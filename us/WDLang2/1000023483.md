


## &lt;Looper&gt;.CancelSwipe (Function)

***In french: &lt;Zone répétée&gt;.AnnuleBalayage***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancels the display of the internal window displayed in a Looper control further to the row swipe.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Code of cancelation button in the internal window
LOOP_CustomerLooper.CancelSwipe()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Looper control>.CancelSwipe()`
---

**`<Looper control>: (Control name)`**

Name of the Looper control to be used.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks
You can manage the swipe of a row in a Looper control. The swipe can:

- Perform no action.

- Run code.

- Automatically delete the swiped element.

- Display an internal window in order to customize the action.




For more details, see [Looper: moves and gestures](../WDChamp/1013295.md).

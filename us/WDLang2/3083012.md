


## LooperDeleteAll (Function)

***In french: ZoneRépétéeSupprimeTout***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes all rows from a Looper control populated programmatically.










<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Reset
LooperDeleteAll(LOOP_Looper1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`LooperDeleteAll(<Looper control>)`
---

**`<Looper control>: (Control name)`**

Name of the Looper control to be used.
If this parameter is equal to an empty string (""), the deletion will be performed in the current Looper control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


### Use conditions
<a name="use_conditions_ELTPARAGRAPHE000096"></a>

**LooperDeleteAll** can only be used on a Looper control populated programmatically.

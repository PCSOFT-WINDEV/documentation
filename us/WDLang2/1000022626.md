


## &lt;Sliding Banner&gt;.Next (Function)

***In french: &lt;Bandeau défilant&gt;.Suivant***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays the next plane of a Sliding Banner control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Displays the next plane
BAN_Banner.Next()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Sliding Banner control>.Next()`
---

**`<Sliding Banner control>: (Control name)`**

Name of Sliding Banner control to use.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


- The invisible planes of the Sliding Banner control are ignored.

- If the current plane is the last visible plane, **&lt;Sliding Banner&gt;.Next** displays the first visible plane.

***In french: BandeauLanceDéfilement***

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=warning_big.png"><B>Warning</B><BR>
	From version <B>24</B>, ***BannerStartScrolling*</**>** is kept for backward compatibility. This function is replaced by ***BannerStartSliding*</**>**.
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enables the automatic slide on a Sliding Banner control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Start sliding
BannerStartSliding(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`BannerStartSliding(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Control name)`**

Name of Sliding Banner control to use.




## BannerNext (Function)

***In french: BandeauSuivant***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays the next plane of a Sliding Banner control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Displays the next plane
BannerNext(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`BannerNext(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Control name)`**

Name of Sliding Banner control to use.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


- The invisible planes of the Sliding Banner control are ignored.

- If the current plane is the last visible plane, **BannerNext** displays the first visible plane.




## ImageStopScrolling (Function)

***In french: ImageArrêteDéfilement***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Stops the automatic scroll of images.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Stops the scroll
ImageStopScrolling(IMG_Banner)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`ImageStopScrolling(<Image control>)`
---

**`<Image control>: (Control name)`**

Name of control to use:

- Image control.

- Clickable Image control.

- Thumbnail control.

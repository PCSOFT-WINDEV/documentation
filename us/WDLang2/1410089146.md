
## WLanguage procedure called by PopupDisplayPage



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
WLanguage procedure ("callback") called by [PopupDisplayPage](../WDLang2/1410087258.md) after the popup page is closed. This is a browser procedure, and it can be a local, global or internal procedure.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`PopupAffichePage_Callback(<Return value>)`
---

**`<Return value>: (Type of the return value)`**

Value that will be returned by the page via the [PopupClosePage](../WDLang2/1410087259.md) function.

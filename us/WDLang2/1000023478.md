
## GalleryClosePopup (Function)

***In french: GalerieFermePopup***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Closes the automatic zoom popup for the image gallery.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
GalleryClosePopup(LOOP_Gallery)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`GalleryClosePopup(<Looper control>)`
---

**`<Looper control>: (Control name)`**

Name of the Looper control in Gallery mode to use.

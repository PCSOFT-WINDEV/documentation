


## &lt;Looper&gt;.ExpandAll (Function)

***In french: &lt;Zone répétée&gt;.DérouleTout***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Expands all breaks in a Looper control with break.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Click the [1.2 stars only] Button control
LOOP_HOTEL.ExpandAll()
LOOP_HOTEL.Collapse(nRow3Star)
LOOP_HOTEL.Collapse(nRow4Star)
LOOP_HOTEL.Collapse(nRow5Star)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Looper control>.ExpandAll()`
---

**`<Looper control>: (Control name)`**

Name of the Looper control to be used.




## Remarks


### Use conditions
<a name="use_conditions_ELTPARAGRAPHE000167"></a>

**&lt;Looper&gt;.ExpandAll** can be used on: 

- a Looper control based on a data file, 

- a Looper control populated programmatically.

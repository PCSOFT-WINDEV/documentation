## Overview
<a name="overview_ELTTEXTE000115"></a>
This help page presents the specific features of the user version of query editor:

- Type of queries

- Data source of report.

- Vertical panes.




**Remark**: The ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=picto_nondispo4.gif) icon (found in the online help) indicates the differences between the user version of query editor and the standard query editor.

## Specific features
<a name="specific_features_ELTTEXTE000139"></a>


### Type of queries
<a name="type_queries_ELTPARAGRAPHE000026"></a>

The user version of query editor cannot be used to create:

- insert queries.

- update queries.

- delete queries.

- queries in SQL code that use the following commands: [INSERT](../Editeurs/2034001.md), [UPDATE](../Editeurs/2034001.md), [DELETE](../Editeurs/2034001.md)




**Reminder**: The user version of the query editor does not allow you to modify the database of the current project.
<a name="NOTE2_2"></a>


### Data source of a query
<a name="data_source_query_ELTPARAGRAPHE000048"></a>

The files and the items that can be used in a query are defined in the data model editor of WINDEV. For more details on the requirements to use a WINDEV application with Reports and Queries, see [Distributing Reports & Queries](../Presentation/3088002.md).
<a name="NOTE2_3"></a>


### Dockable panes
<a name="dockable_panes_ELTPARAGRAPHE000058"></a>

In the user version of query editor, the dockable panes allow you to access:

- the main features of the user query editor editor.

- the accessible elements (reports, queries, ...) of your project.




For more details on dockable panes, see [Dockable panes](../Editeurs/2027001.md).

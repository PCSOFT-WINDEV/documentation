
## Copying controls from the page editor
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000502"></a>
The controls can be copied from the page editor: 

- to the window editor.

- to the report editor.

<a name="NOTE2_1"></a>

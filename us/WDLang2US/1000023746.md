


## &lt;Image&gt;.StopScrolling (Function)

***In french: &lt;Image&gt;.ArrêteDéfilement***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Stops the automatic scroll of images.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Stops the scroll
IMG_Banner.StopScrolling()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Image control>.StopSliding()`
---

**`<Image control>: (Control name)`**

Name of control to use:

- Image control.

- Clickable Image control.

- Thumbnail control.

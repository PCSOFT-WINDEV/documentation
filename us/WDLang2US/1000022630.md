


## &lt;Sliding Banner&gt;.Last (Function)

***In french: &lt;Bandeau défilant&gt;.Dernier***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays the last plane of a Sliding Banner control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Display the last banner plane (the last banner)
BAN_Banner.Last()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Sliding Banner control>.Last()`
---

**`<Sliding Banner control>: (Control name)`**

Name of Sliding Banner control to use.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks
The invisible planes of the Sliding Banner control are ignored.

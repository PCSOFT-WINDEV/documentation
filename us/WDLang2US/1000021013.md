
## FullScreenDisable (Function)

***In french: PleinEcranDésactive***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Disables the "Full screen" mode of browser. 

Remark: To enable the "Full screen" mode, use [FullScreenEnable](../WDLang2/1000021012.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
FullScreenDisable()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`FullScreenDisable()`




## BannerPrevious (Function)

***In french: BandeauPrécédent***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays the previous plane of a Sliding Banner control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Return to previous plane
BannerPrevious(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`BannerPrevious(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Control name)`**

Name of Sliding Banner control to use.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


- The invisible planes of the Sliding Banner control are ignored.

- If the current plane is the first visible plane, **BannerPrevious** displays the last visible plane.




## XMLRename (Function)

***In french: XMLRenomme***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Renames the current element (tag or attribute) in an XML document.






<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = XMLRename(<XML document> , <New element name>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the modification is performed,

- <u><u><u><u>False</u></u></u></u> otherwise (the [ErrorOccurred](../WDLang1/3087001.md) variable is set to <u><u><u><u>True</u></u></u></u>). To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md).




**`<XML document>: (Character string)`**

Name of the XML document where the modification will be performed.

**`<New element name>: (Character string)`**

New name of current element:

- name of tag if the current element is a tag.

- name of attribute if the current element is an attribute.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


### Reading the name of current element
<a name="reading_the_name_current_element_ELTPARAGRAPHE000086"></a>

To read the name of the current element, use [XMLElementName](../WDLang5/3081026.md).
<a name="NOTE0_2"></a>


### Saving the modifications in the XML file
<a name="saving_the_modifications_the_xml_file_ELTPARAGRAPHE000096"></a>

To save the modifications performed in an XML file, you must:

1. Use the [XMLBuildString](../WDLang5/3081016.md) function. This function retrieves and formats the content of an XML document.

2. Save the character string generated by [XMLBuildString](../WDLang5/3081016.md) in an XML file. To do so, use the [fSaveText](../WDLang1/3036046.md) function.




Example:

```wl
// Once the XML document was modified 
XMLSource = XMLBuildString("XMLDoc")
// Save the XML file
fSaveText("ModifiedExample.xml", XMLSource)
```

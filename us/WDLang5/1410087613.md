


## HTMLUnwrapAllChildren (Function)

***In french: HTMLDésencadreTousLesFils***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes the specified node in an HTML document without deleting its children, which take its place. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
hDoc is htmlDocument = "
<html>
	<head/>
	<body>
		<h1></h1><div id="div">
			<p>Text1</p>
			<p>Texte2</p>
		</div><h3></h3>
	</body>
</html>
"
hNode is htmlNode <- HTMLChercheElémentParID(hDoc, "div")
hNodeInsert is htmlNode

hNodeInsert..TagName = "h2"
HTMLWrap(hNode, hNodeInsert)
// body: <h1/><h2><div id="div"><p>Text1</p><p>Text2</p></div></h2><h3/>
Trace(hDoc)
hNodeInsert..TagName = "span"
HTMLWrapAllChildren(hNode, hNodeInsert)
// body: <h1/><h2><div id="div"><span><p>Text1</p><p>Text2</p></span></div></h2><h3/>
Trace(hDoc)
HTMLUnwrapAllChildren(hNode)
// body: <h1/><h2><span><p>Text1</p><p>Text2</p></span></h2><h3/>
Trace(hDoc)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`HTMLUnwrapAllChildren(<Node>)`
---

**`<Node>: (htmlNode variable)`**

Name of the [htmlNode](../WDLang5/1000026081.md) variable that corresponds to the node to be removed.

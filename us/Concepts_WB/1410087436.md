<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
The controls of a page can be linked to a data file or query. These controls can also be linked to a query created when building the control.

In this case, the query is included in the page. It is in the WWH file corresponding to the page. If the WWH file is copied (to another project, for example), the embedded queries used by the page will also be copied.




## WDSQL: Exporting and printing the result of an SQL query
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Présentation
<a name="presentation_ELTTEXTE000106"></a>
Le résultat d'une requête de sélection peut être au choix :

- exporté vers un fichier texte, Excel, XML ou Word,

- imprimé.


**Remarque** : Pour les opérations suivantes, nous considérons que le résultat de la requête est affiché sous WDSQL.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Exporter le résultat d'une requête de sélection
<a name="exporter_resultat_une_requete_selection_ELTTEXTE000130"></a>


### Exporter le résultat de la requête de sélection vers un fichier texte
<a name="exporter_resultat_requete_selection_vers_fichier_texte_ELTPARAGRAPHE000023"></a>Pour exporter le résultat de la requête de sélection vers un fichier texte :

1. Cliquez sur le bouton "Exporter". La fenêtre de sélection du fichier texte s'affiche.

2. Spécifiez le nom et l'emplacement du fichier texte (extension ".TXT" par défaut).

3. Choisissez le type de séparateur utilisé entre les différentes rubriques de chaque enregistrement (option "Type") :

	- ***Séparateur : Tabulation*** : Les rubriques seront séparées par des tabulations.

	- ***Séparateur : Point virgule*** : Les rubriques seront séparées par des points virgule.




4. Validez. Les enregistrements du résultat de la requête sont automatiquement exportés dans le fichier texte.


![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=ExporterFichierTexte.gif)

<a name="NOTE2_2"></a>


### Exporter le résultat de la requête de sélection vers Excel, Word ou XML
<a name="exporter_resultat_requete_selection_vers_excel_word_xml_ELTPARAGRAPHE000040"></a>Pour exporter le résultat de la requête de sélection vers Excel, Word ou XML :

1. Effectuez un clic droit de la souris sur la table contenant le résultat de la requête.

2. Sélectionnez le mode d'export voulu : Word, Excel ou XML.

3. Indiquez le nom du fichier d'export et validez.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Imprimer le résultat d'une requête de sélection
<a name="imprimer_resultat_une_requete_selection_ELTTEXTE000160"></a>
Pour imprimer le résultat de la requête de sélection :

1. Cliquez sur le bouton "Imprimer". La fenêtre de sélection des rubriques à imprimer s'ouvre.

2. Spécifiez et validez les rubriques à imprimer. Le résultat de la requête est affiché dans un aperçu avant impression.

3. Cliquez sur l'icône ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=WDSQL%20-%20HC%20N%B0006%202.gif)
 pour imprimer le résultat de la requête.

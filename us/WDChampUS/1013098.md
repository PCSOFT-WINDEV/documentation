### Control in a dynamic or static page
<a name="control_dynamic_static_page_ELTPARAGRAPHE000016"></a>

The Flash control is used to display a Flash® animation in a WEBDEV page.

**To create a Flash control:**

1. On the "Creation" tab, in the "Integration" group, expand "Plug-in" and click "Flash control".

2. Click where you want to create the control in the page. The control appears in the editor.

3. In the control description ("General" tab), associate the Flash® animation (".swf" file) with the Flash control.




**Remark**: the Flash® animation should be located in the image directory (&lt;Project_Name&gt;_WEB directory).

To view the characteristics of the control, select "Description" in the context menu.

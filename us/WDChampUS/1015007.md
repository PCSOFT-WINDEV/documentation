
## Properties associated with the rows of a TreeView Table control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000108"></a>
The rows of TreeView Table controls in a window or in a page can be handled by WLanguage properties.

**Remark**: To use a row of a Treeview Table control with WLanguage properties, use the following notation: &lt;TreeView Table control&gt;[&lt;Element index&gt;]..&lt;Property&gt;.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

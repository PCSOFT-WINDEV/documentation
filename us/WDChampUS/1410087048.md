
## Layout control properties
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000126"></a>
You can handle Layout controls (and its cells) in a window or page using the following WLanguage properties.

<a name="NOTE2_1"></a>

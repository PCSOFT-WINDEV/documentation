
## H.NbRecModificationQuery

***In french: H.NbEnregRequêteModification***

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Used to find out the number of records added, modified or deleted when running the last query of the current context. 

This variable is read-only. It is updated when running an add, update or delete query (especially through programming with [HExecuteQuery](../WDLang4/3044080.md) and [HExecuteSQLQuery](../WDLang4/3044084.md)).

![Native Connectors (Native Accesses)](https://doc.pcsoft.fr/ext/images/us/AN.png) Access via a Native Oracle Connector: This variable is also updated.

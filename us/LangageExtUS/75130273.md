
## Close (External language)

***In french: Ferme***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Closes the current window and restores the context that existed before the window opening.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```txt
// In C
CALLWD("Close");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Close');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Close")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`Close()`

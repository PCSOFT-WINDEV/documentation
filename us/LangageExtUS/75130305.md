
## WavInit (External language)

***In french: SonInit***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Initializes the sound interpreter. This function must be run once in each program. It is used to calibrate the execution speed of the sounds. This function may take a few minutes.
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`SoundInit([<Sound Initialization>])`
---

**`<Sound Initialization>: (Character string (optional))`**

 Name of the initialization file that will be used to initialize the sound interpretor.

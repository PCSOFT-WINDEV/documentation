
## BurnerMediaFound (Function)

***In french: GraveurMédiaPrésent***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Used to find out whether the burner is empty or not. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MediaFound is boolean
MediaFound = BurnerMediaFound()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = BurnerMediaFound()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if a media is found in the burner. 

- <u><u><u><u>False</u></u></u></u> otherwise.

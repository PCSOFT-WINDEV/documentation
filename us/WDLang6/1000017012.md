
## LockAppOff (Function)

***In french: VerrouAppliDébloque***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Unlocks an application that was locked automatically or by [LockAppOn](../WDLang6/1000017011.md).

**Remark**: This function is mainly used in the window that unlocks the application. It can also be used to unlock an application by programming (use of a specific keyboard shortcut for instance).
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
LockAppOff()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`LockAppOff()`




## Lock functions
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
The following functions are used to lock an application:



|   |   |
| --- | --- |
| [LockAppOff](../WDLang6/1000017012.md) | Unlocks an application that was locked automatically or by [LockAppPassword](../WDLang6/1000017011.md). |
| [LockAppOn](../WDLang6/1000017011.md) | Locks a WINDEV application. |
| [LockAppPassword](../WDLang6/1000017010.md) | Specifies a specific password for unlocking the current application (application locked automatically or by  [LockAppOn](../WDLang6/1000017011.md)). |
| [LockAppTimeout](../WDLang6/1000017006.md) | Returns or modifies the idle time before a WINDEV application is locked. |
| [ValidatePassword](../WDLang6/1000017009.md) | Indicates whether the specified password is valid for the current user. |

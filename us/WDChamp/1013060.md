
## Properties associated with Combo Box controls that display a table
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000068"></a>
A Combo Box control can display a table when it is opened. There are several properties available for this type of control. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>




## Properties associated with a group of controls
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000096"></a>
Several WLanguage properties can be used on a group of controls found in a window, in a page or in a report.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

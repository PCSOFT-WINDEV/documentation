


## Properties associated with a report
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


The following properties can be used on reports:



|   |   |
| --- | --- |
| [AllModifiableWithReportsAndQueries](../Proprietes/2511076.md) | The **AllModifiableWithReportsAndQueries** property is used to determine if a report can be modified in Reports & Queries. |
| [BackgroundImage](../Proprietes/2510056.md) | The **BackgroundImage** property is used to identify and modify the background color of an element: window, page, control or report. |
| [BackgroundImagePrinted](../Proprietes/1000021350.md) | The **BackgroundImagePrinted** is used to: <br><br>- Determine if the background image of a report (form or other) should be printed.<br><br>- Print (or not) the background image of a report. <br><br><br> |
| [BottomMargin](../Proprietes/2511051.md) | The **BottomMargin** property is used to:<br><br>- Find out the bottom margin of a report.<br><br>- Modify the bottom margin of a report.<br><br><br> |
| [BrowsedItem](../Proprietes/2510111.md) | The **BrowsedItem** property is used to:<br><br>- Get the item used to automatically loop through Table, Looper, List Box, ListView and Combo Box controls, supercontrols and reports based on data files.<br><br>- Change the item used to automatically loop through Table, Looper, List Box, ListView and Combo Box controls, supercontrols and reports based on data files. Then, the iteration is reinitialized.<br><br><br> |
| [HSpace](../Proprietes/2511067.md) | The **HSpace** property is used to determine and modify the horizontal spacing between labels in a report. This spacing was defined when creating the Label report. |
| [HTML](../Proprietes/2511040.md) | The **HTML** property is used to: <br><br>- find out whether the current report is printed in HTML mode. <br><br>- manage the HTML type of a Static control found in a report.<br><br><br> |
| [InterfaceModifiableWithReportsAndQueries](../Proprietes/2511075.md) | The **InterfaceModifiableWitReportsAndQueries** property is used to determine if the **interface** of a report can be modified in Reports & Queries. |
| [KeepVisibleHierarchy](../Proprietes/1000021055.md) | The **KeepVisibleHierarchy** property is used to:<br><br>- Find out which elements of the TreeView table will be printed: printing the entire hierarchy or printing the hierarchy that is visible in the control.<br><br>- Modify the print mode of the hierarchy: printing the entire hierarchy or printing the hierarchy that is visible in the control.<br><br><br> |
| [LabelHeight](../Proprietes/2511069.md) | The **LabelHeight** property is used to:<br><br>- Find out the height of labels in a Label report.<br><br>- Modify the height of labels in a Label report.<br><br><br> |
| [LabelNumberToStart](../Proprietes/2511032.md) | The **LabelNumberToStart** property is used to:<br><br>- Find out the number of the label on the label sheet from which the print will be performed.<br><br>- Modify the number of the label from which the print will be performed on the label sheet.<br><br><br> |
| [LabelWidth](../Proprietes/2511055.md) | The **LabelWidth** property is used to:<br><br>- Find out the width of labels in a "Label" report.<br><br>- Modify the width of labels in a "Label" report.<br><br><br> |
| [LeftMargin](../Proprietes/2511048.md) | The **LeftMargin** property is used to:<br><br>- Find out the left margin of a report.<br><br>- Modify the left margin of a report.<br><br><br> |
| [MaxValue](../Proprietes/2510009.md) | The **MaxValue** property is used to:<br><br>- find out and modify the upper bound of a report based on a data file. <br><br>- find out the upper bound defined for the current filter.<br><br>- find out the maximum value of a chart series.<br><br>- find out and modify the maximum value for some types of controls.<br><br><br> |
| [MemZoneName](../Proprietes/2511029.md) | The **MemZoneName** property is used to:<br><br>- find out the name of the memory area used as data source for a report.<br><br>- modify the name of the memory area used as data source for a report.<br><br><br> |
| [MinValue](../Proprietes/2510008.md) | The **MinValue** property is used to:<br><br>- find out and modify the lower bound of a report based on a data file. <br><br>- find out the lower bound defined for the current filter.<br><br>- find out the minimum value of a chart series.<br><br>- find out and modify the minimum value for some types of controls. <br><br><br> |
| [MsgSortSel](../Proprietes/2511045.md) | The **MsgSortSel** property is used to:<br><br>- Find out the wait message displayed when sorting and selecting the report records.<br><br>- Modify the wait message displayed when sorting and selecting the report records.<br><br><br> |
| [Name](../Proprietes/2510082.md) | The **Name** property is used to get:<br><br>- the name of the font associated with a Font variable.<br><br>- the name of a control, report or report block.<br><br>- the name of a control, group of controls or window.<br><br>- the name of a control, group of controls or page.<br><br><br> |
| [NbColumn](../Proprietes/1000018810.md) | The **NbColumn** property is used to get the number of columns in a report. |
| [NbCopy](../Proprietes/2511042.md) | The **NbCopy** property is used to determine and change the number of report copies to print. |
| [NbLabel](../Proprietes/2511050.md) | The **NbLabel** property is used to:<br><br>- Find out the number of labels printed across the page width.<br><br>- Modify the number of labels printed across the page width.<br><br><br> |
| [NbLabelCopy](../Proprietes/2511059.md) | The **NbLabelCopy** property is used to:<br><br>- Get the number of copies of identical labels.<br><br>- Change the number of copies of identical labels.<br><br><br> |
| [NbPrintedRec](../Proprietes/2511058.md) | The **NbPrintedRec** property is used to determine the number of records already printed. |
| [NbRecRead](../Proprietes/2511041.md) | **NbRecRead** is used to find out the number of records read: selected both by the source of the report AND by the filter code (if it exists). |
| [ODBCConnectionName](../Proprietes/2511033.md) | The **ODBCConnectionName** property is used to:<br><br>- Identify the name of the ODBC connection used when printing a report based on a "query performed on a connection".<br><br>- Specify the name of the ODBC connection used when printing a report based on a "query performed on a connection".<br><br><br> |
| [ODBCDatabaseName](../Proprietes/2511034.md) | **ODBCDatabaseName** is used to:<br><br>- Find out the name of the database used when printing a report based on a "query performed on a connection".<br><br>- Specify the name of the database used when printing a report based on a "query performed on a connection".<br><br><br> |
| [ODBCPassword](../Proprietes/2511031.md) | The **ODBCPassword** property is used to:<br><br>- Identify the password used to connect to a database via an ODBC driver or a Native Connector. <br><br>- Specify the password used to connect to a database via an ODBC driver or a Native Connector.<br><br><br> |
| [ODBCSQLCode](../Proprietes/2511028.md) | **ODBCSQLCode** is used to:<br><br>- Find out the SQL code of the "query performed on a connection" (data source of report). <br><br>- Modify the SQL code of the "query performed on a connection" (data source of report).<br><br><br> |
| [ODBCUserName](../Proprietes/2511019.md) | **ODBCUserName** is used to:<br><br>- Identify the username used to connect to a database via an ODBC driver or a Native Connector. <br><br>- Specify the username used to connect to a database via an ODBC driver or a Native Connector. <br><br><br> |
| [PageByPagePrint](../Proprietes/2511057.md) | The **PageByPagePrint** property is used to:<br><br>- Ask a report to be printed page by page. Each page of the report will be sent separately to the printer spooler. <br><br>- Find out the print mode of the report (page by page or all at once).<br><br><br> |
| [PageEndStick](../Proprietes/2511074.md) | The **PageEndStick** property is used to:<br><br>- Find out whether the page footer must be stuck to the last block.<br><br>- Modify the management mode of the Page Footer block.<br><br><br> |
| [PageNumber](../Proprietes/2510106.md) | The **PageNumber** property is used to: <br><br>- get and change the number of the page displayed in an Image control ("multi-page" image files)<br><br>- get and change the page number of a PDF file displayed in an image control of a window<br><br>- get and change the page number of a PDF file displayed in the background of a report. <br><br><br> |
| [Plane](../Proprietes/2510094.md) | The **Plane** property is used to:<br><br>- Get: <br><br>	- the active plane of a window (or report) or the plane to which a control belongs.<br><br>	- the active plane of a WEBDEV element.<br><br><br><br><br>- Change: <br><br>	- the active plane of a window (or report) or associate a control with another plane.<br><br>	- the active plane of a WEBDEV element.<br><br><br><br><br><br> |
| [Process](../Proprietes/1000020878.md) | The **Process** property is used to modify the processes to be executed by the events available on a control, window, page, report, etc. |
| [ReturnedValue](../Proprietes/2510134.md) | The **ReturnedValue** property is used to:<br><br>- Get and change the value returned by a window, page or report.<br><br>- Get and change the value returned by an option in a Radio Button control. <br><br><br> |
| [RightMargin](../Proprietes/2511053.md) | The **RightMargin** property is used to:<br><br>- Find out the right margin of a report.<br><br>- Modify the right margin of a report.<br><br><br> |
| [RightToLeft](../Proprietes/1000017228.md) | The **RightToLeft** property is used to get the writing direction on a control, column, window or report. |
| [SelectionOnly](../Proprietes/2511017.md) | The **SelectionOnly** property is used to:<br><br>- Find out which elements of the Table or TreeView Table control will be printed: print all the records or the selected records only.<br><br>- Change the print mode of rows in the Table or TreeView Table control: print all records or the selected records only.<br><br><br> |
| [SourceName](../Proprietes/2511035.md) | The **SourceName** property is used to:<br><br>- Get the name of the report data source.<br><br>- Change the report data source.<br><br><br> |
| [TextDirection](../Proprietes/2513020.md) | The **TextDirection** property is used to determine and change the writing direction in a window, report or control (useful for languages such as Arabic or Hebrew). |
| [TopMargin](../Proprietes/2511043.md) | The **TopMargin** property is used to:<br><br>- Find out the top margin of a report.<br><br>- Modify the top margin of a report.<br><br><br> |
| [TotalNbRec](../Proprietes/2511047.md) | The **TotalNbRec** property is used to:<br><br>- Get the number of records to be printed.<br><br>- Change the number of records to be printed.<br><br><br> |
| [ViewName](../Proprietes/2511030.md) | The **ViewName** property is used to:<br><br>- find out the name of the HFSQL view, data source of a report<br><br>- modify the name of the HFSQL view, data source of a report.<br><br><br> |
| [VSpace](../Proprietes/2511065.md) | The **VSpace** property is used to get and change the vertical spacing between the labels in a report. This spacing was defined when creating the Label report. |

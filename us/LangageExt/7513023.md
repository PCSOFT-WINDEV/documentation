
## Beep (External language)

***In french: Beep***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Used to emit a beep on the speaker of the computer.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```txt
// In C
CALLWD("Beep");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Beep');
```


<a name="Example3"></a>





```txt
// In VB
Call CALLWD("Beep")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`Beep()`

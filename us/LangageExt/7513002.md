
## WDContinueError: Resume on error
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Use
<a name="use_ELTTEXTE000066"></a>
Forces the execution to resume after an error.

<a name="NOTE2_1"></a>


## Syntax
<a name="syntax_ELTTEXTE000090"></a>

```txt
void      WDContinueError(void);
```

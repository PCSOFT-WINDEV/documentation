
## Tangent (Function)

***In french: Tang***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calculates the tangent of an angle.




<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Tangent of 45
TangentRes = Tangent(45)
// Returns 1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = Tangent(<Angle>)`
---

**`<Result>: (Real)`**

Tangent of the specified angle.

**`<Angle>: (Integer or real)`**

Angle (in degrees) to use. 
A WLanguage error occurs if this parameter is set to 90°, -90°, 270° or -270°.

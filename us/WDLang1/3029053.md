


## dModifySaturation (Function)

***In french: dModifieSaturation***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifies the saturation of an image found: 

- in an Image control, 

- in a variable of type [Image](../WDLang1/1000019650.md).
	


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Caution**: A specific configuration is required to use this function in Linux. For more details, see [The drawings](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Changes the saturation of the image found in the IMG_Image1 control
dModifyLightness(IMG_Image1, 50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = dModifySaturation(<Source image> , <Variation of saturation> [, <Destination image>])`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the change was performed, 

- <u><u><u><u>False</u></u></u></u> otherwise.




**`<Source image>: (Character string or Image variable)`**

Image to use. The image can correspond to: 

- the name of an Image control. 

- the name of a variable of type [Image](../WDLang1/1000019650.md).
	




**`<Variation of saturation>: (Integer)`**

Value included between -100 and +100. This value will be added to the current saturation of each pixel to get a new saturation.

**`<Destination image>: (Optional character string or Image variable)`**

Image where the transformed image will be drawn. The image can correspond to: 

- the name of an Image control. 

- the name of a variable of type [Image](../WDLang1/1000019650.md).
	




If this parameter is not specified, the transformation will be performed on the &lt;Source image&gt;.

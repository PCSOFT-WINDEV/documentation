
## docElementText (Type of variable)

***In french: docElémentTexte***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **docElementText** type is used to handle the characteristics of a Text element found in a [docElement](../WDLang1/1000022484.md) variable.

The characteristics of this type of variable can be handled by several WLanguage properties. 

**CAUTION: This type cannot be used directly. It must be used via the [docElement](../WDLang1/1000022484.md) type**.


<a name="NOTE0"></a>

## Remarks
<a name="NOTE0_1"></a>


### Properties specific to the description of docElementText variables
<a name="properties_specific_the_description_docelementtext_variables_ELTPARAGRAPHE000030"></a>

The following properties can be used to handle a **docElementText** variable:

   | --- |
| --- | --- | --- |
| Property name | Type used | Effect |
| Formatting | [docFormatting](../WDLang1/1000022817.md) variable | Informations for element formatting. |
| Value | Character string | Text of element. |

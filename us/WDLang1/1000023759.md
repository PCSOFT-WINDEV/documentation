


## &lt;TreeView&gt;.CollapseAll (Function)

***In french: &lt;Arbre&gt;.EnrouleTout***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Collapses a branch previously expanded (as well as its child elements): 

- The "child" nodes are no longer visible in the TreeView control. 

- The child nodes are collapsed.




Example of TreeView control:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=Arbre_Vocabulaire.gif)
**Reminder**: At runtime, a node is collapsed when the node name is preceded by "+".
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Collapse the "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TREE_TVRecipe.CollapseAll("Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = <TreeView control>.CollapseAll([<Node to collapse>])`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the node and its children have been collapsed,

- <u><u><u><u>False</u></u></u></u> otherwise.




**`<TreeView control>: (Control name)`**

Name of the TreeView control to be used.

**`<Node to collapse>: (Optional character string)`**

Path of the node to collapse. This parameter has the following format:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
If this parameter is not specified, the TreeView control is collapsed from the root.

If there are two elements with the same name in the path, you can add the IDs of the elements (after their names using [TreeID](../WDLang1/3018027.md)).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


### Node to collapse
<a name="node_collapse_ELTPARAGRAPHE000197"></a>

If the node to collapse:

- is not found, **&lt;TreeView&gt;.CollapseAll** returns <u><u><u><u>False</u></u></u></u>.

- is not specified, **&lt;TreeView&gt;.CollapseAll** returns <u><u><u><u>True</u></u></u></u>. The TreeView control is collapsed from its root.

- is a leaf, **&lt;TreeView&gt;.CollapseAll** returns <u><u><u><u>True</u></u></u></u>.

- is already collapsed, **&lt;TreeView&gt;.CollapseAll** returns <u><u><u><u>True</u></u></u></u>.



<a name="NOTE0_2"></a>


### Selection bar
<a name="selection_bar_ELTPARAGRAPHE000236"></a>

- If the selection bar was positioned on one of the elements of the node to collapse, it is positioned on the collapsed node after the call to **&lt;TreeView&gt;.CollapseAll**.

- The modification code is called whenever the selection bar is moved. If two levels are collapsed and if the selection bar is positioned on the higher node, the modification codes of each level are run.

- The full path of the selected element is returned by [&lt;TreeView&gt;.Select](../WDLang1/1000023767.md).

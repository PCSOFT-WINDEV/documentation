


## EventDestroy (Function)

***In french: SignalDétruit***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Explicitly destroys an event.

**Remark**: All the events are destroyed when the application ends.










<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`EventDestroy(<Event>)`
---

**`<Event>: (Character string)`**

Name of the event to destroy.

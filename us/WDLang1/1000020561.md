
## IWListDeleteAll (Function)

***In french: FIListeSupprimeTout***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes all the internal windows from the list of internal windows browsed by an Internal Window control..

**Remark**: If an internal window is defined in edit for the Internal Window control, this window will be displayed after the call to this function.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
IWListDeleteAll(IW_InternalWindowControl)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`IWListDeleteAll(<Internal Window control>)`
---

**`<Internal Window control>: (Control name)`**

Name of the Internal Window control to use.


## StatCorrelation (Function)

***In french: StatCorrelation***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calculates the coefficient of correlation between two series of values.




<a name="Example1"></a>
<a name="sample_code"></a>

## Example
<a class="notetitle" target="_blank" href="$DOC$=1000003050022&name=statcorrelation_function&product=WD">See additional examples</a>

```wl
// Calculate the coefficient of correlation between row 1 and row 2 of a matrix
ResCorrelationCoef = StatCorrelation("MyMatrix", 1, 2, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = StatCorrelation(<Matrix> [, <1st row/1st column> [, <2nd row/2nd column> [, <Row/Column>]]])`
---

**`<Result>: (Real)`**



- Coefficient of correlation that exist between the two series of values, 

- 0 if the coefficient of correlation is null.




**`<Matrix>: (Character string)`**

Name of the matrix containing the data, defined by [MatCreate](../WDLang1/3050058.md).

**`<1st row/1st column>: (Optional integer)`**

1st row or 1st column containing the data (1 by default).

**`<2nd row/2nd column>: (Optional integer)`**

2nd row or 2nd column containing the data (2 by default).

**`<Row/Column>: (Optional boolean)`**



- <u><u><u><u>True</u></u></u></u> (by default) if the series of values correspond to matrix columns, 

- <u><u><u><u>False</u></u></u></u> if the series of values correspond to matrix rows.

## Remarks




### Calculating the coefficient of correlation
<a name="calculating_the_coefficient_correlation_ELTPARAGRAPHE000094"></a>

![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=STATcORReLATION1.gif)
 StatCorrelation("MyMatrix", 1, 3, False) returns 0,997054.
![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=STATcORReLATION2.gif)
 StatCorrelation("MyMatrix", 1, 3, False) returns 1.
![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=Statcorrelation3.gif)
 StatCorrelation("MyMatrix", 1, 3, False) returns 0.
<a name="NOTE0_2"></a>




### Number of elements found in the matrix
<a name="number_elements_found_the_matrix_ELTPARAGRAPHE000109"></a>

The number of elements found in the matrix is taken as reference. This number of elements corresponds to:

- the number of matrix rows, returned by [MatNbLine](../WDLang1/3050060.md).

- the number of matrix columns, returned by [MatNbColumn](../WDLang1/3050073.md).




If the series contain less elements than the matrix, the missing values are automatically filled with 0. These values are taken into account during the calculations.
<a name="NOTE0_3"></a>




### Managing the errors
<a name="managing_the_errors_ELTPARAGRAPHE000128"></a>

**Caution**: **StatCorrelation** returns no error code. To find out whether errors have been generated when calculating the coefficient of correlation, use [StatError](../WDLang1/3050019.md). To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md) with the *errMessage* constant.

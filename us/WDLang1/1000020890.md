


## GanttOrganizeTask (Function)

***In french: GanttOrganiseTâche***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reorganizes the tasks of a Gantt chart according to the constraints (dependencies and resources). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Réorganise les tâches
GanttOrganiseTâche(COL_Gantt)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`GanttOrganizeTask(<Gantt column>)`
---

**`<Gantt column>: (Control name)`**

Name of the Gantt Chart column (in a Table or TreeView Table control) to be used.




## SpreadsheetDeleteWorksheet (Function)

***In french: TableurSupprimeFeuille***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes a worksheet from a Spreadsheet control found in a window. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Deletes all worksheets except for the first one
FOR I = 1 TO PSHEET_MySpreadsheet.NbWorksheet TO 2 STEP -1
	SpreadsheetDeleteWorksheet(PSHEET_MySpreadsheet, i)
END
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

### Deleting a worksheet identified by its number

`SpreadsheetDeleteWorksheet(<Spreadsheet control> [, <Worksheet number>])`
---

**`<Spreadsheet control>: (Control name)`**

Name of the Spreadsheet control to be used.  

**`<Worksheet number>: (Optional integer)`**

Number of the worksheet to delete. If this parameter is not specified, the current worksheet is deleted. 


<a name="SYNTAX2"></a>

### Deleting a worksheet identified by its name

`SpreadsheetDeleteWorksheet(<Spreadsheet control> , <Worksheet name>)`
---

**`<Spreadsheet control>: (Control name)`**

Name of the Spreadsheet control to be used. 

**`<Worksheet name>: (Character string)`**

Name of the worksheet to delete. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks


- If there is only one worksheet left in the Spreadsheet control, **SpreadsheetDeleteWorksheet** has no effect. 

- This function can only be used on a Spreadsheet control found in a window.




## ArrayInsertReference (Function)

***In french: TableauInsèreRéférence***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adds the reference of an object at given position in a one-dimensional array.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
AnArray is array of AClass
AnObject is AClass
ArrayInsertReference(AnArray, InsertionIndex, AnObject)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`ArrayInsertReference(<WLanguage array> , <Index> , <Object>)`
---

**`<WLanguage array>: (Array)`**

Name of the Array variable to use. This array must be a one-dimensional array.

**`<Index>: (Integer)`**

Index where the element will be inserted into the array. 

**`<Object>: (Object)`**

Name of the object whose reference is added to the array.

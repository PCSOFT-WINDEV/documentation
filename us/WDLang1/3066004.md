
## NetworkUser (Function)

***In french: RéseauUtilisateur***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifies the name of the connected user.

<a name="sample_code"></a>

```wl
// Identify the connected user

```



```wl
// Identify the connected user
```

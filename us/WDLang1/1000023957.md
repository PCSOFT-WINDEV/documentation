


## EditDeleteAllToken (Function)

***In french: SaisieSupprimeToutJeton***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes all the tokens from a "Text token" edit control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
EditDeleteAllToken(EDT_Recipient)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`EditDeleteAllToken(<Edit control>)`
---

**`<Edit control>: (Control name)`**

Name of the "Text token" edit control.

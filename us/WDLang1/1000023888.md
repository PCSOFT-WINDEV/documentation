


## &lt;Window&gt;.Restore (Function)

***In french: &lt;Fenêtre&gt;.Restaure***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays a WINDEV window to its initial size.

**Remark**: **&lt;Window&gt;.Restore** performs the same action as the restore button at the upper-right corner of the windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Display the "WIN_EditCustomer" window in its initial size
WIN_CustomerInput.Restore()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Window>.Restore()`
---

**`<Window>: (Window name)`**

Name of the window that must be displayed in its initial size.



<a name="XComponent"></a>

## Component:
wd280obj.dll

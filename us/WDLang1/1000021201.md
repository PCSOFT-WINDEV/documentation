
## EmulateRadioButton (Function)

***In french: PiloteSélecteur***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emulates the selection of an option in a Radio Button control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Select the 5th option of the radio button
EmulateRadioButton(RADIO_RadioButton, 5)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`EmulateRadioButton(<Control> , <Option number>)`
---

**`<Control>: (Character string)`**

Name of the Radio Button control.

**`<Option number>: (Integer)`**

Index of the option to select in the control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>




## &lt;picLayer variable&gt;.VerticalSymmetry (Function)

***In french: &lt;Variable picCalque&gt;.SymétrieVerticale***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Flips an image contained in a variable of type [picLayer](../WDLang1/1000024604.md) with respect to a vertical axis (symmetry with respect to a vertical axis).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Caution**: A specific configuration is required to use this function in Linux. For more details, see [The drawings](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MyWDPicImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Vertical symmetry of the image contained in the layer
MyResult is boolean
MyResult = MyLayer.VerticalSymmetry()
IF MyResult = False THEN Error("The operation failed")
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = <picLayer image>.VerticalSymmetry()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the symmetry was performed, 

- <u><u><u><u>False</u></u></u></u> otherwise.




**`<picLayer image>: (picLayer variable)`**



Name of the [picLayer](../WDLang1/1000024604.md) variable to be used.

The symmetry is performed in relation to the median vertical axis of the image. The image is directly modified.


## MatFloatMultiply (Function)

***In french: MatRéelMultiplie***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Multiplies each matrix element by a value. This multiplication is performed in the specified matrix.




<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Multiply the matrix elements by 3
MatFloatMultiply("MyMatrix", 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`MatFloatMultiply(<Matrix> , <Value>)`
---

**`<Matrix>: (Character string)`**

Name of the matrix to be used, defined with [MatCreate](../WDLang1/3050058.md).

**`<Value>: (Real)`**

Value used for the multiplication.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks




### Dividing all matrix elements
<a name="dividing_all_matrix_elements_ELTPARAGRAPHE000077"></a>

To divide all matrix elements by a given value, multiply the matrix by: 1/&lt;Value>.
<a name="NOTE0_2"></a>




### Matrix existence
<a name="matrix_existence_ELTPARAGRAPHE000085"></a>

The matrix existence is not checked. To check the matrix existence, use [MatExist](../WDLang1/3050076.md).
<a name="NOTE0_3"></a>




### Managing errors
<a name="managing_errors_ELTPARAGRAPHE000096"></a>

**Caution**: **MatFloatMultiply** returns no error code. To find out whether errors have been generated by the multiplication, use [MatError](../WDLang1/3050044.md). To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md) with the *errMessage* constant.


## Constants for managing projects
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


The following constants are used to manage projects:

| Constant | Description | Used by the functions |
| --- | --- | --- |
| **enumAll** | Enumerates all project elements. | [EnumElement](../WDLang1/3064001.md) |
| **enumContextMenu** | Enumerates the popup menus found in a window. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumDynamicPage** | Enumerates all dynamic pages of project. | [EnumElement](../WDLang1/3064001.md) |
| **enumExternalComponent** | Enumerates the public elements of an external component included in the project. | [EnumElement](../WDLang1/3064001.md) |
| **enumFirst** | Initializes the enumeration. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumGPU** | Constant intended for the User Groupware. | [EnumElement](../WDLang1/3064001.md) |
| **enumGroup** | Enumerates all window groups. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumInternalPage** | Enumerates all internal pages of project. | [EnumElement](../WDLang1/3064001.md) |
| **enumInternalWindow** | Enumerates all internal windows of project. | [EnumElement](../WDLang1/3064001.md) |
| **enumMainMenu** | Enumerates the drop-down menus found in a window. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumPage** | Enumerates all project pages. | [EnumElement](../WDLang1/3064001.md) |
| **enumPageTemplate** | Enumerates the page templates of project. | [EnumElement](../WDLang1/3064001.md) |
| **enumPane** | Enumerates the tab panes. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumQuery** | Enumerates all project queries. | [EnumElement](../WDLang1/3064001.md) |
| **enumReport** | Enumerates all project reports. | [EnumElement](../WDLang1/3064001.md) |
| **enumReportTemplate** | Enumerates the reports templates of project. | [EnumElement](../WDLang1/3064001.md) |
| **enumSideBar** | Enumerates the panes found in a sidebar. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumWindow** | Enumerates all project windows. | [EnumElement](../WDLang1/3064001.md) |
| **enumWindowTemplate** | Enumerates the window templates of project. | [EnumElement](../WDLang1/3064001.md) |
| **piAnalysis** | Type of information to return: Physical name without extension of the .WDD file used by the project or by the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationCCDataPath** | Type of information to return: Path of data directory for the Control Centers defined when creating the executable, the library or the component.<br>If this constants is used in a component, gives the path of the data directory for the application that uses it. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationEmail** | Type of information to return: Email defined when creating the executable, the library or the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationInternetAddress** | Type of information to return: Internet address defined when creating the executable, the library or the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCCDataPath** | Type of information to return: Path of data directory for the Control Centers defined when creating the executable, the library or the component.<br>If this constant is used in the project, gives the path of data directory corresponding to the project. If this constant is used in a component, gives the path of data directory corresponding to the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCompany** | Type of information to return: Name of company specified when creating the executable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piConfiguration** | Type of information to return: Name of project configuration used to generate the executable or the component currently run. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCopyright** | Type of information to return: Copyright defined when creating the executable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piDescription** | Type of information to return: Description of company defined when creating the executable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piEmail** | Type of information to return: Email defined when creating the executable, the library or the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piEXEName** | Type of information to return: Name of executable specified when creating the executable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piGenerationDate** | Type of information to return: Generation date of application or component library (WDL). | [ProjectInfo](../WDLang1/3064004.md) |
| **piInternetAddress** | Type of information to return: Internet address defined when creating the executable, the library or the component. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNameCurrentProject** | Type of information to return: Name of project in which ProjectInfo is called.<br>If this constant is used in the code of main project, the result is the name of main project.<br>If this constant is used in a component, the result corresponds to the name of component project. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNameDeployedSite** | Type of information to return: Real name of the site deployed (for a WEBDEV site during a multi-site deployment). | [ProjectInfo](../WDLang1/3064004.md) |
| **piNewFeaturesHelpContext** | Type of information to return: Context number of the page to display in the help file for the new features. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNewFeaturesHelpName** | Type of information to return: Name of help file for the new features. | [ProjectInfo](../WDLang1/3064004.md) |
| **piProjectName** | Type of information to return: Project name. | [ProjectInfo](../WDLang1/3064004.md) |
| **piRegistry** | Type of information to return: Path in the registry where the information specific to the project can be saved. | [ProjectInfo](../WDLang1/3064004.md) |


## StatAverageDeviation (Function)

***In french: StatEcartMoyen***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calculates the average deviation of values in relation to their arithmetic mean.

**Remark**: The average deviation gives the dispersion in a set of values.




<a name="Example1"></a>
<a name="sample_code"></a>

## Example
<a class="notetitle" target="_blank" href="$DOC$=1000003050024&name=stataveragedeviation_function&product=WD">See additional examples</a>

```wl
// Calculate the average deviation of the elements in row 1 of a matrix
ResAverageDeviation = StatAverageDeviation("MyMatrix", 1, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = StatAverageDeviation(<Matrix> [, <Row/Column number> [, <Row/Column>]])`
---

**`<Result>: (Real)`**



- Average deviation of values, 

- 0 if the average deviation is null.




**`<Matrix>: (Character string)`**

Name of the matrix containing the data, defined by [MatCreate](../WDLang1/3050058.md).

**`<Row/Column number>: (Optional integer)`**

Number of the row or column containing the data (1 by default).

**`<Row/Column>: (Optional boolean)`**



- <u><u><u><u>True</u></u></u></u> (by default) if the series of values corresponds to a matrix column, 

- <u><u><u><u>False</u></u></u></u> if the series of values corresponds to a matrix row.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks




### Number of elements found in a matrix row or in a matrix column
<a name="number_elements_found_matrix_row_matrix_column_ELTPARAGRAPHE000094"></a>

The number of elements found in a matrix row or in a matrix column is taken as a reference. This number of elements corresponds to:

- the total number of matrix rows, returned by [MatNbLine](../WDLang1/3050060.md) (if the calculation is performed on a column), 

- the total number of matrix columns, returned by [MatNbColumn](../WDLang1/3050073.md) (if the calculation is performed on a row).




If the series contain less elements than the matrix, the missing values are automatically filled with 0. These values are taken into account during the calculations.
<a name="NOTE0_2"></a>




### Managing errors
<a name="managing_errors_ELTPARAGRAPHE000113"></a>

**Caution**: **StatAverageDeviation** returns no error code. To find out whether errors have been generating when calculating the average deviation, use [StatError](../WDLang1/3050019.md). To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md) with the *errMessage* constant.

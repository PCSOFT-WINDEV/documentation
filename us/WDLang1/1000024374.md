
## PDF Reader control and PDF file management constants
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
The following constants are used by the functions for managing PDF Reader controls:

| Constant | Detail | Function or type of variable |
| --- | --- | --- |
| **dispWithCoverPage** | Page display mode in the PDF reader: Sets the first page of the PDF file as cover page. The first page is displayed on the right when viewing in 2-page mode. | [pdfParameter](../WDLang6/1410087857.md) type |
| **dispEnableScrolling** | Page display mode in the PDF reader: Display with progressive scrolling: Displays pages in an uninterrupted vertical column occupying the width of one page. Scrolling can be enabled for both 1-page and 2-page display modes. | [pdfParameter](../WDLang6/1410087857.md) type |
| **dispDefault** | Page display mode in the PDF reader: Default display. | [pdfParameter](../WDLang6/1410087857.md) type |
| **dispTwoPage** | Page display mode in the PDF reader: Display two pages side by side in the PDF reader. | [pdfParameter](../WDLang6/1410087857.md) type |
| **dispSinglePage** | Page display mode in the PDF reader: Displays one page of the PDF file at a time in the PDF reader. Margins are not displayed. | [pdfParameter](../WDLang6/1410087857.md) type |
| **pdfXCoord** | X-coordinate of the specified position on the page (expressed in millimetres). | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfYCoord** | Y-coordinate of the specified position on the page (expressed in millimetres). | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfColor** | Color management in the PDF file: Create a color PDF file. | [pdfParameter](../WDLang6/1410087857.md) type |
| **pdfGrayscale** | Color management in the PDF file: Create a black and white PDF file. | [pdfParameter](../WDLang6/1410087857.md) type |
| **pdfPageNumber** | Number of the page that contains the specified position. | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfOriginScreen** | Changes the origin to calculate a position in the PDF Reader control. | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **rotationAnticlockwise** | Rotates the page -90 degrees. | [PDFReaderRotatePage](../WDLang1/1000023940.md) |
| **rotationClockwise** | Rotates the page 90 degrees. | [PDFReaderRotatePage](../WDLang1/1000023940.md) |
| **zoomFitVisible** | Default zoom when the PDF file is opened: Zoom to content. | [pdfParameter](../WDLang6/1410087857.md) type |
| **zoomDefault** | Default zoom when the PDF file is opened: Default zoom. | [pdfParameter](../WDLang6/1410087857.md) type |
| **zoomFullPage** | Default zoom when the PDF file is opened: Zoom to display the whole page. | [pdfParameter](../WDLang6/1410087857.md) type |
| **zoomFullHeight** | Default zoom when the PDF file is opened: Zoom to display the entire vertical space of the page. | [pdfParameter](../WDLang6/1410087857.md) type |
| **zoomFullWidth** | Default zoom when the PDF file is opened: Zoom to display the entire horizontal space of the page. | [pdfParameter](../WDLang6/1410087857.md) type |
| **zoomActualSize** | Default zoom when the PDF file is opened: Zoom at 100%/ | [pdfParameter](../WDLang6/1410087857.md) type |

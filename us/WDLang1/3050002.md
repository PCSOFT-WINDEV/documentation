
## DecimalPart (Function)

***In french: PartieDécimale***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the decimal part of a number.






<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Decimal part of 16.487
ResDecimalPart = DecimalPart(16.487)
// Returns 0.487
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = DecimalPart(<Number>)`
---

**`<Result>: (Real)`**

Requested decimal part.

**`<Number>: (Real)`**

Number to use.

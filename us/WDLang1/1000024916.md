
## DocDeleteTableOfContents (Function)

***In french: DocSupprimeTableDesMatières***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes a table of contents from a Word Processing document. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
DocDeleteTableOfContents(WP_NoName1.Selection.Fragment)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`DocDeleteTableOfContents(<Fragment>)`
---

**`<Fragment>: (docFragment variable)`**

Name of the [docFragment](../WDLang1/1000022482.md) variable that corresponds to the table of contents to delete.

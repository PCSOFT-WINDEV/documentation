


## PicClose (Function)

***In french: PicFerme***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Closes the image found in an Image Editor control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
PicClose(IE_MyControl)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`PicClose(<Image Editor control>)`
---

**`<Image Editor control>: (Control name)`**

Name of the Image Editor control to use. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Remarks
The function has no effect if no image is opened in the Image Editor control.

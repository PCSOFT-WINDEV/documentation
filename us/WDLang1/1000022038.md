
## Background (Type of variable)

***In french: Fond***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Background** type defines the advanced characteristics of the element background. The characteristics of this background can be defined and changed using different WLanguage properties. 

**Remark**: For more details on the declaration of this type of variable and the use of WLanguage properties, see [Declaring a variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Draw a rectangle with blue borders, green background and rounded corners
// in an Image control

// Initializes the drawing in the Image control
dStartDrawing(IMG_Drawing)

Rect is Rectangle
Rect.X = 10
Rect.Y = 15
Rect.Width = 100
Rect.Height = 150
BorderStyle is Border
BorderStyle.Color = LightBlue
BorderStyle.Round = 30
BorderStyle.Thickness = 1
BackgroundStyle is Background
BackgroundStyle.Color = LightGreen
dBorder(Rect, BorderStyle, BackgroundStyle)
```

## Remarks
<a name="NOTE0_1"></a>


### Properties specific to Background variables
<a name="properties_specific_background_variables_ELTPARAGRAPHE000044"></a>

The following properties can be used to handle the background of an element:

| Property name | Type used | Effect |
| --- | --- | --- |
| Color | [Color](../WDLang1/1000019661.md) variable | Background color. |
| Gradient |   | Gradient of the background color: <br><br>- **ColorStart** property ([Color](../WDLang1/1000019661.md) type): Start color of the gradient. <br><br>- **ColorEnd** property ([Color](../WDLang1/1000019661.md) type): End color of the gradient.<br><br>- **Angle** property (integer): Angle that will be used for the selection of colors in the gradient. The angle value is included between 0 (horizontal angle, default value) and 360°.<br><br>- **Color3** property ([Color](../WDLang1/1000019661.md) type): Color that will be used in the gradient.<br><br>- **DistanceColor3** property (integer): Percentage separating the start color from color 3. <br><br>- **Color4** property ([Color](../WDLang1/1000019661.md) type): Color that will be used in the gradient.<br><br>- **DistanceColor4** property (integer): Percentage separating the start color from color 4.  <br><br><br><br><br>The color parameters are used as follows:<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=drectangledegrade.gif)<br><br><br>The angle is used to define the orientation of the gradient.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=angle_degrade.gif)<br><br><br>Gradients with 3 or 4 colors are only available for 0 or 90 degree angles.<br><br>**Remarks**: <br><br>- On some systems (e.g., 256-color Remote Desktop Connection), gradients are not available. In this case, the color defined via the **ColorStart** property is used.<br><br>- Gradient backgrounds are not supported in docx documents. <br><br><br> |
| Hatch | Integer constant | Type of hatching used: <br><br>- *HatchNone*: No hatching. <br><br>- *HatchCrossed*: Criss-crossed hatching.<br><br>- *HatchCrossedDiagonal*: Diagonal crossed hatching.<br><br>- *HatchDiagonal*: Diagonal hatching at 60%.<br><br>- *HatchDiagonalBottom*: Diagonal hatching at 120%.<br><br>- *HatchHorizontal*: Horizontal hatching.<br><br>- *HatchVertical*: Vertical hatching. <br><br><br> |
| Type | Integer constant | Type of background: <br><br>- *BackgroundSolid*: Solid background.<br><br>- *BackgroundTransparent*: Transparent background (the **Color** property is ignored).<br><br>- *BackgroundHatched*: Hatched background (the **Hatch** property is taken into account). <br><br>- *BackgroundGradient*: Background with color gradient (the **Gradient** property is taken into account).<br><br><br> |

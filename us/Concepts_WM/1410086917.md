<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
The data handled by a WINDEV Mobile application can come from:

- an HFSQL Classic or HFSQL Client/Server database (".Fic" files).

- ".INI" files.

- text files (ANSI or UNICODE).

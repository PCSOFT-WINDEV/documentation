
## gglCoordinate (Type of variable)

***In french: gglCoordonnée***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **gglCoordinate** type is used to describe and handle a geographical point on a map. The characteristics of this type of variable can be defined and changed using different WLanguage properties.

This type of variable is used by:

- the [gglMapParameter](../WDLang5/1000017505.md) type.

- the [gglPath](../WDLang5/1000017433.md) type.




**Remarks**: For more details on the declaration of this type of variable and the use of WLanguage properties, see [Declaring a variable](../Motscles/1514032.md).

## Remarks




### Management properties
<a name="management_properties_ELTPARAGRAPHE000050"></a>

The following properties can be used to handle **gglCoordinate** variable:

| Property name | Type used | Effect |
| --- | --- | --- |
| Latitude | Real | Latitude of the point. |
| Longitude | Real | Longitude of the point. |

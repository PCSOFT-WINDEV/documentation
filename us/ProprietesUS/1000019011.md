


## Guest (Property)

***In french: Invité***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Guest** property is used to get and modify a list of guests. This property can be used on:

- a variable of type [gglEvent](../WDLang5/1000017440.md)

- a variable of type [Appointment](../WDLang1/1000019244.md).


## FontUnderlined (Property)

***In french: PoliceSoulignée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **FontUnderlined** property is used to underline text. This property can be used on: 

- controls. For more details, see [FontUnderlined](../Proprietes/2510099.md). 

- a variable of type [docFormatting](../WDLang1/1000022817.md).

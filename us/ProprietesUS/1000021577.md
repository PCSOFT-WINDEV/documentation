
## DownloadProgress (Property)

***In french: AvancementTéléchargement***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **DownloadProgress** property is used to get the download progress notification. This property can be used on:

- a variable of type [httpRequest](../WDLang3/1000021158.md). 

- a variable of type [restRequest](../WDLang3/1000021481.md).

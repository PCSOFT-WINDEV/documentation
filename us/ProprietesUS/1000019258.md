


## WithRepetition (Property)

***In french: AvecRépétition***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **WithRepetition** property is used to determine if an appointment should be repeated. This property can be used on:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [EventTimeline](../WDLang1/1000020612.md).

- a variable of type [Appointment](../WDLang1/1000019244.md).

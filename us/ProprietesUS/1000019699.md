
## DefaultCell

***In french: CelluleDéfaut***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **DefaultCell** property is used to set the style of cells in XLS documents. 

This property can be used on:

- a variable of type [xlsColumn](../WDLang5/1000017471.md).

- a variable of type [xlsRow](../WDLang5/1000017325.md).

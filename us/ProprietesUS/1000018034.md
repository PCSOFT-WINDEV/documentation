
## Modified (Property)

***In french: Modifié***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Modified** property lets you know if an element has been modified. This property can be used on:

- the controls of a window. For more details, see [Modified on the controls of a window](../Proprietes/2510080.md).

- a variable of type [xlsDocument](../WDLang5/1000017464.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [Document](../WDLang1/1000022461.md).

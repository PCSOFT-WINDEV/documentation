
## Height (Property)

***In french: Hauteur***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Height** property is used to get and change the height of an element. This property can be used on:

- an element found in a window, page or report. For more details, see [Height](../Proprietes/2510050.md).

- a variable of type [rssStream](../WDLang5/1000017795.md).

- a variable of type [rssImage](../WDLang5/1000017799.md).

- a variable of type [xlsDocument](../WDLang5/1000017464.md).  

- a variable of type [xlsRow](../WDLang5/1000017325.md).

- a variable of type [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [upnpImage](../WDLang3/1000020905.md).


## ActionLabel (Property)

***In french: LibelléAction***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **ActionLabel** property is used to get and change the label of the notification's action button. This property can be used on:

- a variable of type [Notification](../WDLang3/1000019441.md). 

- a variable of type [notificationAction](../WDLang3/1000021304.md).




## StartDate (Property)

***In french: DateDébut***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **StartDate** property is used to get and change the start date of a time period. This property can be used on:

- Calendar, Organizer and Scheduler controls. For more details, see [StartDate property](../Proprietes/1000017426.md).

- a variable of type [saasSubscription](../WDLang3/1000019088.md).

- a variable of type [gglEvent](../WDLang5/1000017440.md). 

- a variable of type [Appointment](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [GanttTask](../WDLang1/1000020881.md).

- a variable of type [hServerOperation](../WDLang4/1000022471.md).


## Separator (Property)

***In french: Séparateur***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Separator** property is used to manage separators. This property can be used on: 

- [docNumberingLevel](../WDLang1/1000022799.md) variables. 

- [docFormatMulticolumn](../WDLang1/1000023580.md) variables.

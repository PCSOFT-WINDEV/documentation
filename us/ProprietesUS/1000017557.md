
## Procedure

***In french: Procédure***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Procedure** keyword can correspond to:

- the **Procedure** property used on: 

	- a variable of type [hScheduledTask](../WDLang4/1000017448.md).

	- a variable of type [rssStream](../WDLang5/1000017795.md).

	- a variable of type [rssCloud](../WDLang5/1000017785.md).

	- a variable of type [Procedure Description](../WDLang1/1000019317.md).

	- a variable of type [Definition](../WDLang1/1000019540.md).

	- a variable of type [httpProgress](../WDLang3/1000021222.md). 

	- a variable of type [Description of ParallelTask](../WDLang1/1000021586.md).




- a variable of type [Procedure](../Motscles/1000018907.md).

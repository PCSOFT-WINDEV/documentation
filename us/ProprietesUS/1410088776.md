
## Move

***In french: Déplace***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Move** property determines if an element can be moved. This property is available for: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [diagActionAllowed](../WDLang1/1410088840.md). 

- a variable of type [kbActionAllowed](../WDLang1/1410089541.md).

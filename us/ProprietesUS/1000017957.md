
## Caption (Property)

***In french: Libellé***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Caption** property is used to get and change the text of an element. This property can be used on:

- controls, windows, pages and reports. For more details, see [Caption](../Proprietes/2510053.md).

- a variable of type [gglPostalAddress](../WDLang5/1000017575.md).

- a variable of type [gglContact](../WDLang5/1000017434.md).

- a variable of type [gglEmail](../WDLang5/1000017571.md).

- a variable of type [gglEvent](../WDLang5/1000017440.md).

- a variable of type [gglLocation](../WDLang5/1000017441.md).

- a variable of type [gglInstantMessaging](../WDLang5/1000017569.md).

- a variable of type [gglPerson](../WDLang5/1000017446.md).

- a variable of type [gglOrganization](../WDLang5/1000017756.md).

- a variable of type [gglPhoneNumber](../WDLang5/1000017573.md).

- a variable of type [inAppProduct](../WDLang3/1000020868.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [Token](../WDLang1/1000023942.md).

- a variable of type [MarkerImage](../WDLang3/1000025796.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [diagText](../WDLang1/1410088393.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [sysNotificationButton](../WDLang1/1410087947.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [sysNotificationInput](../WDLang1/1410088024.md).

- a variable of type [AssistedInput](../WDLang1/1410088983.md).

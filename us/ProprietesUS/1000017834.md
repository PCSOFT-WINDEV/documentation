


## Region (Property)

***In french: Région***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Region** property is used to get and change a region used by Google AdWords.. This property can be used on:

- a variable of type [sfUserInfo](../WDLang5/1000018423.md).

- a variable of type [Address](../WDLang3/1000019944.md).


## Parameter (Property)

***In french: Paramètre***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Parameter** property is used to get and change certain parameters. This property can be used on:

- a variable of type [hScheduledTask](../WDLang4/1000017448.md).

- a variable of type [API description](../WDLang1/1000019149.md).

- a variable of type [Description of ParallelTask](../WDLang1/1000021586.md).

- a variable of type [Syntax Description](../WDLang1/1410089154.md).


## Attach (Property)

***In french: Attache***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Attach** property gets and changes the attachments of an email. 

This property can be used on:

- the [Email](../WDLang3/3032029.md) structure

- a variable of type [Email](../WDLang3/1000018713.md).

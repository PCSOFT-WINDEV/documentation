
## WholeDay

***In french: JournéeEntière***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **WholeDay** property is used to set an event for an entire day. This property can be used on:

- a variable of type [gglEvent](../WDLang5/1000017440.md).

- a variable of type [Appointment](../WDLang1/1000019244.md).


## AssociatedValue (Property)

***In french: ValeurAssociée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **AssociatedValue** property is used to get the associated value. This property can be used on:

- a variable of type [Option Description](../WDLang1/1000023072.md). 

- a variable of type [Value description](../WDLang1/1000023061.md).

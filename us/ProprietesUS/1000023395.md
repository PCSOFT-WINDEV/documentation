
## NamespaceDeclared (Property)

***In french: NamespaceDéclaré***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **NamespaceDeclared** property is used to manage defined namespaces. This property can be used on:

- a variable of type [xmlNode](../WDLang5/1000018786.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [xmlReader](../WDLang5/1000023392.md).

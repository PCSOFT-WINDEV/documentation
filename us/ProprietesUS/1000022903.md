
## Associated (Property)

***In french: Associé***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Associated** property determines if an element belongs to the base class or to an associated class. This property is used by:

- a variable of type [Procedure Description](../WDLang1/1000019317.md).

- a variable of type [Property Description](../WDLang1/1000019324.md).

- a variable of type [Variable Description](../WDLang1/1000019560.md).

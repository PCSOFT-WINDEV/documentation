
## IndentFirstLine (Property)

***In french: RetraitPremièreLigne***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **IndentFirstLine** property is used to get and change the indent of the first line of an element. This property can be used on:

- [docParagraph](../WDLang1/1000022483.md) variables.

- [docNumberingLevel](../WDLang1/1000022799.md) variables.

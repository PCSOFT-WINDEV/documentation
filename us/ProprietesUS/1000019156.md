
## FreeDLL

***In french: DéchargeDLL***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **FreeDLL** keyword can correspond to: 

- the **FreeDLL** function that is used to unload a 32-bit library (DLL) that was previously loaded in memory. For more details, see [FreeDLL](../WDLang1/3014009.md). 

- the **FreeDLL** property available for a variable of type [API description](../WDLang1/1000019149.md).

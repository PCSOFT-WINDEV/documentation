
## Geodesic (Property)

***In french: Géodésique***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Geodesic** property is used to indicate whether the drawing should follow the curvature of the Earth. This property can be used on: 

- a variable of type [MapPolygon](../WDLang3/1000025476.md).

- a variable of type [MapPolyline](../WDLang3/1000025489.md).

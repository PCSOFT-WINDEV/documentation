
## StartTimestamp (Property)

***In french: DateHeureDébut***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **StartTimestamp** property is used to set the start date and time of an action. It is available for the following types of variables:

- [hbReading](../WDLang4/1000021701.md),

- [hbScanParameter](../WDLang4/1000021715.md).

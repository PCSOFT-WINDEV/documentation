
## Key (Property)

***In french: Clé***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Key** property is used to manipulate keys:

- in a variable of type [xmlSignature](../WDLang3/1000020736.md). 

- in a variable of type [memcachedResult](../WDLang4/1000023359.md).

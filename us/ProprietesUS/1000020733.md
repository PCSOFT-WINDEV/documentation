
## Method (Property)

***In french: Méthode***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Method** property specifies the method used. This property can be used on:

- a variable of type [httpRequest](../WDLang3/1000021158.md). 

- a variable of type [restRequest](../WDLang3/1000021481.md).

- a variable of type [gglOverride](../WDLang5/1000026204.md).

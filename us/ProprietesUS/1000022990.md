
## PrivateKeyPassword (Property)

***In french: MotDePasseCléPrivée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **PrivateKeyPassword** property sets the key file password (for a private key connection). This property is used: 

- by [sshSession](../WDLang3/1000021157.md) variables. 

- by [mqttSession](../WDLang3/1000023023.md) variables.




## NationToSubLanguage (Function)

***In french: NationVersSousLangue***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the default sub-language corresponding to a nation.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
Res = NationToSubLanguage(nationAustralian)
// Res = 3
```

<a name="XSYNTAX"></a>

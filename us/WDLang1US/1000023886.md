


## &lt;Window&gt;.Maximize (Function)

***In french: &lt;Fenêtre&gt;.Maximise***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays a WINDEV window in its maximum size.

**Remark**: **&lt;Window&gt;.Maximize** performs the same action as the maximize button at the upper-right corner of the windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Display the "WIN_EditCustomer" window in its maximum size
WIN_CustomerInput.Maximize()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Window>.Maximize()`
---

**`<Window>: (Window name)`**

Name of the window that must be displayed in its maximum size.



<a name="XComponent"></a>

## Component:
wd280obj.dll

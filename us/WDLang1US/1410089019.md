
## IBANValid (Function)

***In french: IBANValide***



<a name="XUtilisation"></a>
<a name="Utilisation"></a>
<a name="description"></a>
Teste la validité d'un IBAN (International Bank Acount Number).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Example


```wl
// Test d'une variable
MonCompteIBAN est une chaîne = "FR7630006000011234567890189"
IBANValide(MonCompteIBAN) // Renvoie Vrai

// Test d'un champ
SAI_CompteIBAN = "ABC"
IBANValide(SAI_CompteIBAN) // Renvoie Faux
```

<a name="XSYNTAXE"></a>

## Syntax
<a name="SYNTAXE1"></a>

`<Résultat> = IBANValide(<IBAN à vérifier>)`
---

**`<Résultat>: (Booléen)`**



- <u><u><u><u>Vrai</u></u></u></u> si l'élément spécifié contient un IBAN valide.

- <u><u><u><u>Faux</u></u></u></u> dans le cas contraire.




**`<IBAN à vérifier>: (Nom de champ ou variable de type chaîne)`**

IBAN à vérifier. Ce paramètre peut correspondre : 

- au nom d'une variable de type chaîne.

- au nom d'un champ.

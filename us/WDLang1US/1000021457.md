


## SpreadsheetRecalculateAll (Function)

***In french: TableurRecalculeTout***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recalculates all formulas of a Spreadsheet control found in a window (including the formulas that use WLanguage functions/procedures). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
HReadNext(TurnoverFile)
SpreadsheetRecalculateAll(PSHEET_Spreadsheet)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`SpreadsheetRecalculateAll(<Spreadsheet control>)`
---

**`<Spreadsheet control>: (Control name)`**

Name of the Spreadsheet control to be used. 



<a name="XComponent"></a>

## Component:
wd280obj.dll

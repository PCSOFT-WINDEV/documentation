
## WinBackgroundImage (Function)

***In french: FenImageFond***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifies the background image of a window and specifies the display mode of this image.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![User code (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Modify the background image of "WIN_EditImage"
// The new image is "C:\Documents\Bkgd.GIF"
// This image will be centered in the window
Res = WinbackgroundImage(WIN_EditImage, "C:\Documents\Bkgd.GIF", imgCentered)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = WinBackgroundImage(<Window used> , <Image name> , <Display>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the background image was modified, 

- <u><u><u><u>False</u></u></u></u> otherwise.




**`<Window used>: (Window name)`**

Name of the window to be used.
If this parameter corresponds to an empty string (""), the current window is used.

**`<Image name>: (Character string)`**

Name and full (or relative) path of the image to display in the window. A UNC path can be used. No image is displayed in the window if this parameter does not correspond to an existing image.

**`<Display>: (Constant)`**

Indicates the display mode of the image:


|   |   |
| --- | --- |
| *imgCentered* | Centered: Image centered in the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_Centre.GIF)<br> |
| *imgH* | Homothetic: Image enlarged or reduced (by respecting the proportions of the initial image) to occupy the entire surface of the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_HOMO.GIF)<br> |
| *imgHC* | Homothetic centered: Image enlarged or reduced (by respecting the proportions of the initial image) and centered in the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_HomoCentre.GIF)<br> |
| *imgHCE* | Homothetic centered extended: Image enlarged or reduced (by respecting the proportions of the initial image) so that the smaller side of the image is entirely displayed in the window and centered in the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_HomoCentreEtendu.GIF)<br> |
| *imgHCWIS* | Homothetic centered without enlargement: If the image can be entirely displayed in the window, it is displayed at 100% and centered in the window. Otherwise, the image is proportionally reduced so that it can be entirely displayed and centered in the window. |
| *imgHE* | Homothetic extended: Image enlarged or reduced (by respecting the proportions of the initial image) so that the smaller side of the image is entirely displayed in the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_HOMOETENDU.GIF)<br> |
| *imgHWIS* | Homothetic without enlargement: If the image can be entirely displayed in the window, it is displayed at 100%. Otherwise, the image is proportionally reduced in order to be entirely displayed. |
| *imgNormal* | 100%: Image displayed at 100% in the top left corner of the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_100.GIF)<br> |
| *imgStretched* | Stretched: Image stretched across the entire surface of the window.<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_Etiree.GIF)<br> |
| *imgTiled* | Tiled: Image displayed at 100% and tiled to occupy the entire surface of control (tile).<br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=IMG_Mosaique.GIF)<br> |

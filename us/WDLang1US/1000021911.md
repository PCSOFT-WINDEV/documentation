
## TelemetryIsEnabled (Function)

***In french: TélémétrieEstActive***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Allows you to find out whether the telemetry is enabled. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
If TelemetryIsEnabled() = False THEN TelemetryEnable()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = TelemetryIsEnabled()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the telemetry is enabled, 

- <u><u><u><u>False</u></u></u></u> otherwise.

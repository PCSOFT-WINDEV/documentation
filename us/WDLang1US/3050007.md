
## Power (Function)

***In french: Puissance***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Raise a number to a power.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// 8 to the power of 5
ResPower = Power(8, 5)
// Returns 32768
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = Power(<Number to raise> , <Exponent>)`
---

**`<Result>: (Real)`**

Result of &lt;Number to raise&gt; to the power of &lt;Exponent&gt;.

**`<Number to raise>: (Integer or real)`**

Number that will be raised to the power.

**`<Exponent>: (Integer or real)`**

Exponent.

### Equivalence
<a name="equivalence_ELTPARAGRAPHE000076"></a>

**Power** is equivalent to the **^** arithmetic operator. The syntax of this operator is as follows: 


```txt
<Number to Raise> ^ <Exponent>
```

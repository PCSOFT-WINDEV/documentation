
## SemaphoreDestroy (Function)

***In french: SémaphoreDétruit***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Explicitly destroys a semaphore.

**Remark**: All the semaphores are freed when the application ends.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`SemaphoreDestroy(<Semaphore name>)`
---

**`<Semaphore name>: (Character string)`**

Name of the semaphore to destroy.  



<a name="XComponent"></a>

## Component:
wd280vm.dll

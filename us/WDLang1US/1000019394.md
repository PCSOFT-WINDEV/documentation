
## CurrentYear (Function)

***In french: AnnéeEnCours***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the current year in integer format. 






<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MyYear is int
MyYear = CurrentYear()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = CurrentYear()`
---

**`<Result>: (Integer)`**

Current year.

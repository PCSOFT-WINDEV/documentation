


## DocExitEdit (Function)

***In french: DocSortEdition***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exits from the current edit. The entry in edit was performed by [DocGoIntoEdit](../WDLang1/1000023077.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
DocExitEdit(WP_MyDocControl)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`DocExitEdit(<Word Processing control>)`
---

**`<Word Processing control>: (Control name)`**

Name of the Word Processing control to use.

## Component:
wd280mdl.dll

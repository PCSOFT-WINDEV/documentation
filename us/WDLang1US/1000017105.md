
## ServiceContinue (Function)

***In french: ServiceRelance***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restarts a paused service.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Restarts the service found in TABLE_Service
IF ServiceContinue(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = ServiceContinue(<Service name> [, <Remote computer>])`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the restart of the service was requested, 

- <u><u><u><u>False</u></u></u></u> otherwise. To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Character string)`**

Name of the service to restart.

**`<Remote computer>: (Optional character string)`**

IP address of the remote computer in xxx.xxx.xxx.xxx format (returned by [NetIPAddress](../WDLang3/3056023.md)) or name of the remote computer (returned by [NetMachineName](../WDLang3/3056014.md)). This parameter is used to restart a service on another computer. To use this option, the user must be administrator of remote computer. 
If this parameter is not specified or if it corresponds to an empty string (""), the service is handled locally.

### Operating mode
<a name="operating_mode_ELTPARAGRAPHE000055"></a>

**ServiceContinue** sends a request to restart a service to the service manager. This operation may not be performed immediately. To find out the status of the service, use [ServiceWaitStatus](../WDLang1/1000017101.md).

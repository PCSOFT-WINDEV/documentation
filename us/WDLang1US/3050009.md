
## Sin (Function)

***In french: Sinus***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the sine of an angle.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Sine of 30
ResSine = Sin(30)
// Returns 0.5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = Sin(<Angle>)`
---

**`<Result>: (Real)`**

Sine of the specified angle.

**`<Angle>: (Integer or real)`**

Angle (in degrees) to use.

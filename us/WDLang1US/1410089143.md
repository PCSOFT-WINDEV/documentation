


## KanbanDeleteAllCard (Function)

***In french: KanbanSupprimeToutCarte***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deletes all cards from a Kanban control or Kanban List control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Deletes all cards from the Kanban control
KanbanDeleteAllCard(KANBAN_MyKanban)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`KanbanDeleteAllCard(<Control used>)`
---

**`<Control used>: (Control name)`**

Name of the control used: 

- Name of the Kanban control.

- Name of the Kanban List control.




## &lt;TreeView&gt;.Select (Function)

***In french: &lt;Arbre&gt;.Select***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the full path of the selected element in a TreeView control.

Example of TreeView control:
![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=Arbre_Vocabulaire.gif)
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Remark**: **&lt;TreeView&gt;.Select** can be used on:

- a TreeView control populated programmatically, based on a data file, or based on a variable. 

- a single-selection or multi-selection TreeView control. For multi-selection TreeView controls, **&lt;TreeView&gt;.Select** must be used in a loop in order to get all selected rows. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Full path of the selected element in the "TREE_TVRecipe" TreeView control
ResPath = TREE_TVRecipe.Select()
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Selected rows in a multi-selection TreeView Table control populated programmatically
FOR i = 1 _TO_ TREE_TREE1.SelectOccurrence()
	Trace("Selected element: " + TREE_TREE1.Select(False, i))
END

// Equivalent to:
// i is int
// FOR EACH SELECTED ROW i OF TREE_TREE1
// 	Trace("Selected element: " + TREE_TREE1.Select(False, i-1))
// END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = <TreeView control>.Select([<Management of duplicates> [, <Rank>]])`
---

**`<Result>: (Character string)`**

Full path of the selected element, in the following format:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


**`<TreeView control>: (Control name)`**

Name of the TreeView control to be used.

**`<Management of duplicates>: (Optional boolean)`**

Mode for managing duplicates in the elements:

- <u><u><u><u>False</u></u></u></u> (default value): no management is performed.

- <u><u><u><u>True</u></u></u></u>: the names of the path elements are built with an identifier. For more details, see [Managing duplicates in the TreeView controls](../WDChamp/1013325.md).




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) This parameter is ignored.

**`<Rank>: (Optional integer)`**

Rank of the selection for a multi-selection TreeView control. If this parameter is not specified, the index of the first selected row is returned.
For example, to retrieve the first selected element, &lt;Rank&gt; = 1. 
To retrieve the second selected element, &lt;Rank&gt; = 2, etc.
  
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) This parameter is not supported.

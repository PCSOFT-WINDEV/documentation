


## &lt;String type&gt;.Reverse (Function)

***In french: &lt;Type Chaîne&gt;.Inverse***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reverses the characters of a string. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MyString is string = "ABC"
Trace(MyString.Reverse())
// The trace window displays "CBA"
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = <String>.Reverse()`
---

**`<Result>: (Ansi, Unicode or Binary string)`**

Reversed character string.

**`<String>: (Character string)`**

Character string to reverse.




## MultimediaEnd (Function)

***In french: MultimédiaFin***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Positions at the end of media file played in the Multimedia control.








<a name="Example1"></a>
<a name="sample_code"></a>

## Example

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![User code (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"

MultimediaOpen(MM_Multimedia1, MyFile)
MultimediaEnd(MM_Multimedia1)
...
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = MultimediaEnd(<Multimedia control>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the position was set,

- <u><u><u><u>False</u></u></u></u> otherwise. To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Control name)`**

Name of the Multimedia control to be used.


## Root (Function)

***In french: Racine***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Returns the root of a number.


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Square root of 25
ResRoot = Root(25)
// Returns 5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`<Result> = Root(<Number> , <Root Factor>)`
---

**`<Result>: (Real)`**

Root of the number.

**`<Number>: (Integer or real)`**

Number to use.

**`<Root Factor>: (Integer or real)`**

Factor of the root (2 for a square root, 3 for a cubic root, ...). 
If this parameter is not specified, the square root of &lt;Number&gt; is returned.


## RestartProgram (Function)

***In french: RelanceProgramme***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ends the application and restarts it. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Restart the application
RestartProgram()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`RestartProgram([<Code Execution>])`
---

**`<Code Execution>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> to run the code for ending the project. 

- <u><u><u><u>False</u></u></u></u> (default value) to avoid running the closing code of the project.

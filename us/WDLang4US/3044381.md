
## H.TemporaryDirectory (Variable)

***In french: H.RépertoireTemporaire***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Allows you to force the directory used to store the temporary files when reindexing the HFSQL data files.  

The temporary directory of Windows is used by default, then the current directory if the available space is not sufficient.

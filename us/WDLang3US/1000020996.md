


## &lt;upnpDevice variable&gt;.GetMute (Function)

***In french: &lt;Variable upnpPériphérique&gt;.RécupèreMuet***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Retrieves the 'Mute' status of a UPnP device.

**Caution**: To use this function, the device must include a "RenderingControl" service.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
arrDevice is array of upnpDevices 
arrDevice = UPNPListDevice(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
FOR EACH Device of arrDevice
	IF Device.GetMute() THEN
		Trace(Device.Name + " = Mute")
	ELSE
		Trace(Device.Name + " = Not mute")
	END
END
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = <UPnP device>.GetMute()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the device is mute, 

- <u><u><u><u>False</u></u></u></u> otherwise.




**`<UPnP device>: (upnpDevice variable)`**

Name of [upnpDevice](../WDLang3/1000020903.md) variable corresponding to the UPnP device to use.

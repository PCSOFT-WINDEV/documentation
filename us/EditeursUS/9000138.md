
## Copying controls from the window editor
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Overview
<a name="overview_ELTTEXTE000606"></a>
The controls can be copied from the window editor: 

- to the page editor

- to the report editor.

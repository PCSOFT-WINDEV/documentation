
## ReturnReceipt (Property)

***In french: AccuséRéception***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **ReturnReceipt** property gets and changes the email delivery receipt request. 

This property can be used with:

- the [Email](../WDLang3/3032029.md) structure.

- a variable of type [Email](../WDLang3/1000018713.md).

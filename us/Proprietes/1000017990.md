
## Event

***In french: Evénement***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Event** keyword can correspond to:  

- the **[Event](../WDLang1/3015011.md)** function, which is used to intercept Windows events. 

- the **Event** property of a variable of type [gglCalendar](../WDLang5/1000017418.md).


## PageNumber (Property)

***In french: NuméroPage***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **PageNumber** property gets and sets a page number. This property can be used on:

- an Image control or a report. For more details, see [ PageNumber](../Proprietes/2510106.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [ pdfBookmark](../WDLang6/1410089195.md).

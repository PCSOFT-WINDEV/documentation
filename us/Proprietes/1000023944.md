
## CaptionHTML (Property)

***In french: LibelléHTML***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **CaptionHTML** property is used to handle text as HTML code. This property can be used on:

- the controls in a page. For more details, see the help on the [CaptionHTML](../Proprietes/1000020922.md) property.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [diagText](../WDLang1/1410088393.md).

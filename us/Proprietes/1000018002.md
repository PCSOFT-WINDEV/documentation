
## Opacity (Property)

***In french: Opacité***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Opacity** property is used to get and change the opacity of an element. This property can be used on:

- the controls of a window or page. For more details, see [Opacity property on controls](../Proprietes/2514006.md).

- a variable of type [gglPath](../WDLang5/1000017433.md). 

- a variable of type [Color](../WDLang1/1000019661.md).

- a variable of type [Watermark](../WDLang5/1000019922.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [picText](../WDLang1/1000025178.md).

- a variable of type [MapImage](../WDLang3/1000025523.md).

- a variable of type [Marker](../WDLang3/1000019940.md).

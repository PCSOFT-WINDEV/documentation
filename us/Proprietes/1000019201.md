***In french: Altitude***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **ZOrder** property gets and sets the Z-order of an element. This property can be used on:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) the controls of a window or report. For more details, see the help on the [ZOrder](../Proprietes/2510002.md) property.

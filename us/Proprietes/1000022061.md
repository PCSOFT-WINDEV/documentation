
## Property (Property)

***In french: Propriété***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Property** property is used to get and change the properties of an element. This property can be used on:

- a variable of type [Definition](../WDLang1/1000019540.md).

- a variable of type [btleCharacteristic](../WDLang3/1000022060.md).

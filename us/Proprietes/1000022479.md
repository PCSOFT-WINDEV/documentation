
## Replication (Property)

***In french: Réplication***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Replication** property is used to identify and change the replication mode. This property can be used on:

- a data file. For more details, see [Replication](../Proprietes/2512073.md).

- a variable of type [hServerOperation](../WDLang4/1000022471.md).

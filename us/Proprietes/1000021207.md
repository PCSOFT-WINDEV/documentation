
## ReturnedValue (Property)

***In french: ValeurRenvoyée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **ReturnedValue** property is used to handle the return value. This property can be used on:

- windows, pages and controls. For more details, see [ReturnedValue](../Proprietes/2510134.md).

- a variable of type [ParallelTask](../WDLang1/1000021220.md).

- a variable of type [Thread](../WDLang1/1000024879.md).

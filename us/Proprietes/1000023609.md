
## ProcedureTrace (Property)

***In french: ProcédureTrace***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **ProcedureTrace** property allows you to specify the name of the WLanguage procedure used to get the headers and data sent. This property can be used on:

- a variable of type [httpRequest](../WDLang3/1000021158.md). 

- a variable of type [restRequest](../WDLang3/1000021481.md).


## Fragment (Property)

***In french: Fragment***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Fragment** property is used to handle fragments. This property can be used on:

- a variable of type [URI](../WDLang3/1000024027.md). 

- a variable of type [edtHTMLSelection](../WDLang1/1000026093.md).

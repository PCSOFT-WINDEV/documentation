
## LatestDateCovered (Property)

***In french: DernièreDateCouverte***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **LatestDateCovered** property is used to get the date of the last selected record. This property can be used on:

- a variable of type [sfSeekUpdatedResult](../WDLang5/1000018561.md).

- a variable of type [sfSeekDeletedResult](../WDLang5/1000018562.md).


## RoughHeader (Property)

***In french: EntêteBrut***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **RoughHeader** is used to get the header returned with the response. This property can be used on:

- a variable of type [httpResponse](../WDLang3/1000021165.md). 

- a variable of type [restResponse](../WDLang3/1000021477.md).

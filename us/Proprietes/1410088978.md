
## StoredValue (Property)

***In french: ValeurMémorisée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **StoredValue** property gets and sets the stored value of an element. This property can be used on:

- rows of List Box and Table controls, as well as Edit controls with tokens. For more details, see [StoredValue](../Proprietes/2510127.md).

- a variable of type [Token](../WDLang1/1000023942.md). 

- a variable of type [AssistedInput](../WDLang1/1410088983.md).

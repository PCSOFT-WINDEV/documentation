
## SenderAddress (Property)

***In french: AdresseExpéditeur***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **SenderAddress** property allows you to get and change the address the message reply will be sent to (Reply-to).

This property can be used with:

- the [Email](../WDLang3/3032029.md) structure.

- a variable of type [Email](../WDLang3/1000018713.md).


## Round (keyword)

***In french: Arrondi***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Round** keyword can correspond to:

- the [Round](../WDLang1/3050063.md) WLanguage function.

- the **Round** property, which can be used on: 

	- a variable of type [Border](../Motscles/15140100.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [diagSélection](../WDLang1/1410088172.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [diagRectangle](../WDLang1/1410088083.md).


## Movable (Property)

***In french: Déplaçable***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Movable** property is used to define if an element can be moved. This property can be used on:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a column of a Table control. For more details, see the help on the [Movable](../Proprietes/2510032.md) property.

- a variable of type [Marker](../WDLang3/1000019940.md).

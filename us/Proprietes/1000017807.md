


## PhoneNumber (Property)

***In french: Téléphone***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **PhoneNumber** property is used to get and change a phone number. This property can be used on:

- a variable of type [gglContact](../WDLang5/1000017434.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [gpwUser](../WDLang6/1000021088.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [WBAdminAccount](../WDLang2/1410089382.md).

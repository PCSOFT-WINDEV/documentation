
## CompressedSize (Property)

***In french: TailleCompressée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **CompressedSize** property is used to determine the compressed size of an element. This property can be used on:

- a variable of type [zipArchive](../WDLang3/1000018679.md).

- a variable of type [zipArchivedFile](../WDLang3/1000019044.md).

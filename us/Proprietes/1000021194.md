
## Timeout (Property)

***In french: DuréeNonRéponse***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Timeout** property is used to get and change the maximum wait time before a request times out. This property can be used on:

- the stream when using a VideoConference control. For more details, see [Timeout](../Proprietes/2510052.md).

- a variable of type [httpRequest](../WDLang3/1000021158.md).

- a variable of type [restRequest](../WDLang3/1000021481.md).

- a variable of type [wsRequest](../WDLang3/1000020577.md).

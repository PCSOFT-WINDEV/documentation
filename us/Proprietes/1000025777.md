
## Accuracy (Property)

***In french: Précision***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Accuracy** property is used to handle the precision of an element. This property can be used on: 

- a variable of type [geoPosition](../WDLang3/1000019191.md). 

- a variable of type [ocrOption](../WDLang6/1000025768.md).

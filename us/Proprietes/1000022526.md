
## FontStrikeOut (Property)

***In french: PoliceBarrée***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **FontStrikeOut** property is used to determine if a text is struck through. This property can be used on: 

- the controls. For more details, see [FontStrikeOut](../Proprietes/2510091.md). 

- a variable of type [docFormatting](../WDLang1/1000022817.md).

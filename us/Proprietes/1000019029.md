
## NumberExecution (Property)

***In french: NombreExécution***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **NumberExecution** property is used to get and change the number of executions of a task. This property can be used on: 

- a variable of type [hScheduledOptimization](../WDLang4/1000018854.md). 

- a variable of type [hScheduledTask](../WDLang4/1000017448.md).

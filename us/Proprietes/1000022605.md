
## Database (Property)

***In french: BaseDeDonnées***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Database** property is used to get and modify the database associated with a connection. This property can be used on:

- a connection. For more details, see the help on the [Database](../Proprietes/2512101.md) property.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [mongoConnection](../WDLang4/1000022409.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [mongoGridFS](../WDLang4/1000022416.md).

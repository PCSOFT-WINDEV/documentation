
## Attachment (Property)

***In french: PièceJointe***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Attachment** property is used to manage attachments. This property can be used on: 

- a variable of type [pdfDocument](../WDLang6/1000024909.md). 

- a variable of type [gglEvent](../WDLang5/1000017440.md).

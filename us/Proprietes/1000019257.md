


## Repetition (Property)

***In french: Répétition***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Repetition** property is used to configure appointment repetitions. This property can be used on: 

- a variable of type [EventTimeline](../WDLang1/1000020612.md).

- a variable of type [Appointment](../WDLang1/1000019244.md).

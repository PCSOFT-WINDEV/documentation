
## Number (Property)

***In french: Numéro***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
The **Number** property is used to get the number of an element. This property can be used on:

- a variable of type [API description](../WDLang1/1000019149.md).

- a variable of type [gglPhoneNumber](../WDLang5/1000017573.md).  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a variable of type [docSection](../WDLang1/1000022685.md).

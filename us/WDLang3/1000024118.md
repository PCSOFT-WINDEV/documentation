


## &lt;Chart&gt;.LegendFont (Function)

***In french: &lt;Graphe&gt;.PoliceLégende***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifies the font used for the chart legend.

Example of legend in a chart: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=Legende.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Caution**: A specific configuration is required to use this function in Linux. For more details, see [The charts](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Modify the font used for the legend of CHART_MyChart.
// The identifier of font used is 7.
CHART_MyChart.LegendFont(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

### Defining the font used for the chart legend

`<Chart control>.LegendFont(<Font used>)`
---

**`<Chart control>: (Control name)`**

Name of the Chart control to use in the window, page or report editor.


**`<Font used>: (Font)`**

Font to use. This parameter corresponds to a variable of type [Font](../Motscles/1514045.md). The font characteristics have been defined by the [FontCreate](../WDLang1/3060002.md) function or via the [font properties](../Proprietes/2515045.md).


<a name="SYNTAX2"></a>

### Defining the font used for the chart legend (syntax kept for backward compatibility)

`<Chart control>.LegendFont(<Font identifier>)`
---

**`<Chart control>: (Control name)`**

Name of the Chart control to use in the window, page or report editor.


**`<Font identifier>: (Integer)`**

Font identifier, defined by [grCreateFont](../WDLang3/3042001.md). A WLanguage error occurs if this font does not exist.

## Remarks


### Handling legends
<a name="handling_legends_ELTPARAGRAPHE000288"></a>The set of series labels correspond to the chart legend.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=EtiquetteSerie.gif)
The series labels are defined by [&lt;Chart&gt;.SeriesLabel](../WDLang3/1000024073.md). 

The legend font can be modified by **&lt;Chart&gt;.LegendFont**.  

[&lt;Chart&gt;.Legend](../WDLang3/1000024089.md) indicates the presence and position of the legend (i. e. all series labels).
<a name="NOTE0_2"></a>


### Initializing fonts
<a name="initializing_fonts_ELTPARAGRAPHE000315"></a>

The default font used for the chart legends has the following characteristics:

- Font name: Arial, 

- Size: 14,

- Style: Normal, 

- Color: Black.






The data defined by **&lt;Chart&gt;.LegendFont** will be taken into account during the next call to [&lt;Chart&gt;.Draw](../WDLang3/1000024062.md).

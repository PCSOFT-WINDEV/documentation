
## TwoFactorAuthenticationGenerateTOTPKey (Function)

***In french: DoubleAuthGénèreCléTOTP***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Generates a TOTP Key for an application user.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
sTOTPKey is ANSI string 
sTOTPKey = TwoFactorAuthenticationGenerateTOTPKey(User.UserID + User.Email)

// Saves the TOTP key in the User data file
User.sTOTPKey = sTOTPKey
HSave(User)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = TwoFactorAuthenticationGenerateTOTPKey(<Identifier>)`
---

**`<Result>: (Character string)`**

16-character ANSI string corresponding to the generated TOTP key. 

**`<Identifier>: (Character string)`**

User identifier. **This identifier must make it possible to uniquely identify a user** (e.g., concatenation of email, username and unique file key).

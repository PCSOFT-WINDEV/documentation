


## ContactLast (Function)

***In french: ContactDernier***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reads the last contact in a Lotus Notes or Outlook address book.
**Caution**: This function **cannot be used** to read the last contact in an Outlook Express address book.

When reading a contact, the data of this contact is updated in the [Contact structure](../WDLang3/1000006111.md). To retrieve the data of the current contact, all you have to do is retrieve the values from the [Contact structure](../WDLang3/1000006111.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"email/julia.nsf", "C:\Lotus Notes\julia.id")
// Read the last contact
ContactLast(SessionID)
// Add the contacts found into a List Box control
WHILE mContact.Out = False
	ListAdd(LIST_ContactList, mContact.Name + TAB + mContact.FirstName)
	ContactPrevious(SessionID)
END
```

## Syntax

### Reading the last contact in the Lotus Notes or Outlook address book

`<Result> = ContactLast(<Session identifier>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the last contact was read, 

- <u><u><u><u>False</u></u></u></u> otherwise. To get more details on the error, use [ErrorInfo](../WDLang1/3013008.md) with the *errMessage* constant.




**`<Session identifier>: (Integer)`**

Session ID to be used. This identifier is returned by [NotesOpenDatabase](../WDLang3/1000005101.md), [OutlookStartSession](../WDLang3/3032123.md), [EmailStartNotesSession](../WDLang3/3032121.md) or [EmailStartOutlookSession](../WDLang3/3032128.md).

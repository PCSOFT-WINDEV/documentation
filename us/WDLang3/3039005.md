
## Constants for managing streams (sound and video)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constant | Description | Used by the functions or properties |
| --- | --- | --- |
| **StreamAll** | Video + Sound stream (StreamFullDuplexSound + StreamFullDuplexVideo). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamCallerName** | Name of the stream issuer. | [StreamInfo](../WDLang3/3039009.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamCallerNumber** | Number of the stream issuer (IP address for example). | [StreamInfo](../WDLang3/3039009.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamFullDuplexSound** | Incoming and outgoing sound stream (corresponds to StreamOutgoingSound + StreamIncomingSound). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamFullDuplexVideo** | Incoming and outgoing video stream (corresponds to StreamOutgoingVideo + FluxVidéoRéception). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamIncomingDisplay** | Video displayed in the Conference control: The video displayed corresponds to the video received on the current computer. | [Display](../Proprietes/2510010.md) property |
| **StreamIncomingSound** | Incoming sound stream. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamIncomingVideo** | Incoming video stream. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamNoDisplay** | Video displayed in the Conference control: No video is displayed. | [Display](../Proprietes/2510010.md) property |
| **StreamOutgoingDisplay** | Video displayed in the Conference control: The video displayed corresponds to the video sent by the current computer. | [Display](../Proprietes/2510010.md) property |
| **StreamOutgoingSound** | Outgoing sound stream. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamOutgoingVideo** | Outgoing video stream. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamSound** | Type of data sent by the stream: The sound data is emitted. | [OutgoingData](../Proprietes/2510037.md) property |
| **StreamVideo** | Type of data sent by the stream: The Video data is emitted. | [OutgoingData](../Proprietes/2510037.md) property |

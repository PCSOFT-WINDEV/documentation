


## &lt;mqttSession variable&gt;.Disconnect (Function)

***In french: &lt;Variable mqttSession&gt;.Déconnecte***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Allows you to disconnect from a MQTT server (also called "Broker").


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MyMQTTSession is mqttSession
...

MyMQTTSession.Disconnect()
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<MQTT session>.Disconnects()`
---

**`<MQTT session>: (mqttSession variable)`**

Name of [mqttSession](../WDLang3/1000023023.md) variable corresponding to the parameters of the connection to end.

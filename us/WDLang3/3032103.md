
## LDAPReset (Function)

***In french: LDAPRAZ***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinitializes the [LDAPSession structure](../WDLang3/3032101.md) with the default values. This structure is used by [LDAPConnect](../WDLang3/3032100.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
LDAPReset()
```

<a name="SYNTAX1"></a>

## Syntax

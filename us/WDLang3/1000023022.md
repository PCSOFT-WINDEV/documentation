


## MQTTDisconnect (Function)

***In french: MQTTDéconnecte***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Allows you to disconnect from a MQTT server (also called "Broker").


<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
MyMQTTSession is mqttSession
...

MQTTDisconnect(MyMQTTSession)
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`MQTTDisconnect(<MQTT session>)`
---

**`<MQTT session>: (mqttSession variable)`**

Name of [mqttSession](../WDLang3/1000023023.md) variable corresponding to the parameters of the connection to end.

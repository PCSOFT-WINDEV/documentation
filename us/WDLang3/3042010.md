


## grTooltip (Function)

***In french: grBulle***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Displays and formats: 

- the tooltip associated with each chart section (section for a Pie chart, bar for a Column chart, etc.). Also allows you to find out the characteristics of the tooltip associated with each chart section.

- a custom tooltip, associated with a point.




By default, a tooltip is automatically displayed (*grShowTooltip* set to <u><u><u>True</u></u></u>) when a chart is hovered by the mouse cursor.

## Example

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) 
```wl
// Chart created through programming
i is int
// Create the chart
grCreate("MyChart", grPie)
// Destination of chart
grDestinationControl("MyChart", IMG_Image1)
// Display the percentages
grLabel("MyChart", grShowPercent, True)
// Add data
nValue is int
FOR I = 1 TO 200
	nValue = Random(30)
	grAddData("MyChart", 1, nValue)
	grCategoryLabel("MyChart", i, "Test" + i)
	grSeriesLabel("MyChart", 1, "Test series")
END
grTooltip("MyChart", grTooltipFormatPie, -%"[%SERIES%]" + CR + ...
	-%"[%CATEGORY%]" + CR + -%"[%%%]")
grTooltip("MyChart", grShowTooltip, True)
// Draw the chart
grDraw("MyChart")
```
<a name="Example2"></a>

```wl
// Chart control
// Display the percentages on a column chart
grLabel(CHART_MyChart, grShowPercent, True)
grTooltip(CHART_MyChart, grTooltipFormat, -%"[%SERIES%]" + CR + -%"[%CATEGORY%]" + ...
	CR + -%"[%VALUE%]")
grTooltip(CHART_MyChart, grShowTooltip, True)
// Draw the chart
grDraw(CHART_MyChart)
```
<a name="Example3"></a>

```wl
// Chart control
// Display a specific tooltip for a point
grTooltip(CHART_MyChart, 1, 2, "The tooltip content")
// Draw the chart
grDraw(CHART_MyChart)
```

## Syntax

### Displaying and formatting the tooltip associated with a chart

`grTooltip(<Chart name> , <Tooltip option to configure> , <Tooltip option value>)`
---

**`<Chart name>: (Control name or character string)`**

Name of the chart to be used. This name corresponds to:

- the chart name defined through programming with [grCreate](../WDLang3/3042005.md).

- the name of the Chart control in the window editor.




**`<Tooltip option to configure>: (Integer constant)`**

Enables or disables the tooltip and configures its characteristics.


|   |   |
| --- | --- |
| *grShowTooltip* | Enables or disables the tooltip. |
| *grTooltipFormat* | Describes the tooltip format. |
| *grTooltipFormatBubbleChart* | Describes the tooltip format for a Bubble chart.<br> |
| *grTooltipFormatCrosshairHeader* | Describes the header format for the chart crosshairs. |
| *grTooltipFormatHeatMap* | Describes the tooltip format for a Heatmap chart.<br> |
| *grTooltipFormatPie* | Describes the tooltip format for a Pie chart. |
| *grTooltipFormatRangeArea* | Describes the tooltip format for a Range Area chart. |
| *grTooltipFormatScatter* | Describes the tooltip format for a Scatter chart.<br> |
| *grTooltipFormatStock* | Describes the tooltip format for a Stock chart.<br> |



**`<Tooltip option value>: (Character string or boolean)`**

Option value for the selected tooltip. This value depends on the selected option (see the above table).  


<a name="SYNTAX2"></a>

### Finding out the characteristics of tooltip associated with a chart

`<Result> = grTooltip(<Chart name> , <Tooltip option>)`
---

**`<Result>: (Type corresponding to the option (boolean or character string))`**

Value of the sought option.

**`<Chart name>: (Control name or character string)`**

Name of the chart to be used. This name corresponds to:

- the chart name defined through programming with [grCreate](../WDLang3/3042005.md).

- the name of the Chart control in the window editor.




**`<Tooltip option>: (Integer constant)`**

Tooltip option whose value is requested.


|   |   |
| --- | --- |
| *grShowTooltip* | Enables or disables the tooltip. |
| *grTooltipFormat* | Describes the tooltip format. |
| *grTooltipFormatBubbleChart* | Describes the tooltip format for a Bubble chart.<br> |
| *grTooltipFormatCrosshairHeader* | Describes the header format for the chart crosshairs. |
| *grTooltipFormatHeatMap* | Describes the tooltip format for a Heatmap chart.<br> |
| *grTooltipFormatPie* | Describes the tooltip format for a Pie chart. |
| *grTooltipFormatRangeArea* | Describes the tooltip format for a Range Area chart.<br> |
| *grTooltipFormatScatter* | Describes the tooltip format for a Scatter chart.<br> |
| *grTooltipFormatStock* | Describes the tooltip format for a Stock chart. <br> |




<a name="SYNTAX3"></a>

### Displaying and customizing the tooltip associated with a point

`grTooltip(<Chart name> , <Series number> , <Category number> , <Tooltip value>)`
---

**`<Chart name>: (Control name or character string)`**

Name of the chart to be used. This name corresponds to:

- the chart name defined through programming with [grCreate](../WDLang3/3042005.md).

- the name of the Chart control in the window or page editor.




**`<Series number>: (Integer)`**

Number of the series to use.

**`<Category number>: (Integer)`**

Number of the category to use.

**`<Tooltip value>: (Character string)`**

Custom text of tooltip associated with the point.

## Remarks


### Tooltip options to configure
<a name="tooltip_options_configure_ELTPARAGRAPHE000335"></a>

| Constant | Effect | Option value |
| --- | --- | --- |
| *grShowTooltip* | Enables or disables the tooltip. | Boolean<br><br>- <u><u><u><u>True</u></u></u></u> (by default) to enable the tooltip, <br><br>- <u><u><u><u>False</u></u></u></u> to avoid displaying it.<br><br><br>**Caution**: the parameters defined for the tooltip are not re-initialized during the display. |
| *grTooltipFormat* | Describes the tooltip format. | Character string<br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%VALUE%]: Displays the value <br>	Example: "[%VALUE%]"+CR+"[%CATEGORY%]"<br><br><br>By default, the format is as follows: "[%CATEGORY%]"+CR+"[%SERIES%]"+"="+"[%VALUE%]" |
| *grTooltipFormatStock* | Describes the tooltip format for a Stock chart. | Character string<br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%MAX%]: Displays the maximum value of the session. <br>	[%MIN%]: Displays the minimum value of the session. <br>	[%START%]: Displays the first value of the session. <br>	[%END%]: Displays the end value of the session.<br><br><br>By default, the format is as follows: <br>"[%CATEGORY%]"+CR+"Min.=[%MIN%]"+CR+"Max.=[%MAX%]"+CR+"Start=[%START%]"+CR+"End=[%END%]" |
| *grTooltipFormatRadialBar* | Describes the tooltip format for a Radial Bar chart. | Character string <br><br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%VALUE%]: Displays the value <br>	[%%%]: Displays the percentage<br>	Example: "[%VALUE%]"+CR+"[%CATEGORY%]"+CR+"[%%%]"<br><br><br>By default, the format is as follows: "[%CATEGORY%]"+CR+"[%VALUE%]"+CR+"[%%%]" |
| *grTooltipFormatCrosshairHeader* | Describes the header format for the chart crosshairs. | Character string<br><br>- Empty string to use the default format. <br><br>- Character string containing the following string: <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br><br><br>By default, the format is as follows: "[%CATEGORY%]"<br>Stacked Column chart: The total of stack can be added into the tooltip of a stacked column chart by using the following syntax: <br><br><pre><code>grTooltip(MySelf, grTooltipFormatCrosshairHeader, ...<br>"[%CATEGORY%], Sum = [%VALUE%]")</code></pre><br> |
| *grTooltipFormatBubbleChart* | Describes the tooltip format for a bubble chart. | Character string<br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%X%]: Displays the X-coordinate (first series)<br>	[%Y%]: Displays the Y-coordinate (second series)<br>	[%Z%]: Displays the value of the third series. <br><br><br>By default, the format is as follows: "([%X%];[%Y%]): [%Z%]"+CR+"[%CATEGORY%]" |
| *grTooltipFormatScatter* | Describes the tooltip format for a Scatter chart. | Character string<br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%X%]: Displays the X-coordinate.<br><br><br>By default, the format is as follows: "[%X%]"+CR+"[%Y%]"+CR+"[%SERIES%]"+CR+"[%CATEGORY%]" |
| *grTooltipFormatHeatMap* | Describes the tooltip format for a Heatmap chart. | Character string<br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%X%]: Displays the X-coordinate (first series)<br>	[%Y%]: Displays the Y-coordinate (second series)<br>	[%Z%]: Displays the value of the third series. <br><br><br>By default, the format is as follows: "([%X%];[%Y%]): [%Z%]"+CR+"[%CATEGORY%]" |
| *grTooltipFormatPie* | Describes the tooltip format for a Pie chart. | Character string <br><br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%SERIES%]: Displays the series label (defined by [grSeriesLabel](../WDLang3/3042024.md)). <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)). <br>	[%VALUE%]: Displays the value <br>	[%%%]: Displays the percentage<br>	Example: "[%VALUE%]"+CR+"[%CATEGORY%]"+CR+"[%%%]"<br><br><br>By default, the format is as follows: "[%CATEGORY%]"+CR+"[%VALUE%]"+CR+"[%%%]" |
| *grTooltipFormatRangeArea* | Describes the tooltip format for a Range Area chart. | Character string <br><br><br>- Empty string to use the default format.<br><br>- Combination of the following strings: <br>	[%START%]: Displays the start value of the interval. <br>	[%END%]: Displays the end value of the interval.  <br>	[%CATEGORY%]: Displays the category label (defined by [grCategoryLabel](../WDLang3/3042015.md)).<br>	Example: "[%START%]"+CR+"[%END%]"+CR+"[%CATEGORY%]"<br><br><br>By default, the format is as follows: "[%START%]"+CR+"[%END%] |


If the default format is used:

- "[%CATEGORY%]" is ignored if no category label was specified by [grCategoryLabel](../WDLang3/3042015.md).

- "[%SERIES%]" is ignored if no series label was specified with [grSeriesLabel](../WDLang3/3042024.md).


Caution: If you use dynamic string construction ("Allow "[% %]" in strings" in the "Compilation" tab of the project description), a compilation error occurs (unknown identifier). In this case, each string must be preceded by '-%'. Example: 

```wl
grTooltip(CHART_Deadline, grTooltipFormat, "[%CATEGORY%]" + CR + CR+ "[%VALUE%]" + " H")
```
 becomes

```wl
grTooltip(CHART_Deadline, grTooltipFormat, -%"[%CATEGORY%]" + CR + CR + -%"[%VALUE%]" + " H")
```

### Enabling/Disabling the tooltip
<a name="enablingdisabling_the_tooltip_ELTPARAGRAPHE001071"></a>

The tooltip activation/deactivation (*grShowTooltip* constant) is taken into account when running [grDraw](../WDLang3/3042026.md).

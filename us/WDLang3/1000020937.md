


## UPNPModifyVolume (Function)

***In french: UPNPModifieVolume***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Change the volume of a UPnP device.

**Caution**: To use this function, the device must include a "RenderingControl" service.
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
arrDevice is array of upnpDevices 
arrDevice = UPNPListDevice(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
FOR EACH Device of arrDevice
	UPNPModifyVolume(Device, 50)
END
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = UPNPModifyVolume(<UPnP device> , <New value>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> if the volume was modified, 

- <u><u><u><u>False</u></u></u></u> otherwise. 




**`<UPnP device>: (upnpDevice variable)`**

Name of [upnpDevice](../WDLang3/1000020903.md) variable corresponding to the UPnP device to use.

**`<New value>: (Integer)`**

New volume to apply. This parameter must be included between 0 and 100.

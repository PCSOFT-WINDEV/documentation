
## TwoFactorAuthenticationGenerateLink (Function)

***In french: DoubleAuthGénèreLien***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Generates a link to register an account in an authentication application. When associated with a Bar Code control, this link will show a QR code that can be scanned in the authentication app installed on the phone. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
let sURL = TwoFactorAuthenticationGenerateLink(sTOTPKey, sEmail, sAppName)
BAC_MyQRCode = sURL
```

<a name="XSYNTAX"></a>

## Syntax
<a name="SYNTAX1"></a>

`<Result> = TwoFactorAuthenticationGenerateLink(<TOTP key> , <Account> , <Application>)`
---

**`<Result>: (Character string)`**

URL used to create an account in an authentication application.

**`<TOTP key>: (Character string)`**

ANSI string that contains the TOTP key of the user to be authenticated. This TOTP key was generated with [DoubleAuthGenerateKey](../WDLang3/1000025974.md). 

**`<Account>: (Character string)`**

Name of the account to be registered (the email address is generally used).

**`<Application>: (Character string)`**

Name of the application to be registered on the authentication application.




## OBEX functions
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
The OBEX functions are as follow: 



|   |   |
| --- | --- |
| [OBEXConnect](../WDLang3/1000017153.md) | Used to connect to a device that supports the OBEX protocol. <br>Reminder: The OBEX protocol is a protocol for exchanging data via Infrared and Bluetooth. |
| [OBEXDisconnect](../WDLang3/1000017154.md) | Used to disconnect from a device that supports the OBEX protocol. |
| [OBEXGetFile](../WDLang3/1000017152.md) | Retrieves a file from a device that supports the OBEX protocol. |
| [OBEXGetVCard](../WDLang3/1000017156.md) | Retrieves a VCard from a device that supports the OBEX protocol. |
| [OBEXListFile](../WDLang3/1000017148.md) | Lists the files shared by a device that is using the OBEX protocol. |
| [OBEXSendFile](../WDLang3/1000017151.md) | Sends a file to a device that supports the OBEX protocol. |
| [OBEXSendVCard](../WDLang3/1000017155.md) | Sends a VCard file (virtual business card) to a device that supports the OBEX protocol. |

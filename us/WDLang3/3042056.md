


## grTitle (Function)

***In french: grTitre***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifies the caption and/or position of the chart title.

Example of title in a chart: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=Titre.gif)


The chart used can be:

- a chart created through programming ([grCreate](../WDLang3/3042005.md)).

- a Chart control in the window editor, page editor or report editor.




**Remark**: A chart has no title by default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Caution**: A specific configuration is required to use this function in Linux. For more details, see [The charts](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Example


```wl
// Modify the caption and the position of title in "MyChart"
grTitle("MyChart", "Turnover projection", grAtBottom)
```
<a name="Example2"></a>

```wl
// Modify the caption and the position of title in CHART_MyChart
grTitle(CHART_MyChart, "Turnover projection", grAtTop)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Syntax

`grTitle(<Chart name> [, <Title> [, <Position>]])`
---

**`<Chart name>: (Control name or character string)`**

Name of the chart to be used. This name corresponds to:

- the chart name defined through programming with [grCreate](../WDLang3/3042005.md).

- the name of the Chart control in the window, page or report editor.




**`<Title>: (Optional character string)`**

Chart title.

**`<Position>: (Optional constant)`**

Indicates the title position:


|   |   |
| --- | --- |
| *grAtBottom* | Title displayed at the bottom of chart. |
| *grAtTop* | Title displayed at the top of chart. |
| *grNone* | No title. |

<a name="NOTE0_1"></a>

## Remarks


### Displaying the title
<a name="displaying_the_title_ELTPARAGRAPHE000167"></a>

The modifications performed by **grTitle** will be taken into account during the next call to [grDraw](../WDLang3/3042026.md).
<a name="NOTE0_2"></a>


### Manual positioning of the title: grTitle without effect 
<a name="manual_positioning_the_title_docparampagetitleshort_without_effect_ELTPARAGRAPHE000181"></a>

**grTitle** has no effect if the title of the chart is positioned manually ("Manual positioning" in the chart title window).
<a name="NOTE0_3"></a>


### Languages that use a non-Latin character set
<a name="languages_that_use_nonlatin_character_set_ELTPARAGRAPHE000198"></a>

**grTitle** will use the selected non-Latin character set only if the associated project configuration uses UNICODE strings at runtime.

To check the selected option in the current configuration: 

1. On the "Project" tab, in the "Project configuration" group, click "Current configuration". 

2. Go to the "Unicode" tab: "Use UNICODE strings at runtime" must be checked. 

3. Validate if necessary.

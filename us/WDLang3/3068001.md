
## SMSReset (Function)

***In french: SMSRAZ***



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>

<a name="sample_code"></a>

## Example


```wl
// Re-initialize the SMS structure
SMSReset()
```


## NetRenameFile (Function)

***In french: NetRenommeFichier***

	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=3&name=warning_big.png"><B>Warning</B><BR>

<a name="Utilisation"></a>
<a name="description"></a>
Renomme (ou déplace) un fichier présent sur le serveur FTP (File Transfer Protocol) ou RPC (Remote Procedure Call) réalisé avec WINDEV.

**Remarque** : Il est possible d'autoriser ou non l'utilisation de cette fonction sur un serveur FTP/RPC grâce à la fonction [NetOptionServeur](../WDLang3/3056022.md).
<a name="Exemple1"></a>
<a name="sample_code"></a>

## Example


```wl
Res est un entier
IDServeurFTP est un entier
IDServeurFTP = NetConnecte("148.61.125.245", ServeurFTP, "GUEST", "")

// Renommer un fichier sur le serveur
Res = NetRenommeFichier(IDServeurFTP, "C:\HISTO\ANNEE98.TXT", ...
	"C:\HISTO\ANNEE98.BAK")
SI Res <> 0 ALORS
	// Procédure de traitement de l'erreur
  	TraiteErreur() 
FIN
```



<a name="XSYNTAXE"></a>
<a name="SYNTAXE1"></a>

## Syntax

`<Résultat> = NetRenommeFichier(<Identifiant connexion> , <Nom du fichier source> , <Nom du fichier renommé>)`
---

**`<Résultat>: (Entier)`**

Résultat de la modification :

- 0 : Le fichier présent sur le serveur a été renommé

- 2 : Le fichier à renommer n'existe pas

- 5 : Accès refusé

- 17 : Déplacement interdit d'un disque vers un autre

- 19 : Disquette protégée en écriture

- 32 : Le fichier est utilisé par un autre programme

- 183 : Un fichier avec le même nom existe déjà




**`<Identifiant connexion>: (Entier)`**

Identifiant de la connexion du poste client (obtenu par la fonction [NetConnecte](../WDLang3/3056004.md)).

**`<Nom du fichier source>: (Chaîne de caractères Ansi)`**

Nom et chemin d'accès complet du fichier à renommer sur le serveur.

**`<Nom du fichier renommé>: (Chaîne de caractères Ansi)`**

Nom et chemin d'accès complet du fichier renommé. Si le chemin d'accès est différent alors que le nom du fichier est identique, le fichier sera déplacé (sur le même disque uniquement).


## URI constants
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


The following constants are used by the URIxxx functions:

| Constant | Details | Function |
| --- | --- | --- |
| **uriAdd** | The contents of the file or buffer are added at the end of the initial contents of the resource. | [FileToURI](../WDLang3/1410087413.md)<br>[BufferToURI](../WDLang3/1410087415.md) |
| **uriAudio** | Supported MIME types: Allows all types of audio files. | [URISelect](../WDLang3/1000024026.md) |
| **uriCreate** | <br><br>- Opens the picker in "Creation" mode to create a new document. <br><br>- The initial contents of the resource are replaced by the contents of the file or buffer.<br><br><br> | <br><br>- [URISelect](../WDLang3/1000024026.md)<br><br>- [FileToURI](../WDLang3/1410087413.md)<br>	[BufferToURI](../WDLang3/1410087415.md)<br><br><br> |
| **uriImage** | Supported MIME types: Allows all types of image files. | [URISelect](../WDLang3/1000024026.md) |
| **uriInfoName** | Requested information: Name of the resource (without extension). | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoSize** | Requested information: Size of the resource (in bytes). | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoConversionType** | Requested information: MIME types (and corresponding file extensions) into which the resource can be converted when being exported. | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoOriginType** | Requested information: Original MIME type of the resource and corresponding file extension. | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriOpen** | Opens the picker in "Open" mode to open an existing document. | [URISelect](../WDLang3/1000024026.md) |
| **uriVideo** | Supported MIME types: Allows all types of image files. | [URISelect](../WDLang3/1000024026.md) |

## Presentación
<a name="presentacion_ELTTEXTE000172"></a>


### ReplicEdit
<a name="replicedit_ELTPARAGRAPHE000011"></a>

ReplicEdit es una herramienta para definir las características de una replicación. Entonces, esta réplica será usada por la réplica universal asistida en el modo automatic (con ReplicSync y ReplicAdmin) o en el modo programado. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Iniciando ReplicEdit
<a name="iniciando_replicedit_ELTTEXTE000196"></a>


### Iniciando ReplicEdit
<a name="iniciando_replicedit_ELTPARAGRAPHE000020"></a>

Para iniciar ReplicEdit:

- en la pestaña "Herramientas", en el grupo "Base de datos", despliegue "Replicación" y seleccione "ReplicEdit - Editor de replicación".

- ejecute "ReplicEdit.EXE" directamente.



<a name="NOTE2_2"></a>


### Condición de uso
<a name="condicion_uso_ELTPARAGRAPHE000042"></a>

ReplicEdit puede ser distribuido.
<a name="NOTE2_3"></a>

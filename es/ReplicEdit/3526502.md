
## Opciones de replicación (ReplicEdit)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000133"></a>
La creación de una réplica con ReplicEdit le permite especificar varios parámetros:

- Las características de la replicación (dirección de la replicación, modo de gestión de los conflictos, ....)

- Los diferentes ficheros a replicar (fichero base, ficheros relacionado,...).




**Dependiendo del tipo de archivos a replicar, se debe especificar información adicional para realizar una replicación universal asistida por automatic (vía ReplicSynchro) o una replicación programada.**

Las opciones de replicación se utilizan principalmente para especificar la ubicación de la base de datos maestra.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Replicar una base de datos HFSQL Classic
<a name="replicar_una_base_datos_hfsql_classic_ELTTEXTE000157"></a>
Cuando se replica una base de datos HFSQL Classic, la ubicación de los archivos maestros para la replicación debe especificarse en el archivo de replicación.

Para definir el directorio de los archivos de datos maestros de HFSQL Classic: 

1. Mostrar el description de la réplica ("Replication .. Replication description").

2. Seleccionar la pestaña "Ficheros maestros. Esta pestaña le permite:

	- Especifique el directorio para los archivos maestros de HFSQL Classic a través del selector de directorios.

	- Configurar las características de cada archivo. Todo lo que tiene que hacer es hacer doble clic en el nombre del archivo: se muestra la ventana de description del archivo. En la pestaña "General" puede especificar:

		- el nombre físico del archivo. Este parámetro puede corresponder a un ruta completo, a un ruta relativo (que se añadirá al directorio base especificado) o a un nombre de fichero. Si se especifica un nombre de archivo, se buscará en el directorio base especificado.

		- la contraseña del archivo (si es necesario).







**Observación**: Esta información también puede especificarse al preparar el procedimiento instalación o durante el instalación.

## Replicación de una base de datos Cliente/Servidor
<a name="replicacion_una_base_datos_clienteservidor_ELTTEXTE000181"></a>


### Conexiones
<a name="conexiones_ELTPARAGRAPHE000045"></a>

Al replicar una aplicación Cliente/Servidor, ReplicEdit utiliza las conexiones definidas en el análisis. **La contraseña no se inicializa por razones de seguridad.**

Para especificar la contraseña asociada a las conexiones (o para modificar las conexiones), seleccione "Replicación .. description de las conexiones".

Permite modificar la asociación entre un fichero y una conexión:

1. Seleccione "Replicación .. description de los archivos de datos".

2. Seleccione el archivo deseado y haga clic en la búsqueda Icon.

3. Seleccione la conexión deseada.




Observaciones:

- Replicación universal asistida por automatic (a través de ReplicSynchro): Estos parámetros pueden modificarse al preparar el instalación.

- Replicación universal asistida mediante programación: Sólo se puede utilizar ReplicEdit para modificar estos parámetros.



<a name="NOTE3_2"></a>


### Posición de réplica
<a name="posicion_replica_ELTPARAGRAPHE000065"></a>

Al replicar una base de datos que utiliza un acceso nativo, se debe definir un elemento DateTime específico en cada archivo replicado.. Este elemento será actualizado por la aplicación cuando se modifique o añada un Record.

Para especificar la posición de reproducción:

1. Seleccione "Replicación .. description de los archivos de datos".

2. Haga doble clic en el fichero relevante. Se muestra la ventana description del archivo.

3. En la pestaña "Artículos", especifique el artículo para la fecha de Record.

## Seleccionar campos a replicar
<a name="seleccionar_campos_replicar_ELTTEXTE000211"></a>
ReplicEdit también le permite ignorar algunos elementos de los archivos de datos durante la replicación.

Para evitar replicar todos los elementos:

1. Seleccione "Replicación .. description de los archivos de datos".

2. Haga doble clic en el archivo de datos correspondiente. Se muestra la ventana description del archivo.

3. En la pestaña "Items", sólo se replicarán los ítems marcados.

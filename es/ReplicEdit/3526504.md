## Presentación
<a name="presentacion_ELTTEXTE000107"></a>
Cuando se describe la replicación, los archivos de datos deben ser añadidos a la replicación. Esta operación se realiza a través de una asistente.

## Agregar archivos de datos en una réplica
<a name="agregar_archivos_datos_una_replica_ELTTEXTE000131"></a>


### Añadiendo el archivo base
<a name="anadiendo_archivo_base_ELTPARAGRAPHE000021"></a>

1. Seleccione "Replicación .. Añadir un archivo de datos". Comienza la asistente para añadir un archivo.

2. Se muestra la lista de archivos de datos descritos en el análisis actual o encontrados en el servidor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ReplicEdit_FichierDeBase.gif&type=thumb)
 Tiene la posibilidad de seleccionar uno o más archivos de datos para sincronizar. Todos los registros son replicados por Default. 

3. Para los archivos de datos seleccionados, se puede definir: 

	- la dirección de replicación de los archivos de datos: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ReplicEdit_FichierSens.gif&type=thumb)


	- el modo de gestión de los conflictos de modificación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ReplicEdit_FichierConflit.gif&type=thumb)

			Puede seleccionar: 

	- el modo Default definido de antemano (recomendado). 

	- otro modo. 

4. La asistente está terminada. La réplica se describe. El editor de la réplica enumera los archivos de datos que se tienen en cuenta en la réplica. 




Los archivos mostrados en gris son archivos de datos de relacionado: un archivo de datos de relacionado es un archivo de datos que es relacionado a un archivo base (que ya pertenece a la réplica). Dependiendo de la replicación realizada, los datos encontrados en el archivo relacionado serán añadidos o no.. 

Puedes guardar la réplica ("Archivo .. Guardar"). La réplica se guarda en el formato de un archivo ".Wer".. 

Este archivo contiene la description de la réplica. Debe encontrarse junto al ejecutable en los ordenadores de los suscriptores.


### Añadir un archivo relacionado
<a name="anadir_archivo_relacionado_ELTPARAGRAPHE000047"></a>

Para añadir un archivo relacionado: 

1. Doble click en el archivo de datos de relacionado (mostrado en gris) en ReplicEdit. Se inicia la asistente para añadir un archivo de datos de la relacionado.

2. Por Default, los datos del archivo relacionado son los datos correspondientes al archivo base. Por lo tanto, si se filtra el fichero base, los registros encontrados en el fichero relacionado utilizan el mismo filtro. 
	Observación: Para implementar un filtro en el archivo base, despliegue el [descripción de archivo](../ReplicEdit/3526506.md). 

3. Especificar si se debe realizar un filtrado adicional. Si añade un filtro, especifique las características necesarias. 

4. A continuación, especifique la dirección de replicación para el archivo relacionado, así como el modo de gestión de conflictos.

5. Valide el asistente. El archivo relacionado aparece ahora como archivo a replicar.


## Descripción de la replicación (ReplicEdit)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
ReplicEdit se utiliza para crear una réplica a través de una asistente, pero la ventana de réplica description permite identificar o configurar parámetros adicionales. 

**To editar el description de una réplica**, seleccionar "Replication .. Replicación description". 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las diferentes opciones
<a name="las_diferentes_opciones_ELTTEXTE000144"></a>


### Pestaña "General"
<a name="pestana_general_ELTPARAGRAPHE000018"></a>

La pestaña "General" presenta la siguiente información: 

- Nombre de la réplica,

- description de la réplica

- Dirección de replicación, 

- Gestión de conflictos. 




Esta información se introdujo al crear la réplica y puede modificarse en cualquier momento.. 


### "Ficha Parámetros
<a name="ficha_parametros_ELTPARAGRAPHE000030"></a>

La pestaña "Parámetros" indica los diferentes parámetros esperados por la replicación. 

Estos parámetros corresponden a los filtros implementados en los archivos a replicar. 

Así, es posible get el orden de los parámetros utilizados por [ReplicInitialize](../WDLang4/1000017195.md). 


### "Ficha"Archivos maestros
<a name="fichaarchivos_maestros_ELTPARAGRAPHE000042"></a>

Esta pestaña se utiliza para especificar: 

- el directorio principal de los archivos maestros de HFSQL Classic, cuando se replica una base de datos HFSQL Classic. 

- el modo de acceso y la ubicación de los archivos maestros para su replicación. 





### pestaña "Avanzado"
<a name="pestana_avanzado_ELTPARAGRAPHE000050"></a>

Esta pestaña se utiliza para especificar las opciones de replicación avanzadas: 

- ****: Esta opción determina automáticamente el orden de los archivos replicados (según las restricciones de integridad). Si no se selecciona esta opción, el orden se determina por el orden definido en la pestaña "Parámetros".. 

- ****: Esta opción se utiliza para tener en cuenta la modificación de datos de automatic. En ese caso: 

	- Los cambios en la estructura de la base de datos principal se trasladarán a la base de datos de suscriptores.

	- Los nuevos elementos serán tenidos en cuenta por la réplica. 




- <u><u><u><u>False</u></u></u></u> (valor predeterminado). La modificación de datos de automatic realizada en la base de datos maestra no se transfiere a la base de datos de suscriptores.  Atención: este parámetro sólo está disponible para su replicación entre bases de datos HFSQL (Clásico o Cliente/Servidor).

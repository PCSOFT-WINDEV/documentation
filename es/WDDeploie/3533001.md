## Presentación

### WDDeploy
<a name="wddeploy_ELTPARAGRAPHE000011"></a>

WDDDeploy es una herramienta que se utiliza para implementar fácilmente un sitio estático o un sitio PHP.. Hay dos métodos de implementación disponibles:

- **remota implementación (vía FTP)**: Recomendado cuando el servidor no es directamente accesible.

- **implementación por la red**: Recomendado cuando el servidor es directamente accesible.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Inicio de WDDeploy
<a name="inicio_wddeploy_ELTTEXTE000146"></a>
Para iniciar WDDeploy:

- en la pestaña "Herramientas", en el grupo "Web utilities", haga clic en "WDDeploy".

- ejecute "WDDeploy.EXE" directamente.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Condiciones de uso
<a name="condiciones_uso_ELTTEXTE000170"></a>
WDDDeploy puede redistribuirse. WDDDeploy puede utilizarse para instalar cualquier sitio estático (creado por WEBDEV o no).

La licencia WEBDEV se aplica plenamente.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Funcionalidades de WDDeploy
<a name="funcionalidades_wddeploy_ELTTEXTE000194"></a>
WDDDeploy está acostumbrado a:

- instalar los archivos de un sitio estático o PHP en un servidor Web.

- comprimir y agrupar los archivos encontrados en un sitio remoto en archivos.

- restauración de los archivos encontrados en un archivo comprimido.




Para obtener más información, consulte [Funcionalidades de WDDeploy](../WDDeploie/3533002.md).

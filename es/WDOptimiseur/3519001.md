
## WDOptimizer: Edición y modificación de un archivo ".REP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000101"></a>
El archivo &lt;MiProyecto&gt;.REP contiene la lista de archivos utilizados por la aplicación (identificador, nombre lógico y ruta completa del archivo físico).

La GUÍA del análisis es el identificador único del análisis relacionado al proyecto, que contiene el description de los archivos. Este identificador se encuentra en el editor de análisis, en el análisis description (pestaña "Opciones").

El fichero GUID corresponde al identificador del fichero lógico. Este identificador se encuentra en la editor de análisis, en la ventana de archivo description (pestaña "Notas").

Este archivo se crea automáticamente en el directorio de la aplicación y se rellena con el motor HFSQL.

## Archivos REP
<a name="archivos_rep_ELTTEXTE000125"></a>


### ¿Para qué sirven los archivos REP?
<a name="¿para_que_sirven_los_archivos_rep_ELTPARAGRAPHE000024"></a>

El archivo.REP se utiliza para localizar fácilmente los archivos de datos que han sido utilizados por la aplicación WINDEV o por el sitio WEBDEV.

La aplicación WINDEV o el sitio WEBDEV actualiza automáticamente el archivo.REP pero no suele usar el archivo.REP.

Este archivo es utilizado por todas las herramientas que manipular los archivos de la aplicación, y principalmente por la actualización del archivo automatic, etc.
<a name="NOTE2_2"></a>


### Ejemplo: Actualizar una aplicación mientras se modifica el análisis
<a name="ejemplo_actualizar_una_aplicacion_mientras_modifica_analisis_ELTPARAGRAPHE000035"></a>

Cuando se actualiza una aplicación WINDEV, la modificación de los archivos de datos de automatic se inicia automatic ally si se modifica la estructura de la base de datos.

Este Procedure utiliza el archivo .REP para encontrar los archivos físicos utilizados por la aplicación con el fin de modificarlos.

## Edición y modificación de un archivo.REP
<a name="edicion_modificacion_archivorep_ELTTEXTE000155"></a>


### Edición y modificación de un archivo.REP
<a name="edicion_modificacion_archivorep_ELTPARAGRAPHE000046"></a>

Para editar y modificar un archivo REP:

1. Iniciar WDOptimizer.

2. Seleccione "Editar y modificar un archivo.REP".

3. Seleccione el archivo.REP. Se listan los archivos que se encuentran en el archivo.REP seleccionado.

4. Para eliminar todos los archivos que no existen, haga clic en el botón "Limpiar".

5. Para borrar el archivo seleccionado, haga clic en el botón "Borrar".

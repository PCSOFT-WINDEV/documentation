## Presentación
<a name="presentacion_ELTTEXTE000245"></a>


### WDOptimizer
<a name="wdoptimizer_ELTPARAGRAPHE000011"></a>

Las diferentes operaciones que se pueden realizar con WDOptimizer son las siguientes:

- **Optimización de los índices**
	Esta opción le permite realizar varios tipos de operaciones:

	1. **Consulta los índices**: Comprueba la coherencia entre el índice y los archivos de datos.

	2. **Optimizar la velocidad de los índices (recalcular estadísticas)**: Optimiza los índices calculando estadísticas sobre los índices. Estas estadísticas se utilizan para optimizar los filtros, las consultas y las vistas HFSQL.. 
			Observación: Cuanto más se modifique el el archivo de datos, menos las estadísticas reflejarán el contenido del archivo de datos HFSQL. Cuanto más Record s el archivo de datos contiene, menos la modificación de un solo Record impacta las estadísticas generales.

	3. **Reconstruir los índices**: Optimiza el acceso a los registros del archivo de datos y recalcula las estadísticas. Todos los registros cruzados se borran automáticamente. 

	4. **Reconstruir los índices y las notas.**: Optimiza todo el archivo de datos, su índice y sus notas.

	5. **Comprueba y comprime los índices y los memos**: Optimiza todo el archivo de datos, su índice y sus memorandos. Los memos serán comprimidos. Existen opciones avanzadas ("Opciones para comprimir los memos").




- **Ver las propiedades de un archivo**
	Muestra las características de un archivo de datos (nombre, extension, formato, codificación, ...).

- **Editar y modificar un archivo ".REP"**
	Optimiza el acceso a los archivos de datos.

- **Revertir una transacción o eliminar registros de una transacción**
	Cancela una transacción sin interrumpir la ejecución del programa.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000275"></a>


### Iniciando WDOptimizer
<a name="iniciando_wdoptimizer_ELTPARAGRAPHE000043"></a>

WDOptimizer puede iniciarse en modo interactivo a través de uno de los siguientes métodos:

- Desde WINDEV, WEBDEV o WINDEV Mobile, en la pestaña "Herramientas", en el grupo "Base de datos", haga clic en "WDOptimizer".

- Ejecute directamente "WDOptimizer.EXE".




Las diferentes operaciones que se pueden realizar con WDOptimizer son las siguientes:

1. Optimizar índices. Para obtener más información, consulte [Optimizar índices](../WDOptimiseur/3519004.md).

2. Visualización de las propiedades de un archivo. Este archivo de datos puede seleccionarse en la lista que se muestra en la pantalla o puede seleccionarse mediante el selector de archivos.

3. Edición de un archivo.REP: Seleccione el archivo .REP deseado. Para obtener más información, consulte [Edición y modificación de un fichero ".REP](../WDOptimiseur/3519001.md)

4. Gestión de las transacciones. Para obtener más información, consulte [Retroceder una transacción o eliminar registros de una transacción](../WDOptimiseur/3519006.md)




**Observación**: En el ordenador de desarrollo, WDOptimizer puede iniciarse directamente desde WDTool.

WDOptimizer también puede ser ejecutado en el comando Line. Para obtener más información, consulte [Iniciar WDOptimizer en una línea de comandos](../WDOptimiseur/3519005.md).

### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000086"></a>

WDOptimizer puede ser distribuido libremente. WDOptimizer se puede instalar con las aplicaciones desarrolladas con WINDEV o WEBDEV. El acuerdo de licencia de WINDEV o WEBDEV se aplica en su totalidad.

Al crear el instalación Procedure, WDOptimizer se suministra automáticamente con sus aplicaciones.

**Recordatorio**: El procedimiento de instalación se puede crear: 

- mediante el asistente: en la pestaña "Proyecto", en el grupo "Generación", haga clic en "Procedimiento de instalación". 

- desde el editor de instalación: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

## Recordatorio: Índice de archivos
<a name="recordatorio_indice_archivos_ELTTEXTE000311"></a>
El índice de un archivo de datos es un archivo específico (.NDX). Este archivo se crea de acuerdo con las teclas de navegación definidas en el archivo de datos. Le permite encontrar rápidamente datos en un archivo de datos.

Para un mejor rendimiento de las búsquedas realizadas en sus archivos de datos, el índice debe ser optimizado regularmente.

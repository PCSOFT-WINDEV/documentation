
## ¿Cómo saber si se imprimió una reporte?
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


### Problemas
<a name="problemas_ELTPARAGRAPHE000010"></a>

Cuando se imprime una reporte, es muy común dirigir la impresión a una vista previa. 

En algunos casos, un papel Document debe ser impreso físicamente.

¿Cómo averiguar mediante programación si la Document se imprimió o no?
<a name="NOTE1_2"></a>


### Soluciones
<a name="soluciones_ELTPARAGRAPHE000021"></a>

Hay varias funciones disponibles: 

- [iDocumentoImpreso](../WDLang5/3046029.md) se utiliza para saber si el reporte se imprimió desde la vista previa.

- [iDocumentCanceled](../WDLang5/3046006.md) se utiliza para saber si el usuario:

	- cerró la vista previa de impresión antes del final de las impresiones,

	- hizo clic en el botón Cancel en la ventana de impresión actual.

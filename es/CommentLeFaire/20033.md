
## ¿Cómo gestionar las enumeraciones mediante programación?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una enumeración es un tipo de variabble que contiene una lista de valores. Este tipo de Variable se utiliza, por ejemplo, para normalizar los posibles valores de un parámetro. Por lo tanto, el desarrollador podrá inicializar este parámetro a partir de los posibles valores impuestos por la enumeración. 

La Variable associated con esta enumeración puede tomar un solo valor de lista.

Para más detalles sobre las enumeraciones, véase [Enumeración (Tipo de Variable)](../Motscles/1514067.md).

## Declaración y uso
<a name="declaracion_uso_ELTTEXTE000116"></a>


### Sintaxis
<a name="sintaxis_ELTPARAGRAPHE000023"></a>


```txt
EnumerationName is Enumeration
	Possible value 1
	Possible value 2
	...
END


VariableName is EnumerationName 

VariableName = Possible value 1
```



### Ejemplo
<a name="ejemplo_ELTPARAGRAPHE000028"></a>

- Declarar la enumeración: 
	
	```wl
	// Global declarations of the XXX window
	PossibleAction is Enumeration
		Creation
		Modification
		Deletion
		Printing
	END
	```

	```wl
	// Use the enumeration
	ActionParam is PossibleAction
	
	ActionParam = Creation
	
	Open(WIN_Customer, ActionParam)
	```


- En las "Declaraciones globales" Event de WIN_Customer: 
	
	```wl
	PROCEDURE WIN_Customer(WorkingMode is PossibleAction)
	```

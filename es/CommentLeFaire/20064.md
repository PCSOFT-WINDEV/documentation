
## ¿Cómo recuperar la estructura de análisis mediante programación?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000091"></a>
El análisis representa la estructura de los archivos de datos manejados por el programa. 

Tiene la posibilidad de recuperar la estructura de análisis a través de la programación para realizar comprobaciones o proponer una interfaz de usuario para interrogar a una base de datos de forma genérica.

Las siguientes funciones son necesarias para recuperar la estructura de análisis: 

- [HOpenAnalysis](../WDLang4/3044106.md)para abrir un análisis.

- [HListFile](../WDLang4/3044045.md)para recuperar la lista de archivos de análisis.

- [HListItem](../WDLang4/3044230.md)a get la lista de elementos de archivo.

- [HListKey](../WDLang4/3044060.md)para recuperar la lista de claves de archivo.

- [HListLink](../WDLang4/3044138.md)para recuperar los enlaces entre los archivos.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ejemplo de recuperación
<a name="ejemplo_recuperacion_ELTTEXTE000115"></a>
Este ejemplo se utiliza para recuperar la lista de todos los archivos encontrados en el análisis actual. Para cada archivo listado, el ejemplo recupera: 

- la lista de artículos,

- la lista de llaves,

- la lista de enlaces.




Código de ejemplo: 


```wl
// Declare the variables
sFileList is string
sItemList is string
sKeyList is string
sLinkList is string

// Load the list of analysis files
sFileList = HListFile()

// Browse the list of analysis files
FOR EACH STRING sFileName OF sFileList SEPARATED BY CR
	
	// Load the file items
	sItemList = HListItem(sFileName, hLstDetail)
	// Load the file keys
	sKeyList = HListKey(sFileName, hLstDetail)
	// Load the file links
	sLinkList = HListLink(sFileName)
	
	
	Trace("================ FILE ================")
	Trace("File: ", sFileName)
	Trace("--- List of items ---")
	FOR EACH STRING sItem OF sItemList SEPARATED BY CR
		Trace(sItem)
	END
	Trace("--- List of keys ---")
	FOR EACH STRING sKey OF sKeyList SEPARATED BY CR
		Trace(sKey)
	END
	Trace("--- List of links ---")
	FOR EACH STRING sLink OF sLinkList SEPARATED BY CR
		Trace(sLink)
	END
END
```

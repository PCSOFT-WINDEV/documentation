
## ¿Cómo leer los correos electrónicos a través del protocolo POP3?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Principio
<a name="principio_ELTTEXTE000096"></a>
Para leer correos electrónicos a través del protocolo POP3 sin utilizar un software de mensajería, debe: 

1. Recuperar los parámetros de conexión a su servidor POP3 (estos parámetros son suministrados por su proveedor de servicios INTERNET o por su administrador de red). Estos parámetros deben almacenarse a través de un [emailPOP3Session](../WDLang3/1000018759.md) Variable.

2. Conectar con el servidor POP3 a través de [EmailStartPOP3Session](../WDLang3/3032025.md). 

3. Realiza un bucle para leer los correos electrónicos desde el principio hasta el final utilizando [EmailReadFirst](../WDLang3/3032014.md) y [EmailReadNext](../WDLang3/3032004.md). El valor de la variable **Email.Out** es: 

	- <u><u><u><u>True</u></u></u></u> si no hay más mensajes por leer.

	- <u><u><u><u>False</u></u></u></u> en caso contrario.




4. Utilizar una [Email](../WDLang3/1000018713.md) Variable para recuperar el contenido de Email que fue leído. 

5. Cerrar la sesión POP3 mediante [EmailCloseSession](../WDLang3/3032006.md). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000120"></a>

```wl
MyMessage is Email
MyPOP3Session is emailPOP3Session


// Simple POP3 session
MyPOP3Session.ServerAddress = "pop3.mycompany.us"
MyPOP3Session.Name = "user"
MyPOP3Session.Password = "secret"

// Start the POP3 session
IF NOT EmailStartSession(MyPOP3Session) THEN
	// Error starting the IMAP session
	Error("Unable to start the POP3 session.", ErrorInfo())
ELSE
	
	// Read the messages
	EmailReadFirst(MyPOP3Session, MyMessage)
	WHILE NOT Email.Out
		// Process the message read
		Trace(MyMessage.Subject, MyMessage.Sender, MyMessage.Message)

		// Read the next message
		EmailReadNext(MyPOP3Session, MyMessage)
	END
	
	// Disconnect from the session
	EmailCloseSession(MyPOP3Session)
	
END
```

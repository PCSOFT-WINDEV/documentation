## Notas de sesión (LotusNotes)
<a name="notas_sesion_lotusnotes_ELTTEXTE000139"></a>
Para iniciar la sesión de Notes, utilice [EmailStartNotesSession](../WDLang3/3032121.md). Esta función establece una conexión con el software de mensajería de Lotus Notes.
En este caso, todas las funciones utilizadas ([EmailSend](../WDLang3/3032003.md), [EmailReadFirst](../WDLang3/3032014.md), ...) serán procesadas por el software que manipular la base de datos Notes.
Todo está gestionado por Notes (seguridad, historial, envío,...).
Lotus Notes debe estar instalado en el ordenador del usuario.

## Sesión de Outlook
<a name="sesion_outlook_ELTTEXTE000163"></a>
Para iniciar la sesión de Outlook, utilice [EmailStartOutlookSession](../WDLang3/3032128.md). Esta función establece una conexión con el software de mensajería Outlook.
En este caso, todas las funciones utilizadas ([EmailSend](../WDLang3/3032003.md), [EmailReadFirst](../WDLang3/3032014.md), ...) serán procesadas por el software que manipular la base de datos de Outlook.
Todo está gestionado por Outlook (seguridad, historial, envío,...). 
Outlook debe estar instalado en el ordenador del usuario.

## Sesión de intercambio
<a name="sesion_intercambio_ELTTEXTE000187"></a>
Para iniciar la sesión de intercambio, utilice [EmailStartSession](../WDLang3/3032028.md). Esta función establece una conexión con un servidor Exchange.
En este caso, todas las funciones utilizadas ([EmailSend](../WDLang3/3032003.md), [EmailReadFirst](../WDLang3/3032014.md), ...) serán procesadas por el servidor Exchange.
Todo está gestionado por Exchange (seguridad, historial, envío,...).

## Conexión SMTP (sólo envío)
<a name="conexion_smtp_solo_envio_ELTTEXTE000211"></a>
Para una conexión SMTP, utilice [EmailStartSMTPSession](../WDLang3/3032025.md). Esta función establece una conexión con el servidor de mensajería según el protocolo SMTP.
 Este protocolo se utiliza para enviar correos electrónicos. No se puede utilizar para leer los correos electrónicos.
**Atención**: no se realiza ninguna copia de seguridad cuando se envían correos electrónicos.

## Conexión POP3 (sólo lectura)
<a name="conexion_pop3_solo_lectura_ELTTEXTE000235"></a>
Para una conexión POP3, utilice [EmailStartPOP3Session](../WDLang3/3032022.md). Esta función establece una conexión con el servidor de mensajería de acuerdo con el protocolo POP3.. 
Este protocolo se utiliza para leer correos electrónicos. No se puede utilizar para enviar los correos electrónicos.

<a name="NOTE6"></a>
<a name="NOTE6_1"></a>


## Conexión POP3 (sólo lectura)
<a name="conexion_pop3_solo_lectura_ELTTEXTE000259"></a>
Para una conexión mixta (SMTP+POP3), utilice [EmailStartSession](../WDLang3/3032028.md). Esta función establece una conexión con el servidor de mensajería según los protocolos SMTP y POP3.. Estos protocolos se utilizan respectivamente para enviar correos electrónicos y para leer correos electrónicos.
**Atención**: no se realiza ninguna copia de seguridad de la traza cuando se envían correos electrónicos.

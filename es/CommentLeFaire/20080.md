
## ¿Cómo set el foco a una ventana control?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Principio
<a name="principio_ELTTEXTE000097"></a>
Se pueden utilizar dos métodos para enfocar set a un control:

- Utilización de la función [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md) o [SetFocus](../WDLang1/3025001.md). 

- Utilización de la función [ScreenFirst](../WDLang1/3025039.md). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Primer método: utilizando SetFocus AndReturnToUserInput o SetFocus
<a name="primer_metodo_utilizando_setfocus_andreturntouserinput_setfocus_ELTTEXTE000121"></a>
[SetFocusAndReturnToUserInput](../WDLang1/1410088107.md) (o [SetFocus](../WDLang1/3025001.md)) da el foco al control especificado y permite volver a editar la ventana **sin ejecutar el código WLanguage que se encuentra después de la llamada a la función**.

Ejemplo de código: 


```wl
IF EDT_CODE = "" THEN	
	SetFocusAndReturnToUserInput(EDT_CODE)
	// Code not run
ELSE
	SetFocusAndReturnToUserInput(EDT_HEADING)
	// Code not run
END
// Code not run
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Segundo método: utilizando ScreenFirst
<a name="segundo_metodo_utilizando_screenfirst_ELTTEXTE000145"></a>
[ScreenFirst](../WDLang1/3025039.md) se utiliza para almacenar el control que ganará atención. El código WLanguage que sigue a la llamada a la función continúa ejecutándose. Esto le permite cambiar la control sobre la que quiere posicionarse antes de volver a editar la ventana.

Ejemplo de código: 


```wl
// Depending on the result of processes 1 and 2 
// the position is set at the end of the process on EDT_QTY or EDT_DISCOUNTCODE

// Run process part 1
...
If nResPart1 = 0 then ScreenFirst(EDT_QTY)
...
// Run process part 2
...
IF bResPart2 = False then ScreenFirst(EDT_DISCOUNTCODE)
...
```

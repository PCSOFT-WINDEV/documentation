
## ¿Cómo gestionar una combinación a través de la programación?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una combinación es un tipo de variabble que contiene una lista de valores. Este tipo de Variable se utiliza para normalizar los posibles valores de un parámetro. Por ejemplo, el desarrollador podrá inicializar este parámetro a partir de los posibles valores impuestos por la combinación. 

El Variable associated con esta combinación puede tomar varios valores de la lista al agregarlos.

Para obtener más información, consulte [Combinación (Tipo de Variable)](../Motscles/1514074.md).

## Declaración y uso
<a name="declaracion_uso_ELTTEXTE000120"></a>


### Sintaxis
<a name="sintaxis_ELTPARAGRAPHE000023"></a>

La siguiente sintaxis se utiliza para declarar una combinación: 


```txt
CombinationName is Combination 
	Possible value 1
	Possible value 2
	Possible value 3
	...
END
```


Para utilizar una combinación declarada de antemano, utilice la siguiente sintaxis: 


```txt
VariableName is CombinationName 

VariableName = Possible value 1 + Possible value 3
```



### Ejemplo
<a name="ejemplo_ELTPARAGRAPHE000034"></a>


```wl
PossibleMode is Combination 
	Bold
	Italic
	Underlined
END

ParamMode is PossibleMode 

ParamMode = Bold + Underlined
```

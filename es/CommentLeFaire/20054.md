
## ¿Cómo asociar una Image a un elemento que se encuentra en una Tabla TreeView?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Cuando se llena un control Tabla TreeView, una imagen es associated con cada elemento. Es un pequeño símbolo que representa una carpeta abierta o cerrada.. De hecho, cada nivel y cada elemento de la jerarquía puede serlo:

- colapsado (imagen de la carpeta cerrada de Default).

- ampliado (imagen de la carpeta abierta por Default).


Estas imágenes pueden modificarse mediante programación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Método
<a name="metodo_ELTTEXTE000102"></a>
Para modificar las imágenes que representan el elemento "colapsado" o "expandido", utilice las propiedades [CollapsedImage](../Proprietes/2513016.md) y [ExpandedImage](../Proprietes/2513015.md). La siguiente sintaxis le permite utilizar estas propiedades:


```txt
<TreeView Table name>[Row index].CollapsedImage = <Image name>
<TreeView Table name>[Row index].ExpandedImage = <Image name>
```


Ejemplos de código:


```wl
TVT_Product[10]..CollapsedImage = "ProductNormalPrice.jpg"
TVT_Product[11]..CollapsedImage = "ProductBargainPrice.jpg"
```

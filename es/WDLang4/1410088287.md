


## Funciones de gestión de Redis (sintaxis de prefijos)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular Redis bases de datos, utilizando la sintaxis de prefijo: 



|   |   |
| --- | --- |
| [.Connect](../WDLang4/1410088280.md) | Abre una conexión a un servidor (o cluster) de Redis. |
| [.ExecuteCommand](../WDLang4/1410088282.md) | Ejecuta un comando en un servidor Redis y recupera el resultado. |
| [.get](../WDLang4/1410088278.md) | Obtiene el valor de una clave de un servidor Redis como una cadena. |
| [.PipelineAdd](../WDLang4/1410088283.md) | Agrega un comando Redis en un pipeline. |
| [.set](../WDLang4/1410088245.md) | Cambia el valor de una clave en un servidor Redis. |
| [.Subscribe](../WDLang4/1410088279.md) | Suscripción a un canal de emisión de Redis. |
| [Anular suscripción](../WDLang4/1410088281.md) | Cancela una suscripción a un canal de difusión de Redis. |
| [PipelineGet](../WDLang4/1410088285.md) | Recupera el resultado de la ejecución de un comando Redis que fue añadido por [RedisPipelineAdd](../WDLang4/1410088283.md). |
| [PipelineReset](../WDLang4/1410088284.md) | Ignora los resultados de la ejecución de un comando Redis que fue añadido por [RedisPipelineAdd](../WDLang4/1410088283.md). |
| [Publicar](../WDLang4/1410088286.md) | Publica un mensaje en un canal de difusión de Redis. |

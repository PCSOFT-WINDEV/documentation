
## Constantes para la replicación asistida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por la función |
| --- | --- | --- |
| **replicaImmediate** | La réplica se crea inmediatamente, durante la sincronización.. Este modo de replicación puede ser bastante largo, pero garantiza que se actualice desde la base de datos maestra. | [ReplicSynchronize](../WDLang4/1000017196.md) |
| **replicaPeriodic** | La réplica se crea después de cada sincronización. Durante la siguiente réplica, la réplica que se creó previamente se utiliza automáticamente.. Este modo de replicación es rápido, pero no garantiza una "actualización completa" de la base de datos maestra. | [ReplicSynchronize](../WDLang4/1000017196.md) |
| **replicaNoProgressBar** | Desde la versión 190044, una barra de progreso es mostrada por Default durante la sincronización. Esta barra de progreso está adaptada a la plataforma actual. <br>Esta constante le permite no mostrar la barra de progreso y restaurar el comportamiento anterior.. <br>Esta constante puede combinarse con las constantes *replicaImmediate* y *replicaPeriodic*. | [ReplicSynchronize](../WDLang4/1000017196.md) |
| **replicSynchronizationRequestDate** | Fecha de la última solicitud de sincronización. | [ReplicInfo](../WDLang4/1000017197.md) |
| **replicSynchronizationDate** | Fecha de la última sincronización. | [ReplicInfo](../WDLang4/1000017197.md) |
| **replicExist** | Verifica la existencia de la réplica. | [ReplicInfo](../WDLang4/1000017197.md) |
| **replicInitialized** | Verifica si la réplica está inicializada. | [ReplicInfo](../WDLang4/1000017197.md) |

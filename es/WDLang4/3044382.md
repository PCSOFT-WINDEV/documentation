
## H.LanguageCreationUnicode (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite definir las características del tipo de los artículos Unicode que se encuentran en un archivo de datos. Este Variable debe utilizarse antes de crear los archivos de datos ([HCreation](../WDLang4/3044255.md) y [HCreationIfNotFound](../WDLang4/3044254.md)). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
H.LanguageCreationUnicode = "AR" + TAB + "MA"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`H.LanguageCreationUnicode(<Language characteristics>)`
---

**`<Language characteristics>: (Cadena de caracteres)`**



- Características del idioma que se utilizará para la clasificación, en el siguiente formato: 
	
	```txt
	<Language identifier> + TAB + <Country identifier> + TAB + <Variant>
	```


- Constante utilizada para definir una clasificación específica: 
	


|   |   |
| --- | --- |
| *hLanguageBinarySort* | La ordenación se realizará comparando el valor binario de los bytes. Este tipo de clasificación es muy rápida. Se debe utilizar para realizar una búsqueda de coincidencia exacta, por ejemplo. |
| *hLanguageFastSort* | El orden se realizará comparando los caracteres uno por uno de acuerdo a un orden genérico. |


## HInfoLog (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre los registros del servidor.

**Recordatorio**: Los registros del servidor se utilizan para mostrar las llamadas al servidor realizadas por los clientes.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Information is string
// Retrieve the correspondence between the function numbers 
// and the name of the WLanguage functions
Information = HInfoLog("MyConnection", hCorrespondence)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HInfoLog(<Connection name> , <Type of information>)`
---

**`<Result>: (Cadena de caracteres)`**

Resultado detallado del tipo de información solicitada. 

Si el tipo de información solicitada corresponde a la constante *hCorrespondence*, &lt;Resultado&gt; tiene el siguiente formato: 

```txt
<Number 1> + TAB + <Parameter 1> + TAB + <Value 1> + TAB + <French name 1> + TAB + 
<English name 1> + CR + ... + <Number N> + TAB + <Parameter N> + TAB + <Value N> + TAB + 
<French name N> + TAB + <English name N>
```

Donde:

- &lt;Number&gt; corresponde al número utilizado para identificar la función en los registros.

- &lt;Parámetro&gt; corresponde al parámetro del registro que sirve para diferenciar entre funciones similares ([HReadFirst](../WDLang4/3044051.md)/[HReadNext](../WDLang4/3044037.md) por ejemplo).

- &lt;El valor&gt; corresponde al valor del parámetro.

- &lt;El nombre francés&gt; corresponde al nombre de la función WLanguage en francés.

- &lt;El nombre en inglés&gt; corresponde al nombre de la función WLanguage en inglés.




**`<Connection name>: (Cadena de caracteres)`**

Nombre de la conexión con el servidor para el que se solicita información sobre los registros.

**`<Type of information>: (Constante de tipo Integer)`**

Especifica el tipo de información solicitada: 


|   |   |
| --- | --- |
| *hCorrespondence* | Da la correspondencia entre los números de las funciones de registro y los nombres de las funciones de WLanguage. |

## Observaciones


### Obtener registros
<a name="obtener_registros_ELTPARAGRAPHE000062"></a>

- [HSetServer](../WDLang4/3044359.md) se utiliza para recuperar y modificar información sobre los registros (tamaño máximo, ruta, ...).

- [HRetrieveLog](../WDLang4/3044357.md) se utiliza para recuperar los registros del servidor (entre dos fechas) en un archivo en formato de texto.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000218"></a>


### De WINDEV 7 y WEBDEV 7 versión 149
<a name="windev_7_webdev_7_version_149_ELTPARAGRAPHE000011"></a>

A partir de ahora, los siguientes formatos están disponibles para las fechas y horas almacenadas en los artículos:


|   |   |
| --- | --- |
| **Formatos disponibles para los elementos** | **description** |
| Fecha en formato AAAAMMDD | Formato de fecha disponible en WINDEV 5.5, WINDEV 7.5 y versiones posteriores. |
| Fecha y Hora | Nuevo formato de fecha, utilizado para almacenar tanto la fecha como la hora: YYYYMMDDHHmmSSCCC (precisión: milésimas de segundo) |
| Duración | Nuevo formato, utilizado para gestionar las duraciones: +DHHMMSSCCC (precisión: milésimas de segundo) |
| Hora en formato HHMM | Formato de tiempo disponible en WINDEV 5.5, WINDEV 7.5 y versiones posteriores. |

## ¿Cómo funcionará la modificación de datos del automatic?
<a name="¿como_funcionara_modificacion_datos_del_automatic_ELTTEXTE000242"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

Conversión tabla al pasar del "formato antiguo" a un nuevo formato durante la modificación de los archivos de datos de automatic.


|   |   |   |   |   |   |   |
| --- | --- | --- | --- | --- | --- | --- |
| Nuevo formato | momento | salir con | Date/Time | Duración | Cadena / <br>Binary /<br> Memo | Numérico y moneda |
| Formato antiguo |
| momento | X | 0 | Date/Time<br>El tiempo se inicializa con el valor <br>La fecha se inicializa en 00/00/00 (fecha no válida) | Duración | Cadena | 0 |
| salir con | 0 | X | Date/Time<br>El tiempo se inicializa en 00: 00<br>La fecha se inicializa con el valor | 0 | Cadena | 0 |
| Date/Time | momento | salir con | X | 0 | Cadena | 0 |
| Duración | momento | 0 | Date/Time<br>El tiempo se inicializa con el valor <br>La fecha se inicializa en 00/00/00 (fecha no válida) | X | Cadena | 0 |
| Cadena/Binario/Memo | momento | salir con | 0 | 0 | X | Conversión de idiomas |
| Numérico y moneda |   | 0 | 0 | Duración | Cadena | X |


**Observación**: Para una red instalación, la modificación de datos de la automatic modifica tanto los archivos de datos encontrados en el servidor como los archivos de datos encontrados en los ordenadores de los usuarios.

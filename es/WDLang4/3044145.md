


## Funciones Hyper File disponibles para compatibilidad con versiones anteriores
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
A continuación se presentan las funciones de Hyper File (utilizadas en la versión 5.5) que se mantienen por compatibilidad y las funciones equivalentes en la versión 28.


|   |   |
| --- | --- |
| <a name="BinaryLoad"></a><br>BinaryLoad | Carga un archivo binario en una posición de memo 4.1 (inicializa un memo binario con el contenido del archivo).<br><br>Función equivalente: [HLinkMemo](../WDLang4/3044173.md). |
| <a name="BinarySave"></a><br>BinarySave | Guarda el contenido de una nota binaria 4.1 en un archivo.<br><br>Función equivalente: [HExtractMemo](../WDLang4/3044072.md) |
| <a name="HReadFirstNoLock"></a><br>Bloqueo HReadFirstNoLock | Lee la primera Record según la clave especificada. Si el archivo está bloqueado, la lectura de Record no está bloqueada.<br><br>Función equivalente: [HReadFirst](../WDLang4/3044051.md) con constantes de bloqueo. |
| <a name="HReadLock"></a><br>Bloqueo HReadLock | Lee y bloquea una Record según el número de Record especificado. <br><br>Función equivalente: [HRead](../WDLang4/3044047.md) con constantes de bloqueo. |
| <a name="HReadSeekLock"></a><br>Bloqueo HReadSeekLock | Lee y bloquea la primera Record cuyo valor de la clave especificada sea igual o mayor que el valor del criterio de búsqueda.<br><br>Función equivalente: [HReadSeek](../WDLang4/3044050.md) con constantes de bloqueo.<br><br>Función equivalente: [HReadSeek](../WDLang4/3044050.md) con constantes de bloqueo. |
| <a name="HReadLastNoLock"></a><br>BLOQUEO LEADLAST NoLOCK | Lee el último archivo Record según la clave pasada como parámetro. Si el archivo está bloqueado, la lectura de Record no está bloqueada.<br><br>Función equivalente: [HReadLast](../WDLang4/3044140.md) con constantes de bloqueo. |
| <a name="HReadLastLock"></a><br>Cerradura HReadLastLock | Lee y bloquea la última Record del fichero según la clave pasada como parámetro.<br><br>Función equivalente: [HReadLast](../WDLang4/3044140.md) con constantes de bloqueo. |
| <a name="HAliasExternal"></a><br>External HAlias | Carga los archivos description de Hyper File contenidos en un análisis externo a la aplicación sin cerrar el análisis actual.<br><br>Funciones equivalentes: <br><br>- [HDeclare](../WDLang4/3044203.md): Importa un archivo description de un análisis distinto al actual.<br><br>- [HDeclareExternal](../WDLang4/3044204.md): Importa un archivo description de un archivo existente.<br><br><br> |
| <a name="HAlias_55"></a><br>HAlias_55 | Crea un alias para un archivo lógico o cancela todos los alias existentes de un archivo lógico.<br><br>Función equivalente: [HAlias](../WDLang4/3044176.md). |
| <a name="HCopyRecord_55"></a><br>HCopyRecord_55 | Copia la Record actual de un fichero a la Record actual de otro fichero con la misma estructura.<br><br>Función equivalente: [HCopyRecord](../WDLang4/3044261.md) |
| <a name="HCreateView_55"></a><br>HCreateView_55 | Crea una vista en un archivo Hyper File.<br><br>Función equivalente: [HCreateView](../WDLang4/3044159.md). |
| <a name="HDescribeKey"></a><br>HDescribeKey | Describe una clave simple o compuesta en un archivo temporal en formato Hyper File. <br><br>Método equivalente: <br><br>1. Declarar una variable de tipo [Artículo description](../WDLang4/1514071.md).<br><br>2. Defina todas las propiedades del elemento con las propiedades HFSQL.<br><br>3. Utilice [HDescribeItem](../WDLang4/3044233.md) para validar la creación del artículo.<br><br><br> |
| <a name="HDescribeTrigger_55"></a><br>HDescribeTrigger_55 | Añade o modifica un activador en Hyper File. <br><br>Función equivalente: [HDescribeTrigger](../WDLang4/3044074.md) |
| <a name="HDuplicates"></a><br>HDuplicates | Comprueba la unicidad de una clave.<br><br>Método equivalente: esta función ha sido sustituida por:<br><br>- una gestión automatic de los duplicados, que permite al usuario introducir un valor diferente para el elemento que genera un error de duplicación.<br><br>- la función [HErrorDuplicates](../WDLang4/3044077.md).<br><br><br> |
| <a name="HEndLock"></a><br>HEndLock | Marca el fin del bloqueo de los archivos: todos los registros bloqueados están desbloqueados. <br><br>Funciones equivalentes: <br><br>- Utilice las funciones de lectura con las constantes adecuadas: [HRead](../WDLang4/3044047.md), [HReadFirst](../WDLang4/3044051.md), [HReadNext](../WDLang4/3044037.md), .... <br><br>- Abrir el archivo de datos en modo lectura/escritura con [HOpen](../WDLang4/3044120.md).  <br><br>- función [HUnlockFile](../WDLang4/3044160.md). <br><br><br> |
| <a name="HImportText_55"></a><br>HImportText_55 | Importa un archivo de texto en un archivo Hyper File. <br><br>Función equivalente: [HImportText](../WDLang4/3044011.md). |
| <a name="HIndex_55"></a><br>HIndex_55 | Reindexa un archivo Hyper File perteneciente al análisis actual.<br><br>Función equivalente: [HIndex](../WDLang4/3044133.md). |
| <a name="HInfoGene"></a><br>HInfoGene | Devuelve información general sobre un archivo Hyper File.<br><br>Método equivalente: esta función ha sido sustituida por:<br><br>- (por ejemplo, [HNbRec](../WDLang4/3044006.md) sustituye al H.NbRec Variable).<br><br>- [Propiedades HFSQL](../Proprietes/2512070.md) (por ejemplo, el directorio **** Property sustituye al H.Dir Variable).<br><br><br> |
| <a name="HInfoItem"></a><br>HInfoItem | Se utiliza para la información de get sobre un elemento de análisis. <br><br>Método equivalente: esta función ha sido sustituida por [Propiedades HFSQL](../Proprietes/2512070.md). |
| <a name="HIsLocked"></a><br>HIsLocked | Comprueba si la última operación de WINDEV se ha ejecutado o no. <br><br>Método equivalente: esta función ha sido sustituida por:<br><br>- una gestión de cerraduras automatic (que se puede desactivar si es necesario),<br><br>- la función [HErrorLock](../WDLang4/3044099.md).<br><br><br> |
| <a name="HReadFirstLock"></a><br>HReadFirstLock | Lee y bloquea la primera Record del archivo según la clave especificada.<br><br>Función equivalente: [HReadFirst](../WDLang4/3044051.md) con constantes de bloqueo. |
| <a name="HReadNextLock"></a><br>HReadNextLock | Lee y bloquea la Record anterior según la clave especificada.<br><br>Función equivalente: [HReadNext](../WDLang4/3044037.md) con constantes de bloqueo. |
| <a name="HReadNextNoLock"></a><br>HReadNextNoLock | Lee la siguiente Record según la clave especificada. Si el archivo está bloqueado, la lectura de Record no está bloqueada.<br><br>Función equivalente: [HReadNext](../WDLang4/3044037.md) con constantes de bloqueo. |
| <a name="HReadNoLock"></a><br>HReadNoLock | Lee el Record según el número especificado de Record. Si el archivo está bloqueado, la lectura de Record no está bloqueada.<br><br>Función equivalente: [HRead](../WDLang4/3044047.md) con constantes de bloqueo. |
| <a name="HReadPreviousLock"></a><br>HReadPreviousLock | Lee y bloquea la Record anterior según la clave especificada.<br><br>Función equivalente: [HReadPrevious](../WDLang4/3044048.md) con constantes de bloqueo. |
| <a name="HReadPreviousNoLock"></a><br>HReadPreviousNoLock | Lee el anterior Record según la clave especificada. Si el archivo está bloqueado, la lectura de Record no está bloqueada. <br><br>Función equivalente: [HReadPrevious](../WDLang4/3044048.md) con constantes de bloqueo. |
| <a name="HReadSeekNoLock"></a><br>HReadSeekNoLock | Lee la primera Record qué valor de la clave especificada es igual o mayor que el valor del criterio de búsqueda. Si el archivo está bloqueado, la lectura de Record no está bloqueada.<br><br>Función equivalente: [HReadSeek](../WDLang4/3044050.md) con constantes de bloqueo. |
| <a name="HStartLock"></a><br>HStartLock | Marca el inicio del bloqueo de lectura de los archivos. Todas las funciones de lectura "simple" bloquean automáticamente los registros que se han leído.<br><br><br>Funciones equivalentes: <br><br>- Utilice las funciones de lectura con las constantes de bloqueo adecuadas: [HRead](../WDLang4/3044047.md), [HReadFirst](../WDLang4/3044051.md), [HReadNext](../WDLang4/3044037.md), .... <br><br>- Abrir el archivo de datos en modo lectura con [HOpen](../WDLang4/3044120.md).  <br><br>- función [HLockFile](../WDLang4/3044153.md)<br><br><br> |
| <a name="HSubstName"></a><br>HSubstName | Modifica el nombre físico de un archivo de datos.<br><br>Función equivalente: [HChangeName](../WDLang4/3044141.md). |
| <a name="HViewToTable"></a><br>HViewToTable | Rellena una control Tabla de forma programada con una vista.<br><br>Función equivalente: [FileToMemoryTable](../WDLang1/3074005.md). |
| <a name="BinaryInfo"></a><br>Información binaria | Se utiliza para la información de get sobre el memo binario 4.1 (tipo, nombre, fecha, hora y tamaño del archivo fuente).<br><br>Función equivalente: [HInfoMemo](../WDLang4/3044008.md) |
| <a name="HAssign"></a><br>Signo HA | Modifica el acceso ruta a un archivo.<br><br>Función equivalente: [HChangeDir](../WDLang4/3044168.md). |




## HFree (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Transforma los registros cruzados ([HCross](../WDLang4/3044121.md)) encontrados en un archivo de datos en registros borrados. El espacio ocupado por los registros cruzados se libera y puede reutilizarse en las siguientes adiciones ([HAdd](../WDLang4/3044147.md)).

**Recordatorio**: Un Record cruzado es Inactive y desaparece del índice. Los datos que se encuentran en esta Record cruzada no se borran (a diferencia de [HDelete](../WDLang4/3044018.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all crossed records from "Customer"
HFree(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HFree([<Data file>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el registro,

- <u><u><u><u>False</u></u></u></u> si se produce un problema: el Record no se borra. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre, la función **HFree** utilizará el último archivo de datos utilizado por la última función HFSQL (que empieza por "H").  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no es adecuada para los archivos xBase.

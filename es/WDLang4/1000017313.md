


## HSaveParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un valor persistente. Este valor puede ser leído por [HLoadParameter](../WDLang4/1000017314.md).  

Observación: Los parámetros se guardan por base de datos en el archivo "user_param.fic".


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save a value from a stored procedure
// (Server code)
HSaveParameter("MyParameter", 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Guardando un parámetro en el servidor (en el código de Procedure almacenado)

`<Result> = HSaveParameter(<Parameter> , <Parameter value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la copia de seguridad, 

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a guardar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Parameter value>: (Tipo del valor a guardar)`**

Valor a ahorrar.  


<a name="SYNTAX2"></a>

### Guardar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = HSaveParameter(<Connection> , <Parameter> , <Parameter value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la copia de seguridad,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a guardar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Parameter value>: (Tipo del valor a guardar)`**

Valor a ahorrar.

## Observaciones


### Tipo de variables guardadas
<a name="tipo_variables_guardadas_ELTPARAGRAPHE000142"></a>

**HSaveParameter** se utiliza para guardar las variables simples (cadena, entero, ...).

Para guardar variables complejas (objetos, estructuras, etc.), estas variables deben ser serializadas como cadenas utilizando [Serialize](../WDLang1/3013065.md).


## HDescribeFullTextIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Describe un índice de texto completo de un archivo de datos creado mediante programación.

Tras la llamada a **HDescribeFullTextIndex**:

- La posición se puede tratar como cualquier posición de análisis.

- El artículo description ya no puede ser modificado.

- La description Variable del artículo se reinicia. Se puede realizar un nuevo índice description.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyFullTextIndex is description of FullTextIndex

// Describe the full-text index
MyFullTextIndex.MinimumLength = 4
MyFullTextIndex.Name = "MyFullTextIndex"
MyFullTextIndex.Items = "LASTNAME+FIRSTNAME"
MyFullTextIndex.AccentSensitive = True
MyFullTextIndex.CaseSensitive = False

HDescribeFullTextIndex(AFile, MyFullTextIndex)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDescribeFullTextIndex(<Data file> , <Full-text index>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos HFSQL (descrito por [HDescribeFile](../WDLang4/3044206.md)).

**`<Full-text index>: (FullTextIndex description Variable)`**

Nombre de la variable [FullTextIndex Description](../WDLang4/1000017461.md) que contiene el description del índice de texto completo.

## Observaciones


### ¿Cómo describir un índice de texto completo mediante programación?
<a name="¿como_describir_indice_texto_completo_mediante_programacion_ELTPARAGRAPHE000083"></a>

Para describir un índice de texto completo a través de la programación, debe:

1. Utilice las variables [Expediente description](../WDLang4/1514065.md), [Artículo description](../WDLang4/1514071.md), [Descripción de la relación](../WDLang4/1514039.md) y [FullTextIndex Description](../WDLang4/1000017461.md) (si es necesario).

2. Describa las características de los archivos de datos, los artículos, los enlaces y los índices de texto completo a través de las propiedades del HFSQL.

3. Definir la description de artículos con [HDescribeItem](../WDLang4/3044233.md).

4. Definir la descripción de los índices de texto completo con la función **HDescribeFullTextIndex**:

5. Definir el description del archivo de datos ([HDescribeFile](../WDLang4/3044206.md)) y los enlaces ([HDescribeLink](../WDLang4/3044191.md)).

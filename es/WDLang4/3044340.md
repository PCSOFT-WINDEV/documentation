## Detalles de la estructura del cliente
<a name="detalles_estructura_del_cliente_ELTTEXTE000194"></a>


### Variables de la estructura
<a name="variables_estructura_ELTPARAGRAPHE000066"></a>

Las siguientes variables se utilizan para gestionar los ordenadores de los Clientes:


|   |   |
| --- | --- |
| Aplicación HClient | Cadena de caracteres.<br>Nombre de la aplicación. <br>Este nombre es devuelto por [HListConnectedUser](../WDLang4/3044291.md). |
| Base de datos de clientes | Cadena de caracteres.<br>Base de datos HFSQL Client/Server. |
| Archivo HClient. | Cadena de caracteres.<br>Archivo de datos. |
| HClient.login | Cadena de caracteres<br>Nombre de usuario. |
| Ordenador.cliente | Cadena de caracteres.<br>Ordenador del usuario: Nombre de el equipo o dirección IP de el equipo. |


Las funciones que utilizan esta estructura se aplicarán a las conexiones cuyo nombre de usuario es **HClient.login** y que se realizan mediante una aplicación **HClient.Application**. Esta aplicación se ejecuta en el ordenador **HClient.Computer** y utiliza el archivo **HClient.File** de la base de datos **HClient.Database**.

Cada Variable de la estructura puede tener un valor "" (cadena vacía) que significa "cualquier valor".

## Funciones que utilizan la estructura HClient
<a name="funciones_que_utilizan_estructura_hclient_ELTTEXTE000218"></a>
Las siguientes funciones utilizan el **HClient** estructura: 



|   |   |
| --- | --- |
| [HDisconnectClient](../WDLang4/3044339.md) | Muestra un mensaje en los equipos Cliente y desconecta la aplicación. |
| [HResetClient](../WDLang4/3044341.md) | Inicializa la estructura para la gestión de los ordenadores Cliente (estructura[HClient](../WDLang4/3044340.md)). |
| [HSendMessageToClient](../WDLang4/3044346.md) | Muestra un mensaje en el equipo de los clientes. |

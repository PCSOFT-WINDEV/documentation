
## H.VisibleAutoIndexing (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la ventana de indexación debe mostrarse durante la reindexación automática de los archivos de datos HFSQL.

**Observación**: No importa si se muestra la ventana de indexación o no, la aplicación actual no se puede utilizar durante la operación de reindexación.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Valor de H.VisibleAutoIndexing
<a name="valor_hvisibleautoindexing_ELTPARAGRAPHE000022"></a>

- Por defecto, la variable **H.VisibleAutoIndexing** se establece en <u><u><u><u>True</u></u></u></u>: la ventana de indexación se mostrará durante la operación de reindexación automática.

- Si el valor de **H.VisibleAutoIndexing** es <u><u><u><u>False</u></u></u></u>, la ventana de indexación automática sera invisible.




**Atención**: Si se produce un error de bloqueo durante la operación de reindexación (cuando el archivo de datos o un registro encontrado en el archivo de datos se bloquea en lectura/escritura, por ejemplo), la reindexación automática se detiene y se muestra un error fatal (índice corrupto). Entonces, tendrás que reindexar a el archivo de datos.
<a name="NOTE0_2"></a>




### Personalización de la ventana para la indexación automática
<a name="personalizacion_ventana_para_indexacion_automatica_ELTPARAGRAPHE000041"></a>

La ventana para la indexación automática se suministra en francés y en inglés.. Esta ventana se puede personalizar. Para obtener más información, consulte [Gestión asistida de errores HFSQL](../WDLang4/3044188.md).

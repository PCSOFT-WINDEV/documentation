
## mongoDatabase (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoDatabase** permite definir las características de una base de datos de un servidor MongoDB. Estas características pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cntMDB is mongoConnection
cntMDB = MongoCreate("mongodb://db1.example.net")
FOR EACH MyDatabase OF cntMDB.Database
	Trace(MyDatabase.Name)
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables mongoDatabase
<a name="propiedades_especificas_las_variables_mongodatabase_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una base de datos MongoDB:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Colección | Array de [mongoCollection](../WDLang4/1000022411.md) | La asociación array que contiene las colecciones de la base de datos de MongoDB. |
| denominar | Cadena de caracteres | Nombre de la base de datos MongoDB. |

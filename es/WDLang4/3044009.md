


## HLogInfo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega comentarios en el registro al guardar la operación. Estos comentarios se utilizarán para todas las operaciones realizadas en el contexto HFSQL actual, hasta que se vuelva a llamar a **HLogInfo**.

**Observación**: El registro Process no está disponible cuando se accede a una base de datos a través de .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HLogInfo("Adding " + Customer.CustomerName)
HAdd(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Definiendo una comment

`<Result> = HLogInfo([<Message>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Message>: (Cadena de caracteres opcional)`**

Mensaje que se debe insertar en la información de registro (hasta 16 caracteres, la información se truncará si la cadena es más larga).. Esta información puede ser recuperada por [WDLog](../WDJournal/3516012.md). 
No se inserta ningún mensaje si no se especifica este parámetro.. 
Este mensaje se tendrá en cuenta para todos los archivos de datos para los que no se haya definido un comment específico (sintaxis 2).  


<a name="SYNTAX2"></a>

### Definir un comentario específico por cada archivo de datos con registro

`<Result> = HLogInfo(<Data file> , <Message>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos registrado con el que se relaciona el mensaje. Si este parámetro es set a "\*", se tendrán en cuenta todos los archivos de datos registrados que se encuentren en el análisis.

**`<Message>: (Cadena de caracteres)`**

Mensaje que se debe insertar en la información de registro (hasta 16 caracteres, la información se truncará si la cadena es más larga).. Esta información puede ser recuperada por [WDLog](../WDJournal/3516012.md). 
No se inserta ningún mensaje si no se especifica este parámetro..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para insertar diferentes tipos de información durante cada operación, se debe llamar a **HLogInfo** antes de cada una de ellas.

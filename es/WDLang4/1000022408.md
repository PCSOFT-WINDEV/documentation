
## MongoCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una conexión a un servidor o cluster MongoDB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000022624&name=mongocreate_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
cntMDB is mongoConnection
cntMDB = MongoCreate("mongodb://db1.example.net")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MongoCreate(<Connection string>)`
---

**`<Result>: (mongoConnection Variable)`**

[mongoConnection](../WDLang4/1000022409.md) Variable que representa la conexión a la base de datos MongoDB. Si se produce un problema, la función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.

**`<Connection string>: (Cadena de caracteres)`**

Cadena de conexión al servidor o cadena de conexión del clúster que alberga la base de datos MongoDB. Para más detalles, consulte la documentación sobre MongoDB. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función es equivalente a la siguiente sintaxis: 

```txt
<ConnectionName> is mongoConnection(<Connection string>)
```

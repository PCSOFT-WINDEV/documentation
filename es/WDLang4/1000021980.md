


## &lt;hdfsConnection Variable&gt;.CopyFrom (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia un archivo encontrado en un servidor Hadoop al ordenador local.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
MyConnection.OpenConnection()
// Copy from the server to the local computer
MyConnection.CopyFrom("/tmp/test.data", "c:\temp\test.data")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.opyFrom(<Source file> , <Destination file> [, <Progress Bar>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el copy, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable que corresponde a la conexión con el servidor Hadoop.

**`<Source file>: (Cadena de caracteres)`**

ruta absoluta del archivo encontrado en el servidor Hadoop. Esta ruta distingue entre mayúsculas y minúsculas.

**`<Destination file>: (Cadena de caracteres)`**

Ruta del archivo local correspondiente al destino (hasta 260 caracteres). Este parámetro puede corresponder a:

- un nombre de archivo. Este archivo se creará automáticamente en el directorio actual (devuelto por [fCurrentDir](../WDLang1/3036022.md)).

- un nombre de archivo con su ruta completa o relativa. Se puede utilizar una ruta de tipo UNC.




**`<Progress Bar>: (Nombre de control opcional)`**

Nombre de la control Barra de progreso utilizada para ver el progreso de la recuperación de archivos.

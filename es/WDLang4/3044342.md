
## HRplPass (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece las contraseñas que protegen los archivos de la replicación universal. Los archivos se cifran con esta contraseña.

**Observación: Esta función solo se puede utilizar durante la replicación universal.**



Notas: La replicación no está disponible al acceder a una base de datos mediante un .




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the password
HRplPass("A")
// Create replication file
HCreateMovableReplica("", "")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HRplPass([<Password>])`
---

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña asociada a los archivos de réplica. Esta contraseña permite cifrar los archivos. Esta contraseña debe introducirse para ejecutar los archivos de réplica. 
Si no se especifica este parámetro o si corresponde a una cadena vacía (""), no se define ninguna contraseña para los archivos de réplica. El tamaño de la contraseña no está limitado.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las aplicaciones WINDEV, WEBDEV o WINDEV Mobile en formato 10 no pueden leer réplicas cifradas.

- Para cancelar el modo cifrado, utilice una contraseña vacía ("").

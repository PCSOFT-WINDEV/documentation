
## HListConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de conexiones descritas actualmente en la aplicación. Tiene la posibilidad de listar las conexiones descritas en el editor de análisis así como las descritas mediante programación (con [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md)).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the list of connections
CntList is string = HListConnection(hLstDetail + hLstAll + HLstSorted)

// Insert the list into COMBO_Connection
ListAdd(COMBO_Connection, CntList)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListConnection([<Options> [, <Separator of connections> [, <Separator of information>]]])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de conexiones definidas. Las diferentes conexiones están separadas por el &lt;Separador de conexiones&gt; (CR - Carriage Return - por Default).


|   |   |
| --- | --- |
| *hLstDetail* no especificado | &lt;Nombre de la conexión 1&gt; + &lt;Separador de conexiones&gt; + &lt;Nombre de la conexión 2&gt; + &lt;Separador de conexiones&gt; +.... |
| Se especifica *hLstDetail* | &lt;Nombre de la conexión 1&gt; + &lt;Separador de información&gt; + &lt;Usuario&gt; + &lt;Separador de información&gt; + &lt;Fuente de datos&gt; + &lt;Separador de información&gt; + &lt;Nombre del proveedor&gt; + &lt;Separador de información&gt; + &lt;Información opcional&gt; + &lt;Separador de conexiones&gt; +... |



**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Configura el tipo de lista a realizar:


|   |   |
| --- | --- |
| *hLstAll* | Enumera todas las conexiones (conexiones de análisis y conexiones definidas dinámicamente por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md)). |
| *hLstDetail* | Información detallada. |
| *hLstNormal*<br>(valor predeterminado) | Lista de conexiones, no clasificadas, sin las conexiones descritas por programación. |
| *hLstOpened* | Enumera sólo las conexiones abiertas. |
| *hLstSorted* | Clasifica la lista en orden ascendente. |



**`<Separator of connections>: (Cadena de caracteres opcional)`**

Separador utilizado para separar las diferentes conexiones. Este separador corresponde a un Carriage Return (CR) de Default.

**`<Separator of information>: (Cadena de caracteres opcional)`**

Separador utilizado para separar la información devuelta sobre las conexiones. Este separador corresponde a una tabulación - TAB - de Default.

## Observaciones




### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000117"></a>

&lt;Resultado&gt; puede asignarse directamente a un control List Box o Combo Box mediante la función [ListAdd](../WDLang1/3049004.md). Si el &lt;Separador de conexiones&gt; corresponde a CR, cada fila contendrá el nombre completo de una de las conexiones encontradas.

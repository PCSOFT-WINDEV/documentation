


## &lt;Fuente&gt;.RecordDate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha y hora de la última operación de escritura realizada en una Record en un archivo de datos HFSQL (Clásico o Cliente/Servidor).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the date and time of the last modification
// on the current record
Info(Customer.RecordDate())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.RecordDate([<Record number>])`
---

**`<Result>: (Cadena de caracteres)`**



- Fecha y hora de la última operación de escritura en formato AAAAMMDDHHmmSS. La fecha y la hora se expresan en formato UTC (Universal Time Coordinated). 

- Cadena vacía ("") si se produce un error o si el archivo no admite esta función.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Record number>: (Entero de 8 bytes opcional)`**

El número de la Record utilizada. 
Si no se especifica este parámetro, **&lt;Fuente&gt;.RecordDate** manipular la actual Record.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ¿Qué fecha y hora se tendrán en cuenta?
<a name="¿que_fecha_hora_tendran_cuenta_ELTPARAGRAPHE000231"></a>

La fecha y la hora (con una precisión de un segundo) se toman a partir del momento en que [&lt;Fuente&gt;.Add](../WDLang4/1000024187.md), [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md), [&lt;Fuente&gt;.Cross](../WDLang4/1000025051.md), [&lt;Fuente&gt;.Delete](../WDLang4/1000025071.md) o cualquier funcionalidad escribió en los archivos de datos (modificación de archivos de datos por parte de automatic, funciones de importación, reindexación, implementación de reglas de integridad, etc.).
<a name="NOTE0_2"></a>

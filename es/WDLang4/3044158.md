


## HActivateFilter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa el filtro que se creó previamente para el archivo de datos (vista o consulta) especificado. Este filtro se creó con la función [HFilter](../WDLang4/3044100.md) y desactivó con la función [HDeactivateFilter](../WDLang4/3044085.md). Si no se declaró ningún filtro, la función **HActivateFilter** no tiene ningún efecto.

Esta función se puede utilizar con archivos de datos, vistas HFSQL o consultas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare the filter: select customers
// between 20 and 60 years old
HFilter(Customer, Age_Customer, 20, 60)
...
HDeactivateFilter(Customer) // Disabled filter
...
// Browse the data file with a filter on age
// Process the customers who are between 20 and 60 years old
// (ProcessCustomer2060 procedure)
HActivateFilter(Customer)
HReadFirst(Customer, Age_Customer)
WHILE HFound(Customer) = True
	ProcessCustomer2060()
	HReadNext(Customer, Age_Customer)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HActivateFilter([<Data file>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se activó el filtro,

- <u><u><u><u>False</u></u></u></u> si se produce un error (archivo o elemento inexistente).




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos, vista HFSQL o consulta utilizada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Activar y desactivar un filtro
<a name="activar_desactivar_filtro_ELTPARAGRAPHE000097"></a>

El filtro se activa una vez que ha sido declarado (función [HFilter](../WDLang4/3044100.md)). El filtro se puede desactivar con la función [HDeactivateFilter](../WDLang4/3044085.md).
<a name="NOTE0_2"></a>


### Número de filtros activos posibles
<a name="numero_filtros_activos_posibles_ELTPARAGRAPHE000112"></a>

Solo se puede activar un filtro a la vez para una archivo de datos específico.
<a name="NOTE0_3"></a>

### Modo de funcionamiento en OLE DB
<a name="modo_funcionamiento_ole_ELTPARAGRAPHE000123"></a>

Si se utiliza la función **HActivateFilter** para navegar por un archivo de datos, la navegación se detiene. Para tener en cuenta el filtro, se debe reiniciar la navegación con las funciones [HReadFirst](../WDLang4/3044051.md)/[HFirst](../WDLang4/3044113.md) o [HReadLast](../WDLang4/3044140.md)/[HLast](../WDLang4/3044101.md).

La posición en curso y todas las posiciones guardadas se pierden. La posición se restablecerá de acuerdo con el valor del mejor elemento.

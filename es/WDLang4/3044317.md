


## HStopServer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene un servidor HFSQL. A continuación, el servidor puede ser reiniciado por [HStartServer](../WDLang4/3044318.md).

**Observación**: Para realizar esta acción, el usuario debe tener derechos específicos en el servidor.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", ...
	"MyDatabase", hAccessHFClientServer, HORead)
// Open the connection
HOpenConnection("MyConnection")
...
HStopServer("MyConnection")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HStopServer(<Connection>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se detuvo el servidor,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que debe detenerse el servidor. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Detener el servidor
<a name="detener_servidor_ELTPARAGRAPHE000083"></a>

Si las aplicaciones Cliente se utilizan actualmente cuando se detiene el servidor, aparecerá un mensaje de error ("Servidor no disponible") durante la siguiente acción realizada por la aplicación cliente en el servidor.

**Observación**: El servidor no se puede reiniciar inmediatamente una vez que se ha detenido.. El sistema operativo no cierra inmediatamente las conexiones ya establecidas entre los ordenadores cliente y el servidor. Simplemente espere unos minutos (dependiendo del tiempo de espera del sistema operativo) antes de reiniciar el servidor.
<a name="NOTE0_2"></a>


### Administrar permisos
<a name="administrar_permisos_ELTPARAGRAPHE000094"></a>

- Para detener el servidor, el usuario o grupo de usuarios debe tener derechos específicos (constante *hRightsStopServer*).

- Para saber si un usuario o un grupo de usuarios tiene suficientes derechos, utilice [HInfoServerRights](../WDLang4/3044322.md).

- Para modificar los derechos concedidos a un usuario o grupo de usuarios en el servidor, utilice [HModifyServerRights](../WDLang4/3044328.md).

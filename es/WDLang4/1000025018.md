


## &lt;Fuente&gt;.ListSynonym (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de sinónimos utilizados por un índice de texto completo. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves all the synonym words
let sAllSynonyms = DataFile.ListSynonym(MyFullTextIndex)
// For each group of synonym words
FOR EACH STRING sSynonymList OF sAllSynonyms SEPARATED BY CR
	let sString = "The following words are synonyms: " 	
	// Extracts each synonym
	FOR EACH STRING sSynonym OF sSynonymList SEPARATED BY TAB
		// Adds the word to the string
		sString += sSynonym + " "
	END
	Trace(sString)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Source>.ListSynonym(<Full-text index> [, <Path of the WDD file> [, <Password of the WDD file>]])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de sinónimos definidos para el índice de texto completo especificado en el archivo de datos especificado. Esta lista tiene el siguiente formato: 

```txt
<Word 1> + TAB + <Synonym 1A> + TAB + <Synonym 1B> + CR + ...
<Word 2> + TAB + <Synonym 2> + ... + CR + ...
<Word N> + TAB + <Synonym N>
```
En esta lista: 

- Los sinónimos están separados por una tabulación (TAB). 

- Los diferentes grupos de sinónimos están separados por un Carriage Return (CR). 


Observaciones: 

- Las palabras pueden escribirse en mayúsculas si el índice de texto completo no es sensible al mayúsculas o sin caracteres acentuados si el índice de texto completo no es sensible a los caracteres acentuados.

- Las palabras cuya longitud sea inferior a la longitud mínima especificada para el índice de texto completo pueden no aparecer en esta lista.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos que contiene el índice de texto completo.

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado.

**`<Path of the WDD file>: (Cadena de caracteres opcional)`**

ruta completo de análisis (archivo .WDD) donde se describe el archivo de datos. Si este parámetro no se especifica o si es igual a una cadena vacía (""), se busca el archivo de datos en el análisis actual.

**`<Password of the WDD file>: (Cadena de caracteres opcional)`**

Contraseña definida para el análisis. Si no se define ninguna contraseña, este parámetro debe ser igual a una cadena vacía ("").

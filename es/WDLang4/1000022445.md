


## MongoDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todos los documentos (correspondientes al filtro especificado) de una colección de una base de datos MongoDB.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFilter is string
sFilter = "{ ""name"": ""Andy"" }"
MongoDeleteAll(CollectMDB, sFilter)
```
<a name="Example2"></a>

```wl
// Delete all the contacts whose "FirstName" member is "Robert"

// Search filter
sFilter is string = "{ ""FirstName"": ""Robert"" } "

MongoDeleteAll(gclCollection, sFilter)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MongoDeleteAll(<Collection> , <Filter>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Collection>: (Variable de tipo mongoCollection)`**

Nombre de la [mongoCollection](../WDLang4/1000022411.md) Variable que representa la colección de MongoDB en la que se realiza el borrado.

**`<Filter>: (Tipo correspondiente al filtro utilizado)`**

description de filtros en formato MongoDB que se aplicará para seleccionar los documentos a eliminar. Este filtro corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- una variante.

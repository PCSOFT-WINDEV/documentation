


## HNoDatabaseAccess (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deniega las nuevas conexiones a una o a todas las bases de datos. 

Observaciones: 

- Las operaciones de lectura y escritura funcionan como de costumbre. Sólo se rechazan las nuevas conexiones a la base de datos.. 

- Para volver a habilitar las conexiones, utilice [HEndNoDatabaseAccess](../WDLang4/3044287.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "CustomerDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
IF HNoDatabaseAccess("MyConnection", "CustomerDatabase") THEN
	Info("Unable to connect to the database")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HNoDatabaseAccess(<Connection> [, <Database>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el acceso (es decir, la conexión) a la base de datos especificada (o a todas las bases de datos a las que accede la conexión) es denegado,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión que contiene la base de datos a la que se debe denegar el acceso. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Database>: (Cadena de caracteres opcional)`**

Nombre de la base de datos cuyo acceso debe denegarse.
Si este parámetro es set a "\*", se denegará el acceso a todas las bases de datos.
Si no se especifica este parámetro, se denegará el acceso a las bases de datos utilizadas por &lt;Connection&gt;..

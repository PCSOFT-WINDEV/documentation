


## &lt;Fuente&gt;.EndNoModif (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desbloquea un archivo de datos que fue bloqueada por el mismo programa con [&lt;Fuente&gt;.NoModif](../WDLang4/1000024999.md).

**Atención**: No desbloquea los bloqueos realizados por otros programas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Invoice.NoModif()
// Process
Invoice.EndNoModif()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.EndNoMotif()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si no se ha encontrado ningún problema,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado.

<a name="NOTE0_1"></a>

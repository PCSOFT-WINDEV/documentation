


## MongoDeleteOne (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una Document de una colección en una base de datos de MongoDB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCollection is mongoCollection
let sFilter = "{ ""name"": ""Andy"" }"
let sSort = "{ ""rating"": 1 }"
let d = MongoDeleteOne(MyCollection, sFilter, sSort)
```
<a name="Example2"></a>

```wl
// Delete the first contact whose "FirstName" member is set to "Joey"
// Search filter
sFilter is string = "{ ""FirstName"": ""Joey"" }"
MongoDeleteOne(gclCollection, sFilter)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MongoDeleteOne(<Collection> , <Filter> [, <Sort>])`
---

**`<Result>: (Variant)`**

Borrado Document. 

Si se produce un problema, la función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.

**`<Collection>: (Variable de tipo mongoCollection)`**

Nombre de la [mongoCollection](../WDLang4/1000022411.md) Variable que representa la colección de MongoDB en la que se realiza el borrado.

**`<Filter>: (Tipo correspondiente al filtro utilizado)`**

description de filtros en formato MongoDB que se aplicará para seleccionar la Document a borrar. Este filtro corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- una variante. 




**`<Sort>: (Tipo opcional correspondiente al formato utilizado)`**

Opciones de clasificación utilizadas para definir la primera Document que se borrará cuando varias Document correspondan al filtro. Corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- una variante.

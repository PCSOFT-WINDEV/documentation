


## Tiempo de espera HCS (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La variable **HCS.Timeout** permite modificar el tiempo de espera de la conexión:

- para las conexiones establecidas por [HOpenConnection](../WDLang4/3044107.md).

- para las aberturas de conexión de automatic.




**Observación**: El **HCS.Timeout** Variable se ignora para los accesos al servidor que no sean la conexión. Una solicitud cuya ejecución sea más larga que el valor especificado en el **HCS.Timeout** Variable no se interrumpiría, por ejemplo...

Este tiempo de espera debe ser especificado en la aplicación cliente, antes de abrir una conexión en el servidor.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HCS.Timeout = Intranet
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HCS.Timeout = <Value>`
---

**`<Value>: (Constante de entero, entero o duración)`**

Valor del tiempo de espera de la conexión. Corresponde a un tiempo de espera en milisegundos (mayor de 500) o a una de las siguientes constantes preestablecidas: 


|   |   |
| --- | --- |
| *INTERNET* | El tiempo de espera utilizado para la conexión: Valor adecuado para INTERNET (largo tiempo de respuesta) |
| *INTRANET* | El tiempo de espera utilizado para la conexión: Valor adecuado para INTRANET (tiempo de respuesta corto) |


Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, "1s" o "10ms").

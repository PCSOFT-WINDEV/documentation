


## HCheckServerUpdate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si hay una versión más reciente del servidor HFSQL que pueda actualizarse automáticamente. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

let sNewVersion = HCheckServerUpdate(MyConnection)
IF sNewVersion<>"" THEN Info(sNewVersion)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HCheckServerUpdate(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Versión de la actualización disponible en el servidor HFSQL. 

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Connection](../WDLang4/1514073.md).







## Observaciones
El servidor de actualización (que contiene el paquete para actualizar el servidor HFSQL) puede ser definido con la función [HManageServerUpdate](../WDLang4/1000022635.md).

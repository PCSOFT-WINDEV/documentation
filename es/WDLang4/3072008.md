
## SQLDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la conexión actual y libera la memoria utilizada por la conexión.. Se debe llamar sistemáticamente a la función **SQLDisconnect** para cerrar la conexión, incluso si falla la conexión.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)![HFSQL Classic](https://doc.pcsoft.fr/ext/images/us/HF.png)![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png)![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) 
```wl
// Example associated with the SALESMGT analysis
// TABLE_QryTable is a Table control populated programmatically
ResConnect is int
ResExec is boolean
MyDatabase is string = "C:\SALESMGT.WDD"
ResConnect = SQLConnect(MyDatabase, "", "")
IF ResConnect <> 0 THEN
	ResExec = SQLExec("SELECT NAME, CITY FROM POP", "QRY1")
	IF ResExec = True THEN
		// Retrieve the data in the table
		SQLTable("QRY1", TABLE_QryTable)
	ELSE
		// Error while running the query 
		// Retrieve information about the query run
		SQLInfo("QRY1")
		Error(SQL.MesError)
	END
	// Free the query in any case
	SQLClose("QRY1")
ELSE
	 // Connection error
	 Error("The connection to the database" + MyDatabase + " failed")
END
// Free the query in any case
SQLDisconnect()
```

### Conexión de análisis HFSQL
<a name="conexion_analisis_hfsql_ELTPARAGRAPHE000087"></a>

En una conexión con un análisis HFSQL, **SQLDisconnect** cierra el análisis (igual que [HCloseAnalysis](../WDLang4/3044054.md)). Para seguir utilizando los archivos de análisis, éstos deben ser abiertos por [HOpenAnalysis](../WDLang4/3044106.md).


<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000104"></a>

- Se puede establecer una única conexión en un momento dado.

- La memoria asignada durante la conexión (ejecución de "SQLFreeConnect") se libera automáticamente.

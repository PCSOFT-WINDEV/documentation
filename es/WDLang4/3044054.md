
## HCloseAnalysis( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el análisis actual. Todos los archivos de fecha HFSQL, vistas y consultas abiertas actualmente se cierran y se liberan.. Después de llamar a la función **HCloseAnalysis**, no hay ningún análisis en curso.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)![HFSQL Classic](https://doc.pcsoft.fr/ext/images/us/HF.png) 
```wl
// Opens an analysis
HOpenAnalysis("C:\SalesMgt\SalesMgt.WDD")
// Processes 
...
// Closes the analysis currently used by the application
HCloseAnalysis()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HCloseAnalysis()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cerró el análisis,

- <u><u><u><u>False</u></u></u></u> si se produce un problema: el análisis no está cerrado. La función [HError](../WDLang4/3044088.md) permite identificar el error.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se puede abrir un solo análisis en un momento dado.

- Solo un análisis abierto con la función [HOpenAnalysis](../WDLang4/3044106.md) puede cerrarse con la función **HCloseAnalysis**.

- Si no hay ningún análisis en curso, la función **HCloseAnalysis** no tiene ningún efecto.

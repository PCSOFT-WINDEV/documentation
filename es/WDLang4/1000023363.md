
## MemcacheGet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera los valores de las claves almacenadas en un servidor Memcached.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Results is array of memcachedResult = MemcachedGet(memConnection, ["key1", "key2"])
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperar el valor de una llave

`<Result> = MemcacheGet(<Connection> , <Key>)`
---

**`<Result>: (memcachedResult Variable)`**

[memcachedResult](../WDLang4/1000023359.md) variable que corresponde a la clave deseada. Si no se encuentra la clave, las propiedades **Key** y **Value** del [memcachedResult](../WDLang4/1000023359.md) Variable estarán vacías. 

**`<Connection>: (memcachedConnection Variable)`**

Nombre del [memcachedConnection](../WDLang4/1000023339.md) Variable que corresponde a la conexión con Memcached. 

**`<Key>: (Cadena de caracteres)`**

Clave para encontrar.


<a name="SYNTAX2"></a>

### Recuperando los valores de una set de llaves

`<Result> = MemcacheGet(<Connection> , <Keys>)`
---

**`<Result>: (Array de memcachedResult)`**

Array de las variables de [memcachedResult](../WDLang4/1000023359.md) que contienen las claves encontradas.

**`<Connection>: (memcachedConnection Variable)`**

Nombre del [memcachedConnection](../WDLang4/1000023339.md) Variable que corresponde a la conexión con Memcached. 

**`<Keys>: (Array de cadenas de caracteres)`**

Nombre de la Array Variable que contiene las claves deseadas.




## HDeleteQuery (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una consulta (utilizada por procedimientos almacenados) de un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the "ManageEmail" query from the server
HDeleteQuery("MyConnection", ManageEmail)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDeleteQuery(<Connection> , <Query to delete>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la consulta,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL que contiene la consulta a eliminar. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Query to delete>: (Cadena de caracteres)`**

Nombre de la consulta que se debe eliminar del servidor.

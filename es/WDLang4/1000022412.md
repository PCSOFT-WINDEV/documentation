
## mongoDatabaseInfo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoDatabaseInfo** permite conocer las características avanzadas de la información de una base de datos MongoDB. Las características de esta información son devueltas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info is mongoDatabaseInfo 
Info = MongoInfo(DatabaseMDB)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables mongoDatabaseInfo
<a name="propiedades_especificas_las_variables_mongodatabaseinfo_ELTPARAGRAPHE000040"></a>

Las siguientes propiedades pueden ser utilizadas para manipular la información sobre una base de datos MongoDB:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| DataSize | Integro | Tamaño total (en bytes) de los datos encontrados en la base de datos.<br><br>Observaciones: <br><br>- El acolchado está incluido en este total. <br><br>- El factor de escala afecta a este valor. <br><br>- El tamaño no se reduce cuando se reduce un Document o cuando se borran Document s.<br><br><br>**Esta propiedad es de solo lectura**. |
| denominar | Cadena de caracteres | Nombre de la base de datos.<br><br>**Esta propiedad es de solo lectura**. |
| Espacio de nombresTamaño | Integro | Tamaño total de los ficheros namespace (ficheros con .ns extension) de la base de datos.<br><br>**Esta propiedad es de solo lectura**. |
| FileSize | Integro | Tamaño total (en bytes) de los archivos de datos de la base de datos. <br><br>Observaciones: <br><br>- El espacio preasignado se incluye en este total. <br><br>- El factor de relleno se incluye en este total.<br><br><br>**Esta propiedad es de solo lectura**. |
| NbCollections | Integro | Número de colecciones en la base de datos.<br><br>**Esta propiedad es de solo lectura**. |
| NbDocuments | Integro | Número de documentos en la base de datos (entre todas las colecciones).<br><br>**Esta propiedad es de solo lectura**. |
| NbExtents | Integro | Número de extensiones en la base de datos.<br><br>**Esta propiedad es de solo lectura**. |
| NbIndexes | Integro | Número de índices.<br><br>**Esta propiedad es de solo lectura**. |
| Tamaño de almacenamiento | Integro | Tamaño total (en bytes) del espacio utilizado para almacenar datos.<br><br>**Esta propiedad es de solo lectura**. |
| Tamaño del índice | Integro | Tamaño total (en bytes) de los índices de la base de datos.<br><br>**Esta propiedad es de solo lectura**. |
| Tamaño promedio del documento | Integro | Tamaño medio (en bytes) de los documentos. Este tamaño corresponde al tamaño de la base de datos dividido por el número de documentos. <br><br>**Esta propiedad es de solo lectura**. |
| Versión de DataFile | Cadena de caracteres | Versión del formato del disco de almacenamiento disque (sólo para mmapv1). El valor tiene el siguiente formato: **Mayor.menor**.<br><br>**Esta propiedad es de solo lectura**. |

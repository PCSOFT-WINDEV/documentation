


## &lt;Variable Connection&gt;.InfoFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las características de un archivo de datos HFSQL Classic o Client/Server.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="3044311_Example2"></a>

```wl
// Get the size of a data file by using a connection 

// Connection parameters
CntTest is Connection
CntTest.Provider = hAccessHFClientServer
CntTest.User = "admin"
CntTest.Password = "pass_admin"
CntTest.Server = "server:4930"
CntTest.Database = "TestDatabase"

Trace("Total size of Customer file: " + ...
	CntTest.InfoFile("FileName.FIC", hTotalSize))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

### Obtener información sobre un archivo de datos HFSQL Client/Server

`<Result> = <Connection>.FileInfo(<Data file> , <Information>)`
---

**`<Result>: (Tipo de información solicitada)`**



- Información solicitada,

- 0 si la conexión existe pero no se encuentra el archivo de datos en la unidad.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Data file>: (Cadena de caracteres)`**

Nombre físico del archivo de datos para analizar.

**`<Information>: (Constante de tipo Integer)`**

Solicitó información sobre el archivo de datos:


|   |   |
| --- | --- |
| *hFicSize* | Tamaño del archivo de datos en bytes (archivo "Fic"). |
| *hFtxSize* | Tamaño del archivo de índice de texto completo en bytes (archivo".Ftx"). |
| *hMmoSize* | Tamaño del archivo memo en bytes (archivo".Mmo"). |
| *hNdxSize* | Tamaño del archivo índice en bytes (archivo".Ndx"). |
| *hTotalSize* | Tamaño total del archivo de datos en bytes (tamaño acumulado del archivo de datos, archivo de notas y archivo de índices). |

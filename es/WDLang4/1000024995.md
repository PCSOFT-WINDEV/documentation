


## &lt;Fuente&gt;.InfoLock (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el bloqueo de un archivo de datos, un registro o todos los registros de un archivo de datos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LockList is string
// Lists all the locks of the Customer data file
LockList = Customer.InfoLock()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Información de bloqueo de un archivo de datos identificado por su nombre

`<Result> = <Source>.InfoLock([<Record number>])`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena vacía ("") si no hay ningún bloqueo o si el archivo de datos utilizado no es un archivo de datos HFSQL Client/Server.

- Información sobre la cerradura.  Esta información está separada por caracteres TAB y tiene el siguiente formato:
	
	```txt
	<Physical name of the data file relative to the database> + TAB + 
	<Record number> + TAB + 
	<User> + TAB + <Name of client computer> + TAB + 
	<IP address of client computer> + TAB + <Name of client application> + TAB + 
	<Type of lock> + TAB + <Date of lock implementation> + TAB + 
	<Number of seconds since start of lock> + CR
	```


	- &lt;número de Record&gt; es set a -1 si la cerradura era set en el archivo de datos.

	- El parámetro &lt;Tipo de bloqueo&gt; puede corresponder a:

		- W: Bloqueo de escritura.

		- RW: Bloqueo de lectura/escritura.

		- W-T: Bloquear en modo de escritura set por una transacción.

		- RW-T: Bloquear la lectura/escritura de set por una transacción.




- El parámetro &lt;Fecha de inicio del bloqueo&gt; tiene el siguiente formato: AAAAMMDDHHmmSS. Esta fecha está en formato UTC (Universal Time Coordinated).




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Record number>: (Entero opcional)`**

Número de la Record para la que se solicita la información de la cerradura. 
Si no se especifica este parámetro, la función **&lt;Fuente&gt;.InfoLock** lista todos los bloqueos del archivo de datos.




## HDeleteLink (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una regla de integridad entre dos archivos de datos de un servidor. La regla de integridad a eliminar está definida por una [Descripción de la relación](../WDLang4/1514039.md) Variable.

La eliminación será efectiva para la aplicación que la realizó una vez que la conexión se haya cerrado y reabierto.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
MyLink is Link Description
// Describe the "LIVES" link
MyLink.SourceFile = ".\city.fic"
MyLink.LinkedFile = ".\customer.fic"
MyLink.SourceKey = "CITYNAME"
MyLink.LinkedKey = "CUSTCITY"

// Default link (0-1, 0-1)
HDeleteLink(MyConnection, MyLink)
```

```wl
// Delete all links from a database 
// found on an HFSQL Client/Server server 
// independently of the current analysis
CntHFSQL is Connection
CntHFSQL.Provider = hAccessHFClientServer
CntHFSQL.User = "admin"
CntHFSQL.Password = ""
CntHFSQL.Server = "HFSQLServer:4900"
CntHFSQL.Database = "CRM"
HOpenConnection(CntHFSQL)

sListLinks is string = HListLink("*", CntHFSQL)
sALinkInfo is string
ALink is Link Description

FOR EACH STRING sALinkInfo OF sListLinks SEPARATED BY CR
	IF sALinkInfo <> "" THEN
		ALink.Name = ExtractString(sALinkInfo, 1, TAB)
		ALink.SourceFile = Replace(ExtractString(sALinkInfo, 2, TAB), ...
			CntHFSQL.Database, ".", IgnoreCase)
		ALink.SourceKey = ExtractString(sALinkInfo, 3, TAB)
		ALink.LinkedFile = Replace(ExtractString(sALinkInfo, 5, TAB), ...
			CntHFSQL.Database, ".", IgnoreCase)
		ALink.LinkedKey = ExtractString(sALinkInfo, 6, TAB)
		// Delete the link from the HFSQL Client/Server server
		Trace("Deletion >> " + ALink.Name + " >> " + ...
			HDeleteLink(CntHFSQL, ALink))		
	END
END
```

## Sintaxis

`<Result> = HDeleteLink(<Source connection> [, <Linked connection>] , <Link description>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Source connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a la que pertenece el fichero fuente del enlace. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Linked connection>: (Cadena de caracteres opcional o variable de tipo Connection)`**

Conexión a la que pertenece el fichero relacionado del enlace. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).


Si no se especifica este parámetro, ambos archivos de datos pertenecerán a la misma conexión (&lt;Source connection&gt;).

**`<Link description>: (Variable de tipo Link Description)`**

Nombre de la [Descripción de la relación](../WDLang4/1514039.md) Variable que contiene las características del enlace a eliminar. El enlace para borrar está definido por el archivo de datos de la fuente, el ítem de la fuente y el ítem relacionado.

## Observaciones


### Descripción de la relación
<a name="descripcion_relacion_ELTPARAGRAPHE000133"></a>

- Los nombres de los archivos de datos en el enlace description (propiedades [SourceFile](../Proprietes/2512034.md) y [LinkedFile](../Proprietes/2512031.md)) deben corresponder al nombre físico de los archivos de datos, en relación con la base de datos correspondiente. La base de datos de cada archivo de datos es devuelta por los parámetros &lt;Conexión de la fuente&gt; y &lt;Conexión enlazada&gt;.

- Para borrar un enlace, no es necesario describir todas las características del mismo.. Sólo se requieren las siguientes propiedades:
	


|   |   |
| --- | --- |
| [LinkedKey](../Proprietes/2512007.md) | Elemento del fichero de datos relacionado utilizado en el enlace. |
| [SourceKey](../Proprietes/2512003.md) | Elemento del archivo de datos de origen utilizado en el enlace. |
| [LinkedFile](../Proprietes/2512031.md) | Nombre del archivo de datos de relacionado utilizado en el enlace. |
| [SourceFile](../Proprietes/2512034.md) | Nombre del archivo de datos de origen utilizado en el enlace. |
| [denominar](../Proprietes/2510082.md) | Nombre del enlace. |

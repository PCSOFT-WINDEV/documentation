
## HRplManageItem( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica las opciones de replicación para un elemento.: el artículo puede ser replicado o no.  




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HRplManageItem(<File name> , <Item name> , <Replication mode>)`
---

**`<File name>: (Cadena de caracteres)`**

Nombre del archivo de datos para configurar.

**`<Item name>: (Cadena de caracteres)`**

Nombre del elemento de archivo a configurar.

**`<Replication mode>: (Constante de tipo Integer)`**

Constante que indica el modo de replicación del artículo:


|   |   |
| --- | --- |
| *rplReplicatedItem* | La posición se reproduce. |
| *rplUnreplicatedItem* | La posición no se reproduce. |

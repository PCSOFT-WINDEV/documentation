


## HStatNbRecRange (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve una estimación del número de entradas para un elemento clave en un intervalo de valores determinado. Esta estimación se deduce de la información obtenida durante el último cálculo estadístico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about the CustName key from the Customer data file
ResStat = HStatNbRecRange(Customer, CustName, "Clark", "Smith")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HStatNbRecRange([<Data file> [, <Key item>, ]] <Lower bound> , <Upper bound>)`
---

**`<Result>: (Integro)`**

Número de entradas encontradas para el artículo especificado en el interval solicitado. 
Corresponde a la constante *hInvalidRecNum* si:

- las estadísticas nunca se han calculado para la posición especificada.

- las estadísticas no están soportadas por el tipo de posición especificado. HFSQL permite realizar cálculos estadísticos sólo en posiciones clave.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre, la función **HStatNbRecRange** utilizará el último archivo de datos utilizado por la última función HFSQL (que empieza por "H").

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre del elemento clave utilizado para calcular las estadísticas. Si no se especifica este nombre, **HStatNbRecRange** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").

**`<Lower bound>: (Tipo de campo clave)`**

El valor más bajo de interval.

**`<Upper bound>: (Tipo de campo clave)`**

Valor superior de interval.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estadísticas sobre claves compuestas
<a name="estadisticas_sobre_claves_compuestas_ELTPARAGRAPHE000105"></a>La siguiente sintaxis permite calcular estadísticas sobre una clave compuesta: 

```txt
HStatNbRecRange(<Data file>, <Name of composite key>), 
	[<Lower bound of first element of composite key>, 
			<Lower bound of second element of composite key>], 
	[<Upper bound of first element of composite key>, 
			<Upper bound of second element of composite key>, ...])
```

Ejemplo:

```wl
HStatNbRecRange(Customer, LastName_FirstName, ["Black", "Alicia"], ["Morgan","Vince"])
```

<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000128"></a>

- Las estadísticas pueden ser actualizadas por [HIndex](../WDLang4/3044133.md) y [HStatCalculate](../WDLang4/3044134.md).

- En los archivos de datos de HFSQL Classic, los cálculos estadísticos sólo pueden realizarse en las partidas clave.

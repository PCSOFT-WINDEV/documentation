


## HLoadParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee un parámetro que fue guardado de un Procedure almacenado por [HSaveParameter](../WDLang4/1000017313.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HLoadParameter("MyParameter")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cargando un parámetro en el servidor (en el código de Procedure almacenado)

`<Result> = HLoadParameter(<Parameter> [, <Default value>])`
---

**`<Result>: (Variant)`**

Valor del parámetro si existe, &lt;Valor de Default&gt; si el parámetro no existe.

**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a restaurar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Default value>: (Variante opcional)`**

Valor de Default devuelto si no se encuentra el parámetro. Si no se especifica este parámetro, el valor de Default corresponde a una cadena vacía ("").  


<a name="SYNTAX2"></a>

### Cargar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = HLoadParameter(<Connection> , <Parameter> , <Default value>)`
---

**`<Result>: (Variant)`**

Valor del parámetro si existe, &lt;Valor de Default&gt; si el parámetro no existe.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a restaurar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Default value>: (Variant)`**

Valor de Default devuelto si no se encuentra el parámetro. Si no se especifica este parámetro, el valor de Default corresponde a una cadena vacía ("").

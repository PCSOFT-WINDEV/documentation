
## HDBDescribeField( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Describe mediante programación cada elemento que se encuentra en la estructura de un archivo xBase descrito por [HDBDescribeFile](../WDLang4/3044172.md).

**Observación**: los archivos descritos por la programación son archivos en formato dBase3.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDBDescribeFile("CUSTOMER", "DA", "C:\FILE\CUSTOMER.DBF")
HDBDescribeField("LASTNAME,C,20")
HDBDescribeField("FIRSTNAME,C,20")
HDBDescribeField("MARRIED,L")
HDBDescribeField("DOB,D")
HDBDescribeField("INFO,M")
HDBDescribeIndex("C:\FILE\LASTNFILE.NDX", "LASTNAME")
HDBDescribeIndex("C:\FILE\LNFNFILE.NDX", "FULLNAME,C,40", "LASTNAME+FIRSTNAME")
HDBCreation()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDBDescribeField(<Name and type of item>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Name and type of item>: (Cadena de caracteres)`**

Nombre dado al artículo y tipo de artículo. Este parámetro tiene el siguiente formato: 

```txt
<Item name>, <Item type>
```
donde: 

- &lt;Nombre del artículo&gt;&gt;Nombre del artículo: Nombre que se usará para manipular este ítem por programación.

- &lt;Tipo de elemento&gt;: Tipo de elemento a crear:
	


|   |   |
| --- | --- |
| C,&lt;Tamaño | Posición de texto, &lt;Tamaño&gt; es el tamaño de la cadena de caracteres |
| N,&lt;Tamaño&gt;,&lt;NbDecimales&gt; | Partida numérica. &lt;Size&gt; es el tamaño del valor numérico, &lt;NbDecimals&gt; es el número de decimales en el valor numérico. |
| M | posición booleana |
| D | Posición de fecha (en formato "AAMMDD") |
| M | Posición de nota de texto |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Uso de caracteres de espacio en los parámetros
<a name="uso_caracteres_espacio_los_parametros_ELTPARAGRAPHE000077"></a>

Los parámetros no deben estar separados por caracteres de espacio; por ejemplo, no se debe escribir:


```wl
HDBDescribeField("FIRSTNAME, C, 20")
```


Debes escribir:


```wl
HDBDescribeField("FIRSTNAME,C,20")
```

<a name="NOTE0_2"></a>




### Describir un artículo
<a name="describir_articulo_ELTPARAGRAPHE000091"></a>

- Si el archivo ya está abierto al llamar a la función **HDBDescribeField**, se generará un error 1009.

- Las claves (o índices) son definidas por [HDBDescribeIndex](../WDLang4/3044166.md).

- El número de elementos creados en un archivo de datos con la función **HDBDescribeField** está limitado a 128.

- El tamaño máximo de un artículo:

	- no puede exceder los 254 caracteres para una posición de texto no clave.

	- no puede exceder los 100 caracteres para una posición de texto clave.

	- no puede superar los 19 dígitos en la parte entera y los 15 dígitos en la parte decimal de un elemento numérico.




- Para utilizar **HDBDescribeField**, el archivo fue descrito por [HDBDescribeFile](../WDLang4/3044172.md).



<a name="NOTE0_3"></a>

<a name="limitaciones_ELTPARAGRAPHE000119"></a>

- Esta función no está disponible cuando se utiliza un ejecutable independiente..

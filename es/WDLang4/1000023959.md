


## &lt;Variable Connection&gt;.FileExist (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si un archivo de datos existe, es decir, si se ha creado físicamente ([&lt;Fuente&gt;.Creation](../WDLang4/1000024207.md) o [&lt;Fuente&gt;.CreationIfNotFound](../WDLang4/1000024208.md)). 

Observación: Se tienen en cuenta las sustituciones realizadas actualmente en el archivo de datos (por [HSubstDir](../WDLang4/3044028.md), [&lt;Fuente&gt;.ChangeDir](../WDLang4/1000024202.md) y [&lt;Fuente&gt;.ChangeName](../WDLang4/1000024201.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="3044096_Example2"></a>

```wl
// Describe the connection
MyConnection is Connection

// Check whether a Client/Server data file exists
IF MyConnection.FileExist("MyFile.FIC") = True THEN
	// Processes
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

### Comprobar si un archivo de datos HFSQL Client/Server existe

`<Result> = <Connection>.FileExist(<Physical file>)`
---

**`<Result>: (Booleano)`**



- <u><u><u>True</u></u></u> si el archivo de datos físico existe en el servidor,

- <u><u><u>False</u></u></u> en caso contrario.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión asociada a el archivo de datos.

**`<Physical file>: (Cadena de caracteres)`**

Nombre físico del archivo de datos buscado. Se debe especificar la extensión del archivo ("Cliente.fic" por ejemplo).


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **&lt;Variable Connection&gt;.FileExist** puede usarse sin importar si el archivo de datos está abierto o no.

- **&lt;Variable Connection&gt;.FileExist** no abre el archivo de datos si está cerrado.




## HDeleteParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un parámetro previamente guardado con [HSaveParameter](../WDLang4/1000017313.md). 
 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDeleteParameter("MyParameter")
```



```wl
MyConnection is Connection

HDeleteParameter(MyConnection, "MyParameter")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Borrar un parámetro en el servidor (en el código de Procedure almacenado)

`<Result> = HDeleteParameter(<Parameter>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a borrar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Retorno de carro)..  


<a name="SYNTAX2"></a>

### Eliminar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = HDeleteParameter(<Connection> , <Parameter>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a borrar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Retorno de carro)..

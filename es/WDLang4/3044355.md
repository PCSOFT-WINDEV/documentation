


## HDeleteSet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un conjunto de procedimientos almacenados de un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the set of procedures ("ManageEmail") from the server
HDeleteSet("MyConnection", ManageEmail)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDeleteSet(<Connection> , <Set of procedures>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el conjunto de procedimientos,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL que contiene el conjunto de procedimientos a eliminar. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Connection](../WDLang4/1514073.md).




**`<Set of procedures>: (Cadena de caracteres)`**

Nombre del conjunto de procedimientos a eliminar del servidor.

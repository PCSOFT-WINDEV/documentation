
## HDBDescribeFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Describe un archivo de datos en formato dBase3 (el más común) mediante programación. Esta función se utiliza para especificar el nombre, la abreviatura y el acceso a ruta del archivo xBase.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDBDescribeFile("DBCUSTOMER", "CD", "C:\FILE\CUSTOMER.DBF")
HDBDescribeField("LASTNAME,C,20")
HDBDescribeField("FIRSTNAME,C,20")
HDBDescribeField("AGE,N,3,0")
HDBDescribeField("BALANCE,N,10,2")
HDBDescribeField("MARRIED,L")
HDBDescribeField("DOB,D")
HDBDescribeField("INFO,M")
HDBCreation()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDBDescribeFile(<Logical name of xBase file> , <File prefix> , <Full name of physical file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Logical name of xBase file>: (Cadena de caracteres)`**

Nombre lógico del archivo xBase a crear. Este nombre será utilizado por todas las funciones de los archivos de manipular xBase.

**`<File prefix>: (Cadena de caracteres)`**

Abreviatura del archivo (2 caracteres).

**`<Full name of physical file>: (Cadena de caracteres)`**

Ruta donde se creará el archivo en el disco y nombre completo del archivo (con la extensión).

<a name="NOTE0_1"></a>

## Observaciones




### Usando la palabra clave EXTERN
<a name="usando_palabra_clave_extern_ELTPARAGRAPHE000051"></a>

Le recomendamos que utilice la palabra clave EXTERN para declarar un archivo que se describirá y creará, con el fin de evitar advertencias en el editor de código.. Por ejemplo:


```wl
EXTERN Customer
HDBDescribeFile("CUSTOMER", "DA", "C:\FILE\CUSTOMER.DBF")
```

<a name="NOTE0_2"></a>




### Descripción del archivo 
<a name="descripcion_del_archivo_ELTPARAGRAPHE000061"></a>

- Si el archivo ya está abierto al llamar a la función **HDBDescribeFile**, se generará un error 1009.

- El archivo será creado en el disco por [HDBCreation](../WDLang4/3044171.md).

- La estructura de los archivos está descrita por [HDBDescribeField](../WDLang4/3044170.md). El número de elementos creados en un archivo está limitado a 128.

- Los índices son descritos por [HDBDescribeIndex](../WDLang4/3044166.md).



<a name="NOTE0_3"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000083"></a>

- Esta función no está disponible cuando se utiliza una base de datos a través de .

- Esta función no está disponible cuando se utiliza un ejecutable independiente..

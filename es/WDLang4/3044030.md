


## HStatNbRec (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de entradas para un elemento clave determinado. Este número fue calculado durante el último cálculo estadístico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about the CustName key from the Customer data file
ResStat = HStatNbRec(Customer, CustName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HStatNbRec([<Data file> [, <Key item>]])`
---

**`<Result>: (Integro)`**

Número de entradas encontradas para el artículo especificado. 
Corresponde a la constante *hInvalidRecNum* si:

- las estadísticas nunca se han calculado para la posición especificada.

- las estadísticas no están soportadas por el tipo de posición especificado. HFSQL permite realizar cálculos estadísticos sólo en posiciones clave.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre, la función **HStatNbRec** utilizará el último archivo de datos utilizado por la última función HFSQL (que empieza por "H").

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre del elemento clave utilizado para calcular las estadísticas. Si no se especifica este nombre, **HStatNbRec** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las estadísticas pueden ser actualizadas por [HIndex](../WDLang4/3044133.md) y [HStatCalculate](../WDLang4/3044134.md).

- En los archivos de datos de HFSQL Classic, los cálculos estadísticos sólo pueden realizarse en las partidas clave.

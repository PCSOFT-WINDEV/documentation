


## &lt;Variable Connection&gt;.InfoGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el grupo de usuarios especificado. Esta información se asigna automáticamente a [a las variables de gestión de grupos de usuarios](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Associate the data file with the connection
Customer.ChangeConnection(MyConnection)
...
// Find the characteristics of a group
Res is boolean
Res = MyConnection.InfoGroup("Test_Group")
IF Res = True THEN
	Trace("Structure of the group updated")
	Trace("Name of the group: " + HGroup.Name)
	Trace("Description of the group: " + HGroup.Description)
	Trace("Users of the group: " + HGroup.Users)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.InfoGroup(<Group>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la información relativa al grupo,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




Observación: La función [HErrorInfo](../WDLang4/3044071.md) devolverá el error "Grupo desconocido" si el usuario actual no tiene permisos para ver a los usuarios (constante *hRightsSeeUser*). Estos derechos pueden definirse mediante [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md).

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Group>: (Cadena de caracteres)`**

Nombre del grupo de usuarios a gestionar.

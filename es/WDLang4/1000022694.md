


## &lt;Variable Connection&gt;.ListGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de grupos de usuarios definidos para una conexión.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
Res is string
Res = MyConnection.ListGroup()
// Add the groups into a List Box control
ListAdd(LIST_MyList, Res)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ListGroup()`
---

**`<Result>: (Character string)`**

Nombre de los grupos existentes para la conexión especificada. Los nombres de los grupos están separados por caracteres CR (Carriage Return).

**Observación**: Si el usuario actual no tiene permisos para ver usuarios (constante *hRightsSeeUser*) en el servidor (función [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md) o [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md)), la función **&lt;Variable Connection&gt;.ListGroup** solo devuelve el grupo actual.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión utilizada.




## &lt;Fuente&gt;.StatTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la hora de la última actualización de las estadísticas de índice. Estas estadísticas se han calculado con [&lt;Fuente&gt;.StatCalculate](../WDLang4/1000025064.md) o [&lt;Fuente&gt;.Index](../WDLang4/1000025060.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about the CustName key from the Customer data file
Customer.StatCalculate(CustName)

// Find out the date when these statistics were calculated
ResTime = Customer.StatTime()
Info("The statistics performed on the CustName item " + ...
	"were calculated at: " + ...
	TimeToString(ResTime))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.StatTime([<Key item>])`
---

**`<Result>: (Cadena de caracteres)`**

Hora en formato HHMMSS (HH corresponde a las horas, MM a los minutos y SS a los segundos). La hora se devuelve en formato UTC (Universal Time Coordinate).

**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado.

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre de la posición clave sobre la que se han calculado las estadísticas. Si no se especifica este nombre, **&lt;Fuente&gt;.StatTime** utilizará el último elemento clave utilizado por la última función de gestión HFSQL (función que empieza por "H").



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En los archivos de datos HFSQL, las estadísticas se pueden calcular sólo en los elementos clave.

- Esta función no está disponible cuando se utiliza una base de datos a través de .

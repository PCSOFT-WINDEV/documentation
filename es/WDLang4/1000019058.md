
## HClusterStop( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Suspende la ejecución de un cluster HFSQL. Los nodos del cluster ya no aceptarán ninguna conexión de las aplicaciones cliente.

El cluster puede ser reiniciado por [HClusterStart](../WDLang4/1000019057.md).

**Observación**: Esta función está reservada para un uso avanzado. Le recomendamos que utilice el HFSQL control Center para gestionar un cluster HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HClusterStop("Host1", "MyPassword")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterStop(<Coordinator> , <Password>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster. 

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

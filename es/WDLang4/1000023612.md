


## &lt;mongoDatabase Variable&gt;.CrearColección (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una colección en una base de datos MongoDB mientras especifica las opciones de esta colección.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MDBDatabase is mongoDatabase
Option is mongoCollectionOption
Option.MaxSize = 100*1024*1024
MDBDatabase.CreateCollection("product", Option)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Database>.CreateCollection(<Collection> , <Options>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Database>: (Variable de tipo mongoDatabase)`**

Nombre de la [mongoDatabase](../WDLang4/1000022410.md) Variable que corresponde a la base de datos Mongo donde se creará la colección.
Esta base de datos se crea automáticamente si no existe.

**`<Collection>: (Cadena de caracteres)`**

Nombre de la colección a crear.

**`<Options>: (mongoCollectionOption Variable)`**

Nombre de la [mongoCollectionOption](../WDLang4/1000022413.md) Variable que corresponde a las opciones de creación de la colección.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se escribe en una colección inexistente, esta colección se crea automáticamente con las opciones de Default.

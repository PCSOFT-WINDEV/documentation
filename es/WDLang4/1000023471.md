


## &lt;Conexión Variable&gt;.RSAddConfig (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una replicación entre dos servidores HFSQL en el servidor maestro. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

ConfigReplication1 is hRSConfig
ConfigReplication1.Server = "atlas.mycompany.com:4996"
Add(ConfigReplication1.File, "MyDatabase")
Add(ConfigReplication1.File, "-MyDatabase\MyFilefic")
MyConnection.RSAddConfig(ConfigReplication1, hrsNoCopy)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.RSAddConfig(<Replication to perform> , <Option> [, <Progress Bar>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó la replicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión con el servidor HFSQL maestro.

**`<Replication to perform>: (hRSConfig Variable)`**

Nombre de [hRSConfig](../WDLang4/1000020484.md) Variable que describe la replicación a realizar.

**`<Option>: (Constante de tipo Integer)`**

Acción a realizar: 


|   |   |
| --- | --- |
| *hrsCopy* | Permite la copia inicial de elementos replicados desde el servidor de conexión (servidor maestro) al otro servidor (servidor de abonado, definido en el [hRSConfig](../WDLang4/1000020484.md) Variable). Este copy está bloqueando. <br><br>Compatibilidad con versiones anteriores: Corresponde a <u><u><u><u>True</u></u></u></u> en las versiones anteriores. |
| *hrsCopyBackgroundTask* | Permite la copia inicial de elementos replicados desde el servidor de conexión (servidor maestro) al otro servidor (servidor de abonado, definido en el [hRSConfig](../WDLang4/1000020484.md) Variable). Este copy se realiza en la tarea de Background y no se bloquea. <br><br>Para averiguar si el copy se realizó: <br><br>1. Lista de las réplicas disponibles con [&lt;Conexión Variable&gt;.RSListConfig](../WDLang4/1000023554.md). <br><br>2. Si la réplica deseada no aparece en la lista, significa que no se ha añadido. <br><br>3. Si la réplica deseada aparece en la lista, compruebe el valor del **Inicialización** Property del [hRSConfig](../WDLang4/1000020484.md) Variable. Si este Property: <br><br>	- se incluye entre 0 y 1000, la réplica se añade actualmente. <br><br>	- es set al 1000, la réplica fue añadida. <br><br>	- no se especifica, la réplica no se ha añadido.  <br><br><br><br><br><br> |
| *hrsNoCopy* | No se realiza ningún copy. <br><br>Compatibilidad con versiones anteriores: Corresponde a <u><u><u><u>False</u></u></u></u> en las versiones anteriores. |



**`<Progress Bar>: (Nombre de control opcional)`**

Nombre del control Barra de progreso utilizado para ver el progreso del copy inicial.

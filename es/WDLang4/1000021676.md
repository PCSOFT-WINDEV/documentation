


## hdfsRenombre (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Renombra un archivo o un directorio que se encuentra en un servidor Hadoop.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
hdfsOpenConnection(MyConnection)
// Rename the file on the server
hdfsRename(MyConnection, "/tmp/former.txt", "/tmp/new.txt")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = hdfsRename(<Connection> , <Former name> , <New name> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cambió el nombre de la clave de registro,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable que corresponde a la conexión con el servidor Hadoop.

**`<Former name>: (Cadena de caracteres)`**

ruta absoluta del fichero o directorio a renombrar en el servidor Hadoop. Esta ruta distingue entre mayúsculas y minúsculas.

**`<New name>: (Cadena de caracteres)`**

Nueva ruta absoluta del fichero o directorio en el servidor Hadoop. Esta ruta distingue entre mayúsculas y minúsculas.

**`<Options>: (Constante opcional de tipo Integer)`**

Opciones para renombrar:


|   |   |
| --- | --- |
| *hdfsDefault*<br>(Valor predeterminado) | Si el archivo de destino ya existe, no se realiza el cambio de nombre. |
| *hdfsOverwrite* | Si el archivo de destino ya existe, se reemplaza por el archivo con el nuevo nombre. |




## Funciones memorizadas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las bases de datos de Memcached: 



|   |   |
| --- | --- |
| [MemcachedAdd](../WDLang4/1000023340.md) | Agrega una nueva pareja clave/valor en el caché Memcached. |
| [MemcachedAddKeyGroup](../WDLang4/1000023349.md) | Agrega una nueva pareja clave/valor en el caché Memcached en un grupo de servidores. |
| [MemcachedAddPrefix](../WDLang4/1000023343.md) | Añade un valor al principio del valor existente en la caché para la clave dada. |
| [MemcachedAddPrefixKeyGroup](../WDLang4/1000023357.md) | Añade un valor al inicio del valor existente en la caché para la clave dada en un grupo de servidores. |
| [MemcachedAddSuffix](../WDLang4/1000023344.md) | Añade un valor al final del valor existente en la caché para la clave dada. |
| [MemcachedAddSuffixKeyGroup](../WDLang4/1000023350.md) | Añade un valor al final del valor existente en la caché para la clave dada en un grupo de servidores. |
| [MemcachedAssign](../WDLang4/1000023342.md) | Modifica un par de clave / valor en la memoria caché de Memcached:<br><br>- Si la clave existe, se modifica el valor.<br><br>- Si la clave no existe, se añade el par clave/valor.<br><br><br> |
| [MemcachedAssignKeyGroup](../WDLang4/1000023354.md) | Modifica una pareja clave/valor en el caché Memcached en un grupo de servidores:<br><br>- Si la clave existe, se modifica el valor.<br><br>- Si la clave no existe, se añade el par clave/valor.<br><br><br> |
| [MemcachedCAS](../WDLang4/1000023348.md) | Modifica el valor de una tecla si ésta no fue modificada. |
| [MemcachedCASKeyGroup](../WDLang4/1000023356.md) | Modifica el valor de una clave en un grupo de servidores si ésta no fue modificada. |
| [MemcachedDecrement](../WDLang4/1000023347.md) | Disminuye el valor de associated con una llave. La clave se crea si no existe. |
| [MemcachedDecrementKeyGroup](../WDLang4/1000023355.md) | Disminuye el valor de associated con una llave en un grupo de servidores. |
| [MemcachedDelete](../WDLang4/1000023345.md) | Borra una clave de la caché de Memcached. |
| [MemcachedDeleteKeyGroup](../WDLang4/1000023353.md) | Elimina una clave de la caché de Memcached en un grupo de servidores. |
| [MemcachedGetKeyGroup](../WDLang4/1000023364.md) | Recupera los valores de las claves almacenadas en un grupo de servidores Memcached. |
| [MemcachedIncrement](../WDLang4/1000023346.md) | Incrementa el valor de associated con una llave. |
| [MemcachedIncrementKeyGroup](../WDLang4/1000023352.md) | Incrementa el valor de associated con una llave en un grupo de servidores. |
| [MemcachedModify](../WDLang4/1000023341.md) | Modifica un pareja de clave/valor en el caché Memcached. |
| [MemcachedModifyKeyGroup](../WDLang4/1000023351.md) | Modifica una pareja clave/valor en el caché Memcached en un grupo de servidores. |
| [MemcacheGet](../WDLang4/1000023363.md) | Recupera los valores de las claves almacenadas en un servidor Memcached. |

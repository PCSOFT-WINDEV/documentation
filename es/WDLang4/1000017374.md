


## HModifyTask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una tarea programada en el servidor HFSQL definido por la conexión. Esta tarea está en un [hScheduledTask](../WDLang4/1000017448.md) Variable.

**Observación**: [HInfoTask](../WDLang4/1000017409.md) se utiliza para recuperar las características de una tarea programada en una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
MyTask is hScheduledTask

// Retrieve the characteristics of the task
MyTask = HInfoTask(MyConnection, TaskID)
// Modify the task
MyTask.Hour = EDT_Time.Hour+1
HModifyTask(MyConnection, TaskID, MyTask)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyTask(<Connection> , <Task identifier> , <Scheduled task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la tarea programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a la que debe añadirse una tarea. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Task identifier>: (Integro)`**

Identificador de la tarea a modificar. Este identificador es devuelto por [HAddTask](../WDLang4/1000017091.md).

**`<Scheduled task>: (Variable de tipo hScheduledTask)`**

Nombre de la [hScheduledTask](../WDLang4/1000017448.md) Variable que contiene las modificaciones. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000112"></a>

Para modificar una tarea, debe tener:

- los permisos correspondientes (constante *hRightsManageTask*) en el servidor, 

- los derechos para ejecutar el comando relacionado en la tarea programada.




La tarea se ejecutará con la cuenta del usuario que la creó.

**Observación**: La tarea programada no se realizará si el servidor está apagado.. La tarea programada se realizará en la siguiente fecha disponible cuando se encienda el servidor.

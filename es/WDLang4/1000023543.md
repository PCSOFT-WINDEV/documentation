


## RedisPipelineGet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el resultado de la ejecución de un comando de Redis que fue añadido por [RedisPipelineAñadir](../WDLang4/1000023542.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RedisPipelineAdd(gRedis, "set p1 pv1")
RedisPipelineAdd(gRedis, "set p2 pv2")
RedisPipelineAdd(gRedis, "get p2")
RedisPipelineAdd(gRedis, "get p1")

r1 is redisResponse = RedisPipelineGet(gRedis)
r2 is redisResponse = RedisPipelineGet(gRedis)
r3 is redisResponse = RedisPipelineGet(gRedis)
r4 is redisResponse = RedisPipelineGet(gRedis)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RedisPipelineGet(<Connection>)`
---

**`<Result>: (redisResponse Variable)`**

[redisResponse](../WDLang4/1000023534.md) Variable que contiene el resultado del comando Redis.

**`<Connection>: (Variable de tipo redisConnection)`**

Nombre del [redisConnection](../WDLang4/1000023527.md) Variable que corresponde a la conexión del servidor Redis.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Cómo funciona el oleoducto
<a name="como_funciona_oleoducto_ELTPARAGRAPHE000049"></a>

Cuando se han añadido todos los comandos deseados a la cadena, los resultados se pueden recuperar con [RedisPipelineObtener](../WDLang4/1000023543.md).

Para evitar la recuperación de los resultados, es posible utilizar [RedisPipelineReset](../WDLang4/1000023549.md).

Si no se llama a la función If [RedisPipelineReset](../WDLang4/1000023549.md)** y a la función RedisPipelineGet.

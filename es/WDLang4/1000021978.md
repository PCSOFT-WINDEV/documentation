


## Funciones HDFS
			



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>
Las siguientes funciones se utilizan para administrar archivos en un sistema HDFS:



|   |   |
| --- | --- |
| [hdfsCloseConnection](../WDLang4/1000021867.md) | Cierra una conexión con un servidor Hadoop. |
| [hdfsCopyFrom](../WDLang4/1000021678.md) | Copia un archivo encontrado en un servidor Hadoop al ordenador local. |
| [hdfsCopyTo](../WDLang4/1000021679.md) | Copia un archivo ubicado en la máquina local a un servidor Hadoop. |
| [hdfsCreateDirectory](../WDLang4/1000021675.md) | Crea un directorio en un servidor Hadoop. |
| [hdfsDelete](../WDLang4/1000021674.md) | Elimina un archivo o directorio de un servidor Hadoop. |
| [hdfsList](../WDLang4/1000021677.md) | Muestra el contenido de un directorio que se encuentra en un servidor Hadoop. |
| [hdfsOpenConnection](../WDLang4/1000021866.md) | Abre una conexión con un servidor Hadoop. |
| [hdfsRenombre](../WDLang4/1000021676.md) | Renombra un archivo o un directorio que se encuentra en un servidor Hadoop. |



  


Las funciones HDFS utilizan los siguientes tipos:



|   |   |
| --- | --- |
| [hdfsConnection](../WDLang4/1000021650.md) | El tipo **hdfsConnection** permite definir las características avanzadas de una conexión al sistema de archivos Hadoop (HDFS). |
| [hdfsRights](../WDLang4/1000021662.md) | El tipo **hdfsRights** permite definir las características avanzadas de los permisos de acceso a un elemento (archivo, enlace, directorio) del sistema de archivos Hadoop. |
| [hdfsStatus](../WDLang4/1000021973.md) | El tipo **hdfsStatus** permite definir las características avanzadas del estado de un elemento (archivo, directorio, enlace) del sistema de archivos Hadoop. |




Atención: Las funciones de HBase están disponibles en tiempo de edición y ejecución desde Windows Vista o Windows Server 2008.

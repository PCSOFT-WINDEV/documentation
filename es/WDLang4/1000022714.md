


## &lt;Variable Connection&gt;.ModifyGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la información del grupo en función de los elementos que se encuentran en la correspondiente [variables para la gestión de grupos](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
...
// Modify the members of the group
HGroup.Users = "Flo" + CR + "Julia" + CR + "Luke"
// Modify the group
IF MyConnection.ModifyGroup("TESTERS") = True THEN
	Info("Group modified")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ModifyGroup(<Group>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el grupo,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que hay que modificar un grupo.

**`<Group>: (Cadena de caracteres)`**

Nombre del grupo cuyas características deben modificarse.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios para modificar un grupo
<a name="derechos_necesarios_para_modificar_grupo_ELTPARAGRAPHE000250"></a>

Para modificar un grupo (que no sea el del usuario actual), debe tener los permisos para ver usuarios (constante *hRightsSeeUser*). De lo contrario, la función **&lt;Variable Connection&gt;.ModifyGroup** devolverá <u><u><u><u>False</u></u></u></u> y se mostrará el mensaje de error "Grupo desconocido".

Para modificar el nombre de un grupo, se deben tener permisos para ver a los usuarios (constante *hRightsSeeUser*) y permisos para administrarlos (constante *hRightsManageUser*).

Observación: Estas constantes se utilizan en [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) y [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md).
<a name="NOTE0_2"></a>


### Gestión de derechos
<a name="gestion_derechos_ELTPARAGRAPHE000280"></a>

Las siguientes funciones se utilizan para conceder derechos a un grupo:

- [&lt;Variable Connection&gt;.ModifyDatabaseRights](../WDLang4/1000022707.md): se utiliza para conceder derechos sobre una base de datos.

- [&lt;Variable Connection&gt;.ModifyFileRights](../WDLang4/1000022709.md): se utiliza para conceder derechos sobre un archivo de datos.

- [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md): se utiliza para conceder derechos sobre un servidor.




Para obtener más información, consulte [Gestión de los derechos en HFSQL Client/Server](../WDLang4/3044333.md).

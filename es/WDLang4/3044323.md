


## HInfoDatabaseProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer las propiedades de una base de datos que está en un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", ...
	"MyHFServer", "MyDatabase", hAccessHFClientServer, HORead)
// Open the connection
HOpenConnection("MyConnection")
...
OwnerName is string
OwnerName = HInfoDatabaseProperty("MyConnection", ...
	hOwner, "MyDatabase")
IF OwnerName <> "" THEN 
	Trace("Owner of the database: " + OwnerName)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HInfoDatabaseProperty(<Connection> , <Property> , <Database>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de Property solicitado,

- Cadena vacía ("") si se produce un error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que se solicita un valor Property. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Property>: (Constante de tipo Integer)`**

Property a get de la base de datos: 


|   |   |
| --- | --- |
| *hApplication* | Nombre de la aplicación a partir de la cual se creó la base de datos. <br>Si se utiliza esta constante, &lt;Resultado&gt; es una cadena de caracteres con el siguiente formato: &lt;Nombre del ejecutable&gt;(&lt;Nombre del proyecto>). |
| *hComputer* | Nombre de el equipo a partir de la cual se creó el elemento. <br>Si se utiliza esta constante, &lt;Resultado&gt; tiene el siguiente formato: &lt;Nombre del ordenador cliente&gt; + TAB + &lt;IP Address del ordenador cliente&gt;. |
| *hCreator* | Nombre del creador de la base de datos. Esta información no tiene ninguna influencia. <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |
| *hDate* | Fecha de creación de la base de datos. <br>Si se utiliza esta constante, &lt;Resultado&gt; es una cadena con el siguiente formato: YYYYMMDDHHmmSSCCC. |
| *hNotes* | Base de datos description o cualquier otra información associated con la base de datos por [HModifyDatabaseProperty](../WDLang4/3044329.md). <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |
| *hOwner* | Propietario de la base de datos. <br>El propietario es el propietario de la base de datos. El propietario de una base de datos puede ser modificado por [HModifyDatabaseProperty](../WDLang4/3044329.md). <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |



**`<Database>: (Cadena de caracteres)`**

Nombre de la base de datos cuyas propiedades se solicitan. 
Si este parámetro corresponde a una cadena vacía ("""), se utilizará la base de datos definida en la conexión.. 
Se produce un error de WLanguage si no se define ninguna base de datos en la conexión.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar las propiedades de una base de datos, utilice [HModifyDatabaseProperty](../WDLang4/3044329.md).

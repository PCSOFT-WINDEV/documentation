
## Errores de gestión específicos de Conectores Nativos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000288"></a>
Cuando una función HFSQL falla (devuelve <u><u><u><u>False</u></u></u></u>, por ejemplo), se puede conocer mediante programación el número del error correspondiente con la función [HError](../WDLang4/3044088.md). Tienes la capacidad de Process el error y de reintentar la operación.

Esta Page presenta los principales errores de gestión de ficheros que pueden producirse al utilizar los Accesos Nativos. Para cada error, usted encontrará:

- su número

- los detalles del error




Encontrará los códigos de error comunes a todos los accesos nativos y, a continuación, los códigos de error específicos de Native MySQL Access.

**Observación**: [HErrorInfo](../WDLang4/3044071.md) devuelve los detalles del último error que se ha producido al gestionar los archivos a través de un Acceso Nativo.

## Los diferentes códigos de error comunes a todos los Accesos Nativos
<a name="los_diferentes_codigos_error_comunes_todos_los_accesos_nativos_ELTTEXTE000318"></a>
**1: El motor HFSQL y el Conector Nativo no están sincronizados**
El motor HyperFileSQL y el Native Access ya no están sincronizados.. El número de elementos de Record suministrados por el motor HFSQL no es válido.

**2: El motor SQL y el Conector Nativo no están sincronizados**
El motor SQL y el Native Access no están sincronizados. El tipo de consulta especificado por el motor SQL es desconocido para el Native Access. 

**4: Error interno del Conector Nativo**
Se requiere una Record actual válida para la operación actual. No se define ninguna Record actual.

**5: Error en el motor HFSQL**
Esta operación requiere una búsqueda inicial.. No se ha iniciado ninguna búsqueda en el elemento especificado.

**6: Error en el motor HFSQL**
No se encuentra el elemento de búsqueda especificado.

**7: Error en el motor HFSQL**
Esta operación requiere una búsqueda iniciada. No se inició ninguna búsqueda en el archivo de datos especificado.

**8: Error del Conector Nativo**
Esta operación no está soportada por Native Access.

**9: Error en el motor HFSQL**
La transacción especificada no es válida.

**10: Error del Conector Nativo**
No se puede validar la transacción especificada: la transacción interna correspondiente no ha finalizado..
Se debe finalizar la transacción interna.

**11: Error en el motor HFSQL**
El identificador de la posición almacenada no es válido.

**12: Error del sistema**
Memoria insuficiente. El sistema rechazó la asignación de memoria requerida por la operación.

**13: Fallo de funcionamiento**
Fallo de funcionamiento. Este fallo puede ser causado por una operación de otro usuario. La operación actual requiere releer el actual Record en la base de datos. El actual Record no fue encontrado. Ha sido modificado o eliminado de la base de datos.

**14: Error en el motor HFSQL**
La conexión especificada no existe.

**15: Error del sistema**
No se puede leer o abrir el archivo especificado.

**16: Incompatible libraries**
Las librerías utilizadas corresponden a varias versiones incompatibles. La librería HFSQL es demasiado antigua para ser usada por el Native Access. Reinstale las bibliotecas compatibles.

**17: WLanguage error**
Error de programación causado por las características de Native Access. La estructura del archivo de datos no apoya esta operación. No se encontró ningún campo de navegación adecuado. .

**18: WLanguage error**
La operación no puede realizarse: el número de Record especificado no coincide con el número actual de Record (0).
Especifique 0 como número de Record.

**19: Error de HFSQL**
El archivo especificado o tabla no se abre.

**20: Error de programación**
Error de programación o modificación incompatible de la estructura de archivos. El description del archivo en la base de datos no es compatible con el description del archivo en el análisis.
Se pueden implementar varias soluciones:

- Sincronizar el análisis con la base de datos y recompilar el programa.

- Modificar la estructura del archivo en la base de datos.

- Volver a crear el archivo con **HCreation** (los registros del archivo se pierden). 




**21: Error devuelto por la base de datos con respecto al identificador automatic**
Fallo causado por un error devuelto por la base de datos. El mensaje de este error se añade al siguiente mensaje: Fallo en la definición del valor del identificador automatic para el archivo especificado.

**22: Error devuelto por la base de datos**
Los detalles de este error se encuentran en el mensaje de error.

**23: Error en la capa de cliente**
Fallo general de protección en la capa de cliente utilizada por Native Access.
Posible razón: La capa de cliente no es compatible con el acceso nativo (versión demasiado antigua o demasiado reciente en relación con el acceso nativo).

**24: Fallo de posicionamiento**
Error al posicionarse en el registro especificado (por ejemplo, con la función **HRestorePosition**).
Para evitar este error, agregue un identificador de automatic en el archivo.

**26: Error en la base de datos**
El artículo especificado como parte de un índice no se encuentra en la lista de artículos de tabla..
Comprueba la coherencia entre el tabla description y el índice description de la base de datos.

**27: Error de configuración**
La capa de cliente no está instalada correctamente. Reinstale la capa de cliente.

**29: Error causado por transacciones en el motor HFSQL**
Intentar bloquear mientras no hay ninguna transacción en curso. La función **SQLTransaction** debe utilizarse antes de la función **SQLLock**.

**30: Error de programación**
El archivo no se encuentra en la base de datos. Utilice la función **HCreation** para crear el archivo.

**31: Error de configuración**
El número de serie del Native Access no es válido. Vuelva a instalar el acceso nativo.

**32: Error en el motor SQL**
Los datos de corrección de la consulta transmitidos al Native Access por el motor SQL no son consistentes..
Para ejecutar esta consulta, utilice hQueryWithoutCorrection.
Este problema también puede ser causado por un alias de elemento calculado. Si se nombra un alias de posición calculada como posición del fichero fuente, se debe modificar el nombre de la posición calculada.. 

**34: Error devuelto por la base de datos**
Error al reindexar el archivo en la clave especificada.

**35: Error de programación**
Esta operación no está soportada por el Native Access para una consulta "sin corrección"..
No especifiques hQueryWithoutCorrection ni utilices otra consulta para realizar esta operación.

**37: Error de configuración**
Una función requerida por el Native Access no se encontró en la capa de cliente. La capa de cliente no es compatible con el Native Access.
Reinstalar la capa de cliente.   

**38: Error de programación**
Error de sintaxis: no se encontró el delimitador de la Variable especificada.
Debe corregir la consulta.

**39: Error de programación**
Error de sintaxis: el nombre de la Variable no fue encontrado.
Debe corregir la consulta.

**40: Error de programación**
Error de sintaxis: la entrada/salida especificada Variable no se utiliza en la consulta. El Variable probablemente fue mal escrito en la consulta o cuando se declaró.
Debe corregir la consulta.

**41: Error de programación**
La operación solicitada requiere la adición o la finalización de la cláusula WHERE. Esto no es posible: la consulta se ejecutó con hQueryWithoutCorrection.
Si esta consulta no se puede ejecutar sin hQueryWithoutCorrection, empiece a navegar por el resultado de la consulta antes de realizar esta operación..  

**50: Error del motor HFSQL en las funciones SQL**
El índice de la columna especificada es mayor que el número de columnas que se encuentran en el resultado de la consulta.

**51: Error del motor HFSQL en las funciones SQL**
La índice de la columna especificada no es válida.

### Códigos de error específicos de Native MySQL Access
<a name="codigos_error_especificos_native_mysql_access_ELTPARAGRAPHE000229"></a>

**100: Error al configurar el servidor**
La consulta excede la longitud máxima autorizada en el servidor.
Aumente la longitud máxima autorizada en el servidor.

**101: Error en la capa de cliente MySQL**
Fallo general de protección en la capa de cliente MySQL. La capa de cliente no es compatible con el Native Access.
Instalar una capa de cliente compatible (para ello, copy una librería libmysql.dll compatible junto al ejecutable WINDEV).

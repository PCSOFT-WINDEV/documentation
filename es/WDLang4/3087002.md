
## H.NbRecModificationQuery

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer el número de registros añadidos, modificados o eliminados al ejecutar la última consulta del contexto actual.. 

Este Variable es de sólo lectura. Se actualiza cuando se ejecuta una consulta de adición, actualización o eliminación (especialmente mediante la programación con [HExecuteQuery](../WDLang4/3044080.md) y [HExecuteSQLQuery](../WDLang4/3044084.md)).

![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) Acceso a través de una Conector Nativo Oracle: Esta Variable también se actualiza.

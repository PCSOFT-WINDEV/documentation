


## &lt;Variable Connection&gt;.RefreshQuery (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea o actualiza una consulta en un servidor HFSQL. Entonces, la consulta (archivo .WDR) puede ser usada en un Procedure almacenado en el servidor.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Update or create the query
MyConnection.UpdateQuery(Query1)

// Update the query even if there is a more recent version of the query on the server
MyConnection.UpdateQuery(Query1, True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.UpdateQuery(<Query> [, <Force>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la consulta fue creada o actualizada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión con el servidor en el que se debe crear o actualizar la consulta.

**`<Query>: (Cadena de caracteres)`**

Nombre de la consulta a crear o actualizar en el servidor. 
Si este parámetro corresponde a "\*", todas las consultas serán actualizadas o creadas en el servidor.

**`<Force>: (Booleano opcional)`**



- <u><u><u><u>False</u></u></u></u> (valor predeterminado): la consulta será creada o actualizada:

	- si la consulta no se encuentra en el servidor.

	- si se encuentra una versión anterior de la consulta en el servidor.




- <u><u><u><u>True</u></u></u></u>: la consulta se creará en cualquier caso.

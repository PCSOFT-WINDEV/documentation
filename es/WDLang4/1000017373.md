


## HDeleteBackup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una copia de seguridad creada previamente con [HBackup](../WDLang4/1000017081.md). La copia de seguridad se elimina físicamente del disco.



<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDeleteBackup(<Connection> , <Backup to delete>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que debe eliminarse la copia de seguridad. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Backup to delete>: (Integro)`**

Identificador de la copia de seguridad a eliminar. Este identificador es devuelto por [HBackup](../WDLang4/1000017081.md) o [HInfoBackup](../WDLang4/1000017083.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la copia de seguridad se está creando cuando se llama a **HDeleteBackup**, se cancela y se borra.

- Para eliminar una copia de seguridad, el usuario debe ser superusuario o debe tener los permisos para realizar copias de seguridad (constante *hRightsBackup*).

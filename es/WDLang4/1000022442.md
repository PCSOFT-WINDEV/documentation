


## Funciones de MongoDB
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las bases de datos MongoDB y GridFS:



|   |   |
| --- | --- |
| [MongoAdd](../WDLang4/1000022432.md) | Agrega uno o más documentos a una colección de la base de datos de MongoDB. |
| [MongoAggregationPipeline](../WDLang4/1000025267.md) | Permite recuperar documentos que pasan por una operación de agregación en múltiples etapas y se transforman en resultados calculados. |
| [MongoCreate](../WDLang4/1000022408.md) | Crea una conexión a un servidor o cluster MongoDB. |
| [MongoCreateCollection](../WDLang4/1000022434.md) | Crea una colección en una base de datos MongoDB mientras especifica las opciones de esta colección. |
| [MongoDeleteAll](../WDLang4/1000022445.md) | Elimina todos los documentos (correspondientes al filtro especificado) de una colección de una base de datos MongoDB. |
| [MongoDeleteCollection](../WDLang4/1000022558.md) | Elimina una colección de una base de datos MongoDB. |
| [MongoDeleteDatabase](../WDLang4/1000022444.md) | Elimina una base de datos MongoDB así como sus colecciones. |
| [MongoDeleteOne](../WDLang4/1000022446.md) | Borra una Document de una colección en una base de datos de MongoDB. |
| [MongoExecuteCommand](../WDLang4/1000022435.md) | Ejecuta un comando genérico en una base de datos MongoDB o en una colección MongoDB. |
| [MongoExecuteOperations](../WDLang4/1410089189.md) | Realiza operaciones de escritura en MongoDB de forma masiva. Esto evita los problemas de latencia de red que podrían producirse si se realizan las operaciones de forma secuencial. |
| [MongoFind](../WDLang4/1000022433.md) | Encuentra documentos en una colección de MongoDB. |
| [MongoGridFSDeleteFile](../WDLang4/1000022440.md) | Elimina un archivo identificado por su nombre de un sistema de archivos MongoDB. |
| [MongoGridFSDeleteFileByID](../WDLang4/1000022596.md) | Elimina un archivo identificado por su identificador de un sistema de archivos MongoDB. |
| [MongoGridFSGetFile](../WDLang4/1000022439.md) | Recupera (en la computadora local) un archivo identificado por su nombre y encontrado en un sistema de archivos MongoDB. |
| [MongoGridFSGetFileByID](../WDLang4/1000022595.md) | Recupera (en la computadora local) un archivo identificado por su identificador y encontrado en un sistema de archivos MongoDB. |
| [MongoGridFSGetFileInfo](../WDLang4/1000023495.md) | Le permite a get información sobre un archivo de MongoDB (gridFS). |
| [MongoGridFSListFile](../WDLang4/1000022438.md) | Recupera la lista de archivos MongoDB encontrados en el sistema de archivos MongoDB (gridFS). |
| [MongoGridFSSendFile](../WDLang4/1000022436.md) | Copia un archivo del ordenador local a un sistema de archivos MongoDB (gridFS). |
| [MongoInfo](../WDLang4/1000022441.md) | Devuelve información sobre una base de datos MongoDB. |
| [MongoModifyAll](../WDLang4/1000022603.md) | Modifica documentos en una colección de una base de datos MongoDB. |
| [MongoModifyOne](../WDLang4/1000022443.md) | Modifica una Document en una colección de la base de datos de MongoDB. |



Los siguientes tipos de variables se utilizan para manipular, las bases de datos MongoDB y GridFS:: 



|   |   |
| --- | --- |
| [mongoCollection](../WDLang4/1000022411.md) | El tipo **mongoCollection** permite definir las características avanzadas de una colección en la base de datos de un servidor MongoDB. |
| [mongoCollectionOption](../WDLang4/1000022413.md) | El tipo **mongoCollectionOption** permite definir las características avanzadas de las opciones de una colección de una base de datos MongoDB. |
| [mongoConnection](../WDLang4/1000022409.md) | El tipo **mongoConnection** permite definir las características avanzadas de una conexión a un servidor MongoDB. |
| [mongoDatabase](../WDLang4/1000022410.md) | El tipo **mongoDatabase** permite definir las características de una base de datos de un servidor MongoDB. |
| [mongoDatabaseInfo](../WDLang4/1000022412.md) | El tipo **mongoDatabaseInfo** permite definir las características avanzadas de la información de una base de datos MongoDB. |
| [mongoFindOption](../WDLang4/1000022414.md) | El tipo **mongoFindOption** permite definir las características avanzadas de una búsqueda en una base de datos MongoDB. |
| [mongoGridFS](../WDLang4/1000022416.md) | El tipo **mongoGridFS** permite definir las características avanzadas de un sistema de archivos de una base de datos MongoDB. |
| [mongoGridFSFileInfo](../WDLang4/1000022417.md) | El tipo **mongoGridFSFileInfo** permite definir las características avanzadas de un archivo en una base de datos MongoDB. |
| [mongoOperation](../WDLang4/1410089157.md) | El tipo **mongoOperation** permite definir las diferentes operaciones a realizar de forma masiva. Estas operaciones se almacenan en un array de variables de tipo **mongoOperation**. |
| [mongoOperationAdd](../WDLang4/1410089163.md) | El tipo **mongoOperationAdd** permite definir todas las características avanzadas de una operación masiva de adición de documentos. |
| [mongoOperationDeleteAll](../WDLang4/1410089162.md) | El tipo **mongoOperationDeleteAll** permite definir todas las características avanzadas de una operación de eliminación de documentos que se agregará a una lista de operaciones masivas. Se eliminarán todos los documentos que coincidan con el filtro de selección. |
| [mongoOperationDeleteOne](../WDLang4/1410089161.md) | El tipo **mongoOperationDeleteOne** permite definir todas las características avanzadas de una operación de eliminación de documentos que se agregará a una lista de operaciones masivas. Se eliminará el primer documento que coincida con el filtro. |
| [mongoOperationReplaceOne](../WDLang4/1410089166.md) | El tipo **mongoOperationReplaceOne** permite definir todas las características avanzadas de una operación de reemplazo de documentos que se agregará a una lista de operaciones masivas. Se reemplazará el primer documento que coincida con el filtro. |
| [mongoOperationUpdateAll](../WDLang4/1410089165.md) | El tipo **mongoOperationUpdateAll** permite definir todas las características avanzadas de una operación de actualización de documentos que se agregará a una lista de operaciones masivas.. Se actualizarán todos los documentos que coincidan con el filtro de selección. |
| [mongoOperationUpdateOne](../WDLang4/1410089164.md) | El tipo **mongoOperationUpdateOne** permite definir todas las características avanzadas de una operación de actualización de documentos que se agregará a una lista de operaciones masivas. Se actualizará el primer documento que coincida con el filtro de selección. |
| [mongoResult](../WDLang4/1000022415.md) | Le tipo **mongoResult** permite conocer el resultado de una búsqueda en una base de datos de tipo MongoDB. |
| [mongoWriteConcern](../WDLang4/1000022597.md) | El tipo **mongoWriteConcern** permite definir el nivel de acuse de recibo solicitado para las operaciones de escritura en un mongod independiente, en un conjunto de réplicas o en un clúster fragmentado. |




## &lt;Variable Connection&gt;.ListScheduledOptimization (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista las tareas de optimización programadas de los archivos de datos HFSQL Client/Server para una conexión.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
...
arrOptim is array of hScheduledOptimization
arrOptim = MyConnection.ListScheduledOptimization()
FOR EACH Optim OF arrOptim
	DisplayInfo(Optim)	
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ListScheduledOptimization()`
---

**`<Result>: (Array of hScheduledOptimization variables)`**

Array de una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). Cada elemento encontrado en esta array contiene las características de una tarea de optimización programada.. El número de elementos en esta array corresponde al número de tareas de optimización programadas disponibles.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se deben listar las tareas de optimización programadas.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000210"></a>

Para listar las tareas de optimización programadas, debe tener los permisos de conexión (constante *hRightsConnection* o *hRightsEncryptedConnection*) para cada base de datos afectada,

Para una optimización completa del servidor, el usuario debe ser un superusuario.

Para saber si estos derechos están concedidos a un usuario o a un grupo, utilice [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) o [&lt;Variable Connection&gt;.InfoDatabaseRights](../WDLang4/1000022666.md).

Para modificar los derechos concedidos a un usuario o a un grupo de usuarios, utilice [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md) o [&lt;Variable Connection&gt;.ModifyDatabaseRights](../WDLang4/1000022707.md).

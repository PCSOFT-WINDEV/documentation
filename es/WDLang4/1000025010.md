


## &lt;Fuente&gt;.Free (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Transforma los registros cruzados ([&lt;Fuente&gt;.Cross](../WDLang4/1000025051.md)) encontrados en un archivo de datos en registros borrados. El espacio ocupado por los registros cruzados se libera y puede reutilizarse en las siguientes adiciones ([&lt;Fuente&gt;.Add](../WDLang4/1000024187.md)).

**Recordatorio**: Un Record cruzado es Inactive y desaparece del índice. Los datos que se encuentran en esta Record cruzada no se borran (a diferencia de [&lt;Fuente&gt;.Delete](../WDLang4/1000025071.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all crossed records from "Customer"
Customer.Free()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.Free()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el registro,

- <u><u><u><u>False</u></u></u></u> si se produce un problema: el Record no se borra. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no es adecuada para los archivos xBase.




## HTransactionList (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de transacciones en curso o interrumpidas en el servidor de la conexión especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

// Retrieve the list of transactions
sListTrs is string = HTransactionList(MyConnection)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HTransactionList(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de transacciones actuales o interrumpidas encontradas en el servidor associated con la conexión. Esta lista tiene el siguiente formato: 

```txt
<Transaction identifier> + TAB + <Application name> + TAB + 
<Client computer > + TAB + <Value of HComputer> + TAB + 
<Transaction status> + CR +...
```
&lt;Estado de la transacción&gt; es set a 0 para una transacción en curso y a 1 para una transacción interrumpida.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que deben listarse las operaciones. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




## HInfoUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza [las variables de gestión de usuarios](../WDLang4/3044315.md) con la información relativa al usuario especificado. Por razones de seguridad, la contraseña de usuario no se recupera.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
// Associate the data file with the connection
HChangeConnection(Customer, "MyConnection")
IF HInfoUser("MyConnection", "Flora") = True THEN
	Info("The user characteristics are as follows" + CR + ...
		"Login: " + HUser.Login + ...
		"Full Name: " + HUser.FullName + ...
		"Email: " + HUser.Email)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HInfoUser(<Connection> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se actualizó la estructura para administrar usuarios,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




Observación: [HErrorInfo](../WDLang4/3044071.md) devolverá un error "Usuario desconocido" si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser*). Estos derechos pueden definirse mediante [HModifyServerRights](../WDLang4/3044328.md).

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que se debe recuperar la información del usuario. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<User>: (Cadena de caracteres)`**

Nombre del usuario para el que se debe recuperar la información. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.




## &lt;Variable Connection&gt;.StartServerUpdate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite iniciar inmediatamente la actualización automática del servidor. Los usuarios serán desconectados al iniciar esta operación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Starts the server update
MyConnection.StartServerUpdate()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.StartServerUpdate([<Version>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema.


La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión a utilizar para la actualización del servidor.

**`<Version>: (Cadena de caracteres opcional)`**

Versión para la que se debe realizar la actualización (por ejemplo: "240036"). Por defecto, se utiliza la última actualización disponible si no se especifica este parámetro. 




## Observaciones
El servidor de actualización (que contiene el paquete para actualizar el servidor HFSQL) puede ser definido con la función [&lt;Variable Connection&gt;.ManageServerUpdate](../WDLang4/1000022662.md).




## &lt;Variable Connection&gt;.InfoView (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre una vista materializada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

Info("The data of the view was produced on " + ...
	MyConnection.InfoView(MyView, hDateLastRefresh)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.InfoView(<View> , <Requested information>)`
---

**`<Result>: (Cadena de caracteres)`**

Información solicitada.

**`<Connection>: (Variable de tipo Connection)`**

Nombre del [Conexión](../WDLang4/1514073.md) Variable que describe la conexión con el servidor HFSQL que contiene la vista materializada.

**`<View>: (Cadena de caracteres)`**

Nombre de la vista materializada que se encuentra en el servidor y para la que se solicita la información.

**`<Requested information>: (Constante de tipo Integer)`**

Información solicitada sobre la vista materializada especificada. 


|   |   |
| --- | --- |
| *hDateLastRefresh* | &lt;El resultado contendrá la fecha (en formato AAAAMMDD) de la última actualización exitosa de la vista.. |
| *hErrorLastRefresh* | &lt;El resultado&gt; corresponderá a: <br><br>- el error que se produjo durante la última actualización. <br><br>- Vacíe la cadena (""") si la última actualización tuvo éxito.<br><br><br> |
| *hRefreshInProgress* | &lt;El resultado&gt; corresponderá a: <br><br>- 1 si la vista se está actualizando,<br><br>- 0 de lo contrario.<br><br><br> |
| *hSQLCode* | &lt;El resultado contendrá el código SQL del contenido de la vista.. |




## &lt;mongoCollection Variable&gt;.ExecuteCommand (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un comando genérico en una colección de MongoDB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MDBCollection is MongoCollection
// Running a command on a MongoDB collection
MDBCollection.ExecuteCommand("{ ""ping"" : 1 }" )
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Ejecutar un comando en una colección de MongoDB

`<Result> = <Collection>.RunCommand(<Command>)`
---

**`<Result>: (Variable de tipo mongoResult)`**

[mongoResult](../WDLang4/1000022415.md) variable correspondiente al resultado del comando. Si ocurre un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.

**`<Collection>: (Variable de tipo mongoCollection)`**

Nombre de la [mongoCollection](../WDLang4/1000022411.md) Variable que representa la colección de MongoDB en la que se ejecutará el comando.

**`<Command>: (Cadena de caracteres, buffer o variante)`**

Document describiendo el comando a ejecutar. Este Document puede corresponder a: 

- un Document en formato JSON. En este caso, este parámetro corresponde a un string. 

- un Document en formato BSON. En este caso, este parámetro corresponde a un buffer. 

- a variant.

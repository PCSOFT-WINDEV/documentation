## Limitaciones conocidas
<a name="limitaciones_conocidas_ELTTEXTE000206"></a>



|   |   |
| --- | --- |
| <br><br>- Los nombres largos no son soportados por los archivos xBase.<br><br><br> |   |
| <br><br>- Número máximo de registros por archivo de datos:<br><br><br> | **ilimitado** |
| <br><br>- Número máximo de archivos xBase abiertos:<br><br><br> | **1024** |
| <br><br>- Número máximo de controles por archivo:<br><br><br> |   |
| - Archivo dBase 4: | **255** |
| - DBX: | **512** |
| - Archivo Clipper5: | **1.023** |
| - otro: | **128** |
| <br><br>- Tamaño máximo de una Record:<br><br><br> |   |
| - Archivo Clipper5: | **8.192** |
| - otro: | **4.000** |
| <br><br>- Tamaño máximo de un texto control:<br><br><br> |   |
| - Archivo Clipper5: | **2.048** |
| - otro: | **254** |
| <br><br>- Tamaño máximo de un control numérico:<br><br><br> |   |
| - Archivo dBase 4: | **20** |
| - FoxPro: | **20** |
| - otro: | **19** |
| <br><br>- Tamaño máximo de un control numérico decimal:<br><br><br> |   |
| - Archivo dBase 4: | **18** |
| - FoxPro: | **18** |
| - otro: | **15** |
| <br><br>- Número máximo de registros por archivo índice:<br><br><br> | **ilimitado** |
| <br><br>- Tamaño máximo de una llave:<br><br><br> | 100 |
| <br><br>- Longitud máxima de una expresión de clave:<br><br><br> | 100 |
| <br><br>- Funciones permitidas en las expresiones clave:<br><br>	- **DTOC**<br><br>	- **DTOS**<br><br>	- **LEFT**<br><br>	- **RIGHT**<br><br>	- **ROS**<br>			En Clipper, se debe especificar un tamaño. <br><br>	- **SUBSTRATO**<br><br>	- **UPPER**<br><br>	- **LOWER**<br><br><br><br><br><br> |   |
| <br><br>- Número máximo de índices en un archivo MDX (archivo de índice de dBase 4):<br><br><br> | 47 |
| <br><br>- Número máximo de registros por archivo de memo:<br><br><br> | **ilimitado** |
| <br><br>- Tamaño máximo de una nota de texto (editable):<br><br><br> | 24 KB |
| <br><br>- Tamaño máximo de un nombre de control:<br><br><br> | 10 |
| <br><br>- Tamaño máximo de un nombre de clave:<br><br><br> | 10 |


## hComputer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Almacena un número o identificador único de equipo para usar registros y transacciones en la red. El equipo corresponde al computador del usuario. 

De forma predeterminada, el equipo está identificado con un número único y el nombre del equipo (definido en Windows). Para identificar fácilmente el equipo que realiza las operaciones de transacción, se puede definir un identificador específico para el equipo con la función **hComputer**. Este identificador opcional reemplaza el nombre del equipo y se puede ver en [WDTrans](../WDTrans/3524005.md) y [WDLog](../WDJournal/3516012.md).




<a name="sample_code"></a>

## Ejemplo

<a name="Example1"></a>



```wl
HComputer("45000")
```


<a name="Example2"></a>



```wl
// Assign the computer number read in "COMPUTERNUM" entry
// of "MYPROGRAM" section of "WIN.INI" file
ComputerNum is a string
ComputerNum = INIRead("MYPROGRAM", "COMPUTERNUM", "")
IF ComputerNum = "" THEN
	Error("Problem retrieving the computer number")
ELSE
	HComputer(ComputerNum)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HComputer(<Identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se almacenó el número de equipo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Identifier>: (Cadena de caracteres)`**

Identifica el equipo actual (máximo 16 caracteres). Este identificador se puede ver en WDTrans y WDLog. Este identificador corresponde al nombre del ordenador cliente (devuelto por [NetMachineName](../WDLang3/3056014.md)) o al nombre de el equipo donde se encuentran los archivos de datos (para un análisis remoto).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En general, la función **hComputer** se utiliza en la red con un archivo de parámetros que se encuentra en el equipo local y que indica su número.

- La función **hComputer** no tiene ningún efecto si los archivos de datos no tienen un registro y/o si no son la fuente de una transacción.

- Si se utiliza la función **hComputer** después de la función [HTransactionStart](../WDLang4/3044002.md), se modificará el número de equipo para las siguientes operaciones realizadas.

- Esta función no está disponible cuando se utiliza una base de datos a través de .

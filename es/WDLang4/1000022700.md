


## &lt;Variable Connection&gt;.ListTask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista las tareas programadas de un servidor HFSQL Client/Server para una conexión dada.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ListTask()`
---

**`<Result>: (Character string)`**

Lista de tareas programadas encontradas en el servidor en el siguiente formato:

```txt
<Identifier of task 1> + TAB + <Description of task 1> + TAB + ...
<Task 1 enabled or not> + TAB + <User> + TAB + <Month> + TAB + ...
<Day of month> + TAB + <Day of week> + TAB + <Hour> + TAB + ...
<Minute> + TAB + <Day of month or day of week> + TAB + ...
<Number of executions> + TAB + <Procedure name> + EOT +...
```

**Observación**: Para get las características de una tarea, utilice [&lt;Variable Connection&gt;.InfoTask](../WDLang4/1000022686.md).

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se deben listar las tareas programadas.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para listar las tareas de otros usuarios, debe tener los derechos para ver a los usuarios (constante *hRightSeeUser* en [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) y [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md)).

- Para listar las tareas programadas, debe tener los derechos para conectarse (constante *hRightsConnection* o *hRightsEncryptedConnection*) a la base de datos que contiene el procedimiento almacenado. 

- Se puede utilizar la función **&lt;Variable Connection&gt;.ListTask** para verificar la existencia de una tarea. Todo lo que tiene que hacer es comprobar si el identificador de la tarea se encuentra entre las tareas devueltas por la función.


## HClusterStart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia un clúster HFSQL. Esta agrupación fue detenida de antemano (por [HClusterStop](../WDLang4/1000019058.md), por ejemplo).

**Observación**: Esta función está reservada para un uso avanzado. Le recomendamos que utilice el HFSQL control Center para gestionar un cluster HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the cluster
HClusterStart("clustermanager.mydomain.com", "secret")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterStart(<Coordinator> , <Password>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

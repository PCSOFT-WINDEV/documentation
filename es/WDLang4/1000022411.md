
## mongoCollection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoCollection** permite definir las características avanzadas de una colección en la base de datos de un servidor MongoDB. Las características de esta colección pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Cnt is mongoConnexion = MongoCreate(MongoDBServerAddress)
FOR EACH MongoDatabase OF cnt.Database
	Trace(MongoDatabase.Name)
	FOR EACH MongoCollection OF MongoDatabase.Collection
		Trace(TAB + MongoCollection.Name)
	END
END
```

### Propiedades específicas de las variables mongoCollection
<a name="propiedades_especificas_las_variables_mongocollection_ELTPARAGRAPHE000040"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a Mongo collection:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre de la colección.<br>**Esta propiedad es de solo lectura**. |
| WriteConcern | Variable de tipo [mongoWriteConcern](../WDLang4/1000022597.md) | Parámetros para escribir en la colección. |

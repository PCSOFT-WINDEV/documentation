


## &lt;Variable Connection&gt;.DeleteGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina del servidor a un grupo de usuarios asociados a una conexión .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Associate the data file with the connection
Customer.ChangeConnection(MyConnection)
...
IF MyConnection.DeleteGroup("User1") = True THEN
	Info("User1 group deleted")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteGroup(<Group>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el grupo,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




Observación: [HErrorInfo](../WDLang4/3044071.md) devolverá un error "Grupo desconocido" si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser*) y quiere eliminar otro grupo.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se debe eliminar un grupo.

**`<Group>: (Cadena de caracteres)`**

Nombre del grupo a eliminar.

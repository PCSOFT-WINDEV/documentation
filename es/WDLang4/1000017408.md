


## Funciones de enlace
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de enlace se utilizan para enlazar la interfase con los ficheros de datos (y viceversa).. Estas funciones son: 



|   |   |
| --- | --- |
| [ControlToSource](../WDLang4/1000021384.md) | Asigna el valor de control a la fuente de un enlace control. |
| [FileToPage](../WDLang4/3044148.md) | Actualiza automáticamente los controles de una página con:<br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos descritos en el editor de análisis. Esta operación se realiza independientemente del estado de los mandos (grises, inactivos o Invisible).<br><br>- los valores de las variables vinculadas de WLanguage. Esta operación se realiza independientemente del estado de los mandos (grises, inactivos o Invisible).<br><br><br> |
| [FileToScreen](../WDLang4/3044210.md) | Actualiza automáticamente los controles de una ventana o Page con:<br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos descritos en el editor de análisis. <br><br>- los valores de las variables vinculadas de WLanguage. <br><br><br>Esta operación se realiza independientemente del estado de los mandos (grises, inactivos o Invisible). |
| [PageFromSource](../WDLang4/1000021400.md) | Actualiza automáticamente los controles de una Page con: <br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos HFSL descrito en el editor de análisis. <br><br>- los valores de las variables vinculadas de WLanguage. <br><br><br>Esta operación se realiza independientemente del estado de los mandos (grises, inactivos o Invisible). |
| [PageToFile](../WDLang4/3044022.md) | Se actualiza automáticamente:<br><br>- el valor de los elementos en un archivo de datos con el valor de los controles en Page. Los elementos del fichero de datos vinculados a los controles de la ventana se actualizan automáticamente con los valores de dichos controles.<br><br>- el valor de las variables WLanguage con el valor de los controles de ventana. Las variables WLanguage vinculadas a los controles Page se actualizan automáticamente con los valores de dichos controles.<br><br><br>Esta operación se realiza independientemente del estado de los mandos (grises, inactivos o Invisible). |
| [PageToSource](../WDLang4/1000017406.md) | Se actualiza automáticamente:<br><br>- el valor de los elementos en un archivo de datos con el valor de los controles en Page.<br><br>- el valor de las variables WLanguage con el valor de los controles Page.<br><br><br> |
| [ScreenToFile](../WDLang4/3044146.md) | Se actualiza automáticamente:<br><br>- los valores de los elementos de un archivo de datos en memoria con los valores de los controles de la ventana. <br><br>- el valor de las variables WLanguage con el valor de los controles de ventana. <br><br><br>Esta función también está disponible para las páginas WEBDEV. |
| [ScreenToSource](../WDLang4/1000017342.md) | Inicializa automáticamente: <br><br>- los valores de los elementos de un archivo de datos en memoria con los valores de los controles de la ventana.<br><br>- el valor de las variables WLanguage con el valor de los controles de ventana.<br><br><br><br><br>Esta función también está disponible para las páginas WEBDEV. |
| [SourceFromPage](../WDLang4/1000021404.md) | Se actualiza automáticamente:<br><br>- el valor de los elementos en un archivo de datos con el valor de los controles en Page.<br><br>- el valor de las variables WLanguage con el valor de los controles Page.<br><br><br> |
| [SourceFromWindow](../WDLang4/1000021402.md) | Actualiza automáticamente: <br><br>- los valores de los elementos de un archivo de datos en memoria con los valores de los controles de la ventana.<br><br>- el valor de las variables WLanguage con el valor de los controles de ventana.<br><br><br>Esta función también está disponible para las páginas WEBDEV. |
| [SourceToPage](../WDLang4/1000017405.md) | Actualiza automáticamente los controles de una Page con: <br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos HFSL descrito en el editor de análisis.<br><br>- los valores de las variables vinculadas de WLanguage.<br><br><br> |
| [SourceToScreen](../WDLang4/1000017341.md) | Actualiza automáticamente los controles de una ventana o Page con: <br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos HFSL descrito en el editor de análisis.<br><br>- los valores de las variables vinculadas de WLanguage. <br><br><br> |
| [WindowFromSource](../WDLang4/1000021396.md) | Inicializa automáticamente los controles de una ventana con: <br><br>- los valores de los elementos vinculados en el Record actual (cargado en memoria) del archivo de datos HFSL descrito en el editor de análisis.<br><br>- los valores de las variables vinculadas de WLanguage. <br><br><br> |


## HClusterSynchronizationInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el progreso de la sincronización de un nodo de un cluster HFSQL. 

Recordatorio: Una sincronización se realiza automáticamente cuando un nodo de cluster se detiene anormalmente, por ejemplo: cuando se reinicia, este nodo se sincroniza con otro nodo que contiene los datos actualizados.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Status of synchronization
CurrentStatus is string
CurrentStatus = HClusterSynchronizationInfo("MyCoordinator","MyPassword", "NodeName")
Trace("Current status " + ExtractString(CurrentStatus, 1, CR) + ...
	"Step number " + ExtractString(CurrentStatus, 2, CR) + ...
	"Number of steps " + ExtractString(CurrentStatus, 3, CR) + ...
	"Progress " + ExtractString(CurrentStatus, 4, CR))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterSynchronizationInfo(<Coordinator> , <Password> , <Node>)`
---

**`<Result>: (Cadena de caracteres)`**

Información sobre la sincronización de nodos. Este resultado tiene el siguiente formato: 

```txt
<Description of Current Step> + CR + <Number of Current Step> + CR + 
<Number of Steps> + CR + <Progress (in per thousand)>
```


**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

**`<Node>: (Cadena de caracteres)`**

Nombre DNS del nodo actualmente sincronizado.

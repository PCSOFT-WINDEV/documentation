
## Diferencias y características específicas de una aplicación HFSQL Client/Server
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000221"></a>
Es posible que haya que adaptar el modo de gestión de los archivos de datos cuando se utilice una aplicación HFSQL Client/Server.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Diferencias específicas de una aplicación HFSQL Client/Server
<a name="diferencias_especificas_una_aplicacion_hfsql_clientserver_ELTTEXTE000251"></a>


### senderos
<a name="senderos_ELTPARAGRAPHE000020"></a>

Las rutas especificadas en las siguientes funciones y propiedades deben ser relativas a la base de datos que se encuentra en el servidor (base de datos utilizada en la conexión). Este modo de funcionamiento le permite ser independiente de la ubicación del servidor utilizado.

Por ejemplo, en la Diagram de abajo, las bases de datos corresponden a los siguientes directorios: "AplicaciónDatosA", "AplicaciónDatosB" y "AplicaciónDatosC".
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ClientServeur_hchangerep.gif)


En este ejemplo, después de usar la función **HChangeDir**, la nueva ruta de acceso del archivo de datos "Customer.fic" será ".\\DataAplicationA\\Customer2004\\Customer.fic".


```wl
HChangeDir(CUSTOMER, "Customer2004")
```


o


```wl
HChangeDir(CUSTOMER, ".\Customer2004")
```





|   |   |
| --- | --- |
| [DescribedDirectory](../Proprietes/2512074.md) | El directorio del archivo de datos debe ser relativo a la base de datos encontrada en el servidor. |
| [Directorio](../Proprietes/2512078.md) | El directorio del archivo de datos debe ser relativo a la base de datos encontrada en el servidor. |
| [HChangeDir](../WDLang4/3044168.md) | El acceso de ruta a el archivo de datos debe ser relativo a la base de datos que se encuentra en el servidor. |
| [HDeclareExternal](../WDLang4/3044204.md) | El acceso de ruta a el archivo de datos para importar debe ser relativo a la base de datos que se encuentra en el servidor. |

### Funciones WLanguage
<a name="funciones_wlanguage_ELTPARAGRAPHE000061"></a>

Las siguientes funciones tienen un modo de funcionamiento diferente en una aplicación HFSQL Client/Server:


|   |   |
| --- | --- |
| [HCreateView](../WDLang4/3044159.md) | La constante *hViewBreakable* no está disponible en una aplicación WINDEV HFSQL Client/Server.<br>El parámetro &lt;Tamaño de página&gt; se ignora. |
| [HDeclareExternal](../WDLang4/3044204.md) | La sintaxis 3 (uso con OLE DB/Native Access) no está disponible en una aplicación WINDEV HFSQL Client/Server. |
| [HListLink](../WDLang4/3044138.md) | El nombre de los archivos encontrados en &lt;Result&gt; corresponde al nombre y a la ruta relativa de estos archivos. |
| [HReadSeek](../WDLang4/3044050.md) | La constante *hLimitParsing* no está disponible en una aplicación WINDEV HFSQL Client/Server. |

### Notas
<a name="notas_ELTPARAGRAPHE000127"></a>

Al igual que en una aplicación WINDEV HFSQL Network, los datos manejados por las siguientes funciones son suministrados por los clientes. En consecuencia, estos archivos de datos deben encontrarse en el ordenador del cliente (o ser accesibles por el ordenador del cliente).

Por lo tanto, la ruta de los ficheros de datos se define en relación con el ordenador cliente.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ClientServeur_FicClient.gif)



|   |   |
| --- | --- |
| HDeclare | El análisis (archivo".WDD") debe encontrarse en el ordenador del cliente (o debe ser accesible desde el ordenador del cliente). |
| HExportXML | El archivo XML a generar debe encontrarse en el ordenador cliente (o debe ser accesible desde el ordenador cliente). |
| HExtractMemo | El archivo en el que se guardará el contenido del elemento debe encontrarse en el ordenador del cliente (o debe ser accesible desde el ordenador del cliente). |
| HImportHF55 | El archivo a importar y el archivo importado deben encontrarse en el ordenador del cliente (o deben ser accesibles desde el ordenador del cliente). |
| HImportText | El archivo de texto debe encontrarse en el ordenador del cliente (o debe ser accesible desde el ordenador del cliente). |
| HImportXML | El archivo XML debe encontrarse en el ordenador cliente (o debe ser accesible desde el ordenador cliente). |
| HInfoMemo | El archivo adjunto se encuentra en el ordenador cliente (o es accesible desde el ordenador cliente). |
| HLinkMemo | El archivo a adjuntar debe encontrarse en el ordenador cliente (o debe ser accesible desde el ordenador cliente). |
| HOpenAnalysis | El análisis (archivo".WDD") debe encontrarse en el ordenador del cliente (o debe ser accesible desde el ordenador del cliente). |
| HSetREP | El archivo ".REP" debe encontrarse en el ordenador cliente (o debe ser accesible desde el ordenador cliente). |

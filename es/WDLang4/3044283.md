


## HInfoGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el grupo de usuarios especificado. Esta información se asigna automáticamente a [a las variables de gestión de grupos de usuarios](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
// Associate the data file with the connection
HChangeConnection(Customer, "MyConnection")
...
// Find the characteristics of a group
Res is boolean
Res = HInfoGroup("MyConnection", "Test_Group")
IF Res = True THEN
	Trace("Structure of the group updated")
	Trace("Name of the group: " + HGroup.Name)
	Trace("Description of the group: " + HGroup.Description)
	Trace("Users of the group: " + HGroup.Users)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HInfoGroup(<Connection> , <Group>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la información relativa al grupo,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




Observación: La función [HErrorInfo](../WDLang4/3044071.md) devolverá el error "Grupo desconocido" si el usuario actual no tiene permisos para ver a los usuarios (constante *hRightsSeeUser*). Estos derechos pueden definirse mediante [HModifyServerRights](../WDLang4/3044328.md).

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Group>: (Cadena de caracteres)`**

Nombre del grupo de usuarios a gestionar.




## HoperationInProgress (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de operaciones automáticas en curso en un servidor HFSQL.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// List of all operations in progress
arrOp is dynamic array of hServerOperation 
arrOp = HOperationInProgress(cnt)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HOperationInProgress(<Connection>)`
---

**`<Result>: (Array de hServerOperation)`**

Array de las variables [hServerOperation](../WDLang4/1000022471.md) que contienen el description de todas las operaciones actuales en el servidor HFSQL.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a una base de datos. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

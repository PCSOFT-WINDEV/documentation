
## mongoConnection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoConnection** permite definir las características avanzadas de una conexión a un servidor MongoDB. Las características de esta conexión pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cntMDB is mongoConnection
cntMDB = MongoCreate("mongodb://db1.example.net")
FOR EACH MyDatabase OF cntMDB.Database
	Trace(MyDatabase.Name)
END
```

```wl
cntMDB is mongoConnection("mongodb://db1.example.net")
FOR EACH MyDatabase OF cntMDB.Database
	Trace(MyDatabase.Name)
END
```





## Declaración
<a name="declaracion_ELTTEXTE000155"></a>
<a name="SYNTAX1"></a>

`MyVariable is mongoConnection(<Connection String>)`
---

**`<Connection string>: (Cadena de caracteres)`**

Cadena de conexión al servidor o cadena de conexión del clúster que alberga la base de datos MongoDB. Para más detalles, consulte la documentación sobre MongoDB.

## Observaciones

### Propiedades específicas de las variables mongoConnection
<a name="propiedades_especificas_las_variables_mongoconnection_ELTPARAGRAPHE000048"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una conexión MongoDB:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Base de datos | Array de [mongoDatabase](../WDLang4/1000022410.md) | array asociativa que contiene las bases de datos creadas en el servidor de MongoDB. |

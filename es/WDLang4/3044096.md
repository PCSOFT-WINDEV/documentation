


## HFileExist (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber:

- si un archivo de datos existe, es decir, si se ha creado físicamente ([HCreation](../WDLang4/3044255.md) o [HCreationIfNotFound](../WDLang4/3044254.md)). 
	Observación: Se tienen en cuenta las sustituciones realizadas actualmente en el archivo de datos (por [HSubstDir](../WDLang4/3044028.md), [HChangeDir](../WDLang4/3044168.md) y [HChangeName](../WDLang4/3044141.md)).

- si se ha definido una vista o una consulta.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003044096&name=hfileexist_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Check whether the "Customer" data file exists
IF HFileExist(Customer) = True THEN
	// Processes
	...
END
```
<a name="Example2"></a>

```wl
// Check whether a Client/Server data file exists
IF HFileExist(MyConnection, "MyFile.FIC") = True THEN
	// Processes
	...
END
```

## Sintaxis

### Comprobar si un archivo de datos, consulta o vista existe

`<Result> = HFileExist([<Element>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el archivo físico, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element>: (Cadena de caracteres opcional)`**

Nombre del elemento a utilizar: 

- Nombre del archivo de datos.

- Nombre de la vista o consulta.




Si no se especifica este nombre, **HFileExist** utilizará el último archivo de datos, consulta o vista utilizado por la última función de gestión HFSQL (función que empieza por "H").

### Comprobar si un archivo de datos HFSQL Client/Server existe

`<Result> = HFileExist(<Connection> , <Physical file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos físico existe en el servidor,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión de associated con el archivo de datos. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Physical file>: (Cadena de caracteres)`**

Nombre físico del archivo de datos buscado. Se debe especificar la extensión del archivo ("Cliente.fic" por ejemplo).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **HFileExist** puede usarse sin importar si el archivo de datos está abierto o no.

- **HFileExist** no abre el archivo de datos si está cerrado.

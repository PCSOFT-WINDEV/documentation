
## OLE DB: Configuración necesaria 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
Para manipular una base de datos a través de OLE DB, los siguientes elementos deben ser instalados en el ordenador de desarrollo y en el ordenador implementación:

- un controlador OLE DB (OLEDBORA, ORACLE, ...),

- MDAC (Microsoft DATA Access Component) versión 2.7. mínimo.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Notas
<a name="notas_ELTTEXTE000106"></a>


- Si MDAC no está instalado, sólo será posible la conexión a la base de datos OLE DB.. 

- Por Default, la instalación de WINDEV y WEBDEV propone instalar MDAC 2.7.
	También se puede descargar desde el siguiente sitio web: [http://www.microsoft.com/downloads](http://www.microsoft.com/downloads).




**Observación**: En el momento de redactar esta documentación, este módulo puede ser distribuido gratuitamente. El módulo a descargar se llama: MDAC_TYP.EXE.
<a name="NOTE2_3"></a>


### Caso especial
<a name="caso_especial_ELTPARAGRAPHE000033"></a>

El decimal de la capa ADO utilizada por el acceso OLE DB está limitado a 28 dígitos.. Si el valor excede este límite, se devuelve a la base de datos en formato real.




## RedisGet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obtiene el valor de una clave de un servidor Redis como una cadena.

Observación: Para get el valor de una clave como cadena binaria, utilice [RedisEjecutarComando](../WDLang4/1000023541.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gRedis is redisConnection
gRedis.Server = "MyRedisServer:7001"

RedisSet(gRedis, "MyKey", "MyValue", 1h)
Trace(RedisGet(gRedis, "MyKey"))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RedisGet(<Connection> , <Key>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de la clave.

- Nula, si la clave no tiene valor. 




**`<Connection>: (Variable de tipo redisConnection)`**

Nombre del [redisConnection](../WDLang4/1000023527.md) Variable que corresponde a la conexión del servidor Redis.

**`<Key>: (Cadena de caracteres)`**

Nombre de la llave de get.

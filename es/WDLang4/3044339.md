


## HDisconnectClient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje en los ordenadores del Cliente y desconecta la aplicación. Los ordenadores cliente afectados son los especificados en la estructura [HClient](../WDLang4/3044340.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disconnect all users from the "MAILBOX" database
HClient.Database = "MAILBOX"
HDisconnectClient("MyConnection", "Caution: You will be disconnected soon", 300)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDisconnectClient(<Connection> , <Disconnection message> , <Display duration>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. 


**HDisconnectClient** es una función de bloqueo: el resto del programa se ejecutará cuando los diferentes clientes estén desconectados.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Disconnection message>: (Cadena de caracteres)`**

Mensaje que se mostrará en el ordenador cliente antes de la desconexión.

**`<Display duration>: (Entero o Duración)`**

Duración del mensaje en pantalla (en segundos). Este parámetro puede corresponder a:

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s).

<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000131"></a>

El ordenador cliente especificado en la estructura [HClient](../WDLang4/3044340.md) muestra el mensaje durante la duración especificada y ejecuta [EndProgram](../WDLang1/3013033.md) cuando se cierra el cuadro de mensajes.
<a name="NOTE0_2"></a>


### Personalización de la visualización de mensajes
<a name="personalizacion_visualizacion_mensajes_ELTPARAGRAPHE000146"></a>

[HOnServerCall](../WDLang4/3044343.md) se utiliza para personalizar la gestión de la visualización de mensajes en el ordenador cliente.




## &lt;mongoDatabase Variable&gt;.DeteleBase de datos (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una base de datos MongoDB así como sus colecciones.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CntMDB is mongoConnection
MyMongoDatabase is mongoDatabase
MyMongoDatabase = CntMDB.Database["CRM"]
MyMongoDatabase.DeleteDatabase()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Database>.DeleteDatabase()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Database>: (Variable de tipo mongoDatabase)`**

Nombre de la [mongoDatabase](../WDLang4/1000022410.md) Variable que corresponde a la base de datos Mongo a eliminar.

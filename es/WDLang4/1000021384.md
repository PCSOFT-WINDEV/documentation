
## ControlToSource( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Asigna el valor de control a la fuente de un enlace control.

**Observación**: Esta función es useful cuando se desarrolla una aplicación en modo MVP. Utilizada en un Process de modificación, esta función permite modificar un miembro de la presentación en lugar de toda la presentación.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify EDIT_NAME
ControlToSource()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ControlToSource([<Control used>])`
---

**`<Control used>: (Nombre de control opcional)`**

Nombre del control a manipular. Si no se especifica este parámetro, se utiliza el elemento diseñado por la palabra clave "MySelf".. La función no tiene ningún efecto si este elemento no es un control. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es equivalente a [ScreenToSource](../WDLang4/1000017342.md) utilizada en un solo control en lugar de toda la ventana. 

- Esta función es equivalente al siguiente código: 

	- control relacionado a un artículo:
			
		```txt
		DataFile.Item = Control
		```


	- control relacionado a un miembro de la clase: 
		```txt
		Class.Member = Control
		```





- La función no tiene ningún efecto si la control no tiene ningún vínculo.


## hbWrite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe celdas (valores de elementos) en una o varias filas (registros) de una HBase tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Description of data to write
Writing is hbWriting
Writing.Row = "post1"
Cell is hbCell 
Cell.Column.Family = "post"
Cell.Column.Qualifier = "title"
Cell.Value = "This is my blog post."
Add(Writing.Cell, Cell)
// Write data
hbWrite(Connection, "blog", Writing)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = hbWrite(<Connection> , <HBase Table> , <Data to write>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los datos fueron escritos, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase.

**`<HBase Table>: (Cadena de caracteres)`**

Nombre de la HBase tabla a utilizar.

**`<Data to write>: (hbWriting Variable)`**

Nombre de la variable [hbWriting](../WDLang4/1000021710.md) que corresponde al description de la operación de escritura.

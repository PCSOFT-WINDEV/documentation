


## &lt;Variable Connection&gt;.ModifyServerProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las propiedades de un servidor HFSQL.

**Observación**: Para conocer las propiedades de un servidor HFSQL, utilice [&lt;Variable Connection&gt;.InfoServerProperty](../WDLang4/1000022675.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

ResModif is boolean
ResModif = MyConnection.ModifyServerProperty(hNotes, "HFSQL Client/Server server")
IF ResModif = True THEN
	Info("Modification performed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.MoodifyServerProperty(<Property to modify> , <New value>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificaron las propiedades,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable [Conexión](../WDLang4/1514073.md) que describe la conexión para la que hay que modificar el valor de un Property.

**`<Property to modify>: (Constante de tipo Integer)`**

Property del servidor a modificar: 


|   |   |
| --- | --- |
| *hNotes* | description del servidor o cualquier otra información associated con el servidor. <br>Si se utiliza esta constante, &lt;Valor nuevo&gt; es una cadena de caracteres que contiene el texto de la note. |



**`<New value>: (Cadena de caracteres)`**

Nuevo valor para el Property.

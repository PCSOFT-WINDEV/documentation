


## HStartServerUpdate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite iniciar inmediatamente la actualización automática del servidor. Los usuarios serán desconectados al iniciar esta operación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts the server update
HStartServerUpdate(MyConnection)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HStartServerUpdate(<Connection> [, <Version>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema.


La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a una base de datos. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis.

- el nombre de la conexión definida por [HDescribeConnection](../WDLang4/3044205.md). La descripción de la conexión debe conocerse al ejecutar la función.

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Version>: (Cadena de caracteres opcional)`**

Versión para la que se debe realizar la actualización (por ejemplo: "240036"). Por defecto, se utiliza la última actualización disponible si no se especifica este parámetro. 




## Observaciones
El servidor de actualización (que contiene el paquete para actualizar el servidor HFSQL) puede ser definido con la función [HManageServerUpdate](../WDLang4/1000022635.md).

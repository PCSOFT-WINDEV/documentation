


## &lt;hdfsConnection Variable&gt;.Borrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un archivo o directorio de un servidor Hadoop.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
MyConnection.OpenConnection()
// Delete the file from the server
MyConnection.Delete("/tmp/test.data")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.Delete(<Path> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable que corresponde a la conexión con el servidor Hadoop.

**`<Path>: (Cadena de caracteres)`**

ruta absoluta del archivo o directorio a borrar del servidor Hadoop. Esta ruta distingue entre mayúsculas y minúsculas.

**`<Options>: (Constante opcional de tipo Integer)`**

Opciones de borrado:


|   |   |
| --- | --- |
| *hdfsDefault*<br>(Valor predeterminado) | Un directorio que no esté vacío no será eliminado. |
| *hdfsRecursive* | También se borran los contenidos de los directorios y subdirectorios. |

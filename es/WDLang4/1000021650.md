
## hdfsConnection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **hdfsConnection** permite definir las características avanzadas de una conexión al sistema de archivos Hadoop (HDFS). Las características de esta conexión pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
IF hdfsOpenConnection(MyConnection) = False THEN
	Error(ErrorInfo())
END
```

### Propiedades específicas de las variables hdfsConnection
<a name="propiedades_especificas_las_variables_hdfsconnection_ELTPARAGRAPHE000037"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una conexión:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| drogadicta | Cadena de caracteres | Nombre de usuario. |
| purposeode | Cadena de caracteres | DNS nombre o IP Address de "Nodo de Nombre" (servidor) del clúster Hadoop. |

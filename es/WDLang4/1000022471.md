
## hServerOperation (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **hServerOperation** se utiliza para get las características de una operación realizada en un servidor HFSQL. Las características de esta operación son devueltas por varias propiedades de WLanguage. 

Este tipo de Variable es utilizado por [Operación en curso](../WDLang4/1000022480.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrOp is array of hServerOperation = HOperationInProgress(cnt)
```

## Observaciones

### Propiedades específicas de la descripción de las variables hServerOperation
<a name="propiedades_especificas_descripcion_las_variables_hserveroperation_ELTPARAGRAPHE000034"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a **hServerOperation** Variable LE:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Copia de seguridad programada | Variable de tipo [hBackupDescription](../WDLang4/1000017455.md) | Sólo se asigna un valor a esta propiedad si se realiza una "Copia de seguridad" (propiedad **Type** establecida como *hlpBackup*). <br>**Esta propiedad es de solo lectura**. |
| escribir a máquina | Constante de tipo Integer | Tipo de operación: <br><br>- *hlpCalcStat*: Cálculo estadístico. <br><br>- *hlpKeyDetection*: Detección de llaves. <br><br>- *hlpOptimization*: Optimización de archivos de datos. <br><br>- *hlpReplication*: Replicación entre servidores. <br><br>- *hlpBackup*: reserva. <br><br>- *hlpTask*: Tarea programada. <br><br>- *hlpMaterializedView*: Actualización de una vista materializada. <br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Optimización programada | Variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md) | Sólo se asigna un valor a esta propiedad si se realiza una "Optimización de archivos" (propiedad **Type** ajustada a *hlpOptimization*).<br>**Esta propiedad es de solo lectura**. |
| Replicación | Variable de tipo [hRSConfig](../WDLang4/1000020484.md) | Sólo se asigna un valor a esta propiedad si se realiza una "Replicación" (propiedad **Type** establecida como *hlpReplication*). <br>**Esta propiedad es de solo lectura**. |
| StartDate | FechaHora | Fecha y hora de inicio de la operación.<br>**Esta propiedad es de solo lectura**. |
| Tareas programadas | Variable de tipo [hScheduledTask](../WDLang4/1000017448.md) | Sólo se asigna un valor a esta Property si se realiza una "Tarea" (**Tipo** Property set a *hlpTask*). <br>**Esta propiedad es de solo lectura**. |
| Vista ProgramadaMaterializada | Variable de tipo [hScheduleMaterializedView](../WDLang4/1000021444.md) | Sólo se asigna un valor a esta propiedad si se realiza una operación de "Refrescar vista" (propiedad **Type** establecida en *hlpMaterializedView*).<br>**Esta propiedad es de solo lectura**. |

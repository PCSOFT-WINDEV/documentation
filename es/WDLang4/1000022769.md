


## &lt;Variable Connection&gt;.DeleteUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un usuario asociado a una conexión en el servidor HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Associate the data file with the connection
Customer.ChangeConnection(MyConnection)
...
IF MyConnection.DeleteUser("Flo") = True THEN
	Info("User deleted: Flo")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteUser(<User to delete>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el usuario,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**Observación**: [HErrorInfo](../WDLang4/3044071.md) devolverá un error "Usuario desconocido" si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser*).

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se debe eliminar un usuario.

**`<User to delete>: (Cadena de caracteres)`**

Nombre del usuario a eliminar. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.




## &lt;Fichero HFSQL&gt;.ExportJSONString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta los registros de un archivo de datos (HFSQL u OLE DB), vista o consulta a una cadena de JSON.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Export all customers 
sString is string
sString = CUSTOMER.ExportJSONString()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <File>.ExportJSONString([<Items to export>] [, <Options> [, <Progress Bar>]])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena JSON.

**`<File>: (Nombre del control)`**

Nombre del archivo de datos, vista o consulta utilizada.

**`<Items to export>: (Cadena de caracteres opcional)`**

Lista de nombres de artículos a exportar. Los nombres están delimitados por comillas simples y están separados por comas o caracteres CR. Se ignoran los nombres de los elementos clave compuestos, los elementos binarios y los elementos de la nota binaria..
Si este parámetro es una cadena vacía (""), se exportan todos los elementos de del archivo de datos (excepto los elementos de clave compuesta, binario, memo binario y contraseña).

**`<Options>: (Constante opcional de tipo Integer)`**

Especifica el modo de creación del archivo JSON:


|   |   |
| --- | --- |
| *hExpBreakable* | La exportación puede interrumpirse pulsando Esc. <br>Si la exportación se interrumpe, la cadena JSON es válida y contiene sólo los registros exportados antes de la interrupción. |
| *hExpNormal*<br>(valor predeterminado) | Funcionamiento estándar. |



**`<Progress Bar>: (Nombre de control o ventana opcional)`**

Nombre de la control Barra de progreso o ventana que muestra el progreso de la creación de la cadena JSON.




## &lt;Fuente&gt;.SetPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la posición en un registro, a partir de la posición aproximada de uno de sus elementos. Se lee el Record.

Esta función se puede utilizar con archivos de datos, vistas HFSQL o consultas.

**Atención**: **&lt;Fuente&gt;.SetPosition** sólo puede utilizarse para set una posición. Para conocer la posición actual, utilice [&lt;Fuente&gt;.GetCurrentPosition](../WDLang4/1000025045.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Customer.SetPosition(name, 500)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.SetPosition([<Item>, ] <Position> [, <Precision>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectivo el posicionamiento en el registro,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema. Si no corresponde ningún registro, la función [&lt;Fuente&gt;.Out](../WDLang4/1000024605.md) devuelve <u><u><u><u>True</u></u></u></u> y no se lee ningún registro.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos, vista o consulta utilizada.

**`<Item>: (Cadena de caracteres opcional)`**

Nombre del campo utilizado. 
Si no se especifica este nombre, **&lt;Fuente&gt;.SetPosition** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").

**`<Position>: (Entero)`**

Posición aproximada a fijar. Esta posición debe encontrarse en el [ 1;1000] interval.

**`<Precision>: (Entero opcional)`**

Define la precisión con la que se realiza la posición. Atención: Cuanto mayor es la precisión, mayor es el tiempo que se tarda en posicionarse en el archivo.


|   |   |
| --- | --- |
| *hPrecisionHigh* | Alta precisión. |
| *hPrecisionLow* | Baja precisión. |
| *hPrecisionMedium*<br>(valor predeterminado) | Precisión media. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Posición en relación con un elemento clave o no clave
<a name="posicion_relacion_con_elemento_clave_clave_ELTPARAGRAPHE000289"></a>

- **Para un elemento clave**, la posición es relativa al orden del Record en el índice de este elemento.

- **Para un elemento no clave**, la posición es relativa al número del Record en el archivo de datos (véase [&lt;Fuente&gt;.RecNum](../WDLang4/1000025041.md)).



<a name="NOTE0_2"></a>


### Posición en una iteración filtrada
<a name="posicion_una_iteracion_filtrada_ELTPARAGRAPHE000305"></a>

Si se define un filtro con límites ([&lt;Fuente&gt;.Filter](../WDLang4/1000024960.md) sin cadena de condición pero con límites inferior y superior), la posición es relativa a los registros seleccionados por el filtro.

Si se da una cadena de condiciones a [&lt;Fuente&gt;.Filter](../WDLang4/1000024960.md), la posición ignora el filtro.

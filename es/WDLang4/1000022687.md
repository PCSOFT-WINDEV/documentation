


## &lt;Variable Connection&gt;.InfoUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza [las variables de gestión de usuarios](../WDLang4/3044315.md) con la información relativa al usuario especificado. Por razones de seguridad, la contraseña de usuario no se recupera.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Associate the data file with the connection
Customer.ChangeConnection(MyConnection)
IF MyConnection.InfoUser("Flora") = True THEN
	Info("The user characteristics are as follows" + CR + ...
		"Login: " + HUser.Login + ...
		"Full Name: " + HUser.FullName + ...
		"Email: " + HUser.Email)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.UserInfo(<User>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se actualizó la estructura para administrar usuarios,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




Observación: [HErrorInfo](../WDLang4/3044071.md) devolverá un error "Usuario desconocido" si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser*). Estos derechos pueden definirse mediante [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md).

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se debe recuperar la información del usuario.

**`<User>: (Cadena de caracteres)`**

Nombre del usuario para el que se debe recuperar la información. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.

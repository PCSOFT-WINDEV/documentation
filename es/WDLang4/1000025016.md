


## &lt;Fuente&gt;.ListQueryParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de parámetros de una consulta creada en el editor de consultas.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the parameters of QRY_CustomerZip
ParamList is string
ParamList = QRY_CustomerZip.ListQueryParameter()

// Adds the list of parameters into COMBO_COMBO1
COMBO_COMBO1.Add(ParamList)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.ListQueryParameter()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de parámetros de consulta. Los diferentes parámetros están separados por caracteres CR (Carriage Return).

**`<Source>: (Tipo de fuente especificada)`**

 
Nombre de una consulta creada con el editor de consultas. Este nombre corresponde al nombre lógico de la consulta definida en el editor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000193"></a>

&lt;Resultado&gt; puede asignarse directamente a un control List Box o Combo Box mediante la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md). Cada Line tendrá el nombre de uno de los parámetros de la consulta.

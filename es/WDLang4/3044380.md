
## Partial indexación de memos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000169"></a>
Tiene la posibilidad de indexar el contenido de un elemento de nota de texto. Siendo el tamaño de los memorandos de texto bastante importante, se utiliza el mecanismo de indexación de Partial para los memorandos.

La indexación de Partial de los memorandos de texto se utiliza para definir el número de caracteres que se indexarán en el memorando de texto. Este mecanismo se utiliza para optimizar el tamaño de los índices y la velocidad de búsqueda.

Por ejemplo, si el contenido de una factura (que se encuentra en un memorando de texto) está indexado, no tendrá que indexar necesariamente para cada factura las tres páginas de condiciones generales de venta que siguen a los datos de useful.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo indexar parcialmente los memos de texto?
<a name="¿como_indexar_parcialmente_los_memos_texto_ELTTEXTE000193"></a>
Se puede realizar la indexación de notas de texto:

- en el editor de análisis (para los archivos de datos descritos en el análisis)

- en WLanguage para los archivos de datos creados mediante programación.



<a name="NOTE2_2"></a>


### Implementar la indexación de los memorandos de texto en el editor de análisis
<a name="implementar_indexacion_los_memorandos_texto_editor_analisis_ELTPARAGRAPHE000046"></a>

Para implementar la indexación de notas de texto:

1. En el editor de análisis, muestra el description de los elementos que se encuentran en el un archivo de datos que contiene un memo de texto.

2. Seleccione el elemento associated con un memorándum de texto.

3. En la sección derecha de la pantalla, puede especificar si la nota debe ser una clave o no.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Indexation%20des%20m%E9mos%20-%20HC%20N%B0001.gif)


4. Especifique:

	- el tipo de llave de Partial: no una llave, una llave única o una llave con duplicados.

	- la dirección de la iteración: ascendiendo o descendiendo.

	- el tamaño de la parte indexada de la nota. Este tamaño debe especificarse en número de caracteres.

	- si la búsqueda debe ser sensible al caso, a los caracteres acentuados y a los caracteres espaciados. 




5. Validar.  

6. Generar el análisis. Si los archivos de datos ya han sido utilizados, se realiza una modificación de automatic de los archivos de datos.



<a name="NOTE2_3"></a>


### Implementar la indexación de memos de texto a través de la programación
<a name="implementar_indexacion_memos_texto_traves_programacion_ELTPARAGRAPHE000066"></a>

Para implementar la indexación de las notas de texto al describir un archivo de datos mediante programación, basta con utilizar las propiedades WLanguage disponibles en los elementos para especificar las características de la clave de la nota de texto.

| Nombre de la propiedad | Efecto |
| --- | --- |
| [denominar](../Proprietes/2510082.md) | Nombre del campo |
| [AccentSensitive](../Proprietes/2512077.md) | Tenga en cuenta los caracteres acentuados al realizar búsquedas en el elemento. |
| [Sensible a mayúsculas y minúsculas](../Proprietes/2512085.md) | Tenga en cuenta el caso (caracteres en mayúscula / minúscula) al realizar búsquedas en el elemento. |
| [PuntuaciónSensible](../Proprietes/2512087.md) | Tenga en cuenta los caracteres de puntuación (coma, punto,...) al realizar búsquedas en el elemento. |
| [envergadura](../Proprietes/2510125.md) | Tamaño de una posición de texto (cadena de caracteres, nota de texto, carácter). |
| [Tamaño de la llave](../Proprietes/1000017277.md) | Tamaño de la sección a indexar en la nota de texto. |
| [escribir a máquina](../Proprietes/2510131.md) | Identifica y modifica el tipo de un elemento. |
| [KeyType](../Proprietes/2512060.md) | Características de un elemento clave. |
| [SortType](../Proprietes/2512091.md) | Características de la clase en un artículo. |

Ejemplo:


```wl
MyMemoIndex is Item Description
MyMemoIndex.Name = "Comments"
MyMemoIndex.Type = hItemTextMemo
MyMemoIndex.KeyType = hUniqueKey
MyMemoIndex.SortType = hIncreasing
MyMemoIndex.AccentSensitive = False
MyMemoIndex.CaseSensitive = False
MyMemoIndex.PunctuationSensitive = False
MyMemoIndex.KeySize = 50
HDescribeItem(Book, MyMemoIndex)
```


## HDBIndex( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un fichero índice xBase. Los archivos de índice de un archivo de datos xBase abierto con [HDBOpen](../WDLang4/3044163.md) deben abrirse con **HDBIndex** para ser utilizados en WINDEV o WEBDEV. Si los archivos de índice no están abiertos, no se actualizarán cuando se escriba en el archivo.

Observación: no es necesario abrir el archivo de datos o los archivos de índice si el archivo acaba de ser creado por [HDBCreation](../WDLang4/3044171.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDBOpen("CUSTOMER", "CU", "C:\FILE\CUSTOMER.DBF")
HDBIndex("CUSTOMER", "C:\FILE\CUSTNAME.NTX", "CustName")
HDBIndex("CUSTOMER", "C:\FILE\CUSTPHONE.NTX", "Phone")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDBIndex(<Logical name of xBase file> , <Full name of index file> , <Key name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Logical name of xBase file>: (Cadena de caracteres)`**

Nombre dado al archivo xBase. Este archivo fue abierto por [HDBOpen](../WDLang4/3044163.md).

**`<Full name of index file>: (Cadena de caracteres)`**

Acceder a ruta para indexar el archivo y el nombre completo del archivo (con la extensión)

**`<Key name>: (Cadena de caracteres)`**

Denominación de la clave de índice. Este parámetro es obligatorio para los ficheros en formato.NDX,.NTX y.IDX (de hecho, el nombre de la clave no aparece en el fichero).. Es inútil e ignorado para archivos en formato.MDX (porque el nombre de la clave aparece en el archivo)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Nombre de la llave
<a name="nombre_llave_ELTPARAGRAPHE000063"></a>

Si la clave es una clave calculada (múltiple), no debe especificar todos los elementos incluidos en la clave, sino cualquier nombre que se utilizará en la programación.. Para mayor claridad, se utiliza a menudo el nombre del archivo de índice correspondiente.
<a name="NOTE0_2"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000071"></a>

- Esta función no está disponible cuando se utiliza una base de datos a través de .

- Esta función no está disponible cuando se utiliza un ejecutable independiente..




## &lt;Variable Connection&gt;.DuplicateDatabase (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Duplica la base de datos asociada a una conexión en un servidor HFSQL. Las tablas, los derechos, los procedimientos y los desencadenantes están duplicados.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

MyConnection.DuplicateDatabase("MyCopiedDatabase")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.DuplicateDatabase(<Duplicated database> [, <Progress Bar>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se duplicó la base de datos.

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión associated con la base de datos a duplicar.

**`<Duplicated database>: (Cadena de caracteres)`**

Nombre de la nueva base de datos a crear (copy de la base de datos actual). 
Observaciones:

- La nueva base de datos a crear no debe existir. 

- Se produce un error de WLanguage si la nueva base de datos ya existe.




**`<Progress Bar>: (Nombre de ventana o control opcional)`**



- Nombre de la ventana en la que se mostrará la barra de progreso. 

- Nombre del control Barra de progreso utilizado para mostrar el progreso. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000215"></a>

Para duplicar una base de datos, el usuario debe tener: 

- los derechos para realizar copias de seguridad. 

- los derechos de creación de bases de datos.

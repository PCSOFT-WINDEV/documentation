
## HSetReplication( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva temporalmente (o vuelve a activar) la replicación. La réplica puede ser una réplica basada en un registro o una réplica universal.

La replicación entre archivos de datos HFSQL se gestiona automáticamente.

Cuando la replicación está "desactivada", tiene la posibilidad de acceder a los archivos de datos de HFSQL Classic sin almacenar la información requerida por la replicación. A continuación, se debe restaurar la gestión de la replicación.

**Atención: El uso de esta función está reservado a programadores experimentados. No le aconsejamos que desactive temporalmente la gestión de la replicación y que realice modificaciones en los archivos de datos.. De hecho, pueden ocurrir problemas durante la sincronización.**

Observación: La replicación no está disponible al acceder a una base de datos mediante un .

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Solo está disponible la replicación universal.

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)![HFSQL Classic](https://doc.pcsoft.fr/ext/images/us/HF.png)![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) 
```wl
// Disables the log-based replication
HSetReplication(False)
// Add a value that will not be replicated
HAdd(Customer)
// Re-enables the log-based replication
HSetReplication(True)
```
<a name="Example2"></a>



```wl
// Enables the universal replication
HSetReplication(rplReplicationUniversal)
```

### Gestión de la replicación basada en registros entre bases de datos HFSQL

`<Result> = HSetReplication(<Management mode>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se admite la replicación antes de la llamada a **HSetReplication**,

- <u><u><u><u>False</u></u></u></u> si la replicación no era compatible antes de utilizar **HSetReplication**.




**`<Management mode>: (booleano)`**



- <u><u><u><u>False</u></u></u></u> para desactivar temporalmente la gestión de la replicación,

- <u><u><u><u>True</u></u></u></u> para reactivar la gestión de la replicación.  





<a name="SYNTAX2"></a>

### Gestión de la replicación basada en registros (entre bases de datos HFSQL) o la replicación universal

`<Result> = HSetReplication(<Management mode>)`
---

**`<Result>: (Constante de tipo Integer)`**

Antiguo modo de gestión de la réplica (antes de la modificación):


|   |   |
| --- | --- |
| *rplReplicationDisabled* | La replicación no se soportaba antes de llamar a **HSetReplication**. |
| *rplReplicationLogged* | La replicación basada en el registro entre archivos de datos HFSQL se gestionó antes de la llamada a **HSetReplication**. |
| *rplReplicationUniversal* | La replicación realizada antes de llamar a **HSetReplication** era una replicación universal (entre archivos de datos HFSQL y archivos de datos manejados por un Conector Nativo, por ejemplo).<br>**Observación**: se puede utilizar la constante *rplReplicationHeterogeneous* para compatibilidad con versiones anteriores. |



**`<Management mode>: (Constante de tipo Integer)`**

Nuevo modo de gestión a utilizar:


|   |   |
| --- | --- |
| *rplReplicationDisabled* | Se utiliza para desactivar temporalmente la gestión de la replicación. |
| *rplReplicationLogged* | Permite realizar una replicación basada en registros entre los archivos de datos HFSQL (utiliza los archivos de datos HFSQL con registros). |
| *rplReplicationUniversal* | Se utiliza para realizar una replicación universal (entre una base de datos HFSQL y una base de datos Oracle, por ejemplo).<br>**Observación**: se puede utilizar la constante *rplReplicationHeterogeneous* para compatibilidad con versiones anteriores. |

## Observaciones
El registro Process se desactiva cuando se desactiva la replicación. El registro Process puede volver a ser activado por [HSetLog](../WDLang4/3044052.md).




## &lt;mongoCollection Variable&gt;.ModificarTodos (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica documentos en una colección de una base de datos MongoDB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCollection is mongoCollection

let sFilter = "{ ""name"": ""Andy"" }"
let sModification = "{ ""$inc"": { ""score"": 1 } }"
MyCollection.ModifyAll(sFilter, sModification)
```
<a name="Example2"></a>

```wl
gclCollection is mongoCollection
// Find all contacts whose "Name" member is set to "SMITH" and modifies them to "PAULSON"

// Search filter
sFilter is string = "{ ""Name"": ""SMITH"" }"

// Modification criterion
sModification is string = "{ ""$set"": {""Name"": ""PAULSON"" } }"

gclCollection.ModifyAll(sFilter, sModification)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Collection>.ModifyAll(<Filter> , <Modification> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Collection>: (Variable de tipo mongoCollection)`**

Nombre de la [mongoCollection](../WDLang4/1000022411.md) Variable que representa la colección de MongoDB en la que se debe realizar la modificación.

**`<Filter>: (Tipo correspondiente al filtro utilizado)`**

description de filtros en formato MongoDB que se aplicará para seleccionar los documentos a modificar. Este filtro corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- una variante.




**`<Modification>: (Tipo correspondiente al formato utilizado)`**

description de las modificaciones que se harán en los documentos correspondientes al filtro. Corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- a variant.




**`<Options>: (Constante opcional de tipo Integer)`**

Opciones de modificación: 


|   |   |
| --- | --- |
| *mongoAddIfNotFound* | Se utiliza para añadir el Document si no existe en la colección. |
| *mongoWithoutValidation* | La estructura de Document no está comprobada. |

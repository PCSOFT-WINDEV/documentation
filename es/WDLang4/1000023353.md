
## MemcachedDeleteKeyGroup( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una clave de la caché de Memcached en un grupo de servidores.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MemcachedDeleteKeyGroup(memConnection, "MyApp", "key")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MemcachedDeleteKeyGroup(<Connection> , <Group> , <Key>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Connection>: (memcachedConnection Variable)`**

Nombre del [memcachedConnection](../WDLang4/1000023339.md) Variable que corresponde a la conexión con Memcached. 

**`<Group>: (Cadena de caracteres)`**

Sección clave utilizada para elegir el servidor que se utilizará. 

**`<Key>: (Cadena de caracteres)`**

Resto de la clave a borrar.


## Constantes para la gestión de la replicación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




| Constante | Detalles | Utilizado por las funciones |
| --- | --- | --- |
| **rplBidirectional** | <br><br>- Actualice la base de datos maestra Y la base de datos de suscriptores.<br><br>- Actualice el fichero maestro Y el fichero de suscriptores.<br><br><br> | [HSynchronizeReplica](../WDLang4/3044014.md),<br>[HRplManageFile](../WDLang4/1000017215.md) |
| **rplDateSynchroFromMaster** | Información solicitada: Fecha de la última sincronización desde la base de datos maestra. | [HInfoReplica](../WDLang4/3044015.md) |
| **rplDateSynchroFromSubscriber** | Información solicitada: Fecha de la última sincronización | [HInfoReplica](../WDLang4/3044015.md) |
| **rplDateSynchroToMaster** | Información solicitada: Fecha de la última sincronización hacia la base de datos maestra. | [HInfoReplica](../WDLang4/3044015.md) |
| **rplDateSynchroToSubscriber** | Información solicitada: Fecha de la última sincronización hacia la base de datos de suscriptores. | [HInfoReplica](../WDLang4/3044015.md) |
| **rplDeleteConflict** | La Record fue borrada de la réplica de destino y fue modificada o borrada en la réplica de origen. | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplHAdd** | Replicar una adición | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplHDelete** | Reproducir una supresión | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplHModify** | Reproducir una modificación | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplMasterFirst** | Los datos encontrados en la base de datos maestra tienen prioridad durante la reproducción. | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplMasterIsUpToDate** | Información solicitada: La réplica maestra se actualiza o no. | [HInfoReplica](../WDLang4/3044015.md) |
| **rplModifyConflict** | El Record fue modificado tanto en la réplica de origen como en la de destino | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplMostRecentFirst** | Los datos más recientes tienen prioridad. Atención, los datos más recientes se evalúan en función de la hora de la réplica del abonado. | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplNoConflict** | No se detectó ningún conflit | [HSynchronizeReplica](../WDLang4/3044014.md) |
| **rplNumMovableReplica** | Información solicitada: Número de generación de la última réplica móvil que se generó. | [HInfoReplica](../WDLang4/3044015.md) |
| **rplReplicatedItem** | Artículo replicado. | [HRplManageItem](../WDLang4/1000017216.md) |
| **rplReplicationConflictManagement** | El archivo utilizará el modo de gestión de conflictos especificado por HSynchronizeReplica. | [HRplManageFile](../WDLang4/1000017215.md) |
| **rplReplicationDisabled** | Se utiliza para desactivar temporalmente la gestión de la replicación. | [HSetReplication](../WDLang4/3044067.md) |
| **rplReplicationHeterogeneous** | Constantemente mantenida para compatibilidad con versiones anteriores. Esta constante es idéntica a *rplReplicationUniversal*. | [HSetReplication](../WDLang4/3044067.md) |
| **rplReplicationLogged** | Le permite realizar una replicación entre archivos HFSQL (utiliza los archivos HFSQL registrados). | [HSetReplication](../WDLang4/3044067.md) |
| **rplReplicationUniversal** | Se utiliza para realizar una replicación universal (entre una base de datos HFSQL y una base de datos Oracle, por ejemplo). | [HSetReplication](../WDLang4/3044067.md) |
| **rplSubscriberFirst** | Los datos encontrados en la base de datos de suscriptores (o en el fichero de suscriptores) tienen prioridad durante la replicación. | [HSynchronizeReplica](../WDLang4/3044014.md),<br>[HRplManageFile](../WDLang4/1000017215.md) |
| **rplSubscriberIsUpToDate** | Información solicitada: La réplica del suscriptor se actualiza o no | [HInfoReplica](../WDLang4/3044015.md) |
| **rplToMaster** | <br><br>- Actualice la base de datos maestra según la base de datos de suscriptores.<br><br>- Actualice el fichero maestro según la base de datos de suscriptores.<br><br><br> | [HSynchronizeReplica](../WDLang4/3044014.md),<br>[HRplManageFile](../WDLang4/1000017215.md) |
| **rplToSubscriber** | <br><br>- Actualice la base de datos de suscriptores según la base de datos maestra.<br><br>- Actualice el fichero de suscriptores según la base de datos maestra.<br><br><br> | [HSynchronizeReplica](../WDLang4/3044014.md),<br>[HRplManageFile](../WDLang4/1000017215.md) |
| **rplUnreplicatedItem** | Artículo no replicado. | [HRplManageItem](../WDLang4/1000017216.md) |

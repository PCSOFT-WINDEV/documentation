### Operaciones necesarias para ejecutar los conectores de acceso nativo en WINDEV y WEBDEV
<a name="operaciones_necesarias_para_ejecutar_los_conectores_acceso_nativo_windev_webdev_ELTPARAGRAPHE000011"></a>

Este tipo de Conector Nativo permite acceder a las bases de datos ACCESS para mejorar la interoperabilidad. Le recomendamos que utilice HFSQL, el motor de base de datos incluido en WINDEV y WEBDEV.



Para utilizar un conector de acceso nativo, es necesario instalar:

- MDAC.

- el proveedor de Microsoft Jet (instalado con Access o INTERNET Explorer 6.0 por ejemplo).




Observación: Los conectores nativos de Access están disponibles en Access 2007 y posteriores (archivos ACCDB y MDB).

## Creación de una aplicación que utiliza un conector de acceso nativo
<a name="creacion_una_aplicacion_que_utiliza_conector_acceso_nativo_ELTTEXTE000149"></a>
Para crear una aplicación utilizando tablas de Access, importe la estructura de estas tablas en el WINDEV o WEBDEV editor de análisis: 

1. Abrir el análisis associated con el proyecto: haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_Analyse.gif) en los botones de acceso rápido.

2. En la pestaña "Análisis", en el grupo "Creación", despliegue "Importar" y seleccione "Importar descripciones de archivos/tablas...".




**Si decide conservar los datos en formato Access**, las tablas Access se pueden manipular directamente con las funciones WLanguage de gestión de archivos de datos (que comienzan con la letra H). Se establece automáticamente una conexión con la base de datos de Access. 
Observación: si el ruta de las tablas Access utilizadas difiere durante el desarrollo y al ejecutar la aplicación, utilice [HChangeDir](../WDLang4/3044168.md) y [HChangeName](../WDLang4/3044141.md).

**Si elige convertir los datos a formato HFSQL** , los archivos de datos en formato HFSQL se crearán a partir de la estructura de las tablas de Access.. Estos archivos HFSQL pueden ser manejados por las funciones de gestión de los archivos de datos (empezando por la letra H).

## Conector de acceso nativo con WINDEV 5.5. ¿Qué hacer con una nueva versión de WINDEV?
<a name="conector_acceso_nativo_con_windev_55_¿que_hacer_con_una_nueva_version_windev_ELTTEXTE000173"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) En WINDEV 5.5, [SQLConnect](../WDLang4/3072005.md) permitía utilizar el conector de acceso nativo. Una vez conectado a una base de datos Access, todas las funciones SQL estándar le permiten acceder y modificar los datos encontrados en la base de datos.

Este modo de operación se mantiene en las versiones posteriores de WINDEV.

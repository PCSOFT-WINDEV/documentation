


## &lt;Variable Connection&gt;.ListUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de usuarios definidos para una conexión.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
Result1 is string
Result1 = MyConnection.ListUser()
// Add the users into a List Box control
LIST_Mylist.Add(Result1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.UserList()`
---

**`<Result>: (Character string)`**

Nombre de los usuarios existentes para la conexión especificada. Los nombres de usuarios están separados por retornos de carro (CR).

**Observación**: Si el usuario actual no tiene permisos para ver usuarios (constante *hRightsSeeUser*) en el servidor (función [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md) o [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md)), la función **&lt;Variable Connection&gt;.ListUser** solo devuelve el usuario actual.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Connection](../WDLang4/1514073.md) que describe la conexión a utilizar.

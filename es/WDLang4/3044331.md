


## HModifyServerProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las propiedades de un servidor HFSQL.

**Observación**: Para conocer las propiedades de un servidor HFSQL, utilice [HInfoServerProperty](../WDLang4/3044325.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ResModif is boolean
ResModif = HModifyServerProperty("MyConnection", hNotes, "HFSQL Client/Server server")
IF ResModif = True THEN
	Info("Modification performed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyServerProperty(<Connection> , <Property to modify> , <New value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificaron las propiedades,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que hay que modificar el valor de una Property. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Property to modify>: (Constante de tipo Integer)`**

Property del servidor a modificar: 


|   |   |
| --- | --- |
| *hNotes* | description del servidor o cualquier otra información associated con el servidor. <br>Si se utiliza esta constante, &lt;Valor nuevo&gt; es una cadena de caracteres que contiene el texto de la note. |



**`<New value>: (Cadena de caracteres)`**

Nuevo valor para el Property.

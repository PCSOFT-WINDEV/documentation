
## HListAnalysis( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de análisis WINDEV o WEBDEV (en formato HFSQL Classic) disponibles en un directorio determinado. La búsqueda se realiza recursivamente en toda la estructura de árbol del directorio especificado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003044049&name=hlistanalysis_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Fills a Combo Box control with the list of analyses
// found in \My Directories
ListAdd(COMBO_Combo, HListAnalysis("\My Directories"))

// Fills a multiline edit control 
// By using commas and CRs as separator
EDT_Edit1 = HListAnalysis("\My Projects", "," + CR)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListAnalysis(<Search directory> [, <Separator>])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre completo de los análisis encontrados, en el siguiente formato: 

```txt
<analysis 1> + <separator> + <analysis 2> +...
```

donde &lt;análisis X&gt; corresponde al nombre completo del análisis (.WDD) encontrado.

**`<Search directory>: (Cadena de caracteres)`**

Ruta del directorio donde debe iniciarse la búsqueda de análisis. Si este parámetro es set a "\*" o a "\*": se buscarán en todos los directorios de todas las unidades accesibles (dependiendo del caso, esta búsqueda puede llevar bastante tiempo).

**`<Separator>: (Cadena de caracteres opcional)`**

Cadena de caracteres utilizada para separar los diferentes análisis en &lt;Resultado&gt;.. Si no se especifica este parámetro, el CR (Carriage Return) es utilizado por Default.

## Observaciones




### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000041"></a>

Si se utiliza el retorno de carro (CR) como separador, &lt;Resultado&gt; puede asignarse directamente a un cuadro de lista o control Combo Box con [ListAdd](../WDLang1/3049004.md). Cada fila contendrá el nombre completo de uno de los análisis encontrados.

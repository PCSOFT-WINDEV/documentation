


## &lt;Variable Connection&gt;.InfoServerProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer las propiedades de un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
...
ServerNotes is string
NotesServer = MyConnection.InfoServerProperty(hNotes)
IF ServerNotes <> "" THEN 
	Trace("Description of the server: " + NotesServer)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.InfoServerProperty(<Property>)`
---

**`<Result>: (Character string)`**



- Valor de Property solicitado,

- Cadena vacía ("") si se produce un error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable [Conexión](../WDLang4/1514073.md) que describe la conexión para la que se solicita un valor Property.

**`<Property>: (Constante de tipo Integer)`**

Property para encontrar para el servidor: 


|   |   |
| --- | --- |
| *hNotes* | description del servidor o cualquier otra información associated con el servidor por [&lt;Variable Connection&gt;.ModifyServerProperty](../WDLang4/1000022720.md). <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar las propiedades de un servidor, utilice [&lt;Variable Connection&gt;.ModifyServerProperty](../WDLang4/1000022720.md).

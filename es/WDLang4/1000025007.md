


## &lt;Fuente&gt;.LogRestart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia el registro del archivo de datos. El registro se detuvo con la función [&lt;Fuente&gt;.LogStop](../WDLang4/1000025008.md).

**Observación**: El registro Process no está disponible cuando se accede a una base de datos a través de .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Customer.LogInfo("start printing " + Criterion)
Customer.LogStop()  // Stop log
Customer.ReadFirst(Criterion)
WHILE Customer.Found()
	// Process
	Customer.ReadNext()
END

Customer.LogRestart() // Restart log
Customer.LogInfo("end of print of" + Criterion)  // Write
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.LogRestart()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos cuyo registro debe ser reiniciado.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al reiniciar el registro, el archivo de datos debe estar en su estado inicial, es decir, en el estado en el que se encontraba antes de utilizar la función [&lt;Fuente&gt;.LogStop](../WDLang4/1000025008.md).

- Entre las funciones [&lt;Fuente&gt;.LogStop](../WDLang4/1000025008.md) y **&lt;Fuente&gt;.LogRestart**, no se guardará la información en el registro.

- La función **&lt;Fuente&gt;.LogRestart** equivale a la función [&lt;Fuente&gt;.SetLog](../WDLang4/1000024986.md) con el parámetro <u><u><u><u>True</u></u></u></u>.

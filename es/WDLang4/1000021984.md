


## &lt;hdfsConnection Variable&gt;.Lista (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el contenido de un directorio que se encuentra en un servidor Hadoop.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
MyConnection.OpenConnection()
// Retrieve the directory content
Content is array of hdfsStatus = MyConnection.List("/")
// Display the directory content
FOR EACH Elem OF Content
	Trace(Elem.Path)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.List(<Path>)`
---

**`<Result>: (Array)`**

Array de las variables [hdfsStatus](../WDLang4/1000021973.md) correspondientes al contenido del directorio.

**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable que corresponde a la conexión con el servidor Hadoop.

**`<Path>: (Cadena de caracteres)`**

ruta absoluta del directorio en el servidor Hadoop cuyo contenido será listado. Esta ruta distingue entre mayúsculas y minúsculas.

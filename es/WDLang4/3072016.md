
## SQLChangeConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la conexión actual. La conexión actual corresponde a la última conexión abierta o a la conexión especificada con **SQLChangeConnection**.

**Importante**: Las bases de datos HFSQL no permiten abrir varias conexiones simultáneamente. La anidación de conexiones sólo está disponible para las bases de datos que utilizan un acceso mediante ODBC, OLE DB o Conectores Nativos. El Conector AS/400 nativo permite el máximo número de conexiones permitidas por la licencia.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The customers are found on an ORACLE database
Conn1 is int = SQLConnect("ORACLE", "", "")
// The accounting is found on an SQLSERVER database
Conn2 is int = SQLConnect("SQLSERVER", "", "")
...
// Access the customers
SQLChangeConnection(Conn1)
...
// Access the accounting
SQLChangeConnection(Conn2)
...
// Close the current connection
SQLDisconnect()
// Close the current connection
SQLDisconnect()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SQLChangeConnection(<Connection number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio de conexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Connection number>: (Integro)`**

Número de la conexión a utilizar, devuelto por [SQLConnect](../WDLang4/3072005.md).

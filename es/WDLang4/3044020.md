


## HVersión (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber:

- si el contenido de un archivo de datos fue modificado.

- si el contenido de un archivo de datos utilizado por una consulta fue modificado.




Esta función se usa para detectar las modificaciones hechas a un archivo de datos compartidas entre varios programas. Si se detecta una modificación, usted tiene la posibilidad de actualización los datos mostrados en la ventana o en la página.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
VerNum = HVersion(Customer)
```
<a name="Example2"></a>

```wl
// First evaluation of the data file version
VerNum is int
VerNum = HVersion(Customer)
// Processes
// Second evaluation of the data file version
IF VerNum <> HVersion(Customer) THEN
	Info("The Customer data file was modified.")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HVersion([<Data file or query>])`
---

**`<Result>: (Integro)`**

Número de versión del archivo de datos.
![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) El número de versión es siempre igual a 0.

**`<Data file or query>: (Cadena de caracteres opcional)`**



- Nombre del archivo de datos utilizado.

- Nombre de la consulta utilizada.

- Si no se especifica este parámetro, **HVersión** manipular el último archivo de datos (última consulta) utilizado.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El archivo de datos utilizado debe ser un archivo de datos HFSQL. De lo contrario, el número de versión siempre es igual a 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Incrementando el número de versión del archivo de datos
<a name="incrementando_numero_version_del_archivo_datos_ELTPARAGRAPHE000113"></a>

Cuando se crea el archivo de datos, el número de versión se inicializa a 1. Este número se incrementa cada vez que se modifica el contenido del archivo ([HAdd](../WDLang4/3044147.md), [HWrite](../WDLang4/3044092.md), [HModify](../WDLang4/3044042.md), [TableSave](../WDLang1/3074008.md), [TableDelete](../WDLang1/3074032.md), [HFree](../WDLang4/3044064.md), [HDelete](../WDLang4/3044018.md), [HCross](../WDLang4/3044121.md)). El número se incrementa si la función tiene éxito.

**Observación**: La función **HVersión** puede utilizarse en archivos de datos indexados o no indexados.
<a name="NOTE0_2"></a>


### Consultas de múltiples archivos
<a name="consultas_multiples_archivos_ELTPARAGRAPHE000150"></a>

En el caso de una consulta de varios archivos, **HVersión** devuelve la suma de los **HVersión** de todos los archivos de datos utilizados en la consulta.

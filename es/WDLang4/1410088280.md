


## <redisConnection>.Connect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una conexión a un servidor (o cluster) de Redis.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gRedis is redisConnection
gRedis.Server = "MyRedisServer:7001"
IF gRedis.Connect() = False THEN
	ErrorInfo()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.Connect()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se estableció la conexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error. Para más detalles, véase la documentación sobre Redis.




**`<Connection>: (Variable de tipo redisConnection)`**

Nombre del [redisConnection](../WDLang4/1000023527.md) Variable que corresponde a la conexión con un servidor Redis (o cluster).

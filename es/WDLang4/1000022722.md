


## &lt;Variable Connection&gt;.ModifyTask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una tarea programada en el servidor HFSQL definido por la conexión. Esta tarea está en un [hScheduledTask](../WDLang4/1000017448.md) Variable.

**Observación**: [&lt;Variable Connection&gt;.InfoTask](../WDLang4/1000022686.md) se utiliza para recuperar las características de una tarea programada en una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
MyTask is hScheduledTask

// Retrieve the characteristics of the task
MyTask = MyConnection.InfoTask(TaskID)
// Modify the task
MyTask.Hour = EDT_Time.Hour+1
MyConnection.ModifyTask(TaskID, MyTask)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ModifyTask(<Task identifier> , <Scheduled task>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la tarea programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que hay que añadir una tarea.

**`<Task identifier>: (Entero)`**

Identificador de la tarea a modificar. Este identificador es devuelto por [&lt;Variable Connection&gt;.AddTask](../WDLang4/1000022645.md).

**`<Scheduled task>: (Variable de tipo hScheduledTask)`**

Nombre de la [hScheduledTask](../WDLang4/1000017448.md) Variable que contiene las modificaciones.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000241"></a>

Para modificar una tarea, debe tener:

- los permisos correspondientes (constante *hRightsManageTask*) en el servidor, 

- los derechos para ejecutar el comando relacionado en la tarea programada.




La tarea se ejecutará con la cuenta del usuario que la creó.

**Observación**: La tarea programada no se realizará si el servidor está apagado.. La tarea programada se realizará en la siguiente fecha disponible cuando se encienda el servidor.

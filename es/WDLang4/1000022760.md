


## &lt;Variable Connection&gt;.DeleteScheduledOptimization (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una tarea de optimización programada de archivos de datos HFSQL Client/Server.



<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteScheduledOptimization(<Task to delete>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la tarea,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se debe eliminar la tarea.

**`<Task to delete>: (Integer)`**

Identificador de la tarea a eliminar. Este identificador viene dado por [&lt;Variable Connection&gt;.AddScheduledOptimization](../WDLang4/1000022641.md) y [&lt;Variable Connection&gt;.ListScheduledOptimization](../WDLang4/1000022695.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000198"></a>

Para tener los derechos de eliminar una tarea de optimización programada, debe hacerlo:

- ser el dueño de la tarea. 

- ser super-usuario.

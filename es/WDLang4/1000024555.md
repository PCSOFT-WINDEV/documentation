


## HUpdateLink (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza todas las relaciones de un archivo de datos HFSQL Client/Server en el servidor.

Observación: Todos los archivos de datos vinculados con las relaciones deben haber sido configurados con la función [HChangeDir](../WDLang4/3044168.md), si es necesario.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Updates the links of the Customer data file
HUpdateLink(Customer)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HUpdateLink(<Data file> [, <Option>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la actualización,

- <u><u><u><u>False</u></u></u></u> si se produce un error. 




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos que permite conocer la lista de relaciones que se tuvieron en cuenta. Este parámetro también puede corresponder a: 

- "\*", para indicar que todos los archivos de datos se deben tener en cuenta,

- el nombre de un grupo de archivos de datos definido en el análisis. 




**`<Option>: (Constante o combinación de constantes opcional)`**

Opciones para la modificación automática de datos: 


|   |   |
| --- | --- |
| *hmsNoCheck* | La modificación automática de datos se realiza incluso si:<br><br>- la descripción del archivo de datos es más reciente que la del análisis,<br><br>- el GUID del archivo de datos y el GUID del análisis son diferentes.<br><br><br> |
| *hmsNormal* <br>(Valor predeterminado) | Modo de funcionamiento predeterminado de la modificación automática de datos. |


**Observación**: Los archivos de datos utilizados no deben estar abiertos (por la aplicación o por otra aplicación).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los elementos utilizados por las relaciones deben existir.

- Todos los archivos del análisis deben estar presentes si utiliza el parámetro "\*"

- Cuando se actualizan las relaciones, las relaciones eliminadas del análisis también se eliminan del servidor HFSQL. 

- Puede utilizar la función [HModifyStructure](../WDLang4/1000019439.md) antes de la función **HUpdateLink**.

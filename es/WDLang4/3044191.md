
## HDescribeLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Describe un enlace entre dos archivos de datos mediante programación.

Tras la llamada a **HDescribeLink**:

- El description del enlace está validado.

- El enlace description ya no puede ser modificado.

- La variable para el enlace description se reinicia. Se puede hacer un nuevo enlace description.




**Observación**: Para añadir una relación a una base de datos HFSQL Client/Server, utilice [HAddLink](../WDLang4/3044312.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDescribeLink(CustomerProductLink)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDescribeLink(<Link>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Link>: (Variable de tipo Link Description)`**

Nombre de la [Descripción de la relación](../WDLang4/1514039.md) Variable que corresponde al enlace a crear.

<a name="NOTE0_1"></a>

## Observaciones




### ¿Cómo describir un enlace mediante la programación?
<a name="¿como_describir_enlace_mediante_programacion_ELTPARAGRAPHE000077"></a>

Para describir un enlace a través de la programación, debes:

1. Utilice las variables de [Descripción de la relación](../WDLang4/1514039.md).

2. Describa las características del enlace a través de las propiedades HFSQL.

3. Utilice **HDescribeLink** para guardar el description del enlace.




**Observación**: Para describir un archivo de datos mediante programación, utilice [HDescribeFile](../WDLang4/3044206.md). Para describir elementos mediante programación, utilice [HDescribeItem](../WDLang4/3044233.md).

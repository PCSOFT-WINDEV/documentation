## Presentación
<a name="presentacion_ELTTEXTE000169"></a>
WINDEV, WEBDEV y WINDEV Mobile proponen diferentes métodos para describir una conexión:

- En el editor de análisis, 

- Con [HDescribeConnection](../WDLang4/3044205.md), [HChangeConnection](../WDLang4/3044150.md), etc.

- Utilizando el tipo [Conexión](../WDLang4/1514073.md) y las propiedades de WLanguage.

## Describir una conexión en el editor de análisis
<a name="describir_una_conexion_editor_analisis_ELTTEXTE000199"></a>


### Para describir una conexión en el editor de análisis: 
<a name="para_describir_una_conexion_editor_analisis_ELTPARAGRAPHE000040"></a>

1. Abra el editor de análisis.

2. En la pestaña "Análisis", en el grupo "Conexión", haga clic en "Conexiones".

3. En la asistente para la creación de la conexión, especifique:

	- El tipo de conexión a crear (HFSQL Client/Server o base de datos externa), 

	- Las características de la conexión (puerto, base de datos, usuario, contraseña,...).




4. Si ya se ha descrito una conexión en el análisis, se muestra la lista de conexiones así como sus características (esta lista puede modificarse y probarse).

## Describir una conexión con HDescribeConnection
<a name="describir_una_conexion_con_hdescribeconnection_ELTTEXTE000223"></a>
Las funciones [HDescribeConnection](../WDLang4/3044205.md) y [HChangeConnection](../WDLang4/3044150.md), ... permitirle describir una conexión y utilizarla. El principio es el siguiente:

- [HDescribeConnection](../WDLang4/3044205.md) describe una nueva conexión con una base de datos externa. Esta conexión se llama. Se pueden definir parámetros específicos como el modo de gestión de los cursores (si la base de datos a la que se accede lo permite).

- Para asociar esta conexión con una tabla o con un archivo, utilice [HChangeConnection](../WDLang4/3044150.md).

- Para abrir la conexión, utilice [HOpenConnection](../WDLang4/3044107.md). Si los parámetros de la conexión descritos por **HDescribeConnection** son incorrectos, [HOpenConnection](../WDLang4/3044107.md) devolverá un error.

- Con [HDescribeConnection](../WDLang4/3044205.md), una descripción de Oracle tabla importada en el análisis puede ser utilizada directamente por los archivos de datos HFSQL Classic. Basta con describir una conexión temporal a una base de datos HFSQL Classic y asociarla con el archivo description de en el análisis.

## Descripción de una conexión a través del tipo de conexión y de sus propiedades
<a name="descripcion_una_conexion_traves_del_tipo_conexion_sus_propiedades_ELTTEXTE000249"></a>
Un [Conexión](../WDLang4/1514073.md) Variable se utiliza para describir una o varias conexiones a bases de datos externas (HFSQL Client/Server, Conector Nativo, etc.).

La conexión puede establecerse mediante [HOpenConnection](../WDLang4/3044107.md).

El [Conexión](../WDLang4/1514073.md) Variable también se puede utilizar para:

- describir una nueva conexión (equivalente a [HDescribeConnection](../WDLang4/3044205.md)).

- crear una conexión (que será abierta por [HOpenConnection](../WDLang4/3044107.md)).

- modificar una conexión existente.

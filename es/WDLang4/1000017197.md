
## ReplicInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre una réplica. Esta replicación fue definida por el editor de replicación (ReplicEdit).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ReplicInfo("MyReplication", replicExist) = True THEN
	ReplicSynchronize("MyReplication")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ReplicInfo(<Replication name> , <Requested information>)`
---

**`<Result>: (Tipo correspondiente a la información solicitada)`**

Información solicitada por &lt;Información solicitada&gt;.

**`<Replication name>: (Cadena de caracteres)`**

Nombre de la réplica a utilizar. Este nombre corresponde a:

- el nombre del archivo ".wer", creado por ReplicEdit (para una replicación en línea). Este archivo debe encontrarse junto al ejecutable.

- el nombre de la publicación (para un servidor de replicación).




**`<Requested information>: (Constante de tipo Integer)`**

Información solicitada sobre la réplica especificada:


|   |   |
| --- | --- |
| *replicExist* | &lt;Resultado&gt; es set a:<br><br>- <u><u><u><u>True</u></u></u></u> si existe la replicación (es decir, si el archivo .WER se encuentra junto al ejecutable).<br><br>- <u><u><u><u>False</u></u></u></u> si no existe la replicación.<br><br><br>**Observación**: Para una replicación realizada mediante un servidor de replicación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u> si los parámetros pasados a la función  [ReplicOpen](../WDLang4/1000017194.md) son correctos. |
| *replicInitialized* | &lt;Resultado&gt; es set a:<br><br>- <u><u><u><u>True</u></u></u></u> si se inicializa la replicación,<br><br>- <u><u><u><u>False</u></u></u></u> si no se inicializa la replicación.<br><br><br>Se produce un error no fatal si la réplica no existe.. Este error puede ser procesado por el [ErrorOccurred](../WDLang1/3087001.md) Variable. |
| *replicSynchronizationDate* | &lt;El resultado corresponde a la fecha de la última sincronización.<br><br>Se produce un error no fatal si la réplica no existe.. Este error puede ser procesado por el [ErrorOccurred](../WDLang1/3087001.md) Variable. |
| *replicSynchronizationRequestDate* | &lt;El resultado corresponde a la fecha de la última petición de sincronización.<br><br>Se produce un error no fatal si la réplica no existe.. Este error puede ser procesado por el [ErrorOccurred](../WDLang1/3087001.md) Variable. |

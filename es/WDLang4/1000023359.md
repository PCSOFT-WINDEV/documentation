
## memcachedResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **memcachedResult** permite definir las características de un valor leído en un servidor Memcached. Las características de este valor pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
R is memcachedResult = MemcachedGet(memConnection, "key")
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables memcachedResult
<a name="propiedades_especificas_las_variables_memcachedresult_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a **memcachedResult** Variable LE:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| CAS | Integro | Valor de la versión anterior leída.<br>**Esta propiedad es de solo lectura**.<br><br>Atención: El servidor Memcached debe soportar CAS (**SupportCAS** Property de la variable [memcachedConnection](../WDLang4/1000023339.md) establecida en <u><u><u><u>True</u></u></u></u>). CAS (Check And set) se utiliza para realizar una modificación de datos sólo si estos datos no fueron modificados desde su última lectura. |
| Clave | Cadena de caracteres | Nombre de la clave.<br>**Esta propiedad es de solo lectura**. |
| languidecer | Integro | Valor adicional associated con la pareja Llave/Valor.<br>**Esta propiedad es de solo lectura**. |
| Valor | Buffer | Valor de la clave.<br>**Esta propiedad es de solo lectura**. |

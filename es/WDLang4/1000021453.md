


## HExecuteScheduling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta inmediatamente una tarea programada independientemente de su programación: 

- procedimiento almacenado, 

- refuerzos,

- optimización, 

- actualización de una vista materializada.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrBckDesc is array dynamic of hBackupDescription 
arrBckDesc = HListScheduling(MyDatabaseConnection, hlpBackup)
hBckDesc is hBackupDescription
FOR EACH hBckDesc OF arrBckDesc
	HExecuteScheduling(MyDatabaseConnection, hBckDesc)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Ejecutar una tarea programada inmediatamente

`<Result> = HExecuteScheduling(<Connection> , <Scheduled task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó la tarea programada, 

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Scheduled task>: (Variable de tipo hScheduledTask)`**

Nombre de la variable [hScheduledTask](../WDLang4/1000017448.md) que describe la tarea que desencadena un Procedure almacenado.


<a name="SYNTAX2"></a>

### Ejecutar una tarea de optimización programada

`<Result> = HExecuteScheduling(<Connection> , <Scheduled optimization task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó la tarea de optimización programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Scheduled optimization task>: (Variable de tipo hScheduledOptimization)`**

Nombre de la [hScheduledOptimization](../WDLang4/1000018854.md) Variable utilizada.


<a name="SYNTAX3"></a>

### Ejecutar una programación de copia de seguridad

`<Result> = HExecuteScheduling(<Connection> , <Scheduled backup> [, <Type of backup>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó la copia de seguridad programada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Scheduled backup>: (Variable de tipo hBackupDescription)`**

Nombre de la [hBackupDescription](../WDLang4/1000017455.md) Variable utilizada. 

**`<Type of backup>: (Booleano opcional)`**

Tipo de backup a realizar: 

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): Copia de seguridad diferencial. 

- <u><u><u><u>False</u></u></u></u>: Copia de seguridad completa. 




**Observación**: Una copia de seguridad diferencial sólo se puede realizar si la tarea de copia de seguridad incluye una programación diferencial. 


<a name="SYNTAX4"></a>

### Ejecutar una programación de actualización de una vista materializada

`<Result> = HExecuteScheduling(<Connection> , <Materialized view>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó la planificación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Materialized view>: (Variable de tipo hScheduleMaterializedView)`**

Nombre de la variable [hScheduleMaterializedView](../WDLang4/1000021444.md) que contiene el description de las vistas materializadas a actualización.

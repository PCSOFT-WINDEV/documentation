


## HDeactivateServerTrigger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva un trigger del Servidor HFSQL Client/Server en un servidor. Este trigger está deshabilitado para todos los clientes de este servidor. Este activador puede volver a ser activado por [HActivateServerTrigger](../WDLang4/1000017034.md).  

**Observación**: Las funciones para manejar los disparadores del servidor son funciones avanzadas. Los triggers del servidor se pueden crear directamente en el editor de análisis. Para obtener más información, consulte [Disparadores de servidor](../WDLang4/3044369.md).




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Desactivar un disparador de servidor en archivos lógicos HFSQL

`HDeactivateServerTrigger(<Trigger> , <Logical HFSQL Client/Server files>)`
---

**`<Trigger>: (Cadena de caracteres)`**

Nombre del disparador para desactivar. Este nombre se definió con [HDescribeServerTrigger](../WDLang4/1000017033.md).

**`<Logical HFSQL Client/Server files>: (Cadena de caracteres)`**

Nombre lógico de uno o más archivos de datos de HFSQL Client/Server. 
Para especificar varios nombres, los nombres de los archivos de datos deben estar separados por una coma (",").  


<a name="SYNTAX2"></a>

### Desactivar un trigger de servidor en archivos HFSQL físicos de un servidor

`<Result> = HDeactivateServerTrigger(<Connection> , <Trigger> [, <Physical HFSQL Client/Server files>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Trigger>: (Cadena de caracteres)`**

Nombre del disparador para desactivar. Este nombre se definió con [HDescribeServerTrigger](../WDLang4/1000017033.md).

**`<Physical HFSQL Client/Server files>: (Cadena de caracteres opcional)`**

Nombre físico de uno o más archivos de datos de HFSQL Client/Server associated con la conexión (por ejemplo "CUSTOMER.FIC"). 
Para especificar varios nombres, los nombres de los archivos de datos deben estar separados por una coma (","). 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el trigger se desactivará en todos los ficheros de datos de las bases de datos..

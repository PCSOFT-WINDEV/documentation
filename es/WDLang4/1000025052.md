


## &lt;Fuente&gt;.Reset (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa una o más variables de los elementos encontrados en un archivo de datos con sus valores de Default. Los valores de Default se definen cuando el archivo de datos se describe en el editor de análisis.

**Observaciones**: 

- La función **&lt;Fuente&gt;.Reset** no modifica el registro en el archivo de datos, solo modifica los datos en memoria. 

- La función **&lt;Fuente&gt;.Reset** modifica la propiedad [NewRecord](../Proprietes/1000017090.md) del archivo de datos correspondiente: esta propiedad cambia a <u><u><u>True</u></u></u>.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Assign the default values to the items
Customer.Reset()
// Display
PAGE_Customer.FromFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Inicializar los elementos de un archivo de datos con sus valores predeterminados

`<Result> = <Source>.Reset([<Item>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la inicialización,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) devuelve más detalles.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres opcional)`**

Nombre del elemento a inicializar. Si no se especifica este parámetro o si es igual a "\*", la inicialización se realizará para todos los elementos del archivo de datos.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta. Todos los elementos se reinicializan automáticamente.

<a name="NOTE0_1"></a>

## Observaciones


### Abrir archivos
<a name="abrir_archivos_ELTPARAGRAPHE000285"></a>

La función **&lt;Fuente&gt;.Reset** abre el archivo de datos utilizado. Por lo tanto, el archivo de datos debe haber sido creado.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Durante la gestión personalizada de errores, se debe probar la función [HErrorLock](../WDLang4/3044099.md) después de llamar a la función **&lt;Fuente&gt;.Reset**.

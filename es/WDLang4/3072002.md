### Recordatorio
<a name="recordatorio_ELTPARAGRAPHE000013"></a>






|   |   |
| --- | --- |
| [HExecuteQuery](../WDLang4/3044080.md) | Declara una consulta creada en el editor de consultas al motor HFSQL y ejecuta la consulta. |
| [HExecuteSQLQuery](../WDLang4/3044084.md) | Inicializa una consulta escrita en lenguaje SQL y declara esta consulta al motor HFSQL. |




La lista de funciones SQL disponibles en WLanguage es la siguiente: 




|   |   |
| --- | --- |
| [SQLAssociate](../WDLang4/3072004.md) | Automáticamente asocia cada columna del resultado de la consulta con un control o con una Variable de la aplicación. |
| [SQLChangeConnection](../WDLang4/3072016.md) | Modifica la conexión actual. |
| [SQLClose](../WDLang4/3072015.md) | Declara el final de la ejecución de la consulta y libera los recursos de memoria asignados durante la ejecución de la consulta. |
| [SQLCol](../WDLang4/3072018.md) | Recupera el contenido de la columna especificada del resultado de la consulta, para la actual Line. |
| [SQLColumn](../WDLang4/3072001.md) | Devuelve las características de todas las columnas (o posiciones):<br><br>- para una determinada tabla.<br><br>- para una consulta determinada.<br><br><br> |
| [SQLColumnFormat](../WDLang4/3072006.md) | Devuelve los diferentes tipos de columnas soportadas: <br><br>- por la fuente de datos identificada por [SQLConnect](../WDLang4/3072005.md).<br><br>- por la base de datos a la que se accede por una conexión específica (conexión descrita en el análisis o descrita por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md)).<br><br><br> |
| [SQLConnect](../WDLang4/3072005.md) | Conecta la aplicación actual a una base de datos que debe ser interrogada mediante SQL. |
| [SQLConnectWS](../WDLang4/1000021031.md) | Conecta la aplicación actual a una base de datos que será consultada por SQL a través de un Webservice Proxy( SOAP). |
| [SQLDisconnect](../WDLang4/3072008.md) | Cierra la conexión actual y libera la memoria utilizada por la conexión. |
| [SQLExec](../WDLang4/3072007.md) | Nombra y ejecuta una consulta SQL. |
| [SQLExecWDR](../WDLang4/3072010.md) | Ejecuta el código SQL de una consulta creada en el editor de consultas. |
| [SQLFetch](../WDLang4/3072011.md) | Va al siguiente fila (es decir, al siguiente Record) del resultado de la consulta. |
| [SQLFirst](../WDLang4/3072017.md) | Posiciones en la primera fila del resultado de la consulta. |
| [SQLGetCol](../WDLang4/3072019.md) | Recupera el contenido de la columna especificada del resultado de la consulta, para la actual Line. |
| [SQLGetMemo](../WDLang4/3072020.md) | Recupera el contenido de una columna Memo del resultado de la consulta, para la fila actual. |
| [SQLGetTextMemo](../WDLang4/3072012.md) | Recupera el contenido de una columna de Memorias de Texto que se encuentra en el resultado de la consulta, para el actual Line. |
| [SQLInfo](../WDLang4/3072028.md) | Inicializa las diferentes variables SQL con información relativa a la última consulta ejecutada. |
| [SQLLast](../WDLang4/3072009.md) | Establece la posición en la última fila del resultado de la consulta. |
| [SQLListSource](../WDLang4/3072027.md) | Enumera las fuentes de datos declaradas en un sitio local. |
| [SQLListTable](../WDLang4/3072021.md) | Enumere todas las tablas a las que puede acceder el sistema de base de datos conectado. |
| [SQLLock](../WDLang4/3072003.md) | Solía cerrar con llave:<br><br>- Todos los datos tabla, en modo de lectura y escritura. Sus datos no pueden ser accedidos por otros ordenadores.<br><br>- Los registros seleccionados por la consulta. Los otros ordenadores no pueden acceder a estos datos. Este método se puede utilizar para actualizar registros, por ejemplo.<br><br><br> |
| [SQLModify](../WDLang4/3072023.md) | Modifica el contenido de una fila en el resultado de la consulta. |
| [SQLNext](../WDLang4/3072030.md) | Posiciones en la próxima fila del resultado de la consulta. |
| [SQLPrevious](../WDLang4/3072022.md) | Establece la posición en la fila anterior del resultado de la consulta. |
| [SQLReqExists](../WDLang4/3072026.md) | Verifica la existencia de una consulta. |
| [SQLSeek](../WDLang4/3072024.md) | Posiciones sobre el resultado especificado fila de la consulta. |
| [SQLTable](../WDLang4/3072031.md) | Transfiere el resultado de una consulta a una control Tabla poblada programáticamente (una Caja de Listas o control Combo Box), con la posibilidad de Partial Fetch (el resultado se recupera por bloques de filas). |
| [SQLTransaction](../WDLang4/3072029.md) | Inicia, compromete o revierte una transacción:<br><br>- sobre los archivos de una base de datos abierta por [SQLConnect](../WDLang4/3072005.md).,<br><br>- en los archivos de una base de datos abierta por una conexión (OLE DB o Native Access/Conector Nativo) descrita en el editor de análisis o por [HOpenConnection](../WDLang4/3044107.md).<br><br><br> |


## HServerStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer el estado de un servidor HFSQL (esta función utiliza el servicio MantaManager).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Status of a server identified by the computer name
SWITCH HServerStatus("MyServer:4905")
	CASE hStopped: Info("Server stopped")
	CASE hStarted: Info("Server started")
	CASE hInaccessible: Info("Server not available")
END
```


<a name="Example2"></a>





```wl
// Status of a server identified by a Connection variable
Cnt_HFSQL is Connection
Cnt_HFSQL.Provider = hAccessHFClientServer
Cnt_HFSQL.User = "admin"
Cnt_HFSQL.Password = ""
Cnt_HFSQL.Server = "GB:4916"
Cnt_HFSQL.Database = "CRM"

SWITCH HServerStatus(Cnt_HFSQL.Server)
	CASE hStopped: Info("Server stopped")
	CASE hStarted: Info("Server started")
	CASE hInaccessible: Info("Server not available")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Estado de un servidor (con capacidad para especificar el puerto del servidor HFSQL)

`<Result> = HServerStatus(<Computer name>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del servidor HFSQL: 


|   |   |
| --- | --- |
| *hInaccessible* | Estado del servidor no definido |
| *hStarted* | Servidor iniciado |
| *hStopped* | El servidor se detuvo |



**`<Computer name>: (Cadena de caracteres)`**

Nombre del servidor. Se puede especificar el número de puerto utilizado por el servidor HFSQL. De forma predeterminada, se utiliza el puerto 4900. Si el número de puerto del servidor HFSQL es diferente, utilice la siguiente notación:

&lt;Nombre de equipo&gt;:&lt;Número de puerto&gt;

Ejemplo: "NombreEquipo:4901"

El número de puerto corresponde al número de puerto utilizado en [HOpenConnection](../WDLang4/3044107.md).  


<a name="SYNTAX2"></a>

### Estado de un servidor mientras se especifica el nombre del servidor HFSQL

`<Result> = HServerStatus(<Computer name> , <Name of the HFSQL server>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del servidor HFSQL: 


|   |   |
| --- | --- |
| *hInaccessible* | Estado del servidor no definido |
| *hStarted* | Servidor iniciado |
| *hStopped* | El servidor se detuvo |



**`<Computer name>: (Cadena de caracteres)`**

Nombre del servidor.

**`<Name of the HFSQL server>: (Cadena de caracteres)`**

Nombre del servidor HFSQL a reiniciar (este nombre aparece en el Centro HFSQL control). 


<a name="SYNTAX3"></a>

### Estado de un servidor al especificar una conexión

`<Result> = HServerStatus(<Connection name>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del servidor HFSQL: 


|   |   |
| --- | --- |
| *hInaccessible* | Estado del servidor no definido |
| *hStarted* | Servidor iniciado |
| *hStopped* | El servidor se detuvo |



**`<Connection name>: (Cadena de caracteres)`**

Nombre de la conexión para la que se debe comprobar el estado del servidor. Esta conexión fue definida en el editor de análisis, o por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estado indefinido
<a name="estado_indefinido_ELTPARAGRAPHE000112"></a>

El estado del servidor es indefinido cuando no se puede establecer una conexión con MantaManager.. Este fallo de conexión puede deberse a los siguientes problemas:

- El servicio MantaManager no está iniciado,

- El servicio MantaManager no responde,

- Un firewall impide el acceso al servicio MantaManager,

- ...




Debe reiniciar el servicio MantaManager desde el administrador de servicios de Windows y debe comprobar si el puerto 5002 o el puerto 4999 están abiertos (sin firewall ni antivirus).

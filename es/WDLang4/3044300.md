
## HRplFilterProcedure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el WLanguage Procedure que se llamará cada vez que se realice una operación de replicación en un archivo de datos específico. Esta Procedure está dirigida por [HCreateMoveableReplica](../WDLang4/3044209.md) y [HSynchronizeReplica](../WDLang4/3044014.md).

**Observación: Esta función solo se puede utilizar durante la replicación universal.**

Nota: La replicación no está disponible al acceder a una base de datos mediante un .




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The procedure validates the messages. 
// MessageValidation will be called for each replicated message
HRplFilterProcedure(Messages, "MessageValidation")

// Prepare to send messages to Paris (from MPL)
HCreateMovableReplica("P:\MSG\MSG.RPM", "Paris", "C:\Temp\MsgMPL.zip")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HRplFilterProcedure(<File name> , <Procedure name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema (por ejemplo, si las tablas o elementos/columnas especificadas no existen). La función [HError](../WDLang4/3044088.md) permite identificar el error. 
	Atención: &lt;Resultado&gt; es set a <u><u><u><u>True</u></u></u></u> incluso si los archivos/tablas especificados no están replicados.




**`<File name>: (Cadena de caracteres)`**

Nombre del archivo de datos o tabla en el que se debe ejecutar el Procedure.

**`<Procedure name>: (Cadena de caracteres correspondiente a un WLanguage Procedure)`**

Nombre de un WLanguage Procedure que se encuentra en la aplicación WINDEV o en el sitio WEBDEV. Esta Procedure debe ser creada en su aplicación. Este Procedure no espera ningún parámetro. Este procedimiento se llama antes de cada operación realizada en el archivo de destino. Varias variables pueden ser usadas en este Procedure. Si este procedimiento devuelve <u><u><u><u>False</u></u></u></u>, la operación no se realiza. Este Procedure está acostumbrado:

- filtrar los registros que no deben ser replicados.

- gestionar los conflictos, ajustando los valores antes de que se produzca un error duplicado.

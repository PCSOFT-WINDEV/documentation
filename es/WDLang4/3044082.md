
## HErrorPassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Después de la ejecución de una función de lectura o escritura en un archivo de datos, permite averiguar si se produjo un error causado por una contraseña incorrecta en este archivo de datos.

Se recomienda utilizar **HErrorPassword** después de la primera función que maneja un archivo de datos HFSQL.

Esta función se puede utilizar con los archivos de datos HFSQL, vistas HFSQL o consultas.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) De forma predeterminada, WINDEV incluye una [automatic gestión de errores de contraseña](../WDLang4/3044188.md). Esta función no está disponible en el código de los procedimientos almacenados.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sólo se notifican los errores de contraseña que se produjeron en los archivos de datos HFSQL.. 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the status report of HReadFirst
IF HReadFirst(Customer, Name) = False THEN
	// Problem during the read operation
	// Checks whether the error is caused by a wrong password
	IF HErrorPassword() = True THEN
		Info("The password given for the Customer file is invalid")
		 // Open the window used to enter the password
		Password = Open(WIN_Enter_Password)
		HPass(Customer, Password)
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HErrorPassword()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la contraseña es incorrecta,

- <u><u><u><u>False</u></u></u></u> si la contraseña es correcta.

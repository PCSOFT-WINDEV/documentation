
## MemcachedDecrementKeyGroup( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Disminuye el valor de associated con una llave en un grupo de servidores.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MemcachedDecrementKeyGroup(memConnection, "MyApp", "counter")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MemcachedDecrementKeyGroup(<Connection> , <Group> , <Key> [, <Decrement> [, <Initial value> [, <Expiration>]]])`
---

**`<Result>: (Integro)`**

Valor clave anterior. 

**`<Connection>: (memcachedConnection Variable)`**

Nombre del [memcachedConnection](../WDLang4/1000023339.md) Variable que corresponde a la conexión con Memcached. 

**`<Group>: (Cadena de caracteres)`**

Sección clave utilizada para elegir el servidor que se utilizará.

**`<Key>: (Cadena de caracteres)`**

Clave de uso.

**`<Decrement>: (Entero opcional)`**

Valor de disminución (1 por Default). 

**`<Initial value>: (Entero opcional)`**

Valor de la clave inicial si la clave buscada no existe. De hecho, la clave buscada se crea automáticamente si no existe y su valor es set a &lt;Valor Inicial&gt;. 
ATENCIÓN: para utilizar este parámetro, el servidor de Memcached debe utilizar el protocolo de comunicación binario (**BinaryProtocol** Property de la variable [memcachedConnection](../WDLang4/1000023339.md) ajustado a <u><u><u><u>True</u></u></u></u>). 
Se devuelve un error si no se especifica este parámetro..

**`<Expiration>: (Duración entera u opcional)`**

Duración del almacenamiento de la clave en la memoria caché. Esta duración se expresa en segundos. Esta duración puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration. 

- la indicación directa de la duración (por ejemplo, "10s"). 


Esta duración es set a 0 por Default. En este caso, la duración es ilimitada: si no hay suficiente memoria, las teclas más antiguas se borran.




## <HFSQL connection>TransactionRollbackToSavepoint (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la transacción al punto de guardado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

// Start transactions
MyConnection.TransactionStart()

// Actions A

// Declare a savepoint
MyConnection.TransactionSavepoint("MySavepoint")

// Actions B

// Undo all actions after the declaration of MySavepoint
MyConnection.TransactionRollbackToSavepoint("MySavepoint")

// Actions C

// Declare a savepoint
MyConnection.TransactionSavepoint("MySavepoint2")

// Actions D

// Validate (commit) transactions
// Actions A, C and D are taken into account, 
// but not B, because they are between TransactionSavepoint() 
// and TransactionRollbackToSavepoint

MyConnection.TransactionEnd()

// MySavepoint2 is disabled
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.TransactionRollbackToSavepoint(<Savepoint name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se revierte con éxito,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. [HErrorInfo](../WDLang4/3044071.md) devuelve más detalles sobre el error.




**`<Connection>: (Nombre del control)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión a un servidor HFSQL.

**`<Savepoint name>: (Cadena de caracteres)`**

Nombre del punto de guardado al que se va a revertir la transacción. Si el punto de guardado no existe, se devuelve un error.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función debe utilizarse en una transacción. 

- También se puede utilizar la correspondiente sentencia SQL: "ROLLBACK TO SAVEPOINT" identificador.

- Esta función no está disponible: 

	- en archivos HFSQL en modo cluster,

	- con archivos manejados a través de un Conector Nativo.

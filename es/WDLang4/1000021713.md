
## hbDeleteTable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una HBase tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Delete the table
hbDeleteTable(Connection, "blog")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = hbDeleteTable(<Connection> , <Table>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase.

**`<Table>: (Cadena de caracteres)`**

Nombre de HBase tabla a borrar.

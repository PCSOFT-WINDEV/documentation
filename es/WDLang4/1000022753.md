


## &lt;Variable Connection&gt;.SaveParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un valor persistente. Este valor puede ser leído por [&lt;Variable Connection&gt;.LoadParameter](../WDLang4/1000022649.md).  

Observación: Los parámetros se guardan por base de datos en el archivo "user_param.fic".


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save a value 
MyConnection is Connection
MyConnection.SaveParameter("MyParameter", 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

### Guardar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = <Connection>.SaveParameter(<Parameter> , <Parameter value>)`
---

**`<Result>: (Booleano)`**



- <u><u><u>True</u></u></u> si se realizó la copia de seguridad,

- <u><u><u>False</u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a guardar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Parameter value>: (Tipo del valor a guardar)`**

Valor a ahorrar.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tipo de variables guardadas
<a name="tipo_variables_guardadas_ELTPARAGRAPHE000230"></a>

**&lt;Variable Connection&gt;.SaveParameter** se utiliza para guardar las variables simples (cadena, entero, ...).

Para guardar variables complejas (objetos, estructuras, etc.), estas variables deben ser serializadas como cadenas utilizando [Serialize](../WDLang1/3013065.md).

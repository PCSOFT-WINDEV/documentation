
## SQLClose( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Declara el fin de la ejecución de una consulta y libera los recursos de memoria asignados durante la ejecución de la consulta ([SQLExec](../WDLang4/3072007.md) o [SQLExecWDR](../WDLang4/3072010.md)).

**Atención**: La consulta debe ser necesariamente liberada aunque la ejecución de la consulta haya fallado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Run the query and retrieve the information
ResSQL = SQLExec("SELECT* FROM INV", "QRY1")
IF ResSQL = False THEN
	SQLInfo("QRY1")
	Info("SQL error: " + SQL.MesError)
ELSE
	SQLTable("QRY1", TABLE_Table1)
END
SQLClose("QRY1")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SQLClose(<Query name>)`
---

**`<Query name>: (Cadena de caracteres)`**

Nombre de la consulta: 

- creado y dirigido por [SQLExec](../WDLang4/3072007.md),

- dirigido por [SQLExecWDR](../WDLang4/3072010.md).




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las consultas creadas con el editor de consultas deben especificarse en el formato de una cadena de caracteres.




## WithSpace( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade o elimina los espacios que se encuentran a la derecha de un elemento de texto al leerlo.

**Atención**: Esta función se tiene en cuenta para los ficheros de datos para los que el "modo 5.5-compatible: los elementos de texto rellenos de espacios" se comprobaron en la editor de análisis (pestaña "Detalles" de la descripción del archivo de datos).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003044142&name=withspace_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Manage significant spaces (add spaces) for all the files
WithSpace(True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WithSpace([<File name>] [, <Item name>] , <Options>)`
---

**`<Result>: (booleano)`**

Antiguo modo de gestión de espacios significativos.

**`<File name>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este parámetro, **WithSpace** se manipular **todos los archivos de datos**.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta.

**`<Item name>: (Cadena de caracteres opcional)`**

Nombre del campo utilizado. Si no se especifica este parámetro, **WithSpace** se manipular **todos los elementos** del archivo de datos especificado.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta.

**`<Options>: (booleano)`**



- <u><u><u><u>True</u></u></u></u>: el contenido de las posiciones de texto se rellena con espacios a la derecha. 

- <u><u><u><u>False</u></u></u></u>: se eliminan los espacios que se encuentran a la derecha de los elementos de texto.


![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) La gestión de los espacios significativos se realizará para todos los ficheros de datos y para todos los elementos de texto que se encuentren en estos ficheros de datos.

## Observaciones


### Generalizar la gestión de espacios
<a name="generalizar_gestion_espacios_ELTPARAGRAPHE000082"></a>

- Para aplicar el modo de gestión de espacios a **todos los elementos de un archivo de datos**, el parámetro &lt;Nombre del elemento&gt; debe ser igual a "\*". 
	Ejemplo: 
	
	```txt
	WithSpace(<File name>, "*", <Options>)
	```


- Para aplicar el modo de gestión de espacios a **todos los elementos de todos los archivos de datos**, los parámetros &lt;Nombre del archivo&gt; y &lt;Nombre del elemento&gt; deben ser iguales a "\*". 
	Ejemplo: 
	
	```txt
	WithSpace("*", "*", <Options>)
	```
En este caso, **WithSpace** devuelve <u><u><u><u>False</u></u></u></u> si la gestión de los espacios fue diferente en al menos uno de los elementos relevantes.

- Si sólo se especifica el parámetro &lt;Opciones&gt;, el modo de gestión especificado se aplica a todos los archivos de datos y a todos los elementos.





### Función de replicación y WithSpace
<a name="funcion_replicacion_withspace_ELTPARAGRAPHE000104"></a>

La replicación no es compatible con la función **WithSpace** en el código de la aplicación cliente. Esta función cambia el comportamiento de HFSQL. El uso de esta función en la aplicación cliente puede afectar a la eficacia de la replicación..
A partir de la versión 21 actualización 3 (versión 210065), las funciones de sincronización (replicación universal programada o asistida) devuelven un error a la aplicación cliente si se utiliza **WithSpace** en al menos uno de los ficheros replicados.

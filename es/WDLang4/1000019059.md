
## HClusterState( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de un cluster HFSQL interrogando a su coordinador.

**Observación**: Esta función está reservada para un uso avanzado. Le recomendamos que utilice el HFSQL control Center para gestionar un cluster HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF HClusterState("clustermanager.mydomain.com", "secret") = 5 THEN
	STC_Alert = "CAUTION: the cluster is waiting for synchronization!"
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterState(<Coordinator> , <Password>)`
---

**`<Result>: (Integro)`**

Status del cluster. Los valores posibles son los siguientes:

| Valor | Estado |
| --- | --- |
| 0 | Se produjo un error durante la llamada al coordinador.<br><br>[ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error. |
| 1 | El clúster está detenido.<br><br>Puede ser reiniciado por [HClusterStart](../WDLang4/1000019057.md). |
| 2 | El clúster ha sido iniciado.<br><br>Se puede suspender por [HClusterStop](../WDLang4/1000019058.md). |
| 3 | El clúster se está deteniendo. |
| 4 | El clúster se está iniciando. |
| 5 | Todos los nodos del cluster actualmente conectados al coordinador están a la espera de ser sincronizados. |



**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

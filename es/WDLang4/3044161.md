
## HDBSortType( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Retorna o modifica la secuencia de elementos de texto en los archivos xBase.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nSortType is int
// Retrieves the sort order
nSortType = HDBSortType()

// Changes the type of sort
SWITCH nSortType
	CASE hDBASCII: Info( "ASCII sort" )
	CASE hDBLexicographic
			  HDBSortType(HDBASCII)
			  Info( "Lexicographic sort changed to ASCII sort" )
	OTHER CASE
			  Info("Error: Unknown sort")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificando la secuencia de orden

`<Result> = HDBSortType(<Type of sort>)`
---

**`<Result>: (Constante)`**

Tipo de orden de clasificación utilizado antes de cambiar el orden:


|   |   |
| --- | --- |
| *hDBASCII* | Clasificar según el orden ASCII. |
| *hDBLexicographic* | De tipo lexicográfico. |



**`<Type of sort>: (Constante)`**

Tipo de ordenación a utilizar:


|   |   |
| --- | --- |
| *hDBASCII*<br>(Valor predeterminado) | Clasificar según el orden ASCII. |
| *hDBLexicographic* | De tipo lexicográfico. |




<a name="SYNTAX2"></a>

### Averiguar la secuencia de orden

`<Result> = HDBSortType()`
---

**`<Result>: (Constante)`**

Tipo de orden que se utiliza actualmente:


|   |   |
| --- | --- |
| *hDBASCII*<br>(Valor predeterminado) | Clasificar según el orden ASCII. |
| *hDBLexicographic* | De tipo lexicográfico. |

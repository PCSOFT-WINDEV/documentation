


## &lt;Variable Connection&gt;.DeleteParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un parámetro previamente guardado con [&lt;Variable Connection&gt;.SaveParameter](../WDLang4/1000022753.md). 
 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

MyConnection.DeleteParameter("MyParameter")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

### Eliminar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = <Connection>.DeleteParameter(<Parameter>)`
---

**`<Result>: (Booleano)`**



- <u><u><u>True</u></u></u> si se realizó la eliminación correctamente,

- <u><u><u>False</u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a borrar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Retorno de carro)..

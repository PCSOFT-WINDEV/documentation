
## MemcachedCAS( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el valor de una tecla si ésta no fue modificada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MemcachedCAS(memConnection, "key", "value")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MemcachedCAS(<Connection> , <Key> , <Value> , <Cas> [, <Expiration> [, <Additional value>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Connection>: (memcachedConnection Variable)`**

Nombre del [memcachedConnection](../WDLang4/1000023339.md) Variable que corresponde a la conexión con Memcached. 

**`<Key>: (Cadena de caracteres)`**

Clave para modificar. 

**`<Value>: (Cadena de caracteres o memoria intermedia)`**

Valor a almacenar. 

**`<Cas>: (Integro)`**

Valor de la versión anterior. 

**`<Expiration>: (Duración entera u opcional)`**

Duración del almacenamiento de la clave en la memoria caché. Esta duración se expresa en segundos. Esta duración puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration. 

- la indicación directa de la duración (por ejemplo, "10s"). 


Esta duración es set a 0 por Default. En este caso, la duración es ilimitada: si no hay suficiente memoria, las teclas más antiguas se borran.

**`<Additional value>: (Entero opcional)`**

Se utiliza para almacenar un valor adicional associated con la pareja Llave/Valor. Este parámetro puede utilizarse para describir la clave o el valor, por ejemplo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para utilizar esta función, el servidor Memcached debe admitir CAS (**SupportCAS** Property de la variable [memcachedConnection](../WDLang4/1000023339.md) establecida en <u><u><u><u>True</u></u></u></u>). CAS (Check And set) se utiliza para realizar una modificación de datos sólo si estos datos no fueron modificados desde su última lectura.

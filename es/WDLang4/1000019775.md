
## HListCustomFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de carpetas personalizadas (también llamadas grupos) definidas en el análisis. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(HListCustomFolder())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HListCustomFolder([<Path of the WDD file> [, <Password of the WDD file>]])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de carpetas personalizadas definidas en el análisis actual o en el análisis especificado. Los nombres de las carpetas personalizadas están separados por caracteres CR (Carriage Return). El resultado tiene el siguiente formato: 

```txt
<Custom-Folder 1> + CR + <Custom-Folder 2> + ... + <Custom-Folder N>
```


**`<Path of the WDD file>: (Cadena de caracteres opcional)`**

Ruta completa del análisis (es decir, del archivo .WDD) a utilizar. Si este parámetro no se especifica o si es igual a una cadena vacía ("""), se utiliza el análisis actual.

**`<Password of the WDD file>: (Cadena de caracteres opcional)`**

Contraseña definida para el análisis. Si no se define ninguna contraseña, este parámetro debe ser igual a una cadena vacía ("").

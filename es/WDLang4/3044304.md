## Manta server
<a name="manta_server_ELTTEXTE000312"></a>
El **Servidor Manta** incluye:

- el servicio **MantaManager** utilizado para comunicarse con la aplicación "HFSQL control Center" (herramienta de gestión remota) y para detener y/o iniciar los servidores.

- el servicio **Manta**.. 
	Por Default, un solo servicio **Manta** se encuentra en el servidor. Sin embargo, se pueden encontrar varios servicios **Manta** en el mismo servidor. Esta configuración le permite asociar una aplicación WINDEV/WEBDEV (o un pequeño número de aplicaciones WINDEV/WEBDEV) a un servicio **Manta** por ejemplo.. Por lo tanto, si una aplicación sobrecarga el servicio Manta, sólo las aplicaciones associated con este servicio se bloquearán o ralentizarán.
	Repita esta acción para los controles "reference" y "Bar code".
	Observación: Si se produce un fallo, el servicio se reinicia automáticamente.

## ¿Dónde está instalado el servidor de Manta?
<a name="¿donde_esta_instalado_servidor_manta_ELTTEXTE000342"></a>


### ¿Cómo instalar HFSQL Client/Server?
<a name="¿como_instalar_hfsql_clientserver_ELTPARAGRAPHE000037"></a>

La instalación de la HFSQL Client/Server está disponible:

- En el paquete instalación de WINDEV, WEBDEV y WINDEV Mobile.

- En el directorio INSTALL de WINDEV, WEBDEV y WINDEV Mobile.

- a través del sistema integrado de Debian.

- a través del sistema integrado de RedHat.

- a través de Docker.




El servidor HFSQL Client/Server se puede instalar con sus aplicaciones. Todo lo que tienes que hacer es incluirlo al crear el programa instalación.

**Observación**: El servidor de Manta debe ser instalado en un ordenador con una versión de servidor de Windows (todas las versiones soportadas por Microsoft). El servidor Manta también puede instalarse en un ordenador con Linux (procesador x86 o compatible).
<a name="NOTE2_2"></a>


### Configuración en Linux
<a name="configuracion_linux_ELTPARAGRAPHE000060"></a>

Para instalar el servidor HFSQL Client/Server en un servidor LINUX: 

1. copy los siguientes archivos al servidor: 

	- manta_install

	- mantainstall.zip

	- README
			Estos archivos están localizados en el subdirectorio "HFSQL Server\\Linux4" del paquete instalación. 




2. Conéctese al servidor como "root"

3. Ejecute el archivo manta_install. Es posible: 

	- Instalar un nuevo servidor.

	- Actualizar un servidor existente. 

	- Desinstalar un servidor.




4. Siga las instrucciones que aparecen en la pantalla. 


**Atención**: La licencia debe desplazarse completamente para mostrar las siguientes pantallas. 

Para instalar el servidor HFSQL Client/Server de 64 bits en un servidor LINUX de 64 bits: 

1. copy los siguientes archivos al servidor: 

	- manta_install64

	- mantainstall.zip

	- README
			Estos archivos están localizados en el subdirectorio "HFSQL Server\\Linux44_64" del paquete instalación.




2. Conéctese al servidor como "root"

3. Run manta_install64. Es posible: 

	- Instalar un nuevo servidor.

	- Actualizar un servidor existente. 

	- Desinstalar un servidor.




4. Siga las instrucciones que aparecen en la pantalla. 


**Atención**: La licencia debe desplazarse completamente para mostrar las siguientes pantallas.

### Configuración a través del sistema Debian integrado
<a name="configuracion_traves_del_sistema_debian_integrado_ELTPARAGRAPHE000105"></a>Para usar este modo de instalación:
Ejecute la siguiente acción (a partir de Ubuntu 22.04 y Debian 11): 
```txt
curl -sS https://package.windev.com/keys/hfsql_public.asc | \
gpg --dearmor | \
sudo tee /usr/share/keyrings/pcsoft.gpg > /dev/null

echo "deb [signed-by=/usr/share/keyrings/pcsoft.gpg] http://package.windev.com/fr/debian/ debian main" | \
sudo tee /etc/apt/sources.list.d/pcsoft.list

sudo apt update
sudo apt install hfsql
```


Para las distribuciones de Linux más antiguas, puede ejecutar la siguiente acción: 
```txt
curl -sSL https://package.windev.com/keys/hfsql_public.asc | sudo apt-key add -
sudo echo "deb http://package.windev.com/fr/debian/ debian main" 
	> /etc/apt/sources.list.d/pcsoft.list
sudo apt-get update
sudo apt-get install hfsql
```


**Atención**: Si usted get un error en el siguiente formato: "gpg: falló al iniciar el dirmngr" entonces ejecute el siguiente comando: 

```txt
sudo apt-get install dirmngr
```


Observación: Este modo de instalación funciona con las distribuciones de Debian versión 8 y posteriores.

### Configuración a través del sistema integrado de RedHat y sus derivados
<a name="configuracion_traves_del_sistema_integrado_redhat_sus_derivados_ELTPARAGRAPHE000125"></a>Para utilizar este modo de instalación, realice las siguientes acciones:


```txt

sudo dnf install hfsql
sudo systemctl start hfsql
```

## Centro de Control HFSQL: Herramienta de administración remota de servidores
<a name="centro_control_hfsql_herramienta_administracion_remota_servidores_ELTTEXTE000396"></a>
El **HFSQL control Center** se utiliza para:

- gestionar los diferentes servidores utilizados.

- gestionar los usuarios y los grupos de usuarios de las bases de datos.

- ver y/o modificar el contenido de una base de datos.

- realizar búsquedas en una base de datos.

- ejecutar consultas en una base de datos.

- ver las conexiones actuales a una base de datos o a un archivo de datos.

- detener, iniciar, instalar y/o desinstalar un servidor.

- ...

## Desinstalación de los servicios de un servidor
<a name="desinstalacion_los_servicios_servidor_ELTTEXTE000444"></a>


### Los servicios de servidor se pueden desinstalar
<a name="los_servicios_servidor_pueden_desinstalar_ELTPARAGRAPHE000262"></a>

- a través del paquete instalación del módulo del servidor HFSQL. Los servicios a desinstalar son Manta y MantaManager.

- directamente desde el panel de control de Windows (opción "Agregar o quitar programas").

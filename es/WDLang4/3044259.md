
## HBuildKeyValueANSI (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**Independientemente de la plataforma utilizada, los datos almacenados en los archivos HFSQL están en formato ANSI.**

En una plataforma UNICODE, **HBuildKeyValueANSI** construye el valor de una clave compuesta para almacenarlo en un archivo de datos HFSQL.

En una plataforma ANSI, esta función equivale a [HBuildKeyValue](../WDLang4/3044258.md).

**La función HBuildKeyValueANSI no debe utilizarse** para:

- definir un filtro (función [HFilter](../WDLang4/3044100.md)) en una clave compuesta.

- realizar una búsqueda (funciones [HReadSeekFirst](../WDLang4/3044036.md), [HReadSeekLast](../WDLang4/3044034.md), etc.) en una clave compuesta.




Para los filtros o las búsquedas, utilice la función [HBuildKeyValue](../WDLang4/3044258.md).

**Recordatorio**: Las claves compuestas son cadenas binarias cuyo contenido no se puede mostrar directamente. Para obtener más información, consulte [Claves compuestas](../WDLang4/3044178.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Assign an item in a linked data file
Invoice.CustLastNameFirstName = HBuildKeyValueANSI(Customer, ...
	CustLastNameFirstName, Customer.LastName, Customer.FirstName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HBuildKeyValueANSI(<Data file> , <Composite key> , <Value of components>)`
---

**`<Result>: (Cadena de caracteres)`**

Valor de la clave compuesta a guardar en el archivo de datos.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos HFSQL utilizado.

**`<Composite key>: (Cadena de caracteres)`**

Nombre del elemento que corresponde a la clave compuesta. 

**`<Value of components>: (Cadena de caracteres)`**

Valor que se asignará a cada componente de la clave compuesta. 
Este parámetro tiene el siguiente formato: &lt;Componente 1&gt; , &lt;Componente 2&gt;, ...,&lt;Componente N&gt;. 
Por ejemplo: "MEDINA", "Felipe".

### ¿Cuándo utilizar la función HBuildKeyValueANSI?
<a name="¿cuando_utilizar_funcion_docparampagetitleshort_ELTPARAGRAPHE000087"></a>

**HBuildKeyValueANSI** puede utilizarse para almacenar el valor de una clave compuesta utilizada como clave de enlace en un archivo de datos.

Ejemplo:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HconstruitValcleANSI.gif)


Para añadir una nueva factura en una plataforma UNICODE, utilice el siguiente código:


```wl
Invoice.InvoiceNum = ... 
...
Invoice.CustLastNameFirstName = HBuildKeyValueANSI(Customer, ...
	CustLastNameFirstName, Customer.LastName, Customer.FirstName)
...
HAdd(Invoice)
```

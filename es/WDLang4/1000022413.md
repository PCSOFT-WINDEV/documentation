
## mongoCollectionOption (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoCollectionOption** permite definir las características avanzadas de las opciones de una colección de una base de datos MongoDB. Las características de estas opciones se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Option is mongoCollectionOption
Option.MaxSize = 100*1024*1024
MongoCreateCollection(MDBDatabase, "product", Option)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables mongoCollectionOption
<a name="propiedades_especificas_las_variables_mongocollectionoption_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden ser utilizadas para manipular las opciones de colección:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| MaxNbDocumento | Integro | Número máximo de documentos en la colección. Los documentos más antiguos se eliminarán cuando el número de documentos de la colección alcance este límite.. <br><br>La colección no está limitada si este Property no está lleno. |
| MaxSize | Integro | Tamaño máximo de una colección. Los documentos más antiguos se eliminarán cuando el tamaño de la colección alcance este límite.<br><br>La colección no está limitada si este Property no está lleno. |
| Motor de almacenamiento | String de caracteres, memoria intermedia o variante | Document en formato JSON/BSON/variante que describe la lista de opciones pasadas al motor de almacenamiento.. <br><br>Para más detalles, consulte la documentación sobre MongoDB. |
| Nivel de validación | Cadena de caracteres | Nivel de validación cuando se modifica un Document existente. Las opciones disponibles son: <br><br>- "apagado",<br><br>- "estricto",<br><br>- "moderado".<br><br><br>Para más detalles, consulte la documentación sobre MongoDB. |
| Validator | String de caracteres, memoria intermedia o variante | Document en formato JSON/BSON/variante que describe las condiciones de validación de los Document s que se añadirán a la colección. <br><br>Para más detalles, consulte la documentación sobre MongoDB. |

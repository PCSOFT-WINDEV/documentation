


## &lt;mongoGridFS Variable&gt;.DeleteFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un archivo identificado por su nombre de un sistema de archivos MongoDB.

**Observación**: Para eliminar un archivo identificado por su identificador, utilice [&lt;mongoGridFS Variable&gt;.DeleteFileByID](../WDLang4/1000025395.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="Example2"></a>

```wl
gclGridFS is mongoGridFS

// Delete the file found in the table column
gclGridFS.DeleteFile(TABLE_GridFS.COL_Name)

ToastDisplay("Deletion successfully performed.")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <File system>.DeleteFile(<File name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<File system>: (mongoGridFS Variable)`**

Nombre del [mongoGridFS](../WDLang4/1000022416.md) Variable que corresponde al sistema de archivos Mongo a utilizar.  

**`<File name>: (Cadena de caracteres)`**

Nombre del archivo a eliminar.

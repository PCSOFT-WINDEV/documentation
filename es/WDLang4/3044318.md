
## HStartServer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite iniciar un servidor HFSQL (esta función utiliza el servicio MantaManager).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Restart the server 
HStartServer("CSServer")

// Restart the server by using a port number
HStartServer("CSServer:4901")

// Restart the server by using a server name
HStartServer("CSServer", "HFSQL server")
```


<a name="Example2"></a>



```wl
// Restart the server by using a connection variable

Cnt_HFSQL is Connection
Cnt_HFSQL.Provider = hAccessHFClientServer
Cnt_HFSQL.User = "admin"
Cnt_HFSQL.Password = ""
Cnt_HFSQL.Server = "GB:4916"
Cnt_HFSQL.Database = "CRM"

HStartServer(Cnt_HFSQL.Server)

SWITCH HServerStatus(Cnt_HFSQL.Server)
	CASE hStopped: Info("Server stopped")
	CASE hStarted: Info("Server started")
	CASE hInaccessible: Info("Server not available")
END
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Reiniciar un servidor (con capacidad para especificar el puerto del servidor HFSQL)

`<Result> = HStartServer(<Server>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se reinició el servidor,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Server>: (Cadena de caracteres)`**

Nombre del servidor. Se puede especificar el número de puerto utilizado por el servidor HFSQL. De forma predeterminada, se utiliza el puerto 4900. Si el número de puerto del servidor HFSQL es diferente, utilice la siguiente notación:

&lt;Nombre de equipo&gt;: &lt;Número de puerto&gt;

Ejemplo: "NombreEquipo:4901"

El número de puerto corresponde al número de puerto utilizado en [HOpenConnection](../WDLang4/3044107.md).  


<a name="SYNTAX2"></a>

### Reiniciar un servidor mientras se especifica el nombre del servidor HFSQL

`<Result> = HStartServer(<Server> , <HFSQL server>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se reinició el servidor,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Server>: (Cadena de caracteres)`**

Nombre del servidor.

**`<HFSQL server>: (Cadena de caracteres)`**

Nombre del servidor HFSQL a reiniciar (este nombre aparece en el Centro HFSQL control).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Reiniciar un servidor
<a name="reiniciar_servidor_ELTPARAGRAPHE000079"></a>

- Si el servidor ya está iniciado cuando se llama a **HStartServer**, el servidor no se reinicia. [HServerStatus](../WDLang4/3044319.md) devuelve el estado de un servidor.

- No se requieren derechos específicos para iniciar un servidor.




**Observación**: El servidor no se puede reiniciar inmediatamente una vez que se ha detenido.. El sistema operativo no cierra inmediatamente las conexiones ya establecidas entre los ordenadores cliente y el servidor. Simplemente espere unos minutos (dependiendo del tiempo de espera del sistema operativo) antes de reiniciar el servidor.


## HTransactionInProgress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar si una transacción está en curso.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Tries to determine if a transaction is in progress
// ISAM database (HFSQL Classic)
bConnectionInProgress is boolean = HTransactionInProgress()
```



```wl
// Tries to determine if a transaction with MyConnection is in progress
// Client/Server database (HFSQL Client/Server)
MyConnection is Connection
bConnectionInProgress is boolean = HTransactionInProgress(MyConnection)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HTransactionInProgress([<Connection>])`
---

**`<Result>: (booleano)`**

**Si se especifica una conexión**: 

- <u><u><u><u>True</u></u></u></u> si hay una transacción en la conexión especificada o si hay una transacción global (para HFSQL Classic (ISAM) y todas las conexiones HFSQL Client/Server).

- <u><u><u><u>False</u></u></u></u> si no hay ninguna transacción en curso en la conexión especificada. 


**Si no se especifica ninguna conexión**: 

- <u><u><u><u>True</u></u></u></u> si hay una transacción global (para HFSQL Classic (ISAM) y todas las conexiones HFSQL Client/Server).

- <u><u><u><u>False</u></u></u></u> en caso contrario. Atención: puede haber transacciones en curso sobre conexiones específicas.  




**`<Connection>: (Cadena de caracteres opcional o Conexión opcional Variable)`**

Conexión a utilizar. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

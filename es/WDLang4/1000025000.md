


## &lt;Fuente&gt;.LogInfo (Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega comentarios en el registro al guardar la operación. Estos comentarios se utilizarán para todas las operaciones realizadas en el contexto HFSQL actual, hasta que se vuelva a llamar a **&lt;Fuente&gt;.LogInfo**.

**Observación**: El registro Process no está disponible cuando se accede a una base de datos a través de .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Customer.LogInfo("Adding " + Customer.CustomerName)
Customer.Add()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX2"></a>

### Definir un comentario específico por cada archivo de datos con registro

`<Result> = <Source>.LogInfo(<Message>)`
---

**`<Result>: (Booleano)`**



- <u><u><u>True</u></u></u> si se realizó la operación,

- <u><u><u>False</u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos registrado con el que se relaciona el mensaje. 

**`<Message>: (Cadena de caracteres)`**

Mensaje que se debe insertar en la información de registro (hasta 16 caracteres, la información se truncará si la cadena es más larga).. Esta información puede ser recuperada por [WDLog](../WDJournal/3516012.md). 
No se inserta ningún mensaje si no se especifica este parámetro..


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para insertar diferentes tipos de información durante cada operación, se debe llamar a **&lt;Fuente&gt;.LogInfo** antes de cada una de ellas.




## HModifyGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la información del grupo en función de los elementos que se encuentran en la correspondiente [variables para la gestión de grupos](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
...
// Modify the members of the group
HGroup.Users = "Flo" + CR + "Julia" + CR + "Luke"
// Modify the group
IF HModifyGroup("MyConnection", "TESTERS") = True THEN
	Info("Group modified")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyGroup(<Connection> , <Group>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el grupo,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que se debe modificar un grupo. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Group>: (Cadena de caracteres)`**

Nombre del grupo cuyas características deben modificarse.

<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios para modificar un grupo
<a name="derechos_necesarios_para_modificar_grupo_ELTPARAGRAPHE000120"></a>

Para modificar un grupo (que no sea el del usuario actual), debe tener los permisos para ver usuarios (constante *hRightsSeeUser*). De lo contrario, la función **HModifyGroup** devolverá <u><u><u><u>False</u></u></u></u> y se mostrará el mensaje de error "Grupo desconocido".

Para modificar el nombre de un grupo, se deben tener permisos para ver a los usuarios (constante *hRightsSeeUser*) y permisos para administrarlos (constante *hRightsManageUser*).

Observación: Estas constantes se utilizan en [HInfoServerRights](../WDLang4/3044322.md) y [HModifyServerRights](../WDLang4/3044328.md).
<a name="NOTE0_2"></a>


### Gestión de derechos
<a name="gestion_derechos_ELTPARAGRAPHE000145"></a>

Las siguientes funciones se utilizan para conceder derechos a un grupo:

- [HModifyDatabaseRights](../WDLang4/3044326.md): se utiliza para conceder derechos sobre una base de datos.

- [HModifyFileRights](../WDLang4/3044327.md): se utiliza para conceder derechos sobre un archivo de datos.

- [HModifyServerRights](../WDLang4/3044328.md): se utiliza para conceder derechos sobre un servidor.




Para obtener más información, consulte [Gestión de los derechos en HFSQL Client/Server](../WDLang4/3044333.md).

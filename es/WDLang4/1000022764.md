


## &lt;Variable Connection&gt;.DeleteQuery (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una consulta (utilizada por procedimientos almacenados) de un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

// Delete the "ManageEmail" query from the server
MyConnection.DeleteQuery(ManageEmail)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteQuery(<Query to delete>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la consulta,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión con el servidor HFSQL que contiene la consulta a eliminar.

**`<Query to delete>: (Cadena de caracteres)`**

Nombre de la consulta que se debe eliminar del servidor.

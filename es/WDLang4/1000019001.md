


## HDeleteDirectory (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un directorio de una base de datos HFSQL Client/Server. Este directorio debe estar vacío. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

HDeleteFile(MyConnection, "ModuleCRM\Prospect.fic")
HDeleteDirectory(MyConnection, "ModuleCRM")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HDeleteDirectory(<Connection> , <Directory to delete>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el directorio,

- <u><u><u><u>False</u></u></u></u> si se produce un error (por ejemplo, si el directorio no existe en el servidor o no está vacío). La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión en la que se debe eliminar el directorio. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Directory to delete>: (Cadena de caracteres)`**

Ruta del directorio a eliminar. Este ruta es relativo a la base de datos: el nombre de la base de datos no debe especificarse en el ruta. Este directorio debe estar vacío.

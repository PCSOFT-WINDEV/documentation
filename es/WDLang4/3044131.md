


## HIndexingInProgress (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si se está realizando una reindexación en un archivo de datos HFSQL y devuelve el porcentaje de la reindexación realizada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF HIndexingInProgress(Customer) <> 0 THEN
	Trace("Operation unavailable, reindexing")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HIndexingInProgress(<Data file> [, <Progress Bar>])`
---

**`<Result>: (Integro)`**



- 0 si no hay ninguna operación de reindexación en curso en el archivo de datos,

- un número entero comprendido entre 1 y 100 que indica el porcentaje de reindexación ya realizada




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Progress Bar>: (Nombre de ventana, control o entero opcional)`**



- Nombre de la ventana en la que se va a mostrar el indicador, o nombre del campo del tipo de indicador.

- Mango de la ventana que muestra la barra de progreso.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Reindexación
<a name="reindexacion_ELTPARAGRAPHE000093"></a>Los índices se reorganizan mediante la operación de reindexación. No se puede acceder a los datos durante la operación de reindexación.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Cuando se accede a un archivo de datos en el que se realiza una operación de reindexación, el motor HFSQL muestra una ventana que indica que se está realizando una operación de reindexación. Esta ventana se cerrará automáticamente al final de la reindexación.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HF_Reindexation.gif)
 Para más detalles, véase [Gestión asistida de errores HFSQL](../WDLang4/3044188.md) sobre esta ventana y su personalización.

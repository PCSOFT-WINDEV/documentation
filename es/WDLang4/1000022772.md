


## &lt;Variable Connection&gt;.CheckServerUpdate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si hay una versión más reciente del servidor HFSQL que pueda actualizarse automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

let sNewVersion = MyConnection.CheckServerUpdate()
If sNewVersion<>"" THEN Info(sNewVersion)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.CheckServerUpdate()`
---

**`<Result>: (Character string)`**

Versión de la actualización disponible en el servidor HFSQL.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Connection](../WDLang4/1514073.md) que describe la conexión con el servidor HFSQL.




## Observaciones
El servidor de actualización (que contiene el paquete para actualizar el servidor HFSQL) puede ser definido con la función [&lt;Variable Connection&gt;.ManageServerUpdate](../WDLang4/1000022662.md).

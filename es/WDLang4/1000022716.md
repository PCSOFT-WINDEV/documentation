


## &lt;Variable Connection&gt;.ModifyScheduling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una tarea programada en un servidor HFSQL: 

- procedimiento almacenado, 

- refuerzos,

- optimización, 

- actualización de una vista materializada.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyDatabaseConnection is Connection

// Delete all scheduled backups
arrBckDesc is array dynamic of hBackupDescription 
arrBckDesc = MyDatabaseConnection.ListScheduling(hlpBackup)
hBckDesc is hBackupDescription
FOR EACH hBckDesc OF arrBckDesc
	IF hBckDesc.LimitNumberBackups = 3 THEN 
		hBckDesc.LimitNumberBackups = 5
		MyDatabaseConnection.ModifyScheduling(hBckDesc)
	END
END
```

## Sintaxis
<a name="SYNTAX1"></a>

### Modificar una tarea programada

`<Result> = <Connection>.ModifyScheduling(<Scheduled task>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la tarea programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que debe modificarse una tarea programada.

**`<Scheduled task>: (Variable de tipo hScheduledTask)`**

Nombre de la variable [hScheduledTask](../WDLang4/1000017448.md) que describe la tarea que desencadena un Procedure almacenado.


<a name="SYNTAX2"></a>

### Modificar una tarea de optimización programada

`<Result> = <Connection>.ModifyScheduling(<Scheduled optimization task>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la tarea de optimización programada

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

**`<Scheduled optimization task>: (Variable de tipo hScheduledOptimization)`**

Nombre de la [hScheduledOptimization](../WDLang4/1000018854.md) Variable utilizada.

- <u><u><u><u>True</u></u></u></u> si se modificó la planificación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que hay que modificar la programación de una vista materializada actualización.

Nombre de la variable [hScheduleMaterializedView](../WDLang4/1000021444.md) que contiene el description de las vistas materializadas a actualización.

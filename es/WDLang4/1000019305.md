
## HClusterParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee y modifica los parámetros de un clúster HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the DNS name of server
sDNS is string 
sDNS = HClusterParameter("clustermanager.mydomain.com", "secret", hCPDNS)

InfoBuild("Use <%1> as server address in your client applications" + ...
    " to connect to the cluster.", sDNS)
```




## Sintaxis

### Averiguar el valor de un parámetro del cúmulo

`<Result> = HClusterParameter(<Coordinator> , <Password> , <Parameter>)`
---

**`<Result>: (Cadena de caracteres)`**

Valor del parámetro. 

**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

**`<Parameter>: (Constante de tipo Integer)`**

Parámetro a leer. Los valores posibles son:


|   |   |
| --- | --- |
| *hCPClientPort* | Puerto de escucha de las conexiones del cliente. |
| *hCPDNS* | Nombre DNS del cluster. Este parámetro es meramente informativo.<br><br>**Observación**: Es importante no confundir:<br><br>- el nombre que identifica el cúmulo (correspondiente al valor associated con este parámetro): que se usará para acceder a las bases de datos del clúster.<br><br>- el nombre que identifica al coordinador (utilizado en el parámetro &lt;Coordinador&gt;): Nombre DNS del ordenador donde se instaló el coordinador del clúster.<br><br>- el nombre que identifica al nodo (no utilizado por esta función): Nombre DNS del ordenador donde se instaló el nuevo servidor HFSQL.<br><br><br> |





### Modificar el valor de un parámetro de racimo

`<Result> = HClusterParameter(<Coordinator> , <Password> , <Parameter> [, <New value>])`
---

**`<Result>: (Cadena de caracteres)`**

Valor anterior del parámetro.

**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

**`<Parameter>: (Constante de tipo Integer)`**

Parámetro a modificar. Los valores posibles son:


|   |   |
| --- | --- |
| *hCPClientPort* | Puerto de escucha de las conexiones del cliente.<br><br>**Observación**: La modificación de este parámetro se tendrá en cuenta al reiniciar el cluster. |
| *hCPDNS* | Nombre DNS del cluster. Este parámetro es meramente informativo.<br><br>**Observación**: Es importante no confundir:<br><br>- el nombre que identifica el cúmulo (correspondiente al valor associated con este parámetro): que se usará para acceder a las bases de datos del clúster.<br><br>- el nombre que identifica al coordinador (utilizado en el parámetro &lt;Coordinador&gt;): Nombre DNS del ordenador donde se instaló el coordinador del clúster.<br><br>- el nombre que identifica al nodo (no utilizado por esta función): Nombre DNS del ordenador donde se instaló el nuevo servidor HFSQL.<br><br><br> |



**`<New value>: (Cadena de caracteres opcional)`**

Nuevo valor para el parámetro.

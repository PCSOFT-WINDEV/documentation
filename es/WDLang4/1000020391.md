


## HNotifListCCRecipiente (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de destinatarios de una Notification enviada a través de los Centros control (herramienta de mensajería de WDBal). Esta notificación es enviada por el servidor HFSQL cuando ocurre un problema. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ctCSConnection is Connection
ctCSConnection.Provider = hAccessHFClientServer
ctCSConnection.User = "admin"
ctCSConnection.Password = ""
ctCSConnection.Server = "test:4900"
HOpenConnection(ctCSConnection)

ListEmailRecip is string
ListEmailRecip = HNotifListCCRecipient(ctCSConnection)

FOR EACH STRING ARecipient OF ListEmailRecip SEPARATED BY CR
	sRecipient is string = ExtractString(ARecipient, 1, TAB)
	Notif is int = ExtractString(ARecipient, 2, TAB)
	NotifLevel is string
	SWITCH Notif
		CASE hNotifError: NotifLevel = "Minimum level: Errors"
		CASE hNotifWarning: NotifLevel = "Minimum level: Warning"
		CASE hNotifCritical: NotifLevel = "Minimum level: Critical"			
	END
	Trace(sRecipient + TAB + NotifLevel)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HNotifListCCRecipient(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de beneficiarios de Notification a través de los Centros control. La lista tiene el siguiente formato: 

```txt
<Recipient Name> + TAB + <Notification Level> + CR + ...
```
donde: 

- &lt;Nombre del destinatario&gt;&gt;Recipiente: cadena de caracteres correspondiente al nombre del destinatario de Notification. 

- Nivel de la Escalera&gt; &lt;Notification: constante entera que corresponde al nivel de gravedad mínimo del problema desde el que se envía una Notification. Los valores disponibles son los siguientes: 
	


|   |   |
| --- | --- |
| *hNotifCritical* | Las notificaciones se enviarán desde el nivel crítico (para los problemas críticos y de error). |
| *hNotifError* | Las notificaciones se enviarán sólo para los problemas de error. |
| *hNotifWarning* | Las notificaciones se enviarán desde el nivel de alerta (para los problemas de advertencia, críticos y de error). |






**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

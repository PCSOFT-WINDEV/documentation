
## HInitHF55( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite acceder a funciones específicas para la gestión de datos de xBase (o dBase) si el Acceso nativo a xBase proporcionado con WINDEV o WEBDEV no está disponible.

Observaciones: 

- Le aconsejamos que utilice el acceso nativo a xBase proporcionado con WINDEV.

- Esta función no está disponible cuando se utiliza un ejecutable independiente.. 




**Para acceder a un archivo xBase sin utilizar el acceso estándar Native xBase:**

- **WINDEV o WEBDEV aplicación sin Hyper File 5.5 archivo de datos en el análisis:**
	Debe utilizar un análisis creado por WINDEV 5.5. 
	Un análisis en formato WINDEV 5.5 está disponible en el subdirectorio "Personal\\Templates" del directorio instalación de su producto (archivo **ForxBase.WDD**). Este análisis debe copiarse en el directorio del proyecto y asociase al proyecto con la función **HInitHF55**.

- **Aplicación WINDEV o WEBDEV con uno o más archivos de datos Hyper File 5.5 en el análisis:**
	no se requiere ninguna configuración específica.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// WINDEV application without file in WinDev 5.5 format
HInitHF55("C:\File\ForxBase.WDD")
HDBDescribeFile("DBCUSTOMER","CD","C:\FILE\CUSTOMER.DBF")
HDBDescribeField("LASTNAME,C,20")
HDBDescribeField("FIRSTNAME,C,20")
HDBDescribeField("AGE,N,3,0")
HDBDescribeField("BALANCE,N,10,2")
HDBDescribeField("MARRIED,L")
HDBDescribeField("DOB,D")
HDBDescribeField("INFO,M")
HDBCreation()
```

## Sintaxis

`<Result> = HInitHF55(<Full Path of WDD file> [, <Password of WDD file>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si no se ha encontrado ningún problema,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Full Path of WDD file>: (Cadena de caracteres)`**

Acceder a ruta al fichero de análisis.WDD en formato Hyper File 5.5. Se pueden utilizar todos los formatos de análisis reconocidos por Hyper File 5.5 (WINDEV 5.5B, 5.5, 5.0 y 4.1).. Este ruta se tiene en cuenta si no se abre ningún análisis o si el análisis abierto no contiene ningún archivo de datos en formato 5.5 (pestaña "Detalles" del análisis description).

**`<Password of WDD file>: (Cadena de caracteres opcional)`**



- Contraseña asociada al análisis. 

- Cadena vacía (""") si no se ha definido ninguna contraseña.

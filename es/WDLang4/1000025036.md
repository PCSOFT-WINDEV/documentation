


## &lt;Fuente&gt;.MigrateLinkedCompositeKey (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Migra el valor de las claves compuestas relacionadas que provienen de un archivo en formato Hyper File 5.5 al formato HFSQL Classic.


|   |
| --- |
| **¿Cuándo debo utilizar esta función?**<br><br>El formato de las claves compuestas ha cambiado entre Hyper File 5.5 y HFSQL Classic (monopuesto o multipuesto).<br><br>Consecuencia: el valor de las claves relacionado correspondientes a una clave compuesta es incorrecto una vez que los archivos han sido migrados de Hyper File 5.5 a HFSQL Classic. **&lt;Fuente&gt;.MigrateLinkedCompositeKey** cambia el valor de las claves relacionado y recupera el valor de la clave compuesta en la clave relacionado.<br><br>Condiciones de uso:<br><br>Se debe utilizar esta función:<br><br>- con archivos en formato HFSQL Classic (archivos en formato 5.5 migrados al formato Classic),<br><br>- una vez por cada tecla para migrar<br><br>- justo después de la migración de Hyper File 5.5 a HFSQL Classic. Si se añaden registros entre la migración y la llamada a **&lt;Fuente&gt;.MigrateLinkedCompositeKey**, los valores de los nuevos registros pueden estar corruptos.<br><br><br> |


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the values of Orders.CustomerCompKey
Customer.MigrateLinkedCompositeKey(CompKey, Orders, CustomerCompKey)
```

## Sintaxis

`<Result> = <Source>.MigrateLinkedCompositeKey(<Source composite key> , <Linked file> , <Linked key> [, <Progress Bar>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo fuente del enlace, que contiene la description de la clave compuesta.

**`<Source composite key>: (Cadena de caracteres)`**

Nombre del elemento "Clave compuesta" utilizado en el archivo fuente del enlace.

**`<Linked file>: (Cadena de caracteres)`**

Nombre del archivo relacionado que contiene el valor de la clave compuesta a migrar

**`<Linked key>: (Cadena de caracteres)`**

Nombre del elemento clave en el fichero relacionado. El valor de este elemento debe migrarse.

**`<Progress Bar>: (Nombre de ventana o control opcional)`**

Nombre de control Barra de progreso o nombre de la ventana utilizada para mostrar el progreso de la migración.

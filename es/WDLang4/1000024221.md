


## &lt;Fuente&gt;.DeleteView (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una vista creada previamente. Esto borra todo lo relacionado con la vista, incluyendo su description. Durante este borrado, los registros bloqueados por la vista (si existen) se desbloquean.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the Customer34 view
IF Customer34.DeleteView() = False THEN
	Error("Error while deleting a view: " + HError())
ELSE
	Info("The view was deleted.")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.DeleteView()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la vista,

- <u><u><u><u>False</u></u></u></u> si se produce un problema: la vista no se borra. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Source>: (Tipo de fuente especificada)`**

Nombre de la vista a borrar. Este nombre se definió con [&lt;Fuente&gt;.CreateView](../WDLang4/1000024209.md). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Por Default, al salir de la aplicación, se borran todas las vistas existentes.

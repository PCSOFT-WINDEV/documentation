
## HRplDeclareLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
1. Declara que existe un vínculo (1,1) (0,n) entre dos cuadros. Sólo las Record s encontradas en el objetivo tabla con una Record correspondiente en la fuente tabla (respetando el filtro de ésta) serán replicadas durante una replicación universal.

**Observación: Esta función solo se puede utilizar durante la replicación universal.**

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Solo está disponible la replicación universal. 

Nota: La replicación no está disponible al acceder a una base de datos mediante un .




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve a subscriber as well as his messages
HCreateSubscriberReplica("", "", "Mr X", 0, "Messages" + CR + "Recipients")
// Filter the messages
HFilter(Recipients, Name, "X")
// Declare the link
HRplDeclareLink(Recipients, Messages, MessagesID, MessagesID)
HCreateMovableReplica("", "", rplToSubscriber)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HRplDeclareLink(<Name of source file> , <Name of source item> , <Name of target file> , <Name of target item>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema (por ejemplo, si las tablas o elementos/columnas especificadas no existen). La función [HError](../WDLang4/3044088.md) permite identificar el error. Atención: &lt;Resultado&gt; es set a <u><u><u><u>True</u></u></u></u> incluso si los archivos/tablas especificados no están replicados.




**`<Name of source file>: (Cadena de caracteres)`**

Nombre del archivo o tabla cuya multiplicidad es set a (1, 1).

**`<Name of source item>: (Cadena de caracteres)`**

Nombre del elemento afectado por el enlace en el archivo fuente.

**`<Name of target file>: (Cadena de caracteres)`**

Nombre del archivo o tabla cuya multiplicidad es set a (0, n).

**`<Name of target item>: (Cadena de caracteres)`**

Nombre del elemento afectado por el enlace en el fichero destino.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La operación se tiene en cuenta sólo si la fuente tabla/fichero se especifica DESPUÉS del objetivo tabla/fichero en la lista de tablas que se pasa a [HCreateSubscriberReplica](../WDLang4/3044253.md).

- No se respeta el filtrado realizado en el segundo archivo/tabla.

- Tan pronto como un archivo es objeto de un enlace declarado, este archivo se replica únicamente a través de este enlace.

- Si un archivo es la fuente de dos enlaces, se replicará un Record si pertenece a uno de los enlaces.

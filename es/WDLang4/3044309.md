


## HModifyUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la información de un usuario en función de los elementos que se encuentran en la correspondiente [variables para la gestión de usuarios](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
// Associate the data file with the connection
HChangeConnection(Customer, "MyConnection")
IF HInfoUser("MyConnection", "Flora") = True THEN
	// Modify the characteristics
	HUser.Groups = "ADMIN" + CR + "TEST" + CR + "DOC"
	IF HModifyUser("MyConnection", "Flora") = True THEN
		Info("User modified")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyUser(<Connection> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el usuario,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que se debe modificar un usuario. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<User>: (Cadena de caracteres)`**

Nombre del usuario cuyas características deben modificarse. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.

<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios para modificar un usuario
<a name="derechos_necesarios_para_modificar_usuario_ELTPARAGRAPHE000120"></a>

Para modificar un usuario (que no sea el actual), debe tener el derecho de ver a los usuarios (constante *hRightsSeeUser*). En caso contrario, **HModifyUser** devolverá <u><u><u><u>False</u></u></u></u> y aparecerá un mensaje de error "Usuario desconocido".

Para modificar el nombre de un usuario, debe tener permisos para ver otros usuarios (constante *hRightsSeeUser*) y para administrarlos (constante *hRightsManageUser*).

Observación: Estas constantes se utilizan en [HInfoServerRights](../WDLang4/3044322.md) y [HModifyServerRights](../WDLang4/3044328.md).




## HLogStop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene el tronco Process de un archivo de datos. Las operaciones realizadas en el archivo registrado ya no se guardan.

**Observación**: Los registros no están disponibles cuando se accede a una base de datos mediante un [Acceso remoto](../WDLang3/3056029.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HLogInfo(Customer, "start printing " + Criterion)
HLogStop(Customer)  // Stop the log
HReadFirst(Customer, Criterion)
WHILE HFound() = True
	// Process
	 HReadNext(Customer)
END

// Restart log
HLogRestart(Customer) 
// Write to log
HLogInfo(Customer, "end of print of " + Criterion)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HLogStop(<Data file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos registrado cuyo registro Process debe ser detenido.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las operaciones realizadas una vez utilizada esta función no se registrarán y se perderán si se produce un problema.

- No olvide reiniciar el registro Process a través de [HLogRestart](../WDLang4/3044019.md) y asegúrese de que el archivo de datos tiene estrictamente su estado inicial.

- El registro se detiene hasta que se ejecuta [HLogRestart](../WDLang4/3044019.md).

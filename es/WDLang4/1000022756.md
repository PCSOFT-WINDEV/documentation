


## &lt;Variable Connection&gt;.DeleteSet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un conjunto de procedimientos almacenados de un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Delete the set of procedures ("ManageEmail") from the server
MyConnection.DeleteSet(ManageEmail)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteSet(<Set of procedures>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el conjunto de procedimientos,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Connection](../WDLang4/1514073.md) que describe la conexión al servidor HFSQL con el conjunto de procedimientos a eliminar.

**`<Set of procedures>: (Cadena de caracteres)`**

Nombre del conjunto de procedimientos a eliminar del servidor.

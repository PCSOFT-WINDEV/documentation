


## &lt;Fuente&gt;.ListFullTextIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de índices de texto completo para un archivo de datos reconocidos por el motor HFSQL. Puedes get:

- la expresión del índice de texto completo,

- si el índice de texto completo es sensible al mayúsculas y minúsculas o a los caracteres acentuados,

- la longitud mínima de las palabras indexadas,

- si el índice de texto completo es compatible con el formato HTML o RTF,

- si el índice de texto completo está usando una lista de palabras vacías

- si el índice de texto completo utiliza una lista de sinónimos

- qué idioma se utiliza para la obtención de la información....




Se tienen en cuenta los ficheros definidos por [&lt;Fuente&gt;.DescribeFile](../WDLang4/1000024215.md), [&lt;Fuente&gt;.DeclareExternal](../WDLang4/1000024214.md) y [&lt;Fuente&gt;.Declare](../WDLang4/1000024213.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the full-text indexes of a data file in the current analysis
ListFullText is string
ListFullText = Customer.ListFullTextIndex(hLstDetail)
// Adds the list of indexes in COMBO_COMBO1
COMBO_COMBO1.Add(ListFullText)
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.ListFullTextIndex([<Path of the WDD file> [, <Password of the WDD file> [, <Options>]]])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de índices de texto completo que se encuentran en el archivo de datos especificado. Las descripciones de cada índice de texto completo están separadas por caracteres CR (Carriage Return). El resultado es más o menos detallado en función de la constante utilizada:


|   |   |
| --- | --- |
| *hLstDetail* no especificado | &lt;Nombre del índice de texto completo 1&gt; + CR + &lt;Nombre del índice de texto completo 2 + CR + CR ...<br>El resultado es idéntico con la constante *hLstSorted*. |
| Se especifica *hLstDetail* | <br><br>- **Antes de la versión 20**: &lt;Nombre del índice de texto completo 1&gt; + TAB + &lt;Expresión&gt; + CR +....<br><br>- **Después de la versión 20**: &lt;Nombre del índice de texto completo 1&gt; + TAB + &lt;Fórmula&gt; + TAB+ &lt;Distinguir mayúsculas y minúsculas&gt; + TAB + &lt;Distinguir acentos&gt; + TAB + &lt;Longitud mínima&gt; + TAB + &lt;Soporte HTML&gt; + TAB + &lt;Soporte RTF&gt; + TAB + &lt;Palabras vacías&gt; + TAB + &lt;Sinónimos&gt; + TAB + &lt;Stemming&gt; + CR +...<br><br><br> |

Donde:


- Nombre del índice de texto completo&gt; &lt;Nombre del índice de texto completo &gt;&gt;.: nombre de la posición de índice de texto completo

- &lt;Expresión&gt;&gt;Expresión: lista de elementos incluidos en el índice de texto completo. Las posiciones están separadas por el carácter "+".




- &lt;Sensible a mayúsculas y minúsculas: "1" si el índice de texto completo es sensible al caso, "0" en caso contrario.

- &lt;Enfoque sensible: "1" si el índice de texto completo es sensible a los caracteres acentuados, "0" en caso contrario.

- &lt;Longitud mínima&gt;&gt;Longitud mínima: longitud mínima de las palabras indexadas (0 si todas las palabras están indexadas).

- &lt;Soporte HTML&gt;: "1" para que las etiquetas HTML no sean indexadas, "0" para que el formato HTML no sea soportado.

- Soporte &lt;RTF&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;: "1" para que las etiquetas RTF no sean indexadas, "0" para que el formato RTF no sea soportado.

- &lt;Palabras de parada&gt;&gt;Palabras de parada: "1" si el índice de texto completo está usando una lista de palabras vacías, "0" en caso contrario.

- Sinónimos&gt; &lt;Sinónimos&gt;: "1" si el índice de texto completo utiliza una lista de sinónimos, "0" si no.

- &lt;Stemming&gt;: número entero correspondiente al idioma utilizado para el despalillado, "0" si el índice de texto completo no utiliza el despalillado.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos para el que deben listarse los índices de texto completo.

**`<Path of the WDD file>: (Cadena de caracteres opcional)`**

ruta completo de análisis (archivo .WDD) donde se describe el archivo de datos. Si este parámetro no se especifica o si es igual a una cadena vacía (""), se busca el archivo de datos en el análisis actual.

**`<Password of the WDD file>: (Cadena de caracteres opcional)`**

Contraseña definida para el análisis. Si no se define ninguna contraseña, este parámetro debe ser igual a una cadena vacía ("").

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Configura el tipo de lista a realizar:


|   |   |
| --- | --- |
| *hLstDetail* | Información detallada. |
| *hLstNormal*<br>(valor predeterminado) | Información no detallada. |
| *hLstSorted* | Clasifica la lista en orden ascendente. |

## Observaciones


### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000298"></a>

&lt;Resultado&gt; puede asignarse directamente a un control List Box o Combo Box mediante la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md). Cada fila contendrá el nombre de uno de los índices de texto completo encontrados.
<a name="NOTE0_2"></a>


### Extracción de una información del resultado
<a name="extraccion_una_informacion_del_resultado_ELTPARAGRAPHE000313"></a>

Para extraer una información del resultado detallado, le recomendamos que utilice [&lt;Tipo String&gt;.ExtractString](../WDLang1/1000024876.md).

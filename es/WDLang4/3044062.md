
## HFreePosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una posición guardada por [HSavePosition](../WDLang4/3044136.md). Permite liberar la memoria asignada a la gestión de posiciones.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the position
nPos is int = HSavePosition(Customer, Name)
...
// Processes
...
// Delete the useless position
HFreePosition(nPos)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HFreePosition(<Position>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la posición especificada ha sido borrada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Position>: (Integro)`**

Número de una posición guardada de antemano, devuelto por [HSavePosition](../WDLang4/3044136.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las posiciones no borradas por programación se borrarán automáticamente cuando se cierre la aplicación.

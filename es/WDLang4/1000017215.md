
## HRplManageFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define las opciones utilizadas para la réplica universal de un archivo de datos:

- la dirección de replicación.

- el modo de gestión de los conflictos.




En Default, todos los archivos de datos utilizan las características de sincronización definidas por [HSynchronizeReplica](../WDLang4/3044014.md). **HRplManageFile** se puede utilizar para especificar que los cambios realizados en un archivo de datos determinado se replicarán del maestro al suscriptor (la replicación será bidireccional para todos los demás archivos de datos).

**Esta función sólo se puede utilizar en la reproducción universal.**




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The Product file will be replicated from master to subscriber only
HRplManageFile(Product, rplReplicationConflictManagement, rplToSubscriber)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HRplManageFile(<File name> , <Management of conflicts> [, <Replication direction>])`
---

**`<File name>: (Cadena de caracteres)`**

Nombre del archivo de datos para configurar.

**`<Management of conflicts>: (Constante de tipo Integer)`**

Constante que indica el modo de gestión de los conflictos:


|   |   |
| --- | --- |
| *rplMasterFirst* | Los datos que se encuentran en el fichero de datos maestros tienen prioridad durante la replicación. |
| *rplMostRecentFirst* | Los datos más recientes tienen prioridad. Atención, los datos más recientes se evalúan en función de la hora de la réplica del abonado. |
| *rplReplicationConflictManagement*<br>(valor predeterminado) | El archivo utilizará el modo de gestión de conflictos especificado por [HSynchronizeReplica](../WDLang4/3044014.md). |
| *rplSubscriberFirst* | Los datos que se encuentran en el archivo de datos de los suscriptores tienen prioridad durante la replicación. |



**`<Replication direction>: (Constante opcional de tipo Integer)`**

Dirección en la que se realizará la replicación:


|   |   |
| --- | --- |
| *rplBidirectional* <br>(valor predeterminado) | Actualice el fichero maestro Y el fichero de suscriptores. |
| *rplToMaster* | Actualice el fichero maestro según la base de datos de suscriptores. |
| *rplToSubscriber* | Actualice el fichero de suscriptores según la base de datos maestra. |

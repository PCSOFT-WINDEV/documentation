
## HServerWorkingDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ruta de un directorio temporal en el servidor HFSQL. Este directorio se crea automáticamente en el servidor. Los procedimientos almacenados que se ejecutan en el servidor tendrán los derechos para escribir en este directorio. Este directorio se borrará y destruirá automáticamente cuando el cliente se desconecte.

**Observación**: Esta función sólo puede ser ejecutada desde un Procedure almacenado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code of a stored procedure
fSaveText(CompleteDir(HServerWorkingDir()) + "MyFile.txt", MyTextString)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HServerWorkingDir()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre completo del directorio temporal en el servidor HFSQL. Un Procedure almacenado podrá crear archivos de texto en este directorio por ejemplo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para forzar la eliminación del directorio y el borrado de su contenido, utilice [HClearWorkingDir](../WDLang4/1000017767.md).

- [HSetServer](../WDLang4/3044359.md) también devuelve información sobre las rutas de los diferentes directorios que se encuentran en el servidor (ruta del directorio de la base de datos, ruta del directorio de registros, ruta del directorio de copia de seguridad de los registros, etc.)

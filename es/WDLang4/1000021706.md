
## hbRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee las celdas (valores de los elementos) de un fila (registro) en una HBase tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Description of data to read
Reading is hbReading
Reading.Row = "post1"
Reading.Version = 2
// Column description
Col is hbColumn
Col.Family = "Post"
Add(Reading.Column, Col)
// Read data
MyResult is array of hbCells = hbRead(Connection, "blog", Reading)
// Display the data read
FOR EACH Cell OF MyResult DO
	Trace("Column: " + Cell.Column.Qualifier + "; Value: " + ...
	Cell.Value + "; timestamp: " + Cell.Timestamp)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = hbRead(<Connection> , <Table> , <Reading>)`
---

**`<Result>: (Array de hbCell)`**

Array de las variables [hbCell](../WDLang4/1000021707.md) correspondientes a los valores de las celdas leídas.

**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase.

**`<Table>: (Cadena de caracteres)`**

Nombre de la HBase tabla a utilizar.

**`<Reading>: (hbReading Variable)`**

Nombre de [hbReading](../WDLang4/1000021701.md) Variable que contiene las características de la operación de lectura a realizar.

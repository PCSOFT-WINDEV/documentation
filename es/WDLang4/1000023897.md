


## Funciones de MongoDB (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las bases de datos MongoDB (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;mongoCollection Variable&gt;.Añadir](../WDLang4/1000025345.md) | Agrega uno o más documentos a una colección de la base de datos de MongoDB. |
| [&lt;mongoCollection Variable&gt;.BorrarColección](../WDLang4/1000025400.md) | Elimina una colección de una base de datos MongoDB. |
| [&lt;mongoCollection Variable&gt;.BorrarTodo](../WDLang4/1000025401.md) | Elimina todos los documentos (correspondientes al filtro especificado) de una colección de una base de datos MongoDB. |
| [&lt;mongoCollection Variable&gt;.BorrarUno](../WDLang4/1000025402.md) | Borra una Document de una colección en una base de datos de MongoDB. |
| [&lt;mongoCollection Variable&gt;.Encontrar](../WDLang4/1000025356.md) | Encuentra documentos en una colección de MongoDB. |
| [&lt;mongoCollection Variable&gt;.ExecuteCommand](../WDLang4/1000023819.md) | Ejecuta un comando genérico en una colección de MongoDB. |
| [&lt;mongoCollection Variable&gt;.ModificarTodos](../WDLang4/1000025397.md) | Modifica documentos en una colección de una base de datos MongoDB. |
| [&lt;mongoCollection Variable&gt;.ModifyOne](../WDLang4/1000025398.md) | Modifica una Document en una colección de la base de datos de MongoDB. |
| [&lt;mongoDatabase Variable&gt;.CrearColección](../WDLang4/1000023612.md) | Crea una colección en una base de datos MongoDB mientras especifica las opciones de esta colección. |
| [&lt;mongoDatabase Variable&gt;.DeteleBase de datos](../WDLang4/1000025399.md) | Elimina una base de datos MongoDB así como sus colecciones. |
| [&lt;mongoDatabase Variable&gt;.ExecuteCommand](../WDLang4/1000023636.md) | Ejecuta un comando genérico en una base de datos MongoDB. |
| [&lt;mongoGridFS Variable&gt;.DeleteFile](../WDLang4/1000025394.md) | Elimina un archivo identificado por su nombre de un sistema de archivos MongoDB. |
| [&lt;mongoGridFS Variable&gt;.DeleteFileByID](../WDLang4/1000025395.md) | Elimina un archivo identificado por su identificador de un sistema de archivos MongoDB. |
| [&lt;mongoGridFS Variable&gt;.GetFile](../WDLang4/1000025391.md) | Recupera (en la computadora local) un archivo identificado por su nombre y encontrado en un sistema de archivos MongoDB. |
| [&lt;mongoGridFS Variable&gt;.ListFile](../WDLang4/1000025358.md) | Recupera la lista de archivos MongoDB encontrados en el sistema de archivos MongoDB (gridFS). |
| [&lt;mongoGridFS Variable&gt;.SendFile](../WDLang4/1000025357.md) | Copia un archivo del ordenador local a un sistema de archivos MongoDB (gridFS). |
| [&lt;mongoVariable de la base de datos&gt;.Info](../WDLang4/1000025396.md) | Devuelve información sobre una base de datos MongoDB. |
| [&lt;Variable mongoCollection&gt;.ExecuteOperations](../WDLang4/1410089633.md) | Realiza operaciones de escritura en MongoDB de forma masiva. Esto evita los problemas de latencia de red que podrían producirse si se realizan las operaciones de forma secuencial. |
| [&lt;Variable mongoGridFS variable&gt;.GetFileInfo](../WDLang4/1000025393.md) | Utilizado para get información sobre un archivo de MongoDB (gridFS). |
| [&lt;Variable mongoGridFS&gt;.GetFileById](../WDLang4/1000025392.md) | Recupera (en la computadora local) un archivo identificado por su identificador y encontrado en un sistema de archivos MongoDB. |




## HLogRestart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia el registro del archivo de datos. El registro se detuvo con la función [HLogStop](../WDLang4/3044035.md).

**Observación**: El registro Process no está disponible cuando se accede a una base de datos a través de .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HLogInfo("Customer", "start printing " + Criterion)
HLogStop("Customer")  // Stop the log
HReadFirst("Customer", Criterion)
WHILE HFound()
	// Process
	HReadNext("Customer")
END

HLogRestart("Customer") // Restart the log
HLogInfo("Customer", "done printing" + Criterion)  // Write
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HLogRestart(<Data file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos cuyo registro debe ser reiniciado.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al reiniciar el registro, el archivo de datos debe estar en su estado inicial, es decir, en el estado en el que se encontraba antes de utilizar la función [HLogStop](../WDLang4/3044035.md).

- Entre las funciones [HLogStop](../WDLang4/3044035.md) y **HLogRestart**, no se guardará la información en el registro.

- La función **HLogRestart** equivale a la función [HSetLog](../WDLang4/3044052.md) con el parámetro <u><u><u><u>True</u></u></u></u>.

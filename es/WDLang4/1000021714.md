


## hbDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina: 

- una o más celdas de una HBase tabla para una o más columnas. 

- a fila desde una HBase tabla. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Delete the entire row
hbDelete(Connection, "blog", "post1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borrar un fila o un associated de celda con una columna

`<Result> = hbDelete(<Connection> , <Table> , <Row> [, <Column> [, <Timestamp>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase.

**`<Table>: (Cadena de caracteres)`**

Nombre de HBase tabla

**`<Row>: (Cadena de caracteres)`**



- Identificador de fila a eliminar (si no se especifica &lt;Columna&gt;). 

- Identificador de fila que contiene la columna a borrar (la celda será borrada). 




**`<Column>: (Opcional hbColumn Variable)`**

Nombre de la Variable de tipo [hbColumn](../WDLang4/1000021698.md) que corresponde a la columna o grupo de columnas a eliminar.

**`<Timestamp>: (DateTime opcional)`**

Fecha límite para borrar los valores encontrados en la celda o en el fila. Sólo se borrarán los valores encontrados antes de esta fecha.


<a name="SYNTAX2"></a>

### Borrar un fila o las celdas associated con varias columnas

`<Result> = hbDelete(<Connection> , <Table> , <Row> [, <Array of columns> [, <Timestamp>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase.

**`<Table>: (Cadena de caracteres)`**

Nombre de HBase tabla

**`<Row>: (Cadena de caracteres)`**



- Identificador de la fila a eliminar (si no se especifica &lt;Array de columnas&gt;). 

- Identificador de la fila que contiene las columnas a borrar (las celdas serán borradas). 




**`<Array of columns>: (array opcional de variables hbColumn)`**

Array de variables de tipo [hbColumn](../WDLang4/1000021698.md) que corresponden a las columnas de grupos de columnas que se van a eliminar.

**`<Timestamp>: (DateTime opcional)`**

Fecha límite para borrar los valores encontrados en las celdas o en el fila. Sólo se borrarán los valores encontrados antes de esta fecha.

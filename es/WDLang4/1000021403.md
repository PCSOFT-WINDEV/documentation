
## <Source>.FromWindow (Función)

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se actualiza automáticamente:

- los valores de los elementos de un archivo de datos en memoria con los valores de los controles de la ventana. Los elementos del archivo de datos vinculados a los controles de la ventana se actualizan automáticamente con los valores de estos controles.

- el valor de las variables WLanguage con el valor de los controles de ventana. Las variables WLanguage vinculadas a los controles de la ventana se actualizan automáticamente con los valores de dichos controles.




**Observaciones**:

- **<Source>.FromWindow** no se puede usar para escribir en el archivo de datos: las variables del archivo de datos sólo pueden ser actualizadas (es decir, las variables de la Record cargadas en memoria). Para modificar los registros, utilice [&lt;Fuente&gt;.Add](../WDLang4/1000024187.md), [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md), [&lt;Fuente&gt;.Write](../WDLang4/1000024331.md), etc.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the "Add" button
Customer.FromWindow()
HAdd(Customer)
IF HErrorDuplicates() = True THEN Error("This number already exists")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Source>.FromWindow([<Window>])`
---

**`<Source>: (Tipo de fuente especificada)`**

Nombre de la fuente que contiene los datos a actualizar. Por ejemplo, esta fuente puede ser:

- El nombre de un archivo de datos: sólo se tienen en cuenta los controles vinculados a los elementos del archivo de datos.

- El nombre de un WLanguage Variable: sólo se tienen en cuenta los controles vinculados a la lengua de trabajo Variable. El control se vinculará a los datos Variable (por ejemplo, un Variable que contenga el nombre de otro Variable). 




**`<Window>: (Nombre de ventana)`**

Nombre de la ventana que contiene los controles.

Resumen: Se llama cuando una control gana el foco. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Vinculación entre un control y un WLanguage Variable o entre un control y un elemento de un archivo de datos
<a name="vinculacion_entre_control_wlanguage_variable_entre_control_elemento_archivo_datos_ELTPARAGRAPHE000235"></a>

Se puede determinar la vinculación entre un control y un WLanguage Variable o entre un control y un elemento de un archivo de datos:

- En el editor, en la ventana de control description (pestaña "Binding").

- De forma programada, con el programa [DataBinding](../Proprietes/2510060.md) Property.



<a name="NOTE0_2"></a>


### Actualización de los artículos: Casos especiales
<a name="actualizacion_los_articulos_casos_especiales_ELTPARAGRAPHE000251"></a>

- Para asignar automáticamente el valor NULL a un elemento con la función **<Source>.FromWindow**:

	1. Marque "NULL supported" para el elemento en el editor de análisis.

	2. Seleccione la opción "NULL si está vacío" en la Edición control vinculada al artículo.




- Los elementos del archivo de datos que no están vinculados a un control no se actualizan con **<Source>.FromWindow** .

- Si varios controles en la ventana están vinculados al mismo elemento del archivo de datos, es posible que **<Source>.FromWindow** no actualice correctamente el elemento vinculado. ***No se recomienda vincular varios controles de ventana al mismo elemento.***

- **<Source>.FromWindow** no actualiza elementos o variables con los valores de los controles Image. Utiliza la función [&lt;Fuente&gt;.LinkMemo](../WDLang4/1000024194.md).

- **<Source>.FromWindow** no está disponible en las vistas.

- **Vinculación con elementos de "Contraseña segura"**: El valor del elemento "Contraseña" no puede modificarse hasta que el usuario cambie el valor en la control vinculada al elemento.

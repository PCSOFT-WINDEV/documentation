


## &lt;Fuente&gt;.DeactivateAutoFilter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva un filtro automático en los archivos de datos relacionados al navegar por un archivo XML. Este filtro se activó con la función [&lt;Fuente&gt;.ActivateAutoFilter](../WDLang4/1000024171.md).

**Atención**: Esta función solo está disponible para el Conector Nativo XML.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable all the possible automatic filters on the Customer data file
Customer.ActivateAutoFilter("*")
// except between Customer and Order
Customer.DeactivateAutoFilter(Order)

/////////////////////////////

// Browse orders
Order.ReadFirst(OrderID)
WHILE Order.Out() = False
	// Browses the order lines of the current order
	OrdLine.ReadFirst(OrderID)
	WHILE OrdLine.Out() = False
		OrdLine.ReadNext(OrderID)
	END
	// Next order
	Order.ReadNext(OrderID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.DeactivateAutoFilter(<Data file not to filter>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se desactivó el filtro,

- <u><u><u><u>False</u></u></u></u> si se produce un error (archivo de datos o elemento inexistente).




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado para la navegación.

**`<Data file not to filter>: (Cadena de caracteres)`**



- Nombre del archivo de datos que no se debe filtrar cuando se establece la posición en el archivo de datos utilizado para la navegación.

- "\*" para desactivar todos los filtros automáticos.

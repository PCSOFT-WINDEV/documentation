
## HError (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número del último error provocado por el motor HFSQL.

**Observación**: Se debe llamar a esta función justo después de la función que produjo el error. La llamada a una función distinta de la función de error podría reiniciar y cambiar el informe devuelto por la función **HError**.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003044088&name=herror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Manage an error wile creating a file
IF HCreation(Customer) = False THEN
	Error("HFSQL error: " + HError())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HError([<Type of error>])`
---

**`<Result>: (Entero)`**



- Número del último error que se produjo

- Constante *hNoError* (cuyo valor es 0) en los siguientes casos:

	- No se produjo ningún error.

	- El tipo de error solicitado correspondía a un suberror, pero no se produjo ningún suberror.







**`<Type of error>: (Constante opcional de tipo Integer.)`**

Permite obtener los números de los suberrores.


|   |   |
| --- | --- |
| *hErrCurrent*<br>(valor predeterminado) | Número del error en curso. |
| *hErrMain* | Número del error principal. |
| *hSubErrFirst* | Número del primer suberror. |
| *hSubErrLast* | Número del último suberror. |
| *hSubErrNext* | Número del siguiente suberror. |
| *hSubErrPrevious* | Número del suberror anterior. |

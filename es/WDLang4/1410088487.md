


## HExportJSONString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta los registros de un archivo de datos (HFSQL u OLE DB), vista o consulta a una cadena de JSON.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Export all customers 
sString is string
sString = HExportJSONString(CUSTOMER)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HExportJSONString(<File> [, <Items to export>] [, <Options> [, <Progress Bar>]])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena JSON.

**`<File>: (Cadena de caracteres)`**

Nombre del archivo de datos, vista o consulta utilizada.

**`<Items to export>: (Cadena de caracteres opcional)`**

Lista de nombres de artículos a exportar. Los nombres están delimitados por comillas simples y están separados por comas o caracteres CR. Se ignoran los nombres de los elementos clave compuestos, los elementos binarios y los elementos de la nota binaria..
Si este parámetro es una cadena vacía (""), se exportan todos los elementos de del archivo de datos (excepto los elementos de clave compuesta, binario, memo binario y contraseña).

**`<Options>: (Constante opcional de tipo Integer)`**

Especifica el modo de creación del archivo JSON:


|   |   |
| --- | --- |
| *hExpBreakable* | La exportación puede interrumpirse pulsando Esc. <br>Si la exportación se interrumpe, la cadena JSON es válida y contiene sólo los registros exportados antes de la interrupción. |
| *hExpNormal*<br>(valor predeterminado) | Funcionamiento estándar. |



**`<Progress Bar>: (Nombre de control o ventana opcional)`**

Nombre de la control Barra de progreso o ventana que muestra el progreso de la creación de la cadena JSON.

<a name="NOTE0_1"></a>

## Observaciones


### Exportación del contenido de la consulta
<a name="exportacion_del_contenido_consulta_ELTPARAGRAPHE000190"></a>

Para exportar el contenido de una consulta, debe inicializarla (función [HExecuteQuery](../WDLang4/3044080.md)) antes de utilizar la función **HExportJSONString**.
<a name="1000021964_NOTE0_2"></a>


### Exportar un solo Record
<a name="exportar_solo_record_ELTPARAGRAPHE000204"></a>

Para exportar una única Record a una cadena JSON, utilice [HRecordToJSON](../WDLang4/1000021909.md).
<a name="1000021964_NOTE0_3"></a>


### Campos de tipo Array
<a name="campos_tipo_array_ELTPARAGRAPHE000216"></a>Si uno de los elementos exportados es un elemento Array, cada elemento del elemento array se exporta en un nuevo nivel:


```txt
{"File":
	{
	"Item1":Data of item 1,
	"Item2":Data of item 2,
	"ArrItem":
		[Data element 1, Data element 2, ...]
	...
	}
}
```

**Recordatorio**: Se ignoran los elementos de clave compuesta, binario, memo binario y contraseña.




## Estructura de los ficheros log (HFSQL Client/Server)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000256"></a>
Los siguientes archivos se crean automáticamente cuando se registra un archivo de datos de HFSQL Client/Server:


|   |   |
| --- | --- |
| **JNL_FILES.FIC** | description de los archivos registrados. |
| **JNL_OPERATION.FIC** | description de acciones realizadas en un archivo registrado. |
| **JNL_USERS.FIC** | description de usuarios que realizan una acción en un archivo registrado. |
| **xxxJNL.FIC** | Archivo creado para cada archivo registrado. Contiene el valor de los registros utilizados antes y después de cada operación. |

Estos ficheros se crean según la siguiente estructura:


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Journal_CS.gif) | **1. Directorio JNL**<br>Este directorio contiene:<br><br>- El registro de archivos<br><br>- El log de usuario<br><br>- El registro de operaciones<br><br>- Para cada archivo registrado de cada base de datos, el archivo de registro correspondiente. La estructura de árbol utilizada corresponde a la estructura de árbol de los archivos de datos associated con la base de datos.<br><br><br>**Por ejemplo**, si el archivo de órdenes de DataApplicationA está registrado, el subdirectorio __JNL\\DataApplicationA contendrá el archivo OrdersJNL.FIC.<br><br>**Otro ejemplo**: si el archivo del cliente de DataApplicationB está registrado, el subdirectorio __JNL\\DataApplicationB\\Customer contendrá el archivo CustomerJNL.FIC.<br><br>**2. Directorio de la base de datos** DataApplicationA<br><br>**3. Directorio de la base de datos** DataApplicationB |

Esta Page presenta la estructura de estos ficheros.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fichero JNL_FILES.FIC
<a name="fichero_jnlfilesfic_ELTTEXTE000286"></a>
Los diferentes elementos que se encuentran en JNL_FILES.FIC son los siguientes:


|   |   |
| --- | --- |
| **JNLFile_ID** | Identificador de archivo. |
| **Source_DB_File_Location** | Nombre de la base de datos + '\\' + ruta relativa del archivo de registro. |
| **Source_File_GUID** | GUID del archivo registrado. |
| **JNL_File_GUID** | GUID del registro de escritura a archivo. |



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## JNL_OPERATION.FIC archivo
<a name="jnloperationfic_archivo_ELTTEXTE000310"></a>
Los diferentes elementos que se encuentran en JNL_OPERATION.FIC son los siguientes:


|   |   |
| --- | --- |
| **Operation_ID** | Identificador de la operación. |
| **USUARIO_ID** | Identificador del usuario que realiza la operación. Este identificador corresponde al identificador de JNL_USER.Fic |
| **JNLFile_ID** | Identificador del archivo registrado donde se realiza la operación. Este identificador corresponde al identificador de JNL_FILES.Fic. |
| **WLFunction_ID** | Identificador de la función WLanguage utilizada en el programa. Para saber el nombre de la función WLanguage utilizada, basta con consultar el fichero ListeDéfinitionHF.WL (que se encuentra en el subdirectorio "Personal\\Extern\\" del producto) |
| **Server_Time** | Fecha de funcionamiento en el servidor (en formato UTC, hora universal) |
| **Record_Number** | Número de la Record modificada |
| **Source_Record_Backup_ID** | Identificador en el archivo de valores. Corresponde al valor de Record antes de la operación. |
| **Modified_Record_Backup_ID** | Identificador en el archivo de valores. Corresponde al valor de Record después de la operación. |
| **HLogInfoMsg** | Valor definido por [HLogInfo](../WDLang4/3044009.md). |
| **hComputer_ID** | Texto definido por [HComputer](../WDLang4/3044111.md). |

## JNL_USERS.FIC archivo
<a name="jnlusersfic_archivo_ELTTEXTE000334"></a>
Los diferentes elementos que se encuentran en JNL_USERS.FIC son los siguientes:


|   |   |
| --- | --- |
| **USUARIO_ID** | Identificador del usuario. |
| **Usuario** | Nombre de usuario. |
| **Workstation_Name** | Nombre del ordenador del usuario. |
| **Dirección_IP de la estación de trabajo** | IP Address de la computadora del usuario. |
| **Aplicación** | Nombre de la aplicación |



<a name="NOTE5"></a>
<a name="NOTE5_1"></a>
El fichero \*JNL.Fic contiene:

- los artículos que se encuentran en la tabla de abajo,

- todos los elementos registrados del correspondiente archivo HFSQL registrado.





|   |   |
| --- | --- |
| **USUARIO_ID** | Identificador del usuario que realizó la operación. Este número se utiliza para identificar la Record en JNL_Users.Fic |
| **Record_Number** | Número de la Record en el archivo inicial registrado. |
| **Record_Status** | Estado de la Record (modificado, eliminado, añadido, ...) |
| **WLFunction_ID** | Identificador de la función WLanguage utilizada en el programa. Para conocer el nombre de la función WLanguage utilizada, basta con consultar el fichero ListeDéfinitionHF.WL (que se encuentra en el subdirectorio "Personal\\Extern\\" del producto utilizado) |
| **Server_Time** | Fecha de funcionamiento en el servidor (en formato UTC, hora universal) |
| **HLogInfoMsg** | Valor definido por [HLogInfo](../WDLang4/3044009.md). |
| **hComputer_ID** | Texto definido por [HComputer](../WDLang4/3044111.md). |
| **IDJNL** | Identificador del registro |




## &lt;Variable Connection&gt;.InfoFileProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer las propiedades de un archivo de datos ubicado en un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
...
OwnerName is string
OwnerName = MyConnection.InfoFileProperty(hOwner, ...
	"MyDatabase", "Customer.fic")
IF OwnerName <> "" THEN 
	Trace("Owner of the database: " + OwnerName)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.InfoFileProperty(<Property> , <Database> , <Data file>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de Property solicitado,

- Cadena vacía ("") si se produce un error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable [Conexión](../WDLang4/1514073.md) que describe la conexión para la que se solicita un valor Property.

**`<Property>: (Constante de tipo Integer)`**

Archivo solicitado Property: 


|   |   |
| --- | --- |
| *hApplication* | Nombre de la aplicación a partir de la cual se creó el archivo de datos. <br>Si se utiliza esta constante, &lt;Result&gt; será una cadena de caracteres con el siguiente formato: <br>&lt;Nombre ejecutable&gt;(&lt;Nombre del proyecto>). |
| *hComputer* | Nombre de el equipo a partir del cual se creó el archivo de datos. <br>Si se utiliza esta constante, &lt;Result&gt; tendrá el siguiente formato: <br>&lt;Nombre del ordenador cliente&gt; + TAB + &lt;IP Address del ordenador cliente&gt;. |
| *hCreator* | Nombre del propietario del archivo.<br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |
| *hDate* | Fecha de creación del archivo de datos. <br>Si se utiliza esta constante, &lt;Resultado&gt; es una cadena con el siguiente formato: YYYYMMDDHHmmSSCCC. |
| *hNotes* | description del archivo de datos o cualquier otra información asociada a el archivo de datos por [&lt;Variable Connection&gt;.ModifyFileProperty](../WDLang4/1000022718.md). <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |
| *hOwner* | Propietario del archivo de datos. <br>Si se utiliza esta constante, &lt;Result&gt; es una cadena de caracteres. |



**`<Database>: (Cadena de caracteres)`**

Nombre de la base de datos que contiene el archivo cuyas propiedades se solicitan. 
Si este parámetro corresponde a una cadena vacía ("""), se utilizará la base de datos definida en la conexión.. 
Se produce un error de WLanguage si no se define ninguna base de datos en la conexión.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos HFSQL cuyas propiedades se solicitan. Se debe especificar la extensión del archivo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para modificar las propiedades de un archivo de datos, utilice [&lt;Variable Connection&gt;.ModifyFileProperty](../WDLang4/1000022718.md).

- Atención: el el archivo de datos utilizado debe haber sido creado en el servidor ([&lt;Fuente&gt;.Creation](../WDLang4/1000024207.md) o [&lt;Fuente&gt;.CreationIfNotFound](../WDLang4/1000024208.md)).

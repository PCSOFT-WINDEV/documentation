
## hbCreateTable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una HBase tabla o modifica la descripción de una HBase tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Describe the table
TableDesc is hbTableDescription
TableDesc.Name = "blog"
// Describe the "Post" column
ColDesc is hbColumnDescription
ColDesc.Name = "Post"
Add(TableDesc.Column, ColDesc)
// Create the table
hbCreateTable(Connection, TableDesc)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = hbCreateTable(<Connection> , <Description>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación, 

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hbConnection Variable)`**

Nombre del [hbConnection](../WDLang4/1000021685.md) Variable que corresponde a la conexión con el servidor HBase. 

**`<Description>: (hbTableDescription Variable)`**

Nombre de la variable [hbTableDescription](../WDLang4/1000021688.md) correspondiente al tabla description.


## SQLColumnFormat( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve los diferentes tipos de columnas soportadas:

- por la fuente de datos identificada por [SQLConnect](../WDLang4/3072005.md)

- por la base de datos a la que se accede mediante una conexión específica (descrita en el análisis, o descrita por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md)).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Str is string
Str = SQLColumnFormat()
// The Str string contains for example:
// "CHAR" + TAB + "9" + CR + "VARCHAR" + TAB + "20" + CR
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SQLColumnFormat([<Connection name>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de tipos soportados:

- por la fuente de datos (definida por [SQLConnect](../WDLang4/3072005.md))

- por &lt;Nombre de la conexión&gt; si se especifica este parámetro.




Esta lista tiene el siguiente formato: &lt;Tipo de columna SQL&gt; + TAB + &lt;WINDEV/WEBDEV type&gt; + CR. Para obtener más información, consulte [observaciones](#NOTE0_1).

**`<Connection name>: (Cadena de caracteres opcional)`**

Si se especifica este parámetro, se utiliza para especificar la conexión para la que se debe devolver el formato de las columnas.. Este nombre corresponde a una conexión descrita en el análisis, o a una conexión descrita por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

Si no se especifica este parámetro, se utilizará la fuente de datos definida por [SQLConnect](../WDLang4/3072005.md).

## Observaciones




### Lista de tipos soportados
<a name="lista_tipos_soportados_ELTPARAGRAPHE000096"></a>

El resultado de la función **SQLColumnFormat** tiene el siguiente formato:

```txt
<Type of SQL column> + TAB + <WINDEV type> + CR
```

| tipo SQL | Corresponde a | Tipo WINDEV/WEBDEV |
| --- | --- | --- |
| SQL_WLONGVARCHAR | Memo Unicode | -2 |
| SQL_WCHAR | Cadena Unicode | 0 |
| SQL_BIT | trozo | 1 |
| SQL_TINYINT | Integro | 2 |
| SQL_BIGINT | Integro | 3 |
| SQL_LONGVARBINARY | Memorándum binario | 4 |
| SQL_VARBINARY | Cadena binaria de longitud fija | 5 |
| SQL_BINARY | Binario | 6 |
| SQL_LONGVARCHAR | Nota de cadena ASCII | 7 |
| SQL_CHAR | Cadena de longitud fija | 9 |
| SQL_NUMERIC | Numeric | 10 |
| SQL_DECIMAL | Número decimal | 11 |
| SQL_INTEGER | Integro | 12 |

| SQL_FLOAT | Real | 14 |

| SQL_DATE | salir con | 17 |




## HModifyDatabaseProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las propiedades de una base de datos ubicada en un servidor HFSQL.

**Observación**: Para conocer las propiedades de una base de datos, utilice [HInfoDatabaseProperty](../WDLang4/3044323.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ResModif is boolean
ResModif = HModifyDatabaseProperty("MyConnection", hOwner, "Flora", "MyDatabase")
IF ResModif = True THEN
	Info("Modification performed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyDatabaseProperty(<Connection> , <Property to modify> , <New value> , <Database>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificaron las propiedades,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que hay que modificar el valor de una Property. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Property to modify>: (Constante de tipo Integer)`**

Base de datos Property para modificar: 


|   |   |
| --- | --- |
| *hNotes* | description de la base de datos o cualquier otra información associated con la base de datos. <br>Si se utiliza esta constante, &lt;Valor nuevo&gt; es una cadena de caracteres que contiene el texto de la note. |
| *hOwner* | Propietario de la base de datos. El propietario es el propietario de la base de datos. El propietario de una base de datos puede cambiar. La noción de propietario repercute en la gestión de los derechos.<br>Si se utiliza esta constante, &lt;Nuevo valor&gt; es una cadena de caracteres que contiene el nuevo nombre del propietario. |



**`<New value>: (Cadena de caracteres)`**

Nuevo valor para el Property.

**`<Database>: (Cadena de caracteres)`**

Nombre de la base de datos cuyas propiedades deben ser modificadas. 
Si este parámetro corresponde a una cadena vacía ("""), se utilizará la base de datos definida en la conexión.. 
Se produce un error de WLanguage si no se define ninguna base de datos en la conexión..

<a name="NOTE0_1"></a>

## Observaciones


### Cambiar el propietario de una base de datos
<a name="cambiar_propietario_una_base_datos_ELTPARAGRAPHE000132"></a>

- El usuario especificado como nuevo propietario debe ser un usuario que se encuentre en el servidor. Se produce un error de WLanguage si el usuario no existe.

- Para modificar el propietario, el usuario que realiza la modificación debe:

	- ser un "superusuario".

	- tener permisos para cambiar el propietario de la base de datos (permisos correspondientes a la constante *hRightsChangeOwner*).





Para determinar si el usuario tiene los derechos adecuados, utilice [HInfoDatabaseProperty](../WDLang4/3044323.md).

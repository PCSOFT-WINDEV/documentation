
## HClusterMasterNode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del nodo maestro en un cluster HFSQL. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
let sMasterString = HClusterMasterNode("clustermanager.mycompany.com", ... 
	 "secret", "host3.mycompany.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterMasterNode(<Coordinator> , <Password>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del nodo maestro del cluster HFSQL. 

- Cadena vacía ("") si no se detecta el maestro. Puede que no haya maestro si los nodos no se sincronizan o se desconectan del coordinador.




**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS de la máquina donde se encuentra el coordinador del cluster HFSQL.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se proporcionó al instalar el clúster HFSQL.

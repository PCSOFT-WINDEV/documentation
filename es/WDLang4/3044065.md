
## HSetTrigger( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o no el manejo de disparadores. Para obtener más información, consulte [Gestión de triggers](../WDLang4/3044271.md). 

**Observación**: El manejo de los disparadores está habilitado por Default. **Para acelerar su aplicación**le aconsejamos que desactive los disparadores si no se utilizan.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disables the management of triggers
HSetTrigger(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HSetTrigger(<Management>)`
---

**`<Result>: (booleano)`**

Anterior modo de gestión de triggers (antes de llamar a la función **HSetTrigger**): 

- <u><u><u><u>True</u></u></u></u> si la gestión de triggers estaba activa,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si se ha desactivado la gestión de los desencadenantes.




**`<Management>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la gestión de triggers está activada,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si esta gestión debe ser desactivada.

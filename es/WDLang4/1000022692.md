


## &lt;Variable Connection&gt;.ListDatabase (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista las bases de datos Client/Server asociadas a una conexión.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
...
// Add the databases of MyConnection into an edit control
EDT_Edit1 = MyConnection.ListDatabase()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ListDatabase([<Options>])`
---

**`<Result>: (Character string)`**

Lista de bases de datos del servidor definidas por la conexión. El resultado es más o menos detallado en función de la constante utilizada:


|   |   |
| --- | --- |
| *hLstNormal* | Enumera las bases de datos del servidor definidas por la conexión:<br>&lt;Nombre de la base de datos 1&gt; + TAB + &lt;Nombre de la base de datos 2&gt; + TAB +... |
| *hLstFile* | Enumera las bases de datos y sus archivos para el servidor definido por la conexión:<br>&lt;Nombre de la base de datos 1&gt; + TAB + &lt;Nombre del archivo de datos 1 en la base de datos 1&gt; + TAB + ... + CR +&lt;Nombre de la base de datos 2&gt; + TAB + &lt;Nombre del archivo de datos 1 en la base de datos 2&gt; + TAB + ... + CR +... |
| *hLstDatabaseSize* | Enumera las bases de datos y su tamaño (en bytes) para el servidor definido por la conexión:<br>&lt;Nombre de la base de datos 1&gt; + TAB + &lt;Tamaño de la base de datos 1&gt; + CR + &lt;Nombre de la base de datos 2&gt; + TAB + &lt;Tamaño de la base de datos 2&gt; + CR + ...<br> |


Donde:

- &lt;Nombre de la base de datos X&gt;&gt;: Nombre de la base de datos.

- &lt;Nombre del archivo de datos Y en la base de datos X&gt;: Nombre del archivo de datos físicos Y localizado en la base de datos X.

- &lt;Tamaño de la base de datos X&gt;&gt;: Tamaño de todos los archivos ubicados en la base de datos en bytes. 




Si se produce un error, la lista de bases de datos corresponde a una cadena vacía ("") y se utiliza [HError](../WDLang4/3044088.md) para identificar el error.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Tipo de lista a realizar:


|   |   |
| --- | --- |
| *hLstFile* | Lista detallada, utilizada para averiguar los archivos associated con cada base de datos. |
| *hLstNormal*<br>(valor predeterminado) | Lista no detallada, que contiene el nombre de las bases de datos associated con una conexión. |
| *hLstDatabaseSize* | Lista detallada, utilizada para averiguar el tamaño de todos los archivos associated con cada base de datos.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No disponible. |

### &lt;Variable Connection&gt;.ListDatabase y el conector nativo de MySQL
<a name="docparampagetitleshort_conector_nativo_mysql_ELTPARAGRAPHE000293"></a>

- [HListFile](../WDLang4/3044045.md) devuelve la lista de archivos encontrados en la base de datos actual. 

- **&lt;Variable Connection&gt;.ListDatabase** lista las bases de datos MySQL accesibles y sus archivos.

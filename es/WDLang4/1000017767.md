
## HClearWorkingDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra y destruye el directorio temporal creado previamente al ejecutar [HServerWorkingDir](../WDLang4/1000017768.md). Este directorio se borra y destruye automáticamente durante la desconexión del cliente.

**Observación**: Esta función sólo puede ser ejecutada desde un Procedure almacenado.  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code of a stored procedure
fSaveText(CompleteDir(HServerWorkingDir()) + "MyFile.txt", MyTextString)
...
HClearWorkingDir()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HClearWorkingDir()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se destruyó el directorio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.

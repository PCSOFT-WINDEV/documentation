


## HListStoredElement (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de los elementos almacenados en un servidor HFSQL (conjuntos de procedimientos, procedimientos almacenados o consultas).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the elements found on the server
ElementList is string
ElementList = HListStoredElement("MyConnection", hLstDetail + hLstAll)

// Adds the list of element into COMBO_COMBO1
ListAdd(COMBO_COMBO1, ElementList)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListStoredElement(<Connection> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de elementos solicitados. El resultado es más o menos detallado en función de la constante utilizada.


|   |   |
| --- | --- |
| *hLstDetail* no especificado | &lt;Nombre del elemento 1&gt; + TAB + &lt;Tipo&gt; + CR + &lt;Nombre del elemento 2&gt; + CR +.... |
| Se especifica *hLstDetail* | &lt;Nombre del elemento 1&gt; + TAB + &lt;Tipo&gt; + TAB + &lt;Nombre del autor&gt; + TAB + &lt;Fecha y hora de actualización&gt; + TAB + &lt;Nombre del autor de la actualización&gt; + TAB + &lt;Nombre del ordenador de actualización&gt; + TAB + &lt;Tamaño&gt; + CR +.... |

Donde:

- &lt;Nombre del elemento X: El nombre lógico del conjunto de Procedure s, el Procedure o la consulta. El nombre de la Procedure tiene el siguiente formato: &lt;Nombre del conjunto de Procedure s&gt;.&lt;Nombre de Procedure&gt;. 

- &lt;Tipo&gt;: Tipo de elemento: Valor utilizado para identificar el elemento listado:
	


|   |   |
| --- | --- |
| *eltProcedure* | El elemento listado es una Procedure. |
| *eltQuery* | El elemento listado es una consulta. |
| *eltSetOfProcedures* | El elemento listado es una set de procedimientos. |



- &lt;Nombre del autor&gt;&gt;Nombre del autor: Nombre de la persona que creó el elemento.

- Fecha y hora de actualización&gt; &lt;Fecha y hora de actualización: Fecha y hora de la última actualización realizada en el elemento.

- Nombre del autor de la actualización&gt; &lt;Nombre del autor de la actualización: Nombre de la persona que realizó la última actualización del elemento.

- Nombre del ordenador de actualización&gt; &lt;Nombre del ordenador de actualización: Nombre de el equipo que realizó la última actualización del elemento.




## HRSExecute (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta inmediatamente una replicación recurrente entre servidores HFSQL: la replicación se activa antes de la programación.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Cnt is Connection
ConfigReplication1 is hRSConfig
...
HRSExecute(Cnt, ConfigReplication1..Identifier)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HRSExecute(<Connection> , <Replication to trigger>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se activó la replicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor maestro HFSQL para replicación. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Replication to trigger>: (Integro)`**

Identificador de la réplica a desencadenar. Este identificador corresponde a la propiedad **Identifier** de la variable [hRSConfig](../WDLang4/1000020484.md) que describe la replicación.




## HRSInfo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la configuración de la replicación de un servidor HFSQL que participa en una o más replicaciones.

Esta función se puede ejecutar en un servidor maestro o en un servidor de suscriptores.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Cnt is Connection

Trace(HRSInfo(Cnt))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HRSInfo(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Información sobre las diferentes replicaciones realizadas en este servidor. Esta información tiene el siguiente formato:  


```txt
<Server role> + TAB + <Server identifier> + TAB + <Port> + TAB + <Priority>
```
donde: 

- &lt;Server role&gt; corresponde a una constante entera utilizada para averiguar el papel del servidor en la replicación.: 
	


|   |   |
| --- | --- |
| *hrsInactive* | Inactive en la replicación entre servidores HFSQL. |
| *hrsMaster* | Ordenador maestro en la replicación entre servidores HFSQL. |
| *hrsMaster* + *hrsSubscriber* | Ordenadores maestros y de abonado en la replicación entre servidores HFSQL (para una replicación bidireccional). |
| *hrsSubscriber* | Ordenador de abonado en la replicación entre servidores HFSQL. |



- &lt;El identificador de servidor&gt; corresponde al número utilizado para identificar el servidor HFSQL actual entre todos los servidores que participan en una replicación.

- &lt;Port&gt; corresponde al puerto TCP/IP utilizado para la comunicación entre servidores.

- &lt;Prioridad&gt; corresponde al número de prioridad del servidor utilizado en caso de resolución por prioridad durante un conflicto de modificación.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

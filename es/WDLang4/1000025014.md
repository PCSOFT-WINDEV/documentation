


## &lt;Fuente&gt;.ListLink (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de enlaces de un archivo de datos (Merise) en el análisis actual o en un análisis específico.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the links of the Customer data file in the current analysis
ListLink is string
ListLink = Customer.ListLink(hLstDetail)
// Adds the list of links in COMBO_COMBO1
COMBO_COMBO1.Add(ListLink)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Listado de enlaces de un archivo de datos en un análisis

`<Result> = <Source>.ListLink([<Path of the WDD file> [, <Password of the WDD file>]] [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de relaciones. Las diferentes descripciones de los enlaces están separadas por caracteres CR (Carriage Return). El resultado es más o menos detallado en función de la constante utilizada:


|   |   |
| --- | --- |
| *hLstDetail* no especificado | &lt;Nombre de la relación 1&gt; + TAB + &lt;Nombre del archivo 1.1&gt; + TAB + &lt;Nombre del elemento1.1&gt; + TAB + &lt;Cardinalidad 1.1&gt; + TAB + &lt;Nombre del archivo 1.2&gt; + TAB + &lt;Nombre del elemento 1.2&gt; + TAB + &lt;Cardinalidad 1.2&gt; + CR +... |
| Se especifica *hLstDetail* | &lt;Nombre de la relación1&gt; + TAB + &lt;Nombre del archivo1.1&gt; + TAB + &lt;Nombre del elemento1.1&gt; + TAB + &lt;Cardinalidad 1.1&gt; + TAB + &lt;Nombre del archivo1.2&gt; + TAB + &lt;Nombre del elemento1.2&gt; + TAB + &lt;Cardinalidad 1.2&gt; + TAB + &lt;Regla de modificación&gt; + TAB + &lt;Regla de eliminación&gt; + CR +... |


Donde:

- &lt;Nombre del enlace X&gt;: nombre del enlace.

- &lt;Archivo de nombre x.1>:: El nombre del archivo de datos relacionado con la clave primaria.

- &lt;Nombre del ítem x.1&gt;: Nombre del elemento relacionado con la clave primaria o nombre del archivo de datos especificado.

- &lt;Cardinalidad x.1&gt;: Cardinalidad del vínculo relacionado con la clave primaria en el siguiente formato: "&lt;Cardinalidad minúscula&gt;, &lt;Cardinalidad máx.&gt;" ("0, N" o "1, N" por ejemplo).

- &lt;Archivo de nombre x.2&gt;: El nombre del archivo de datos relacionado con la llave extranjera.

- &lt;Nombre del ítem x.2&gt;: Nombre del elemento relacionado con la clave externa.

- &lt;Cardinalidad x.2&gt;: Cardinalidad del vínculo relacionado con la clave foránea en el siguiente formato: "&lt;Cardenalidad minúscula&gt;, &lt;Cardenalidad máxima&gt;" ("0, N" o "1, N" por ejemplo).

- &lt;Regla de modificación: Regla a aplicar si se modifica la llave primaria.

- &lt;Regla de borrado: Regla a aplicar si se borra la llave primaria.


![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) El nombre de los archivos de datos corresponde al nombre de la base de datos + '\\' + el nombre físico relativo de estos archivos de datos.![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) Si uno de los enlaces listados se realiza en una clave compuesta, el parámetro &lt;Name item 1.2&gt; devuelto corresponderá al nombre de los diferentes componentes clave. Los nombres estarán separados por una coma.

**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos cuyos enlaces deben ser listados. Los enlaces que se originan desde este archivo o que terminan en este archivo se enumeran a continuación. 

**`<Path of the WDD file>: (Cadena de caracteres opcional)`**

ruta completa del archivo de análisis (archivo.WDD). Si este parámetro no se especifica o si es igual a una cadena vacía ("""), los enlaces se buscan en el análisis actual.

**`<Password of the WDD file>: (Cadena de caracteres opcional)`**

Contraseña definida para el análisis. Si no se define ninguna contraseña, este parámetro debe ser igual a una cadena vacía ("").

**`<Options>: (Constante opcional de tipo Integer)`**

Tipo de lista a realizar:


|   |   |
| --- | --- |
| *hLstDetail* | Enumera la información detallada sobre los enlaces. |
| *hLstNormal*<br>(valor predeterminado) | Enumera los enlaces. |

![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta.

## Observaciones


### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000370"></a>

&lt;Resultado&gt; puede asignarse directamente a un control List Box o Combo Box mediante la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md). Cada fila contendrá el nombre completo de uno de los enlaces encontrados.
<a name="NOTE0_2"></a>


### Regla de borrado y modificación
<a name="regla_borrado_modificacion_ELTPARAGRAPHE000385"></a>

Las reglas de modificación y supresión devueltas por **&lt;Fuente&gt;.ListLink** pueden corresponder a las siguientes constantes:


|   |   |
| --- | --- |
| *hIntegrityCascade* | Gestionar la modificación y eliminación en cascada. |
| *hIntegrityDefaultValue* | Durante la eliminación, se asigna un valor Default a los registros relacionado. |
| *hIntegrityForbidden* | Estricto respeto a la integridad. |
| *hIntegrityNone* | No hay restricciones de integridad. |

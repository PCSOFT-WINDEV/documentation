


## &lt;Conexión Variable&gt;.RSModifyConfig (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica algunos parámetros de una replicación existente entre dos servidores HFSQL. Esta función debe ejecutarse en un servidor maestro HFSQL para la replicación.. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
ConfigReplication1 is hRSConfig
...

ConfigReplication1.Scheduling.Hour = "*/2"
MyConnection.RSModifyConfig(ConfigReplication1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.RSModifyConfig(<Replication>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la replicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre del [Conexión](../WDLang4/1514073.md) Variable que describe la conexión con el servidor HFSQL maestro en el que se debe cambiar la configuración de la replicación.

**`<Replication>: (hRSConfig Variable)`**

Nombre de la [hRSConfig](../WDLang4/1000020484.md) Variable que describe la réplica modificada a actualizar.

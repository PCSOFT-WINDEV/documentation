


## HClose (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un archivo de datos o todos los archivos de datos abiertos por el usuario actual: todos los archivos de datos físicos correspondientes se cierran para el usuario actual. Para los archivos de datos HFSQL, se cierran los siguientes archivos:

- el archivo de datos (.FIC),

- el archivo índice (.NDX) si existe,

- el archivo de memo (.MMO) si existe.

- el archivo de índice de texto completo (.FTX) si existe.




**Al cerrar el archivo de datos**: 

- Las cerraduras (en el archivo de datos y/o algunos de sus registros) están desbloqueadas.

- Los filtros definidos en el archivo de datos con [HFilter](../WDLang4/3044100.md) se borran automáticamente. 

- Las iteraciones actuales en el archivo de datos se cancelan. 

- Las búsquedas actuales en el archivo de datos se cancelan. 




Una vez que se cierra el archivo de datos, se conservan las reasignaciones realizadas por [HChangeName](../WDLang4/3044141.md), [HChangeDir](../WDLang4/3044168.md) y [HSubstDir](../WDLang4/3044028.md).

Si se ha ejecutado una consulta en el archivo de datos, se debe llamar a [HCancelDeclaration](../WDLang4/3044174.md) en la consulta para que el cierre sea efectivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the "Supplier" data file
HClose(Supplier)
```
<a name="Example2"></a>

```wl
IF NOT HClose("") THEN
	Error ("Problem when closing data files")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HClose([<Data file>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos estuviera cerrado,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos a cerrar. 
Si no se especifica este nombre, **HClose** cerrará el último archivo de datos utilizado por la última función de gestión HFSQL (función que empieza por "H").
Si este parámetro es igual a "\*" " o si corresponde a una cadena vacía (""), se cerrarán todos los archivos de datos abiertos.. 
Este parámetro también puede corresponder al nombre de un [grupo de archivos (o Carpeta Personal) definido en el análisis](../Editeurs/2011003.md) (no disponible en Android y Android Widget).

## Observaciones


### Cerrar las mesas y la conexión
<a name="cerrar_las_mesas_conexion_ELTPARAGRAPHE000141"></a>

Cuando todas las tablas que utilizan la misma conexión (tablas abiertas en la misma base de datos con los mismos parámetros de conexión) se cierran, la conexión se cierra automáticamente.

Para cerrar una conexión, utilice [HCloseConnection](../WDLang4/3044095.md).

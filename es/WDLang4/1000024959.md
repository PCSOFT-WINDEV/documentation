


## &lt;Fuente&gt;.FileExist (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si un archivo de datos existe, es decir, si se ha creado físicamente ([&lt;Fuente&gt;.Creation](../WDLang4/1000024207.md) o [&lt;Fuente&gt;.CreationIfNotFound](../WDLang4/1000024208.md)). 

Observación: Se tienen en cuenta las sustituciones realizadas actualmente en el archivo de datos (por [HSubstDir](../WDLang4/3044028.md), [&lt;Fuente&gt;.ChangeDir](../WDLang4/1000024202.md) y [&lt;Fuente&gt;.ChangeName](../WDLang4/1000024201.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check whether the "Customer" data file exists
IF Customer.FileExist() = True THEN
	// Processes
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Comprobar si un archivo de datos, consulta o vista existe

`<Result> = <Source>.FileExist()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si existe el archivo físico, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del elemento a utilizar: 

- Nombre del archivo de datos.

- Nombre de la vista o consulta.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **&lt;Fuente&gt;.FileExist** puede usarse sin importar si el archivo de datos está abierto o no.

- **&lt;Fuente&gt;.FileExist** no abre el archivo de datos si está cerrado.

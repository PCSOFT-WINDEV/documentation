


## HUpdateView (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea o actualiza la descripción de una vista SQL o vista materializada del análisis a un servidor HFSQL.

Observación: Los datos de la vista son actualizados por [HRefreshView](../WDLang4/1000021376.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Updates the description of the view on the HFSQL server
HUpdateView(MyConnection, MyView)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HUpdateView(<Connection> , <View> [, <Force>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó o actualizó la vista (SQL o materializada). 

- <u><u><u><u>False</u></u></u></u> si se produce un problema.  La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor en el que se debe crear o actualizar la vista SQL o materializada.. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<View>: (Cadena de caracteres)`**

Nombre del SQL o vista materializada a utilizar. Esta vista debe ser creada o actualizada en el servidor HFSQL. 

**`<Force>: (Booleano opcional)`**



- <u><u><u><u>False</u></u></u></u> (valor predeterminado): la vista se creará o actualizará:

	- si la vista se encuentra en el servidor.

	- si se encuentra una versión anterior de la vista en el servidor.




- <u><u><u><u>True</u></u></u></u>: la vista se actualizará en cualquier caso.

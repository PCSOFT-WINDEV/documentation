


## MongoFind (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra documentos en una colección de MongoDB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000022623&name=mongofind_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
sFilter is string
sFilter = "{ ""name"": ""France"" }"

let MySearch = MongoFind(CollecMDB, sFilter)

// Read the result
FOR EACH item of MySearch
	Trace(VariantToJson(item))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MongoFind(<Collection> , <Filter> [, <Option>])`
---

**`<Result>: (Variable de tipo mongoResult)`**

[mongoResult](../WDLang4/1000022415.md) variable correspondiente al resultado de la búsqueda. Si se produce un problema, la función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.

**`<Collection>: (Variable de tipo mongoCollection)`**

Nombre de la [mongoCollection](../WDLang4/1000022411.md) Variable correspondiente a la colección de MongoDB donde se realizará la búsqueda.

**`<Filter>: (Tipo correspondiente al filtro utilizado)`**

description de filtros en formato MongoDB que se aplicarán para realizar la búsqueda. Este filtro corresponde a: 

- un Document en formato JSON, en formato de cadena de caracteres. 

- un Document en formato BSON, en formato buffer. 

- una variante. 




**`<Option>: (Opcional mongoFindOption Variable)`**

Nombre de [mongoFindOption](../WDLang4/1000022414.md) Variable correspondiente a las opciones de búsqueda.

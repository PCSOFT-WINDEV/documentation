


## &lt;Fuente&gt;.TrackStop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deja de rastrear a un archivo de datos en un servidor HFSQL. Este tema fue iniciado por [&lt;Fuente&gt;.Track](../WDLang4/1000025029.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts tracking the Order data file
// The ProcessOrder procedure is called whenever a record is added to the data file
Order.Track(ProcessOrder, hRecNumAll, hsAdd)

...
// Stops tracking the Order data file
Order.TrackStop(ProcessOrder, hRecNumAll)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Source>.TrackStop(<WLanguage procedure> [, <Record number>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si de detuvo la supervisión,

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, supervisión inexistente). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos HFSQL a utilizar. 

**`<WLanguage procedure>: (Nombre de procedimiento)`**

El nombre de WLanguage Procedure no se llamará más para rastrear a el archivo de datos.

**`<Record number>: (Entero opcional)`**

Número de la Record que ya no se puede rastrear. 
Si este parámetro no se especifica o corresponde a la constante *hRecNumAll*, **&lt;Fuente&gt;.TrackStop** dejará de seguir todos los registros de el archivo de datos (es decir, todo el archivo de datos).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las funciones [&lt;Fuente&gt;.Track](../WDLang4/1000025029.md) y **&lt;Fuente&gt;.TrackStop** se pueden utilizar en un servicio de Windows.

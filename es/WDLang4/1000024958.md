


## &lt;Fuente&gt;.Close (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un archivo de datos o todos los archivos de datos abiertos por el usuario actual: todos los archivos de datos físicos correspondientes se cierran para el usuario actual. Para los archivos de datos HFSQL, se cierran los siguientes archivos:

- el archivo de datos (.FIC),

- el archivo índice (.NDX) si existe,

- el archivo de memo (.MMO) si existe.

- el archivo de índice de texto completo (.FTX) si existe.




**Al cerrar el archivo de datos**: 

- Las cerraduras (en el archivo de datos y/o algunos de sus registros) están desbloqueadas.

- Los filtros definidos en el archivo de datos con [&lt;Fuente&gt;.Filter](../WDLang4/1000024960.md) se borran automáticamente. 

- Las iteraciones actuales en el archivo de datos se cancelan. 

- Las búsquedas actuales en el archivo de datos se cancelan. 




Una vez que se cierra el archivo de datos, se conservan las reasignaciones realizadas por [&lt;Fuente&gt;.ChangeName](../WDLang4/1000024201.md), [&lt;Fuente&gt;.ChangeDir](../WDLang4/1000024202.md) y [HSubstDir](../WDLang4/3044028.md).

Si se ha ejecutado una consulta en el archivo de datos, se debe llamar a [&lt;Fuente&gt;.CancelDeclaration](../WDLang4/1000024192.md) en la consulta para que el cierre sea efectivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the "Supplier" data file
Supplier.Close()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.Close()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos estuviera cerrado,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos a cerrar.

## Observaciones


### Cerrar las mesas y la conexión
<a name="cerrar_las_mesas_conexion_ELTPARAGRAPHE000270"></a>

Cuando todas las tablas que utilizan la misma conexión (tablas abiertas en la misma base de datos con los mismos parámetros de conexión) se cierran, la conexión se cierra automáticamente.

Para cerrar una conexión, utilice [&lt;Variable Connection&gt;.CloseConnection](../WDLang4/1000023941.md).

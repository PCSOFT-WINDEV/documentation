


## &lt;Fuente&gt;.SetMemo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite modificar el modo de gestión de elementos de tipo memo. Tienes la habilidad de manejar todos los memos de un archivo de datos o un memo específico de un archivo de datos. La gestión de los memorandos está habilitada por Default.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) La gestión de memorandos puede ser implementada sólo para el archivo de datos: las notas no pueden gestionarse para las posiciones.![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png) Cuando se llama a **&lt;Fuente&gt;.SetMemo**, se cierran todas las iteraciones iniciadas. Se mantienen el filtro y los criterios de búsqueda.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable management of memos on observations of the Customer data file
Customer.SetMemo(Item_Observations, hMemoNo)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.SetMemo([<Memo item>, ] <Management mode>)`
---

**`<Result>: (Entero)`**

Antiguo modo de gestión de memorias utilizado (antes de llamar a **&lt;Fuente&gt;.SetMemo**).


|   |   |
| --- | --- |
| *hMemoAuto*<br>(valor predeterminado) | Las notas se leen durante el acceso a la posición. |
| *hMemoNo* | No hay gestión de memos (no se realizará ninguna operación de lectura o escritura en el archivo de datos o en el elemento especificado). |
| *hMemoYes* | Los memorándums de texto se leen cada vez que se lee la Record.<br>Los memos binarios se leen durante el acceso al elemento. |



**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Memo item>: (Cadena de caracteres opcional)`**

Nombre de la posición de nota a gestionar o no. 
Si no se especifica este nombre o si es igual a "\*" ", se tienen en cuenta todas las partidas pro memoria..
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) La gestión de memorandos puede ser implementada sólo para el archivo de datos: las notas no pueden gestionarse para las posiciones.

**`<Management mode>: (Constante de tipo Integer)`**

Modo de gestión de las posiciones pro memoria:


|   |   |
| --- | --- |
| *hMemoAuto*<br>(valor predeterminado) | Las notas se leen durante el acceso a la posición. |
| *hMemoNo* | No hay gestión de memos (no se realizará ninguna operación de lectura o escritura en el archivo de datos o en el elemento especificado). |
| *hMemoYes* | Los memorándums de texto se leen cada vez que se lee la Record.<br>Los memos binarios se leen durante el acceso al elemento. |


![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png) La constante *hMemoAuto* es equivalente a la constante *hMemoYes*.

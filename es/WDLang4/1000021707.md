
## hbCell (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **hbCell** se utiliza para definir las características avanzadas del contenido de una celda (valor del elemento) en una HBase tabla. Las características de este contenido pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 
Este tipo de Variable es utilizado por: 

- la función [hbLeer](../WDLang4/1000021706.md): el resultado de la función [hbLeer](../WDLang4/1000021706.md) es un array de **hbCell**.

- variables de tipo [hbWriting](../WDLang4/1000021710.md), para especificar los valores a escribir. 




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Description of data to write
Writing is hbWriting
Writing.Row = "post1"
Cell is hbCell 
Cell.Column.Family = "post"
Cell.Column.Qualifier = "title"
Cell.Value = "This is my blog post."
Add(Writing.Cell, Cell)
// Write data
hbWrite(Connection, "blog", Writing)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables hbCell
<a name="propiedades_especificas_description_las_variables_hbcell_ELTPARAGRAPHE000043"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un contenido celular:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Columna | Variable de tipo [hbColumn](../WDLang4/1000021698.md) | Columna associated con la célula. |
| Timestamp | FechaHora | Marca de tiempo del valor leído. |
| Valor | Buffer | Contenido de la célula. |

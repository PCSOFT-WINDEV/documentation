


## &lt;Fuente&gt;.RestaurarPosición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restablece el contexto previamente guardado de un archivo de datos ([&lt;Fuente&gt;.SavePosition](../WDLang4/1000025062.md)): registro actual, filtro, punteros.

La función **&lt;Fuente&gt;.RestaurarPosición** permite restaurar:

- los diferentes punteros del índice,

- el filtro actual.




Esta función se puede utilizar con archivos de datos, vistas HFSQL o consultas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find a record
Customer.ReadSeekFirst(Name, Name)
WHILE Customer.Found() = True
	// Saves the current context
	nPos = Customer.SavePosition(Name)
	// Check a customer with another name
	Customer.ReadSeek(Name, Name2)
	IF Customer.Found() = True THEN Counter = Counter + 1
	// Restores the context
	HRestorePosition(nPos)
	Customer.ReadNext(Name)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.RestorePosition()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se restauró la posición especificada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos cuyo contexto debe ser restaurado.

## Observaciones
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000251"></a>

- El número de llamadas a [&lt;Fuente&gt;.SavePosition](../WDLang4/1000025062.md) no está limitado.

- Puede anidar [&lt;Fuente&gt;.SavePosition](../WDLang4/1000025062.md) y **&lt;Fuente&gt;.RestaurarPosición** para el mismo archivo de datos.

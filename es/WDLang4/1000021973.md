
## hdfsStatus (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **hdfsStatus** permite conocer el estado de un elemento (archivo, directorio, enlace) del sistema de archivos Hadoop. Las características de este estado son devueltas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
hdfsOpenConnection(MyConnection)
// Retrieve the directory content
Content is array of hdfsStatus = hdfsList(MyConnection, "/")
// Display the directory content
FOR EACH Elem OF Content
	Trace(Elem.Path)
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables hdfsStatus
<a name="propiedades_especificas_descripcion_las_variables_hdfsstatus_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser usadas para manipular un estado de elemento en el sistema de archivos Hadoop:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| DateLastAccessed | FechaHora | Fecha y hora del último acceso al elemento.<br>**Esta propiedad es de solo lectura**.<br><br>Esta constante corresponde a "LastAccessDate" en las versiones anteriores. |
| DateModified | FechaHora | Fecha y hora de la última modificación realizada en el elemento.<br>**Esta propiedad es de solo lectura**.<br><br>Esta constante corresponde a "ModificationDate" en las versiones anteriores. |
| Derechos | Variable de tipo [hdfsRights](../WDLang4/1000021662.md) | Derechos de acceso al elemento.<br>**Esta propiedad es de solo lectura**. |
| envergadura | Integro | Tamaño del elemento en bytes.<br>**Esta propiedad es de solo lectura**. |
| escribir a máquina | Integro | Tipo de elemento:<br><br>- *hdfsFile* si el elemento es un archivo.<br><br>- *hdfsFile* si el elemento es un directorio.<br><br>- *hdfsLink* si el elemento es un enlace a un archivo.<br><br><br>**Esta propiedad es de solo lectura**. |
| Grupo | Cadena de caracteres | Nombre del grupo al que pertenece el elemento.<br>**Esta propiedad es de solo lectura**. |
| NúmeroReplicat | Integro | Número de copias almacenadas para los elementos.<br>**Esta propiedad es de solo lectura**. |
| Propietario | Cadena de caracteres | Propietario del elemento.<br>**Esta propiedad es de solo lectura**. |
| Ruta | Cadena de caracteres | Ruta del elemento en el sistema de archivos Hadoop<br>**Esta propiedad es de solo lectura**. |
| Tamaño de bloque | Integro | Tamaño de los bloques para la distribución del elemento en los diferentes servidores.<br>**Esta propiedad es de solo lectura**. |
| vincular | Cadena de caracteres | Destino del enlace, si el elemento es un enlace.<br>**Esta propiedad es de solo lectura**. |

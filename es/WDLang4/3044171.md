
## HDBCreation( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la descripción de la estructura de un fichero de datos xBase mediante programación. El archivo de datos que se acaba de describir se crea en el disco en el ruta especificado en [HDBDescribeFile](../WDLang4/3044172.md). Los archivos de datos, notas e índices se crean en el disco.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDBDescribeFile("DBCUSTOMER", "CD", "C:\FILE\CUSTOMER.DBF")
HDBDescribeField("LASTNAME,C,20")
HDBDescribeField("FIRSTNAME,C,20")
HDBDescribeField("AGE,N,3,0")
HDBDescribeField("BALANCE,N,10,2")
HDBDescribeField("MARRIED,L")
HDBDescribeField("DOB,D")
HDBDescribeField("INFO,M")
HDBCreation()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDBCreation()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Creación de un archivo xBase a través del Native xBase Connector
<a name="creacion_archivo_xbase_traves_del_native_xbase_connector_ELTPARAGRAPHE000047"></a>

- Los archivos de datos e índices se crean en formato dBase 3 (archivo NDX).

- Después de la función **HDBCreation**, se puede utilizar el archivo sin llamar a las funciones [HDBOpen](../WDLang4/3044163.md) y [HDBIndex](../WDLang4/3044165.md).



<a name="NOTE0_2"></a>




### Creación de un archivo xBase
<a name="creacion_archivo_xbase_ELTPARAGRAPHE000064"></a>

- Si el archivo ya existe, se sobrescribe con el nuevo archivo.

- El error 1009 se producirá si el archivo se abre con **HDBCreation**.



<a name="NOTE0_3"></a>




### Creación de un fichero xBase en la editor de análisis
<a name="creacion_fichero_xbase_editor_analisis_ELTPARAGRAPHE000075"></a>

También se puede describir y crear un archivo xBase en el editor. También tiene la posibilidad de importar un archivo xBase existente en la editor de análisis.
<a name="NOTE0_4"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000082"></a>

- Esta función no está disponible cuando se utiliza una base de datos a través de .

- Esta función no está disponible cuando se utiliza un ejecutable independiente..

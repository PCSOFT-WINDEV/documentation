


## HSetTransaction (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la gestión de transacciones para uno o más archivos de datos.

Cuando se desactiva la gestión de las transacciones, se inhiben todas las funciones relacionadas con las transacciones ([HTransactionStart](../WDLang4/3044002.md), [HTransactionEnd](../WDLang4/3044032.md), [HTransactionCancel](../WDLang4/3044001.md)).

**Observación**: La gestión de las transacciones está habilitada por Default.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable the management of transactions on the Customer data file
HSetTransaction(Customer, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HSetTransaction([<Data file>, ] <Options>)`
---

**`<Result>: (booleano)`**

Corresponde a la anterior gestión de transacciones (antes de utilizar la función **HSetTransaction**): 

- <u><u><u><u>True STRIMEULB</u></u></u></u> si la gestión de las transacciones está habilitada,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si se ha desactivado la gestión de operaciones. 


Si &lt;Archivo de datos&gt;= "\*", &lt;Resultado&gt; es set a <u><u><u><u>True</u></u></u></u> si la gestión de las transacciones se habilitó para al menos un archivo de datos en el modo de gestión de transacciones anterior.

**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre o si corresponde a "\*" ", la gestión de las transacciones se tiene en cuenta para todos los ficheros de datos encontrados en el análisis.

**`<Options>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la gestión de las transacciones debe ser habilitada para el archivo de datos, 

- <u><u><u><u>False STRIMEULB</u></u></u></u> si esta gestión debe ser desactivada.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de transacciones habilitada
<a name="gestion_transacciones_habilitada_ELTPARAGRAPHE000121"></a>

Si se habilita la gestión de transacciones para un solo fichero, sólo se tendrán en cuenta en la gestión de transacciones las operaciones realizadas en este fichero de datos (independientemente de los ficheros de datos especificados en [HTransactionStart](../WDLang4/3044002.md)).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Límite 
<a name="limite_ELTPARAGRAPHE000135"></a>

La gestión de las transacciones no se aplica a los archivos xBase.

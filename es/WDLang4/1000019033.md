## Funciones para gestionar un clúster HFSQL
<a name="funciones_para_gestionar_cluster_hfsql_ELTTEXTE000066"></a>
Las siguientes funciones se utilizan para gestionar un clúster HFSQL:



|   |   |
| --- | --- |
| [HClusterAddNode](../WDLang4/1000019061.md) | Habilita un nodo en un cluster HFSQL. |
| [HClusterDeleteNode](../WDLang4/1000019062.md) | Desactiva un nodo en un cluster HFSQL. |
| [HClusterIgnoreSynchro](../WDLang4/1000019243.md) | Define un nodo del cluster HFSQL como fuente de datos para realizar la sincronización del cluster. |
| [HClusterMasterNode](../WDLang4/1410087885.md) | Devuelve el nombre del nodo maestro de un cluster HFSQL. |
| [HClusterNodeInfo](../WDLang4/1000019060.md) | Retorna el estado cada nodo del cluster interrogando a su coordinador. |
| [HClusterParameter](../WDLang4/1000019305.md) | Lee y modifica los parámetros de un clúster HFSQL. |
| [HClusterStart](../WDLang4/1000019057.md) | Inicia un clúster HFSQL. Este cúmulo fue detenido de antemano (por [HClusterStop](../WDLang4/1000019058.md) por ejemplo). |
| [HClusterState](../WDLang4/1000019059.md) | Devuelve el estado de un cluster HFSQL interrogando a su coordinador. |
| [HClusterStop](../WDLang4/1000019058.md) | Suspende la ejecución de un cluster HFSQL. Los nodos del cluster ya no aceptarán ninguna conexión de las aplicaciones cliente.. |
| [HClusterSynchronizationErrorInfo](../WDLang4/1000021975.md) | Devuelve el error que se produjo durante el paso de sincronización anterior del clúster HFSQL. |
| [HClusterSynchronizationInfo](../WDLang4/1000021974.md) | Devuelve el progreso de la sincronización de un nodo de un cluster HFSQL. |





También tiene la posibilidad de utilizar las siguientes funciones: 



|   |   |
| --- | --- |
| [HDeleteParameter](../WDLang4/1000017315.md) | Elimina un parámetro guardado previamente con la función [HSaveParameter](../WDLang4/1000017313.md). |
| [HLoadParameter](../WDLang4/1000017314.md) | Lee un parámetro guardado previamente desde un procedimiento almacenado con la función [HSaveParameter](../WDLang4/1000017313.md). |
| [HSaveParameter](../WDLang4/1000017313.md) | Guarda un valor persistente en el servidor HFSQL. |

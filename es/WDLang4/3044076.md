
## H. NumRec (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Contiene el número de la Record física cargada en la memoria (lo que significa el número de la última Record leída).

Esta Variable se actualiza cada vez que se utiliza una función de lectura ([HReadFirst](../WDLang4/3044051.md), [HReadNext](../WDLang4/3044037.md), [HRead](../WDLang4/3044047.md), etc.)


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Uso de H.NumRec
<a name="uso_hnumrec_ELTPARAGRAPHE000031"></a>

El **H.NumRec** Variable puede ser usado en: 

- los archivos de datos HFSQL.

- las vistas HFSQL.

- las consultas en los archivos de datos HFSQL. 




En estos dos últimos casos, la variable **H.NumRec** corresponde al número Record del archivo de datos associated con la vista o con la consulta.

En los archivos de xBase, el **H.NumRec** Variable es siempre igual a 0.

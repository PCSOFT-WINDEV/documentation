


## HToFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia una fuente de datos (archivo, consulta, vista, ...) a un archivo físico de datos HFSQL con la misma description. Este archivo de datos no está ni cifrado ni protegido por contraseña.  

Esta función se utiliza para grabar un historial, por ejemplo: una consulta selecciona todos los registros encontrados para el año 2019, el resultado se transfiere a un archivo de datos y una nueva consulta borra todos los registros encontrados para el año 2019.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF HToFile(MyView, "C:\temp\Test.fic") = True THEN
	Info("Data transferred into a data file")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HToFile(<Data source> , <Destination file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el archivo de destino ya existe, por ejemplo).




**`<Data source>: (Cadena de caracteres)`**

Nombre lógico de la fuente de datos a copy (archivo de datos, vista, consulta, ...).

**`<Destination file>: (Cadena de caracteres)`**

Nombre completo del archivo de datos físicos en el que deben copiarse los registros encontrados en la fuente. Este archivo de datos y la fuente de datos tendrán la misma description. Para acceder a este archivo de datos, utilice, por ejemplo, [HDeclareExternal](../WDLang4/3044204.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Fuente filtrada
<a name="fuente_filtrada_ELTPARAGRAPHE000095"></a>

Sólo se copiarán los registros correspondientes al filtro si se define y habilita un filtro en la fuente de datos.
<a name="NOTE0_2"></a>


### Fuente de datos: Ver o consultar
<a name="fuente_datos_ver_consultar_ELTPARAGRAPHE000104"></a>

Si la fuente de datos es una vista o una consulta, todos los elementos serán claves del archivo de datos creado, excepto:

- Partidas pro memoria.

- cuyo tamaño supere los 260 caracteres y que no se introduzcan en el archivo fuente de la vista o consulta.




## Funciones para la gestión de HFSQL Client/Server (sintaxis del prefijo)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
Se utilizan varias funciones HFSQL para gestionar los datos de HFSQL Client/Server.

Estas funciones se utilizan principalmente para:

- Gestionar los usuarios y los grupos de usuarios (creación, derechos,...)

- Gestionar los archivos de datos que se encuentran en el servidor

- Gestionar la integridad referencial (funciones para la gestión de los enlaces)

- Gestionar el servidor.


## HListServerTrigger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los disparadores disponibles en una conexión o en los archivos de conexión.  

**Observación**: Las funciones para manejar los disparadores del servidor son funciones avanzadas. Los triggers del servidor se pueden crear directamente en el editor de análisis. Para obtener más información, consulte [Disparadores de servidor](../WDLang4/3044369.md).




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListServerTrigger(<Connection> [, <Physical HFSQL Client/Server files>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de activadores de servidor aplicados a la conexión especificada. Los diferentes desencadenantes están separados por caracteres CR (Carriage Return).
La lista tiene el siguiente formato:

```txt
<Trigger> + TAB + <Data file> + TAB + <WLanguage procedure> + TAB + 
<Function number> + TAB + <Type of trigger> + TAB + <Trigger status> + CR +...
```



|   |   |
| --- | --- |
| &lt;Disparador&gt; | Nombre del trigger. |
| &lt;Fecha del expediente&gt; | Nombre físico del archivo de datos al que se aplica el disparador. |
| &lt;WLanguage Procedure&gt; | Nombre del WLanguage Procedure almacenado llamado por el gatillo. |
| &lt;Número de función&gt; | Identificador de la función WLanguage en la que está activado el trigger del servidor. <br>Para conocer el nombre de la función WLanguage utilizada, consulte el archivo ListeDéfinitionHF.WL (que se encuentra en el subdirectorio "Personal\\External\\" de WINDEV, WINDEV Mobile o WEBDEV). |
| &lt;Tipo de trigger&gt; | <br><br>- "A" para un disparador "ANTES" (el Procedure associated con el disparador se ejecuta **antes de que** la función HFSQL del disparador).<br><br>- "P" para un disparador "DESPUÉS" (el Procedure associated con el disparador se ejecuta **after** la función HFSQL del disparador).<br><br><br> |
| &lt;Estado del Trigger&gt; | <br><br>- "1" para una activación de active ([HActivateServerTrigger](../WDLang4/1000017034.md)),<br><br>- "0" para una activación de inactive ([HDeactivateServerTrigger](../WDLang4/1000017033.md)).<br><br><br> |



**`<Connection>: (Cadena de caracteres opcional o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Physical HFSQL Client/Server files>: (Cadena de caracteres opcional)`**

Nombre físico de uno o más archivos de datos de HFSQL Client/Server associated con la conexión (por ejemplo "CUSTOMER.FIC"). Para especificar varios nombres, los nombres de los archivos de datos deben estar separados por una coma (","). 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el trigger se habilitará en todos los ficheros de datos de la base de datos..

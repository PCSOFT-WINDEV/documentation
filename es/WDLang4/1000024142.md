


## Funciones de gestión de Redis
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para administrar una base de datos Redis: 



|   |   |
| --- | --- |
| [RedisConnect](../WDLang4/1000023530.md) | Abre una conexión a un servidor (o cluster) de Redis. |
| [RedisEjecutarComando](../WDLang4/1000023541.md) | Ejecuta un comando en un servidor Redis y recupera el resultado. |
| [RedisGet](../WDLang4/1410088171.md) | Obtiene el valor de una clave de un servidor Redis como una cadena. |
| [RedisPipelineAdd](../WDLang4/1000023542.md) | Agrega un comando Redis en un pipeline. |
| [RedisPipelineGet](../WDLang4/1000023543.md) | Recupera el resultado de la ejecución de un comando Redis que fue añadido por [RedisPipelineAdd](../WDLang4/1000023542.md). |
| [RedisPipelineReset](../WDLang4/1000023549.md) | Ignora los resultados de la ejecución de un comando Redis que fue añadido por [RedisPipelineAdd](../WDLang4/1000023542.md). |
| [RedisPublish](../WDLang4/1000023551.md) | Publica un mensaje en un canal de difusión de Redis. |
| [RedisSet](../WDLang4/1410088170.md) | Cambia el valor de una clave en un servidor Redis. |
| [redisSubscribe](../WDLang4/1000023552.md) | Suscripción a un canal de emisión de Redis. |
| [redisUnsubscribe](../WDLang4/1000023553.md) | Cancela una suscripción a un canal de difusión de Redis. |




## &lt;Variable Connection&gt;.ModifyUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la información de un usuario en función de los elementos que se encuentran en la correspondiente [variables para la gestión de usuarios](../WDLang4/3044315.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Associate the data file with the connection
Customer.ChangeConnection(MyConnection)
IF MyConnection.InfoUser("Flora") = True THEN
	// Modify the characteristics
	HUser.Groups = "ADMIN" + CR + "TEST" + CR + "DOC"
	IF MyConnection.ModifyUser("Flora") = True THEN
		Info("User modified")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.ModifyUser(<User>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el usuario,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que hay que modificar un usuario.

**`<User>: (Cadena de caracteres)`**

Nombre del usuario cuyas características deben modificarse. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios para modificar un usuario
<a name="derechos_necesarios_para_modificar_usuario_ELTPARAGRAPHE000245"></a>

Para modificar un usuario (que no sea el actual), debe tener el derecho de ver a los usuarios (constante *hRightsSeeUser*). En caso contrario, **&lt;Variable Connection&gt;.ModifyUser** devolverá <u><u><u><u>False</u></u></u></u> y aparecerá un mensaje de error "Usuario desconocido".

Para modificar el nombre de un usuario, debe tener permisos para ver otros usuarios (constante *hRightsSeeUser*) y para administrarlos (constante *hRightsManageUser*).

Observación: Estas constantes se utilizan en [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) y [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md).

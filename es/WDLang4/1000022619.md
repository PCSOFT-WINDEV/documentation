
## Bases de datos MongoDB/GRIDFS
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000138"></a>
MongoDB es una base de datos NoSQL orientada a Document. MongoDB almacena documentos en formato JSON en "colecciones".. Una «collection» es un conjunto de documentos.

MongoDB propone métodos avanzados para encontrar estos documentos.

MongoDB propone "reducir mapa".

GridFS es un sistema de archivos distribuidos gestionados por MongoDB.

## Manejo de datos MongoDB en WLanguage

### Manejo de una base de datos MongoDB
<a name="manejo_una_base_datos_mongodb_ELTPARAGRAPHE000033"></a>

Varios tipos de WLanguage y funciones le permiten manipular una base de datos MongoDB. 

Estas funciones se utilizan para: 

- Crear una conexión, una base de datos o una colección ([MongoCreate](../WDLang4/1000022408.md), [MongoCreateCollection](../WDLang4/1000022434.md), etc.). 

- Añadir, editar o eliminar documentos en una base de datos MongoDB ([MongoAdd](../WDLang4/1000022432.md), [MongoModifyOne](../WDLang4/1000022443.md), [MongoModifyAll](../WDLang4/1000022603.md), [MongoDeleteOne](../WDLang4/1000022446.md), etc.). 

- Borrar una base de datos, una colección ([MongoDeleteDatabase](../WDLang4/1000022444.md), [MongoDeleteColección](../WDLang4/1000022558.md)). 

- Realizar una búsqueda ([MongoFind](../WDLang4/1000022433.md)) o ejecutar un comando ([MongoExecuteCommand](../WDLang4/1000022435.md)). 


...

Las siguientes funciones le permiten manipular las bases de datos MongoDB: 



|   |   |
| --- | --- |
| [MongoAdd](../WDLang4/1000022432.md) | Agrega uno o más documentos a una colección de la base de datos de MongoDB. |
| [MongoCreate](../WDLang4/1000022408.md) | Crea una conexión a un servidor o cluster MongoDB. |
| [MongoCreateCollection](../WDLang4/1000022434.md) | Crea una colección en una base de datos MongoDB mientras especifica las opciones de esta colección. |
| [MongoDeleteAll](../WDLang4/1000022445.md) | Elimina todos los documentos (correspondientes al filtro especificado) de una colección de una base de datos MongoDB. |
| [MongoDeleteCollection](../WDLang4/1000022558.md) | Elimina una colección de una base de datos MongoDB. |
| [MongoDeleteDatabase](../WDLang4/1000022444.md) | Elimina una base de datos MongoDB así como sus colecciones. |
| [MongoDeleteOne](../WDLang4/1000022446.md) | Borra una Document de una colección en una base de datos de MongoDB. |
| [MongoExecuteCommand](../WDLang4/1000022435.md) | Ejecuta un comando genérico en una base de datos MongoDB o en una colección MongoDB. |
| [MongoFind](../WDLang4/1000022433.md) | Encuentra documentos en una colección de MongoDB. |
| [MongoInfo](../WDLang4/1000022441.md) | Devuelve información sobre una base de datos MongoDB. |
| [MongoModifyAll](../WDLang4/1000022603.md) | Modifica documentos en una colección de una base de datos MongoDB. |
| [MongoModifyOne](../WDLang4/1000022443.md) | Modifica una Document en una colección de la base de datos de MongoDB. |

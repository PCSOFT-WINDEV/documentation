


## &lt;Fuente&gt;.StatNbRecRange (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve una estimación del número de entradas para un elemento clave en un intervalo de valores determinado. Esta estimación se deduce de la información obtenida durante el último cálculo estadístico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about the CustName key from the Customer data file
ResStat = Customer.StatNbRecRange(CustName, "Clark", "Smith")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.StatNbRecRange([<Key item>, ] <Lower bound> , <Upper bound>)`
---

**`<Result>: (Entero)`**

Número de entradas encontradas para el artículo especificado en el interval solicitado. 
Corresponde a la constante *hInvalidRecNum* si:

- las estadísticas nunca se han calculado para la posición especificada.

- las estadísticas no están soportadas por el tipo de posición especificado. HFSQL permite realizar cálculos estadísticos sólo en posiciones clave.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre del elemento clave utilizado para calcular las estadísticas. Si no se especifica este nombre, **&lt;Fuente&gt;.StatNbRecRange** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").

**`<Lower bound>: (Tipo de campo clave)`**

El valor más bajo de interval.

**`<Upper bound>: (Tipo de campo clave)`**

Valor superior de interval.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estadísticas sobre claves compuestas
<a name="estadisticas_sobre_claves_compuestas_ELTPARAGRAPHE000235"></a>La siguiente sintaxis permite calcular estadísticas sobre una clave compuesta: 

```txt
<Data file>.StatNbRecRange(<Name of composite key>, 
	[<Lower bound of first element of composite key>, 
			<Lower bound of second element of composite key>], 
	[<Upper bound of first element of composite key>, 
			<Upper bound of second element of composite key>, ...])
```

Ejemplo:

```wl
Customer.StatNbRecRange(LastName_FirstName, ["Black", "Alicia"], ["Morgan","Vince"])
```

<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000259"></a>

- Las estadísticas pueden ser actualizadas por [&lt;Fuente&gt;.Index](../WDLang4/1000025060.md) y [&lt;Fuente&gt;.StatCalculate](../WDLang4/1000025064.md).

- En los archivos de datos de HFSQL Classic, los cálculos estadísticos sólo pueden realizarse en las partidas clave.

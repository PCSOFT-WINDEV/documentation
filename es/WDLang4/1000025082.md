


## &lt;Fuente&gt;.Version (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber:

- si el contenido de un archivo de datos fue modificado.

- si el contenido de un archivo de datos utilizado por una consulta fue modificado.




Esta función se usa para detectar las modificaciones hechas a un archivo de datos compartidas entre varios programas. Si se detecta una modificación, usted tiene la posibilidad de actualización los datos mostrados en la ventana o en la página.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
VerNum = Customer.Version()
```
<a name="Example2"></a>

```wl
// First evaluation of the data file version
VerNum is int
VerNum = Customer.Version()
// Processes
// Second evaluation of the data file version
IF VerNum <> Customer.Version() THEN
	Info("The Customer data file was modified.")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.Version()`
---

**`<Result>: (Entero)`**

Número de versión del archivo de datos.
![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) El número de versión es siempre igual a 0.

**`<Source>: (Tipo de fuente especificada)`**



- Nombre del archivo de datos utilizado.

- Nombre de la consulta utilizada.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El archivo de datos utilizado debe ser un archivo de datos HFSQL. De lo contrario, el número de versión siempre es igual a 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Incrementando el número de versión del archivo de datos
<a name="incrementando_numero_version_del_archivo_datos_ELTPARAGRAPHE000243"></a>

Cuando se crea el archivo de datos, el número de versión se inicializa a 1. Este número se incrementa cada vez que se modifica el contenido del archivo ([&lt;Fuente&gt;.Add](../WDLang4/1000024187.md), [&lt;Fuente&gt;.Write](../WDLang4/1000024331.md), [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md), [&lt;Tabla&gt;.Save](../WDLang1/1000024249.md), [.Delete](../WDLang1/1000024289.md), [&lt;Fuente&gt;.Free](../WDLang4/1000025010.md), [&lt;Fuente&gt;.Delete](../WDLang4/1000025071.md), [&lt;Fuente&gt;.Cross](../WDLang4/1000025051.md)). El número se incrementa si la función tiene éxito.

**Observación**: La función **&lt;Fuente&gt;.Version** puede utilizarse en archivos de datos indexados o no indexados.<a name="NOTE0_2"></a>


### Consultas de múltiples archivos
<a name="consultas_multiples_archivos_ELTPARAGRAPHE000298"></a>

En el caso de una consulta de varios archivos, **&lt;Fuente&gt;.Version** devuelve la suma de los **&lt;Fuente&gt;.Version** de todos los archivos de datos utilizados en la consulta.

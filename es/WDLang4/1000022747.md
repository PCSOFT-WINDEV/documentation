


## &lt;Variable Connection&gt;.PriorityClient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la prioridad de una aplicación cliente. Las aplicaciones cuya prioridad debe modificarse son las aplicaciones cliente descritas en la estructura [HClient](../WDLang4/3044340.md).

**Observación**: Cuanto mayor sea la prioridad de la aplicación cliente, más rápido se procesarán las solicitudes de esta aplicación por parte del servidor..



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Define the HClient structure
HClient.Application = "MyClientApplication"

// Change the priority of the application
MyConnection.PriorityClient(5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.PriorityClient(<Priority>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio de prioridad,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión con el servidor HFSQL.

**`<Priority>: (Integer)`**

Nuevo valor de la prioridad que se asignará a la aplicación cliente. Este valor se encuentra entre -10 (prioridad baja) y 10 (prioridad alta). 
Por defecto, la prioridad es set a 0 cuando se inicia la aplicación (prioridad estándar).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar la prioridad de la aplicación cliente especificada, se requieren los derechos para modificar la prioridad.. Este permiso corresponde a la constante *hRightsPriority* (utilizada en las funciones [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) y [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md)).




## HModifyScheduledBackup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la programación de una copia de seguridad. Esta programación fue descrita por un [hBackupDescription](../WDLang4/1000017455.md) Variable. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyDatabaseConnection is Connection

arrBck is dynamic array of hBackupDescription
arrBck = HListScheduledBackup(MyDatabaseConnection)
hBck is hBackupDescription

FOR EACH hBck OF arrBck
	IF hBck.LimitNumberBackups = 3 THEN 
		hBck.LimitNumberBackups = 5
		HModifyScheduledBackup(MyDatabaseConnection, hBck)
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyScheduledBackup(<Connection> , <Scheduled backup>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la copia de seguridad programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que debe modificarse una copia de seguridad programada. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Scheduled backup>: (Variable de tipo hBackupDescription)`**

Nombre de la [hBackupDescription](../WDLang4/1000017455.md) Variable a modificar.

<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000096"></a>

Para tener los derechos para modificar una copia de seguridad programada, debe: 

- ser el propietario de la copia de seguridad programada. 

- ser super-usuario.

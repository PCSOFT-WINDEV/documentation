


## HRSWaitForDataProcess (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Espera: 

- los datos a replicar en el servidor maestro para ser enviados al servidor de suscriptores.

- los datos a replicar recibidos en el servidor del suscriptor para ser aplicados en su totalidad.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cnt is Connection

HRSWaitForDataProcess(cnt, RplID, 10000)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Esperando a que los datos se repliquen en el servidor maestro para ser enviados al servidor del suscriptor

`<Result> = HRSWaitForDataProcess(<Connection> , <Replication> , <Timeout>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se envían los datos,

- <u><u><u><u>False</u></u></u></u> si se produce un error o si se agotó el tiempo de espera. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor maestro HFSQL. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Replication>: (Integro)`**

Identificador de réplica. Este identificador corresponde a la propiedad **Identifier** de la variable [hRSConfig](../WDLang4/1000020484.md) que describe la replicación. 

**`<Timeout>: (Integro)`**

Tiempo máximo de espera en milisegundos. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la indicación directa de la duración (por ejemplo, "10ms"). 





<a name="SYNTAX2"></a>

### En espera de que se apliquen íntegramente los datos para replicar los recibidos en el servidor del suscriptor

`<Result> = HRSWaitForDataProcess(<Connection> , <Master server> , <Replication> , <Timeout>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se aplican los datos,

- <u><u><u><u>False</u></u></u></u> si se produce un error o si se agotó el tiempo de espera. 




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor de suscriptores HFSQL. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Master server>: (Integro)`**

Identificador del servidor maestro. 

**`<Replication>: (Integro)`**

Identificador de réplica. Este identificador corresponde a la propiedad **Identifier** de la variable [hRSConfig](../WDLang4/1000020484.md) que describe la replicación. 

**`<Timeout>: (Integro)`**

Tiempo máximo de espera en milisegundos. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la indicación directa de la duración (por ejemplo, "10ms").

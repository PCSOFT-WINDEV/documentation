
## HSuggestKeyEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Termina de recoger información sobre las consultas ejecutadas para proponer claves de optimización. 

Las consultas ejecutadas a partir de la llamada a [HSuggestKeyStart](../WDLang4/1000025905.md) se analizan para proponer un set de claves que se añade a sus archivos de datos y mejorar el rendimiento de las consultas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HSuggestKeyStart(MyConnection)
// Execute queries on MyConnection 
// with HExecuteSQLQuery and HExecuteQuery
...
// Get suggestions
sSuggestedKeys is string = RTFToText(HSuggestKeyEnd(MyConnection))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HSuggestKeyEnd(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Mensaje legible para el desarrollador, que contiene la lista de claves que hay que añadir a los archivos de datos para optimizar las consultas. 

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL que contiene las consultas a analizar. Esta conexión corresponde a:

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El resultado obtenido con [HSuggestKeyEnd](../WDLang4/1000025906.md) tendrá en cuenta las 10 consultas más lentas. Se recomienda ejecutar la misma consulta varias veces con diferentes parámetros para obtener más resultados de useful.

- Para analizar una consulta con un parámetro, debe proporcionar el valor del parámetro cuando se ejecuta la consulta. 
	Si no se da un valor a los parámetros de consulta, la condición de consulta no se tendrá en cuenta. En ese caso, no se sugerirá ninguna llave.

- Las claves se sugieren de acuerdo con los datos contenidos en el archivo de datos. Si un archivo de datos no contiene datos, no se sugerirá ninguna clave.




## HListScheduledOptimization (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista las tareas de optimización programadas de los archivos de datos HFSQL Client/Server para una conexión.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
...
arrOptim is array of hScheduledOptimization
arrOptim = HListScheduledOptimization(MyConnection)
FOR EACH Optim OF arrOptim
	DisplayInfo(Optim)	
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListScheduledOptimization(<Connection>)`
---

**`<Result>: (Array de las variables de la hScheduledOptimization)`**

Array de una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). Cada elemento encontrado en esta array contiene las características de una tarea de optimización programada.. El número de elementos en esta array corresponde al número de tareas de optimización programadas disponibles.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que deben listarse las tareas de optimización programadas. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000085"></a>

Para listar las tareas de optimización programadas, debe tener los permisos de conexión (constante *hRightsConnection* o *hRightsEncryptedConnection*) para cada base de datos afectada,

Para una optimización completa del servidor, el usuario debe ser un superusuario.

Para saber si estos derechos están concedidos a un usuario o a un grupo, utilice [HInfoServerRights](../WDLang4/3044322.md) o [HInfoDatabaseRights](../WDLang4/3044320.md).

Para modificar los derechos concedidos a un usuario o a un grupo de usuarios, utilice [HModifyServerRights](../WDLang4/3044328.md) o [HModifyDatabaseRights](../WDLang4/3044326.md).

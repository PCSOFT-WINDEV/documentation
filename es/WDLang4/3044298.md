


## HDeactivateAutoFilter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva un filtro automático en los archivos de datos relacionados al navegar por un archivo XML. Este filtro se activó con la función [HActivateAutoFilter](../WDLang4/3044297.md).

**Atención**: Esta función solo está disponible para el Conector Nativo XML.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable all the possible automatic filters on the Customer data file
HActivateAutoFilter(Customer, "*")
// except between Customer and Order
HDeactivateAutoFilter(Customer, Order)

/////////////////////////////

// Browse orders
HReadFirst(Order, OrderID)
WHILE HOut() = False
	// Browses the order lines of the current order
	HReadFirst(OrdLine, OrderID)
	WHILE HOut() = False
		HReadNext(OrdLine, OrderID)
	END
	// Next order
	HReadNext(Order, OrderID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDeactivateAutoFilter(<Browse data file> , <Data file not to filter>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se desactivó el filtro,

- <u><u><u><u>False</u></u></u></u> si se produce un error (archivo de datos o elemento inexistente).




**`<Browse data file>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado para la navegación.

- "\*" para desactivar todos los filtros automáticos.




Si los parámetros &lt;Archivo de datos de navegación&gt; y &lt;Archivo de datos a no filtrar&gt; corresponden a "\*", &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u> y no se desactiva ningún filtro.

**`<Data file not to filter>: (Cadena de caracteres)`**



- Nombre del archivo de datos que no se debe filtrar cuando se establece la posición en el archivo de datos utilizado para la navegación.

- "\*" para desactivar todos los filtros automáticos.




Si los parámetros &lt;Archivo de datos de navegación&gt; y &lt;Archivo de datos a no filtrar&gt; corresponden a "\*", &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u> y no se desactiva ningún filtro.

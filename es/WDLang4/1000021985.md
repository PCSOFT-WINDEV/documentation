


## &lt;hdfsConnection Variable&gt;.OpenConnection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una conexión con un servidor Hadoop.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
IF MyConnection.OpenConnection() = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.OpenConnection()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se estableció la conexión,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable correspondiente a la conexión a establecer con el servidor Hadoop.

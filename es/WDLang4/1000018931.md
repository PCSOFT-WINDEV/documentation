


## HModifyScheduledOptimization (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una tarea de optimización programada en el servidor HFSQL definido por la conexión. Esta tarea se encuentra en un [hScheduledOptimization](../WDLang4/1000018854.md) avanzado Variable.



<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HModifyScheduledOptimization(<Connection> , <Scheduled optimization task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la tarea de optimización programada,

- <u><u><u><u>False</u></u></u></u> si se produce un error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la que se debe modificar una tarea de optimización programada. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Scheduled optimization task>: (Variable de tipo hScheduledOptimization)`**

Nombre de la [hScheduledOptimization](../WDLang4/1000018854.md) Variable utilizada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000084"></a>

Para modificar una tarea, debe tener: 

- los permisos correspondientes (constante *hRightsManageTask*) en el servidor,

- los derechos para ejecutar el comando relacionado en la tarea programada.


La tarea se ejecutará con la cuenta del usuario que la creó.. 

Observación: La tarea programada no se realizará si el servidor está apagado.. La tarea programada se realizará en la siguiente fecha disponible cuando se encienda el servidor.

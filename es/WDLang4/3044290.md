


## HListUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de usuarios definidos para una conexión.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
Result1 is string
Result1 = HListUser("MyConnection")
// Add the users into a List Box control 
ListAdd(LIST_MyList, Result1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListUser(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de los usuarios existentes para la conexión especificada. Los nombres de usuarios están separados por retornos de carro (CR).

**Observación**: Si el usuario actual no tiene permisos para ver usuarios (constante *hRightsSeeUser*) en el servidor (función [HModifyServerRights](../WDLang4/3044328.md) o [HInfoServerRights](../WDLang4/3044322.md)), la función **HListUser** solo devuelve el usuario actual. 

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Connection](../WDLang4/1514073.md).


## hbReading (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **hbReading** permite definir las características de una operación de lectura. Las características de esta operación de lectura pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Connection is hbConnection
Connection.Server = "MyServer"
// Description of data to read
Reading is hbReading
Reading.Row = "post1"
Reading.Version = 2
// Column description
Col is hbColumn
Col.Family = "Post"
Add(Reading.Column, Col)
// Read data
MyResult is array of hbCells = hbRead(Connection, "blog", Reading)
// Display the data read
FOR EACH Cell OF MyResult DO
	Trace("Column: " + Cell.Column.Qualifier + "; Value: " + ...
	Cell.Value + "; timestamp: " + Cell.Timestamp)
END
```

## Observaciones

### Propiedades específicas de las variables hbReading
<a name="propiedades_especificas_las_variables_hbreading_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden utilizarse para describir la operación de lectura a realizar:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Columna | Array de [hbColumn](../WDLang4/1000021698.md) | description de columnas para leer. Si la array está vacía, se lee toda la fila. |
| Fila | Buffer | Identificador de la fila (Record). |
| Fin del sello de fecha y hora | FechaHora | Fecha máxima de las versiones a leer (opcional). Se leerán los datos con una fecha anterior o igual a esta fecha. |
| InicioSello de fecha y hora | FechaHora | Fecha mínima de las versiones a leer (opcional). Si se especifica este Property, también debe especificarse el **EndTimestamp** Property. Se leerán los datos con una fecha posterior o igual a esta fecha. |
| variante | Integro | Número mínimo de versiones a leer (opcional). |

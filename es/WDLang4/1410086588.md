


## Funciones HBase
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Para la gestión de las bases de datos de HBase se utilizan las siguientes funciones: 




|   |   |
| --- | --- |
| [hbCreateTable](../WDLang4/1000021696.md) | Crea una HBase tabla o modifica la descripción de una HBase tabla. |
| [hbDelete](../WDLang4/1000021714.md) | Elimina:<br><br>- una o varias celdas de una HBase tabla para una o varias columnas.<br><br>- a fila de una HBase tabla.<br><br><br> |
| [hbDeleteTable](../WDLang4/1000021713.md) | Borra una HBase tabla. |
| [hbGetTableDescription](../WDLang4/1000021695.md) | Recupera la description de una HBase tabla. |
| [hbListTable](../WDLang4/1000021687.md) | Recupera la lista de tablas encontradas en un servidor HBase. |
| [hbRead](../WDLang4/1000021706.md) | Lee las celdas (valores de los elementos) de un fila (registro) en una HBase tabla. |
| [hbWrite](../WDLang4/1000021712.md) | Escribe celdas (valores de elementos) en una o varias filas (registros) de una HBase tabla. |





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>
Las funciones HBase utilizan los tipos siguientes: 



|   |   |
| --- | --- |
| [hbCell](../WDLang4/1000021707.md) | El tipo **hbCell** se utiliza para definir las características avanzadas del contenido de una celda (valor del elemento) en una HBase tabla. |
| [hbColumn](../WDLang4/1000021698.md) | El tipo **hbColumn** se utiliza para definir las características avanzadas de una columna (elemento) en una HBase tabla. |
| [hbColumnDescription](../WDLang4/1000021694.md) | El tipo **hbColumnDescription** permite definir las características avanzadas de una familia de columnas (elementos). |
| [hbConnection](../WDLang4/1000021685.md) | El tipo **hbConnection** permite definir las características de una conexión a un servidor HBase. |
| [hbReading](../WDLang4/1000021701.md) | El tipo **hbReading** permite definir las características de una operación de lectura. |
| [hbResultScan](../WDLang4/1000021724.md) | El tipo **hbScanResult** se utiliza para definir todas las características avanzadas del resultado de una iteración realizada en una base de datos HBase. |
| [hbScanParameter](../WDLang4/1000021715.md) | El tipo **hbScanParameter** se utiliza para definir los parámetros para iterar sobre los registros de una base de datos HBase. |
| [hbTableDescription](../WDLang4/1000021688.md) | El tipo **hbTableDescription** se utiliza para definir y conocer las características de una HBase tabla. |
| [hbWriting](../WDLang4/1000021710.md) | El tipo **hbWriting** se utiliza para definir las características de una operación de escritura que se realizará en un fila (registro) de una HBase tabla. |

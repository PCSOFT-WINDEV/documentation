


## &lt;Fuente&gt;.SetTransaction (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la gestión de transacciones para uno o más archivos de datos.

Cuando se desactiva la gestión de las transacciones, se inhiben todas las funciones relacionadas con las transacciones ([&lt;variable Connection&gt;.TransactionStart](../WDLang4/1000023967.md), [&lt;Variable Connection&gt;.TransactionEnd](../WDLang4/1000023968.md), [&lt;Variable Connection&gt;.TransactionCancel](../WDLang4/1000023966.md)).

**Observación**: La gestión de las transacciones está habilitada por Default.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable the management of transactions on the Customer data file
Customer.SetTransaction(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.SetTransaction(<Options>)`
---

**`<Result>: (Booleano)`**

Corresponde a la anterior gestión de transacciones (antes de utilizar la función **&lt;Fuente&gt;.SetTransaction**): 

- <u><u><u><u>True STRIMEULB</u></u></u></u> si la gestión de las transacciones está habilitada,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si se ha desactivado la gestión de operaciones. 




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Options>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la gestión de las transacciones debe ser habilitada para el archivo de datos, 

- <u><u><u><u>False STRIMEULB</u></u></u></u> si esta gestión debe ser desactivada.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de transacciones habilitada
<a name="gestion_transacciones_habilitada_ELTPARAGRAPHE000257"></a>

Si se habilita la gestión de transacciones para un solo fichero, sólo se tendrán en cuenta en la gestión de transacciones las operaciones realizadas en este fichero de datos (independientemente de los ficheros de datos especificados en [&lt;variable Connection&gt;.TransactionStart](../WDLang4/1000023967.md)).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Límite 
<a name="limite_ELTPARAGRAPHE000274"></a>

La gestión de las transacciones no se aplica a los archivos xBase.

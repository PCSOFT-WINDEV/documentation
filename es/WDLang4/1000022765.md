


## &lt;Variable Connection&gt;.DeleteBackup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una copia de seguridad creada previamente con [&lt;Variable Connection&gt;.Backup](../WDLang4/1000022752.md). La copia de seguridad se elimina físicamente del disco.



<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteBackup(<Backup to delete>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión de la que hay que borrar la copia de seguridad.

**`<Backup to delete>: (Integer)`**

Identificador de la copia de seguridad a eliminar. Este identificador es devuelto por [&lt;Variable Connection&gt;.Backup](../WDLang4/1000022752.md) o [&lt;Variable Connection&gt;.InfoBackup](../WDLang4/1000022679.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la copia de seguridad se está creando cuando se llama a **&lt;Variable Connection&gt;.DeleteBackup**, se cancela y se borra.

- Para eliminar una copia de seguridad, el usuario debe ser superusuario o debe tener los permisos para realizar copias de seguridad (constante *hRightsBackup*).

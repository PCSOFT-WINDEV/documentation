
## mongoResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le tipo **mongoResult** permite conocer el resultado de una búsqueda en una base de datos de tipo MongoDB. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFilter is string
sFilter = "{ ""name"": ""France"" }"

let MySearch = MongoFind(CollecMDB, sFilter)

// Read the result
FOR EACH item of MySearch
	Trace(VariantToJson(item))
END
```





<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000040"></a>

- El tipo **mongoResult** se puede navegar con la instrucción FOR EACH. 

- Cada elemento navegado es un **Variante**. 

- Cada elemento representa una Document de MongoDB.

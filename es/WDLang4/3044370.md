
## LINUX: Acceso a las bases de datos ODBC
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
El acceso nativo a HFSQL Classic y Cliente/Servidor está disponible para las aplicaciones WINDEV que se ejecutan en Linux (aplicaciones "de consola") y para los sitios WEBDEV alojados en Linux. 

El acceso nativo a MySQL también está disponible en Linux.

El acceso a todas las bases de datos equipadas con un controlador ODBC de Linux también está disponible.: Oracle, DB2, Postgresql, Interbase, Ingres, SQL Server, Informix, ....



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000119"></a>


### Manejo de bases de datos mediante un controlador ODBC de Linux
<a name="manejo_bases_datos_mediante_controlador_odbc_linux_ELTPARAGRAPHE000024"></a>

Para manipular las bases de datos a través de un controlador ODBC Linux: 

1. Compruebe si la capa estándar para la gestión ODBC no está instalada en el ordenador Linux.. Si este no es el caso, instale esta capa para la administración de ODBC.

2. Instale (si es necesario) el controlador ODBC correspondiente a la base de datos a utilizar.

3. En su aplicación o sitio Linux, manipular las bases de datos a través del programa [Funciones SQL de WLanguage](../WDLang4/3072002.md).



<a name="NOTE2_2"></a>


### Instalando la capa ODBC en Debian
<a name="instalando_capa_odbc_debian_ELTPARAGRAPHE000038"></a>

Para instalar la capa ODBC en Debian, utilice el siguiente código:


```txt
apt-get install unixodbc
```

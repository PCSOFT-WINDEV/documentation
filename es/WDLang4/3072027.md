


## SQLListSource( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las fuentes de datos declaradas en un sitio local. No es necesario estar conectado a la base de datos para realizar esta operación.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the accessible sources in a List Box control
ListAdd(LIST_DataList, SQLListSource())

// Display the accessible databases in a List Box control 
ListAdd(LIST_DataList, SQLListSource("DATABASE"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Enumeración de las fuentes de datos disponibles para todas las bases de datos

`<Result> = SQLListSource()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de todas las fuentes de datos disponibles. Los diferentes elementos están separados por caracteres CR y pueden añadirse directamente en una lista ([ListAdd](../WDLang1/3049004.md)).
![ODBC](https://doc.pcsoft.fr/ext/images/us/ODBC.png) Lista de todas las fuentes de datos definidas en el administrador ODBC del ordenador actual.


<a name="SYNTAX2"></a>

### Enumeración de las bases de datos disponibles

`<Result> = SQLListSource(<"DATABASE">)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de todos los tipos de bases de datos disponibles en el ordenador actual. Los diferentes elementos están separados por caracteres CR y pueden añadirse directamente en una lista ([ListAdd](../WDLang1/3049004.md)). Los posibles tipos de bases de datos son:


|   |   |
| --- | --- |
| ACCEDER | El proveedor de Microsoft Jet 4 y MDAC están instalados en el ordenador actual. |
| DB2 | El conector nativo DB2 se instala en el ordenador actual. |
| HYPER FICHERO | Acceso a bases de datos HFSQL. |
| HIVE | El Native Hive Connector se instala en el ordenador actual. |
| INFORMIX | El Native INFORMIX Connector se instala en el ordenador actual. |
| MariaDB | El Native MariaDB Connector se instala en el ordenador actual. |
| MYSQL | El Native MYSQL Connector se instala en el ordenador actual. |
| ODBC | La DLL ODBC32 se instala en el ordenador actual. |
| OLEDB | MDAC se instala en el ordenador actual. |
| ORACLE | El conector ORACLE nativo se instala en el ordenador actual. |
| POSTGRESQL | El conector POSTGRESQL nativo se instala en el ordenador actual. |
| PROGRESO | El Native PROGRESS Connector se instala en el ordenador actual. |
| SQL AZURE | El Native SQL AZURE Connector se instala en el ordenador actual. |
| SQL SERVER | El Native SQL SERVER Connector se instala en el ordenador actual..<br>**Observación**: Se tiene en cuenta el valor de la [H.SQLServerMode](../WDLang4/3044276.md) Variable. Si la antigua Conector Nativo es forzada y si no está en el equipo, el Conector Nativo SQL Server no aparecerá. Si la nueva Conector Nativo es forzada y si no está en la el equipo, la Conector Nativo SQL Server no aparecerá en la lista. |
| SYBASE | El conector SYBASE nativo se instala en el ordenador actual. |



**`<"DATABASE">: (Cadena de caracteres)`**

"BASE DE DATOS": Cadena de caracteres utilizada para listar las bases de datos disponibles en el ordenador actual.  


<a name="SYNTAX3"></a>

### Enumeración de la fuente de datos disponible para un tipo específico

`<Result> = SQLListSource(<"DATABASE"> , <Type of database>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de todas las fuentes de datos disponibles para el tipo especificado. Los diferentes elementos están separados por caracteres CR y pueden añadirse directamente en una lista ([ListAdd](../WDLang1/3049004.md)). Los resultados posibles según el tipo de base de datos son los siguientes:


|   |   |
| --- | --- |
| Acceso | Cuerda vacía. |
| HYPER FICHERO | Cadena vacía. Le recomendamos que utilice [HListAnalysis](../WDLang4/3044049.md) para get la lista de análisis. |
| ODBC | Lista de bases de datos ODBC definidas en el administrador ODBC del ordenador actual. |
| OLEDB | Cuerda vacía. |



**`<"DATABASE">: (Cadena de caracteres)`**

"BASE DE DATOS": Cadena de caracteres utilizada para listar las fuentes de datos disponibles para un tipo específico.

**`<Type of database>: (Cadena de caracteres)`**

Tipo de bases de datos buscadas: "ODBC", ...

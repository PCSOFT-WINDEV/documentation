


## HFSQL Client/Server: Configuración de los archivos de datos en el servidor
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000150"></a>
**Si modifica una aplicación WINDEV existente en una aplicación HFSQL Client/Server, se pueden utilizar dos métodos para convertir los archivos HFSQL Classic en archivos HFSQL Client/Server: desde el editor de análisis o desde el HFSQL control Center.
Para más detalles sobre cómo convertir los archivos HFSQL Classic en Cliente/Servidor, véase [Convertir una base de datos HFSQL Classic en HFSQL Client/Server](../WDLang4/9000155.md). 

**Para crear una aplicación HFSQL Client/Server**, no se requiere ninguna operación específica. Los archivos de datos se crearán en el servidor durante la llamada a [HCreation](../WDLang4/3044255.md) o [HCreationIfNotFound](../WDLang4/3044254.md).

### Nombres en un servidor Linux
<a name="nombres_servidor_linux_ELTPARAGRAPHE000033"></a>

En un servidor Linux, los nombres de los archivos de datos deben escribirse en minúsculas y sin caracteres acentuados.

Por lo tanto, el archivo llamado "Tasks.fic" debe ser renombrado a "tasks.fic".. [HCopyFile](../WDLang4/3044282.md) se utiliza para copy los archivos de datos teniendo en cuenta las características específicas relativas al caso de los nombres de archivo en Linux.
<a name="NOTE2_2"></a>


### Importante
<a name="importante_ELTPARAGRAPHE000045"></a>

En HFSQL Client/Server:

- [HSubstDir](../WDLang4/3044028.md) no se tiene en cuenta. De hecho, esta función sólo se puede utilizar para sustituir los directorios de los archivos de la red HFSQL.

- Si se utiliza [HChangeDir](../WDLang4/3044168.md), debe especificar un ruta relativo al directorio de la base de datos de la aplicación en el servidor. Este modo de funcionamiento le permite ser independiente de la ubicación del servidor utilizado.

- Si se especifica un directorio ruta en el análisis description, en el description de los archivos de datos encontrados en el análisis o en el proyecto description, este ruta debe ser relativo al directorio de la base de datos de la aplicación en el servidor.

- Los archivos de datos a los que se accede por un servidor HFSQL son accesibles a través de este servidor solamente. No se puede acceder a ellos directamente por otra aplicación, especialmente a través del motor HFSQL Classic. Se requiere una conexión con el servidor para acceder a estos archivos de datos.



<a name="NOTE2_3"></a>


### Gestión de las limitaciones de integridad
<a name="gestion_las_limitaciones_integridad_ELTPARAGRAPHE000062"></a>

Si una aplicación HFSQL ISAM fue modificada en una aplicación HFSQL Client/Server creando los archivos de datos en el servidor, las restricciones de integridad son soportadas.

Si una aplicación HFSQL ISAM fue modificada en una aplicación HFSQL Client/Server sin crear los archivos de datos HFSQL en el servidor (por simple copy), las restricciones de integridad no son administradas.

Se pueden utilizar varios métodos para**gestionar las restricciones de integridad en su aplicación HFSQL**:

- Utilice la función [HAddLink](../WDLang4/3044312.md). 

- Realizar la modificación de los archivos de datos de automatic al instalar la aplicación. La modificación de automatic de los archivos de datos se utiliza para crear las restricciones de integridad en el servidor.

## Importación de las estructuras
<a name="importacion_las_estructuras_ELTTEXTE000216"></a>
Las estructuras de los archivos encontrados en un servidor HFSQL existente pueden ser importadas a un análisis.: en la pestaña "Análisis", en el grupo "Creación", amplíe "Importar" y seleccione "Importar archivo/descripciones tabla...".

<a name="NOTE4_1"></a>


## Limitaciones
<a name="limitaciones_ELTTEXTE000240"></a>
El modo de gestión de los archivos de datos puede tener que adaptarse cuando se utiliza una aplicación de HFSQL Client/Server. Para saber más sobre las características específicas que hay que tener en cuenta, consulte [Diferencias y limitaciones específicas de una aplicación de HFSQL Client/Server](../WDLang4/3044302.md).

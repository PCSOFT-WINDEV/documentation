


## &lt;Fuente&gt;.Save (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el registro actual si existe, si no existe, agrega un nuevo registro.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Customer.ToFile()
Customer.Save()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Source>.Save([<Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición o modificación,

- <u><u><u><u>False</u></u></u></u> si se produce un error.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos, vista o consulta utilizada. 

**`<Options>: (Constante opcional de tipo Integer)`**

Permite configurar:

- el efecto de la adición o modificación de registros en la navegación actual.

- el modo de cálculo del identificador automático. Para obtener más información, consulte [Gestión del identificador automático al agregar un registro](../WDLang4/3044183.md).

- el modo de bloqueo del registro agregado o modificado.

- la gestión de duplicados.

- la gestión de la integridad.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la propiedad [NewRecord](../Proprietes/1000017090.md) se establece en <u><u><u><u>True</u></u></u></u>, la función **&lt;Fuente&gt;.Save** agrega el registro al archivo de datos (función [&lt;Fuente&gt;.Add](../WDLang4/1000024187.md)). Para obtener más información, consulte la documentación de [&lt;Fuente&gt;.Add](../WDLang4/1000024187.md). 

- Si la propiedad [NewRecord](../Proprietes/1000017090.md) se establece en <u><u><u><u>False</u></u></u></u>, la función **&lt;Fuente&gt;.Save** modifica el registro en el archivo de datos (función [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md)). Para obtener más información, consulte la documentación de [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md).

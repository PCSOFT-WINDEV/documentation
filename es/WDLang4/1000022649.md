


## &lt;Variable Connection&gt;.LoadParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee un parámetro que fue guardado de un Procedure almacenado por [&lt;Variable Connection&gt;.SaveParameter](../WDLang4/1000022753.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyParameter is Connection

MyParameter.LoadParameter()
```

<a name="XSYNTAX"></a>

### Cargar un parámetro en el servidor (desde el equipo Cliente)

`<Result> = <Connection>.LoadParameter(<Parameter> , <Default value>)`
---

**`<Result>: (Variant)`**

Valor del parámetro si existe, &lt;Valor de Default&gt; si el parámetro no existe.

**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<Parameter>: (Cadena de caracteres)`**

Nombre del parámetro a restaurar. Este parámetro no debe corresponder a una cadena vacía y no debe contener el carácter "CR" (Carriage return).

**`<Default value>: (Variant)`**

Valor de Default devuelto si no se encuentra el parámetro. Si no se especifica este parámetro, el valor de Default corresponde a una cadena vacía ("").

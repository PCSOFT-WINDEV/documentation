


## &lt;hdfsConnection Variable&gt;.CrearDirectorio (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un directorio en un servidor Hadoop.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection is hdfsConnection
MyConnection.namenode = "MyServer"
MyConnection.User = "Admin"
MyConnection.OpenConnection()
// Define the rights
Rights is hdfsRights
Rights.UserReading = True
Rights.UserWriting = True
// Create the directory
MyConnection.CreateDirectory("/tmp/subdirectory", Rights)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.CreateDirectory(<Path> [, <Rights>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el directorio,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Connection>: (hdfsConnection Variable)`**

Nombre del [hdfsConnection](../WDLang4/1000021650.md) Variable que corresponde a la conexión con el servidor Hadoop.

**`<Path>: (Cadena de caracteres)`**

ruta absoluta del directorio que se creará en el servidor de Hadoop. Esta ruta distingue entre mayúsculas y minúsculas.

**`<Rights>: (Opcional hdfsRights Variable)`**

Nombre de [hdfsRights](../WDLang4/1000021662.md) Variable correspondiente a los derechos que se concederán al directorio creado.
Por Default, todos los derechos son concedidos a todos los usuarios.

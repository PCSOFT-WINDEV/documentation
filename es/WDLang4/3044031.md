


## HStatNbDuplicados (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de duplicados de un elemento clave dado. Este número de duplicados se calculó durante el último cálculo de las estadísticas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about duplicates on the CustName key of the Customer data file
ResStat = HStatNbDuplicates(Customer, CustName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HStatNbDuplicates([<Data file> [, <Key item>]])`
---

**`<Result>: (Integro)`**

Número de duplicados encontrados. Corresponde a la constante *hInvalidRecNum* si:

- las estadísticas nunca se han calculado para la posición especificada.

- las estadísticas no están soportadas por el tipo de posición especificado. HFSQL permite realizar cálculos estadísticos sólo en posiciones clave.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre, la función **HStatNbDuplicados** utilizará el último archivo de datos utilizado por la última función HFSQL (que empieza por "H").

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre del elemento clave utilizado para calcular las estadísticas. Si no se especifica este nombre, **HStatNbDuplicados** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las estadísticas también pueden ser actualizadas por [HIndex](../WDLang4/3044133.md). [HIndex](../WDLang4/3044133.md) y [HStatCalculate](../WDLang4/3044134.md) son las dos únicas funciones que actualizan la estadística del archivo de datos.

- En los archivos de datos de HFSQL Classic, los cálculos estadísticos sólo pueden realizarse en las partidas clave.

- El número de duplicados corresponde al número total de entradas menos el número de valores distintos.. Por ejemplo, si las entradas son los números: 1, 3, 3, 5, 5, 5, 6, 7, 7, 7, 8, 9, entonces el número de duplicados es 12-7 = 5.

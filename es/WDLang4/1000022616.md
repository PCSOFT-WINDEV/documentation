Las siguientes constantes son utilizadas por las funciones Mongoxxx:

| Constante | Detalles | Función |
| --- | --- | --- |
| **mwcPor defecto** | Número mínimo de nodos que deben haber reconocido la Process para considerar que la escritura se realiza: un solo nodo debe haber reconocido a la Process para considerar que la escritura se realiza. | Tipo [mongoWriteConcern](../WDLang4/1000022597.md) |
| **mwcMajority** | Número mínimo de nodos que deben haber reconocido la Process para considerar que la escritura se realiza: la mayoría de los nodos deben haber reconocido a la Process para considerar que la escritura se realiza. | Tipo [mongoWriteConcern](../WDLang4/1000022597.md) |
| **mwcUnacknowledged** | Número mínimo de nodos que deben haber reconocido la Process para considerar que la escritura se realiza: Ningún nodo debe haber reconocido a la Process para considerar que la escritura se realiza. Atención, no se devuelve ningún error en este caso. | Tipo [mongoWriteConcern](../WDLang4/1000022597.md) |
| **mwcTag** | Número mínimo de nodos que deben haber reconocido la Process para considerar que la escritura se realiza: el número de nodos es el definido en una preconfiguración definida en el servidor. | Tipo [mongoWriteConcern](../WDLang4/1000022597.md) |
| **mongoAddIfNotFound** | Se utiliza para añadir un Document si no existe en la colección. | [MongoModifyAll](../WDLang4/1000022603.md)<br>[MongoModifyOne](../WDLang4/1000022443.md) |
| **mongoContinueIfError** | Se usa para añadir el siguiente Document incluso si el Document anterior provocó un error. | [MongoAdd](../WDLang4/1000022432.md) |
| **mongoReturnValueAfter** | Se utiliza para devolver el Document después de la modificación. Por defecto, si no se utiliza esta constante, el Document devuelto corresponde al Document antes de la modificación. | [MongoModifyOne](../WDLang4/1000022443.md) |
| **mongoWithoutValidation** | No comprueba la estructura de Document. | [MongoAdd](../WDLang4/1000022432.md)<br>[MongoModifyAll](../WDLang4/1000022603.md) |

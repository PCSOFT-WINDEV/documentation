


## &lt;Fuente&gt;.ActivateFilter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa el filtro que se creó previamente para el archivo de datos (vista o consulta) especificado. Este filtro se creó con la función [&lt;Fuente&gt;.Filter](../WDLang4/1000024960.md) y desactivó con la función [&lt;Fuente&gt;.DeactivateFilter](../WDLang4/1000024218.md). Si no se declaró ningún filtro, la función **&lt;Fuente&gt;.ActivateFilter** no tiene ningún efecto.

Esta función se puede utilizar con archivos de datos, vistas HFSQL o consultas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare the filter: select customers
// between 20 and 60 years old
Customer.Filter(Age_Customer, 20, 60)
...
Customer.DeactivateFilter() // Disabled filter
...
// Browse the data file with a filter on age
// Process the customers who are between 20 and 60 years old
// (ProcessCustomer2060 procedure)
Customer.ActivateFilter()
Customer.ReadFirst(Age_Customer)
WHILE Customer.Found() = True
	ProcessCustomer2060()
	Customer.ReadNext(Age_Customer)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.ActivateFilter()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se activó el filtro,

- <u><u><u><u>False</u></u></u></u> si se produce un error (archivo o elemento inexistente).




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos, vista HFSQL o consulta utilizada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Activar y desactivar un filtro
<a name="activar_desactivar_filtro_ELTPARAGRAPHE000236"></a>

El filtro se activa una vez que ha sido declarado (función [&lt;Fuente&gt;.Filter](../WDLang4/1000024960.md)). El filtro se puede desactivar con la función [&lt;Fuente&gt;.DeactivateFilter](../WDLang4/1000024218.md).
<a name="NOTE0_2"></a>


### Número de filtros activos posibles
<a name="numero_filtros_activos_posibles_ELTPARAGRAPHE000256"></a>

Solo se puede activar un filtro a la vez para una archivo de datos específico.
<a name="NOTE0_3"></a>

### Modo de funcionamiento en OLE DB
<a name="modo_funcionamiento_ole_ELTPARAGRAPHE000268"></a>

Si se utiliza la función **&lt;Fuente&gt;.ActivateFilter** para navegar por un archivo de datos, la navegación se detiene. Para tener en cuenta el filtro, se debe reiniciar la navegación con las funciones [&lt;Fuente&gt;.ReadFirst](../WDLang4/1000025022.md)/[&lt;Fuente&gt;.First](../WDLang4/1000025048.md) o [&lt;Fuente&gt;.ReadLast](../WDLang4/1000025020.md)/[&lt;Fuente&gt;.Last](../WDLang4/1000024217.md).

La posición en curso y todas las posiciones guardadas se pierden. La posición se restablecerá de acuerdo con el valor del mejor elemento.

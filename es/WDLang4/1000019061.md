
## HClusterAddNode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita un nodo en un cluster HFSQL.

Notas: 

- La función **HClusterAddNode** se llama automáticamente durante la [instalación de un nuevo nodo](../WDLang4/1000019031.md). Esta función está reservada para un uso avanzado.

- La función **HClusterAddNode** no instala el servidor HFSQL y no añade registros DNS para permitir el acceso al nuevo nodo.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add the HFSQL server installed on host3.mycompany.com into the
// cluster managed by clustermanager.mycompany.com
HClusterAddNode("clustermanager.mycompany.com", "secret", "host3.mycompany.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HClusterAddNode(<Coordinator> , <Password> , <New node>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Coordinator>: (Cadena de caracteres)`**

Nombre DNS del ordenador donde está instalado el coordinador del clúster.

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para conectarse con el coordinador. Esta contraseña se dio cuando [instalación del cluster](../WDLang4/1000019031.md).

**`<New node>: (Cadena de caracteres)`**

Nombre del nodo a añadir. Este parámetro corresponde al nombre DNS de el equipo en el que se instaló el nuevo servidor HFSQL.






## Observaciones


### Modo de funcionamiento de los nodos de un cluster
<a name="modo_funcionamiento_los_nodos_cluster_ELTPARAGRAPHE000065"></a>

La Address del coordinador del clúster se escribe en el archivo de configuración de cada uno de los nodos cuando se instalan. Por lo tanto, al iniciar el servicio HFSQL Server, el nodo se conecta automáticamente al clúster.

Las funciones [HClusterAddNode](../WDLang4/1000019061.md) y [HClusterDeleteNode](../WDLang4/1000019062.md) pueden utilizarse para incluir o excluir un nodo de la lista de nodos activos del clúster.

Cuando un nodo es habilitado por [HClusterAddNode](../WDLang4/1000019061.md), este nodo sincroniza sus bases de datos con las de los otros nodos y procesa las solicitudes de conexión de los clientes.

Cuando un nodo es desactivado por [HClusterDeleteNode](../WDLang4/1000019062.md), este nodo deja de sincronizar sus bases de datos y ya no acepta ninguna conexión de las aplicaciones cliente.

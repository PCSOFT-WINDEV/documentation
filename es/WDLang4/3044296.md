


## HDeleteUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un usuario asociado a una conexión en el servidor HFSQL.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", ...
		"MyDatabase", hAccessHFClientServer, HORead)
// Open the connection
HOpenConnection("MyConnection")
// Associate the data file with the connection
HChangeConnection(Customer, "MyConnection")
...
IF HDeleteUser("MyConnection", "Flo") = True THEN
	Info("User deleted: Flo")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDeleteUser(<Connection> , <User to delete>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el usuario,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**Observación**: [HErrorInfo](../WDLang4/3044071.md) devolverá un error "Usuario desconocido" si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser*).

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión para la cual un usuario debe ser eliminado. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<User to delete>: (Cadena de caracteres)`**

Nombre del usuario a eliminar. Este nombre debe corresponder al **HUser.Login** Variable para uno de los usuarios de la conexión especificada.

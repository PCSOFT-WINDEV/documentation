


## &lt;Variable Connection&gt;.RefreshView (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vuelve a calcular el contenido de una vista materializada. 

**Observación**: También tiene la posibilidad de actualización la vista mediante la sentencia SQL ["REFRESH MATERIALIZED VIEW"](../Editeurs/2034001.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

MyConnection.RefreshView(MyView)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.RefreshView(<View> [, <Option>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se actualizó la vista materializada, 

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la variable de tipo [Conexión](../WDLang4/1514073.md) que describe la conexión a utilizar.

**`<View>: (Cadena de caracteres)`**

Nombre de la vista materializada cuyos datos deben recalcularse.. Esta vista fue creada: 

- a través de la editor de análisis. 

- mediante el comando SQL ["CREATE MATERIALIZED VIEW"](../Editeurs/2034001.md).




**`<Option>: (Constante opcional de tipo Integer)`**

Modo para refrescar la vista materializada. 


|   |   |
| --- | --- |
| *hNoWait* | La actualización de vista materializada no es una operación de bloqueo: se realiza en la tarea de Background. [&lt;Variable Connection&gt;.InfoView](../WDLang4/1000022688.md) devuelve la información de la vista materializada después de ser refrescada. |
| Parámetro no especificado (por Default) | El actualización de vista materializada se bloquea para quien lo solicitó.. &lt;Resultado&gt; permite conocer el informe de la actualización. |

<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000244"></a>

Para recalcular el contenido de la vista materializada, debe tener: 

- los permisos correspondientes (constante *hRightsManageTask*) en el servidor,

- para cada base de datos correspondiente (es decir, que contenga una vista para procesar), los permisos de conexión (constante *hRightsConnection* o *hRightsEncryptedConnection*). 




Para saber si estos derechos están concedidos a un usuario o a un grupo, utilice [&lt;Variable Connection&gt;.InfoServerRights](../WDLang4/1000022668.md) o [&lt;Variable Connection&gt;.InfoDatabaseRights](../WDLang4/1000022666.md).

Para modificar los derechos concedidos a un usuario o a un grupo de usuarios, utilice [&lt;Variable Connection&gt;.ModifyServerRights](../WDLang4/1000022710.md) o [&lt;Variable Connection&gt;.ModifyDatabaseRights](../WDLang4/1000022707.md).

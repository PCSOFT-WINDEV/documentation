


## &lt;Variable mongoGridFS&gt;.GetFileById (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera (en la computadora local) un archivo identificado por su identificador y encontrado en un sistema de archivos MongoDB.

**Observación**: Para recuperar un archivo identificado a partir de su nombre, utilice [&lt;mongoGridFS Variable&gt;.GetFile](../WDLang4/1000025391.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gclGridFS is mongoGridFS
FOR EACH file of MongoGridFSListFile(gclGridFS)
	gclGridFS.GetFileByID(file.ID, fDataDir() + ["\"] + file.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <File system>.GetFileByID(<Identifier of file to retrieve> , <Name of local file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<File system>: (mongoGridFS Variable)`**

Nombre del [mongoGridFS](../WDLang4/1000022416.md) Variable que corresponde al sistema de archivos Mongo a utilizar.

**`<Identifier of file to retrieve>: (Variant)`**

Identificador del archivo en el sistema de archivos de MongoDB que se recuperará en el ordenador local.

**`<Name of local file>: (Cadena de caracteres)`**

Nombre y ruta del archivo que se recuperará en el ordenador local.

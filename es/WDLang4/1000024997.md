


## &lt;Fuente&gt;.InfoMemo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las características de un memo binario y de texto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Product.ReadFirst(Name)
MemoInfo is string 
MemoInfo = Product.InfoMemo(Name)
// Retrieve the name of source file of memo
FileName is string 
FileName = MemoInfo.ExtractString(2, TAB)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Conocer las características de un elemento Memo

`<Result> = <Source>.MemoInfo(<Memo item>)`
---

**`<Result>: (Cadena de caracteres)`**



- Información sobre el memo binario. 

- Vaciar la cadena (""") si la nota está vacía.




La información está separada por caracteres TAB. Esta información es la siguiente: 

```txt
<Type>+TAB+<Name>+TAB+<Size>+TAB+<Date>+TAB+<Time>+TAB+<Info>
```



|   |   |
| --- | --- |
| escribir a máquina | Tipo de memo. Este parámetro puede tomar los siguientes valores: "TXT" para las notas de texto, "IMG", "WAV", "OLE", "BIN" o "BAD" ("MAL" indica que la nota binaria no está almacenada correctamente). |
| denominar | Nombre del archivo fuente tal y como se especificó en [&lt;Fuente&gt;.LinkMemo](../WDLang4/1000024194.md). |
| envergadura | Tamaño del archivo fuente en bytes. Para una nota de texto, este tamaño se expresa en número de caracteres. |
| salir con | Fecha del archivo fuente en formato "AAAAMMDD". |
| momento | Hora del archivo fuente en formato "HHMMSS |
| Info | Información del usuario tal y como se especificó en [&lt;Fuente&gt;.LinkMemo](../WDLang4/1000024194.md) |


**Si el memo es un memo de texto**el nombre, la fecha, la hora y la información del usuario no están rellenados. El tipo es set a "TXT". 
Ejemplo de valor devuelto por una nota de texto cuyo tamaño es 25: "TXT"+TAB+TAB+25+TAB+TAB+TAB.

**Si la nota es una nota binaria** que no ha sido asignada por [&lt;Fuente&gt;.LinkMemo](../WDLang4/1000024194.md), el nombre, la fecha, la hora y la información no se especifican. El tipo es set a "BIN". 
Ejemplo de valor devuelto por una nota binaria cuyo tamaño es 1202: "BIN"+TAB+TAB+1202+TAB+TAB+TAB.
![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) La información sobre el memo no se guardará. La única información devuelta por la función **&lt;Fuente&gt;.InfoMemo** corresponde a: 

- del tamaño de un memo,

- una cadena vacía (""") si no hay tamaño. 


Por ejemplo: "TXT" (o "BIN")+TAB+TAB+25+TAB+TAB+TAB.

**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos (consulta de vista) utilizado.

**`<Memo item>: (Cadena de caracteres)`**

Nombre de la posición de nota a utilizar.

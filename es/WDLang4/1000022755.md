


## &lt;Variable Connection&gt;.DeleteDatabase (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una base de datos de un servidor HFSQL. Todas las conexiones que utilizan la base de datos eliminada están cerradas.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
MyConnection is Connection

// Open the connection
MyConnection.OpenConnection()
// Delete the database
MyConnection.DeleteDatabase()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteDatabase([<Database>] [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la base de datos,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la base de datos debe ser eliminada.

**`<Database>: (Cadena de caracteres opcional)`**

Nombre de la base de datos a eliminar. 
Si no se especifica este parámetro, se utilizará la base de datos definida en la conexión.. Se produce un error de WLanguage si no se define ninguna base de datos en la conexión.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Por Default, al suprimir la base de datos, los archivos correspondientes a los registros no se eliminan y los archivos correspondientes a las transacciones se suprimen. Este parámetro se utiliza para modificar este comportamiento:


|   |   |
| --- | --- |
| *hDeleteLog* | Elimina los archivos de registro y todas las operaciones registradas. |
| *hKeepTRS* | Almacena los ficheros de transacción. |

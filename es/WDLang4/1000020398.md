


## HNotifDeleteCCRecipient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los destinatarios de una notificación enviada a través de los Centros de Control (herramienta de mensajería WDBal). Esta notificación es enviada por el servidor HFSQL cuando ocurre un problema. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ctCSConnection is Connection
ctCSConnection.Provider = hAccessHFClientServer
ctCSConnection.User = "admin"
ctCSConnection.Password = ""
ctCSConnection.Server = "test:4900"
HOpenConnection(ctCSConnection)

HNotifConfigure(ctCSConnection, hNotifCCCS, "DOC:4988", "CC_DOC", "DOC", "")
HNotifAddCCRecipient(ctCSConnection, "Vince", hNotifWarning)

HNotifDeleteCCRecipient(ctCSConnection, "Vince")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HNotifDeleteCCRecipient(<Connection> , <Recipient>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md). 




**`<Recipient>: (Cadena de caracteres)`**

Nombre del destinatario que desea eliminar. Este nombre debe corresponder a un usuario definido en la base de datos de los Centros control.

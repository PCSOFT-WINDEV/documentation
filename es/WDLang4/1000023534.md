
## redisResponse (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **redisResponse** permite definir todas las características avanzadas de una respuesta recibida por [RedisEjecutarComando](../WDLang4/1000023541.md) 00 o [RedisPipelineObtener](../WDLang4/1000023543.md). Las características de esta respuesta se pueden conocer utilizando diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
INTERNAL PROCEDURE DisplayResult (r is redisResponse, sIdent="")
	SWITCH r.Type
		CASE redisTypeInt
			Trace(sIdent + "integer: " + r.Integer)
		CASE redisTypeError
			Trace(sIdent + "error: " + r.String)
		CASE redisTypeStatus
			Trace(sIdent + "status: " + r.String)
		CASE redisTypeNil
			Trace(sIdent + "Nil")
		CASE redisTypeArray
			Trace(sIdent + "array: ")
			FOR EACH r2 OF r.Array
				ShowResponse(r2, sIdent + TAB)
			END
		CASE redisTypeBuffer
			Trace(sIdent + "buffer in string: " + r.String)
			Trace(sIdent + "buffer in hexa: " + BufferToHexa(r.Buffer))
	END
END

r1 is redisResponse = RedisExecuteCommand(gRedis, "GET keyString")
DisplayResult(r1)
```

### Propiedades específicas de las variables redisResponse
<a name="propiedades_especificas_las_variables_redisresponse_ELTPARAGRAPHE000035"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una respuesta Redis:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Array | Array de redisResponse | Si la propiedad If **..Type** corresponde a la constante *redisTypeArray*, esta propiedad contiene un array de variables de tipo **redisResponse**. |
| Buffer | Buffer | Valor binario si la propiedad **Type** corresponde a la constante *redisTypeBuffer*. <br><br>**Esta propiedad es de solo lectura**. |
| Cadena | Cadena de caracteres | <br><br>- Si la propiedad **Type** corresponde a la constante *redisTypeStatus* , contiene la cadena correspondiente al estado devuelto. <br><br>- Si la propiedad **Type** corresponde a la constante *redisTypeError* , contiene el mensaje de error. <br><br>- Si la propiedad **Type** corresponde a la constante *redisTypeBuffer* , contiene el búfer convertido a una cadena usando UTF-8 encoding .<br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| escribir a máquina | Integro | Tipo del resultado. Este tipo puede corresponder a una de las siguientes constantes: <br><br>- *redisTypeBuffer*: El resultado corresponde a un buffer. <br><br>- *redisTypeInt*: El resultado corresponde a un número entero.<br><br>- *redisTypeError*: El resultado corresponde a un error. <br><br>- *redisTypeNil*: El resultado no tiene valor. <br><br>- *redisTypeStatus*: El resultado corresponde a un status. <br><br>- *redisTypeArray*: El resultado corresponde a una array. <br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Integro | Integro | Valor entero si la propiedad **..Type** corresponde a la constante *redisTypeInt*.<br><br>**Esta propiedad es de solo lectura**. |

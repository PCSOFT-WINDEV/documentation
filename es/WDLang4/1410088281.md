


## <redisConnection>Anular suscripción (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela una suscripción a un canal de difusión de Redis.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gRedis.Unsubscribe("MyChannel")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Connection>.Unsubscribe(<Channel>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Connection>: (Variable de tipo redisConnection)`**

Nombre del [redisConnection](../WDLang4/1000023527.md) Variable que corresponde a la conexión del servidor Redis.

**`<Channel>: (Cadena de caracteres)`**

Nombre del canal del que la aplicación va a darse de baja.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Operaciones al darse de baja
<a name="operaciones_darse_baja_ELTPARAGRAPHE000181"></a>

Al darse de baja, se ejecutará la Procedure especificada en [.Subscribe](../WDLang4/1410088279.md).




## &lt;Fuente&gt;.StatNbRec (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de entradas para un elemento clave determinado. Este número fue calculado durante el último cálculo estadístico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate statistics about the CustName key from the Customer data file
ResStat = Customer.StatNbRec(CustName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source>.StatNbRec([<Key item>])`
---

**`<Result>: (Entero)`**

Número de entradas encontradas para el artículo especificado. 
Corresponde a la constante *hInvalidRecNum* si:

- las estadísticas nunca se han calculado para la posición especificada.

- las estadísticas no están soportadas por el tipo de posición especificado. HFSQL permite realizar cálculos estadísticos sólo en posiciones clave.




**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos utilizado. 

**`<Key item>: (Cadena de caracteres opcional)`**

Nombre del elemento clave utilizado para calcular las estadísticas. Si no se especifica este nombre, **&lt;Fuente&gt;.StatNbRec** utilizará el último elemento utilizado por la última función de gestión HFSQL (función que empieza por "H").



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las estadísticas pueden ser actualizadas por [&lt;Fuente&gt;.Index](../WDLang4/1000025060.md) y [&lt;Fuente&gt;.StatCalculate](../WDLang4/1000025064.md).

- En los archivos de datos de HFSQL Classic, los cálculos estadísticos sólo pueden realizarse en las partidas clave.

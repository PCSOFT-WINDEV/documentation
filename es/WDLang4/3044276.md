


## Modo H.SQLServer (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite especificar la versión de Conector Nativo SQL Server (o Native SQL Server Access) que se utilizará. 

La constante *hNativeAccessSQLServer* no cambia. Debe utilizarse siempre en las funciones de tratamiento de la Conector Nativo SQL Server ([HDeclareExternal](../WDLang4/3044204.md), [HDescribeConnection](../WDLang4/3044205.md), [HListFile](../WDLang4/3044045.md), [HOpenConnection](../WDLang4/3044107.md)).
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`H.SQLServerMode = <Value>`
---

**`<Value>: (Integro)`**



- **0** para forzar el uso de la Conector Nativo SQL Server a través de la OLEDB.

- **1** (Default) para utilizar el Conector Nativo instalado . Si se instalan ambas versiones de la Conector Nativo, se utilizará la Conector Nativo vía OLE DB. 

- **3** para forzar el uso de la Conector Nativo SQL Server vía ODBC.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000041"></a>

Por Default, se utiliza la Conector Nativo seleccionada durante la instalación.. Si se instalan la Conectores Nativos vía OLE DB y la ODBC, se utilizará la Conector Nativo vía OLE DB.

El **H.SQLServerMode** Variable se utiliza para forzar un modo. El Variable debe ser modificado antes de utilizar el [HDescribeConnection](../WDLang4/3044205.md) y/o el [HOpenConnection](../WDLang4/3044107.md). 
La palabra clave **WD CLIENT VERSION** especificada en el [información opcional de conexión](../WDLang4/9000124.md) se utiliza para indicar el cliente de SQL Server a utilizar. 
Por ejemplo, la cadena "WD CLIENT VERSION = 2012" se utiliza para forzar el acceso a través de un cliente SQL Server 2012 instalado en la estación de trabajo (msoledbsql), preferible al controlador proporcionado por Windows (sqloledb).
<a name="NOTE0_2"></a>


### Nota
<a name="nota_ELTPARAGRAPHE000063"></a>

[HListProvider](../WDLang4/3044232.md) y [Fuente de SQLList](../WDLang4/3072027.md) tienen en cuenta el **H.SQLServerMode** Variable. 
<a name="NOTE0_3"></a>


### Las DLLs utilizadas
<a name="las_dlls_utilizadas_ELTPARAGRAPHE000075"></a>

- La Conector Nativo SQL Server vía OLE DB corresponde a: 

	- la biblioteca wd28 0sqlserver.dll (en modo de 32 bits),

	- la biblioteca wd28 0sqlserver64.dll (en modo de 64 bits).




- La Conector Nativo SQL Server vía ODBC corresponde a: 

- la biblioteca wd28 0sqlserver2.dll (en modo de 32 bits),

- la biblioteca wd28 0sqlserver2_64.dll (en modo de 64 bits).

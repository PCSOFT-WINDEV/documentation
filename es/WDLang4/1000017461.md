
## Descripción del FullTextIndex (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una variable **FullTextIndex description** se utiliza para describir uno o más índices de texto completo por programación.. Las características de estos índices pueden ser definidas por varios [Propiedades de WLanguage](#NOTE0_1)

Para añadir la descripción de este índice a la descripción de un archivo de datos, utilice [HDescribeFullTextIndex](../WDLang4/1000017407.md). La variable "FullTextIndex Description" se reinicializa y se puede utilizar para describir otro índice de texto completo por programación.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
MyFullTextIndex is FullTextIndex Description
CUSTOMER is Data Source

// Describe the "CUSTOMER" file
AFile.Name = "CUSTOMER"
AFile.Type = hFileNormal
AFile.FicCryptMethod = hEncryptionStandard

// Description of the "LASTNAME" item
AnItem.Name = "LASTNAME"
AnItem.Type = hItemText
AnItem.Size = 40
AnItem.KeyType = hDuplicatesKey
HDescribeItem(AFile, AnItem)

// Describe the "FIRSTNAME" item
AnItem.Name = "FIRSTNAME"
AnItem.Type = hItemText
AnItem.Size = 40
HDescribeItem(AFile, AnItem)

// Describe the full-text index
MyFullTextIndex.MinimumLength = 4
MyFullTextIndex.Name = "MyFullTextIndex"
MyFullTextIndex.Items = "LASTNAME+FIRSTNAME"
MyFullTextIndex.AccentSensitive = True
MyFullTextIndex.CaseSensitive = False
HDescribeFullTextIndex(AFile, MyFullTextIndex)

// Validate the file description
HDescribeFile(AFile)
// Create the Customer file (recognized because declared as data source)
HCreation(CUSTOMER)
// Create a record
CUSTOMER.LASTNAME = "Doe"
CUSTOMER.FIRSTNAME = "Andrew"
HAdd(CUSTOMER)
// Create a record
CUSTOMER.LASTNAME = "Doe"
CUSTOMER.FIRSTNAME = "Vince"
HAdd(CUSTOMER)
```

## Sintaxis

### Declarando una o más descripciones de FullTextIndex

`<Variable name> is FullTextIndex description
<Variable name 1>, <Variable name 2> are FullTextIndex descriptions
`
---

**`<Variable name>: ()`**

Nombre de la variable "hFullTextIndex Description" a declarar.

Observación: Las palabras clave <u><u><u><u>a</u></u></u></u> y <u><u><u><u>an</u></u></u></u> son opcionales: proporcionan una mejor legibilidad.  


<a name="SYNTAX2"></a>

### Describiendo una variable "Descripción del índice de texto completo"...

``
---

**Para describir una variable "FullTextIndex Description" (Descripción de FullTextIndex)**: 
Utilice las propiedades WLanguage específicas de las descripciones de los índices de texto completo. Para obtener más información, consulte [observaciones](#NOTE0_1).

**Para validar la descripción de una variable de tipo "FullTextIndex Description"**:
Utilice [HDescribeFullTextIndex](../WDLang4/1000017407.md).

### ¿Cómo describir los archivos de datos por programación?
<a name="¿como_describir_los_archivos_datos_por_programacion_ELTPARAGRAPHE000196"></a>

Para describir los archivos de datos por programación, usted debe:

**1.** Declarar las variables "[Descripción del archivo](../WDLang4/1514065.md)", "[Descripción de elemento](../WDLang4/1514071.md)", "**FullTextIndex Description**" y "[Descripción de la relación](../WDLang4/1514039.md)" (si es necesario).

**2.** Para cada archivo de datos:

- Describa las características de del archivo de datos a través del [Propiedades HFSQL](../Proprietes/2512070.md).

- Describir las características de los artículos a través de la [Propiedades HFSQL](../Proprietes/2512070.md).

- Validar la descripción de cada artículo ([HDescribeItem](../WDLang4/3044233.md)).

- Describir (si es necesario) las características de los índices de texto completo a través de [Propiedades HFSQL](../Proprietes/2512070.md).

- Validar la descripción de cada índice de texto completo ([HDescribeFullTextIndex](../WDLang4/1000017407.md)).

- Valide la descripción del archivo de datos ([HDescribeFile](../WDLang4/3044206.md)).




**3.** Describa (si es necesario) las características de los enlaces a través de [Propiedades HFSQL](../Proprietes/2512070.md).

**4.** Validar, si es necesario, la descripción de cada relación (función [HDescribeLink](../WDLang4/3044191.md).




## &lt;Fuente&gt;.FTListWord (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve una lista de las palabras del registro actual que tendrá en cuenta el índice de texto completo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrToken is array of strings = Document.FTListWord(FTIndexTitleContent)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Source>.FTListWord(<Full-text index>)`
---

**`<Result>: (Array de cadenas)`**

Variable de tipo Array que contiene las palabras que se tienen en cuenta en la indexación de texto completo.

**`<Source>: (Tipo de fuente especificada)`**

Nombre del archivo de datos HFSQL utilizado.

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo del archivo.


## Navegación por los archivos de datos HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
Los elementos "Clave" sirven para organizar los registros en un archivo de datos con el fin de optimizar la navegación o la búsqueda.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Búsqueda realizada en un archivo sin clave
<a name="busqueda_realizada_archivo_sin_clave_ELTTEXTE000119"></a>
Se leen todas las Record para encontrar al buscado Record.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=rubrique_cle1.gif)


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Búsqueda realizada en un archivo con una clave
<a name="busqueda_realizada_archivo_con_una_clave_ELTTEXTE000143"></a>
El uso de una clave permite crear un ÍNDICE. El índice da acceso directo a la Record buscada.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=rubrique_cle2.gif)

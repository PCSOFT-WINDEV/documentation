


## RedisPublish (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Publica un mensaje en un canal de difusión de Redis.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RedisPublish(gRedis, "MyChannel", "Hello Redis from WINDEV")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RedisPublish(<Connection> , <Channel> , <Message>)`
---

**`<Result>: (Integro)`**

Número de solicitudes que recibieron el mensaje. 

Observación: Esta información no está disponible cuando se conecta a un cluster de Redis.

**`<Connection>: (Variable de tipo redisConnection)`**

Nombre del [redisConnection](../WDLang4/1000023527.md) Variable que corresponde a la conexión del servidor Redis.

**`<Channel>: (Cadena de caracteres)`**

Nombre del canal en el que debe publicarse el mensaje. Esta cadena está codificada en formato UTF8.

**`<Message>: (Cadena de caracteres o memoria intermedia)`**

Mensaje para publicar en el canal: 

- Si es un parámetro String, se convertirá al formato UTF8.

- Si es un parámetro Buffer, se enviará en formato binario.

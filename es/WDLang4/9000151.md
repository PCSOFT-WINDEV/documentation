## Versión 19: HyperFileSQL y HFSQL
<a name="version_19_hyperfilesql_hfsql_ELTTEXTE000051"></a>
HFSQL es el nuevo nombre de HyperFileSQL. 

¡Lo que ha evolucionado desde su primera versión en 1989!

Para simplificar su nombre, la base de datos ahora es HFSQL, un nombre ya usado por muchos desarrolladores y administradores de base de datos.

HFSQL e HyperFileSQL son totalmente compatibles, en Classic, red, Cliente/Servidor, nube, modos embebidos.

Recordemos que HFSQL es la única base de datos del mundo compatible en Windows, Linux, iOS, MacOS, Android, independientemente del modo de uso: Clásico, móvil, Cloud, Cliente/Servidor.

¡Para mover una base de datos de un servidor Windows a Android (por ejemplo), lo único que tiene que hacer es copiar la base de datos y sus índices, y todo estará operando de manera directa!

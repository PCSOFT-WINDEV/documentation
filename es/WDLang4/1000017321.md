


## HListParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de parámetros guardados a partir de procedimientos almacenados en el servidor.

**Observación**: Los parámetros se guardan por base de datos en el archivo "user_param.fic".




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the list of values saved
// (Code of a stored procedure)
MyList is string
MyList = HListParameter()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la lista de parámetros guardados en el servidor (en el código de Procedure almacenado)

`<Result> = HListParameter()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de nombres de parámetros separados por caracteres CR (Carriage Return).  


<a name="SYNTAX2"></a>

### Lista de parámetros guardados en el servidor (desde el equipo Cliente)

`<Result> = HListParameter(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de nombres de parámetros separados por caracteres CR (Carriage Return). 

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




## HListGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de grupos de usuarios definidos para una conexión.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", ...
	"MyHFServer", "MyDatabase", hAccessHFClientServer, HORead)
// Open the connection
HOpenConnection("MyConnection")
Res is string
Res = HListGroup("MyConnection")
// Add the groups into a List Box control
LIST_Mylist.Add(Res)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListGroup(<Connection>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de los grupos existentes para la conexión especificada. Los nombres de los grupos están separados por caracteres CR (Carriage Return).

**Observación**: Si el usuario actual no tiene permisos para ver usuarios (constante *hRightsSeeUser*) en el servidor (función [HModifyServerRights](../WDLang4/3044328.md) o [HInfoServerRights](../WDLang4/3044322.md)), la función **HListGroup** solo devuelve el grupo actual.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión utilizada. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).

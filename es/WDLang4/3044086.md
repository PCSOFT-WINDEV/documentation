
## HErrorIntegrity (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si se produjo un error de integridad.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) De forma predeterminada, WINDEV incluye una [gestión automática de errores de integridad](../WDLang4/3044188.md). Esta gestión automática no está disponible en el código de procedimientos almacenados.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo se notifican los errores de integridad que se producen en los archivos de datos HFSQL.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Manage an integrity error
IF HModify(Customer) = False THEN
	IF HErrorIntegrity() = True THEN 
		Error("Integrity error: this record cannot be modified")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HErrorIntegrity()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se produjo un error de integridad,

- <u><u><u><u>False</u></u></u></u> en caso contrario (sin errores).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
De forma predeterminada, todas las reglas de integridad definidas en la descripción del análisis están activadas. Para activar o desactivar la gestión de una restricción de integridad, utilice la función [HSetIntegrity](../WDLang4/3044058.md).

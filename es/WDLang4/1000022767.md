


## &lt;Variable Connection&gt;.DeleteTask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una tarea programada de un servidor HFSQL Client/Server.

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Connection>.DeleteTask(<Task to delete>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la tarea,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Variable de tipo Connection)`**

Nombre de la [Conexión](../WDLang4/1514073.md) Variable que describe la conexión para la que se debe eliminar la tarea.

**`<Task to delete>: (Integer)`**

Identificador de la tarea a eliminar. Este identificador lo dan [&lt;Variable Connection&gt;.AddTask](../WDLang4/1000022645.md) y [&lt;Variable Connection&gt;.ListTask](../WDLang4/1000022700.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000206"></a>

Para tener los derechos para eliminar una tarea, debe hacerlo:

- ser el dueño de la tarea.

- ser super-usuario.

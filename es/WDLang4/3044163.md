
## HDBOpen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el archivo de datos de la xBase y el archivo "memo" si existe.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HDBOpen("ORDERS", "OR", "C:\FILE\ORDERS.DBF")
HDBIndex("ORDERS", "C:\FILE\ORDERS.NTX", "OrdNum")
HDBOpen("PRODUCT", "PR", "C:\FILE\PRODUCT.DBF")
HDBIndex("PRODUCT", "C:\FILE\PRODUCT.NTX", "ProdNum")

// Display the characteristics of the order 
HReadFirst("ORDERS", "OrdNum")
IF HOut() = False THEN
	HReadSeek("PRODUCT", "ProdNum", CO.PRODNUM)
	FileToScreen()
ELSE
	Error("Empty file")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HDBOpen(<Logical name of xBase file> , <Abbreviation of xBase file> , <Full name of xBase file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Logical name of xBase file>: (Cadena de caracteres)`**

Nombre lógico dado al archivo xBase. Este nombre se utilizará para manipular el archivo xBase por programación.

**`<Abbreviation of xBase file>: (Cadena de caracteres)`**

Prefijo (2 caracteres) utilizado a través de la programación para identificar un elemento del archivo. Por ejemplo, si el prefijo del fichero "Cliente" es CU, la sintaxis para utilizar el elemento "Nombre" de este fichero es CU.NOMBRE.
Atención: Varios archivos de datos abiertos con **HDBOpen** no pueden tener la misma abreviatura, aunque los archivos se hayan cerrado.

**`<Full name of xBase file>: (Cadena de caracteres)`**

Ruta y nombre completo del archivo xBase a abrir (con la extensión).

<a name="NOTE0_1"></a>

## Observaciones


### Enlace entre archivos xBase
<a name="enlace_entre_archivos_xbase_ELTPARAGRAPHE000053"></a>

Los enlaces entre los archivos xBase no se gestionan automáticamente. Por lo tanto, debes escribir el código de "enlace" para acceder a un Record en un archivo relacionado. La Record del archivo relacionado debe ser leída siempre que se lea el archivo:


```wl
// Read the file
IF HOut() = False THEN
	HReadSeek(FileName, KeyName, Key_Value)
END
```

<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000062"></a>

- Esta función no está disponible cuando se utiliza una base de datos a través de .

- Esta función no está disponible cuando se utiliza un ejecutable independiente..

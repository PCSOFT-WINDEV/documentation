


## HListREP (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de asignaciones de los archivos de datos manipulados por la aplicación actual, es decir, la lista de archivos físicos que corresponden a los archivos de datos descritos en el análisis.

Para más detalles sobre esta lista, véase [Lista de archivos de datos manipulados por una aplicación](../WDLang4/3044186.md).

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sAssignments is string
// Retrieve all the assignments of logical file
// CUSTOMER whose physical name is CUSTOMER2000 and that
// exist on disk
sAssignments = HListREP(".", "CUSTOMER", "CUSTOMER2000", hExisting)
```

## Sintaxis

### Lista de asignaciones a partir del archivo .REP

`<Result> = HListREP([<Name of REP file> [, <Logical file name> [, <Physical file name>]]] [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**



- Lista de asignaciones,

- Cadena vacía ("") si se produce un error. 


La lista de asignaciones tiene el siguiente formato:

```txt
<Logical name> + <TAB> + <Physical name> + TAB + <Directory> + CR +...
```

Dónde:

- &lt;Nombre lógico&gt;: Nombre del archivo de datos en el análisis. 

- &lt;Nombre físico&gt;: Nombre físico del archivo, sin ruta ni extensión.

- &lt;Directorio>: Directorio del archivo físico. 


![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) &lt;Resultado&gt; se completa con 2 nuevos valores:

- &lt;Nombre de servidor&gt;: Nombre del servidor HFSQL Client/Server.

- &lt;Nombre de base de datos&gt;: Nombre de la base de datos HFSQL Client/Server.




**`<Name of REP file>: (Cadena de caracteres opcional)`**

Nombre completo del archivo .REP analizado. Si este parámetro corresponde a una cadena vacía o a **"."** se analiza el archivo ".REP" actual. El archivo ".REP" actual también se analiza si no se especifican los tres primeros parámetros opcionales.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) No es posible especificar solo este parámetro. La sintaxis HListREP(&lt;Nombre del archivo .REP&gt;) no está disponible.

**`<Logical file name>: (Cadena de caracteres opcional)`**

Nombre lógico del archivo de datos cuyas asignaciones deben ser listadas. Si no se especifica este parámetro o es igual a **"\*"**, se buscarán las asignaciones de todos los archivos de datos.

**`<Physical file name>: (Cadena de caracteres opcional)`**

Nombre físico del archivo de datos cuyas asignaciones deben ser listadas. Si no se especifica este parámetro, se buscarán todas las asignaciones de &lt;Nombre lógico archivo&gt;.

**`<Options>: (Constante opcional de tipo Integer)`**

Todas las asignaciones se buscan de forma predeterminada. La constante *hExisting* permite buscar solo las asignaciones para las que realmente existe un archivo de datos.

### Lista de asignaciones a partir del archivo .WDD

`<Result> = HListREP([<Name of WDD File> [, <Logical file name> [, <Physical file name>]]] [, <Options>])`
---


**`<Name of WDD File>: (Cadena de caracteres opcional)`**

**`<Options>: (Constante opcional de tipo Integer)`**

Todas las asignaciones se buscan de forma predeterminada. La constante *hExisting* permite buscar solo las asignaciones para las que realmente existe un archivo de datos.




## HListConnectedUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los usuarios actualmente conectados a uno o más archivos manejados por una conexión Cliente/Servidor.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the connection
HDescribeConnection("MyConnection", "Julia", "Password", "MyHFServer", "MyDatabase", ...
	hAccessHFClientServer, hORead)
// Open the connection
HOpenConnection("MyConnection")
Result1 is string
Result1 = HListConnectedUser("MyConnection")
// Add the users into a List Box control
ListAdd(LIST_MyList, Result1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HListConnectedUser(<Connection name> [, <Database> [, <Physical file name>]])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de los diferentes usuarios que están utilizando:

- el archivo especificado.

- la base de datos especificada.




Si sólo se especifica el nombre de la conexión, muestra una lista de todos los usuarios conectados al servidor..

La información se devuelve en el siguiente formato:

```txt
<Username> + TAB + <Name of client computer> + TAB + 
<Name of client application> + TAB + <Internal version of client HFSQL library> + 
TAB + <Version of client HFSQL library> + TAB + 
<Priority of client application> + TAB + 
<Name of server node (if HFSQL cluster)> + <IP address of client computer> + CR + ...
```

**Observación**: Si el usuario actual no tiene derechos para ver los usuarios (constante *hRightsSeeUser* ) en el servidor ([HModifyServerRights](../WDLang4/3044328.md) o [HInfoServerRights](../WDLang4/3044322.md)), **HListConnectedUser** devuelve las conexiones del usuario actual.

**`<Connection name>: (Cadena de caracteres)`**

Nombre de la conexión utilizada. Esta conexión fue definida en el editor de análisis, o por el [HDescribeConnection](../WDLang4/3044205.md) o el [HOpenConnection](../WDLang4/3044107.md).

**`<Database>: (Cadena de caracteres opcional)`**

Nombre de la base de datos para la que queremos conocer a los usuarios conectados. 
Si no se especifica este parámetro y el parámetro &lt;Nombre físico del archivo&gt;, la función **HListConnectedUser** lista los usuarios conectados al servidor.

**`<Physical file name>: (Cadena de caracteres opcional)`**

Nombre del archivo para el que queremos conocer a los usuarios conectados. 
Si no se especifica este parámetro, la función **HListConnectedUser** lista los usuarios conectados a la base de datos.
Si no se especifica este parámetro y el parámetro &lt;Base de datos&gt;, la función **HListConnectedUser** lista los usuarios conectados al servidor.

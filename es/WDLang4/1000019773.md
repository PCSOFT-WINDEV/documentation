
## HInfoAnalysis( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Retorna información de un análisis (archivo WDD).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(HInfoAnalysis())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HInfoAnalysis([<Path of the WDD file> [, <Password of the WDD file>]])`
---

**`<Result>: (Cadena de caracteres)`**

Información del análisis. Esta información utiliza el siguiente formato: 

```txt
<Analysis GUID> + CR + <Generation Number>
```


**`<Path of the WDD file>: (Cadena de caracteres opcional)`**

Ruta completa del análisis (es decir, del archivo .WDD) a utilizar. 
Si este parámetro no se especifica o si es igual a una cadena vacía ("""), se utiliza el análisis actual.

**`<Password of the WDD file>: (Cadena de caracteres opcional)`**

Contraseña definida para el análisis. Si no se define ninguna contraseña, este parámetro debe ser igual a una cadena vacía ("").




## Funciones HDFS (sintaxis del prefijo)
			



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>
Las siguientes funciones se utilizan para gestionar archivos en un sistema HDFS (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;hdfsConnection Variable&gt;.Borrar](../WDLang4/1000021987.md) | Borra un fichero o un directorio en un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.CloseConnection](../WDLang4/1000021983.md) | Cierra una conexión con un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.CopyFrom](../WDLang4/1000021980.md) | Copia un archivo encontrado en un servidor Hadoop al ordenador local. |
| [&lt;hdfsConnection Variable&gt;.CopyTo](../WDLang4/1000021981.md) | Copia un fichero situado en la maquina local hacia un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.CrearDirectorio](../WDLang4/1000021982.md) | Crea un directorio en un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.Lista](../WDLang4/1000021984.md) | Muestra el contenido de un directorio que se encuentra en un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.OpenConnection](../WDLang4/1000021985.md) | Abre una conexión con un servidor Hadoop. |
| [&lt;hdfsConnection Variable&gt;.Renombrar](../WDLang4/1000021986.md) | Renombra un archivo o un directorio que se encuentra en un servidor Hadoop. |



Las funciones HDFS utilizan los siguientes tipos:



|   |   |
| --- | --- |
| [hdfsConnection](../WDLang4/1000021650.md) | El tipo **hdfsConnection** permite definir las características avanzadas de una conexión al sistema de archivos Hadoop (HDFS). |
| [hdfsRights](../WDLang4/1000021662.md) | El tipo **hdfsRights** permite definir las características avanzadas de los permisos de acceso a un elemento (archivo, enlace, directorio) del sistema de archivos Hadoop. |
| [hdfsStatus](../WDLang4/1000021973.md) | El tipo **hdfsStatus** permite definir las características avanzadas del estado de un elemento (archivo, directorio, enlace) del sistema de archivos Hadoop. |




Atención: Las funciones de HBase están disponibles en tiempo de edición y ejecución desde Windows Vista o Windows Server 2008.

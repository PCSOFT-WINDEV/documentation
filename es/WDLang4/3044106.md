
## HOpenAnalysis( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un análisis en formato HFSQL Classic. Esto permite acceder a los archivos de datos descritos en el análisis, así como a la estructura de estos archivos.

**Recordatorio**: Se puede abrir un solo análisis en un momento dado. Si un análisis ya estaba abierto antes de la llamada a la función, este análisis se cierra automáticamente, así como los archivos de datos de associated.

**Atención**: Al utilizar la función **HOpenAnalysis**, el contexto HFSQL se reinicializa automáticamente. Las siguientes operaciones se realizan automáticamente:

- Todos los archivos de datos abiertos se cierran.

- Todas las vistas HFSQL existentes se eliminan.

- Todas las consultas existentes se eliminan.

- Todas las declaraciones realizadas con las funciones [HDeclareExternal](../WDLang4/3044204.md) y [HDescribeFile](../WDLang4/3044206.md) se eliminan.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the SALESMGT analysis
HOpenAnalysis("C:\SALESMGT\SALESMGT.WDD", "Password")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Abrir un análisis HFSQL localmente

`<Result> = HOpenAnalysis(<Path of .WDD> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió el análisis,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite obtener más detalles sobre el problema.




**`<Path of .WDD>: (Cadena de caracteres)`**

Ruta completa del archivo ".WDD" del análisis. Este archivo contiene la descripción del análisis a abrir.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si sólo se especifica la Drive y el nombre del análisis, éste se busca en el directorio actual de la Drive especificada. 
Si no se especifica ni la unidad ni la ruta del análisis, el análisis se busca en el directorio actual del disco actual.
La siguiente sintaxis &lt;Unidad&gt;:&lt;Nombre del análisis&gt; se utiliza para buscar el análisis en el directorio actual de la Drive especificada . 

**`<Password>: (Cadena de caracteres opcional)`**



- Contraseña del análisis, definida en el editor de análisis. 

- Vaciar la cadena ("") si no hay contraseña es associated con el análisis.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Iniciando una aplicación associated con un análisis
<a name="iniciando_una_aplicacion_associated_con_analisis_ELTPARAGRAPHE000096"></a>

- Al iniciar una aplicación, el análisis associated con esta aplicación se abre automáticamente. El análisis se busca en la biblioteca principal de la aplicación, luego en el directorio de la aplicación.. 
	Para asegurarse de que se encuentra el análisis, le aconsejamos que lo incluya en la biblioteca principal de la aplicación.

- Si se llama a la función **HOpenAnalysis** desde una aplicación asociada al análisis, esta función:

	- no tiene efecto si el ruta del análisis especificado corresponde al ruta del análisis associated con la aplicación.

	- devuelve <u><u><u><u>False</u></u></u></u> si la ruta del análisis especificado no existe. En este caso, la aplicación sigue trabajando con el análisis actual.

	- abre el análisis y devuelve <u><u><u><u>True</u></u></u></u> si el análisis especificado existe y es válido.






<a name="NOTE0_2"></a>




### Iniciar una aplicación no associated con un análisis
<a name="iniciar_una_aplicacion_associated_con_analisis_ELTPARAGRAPHE000121"></a>

Cuando se inicia una aplicación que no es associated con un análisis, no se abre automáticamente ningún análisis. Por lo tanto, es necesario utilizar **HOpenAnalysis**.

Un análisis abierto con la función **HOpenAnalysis** se busca:

- primero, en la biblioteca principal de la aplicación según su el nombre físico completo.

- después, en el disco actual.



<a name="NOTE0_3"></a>




### Apertura de un análisis ya abierto
<a name="apertura_analisis_abierto_ELTPARAGRAPHE000138"></a>

Si el análisis abierto con la función **HOpenAnalysis** ya está abierto, la función **HOpenAnalysis** no tiene ningún efecto.
<a name="NOTE0_5"></a>




### Versión de análisis abierta con la función HOpenAnalysis
<a name="version_analisis_abierta_con_funcion_docparampagetitleshort_ELTPARAGRAPHE000150"></a>

**HOpenAnalysis no puede ser usado para abrir los análisis en formato Hyper File 5.5 (y anterior). Sólo se pueden abrir los análisis de HFSQL Classic.**

Para utilizar **HOpenAnalysis** en archivos Hyper File 5.5, debe utilizar el archivo " .WDD" en un análisis HFSQL Classic que contenga archivos Hyper File 5.5. En este caso, el archivo ".WDD" del análisis de la fuente WINDEV 5.5 debe encontrarse necesariamente en el directorio del ejecutable.

**También puede utilizar [HDeclare](../WDLang4/3044203.md) para manipular directamente un archivo Hyper File 5.5.**

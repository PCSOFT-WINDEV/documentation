


## HPriorityClient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la prioridad de una aplicación cliente. Las aplicaciones cuya prioridad debe modificarse son las aplicaciones cliente descritas en la estructura [HClient](../WDLang4/3044340.md).

**Observación**: Cuanto mayor sea la prioridad de la aplicación cliente, más rápido se procesarán las solicitudes de esta aplicación por parte del servidor..



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the HClient structure
HClient.Application = "MyClientApplication"

// Change the priority of the application
HPriorityClient("MyConnection", 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HPriorityClient(<Connection> , <Priority>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio de prioridad,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [HError](../WDLang4/3044088.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión al servidor HFSQL. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<Priority>: (Integro)`**

Nuevo valor de la prioridad que se asignará a la aplicación cliente. Este valor se encuentra entre -10 (prioridad baja) y 10 (prioridad alta). 
Por defecto, la prioridad es set a 0 cuando se inicia la aplicación (prioridad estándar).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar la prioridad de la aplicación cliente especificada, se requieren los derechos para modificar la prioridad.. Este permiso corresponde a la constante *hRightsPriority* (utilizada en las funciones [HInfoServerRights](../WDLang4/3044322.md) y [HModifyServerRights](../WDLang4/3044328.md)).

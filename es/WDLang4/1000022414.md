
## mongoFindOption (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **mongoFindOption** permite definir las características avanzadas de una búsqueda en una base de datos MongoDB. Las características de esta búsqueda pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyOptions is mongoFindOption
MyOptions.Limit = 5

let MySearch = MongoFind(CntMdb.Database["CRM"].Collection["products"], "{}", MyOptions)
```
<a name="Example2"></a>

```wl
Option is mongoFindOption
// Sort according to the age	
Option.Sort = "{ ""age"": -1 }"

sFilter is string = "{}"	
let MyResult = MongoFind(CntMdb.Database["CRM"].Collection["Actors"], sFilter, Option)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables mongoFindOption
<a name="propiedades_especificas_descripcion_las_variables_mongofindoption_ELTPARAGRAPHE000041"></a>

Las siguientes propiedades pueden ser utilizadas para manipular las opciones de búsqueda:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| brincar | Integro | Número de documentos omitidos. El primer Document devuelto será el Document correspondiente a este valor +1. |
| clasificar | Variante, Holgura o Cadena de caracteres | Document en formato JSON (Cadena de caracteres), en formato BSON (buffer) o variante que define los parámetros de clasificación.<br><br>Para get un tipo ascendente, el nombre de la clave debe ser seguido por ":1". <br><br>Para get una clase descendente, el nombre de la clave debe ser seguido por ":-1". |
| Límite | Integro | Número máximo de documentos que se devolverán durante la búsqueda. |
| Proyección | Variante, Holgura o Cadena de caracteres | Document en formato JSON (cadena de caracteres), en formato BSON (memoria intermedia) o variante que define las posiciones a devolver. |




## HRefreshView (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vuelve a calcular el contenido de una vista materializada. 

**Observación**: También tiene la posibilidad de actualización la vista mediante la sentencia SQL ["REFRESH MATERIALIZED VIEW"](../Editeurs/2034001.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HRefreshView(MyConnection, MyView)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HRefreshView(<Connection> , <View> [, <Option>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se actualizó la vista materializada, 

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [HErrorInfo](../WDLang4/3044071.md) permite identificar el error.




**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Conexión a utilizar. Esta conexión corresponde a: 

- el nombre de la conexión definida en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- el nombre de una variable de tipo [Conexión](../WDLang4/1514073.md).




**`<View>: (Cadena de caracteres)`**

Nombre de la vista materializada cuyos datos deben recalcularse.. Esta vista fue creada: 

- a través de la editor de análisis. 

- mediante el comando SQL ["CREATE MATERIALIZED VIEW"](../Editeurs/2034001.md).




**`<Option>: (Constante opcional de tipo Integer)`**

Modo para refrescar la vista materializada. 


|   |   |
| --- | --- |
| *hNoWait* | La actualización de vista materializada no es una operación de bloqueo: se realiza en la tarea de Background. [HInfoView](../WDLang4/1000021619.md) devuelve la información de la vista materializada después de ser refrescada. |
| Parámetro no especificado (por Default) | El actualización de vista materializada se bloquea para quien lo solicitó.. &lt;Resultado&gt; permite conocer el informe de la actualización. |

## Observaciones


### Derechos necesarios
<a name="derechos_necesarios_ELTPARAGRAPHE000123"></a>

Para recalcular el contenido de la vista materializada, debe tener: 

- los permisos correspondientes (constante *hRightsManageTask*) en el servidor,

- para cada base de datos correspondiente (es decir, que contenga una vista para procesar), los permisos de conexión (constante *hRightsConnection* o *hRightsEncryptedConnection*). 




Para saber si estos derechos están concedidos a un usuario o a un grupo, utilice [HInfoServerRights](../WDLang4/3044322.md) o [HInfoDatabaseRights](../WDLang4/3044320.md).

Para modificar los derechos concedidos a un usuario o a un grupo de usuarios, utilice [HModifyServerRights](../WDLang4/3044328.md) o [HModifyDatabaseRights](../WDLang4/3044326.md).

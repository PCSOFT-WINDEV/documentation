


## HReset (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa: 

- una o todas las variables de los elementos de archivo con sus valores Default. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) uno o todos los elementos de una [Record](../WDLang4/1000020962.md) Variable con sus valores Default.


Los valores de Default se definen cuando el archivo de datos se describe en el editor de análisis.

**Observaciones**: 

- La función **HReset** no modifica el registro en el archivo de datos, solo modifica los datos en memoria. 

- La función **HReset** modifica la propiedad [NewRecord](../Proprietes/1000017090.md) del archivo de datos correspondiente: esta propiedad cambia a <u><u><u>True</u></u></u>.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Assign the default values to the items
HReset(Customer)
// Display
FileToPage()
```

## Sintaxis

### Inicializar los elementos de un archivo de datos con sus valores predeterminados

`<Result> = HReset([<Data file> [, <Item>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la inicialización,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) devuelve más detalles.




**`<Data file>: (Cadena de caracteres opcional)`**

Nombre del archivo de datos utilizado. Si no se especifica este nombre, la función **HReset** utilizará el último archivo de datos utilizado por la última función HFSQL (que empieza por "H").

**`<Item>: (Cadena de caracteres opcional)`**

Nombre del elemento a inicializar. Si no se especifica este parámetro o si es igual a "\*", la inicialización se realizará para todos los elementos del archivo de datos.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta. Todos los elementos se reinicializan automáticamente.

### Inicializando los elementos de Record con sus valores por defecto (Record Variable)

`<Result> = HReset(<Record> [, <Item>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la inicialización,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. La función [HErrorInfo](../WDLang4/3044071.md) devuelve más detalles.




**`<Record>: (Variable de tipo Record)`**

Nombre de la [Record](../WDLang4/1000020962.md) Variable utilizada. 

**`<Item>: (Cadena de caracteres opcional)`**

Nombre del elemento a inicializar. Si no se especifica este parámetro o si es igual a "\*", la inicialización se realizará para todos los elementos de Record.
![Hyper File 5.5](https://doc.pcsoft.fr/ext/images/us/HF55.png) Este parámetro no se tiene en cuenta. Todos los elementos se reinicializan automáticamente.

## Observaciones


### Abrir archivos
<a name="abrir_archivos_ELTPARAGRAPHE000198"></a>

La función **HReset** abre el archivo de datos utilizado. Por lo tanto, el archivo de datos debe haber sido creado.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Durante la gestión personalizada de errores, se debe probar la función [HErrorLock](../WDLang4/3044099.md) después de llamar a la función **HReset**.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Variable de tipo Record
<a name="variable_tipo_record_ELTPARAGRAPHE000223"></a>

Para restablecer una Variable de tipo [Record](../WDLang4/1000020962.md), también puede utilizar [VariableReset](../WDLang1/3013069.md).

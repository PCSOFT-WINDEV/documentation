


## &lt;Captcha&gt;.display (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un nuevo Captcha en un Captcha control. Si un captcha ya fue mostrado en el control, sus parámetros son reiniciados: 

- nuevo texto.

- nueva deformación del texto, ....



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Proposes a new captcha to the user
CPTCH_Captcha1.Display()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Captcha control>.Display()`
---

**`<Captcha control>: (Nombre del control)`**

Nombre del Captcha control donde debe mostrarse el captcha.

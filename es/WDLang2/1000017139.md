


## CellDisplayDialog (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un control Celda en una página con un efecto OVI (Oscurecimiento de Ventanas Inactivas).. Esta función se utiliza para simular fácilmente una ventana de diálogo en el código del navegador, utilizando un control Celda en una página.

Para obtener más información, consulte [Comunicación con el usuario de la Web](../WDChamp/1012026.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display a Cell control containing a question
CellDisplayDialog(CELL_Question, cellCenter, True, 50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`CellDisplayDialog(<Cell control> [, <Display position> [, <Take DDW into account> [, <Dimming rate>]]])`
---

**`<Cell control>: (Nombre del control)`**

Nombre de la control Celda que se mostrará.

**`<Display position>: (Constante opcional de tipo Integer)`**

Visualizar la posición de la celda:


|   |   |
| --- | --- |
| *cellBottomLeft* | El control Celda está en la parte inferior izquierda de la parte de Visible de la página actual. |
| *cellBottomRight* | El control Celda está en la parte inferior derecha de la parte de Visible de la página actual. |
| *cellCenter*<br>(Valor predeterminado) | El control Celda está centrado en la parte de Visible de la página actual. |
| *cellPositionXY* | La control Celda se mantiene en su posición actual. |
| *cellTopLeft* | El control Celda está en la parte superior izquierda de la parte de Visible de la página actual. |
| *cellTopRight* | El control Celda está en la parte superior derecha de la parte de Visible de la página actual. |



**`<Take DDW into account>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para mostrar el control Celda con el efecto OVI (Oscurecimiento de Ventanas Inactivas). La página que se encuentra detrás del control Celda se muestra en gris

- <u><u><u><u>False</u></u></u></u> para desactivar el efecto OVI.




**`<Dimming rate>: (Integro)`**

Tasa de grises (incluidos entre 0 y 100) utilizada por la OVI. 
El valor máximo (100) se utilizará si el valor especificado es superior a 100. 

La OVI se ignora si el valor especificado es inferior a 0. La tasa de gris por defecto es set a 18%.

<a name="NOTE0_1"></a>

## Observaciones


### Características de la control Celda
<a name="caracteristicas_control_celda_ELTPARAGRAPHE000098"></a>

- La control Celda debe ser superpuesta (pestaña "Detalles" de la ventana control Celda description).

- Para evitar que aparezcan barras de desplazamiento innecesarias en el navegador, marque la opción "Posicionamiento automático al cargar para evitar las barras de desplazamiento" (pestaña "Detalles" de la ventana control Celda description). Esta opción se utiliza para posicionar automáticamente el control Celda en la posición (0,0) al cargar la página.. A continuación, se tiene en cuenta la posición de visualización definida con la función **CellDisplayDialog**.




**Observación**: Para ejecutar el código del servidor desde un control Celda, utilice [AJAXExecute](../WDLang2/3055114.md) o [AJAXExecuteAsynchronous](../WDLang2/3055115.md).
<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000122"></a>

- Se debe utilizar la función **CellDisplayDialog** para que se muestre el control Celda:

	- en un botón "Ninguno. En este caso, sólo se ejecutará el código del navegador del botón.

	- en un botón "Submit" y activar AJAX en el código de este botón. En este caso, se ejecutarán los códigos del navegador y del servidor del botón.




- La control Celda mostrada puede ocultarse con [CeldaCerrarDiálogo](../WDLang2/1000017140.md).

- Los navegadores no actualización la visualización en medio de un código JavaScript: **CellDisplayDialog** y [CeldaCerrarDiálogo](../WDLang2/1000017140.md) deben utilizarse en diferentes eventos/procesos. Si estas funciones se utilizan en el mismo evento/Process, el control Celda no se muestra.

- No se pueden mostrar dos controles Celda diferentes al mismo tiempo utilizando la función **CellDisplayDialog**.

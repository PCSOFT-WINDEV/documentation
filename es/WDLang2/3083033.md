Las siguientes constantes se utilizan para gestionar los controles del Looper:




|   |   |   |
| --- | --- | --- |
| **Constante** | **Detalles** | **Utilizado por las funciones** |
| **liControlName** | El nombre de la control que se encuentra en x,y. | [LooperInfoXY](../WDLang2/3083022.md) |
| **liBreakName** | Nombre de la ruptura en el punto x, y. | [LooperInfoXY](../WDLang2/3083022.md) |
| **liLineNumber** | Número de la fila en el punto x, y. | [LooperInfoXY](../WDLang2/3083022.md) |
| **liOriginScreen** | El origen de la pantalla corresponde al punto (0.0)). | [LooperInfoXY](../WDLang2/3083022.md) |
| **loopAnimationNone** | No se reproduce ninguna animación durante la eliminación. | [LooperDelete](../WDLang2/3083008.md) |
| **loopAnimationDelete** | Reproduce una animación durante el borrado. Esta animación corresponde a un efecto de desaparición al reducir la altura de la fila. | [LooperDelete](../WDLang2/3083008.md) |

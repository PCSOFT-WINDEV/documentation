
## GglAnalyticsAddEvent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Event en Google Analytics. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
GglAnalyticsAddEvent("Video", "play", "product trial", "pushchair")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GglAnalyticsAddEvent(<Category> , <Action> [, <Caption> [, <Event number> [, <Options>]]])`
---

**`<Category>: (Cadena de caracteres)`**

Nombre de la categoría. Este parámetro se utiliza para agrupar varios eventos ("Video" por ejemplo).

**`<Action>: (Cadena de caracteres)`**

Nombre de la acción relacionado al Event("play", por ejemplo). 

**`<Caption>: (Cadena de caracteres opcional)`**

Leyenda adicional utilizada para simplificar la presentación de eventos ("Publicidad" por ejemplo).
Este parámetro corresponde a una cadena vacía de Default. 

**`<Event number>: (Entero opcional)`**

Número asociado a la Event. De forma predeterminada, este parámetro corresponde a 0. 

**`<Options>: (Variant opcional)`**

Opciones adicionales dependientes de Google Analytics. Para más detalles sobre los posibles valores, consulte la documentación de Google Analytics en [https://developers.google.com/analytics/devguides/collection/analyticsjs/events](https://developers.google.com/analytics/devguides/collection/analyticsjs/events). 
Este parámetro es set a Nula por Default.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para utilizar esta función, debe tener y guardar un identificador de Google Analytics en el proyecto. Para obtener más información, consulte [WEBDEV y Google Analytics](../WDLang2/1000022374.md).

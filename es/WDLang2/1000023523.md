


## &lt;Posición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve la barra de desplazamiento para mostrar una fila específica o devuelve el índice de la primera fila que se muestra en el control Looper.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Index of the first row displayed in "LOOP_Product"
ResPosition = LOOP_Product.Position()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Mostrando un control Looper de un fila específico

`<Looper control>.Position(<Index>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**Atención**: No se puede utilizar control Looper lineal. 

**`<Index>: (Integro)`**

Índice del fila que se mostrará en la primera posición del control Looper. 
El último fila es seleccionado por Default si este parámetro es mayor que el número de filas del control Looper (devuelto por [&lt;Looper&gt;.Count](../WDLang2/1000023522.md)).


<a name="SYNTAX2"></a>

### Obtener el índice de la primera fila mostrada en una control Looper

`<Result> = <Looper control>.Position()`
---

**`<Result>: (Integro)`**

Índice del primer fila mostrado en el control Looper especificado .

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**Atención**: No se puede utilizar control Looper lineal.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Barra de selección y función &lt;Posición
<a name="barra_seleccion_funcion_docparampagetitleshort_ELTPARAGRAPHE000269"></a>

**&lt;Posición** no cambia el fila actual (es decir, el fila resaltado por banda de selección ).

Por lo tanto, si el control Looper tiene el foco al ejecutar la función **&lt;Posición**, la fila actual y la fila especificada deben ser visibles al mismo tiempo. Si el fila actual y el fila especificado no pueden visualizarse al mismo tiempo, el control Looper se visualizará desde el fila actual (y no desde el fila en <Índice>).
<a name="NOTE0_2"></a>


### Mostrar una fila en la primera posición de la control Looper (sintaxis 1)
<a name="mostrar_una_fila_primera_posicion_control_looper_sintaxis_1_ELTPARAGRAPHE000288"></a>

La fila que aparece en la primera posición puede no ser la fila solicitada . De hecho, la función **&lt;Posición** intenta mostrar un número máximo de filas en el control Looper.

La función **&lt;Posición** mostrará la fila especificada solo si esta fila va seguida de un número suficiente de filas no vacías.
<a name="NOTE0_3"></a>




## Funciones de banner deslizante (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular la Bandera deslizante control a través de la sintaxis del prefijo: 



|   |   |
| --- | --- |
| [&lt;Bandera deslizante&gt;.Primero](../WDLang2/1000022629.md) | Muestra el primer plano de un Banner deslizante control. |
| [&lt;Bandera deslizante&gt;.Siguiente](../WDLang2/1000022626.md) | Muestra el siguiente plano de un Banner deslizante control. |
| [&lt;Banner deslizante&gt;.anterior](../WDLang2/1000022627.md) | Muestra el plano anterior de un Banner Deslizante control. |
| [&lt;Banner deslizante&gt;.StartSliding](../WDLang2/1000022631.md) | Permite el deslizamiento de automatic en un banner deslizante control. |
| [&lt;Banner deslizante&gt;.StopSliding](../WDLang2/1000022628.md) | Detiene el deslizamiento de automatic en una pancarta deslizante control. |
| [&lt;Banner deslizante&gt;Último](../WDLang2/1000022630.md) | Muestra el último plano de un Banner deslizante control. |

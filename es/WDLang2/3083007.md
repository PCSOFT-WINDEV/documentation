


## Funciones de control Looper
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>






Las siguientes funciones se utilizan para gestionar los controles del Looper:



|   |   |
| --- | --- |
| [GaleríaCerrarPopup](../WDLang2/1000023478.md) | Cierra el popup de zoom de automatic para la galería de Image. |
| [GaleríaDisplayPopup](../WDLang2/1000023476.md) | Muestra la ventana emergente de zoom automático para el control Galería de imágenes. |
| [GalleryPopupDisplayed](../WDLang2/1000024975.md) | Indica si se muestra la popup automática de zoom del control Galería de imágenes. |
| [LooperAdd](../WDLang2/3083006.md) | Añade una fila al final de una control Looper. |
| [LooperAddInProgress](../WDLang2/1000021638.md) | Modifica o devuelve el estado de visibilidad de la ventana de espera interna al añadir elementos adicionales en un Infinite control Looper. |
| [LooperAddIP](../WDLang2/1410089366.md) | Agrega una fila como página interna al final de un control Looper rellenado mediante programación. |
| [LooperAddIW](../WDLang2/1000023439.md) | Agrega una fila como ventana interna al final de un control Looper rellenado mediante programación. |
| [LooperAddLine](../WDLang2/3083004.md) | Añade una fila al final de una control Looper. |
| [LooperBreakIndex](../WDLang2/3083029.md) | Devuelve el índice de la cabecera y el pie de página de una pausa dada en una control Looper. |
| [LooperCancelSwipe](../WDLang2/1000021988.md) | Cancela la visualización de la ventana interna mostrada en un control Looper después del deslizamiento de fila. |
| [LooperCollapse](../WDLang2/1000019770.md) | Contrae una ruptura previamente expandida en un control Looper. |
| [LooperCollapseAll](../WDLang2/1000019772.md) | Colapsa todas las roturas expandidas en una control Looper. |
| [LooperCollapseExpand](../WDLang2/1000019970.md) | En una determinada fila de un control Looper: <br><br>- colapsa una pausa (si está expandida),<br><br>- amplía una pausa (si está colapsada).<br><br><br> |
| [LooperCount](../WDLang2/3083014.md) | Devuelve el número de filas de una control Looper. |
| [LooperDelete](../WDLang2/3083008.md) | Borra una fila de una control Looper. |
| [LooperDeleteAll](../WDLang2/3083012.md) | Elimina todas las filas de una control Looper poblada de forma programática. |
| [LooperDeleteSelect](../WDLang2/1000023336.md) | Elimina las filas seleccionadas de una control Looper. |
| [LooperDisplay](../WDLang2/3083001.md) | Actualiza a control Looper a partir de un archivo de datos desde una posición determinada: las modificaciones realizadas en el archivo relacionado se aplican al looper. |
| [LooperEndFilling](../WDLang2/1000022371.md) | Finaliza la Process para rellenar una control Looper de forma programada. |
| [LooperExpand](../WDLang2/1000019771.md) | Amplía una pausa en la control Looper en una fila determinada . |
| [LooperExpandAll](../WDLang2/1000019774.md) | Expande todas las pausas en una control Looper con pausa. |
| [LooperInfoXY](../WDLang2/3083022.md) | Se utiliza para encontrar la fila o la control situada en una posición específica en la control Looper (X, Y). |
| [LooperInsert](../WDLang2/3083003.md) | Inserta una fila en una control Looper. |
| [LooperInsertIP](../WDLang2/1410089367.md) | Inserta una fila como página interna en un control Looper rellenado mediante programación. |
| [LooperInsertIW](../WDLang2/1000023440.md) | Inserta una fila como ventana interna en un control Looper rellenado mediante programación. |
| [LooperInsertLine](../WDLang2/3083002.md) | Inserta una fila en una control Looper. |
| [LooperModify](../WDLang2/3083013.md) | Cambia los valores de los atributos en una fila de una control Looper. |
| [LooperModifyIP](../WDLang2/1410089368.md) | Modifica una fila que corresponde a una página interna en un control Looper rellenado mediante programación. |
| [LooperModifyIW](../WDLang2/1000023441.md) | Modifica una fila que corresponde a una ventana interna de un control Looper rellenado mediante programación. |
| [LooperModifyLine](../WDLang2/3083015.md) | Cambia los valores de los atributos en una fila de una control Looper basada en un archivo de datos o poblada programáticamente. |
| [LooperMoveLine](../WDLang2/1000017248.md) | Mueve un fila o intercambia dos filas en un control Looper. |
| [LooperPosition](../WDLang2/3083023.md) | Mueve la barra de desplazamiento para mostrar una fila específica o devuelve el índice de la primera fila que se muestra en el control Looper. |
| [LooperRecNum](../WDLang2/3083010.md) | Devuelve el número de la Record actual en una control Looper basada en un archivo de datos. |
| [LooperRefreshVisible](../WDLang2/1000021195.md) | Cambia o devuelve el estado de visibilidad de la barra actualización en un control Looper. |
| [LooperRestorePositionAndSelection](../WDLang2/1000021789.md) | Recupera la posición de la barra de desplazamiento en un control Looper así como los elementos seleccionados. |
| [LooperSave](../WDLang2/3083028.md) | Valida la modificación o adición del Record associated con el fila actual en el control Looper. |
| [LooperSavePositionAndSelection](../WDLang2/1000021788.md) | Ahorra:<br><br>- la posición de la barra de desplazamiento en un control Looper,<br><br>- el elemento o elementos seleccionados.<br><br><br>Estos elementos pueden ser restaurados por [LooperRestorePositionAndSelection](../WDLang2/1000021789.md). |
| [LooperSearch](../WDLang2/3083005.md) | Busca un valor en un atributo de una control Looper. |
| [LooperSearchEverywhere](../WDLang2/1000021997.md) | Busca un valor en todos los atributos de una control Looper. |
| [LooperSelect](../WDLang2/3083024.md) | Devuelve el índice del elemento seleccionado en una control Looper. |
| [LooperSelectCount](../WDLang2/3083027.md) | Devuelve el número de filas seleccionadas en una control Looper. |
| [LooperSelectMinus](../WDLang2/3083026.md) | Deselecciona una fila en una control Looper. |
| [LooperSelectPlus](../WDLang2/3083025.md) | Selecciona un fila en un control Looper. |
| [LooperSort](../WDLang2/3083009.md) | Permite:<br><br>- Ordena una control Looper según uno o más atributos.<br><br>- Ordena un control Looper en todos sus atributos.<br><br>- Cancel una clase que se realizó de antemano (lo que significa ignorar la clase).<br><br><br> |
| [LooperStartFilling](../WDLang2/1000022370.md) | Inicia la Process para rellenar una control Looper de forma programada. |
| [LooperToExcel](../WDLang2/3083030.md) | Crea un fichero Excel a partir de los datos encontrados en una control Looper (hojeando control Looper o control Looper de la memoria). |
| [LooperToWord](../WDLang2/3083031.md) | Crea un archivo de Word (.RTF) utilizando los datos de un archivo control Looper. |
| [LooperToXML](../WDLang2/3083032.md) | Crea un archivo XML con los datos de una control Looper (control Looper basada en un archivo de datos o rellenada mediante programación). |

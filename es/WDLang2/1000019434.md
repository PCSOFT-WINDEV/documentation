


## ImageOccurrence (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número total de imágenes que tiene en cuenta el scroll.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the number of images taken into account by the scroll
Trace(ImageOccurrence(IMG_Banner))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ImageOccurrence(<Image control>)`
---

**`<Result>: (Integro)`**

Número total de imágenes para el scroll.

**`<Image control>: (Nombre del control)`**

Nombre del control a utilizar:

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.




## WBAdminUpdateAccount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza las características de una cuenta WEBDEV en un WEBDEV Application Server. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
wbAuthLogin is WBAdminAuth
wbAuthLogin.ServerAddress = "https://my-server.internal/"
wbAuthLogin.User = "WWW\DATA"
wbAuthLogin.Password = "www"

list_accounts is array of WBAdminAccounts = WBAdminListAccount(wbAuthLogin)
WBAccount is WBAdminAccount = list_accounts[1]
WBAccount.StatisticsAccount = True

Result_Update is boolean = WBAdminUpdateAccount(wbAuthLogin, WBAccount)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WBAdminUpdateAccount(<WEBDEV Application Server> , <Account>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se actualizó la cuenta,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<WEBDEV Application Server>: (Variable de tipo WBAdminAuth)`**

Nombre de la variable de tipo [WBAdminAuth](../WDLang2/1410089328.md) que describe el servidor y la cuenta de administrador utilizada para consultar a WEBDEV Application Server. 

**`<Account>: (Variable de tipo WBAdminAccount)`**

Nombre de la variable de tipo [WBAdminAccount](../WDLang2/1410089382.md) que contiene la descripción de la cuenta que debe actualizarse.


## FramesetRefresh( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza un conjunto de marcos que se muestra en el navegador del usuario desde el contexto del servidor. Los conjuntos de marcos actualizados corresponden a los conjuntos de marcos definidos en el editor, en su estado inicial.. Se muestran las páginas iniciales del conjunto de marcos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refresh the Frameset1
FramesetRefresh(Frameset1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`FramesetRefresh(<Frameset name>)`
---

**`<Frameset name>: (Cadena de caracteres)`**

Nombre del conjunto de marcos a actualización. 
Si este parámetro corresponde a una cadena vacía (""), el nombre del conjunto de marcos es el del conjunto de marcos al que pertenece el actual Event.

**Atención**: El contexto del conjunto de marcos debe existir en el servidor. El contexto del conjunto de marcos es creado por [FramesetDisplay](../WDLang2/3058006.md) y [ContextoAbrir](../WDLang2/3058004.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se produce un error de WLanguage si se utiliza **FramesetRefresh** sin parámetros en el código de inicialización del conjunto de marcos.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

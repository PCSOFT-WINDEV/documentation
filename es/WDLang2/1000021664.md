


## &lt;Página&gt;. uso (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una página WEBDEV en el navegador del usuario y cierra todos los contextos de página actualmente abiertos en el servidor.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch the application to English
Nation(nationEnglish)
// Redisplay the PAGE_Login page with captions in English
PAGE_Login.Use()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Page>.Use([<Parameter 1> [... [, <Parameter N>]]])`
---

**`<Page>: (Nombre de la página)`**

Nombre de la página a mostrar.br>En modo de prueba, especifique la ruta completa de la página si la página a abrir se encuentra en un directorio distinto al directorio del proyecto

**`<Parameter 1>: (Tipo de valor enviado a la página (opcional))`**

Parámetros a pasar a la apertura Event de la página a abrir ("Declaraciones globales" Event de la página). Estos parámetros son pasados por reference y son considerados como variables globales a la página.



**`<Parameter N>: (Tipo de valor enviado a la página (opcional))`**

Parámetros a pasar a la apertura Event de la página a abrir ("Declaraciones globales" Event de la página). Estos parámetros son pasados por reference y son considerados como variables globales a la página.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Parámetros pasados a la página para abrir
<a name="parametros_pasados_pagina_para_abrir_ELTPARAGRAPHE000220"></a>Los parámetros se recuperan en la apertura Event de la página ("Declaraciones globales" Event de la página). Simplemente escriba la siguiente línea al principio de las "declaraciones globales" Event:


```txt
PROCEDURE <Page name> (<Parameter 1> [, ...[, <Parameter N>]]])
```

<a name="NOTE0_2"></a>


### Sitios multilingües
<a name="sitios_multilingues_ELTPARAGRAPHE000232"></a>

**&lt;Página&gt;. uso** puede utilizarse en sitios multilingües. Esta función sirve para reabrir una página teniendo en cuenta el idioma especificado por [Nación](../WDLang1/3054014.md).
<a name="NOTE0_3"></a>


### Principio para abrir una página
<a name="principio_para_abrir_una_pagina_ELTPARAGRAPHE000247"></a>

**&lt;Página&gt;. uso** hace lo siguiente:

- cerrar todo el contexto abierto en el servidor

- crear un contexto de página en el servidor

- devolver la página al navegador del usuario web.




Observación: Si se utiliza la función **&lt;Página&gt;. uso** en el código de un control Botón, la página se abrirá en el destino definido para este control Botón(pestaña "General" de la descripción del control o función [ChangeTarget](../WDLang2/3058005.md)).
<a name="NOTE0_4"></a>


### Eventos y códigos que se ejecutan cuando se llama a &lt;Página&gt;. uso
<a name="eventos_codigos_que_ejecutan_cuando_llama_docparampagetitleshort_ELTPARAGRAPHE000270"></a>

Los siguientes eventos se ejecutan cuando se muestra la página:

- Código de apertura de la página ("Declaraciones globales" Event de la página). 

- "Iniciando" Event de los controles de la página. 

- Mostrar página en el navegador.

- Código Javascript associated con la página y los controles.

- Código WLanguage después de la llamada a **&lt;Página&gt;. uso** (si se utiliza).

### Abrir una página de un componente externo
<a name="abrir_una_pagina_componente_externo_ELTPARAGRAPHE000291"></a>Para abrir una página de un componente externo, utilice el nombre de la página del componente externo (el componente incluido en el proyecto).. Por ejemplo:

```wl
ComponentPage.Use()
```
 Si se produce un conflicto con un elemento del proyecto, el nombre del elemento debe ir precedido del nombre del componente externo. Por ejemplo:

```wl
MyComponent.Page.Use()
```
 Para utilizar el nombre de la página en un Variable, se debe especificar el nombre del componente externo. Por ejemplo:

```wl
sPage is string = "MyComponent.MyPage"

sPage.Use()
```

<a name="NOTE0_6"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000321"></a>

- Para volver a mostrar una página, se recomienda utilizar [&lt;Página&gt;.Refresh](../WDLang2/1000021659.md).

- Para mostrar una página sin cerrar los contextos abiertos en el servidor, le recomendamos que utilice [&lt;Página&gt;.display](../WDLang2/1000021661.md).

- No debe utilizar el código WLanguage después de una llamada a **&lt;Página&gt;. uso**.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

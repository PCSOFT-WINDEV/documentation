
## ChangeSourcePage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia dinámicamente la página que se muestra en un control Página interna. Durante esta modificación, se pueden pasar parámetros a la nueva página interna.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load IPAGE_InternalPage
// in IPAGE_InternalPageControl
ChangeSourcePage(IPAGE_InternalPageControl, IPAGE_InternalPage)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ChangeSourcePage(<Internal Page control> , <Internal page> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Internal Page control>: (Nombre del control)`**

Nombre del control Página interna que se modificará.

**`<Internal page>: (Nombre de la página interna)`**

Nombre de la página interna que se asociará al control Página interna.

**`<Parameter 1>: (Tipo de parámetro (opcional))`**

Primer parámetro que se pasará a la página interna.



**`<Parameter N>: (Tipo de parámetro (opcional))`**

Enésimo parámetro que se pasará a la página interna.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Eventos ejecutados al cambiar la página interna
<a name="eventos_ejecutados_cambiar_pagina_interna_ELTPARAGRAPHE000037"></a>

Los siguientes eventos se ejecutan al llamar a la función **ChangeSourcePage**:

- Evento "Antes de descargar la página interna". Este evento está asociado al control Página interna. 

- Evento "Cierre" de la anterior página interna.

- Evento "Declaraciones globales" de la nueva página interna.

- Evento "Inicialización" de los controles de la nueva página interna. 

- Evento "Inicialización" de la página interna.

- Evento "Después de cargar la página interna". Este evento está asociado al control Página interna. 



<a name="NOTE0_2"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000054"></a>

- Los nombres y parámetros de las funciones llamadas deben ser idénticos en ambas páginas. 

- Si se utiliza un control de la página interna directamente desde el exterior, debe haber un control con el mismo nombre en la nueva página interna.

- Los valores de las variables globales de la anterior página interna se pierden.



<a name="NOTE0_3"></a>




### Propiedad SourcePage
<a name="propiedad_sourcepage_ELTPARAGRAPHE000064"></a>

La propiedad [SourcePage](../Proprietes/1000021228.md) devuelve el nombre de la página interna que se muestra en un control Página interna.

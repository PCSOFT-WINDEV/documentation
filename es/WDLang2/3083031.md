


## LooperToWord (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un archivo de Word (.RTF) utilizando los datos de un archivo control Looper.

**Observación**: Este archivo puede abrirse en Word u OpenOffice Writer (dependiendo del software instalado en el ordenador actual).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
sFile is string
// File name
sFile = fSelect("", "", "Choose the export file", ...
	"Word files (*.rtf) *.rtf", "rtf", fSelCreate + fSelExist)
// The export is canceled if no name was specified
// Otherwise, export the content of the LOOP_Customer Looper control
IF sFile <> "" THEN LooperToWord(LOOP_Customer, sFile)
```
<a name="Example2"></a>

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperToWord(<Looper control> , <Doc file name> [, <Options> [, <Start row> [, <End row>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar. 

**`<Doc file name>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo Word a crear. El archivo se sustituye por el nuevo si ya existe.

**`<Options>: (Constante opcional de tipo Integer)`**

Configura el archivo de Word para crear.


|   |   |
| --- | --- |
| *taColumnsTitles*<br>(Valor predeterminado) | El título de las columnas se inserta en el archivo. |
| *taNoTitle* | Sólo se copian los datos en el archivo. |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |



**`<Start row>: (Entero opcional)`**

Número de la fila desde la que se iniciará la exportación. Si no se especifica este parámetro, el inicio fila será el primer fila encontrado en el control Looper.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. Si no se especifica este parámetro, el final fila es el último fila encontrado en el control Looper.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El archivo creado se puede abrir con Word 95 (o posterior) o con OpenOffice Writer 2 (o posterior).

- Esta función se puede utilizar en los controles de Looper basados en un archivo de datos o rellenarlos de forma programada.

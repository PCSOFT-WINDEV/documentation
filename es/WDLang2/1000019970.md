


## LooperCollapseExpand (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
En una determinada fila de un control Looper: 

- colapsa una rotura (si se expande),

- expande una ruptura (si se colapsa).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Expands on user click
LooperCollapseExpand(LOOP_Products)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LooperCollapseExpand(<Looper control> [, <Row number> [, <Break name>]])`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del fila antes de que se llame **LooperCollapseExpand**: 



|   |   |
| --- | --- |
| *taCollapse* | fila colapsada . |
|  *taError* | No se ha encontrado fila. |
| *taExpand* | fila ampliada . |



**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es una cadena vacía (""), se utiliza la control Looper actual.

**`<Row number>: (Entero opcional)`**

Número de la fila a ocultar o desglosar.
Si no se especifica este parámetro, se utiliza el fila actual.

**`<Break name>: (Cadena de caracteres opcional)`**

Nombre de la ruptura a modificar. Este parámetro debe especificarse en rupturas anidadas. Se utiliza para definir la ruptura que se manejará. Por la Default, ninguna ruptura se expande o colapsa si las rupturas están anidadas.




## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000083"></a>

La función **LooperCollapseExpand** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.

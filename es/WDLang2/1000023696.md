


## &lt;Looper&gt;.SavePositionAndSelection ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra:

- la posición de la barra de desplazamiento en una control Looper,

- los elementos seleccionados.


Estos elementos pueden ser restaurados por [&lt;Looper&gt;.RestorePositionAndSelection](../WDLang2/1000023525.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sSavePosition is string
sSavePosition = LOOP_Looper1.SavePositionAndSelection()
LOOP_Looper1.Display(taInit)
LOOP_Looper1.RestorePositionAndSelection(sSavePosition)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Looper control>.SavePositionAndSelection()`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [&lt;Looper&gt;.RestorePositionAndSelection](../WDLang2/1000023525.md).

- Cadena vacía ("") si la control Looper está vacía.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.


## ConnectionCount( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de instancias del sitio WEBDEV actualmente ejecutadas en el servidor.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info("There are " + ConnectionCount() + "connected Web users")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ConnectionCount()`
---

**`<Result>: (Integro)`**

Número de instancias del sitio WEBDEV que se ejecutan actualmente en el servidor.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**ConnectionCount** le permite realizar fácilmente o get lo siguiente:

- Pruebas de validez de la licencia, para sitios limitados a un número determinado de conexiones, o cobradas en función del número de conexiones.

- estadísticas para determinar el número máximo de personas conectadas a un sitio, por ejemplo.


Cada conexión puede identificarse a partir del identificador Process devuelto por [ExeGetPID](../WDLang1/1000017239.md).

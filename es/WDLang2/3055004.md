
## AddFavorite( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	 Esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función sólo gestiona los Favoritos de INTERNET Explorer. Desde el 15 de junio de 2022, la aplicación de escritorio INTERNET Explorer 11 ya no es compatible con algunas versiones de Windows 10. 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un INTERNET Address a la lista de favoritos del usuario de la web.

**Atención**: Esta operación funciona sólo en los navegadores INTERNET Explorer.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds the favorite
IF Position(BrowserName(), "Netscape") = 0 THEN
	AddFavorite("PC SOFT, WEBDEV editor", "www.windev.com")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AddFavorite(<Title> , <Internet address>)`
---

**`<Title>: (Cadena de caracteres)`**

Título del favorito visible en el menú "Favoritos" del Explorador de INTERNET.

**`<Internet address>: (Cadena de caracteres)`**

INTERNET Address del favorito. Esta dirección puede ser:

- una IP Address en formato http://xxx.xxx.xxx.xxx

- una URL (http://www.windev.com/webdev/ por ejemplo/)

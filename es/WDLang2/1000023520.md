


## &lt;ModifyLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia los valores de los atributos en una fila de una control Looper.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para los controles de Looper sin atributos, se recomienda cambiar directamente el valor de los controles correspondientes. Para obtener más información, consulte Modificar un control Looper sin atributos.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify a row in a Looper control
LOOP_Looper1.ModifyLine(Index, ProductName, ProductCode)
```
<a name="Example2"></a>

```wl
// Modifies a value in the Looper control at position 12
LOOP_Looper1.ModifyLine(12, LastName, FirstName, PhoneNum, FaxNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Looper control>.ModifyRow([<Index> [, <Value 1> [, <Value N>]]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Index>: (Entero opcional)`**

Índice de la fila a modificar.

- Si no se especifica este parámetro, la modificación se realizará en la unidad actual.

- Se muestra un error de WLanguage si <Índice> es mayor que el número de filas.




**`<Value 1>: (Tipo de atributo associated (opcional))`**

Valor del primer atributo de la fila que debe modificarse en la control Looper especificada.
Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Si no se especifica ninguno de estos parámetros, la función **&lt;ModifyLine** asigna una cadena vacía ("") o 0.

**`<Value N>: (Tipo de atributo associated (opcional))`**

Valor del atributo Nth del fila que debe modificarse en el control Looper especificado..
Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Si no se especifica ninguno de estos parámetros, la función **&lt;ModifyLine** asigna una cadena vacía ("") o 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000275"></a>

La función **&lt;ModifyLine** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Características de la fila
<a name="caracteristicas_fila_ELTPARAGRAPHE000304"></a>Si un valor corresponde al resultado de un cálculo, la expresión numérica debe figurar entre paréntesis.. Por ejemplo:

```wl
ZR_CalculationZone.ModifyLine(5, (53+29), (83-21))
```

<a name="NOTE0_3"></a>


### clasificar
<a name="clasificar_ELTPARAGRAPHE000320"></a>

Si **&lt;ModifyLine** modifica un atributo de clasificación del control Looper actual, se eliminará el orden de clasificación set con este atributo.
<a name="NOTE0_4"></a>


### Modificar un control Looper sin atributos
<a name="modificar_control_looper_sin_atributos_ELTPARAGRAPHE000332"></a>

Para modificar una control Looper sin atributos, se recomienda cambiar directamente el valor de los controles correspondientes.

Por ejemplo:


```wl
// Modify the value of controls
// in "LOOP_Looper1" for row 23
LOOP_Looper1[23].LastNameControl = "Hobbes"
LOOP_Looper1[23].FirstNameControl = "Samantha"
```




## <Page>.ClosePopup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una página que se muestra como una ventana emergente a través de [.DisplayPopup](../WDLang2/1410087260.md).<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Closes the page displayed as a popup
PAGE_ProductDetails.ClosePopup()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Page>.ClosePopup()`
---

**`<Page>: (Nombre de la página)`**

Nombre de la página que se muestra como una ventana emergente que se ocultará. Esta página debe haber sido visualizada por [.DisplayPopup](../WDLang2/1410087260.md). En caso contrario, **<Page>.ClosePopup** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [.DisplayPopup](../WDLang2/1410087260.md) y **<Page>.ClosePopup** deben utilizarse en diferentes proceos/eventos. Si estas funciones se utilizan en el mismo proceso/Event, la página Popup no se muestra.

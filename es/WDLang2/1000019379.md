
## LocalStorageAvailable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el almacenamiento local está disponible o no para el navegador actual.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF LocalStorageAvailable() = False THEN 
	Info("Your browser does not support the browser contexts")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LocalStorageAvailable()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el almacenamiento local está disponible,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

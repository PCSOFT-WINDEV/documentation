


## &lt;ToExcel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un fichero Excel a partir de los datos encontrados en una control Looper (hojeando control Looper o control Looper de la memoria).

**Observación**: Este archivo puede ser abierto por Excel o por OpenOffice Calc (dependiendo del software instalado en el ordenador actual).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
sFile is string
// File name
sFile = fSelect("", "", "Choose the export file", ...
	"Excel files (*.xls) *.xls", "xls", fSelCreate + fSelExist)
// The export is canceled if no name was specified
// Otherwise, export the content of the LOOP_Customer Looper control
IF sFile <> "" THEN 
	LOOP_Customer.ToExcel(sFile)
END
```

## Sintaxis

`<Result> = <Looper control>.ToExcel(<Name of XLS file> [, <Options> [, <Start row> [, <End row>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Name of XLS file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo Excel a crear. El archivo se sustituye por el nuevo si ya existe.

**`<Options>: (Constante opcional de tipo Integer)`**

Configura el archivo XLS para crear.


|   |   |
| --- | --- |
| *taColumnsTitles*<br>(valor predeterminado) | El nombre de los atributos se utiliza en el fichero Excel como título de las columnas. |
| *taMerge* | Si el archivo XLS ya existe, fusiona el contenido existente con el nuevo contenido: las fórmulas, el formato, ... de las células existentes se mantiene. Los datos existentes no se almacenan. Sólo se conservan los datos de las columnas y filas encontradas en el archivo XLS existente. |
| *taNoTitle* | Sólo se copian los datos en el archivo. El nombre de los atributos no se especifica en el título de columna del fichero Excel. |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |
| *taWithLayout* | Exporta la color y la fuente utilizada en la control. |



**`<Start row>: (Entero opcional)`**

Número de la fila desde la que se iniciará la exportación. 
Si no se especifica este parámetro, el inicio fila será el primer fila encontrado en el control Looper.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. 
Si no se especifica este parámetro, el final fila es el último fila encontrado en el control Looper.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los archivos de Excel no pueden exceder las 65536 filas y 256 columnas (lo que significa 65536 filas que contienen 256 controles cada una).. Este límite puede superarse utilizando la extensión ".xlsx" para crear archivos en formato Excel 2007. Atención: en este caso, las versiones anteriores de Excel deben estar equipadas con el "Paquete de compatibilidad Office 2007" que puede descargarse gratuitamente del sitio de Microsoft para poder abrir los documentos generados.

- Sólo se exportan los primeros 256 caracteres de cada control.

- El archivo creado puede ser abierto por Excel 97 (o posterior) o por OpenOffice Calc 2 (o posterior).

- Las columnas "Numéricas de alta precisión" (38 dígitos significativos) se exportarán como número exponencial.

- Las columnas Unicode se exportan en formato Unicode a Excel.

- Las columnas de casillas de verificación se exportan como casillas de verificación.

- Los siguientes elementos no se exportan al fichero Excel: 

	- Las imágenes, 

	- Los gráficos,

	- Los códigos de barras.




## &lt;Image&gt;.last (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el último Image para el pergamino.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the last banner
IMG_Banner.Last()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Image control>.Last()`
---

**`<Image control>: (Nombre del control)`**

Nombre del control a utilizar:

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.

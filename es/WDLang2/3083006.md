


## LooperAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una fila al final de una control Looper.








	<B><B><B>Advertencia</B></B></B>

	 Se recomienda utilizar la función [LooperAddLine](../WDLang2/3083004.md). [LooperAddLine](../WDLang2/3083004.md) le permite añadir:

- TAB en los atributos/controles.

- todos los tipos de valores en los atributos/controles.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the Looper control named LOOP_LOOPER1 
LooperDeleteAll(LOOP_LOOPER1) 
HReadFirst(Product, ProdName) 
WHILE HOut() = False
	LooperAdd(LOOP_LOOPER1, Product.ProdName + TAB + Product.ProductCode) 
	HReadNext(Product, ProdName) 
END
```
<a name="Example2"></a>

```wl
// Add an empty row 
LooperAdd(LOOP_Looper1) 
// Position on the empty row 
Index = LooperCount(LOOP_Looper1) 
// Then, initialize the attributes 
ATT_AttributeLastName[Index] = "Hobbes"
ATT_AttributeFirstName[Index] = "Samantha"

// Or initialize the controls
// LOOP_Looper1[Index].ControlLastName = "Hobbes"
// LOOP_Looper1[Index].ControlFirstName = "Samantha"
```

<a name="SYNTAX1"></a>

## Sintaxis

### Añadir un fila a un control Looper con atributos

`<Result> = LooperAdd(<Looper control> [, <Value>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la adición fue exitosa,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es una cadena vacía (""), la fila se añadirá en la control Looper actual .

**`<Value>: (Cadena de caracteres opcional)`**

Lista de valores que se insertarán en la nueva fila. Este parámetro debe tener el siguiente formato: 

```txt
<Value 1> + TAB + ... + <Value N>
```

Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Para no especificar ningún valor para uno de los atributos, utilice una cadena vacía ("") o 0 (según el tipo de atributo).
Se añadirá una fila vacía si este parámetro corresponde a una cadena vacía ("").

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000158"></a>

La función **LooperAdd** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Añadir una fila vacía
<a name="anadir_una_fila_vacia_ELTPARAGRAPHE000186"></a>

Después de añadir una fila vacía, cada atributo o control debe inicializarse individualmente.

Por ejemplo:

- Para una control Looper con atributos:
	
	```wl
	// Add an empty row
	LooperAdd(LOOP_Looper1)
	
	// Position on the empty row
	Index = LooperCount(LOOP_Looper1)
	// Then, initialize the attributes
	ATT_AttributeLastName[Index] = "Hobbes"
	ATT_AttributeFirstName[Index] = "Samantha"
	ATT_ColorAttributeFirstName[Index] = LightRed
	ATT_AttributeAddressVisible[Index] = False
	```


- Para una control Looper sin atributos: 
	
	```wl
	// Add an empty row
	LooperAdd(LOOP_Looper1)
	
	// Position on the empty row
	Index = LooperCount(LOOP_Looper1)
	// Then, initialize the controls
	LOOP_Looper1[Index].ControlLastName = "Hobbes"
	LOOP_Looper1[Index].ControlFirstName = "Samantha"
	LOOP_Looper1[Index].ControlFirstName.BackgroundColor = LightRed
	LOOP_Looper1[Index].ControlAddress.Visible = False
	```




<a name="NOTE0_3"></a>


### Características de la fila
<a name="caracteristicas_fila_ELTPARAGRAPHE000214"></a>Si un valor corresponde al resultado de un cálculo, la expresión numérica debe figurar entre paréntesis.. Por ejemplo:

```wl
LooperAdd(LOOP_CalcLooper, (53+29) + TAB + (83-21))
```

<a name="NOTE0_4"></a>


### Posición de la fila agregada
<a name="posicion_fila_agregada_ELTPARAGRAPHE000229"></a>

Por Default, el fila se añade al final del control Looper.

Si el control Looper está ordenado ([LooperSort](../WDLang2/3083009.md)), el fila añadido se posiciona en el control Looper según la ordenación actual.
<a name="NOTE0_5"></a>


### Manejo de un memo binario
<a name="manejo_memo_binario_ELTPARAGRAPHE000243"></a>

Para añadir una nota binaria en una control Looper, utilice [LooperAddLine](../WDLang2/3083004.md).

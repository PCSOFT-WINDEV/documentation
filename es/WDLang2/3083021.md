


## Funciones para la gestión de los controles de la miga de pan
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles de la miga de pan:



|   |   |
| --- | --- |
| [BreadcrumbAddLink](../WDLang2/3083016.md) | Añade un enlace a un control Migas de pan. |
| [BreadcrumbDeleteAll](../WDLang2/3083020.md) | Borra todos los enlaces de un control Migas de pan. |
| [BreadcrumbDeleteLink](../WDLang2/3083019.md) | Borra un enlace de un control Migas de pan. |
| [BreadcrumbInsertLink](../WDLang2/3083017.md) | Inserta un enlace en un control Migas de pan. |
| [BreadcrumbModifyLink](../WDLang2/3083018.md) | Modifica un enlace en un control Migas de pan. |

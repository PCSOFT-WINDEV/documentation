


## LooperSearchEverywhere (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un valor en todos los atributos de una control Looper. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click code of the Button control [Search]
nPos is int
nPos = LooperSearchEverywhere(LOOP_Order, EDT_Search, searchContains)
// If found
IF nPos > 0 THEN
	// Selects the row
	LOOP_Order = nPos
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LooperSearchEverywhere(<Looper control> , <Sought value> [, <Type of search> [, <Start>]])`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado,

- -1 si la búsqueda falla.




**`<Looper control>: (Nombre del control)`**

Nombre de la control Looper donde se realizará la búsqueda. La búsqueda se realiza en todos los atributos.

**`<Sought value>: (Cadena de caracteres)`**

Valor buscado en todos los atributos.

**`<Type of search>: (Constante opcional de tipo Integer)`**

Tipo de búsqueda que se realizará: 



|   |   |
| --- | --- |
| *searchContains*<br>(Valor predeterminado) | **Búsqueda genérica de tipo "Contiene"**: <br> corresponde al índice de la fila cuyos elementos tienen un valor que contiene . |
| *searchExactMatch* | **Búsqueda de coincidencia exacta cuyo tipo es "Igual a"**: <br> corresponde al índice de las filas cuyos elementos tienen un valor estrictamente igual a . |
| *searchStartsWith* | **Búsqueda genérica de tipo "Comienza por"**: <br> corresponde al índice de la fila cuyos elementos tienen un valor que comienza por . |



**`<Start>: (Entero opcional)`**

Índice de inicio de la búsqueda (1 de Default).


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000092"></a>

- La función **LooperSearchEverywhere** puede utilizarse en: 

	- a control Looper basada en un archivo de datos. Sin embargo, esta función no está disponible para el controles Tabla basado en un archivo de datos con acceso directo.  

	- a control Looper poblada programáticamente.




- El valor se busca entre los atributos que son relacionado a un valor de texto visible (por ejemplo, [Valor](../Proprietes/2510130.md) en un control Edit o [Pie de foto](../Proprietes/2510053.md) en un control Estático)

- La búsqueda no se realiza en los atributos ocultos.



<a name="NOTE0_2"></a>


### Sensibilidad a los casos
<a name="sensibilidad_los_casos_ELTPARAGRAPHE000114"></a>

La búsqueda no distingue entre mayúsculas y minúsculas, incluso si se trata de una búsqueda por coincidencia exacta.

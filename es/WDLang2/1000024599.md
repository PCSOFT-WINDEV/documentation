
## PageCurrentLayout (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del layout actual: 

- en la página en curso.

- en la página especificada.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF PageCurrentLayout() = 2 THEN
	STC_Details = Customer.Details
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificar el layout de la página actual

`<Result> = PageCurrentLayout()`
---

**`<Result>: (Entero)`**



- Índice del layout actual.

- 0 si la página no tiene ningún layout. 

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *layoutPrint* | Se está utilizando el diseño específico para la impresión. |
| *layoutWINDEV* | Se está utilizando la disposición específica para WINDEV. |







<a name="SYNTAX2"></a>

### Identificar el layout de una página específica

`<Result> = PageCurrentLayout(<Page>)`
---

**`<Result>: (Integro)`**



- Índice del layout actual.

- 0 si la página interna no tiene ningún layout. 

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *layoutPrint* | Se está utilizando el diseño específico para la impresión. |
| *layoutWINDEV* | Se está utilizando la disposición específica para WINDEV. |






**`<Page>: (Nombre de la página)`**

Nombre de la página manipulada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los layouts se numeran en el orden en el que se crearon.

- El layout cambia si la orientación del dispositivo cambia o si se ejecuta la función [PageChangeLayout](../WDLang2/1000024598.md).

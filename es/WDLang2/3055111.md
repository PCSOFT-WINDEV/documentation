
## AJAXAsynchronousCallPending( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Determina si se está ejecutando un servidor Procedure llamado por [AJAXExecuteAsynchronous](../WDLang2/3055115.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is AJAX supported by the current browser?
IF AJAXAvailable() = True THEN
	// Run the server procedure named "MyProc"
	// As soon as the result of the "MyProc" server procedure
	// will be available, 
	// the "ProcResult" browser procedure will be run
	Res is int
	Res = AJAXExecuteAsynchronous(MyProc, ProcResult)
	// The following processes are run without waiting for the result
	...
	// Progress of the server procedure named "MyProc"
	IF AJAXAsynchronousCallPending(Res) = True THEN
		// Cancel the execution 
		// of the "ProcResult" browser procedure
		// The result of the "MyProc" server procedure 
		// cannot be processed
		AJAXCancelAsynchronousCall(Res)
	END
ELSE
	// Processes without using AJAX
	... 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AJAXAsynchronousCallPending(<Identifier of server procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el procedimiento servidor especificado se está ejecutando,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Identifier of server procedure>: (Integro)`**

Identificador del servidor de Procedure para usar. Este identificador fue definido por [AJAXExecuteAsynchronous](../WDLang2/3055115.md).

<a name="NOTE0_1"></a>

## Observaciones


### Recordatorio
<a name="recordatorio_ELTPARAGRAPHE000045"></a>

[AJAXCancelAsynchronousCall](../WDLang2/3055110.md) se utiliza para cancelar la ejecución de automatic del procedimiento de navegación llamado por [AJAXExecuteAsynchronous](../WDLang2/3055115.md).
<a name="NOTE0_2"></a>


### Navegadores antiguos
<a name="navegadores_antiguos_ELTPARAGRAPHE000057"></a>

La función [AJAXAvailable](../WDLang2/3055112.md) permite determinar si el navegador actual soporta AJAX. Si un proceso que utiliza AJAX se ejecuta en un navegador que no es compatible con esta tecnología, el proceso se ejecuta "como si" no utilizara AJAX (por ejemplo, se actualiza toda la página).

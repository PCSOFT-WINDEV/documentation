
## ProgressBarDisable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la actualización periódica de una barra de progreso vía Ajax.

**Atención**: Esta función no puede ser usada en un sitio PHP o AWP. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stop the refresh of the Progress Bar
ProgressBarDisable(PROGBAR_Copy)
// Hide the Progress Bar displayed in a cell
CellCloseDialog(CELL_ProgBar)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ProgressBarDisable(<Progress Bar control>)`
---

**`<Progress Bar control>: (Nombre del control)`**

Nombre de la control Barra de progreso a utilizar



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función no tiene ningún efecto si no se activa actualización.

- Si la actualización se inició con la función [ProgresoBarEnable](../WDLang2/1000019477.md), la función **ProgressBarDisable** detiene las llamadas periódicas al servidor. Si un servidor Process está habilitado, su ejecución continúa.

- Si la actualización se inició con la función [ProgresoBarExecute](../WDLang2/1000019481.md), la función **ProgressBarDisable** detiene las llamadas periódicas a la función navegador. Por lo tanto, las últimas llamadas a la función no se ejecutan.

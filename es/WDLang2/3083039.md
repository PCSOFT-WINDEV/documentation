
## Funciones para la gestión de las migas de pan (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las rutas del mapa del sitio a través de la sintaxis del prefijo:



|   |   |
| --- | --- |
| [&lt;DeleteAll](../WDLang2/3083038.md) | Borra todos los enlaces de un control Migas de pan. |
| [&lt;DeleteLink](../WDLang2/3083037.md) | Borra un enlace de un control Migas de pan. |
| [&lt;Miga de pan&gt;.InsertLink](../WDLang2/3083035.md) | Inserta un enlace en un control Migas de pan. |
| [&lt;Migajas de pan&gt;.ModifyLink](../WDLang2/3083036.md) | Modifica un enlace en un control Migas de pan. |
| [&lt;Migas de pan&gt;.AddLink](../WDLang2/3083034.md) | Añade un enlace a un control Migas de pan. |

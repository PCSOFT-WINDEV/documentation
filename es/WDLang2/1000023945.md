


## Funciones control Looper (sintaxis prefijada)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar los controles del Looper (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;AddInProgress](../WDLang2/1000023482.md) | Modifica o devuelve el estado de visibilidad de la ventana de espera interna al añadir elementos adicionales en un Infinite control Looper. |
| [&lt;AddLine](../WDLang2/1000023475.md) | Añade una fila al final de control Looper. |
| [&lt;Borrar](../WDLang2/1000023701.md) | Borra una fila de una control Looper. |
| [&lt;Cancelar el deslizamiento](../WDLang2/1000023483.md) | Cancela la visualización de la ventana interna mostrada en un control Looper después del deslizamiento de fila. |
| [&lt;Clasificación](../WDLang2/1000023704.md) | Permite:<br><br>- Ordena una control Looper según uno o más atributos.<br><br>- Ordena un control Looper en todos sus atributos.<br><br>- Cancel una clase que se realizó de antemano (lo que significa ignorar la clase).<br><br><br> |
| [&lt;Colapso](../WDLang2/1000023497.md) | Contrae una ruptura previamente expandida en un control Looper. |
| [&lt;CollapseExpand](../WDLang2/1000023499.md) | En una determinada fila de un control Looper: <br><br>- colapsa una pausa (si está expandida),<br><br>- amplía una pausa (si está colapsada).<br><br><br> |
| [&lt;Eliminar todo](../WDLang2/1000023703.md) | Elimina todas las filas de una control Looper poblada de forma programática. |
| [&lt;EliminarSeleccionar](../WDLang2/1000023702.md) | Elimina las filas seleccionadas de una control Looper. |
| [&lt;EndFilling](../WDLang2/1000023501.md) | Finaliza la Process para rellenar una control Looper de forma programada. |
| [&lt;ExpandAll](../WDLang2/1000023490.md) | Expande todas las pausas en una control Looper con pausa. |
| [&lt;Expandir](../WDLang2/1000023489.md) | Amplía una pausa en la control Looper en una fila determinada. |
| [&lt;Guardar](../WDLang2/1000023496.md) | Valida la modificación o adición del Record associated con el fila actual en el control Looper. |
| [&lt;InfoXY](../WDLang2/1000023508.md) | Se utiliza para encontrar la fila o la control situada en una posición específica en la control Looper (X, Y). |
| [&lt;Insertar](../WDLang2/1000023509.md) | Inserta una fila en una control Looper. |
| [&lt;InsertIW](../WDLang2/1000023510.md) | Inserta una fila como ventana interna en una control Looper poblada programáticamente. |
| [&lt;Looper&gt;.AddIP](../WDLang2/1410089490.md) | Agrega una fila como página interna al final de un control Looper rellenado mediante programación. |
| [&lt;Looper&gt;.BreakIndex](../WDLang2/1000023502.md) | Devuelve el índice de la cabecera y el pie de página de una pausa dada en una control Looper. |
| [&lt;Looper&gt;.InsertIP](../WDLang2/1410089491.md) | Inserta una fila como página interna en un control Looper rellenado mediante programación. |
| [&lt;Looper&gt;.ModifyIP](../WDLang2/1410089492.md) | Modifica una fila que corresponde a una página interna en un control Looper rellenado mediante programación. |
| [&lt;Looper&gt;.RestorePositionAndSelection](../WDLang2/1000023525.md) | Recupera la posición de la barra de desplazamiento en un control Looper así como los elementos seleccionados. |
| [&lt;Looper&gt;.SavePositionAndSelection](../WDLang2/1000023696.md) | Ahorra:<br><br>- la posición de la barra de desplazamiento en un control Looper,<br><br>- el elemento o elementos seleccionados.<br><br><br>Estos elementos pueden ser restaurados por [&lt;Looper&gt;.RestorePositionAndSelection](../WDLang2/1000023525.md). |
| [&lt;Looper&gt;.SearchEverywhere](../WDLang2/1000023485.md) | Busca un valor en todos los atributos de una control Looper. |
| [&lt;Modificar](../WDLang2/1000023512.md) | Cambia los valores de los atributos en una fila de una control Looper. |
| [&lt;ModificarIW](../WDLang2/1000023519.md) | Modifica una fila correspondiente a una ventana interna de una control Looper poblada programáticamente. |
| [&lt;ModifyLine](../WDLang2/1000023520.md) | Cambia los valores de los atributos en una fila de una control Looper. |
| [&lt;Pantalla](../WDLang2/1000023451.md) | Actualiza a control Looper a partir de un archivo de datos desde una posición determinada: las modificaciones realizadas en el archivo relacionado se aplican al looper. |
| [&lt;Posición](../WDLang2/1000023523.md) | Mueve la barra de desplazamiento para mostrar una fila específica o devuelve el índice de la primera fila que se muestra en el control Looper. |
| [&lt;RecNum](../WDLang2/1000023521.md) | Devuelve el número de la Record actual en una control Looper basada en un archivo de datos. |
| [&lt;RefreshVisible](../WDLang2/1000023524.md) | Cambia o devuelve el estado de visibilidad de la barra actualización en un control Looper. |
| [&lt;Seleccione](../WDLang2/1000023697.md) | Devuelve el índice del elemento seleccionado en una control Looper. |
| [&lt;SelectCount](../WDLang2/1000023699.md) | Devuelve el número de filas seleccionadas en una control Looper. |
| [&lt;SelectMinus](../WDLang2/1000023698.md) | Deselecciona una fila en una control Looper. |
| [&lt;SelectPlus](../WDLang2/1000023700.md) | Selecciona un fila en un control Looper. |
| [&lt;StartFilling](../WDLang2/1000023487.md) | Inicia la Process para rellenar una control Looper de forma programada. |
| [&lt;ToExcel](../WDLang2/1000023705.md) | Crea un fichero Excel a partir de los datos encontrados en una control Looper (hojeando control Looper o control Looper de la memoria). |
| [&lt;ToWord](../WDLang2/1000023706.md) | Crea un archivo de Word (.RTF) utilizando los datos de un archivo control Looper. |
| [&lt;ToXML](../WDLang2/1000023707.md) | Crea un archivo XML con los datos de una control Looper (control Looper basada en un archivo de datos o rellenada mediante programación). |
| [.Add](../WDLang2/1000023469.md) | Añade una fila al final de una control Looper. |
| [.AddIW](../WDLang2/1000023474.md) | Agrega una fila como ventana interna al final de un control Looper rellenado mediante programación. |
| [.CollapseAll](../WDLang2/1000023500.md) | Colapsa todas las roturas expandidas en una control Looper. |
| [.DisplayPopup](../WDLang2/1000023481.md) | Muestra la ventana emergente de zoom automático para el control Galería de imágenes. |

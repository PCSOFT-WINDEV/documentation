
## BrowserSlice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número del rango de resolución que se muestra en una página en modo Responsive Web Design.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the "Tablet" slice is displayed
IF BrowserSlice() = 2 THEN
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserSlice()`
---

**`<Result>: (Integro)`**

Número de rango de resolución actual.

Observación: Si la página no está usando el modo **Responsive Web Design** , &lt;Resultado&gt; es set a 1.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El número de Default rango de resolución es 1. 

- Entonces, el número de rango de resolución aumenta para cada rango de resolución siguiente:

	- Si "Escritorio" es el Default rango de resolución, los números aumentan desde el rango de resolución más grande ("Escritorio") hasta el rango de resolución más pequeño ("Móvil").

	- Si "Móvil" es el Default rango de resolución, los números aumentan desde el rango de resolución más pequeño ("Móvil") hasta el rango de resolución más grande ("Escritorio").

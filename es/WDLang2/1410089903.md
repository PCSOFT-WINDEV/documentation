


## Funciones remotas control del servidor de aplicaciones WEBDEV (sintaxis de prefijos)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar usuarios y conexiones en un Servidor de Aplicaciones WEBDEV, utilizando la sintaxis de prefijos:




|   |   |
| --- | --- |
| [&lt;Variable WBAdminAuth&gt;.CreateAccount](../WDLang2/1410089712.md) | Crea una cuenta WEBDEV en un WEBDEV Application Server. |
| [&lt;Variable WBAdminAuth&gt;.DeleteAccount](../WDLang2/1410089717.md) | Elimina una cuenta WEBDEV de un WEBDEV Application Server. |
| [&lt;Variable WBAdminAuth&gt;.Disconnect](../WDLang2/1410089713.md) | Elimina una conexión de WEBDEV Application Server. |
| [&lt;Variable WBAdminAuth&gt;.ListAccount](../WDLang2/1410089714.md) | Devuelve la lista de cuentas WEBDEV de WEBDEV Application Server. |
| [&lt;Variable WBAdminAuth&gt;.ListConnection](../WDLang2/1410089715.md) | Devuelve la lista de conexiones actuales en WEBDEV Application Server. |
| [&lt;Variable WBAdminAuth&gt;.UpdateAccount](../WDLang2/1410089716.md) | Actualiza los parámetros de una cuenta WEBDEV en WEBDEV Application Server. |

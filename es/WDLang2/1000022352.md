
## AppServerListScheduledTask( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de tareas programadas para la aplicación (sitio o servicio web).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FOR EACH AScheduledTask of AppServerListScheduledTask()
	Trace(AScheduledTask.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppServerListScheduledTask()`
---

**`<Result>: (Array de las variables de la appServerScheduledTask)`**

Array de las variables de [appServerScheduledTask](../WDLang2/1000022367.md) correspondientes a las tareas programadas de la aplicación (sitio o Webservice).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si la aplicación (sitio o servicio web) no tiene tareas programadas, la función devuelve una array vacía.

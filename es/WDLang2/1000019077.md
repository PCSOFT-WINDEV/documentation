


## Funciones de carga (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la carga de archivos (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;Arranque](../WDLang2/1000019063.md) | Inicia el envío de los archivos seleccionados a una control Upload. |
| [&lt;CurrentFileSize](../WDLang2/1000019076.md) | Devuelve el tamaño total (en bytes) del archivo cargado actualmente a través de un control Upload. |
| [&lt;CurrentFileSizeSizeSent](../WDLang2/1000019074.md) | Devuelve el tamaño (en bytes) ya enviado para el archivo actualmente cargado a través de un control Upload. |
| [&lt;Subir&gt;.FileName](../WDLang2/1000019064.md) | Devuelve el nombre de un archivo "cargado" por el usuario web.. Este upload puede haberse realizado: <br><br>- a través de un"Upload" control Campo de entrada. <br><br>- a través de un solo archivo o de varios archivos control Upload.<br><br><br> |
| [&lt;Tamaño](../WDLang2/1000019070.md) | Devuelve el tamaño total (en bytes) del archivo cargado actualmente a través de un control Upload. |
| [&lt;Tamaño del archivo](../WDLang2/1000019075.md) | Devuelve el tamaño total (en bytes) de un archivo encontrado en una control Upload. |
| [&lt;Upload&gt;.CopyFile](../WDLang2/1000019046.md) | Saves on the server a file "uploaded" by the Web user. This upload may have been performed: <br><br>- via an "Upload" edit control. <br><br>- via a single-file or multi-file Upload control.<br><br><br> |
| [&lt;Upload&gt;.CurrentFile](../WDLang2/1000019047.md) | Indica el archivo cargado actualmente a través de control Upload. |
| [&lt;Upload&gt;.Delete](../WDLang2/1000019065.md) | Elimina un archivo de la lista de archivos a cargar: el archivo no se cargará en el servidor. |
| [&lt;Upload&gt;.DeleteAll](../WDLang2/1000019069.md) | Borra la lista de archivos a cargar: no se cargará ningún archivo en el servidor. |
| [&lt;Upload&gt;.SizeSent](../WDLang2/1000019071.md) | Devuelve el tamaño total (en bytes) de los archivos ya enviados por el upload actual a través de un control Upload. |

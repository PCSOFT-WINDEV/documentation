


## UploadSizeSizeSent (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño total (en bytes) de los archivos ya enviados por el upload actual a través de un control Upload. Para una control Upload multiarchivo, este tamaño corresponde a la suma de los tamaños individuales de cada archivo ya enviado con el tamaño enviado del archivo actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Event "Progress of transfer"
// Displays the size sent in KB
STC_SizeSent = Round(UploadSizeSent(MySelf) / 1024, 0) + "KB"
```
<a name="Example2"></a>

```wl
// -- Event "Progress of transfer"
// Displays the number of bytes sent according to the total size to send
STC_UploadMulti_Global = Round(UploadSizeSent(MySelf) / UploadSize(MySelf) * 100) + "%"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UploadSizeSent(<Upload control>)`
---

**`<Result>: (Integro)`**

Tamaño de los archivos ya enviados (en bytes).

**`<Upload control>: (Nombre del control)`**

Nombre de la control Upload a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ¿En qué Event debo usar esta función?
<a name="¿en_que_event_debo_usar_esta_funcion_ELTPARAGRAPHE000059"></a>

**UploadSizeSizeSent** sólo puede utilizarse en el navegador Event "Progreso de la transferencia". Este Event es associated: 

- al botón "Enviar" para un archivo único control Upload. 

- al botón "Añadir" para un control Upload de varios archivos.


Utilizado en otro Event, **UploadSizeSizeSent** devolverá 0.
<a name="NOTE0_2"></a>


### Tamaño máximo de los archivos a cargar
<a name="tamano_maximo_los_archivos_cargar_ELTPARAGRAPHE000206"></a>

**By Default,** el tamaño de los archivos a subir no está limitado en el motor WEBDEV. Este tamaño puede ser limitado por el servidor de aplicaciones WEBDEV. 
El archivo no se cargará si su tamaño excede el tamaño límite.
Para modificar el tamaño máximo de los archivos a cargar en WEBDEV servidor de aplicaciones, el registro debe ser modificado en el servidor:  

1. Para modificar el tamaño de los archivos cargados para todas las aplicaciones WEBDEV 28, seleccione la siguiente tecla: HKEY_LOCAL_MACHINE\\SOFTWARE\\PC SOFT\\WEBDEV\\28.0 en un ordenador de 32 bits y HKEY_LOCAL_MACHINE\\SOFTWARE\\WOW6432Node\\PC SOFT\\WEBDEV\\28.0 en un ordenador de 64 bits.

2. Añada la cadena MAX_UPLOAD. El valor de esta cadena será el tamaño máximo de los archivos a cargar (en KB).

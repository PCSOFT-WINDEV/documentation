


## &lt;Eliminar todo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las filas de una control Looper poblada de forma programática.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reset
LOOP_Looper1.DeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Looper control>.DeleteAll()`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000215"></a>

**&lt;Eliminar todo** solo se puede usar en un control Looper completado mediante programación.

## Componente:
wd280obj.dll

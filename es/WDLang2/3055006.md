
## BrowserOpen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una nueva ventana del navegador.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
//Opens a new window
BrowserOpen("http://www.windev.com")
```


<a name="Example2"></a>



```wl
// Displays a page in new resizable browser
// whose dimensions are 500x400 
PageName is string = FolderWeb() + "page.html"
BrowserOpen(PageName, ...
	"NEWBRW", ONDimension + ONScrollBar, 500, 400)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BrowserOpen(<Page address> [, <Window name> [, <Options> [, <Width> [, <Height> [, <Horizontal position> [, <Vertical position> [, <JavaScript parameters>]]]]]]])`
---

**`<Result>: (Integro)`**

Identificador de la ventana del navegador abierta. Este identificador puede ser utilizado por [BrowserClose](../WDLang2/3055007.md) para cerrar esta ventana del navegador.

Observación: Este identificador corresponde a un objeto JavaScript que sólo puede utilizarse con [BrowserClose](../WDLang2/3055007.md). Su valor no puede ser visto como un entero estándar.

**`<Page address>: (Cadena de caracteres)`**

URL Address de la página que debe mostrarse en la nueva ventana del navegador. Si este parámetro corresponde a una cadena vacía ("""), se abre una página en blanco ("about:blank").

**`<Window name>: (Cadena de caracteres opcional)`**

Nombre de la nueva ventana del navegador. Este parámetro permite volver a mostrar una página en una ventana del navegador con el mismo nombre (si se abren varias ventanas del navegador en el equipo del usuario de la web).

Este parámetro también puede corresponder a un marco o al alias de un iFrame control.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Parámetros de la nueva ventana del navegador:


|   |   |
| --- | --- |
| *ONFull*<br>(Valor predeterminado) | La ventana del nuevo navegador incluirá todas las opciones (equivalentes a la combinación de todas las constantes). |
| *ONLink* | Se mostrará la barra de enlace. |
| *ONLocation* | Se mostrará la barra de Address. |
| *ONMenuBar* | Aparecerá la barra de menú. |
| *ONResizable* | La ventana del nuevo navegador será redimensionable. |
| *ONSatusBar* | Aparecerá la barra de estado. |
| *ONScrollBar* | Aparecerán las barras de desplazamiento. |
| *ONSimple* | La ventana del nuevo navegador será una ventana simple (sin combinación de constantes). |
| *ONToolBar* | Aparecerá la barra de herramientas. |



**`<Width>: (Entero opcional)`**

Ancho (en píxeles) de la nueva ventana del navegador. Puede tomar como valor la constante *Default*.

**`<Height>: (Entero opcional)`**

Altura en píxeles de la nueva ventana del navegador. Puede tomar como valor la constante *Default*.

**`<Horizontal position>: (Entero opcional)`**

Posición horizontal (en píxeles) de la nueva ventana del navegador con respecto a la Corner superior izquierda de la pantalla principal. Puede tomar como valor la constante *Default*.

**`<Vertical position>: (Entero opcional)`**

Posición vertical en píxeles de la nueva ventana del navegador en relación con la parte superior izquierda de la pantalla Corner. Puede tomar como valor la constante *Default*.

**`<JavaScript parameters>: (Cadena de caracteres opcional)`**

Otros parámetros de JavaScript que deben utilizarse al abrir un nuevo navegador si &lt;Target&gt; es set a "_blank".

## Observaciones




### Mostrar un sitio dentro de un marco
<a name="mostrar_sitio_dentro_marco_ELTPARAGRAPHE000101"></a>Para mostrar un sitio INTERNET en un marco de un sitio WEBDEV, utilice el siguiente código:


```txt
BrowserOpen("http://www.windev.com", <FRAMENAME>)
```


donde &lt;FRAMENAME&gt; es el nombre del marco donde se debe mostrar el sitio.

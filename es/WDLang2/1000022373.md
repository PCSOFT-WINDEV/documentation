


## Funciones Google Analytics
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para enviar datos a Google Analytics: 



|   |   |
| --- | --- |
| [GglAnalyticsAddEvent](../WDLang2/1000022356.md) | Añade una Event en Google Analytics |
| [GglAnalyticsAddException](../WDLang2/1000022360.md) | Añade una excepción además de Google Analytics. |
| [GglAnalyticsAddPage](../WDLang2/1000022357.md) | Añade una Page junto a Google Analytics. |
| [GglAnalyticsAddSocialNetworkAction](../WDLang2/1000022358.md) | Añade una acción de red social además de Google Analytics. |
| [GglAnalyticsAddTiming](../WDLang2/1000022359.md) | Añade una duración al lado de Google Analytics. |

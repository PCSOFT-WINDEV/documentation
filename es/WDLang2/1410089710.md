
## Controlar WEBDEV Application Server
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
A partir de la versión 28, puede controlar WEBDEV Application Server de forma remota. Esta novedad permite automatizar acciones, hacer que estas acciones estén disponibles mediante una UI personalizada y crear herramientas de seguimiento y estadísticas personalizadas.

Esto puede hacerse desde una aplicación WINDEV o un sitio WEBDEV.

Las siguientes son algunas de las acciones que pueden automatizarse:

- Administrar cuentas WEBDEV: crear, eliminar y listar cuentas...

- Administrar conexiones: crear una lista de conexiones, desconectar usuarios, obtener el número de usuarios conectados, etc.

- ...

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000150"></a>


### Administrar cuentas WEBDEV 
<a name="administrar_cuentas_webdev_ELTPARAGRAPHE000024"></a>

Para administrar cuentas WEBDEV de forma remota: 

1. Defina una variable de tipo [WBAdminAuth](../WDLang2/1410089328.md). Esta variable permite definir las características de la API utilizada para administrar WEBDEV Application Server de forma remota.   

2. Utilice una variable de tipo [WBAdminAccount](../WDLang2/1410089382.md).

	- Para crear una cuenta, defina todas las características de la variable [WBAdminAccount](../WDLang2/1410089382.md) y, a continuación, utilice la función [WBAdminCreateAccount](../WDLang2/1410089450.md). 

	- Para actualizar o eliminar una cuenta, identifique la variable [WBAdminAccount](../WDLang2/1410089382.md) que corresponde a la cuenta deseada utilizando la función [WBAdminListAccount](../WDLang2/1410089449.md) y, a continuación, utilice las funciones [WBAdminUpdateAccount](../WDLang2/1410089471.md) y [WBAdminDeleteAccount](../WDLang2/1410089472.md). 







Atención: Para agregar cuentas WEBDEV, la cuenta Windows utilizada debe ser administrador del servidor. 


### Administrar conexiones
<a name="administrar_conexiones_ELTPARAGRAPHE000061"></a>

Para administrar las conexiones de forma remota: 

1. Defina una variable de tipo [WBAdminAuth](../WDLang2/1410089328.md). Esta variable permite definir las características de la API utilizada para administrar WEBDEV Application Server de forma remota.   

2. Utilice una variable de tipo [WBAdminConnection](../WDLang2/1410089344.md). La lista de conexiones activas puede consultarse con la función [WBAdminListConnection](../WDLang2/1410089342.md). Puede eliminar una conexión utilizando la función [WBAdminDisconnect](../WDLang2/1410089343.md).

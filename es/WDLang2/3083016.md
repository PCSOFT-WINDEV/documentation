


## BreadcrumbAddLink (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un enlace a un control Migas de pan.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
BreadcrumbAddLink(SMP_MyBreadcrumbControl, "Help", "Help_Page", "Help")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`BreadcrumbAddLink(<Breadcrumb control> , <Caption> , <URL or page name> [, <Tooltip>])`
---

**`<Breadcrumb control>: (Nombre del control)`**

Nombre de la control Migas de pan a la que se añadirá el enlace.

**`<Caption>: (Cadena de caracteres)`**

Leyenda del enlace.

**`<URL or page name>: (Cadena de caracteres)`**

URL associated con el enlace o nombre lógico de la página WEBDEV para abrir.
**Atención**: La URL de un sitio externo debe comenzar necesariamente con "http://".. 

**`<Tooltip>: (Cadena de caracteres opcional)`**

description del enlace, mostrada en un tooltip cuando el ratón pasa por encima del enlace.

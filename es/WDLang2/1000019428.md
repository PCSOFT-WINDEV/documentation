


## ImageStopScrolling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene el desplazamiento de imágenes de automatic.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops the scroll
ImageStopScrolling(IMG_Banner)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ImageStopScrolling(<Image control>)`
---

**`<Image control>: (Nombre del control)`**

Nombre del control a utilizar:

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.

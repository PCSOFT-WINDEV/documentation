


## &lt;RecNum (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la Record actual en una control Looper basada en un archivo de datos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of the current record in "LOOP_ProductLooper"
ResRecNum = LOOP_ProductLooper.RecNum()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Looper control>.RecNum()`
---

**`<Result>: (Integro)`**



- El número actual de Record en el archivo de datos relacionado al control Looper, 

- 0 si no hay ningún fila actual o si el fila actual no está guardado todavía en el archivo de datos relacionado al control Looper.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### control Looper basado en una consulta
<a name="control_looper_basado_una_consulta_ELTPARAGRAPHE000182"></a>

Si la consulta se ejecutó con la constante *hWithFilter* (opción marcada en el editor de consultas), **&lt;RecNum** devuelve el número del Record actual en la consulta.

Si la consulta se ejecutó sin la constante *hWithFilter* , **&lt;RecNum** devuelve un resultado indefinido.

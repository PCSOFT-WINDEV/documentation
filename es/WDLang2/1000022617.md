


## Funciones Banner deslizante
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del Sliding Banner: 



|   |   |
| --- | --- |
| [BannerFirst](../WDLang2/1000022513.md) | Muestra el primer plano de un Banner deslizante control. |
| [BannerLast](../WDLang2/1000022516.md) | Muestra el último plano de un Banner deslizante control. |
| [BannerNext](../WDLang2/1000022514.md) | Muestra el siguiente plano de un Banner deslizante control. |
| [BannerPrevious](../WDLang2/1000022515.md) | Muestra el plano anterior de un Banner Deslizante control. |
| [BannerStartSliding](../WDLang2/1000022511.md) | Permite el deslizamiento de automatic en un banner deslizante control. |
| [BannerStopSliding](../WDLang2/1000022512.md) | Detiene el deslizamiento de automatic en una pancarta deslizante control. |

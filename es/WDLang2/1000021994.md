
## DynamicServingConfigure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el entorno que utilizará la "Publicación Dinámica" para elegir el conjunto de páginas que se mostrarán.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Menu option
// Always display the Computer version of pages
DynamicServingConfigure(dsDesktop)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DynamicServingConfigure(<Environment>)`
---

**`<Environment>: (Constante de tipo Integer)`**

Tipo de ambiente a utilizar: 


|   |   |
| --- | --- |
| *dsAutomatic* | La elección de las páginas a mostrar es realizada automáticamente por WEBDEV en función de la plataforma actual.. |
| *dsDesktop* | Las páginas en modo "Ordenador" son obligatorias. |
| *dsMobile* | Las páginas en modo "Móvil" son obligatorias. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La llamada a **DynamicServingConfigure** desencadena una llamada a [PageUse](../WDLang2/3058016.md) para todas las páginas actualmente abiertas: 

- que son associated con servicio dinámico,

- y cuyo modo de visualización no corresponde al seleccionado (incluida la página actual).

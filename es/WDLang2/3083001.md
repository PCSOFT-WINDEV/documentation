


## LooperDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza a control Looper a partir de un archivo de datos desde una posición determinada: las modificaciones realizadas en el archivo de datos de relacionado se aplican al control Looper.

**Observación**: **LooperDisplay** se puede usar en controles Looper basados en un archivo de datos o se puede completar mediante programación.












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refresh "LOOP_ProductLooper" from the start of the control 
LooperDisplay(LOOP_ProductLooper, taStart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LooperDisplay(<Looper control> [, <Position>])`
---

**`<Looper control>: (Nombre del control)`**

Nombre de la control Looper a actualización.
Si este parámetro corresponde a una cadena vacía (""), se actualiza la control Looper a la que pertenece la Event actual.

**`<Position>: (Entero, carácter o constante (opcional))`**

Indica la posición de inicio para la actualización. Si no se especifica este parámetro, los elementos de la control Looper se vuelven a visualizar a partir del elemento actual de la control Looper.

Este parámetro puede corresponder a:

- Un entero. La actualización se realiza a partir de esta posición. No se modifica la fila seleccionada. El evento "Visualización de una fila" de &lt;Posición&gt; se ejecuta. El elemento actual en el control Looper es el mismo que el registro actual en el archivo de datos. Si este entero no es válido (porque está fuera de los límites autorizados), se produce un error al llamar a la función **LooperDisplay**.

- Un carácter o una constante:
	


|   |   |
| --- | --- |
| *taCurrentFirst* | <br><br>	- Actualiza el control Looper del actual Record en el archivo de datos.<br><br>	- Se selecciona la primera fila de la control Looper. Esta fila corresponde al registro actual de la control Looper.<br><br><br> |
| *taCurrentSelection* | <br><br>	- Actualiza el control Looper desde la posición de banda de selección en el control Looper.<br><br>	- La posición de la banda de selección no se modifica. El registro actual en el archivo de datos corresponderá al registro seleccionado en el control Looper (después de llamar a la función **LooperDisplay**).<br><br><br> |
| *taInit* | <br><br>	- Ejecuta los códigos de inicialización y de fin de inicialización del control Looper.<br><br>	- Llena la control Looper.<br><br>	- Se selecciona la primera fila de la control Looper. Este fila corresponde al Record actual al final del código de inicialización del control Looper.<br><br><br> |
| *taNext* | <br><br>	- Vuelve a mostrar la control Looper y apunta al principio de la control Looper en la siguiente Page (lo mismo que utilizar la r Page para acceder a la siguiente Page).<br><br>	- El primer Visible fila del control Looper es seleccionado. Esta fila corresponde al registro actual de la control Looper.<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *taPrevious* | <br><br>	- Actualiza el control Looper y apunta al principio del control Looper en el Page anterior (lo mismo que utilizar el Page r para acceder al Page anterior ).<br><br>	- El primer Visible fila del control Looper es seleccionado. Esta fila corresponde al registro actual de la control Looper.<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *taReExecuteQuery* | <br><br>	- Reejecutar la consulta relacionado a la control Looper. Para obtener más información, consulte [Tablas y consultas](../WDChamp/1013203.md).<br><br>	- Actualiza el control Looper y fija la posición al principio del control.<br><br>	- Se selecciona la primera fila de la control Looper. Esta fila corresponde al primer registro de la control Looper.<br><br><br>**Atención**: Sólo se ejecuta la consulta relacionado al control Looper. Si se han especificado opciones específicas con [HExecuteQuery](../WDLang4/3044080.md) ( *hModifyFile* , por ejemplo), estas opciones se ignoran. |
| *taStart*<br>(Valor predeterminado) | <br><br>	- Actualiza el control Looper y fija la posición al principio del control.<br><br>	- Se selecciona la primera fila de la control Looper. Esta fila corresponde al primer registro de la control Looper.<br>		Observación: En el caso de una control Looper rellenada por programa, el primer elemento no se selecciona. <br><br><br> |

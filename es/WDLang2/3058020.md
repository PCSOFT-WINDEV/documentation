


## PageSubmit (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Valida la página especificada e inicia la ejecución de un botón.. Sólo se ejecuta el código del servidor del botón.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Browser click code of BTN_Close button
// Validates the page and starts running the server click code of BTN_Button1
PageSubmit("", "BTN_Button1")
```
<a name="Example2"></a>

```wl
// -- Browser click code on an image
// Run the click event of a SUBMIT button 
// in order for the content of page controls to be sent to the server

// The BTN_FAMILY_CHOICE button opens a page in a new simplified browser
PageSubmit("", "BTN_FAMILY_CHOICE", NewBrowser, "Family Choice", ONSimple)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`PáginaEnviar(<Page> , <Button> [, <Target> [, <Window of the browser> [, <Options> [, <Width> [, <Height> [, <Horizontal position> [, <Vertical position> [, <Javascript parameters>]]]]]]]])`
---

**`<Page>: (Page name)`**



- Nombre de la página a validar,

- Vaciar la cadena (""") para validar la página actual. 


La página a validar debe ser una de las páginas que actualmente se muestran en el navegador.

**`<Button>: (Control name)`**

Nombre de la control Botón cuyo código de clic del servidor se ejecutará. Este control debe pertenecer a la página especificada.

**`<Target>: (Optional character string or constant)`**

Nombre de la trama de destino. &lt;Destino&gt; también puede tomar los siguientes valores:


|   |   |
| --- | --- |
| *ParentFrame* | el objetivo es el contenedor de la página actual (conjunto de marcos padre, navegador padre). <br>Este parámetro también puede corresponder a la cadena "_parent" (compatibilidad con WEBDEV 1.5). |
| *CurrentFrame* | el objetivo es el cuadro actual (valor de Default). <br>Este parámetro también puede corresponder a la cadena "_self" (compatibilidad con WEBDEV 1.5). |
| *CurrentBrowser* | el objetivo es el navegador actual. <br>Este parámetro también puede corresponder a la cadena "_top" (compatibilidad con WEBDEV 1.5). |
| *NewBrowser* | el objetivo es un nuevo navegador (se abre una nueva ventana del navegador). Los siguientes parámetros de la función **PageSubmit** permiten configurar esta nueva ventana. <br>Este parámetro también puede corresponder a la cadena "_blank" (compatibilidad con WEBDEV 1.5). |



**`<Window of the browser>: (: Optional character string (with quotes))`**

Nombre de la ventana en el nuevo navegador si &lt;Destino&gt; es igual a la constante *NewBrowser*. Este parámetro le permite volver a mostrar una página en un navegador con el mismo nombre (si se abren varios navegadores en el ordenador del usuario Web).

**`<Options>: (Optional Integer constant (or combination of constants))`**

Parámetros de la ventana en el nuevo navegador si &lt;Destino&gt; es igual a la constante *NewBrowser*:


|   |   |
| --- | --- |
| *ONLocation* | Se mostrará la barra de Address. |
| *ONScrollBar* | Aparecerán las barras de desplazamiento. |
| *ONFull* <br>(Valor predeterminado) | La ventana del nuevo navegador incluirá todas las opciones (equivalente a la combinación de todas las constantes). |
| *ONResizable* | La ventana del nuevo navegador será redimensionable. |
| *ONLink* | Se mostrará la barra de enlace. |
| *ONMenuBar* | Aparecerá la barra de menú. |
| *ONSatusBar* | Aparecerá la barra de estado. |
| *ONToolBar* | Aparecerá la barra de herramientas. |
| *ONSimple* | La ventana del nuevo navegador será una ventana simple (sin combinación de constantes). |



**`<Width>: (Optional integer)`**

Ancho de ventana en el nuevo navegador (en píxeles).

**`<Height>: (Optional integer)`**

Altura de la ventana en el nuevo navegador (en píxeles).

**`<Horizontal position>: (Optional integer)`**

Posición horizontal (en píxeles) de la ventana en el nuevo navegador (en relación con el Corner superior izquierdo de la pantalla).

**`<Vertical position>: (Optional integer)`**

Posición vertical (en píxeles) de la ventana en el nuevo navegador (en relación con el Corner superior izquierdo de la pantalla).

**`<Javascript parameters>: (Optional character string (with quotes))`**

Otros parámetros de Javascript que deben ser usados al abrir un nuevo navegador si &lt;Target&gt; es igual a la constante *NewBrowser*.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Atención: se puede realizar una sola acción a la vez en la misma página
<a name="atencion_puede_realizar_una_sola_accion_vez_misma_pagina_ELTPARAGRAPHE000119"></a>

La función **PageSubmit** provoca una acción en la página que se muestra en el navegador del usuario. Sin embargo, los navegadores de Internet permiten una sola acción en la página por Process.

Por ejemplo, si un Process envía una página al servidor (**PageSubmit**) y abre el software de mensajería ([EmailOpenMail](../WDLang2/3032021.md)), sólo se realizará la última acción (abrir el software de mensajería).

**Atención**: Algunos tipos de botones desencadenan automáticamente una acción en la página (botón"enviar al servidor", botón"otro" cuya acción difiere de la de"ninguno").. Si se utiliza la función **PageSubmit** en el proceso de clic navegador de estos botones (o en un procedimiento llamado por este proceso), esta función se ignorará: sólo se ejecutará la acción del botón.

**Consecuencias**: **PageSubmit** no se puede utilizar en un código que desencadena una acción (automáticamente o no). Esta función se ignora:

- En el código de clic de un botón "enviar".

- En el código de clic de un botón "otro" cuya acción es diferente de "ninguno".

- Si va seguido de **PageSubmit** o [EmailOpenMail](../WDLang2/3032021.md) (en el Process actual, en un procedimiento llamado por el Process actual o en otro Process llamado por [Ejecución](../WDLang1/3013041.md)).



<a name="NOTE0_2"></a>


### Uso de PageSubmit de un control Botón
<a name="uso_docparampagetitleshort_control_boton_ELTPARAGRAPHE000155"></a>

Para usar el **PageSubmit** de un control Botón (código de clic del navegador del control) este control debe tener las siguientes características:

- Tipo de botón: "Otro".

- Acción: "Ninguna".



<a name="NOTE0_3"></a>


### El nombre de la control Botón corresponde a un Variable o una cuerda construida. 
<a name="nombre_control_boton_corresponde_variable_una_cuerda_construida_ELTPARAGRAPHE000169"></a>

Si el nombre de la control Botón corresponde a un Variable o a una cadena construida (por ejemplo: CONTROL"+nControlNumber"), no se puede realizar ninguna verificación durante la compilación.. Cualquier error provocará un error de JavaScript o un error HTTP al utilizar la función.

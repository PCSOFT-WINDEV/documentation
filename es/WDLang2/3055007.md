
## BrowserClose( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la ventana del navegador actual y detiene la ejecución del código actual.. También tiene la posibilidad de cerrar un navegador específico.

Casos especiales:

- **Internet Explorer**: Por razones de seguridad, el navegador solicita una autorización antes de cerrar.. Si el usuario responde "No" a la solicitud de cierre enviada por el navegador, el navegador no se cierra pero se detiene el código que se está ejecutando actualmente.

- **Firefox**: Por razones de seguridad, Firefox se niega a cerrar el navegador si éste no fue abierto por [NavegadorAbrir](../WDLang2/3055006.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
BrowserClose()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cerrando el navegador actual

`BrowserClose()`
---


<a name="SYNTAX2"></a>

### Cerrar el navegador identificado por un identificador

`BrowserClose(<Identifier>)`
---

**`<Identifier>: (Integro)`**

Identificador de la ventana del navegador para cerrar. Este identificador es devuelto por [NavegadorAbrir](../WDLang2/3055006.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Detener el sitio WEBDEV
<a name="detener_sitio_webdev_ELTPARAGRAPHE000047"></a>

**BrowserClose** sólo cierra la ventana actual del navegador: esta función no se puede utilizar para detener un sitio WEBDEV. Para detener un sitio WEBDEV, utilice [EndProgram](../WDLang1/3013033.md).

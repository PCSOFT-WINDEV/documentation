


## <Looper>.DisplayPopup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana emergente de zoom automático para el control Galería de imágenes. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF LOOP_Gallery[LOOP_Gallery].ATT_HighQualityImage THEN
	LOOP_Gallery.DisplayPopup(LOOP_Gallery)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Looper control>.DisplayPopup([<Subscript>])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper en modo Galería a utilizar.

**`<Subscript>: (Entero opcional)`**

Índice de elemento para acercar el control Looper. Este parámetro corresponde a la actual índice de Default.

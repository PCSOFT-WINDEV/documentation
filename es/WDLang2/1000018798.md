


## UploadCurrentFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el archivo que se está cargando actualmente a través de control Upload. Esta función es useful para controles de carga de múltiples archivos. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the name of file currently uploaded
STC_UploadMulti_File = UPL_UploadMulti[UploadCurrentFile(MySelf)]
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UploadCurrentFile(<Upload control>)`
---

**`<Result>: (Integro)`**

Índice del archivo actualmente cargado a través del control Upload: 1 para el primer archivo de la lista, 2 para el segundo, ....

**`<Upload control>: (Cadena de caracteres)`**

Nombre de la control Upload a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ¿En qué evento se debe utilizar la función **UploadCurrentFile**?
<a name="¿en_que_evento_debe_utilizar_funcion_docparampagetitleshort_ELTPARAGRAPHE000048"></a>

**UploadCurrentFile** solo se puede usar en el navegador Event "Progreso de la transferencia". Este Event es associated: 

- al botón "Enviar" para un expediente único control Upload. 

- al botón "Añadir" para un expediente múltiple control Upload. 




Usado en otro Event , **UploadCurrentFile** devolverá 0. 
<a name="NOTE0_2"></a>


### Tamaño máximo de los archivos a cargar
<a name="tamano_maximo_los_archivos_cargar_ELTPARAGRAPHE000197"></a>

**By Default,** el tamaño de los archivos a subir no está limitado en el motor WEBDEV. Este tamaño puede ser limitado por el servidor de aplicaciones WEBDEV. 
El archivo no se cargará si su tamaño excede el tamaño límite.
Para modificar el tamaño máximo de los archivos a cargar en WEBDEV servidor de aplicaciones, el registro debe ser modificado en el servidor:  

1. Para modificar el tamaño de los archivos cargados para todas las aplicaciones WEBDEV 28, seleccione la siguiente tecla: HKEY_LOCAL_MACHINE\\SOFTWARE\\PC SOFT\\WEBDEV\\28.0 en un ordenador de 32 bits y HKEY_LOCAL_MACHINE\\SOFTWARE\\WOW6432Node\\PC SOFT\\WEBDEV\\28.0 en un ordenador de 64 bits.

2. Añada la cadena MAX_UPLOAD. El valor de esta cadena será el tamaño máximo de los archivos a cargar (en KB).

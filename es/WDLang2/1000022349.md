
## AppServerDeleteScheduledTask( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una tarea programada associated con la aplicación (sitio o Webservice).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
aScheduledTask is appServerScheduledTask
aScheduledTask.Name = "MyTask" 
// Tries to delete the scheduled task whose name is "MyTask"
AppServerDeleteScheduledTask(aScheduledTask)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppServerDeleteScheduledTask(<Scheduled task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la tarea programada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Scheduled task>: (Tipo associated)`**

Tarea programada a eliminar. Este parámetro puede corresponder a: 

- una variable de tipo [appServerScheduledTask](../WDLang2/1000022367.md). 

- un nombre de Procedure. La tarea programada asociada al nombre de esta Procedure será eliminada. 

- el nombre de una tarea programada. Se eliminará la tarea programada correspondiente.

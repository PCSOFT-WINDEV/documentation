
## BrowserOS( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el sistema operativo instalado en el navegador del usuario de la web.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Proposes the application in the "store" corresponding to the OS
SWITCH BrowserOS()
	CASE browserOSiOS
		// ...
	CASE browserOSAndroid
		// ...
	CASE browserOSWindowsMobile
		// ...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserOS()`
---

**`<Result>: (Constante de tipo Integer)`**

Sistema operativo del usuario de la web : 



|   |   |
| --- | --- |
| *browserOSAndroid* | El navegador se inició desde un sistema Android. |
| *browserOSiOS* | El navegador se inició desde un sistema iOS. |
| *browserOSLinux* | El navegador se inició desde un sistema Linux. |
| *browserOSMacOS* | El navegador se inició desde un sistema MacOS. |
| *browserOSOther* | El navegador se inició desde otro sistema. |
| *browserOSWindows* | El navegador se inició desde un sistema Windows. |
| *browserOSWindowsMobile* | El navegador se inició desde un sistema Windows para móviles (por ejemplo: Windows CE, Windows 10 Mobile, etc.).. |

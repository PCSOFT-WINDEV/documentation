


## LooperCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de filas de una control Looper.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of rows found in the "LOOP_Looper1" Looper control?
Result1 is int
Result1 = LooperCount(LOOP_Looper1)
```
<a name="Example2"></a>

```wl
// Displays the number of answers found: (ex: "There are 31 answers")
STC_CAPTION1 = "There are " + LooperCount(LOOP_ResultLooper) + " answers"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperCount(<Looper control> [, <Row/Attribute>])`
---

**`<Result>: (Integro)`**



- Número de filas encontradas en una control Looper,

- 0 si la control Looper está vacía.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), el recuento se realizará sobre la control Looper actual .

**`<Row/Attribute>: (Constante opcional)`**




|   |   |
| --- | --- |
| *toColumn* | Número de atributos que se encuentran en la control Looper. |
| *toShown*<br> | Número de filas de Visible no vacías.<br><br><br> |
| *toTotal*<br>(Valor predeterminado)<br> | **control Looper basado en un archivo de datos**:<br><br>- Número total de registros:<br><br>	- en el archivo de datos de relacionado (si no hay filtro). <br><br>	- filtrado en el archivo de datos de relacionado (si hay un filtro).<br><br>	- en la consulta (si control Looper es relacionado a una consulta).<br><br><br><br><br>- Número de registros de Visible.<br><br><br>**Atención**: el "Fin de la inicialización" Event se ejecuta una vez calculado el total.<br><br>**control Looper completado mediante programación**:<br>Número total de filas añadidas por [LooperAdd](../WDLang2/3083006.md), [LooperAddLine](../WDLang2/3083004.md), [LooperInsert](../WDLang2/3083003.md) o [LooperInsertLine](../WDLang2/3083002.md). |
| *toVisible* | Número de filas de Visible (incluyendo las filas vacías o las filas parcialmente mostradas). Corresponde al número máximo de filas que se pueden visualizar en el control Looper.<br><br><br> |

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000192"></a>

La función **LooperCount** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.


## BrowserHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la altura (en píxeles) del área de cliente donde se muestra la página.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If there is not enough available size.
IF BrowserHeight() < 500 THEN
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserHeight()`
---

**`<Result>: (Integro)`**

Altura del área de clientes en píxeles.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta altura incluye las posibles barras de desplazamiento.

- Para una página mostrada en un "iFrame" control, la altura devuelta es la correspondiente a la dimensión del iFrame control.




<a name="XComponent"></a>

## Componente:
WDJS.DLL

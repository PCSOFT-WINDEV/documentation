


## &lt;Variable WBAdminAuth&gt;.CreateAccount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una cuenta WEBDEV en un WEBDEV Application Server.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
wbAuthLogin is WBAdminAuth
wbAuthLogin.ServerAddress = "https://my-server.internal/"
wbAuthLogin.User = "WWW\DATA"
wbAuthLogin.Password = "www"

WBAccount is WBAdminAccount
WBAccount.User = "USER_X"
WBAccount.VirtualServer = WBAccount.User

ResultCreate is boolean = wbAuthLogin.CreateAccount(WBAccount, "USERXPASSWORD")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <WEBDEV Application Server>.CreateAccount(<Account> , <Password>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la cuenta,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WEBDEV Application Server>: (Variable de tipo WBAdminAuth)`**

Nombre de la variable de tipo [WBAdminAuth](../WDLang2/1410089328.md) que describe el servidor y la cuenta de administrador utilizada para consultar a WEBDEV Application Server. 
Atención: Para agregar cuentas WEBDEV, la cuenta Windows utilizada debe ser administrador del servidor.

**`<Account>: (Variable de tipo WBAdminAccount)`**

Nombre de la variable de tipo [WBAdminAccount](../WDLang2/1410089382.md) que contiene la descripción de la cuenta que se va a crear.

**`<Password>: (Cadena de caracteres)`**

Contraseña de la cuenta a crear.




## &lt;Página&gt;.Refresh (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza una página mostrada en el navegador del usuario a partir del contexto en el servidor. Esta función se utiliza principalmente para los sitios en el modo Sesión. 

**Atención**: El contexto de la página debe existir en el servidor. El contexto de la página es creado por [&lt;Página&gt;.display](../WDLang2/1000021661.md) y [ContextoAbrir](../WDLang2/3058004.md). Todos los eventos relacionado a la página o a los controles ya han sido ejecutados. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refresh PAGE_Page1
PAGE_Page1.Refresh()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Page>.Refresh()`
---

**`<Page>: (Nombre de la página)`**

Nombre de la página de actualización.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Utilización de &lt;Página&gt;.Refresh en un Event en modo AJAX
<a name="utilizacion_docparampagetitleshort_event_modo_ajax_ELTPARAGRAPHE000190"></a>

**Recordatorio**: AJAX se utiliza para actualización los datos modificados en una página HTML sin volver a mostrar toda la página. Para obtener más información, consulte [AJAX](../WDLang2/3055101.md).

Si **&lt;Página&gt;.Refresh** se utiliza en un Event en modo AJAX (se muestra "AJAX habilitado" en la barra de Event) y si este Event se utiliza para actualización única o sistemáticamente una página, la tecnología AJAX se vuelve inútil. En el modo AJAX, **&lt;Página&gt;.Refresh** debe reservarse para casos especiales.

Durante una Process larga, [AJAXExecuteAsynchronous](../WDLang2/3055115.md) permite mostrar un mensaje de espera.


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000218"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".




## LooperSavePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra:

- la posición de la barra de desplazamiento en una control Looper,

- los elementos seleccionados.


Estos elementos pueden ser restaurados por [LooperRestorePositionAndSelection](../WDLang2/1000021789.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sSavePosition is string
sSavePosition = LooperSavePositionAndSelection(LOOP_Looper1)
LooperDisplay(LOOP_Looper1, taInit)
LooperRestorePositionAndSelection(LOOP_Looper1, sSavePosition)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LooperSavePositionAndSelection(<Looper control>)`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [LooperRestorePositionAndSelection](../WDLang2/1000021789.md).

- Cadena vacía ("") si la control Looper está vacía.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se utiliza la control Looper a la que pertenece la Event actual.

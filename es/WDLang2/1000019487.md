
## BrowserType( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tipo de navegador utilizado por el usuario Web.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a specific page for IE9 
IF BrowserType() = BrowserTypeInternetExplorer THEN
	SWITCH BrowserVersion(BrowserMajorVersion)
	CASE 9
		PageDisplay(PAGE_Form_IE9)
	OTHER CASE
		PageDisplay(PAGE_Form_IE)
	END
ELSE
	PageDisplay(PAGE_Form)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserType()`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de navegador utilizado por el usuario para visualizar el sitio web: 



|   |   |
| --- | --- |
| *BrowserTypeChrome* | Cromo. |
| *BrowserTypeEdge* | Microsoft Edge. |
| *BrowserTypeFirefox* | Firefox. |
| *BrowserTypeGecko* | Navegador que usa el motor Gecko (aparte de Firefox). |
| *BrowserTypeInternetExplorer* | Internet Explorer. |
| *BrowserTypeOpera* | Opera. |
| *BrowserTypeRobot* | Robot de motor de búsqueda. |
| *BrowserTypeSafari* | Safari. |
| *BrowserTypeUnknown* | El navegador utilizado es desconocido o falta la información. |
| *BrowserTypeWebKit* | Navegador que usa el motor WebKit (aparte de Chrome y Safari). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función devuelve la constante *BrowserTypeUnknown* si el navegador utilizado por el usuario de la Web es desconocido.

- En el caso de un sitio dinámico (sitio no AWP), el valor devuelto corresponde al navegador utilizado al conectarse al sitio. Si el usuario cambia de navegador (a través de una copy-pegar la URL en otro navegador por ejemplo), este cambio no se detecta.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

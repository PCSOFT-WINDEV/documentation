


## &lt;InsertLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una fila en una control Looper.

**Observación**: Para añadir elementos al final de control Looper, utilice [&lt;AddLine](../WDLang2/1000023475.md).










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add values into the Looper control at position 12
LOOP_Looper1.InsertLine(12, LastName, FirstName, PhoneNum, FaxNum)
```
<a name="Example2"></a>

```wl
// Insert searched information into the database
i is int = 1
LOOP_LOOPER1.DeleteAll()
HReadSeek(Product, ProdName, "Window")
WHILE HOut() = False
	LOOP_LOOPER1.InsertLine(i, Product.ProdName, Product.Code)
	HReadNext(Product, ProdName)
	i ++
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un fila en un control Looper con atributos

`<Looper control>.InsertLine([<Subscript> [, <Value 1> [, <Value N>]]])`
---

**`<Looper control>: (Control name)`**

Nombre del control Looper a utilizar.

**`<Subscript>: (Optional integer)`**

Índice del fila a partir del cual se insertará el nuevo fila. 



|   |   |
| --- | --- |
| Índice no especificado | La fila se inserta antes de la fila seleccionada. Si no se selecciona ningún fila, el fila se insertará en la última posición del control Looper. |
| Índice mayor que el número de filas de control | La fila se insertará en la última posición de la control Looper. En este caso la función **&lt;InsertLine** equivale a la función [&lt;AddLine](../WDLang2/1000023475.md). |
| Índice igual a 0 | La fila se insertará en la primera posición de la control Looper. |
| Índice igual a -1 | La fila se inserta antes de la fila seleccionada. Si no se selecciona ningún fila, el fila se insertará en la última posición del control Looper. |



**`<Value 1>: (Type of associated attribute (optional))`**

Valor del 1er atributo para que el fila se inserte en el control Looper especificado.
Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Para no especificar ningún valor para uno de los atributos, utilice una cadena vacía ("") o 0 (según el tipo de atributo).
Si no se especifica ninguno de estos parámetros, se insertará una fila vacía en la control Looper.

**`<Value N>: (Type of associated attribute (optional))`**

Valor del atributo N para que el fila se inserte en el control Looper especificado.
Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Para no especificar ningún valor para uno de los atributos, utilice una cadena vacía ("") o 0 (según el tipo de atributo).
Si no se especifica ninguno de estos parámetros, se insertará una fila vacía en la control Looper.

### Insertar un fila en un control Looper sin atributos

`<Looper control>.InsertLine([<Subscript>])`
---

**`<Looper control>: (Control name)`**

Nombre del control Looper a utilizar.

**`<Subscript>: (Optional integer)`**

Índice del fila a partir del cual se insertará el nuevo fila.

- Si este parámetro no se especifica o es igual a -1, el fila se insertará antes del fila actual . Si no se selecciona ninguna fila, la fila se insertará al final del control Looper.

- Si &lt;Index&gt; es mayor que el número de filas, la fila se insertará en la última posición del control Looper. En este caso la función **&lt;InsertLine** equivale a la función [&lt;AddLine](../WDLang2/1000023475.md).

- Si este parámetro es 0, la fila se insertará en la primera posición del control Looper.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000333"></a>

La función **&lt;InsertLine** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Añadir una fila vacía
<a name="anadir_una_fila_vacia_ELTPARAGRAPHE000362"></a>

Una vez añadida una fila vacía, hay que inicializar cada atributo o cada control uno por uno.

Por ejemplo:

- Para una control Looper con atributos:
	
	```wl
	// Add an empty row
	LOOP_LOOPER1.InsertLine(14)
	
	// Then, initialize the attributes
	AttributeLastName[14] = "Hobbes"
	AttributeFirstName[14] = "Samantha"
	AttributeFirstNameColor[14] = LightRed
	AttributeAddressVisible[14] = False
	```


- Para una control Looper sin atributos: 
	
	```wl
	// Add an empty row
	LOOP_LOOPER1.InsertLine(14)
	
	// Then, initialize the controls
	LOOP_Looper1[14].LastNameControl = "Hobbes"
	LOOP_Looper1[14].FirstNameControl = "Samantha"
	LOOP_Looper1[14].ControlFirstName.BackgroundColor = LightRed
	LOOP_Looper1[14].ControlAddress.Visible = False
	```




<a name="NOTE0_3"></a>


### Características de la fila
<a name="caracteristicas_fila_ELTPARAGRAPHE000391"></a>

Si un valor corresponde al resultado de un cálculo, la expresión numérica debe figurar entre paréntesis.. 

Por ejemplo:

```wl
LOOP_CalculationZone.InsertLine(10, (53+29), (83-21))
```

<a name="NOTE0_4"></a>


### clasificar
<a name="clasificar_ELTPARAGRAPHE000409"></a>

La función **&lt;InsertLine** no tiene en cuenta el orden.




## BannerStartSliding (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite el deslizamiento de automatic en un banner deslizante control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start sliding
BannerStartSliding(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BannerStartSliding(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Nombre del control)`**

Nombre de la pancarta deslizante control para usar.

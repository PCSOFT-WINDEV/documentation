


## BannerNext (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el siguiente plano de un Banner deslizante control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the next plane
BannerNext(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BannerNext(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Nombre del control)`**

Nombre de la pancarta deslizante control para usar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los aviones Invisible del Sliding Banner control son ignorados.

- Si el plano actual es el último plano visible, la función **BannerNext** muestra el primer plano visible.

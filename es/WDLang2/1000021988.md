


## LooperCancelSwipe (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la visualización de la ventana interna mostrada en un control Looper después del deslizamiento de fila.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code of cancellation button in the internal window
LooperCancelSwipe(LOOP_CustomerLooper)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LooperCancelSwipe(<Looper control>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Puede gestionar el deslizamiento de una fila en un control Looper. La lata de refrescos:

- No realizar ninguna acción.

- Ejecutar código.

- Borrar automáticamente el elemento deslizado.

- Mostrar una ventana interna para personalizar la acción.




Para obtener más información, consulte [Looper: desplazamientos y gestos](../WDChamp/1013295.md).

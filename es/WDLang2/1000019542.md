
## Constantes del navegador
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **BrowserFullVersion** | Número de versión completa del navegador utilizado por el usuario web. | [versión de navegador](../WDLang2/1000019488.md) |
| **BrowserMajorVersion** | Número de versión principal del navegador utilizado por el usuario web. | [versión de navegador](../WDLang2/1000019488.md) |
| **browserOSAndroid** | El navegador se inició desde un sistema Android. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSiOS** | El navegador se inició desde un sistema iOS. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSLinux** | El navegador se inició desde un sistema Linux. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSMacOS** | El navegador se inició desde un sistema MacOS. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSOther** | El navegador se inició desde otro sistema. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSWindows** | El navegador se inició desde un sistema Windows. | [BrowserOS](../WDLang2/1000021857.md) |
| **browserOSWindowsMobile** | El navegador se inició desde un sistema Windows para móviles. | [BrowserOS](../WDLang2/1000021857.md) |
| **BrowserPlatformAndroid** | El usuario muestra el sitio en un Smartphone que funciona con Android. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformBlackBerry** | El usuario muestra el sitio en un BlackBerry. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformiPad** | El usuario muestra el sitio en un iPad. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformiPhone** | El usuario muestra el sitio en un iPhone. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformMobile** | El usuario muestra el sitio en un Smartphone (distinto de los mencionados). | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformRobot** | El usuario es un robot de un motor de búsqueda (y por lo tanto indexa el sitio). | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformStandard** | El usuario no utiliza una plataforma específica (PCs, ordenadores portátiles, Mac,...). | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformWindowsMobile** | El usuario muestra el sitio en un Smartphone con Windows Mobile. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserPlatformWindowsPhone** | El usuario muestra el sitio en una SmartPhone que funciona con Windows Phone. | [NavegadorPlataforma](../WDLang2/1000019486.md) |
| **BrowserTypeChrome** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Cromo | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeEdge** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Microsoft Edge | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeFirefox** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Firefox | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeGecko** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Navegador que usa el motor Gecko (aparte de Firefox). | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeInternetExplorer** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Internet Explorer. | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeOpera** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Opera | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeRobot** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Robot de motor de búsqueda. | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeSafari** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Safari | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeUnknown** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: El navegador utilizado es desconocido o falta información | [Tipo de navegador](../WDLang2/1000019487.md) |
| **BrowserTypeWebKit** | Tipo de navegador utilizado por el usuario para visualizar el sitio web: Navegador que usa el motor WebKit (aparte de Chrome y Safari). | [Tipo de navegador](../WDLang2/1000019487.md) |
| **cookieHTTPOnly** | Tipo de Cookie: Cookie que sólo se puede recuperar en el código del servidor. Esta Cookie no puede ser leída en el código del navegador. | [CookieWrite](../WDLang2/3012008.md) |
| **cookieNormal** | Tipo de Cookie: Cookie sin opciones específicas. | [CookieWrite](../WDLang2/3012008.md) |
| **cookieSameSiteStrict** | Tipo de Cookie: SameSite Cookie. Este tipo de Cookie se utiliza para prevenir los ataques de Falsificación de Peticiones en Sitios Cruzados (CSRF). | [CookieWrite](../WDLang2/3012008.md) |
| **cookieSecure** | Tipo de Cookie: Cookie transmitido sólo en una petición HTTPS. | [CookieWrite](../WDLang2/3012008.md) |

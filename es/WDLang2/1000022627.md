


## &lt;Banner deslizante&gt;.anterior (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el plano anterior de un Banner Deslizante control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Return to previous plane
BAN_Banner.Previous()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Sliding Banner control>.Previous()`
---

**`<Sliding Banner control>: (Nombre del control)`**

Nombre de la pancarta deslizante control para usar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los aviones Invisible del Sliding Banner control son ignorados.

- Si el plano actual es el primer plano visible, la función **&lt;Banner deslizante&gt;.anterior** muestra el último plano visible.

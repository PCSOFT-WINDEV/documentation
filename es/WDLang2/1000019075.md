


## &lt;Tamaño del archivo (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño total (en bytes) de un archivo encontrado en una control Upload. 

**Observación**: Esta función se puede utilizar incluso si no hay ningún archivo cargado actualmente..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the total file size in KB
STC_TotalFileSize = Round(UPL_UploadControl.FileSize(1) / 1024, 0) + "KB"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Upload control>.FileSize([<File index>])`
---

**`<Result>: (Integro)`**

Tamaño total (en bytes) del archivo encontrado en el archivo control Upload.

**`<Upload control>: (Nombre del control)`**

Nombre de la control Upload utilizada.

**`<File index>: (Entero opcional)`**

Índice del fichero (que se encuentra en el control Upload) a gestionar. El valor predeterminado de este parámetro es 1.




## LooperRecNum (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la Record actual en una control Looper basada en un archivo de datos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of the current record in "LOOP_ProductLooper"
ResRecNum = LooperRecNum(LOOP_ProductLooper)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperRecNum(<Looper control>)`
---

**`<Result>: (Integro)`**



- El número actual de Record en el archivo de datos relacionado al control Looper, 

- 0 si no hay ningún fila actual o si el fila actual no está guardado todavía en el archivo de datos relacionado al control Looper.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), el control Looper utilizado es el control Looper al que pertenece el Event actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### control Looper basado en una consulta
<a name="control_looper_basado_una_consulta_ELTPARAGRAPHE000063"></a>

Si la consulta se ejecutó con la constante *hWithFilter* (opción marcada en el editor de consultas), **LooperRecNum** devuelve el número del Record actual en la consulta.

Si la consulta se ejecutó sin la constante *hWithFilter* , **LooperRecNum** devuelve un resultado indefinido.

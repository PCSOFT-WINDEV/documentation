
## FullScreenDisable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva el modo "Pantalla completa" del navegador. 

Observación: Para activar el modo "Pantalla completa", utilice [FullScreenEnable](../WDLang2/1000021012.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FullScreenDisable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`FullScreenDisable()`

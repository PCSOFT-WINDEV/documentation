


## &lt;Imagen&gt;.ScrollingPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el pergamino del Image especificado o devuelve el nombre del Image mostrado en el pergamino.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the next image (equivalent to ImageName.Next())
IMG_Banner.ScrollingPosition(IMG_Banner.ScrollingPosition() + 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Mostrando un pergamino de un Image especificado

`<Image control>.ScrollingPosition(<Subscript>)`
---

**`<Image control>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.




**`<Subscript>: (Integro)`**

Índice de la Image a mostrar. Si este parámetro es mayor que el número de imágenes a desplazar (devuelto por [&lt;Imagen&gt;.Occurrence](../WDLang2/1000023750.md)), la primera imagen es mostrada por Default.


<a name="SYNTAX2"></a>

### Averiguar el índice de la Image que aparece en un rollo

`<Result> = <Image control>.ScrollingPosition()`
---

**`<Result>: (Integro)`**

Índice de la Image que se muestra actualmente.

**`<Image control>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se utiliza en un código de servidor, **&lt;Imagen&gt;.ScrollingPosition** devuelve el índice del Image actual sólo si el código de servidor ha sido ejecutado por el "Enviar valores al servidor (submit)" control.




## Funciones de carga
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el upload de archivos:



|   |   |
| --- | --- |
| [UploadCopyFile](../WDLang2/3012023.md) | Guarda en el servidor un archivo "cargado" por el usuario Web. Este upload puede haberse realizado: <br><br>- a través de un"Upload" control Campo de entrada. <br><br>- a través de un solo archivo o de varios archivos control Upload.<br><br><br> |
| [UploadCurrentFile](../WDLang2/1000018798.md) | Indica el archivo cargado actualmente a través de control Upload. |
| [UploadCurrentFileSize](../WDLang2/1000018797.md) | Devuelve el tamaño total (en bytes) del archivo cargado actualmente a través de un control Upload. |
| [UploadCurrentFileSizeSizeSent](../WDLang2/1000018796.md) | Devuelve el tamaño (en bytes) ya enviado para el archivo actualmente cargado a través de un control Upload. |
| [UploadDelete](../WDLang2/1000018808.md) | Elimina un archivo de la lista de archivos a cargar: el archivo no se cargará en el servidor. |
| [UploadDeleteAll](../WDLang2/1000018809.md) | Borra la lista de archivos a cargar: no se cargará ningún archivo en el servidor. |
| [UploadFileName](../WDLang2/3012022.md) | Devuelve el nombre de un fichero "subido" por el usuario Web. Este upload puede haberse realizado: <br><br>- a través de un "Upload" control Campo de entrada. <br><br>- a través de un solo archivo o de varios archivos control Upload.<br><br><br> |
| [UploadFileSize](../WDLang2/1000019035.md) | Devuelve el tamaño total (en bytes) de un archivo encontrado en una control Upload. |
| [UploadSize](../WDLang2/1000018795.md) | Devuelve el tamaño total (en bytes) del archivo cargado actualmente a través de un control Upload. |
| [UploadSizeSizeSent](../WDLang2/1000018794.md) | Devuelve el tamaño total (en bytes) de los archivos ya enviados por el upload actual a través de un control Upload. |
| [UploadStart](../WDLang2/1000018793.md) | Inicia el envío de los archivos seleccionados a una control Upload. |

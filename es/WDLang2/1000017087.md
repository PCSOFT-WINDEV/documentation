


## PageActivateDDW( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la OVI (Dim Disabled Window) al mostrar una página modal (las páginas utilizadas para comunicarse con el usuario, por ejemplo).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PageActivateDDW(True)
PageRateDDW(40)
YesNo("Do you want to exit?")
PageActivateDDW(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PageActivateDDW(<Activation of DDW>)`
---

**`<Result>: (booleano)`**

Estado de activación previa de OVI:

- <u><u><u><u>True</u></u></u></u>: OVI fue habilitado para las páginas del sitio.

- <u><u><u><u>False</u></u></u></u>: la OVI no está habilitada para las páginas del sitio.




**`<Activation of DDW>: (booleano)`**

Nuevo estado de activación de OVI:

- <u><u><u><u>True</u></u></u></u>: OVI está habilitado para las páginas del sitio.

- <u><u><u><u>False</u></u></u></u>: OVI está desactivado para las páginas del sitio.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Tasa de Gris**: La tasa de gris por defecto para las páginas es set a 60%. Para cambiar esta tasa, utilice [PáginaTasaDDDDWW](../WDLang2/1000017086.md).

- El valor utilizado en el código del navegador se inicializa por el valor dado en el código del servidor. Las modificaciones realizadas en el código del navegador no se transfieren al código del servidor.




## ImagePrevious (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el anterior Image para el pergamino.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Goes back to the previous banner
ImagePrevious(IMG_Banner)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ImagePrevious(<Image control>)`
---

**`<Image control>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.

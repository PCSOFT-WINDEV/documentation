


## LooperDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las filas de una control Looper poblada de forma programática.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reset
LooperDeleteAll(LOOP_Looper1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LooperDeleteAll(<Looper control>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es igual a una cadena vacía (""), el borrado se realizará en la control Looper actual .



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000096"></a>

**LooperDeleteAll** solo se puede usar en un control Looper completado mediante programación.

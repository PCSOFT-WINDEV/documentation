


## Funciones Captcha (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles Captcha a través de la sintaxis del prefijo: 



|   |   |
| --- | --- |
| [&lt;Captcha&gt;.display](../WDLang2/1000019493.md) | Muestra un nuevo Captcha en un Captcha control. |
| [&lt;Captcha&gt;.Verify](../WDLang2/1000019567.md) | Comprueba si el valor tecleado por el usuario se corresponde con la cadena mostrada en un Captcha control. |

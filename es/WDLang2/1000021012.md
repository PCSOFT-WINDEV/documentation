
## FullScreenEnable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el navegador al modo "Pantalla completa".. El navegador ocupa toda la pantalla. La barra de herramientas ya no se muestra.  

Observación: Para salir del modo "Pantalla completa", utilice [FullScreenDisable](../WDLang2/1000021013.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FullScreenEnable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`FullScreenEnable()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El cambio al modo "Pantalla completa" sólo está disponible para los navegadores recientes.. 

- Atención: La función **FullScreenEnable** requiere una acción del usuario: Esta función no puede ser usada en un Event "Onload".. 
	Observación: Los navegadores bloquean este comando por razones de seguridad. Si es uno de los comandos que necesariamente requieren una acción del usuario.

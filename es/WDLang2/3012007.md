
## ASPExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Llama a un script.asp externo y devuelve el resultado en una cadena. Se realiza una petición HTTP: tienes la capacidad de usar un método POST o un método get.

Se recomienda utilizar esta función para ejecutar scripts ASP (en lugar de [ScriptExecute](../WDLang2/3012016.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Runs the "Script1.asp" script found at the root of the Web site 
// and returns the result in S
MyBuffer is Buffer
MyBuffer = ASPExecute("/Script1.asp")
```


<a name="Example2"></a>



```wl
// Login page in asp. Sends 3 parameters with the POST method
// The script is found in the ASP directory
MyBuffer is Buffer
MyBuffer = ASPExecute("/ASP/Login.asp", "Name=BOB&Pass=My+Secret&Lang=3", True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ASPExecute(<Script name> [, <"Name1=Param1&Name2=Param2&..."> [, <Runtime method>]])`
---

**`<Result>: (Buffer o cadena de caracteres)`**

Resultado del script ASP. En la mayoría de los casos, este resultado puede ser mostrado por [Pantalla StringDisplay](../WDLang2/3012001.md).

**`<Script name>: (Cadena de caracteres)`**

Address del guión. La URL del script puede tener el siguiente formato:

- "http://ServerName/Address.asp": el script Address.asp se encuentra en el ordenador "ServerName".

- "/Scripts/MyScript.asp": MyScript.asp se encuentra en el servidor actual, en el subdirectorio "/Scripts" del sitio.


Tiene la posibilidad de utilizar una secure Address (https)

**`<"Name1=Param1&Name2=Param2&...">: (Cadena de caracteres opcional)`**

Parámetros destinados al script. Esta cadena contiene para cada parámetro:

- el nombre del parámetro,

- el cartel de =,

- el valor del parámetro.




Dos parámetros están separados por el signo &.

**`<Runtime method>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para ejecutar un método POST,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para ejecutar un método GET.

## Observaciones




### Varios
<a name="varios_ELTPARAGRAPHE000073"></a>

**ASPExecute** es más lento que [ASPDisplay](../WDLang2/3012003.md).
<a name="NOTE0_2"></a>




### Parámetros de codificación
<a name="parametros_codificacion_ELTPARAGRAPHE000086"></a>

El contenido de los parámetros se codifica automáticamente a get una URL válida:

- transformación en UTF-8 (según las opciones de la página y del proyecto).

- codificación de caracteres especiales.




## PageVisible( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la página es Visible al usuario. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Page visible? Yes => initialize it
IF PageVisible() THEN Initialize()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PageVisible()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la página es visible para el usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## BrowserIsConnected( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el navegador está conectado a la red.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF BrowserIsConnected() = False THEN Info("You are disconnected")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserIsConnected()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el navegador está conectado a la red,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

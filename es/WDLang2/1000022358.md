
## GglAnalyticsAddSocialNetworkAction( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una acción de red social junto a Google Analytics. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
GglAnalyticsAddSocialNetworkAction("Facebook", "like", "http://mysite.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GglAnalyticsAddSocialNetworkAction(<Social network> , <Action> , <Target> [, <Other>])`
---

**`<Social network>: (Cadena de caracteres)`**

Nombre de la red social a la que se añadirá la acción (por ejemplo: Facebook, Twitter, ....). 

**`<Action>: (Cadena de caracteres)`**

Tipo de acción a añadir (Like, Send, Tweet, ....).

**`<Target>: (Cadena de caracteres)`**

Objetivo de la acción en la red social. Este parámetro puede ser la Address de un sitio web o cualquier otra cadena de caracteres.

**`<Other>: (Variant opcional)`**

Opciones adicionales según la red social. Para más detalles sobre los posibles valores, consulte la documentación de Google Analytics ([https://developers.google.com/analytics/devguides/collection/analyticsjs/social-interactions](https://developers.google.com/analytics/devguides/collection/analyticsjs/social-interactions)) 
Este parámetro es set a Nula por Default.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para utilizar esta función, debe tener y guardar un identificador de Google Analytics en el proyecto. Para obtener más información, consulte [WEBDEV y Google Analytics](../WDLang2/1000022374.md).




## CaptchaVerify (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si el valor tecleado por el usuario se corresponde con la cadena mostrada en un Captcha control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Validates the user input
TestCaptcha is boolean
TestCaptcha = CaptchaVerify(CPTCH_Captcha, EDT_VerifCaptcha) 

IF TestCaptcha = True THEN
	Info("Congratulations, the Captcha was successfully typed.")
ELSE
	Error("Check your input, the values are different.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CaptchaVerify(<Captcha control> , <Value to test>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor captcha es correcto,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Captcha control>: (Nombre del control)`**

Nombre del Captcha control a utilizar.

**`<Value to test>: (Cadena Unicode)`**

Valor introducido por el usuario. Este parámetro puede corresponder al nombre de control en el que el usuario introdujo el valor del Captcha. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Permisibilidad
<a name="permisibilidad_ELTPARAGRAPHE000059"></a>

Algunas fuentes de caracteres muestran caracteres diferentes utilizando el mismo dibujo o dibujos similares.. Por ejemplo, la letra L (minúscula) y el dígito 1 están muy cerca.: " l " " 1 ". 

La función **CaptchaVerify** tiene en cuenta las similitudes más comunes y realiza la comparación aceptando una sustitución de estos caracteres.

**CaptchaVerify** no distingue entre mayúsculas y minúsculas.

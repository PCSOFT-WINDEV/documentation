


## PopupClose (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una Control Popup mostrada en la página a través de [PopupDisplay](../WDLang2/1000019788.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PopupClose(POPUP_Question)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PopupClose([<Popup>])`
---

**`<Popup>: (Nombre del control)`**

Nombre de la Control Popup que se ocultará. Esta Control Popup es mostrada por [PopupDisplay](../WDLang2/1000019788.md). 
En caso contrario, **PopupClose** no tiene ningún efecto.
Este parámetro es opcional en el código de un control que se encuentra en el popup. Si no se especifica este parámetro, se cierra el popup que contiene la control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [PopupDisplay](../WDLang2/1000019788.md) y **PopupClose** deben utilizarse en diferentes proceos/eventos. Si estas funciones se utilizan en la misma Process/Event, la Control Popup no se visualiza.

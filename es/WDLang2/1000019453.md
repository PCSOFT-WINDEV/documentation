[Mostrar solo las constantes utilizadas por las sintaxis sin prefijo](#" onclick="$('.object').hide();$('.not_object').show();)Las siguientes constantes se utilizan para gestionar los controles Image:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **scrollCurrent** | Rellena la lista de imágenes e inicia el desplazamiento desde la posición actual si es posible. | [ImageStartScrolling](../WDLang2/1000019427.md) |
| **scrollInitialize** | Rellena la lista de imágenes e inicia el desplazamiento desde el principio. | [ImageStartScrolling](../WDLang2/1000019427.md) |
| **scrollStart** | Inicia (o reinicia) el desplazamiento desde la posición actual. | [ImageStartScrolling](../WDLang2/1000019427.md) |

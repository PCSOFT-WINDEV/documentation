
## GglAnalyticsAddPage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Page junto a Google Analytics. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
GglAnalyticsAddPage("/domain.com/pricing.shtml")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GglAnalyticsAddPage([<Page path> [, <Options>]])`
---

**`<Page path>: (Cadena de caracteres opcional)`**

Sección de la página Address representando a la ruta. Este valor debe comenzar con una barra oblicua (/).
Por Default, este parámetro corresponde al ruta del Page actual. 

**`<Options>: (Variant opcional)`**

Opciones adicionales dependientes de Google Analytics. Para más detalles sobre los posibles valores, consulte la documentación de Google Analytics en [https://developers.google.com/analytics/devguides/collection/analyticsjs/pages](https://developers.google.com/analytics/devguides/collection/analyticsjs/pages). 
Este parámetro es set a Nula por Default.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para utilizar esta función, debe tener y guardar un identificador de Google Analytics en el proyecto. Para obtener más información, consulte [WEBDEV y Google Analytics](../WDLang2/1000022374.md). 

- La Page actual se añade automáticamente (al incluir analytics.js).


## ChangeTarget( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el objetivo de una acción de botón.

Por Default, el objetivo se define en la pestaña "General" del botón de ejecución o de la página actual


|   |
| --- |
| **Atención: [ChangeTarget](../WDLang2/3058005.md) se puede utilizar directamente en el navegador process correspondiente a la acción del servidor de control:**<br><br>- El navegador **hace clic en el código para un botón, un enlace o control Imagen.**<br><br>- **Código "Siempre que se modifica" para los controles del Botón de Radio, Casilla de Control, Casilla de Lista o Combo Box.**<br><br><br>**ChangeTarget no tiene ningún efecto en los otros procesos del navegador o en un navegador local Procedure, incluso si este Procedure es llamado por uno de los procesos Above-mencionados.** |


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Browser click code of the Button control
// The target of the button defined in the description window (current frame)
ChangeTarget(CurrentBrowser)

// -- Server click code of the Button control
IF CheckUserName(EDT_UserName) = False THEN
	// Refreshes the PAGE_Identification page
	PageRefresh(PAGE_Identification)
ELSE
	// Displays the home page
	PageDisplay(PAGE_Welcome)
END
```

## Sintaxis

`ChangeTarget(<Target> [, <Window name> [, <Options> [, <Width> [, <Height> [, <Horizontal position> [, <Vertical position> [, <JavaScript parameters>]]]]]]])`
---

**`<Target>: (Cadena de caracteres o constante)`**

Nombre del objetivo o alias del objetivo iFrame control.

&lt;Target&gt; también puede tomar los siguientes valores:


|   |   |
| --- | --- |
| *CurrentBrowser* | El objetivo es el navegador actual. <br>Este parámetro también puede corresponder a la cadena "_top" (compatibilidad con WEBDEV 1.5). |
| *CurrentFrame* | El objetivo es el cuadro actual (valor de Default)<br>Este parámetro también puede corresponder a la cadena "_self" (compatibilidad con WEBDEV 1.5). |
| *NewBrowser* | El objetivo es un nuevo navegador (se abre una nueva ventana del navegador). Los siguientes parámetros de **ChangeTarget** se utilizan para configurar la ventana del navegador. <br>Este parámetro también puede corresponder a la cadena "_blank" (compatibilidad con WEBDEV 1.5). |
| *ParentFrame* | El objetivo es el contenedor de la página actual (conjunto de marcos padre, navegador padre). <br>Este parámetro también puede corresponder a la cadena "_parent" (compatibilidad con WEBDEV 1.5). |



**`<Window name>: (: Cadena de caracteres opcional)`**

Nombre de la ventana del nuevo navegador si &lt;Destino&gt; es igual a la constante *NewBrowser* (o a "_blank"). Este parámetro permite volver a mostrar una página en un navegador con el mismo nombre (si se abren varios navegadores en la el equipo del usuario de la Web). Este nombre es un nombre único (equivalente al nombre de un JavaScript Variable). 
**Atención**: Este nombre no debe contener ningún carácter de espacio, ningún carácter acentuado y ningún carácter especial.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Parámetros de la nueva ventana del navegador si &lt;Destino&gt; es igual a la constante *NewBrowser* (o a "_blank"):


|   |   |
| --- | --- |
| *ONFull* <br>(Valor predeterminado) | La ventana del nuevo navegador incluirá una barra de Address y barras de desplazamiento (equivalente a la combinación de todas las constantes anteriores) |
| *ONLink* | Se mostrará la barra de enlaces |
| *ONLocation* | La barra de Address será mostrada |
| *ONMenuBar* | Aparecerá la barra de menú |
| *ONResizable* | La ventana del nuevo navegador será redimensionable |
| *ONSatusBar* | Aparecerá la barra de estado |
| *ONScrollBar* | Aparecerán las barras de desplazamiento |
| *ONSimple* | La ventana del nuevo navegador será una ventana simple (sin combinación de constantes). |
| *ONToolBar* | Aparecerá la barra de herramientas |



**`<Width>: (Entero opcional)`**

Ancho de ventana en el nuevo navegador (en píxeles).

**`<Height>: (Entero opcional)`**

Altura de la ventana en el nuevo navegador (en píxeles).

**`<Horizontal position>: (Entero opcional)`**

Posición horizontal (en píxeles) de la ventana en el nuevo navegador (en relación con la Corner superior izquierda de la pantalla).

**`<Vertical position>: (Entero opcional)`**

Posición vertical (en píxeles) de la ventana en el nuevo navegador (en relación con la Corner superior izquierda de la pantalla).

**`<JavaScript parameters>: (Cadena de caracteres opcional)`**

Otros parámetros de JavaScript que se utilizan al abrir un nuevo navegador si &lt;Target&gt; es igual a la constante *NewBrowser* (o a "_blank").

## Observaciones




### Consejo: optimizar el tamaño de sus archivos HTML
<a name="consejo_optimizar_tamano_sus_archivos_html_ELTPARAGRAPHE000129"></a>

Para [PageDisplay](../WDLang2/3058008.md), [SemiDynamicPageDisplay](../WDLang2/3058018.md), [DynamicSiteDisplay](../WDLang2/3058024.md) y [FileDisplay](../WDLang2/3012005.md), le recomendamos que cambie el objetivo en estas funciones directamente. De hecho, el código JavaScript correspondiente incluido en los archivos HTML es más pequeño.




## &lt;Página&gt;.existente (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite saber si existe un contexto de página en el servidor (lo que significa si la página se abrió).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refreshes the page if the context exists otherwise opens the page
IF PAGE_Page3.Exist() = True THEN
	PAGE_Page3.Refresh()
ELSE
	PAGE_Page3.Display()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Page>.Exist()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el contexto de la página especificada existe (lo que significa si la página especificada se abre),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Page>: (Nombre de la página)`**

Nombre de la página cuya existencia de contexto se debe verificar.

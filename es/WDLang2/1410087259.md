


## PopupClosePage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una página que se muestra como una ventana emergente a través de [PopupDisplayPage](../WDLang2/1410087258.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Closes the page displayed as a popup
PopupClosePage(PAGE_ProductDetails)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PopupClosePage([<Page>])`
---

**`<Page>: (Nombre de la página opcional)`**

Nombre de la página que se muestra como una ventana emergente que se ocultará. Esta página debe haber sido visualizada por [PopupDisplayPage](../WDLang2/1410087258.md). En caso contrario, **PopupClosePage** no tiene ningún efecto.

Si se utiliza **PopupClosePage** en el código de la página principal, debe especificarse este parámetro: se debe especificar el nombre de la página emergente que se va a cerrar. 
Si se utiliza **PopupClosePage** en el código de la página actual (mostrada como una ventana emergente), este parámetro no es necesario: la página actual está cerrada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [PopupDisplayPage](../WDLang2/1410087258.md) y **PopupClosePage** deben utilizarse en diferentes proceos/eventos. Si estas funciones se utilizan en el mismo proceso/Event, la página Popup no se muestra.

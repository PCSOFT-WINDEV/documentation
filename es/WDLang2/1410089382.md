
## WBAdminAccount (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **WBAdminAccount** se utiliza para definir todas las características avanzadas de una cuenta WEBDEV en un servidor de aplicaciones WEBDEV. Las características de esta cuenta WEBDEV pueden definirse y modificarse utilizando diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
wbAuthLogin is WBAdminAuth
wbAuthLogin.ServerAddress = "https://my-server.internal/"
wbAuthLogin.User = "WWW\DATA"
wbAuthLogin.Password = "www"

WBAccounts is array of WBAdminAccounts = WBAdminListAccount(wbAuthLogin)
FOR i = 1 _TO_ WBAccounts.Count
	Trace(WBAccounts[i].User)
END
```

## Properties

Las siguientes propiedades pueden utilizarse para manipular una cuenta WEBDEV:

| Nombre de la propiedad | Tipo utilizado | Efecto |

| Address | Cadena de caracteres | Address del usuario de la cuenta. |

| AplicaciónUsuario | Cadena de caracteres | Nombre de la cuenta de Windows utilizada para ejecutar sitios y servicios web. La contraseña de associated es la que se utiliza para la cuenta de Windows. |
| CuentaActiva | booleano | <br><br>- <u><u><u>True</u></u></u> si la cuenta es Active ,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br> |
| Datos de carpeta | Cadena de caracteres | Directorio de archivos de datos para las aplicaciones y sitios WEBDEV associated con esta cuenta. |
| DeploymentAccount | booleano | <br><br>- <u><u><u>True</u></u></u> si el usuario puede implementar sitios y servicios,<br><br>- <u><u><u>False</u></u></u> en caso contrario.<br><br><br> |
| DirectorioUsuario | Cadena de caracteres | Directorio de aplicaciones y sitios WEBDEV associated con esta cuenta. |
| drogadicta | Cadena de caracteres | Nombre de la cuenta de Windows utilizada para la instalación (implementación) y la administración remota. La contraseña de associated es la que se utiliza para la cuenta de Windows. |
| Email | Cadena de caracteres | Email Address del usuario de la cuenta. |
| Empresa | Cadena de caracteres | Empresa del usuario de la cuenta. |
| EstadísticasCuenta | booleano | <br><br>- <u><u><u>True</u></u></u> si el usuario puede ver las estadísticas de los sitios,<br><br>- <u><u><u>False</u></u></u> en caso contrario.<br><br><br> |
| Foto | Buffer | Foto del usuario. |

| VirtualServerDirectory | Cadena de caracteres | Directorio raíz del servidor virtual. |

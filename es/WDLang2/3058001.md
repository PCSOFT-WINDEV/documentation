


## Funciones de gestión de páginas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las páginas:



|   |   |
| --- | --- |
| [CancelAWPContext](../WDLang2/1000017170.md) | Elimina del contexto de AWP una Variable que fue añadida por [DeclararAWPContext](../WDLang2/3058028.md). |
| [CellCloseDialog](../WDLang2/1000017140.md) | Oculta una control Celda previamente mostrada en la página a través de [CellDisplayDialog](../WDLang2/1000017139.md). |
| [CellDisplayDialog](../WDLang2/1000017139.md) | Muestra un control Celda en una página con un efecto OVI (Oscurecimiento de Ventanas Inactivas).. Esta función se utiliza para simular fácilmente una ventana de diálogo en el código del navegador, utilizando un control Celda en una página. |
| [ChangeAction](../WDLang2/3058007.md) | Se utiliza para especificar la acción a realizar cuando la página HTML mostrada en el navegador ya no está sincronizada con el contexto de la página en el servidor. |
| [ChangeSourcePage](../WDLang2/1000024600.md) | Cambia dinámicamente la página que se muestra en un control Página interna |
| [ChangeTarget](../WDLang2/3058005.md) | Cambia el objetivo de una acción de botón. |
| [ConfigureAWPContext](../WDLang2/1000017066.md) | Configura el modo de funcionamiento del contexto AWP. |
| [ContextClose](../WDLang2/3058003.md) | Cierra un contexto de página. |
| [ContextExist](../WDLang2/3058002.md) | Le permite saber si existe un contexto de página en el servidor (lo que significa si la página se abrió). |
| [ContextOpen](../WDLang2/3058004.md) | Abre un nuevo contexto de página sin devolver la información al navegador. |
| [CurrentPage](../WDLang2/3058009.md) | Devuelve el nombre de la página que contiene el código WLanguage actualmente en ejecución. |
| [DeclareAWPContext](../WDLang2/3058028.md) | Permite declarar una lista de variables cuyo valor será persistente entre las sucesivas visualizaciones de las páginas de Active WEBDEV. |
| [Dirección de página](../WDLang2/3058010.md) | Se utiliza para averiguar la INTERNET Address de una página. |
| [DynamicSiteDisplay](../WDLang2/3058024.md) | Inicia un sitio WEBDEV dinámico desde una página estática o desde una página WEBDEV dinámica. |
| [FramesetDisplay](../WDLang2/3058006.md) | Muestra un conjunto de marcos WEBDEV en el navegador del usuario. |
| [FramesetRefresh](../WDLang2/3058014.md) | Actualiza un conjunto de marcos que se muestra en el navegador del usuario a partir del contexto en el servidor. |
| [FramesetUse](../WDLang2/3058012.md) | Muestra un conjunto de marcos WEBDEV en el navegador del usuario y cierra todos los contextos de la página actual y del conjunto de marcos. |
| [FreeAWPContext](../WDLang2/1000020429.md) | Libera el contexto de AWP por adelantado (en disco) para permitir que la otra llamada en el mismo contexto de AWP sea procesada en paralelo. |
| [IdentifierAWPContext](../WDLang2/1000017144.md) | Devuelve el identificador del contexto de AWP. |
| [PageActivateDDW](../WDLang2/1000017087.md) | Habilita o deshabilita la OVI (Dim Disabled Window) al mostrar una página modal (las páginas utilizadas para comunicarse con el usuario, por ejemplo). |
| [PageChangeLayout](../WDLang2/1000024598.md) | Aplica un layout específico a una página. |
| [PageCloseDialog](../WDLang2/1000017166.md) | Cierra la página actual. Esta página fue abierta por [PageDisplayDialog](../WDLang2/1000017124.md). Un valor de retorno puede ser devuelto a la página de llamada. |
| [PageCurrentLayout](../WDLang2/1000024599.md) | Devuelve el índice del layout actual: <br><br>- en la página en curso.<br><br>- en la página especificada.<br><br><br> |
| [PageDisplay](../WDLang2/3058008.md) | Muestra una página web en el navegador del usuario. |
| [PageDisplayDialog](../WDLang2/1000017124.md) | Muestra una página en modo modal. Esta función se utiliza para establecer un diálogo con el usuario. La página se muestra en primer plano mientras que la página de apertura se muestra en segundo plano, Grayed por el mecanismo de OVI. |
| [PageExist](../WDLang2/3058013.md) | Comprueba si la página especificada se visualiza actualmente en el browser del usuario Web. Permite saber si una página se muestra en un marco distinto al actual. |
| [PageInitialization](../WDLang2/3058025.md) | Restablece a cero (o no) los controles que se encuentran en la página actual e inicia los eventos para la inicialización de los controles y la página. |
| [PageParameter](../WDLang2/3058026.md) | Devuelve el valor de un parámetro pasado a la página actual. |
| [PagePosition](../WDLang2/3058021.md) | Desplaza una página hacia arriba para posicionar un control en la sección Visible de la página (arriba) del navegador. |
| [PageRateDDW](../WDLang2/1000017086.md) | Permite conocer y modificar la tasa de grises de las páginas que utilizan OVI (Dim disabled pages). |
| [PageReadHeaderHTTP](../WDLang2/1000021635.md) | Lee y devuelve una o más cabeceras HTTP recibidas por la Page actual (estas cabeceras son enviadas por el cliente al servidor en la petición actual). |
| [PageRefresh](../WDLang2/3058011.md) | Actualiza una página mostrada en el navegador del usuario a partir del contexto en el servidor. |
| [PageSubmit](../WDLang2/3058020.md) | Valida la página especificada e inicia la ejecución de un botón.. |
| [PageToASP](../WDLang2/3058019.md) | Envía los datos encontrados en una página que actualmente se muestra en el navegador a un servidor ASP. |
| [PageToEmail](../WDLang2/3058027.md) | Envía por correo electrónico los datos encontrados en una página que actualmente se muestra en el navegador. |
| [PageToJSP](../WDLang2/3058022.md) | Envía los datos encontrados en una página que actualmente se muestra en el navegador a un servidor JSP. |
| [PageToPHP](../WDLang2/3058015.md) | Envía los datos encontrados en una página actualmente mostrada en el navegador a un servidor PHP. |
| [PageVisible](../WDLang2/1000021014.md) | Indica si la página es Visible al usuario. |
| [PageWriteHeaderHTTP](../WDLang2/1000021636.md) | Añade un encabezado HTTP adicional que será devuelto al navegador: <br><br>- durante la siguiente función de visualización de la página ([PageDisplay](../WDLang2/3058008.md), [PageRefresh](../WDLang2/3058011.md), ...),<br><br>- durante el retorno de automatic de la página actual al final de la acción actual.<br><br><br> |
| [PopupAnimate](../WDLang2/1000020313.md) | Muestra una ventana emergente en una celda de página. |
| [PopupClose](../WDLang2/1000019789.md) | Oculta una ventana emergente que se muestra en la página a través de [PopupDisplay](../WDLang2/1000019788.md). |
| [PopupClosePage](../WDLang2/1410087259.md) | Oculta una página mostrada como popup a través de [PopupDisplayPage](../WDLang2/1410087258.md). |
| [PopupDisplay](../WDLang2/1000019788.md) | Muestra una ventana emergente en una Page con efecto OVI (Oscurecimiento de Ventanas Inactivas). |
| [PopupDisplayPage](../WDLang2/1410087258.md) | Muestra una Page WEBDEV como una ventana emergente en la Page actual con un efecto OVI (Oscurecimiento de Ventanas Inactivas). |
| [PreviousPage](../WDLang2/3058017.md) | Devuelve el nombre de la página anterior. |
| [SemiDynamicPageDisplay](../WDLang2/3058018.md) | Muestra una página semidinámica en el browser del usuario Web desde una página WEBDEV dinámica o estática. |
| [SiteAddress](../WDLang2/3058023.md) | Devuelve el INTERNET Address para conectarse a un sitio WEBDEV dinámico (en modo Sesión) que se encuentra en el mismo servidor. |
| [Uso de la página](../WDLang2/3058016.md) | Muestra una página WEBDEV en el navegador del usuario y cierra todos los contextos de página actualmente abiertos en el servidor. |

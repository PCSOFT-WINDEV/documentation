


## &lt;Image&gt;.first (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el primer Image para el pergamino.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the first banner
IMG_Banner.First()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Image control>.First()`
---

**`<Image control>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Imagen, 

- Control Imagen con enlace, 

- Control Miniatura.

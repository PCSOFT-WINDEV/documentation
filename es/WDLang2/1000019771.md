


## LooperExpand (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Amplía una pausa en la control Looper en una fila determinada . 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the [4 stars] Button control
nRow4Star is int
nRow4Star = LooperSearch(LOOP_HOTEL, ATT_NBSTARS, 4)
LooperExpand(LOOP_HOTEL, nRow4Star)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LooperExpand(<Looper control> [, <Row> [, <Break name>]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es una cadena vacía (""), se utiliza la control Looper actual.

**`<Row>: (Entero opcional)`**

Número de la fila a ampliar. 
Si no se especifica este parámetro, se desglosa el fila actual.. 

**`<Break name>: (Cadena de caracteres opcional)`**

Nombre de la pausa a ampliar. Este parámetro debe especificarse en rupturas anidadas. Se usa para definir la ruptura que se expandirá. 
Si no se especifica este parámetro, la ruptura Above el fila actual se expandirá (incluso si hay varios niveles de ruptura). 




## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000059"></a>

La función **LooperExpand** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.

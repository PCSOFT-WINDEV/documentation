


## &lt;Banner deslizante&gt;Último (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el último plano de un Banner deslizante control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the last banner plane (the last banner)
BAN_Banner.Last()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Sliding Banner control>.Last()`
---

**`<Sliding Banner control>: (Nombre del control)`**

Nombre de la pancarta deslizante control para usar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los aviones Invisible del Sliding Banner control son ignorados.

## Componente:
wd280page.dll

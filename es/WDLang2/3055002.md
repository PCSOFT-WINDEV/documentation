


## Funciones de gestión del navegador
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los navegadores:



|   |   |
| --- | --- |
| [AddFavorite](../WDLang2/3055004.md) | Añade un INTERNET Address a la lista de favoritos del usuario de la web. |
| [BrowserClose](../WDLang2/3055007.md) | Cierra la ventana del navegador actual y detiene la ejecución del código actual. |
| [BrowserHeight](../WDLang2/1000022548.md) | Devuelve la altura (en píxeles) del área de cliente donde se muestra la página. |
| [BrowserHistoryAdd](../WDLang2/1000023458.md) | Añade una entrada en la historia de la navegación asociando datos. Estos datos se transmitirán al volver a esta entrada. |
| [BrowserHistoryModify](../WDLang2/1000023459.md) | Modifica los datos de la entrada actual en el historial de navegación. Estos datos se transmitirán al volver a la entrada actual. |
| [BrowserIPAddress](../WDLang2/3055003.md) | Devuelve la IP Address del ordenador cliente conectado al sitio WEBDEV. |
| [BrowserIsConnected](../WDLang2/1000019392.md) | Indica si el navegador está conectado a la red. |
| [BrowserMobile](../WDLang2/1000021856.md) | Permite averiguar si el navegador utilizado por el usuario de la Web se inicia en un dispositivo móvil. |
| [BrowserName](../WDLang2/3055005.md) | Devuelve el nombre del navegador del usuario Web ("user agent"). |
| [BrowserOpen](../WDLang2/3055006.md) | Abre una nueva ventana del navegador. |
| [BrowserOS](../WDLang2/1000021857.md) | Indica el sistema operativo instalado en el navegador del usuario de la web. |
| [BrowserPlatform](../WDLang2/1000019486.md) | Devuelve la plataforma del navegador del usuario. |
| [BrowserPrint](../WDLang2/1000024545.md) | Imprime la página que se muestra en el navegador. |
| [BrowserRunApp](../WDLang2/1000019424.md) | Abre el navegador Default del dispositivo actual. |
| [BrowserSlice](../WDLang2/1000022546.md) | Devuelve el número del rango de resolución que se muestra en una página en modo Responsive Web Design. |
| [BrowserType](../WDLang2/1000019487.md) | Devuelve el tipo de navegador utilizado por el usuario Web. |
| [BrowserVersion](../WDLang2/1000019488.md) | Devuelve la versión del navegador utilizado por el usuario web. |
| [BrowserWidth](../WDLang2/1000022547.md) | Devuelve el ancho (en píxeles) del área de cliente donde se muestra la página. |
| [CookieDelete](../WDLang2/1000025874.md) | Borra una Cookie en el ordenador del usuario. |
| [CookieRead](../WDLang2/3012009.md) | Recupera el valor de una Cookie guardada en la el equipo del usuario de la Web. |
| [CookieWrite](../WDLang2/3012008.md) | Escribe una Cookie en la el equipo del usuario de la Web. |
| [EmailOpenMail](../WDLang2/3032021.md) | Abre el software de mensajería Default: <br><br>- de usuario Web en el ordenador del navegador.<br><br>- del usuario en el ordenador Windows actual.<br><br>- en la Phone.  <br><br><br> |
| [FullScreenDisable](../WDLang2/1000021013.md) | Desactiva el modo "Pantalla completa" del navegador. |
| [FullScreenEnable](../WDLang2/1000021012.md) | Cambia el navegador al modo "Pantalla completa".. El navegador ocupa toda la pantalla. |
| [NotificationDisplay](../WDLang2/1000021021.md) | Muestra un mensaje fuera del navegador. |

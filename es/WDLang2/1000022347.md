
## AppServerAddScheduledTask( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva tarea programada a la aplicación (sitio o servicio web).

**Atención**: La nueva tarea programada no es persistente. Se borrará al reiniciar la WEBDEV servidor de aplicaciones o al desplegar el sitio.
Se recomienda utilizar la "Configuración del sitio": Iniciar servidor de aplicaciones" Event para utilizar **AppServerAddScheduledTask**. La tarea programada se recreará automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Programs the task so that it is run every hour
MyScheduledTask is appServerScheduledTask
MyScheduledTask.Name = "MyTask"
MyScheduledTask.Procedure = MyProc
MyScheduledTask.Month = "*"
MyScheduledTask.DayOfMonth = "*"
MyScheduledTask.DayOfWeek = "*"
MyScheduledTask.Hour = "*"
MyScheduledTask.Minute = "0"

IF NOT AppServerAddScheduledTask(MyScheduledTask) THEN
	// Error case (if a task with the same name already exists for example)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppServerAddScheduledTask(<Scheduled task>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó la tarea programada. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Scheduled task>: (appServerScheduledTask Variable)`**

Nombre de la variable [appServerScheduledTask](../WDLang2/1000022367.md) correspondiente al description de la tarea programada a añadir.

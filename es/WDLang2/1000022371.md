


## LooperEndFilling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la Process para rellenar una control Looper de forma programada. 

Recordatorio: El inicio del llenado fue especificado por [LooperStartFilling](../WDLang2/1000022370.md). Se representa el progreso de esta operación:

- mediante una barra de progreso de Infinite que se muestra en la parte inferior de control.

- por una ventana interna.




Atención: Antes de utilizar esta función, es necesario habilitar el mecanismo de gestión de la operación de llenado en la ventana de control description. Para obtener más información, consulte [Características de un control Looper en una ventana](../WDChamp/1013294.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LooperStartFilling(LOOP_MyLooper)

FOR i = 1 TO 10000
	LooperAddLine(LOOP_MyLooper)
	Multitask(-1)
END

LooperEndFilling(LOOP_MyLooper)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LooperEndFilling(<Looper control>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Looper a la que pertenezca la Event actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no tiene ningún efecto en los controles del Looper de relacionado a un archivo de datos o la consulta.




## PageCloseDialog( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la página actual. Esta página fue abierta por [PageDisplayDialog](../WDLang2/1000017124.md). Un valor de retorno puede ser devuelto a la página de llamada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the page used to dialog while returning True
PageCloseDialog(True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`PageCloseDialog([<Returned value>])`
---

**`<Returned value>: (Tipo correspondiente al valor devuelto (opcional))`**

El valor devuelto por la página cuando se cierra. Este valor se recupera sólo si la página fue abierta por [PageDisplayDialog](../WDLang2/1000017124.md). Si la página no se ha abierto con [PageDisplayDialog](../WDLang2/1000017124.md), se devuelve un error.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **PageCloseDialog**() equivale a [ContextoCerrar](../WDLang2/3058003.md)("").

- **PageCloseDialog**(Valor) equivale a [ContextoCerrar](../WDLang2/3058003.md)("", Valor).

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

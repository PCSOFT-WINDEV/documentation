


## &lt;Colapso (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Contrae una ruptura previamente expandida en un control Looper. Los elementos de ruptura ya no son Visible en el control Looper.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nRow4Star is int
nRow4Star = LOOP_HOTEL.Search(ATT_NBSTARS, 4)
LOOP_HOTEL.Collapse(nRow4Star)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Looper control>.Collapse([<Row> [, <Break name>]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Row>: (Entero opcional)`**

Número de la fila a contraer. 
Si no se especifica este parámetro, se contraerá el fila actual..

**`<Break name>: (Cadena de caracteres opcional)`**

Nombre de la ruptura al colapso. Este parámetro debe especificarse en rupturas anidadas. Permite definir la ruptura que se colapsará. La función se colapsa desde la primera ruptura de Default (incluso si hay varios niveles de ruptura).




## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000177"></a>

La función **&lt;Colapso** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.

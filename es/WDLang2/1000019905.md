
## ColorPalette( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un Color encontrado en el actual Palette.
<a name="Example1"></a>
<a name="sample_code"></a>

```wl
Trace(ColorPalette(colorBackgroundGeneral))
```

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000225"></a>

La función **ColorPalette** selecciona la paleta según el contexto:

- En el código de un control o en el código de una página, la función devuelve los colores que se encuentran en la Palette de la página.

- En el código del proyecto, la función devuelve los colores que se encuentran en la Palette del proyecto.




Si no existe la Palette deseada, **ColorPalette** utiliza la Palette de un elemento más general:

- En una página (sin Palette o que utiliza el Palette del proyecto) : **ColorPalette** busca el palette del componente padre (si la página está en un componente) o del proyecto (si la página está directamente en la aplicación).

- En un componente (sin Palette o que utiliza el Palette por defecto), **ColorPalette** busca el Palette del componente padre (si el componente está en otro componente) o del proyecto (si el componente está directamente en la aplicación).

- En el proyecto (sin Palette o que utiliza el Palette por defecto), **ColorPalette** devuelve un error porque el Palette por defecto no se puede utilizar mediante programación.

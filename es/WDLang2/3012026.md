


## Funciones varias de WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las diferentes funciones de WEBDEV son las siguientes: 



|   |   |
| --- | --- |
| [ASPDisplay](../WDLang2/3012003.md) | Llama a un script ASP externo y devuelve la página de resultados en la ventana actual del navegador. |
| [ASPExecute](../WDLang2/3012007.md) | Llama a un script.asp externo y devuelve el resultado en una cadena. |
| [CapsLockVerify](../WDLang2/1000020557.md) | Comprueba si la tecla CapsLock está pulsada. |
| [CertificateClientInfo](../WDLang2/1000019393.md) | Devuelve la información sobre la Certificate utilizada por el ordenador cliente. |
| [ColorPalette](../WDLang2/1000019905.md) | Devuelve un Color encontrado en el actual Palette. |
| [ConnectionCount](../WDLang2/3012004.md) | Devuelve el número de instancias del sitio WEBDEV actualmente ejecutadas en el servidor. |
| [DynamicServingConfigure](../WDLang2/1000021994.md) | Indica el entorno que utilizará la "Publicación Dinámica" para elegir el conjunto de páginas que se mostrarán. |
| [ExecuteDelayedProcedure](../WDLang2/1000020682.md) | Dirige un procedimiento diferido (en el WEBDEV servidor de aplicaciones). |
| [FileDisplay](../WDLang2/3012005.md) | Muestra un fichero en el navegador del usuario Web. |
| [FolderData](../WDLang2/3036007.md) | FolderData se guarda para compatibilidad con versiones anteriores. |
| [FolderWeb](../WDLang2/3012015.md) | Devuelve el ruta del directorio que contiene:<br><br>- las imágenes,<br><br>- los archivos JavaScript,<br><br>- los archivos del applet Java,<br><br>- los otros archivos accesibles desde el navegador.<br><br><br> |
| [FolderWeb_15](../WDLang2/3012018.md) | Atención: al migrar un proyecto de WEBDEV 1.5 a WEBDEV 17: <br><br>- FolderWeb fue sustituido por FolderWeb_15. <br><br>- se genera automáticamente un aviso en su proyecto.<br><br><br> |
| [HTMLClassAdd](../WDLang2/1000021550.md) | Añade una clase a las clases de HTML de un control. |
| [HTMLClassDelete](../WDLang2/1000021552.md) | Borra una clase de las clases de HTML de un control. |
| [HTMLClassToggle](../WDLang2/1000021551.md) | Cambia una clase en las clases de HTML de un control: si la clase no existe, se añade; si la clase ya existe, se elimina. |
| [jQuery](../WDLang2/1000021070.md) | Ejecuta un método JavaScript (o varios métodos encadenados) de la biblioteca jQuery en un elemento de la página. |
| [jQueryExecute](../WDLang2/1000021008.md) | Ejecuta un método JavaScript de la biblioteca jQuery en un elemento de la página. |
| [JSEndEvent](../WDLang2/1000018925.md) | Elimina la asociación entre una función de navegador de WLanguage y una Event( implementada por [JSEvent](../WDLang2/1000018924.md)). |
| [JSEvent](../WDLang2/1000018924.md) | Asocia una Procedure del navegador con una Event en un objeto del código del navegador. |
| [JSInfoEvent](../WDLang2/3036067.md) | Usó para manipular las propiedades de JavaScript del navegador Event que desencadenó la ejecución del código. |
| [JSInterruptEvent](../WDLang2/1000019185.md) | Interrumpe la Process de la actual Event. |
| [JSMethod](../WDLang2/3012002.md) | Se utiliza para ejecutar un método JavaScript en un elemento que se encuentra en la página actual. |
| [JSONExecute](../WDLang2/1000017284.md) | Llama a una URL del servidor que devuelve datos en formato JSON (JavaScript Object Notation). |
| [JSONExecuteExternal](../WDLang2/1000017286.md) | Llama a una URL de un servidor externo que devuelve datos en formato JSON (JavaScript Object Notation). |
| [JSProperty](../WDLang2/3012019.md) | Se utiliza para las características específicas de manipular en los objetos que se encuentran en la página actual. |
| [PHPDisplay](../WDLang2/3012021.md) | Llama a un script PHP externo y devuelve la página de resultados en la ventana actual del navegador. |
| [PHPExecute](../WDLang2/3012013.md) | Llama a un script.php externo y devuelve el resultado en una cadena. |
| [ScriptDisplay](../WDLang2/3012017.md) | Llama a un script o página externa (.php,.asp,.mhtml o.mht) y devuelve la página de resultados en la ventana actual del navegador. |
| [ScriptExecute](../WDLang2/3012016.md) | Llama a un script externo (.asp o.php) y devuelve el resultado en una cadena. |
| [SessionHeartBeatEnable](../WDLang2/1000026248.md) | Activa o desactiva el "latido" de la sesión actual. |
| [SessionPrologue](../WDLang2/1000026262.md) | Activa o desactiva una Procedure WLanguage llamada en cada solicitud de una sesión. |
| [SessionRequestedPage](../WDLang2/1410089106.md) | Devuelve el nombre de la Page solicitada al inicio de la sesión. |
| [SSLActive](../WDLang2/3012014.md) | Permite activar o desactivar el modo secure SSL. |
| [StringDisplay](../WDLang2/3012001.md) | Devuelve una cadena específica (o un búfer) al navegador del cliente en respuesta a una solicitud. |

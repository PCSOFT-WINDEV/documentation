


## PageInitialization( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restablece a cero (o no) los controles que se encuentran en la página actual e inicia los eventos para la inicialización de los controles y la página.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the controls 
//(the controls are reset to blank first)
PageInitialization()
```
<a name="Example2"></a>

```wl
// Initialize the controls 
//(don't reset the controls first)
PageInitialization(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`PageInitialization([<Reset>])`
---

**`<Reset>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor por defecto) si los controles deben ser reseteados antes de ejecutar el Event para inicializar los controles y la página,

- <u><u><u><u>False</u></u></u></u> si los controles no se deben volver a poner en blanco.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ¿Cómo saber si la inicialización de un control fue activada por PageInitialization?
<a name="¿como_saber_inicializacion_control_fue_activada_por_docparampagetitleshort_ELTPARAGRAPHE000039"></a>

Utilice el **OPENING** Variable (booleano) en el código de inicialización del control a utilizar.

La variable **OPENING** es set a:

- <u><u><u><u>True</u></u></u></u> si la inicialización Event fue activada cuando se abrió la página,

- <u><u><u><u>False</u></u></u></u> si el evento de inicialización fue lanzado por **PageInitialization**.




Por lo tanto, la ejecución de los procesos en la inicialización del control puede depender del valor del <u><u><u><u>OPENING</u></u></u></u> Variable.

Por ejemplo, aquí está cómo inicializar los controles de la Lista sólo si **OPENING** es set a <u><u><u><u>True</u></u></u></u>:


```wl
// Control initialization
IF OPENING = True THEN
	...
END
```

<a name="NOTE0_2"></a>


### Atención 
<a name="atencion_ELTPARAGRAPHE000075"></a>

- **PageInitialization** no debe ser llamado en la "Apertura" de una página Event.

- No confunda **PageInitialization** con [Reiniciar](../WDLang1/3025026.md), que reinicia todos los controles de la página (sin ejecutar el Event de "Inicialización" de los controles).

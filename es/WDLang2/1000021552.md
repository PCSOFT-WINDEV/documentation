
## HTMLClassDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una clase de las clases de HTML de un control.

Observación: También puede determinar y modificar el valor del atributo "class" de HTML de un control a través de la [HTMLClass](../Proprietes/1000021530.md) Property. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLClassDelete(EDT_Edit1, "InputInError")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLClassDelete(<Control> , <Class to delete>)`
---

**`<Control>: (Nombre del control)`**

Nombre del control a manipular.

**`<Class to delete>: (Cadena de caracteres)`**

Nombre de la clase a eliminar. Si la clase no existe, la función no tiene efecto.

Si se deben eliminar varias clases, este parámetro debe contener el nombre de las diferentes clases separadas por caracteres de espacio. Cada clase se elimina de forma independiente.

Atención: El nombre de las clases es sensible a las mayúsculas y minúsculas.

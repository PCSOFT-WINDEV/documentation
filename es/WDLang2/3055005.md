
## BrowserName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del navegador del usuario Web ("user agent").. El nombre del navegador es devuelto por el software de navegación del usuario Web.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NameOfBrowser is string
NameOfBrowser = BrowserName()
IF Position(NameOfBrowser, "MSIE") <> 0 THEN
	// The browser is Internet Explorer
ELSE
	// The browser can be Netscape, Mozilla, Opera ...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BrowserName()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre del navegador del usuario Web.

<a name="NOTE0_1"></a>

## Observaciones


### Ejemplos de nombres de navegadores
<a name="ejemplos_nombres_navegadores_ELTPARAGRAPHE000031"></a>

El nombre devuelto cambia según la versión y el navegador. Algunos ejemplos de nombres devueltos para los navegadores más comunes:  

- INTERNET Explorer 6: Mozilla/4.0 (compatible;MSIE 6.0;Windows NT 5.1;SV1;.NET CLR 1.1.4322)

- INTERNET Explorer 11: Mozilla/5.0 (Windows NT 6.1; Win64; x64; Trident/7.0; rv:11.0) como Gecko

- Ventaja en Windows 10: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, como Gecko) Chrome/42.0.2311.135 Safari/537.36 Edge/12.10240

- Edge en Windows 10 Mobile: Mozilla/5.0 (Android 4.2.1; NOKIA; Lumia 1320) AppleWebKit/537.36 (KHTML, como Gecko) Chrome/46.0.2486.0 Mobile Safari/537.36 Edge/13.10581

- FireFox 44.0: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:44.0) Gecko/20100101 Firefox/44.0

- Google Chrome 47 (versión para Windows): Mozilla/5.0 (Windows NT 6.1; WOW64) AppleWebKit/537.36 (KHTML, como Gecko) Chrome/47.0.2526.58 Safari/537.36

- Safari (versión MacOS): Mozilla/5.0 (Macintosh; Intel Mac OS X 10_11_1) AppleWebKit/601.2.7 (KHTML, like Gecko) Version/9.0.1 Safari/601.2.7

- Safari en el iPhone: Mozilla/5.0 (iPhone; CPU iPhone OS 9_1 como Mac OS X) AppleWebKit/601.1.46 (KHTML, como Gecko) Versión/9.0 Mobile/13B143 Safari/601.1





### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000047"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

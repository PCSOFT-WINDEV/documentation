
## &lt;Clasificación (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite clasificar una control Looper de acuerdo a uno o más de sus atributos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sort according to one of the Looper control attributes selected in a Radio Button control
SWITCH RADIO_RADIO1
	CASE 1: LOOP_Customer.Sort("NAME")
	CASE 2: LOOP_Customer.Sort("COMPANY")
	CASE 3: LOOP_Customer.Sort("CITY")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Ordenar una control Looper según una lista de atributos

`<Result> = <Looper control>.Sort(<List of attributes>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la ordenación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<List of attributes>: (Nombre del control)`**

Nombre de los atributos según los cuales se clasificará la control Looper (hasta 10 atributos). Este string debe tener el siguiente formato:

```txt
[<Direction>]<Attribute 1> + TAB + [<Direction>]<Attribute 2> + TAB + ...
```
donde:

- &lt;Dirección&gt;: (opcional), puede tomar los siguientes valores:

	- "+": la clasificación se realizará en orden ascendente (valor Default).

	- "-": la ordenación se realizará en orden descendente.




- : nombre del atributo utilizado para la clasificación.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000200"></a>

La función **&lt;Clasificación** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_3"></a>


### clasificar
<a name="clasificar_ELTPARAGRAPHE000224"></a>

La clasificación se tendrá en cuenta durante la siguiente adición de una fila en la control Looper ([.Add](../WDLang2/1000023469.md), [&lt;AddLine](../WDLang2/1000023475.md), [&lt;Insertar](../WDLang2/1000023509.md) y [&lt;InsertLine](../WDLang2/1000023511.md)).<a name="NOTE0_4"></a>


### Looper vinculado a una fuente de datos y utilizando una iteración programada
<a name="looper_vinculado_una_fuente_datos_utilizando_una_iteracion_programada_ELTPARAGRAPHE000255"></a>

**&lt;Clasificación** no funciona en un control Looper con las siguientes características: 

- control Looper vinculada a una fuente de datos,

- control Looper con la opción control **Recorrer mediante programación** (pestaña "Contenido" de la ventana de descripción del control). 




Para cambiar el orden de visualización de un control Looper con **navegación mediante programación**, se debe adaptar el código de los procesos de lectura del control Looper (proceso Lectura primero registro, anterior registro, siguiente registro, etc.).

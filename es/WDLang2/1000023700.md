


## &lt;SelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona un fila en un control Looper. El fila seleccionado corresponde al fila actual en el que se muestra el banda de selección.

**Observación**: La función **&lt;SelectPlus** puede utilizarse en:

- a control Looper basada en un archivo de datos o rellenada mediante programación.

- una selección única o una selección múltiple control Looper.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the row 5 in the "LOOP_Product" looper
LOOP_Product.SelectPlus(5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Looper control>.SelectPlus([<Index 1> [... [, <Index N>]]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Index 1>: (Entero opcional)`**

Primer índice de la fila a seleccionar. 
Si no se especifica este parámetro, se seleccionan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila ya seleccionada, la función **&lt;SelectPlus** no tiene ningún efecto. Puede get el índice de un fila seleccionado con [&lt;Seleccione](../WDLang2/1000023697.md).



**`<Index N>: (Entero opcional)`**

Enésimo índice de la fila a seleccionar. 
Si no se especifica este parámetro, se seleccionan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila ya seleccionada, la función **&lt;SelectPlus** no tiene ningún efecto. Puede get el índice de un fila seleccionado con [&lt;Seleccione](../WDLang2/1000023697.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Evento "Selección de una fila"
<a name="evento_seleccion_una_fila_ELTPARAGRAPHE000250"></a>

El evento "Selección de una fila" no se ejecuta al llamar a la función **&lt;SelectPlus**.

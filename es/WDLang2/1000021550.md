
## HTMLClassAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una clase a las clases de HTML de un control.

Observación: También puede determinar y modificar el valor del atributo "class" de HTML de un control a través de la [HTMLClass](../Proprietes/1000021530.md) Property. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLClassAdd(EDT_Edit1, "InputInError")
```




## Sintaxis
<a name="SYNTAX1"></a>

`HTMLClassAdd(<Control> , <Class to add>)`
---

**`<Control>: (Cadena de caracteres)`**

Nombre del control a manipular.

**`<Class to add>: (Cadena de caracteres)`**

Nombre de la clase a añadir. La clase no se añade dos veces si ya existe.

Si deben añadirse varias clases, este parámetro debe contener el nombre de las diferentes clases separadas por caracteres de espacio. Cada clase se añade de forma independiente.

Atención: El nombre de las clases es sensible a las mayúsculas y minúsculas.

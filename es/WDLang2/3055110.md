
## AJAXCancelAsynchronousCall( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la ejecución automática del procedimiento **navegador** llamado con la función [AJAXExecuteAsynchronous](../WDLang2/3055115.md). El procedimiento **server** llamado por la función [AJAXExecuteAsynchronous](../WDLang2/3055115.md) continuará ejecutándose.

**Recordatorio**: El navegador Procedure se utiliza para Process el resultado devuelto por el servidor Procedure. Cuando se llama a **AJAXCancelAsynchronousCall**, no se procesa el resultado del servidor Procedure
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is AJAX supported by the current browser?
IF AJAXAvailable() = True THEN
	// Run the server procedure named "MyProc"
	// As soon as the result of the "MyProc" server procedure
	// will be available, 
	// the "ProcResult" browser procedure will be run
	Res is int
	Res = AJAXExecuteAsynchronous(MyProc, ProcResult)
	// The following processes are run without waiting for the result
	...
	// Progress of the server procedure named "MyProc"
	IF AJAXAsynchronousCallPending(Res) = True THEN
		// Cancel the execution 
		// of the "ProcResult" browser procedure
		// The result of the "MyProc" server procedure 
		// cannot be processed
		AJAXCancelAsynchronousCall(Res)
	END
ELSE
	// Processes without using AJAX
	... 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AJAXCancelAsynchronousCall(<Identifier of server procedure>)`
---

**`<Identifier of server procedure>: (Integro)`**

Identificador del servidor Procedure cuyo resultado no será procesado. Este identificador fue definido por [AJAXExecuteAsynchronous](../WDLang2/3055115.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Recordatorio
<a name="recordatorio_ELTPARAGRAPHE000046"></a>

[AJAXAsynchronousCallPending](../WDLang2/3055111.md) se utiliza para averiguar si un procedimiento del servidor llamado por [AJAXExecuteAsynchronous](../WDLang2/3055115.md) se está ejecutando actualmente.
<a name="NOTE0_2"></a>


### Navegadores antiguos
<a name="navegadores_antiguos_ELTPARAGRAPHE000058"></a>

La función [AJAXAvailable](../WDLang2/3055112.md) permite determinar si el navegador actual soporta AJAX. Si un proceso que utiliza AJAX se ejecuta en un navegador que no es compatible con esta tecnología, el proceso se ejecuta "como si" no utilizara AJAX (por ejemplo, se actualiza toda la página).


## FileDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un fichero en el navegador del usuario Web.. Le permite visualizar en el browser de documentos de usuario Web que no son accesibles para el servidor Web.: archivos ocultos por razones de seguridad, memos binarios, ....

## Observaciones
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Elegir el destino del archivo
<a name="elegir_destino_del_archivo_ELTPARAGRAPHE000196"></a>

Para seleccionar el destino del archivo, se recomienda utilizar el parámetro &lt;Destino&gt; de la función **FileDisplay** en lugar de la función [ChangeTarget](../WDLang2/3058005.md). De hecho, en este caso, el código JavaScript correspondiente es más pequeño y el tamaño de sus páginas está optimizado.
<a name="NOTE0_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Recuperación de los parámetros pasados a una página de Active WEBDEV
<a name="recuperacion_los_parametros_pasados_una_pagina_active_webdev_ELTPARAGRAPHE000210"></a>

Se pueden utilizar varios métodos para recuperar los parámetros pasados a una página de Active WEBDEV:

1. Recuperando los parámetros en la "Declaración global" de la página de Event.

2. Recuperación de los parámetros con [PageParameter](../WDLang2/3058026.md).




Para obtener más información, consulte [Visualización de una página de Active WEBDEV desde otra página](../WDChamp/3539047.md).
<a name="NOTE0_3"></a>




### Descarga de archivos grandes
<a name="descarga_archivos_grandes_ELTPARAGRAPHE000229"></a>

La descarga de archivos con **FileDisplay** puede llevar más tiempo que la descarga de archivos directamente a través de un enlace http. 

Cuando se utiliza **FileDisplay**, el motor WEBDEV: 

1. Abre el archivo para descargar. 

2. Transfiere el archivo al servidor Web mientras solicita enviar el archivo al usuario Web.. 




Cuando se utiliza un enlace http, el navegador del usuario Web encuentra directamente el archivo a descargar.

**Para los archivos grandes, le recomendamos que utilice un enlace http**. En este caso, el ruta y el nombre del archivo que se va a descargar se pueden configurar con el [URL](../Proprietes/2510132.md) Property.
<a name="NOTE0_4"></a>


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000254"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".




## LooperToXML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un archivo XML con los datos de una control Looper (control Looper basada en un archivo de datos o rellenada mediante programación).

**Observación**: Default genera automáticamente una hoja de estilo (archivo ".XSL") al crear el archivo XML.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
sFile is string
// File name
sFile = fSelect("", "", "Choose the export file", ...
	"XML files (*.XML) *.xml", "xml", fSelCreate + fSelExist)
// The export is canceled if no name was specified
// Otherwise, export the content of the LOOP_Customer Looper control
IF sFile <> "" THEN LooperToXML(LOOP_Customer, sFile)
```

## Sintaxis

`<Result> = LooperToXML(<Looper control> , <Name of the XML file> [, <Options> [, <Start row> [, <End row>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Name of the XML file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo XML a crear. El archivo se sustituye por el nuevo si ya existe.

**`<Options>: (Constante opcional de tipo Integer)`**

Configura el archivo XML a crear.


|   |   |
| --- | --- |
| *taColumnsTitles* | El título de las columnas se inserta en el archivo. |
| *taNoTitle* | Sólo se copian los datos en el archivo. |
| *taNoXSL* | No se crea la hoja de estilo (archivo ".XSL") |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |



**`<Start row>: (Entero opcional)`**

Número de la fila desde la que se iniciará la exportación. 
Si no se especifica este parámetro, el inicio fila será el primer fila encontrado en el control Looper.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. 
Si no se especifica este parámetro, el final fila es el último fila encontrado en el control Looper.

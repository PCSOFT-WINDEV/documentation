
## ProgressBarEnable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita la actualización de una barra de progreso vía Ajax.

**Atención**: Esta función no puede ser usada en un sitio PHP o AWP. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000019562&name=progressbarenable_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Display the Progress Bar in the middle of the screen in a cell with DDW
CellDisplayDialog(CELL_ProgBar)
// Start monitoring the progress of the copy
ProgressBarEnable(PROGBAR_Copy)
```



```wl
// -- Server code of BTN_Process button
FOR I = PROGBAR_NoName1.MinValue _TO_ PROGBAR_NoName1.MaxValue
	PROGBAR_NoName1 = I
	Multitask(-1)
END

// -- Browser code of BTN_Process button
ProgressBarEnable(PROGBAR_NoName1, 10)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ProgressBarEnable(<Progress Bar control> [, <Period> [, <Automatic stop>]])`
---

**`<Progress Bar control>: (Nombre del control)`**

control Barra de progreso para usar.

**`<Period>: (Entero o duración opcional)`**

Período para refrescar la Barra de Progreso en centésimas de segundo. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, "1s", "10cs" o "10ms"). 


Este período es set a 1000 centésimas de segundo por Default.

**`<Automatic stop>: (Booleano opcional)`**

Indica si la actualización de la barra de progreso debe detenerse automáticamente (<u><u><u><u>True</u></u></u></u>, valor predeterminado) al final de la llamada al servidor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El actualización de la barra de progreso se realiza regularmente desde el navegador interrogando al servidor.

En el código del servidor, la posición de la barra de progreso se cambia modificando el valor (propiedad [Valor](../Proprietes/2510130.md)) de la propiedad control Barra de progreso.

Al ser provocada la actualización por el navegador, las modificaciones en el código del servidor pueden no aplicarse inmediatamente a la control del navegador.

**Atención**: No se debe llamar al servidor con demasiada frecuencia para evitar realizar demasiadas peticiones.

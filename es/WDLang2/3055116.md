


## Funciones de gestión AJAX
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de AJAX son las siguientes:



|   |   |
| --- | --- |
| [AJAXAsynchronousCallPending](../WDLang2/3055111.md) | Se utiliza para averiguar si un servidor Procedure llamado por [AJAXExecuteAsynchronous](../WDLang2/3055115.md) se ejecuta actualmente. |
| [AJAXAvailable](../WDLang2/3055112.md) | Se utiliza para saber si la tecnología AJAX es compatible con el navegador actual. |
| [AJAXCancelAsynchronousCall](../WDLang2/3055110.md) | Cancela la ejecución de automatic del navegador Procedure llamado por [AJAXExecuteAsynchronous](../WDLang2/3055115.md). |
| [AJAXExecute](../WDLang2/3055114.md) | Ejecuta un servidor Procedure sin refrescar el Page. |
| [AJAXExecuteAsynchronous](../WDLang2/3055115.md) | Ejecuta un servidor Procedure sin refrescar el Page. |

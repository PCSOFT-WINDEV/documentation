


## &lt;Image&gt;.LayerVisible (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se usa para hacer visible una capa que se encuentra en una imagen WDPIC o Invisible.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the product image in an Image control
PROCEDURE DisplayProduct(sImageControl is Control, sWDPicName is string, ...
			bPromo is boolean)

MyImage is WDPic
MyImage.FileName = sWDPicName

sImageControl.Value = MyImage
sImageControl..LayerVisible("Promo layer", bPromo)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <WDPic image>.LayerVisible(<Layer name> , <Visible>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe la capa,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si el layer no existe. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) permite conocer los detalles del error.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de [WDPic](../WDLang1/1000023503.md) Variable correspondiente al Image utilizado.

**`<Layer name>: (Cadena de caracteres)`**

Nombre de la capa a utilizar. Este nombre debe ser idéntico al nombre definido en el editor de Image.

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la capa debe ser visible,

- <u><u><u><u>False</u></u></u></u> si la capa debe ser Invisible.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Una capa puede hacerse visible o Invisible mediante programación sólo si la capa incluye la opción "Visible/Invisible mediante programación". Esta opción está disponible en el menú contextual de la capa. Esta opción puede ser activada: 

- en el editor de Image,

- en el editor de imágenes control.


Para obtener más información, consulte [Gestión de capas](../Editeurs/1410086606.md).

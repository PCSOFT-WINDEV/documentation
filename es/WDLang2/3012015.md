
## FolderWeb( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la ruta relativa del directorio que contiene:

- imágenes, 

- Archivos JavaScript,

- Archivos del applet Java,

- otros archivos accesibles desde el navegador.




Observación: Para get el directorio "&lt;nombre del proyecto&gt;_WEB", puede utilizar: 

- la función **FolderWeb**: esta función devuelve una ruta relativa para ser interpretada por el navegador. 

- la función [fWebDir](../WDLang1/3036052.md): esta función devuelve un ruta absoluto para ser utilizado desde el servidor con el [funciones de gestión de archivos externos (fxxx)](../WDLang1/3036034.md).

## Observaciones
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) 

### Directorio de imágenes (en código de navegador)
<a name="directorio_imagenes_codigo_navegador_ELTPARAGRAPHE000072"></a>

Por Default, "Localizar el Image en el directorio _WEB en el código del navegador" se comprueba al describir un Image. En este caso, en el código del navegador, el Image se busca automáticamente en el directorio _WEB del sitio. No se requiere ningún código específico.

Si "Localizar el Image en el directorio _WEB en el código del navegador" está desmarcado, se debe especificar el Image ruta completo para encontrar el Image. Para ello, utilice la función **FolderWeb**.
<a name="NOTE0_2"></a>




### Especificar el directorio de imágenes en WLanguage para generar el código HTML
<a name="especificar_directorio_imagenes_wlanguage_para_generar_codigo_html_ELTPARAGRAPHE000084"></a>

La función **FolderWeb** se puede utilizar para especificar el directorio de imágenes al generar el código HTML en WLanguage.

Si el código HTML debe contener referencias a imágenes o a cualquier otro objeto estático (archivo Java o JavaScript), se debe utilizar la función **FolderWeb** para localizar estos archivos.
<a name="NOTE0_3"></a>




### Instalación
<a name="instalacion_ELTPARAGRAPHE000098"></a>

En la instalación se realiza a través de medios físicos, el directorio de los archivos de datos se especifica durante la instalación.

En el caso de una instalación remota (FTP), el proveedor de INTERNET suministra el directorio principal de los archivos de datos y el usuario suministra el nombre del subdirectorio en el que están instalados los archivos de datos.

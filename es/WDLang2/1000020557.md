
## CapsLockVerify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si la tecla CapsLock está pulsada.

**Atención**: Esta función sólo debe utilizarse en el código del navegador de la tecla pulsada (onkeypress).. La comprobación se realizará cuando se escriba la primera letra.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a warning if the CapsLock key is pressed
IF CapsLockVerify() = True THEN Info("The CapsLock key is pressed.")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CapsLockVerify()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la tecla CapsLock está presionada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

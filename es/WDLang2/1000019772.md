


## LooperCollapseAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Colapsa todas las roturas expandidas en una control Looper. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the [4 stars only] Button control
LooperCollapseAll(LOOP_HOTEL)
nRow4Star is int
nRow4Star = LooperSearch(LOOP_HOTEL, ATT_NBSTARS, 4)
LooperExpand(LOOP_HOTEL, nRow4Star)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LooperCollapseAll(<Looper control>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es una cadena vacía (""), se utiliza la control Looper actual.




## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000048"></a>

La función **LooperCollapseAll** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.

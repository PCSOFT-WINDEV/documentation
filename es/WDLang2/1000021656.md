


## &lt;Cerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una Control Popup mostrada en la página a través de [&lt;Pantalla](../WDLang2/1000021654.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
POPUP_Question.Close()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Popup>.Close()`
---

**`<Popup>: (Nombre del control)`**

Nombre de la Control Popup que se ocultará.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [&lt;Pantalla](../WDLang2/1000021654.md) y **&lt;Cerrar** deben utilizarse en diferentes proceos/eventos. Si estas funciones se utilizan en la misma Process/Event, la Control Popup no se visualiza.

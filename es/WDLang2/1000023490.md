


## &lt;ExpandAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Expande todas las pausas en una control Looper con pausa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the [1.2 stars only] Button control
LOOP_HOTEL.ExpandAll()
LOOP_HOTEL.Collapse(nRow3Star)
LOOP_HOTEL.Collapse(nRow4Star)
LOOP_HOTEL.Collapse(nRow5Star)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Looper control>.ExpandAll()`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.




## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000167"></a>

La función **&lt;ExpandAll** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente.

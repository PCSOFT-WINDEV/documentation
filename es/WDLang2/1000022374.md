## Presentación
<a name="presentacion_ELTTEXTE000203"></a>
Google Analytics es una herramienta muy popular para medir el tráfico del sitio.

Google Analytics se utiliza para medir el número de visitas de cada página del sitio, el tiempo de carga de cada página, la navegación ruta, el tiempo de carga de los recursos de la página (imágenes), la tasa de rebote, el entorno del usuario Web (dispositivo, ubicación geográfica, idioma, navegador,...), la fuente de navegación (página de inicio, anuncio,...), etc.

La versión 22 de WEBDEV incluye automáticamente en sus sitios los seguidores necesarios para recopilar estadísticas en Google Analytics: sabrás todo sobre las actividades de los usuarios de la Web en tus sitios!

Si es necesario, usted tiene la capacidad de incluir rastreadores adicionales en sus sitios, para manejar casos especiales.: 

- descargando un archivo,

- usando aviones....




Esta página de documentación presenta: 

- [Cómo utilizar Google Analytics en sus sitios WEBDEV](#NOTE2_1). 

- [Los rastreadores adicionales disponibles (y las funciones de associated)](#NOTE3_1). 




Observación: Las páginas que tiene en cuenta Google Analytics son las páginas referenciables, es decir, las páginas estáticas, las páginas de Active WEBDEV y las páginas PHP.

## ¿Cómo incluir Google Analytics en sus sitios WEBDEV?
<a name="¿como_incluir_google_analytics_sus_sitios_webdev_ELTTEXTE000227"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000038"></a>

Para vincular Google Analytics con su sitio web, debe: 

- definir una cuenta en Google Analytics. Esta cuenta debe ser relacionado a su sitio. 

- recuperar el identificador de rastreo de Google Analytics associated con su cuenta. 

- especificar el identificador de seguimiento en el proyecto WEBDEV. 

- implementar el proyecto WEBDEV. 





### Registro en Google Analytics
<a name="registro_google_analytics_ELTPARAGRAPHE000048"></a>

Para vincular Google Analytics y su sitio web, se debe definir una cuenta en Google Analytics. Esta cuenta debe ser relacionado a su sitio. 

Para definir una cuenta relacionado a un sitio en Google Analytics: 

1. Ir al sitio: [https://www.google.fr/intl/fr/analytics/](https://www.google.fr/intl/fr/analytics/). Conéctese (por ejemplo, con su cuenta de Google). 

2. Regístrese si aún no lo ha hecho: haga clic en el botón "Registro. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_ana_1.gif)


3. Especifique las características de la cuenta de Google Analytics: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_ana_2.gif)


	- Seguimiento de un sitio web. 

- Nombre de la cuenta

- Nombre del sitio Web a rastrear. Este parámetro es obligatorio y se utiliza para identificar el sitio en el Google Analytics cuadro de mando. 

- URL del sitio Web. Este parámetro es obligatorio. Se utiliza para asociar el sitio web con Google Analytics.. 

- Otra información diversa. 

4. Entonces, haga clic en el botón "get un ID de rastreo". 

5. La cuenta se crea y se get un ID de seguimiento. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_ana_3.gif)






### Especificar el identificador de seguimiento en el proyecto WEBDEV
<a name="especificar_identificador_seguimiento_proyecto_webdev_ELTPARAGRAPHE000073"></a>

Para especificar el identificador de seguimiento en el proyecto WEBDEV: 

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. En la pestaña "General", copy el identificador de rastreo correspondiente a su sitio: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_Analytics%20-%20HC%20N%B0001.gif&type=thumb)





Puede desplegar su sitio. Las páginas estáticas, AWP y PHP serán tomadas en cuenta automáticamente por Google Analytics.

## Los "seguidores" adicionales disponibles
<a name="los_seguidores_adicionales_disponibles_ELTTEXTE000263"></a>
Además de las estadísticas generales de navegación administradas automáticamente por WEBDEV, usted tiene la capacidad de medir comportamientos específicos:

- Personalizado Event: ...presiona el botón de orden, pasa en un control, ...

- Interacción con la red social.

- Cronometraje personalizado: medir el tiempo que pasa en un Process.

- Excepción en la programación: error de programación.

- Vista de la página.





### Personalizado Event
<a name="personalizado_event_ELTPARAGRAPHE000109"></a>

El seguimiento de una Event personalizada se utiliza para medir la frecuencia de una acción específica realizada por el usuario de la Web: como hacer clic en un botón, descargar un archivo, hacer clic en un banner publicitario, reproducir un vídeo...

Para realizar este seguimiento, basta con llamar a [GglAnalyticsAddEvent](../WDLang2/1000022356.md) en el código correspondiente.


### Interacción con la red social
<a name="interaccion_con_red_social_ELTPARAGRAPHE000119"></a>

Este rastreador se utiliza para medir el número de clics realizados en un "botón social" incluido en la página: Como, Tweet, ...

Para realizar este seguimiento, basta con llamar a [GglAnalyticsAddSocialNetworkAction](../WDLang2/1000022358.md) en el código correspondiente.


### Cronometraje personalizado
<a name="cronometraje_personalizado_ELTPARAGRAPHE000129"></a>

Este rastreador le permite hacer estadísticas de get sobre el tiempo empleado durante los procesos o acciones específicas del usuario: medir el tiempo para mostrar una lista de productos, medir el tiempo para rellenar un formulario, ....

Para realizar este seguimiento, basta con llamar a [GglAnalyticsAddTiming](../WDLang2/1000022359.md) en el código correspondiente.


### Errores de programación
<a name="errores_programacion_ELTPARAGRAPHE000139"></a>

Este rastreador se utiliza para medir el número de errores inesperados en el código del navegador del sitio.. 

Todo lo que tiene que hacer es utilizar el comando "When Exception" de WLanguage y luego llamar a [GglAnalyticsAddException](../WDLang2/1000022360.md) cuando se cancele la excepción.


### Vista de la página
<a name="vista_pagina_ELTPARAGRAPHE000149"></a>

Este tipo de rastreador personalizado se utiliza para mejorar las vistas de las páginas complejas.: planos, ... 

Para realizar este seguimiento, basta con llamar a [GglAnalyticsAddPage](../WDLang2/1000022357.md).




## Funciones de gestión de páginas (sintaxis de prefijos)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las páginas (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;Celda&gt;.CloseDialog](../WDLang2/1000021653.md) | Oculta el control Celda mostrado previamente en la página a través de &lt;control Celda&gt;.DisplayDialog. |
| [&lt;Celda&gt;.DisplayDialog](../WDLang2/1000021652.md) | Muestra un control Celda en una página con un efecto OVI (Oscurecimiento de Ventanas Inactivas).. Esta función se utiliza para simular fácilmente una ventana de diálogo en el código del navegador, utilizando un control Celda en una página. |
| [&lt;Cerrar](../WDLang2/1000021656.md) | Oculta una ventana emergente que se muestra en la página a través de [&lt;Pantalla](../WDLang2/1000021654.md). |
| [&lt;Página&gt;. uso](../WDLang2/1000021664.md) | Muestra una página WEBDEV en el navegador del usuario Web y cierra todos los contextos de página actuales. |
| [&lt;Página&gt;.Address](../WDLang2/1000021660.md) | Se utiliza para averiguar la INTERNET Address de una página. |
| [&lt;Página&gt;.display](../WDLang2/1000021661.md) | Muestra una página web en el navegador del usuario. |
| [&lt;Página&gt;.DisplayDialog](../WDLang2/1000021663.md) | Muestra una página en modo modal. Esta función se utiliza para establecer un diálogo con el usuario. La página se muestra en primer plano mientras que la página de apertura se muestra en segundo plano, Grayed por el mecanismo de OVI. |
| [&lt;Página&gt;.existente](../WDLang2/1000021658.md) | Le permite saber si existe un contexto de página en el servidor (lo que significa si la página se abrió). |
| [&lt;Página&gt;.Refresh](../WDLang2/1000021659.md) | Actualiza una página mostrada en el navegador del usuario a partir del contexto en el servidor. |
| [&lt;Pantalla](../WDLang2/1000021654.md) | Muestra una ventana emergente en una Page con efecto OVI (Oscurecimiento de Ventanas Inactivas). |
| [&lt;Popup&gt;.Animate](../WDLang2/1000021655.md) | Muestra una ventana emergente en una celda de página. |
| [.ClosePopup](../WDLang2/1410087261.md) | Oculta una ventana emergente Page mostrada en la Page a través de [.DisplayPopup](../WDLang2/1410087260.md). |
| [.DisplayPopup](../WDLang2/1410087260.md) | Muestra una Page WEBDEV como una ventana emergente en la Page actual con un efecto OVI (Oscurecimiento de Ventanas Inactivas). |

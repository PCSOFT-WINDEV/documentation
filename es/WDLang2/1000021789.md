


## LooperRestorePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la posición de la barra de desplazamiento en un control Looper así como los elementos seleccionados.. Estos elementos han sido guardados por [LooperSavePositionAndSelection](../WDLang2/1000021788.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sSavePosition is string
sSavePosition = LooperSavePositionAndSelection(LOOP_Looper1)
LooperDisplay(LOOP_Looper1, taInit)
LooperRestorePositionAndSelection(LOOP_Looper1, sSavePosition)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LooperRestorePositionAndSelection(<Looper control> [, <Backup>])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Backup>: (Cadena de caracteres opcional)`**

Guardar características devueltas por [LooperSavePositionAndSelection](../WDLang2/1000021788.md). 
Si no se especifica este parámetro:

- Se utiliza el último guardado realizado por [LooperSavePositionAndSelection](../WDLang2/1000021788.md) en la control (si existe).

- Se produce un error si no existe ninguna copia de seguridad.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El nombre de la control utilizada para guardar ([LooperSavePositionAndSelection](../WDLang2/1000021788.md)) puede diferir del nombre utilizado para la operación de restauración. Esta característica puede ser useful para duplicar la posición y la selección en dos controles de Looper que se muestran alternativamente por ejemplo.

- Si el control está vacío, la función **LooperRestorePositionAndSelection** no tiene ningún efecto.




## ContextOpen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un nuevo contexto de página sin devolver la información al navegador.. Esta función se utiliza para pasar parámetros a una de las páginas del conjunto de marcos.

Si un contexto con el mismo nombre ya está abierto, éste se destruye automáticamente (se ejecuta el código de cierre de la página) y se abre.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the context of PAGE_Page4
ContextOpen(PAGE_Page4)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ContextOpen(<Page> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Page>: (Nombre de la página)`**

Nombre de la página cuyo contexto debe abrirse. Si este parámetro es igual a una cadena vacía (""), el nombre de la página corresponde a la página a la que pertenece el actual Process.

**`<Parameter 1>: (Tipo de valor enviado a la página (opcional))`**

Parámetros esperados por la apertura Process de la página a mostrar. Estos parámetros son pasados por reference y son considerados como variables globales a la página.



**`<Parameter N>: (Tipo de valor enviado a la página (opcional))`**

Parámetros esperados por la apertura Process de la página a mostrar. Estos parámetros son pasados por reference y son considerados como variables globales a la página.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ContextOpen, PageDisplay y PageRefresh
<a name="docparampagetitleshort_pagedisplay_pagerefresh_ELTPARAGRAPHE000044"></a>

**ContextOpen** tiene el mismo efecto que [PageDisplay](../WDLang2/3058008.md) pero no devuelve la página al navegador. Para devolver la página al navegador, utilice [PageRefresh](../WDLang2/3058011.md).
<a name="NOTE0_2"></a>


### Pasar parámetros a ContextOpen
<a name="pasar_parametros_docparampagetitleshort_ELTPARAGRAPHE000060"></a>

Al abrir un conjunto de marcos cuyas páginas esperan parámetros, debe (antes de mostrar el conjunto de marcos) abrir el contexto de las páginas del conjunto de marcos especificando los parámetros esperados por estas páginas.
<a name="NOTE0_3"></a>


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000070"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

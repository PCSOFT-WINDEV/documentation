


## PHPExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Llama a un script.php externo y devuelve el resultado en una cadena. Se realiza una petición HTTP: tienes la capacidad de usar un método POST o un método get.

Se recomienda utilizar esta función para ejecutar scripts PHP (en lugar de [ScriptExecute](../WDLang2/3012016.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Runs the "Script1.php" script found at the root 
// of the Web site and returns the result in S
MyBuffer is Buffer
MyBuffer = PHPExecute("/Script1.php")
```
<a name="Example2"></a>

```wl
// Login page in php. Sends 3 parameters with the POST method
// The script is found in the php directory
MyBuffer is Buffer
MyBuffer = PHPExecute("/php/Login.php", "Name=BOB&Pass=My+Secret&Lang=3", True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PHPExecute(<Script name> [, <"Name1=Param1&Name2=Param2&..."> [, <Runtime method>]])`
---

**`<Result>: (Buffer o cadena de caracteres)`**

Resultado del guión. En la mayoría de los casos, este resultado puede ser mostrado por [Pantalla StringDisplay](../WDLang2/3012001.md).

**`<Script name>: (Cadena de caracteres)`**

Address del guión. La URL del script puede tener el siguiente formato:

- "http://ServerName/Address.php": el script Address.php se encuentra en el ordenador "ServerName

- "/Scripts/MyScript.php": MyScript.php se encuentra en el servidor actual, en el subdirectorio "/Scripts" del sitio.

- tiene la posibilidad de utilizar un secure Address (https)




**`<"Name1=Param1&Name2=Param2&...">: (Cadena de caracteres)`**

Parámetros destinados al script. Esta cadena contiene para cada parámetro:

- el nombre del parámetro,

- el signo "=",

- el valor del parámetro.




Dos parámetros están separados por el signo &.

**`<Runtime method>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para ejecutar un método POST,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para ejecutar un método GET.

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000063"></a>

**PHPExecute** es más lento que [PHPDisplay](../WDLang2/3012021.md).
<a name="NOTE0_2"></a>


### Parámetros de codificación
<a name="parametros_codificacion_ELTPARAGRAPHE000075"></a>

El contenido de los parámetros se codifica automáticamente a get una URL válida:

- transformación en UTF-8 (según las opciones de la página y del proyecto).

- codificación de caracteres especiales.




## Funciones de control Barra de progreso
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de la barra de progreso en un sitio WEBDEV:



|   |   |
| --- | --- |
| [ProgressBarDisable](../WDLang2/1000019478.md) | Detiene la actualización periódica de una barra de progreso vía Ajax. |
| [ProgressBarEnable](../WDLang2/1000019477.md) | Habilita la actualización de una barra de progreso vía Ajax. |
| [ProgressBarExecute](../WDLang2/1000019481.md) | Inicia un largo navegador Process y llena una barra de progreso vía Ajax de acuerdo al progreso de esta Process. |

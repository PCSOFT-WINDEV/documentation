


## LooperInsert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una fila en una control Looper.

**Observación**: Se recomienda utilizar la función [LooperInsertLine](../WDLang2/3083002.md). Efectivamente, [LooperInsertLine](../WDLang2/3083002.md) le permite insertar:

- TAB en los atributos/controles.

- todos los tipos de valores en los atributos/controles.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para los controles de Looper sin atributos, le recomendamos que utilice [LooperInsertLine](../WDLang2/3083002.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the Looper control named LOOP_LOOPER1
LooperDeleteAll(LOOP_LOOPER1)
HReadFirst(Product, ProdName)
WHILE HOut() = False
	LooperInsert(LOOP_LOOPER1, Product.ProdName + TAB + Product.Code)
	HReadNext(Product, ProdName)
END
```
<a name="Example2"></a>

```wl
// Add values into the Looper control at position 12
LooperInsert(LOOP_Looper1, LastName + TAB + FirstName + TAB + PhoneNum + TAB + FaxNum, 12)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperInsert(<Looper control> , <Value> [, <Index>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la adición fue exitosa,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es una cadena vacía (""), la fila se añadirá en la control Looper actual .

**`<Value>: (Cadena de caracteres)`**

Lista de valores que se insertarán en la nueva fila. Este parámetro debe tener el siguiente formato:

```txt
<Value 1> + TAB + ... + <Value N>
```

Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Para no especificar ningún valor para un atributo, utilice una cadena vacía ("") o 0 (según el tipo de atributo).

**`<Index>: (Entero opcional)`**

Índice del fila a partir del cual se insertará el nuevo fila.

- Si este parámetro no se especifica o es igual a -1, el fila se insertará antes del fila actual . Si no se selecciona ninguna fila, la fila se insertará al final del control Looper.

- Si &lt;Index&gt; es mayor que el número de filas, la fila se insertará en la última posición del control Looper. En este caso la función **LooperInsert** equivale a la función [LooperAdd](../WDLang2/3083006.md).

- Si este parámetro es 0, la fila se insertará en la primera posición del control Looper.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000149"></a>

La función **LooperInsert** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Características de la fila
<a name="caracteristicas_fila_ELTPARAGRAPHE000163"></a>Si un valor corresponde al resultado de un cálculo, la expresión numérica debe figurar entre paréntesis.. Por ejemplo:

```wl
LooperInsert(LOOP_CalculationZone, (53+29) + TAB + (83-21), 12)
```

<a name="NOTE0_3"></a>


### Manejo de un memo binario
<a name="manejo_memo_binario_ELTPARAGRAPHE000178"></a>

Para insertar una nota binaria en una control Looper, utilice [LooperInsertLine](../WDLang2/3083002.md).
<a name="NOTE0_4"></a>


### clasificar
<a name="clasificar_ELTPARAGRAPHE000190"></a>

La función **LooperInsert** no tiene en cuenta el orden.




## LooperSelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de filas seleccionadas en una control Looper. Las líneas seleccionadas corresponden a las líneas actuales en las que se visualiza el banda de selección.

**Observación**: La función **LooperSelectCount** puede utilizarse en:

- a control Looper basada en un archivo de datos o rellenada mediante programación.

- una selección única o una selección múltiple control Looper.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
i is int 
NbSelected is int 
NbSelected = LooperSelectCount(LOOP_PRODUCT) 
FOR i = 1 TO NbSelected
	Trace("Selected row: " + i) 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperSelectCount(<Looper control>)`
---

**`<Result>: (Integro)`**



- Número de filas seleccionadas en el control Looper especificado,

- 0 si no se selecciona fila. 


Este número puede ser mayor que 1 para los controles de Looper de selección múltiple.

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se utiliza el control Looper al que pertenece el Event actual. 
Se produce un error WLanguage si este parámetro no corresponde al nombre de un control Looper.

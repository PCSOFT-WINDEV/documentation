


## LooperSelect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del elemento seleccionado en una control Looper. El elemento seleccionado corresponde al fila actual en el que se muestra el banda de selección.

**Observación**: La función **LooperSelect** puede utilizarse en:

- a control Looper basada en un archivo de datos o rellenada mediante programación.

- una selección única o multiselección control Looper. Para los controles Looper de selección múltiple, se debe utilizar la función **LooperSelect** en un bucle para obtener todas las filas seleccionadas.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Index of the selected row in "LOOP_Product"
ResSelect = LooperSelect(LOOP_Product)
```
<a name="Example2"></a>

```wl
// Rows selected in the multi-selection Looper control populated programmatically
i is int
NbSelected is int = LooperSelectCount(LOOP_Looper1)
FOR i = 1 TO NbSelected
	Trace("Selected element: " + LooperSelect(LOOP_Looper1, i))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperSelect(<Looper control> [, <Rank>])`
---

**`<Result>: (Integro)`**



- Índice del fila seleccionado en el control Looper especificado ,

- -1 si no se selecciona fila.




**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se utiliza la control Looper a la que pertenece la Event actual.

**`<Rank>: (Entero opcional)`**

Rango de selección para una multiselección control Looper. 
Si no se especifica este parámetro, se devuelve el índice de la primera fila seleccionada.
Por ejemplo, para obtener el primer elemento seleccionado, &lt;Rank&gt; = 1. 
Para recuperar el segundo elemento seleccionado,  = 2, ...




## LooperSearch( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un valor en un atributo de una control Looper.

Observación: Para realizar una búsqueda en todos los atributos de una control Looper, utilice [LooperSearchEverywhere](../WDLang2/1000021997.md). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Counts the names starting with "SMI" 
// The first one
Index is int = LooperSearch(ATT_NAME, "SMI", False)
Cnt is int = 0
WHILE Index <> -1
	Cnt++
	// Next one
	Index = LooperSearch(ATT_NAME, "SMI", False, Index + 1) 
END
Info("There are " + Cnt + " names starting with 'SMI'")
```
<a name="Example2"></a>

```wl
// Finds the position of customer #1234
Index is int = LooperSearch(ATT_IDCUSTOMER, "1234")
// If this customer was found
IF Index <> -1 THEN
	// Positions the Looper control on the found customer
	LOOP_LOOPER1 = Index
	// Retrieves the name of the customer (ATT_NAME is an "attribute" of LOOP_LOOPER1) 
	// and assigns it to a Static control
	STC_CAPTION1 = " Information about " + ATT_NAME
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LooperSearch(<Attribute> , <Sought value> [, <Type of search> [, <Start>]])`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado,

- -1 si la búsqueda falla.




**`<Attribute>: (Cadena de caracteres)`**

Atributo de la control Looper sobre la que se debe realizar la búsqueda.
Si este parámetro corresponde a una cadena vacía (""), la búsqueda se realiza en el atributo al que pertenece la Event actual.

**`<Sought value>: (Cadena de caracteres)`**

Valor buscado en el atributo especificado.

**`<Type of search>: (Constante opcional de tipo Integer)`**

Tipo de búsqueda que se realizará: 



|   |   |
| --- | --- |
| *searchContains* | **Búsqueda genérica de tipo "Contiene"**<br><br>Se devuelve el índice de los elementos que contienen  . <br> |
| *searchExactMatch* (o <u><u><u><u>True</u></u></u></u> para compatibilidad)<br>(Valor predeterminado) | **Búsqueda exacta** ("Igual a")<br><br>Se devuelve el índice de los elementos cuyo valor es estrictamente igual a  . |
| *searchStartsWith* (o <u><u><u><u>False</u></u></u></u> para compatibilidad) | **Búsqueda genérica de tipo "Comienza por"**<br><br>Se devuelve el índice de los elementos cuyo valor comienza por . |



**`<Start>: (Entero opcional)`**

Índice de inicio de la búsqueda (1 de Default).

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000178"></a>

La función **LooperSearch** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Sensibilidad a los casos
<a name="sensibilidad_los_casos_ELTPARAGRAPHE000187"></a>

La búsqueda no distingue entre mayúsculas y minúsculas, incluso si se trata de una búsqueda por coincidencia exacta.

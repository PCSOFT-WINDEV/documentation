


## PagePosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desplaza una página hacia arriba para posicionar un control en la sección Visible de la página (arriba) del navegador.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// LOOP_LOOPER is a Looper control. 
// Position on the last row of the Looper control
PagePosition("", LOOP_LOOPER, LOOP_LOOPER.Occurrence)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`PagePosition(<Page name> , <Control name> [, <Index>])`
---

**`<Page name>: (Cadena de caracteres)`**

Nombre de la página a desplazar. 
La página actual se utilizará si el nombre de la página corresponde a una cadena vacía ("""). **PagePosition** no tiene efecto si el nombre de la página es incorrecto.

**`<Control name>: (Cadena de caracteres)`**

Nombre del control a mostrar en la parte de Visible de la página especificada. **PagePosition** no tiene efecto si el control ya está visible en la página o si la página está completamente visible en el navegador. 

**`<Index>: (Entero opcional (para los controles de Looper y de Tabla que se rellenan mediante programación))`**

Índice de la fila que contiene la control. la función **PagePosition** no tiene ningún efecto: 

- si el control no es un Looper o control Tabla poblado programáticamente,

- si el índice es mayor que el número de filas en la Tabla especificada o control Looper poblada programáticamente.

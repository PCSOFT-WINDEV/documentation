


## &lt;SelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un fila en un control Looper. El fila seleccionado corresponde al fila actual en el que se muestra el banda de selección.

**Observación**: La función **&lt;SelectMinus** puede utilizarse en:

- a control Looper basada en un archivo de datos o rellenada mediante programación.

- una selección única o una selección múltiple control Looper.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselects row 5 in the "LOOP_Product" Looper control
LOOP_Product.SelectMinus(5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Looper control>.SelectMinus([<Index 1> [... [, <Index N>]]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.

**`<Index 1>: (Entero opcional)`**

Primer índice de la fila a deseleccionar. 
Si no se especifica este parámetro, se desmarcan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila no seleccionada, la función **&lt;SelectMinus** no tiene ningún efecto. Puede get el índice del fila seleccionado con [&lt;Seleccione](../WDLang2/1000023697.md).



**`<Index N>: (Enteros opcionales)`**

Enésimo índice de la fila a deseleccionar. 
Si no se especifica este parámetro, se desmarcan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila no seleccionada, la función **&lt;SelectMinus** no tiene ningún efecto. Puede get el índice del fila seleccionado con [&lt;Seleccione](../WDLang2/1000023697.md).

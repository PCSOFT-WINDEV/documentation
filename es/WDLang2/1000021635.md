


## PageReadHeaderHTTP( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee y devuelve una o más cabeceras HTTP recibidas por la Page actual (estas cabeceras son enviadas por el cliente al servidor en la petición actual).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sReferer is string = PageReadHeaderHTTP("Referer")
```
<a name="Example2"></a>

```wl
// Example for retrieving all incoming HTTP headers
// Browse the incoming HTTP headers
nHeaderSubscript is int
sAHeader is string
sAllHeaders is string

LOOP
	// Next header
	nHeaderSubscript ++
	sAHeader = PageReadHeaderHTTP(nHeaderSubscript)
	// Header read?
	IF sAHeader = "" THEN BREAK  // no other header
	sAllHeaders += [CR] + sAHeader
END
Info(sAllHeaders)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### La lectura de un encabezado identificado por su nombre

`<Result> = PageReadHeaderHTTP(<Header to read>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de cabecera. 

- Cadena vacía ("") si no se encuentra la cabecera.




**`<Header to read>: (Cadena de caracteres)`**

Nombre del encabezado a leer.


<a name="SYNTAX2"></a>

### Lectura de una cabecera identificada por su índice

`<Result> = PageReadHeaderHTTP(<Subscript>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de cabecera. El nombre de la cabecera se devuelve en mayúsculas y los guiones se sustituyen por "guiones bajos" (no está disponible el nombre inicial de la cabecera HTTP tal y como la envió el cliente). 

- Cadena vacía ("") si no hay cabecera que leer.




**`<Subscript>: (Integro)`**

Subíndice del encabezado para leer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En un sitio dinámico, las cabeceras son las cabeceras HTTP recibidas al conectarse al sitio.

- Para listar todas las cabeceras, utilice la sintaxis de índice hasta que la función devuelva una cadena vacía.

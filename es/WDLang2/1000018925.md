
## JSEndEvent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina la asociación entre una función del navegador WLanguage y una Event( implementada por [JSEvent](../WDLang2/1000018924.md)). La Procedure asociada a la Event no será llamada más.

**Observación**: La interceptación de eventos se detiene automáticamente cuando se cierra la página de associated.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Cancels the interception of all the clicks on the page
// gnClick is an integer returned by JSEvent
JSEndEvent(gnPageClick)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = JSEndEvent(<Event number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se canceló la interceptación del evento,

- <u><u><u><u>False</u></u></u></u> si se produce un error, (número de evento incorrecto, por ejemplo).




**`<Event number>: (Integro)`**

Número de la Event a Cancel. Este entero fue devuelto por [JSEvent](../WDLang2/1000018924.md).

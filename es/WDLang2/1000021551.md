
## HTMLClassToggle( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia una clase en las clases de HTML de un control: si la clase no existe, se añade; si la clase ya existe, se elimina.

Observación: También puede determinar y modificar el valor del atributo "class" de HTML de un control a través de la [HTMLClass](../Proprietes/1000021530.md) Property. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLClassToggle(EDT_Edit1, "InputInError")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLClassToggle(<Control> , <Class to toggle>)`
---

**`<Control>: (Nombre del control)`**

Nombre del control a manipular.

**`<Class to toggle>: (Cadena de caracteres)`**

Nombre de la clase que se va a alternar. Si una clase se encuentra n veces en la cuerda, esta clase se conmuta n veces.

Si se deben conmutar varias clases, este parámetro debe contener el nombre de las diferentes clases separadas por caracteres de espacio. Cada clase se alternará de forma independiente.

Atención: El nombre de las clases es sensible a las mayúsculas y minúsculas.


## JSInfoEvent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite a manipular las propiedades de JavaScript del navegador Event que desencadenó la ejecución del código. Permite: 

- leer un JavaScript Property de la Event.

- asignar un valor a una Property JavaScript de la Event. 




Esta función es una función avanzada: se requiere un buen conocimiento de Javascript.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the position of the mouse during a click
nXPos is int = JSInfoEvent("customerX")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperar el valor de un evento Property

`<Result> = JSInfoEvent([<Property>])`
---

**`<Result>: (Variant)`**

Valor de Property.

**`<Property>: (Cadena de caracteres opcional)`**

Property para usar. Si no se especifica este parámetro, la función **JSInfoEvent** recupera el objeto directamente. **JSInfoEvent** puede utilizarse como argumento de un navegador Procedure escrito en JavaScript.  


<a name="SYNTAX2"></a>

### Modificar una propiedad de la Event

`JSInfoEvent(<Property>)`
---

**`<Property>: (Cadena de caracteres)`**

Property para usar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los objetos "evento" son diferentes según los navegadores. Algunas propiedades no están disponibles para todos los navegadores.

- El Event que desencadenó la ejecución del código puede no estar disponible. Antes de utilizar esta Event, le recomendamos que compruebe su existencia con la siguiente Line de código por ejemplo:
	
	```wl
	IF JSInfoEvent() THEN ...
	```


- **JSInfoEvent** no debe utilizarse en procesos WEBDEV introducidos directamente en JavaScript: use el objeto **Event** directamente.

- La función **JSInfoEvent** no está disponible:

	- en los procedimientos de navegación.

	- en los eventos llamados directa o indirectamente desde un navegador procedure (a través de [ExecuteProcess](../WDLang1/3013031.md) por ejemplo).

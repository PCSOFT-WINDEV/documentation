
## LocalStorageValueName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de un valor para el almacenamiento local.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FOR i = 1 TO LocalStorageOccurrence()
	Trace(LocalStorageGet(LocalStorageValueName(i)))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LocalStorageValueName(<Subscript> [, <Persistent>])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre del valor encontrado en el almacenamiento local y correspondiente al índice especificado.

**`<Subscript>: (Integro)`**

Subíndice de valor cuyo nombre será recuperado. La índice del primer valor es set a 1. Esta índice corresponde a la orden de adición en el almacenamiento local.. 

**`<Persistent>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para trabajar en el almacenamiento local persistente (valor predeterminado),

- <u><u><u><u>False</u></u></u></u> para trabajar en el almacenamiento local a una ventana de navegador.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Almacenamiento local persistente o no
<a name="almacenamiento_local_persistente_ELTPARAGRAPHE000043"></a>

Durante un almacenamiento local persistente, los valores almacenados pueden ser compartidos por todas las ventanas y pestañas del mismo navegador en un ordenador.: por lo tanto, se pueden recuperar una vez que el navegador se ha cerrado y se ha vuelto a abrir o en otra ventana del navegador.. 

Durante un almacenamiento local no persistente, los valores almacenados son específicos de una ventana (o de una pestaña) del navegador.. en este caso, los valores se perderán cuando se cierre la ventana (o pestaña) del navegador. Por el contrario, estos valores pueden ser leídos por las ventanas del navegador abiertas desde la ventana donde se han almacenado los valores..


## BrowserMobile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite averiguar si el navegador utilizado por el usuario de la Web se inicia en un dispositivo móvil.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Redirects to the mobile page
IF BrowserMobile() = True THEN
	PageDisplay(PAGE_Home_Mobile)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserMobile()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el navegador se inicia en un dispositivo móvil. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las tabletas no se consideran como dispositivos móviles.

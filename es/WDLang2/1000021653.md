


## &lt;Celda&gt;.CloseDialog (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una control Celda previamente mostrada en la página a través de [&lt;Celda&gt;.DisplayDialog](../WDLang2/1000021652.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the question Cell control
CELL_Question.CloseDialog()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Cell control>.CloseDialog()`
---

**`<Cell control>: (Nombre del control)`**

Nombre de la celda a ocultar. Esta celda es superponible y se mostró con [&lt;Celda&gt;.DisplayDialog](../WDLang2/1000021652.md). De lo contrario, la función **&lt;Celda&gt;.CloseDialog** no realiza ninguna acción.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [&lt;Celda&gt;.DisplayDialog](../WDLang2/1000021652.md) y **&lt;Celda&gt;.CloseDialog** deben utilizarse en eventos/procesos diferentes. Si estas funciones se utilizan en el mismo evento/Process, el control Celda no se muestra.

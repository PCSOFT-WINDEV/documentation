


## LooperDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una fila de una control Looper. La control Looper se actualiza automáticamente. 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete row 1
LooperDelete(LOOP_ProdLooper, 1)
```
<a name="Example2"></a>

```wl
// Delete the current row
LooperDelete(LOOP_ProdLooper)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LooperDelete(<Looper control> [, <Index> [, <Animation>]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre de la control Looper que se va a utilizar.
Si este parámetro es igual a una cadena vacía ("""), la eliminación se realizará en el control Looper actual.

**`<Index>: (Entero opcional)`**

Índice de la fila a eliminar.

- Si no se especifica este parámetro, el borrado se realizará en el fila actual.

- Si &lt;Index&gt; es mayor que el número de filas, **LooperDelete** no tiene efecto.




**`<Animation>: (Constante opcional de tipo Integer)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Gestión de la animación durante el borrado: 



|   |   |
| --- | --- |
| *loopAnimationDelete* | Reproduce una animación durante el borrado. Esta animación corresponde a un efecto de desaparición al reducir la altura de la fila. |
| *loopAnimationNone*<br>(Valor predeterminado) | No se reproduce ninguna animación durante el borrado. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000134"></a>

La función **LooperDelete** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 





### Utilización con una control Looper basada en un archivo de datos
<a name="utilizacion_con_una_control_looper_basada_archivo_datos_ELTPARAGRAPHE000146"></a>

Si el control Looper está basado en un archivo de datos: 

- el Record se borra de el archivo de datos. 

- se selecciona la siguiente fila (si se borra la última fila, se seleccionará la fila anterior). El Record seleccionado en el archivo de datos de relacionado corresponde al fila seleccionado.


## BrowserWidth( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ancho (en píxeles) del área de cliente donde se muestra la página.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If there is not enough available size
IF BrowserWidth() < 500 THEN
...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserWidth()`
---

**`<Result>: (Integro)`**



- En el modo de zonificación, corresponde a la anchura del área de clientes en píxeles.. 

- En modo "Responsive Web Design", corresponde al ancho en píxeles del rango en curso en una página.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Este ancho incluye las posibles barras de desplazamiento.

- Para una página mostrada en un "iFrame" control, el ancho devuelto es el correspondiente a la dimensión del iFrame control.




<a name="XComponent"></a>

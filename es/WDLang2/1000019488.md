
## BrowserVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la versión del navegador utilizado por el usuario web.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a specific page for IE9 
IF BrowserPlatform() = BrowserTypeInternetExplorer THEN
	SWITCH BrowserVersion(BrowserMajorVersion)
	CASE 9
		PageDisplay(PAGE_Form_IE9)
	OTHER CASE
		PageDisplay(PAGE_Form_IE)
	END
ELSE
	PageDisplay(PAGE_Form)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserVersion([<Information>])`
---

**`<Result>: (Variant)`**



- Versión del navegador (el tipo depende de la información solicitada):

	- Si se usó la constante *BrowserFullVersion*, &lt;Resultado&gt; es una cadena de caracteres. 

	- Si se usó la constante *BrowserMajorVersion*, &lt;Resultado&gt; es un número entero.




- 0 si el navegador utilizado por el usuario web es desconocido. 




**`<Information>: (Constante opcional de tipo Integer)`**

Información solicitada:



|   |   |
| --- | --- |
| *BrowserFullVersion* | Número de versión completa del navegador utilizado por el usuario web. |
| *BrowserMajorVersion* <br>(Valor predeterminado) | Número de versión principal del navegador utilizado por el usuario web. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En el caso de un sitio dinámico (sitio no AWP), el valor devuelto corresponde al navegador utilizado al conectarse al sitio. Si el usuario cambia de navegador (a través de una copy-pegar la URL en otro navegador por ejemplo), este cambio no se detecta.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

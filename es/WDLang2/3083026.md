


## LooperSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un fila en un control Looper. El fila seleccionado corresponde al fila actual en el que se muestra el banda de selección.

**Observación**: La función **LooperSelectMinus** puede utilizarse en:

- a control Looper basada en un archivo de datos o rellenada mediante programación.

- una selección única o una selección múltiple control Looper.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselects row 5 in the "LOOP_Product" Looper control
LooperSelectMinus(LOOP_Product, 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LooperSelectMinus(<Looper control> [, <Index 1> [... [, <Index N>]]])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro corresponde a una cadena vacía (""), el control Looper utilizado es el control Looper al que pertenece el Event actual.

**`<Index 1>: (Entero opcional)`**

Primer índice de la fila a deseleccionar. 
Si no se especifica este parámetro, se desmarcan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila no seleccionada, la función **LooperSelectMinus** no tiene ningún efecto. Puede get el índice del fila seleccionado con [LooperSelect](../WDLang2/3083024.md).



**`<Index N>: (Enteros opcionales)`**

Enésimo índice de la fila a deseleccionar. 
Si no se especifica este parámetro, se desmarcan todas las líneas que se encuentran en el control Looper especificado.. Si este parámetro corresponde a una fila no seleccionada, la función **LooperSelectMinus** no tiene ningún efecto. Puede get el índice del fila seleccionado con [LooperSelect](../WDLang2/3083024.md).

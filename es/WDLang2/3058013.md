


## PageExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si la página especificada se muestra actualmente en el navegador del usuario Web.

Permite saber si una página se muestra en un marco distinto al actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the "PAGE_Registration" page is displayed
PageExist(PAGE_Registration)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PageExist(<Page>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la página se muestra actualmente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Page>: (Nombre de la página)`**

Nombre de la página cuya existencia debe comprobarse.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para comprobar si una página está abierta en el servidor, utilice [ContextExist](../WDLang2/3058002.md).

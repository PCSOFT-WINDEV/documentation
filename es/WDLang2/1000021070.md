
## jQuery (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un método JavaScript (o varios métodos encadenados) de la biblioteca jQuery en un elemento de la página.

Esta función es una función avanzada: Se requiere un buen conocimiento de JavaScript y de la biblioteca jQuery.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// A single jQuery function is used 
jQuery(STC_1).fadeOut()
```


<a name="Example2"></a>

```wl
// Use with 2 chained functions
// The opacity of control is modified, then the control explodes before it disappears
jQuery(CELL_Square).fadeTo("opacity",0.1).effect("explode")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Ejecutando un comando de la biblioteca jQuery

`<Result> = jQuery(<WEBDEV control>).<jQuery function>(<Parameter 1>, ..., <Parameter N>)`
---

**`<Result>: (Tipo esperado)`**

Valor de retorno del método jQuery llamado.

**`<WEBDEV control>: (Nombre del control)`**

Nombre de control que se utilizará en la página. 

Observación: La opción "El control se puede superponer" debe estar seleccionada en el control (pestaña "UI" de la ventana de descripción del control). 

**`<jQuery function>: (Cadena de caracteres)`**

Nombre de la función jQuery a ejecutar. 

Atención: Como JavaScript distingue entre mayúsculas y minúsculas, el nombre de la función debe respetar las mayúsculas y minúsculas utilizadas en la biblioteca jQuery. 

**`<Parameter 1>: (Cadena de caracteres)`**

Primer parámetro esperado por la función jQuery. 

**`<Parameter N>: (Cadena de caracteres)`**

Enésimo parámetro esperado por la función jQuery. 


<a name="SYNTAX2"></a>

### Ejecutando varios comandos encadenados de la biblioteca jQuery

`<Result> = jQuery(<WEBDEV control>).<jQuery function 1>(<Parameter 1>, ..., <Parameter N>).<jQuery function 2>(<Parameter 1>, ..., <Parameter N>)`
---

**`<Result>: (Tipo esperado)`**

Valor de retorno del método jQuery llamado.

**`<WEBDEV control>: (Nombre del control)`**

Nombre de control que se utilizará en la página.

**`<jQuery function 1>: (Cadena de caracteres)`**

Nombre de la primera función jQuery a ejecutar. Atención: Dado que JavaScript distingue entre mayúsculas y minúsculas, el nombre de la función debe respetar las mayúsculas y minúsculas utilizadas en la biblioteca jQuery.

**`<Parameter 1>: (Cadena de caracteres)`**

Primer parámetro esperado por la función jQuery.

**`<Parameter N>: (Cadena de caracteres)`**

Nth esperado por la función jQuery.

**`<jQuery function 2>: ()`**

Nombre de la segunda función jQuery a ejecutar. Atención: Dado que JavaScript distingue entre mayúsculas y minúsculas, el nombre de la función debe respetar las mayúsculas y minúsculas utilizadas en la biblioteca jQuery.

## Observaciones


### Parámetros de la función jQuery
<a name="parametros_funcion_jquery_ELTPARAGRAPHE000069"></a>

Si hay que pasar una Procedure ("Callback") a una de las funciones de jQuery, hay que pasar la Procedure del navegador por la reference (nombre de la función precedido de "&"). Por ejemplo: 

```wl
jQuery(BTN_Add).on("click", &OnAddition)
```



### Cargando la librería jQuery y sus extensiones
<a name="cargando_libreria_jquery_sus_extensiones_ELTPARAGRAPHE000076"></a>

Con WEBDEV se proporciona una librería jQuery pero hay que incluir otra librería en sustitución o incluir extensiones (jQuery-UI por ejemplo). 

Es posible: 

- incluir el script local o remoto (método recomendado). 

- usar la etiqueta "&lt;script src = ....&gt;. 




**Método 1: Incluyendo el archivo de script (local o remoto (URL))** (método recomendado)

1. Mostrar la ventana description de la página ("description" en el menú contextual). 

2. Seleccione la pestaña "Avanzado. 

3. Seleccione la pestaña "Recursos externos (JavaScript/CSS)".

4. A reference: 

	- un script remoto, copie/pegue la Address (URL) del archivo de script en la columna "Recursos JavaScript. 

	- un script local (archivo que se encuentra en su ordenador, por ejemplo), haga clic en el botón "+" de "Recursos JavaScript" y seleccione el archivo correspondiente. 





Atención: Comprobar el orden de las dependencias de los scripts (la librería jQuery y luego sus extensiones). 

**Método 2: Añadir el etiquetado para la inserción de scripts**: 

1. Mostrar la ventana description de la página ("description" en el menú contextual). 

2. Seleccione la pestaña "Avanzado. 

3. Seleccione la pestaña "HTML. 

4. copy las siguientes líneas de código en la sección "Código HTML añadido a la cabecera de la página. Por ejemplo: 
	
	```html
	<script src="http://code.jquery.com/jquery-1.9.1.js"></script>
	<script src="http://code.jquery.com/ui/1.10.3/jquery-ui.js"></script>
	```





Atención: 

- El código debe insertarse en la sección "Código HTML añadido al encabezado de la página" y no en la sección "Código HTML añadido al final de la página".. Exception: Si desea incluir una extensión sin incluir otra biblioteca jQuery, debe: 

	- Usar el método que utiliza el etiquetado. 

	- copy el código en la sección "Código HTML añadido al final de la página".




- Este método no se recomienda porque la carga del script interrumpe la carga de la página. La carga y la visualización de la página se ralentizan.

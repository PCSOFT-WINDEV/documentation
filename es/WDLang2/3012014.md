
## SSLActive( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite activar o desactivar el modo secure SSL.

**Atención**: se requiere un SSL Certificate activo para usar esta función. Para obtener más información, consulte [Transacciones de secure](../WDLang2/3012011.md) y [Obtención de una Certificate SSL (IIS versión 2 a 5)](../WDLang2/3012012.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- browser click code of button
SSLActive(True)

// -- server click code of button 
PageDisplay(PAGE_Secure_Page1)
// "PAGE_Secure_Page1" is opened in secure mode. 
// All the actions performed from this page
// will be secured
```


<a name="Example2"></a>



```wl
// -- browser click code of button
SSLActive(False)
// -- server click code of button 
PageDisplay(PAGE_Page1)
// "PAGE_Page1" is opened in standard mode. 
// All the actions performed from this page
// will not be secured
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SSLActive(<Mode>)`
---

**`<Mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe activar el modo SSL,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000050"></a>

Esta función sólo se puede utilizar: 

- en el código de clic de los botones que realizan una operación de "Enviar el valor de los control al servidor (submit)" (pestaña "General" en la ventana description de la control). 

- en el código de selección de la opción de menú "Enviar el valor de los controles al servidor (submit)" (pestaña "General" en la ventana de description de la opción de menú).

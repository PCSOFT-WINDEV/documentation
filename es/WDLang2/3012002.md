


## JSMethod( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para ejecutar un método JavaScript en un elemento que se encuentra en la página actual.

Esta función es una función avanzada: se requiere un buen conocimiento de Javascript.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`JSMethod(<Full element name> , <Method> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Full element name>: (Cadena de caracteres)`**

"JavaScript" nombre del elemento a utilizar. En la mayoría de los casos, este nombre comienza con "Document".. Debe utilizar los alias de los controles. Para más detalles, ver Observaciones.

**`<Method>: (Cadena de caracteres)`**

Método del elemento a utilizar. El nombre del método debe especificarse directamente: no puedes usar una cadena construida.

**`<Parameter 1>: (Cadena de caracteres opcional)`**

Parámetros del método.



**`<Parameter N>: (Cadena de caracteres opcional)`**

Parámetros del método.

## Observaciones


### JSMethod maneja los alias de los controles
<a name="jsmethod_maneja_los_alias_los_controles_ELTPARAGRAPHE000040"></a>

El lenguaje JavaScript distingue entre mayúsculas y minúsculas para los identificadores: el nombre completo del elemento debe respetar el caso definido para los alias de control: todas las páginas y los alias de control están escritos en mayúsculas.

Para construir el nombre del elemento nombre, debes saber el alias control. Este alias corresponde al nombre de control en la página HTML. Para encontrar este alias:

- en el editor de la página, pasa el cursor por el elemento para mostrar el alias usado en la punta de la herramienta control.

- en el editor de código, pulse Mayúsculas + F1.

- mediante programación, utilice la propiedad [Alias](../Proprietes/2510004.md).




Al construir el nombre completo del elemento, tiene la posibilidad de utilizar "_PAGE_" para acceder a la página.




## &lt;Popup&gt;.Animate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un Control Popup en un control Celda de la página. Si la celda ya contiene una Control Popup, se reproduce una animación durante la transición.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
POPUP_Details.Animate(CELL_Main)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Popup control>.Animates(<Destination Cell control> [, <Animation> [, <Duration>]])`
---

**`<Popup control>: (Nombre del control)`**

Nombre del control Popup a mostrar.

**`<Destination Cell control>: (Nombre del control)`**

Nombre de control Celda donde se mostrará la ventana emergente.

**`<Animation>: (Entero opcional)`**

Animación utilizada: 


|   |   |
| --- | --- |
| *animCoverUp* | Deslice el Control Popup hacia arriba para cubrir el contenido anterior. |
| *animSlideLeft*<br>(Valor predeterminado) | Deslice el Control Popup hacia la izquierda. |
| *animSlideRight* | Deslice el Control Popup hacia la derecha. |
| *animUncoverDown* | Deslice el contenido anterior hacia abajo y descubra la Control Popup. |



**`<Duration>: (Entero o duración opcional)`**

Duración de la animación en centésimas de segundo (100 centésimas de segundo que significa 1 segundo por Default). Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000205"></a>

Al llamar la función, si:

- No se visualiza ningún Control Popup en el control Celda:

	- El contenido de la control Celda se destruye.

	- El Control Popup se muestra inmediatamente (sin animación).




- En la celda se visualiza un Control Popup:

	- La nueva Control Popup se muestra gradualmente utilizando la animación especificada.







Los controles de Celda y Popup deben tener las siguientes características:

- Los controles Cell y Popup deben estar en la misma página.

- El desbordamiento horizontal de la control Celda debe ser Invisible.

- Los controles Popup y Cell deben tener el mismo ancho.

- Los controles Popup y Cell deben tener al menos la misma altura.



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000230"></a>

- Cuando se muestra un Control Popup por primera vez en un control Celda, se borra el contenido del control Celda.

- Si un Control Popup aparece en un control Celda, el Control Popup ya no puede ser utilizado a través de la programación, excepto con [&lt;Popup&gt;.Animate](../WDLang2/1000021655.md).

- El Control Popup mostrado no se mantiene cuando se vuelve a mostrar la página.. Si se utiliza un control Botón servidor (no AJAX) para actualizar la página, se debe utilizar la función **&lt;Popup&gt;.Animate** para volver a mostrar el control Popup correcto.

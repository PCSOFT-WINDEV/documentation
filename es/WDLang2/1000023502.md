


## &lt;Looper&gt;.BreakIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de la cabecera y el pie de página de una pausa dada en una control Looper: 

- El índice de la cabecera de una ruptura determinada también corresponde al índice de la primera fila de esta ruptura.

- El índice del pie de página de una pausa determinada también corresponde al índice de la última fila de esta pausa.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The LOOP_Invoices Looper control lists the invoices of each customer
// A break is performed for each customer
// The break footer contains the TotalInvoice control 
// linked to the ATT_Total attribute
// This control displays the total of all invoices for each customer
MyRow is int = 1	// Beginning of the Looper control
BreakHeader is int		// Index of break header
BreakFooter is int		// Index of break footer
Index is int		// Index of rows
Total is int		// Total for each break

// Calculate the totals for the entire Looper control
WHILE MyRow <= LOOP_Invoices.Count
	// Index of the current break
	BreakHeader = BRK_CustHeader.BreakIndex(MyRow)
	BreakFooter = BRK_CustFooter.BreakIndex(MyRow)

	// Browse the rows found in the current break
	// to calculate the total
	FOR Index = BreakHeader TO BreakFooter
		Total += ATT_Value[Index]
	END

	// Assign the total
	ATT_Total[BreakFooter] = Total

	// Go to the next break
	MyRow = BreakFooter + 1
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Break header/footer>.BreakIndex([<Subscript>])`
---

**`<Result>: (Integro)`**



- Índice de la cabecera/pie de ruptura especificada en la ruptura que contiene el fila cuyo índice es &lt;Índice>, 

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Break header/footer>: (Nombre del control)`**

Nombre de la cabecera o pie de página de la pausa que se va a utilizar. 
Este parámetro puede corresponder a:

- el nombre de la cabecera o pie de página de la ruptura directamente.

- el &lt;Nombre de control Looper&gt;.&lt;Nombre de la cabecera/pie de página de la ruptura&gt;




**`<Subscript>: (Entero opcional)`**

Índice de una de las filas en la pausa deseada.
Si no se especifica este parámetro, se devuelve el índice del encabezado/pie de ruptura que contiene la fila actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000216"></a>

La función **&lt;Looper&gt;.BreakIndex** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### Utilización de la función &lt;Looper&gt;.BreakIndex
<a name="utilizacion_funcion_docparampagetitleshort_ELTPARAGRAPHE000240"></a>

Se recomienda llamar a la función **&lt;Looper&gt;.BreakIndex** después de mostrar todo el control Looper. De lo contrario, los índices devueltos por la función **&lt;Looper&gt;.BreakIndex** pueden no corresponder con los índices que se muestran realmente en el control Looper.

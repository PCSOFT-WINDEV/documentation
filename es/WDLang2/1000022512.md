


## BannerStopSliding (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene el deslizamiento de automatic en una pancarta deslizante control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stop sliding (stays on the current banner)
BannerStopSliding(BAN_Banner)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BannerStopSliding(<Sliding Banner control>)`
---

**`<Sliding Banner control>: (Nombre del control)`**

Nombre de la pancarta deslizante control para usar.




## CellCloseDialog (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta una control Celda previamente mostrada en la página a través de [Diálogo CellDisplayDialog](../WDLang2/1000017139.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the question Cell control
CellCloseDialog(CELL_Question)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`CellCloseDialog(<Cell control>)`
---

**`<Cell control>: (Nombre del control)`**

Nombre de la celda a ocultar. Esta celda es superponible y se mostró con [Diálogo CellDisplayDialog](../WDLang2/1000017139.md). De lo contrario, la función **CellCloseDialog** no realiza ninguna acción.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los navegadores no actualización la visualización en medio de un código JavaScript: las funciones [Diálogo CellDisplayDialog](../WDLang2/1000017139.md) y **CellCloseDialog** deben utilizarse en eventos/procesos diferentes. Si estas funciones se utilizan en el mismo evento/Process, el control Celda no se muestra.

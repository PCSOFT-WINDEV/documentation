


## LooperModify (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia los valores de los atributos en una fila de una control Looper. 

**Observación**: Se recomienda utilizar la función [LooperModifyLine](../WDLang2/3083015.md). Efectivamente, [LooperModifyLine](../WDLang2/3083015.md) le permite insertar:

- TAB en los atributos/controles.

- todos los tipos de valores en los atributos/controles.

Para obtener más información, consulte ["Modificando un control Looper sin atributos](#NOTE0_4). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify a row in a Looper control
LooperModify(LOOP_Looper1, ProductName + TAB + ProductCode, Index)
```
<a name="Example2"></a>

```wl
// Modifies a value in the Looper control at position 12
LooperModify(LOOP_Looper1, LastName + TAB + FirstName + TAB + ...
	PhoneNum + TAB + FaxNum, 12)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LooperModify(<Looper control> , <Value> [, <Index>])`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.
Si este parámetro es igual a una cadena vacía (""), la modificación se realizará en la control Looper actual .

**`<Value>: (Cadena de caracteres)`**

Lista de valores modificados. Este parámetro debe tener el siguiente formato: 

```txt
<Value 1> + TAB + ... + <Value N>
```

Los diferentes valores deben ser especificados en el orden de los atributos (ventana control Looper description).
Para no especificar ningún valor para un atributo, utilice una cadena vacía ("") o 0 (según el tipo de atributo).

**`<Index>: (Entero opcional)`**

Índice de la fila a modificar.

- Si no se especifica este parámetro, la modificación se realizará en la unidad actual.

- Si &lt;Índice&gt; es superior al número de filas, la función **LooperModify** equivale a la función [LooperAdd](../WDLang2/3083006.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000145"></a>

La función **LooperModify** puede utilizarse en: 

- a control Looper basada en un archivo de datos, 

- a control Looper poblada programáticamente. 



<a name="NOTE0_2"></a>


### clasificar
<a name="clasificar_ELTPARAGRAPHE000159"></a>

Si **LooperModify** modifica un atributo de clasificación del control Looper actual, se eliminará el orden de clasificación set con este atributo.
<a name="NOTE0_3"></a>


### Manejo de un memo binario
<a name="manejo_memo_binario_ELTPARAGRAPHE000170"></a>

Para modificar una nota binaria en un control Looper, utilice [LooperModifyLine](../WDLang2/3083015.md).

### Modificar un control Looper sin atributos
<a name="modificar_control_looper_sin_atributos_ELTPARAGRAPHE000184"></a>

Para modificar una control Looper sin atributos, se recomienda cambiar directamente el valor de los controles correspondientes.

Por ejemplo:

```wl
// Modify the value of controls
// in "LOOP_Looper1" for row 23
LOOP_Looper1[23].LastNameControl = "Hobbes"
LOOP_Looper1[23].FirstNameControl = "Samantha"
```

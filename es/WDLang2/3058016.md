


## Uso de la página (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una página WEBDEV en el navegador del usuario y cierra todos los contextos de página actualmente abiertos en el servidor.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch the application to English
Nation(nationEnglish)
// Redisplay the PAGE_Login page with captions in English
PageUse(PAGE_Login)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`PageUse(<Page> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Page>: (Nombre de la página)`**

Nombre de la página a mostrar.
Si este parámetro corresponde a una cadena vacía (""), el nombre de la página es el nombre de la página a la que pertenece el actual Event.
En el modo de prueba, especifique la ruta completa de la página si la página que desea abrir se encuentra en un directorio que no sea el directorio del proyecto.

**`<Parameter 1>: (Tipo de valor enviado a la página (opcional))`**

Parámetros a pasar a la apertura Event de la página a abrir ("Declaraciones globales" Event de la página). Estos parámetros son pasados por reference y son considerados como variables globales a la página.



**`<Parameter N>: (Tipo de valor enviado a la página (opcional))`**

Parámetros a pasar a la apertura Event de la página a abrir ("Declaraciones globales" Event de la página). Estos parámetros son pasados por reference y son considerados como variables globales a la página.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Parámetros pasados a la página para abrir
<a name="parametros_pasados_pagina_para_abrir_ELTPARAGRAPHE000076"></a>Los parámetros se recuperan en la apertura Event de la página ("Declaraciones globales" Event de la página). Simplemente escriba la siguiente línea al principio de las "declaraciones globales" Event:


```txt
PROCEDURE <Page name> (<Parameter 1> [, ...[, <Parameter N>]]])
```

<a name="NOTE0_2"></a>


### Sitios multilingües
<a name="sitios_multilingues_ELTPARAGRAPHE000087"></a>

**Uso de la página** puede utilizarse en sitios multilingües. Esta función sirve para reabrir una página teniendo en cuenta el idioma especificado por [Nación](../WDLang1/3054014.md).
<a name="NOTE0_3"></a>


### Principio para abrir una página
<a name="principio_para_abrir_una_pagina_ELTPARAGRAPHE000101"></a>

**Uso de la página** hace lo siguiente:

- cerrar todo el contexto abierto en el servidor

- crear un contexto de página en el servidor

- devolver la página al navegador del usuario web.




Observación: Si se utiliza la función **Uso de la página** en el código de un control Botón, la página se abrirá en el destino definido para este control Botón(pestaña "General" de la descripción del control o función [ChangeTarget](../WDLang2/3058005.md)).
<a name="NOTE0_4"></a>


### Eventos y códigos que se ejecutan cuando se llama a Uso de la página
<a name="eventos_codigos_que_ejecutan_cuando_llama_docparampagetitleshort_ELTPARAGRAPHE000123"></a>

Los siguientes eventos se ejecutan cuando se muestra la página:

- Código de apertura de la página ("Declaraciones globales" Event de la página). 

- "Iniciando" Event de los controles de la página. 

- Mostrar página en el navegador.

- Código Javascript associated con la página y los controles.

- Código WLanguage después de la llamada a **Uso de la página** (si se utiliza).



<a name="NOTE0_5"></a>

### Abrir una página de un componente externo
<a name="abrir_una_pagina_componente_externo_ELTPARAGRAPHE000143"></a>Para abrir una página de un componente externo, utilice el nombre de la página del componente externo (el componente incluido en el proyecto).. Por ejemplo:

```wl
PageUse(ComponentPage)
```
 Si se produce un conflicto con un elemento del proyecto, el nombre del elemento debe ir precedido del nombre del componente externo. Por ejemplo:

```wl
PageUse(MyComponent.Page)
```
 Para utilizar el nombre de la página en un Variable, se debe especificar el nombre del componente externo. Por ejemplo:

```wl
sPage is string = "MyComponent.MyPage"

PageUse(sPage)
```

<a name="NOTE0_6"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000172"></a>

- Para volver a mostrar una página, se recomienda utilizar [PageRefresh](../WDLang2/3058011.md).

- Para mostrar una página sin cerrar los contextos abiertos en el servidor, le recomendamos que utilice [PageDisplay](../WDLang2/3058008.md).

- No debe utilizar el código WLanguage después de una llamada a **Uso de la página**.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

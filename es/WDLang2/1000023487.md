


## &lt;StartFilling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la Process para rellenar una control Looper de forma programada. Se representa el progreso de esta operación:

- mediante una barra de progreso de Infinite que se muestra en la parte inferior de control.

- por una ventana interna.




**Atención**: Antes de utilizar esta función, es necesario habilitar el mecanismo de gestión de la operación de llenado en la ventana de control description. Para obtener más información, consulte [Características de un control Looper en una ventana](../WDChamp/1013294.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LOOP_MyLooper.StartFilling()

FOR i = 1 TO 10000
	LOOP_MyLooper.AddLine()
	Multitask(-1)
END

LOOP_MyLooper.EndFilling()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Looper control>.StartFilling()`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al final de la Process para cargar la control Looper, se debe llamar a [&lt;EndFilling](../WDLang2/1000023501.md) para especificar que la operación de carga ha finalizado.

- Esta función no tiene ningún efecto en los controles del Looper de relacionado a un archivo de datos o en la fuente de datos de consulta.

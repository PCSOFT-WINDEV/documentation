


## jQueryExecute (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un método JavaScript de la biblioteca jQuery en un elemento de la página.

Observaciones: 

- [jQuery](../WDLang2/1000021070.md) se utiliza para ejecutar varios métodos encadenados. 

- Esta función es una función avanzada: se requiere un buen conocimiento de JavaScript y de la biblioteca jQuery



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The CELL_Square control explodes before it disappears
jQueryExecute(CELL_Square,"effect","explode")

// Modify the opacity of the STC_1 control
jQueryExecute(STC_1, "fadeTo", "opacity", 0.5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = jQueryExecute(<Control used> [, <Category>] , <jQuery function> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Result>: (Tipo esperado)`**

Valor de retorno del método jQuery llamado.

**`<Control used>: (Nombre del control)`**

Nombre de control que se utilizará en la página. 

Observación: La opción "El control puede ser superpuesto" debe ser marcada para el control (pestaña "UI" en la ventana de control de description).

**`<Category>: (Cadena de caracteres opcional)`**

Nombre de la categoría de funciones en la biblioteca jQuery. 

**`<jQuery function>: (Cadena de caracteres)`**

Nombre de la función jQuery a ejecutar. 

Atención: Dado que JavaScript distingue entre mayúsculas y minúsculas, el nombre de la función debe respetar las mayúsculas y minúsculas utilizadas en la biblioteca jQuery.

**`<Parameter 1>: (Cadena de caracteres opcional)`**

Primer parámetro esperado por la función jQuery. 



**`<Parameter N>: (Cadena de caracteres opcional)`**

Nth esperado por la función jQuery.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Parámetros de la función jQuery
<a name="parametros_funcion_jquery_ELTPARAGRAPHE000052"></a>Si se debe pasar un procedimiento ("Callback") como parámetro a la función jQuery, el procedimiento del navegador debe ser pasado por reference (nombre de la función precedido por "&"). Por ejemplo: 


```wl
jQueryExecute(BTN_Addition, "click", &OnAddition)
```



### Cargando las librerías jQuery
<a name="cargando_las_librerias_jquery_ELTPARAGRAPHE000059"></a>

Hay varias versiones disponibles para la biblioteca jQuery. Varios archivos Javascript se pueden encontrar en la biblioteca jQuery.

Por lo tanto, las bibliotecas adicionales de JavaScript deben ser cargadas. Hay dos métodos disponibles: 

- incluir el código de carga de JavaScript en el código de la cabecera de la página. 

- descargar las librerías Javascript jQuery en el servidor e incluir los archivos Javascript en su proyecto o página.




**Método 1: Para incluir el código de carga en la página**: 

1. Mostrar la ventana description de la página ("description" en el menú contextual). 

2. Seleccione la pestaña "Avanzado. 

3. Seleccione la pestaña "HTML. 

4. copy las siguientes líneas de código en el área de "código HTML añadido al encabezado de la página".:   



```html
<script src="http://code.jquery.com/jquery-1.9.1.js"></script>
<script src="http://code.jquery.com/ui/1.10.3/jquery-ui.js"></script>
```
Atención: Cuando se escribió esta página de ayuda, las librerías estaban disponibles en las versiones 1.9.1 y 1.10.3. Le aconsejamos que compruebe si estas versiones siguen siendo las actuales. 

**Método 2: Para incluir los archivos Javascript en su página (después de la descarga)**: 

1. Mostrar la ventana description de la página ("description" en el menú contextual). 

2. Seleccione la pestaña "Avanzado. 

3. Seleccione la pestaña "HTML".

4. Pulse el botón "Añadir" y seleccione los archivos JavaScript necesarios (.js). Atención: le aconsejamos que compruebe qué versiones son las actuales.

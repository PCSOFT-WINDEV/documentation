
## GaleríaCerrarPopup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el popup de zoom de automatic para la galería de Image.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
GalleryClosePopup(LOOP_Gallery)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GalleryClosePopup(<Looper control>)`
---

**`<Looper control>: (Nombre del control)`**

Nombre del control Looper en modo Galería a utilizar.

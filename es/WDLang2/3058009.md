


## CurrentPage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de la página que contiene el código WLanguage actualmente en ejecución.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Str is string
Str = CurrentPage()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CurrentPage()`
---

**`<Result>: (Cadena de caracteres)`**




|   |   |
| --- | --- |
| Nombre de la página actual en mayúsculas | Si el código WLanguage que se está ejecutando es:<br><br>- un Event associated con la página: <br><br><br><br><br>- un Event associated con un control (inicializando, clic).<br><br>- un Procedure local a la página.<br><br><br> |
| Cadena vacía ("") | Si el código que se está ejecutando es:<br><br><br> |

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El caso del nombre de la página corresponde al caso real del nombre de la página. Por ejemplo, si la página actual corresponde a la página "Home.WWH", el nombre devuelto será "Home".

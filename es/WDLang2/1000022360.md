
## GglAnalyticsAddException( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una excepción además de Google Analytics. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Message is Variant
Message.exDescription = ErrorInfo()
Message.exFatal = False
GglAnalyticsAddException(Message)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GglAnalyticsAddException([<Options>])`
---

**`<Options>: (Variante opcional)`**

Opciones adicionales dependientes de Google Analytics. Para más detalles sobre los posibles valores, consulte la documentación de Google Analytics en [https://developers.google.com/analytics/devguides/collection/analyticsjs/exceptions](https://developers.google.com/analytics/devguides/collection/analyticsjs/exceptions). 
Este parámetro es set a Nula por Default.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para utilizar esta función, debe tener y guardar un identificador de Google Analytics en el proyecto. Para obtener más información, consulte [WEBDEV y Google Analytics](../WDLang2/1000022374.md).

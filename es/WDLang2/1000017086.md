


## PageRateDDW( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer y modificar la tasa de grises de las páginas que utilizan OVI (Dim disabled pages).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PageActivateDDW(True)
PageRateDDW(40)
YesNo("Do you want to exit?")
PageActivateDDW(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la tasa de atenuación

`<Result> = PageRateDDW()`
---

**`<Result>: (Integro)`**

Tasa de atenuación actual (entre 0 y 100). La tasa de gris por defecto es set a 60%.  


<a name="SYNTAX2"></a>

### Cambio de la tasa de atenuación

`<Result> = PageRateDDW(<Dimming rate>)`
---

**`<Result>: (Integro)`**

Tasa de atenuación inicial (entre 0 y 100).

**`<Dimming rate>: (Integro)`**

Nueva tasa de atenuación (entre 0 y 100). 
El valor máximo (100) se utilizará si el valor especificado es superior a 100. 
La función **PageRateDDW** no tendrá ningún efecto si se indica un valor inferior a 0. La tasa de gris por defecto es set a 60%.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para activar la función OVI en las páginas, utilice [PáginaActivarDW](../WDLang2/1000017087.md).

- El valor utilizado en el código del navegador se inicializa por el valor dado en el código del servidor. Las modificaciones realizadas en el código del navegador no se transfieren al código del servidor.

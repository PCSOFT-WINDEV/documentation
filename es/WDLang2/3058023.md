
## SiteAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el INTERNET Address para conectarse a un sitio WEBDEV dinámico (en modo Sesión) que se encuentra en el mismo servidor.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate a link that starts the site named "MySite2"
HTM_HTMLControl.Value = "<A HREF=" + ...
	SiteAddress(MySite2) + ">Click me to go to Site 2</A>"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SiteAddress([<Site name>])`
---

**`<Result>: (Cadena de caracteres ANSI)`**

Address del sitio. .

**`<Site name>: (Cadena de caracteres opcional)`**

Nombre del sitio dinámico WEBDEV (en modo Sesión) cuyo Address se solicita. Este sitio dinámico y el sitio actual deben estar instalados en el mismo servidor.  

Si este parámetro es igual a una cadena vacía ("""), el nombre del sitio corresponde al sitio actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Ejemplos de uso
<a name="ejemplos_uso_ELTPARAGRAPHE000044"></a>

- Este Address puede utilizarse, por ejemplo, en un Email que contenga un enlace al sitio.

- Este Address también es useful cuando el código HTML que contiene acciones específicas es generado por el programa.

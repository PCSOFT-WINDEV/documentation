
## BrowserPlatform( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la plataforma del navegador del usuario.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a smartphone-specific page

SWITCH BrowserPlatform()
	CASE BrowserPlatformAndroid, BrowserPlatformBlackBerry, ... 
		BrowserPlatformiPhone, BrowserPlatformMobile
		PageDisplay(PAGE_Details_Mobile)
	OTHER CASE
		PageDisplay(PAGE_Details)
END
```



```wl
// Displays a smartphone-specific page in an AWP site

bBrwVersion is boolean

ConfigureAWPContext(ctxDisk, ctxIDCookie)
DeclareAWPContext(bBrwVersion)

IF bBrwVersion = False THEN
	SWITCH BrowserPlatform()
		CASE BrowserPlatformAndroid, BrowserPlatformBlackBerry, ... 
			BrowserPlatformiPhone, BrowserPlatformMobile
			PageDisplay("List-Travels-smartphones")
		OTHER CASE
			PageDisplay(PAGE_Home)
	END
END
bBrwVersion = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BrowserPlatform()`
---

**`<Result>: (Constante de tipo Integer)`**

Plataforma de navegación:



|   |   |
| --- | --- |
| *BrowserPlatformAndroid* | El usuario muestra el sitio en un Smartphone que funciona con Android. |
| *BrowserPlatformBlackBerry* | El usuario muestra el sitio en un BlackBerry. |
| *BrowserPlatformiPad* | El usuario muestra el sitio en un iPad. |
| *BrowserPlatformiPhone* | El usuario muestra el sitio en un iPhone. |
| *BrowserPlatformMobile* | El usuario muestra el sitio en un Smartphone (distinto de los mencionados). |
| *BrowserPlatformRobot* | El usuario es un robot de un motor de búsqueda (y por lo tanto indexa el sitio). |
| *BrowserPlatformStandard* | El usuario no utiliza una plataforma específica (PCs, ordenadores portátiles, Mac,...). |
| *BrowserPlatformWindowsMobile* | El usuario muestra el sitio en un Smartphone con Windows Mobile.<br><br>Observación: Esta plataforma ya no está disponible a partir de la versión 27. |
| *BrowserPlatformWindowsPhone* | El usuario muestra el sitio en un SmartPhone con Windows Phone o Windows 10 Mobile.<br><br>Observación: Esta plataforma ya no está disponible. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función devuelve la constante *BrowserPlatformStandard* si el navegador utilizado por el usuario de la web es desconocido.

- En el caso de un sitio dinámico (sitio no AWP), el valor devuelto corresponde al navegador utilizado al conectarse al sitio. Si el usuario cambia a otro navegador (copia y pega una URL en otro navegador, por ejemplo), este cambio no se detecta.

- Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".


## FramesetDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un conjunto de marcos WEBDEV en el navegador del usuario.

Si se utiliza **FramesetDisplay** en un código de botón, el conjunto de marcos se abrirá en el objetivo definido para este botón (pestaña "General" del control description o [ChangeTarget](../WDLang2/3058005.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Redisplay the FSET_LOGIN frameset 
FramesetDisplay(FSET_LOGIN)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`FramesetDisplay(<Frameset name> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Frameset name>: (Cadena de caracteres)`**

Nombre del conjunto de marcos a mostrar. 
Si este parámetro corresponde a una cadena vacía (""), el nombre del conjunto de marcos es el del conjunto de marcos al que pertenece el actual Event.

**`<Parameter 1>: (Tipo de valor enviado a la página (opcional))`**

Parámetros que se pasarán a las "Declaraciones Globales" Event del conjunto de marcos para abrir. Estos parámetros son pasados por reference y son considerados como variables globales al conjunto de marcos y a todas las páginas del conjunto de marcos.



**`<Parameter N>: (Tipo de valor enviado a la página (opcional))`**

Parámetros que se pasarán a las "Declaraciones Globales" Event del conjunto de marcos para abrir. Estos parámetros son pasados por reference y son considerados como variables globales al conjunto de marcos y a todas las páginas del conjunto de marcos.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Recuperación de parámetros
<a name="recuperacion_parametros_ELTPARAGRAPHE000060"></a>Para recuperar los parámetros, en las "Declaraciones globales" Event del conjunto de marcos, especifique el siguiente Line al principio del código:


```txt
PROCEDURE <Frameset name>(<Parameter 1> [,...[, <Parameter N>]])
```

<a name="NOTE0_2"></a>




### Principio para abrir un juego de marcos
<a name="principio_para_abrir_juego_marcos_ELTPARAGRAPHE000070"></a>

**FramesetDisplay** hace lo siguiente:

- crear un contexto de conjunto de marcos en el servidor

- crear un contexto de página en el servidor para todas las páginas del conjunto de marcos del servidor

- devolver el conjunto de marcos al navegador del usuario.



<a name="NOTE0_3"></a>




### Las operaciones se ejecutan al abrir el conjunto de marcos
<a name="las_operaciones_ejecutan_abrir_conjunto_marcos_ELTPARAGRAPHE000084"></a>

Las diferentes operaciones que se pueden realizar al visualizar el conjunto de marcos son las siguientes:

- Ejecutando las "Declaraciones globales" e "Iniciando" eventos del conjunto de marcos.

- Abrir las páginas del juego de marcos (ejecutar los eventos de "Declaraciones globales" e "Inicialización" para cada página).

- Código WLanguage después de la llamada a **FramesetDisplay** (si se utiliza).



<a name="NOTE0_4"></a>


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000099"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

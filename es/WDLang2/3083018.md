


## BreadcrumbModifyLink (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un enlace en un control Migas de pan.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies a link
BreadcrumbModifyLink(SMP_MyBreadcrumb, 4, "Help", "Help_Page", "Help")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`BreadcrumbModifyLink(<Breadcrumb control> , <Number> , <Caption> , <URL or page name> [, <Tooltip>])`
---

**`<Breadcrumb control>: (Nombre del control)`**

Nombre de la control Migas de pan en la que se modificará el enlace.

**`<Number>: (Integro)`**

Número del enlace a modificar. Este número está comprendido entre 1 y el número de apariciones de la &lt;control Migas de pan&gt; (devuelta por la [Conde](../Proprietes/2510097.md) Property).

**`<Caption>: (Cadena de caracteres)`**

Nuevo título para el enlace.

**`<URL or page name>: (Cadena de caracteres)`**

Nueva URL associated con el enlace o nombre lógico de la página WEBDEV a abrir.

**`<Tooltip>: (Cadena de caracteres opcional)`**

description del enlace, mostrada en un tooltip cuando el ratón pasa por encima del enlace.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Propiedades de los enlaces
<a name="propiedades_los_enlaces_ELTPARAGRAPHE000070"></a>

Para acceder a las propiedades de cada enlace en el control Migas de pan, también puede utilizar el operador [] con el Property asociado.

Por ejemplo:


```wl
SMP_Breadcrumb[1].Caption = "Home"
```

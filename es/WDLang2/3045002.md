


## ImageArea( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la zona de Image en la que ha hecho clic el usuario de la web. 
Esta función sólo puede utilizarse en el código de clic de un control Imagen con enlace (control con una acción definida como "Áreas clicables definidas en modo de edición (Mapa de imágenes)" en la pestaña "General" de la ventana del control description).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on IMG_IMAGE1
n is int = ImageArea()
Trace(n)
SWITCH n
	CASE 0: STC_Caption1 = "External area"
	CASE 1: STC_Caption1 = "Area 1"
	CASE 2: STC_Caption1 = "Area 2"
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ImageArea()`
---

**`<Result>: (Integro)`**

Número de la zona en la que se ha hecho clic. Este número es indefinido si **ImageArea** se utiliza en un Event que no sea el "Click" Event de un control Imagen. 
Observación: En la pestaña "General" de la ventana control Imagen description, la "Acción al hacer clic" corresponde a "Zonas clicables definidas en modo de edición (mapa Image)".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000038"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".

## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
En un sitio Web, el almacenamiento local se utiliza para almacenar y recuperar valores en el código del navegador.

Estos valores pueden ser: 

- compartido por todas las ventanas y pestañas del mismo navegador en un ordenador: por lo tanto, se pueden recuperar una vez que el navegador se ha cerrado y se ha vuelto a abrir o en otra ventana del navegador.. **En este caso, el almacenamiento local es persistente.** 

- específico de una ventana o de una pestaña del navegador: en este caso, los valores se perderán cuando se cierre la ventana del navegador. Por el contrario, estos valores pueden ser leídos por las ventanas del navegador abiertas desde la ventana donde se han almacenado los valores.. **En este caso, el almacenamiento local no es persistente.**




**Observaciones**: 

- El almacenamiento de parámetros locales no está disponible por Page. 

- El almacenamiento de parámetros locales sólo está disponible para los navegadores recientes (compatible con el estándar HTML 5).

- Incluso para el almacenamiento local persistente, el almacenamiento no es compartido por los diferentes navegadores (un valor almacenado en el Explorador INTERNET no puede ser recuperado por Firefox).






## Funciones WLanguage
<a name="funciones_wlanguage_ELTTEXTE000100"></a>
Las siguientes funciones se utilizan para realizar un almacenamiento local:



|   |   |
| --- | --- |
| [LocalStorageAdd](../WDLang2/1000019382.md) | Añade un valor al almacenamiento local. |
| [LocalStorageAvailable](../WDLang2/1000019379.md) | Indica si el almacenamiento local está disponible o no para el navegador actual.. |
| [LocalStorageDelete](../WDLang2/1000019385.md) | Elimina un valor del almacenamiento local. |
| [LocalStorageDeleteAll](../WDLang2/1000019386.md) | Borra todos los valores del almacenamiento local. |
| [LocalStorageGet](../WDLang2/1000019384.md) | Recupera un valor del almacenamiento local. |
| [LocalStorageOccurrence](../WDLang2/1000019387.md) | Devuelve el número de valores para el almacenamiento local. |
| [LocalStorageValueName](../WDLang2/1000019388.md) | Devuelve el nombre de un valor para el almacenamiento local. |




## ImageYPos( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición vertical (Y) del cursor del ratón con respecto a la control Imagen especificada.. 

Esta función sólo puede utilizarse en el código de clic de una control Imagen. Esta control Imagen tiene las siguientes características (en la pestaña "General" de la ventana de control description): 

- Acción del clic: "Llamar al servidor"

- Durante la acción: "Enviar coordenadas del clic".





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click code of IMG_IMAGE1
X is int = ImageXPos()
Y is int = ImageYPos()
IF X > Y THEN
	STC_CAPTION1 = "Top - Right"
ELSE
	STC_CAPTION1 = "Bottom - Left"
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ImageYPos()`
---

**`<Result>: (Integro)`**

Posición vertical (coordenada Y) en píxeles del cursor del ratón en relación con la Corner superior izquierda de la control Imagen sobre la que se ha pulsado el botón izquierdo del ratón. 
Este valor es indefinido si se utiliza **ImageYPos** en un evento que no sea el código de clic de un control Imagen con una acción configurada como "Enviar coordenadas de clic" (pestaña "General" de la ventana del control description).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Sesiones preiniciadas
<a name="sesiones_preiniciadas_ELTPARAGRAPHE000172"></a>

Si el proyecto utiliza sesiones preiniciadas, esta función no debe utilizarse en el evento "Inicialización" del proyecto. Debe utilizarse en el evento "Inicialización del proyecto después de la conexión al sitio".




## Funciones de gestión de imágenes (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Se pueden utilizar varias funciones para gestionar imágenes: 

- Funciones de gestión de Image específicas de la web (especialmente las que gestionan el Scrolling de Image). 

- Funciones para la gestión de Image en los controles de Image. 





|   |   |
| --- | --- |
| [ImageArea](../WDLang2/3045002.md) | Devuelve el número del área de Image en el que ha hecho clic el usuario de la web. |
| [&lt;Image&gt;.first](../WDLang2/1000023753.md) | Muestra el primer Image para el pergamino. |
| [&lt;Image&gt;.last](../WDLang2/1000023748.md) | Muestra el último Image para el pergamino. |
| [&lt;Image&gt;.LayerVisible](../WDLang2/1000023747.md) | Se usa para hacer visible una capa que se encuentra en una imagen WDPIC o Invisible. |
| [ImageNext](../WDLang2/1000023754.md) | Muestra la siguiente Image para el pergamino. |
| [&lt;Imagen&gt;.Occurrence](../WDLang2/1000023750.md) | Devuelve el número total de imágenes que tiene en cuenta el scroll. |
| [ImagePrevious](../WDLang2/1000023752.md) | Muestra el anterior Image para el pergamino. |
| [ImageScrollingPosition](../WDLang2/1000023751.md) | Devuelve la Image visualizada. |
| [&lt;Imagen&gt;.StartScrolling](../WDLang2/1000023749.md) | Inicia el ciclo de imágenes. |
| [ImageStopScrolling](../WDLang2/1000023746.md) | Detiene el desplazamiento de imágenes de automatic. |
| [ImageXPos](../WDLang2/3045003.md) | Devuelve la posición horizontal (X) del cursor del ratón con respecto a la control Imagen especificada. |
| [ImageYPos](../WDLang2/3045001.md) | Devuelve la posición vertical (Y) del cursor del ratón con respecto a la control Imagen con enlace especificada. |

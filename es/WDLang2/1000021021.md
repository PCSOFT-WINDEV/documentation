
## NotificationDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje fuera del navegador. 

Un navegador puede mostrar información fuera de su ventana. Por ejemplo, un navegador en Windows puede mostrar una notificación de un mensaje recibido en la parte inferior derecha del escritorio. Incluso si el navegador está minimizado, se mostrará la información. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notification_web.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a message outside the browser
NotificationDisplay("Messaging", "New incoming email", ...
		 "Message.gif")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`NotificationDisplay(<Title> , <Message> [, <Image>])`
---

**`<Title>: (Cadena de caracteres)`**

Título del mensaje mostrado. Si este parámetro corresponde a una cadena vacía (""), no se mostrará el título. 

**`<Message>: (Cadena de caracteres)`**

Mensaje que se mostrará en la notificación. 

**`<Image>: (Cadena de caracteres opcional)`**

Imagen del mensaje mostrado. 

En una página dinámica, si la imagen se encuentra en el directorio "_WEB" del proyecto, se debe especificar la ruta virtual del directorio, utilizando la función [FolderWeb](../WDLang2/3012015.md). Por ejemplo: FolderWeb() + "MiImagen.gif".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las notificaciones deben estar activadas en Windows en el equipo del usuario. De lo contrario, no se mostrará la notificación. 

- Algunos navegadores no admiten las notificaciones. Si el navegador no soporta Notification s, no se mostrará el Notification.

- Si las notificaciones no están permitidas, se mostrará una solicitud de autorización en el navegador.
	Atención: Si se llama a **NotificationDisplay** desde un botón que ejecuta código del servidor, la petición Notification Permission del navegador se eliminará cuando se actualice la página: por lo tanto, el usuario no podrá autorizar las notificaciones.

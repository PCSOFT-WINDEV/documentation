
## TmeLine control relacionado a un archivo de datos o Variable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000125"></a>
Puede crear controles TimeLine directamente relacionado a los datos. Puede crear: 

- a **control Línea de tiempo basado en un archivo de datos**: el control Línea de tiempo será relacionado a un archivo de datos o a una consulta. En este caso, el archivo de datos contiene las características de los eventos. 

- a **control Línea de tiempo basado en un Variable**: el control Línea de tiempo será relacionado a un Variable (una instancia de clase en la mayoría de los casos). En este caso, la clase lista los eventos y sus características. 




La programación de los controles TimeLine se simplifica. 





## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000149"></a>


### Creación de una control Línea de tiempo basada en un archivo de datos
<a name="creacion_una_control_linea_tiempo_basada_archivo_datos_ELTPARAGRAPHE000020"></a>

Para crear una línea de tiempo control basada en un archivo de datos: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Contenido" seleccione "Archivo/Consulta" y seleccione la fuente de datos utilizada.. Esta fuente de datos puede ser: 

	- un archivo de datos definido en el análisis,

	- una consulta del proyecto, 

	- una consulta incluida en la ventana. 




3. Asociar los diferentes elementos de la fuente de datos con las características del Event. Los elementos precedidos de una estrella son obligatorios. 
	**Observación**: Si el archivo de datos asociado no contiene directamente el elemento correspondiente, puede set un enlace de varios archivos. 

4. Validar la ventana control Línea de tiempo description. El control Línea de tiempo se basa en el un archivo de datos. 







### Creación de una control Línea de tiempo basada en una Variable
<a name="creacion_una_control_linea_tiempo_basada_una_variable_ELTPARAGRAPHE000037"></a>

Para crear una control Línea de tiempo basada en una Variable: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Contenido", seleccione la opción "Por variable", y luego seleccione la fuente de datos utilizada (clase, etc.). 

3. Asociar los diferentes elementos (miembros de la clase por ejemplo) de la fuente de datos con las características de la Event. Los elementos precedidos de una estrella son obligatorios. 

4. Validar la ventana control Línea de tiempo description. El control Línea de tiempo se basa en un Variable. 








## Gestión de una control Línea de tiempo basada en un archivo de datos o Variable
<a name="gestion_una_control_linea_tiempo_basada_archivo_datos_variable_ELTTEXTE000179"></a>


### control Línea de tiempo basada en un archivo de datos
<a name="control_linea_tiempo_basada_archivo_datos_ELTPARAGRAPHE000049"></a>

Los controles TimeLine basados en un archivo de datos son muy fáciles de manipular: 

- Si el usuario maneja la línea de tiempo a través del menú contextual, todas las adiciones, eliminaciones y cambios realizados en los eventos se aplicarán automáticamente a el archivo de datos relacionado al control.

- Si el usuario maneja los eventos a través de una ventana específica, todo lo que tiene que hacer es modificar el archivo de datos relacionado a la línea de tiempo. Los datos visualizados se actualizarán automáticamente. También es posible utilizar [Visualización de la línea de tiempo](../WDLang1/1000020683.md) para actualización los datos en el control. 




**Observación**: Si el control es relacionado a una fuente de datos, la eliminación realizada en el control se realiza al mismo tiempo en la fuente de datos del relacionado. 


### control Línea de tiempo basada en un Variable
<a name="control_linea_tiempo_basada_variable_ELTPARAGRAPHE000065"></a>

Los controles de la línea de tiempo basados en una Variable son muy fáciles de manipular: 

- Si el usuario maneja la línea de tiempo a través del menú contextual, todas las adiciones, supresiones y cambios realizados en los eventos se aplicarán automáticamente a la fuente de datos de control.

- Si el usuario maneja los eventos a través de una ventana específica, todo lo que tiene que hacer es modificar los elementos relacionado a la línea de tiempo (miembros de la clase por ejemplo).. Los datos visualizados se actualizarán automáticamente. También es posible utilizar [Visualización de la línea de tiempo](../WDLang1/1000020683.md) para actualización los datos en el control. 




**Observación**: Si el control es relacionado a una fuente de datos, la eliminación realizada en el control se realiza al mismo tiempo en la fuente de datos del relacionado.

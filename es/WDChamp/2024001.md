## Presentación
<a name="presentacion_ELTTEXTE000185"></a>
Su reporte muestra controles que se superponen?

Quieres averiguar la posición de orden Z de tus controles: ¿qué controles se muestran en primer plano?

La pantalla bidimensional pronto se vuelve inadecuada para este tipo de operación.

WINDEV le ofrece una vista espacial de una reporte al permitirle ver la reporte en tres dimensiones. La posición de orden Z de los mandos reporte es inmediatamente Visible. Una control seleccionada en este modo de visualización permanece seleccionada cuando el modo vuelve a las dos dimensiones.

Usted tiene la posibilidad de elegir el mejor ángulo de visión mediante el ratón y las teclas del teclado.

**Atención**: se requiere una configuración específica para utilizar esta función. Para más detalles, ver Observaciones.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000233"></a>
Para visualizar sus informes en 3D: 

1. Visualice el reporte en el editor.

2. En la pestaña "Vista", en el grupo "Opciones", haga clic en "3D".

3. La reporte se muestra en 3D.




**Observación**: Para mostrar una vista tridimensional de su reporte y para animar esta vista, presione Alt + Shift.

El ratón o el teclado pueden utilizarse para optimizar la vista reporte.

   | --- |
| --- |
| Se mueve con el ratón |
| Rotación (botón izquierdo del ratón) | Haga clic en la vista reporte y mueva el ratón manteniendo pulsado el botón izquierdo. |
| Reubicación (botón derecho del ratón) | Haga clic en la vista reporte y mueva el ratón manteniendo pulsado el botón derecho. |
| **Se mueve con el teclado** |
| rotación | Utilice las teclas de flecha, así como las teclas + y - del teclado numérico.<br>También puede utilizar los números del teclado numérico. |
| Traslación | Mantenga pulsada la tecla Mayúsculas y utilice las teclas de flechas del teclado numérico. |
| Zoom | Pulse la siguiente combinación de teclas:<br><br>- Mayúsculas + para acercar la imagen<br><br>- Mayúsculas - para alejar la imagen.<br><br><br> |
| Modificar la separación entre los mandos (modificaciones entre los diferentes planos de visualización) | / (en el teclado numérico) para reducir el espacio entre los controles.<br>"\*" (en el teclado numérico) para aumentar el espacio entre los mandos |
| Reinicialización de la pantalla | Tecla 0 en el teclado numérico: reinicialización completa<br>5 en el teclado numérico: Reinicialización manteniendo el espacio entre los controles. |
| Animación de la vista en 3 dimensiones | Tecla bloq despl.<br>Clave Esc de Cancel |




**Observación**: Para desactivar la vista tridimensional, pulse la tecla Esc.

## Notas
<a name="notas_ELTTEXTE000257"></a>


### Posibles acciones al visualizar un reporte en 3D
<a name="posibles_acciones_visualizar_reporte_3d_ELTPARAGRAPHE000117"></a>

En el modo 3D, sólo se pueden realizar algunas acciones en los controles reporte. Puedes, por ejemplo:

- seleccione un control.

- mostrar el control description.

- mostrar el código fuente de la control.



<a name="NOTE4_2"></a>


### Condiciones necesarias para visualizar una reporte en 3D
<a name="condiciones_necesarias_para_visualizar_una_reporte_3d_ELTPARAGRAPHE000128"></a>

Para ver sus informes en 3D, el ordenador actual debe estar equipado con:

- DirectX versión 8.0.A (o posterior).

- Una tarjeta aceleradora 3D con controladores actualizados.

- Un modo de visualización en 16 bits o 32 bits.




Si utiliza WINDEV, WEBDEV o WINDEV Mobile en modo de 64 bits, para implementar la visualización en 3 dimensiones, debe descargar e instalar el paquete desde la siguiente Address:  [https://www.microsoft.com/fr-fr/download/details.aspx?id=35](https://www.microsoft.com/fr-fr/download/details.aspx?id=35)

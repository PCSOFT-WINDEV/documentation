## Presentación del control Editor de imágenes para móviles
<a name="presentacion_del_control_editor_imagenes_para_moviles_ELTTEXTE000118"></a>
El control Editor de imágenes para aplicaciones móviles permite a los usuarios finales editar imágenes o fotos.  

Este control incluye todas las características estándar de edición de imágenes para dispositivos móviles:

- recorte,

- rotación, 

- corrección de perspectiva,

- brillo, contraste, saturación, 

- anotaciones: dibujo a mano alzada, resaltador, texto,

- simetría, 

- desenfoque,

- OCR sugerido en áreas de texto detectadas,

- etc.

## Crear un control Editor de imágenes para móviles
<a name="crear_control_editor_imagenes_para_moviles_ELTTEXTE000142"></a>
**Para crear un control Editor de imágenes para móviles**: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Ofimática" y seleccione "Editor de imágenes para móviles". 

2. Haga clic en la ubicación en la que desea crear el control.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Características del control Editor de imágenes para móviles
<a name="caracteristicas_del_control_editor_imagenes_para_moviles_ELTTEXTE000170"></a>
Puede elegir las opciones que se muestran en la barra de herramientas del control en la pestaña "General" de la ventana de descripción: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_mobile%20-%20HC%20N%B0001.gif)


- **Mostrar opciones para cambiar de imagen (abrir, tomar una foto, etc.)**: Permite al usuario seleccionar la imagen que se mostrará en el control. El usuario puede tomar una foto o abrir un archivo del dispositivo. La barra de herramientas correspondiente aparece en la esquina superior derecha del control Editor de imágenes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_mobile%20-%20HC%20N%B0002.gif)


- **Mostrar opciones para modificaciones (recorte, rotación, contraste, etc.)**: Permite al usuario editar la imagen que se muestra en el control Editor de imágenes. Esta opción corresponde a los botones "Transformar" y "Ajustes" de la barra de herramientas del control.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_mobile%20-%20HC%20N%B0003.gif)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_mobile%20-%20HC%20N%B0004.gif)


- **Mostrar opciones para anotaciones (lápiz, texto, resaltado, etc.)**: Permite al usuario hacer anotaciones en la imagen agregando texto, por ejemplo. Esta opción corresponde a los botones "Dibujo" y "Texto" de la barra de herramientas del control. 

- **Mostrar opciones para reconocimiento óptico de caracteres (OCR)**: Permite al usuario ejecutar el reconocimiento óptico de caracteres (OCR) en la imagen que se muestra en el control Editor de imágenes. Esta opción corresponde al botón "OCR" disponible en las herramientas del control.

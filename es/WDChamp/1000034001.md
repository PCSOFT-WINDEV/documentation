## Presentación del control Barra de progreso infinita
<a name="presentacion_del_control_barra_progreso_infinita_ELTTEXTE000157"></a>
Las barras de progreso proporcionan información útil al usuario final.

A veces no es posible predecir la duración de un proceso y, por lo tanto, mostrar una barra de progreso que corresponda al estado de avance.

En ese caso, se puede utilizar un control Barra de progreso infinita. Este tipo de barra de progreso muestra una animación de carga infinita. <br>![Ejemplos de barras de progreso "infinitas"](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD-Jauge-Infinie%20-04.jpg)

## Crear un control Barra de progreso infinita
<a name="crear_control_barra_progreso_infinita_ELTTEXTE000181"></a>
Para crear un control Barra de progreso infinita: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Barra de progreso" y seleccione una barra de progreso infinita. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Jauge20.gif)


2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

<a name="NOTE3_1"></a>


## Características del control Barra de progreso infinita
<a name="caracteristicas_del_control_barra_progreso_infinita_ELTTEXTE000205"></a>
Puede definir el tipo de barra de progreso infinita en la pestaña "General" de la ventana de descripción: 

- Lineal nativa.

- Imagen personalizada. En este caso, puede especificar los siguientes elementos en el editor o mediante programación: 

	- la imagen utilizada. 
			 Mediante programación, esta opción corresponde a la propiedad [Image](../Proprietes/2510034.md). 

	- el número de etapas de la animación. 
			![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Mediante programación, esta opción corresponde a la propiedad [NbAnimationSteps](../Proprietes/1000021230.md). 

	- la duración de la animación
			Mediante programación, esta opción corresponde a la propiedad [AnimationDuration](../Proprietes/1000021069.md). 

	- si la animación debe ir primero hacia adelante, y luego hacia atrás. 
			![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Mediante programación, esta opción corresponde a la propiedad [TwoWayAnimation](../Proprietes/1000021231.md).

## Modo de funcionamiento del control Barra de progreso infinita
<a name="modo_funcionamiento_del_control_barra_progreso_infinita_ELTTEXTE000229"></a>
Una barra de progreso infinita es una barra de progreso que nunca se detiene. 

Para utilizar una barra de progreso infinita en una aplicación, le recomendamos crear un control Barra de progreso infinita, y luego, hacerla visible o invisible (propiedad [Visible](../Proprietes/2510138.md)) según sus necesidades.

Observación: La animación de la barra de progreso infinita se realiza en el hilo principal de la aplicación. Si el hilo principal está bloqueado por un largo proceso, la animación se detiene. El proceso asociado a la barra de progreso infinita debe: 

- realizarse en un hilo secundario. 

- ser un proceso asíncrono.

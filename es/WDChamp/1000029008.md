
## Funcionalidades automáticas de los controles Tabla dinámica
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000401"></a>
La control Tabla dinámica ofrece varias características de aplicación de la automatic (AAF) que permiten al usuario aprovechar todas sus capacidades. 

El usuario puede: 

- cambiar el tamaño de la primera columna de la control Tabla dinámica con el ratón (columna que contiene las cabeceras de la fila). 

- contraer/expandir columnas y filas haciendo clic en los iconos de cabecera "+" y "-".. 

- abrir un menú contextual para realizar varias acciones en las filas y columnas. 

- abrir un menú contextual para las celdas. 

- seleccionar varias filas, columnas o celdas mediante las teclas Mayúsculas y Ctrl. Esta opción sólo está disponible si se permite la selección múltiple en la control. Para obtener más información, consulte [description de un control Tabla dinámica](../WDChamp/1000029007.md). 

- realizar una búsqueda en las celdas de un control Tabla dinámica pulsando Ctrl + F.




Las características del menú contextual para las filas y columnas son las siguientes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD.gif)


- [Borrar/Visualizar una o más columnas](#NOTE2_1)

- [Ocultar/Desplegar las columnas](#NOTE2_2),

- [Desplegar un nivel y todos sus subniveles](#NOTE2_2),

- [Añadir grupos (semana/mes/trimestre)](#NOTE2_Add), 

- [Reorganizar los encabezados de las filas y columnas](#NOTE2_Reorganize),

- [Anular filas y columnas](#NOTE2_Reverse),

- [Comparar rangos de fechas](#NOTE2_Compare),

- copy una o más filas,

- [Exportar el contenido de la control Tabla dinámica a un Excel Document](#NOTE2_3),

- [Cargar/guardar el contenido del control Tabla dinámica](#NOTE2_4),

- [Restablecer el tamaño y los grupos iniciales](#NOTE2_5), 

- [Imprimir el contenido de control](../WDChamp/1000029009.md).




Las características del menú contextual para las celdas son las siguientes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Cellule.gif)


- copy el elemento, copy el fila, copy la columna,

- [Exportar el contenido de la control Tabla dinámica a un Excel Document](#NOTE2_3),

- [Cambiar la Color de fondo de una célula](#NOTE2_6). 

- [Cargar/guardar el contenido del control Tabla dinámica](#NOTE2_4),

Observación: Si el control Tabla dinámica permite la selección múltiple, ciertas operaciones están disponibles sólo para las filas, columnas o celdas seleccionadas.

## Características del menú contextual
<a name="caracteristicas_del_menu_contextual_ELTTEXTE000425"></a>
<a name="NOTE2_1"></a>


### Borrar columnas y filas
<a name="borrar_columnas_filas_ELTPARAGRAPHE000122"></a>

Algunas tablas dinámicas pueden contener un gran número de filas y columnas (si la control contiene varios niveles jerárquicos, por ejemplo).

Para get una visualización más concisa, puedes:

- eliminar la columna actual. 

- eliminar todas las columnas identificadas con la misma información. Por ejemplo, en una tabla pivotante que presenta cálculos a lo largo de varios años, tiene la posibilidad de borrar el mes de febrero de todos los años que se muestran en esta tabla.. 

- suprimir todas las columnas, excepto las identificadas con la misma información. Por ejemplo, en un pivote tabla que presenta cálculos a lo largo de varios años, tiene la posibilidad de visualizar los meses de febrero de cada año.. Todos los demás meses serán eliminados. 




Este modo de visualización también está disponible para las líneas.

**Durante el borrado, se vuelve a calcular el contenido del pivote tabla: se actualizan los totales.** 

Las columnas y/o filas borradas se pueden volver a mostrar en cualquier momento ("Re-visualizar todas las columnas borradas" por ejemplo). Durante la visualización, se vuelve a calcular el contenido del pivote tabla: se actualizan los totales. 

**Atención**: Si una columna o un fila no se repite, la única acción posible es borrar la columna o fila seleccionado.
<a name="NOTE2_2"></a>


### Contraer/ampliar columnas o filas
<a name="contraerampliar_columnas_filas_ELTPARAGRAPHE000143"></a>

Para una tabla dinámica con muchos elementos y subelementos, puede: 

- colapsar o expandir todas las columnas y/o filas en una sola acción. 

- expandir un nivel y todos sus subniveles en una sola acción (en la columna o en fila). 




Por ejemplo: 

- Si se desglosan todas las columnas, el pivote tabla se visualiza con más detalle.: todos los resultados intermedios son Visible, desde el nivel más alto al más bajo. 

- Si todas las columnas están colapsadas, el pivote tabla tiene una visualización sintética: sólo presenta los cálculos de primer nivel. 



<a name="NOTE2_Add"></a>


### Añadir grupos: semanal, quincenal, trimestral, ... 
<a name="anadir_grupos_semanal_quincenal_trimestral_ELTPARAGRAPHE000162"></a>

Si el pivote tabla no contiene suficientes detalles, tiene la posibilidad de añadir columnas para mejorar las estadísticas mostradas.. 

El menú contextual de las columnas incluye la opción "Añadir. Esta opción permite añadir grupos que no se muestran, por ejemplo: Semestre, Trimestre, Quincena, ....

Atención: esta opción requiere el recálculo del pivote tabla. Si se deben realizar varias operaciones, se puede posponer el recálculo de datos.. 
<a name="NOTE2_Reorganize"></a>


### Reorganización de las cabeceras de las filas y columnas
<a name="reorganizacion_las_cabeceras_las_filas_columnas_ELTPARAGRAPHE000175"></a>

Esta opción se utiliza para modificar la visualización de filas y columnas en el control Tabla dinámica. 

Cuando se selecciona esta opción, el pivote tabla se vuelve editable.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TCD_Inverser_dim1.gif)


Es posible: 

- hacer filas o columnas Visible o no. 
	Para mostrar los elementos, basta con seleccionar los elementos ocultos en "Cabeceras no mostradas" y desplazarlos a la ubicación deseada en las filas o columnas. 
	Para ocultar elementos, basta con seleccionar el encabezado correspondiente y moverlo a "Encabezados no mostrados". 

- invertir las dimensiones. Simplemente seleccione el encabezado, arrástrelo y suéltelo en la posición deseada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TCD_Inverser_dim2.gif)

### Anulación de la fila y de las cabeceras de columna
<a name="anulacion_fila_las_cabeceras_columna_ELTPARAGRAPHE000197"></a>Esta opción da la vuelta a los datos de la control Tabla dinámica. Todas las columnas se convierten en filas y todas las filas en columnas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD%20-%20TCD%20Faire%20Pivoter.jpg)


**Observación**: Para invertir las filas y las columnas que se encuentran en un control Tabla dinámica a través de la programación, utilice [AAFExecute](../WDLang1/1000022099.md) asociado a la constante *aafPvtReverseRowColumn*.
<a name="NOTE2_Compare"></a>


### Comparación de rangos de fechas
<a name="comparacion_rangos_fechas_ELTPARAGRAPHE000212"></a>

Esta opción se utiliza para comparar los datos mostrados en la control Tabla dinámica en 2 rangos de fechas. 

Cuando se selecciona esta opción: 

- Aparece una ventana que le permite introducir las características del comparison: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Comparer%20-%20HC%20N%B0001.gif&type=thumb)


	- Rango de fechas de reference. 

- Rango de fechas anteriores para comparison. 

- Evolución entre años.

- La validación de esta ventana recalcula los datos mostrados, para obtener sólo los datos de comparison. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Comparer%20-%20HC%20N%B0002.gif&type=thumb)

	Si hay que mostrar la evolución entre años, aparecerá en verde o rojo, dependiendo de si es positiva o negativa. El menú contextual de los datos incluye las dos opciones siguientes para mejorar la comparison: 

	- Mostrar evoluciones. 

	- Mostrar evoluciones en %. 





Para restablecer el estándar control Tabla dinámica, seleccione "Cancelar el rango de fechas comparison" en el menú contextual. 

**Observación**: Para comparar rangos de fechas a través de la programación, utilice [AAFExecute](../WDLang1/1000022099.md) con la constante *aafCompareDateRangePvt*.

### Exportación de control Tabla dinámica a Excel
<a name="exportacion_control_tabla_dinamica_excel_ELTPARAGRAPHE000247"></a>

Esta opción se utiliza para exportar el contenido del pivote tabla a Excel.. Esta opción corresponde a [PVTToExcel](../WDLang1/1000020703.md). Todo lo que tiene que hacer es especificar el nombre del archivo XLS para crear. La Document creada puede abrirse directamente. El contenido de la Document XLS creada corresponde exactamente a los datos mostrados en el pivote tabla. Las columnas ocultas o colapsadas no serán Visible en el archivo XLS. 
<a name="NOTE2_4"></a>


### Cargar/guardar el contenido del pivote tabla
<a name="cargarguardar_contenido_del_pivote_tabla_ELTPARAGRAPHE000257"></a>

Para evitar recalcular una control Tabla dinámica mostrada anteriormente, el contenido de la control puede guardarse en un archivo en el disco. Esto permitirá a los usuarios abrirlo más tarde o enviarlo a otra persona. Esta opción corresponde a [PVTSave](../WDLang1/1000020471.md). 

El archivo creado puede reabrirse más tarde (equivalente a [PVTLoad](../WDLang1/1000020507.md)). 

Atención: El archivo creado sólo puede abrirse en el mismo tipo de pivote tabla (con columnas y filas coincidentes).

Observación: El archivo de copia de seguridad se puede cifrar con una contraseña. 

- Cuando guarde el archivo, simplemente marque "Cifrar con contraseña" al escribir el nombre del archivo de copia de seguridad. En este caso, durante la validación, una nueva ventana solicita la contraseña a utilizar: 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Cryptage%20-%20HC%20N%B0002.gif)


- Al cargar el archivo, esta misma ventana se muestra si el archivo está cifrado. Todo lo que tienes que hacer es escribir la contraseña correspondiente. 




La ventana de introducción de la contraseña está disponible en inglés y en francés.

<a name="guardar_tamano_los_grupos_ELTPARAGRAPHE000295"></a>

Esta opción guarda los últimos ajustes del pivote tabla. Así, la próxima vez que se abra la ventana, estos ajustes se aplicarán automáticamente. 

El usuario ya no tiene que configurar la control cada vez que se muestra.

El menú contextual de una celda permite al usuario cambiar fácilmente la Color de una celda en la control Tabla dinámica. Esto facilita la identificación de elementos importantes en la tabla dinámica. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Cellule_Coul1.gif)

### Configurar el menú de la automatic en una control Tabla dinámica
<a name="configurar_menu_automatic_una_control_tabla_dinamica_ELTPARAGRAPHE000324"></a>

El menú contextual de una control Tabla dinámica se puede desactivar. Para configurar la visualización del menú contextual, debe:

1. Abre la ventana de la description de la control Tabla dinámica.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Es posible: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, se utilizará el menú automatic del control Tabla dinámica (menú AAF). Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Menu2.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [PopupMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.

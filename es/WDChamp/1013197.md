


## Propiedades associated con el panel de pestañas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000099"></a>
Un panel de pestañas que se encuentra en una ventana o en una Page puede manejarse mediante las siguientes propiedades de WLanguage.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propiedades disponibles en una ventana

### Propiedades disponibles en una ventana
<a name="propiedades_disponibles_una_ventana_ELTPARAGRAPHE000020"></a>Las siguientes propiedades de WLanguage son associated con un panel de pestañas:



|   |   |
| --- | --- |
| [Alias](../Proprietes/2510004.md) | La propiedad **Alias** permite: <br><br>- obtener y cambiar el alias de una ventana (WINDEV y WINDEV Mobile).<br><br>- obtener y cambiar el alias de una pestaña dinámica (WINDEV).<br><br>- obtener el nombre interno HTML/JavaScript de un control (WEBDEV).<br><br><br> |
| [BackgroundColor](../Proprietes/2510022.md) | La propiedad **BackgroundColor** permite establecer el color de fondo de un control, ventana, página o bloque de un reporte. |
| [Caption](../Proprietes/2510053.md) | **En una base de datos**, el **Caption** Property se utiliza para:<br><br>- Averigua la leyenda associated con un elemento de archivo. <br><br>- Averigua la leyenda associated con un archivo de datos. <br><br>- Descubrir y modificar la leyenda de conexión. <br><br><br><br><br>**En un reporte**, el **Caption** Property se utiliza para:<br><br>- Averigua el texto que aparece en una estática o control Enlace.<br><br>- Modificar el texto que se muestra en una estática o control Enlace.<br><br><br><br><br>**En una ventana o página**, el **Caption** Property también se utiliza para:<br><br>- Averiguar y modificar la leyenda de un control en una ventana o en un Page.<br><br>- Descubrir y modificar el título de una opción de menú.<br><br>- Averiguar y modificar el título de una ventana o Page.<br><br>- Descubra el contenido de un control Migas de pan.<br><br><br><br><br> |
| [Color](../Proprietes/2510071.md) | El **Color** Property se utiliza para get o cambiar la fuente Color para:<br><br>- una fuente Variable.<br><br>- un texto visualizado en un control Estático o en un control calculado de un reporte.<br><br>- un texto mostrado en una ventana control.<br><br>- un texto mostrado en una página control.<br><br><br> |
| [FullName](../Proprietes/2514005.md) | El **FullName** Property se utiliza para averiguar el nombre completo:<br><br>- de una control.<br><br>- de una ventana.<br><br>- de una opción de menú.<br><br>- de una columna tabla.<br><br><br><br><br> |
| [Grayed](../Proprietes/1000019648.md) | El **Gris** Property se utiliza para: <br><br>- Determinar si un control (o un grupo de controles) está Grayed fuera. <br><br>- Ennegrecer o desennegrecer un control, un grupo de control s o una ventana.<br><br><br> |
| [HelpNumber](../Proprietes/2510141.md) | La propiedad **HelpNumber** permite obtener y modificar el número de ayuda asociado al control. |
| [Image](../Proprietes/2510034.md) | El **Image** Property obtiene y establece la imagen de fondo de un control o bloque. |
| [InitialState](../Proprietes/2510063.md) | El **InitialState** Property se utiliza para get el estado inicial de un control. |
| [InitialVisible](../Proprietes/2510070.md) | La propiedad **..InitialVisible** permite saber si:<br><br>- un control fue visible durante la primera visualización de la ventana o página asociada.<br><br>- una ventana era visible durante su primera visualización.<br><br><br> |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [State](../Proprietes/2510039.md) | La **Estado** Property se utiliza para get y cambiar el estado de un elemento. |
| [ToolTip](../Proprietes/2510005.md) | El **ToolTip** Property se utiliza para get y cambiar el texto mostrado en el tooltip associated con una ventana o página control. |
| [Undocked](../Proprietes/1000026078.md) | El **Undocked** Property se utiliza para determinar si un panel de una Ficha o control Cinta de opciones está desacoplado. |
| [Visible](../Proprietes/2510138.md) | El **Visible** Property se utiliza para:<br><br>- Averiguar si un elemento es Visible.<br><br>- Hacer visible un elemento o Invisible.<br><br><br> |
| [WinEditAllowed](../Proprietes/1000021850.md) | El **WinEditAllowed** Property se utiliza para determinar y configurar las acciones permitidas en un control cuando la ventana pasa al modo "modificación del UI por el usuario final". |





Observación: Dependiendo de la plataforma utilizada, puede que algunas propiedades no estén disponibles. Para obtener más información, consulte la página de ayuda de la propiedad.

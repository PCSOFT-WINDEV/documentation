
## Vinculación de una barra de desplazamiento control a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000106"></a>
Una barra de desplazamiento puede estar vinculada a un elemento en un archivo de datos, consulta, etc.

En este caso, las características de la posición son las siguientes:

- el elemento es de tipo **Integer**.

- el elemento contiene el valor de la posición en la caja de la barra de desplazamiento. Esta posición corresponde al valor de la barra de desplazamiento.

- el valor del elemento debe estar entre el valor mínimo y máximo de la barra de desplazamiento.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manejo del elemento y de la barra de desplazamiento control
<a name="manejo_del_elemento_barra_desplazamiento_control_ELTTEXTE000130"></a>


### Actualización de la barra de desplazamiento control con el valor del elemento
<a name="actualizacion_barra_desplazamiento_control_con_valor_del_elemento_ELTPARAGRAPHE000024"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo relacionado) serán actualizados por [FileToScreen](../WDLang4/3044210.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE2_2"></a>


### Actualización del elemento con los datos de la barra de desplazamiento control
<a name="actualizacion_del_elemento_con_los_datos_barra_desplazamiento_control_ELTPARAGRAPHE000044"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la ventana serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```




## control Enlace associated con un artículo que contiene un correo electrónico Address
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000147"></a>
En una base de datos, el correo electrónico Address de los clientes, proveedores, ... es uno de los elementos almacenados. ¿Por qué no crear directamente los enlaces a estas direcciones en su sitio?

WEBDEV permite crear fácilmente un control Enlace que se utiliza para enviar automáticamente un Email. Por ejemplo, un usuario de la web que esté consultando una lista de proveedores puede enviar directamente una Email a uno de ellos. Al hacer clic en la control Enlace, se abrirá el software de mensajería del usuario de la web y se mostrará el correo electrónico Address del destinatario.

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000171"></a>


### Para crear una control Enlace asociada a un artículo que contenga un correo electrónico Address
<a name="para_crear_una_control_enlace_asociada_articulo_que_contenga_correo_electronico_address_ELTPARAGRAPHE000020"></a>

Para crear un control de enlace asociado a un elemento que contenga un correo electrónico Address: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Enlace".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.

3. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

4. Ir a la pestaña "General": 

	- En la sección "Acción de enlace", seleccione el tipo "Enlace personalizado (correo electrónico, Phone, etc.)".

	- En la sección "Address", seleccione "Email".




5. Vaya a la pestaña "Detalles" y seleccione:

	- "Agregar prefijo automáticamente a la propiedad URL".

	- "..Value representa la URL del enlace (compatibilidad con WEBDEV 1.5)".




6. Vaya a la pestaña "Binding" y seleccione el elemento que desea vincular a la control.

7. Validar el control description.



<a name="NOTE2_2"></a>


### RAD Truco de página: Creación automática del enlace associated con el correo electrónico Address
<a name="rad_truco_pagina_creacion_automatica_del_enlace_associated_con_correo_electronico_address_ELTPARAGRAPHE000061"></a>

1. En el editor de análisis, visualice la ventana de partidas del expediente correspondiente ("Descripción de partidas" de el menú contextual).

2. Seleccione el elemento que contiene un correo electrónico Address.

3. Haga clic en "Parámetros de la control vinculados al elemento seleccionado (información compartida)".

4. Aparece la ventana de gestión de la información compartida. 

5. Seleccione "Enlace" en "Tipo de control". Se muestra la ventana description de la control Enlace creada por la RAD.

6. Ir a la pestaña "General": 

	- En la sección "Acción de enlace", seleccione el tipo "Enlace personalizado (correo electrónico, Phone, etc.)".

	- En la sección "Address", seleccione "Email".




7. Vaya a la pestaña "Detalles" y seleccione:

	- "Agregar prefijo automáticamente a la propiedad URL".

	- "..Value representa la URL del enlace (compatibilidad con WEBDEV 1.5)".




8. Vaya a la pestaña "Binding" y seleccione el elemento que desea vincular a la control.

9. Validar el control description y la ventana de elementos del archivo de datos.

10. Generar el análisis: en la pestaña "Análisis", en el grupo "Análisis", despliegue "Generación" y seleccione "Generación".




Al crear una página RAD, la control Enlace se creará automáticamente con todas las características necesarias.

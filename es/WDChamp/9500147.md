<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000253"></a>
Esta ayuda Page presenta las características de un control Imagen que se encuentra en una ventana. 

Recordatorio: Para crear un control Imagen: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Imagen".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual del control.

## Opciones de la ventana de description
<a name="opciones_ventana_description_ELTTEXTE000277"></a>


### Pestaña "General"
<a name="pestana_general_ELTPARAGRAPHE000037"></a>

En la pestaña "General" se puede definir la imagen que se muestra y sus características. 

**Elegir la imagen que se muestra en el control**: 
La imagen visualizada por el control se selecciona mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), opción "Examinar".

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Si el control Imagen permite mostrar un archivo PDF** , simplemente seleccione el archivo para mostrar. Las siguientes propiedades de WLanguage se utilizan para configurar el PDF Page visualizado : 


|   |   |
| --- | --- |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [ImageMode](../Proprietes/2510058.md) | Permite conocer y cambiar el modo de visualización de una Image. |
| [NumberPage](../Proprietes/2510084.md) | Se utiliza para averiguar el número de páginas de un archivo PDF que se muestra en una control Imagen. |
| [PageNumber](../Proprietes/2510106.md) | Permite conocer y modificar el número Page de un fichero PDF visualizado en una control Imagen. |





![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Atención**: La visualización de un archivo PDF en un Image requiere el [Framework GDI+](../Editeurs/2020007.md). 



**Otras opciones**: Es posible definir: 

- el uso de [Escala de 9 sectores](../WDChamp/1010040.md).

- el modo de visualización, la transparencia, la posición y la simetría de las imágenes. Para obtener más información, consulte [Seleccionar el modo de visualización de la imagen](../WDChamp/1013134.md).





### Pestaña " UI "
<a name="pestana_ELTPARAGRAPHE000098"></a>

En la pestaña "UI", puede definir: 

- el estado inicial de la control,

- si se muestra o no un símbolo de representación en la control. Este símbolo es una miniatura mostrada por Default para representar el control Imagen en el editor.
	<br>![Image con símbolo de representación](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Image_Mire%20-%20HC%20N%B0001.gif)<br>![Image sin símbolo de representación](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Image_Mire%20-%20HC%20N%B0001%201.gif)
 Esta opción también se puede activar/desactivar para todos los controles Image en las opciones del editor de ventanas. Para obtener más información, consulte [Opciones del editor de ventanas](../Editeurs/9000166.md). 

- la acción realizada durante un clic derecho,

- el menú contextual del control, 

- ...





### pestaña "Detalles"
<a name="pestana_detalles_ELTPARAGRAPHE000120"></a>

La pestaña "Detalles" permite definir:

- Los parámetros de visualización

	- **Cargar en segundo plano**: Esta opción se utiliza para evitar el bloqueo de la aplicación si la Image debe ser cargada desde un sitio. Una animación se muestra automáticamente en el control Imagen si es necesario.. 
			**Observación**: Si la control Imagen se encuentra en una control Looper, la Image siempre se carga en la tarea de la Background (sin importar si la opción está habilitada o no).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) **Oriente de acuerdo con los datos Exif de Image**: Si esta opción está activada, una foto se mostrará automáticamente en la dirección adecuada (vertical u horizontal) según los datos Exif de la foto.
			

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Barras de desplazamiento y zoom automáticos**:  Esta opción es necesaria para mostrar los archivos PDF en los controles de Image. Si el archivo PDF no se puede mostrar por completo en el control Imagen, se muestran las barras de desplazamiento y el zoom está disponible pulsando Ctrl + rueda del ratón.. El zoom también puede manejarse mediante programación con [Zoom](../Proprietes/1000017201.md).




- Las opciones de animación
	Para obtener más información, consulte [Animar una imagen](../WDChamp/1013244.md).

- Opciones de clic
	Para obtener más información, consulte [Image clicable en una ventana](../WDChamp/1013133.md).





### Pestaña "Estilo"
<a name="pestana_estilo_ELTPARAGRAPHE000173"></a>

La pestaña "Estilo" permite definir el estilo de los distintos elementos de control Imagen. Es posible definir: 

- Una animación al cambiar la imagen mostrada en la control (elemento "animación control"). El botón "Parámetros" sirve para definir la duración de la animación.

- Un borde (elemento "Borde exterior").

- Una sombra difuminada (elemento "sombra difuminada").




**Para definir el borde Image**: 

1. Seleccione el elemento "Fuera Border ". 

2. Ampliar la lista "Frontera. Se muestran los bordes disponibles. 

	- Las opciones "Redondo xxx" permiten definir un borde redondo. Si se seleccionan estas opciones, una flecha azul permite abrir la ventana de configuración de estilos en la pestaña "Estilo".. 

	- La opción "Image (escalado 9-rango de resolución)" permite elegir un Image para la frontera y configurarlo mediante el escalado 9-rango de resolución. 

	- La opción "Personalizado" se utiliza para definir un borde personalizado. Si se selecciona esta opción, una flecha azul permite abrir la configuración de la forma personalizada.  

	- La opción "Redondo" sirve para crear una Image redonda . Si se seleccionan estas opciones, una flecha azul permite abrir la ventana de configuración (espesor y estilo de Line)
			Atención: Para evitar que se distorsione, el modo de visualización de la Image debe ser homotético.







Observación: El ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bouton_rond%20-%20HC%20N%B0001%202.gif) (a la derecha de la opción "Bordes") permite configurar la sombra de los bordes. Si utiliza una sombra, puede elegir: 

- el desplazamiento X e Y de la sombra,

- el porcentaje de opacidad,

- el radio de desenfoque,

- el color de la sombra.

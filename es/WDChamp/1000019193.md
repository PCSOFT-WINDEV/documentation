## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
El gráfico de embudo muestra los datos en formato de embudo. Una vez totalizados, estos datos son iguales a 100%. Este tipo de gráfico es un gráfico con una sola serie. Representa los datos como partes de 100%. Este tipo de gráfico no utiliza ningún eje.

Los gráficos de embudo se utilizan a menudo para representar los pasos de un Process de ventas y para indicar la cantidad de ingresos potenciales esperados para cada paso. 

El gráfico de Embudo es especialmente adecuado para ilustrar las oportunidades de negocio. Las oportunidades más probables de ser firmadas se encuentran en la parte inferior del embudo, mientras que las menos prometedoras se encuentran en la parte superior.. 

Esta es una solución muy useful para ayudar a los empresarios en sus decisiones.

## Características de un gráfico de embudo
<a name="caracteristicas_grafico_embudo_ELTTEXTE000168"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=genEntonnoir.gif&type=thumb)


En este gráfico:

**1. Título del gráfico**. Las características del título del gráfico pueden definirse en la pestaña "Detalles" de la descripción del control (botón "Editar título del gráfico").

Puede definir el título del gráfico, la posición y estilo.

**2. Cuello de embudo**. Las características del cuello del embudo pueden definirse en la pestaña "General" del control description: 

- Anchura del cuello (en porcentaje). 

- Altura del cuello (en porcentaje). 




**3. Primer valor**. Usted tiene la capacidad de mostrar (o no) el valor total en el gráfico de embudo. Para evitar la visualización de este valor, basta con marcar "Ocultar primer valor" (pestaña "General" de la ventana de description).  

**4. mito**. Las características de la leyenda pueden definirse en la pestaña "Detalles" del control description ("Editar leyenda del gráfico").

Puede definir su posición y estilo.

**5. Etiqueta de categoría**. Las etiquetas de las categorías se muestran en la leyenda. Las etiquetas de las categorías se pueden definir en la pestaña "Serie" del control description. La fuente de las etiquetas de categoría puede corresponder a una columna tabla, un WLanguage array, una lista, ... 

Las etiquetas de las categorías también se pueden mostrar directamente en cada sección del embudo.. Para ello, seleccione "Mostrar etiquetas de categoría" en la pestaña "General" de la ventana de control de description. 

**6. Color de las secciones**. La Color de las secciones se define automáticamente. Para modificar la Color de las secciones, utilice [grSeriesColor](../WDLang3/3042013.md) y especifique el número de sección. 

**7. Gradient colores**. Las características relativas al aspecto del gráfico se definen en la pestaña "Detalles" de la ventana del gráfico description. Las siguientes opciones están disponibles:

- Reducir los efectos dentados.

- Colores de borde automáticos.

- Colores de Gradient (con posible Gradient horizontal).




**Otras características**: 

- **Distancia entre secciones**. El espacio entre secciones puede definirse mediante "Espacio entre las secciones: " (pestaña "General" de la ventana de descripción del control) o mediante [grColumnSpacing](../WDLang3/3042014.md). 

- **Tamaño de las secciones**. La pestaña "General" de la ventana de control description (o [grParameter](../WDLang3/1000018337.md)) permite elegir el tamaño de las secciones: 

	- Proporcional a la superficie: El valor está simbolizado por el área de la sección de embudo.

	- Proporcional a la altura: El valor está simbolizado por la altura de la sección del embudo.

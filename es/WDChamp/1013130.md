


## Recuperando las coordenadas de un clic en un Image clicable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000091"></a>
Si la acción de clic de la Image es set a "Llamada al servidor", puede get las coordenadas de clic cuando se hace clic en la Image en el navegador.

En función de la posición del ratón, podrá realizar una Process específica (cambiar Color, mostrar un nuevo Image, etc.).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperación de las coordenadas del ratón
<a name="recuperacion_las_coordenadas_del_raton_ELTTEXTE000115"></a>
La posición del cursor del ratón sólo se puede recuperar en el código del servidor. Para recuperar la posición del cursor del ratón, utilice las siguientes funciones:


|   |   |
| --- | --- |
| [ImagenYPos](../WDLang2/3045001.md) | Devuelve la posición vertical (Y) del cursor del ratón en relación con la control Imagen especificada. |
| [ImageXPos](../WDLang2/3045003.md) | Devuelve la posición horizontal (X) del cursor del ratón con respecto a la control Imagen especificada.. |



**Ejemplo**: Visualización de un cuadro de información con la posición del ratón en la Image clicable (IMG_Click1).


```wl
// -- server code: Click on IMG_Click1
PositionX, PositionY are int
// Retrieve the position of the mouse in a clickable image
PositionX = ImageXPos()
PositionY = ImageYPos()
Info("You've clicked at position: ", PositionX, PositionY)
```


Observación: También puede utilizar [PageParameter](../WDLang2/3058026.md) con el parámetro POS. Por ejemplo: 


```wl
// Retrieve the position of the mouse in a clickable image
Parameter is string = PageParameter("POS")
// Ex: Parameter ="15,23"
x,y are int
x = ExtractString(Parameter, 1, ",")
y = ExtractString(Parameter, 2, ",")
```

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000210"></a>
El control Botón de opción permite al usuario seleccionar una opción entre varias.

Un botón de opción puede mostrar hasta 256 opciones en varias filas y columnas. El usuario solo puede seleccionar una opción. El botón de opción devuelve el número de la opción seleccionada.

## Crear un control Botón de opción
<a name="crear_control_boton_opcion_ELTTEXTE000234"></a>


### Crear un control Botón de opción
<a name="crear_control_boton_opcion_ELTPARAGRAPHE000063"></a>

Para crear un control Botón de opción:

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Botón de opción".

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control. El control aparece en el editor.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Consejo**: Se puede crear un botón de opción "predefinido" desplegando "Botón de opción". La lista de controles disponibles se muestra con la plantilla del proyecto. Simplemente haga clic en el control que desea crear.

Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Se puede definir un tooltip general para el control o un tooltip específico para cada opción.
<a name="NOTE2_2"></a>
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Limitaciones en Java
<a name="limitaciones_java_ELTPARAGRAPHE000097"></a>

Las siguientes características no están disponibles en Java:

- columnas proporcionales (si las opciones del botón de opción se muestran en varias columnas).

- títulos multilínea de los botones de opción.

- texto con formato de las opciones de los botones de opción (R. T. F.).

- número de ayuda asociado a cada opción del botón de opción.

## Modos de visualización disponibles
<a name="modos_visualizacion_disponibles_ELTTEXTE000270"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
WINDEV y WINDEV Mobile proponen diferentes modos de visualización para los botones de opción:

- botón de opción estándar,

- botón de opción estándar con subtítulos,

- botón de opción con estilo Vista (se puede usar una imagen para cada opción),

- botón de opción con estilo Vista y subtítulos (se puede usar una imagen para cada opción). 


El modo de visualización del botón de opción se selecciona en la pestaña "General" de la descripción del control. En la pestaña "Contenido" se pueden introducir las características propias del modo de visualización (imagen, subtítulo, alto de fila, etc.) en los diferentes idiomas de la aplicación. Estos elementos se pueden modificar por programación.

El modo de programación puede cambiar según el tipo de visualización seleccionado. Para obtener más información, consulte [Manipular un botón de opción por programación](../WDChamp/1013254.md).

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Solo está disponible el modo estándar y el modo estándar con subtítulos.

## Cambiar
<a name="cambiar_ELTTEXTE000294"></a>
WINDEV, WINDEV Mobile y WEBDEV permiten transformar automáticamente un control Botón de opción en un control Casilla de verificación (y viceversa): 

1. Seleccione el control que desea transformar. 

2. En la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Botón de opción/Casilla de verificación".

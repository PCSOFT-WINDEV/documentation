## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
En un gráfico, las escalas se utilizan para estimar rápidamente el valor de los diferentes puntos que se encuentran en un gráfico.

WINDEV, WEBDEV y WINDEV Mobile proponen una gestión avanzada de las escalas mostradas en un control Gráfico.

## Características de las balanzas
<a name="caracteristicas_las_balanzas_ELTTEXTE000129"></a>


### Características generales
<a name="caracteristicas_generales_ELTPARAGRAPHE000028"></a>

En la mayoría de los gráficos que incluyen escalas, la control Gráfico está acostumbrada:

- defina el ***paso para las marcas***. Por Default, las marcas se calculan automáticamente a partir de los valores encontrados en las diferentes series. Para personalizar las marcas, basta con especificar el paso deseado en la opción "Marcas" de la pestaña "General" de control.

- definir ***la orientación de las marcas***. Mediante Default, la orientación de las marcas se adapta al espacio de las mismas. Tiene la posibilidad de forzar la visualización horizontal o vertical del texto en las marcas. Para personalizar la orientación de las marcas, basta con seleccionar la orientación deseada en la pestaña "General" de control.

- definir ***el valor mínimo y máximo de las marcas***. Según Default, estos valores dependen de los valores encontrados en las diferentes series. Estos valores pueden definirse en la pestaña "General" del control description.

- defina el espacio en píxeles entre 2 marcas horizontales (opción "Espacio" en la pestaña "General" del control description).

- definir el formato de las marcas verticales ("Formato de las marcas" en la pestaña "General" del control description). 





### Características específicas
<a name="caracteristicas_especificas_ELTPARAGRAPHE000039"></a>

También tiene la capacidad de definir escalas específicas:

- **Escala de las abscisas espaciadas según el tiempo**. Esta opción funciona con datos de Fecha u Hora procedentes de: 

	- un control Tabla, 
			

	- un elemento HFSQL. 

	- un array. 
			


Esta opción no puede utilizarse con una serie rellenada mediante programación.

- **Escala de coordenadas logarítmicas Y**.

- **Escala secundaria**: Para cada serie, se puede especificar si la escala utilizada para la serie es la escala estándar o la escala secundaria.. La escala secundaria se muestra a la derecha del gráfico.

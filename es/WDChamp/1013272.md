<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000125"></a>
WEBDEV le permite manipular un IFrame control a través de la programación. Para ello, utilice el Variable del IFrame control directamente en el código.

El Variable del IFrame control corresponde al nombre del IFrame control. El tipo Variable es una cadena de caracteres.

**Importante**: Un IFrame control puede ser programado en código de servidor.

## Mostrando una página en un IFrame control
<a name="mostrando_una_pagina_iframe_control_ELTTEXTE000149"></a>


### Cambiando el objetivo del IFrame control en el editor
<a name="cambiando_objetivo_del_iframe_control_editor_ELTPARAGRAPHE000022"></a>

El objetivo de un iFrame control se puede cambiar fácilmente a través de un botón o un enlace.

El objetivo puede ser cambiado sólo si el objetivo del iFrame control pertenece al sitio actual.

Para ello, cree un botón o un enlace con las siguientes características:

- Operación en los controles: "Ninguna".

- Acción: "Mostrar la página XXX". Todo lo que tiene que hacer es seleccionar la página a mostrar.

- Destino: "En un IFrameXXX". Todo lo que tienes que hacer es seleccionar el nombre del iFrame control.




**Observación**: Si el objetivo del control del iFrame no pertenece al sitio actual, el navegador INTERNET no permite modificar este objetivo (por razones de seguridad).
<a name="NOTE2_2"></a>


### Cambiar el objetivo del IFrame control mediante programación
<a name="cambiar_objetivo_del_iframe_control_mediante_programacion_ELTPARAGRAPHE000039"></a>

Para cambiar el objetivo del IFrame control a través de la programación, utilice:

- [PageDisplay](../WDLang2/3058008.md) y especifique el alias del iFrame control en el que se debe mostrar la página (código del navegador).

- [ChangeTarget](../WDLang2/3058005.md) y especificar el alias del iFrame control en el que se debe mostrar la página (código del navegador) y luego [PageDisplay](../WDLang2/3058008.md) en el código del servidor.




El objetivo puede ser cambiado sólo si el objetivo del iFrame control pertenece al sitio actual.

**Observación**: Si el objetivo del control del iFrame no pertenece al sitio actual, el navegador INTERNET no permite modificar este objetivo (por razones de seguridad).

## Propiedades associated con el IFrame control
<a name="propiedades_associated_con_iframe_control_ELTTEXTE000179"></a>
Las siguientes propiedades se utilizan para gestionar los controles IFrame por programación.


|   |   |
| --- | --- |
| [escribir a máquina](../Proprietes/2510131.md) | Permite get el tipo de un objeto |
| [InitialValue](../Proprietes/2510128.md) | Permite get el valor inicial de un control. Para un control IFrame, este Property corresponde al nombre de la página del sitio o a una cadena vacía si el control es associated con una página de otro sitio. |

Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con un IFrame control, consulte [Propiedades associated con un IFrame control](../WDChamp/1013271.md).

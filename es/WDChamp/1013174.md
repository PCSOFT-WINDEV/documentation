
## control ListView relacionado a un archivo de datos (o a una consulta)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000098"></a>
Se puede utilizar un control ListView para mostrar el contenido de un archivo de datos (o el contenido de una consulta).

Cuando un control ListView es relacionado a un archivo de datos (o a una consulta), este control puede ser inicializado con el contenido del archivo de datos:

- automáticamente: No se requiere un código específico para inicializar el control ListView. Las características de la navegación realizada en el archivo de datos relacionado al control se describen en la pestaña "Contenido" del control description.

- mediante programación: El código utilizado para navegar de el archivo de datos relacionado a control ListView debe ser introducido en el código de inicialización del control. Este método no es recomendable.




El contenido de dos artículos diferentes se muestra en el control ListView:

- una posición correspondiente a los títulos (por ejemplo, posición"Texto").

- un elemento que corresponda a las imágenes (por ejemplo, un campo de tipo "Sonido, imagen, binario").




**Observación**: Al crear una control ListView, la asistente propone crear los diferentes tipos de vistas de listas. Para una control ListView relacionado a un archivo de datos o a una consulta, la asistente pide la información necesaria (fichero de datos utilizado, elementos utilizados, etc.).

## Código de inicialización de una control ListView programada
<a name="codigo_inicializacion_una_control_listview_programada_ELTTEXTE000122"></a>
Si se utiliza un control ListView basado en un archivo de datos con inicialización programada, el código para navegar por el archivo de datos debe introducirse en el código de inicialización del control.

**Ejemplo**: Iniciando un control ListView con las fotografías encontradas en el archivo de datos de VIAJES.


```wl
// -- Initialization code of ListView control
HReadFirst(TRAVEL, Photo)
WHILE NOT HOut()
	ListAdd(LSV_TravelPhoto, TRAVEL.Photo)
	HReadNext(TRAVEL, Photo)
END
```

## Miniaturas HFSQL
<a name="miniaturas_hfsql_ELTTEXTE000146"></a>
El ítem relacionado a las imágenes mostradas en el control ListView puede usar miniaturas HFSQL. En este caso, se optimizará la visualización de las imágenes.

Para obtener más información, consulte [Miniaturas HFSQL](../WDLang4/3044348.md).

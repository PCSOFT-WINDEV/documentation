
## Manejo de una pantalla formateada control mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
WEBDEV le permite manipular un formato de visualización control a través de la programación. Para hacerlo, usa el Variable de la pantalla formateada control en el código.

El Variable de la pantalla formateada control corresponde al nombre de la pantalla formateada control. El tipo de la Variable depende del tipo de control (numérico, texto, moneda, ...). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Iniciando una pantalla formateada control (código de servidor)
<a name="iniciando_una_pantalla_formateada_control_codigo_servidor_ELTTEXTE000135"></a>
Para inicializar un control de pantalla formateado, una simple asignación del control Variable es suficiente. Esta inicialización debe realizarse en código del servidor. Por ejemplo:


```wl
// Initialize the formatted display control named "FSTC_Date"
FSTC_Date = 19990506 
// Other possibility: Date.Value = "19990506"
```


**Observación**: también puede utilizar las propiedades [Valor](../Proprietes/2510130.md) o [Pie de foto](../Proprietes/2510053.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Recuperar el valor de una pantalla formateada control (código de servidor)
<a name="recuperar_valor_una_pantalla_formateada_control_codigo_servidor_ELTTEXTE000159"></a>
Para recuperar el contenido de una pantalla formateada control (estática o dinámica) en un código de servidor:

- realizar una asignación simple. Por ejemplo:
	
	```wl
	CurrentDispContent = FSTC_FORMATTED1
	```


- manipular la pantalla formateada control directamente. Por ejemplo:
	
	```wl
	Info(FSTC_FORMATTED1)
	```


- utilizar las propiedades [Valor](../Proprietes/2510130.md) o [Pie de foto](../Proprietes/2510053.md).

## Utilice el ejemplo
<a name="utilice_ejemplo_ELTTEXTE000183"></a>


### ¿Cómo puedo mostrar el siguiente texto en una página? "fecha de la última actualización: dd/mm/aaaaa":
<a name="¿como_puedo_mostrar_siguiente_texto_una_pagina_fecha_ultima_actualizacion_ddmmaaaaa_ELTPARAGRAPHE000058"></a>

1. Crear un control de visualización con formato.

2. En el código de inicialización de la control, introduzca el siguiente código:
	
	```wl
	MySelf = fDate(CompleteDir(fExeDir()) + "MySite.WDL", "", fModification)
	```
Donde "MySite.WDL" corresponde a la BDM de su sitio.

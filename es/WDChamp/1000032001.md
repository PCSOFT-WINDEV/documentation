## Resumen de control Área de texto enriquecido
<a name="resumen_control_area_texto_enriquecido_ELTTEXTE000211"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

La control Área de texto enriquecido (también llamada RTA) es una control Estático que ofrece varias posibilidades de disposición: 

- Formato de texto,

- Inserción de imágenes o enlaces,

- Inserción de iconos procedentes de la fuente,

- ...




La control Área de texto enriquecido se puede utilizar para formatear una sección de texto en un sitio.. 

En edición, el HTML control Área de texto enriquecido permite al desarrollador escribir el texto a mostrar (con todas las opciones de formato). 

En tiempo de ejecución, el HTML control Área de texto enriquecido permite al usuario ver un texto con formato, imágenes y enlaces.

## Creación de una control Área de texto enriquecido
<a name="creacion_una_control_area_texto_enriquecido_ELTTEXTE000235"></a>


### Crear control
<a name="crear_control_ELTPARAGRAPHE000033"></a>

Para crear una control Área de texto enriquecido: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Texto".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_Libelle_HTML.gif)


2. Seleccione "Área de texto enriquecido".. 

3. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.





### Escribir un texto en el control Área de texto enriquecido
<a name="escribir_texto_control_area_texto_enriquecido_ELTPARAGRAPHE000051"></a>

Para escribir un texto en el campo control Área de texto enriquecido: 

1. Seleccione la control. 

2. Haga clic en la control: una barra amarilla aparece alrededor de la control y la control cambia al modo de edición. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ZTR_Edition_bandeau_Jaune.gif)


3. En el modo de edición: 

	- tiene la posibilidad de escribir su texto directamente. Puede, por ejemplo, pegar un texto procedente de un software como Word: el formato se mantendrá. 

	- la cinta de edición se adapta a la entrada en el control: se visualizan todas las opciones de tratamiento disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ZTR_Edition_ruban.gif)
 Estas opciones de edición se agrupan en 4 categorías: 

		- tipo de letra: se usa para especificar los atributos de fuente para el texto seleccionado. 

		- Párrafo: se utiliza para especificar los atributos de los párrafos (sangrías, viñetas, listas, centrado y justificación)

		- Estilo CSS - Semánticas: se usa para asociar un estilo CSS y semántica al texto seleccionado. 

		- Controles: se utiliza para añadir controles a la control Área de texto enriquecido: 

			- un control Imagen, 

			- a control Enlace,

			- un Icon que viene de la fuente,

			- otro control: todos los controles que se encuentran en el panel "Creación" están disponibles.








### Insertar un enlace en el control Área de texto enriquecido
<a name="insertar_enlace_control_area_texto_enriquecido_ELTPARAGRAPHE000080"></a>

Para insertar un enlace en el control Área de texto enriquecido: 

1. Entrar en edición en el control Área de texto enriquecido.

2. Seleccione el texto que debe ser convertido en enlace. 

3. En la pestaña "Texto", en el grupo "Controles", haga clic en "Enlace". Aparece el enlace de la ventana de description. Esta ventana se utiliza para especificar las características del enlace. La acción associated con el enlace se puede definir en la pestaña "General".




Observaciones: 

- Cuando se inserta un enlace en la control Área de texto enriquecido, este enlace corresponde a un control Enlace. Esta control está incluida en el Área de Texto Enriquecido y puede ser manejada mediante programación a través de su nombre. 

- La lista de controles que se encuentran en la control Área de texto enriquecido se muestra en la pestaña "General" del control description. 

- Para crear un enlace en un control Área de texto enriquecido, también tiene la posibilidad de arrastrar y soltar un control Enlace existente.





### Insertar un Image en el control Área de texto enriquecido
<a name="insertar_image_control_area_texto_enriquecido_ELTPARAGRAPHE000107"></a>

Para insertar un Image en el control Área de texto enriquecido: 

1. Entrar en edición en el control Área de texto enriquecido.

2. Posición en el lugar donde se insertará Image. 

3. En la pestaña "Texto", en el grupo "Controles", haga clic en "Imagen".

4. La imagen de la ventana de description aparece. Esta ventana se utiliza para especificar las características de Image. La pestaña "General" sirve para:  

	- especificar la Image a mostrar,

	- especificar el modo de envoltura de Image y el modo de visualización de Image.  







Observaciones: 

- Cuando se inserta un Image en el control Área de texto enriquecido, este Image corresponde a un control Imagen. Esta control está incluida en el Área de Texto Enriquecido y puede ser manejada mediante programación a través de su nombre. 

- La lista de controles que se encuentran en la control Área de texto enriquecido se muestra en la pestaña "General" del control description. 

- Para crear una imagen en una control Área de texto enriquecido, también tiene la posibilidad de arrastrar y soltar una control Imagen existente.





### Insertar una fuente Icon en el control Área de texto enriquecido
<a name="insertar_una_fuente_icon_control_area_texto_enriquecido_ELTPARAGRAPHE000138"></a>

Para añadir una Icon en un pie de foto HTML: 

1. Posicione el cursor del ratón en el texto HTML a modificar.

2. Haga clic en el botón **Íconos de fuente** ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion_des_polices_icones%20-%20HC%20N%B0001%201.gif). 

3. La siguiente ventana se abre.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion_des_polices_icones%20-%20HC%20N%B0002.gif&type=thumb)


4. Seleccione la familia de iconos en la lista de fuentes.

5. Haga doble clic en la Icon solicitada o seleccione la Icon y valide la ventana. La Icon se añade en el título HTML en la posición del cursor.




Atención: La configuración actual del proyecto debe manipular cadenas Unicode en tiempo de ejecución. Esta opción está disponible en la ventana de configuración de description.

## Características de control Área de texto enriquecido
<a name="caracteristicas_control_area_texto_enriquecido_ELTTEXTE000296"></a>
Se puede acceder a las principales características de la control Área de texto enriquecido a través de la ventana de control de la description. La ventana de la description de la control Área de texto enriquecido se utiliza para: 

- editar el código fuente HTML de la control (pestaña "General"). Este código puede ser modificado directamente en el editor. Es posible: 

	- Añadir código HTML. Si es posible, el resultado del código HTML añadido se verá en el editor (sólo las animaciones no son compatibles).. Estas adiciones se almacenan en el control. 

	- Borrar una sección de código HTML. Tiene la posibilidad de eliminar el código HTML correspondiente a un enlace, por ejemplo. 

	- Rellenar el código HTML generado. Puede, por ejemplo, añadir etiquetas específicas para mostrar un tooltip de ayuda. 




- ver la lista de controles que se encuentran en el área (pestaña"General"). 

- mostrar la ventana de description de los controles que se encuentran en el área (botón "Editar" en la pestaña "General"). 

- definir las opciones de interfaz (pestaña "UI"). Es posible definir: 

	- si la control Área de texto enriquecido puede apilarse.

	- la semántica associated con el control Área de texto enriquecido. 







Observaciones: 

- La ventana description del control s que se encuentra en el control Área de texto enriquecido difiere de la del control fuera del área de texto enriquecido. De hecho, alguna información (especialmente la posición y el estilo) se define en relación con la control Área de texto enriquecido (y no en relación con el Link o control Imagen).

- En una control Estático que se encuentra en una control Área de texto enriquecido, se puede definir un ancla a través de la ventana de la control Estático description (pestaña "General", "Definir como ancla").

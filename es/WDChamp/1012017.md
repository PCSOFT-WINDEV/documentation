## Presentación
<a name="presentacion_ELTTEXTE000080"></a>
En algunos casos, es posible que tenga que proponer la validación de una página mediante un botón (o un enlace) que se encuentra en un marco diferente.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000104"></a>
Para validar una página que no sea la página actual (lo que significa que se encuentra en un marco diferente):

1. Crear un botón (o un enlace) en la página que contiene la información a validar (página que contiene el tabla en el ejemplo). Este botón (o este enlace) puede ser Invisible. Sólo se ejecutará el código de clic del servidor de este botón.

2. En el código del botón (o enlace) utilizado para validar la página (enlace "Validar las correcciones" en el ejemplo), llame a [PageSubmit](../WDLang2/3058020.md) y especifique:

	- el nombre de la página a validar (página que contiene el tabla en el ejemplo).

	- el nombre del botón (o enlace) utilizado para validar la página (control creado en el paso 1).

	- el destino (el nombre de la trama) donde se debe mostrar el resultado de la validación.

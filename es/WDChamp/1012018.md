## Vista general de las páginas
<a name="vista_general_las_paginas_ELTTEXTE000277"></a>
Las páginas son la interfaz principal de un sitio WEBDEV. Las páginas permiten a los usuarios de la web ver y manipular información.

Esta información puede:

- Información dinámica procedente de ficheros de datos, consultas, etc.. Esta información cambia de acuerdo con las peticiones de los usuarios de la web.

- Estática. La información es fija y no cambia.

## Crear una página
<a name="crear_una_pagina_ELTTEXTE000301"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000023"></a>

Para crear una Page: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Página", y luego en "Página".

	- El asistente de creación de páginas se abre automáticamente.




2. Seleccione el tipo de Page que desea crear:

	- **la opción "Estándar"** le permite crear: 

		- una página en blanco sin ningún control. 
						El modo de edición de esta página se define en la pestaña "General" de la ventana de la página description.: 

			- **Responsive (zoning y layouts)** (valor predeterminado): Page en blanco con zonificación y trazados.
										Al utilizar los diseños definidos en escritorio y móvil, este tipo de Page permite crear rápidamente un sitio "Mobile-friendly".. 

			- **Rangos de resolución (modo Responsive heredado)**: Page en blanco con puntos de ruptura Responsive Web Design. 
										Este tipo de Page se mantiene por compatibilidad para crear sitios "Mobile-friendly"..
										Para obtener más información, consulte [Creación de un diseño web responsivo Page](../WDChamp/9000168.md). 

			- **Modo HTML**: Page en blanco cuyo tipo es HTML Stream. 




- un Page basado en un proyecto modelo, si el proyecto contiene un Page modelo.  

- una Page basada en una modelo preestablecida (2 o 3 columnas, barra lateral izquierda o derecha, etc.). 
						Puede generar el Page y el Page modelo, o sólo el Page. 
						La Page generada contiene controles que proporcionan una vista previa. Estos controles pueden ser modificados o borrados posteriormente. 

- **Las "páginas internas"** se utilizan para crear: 

	- una Page interna en blanco que puede incluirse (con su código) en otra Page a través de los controles "Page" internos. 

	- páginas internas específicas para la gestión de los cuadros de diálogo. 




Para visualizar las características de la Page creada, utilice uno de los siguientes métodos: 

- Seleccione la opción "Descripción" en el menú contextual de la página.

- En la pestaña "Modificación", en el grupo "Edición rápida", haga clic en "Descripción".

### Archivos correspondientes a las páginas
<a name="archivos_correspondientes_las_paginas_ELTPARAGRAPHE000105"></a>

Las páginas corresponden a ficheros WEBDEV ".WWH. Por Default, estos archivos se crean en el directorio principal del proyecto.

Al generar las páginas del sitio, los archivos ".WWH" se convierten automáticamente en archivos ".HTM".. Estos archivos HTML se crean en distintos directorios en función del tipo de Page:

- ***página dinámica***: subdirectorios de los idiomas del proyecto (por ejemplo, "US", "FR", etc.).

- ***página estática***: subdirectorios de los idiomas que se encuentran en el directorio "_WEB" del proyecto ("MySite_Web/US", por ejemplo).

### Descripción de una Page
<a name="descripcion_una_page_ELTPARAGRAPHE000125"></a>

La ventana description de un Page contiene varias pestañas que permiten definir todas las características del Page:

- Pestaña " **Generales** ": Se utiliza para definir: 

	- el nombre de la Page, 

	- la descripción, 

	- su título, 

	- su tipo (Sesión, AWP, estática, PHP, etc.), 

	- su modo de edición (RWD basado en diseño o RWD heredado, etc.).




- Pestaña "**UI**" : Se utiliza para definir: 

	- las características del indicador automatic Ajax, 

	- varias opciones de visualización: 

		- orden de tabulación, 

		- planes,

		- [botón atrás](../WDChamp/1012001.md),




- [adaptación para móviles](../NonVisible/2030057.md), 

- Aplicaciones web para iOS, 

- cursor hover de la Page. 

- Talla Page. 

- opciones de anclaje de la Page. 

- Pestaña **"Detalles"**:  Se utiliza para definir:

	- el contexto HFSQL utilizado, 

	- [Page SEO](../Editeurs/2030037.md),

	- [las opciones para la validez de la Page](../WDChamp/1012021.md),

	- [los archivos JavaScript (.js) que hay que añadir a Page](../Editeurs/1012027.md). Estos archivos serán utilizados por el HTML generado Page.

	- los archivos CSS que hay que añadir a Page. Estos archivos serán utilizados por el HTML generado Page.




- Pestaña **"Cuadro"**: Se utiliza para definir las características de los marcos.

- Pestaña **"Idiomas"**: Sirve para definir las lenguas disponibles en el sistema Page.

- Pestaña **" note "**: Permite definir el título de las notas y su contenido.. Esta información se imprimirá en la documentación técnica.

- Pestaña " **Avanzado** ": Está acostumbrado:

	- definir la [opciones avanzadas para la generación de Page](../WDChamp/1012030.md).

	- añadir código HTML a la cabecera de la Page.




- Pestaña "**Estilo**": Se utiliza para definir:

	- las características del tema associated con el Page. Para obtener más información, consulte [Gestión de las pieles](../Editeurs/2016020.md). 

	- la hoja de estilo associated con la hoja de estilo Page.

	- las opciones del estilo Page: 

		- Fondo navegador. 

		- Page Background y fronteras. Por ejemplo, la imagen de fondo associated con el Page. Para obtener más información, consulte [Página y fondo Image](../WDChamp/1012031.md). 

		- Colores predeterminados: Esta opción le permite definir los diferentes colores utilizados para el texto de la Page, enlaces, enlaces visitados, ... Esta información modifica el código de la etiqueta  del HTML Page.

### Creación de una Page a partir de una Page interna (e inversamente)
<a name="creacion_una_page_partir_una_page_interna_inversamente_ELTPARAGRAPHE000199"></a>

WEBDEV permite crear una Page a partir de una Page interna (e inversamente).

- Para crear una página interna a partir de la página actual, en la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar página como página interna".

- Para crear una página a partir de la página interna actual, en la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar página interna como página".

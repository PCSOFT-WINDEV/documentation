## ¿Barra de progreso simple o simétrica?
<a name="¿barra_progreso_simple_simetrica_ELTTEXTE000111"></a>
Le recomendamos que lo haga:

- utilice el **barras de progreso simples** para ver un progreso (por ejemplo, una transferencia de archivos). La barra se mueve desde el origen de la barra de progreso hasta el final durante toda la Process.

- usar **barreras de progreso simétrico** para ver la evolución de un valor en relación con un valor reference. Esto le permite ver la evolución de una temperatura, por ejemplo. Por ejemplo:
	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Jauge_Sym1.GIF)![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Jauge_Sym2.GIF)




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No se puede crear una barra de progreso simétrica.

## Image y barra de progreso: Visualización del progreso
<a name="image_barra_progreso_visualizacion_del_progreso_ELTTEXTE000135"></a>
Para ver la progresión de una barra de progreso, puede hacerlo:

- definir un estilo específico para la barra de progreso (y para la barra de progreso de Active en caso de una barra de progreso simétrica). Seleccione el estilo deseado en la pestaña "Estilo" de la barra de progreso.

- usar imágenes.




Una barra de progreso puede contener:

- un fondo Image.

- un Active Image. Esta Image se mostrará durante la progresión de la barra de progreso.




Estas dos imágenes se pueden combinar para producir efectos especiales.

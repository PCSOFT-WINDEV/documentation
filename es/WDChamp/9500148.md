## Presentación
<a name="presentacion_ELTTEXTE000182"></a>
Esta página de ayuda presenta las características de un control Imagen que se encuentra en una página. 

Recordatorio: Para crear una control Imagen: 

1. En el panel "Creación", en el grupo "Controles habituales", expanda "Image" y seleccione "Image".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en modo de creación.




**Para ver las características del control**, seleccione la opción "Descripción" en el menú contextual del control..

## Opciones de la ventana de description

### "Ficha"General
<a name="fichageneral_ELTPARAGRAPHE000025"></a>

En la pestaña "General" se puede definir la imagen que se muestra y sus características. 

**Image source**: 

- **TTipo de Image**: Varios tipos de imágenes están disponibles en un sitio WEBDEV. Cada tipo corresponde a un uso específico. Ver [Eligiendo el tipo de Image: dinámico, estático, generado, ....](../WDChamp/1013132.md) para más detalles. 
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Las imágenes dinámicas no son compatibles.

- **Image**: La imagen mostrada por la control se selecciona mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), opción "Examinar".. Dependiendo de la Image seleccionada, se muestra un asistente que le permite:

	- **comprimir el Image automáticamente** (para los Image s cuyo tamaño exceda los 20 KB). Esta opción también permite convertir la imagen en uno de los formatos adecuados para las páginas de INTERNET. Si esta opción no está seleccionada:

		- Si el Image está usando un formato no adaptado a las páginas del INTERNET, el Image se convertirá cada vez que se genere la página, por lo que esta generación puede ralentizarse.

		- Si la Image utiliza un formato adaptado a las páginas, pero si su tamaño es demasiado grande, el tiempo de visualización de la página puede ser más lento.


 Si elige realizar la conversión inmediatamente, tiene la posibilidad de seleccionar el formato Image y su calidad. También tiene la posibilidad de utilizar la función ***JPEG progresivo*** modelo. Este formato muestra una página progresivamente mientras el navegador carga la página.. 

- **convierte automáticamente la imagen** en uno de los formatos adecuados para las páginas de INTERNET. Si no se selecciona esta opción, el Image se convertirá cada vez que se genere la página, por lo que esta generación puede ralentizarse. Si elige realizar la conversión inmediatamente, tiene la posibilidad de seleccionar el formato Image y su calidad.

- **define la ubicación de Image**: puedes copiar el Image en el directorio del sitio (directorio _WEB) para optimizar el tiempo de generación de la página o dejar el Image en su ubicación (recomendado cuando el archivo se comparte entre varios colaboradores).




**Parámetros de visualización**

- **The Image es una hoja de animación**: Esta opción le permite utilizar un Image que será animado por WEBDEV. Todo lo que tienes que hacer es proporcionar un archivo Image que contiene los diferentes dibujos de animación. Ver [Animación de un Image](../WDChamp/1013244.md) para más detalles.  

- **Modo visualización**: Vea [Eligiendo el modo de visualización de Image](../WDChamp/1013134.md) para más detalles.




**Transiciones - Efectos - Desplazamiento automático**

- **Transición/Efectos**: Cuando se modifica el valor de Image control, se puede seleccionar una transición como Desvanecido, Superposición o Golpe para hacer más armonioso el cambio de Image. Vea [Animando la pantalla de Image](../WDChamp/9500117.md) para más detalles.

- **Habilitar el pergamino automatic de imágenes**: Esta opción es accesible para las imágenes "dinámicas" o para las imágenes "de una base de datos: ruta". Vea [Imagen con el pergamino automatic](../WDChamp/9500118.md) para más detalles.




**Acción**
Una acción también puede ser associated con el Image. Consulte [Eligiendo la acción associated con un control](../WDChamp/1013165.md) para obtener más detalles.


### pestaña "Detalles"
<a name="pestana_detalles_ELTPARAGRAPHE000083"></a>

La pestaña "Detalles" sirve para especificar opciones avanzadas. 

**Localiza el Image en el código del navegador _directorio web**: 
Esta opción es marcada por Default cuando se crea un control Imagen. En este caso, en el código del navegador, el Image se busca automáticamente en el directorio _WEB del sitio. No se requiere ningún código específico.
Si esta opción está desmarcada, se debe especificar el Image ruta completo para encontrar el Image. Para hacerlo, usa [FolderWeb](../WDLang2/3012015.md).

**Image protección** : 
Si esta opción está marcada, la el menú contextual de Image no se mostrará en el navegador. Algunas de las operaciones de Image disponibles a nivel de navegador (copia de seguridad, impresión, ...) no se realizarán. Esta opción también está disponible en los controles de [Imagen con enlace](../WDChamp/1013140.md) y en los controles de [Miniaturas en miniatura](../WDChamp/1013275.md).

**Zoom automático**: 
WEBDEV propone el zoom automatic de un Image en un popup. Consulte [Zoom en una imagen](../WDChamp/9500127.md) para obtener más detalles. 

**Arrastrar/Soltar**: 
Esta opción se utiliza para configurar la control Imagen para la gestión de Arrastrar y Soltar. Ver [Arrastrar y soltar automáticamente](../WDLang1/3030012.md) para más detalles. 

**Retraso en la carga**: 
La opción "Cargar el Image cuando el control se muestra en la pantalla" se utiliza para delay el Image cargando. De hecho, cuando un navegador realiza el renderizado de una página, carga la página y sus imágenes desde el servidor.. Este tiempo de carga puede ser bastante largo en algunos casos.. Esta opción permite que el navegador cargue la imagen sólo cuando ésta se muestre en la sección de la página Visible. También tiene la capacidad de especificar la espera Image, usada durante la carga retrasada de Image. 

**Zoom y desplazamiento con los dedos**
Esta opción se utiliza para gestionar el multitáctil en el navegador para las imágenes. Ver [Multi-toque en una control Imagen](../WDChamp/9500131.md) para más detalles.

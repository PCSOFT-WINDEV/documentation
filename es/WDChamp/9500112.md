## Formato HTML
<a name="formato_html_ELTTEXTE000169"></a>
HyperText Markup Language (HTML, por sus siglas en inglés) es un lenguaje de marcado para páginas Web.

En WEBDEV es posible manipular texto HTML mediante la opción "Formato HTML" de los controles Campo de entrada y el [HTML control](../WDChamp/1013105.md).

- Si el texto HTML se muestra en un control que no soporta este formato, se mostrarán las etiquetas. 

- Si el texto HTML se visualiza en una control que admita HTML, las etiquetas se interpretarán automáticamente y el texto se formateará en consecuencia.

## Control Campo de entrada HTML
<a name="control_campo_entrada_html_ELTTEXTE000193"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000026"></a>

La edición HTML control se utiliza para: 

- mostrar contenido HTML,

- introducir contenido HTML.




El control almacena automáticamente el texto introducido como HTML. Este texto contendrá todas las etiquetas de formato necesarias.


### Características de un control
<a name="caracteristicas_control_ELTPARAGRAPHE000036"></a>

El control HTML Edit dispone de varias opciones (pestaña "General" de la ventana description): 

- **Barra de herramientas HTML**: Se puede mostrar una barra de formato en la parte superior del control. Esta barra de herramientas permite al usuario definir fácilmente el formato del texto introducido. Para obtener más información, consulte [Barra de formato HTML](../WDChamp/9500113.md).

- **Acción en la tecla Enter**: Define lo que hace la tecla Intro en la edición HTML control. 

- **Edición de correos electrónicos**: Habilita un modo de entrada "Email" que limita las funcionalidades HTML a las reconocidas por la mayoría de los clientes Email.

- **Contenido independiente del estilo de la página**: Aplica el estilo sólo al campo de entrada. No se aplicará ningún otro estilo al texto del campo de entrada. 




**Observación**: Una misma cadena será más larga en un control Campo de entrada HTML que en un control estándar, porque se añaden etiquetas HTML.


### WLanguage eventos associated con el control
<a name="wlanguage_eventos_associated_con_control_ELTPARAGRAPHE000067"></a>

El HTML Editar control incluye todos los eventos WLanguage associated con Editar control s. Para obtener más información, consulte [Eventos asociados a los controles Campo de entrada](../WDChamp/1013229.md).  

La función opcional "Key pushed" Event, disponible en el código del navegador, permite crear accesos directos mediante las teclas F, etc.

## Definir el contenido de un control Campo de entrada HTML
<a name="definir_contenido_control_campo_entrada_html_ELTTEXTE000229"></a>
Para definir el contenido de un control Campo de entrada HTML, simplemente especifique la propiedad [Valor](../Proprietes/2510130.md) (en el código de inicialización del control, por ejemplo).

Por ejemplo: 


```wl
// Read an HTML file
HTMLPage is string
HTMLPage = fLoadText(SysDir(srMyDocuments) + ["\"] + "index.html")

// Initialize the edit control
EDT_HTMLControl = HTMLPage
```

## Guardar un texto en un archivo HTML
<a name="guardar_texto_archivo_html_ELTTEXTE000253"></a>
En un control Campo de entrada HTML, se aplica automáticamente el formato HTML al texto escrito.

**Para guardar el contenido de un control Campo de entrada HTML** en un archivo HTML, simplemente utilice la función [fSaveText](../WDLang1/3036046.md) en el control.
<a name="NOTE5_2"></a>


### Ejemplo
<a name="ejemplo_ELTPARAGRAPHE000108"></a>

En el código a continuación, se crea un archivo "index.html". Este archivo contiene el texto introducido en el control EDT_HTMLText. 


```wl
// Select the path and name of the file
FileName is string
FileName = fSelect("", "", "Select a file...", ...
		"HTML files" + TAB + "*.html" + CR + "All files (*.*)" + TAB + "*.*", ...
		"HTML", fselCreate + fselExist)

// Backup
fSaveText(FileName, EDT_HTMLText)
```

<a name="NOTE5_3"></a>




### Guardar texto HTML en un campo de un archivo de datos HFSQL
<a name="guardar_texto_html_campo_archivo_datos_hfsql_ELTPARAGRAPHE000118"></a>

Para guardar texto HTML en un campo de un archivo de datos, se recomienda enlazar el control Campo de entrada a un campo de tipo Memo de texto.

Como el HTML incluye muchas etiquetas, los campos de tipo "Cadena de caracteres" suelen ser demasiado pequeños.

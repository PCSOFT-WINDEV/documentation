
## Manipular controles Menú contextual mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000136"></a>
WEBDEV incluye diferentes funciones y propiedades WLanguage para manipular controles Menú contextual mediante programación.

Puede manipular el menú o sus opciones mediante programación: para manipular una opción de menú desde la página, utilice la sintaxis "&lt;Nombre de la opción&gt;".





## Manipular controles Menú contextual mediante programación
<a name="manipular_controles_menu_contextual_mediante_programacion_ELTTEXTE000160"></a>


### Asociar un menú contextual a un control o página 
<a name="asociar_menu_contextual_control_pagina_ELTPARAGRAPHE000017"></a>

Puede asociar un menú contextual a un control o página: 

- En el editor de páginas (a través de la opción "Menú contextual" en la pestaña "UI" de la ventana de descripción del elemento). 

- Mediante programación: 

	- Usando la propiedad [ContextMenu](../Proprietes/2510077.md). 

	- Usando la función [OpenContextMenu](../WDLang1/3025029.md) (solo código Navegador). 







Observación: la función [ControlContextMenu](../WDLang1/3025012.md) permite obtener el nombre del control que abrió el menú contextual actual. 






### Funciones WLanguage para menús y opciones de menú
<a name="funciones_wlanguage_para_menus_opciones_menu_ELTPARAGRAPHE000041"></a>Las siguientes funciones permiten manipular controles Menú contextual: 



|   |   |
| --- | --- |
| [MenuAddMenu](../WDLang1/1000017266.md) | Agrega un nuevo menú a un menú desplegable o contextual de una ventana o página. |
| [MenuAddOption](../WDLang1/1000017262.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. |
| [MenuAddPopup](../WDLang1/1000020583.md) | Transforma una opción de menú de una Page para que esta opción abra una ventana emergente. |
| [MenuAddSeparator](../WDLang1/1000017270.md) | Añade un nuevo separador en un menú (desplegable o contextual). |
| [MenuAddURLOption](../WDLang1/1000018529.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú abre la página que corresponde a la URL especificada. |
| [MenuDelete](../WDLang1/3038058.md) | Borra un menú o una opción de menú. |








### Propiedades WLanguage
<a name="propiedades_wlanguage_ELTPARAGRAPHE000048"></a>

Las siguientes propiedades permiten manipular las opciones de menú de los menús desplegables o contextuales: 


|   |   |
| --- | --- |
| [Caption](../Proprietes/2510053.md) | Obtiene y establece el texto de una opción de menú. |
| [Checked](../Proprietes/2510019.md) | Permite utilizar una marca de verificación (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuCtxMarque.gif)) junto a una opción de menú. |
| [Name](../Proprietes/2510082.md) | Obtiene el nombre de una opción de menú. |
| [Selected](../Proprietes/2510112.md) | Determina si la opción de menú está seleccionada. |
| [Type](../Proprietes/2510131.md) | Determina si el elemento utilizado es una opción de menú. |
| [Visible](../Proprietes/2510138.md) | Determina si una opción de menú es visible y muestra u oculta la opción. |





Para obtener la lista completa de las propiedades WLanguage que pueden utilizarse con los controles Menú contextual, consulte [Propiedades del control Menú contextual](../WDChamp/1410087223.md).

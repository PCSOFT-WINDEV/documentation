
## AAF: Clasificar/Buscar/Filtrar en las columnas de los controles de tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
En los controles de la tabla (basados en un archivo de datos o poblados mediante programación), las columnas con una búsqueda Icon abren un menú contextual con las características de ordenación, filtrado y búsqueda disponibles para la columna. 

Basta con hacer clic con el botón derecho del ratón en la búsqueda Icon para abrir el menú contextual de la columna: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Table_menuTriColonne.gif)


Este menú contextual propone:

- para realizar una búsqueda (opción "Buscar"): 
	Todo lo que tiene que hacer es escribir las primeras letras correspondientes al elemento buscado. El control Tabla muestra todos los elementos correspondientes a las primeras letras tecleadas. 

- para implementar un filtro ("Filtro"): 
	Para ello, seleccione una condición e indique el valor solicitado. El contenido de la control Tabla se filtra según la condición especificada.
	Observación: La condición "Está vacío" se utiliza para mostrar sólo los elementos vacíos en la columna. 

- para borrar un filtro que antes era set en la columna ("Borrar filtro"). 

- para recordar los filtros ("Recordar los filtros").
	Si esta opción está activada, los filtros implementados al cerrar la ventana se ejecutarán automáticamente durante la siguiente apertura de la ventana.. Se mostrará un breve mensaje para informar al usuario e indicar los filtros que son set. 

- para recordar los tipos ("Remember sorts").
	Si esta opción está activada, los tipos definidos al cerrar la ventana serán automáticamente set la próxima vez que se abra la ventana. 

- a set un filtro con uno de los valores de la columna.




**Observaciones**: 

- Los menús contextuales que aparecen al hacer clic con el botón derecho del ratón en el título de la columna y en la búsqueda Icon son diferentes. 

- Estas opciones también están disponibles para los controles de la tabla TreeView. 

- Para más detalles sobre las opciones de este menú contextual, véase [Menú contextual de las columnas](../WDChamp/9500125.md).


## Manejo de una Control de número mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000130"></a>
WINDEV y WINDEV Mobile le permiten manipular a Control de número mediante la programación. Para hacerlo, usa la Variable de la Control de número en el código.

La Variable de la Control de número:

- corresponde al nombre de la Control de número.

- se inicializa con el valor de giro.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperación del valor de centrifugado
<a name="recuperacion_del_valor_centrifugado_ELTTEXTE000154"></a>
Para recuperar el valor de centrifugado, utilice la siguiente sintaxis:


```txt
<Position> = <Spin Name>
```


&lt;Posición&gt; es un entero Variable.

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificar el valor de centrifugado
<a name="modificar_valor_centrifugado_ELTTEXTE000178"></a>
Para modificar el valor de centrifugado, utilice la siguiente sintaxis:


```txt
<Spin Name> = <Value>
```


&lt;Valor&gt; es un valor que se encuentra entre el valor mínimo y máximo de la caja de giro control. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Propiedades específicas para la gestión de un Control de número
<a name="propiedades_especificas_para_gestion_control_numero_ELTTEXTE000202"></a>
Las siguientes propiedades se utilizan para gestionar las características de un spin por programación.


|   |   |
| --- | --- |
| [MaxValue](../Proprietes/2510009.md) | Se utiliza para averiguar y modificar el límite superior de un giro. |
| [MinValue](../Proprietes/2510008.md) | Se utiliza para averiguar y modificar el límite inferior de un giro. |
| [ScrollValue](../Proprietes/2510011.md) | Permite conocer y modificar el paso de incremento de un giro. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una Control de número, consulte [Propiedades associated con controles Control de número](../WDChamp/1013226.md).

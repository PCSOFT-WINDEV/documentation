


## "Pulsación larga" opcional Event
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
La opción Event "Pulsación larga/Abrir menú contextual" se ejecuta durante una pulsación larga en los controles de una ventana. Esta Event puede utilizarse: 

- para realizar una Process específica antes de mostrar el menú contextual. 
	Recordatorio: el menú contextual asociado a control puede definirse en la pestaña "UI" de la ventana description de control. 






Observación: Para los controles Table y Looper, la "Selección Fila" Event se ejecuta ANTES de "Pulsación larga/Abrir menú contextual".. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000126"></a>
Para utilizar el evento opcional "Pulsación prolongada/Abrir menú contextual": 

1. Visualice el código control (pulse F2 en el control, por ejemplo). 

2. Haga clic en el enlace "Añadir otros Event s a xxx" en la parte inferior del código de la ventana, después del último Event.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)
 En la lista de Event s opcionales, seleccione "Pulsación larga/Abrir menú contextual".

3. Validar. La Event aparece en el editor de código.

## Presentación del control List Box
<a name="presentacion_del_control_list_box_ELTTEXTE000234"></a>
El control List Box permite mostrar una lista de elementos y seleccionar uno o más elementos de la lista.

La lista está expandida, por lo que varios elementos de la lista se muestran en la ventana o página.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los siguientes controles List Box están disponibles **en WINDEV y WINDEV Mobile**:


|   |   |   |
| --- | --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LignePersonnalise.gif)<br>Control List Box texto | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=liste_graphique2.gif)<br>Control List Box control con imágenes y texto. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Liste_graphique.gif)<br>Control List Box gráfico de varias columnas utilizado para seleccionar un color fácilmente |

Los elementos de un control List Box pueden:

- definirse al crear el control en el editor.

- definirse mediante programación.

- provenir de un archivo de datos o consulta.

- provenir de una variable (global al proyecto, ventana o página; miembro de clase, etc.).




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) También es posible dibujar en los controles List Box o mostrar una imagen con las [funciones de gestión de cadenas gráficas](../WDLang1/3029028.md).



## Crear un control List Box
<a name="crear_control_list_box_ELTTEXTE000258"></a>
Para crear un control List Box: 

1. En la pestaña "Creación", en el grupo "Datos": 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) despliegue "Tabla y List Box" y seleccione "List Box".




2. Haga clic en la ubicación en la que desea crear el control. El asistente para crear un control List Box se abre automáticamente.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Vocabulario
<a name="vocabulario_ELTTEXTE000282"></a>
Para manipular fácilmente un control List Box, se deben conocer los siguientes conceptos:

| Nombre | Definición |
| --- | --- |
| Banda de selección | Permite destacar el elemento seleccionado. En la mayoría de los casos se utiliza un color específico. |
| Desplazamiento vertical | Dirección de desplazamiento del control List Box.<br>Una barra de desplazamiento vertical aparece automáticamente si el número de elementos en el control List Box supera el número de elementos visibles. <br>Si el tamaño de un elemento es más grande que el ancho del control List Box, este elemento se truncará. Las barras de desplazamiento horizontal no se muestran automáticamente. |
| List Box basado en un archivo de datos | El control List Box está basado en un archivo de datos o consulta. Los elementos del control List Box corresponden a los registros del archivo de datos. Existen dos tipos de controles List Box basados en archivos de datos:<br><br>- ***Control List Box con fuente de datos en memoria***: el contenido del control List Box se carga en la memoria y después se muestra en la pantalla. Esta opción permite acceder a los elementos del control List Box más rápido. Este modo está reservado para archivos de datos con menos de 100 000 registros.<br><br>- ***Control List Box con acceso directo a la fuente de datos***: El contenido del archivo de datos o consulta se muestra directamente en el control List Box . El registro actual corresponde al elemento seleccionado. Al seleccionar un elemento en el control List Box, se accede al archivo de datos relacionado. Este tipo de control List Box está reservado para archivos de datos con más de 100 000 registros.<br><br><br> |
| List Box rellenado mediante programación | El control List Box no está basado ni en un archivo de datos, ni en una consulta. El desarrollador define los elementos que se muestran en el control List Box (en la ventana de descripción o mediante programación) |
| List Box de selección única | El usuario solo puede seleccionar un elemento en el control List Box. |
| List Box de selección múltiple | El usuario puede seleccionar varios elementos al mismo tiempo en el control List Box.<br>Para seleccionar varios elementos, el usuario puede hacer una de las siguientes acciones:<br><br>- seleccionar los elementos uno por uno manteniendo pulsada la tecla Ctrl.<br><br>- seleccionar varios elementos consecutivos manteniendo pulsada la tecla Shift.<br><br><br> |
| List Box basado en una variable | El control List Box está basado en una variable (global al proyecto, ventana o página; miembro de clase, etc.). |

### Limitaciones en Java
<a name="limitaciones_java_ELTPARAGRAPHE000179"></a>

Las siguientes características no están disponibles en Java:

- tooltip en la barra de desplazamiento.

- List Box multicolumna.

- número de ayuda asociado al control List Box.

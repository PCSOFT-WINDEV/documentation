


## Impresión del contenido de un pivote tabla (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000142"></a>
Pivot controles Tabla dispone de un menú contextual automatic que permite a los usuarios realizar diferentes operaciones sobre los datos visualizados en el control. 

Una de las opciones de este menú contextual permite imprimir el contenido de la control Tabla dinámica ("Imprimir"). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Modo de funcionamiento
<a name="modo_funcionamiento_ELTTEXTE000166"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000020"></a>

Pueden ocurrir varios casos:

- Si un reporte fue creado en este control Tabla dinámica por el desarrollador, este reporte se imprime.

- Si ningún reporte es relacionado a este control Tabla dinámica, el usuario puede imprimir el contenido del control Tabla dinámica.




**Observaciones**: 

- La opción "Imprimir" del menú contextual de Pivot controles Tabla requiere las bibliotecas de impresión de WINDEV framework.

- Mediante programación: 

	- [AAFExecute](../WDLang1/1000022099.md) asociada a la constante *aafPrintPvt* se utiliza para imprimir el control Tabla dinámica especificado.

	- [AAFDisable](../WDLang1/1000022018.md) asociada al *aafPrintPvt* elimina la opción "Imprimir" para el control Tabla dinámica especificado.

	- [AAFChangeCaption](../WDLang1/1000022100.md) asociada al *aafPrintPvt* cambia el texto de la opción "Imprimir" para el control Tabla dinámica especificado.








### Impresión de la reporte creada por el desarrollador
<a name="impresion_reporte_creada_por_desarrollador_ELTPARAGRAPHE000043"></a>

Si un reporte fue creado en el control Tabla dinámica por el desarrollador, la opción "Imprimir" se usará para imprimir este reporte. No se requiere programación específica.


### Impresión directa del contenido del pivote tabla
<a name="impresion_directa_del_contenido_del_pivote_tabla_ELTPARAGRAPHE000048"></a>

Una ventana permite al usuario imprimir en modo vertical u horizontal. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Imprimer_TCD%20-%20HC%20N%B0001.gif)
 El contenido del pivote tabla se muestra directamente en el visor reporte.

Si el número de filas encontradas en la control Tabla dinámica es bastante importante (más de 10 000 filas), la aplicación pide confirmación al usuario (la impresión puede ser bastante larga y consumir mucha memoria).

Las características de impresión son las siguientes:

- El título de la impresión corresponde al título de la control Tabla dinámica. Si no se especificó ningún título para el control Tabla dinámica, se utiliza el título de la ventana.

- Los títulos de las columnas se imprimen en cada página.

- El fondo Color de las celdas será el mismo que el definido en el control Tabla dinámica. 

- La fecha de hoy está impresa en la parte superior derecha Corner.

- El número de la página actual y el número total de páginas están impresos en la parte inferior derecha de la Corner.





### Personalizar la ventana de configuración para imprimir la control Tabla dinámica
<a name="personalizar_ventana_configuracion_para_imprimir_control_tabla_dinamica_ELTPARAGRAPHE000065"></a>

WINDEV le permite incluir en su aplicación la ventana de configuración de impresión de la tabla dinámica utilizada por el AAF. Usted tendrá la posibilidad de personalizar:

- el aspecto de esta ventana (por ejemplo, modificación de plantilla),

- los idiomas soportados por esta ventana. Esta característica es muy useful a la hora de traducir esta ventana a los diferentes idiomas disponibles en la aplicación.  




Para incluir la ventana para configurar la impresión en su aplicación: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccione el subdirectorio que contiene los elementos a importar. La ventana de ajustes de impresión se encuentra en el subdirectorio "Programas\\Datos\\Ventanas de preajuste\\AAF" del directorio WINDEV instalación.

3. WINDEV hace una lista de los elementos del directorio. Este directorio contiene los elementos que corresponden a las AAF personalizables. La ventana a importar se llama "WinDevAaf_ConfigPVTPrint.wdw".

4. Validar. La ventana está incluida en el proyecto y se puede modificar. Esta ventana personalizada se tendrá en cuenta automáticamente al ejecutar la aplicación.




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario.

## Vista general de la control Migas de pan
<a name="vista_general_control_migas_pan_ELTTEXTE000159"></a>


### control en una página estática, dinámica, AWP y PHP
<a name="control_una_pagina_estatica_dinamica_awp_php_ELTPARAGRAPHE000011"></a>

El "Breadcrumb" control corresponde a una serie de enlaces que permiten al usuario de la Web volver a la estructura de árbol lógico de un sitio.

El contenido de este control se construye dinámicamente cuando se navega "abajo" en el sitio.

Típicamente, usted tendrá la habilidad de crear menús tales como: Cámara &gt;&gt; Foto &gt;&gt; Digital &gt;&gt; Reflex.

## Creación de una control Migas de pan
<a name="creacion_una_control_migas_pan_ELTTEXTE000183"></a>


### Creación de una control Migas de pan
<a name="creacion_una_control_migas_pan_ELTPARAGRAPHE000038"></a>

Para crear una control Migas de pan: 

1. En la pestaña "Creación", en el grupo "Navegación", haga clic en "Migas de pan".

2. Haga clic en la ubicación en la que desea crear el control. El control se crea automáticamente. En el editor, la control Migas de pan aparece con el siguiente formato: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Chemin_Navig_Editeur.gif)





Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

La pestaña "Detalles" del control description le permite:

- definir el carácter para separar las diferentes migas de pan. 

- definir si el último elemento de una miga de pan es un enlace.



<a name="NOTE2_2"></a>


### Configuración de una control Migas de pan
<a name="configuracion_una_control_migas_pan_ELTPARAGRAPHE000068"></a>

Para configurar un control Migas de pan:

1. Mostrar el control description ("description" de el menú contextual de la control).

2. En la pestaña "General", haga clic en el botón "Editar mapa de navegación".

3. Aparece la ventana para configurar el mapa de navegación ruta. Para obtener más información, consulte [Construcción y modificación del mapa de navegación](../WDChamp/1013301.md).

4. Validar el mapa de navegación.

5. El "Breadcrumb" control está configurado para ser usado en las diferentes páginas de su sitio.




**Observación**: Los diferentes elementos de un "Breadcrumb" control pueden ser manejados y modificados mediante programación con [las funciones de WLanguage para gestionar los controles de Breadcrumb](../WDLang2/3083021.md).

## Consejo
<a name="consejo_ELTTEXTE000213"></a>


### Uso de plantillas
<a name="uso_plantillas_ELTPARAGRAPHE000095"></a>

Si su sitio utiliza una modelo, le aconsejamos que cree las migas de pan de su sitio según la página modelo. Por lo tanto, todas las páginas que utilizan la modelo se actualizarán automáticamente y utilizarán las migas de pan.
<a name="NOTE3_2"></a>


### Pan rallado y tipos de páginas
<a name="pan_rallado_tipos_paginas_ELTPARAGRAPHE000102"></a>

El pan rallado debe contener:

- páginas WEBDEV dinámicas.

- páginas estáticas y/o páginas de AWP.

- Páginas PHP.




Al programar, usted tiene la habilidad de mezclar los tipos de páginas.

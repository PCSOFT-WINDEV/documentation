


## El Cubo control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
El Cubo control es un control que realiza automáticamente rotaciones alrededor de los diferentes ejes. Este control se puede encontrar en un Corner de una ventana por ejemplo.

**Atención**: Este control requiere el .Net framework 3.0 (o posterior) para operar. Este framework está incluido en Vista (y más tarde) y puede ser descargado para Windows XP (desde el sitio de Microsoft).. El primer uso de un control que requiere este framework (control Carrusel, Cubo control o Xaml control) carga este framework, lo que puede causar delay (fuera de nuestro control).

**Para crear un "Cubo" control**: 

1. En la pestaña "Creación", en el grupo "Otros controles", haga clic en "Cubo".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para visualizar las características de control, seleccione "description" en el menú contextual de control.

El control del Cubo debe ser associated con un Image. Esta imagen será animada automáticamente por el cubo control. Esta imagen puede seleccionarse en la pestaña "General" del control description.

## Manejo programático del cubo control
<a name="manejo_programatico_del_cubo_control_ELTTEXTE000170"></a>
WINDEV le permite manipular un Cubo control programáticamente. Para ello, utilice la Variable del cubo control en el código. Esta variable es de tipo numérico.

El Variable del Cubo control corresponde al nombre del Cubo control.

**cubo control funciones**
Las siguientes funciones pueden utilizarse para los controles del Cubo manipular:



|   |   |
| --- | --- |
| [XamlDisable](../WDLang1/1000015004.md) | Simula la ausencia de .Net 3.0 framework. |




**Cubo control propiedades**
Las siguientes propiedades se pueden utilizar para los controles del Cubo manipular:



|   |   |
| --- | --- |
| [AnimationPeriod](../Proprietes/2510152.md) | El **AnimationPeriod** Property se utiliza para get y modificar la velocidad de rotación de la animación en los controles de Cubo o Carrusel. |




Para obtener una lista completa de las propiedades de WLanguage que se pueden utilizar con los controles del Cubo, consulte [Propiedades del cubo control](../WDChamp/1016101.md).

## Notas
<a name="notas_ELTTEXTE000212"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000070"></a>

Las limitaciones del Cubo control son las siguientes:

- La versión 3.0 de framework .Net (o posterior) debe ser instalada en el equipo. Este framework está incluido en Vista y se puede descargar para Windows XP (desde el sitio de Microsoft).. 
	El primer uso de un control que requiere este framework (Carrusel, Cubo o Xaml) carga este framework, lo que puede causar delay (fuera de nuestro control).

- La aplicación no se puede iniciar desde la red.

- La función OVI (Oscurecimiento de Ventanas Inactivas) está desactivada para todas las ventanas que contienen un cubo control.

- Este control no opera bajo el TSE.

- Esta control utiliza la biblioteca wd28 0xaml.dll. Esta biblioteca no se puede renombrar al crear el ejecutable.



<a name="NOTE3_2"></a>


### Simulando la ausencia de .Net 3.0 framework
<a name="simulando_ausencia_net_30_framework_ELTPARAGRAPHE000087"></a>

Cuando se desarrolla una aplicación que contiene un control de cubo, puede ser useful para saber cómo funcionará la aplicación cuando se inicie en un ordenador que no esté equipado con el .Net 3.0 framework (o posterior).

Simular la ausencia de la framework en un ordenador:

- utilice la función [XamlDisable](../WDLang1/1000015004.md).

- renombrar la siguiente clave de registro:
	"HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\.NETFramework\\v3.0".

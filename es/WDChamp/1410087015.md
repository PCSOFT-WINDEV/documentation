
## Propiedades de control Firma
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000066"></a>
Un control Firma puede ser manejado en un reporte por las siguientes propiedades de WLanguage.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propiedades disponibles en un reporte
<a name="propiedades_disponibles_reporte_ELTTEXTE000090"></a>
Las siguientes propiedades son associated con controles de Firma:



|   |   |
| --- | --- |
| [BlurLevel](../Proprietes/1410088348.md) | La propiedad **BlurLevel** obtiene y establece el porcentaje de desenfoque aplicado a un control en un reporte. |
| [Caption](../Proprietes/2510053.md) | **En una base de datos**, el **Caption** Property se utiliza para:<br><br>- Averigua la leyenda associated con un elemento de archivo. <br><br>- Averigua la leyenda associated con un archivo de datos. <br><br>- Descubrir y modificar la leyenda de conexión. <br><br><br><br><br>**En un reporte**, el **Caption** Property se utiliza para:<br><br>- Averigua el texto que aparece en una estática o control Enlace.<br><br>- Modificar el texto que se muestra en una estática o control Enlace.<br><br><br><br><br>**En una ventana o página**, el **Caption** Property también se utiliza para:<br><br>- Averiguar y modificar la leyenda de un control en una ventana o en un Page.<br><br>- Descubrir y modificar el título de una opción de menú.<br><br>- Averiguar y modificar el título de una ventana o Page.<br><br>- Descubra el contenido de un control Migas de pan.<br><br><br><br><br> |
| [Certificate](../Proprietes/1000021684.md) | La propiedad **Certificate** se utiliza para obtener y cambiar el Certificate utilizado por un control Firma. |
| [Height](../Proprietes/2510050.md) | La **altura** Property se usa para get y cambiar la altura:<br><br>- de un control o bloque en un reporte.<br><br>- de una control (en una ventana), ventana, fila de una Tabla, Cuadro de Lista o control ListView, opción de un Botón de Radio o control Casilla de verificación, romper cabecera o pie de una Tabla o control Looper.<br><br>- de un control (en un Page), romper cabecera o pie de página en una Tabla o control Looper. La control puede ser superpuesta o no.<br><br>- de una celda de ruptura en una Tabla o control Looper.<br><br><br> |
| [Hidden](../Proprietes/1410087964.md) | La propiedad **Hidden** permite: <br><br>- Determinar si un control en un reporte está oculto. <br><br>- Ocultar o desenfocar un control en un reporte.<br><br><br> |
| [Image](../Proprietes/2510034.md) | El **Image** Property obtiene y establece la imagen de fondo de un control o bloque. |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [Note](../Proprietes/2510085.md) | La propiedad **Note** permite:<br><br>- Encuentra las notas associated con un control, una ventana, un Page o un reporte.<br><br>- Modifica las notas associated con un control, una ventana un Page o un reporte.<br><br><br> |
| [NoteTitle](../Proprietes/2510121.md) | La propiedad **NoteTitle** permite:<br><br>- get el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte. <br><br>- Cambiar el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte.<br><br><br> |
| [Plane](../Proprietes/2510094.md) | La propiedad **Plane** permite:<br><br>- Obtener: <br><br>	- el plano activo de una ventana (o reporte) o el plano al que pertenece una control.<br><br>	- el plano Active de un elemento WEBDEV.<br><br><br><br><br>- Cambiar: <br><br>	- el plano activo de una ventana (o reporte) o asocia un control a otro plano.<br><br>	- el plano Active de un elemento WEBDEV.<br><br><br><br><br><br> |
| [SignatureMode](../Proprietes/1000024436.md) | El **SignatureMode** Property se utiliza para get y cambiar las opciones de firma de un control Firma en un reporte. |
| [SignatureType](../Proprietes/1000021995.md) | El **SignatureType** Property se utiliza para get y definir el tipo de firma que se utilizará en un control Firma. |
| [Visible](../Proprietes/2510138.md) | El **Visible** Property se utiliza para:<br><br>- Averiguar si un elemento es Visible.<br><br>- Hacer visible un elemento o Invisible.<br><br><br> |
| [Width](../Proprietes/2510059.md) | El **Ancho** Property obtiene y modifica el ancho:<br><br>- de un control o bloque en un reporte.<br><br>- de un control (que se encuentra en una ventana), de una ventana, de una columna que se encuentra en un tabla o de una columna que se encuentra en un cuadro de lista (incluida la listview).<br><br>- de un control (que se encuentra en un Page), columna que se encuentra en un tabla o columna que se encuentra en un cuadro de lista. La control puede ser superpuesta o no.<br><br><br> |
| [X](../Proprietes/2510135.md) | El **X** Property se utiliza para:<br><br>- get la coordenada X de una columna o ventana control, control Tabla (posición en el eje X).<br><br>- get la coordenada X de un control o ventana (posición en el eje X).<br><br><br> |
| [Y](../Proprietes/2510137.md) | El **Y** Property se utiliza para:<br><br>- Averigua la coordenada Y de un control o ventana (posición en el eje Y).<br><br>- Modificar la coordenada Y de un control o ventana.<br><br><br> |
| [ZOrder](../Proprietes/2510002.md) | La propiedad **ZOrder** define el orden Z (profundidad) de un elemento. |

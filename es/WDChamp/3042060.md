## Presentación
<a name="presentacion_ELTTEXTE000178"></a>
Para incluir gráficos en sus aplicaciones o en sus sitios, WINDEV, WEBDEV y WINDEV Mobile incluyen: 

- a [control Gráfico](../WDChamp/1013006.md) (que puede utilizarse en la ventana, la página y editor de reportes). 

- [Funciones de WLanguage](../WDLang3/3042003.md) que se utilizan para crear y manipular gráficos.




Los gráficos se utilizan para ver eficientemente los datos numéricos.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Uso de gráficos en WEBDEV Linux y en WINDEV Linux**
Para utilizar los gráficos en Linux, es necesario instalar la biblioteca QT (Core y GUI) versión 5 (o posterior). 

**En Ubuntu**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```

## ¿Cómo crear gráficos?
<a name="¿como_crear_graficos_ELTTEXTE000337"></a>


### Creación de gráficos en los editores
<a name="creacion_graficos_los_editores_ELTPARAGRAPHE000084"></a>

El editor de ventanas, el editor de páginas y editor de reportes incluyen un control Gráfico. Al crear esta control, los magos le hacen preguntas para definir el tipo de gráfico a crear. La ventana description del control le permite modificar estas opciones en cualquier momento.

Las funciones de WLanguage también se pueden utilizar para modificar los gráficos que se muestran en los controles de gráficos.. Para ello, especifique el nombre de la control Gráfico en las diferentes funciones.



Para más detalles sobre esta control, véase [La control Gráfico](../WDChamp/1013006.md).
<a name="NOTE3_2"></a>


### Creación de gráficos mediante programación
<a name="creacion_graficos_mediante_programacion_ELTPARAGRAPHE000102"></a>

WINDEV, WEBDEV y WINDEV Mobile incluyen varias funciones para manejar gráficos. Estas funciones permiten crear los gráficos, modificarlos, cambiar las opciones de visualización....

Para obtener más información, consulte: 

- [Tablas de programación](../WDLang3/1000021344.md). 

- [Funciones de gestión de gráficos](../WDLang3/3042003.md).

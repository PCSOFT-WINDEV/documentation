
## La información sobre un control se encuentra en una ventana o en una página
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000099"></a>
El editor de ventanas y el editor de páginas le permiten get información sobre los controles utilizados. Es posible:

- ver la información cuando se está flotando sobre un control.

- mostrar y modificar la información cuando se selecciona un control.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## La información que se muestra cuando se cierne un control
<a name="informacion_que_muestra_cuando_cierne_control_ELTTEXTE000123"></a>
Cuando se cierne sobre un control: 

- la control suspendida está resaltada por una luz Line (las esquinas se muestran en rojo). 

- Aparece un tooltip. Se muestra esta información sobre la herramienta:

	- el nombre de la colgada control ("BTN_Display_the_TreeMap" en el ejemplo a continuación).

	- la posición (en pixeles) del control por el que se desliza el cursor ("&lt;640, 129&gt;" en el ejemplo de abajo).

	- el tamaño (en pixeles) del control por el que se desliza el cursor ("&lt;124x24&gt;" en el ejemplo de abajo).

	- el anclaje del control suspendido si el control está anclado ("Ancho" en el ejemplo de abajo).







Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BulleAideChamp.gif)


**Observación**: Si el control que se mantiene en el aire es Invisible al abrir la ventana, la cadena "(Invisible)" aparece en la punta de la herramienta.

## Información que aparece al seleccionar una control
<a name="informacion_que_aparece_seleccionar_una_control_ELTTEXTE000147"></a>
Cuando se selecciona un control, se muestra la siguiente información y se puede modificar en la barra de estado de WINDEV, WEBDEV o WINDEV Mobile:

- el nombre del control seleccionado ("STC_WARNING" en el ejemplo a continuación).

- la posición (en pixeles) del control seleccionado ("15, 222" en el ejemplo de abajo).

- el tamaño (en pixeles) del control seleccionado ("319x58" en el ejemplo de abajo).

- el avión al que pertenece el control seleccionado ("Plano 0" en el ejemplo siguiente).

- el zoom (100% en el siguiente ejemplo). 




Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=InfoChampBarreMessage2.gif)

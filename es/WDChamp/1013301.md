
## Mapa de navegación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
El mapa del sitio ruta se utiliza para implementar la estructura y la secuencia de páginas de su sitio.. Se utiliza el mapa del sitio ruta:

- por la [Miga de pan](../WDChamp/1013265.md) control.

- por la [Mapa del sitio](../WDChamp/1013300.md) control. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Construcción del mapa del sitio ruta
<a name="construccion_del_mapa_del_sitio_ruta_ELTTEXTE000120"></a>
La implementación de un mapa del sitio ruta se requiere tan pronto como se implemente un control utilizado para navegar en el sitio (mapa del sitio ruta o mapa del sitio).
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Plan%20de%20navigation%20-%20HC%20N%B0001.gif)


Es posible:

- **Construye el mapa del sitio ruta automatic aliado** a partir de las secuencias realizadas en tu sitio (botón "Creación de automatic"). Si ya existe un mapa del sitio ruta, éste será borrado.. WEBDEV utiliza la secuencia de páginas descritas en el editor o por programación para construir el mapa del sitio.

- **Build the navigation map manually**: para ello, seleccione las páginas a mostrar y añádalas al mapa del sitio ruta con el botón "Añadir".. Ahora puede organizar la secuencia de páginas realizando operaciones de "Arrastrar y soltar" entre las diferentes páginas de la zona "Mapa del sitio".

- **Añadir o quitar páginas** del mapa de navegación (botones "Añadir" y "Quitar").

- ** Mostrar y modificar el description de cada página** que se encuentra en el mapa del sitio ruta: para ello, haga doble clic en la página que se encuentra en el área "Mapa del sitio ruta".. Es posible definir:

	- la leyenda utilizada en el mapa del sitio ruta o en el mapa del sitio (pestaña"Caption"). El título de la página es usado por Default.

	- el tooltip associated con la página y se muestra cuando el mapa del sitio ruta o el mapa del sitio es pasado por el cursor del ratón (pestaña "Tooltip"). El título de la página es usado por Default.

	- el valor de los parámetros que se utilizarán para mostrar la página (pestaña"Parámetros"). Esta opción es muy useful para las páginas de AWP por ejemplo.







Esta operación debe realizarse para cada página.

**Observación**: Los enlaces encontrados en el mapa del sitio ruta que utilicen URLs con parámetros se tendrán en cuenta automáticamente en el [Archivo SiteMaps](../Editeurs/2030053.md).

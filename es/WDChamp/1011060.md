
## Visor de reportes: Proponer varios informes
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
En la vista previa de Document, los usuarios finales pueden: 

- elegir o cambiar la orientación de sus informes (vertical u horizontal). 

- adaptar el trabajo de impresión a un formato de papel específico: A4, A3, US Letter, etc.




Esta funcionalidad está disponible en el visor reporte a través de los grupos reporte. Los usuarios pueden elegir la mejor disposición en el visor de reporte mediante un cuadro combinado. 

**Observaciones**: 

- Esta funcionalidad puede ser fácilmente mejorada, por ejemplo, para proporcionar al usuario un set de informes estadísticos por año directamente del visor reporte...

- Para utilizar el mismo reporte en diferentes formatos, puede anclar set en los controles de los informes. Para obtener más información, consulte [Controles de anclaje en un reporte](../WDChamp/1013298.md). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000159"></a>
Permitir a los usuarios finales elegir la orientación y el formato de sus informes: 

1. Se crean varias versiones de la misma reporte en el editor de reportes: 

	- a reporte en modo vertical / a reporte en modo horizontal,

	- una reporte en formato A4 / una reporte en formato A3 /....




2. Un grupo reporte se crea mediante la programación. Este grupo reporte contiene las diferentes versiones de reporte. 

3. Uno de los informes del grupo reporte se muestra en el visor de reporte. El usuario puede seleccionar a través de un combo box la configuración de la reporte para imprimir..  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Groupe_etat%20-%20HC%20N%B0001.gif)

## Implementación
<a name="implementacion_ELTTEXTE000183"></a>
Para proponer diferentes modos de impresión de una reporte: 

1. Crear la reporte inicial y las diferentes versiones de la reporte en un formato diferente (vertical, horizontal, A4, 13, etc.). Cada reporte tiene un nombre diferente. 

2. En el código de impresión de la reporte inicial: 

	- Solicitar la visualización en el visor reporte ([iDestination](../WDLang5/3046074.md)). 

	- Añadir los diferentes informes en un grupo a través de [iGroupAñadir](../WDLang5/1000020580.md). Si el reporte espera parámetros, especifique estos parámetros con [iGroupAñadir](../WDLang5/1000020580.md). 

	- Personalizar (si es necesario) la Image y la leyenda que se encuentra en el cuadro combinado para la selección de reporte mediante [iGroupConfigurar](../WDLang5/1000020581.md). Por Default: 

		- se muestra una imagen que corresponde al formato. 

		- se especifican las características del formato (disponibles en la pestaña "Format" de la ventana description del reporte). 




- Inicie la impresión de reporte en el visor de reporte ([iGroupImprimir](../WDLang5/1000020582.md)). El reporte especificado se muestra en el visor de reporte. El usuario puede seleccionar el formato solicitado a través del cuadro combinado que se encuentra en el visor de reporte. 




Ejemplo de código: 


```wl
// Display in the report viewer
iDestination(iViewer)
// Add the report to the group
iGroupAdd(RPT_Chart_Portrait)
// Configure its caption
iGroupConfigure(RPT_Chart_Portrait, "Print in Portrait mode")
// Add the report to the group
iGroupAdd(RPT_Chart_Landscape)
// Configure its caption
iGroupConfigure(RPT_Chart_Landscape, "Print in Landscape mode")
// Start printing the first report in the report viewer
iGroupPrint()
```

## Uso avanzado de los grupos reporte
<a name="uso_avanzado_los_grupos_reporte_ELTTEXTE000207"></a>
Los grupos de reporte pueden ser utilizados para proporcionar diferentes informes desde el visor de reporte. 

Por ejemplo: los grupos reporte pueden utilizarse para mostrar: 

- informes estadísticos por año: el mismo reporte se muestra varias veces. Sólo se modifica el parámetro que especifica el año. El usuario puede mostrar directamente los informes de los diferentes años desde el visor de reporte. 

- informes correspondientes al mismo tema: el grupo reporte incluirá, por ejemplo, el reporte de la ficha de producto, el reporte de las estadísticas de ventas, el reporte de los pedidos del mes en curso, etc.

## Vista general de la control Red social
<a name="vista_general_control_red_social_ELTTEXTE000196"></a>
La "Red social" control es una barra de herramientas que contiene iconos utilizados para establecer el enlace con las principales redes sociales de la Web:

- Facebook, 

- Tweeter, 

- Linkedln,

- Tumblr, 

- Pinterest.

- Instagram.

- TikTok.

- WhatsApp.




En la mayoría de los casos, esta control se inserta en un sitio o en el lateral de un artículo. 

Esta control permite al usuario de la web realizar dos tipos de acciones: 

- Compartir su experiencia con la comunidad de su red social. 

- Evaluar una página o un sitio mediante las funcionalidades "Me gusta" de Facebook o "+1" de Google+. 




**Observación**: La control Red social no está disponible para el Ajax y los bucles del navegador. La control Red social está disponible sólo para los bucles estándar.

## Creación de una control Red social
<a name="creacion_una_control_red_social_ELTTEXTE000220"></a>
Para crear una control Red social:

1. En la pestaña "Creación", en el grupo "Integración", despliegue "Redes sociales" y seleccione el tipo de red social que desea utilizar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20R%E9seau%20social%20-%20HC%20N%B0001.gif)


2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para mostrar las características de la control: 

1. Seleccione la control correspondiente a la red social que desee. 

2. Seleccione "description" en el menú desplegable control.




Observación: Para crear un "WhatsApp" control Red social: 

1. Haz clic directamente en "Redes sociales".. 

2. Haga clic en la ubicación en la que desea crear el control.

3. Seleccione el tipo de red social en la ventana de control description.

## Características de la control Red social
<a name="caracteristicas_control_red_social_ELTTEXTE000244"></a>
La pestaña "General" del control description permite definir: 

- El **tipo** de enlace a mostrar.. Se pueden mostrar uno o más tipos de botones para cada red social: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20R%E9seau%20social%20-%20HC%20N%B0002.gif&type=thumb)


- El **target** de la acción en la red social. Este objetivo puede ser: 

	- el dominio. Un clic en el control Red social creará un enlace al dominio real del sitio actual.. 

	- el sitio. Un clic en la control Red social creará un enlace con el sitio actual (es decir, la Address actual sin el nombre de la Page). 

	- la página actual. Un clic en la control Red social creará un enlace con la Page actual del sitio WEBDEV. Si el sitio WEBDEV es un sitio dinámico, el enlace se realizará hacia el dominio. 

	- un objetivo personalizado. Un clic en control Red social creará un enlace a un sitio que no sea el sitio actual.. Este sitio puede especificarse en la "URL o cuenta de destino" control. 
			Para obtener más información, consulte [Objetivos disponibles para cada tipo de enlace](../WDChamp/1000028004.md).




- **URL de destino**: URL Address a utilizar cuando el objetivo es personalizado (para un enlace "Tweeter" por ejemplo).

- **cuenta de destino**: Address de la cuenta a utilizar cuando se personaliza el objetivo (para un enlace "Seguir", por ejemplo).
	Para obtener más información, consulte [Objetivos disponibles para cada tipo de enlace](../WDChamp/1000028004.md).




Dependiendo del tipo de enlace, usted tiene la capacidad de: 

- **Utilizar el botón nativo recomendado por la red social**. En este caso, se pueden configurar algunas opciones: 

	- **Mostrar el número de veces que se compartió en un tooltip**: Si esta opción está seleccionada, el número de acciones se mostrará en un tooltip al lado de control Red social. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lien_Social_Bulle.gif)


	- **Solicitar a la red social que se muestre en el idioma de la página**: Si se marca esta opción, las características del botón se mostrarán en el idioma de Page (por ejemplo, la leyenda del botón "Recomendar" se traducirá automáticamente).. 

- **Utilizar un enlace de imagen personalizado**. En este caso, todo lo que tienes que hacer es seleccionar el Image a utilizar. 
	**Atención**: Debes cumplir con las condiciones de uso de las redes sociales, especialmente las restricciones relativas a las imágenes en los enlaces que dirigen a sus servicios.. Utilice las imágenes oficiales suministradas por los sitios.

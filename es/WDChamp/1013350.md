
## Columna de tipo Atributo
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
WINDEV y WEBDEV permiten utilizar columnas de tipo "Atributo" en los controles de tabla. Este tipo de columna se utiliza para: 

- almacenar un valor que no se mostrará en el control Tabla. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) simplificar la utilización y la programación de las columnas contenedoras, permitiendo asociar un atributo a una control de la columna contenedora.

## Columna de atributos: Almacenar un valor que no se mostrará
<a name="columna_atributos_almacenar_valor_que_mostrara_ELTTEXTE000159"></a>


### Creación de una columna de tipo Attribute para almacenar un valor
<a name="creacion_una_columna_tipo_attribute_para_almacenar_valor_ELTPARAGRAPHE000021"></a>

Para crear una columna de tipo Atributo en una control Tabla:

1. Si es necesario, cree una control Tabla (basada en un archivo de datos, rellenada programáticamente o basada en una Variable).

2. Crear una columna de atributos (a través de la ventana description o del menú contextual de control Tabla).

3. Si es necesario, valide la ventana de control description. 

4. La columna de atributos no aparece en la control Tabla del editor. La columna está oculta por Default y sólo puede ser manipulada mediante programación. 




Observaciones: 

- La columna Atributo no debe ser associated con un control. 

- Las columnas de atributos están disponibles para los controles de tabla (basados en un archivo de datos, rellenados mediante programación o basados en un Variable) y los controles de tabla TreeView,





### Manipulación de la columna de atributos mediante programación
<a name="manipulacion_columna_atributos_mediante_programacion_ELTPARAGRAPHE000036"></a>

Las columnas de atributos pueden ser manipuladas mediante programación como otros tipos de columnas. La columna y su valor nunca se mostrarán en el control. 

Las columnas de atributos se tienen en cuenta al llamar a [TableAdd](../WDLang1/3074017.md), [TableAddLine](../WDLang1/3074006.md), [TableInsert](../WDLang1/3074025.md) y [TableInsertLine](../WDLang1/3074026.md). 

Se puede manipular directamente, utilizando la siguiente sintaxis: 

```txt
<Column_name>[Row_number}
```

## Columna de atributos: Asociar un atributo a una control (columna contenedora)
<a name="columna_atributos_asociar_atributo_una_control_columna_contenedora_ELTTEXTE000189"></a>


### Creación de una columna de tipo Atributo associated con un control
<a name="creacion_una_columna_tipo_atributo_associated_con_control_ELTPARAGRAPHE000064"></a>

Para crear una columna de tipo Atributo en una control Tabla y asociarla a una control en una columna Contenedor:

1. Si es necesario, cree una control Tabla (basada en un archivo de datos, rellenada programáticamente o basada en una Variable).

2. Crear una columna de atributos (a través de la ventana description o del menú contextual de control Tabla).

3. En la pestaña "General" de la ventana de la columna description: 

	- Seleccione el associated control. Esta control debe estar en una columna contenedora de la control Tabla. 

	- Seleccione el control Property associated con la columna Atributos: el contenido de la columna corresponderá al valor de esta Property. 




4. Valide la ventana de descripción del control. 

5. La columna de atributos no aparece en la control Tabla del editor. La columna está oculta por Default y sólo puede ser manipulada mediante programación. 




Observaciones: 

- Las columnas de atributos están disponibles para los controles de tabla (basados en un archivo de datos, rellenados mediante programación o basados en un Variable) y los controles de tabla TreeView,





### Manipulación de la columna de atributos associated con un control mediante programación
<a name="manipulacion_columna_atributos_associated_con_control_mediante_programacion_ELTPARAGRAPHE000082"></a>

Las columnas de atributos associated con un control pueden manipularse mediante programación como otros tipos de columnas. La columna y su valor nunca se mostrarán en el control. 

Las columnas de atributos pueden facilitar la programación de las columnas contenedoras. 

Esto permite cambiar una característica de un elemento en una columna de Contenedor cambiando sólo el valor correspondiente en la columna de Atributos de associated.

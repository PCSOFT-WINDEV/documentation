## Presentación
<a name="presentacion_ELTTEXTE000138"></a>
Las propiedades WLanguage se pueden utilizar para manipular: 

- el estandarte deslizante control. Para ello, utilice la siguiente sintaxis: 
	
	```txt
	<Sliding Banner control>.<Property name>
	```
Por ejemplo: 
	
	```wl
	BAN_MyBanner.Value = 3
	```
Consulte [Pancarta corredera control propiedades](#NOTE2_1). 

- los aviones associated con el Banner deslizante control. Para ello, utilice la siguiente sintaxis: 
	
	```txt
	<Sliding Banner control>[Plane number].<Property name>
	```
Por ejemplo: 
	
	```wl
	BAN_MyBanner[3].Visible = True
	```
Consulte [Propiedades associated con los planos de un Banner deslizante control](#NOTE3_1).

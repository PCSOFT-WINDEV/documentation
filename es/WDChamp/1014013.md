


## Posición en el orden Z de un control 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
Todos los controles que se encuentran en una ventana (en un Page o en un reporte) tienen una posición de orden Z diferente. La posición de orden z de una control corresponde al orden de aparición de la control en relación con el plano de la ventana (o Page o reporte).

Por ejemplo:<br>![Ventana mostrada en el editor](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Altitude.gif)<br>![Sección vertical de la ventana](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Altitude2.gif)
 La posición de orden Z se utiliza para gestionar los efectos de superposición de varios controles: si se superponen dos control (dos imágenes por ejemplo), el control que aparece en primer plano será el control con el orden z más alto.

<a name="NOTE2_1"></a>


## Cambiar la posición en el orden Z de un control
<a name="cambiar_posicion_orden_control_ELTTEXTE000168"></a>
La posición de orden Z de una control puede ser modificada:

- [en la ventana, Page o editor de reportes](#NOTE2_2).

- [mediante programación](#NOTE2_4).



<a name="NOTE2_2"></a>


### Modificación del orden Z de una control en la ventana, Page o editor de reportes
<a name="modificacion_del_orden_una_control_ventana_page_editor_reportes_ELTPARAGRAPHE000037"></a>

Para cambiar el orden Z de una control en la ventana, Page o editor de reportes: 

1. Seleccione el control deseado.

2. En el panel "Modificación", en el grupo "Profundidad y alineación", seleccione una acción:

	- para colocar una control en la posición más alta del orden Z: expanda "Traer al frente" y seleccione "Traer al frente".

	- para colocar una control en la posición más baja del orden Z: expanda "Enviar hacia atrás" y seleccione "Enviar hacia atrás".

	- para aumentar la posición de orden Z de una control: expanda "Adelantar" y seleccione "Adelantar".

	- para reducir la posición de orden Z de una control: expanda "Enviar hacia atrás" y seleccione "Enviar hacia atrás".






<a name="NOTE2_4"></a>


### Para modificar la posición de orden Z de una control mediante programación
<a name="para_modificar_posicion_orden_una_control_mediante_programacion_ELTPARAGRAPHE000060"></a>

Utilice [ZOrder](../Proprietes/2510002.md) en los controles de una ventana o reporte.

## Nota
<a name="nota_ELTTEXTE000204"></a>


### Posición única en el orden Z
<a name="posicion_unica_orden_ELTPARAGRAPHE000075"></a>

Todos los controles que pertenecen a la misma ventana (o a la misma reporte) tienen una posición diferente en el orden z.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Casos especiales**: 

- **control dentro de un control Pestaña , super control o barra de herramientas**
	Si la ventana contiene un control Pestaña:

	- El control Pestaña tiene una posición en el orden Z. Este orden Z se establece con respecto a la ventana

	- El orden Z de los controles dentro de la control Pestaña es relativo a la control Pestaña. Todos los mandos associated con el control Pestaña tienen una posición de orden Z diferente.


Los supercontroles y las barras de herramientas funcionan de la misma manera que los controles de pestañas.

- **Planos de una ventana**
	Si la ventana utiliza varios planos, el índice z de los controles es global para todos los planos.
	Para más detalles sobre los aviones, véase [Los aviones](../WDChamp/1013078.md).

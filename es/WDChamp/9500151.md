## Presentación
<a name="presentacion_ELTTEXTE000085"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000108"></a>

Los controles de imagen con secuencia automatic se utilizan para visualizar una serie de Image s situados en uno o varios directorios del sitio. Las imágenes se navegan automáticamente. 

El cambio de imágenes se puede realizar de forma automática o manual (a través de las funciones WLanguage).. 

Se pueden aplicar efectos de transición. 

Esto le permite crear fácilmente banners publicitarios o presentaciones de diapositivas.. 

La secuencia automatic está disponible para los siguientes controles: 

- Control Imagen.

- Control Imagen con enlace.

- Control Miniatura.


Observación: Si las diferentes imágenes utilizadas por la secuencia automatic tienen tamaños diferentes, se aplica el modo de visualización especificado en la pestaña "General" del control automatic ally.

## Usando una imagen con automatic se desplaza en un sitio WEBDEV
<a name="usando_una_imagen_con_automatic_desplaza_sitio_webdev_ELTTEXTE000127"></a>
Para utilizar una imagen con secuencia automatic en un sitio WEBDEV: 

1. Cree una control Imagen.

2. Abra la ventana de descripción del control (opción "Descripción" del menú contextual del control Imagen).

3. En la pestaña "UI", "Secuencia automatic", marque "Deslizar imágenes de un directorio o lista programada" y especifique las diferentes opciones de secuencia: 

	- **Duración entre imágenes**: indica los intervalos de tiempo para mostrar la Image.  

	- **Iniciar automáticamente**. Si no se selecciona esta opción, la secuencia puede iniciarse mediante [&lt;Imagen&gt;.StartScrolling](../WDLang2/1000023749.md). 

	- **Examinar directorio de la imagen (Pestaña General)**: Esta opción se utiliza para navegar por todas las imágenes que se encuentran en el directorio actualmente seleccionado para la control Imagen (o el directorio de la imagen actualmente asociada a la control Imagen). En este caso, la navegación no es recursiva: las imágenes localizadas en subdirectorios no se navegan.  

	- **Ordenar contenido del deslizamiento**: Esta opción se utiliza para mostrar las imágenes navegadas según su nombre. Si es necesario seleccione el directorio de navegación de imágenes (control Imagen.




4. Valide la ventana de descripción del control. 




Si ha set la secuencia para que se inicie automáticamente y ha especificado un directorio Image, la secuencia se iniciará automáticamente al probar el Page. 

**Observación**: **Los efectos de transición se pueden combinar con secuencia** .

## Programando una imagen con el pergamino automatic
<a name="programando_una_imagen_con_pergamino_automatic_ELTTEXTE000192"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000215"></a>

Los controles con una secuencia automatic pueden manejarse mediante programación a través de varias propiedades y funciones de WLanguage. Estas funciones y propiedades también están disponibles en el código del navegador y en PHP.


### Definition de imágenes que se mostrarán en la secuencia
<a name="definition_imagenes_que_mostraran_secuencia_ELTPARAGRAPHE000225"></a>

Las imágenes mostradas en el control durante la secuencia pueden ser:: 

- que se encuentra en el directorio seleccionado en la ventana de control de description. 

- associated con el control mediante programación. Ejemplos: 

	- Una lista de imágenes es associated con el control: 
			
		```wl
		// Assignment in server code
		IMG_Image = "digits_0.png" + CR + "digits_1.png" + CR +  ...
				"digits_2.png" + CR + "digits_3.png"
		```


	- Un directorio que contiene imágenes es associated con el control: 
			
		```wl
		IMG_Image = "images"
		```
Este código se utiliza para asociar una set de imágenes o un directorio con la control. La ruta es relativa al directorio de imágenes del sitio (directorio _WEB).








### Funciones de WLanguage
<a name="funciones_wlanguage_ELTPARAGRAPHE000246"></a>Para manipular las imágenes de una secuencia automatic se utilizan varias funciones de WLanguage. Estas funciones son las siguientes:



|   |   |
| --- | --- |
| [&lt;Image&gt;.first](../WDLang2/1000023753.md) | Muestra el primer Image para el pergamino. |
| [&lt;Image&gt;.last](../WDLang2/1000023748.md) | Muestra el último Image para el pergamino. |
| [ImageNext](../WDLang2/1000023754.md) | Muestra la siguiente Image para el pergamino. |
| [&lt;Imagen&gt;.Occurrence](../WDLang2/1000023750.md) | Devuelve el número total de imágenes que tiene en cuenta el scroll. |
| [ImagePrevious](../WDLang2/1000023752.md) | Muestra el anterior Image para el pergamino. |
| [ImageScrollingPosition](../WDLang2/1000023751.md) | Devuelve la Image visualizada. |
| [&lt;Imagen&gt;.StartScrolling](../WDLang2/1000023749.md) | Inicia el ciclo de imágenes. |
| [ImageStopScrolling](../WDLang2/1000023746.md) | Detiene el desplazamiento de imágenes de automatic. |





Programación de secuencias: 

```wl
// Program images in the sliding banner.
// Ability to specify the image directory
// The first time, the first image is selected at random.
// =================
// Declaration of global variables
gnNumberSlideImages is int
gnStartingImage is int
gsSlideImageName	is string

// Page initialization
<BLOCK Sliding Banner>
	gsSlideImageName	= fListFile(fWebDir() + fSep + "Home banner" + 
			fSep + "*.jpg", frNotRecursive)
	FOR EACH STRING sAFile OF gsSlideImageName SEPARATED BY CR	
		gnNumberSlideImages++
	END
	InitRandom()
	gnStartingImage = Random(1, gnNumberSlideImages)
	
	NumSlideStartingImage is int = gnStartingImage
	
	FOR I = 1 TO NumSlideStartingImage
		IMG_Overview.Next()
	END
	IMG_Overview.StartScrolling()
<END>
```

<a name="propiedades_wlanguage_ELTPARAGRAPHE000335"></a>

|   |   |
| --- | --- |

| [DisplayedValue](../Proprietes/2510129.md) | Devuelve la ruta de la Image que se muestra actualmente. |
| [SortOption](../Proprietes/1000017044.md) | Permite definir el modo de clasificación Image de la secuencia. |
| [Valor](../Proprietes/2510130.md) | Devuelve el directorio de ruta de Image. |

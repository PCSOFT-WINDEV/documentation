## Condiciones de uso
<a name="condiciones_uso_ELTTEXTE000119"></a>
**El control Publicidad utiliza la biblioteca Google Play Service de Android.**
Para usar un anuncio control, debes: 

- instalar Google Play Service en el SDK de Android. 
	Para ello, inicie el Administrador del SDK y compruebe si los "Servicios de Google Play", que se encuentran en los complementos, están instalados correctamente.. 

- [especificar el identificador del grupo de anuncios al generar la aplicación Android](#NOTE3_1). 




**Atención**: La visualización de los anuncios de AdMob en una aplicación debe cumplir con reglas específicas (número de anuncios Visible al mismo tiempo, ubicación de los anuncios, ...). Para obtener más información, consulte [https://support.google.com/admob/bin/answer.py?hl=fr&answer=1307237&topic=1307235&ctx=topic](https://support.google.com/admob/bin/answer.py?hl=fr&answer=1307237&topic=1307235&ctx=topic)

## Identificador de AdMob del grupo de anuncios
<a name="identificador_admob_del_grupo_anuncios_ELTTEXTE000143"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000031"></a>

Para poder mostrar anuncios en el control de anuncios, se debe definir un grupo de anuncios en el sitio "AdMob" para definir el tipo de anuncios a get. Durante este registro, un identificador es associated con el grupo de anuncios. Este identificador debe especificarse en el asistente para generar la aplicación Android. 


### Registro de la aplicación en el sitio de AdMob
<a name="registro_aplicacion_sitio_admob_ELTPARAGRAPHE000036"></a>

Atención: Estas operaciones se dan sólo a título informativo. La interfaz puede haber sido modificada desde que este Document fue escrito. La interfaz utilizada en esta página de documentación corresponde a la interfaz disponible en enero de 2014 (nueva interfaz AdMob de Google).. 

**Para registrar una nueva aplicación Android en AdMob**: 

1. Conectarse al sitio de AdMob: [http://fr.admob.com/](http://fr.admob.com/) (se requiere una cuenta de Google o AdMob). 

2. Crear una cuenta de AdMob si es necesario. 

3. Cuando se crea la cuenta, usted tiene la posibilidad de añadir su sitio o aplicación.

4. Especificar, si es necesario, la información necesaria para el pago. 

5. En la página de inicio, haga clic en "Monetizar" o en el botón "Monetizar una nueva aplicación".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Admob_1.gif)


6. Seleccione o cree la aplicación para monetizar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=admob_2.gif)
Es posible: 

	- Encontrar una aplicación existente. 

	- Añadir una aplicación manualmente. 




7. Introduzca los datos de la solicitud: 

	- el nombre de la aplicación. Este nombre puede diferir del nombre final de su aplicación. 

	- la plataforma de aplicación: Android




8. Se puede crear un anuncio una vez que su aplicación ha sido validada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=admob_3.gif)


9. El identificador del bloque de anuncios se suministra una vez validado el grupo de anuncios.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=admob_4.gif)
 Este identificador debe introducirse en la asistente para crear la aplicación Android. 





### Introduciendo el identificador en la generación asistente
<a name="introduciendo_identificador_generacion_asistente_ELTPARAGRAPHE000068"></a>

En el asistente para generar la aplicación Android, especifique el identificador del bloque de anuncios. 

Observación: Se puede producir un error al cargar AdMob siempre que el grupo de anuncios no esté disponible todavía.. Esta operación puede durar varias horas.

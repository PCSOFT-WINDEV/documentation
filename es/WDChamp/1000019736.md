## Presentación
<a name="presentacion_ELTTEXTE000161"></a>


### Control de ventana o página
<a name="control_ventana_pagina_ELTPARAGRAPHE000011"></a>

La control Valoración permite al usuario: 

- indicar el nivel de popularidad de un elemento mediante pictogramas repetidos. 

- averiguar el nivel de popularidad de un elemento mediante pictogramas repetidos. 




En la mayoría de los casos, este tipo de control presenta el nivel de popularidad como una calificación de 1 a 5 estrellas: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notation_Etoile.gif)

También se puede utilizar para indicar la elección del usuario: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notation_Jaime.gif)

WINDEV y WEBDEV le ofrecen la posibilidad de: 

- utilizar cualquier número de pictogramas.

- usar cualquier forma de pictograma. Por ejemplo, tienes la posibilidad de usar los Image del catálogo Image.

## Creación de una control Valoración
<a name="creacion_una_control_valoracion_ELTTEXTE000185"></a>
Para crear una control Valoración:

1. En la pestaña "Creación", en el grupo "Controles gráficos", haga clic en "Valoración".

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control. El control aparece en el editor.




**Consejo**: También puede crear un "preset" control Valoración haciendo clic en la flecha que se encuentra a la derecha de la opción "Rating".. La lista de controles disponibles se muestra con la plantilla del proyecto. Simplemente haga clic en el control que desea crear.

Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Características de control Valoración
<a name="caracteristicas_control_valoracion_ELTTEXTE000209"></a>
La pestaña "General" de la ventana control Valoración description permite definir las características del control: 

- El **máxima puntuación posible**: se utiliza para definir el número de repeticiones de pictogramas en la control Valoración. Esta opción solo está disponible en modo "Misma imagen para cada puntuación". 

- El **número de marcas en el Image**: indica el número de pictogramas que se encuentran en la Image. Esta opción sólo está disponible si el diseño del Image set utiliza el modo "Image diferente para cada marca".. 

- La marca **inicial**, que significa la marca usada cuando se muestra el control. 

- El **granularidad de las marcas**. Esta opción corresponde al modo de selección de las marcas. La calificación puede evolucionar: 

	- por un valor entero: la marca evolucionará de a uno por vez

	- por la mitad del valor: la marca evolucionará 0,5 a la vez. 

	- por valor real: la marca evolucionará en función del valor tecleado por el usuario. 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **valor de una'estrella**: valor associated con una estrella (o cualquier otro símbolo). El valor de la control Valoración corresponderá al valor de una estrella multiplicado por el número de estrellas visualizadas.. 

- **The Image de marks**: Image utilizado para ver las marcas. Si no se especifica este parámetro, la imagen de una estrella es utilizada por Default. El botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) se utiliza para seleccionar: 

- un Image accesible desde su ordenador (opción "Navegar"). 

- un Image en el catálogo Image (opción "Catálogo"). En este caso, la Image seleccionada se adapta automáticamente a las opciones especificadas.

- **Imagen set distribución**: usado para definir el modo de dividir la imagen de control set: 

	- **Misma imagen para cada puntuación**: La Image debe contener un solo pictograma (representado por varios estados). La Repetition del pictograma se utilizará para representar los diferentes valores posibles en la control. 

	- **Imagen diferente para cada puntuación**: La Image debe contener varios pictogramas (cada uno representado por varios estados). Cada pictograma se utilizará para representar los diferentes valores posibles en el control. En este modo, la marca máxima corresponde al número de pictogramas contenidos en la Image. 




- **Diferenciar gráficamente las marcas por debajo de la marca actual**: 

	- Cuando este modo está desactivado, las marcas debajo de la marca actual se muestran como la marca actual.. 

	- Cuando este modo está activado, las marcas que se encuentran debajo de la marca actual se dibujan utilizando el estado 0 del conjunto Image (las marcas que se encuentran debajo de la marca actual serán Invisible si el Image contiene un solo estado).

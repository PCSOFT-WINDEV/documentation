


## Page específica para mostrar los errores del sitio
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000147"></a>
Si se produce un error en el sitio dinámico de WEBDEV, Default muestra una página de error estándar . Esta Page informa al usuario de la Web del error que se ha producido en el sitio. 

Para personalizar este mensaje de error, sólo tiene que incluir la Page que desea utilizar para mostrar los errores en su sitio web. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000171"></a>


### Creación de una Page para la gestión de errores
<a name="creacion_una_page_para_gestion_errores_ELTPARAGRAPHE000020"></a>

El Page de gestión de errores se crea mediante la creación del Page asistente. 

Para crear una Page de gestión de errores: 

1. Cree una nueva página:

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Página", y luego en "Página". 

	- El asistente de creación de páginas se abre automáticamente.




2. Haga clic en la pestaña "Páginas de error" de asistente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Erreur_SiteWB.gif&type=thumb)


3. Seleccione una de las páginas de error disponibles: 

	- Error genérico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ErreurWB_Generique.gif)
 Esta página de error genérica contiene un título y un Image, y puede personalizarse utilizando el plantilla de su sitio, modificando el mensaje, etc.

	- Error simple: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ErreurWB_Simple.gif)
 Esta simple página de error contiene un título, un Image y el título del error WEBDEV. En tiempo de ejecución, la etiqueta [%TITLE%] se sustituye por el título del error. Esta Page puede personalizarse utilizando la plantilla de su sitio, modificando el mensaje, ... 

	- Error detallado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=erreurWB_detail.gif)
 Este error detallado Page contiene: 

		- una imagen, etc, 

		- el título del error WEBDEV. En tiempo de ejecución, la etiqueta [%TITLE%] se sustituye por el título del error. 

		- el mensaje de error completo. En tiempo de ejecución, la etiqueta [%MSJ%] se sustituye por el título del error. 

		- El mensaje del sistema. En tiempo de ejecución, la etiqueta [%SYS%] se sustituye por el título del error. 
						Esta Page puede personalizarse utilizando la plantilla de su sitio, ... 




4. Valide el asistente. 

5. Especifique el nombre de la Page a crear y validar. La Page estática correspondiente al tipo de Page seleccionado se crea automáticamente. 



<a name="NOTE2_2"></a>


### Implementación de la gestión personalizada de errores
<a name="implementacion_gestion_personalizada_errores_ELTPARAGRAPHE000071"></a>

Para especificar el proyecto Page que se utilizará para gestionar los errores: 

1. Abra la ventana de descripción de la configuración actual: en la pestaña "Proyecto", en el grupo "Configuración del proyecto", haga clic en "Configuración actual". 

2. En la pestaña "General", seleccione el error deseado Page: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Erreur_SiteWB_Def.gif&type=thumb)
 Observación: sólo se proponen las páginas estáticas del proyecto. 

3. Valide.



<a name="NOTE2_3"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000096"></a>

Si se ha definido en el proyecto un Page para la gestión personalizada de errores, este Page se muestra automáticamente si se ha producido un error al desplegar el sitio. 

**Caso especial**: Si previamente se ha implementado una personalización de errores a través del administrador WEBDEV para este sitio (pestaña "Gestión" del administrador WEBDEV), esta personalización no se conserva.. Sin embargo, si se ha especificado un título personalizado en el administrador WEBDEV (para un error específico, por ejemplo), este título se mostrará en el error Page.

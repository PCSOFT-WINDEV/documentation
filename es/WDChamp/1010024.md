
## Los diferentes tipos de ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000161"></a>
Los siguientes tipos de ventanas pueden ser creados en una aplicación WINDEV:

- **MDI**: 
	Estas ventanas deben respetar algunas restricciones específicas de las aplicaciones MDI. Hay tres tipos de ventanas disponibles: Ventana padre MDI, ventana hija MDI y ventana hermana MDI. 
	Las ventanas hijas y las ventanas hermanas se abren de forma no modal.. Para obtener más información, consulte [Gestión no modal de una ventana](../WDChamp/1010022.md).

- **no MDI**: 
	Estas ventanas no tienen ninguna restricción específica. Estas ventanas se pueden abrir en modo modal o en modo no modal. 
	Para obtener más información, consulte [Ventanas modales](../WDChamp/1010028.md)
	Para obtener más información, consulte [Gestión no modal de una ventana](../WDChamp/1010022.md).

## Ventana MDI
<a name="ventana_mdi_ELTTEXTE000185"></a>


### Características de las ventanas principales del MDI
<a name="caracteristicas_las_ventanas_principales_del_mdi_ELTPARAGRAPHE000039"></a>

Una ventana MDI padre debe ser necesariamente la primera ventana de un proyecto WINDEV.. Esta ventana se utiliza para mostrar todas las ventanas de la aplicación.

Una ventana padre MDI tiene las siguientes características (no modificables):

- bordes modificables

- botón de minimizar, botón de maximizar, menú del sistema

- barra de título

- Barra de Icon

- capacidad de incluir un menú desplegable y barras de herramientas.




En una ventana principal MDI, sólo pueden editarse los controles que se encuentran en la zona de la "barra Icon".. Fuera de esta zona, no puede haber control en la edición, no hay clic del ratón es Active.
<a name="NOTE2_2"></a>


### Características de las ventanas MDI hijas
<a name="caracteristicas_las_ventanas_mdi_hijas_ELTPARAGRAPHE000056"></a>

Todas las ventanas MDI hijas tienen las mismas características (que no pueden ser modificadas):

- bordes modificables, 

- un botón de minimizar, botón de maximizar, menú del sistema, 

- barra de título, 

- no menu.




Las ventanas MDI hijas se abren necesariamente en la ventana padre. La posición de visualización de una ventana secundaria MDI siempre es relativa a su ventana principal.

**Observación**: No se puede usar ningún plantilla recortado en las ventanas del MDI.

**Programación**
Las ventanas MDI hijas deben ser abiertas por [MDIOpen](../WDLang1/3052005.md).

**Observación**: la prueba de una ventana MDI hija no se puede ejecutar directamente desde el editor. De hecho, debe abrirse necesariamente desde la ventana principal.

**Mostrar mensajes o barras de progreso con las funciones ProgressBar y Message.**

- Si la ventana MDI hija incluye una barra de estado, el mensaje o la barra de progreso se muestra en la ventana MDI hija.

- Si la ventana MDI hija no incluye ninguna barra de estado, el mensaje o la barra de progreso se muestra en la barra de la ventana padre MDI.

## Ventana no MDI (o ventana libre)
<a name="ventana_mdi_ventana_libre_ELTTEXTE000215"></a>
Para las ventanas no MDI, no hay ninguna restricción sobre el tamaño o el tipo de ventana es relacionado al modo de apertura de una ventana.. Por lo tanto, independientemente de su modo de apertura, las ventanas pueden incluir:

- una barra de título o no, 

- un botón de minimizar o no, 

- un botón de maximizar o no, 

- un botón para el menú del sistema o no, 

- un borde de ventana simple, grueso o doble, 

- un menú o no, 

- todos los tipos de controles.




Las ventanas libres se pueden abrir:

- de [entablar](../WDLang1/3038035.md) (apertura modal).

- por [OpenChild](../WDLang1/3038021.md) o [OpenSister](../WDLang1/3038045.md) (apertura no modal).




**Observación**: las ventanas pueden tener esquinas redondeadas en Windows 11. El sistema aplica este efecto a las ventanas con marcos dobles o gruesos, y sin recortes. Las ventanas con borde simple no pueden tener esquinas redondeadas.

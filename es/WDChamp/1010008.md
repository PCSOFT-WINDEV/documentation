
## Manejo de opciones de menú a través del menú contextual
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000190"></a>
WINDEV y WINDEV Mobile soportan dos tipos de menús desplegables:

- **menús principales de las ventanas**. 
	Un menú principal siempre está asociado a una ventana. Este menú puede aparecer bajo la barra de título de la ventana. Para obtener más información, consulte [Gestión de los menús de la ventana principal](../WDChamp/1010010.md).
	

- **menús contextuales personalizados**.
	El usuario puede abrir un menú contextual presionando el botón derecho del ratón. 
	Para obtener más información, consulte [Menú contextual personalizado](../WDChamp/1010017.md).




Un menú desplegable (menú contextual principal o personalizado) está compuesto por una o varias opciones y subopciones. Cada opción ejecuta un código WLanguage.

Por ejemplo:


|   |   |

En el editor, puede manipular las opciones de menú de la misma manera tanto para los menús principales como para los menús contextuales personalizados. Para manipular las opciones de menú, simplemente edite el menú correspondiente.

**Para editar (o mostrar en el editor)**:

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El menú principal de una ventana:

	- Abra la ventana correspondiente en el editor de ventanas.

	- seleccione una de las opciones de menú. Las opciones y subopciones se muestran automáticamente.




- Un menú contextual personalizado: en la pestaña "Ventana", en el grupo "Barras y menús", amplíe "Menús contextuales" y seleccione uno de los menús contextuales existentes.




Observación: WINDEV y WINDEV Mobile también le permiten [manipular las opciones del menú a través de la programación](../WDChamp/1010007.md).

Esta página de ayuda explica cómo manipular (añadir, eliminar, etc.) las opciones de menú en el editor a través del menú contextual. Para obtener más información, consulte:

- [Características de las opciones de menú (ventana description)](../WDChamp/1010009.md).

- [Gestión de las opciones de menú en las ventanas MDI](../WDChamp/1010015.md).

## Manipular las opciones de menú en el editor
<a name="manipular_las_opciones_menu_editor_ELTTEXTE000285"></a>


### Añadir una opción de menú estándar
<a name="anadir_una_opcion_menu_estandar_ELTPARAGRAPHE000032"></a>

Para añadir una opción de menú estándar: 

- **Después** la opción de menú seleccionada, puede hacerlo:

	- abrir el menú contextual de la opción de menú actual (clic con el botón derecho del ratón) y seleccionar "Añadir después".

	- pulse la tecla Aceptar.




- **Antes** la opción de menú seleccionada, puede hacerlo:

	- abrir el menú contextual de la opción de menú actual (clic con el botón derecho del ratón) y seleccionar "Añadir antes".

	- pulse la tecla Ins.




- Escriba el título de la nueva opción de menú.

- Pulse Intro para validar la adición.




**Observaciones**:

- La tecla Esc se utiliza para Cancel la adición de una opción de menú.

- Para pasar de una opción de menú a la siguiente, utilice las teclas de flecha (arriba y abajo).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Si una ventana contiene más de 2 menús, se utiliza una flecha para mostrar los otros menús en el editor de ventanas.



<a name="NOTE2_3"></a>


### Añadir un separador
<a name="anadir_separador_ELTPARAGRAPHE000058"></a>

Para añadir un separador: 

1. Seleccione la opción de menú delante de la cual se debe añadir el separador.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Insertar un separador".


Observación: También tienes la capacidad de hacerlo:

- añadir un separador antes de la opción de menú seleccionada pulsando la tecla Pausa.

- añadir un separador dando el título "-" o "--" o "---" a la opción de menú. La opción se convierte en un separador cuando se valida la entrada.. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Los separadores no están disponibles.
<a name="NOTE2_2"></a>


### Vinculación de una opción de menú a una acción predefinida
<a name="vinculacion_una_opcion_menu_una_accion_predefinida_ELTPARAGRAPHE000078"></a>

Para vincular una opción de menú a una acción predefinida: 

1. Seleccione la opción de menú deseada.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Acción preestablecida...". Aparece la ventana para seleccionar una acción predefinida.

4. Seleccione la acción solicitada y valídela.


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Las acciones preestablecidas no están disponibles.
<a name="NOTE2_4"></a>


### Añadir un submenú
<a name="anadir_submenu_ELTPARAGRAPHE000094"></a>

Para añadir un submenú: 

1. Seleccione la opción de menú a la que se debe añadir una subopción.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione la opción "Transformar para expandir un submenú".

4. Escriba el título de la primera opción del submenú creado.

5. Pulse Intro para validar la adición.




Observaciones:

- El método para añadir subopciones a un submenú es el mismo que el método para añadir opciones a un menú.

- También tiene la posibilidad de añadir un submenú a la opción de menú seleccionada pulsando la tecla Tab.

### Modificando la Color de fondo de los menús y las opciones del menú
<a name="modificando_color_fondo_los_menus_las_opciones_del_menu_ELTPARAGRAPHE000117"></a>

Para modificar el fondo Color de los menús y las opciones del menú: 

1. Seleccione el menú o una de sus opciones.

2. Abra la ventana description ("Opción description" en el menú contextual).

3. En la pestaña "Estilo", configure:

	- el color de la barra de menú cuando Inactive o pasa por encima.

	- el color de las opciones cuando Inactive o pasa por encima.





"automatic" aplica el Color por defecto utilizado por Windows.

Observación: Para que se tengan en cuenta los colores, desmarque la opción "Mostrar el menú en estilo XP" en la pestaña "Estilo" de la ventana description.
<a name="NOTE2_6"></a>


### Borrar una opción de menú
<a name="borrar_una_opcion_menu_ELTPARAGRAPHE000135"></a>

Para borrar una opción de menú: 

1. Seleccione la opción de menú que desea eliminar.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Borrar".


Observación: También puede eliminar la opción del menú actual pulsando la tecla Supr.

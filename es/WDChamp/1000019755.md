## Presentación del control Mapa
<a name="presentacion_del_control_mapa_ELTTEXTE000404"></a>
El control Mapa permite mostrar un mapa para utilizar la geolocalización, mostrar puntos de interés (POI) y más, ... 

El mapa proviene de: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el servicio de Google Maps para sitios Web mediante la API de Google Maps. 
	El uso de un control Mapa está sujeto a términos y condiciones específicos. Dependiendo del modo de uso, es posible que tenga que utilizar una clave de Google ([MapLicenseGgl](../WDLang3/1000021299.md)). Verifique la licencia de Google antes de implementar sus aplicaciones. 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) el servicio de Bing Maps para aplicaciones en modo Universal Windows 10 App.
	 **Atención**: 

	- El uso de un control Mapa está sujeto a términos y condiciones específicos. Se debe utilizar una clave Bing (función [MapLicenseBing](../WDLang3/1000022006.md)). Verifique la licencia de Bing antes de implementar sus aplicaciones. 

	- Si se especifica una dirección con una cadena de caracteres en un control Mapa, se pueden producir resultados inconsistentes. Se recomienda especificar una dirección con una variable de tipo [Address](../WDLang3/1000019944.md) o [geoPosition](../WDLang3/1000019191.md).   







El mapa que se muestra es interactivo: puede acercarlo, moverlo, etc, ... Se puede ver en modo Mapa o satélite. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ModeCarte.gif)

### Crear un control Mapa
<a name="crear_control_mapa_ELTPARAGRAPHE000096"></a>

Para crear un control Mapa: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", haga clic en "Mapa".

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual del control. 
La pestaña "General" permite definir: 

- el modo de visualización: mapa o satélite,

- si se muestra u oculta la información sobre el tráfico.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) si los botones de zoom deben mostrarse (o no) en el control,

- la imagen utilizada para los marcadores de los puntos de interés (POI). 
	

- la popup (ventana emergente) que se muestra de forma predeterminada cuando se hace clic en un marcador. Esta popup puede corresponder a: 

	- Ninguna: se utilizará la ventana emergente predeterminada. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Ventana predefinida: En este caso, una ventana interna se integra automáticamente en el proyecto. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una ventana interna del proyecto.







La pestaña "Detalles" permite definir: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) si se habilita el zoom con doble clic,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) recordar la última posición mostrada.




**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El control Mapa no se puede utilizar en las pruebas automatizadas. 



<a name="NOTE1_3"></a>


### Ventana emergente para mostrar los marcadores
<a name="ventana_emergente_para_mostrar_los_marcadores_ELTPARAGRAPHE000200"></a>

Dependiendo de la plataforma, cuando se utiliza una ventana o página predefinida para las ventanas emergentes de los marcadores, se incluyen los siguientes elementos: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la ventana interna FI_WinDevPopupMarqueur_WD (ventana interna en el componente WDFAA).




Estos elementos contienen: 

- dos controles que muestran el nombre y la descripción del marcador. 

- un código de declaración para manipular la variable actual de tipo Marker.

## Visualización y uso de mapas
<a name="visualizacion_uso_mapas_ELTTEXTE000440"></a>


### Visualización de mapas
<a name="visualizacion_mapas_ELTPARAGRAPHE000224"></a>

Al abrir una ventana que contiene un control Mapa, se muestra un mapa del mundo automáticamente: el mapa está centrado en Europa, con un nivel de zoom medio.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Puede mostrar la última posición: seleccione la opción "Recordar última posición mostrada" en la pestaña "Detalles" de la ventana de descripción del control Mapa. En este caso, se muestra la última posición con el mismo nivel de zoom.


### Mapas utilizados por el usuario
<a name="mapas_utilizados_por_usuario_ELTPARAGRAPHE000237"></a>

El usuario puede desplazarse por el mapa: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) moviéndolo con uno o más dedos. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) moviéndolo con con el ratón.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) mediante los botones de posicionamiento del control (si se seleccionó la opción "Mostrar botones de zoom" en la pestaña "General" de la ventana de descripción del control). 




El usuario puede acercar o alejar el mapa: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) utilizando los botones de zoom del control (si se seleccionó la opción en la pestaña "General" de la ventana de descripción del control). 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) realizando un gesto de "pellizco": separe los dedos para acercar y júntelos para alejar. 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) pulsando dos veces en el mapa para acercar la imagen en una posición (si se seleccionó esta opción en la pestaña "General" de la ventana de descripción del control). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) haciendo doble clic en el mapa para acercar la imagen en una posición (si se seleccionó esta opción en la pestaña "General" de la ventana de descripción del control). 

- mediante programación con la propiedad [Zoom](../Proprietes/1000017201.md). 






![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para obtener más información sobre las opciones disponibles en tiempo de ejecución, consulte [Control Mapa: Funcionalidades Automáticas de la Aplicación (AAF)](../WDChamp/1000019824.md).

### Internet Explorer: Versiones necesarias
<a name="internet_explorer_versiones_necesarias_ELTPARAGRAPHE000304"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una aplicación WINDEV, el control Mapa utiliza ActiveX de Internet Explorer en el equipo cliente. Para utilizar el control Mapa en la aplicación, el equipo cliente debe tener Internet Explorer 10 u 11.

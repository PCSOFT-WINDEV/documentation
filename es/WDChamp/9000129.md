


## Imprimir un texto de Watermark
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000155"></a>
Un reporte (y un duplicado) puede incluir un texto adicional en forma de marca de agua, como por ejemplo, "Confidencial" o "Duplicado" impreso en todo el reporte. 

El Watermark puede ser creado: 

- del espectador de reporte (sólo WINDEV).

- mediante programación. 




**Observaciones**: 

- Si se define un Watermark para un reporte, se define para todas las páginas del reporte.  

- Se puede definir un solo Watermark para cada reporte. No se pueden definir varias marcas de agua. 

- La Watermark se imprime en las páginas que contienen información únicamente. No se imprime en páginas vacías.

## Creando un Watermark desde el visor de reporte
<a name="creando_watermark_desde_visor_reporte_ELTTEXTE000179"></a>
<a name="NOTE2_1"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la pestaña "Anotar" del visor de reporte, la opción "Marca de agua" permite al usuario crear un Watermark en el reporte que se está visualizando. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_Apercu_Annoter.gif&type=thumb)
 Cuando se selecciona esta opción, una ventana específica permite al usuario definir:

- el destino de la Watermark: imprimir y/o duplicar copy. 

- el posicionamiento de la Watermark: diagonal ascendente, diagonal descendente u horizontal. 

- el texto de la Watermark, su fuente, su tamaño, su Color y su opacidad. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen_Filigrane.gif)

## Creación de una Watermark a través de la programación
<a name="creacion_una_watermark_traves_programacion_ELTTEXTE000203"></a>
<a name="NOTE3_1"></a>
Se pueden utilizar diferentes métodos para crear una Watermark a través de la programación: 

- [Método 1: Configuración completa: usando un Watermark Variable](#NOTE3_2). 

- [Método 2: Configuración rápida: empleo directo](#NOTE3_3).



<a name="NOTE3_2"></a>


### Método 1: Configuración completa: usando un Watermark Variable
<a name="metodo_1_configuracion_completa_usando_watermark_variable_ELTPARAGRAPHE000063"></a>

Este método es el siguiente: 

1. Definir [Watermark](../WDLang5/1000019922.md) Variable. Esta variable contiene todas las características de la Watermark a imprimir. 

2. Utilice [iParameterWatermark](../WDLang5/1000019939.md) para añadir la marca de agua en su reporte. 

3. Empiece a imprimir la reporte. 




Por ejemplo: 

- Código WLanguage utilizado para imprimir un Watermark en el Background de un reporte creado con el editor de reportes:
	
	```wl
	// Print a watermark in report background
	MyWatermark is Watermark
	
	// Text of the watermark
	MyWatermark.Text = "Confidential"
	// Font used
	MyWatermark.Font.Name = "Arial"
	// The watermark will be horizontally centered at bottom
	MyWatermark.Position = iCenterH + iBottom
	
	// Print in the report viewer
	iDestination(iViewer)
	
	// Indicates that the watermark will be printed during the print 
	// and in the duplicate copies
	iParameterWatermark(iWatermarkPrinting + ...
		iWatermarkDuplicate, MyWatermark)
	
	// Starts printing the report
	iPrintReport(RPT_Commercial)
	```


- Código de idioma usado para imprimir un Watermark en el Background de un reporte creado con las funciones de impresión:
	
	```wl
	MyWatermark is Watermark
	
	// Text of the watermark
	MyWatermark.Text = "Confidential"
	
	// Print in the report viewer
	iDestination(iViewer)
	
	// Implement a watermark in document background only
	iParameterWatermark(iwatermarkPrinting, MyWatermark)
	
	// Prints several pages with watermark
	FOR i = 6 TO 10
		iPrint(" Page " + i)
		iSkipPage()
	END
	// The last page is empty and it has no watermark
	
	// Ends the print job
	iEndPrinting()
	```

### Método 2: Configuración rápida: empleo directo
<a name="metodo_2_configuracion_rapida_empleo_directo_ELTPARAGRAPHE000089"></a>

Este método es el siguiente: 

1. Utilice [iParameterWatermark](../WDLang5/1000019939.md) para especificar el texto del Watermark que se añadirá a su reporte. En este caso, se utiliza la fuente "Arial", con un tamaño automatic y con una opacidad set a 20%. 

2. Empiece a imprimir la reporte. 




Por ejemplo: 

- Código WLanguage utilizado para imprimir un Watermark en el Background de un reporte creado con el editor de reportes: 
	
	```wl
	// Print in the report viewer
	iDestination(iViewer)
	
	// Indicates that the watermark will be printed during the print 
	// and in the duplicate copies
	iParameterWatermark("Confidential")
	
	// Starts printing the report
	iPrintReport(RPT_Commercial)
	```


- Código de idioma usado para imprimir un Watermark en el Background de un reporte creado con las funciones de impresión:
	
	```wl
	// Print in the report viewer
	iDestination(iViewer)
	
	// Implement a watermark in document background only
	iParameterWatermark(iWatermarkPrinting, "Confidential")
	
	// Prints several pages with watermark
	FOR i = 6 TO 10
		iPrint(" Page " + i)
		iSkipPage()
	END
	// The last page is empty and it has no watermark
	
	// Ends the print job
	iEndPrinting()
	```

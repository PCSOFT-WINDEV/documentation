## Resumen de Control deslizante
<a name="resumen_control_deslizante_ELTTEXTE000180"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000011"></a>

Un deslizador control se utiliza para seleccionar un valor entre dos límites. Su modo de funcionamiento es similar al modo de funcionamiento de una barra de desplazamiento.. El valor actual se identifica por un cursor que el usuario mueve con el ratón (o con la rueda del ratón).

Se pueden mostrar marcas visuales. El cursor no se ajusta a estas marcas: el cursor del deslizador se mueve unidad por unidad.
 Puedes crear:

- deslizadores estándar<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Potentiometre.gif)

## Creación de una Control deslizante
<a name="creacion_una_control_deslizante_ELTTEXTE000204"></a>


### Creación de una Control deslizante
<a name="creacion_una_control_deslizante_ELTPARAGRAPHE000050"></a>

Para crear una Control deslizante: 

1. En la pestaña "Creación", en el grupo "Controles gráficos": 

	- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) ampliar "Control deslizante" y seleccionar el tipo de deslizador a crear:
			<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_pot3.gif)


	- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) haga clic en "Control deslizante".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

### Modificando el valor del deslizador en tiempo de ejecución con la rueda del ratón
<a name="modificando_valor_del_deslizador_tiempo_ejecucion_con_rueda_del_raton_ELTPARAGRAPHE000092"></a>

El usuario puede modificar el valor del control deslizante con la rueda del ratón: el cursor se moverá según el paso (las marcas) definido para el control deslizante. 
<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### canjear
<a name="canjear_ELTPARAGRAPHE000101"></a>

WINDEV puede convertir un Control deslizante estándar en un Control deslizante circular y viceversa: 

1. Seleccione el control que desea transformar. 

2. En la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Control deslizante/Control deslizante circular".

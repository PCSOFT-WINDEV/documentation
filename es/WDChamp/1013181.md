
## Vinculación de una control List Box a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000094"></a>
Una control List Box puede estar vinculada a un elemento de un archivo de datos, consulta, etc.

Los siguientes tipos de artículos pueden estar vinculados a una control List Box:

- Texto, 

- Numérico,

- Fecha u hora,

- Botón de opción, List Box, Combo Box.




**El elemento ligado al control List Box contendrá el índice del elemento seleccionado en el control List Box.**

**Observación**: Cuando un elemento está vinculado a un control List Box, el contenido del elemento del cuadro de lista es el mismo para todos los registros de el archivo de datos. 

El contenido de la control List Box puede ser: 

- introducido directamente en el editor de análisis,

- introducido en el editor de ventanas o páginas,

- asignado en el código (con [ListAdd](../WDLang1/3049004.md)). 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipulación del artículo y de la control List Box
<a name="manipulacion_del_articulo_control_list_box_ELTTEXTE000118"></a>


### Para actualizar la control List Box con el elemento que corresponde al valor almacenado en el artículo:
<a name="para_actualizar_control_list_box_con_elemento_que_corresponde_valor_almacenado_articulo_ELTPARAGRAPHE000038"></a>

**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los controles enlazados a un campo en el archivo de datos se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [FileToScreen](../WDLang4/3044210.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE3_2"></a>


### Para actualizar un elemento con el índice del elemento seleccionado en la control List Box:
<a name="para_actualizar_elemento_con_indice_del_elemento_seleccionado_control_list_box_ELTPARAGRAPHE000045"></a>

**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los campos de el archivo de datos enlazados a un control de la ventana o página se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [ScreenToFile](../WDLang4/3044146.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
El valor de cada control debe asignarse a cada campo utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

## Creación de una control en una reporte
<a name="creacion_una_control_una_reporte_ELTTEXTE000453"></a>


### Creando una simple control
<a name="creando_una_simple_control_ELTPARAGRAPHE000048"></a>

Para crear un control en un reporte:

1. En la pestaña "Creación", haga clic en el control que desea crear:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_Ruban_Creation.gif&type=thumb)


2. Haga clic en la ubicación en la que desea crear el control.



<a name="NOTE2_2"></a>


### Creando un control a partir de un diccionario control
<a name="creando_control_partir_diccionario_control_ELTPARAGRAPHE000063"></a>

Un reporte control puede crearse a partir de un control de un control encontrado en un diccionario. Un diccionario se utiliza para almacenar varios elementos (controles, informes,...) para un uso futuro en nuevos proyectos.. Para obtener más información, consulte [Diccionario](../Editeurs/2023002.md).

Para crear un control a partir de un control encontrado en un diccionario:

1. Visualiza la pantalla de ["Panel "Diccionario](../Editeurs/2027014.md).

2. Abra el diccionario si es necesario ("Abrir" en el menú contextual).

3. Seleccione la inicial control que se encuentra en el diccionario (en la categoría "Reporte control s").

4. Arrastrar y soltar el control del diccionario a la futura posición del control en el reporte.

5. Suscriba el control creado al diccionario si es necesario. Si el control creado se suscribe al diccionario, las modificaciones hechas al control que se encuentran en el diccionario se aplicarán automáticamente al control creado. Para obtener más información, consulte [Suscribir un elemento al diccionario](../Editeurs/2023001.md).



<a name="NOTE2_3"></a>


### Creación de una control a partir del contenido del portapapeles
<a name="creacion_una_control_partir_del_contenido_del_portapapeles_ELTPARAGRAPHE000087"></a>

La editor de reportes le permite crear algunos tipos de controles según el contenido del portapapeles:

| Contenido del portapapeles | Tipo de control creado |
| --- | --- |
| Texto sin formatear, por ejemplo, recuperado en un editor de texto. | Control Estático |
| Texto formateado, por ejemplo, recuperado en un editor de texto. | Control RTF |
| Image desde el Explorador de Windows. | control Imagen |
| Imagen, recuperada de INTERNET por ejemplo. | control Imagen |


Para crear una control a partir del contenido del portapapeles:

1. copy el elemento al portapapeles (Ctrl + C desde el editor de elementos).

2. En editor de reportes, en la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar" (o Ctrl + V).



<a name="NOTE2_4"></a>


### Creando un control Imagen a partir de un archivo Image por medio de arrastrar y soltar
<a name="creando_control_imagen_partir_archivo_image_por_medio_arrastrar_soltar_ELTPARAGRAPHE000133"></a>

El editor de reportes permite crear un control Imagen arrastrando un archivo de imagen desde el explorador de Windows y soltándolo en un WINDEV reporte.

Para crear una control Imagen a partir de un archivo Image:

1. Seleccione el archivo Image deseado en el Explorador de Windows.

2. Mantenga pulsado el botón izquierdo del ratón y mueva el archivo Image a un reporte abierto en el editor.

3. Suelte el botón del ratón. Se crea automáticamente un control Imagen associated con el archivo Image.




**Observación**: 
 
Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).

## Vinculación de un control a un elemento un archivo de datos o Variable
<a name="vinculacion_control_elemento_archivo_datos_variable_ELTTEXTE000526"></a>
Se puede vincular: 

- a reporte control (Estática, Calculada, Image, Código de barras, Casilla de verificación, Enlace o Gráfico) al elemento un archivo de datos encontrado (o no) en la fuente de datos reporte.

- un control reporte (Estático, Calculado, Imagen, Código de barras, Casilla de verificación o Enlace) a una variable en la fuente de datos reporte.




Para vincular un control existente a un elemento de archivo de datos o Variable:

1. Seleccione el control deseado.

2. Haga clic con el botón derecho del ratón y seleccione "description" en el menú contextual para abrir la ventana de control description, luego vaya a la pestaña "Vinculación" (o pestaña "Series" si se trata de un gráfico).

3. Seleccione el elemento encuadernado o Variable.


**Observación**: También puede crear una control Estático directamente desde el elemento un archivo de datos:

1. En la ["Panel "Análisis](../Editeurs/2027012.md), seleccione el elemento que estará vinculado a la control que se va a crear.

2. Arrastre el elemento y suéltelo en la posición en la que desea crear la control vinculada.

## Copiar los controles reporte
<a name="copiar_los_controles_reporte_ELTTEXTE000580"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000199"></a>Se pueden copiar uno o varios controles en la misma reporte o de una reporte a otra.
El description y el código de control copiado serán idénticos a los del control inicial.
<a name="NOTE5_2"></a>


### Copiar un control en el mismo reporte
<a name="copiar_control_mismo_reporte_ELTPARAGRAPHE000207"></a>

Para copiar una control en la misma reporte: 

1. Seleccione el mando a copy.

2. En la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar". 
	Observación: También puede seleccionar "copy" en el menú contextual del control.

3. En la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar".
	Observación: También puede seleccionar "Pegar" en el menú contextual de control.

4. Posiciona la control con el ratón (el contorno de la control aparece en líneas punteadas y te ayuda a posicionar la nueva control).



<a name="NOTE5_3"></a>


### Copiar un control de un reporte en otro reporte
<a name="copiar_control_reporte_otro_reporte_ELTPARAGRAPHE000247"></a>

Para copiar un control de un reporte en otro reporte: 

- ** Método 1**:

	- En el primer reporte, seleccione el control al copy.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar".
			Observación: También puede seleccionar "copy" en el menú contextual del control.

	- Visualice el destino reporte en el editor de reportes.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar".
			Observación: También puede seleccionar "Pegar" en el menú contextual de control.

	- Posiciona la control con el ratón (el contorno de la control aparece en líneas punteadas y te ayuda a posicionar la nueva control).




- ** Método 2**: (ambos informes deben visualizarse en el editor)

	- En el primer reporte, seleccione el control al copy.

	- Mantenga el botón izquierdo del ratón pulsado y arrastre la control hacia el destino reporte.

	- Suelte el botón del ratón una vez que el control esté posicionado en el reporte de destino (el contorno control aparece en líneas de puntos y le ayuda a posicionar el control pegado).

### Mover uno o más controles reporte
<a name="mover_uno_mas_controles_reporte_ELTPARAGRAPHE000307"></a>

Para mover uno o más controles reporte: 

1. Seleccione los controles que desea mover.

2. Mantenga pulsado el botón izquierdo del ratón y mueva los controles con el ratón.. El verificador de interfaz en tiempo real le permite alinear los controles (véase el párrafo siguiente).




Observaciones:

- Un control pertenece necesariamente a un bloque.

- Los controles seleccionados también se pueden mover con las teclas de flecha que se encuentran en el teclado. Para acelerar el movimiento, presione la tecla Ctrl junto con la tecla de flecha. Para ralentizar el movimiento, presione la tecla Mayúsculas junto con la tecla de flecha.



<a name="NOTE5_6"></a>


### Controles de bloqueo
<a name="controles_bloqueo_ELTPARAGRAPHE000322"></a>

En algunos casos, es posible que desee bloquear los controles en una posición fija.: en este caso, el control no puede ser modificado en su posición y el control no es seleccionado durante una selección realizada con el lazo. Esta característica es a menudo useful para los controles de diseño.

Para bloquear un control:

1. Seleccione la control.

2. Usar:

	- "Bloquear" en el menú contextual de control.

	- las teclas Alt + V.

## Usando un reporte control fuera de la zona...
<a name="usando_reporte_control_fuera_zona_ELTTEXTE000640"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000362"></a>

En la mayoría de los casos, los controles pertenecen a un reporte. Por lo tanto, estos controles aparecen en la reporte en modo de edición y en tiempo de ejecución.

Un control también puede ser colocado en la ventana de la casa de un reporte: la control se dice "fuera reporte". Este control será Visible sólo en el modo de edición. Este control no se imprimirá. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtatFenAccueil.gif)
 Un "off-reporte" control sirve para almacenar un reporte control sin imprimirlo. Esto le permite utilizar el contenido del control mediante programación (como una variable).
<a name="NOTE6_2"></a>


### Creación de un control "fuera de reporte
<a name="creacion_control_fuera_reporte_ELTPARAGRAPHE000373"></a>

Para crear un control"fuera de reporte: 

1. En la pestaña "Creación", haz clic en el control que quieras crear.

2. Haga clic en la parte izquierda de la ventana de inicio en la que desea crear el control.

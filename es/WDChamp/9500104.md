
## El control Campo de entrada RTF: Barra de formato automática
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000127"></a>
En los controles Campo de entrada RTF, se puede mostrar automáticamente una barra de formato cuando el control entra en modo de edición. De esta manera, las opciones de formato se gestionan automáticamente.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000151"></a>


### Activar la barra de formato para los controles RTF
<a name="activar_barra_formato_para_los_controles_rtf_ELTPARAGRAPHE000018"></a>

Para activar la barra de formato para los controles RTF:

1. Abra la descripción de un control Campo de entrada RTF.

2. En la pestaña "General", indique si el control utiliza una barra de herramientas RTF: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RTF_ConfigBarre.gif)


3. Valide.




**Atención**: Solo se puede mostrar una barra de herramientas RTF en una ventana. Si hay más de un control en modo "Siempre visible", el control que tiene el foco tiene prioridad y mostrará la barra de herramientas RTF.
<a name="NOTE2_2"></a>


### Funcionalidad disponible en las AAF
<a name="funcionalidad_disponible_las_aaf_ELTPARAGRAPHE000032"></a>

En un control Campo de entrada RTF, el usuario final puede activar la barra de formato mediante la opción "Barra de formato RTF" en el menú contextual del control.
<a name="NOTE2_3"></a>


### Fuente predeterminada en la barra de herramientas RTF
<a name="fuente_predeterminada_barra_herramientas_rtf_ELTPARAGRAPHE000039"></a>

En tiempo de ejecución, la barra de herramientas RTF se inicializa con la fuente, el tamaño y los atributos de fuente especificados en la descripción del control. Esta fuente puede modificarse mediante programación con [las propiedades de gestión de fuentes](../Proprietes/2515045.md). 
<a name="NOTE2_4"></a>


### Desactivar la barra de formato RTF
<a name="desactivar_barra_formato_rtf_ELTPARAGRAPHE000049"></a>

Para no mostrar la opción "Barra de formato RTF" en el menú contextual del control Campo de entrada, utilice la función [AAFDisable](../WDLang1/1000022018.md)  con la constante *aafRTFToolbar*.

**Atención**: Este comando también desactivará la [barra de formato de los controles HTML](../WDChamp/9500113.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Uso de la barra de formato RTF
<a name="uso_barra_formato_rtf_ELTTEXTE000193"></a>
Esta es la barra de formato RTF: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RTF_Barre.gif)
Se pueden utilizar los siguientes atajos de teclado para formatear el texto:

- Cursiva: Ctrl + I

- Negrita: Ctrl + B

- Subrayado: Ctrl + U

- Tachado: Ctrl + 6

- Resaltado: Ctrl + H 




Si el usuario selecciona texto con formato (por ejemplo, texto subrayado), el botón correspondiente se muestra presionado en la barra de herramientas.

El botón "Color" no refleja el color de la selección: corresponde al último valor seleccionado por el usuario.

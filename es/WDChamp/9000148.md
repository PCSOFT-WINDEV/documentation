


## Ventanas no maximizadas en iOS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
Una ventana no maximizada sólo puede ocupar una sección de la pantalla. La mayoría de las veces, las ventanas no maximizadas se utilizan para mostrar mensajes de información.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación de una ventana no maximizada
<a name="creacion_una_ventana_maximizada_ELTTEXTE000159"></a>
Para crear una ventana no maximizada: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Ventana", y luego en "Ventana".

	- El asistente de creación de ventanas se abre. 

	- Selecciona "En blanco para iPad" o "En blanco para iPhone" en la asistente y valídalo.




2. La ventana se crea en el editor. 

3. Visualizar la ventana description ("description" en el menú contextual). 

4. En la pestaña "Estilo", seleccione "Ventana no maximizada".. 
	Observaciones: 

	- La opción "Ventana con flecha" permite crear una ventana [ventana con flecha](../Editeurs/9000146.md).

	- Este tipo de ventana sólo está disponible para iPad.




5. Valide.

## Características de las ventanas no maximizadas
<a name="caracteristicas_las_ventanas_maximizadas_ELTTEXTE000183"></a>


### Tamaño de la ventana
<a name="tamano_ventana_ELTPARAGRAPHE000047"></a>

El tamaño de la ventana cuando se abre puede modificarse en la pestaña "Detalles" de la ventana description. 

Por Default, la ventana está centrada en relación con la pantalla. 

Observación: Las alturas y anchuras mínimas y máximas de la ventana pueden definirse en la pestaña "Estilo" de la ventana description (botón "Tamaño mín./máx."). 
<a name="NOTE2_2"></a>


### Abrir una ventana no maximizada
<a name="abrir_una_ventana_maximizada_ELTPARAGRAPHE000058"></a>

La apertura de una ventana no maximizada es realizada por [OpenChild](../WDLang1/3038021.md) o por [OpenMobileWindow](../WDLang1/1000021018.md).
 
Por ejemplo:


```wl
OpenMobileWindow(WIN_Non_Max)
```

<a name="NOTE2_3"></a>


### Ventana recortada
<a name="ventana_recortada_ELTPARAGRAPHE000218"></a>

Se puede recortar una ventana no maximizada (ventana no rectangular). Para ello, utilice un Image en formato PNG. 

Para recortar una ventana no maximizada:

- En la pestaña "Estilo" de la ventana description, seleccione el fondo "Transparent" color.

- En la pestaña "Imagen" de la ventana description, seleccione: 

	- un fondo Image en formato PNG, 

	- el modo de transparencia "Según Image ".
			Observación: La transparencia de Image se apoya en el formato PNG.






<a name="NOTE2_4"></a>


### Observación
<a name="observacion_ELTPARAGRAPHE000083"></a>

La barra de título de una ventana no maximizada se muestra sólo para las ventanas [Ventana con flecha](../Editeurs/9000146.md).

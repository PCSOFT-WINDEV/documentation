


## Las cartas "Line
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000181"></a>
En los gráficos Line, los datos se interpretan como coordenadas Y sucesivas de puntos. Entonces, los puntos que se encuentran en cada serie son relacionado por una Line.

Los gráficos de Line se utilizan principalmente para representar la tendencia de los datos.

<a name="NOTE2_1"></a>


## Características de las cartas de Line
<a name="caracteristicas_las_cartas_line_ELTTEXTE000205"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Genligne.gif&type=thumb)
En este cuadro:

**1. Título del gráfico**. 
Las características del título del gráfico pueden definirse en la pestaña "Detalles" del control description ("Editar título del gráfico").
Puede definir el título del gráfico, su posición y su estilo.

**2. Título de los ejes**. 
El título de los ejes horizontales y verticales se muestra si el título se definió en la pestaña "General" de la control Gráfico description. Este parámetro también puede definirse mediante programación con [grAxisTitle](../WDLang3/3042057.md).

**3. Marca en el eje vertical**. 
La marca es calculada automáticamente por Default. La pestaña "General" del control description se utiliza para definir:

- una graduación específica.

- una escala de coordenadas X en función del tiempo. Esta opción opera con datos de Fecha o FechaHora provenientes de una control Tabla o de un ítem del HFSQL. Esta opción no puede utilizarse con una serie rellenada mediante programación. 

- una escala logarítmica de coordenadas Y.




También tiene la posibilidad de utilizar la escala secundaria para una o más series (pestaña"Series").

**4. mito**. 
Las características de la leyenda pueden definirse en la pestaña "Detalles" del control description ("Editar leyenda del gráfico").
Puedes definir su posición y su estilo.

**5. Etiquetas de serie**. 
El título de la leyenda corresponde al título de las diferentes series mostradas en el gráfico (definido en la pestaña "Series").

**6. Color de serie**. 
Se puede definir un color para cada serie en la pestaña "Serie" del control description. Un automatic Color es usado por Default.

**7. Etiqueta de categoría**. 
Las etiquetas de las categorías se pueden definir en la pestaña "Serie" del control description.

**8. Líneas de cuadrícula**. 
Las cuadrículas horizontales y verticales pueden definirse en la pestaña "General" del control description.

### Otras características
<a name="otras_caracteristicas_ELTPARAGRAPHE000104"></a>

Usted también tiene la capacidad de **visualizar los valores de los datos**. Este parámetro se visualiza si se cumplen las dos condiciones siguientes:

- si se dibuja el punto de la serie ("Tipo de punto" en la pestaña "Serie" de la ventana de control de description).  

- si la opción "Mostrar valores" está seleccionada en la pestaña "Detalles" de la descripción del control Gráfico.


También se puede seleccionar una **imagen de fondo** para el control Gráfico en la pestaña "Detalles" de la ventana de descripción del control.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos "Line" se pueden mostrar con una animación. Para animar este tipo de gráfico, basta con marcar "Activar animaciones" en la pestaña "Detalles" del control description.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos "Line" se pueden visualizar en tiempo real. Para obtener más información, consulte [Gráfico en tiempo real](../WDChamp/1000021037.md).

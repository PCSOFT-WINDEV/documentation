
## Opción de menú: operación realizada y acción
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
Una opción de un menú desplegable o contextual en una página puede ser associated con una acción. Esta acción tiene varias características: 

- [Tipo de la acción](#NOTE2_1), 

- [Destino de la acción](#NOTE3_1), 

- [Posibilidad de manipular el contenido de los controles](#NOTE4_1).

## Tipo de la acción: las acciones disponibles
<a name="tipo_accion_las_acciones_disponibles_ELTTEXTE000184"></a>


### Acciones predefinidas
<a name="acciones_predefinidas_ELTPARAGRAPHE000030"></a>

Varias acciones preestablecidas son propuestas por Default en un cuadro de lista desplegable. Estas acciones preestablecidas dependen de los elementos de la Page o del proyecto.

Por ejemplo, usted puede elegir:

- Ejecutar códigos Servidor y Navegador. 

- Ejecutar solo código de clic Navegador

- Mostrar una página del sitio. En este caso, deberá seleccionar la Page deseada e indicar los parámetros previstos, si es necesario;  

- Mostrar una página de otro sitio. En este caso, se debe especificar: 

	- Las opciones de seguridad, 

	- La URL de la Page que debe mostrarse. 




- Mostrar un enlace personalizado. En este caso, deberá introducir los distintos elementos de la dirección (correo electrónico, FTP, Phone, etc.). 




Atención: Las acciones se realizan después del código de clic de la opción de menú (código del navegador y código del servidor).
<a name="NOTE2_2"></a>


### Acciones específicas
<a name="acciones_especificas_ELTPARAGRAPHE000050"></a>

La lista "Tipo" también incluye un enlace "Más opciones".. Esta opción le permite añadir diferentes opciones a los tipos de acciones disponibles. Es posible:

- realizar una acción en otro control, 

- posición en un ancla, 

- compartir en una red social, 

- cambiar idioma actual, 

- enviar el valor de los controles a una página o script. 


Observación: Todas estas operaciones también pueden realizarse en el código.

## Destino de la acción
<a name="destino_accion_ELTTEXTE000214"></a>
Si no se elige ningún destino para la acción, ésta se realiza en el destino Default definido para el Page actual.

Los posibles objetivos de una acción realizada por una opción de menú son:


|   |   |
| --- | --- |
| **Nuevo navegador (_blank):**<br>Muestra el resultado en una nueva ventana del navegador. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_nouveau.gif)<br> |
| **Fotograma actual (_self):**<br>Muestra el resultado en el mismo cuadro. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_FrameEnCours.gif)<br> |
| **Marco principal (_parent):**<br>Muestra el resultado en el marco padre (caso de conjuntos de marcos anidados). | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_FrameParent.gif)<br> |
| **Navegador actual (_top):**<br>Muestra el resultado en la Page actual del navegador (sin marcos). | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_NavEnCours.gif)<br> |
| **Otros marcos del conjunto de marcos (si el Page pertenece a un conjunto de marcos):**<br>Muestra el resultado en un marco específico. |   |

## Posibilidad de manipular el contenido de los controles
<a name="posibilidad_manipular_contenido_los_controles_ELTTEXTE000238"></a>
En función de la acción seleccionada, debe especificar si el contenido de los controles debe enviarse o no al servidor. 

Por defecto, no se associated ninguna acción con la opción de menú, pero puede : 

- Enviar valores al servidor (submit). Esta operación es necesaria si se utiliza la opción de menú para pasar al modo SSL a través de [SSLActive](../WDLang2/3012014.md). 

- Borrar el contenido de los controles en el navegador (reset). Esta opción, conservada por compatibilidad, permite restablecer los mandos del Page.




## Manejo de una control Barra de navegación mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000091"></a>
El control Barra de navegación puede ser manejado a través de la programación. Para hacerlo, usa la Variable de la control Barra de navegación en el código. La Variable de control Barra de navegación corresponde al nombre de control Barra de navegación. El tipo Variable es una cadena de caracteres.

Observación: Las diferentes áreas que se encuentran en la control Barra de navegación no pueden ser manejadas de forma independiente a través de la programación. 



## Manejo de la control Barra de navegación mediante programación
<a name="manejo_control_barra_navegacion_mediante_programacion_ELTTEXTE000115"></a>


### Expansión del área colapsada
<a name="expansion_del_area_colapsada_ELTPARAGRAPHE000017"></a>

Cuando el [Colapsado](../Proprietes/2513014.md) Property se utiliza en el código del navegador en el control Barra de navegación, colapsa o expande el área central de la barra de navegación. 
Puede utilizar [Colapsado](../Proprietes/2513014.md) en el código hover de un control por ejemplo. 

La propiedad [Colapsado](../Proprietes/2513014.md) tiene una acción si la control Barra de navegación está en modo compacto solamente. 

Ejemplo: 


```wl
// -- Load code of the page (browser) 
// Variable used to hide the drop-down menu in case of outside click
gbClickOnBurger is boolean = False

// -- Click on the page (browser code)
IF NOT gbClickOnBurger THEN
	// Hide the menu
	NAV_Bar.Collapsed = True
END
gbClickOnBurger = False
```

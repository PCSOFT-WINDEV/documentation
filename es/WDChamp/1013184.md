## Eventos de Default
<a name="eventos_default_ELTTEXTE000201"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) WINDEV y WINDEV Mobile gestionan los siguientes eventos por Default (orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Inicialización | Se ejecuta al abrir la ventana. (\*)<br><br>Caso especial: El cuadro de lista de navegación se rellena automáticamente:<br><br>Cuando se ejecuta este código, la control List Box aún no se ha rellenado.. Para forzar la control List Box a rellenar este código, utilice:<br><br>- la función [ListDisplay](../WDLang1/3049005.md). <br><br>- las propiedades para gestionar los controles del List Box de navegación: [BrowsedFile](../Proprietes/2510040.md), [BrowsedItem](../Proprietes/2510111.md) o [filtros](../Proprietes/2510042.md).<br><br><br> |
| Fin de inicialización | **Código solo para controles List Box basados en un archivo de datos rellenados automáticamente:**<br>Ejecutar una vez que se haya llenado el control List Box. Este código le permite seleccionar, añadir o modificar un elemento de la control List Box. |
| partida | Ejecutar cuando la control gana el foco |
| salida | Ejecutar al pasar de un control a otro (tecla TAB, seleccionar otro control con el ratón, etc.). |
| Seleccionar una fila | Ejecutar cuando el usuario selecciona un elemento en la fila o modifica el elemento seleccionado. |


(\*) De forma predeterminada, el evento "Inicialización" de cada control se ejecuta según el orden en el que se crearon los controles en la ventana. Para modificar este orden de ejecución: 

1. En la pestaña "Ventana", en el grupo "Orden", haga clic en "Inicialización". 

2. En la ventana que aparece, utilice las flechas de la derecha para cambiar el orden en el que se inicializan los controles.

## Eventos opcionales
<a name="eventos_opcionales_ELTTEXTE000231"></a>
Se pueden utilizar varios eventos opcionales.

Para añadir un evento opcional:

1. Seleccione la control deseada.

2. Mostrar la ventana del código de esta control (tecla F2).

3. Haga clic en el enlace "Añadir otras Event s a xxx" en la parte inferior del código de la ventana, después de la última Event.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. Todos los eventos opcionales disponibles para el control se muestran. 

5. Marque la opción Event para agregar y validar. 
	Observación: Puede seleccionar varios eventos opcionales. 

6. El Event opcional seleccionado se agrega automáticamente a los Event s administrados por el control.




Para desactivar un Event opcional, simplemente realice las mismas operaciones para mostrar la lista de Event s opcionales. A continuación, simplemente desmarque los eventos opcionales para eliminar. 
Observación: Si el código desactivado contiene código WLanguage, este código se elimina automáticamente.

Se puede gestionar, por ejemplo:

- el desplazamiento del cursor por encima del control,

- el botón izquierdo del ratón presionado, liberado o con doble clic,

- el botón derecho del ratón presionado, liberado o con doble clic,

- la rueda del ratón, etc.

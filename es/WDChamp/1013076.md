### Control Botón en una ventana
<a name="control_boton_una_ventana_ELTPARAGRAPHE000061"></a>

El control Botón permite al usuario ejecutar una acción específica con un simple clic.

Un control Botón puede contener:

- texto, <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BOUTON_TEXTE.gif)
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los botones no pueden tener texto multilínea.

- imagen, <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Btn_img.gif)


- texto e imagen.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BTN_IMG_TXT.gif)





WINDEV y WINDEV Mobile ofrecen diferentes opciones para personalizar los botones, su estilo o efecto. 

Es posible: 

- definir el [tipo de botón](../WDChamp/1013093.md). Cada tipo de botón permite realizar una acción específica.

- crear [botones On/Off](../WDChamp/1013073.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) crear [botones con flecha](../WDChamp/1013163.md).

- definir imágenes para ilustrar el botón o personalizar su forma. Para obtener más información, consulte [Imágenes y botones](../WDChamp/1013075.md). 

- crear botones animados. Para obtener más información, consulte [Imágenes y botones](../WDChamp/1013075.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilizar el formato RTF para dar formato al texto del botón. 
	**Observación**: Para utilizar el formato RTF en un control Botón, seleccione la opción "RTF" en la pestaña "General" de la descripción del control. La barra de configuración RTF aparece arriba del título del control para configurarlo fácilmente. También puede copiar directamente un código RTF en el título del control.

### Control Botón en una página
<a name="control_boton_una_pagina_ELTPARAGRAPHE000120"></a>

Un botón puede tener diferentes aspectos:


|   |   |
| --- | --- |
| <br><br>- texto<br><br><br> | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BOUTON_TEXTE.gif)<br> |
| <br><br>- Imagen fija o al pasar el cursor<br><br><br> | Una o dos imágenes están asociadas al botón:<br><br>- una imagen para el botón en estado ***"normal"***,<br><br>- una imagen para el estado ***"sobre"*** (cuando se pasa el cursor sobre el botón).<br><br><br>En este caso, se utilizan imágenes independientes. |
| <br><br>- Imágenes con varios estados<br><br><br> | Diferentes dibujos están asociados al botón. Estos dibujos están agrupados en una imagen llamada "conjunto de imágenes". Se pueden usar hasta 5 dibujos. Los dibujos corresponden a los siguientes estados del botón: normal / presionado / en gris / enfocado / sobre. |


La apariencia del control Botón se puede definir en la pestaña "Estilo" de la ventana de descripción del control, en el elemento "Botón (Imágenes)".

Diferente información permite determinar la acción del botón y los eventos que se ejecutan al pulsar el control Botón:

- [el tipo de control Botón](../WDChamp/1013074.md).

- [la acción definida para el control Botón](../WDChamp/1013071.md).

## Crear un control Botón
<a name="crear_control_boton_ELTTEXTE000301"></a>
Para crear un control Botón: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Botón" (o en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_bouton.gif)).

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control. El control aparece en el editor.




**Consejo**: Puede crear directamente un control Botón "predefinido" (que puede contener código) desplegando "Botón". La lista de controles Botón disponibles se muestra con la plantilla del proyecto. Simplemente haga clic en el control Botón que desea crear. 

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

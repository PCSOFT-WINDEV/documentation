


## Manipular un control Combo Box mediante programación (sintaxis con prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
WINDEV, WINDEV Mobile y WEBDEV permiten manipular controles Combo Box mediante programación. Para ello, utilice la variable del control Combo Box en el código. Esta variable es de tipo numérico.

La variable del control Combo Box:

- corresponde al nombre del control Combo Box.

- se inicializa con el índice del elemento seleccionado en el control.




Para manipular un elemento del control Combo Box, utilice la siguiente notación:

```txt
<Combo Box control>[<Element index>]
```


**Observación**: Existen varias funciones WLanguage específicas para los controles Combo Box. Para obtener más información, consulte: [Funciones de gestión de los controles List Box y Combo Box](../WDLang1/1000017419.md) y [funciones específicas de los controles Combo Box](../WDLang1/3026006.md).

## Inicializar y rellenar un control Combo Box
<a name="inicializar_rellenar_control_combo_box_ELTTEXTE000156"></a>


### Inicializar un elemento
<a name="inicializar_elemento_ELTPARAGRAPHE000179"></a>

Para inicializar un elemento en un control Combo Box con o sin entrada, utilice la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md).

Para modificar un elemento en un control Combo Box con o sin entrada, utilice una asignación directa:


```txt
<Combo Box control>[index] = <Value>
```

<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Inicializar la entrada en un control Combo Box
<a name="inicializar_entrada_control_combo_box_ELTPARAGRAPHE000203"></a>

Para inicializar la entrada en un control Combo Box, utilice:

- asignación directa:
	
	```txt
	<Combo Box control> = <Value>
	```


- la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md).
	
	```wl
	// Add elements to a Combo Box control
	COMBO_COMBO1.Add("MOORE")
	COMBO_COMBO1.Add(Cust_Name)
	```




<a name="NOTE2_3"></a>


### Rellenar un control Combo Box
<a name="rellenar_control_combo_box_ELTPARAGRAPHE000235"></a>Para inicializar un control Combo Box, simplemente utilice un bucle de iteración para inicializar todos los elementos del control.

```txt
Index is int
FOR Index = <Start position> TO <End position>
	<Combo Box control>.Add(<Value>)
END
```
Ejemplo: 

```wl
i is int 
FOR i = 1 TO 52 
	COMBO_COMBO.Add(NAME[i]) 
END
```

## Seleccionar un elemento en un control Combo Box
<a name="seleccionar_elemento_control_combo_box_ELTTEXTE000253"></a>


### Seleccionar un elemento en un control Combo Box
<a name="seleccionar_elemento_control_combo_box_ELTPARAGRAPHE000276"></a>

Para seleccionar un elemento en la fila &lt;Index&gt; utilice:

- asignación directa:
	
	```txt
	<Combo Box control> = Index
	```


- la función [&lt;Caja de lista&gt;.SelectPlus](../WDLang1/1000017414.md):
	
	```txt
	<Combo Box control>.SelectPlus(<Index>)
	```



![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) **Observación**: En un control Combo Box con entrada, solo se puede utilizar la segunda sintaxis.

**Observación**: Si el control Combo Box se inicializa con la función [gStoredValue](../WDLang1/1410088363.md), el valor de [gStoredValue](../WDLang1/1410088363.md) debe utilizarse para seleccionar un elemento. 
Por ejemplo, si el control Combo Box se inicializa con el código: 

```wl
ListAdd(COMBO_Person, "My text" + gStoredValue("StoredValue"))
```
el siguiente código permitirá seleccionar un elemento: 

```wl
COMBO_Person = StoredValue
```

<a name="NOTE3_2"></a>


### Seleccionar el último elemento de un control Combo Box
<a name="seleccionar_ultimo_elemento_control_combo_box_ELTPARAGRAPHE000322"></a>

Para fijar la posición en el último elemento del control Combo Box, utilice:

- la propiedad [Count](../WDLang1/3025009.md): 
	
	```txt
	<Combo Box control> = <Combo Box control>.Count
	```


- la función [&lt;Caja de lista&gt;.SelectPlus](../WDLang1/1000017414.md) con la propiedad [Count](../WDLang1/3025009.md):
	
	```txt
	<Combo Box control>.SelectPlus(<Combo Box control>.Count)
	```


- la función [&lt;List Caja&gt;.Cuenta](../WDLang1/1000017334.md):
	
	```txt
	<Combo Box control> = <Combo Box control>.Count()
	```



![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) **Observación**: En un control Combo Box con entrada, solo se puede utilizar la última sintaxis.

## Obtener el elemento seleccionado en el control Combo Box
<a name="obtener_elemento_seleccionado_control_combo_box_ELTTEXTE000372"></a>


### Obtener el índice del elemento seleccionado
<a name="obtener_indice_del_elemento_seleccionado_ELTPARAGRAPHE000395"></a>

Para obtener el índice del elemento seleccionado, utilice una de las siguientes sintaxis:

- el elemento directamente:
	
	```txt
	Index = <Combo Box control>
	```




<a name="NOTE4_2"></a>


### Obtener el valor del elemento seleccionado
<a name="obtener_valor_del_elemento_seleccionado_ELTPARAGRAPHE000409"></a>

Para obtener el valor del elemento seleccionado, utilice una de las siguientes sintaxis:

- Sintaxis 1: Control Combo Box sin entrada
	
	```txt
	index is int = <Combo Box control>
	<Variable> = <Combo Box control>[Index]
	```


- Sintaxis 2: Control Combo Box sin entrada
	
	```txt
	<Variable> = <Combo Box control>[<Combo Box control>]
	```


- Sintaxis 3: Controles Combo Box con o sin entrada: Propiedad [DisplayedValue](../Proprietes/2510129.md)
	
	
	```txt
	<Variable> = <Combo Box control>.DisplayedValue
	```


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sintaxis específica: Control Combo Box con entrada:
	
	```txt
	<Variable> = <Combo Box control>
	```

## Propiedades específicas de la gestión de controles Combo Box
<a name="propiedades_especificas_gestion_controles_combo_box_ELTTEXTE000474"></a>
Las siguientes propiedades permiten manipular los controles Combo Box: 


|   |   |
| --- | --- |
| **Todos los tipos de controles Combo Box** |
| [HorizontalAlignment](../Proprietes/2510023.md) | Obtiene y establece la alineación horizontal de los elementos en el control Combo Box. |
| [Ellipsis](../Proprietes/2510051.md) | Obtiene y establece el tipo de elipsis. |
| [LineHeight](../Proprietes/2510041.md) | Obtiene y establece la altura de las filas en un control Combo Box. |
| [Memory](../Proprietes/2510076.md) | Indica si el control Combo Box especificado se rellena mediante programación o si está basado en un archivo de datos. |
| [Count](../WDLang1/3025009.md) | Obtiene el número de filas de un control Combo Box. |
| [FillType](../Proprietes/1000017723.md) | Indica cómo se rellena un control Combo Box (mediante programación, desde un archivo de datos o desde una variable). |
| [Empty](../Proprietes/2510126.md) | Indica si un control Combo Box está vacío. |
| **Control Combo Box con entrada únicamente** |
| [WithInput](../Proprietes/1000020285.md) | Obtiene y establece el modo de entrada del control Combo Box (con o sin entrada). |
| [Cursor](../Proprietes/2510025.md) | Obtiene y establece la posición del cursor en un control. |
| [CursorEnd](../Proprietes/2510047.md) | Obtiene y establece la posición final de una selección realizada con el cursor. |
| [MemoryFormat](../Proprietes/2510048.md) | Obtiene y establece el formato del valor devuelto. |
| [DisplayMask](../Proprietes/1000020008.md) | Obtiene y establece la máscara de visualización. |
| [InputMask](../Proprietes/2510033.md) | Obtiene y establece la máscara de entrada. |
| [Size](../Proprietes/2510125.md) | Obtiene y establece el número máximo de caracteres en el campo de entrada del control Combo Box. |
| [SelectedText](../Proprietes/1000017193.md) | Obtiene y modifica el texto seleccionado en el campo de entrada del control Combo Box. |
| [InputType](../Proprietes/2510136.md) | Obtiene y establece el tipo de datos introducidos en el campo de entrada del control Combo Box. |
| **Control Combo Box sin entrada únicamente** |
| [SearchAAF](../Proprietes/1000023445.md) | Permite: <br><br>- determinar si la opción de búsqueda está activada en un control Combo Box sin entrada,<br><br>- activar o desactivar la opción de búsqueda mediante las AAF en un control Combo Box sin entrada. <br><br><br> |
| **Control Combo Box rellenado mediante programación únicamente** |
| [InitialContent](../Proprietes/2510021.md) | Obtiene el contenido inicial de un control Combo Box rellenado mediante programación. |
| [Sorted](../Proprietes/2510115.md) | Indica si un control Combo Box rellenado mediante programación está ordenado, u ordena el control. |
| **Control Combo Box basado en un archivo de datos únicamente** |
| [BrowsedFile](../Proprietes/2510040.md) | Obtiene y establece el archivo de datos o consulta utilizada para mostrar registros en controles Combo Box. |
| [AutoBrowse](../Proprietes/2510093.md) | Indica si un control Combo Box se recorre de forma automática o mediante programación. |
| [DisplayedItem](../Proprietes/2510123.md) | Obtiene y establece el campo que se muestra en un control Combo Box. |
| [StoredItem](../Proprietes/2510110.md) | Obtiene y establece el campo almacenado de un control Combo Box. |
| [BrowsedItem](../Proprietes/2510111.md) | Obtiene y establece el campo utilizado para recorrer automáticamente los controles Combo Box. |


Para obtener la lista completa de las propiedades WLanguage que pueden utilizarse con un control Combo Box, consulte [Propiedades asociadas a los controles Combo Box](../WDChamp/1013014.md).




## Menú contextual associated con controles Tabla y sus columnas (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000183"></a>
Las aplicaciones WINDEV incluyen automáticamente un menú contextual para controles Tabla, TreeView controles Tabla y sus columnas. 

Este menú contextual, disponible sin programación, presenta varias funciones de useful: estas características son parte de las características de la aplicación de automatic (AAF). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## AAF disponibles en el menú contextual de Table y TreeView controles Tabla
<a name="aaf_disponibles_menu_contextual_table_treeview_controles_tabla_ELTTEXTE000207"></a>
El menú contextual del sistema (también llamado "Menú AAF") associated por Default con la Tabla y TreeView controles Tabla incluye las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Table_MenuCtx.gif)


- **Copy** la célula, el fila o el contenido de control. Para obtener más información, consulte [Tratamiento de las columnas de una control Tabla en tiempo de ejecución (AAF)](../WDChamp/9500109.md). 

- **Store el status de las pausas** (si control Tabla contiene pausas). Para obtener más información, consulte [Almacenamiento de las pausas en la controles Tabla (AAF)](../WDChamp/9500138.md). 

- **Visualizar información en varias líneas** (para las columnas de texto). Para obtener más información, consulte [Tratamiento de las columnas de una control Tabla en tiempo de ejecución (AAF)](../WDChamp/9500109.md). 

- **automatic cálculos** (suma, media, ...) para algunos tipos de columnas. Para obtener más información, consulte [Cálculos de la automatic en una control Tabla (AAF)](../WDChamp/9500141.md). 

- **Exportar el tabla**: 

	- a Word de Default o a OpenOffice Writer si está instalado.

	- a Excel de Default o a OpenOffice Calc si está instalado.

	- to XML.


Para obtener más información, consulte [Exportación de datos (AAF)](../WDChamp/9500142.md). 

- **Gráfico**: Crear un gráfico para representar datos (en las columnas numéricas). Para obtener más información, consulte [Gráfico en una columna de tabla (AAF)](../WDChamp/9500143.md). 

- **Seleccionar varias columnas** de control Tabla.
	Para simplificar el menú, sólo se proponen las columnas con un título.
	Esta opción permite al usuario mostrar u ocultar las columnas (para limitar el tamaño del menú, sólo se muestran las primeras 32 columnas del control Tabla).

- **Print** el contenido del control Tabla (impresión directa o impresión a través de "Reports & Queries"). Para obtener más información, consulte [Imprimir el contenido de una tabla (AAF)](../WDChamp/9500132.md). 

- Definir el fondo **color**. Para obtener más información, consulte [Colorear una fila, una columna o una celda que se encuentra en una control (AAF)](../WDChamp/9500139.md).




Este menú permite al usuario realizar fácilmente diferentes procesos no necesariamente planificados en la aplicación.

**Observaciones**: 

- La Tabla y la Vista de Árbol controles Tabla proponen varios otros AAF.  

- Tiene la posibilidad de añadir opciones a este menú: 

	- del editor (ver abajo). 

	- mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

## Configurar el menú contextual automatic de Tabla y Treeview controles Tabla
<a name="configurar_menu_contextual_automatic_tabla_treeview_controles_tabla_ELTTEXTE000231"></a>


### Configurando el menú automatic en una Tabla/Revisión control Tabla
<a name="configurando_menu_automatic_una_tablarevision_control_tabla_ELTPARAGRAPHE000090"></a>

El menú contextual de control puede desactivarse. Para configurar la visualización del menú contextual:

1. Abra la ventana de descripción del control.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.":<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Menu2.gif)





Puede elegir visualizar: 

- el menú del sistema (menú AAF). Las opciones se pueden configurar con la opción "Parámetros. Este menú se puede mostrar o no. 

- un menú contextual personalizado. Este menú se puede visualizar por sí solo o como complemento del menú del sistema.. En este caso, se puede especificar si este menú debe visualizarse antes o después del menú del sistema.. Para obtener más información, consulte [Gestión de menús contextuales](../WDChamp/1010011.md).

- no menu. 




**Observación**: La misma configuración se puede realizar para las columnas de la control Tabla. Por defecto, el menú contextual de control Tabla es associated con todas las columnas.  
<a name="NOTE3_2"></a>


### Configurando la opción "Seleccionar columnas...".
<a name="configurando_opcion_seleccionar_columnas_ELTPARAGRAPHE000119"></a>

En algunos casos, es posible que no desee ofrecer al usuario la posibilidad de ocultar las columnas.. Para ello, se puede activar o desactivar la opción "Seleccionar columnas...":

1. Abra la ventana de descripción del control.

2. Seleccione la pestaña "Detalles".

3. Marcar o desmarcar el menú "'Seleccionar columnas...'".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Menu3.gif)

### Configurar algunas opciones a través de la programación: Funciones de gestión de las características de automatic
<a name="configurar_algunas_opciones_traves_programacion_funciones_gestion_las_caracteristicas_automatic_ELTPARAGRAPHE000133"></a>

Varias funciones de WLanguage se utilizan para gestionar la visualización de algunas opciones en el menú contextual del sistema de la Tabla o control Tabla TreeView:



|   |   |
| --- | --- |
| [AAFDisable](../WDLang1/1000022018.md) | Desactiva una función de aplicación de automatic (AAF) en una ventana de control o en la aplicación actual. |
| [AAFExecute](../WDLang1/1000022099.md) | Ejecuta una función de aplicación de automatic (AAF) en una control o ventana. Esta función se puede utilizar para iniciar el corrector ortográfico en un control Campo de entrada, por ejemplo. |

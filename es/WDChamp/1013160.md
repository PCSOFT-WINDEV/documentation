## Principio
<a name="principio_ELTTEXTE000149"></a>
Se pueden utilizar dos métodos para animar una control Estático:

- Scrolling leyenda: el pie de foto se desplaza de un lado de la control al otro. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LibelleScrollant.gif)

## Animación de una control Estático
<a name="animacion_una_control_estatico_ELTTEXTE000173"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000022"></a>

Para animar un control Estático: 

1. Mostrar la ventana description del control estático ("description" del el menú contextual del control estático ).

2. En la pestaña "Detalles" configure las opciones de animación. Elija el tipo de animación y configure las opciones de associated. Las opciones dependen del modo de animación seleccionado.




Por ejemplo, **si la animación de Scrolling se selecciona**, tiene la capacidad de especificar:

- La velocidad de actualización entre cada movimiento.

- El número de píxeles correspondiente a un movimiento.

- La dirección de desplazamiento.




Por ejemplo, **si la animación parpadeante está seleccionada**tiene la capacidad de especificar:

- La velocidad de actualización entre cada parpadeo.

- La parpadeante Color de la leyenda. Se necesitan dos colores para que el título parpadee:

	- la Color estándar de la leyenda.

	- la parpadeante Color de la leyenda. Si este color es Transparent, la leyenda se mostrará cada dos por tres.






<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000046"></a>

- Los textos RTF y los textos con "sombra borrosa" no pueden ser animados.

- La animación de una control Estático sólo es Visible en tiempo de ejecución.

- El texto animado está limitado a 1024 bytes.

- La animación deja de funcionar si se utilizan las funciones para dar formato al texto ([gFontBold](../WDLang1/3029061.md), [gFontSize](../WDLang1/3029064.md), [gFontItalic](../WDLang1/3029062.md), [gFontUnderlined](../WDLang1/3029063.md)).

## Programación
<a name="programacion_ELTTEXTE000203"></a>
Se utilizan varias funciones de WLanguage para gestionar los controles estáticos animados. Estas funciones son las siguientes:


|   |   |
| --- | --- |
| [AnimationGetParameter](../WDLang1/3017003.md) | Lee el valor de un parámetro de animación definido para una Image en una control Estático. |
| [AnimationSetMode](../WDLang1/3017001.md) | Cambia el tipo de animación que se usa en un control Estático. |
| [AnimationSetParameter](../WDLang1/3017002.md) | Cambia uno de los parámetros de animación definidos para un texto en un control Estático. |


<a name="NOTE3_2"></a>
Se utilizan varias propiedades de WLanguage para gestionar los controles estáticos animados. Estas propiedades son las siguientes:


|   |   |
| --- | --- |
| [Animación](../Proprietes/2510006.md) | Permite conocer el estado actual de la animación en un control Estático e iniciar/parar una animación en un control Estático animado. |
| [InitialAnimation](../Proprietes/2510012.md) | Permite conocer el estado inicial de la animación en un control Estático. |

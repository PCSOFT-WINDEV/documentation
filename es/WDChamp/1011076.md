## Creación de una reporte multicolumna
<a name="creacion_una_reporte_multicolumna_ELTTEXTE000161"></a>


### Creación de una nueva reporte
<a name="creacion_una_nueva_reporte_ELTPARAGRAPHE000043"></a>

Para crear una reporte "Multicolumna: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido.

	- Aparece la ventana de creación: haga clic en "Reporte", y luego en "Reporte".

	- El asistente de creación de reportes se abre.




2. Seleccione "Multicolumna reporte". 

3. En esta asistente, debe especificar:

	- El número de columnas

	- El tipo reporte (formulario o tabla)

	- La fuente de datos de la reporte (archivo de datos, consulta, ...). Para obtener más información, consulte [Fuente de datos Reporte](../WDChamp/1011055.md).

	- La información específica de las clasificaciones y rupturas.

	- Las posiciones a imprimir.

	- Los cálculos de automatic sobre los artículos (Suma, Promedio, Contador).

	- El formato del papel y la orientación (vertical, horizontal).

	- La plantilla a aplicar.

	- El título y el nombre de reporte.








### Modificar una reporte existente
<a name="modificar_una_reporte_existente_ELTPARAGRAPHE000070"></a>

Tiene la capacidad de transformar una reporte existente en una reporte multicolumna.. Para ello:

1. Abre la ventana de description del reporte.

2. Seleccione la pestaña "Formato".

3. En el área "Multicolumna", seleccione el número de columnas para la reporte.




**Observación**: Los controles de reporte deberán reposicionarse para adaptarse al nuevo tamaño de reporte.



## Propiedades de WLanguage associated con los informes multicolumna
<a name="propiedades_wlanguage_associated_con_los_informes_multicolumna_ELTTEXTE000191"></a>
Utilice el [NbColumn](../Proprietes/1000018810.md) Property para determinar mediante programación si un reporte tiene varias columnas. 

Para get la lista de propiedades disponibles en un reporte, véase [Propiedades associated con un reporte](../WDChamp/1011029.md).


## Gestión de las opciones de menú en las ventanas MDI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
Las ventanas MDI deben respetar algunas restricciones específicas de las aplicaciones MDI. Hay tres tipos de ventanas disponibles: MDI ventana padre, MDI ventana hija y MDI ventana hermana.

La ventana que abre otra ventana se llama **ventana principal** las ventanas abiertas por la ventana principal se denominan **ventanas para niños**. Todas las ventanas hijas abiertas por la misma ventana padre son **ventanas hermanas**.

Para obtener más información, consulte [Los diferentes tipos de ventanas](../WDChamp/1010024.md).

Las operaciones realizadas en las opciones del menú son las mismas, independientemente de que el menú pertenezca o no a una ventana de MDI.. Sin embargo, hay características adicionales disponibles en las ventanas MDI:

- utilizando [grupos de opciones de menú](../WDChamp/1010015.md) para mostrar algunas opciones de menú en la ventana padre MDI.

- mostrar la [lista de ventanas MDI hijas abiertas](../WDChamp/1010015.md) en el menú principal de la ventana principal del MDI.




Observación: Para más detalles sobre los menús y las opciones de menú, véase:

- [Gestión de los menús emergentes en una aplicación WINDEV](../WDChamp/1010011.md).

- [Gestión de los menús de la ventana principal](../WDChamp/1010010.md).

- [Manipular las opciones de menú en el editor](../WDChamp/1010008.md).

## Grupos de opciones de menú
<a name="grupos_opciones_menu_ELTTEXTE000184"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000051"></a>

WINDEV le permite mostrar algunas opciones de menú en la ventana MDI padre sólo cuando se abre una ventana MDI hija.. Para ello:

- asociar las opciones del menú para mostrar con un grupo.

- asociar la ventana MDI hija a este mismo grupo.




Ejemplo: Los grupos de opciones de menú se utilizan para gestionar las funciones propuestas a los usuarios en función de sus derechos de acceso.

Observaciones:

- La misma opción de menú se puede mostrar cuando se abren varias ventanas secundarias. Todo lo que tiene que hacer es asociar esta opción con varios grupos.

- Para que una opción se muestre independientemente de la ventana secundaria abierta, ningún grupo debe ser associated con esta opción.



<a name="NOTE2_2"></a>


### Asociar una opción de menú a un grupo
<a name="asociar_una_opcion_menu_grupo_ELTPARAGRAPHE000068"></a>

Para asociar una opción de menú a un grupo: 

1. Visualice la pestaña "UI" de la opción description ("Opción description" de el menú contextual de la opción).

2. Desmarcar "Visible".. La opción se Visible automáticamente cuando se abre la ventana MDI hija. De hecho, la visibilidad de la opción es gestionada automáticamente por WINDEV.

3. Haga clic en el botón "Grupo. Se visualiza la lista de grupos existentes.

4. Cree un nuevo grupo si es necesario.

5. Asociar la opción con el grupo (marque la casilla correspondiente). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GroupeOption.gif)




<a name="NOTE2_3"></a>


### Asociar una ventana secundaria de MDI a un grupo
<a name="asociar_una_ventana_secundaria_mdi_grupo_ELTPARAGRAPHE000082"></a>

Para asociar una ventana secundaria de MDI a un grupo: 

1. Abra la ventana MDI hijo.

2. Visualice los "Detalles" en la description de esta ventana ("description" de el menú contextual).

3. Escriba el nombre del grupo a asociar en la opción "Grupo MDI".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OptionGroupeMDI.gif)

## Cómo averiguar qué ventanas MDI hijas se abren
<a name="como_averiguar_que_ventanas_mdi_hijas_abren_ELTTEXTE000220"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000096"></a>

WINDEV le permite mostrar el título de las ventanas MDI hijas abiertas en el menú principal de la ventana MDI padre.. El título de la ventana actual está precedido por una marca de verificación. Para mostrar una de las ventanas abiertas, todo lo que tiene que hacer es seleccionar su título.

Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenFilleMDIOuvertepsd.gif)

<a name="NOTE3_2"></a>


### Visualización de la lista de ventanas MDI abiertas
<a name="visualizacion_lista_ventanas_mdi_abiertas_ELTPARAGRAPHE000106"></a>

Para visualizar la lista de ventanas MDI abiertas: 

1. Seleccione una de las opciones del menú principal. La lista de ventanas MDI hijas abiertas se mostrará después de la última subopción de esta opción.

2. Visualice la pestaña "UI" en el description de esta opción ("description" de el menú contextual).

3. Marque "Menú MDI".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OptionMenuMDI.gif)




## Gestión de menús contextuales en una aplicación WINDEV o WINDEV Mobile
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000145"></a>
Los usuarios pueden abrir el menú contextual de una ventana: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) pulsando el botón derecho del ratón.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) mediante una pulsación larga en la control. 


Este menú permite a los usuarios acceder fácilmente a funciones específicas de una ventana o de control.

WINDEV y WINDEV Mobile le permiten gestionar varios tipos de menús contextuales:

- **Menús contextuales personalizados**. Estos menús se crean en el editor de ventanas y pueden manipularse mediante programación. 
	Para obtener más información, consulte [Menú contextual personalizado](../WDChamp/1010017.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Menús contextuales AAF** ( automatic Características de la aplicación) administrados por WINDEV. Estos menús contextuales aparecen al hacer clic con el botón derecho del ratón en algunos tipos de controles (Edición, Gráfico, controles Tabla, etc.). Estos menús contextuales permiten al usuario final utilizar las funciones automatic de la aplicación. 
	Para más detalles sobre los AAF, véase [Funcionalidades Automáticas de la Aplicación](../Editeurs/9000022.md).
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Los menús contextuales AAF sólo están disponibles en controles Tabla.

- **Menús contextuales del sistema** ( Android , Windows, etc.). Estos menús contextuales aparecen en algunos tipos de controles (ActiveX, etc.). 
	Para obtener más información, consulte [Menú contextual del sistema](../WDChamp/1010023.md).
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Los menús contextuales del sistema no están disponibles.




Ejemplo de menú contextual personalizado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuContextuel.gif)


Observación: En una página WEBDEV, también es posible gestionar los menús contextuales mediante la función [Menú contextual](../WDChamp/1410087217.md) control.

## Menú contextual en los controles Moneda + Euro
<a name="menu_contextual_los_controles_moneda_euro_ELTTEXTE000169"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En los controles "Moneda + Euro" (controles de edición, columna de una control Tabla, etc.), un menú contextual puede permitir a los usuarios elegir la moneda. Este menú contextual presenta el resultado de la conversión para el valor introducido en las diferentes monedas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuctxMonetaireeuro.gif)
 Si este menú contextual está disponible para los usuarios, el menú contextual associated con el control será ignorado.

Para activar este menú contextual del sistema: 

1. Abrir la ventana description de la "Moneda + Euro" Editar control. 

2. En la pestaña "Detalles", en la sección "Gestión de euros", pulse "Opciones".. 

3. En la ventana que aparece, marque "Modificar divisa mediante un menú contextual" y confirme. Validar la ventana. 

4. Valide la ventana de descripción del control.

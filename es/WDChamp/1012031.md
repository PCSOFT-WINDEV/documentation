


## Estilo de las páginas: Imagen de fondo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000176"></a>
Para optimizar el estilo de sus páginas, WEBDEV le permite utilizar: 

- un fondo Image en sus páginas. Sus controles se muestran Above este fondo Image.

- un Image en el navegador Background. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fondo Image de páginas
<a name="fondo_image_paginas_ELTTEXTE000200"></a>
Para definir el Image que se usará en la página Background: 

1. Abra la ventana de descripción de la página: en la pestaña "Página", en el grupo "Descripción", haga clic en "Descripción" (o seleccione la opción "Descripción" en el menú contextual de la página).

2. En la pestaña "Estilo", en el área "Estilo", seleccione el elemento "Página Border/Background".. 

3. Para definir el fondo de la página Color, especifique:

	- la Color de las fronteras

	- el fondo Color

	- el estilo de las cenefas y Background. 




4. Para definir el fondo de la página Image: 

	- Seleccione la Image que se utilizará como fondo de la página (a través del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) del "Fondo Image" control): 

		- La opción "Navegar" se utiliza para seleccionar un archivo Image accesible desde su ordenador.

		- La opción "Catálogo" permite seleccionar un Image en el catálogo de Image proporcionado con WEBDEV.




- Especifique las características de la imagen repetition: 

	- Ninguno,

	- Image de azulejos (en altura y anchura),

	- Imagen repetida X (horizontalmente),

	- Alicatada Image Y (en altura).

5. Valide.




**Observaciones**: 

- El modo de gestión del tamaño de Image es una característica disponible en CSS3. Para utilizar esta función, la página debe generarse en modo HTML 5. Se puede definir el modo de generación de páginas HTML: 

	- para el proyecto (pestaña "Avanzado" del proyecto description, opción "Generación estándar de código HTML"). 

	- para la página (pestaña "General" de la página description, opción "Generación estándar de código HTML"). 




- Para definir la misma página de fondo Image para todas las páginas de su proyecto, defina este Image para la página modelo utilizada por su proyecto. Para obtener más información, consulte [Modelo de páginas](../WDChamp/1012015.md). 

- El fondo Image es mostrado por el navegador. Si el archivo Image utilizado es demasiado grande, la visualización de la página puede ser más lenta.






## Fondo Image del navegador
<a name="fondo_image_del_navegador_ELTTEXTE000224"></a>
Para definir el Image que se utilizará en el navegador Background: 

1. Abra la ventana de descripción de la página: en la pestaña "Página", en el grupo "Descripción", haga clic en "Descripción" (o seleccione la opción "Descripción" en el menú contextual de la página).

2. En la pestaña "Estilo", en el área "Estilo", seleccione el elemento "Browser Background".. 

3. Especifique (si es necesario) el fondo Color del navegador. 

4. Especificar las características del fondo Image: 

	- Seleccione la Image que se utilizará como fondo del navegador (a través del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) del "Fondo Image" control): 

		- La opción "Navegar" se utiliza para seleccionar un archivo Image accesible desde su ordenador.

		- La opción "Catálogo" permite seleccionar un Image en el catálogo de Image proporcionado con WEBDEV.




- Para crear una Gradient, haz clic en el botón "Crear una Gradient".. Este botón permite generar una imagen que contiene un Gradient y utilizarla como navegador Background. Una asistente te ayuda a crear la Gradient.

- Si la opción "Fondo fijo Image" está marcada, el fondo Image del navegador se fijará cuando el usuario de la web desplace la página (barra de desplazamiento vertical).

- Especifique las características de la imagen repetition: 

	- Ninguno,

	- Image de azulejos (en altura y anchura),

	- Imagen repetida X (horizontalmente),

	- Alicatada Image Y (en altura).

5. Valide.




**Observaciones**: 

- El modo de gestión del tamaño de Image es una característica disponible en CSS3. Para utilizar esta función, la página debe generarse en modo HTML 5. Se puede definir el modo de generación de páginas HTML: 

	- para el proyecto (pestaña "Avanzado" del proyecto description, opción "Generación estándar de código HTML"). 

	- para la página (pestaña "General" de la página description, opción "Generación estándar de código HTML"). 




- Para definir el mismo fondo de pantalla Image para todas las páginas de su proyecto, defina este Image para la página modelo utilizada por su proyecto. Para obtener más información, consulte [Modelo de páginas](../WDChamp/1012015.md). 

- El fondo Image es mostrado por el navegador. Si el archivo Image utilizado es demasiado grande, la visualización de la página puede ser más lenta.

## Presentación
<a name="presentacion_ELTTEXTE000753"></a>
En el momento de la ejecución, en el control Hoja de cálculo, el usuario final introduce directamente las fórmulas de cálculo. Las celdas utilizadas se pueden seleccionar con el ratón.

La control Hoja de cálculo propone las principales funciones y operadores de cálculo: suma, resta, multiplicación, división, porcentajes, media, mín-máx, condición (IF), cálculo y operadores comparison, ...

Se pueden utilizar varias funciones de WLanguage en las fórmulas, como las funciones matemáticas, las funciones de cadena, las funciones de fecha, las funciones financieras, ....

En las fórmulas, tiene la posibilidad de utilizar los procedimientos de WLanguage que se han creado para este fin en la aplicación ([Hoja de cálculoPermitirProcedimiento](../WDLang1/1000021473.md)). Por ejemplo, la aplicación puede proponer el "Coste de envío" Procedure que calcula el importe según parámetros avanzados: peso, volumen, velocidad, destino Country, ... 
¡Este Procedure incluso se puede utilizar en la hoja de cálculo!

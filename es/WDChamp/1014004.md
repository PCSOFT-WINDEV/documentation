## Presentación
<a name="presentacion_ELTTEXTE000155"></a>
Por defecto, las Event disponibles en el editor de códigos corresponden a las Event más comunes.

Sin embargo, las ventanas, las páginas y sus controles pueden gestionar eventos opcionales (o procesos opcionales).

Esto le permite gestionar según las plataformas:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para las ventanas y sus controles: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Entrada en una control en rollover y salida de una control en rollover con el ratón

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el desplazamiento del cursor por encima del control,

	- el botón izquierdo abajo, arriba, con doble clic (en algunos controles),

	- el botón derecho abajo, arriba, con doble clic (en algunos controles),

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la rueda del ratón,

	- ...


En estos eventos opcionales, podrá conocer las diferentes acciones realizadas por el usuario (tecla pulsada, uso de la rueda del ratón, etc.).

## Añadiendo una Event opcional
<a name="anadiendo_una_event_opcional_ELTTEXTE000179"></a>
Para añadir una Event opcional en una ventana, una página o una control de una ventana o página:

1. Seleccione la ventana, la página o la control solicitada.

2. Mostrar la ventana de códigos de esta ventana, página o control (tecla F2).

3. Haga clic en el enlace "Añadir otros Event s a xxx" en la parte inferior del código de la ventana, después del último Event.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. Se visualizan todos los eventos opcionales disponibles para el elemento. Por ejemplo, para una ventana: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0002.gif&type=thumb)


5. Marque la opción Event para agregar y validar. 
	Observación: Puede seleccionar varios eventos opcionales. 

6. La Event opcional seleccionada se añade automáticamente a las Event s gestionadas por la ventana, página o control.




**Atención**: Un Event opcional es **siempre** ejecutado (a diferencia de los Event estándar), incluso cuando se llama a [esperar](../WDLang1/3015012.md) o [Multitask](../WDLang1/3015004.md).

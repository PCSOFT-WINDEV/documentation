


## Propiedades associated con un reporte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes propiedades pueden utilizarse en los informes:



|   |   |
| --- | --- |
| [AllModifiableWithReportsAndQueries](../Proprietes/2511076.md) | El **AllModifiableWithReportsAndQueries** Property se utiliza para determinar si un reporte puede ser modificado en Reports & Queries. |
| [BackgroundImage](../Proprietes/2510056.md) | La propiedad **..BackgroundImage** permite conocer y modificar la imagen de fondo de un elemento: ventana, página, control o reporte. |
| [BackgroundImagePrinted](../Proprietes/1000021350.md) | La **BackgroundImagePrinted** se utiliza para: <br><br>- Determina si el fondo Image de un reporte (formulario u otro) debe imprimirse.<br><br>- Imprimir (o no) el fondo Image de una reporte. <br><br><br> |
| [BottomMargin](../Proprietes/2511051.md) | El **BottomMargin** Property se utiliza para:<br><br>- Descubre el margen inferior de un reporte.<br><br>- Modificar el margen inferior de un reporte.<br><br><br> |
| [BrowsedItem](../Proprietes/2510111.md) | El **BrowsedItem** Property se utiliza para:<br><br>- get el elemento utilizado para recorrer automáticamente los controles de Tabla, Looper, Cuadro de Lista, ListView y Combo Box, supercontroles e informes basados en archivos de datos.<br><br>- Cambia el elemento utilizado para recorrer automáticamente los controles de Tabla, Looper, Cuadro de Lista, ListView y Combo Box, los supercontroles y los informes basados en archivos de datos. A continuación, se reinicia la iteración.<br><br><br> |
| [HSpace](../Proprietes/2511067.md) | La función **HSpace** Property se utiliza para determinar y modificar el espaciado horizontal entre etiquetas en un reporte. Este espacio se definió al crear la etiqueta reporte. |
| [HTML](../Proprietes/2511040.md) | El **HTML** Property se utiliza para: <br><br>- averiguar si el reporte actual se imprime en modo HTML. <br><br>- gestiona el tipo HTML de un control Estático que se encuentra en un reporte.<br><br><br> |
| [InterfaceModifiableWithReportsAndQueries](../Proprietes/2511075.md) | La propiedad **InterfaceModifiableWitReportsAndQueries** permite saber si la **interfaz** de un reporte puede editarse en Reports & Queries. |
| [KeepVisibleHierarchy](../Proprietes/1000021055.md) | El **KeepVisibleHierarchy** Property se utiliza para:<br><br>- Averigüe qué elementos de TreeView tabla se imprimirán: imprimiendo toda la jerarquía o imprimiendo la jerarquía que es Visible en el control.<br><br>- Modificar el modo de impresión de la jerarquía: imprimiendo toda la jerarquía o imprimiendo la jerarquía que es Visible en el control.<br><br><br> |
| [LabelHeight](../Proprietes/2511069.md) | El **LabelHeight** Property se utiliza para:<br><br>- Averiguar la altura de las etiquetas en una etiqueta reporte.<br><br>- Modificar la altura de las etiquetas en una etiqueta reporte.<br><br><br> |
| [LabelNumberToStart](../Proprietes/2511032.md) | El **LabelNumberToStart** Property se utiliza para:<br><br>- Averigüe el número de la etiqueta en la hoja de etiquetas desde la que se realizará la impresión..<br><br>- Modificar el número de la etiqueta desde la que se realizará la impresión en la hoja de etiquetas..<br><br><br> |
| [LabelWidth](../Proprietes/2511055.md) | El **LabelWidth** Property se utiliza para:<br><br>- Descubra el ancho de las etiquetas de una "Etiqueta" reporte.<br><br>- Modificar el ancho de las etiquetas en una "Etiqueta" reporte.<br><br><br> |
| [LeftMargin](../Proprietes/2511048.md) | El **LeftMargin** Property se utiliza para:<br><br>- Encuentra el margen izquierdo de un reporte.<br><br>- Modificar el margen izquierdo de un reporte.<br><br><br> |
| [MaxValue](../Proprietes/2510009.md) | El **MaxValue** Property se utiliza para:<br><br>- averiguar y modificar el límite superior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite superior definido para el filtro actual.<br><br>- averiguar el valor máximo de una serie de gráficos.<br><br>- averiguar y modificar el valor máximo para algunos tipos de controles.<br><br><br> |
| [MemZoneName](../Proprietes/2511029.md) | El **MemZoneName** Property se utiliza para:<br><br>- averigua el nombre de la zona de memoria utilizada como fuente de datos para un reporte.<br><br>- modificar el nombre de la zona de memoria utilizada como fuente de datos para un reporte.<br><br><br> |
| [MinValue](../Proprietes/2510008.md) | El **MinValue** Property se utiliza para:<br><br>- averiguar y modificar el límite inferior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite inferior definido para el filtro actual.<br><br>- averiguar el valor mínimo de una serie de gráficos.<br><br>- averiguar y modificar el valor mínimo para algunos tipos de controles. <br><br><br> |
| [MsgSortSel](../Proprietes/2511045.md) | El **MsgSortSel** Property se utiliza para:<br><br>- Descubre el mensaje de espera que aparece al ordenar y seleccionar los registros reporte.<br><br>- Modificar el mensaje de espera que aparece al ordenar y seleccionar los registros reporte.<br><br><br> |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [NbColumn](../Proprietes/1000018810.md) | El **NbColumn** Property se utiliza para get el número de columnas en un reporte. |
| [NbCopy](../Proprietes/2511042.md) | El **NbCopy** Property se utiliza para determinar y modificar el número de copias reporte a imprimir. |
| [NbLabel](../Proprietes/2511050.md) | El **NbLabel** Property se utiliza para:<br><br>- Descubra el número de etiquetas impresas en todo el ancho de la página.<br><br>- Modificar el número de etiquetas impresas en todo el ancho de la página.<br><br><br> |
| [NbLabelCopy](../Proprietes/2511059.md) | El **NbLabelCopy** Property se utiliza para:<br><br>- get el número de copias de etiquetas idénticas.<br><br>- Cambiar el número de copias de etiquetas idénticas.<br><br><br> |
| [NbPrintedRec](../Proprietes/2511058.md) | El **NbPrintedRec** Property se utiliza para determinar el número de registros ya impresos. |
| [NbRecRead](../Proprietes/2511041.md) | **NbRecRead** se utiliza para conocer el número de registros leídos: seleccionado tanto por la fuente de la reporte como por el código del filtro (si existe). |
| [ODBCConnectionName](../Proprietes/2511033.md) | El **ODBCConnectionName** Property se utiliza para:<br><br>- Identifica el nombre de la conexión ODBC utilizada al imprimir un reporte basado en una "consulta realizada en una conexión".<br><br>- Especifique el nombre de la conexión ODBC utilizada al imprimir un reporte basado en una "consulta realizada en una conexión"..<br><br><br> |
| [ODBCDatabaseName](../Proprietes/2511034.md) | **ODBCDatabaseName** se utiliza para:<br><br>- Averiguar el nombre de la base de datos utilizada al imprimir un reporte a partir de una "consulta realizada en una conexión"..<br><br>- Especifique el nombre de la base de datos utilizada al imprimir un reporte basado en una "consulta realizada en una conexión"..<br><br><br> |
| [ODBCPassword](../Proprietes/2511031.md) | El **ODBCPassword** Property se utiliza para:<br><br>- Identificar la contraseña utilizada para conectarse a una base de datos a través de un controlador ODBC o un Conector Nativo. <br><br>- Especifica la contraseña utilizada para conectarse a una base de datos a través de un controlador ODBC o un Conector Nativo.<br><br><br> |
| [ODBCSQLCode](../Proprietes/2511028.md) | **ODBCSQLCode** se utiliza para:<br><br>- Descubra el código SQL de la "consulta realizada en una conexión" (fuente de datos de reporte). <br><br>- Modificar el código SQL de la "consulta realizada en una conexión" (fuente de datos de reporte).<br><br><br> |
| [ODBCUserName](../Proprietes/2511019.md) | **ODBCUserName** se utiliza para:<br><br>- Identifica el nombre de usuario utilizado para conectarse a una base de datos a través de un controlador ODBC o un Conector Nativo. <br><br>- Especifica el nombre de usuario utilizado para conectarse a una base de datos a través de un controlador ODBC o un Conector Nativo. <br><br><br> |
| [PageByPagePrint](../Proprietes/2511057.md) | El **PageByPagePrint** Property se utiliza para:<br><br>- Pida a un reporte que se imprima página por página. Cada página de la reporte se enviará por separado a la cola de impresión.. <br><br>- Descubra el modo de impresión del reporte (página por página o todos a la vez).<br><br><br> |
| [PageEndStick](../Proprietes/2511074.md) | El **PageEndStick** Property se utiliza para:<br><br>- Averiguar si el pie de página debe estar pegado al último bloque.<br><br>- Modificar el modo de gestión del bloque Pie de página.<br><br><br> |
| [PageNumber](../Proprietes/2510106.md) | La propiedad **PageNumber** permite: <br><br>- get y cambiar el número de la página mostrada en un control Imagen (archivos "multipágina" Image)<br><br>- get y cambiar el número de página de un archivo PDF mostrado en un Image control de una ventana<br><br>- get y cambiar el número Page de un archivo PDF que aparece en el Background de un reporte. <br><br><br> |
| [Plane](../Proprietes/2510094.md) | La propiedad **Plane** permite:<br><br>- Obtener: <br><br>	- el plano activo de una ventana (o reporte) o el plano al que pertenece una control.<br><br>	- el plano Active de un elemento WEBDEV.<br><br><br><br><br>- Cambiar: <br><br>	- el plano activo de una ventana (o reporte) o asocia un control a otro plano.<br><br>	- el plano Active de un elemento WEBDEV.<br><br><br><br><br><br> |
| [Process](../Proprietes/1000020878.md) | El **Process** Property se utiliza para modificar los Process es que serán ejecutados por los eventos disponibles en un control, ventana, Page, reporte, etc. |
| [ReturnedValue](../Proprietes/2510134.md) | El **ReturnedValue** Property se utiliza para:<br><br>- get y cambiar el valor devuelto por una ventana, página o reporte.<br><br>- get y cambiar el valor devuelto por una opción en un control Botón de opción. <br><br><br> |
| [RightMargin](../Proprietes/2511053.md) | El **RightMargin** Property se utiliza para:<br><br>- Encuentra el margen derecho de un reporte.<br><br>- Modificar el margen derecho de un reporte.<br><br><br> |

| [TopMargin](../Proprietes/2511043.md) | El **TopMargin** Property se utiliza para:<br><br>- Descubra el margen superior de un reporte.<br><br>- Modificar el margen superior de un reporte.<br><br><br> |
| [TotalNbRec](../Proprietes/2511047.md) | El **TotalNbRec** Property se utiliza para:<br><br>- get el número de registros a imprimir.<br><br>- Modificar el número de registros a imprimir.<br><br><br> |
| [ViewName](../Proprietes/2511030.md) | El **ViewName** Property se utiliza para:<br><br>- encontrar el nombre de la vista HFSQL, fuente de datos de un reporte<br><br>- modifica el nombre de la vista HFSQL, fuente de datos de un reporte.<br><br><br> |
| [VSpace](../Proprietes/2511065.md) | El **VSpace** Property se utiliza para get y cambiar el espacio vertical entre las etiquetas en un reporte. Este espacio se definió al crear la etiqueta reporte. |


## Gráfico de barras radiales
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
El gráfico de barras radiales se utiliza para comparar objetivos o ilustrar el progreso de varios elementos de forma simultánea.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Características de un gráfico de barras radiales
<a name="caracteristicas_grafico_barras_radiales_ELTTEXTE000128"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_Circulaire_de_progres%20-%20HC%20N%B0001.gif)


Elementos del gráfico:

**1.** **Título del gráfico.**
Las características del título del gráfico pueden definirse en la pestaña "Detalles" de la descripción del control (botón "Editar título del gráfico").
Puede definir el título del gráfico, la posición y estilo.

**2.** **Tamaño del agujero**
Un gráfico de barras radiales puede tener un agujero en el centro. El tamaño del agujero se puede definir en la pestaña "General" de la descripción del control.. 

**3.** **Sector del gráfico**
Cada sector del gráfico representa el valor de una fuente de datos específica. Puede mostrar las etiquetas de las categorías y porcentajes. 
Las etiquetas de categoría pueden aparecer dentro o fuera de los sectores. 

**4.** **Tooltip**
El valor de los datos sobre los que se pasa el cursor se muestra en un globo (tooltip).

**5.** **Leyenda del gráfico.**
Las características de la leyenda se pueden definir en la pestaña "Detalles" de la descripción del control (botón "Editar leyenda del gráfico"). Puede definir su posición y estilo.


**Otras características**: 

- Puede elegir el ángulo inicial del gráfico.  

- Las siguientes funciones WLanguage son específicas para este tipo de gráfico: 


|   |   |
| --- | --- |
| [grRBAddData](../WDLang3/1410087792.md) | Añade datos a un gráfico de barras radiales. |
| [grRBMaximumValue](../WDLang3/1410087801.md) | Establece el valor máximo en un gráfico de barras radiales. |
| [grRBRotationDirection](../WDLang3/1410087791.md) | Establece la dirección de rotación en un gráfico de barras radiales. |




Observación: Este tipo de gráfico no acepta valores negativos.

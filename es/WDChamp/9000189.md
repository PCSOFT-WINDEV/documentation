
## Teclado Variable (Uso de teclados físicos en Android)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000139"></a>
El **teclado** Variable permite recuperar los parámetros de los eventos de un teclado físico. De hecho, un dispositivo Android puede tener un teclado físico, o puede estar conectado a un teclado físico (a través del sistema Samsung DeX, por ejemplo). 

Esta Variable debe ser usada sólo en eventos relacionados con el teclado físico: 

- Tecla pulsada (WM_KEYDOWN), 

- Tecla liberada (WM_KEYUP).


En Android, estos dos Event se llaman sólo si son activados por un Event procedente de un teclado físico. 
En Chrome OS, estos dos eventos no se llaman. 



Observación: Este Variable puede ser usado en modo Simulador.

<a name="NOTE2_1"></a>


## Teclado Variable
<a name="teclado_variable_ELTTEXTE000163"></a>
El teclado Variable incluye los siguientes elementos:



|   |   |
| --- | --- |
| Código del teclado | Integro<br><br>Indica el código correspondiente a la tecla que activó el teclado Event. |
| Teclado. | booleano<br><br>Indica si se pulsa la tecla Shift cuando se activa el teclado Event. |
| Teclado.Alt | booleano<br><br>Indica si se pulsa la tecla Alt cuando se activa el teclado Event. |
| Teclado.Ctrl | booleano<br><br>Indica si se pulsa la tecla Ctrl cuando se activa el teclado Event. |






### Gestionar el código de las teclas
<a name="gestionar_codigo_las_teclas_ELTPARAGRAPHE000058"></a>

Para gestionar el código de las teclas: 

1. Incluya el archivo KeyConst.WL en su proyecto. Este archivo asocia una constante específica con cada tecla del teclado. Este archivo se encuentra en el directorio WINDEV/WEBDEV/WINDEV Mobile (subdirectorio "Personal\\External"). Para incluir este archivo en sus aplicaciones, debe utilizar la palabra clave [EXTERN](../Motscles/1511013.md) de la siguiente manera: 
	
	```wl
	EXTERN "KEYCONST.wl"
	```


2. Entonces usa las constantes deseadas con el Teclado Variable.code: 
	
	```wl
	IF Keyboard.Code = VK_Space THEN 
	...
	END
	```



Observaciones: 

- Para gestionar las claves de los caracteres, basta con utilizar el código ASCII de ese carácter (devuelto por [Asc](../WDLang1/3024004.md)).

- Todas las constantes encontradas en el archivo KeyConst.WL comienzan con las letras VK_.. Estas constantes son mostradas por el completador de código en el editor de código desde el momento en que el archivo ha sido integrado con la palabra clave [EXTERN](../Motscles/1511013.md).

## Ejemplo
<a name="ejemplo_ELTTEXTE000193"></a>


### Realizar una acción en el Cayo abajo Event
<a name="realizar_una_accion_cayo_abajo_event_ELTPARAGRAPHE000088"></a>

El siguiente código se utiliza para realizar una acción en la Event "Tecla abajo" de una control Campo de entrada cuando se utiliza Ctrl + V. 


```wl
IF Keyboard.Code = VK_V _AND_ Keyboard.Ctrl THEN 
	Info ("Ctrl V used")
	...
END
```

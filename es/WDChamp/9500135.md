
## AAF: Ampliar el contenido de una control (tabla, hoja de cálculo, cuadro de lista, tabla pivotante, ...)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000139"></a>
Para simplificar la lectura en los control s utilizados para visualizar grandes volúmenes de datos, el usuario final tiene la posibilidad de ampliar los datos que se encuentran en el control. 

Los datos visualizados se pueden ampliar en los siguientes controles: 

- Control Tabla,

- control Tabla encontrada en un control Combo Box,

- Control Tabla TreeView, 

- Control Tabla dinámica, 

- control Hoja de cálculo,

- Control List Box.

## Acercar/alejar el contenido de una control Tabla
<a name="acercaralejar_contenido_una_control_tabla_ELTTEXTE000163"></a>
**To zoom el contenido de control**: 
El usuario puede ampliar el contenido de una Tabla con la tecla Ctrl y la rueda del ratón.. 

**Para restablecer la visualización inicial**, basta con seleccionar "Cancel el zoom" en el menú contextual de control (o pulsar Ctrl + 0). 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Zoom_Table.gif&type=thumb)



### Almacenamiento del zoom
<a name="almacenamiento_del_zoom_ELTPARAGRAPHE000037"></a>

- **En el caso de los Table y TreeView controles Tabla**, el zoom se guarda automáticamente y se vuelve a aplicar la próxima vez que se muestre el control si la opción "Recordar configuración de columnas" está marcada (pestaña "Detalles" de la ventana del control description). 

- **En el control Tabla dinámica**, el zoom se guarda automáticamente.

## Programación
<a name="programacion_ELTTEXTE000193"></a>


### Programación del zoom
<a name="programacion_del_zoom_ELTPARAGRAPHE000051"></a>

Puede programar el zoom que se aplicará a la control (Tabla, Tabla TreeView, Tabla Pivotante, Hoja de Cálculo...) utilizando [Zoom](../Proprietes/1000017201.md) en el control. 

Por ejemplo: 


```wl
// Zoom in 150%
TABLE_Customer.Zoom = 150
```


Si el zoom se realiza mediante programación, el menú contextual de control permite Cancel R el zoom. 


### Desactivar el zoom
<a name="desactivar_zoom_ELTPARAGRAPHE000065"></a>

Para desactivar el zoom en un control, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafZoom*. 

Por ejemplo: 


```wl
// No zoom allowed
AAFDisable(TABLE_Customer, aafZoom)
```



### Cancelar el zoom
<a name="cancelar_zoom_ELTPARAGRAPHE000077"></a>

Para Cancel el zoom en un control Tabla (y restaurar el nivel de zoom inicial), basta con utilizar [AAFExecute](../WDLang1/1000022099.md) con la constante *aafZoom*.


### Modificar la leyenda de la opción utilizada para Cancel el zoom (mediante programación)
<a name="modificar_leyenda_opcion_utilizada_para_cancel_zoom_mediante_programacion_ELTPARAGRAPHE000085"></a>

Para Cancel el zoom, los usuarios finales pueden abrir el menú contextual del control (clic derecho) y seleccionar "Cancel zoom". 

Para modificar el texto de esta opción, basta con utilizar [AAFChangeCaption](../WDLang1/1000022100.md) con la constante *aafZoom*.

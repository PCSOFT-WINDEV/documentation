
## Definir el foco inicial en un control (AAF)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000084"></a>
El menú contextual de los controles Campo de entrada incluye la opción "Poner foco inicial en este control". Esta opción permite definir cuál control tendrá el foco inicial cuando se abra la ventana.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000108"></a>


### Implementación
<a name="implementacion_ELTPARAGRAPHE000019"></a>

La opción "Poner foco inicial en este control" es una opción predeterminada en los Controles campo de entrada. 

Al igual que todas las opciones incluidas en las AAF, esta opción se puede configurar. Para obtener más información, consulte [Configurar los menús de las AAF](../Editeurs/2010040.md). 

Mediante programación: 

- utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aaf1stFocusControl* para eliminar esta opción del menú contextual. 

- utilice la función [AAFChangeCaption](../WDLang1/1000022100.md) para cambiar el texto de la opción.

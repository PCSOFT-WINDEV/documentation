


## Ventanas no maximizadas para Android
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000149"></a>
Una ventana no maximizada es una ventana que ocupa sólo una sección de la pantalla.

También se puede recortar una ventana no maximizada (ventana no rectangular).

Las ventanas no maximizadas pueden crearse a través de la asistente para la creación de ventanas. Para iniciar esta asistente: 

- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

- La ventana de creación de nuevos elementos se abre: haga clic en "Ventana", y luego en "Ventana".

- El asistente de creación de ventanas se abre. 

- Seleccione "Asistente".. 

- En la asistente, seleccione "No maximizado".




La mayoría de las veces, las ventanas no maximizadas se utilizan para mostrar mensajes de información.

La asistente solicita la siguiente información para la creación de ventanas: 

- [el tamaño de la ventana cuando se abre](#NOTE2_1). 

- [los botones que se insertarán en la ventana](#NOTE2_2). 

- el nombre de la ventana.

## Características de las ventanas no maximizadas
<a name="caracteristicas_las_ventanas_maximizadas_ELTTEXTE000173"></a>


### Tamaño de la ventana
<a name="tamano_ventana_ELTPARAGRAPHE000046"></a>

El tamaño de la ventana cuando se abre puede modificarse en la pestaña "Detalles" de la ventana description. 

Por Default, la ventana está centrada en relación con la pantalla. 
Observaciones: Posicionamiento de ventanas no maximizadas: 

- La posición de las ventanas no maximizadas puede cambiarse con el uso de [WinSize](../WDLang1/3038020.md). 

- Puede seleccionar la posición de apertura "relativa a la pantalla" (pestaña "UI" de la descripción de la ventana).



<a name="NOTE2_2"></a>


### Botones que se mostrarán en la ventana
<a name="botones_que_mostraran_ventana_ELTPARAGRAPHE000064"></a>

Cuando el asistente crea una ventana no maximizada para la creación de ventanas, tiene la posibilidad de elegir los botones que se insertarán en esta ventana: 

- OK y Cancel, 

- Botón de Reintentar y Botón de Abandonar, 

- Botón de ignorar y botón de cerrar, 

- Botón de salida.




En este caso, los botones seleccionados (y su código) se añaden automáticamente a la ventana. En función del tipo de botón solicitado, el código de clic del botón devuelve un booleano (Sí para el botón "OK", No para el botón "Abandonar" o "Cancel"). Este valor de retorno puede ser procesado por [OpenMobileWindow](../WDLang1/1000021018.md) (o por [OpenChild](../WDLang1/3038021.md)). Por ejemplo:


```wl
IF OpenMobileWindow(WIN_Non_Max) = Yes THEN Info("You've clicked OK")
```

<a name="NOTE2_3"></a>


### Ventana recortada
<a name="ventana_recortada_ELTPARAGRAPHE000090"></a>

Se puede recortar una ventana no maximizada (ventana no rectangular). Para ello, utilice un Image en formato PNG. 

Para recortar una ventana no maximizada:

- En la pestaña "Estilo" de la ventana description, seleccione el fondo "Transparent" color.

- En la pestaña "Imagen" de la ventana description, seleccione: 

	- un fondo Image en formato PNG, 

	- el modo de transparencia "Según Image ".
			Observación: La transparencia de Image se apoya en el formato PNG.




## Manipular un control Campo de entrada mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000296"></a>
Los controles Campo de entrada se pueden manipular mediante programación. Para ello, utilice la variable que corresponde al control Campo de entrada en el código.

Esta variable es el nombre del control. El tipo de variable depende del tipo del control Campo de entrada (numérico, texto, monetario, etc.).
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Caso especial: [Manipular un control Campo de entrada RTF mediante programación](../WDChamp/1013190.md)

## Mostrar información en un control Campo de entrada
<a name="mostrar_informacion_control_campo_entrada_ELTTEXTE000320"></a>
Para mostrar información en un control Campo de entrada, simplemente asigne un valor.

Por ejemplo:

- asignación directa:
	
	```wl
	EDT_CustomerLastName = "Doe"
	```


- asignación utilizando una variable:
	
	```wl
	EDT_LASTNAME = Customer_LastName
	```


- asignación mediante un cálculo:
	
	```wl
	EDT_LASTNAME = NoSpace(CUSTOMER.LASTNAME) + TAB + NoSpace(CUSTOMER.FIRSTNAME)
	```



**Observación**: Si el control está asociado con un campo de un archivo de datos, la asignación se realiza automáticamente mediante: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la función [FileToScreen](../WDLang4/3044210.md).


También puede utilizar la siguiente notación: CustomerLastName = CUSTOMER.LASTNAME. Para obtener más información, consulte [Enlazar un control Campo de entrada a un campo de una fuente de datos](../WDChamp/1013159.md).

## Obtener el valor introducido en un control Campo de entrada
<a name="obtener_valor_introducido_control_campo_entrada_ELTTEXTE000344"></a>
Para obtener el contenido de un control Campo de entrada:

- realice una asignación simple. 
	Por ejemplo:
	
	```wl
	CurrentEditContent = EDT_EDIT1
	```


- extraiga parte de la información. 
	Por ejemplo:
	
	```wl
	Initial = Left(EDT_LastName, 1) 
	Initial = EDT_LastName[[1]]
	```


- utilice la propiedad [Value](../Proprietes/2510130.md)




**Observación**: Si el control está enlazado a un campo de un archivo de datos, el valor introducido en el control se actualiza automáticamente en la fuente de datos utilizando: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la función [ScreenToFile](../WDLang4/3044146.md).


También puede utilizar la siguiente notación: CUSTOMER.LASTNAME = Edit1. Para obtener más información, consulte [Enlazar un control Campo de entrada a un campo de una fuente de datos](../WDChamp/1013159.md).

## Propiedades específicas de los controles Campo de entrada
<a name="propiedades_especificas_los_controles_campo_entrada_ELTTEXTE000368"></a>
Las siguientes propiedades permiten definir las características de los controles Campo de entrada.


|   |   |
| --- | --- |
| **Cualquier tipo de control Campo de entrada** |
| [HorizontalAlignment](../Proprietes/2510023.md) | Obtiene y establece la alineación horizontal de un control Campo de entrada. |
| [VerticalAlignment](../Proprietes/2510054.md) | Obtiene y establece la alineación vertical de un control Campo de entrada. |
| [Cursor](../Proprietes/2510025.md) | Obtiene y establece la posición del cursor en un control Campo de entrada. |
| [DndTarget](../Proprietes/2510016.md) | Obtiene y establece el comportamiento del control de destino durante una acción de arrastrar y soltar. |
| [DndSource](../Proprietes/2510014.md) | Obtiene y establece el comportamiento del control de origen durante una acción de arrastrar y soltar. |
| [Ellipsis](../Proprietes/2510051.md) | Obtiene y establece el tipo de elipsis en un control Campo de entrada. |
| [CursorEnd](../Proprietes/2510047.md) | Obtiene y establece la posición final de una selección realizada con el cursor. |
| [InputMask](../Proprietes/2510033.md) | Obtiene y establece la máscara de entrada. |
| [LeftIndent](../Proprietes/2513018.md) | Obtiene y establece la sangría izquierda del texto en un control Campo de entrada. |
| [RichEdit](../Proprietes/2510108.md) | Determina si un control es compatible con RTF, y activa o desactiva la compatibilidad RTF para ese control. |
| [SelectedText](../Proprietes/1000017193.md) | Obtiene y cambia el texto seleccionado en un control Campo de entrada. |
| [InputType](../Proprietes/2510136.md) | Obtiene el tipo de control Campo de entrada. |
| **Controles Campo de entrada de tipo Texto únicamente** |
| [DeleteButton](../Proprietes/1000021924.md) | Determina si el control contiene o no un botón para eliminar elementos. |
| [AutomaticLink](../Proprietes/2513017.md) | Obtiene y establece el modo de detección de enlaces en los controles Campo de entrada multilínea. |
| [Password](../Proprietes/2510086.md) | Determina si un control Campo de entrada es de tipo "Contraseña". |
| [Multiline](../Proprietes/2510105.md) | Determina si un control Campo de entrada soporta varias líneas de texto. |
| [AutoLineWrap](../Proprietes/1000021868.md) | Determina si un control Campo de entrada multilínea admite el ajuste de línea automático. |
| **Controles Campo de entrada de tipo Monetario + Euro únicamente** |
| [DisplayCurrency](../Proprietes/2510030.md) | Obtiene y establece divisa utilizada al mostrar e introducir un valor en un control Campo de entrada de tipo "Monetario + Euro". |
| [MemoryCurrency](../Proprietes/2510013.md) | Obtiene y establece la divisa utilizada al manipular el control Campo de entrada de tipo "Monetario + Euro" mediante programación. |
| **Controles Campo de entrada de tipo Numérico y Duración** |
| [MaxValue](../Proprietes/2510009.md) | Obtiene y establece el límite superior de los valores en un control Campo de entrada. |
| [MinValue](../Proprietes/2510008.md) | Obtiene y establece el límite inferior de los valores en un control Campo de entrada. |

Para obtener la lista completa de las propiedades WLanguage que pueden utilizarse con un control Campo de entrada, consulte [Propiedades asociadas al control Campo de entrada](../WDChamp/1013230.md).

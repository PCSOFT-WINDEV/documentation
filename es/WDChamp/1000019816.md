
## Organigrama de relacionado a un archivo de datos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000085"></a>
Tiene la posibilidad de crear controles de organigrama directamente relacionado a los datos.. Puede crear un organigrama **Browsing control**: el control Organigrama será relacionado a un archivo de datos o a una consulta. En este caso, el archivo de datos contiene la información sobre los elementos mostrados en el control. 





## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000109"></a>
Para crear una control Organigrama de navegación, debe modificar la descripción de la control Organigrama.

**Para modificar las características de un control Organigrama existente:**

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Contenido" seleccione "Archivo/Consulta" y seleccione la fuente de datos utilizada.. Esta fuente de datos puede ser: 

	- un archivo de datos definido en el análisis,

	- una consulta del proyecto, 

	- una consulta incluida en la ventana. 




3. Especificar los diferentes elementos a tener en cuenta para construir la estructura de árbol del elemento en el organigrama: 

	- la posición de búsqueda

	- el identificador del progenitor (que se encuentra en la fuente Record) y el elemento al que es relacionado. 




4. Asociar los elementos de la fuente de datos con las características del elemento. Los elementos precedidos de una estrella son obligatorios. 

5. Validar la ventana de description de la control Organigrama. El control Organigrama es un organigrama de navegación. 








## Manejo de una control Organigrama basada en un archivo de datos
<a name="manejo_una_control_organigrama_basada_archivo_datos_ELTTEXTE000133"></a>
Una control Organigrama de navegación puede ser fácilmente manejada. El usuario puede manipular los elementos a través de una ventana específica. Todo lo que tienes que hacer es modificar el archivo de datos relacionado al organigrama. Los datos visualizados se actualizarán automáticamente. También es posible utilizar [OrgDisplay](../WDLang1/1000019764.md) para actualización los datos en el control.


## Manejar una control Estático de forma programada
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000188"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten manipular a control Estático programáticamente. Para hacerlo, usa la Variable de la control Estático en el código.

La Variable de la control Estático corresponde al nombre de la estática. Un control Estático siempre es un control de texto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Inicialización de una control Estático
<a name="inicializacion_una_control_estatico_ELTTEXTE000212"></a>
Para inicializar una control Estático, todo lo que tiene que hacer es asignar la variable de la control Estático.

Por ejemplo: 


```wl
// Assign the last and first names read in the Customer data file to a Static control
STC_MyStatic = Customer.LastName + "  " + Customer.FirstName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Recuperar el valor de una control Estático
<a name="recuperar_valor_una_control_estatico_ELTTEXTE000236"></a>
Para recuperar el contenido de un control Estático:

- realizar una asignación simple. Por ejemplo:
	
	```wl
	CurrentStaticContent = STC_STATIC1
	```


- usar la control Estático directamente. Por ejemplo:
	
	```wl
	Info(STC_STATIC1)
	```


- utilizar las propiedades [Valor](../Proprietes/2510130.md) o [Pie de foto](../Proprietes/2510053.md).

## Propiedades específicas para la gestión de un control Estático
<a name="propiedades_especificas_para_gestion_control_estatico_ELTTEXTE000284"></a>
Las siguientes propiedades se utilizan para gestionar las características de una control Estático a través de la programación.


|   |   |
| --- | --- |
| [Animación](../Proprietes/2510006.md) | Permite conocer el estado actual de la animación en un control Estático e iniciar/parar una animación en un control Estático animado. |
| [InitialAnimation](../Proprietes/2510012.md) | Permite conocer el estado inicial de la animación en un control Estático. |
| [Elipsis](../Proprietes/2510051.md) | Permite conocer y modificar el modo de gestión de las elipsis en una control Estático. |
| [Pie de foto](../Proprietes/2510053.md) | Se utiliza para averiguar y modificar la leyenda de la control. |
| [FontMinSize](../Proprietes/1000024918.md) | Permite conocer y modificar el tamaño mínimo de la fuente utilizada en una control Estático con la opción "Reducir fuente".. |
| [BottomIndent](../Proprietes/1000022891.md) | Permite conocer y modificar el margen inferior de una control Estático. |
| [RightIndent](../Proprietes/1000021110.md) | Permite conocer y modificar el margen derecho de una control Estático. |
| [LeftIndent](../Proprietes/2513018.md) | Permite conocer y modificar el margen izquierdo de un control Estático. |
| [TopIndent](../Proprietes/1000022890.md) | Permite conocer y modificar el margen superior de un control Estático. |
| [RichEdit](../Proprietes/2510108.md) | Permite conocer y modificar la característica RTF (Rich Text Format) de un control Estático. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Estático, consulte [Propiedades associated con un control Estático](../WDChamp/1013002.md).

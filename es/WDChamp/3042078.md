
## Cruce de líneas en un gráfico
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000178"></a>
En un gráfico Line, puede ser difícil leer el valor de los ejes para un punto que está lejos de estos ejes.

A través de la nueva retícula incluida en todos los gráficos, el usuario final conoce con precisión el valor punteado.

Todos los gráficos con ejes se benefician automáticamente de la retícula.

El usuario pasa el ratón por encima de un punto del gráfico y se muestra el punto de mira!
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD%20-%20Graphe%20Mire.jpg)


Si la cruz vertical cruza varias líneas, el valor se muestra para cada intersección.: es el punto de mira múltiple. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD%20-%20Graphe%20Mire%20multiple.jpg)


Para un retículo múltiple, el encabezado del tooltip se puede personalizar.

## Implementación
<a name="implementacion_ELTTEXTE000202"></a>


### Punto de mira en un control Gráfico
<a name="punto_mira_control_grafico_ELTPARAGRAPHE000041"></a>

El punto de mira está disponible en Default para todos los controles de la carta creados a partir de la versión 20. 

Para mostrar una cruz en una control Gráfico: 

1. Abre la ventana de control Gráfico description.

2. En la pestaña "Detalles: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en la zona "Zoom y mira", marque "Activar mira del gráfico". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Mire_Graphe_Editeur.gif)
 La opción "Mira múltiple (líneas y columnas apiladas)" permite utilizar cruces múltiples en los gráficos Line o de columnas. 




3. Valide la ventana de descripción del control. 




**Observaciones**: 

- El punto de mira está disponible para las siguientes clases de gráficos: Área, Burbuja, Line, Columna, MinMax, Radar y Cascada.

### Cruceta en los AAF (Características de la aplicación automatic)
<a name="cruceta_los_aaf_caracteristicas_aplicacion_automatic_ELTPARAGRAPHE000090"></a>

Si la retícula no está activada en un gráfico, el usuario puede activarla a través del menú contextual del gráfico ("Activar la retícula"). 
<a name="NOTE2_3"></a>


### Modificación de la retícula múltiple mediante programación
<a name="modificacion_reticula_multiple_mediante_programacion_ELTPARAGRAPHE000097"></a>

Por Default, el múltiple retículo muestra: 

- en la cabecera, el nombre de la categoría. 

- el contenido de los diferentes tooltips para los puntos correspondientes a la retícula. 




Es posible modificar el encabezado del tooltip de la mira mediante la función [grTooltip](../WDLang3/3042010.md) con la constante *grTooltipFormatCrosshairHeader*. 

Por ejemplo: 


```wl
grTooltip(CHART_Chart1, grTooltipFormatCrosshairHeader, "Category: [%CATEGORY%]")
```

<a name="NOTE2_4"></a>


### Configuración de las opciones de la retícula a través de la programación
<a name="configuracion_las_opciones_reticula_traves_programacion_ELTPARAGRAPHE000116"></a>

El retículo tiene las siguientes características de Default: 

- color basado en el color de cuadrículas en el control Gráfico. 

- espesor: 1 píxel. 

- opacidad: retícula totalmente opaca. 




Estas características pueden ser modificadas por las siguientes propiedades:  



|   |   |
| --- | --- |
| [CrosshairColor](../Proprietes/1000021505.md) | El **CrosshairColor** Property se utiliza para get y cambiar el Color de la retícula en un control Gráfico. |
| [CrosshairOpacity](../Proprietes/1000021517.md) | La función **CrosshairOpacity** Property se utiliza para determinar y cambiar la opacidad de la retícula en un control Gráfico. |
| [CrosshairThickness](../Proprietes/1000021506.md) | El **CrosshairThickness** Property se utiliza para determinar y cambiar el grosor de la retícula en un control Gráfico. |

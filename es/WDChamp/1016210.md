## Presentación
<a name="presentacion_ELTTEXTE000171"></a>
La control Cajón de navegación puede contener otros controles y está pensada para facilitar el diseño de la UI. Se puede ampliar para mostrar (o no) los controles integrados.. La operación de colapso/expansión de control puede hacerse:

- haciendo clic en el título control,

- mediante programación.

### Creando un "Cajón" control en una ventana
<a name="creando_cajon_control_una_ventana_ELTPARAGRAPHE000026"></a>

Para crear un "Cajón" control en una ventana: 

1. En la pestaña "Creación", en el grupo "Contenedores", despliegue "Pestaña y asociados" y seleccione "Control Cajón de navegación".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor. 
	El control Cajón de navegación creado contiene automáticamente un control: este control es el botón usado para "Expandir/Colapsar" el control Cajón de navegación. Esta control contiene el código WLanguage necesario. Tienes la posibilidad de personalizar esta control asociándola con una Image específica.

3. Deje caer sobre la control Cajón de navegación los diferentes controles que deben aparecer/desaparecer cuando se utiliza.


Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Características de una control Cajón de navegación
<a name="caracteristicas_una_control_cajon_navegacion_ELTTEXTE000207"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### control Cajón de navegación encontrada en una ventana
<a name="control_cajon_navegacion_encontrada_una_ventana_ELTPARAGRAPHE000086"></a>

- Cuando el control Cajón de navegación está en edición en el editor de ventanas, se puede manejar como cuando está en ejecución.. El menú emergente de la control le permite expandirla/colapsarla a petición (opción "Colapsar/Expandir la control Cajón de navegación").

- En la description de la control Cajón de navegación, la pestaña "General" se utiliza para especificar:

	- Si se debe realizar una animación al manejar un control Cajón de navegación.

	- Si el tamaño de la ventana debe ser adaptado cuando se usa la control. De hecho, cuando se expande el control Cajón de navegación, los controles que se encuentran debajo del control Cajón de navegación se mueven automáticamente hacia abajo.. Esta opción se utiliza para ampliar automáticamente la ventana hacia abajo para ver los controles que se encuentran debajo de la control Cajón de navegación.

## Programación de una control Cajón de navegación
<a name="programacion_una_control_cajon_navegacion_ELTTEXTE000237"></a>


### Expansión/Colapso de una control Cajón de navegación
<a name="expansioncolapso_una_control_cajon_navegacion_ELTPARAGRAPHE000114"></a>Para contraer una control Cajón de navegación, utilice la siguiente sintaxis:


```txt
<Name of Drawer control> = False
```


Para desglosar una control Cajón de navegación, utilice la siguiente sintaxis:


```txt
<Name of Drawer control> = True
```

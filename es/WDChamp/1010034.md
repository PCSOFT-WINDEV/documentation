
## Eventos associated con las celdas de las barras de estado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000090"></a>
WINDEV gestiona los siguientes eventos de forma predeterminada (en orden de aparición en el editor de código):

| Evento | Condición de ejecución |
| --- | --- |
| Inicialización | Ejecutar al abrir la ventana.<br><br>Este código se ejecuta después de la inicialización Process de todos los controles encontrados en la ventana actual. |
| partida | Ejecutar cuando el cursor de edición entra en la celda (sólo celdas editables). |
| salida | Ejecutar cuando el cursor de edición salga de la celda (tecla TAB, clic en otra celda, etc.). |
| Siempre que se modifique | Ejecutar siempre que el contenido de la celda sea modificado por el usuario (escribiendo o borrando una letra por ejemplo). |
| Haga clic | Ejecutar durante un clic del ratón en la celda de la barra de estado. |
| Cada vez que se actualiza | Ejecutar regularmente para actualizar la celda (sólo celdas"&lt;Caps Lock&gt;"). |

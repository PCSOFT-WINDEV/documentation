


## Control TreeMap
			





## Presentación
<a name="presentacion_ELTTEXTE000137"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000008"></a>

El control Treemap permite representar dinámicamente datos, posiblemente basados en estructura de árbol, de manera ponderada. 

Para llenar esta control, todo lo que tienes que hacer es especificar: 

- el nombre del elemento. 

- posiblemente, el árbol ruta del elemento. 

- el peso del elemento. 




Este tipo de control es muy useful para representar volúmenes de datos de forma intuitiva. 



## Creación de una control TreeMap
<a name="creacion_una_control_treemap_ELTTEXTE000161"></a>
Para crear una control TreeMap: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", haga clic en "TreeMap".

2. Haga clic en la ubicación en la que desea crear el control. El control se crea automáticamente. 




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción en el menú contextual. El control TreeMap puede manejarse mediante programación a través del [Funciones de TreeMap](../WDLang1/1000018995.md) (o del [Funciones TreeMap (sintaxis de prefijo)](../WDLang1/1000019801.md)).



## Características de la control TreeMap
<a name="caracteristicas_control_treemap_ELTTEXTE000189"></a>


### Control de la ventana de description
<a name="control_ventana_description_ELTPARAGRAPHE000051"></a>

La ventana de description de control TreeMap se utiliza para introducir las características de visualización del control. 

La pestaña "General" sirve para seleccionar: 

- el modo de visualización de la leyenda de los elementos de control: leyenda sobre los elementos o sobre los grupos de elementos. 

- el contenido del epígrafe de información para el coeficiente corrector. 




La pestaña "UI" se utiliza para autorizar la selección en el control (esta opción es propuesta por Default). 

La pestaña "Estilo" se utiliza para definir las opciones de fuente, las opciones de Color y otras opciones relativas a los diferentes elementos del TreeMap. 


### Vocabulario associated con el control TreeMap
<a name="vocabulario_associated_con_control_treemap_ELTPARAGRAPHE000065"></a>

Para manipular a control TreeMap, usted debe estar familiarizado con el siguiente vocabulario: 

| denominar | Definición |
| --- | --- |
| Elemento | Elemento visualizado en la control TreeMap. |
| Elemento ruta | La ruta de un elemento corresponde a la ruta y a la leyenda del elemento que se muestra en el control TreeMap. El uso de un ruta al añadir el elemento en el control TreeMap le permite organizar los elementos y agruparlos. |
| Grupo de elementos | set de elementos que utilizan una ruta común. |
| Ponderación | Peso del elemento mostrado en la pantalla control TreeMap. Esta ponderación se puede visualizar (o no) en la pantalla control TreeMap. Esta ponderación se utiliza para representar la importancia del elemento en relación con otros elementos que se encuentran en el control TreeMap. |

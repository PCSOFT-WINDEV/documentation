
## Eventos associated con un banner deslizante control
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Eventos de Default
<a name="eventos_default_ELTTEXTE000099"></a>
WEBDEV gestiona los siguientes eventos de forma predeterminada (orden de aparición en el editor de código):

| **Event** | **Condición de ejecución** |
| --- | --- |
| Inicialización (código servidor) | Se ejecuta al abrir la página. |
| Siempre que cambie de plano (código de navegador) | Se ejecuta cada vez que se cambia de plano (mediante programación, bala o botón). |
| Siempre que cambie de plano (código de servidor) | Se ejecuta cada vez que se cambia de plano (mediante programación, bala o botón). |
| Regreso de AJAX Process después de cada cambio de avión (código de navegador) | Corre cuando el Ajax Process en el servidor esté completo. |
| Clic (código navegador) | Corre durante un clic en el Sliding Banner control. <br><br>Observación: Este evento solo está disponible si se seleccionó la opción "Activar clic en el fondo del banner" (opción disponible en la ventana de descripción del control). |
| Haga clic (código de servidor) | Corre durante un clic en el Sliding Banner control. <br><br>Observación: Este evento solo está disponible si se seleccionó la opción "Activar clic en el fondo del banner" (opción disponible en la ventana de descripción del control). |

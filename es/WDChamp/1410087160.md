## Presentación del control Editor de diagramas
<a name="presentacion_del_control_editor_diagramas_ELTTEXTE000302"></a>
El control Editor de diagramas permite a los usuarios crear y editar diagramas en sus aplicaciones y sitios. Este control está basado en Chromium Embedded Framework (CEF). 

Este control puede utilizarse para representar estructuras, arquitecturas, infraestructuras materiales, planos, procesos, mecanismos, diagramas de flujo no convencionales, etc.

El control Editor de diagramas incluye varias características: 

- formas predefinidas o personalizadas,

- cambio de tamaño de las formas,

- opciones de estilo de las formas: línea, color, etc.

- conectores avanzados entre formas,

- entrada de texto, 

- ...


Estas características pueden ser definidas directamente por el usuario final o mediante programación.

La cinta de opciones de este control permite realizar las acciones más comunes. Para obtener más información, consulte el control Editor de diagramas: Opciones de la cinta y de las pestañas.

También se pueden utilizar varios métodos abreviados de teclado:

## Crear un control Editor de diagramas
<a name="crear_control_editor_diagramas_ELTTEXTE000326"></a>
**Para crear un control Editor de diagramas**:

1. Seleccione la siguiente opción, según el IDE utilizado: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Ofimática" y haga clic en "Editor de diagramas". 




2. Haga clic en la ventana o página en la ubicación en la que desea crear el control.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

### Ventana de descripción
<a name="ventana_descripcion_ELTPARAGRAPHE000172"></a>

La pestaña "General" de la ventana de descripción del control Editor de diagramas permite definir las siguientes opciones para las barras de herramientas y los paneles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0002.gif)


- **Mostrar barra de herramientas**: Muestra la barra de herramientas del control Editor de diagramas. En una aplicación WINDEV, esta barra de herramientas se muestra como una cinta de opciones. En un sitio WEBDEV, se muestra una barra pequeña. 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si se muestra una barra de herramientas, esta se puede definir en "VI fuente".
	Si la opción **"VI fuente"** corresponde a: 

	- "Ninguna": se utilizará la barra de herramientas predeterminada. 

	- "Ventana predefinida": la barra de herramientas predeterminada se puede personalizar.
			Solo tiene que seleccionar la ventana predefinida en "VI fuente". En este caso, la ventana "IW_WinDevAAF_DiagramEditorRibbon" se incluirá automáticamente en el proyecto. Esta ventana interna (disponible en inglés y francés) puede modificarse según sus necesidades. Esta ventana contiene todo el código necesario para el control Editor de diagramas. Le recomendamos comprobar los errores de UI y asegurarse de que la ventana funciona correctamente después de realizar cualquier cambio. 

	- Ventana interna del proyecto: Permite utilizar una ventana interna para incluir una barra de herramientas específica.
			Puede utilizar cualquier ventana interna del proyecto para crear la barra de herramientas.




- **Mostrar panel &lt;Biblioteca&gt;**: Muestra u oculta el panel "Biblioteca" con las bibliotecas predeterminadas y personalizadas del control Editor de diagramas.

La pestaña "General" de la ventana de descripción del control Editor de diagramas permite definir las siguientes opciones de visualización: 

- **Mostrar líneas de cuadrícula**: Muestra u oculta las líneas de cuadrícula del control Editor de diagramas. La propiedad WLanguage correspondiente es [GridlinesVisible](../Proprietes/1000023117.md). 

- **Bordes de página**: Muestra los bordes de página en el control Editor de diagramas. La propiedad WLanguage correspondiente es [PageBorderVisible](../Proprietes/1410088623.md). 

- **Vista 3D isométrica**: Muestra una cuadrícula isométrica para alinear objetos isométricos.

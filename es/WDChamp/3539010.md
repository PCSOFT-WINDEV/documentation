
## La tecnología AWP (Active WEBDEV Page)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000094"></a>
PC SOFT ofrece una nueva tecnología: **Active Páginas WEBDEV ( AWP )** .

Este método AWP (Active WEBDEV Page), dedicado a INTERNET, permite gestionar fácilmente los Page dinámicos autónomos, sin contexto. Esto permite crear una Page cuyo contenido depende de una base de datos pero no del contenido de otras Page s.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ventajas y limitaciones de la tecnología AWP
<a name="ventajas_limitaciones_tecnologia_awp_ELTTEXTE000118"></a>


### ATENCIÓN: El modo AWP está reservado a los desarrolladores que ya están familiarizados con WEBDEV.
<a name="atencion_modo_awp_esta_reservado_los_desarrolladores_que_estan_familiarizados_con_webdev_ELTPARAGRAPHE000020"></a>

Se requiere un buen conocimiento de WEBDEV (incluyendo los conceptos de sesiones y contextos) para utilizar este modo avanzado.

De hecho, los contextos deben gestionarse mediante la programación en este modo.

Algunas características específicas de los contextos de automatic pueden no funcionar más en este modo.
<a name="NOTE2_2"></a>


### Las ventajas del modo AWP son las siguientes:
<a name="las_ventajas_del_modo_awp_son_las_siguientes_ELTPARAGRAPHE000031"></a>

- no hay consumo de recursos en el servidor, 

- fácil creación de sitios dinámicos simples, 

- mejor referenciación de las páginas de Active WEBDEV del sitio, 

- persistencia de las páginas (no hay tiempo de espera), 

- actualización simplificada: cuando se actualiza ad hoc o cuando se añade una Page, cada Page puede cargarse por separado sin tener que bloquear a los usuarios de la Web.




**Los inconvenientes son los siguientes:**

- los contextos deben ser gestionados mediante programación (la gestión Default de los contextos WEBDEV puede ser utilizada simultáneamente en el mismo sitio), 

- no hay posicionamiento de automatic en los archivos de datos, 

- programación específica para generar los enlaces de referencia, 

- algunas funciones de WEBDEV no están disponibles, 

- ...




**Recordatorio**: el modo AWP está reservado a los desarrolladores que están familiarizados con WEBDEV.

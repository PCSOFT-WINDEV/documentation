## Presentación
<a name="presentacion_ELTTEXTE000203"></a>
La editor de reportes permite crear informes "Scheduler".. 

En este caso, la reporte sólo contiene un control Planificador. Este tipo de control se crea al crear un reporte programador: no puede crearse independientemente de reporte. 

Ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_Planning.gif&type=thumb)


En el editor, este control incluye dos bloques: 

- El bloque "Cabecera" que muestra los títulos de las columnas que se encuentran en el programa. 

- El bloque"Programar" que muestra los recursos y las citas en un control Planificador.

## Creación de un planificador reporte
<a name="creacion_planificador_reporte_ELTTEXTE000227"></a>


### Crear un reporte
<a name="crear_reporte_ELTPARAGRAPHE000029"></a>

Para crear un "Planificador" reporte: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

2. La ventana de creación de nuevos elementos se abre: haga clic en "Reporte", y luego en "Reporte". 

3. El asistente de creación de reportes se abre. 

4. En esta asistente, especifique:

	- el tipo de reporte a crear ("Programador reporte") con el [información específica para la creación de un programador control](#NOTE2_2): 

		- el número de días a mostrar.

		- la granularidad de las líneas de grilla.

		- el modo de visualización de los recursos (recursos visualizados por filas o en columnas).  




- la modelo utilizada en caso necesario,

- la fuente de datos del reporte (archivo de datos, consulta, control Planificador, ...). Para obtener más información, consulte [Fuente de datos Reporte](../WDChamp/1011055.md).



<a name="NOTE2_2"></a>


### Información específica para la creación del Planificador
<a name="informacion_especifica_para_creacion_del_planificador_ELTPARAGRAPHE000062"></a>

La siguiente información es específica para la creación de la control Planificador:

- Número de días a mostrar en el control Planificador: El programa se puede imprimir: 

	- por un día,

	- durante una semana en 5 días,

	- durante una semana en 7 días,

	- durante un mes,

	- desde el primer Appointment hasta el último,

	- con un número personalizado de días. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ass_Creation_pln.gif)





- La granularidad de las líneas de cuadrícula. Esta granularidad se utiliza para definir el tamaño de la cuadrícula utilizada para mostrar las líneas de la cuadrícula de la control Planificador. Esta granularidad es set a 30 minutos por Default. Esta información debe ser modificada de acuerdo con el número de días a visualizar.. 
	Esta característica puede ser modificada en el código a través de la [GranularityGridlines](../Proprietes/1000020107.md) Property. 

- El modo de visualización de los recursos. Los recursos pueden mostrarse por filas o en columnas. 




**Observación**:  Si el Planificador reporte es relacionado a una fuente de datos (consulta, archivo de datos), debe especificar los elementos que se tendrán en cuenta para mostrar el horario y las citas. Se debe especificar la siguiente información: 

- el título de Appointment

- el recurso asociado con el Appointment

- las fechas de inicio y fin de Appointment

## Características de un programador reporte y control Planificador
<a name="caracteristicas_programador_reporte_control_planificador_ELTTEXTE000257"></a>


### Características del planificador (editor de reportes)
<a name="caracteristicas_del_planificador_editor_reportes_ELTPARAGRAPHE000094"></a>

Usted tiene la capacidad de modificar las siguientes características de un Programador reporte: 

- La fuente de datos del Programador reporte puede modificarse en la pestaña "Datos" de la ventana description del reporte ("Reporte description" en el menú contextual del reporte). 

- La pestaña "General" de la ventana de description de control Planificador se utiliza para especificar: 

	- el número de días que deben ser mostrados en la control. 

	- la granularidad de las líneas de grilla.

	- el período de tiempo de Visible. 

	- la visualización de las citas de todo el día. Si la opción "Mostrar citas de todo el día" está marcada, la reporte admite las citas para las que la Property "Todo el día" es set a verdadero (para más detalles, véase [Appointment Variable](../WDLang1/1000019244.md)). En este caso, un área específica utilizada para mostrar las citas diarias aparece en el programa.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Planning_journee_Etat.gif)
Observación: Asegúrese de que el área utilizada para mostrar las citas diarias tenga las dimensiones adecuadas en la pantalla de editor de reportes. Se puede cambiar el tamaño si es necesario.

	- la reporte personalizada para mostrar las citas. 

		- Si la opción seleccionada es "Ninguna", se utiliza la pantalla Default. 

		- Si se selecciona "Preset reporte", se añade automáticamente un preset reporte al proyecto.. Este reporte se denomina "RPT_WinDevViewAPT" por Default y se utiliza automáticamente para mostrar las citas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_WinDevvueRDV.gif)
 Este reporte se puede personalizar, renombrar, etc.

		- Si la opción seleccionada corresponde a un proyecto reporte, este reporte se utilizará para mostrar las citas..
						Le aconsejamos personalizar el reporte propuesto por Default.
						La misma reporte se puede utilizar para personalizar las citas en un Programador u Organizador reporte..  




- el modo de visualización de los recursos: en filas o columnas. Si los recursos se muestran por columnas, tiene la posibilidad de especificar el número de recursos mostrados por columnas. 

- la máscara de visualización de las fechas y horas. 

- Si el control Planificador es relacionado a una fuente de datos (archivo de datos, consulta, ...), la pestaña "Contenido" que se encuentra en la ventana description del control se usa para vincular los datos del Appointment con los datos que se encuentran en la fuente de datos. 

- La pestaña "Estilo" en la ventana de la description de la control Planificador se usa para especificar el estilo: 

	- para el título de los recursos.

	- para las citas que se muestran en el control Planificador.

### Funciones de WLanguage
<a name="funciones_wlanguage_ELTPARAGRAPHE000142"></a>Varias funciones de WLanguage están disponibles en editor de reportes para manejar los controles del Programador. Estas funciones son: 



|   |   |
| --- | --- |
| [Cita previa](../WDLang1/1000019273.md) | Añade una nueva Appointment a una control Planificador. |
| [Lista de programación de citas](../WDLang1/1000019276.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Planificador, <br><br>- todas las citas encontradas en un control Planificador,<br><br>- un Appointment específico (seleccionado o suspendido). <br><br><br> |
| [Modo de programación](../WDLang1/1000019992.md) | Devuelve el modo de visualización de un control Planificador: día, semana, mes, .... |
| [PlanificadorBorrarRecurso](../WDLang1/1000019282.md) | Borra una Resource de una control Planificador. |
| [PlanificadorFondoColor](../WDLang1/1000019997.md) | Define una Color de fondo para un periodo de tiempo en una control Planificador (periodo de tiempo correspondiente a la hora de comer, un día festivo, etc.). |
| [Posición del programador](../WDLang1/1000019372.md) | Muestra un control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que aparece en un control Planificador. |
| [SchedulerAddResource](../WDLang1/1000019279.md) | Añade un nuevo Resource en un control Planificador. |
| [SchedulerChangeMode](../WDLang1/1000019440.md) | Modifica el período (es decir, el número de días) que muestra un control Planificador. |
| [SchedulerDeleteAll](../WDLang1/1000019275.md) | Borra todas las citas y recursos de un control Planificador. |
| [SchedulerDeleteAppointment](../WDLang1/1000019274.md) | Borra una Appointment de una control Planificador. |
| [SchedulerPositionDateTime](../WDLang1/1000019998.md) | Muestra un control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que aparece en un control Planificador. |
| [SchedulerTimeDisplayed](../WDLang1/1000019373.md) | Define el intervalo de tiempo que muestra control Planificador en el modo Día o en el modo Semana.. |

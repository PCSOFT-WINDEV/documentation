## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
En la mayoría de los casos, un reporte es relacionado para un proyecto. Para poder extraer los datos de los archivos de datos, el editor de reportes debe conocer el description de los archivos de datos que está almacenado en el análisis, estando el análisis asociado a un proyecto.<br>![En este caso, un análisis WINDEV se comparte entre dos proyectos (WINDEV, WINDEV Mobile o WEBDEV).](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1-Comment-creer-un-etat.gif)


Para cargar un proyecto, en la pestaña "Inicio", en el grupo "General", despliegue "Abrir" y seleccione "Abrir un proyecto".

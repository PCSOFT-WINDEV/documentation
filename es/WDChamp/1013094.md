
## Manejo de los controles de los botones mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000275"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten programar los controles de los botones de manipular. Para hacerlo, usa la Variable de la control Botón en el código.

La Variable de la control Botón:

- corresponde al nombre de la control Botón.

- se inicializa con la leyenda de la control Botón.

## Recuperación y modificación de la leyenda de la control Botón
<a name="recuperacion_modificacion_leyenda_control_boton_ELTTEXTE000299"></a>


### Recuperación de la leyenda
<a name="recuperacion_leyenda_ELTPARAGRAPHE000023"></a>

Para recuperar el título de la control Botón, utilice la siguiente sintaxis:

- sintaxis directa: 
	
	```txt
	<Caption> = <Button control>
	```





- usando la propiedad [Pie de foto](../Proprietes/2510053.md): 
	
	```txt
	<Caption> = <Button control>.Caption
	```





 es una cadena de caracteres Variable.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: Si el subtítulo está en RTF ("RTF" marcado en la pestaña "General" del control description), el subtítulo recuperado contendrá las etiquetas RTF. Para get el título sin las etiquetas RTF, utilice [RTFToText](../WDLang1/3025032.md). 
Por ejemplo: 


```wl
MyCaption is string
MyCaption = RTFToText(BTN_Button.Caption)
```

<a name="NOTE2_2"></a>


### Cambiar título
<a name="cambiar_titulo_ELTPARAGRAPHE000057"></a>

Para modificar el título de la control Botón, utilice la siguiente sintaxis:


```txt
<Button control> = <Caption>
```


 es una cadena de caracteres Variable.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: Si el pie de foto está en RTF ("RTF" marcado en la pestaña "General" del control description), el nuevo pie de foto puede (o no) contener texto RTF.

## Averiguar y modificar el estado de un botón de encendido/apagado
<a name="averiguar_modificar_estado_boton_encendidoapagado_ELTTEXTE000329"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El [Empujado](../Proprietes/2510038.md) Property se utiliza para obtener el estado del control Botón:


```txt
<State> = <Button control>.Pushed
```


&lt;Estado&gt; es una variable de tipo Boolean: 

- <u><u><u><u>True</u></u></u></u> si el botón está presionado,

- <u><u><u><u>False</u></u></u></u> si el botón está liberado.

## Modificando la Image que aparece en el control Botón
<a name="modificando_image_que_aparece_control_boton_ELTTEXTE000389"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para cambiar la imagen que se muestra en la control Botón, utilice la [Image](../Proprietes/2510034.md) Property:


```txt
<Button control>.Image = <Image path>
```


 es una cadena de caracteres Variable

Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).




**Observación**: Para manejar las imágenes en formato Tiff, GIF y JPEG, Wxxximg.dll debe ser instalado con las otras DLL de WINDEV.

## Propiedades específicas para la gestión de imágenes en un control Botón
<a name="propiedades_especificas_para_gestion_imagenes_control_boton_ELTTEXTE000450"></a>
Las siguientes propiedades se utilizan para programar manipular las imágenes utilizadas en un control Botón.


|   |   |
| --- | --- |
| [BackgroundImage](../Proprietes/2510056.md) | Permite descubrir y modificar la imagen associated con el Border de un control Botón. |
| [BackgroundImageState](../Proprietes/2510057.md) | Permite conocer y modificar el número de dibujos que se encuentran en la imagen associated con el Border de un control Botón. |
| [Empujado](../Proprietes/2510038.md) | Permite conocer y modificar el estado de un control Botón encendido/apagado (presionado o no). |
| [Image](../Proprietes/2510034.md) | Permite conocer y modificar la Image de una control Botón. |
| [ImageState](../Proprietes/2510055.md) | Permite conocer y modificar el número de dibujos que se encuentran en la imagen associated con un control Botón. |
| [Valor](../Proprietes/2510130.md) | Permite conocer y modificar la leyenda de una control Botón. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con los controles Button, consulte [Propiedades de control Botón](../WDChamp/1013127.md).

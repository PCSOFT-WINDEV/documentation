## Las ventanas usadas por el visor de reporte
<a name="las_ventanas_usadas_por_visor_reporte_ELTTEXTE000163"></a>
Estas ventanas son las siguientes:



|   |   |
| --- | --- |
| <br>!["Ventana"Vista previa](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParam_apercu20.gif)<br>Esta ventana corresponde a la vista previa de impresión. |
| <br>!["Ventana"WD_Watermark](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParamFiligrane.gif)<br>Esta ventana se utiliza para gestionar las marcas de agua. |
| <br>!["Ventana"WD_Print_Reporte](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParamImpr1.gif)<br>Esta ventana se utiliza al imprimir un reporte de WINDEV desde la ventana de vista previa de impresión.<br>Esta ventana se utiliza para configurar:<br><br>- las páginas para imprimir.<br><br>- el número de copias a imprimir.<br><br>- el destino de impresión (impresora).<br><br>- los parámetros de la impresora, especialmente la orientación del papel.<br><br><br><br>Esta ventana también se utiliza para conocer el estado de la impresora y su tipo. | <br>!["Ventana"WD_Print](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParamImpr2.gif)<br>Esta ventana se utiliza durante una impresión realizada en WLanguage ([iEndPrinting](../WDLang5/3046053.md) llamada después de [iPreview](../WDLang5/3046069.md)). <br>Esta ventana se utiliza para configurar:<br><br>- las páginas para imprimir.<br><br>- el número de copias a imprimir.<br><br><br>Esta ventana también se utiliza para averiguarlo:<br><br>- el nombre de la impresora utilizada,<br><br>- el estado de la impresora utilizada<br><br>- el tipo de impresora utilizado.<br><br><br> |
| !["Ventana"WD_Wait](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenPatienceImpr.gif) |
| <br>!["ventana"WD_Install_ReportsAndQueries](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParam_EtatsR20.gif)<br>Esta ventana se abre cuando el usuario intenta modificar un reporte sin haber instalado "Reports and Queries". |
| <br>!["Ventana"WD_PopupSelColor](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FEN_POP_Couleur.gif)<br>Esta ventana permite seleccionar los colores utilizados para las anotaciones. | <br>!["Ventana"WD_PopupSelThickness](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FEN_POP_Epaisseur.gif)<br>Esta ventana permite seleccionar el grosor de la línea de las anotaciones. |
| <br>!["Ventana"WD_PopupReportGroup](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParam_format.gif)<br>Esta ventana permite seleccionar el tipo de reporte que se va a imprimir (para un grupo de reportes o de reportes con anclajes). | <br>!["Ventana"WD_PopupPrinter](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenParam_ListeImp.gif)<br>Esta ventana permite seleccionar la impresora que se va a utilizar. |
| <br>!["WD_PopupCrosshairOption"](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fenparam_mire.gif)<br>Esta ventana permite definir las opciones de la mira de la vista previa. |
| <br>!["Ventana"WD_SelPrinter](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fenparam_imp.gif)<br>Esta ventana permite mostrar el mensaje de la impresora. |

## Presentación
<a name="presentacion_ELTTEXTE000177"></a>
WINDEV y WEBDEV permiten imprimir automáticamente para una serie de impresoras PCL específicas.

**Si se selecciona una de estas impresoras**Los elementos específicos (líneas, imágenes, colores,...) se imprimirán de acuerdo con esta impresora.

**Si la impresora utilizada no se encuentra en esta lista**tiene la posibilidad de crear un archivo PCL genérico reconocido por la mayoría de las impresoras PCL.. En algunos casos, es posible que las líneas y las imágenes no se impriman correctamente.

Los comandos de impresión también se pueden enviar directamente a su impresora (desarrolle su propio controlador).. En este caso, no debe utilizar las funciones de impresión de WLanguage ni los informes: para imprimir, debe utilizar las funciones escritas por usted mismo.

Para más detalles, consulte la documentación suministrada con su impresora.

Ejemplo:


```wl
// "Proc_Printer" is a set of procedures that contains 
// the print code specific to a printer
// You must now write the WLanguage code that corresponds to each one of your printers
Proc_Printer.InitializeConnectionIR()
Proc_Printer.PrintLine("Title of the print.") 
Proc_Printer.SkipLine()
Proc_Printer.PrintWord("A section of sentence without line skip")
Proc_Printer.PrintWord(" in order for the rest to be printed on the same line.")
Proc_Printer.SkipPage()
Proc_Printer.PrintWord("Page 2 and End")
Proc_Printer.EndPrintout()
```

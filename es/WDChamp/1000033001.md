## Resumen de control Hoja de cálculo
<a name="resumen_control_hoja_calculo_ELTTEXTE000107"></a>
Las hojas de cálculo son una de las herramientas ofimáticas más utilizadas.

La "hoja de cálculo" control es un control nativo: no requiere la instalación de un software de hoja de cálculo en el equipo (y, por tanto, no hay que gestionar ninguna licencia).

Con este control, no más navegación entre las aplicaciones, no hay software externo a control!

Algunos beneficios de este control: 

- El contenido de control Hoja de cálculo se puede guardar (o cargar) directamente en formato Excel (formato XLSX). 

- El control Hoja de cálculo es compatible con Excel, OpenOfficeCalc, FreeOffice, ... 

- El control Hoja de cálculo es compatible con las hojas de cálculo de Excel (XLSX). 

- El control Hoja de cálculo admite archivos XLTX y XLTM (plantillas de Excel). Si se abre un archivo modelo en el control Hoja de cálculo, se crea automáticamente un nuevo libro de trabajo con ese modelo.  


Esta compatibilidad permite reutilizar las hojas de cálculo existentes o los cálculos realizados en el programa control Hoja de cálculo.

El control Hoja de cálculo también admite hojas de trabajo protegidas y celdas bloqueadas. La contraseña de la hoja de trabajo se solicita al realizar cambios.




## Manipular un control Barra lateral mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000172"></a>
WINDEV y WINDEV Mobile permiten manipular controles Barra lateral mediante programación. Para ello, utilice la variable del control Barra lateral en el código.

La variable del control Barra lateral corresponde al número de la pestaña activa. Esta variable es un número entero.

**Observación**: WINDEV y WINDEV Mobile gestionan automáticamente las transiciones cuando el usuario hace clic en las diferentes pestañas.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipular las pestañas de un control Barra lateral
<a name="manipular_las_pestanas_control_barra_lateral_ELTTEXTE000196"></a>
Para manipular una pestaña, utilice la siguiente sintaxis:


```txt
<Sidebar control>[Tab number].PropertyName = PropertyValue
```
Por ejemplo:


```wl
SIDE_MyBox[2].State = Grayed
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Cambiar la pestaña activa
<a name="cambiar_pestana_activa_ELTTEXTE000220"></a>
De forma predeterminada, la pestaña 1 (pestaña de la parte superior) está activada. La pestaña activa puede cambiarse mediante programación.

Para activar una pestaña de la barra lateral:

- Especifique el número de la pestaña a activar:
	
	```txt
	<Sidebar control> = <Tab number>
	```


- Utilice la propiedad [Value](../Proprietes/2510130.md).




El número se asigna en la descripción de las pestañas: en la lista de pestañas, la primera pestaña es la pestaña nº 1, la segunda pestaña es la pestaña nº 2, y así sucesivamente.

Observaciones: 

- Si la pestaña especificada no existe, la pestaña que se muestra no cambia.

- En la misma barra lateral, solo se puede mostrar una pestaña a la vez.

- Al cambiar a otra pestaña:

	- se ejecuta el código de salida del control con foco.

	- se activa el código de entrada del primer control editable de la pestaña.

## Obtener la pestaña activa
<a name="obtener_pestana_activa_ELTTEXTE000244"></a>
Para obtener el número de la pestaña activa:

- realice una asignación simple. Por ejemplo:
	
	```txt
	<Active pane number> = <Sidebar control>
	```


- utilice la propiedad [Value](../Proprietes/2510130.md).

## Funciones WLanguage para manipular controles Barra lateral
<a name="funciones_wlanguage_para_manipular_controles_barra_lateral_ELTTEXTE000268"></a>
Las siguientes funciones permiten manipular las características de los controles Barra lateral.


|   |   |
| --- | --- |
| [ControlClone](../WDLang1/3025017.md) | Clona un control asociándolo a una pestaña de un control Barra lateral. |
| [EnumControl](../WDLang1/3025005.md) | Devuelve el nombre de los controles asociados a un control Barra lateral. |
| [EnumSubElement](../WDLang1/3064002.md) | Devuelve la lista de pestañas de un control Barra lateral. |

## Propiedades específicas de las pestañas de un control Barra lateral
<a name="propiedades_especificas_las_pestanas_control_barra_lateral_ELTTEXTE000292"></a>
Las siguientes propiedades permiten gestionar las características de las pestañas de un control Barra lateral mediante programación.


|   |   |
| --- | --- |
| [Count](../Proprietes/2510097.md) | Obtiene el número de pestañas de un control Barra lateral. |
| [Image](../Proprietes/2510034.md) | Obtiene y establece la imagen de las pestañas de un control Barra lateral. <br>Para manipular la pestaña, utilice la siguiente sintaxis: &lt;Control Barra lateral&gt; [&lt;Índice de la pestaña&gt;]. |


Para obtener la lista completa de las propiedades WLanguage que pueden utilizarse con un control Barra lateral, consulte [Propiedades asociadas a una barra lateral](../WDChamp/1013111.md).


## Ratón magnético: moviendo automáticamente el cursor en el botón Default
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
El cursor del ratón puede posicionarse automáticamente en el botón Default cuando se abre la ventana.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PositionneSouris.GIF)


Esta característica es muy útil para las ventanas en las que el usuario hace clic rápidamente en el botón propuesto por Default.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000106"></a>
Para mover automáticamente el cursor del ratón sobre el botón de la Default cuando se abre la ventana:

1. Abra la ventana de descripción (opción "Descripción" del menú contextual).

2. Despliega la pestaña "Detalles" y selecciona "Mueve el cursor del ratón sobre el botón Default al abrir la ventana".

3. Valide la ventana de descripción.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Botón Default
<a name="boton_default_ELTTEXTE000130"></a>
El botón sobre el que se desplaza el cursor del ratón debe ser un botón "Validación".

Si la ventana contiene varios botones "Validación", el cursor del ratón se moverá al primer botón creado.




## HTML tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000198"></a>
El HTML tabla es un control usado para crear fácilmente diseños específicos en las páginas HTML. El HTML tabla no se puede utilizar para posicionar los controles pero permite presentar la información.

Por ejemplo, las tablas HTML permiten crear enumeraciones con bordes y columnas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML.gif)

## Creación de una tabla HTML
<a name="creacion_una_tabla_html_ELTTEXTE000222"></a>
**Para crear una tabla HTML**: 

- en la pestaña "Creación", en el grupo "Contenedores", haga clic en "Tabla HTML". En la asistente, seleccione las características de la tabla HTML para crear: 

	- Número de filas.

	- Número de columnas.

	- Estilo de tabla y alternancia de filas y columnas.




- en la pestaña "Creación", en el grupo "Contenedores", haga clic en "Tabla HTML" y seleccione directamente el número de filas y columnas de la tabla. En la asistente, seleccione las características de estilo para la tabla así como la alternancia de filas y columnas. 




Cuando se crea el HTML tabla en la página, puede hacerlo:

- moverlo con ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML_deplacement.gif).

- manipular su description a través del menú contextual de control. El menú emergente de ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML_deplacement.gif) permite manipular el tabla, el menú contextual de una celda permite manipular las celdas tabla.

## Usando un HTML tabla
<a name="usando_html_tabla_ELTTEXTE000246"></a>
En cada celda de una tabla HTML se puede agregar un control. La control y la célula tienen el mismo tamaño. La control y las celdas tienen las mismas propiedades Border y Background.

No se pueden incluir varios controles en la misma celda de una tabla HTML.

Sin embargo, una celda HTML que contenga varios controles puede incluirse en una celda de una tabla HTML.

## Manejo de HTML tabla
<a name="manejo_html_tabla_ELTTEXTE000270"></a>


### Seleccionar una celda
<a name="seleccionar_una_celda_ELTPARAGRAPHE000079"></a>

Para seleccionar una celda vacía, debe hacerlo:

1. Mover el cursor del ratón sobre la celda.

2. Haga clic dentro de la celda cuando el cursor del ratón se convierte en una flecha Black diagonal. Se selecciona la celda: aparecen marcas de sombreado en la celda.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML_Cellule_select.gif)



En este caso, la opción "description" que se encuentra en el el menú contextual del tabla le permitirá acceder al description de la célula.

Para seleccionar una célula associated con una control, todo lo que tienes que hacer es seleccionar la control.
En este caso, la opción "description" del menú contextual del tabla le permitirá acceder al description del control. 
Para averiguar la description de la llamada, seleccione "HTML tabla .. Descripción" de el menú contextual.
<a name="NOTE4_2"></a>


### Selección de una columna o de un fila
<a name="seleccion_una_columna_fila_ELTPARAGRAPHE000098"></a>

Para seleccionar una columna o un fila, debe hacerlo:

1. Posicione el cursor del ratón al final de la columna o fila.

2. Haz clic cuando el cursor del ratón se convierta en una flecha Black vertical (para una columna) o una flecha Black horizontal (para un fila). Se selecciona la columna fila o la columna: aparecen marcas de nacimiento.




El menú que aparece le permite hacerlo:

- dividir el elemento seleccionado vertical u horizontalmente

- fusionar las diferentes partes del elemento seleccionado.



<a name="NOTE4_3"></a>


### Borrar un HTML tabla
<a name="borrar_html_tabla_ELTPARAGRAPHE000113"></a>

Para eliminar una tabla HTML, seleccione la opción "Tabla HTML .. Eliminar tabla" en el menú contextual.

## Cambiar el tamaño de un archivo HTML tabla
<a name="cambiar_tamano_archivo_html_tabla_ELTTEXTE000306"></a>


### Modificar el número de celdas en el tabla
<a name="modificar_numero_celdas_tabla_ELTPARAGRAPHE000122"></a>

Para modificar el número de celdas en un HTML tabla, puede hacerlo:

- agregar filas o columnas una por una mediante el menú contextual (opción "Tabla HTML .. Insertar una fila" o "Tabla HTML .. Insertar una columna")

- añadir varias filas o columnas a través de el menú contextual ("HTML tabla .. Ampliar el tabla")



<a name="NOTE5_2"></a>


### Cambiar el tamaño de la tabla
<a name="cambiar_tamano_tabla_ELTPARAGRAPHE000132"></a>

El tamaño de la tabla puede ser modificado de acuerdo a dos métodos:

- Ampliando o reduciendo todas las filas/columnas de tabla proporcionalmente.: para ello, reduzca o amplíe la tabla utilizando la manipular en medio de la última fila o columna.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML_Agrandissement.gif)


- ampliando o reduciendo el tamaño de la última tabla fila/columna: para ello, reduzca o amplíe la tabla haciendo clic en el borde de la última fila o columna.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableau_HTML_Agrandis_col.gif)







## Edición del estilo de un HTML tabla
<a name="edicion_del_estilo_html_tabla_ELTTEXTE000336"></a>
Tiene la capacidad de editar por separado las características del Border y Background del tabla HTML y las características de cada una de las celdas que se encuentran en el tabla HTML. Las propiedades sobre el estilo de la Border y el estilo de la Background son las mismas que para los otros controles. 

El estilo de cada elemento que se encuentra en el HTML tabla también puede ser modificado. Para ello:

1. Abra la ventana HTML tabla description.

2. En la pestaña "Estilo", seleccione el elemento a personalizar en el cuadro combinado "Elemento".

3. Se muestran las propiedades que se pueden configurar para el elemento seleccionado.


**Observación**: La pestaña "Estilo" también puede ser usada para iniciar una asistente para crear un estilo automático para el HTML tabla. Todo lo que tienes que hacer es hacer clic en el enlace "Iniciar la asistente para la creación de estilos".

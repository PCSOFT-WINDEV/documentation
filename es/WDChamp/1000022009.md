
## Manejo de una Control Popup mediante programación (sintaxis de prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000063"></a>
Al ser una control de la Page, tiene la posibilidad de acceder a: 

- los controles Page de la ventana emergente. 

- los controles emergentes de Page. Por ejemplo: 
	
	```wl
	EDT_EditName = POPUP_MyPopup.EDT_Info_to_enter
	```
Las variables de la página son accesibles en los eventos associated con el Control Popup. Por ejemplo, el siguiente código se utiliza para mostrar una ventana emergente de formulario en el elemento seleccionado en un control Looper: 



```wl
// Browser click code on LOOP_Products
POPUP_FORM.IMG_PRODUCT = FolderWeb + "ui/empty.gif"
POPUP_FORM.Animate(CELL_PAGE, animScrollLeft, 25)
PRODUCTID = ATT_PRODUCTID
ExecuteProcess(POPUP_FORM.ZONE_HEADER_POPUP.BTN_Load, trtClick)
```




## Manejo de la Control Popup mediante programación
<a name="manejo_control_popup_mediante_programacion_ELTTEXTE000102"></a>


### Funciones disponibles
<a name="funciones_disponibles_ELTPARAGRAPHE000125"></a>Varias funciones son específicas para la gestión de los controles Popup: 



|   |   |
| --- | --- |
| [&lt;Popup&gt;.Animate](../WDLang2/1000021655.md) | Muestra una ventana emergente en una celda de página. |
| [PopupClose](../WDLang2/1000021656.md) | Oculta una ventana emergente que se muestra en la página a través de [&lt;Pantalla](../WDLang2/1000021654.md). |
| [&lt;Pantalla](../WDLang2/1000021654.md) | Muestra una ventana emergente en una Page con efecto OVI (Oscurecimiento de Ventanas Inactivas). |



Observaciones: 

- [&lt;Celda&gt;.DisplayDialog](../WDLang2/1000021652.md) y [&lt;Celda&gt;.CloseDialog](../WDLang2/1000021653.md) son compatibles con los controles Popup. Cuando los controles Cell se transforman en controles Popup, el código existente permanece compatible y funcional..  

- [&lt;Menú&gt;.AddPopup](../WDLang1/1000020602.md) se utiliza para transformar una opción de menú para que abra un Control Popup.

- Una Control Popup puede ser fijada por [&lt;Pantalla](../WDLang2/1000021654.md).

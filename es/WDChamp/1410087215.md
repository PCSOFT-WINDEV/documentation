
## Manipular controles Editor de imágenes para móviles mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
WINDEV Mobile permite manipular controles Editor de imágenes para móviles mediante programación. 

Puede manipular la imagen que se muestra en el control Editor de imágenes usando la variable que representa el control.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mostrar una imagen en un control Editor de imágenes
<a name="mostrar_una_imagen_control_editor_imagenes_ELTTEXTE000147"></a>
Para mostrar una imagen, solo tiene que asignarla al control Editor de imágenes. Puede utilizar un nombre de archivo o una variable de tipo [Image](../WDLang1/1000019650.md). 

Por ejemplo:

- asignación directa: 
	
	```wl
	IE_MyImage = "C:\Temp\MyImage.bmp"
	```


- asignación utilizando una variable:
	
	```wl
	ImagePath = fCurrentDir() + "\MyImage.bmp"
	IE_MyImage = ImagePath
	```


- asignación utilizando una variable de tipo [Image](../WDLang1/1000019650.md): 
	
	```wl
	MyImage is Image
	MyImage.Width = 500
	MyImage.Height = 100
	// Load from a file
	MyImage = dLoadImage("image.png")
	// Draw in the image
	dStartDrawing(MyImage)
	// Write a text
	dFont("Arial", 12, iNormal, 0)
	dText(4, 0, "Text in an Image control!")
	// End of drawing
	dEndDrawing(MyImage)
	
	// Assignment to a Mobile Image Editor control
	IE_MyImage = MyImage
	```






Observación: Un control Editor de imágenes para móviles también se puede inicializar a partir de una variable de tipo Buffer o Text que contenga el valor de la imagen en formato binario.

## Obtener la imagen que se muestra en el control Editor de imágenes
<a name="obtener_imagen_que_muestra_control_editor_imagenes_ELTTEXTE000171"></a>
La imagen que se muestra y manipula en el control Editor de imágenes puede obtenerse en una variable de tipo [Image](../WDLang1/1000019650.md). 

Por ejemplo: 

```wl
MyEditedImage is Image
MyEditedImage = IE_MyImage
```


Observación: La imagen obtenida solo contiene los cambios validados por el usuario. Los cambios que aún no se hayan validado serán ignorados en la variable [Image](../WDLang1/1000019650.md) devuelta. 

A continuación, la imagen puede guardarse con las funciones [dSaveImagePNG](../WDLang1/3029027.md), [AlbumSave](../WDLang3/1000020191.md), etc.

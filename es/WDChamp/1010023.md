
## Menú emergente del sistema
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000087"></a>
Los menús contextuales del sistema son gestionados por el sistema (Android, Windows, ...). Estos menús emergentes se muestran en algunos tipos de controles (ActiveX, ....).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Los menús contextuales del sistema Default
<a name="los_menus_contextuales_del_sistema_default_ELTTEXTE000111"></a>
Algunos tipos de controles pueden incluir un menú contextual del sistema. Este menú emergente puede corresponder a:

- ***el menú contextual mostrado por Windows***. Este es el caso de los menús emergentes de los controles ActiveX, multimedia o HTML:

	- a HTML control: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuCtxChpHTML.gif)
Observación: El menú contextual del sistema que se muestra es el de INTERNET Explorer. Depende del objeto que se seleccione en la página (Image, texto, enlace, etc.).

	- un ActiveX control: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuCtxActiveX.gif)
Observación: El menú contextual del sistema que se muestra depende del programa ActiveX asociado al control.




- ***el menú contextual de la AAF dirigida por WINDEV***.

	- una edición control: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuCtxChpSaisie.gif)


	- un control Tabla: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Table_MenuCtx.gif)
 Para más detalles, véase [Menú contextual de una control Tabla](../WDChamp/1013207.md).




El menú emergente del AAF puede configurarse (para mostrar sólo algunas opciones) y traducirse.. Para obtener más información, consulte [Configuración del menú contextual del AAF](../Editeurs/2010040.md).




## Visor de reportes: Pestaña "Vista previa"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000156"></a>


El panel "Vista previa" permite al usuario configurar el modo de visualización de la impresión en la vista previa actual.: Zoom, retícula, mover, mover entre páginas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Apercu_Volet_Apercu.gif&type=thumb)

## Las diferentes opciones
<a name="las_diferentes_opciones_ELTTEXTE000180"></a>


### Zoom
<a name="zoom_ELTPARAGRAPHE000029"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_Apercu_Zoom.gif)
El espectador de reporte propone: 

- ver la impresión actual en 100%. 

- adaptar el ancho de la impresión. La impresión se mostrará a todo lo ancho en el visor. 

- mostrar toda la página. 

- mostrar dos páginas una al lado de la otra. En este caso, si se ha definido una portada, al visualizar una al lado de la otra, la primera página se mostrará a la derecha.. 




Observación: Se puede realizar una gestión más precisa del zoom: 

- utilizando la herramienta "Lazo de zoom" en el grupo "Modo".. Esta opción permite hacer zoom en un área específica, definida por el lazo del ratón. 

- con las teclas "Zoom in" y "Zoom out" del grupo "Modo. Seleccione el modo de zoom deseado: un zoom se realizará automáticamente durante cada clic en la vista previa.  

- pulsando Ctrl + (signo'+' en el teclado numérico) y Ctrl - (signo'-' en el teclado numérico). 

- especificando el porcentaje de zoom directamente en la barra de menú del espectador (arriba a la derecha). 



<a name="NOTE2_2"></a>


### Cómo usar el visor de reporte
<a name="como_usar_visor_reporte_ELTPARAGRAPHE000051"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_Apercu_Mode.gif)
Las opciones disponibles son las siguientes: 

- La opción "Mover" se utiliza para moverse en la edición actual mientras se mantiene pulsado el botón izquierdo del ratón.. 

- La opción "Selección" se utiliza para seleccionar el texto en el visor de reporte a copy/paste. Para obtener más información, consulte [Copiar el texto en el visor de reporte](../WDChamp/1011059.md).

- La opción "Cruceta" se utiliza para mostrar un punto de mira. Este punto de mira hace que la lectura de las tablas con muchos datos sea mucho más fácil. Cuando el usuario final apunta a un dato utilizando el retículo, puede identificar más fácilmente la fila y la columna de ese dato, como el título por ejemplo.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Apercu_Etat_Mire%20-%20HC%20N%B0001.gif&type=thumb)
 Es posible configurar el grosor y la Color del retículo haciendo clic en la flecha bajo el retículo. 

- La opción "Anotación" se utiliza para mostrar el panel específico para la gestión de anotaciones.. 

- La opción "Snapshot" se utiliza para hacer una captura de pantalla. Hay dos opciones disponibles: 

	- Selección con lazo: el cursor del ratón se convierte en lazo. La selección definida en la vista previa se copia automáticamente como Image en el portapapeles. 

	- Inteligente: todo el contenido de la página actual en la vista previa se copia automáticamente en el portapapeles. 


Entonces, el contenido del portapapeles puede ser pegado en cualquier herramienta para la gestión de Image. 

- Las opciones "Zoom lazo", "Zoom in" y "Zoom out" completan las opciones de zoom del grupo "Zoom". 



<a name="NOTE2_3"></a>


### Modo de visualización
<a name="modo_visualizacion_ELTPARAGRAPHE000088"></a>

Existen varios modos de visualización: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_Apercu_Affichage.gif)


- **"Modo"Pantalla completa**: La impresión actual ocupa todo el espacio disponible. El menú con forma de cinta ya no es Visible. Para restaurar el modo de visualización estándar, presione la tecla Esc del teclado o haga clic en la pantalla Icon que se encuentra en la parte superior derecha. 

- **Panel miniaturas**:  Muestra las páginas del reporte como **miniaturas**. Las miniaturas se redimensionan automáticamente al modificar el área de visualización de las miniaturas.. Un menú contextual específico para las miniaturas permite imprimir directamente la página deseada. 
	La parte vista está representada en la miniatura. El área de zoom y posicionamiento puede cambiar de tamaño y moverse en la miniatura, el factor de zoom puede modificarse, ...
	Si hay marcadores disponibles, puede mostrarlos en el área de miniaturas.. Estos marcadores se muestran jerárquicamente. Para acceder a una sección específica, basta con hacer clic en el marcador correspondiente.



<a name="NOTE2_4"></a>


### Navegación en las páginas
<a name="navegacion_las_paginas_ELTPARAGRAPHE000105"></a>

Si el visor de reporte muestra una impresión con varias páginas, puede moverse entre las páginas, o mostrar directamente la página deseada introduciendo el número de esa página.

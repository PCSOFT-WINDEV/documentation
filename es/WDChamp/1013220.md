## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000143"></a>
WINDEV gestiona los siguientes eventos de forma predeterminada (en orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Inicialización | Corre cuando se abra la ventana. \*<br><br>**Special case:** Navegación por control ListView se llena automáticamente:<br>Cuando se ejecuta este código, la control ListView aún no se ha rellenado.. Para forzar la control ListView a rellenar este código, utilice:<br><br>- la función [ListDisplay](../WDLang1/3049005.md).<br><br>- las propiedades para gestionar los controles de navegación de ListView: [BrowsedFile](../Proprietes/2510040.md), [BrowsedItem](../Proprietes/2510111.md) o [filtros](../Proprietes/2510042.md).<br><br><br> |
| Fin de inicialización | **Code sólo para navegar por los controles de ListView que se llenan automáticamente:**<br>Ejecutar una vez que se haya llenado el control ListView. Este código le permite seleccionar, añadir o modificar un elemento de la control ListView. |
| partida | Corre cuando la control se enfoque. |
| salida | Ejecutar al pasar de un control a otro (tecla TAB, seleccionar otro control con el ratón, etc.). |
| Visualización de una fila | Ejecutar cuando un nuevo elemento debe ser mostrado en la pantalla de control ListView: volver a mostrar la ListView, desplazarse por la control ListView, etc. |
| Selección de una fila | Ejecutar cuando el usuario selecciona un elemento en la fila o cambia el elemento seleccionado. |


(\*) Por defecto, la "Inicialización" Event de cada control se ejecuta según el orden en que se crearon las control en la ventana. Para modificar este orden de ejecución: 

1. En la pestaña "Ventana", en el grupo "Orden", haga clic en "Inicialización". 

2. En la ventana mostrada, utilice las flechas de la derecha para modificar el orden en el que se inicializan los controles.

## Eventos opcionales
<a name="eventos_opcionales_ELTTEXTE000167"></a>
Se pueden utilizar varios eventos opcionales.

Para añadir un evento opcional:

1. Seleccione la control deseada.

2. Mostrar la ventana del código de esta control (tecla F2).

3. Haga clic en el enlace "Añadir otras Event s a xxx" en la parte inferior del código de la ventana, después de la última Event.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. Todos los eventos opcionales disponibles para el control se muestran. 

5. Marque la opción Event para agregar y validar. 
	Observación: Puede seleccionar varios eventos opcionales. 

6. El Event opcional seleccionado se agrega automáticamente a los Event s administrados por el control.




Para desactivar un Event opcional, simplemente realice las mismas operaciones para mostrar la lista de Event s opcionales. A continuación, simplemente desmarque los eventos opcionales para eliminar. 
Observación: Si el código desactivado contiene código WLanguage, este código se elimina automáticamente.

Se puede gestionar, por ejemplo:

- el desplazamiento del cursor por encima del control,

- botón izquierdo del ratón abajo, arriba, con un doble clic,

- el botón derecho del ratón presionado, liberado o con doble clic 

- la rueda del ratón, etc.

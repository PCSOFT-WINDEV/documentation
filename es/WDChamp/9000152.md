
## Ventana corredera (menú deslizante)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000199"></a>
Las ventanas correderas (también llamadas menús correderos) se utilizan a menudo en las aplicaciones móviles.. 

Las características para utilizar este tipo de ventana son las siguientes: 

- La ventana corrediza puede aparecer desde la izquierda o desde la derecha de la pantalla.. 

- La ventana deslizante puede mostrar parcialmente Above la ventana anterior o puede desplazar parcialmente esta ventana. 




Un uso típico para este tipo de ventana es la visualización de una lista de opciones o la configuración de una aplicación.. 

WINDEV Mobile le permite mostrar este tipo de ventana: 

- a través de un gesto del usuario, 

- mediante programación con la función [WinSlidingVisible](../WDLang1/1000020978.md). 




Atención: Las ventanas deslizantes sólo pueden utilizarse con ventanas maximizadas.

<a name="NOTE2_1"></a>


## ¿Cómo proponer ventanas correderas?
<a name="¿como_proponer_ventanas_correderas_ELTTEXTE000223"></a>
Los diferentes pasos para proponer ventanas correderas son los siguientes: 

1. [Creación de ventanas internas](#NOTE2_2). Estas ventanas internas serán las ventanas correderas. 

2. [Definición de las ventanas deslizantes que se mostrarán en la ventana description de la ventana principal](#NOTE2_3). 

3. Definición del modo de visualización de las ventanas correderas: 

	- por gesto. 

	- mediante programación. 






<a name="NOTE2_2"></a>


### Creación de ventanas internas
<a name="creacion_ventanas_internas_ELTPARAGRAPHE000053"></a>

Las ventanas correderas son ventanas internas.  

Esta ventana interna tiene las siguientes características: 

- [Cerrar](../WDLang1/3038018.md) utilizada en la ventana deslizante o en la ventana principal cerrará ambas ventanas. 

- Los controles de la ventana corredera son accesibles a través de la ventana principal, con las siguientes palabras clave: 

	- [MyLeftSlidingWindow](../Motscles/1511022.md) para identificar la ventana deslizante izquierda. 

	- [Ventana MyRightSliding](../Motscles/1511023.md) para identificar la ventana corrediza correcta. 
			
		```wl
		// Changes the caption color in the right sliding window
		MyRightSlidingWindow.STC_Caption.Color = LightRed
		```





- Aspecto de la ventana corredera: 
	La Color y el borde de la ventana corredera están definidos por la ventana interna. Si la ventana interna tiene un fondo Transparent, el fondo Color de la ventana deslizante será el fondo Color de la ventana principal. 
	



<a name="NOTE2_3"></a>


### Definir las ventanas correderas associated con una ventana
<a name="definir_las_ventanas_correderas_associated_con_una_ventana_ELTPARAGRAPHE000083"></a>

Para definir las ventanas correderas associated con una ventana: 

1. Abra la pestaña "Detalles" de la ventana principal de description. 

2. Seleccione las ventanas internas que se mostrarán como ventanas correderas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenCoulissante_Description.gif&type=thumb)


	- En el área "Ventana corredera izquierda" (o "Ventana corredera derecha"), especifique: 

	- **la ventana interna para mostrar**. Las diferentes ventanas internas de la aplicación se proponen al expandir la lista. 

	- **el ancho de la ventana interna**. 
						El ancho de la ventana corredera puede corresponder a: 

		- el ancho de la ventana interna. En este caso, deje vacía la "Anchura" control. 

		- un porcentaje de la ventana principal. En este caso, se aplicarán los anclajes definidos en la ventana interna. No se mostrará ninguna barra de desplazamiento horizontal. 
										Observaciones: 
										- La altura está fijada por la altura de la ventana interna. Se mostrará una barra de desplazamiento vertical si la altura supera la altura de la pantalla.. 
										- Tiene la posibilidad de utilizar [WinUsefulSize](../WDLang1/1000017089.md) para definir el tamaño de la ventana deslizante de useful.

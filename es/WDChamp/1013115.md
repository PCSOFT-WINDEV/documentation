
## Características de una control Barra lateral
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
Las principales características de una control Barra lateral son configurables:

- en la pestaña "General" de la descripción del control: puede configurar cada pestaña de control Barra lateral.

- en la pestaña "UI" del control description:  puede configurar el comportamiento de las pestañas.

- en la pestaña "Detalles" del control description: puedes elegir el fondo Image de las pestañas.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración de las fichas de control Barra lateral
<a name="configuracion_las_fichas_control_barra_lateral_ELTTEXTE000159"></a>
La pestaña "General" de la ventana de descripción del control Barra lateral permite:

- añadir, borrar, insertar nuevas pestañas.

- configurar (para la pestaña seleccionada en la vista previa):

	- el estado inicial de la ficha.

	- el título.

	- la imagen associated con el texto de la ficha. 
			**Atención**: Esta Image no aparece en el Background de la ficha sino a la izquierda del texto.







**Observación**: No hay barras de desplazamiento automatic en las pestañas de los controles de la barra lateral. Si no se pueden mostrar algunos controles, los controles se truncan.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Gestión de pestañas
<a name="gestion_pestanas_ELTTEXTE000183"></a>
La pestaña "UI" de la ventana control Barra lateral description permite gestionar la visualización de las pestañas.

Puede set:

- la animación utilizada al cambiar de pestaña.

- la opción de recordar la ficha de Active. Si se selecciona esta opción, el estado de la pestaña se guardará cuando se cierre la ventana. Esta pestaña se seleccionará automáticamente la próxima vez que se abra la ventana.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Gestión del fondo Image de fichas
<a name="gestion_del_fondo_image_fichas_ELTTEXTE000207"></a>
La pestaña "Detalles" de la ventana control Barra lateral description permite gestionar el fondo Image de las pestañas. Esta Image es utilizada por todas las fichas. Esta Image puede contener los diferentes estados de la ficha.

**Consejo**: Para get destacados Image s, compruebe el catálogo Image de WINDEV/WINDEV Mobile y seleccione un botón Image con 5 estados.
<a name="NOTE4_2"></a>


### Estados de las pestañas
<a name="estados_las_pestanas_ELTPARAGRAPHE000059"></a>

Puede gestionar los diferentes estados de la ficha a través de la aplicación associated Image.

Cada estado corresponde a un dibujo específico. Todos los dibujos se guardan en un solo Image. Esta imagen es associated con la pestaña: todo lo que tienes que hacer es especificar el número de estados (o dibujos) que se encuentran en el archivo Image.

Ejemplo: Archivo BlueTab.gif Image
<br>![En reposo, pulsada, Grayed fuera, seleccionada, pestaña de rastreo](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bleu0208.gif)


Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).



<a name="NOTE4_3"></a>


### Caso especial: Image de 3-Image fichas
<a name="caso_especial_image_3image_fichas_ELTPARAGRAPHE000075"></a>

WINDEV dividirá automáticamente la pestaña Image en tres Image separadas.

Se utilizan los siguientes criterios:

- la Image se divide en tres secciones

- La sección 1 corresponde a la zona izquierda

- La sección 3 corresponde a la zona derecha

- La sección 2 corresponde a la zona media. El tamaño de esta sección es igual a 16 píxeles. Si el tabulador es proporcional al texto, esta parte se repetirá tantas veces como sea necesario para que quepa el texto.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BoiteOutil3Image.gif)


**Observación**: Si la sección 2 tiene un tamaño inferior a 16 píxeles, no se recomienda utilizar un tabulador proporcional al texto. Si es necesario, integre los Image s adicionales y el texto en el Image.

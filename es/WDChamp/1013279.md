
## AAF: Desglosar u ocultar una estructura de árbol (Tabla o control Tabla TreeView)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
El TreeView y el TreeView controles Tabla ofrecen al usuario final varias opciones a través del menú contextual.

Una de las opciones de este menú contextual es la posibilidad de expandir o colapsar toda la estructura de árbol de control. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Arbre.gif)


La opción "Expandir todo" o "Contraer todo" se utiliza para: 

- Desplegar todas las ramas que se encuentran en la vista de árbol o en la jerarquía seleccionada.. 

- colapsar todas las ramas que se encuentran en la vista de árbol o en la jerarquía seleccionada.

## Ajustes
<a name="ajustes_ELTTEXTE000144"></a>


### Configurando el menú automatic
<a name="configurando_menu_automatic_ELTPARAGRAPHE000025"></a>

El menú contextual del TreeView o de control Tabla TreeView puede ser desactivado. Para configurar cómo se muestra el menú contextual associated con el control:

1. Abra la ventana de descripción del control.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, el menú contextual que se mostrará será el menú automatic del control. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.





### Desactivar la opción "Expandir todo"/"Colapsar todo
<a name="desactivar_opcion_expandir_todocolapsar_todo_ELTPARAGRAPHE000063"></a>

En el código, [AAFDisable](../WDLang1/1000022018.md) se utiliza para eliminar algunas opciones del menú contextual de AAF. 
Para desactivar la opción "Expandir todo"/"Contraer todo", utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafExpandCollapseAll*.

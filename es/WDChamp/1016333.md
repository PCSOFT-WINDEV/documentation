
## Manejo de una control Upload mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000128"></a>
El control Upload se utiliza para subir uno o más archivos. 

El control Upload puede ser manejado a través de la programación. Para ello, utilice el nombre del control Upload (botón"Enviar" o"Añadir").

Las funciones Uploadxxx de WLanguage también se pueden utilizar para manipular y control Upload. Para más detalles, consulte la lista de [funciones para la gestión de uploads](../WDLang2/1000018815.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestión de arrastrar y soltar archivos a un control Upload (HTML5)
<a name="gestion_arrastrar_soltar_archivos_control_upload_html5_ELTTEXTE000152"></a>
Los navegadores que soportan HTML5 (como Firefox y Chrome) le permiten arrastrar y soltar los archivos a la carpeta control Upload. 

La pestaña "UI" en la ventana de control Upload description (botón "Enviar" o "Añadir") se utiliza para gestionar esta función. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dnd_Upload.gif)


En el caso de un control Upload utilizado para cargar varios archivos propuestos por Default, el control sobre el que se puede realizar el arrastrar y soltar corresponde por Default a la celda de control Upload.

## Propiedades associated con un control Upload
<a name="propiedades_associated_con_control_upload_ELTTEXTE000176"></a>
Varias propiedades de WLanguage están disponibles para control Upload. Las principales propiedades son las siguientes: 


|   |   |
| --- | --- |
| [Conde](../Proprietes/2510097.md) | Permite conocer el número de archivos que se encuentran en una control Upload. |
| [NombreNavegadorArchivo](../Proprietes/1000018779.md) | Permite conocer el nombre inicial del archivo a subir que se encuentra en la el equipo del usuario de la Web. Este Property está disponible para los controles de carga de múltiples archivos. |
| [NombreServidorArchivo](../Proprietes/1000018780.md) | Le permite averiguar el nombre del archivo cargado que se encuentra en el servidor.. Este Property está disponible para los controles de carga de múltiples archivos. |
| [Selección múltiple](../Proprietes/1000017054.md) | Permite conocer y modificar el modo de selección de archivos en una control Upload. |
| [TipoArchivos](../Proprietes/1000019034.md) | Permite conocer y modificar el filtro de los ficheros propuestos en el selector de ficheros de la control Upload. |
| [Valor](../Proprietes/2510130.md) | Permite conocer y modificar la lista de archivos encontrados en la control Upload. |


Para obtener una lista completa de las propiedades que se pueden utilizar en el control Upload, consulte [Lista de propiedades disponibles para la control Upload](../WDChamp/1016332.md).

## Visión general de la Web control Cámara
<a name="vision_general_web_control_camara_ELTTEXTE000104"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

La Web control Cámara se utiliza para ver un archivo que se actualiza a intervalos de tiempo regulares.. Este archivo puede corresponder a una imagen emitida por una "cámara web", por ejemplo.


### Crear control
<a name="crear_control_ELTPARAGRAPHE000016"></a>

Para crear una Web control Cámara:

1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Imagen" y seleccione "Cámara Web".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.


Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000134"></a>


### Recuperación de una imagen emitida por una "cámara web".
<a name="recuperacion_una_imagen_emitida_por_una_camara_web_ELTPARAGRAPHE000042"></a>

Recuperar una imagen emitida por un control de "cámara web":

1. **En el equipo relacionado a la "cámara web"**: configurar la grabación de la imagen emitida por una "cámara web" (nombre del archivo, velocidad de actualización, "cámara web" utilizada,....). Este ajuste se puede realizar:

	- por el software suministrado con la "cámara web".

	- por la aplicación "WD WebCam.exe" disponible en el sitio de [PC SOFT](https://www.windev.com/ts/download/index.html) (página de descarga, párrafo "Otro software").
			El archivo de imagen debe ser accesible desde cualquier ordenador que utilice el sitio WEBDEV.
			**Observación**: El código fuente de "WD WebCam" está disponible en los ejemplos proporcionados con WINDEV.




2. **En el ordenador de desarrollo**: especificar el archivo de imagen relacionado a la Web control Cámara (pestaña "General" en la ventana de descripción de control).




## Manejo de una barra de herramientas mediante programación (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
WINDEV le permite manipular una barra de herramientas control a través de la programación. Para hacerlo, usa la Variable de la "Barra de herramientas" control en el código.

La Variable de la barra de herramientas control corresponde al nombre de la barra de herramientas. Este Variable no tiene ningún valor.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manejando los control en una barra de herramientas control
<a name="manejando_los_control_una_barra_herramientas_control_ELTTEXTE000114"></a>
Se puede acceder a los controles que se encuentran en la barra de herramientas:

- como cualquier otro control, usando su nombre directamente.

- especificando el nombre de la barra de herramientas control, luego el nombre de la control. Por ejemplo: 
	
	```txt
	Toolbar1.Button1
	```





**Observación**: las propiedades que afectan el tamaño y la posición de los controles que se encuentran en la barra de herramientas se pueden acceder en modo de sólo lectura.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## ¿Cómo cerrar la barra de herramientas flotante control?
<a name="¿como_cerrar_barra_herramientas_flotante_control_ELTTEXTE000147"></a>
Para cerrar una barra de herramientas flotante control usando la X, debe escribir la siguiente línea de código en la barra de herramientas Event de "Cierre" de la barra de herramientas control:


```wl
MySelf..Visible = False
```
**Atención**: No se recomienda el uso de [Cerrar](../WDLang1/3038018.md). Esta función cierra la ventana y no la barra de herramientas control.

## Funciones específicas para la gestión de los controles de la barra de herramientas
<a name="funciones_especificas_para_gestion_los_controles_barra_herramientas_ELTTEXTE000179"></a>
Las siguientes funciones se utilizan para manipular la barra de herramientas control a través de la programación:



|   |   |
| --- | --- |
| [&lt;Barra de herramientas&gt;.posición](../WDLang1/3020007.md) | Indica el modo de visualización de una barra de herramientas control. |
| [ToolbarDock](../WDLang1/3020006.md) | Acopla una barra de herramientas control a una ventana o cambia la posición de anclaje de una barra de herramientas acoplada control. |
| [ToolbarUndock](../WDLang1/3020008.md) | Transforms a docked Toolbar control into a floating Toolbar control. |

## Propiedades específicas de la gestión de una barra de herramientas control
<a name="propiedades_especificas_gestion_una_barra_herramientas_control_ELTTEXTE000231"></a>
Las siguientes propiedades se utilizan para gestionar las características de una Barra de Herramientas control a través de la programación: 


|   |   |
| --- | --- |
| [Altura](../Proprietes/2510050.md) | Se utiliza para averiguar la altura de una barra de herramientas control. |
| [anchura](../Proprietes/2510059.md) | Se utiliza para averiguar y modificar el ancho de una barra de herramientas control. |
| [InitialHeight](../Proprietes/2510043.md) | Se utiliza para averiguar la altura inicial de una barra de herramientas control. |
| [InitialWidth](../Proprietes/2510067.md) | Se utiliza para averiguar el ancho inicial de una barra de herramientas control. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una barra de herramientas control, consulte [Propiedades associated con los controles de la barra de herramientas](../WDChamp/1013124.md).

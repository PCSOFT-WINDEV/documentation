## Visión general de Flexbox control
<a name="vision_general_flexbox_control_ELTTEXTE000122"></a>
Flexbox es un sistema de diseño estándar para páginas web. Los controles Flexbox se introdujeron en WEBDEV 28. 

Más fácil imposible: 

- Suelte los control s en el Flexbox control.

- A continuación, el Flexbox control posiciona automáticamente los control s en función de los parámetros deseados.




Por ejemplo, la Flexbox control le permite: 

- Definir la posición de los controles al cambiar el tamaño del navegador. Para obtener más información, consulte [Ejemplo de Flexbox: automatic control disposición](../WDChamp/1410087285.md). 

- Relaciones set para distribuir el espacio disponible entre los mandos. Para obtener más información, consulte [Ejemplo de Flexbox: ratios para distribuir el espacio disponible entre los controles](../WDChamp/1410087275.md).

## Creación de un Flexbox control
<a name="creacion_flexbox_control_ELTTEXTE000146"></a>
Para crear un Flexbox control:

1. En la pestaña "Creación", en el grupo "Contenedores", haga clic en "Flexbox". 

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Observación: Al desplegar la opción "Flexbox" aparecen varios controles Flexbox predefinidos, entre ellos: 

- Flexbox en columnas, 

- Flexbox en filas, 

- Tarjetas Flexbox. 




Para mostrar las características de la control: 

1. Seleccione la control. 

2. Seleccione "description" en el menú contextual de control.

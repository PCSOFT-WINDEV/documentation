## Presentación
<a name="presentacion_ELTTEXTE000343"></a>
WEBDEV permite crear modelos de páginas. Estas plantillas contienen todos los elementos gráficos comunes a todas las páginas de su sitio.

Los cambios realizados en un modelo de páginas se aplican automáticamente a todas las páginas que utilizan dicho modelo.

Un modelo de páginas permite utilizar el tema gráfico definido para un sitio.

**Se recomienda utilizar plantillas en sus sitios WEBDEV en lugar de conjuntos de marcos.** El uso de plantillas Page en sus sitios le permite:

- optimizar la velocidad de visualización de las páginas del sitio.

- limitar el número de archivos HTML generados (y por lo tanto el número de archivos HTML descargados a el equipo del usuario de la Web).

- crear las páginas del sitio fácilmente.

- definir el diseño de las páginas del sitio.

- actualizar el tema gráfico del sitio fácilmente.

- simplificar la gestión de los objetivos de las páginas del sitio.

- ...

## Creación de una Page modelo
<a name="creacion_una_page_modelo_ELTTEXTE000367"></a>
Se puede crear una Page modelo:

- directamente, como cualquier otro Page.

- de un Page existente.



<a name="NOTE2_2"></a>


### Creación de una Page modelo
<a name="creacion_una_page_modelo_ELTPARAGRAPHE000040"></a>

Para crear una Page modelo:

1. Crear una nueva Page modelo: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Page" y luego en "Page modelo".




2. Especifique (si es necesario) si el Page modelo debe: 

	- estar en blanco, 

	- basarse en una de las plantillas del proyecto, 

	- usando un modelo preestablecido.  




3. Crear todos los controles necesarios. Estos controles se incluirán en todos los Page s que utilicen el Page modelo.

4. Introduzca (si es necesario):

	- el WLanguage eventos / procesos associated con el Page modelo.

	- los eventos / procesos WLanguage associated con los diferentes controles del Page modelo.
			Estos eventos y procesos se importarán automáticamente a las páginas que utilicen el sistema modelo.




5. Dé un nombre a su modelo (pestaña "General" de la ventana description).

6. En la pestaña "Inicio", en el grupo "General", haga clic en "Guardar" (o ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif) en los botones de acceso rápido).




La Page modelo corresponde a un fichero cuya extensión es ".WWT"..<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=modele_carreJaune.gif&type=thumb)

<a name="NOTE2_3"></a>


### Creando un Page modelo a partir de un Page existente
<a name="creando_page_modelo_partir_page_existente_ELTPARAGRAPHE000085"></a>

Para crear una Page modelo a partir de una Page existente:

1. Abra el Page elegido como modelo. 

2. Mantenga los controles requeridos por la modelo. Estos controles se incluirán en todos los Page s que utilicen el Page modelo.

3. Mantener el código de procesos requerido por la modelo. Estos procesos se importarán automáticamente a las páginas que utilizan la modelo.

4. Dé un nombre a su página (pestaña "General" de la ventana description de la página).

5. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar como modelo". La Page actual se convierte en una modelo:

	- su nombre va precedido de "Plantilla".

	- aparece un borde amarillo alrededor de la Page en el editor Page.

	- el Page modelo aparece en el panel "Explorador de proyectos", en la sección "Plantillas de Page s".

	- la extension del fichero modelo es ".WWT".

## Aplicación de una Page modelo
<a name="aplicacion_una_page_modelo_ELTTEXTE000403"></a>


### Crear una nueva Page que utilice una Page modelo
<a name="crear_una_nueva_page_que_utilice_una_page_modelo_ELTPARAGRAPHE000117"></a>

Para crear una nueva Page que utilice una Page modelo:

1. Crear una nueva Page: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido.

	- En la ventana que aparece, haga clic en "Page" y luego en "Page".




2. En el asistente, seleccione la opción "Página" a la izquierda y enseguida seleccione el modelo personalizado a la derecha.

3. Valide la creación de la página. Se crea una Page associated en blanco con su Page modelo. 




Se utiliza un cuadrado amarillo para identificar los elementos pertenecientes a la Page modelo. Estos elementos pueden ser modificados:

- en el Page modelo: la modificación será efectiva para todas las páginas que usen esta modelo.

- en la página, con el mecanismo de "control overload": la modificación será efectiva sólo para el Page.



<a name="NOTE3_2"></a>


### Aplicar una modelo a una Page existente 
<a name="aplicar_una_modelo_una_page_existente_ELTPARAGRAPHE000138"></a>

Para aplicar una modelo a una Page existente:

1. Visualizar la Page en el editor Page.

2. En la pestaña "Página", en el grupo "Descripción", despliegue "Modelos" y seleccione "Heredar de un modelo".

3. Seleccione la Page modelo que desee aplicar.

4. Valide.

5. Especifique si desea actualización las propiedades de la página desde el menú modelo. Si responde Sí, la Page se relacionado automáticamente a la modelo para la mayoría de sus características.




**Observación**: las propiedades de la Page generada por la modelo son las siguientes: Imagen, imagen de fondo fija, fondo color, texto color, enlace color, color del enlace visitado, color del enlace Active, enlace subrayado, centrar la página, anclar los control s en la página, título, control para mostrar la barra de progreso, posición de la barra de progreso Ajax.

## Abrir el modelo associated con un Page
<a name="abrir_modelo_associated_con_page_ELTTEXTE000433"></a>
Para abrir el modelo associated con un Page:

1. Visualizar la Page en el editor Page.

2. En el menú contextual de un elemento modelo, seleccione "Abrir modelo".




**Otra posibilidad**: 

- Listar las plantillas associated con la Page: en la pestaña "Página", en el grupo "Descripción", despliegue "Modelos" y seleccione "Lista de modelos utilizados". 

- Seleccione el modelo deseado y haga clic en el botón "Abrir modelo" (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_ouvrir_modele.gif)).

## Aplicando las modificaciones realizadas en un modelo
<a name="aplicando_las_modificaciones_realizadas_modelo_ELTTEXTE000457"></a>
Cuando se modifica un Page modelo, las modificaciones pueden aplicarse a uno o más Page s que utilicen este modelo en el proyecto actual.

La aplicación de las modificaciones se propone al cerrar el modelo (si no se han guardado las modificaciones).. 

Esta operación se puede realizar en cualquier momento:

- de la modelo: en la pestaña "Página", en el grupo "Descripción", despliegue "Modelos" y seleccione "Actualizar páginas asociadas".

- de cualquier Page que utilice el modelo: 

	- abriendo la lista de plantillas utilizadas: en la pestaña "Página", en el grupo "Descripción", despliegue "Modelos" y seleccione "Lista de modelos utilizados". 

	- y pulsando el botón "Actualizar".




- de la página modelo, a través del ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modele_MAJ.gif) Icon en la barra modelo.




**Observación**: Las modificaciones realizadas en la modelo se aplican según las características sobrecargadas en las páginas que utilizan la modelo.

## Lista de modelos utilizados
<a name="lista_modelos_utilizados_ELTTEXTE000505"></a>
Para ver la lista de plantillas utilizadas en un Page, vaya a la pestaña "Página", grupo "Descripción", despliegue "Modelos" y seleccione "Lista de plantillas utilizadas".
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ModelePage_ListeModele.gif)

Desde esta ventana, puedes hacerlo:

- "Abrir una modelo": abre el modelo seleccionado en el editor de Page.

- "Actualizar": los controles y el código del modelo se actualizan en el Page.

- "Añadir": añade una modelo a la Page actual .

- "Disociar": los controles modelo que se encuentran en la página están disociados de los controles modelo. La modelo se elimina de la lista de plantillas utilizadas, pero sus controles no se eliminan de la Page.

- "Borrar": borra la modelo (controles y código) de la Page.

## Creación
<a name="creacion_ELTTEXTE000282"></a>


### Creación de un conjunto de marcos
<a name="creacion_conjunto_marcos_ELTPARAGRAPHE000048"></a>

WEBDEV propone varios métodos para crear un conjunto de marcos:

- Creación de un conjunto de marcos en blanco.

- Usando una configuración de fotogramas preestablecida.

- Construcción de un sitio basado en conjuntos de marcos a partir del análisis.




Para obtener más información, consulte [Creación de un conjunto de marcos](../WDChamp/1012002.md).
<a name="NOTE2_2"></a>


### Abrir un juego de marcos
<a name="abrir_juego_marcos_ELTPARAGRAPHE000064"></a>

Para abrir un conjunto de marcos: 

1. En el panel "Inicio", en el grupo "General", expanda "Abrir" y seleccione "Abrir" (Ctrl + O).

2. Seleccione el conjunto de marcos que desea abrir.




Otras operaciones:

- Haga doble clic en el nombre del conjunto de marcos mostrado en el panel "Explorador de proyectos", opción "Páginas".

- Haga doble clic en la representación del conjunto de marcos en el gráfico del proyecto, en el editor del proyecto.

## Asociar un marco con una página
<a name="asociar_marco_con_una_pagina_ELTTEXTE000314"></a>


### Asociar un marco a una página
<a name="asociar_marco_una_pagina_ELTPARAGRAPHE000086"></a>

Para asociar un marco a una página: 

1. En el panel "Explorador de proyectos", seleccione la página que desea asociar.

2. Arrastre y suelte la página que se encuentra en el panel "Explorador de proyectos" hacia el marco donde se debe mostrar la página.

3. Guardar FrameSet: En el panel "Página", en el grupo "Edición", expanda "Cuadro" y seleccione "Guardar el cuadro".




Observaciones:

- Para asociar un marco con una página, también puede hacer clic dentro de un marco y abrir la página que debe ser associated con el marco actual. La página abierta aparece en el marco actual.

- Al crear y guardar el juego de marcos, el estado inicial del juego de marcos se define automáticamente. Para obtener más información, consulte [Definir el estado inicial de un conjunto de marcos](#NOTE6_1).

## Adición o eliminación de un marco en/desde un conjunto de marcos
<a name="adicion_eliminacion_marco_endesde_conjunto_marcos_ELTTEXTE000340"></a>


### Añadir un marco a un conjunto de marcos
<a name="anadir_marco_conjunto_marcos_ELTPARAGRAPHE000117"></a>

Para añadir un marco a un conjunto de marcos: 

1. En el panel "Página", en el grupo "Editar", expanda "Cuadro" y seleccione: 

	- "Vertical" para realizar una división vertical. 

	- "Horizontal" para realizar una división horizontal.




2. Seleccione la posición del nuevo marco (a la izquierda, a la derecha, arriba, abajo) en relación con el marco actual.


El marco seleccionado se divide. Tiene la posibilidad de cambiar el tamaño de los dos cuadros nuevos. Para obtener más información, consulte [Modificar las dimensiones de un marco](#NOTE5_1).
<a name="NOTE4_2"></a>


### Eliminar un fotograma de un conjunto de fotogramas
<a name="eliminar_fotograma_conjunto_fotogramas_ELTPARAGRAPHE000145"></a>

Para eliminar un fotograma de un conjunto de fotogramas: 

1. Seleccione el marco que desea eliminar. El contorno del marco seleccionado aparece en rojo.

2. En la pestaña "Página", en el grupo "Editar", expanda "Marco" y seleccione "Eliminar el marco". Los marcos restantes se redimensionan automáticamente para ocupar el espacio disponible.

## Modificar las dimensiones de un marco
<a name="modificar_las_dimensiones_marco_ELTTEXTE000370"></a>
Para modificar las dimensiones de un marco: 

1. Haga clic dentro del marco que debe ser modificado. El contorno del marco seleccionado aparece en rojo.

2. Pase el cursor por el Border para cambiar el tamaño. Cuando el cursor del ratón se convierta en una flecha doble, mantenga pulsado el botón izquierdo del ratón mientras cambia el tamaño del marco.

3. Suelte el botón izquierdo del ratón. Se realiza el redimensionamiento.


Para evitar tener demasiadas barras de desplazamiento en los marcos, los controles que se encuentran en las páginas associated con los marcos deben ser Visible en el conjunto de marcos.

## Definición de las páginas iniciales del conjunto de marcos
<a name="definicion_las_paginas_iniciales_del_conjunto_marcos_ELTTEXTE000394"></a>
Las páginas iniciales de un conjunto de marcos deben ser definidas: estas páginas serán abiertas por Default cuando el frameset se abra en el navegador del usuario de la web.

Para definir las páginas iniciales del conjunto de marcos:

1. Asociar las páginas solicitadas con los diferentes marcos.

2. En el panel "Página", en el grupo "Editar", expanda "Cuadro" y seleccione "Definir el estado inicial".




**Recordatorio**: Un conjunto de marcos se utiliza para mostrar varias páginas en la misma ventana de un navegador.. El archivo HTML correspondiente al conjunto de marcos contiene:

- La Definition de la división de los diferentes marcos en el navegador.

- El nombre de las páginas abiertas cuando el conjunto de marcos se muestra en el navegador.



<a name="NOTE6_2"></a>


### Anidar los juegos de marcos
<a name="anidar_los_juegos_marcos_ELTPARAGRAPHE000205"></a>

Por Default, se utiliza un conjunto de marcos para mostrar varias páginas en la misma ventana del navegador. Una de las páginas mostradas en el conjunto de marcos puede ser un conjunto de marcos en sí mismo: hablamos de "conjuntos de marcos anidados". Los conjuntos de marcos anidados se pueden utilizar para implementar un diseño excepcional en sus páginas, por ejemplo.

Para obtener más información, consulte [Anidar los conjuntos de marcos](../WDChamp/1012006.md)

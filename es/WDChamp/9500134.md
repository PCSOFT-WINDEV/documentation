## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
Un control Tabla puede estar basado en un archivo de datos, en una variable o rellenarse mediante programación: 

- Un control Tabla basado en un archivo de datos muestra los datos de una fuente de datos específica (archivo de datos, consulta, etc.).

- Un control Tabla basado en una variable muestra los datos contenidos en la variable.

- Un control Tabla rellenado mediante programación muestra los datos definidos en el código.

## Contenido del control Tabla
<a name="contenido_del_control_tabla_ELTTEXTE000120"></a>
El contenido se define al crear el control y puede modificarse en la ventana descripción (pestaña "Contenido").<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Table_Contenu.gif)
Esta ventana permite definir si el control es una: 

- Tabla basada en un archivo de datos (opción "Archivo/Consulta"),

- Tabla rellenada mediante programación (opción "Mediante programación"),

- Tabla basada en una variable (opción "Mediante variable"). 




**Para los controles Tabla basados en una variable**, es posible definir la fuente de datos y la variable almacenada. Para obtener más información sobre este control, consulte [Control Tabla basado en una variable](../WDChamp/1013306.md). 

**Para los control Tabla basados en un archivo de datos**, es posible definir las características de la iteración realizada en la fuente de datos. Para obtener más información, consulte [Vincular un control Tabla a un archivo de datos](../WDChamp/1013236.md).

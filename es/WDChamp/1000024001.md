## Vista general de la control Zona multilínea
<a name="vista_general_control_zona_multilinea_ELTTEXTE000261"></a>
La control Zona multilínea le permite crear fácilmente interfaces al estilo iOS. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ZM_Exemple.gif)


Este control permite agrupar varios controles: 

- opciones de una categoría, 

- información sobre un contacto, etc 

- ...




Este control puede contener: 

- filas definidas en el editor (filas estáticas). 

- filas definidas en tiempo de ejecución, mediante programación (filas dinámicas).

## Creación de una control Zona multilínea
<a name="creacion_una_control_zona_multilinea_ELTTEXTE000285"></a>
Para crear una control Zona multilínea:

1. En la pestaña "Creación", en el grupo "Datos": 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) expandir "Looper" y hacer clic en "Multiline zone".

		1. Haga clic en la ubicación en la que desea crear el control.







**Observaciones**: 




Por Default, el control creado contiene un solo fila "estático" vacío. Este control está personalizado en su ventana de description. 

Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Características de la control Zona multilínea
<a name="caracteristicas_control_zona_multilinea_ELTTEXTE000309"></a>


### Fila description
<a name="fila_description_ELTPARAGRAPHE000094"></a>

La pestaña "General" en la ventana de description de una zona multilínea se utiliza para: 

- Definir los parámetros de la fila actual,

- agregar una nueva fila, 

- duplicar una fila existente,

- borrar una fila existente. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Zml%20-%20HC%20N%B0001.gif)
Cuando se añade un nuevo fila ("Nueva fila") o se modifica el fila actual ("Modificar tipo de fila" o doble clic en el fila), se inicia un asistente que permite crear contenidos Default correspondientes a los casos de uso más habituales:

- un botón de casilla de verificación,

- un subtítulo con un subtítulo que dirige a una lista de opciones, ....

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ZM_Ligne.gif)
Al validar la ventana description, la zona multilínea control se actualiza con las filas seleccionadas. 

Los controles correspondientes a las opciones seleccionadas de fila se crean automáticamente. 
<a name="NOTE3_2"></a>


### Estilo de la control Zona multilínea
<a name="estilo_control_zona_multilinea_ELTPARAGRAPHE000140"></a>

La pestaña "Estilo" de la ventana control Zona multilínea description se utiliza para configurar las características: 

- de la leyenda de control. 

- del exterior de Border. 

- del interior de Border. 

- de la fila seleccionada. 



<a name="NOTE3_3"></a>


### Uso estático de la control Zona multilínea
<a name="uso_estatico_control_zona_multilinea_ELTPARAGRAPHE000152"></a>

Una control Zona multilínea es estática si la opción "Definir como fila repetida del control" (pestaña "General") no está marcada para ninguna de las filas.

En este caso, la control Zona multilínea se utiliza para simplificar la interfaz agrupando los elementos por "tema".. 

**Por ejemplo**, un control Zona multilínea estático será usado en un formulario para introducir el nombre y el apellido de un Contact, su Address, ... <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ZM_Statique.gif)
 En este caso, el contenido será estático y el número de filas será fijo. Por ejemplo, para la identidad: 

- un fila para la empresa,

- un fila para el apellido,

- un fila para el primer nombre. 


En este caso, los controles en cada fila son relacionado al artículo correspondiente en el archivo de datos de los clientes.  

Otro ejemplo: usando un control Zona multilínea estático para comportarse como un botón de radio: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ZM_Selecteur.gif)

<a name="NOTE3_4"></a>


### Uso dinámico de una control Zona multilínea
<a name="uso_dinamico_una_control_zona_multilinea_ELTPARAGRAPHE000185"></a>

Una control Zona multilínea es dinámica si la opción "Definir como fila repetida del control" (pestaña "General") está marcada para cualquiera de las filas. Esta opción puede ser marcada para un solo fila. 

En este caso, se pueden añadir filas mediante programación (por ejemplo, una lista de opciones o una lista de valores posibles según una elección o a una base de datos). 

Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ZM_Dynamique.gif)
 Aquí, el primer fila es estático y las filas de productos corresponden a un fila dinámico en el editor. 

Una Line puede ser añadida por [mlzAddLine](../WDLang1/1000020199.md). Para más detalles sobre las funciones de manejo de los controles dinámicos de la Zona Multilínea, véase [Funciones de gestión de control Zona multilínea](../WDLang1/1000020215.md). 
<a name="NOTE3_5"></a>


### Estado de la control Zona multilínea
<a name="estado_control_zona_multilinea_ELTPARAGRAPHE000210"></a>

Si el control Zona multilínea está "inactivo" o "en gris", el clic Process no se ejecuta y sus filas no pueden ser seleccionadas.

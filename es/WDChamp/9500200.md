### control en una ventana de WINDEV y WINDEV Mobile
<a name="control_una_ventana_windev_windev_mobile_ELTPARAGRAPHE000011"></a>

El control multimedia se utiliza para reproducir archivos multimedia.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **En un PC**, los siguientes elementos deben estar instalados: Windows Media Player, una versión reciente de DirectX, así como los códecs necesarios para leer los archivos de vídeo.. Los vídeos no se pueden reproducir si no están instalados los códecs necesarios.

## Creación de un control multimedia
<a name="creacion_control_multimedia_ELTTEXTE000232"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para crear un control Multimedia en una ventana: 

1. En la pestaña "Creación", en el grupo "Controles gráficos": 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ampliar "Vídeo y captura" y seleccionar "Multimedia".




2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Manejando un control multimedia
<a name="manejando_control_multimedia_ELTTEXTE000256"></a>
Una control multimedia puede manejarse mediante programación a través de:

- la [Funciones multimedia de WLanguage](../WDLang1/1000015005.md). Estas funciones se utilizan para asociar un archivo multimedia con el control y para reproducirlo.

- la [Propiedades de WLanguage](../WDChamp/9500201.md). [Valor](../Proprietes/2510130.md) permite asociar un archivo multimedia a un control y reproducirlo.





### Observaciones
<a name="observaciones_ELTPARAGRAPHE000136"></a>

- Los controles multimedia no se pueden utilizar en los controles de Looper.

- No se puede capturar la imagen de un video pagado en un control multimedia.

## Resumen de control Código de barras
<a name="resumen_control_codigo_barras_ELTTEXTE000227"></a>


### Control de ventana, página o reporte
<a name="control_ventana_pagina_reporte_ELTPARAGRAPHE000011"></a>

La control Código de barras le permite visualizar fácilmente los códigos de barras en sus ventanas, páginas o informes.. 

El valor de una control Código de barras corresponde a una cadena alfanumérica.

**Atención**: Para leer directamente el valor de un código de barras en la pantalla con un dispositivo especial, asegúrese de utilizar el dispositivo adecuado.

## Creación de una control Código de barras
<a name="creacion_una_control_codigo_barras_ELTTEXTE000251"></a>
Para crear una control Código de barras: 

1. En la pestaña "Creación": 

	- en el editor de ventanas o en el editor de páginas: en el grupo "Controles gráficos", haga clic en "Código de barras".

	- en la editor de reportes: en el grupo "Datos", haga clic en "Código de barras".




2. Haga clic en la posición donde debe crearse la control en la ventana, página o reporte. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. Ahora tiene la posibilidad de elegir el tipo de código de barras (pestaña"General"). Este tipo no se puede modificar mediante programación.

**Consejo**: También puede crear una control Campo de entrada "preformateada" ampliando la opción "Código de barras".. Aparece la lista de los tipos de código de barras disponibles con su vista previa en la plantilla del proyecto.. Simplemente haga clic en el control que desea crear. 


### Observación: Tamaño del código QR
<a name="observacion_tamano_del_codigo_ELTPARAGRAPHE000095"></a>

Un código QR es una matriz de puntos cuyo tamaño varía según los datos contenidos y la tasa de corrección. 

La norma de gestión del código QR requiere que todos los puntos sean del mismo tamaño. 

Para seguir siendo legible, el tamaño del código QR no siempre es proporcional al tamaño de la control. De un código QR a otro, puede aparecer un margen dentro de la control para asegurarse de que el código cumple con la norma y que puede ser leído.

## Características de control Código de barras

### Características de control Código de barras
<a name="caracteristicas_control_codigo_barras_ELTPARAGRAPHE000108"></a>


|   |   |
| --- | --- |
| Un control Código de barras incluye:<br><br>- la leyenda de la control Código de barras.<br><br>- la Image del código de barras.<br><br>- el valor del código de barras (que corresponde a una cadena alfanumérica).<br><br><br> | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=codeBarre_Detail.gif)<br> |

**El editor de ventanas o el editor de páginas** se utiliza para seleccionar:

- el tipo de código de barras a visualizar (con sus características si es necesario).

	- Códigos de barras con cheque: El último carácter del código de barras corresponde a un carácter de control.

	- Código de barras extendido: El valor del código de barras puede contener los 128 caracteres que se encuentran en el código ASCII.

	- Tamaño proporcional: La altura del código de barras es proporcional a su anchura (para los tipos UPC y EAN).

	- Valor del código de barras visible: Muestra (o no) el valor del código de barras debajo de su dibujo..
			Para conocer los diferentes tipos de códigos de barras soportados por WINDEV, WINDEV Mobile o WEBDEV, consulte [Los diferentes tipos de códigos de barras](../WDLang5/3046064.md).




- la orientación y posición del código de barras (pestaña"Detalles").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) la Border del código de barras (pestaña "Estilo" del editor de la ventana).




**El editor de reportes** permite indicar:

- el valor inicial del código de barras. Por ejemplo, tienes la capacidad de escribir el INTERNET Address relacionado a un código de barras QR. 

- el tipo de código de barras a mostrar (con sus características específicas si es necesario).

	- Con verificación: El último carácter del código de barras corresponde a un carácter de control.

	- Código expandido: El valor del código de barras puede contener los 128 caracteres que se encuentran en el código ASCII.

	- Valor del código de barras visible: Muestra (o no) el valor del código de barras debajo de su dibujo.

	- Tamaño proporcional: La altura del código de barras es proporcional a su anchura (para los tipos UPC y EAN).

	- Ignorar caracteres de formato: Ignora los corchetes en un código de barras 128 o EAN-128.

	- El valor contiene caracteres de control: Código de barras que contiene caracteres control (válido sólo para el CÓDIGO 128, DATAMATRIX, EAN 128 y códigos de barras PDF 417). Por Default, los caracteres de control están encerrados entre los caracteres "&lt;" y "&gt;".. Para que estos personajes no sean interpretados como personajes de control, deben ser doblados.

	- Juegos de caracteres: Código de barras que utiliza uno o más conjuntos de caracteres para comprimir el código de barras. Esta set de caracteres debe ser definida en el valor del código de barras.

	- Caracteres de control: Código de barras con caracteres de control (válido para los códigos de barras CODE 128, DATAMATRIX, EAN 128 y PDF 417).
			Para conocer los diferentes tipos de códigos de barras soportados por WINDEV o WEBDEV, consulte [Los diferentes tipos de códigos de barras](../WDLang5/3046064.md).




- la orientación y posición del código de barras.

- la Border del código de barras (pestaña "Estilo").

### Arrastrar - Soltar
<a name="arrastrar_soltar_ELTPARAGRAPHE000173"></a>

En una ventana, se puede utilizar un control Código de barras para realizar un Arrastrar y soltar programado.. Para obtener más información, consulte [Arrastrar y soltar programados](../WDLang1/3030013.md).


## Efecto Parallax
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000200"></a>
Los efectos "Parallax" permiten desplazar los elementos de la interfaz a diferentes velocidades.. 

Es posible definir: 

- **un paralaje de controles**: En una ventana o en una Page, las animaciones de paralaje sirven para configurar la velocidad a la que se desplaza o se reduce la control en relación con el desplazamiento de la barra de desplazamiento en la ventana (o en la Page).

- **un paralaje de fondo Image**: En una Page, se puede utilizar una animación de paralaje en las imágenes Background. Cuando se utiliza la barra de desplazamiento, la Image de fondo se moverá a una velocidad específica en relación con el movimiento de la barra de desplazamiento.




Los efectos Parallax están disponibles: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) para todos los controles de ventanas. Para obtener más información, consulte [El efecto paralaje en el editor de ventanas](#NOTE2_1).

## El efecto paralaje en el editor de ventanas
<a name="efecto_paralaje_editor_ventanas_ELTTEXTE000224"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000047"></a>

El editor de la ventana propone un efecto de paralaje de control. 

Por ejemplo: 

- Si no se define ningún paralaje, los controles y la barra de desplazamiento de la ventana se mueven a la misma velocidad.. 

- Si la control está fija, la control no se moverá aunque se use la barra de desplazamiento. 

- El paralaje se utiliza para realizar efectos específicos, por ejemplo haciendo que la velocidad de movimiento de la control (o su tasa de reducción) cambie en relación con la velocidad de movimiento de la barra de desplazamiento de la ventana. 









### Definición del efecto paralaje en el editor de ventanas
<a name="definicion_del_efecto_paralaje_editor_ventanas_ELTPARAGRAPHE000065"></a>

Para definir el efecto de paralaje de una control: 

1. Abra la ventana de descripción del control. 

2. En la pestaña "UI", despliegue la opción "Parallax". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Effet_de_parallaxe%20-%20HC%20N%B0001.gif&type=thumb)


3. Seleccione la opción deseada: 

	- **Ningún parallax**: La control se desplaza verticalmente siguiendo el movimiento de la barra de desplazamiento de la ventana.

	- **Efecto Material Design sin reducción**: La control se desplaza verticalmente de forma proporcional al movimiento de la barra de desplazamiento de la ventana. En este caso, la velocidad de desplazamiento es set al 50% y la velocidad de reducción es set al 0%. 

	- **Efecto Material Design con reducción**: La control se reduce verticalmente en relación con el movimiento de la barra de desplazamiento en la ventana. En este caso, la velocidad de movimiento es set al 50% y la velocidad de reducción es set al 50%. 
			Para un control Imagen, el Image sigue el modo de visualización. 

		- Si la imagen se estira, se distorsionará para que se muestre completamente a la altura de la control. 

		- Si la imagen es homotética centrada, se reducirá proporcionalmente a la altura de control.




- **Control estático**: El control está fijado. No se mueve en la ventana. Siempre es Visible cuando la barra de desplazamiento se mueve en la ventana. En este caso, la velocidad de movimiento es set al 0% y la velocidad de reducción es set a 100%.

- **Personalizar efecto**: Tienes la capacidad de especificar la tasa de paralaje de Y (velocidad de movimiento de control) y la tasa de paralaje de Altura (velocidad de reducción de control).




Observación: El efecto paralaje está disponible para todos los tipos de controles. 


### Definir el paralaje a través de la programación
<a name="definir_paralaje_traves_programacion_ELTPARAGRAPHE000088"></a>Las siguientes propiedades se utilizan para definir el paralaje de una control a través de la programación: 



|   |   |
| --- | --- |
| [ParallaxRateHeight](../Proprietes/1000021921.md) | El **ParallaxRateHeight** Property se usa para determinar y especificar qué tan rápido se reduce un control cuando se mueve la barra de desplazamiento en la ventana, ventana interna o Page . |
| [ParallaxRateY](../Proprietes/1000021922.md) | El **ParallaxRateY** Property se usa para determinar y especificar qué tan rápido se mueve un control cuando se mueve la barra de desplazamiento en la ventana, ventana interna o Page . |

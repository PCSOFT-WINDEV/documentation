


## ¿Cómo evitar que se muestre la URL de las páginas de WEBDEV en el navegador?
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000101"></a>
Por defecto, cuando el usuario se conecta a un sitio desarrollado con WEBDEV (sitio dinámico en modo de sesión), la barra del navegador Address contiene una Address en el siguiente formato:  

- URL corto: www.mydomain.com/MyApp

- o la URL completa si el servidor web no acepta una URL corta: www.mydomain.com/WD28 0AWP/WD28 0AWP.exe/CONNECT/MyApp




Esta dirección puede ocultarse fácilmente al usuario encapsulándola en un iFrame de una página estática.

**Observaciones**: 

- Esta característica sólo afecta a los sitios dinámicos estándar en el modo de sesión. Esta característica no afecta a los sitios de AWP, los sitios PHP o los sitios estáticos.

- En el caso de los sitios AWP, el nombre del servidor de aplicaciones WEBDEV puede ocultarse en la URL de la página a través de la función [Reescritura de URL](../Editeurs/2030054.md).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000125"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000035"></a>

Para mostrar una URL sin cambios cuando se utiliza un sitio WEBDEV, simplemente muestre el sitio en el iFrame de una página estática. El Address que se muestra en el navegador es el Address de la página estática y el sitio WEBDEV se muestra en un iFrame.
<a name="NOTE2_2"></a>


### Implementación
<a name="implementacion_ELTPARAGRAPHE000042"></a>

Se puede crear el conjunto de marcos estáticos que albergará el sitio dinámico en el modo de sesión:

- por cualquier editor.

- por WEBDEV. En este caso, se recomienda utilizar un proyecto distinto del proyecto dinámico en modo de sesión para visualizar.




Ejemplo de código HTML para el conjunto de marcos estáticos a crear:


```html
<html>
<head><meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
<title>Site title</title>
<iframe src="/sitetostart" name="MySite" style="border:0;overflow:auto;width:100%;height:100%;
position:absolute;top:0;left:0;"></iframe>
</html>
```
En la mayoría de los casos, esta página HTML será la página definida por Default para el servidor Web (IIS, Apache, etc.)

Contacte con el administrador del servicio INTERNET para redirigir el http://www.mydomain.com Address a esta página.

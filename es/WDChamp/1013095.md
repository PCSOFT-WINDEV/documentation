## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000131"></a>
WINDEV gestiona los siguientes eventos de forma predeterminada (en orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Inicialización de la control Conferencia | Corre cuando se abra la ventana. \* |


(\*) Por defecto, la "Inicialización" Event de cada control se ejecuta según el orden en que se crearon las control en la ventana. Para modificar este orden de ejecución: 

1. En la pestaña "Ventana", en el grupo "Orden", haga clic en "Inicialización".

2. En la ventana mostrada, utilice las flechas de la derecha para modificar el orden en el que se inicializan los controles.

## Eventos del arroyo associated con el control Conferencia
<a name="eventos_del_arroyo_associated_con_control_conferencia_ELTTEXTE000155"></a>
WINDEV también permite ver los eventos específicos de la corriente associated con el control Conferencia. Para ver estos eventos, seleccione la opción "Código de flujo associated" en el menú contextual de control Conferencia.

| Event | Condición de ejecución |
| --- | --- |
| Inicialización (\*) | Ejecutar al abrir la ventana. |
| Solicitud de conexión de un nuevo flujo | Ejecutar cuando se detecta una nueva secuencia. |
| Fin de la conexión | Ejecutar cuando el corresponsal de la corriente se desconecta. |


Para obtener más información, consulte [WINDEV y los arroyos](../WDLang3/3039001.md).

## Event a la solicitud de conexión de un nuevo stream
<a name="event_solicitud_conexion_nuevo_stream_ELTTEXTE000179"></a>
El Event **"Solicitud de conexión de un nuevo flujo"** se ejecuta cuando un nuevo flujo intenta conectarse a la aplicación actual.

**Este Event toma un parámetro**.

Para manipular este parámetro, se declara automáticamente un Procedure en la "Solicitud de conexión de un nuevo flujo" Event.

Esta Procedure utiliza la siguiente sintaxis:


```wl
PROCEDURE NewStream(nStreamID)
```
Donde "nStreamID" es un entero Variable correspondiente al identificador del flujo detectado.

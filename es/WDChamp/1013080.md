


## Los aviones en WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000256"></a>
Los planos se utilizan para agrupar los controles de los elementos. Un plano es una "capa" de un elemento. Cada elemento puede estar formado por varios planos.

WEBDEV propone la gestión de planos para los siguientes elementos: 

- las páginas,

- las páginas internas,

- celdas, 

- las áreas de disposición,

- las ventanas emergentes,

- supercontroles,

- modelos de controles, 

- las plantillas de página. 




Los mandos se pueden posicionar en diferentes planos. Un solo plano puede ser mostrado a la vez en el editor de páginas y cuando se ejecuta el sitio.

Cada elemento control puede:

- pertenecen a un avión. Este control se mostrará cuando se active el plano correspondiente (lo que significa Visible).

- pertenecen a varios planos. Este control se mostrará cuando uno de los planos asociados al control esté habilitado (lo que significa Visible).

## Implementación: Permitiendo la gestión de planos
<a name="implementacion_permitiendo_gestion_planos_ELTTEXTE000280"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000036"></a>

Para utilizar los aviones en WEBDEV, la gestión de los aviones debe estar habilitada para el elemento deseado: 

1. Abrir la ventana del elemento description. 

2. En la pestaña "UI", seleccione la opción "Activar gestión de planos". 

3. Validar la ventana del elemento description. 

4. Los planos están disponibles para el elemento. Si la visualización de planos está permitida en el editor, la cifra 1 utilizada para identificar el primer plano aparece en la parte superior derecha del elemento.. 
	Observación: Para que se muestre el número del plano en el editor, en la pestaña "Vista", en el grupo "Mostrar (todos los controles)", seleccione la opción "Planos". 




Cuando se habilita la gestión de los aviones, se pueden crear los controles deseados y asociarlos a los diferentes planos. 
<a name="NOTE2_2"></a>


### Carga plana retardada
<a name="carga_plana_retardada_ELTPARAGRAPHE000068"></a>

Al activar los planos, se puede seleccionar la opción "Carga diferida del contenido de los planos".

Esta opción se utiliza para cargar los datos de los aviones más rápido. De hecho, si los aviones contienen muchos datos para mostrar, la página puede tardar bastante en cargarse. Puede mejorar esto cargando los datos sólo cuando se muestra el plano correspondiente: esto se llama "lazy loading". 

Si se selecciona la opción "Carga diferida del contenido de los planos" al activar los planos, se asocian dos nuevos eventos al elemento: 

- Aplazar la carga de un plano (servidor) en AJAX. 

- Aplazar la carga de un avión (Navegador). 




En estos eventos, se puede utilizar la [Tela impermeable](../Proprietes/2510094.md) Property para identificar el plano Active. Esto le permite ejecutar sólo la inicialización Process de los controles ubicados en el plano correspondiente.

Ejemplo de código: 


```wl
// Code for delayed load (server) of CELL_VARIOUS
SWITCH CELL_VARIOUS.Plane
	CASE 1 // Displaying the list of customers
   	ProcLoadCustomerTable()
	CASE 2 // Displaying the orders of the selected customer
   	CustomerID is int = TABLE_Customer.COL_CustomerNum
   	ProcLoadCustomerOrd(CustomerID)
	CASE 3 // Order details
	OrdNum is int = TABLE_Order.COL_OrderNum
	ProcLoadOrderDetails(OrdNum)
END
```

### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000099"></a>

- **Control Looper**: Puedes usar aviones para cada fila. Simplemente crea una célula para cada fila. Esta celda contiene los planos a visualizar. Se utiliza un atributo "Plane" para gestionar el número de plano visualizado.

- **control Tabla**: Tiene la capacidad de utilizar planos en una columna de contenedor, utilizando una celda, por ejemplo. 

- **Zoning**: 

	- Si los planos se gestionan a nivel de página, la zonificación se define para cada plano. 

	- Se pueden utilizar varios planos en cada área de disposición. 




- **Scrolling Banner control**: La pancarta deslizante control utiliza planos para mostrar los diferentes elementos de las pancartas. En este caso, la gestión de los aviones es específica. Para obtener más información, consulte [Control Banner deslizante](../WDChamp/1410087016.md).

## Operaciones en el editor
<a name="operaciones_editor_ELTTEXTE000316"></a>


### Asignación de uno o más controles a uno o más planos
<a name="asignacion_uno_mas_controles_uno_mas_planos_ELTPARAGRAPHE000117"></a>

**Caso 1: Creación de controles**: 
Cuando se crea un control en un elemento que contiene planos, el control creado se asocia automáticamente con el plano Visible. 

**Caso 2: Asociar controles existentes con uno o más planos**: 
Para asignar uno o más controles a uno o más planos: 

1. Seleccione los controles. 

2. Seleccione:

	- "Asociar .. a un plano" en el menú contextual de la selección. 

	- "Asociar .. a un plano" en el menú contextual de la control. 




3. En la ventana que se visualiza, seleccione: 

	- uno de los aviones disponibles.

	- varios planos (para ello, mantener pulsada la tecla Ctrl durante la selección de planos). 




4. Validar. 




**Observaciones**: 

- Un control posicionado en un elemento que contiene aviones es siempre associated con un avión. 
	No hay un plano 0, utilizado para hacer un control Visible independientemente del plano mostrado. Para gestionar este caso, todo lo que tiene que hacer es definir diferentes áreas, por ejemplo, un área que no contenga ningún plano y un área que contenga planos.. 

- Si un control es associated con varios planos, se convierte automáticamente en apilable. 

- El número de planos es ilimitado. Sin embargo, el editor de páginas puede impedirle seleccionar los planos superiores a 98.

- También puede asignar un control a un plano desde la ventana de descripción del control (pestaña "UI", opción "Plano").

- Al asignar una control a un plano, el plano al que se asigna la control aparece automáticamente en el editor. 

- La ventana de asignaciones lista los planos utilizados y luego los planos en blanco.. Los botones que se encuentran en la parte inferior de esta ventana simplifican el movimiento entre los diferentes tipos de aviones.



<a name="NOTE3_2"></a>


### Seleccionar el plano visualizado en el editor
<a name="seleccionar_plano_visualizado_editor_ELTPARAGRAPHE000152"></a>

Para visualizar un plano específico en el editor, puede hacerlo:

1. Seleccionar (si es necesario) el elemento para el que se deben visualizar los planos.. 

2. Presione las teclas PgDn y PgUp. Estas teclas se utilizan para mostrar los diferentes planos del elemento actual (o los diferentes planos de la control padre de la selección). 




Si la visualización de planos está permitida en el editor, el número del plano actual se muestra en la parte superior derecha del elemento..

## Cambio de plano a través de la programación
<a name="cambio_plano_traves_programacion_ELTTEXTE000346"></a>
La propiedad [Tela impermeable](../Proprietes/2510094.md) permite:

- averiguar y cambiar el plano actual de un elemento.

- averigua el avión associated con una control.



<a name="NOTE4_2"></a>


### Cambio de plano en un elemento
<a name="cambio_plano_elemento_ELTPARAGRAPHE000177"></a>

El cambio de plano en un elemento se puede realizar en cualquier momento. Una control Botón se utiliza a menudo para cambiar de avión. 

Ejemplo: 


```wl
// Switch to plane 4 for adding orders
PAGE_CRM.CELL_OrdCell.Plane = 4
```
El cambio de avión es inmediato: sólo los controles associated con el avión se muestran. 
<a name="NOTE4_3"></a>


### Efectos al cambiar de plano
<a name="efectos_cambiar_plano_ELTPARAGRAPHE000189"></a>

El cambio de plano se puede hacer con efectos. tiene la capacidad de definir efectos: 

- **para el elemento que gestiona los planos**. En la pestaña "Estilo" debe: 

	- seleccionar el elemento "Efectos. 

	- haz clic en "Añadir un efecto". 

	- seleccione un efecto para cambiar de avión y configúrelo mediante el botón "Detalles".. Tiene la posibilidad de especificar cómo se activará el efecto (cambio de plano, ir al plano anterior,...).




- **para los elementos encontrados en el plano**mediante la gestión de efectos específicos para la aparición o desaparición de controles. En la pestaña "Estilo" debe: 

	- seleccionar el elemento "Efectos. 

	- haz clic en "Añadir un efecto". 

	- seleccione un efecto de aparición o desaparición y configúrelo mediante el botón "Detalles".: se proponen varias animaciones.







Atención: Los efectos sólo funcionan si la página no se vuelve a mostrar. Para que el efecto sea visible, el [Tela impermeable](../Proprietes/2510094.md) Property debe ser llamado en el código del navegador o en el código del servidor Ajax.

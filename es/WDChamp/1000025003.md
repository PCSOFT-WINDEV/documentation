
## Propiedades associated con las barras de acción
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
Una "Action Bar" encontrada en una ventana puede ser manejada en WLanguage por las siguientes propiedades.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Propiedades disponibles en una ventana
<a name="propiedades_disponibles_una_ventana_ELTTEXTE000106"></a>


### Action Bar: Propiedades disponibles en una ventana
<a name="action_bar_propiedades_disponibles_una_ventana_ELTPARAGRAPHE000017"></a>Las siguientes propiedades son associated con "Barras de Acción":



|   |   |
| --- | --- |
| [BackgroundColor](../Proprietes/2510022.md) | La propiedad **BackgroundColor** permite establecer el color de fondo de un control, ventana, página o bloque de un reporte. |
| [BackgroundImage](../Proprietes/2510056.md) | La propiedad **..BackgroundImage** permite conocer y modificar la imagen de fondo de un elemento: ventana, página, control o reporte. |
| [Color](../Proprietes/2510071.md) | El **Color** Property se utiliza para get o cambiar la fuente Color para:<br><br>- una fuente Variable.<br><br>- un texto visualizado en un control Estático o en un control calculado de un reporte.<br><br>- un texto mostrado en una ventana control.<br><br>- un texto mostrado en una página control.<br><br><br> |
| [Extra](../Proprietes/1410089327.md) | La propiedad **Extra** permite:<br><br>- Obtener información adicional sobre un control, ventana o página.<br><br>- Editar información adicional de un control, ventana o página.<br><br><br> |
| [Image](../Proprietes/2510034.md) | El **Image** Property obtiene y establece la imagen de fondo de un control o bloque. |
| [Note](../Proprietes/2510085.md) | La propiedad **Note** permite:<br><br>- Encuentra las notas associated con un control, una ventana, un Page o un reporte.<br><br>- Modifica las notas associated con un control, una ventana un Page o un reporte.<br><br><br> |
| [NoteTitle](../Proprietes/2510121.md) | La propiedad **NoteTitle** permite:<br><br>- get el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte. <br><br>- Cambiar el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte.<br><br><br> |
| [Opacity](../Proprietes/2514006.md) | El **Opacidad** Property se utiliza para get efinir el porcentaje de opacidad de una ventana, o de los controles de una ventana o Page. |
| [Pane](../Proprietes/1000021098.md) | La propiedad **Pane** permite: <br><br>- Averigua las pestañas del panel associated con un control. <br><br>- Definir las pestañas del panel associated con un control.<br><br><br> |
| [Process](../Proprietes/1000020878.md) | El **Process** Property se utiliza para modificar los Process es que serán ejecutados por los eventos disponibles en un control, ventana, Page, reporte, etc. |
| [SearchValue](../Proprietes/1000021602.md) | El **SearchValue** Property se utiliza para get y cambiar el valor del campo de búsqueda en un Action Bar. |
| [Style](../Proprietes/1000021472.md) | La propiedad **Style** permite obtener y asignar el estilo completo de un control (incluyendo las sobrescrituras). |
| [Value](../Proprietes/2510130.md) | La propiedad **Value** permite:<br><br>- Averiguar y modificar el valor de un reporte control.<br><br>- Averiguar y modificar el valor de un control o el título de una ventana.<br><br>- Averiguar el identificador de una secuencia. <br><br>- Averiguar y modificar el valor de un control o el título de un Page.<br><br><br> |
| [Visible](../Proprietes/2510138.md) | El **Visible** Property se utiliza para:<br><br>- Averiguar si un elemento es Visible.<br><br>- Hacer visible un elemento o Invisible.<br><br><br> |
| [VisibleWithAnimation](../Proprietes/1410087232.md) | La propiedad **VisibleConAnimación** se utiliza para hacer visible o Invisible un control mediante la animación "Aparecer/Desaparecer". |




## Barra de estado de las ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000171"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000011"></a>

La barra de estado de una ventana permite mostrar varios tipos de información. Por ejemplo:

- un mensaje de información relacionado con el control actual. 
	Este mensaje se puede introducir en la pestaña "Ayuda" de la ventana de descripción del control. 
	El mensaje se mostrará en la barra de estado cuando el usuario seleccione el control.

- un mensaje de información específico. Este mensaje se muestra mediante programación con la función [Message](../WDLang1/3021006.md).

- una barra de progreso. Esta barra de progreso se muestra mediante programación con la función [ProgressBar](../WDLang1/3021008.md).




**Para Mostrar diferente información de forma simultánea**, la barra de estado se puede dividir en varias celdas.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000195"></a>


### Mostrar una barra de estado
<a name="mostrar_una_barra_estado_ELTPARAGRAPHE000036"></a>

Para agregar una barra de estado en la ventana actual, realice una de las siguientes acciones: 

- seleccione la opción "Mostrar barra de estado" en la pestaña "Estilo" de la descripción de la ventana.

- en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Barra de estado" y seleccione "Con barra de estado".

- utilice la propiedad [StatusBar](../Proprietes/2510007.md).



<a name="NOTE2_2"></a>


### Barra de estado con varias celdas
<a name="barra_estado_con_varias_celdas_ELTPARAGRAPHE000066"></a>

Puede crear una barra de estado con varias celdas mediante una de las siguientes opciones:

- desde la ventana de descripción de la barra de estado: en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Barra de estado" y seleccione "Editar".

- mediante el menú contextual de la barra de estado (opción "Insertar una celda").

- mediante programación con la función [StatusBarAddCell](../WDLang1/3021001.md).




Cada celda puede mostrar un tipo de información específico. Para obtener más información, consulte [Información en una barra de estado con varias celdas](../WDChamp/1010037.md).

## Ventana de descripción de la barra de estado
<a name="ventana_descripcion_barra_estado_ELTTEXTE000225"></a>
La ventana de descripción de una barra de estado permite:

- insertar y/o eliminar celdas, y modificar el orden de las mismas.

- especificar las características de cada celda (para obtener más información, consulte [Información en una barra de estado con varias celdas](../WDChamp/1010037.md)).

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenDescriptionBarreMessage.gif)
Para abrir la ventana de descripción de la barra de estado:

- haga doble clic en la barra de estado.

- seleccionar la opción "Descripción" en el menú contextual de la barra de estado.

- en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Barra de estado" y seleccione "Editar".

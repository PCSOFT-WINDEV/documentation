## Creación de una control Hoja de cálculo en un reporte
<a name="creacion_una_control_hoja_calculo_reporte_ELTTEXTE000128"></a>


### Crear control
<a name="crear_control_ELTPARAGRAPHE000011"></a>

Para crear una control Hoja de cálculo: 

1. En la pestaña "Creación", en el grupo "Datos", haga clic en "Hoja de cálculo". 

2. Haga clic en la ubicación en la que desea crear el control.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. 

**Observación**: Los datos mostrados por un control Hoja de cálculo en un reporte pueden: 

- determinarse mediante programación (por ejemplo, la visualización directa de un archivo xls o xlsx). 

- corresponden a una nota de un archivo de datos o a una consulta, ... 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) corresponden a una control Hoja de cálculo en una ventana. 

	- corresponden a una [xlsDocument](../WDLang5/1000017464.md) variable.

## Características de control Hoja de cálculo en un reporte
<a name="caracteristicas_control_hoja_calculo_reporte_ELTTEXTE000152"></a>
Las principales características de un control Hoja de cálculo en un reporte se definen en las diferentes pestañas de la ventana description del control: 

- Pestaña "General": La opción **Escalado** corresponde a la propiedad [AdjustmentMode](../Proprietes/1000021790.md). Las opciones disponibles son: 

	- Ninguna escala: La hoja de trabajo se imprime con sus dimensiones. La impresión se puede realizar en varias páginas de ancho y alto si es necesario.. 

	- Ajustar hoja de cálculo a las dimensiones del control: La hoja de trabajo se reducirá para que se muestre en su totalidad en el control Hoja de cálculo. 

	- Ajustar todas las columnas al ancho del control: Las columnas de la hoja de trabajo se reducirán o ampliarán para ajustarse al ancho del control. 

	- Ajustar todas las filas al alto del control: Las filas que se encuentran en la hoja de trabajo se reducirán o ampliarán para que todas las filas se muestren a la altura de control. 




- Pestaña "Binding" (encuadernación): Permite definir la vinculación de la control Hoja de cálculo. El control puede ser: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) vinculado a una control en una ventana de aplicación.

	- vinculado a una Variable. 

	- programado ("Ninguno" está seleccionado). Para obtener más información, consulte [Manejar una control Hoja de cálculo mediante programación (en una reporte)](../WDChamp/1000033010.md). 








### Manejo de una control Hoja de cálculo en una reporte mediante programación
<a name="manejo_una_control_hoja_calculo_una_reporte_mediante_programacion_ELTPARAGRAPHE000077"></a>

Puede asociar la hoja de cálculo de un reporte con: 

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md). Simplemente usa el código: 
	
	```txt
	<Spreadsheet control> = <xlsDocument variable>
	```


- un archivo xls/xlsx. Simplemente usa el código: 
	
	```txt
	<Spreadsheet control> = <Path of xls/xlsx file>
	```





Atención: Las funciones de gestión de control Hoja de cálculo no se pueden utilizar en el control Hoja de cálculo de un reporte.

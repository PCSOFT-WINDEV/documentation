
## Transparencia de las ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000107"></a>
WINDEV le permite aplicar un coeficiente de transparencia a las ventanas.

Esta característica es muy useful. Se utiliza para mostrar una ventana de alerta o de aviso "encima" de otra ventana, sin detener la Process actual.

Imaginemos una aplicación de supervisión: cada vez que se produce una alerta, se abre una ventana de Transparent que muestra un mensaje mientras la ventana que se encuentra debajo permanece activa y Visible.

Imaginemos cómo podría hacer un seguimiento de los correos urgentes, controlar las cadenas de montaje, ...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000131"></a>


### Modificación de la transparencia de las ventanas
<a name="modificacion_transparencia_las_ventanas_ELTPARAGRAPHE000024"></a>

La transparencia de las ventanas puede ser modificada:

- [en el editor de ventanas](#NOTE2_2).

- [mediante programación](#NOTE2_3).



<a name="NOTE2_2"></a>


### Modificar la opacidad de una ventana en edición
<a name="modificar_opacidad_una_ventana_edicion_ELTPARAGRAPHE000038"></a>

Modificar la opacidad de una ventana:

1. Abra el menú contextual de la ventana y seleccione "description".

2. Vaya a la pestaña "UI" y set el nivel de opacidad de la ventana:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Transparence%20-%20HC%20N%B0001.gif)



Este nivel puede cambiar de 0 (ventana de Transparent) a 100 (ventana opaca).
<a name="NOTE2_3"></a>


### Cambiar la opacidad de la ventana mediante programación
<a name="cambiar_opacidad_ventana_mediante_programacion_ELTPARAGRAPHE000051"></a>

También puede cambiar la opacidad de la ventana mediante programación con el programa [Opacidad](../Proprietes/2514006.md) Property.

Ejemplo: Mostrar efectos al cerrar una ventana:


```wl
// Make the window invisible little by little (fade out)
i is int
FOR i = 1 TO 100
	WIN_EditWindow.Opacity = 100 - I
END
Close()
```

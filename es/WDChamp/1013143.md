
## Características de una control indexada 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Descripción de una control indexada
<a name="descripcion_una_control_indexada_ELTTEXTE000093"></a>
Existe una ventana description para cada control encontrado en un control indexado . El nombre de la control es el mismo. Sólo cambia el índice control. Se pueden definir diferentes opciones para cada control que se encuentre en una control indexada .

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Diferencias con WINDEV 5.5
<a name="diferencias_con_windev_55_ELTTEXTE000117"></a>
**En la versión 5.5**, el índice se mostraba en la barra de estado del editor de ventanas. La ventana description era la misma para todos los controles indexados. Cada modificación realizada en la ventana description (ventana de 7 pestañas) se aplicó a todos los controles indexados.

**A partir de la versión 9**, el índice se muestra en la ventana de descripción. Cada control indexado tiene su propia description (ventana de 7 fichas) y sus propias características. Un control que se encuentra en un control indizado puede ser verde mientras que otro control en el mismo control indizado puede ser azul. Del mismo modo, dos control s pertenecientes a la misma control indexada pueden utilizar menús emergentes diferentes.

***Para modificar las características de todos los controles indizados en una sola operación***, debe:

1. Seleccione todos los controles indexados (utilizando el lazo, por ejemplo)

2. Muestra la ventana de description y realiza las modificaciones solicitadas.

## Código de la control indexada
<a name="codigo_control_indexada_ELTTEXTE000141"></a>
Todos los controles indexados con el mismo nombre son associated con los mismos procesos.

Por ejemplo, el código de inicialización del control indexado llamado Edit1[ 1] es el mismo que el código de inicialización del control indexado llamado Edit1[ 5].

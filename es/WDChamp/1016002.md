


## Manejo de una control Carrusel mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
WINDEV le permite manipular a control Carrusel mediante la programación. Para hacerlo, usa la Variable de la control Carrusel en el código.

Los controles Carrusel y ListView utilizan el mismo grupo de funciones ([ListaXXX](../WDLang1/3049001.md)).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Rellenar una control Carrusel
<a name="rellenar_una_control_carrusel_ELTTEXTE000159"></a>


### Asignar un valor a un elemento
<a name="asignar_valor_elemento_ELTPARAGRAPHE000023"></a>

Para asignar un valor inicial a un elemento en una control Carrusel, utilice [ListAdd](../WDLang1/3049004.md). 


```wl
// Add elements
ListAdd(CAR_Carousel, "Customer", "Customer.gif")
ListAdd(CAR_Carousel, "Order", "Order.gif")
```
**Observación**: El texto ("Customer" u "Order" en este ejemplo) se muestra en la esquina superior izquierda del control Carrusel cuando se pasa el cursor por encima del elemento.
<a name="NOTE2_2"></a>


### Rellenar una control Carrusel
<a name="rellenar_una_control_carrusel_ELTPARAGRAPHE000036"></a>Para asignar valores a una control Carrusel, utilice un bucle de iteración para inicializar todos los elementos de la control. Por ejemplo: 


```wl
i is int
FOR i = 1 to 52
	ListAdd(CAR_Carousel, Name[i], CarouselImage[i])
END
```

## Despejar una control Carrusel
<a name="despejar_una_control_carrusel_ELTTEXTE000189"></a>


### Cómo eliminar elementos de una control Carrusel
<a name="como_eliminar_elementos_una_control_carrusel_ELTPARAGRAPHE000047"></a>

Para eliminar todos los elementos, utilice [ListDeleteAll](../WDLang1/3049013.md).


```wl
// Remove all elements from the Carousel control
ListDeleteAll(CAR_Carousel)
```

## Propiedades específicas de los controles de carrusel
<a name="propiedades_especificas_los_controles_carrusel_ELTTEXTE000213"></a>
Las siguientes propiedades se utilizan para manipular una control Carrusel de forma programada.


|   |   |
| --- | --- |
| [Animación](../Proprietes/2510006.md) | Detener o reanudar la animación de la control Carrusel. |
| [InitialAnimation](../Proprietes/2510012.md) | Permite configurar la animación del control Carrusel cuando se inicializa. |
| [ObserverOrientation](../Proprietes/2515046.md) | Permite identificar y cambiar el ángulo secundario de una control Carrusel. Este ángulo secundario hace que la control Carrusel se "incline". |
| [Período de animación](../Proprietes/2510152.md) | Permite get y cambiar la velocidad de rotación de un control Carrusel. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Carrusel, consulte [Propiedades associated con un control Carrusel](../WDChamp/1016003.md).

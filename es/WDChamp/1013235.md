## Vista general de la control Paginación
<a name="vista_general_control_paginacion_ELTTEXTE000146"></a>
La control Paginación se utiliza para acceder rápidamente a los registros de una Tabla o control Looper mostrados en varias páginas.. El usuario web hace clic en el número de página para visualizar los registros correspondientes.. 


Un paginador se associated automáticamente con control s que no son de tipo Ajax (basados en un archivo de datos o poblados programáticamente) cuando el número máximo de filas por página es mayor que el número de filas en el control. 

También puede crear una control Paginación y asociarla con una Tabla o control Looper después.


### Creación de una control Paginación
<a name="creacion_una_control_paginacion_ELTPARAGRAPHE000019"></a>

Para crear una control Paginación: 

1. En la pestaña "Creación", en el grupo "Navegación", haga clic en "Paginación".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

Para asociar el localizador con una Tabla o control Looper existente: 

1. Abre la ventana de control Paginación description. 

2. Selecciona la tabla o control Looper en la pestaña "General".

## Notas
<a name="notas_ELTTEXTE000176"></a>


### Enviar el valor de los controles al servidor
<a name="enviar_valor_los_controles_servidor_ELTPARAGRAPHE000050"></a>

La opción "Enviar el valor de los controles al servidor" (pestaña "General" de la ventana description) se utiliza para realizar las siguientes operaciones cuando se utiliza un control Paginación:

- validar los controles de la página actual.

- devuelve el valor de estos controles al servidor.




A continuación, el servidor realiza la acción especificada.
<a name="NOTE2_2"></a>


### description de una control Tabla y una control Looper
<a name="description_una_control_tabla_una_control_looper_ELTPARAGRAPHE000062"></a>

Al describir una Tabla o control Looper, tiene la posibilidad de especificar el número máximo de filas por página:

- control Tabla: Pestaña"Detalles" en las características de control Tabla.

- Control Looper: Pestaña"General" en las características de control Looper.




Si el número de filas mostradas en la página es mayor que el número máximo especificado, se associated automáticamente un localizador con la Tabla o control Looper. La control Paginación aparece debajo de la Tabla y/o control Looper. 

**Observación**: Esta opción está disponible sólo para controles Tabla no-Ajax. controles Tabla en modo "Servidor + AJAX" utiliza una barra de desplazamiento.
<a name="NOTE2_3"></a>


### Active WEBDEV Pager
<a name="active_webdev_pager_ELTPARAGRAPHE000078"></a>

El localizador en modo AWP gestiona automáticamente el método de paso de parámetros, sin programación. 

Este tipo de paginador te permite reference todas las páginas de relacionado y evita tener que usar cookies para almacenar la búsqueda actual.

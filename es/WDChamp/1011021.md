
## Impresión de informes desde un sitio WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
Por Default, el "Imprimir" Icon permite al usuario de la Web imprimir la página actual del sitio (Ctrl + P). Sin embargo, dependiendo del diseño de la página, la impresión puede ser incompleta (líneas truncadas, imágenes demasiado grandes, etc.).

WEBDEV le permite imprimir fácilmente informes en sitios estáticos y dinámicos (en modo Sesión o AWP).

Un WEBDEV reporte se utiliza para dar formato a la información para que el usuario de la Web pueda get una impresión perfecta.

WEBDEV propone varios métodos para imprimir una reporte. Un reporte puede serlo:

- [mostrarse en el navegador del usuario Web en formato PDF](../WDChamp/1011015.md). La impresión final será gestionada por el software para visualizar el archivo PDF.

- [descargado en la el equipo del usuario de la Web en formato PDF](../WDChamp/1011005.md).

- [visualizado en el navegador del usuario Web en formato HTML](../WDChamp/1011051.md).

- [impreso en una de las impresoras conectadas al servidor web](../WDChamp/1011002.md) (sólo en los sitios de intranet).

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000159"></a>
Un control en una ventana o página puede ser associated con un carácter alfanumérico, una tecla o combinación de teclas ("atajo de teclado"), etc.

Este atajo permite acceder directamente a la página web de control.

**Observación**: Por defecto, puede set una "Hotkey" utilizando el texto de la control (carácter '&' antes de la letra a definir como hotkey). Para obtener más información, consulte ["y carácter"](#NOTE3_1).

### Asignar un atajo de teclado a una control en una ventana
<a name="asignar_atajo_teclado_una_control_una_ventana_ELTPARAGRAPHE000035"></a>

Para asignar un atajo de teclado a una control en una ventana: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

2. Ve a la pestaña "UI" y haz clic en el botón que aparece junto a "Atajo de teclado".

3. Seleccione las teclas que constituyen la tecla de acceso directo deseada. 
	El botón "Detección" se utiliza para detectar la tecla (o la combinación de teclas) utilizada en el teclado.

4. Valide.

## y carácter
<a name="caracter_ELTTEXTE000213"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Gestionar el carácter & en los controles de ventana
<a name="gestionar_caracter_los_controles_ventana_ELTPARAGRAPHE000072"></a>

Mediante Default, puede utilizar una tecla de acceso directo como atajo de teclado. Esta tecla de acceso directo se asigna utilizando el texto de la control (carácter '&' antes de la letra a definir como tecla de acceso directo).

El atajo associated con al control será Alt + Letra después de &.

Para que aparezca el carácter '&' en una leyenda, este carácter debe escribirse dos veces. Por ejemplo:

- "War&Peace" se mostrará como "War<u><u><u><u>P</u></u></u></u>eace" (con Alt + P como tecla de acceso directo).

- "War&&Peace" se mostrará como "War&Peace".




Para definir cómo se interpretará el carácter '&', utilice la función [InterpretarAmpersand](../Proprietes/2510151.md) Property.

Para que el carácter '&' se inserte automáticamente antes de la leyenda de los nuevos controles: 

1. Mostrar las opciones del editor de ventanas: en la pestaña "Inicio", en el grupo "Entorno", expanda "Opciones" y seleccione "Opciones del editor de ventanas". 

2. En la pestaña "Controles", marque "Tecla de acceso rápido automática al crear un control".

3. Validar.

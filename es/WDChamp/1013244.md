## Presentación
<a name="presentacion_ELTTEXTE000185"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten manipular varios tipos de imágenes animadas:

- Las **imágenes animadas de forma predeterminada**. Estas Image s tienen un formato animado de Image (GIF animado por ejemplo). Estas imágenes animadas se gestionan automáticamente. No se requiere ninguna configuración específica.
	

- El **imágenes animadas por WINDEV/WINDEV Mobile/WEBDEV**. Estas imágenes contienen todos los dibujos animados.






Esta ayuda Page explica cómo manipular las imágenes animadas por WINDEV, WINDEV Mobile y WEBDEV.

<a name="NOTE2_1"></a>


## Creando un Image animado por WINDEV/WINDEV Mobile/WEBDEV
<a name="creando_image_animado_por_windevwindev_mobilewebdev_ELTTEXTE000209"></a>
Para crear un Image animado: 

1. Crear un archivo Image.

2. En este archivo Image, inserte los diferentes dibujos de animación. Estos dibujos pueden organizarse en filas o en columnas, o en una combinación de filas y columnas.




Atención: Todos los dibujos deben tener el mismo tamaño y el espacio entre ellos debe ser idéntico.

### Usando un Image animado en una aplicación WINDEV o WINDEV Mobile
<a name="usando_image_animado_una_aplicacion_windev_windev_mobile_ELTPARAGRAPHE000081"></a>

Para usar un Image animado personalizado en una aplicación WINDEV o WINDEV Mobile:

1. Cree una control Imagen.

2. Abra la ventana de descripción del control (opción "Descripción" del menú contextual del control Imagen).

3. En la pestaña "General", seleccione el archivo que contiene la animación Image.

4. En la pestaña "Detalles" configure las opciones de animación. Elija el tipo de animación y configure las opciones de associated.




Las opciones pueden variar según el modo de visualización seleccionado.

Las opciones "one-way" y "alternate" permiten configurar el modo de lectura de los dibujos del fichero Image. Las imágenes se pueden leer de izquierda a derecha y de arriba hacia abajo ("dirección estándar") o en la "dirección inversa".

Es posible indicar:

- El tiempo de visualización de cada dibujo de la animación ("Duration/Image").

- El número de dibujos encontrados en el ancho y en el alto en el archivo de Image. Dependiendo del número especificado, la Image se dividirá automáticamente en tantos dibujos como sea necesario.




La animación se llevará a cabo mostrando cada dibujo en el tiempo regular interval.

## Programación de un Image animado
<a name="programacion_image_animado_ELTTEXTE000263"></a>
Varias funciones de WLanguage se utilizan para gestionar las imágenes animadas. Estas funciones son las siguientes:



|   |   |
| --- | --- |
| [AnimationGetParameter](../WDLang1/3017003.md) | Lee el valor de uno de los parámetros de animación definidos para una Image o control Estático. |
| [AnimationSetMode](../WDLang1/3017001.md) | Cambia el tipo de animación: <br><br>- para una Image en una control Imagen. <br><br>- para un texto en un control estático.<br><br><br> |
| [AnimationSetParameter](../WDLang1/3017002.md) | Cambia uno de los parámetros de animación usados para: <br><br>- un Image animado (en un control Imagen),<br><br>- un texto animado (en un control Estático).<br><br><br> |




<a name="NOTE4_2"></a>
Se utilizan varias propiedades de WLanguage para gestionar las imágenes animadas. Estas propiedades son las siguientes:



|   |   |
| --- | --- |
| [Animation](../Proprietes/2510006.md) | La propiedad **Animation** permite animar una imagen. |
| [InitialAnimation](../Proprietes/2510012.md) | La propiedad **InitialAnimation** determina si la animación de un control Imagen, Botón, Estático o Carrusel está activada cuando se crea el control. |


## Propiedades associated con las filas de un control Tabla TreeView
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
Las filas del TreeView controles Tabla en una ventana o en una Page pueden ser manejadas por propiedades WLanguage.

**Observación**: Para usar un fila de un control Tabla de Treeview con propiedades de WLanguage, use la siguiente notación: &lt;control Tabla TreeView&gt;[&lt;Índice del elemento&gt;]..&lt;Property&gt;.

<a name="NOTE2_1"></a>

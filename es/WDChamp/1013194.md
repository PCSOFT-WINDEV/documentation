## Presentación del control Pestaña
<a name="presentacion_del_control_pestana_ELTTEXTE000224"></a>
El control Pestaña permite mostrar diferentes controles al mismo tiempo. Cuando se activa una pestaña:

- los controles asociados a la pestaña se muestran en la ventana o página.

- los controles asociados a otras pestañas no se muestran,

- los controles que no están asociados a ninguna pestaña mantienen el mismo estado de visualización.

## Crear un control Pestaña
<a name="crear_control_pestana_ELTTEXTE000248"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para crear un control Pestaña:

1. En la pestaña "Creación", en el grupo "Contenedores", despliegue "Pestaña y asociados" y seleccione "Pestaña".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual. 

La ventana de descripción del control permite: 

- configurar las pestañas (etiqueta, Icon, etc.). 

- [definir el nombre de las pestañas](../WDChamp/1013196.md). 

- seleccionar [el modo de visualización de las pestañas](../WDChamp/1013274.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilizar [pestañas dinámicas](../WDChamp/1013299.md).

## Vocabulario asociado al control Pestaña
<a name="vocabulario_asociado_control_pestana_ELTTEXTE000282"></a>
Para manipular fácilmente un control Pestaña, se deben conocer los siguientes conceptos:

| denominar | Definition |
| --- | --- |
| lengüeta | Botón de tabulación utilizado para cambiar la vista Active al panel correspondiente. La pestaña puede contener una Image o una etiqueta. Esta etiqueta puede estar compuesta por varias líneas separadas por CR. |
| Panel de pestañas | Área que contiene los controles associated con la pestaña. |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Pestaña dinámica | ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Pestaña que muestra una ventana interna. Permite crear una interfaz similar a la de los navegadores de Internet:<br><br>- las pestañas se pueden "desacoplar" de la ventana principal.<br><br>- las pestañas se pueden reorganizar.<br><br>- se pueden cerrar y abrir pestañas.<br><br><br>Para obtener más información, consulte [Pestañas dinámicas](../WDChamp/1013299.md). |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Ficha Estática | ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Ficha estándar, a diferencia de las fichas dinámicas. |

## Limitaciones
<a name="limitaciones_ELTTEXTE000306"></a>
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Limitaciones en Java
<a name="limitaciones_java_ELTPARAGRAPHE000164"></a>

Las siguientes características no están disponibles en Java:

- imágenes de pestañas.

- imágenes del panel de pestañas.

- número de ayuda asociado a la pestaña.

- posición de las pestañas. Por Default, las pestañas se colocan en la parte superior de la control.

- modo de visualización de las imágenes. 

- pestañas dinámicas.

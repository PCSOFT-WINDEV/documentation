
## Botón desplegable
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000180"></a>
El botón desplegable es una control Botón que sirve para ampliar: 

- un menú emergente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bouton_fleche_Menu.gif)


- una ventana "Popup".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=bouton_fleche_popup.gif)





Este tipo de control Botón se utiliza para proponer un set de acciones en un solo botón. 

- Al hacer clic en el botón se ejecuta la elección de Default. 

- El clic en la flecha expande el menú o la ventana emergente.

## Implementación
<a name="implementacion_ELTTEXTE000204"></a>


### Crear un desplegable control Botón
<a name="crear_desplegable_control_boton_ELTPARAGRAPHE000027"></a>

Para crear un desplegable control Botón:

1. Cree un control Botón.

2. Abra la pestaña "UI" en la ventana del control description y seleccione el tipo de flecha. Las opciones disponibles son: 

	- No hay flecha (valor de Default): el control Botón es un botón estándar. 

	- Botón desplegable: el botón y la flecha realizan la misma acción. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=bouton_fleche.gif)
Es posible: 

		- Ejecutar el código de clic de la control Botón. 

		- ampliar el menú contextual associated con el control Botón. 

		- mostrar una de las ventanas de la aplicación (una ventana emergente que presenta un menú específico, por ejemplo). 




- Botón dividido: la control Botón y la flecha ejecutan dos acciones diferentes. 

	- El botón ejecuta su código. 

	- La flecha corre: 
						- el código del clic de la flecha. 
						- la acción especificada: 
						<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=bouton_fleche_distinct.gif)
Es posible: 

		- Ejecutar el código de clic de la control Botón. 

		- ampliar el menú contextual associated con el control Botón. 
										En este caso, es posible seleccionar la opción "Al hacer clic en el botón se ejecutará automáticamente la última opción de menú seleccionada mediante la flecha". 

		- mostrar una de las ventanas de la aplicación (una ventana emergente que presenta un menú específico, por ejemplo). 

3. Valide la ventana de descripción.

## Características y modo de funcionamiento de los botones desplegables
<a name="caracteristicas_modo_funcionamiento_los_botones_desplegables_ELTTEXTE000228"></a>


### Posición de la flecha
<a name="posicion_flecha_ELTPARAGRAPHE000070"></a>

Según el tamaño y la posición de la etiqueta del botón, o si hay una Icon en el control Botón, la flecha puede situarse a la derecha o debajo del control Botón.  


### Botón desplegable que abre una ventana emergente
<a name="boton_desplegable_que_abre_una_ventana_emergente_ELTPARAGRAPHE000075"></a>

Si su botón debe abrir una ventana, le recomendamos que utilice una ventana: 

- sin títulos,

- sin menú,

- con un tamaño de centro comercial,

- que utiliza una simple Border. 




Observación: Tiene la posibilidad de crear una ventana emergente directamente ("Blanco para ventana emergente" en la pestaña "Estándar" de la Nueva ventana asistente). 


### Botón de división que abre un menú y guarda la opción seleccionada
<a name="boton_division_que_abre_menu_guarda_opcion_seleccionada_ELTPARAGRAPHE000087"></a>

Cuando se crea un botón de división y se selecciona la opción "Al hacer clic en el botón se ejecutará automáticamente la última opción de menú seleccionada mediante la flecha", el control Botón tendrá el siguiente comportamiento: 

- la primera vez que se abra la ventana que contiene el botón, la etiqueta del botón mostrará la primera opción del menú contextual (etiqueta de la opción y Image). 

- cuando el usuario expande el el menú contextual y selecciona una opción: 

	- la etiqueta de la opción y la associated Image aparecen automáticamente en el botón.

	- se ejecuta el código de la opción seleccionada. 







Observación: si la opción de menú contiene una Image, WINDEV gestiona automáticamente los diferentes estados en la Image que se muestra en el botón en tiempo de ejecución. 




### Botón desplegable utilizado en una control Cinta de opciones
<a name="boton_desplegable_utilizado_una_control_cinta_opciones_ELTPARAGRAPHE000105"></a>

Si un control Botón está en el grupo de un control Cinta de opciones, la pestaña "UI" del control Botón permite configurar el comportamiento del botón cuando la cinta se reduce. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ruban_Champ_Bouton.gif)


Es posible: 

- evitar la reducción de la control. 

- reducirlo para que aparezca el botón Image y el texto. 

- redúzcalo para que aparezca el botón Image. 


Las dos últimas opciones son útiles para los botones que muestran una Icon y una etiqueta específicas. 


### Aspecto del botón desplegable
<a name="aspecto_del_boton_desplegable_ELTPARAGRAPHE000119"></a>

La pestaña "Estilo" de un botón desplegable de la ventana description permite definir las características del divisor. Puedes set: 

- la opacidad,

- el tipo de flecha (triángulo o cheurón),

- el divisor Color. 
	La opción "Transparent" define un botón desplegable sin separador. 


Estas opciones pueden ser set para un solo estado del botón o para todos los estados del botón.

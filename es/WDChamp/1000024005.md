


## Manejo de una control Zona multilínea mediante programación (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000065"></a>
WINDEV y WINDEV Mobile permiten manipular controles Zona multilínea mediante programación: 

- Un control Zona multilínea puede manipularse mediante programación usando el nombre del control. Esta página presenta las diferentes operaciones disponibles. 

- Hay [varias funciones WLanguage](../WDLang1/1000020220.md) a manipular las filas dinámicas de un control Zona multilínea.




En esta página de ayuda se explica cómo programar los controles de zona multilínea de manipular. 





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipular controles Zona multilínea mediante programación
<a name="manipular_controles_zona_multilinea_mediante_programacion_ELTTEXTE000118"></a>
**Para acceder a los controles de un fila estático de un control Zona multilínea**, basta con utilizar la siguiente sintaxis:


```txt
<Multiline Zone control>.<Control name>
```
Por ejemplo: 


```wl
MZ_General.STC_N_of_xxx = StringBuild("Number of Order Form: %1", ORDER.OrderNum)
```


**Para manipular los elementos que se encuentran en una fila dinámica de un control Zona multilínea**, utilice el índice de la fila a manipular. La sintaxis es la siguiente: 

```txt
<Multiline Zone control>[<Index>].<Control name> = <Value>
```


Por ejemplo:

```wl
nRow is int = MZ_Products.AddLine()
MZ_Products[nRow].STC_NameProdRow = stRow.sProdCap
MZ_Products[nRow].STC_NameProdRow.Color = LightBlue
MZ_Products[nRow].EDT_QtyRow = stRow.nQuantity
```


**Observaciones**: 

- El índice "1" corresponde a la primera fila del control Zona multilínea. Todas las filas están numeradas (filas estáticas y filas dinámicas). Si el control Zona multilínea contiene filas estáticas y dinámicas, se debe tener en cuenta el número de filas estáticas.

- Las filas dinámicas de un control Zona multilínea pueden manipularse usando diferentes [Funciones de WLanguage](../WDLang1/1000020220.md).

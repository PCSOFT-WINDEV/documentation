## Presentación
<a name="presentacion_ELTTEXTE000227"></a>
La pestaña "Series" sirve para definir las diferentes series utilizadas en la control Gráfico, así como la fuente de las etiquetas de las categorías.. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fuente de las etiquetas de categoría
<a name="fuente_las_etiquetas_categoria_ELTTEXTE000251"></a>
Las etiquetas de categoría son las etiquetas que representan las categorías mostradas en el gráfico. 

Se visualizan las etiquetas de categoría:

- debajo de las marcas horizontales para los gráficos Line y los gráficos de acciones (Candlestick y BarCharts),

- debajo del eje X para los gráficos de columnas y los gráficos MinMax.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)
Puede seleccionar: 

- la fuente: 

	- Rellenado mediante programación, 

	- Buscar realizado en un archivo de datos o consulta,

	- Columna de tabla, 
			

	- Control List Box, 
			

	- Array WLanguage. 
			




- los datos. La información a dar depende de la fuente seleccionada. 




Observación: En un reporte, las fuentes de datos disponibles son: 

- Rellenado mediante programación, 

- Búsqueda realizada en un archivo de datos o consulta,

- Array WLanguage, 

- Fuente de datos de reporte.

## Opciones de las series seleccionadas
<a name="opciones_las_series_seleccionadas_ELTTEXTE000299"></a>
Se pueden definir opciones específicas para cada serie seleccionada en el array de la serie: 

- **Tipo de serie**: Tipo de gráfico associated con la serie. Por Default, este parámetro corresponde al tipo de gráfico que se muestra en la control (definido en la pestaña "General"). 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Para crear cartas compuestas, tiene la posibilidad de elegir diferentes tipos de cartas para cada serie.

- **Color**: Color de serie. 

- **Opacidad**: Porcentaje de opacidad de la serie. 

- **Tipo de punto**: Forma del punto (círculo, cuadrado,...). Tienes la habilidad de no dibujar el punto. 

- **Navegar por la fuente de datos cada vez que se realice un dibujo para actualizar los valores.**: Esta opción se utiliza para actualizar los datos del gráfico para cada nuevo dibujo (visualización) del gráfico.. 

- **Usar escala secundaria**: La serie se visualizará utilizando la escala secundaria. 




Dependiendo del tipo de serie seleccionado (Line, gráfico de Área, Dispersión y Columna), las propiedades pueden ser definidas o anuladas según el tipo de gráfico definido en la pestaña "General".

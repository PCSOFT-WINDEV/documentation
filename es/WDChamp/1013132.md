## Presentación
<a name="presentacion_ELTTEXTE000189"></a>
Las imágenes se utilizan a menudo para ilustrar las páginas del sitio.

**Hay varios tipos de imágenes disponibles en WEBDEV para una página dinámica**. Cada tipo corresponde a un uso específico de Image.

El tipo de la imagen se elige en la pestaña "General" de la imagen description.

Se proponen cuatro tipos de imágenes:

- Estática Image.

- Dinámica Image.

- Imagen generada.

- De una base de datos.


Para utilizar las imágenes generadas y almacenadas en una base de datos en Linux, debe instalar la biblioteca QT (Core y GUI), versión 5 o posterior.

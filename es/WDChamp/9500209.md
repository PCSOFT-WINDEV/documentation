### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>El control "Peeling Corner" permite crear efectos visuales en una página presentando una Image parcialmente cubierta por otra. Cuando el control es pasado por el cursor del ratón, el Above Image se pliega progresivamente para hacer aparecer el Image de abajo, como un Corner pelado debajo del cual aparecería la página siguiente.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cornage1.gif)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cornage2.gif)

## Crear un control Esquina doblada
<a name="crear_control_esquina_doblada_ELTTEXTE000126"></a>
Para crear una control Esquina Doblada:

1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Imagen" y seleccione "Esquina doblada".

2. Haga clic en la ubicación en la que desea crear el control.

<a name="NOTE3_1"></a>


## Características de una control Esquina Doblada
<a name="caracteristicas_una_control_esquina_doblada_ELTTEXTE000150"></a>
La ventana description del control permite definir: 

- el Image que será descubierto por el Corner pelado (pestaña "General"). 

- la acción que se realizará al hacer clic en la control (pestaña "General"). Esta información es la misma que la definida para los controles de los botones. Para obtener más información, consulte [Seleccionar la acción asociada a un control](../WDChamp/1013074.md) y [elegir el tipo de acción associated con un control](../WDChamp/1013071.md). 

- la imagen utilizada para el efecto esquina doblada (pestaña "Estilo"). 

- la esquina descubierta de la Image (pestaña "Estilo").

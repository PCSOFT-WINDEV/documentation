## Presentación
<a name="presentacion_ELTTEXTE000240"></a>
Varios tipos de leyendas están soportadas en los gráficos. Los nombres de estas leyendas difieren según sus posiciones: 

- **Etiquetas de series**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
Todas las etiquetas de las series corresponden a la **leyenda del gráfico**.  

- **Etiqueta de categorías**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)


- **Título del gráfico**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Libelle_Graphe.gif)
 El título del gráfico puede incluir tres elementos:

	- las etiquetas de las categorías

	- el valor de las categorías

	- el porcentaje de categorías




- **Título del gráfico**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)


- **Título de los ejes**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titreaxe.gif)

## ¿Cómo utilizar los diferentes tipos de leyendas?
<a name="¿como_utilizar_los_diferentes_tipos_leyendas_ELTTEXTE000264"></a>


### Manejo de las etiquetas de las categorías
<a name="manejo_las_etiquetas_las_categorias_ELTPARAGRAPHE000036"></a>

Se visualizan las etiquetas de categoría:

- por debajo de las marcas horizontales para los gráficos de Line y de acciones (Candlestick y BarCharts),

- debajo del eje X para los gráficos de Columna y MinMax.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)


Las etiquetas de categoría no pueden utilizarse en los gráficos de dispersión.

Las etiquetas de las categorías son definidas por [grCategoryLabel](../WDLang3/3042015.md).

El tipo de letra de la etiqueta puede ser modificado por [grLabelFont](../WDLang3/3042012.md).

Las etiquetas de categoría definidas con [grCategoryLabel](../WDLang3/3042015.md) pueden visualizarse en las etiquetas de los gráficos circulares (*grShowLabel* constante utilizada con [grLabel](../WDLang3/3042046.md)).

**Caso especial: las etiquetas de las categorías de fecha u hora**
Un control Gráfico puede mostrar etiquetas de categoría de fecha. La opción "Escala de coordenadas X espaciadas en función del tiempo" (pestaña "General" de la ventana description de control Gráfico) se utiliza para get una distribución adecuada de las etiquetas. 

Si el control Gráfico muestra las coordenadas X en formato DateTime, un zoom realizado por el usuario en el gráfico se utilizará para get una escala de tiempo con varios niveles, para cada unidad de tiempo. 
<br>![La visualización y presentación del año, meses y días es automática](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=073-abscisses_temporelles.jpg)


Los niveles posibles son los siguientes: 

- Milisegundos, 

- Segundos, 

- Minutos, 

- Horas, 

- Días, 

- Meses, 

- Años. 




Por Default: 

- El nivel mínimo a visualizar se define en función del rango de datos.

- El nivel máximo se define utilizando dos parámetros:

	- El número máximo de niveles a mostrar en la carta (3 por Default). 

	- El nivel máximo permitido para la visualización (por Default: año).







Las siguientes funciones se utilizan para configurar los niveles visualizados: 



|   |   |
| --- | --- |
| [grTimeLevelMask](../WDLang3/1000023017.md) | Define la máscara de visualización para un nivel de escala de tiempo en un gráfico. |
| [grTimeLevelMax](../WDLang3/1000023016.md) | Fija el número de niveles para mostrar en la escala temporal de un gráfico, así como el nivel máximo. |

### Manipulación de las etiquetas de serie
<a name="manipulacion_las_etiquetas_serie_ELTPARAGRAPHE000100"></a>

Las etiquetas de la serie set corresponden a la leyenda de la carta. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)


Las etiquetas de las series no pueden utilizarse en los gráficos circulares.

Las etiquetas de las series se definen con [grSeriesLabel](../WDLang3/3042024.md). 

La fuente de la leyenda puede cambiarse con [grLegendFont](../WDLang3/3042037.md).  

[grLegend](../WDLang3/3042047.md) indica la presencia y la posición de la leyenda (es decir, todas las etiquetas de las series).

**Observación**: Para definir las etiquetas de las series en los gráficos de "dispersión", le recomendamos que utilice [grScatterSeriesLabel](../WDLang3/3042038.md).
<a name="NOTE2_3"></a>


### Manejo de la leyenda del gráfico
<a name="manejo_leyenda_del_grafico_ELTPARAGRAPHE000129"></a>

El título del gráfico puede incluir tres elementos:

- las etiquetas de las categorías,

- el valor de las categorías,

- el porcentaje de categorías.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Libelle_Graphe.gif)


Los títulos de los gráficos sólo pueden mostrarse en los gráficos circulares, de columnas y de columnas apiladas.

[grLabel](../WDLang3/3042046.md) se utiliza para mostrar los diferentes elementos de la leyenda del gráfico. Se visualizan diferentes elementos según el tipo de gráfico.

Las etiquetas de las categorías mostradas corresponden a las definidas por [grCategoryLabel](../WDLang3/3042015.md).

El valor de las categorías depende de los datos del gráfico.

El porcentaje de categorías se calcula automáticamente.

La pantalla de información puede ser formateada por [grMask](../WDLang3/3042034.md).
<a name="NOTE2_4"></a>


### Manejo del título del gráfico
<a name="manejo_del_titulo_del_grafico_ELTPARAGRAPHE000162"></a>

El título de la carta puede ser mostrado Above o debajo de la carta. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)


El contenido del título del gráfico se define con [grTitle](../WDLang3/3042056.md). Esta función también puede utilizarse para set la posición del título.

El tipo de letra utilizado para el título del gráfico puede ser modificado por [grTitleFont](../WDLang3/3042048.md).
<a name="NOTE2_5"></a>


### Manejo del título de los ejes del gráfico
<a name="manejo_del_titulo_los_ejes_del_grafico_ELTPARAGRAPHE000180"></a>

El título de los ejes del gráfico se puede mostrar para los diferentes ejes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titreaxe.gif)


El contenido del título de los ejes del gráfico está definido por [grAxisTitle](../WDLang3/3042057.md). Esta función también se puede utilizar para posicionar este título según el eje deseado. No se puede definir el posicionamiento y/o la orientación del título en relación al eje.

El tipo de letra utilizado para el título de los ejes del gráfico puede ser modificado por [grAxisTitleFont](../WDLang3/3042059.md).

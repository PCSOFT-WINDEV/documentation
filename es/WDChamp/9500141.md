
## Cálculos de la automatic en una control Tabla (AAF)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
En los controles de la Tabla y de la Tabla TreeView, el usuario final tiene la posibilidad de añadir cálculos de automatic. Estos cálculos están disponibles para todos los tipos de columnas. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mostrando (o no) un cálculo de automatic
<a name="mostrando_calculo_automatic_ELTTEXTE000155"></a>


### Mostrando un cálculo de automatic
<a name="mostrando_calculo_automatic_ELTPARAGRAPHE000016"></a>

Para mostrar un cálculo de automatic en una columna de tabla: 

1. Abrir el menú contextual de la columna solicitada. 

2. Seleccione "Cálculo automatic (suma, media, ...)" y luego seleccione el tipo de cálculo deseado: 

	- cifra

	- Número de elementos

	- Promedio

	- Mínimo

	- Máximo




3. El cálculo se realiza automáticamente y se añade debajo del tabla. Siempre es Visible. 




Atención: 

- El menú emergente utilizado para añadir o eliminar los cálculos de automatic no está disponible para todos los tipos de columnas. 

- El menú emergente para añadir o eliminar cálculos de automatic no está disponible en los controles de la tabla de "Acceso directo" con más de 1000 filas (para optimizar los tiempos de cálculo). 

- La ejecución de los cálculos automatic en un control Tabla (arriba o abajo del control y las pausas) es un Process sincrónico.





### Borrar un cálculo de automatic
<a name="borrar_calculo_automatic_ELTPARAGRAPHE000041"></a>

Para borrar un cálculo de automatic de una columna de tabla: 

1. Abrir el menú contextual de la columna solicitada. 

2. Seleccione "Cálculo automatic (suma, media, ...)" y luego seleccione el tipo de cálculo a eliminar: 

	- cifra

	- Número de elementos

	- Promedio

	- Mínimo

	- Máximo




3. El cálculo se borra automáticamente para la columna seleccionada.. 
	Atención: Si el cálculo existe para otras columnas, el cálculo Line todavía se mostrará debajo del tabla. 




Atención: El menú emergente utilizado para añadir o eliminar los cálculos de automatic no está disponible para todos los tipos de columnas.

## Programación
<a name="programacion_ELTTEXTE000185"></a>


### Proponiendo un cálculo de automatic
<a name="proponiendo_calculo_automatic_ELTPARAGRAPHE000069"></a>

Los cálculos automatic son propuestos por Default en el menú contextual de las columnas tabla (para los tipos de columnas que lo permiten). 

recordatorios: 

- Un cálculo de automatic puede ser mostrado en una tabla o control Tabla TreeView. Simplemente define el cálculo a realizar en la columna de la ventana de description. Para obtener más información, consulte [Cálculos en las columnas de un control Tabla](../WDChamp/1013201.md). 

- También se pueden añadir cálculos personalizados de automatic en las columnas de un control Tabla. Para obtener más información, consulte [Programación de cálculos personalizados en las tablas](../WDChamp/1013331.md).

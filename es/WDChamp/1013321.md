


## Manejo de una control Botón de opción mediante programación (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten programar los controles de los botones de radio de manipular. Para hacerlo, usa la Variable de control Botón de opción en el código.

Independientemente del tipo de radiobotón, cada opción puede ser associated con:

- ***un número entero ***.. Este número entero puede tener un valor de 1 a N, siendo N el número total de opciones que se encuentran en el botón de radio. La primera opción es la opción #1, la segunda opción es la opción #2, ... Las opciones están numeradas de arriba a abajo y de izquierda a derecha.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) ***a valor específico de retorno***.. Este valor devuelto se especifica en la pestaña "Contenido" de la ventana de control de description.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Utilizando la numeración o los valores devueltos?**
El método de programación de los botones de selección que devuelven un número entero difiere del método de programación de los botones de selección que devuelven un valor. El beneficio de usar los valores devueltos es Visible cuando se inserta o se mueve una opción en el botón de radio:

- ***Si el botón de opción devuelve un número entero***, insertar o mover una opción en el botón de opción obliga a revisar todo el código para gestionar el valor del botón de opción (para gestionar el cambio en los números).

- ***Si el botón de opción devuelve un valor específico***, la inserción de una opción en el botón de opción sólo requiere tener en cuenta la nueva opción.. Mover una opción no requiere ninguna modificación.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Varias funciones de WLanguage se utilizan para añadir, insertar o borrar opciones en un control Botón de opción. Para obtener más información, consulte [Funciones de gestión de control Botón de opción](../WDLang1/1000021190.md).

## Inicializar la elección de una control Botón de opción
<a name="inicializar_eleccion_una_control_boton_opcion_ELTTEXTE000162"></a>
Por Default, la primera opción se selecciona cuando se muestra un control Botón de opción.

**Para seleccionar una opción de un control Botón de opción**: 

- si el control Botón de opción devuelve un número entero, utilice la siguiente sintaxis:
	
	```txt
	<Radio Button control> = <Option index>
	```


- si el control Botón de opción devuelve un valor específico (sólo en WINDEV o WINDEV Mobile), utilice la siguiente sintaxis:
	
	```txt
	<Radio Button control> = <Option value>
	```



La selección de una opción desmarca la opción que está seleccionada actualmente.

**Observación**: Si el control es associated con el elemento un archivo de datos, la asignación se realiza automáticamente mediante: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) la función [.fromFile](../WDLang4/1000021387.md). 


Para obtener más información, consulte [Vinculación de una control Botón de opción a un elemento](../WDChamp/1013250.md).<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Mostrando un control Botón de opción sin la opción Default
<a name="mostrando_control_boton_opcion_sin_opcion_default_ELTPARAGRAPHE000231"></a>

Un control Botón de opción siempre se coloca en una posición (1 por Default).

Para mostrar una control Botón de opción sin la opción Default, todo lo que tienes que hacer es asignar el valor inicial de control a:

- -1 para una control Botón de opción que devuelve un entero,

- "" (cadena vacía) para una control Botón de opción que devuelve un valor.




**Observación**: Un control Botón de opción sin valor Default no puede ser accedido por TAB.

**Ejemplo**:


```wl
// Display the Radio Button control without initial value
RADIO_MyRadioButton = -1 
// or RADIO_MyRadioButton = ""
```

## Recuperar la opción seleccionada en un control Botón de opción
<a name="recuperar_opcion_seleccionada_control_boton_opcion_ELTTEXTE000247"></a>


### Caso de una control Botón de opción que devuelve un entero
<a name="caso_una_control_boton_opcion_que_devuelve_entero_ELTPARAGRAPHE000270"></a>

**Para get el índice de la opción seleccionada** en el control Botón de opción, utilice la siguiente sintaxis: 

```txt
<Option index> = <Radio Button control>
```

En este caso, el parámetro &lt;Índice de opción&gt; es un entero que corresponde al número de la opción seleccionada.

**To busque el título de la opción seleccionada** en el control Botón de opción, utilice la siguiente sintaxis:

```txt
<Option caption> = <Radio Button control>[Radio Button control].Caption
```

**Observación**: Si el control es associated con un elemento encontrado en un archivo de datos, el valor tecleado se recupera automáticamente utilizando: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) la función [.ToFile](../WDLang4/1000021385.md).


Para obtener más información, consulte [Enlace de un control Botón de opción a un artículo](../WDChamp/1013250.md).<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Para una control Botón de opción que devuelve un valor
<a name="para_una_control_boton_opcion_que_devuelve_valor_ELTPARAGRAPHE000315"></a>

**To recupera el valor de la opción seleccionada** en el control Botón de opción, utilice la siguiente sintaxis:

```txt
<Option value> = <Radio Button control>
```

En este caso, el parámetro &lt;Valor de la opción&gt; es una cadena de caracteres que corresponde al valor devuelto de la opción seleccionada.

**Observación**: El [ReturnedValue](../Proprietes/2510134.md) Property también se utiliza para obtener y set el valor devuelto por cada opción del control Botón de opción.

**Para obtener el índice de la opción seleccionada**, se debe realizar un Process específico. Ejemplo:

```wl
// Value returned by the selected option
Result1 = RADIO_TodaysDish
// Find the index of selected option
Index is int
I is int
FOR I = 1 _TO_ RADIO_TodaysDish.Count
	IF RADIO_TodaysDish[I].ReturnedValue = Result1 THEN Index = I
END
```


**To busque el título de la opción seleccionada** en el control Botón de opción, utilice la siguiente sintaxis:

```txt
<Option caption> = <Radio Button control>[Option index].Caption
```
**Observación**: Si el control es associated con un elemento encontrado en un archivo de datos, el valor tecleado se recupera automáticamente utilizando: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) la función [.ToFile](../WDLang4/1000021385.md).


Para obtener más información, consulte [Enlace de un control Botón de opción a un artículo](../WDChamp/1013250.md).<a name="NOTE3_3"></a>

## Modificar el título de una control Botón de opción
<a name="modificar_titulo_una_control_boton_opcion_ELTTEXTE000382"></a>
La propiedad [Pie de foto](../Proprietes/2510053.md) permite:

- **cambiar el título del control Botón de opción**: 
	
	```txt
	<Name of Radio Button control>.Caption = <Caption>
	```


- **cambiar el título de todas las opciones que se encuentran en control Botón de opción**:
	Las opciones deben estar separadas por el carácter TAB.
	Sólo puede especificar las primeras opciones.
	
	```txt
	<Radio Button control>.Caption = <Option 1>+TAB+<Option 2>+TAB+<Option 3> ...
	```


- **recuperar y modificar el título de una opción**:
	
	```txt
	// Get the sub-caption
	<Sub-caption> = <Radio Button control>[<Option index>].SubCaption
	
	// Modify the sub-caption
	<Radio Button control>[<Option index>].SubCaption = <New sub-caption>
	```

### Caso de los controles de los Botones de radio con subtítulos
<a name="caso_los_controles_los_botones_radio_con_subtitulos_ELTPARAGRAPHE000432"></a>

El [SubCaption](../Proprietes/1000017232.md) Property se utiliza para get y cambiar los subtítulos de las opciones en un control Botón de opción.


```txt
// Get the sub-caption
<Sub-caption> = <Radio Button control>[<Option index>].SubCaption

// Modify the sub-caption
<Radio Button control>[<Option index>].SubCaption = <New sub-caption>
```

## Cambiar los colores de los subtítulos de las opciones en una control Botón de opción
<a name="cambiar_los_colores_los_subtitulos_las_opciones_una_control_boton_opcion_ELTTEXTE000442"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Puede utilizar la siguiente sintaxis para cambiar los colores de los subtítulos de las opciones de una control Botón de opción: 

```txt
<Radio Button control>.Caption = gPen(<Color of option 1>) + ...
	<Caption of option 1> + TAB + ...
	gPen(<Color of option N>) + <Caption of option N>
```
&lt;Color de la opción&gt; corresponde a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

## Propiedades específicas de los controles de los botones de radio
<a name="propiedades_especificas_los_controles_los_botones_radio_ELTTEXTE000485"></a>
Las siguientes propiedades se utilizan para gestionar mediante programación las características de un control Botón de opción.


|   |   |
| --- | --- |
| [BackgroundColor](../Proprietes/2510022.md) | Se utiliza para averiguar y modificar el fondo Color del control Botón de opción. |
| [Color](../Proprietes/2510071.md) | Se utiliza para averiguar y modificar el Color del texto para las opciones que se encuentran en el control Botón de opción. |
| [Conde](../WDLang1/3025009.md) | Permite conocer el número de opciones de una control Botón de opción. |
| [HorizontalAlignment](../Proprietes/2510023.md) | Sirve para conocer y modificar la alineación horizontal de una control. |
| [InitialValue](../Proprietes/2510128.md) | Permite conocer el valor inicial de una control Botón de opción. |
| [NumberColumn](../Proprietes/2510083.md) | Permite conocer el número de columnas de un control Botón de opción. |
| [ReturnedValue](../Proprietes/2510134.md) | Permite conocer y modificar el valor devuelto por una opción que se encuentra en un control Botón de opción. |
| [SubCaption](../Proprietes/1000017232.md) | Permite conocer y modificar el subtítulo de una opción de un control Botón de opción. |
| [VerticalAlignment](../Proprietes/2510054.md) | Se utiliza para averiguar y modificar la alineación vertical de un control. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Botón de opción, consulte [Propiedades associated con un control Botón de opción](../WDChamp/1013253.md).

## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
WEBDEV le permite crear sitios estáticos y/o sitios dinámicos.

Estos sitios pueden combinar varios tipos de páginas:

- [páginas estáticas](#NOTE2_1).

- [páginas dinámicas (en modo Sesión o AWP)](#NOTE4_1).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Página estática
<a name="pagina_estatica_ELTTEXTE000137"></a>
**Características**
Una página estática se utiliza para mostrar texto e imágenes, ... Este tipo de página no permite la interacción real con el usuario.. 
Para visualizar varios formularios de producto, debe crear tantas páginas como formularios de producto numéricos.. Los datos se guardan en la página.

**Desarrollo**
Las páginas estáticas se crean en el editor WEBDEV.

**Instalación**
En el servidor, simplemente copy la página al servidor Web.

## Página dinámica (en modo Sesión o AWP)
<a name="pagina_dinamica_modo_sesion_awp_ELTTEXTE000161"></a>
**Características**
Una página dinámica le permite mostrar texto, imágenes y datos provenientes de una base de datos., ...
Una sola página dinámica es suficiente para ver varios formularios de productos: los datos de la página se rellenarán según el contenido de la base de datos y según el producto solicitado por el usuario de la Web. No se graba ningún dato de la base de datos en la página.

**Desarrollo**
Las páginas dinámicas se crean en el editor WEBDEV.

**Instalación**
En el servidor, debe instalar:

- el motor WEBDEV: este motor muestra las páginas HTML con los datos encontrados en la base de datos. 

- la base de datos.

- las páginas HTML generadas por WEBDEV.

## Aspecto del gráfico
<a name="aspecto_del_grafico_ELTTEXTE000337"></a>
Las características relativas a la apariencia de la carta son las siguientes: 

- **Suavizar efectos de dientes de sierra (Antialiasing)**: le permite usar el antialiasing si GDI+ está instalado. 
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta opción no está disponible.

- **Efecto de luz realista**: permite utilizar un efecto de luz en el gráfico. Si esta opción está seleccionada, el porcentaje de biselado se utiliza para configurar la inclinación de la luz.. 
	

- **Colores de borde automáticos**: Si esta opción está marcada, los colores para la Border de la serie se calculan automáticamente. 

- **Colores degradados**: Permite utilizar los colores de Gradient en la visualización de series. La Gradient es vertical por defecto, pero tiene la posibilidad de utilizar una Gradient horizontal. 

- **Mostrar también etiquetas superpuestas**: Esta opción permite apilar las etiquetas en las tablas Pie, Semicircular y Donut., ... 
	

- **Mostrar etiquetas fuera de los sectores**: Esta opción permite que las etiquetas se posicionen fuera de las secciones de las tablas Pie, Semicircular y Donut., ... 
	

- **Mostrar valores**: Se utiliza para visualizar los valores además de los porcentajes en las etiquetas.. Usted tiene la capacidad de definir la máscara de visualización que se utilizará. 
	El **Mostrar valores verticalmente** muestra los valores verticalmente.

## Zoom y retícula (no disponible en los informes)
<a name="zoom_reticula_disponible_los_informes_ELTTEXTE000361"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Gestión del zoom
<a name="gestion_del_zoom_ELTPARAGRAPHE000118"></a>

Habilitar el zoom permite al usuario realizar un zoom sobre los datos mostrados en el gráfico.

- **Permitir zoom horizontal**: permite activar el zoom de los valores en función del eje horizontal. La barra de desplazamiento del zoom se sitúa debajo de la control. 

- **Permitir zoom vertical**: permite activar el zoom de los valores en función del eje vertical. La barra de desplazamiento del zoom se sitúa a la derecha de control. 



![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Gestión del punto de mira
<a name="gestion_del_punto_mira_ELTPARAGRAPHE000136"></a>

El punto de mira permite al usuario conocer con precisión el valor apuntado. Para obtener más información, consulte [Cruce de líneas en un gráfico](../WDChamp/3042078.md). 

La retícula se habilita a través de "Activar mira del gráfico".

La opción "Mira múltiple (líneas y columnas apiladas)" permite utilizar múltiples cruces en los gráficos Line o de columnas.

Observación: El punto de mira está disponible para las siguientes clases de gráficos: Área, Burbuja, Line, Columna, MinMax, Radar y Cascada.

## Título y leyenda
<a name="titulo_leyenda_ELTTEXTE000391"></a>
El título y la leyenda se pueden configurar en un control Gráfico. 

El botón "Editar título del gráfico" sirve para mostrar la ventana del título del gráfico description. Esta ventana permite: 

- Escriba (si es necesario) el título del gráfico. 

- Configurar su posicionamiento. Si se selecciona un posicionamiento manual, tiene la posibilidad de mover el área de título en el editor. 

- Configurar el estilo utilizado para mostrar el título del gráfico. 




El botón "Editar leyenda del gráfico" permite visualizar la ventana de la leyenda del gráfico description. Esta ventana permite: 

- Definir el posicionamiento de la leyenda. 
	Este posicionamiento puede ser automatic o manual. 
	Si se selecciona un posicionamiento manual, se puede: 

	- mover el área de leyenda en el editor. 

	- elegir la posición de las leyendas (en fila o en la columna)

	- adaptar el tamaño de la leyenda al contenido




- Definir el aspecto de la leyenda: Se puede utilizar una Border redondeada para la leyenda. 

- Definir el estilo de la leyenda (fuente, ....).

## Animaciones y fondos Image
<a name="animaciones_fondos_image_ELTTEXTE000415"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Animaciones
<a name="animaciones_ELTPARAGRAPHE000199"></a>Algunos gráficos pueden ser animados. 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para los gráficos 2D, tiene la posibilidad de habilitar (o no) las animaciones. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para los gráficos 3D, se pueden configurar varias opciones: 

- Acercar, 

- Construcción progresiva, 

- Rotación,

- Colores progresivos. 




Para obtener más información, consulte [Animación de control Gráfico](../WDChamp/1013296.md). 


### Imagen de fondo
<a name="imagen_fondo_ELTPARAGRAPHE000230"></a>La Background Image aparecerá en el gráfico Background. 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para este Image se puede utilizar el modo de escalado 9-rango de resolución.

## Presentación
<a name="presentacion_ELTTEXTE000066"></a>
WINDEV, WINDEV Mobile y WEBDEV permiten crear interfaces personalizadas con el editor de ventanas y el editor Page.

El editor de ventanas y el editor Page te permiten:

- crear las ventanas y las páginas de su aplicación.

- crear controles en sus ventanas y páginas. Estos controles permitirán a los usuarios visualizar e introducir los datos de la aplicación.. Para todos los tipos de controles, se pueden configurar varias características en el editor o mediante programación. 




Además, WINDEV, WINDEV Mobile y WEBDEV le permiten crear fácilmente informes que agrupan toda la información necesaria.. La editor de reportes propone diferentes tipos de informes y diferentes tipos de controles para crear sus impresiones.


## Ejemplo de Flexbox: ratios para distribuir el espacio disponible entre los controles
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000088"></a>
En un contenedor Flexbox, puede definir las proporciones de los distintos controles para que ocupen un espacio determinado.

Con el Flexbox control, puede set una proporción para cada control. Este ratio se aplicará a medida que se amplíe el Page.

En el siguiente ejemplo, hay 4 áreas de entrada (controles de edición) dentro de una control Looper.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple2%20-%20HC%20N%B0005.gif&type=thumb)
 Cuando el navegador se amplía horizontalmente, las 2 primeras áreas de entrada deben permanecer con una anchura fija y el espacio extra debe distribuirse de la siguiente manera: 20 % para la tercera zona (Nombre completo), 80 % para la cuarta zona (description). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple2%20-%20HC%20N%B0004.gif&type=thumb)


Estas anclas se pueden definir con un Flexbox control.

Observación: Esta Page se basa en el ejemplo "WW_Flexbox".





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000112"></a>
A los anclajes específicos de set en las zonas de entrada dentro de control Looper: 

1. Crear un Flexbox control. Este Flexbox control debe estar dentro de un fila del control Looper y contener las diferentes áreas de entrada (Editar control s).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple2%20-%20HC%20N%B0001.gif&type=thumb)


2. Los 4 controles de edición aparecen en la pestaña "General" del control Flexbox description. Para get el comportamiento de redimensionamiento deseado, basta con especificar un valor flex-grow para el tercer y cuarto controles Edit. En nuestro caso: 

	- la tercera Edición control tiene un valor de flexión de 20. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple2%20-%20HC%20N%B0003.gif&type=thumb)


	- la cuarta Edición control tiene un valor de flexión del 80. 

3. No es necesario realizar cambios en la pestaña "Detalles"..

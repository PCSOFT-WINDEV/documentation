
## Manejo de una control Interruptor mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000148"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten manipular a control Interruptor mediante la programación. Simplemente usa la Variable de la control Interruptor en el código.

El Variable del control Interruptor corresponde al nombre del control. Este Variable es un número entero.

Esta página de ayuda explica cómo manipular un control Interruptor mediante programación. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>




## Inicialización de una control Interruptor
<a name="inicializacion_una_control_interruptor_ELTTEXTE000172"></a>
El control Interruptor es un control que propone sólo dos opciones: ON y OFF. La primera opción del interruptor corresponde a OFF y la segunda a ON.. 

Por defecto, cuando se muestra, el control Interruptor es set a OFF. 

Para set el cambio a ON, puedes usar: 

- la asignación directa. En este caso, basta con especificar el valor devuelto correspondiente a la elección deseada mediante la programación. Este valor ha sido definido en la pestaña "General"' de la ventana de control description. 
	
	```txt
	<Switch control> = 1 // If the value returned is 1
	<Switch control> = True // If the value returned is True
	<Switch control> = "Option 2" // If the value returned is Option 2
	```


- la [Valor](../Proprietes/2510130.md) property. 
	
	```txt
	Switch control>.Value = 1 // If the value returned is 1
	Switch control>.Value = True // If the value returned is True
	Switch control>.Value = "Option 2" // If the value returned is Option 2
	```





**Observación**: Si el control es associated con el elemento un archivo de datos, la asignación se realiza automáticamente mediante: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la función [FileToScreen](../WDLang4/3044210.md).

## Recuperar el valor de una control Interruptor
<a name="recuperar_valor_una_control_interruptor_ELTTEXTE000196"></a>
Para recuperar el valor de un control Interruptor, utilice la siguiente sintaxis:


```txt
<Control value> = <Switch control>
```


En este caso, &lt;Valor de control&gt; corresponde al valor devuelto por cada una de las opciones de la ventana de control description. 

El valor devuelto por cada opción (OFF y ON) puede cambiarse mediante programación con el programa [ReturnedValue](../Proprietes/2510134.md) Property. Por ejemplo: 

```txt
Switch control>[1].ReturnedValue = <Value for option OFF
Switch control>[2].ReturnedValue = <Value for option ON
```
Recordatorio: La primera opción del interruptor corresponde a OFF y la segunda a ON.. 

**Observación**: Si el control es associated con elemento un archivo de datos, el valor introducido se recupera automáticamente a través de: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la función [ScreenToFile](../WDLang4/3044146.md).

## Propiedades específicas de los controles de Switch
<a name="propiedades_especificas_los_controles_switch_ELTTEXTE000220"></a>
El control Interruptor puede ser manipulado a través de las propiedades WLanguage. 

Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Interruptor, consulte [Propiedades associated con un control Interruptor](../WDChamp/1410087190.md).

Observación: para cambiar los colores o la apariencia del control al mover el cursor, es necesario configurar el estilo de los elementos de control (pestaña "Style" de la ventana de control description).

## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000231"></a>


### Eventos soportados por WINDEV y WINDEV Mobile
<a name="eventos_soportados_por_windev_windev_mobile_ELTPARAGRAPHE000031"></a>

Los siguientes eventos son propuestos por Default (en orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Inicialización | Corre cuando se abra la ventana. \* |
| Haga clic en un enlace | Se ejecuta cuando se hace clic en un enlace en la pantalla HTML control. Sólo los enlaces externos (a otra página) activan esta Event. Los enlaces internos (a un ancla en la misma página, por ejemplo) no activan esta Event. <br><br>Este Event acepta parámetros: <br><br><pre><code>PROCEDURE ClickLink(sURL is string, bInNewWindow is boolean, <br>	sDestination is string)</code></pre><br>donde &lt;sURL&gt; es una cadena opcional que contiene la URL del enlace pulsado en la pantalla HTML control. <br><br>Si este evento devuelve <u><u><u><u>False</u></u></u></u>, la acción asociada al enlace no se realizará.<br> <br>Atención: el código no debe contener Process es o Process lentos que permitan al control volver a centrarse. En este caso, la acción de Default seguirá realizándose.<br><br>Ejemplo de código: <br><br><br><pre><code>// Open links with a new window as destination, <br>// in another specific window of the application<br>PROCEDURE ClickLink(sURL is string, bInNewWindow is boolean, <br>	sDestination is string)<br>IF bInNewWindow THEN<br>	// Open another requested window: <br>	// opening process of the other window, <br>	// delayed so that the current process responds <br>	// before opening the new window<br>	OpenNewBrowser(sURL)	<br>	RESULT False // Overrides default behavior<br>ELSE<br>	RESULT True // Keeps default behavior<br>END<br><br>INTERNAL PROCEDURE DisplayNewBrowser(sURLParam) <Delay=10ms> <br>	// Important: specify a delay, even if it is very small<br>	<COMPILE IF ConfigurationType = Android OR ConfigurationType = iOS><br>		OpenMobileWindow(WIN_POPUP, sURLParam) <br>	<ELSE>	<br>		OpenChild(WIN_POPUP. sURLParam) <br>	<END><br>END</code></pre><br> |
| Antes de cargar la página HTML | Ejecutar antes de cargar el HTML Document en la pantalla HTML control. Si el usuario navega utilizando los enlaces en el HTML Display control, esta Event se ejecuta una vez que se ha cargado cada HTML Document mostrado en la control. <br><br>Este Event acepta parámetros: <br><br><pre><code>PROCEDURE ChangePage(sURL is string, sDestination is string)</code></pre><br>donde &lt;sURL&gt; es una cadena opcional que contiene el URL de la página a mostrar. <br><br>Si esta Event devuelve <u><u><u><u>False</u></u></u></u>, la página no se cargará. |
| Después de cargar la página HTML | Ejecutar después de que el HTML Document se haya cargado en el HTML Display control. Si el usuario navega utilizando los enlaces en el HTML Display control, esta Event se ejecuta una vez que se ha cargado cada HTML Document mostrado en la control. |
| Cargar los recursos | Permite que se le notifique e impide que se carguen scripts o imágenes de la aplicación (con fines de seguridad, por ejemplo). <br><br>Este Event acepta parámetros: <br><br><pre><code>PROCEDURE LoadResources(sURL is string, nType is int)</code></pre><br> donde: <br><br>- &lt;sURL&gt; es una cadena de caracteres que corresponde al URL de la Resource a cargar.<br><br>- &lt;nTipo&gt; es una constante utilizada para identificar el tipo de Resource que se va a descargar: <br><br>	- *typehtmlImageRes*: la Resource a descargar es una Image. <br><br>	- *typehtmlScriptRes*: la Resource para descargar es un guión. <br><br>	- *typehtmlCssRes*: la Resource a descargar es un estilo CSS.  <br><br>	- *typehtmlXMLHttpRequestRes*: la Resource a descargar es una Document XML generada como respuesta a una petición HTTP. <br><br>	- *typehtmlOther*: la Resource a descargar es de un tipo desconocido. <br><br><br><br><br><br><br><br>Si este evento devuelve <u><u><u><u>False</u></u></u></u>, el Resource no se cargará. |
| Clic derecho en un enlace | Se ejecuta cuando el usuario hace clic con el botón derecho en un enlace (para abrir un menú contextual). Este Event tiene el siguiente formato: <br><pre><code>PROCEDURE RightClickLink(sURL is string)</code></pre><br> donde &lt;sURL&gt; es una cadena que contiene la URL del enlace. |


(\*) Por defecto, la "Inicialización" Event de cada control se ejecuta según el orden en que se crearon las control en la ventana. Para modificar este orden de ejecución: 

1. En la pestaña "Ventana", en el grupo "Orden", haga clic en "Inicialización". 

2. En la ventana mostrada, utilice las flechas de la derecha para modificar el orden en el que se inicializan los controles.

## Eventos opcionales
<a name="eventos_opcionales_ELTTEXTE000255"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Se pueden utilizar varios eventos opcionales.
Para añadir un evento opcional:

1. Seleccione la control deseada.

2. Mostrar la ventana del código de esta control (tecla F2).

3. Haga clic en el enlace "Añadir otras Event s a xxx" en la parte inferior del código de la ventana, después de la última Event.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. Todos los eventos opcionales disponibles para el control se muestran. 

5. Marque la opción Event para agregar y validar. 
	Observación: Puede seleccionar varios eventos opcionales. 

6. El Event opcional seleccionado se agrega automáticamente a los Event s administrados por el control.




Para desactivar un Event opcional, simplemente realice las mismas operaciones para mostrar la lista de Event s opcionales. A continuación, simplemente desmarque los eventos opcionales para eliminar. 
Observación: Si el código desactivado contiene código WLanguage, este código se elimina automáticamente.Se puede gestionar, por ejemplo:

- el desplazamiento del cursor por encima del control,

- botón izquierdo del ratón abajo, arriba, con un doble clic,

- el botón derecho del ratón presionado, liberado o con doble clic 

- la rueda del ratón, etc.

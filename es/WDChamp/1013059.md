## Resumen de la partida control
<a name="resumen_partida_control_ELTTEXTE000125"></a>


### Reporte control
<a name="reporte_control_ELTPARAGRAPHE000014"></a>

La control "Artículo" se utiliza para editar el contenido de un artículo encontrado en un archivo de datos. El artículo puede ser un artículo del archivo de datos de la base de reporte o de un archivo de datos accesible por un enlace. Este control también puede ser asignado por programación.



**Para crear un control "Elemento"**:

1. En el panel "Creación", en el grupo "Controles habituales", haga clic en "Elemento".

2. Haga clic en la ubicación en la que desea crear el control.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo vincular un artículo control a un artículo?
<a name="¿como_vincular_articulo_control_articulo_ELTTEXTE000149"></a>
Para definir el artículo relacionado al "artículo" control: 

1. Mostrar la pestaña "General" en la ventana de description del control ("description" de el menú contextual del control).

2. **Para relacionar el control con un elemento que pertenece a la fuente de datos del reporte**, seleccione directamente el elemento el Combo Box "Elemento relacionado".

3. **Para relacionar el control con un elemento que no pertenece a la fuente de datos del reporte**, haga clic en el botón "Otros" y seleccione el elemento relacionado con el control actual.

## Notas
<a name="notas_ELTTEXTE000173"></a>


### RTF Artículo control: Cambiar un "artículo" control en un "RTF" control
<a name="rtf_articulo_control_cambiar_articulo_control_rtf_control_ELTPARAGRAPHE000047"></a>

Para mostrar un texto en formato RTF en una "Partida" control: 

1. Abra la ventana de descripción del control (opción "Descripción" del menú contextual del control). 

2. En la pestaña "Detalles", marque "Formato RTF".

3. Valide la ventana de descripción. 




El "Item" control se convierte en un "RTF" control. Se mantienen todas las características del "Item" control.
<a name="NOTE3_2"></a>


### Artículo especial control: "Monetario + Euro"
<a name="articulo_especial_control_monetario_euro_ELTPARAGRAPHE000060"></a>

Para un artículo "Moneda + Euro" control, tiene la posibilidad de gestionar:

- los **moneda almacenada**: moneda de los datos encontrados en la fuente de datos de reporte. Si la control se maneja por programación, la moneda correspondiente será la almacenada.

- los **divisa visualizada**: moneda utilizada para visualizar/imprimir el valor en el reporte. Esta moneda puede diferir de la moneda almacenada. La conversión se realiza automáticamente.

- los **doble indicación**: se pueden vincular dos controles "Moneda + Euro".. El valor mostrado en la base control se convierte automáticamente en el rastreador control.




Este tipo de control también puede ser usado en una ventana. Para más detalles sobre esta control, véase [El tipo "Euro + Moneda](../WDLang1/3033007.md).


### Artículo control y cadena de caracteres UNICODE
<a name="articulo_control_cadena_caracteres_unicode_ELTPARAGRAPHE000074"></a>

Los elementos HFSQL "UNICODE string" se muestran automáticamente en los controles de elementos de los informes.. No se requiere ninguna operación para realizar la visualización.




## Impedir el uso del botón "Back" del navegador
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
Si se utiliza el botón "Atrás" del navegador para mostrar la página anterior, esta acción no tendrá ningún efecto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Modo de funcionamiento
<a name="modo_funcionamiento_ELTTEXTE000170"></a>
Al desactivar la función "Página anterior" del navegador se inserta el siguiente código JavaScript en la página HTML generada:


```txt
<SCRIPT LANGUAGE="JavaScript">
history.forward()
</SCRIPT>
```
Cuando se ejecuta la página en un navegador, no es posible volver a esta página utilizando el botón "Atrás".
<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000026"></a>

- Esta acción no tendrá ningún efecto.

- Si hace clic en el botón "Atrás" del navegador, la página parpadeará.

- Este mecanismo puede fallar si se pulsa el botón [ STOP] del navegador antes de que el navegador ejecute la sentencia forward().




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000200"></a>


### Desactivar el botón "Atrás" del navegador para una página específica
<a name="desactivar_boton_atras_del_navegador_para_una_pagina_especifica_ELTPARAGRAPHE000043"></a>

Para desactivar el botón "Atrás" del navegador para una página específica: 

1. Abra la pestaña "UI" de la ventana de la página description (opción "description" en el menú emergente de la página).

2. En el área "Opciones", para la opción "Uso del botón "Atrás" del navegador", seleccione "Prohibido".

3. Valide.



<a name="NOTE3_2"></a>


### Desactivar el botón "Atrás" del navegador para un conjunto de marcos
<a name="desactivar_boton_atras_del_navegador_para_conjunto_marcos_ELTPARAGRAPHE000058"></a>

Para desactivar el botón "Atrás" del navegador para un conjunto de marcos: 

1. Abra la pestaña "UI" de la ventana del conjunto de marcos description (opción "description" en el menú emergente del marco).

2. En el área "Opciones", para la opción "Uso del botón "Atrás" del navegador", seleccione "Prohibido".

3. Valide.



<a name="NOTE3_3"></a>


### Desactivar el botón "Atrás" del navegador para todas las páginas del proyecto
<a name="desactivar_boton_atras_del_navegador_para_todas_las_paginas_del_proyecto_ELTPARAGRAPHE000073"></a>

Para desactivar el botón "Atrás" del navegador para todas las páginas del proyecto: 

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. En la pestaña "Avanzado", para la opción "Opción predeterminada para utilizar el botón "Atrás" del navegador", marque "Prohibido".

3. Validar. Esta opción se tendrá en cuenta automáticamente para todas las nuevas páginas del sitio.

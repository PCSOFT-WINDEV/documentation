## Visión general del iFrame control
<a name="vision_general_del_iframe_control_ELTTEXTE000123"></a>
iFrame es un concepto HTML. Este concepto indica que un contenido Web independiente se inserta en un área específica de una página.. Esto le permite visualizar en una página HTML:

- una página HTML procedente de un sitio que no sea el sitio WEBDEV actual.

- una página HTML que se encuentra en el sitio actual.

## Creación de un iFrame control
<a name="creacion_iframe_control_ELTTEXTE000147"></a>


### Creación de un iFrame control
<a name="creacion_iframe_control_ELTPARAGRAPHE000021"></a>

Para crear un iFrame control:

1. En la pestaña "Creación", en el grupo "Integración", haga clic en "iFrame".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.
<a name="NOTE2_2"></a>


### Asociar una página con un iFrame control
<a name="asociar_una_pagina_con_iframe_control_ELTPARAGRAPHE000045"></a>

Para asociar una página con un iFrame control:

1. Mostrar la ventana description del iFrame control (seleccionar "description" de el menú contextual del control).

2. En la pestaña "General", seleccione la página que se muestra en la control:

	- una página encontrada en el proyecto actual.

	- introduzca la address de un sitio o de una página del sitio (por ejemplo, [www.windev.es](http://www.windev.com)), así como el modo de seguridad deseado.




3. Valide.

## Manejando un iFrame control
<a name="manejando_iframe_control_ELTTEXTE000177"></a>


### Cambiando el objetivo del iFrame control
<a name="cambiando_objetivo_del_iframe_control_ELTPARAGRAPHE000064"></a>

El objetivo de un iFrame control puede cambiarse fácilmente a través de un botón o de un control Enlace.

El objetivo puede ser cambiado sólo si el objetivo del iFrame control pertenece al sitio actual.

Simplemente crea un Botón o una control Enlace con las siguientes características:

- escribir a máquina: "Mostrar una página del sitio". Todo lo que tiene que hacer es seleccionar la página a mostrar.

- Destino: "En un iFrame XXX". Todo lo que tienes que hacer es seleccionar el nombre del iFrame control.

- Durante la acción: "No enviar nada al servidor". 




**Observación**: Si el objetivo del control del iFrame no pertenece al sitio actual, el navegador INTERNET no permite modificar este objetivo (por razones de seguridad).

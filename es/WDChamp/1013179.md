## Presentación del control Estático
<a name="presentacion_del_control_estatico_ELTTEXTE000356"></a>


### Control de ventana, página (dinámica o estática) y reporte
<a name="control_ventana_pagina_dinamica_estatica_reporte_ELTPARAGRAPHE000011"></a>

Los controles Estático permiten mostrar texto estático. El usuario no puede introducir contenido en el control.

Para aplicar **efectos al texto**, es posible utilizar:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El estilo del control para configurar el elemento "Sombra". 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LIBeLLe_FLOU.gif)


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El tipo "RTF" para mostrar un texto en formato RTF.
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LIBeLLe_RTF.gif)


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Una animación específica (desplazamiento o parpadeo). 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LibelleScrollant.gif)





**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los controles Estático también pueden configurarse con una máscara de visualización para aplicar automáticamente un formato al valor que se les asigna.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Puede configurar los diferentes efectos del control Estático en la pestaña "Estilo" de la ventana de descripción. Puede: 

	- configurar los bordes del control. Por ejemplo, esta opción le permite mostrar una imagen en el control Estático. 
			El ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bouton_rond%20-%20HC%20N%B0001%202.gif) (a la derecha de la opción "Bordes") permite configurar la sombra de los bordes. Si utiliza una sombra, puede definir el desplazamiento X e Y, el porcentaje de opacidad, el radio de desenfoque y el color de la sombra. 

	- establecer las características de la fuente mediante el botón <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Trad_param.gif)
. Es posible definir: 

		- el interlineado,

		- el espaciado entre caracteres,

		- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el ángulo del texto.

## Crear un control Estático
<a name="crear_control_estatico_ELTTEXTE000380"></a>

### Crear un control Estático en una ventana o reporte
<a name="crear_control_estatico_una_ventana_reporte_ELTPARAGRAPHE000105"></a>

**Para crear un control Estático en una ventana**: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_libelle18.gif).

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




**Para crear un control Estático en un reporte**: 

1. En la pestaña "Creación", en el grupo "Texto", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_libelle18.gif).

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual. Puede introducir texto en el control o definir el tipo de información que se mostrará. 

Observaciones: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los reportes no están disponibles.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En las ventanas, los controles Estático multilínea pueden utilizar una elipsis: si el texto es demasiado largo para mostrarse en el control, el final del texto se reemplaza con tres puntos suspensivos. El uso de la elipsis se puede definir en la pestaña "Detalles" del control Estático.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En las ventanas, los controles Estático multilínea pueden justificarse ("Alineación horizontal: Justificado").

### Crear un control Estático de tipo RTF en una ventana
<a name="crear_control_estatico_tipo_rtf_una_ventana_ELTPARAGRAPHE000160"></a>

Para crear un control Estático de tipo RTF en una ventana: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_libelle18.gif).

2. Abra la ventana de descripción del control (seleccione "Descripción" del menú contextual). 

3. En la pestaña "General": 

	- Seleccione el tipo de control Estático: "RTF". 

	- Introduzca el texto RTF: una barra de herramientas específica se muestra automáticamente para definir el formato del texto.







Observaciones:

- También puede crear un control Estático de tipo RTF insertando el control Smart "Texto RTF". 

- Puede crear un control Estático RTF pegando el texto RTF directamente en la ventana.

- Para usar un control Estático de tipo RTF, el archivo "RICHED20.DLL" debe estar en el equipo actual. En la mayoría de los casos, el archivo "RICHED20.DLL" se encuentra en el directorio del sistema de Windows.

- También puede utilizar la propiedad [RichEdit](../Proprietes/2510108.md) para modificar el tipo RTF de un control Estático mediante programación.

### Introducir texto en el control Estático
<a name="introducir_texto_control_estatico_ELTPARAGRAPHE000221"></a>

Para escribir el texto del control Estático:

1. Haga clic en el control Estático en el editor. El cursor cambia a un cursor de texto.

2. Introduzca el texto del control Estático.
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En el caso de los controles Estático de tipo RTF (de una ventana o reporte), el formato de cada elemento puede definirse mediante una barra de herramientas.




Observación: El texto del control Estático también puede editarse en la ventana de descripción del control. 
<a name="NOTE3_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Contenido de un control Estático (Editor de reportes)
<a name="contenido_control_estatico_editor_reportes_ELTPARAGRAPHE000239"></a>

- **La cadena de caracteres que se muestra en el control Estático puede contener el valor de un campo impreso en el reporte**. Para insertar el valor de un campo o control del reporte en el texto del control Estático, introduzca el nombre del campo o control entre [% y %]. 
	Por ejemplo, el texto del control puede corresponder a: 
	El proveedor **[**%**NombreProveedor**%**]** vende el producto **[**%**NombreProducto**%**]** por **[**%**PrecioConImpuestos**%**]**. 
	Los campos NombreProveedor, NombreProducto y PrecioConImpuestos serán reemplazados por su valor cuando se ejecute el reporte.

- **El texto puede ocupar varias líneas**. En este caso, el control debe ser multilínea y su tamaño debe ser lo suficientemente grande. 
	Observación: Si se aplica un ángulo de orientación a un control multilínea, se ignoran los retornos de carro (CR) automáticos. Solo se tienen en cuenta los saltos de línea manuales.




**Recordatorio**: La opción "Ajuste automático" permite aumentar automáticamente el tamaño del control según el contenido.

## Notas
<a name="notas_ELTTEXTE000446"></a>


- El contenido de un control Estático se puede introducir en el editor o definirse mediante programación.

- Un control Estático se puede enlazar a un campo de un archivo de datos. En este caso, el control muestra el contenido de los campos del archivo de datos.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En las ventanas, los controles Estático tienen una máscara de visualización para aplicar automáticamente un formato a los datos que se les asignan. Esta máscara puede modificarse mediante programación con la propiedad [InputMask](../Proprietes/2510033.md).

## Pasar parámetros a una Page al abrirla
<a name="pasar_parametros_una_page_abrirla_ELTTEXTE000132"></a>




### Para pasar parámetros a una Page cuando se abre:
<a name="para_pasar_parametros_una_page_cuando_abre_ELTPARAGRAPHE000012"></a>

**1.** Declare un Procedure en las "Declaraciones globales" Event del Page. El nombre de esta Procedure y el de la Page deben ser idénticos. Los parámetros de esta Procedure corresponden a los parámetros que deben pasarse a la Page.

**Atención**: La declaración Procedure (palabra clave Procedure) debe corresponder a la primera línea de las "Declaraciones globales" Event.

Por ejemplo, la página PAGE_Calendar se utiliza para gestionar la selección de una fecha en un Calendar. Esta Page espera un parámetro que indique la fecha a seleccionar.


```wl
// -- Global declarations of PAGE_CALENDAR --

PROCEDURE PAGE_Calendar(sSelDate)
// sSelDate: date specifying the date to select.
```


Este parámetro puede manejarse desde cualquier Event de la Page (botón, procedimientos locales, etc.).

Por ejemplo, en el evento "Inicialización" de PAGE_Calendar:


```wl
// -- Initializing PAGE_CALENDAR --

// EDT_DateControl is an edit control
// It contains the value of the sSelDate parameter
EDT_DateControl = sSelDate
```


**2.** Pase el parámetro esperado por el Page cuando se abre ( [PageDisplay](../WDLang2/3058008.md) , [PageUse](../WDLang2/3058016.md) 02.30, etc.). Por ejemplo, BTN_Calendar se utiliza para abrir PAGE_Calendar. Cuando se abre esta Page, la fecha de hoy se pasa como parámetro.


```wl
// -- Click code of BTN_CALENDAR --

// Open PAGE_CALENDAR
// Today's date is passed as parameter
PageDisplay(PAGE_CALENDAR, Today())
```

## Ejecución de la prueba de una Page con parámetros
<a name="ejecucion_prueba_una_page_con_parametros_ELTTEXTE000156"></a>
Para ejecutar la prueba de una Page con parámetros:

1. Abra el Page con parámetros en el editor Page.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Go_Page_bl.gif) (o pulse F9). La siguiente ventana se abre:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Page_Parametree%20-%20HC%20N%B0001.gif)


3. Especifique el valor de los parámetros que se utilizarán para ejecutar la prueba de la Page. Para utilizar el valor Default de los parámetros, escriba el carácter "\*".

4. Validar. La Page se visualiza según los valores especificados.

## Notas
<a name="notas_ELTTEXTE000180"></a>


### Parámetros pasados por valor
<a name="parametros_pasados_por_valor_ELTPARAGRAPHE000058"></a>

Si los parámetros pasados a un Page se modifican en este Page, estas modificaciones sólo se tendrán en cuenta en este Page. El valor de estos parámetros no se modifica en el llamado Event.

Por ejemplo:

1. La variable *MyDate* se declara en el código de BTN_ Calendar. Esta variable contiene la fecha del día (por ejemplo: MyDate = Today()).

2. Esta variable se pasa en parámetro a PAGE_Calendar. El parámetro *sSelDate* contiene el valor de la variable *MyDate*.

3. Se modifica el valor del parámetro *sSelDate* en PAGE_ Calendar (por ejemplo: sFechaSel = "20020701")

4. El valor de *MyDate* Variable no se modifica.



<a name="NOTE3_2"></a>


### Dando un valor Default al parámetro en la declaración
<a name="dando_valor_default_parametro_declaracion_ELTPARAGRAPHE000072"></a>

Se puede dar un valor Default a los parámetros cuando se declaran los parámetros.

Por ejemplo, para dar un valor por defecto en el ejemplo anterior, introduzca el siguiente código en el evento "Declaraciones globales" de PAGE_Calendar:


```wl
// -- Global declarations of PAGE_CALENDAR --

PROCEDURE PAGE_Calendar(sSelDate = 20030101)
// sSelDate: date specifying the date to select.
```

<a name="NOTE3_3"></a>


### Alcance de los parámetros
<a name="alcance_los_parametros_ELTPARAGRAPHE000083"></a>

Los parámetros pasados a una página son globales a todos los eventos de esta página (inicialización, código de un botón, código de un Procedure local, etc.).

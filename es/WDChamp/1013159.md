
## Vinculación de una control de edición a un elemento

<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000099"></a>
Una edición control se puede vincular a un elemento en un archivo de datos , consulta, etc., ... El tipo de control encuadernada puede ser diferente según el tipo de artículo.

La tabla que aparece a continuación muestra los diferentes tipos de elementos y los correspondientes controles de edición creados automáticamente por la metodología RAD:

| Tipo de elemento | Tipo de control |
| --- | --- |
| texto | Texto control Campo de entrada |
| Numeric | Control Campo de entrada numérico |
| Moneda | Moneda control Campo de entrada |
| salir con | Fecha control Campo de entrada |
| momento | Horario control Campo de entrada |

## Actualizar la control de edición con el valor del artículo
<a name="actualizar_control_edicion_con_valor_del_articulo_ELTTEXTE000123"></a>
**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los controles enlazados a un campo en el archivo de datos se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [FileToScreen](../WDLang4/3044210.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

## Actualizar un artículo con los datos de la edición de control
<a name="actualizar_articulo_con_los_datos_edicion_control_ELTTEXTE000168"></a>
**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los campos de el archivo de datos enlazados a un control de la ventana o página se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [ScreenToFile](../WDLang4/3044146.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
El valor de cada control debe asignarse a cada campo utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```




## La página de inicio
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
La página de inicio es la primera página de su sitio WEBDEV (estática o en modo de sesión). Esta página es siempre una página estática.

En un sitio en modo de sesión, la página de inicio es uno de los elementos utilizados para reference el sitio.

Usted tiene la posibilidad de elegir una página estática que se encuentra en su sitio.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Definición de la página de inicio
<a name="definicion_pagina_inicio_ELTTEXTE000126"></a>
WEBDEV permite definir la página de inicio de un sitio a través del panel "Explorador de proyectos".

**Observación**: También se propone una página de inicio durante la prueba de un proyecto estático, si no se ha definido ninguna página de inicio.. Para obtener más información, consulte [Probar un proyecto](../Editeurs/2019003.md).
<a name="NOTE2b_2"></a>


### Definir la casa en el panel "Explorador de proyectos
<a name="definir_casa_panel_explorador_proyectos_ELTPARAGRAPHE000032"></a>

Definir la página de inicio del proyecto:

1. En el panel "Explorador de proyectos", seleccione la página que se convertirá en la nueva página de inicio.. Esta página debe ser una página estática de su sitio.

2. Seleccione "Página de inicio" en el menú contextual. Esta opción sólo está disponible si la página seleccionada es una página estática.

3. Se genera automáticamente la página HTML correspondiente y se define la página de inicio.. El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_accueil.gif) Icon se utiliza para identificar la página de inicio en el panel "Explorador de proyectos".

## Archivos creados correspondientes a la página de inicio
<a name="archivos_creados_correspondientes_pagina_inicio_ELTTEXTE000156"></a>
Cuando se define una página de inicio para su sitio estático o su sitio en modo de sesión, los siguientes archivos se crean automáticamente en el directorio &lt;Nombre del proyecto&gt;_WEB, así como en cada subdirectorio del idioma:

- Inicio.htm

- Default.htm

- Index.htm




Al instalar un sitio WEBDEV, esta página se utilizará como primera página del sitio.. El Address para iniciar el sitio tendrá que iniciar esta página.


## Definir las áreas sobre las que se puede hacer clic
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
En una página, una control Imagen puede contener áreas clicables. En este caso, en la pestaña "General" de la ventana de control description, "Acción al hacer clic" es set a "Zonas clicables definidas en modo de edición (mapa Image)". 

A continuación, puede incluir zonas en las que se pueda hacer clic en el Image. Estas áreas pueden tener diferentes formas: círculo, rectángulo, Polygon, etc.

Se puede asignar una acción específica a cada área y a toda la Image. La acción correspondiente se realizará cuando el usuario Web haga clic en una de estas áreas.

## Dibujando un área de clicks...
<a name="dibujando_area_clicks_ELTTEXTE000159"></a>


### Dibujar una zona clicable en un mapa control Imagen (Image)
<a name="dibujar_una_zona_clicable_mapa_control_imagen_image_ELTPARAGRAPHE000022"></a>

Para dibujar una zona clicable en un mapa control Imagen (Image):

1. En la pestaña "General" de la control Imagen description, asegúrese de que la acción es set a "Áreas clicables definidas en modo de edición (mapa Image)".

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la Image y seleccione "Editar áreas clicables". La control Imagen pasa al modo de edición: se muestra un contorno amarillo. 

3. Abrir el menú contextual de la control Imagen en modo de edición. Dependiendo de la forma del área a crear, seleccione una de las siguientes opciones en el menú contextual:

	- "Agregar un rectángulo": para definir áreas fijas simples.

	- "Agregar un círculo": para definir áreas redondeadas.

	- "Agregar un polígono": para definir áreas complejas como un mapa geográfico.




4. **Para un círculo o un rectángulo**se crea automáticamente el área. 
	Para mover el área, haga clic en el área y mantenga pulsado el botón izquierdo del ratón mientras mueve el área.. 
	Para cambiar el tamaño del área, haz clic en el área y selecciona uno de los mangos de Black.

5. **Para un Polygon**, haga clic en el lugar donde debe dibujarse el área. Luego, haga clic para definir cada punto de la Polygon. Para cerrar la Polygon, usa el botón derecho del ratón.




**Observación**: Cada área sobre la que se puede hacer clic se identifica con un número. Este número corresponde al orden en que se han creado las áreas. Este número se muestra automáticamente en el área.
<a name="NOTE2_2"></a>


### Consejos para un área de clicks de Polygon
<a name="consejos_para_area_clicks_polygon_ELTPARAGRAPHE000045"></a>

**Para seleccionar con precisión los puntos del área**tiene la posibilidad de ampliar la página en el editor de páginas. Para ello, seleccione el porcentaje de zoom en la barra de estado de WEBDEV. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zoom.gif)


**To añadir vértices a la Polygon**:

1. Mantenga pulsada la tecla Ctrl.

2. Haga clic en el lugar deseado del contorno del área (aparece un signo "+" debajo del cursor).

3. Suelte la tecla Ctrl.

## Asociar una acción con un área clicable
<a name="asociar_una_accion_con_area_clicable_ELTTEXTE000189"></a>
Para asociar una acción a un área en la que se puede hacer clic:

1. Seleccione el área en la que se puede hacer clic.

2. Seleccione "Acción de área" en el menú contextual de la zona en la que se puede hacer clic.

3. En la ventana que aparece, seleccione la acción deseada:

	- Ejecutando un enlace o un botón que se encuentra en la página, ....

	- Mostrar una página, ...

	- Enviar un Email, ...




4. Valide.



<a name="NOTE3_2"></a>


### Gestión de la acción associated con una zona clicable (mapa Image) de forma programática
<a name="gestion_accion_associated_con_una_zona_clicable_mapa_image_forma_programatica_ELTPARAGRAPHE000075"></a>

En el código del servidor de la control Imagen (mapa de la Image), se puede gestionar la acción a realizar según la zona pulsada. Para ello, basta con llamar a [ImageArea](../WDLang2/3045002.md) en el código de clicks del servidor de control Imagen.

Por ejemplo, el siguiente código se utiliza para mostrar una página diferente según el área en la que se ha hecho clic. Las áreas se identifican mediante un número. Este número corresponde al orden en que se han creado las áreas. Este número se muestra automáticamente en el área.

**Atención**: Si se elimina un área, se modificará el número de las áreas creadas después (número inicial -1).


```wl
// -- Server code: Click on IMG_Click1
Area is int
// Retrieve the area that was clicked
Area = ImageArea()
SWITCH Area 
	CASE 1: PageDisplay(PAGE_Home)
	CASE 2: PageDisplay(PAGE_AboutUs)
END
```

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del campo
<a name="caracteristicas_del_campo_ELTTEXTE000113"></a>
Una control Código de barras puede estar vinculada a un elemento de un archivo de datos, consulta, etc.

Hay dos tipos de elementos que se pueden vincular a una control Código de barras:

- un elemento "Texto".

- un elemento "Numérico".








<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Enlazar un control Código de barras a un campo
<a name="enlazar_control_codigo_barras_campo_ELTTEXTE000137"></a>
**Para definir el elemento que se vinculará al control Código de barras directamente desde la ventana, página o editor de reportes**:

1. Haga clic con el botón derecho del ratón y seleccione "description" en el menú contextual. A continuación, vaya a la pestaña "Binding" de la ventana de control description.

2. Seleccione el archivo de datos o la consulta y, a continuación, el elemento que desea vincular a la control Código de barras.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Para actualizar la partida con los datos de la control Código de barras
<a name="para_actualizar_partida_con_los_datos_control_codigo_barras_ELTTEXTE000161"></a>
**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los campos de el archivo de datos enlazados a un control de la ventana o página se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [ScreenToFile](../WDLang4/3044146.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
El valor de cada control debe asignarse a cada campo utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

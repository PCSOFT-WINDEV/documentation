


## Funciones WLanguage disponibles para un reporte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000273"></a>
La mayoría de las funciones WLanguage se pueden utilizar en los eventos de un reporte (creado con el editor de reportes).

Sin embargo, algunas de ellas no son recomendables, como las funciones de impresión específicas para la impresión en WLanguage o las funciones para acceder a los registros utilizados por reporte.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Funciones WLanguage no recomendadas en un reporte
<a name="funciones_wlanguage_recomendadas_reporte_ELTTEXTE000297"></a>
No se recomienda utilizar las siguientes funciones en los eventos de reporte, bloques o controles. Si se utilizan estas funciones, la ejecución de reporte puede ser incorrecta.

**Funciones de impresión**: [iEllipse](../WDLang5/3046009.md), [iEndPrinting](../WDLang5/3046053.md), [iImageHeight](../WDLang5/3046057.md), [iPrint](../WDLang5/3046031.md), [iPrintBarCode](../WDLang5/3046020.md), [iPrintImage](../WDLang5/3046017.md), [iPrintWord](../WDLang5/3046034.md), [iPrintArea](../WDLang5/3046027.md), [iPrintAreaRTF](../WDLang5/3046023.md), [iImageWidth](../WDLang5/3046061.md), [iLink](../WDLang5/3046018.md), [iNewLine](../WDLang5/3046035.md), [iXPos](../WDLang5/3046015.md), [iYPos](../WDLang5/3046075.md), [iSkipLine](../WDLang5/3046071.md), [iLine](../WDLang5/3046050.md), [iHLine](../WDLang5/3046051.md) y [iVLine](../WDLang5/3046066.md).

**Funciones de acceso a un registro**: Todas las funciones HFSQL que modifican un Record en memoria o el Record actual en el archivo base del reporte no se recomiendan en los eventos de un reporte, bloquean o controlan. De hecho, estas funciones pueden modificar la navegación realizada en el archivo base de la reporte.

Estas funciones son: [HAdd](../WDLang4/3044147.md), [HCreation](../WDLang4/3044255.md), [HCreationIfNotFound](../WDLang4/3044254.md), [HLast](../WDLang4/3044101.md), [HWrite](../WDLang4/3044092.md), HLitxxxx, [HModify](../WDLang4/3044042.md), [HPrevious](../WDLang4/3044112.md), [HSetPosition](../WDLang4/3044110.md), [HFirst](../WDLang4/3044113.md), [HSeek](../WDLang4/3044130.md), [HBackward](../WDLang4/3044126.md), [HCross](../WDLang4/3044121.md), [HNext](../WDLang4/3044027.md), [HDelete](../WDLang4/3044018.md), etc.|   |   |   |

## Funciones de impresión de WLanguage que se pueden utilizar en un reporte
<a name="funciones_impresion_wlanguage_que_pueden_utilizar_reporte_ELTTEXTE000339"></a>
En una reporte se pueden utilizar las siguientes funciones de impresión: 



|   |   |
| --- | --- |
| [función iReset](../WDLang5/3046067.md) | Reinicia los ajustes de impresión (márgenes, fuente, formato de papel, orientación, ...) almacenados en la biblioteca de impresión. |
| [iAreaHeight](../WDLang5/3046062.md) | Calcula la altura de un área de ancho fijo necesaria para imprimir todo el texto que utiliza la fuente actual. |
| [iBorder](../WDLang5/3046044.md) | Imprime una Border en las coordenadas especificadas. |
| [iCloseReport](../WDLang5/3046041.md) | Detiene la impresión de la reporte actual inmediatamente. |
| [iConfigure](../WDLang5/3046072.md) | Configura la impresora para una impresión realizada en WLanguage:<br><br>- abriendo la ventana de configuración. <br><br>- modificando la impresora utilizada para la impresión actual. <br><br><br> |
| [iCreateFont](../WDLang5/3046045.md) | Crea una nueva fuente de impresión. |
| [iDidotFont](../WDLang5/3046016.md) | Selecciona la unidad que se utilizará para la altura de las fuentes (creada por [iCreateFont](../WDLang5/3046045.md)): Punto DIDOT o punto PICA. |
| [iDocumentCanceled](../WDLang5/3046006.md) | Permite saber si la impresión actual ha sido cancelada por el usuario. |
| [iDocumentPrinted](../WDLang5/3046029.md) | Determina si el usuario imprimió la Document desde el visor de reporte. |
| [iEndReport](../WDLang5/3046040.md) | Obliga a detener la impresión de reporte. |
| [iEscape](../WDLang5/3046054.md) | Envía un comando ESCAPE o un comando de datos a una impresora. |
| [iFont](../WDLang5/3046014.md) | Selecciona la fuente Default. |
| [iForceComplement](../WDLang5/3046046.md) | Fuerza la impresión de un bloque de Body Complement. |
| [iInfoPrinter](../WDLang5/3046022.md) | Recupera las características de la impresora actual o Default. |
| [iInitReportQuery](../WDLang5/3046021.md) | Inicializa la consulta relacionado al reporte (para una consulta con parámetros). |
| [iInitReportQueryConnection](../WDLang5/3046028.md) | Inicia la consulta relacionado en el reporte. |
| [iLastFile](../WDLang5/3046068.md) | Devuelve: <br><br>- el nombre completo del último archivo generado durante la impresión (PDF, RTF, texto, ....).<br><br>- la lista de archivos duplicados creados. <br><br><br> |
| [iListNestedReports](../WDLang5/3046005.md) | Devuelve : <br><br>- la lista de informes anidados que se imprimen actualmente (informes ejecutados por [iPrintReport](../WDLang5/3046032.md) en uno de los procesos de la reporte que se imprimen actualmente). <br><br>- la lista de informes secuenciados que se imprimen actualmente ([iSequenceAdd](../WDLang5/1000020059.md)).<br><br><br> |
| [iListPrinter](../WDLang5/3046063.md) | Devuelve la lista de impresoras instaladas en el ordenador actual. |
| [iMargin](../WDLang5/3046011.md) | Define los márgenes de impresión "lógicos". |
| [iMMToPica](../WDLang5/3046004.md) | Convierte las coordenadas de un punto (expresadas en milímetros) en coordenadas del sistema. |
| [iNestedHeaderFooter](../WDLang5/3046047.md) | Imprime (o no) las cabeceras y pies de página de reporte anidada. |
| [iPageEnd](../WDLang5/3046038.md) | Fuerza el movimiento a la página siguiente durante la impresión de reporte. |
| [iPageHeight](../WDLang5/3046052.md) | Calcula la altura de la página imprimible (en milímetros) teniendo en cuenta los márgenes superior e inferior. |
| [iPageHeightLeft](../WDLang5/3046060.md) | Calcula la altura disponible (en milímetros) a la izquierda de la página actual teniendo en cuenta los márgenes definidos (superior o inferior). |
| [iPageNum](../WDLang5/3046013.md) | Devuelve o inicializa el número de la página actualmente impresa. |
| [iPageWidth](../WDLang5/3046036.md) | Calcula el ancho (en milímetros) de la página imprimible teniendo en cuenta los márgenes izquierdo o derecho. |
| [iParameter](../WDLang5/3046008.md) | Recupera la configuración de la impresora y configura la impresión. |
| [iParameterPDF](../WDLang5/3046007.md) | Define las opciones para el archivo PDF generado. |
| [iParameterPreview](../WDLang5/3046001.md) | Configura la vista previa de impresión:<br><br>- muestra un mensaje sobre los recursos del equipo actual.<br><br>- muestra u oculta las pestañas de la cinta de opciones (así como los botones). <br><br>- las opciones de exportación disponibles en la mini ventana de vista previa. <br><br><br> |
| [iPicaToMM](../WDLang5/3046010.md) | Convierte las coordenadas de un punto (expresadas en coordenadas del sistema) en milímetros. |
| [iPreview](../WDLang5/3046069.md) | Configura el destino de impresión. |
| [iPrintBlock](../WDLang5/3046030.md) | Obliga a imprimir un bloque reporte. |
| [iPrintBodyComplement](../WDLang5/3046019.md) | Fuerza a imprimir un bloque de complemento de cuerpo mientras especifica la altura del bloque de complemento. |
| [iPrintReport](../WDLang5/3046032.md) | Imprime una reporte creada con el editor de reportes. |
| [iReportPrintingStatus](../WDLang5/3046043.md) | Devuelve el estado de la impresión actual. |
| [iRoundBorder](../WDLang5/3046024.md) | Imprime una Border con contornos redondeados en coordenadas especificadas. |
| [iSelectFont](../WDLang5/3046055.md) | Crea una nueva fuente de impresión desde la ventana estándar para la selección de fuentes.... |
| [iSkipPage](../WDLang5/3046073.md) | Genera un salto de página: el código correspondiente se envía a la cola de impresión. |
| [iSubstBlock](../WDLang5/3046048.md) | Sustituye un bloque de la reporte por otro bloque mientras se imprime la reporte. |
| [iTextHeight](../WDLang5/3046070.md) | Calcula la altura de la fuente para el texto a imprimir (en milímetros). |
| [iTextWidth](../WDLang5/3046033.md) | Calcula el ancho (en milímetros) del texto a imprimir, de acuerdo con la fuente especificada. |
| [iTransparentMagenta](../WDLang5/3046003.md) | Modifica el modo de gestión del color Magenta para que este color sea considerado como Transparent (o no) en todas las próximas impresiones. |
| [iWindowCancel](../WDLang5/3046042.md) | Configura la ventana de impresión de Cancel.. |


## Manejo de una control Tabla dinámica mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000197"></a>
Una control Tabla dinámica puede ser manejada a través de la programación. 

Para manipular a control Tabla dinámica mediante programación, WINDEV propone el [Funciones PVT](../WDLang1/1000020704.md). 

Esta ayuda Page explica cómo manipular a control Tabla dinámica a través de la programación. 





## Manejo de la control Tabla dinámica mediante programación
<a name="manejo_control_tabla_dinamica_mediante_programacion_ELTTEXTE000221"></a>


### Colorear los valores
<a name="colorear_los_valores_ELTPARAGRAPHE000022"></a>

Desde el menú contextual de la control Tabla dinámica, el usuario final puede fácilmente color un valor para resaltarlo. Para obtener más información, consulte [Funcionalidades automáticas de los controles Tabla dinámica](../WDChamp/1000029008.md). 

A través de la programación, usted tiene la capacidad de acceder a los valores con el fin de modificar su estilo (texto o fondo color, ...). Para ello, modifique las propiedades de los valores en la "Visualización de celdas" Event. 

Por ejemplo: 


```wl
// If the turnover is less than 500, displays the value in red
IF VAL_TO <= 500 THEN
	VAL_TO.BackgroundColor = PastelRed
END
```


Observación: Tiene la capacidad de personalizar el estilo de cada elemento de la celda: Color, fuente, fondo Color, ... a través de varias propiedades. Para obtener más información, consulte [Propiedades associated con los valores de un control Tabla dinámica](../WDChamp/1000029003.md). 




### Añadir una columna o un título de fila
<a name="anadir_una_columna_titulo_fila_ELTPARAGRAPHE000046"></a>

Por Default, las filas o columnas de un control Tabla dinámica no tienen título. Puede añadir una utilizando el nombre de cabecera [Pie de foto](../Proprietes/2510053.md) Property. 

Por ejemplo: 

- La tabla pivotante no muestra un título: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=tcd_sanstitre.gif)


- Tabla pivotante con un título: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=tcd_avectitre.gif)





Se utiliza el siguiente código: 


```wl
// Initialize PVT_Stats
COL_Continent.Caption = "Continent"
COL_Country.Caption = "Country"
COL_Vehicle_Type.Caption = "Vehicle"
COL_OrderDate_Year.Caption = "Year"
COL_OrderDate_Quarter.Caption = "Quarter"
COL_OrderDate_Month.Caption = "Month"
```



### Hacer un valor Invisible
<a name="hacer_valor_invisible_ELTPARAGRAPHE000067"></a>

Por Default, todos los valores de una Tabla Pivotante se muestran. En algunos casos, puede ser interesante calcular los valores pero no mostrarlos.. Los valores existen en el control Tabla dinámica y se pueden utilizar para realizar un cálculo o para rellenar un gráfico, por ejemplo, ....

Para ocultar un valor, basta con utilizar el botón [Visible](../Proprietes/2510138.md) Property. 

Ejemplo: Una tabla pivotante muestra en cada celda el volumen de ventas y la cantidad vendida. Un gráfico se construye a partir de la cantidad vendida y esta cantidad no debe ser mostrada en el control Tabla dinámica. Para ello, utilice el siguiente código Line en el código de inicialización del control Tabla dinámica: 


```wl
VAL_Qty.Visible = False
```



### Descubrir y modificar la máscara utilizada para mostrar los valores
<a name="descubrir_modificar_mascara_utilizada_para_mostrar_los_valores_ELTPARAGRAPHE000083"></a>

Para obtener y establecer el tipo que se muestra para un valor en un control Tabla dinámica, utilice el [InputType](../Proprietes/2510136.md) Property. Puede utilizar la siguiente sintaxis: 


```txt
{{"Value Name"}}..InputType = typInputDuration
```


o: 


```txt
ValueName.InputType = typInputDuration
```





### Modificación de los ejes visualizados
<a name="modificacion_los_ejes_visualizados_ELTPARAGRAPHE000100"></a>

Tiene la posibilidad de modificar los ejes mostrados o de invertir las filas y columnas con [PVTAxisXY](../WDLang1/1000021113.md). 

Ejemplo: 


```wl
// Modifies the rows and columns displayed
PVTAxisXY(PVT_Stats, "COL_Year" + CR + "COL_Quarter", "COL_VehicleType" + CR + "COL_Model")
```





### Modificación de la fuente de datos
<a name="modificacion_fuente_datos_ELTPARAGRAPHE000113"></a>

También tiene la posibilidad de modificar la fuente de datos de un control Tabla dinámica a través de [BrowsedFile](../Proprietes/2510040.md). 

Ejemplo: 


```wl
// Modifies the rows and columns displayed
PVT_Stats.BrowsedFile = Stat2015
```


Atención: 

- Si el control Tabla dinámica está basado en el un archivo de datos, el nuevo archivo de datos de la fuente debe tener la misma estructura que el archivo usado cuando se creó el control. 

- Si la control Tabla dinámica se basa en un array de estructuras: 

	- la nueva fuente array de estructuras debe tener la misma estructura que la Variable usada al crear el control. 

	- el nombre de la nueva estructura debe ir precedido de ":". Por ejemplo: 
			
		```wl
		PVT_Stats.BrowsedFile = ":MyNewStructure"
		```









### Progreso del cálculo
<a name="progreso_del_calculo_ELTPARAGRAPHE000280"></a>

- Se muestra una barra de progreso durante el cálculo de la control Tabla dinámica. 

- El usuario tiene la capacidad de Cancel el cálculo a través de un botón "Cancel".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_TCD_Cellule_Jauge.gif)
 Para saber si el cálculo se ha realizado en su totalidad, utilice el [ErrorOccurred](../WDLang1/3087001.md) Variable. 
	Ejemplo: 
	
	```wl
	PROCEDURE Load_PVT()
	
	PVTCalculateAll(PVT_Statistics)
	IF ErrorOccurred = True THEN
		SWITCH Dialog("Do you want to cancel the current process?")
			// Cancel
			CASE 1
				Close()
			// Continue
			CASE 0
				Load_PVT()
		END
	END
	```








### Propiedades específicas de la gestión de Pivot controles Tabla
<a name="propiedades_especificas_gestion_pivot_controles_tabla_ELTPARAGRAPHE000142"></a>Las siguientes propiedades se utilizan para gestionar la Pivot controles Tabla: 



|   |   |
| --- | --- |
| [AutoLineWrap](../Proprietes/1000021868.md) | El **AutoLineWrap** Property se utiliza para: <br><br>- determinar si la función de envoltura automatic Line está activada en un multiLine Editar control, en una columna de una Tabla o control TreeView o en una cabecera fila de un control Tabla dinámica. <br><br>- set el modo de envoltura Line en un multiLine Editar control, en una columna de una Tabla o control TreeView o en una cabecera fila de un control Tabla dinámica.<br><br><br> |
| [Cumulated](../Proprietes/1000020702.md) | El **Acumulado** Property se utiliza para determinar si el valor de una columna o fila en control Tabla dinámica corresponde a un total (total al final de un fila o columna). |
| [DisplayOrphan](../Proprietes/1000020697.md) | La propiedad **DisplayOrphan** permite: <br><br>- Determinar si una fila o columna en un control Tabla dinámica se muestra cuando no tiene padre. <br><br>- Cambiar el modo de visualización de una fila o una columna en un control Tabla dinámica cuando no tiene padre. <br><br><br> |
| [FilterProcedure](../Proprietes/1000020655.md) | La propiedad **FilterProcedure** permite obtener y modificar el procedimiento utilizado para aplicar un filtro en un encabezado de fila o columna al calcular un control Tabla dinámica. |
| [ProgressBar](../Proprietes/1000020706.md) | El **ProgressBar** Property se utiliza para identificar y cambiar el control Barra de progreso utilizado al calcular un control Tabla dinámica (funciones [PVTCalculateAll](../WDLang1/1000020539.md) y [PVTCalculateUpdate](../WDLang1/1000020534.md)). |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Tabla dinámica, consulte [Propiedades associated con Pivote controles Tabla](../WDChamp/1000029003.md).

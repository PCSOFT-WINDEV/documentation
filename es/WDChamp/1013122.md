
## Posicionamiento de una barra de herramientas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000093"></a>
Se puede crear una barra de herramientas control en una ventana "Libre" o en una ventana "MDI Parent".. La gestión del anclaje de la barra de herramientas (y sus consecuencias en la disposición de los controles de la ventana) depende del tipo de ventana.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Barra de herramientas en una ventana "Libre
<a name="barra_herramientas_una_ventana_libre_ELTTEXTE000117"></a>
En una ventana libre, cuando el usuario acopla la barra de herramientas, todos los controles de la ventana se moverán proporcionalmente al tamaño de la barra de herramientas.. El movimiento de los controles se realizará cumpliendo con las características de anclaje de los controles de las ventanas.

## Barra de herramientas en una ventana principal de MDI
<a name="barra_herramientas_una_ventana_principal_mdi_ELTTEXTE000141"></a>
En una ventana MDI padre, cuando el usuario acopla la barra de herramientas, se modificará el tamaño del área MDI.

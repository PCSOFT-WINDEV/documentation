


## Propiedades associated con la serie de cartas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes propiedades se pueden utilizar con las series de un "Gráfico":



|   |   |
| --- | --- |
| [AutoRefresh](../Proprietes/2510146.md) | El **AutoRefresh** Property se utiliza para:<br><br>- Averiguar si los datos encontrados en la serie se recalculan automáticamente cada vez que se llama a [grDraw](../WDLang3/3042026.md).<br><br>- Modificar el modo para recalcular los datos encontrados en la serie.<br><br><br> |
| [Legend](../Proprietes/2510143.md) | La propiedad **Legend** permite:<br><br>- Descubra la etiqueta de una serie.<br><br>- Modificar la etiqueta de una serie.<br><br><br> |
| [MaxValue](../Proprietes/2510009.md) | El **MaxValue** Property se utiliza para:<br><br>- averiguar y modificar el límite superior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite superior definido para el filtro actual.<br><br>- averiguar el valor máximo de una serie de gráficos.<br><br>- averiguar y modificar el valor máximo para algunos tipos de controles.<br><br><br> |
| [MinValue](../Proprietes/2510008.md) | El **MinValue** Property se utiliza para:<br><br>- averiguar y modificar el límite inferior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite inferior definido para el filtro actual.<br><br>- averiguar el valor mínimo de una serie de gráficos.<br><br>- averiguar y modificar el valor mínimo para algunos tipos de controles. <br><br><br> |
| [SecondaryAxis](../Proprietes/1000019187.md) | La propiedad **SecondaryAxis** permite: <br><br>- determinar si una serie del gráfico está en el eje secundario. <br><br>- establecer la posición de una serie del gráfico en el eje secundario.<br><br><br> |
| [SeriesColor](../Proprietes/2510145.md) | La propiedad **SeriesColor** permite:<br><br>- Encuentra la Color de una serie en un gráfico.<br><br>- Modificar la Color de una serie en un gráfico. <br><br><br> |
| [Source](../Proprietes/2510147.md) | La propiedad **Source** se puede utilizar en:<br><br>- **un control Gráfico**: en este caso, el **Fuente** Property se utiliza para get la fuente de la serie: Columna de la tabla, lista de valores, ...<br><br>- **una conexión**: en este caso, el **Fuente** Property se utiliza para obtener o set la fuente de una conexión (HFSQL Client/Server, OLE DB, etc.).<br><br><br><br><br> |
| [SourceDescription](../Proprietes/2510148.md) | El **SourceDescription** Property se utiliza para get el nombre del elemento utilizado como fuente de datos de una serie. |
| [Thickness](../Proprietes/2510144.md) | La propiedad **Thickness** permite:<br><br>- Averigua el grosor de la Line para una serie (sólo en las tablas de la Line). <br><br>- Modificar el grosor de la Line para una serie (sólo en las tablas de la Line).<br><br><br> |

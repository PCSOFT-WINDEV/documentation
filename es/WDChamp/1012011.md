## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
La visualización de un nuevo conjunto de marcos en un sitio WEBDEV puede realizarse según varios métodos:

- [desde la description del control que debe abrir el frameset (botón, enlace, menú, Image pulsable, etc.)](#NOTE2_1).

- [mediante programación en WLanguage](#NOTE3_1).

## Mostrando un juego de marcos de la description de un control
<a name="mostrando_juego_marcos_description_control_ELTTEXTE000133"></a>
Al describir un botón, un enlace o un Image sobre el que se puede hacer clic, se tiene la capacidad de definir la acción realizada y el destino de esta acción.

Para abrir un frameset "xxx" desde un botón, un enlace, un Image, ...:

1. Abra la descripción del elemento (opción "Descripción" del menú contextual del elemento)

2. Seleccione la acción a realizar: "Mostrar la página xxx". Esta Page debe corresponder al conjunto de cuadros.

3. Seleccione el destino de la acción: página actual, navegador actual, ...

4. Valide.

## Visualización de un conjunto de marcos mediante programación (en WLanguage)
<a name="visualizacion_conjunto_marcos_mediante_programacion_wlanguage_ELTTEXTE000157"></a>
Se pueden utilizar varias funciones de WLanguage para abrir un conjunto de marcos.



|   |   |
| --- | --- |
| [FramesetDisplay](../WDLang2/3058006.md) | Muestra un conjunto de marcos WEBDEV en el navegador del usuario. |
| [FramesetRefresh](../WDLang2/3058014.md) | Actualiza un conjunto de marcos que se muestra en el navegador del usuario a partir del contexto en el servidor. |
| [FramesetUse](../WDLang2/3058012.md) | Muestra un conjunto de marcos WEBDEV en el navegador del usuario y cierra todos los contextos de la página actual y del conjunto de marcos. |





Al abrir el conjunto de marcos, usted tiene la capacidad de pasar los parámetros:

- al conjunto de marcos. Todo lo que tiene que hacer es especificar estos parámetros en una de las funciones para abrir el conjunto de marcos.

- a una de las páginas del conjunto de marcos. Basta con abrir el contexto de la Page correspondiente y especificar los parámetros a pasar ([ContextoAbrir](../WDLang2/3058004.md)). Esta función debe utilizarse antes de abrir el juego de marcos.




 
**Observación**: De forma predeterminada, cuando se abre una página mediante programación, el destino seleccionado es:

- el objetivo definido en description del objeto que desencadena la apertura de Page (botón, enlace, etc.).

- el destino establecido para la página actual.

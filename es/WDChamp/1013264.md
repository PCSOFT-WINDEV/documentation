
## Programación de las células tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000132"></a>
Las mesas se pueden manipular:

- por fila, 

- por columna, 

- por celda.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000156"></a>
Para manipular las celdas de una control Tabla mediante programación, utilice la siguiente sintaxis:


```txt
<Column Name> [ <Row Number>]
```


Varias propiedades de WLanguage pueden ser usadas en las celdas de una control Tabla a través de la siguiente sintaxis:

- 
	```txt
	<Column Name>[<Row Number>]..<Property> = <Value>
	```


- 
	```txt
	<Table Name>[<Row Number>, <Column Number>]..<Property> = <Value>
	```





**Observación**: Si la celda corresponde a una casilla de verificación, no es necesario especificar el número de la opción en la casilla de verificación.

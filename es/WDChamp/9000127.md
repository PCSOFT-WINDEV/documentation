<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
WINDEV y WINDEV Mobile permiten añadir eventos opcionales a los propuestos por Default para los diferentes elementos de aplicación. Para obtener más información, consulte [Eventos opcionales](../WDChamp/1014004.md). 

Dependiendo del Event opcional utilizado, se puede recuperar información específica a través de las variables _EVE.LPARAM y _EVE.WPARAM.

<a name="NOTE3_1"></a>

## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
La control de visualización HTML se utiliza para visualizar una Page Web en una aplicación. Así, en 1 Line de código, sin utilizar un navegador externo, sus aplicaciones WINDEV pueden mostrar un Page de un sitio web.

La pantalla HTML control se puede manipular directamente en el código. Basta con utilizar el control de visualización HTML Variable. La pantalla HTML control Variable es el nombre de la pantalla HTML control. El tipo Variable es una cadena de caracteres.

También puede manipular los controles de visualización de HTML en el código utilizando [Visualización HTML Funciones control](../WDLang1/1000026264.md). 

Esta página de ayuda explica cómo manipular una pantalla HTML control en el código. 





## Manipulación de una pantalla HTML control en el código
<a name="manipulacion_una_pantalla_html_control_codigo_ELTTEXTE000159"></a>


### Visualización de información en una pantalla HTML control
<a name="visualizacion_informacion_una_pantalla_html_control_ELTPARAGRAPHE000024"></a>

Para mostrar la información en una control HTML, basta con asignar un valor

Por ejemplo:

- Asignación directa:
	
	```wl
	HTM_WebPage = "http://www.windev.com"
	```


- Asignación de Variable:
	
	```wl
	HTM_WebPage = MyPage
	```



También puede utilizar las propiedades [Valor](../Proprietes/2510130.md) o [URL](../Proprietes/2510132.md).


### Obtener el valor de una pantalla HTML control
<a name="obtener_valor_una_pantalla_html_control_ELTPARAGRAPHE000046"></a>

Para get el contenido de una pantalla HTML control (el usuario puede utilizar los enlaces de la página mostrada en el control):

- Basta con asignar un valor. Por ejemplo:
	
	```wl
	HTMLContent = HTM_HTMLDisplayControl
	```


- Utilice la propiedad [Valor](../Proprietes/2510130.md). 







### Impedir que el usuario utilice un enlace en una Page
<a name="impedir_que_usuario_utilice_enlace_una_page_ELTPARAGRAPHE000060"></a>

Mediante Default, en la pantalla HTML control, el usuario puede utilizar los distintos enlaces de la página mostrada en el control. 

Para limitar el uso de estos enlaces, puede utilizar la opción "Antes de cargar la página HTML" de Event. 

Para el siguiente ejemplo, vamos a suponer que la lista de sitios autorizados está contenida en una WLanguage array. 


```wl
PROCEDURE ChangePage(sURL is string, sDestination is string)
ArrAuthorizedSites is array of 2 strings
ArrAuthorizedSites[1] = "windev.com"
ArrAuthorizedSites[2] = "google.com"
let sDomain = URLExtractPath(sURL, urlDomain)
IF ArraySeek(ArrAuthorizedSites, asLinearFirst, sDomain) = -1 THEN
		ToastDisplay("Unauthorized site", toastShort, vaMiddle)
		RESULT False
END
RETURN True
```


Este código devuelve <u><u><u><u>False</u></u></u></u> si el sitio no está autorizado y muestra un toast. 







## Propiedades WLanguage asociadas

### Propiedades específicas de los controles HTML Display
<a name="propiedades_especificas_los_controles_html_display_ELTPARAGRAPHE000080"></a>Las siguientes propiedades se utilizan para gestionar las características de una pantalla HTML control en el código: 



|   |   |
| --- | --- |
| [Body](../Proprietes/1000025914.md) | El **Body** Property se utiliza para identificar y modificar el cuerpo HTML de un control Display HTML o un editor HTML control. |
| [Document](../Proprietes/1000026145.md) | La propiedad **Document** permite obtener o modificar el documento HTML asociado a un control de visualización HTML o Editor HTML. |
| [SecurityHtml](../Proprietes/1000019121.md) | El **SecurityHtml** Property se utiliza para get y cambiar el estado del mecanismo de seguridad de los controles HTML Display y HTML Editor. |
| [URL](../Proprietes/2510132.md) | La propiedad **URL** permite:<br><br>- Obtener el Address associated con un control.<br><br>- Cambiar el Address associated por un control.<br><br><br> |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control de visualización de HTML, consulte [Propiedades associated con controles HTML Display](../WDChamp/1410087146.md).

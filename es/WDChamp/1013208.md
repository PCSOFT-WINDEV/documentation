## Propiedades y métodos del ActiveX
<a name="propiedades_metodos_del_activex_ELTTEXTE000135"></a>
Un ActiveX es associated con propiedades y métodos. Estas propiedades y métodos pueden ser usados en WLanguage para manipular el ActiveX control directamente. Para ello, utilice la sintaxis de automatización de WLanguage.

Por ejemplo:


```wl
// AVIReader is an ActiveX control
AVIReader>>Zoom=12 // Assign a property
AVIReader>>Pause(10) // Call a method
```

<a name="NOTE2_2"></a>
Para conocer las propiedades y métodos de associated con un ActiveX, puedes:

- ver la documentación del ActiveX control.

- utilizar [WDXVIEW](../WDXView/3526002.md) suministrada con WINDEV.

- usar la terminación de automatic de WINDEV. Las propiedades y métodos del ActiveX son propuestos automáticamente por el editor de código. Por ejemplo, para el ActiveX del "Navegador WEB de Microsoft", nosotros get:

## Eventos associated con el ActiveX
<a name="eventos_associated_con_activex_ELTTEXTE000165"></a>
WLanguage permite gestionar los eventos generados por un ActiveX control con [ActiveXEvent](../WDLang1/3025010.md). Esta función le permite asociar un Procedure de su proyecto con el Event solicitado.

Por ejemplo:


```wl
// Initialization code of the ActiveX control named AVIReader
ActiveXEvent("My_procedure","AVIReader","OnClick")
// "my procedure" will be called whenever the 
// "OnClick" will be triggered by the "AVIReader" control.
```


Para averiguar los eventos associated con un ActiveX, puedes:

- ver la documentación del ActiveX control.

- utilizar [WDXVIEW](../WDXView/3526002.md) suministrada con WINDEV.

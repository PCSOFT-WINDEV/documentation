## Presentación
<a name="presentacion_ELTTEXTE000190"></a>
Para optimizar el estilo de sus pantallas, WINDEV y WINDEV Mobile le permiten usarlas:

- imágenes en sus ventanas. Estas imágenes se utilizan como fondo de la ventana. Sus controles se muestran Above este Image.

- colores de fondo. Se pueden utilizar varios colores estándar.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) También tiene la posibilidad de construir colores Gradient Background. Estos colores de Gradient se pueden utilizar: 

- en la ventana Background,

- en segundo plano Color para los diferentes controles de las ventanas. 




**Observación**: no se puede utilizar la función Aero Snap en ventanas con plantilla recortada.. Aero Snap requiere una ventana estándar redimensionable.

## Background imágenes de ventanas
<a name="background_imagenes_ventanas_ELTTEXTE000214"></a>


### Imagen de fondo
<a name="imagen_fondo_ELTPARAGRAPHE000030"></a>

Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).




Ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen_Imagefond.gif)


**Observación**: Para personalizar sus ventanas, también puede utilizar un plantilla. A plantilla es una interfaz preestablecida. Para obtener más información, consulte [Plantilla](../Editeurs/2016011.md).

### Configuración de las imágenes de Background
<a name="configuracion_las_imagenes_background_ELTPARAGRAPHE000055"></a>

La configuración se define en la pestaña "Imagen" de la descripción de la ventana.

Para seleccionar el fondo Image, vaya a "Fondo Image" y haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif): 

- La opción "Navegar" se utiliza para seleccionar un archivo Image accesible desde su ordenador.

- La opción "Catálogo" permite seleccionar una imagen del catálogo de imágenes.




**Observación**: También puede arrastrar y soltar el archivo Image encontrado en el explorador hacia la ventana de WINDEV o WINDEV Mobile.

## Color de fondo
<a name="color_fondo_ELTTEXTE000275"></a>


### Selección del fondo Color
<a name="seleccion_del_fondo_color_ELTPARAGRAPHE000109"></a>

Para seleccionar el fondo Color de una ventana:

1. Abra la ventana de descripción.

2. En la pestaña "Estilo", abra la lista "Color de fondo". Se muestra una lista de colores disponibles: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Colores disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen_couleur_fond.gif)





3. Seleccione el fondo solicitado Color.

4. Valide la ventana de descripción.



<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Color fondo degradado
<a name="color_fondo_degradado_ELTPARAGRAPHE000132"></a>

En la ventana de selección de los colores de fondo, Gradient varios colores de fondo Default. La opción "Otros Gradient s" permite mostrar la ventana de configuración de Gradient.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen_degrade.gif)


Esta ventana le permite:

- Seleccione un gradiente Color propuesto por Default.

- Modificar uno de los colores de Gradient propuestos por Default: tiene la posibilidad de configurar los colores, el número de colores que tiene en cuenta la Gradient y el ángulo de la Gradient. Sólo usa tu creatividad.




**Observación**: Los Gradient Color s también pueden utilizarse como Color de fondo para los controles: Para ello, modifique el color de fondo del borde externo del control en la pestaña "Estilo" de la ventana de description.

## Presentación
<a name="presentacion_ELTTEXTE000269"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

La control Multimedia sirve para visualizar fácilmente un vídeo en el navegador.
 Observación: En versiones anteriores, esta control se llamaba control Video.

El control busca el mejor reproductor de vídeo encontrado en el ordenador cliente según el formato del archivo de vídeo a visualizar. 

Si el navegador utilizado es compatible con el estándar HTML 5, el vídeo se reproducirá automáticamente sin ningún controlador de vídeo específico.. 

El vídeo puede controlarse en código de navegador a través de [Funciones multimedia](../WDLang1/1000015005.md).


### Creación de un control multimedia
<a name="creacion_control_multimedia_ELTPARAGRAPHE000029"></a>

Para crear una control multimedia: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Imagen" y seleccione "Control Multimedia".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

En las características de la control, tienes la posibilidad de elegir directamente el archivo de vídeo a visualizar. El archivo también se puede asignar por programación (en código del servidor).

## Características de una control multimedia
<a name="caracteristicas_una_control_multimedia_ELTTEXTE000299"></a>


### Fuente de vídeo (Archivo de vídeo)
<a name="fuente_video_archivo_video_ELTPARAGRAPHE000057"></a>

El archivo de vídeo puede asignarse a la control sólo en código de servidor.

El archivo de vídeo puede ser:

- un ruta y un nombre de archivo: en este caso, el archivo se buscará en el directorio _WEB (o en uno de sus subdirectorios), el ruta dado a la control debe ser relativo a este directorio.

- una URL (que comienza con http:// o https://) que termina con el nombre de un archivo de vídeo (por ejemplo: http://videoserver.mycompany.com/films/presentation.wmv)

- una URL (que comienza con http:// o https://) que no termina con el nombre de un archivo de vídeo (por ejemplo: http://videoserver.mycompany.com/film.awp?id=51452). En este caso, hay que rellenar [Lector Multimedia](../Proprietes/1000019055.md).

- una URL completa de un video alojado fuera del sitio (video de YouTube por ejemplo). 
	Observación: En el caso de un vídeo de YouTube, la personalización de los subtítulos no está disponible.




El reproductor multimedia utilizado por el control depende del formato del archivo y del sistema operativo.
Si el navegador utilizado es compatible con el estándar HTML 5, el vídeo se reproducirá automáticamente sin ningún controlador de vídeo específico.. 
En la tabla que figura a continuación se resumen los principales casos:

| Extensión del archivo | Reproductor multimedia utilizado por el vídeo control |
| --- | --- |
| *Formato Windows Media*:<br><br>- .wmv<br><br>- .wvxx<br><br>- .wma<br><br>- .cera<br><br>- ...asf.<br><br>- .asx<br><br>- .avi<br><br>- ...wav...<br><br><br> | En Windows, se utiliza el reproductor de Windows Media.<br><br>En los demás sistemas operativos, el reproductor a utilizar es elegido por el navegador. |
| *Vídeo Flash*:<br><br>- .flv<br><br><br> | El software Flash se utilizará si se encuentra. De lo contrario, el reproductor de vídeo a utilizar es elegido por el navegador. |
| *MPEG-4 Part 14 (ISO/IEC 14496-14:2003)*:<br><br>- .mp4<br><br><br> | El software QuickTime se utilizará si se encuentra. De lo contrario, el reproductor de vídeo a utilizar es elegido por el navegador. |
| *MPEG*:<br><br>- .mpg<br><br>- .mpeg<br><br>- .mp3<br><br><br> | En Windows con INTERNET Explorer, se utiliza el reproductor de Windows Media.<br><br>En los otros sistemas operativos o con otros navegadores, el reproductor a utilizar es elegido por el navegador. |
| *Apple QuickTime*:<br><br>- .mov<br><br>- .aiff<br><br><br> | El software QuickTime se utilizará si se encuentra. De lo contrario, el reproductor de vídeo a utilizar es elegido por el navegador. |
| *Ogg Vobis*:<br><br>- ...ogg.<br><br>- ...ogm.<br><br>- ...oga.<br><br>- .ogv<br><br><br> | En Firefox (versión 3.5 y posteriores), se utiliza una etiqueta "&lt;VIDEO&gt;" en el estándar HTML 5.<br><br>Con los otros navegadores, el reproductor de vídeo a utilizar es elegido por el navegador. |
| Otros casos | El reproductor de vídeo a utilizar es elegido por el navegador. |


El reproductor de vídeo utilizado es devuelto por el [Lector Multimedia](../Proprietes/1000019055.md) Property.

**Observaciones**: 

- El Multimedia control no contiene ningún códec de reproducción. Los codecs apropiados deben ser instalados en la el equipo del usuario de la Web.

- Si la elección del reproductor multimedia utilizado fue hecha por el navegador, [las funciones multimedia](../WDLang1/1000015005.md) no puede ser utilizado.

- Dependiendo del reproductor multimedia utilizado, es posible que algunas funciones no estén disponibles (por ejemplo, el reproductor Flash de Adobe no es compatible con el parámetro "Permitir el modo de pantalla completa").

- Si el servidor web utilizado es IIS, se debe añadir un tipo MIME al servidor web para cada tipo de vídeo.. 

- Es posible que el navegador utilizado no sea compatible con todos los formatos. Para comprobar si el navegador soporta el formato deseado (AVI por ejemplo), basta con soltar el archivo deseado en el navegador. 





### Subtítulos
<a name="subtitulos_ELTPARAGRAPHE000167"></a>

Para añadir subtítulos a un archivo multimedia, debe: 

1. Haga clic en el botón "+". 

2. En la pantalla tabla, seleccione: 

	- el tipo de subtítulos (diálogos, comentarios, ....). 

	- el idioma de los subtítulos. 

	- el título de los subtítulos. Este subtítulo permite a los usuarios seleccionar el tipo de subtítulos en tiempo de ejecución. 

	- el archivo que contiene los subtítulos. La extensión de este archivo puede ser "vtt", "ttml" o "srt".. 







Atención: 

- Esta característica es una característica de HTML 5. La página debe estar en formato HTML 5. 

- Esta característica es implementada directamente por el navegador. Dependiendo del navegador utilizado, algunas características pueden ser soportadas o no. 





### Otras opciones
<a name="otras_opciones_ELTPARAGRAPHE000185"></a>

- **Iniciar automáticamente**: Esta opción se utiliza para iniciar automáticamente el vídeo cuando se muestra la página.

- **Repetir automáticamente**: Esta opción se utiliza para reproducir el vídeo en bucle.  

- **Permitir modo de 'pantalla completa'**: Esta opción permite al usuario cambiar al modo de "pantalla completa".. 

- **Barra Multimedia**: Esta opción se utiliza para mostrar la barra de gestión de vídeo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_multimedia.gif)





**Atención**: Las opciones de esta control dependen del tipo de vídeo reproducido.

## Manejo de un control multimedia mediante programación
<a name="manejo_control_multimedia_mediante_programacion_ELTTEXTE000335"></a>


### Modificación del vídeo de forma dinámica mediante programación
<a name="modificacion_del_video_forma_dinamica_mediante_programacion_ELTPARAGRAPHE000216"></a>Para cambiar dinámicamente el archivo de vídeo associated con el control, basta con asignar el nuevo archivo al control. Ejemplo: 


```wl
// Modify the video displayed
// The new video is found in a "Video" subdirectory of the _WEB directory of the project
VIDEO_Video1 = "/" + FolderWeb() + "/Video/Wildlife.wmv"
```


## Editor de diagramas: cinta de opciones y paneles
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000205"></a>
El control Editor de diagramas incluye diferentes características a las que se puede acceder mediante: 

- La cinta de opciones: abrir, guardar, buscar, imprimir, etc.

- El panel "Biblioteca", que presenta los diferentes elementos que se pueden utilizar en el diagrama. 

- El panel "Modificador", que permite editar y configurar el elemento actual.  




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## La cinta de opciones
<a name="cinta_opciones_ELTTEXTE000229"></a>
La cinta de opciones del control Editor de diagramas permite realizar todas las acciones básicas en un diagrama. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003.gif&type=thumb)
Esta cinta de opciones contiene los siguientes grupos: 

- **Archivo**: Permite abrir, imprimir o guardar un diagrama.

- **Estilo**: Permite definir el estilo de los elementos. 

- **Portapapeles**: Permite gestionar el portapapeles (copia). 

- **Zoom**: Permite acercar o alejar el diagrama.

- **Modo de edición**: Permite dibujar a mano alzada.

- **Acción**: Permite rehacer o deshacer una acción. 

- **Edición**: Permite gestionar las opciones de visualización.

- **Buscar**: Permite buscar en el diagrama.   





### Grupo "Archivo"
<a name="grupo_archivo_ELTPARAGRAPHE000036"></a>

El grupo **Archivo** permite abrir, exportar, imprimir o guardar un diagrama. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%202.gif)
Es posible: 

- Crear un nuevo diagrama.

- Abrir un archivo de diagrama existente. 

- Exportar el diagrama actual a PNG o SVG. 

- Imprimir el diagrama actual. La ventana de configuración de la impresión se abre (con una vista previa del documento). Puede configurar: 

	- las propiedades de la impresora: Nombre de la impresora, orientación, modo de impresión, número de copias. 

	- las páginas a imprimir: todas las páginas, la página actual o una selección de páginas (páginas separadas por '-' para una selección, página separada por ',' para páginas independientes). 


 También es posible:

- previsualizar las diferentes páginas a imprimir. 

- exportar como archivo PDF o como correo electrónico con el archivo PDF adjunto (botón "Exportar"). 

- Guardar el diagrama actual. Esta opción guarda el diagrama en su estado actual. Puede guardar el archivo actual o utilizar la opción "Guardar como". 





### Grupo "Estilo"
<a name="grupo_estilo_ELTPARAGRAPHE000058"></a>

El grupo **Estilo** permite aplicar el formato de un elemento a otro elemento. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%203.gif)
Para utilizar esta opción:: 

1. Seleccione el elemento del que desea copiar el formato. 

2. Seleccione la opción "Copiar formato" en la cinta de opciones. 

3. Haga clic en el elemento deseado. El formato del primer elemento se aplicará al segundo elemento.  




Observación: Puede copiar el formato de un elemento determinado y aplicarlo a otro tipo de elemento (por ejemplo, un círculo y un rectángulo). 


### Grupo "Portapapeles"
<a name="grupo_portapapeles_ELTPARAGRAPHE000071"></a>

El grupo **Portapapeles** permite copiar el contenido de la selección. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%204.gif)
Este contenido se puede pegar en otra aplicación (bloc de notas, etc.). 

Observación: La copia también puede realizarse a través del menú contextual de la selección. 


### Grupo "Modo de edición"
<a name="grupo_modo_edicion_ELTPARAGRAPHE000082"></a>

En el grupo **Modo de edición**, puede activar o desactivar el modo "Forma libre". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%208.gif)
Si se activa este modo, el cursor se convierte en un pincel y permite dibujar a "mano alzada" en el editor de diagramas. Para volver al modo de edición estándar, haga clic de nuevo en la opción "Forma libre". 


### Grupo "Zoom"
<a name="grupo_zoom_ELTPARAGRAPHE000089"></a>

El grupo **Zoom** permite acercar y alejar el diagrama. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%205.gif)
Es posible: 

- Ver el diagrama con un zoom del 100%.. 

- Ajustar el diagrama para que se muestren todos los elementos (opción "Ajustar a la ventana")

- Personalizar el zoom. 





### Grupo "Edición"
<a name="grupo_edicion_ELTPARAGRAPHE000100"></a>

El grupo **Edición** contiene las opciones de visualización de los elementos del diagrama. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%207.gif)
Es posible: 

- Traer el elemento seleccionado delante de los demás elementos,

- Enviar el elemento seleccionado detrás de los demás elementos,

- Mover el elemento seleccionado un nivel hacia delante o hacia atrás,

- Agrupar los elementos seleccionados,

- Desagrupar la selección. 





### Grupo "Buscar"
<a name="grupo_buscar_ELTPARAGRAPHE000113"></a>

El grupo **Buscar** permite buscar un texto determinado en el diagrama. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0003%201.gif)


Para realizar una búsqueda: 

1. Haga clic en el cuadro de búsqueda de la cinta de opciones (o pulse Ctrl + F). 

2. Introduzca el texto que desea buscar. A medida que se escribe, se seleccionan los elementos que coinciden con la entrada. Las flechas permiten seleccionar el siguiente elemento si el diagrama contiene varios resultados.

## Panel "Biblioteca"
<a name="panel_biblioteca_ELTTEXTE000295"></a>
El panel "Biblioteca" contiene los diferentes elementos que se pueden insertar en el diagrama. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0004%203.gif)


Cada pestaña del panel Biblioteca corresponde a un diagrama. Este diagrama contiene diferentes elementos o imágenes que se pueden reutilizar en el editor de diagramas.  

El panel Biblioteca permite: 

- Crear elementos en el diagrama (arrastrar y soltar imágenes en el diagrama). 

- Crear una biblioteca personalizada (opción "Nueva biblioteca" del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0004%201.gif)). Esta opción guarda un archivo de biblioteca con la extensión ".wddiag". Este archivo aparece en el panel Biblioteca. Puede mover los elementos del diagrama actual a este panel para importarlos a la biblioteca. 

- Cargar una biblioteca personalizada (opción "Cargar una biblioteca" del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0004%201.gif)).

## Panel "Modificador"
<a name="panel_modificador_ELTTEXTE000319"></a>
El panel "Modificador" proporciona información sobre el elemento seleccionado y, si es necesario, le permite editar esta información. <br>![Características de la página](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_diagrammes_Ruban%20-%20HC%20N%B0004%202.gif)


El panel "Modificador" permite: 

- Definir el formato del texto en un área de texto: fuente, tamaño, posición, etc.

- Definir las características de una forma: Color de fondo, color del borde, estilo del borde, etc.

- Definir las características de los conectores: enrutamiento, estilo de la imagen de origen, estilo de la imagen de destino, etc.

- Definir las opciones de la página, etc.

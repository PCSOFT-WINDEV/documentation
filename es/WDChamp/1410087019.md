


## Manejo de una pancarta deslizante control mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000194"></a>
Una pancarta deslizante control puede ser manejada a través de la programación. Para hacerlo, usa la Variable de Sliding Banner control en el código.

La Variable del control Sliding Banner corresponde al número del panel Active. Esta variable es un número entero.

Para manipular una pancarta deslizante control mediante programación, WEBDEV propone el [Funciones de la pancarta corredera control](../WDLang2/1000022617.md).

En esta página de ayuda se explica cómo manipular una pancarta deslizante control mediante programación. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Planos de deslizamiento y navegación
<a name="planos_deslizamiento_navegacion_ELTTEXTE000218"></a>


### automatic secuencia de diapositivas en un Banner deslizante control
<a name="automatic_secuencia_diapositivas_banner_deslizante_control_ELTPARAGRAPHE000031"></a>

Para utilizar una secuencia automatic en una Bandera deslizante control, puede: 

- marque "Iniciar secuencia automáticamente" en la pestaña "Detalles" de la ventana de control description.

- utilizar la función WLanguage [BannerStartSliding](../WDLang2/1000022511.md). 




En este caso, los diferentes planos de Visible de Sliding Banner control se mostrarán automáticamente en el control: 

- según el orden definido en la pestaña "General" del Banner Deslizante control. 

- con la duración de la pantalla definida en la pestaña "General" del Banner deslizante control. 


Para detener el deslizamiento de automatic, utilice [BannerStopSliding](../WDLang2/1000022512.md). 
<a name="NOTE2_2"></a>


### Cambio de diapositivas con las viñetas y los botones izquierdo y derecho
<a name="cambio_diapositivas_con_las_vinetas_los_botones_izquierdo_derecho_ELTPARAGRAPHE000060"></a>

Si se utilizan los botones izquierdo y derecho, así como las balas propuestas en el Sliding Banner control, no se requiere ninguna programación específica. Ir de un avión a otro se gestiona automáticamente. 

Observaciones: 

- El orden de los aviones es el definido en la ventana de description del Banner Deslizante control.

- Los aviones de Invisible son ignorados.   



<a name="NOTE2_3"></a>


### Personalizar el deslizamiento de diapositivas
<a name="personalizar_deslizamiento_diapositivas_ELTPARAGRAPHE000074"></a>

Si se usan control s específicos para ir de un avión a otro (botones específicos, por ejemplo, colocados fuera de control), se tiene la habilidad de manejar el movimiento de un avión a otro: 

- utilizando las siguientes funciones de WLanguage: 
	


|   |   |
| --- | --- |
| [BannerFirst](../WDLang2/1000022513.md) | Muestra el primer plano de un Banner deslizante control. |
| [BannerLast](../WDLang2/1000022516.md) | Muestra el último plano de un Banner deslizante control. |
| [BannerNext](../WDLang2/1000022514.md) | Muestra el siguiente plano de un Banner deslizante control. |
| [BannerPrevious](../WDLang2/1000022515.md) | Muestra el plano anterior de un Banner Deslizante control. |

- utilizando [Valor](../Proprietes/2510130.md) para mostrar el plano deseado. 
	Por ejemplo: 
	
	```wl
	// Displays plane 3 in the Sliding Banner control
	BAN_Banner.Value = 3
	```


- asignando el valor del plano deseado a la pancarta deslizante control. 
	Por ejemplo: 
	
	```txt
	NameSlidingBannerControl = PlaneNumber
	```



Observación: Los números de los aviones y su orden se definen en la ventana de description del Banner Deslizante control.

## Manejando los aviones de un Sliding Banner control
<a name="manejando_los_aviones_sliding_banner_control_ELTTEXTE000284"></a>


### Hacer un avión Invisible
<a name="hacer_avion_invisible_ELTPARAGRAPHE000105"></a>

Puede utilizar [Visible](../Proprietes/2510138.md) para hacer un plano Visible (y a la inversa). La visibilidad de los planos se tiene en cuenta a la hora de navegar por los diferentes planos utilizados por el control de la Bandera Deslizante (automatic o iteración programada). 

Por ejemplo, el siguiente código se utiliza para hacer de Invisible el avión 5 que se encontró en el BAN_Ad control en agosto: 


```wl
TodayDate is Date
IF TodayDate.Month = 8 THEN
	BAN_Ad[5].Visible = False
END
```
Consejo: Puedes preparar un avión (para una promoción por ejemplo) y hacerlo Visible en una fecha determinada, por un período determinado. 


### Recuperar el avión actual
<a name="recuperar_avion_actual_ELTPARAGRAPHE000120"></a>

Para recuperar el número del avión actual: 

- realice una asignación simple. Por ejemplo:
	
	```txt
	ActivePlaneNum = NameSlidingBannerControl
	```


- utilice la propiedad [Valor](../Proprietes/2510130.md).

## Propiedades disponibles para un Sliding Banner control
<a name="propiedades_disponibles_para_sliding_banner_control_ELTTEXTE000314"></a>
Varias propiedades de WLanguage pueden ser usadas para manipular: 

- el estandarte deslizante control. 

- los aviones associated con el Banner deslizante control. 


Para conocer la lista completa de propiedades de WLanguage que pueden utilizarse con una pancarta deslizante control, consulte [Propiedades associated con controles deslizantes Banner](../WDChamp/1410087018.md).




## Los gráficos de la "Columna
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000227"></a>
Los datos se interpretan como "columna" cuya altura es proporcional al valor de los datos.

Los gráficos de columnas se utilizan para representar la tendencia de los datos en un período de tiempo determinado.. También se utilizan para realizar comparaciones entre elementos.

Las categorías están organizadas horizontalmente y los valores están organizados verticalmente para resaltar su variación en el tiempo.

**Observación**: También está disponible un gráfico de columnas horizontal.

## Características de un gráfico de columna
<a name="caracteristicas_grafico_columna_ELTTEXTE000251"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Genhisto.gif&type=thumb)
En este cuadro:

**1. Valores de los datos.** 
Este parámetro aparece si se marca "Mostrar valores" en la pestaña "Detalles" del control Gráfico description.

**2. Título del eje vertical**. 
Este parámetro se muestra si el título del eje se definió en la pestaña "General" de la control Gráfico description. Este parámetro también puede definirse mediante programación con [grAxisTitle](../WDLang3/3042057.md). También se puede definir un título para el eje horizontal.

**3. Marca en el eje vertical**. 
La marca es calculada automáticamente por Default. La pestaña "General" del control description se utiliza para:

- definir una marca específica.

- definir una escala en el eje X en función del tiempo. Esta opción opera con datos de Fecha o FechaHora provenientes de una control Tabla o de un ítem del HFSQL. Esta opción no puede utilizarse con una serie rellenada mediante programación.

- definir una escala logarítmica en el eje Y.

- especificar la orientación de las marcas.




También tiene la posibilidad de utilizar la escala secundaria para una o más series (pestaña"Series").

**4. Título del gráfico**. 
Las características del título del gráfico pueden definirse en la pestaña "Detalles" del control description ("Editar título del gráfico").

Puede definir el título del gráfico, su posición y su estilo.

**5. mito**. 
Las características de la leyenda pueden definirse en la pestaña "Detalles" del control description ("Editar leyenda del gráfico").

Puedes definir su posición y su estilo.

**6. Etiquetas de serie**. 
El título de la leyenda corresponde al título de las diferentes series mostradas en el gráfico (definido en la pestaña "Series").

**7. Color de serie**. 
Se puede definir un color para cada serie en la pestaña "Serie" del control description. Un automatic Color es usado por Default.

**8. Etiqueta de categoría**. 
Las etiquetas de las categorías se pueden definir en la pestaña "Serie" del control description.

**9. Espacio entre las columnas del gráfico de columnas**. 
Este espacio está definido en la pestaña "General" del control description.

**10. Líneas de cuadrícula**. 
Las cuadrículas horizontales y verticales pueden definirse en la pestaña "General" del control description.

### Características adicionales
<a name="caracteristicas_adicionales_ELTPARAGRAPHE000125"></a>

- ***Disposición de columnas en el gráfico de columnas***: las columnas del gráfico de columnas pueden ser agrupadas, apiladas o en profundidad (pestaña "General" del control description). Los gráficos de columnas apilados no soportan los valores negativos.

- Durante una visualización en 3 dimensiones, se puede utilizar ***columnas cilíndricas***.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) A ***acercamiento*** (horizontal y/o vertical) se puede aplicar en un gráfico de columnas. En este caso, el usuario puede realizar un zoom en el gráfico para ver un dato.

- También tienes la capacidad de mostrar ***y Image en el gráfico Background***. Esta imagen se puede seleccionar en la pestaña "Detalles" del control description.

- Puede utilizar una imagen de ***para las columnas que se encuentran en el gráfico de columnas*** (pestaña "Serie" en la ventana del gráfico description).

- Puedes definir ***un Color específico para el Border de las columnas*** que se encuentran en los gráficos de las columnas:

	1. Verifique que el color de los bordes no sea automático (opción "Colores de borde automáticos" en la pestaña "Detalles" de la descripción del control).

	2. En la pestaña "Estilo", seleccione el elemento "Borde Color" y seleccione el Color solicitado.




- La pestaña "Estilo" sirve para configurar el estilo de varios elementos: Color de cuadrículas y ejes, fuente para el título y los ejes, ...

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos "Columna" se pueden mostrar con una animación. Para animar este tipo de gráfico, basta con marcar "Activar animaciones" en la pestaña "Detalles" del control description. 

- **Gráfico de columnas apiladas**: Por Default, un tooltip muestra el valor de cada serie. 
	Tiene la posibilidad de mostrar el total de la pila además del total de cada serie utilizando [grTooltip](../WDLang3/3042010.md). Ejemplo:  
	
	```wl
	grTooltip(MySelf, grTooltipFormatCrosshairHeader, "[%CATEGORY%], Sum = [%VALUE%]")
	```


- **Agregación de datos**: 

	- Para los gráficos sin escala de tiempo, la agregación no está habilitada por Default.

	- Para los gráficos con escala de tiempo, la agregación de Default es la suma de las columnas.




- El algoritmo de agregación puede ser definido por [grColumnAggregation](../WDLang3/1000023015.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos de"Columna" se pueden mostrar en tiempo real. Para obtener más información, consulte [Gráfico en tiempo real](../WDChamp/1000021037.md).

## Los diferentes tipos de controles Campo de entrada
<a name="los_diferentes_tipos_controles_campo_entrada_ELTTEXTE000300"></a>


### Control Campo de entrada de tipo Texto
<a name="control_campo_entrada_tipo_texto_ELTPARAGRAPHE000011"></a>

Este tipo de control permite mostrar o introducir cadenas de caracteres (nombre, dirección, etc.).

Observaciones: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los controles Campo de entrada de tipo texto pueden mostrar un botón "Borrar" (X). Este botón permite al usuario eliminar el contenido del control.. El botón puede aparecer en el área de entrada: 

	- solo cuando el usuario introduce datos en el control. 

		- solo cuando el control pierde el foco. 
						

		- siempre (tanto si el cursor de texto está dentro como fuera del control). 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Cuando un control Campo de entrada de tipo texto se amplía en el editor, se convierte automáticamente en un control multilínea.



<a name="NOTE1_2"></a>


### Control Campo de entrada de tipo Texto multilínea
<a name="control_campo_entrada_tipo_texto_multilinea_ELTPARAGRAPHE000041"></a>

Este tipo de control permite mostrar o introducir varias líneas separadas por un retorno de carro (CR).

Observaciones: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si las líneas están separadas por un retorno de carro (CR), es posible mostrar una barra de desplazamiento vertical u horizontal.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) También puede mostrar direcciones web y rutas de archivos como enlaces.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Cuando un control Campo de entrada de tipo texto se amplía en el editor, se convierte automáticamente en un control multilínea.

### Control Campo de entrada de tipo Texto RTF
<a name="control_campo_entrada_tipo_texto_rtf_ELTPARAGRAPHE000062"></a>

Este tipo de control permite mostrar e introducir contenido RTF. Para obtener más información, consulte [Manipular texto RTF en un control Campo de entrada](../WDChamp/1013190.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para las aplicaciones que se ejecutan en Windows Vista (o posterior) o en Tablet PC, puede incluir la opción "Entrada manuscrita (PC tablet)". Esta opción permite al usuario escribir directamente en el control utilizando el ratón o un lápiz óptico. Esta funcionalidad está disponible de forma predeterminada en las aplicaciones móviles.

### Campo de entrada de tipo Markdown (solo lectura)
<a name="campo_entrada_tipo_markdown_solo_lectura_ELTPARAGRAPHE000104"></a>

Este tipo de control permite mostrar texto en formato Markdown. El usuario puede seleccionar o desplazarse por el texto. 

Recordatorio: La opción "Admitir Markdown" en la pestaña "UI" de la ventana de descripción de los controles Campo de entrada solo se aplica al título, tooltip y tokens asociados al control. Esta propiedad no se aplica al contenido del control Campo de entrada.

Para cambiar el tipo del control Campo de entrada (de Texto a Markdown, por ejemplo), utilice la propiedad [DisplayAsMarkdown](../Proprietes/1410089587.md).
<a name="NOTE1_5"></a>


### Control Campo de entrada de tipo Contraseña
<a name="control_campo_entrada_tipo_contrasena_ELTPARAGRAPHE000122"></a>

En este tipo de control, todos los caracteres introducidos son reemplazados por asteriscos en la pantalla. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El usuario puede comprobar la contraseña introducida (FAA). Para obtener más información, consulte [Mostrar una contraseña (FAA)](../WDChamp/9500129.md). 
<a name="NOTE1_6"></a>


### Control Campo de entrada de tipo Numérico, Monetario o Monetario + Euro
<a name="control_campo_entrada_tipo_numerico_monetario_monetario_euro_ELTPARAGRAPHE000138"></a>

Estos diferentes tipos de controles permiten manipular valores numéricos: 

- El control Campo de entrada de tipo Numérico permite mostrar o introducir valores numéricos (peso, etc.). Si el control no es lo suficientemente grande para mostrar todo el número, se muestran los caracteres "+++".

- El control Campo de entrada de tipo Monetario permite mostrar o introducir valores monetarios.

- El control Campo de entrada de tipo Monetario + Euro permite manipular precios, por ejemplo.
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este tipo de control Campo de entrada no está disponible.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los controles Campo de entrada de tipo Numérico, Monetario y Monetario + Euro pueden mostrar una **calculadora** para que los usuarios realicen cálculos (opción "Mostrar botón calculadora"). Para este tipo de control existen varias Funcionalidades Automáticas de la Aplicación (FAA). Por ejemplo, la posibilidad de cambiar el valor con la rueda del ratón. Para obtener más información, consulte [Funcionalidades Automáticas de la Aplicación para controles numéricos](../Editeurs/9500122.md).
<a name="NOTE1_7"></a>


### Control Campo de entrada de tipo fecha
<a name="control_campo_entrada_tipo_fecha_ELTPARAGRAPHE000166"></a>

Este tipo de control permite mostrar o introducir fechas.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El usuario puede utilizar las teclas "+" y "-" del teclado numérico para cambiar la fecha.
Si el control no está en una ventana emergente, se puede mostrar un botón de tipo calendario para permitir que el usuario introduzca una fecha (opción "Mostrar un botón calendario"). En este caso, se pueden definir las características del calendario. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ***Funcionalidad Automática de la Aplicación (FAA)***: Aún si no se muestra el botón Calendario, el usuario podrá introducir una fecha en un calendario (opción "Calendario" en el menú contextual del control). Para eliminar esta Funcionalidad Automática de la Aplicación, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafCalendar*. Para obtener más información, consulte [Funcionalidades Automáticas de la Aplicación para controles de tipo Fecha](../Editeurs/9500124.md).




## Image "clicable" en una ventana
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
WINDEV y WINDEV Mobile le ofrecen la posibilidad de utilizar imágenes en las que puede hacer clic.. Una Image seleccionable es una Image (mostrada en una control Imagen) que puede ser seleccionada por el usuario. Se ejecuta una Process específica (código de clic de la Image).

A diferencia de un botón, no se ve ningún efecto de presión cuando se hace clic en una Image.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Usando una Image pulsable
<a name="usando_una_image_pulsable_ELTTEXTE000147"></a>
Para usar una Image pulsable en una aplicación:

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Imagen".

2. Abra la descripción del control (opción "Descripción" del menú contextual de la imagen).

3. En la pestaña "Detalles", utilice los ajustes de clic: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_MenuContextuelImage%20-%20HC%20N%B0001%201.gif)


	- La opción "Esta imagen es un área de clic" hace que la imagen sea una imagen con enlace. El "clic" Event se añade automáticamente a la WLanguage Event s associated con el control.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) La opción "Prioridad para los clics" permite interceptar el clic en la imagen, independientemente del Z-Order del control Imagen en la ventana. Si esta opción no está seleccionada, el control Imagen debe aparecer en primer plano para interceptar el clic del ratón.

- Especificar si la Image será manejada por el usuario. Para obtener más información, consulte [Menú contextual de los controles de Image](../WDChamp/9500105.md).

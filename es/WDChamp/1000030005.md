
## Menú contextual de los controles del diagrama de Gantt (AAF)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000157"></a>
Cada control Diagrama de Gantt en una ventana tiene un menú Default que está a disposición de los usuarios.

Este menú, disponible sin programación, incluye varias acciones:

- Modificar la duración de una tarea,

- Borrar tareas,

- Agregar tareas, 

- Enlazar tareas,

- Imprime el gráfico de Gantt.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Gantt.gif)


**Observación**: Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

<a name="NOTE2_1"></a>


## Ajustes
<a name="ajustes_ELTTEXTE000181"></a>


### Configuración del menú automatic en un control Diagrama de Gantt
<a name="configuracion_del_menu_automatic_control_diagrama_gantt_ELTPARAGRAPHE000031"></a>

El menú contextual de control Diagrama de Gantt puede desactivarse. Para configurar la visualización del menú contextual associated con el control Diagrama de Gantt, debe:

1. Abra la ventana de descripción del control Tabla o Tabla TreeView.

2. Seleccione la columna "Diagrama de Gantt. 

3. Seleccione la pestaña "UI".

4. Elija el menú deseado en la opción "Menú contextual". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, aparecerá el menú automatic del control Diagrama de Gantt. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú FAA](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las FAA. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las FAA y restaurar el menú estándar de Windows para un control determinado.

- [AAFDisable](../WDLang1/1000022018.md) se utiliza para eliminar algunas opciones del menú contextual de las AAF específicas de los controles del diagrama de Gantt. Se pueden utilizar diferentes constantes según la opción de deshabilitar: 

	- *aafGanttAddDependency*: usado para deshabilitar: 

		- la adición de una tarea de condición previa,

		- la supresión de todas las tareas anteriores. 




- *aafGanttAddTask*: usado para deshabilitar la adición de una tarea. 

- *aafGanttEditTask*: usado para desactivar la edición de una tarea. 

- *aafGanttDeleteTask*: usado para deshabilitar la eliminación de una tarea. 

- *aafPrintReportOnTable*: usado para desactivar la impresión del diagrama de Gantt.

## Impresión de un diagrama de Gantt
<a name="impresion_diagrama_gantt_ELTTEXTE000205"></a>
<a name="NOTE3_1"></a>
La opción "Imprimir" se utiliza para imprimir el contenido de la columna del diagrama de Gantt.   

Las características de impresión son las siguientes:

- El título de la impresión corresponde a la leyenda de la Tabla o control TreeView que muestra el control Diagrama de Gantt. Si no se ha especificado ningún título, reporte no muestra ningún título.

- La impresión respeta la Color utilizada en el control Diagrama de Gantt.

- La fecha de hoy está impresa en la parte superior derecha Corner.

- El número de la Page actual y el número total de Page s se imprimen en la parte inferior derecha Corner.

- El documento se imprime utilizando la dirección de escritura especificada con la propiedad [TextDirection](../Proprietes/2513020.md) a nivel de ventana y/o control Diagrama de Gantt.




**Observaciones**: 

- La opción "Imprimir" del menú contextual de las columnas del diagrama de Gantt requiere proporcionar las bibliotecas de impresión de WINDEV framework con el ejecutable.

- Para deshabilitar la opción "Imprimir" en el menú contextual de los controles del diagrama de Gantt, use [AAFDisable](../WDLang1/1000022018.md) con el *aafPrintReportOnTable* 00720074004f006e005400610062006c006500 constante.

- Para personalizar la reporte asociada a un control Diagrama de Gantt específico, debe crear un nuevo diagrama de Gantt reporte basado en este control Diagrama de Gantt. Esta reporte se asociará automáticamente a la control Diagrama de Gantt y se imprimirá mediante la opción "Imprimir".. Para obtener más información, consulte [Diagrama de Gantt reporte](../WDChamp/1011084.md).

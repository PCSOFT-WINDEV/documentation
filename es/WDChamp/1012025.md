
## Estandarización del W3C
			






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, todas las páginas generadas por WEBDEV cumplen con el estándar W3C. Por lo tanto, esta página de ayuda está obsoleta a partir de esta versión: las operaciones presentadas en esta página están disponibles a partir de las versiones anteriores a la versión 17.  



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000112"></a>
¿Debe cumplir con el estándar W3C (HTML 4.01) en sus proyectos WEBDEV? WEBDEV le ofrece la posibilidad de generar un código HTML que cumple con el estándar W3C.

Las páginas generadas por WEBDEV pueden ser enviadas al servicio de validación del estándar W3C ([http://validator.w3.org](http://validator.w3.org)). Para ello, guarde las páginas de su sitio antes de enviarlas.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000136"></a>


### Generación de páginas HTML que cumplen con el estándar W3C
<a name="generacion_paginas_html_que_cumplen_con_estandar_w3c_ELTPARAGRAPHE000027"></a>

Generar páginas HTML que cumplan con el estándar W3C:

1. Abra la descripción del proyecto (opción "Proyecto .. Descripción del proyecto").

2. En la pestaña "Avanzado" haga clic en el botón "Avanzado...".: se muestra la lista de opciones para generar el código HTML.

3. Seleccione "Respetar la norma transitoria W3C".

4. Regenera las páginas de tu sitio.




**Observación**: Para obtener una visualización correspondiente al estándar del W3C en todos los navegadores, debe proporcionar el archivo DTD correspondiente al estándar del W3C ("Incluir el reference al archivo DTD"). Sin embargo, esta opción puede desmarcarse si se producen problemas de visualización.
<a name="NOTE2_2"></a>


### Aplicación del estándar W3C a una página
<a name="aplicacion_del_estandar_w3c_una_pagina_ELTPARAGRAPHE000041"></a>

Para que una página cumpla con el estándar W3C:

1. Abra la página en el editor.

2. Abra la descripción de la página.

3. En la pestaña "Avanzado", seleccione "Personalizado" y haga clic en el botón [...].

4. Seleccione "Respetar la norma transitoria W3C".

5. Valide.




Esta opción se tendrá en cuenta en la próxima generación de la página.
<a name="NOTE2_3"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000062"></a>

- Una página con la opción W3C puede no pasar la validación de W3C si esta página contiene una página interna que no tiene la opción W3C.

- Una página con las opciones del W3C y "Incluir el reference al archivo DTD" puede no mostrarse correctamente si esta página contiene una página interna que no tiene las opciones del W3C y "Incluir el reference al archivo DTD".. Si una página contiene páginas internas de componentes externos para las que no se sabe si han sido generadas con las opciones W3C y "Incluir el reference al archivo DTD" y si la página no se muestra correctamente, desmarque "Incluir el reference al archivo DTD" para la página.

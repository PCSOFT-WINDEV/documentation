
## Vinculación de una control Casilla de verificación a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000097"></a>
Una control Combo Box puede estar vinculada a un elemento de un archivo de datos, consulta, etc.

Los siguientes tipos de artículos pueden estar vinculados a una control Casilla de verificación:

- Numérico,

- Booleano,

- Casilla de verificación.




**Si la casilla de verificación contiene una sola opción**el ítem se tomará por valor:

- 1 si la casilla está marcada

- 0 si la casilla no está marcada.




**If la casilla de verificación contiene varias opciones**, se requiere un elemento array. Cada elemento del elemento array corresponderá a una opción de casilla de verificación, y tomará el siguiente valor:

- 1 si la casilla está marcada

- 0 si la casilla no está marcada.

<a name="NOTE2_1"></a>


## Actualizar un artículo con los datos de la control Casilla de verificación
<a name="actualizar_articulo_con_los_datos_control_casilla_verificacion_ELTTEXTE000121"></a>
**1er caso: los controles de la ventana, de la página o de reporte están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control de la ventana, página o reporte serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana, de la página o de reporte no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada artículo.

1. Si la control Casilla de verificación tiene una sola opción: utilice la siguiente sintaxis para actualizar el elemento: 
	
	```txt
	<Data file>.<Item> = <Check Box control>
	```


2. Si la control Casilla de verificación tiene varias opciones y está vinculada a un elemento array: utilice la siguiente sintaxis para actualizar el elemento:
	
	```txt
	<Data file>.<Item>[Option index] = <Check Box control>[Option index]
	```


## Manejo de una control Enlace mediante programación (Página)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000098"></a>
WEBDEV le permite manipular a control Enlace a través de la programación. Para hacerlo, usa la Variable de la control Enlace en el código.

El Variable del control Enlace corresponde al nombre del control. Un control Enlace es siempre un control de texto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Descubrir y modificar la leyenda (código del servidor)
<a name="descubrir_modificar_leyenda_codigo_del_servidor_ELTTEXTE000122"></a>
Para manipular el título de un control Enlace, utilice el [Pie de foto](../Proprietes/2510053.md) Property.

Por ejemplo, la siguiente línea de código modifica el título del LINK_Email control:


```wl
LINK_Email.Caption = "Write to me directly"
```


**Observación**: Si la opción "..El valor representa la URL del enlace" está desmarcada (pestaña "Detalles" del enlace description), la leyenda del enlace también puede ser devuelta por [Valor](../Proprietes/2510130.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Recuperación y modificación de la URL del enlace (para un enlace "mailto" en el código del servidor)
<a name="recuperacion_modificacion_url_del_enlace_para_enlace_mailto_codigo_del_servidor_ELTTEXTE000146"></a>
Para manipular la URL de un "MailTo" de tipo control Enlace, basta con utilizar el [URL](../Proprietes/2510132.md) Property. 

Por ejemplo, la siguiente línea de código modifica el correo electrónico Address del LINK_Email control:


```wl
LINK_Email.URL = "address@laposte.net"
```


**Observación**: Si está marcada la opción "..El valor representa la URL del enlace" (pestaña "Detalles" del enlace description), la URL del enlace también puede ser devuelta por [Valor](../Proprietes/2510130.md).

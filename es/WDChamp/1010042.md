
## Manejo de las opciones de menú de WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000233"></a>
El menú contextual de opciones de menú permite realizar todas las operaciones necesarias para definir: 

- un menú desplegable.




Para manipular las opciones de un menú desplegable: 

- Seleccione el menú. 

- Haga clic en el menú seleccionado: un marco amarillo aparece alrededor. 

- El menú pasa al modo de edición. Puede hacer clic con el botón derecho en las opciones del menú para añadir otras opciones.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_creation_Option.gif)


Para manipular las opciones de un menú contextual: 

- Realice una de las siguientes acciones para abrir el Contexto control Menú: 

	- haga doble clic en el nombre en el panel "Explorador de proyectos". 

	- en la pestaña "Página", en el grupo "Edición", despliegue "Menús contextuales" y seleccione "&lt;NombreMenuContextual&gt;".




- Seleccione una opción. Puede hacer clic con el botón derecho en las opciones del menú para añadir otras opciones.




**Observaciones**: 

- Pulse Esc para salir del modo de edición (el borde amarillo desaparecerá).

- Para editar el texto de una opción en un menú desplegable o contextual, sólo tiene que seleccionar la opción y pulsar la barra espaciadora (o seleccionar "Editar título de opción" en el menú contextual): un marco amarillo aparece alrededor.

- Para asociar una acción a una opción de menú, puede utilizar la ventana description de la opción de menú o el código associated con la opción. Para obtener más información, consulte [Programación de un menú desplegable](../WDChamp/1010010.md).

- Para crear un menú desplegable emergente, basta con asociar la opción de menú a una ventana emergente en la aplicación Page. Para obtener más información, consulte [Opciones del menú que aparecen](#NOTE3_1). 

- Las siguientes funciones pueden utilizarse para añadir opciones a un menú contextual o desplegable existente: 
	


|   |   |
| --- | --- |
| [MenuAddMenu](../WDLang1/1000017266.md) | Agrega un nuevo menú a un menú desplegable o contextual de una ventana o página. |
| [MenuAddOption](../WDLang1/1000017262.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. |
| [MenuAddPopup](../WDLang1/1000020583.md) | Transforma una opción de menú de una Page para que esta opción abra una ventana emergente. |
| [MenuAddSeparator](../WDLang1/1000017270.md) | Añade un nuevo separador en un menú (desplegable o contextual). |
| [MenuAddURLOption](../WDLang1/1000018529.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú abre la página que corresponde a la URL especificada. |

## Las diferentes operaciones
<a name="las_diferentes_operaciones_ELTTEXTE000294"></a>


### Cambiar el título de una opción de menú
<a name="cambiar_titulo_una_opcion_menu_ELTPARAGRAPHE000073"></a>

Para cambiar el título de una opción de menú:

1. Seleccione la opción.

2. Pulse la barra espaciadora.

3. Escriba el nuevo título y pulse la tecla Intro para validar.



<a name="NOTE2_2"></a>


### Agregar una opción de menú
<a name="agregar_una_opcion_menu_ELTPARAGRAPHE000084"></a>

Para agregar una opción de menú:

1. Para añadir una opción de menú después de una posición determinada, haga clic con el botón derecho del ratón en la opción de menú deseada y seleccione "Añadir después de".

2. Presione la tecla Entrar.

3. Escriba el título de la nueva opción y pulse la tecla Intro para validar.


Observación: La tecla Esc se utiliza para Cancel la adición de una opción de menú.
<a name="NOTE2_3"></a>


### Añadir un separador
<a name="anadir_separador_ELTPARAGRAPHE000097"></a>

Para añadir un separador:

1. Seleccione la opción del submenú tras la cual debe añadirse el separador.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Añadir un separador".



<a name="NOTE2_4"></a>


### Añadir un submenú
<a name="anadir_submenu_ELTPARAGRAPHE000108"></a>

Para añadir un submenú:

1. Seleccione la opción de menú a la que se debe añadir una subopción.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Insertar un submenú".

4. Pulse la barra espaciadora.

5. Escriba el texto de la primera opción del nuevo submenú.

6. Pulse Intro para validar la adición.




Observaciones: 

- Puede utilizar el mismo método para añadir opciones en los menús y subopciones en los submenús.



<a name="NOTE2_5"></a>


### Borrar una opción de menú
<a name="borrar_una_opcion_menu_ELTPARAGRAPHE000136"></a>

Para eliminar una opción de menú:

1. Seleccione la opción de menú que desea eliminar.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción.

3. Seleccione "Borrar".


Observación: También puede borrar la opción de menú actual con la tecla Supr.

## Configuración de las opciones de un menú emergente (sólo menús desplegables)
<a name="configuracion_las_opciones_menu_emergente_solo_menus_desplegables_ELTTEXTE000342"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000152"></a>

En un menú "Popup", al pasar el ratón por encima de una opción de la barra de menús, se abre un Popup Page totalmente personalizable. . 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_popup_WB.gif)


La ventana emergente Page permite crear fácilmente menús gráficos. Tiene la posibilidad de insertar cualquier tipo de control en la ventana emergente. 
<a name="NOTE3_2"></a>


### Creación de las opciones de un menú emergente
<a name="creacion_las_opciones_menu_emergente_ELTPARAGRAPHE000163"></a>

Para crear las opciones de un menú emergente: 

1. Mostrar el menú en modo edición: el menú debe ser encerrado por una Line amarilla. 

2. Seleccione la opción del menú que debe mostrar la ventana emergente Page. 

3. Despliegue el menú contextual de la opción (clic con el botón derecho del ratón) y seleccione "Desplegar una ventana emergente Page".. 

4. Aparece la pestaña "UI" de la ventana description de la opción de menú. En esta ventana: 

	- Seleccione la ventana emergente Page que debe mostrarse. Si la ventana Page no existe, haga clic en "" para crearla.. 

	- Seleccione la posición de la ventana de diálogo en relación con la opción (abajo, a la derecha, a la izquierda, ....). 




5. Valide la ventana de descripción. Si ha elegido una nueva ventana emergente, la ventana emergente aparece en la creación en el editor. 

6. El popup associated con la opción también se puede ver en el editor: 

	- mostrar (si es necesario) el menú en modo de edición (con la Border amarilla). 

	- haga clic en la opción de menú associated con el popup. 





Observación: También tiene la capacidad de transformar una opción de menú existente en una opción que abre una ventana emergente.: 

1. Mostrar el menú en modo edición: el menú debe ser encerrado por una Line amarilla. 

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la opción y seleccione "Opción description".

3. En la pestaña "UI", seleccione la ventana emergente que debe ser associated con la opción.

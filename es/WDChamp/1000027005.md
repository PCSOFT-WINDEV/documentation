
## Vinculación de un deslizador WEBDEV a un elemento
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000124"></a>
Un deslizador WEBDEV puede estar vinculado a un elemento en un archivo de datos, consulta, etc.

**Para una corredera lineal**el elemento contiene el valor definido por el cursor de la barra deslizante. 

Las características del artículo son las siguientes:

- el elemento es de tipo **Integer**.

- el elemento contiene el valor de la posición del cursor del deslizador. Esta posición corresponde al valor de la corredera.

- el valor del elemento debe estar entre el valor mínimo y máximo del deslizador.




**Para un deslizador de rango**, dos elementos están ligados al control: 

- una posición que contiene el valor del cursor principal (valor inferior). 

- una posición que contiene el valor secundario (valor superior). 




Las características de los artículos son las siguientes:

- los artículos son **Números enteros**. 

- los elementos contienen los valores de los cursores deslizantes. Estos valores corresponden a los valores mínimo y máximo de la corredera.

- el valor de cada elemento debe estar entre el valor mínimo y máximo del deslizador.

## Manipulación del artículo y del control deslizante
<a name="manipulacion_del_articulo_del_control_deslizante_ELTTEXTE000148"></a>


### Actualización de la Control deslizante con el valor del artículo
<a name="actualizacion_control_deslizante_con_valor_del_articulo_ELTPARAGRAPHE000036"></a>

**1er caso: los controles de la página están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo relacionado) serán actualizados por [ArchivoPágina](../WDLang4/3044148.md).

**2º caso: los controles de la página no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:

- Control deslizante Lineal: 
	
	```txt
	<Slider control> = <Data file>.<Item>
	```


- Control deslizante de rango: 
	
	```txt
	<Slider control>.LowerValue = <Data file>.<Item1>
	<Slider control>.UpperValue = <Data file>.<Item2>
	```




<a name="NOTE2_2"></a>


### Actualización de la partida con los datos de la Control deslizante
<a name="actualizacion_partida_con_los_datos_control_deslizante_ELTPARAGRAPHE000059"></a>

**1er caso: los controles de la página están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la página serán actualizados por [PáginaArchivo](../WDLang4/3044022.md).

**2º caso: los controles de la página no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:

- Control deslizante Lineal: 
	
	```txt
	<Data file>.<Item> = <Slider control>
	```


- Control deslizante de rango:
	
	```txt
	<Data file>.<Item1> = <Slider control>.LowerValue
	<Data file>.<Item2> = <Slider control>.UpperValue
	```

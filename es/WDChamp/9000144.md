


## Zona fija en una Page: zona fija en la pantalla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000209"></a>
Un área fija en una página es una sección de la página que permanece Visible incluso si el usuario de la web mueve la barra de desplazamiento del navegador. Esta zona (así como los controles que se encuentran en ella) permanece Visible mientras el contenido de Page se desplaza. Esta área es fija. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_ZoneFixe.gif)
 WEBDEV ofrece una gestión de áreas fijas para varios tipos de elementos: 

- [las áreas de disposición](#NOTE2_1) (en modo Zonificación). 

- [los controles de Cell](#NOTE3_1). 

- [los popups](#NOTE4_1). 




**Condiciones de uso**: Para utilizar esta función, la Page debe generarse en HTML5. Se puede definir el modo de generación: 

- en la pestaña "Avanzado" del proyecto description, opción "Modo de generación. Para abrir la ventana de descripción del proyecto, en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". En este caso, el modo de generación se aplica a todas las páginas del proyecto.. 

- en la pestaña "General" de la página description, opción "Modo de generación. Para abrir la ventana de descripción de la página, seleccione la opción "Descripción" del menú contextual de la página. Esta opción permite especificar el modo de generación que se utilizará para el Page (puede diferir del modo de generación seleccionado para los demás Page del proyecto).

## Fijación de un área de disposición
<a name="fijacion_area_disposicion_ELTTEXTE000233"></a>
Para fijar una zona de disposición en una Page:  

1. Seleccione la zona de maquetación que desea fijar y abra la ventana description ("Zona description" en el menú contextual). 

2. En la pestaña "UI", seleccione la opción "Fijo". Hay tres modos disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_ZoneEpingle_modes.gif&type=thumb)


	- **siempre arreglado**: El área de disposición se fija según las opciones de anclaje definidas para el área. Estas opciones de anclaje también se definen en la pestaña "UI" de la ventana description del área de maquetación. Por ejemplo, si el área de disposición está anclada con las siguientes características:

	- X: Seguir el borde izquierdo del navegador, 

	- Y: Seguir el centro del navegador,

## Arreglar una control Celda
<a name="arreglar_una_control_celda_ELTTEXTE000257"></a>
Para fijar una control Celda en un Page: 

1. Visualizar la Page que contiene el control Celda. 

2. Seleccione la control Celda que desea corregir y abra la ventana description ("description" en el menú contextual). 

3. En la pestaña "UI", seleccione la opción "Fijo". 

4. Haz clic en "Parámetros" para gestionar cómo se fija la control. Hay tres modos disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_CelluleEpingle_modes.gif&type=thumb)


	- **Siempre fijo + anclaje**: La celda se fija según las opciones de anclaje definidas para la celda. Estas opciones de anclaje también se definen en la pestaña "UI" de la ventana de la célula description. Por ejemplo, si el control Celda está anclado con las siguientes características:

	- X: Seguir el borde izquierdo del navegador

	- Y: Seguir el centro del navegador

## Arreglar una ventana emergente Page
<a name="arreglar_una_ventana_emergente_page_ELTTEXTE000281"></a>
Una ventana emergente sólo puede arreglarse mediante programación. 

Para ello, utilice la constante *popupFixed* durante la llamada a [PopupDisplay](../WDLang2/1000019788.md). Por ejemplo: 

```wl
PopupDisplay(POPUP_Explanation, popupFixed + popupCenterLeft)
```

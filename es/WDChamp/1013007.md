## Sugerencia sobre los valores del gráfico
<a name="sugerencia_sobre_los_valores_del_grafico_ELTTEXTE000201"></a>
 
Por Default, en un gráfico, se muestra un tooltip cuando se pasa el cursor del ratón por un dato. Este tooltip contiene el valor y el título de las series y categorías. Esta información sobre la herramienta puede ser personalizada por [grTooltip](../WDLang3/3042010.md). 

Las características de este tooltip son las siguientes: 

- Visualización inmediata para una mejor visibilidad,

- Borde de la punta de la herramienta que utiliza el Color de la serie. La asociación entre el tooltip y la serie es inmediata. 

- Punta de herramienta Semi-Transparent: la tabla de abajo es Visible.

- Flecha apuntando con precisión al punto de destino.

## Las diferentes opciones del menú contextual
<a name="las_diferentes_opciones_del_menu_contextual_ELTTEXTE000225"></a>
Un gráfico es associated con un menú contextual de Default (mostrado a través de un clic con el botón derecho del ratón).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_MenuCtx.gif)


Este menú contextual permite a los usuarios finales realizar varias operaciones.  

El menú contextual de un gráfico se utiliza para:

- configurar el tipo de gráfico: pastel, Line, columna, gráfico de acciones, ... Esto le permite elegir el mejor modo de visualización de acuerdo con los datos mostrados.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=graphe_Courbe.gif&type=thumb)

	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=graphe_Aire.gif&type=thumb)


- mostrar (o no) la leyenda y configurar su posición.

- mostrar (o no) las líneas de la cuadrícula.

- configurar la posición de los diferentes ejes. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_axes2.gif&type=thumb)

	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_axes1.gif&type=thumb)


- ***configurar el estilo del gráfico***: ver las líneas suavizadas y los colores de Gradient.

- activar el zoom (ver [párrafo siguiente](#NOTE4_1)).

- activar animaciones. 

- activar el retículo.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_Mire.gif&type=thumb)
 Para más detalles, véase [Chart crosshair](../WDChamp/3042078.md). 

- guardar el gráfico en formato Image. Se guarda el gráfico que se muestra actualmente (con todas sus características).

- imprimir el gráfico en la impresora seleccionada. El gráfico se imprimirá en Color en una impresora Color.




**Observación**: Usted tiene la posibilidad de configurar este menú:

- se pueden añadir opciones adicionales. Para obtener más información, consulte [Configurar el menú contextual que muestran los AAF](../Editeurs/2010040.md).

- [grMenuOpción](../WDLang3/3042042.md) se utiliza para cambiar el título de una opción o para evitar que se muestre una opción.

## Ampliar un gráfico
<a name="ampliar_grafico_ELTTEXTE000249"></a>


### Activar el zoom
<a name="activar_zoom_ELTPARAGRAPHE000098"></a>

El menú contextual del gráfico se utiliza para activar el zoom en el gráfico. Usted tiene la capacidad de habilitar:

- sólo el zoom horizontal.

- sólo el zoom vertical.

- el zoom vertical y horizontal.




Este zoom también puede ser activado por Default cuando se muestra el gráfico.
<a name="NOTE3_2"></a>


### Manejo del zoom
<a name="manejo_del_zoom_ELTPARAGRAPHE000111"></a>Cuando el zoom está activado, el gráfico se puede manejar mediante uno de los siguientes métodos:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_zoom.gif&type=thumb)


- Uso de barras de desplazamiento verticales y horizontales: tiene la posibilidad de especificar el área a ampliar moviendo los diferentes cuadros de la barra de desplazamiento.

- Pulsando Ctrl + la rueda del ratón: el área señalada por el ratón se amplía automáticamente. 
	La rueda del zoom también funciona cuando la control Gráfico está suspendida en el aire (incluso si la control no tiene foco).

- Usando el lazo del ratón: el área seleccionada por el lazo del ratón se amplía automáticamente. Para detener el zoom, pulse la tecla Esc.

- Pulsando Ctrl + (signo'+' en el teclado numérico) y Ctrl - (signo'-' en el teclado numérico): el área que se encuentra en el centro del gráfico se amplía automáticamente.




Para restaurar el gráfico a su estado inicial:

- Seleccionando "Zoom ... Zoom inicial" en el menú contextual del gráfico.

- Presionando Ctrl + 0.




**Observación**: Si el gráfico es associated con un fondo Image, el fondo Image no se amplía.




## Comparación de los controles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
En algunos casos, puede ser useful para comparar dos controles encontrados en la misma ventana o en dos ventanas diferentes.

Por ejemplo, si se utiliza un control modelo, se puede comparar las sobrecargas realizadas en las dos plantillas. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000126"></a>


### Comparación de dos controles que se encuentran en la misma ventana
<a name="comparacion_dos_controles_que_encuentran_misma_ventana_ELTPARAGRAPHE000020"></a>

Para comparar dos controles que se encuentran en la misma ventana: 

1. Seleccione dos controles para comparar. 

2. Seleccione "Comparar los dos elementos" en el menú contextual de la selección (clic derecho del ratón). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Comparer_Champs%20-%20HC%20N%B0001.gif)


3. Aparece la ventana de resultados. ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Comparer_Champs%20-%20HC%20N%B0002%201.gif) Para ver las diferencias, haga clic en <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Comparer_Champs%20-%20HC%20N%B0002.gif&type=thumb)


4. Se muestran los detalles de los diferentes elementos: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Comparer_Champs%20-%20HC%20N%B0003.gif&type=thumb)

### Comparación de dos controles que se encuentran en ventanas diferentes
<a name="comparacion_dos_controles_que_encuentran_ventanas_diferentes_ELTPARAGRAPHE000037"></a>

Para comparar dos controles que se encuentran en diferentes ventanas: 

1. Seleccione el primer control para comparar. 

2. En el menú contextual (clic derecho del ratón), seleccione "Comparar con". 

3. Se inicia un comparison asistente que permite seleccionar la ventana que contiene el control a comparar. Esta ventana puede ser: 

	- En el proyecto actual. Todo lo que tiene que hacer es seleccionar la ventana solicitada de la lista. 

	- En otro proyecto. Todo lo que tienes que hacer es seleccionar el archivo físico ruta de la ventana. 




4. La asistente propone la lista de controles que se encuentran en la ventana. Sólo hay que seleccionar el que se utilizará para el comparison. 

5. Especifique las opciones de comparison: 

	- Ignorar líneas en blanco: Si esta opción está seleccionada, las diferencias de líneas vacías en el código serán ignoradas.. 

	- Ignorar las diferencias de los comentarios: Si esta opción está seleccionada, las diferencias de comentarios en el código serán ignoradas.

	- Comparar subelementos: Si se selecciona esta opción, se comparan los subelementos (para un control Tabla por ejemplo, se comparan las columnas).. 




6. Aparece la ventana de resultados. Para ver las diferencias, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Comparer_Champs%20-%20HC%20N%B0002%201.gif)

7. Se muestran los detalles de los diferentes elementos.

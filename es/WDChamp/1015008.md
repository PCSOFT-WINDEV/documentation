
## Impresión del contenido de una tabla TreeView (AAF)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
El TreeView controles Tabla incluye un menú contextual automatic que permite a los usuarios realizar múltiples acciones sobre los datos mostrados en el control. 

Una de las opciones de este menú contextual permite imprimir el contenido de la control Tabla TreeView (opción "Imprimir").

## Modo de funcionamiento
<a name="modo_funcionamiento_ELTTEXTE000141"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000020"></a>

Pueden ocurrir varios casos:

- Si el desarrollador ha creado una reporte basada en esta control Tabla TreeView, se imprime la reporte.

- Si ningún reporte es relacionado para este control Tabla TreeView , el usuario puede imprimir directamente el contenido del control .




**Observaciones**: 

- La opción "Imprimir" del menú contextual de TreeView controles Tabla requiere las bibliotecas de impresión WINDEV framework.

- En el código, [AAFDisable](../WDLang1/1000022018.md) con la constante *aafPrintReportOnTable*, elimina la opción "Imprimir" para el control Tabla TreeView especificado.





### Impresión de la reporte creada por el desarrollador
<a name="impresion_reporte_creada_por_desarrollador_ELTPARAGRAPHE000036"></a>

Si el desarrollador ha creado una reporte basada en la control Tabla TreeView, la opción "Imprimir" imprimirá directamente la reporte. No se requiere programación específica. 


### Imprimir directamente la control Tabla TreeView
<a name="imprimir_directamente_control_tabla_treeview_ELTPARAGRAPHE000041"></a>

Cuando se imprime el control Tabla TreeView, el contenido del control se muestra directamente en el visor del reporte.

Si la control Tabla TreeView tiene demasiadas filas (más de 10.000), la aplicación pide confirmación al usuario (el trabajo de impresión puede durar bastante y consumir mucha memoria).

Las características de impresión son las siguientes:

- El título de la Document es el título de la control Tabla TreeView. Si no se ha especificado ningún título para la control TreeView, se imprime el texto "Reporte en tabla (AAF)".

- Los títulos de las columnas se imprimen en cada página.

- El fondo Color de las celdas será el mismo que el definido en el control Tabla TreeView.

- El trabajo de impresión no refleja los atributos RTF del texto en el control Tabla TreeView.  

- Los contadores, los totales, ... definidos para la control Tabla TreeView no se imprimen. Sólo se imprime un contador fila debajo del control Tabla TreeView. La leyenda corresponde a la leyenda definida para el contador en el editor de ventanas.

- La fecha de hoy está impresa en la parte superior derecha Corner.

- El número de la página actual y el número total de páginas se imprimen en la parte inferior derecha Corner.

- La fusión horizontal de las columnas se mantiene, excepto en los siguientes casos:

	- se fusionan con una columna de Invisible

	- fusión de columnas que se "superponen" en 2 páginas.




- Las imágenes del TreeView controles Tabla se imprimen sólo si se especifica su ruta. Las imágenes de los memorandos no se imprimen.

- El documento se imprime utilizando la dirección de escritura especificada con la propiedad [TextDirection](../Proprietes/2513020.md) a nivel de ventana y/o control Tabla TreeView.

- Las columnas definidas como no exportables y no imprimibles no se imprimirán (propiedad [VisibleInExportAndPrint](../Proprietes/1000022320.md) u opción "Exportar e imprimir" en la pestaña "General" de la ventana de la columna description).

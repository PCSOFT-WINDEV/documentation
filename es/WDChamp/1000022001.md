## Vista general de la Control Popup
<a name="vista_general_control_popup_ELTTEXTE000244"></a>
El Control Popup se utiliza para asociar una ventana emergente a un Page. Esta ventana emergente se mostrará desde la Page en la que se inserte. Esta ventana emergente se utiliza para establecer fácilmente un diálogo con el usuario. 

Hasta la versión 17, este tipo de diálogo se establecía a través de un [Control Celda](../WDChamp/1013096.md). 

El uso de una Control Popup permite manipular la ventana de diálogo en el editor como un elemento diferente de la Page, mientras que se incluye en la Page. A diferencia del uso de celdas, el uso de ventanas emergentes no afecta a la disposición de Page en el editor.

## Creación de una Control Popup
<a name="creacion_una_control_popup_ELTTEXTE000268"></a>


### Crear una Control Popup
<a name="crear_una_control_popup_ELTPARAGRAPHE000025"></a>

Puede crear Page s emergentes desde la barra Page: 

1. En la barra de páginas, haga clic en "Páginas Popup". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_Popup_creation%20-%20HC%20N%B0001.gif)


2. En el menú que aparece, haga clic en "Nueva popup". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_Popup_creation%20-%20HC%20N%B0002.gif)


3. La página popup aparece en el editor. 


Para visualizar las características del Control Popup, seleccione "description" en el menú contextual del control.

**Observaciones**: 

- Una Page puede contener una o varias ventanas emergentes. Para mostrar la ventana emergente deseada en tiempo de ejecución, basta con utilizar [PopupDisplay](../WDLang2/1000019788.md). 

- Una Page modelo puede contener una o varias ventanas emergentes. Las páginas que utilicen esta modelo podrán utilizar estos popups. 

- Un control modelo puede utilizar uno o varios popups (un selector color por ejemplo).

- El menú que se encuentra en una Page puede utilizar las opciones Popup. Para obtener más información, consulte [Configurar un menú contextual](../WDChamp/1010042.md). 

- Una Page puede mostrarse como una ventana emergente utilizando [PopupDisplayPage](../WDLang2/1410087258.md).. 





### Otros modos de creación
<a name="otros_modos_creacion_ELTPARAGRAPHE000059"></a>

- **Crear una ventana emergente a partir de una celda existente**: 
	Para crear una ventana emergente desde una celda existente: 

	1. Seleccione la celda deseada.... 

	2. En la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Celda/Página Popup".


 Observación: Si esta celda no se encuentra en una modelo y si es apilable, también se puede utilizar una opción del menú contextual para transformarla en emergente. 

- **Crear una ventana emergente a partir de un set de controles**: 

	1. Seleccione los controles deseados en el menú Page. 

	2. Realice una de las siguientes operaciones: 

		- seleccione "Refactorización ... Crear una ventana emergente Page a partir de la selección" en el menú contextual.

		- en la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Crear una página Popup a partir de la selección".

## Características de la Control Popup
<a name="caracteristicas_control_popup_ELTTEXTE000298"></a>


### Ventana de descripción
<a name="ventana_descripcion_ELTPARAGRAPHE000144"></a>

La ventana de la description de la Control Popup le permite definir: 

- los diferentes procedimientos locales (servidor y navegador) associated con el popup (pestaña "General"). 

- la capacidad de mover el popup con el ratón (pestaña "UI"). 

- la tasa de OVI (Ventana Desactivada Atenuada) associated con el control (60% por Default)

- el estilo del popup (pestaña"Estilo"). Tienes la capacidad de seleccionar una imagen de fondo específica para el popup o definir un Border.

## Manejo de una Control Popup en el editor
<a name="manejo_una_control_popup_editor_ELTTEXTE000322"></a>


### Ventana para la gestión de ventanas emergentes
<a name="ventana_para_gestion_ventanas_emergentes_ELTPARAGRAPHE000158"></a>

Se pueden identificar las diferentes ventanas emergentes associated con una Page: 

- en el panel "Explorador de proyectos", con los detalles de Page. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Popup_Description.gif)


- en la pestaña "Página", en el grupo "Edición", haga clic en "Páginas Popup".

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Popup_FenDescription.gif)
Esta última opción se utiliza para: 

- Cree una nueva ventana emergente.

- Eliminar una ventana emergente existente. 

- Abrir la ventana description de una ventana emergente.

- Abra una ventana emergente en el editor.

- Renombrar una ventana emergente. 



<a name="NOTE4_2"></a>


### Manejo de las ventanas emergentes associated con un Page
<a name="manejo_las_ventanas_emergentes_associated_con_page_ELTPARAGRAPHE000190"></a>

En el editor, se puede acceder a los popups associated con el Page a través del botón "Popup Page s" de la barra Page. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_Popup_creation%20-%20HC%20N%B0001.gif)
 Al hacer clic en este botón se abre una vista previa de las ventanas emergentes associated con el Page. Puede abrir la ventana emergente deseada en el editor haciendo clic en la vista previa correspondiente. Aparece una nueva pestaña para cada ventana emergente abierta en el editor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=popup_barre_doc.gif)

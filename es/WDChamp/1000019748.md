
## Manejo de los controles de calificación mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000161"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten programar los controles de manipular Rating. Para hacerlo, usa la Variable de la control Valoración en el código.

La Variable del control Valoración:

- corresponde al nombre de la control Valoración.

- se inicializa con la clasificación mostrada por el control Valoración.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperando la clasificación mostrada en la control
<a name="recuperando_clasificacion_mostrada_control_ELTTEXTE000185"></a>
Para recuperar la clasificación, utilice la siguiente sintaxis:


```txt
<Rating> = <Rating control>
```


&lt;La clasificación es un número entero o una variable real, según la granularidad definida para las clasificaciones en la ventana de description del control.. 

**Observaciones**: 

- También puede utilizar la [Valor](../Proprietes/2510130.md) Property. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El &lt;Rating&gt; tiene en cuenta el valor especificado por estrella (en el control description o con el [ValuePerStar](../Proprietes/1000021996.md) Property). 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificando la clasificación mostrada en la control
<a name="modificando_clasificacion_mostrada_control_ELTTEXTE000209"></a>
Para modificar la clasificación, utilice la siguiente sintaxis:


```txt
<Rating control> = <Rating>
```


&lt;Calificación&gt; es un valor entre la calificación mínima y máxima de la control Valoración. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observaciones**: 

- También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Atención: si se especifica un valor por estrella (en el control description o con el [ValuePerStar](../Proprietes/1000021996.md) Property), el &lt;Rating&gt; debe tener en cuenta este valor (valor de una estrella multiplicado por el número de estrellas mostradas).

## Propiedades específicas de la gestión de la calificación
<a name="propiedades_especificas_gestion_calificacion_ELTTEXTE000233"></a>
Las siguientes propiedades se pueden utilizar para programar los controles de calificación de manipular.


|   |   |
| --- | --- |
| [MaxValue](../Proprietes/2510009.md) | Permite conocer y modificar el valor máximo de una control Valoración. |
| [MinValue](../Proprietes/2510008.md) | Permite conocer y modificar el valor mínimo de un control Valoración. Esta Property es particularmente useful para los controles de calificación con una función "Me gusta/No me gusta".<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notation_Jaime.gif)<br> |
| [ValuePerStar](../Proprietes/1000021996.md) | Permite conocer y modificar el valor de una estrella en una control Valoración. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Valoración, consulte [Propiedades associated con controles de Rating](../WDChamp/1000019744.md).

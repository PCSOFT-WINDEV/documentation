
## Impresión de varias páginas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
Una reporte puede ser muy ancha, con un ancho que excede el de una hoja de papel. Este es el caso de los informes de la Tabla, el Planificador o el Gráfico de Gantt, ...

La impresión multipágina se utiliza para imprimir una reporte en varias páginas (2 o más) que pueden colocarse una al lado de la otra. 

Esta función le permite leer tablas o calendarios de gran tamaño. 

En el editor, el reporte muestra las diferentes páginas una al lado de la otra: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_Multipage_Editeur.gif&type=thumb)


En tiempo de ejecución, todas las páginas se muestran en el visor de reporte y se pueden imprimir: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_Multipage_Go.gif&type=thumb)

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000150"></a>
Para imprimir un reporte en modo multipágina, puedes: 

- seleccione "Imprimir en X páginas" al crear la reporte. 

- configurar el formato de impresión en las características de reporte. 



<a name="NOTE2_2"></a>


### Creando un reporte de varias páginas
<a name="creando_reporte_varias_paginas_ELTPARAGRAPHE000034"></a>

Al final de la asistente para la creación de la reporte, si la reporte es demasiado grande para caber en el formato especificado, la asistente propone las siguientes opciones: 

- Imprimir la reporte en varias páginas. En este caso, se crea automáticamente un reporte de varias páginas. 

- Utilizar el modo horizontal. 

- Reducir el tamaño de reporte. En este caso, según el porcentaje de reducción, el reporte impreso puede volverse ilegible. 



<a name="NOTE2_3"></a>


### Modificar una reporte existente
<a name="modificar_una_reporte_existente_ELTPARAGRAPHE000045"></a>

Un reporte existente se puede convertir en un reporte de varias páginas. 

Para cambiar el formato de reporte: 

1. Abra la descripción del reporte (seleccione "Descripción del reporte" en el menú contextual). 

2. Seleccione la pestaña "Formato".. 

3. En la zona "Multipágina en ancho", especifique el número de páginas que se utilizarán en ancho: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_Multipage_Format.gif&type=thumb)


4. Validar. 

5. La reporte se muestra en el editor con el número de páginas especificado. Tienes la capacidad de mover y ampliar los diferentes controles que se encuentran en tu reporte. 

6. Configurar la gestión de los bloques (especialmente los bloques "Cabecera de página" y "Pie de página") para el modo multipágina: 

	- Abra la descripción del reporte (seleccione "Descripción del reporte" en el menú contextual). 

	- En la pestaña "Bloques", defina las opciones: los bloques pueden ser: 

		- expandido en ancho en todas las páginas.

		- repetido en cada página.  




- Validar. 


Observación: Para una mesa reporte, el bloque de cabecera es necesariamente estirado.

<a name="NOTE3_1"></a>


## Imprimiendo la multipágina reporte
<a name="imprimiendo_multipagina_reporte_ELTTEXTE000186"></a>
Cuando se imprime un reporte de varias páginas, las diferentes páginas se imprimen automáticamente. Esta opción se tiene en cuenta: 

- en el visor de reporte,

- durante una impresión en la impresora,

- durante una impresión en formato PDF. 


Las páginas se imprimen de izquierda a derecha y luego hacia abajo..

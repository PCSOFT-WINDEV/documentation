


## Controles de looper poblados programáticamente: Operaciones en WLanguage (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000101"></a>
A continuación se muestra un resumen de cómo poblar los controles de manipular Looper mediante programación en WLanguage. Hay múltiples funciones de WLanguage para manipular este tipo de control.

No dude en consultar la documentación sobre estas funciones. Para obtener más información, consulte [Funciones de gestión de control Looper](../WDLang2/1000023945.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Inicialización de una población control Looper mediante programación
<a name="inicializacion_una_poblacion_control_looper_mediante_programacion_ELTTEXTE000132"></a>
Las filas no se crean automáticamente en un control Looper. Esto se debe especificar llamando:

- [.Add](../WDLang2/1000023469.md) o [&lt;AddLine](../WDLang2/1000023475.md).

- [&lt;Insertar](../WDLang2/1000023509.md) o [&lt;InsertLine](../WDLang2/1000023511.md).



<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Añadir filas en un control Looper rellenado programáticamente
<a name="anadir_filas_control_looper_rellenado_programaticamente_ELTTEXTE000180"></a>
Se pueden añadir filas en una control Looper poblada de forma programática:

- con [&lt;AddLine](../WDLang2/1000023475.md) y [&lt;InsertLine](../WDLang2/1000023511.md): en cada fila, el valor de cada atributo o control está separado por comas.

- con [.Add](../WDLang2/1000023469.md) y [&lt;Insertar](../WDLang2/1000023509.md): en un fila, cada valor de atributo o valor de control está separado por TAB.


<a name="NOTE3_2"></a>


### Observación
<a name="observacion_ELTPARAGRAPHE000235"></a>

Para facilitar la lectura del código, puede utilizar el siguiente método:

1. Añade una fila vacía.

2. Asigne cada atributo o control uno por uno.


Un error ocurre automáticamente cuando se borra un atributo o un control. Sin embargo, el tiempo de visualización puede ser mayor.
<a name="NOTE3_3"></a>


### Selección de una fila
<a name="seleccion_una_fila_ELTPARAGRAPHE000250"></a>

No se puede seleccionar una fila.

## Modificar los atributos o controles de un fila
<a name="modificar_los_atributos_controles_fila_ELTTEXTE000255"></a>


### Modificar todos los atributos o controles de una fila
<a name="modificar_todos_los_atributos_controles_una_fila_ELTPARAGRAPHE000278"></a>

Para editar los atributos o controles de un fila específico, utilice [&lt;Modificar](../WDLang2/1000023512.md) o [&lt;ModifyLine](../WDLang2/1000023520.md) y especifique el índice del fila que desea modificar.

Por ejemplo:

```wl
// Modify the row #3
LOOP_Looper1.Modify("Highlander" + TAB + "103min" + TAB + "Unknown" + ...
		TAB + "C. Lambert" + TAB + "" + TAB + "RO120", 3)
```


Para editar los atributos o controles de la fila actual, no es necesario especificar el índice.
<a name="NOTE4_2"></a>


### Modificar un atributo o un control
<a name="modificar_atributo_control_ELTPARAGRAPHE000308"></a>

Para modificar un atributo o un control, especifica el nombre del atributo o control a modificar.

- Modificar el atributo o control de un fila específico: Debe especificarse el índice fila.

- Modificar un atributo o control en el actual fila: No es necesario especificar el índice.

## Borrar una fila
<a name="borrar_una_fila_ELTTEXTE000316"></a>
La supresión de una fila debe ser solicitada explícitamente por [&lt;Borrar](../WDLang2/1000023701.md). Se utiliza la siguiente sintaxis:

```txt
<Looper control>.Delete([Index])
```


Si se especifica el índice, [&lt;Borrar](../WDLang2/1000023701.md) borra el fila que corresponde a ese índice. De lo contrario, se borra el fila actual.. Por ejemplo:

```wl
LOOP_CUSTOMER.Delete()
```
 Borrar un fila en un control Looper también borra todos los valores de los atributos o controles de ese fila.


## AAF: Exportar datos (Tabla, Tabla TreeView, Looper, TreeView...)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000158"></a>
Quieres realizar simulaciones con Excel, Word, LibreOffice, ...? Nada es más fácil. Simplemente "exportar" el contenido de control al software deseado. El archivo correspondiente se crea automáticamente a partir del contenido del control exportado y puede ser utilizado inmediatamente por otro software. 

Los controles que proponen esta opción son los siguientes: 

- Tabla y tabla TreeView. 

- TreeView. 

- Looper.

- Pivote tabla (sólo para Excel). 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Realización de una exportación
<a name="realizacion_una_exportacion_ELTTEXTE000182"></a>
Para realizar una exportación a Excel, Word, ....:

1. Abra el menú contextual de tabla y seleccione "Exportar tabla a...". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Exportez%20la%20table%20vers.jpg)

	Observación: Si la opción va seguida de un candado o una estrella, se solicita una contraseña antes de realizar la exportación.

2. Especifique el nombre del archivo de Excel que desea crear, así como su ubicación. 

3. La aplicación propone abrir directamente el archivo creado. 




Observaciones : 

- Si el archivo ya existe, puede sobrescribirlo o fusionarlo con el archivo que se está creando. La operación de fusión le permite mantener todas las modificaciones realizadas (por ejemplo, para un fichero Excel, formato de celdas, fórmulas,...). 

- Las columnas "numéricas de alta precisión" (con 38 dígitos significativos) se exportan a Excel como números exponenciales.

## Programación
<a name="programacion_ELTTEXTE000206"></a>


### Nombre del fichero de exportación
<a name="nombre_del_fichero_exportacion_ELTPARAGRAPHE000044"></a>

Por Default, el nombre de archivo sugerido corresponde a la leyenda del control Tabla. Este nombre puede modificarse mediante la programación de [Pie de foto](../Proprietes/2510053.md) Property. 

Por ejemplo: 


```wl
// In order for the file proposed to export to be "ExportStat2016"
TABLE_MyTable.Caption = "ExportStat2016"
```



### Desactivar la opción de exportación
<a name="desactivar_opcion_exportacion_ELTPARAGRAPHE000056"></a>

Para desactivar la opción de exportación en una Tabla, Tabla TreeView, TreeView o control Looper, simplemente utilice [AAFDisable](../WDLang1/1000022018.md): 

- con la constante *aafTableToExcel* para desactivar el menú de exportación a Excel.

- con la constante *aafTableToWord* para desactivar el menú de exportación a Word. 

- con la constante *aafTableToXML* para desactivar el menú de exportación a XML. 




Las opciones de los menús correspondientes ya no se mostrarán.. 


### Ejecución de una exportación
<a name="ejecucion_una_exportacion_ELTPARAGRAPHE000070"></a>

Para exportar los datos de una Tabla, Tabla TreeView, TreeView o control Looper mediante programación, basta con utilizar [AAFExecute](../WDLang1/1000022099.md): 

- con la constante *aafTableToExcel* para exportar el contenido del control a Excel.

- con la constante *aafTableToWord* para exportar el contenido del control a Word. 

- con la constante *aafTableToXML* para exportar el contenido del control a XML. 





### Modificar el título de la opción
<a name="modificar_titulo_opcion_ELTPARAGRAPHE000082"></a>

El título de la opción de menú que permite al usuario exportar el contenido de control es "Exportar a Excel", "Exportar a Word", "Exportar a XML"., ... Para modificar esta leyenda, utilice [AAFChangeCaption](../WDLang1/1000022100.md): 

- con la constante *aafTableToExcel* para modificar la opción de exportación a Excel.

- con la constante *aafTableToWord* para modificar la opción de exportación a Word. 

- con la constante *aafTableToXML* para modificar la opción de exportación a XML. 





### Asegurar las exportaciones
<a name="asegurar_las_exportaciones_ELTPARAGRAPHE000098"></a>

Los datos mostrados en los controles Table, TreeView Table, TreeView o Looper se pueden exportar.. Si se visualizan datos confidenciales en estos controles, puede: 

- prohibir la exportación. En este caso, no se propone la opción correspondiente. 

- solicitar una contraseña. En ese caso: 

	- en la opción de menú correspondiente aparece un candado (o una estrella ' '\*). 

	- aparece una ventana pidiendo una contraseña antes de realizar la acción. Si la contraseña no es válida, la acción no se realiza.







Para asegurar las exportaciones realizadas a través del menú contextual que se encuentra en los controles Table, TreeView Table, TreeView o Looper, utilice [AAFExportSecurity](../WDLang1/1000022481.md).

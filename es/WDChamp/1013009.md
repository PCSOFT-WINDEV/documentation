
## Llenado de una control Gráfico
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000179"></a>
Los datos que se mostrarán en la control Gráfico pueden provenir de:

- un artículo de un archivo de datos o una consulta.

- una columna de una control Tabla encontrada en la ventana actual o Page.
	

- a control List Box encontrada en la ventana actual o Page.
	

- un WLanguage array.
	

- datos especificados en la ventana description del control o mediante programación.

- un control del actual reporte (sólo para un reporte).




El modo de rellenar el gráfico se especifica en la asistente para la creación del control y en la ventana de descripción del control (["Pestaña"Series](../WDChamp/3042081.md)).

## Gráfico rellenado mediante programación
<a name="grafico_rellenado_mediante_programacion_ELTTEXTE000203"></a>
La programación necesaria para llenar el gráfico se realiza mediante las siguientes funciones: 



|   |   |
| --- | --- |
| [grAddData](../WDLang3/3042023.md) | Añade un dato a un gráfico. |
| [grHMAddData](../WDLang3/1000021628.md) | Agrega un dato a un gráfico de Heatmap. |
| [grIncreaseData](../WDLang3/3042030.md) | Agrega un valor a un dato en un gráfico. |
| [grScatterAddDataXY](../WDLang3/3042032.md) | Añade un dato a un gráfico "Scatter". |
| [grStockAddData](../WDLang3/3042009.md) | Añade datos en un gráfico de existencias (candelabros, gráficos de barras o minmax). |
| [grSunburstAddData](../WDLang3/1000021285.md) | Agrega un dato a una carta de Sunburst. |
| [grSurfaceAddData](../WDLang3/1000020425.md) | Agrega un dato a un gráfico de superficie. |





Por ejemplo:

- **control Gráfico en una ventana o Page**:
	El "CHART_MySalary" control Gráfico se utiliza para ver la evolución de un salario en los últimos cuatro años en un gráfico Line..
	Los controles de edición "EDT_Salario1", "EDT_Salario2", "EDT_Salario3" y "EDT_Salario4" se utilizan para escribir el salario del año correspondiente.
	En este ejemplo, escribe el siguiente código en la Event "Iniciando" de la control Gráfico:
	
	```wl
	// Add data into the chart
	grAddData(CHART_MySalary, 1, EDT_Salary1)
	grAddData(CHART_MySalary, 1, EDT_Salary2)
	grAddData(CHART_MySalary, 1, EDT_Salary3)
	grAddData(CHART_MySalary, 1, EDT_Salary4)
	```


- **control Gráfico en un reporte**: 
	El "RPT_MyReport" reporte imprime los detalles (apellido, nombre, volumen de negocios) de los 10 mejores clientes..
	El gráfico "CHART_MyChart" muestra el volumen de negocios de cada cliente en un gráfico circular.
	En este ejemplo, escriba el siguiente código en la Event "Antes de imprimir" del bloque Cuerpo:
	
	```wl
	// Add data into the chart
	grAddData(CHART_MyChart, 1, MySource.Turnover)
	```




<a name="NOTE3_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000085"></a>

- Si el modo de llenado es automatic, le aconsejamos que no programe el llenado del gráfico (con [grAddData](../WDLang3/3042023.md), [grScatterAddDataXY](../WDLang3/3042032.md), [grStockAddData](../WDLang3/3042009.md) y [grIncreaseData](../WDLang3/3042030.md)).

- El editor de ventanas, el editor Page o editor de reportes crean automáticamente el gráfico.. No es necesario utilizar [grCreate](../WDLang3/3042005.md) para crearlo.

- Se puede modificar el aspecto del gráfico y/o la visualización del texto en el gráfico:

	- en el gráfico description ("description" en el menú contextual de control). Para obtener más información, consulte [La control Gráfico: Características en el editor](../WDChamp/3042076.md).

	- a través de la [funciones para la gestión de gráficos](../WDLang3/3042003.md).

## Caso especial
<a name="caso_especial_ELTTEXTE000284"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### control Gráfico en una ventana o en una página basada en un control Tabla construido por BuildBrowsingTable
<a name="control_grafico_una_ventana_una_pagina_basada_control_tabla_construido_por_buildbrowsingtable_ELTPARAGRAPHE000122"></a>

Si está creando un nuevo gráfico cuya fuente es una control Tabla rellenada por [BuildBrowsingTable](../WDLang1/3074002.md) a partir de una consulta, debe especificar que las columnas corresponden a _COL1, _COL2, ...

También puede utilizar [grSourceSeries](../WDLang3/3042067.md) para modificar la fuente de la serie.

## Resumen de la control calculada
<a name="resumen_control_calculada_ELTTEXTE000128"></a>


### control de una reporte
<a name="control_una_reporte_ELTPARAGRAPHE000011"></a>

El control "Calculado" permite realizar cálculos en un reporte. Estos cálculos se pueden hacer:

- en un ítem de la fuente de datos.

- en un control encontrado en el reporte.

- mediante programación.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Crear un control calculado
<a name="crear_control_calculado_ELTTEXTE000152"></a>
Para crear un control "Calculado":

1. En la pestaña "Creación", en el grupo "Texto", haga clic en "Cálculo".

2. Haga clic en la ubicación en la que desea crear el control.

3. En la creación de control calculada asistente, especifique: 

	- si el calculado control cuenta los registros. 

	- si el control calculado realiza un cálculo sobre un artículo o control. En este caso, seleccione el tipo de cálculo (suma, promedio, ...) y la fuente de cálculo (nombre de control o nombre del artículo).

	- si la control calculada muestra un cálculo realizado mediante programación. En la lista, seleccione la máscara que desea utilizar. 







Esta información puede ser modificada en la ventana de description del control ("description" de el menú contextual).
<a name="NOTE2_2"></a>


### Importante: Diferencia entre el cálculo y el cálculo en la ruptura
<a name="importante_diferencia_entre_calculo_calculo_ruptura_ELTPARAGRAPHE000066"></a>

En los cálculos en pausa, el control se reinicia después de cada pausa.

## Notas
<a name="notas_ELTTEXTE000182"></a>


### Control "Calculado" específico: "Monetario + Euro"
<a name="control_calculado_especifico_monetario_euro_ELTPARAGRAPHE000075"></a>

Para un "Moneda + Euro" calculado control, tienes la habilidad de manejar:

- los ***moneda almacenada***: moneda de los datos encontrados en la fuente de datos de reporte. Si la control se maneja a través de la programación, la moneda utilizada para el valor será la moneda almacenada.

- los ***divisa visualizada***: moneda utilizada para visualizar el valor en el reporte. Esta moneda puede diferir de la moneda almacenada. La conversión se realiza automáticamente.

- los ***doble indicación***: se pueden vincular dos controles "Moneda + Euro".. El valor mostrado en la base control se convierte automáticamente en el rastreador control.

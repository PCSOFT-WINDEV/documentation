## Presentación
<a name="presentacion_ELTTEXTE000108"></a>


### Control de página dinámica o estática
<a name="control_pagina_dinamica_estatica_ELTPARAGRAPHE000014"></a>

SilverLight control se utiliza para visualizar una animación SilverLight 2 en un WEBDEV Page.

Para crear un SilverLight control: 

1. En la pestaña "Creación", en el grupo "Integración", despliegue "Plug-in" y haga clic en "Silverlight control".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.

3. En el control description, asocie la animación (archivo ".xap") al SilverLight control.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual. 


### Observación: Localización de la animación SilverLight 2
<a name="observacion_localizacion_animacion_silverlight_2_ELTPARAGRAPHE000027"></a>

La animación SilverLight 2 debe encontrarse en el directorio Image (directorio _WEB). Los archivos dependientes de la animación (imágenes, archivos de texto, ...) también deben copiarse en el directorio _WEB.

Si la animación utiliza archivos JavaScript (.js), estos archivos deben incluirse en la página (pestaña "Avanzado" de la página description ).

## Mostrando una página que contiene un SilverLight control
<a name="mostrando_una_pagina_que_contiene_silverlight_control_ELTTEXTE000138"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000042"></a>

Para mostrar una control de SilverLight, el framework de Microsoft debe estar instalado en el equipo. Si este framework no está instalado en el el equipo del usuario Web, una ventana le propone descargar e instalar "Microsoft Silverlight". Después de instalación, basta con actualización el Page para visualizarlo correctamente.
<a name="NOTE2_2"></a>


### Caso especial: Desarrollo de un ordenador o servidor Vista (o posterior) en IIS7 con la UAC habilitada: 
<a name="caso_especial_desarrollo_ordenador_servidor_vista_posterior_iis7_con_uac_habilitada_ELTPARAGRAPHE000049"></a>

- Si el equipo funciona con Windows Vista (o posterior), el instalación de "Microsoft Silverlight" debe realizarse como administrador.

- En algunos casos, es necesario realizar una modificación en IIS 7 para que el fichero ".xap" sea interpretado en el servidor o en los ordenadores de desarrollo (durante las pruebas). Se debe añadir el tipo MIME correspondiente.

	1. Abra el administrador de servicios de Internet de Microsoft (Opción "Iniciar .. Ejecutar .. Inetmgr.exe" desde el escritorio de Windows).

	2. Haga doble clic en "Tipo MIME".

	3. Seleccione "Añadir".

	4. Especifique:

		- la extension del expediente: ".xap".

		- la clase MIME: "application/x-silverlight-app"




5. Valide.

## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000121"></a>
WEBDEV gestiona los siguientes eventos de forma predeterminada (orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Acción (Navegador) | Código que puede ser llamado directamente desde el Componente Web. Se utiliza para ejecutar una acción a través del código HTML o JavaScript de la control. |
| Acción (Servidor) | Código que puede ser llamado directamente desde el Componente Web. Se utiliza para ejecutar una acción a través del código HTML o JavaScript de la control. |
| Asignación del valor Property (Navegador) | Código utilizado para asignar un valor al Componente Web control. |
| Cargar (onload) (Navegador) | Corre al cargar el componente. Por defecto, este Event contiene el siguiente Line de código: <br><br><pre><code>ExecuteProcess(<Control name>, trtValueAssignment)</code></pre><br>Esta línea de código se utiliza para ejecutar la "Asignación de la propiedad del valor" Event de la control. <br>Esto le permite dar un valor a la control por ejemplo. |
| Declaraciones globales (Servidor) | Declarando las variables globales relacionado al control. |
| Enviar (onsubmit) (Browser) | Corre: <br><br>- al validar la página (presentar). <br><br>- cuando se ejecuta la acción de servidor de una control. <br><br><br><br><br>Por defecto, este Event contiene el siguiente Line de código: <br><br><pre><code>ExecuteProcess(<Control name>, trtValueRetrieval)</code></pre><br>Esta línea de código se utiliza para ejecutar la Event "Recuperación de la propiedad del valor" de la control. <br>Esto le permite almacenar el valor de control por ejemplo. |
| Inicialización (Servidor) | Se ejecuta al abrir la página. |
| Obtener el valor Property (Navegador) | Código utilizado para definir el contenido del Valor Property del Componente Web control. |

## Eventos adicionales
<a name="eventos_adicionales_ELTTEXTE000145"></a>
Varios eventos adicionales se muestran en el editor de código: 

- CSS Event: "Declarando los estilos CSS": Código correspondiente al código CSS del componente web. Este código fue escrito cuando se creó el control. 

- JS Event: "Iniciando el componente Web": Código JS del componente web. Este código fue escrito cuando se creó el control.

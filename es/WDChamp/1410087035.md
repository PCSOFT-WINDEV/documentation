## Presentación
<a name="presentacion_ELTTEXTE000514"></a>
El editor de imágenes control propone varias características accesibles a través de: 

- La cinta de opciones. Hay dos paneles disponibles: 

	- El panel "Inicio" que se utiliza para realizar las operaciones más habituales en un Image. 

	- El panel "Otras herramientas" que se utiliza para realizar operaciones más complejas.. 




- El panel de edición, que se utiliza para realizar fácilmente varios ajustes. 




Observaciones: Las flechas que se encuentran en la parte superior de la cinta de opciones sirven para gestionar la función "Deshacer - Rehacer" del editor Image control.


### Caso especial: Archivo de solo lectura.
<a name="caso_especial_archivo_solo_lectura_ELTPARAGRAPHE000020"></a>

Cuando se abre un Image en modo sólo lectura, aparece una barra "Sólo lectura" en lugar de las flechas para la gestión de Deshacer/Rehacer. En este caso, sólo se pueden realizar algunas operaciones. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_LSeule%20-%20HC%20N%B0001.gif)

## Cinta: Pestaña "Inicio"
<a name="cinta_pestana_inicio_ELTTEXTE000544"></a>
El panel "Inicio" permite realizar las operaciones más habituales en un Image. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001.gif&type=thumb)


Esta pestaña contiene los siguientes grupos: 

- **General**: Se utiliza para abrir, crear, guardar o imprimir un Image. 

- **Portapapeles**: Se utiliza para manejar el portapapeles (copy/pegar). 

- **Tamaño**: Usado a manipular el tamaño de Image. 

- **Dibujo**: Se utiliza para aplicar las herramientas de dibujo a la actual Image o capa. 

- **Selección**: Se utiliza para realizar una selección en la capa actual, para moverla y cambiar su tamaño. 

- **Líneas**: Se utiliza para definir el grosor de la línea y su Color. 

- **Mostrar**: Se usa para hacer zoom y para mostrar u ocultar el panel. 







### Grupo "General"
<a name="grupo_general_ELTPARAGRAPHE000047"></a>

El grupo **General** se utiliza para abrir, crear, guardar o imprimir un Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%201.gif)
Es posible: 

- Crear una nueva imagen. 

- Abrir un Image existente. 

- Escanear un documento. Aparece la ventana de opciones de escaneo. En esta ventana, puede seleccionar:

	- la deseada Scanner.

	- la fuente (cama plana, alimentador, etc.).

	- el modo Color (Color, escala de grises, negro y White, etc.).

	- la resolución. Puede obtener una vista previa de la Document e iniciar la exploración mediante los botones "Vista previa" y "Exploración", respectivamente. El resultado de la exploración se muestra en el editor de Image. El resultado de la exploración sustituye a la Image mostrada anteriormente.   




- Guardar (o guardar como) un Image. 

- Imprimir un Image. La ventana de configuración de la impresión se abre (con una vista previa del documento). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_imprimer%20-%20HC%20N%B0001.gif&type=thumb)
Es posible configurar: 

	- las propiedades de la impresora: Nombre de la impresora, orientación, modo de impresión, número de copias. 

	- la posición Image en el Page impreso, así como los márgenes,

	- el tamaño de Image. 










### Grupo "Portapapeles"
<a name="grupo_portapapeles_ELTPARAGRAPHE000074"></a>

El grupo **Portapapeles** le permite: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%202.gif)


- gestionar las funciones del portapapeles (copy / pegar). 
	Observación: También puede utilizar los atajos estándar: Ctrl + C, Ctrl + V, Ctrl + X. 

- hacer una captura de pantalla: La opción "Recortar" permite seleccionar y capturar un área de la pantalla. Esta captura se añade a la Image actual en una nueva capa.







### "Grupo "Tamaño
<a name="grupo_tamano_ELTPARAGRAPHE000088"></a>

El grupo **Tamaño** permite gestionar principalmente el tamaño de la imagen. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%209.gif)
Es posible: 

- **Recortar el Image**: Esta opción cambia el cursor del ratón en una herramienta específica. Para usar esta herramienta, seleccione el área que desea mantener. El Image se recorta automáticamente. 

- **Cambiar** el tamaño del Image: En la ventana que se muestra, especifique el nuevo tamaño o el porcentaje de ampliación/reducción.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0001.gif)

	Observaciones: 

	- El redimensionamiento se realiza para la capa actual. Si selecciona la capa "Toda Image", el redimensionamiento se realizará para todas las capas. 

	- Un clic en la cadena se utiliza para realizar un redimensionamiento no proporcional.. 




- **Ampliar el espacio de trabajo**: En la ventana que se muestra, introduzca: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0002.gif)


	- El nuevo tamaño del espacio de trabajo. 

- El modo para ampliar la Image. 







### "Grupo "Dibujo
<a name="grupo_dibujo_ELTPARAGRAPHE000110"></a>

El grupo **Dibujo** se utiliza para aplicar las herramientas de dibujo al Image o capa actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%206.gif)


- **La herramienta "Lápiz"** se utiliza para dibujar a mano alzada en la capa actual. El lápiz Color y el grosor pueden seleccionarse en el grupo "Líneas".. Estas características también pueden modificarse en el programa [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Haga clic en la herramienta "Lápiz" de la cinta de opciones. 

	2. Haga clic en la posición donde el dibujo comenzará en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que hayas terminado de dibujar. 


El editor Image reconoce automáticamente los dibujos a mano alzada. Así, una Line, un círculo o un rectángulo dibujados a mano alzada serán reconocidos automáticamente y redibujados.

- **La herramienta " Line "** se usa para dibujar un Line en la capa actual. La línea Color y su grosor pueden seleccionarse en el grupo "Line s".. Estas características también pueden modificarse en el programa [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Haga clic en la herramienta "Line" de la cinta de opciones. 

	2. Haga clic en la posición donde la línea comenzará en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que hayas terminado de dibujar. 
			Observación: Si mantienes pulsada la tecla Shift, podrás get un Line perfectamente horizontal o vertical. 




- **La herramienta "Borrador"** se usa para borrar a mano alzada en la capa actual. El borrador borra el dibujo en la capa actual. El grosor de la goma de borrar se define en el grupo "Líneas" o en [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Haga clic en la herramienta "Borrador" de la cinta de opciones. 

	2. Haga clic en la posición donde comenzará el borrado en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta el final del borrado. 




- **La herramienta "Rellenar"** se utiliza para rellenar un área con un Color en la capa actual. El relleno color puede seleccionarse en el grupo "Líneas" o en el [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Seleccione el área que desea rellenar (con las herramientas rectángulo, lazo o varita mágica, por ejemplo, disponibles en el grupo "Selección").. 

	2. Haga clic en la herramienta "Relleno" de la cinta de opciones. 

	3. Haga clic en el área para rellenar. El relleno se realiza automáticamente. 




- **La herramienta "Elipse"** se usa para dibujar una elipse o un círculo en la capa actual. La línea y el relleno Color pueden seleccionarse en el grupo "Line s".. Estas características también pueden modificarse en el programa [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Haga clic en la herramienta "Elipse" de la cinta de opciones. 

	2. Haga clic en la posición donde el dibujo de la elipse comenzará en la Image. 
			Observación: La elipse se dibuja en un área. La posición inicial del dibujo corresponde a una Corner de la zona en la que se dibujará la elipse. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que hayas terminado de dibujar. 
			Observación: Si mantienes pulsada la tecla Shift, harás get un círculo perfecto.




- **La herramienta "Rectángulo"** se usa para dibujar un rectángulo o un cuadrado en la capa actual. La línea y el relleno Color pueden seleccionarse en el grupo "Line s".. Estas características también pueden modificarse en el programa [Panel del editor de Image](#NOTE4_1). Para usar esta herramienta: 

	1. Haga clic en la herramienta "Rectángulo" de la cinta de opciones. 

	2. Haga clic en la posición donde el rectángulo comenzará en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que hayas terminado de dibujar. 
			Observación: Si mantienes pulsada la tecla Shift, tendrás get un cuadrado perfecto.




- **La herramienta "Flecha"** le permite dibujar una flecha en la capa actual. La flecha color puede seleccionarse en el grupo "Líneas".. La [Panel del editor de Image](#NOTE4_1) se utiliza para configurar características adicionales como el tamaño de la flecha, la punta de la flecha, etc., ... Para usar esta herramienta: 

	1. Haga clic en la herramienta "Flecha" de la cinta de opciones. 

	2. Haga clic en la posición donde la flecha comenzará en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que hayas terminado de dibujar. 
			Observación: Si mantienes pulsada la tecla Shift, la flecha será horizontal o vertical.




- **La herramienta "Texto"** se utiliza para escribir un texto en la capa actual. Para usar esta herramienta: 

	1. Haga clic en la herramienta "Texto" de la cinta de opciones. 

	2. En la Image, cree el área donde el texto será posicionado. 

	3. Aparecerá el cuadro de texto, así como una pestaña "Texto", que le permitirá configurar el texto: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_Outil%20texte%20-%20HC%20N%B0001.gif)


		- Fuente utilizada y tamaño de la fuente,

- Características de los personajes: negrita, cursiva, subrayado, ...

- Fondo Color, texto Color,

- Diseño (centrado, alineado a la derecha, ...). 

4. Introduzca un texto. Para salir del área de texto, haga clic en cualquier lugar de la Image. 

- **La herramienta " Watermark "** se usa para crear una nueva capa de texto en el Image actual. Para usar esta herramienta: 

	1. Haga clic en "Marca de agua" en la cinta de opciones. 

	2. Haga clic en la imagen en la posición donde la Watermark debe aparecer. 

	3. Se muestra la ventana para ajustar las características de la Watermark. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_Filigrane%20-%20HC%20N%B0001.gif)
 En esta ventana, puede especificar: 

		- Fuente, tamaño, color y características (negrita, cursiva, subrayado). 

		- Color del texto.

		- Alineación del texto (horizontal y vertical), ángulo y opacidad.

		- Texto a mostrar. 

		- La opción "Vista previa de la imagen" permite previsualizar directamente la Watermark sin validar este cuadro de diálogo. 




4. Validar. La capa de texto que contiene la Watermark se crea. 

- **La herramienta "Desenfoque"** le permite desenfocar una parte del Image. Para usar esta herramienta: 

	1. Haga clic en la herramienta "Blur" de la cinta de opciones. 

	2. Seleccione la zona de la Image que desea difuminar. 

	3. La Blur se aplica automáticamente. 


Observación: Si la Blur se aplica varias veces en la misma zona, el efecto se intensificará. 





### "Grupo "Selección
<a name="grupo_seleccion_ELTPARAGRAPHE000230"></a>El grupo **Selección** se utiliza para realizar una selección en la capa actual, moverla y cambiar su tamaño. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%207.gif)


- Selección: Se utiliza para realizar una selección rectangular en la capa actual. Para usar esta herramienta: 

	1. Haga clic en la herramienta "Selección" de la cinta de opciones. 

	2. Haga clic en la posición donde el rectángulo de selección comenzará en la Image. 

	3. Mantenga el botón izquierdo del ratón pulsado hasta que haya seleccionado el área deseada.

	4. Entonces, la selección puede ser manipulada (mover, llenar, etc.). 




- Lazo: Se utiliza para realizar una selección a mano alzada en la capa actual. Para usar esta herramienta: 

	1. Haga clic en la herramienta "Lazo" de la cinta de opciones.. 

	2. Haga clic en la posición donde la selección comenzará en la Image. 

	3. Mantén el botón izquierdo del ratón pulsado hasta que llegues al punto en el que la selección terminará. Si este punto no corresponde al punto de partida, las dos extremidades del lazo son automáticamente relacionado.

	4. Entonces, la selección puede ser manipulada (mover, llenar, etc.). 




- Varita mágica: Se utiliza para realizar una selección según los esquemas existentes en la capa actual. Para usar esta herramienta: 

	1. Haga clic en la herramienta "Varita" de la cinta de opciones. 

	2. Haga clic en la posición donde se realizará la selección en la Image. Los píxeles correspondientes se seleccionan automáticamente. 
			Observación: La tolerancia puede ser especificada para la varita: todo lo que tienes que hacer es expandir el menú que se encuentra a la derecha de la opción Wand. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_Image_baguette%20-%20HC%20N%B0001.gif)


	3. Entonces, la selección puede ser manipulada (mover, llenar, etc.). 

- Mover y cambiar el tamaño (Ctrl + T): Esta opción se utiliza para set la selección actual con el fin de:

	- Muévete.: todo lo que tienes que hacer es mover la selección con el ratón,

	- cambiar el tamaño de la misma: todo lo que tienes que hacer es usar las manijas de tamaño alrededor de la selección. 
			Observación: Si se pulsa la tecla SHIFT durante el redimensionamiento, esta operación de redimensionamiento se realiza en modo homotético (se mantiene la relación altura/anchura).

	- realizar una rotación: basta con alejar el ratón de los tiradores del área para que aparezca un cursor de rotación (doble flecha curva). A continuación, mantenga el botón izquierdo del ratón pulsado cuando gire el objeto en la dirección deseada. Soltando el botón izquierdo del ratón se valida la rotación.

## Cinta: "Panel "Otras herramientas
<a name="cinta_panel_otras_herramientas_ELTTEXTE000610"></a>
El panel "Otras herramientas" permite realizar diversas operaciones en una capa Image o Image. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0002.gif)


Las opciones se agrupan en varios grupos: 

- **Capas**: Se utiliza para gestionar las capas. 

- **Rotación y simetría**: Utilizado para manejar transformaciones simples en la Image. 

- **OCR**: Permite utilizar el OCR sobre el texto presente en el Image. 

- **Efectos y ajustes**: Utilizado para manipular el contenido de Image. 

- **Transformación**: Usado para transformar el Image. 





### "Grupo "Capas
<a name="grupo_capas_ELTPARAGRAPHE000325"></a>

El grupo "Capas" permite gestionar las capas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0004%201.gif)
Es posible: 

- Crear una nueva capa, ya sea en blanco o a partir de un Image.

- Duplicar una capa existente. 

- Aplanar las capas de Image. Estas operaciones también pueden realizarse a través del panel del editor de Image. Para obtener más información, consulte [Características del panel](#NOTE4_1).







### "Grupo "Rotación y simetría
<a name="grupo_rotacion_simetria_ELTPARAGRAPHE000340"></a>

El grupo **Rotación y simetría** le permite administrar transformaciones simples en el Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0002%201.gif)
 Las diferentes opciones son: 

- **Rotación**: Se utiliza para realizar una rotación de la capa actual (rotación preestablecida o personalizada).. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0003.gif)


- **Simetría**: Se utiliza para realizar una simetría de la capa de corriente (horizontal y/o vertical). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0004.gif)








### "Grupo "OCR
<a name="grupo_ocr_ELTPARAGRAPHE000355"></a>

La opción "OCR" analiza el texto contenido en una parte de la Image visualizada en el editor Image. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0002%204.gif)


Para utilizar esta opción: 

1. Haga clic en "OCR".. 

2. Arrastre el cursor sobre la parte de la Image que contiene el texto a analizar. 

3. El texto se interpreta y se copia en el portapapeles. Un mensaje permite al usuario saber que el texto está en el portapapeles. Luego se puede pegar en cualquier aplicación. 




**Administrar idiomas de OCR**: Por Default, el OCR es compatible con el inglés, el español y el francés. El idioma utilizado depende del parámetro pasado a [Nación](../WDLang1/3054014.md). 
Para añadir más idiomas: 

1. Si es necesario, descargue el modelo para el idioma deseado (lea la documentación de [OCR nativo](../WDLang6/1000025951.md)). 

2. Utilice la función [OCRLoadLanguage](../WDLang6/1000025790.md) para cargar el idioma deseado. 

3. Cambia de idioma con [Nación](../WDLang1/3054014.md). 







### "Grupo "Efectos y ajustes
<a name="grupo_efectos_ajustes_ELTPARAGRAPHE000389"></a>

El grupo **Efectos y configuraciones** se usa principalmente para administrar el contenido Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0002%202.gif)
 Las opciones disponibles son: 

- **Ligereza de contraste**: Esta opción se utiliza para definir el contraste y la luminosidad de Image a través de una ventana específica. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0005.gif)
 El contraste se incluye entre -100 y +100. La ligereza se incluye entre -255 y +255. Estos ajustes pueden visualizarse en tiempo real en el Image (opción "Previsualización Image").

- **Color transparente**: Esta opción se usa para seleccionar el color que se convertirá en Transparent en el Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0008.gif)
 Estos ajustes pueden visualizarse en tiempo real en Image (opción "Previsualización Image"). La pipeta sirve para seleccionar directamente una color en su Image. 

- **Ajustes HSL**: Esta opción se utiliza para modificar las opciones de Tono, Saturación y Luminosidad de Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0009.gif)
 La tonalidad está comprendida entre -180 y +180. La saturación se incluye entre -100 y +100. La ligereza se incluye entre -255 y +255. Estos ajustes pueden visualizarse en tiempo real en Image (opción "Previsualización Image"). 

- **Colores transformados**: Esta opción permite sustituir una color por otra en la Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0010.gif&type=thumb)
 Estos ajustes pueden visualizarse en tiempo real en Image (opción "Previsualización Image").

- **Sombra**: Se utiliza para aplicar un efecto de sombra en los elementos de Image en relación con el Image Background. Los efectos de sombra aumentan cada vez que se hace clic en la opción "Sombra".. La fuente de luz utilizada para generar la sombra proviene de la Corner superior izquierda.

- **aureola**: Se utiliza para definir una sombra alrededor de los elementos de Image (equivalente a una sombra en todos los lados). Puedes definir el grosor del halo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0011.gif)
 Estos ajustes pueden visualizarse en tiempo real en Image (opción "Previsualización Image").

- **Blanco y negro**: Se usa para convertir la imagen en Color en negro y White (escala de grises). 







### "Grupo "Transformación
<a name="grupo_transformacion_ELTPARAGRAPHE000415"></a>

El grupo **Transformación** se utiliza principalmente para administrar el tamaño Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0002%203.gif)
 Las opciones disponibles son: 

- **Icon**: Esta opción se utiliza para cambiar la actual Image en Icon. Los diferentes formatos de los iconos propuestos por Default se muestran a la izquierda del editor. Tiene la posibilidad de añadir formatos específicos (botón "+"), suprimir formatos inútiles, regenerar los iconos modificando el tamaño de la fuente Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0007.gif&type=thumb)


- **Multi DPI**: Se utiliza para crear una set de imágenes correspondientes a los diferentes DPIs . Los diferentes DPI propuestos por Default se muestran a la izquierda del editor. Tiene la posibilidad de añadir DPI específicos (botón "+"), suprimir DPI inútiles, regenerar los Image s modificando el tamaño de los Image de origen . La opción **"Exportar DPI"** que se encuentra en el panel "Inicio" de la cinta se utiliza para generar los archivos correspondientes. El nombre de estas imágenes está formateado de la siguiente manera: 
	
	```txt
	omImage@dpi99x.Extension
	```
Donde 99x corresponde al coeficiente multiplicador en relación con el editor DPI.
	El editor opera en 160 DPI. Por ejemplo, para un "Logo.png" Image, el set de Image s será:

	- "Logo@dpi1x.png" para los 160 DPI.

	- "Logo@dpi1_5x.png" para los 240 PPP. 

	- "Logo@dpi2x.png" para los 320 DPI.

	- "Logo@dpi3x.png" para los 480 PPP. 

	- "Logo@dpi4x.png" para los 640 PPP. 




- **Borde**: Esta opción se utiliza para añadir una Border alrededor del Image a través de una ventana específica. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_reglages%20-%20HC%20N%B0006.gif)
Es posible: 

	- usar un borde preestablecido: simple Black, redondo, ...

	- personalizar un borde preestablecido.

	- usar un borde personalizado.

## El panel del editor de Image
<a name="panel_del_editor_image_ELTTEXTE000664"></a>
El panel que se encuentra a la derecha del editor de Image se utiliza para: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_ruban%20-%20HC%20N%B0001%204.gif)


- Previsualización de la Image y la selección realizada en la Image. 

- Ampliar la Image mostrada en el editor mediante el cursor de tamaño. 

- Seleccione la color utilizada para las diferentes herramientas (relleno, pluma, Line, ...). La Color también puede ser seleccionada a través de la pipeta. 

- Seleccione el grosor de las diferentes herramientas (Line, bolígrafo, ...). 

- Gestionar y seleccionar las capas. 





### Capas
<a name="capas_ELTPARAGRAPHE000459"></a>

Las capas están apiladas una Above otra, cada una contiene algunos elementos de Image. Por ejemplo, puede crear una capa para mostrar un sello de "Promoción" en una Image. 

Las siguientes opciones están disponibles para las capas: 


- Hacer visible una capa o Invisible (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%201.gif)),

- Añadir una nueva capa (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%202.gif)),

- Eliminar la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%203.gif)),

- Duplicar la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%204.gif)),

- Mover la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%205.gif)),

- Aplanar la Image (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%206.gif)).


 Cada capa incluye un menú contextual que te permite manipular arla (duplicar, fusionar, mover, ...).

La opción "Visible/Invisible mediante programación" le permite hacer que la capa sea utilizable mediante programación: Esto le permite mostrar la imagen en un control Imagen mientras especifica la capa que se hará visible o Invisible.

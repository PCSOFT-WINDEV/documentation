## Características
<a name="caracteristicas_ELTTEXTE000151"></a>


### Abrir una Image en el editor
<a name="abrir_una_image_editor_ELTPARAGRAPHE000017"></a>

Para abrir un Image en el editor de Image control, puede: 

- soltar un archivo de imagen directamente en el control,

- dejar caer una Image de la página. 

- haga clic en el botón "+" de la cinta de opciones en la parte inferior del editor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0002.gif)
 El menú le permite: 

	- abrir una Image desde el equipo: simplemente especifica el archivo ruta en la ventana que aparece (o suelta el archivo deseado). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0003.gif)


	- abrir una Image desde una URL: simplemente especifique la URL correspondiente en la ventana que aparece. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0004.gif)





### Basándose en la Image
<a name="basandose_image_ELTPARAGRAPHE000033"></a>

El control del Editor de imágenes le permite dibujar libremente sobre la imagen mostrada, utilizando el lápiz, la Marker o la herramienta de resaltado. <br>![Añadir un dibujo](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0005.gif)
 Cuando se selecciona una de estas opciones, aparece una nueva barra de herramientas en la parte inferior del control, que permite configurar el Color y el grosor del Line. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0012.gif)
 También puede borrar la Line actual. 


### Insertar una forma en la Image
<a name="insertar_una_forma_image_ELTPARAGRAPHE000043"></a>

El editor de Image control le permite insertar: 

- una línea o una flecha en la pantalla de Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0006.gif)


- una forma (triángulo, rectángulo o círculo). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0007.gif)





Cuando se selecciona una de estas opciones, aparece una nueva barra de herramientas en la parte inferior del control, que permite configurar la Color y el grosor de la Line. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0012.gif)
 También puedes borrar la forma actual. 


### Inserción de texto en la Image
<a name="insercion_texto_image_ELTPARAGRAPHE000057"></a>

El control Image Editor permite insertar texto en un Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0008.gif)
 Cuando se selecciona una de estas opciones, el texto editable aparece en un área de texto dentro de la imagen, y aparece una nueva barra de herramientas en la parte inferior del control, que permite configurar la fuente, Color, etc., .... <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0010.gif)
 También puede borrar el texto actual. 


### Manipulación de la Image
<a name="manipulacion_image_ELTPARAGRAPHE000066"></a>

El control del editor de Image le permite manipular Image s. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0011.gif)
 Puede recortar, ampliar y girar el Image. La barra de herramientas de la parte inferior de control se adapta para mostrar las diferentes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0009.gif)


Para ampliar el Image, puede utilizar: 

- la rueda del ratón,

- los botones + y - de la barra de herramientas de la parte superior del editor. 




Puede guardar la Image en cualquier momento descargándola a través del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_images_WB%20-%20HC%20N%B0010%201.gif) de la barra de herramientas situada en la parte superior del editor.

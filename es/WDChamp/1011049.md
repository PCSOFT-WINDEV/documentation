<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000181"></a>
Las diferentes características de un bloque pueden ser vistas y/o modificadas en la ventana de description de este bloque.

Dependiendo del tipo de bloque, la ventana de description se utiliza para configurar:

- el nombre del bloque.

- el elemento de ruptura relacionado al bloque (sólo bloques de ruptura).

- los parámetros y las opciones de impresión del bloque. Para más detalles, consulte la página web de [Parámetros y opciones de impresión de un bloque](#NOTE3_1).

- la altura del bloque.

- el estilo de bloque (fondo Color, Border, ...).

## Descripción de los bloques de reporte
<a name="descripcion_los_bloques_reporte_ELTTEXTE000205"></a>
Para visualizar la ventana de description de un bloque, realice una de las siguientes operaciones:

- haga doble clic en el título del bloque.

- en la pestaña "Vista", en el grupo "Opciones", haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) o pulse F12. En la ventana que aparece, seleccione el bloque y haga clic en el botón "description".

- seleccionar "Bloque description" de el menú contextual del bloque (clic con el botón derecho del ratón en el título del bloque).




Por ejemplo, la ventana de description de un bloque de "Romper la cabecera": <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BlocDescription.gif&type=thumb)

## Parámetros y opciones de impresión de un bloque
<a name="parametros_opciones_impresion_bloque_ELTTEXTE000229"></a>
Los parámetros y las opciones de impresión de un bloque se definen en la pestaña "UI" de su description (doble clic en la leyenda del bloque).

Los parámetros y las opciones de impresión de los bloques reporte son los siguientes:

- **Visibilidad**: 

	- **Visible**:
			El bloque se puede visualizar o no. Si un bloque es Invisible, el bloque y sus controles no se mostrarán en el reporte.
			Esta característica puede modificarse de forma programada con el programa [Visible](../Proprietes/2510138.md) Property.

	- **Siempre visible**: 
			El bloque y sus controles se mostrarán en todas las páginas de la reporte.
			 Esta opción está disponible para los bloques Encabezado de página y Pie de página.. 

	- **Siempre invisible**: 
			El bloque y sus controles no se mostrarán en la pantalla de reporte.
			 Esta opción está disponible para los bloques Encabezado de página y Pie de página.. 

	- **Visible solo en la primera página**: 
			El bloque y sus controles se mostrarán en la primera página de la reporte.
			 Esta opción está disponible para los bloques Encabezado de página y Pie de página.. 

	- **Invisible solo en la primera página**: 
			El bloque y sus controles se mostrarán en todas las páginas de la reporte excepto en la primera..
			 Esta opción está disponible para los bloques Encabezado de página y Pie de página.. 




- **Capacidad de rotura**: 

	- **Bloque separable**: 
			Si un bloque se puede romper y no hay suficiente espacio en una página para imprimir el bloque completo, el contenido del bloque se imprime en varias páginas.. 
			Si un bloque es irrompible y no hay suficiente espacio en la página para imprimir el bloque completo, el contenido del bloque se imprime en la página siguiente.




- **Rellenar la página**: 

	- **Completar hasta el final de la página**:
			Se puede estirar un bloque hasta que llegue al final de la página..
			**Observación**: Esta opción es útil si el bloque tiene una imagen de fondo o un fondo Color.

	- **Alarga las columnas tabla hasta el final de la página** (para los bloques que contienen un tabla):
			Las columnas tabla se pueden ampliar hasta llegar al final de la página.
			**Observación**: Para extender las columnas de una tabla hasta el pie de página, también es posible agregar un bloque *Complemento de cuerpo*. El *Complemento del cuerpo* se estiran automáticamente hasta que llegan al final de la página.

	- **Repetir hasta el final de la página**:
			El bloque se puede copiar hasta el final de la página.
			Esta opción está disponible especialmente para el bloque de cuerpo. 

	- **Conservar el espacio del pie de página cuando sea invisible** (solo para los bloques Pie de página)**:
			Por defecto, el espacio ocupado por el bloque del pie de página se mantendrá siempre en las páginas de reporte (no importa si el bloque es Visible o no). Con esta opción, el espacio ocupado por el bloque del pie de página se mantendrá sólo en las páginas en las que este bloque es Visible. Esta opción se utiliza para optimizar el método de llenado de las páginas de reporte.

	- **Reiniciar el número de páginas después de imprimir el bloque**: En Default, el número de páginas en los controles de preajuste "Número de páginas" y "Número de páginas y número de páginas" no se restablece durante el trabajo de impresión. Esta opción permite restablecer automáticamente estos controles preestablecidos una vez que se imprime el bloque (durante una pausa o al final de la impresión). 
			Esta opción puede utilizarse cuando se imprime una set de facturas, por ejemplo, para numerar las páginas de cada factura de forma independiente.




- **Salto automático**:

	- **Salto de página después del bloque**:
			Se puede generar automáticamente un salto de página una vez que se imprime el bloque.

	- **Salto de columna después del bloque**: (sólo para el bloque de cuerpo de informes multicolumna)
			Se puede generar automáticamente un salto de columna una vez que se imprime el bloque.

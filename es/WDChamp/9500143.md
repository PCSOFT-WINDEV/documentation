
## AAF: Gráfico en una columna de tabla
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
Cuando una columna tabla es una columna Numérica, el usuario final puede crear un gráfico en los datos de la columna y por lo tanto beneficiarse del poder de los gráficos creados con WINDEV.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualización de un gráfico
<a name="visualizacion_grafico_ELTTEXTE000120"></a>
Para crear un gráfico a partir de los datos que se encuentran en una columna (tabla o TreeView Table): 

1. Abrir el menú contextual de la columna solicitada. 

2. Seleccione"Gráfico".

3. El gráfico creado a partir de los datos encontrados en la columna se muestra automáticamente en una nueva ventana. Las funciones del menú contextual del gráfico le permiten visualizar los datos según sus propias necesidades. Para obtener más información, consulte [Manejo de los controles del gráfico en tiempo de ejecución (AAF)](../WDChamp/1013007.md).




Observación: No puede seleccionar las columnas que se incluirán en el gráfico. Para elegir algunas columnas específicas de la control Tabla, le aconsejamos que cree un nuevo gráfico a partir de la control Tabla. Este gráfico se mostrará en una ventana, por ejemplo.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Programación
<a name="programacion_ELTTEXTE000144"></a>


### Desactivación de los gráficos
<a name="desactivacion_los_graficos_ELTPARAGRAPHE000032"></a>

Para desactivar la opción de crear gráficos, basta con utilizar [AAFDisable](../WDLang1/1000022018.md) con la constante *aafChart*.


### Visualización del gráfico
<a name="visualizacion_del_grafico_ELTPARAGRAPHE000040"></a>

Para visualizar el gráfico de una columna tabla mediante programación, basta con utilizar [AAFExecute](../WDLang1/1000022099.md) con la constante *aafChart*.

Ejemplo: 


```wl
AAFExecute(TABLE_Stat.COL_Result2016, aafChart)
```

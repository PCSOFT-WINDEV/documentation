## Características de un gráfico de área
<a name="caracteristicas_grafico_area_ELTTEXTE000215"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GenAire.gif&type=thumb)
En este cuadro:

**1.** **Señala con su valor.**
El valor del punto se visualiza si se cumplen las siguientes condiciones:

- la opción "Mostrar valores" está seleccionada en la pestaña "Detalles" de la descripción del control Gráfico.

- el tipo de punto se define para cada serie en la pestaña "Series" de la ventana del punto description. Si el tipo de punto es "No dibujado", el punto no se iluminará y no se mostrará su valor.




El tamaño de los puntos se define en la pestaña "General" de la ventana de control de description.  

**2. Título de los ejes**. 
El título de los ejes horizontales y verticales se muestra si el título se definió en la pestaña "General" de la control Gráfico description. Este parámetro también puede definirse mediante programación con [grAxisTitle](../WDLang3/3042057.md).

**3. Título del gráfico**. 
Las características del título del gráfico pueden definirse en la pestaña "Detalles" del control description ("Editar título del gráfico").
Puede definir el título del gráfico, su posición y su estilo.

**4. mito**. 
Las características de la leyenda pueden definirse en la pestaña "Detalles" del control description ("Editar leyenda del gráfico").
Puedes definir su posición y su estilo.

**5. Etiquetas de serie**. 
El título de la leyenda corresponde al título de las diferentes series mostradas en el gráfico (definido en la pestaña "Series").

**6. Color de serie**. 
Se puede definir un color para cada serie en la pestaña "Serie" del control description. Un automatic Color es usado por Default.

**7. Etiquetas y marcas de categoría**.
Las etiquetas de las categorías se pueden definir en la pestaña "Serie" del control description.

La marca es calculada automáticamente por Default. La pestaña "General" del control description se utiliza para:

- definir una marca específica.

- definir una escala en el eje X en función del tiempo. Esta opción opera con datos de Fecha o FechaHora provenientes de una control Tabla o de un ítem del HFSQL. Esta opción no puede utilizarse con una serie rellenada mediante programación.

- definir una escala logarítmica en el eje Y.




También tiene la posibilidad de utilizar la escala secundaria para una o más series (pestaña"Series").

**8. Líneas de cuadrícula**. 
Para visualizar las retículas, seleccione "Cuadrícula" y "Cuadrícula" en la pestaña "General" del control description.

**9. Barras de desplazamiento utilizadas para realizar un zoom**. 
La capacidad de acercar un gráfico se puede definir en la pestaña "Detalles" del control description. Esta opción sólo está disponible en WINDEV.

**Otras características**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos de"Área" pueden ser mostrados con una animación. Para animar este tipo de gráfico, basta con marcar "Activar animaciones" en la pestaña "Detalles" del control description. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los gráficos de"área" se pueden mostrar en tiempo real. Para obtener más información, consulte [Gráfico en tiempo real](../WDChamp/1000021037.md).

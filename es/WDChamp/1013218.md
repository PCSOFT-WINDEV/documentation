## Presentación del control Imagen
<a name="presentacion_del_control_imagen_ELTTEXTE000250"></a>


### Control de ventana, página o reporte
<a name="control_ventana_pagina_reporte_ELTPARAGRAPHE000011"></a>

El control Imagen permite:

- Mostrar imágenes estáticas o animadas en sus aplicaciones.

- Dibujar.

- Mostrar gráficos creados con el diseñador de gráficos. Sin embargo, le recomendamos utilizar un [control Gráfico](../WDChamp/1013006.md).

- Ejecutar procesos específicos cuando un usuario hace clic en la imagen (equivalente a las áreas de clic de WINDEV 5.5).

- Mostrar videos con las [funciones MCI](../WDLang1/3051010.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Mostrar archivos PDF](../WDChamp/9500147.md). 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta funcionalidad requiere el [Framework GDI+](../Editeurs/2020007.md).

## Crear un control Imagen
<a name="crear_control_imagen_ELTTEXTE000274"></a>


### En una ventana o página
<a name="una_ventana_pagina_ELTPARAGRAPHE000105"></a>

Para crear un control Imagen en una ventana o página: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Imagen".

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual del control. Las principales opciones disponibles se pueden consultar:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) en la página [Control Imagen en una ventana](../WDChamp/9500147.md). 




**Para editar la imagen de un control Imagen**, seleccione la opción "Editar imagen" en el menú contextual. El editor de imágenes se abre automáticamente. Para obtener más información, consulte [Editor de imágenes](../Editeurs/1410086603.md). 
<a name="NOTE2_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### En un reporte
<a name="reporte_ELTPARAGRAPHE000149"></a>

Para crear un control Imagen:

1. En la pestaña "Creación", en el grupo "Visual", haga clic en "Imagen".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual. 

Para seleccionar la imagen (o el archivo PDF) que se muestra en el control, haga clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), opción "Examinar".

Para más información sobre el modo de visualización y transparencia de la imagen, consulte [Seleccionar el modo de visualización de la imagen](../WDChamp/1013134.md).

## Imágenes multilingües
<a name="imagenes_multilingues_ELTTEXTE000304"></a>
Si la aplicación es multilingüe, puede seleccionar una imagen diferente para cada idioma. Esta opción es útil cuando las imágenes contienen texto, por ejemplo.

Para gestionar imágenes multilingües: 

1. Abra la descripción del control Imagen, pestaña "General". 

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) y seleccione la opción "Multilingüe". La ventana de gestión de imágenes multilingües se abre. De forma predeterminada, el mismo archivo está asociado a todos los idiomas de la ventana, página o reporte.

3. Para cada idioma, haga clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif), opción "Examinar", y seleccione la imagen que corresponde al idioma. En tiempo de ejecución, se mostrará la imagen correspondiente según el idioma seleccionado con la función [Nation](../WDLang1/3054014.md).




**Atención**: La función multilingüe solo gestiona los archivos de imagen, no las características de visualización de la imagen.

**Observación**: Si se definió una imagen multilingüe, el mensaje "&lt;Valor multilingüe&gt;" se muestra en el área que corresponde al nombre del archivo de imagen en la ventana de descripción del control Imagen.

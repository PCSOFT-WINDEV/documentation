
## Vinculación de una Control de número a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000080"></a>
Una Control de número puede estar vinculada a un elemento de un archivo de datos, consulta, etc.

En este caso, las características de la posición son las siguientes:

- el elemento es de tipo **Integer**.

- el elemento contiene el valor de la Control de número.

- el valor del artículo debe estar entre el valor mínimo y máximo de la Control de número.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipulación del artículo y de la Control de número
<a name="manipulacion_del_articulo_control_numero_ELTTEXTE000104"></a>


### Actualización de la Control de número con los datos del artículo
<a name="actualizacion_control_numero_con_los_datos_del_articulo_ELTPARAGRAPHE000024"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo relacionado) serán actualizados por [FileToScreen](../WDLang4/3044210.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE2_2"></a>


### Actualizar un artículo con los datos de la Control de número
<a name="actualizar_articulo_con_los_datos_control_numero_ELTPARAGRAPHE000031"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la ventana serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

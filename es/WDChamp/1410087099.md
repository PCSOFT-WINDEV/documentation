<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
Un lector de PDF control puede ser manejado a través de la programación. 

Para manipular un control Lector PDF mediante programación, WINDEV propone las [funciones para gestionar el lector de PDF control](../WDLang1/1000023877.md). 

Esta página de ayuda explica como manipular un control Lector PDF mediante programación.

<a name="NOTE2_1"></a>


## Manejo del lector de PDF control mediante programación
<a name="manejo_del_lector_pdf_control_mediante_programacion_ELTTEXTE000119"></a>


### Iniciando un lector de PDF control
<a name="iniciando_lector_pdf_control_ELTPARAGRAPHE000142"></a>

Para mostrar un archivo PDF en un lector de PDF control, puede: 

- Definir el documento inicial (pestaña "General" de la ventana description de control). 

- Asignar el lector de PDF control con el archivo PDF. Por ejemplo: 
	
	```wl
	// Selects a .pdf file in the current directory
	sFileName is string
	sFileName = fSelect("", "Presentation.pdf", ...
		"Selection of PDF files", ...
		"Pdf file (*.pdf)" + TAB + "*.pdf", "*.pdf", fselOpen)
	
	IF sFileName ~="" THEN
		// No selected file
		RETURN
	END
	
	// Load PDF file
	PDF_MyDocument = sFileName
	```
 Si se usan contraseñas: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si se utiliza una contraseña para proteger el archivo al abrirlo, aparecerá una ventana para escribir la contraseña. 







- Utilice la función [Abrir](../WDLang1/1000023776.md). Por ejemplo: 
	
	```wl
	PDF_MyDocument.Open("C:\Users\test\Documents\file.pdf")
	
	IF ErrorOccurred() THEN
		Error(ErrorInfo())
		RETURN
	END
	```
 Observación: [Abrir](../WDLang1/1000023776.md) permite dar la contraseña del archivo a través de la programación.


<a name="NOTE2_2"></a>


### Imprimir el contenido de un lector de PDF control
<a name="imprimir_contenido_lector_pdf_control_ELTPARAGRAPHE000187"></a>

Para imprimir el contenido de un PDF Reader control, basta con utilizar [iPrintPDF](../WDLang5/1000020519.md). Por ejemplo: 

```wl
iPrintPDF(PDF_MyPDFReader)
```

### Propiedades específicas del lector de PDF control
<a name="propiedades_especificas_del_lector_pdf_control_ELTPARAGRAPHE000202"></a>Las siguientes propiedades son específicas del lector de PDF control. Se utilizan principalmente para manipular las características del control: 



|   |   |
| --- | --- |
| [DisplayMode](../Proprietes/1000023378.md) | El **DisplayMode** Property obtiene y cambia: <br><br>- el modo de visualización en un control Procesador de texto, <br><br>- el modo de visualización en un lector PDF control<br><br>- el modo de visualización en un editor HTML control, <br><br>- el modo de visualización en un Kanban control<br><br><br> |
| [DisplayModeAutoWhenLoading](../Proprietes/1000023467.md) | El **DisplayModeAutoWhenLoading** Property se utiliza para determinar y cambiar el modo de visualización al cargar un PDF Document. |
| [FilePath](../Proprietes/1000023104.md) | El **FilePath** Property se utiliza para identificar: <br><br>- el nombre del archivo xlsx associated con un control Hoja de cálculo.<br><br>- el nombre del archivo associated con un editor de imágenes control. <br><br>- el nombre del archivo PDF associated con un lector de PDF control. <br><br>- el nombre del archivo DOCX associated con un control Procesador de texto. <br><br>- el nombre del archivo wddiag asociado a un editor Diagram control. <br><br><br> |
| [NavigationPaneVisible](../Proprietes/1000023437.md) | El **NavigationPaneVisible** Property se utiliza para mostrar el panel de miniaturas en el PDF Reader control. |
| [NumberAccessiblePages](../Proprietes/1000023477.md) | El **NúmeroDePáginasAccesibles** Property se utiliza para get el número de páginas cargadas actualmente en un lector de PDF o control Procesador de texto. |
| [NumberDisplayedPage](../Proprietes/1000023479.md) | La propiedad **NumberDisplayedPage** permite identificar y cambiar el número de la página que se muestra actualmente en el control Lector PDF o Procesador de texto. |
| [NumberPage](../Proprietes/2510084.md) | La propiedad **NumberPage** permite obtener: <br><br>- el número de páginas de un archivo "multipágina" Image. Esta Image se muestra en una control Imagen o en el fondo de una control Gráfico.<br><br>- el número de páginas de un archivo PDF que se muestra en un control Imagen.<br><br>- el número de páginas que se encuentran en un archivo PDF que se muestra en un lector de PDF control.<br><br>- el número de páginas que se encuentran en un archivo DOCX mostrado en un control Procesador de texto. <br><br><br> |
| [UseMode](../Proprietes/1000023449.md) | El **modo de uso** Property se utiliza para determinar o cambiar cómo reacciona el lector de PDF control a la entrada del mouse.. El ratón se utiliza para seleccionar el texto en la Document o para moverse dentro de las páginas de Document. |
| [Zoom](../Proprietes/1000017201.md) | La propiedad **Zoom** obtiene o establece el valor del zoom: <br><br>- en un control Imagen.<br><br>- en un control Mapa. <br><br>- en un control Organigrama.<br><br>- en un control Tabla, Tabla TreeView, Tabla dinámica, Hoja de cálculo o List Box. <br><br>- en un control Procesador de texto. <br><br>- en un control Lector PDF.<br><br>- en un control Cámara.<br><br>- en un control Editor de diagramas; <br><br>- en una ventana.<br><br><br> |





Para obtener una lista completa de las propiedades de WLanguage que se pueden utilizar con los controles de PDF Reader, consulte [PDF Reader Propiedades de control](../WDChamp/1410087038.md).

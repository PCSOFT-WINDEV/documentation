
## AAF: Clasificar en las columnas de un control Tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
WINDEV permite al usuario ordenar fácilmente las columnas de un control Tabla. 

Las clasificaciones también pueden realizarse mediante programación. 

Observación: Estas operaciones también se pueden realizar en un control Tabla TreeView. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Clasificación de las columnas (AAF)
<a name="clasificacion_las_columnas_aaf_ELTTEXTE000170"></a>


### Hechizo simple
<a name="hechizo_simple_ELTPARAGRAPHE000022"></a>

Para ordenar las columnas de una control Tabla, el usuario debe hacer clic en el título de la columna.

Las columnas clasificables se identifican con una flecha doble.

Cuando la ordenación está activada, una pequeña flecha indica la dirección de ordenación de la columna.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Triez%20les%20donn%E9es%20affich%E9es%20dans%20vos%20tables%20en%20un%20clic.jpg)


**Consejo**: Para implementar esta característica, "Sortable" debe ser marcada en la pestaña "General" de la ventana de description de la columna.
<a name="NOTE2_2"></a>


### Destino múltiple
<a name="destino_multiple_ELTPARAGRAPHE000037"></a>

Para clasificar el contenido de una control Tabla en varias columnas, el usuario debe:

1. Mantenga pulsada la tecla Ctrl.

2. Haga clic en el título de las columnas que se tendrán en cuenta para la ordenación. Las columnas deben seleccionarse según el orden solicitado. Por ejemplo, para ordenar por empresa y por nombre, debe hacer clic en la columna "Empresa" y luego en la columna "Nombre".




**Observación**: La clasificación múltiple sólo está disponible para la memoria controles Tabla.
<a name="NOTE2_3"></a>


### Recordar orden
<a name="recordar_orden_ELTPARAGRAPHE000051"></a>

Cuando el usuario ha realizado una o más clasificaciones en las columnas de un control Tabla, estas clasificaciones pueden almacenarse para que Default las aplique.. 

Para habilitar esta función, debe: 

1. Abrir el menú contextual de la columna de clasificación de la control Tabla. Para obtener más información, consulte [Menú contextual de las columnas](../WDChamp/9500125.md). 

2. Selecciona "Guardar las clasificaciones"..

## Desarrollo y programación
<a name="desarrollo_programacion_ELTTEXTE000206"></a>
**Al crear un control Tabla**:
Para hacer una columna ordenable, todo lo que tienes que hacer es seleccionar "Ordenable" en la pestaña "General" de la ventana de description de la columna. 

**Mediante programación**: 

- Para ordenar el contenido de una control Tabla, basta con utilizar [TableSort](../WDLang1/3074035.md). 

- [TableSortedColumn](../WDLang1/3074010.md) devuelve la lista de columnas ordenadas en control Tabla. 

- La [TiendaSortAAF](../Proprietes/1000020069.md) Property se utiliza para determinar y especificar si el orden de clasificación definido por el usuario en una Tabla o control Tabla TreeView se mantiene cuando se cierra y se vuelve a abrir una aplicación.

- El [SortOption](../Proprietes/1000017044.md) Property se utiliza para get y cambiar las opciones de ordenación en las columnas de un control Tabla. 

- La [Clasificable](../Proprietes/2510116.md) Property se utiliza para saber si una columna es ordenable y a las columnas ordenables. 

- Si no quiere mostrar las opciones "Recordar filtros" y "Recordar ordenaciones" en el menú contextual, simplemente utilice [AAFDisable](../WDLang1/1000022018.md) (o [DisableAAF](../WDLang1/3013061.md)) con la constante *aafSaveFilterAndSort*.

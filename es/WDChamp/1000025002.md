## Eventos predeterminados
<a name="eventos_predeterminados_ELTTEXTE000188"></a>
**Para el control Action Bar** , WINDEV Mobile admite los siguientes eventos por Default (orden de aparición en el editor de código):


|   |   |
| --- | --- |
| **Evento** | **Condición de ejecución** |
| Haga clic en el botón izquierdo | Se ejecuta cuando el usuario hace clic en el botón izquierdo de la ventana de Action Bar. Se puede ejecutar una acción preestablecida después de esta Event. Esta acción se define en la pestaña "General" (opción "1", botón de la izquierda).<br><br><br><br>Para más detalles, véase "Botón izquierdo" en la Page denominada [Descripción del control Action Bar](../WDChamp/1000025005.md).<br><br>Observación: Esta Event sustituye a la Event"Haga clic en el icono" , disponible en la versión 18. |
|   | Se ejecuta cuando el usuario cambia el campo de búsqueda de la Action Bar. <br><br>Este evento solo está disponible si se activó la búsqueda en el Action Bar (opción "Permitir búsqueda en el Action Bar" en la pestaña "Detalles" de la ventana de descripción del control Action Bar). <br><br>Observaciones: <br><br>- El campo de búsqueda de la Action Bar puede convertirse en Visible por [ActionBarSearchVisible](../WDLang1/1000021599.md). <br><br>- El valor introducido en el campo de búsqueda puede recuperarse y modificarse con la [SearchValue](../Proprietes/1000021602.md) Property. <br><br><br> |
|   | Se ejecuta cuando el usuario valida la entrada en el campo de búsqueda de la Action Bar. <br><br>Este evento solo está disponible si se activó la búsqueda en el Action Bar (opción "Permitir búsqueda en el Action Bar" en la pestaña "Detalles" de la ventana de descripción del control Action Bar). <br><br>Observaciones: <br><br>- El campo de búsqueda de la Action Bar puede convertirse en Visible por [ActionBarSearchVisible](../WDLang1/1000021599.md). <br><br>- El valor introducido en el campo de búsqueda puede ser recuperado con la [SearchValue](../Proprietes/1000021602.md) Property. <br><br><br> |
| Modificar la vista visualizada | Ejecutar al cambiar la vista visualizada. Esta Event permite, por ejemplo, modificar la ventana interna que muestra la Zona Action Bar control. |


**Para la zona Action Bar** , WINDEV Mobile admite los siguientes eventos por Default (orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Después de cargar la ventana interna | Se ejecuta cuando la ventana interna es modificada por [ChangeSourceWindow](../WDLang1/3038053.md). Esta Event se ejecuta después de las Event s "Declaraciones globales" e "Inicialización" de la nueva ventana interna.<br><br>Esta Event le permite, por ejemplo, inicializar los controles o restablecer los valores en la ventana interna.<br><br>No se debe utilizar la función [ChangeSourceWindow](../WDLang1/3038053.md) en este evento. |
| Antes de descargar la ventana interna | Se ejecuta cuando la ventana interna es modificada por [ChangeSourceWindow](../WDLang1/3038053.md). Este Event se ejecuta antes del cierre del Event de la ventana interna inicial.<br><br>Esta Event se utiliza, por ejemplo, para guardar los valores presentes en la ventana interna.<br><br>No se debe utilizar la función [ChangeSourceWindow](../WDLang1/3038053.md) en este evento. |

## Vista general del applet de Java control
<a name="vista_general_del_applet_java_control_ELTTEXTE000140"></a>


### Control de página dinámica o estática
<a name="control_pagina_dinamica_estatica_ELTPARAGRAPHE000011"></a>

A Java Applet control le permite incluir fácilmente los Applets Java en sus sitios WEBDEV.

Para crear un Applet Java control, debe tener:

- el applet Java (archivo "Class").

- los diferentes parámetros del applet.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación de un applet de Java control
<a name="creacion_applet_java_control_ELTTEXTE000164"></a>
Para crear un Applet Java control: 

1. En la pestaña "Creación", en el grupo "Integración", haga clic en "Applet java".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Características específicas del Applet Java control
<a name="caracteristicas_especificas_del_applet_java_control_ELTTEXTE000188"></a>


### Control description
<a name="control_description_ELTPARAGRAPHE000049"></a>

Para configurar el control del Applet de Java, especifique los siguientes detalles en la pestaña "General" del control description:

- **el archivo Jar** si fuera necesario. Este archivo debe ser especificado cuando el applet de Java se incluye en un archivo. 

- el nombre **del fichero Applet** (sin extension) correspondiente al applet principal a ejecutar (fichero ".CLASS").
	Ejemplo: "Pergamino"
	Observación: Los nombres de los applets son "sensibles a mayúsculas y minúsculas".. Preste especial atención a los caracteres en mayúsculas y minúsculas.

- la **base**, que significa el nombre del directorio que contiene todos los componentes del applet Java. Esta cadena debe ir precedida y seguida del carácter "/".. Le aconsejamos que coloque todos los objetos necesarios para ejecutar el applet en el directorio "&lt;Nombredelproyecto&gt;_WEB" del proyecto o en un subdirectorio del directorio "&lt;Nombredelproyecto&gt;_WEB". 
	Ejemplo: "/JavaSite_web/MisApplets/"
	Observación: Dependiendo del servidor Web utilizado, los nombres de los directorios son "sensibles a mayúsculas y minúsculas".. Preste especial atención a los caracteres en mayúsculas y minúsculas.

- la **tabla de parámetros**. Este tabla agrupa todos los parámetros necesarios para ejecutar el applet. La lista de parámetros es suministrada por el distribuidor de Applet. 
	***Para agregar un parámetro***, haga clic en el botón "Agregar": Especifique el nombre del parámetro (columna"Nombre del parámetro") y su valor (columna"Valor").. 
	***Para eliminar un parámetro***, seleccione el parámetro y haga clic en el botón "Eliminar".
	***Si el applet Java fue creado con WINDEV***, el nombre de los parámetros corresponde al nombre de los parámetros recuperados por [CommandLine](../WDLang1/3013051.md). 
	Ejemplo: Código de inicialización del proyecto WINDEV utilizado para crear el applet:
	
	```wl
	P1 is string
	P1 = CommandLine("Param1")
	```

	En este caso, el nombre del parámetro que se utilizará en WEBDEV es "Param1".



<a name="NOTE3_2"></a>


### Directorio actual
<a name="directorio_actual_ELTPARAGRAPHE000084"></a>

Para especificar la base de datos y los parámetros, el ruta es relativo al directorio actual. El directorio actual es el directorio donde se genera el archivo HTM correspondiente a la Page.

- **Para un Page**dinámico, el directorio actual es el subdirectorio de idioma del directorio del proyecto (FR, UK, etc.). 
	Ejemplo: Si el parámetro es un archivo ubicado en el directorio "&lt;NombreProyecto&gt;_WEB", el valor del parámetro corresponde a "/&lt;NombreProyecto&gt;_WEB/&lt;NombreArchivo&gt;".

- **Para un Page**estático, el directorio actual es el subdirectorio de idioma del directorio &lt;NombreProyecto&gt;_WEB (FR, UK, etc.).




**Ejemplo**: Si el parámetro es un archivo que se encuentra en el directorio "&lt;Nombre del proyecto&gt;_WEB", el valor del parámetro corresponde a "../&lt;Nombre del archivo&gt;".

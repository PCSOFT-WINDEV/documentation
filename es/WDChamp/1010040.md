


## Escala de 9 sectores
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
La escala de 9-rango de resolución le permite redimensionar las imágenes de forma armoniosa. La escala 9-rango de resolución se puede utilizar en Image s en un control Imagen o en el fondo Image de una ventana.

El principio de la escalada de 9-rango de resolución es simple: la Image se divide en 9 secciones. Entonces, tienes la capacidad de definir cómo se comportará cada sección cuando la Image control o la ventana sea redimensionada.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fondo Image de una ventana
<a name="fondo_image_una_ventana_ELTTEXTE000184"></a>
Para definir la escala 9-rango de resolución para el fondo Image de una ventana: 

1. Abre la ventana description.

2. En la pestaña "Image", seleccione "Utilizar escala de 9 sectores" y configure este modo mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=btnrupture.gif).

3. La siguiente ventana se abre:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Mode9image.gif)


4. Configure el escalado de 9-rango de resolución según sus necesidades. El botón "Vista previa" le permite ver cómo se mostrará el Image en tiempo de ejecución.




**Observación**: Algunos consejos para optimizar la velocidad de visualización de las ventanas que utilizan la escala de 9-rango de resolución:

- no utilice una "Transparent GIF" Image con escala 9-rango de resolución.

- usar un fondo plano para la imagen y para el fondo Color de la ventana.

- intente definir márgenes que sean múltiplos de 4.

## control Imagen
<a name="control_imagen_ELTTEXTE000208"></a>
Para definir la escala de 9-rango de resolución en un control Imagen:

1. Abra la ventana de descripción del control.

2. En la pestaña "General", seleccione "Utilizar escala de 9 sectores" y configure este modo mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=btnrupture.gif).




**Observación**: no se recomienda utilizar una "Transparent GIF" Image con escala 9-rango de resolución, ya que la visualización puede ser más lenta.

<a name="NOTE4_1"></a>


## Control Botón
<a name="control_boton_ELTTEXTE000232"></a>
Para definir la escala de 9-rango de resolución en un control Botón:

1. Abra la ventana de descripción del control.

2. En la pestaña "General", seleccione "Estirar: Escalado personalizado 9-rango de resolución" y configure este modo con el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=btnrupture.gif).




**Observación**: no se recomienda utilizar una "Transparent GIF" Image con escala 9-rango de resolución, ya que la visualización puede ser más lenta.

## Configuración de la escala de 9-rango de resolución
<a name="configuracion_escala_9rango_resolucion_ELTTEXTE000256"></a>
El método de escalado 9-rango de resolución consiste en dividir el fondo Image en 9 zonas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Mode9image3.gif)


**Para cada área**, puede definir si la sección de la Image correspondiente al área debe ser mostrada o no: simplemente haga clic en el área para hacerla visible o Invisible. Si la sección de imágenes no se muestra, será sustituida por el fondo Color de la ventana o control.

**Los márgenes** se utilizan para definir el tamaño de cada área. Tiene la posibilidad de introducir los valores de las diferentes áreas o de mover la Line correspondiente al margen.

Al redimensionar la ventana, sólo se redimensionan las áreas 2, 4, 6, 8 y 5 para ocupar todo el espacio.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Mode9image2.gif)


El método de escalado de 9-rango de resolución se utiliza para gestionar el comportamiento de cada área al redimensionar la ventana. Para cada área, la Image será ampliada:

- repitiendo la Image que constituye el área

- estirando la Image que constituye el área


En la ventana de configuración del modo de escalado 9-rango de resolución:

- La opción "Modo ancho" permite gestionar la operación de redimensionamiento de las zonas 2 y 8 (zona superior e inferior)

- La opción "Modo Altura" se utiliza para gestionar la operación de redimensionamiento de las áreas 4 y 6 (Área izquierda y Área derecha).

- La opción "Modo centro" se utiliza para gestionar la operación de redimensionamiento para el área 5 (Área central).

## Optimización de la escala de 9-rango de resolución
<a name="optimizacion_escala_9rango_resolucion_ELTTEXTE000280"></a>
Algunos consejos optimizan la escala de 9-rango de resolución:

- Utilice múltiplos de 8 para los márgenes en anchura.

- El ancho del área central debe ser al menos igual a 32 píxeles.

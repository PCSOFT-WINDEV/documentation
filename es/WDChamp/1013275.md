## Vista general de la control Miniatura
<a name="vista_general_control_miniatura_ELTTEXTE000230"></a>
El control Miniatura se utiliza para gestionar automáticamente la visualización de imágenes en formato de miniaturas.

Durante un clic en la miniatura, la Image inicial se mostrará automáticamente: 

- en una nueva ventana del navegador o en una pestaña del navegador (según la configuración del navegador). 

- como un popup (automatic o en una control Imagen de una célula). 
	La ventana emergente cambia de tamaño junto con el navegador, y se utiliza automáticamente una transición cuando se muestra la ventana emergente. 
	Esta ventana emergente tiene las siguientes características: 

	- Posibilidad de cerrar haciendo clic fuera de la ventana, pulsando Esc o haciendo clic en el botón Cerrar Icon. 

	- Barra de herramientas con efectos de entrada, salida y desplazamiento, con botones de rotación y zoom

	- Rotación en incrementos de 90 grados.

	- Zoom de 2,5x con la rueda del ratón.

	- Zoom en incrementos del 100%.

	- Posibilidad de arrastrar y soltar los archivos Image.







No se requiere el código Line.

Este tipo de control simplifica la creación de páginas de "Catálogo".

## Creación de una control Miniatura
<a name="creacion_una_control_miniatura_ELTTEXTE000254"></a>


### Creación de una control Miniatura
<a name="creacion_una_control_miniatura_ELTPARAGRAPHE000080"></a>

Para crear una control Miniatura:

1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Imagen" y seleccione "Miniatura".

2. Haga clic en la ubicación en la que desea crear el control. El control se crea automáticamente.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.
<a name="NOTE2_2"></a>


### Describiendo un control Miniatura
<a name="describiendo_control_miniatura_ELTPARAGRAPHE000104"></a>

Las principales características de una control Miniatura son las siguientes. Estas características se encuentran en las diferentes pestañas de la ventana de control de description. 

**Pestaña "General"**: 

- Fuente Image: 

	- **Image**: Se utiliza para especificar la ruta de la imagen de gran tamaño asociada a la control Miniatura.. Esta Image se mostrará automáticamente cuando se haga un clic en la miniatura. 

	- **Thumbnail**. Se utiliza para especificar la ruta de la Image que se utilizará como miniatura. Esto le permite mostrar una Image que difiere de la fuente Image en la control Miniatura. Si esta control no se rellena, la Image de la miniatura es generada automáticamente por WEBDEV a partir de la Image de origen.




- Ajustes de visualización: 

	- **Modo de visualización**. Para obtener más información, consulte [Seleccionar el modo de visualización de la imagen](../WDChamp/1013134.md). Sólo están disponibles los modos "Homothetic centered" y "Stretched".

	- **Transparencia**. Para obtener más información, consulte [Seleccionar el modo de visualización de la imagen](../WDChamp/1013134.md). 

	- **Visualización en formato grande**: Esta opción permite elegir el lugar donde debe mostrarse la imagen inicial durante un clic en la miniatura (nueva pestaña del navegador de Default).
			Se pueden definir varios destinos para una visualización específica de la miniatura Image: 

		- **Popup automática**: la miniatura Image se muestra en formato popup en la página (el fondo de la página es Grayed). El estilo del popup de automatic se puede definir en la pestaña "Estilo" de control Miniatura (elemento "popup de automatic").  

		- **Popup personalizada**: 

			- si tu página contiene una celda y una control Imagen en la celda, tienes la capacidad de mostrar la Image en esta celda. Para ello, seleccione el nombre de la celda y el nombre de la control Imagen. La imagen de la miniatura se muestra en la celda como popup en la página (el fondo de la página es Grayed). 

			- si tu página contiene un popup y una control Imagen en el popup, tienes la posibilidad de mostrar la Image en este popup. Para ello, basta con seleccionar el nombre de la ventana emergente y la control Imagen. La miniatura Image se muestra en la control Imagen del popup. 







**Pestaña "UI"**: 

- **Transición**: Cuando se modifica el valor de la miniatura control, se puede seleccionar una transición como Faded, Overlap o Swipe para que el cambio de Image sea más armonioso. Para obtener más información, consulte [Animar una imagen](../WDChamp/9500117.md). 

- **Deslizamiento automático**: Esta opción está disponible para imágenes "dinámicas" o imágenes "de una base de datos".: ruta". Para obtener más información, consulte [Imagen con deslizamiento automático](../WDChamp/9500118.md).




**Pestaña "Detalles"**: 

- **Carga diferida**: La opción "Cargar imagen cuando el control está en la pantalla" permite retrasar la carga de la imagen. De hecho, cuando un navegador realiza el renderizado de una página, carga la página y sus imágenes desde el servidor.. Este tiempo de carga puede ser bastante largo en algunos casos.. Esta opción permite que el navegador cargue la imagen sólo cuando ésta se muestre en la sección de la página Visible. También tiene la capacidad de especificar la espera Image, usada durante la carga retrasada de Image. 

- **Zoom al pasar el cursor**: Esta opción se utiliza para mostrar la Image inicial en un popup cuando se pasa por encima de la miniatura. Es posible elegir: 

	- **Popup automática**: la miniatura Image se muestra en formato popup en la página (el fondo de la página es Grayed). El estilo del popup de automatic se puede definir en la pestaña "Estilo" de control Miniatura (elemento "popup de automatic").  

	- **Popup personalizada**: 

		- si tu página contiene una celda y una control Imagen en la celda, tienes la capacidad de mostrar la Image en esta celda. Para ello, seleccione el nombre de la celda y el nombre de la control Imagen. La imagen de la miniatura se muestra en la celda como popup en la página (el fondo de la página es Grayed). 

		- si tu página contiene un popup y una control Imagen en el popup, tienes la posibilidad de mostrar la Image en este popup. Para ello, basta con seleccionar el nombre de la ventana emergente y la control Imagen. La miniatura Image se muestra en la control Imagen del popup. 




- **Options**: 

	- **Permitir indirecciones (ralentiza la ejecución)**: Esta opción permite manipular la control mediante indirecciones. Para obtener más información, consulte [Operadores indirectos](../Motscles/1512005.md). 

	- ** Desactivar la opción para guardar imágenes en el menú contextual del navegador**: Si esta opción está marcada, el menú contextual de la miniatura no se mostrará en el navegador. Algunas operaciones para manejar la miniatura en el navegador (guardar, imprimir, etc.) no estarán disponibles. 
			**Atención**: Sólo la Image mostrada como miniatura está protegida. La Image abierta en el nuevo navegador no está protegida. Se mostrará el menú contextual de esta Image.




- **Compatibilidad con versiones anteriores**: Estas opciones recuperan las opciones de configuración de las imágenes disponibles en las versiones 23 y anteriores.. Estas opciones están disponibles para la compatibilidad. Ahora, el tipo de Image se detecta automáticamente. Para más detalles sobre las opciones disponibles para la compatibilidad, consulte [Seleccionar el tipo de imagen: dinámico, estático, generado, ....](../WDChamp/1013132.md).

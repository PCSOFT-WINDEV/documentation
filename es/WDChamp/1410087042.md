## Vista general del contenedor nativo control
<a name="vista_general_del_contenedor_nativo_control_ELTTEXTE000091"></a>
El contenedor nativo control se utiliza para gestionar los control s no creados con WINDEV o WINDEV Mobile. Esto le permite incluir fácilmente en una ventana controles procedentes de un módulo externo como el SDK. 

El control «Container» devuelve el handle (UIView en iOS, view Identifier en Android) que espera el SDK. Entonces, este manipular puede ser usado directamente para manipular el control.

## Creación de una control Contenedor nativo
<a name="creacion_una_control_contenedor_nativo_ELTTEXTE000115"></a>
Para crear una control Contenedor nativo:

1. En la pestaña "Creación", en el grupo "Otros controles", haga clic en "Contenedor nativo". 

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.
	Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si es necesario, cambie el tamaño del control usando los controladores de tamaño. 




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

Para manipular el control Contenedor nativo, véase [Manejo de una control Contenedor nativo mediante programación](../WDChamp/1410087045.md).

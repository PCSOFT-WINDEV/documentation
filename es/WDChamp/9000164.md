## Presentación
<a name="presentacion_ELTTEXTE000162"></a>
El texto enriquecido admite múltiples funciones avanzadas: fuente, peso, tamaño, color, atributo, ...

Puede utilizar texto enriquecido en muchos controles:

- corregir

- Combo Box

- List Box

- Casilla de verificación

- Botón de opción

- vincular




El texto enriquecido puede corresponder a: 

- el título de control. 

- el título de las opciones (por ejemplo, para los controles del Botón de radio o de la Casilla de verificación).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000186"></a>


### Escribir subtítulos en texto enriquecido
<a name="escribir_subtitulos_texto_enriquecido_ELTPARAGRAPHE000035"></a>

Para utilizar texto enriquecido en el pie de foto de control: 

1. Seleccione la control en el Page. 

2. Haga clic con el botón derecho del ratón para abrir el menú contextual de la control y seleccione "Editar pie de foto".. 

3. La leyenda control pasa al modo "Edición: aparece un borde amarillo alrededor del título seleccionado. 

4. Tiene la posibilidad de personalizar el enriquecimiento a través del panel "Texto" que aparece en la cinta de opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Texte_Riche_2.gif&type=thumb)
Es posible: 

	- Personalizar el formato del texto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Texte_Riche_1.gif)


	- Añadir un enlace. 

- Agregar una imagen. 

5. Para validar, haga clic en cualquier lugar de Page. 




Por ejemplo:

- Una control Estático que contiene un enlace: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-ZTR-Lien%20dans%20texte.jpg)


- Una control Campo de entrada cuyo título contiene un enlace: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-ZTR-Saisie.jpg)


- Una control Casilla de verificación cuyo título está en negrita y cuyo título de opción contiene un enlace: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-ZTR-Lien%20dans%20interrupteur.jpg)


- Una control Botón de opción cuyo título está en negrita y cuyo título de opción contiene un enlace: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-ZTR-S%E9lecteur.jpg)


- Una petición para Email, con una estrella roja: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB%20-%20Texte%20riche%20dans%20les%20champs.jpg)





**Observaciones**: 

- Para que el título sea editable, también puede: 

	- haga clic en la control dos veces en el editor. 

	- haga clic en "Editar con formato" en la pestaña "General" de la ventana de control description. 




- Para editar una opción de un Botón de Radio o control Casilla de verificación: 

	- del editor, seleccione la control y haga clic en la opción deseada: la opción se vuelve editable. 

	- vaya a la ventana de control description, pestaña "Contenido", seleccione la opción deseada y haga clic en "Editar con formato"..

### Edición del código HTML de los subtítulos
<a name="edicion_del_codigo_html_los_subtitulos_ELTPARAGRAPHE000087"></a>

Usted tiene la capacidad de editar el código fuente HTML de los subtítulos de WINDEV. Esto le permite modificar el código HTML generado, añadir su propio código HTML e incluir un efecto específico por ejemplo, etc.

Para editar el código HTML de un título: 

1. Seleccione la control en el Page. 

2. Abra la ventana de descripción del control (seleccione "Descripción" del menú contextual). 

3. En la pestaña "General", haga clic en "Editar código HTML". 

4. Aparece el editor de código HTML. Tiene la posibilidad de modificar el código HTML. 




Observaciones: 

- Si es posible, las modificaciones realizadas en el editor de código HTML se Visible en WYSIWYG en el editor Page. Los códigos HTML avanzados (tipo de animación) no son Visible. 

- El editor de código HTML permite escribir código HTML que contenga funciones de JavaScript. Para más detalles, véase Editor HTML.

## Creación de un organizador reporte
<a name="creacion_organizador_reporte_ELTTEXTE000215"></a>


### Crear un reporte
<a name="crear_reporte_ELTPARAGRAPHE000037"></a>

Crear un "Organizador" reporte: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Reporte", y luego en "Reporte".

	- El asistente de creación de reportes se abre.




2. Seleccione un "Organizador o Programador" reporte. 

3. En el resto de la asistente, especifique: 

	- el tipo de reporte a crear ("Organizador reporte") con el [información específica para la creación del organizador](#NOTE2_2): 

		- el modo de visualización inicial.

		- el número de días visualizados. 

		- la granularidad de las líneas de cuadrícula. 




- la modelo utilizada en caso necesario, 

- la fuente de datos de reporte (archivo de datos, consulta, control Agenda, ...). Para obtener más información, consulte [Fuente de datos Reporte](../WDChamp/1011055.md).



<a name="NOTE2_2"></a>


### Información específica para la creación de la Agenda reporte
<a name="informacion_especifica_para_creacion_agenda_reporte_ELTPARAGRAPHE000067"></a>

La información específica para la creación del Organizador reporte es:

- Modo de visualización inicial: El organizador se puede mostrar: 

	- a diario,

	- sobre una base semanal,

	- mensualmente. 




- La granularidad de las líneas de cuadrícula. Esta granularidad se utiliza para definir el tamaño de la cuadrícula utilizada para mostrar las líneas de la cuadrícula de control Agenda. Esta granularidad es set a 30 minutos por Default. Esta información debe ser modificada de acuerdo con el número de días a visualizar.. 
	Esta característica puede ser modificada en el código a través de la [GranularityGridlines](../Proprietes/1000020107.md) Property. 




**Observación**:  Si el Organizador reporte es relacionado a una fuente de datos (consulta, archivo de datos), debe especificar los elementos que se tendrán en cuenta para mostrar las citas. Se debe especificar la siguiente información: 

- el título de Appointment. 

- las fechas de inicio y fin de Appointment.

## Características del organizador reporte y control Agenda
<a name="caracteristicas_del_organizador_reporte_control_agenda_ELTTEXTE000245"></a>
Tiene la posibilidad de modificar las siguientes características del Organizador reporte: 

- La fuente de datos del organizador reporte puede modificarse en la pestaña "Datos" de la ventana description de reporte ("Reporte description" en el menú contextual de reporte). 

- La pestaña "General" de la ventana description de control Agenda permite especificar: 

	- el modo de visualización inicial: Día, semana, mes. 

	- el número de días visualizados. 

	- el período de tiempo de Visible. 

	- el primer día de la semana.

	- la granularidad de las líneas de grilla.

	- la visualización de las citas de todo el día. Si la opción "Mostrar citas de todo el día" está marcada, la reporte admite las citas para las que la Property "Todo el día" es set a verdadero (para más detalles, véase [Appointment Variable](../WDLang1/1000019244.md)). En este caso, en el organizador aparece un área utilizada para visualizar las citas diarias.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Agenda_journee_Etat.gif)
Observación: Asegúrese de que el área utilizada para mostrar las citas diarias tenga las dimensiones adecuadas en la pantalla de editor de reportes. Se puede redimensionar si es necesario. 

	- la máscara de visualización para la fecha y los horarios. 

	- la reporte personalizada para mostrar las citas. 

		- Si la opción seleccionada es "Ninguna", se utiliza la visualización Default. 

		- Si se selecciona "Preajuste reporte", se añade automáticamente al proyecto un preajuste reporte. Este reporte se denomina "RPT_WinDevViewAPT" por Default y se utiliza automáticamente para mostrar las citas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_WinDevvueRDV.gif)
 Este reporte se puede personalizar, renombrar, etc.

		- Si la opción seleccionada corresponde a un proyecto reporte, este reporte se utilizará para mostrar las citas..
						Le aconsejamos personalizar el reporte propuesto por Default.
						La misma reporte se puede utilizar para personalizar las citas en un Programador u Organizador reporte..  




- Si el control Agenda se relacionado a una fuente de datos (archivo de datos, consulta, ...), la pestaña "Contenido" que se encuentra en la ventana de descripción del control se utiliza para relacionado los datos Appointment con los datos que se encuentran en la fuente de datos. 

- La pestaña "Estilo" de la ventana control Agenda description permite especificar el estilo de las citas que aparecen en control Agenda.

| [OrganizadorBorrarTodo](../WDLang1/1000018882.md) | Borra todas las citas de una control Agenda. |
| [OrganizadorFondoColor](../WDLang1/1000020004.md) | Define un fondo Color para una franja horaria en un control Agenda. |
| [OrganizerChangeMode](../WDLang1/1000019497.md) | Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). |
| [OrganizerChangeZoom](../WDLang1/1000018883.md) | Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). Esta función se conserva por motivos de compatibilidad con versiones anteriores. |
| [OrganizerDeleteAppointment](../WDLang1/1000018881.md) | Borra una Appointment de una control Agenda. |
| [OrganizerListAppointment](../WDLang1/1000018880.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Agenda, <br><br>- todas las citas encontradas en un control Agenda,<br><br>- una Appointment específica (seleccionada o suspendida).<br><br><br> |
| [OrganizerMode](../WDLang1/1000019545.md) | Devuelve el modo de visualización de un control Agenda: día, semana, mes, ... |
| [OrganizerPosition](../WDLang1/1000018889.md) | Muestra un control Agenda a partir de una fecha especificada o devuelve la fecha del primer día (o la fecha del último día) que aparece en un control Agenda. |
| [OrganizerTimeDisplayed](../WDLang1/1000018888.md) | Define el intervalo de tiempo que muestra control Agenda en el modo Día o en el modo Semana. |
| [OrganizerZoom](../WDLang1/1000018884.md) | Devuelve el intervalo de tiempo mostrado por un control Agenda. |

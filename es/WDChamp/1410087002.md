## Vista general de la control Barra de navegación
<a name="vista_general_control_barra_navegacion_ELTTEXTE000270"></a>


### Control en una página estática o dinámica
<a name="control_una_pagina_estatica_dinamica_ELTPARAGRAPHE000011"></a>

La control Barra de navegación se utiliza para crear un área de menú que puede adaptarse fácilmente al tamaño del navegador.. Esta área puede contener todo tipo de controles. 

Hay dos modos de visualización disponibles: 

- **Modo ampliado**: La barra de navegación incluye 3 secciones: Izquierda, centro, derecha. Se visualizan las 3 secciones. <br>![Una barra de navegación en modo extendido](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-Responsive-Barre-de-navigation-2-colonnes.jpg)


- **Compact mode**: La barra de navegación incluye las secciones izquierda y derecha, así como una sección central que se puede ampliar (al hacer clic en un botón, por ejemplo).. <br>![Una barra de navegación en modo compacto](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-Responsive-Barre-de-navigation-1-colonne.jpg)<br>![Una barra de navegación en modo compacto con la sección central ampliada](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB-Responsive-Barre-de-navigation-1-colonne-menu-deroule.jpg)





Marca **Remark**: Este control sólo está disponible para las páginas en modo "Diseño Web con capacidad de respuesta"..

## Creación de una control Barra de navegación
<a name="creacion_una_control_barra_navegacion_ELTTEXTE000294"></a>
**Para crear un control Barra de navegación**:

1. Crear o abrir una página en modo "Diseño Web con capacidad de respuesta. 

2. En el panel "Creación", en el grupo "Navegación", expande "Barra de nav." y selecciona el tipo de barra de navegación para crear: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre%20de%20navigation%20-%20Creation%20-%20HC%20N%B0001.gif)


	- **Barra de navegación en blanco**: Esta barra de navegación no contiene ningún control. Se puede personalizar por completo. Se puede utilizar, por ejemplo, para gestionar un diseño específico.

- **Barra de navegación con menú'Burger**: Esta barra de navegación se rellena previamente para gestionar la visualización de los menús en las diferentes plataformas. Algunos parámetros se especifican por defecto, como el cambio de visualización del primer rango de resolución.

3. Haga clic en la posición donde se creará el control en la página. El control aparece en el modo de creación.



<a name="NOTE2_2"></a>


### Características de control Barra de navegación con Burger creadas por defecto
<a name="caracteristicas_control_barra_navegacion_con_burger_creadas_por_defecto_ELTPARAGRAPHE000061"></a>

La control Barra de navegación se adapta automáticamente al modo Responsive Web Design.: 

- En el mayor rango de resolución: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Creation%20-%20HC%20N%B0001.gif&type=thumb)
 En este rango de resolución: 

	- el control se visualiza en el modo "Extendido",

	- el botón "Hamburguesa" que sirve para ampliar la zona central de la barra de navegación está oculto. 

	- aparece un menú desplegable. Este menú se puede borrar o personalizar según sus necesidades.




- En las rebanadas más pequeñas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Creation%20-%20HC%20N%B0002.gif&type=thumb)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Creation%20-%20HC%20N%B0003.gif&type=thumb)
 En estas rebanadas: 

	- el control Barra de navegación se anula para ser mostrado en modo "Compacto".. 

	- se visualiza el botón "Hamburguesa" que sirve para ampliar la zona central de la barra de navegación. La acción definida para este control es "Contraer/Expandir la barra de navegación".. Esta acción se utiliza para mostrar la sección central en formato popup.







Para mostrar las características de control, seleccione "Descripción" de el menú contextual de control.

### Selección del modo de visualización
<a name="seleccion_del_modo_visualizacion_ELTPARAGRAPHE000090"></a>

Para configurar el modo de visualización de control Barra de navegación: 

1. Mostrar la ventana de descripción de la control Barra de navegación. 

2. En la pestaña "UI" configure el modo de visualización del control: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Options%20-%20HC%20N%B0001.gif)


3. Validar. 




Observación: El modo de visualización puede ser anulado para cada rango de resolución. Por defecto: 

- Cuando se crea una control Barra de navegación con una hamburguesa, se anulan todas las rebanadas más pequeñas.

- Cuando se crea un control Barra de navegación blanco, los segmentos más pequeños no se anulan.

### El ancho de los paneles laterales
<a name="ancho_los_paneles_laterales_ELTPARAGRAPHE000111"></a>

El tamaño de los cristales laterales se utiliza para definir el comportamiento de las dos secciones exteriores de la control Barra de navegación. Tiene la posibilidad de utilizar una de las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Options%20-%20HC%20N%B0002.gif)


- No hay ancho mínimo: Si se elige esta opción, los 3 paneles mantienen siempre un tamaño proporcional. 

- Anchura mínima de edición: Si esta opción está activada, el tamaño de la rango de resolución define el umbral a partir del cual los paneles laterales ya no se reducen. A partir de este nivel, sólo se reduce el panel central. 

- Ancho fijo por rango de resolución: Si se elige esta opción, el ancho de los paneles izquierdo y derecho no cambia en la ventana control Barra de navegación. Sólo el panel central se agranda o se reduce.  




**RS**: El tamaño de los paneles se puede definir para cada rango de resolución. En este caso, la opción elegida para el ancho de los cristales se aplica al nuevo tamaño de los mismos.. 

**Tip**: Escondiendo un panel lateral y seleccionando "Ancho fijo por rango de resolución", el control Barra de navegación puede utilizarse para crear un diseño que incluya una sección fija (para un banner publicitario, por ejemplo) y una sección fluida.

Al crear la control Barra de navegación (en blanco o con Burger), la opción seleccionada es "Anchura mínima de edición" por defecto.

## Operaciones en el editor
<a name="operaciones_editor_ELTTEXTE000360"></a>


### Manejo del control en el editor
<a name="manejo_del_control_editor_ELTPARAGRAPHE000143"></a>

El control Barra de navegación es un control de envases cuyo aspecto depende del modo de visualización utilizado.. 

En el editor, la control Barra de navegación se divide en 3 paneles: izquierda (1), centro (2) y derecha (3). El tamaño y la visualización de los paneles dependen del modo de visualización de control Barra de navegación. 

- En modo "Extendido: Los 3 paneles se muestran en el editor. Los controles de <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Creation%20-%20HC%20N%B0001%202.gif)
 T pueden ser posicionados en los diferentes paneles (logo, menú, enlaces, ...).  

- En modo "Compacto: El panel central (2) ya no se muestra (está sombreado en el editor). Los controles se encuentran en el área ampliada. 
	Para ver y editar esta área expandida en el editor, sólo tienes que hacer clic en el área sombreada (también tienes la posibilidad de seleccionar "Editar el panel colapsado" desde el el menú contextual del control). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Edition%20-%20HC%20N%B0001.gif&type=thumb)
 TEl área expandida aparece en edición (encerrada en un borde amarillo). Puede realizar las modificaciones solicitadas. Para salir del modo de edición, pulse la tecla Esc.. 




Observaciones: 

- Se puede utilizar cualquier tipo de control en los cristales. 

- Los paneles no pueden ser manejados por programación. 



<a name="NOTE4_2"></a>


### Personalizando los trozos
<a name="personalizando_los_trozos_ELTPARAGRAPHE000166"></a>

Los siguientes elementos de la control Barra de navegación pueden ser personalizados para cada rango de resolución: 

- la posición de los mandos en los cristales.

- la posición de los controles en el panel central (modo plegado o expandido). 

- el modo de visualización del control (extendido o compacto). 

- el tamaño de los cristales. 




También puedes esconder un cristal en un rango de resolución.

<a name="visualizacion_del_area_desglosada_tiempo_ejecucion_ELTTEXTE000390"></a>
Se pueden utilizar dos métodos para mostrar el área central de la barra de modificación en tiempo de ejecución: 

- [Utilizar una acción predefinida](#NOTE5_2). 

- [Use la propiedad ..Collapsed programando](#NOTE5_3). 



<a name="NOTE5_2"></a>


### Uso de una acción predefinida
<a name="uso_una_accion_predefinida_ELTPARAGRAPHE000194"></a>

La acción predefinida "Contraer/Expandir la barra de navegación" se utiliza para mostrar el área central de la barra de navegación en una ventana emergente.. Esta acción está disponible en los controles de botones y enlaces, por ejemplo. Para usar esta acción: 

1. Mostrar la ventana de descripción del Botón o control Enlace. 

2. En la pestaña "General", en el área "Acción de los botones", expanda "Tipo" y haga clic en "Más opciones".. Seleccione "Hacer una acción en otro control". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Action%20-%20HC%20N%B0001.gif)


3. En el área de "Control", seleccione "Colapsar/Expandir la barra de navegación". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_navigation_Action%20-%20HC%20N%B0002.gif)


4. Validar la ventana de descripción del control.




Observaciones:  

- El control asociado a la acción predefinida "Contraer/Expandir la barra de navegación" no se encuentra necesariamente en la barra de navegación.. 

- Por defecto, control Barra de navegación with Burger utiliza un botón "Hamburger" para ampliar la barra de navegación.. 



<a name="NOTE5_3"></a>


### Usando la propiedad ..Collapsed por programación
<a name="usando_propiedad_collapsed_por_programacion_ELTPARAGRAPHE000213"></a>

El [...colapsó...](../Proprietes/2513014.md) usado en el código del navegador en el control Barra de navegación se usa para colapsar o expandir el área media de la barra de navegación. 
Tienes la capacidad de usar [...colapsó...](../Proprietes/2513014.md) en el código de rotación de un control, por ejemplo.

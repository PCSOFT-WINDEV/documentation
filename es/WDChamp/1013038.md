


## Manejo de una control TreeView mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000138"></a>
WINDEV y WEBDEV permiten manipular a control TreeView mediante programación. Para hacerlo, usa la Variable de la control TreeView en el código.

El Variable del control TreeView corresponde al nombre del control TreeView.

Esta ayuda Page explica cómo manipular a control TreeView a través de la programación. 

Observaciones:

- Varias funciones de WLanguage son específicas para la gestión de los controles de TreeView. Para obtener más información, consulte [las funciones de gestión de los controles TreeView](../WDLang1/3018025.md).

- También tiene la posibilidad de manipular un elemento encontrado en una control TreeView a través de la programación. Para obtener más información, consulte [Manipulación de un elemento de una control TreeView mediante programación](../WDChamp/1013040.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Un control TreeView también puede ser relacionado a un archivo de datos o Variable.

## Inicialización de una control TreeView
<a name="inicializacion_una_control_treeview_ELTTEXTE000162"></a>


### Rellenar una control TreeView
<a name="rellenar_una_control_treeview_ELTPARAGRAPHE000042"></a>

Para inicializar una control TreeView, basta con describir todos los elementos del árbol con [TreeAdd](../WDLang1/3018012.md).

```wl
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert", "CAKE-.BMP", "CAKE-.BMP")
TreeAdd(TREE_TVRecipe ,"Recipe" + TAB + "Dessert" + TAB + "Ice cream")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Ice cream" + TAB +  ...
		"Vanilla", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Ice cream" + TAB +  ...
		"Strawberry", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Ice cream" + TAB +  ...
		"Chocolate", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Cup")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Cup" + TAB + ...
		"Peach Melba", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Cup" + TAB + ...
		"Cherry Garcia", ...
	"NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Cup" + TAB + ...
		"Colonel", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Apple pie")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Apple pie" + TAB + ...
	"with whipped cream", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Apple pie" + TAB + ...
	"with apple brandy", "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Crème brûlée", ...
	 "NEXT5-.BMP")
TreeAdd(TREE_TVRecipe, "Recipe" + TAB + "Dessert" + TAB + "Meringue with cream", ...
	"NEXT5-.BMP")
TreeExpand(TREE_TVRecipe, "Recipe")
```

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: Para rellenar correctamente un control TreeView, puede utilizar el [AddChildDelayed](../Proprietes/1000020927.md) Property en las filas del control TreeView. Esta propiedad se utiliza para definir la Procedure que se ejecutará para llenar el nivel para expandir. La búsqueda de niños y el relleno de niños se realiza sólo cuando el usuario quiere verlos..

## Propiedades de WLanguage associated con los controles de TreeView
<a name="propiedades_wlanguage_associated_con_los_controles_treeview_ELTTEXTE000186"></a>
Varias propiedades de WLanguage pueden ser usadas en el control TreeView. Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control TreeView, consulte [Propiedades associated con controles TreeView](../WDChamp/1013028.md).
<a name="NOTE3_2"></a>


### Uso de las propiedades en una control TreeView
<a name="uso_las_propiedades_una_control_treeview_ELTPARAGRAPHE000083"></a>

Ejemplo: Cómo conocer el número de elementos de una control TreeView:

```wl
Trace(TREE_MyTreeView.Count)
```

Ejemplo: Cómo modificar el tamaño de la fuente para todos los elementos de una control TreeView:

```wl
TREE_MyTreeView.FontSize = 14
```

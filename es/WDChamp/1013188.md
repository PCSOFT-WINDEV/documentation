## Visión general de la control Cámara
<a name="vision_general_control_camara_ELTTEXTE000172"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000011"></a>

El control Cámara permite: 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) ver el vídeo capturado por una cámara conectada al ordenador actual.




Puede guardar el video o una foto del mismo.

**Esta página de ayuda presenta la utilización de control Cámara en una ventana WINDEV. Para WINDEV Mobile, véase [control Cámara (aplicación móvil)](../WDChamp/1013349.md).**




### Creación de una control Cámara en una ventana WINDEV
<a name="creacion_una_control_camara_una_ventana_windev_ELTPARAGRAPHE000055"></a>

Para crear un control Cámara:

1. En la pestaña "Creación", en el grupo "Controles gráficos", amplíe "Video y captura" y seleccione "Cámara".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

El control Cámara puede manipularse mediante programación usando las [funciones Camera](../WDLang1/1410086597.md). 
Observación: Estas funciones también están disponibles con una sintaxis con prefijo. Para obtener más información, consulte [Funciones Cámara (Sintaxis con prefijo)](../WDLang1/1000022386.md).

**Observaciones**: 

- El control Cámara funciona con todas las versiones de Windows, independientemente del controlador instalado, a través de la tecnología DirectShow. El comportamiento de las versiones anteriores se puede restaurar seleccionando la opción "Use the former 'Video for Windows' APIs (compatibility)".. 
	Observación: Un control Cámara que utiliza la tecnología DirectShow y un control Cámara compatible no deben utilizarse en la misma ventana.

- Para recuperar una imagen transmitida por una cámara, especifique el controlador de vídeo utilizado (pestaña "General" de la ventana de control de description). Este controlador puede definirse mediante programación con la función [Controlador](../Proprietes/2510061.md) Property o [CameraChangeDriver](../WDLang1/1000022451.md).

## Visualización de una secuencia de vídeo
<a name="visualizacion_una_secuencia_video_ELTTEXTE000202"></a>
Solo se puede ver el video que se está capturando en un control Cámara.

Para ver:

- una imagen capturada con la función [CameraCapture](../WDLang1/1000022450.md), utilice un [control Imagen](../WDChamp/1013218.md).

- un video capturado con la función [CameraCapture](../WDLang1/1000022450.md), utilice un [Control Multimedia](../WDChamp/9500200.md).

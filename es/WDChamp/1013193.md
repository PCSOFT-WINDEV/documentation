## Asociar un control a un panel de pestañas
<a name="asociar_control_panel_pestanas_ELTTEXTE000124"></a>
Para asociar un control a un panel de pestañas: 

- Método 1: Ubique el control en el panel de pestañas deseado. El control se asocia automáticamente a la pestaña actual.

- Método 2: 

	- Abra el menú contextual del control. 

	- Seleccione "Asociar ... a una pestaña". 

	- En la ventana que aparece, seleccione la pestaña asociada al control. 

	- Valide. 







**Observaciones**: 

- Para asociar un control a una pestaña sin ubicarlo en esta pestaña, elija el segundo método. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Un control puede estar asociado a varias pestañas. Simplemente utilice el segundo método y seleccione las pestañas deseadas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NV_Affecter_onglet.gif)


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta página de ayuda solo presenta pestañas estáticas.

## No asociar un control a un panel de pestañas
<a name="asociar_control_panel_pestanas_ELTTEXTE000148"></a>
Un control ubicado en un panel de pestañas no tiene que estar necesariamente asociado a él. A veces, es posible que desee que aparezca una control en el panel de pestañas independientemente de la pestaña seleccionada.

Para no asociar una control a un panel de pestañas:: 

1. Seleccione la opción "Asociar .. a una pestaña" en el menú contextual del control.

2. Seleccione "Ninguno".

## Funciones y propiedades WLanguage específicas de la gestión de pestañas
<a name="funciones_propiedades_wlanguage_especificas_gestion_pestanas_ELTTEXTE000172"></a>




### Nombre del control Pestaña asociado a un control
<a name="nombre_del_control_pestana_asociado_control_ELTPARAGRAPHE000053"></a>

Para obtener el nombre del control Pestaña asociado a un control, utilice [ControlTab](../WDLang1/3025028.md). 

Ejemplo: 

```wl
// EDT_Name1 is on the TAB_TabInfo control
Info(ControlTab(EDT_Name1))     // Returns "TAB_TabInfo"
```

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Paneles de pestañas asociados a un control
<a name="paneles_pestanas_asociados_control_ELTPARAGRAPHE000068"></a>

Para obtener y cambiar el o los paneles de pestañas asociados a un control, utilice la propiedad [Pane](../Proprietes/1000021098.md). 

Ejemplo: 


```wl
// Associate the Validate button to panes 1, 2 and 4
BTN_Validate.Pane = "1,2,4"

// Find out the panes associated with a button
IF BTN_Validate.Pane [=] "2" THEN Info("The control is associated with pane 2")
```

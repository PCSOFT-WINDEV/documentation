
## Manejo de una página interna control mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
La página interna control se utiliza para mostrar una página interna existente. Se pueden realizar varias operaciones en la página interna control a través de la programación.

**Atención: No se deben confundir los términos "Control Página interna" con "Página interna": la "página interna" control es un tipo de control mientras que la "página interna" es un tipo de página.**

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Programación en WLanguage
<a name="programacion_wlanguage_ELTTEXTE000100"></a>


### Pasar parámetros a una página interna
<a name="pasar_parametros_una_pagina_interna_ELTPARAGRAPHE000020"></a>

Para pasar parámetros a una página interna, le aconsejamos que lo haga:

1. Crear una inicialización Procedure en la página interna.

2. Llama a esto Procedure en la página que contiene la "Página Interna" control.




Ejemplo:


```wl
// -- PAGE_Page1 initialization code
// MyInternalPage is a PAGE_Page1 control
// that is linked to an internal page 
// This internal page contains a 
// procedure Init()
MyInternalPage.Init(Param1, Param2)
```

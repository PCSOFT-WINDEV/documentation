## Presentación
<a name="presentacion_ELTTEXTE000282"></a>
WINDEV le permite personalizar:

- las ventanas y los controles que se encuentran en estas ventanas usando el tema de Windows XP/Vista.

- los menús (menús principales o emergentes) de las ventanas que utilizan el estilo XP/Vista (característica disponible incluso si el equipo actual no ejecuta Windows XP/Vista).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Usando el tema de Windows XP/Vista
<a name="usando_tema_windows_xpvista_ELTTEXTE000306"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000024"></a>

El tema de Windows XP/Vista corresponde a las características de la interfaz en un ordenador con Windows XP/Vista. Este tema se define en el panel de configuración de Windows.

WINDEV le permite forzar el uso del tema Windows XP/Vista en las ventanas y en los controles que se encuentran en estas ventanas.. En este caso, tan pronto como la aplicación se abra en un ordenador con Windows XP/Vista, se aplicará el tema de Windows XP/Vista definido en este ordenador.

**Observación**: Si la aplicación se abre en un ordenador que utiliza otro sistema operativo, las características de estilo (definidas en la pestaña "Estilo" de las ventanas de description) se aplicarán automáticamente.
<a name="NOTE2_2"></a>


### Forzar el uso del tema Windows XP/Vista en una ventana
<a name="forzar_uso_del_tema_windows_xpvista_una_ventana_ELTPARAGRAPHE000035"></a>

Para forzar el uso del tema Windows XP/Vista en una ventana:

1. Abra la ventana correspondiente en WINDEV (del explorador del proyecto, por ejemplo).

2. Mostrar el panel "Modificador": en la pestaña "Inicio", en el grupo "Entorno", despliegue "Paneles", seleccione "Paneles" y, a continuación, haga clic en "Modificador".

3. En el modificador, en la categoría "Estilo", cambia la opción "Tema XP" a "Sí".. 
	En la pestaña "Estilo" de la ventana de description de la ventana actual, se muestra la opción "Forzar el tema Active en Windows XP/Vista" y se marca automáticamente.




**Observaciones**:

- Las ventanas que utilizan el tema de Windows XP/Vista mantienen sus características de estilo (definidas en la pestaña "Estilo" de la ventana de description)

- Si se usa un fondo Image, este Image se mantiene. De lo contrario, se aplica el correspondiente tema XP/Vista (fondo Color de la ventana).



<a name="NOTE2_3"></a>


### Forzar el uso del tema de Windows XP/Vista en una ventana control
<a name="forzar_uso_del_tema_windows_xpvista_una_ventana_control_ELTPARAGRAPHE000065"></a>

Para forzar el uso del tema de Windows XP/Vista en una ventana control: 

1. Forzar el uso del tema de Windows XP en la ventana a la que pertenece el control (véase el párrafo Above).

2. Seleccione la control deseada.

3. Mostrar la pestaña "Estilo" en la ventana de description de este control ("description" de el menú contextual).

4. Seleccione "XP/Vista theme" en la lista desplegable "Element".

5. Marque "Usar el tema Active en Windows XP/Vista".




Observación: Si el tema no fue forzado a nivel de la ventana, la marca de verificación no se encontrará a nivel de control. 
<a name="NOTE2_4"></a>

## Uso del estilo XP/Vista
<a name="uso_del_estilo_xpvista_ELTTEXTE000348"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000196"></a>

WINDEV le permite usar el estilo XP/Vista en los menús (menús principales y/o emergentes) que se encuentran en las ventanas.. Para ver los menús al estilo XP/Vista, el ordenador actual no tiene que ejecutar necesariamente Windows XP/Vista.

### Uso del estilo XP/Vista en el menú principal de una ventana
<a name="uso_del_estilo_xpvista_menu_principal_una_ventana_ELTPARAGRAPHE000214"></a>

Para que el menú principal de una ventana utilice el estilo XP/Vista: 

1. Abra la ventana solicitada en WINDEV.

2. Inserte el menú principal si es necesario: en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Menú principal" y seleccione "Agregar menú principal".

3. Mostrar la pestaña "Estilo" en la ventana de description de la ventana actual ("description" de el menú contextual).

4. Marque "Mostrar el menú en el estilo XP/Vista".




**Observación**: Los menús que usan el estilo XP mantienen las características de la fuente usada para las opciones del menú (tamaño, Color, ...). Estas características han sido definidas en la pestaña "Estilo" de la ventana de description de la opción de menú.

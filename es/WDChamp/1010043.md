
## Programación de un menú desplegable WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
Una vez creado el menú desplegable de su sitio, todo lo que tiene que hacer es asociar una acción a cada opción del menú.

Se puede programar las opciones del menú:

- En el editor de la página, en la opción description.

- En el editor de código, programando en WLanguage o en JavaScript.




También se pueden utilizar varias funciones de WLanguage para programar las opciones del menú manipular. Para obtener más información, consulte [Funciones de gestión de menús](../WDLang1/3038016.md). 

Observación: Usted tiene la capacidad de programar solamente las opciones finales del menú.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Editor de páginas: definiendo las acciones associated con las opciones del menú
<a name="editor_paginas_definiendo_las_acciones_associated_con_las_opciones_del_menu_ELTTEXTE000175"></a>
La acción asociada a la opción de menú se define en la ventana description de la opción de menú.

**Recordatorio**: Para mostrar la ventana de description de una opción de menú:

1. Haga doble clic en el menú: un marco amarillo aparece alrededor.

2. Seleccione la opción de menú deseada.

3. Abra el menú contextual de la opción y seleccione "Opción description".




Para obtener más información, consulte [Eligiendo la acción associated con una opción de menú](../WDChamp/1010044.md).

## Editor de código: definiendo las acciones associated con las opciones del menú
<a name="editor_codigo_definiendo_las_acciones_associated_con_las_opciones_del_menu_ELTTEXTE000199"></a>


### Mostrar los eventos associated con una opción de menú
<a name="mostrar_los_eventos_associated_con_una_opcion_menu_ELTPARAGRAPHE000048"></a>

Para mostrar los eventos associated con una opción de menú:

1. Haga doble clic en el menú: un marco amarillo aparece alrededor.

2. Seleccione la opción de menú deseada.

3. Abra el menú contextual de la opción y seleccione "Código".




Hay dos eventos disponibles:

- Seleccionando el menú en código de navegador.

- Seleccionando el menú en código de servidor (disponible sólo para las páginas dinámicas).




En estos casos, las funciones de WLanguage pueden ser utilizadas para:

- Mostrar una Page ([PageDisplay](../WDLang2/3058008.md)).

- Etc.




Observación: Estos eventos están disponibles: 

- para las opciones finales del menú y los submenús. 

- para las opciones principales del menú (las que expanden los submenús). 



<a name="NOTE3_2"></a>


### Eventos associated de Default con opciones de menú
<a name="eventos_associated_default_con_opciones_menu_ELTPARAGRAPHE000079"></a>

WEBDEV gestiona los siguientes eventos de forma predeterminada (orden de aparición en el editor de código):

| Event | Condición de ejecución |
| --- | --- |
| Selección de la opción (código del navegador) | Ejecutar cuando el usuario Web selecciona la opción. |
| Selección de la opción (Código de servidor) | Ejecutar cuando el usuario Web selecciona la opción. |

Observación: Solo los códigos Navegador están disponibles en una página estática.
<a name="NOTE3_3"></a>


### Programación dinámica de opciones de menú
<a name="programacion_dinamica_opciones_menu_ELTPARAGRAPHE000101"></a>Las siguientes funciones de WLanguage se utilizan para manipular dinámicamente las opciones de menú de un Page:



|   |   |
| --- | --- |
| [MenuAddMenu](../WDLang1/1000017266.md) | Agrega un nuevo menú a un menú desplegable o contextual de una ventana o página. |
| [MenuAddOption](../WDLang1/1000017262.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. |
| [MenuAddPopup](../WDLang1/1000020583.md) | Transforma una opción de menú de una Page para que esta opción abra una ventana emergente. |
| [MenuAddSeparator](../WDLang1/1000017270.md) | Añade un nuevo separador en un menú (desplegable o contextual). |
| [MenuAddURLOption](../WDLang1/1000018529.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú abre la página que corresponde a la URL especificada. |
| [MenuDelete](../WDLang1/3038058.md) | Borra un menú o una opción de menú. |

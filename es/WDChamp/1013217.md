
## Interacciones entre una animación Flash® y WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
Una animación Flash® puede utilizarse para mostrar información en los controles de una página WEBDEV.

Se requiere una simple modificación de la animación Flash®.


	<B><B><B><B>Advertencia</B></B></B></B>

	 Flash® ya no es compatible con los navegadores desde el 01/01/2021.


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000120"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Modificación a realizar en la animación Flash
<a name="modificacion_realizar_animacion_flash_ELTPARAGRAPHE000024"></a>

1. Añade un Variable global. Este Variable global contendrá el nombre del WEBDEV control a modificar.

2. Modificar el código "OnRelease" para asignar el control según el nombre de la Variable.

3. Publicar la animación Flash®.



<a name="NOTE2_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Modificación a realizar en la página WEBDEV
<a name="modificacion_realizar_pagina_webdev_ELTPARAGRAPHE000035"></a>

1. Asociar la animación Flash® con la control Flash. Este control debe ser dinámico.

2. En la pestaña "Detalles" de la ventana de description del Flash control, marque "control accesible desde Javascript".

3. Inserte el siguiente código en el código de carga de la página:





```wl
// Retrieves the alias of FLASH_FlashControl
sFlash is string = FLASH_FlashControl..Alias
// Retrieves the alias of the edit control
sEdit is string = EDT_EditControl..Alias
// Initializes the variables of the Flash animation
InitFlash(sFlash, sEdit)
```


El **InitFlash** Procedure es un Procedure en JavaScript (y no en código de lenguaje WLanguage). El código del procedimiento es el siguiente:


```wl
function InitFlash(sFlash, sEdit)
{
// call SetVariable on the Flash control
// document.<Page Name>.<Name of Flash Control>.SetVariable(<Name of Flash Variable>, 
// <Name of Control Alias );
eval("document.THEPAGE." + sFlash).SetVariable("gsControlName", sEdit);
}
```




## Menú contextual de automatic para los controles del Programador
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000242"></a>
Cada control Planificador que se encuentra en una ventana propone un menú al usuario.

Este menú, disponible sin programación, incluye varias acciones:

- Cancel o rehacer una acción.

- Selección del modo de visualización del programa: Día, Semana Completa, Semana Laboral, Mes.

- Mostrando los elementos de hoy.

- Gestión de las citas: Adición, eliminación, modificación.

- Copiando el contenido de Appointment.

- Imprimir el programa.

- Envío del programa por Email. 




Las acciones realizadas en la control Planificador pueden ser: 

- deshacer mediante "Deshacer" en el menú contextual o mediante Ctrl + Z. 

- rehacer mediante "Rehacer" en el menú contextual o mediante Ctrl + Y.


Estas acciones corresponden principalmente al movimiento, cambio de tamaño, eliminación, ... de citas.

**Observación**: Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

## Ajustes
<a name="ajustes_ELTTEXTE000266"></a>


### Configurar el menú en una control Planificador
<a name="configurar_menu_una_control_planificador_ELTPARAGRAPHE000046"></a>

El menú contextual de control Planificador puede desactivarse. Para configurar la visualización del menú contextual associated con el control Planificador, debe:

1. Abra la ventana de control Planificador description.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, aparecerá el menú automatic de control Planificador. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.

- A través de la programación, [AAFDisable](../WDLang1/1000022018.md) permite eliminar algunas opciones del menú contextual del AAF específicas de los controles del Programador. Se pueden utilizar diferentes constantes según la opción de deshabilitar: 

	- *aafPrintSchedule*: se usa para borrar la opción de imprimir el horario. Atención, esta constante también puede borrar la opción de impresión propuesta por los controles del Organizador..  

	- *aafSendFileByEmail*: utilizado para eliminar la opción de enviar el programa por Email. Atención, esta constante también puede borrar la opción de envío de Email propuesta por los controles del Organizador y por los controles de edición del Camino.

### "Opción"Imprimir
<a name="opcionimprimir_ELTPARAGRAPHE000098"></a>

La opción "Imprimir" se utiliza para imprimir el contenido de la control Planificador. Aparece una ventana específica que le permite configurar la impresión. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Imp_Planning.gif)


Esta ventana permite configurar: 

- la orientación de reporte.

- el tiempo braket per Page: día, semana, mes, ....

- el período a imprimir (set de fechas). 




**Observación**: Dependiendo de las opciones especificadas para la franja horaria de Page s' y el período, la impresión del planificador puede requerir más de un Page. Por ejemplo, un horario impreso diariamente durante una semana imprimirá 1 Page al día. 

Con la tecla "Imprimir" se inicia la impresión. 

El contenido de la agenda control se visualiza en el visor de reporte.

Independientemente de la orientación seleccionada, el contenido del Horario control se imprime completamente entre los límites especificados.   

Las características de impresión son las siguientes:

- El título de la impresión corresponde a la leyenda de control Planificador. Si no se ha especificado ningún título para la control Planificador, reporte no muestra ningún título.

- La impresión respeta los colores utilizados en la control Planificador.

- La fecha de hoy se imprime en la parte inferior izquierda.

- El número de la Page actual y el número total de Page s se imprimen en la parte inferior derecha Corner.

- El documento impreso tiene en cuenta la dirección del texto especificada con el [TextDirection](../Proprietes/2513020.md) Property en la ventana y/o en el control Planificador.




**Observaciones**: 

- Las librerías de impresión de WINDEV framework deben ser proporcionadas con el ejecutable para utilizar la opción "Imprimir" en el menú contextual de los controles del Programador.

- Para desactivar la opción "Imprimir" en el menú contextual del control Planificador, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante ***aafPrintSchedule***.

- Para personalizar la reporte asociada a una control Planificador específica, debe crear un nuevo Scheduler reporte basado en esta control Planificador. Se associated automáticamente con el control Planificador y se imprime a través de la opción "Imprimir".

### Personalizar la ventana de configuración para imprimir el planificador (disponible a partir de la versión 170078)
<a name="personalizar_ventana_configuracion_para_imprimir_planificador_disponible_partir_version_170078_ELTPARAGRAPHE000140"></a>

WINDEV le permite incluir la ventana de configuración para imprimir el planificador utilizado por la AAF en su aplicación.. Usted tendrá la posibilidad de personalizar:

- el aspecto de esta ventana (por ejemplo, modificación de plantilla)

- los idiomas soportados por esta ventana. Esta característica es muy useful a la hora de traducir esta ventana a los diferentes idiomas disponibles en la aplicación.  




**Para incluir la ventana de configuración de la impresión del calendario en su aplicación**: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccione el subdirectorio que contiene los elementos a importar. La ventana de ajustes de impresión se encuentra en el subdirectorio "Programas\\Datos\\Ventanas de preajuste\\AAF" del directorio WINDEV instalación.

3. WINDEV hace una lista de los elementos del directorio. Este directorio contiene los elementos que corresponden a las AAF personalizables. La ventana para importar se llama "WinDevAaf_ConfigPrintingSchedulerOrganizer.wdw".

4. Validar. La ventana está incluida en el proyecto y se puede modificar. Esta ventana personalizada se tendrá en cuenta automáticamente al ejecutar la aplicación.




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario. 

- Si está utilizando un programador control en su proyecto, el AAF de impresión del control Planificador también utilizará esta ventana.

### Opción "Enviar por correo electrónico"
<a name="opcion_enviar_por_correo_electronico_ELTPARAGRAPHE000178"></a>

La opción "Enviar por Email" se utiliza para: 

- imprimir el contenido de la control Planificador en un archivo "Schedule.pdf. 

- crear un nuevo mensaje en el software de mensajería configurado por Default y adjuntar el archivo "Schedule.pdf" generado a este mensaje. 




La Procedure para imprimir el organizador en el archivo PDF es idéntica a la impresión que la Procedure realiza a través de la opción "Imprimir el horario" (ver el párrafo anterior). 

El Planificador reporte generado no se visualiza en el visor de reporte, sino que se imprime directamente en un archivo en formato PDF.. 

**Observaciones**: 

- Las librerías de impresión y de generación de archivos PDF de WINDEV framework deben ser proporcionadas con el ejecutable para utilizar la opción "Imprimir" en el menú contextual de los controles del Programador.

- Para desactivar la opción "Enviar por correo electrónico" en el menú contextual de los controles del Programador, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante ***aafSendFileByEmail***. Atención, esta constante también puede borrar la opción de envío de Email propuesta por los controles del Organizador y por los controles de edición del Camino.

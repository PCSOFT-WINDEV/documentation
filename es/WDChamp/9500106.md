


## AAF: Aumentar la altura de las filas en los controles Table o Looper
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
En los controles de la tabla, el usuario puede cambiar el tamaño de la altura de las filas, en lectura o en edición.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Redim-hauteur-ligne.gif)


Esta función también está disponible: 

- en los controles del Looper.

- en los controles de la tabla TreeView. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000126"></a>


### Permitiendo al usuario cambiar el tamaño de la altura de las filas en una Tabla (o Tabla de Vista de Árbol) control
<a name="permitiendo_usuario_cambiar_tamano_altura_las_filas_una_tabla_tabla_vista_arbol_control_ELTPARAGRAPHE000024"></a>

Para permitir al usuario cambiar el tamaño de la altura de las filas en una Tabla (o Tabla de Vista de Árbol) control: 

1. Abra el control description.

2. En la pestaña "Detalles" del control description, marque "Alto de las filas redimensionable".

3. Valide.



<a name="NOTE2_2"></a>


### Permitir al usuario redimensionar la altura de las filas en una control Looper
<a name="permitir_usuario_redimensionar_altura_las_filas_una_control_looper_ELTPARAGRAPHE000039"></a>

Para permitir al usuario redimensionar la altura de las filas en una control Looper: 

1. Mostrar la description de control Looper.

2. En la pestaña "Detalles" de la control Looper description, marque "Redimensionable" para la altura de la fila.

3. Valide.



<a name="NOTE2_3"></a>


### Desactivar la capacidad de cambiar el tamaño de las filas
<a name="desactivar_capacidad_cambiar_tamano_las_filas_ELTPARAGRAPHE000050"></a>

Para no permitir al usuario redimensionar la altura de las filas en un control Tabla, Tabla TreeView o Looper, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafResizeRowHeight*.


## FAA: Menú contextual de los controles de Image
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000180"></a>
Un menú contextual está disponible para los controles de Image. Este menú se utiliza para: 

- modificar la Image que se encuentra en el control (a través de un mini-bar o un editor de Image).

- copy, guardar y mostrar el Image seleccionado en modo de alta calidad. 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_MenuContextuelImage%20-%20HC%20N%B0003.gif)


Observación: Algunas opciones de menú pueden ir seguidas de una estrella o un candado para indicar que se requiere una contraseña para realizar la opción solicitada.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000204"></a>


### Permitir a los usuarios abrir un menú contextual en las imágenes
<a name="permitir_los_usuarios_abrir_menu_contextual_las_imagenes_ELTPARAGRAPHE000030"></a>

Para permitir a los usuarios abrir un menú contextual en las imágenes:

1. Abrir la description de la control Imagen.

2. En la pestaña "Detalles", seleccione la opción "Esta imagen es un área de clic".

3. En la opción "Opciones de la imagen (para el usuario)", seleccione el tipo de menú contextual que desea visualizar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_MenuContextuelImage%20-%20HC%20N%B0001.gif&type=thumb)


	- "No menu": No se mostrará ningún menú contextual para la control. 

- "Menú simple (copiar, guardar como, etc.)":  El menú contextual permite copiar, guardar y visualizar la Image seleccionada en modo de alta calidad.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_MenuContextuelImage%20-%20HC%20N%B0002.gif)

4. Valide.



<a name="NOTE2_2"></a>


### Configuración del menú contextual de Image
<a name="configuracion_del_menu_contextual_image_ELTPARAGRAPHE000061"></a>

Si se ha activado el menú contextual AAF simple para el control Imagen, puede configurarlo. Para configurar el menú contextual del control Imagen, debe:

1. Mostrar la ventana de description de control Imagen.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contextual". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, el menú de automatic de control Imagen se mostrará. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú FAA](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las FAA. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las FAA y restaurar el menú estándar de Windows para un control determinado. 

- Si se ha asignado un contenido binario al Image (a través de [fLoadText](../WDLang1/3036006.md) o de una nota binaria, por ejemplo), la opción "Alta calidad" no estará disponible en el menú contextual.

- La opción "Editar Image" se configura a través de la pestaña "Detalles" de control Imagen.

## Programación
<a name="programacion_ELTTEXTE000234"></a>


### Texto de las opciones del menú contextual
<a name="texto_las_opciones_del_menu_contextual_ELTPARAGRAPHE000112"></a>

El texto de las opciones del menú contextual AAF (copy, guardar como, etc.) puede modificarse a través de [AAFChangeCaption](../WDLang1/1000022100.md). 
<a name="NOTE3_2"></a>


### Protección de las opciones de exportación
<a name="proteccion_las_opciones_exportacion_ELTPARAGRAPHE000124"></a>

La control Imagen propone opciones de exportación: "Copiar", "Guardar como", "Abrir con el programa Default", etc.

[AAFExportSecurity](../WDLang1/1000022481.md) se utiliza para asegurar estas características: 

- solicitando una contraseña antes de realizar la exportación. En este caso, un candado o una estrella se muestra en el título de la opción. 

- prohibiendo la exportación. En este caso, las opciones correspondientes ya no se mostrarán..




## Opciones para generar las páginas HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
Se pueden definir las opciones para generar las páginas HTML en un proyecto WEBDEV:

- en la pestaña "Avanzado" del proyecto description (en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción"). Las opciones de generación se tienen en cuenta para todas las páginas del proyecto.

- al describir las páginas ("description" en el menú contextual de la página en el editor, pestaña "Avanzado"). Las opciones de generación seleccionadas se tendrán en cuenta sólo para la página actual.

## Opciones para generar el código HTML
<a name="opciones_para_generar_codigo_html_ELTTEXTE000113"></a>
Tienes la capacidad de generar:

- un código HTML compatible con todos los navegadores existentes.

- un código HTML optimizado para los navegadores recientes. En este caso, el botón "Avanzado" se utiliza para especificar las diferentes optimizaciones que se pueden realizar.


**Observación**: Algunas optimizaciones no son seleccionadas por Default porque pueden provocar algunas modificaciones en el código WLanguage existente. Le aconsejamos que realice la prueba de la aplicación después de cada tipo de optimización realizada.

WEBDEV te da la posibilidad de generar una segunda set de páginas que son "Compatible con todos los navegadores". En este caso, todas las páginas del proyecto se generarán automáticamente dos veces. La segunda set de las páginas va precedida del carácter "_".

WEBDEV también le permite codificar las páginas HTML en UTF8.

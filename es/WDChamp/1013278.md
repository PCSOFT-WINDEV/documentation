## Modo de selección
<a name="modo_seleccion_ELTTEXTE000152"></a>
La pestaña "UI" de la ventana de descripción del control List Box permite definir el modo de selección en el control. 

Los modos de selección disponibles son los siguientes: 

- **Selección única**: La banda de selección se muestra en la fila seleccionada.. El código de selección de filas se ejecuta.

- **Selección múltiple**: La banda de selección se muestra en las filas seleccionadas. Para seleccionar varias filas, mantenga pulsada la tecla Mayús o Ctrl. El código de selección de filas se ejecuta cada vez que se selecciona una fila.

## Desplazamiento
<a name="desplazamiento_ELTTEXTE000176"></a>
Las opciones para desplazarse por el control List Box pueden definirse en la pestaña "Detalles" de la ventana de descripción del control. Es posible definir: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **El modo de visualización de la barra de desplazamiento vertical y horizontal**. Puede ser:

	- automático: la barra de desplazamiento se mostrará cuando sea necesario.

	- siempre visible: la barra de desplazamiento siempre se mostrará.

	- nunca visible: la barra de desplazamiento nunca se mostrará.




- **El modo de desplazamiento**: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Desplazamiento por pixel: las filas pueden mostrarse parcialmente según la posición de la barra de desplazamiento.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Desplazamiento táctil: permite manipular la barra de desplazamiento en una pantalla táctil.

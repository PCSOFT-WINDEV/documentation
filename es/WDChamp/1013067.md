
## Vinculación de una control ListView a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000076"></a>
Un control ListView se puede vincular a un elemento en un archivo de datos , consulta, etc.. 

**Por Default, el elemento vinculado al control contendrá el índice del elemento seleccionado en el control ListView.**

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipulación del artículo y de la control ListView
<a name="manipulacion_del_articulo_control_listview_ELTTEXTE000100"></a>


### Para actualizar la control ListView con el elemento que corresponde al valor almacenado en el artículo:
<a name="para_actualizar_control_listview_con_elemento_que_corresponde_valor_almacenado_articulo_ELTPARAGRAPHE000020"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo relacionado) serán actualizados por [FileToScreen](../WDLang4/3044210.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE3_2"></a>


### Para actualizar un elemento con el índice del elemento seleccionado en la control ListView:
<a name="para_actualizar_elemento_con_indice_del_elemento_seleccionado_control_listview_ELTPARAGRAPHE000027"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la ventana serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

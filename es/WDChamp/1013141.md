
## Manejo de una control Barra de progreso mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000228"></a>
Una control Barra de progreso puede ser manejada a través de la programación. Para hacerlo, usa la Variable de la control Barra de progreso en el código.

La Variable de la control Barra de progreso:

- corresponde al nombre de la control Barra de progreso.

- se inicializa con la posición de la barra de progreso.

### Barra de progreso no simétrica (lineal o circular)
<a name="barra_progreso_simetrica_lineal_circular_ELTPARAGRAPHE000025"></a>

El progreso se realiza desde el límite inferior hasta el límite superior.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=JaugeSimple.GIF)



```wl
FOR I = PROGBAR_ProgBar1.MinValue _TO_ PROGBAR_ProgBar1.MaxValue
	PROGBAR_ProgBar1 = I
END
```

### Barra de progreso simétrica
<a name="barra_progreso_simetrica_ELTPARAGRAPHE000037"></a>

El progreso se realiza de la siguiente manera:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=JaugeSymetrique_prog.gif)



```wl
FOR I = 2 * PROGBAR_ProgBar1.MinValue - PROGBAR_ProgBar1.MaxValue _TO_ PROGBAR_ProgBar1.MaxValue
	PROGBAR_ProgBar1 = I
END
```

## Recuperación de la posición de la barra de progreso
<a name="recuperacion_posicion_barra_progreso_ELTTEXTE000288"></a>
Para recuperar la posición de una barra de progreso, utilice la siguiente sintaxis: 


```txt
<Position> = <Progress Bar control>
```


&lt;Posición&gt; es un entero Variable. 

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modificación de la posición de la barra de progreso
<a name="modificacion_posicion_barra_progreso_ELTTEXTE000312"></a>
Para modificar la posición de una barra de progreso, utilice la siguiente sintaxis: 


```txt
<Progress Bar control> = <Value>
```


&lt;Valor&gt; es un valor que se encuentra entre el valor mínimo y el valor máximo de la barra de progreso.. Estos valores se han descrito en la ventana de control description o mediante programación con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

## Propiedades específicas de los controles de la barra de progreso
<a name="propiedades_especificas_los_controles_barra_progreso_ELTTEXTE000336"></a>
Las siguientes propiedades se utilizan para manipular una control Barra de progreso de forma programada.


|   |   |
| --- | --- |
| [BackgroundImage](../Proprietes/2510056.md) | Permite get y cambiar el fondo Image de un control Barra de progreso. |
| [HorizontalAlignment](../Proprietes/2510023.md) | Permite get y cambiar la alineación horizontal del porcentaje de progreso mostrado en el control Barra de progreso. |
| [Image](../Proprietes/2510034.md) | Permite get y cambiar el progreso Image de un control Barra de progreso. |
| [MaxValue](../Proprietes/2510009.md) | Permite get y cambiar el límite superior de un control Barra de progreso. |
| [MinValue](../Proprietes/2510008.md) | Permite get y cambiar el límite inferior de un control Barra de progreso. |
| [TextProgressBar](../Proprietes/1000017246.md) | Permite get y cambiar el texto que se muestra en un control Barra de progreso (en lugar del porcentaje). |
| [Vertical](../Proprietes/2510133.md) | Permite saber si la barra de progreso es horizontal o vertical. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Barra de progreso, consulte [Propiedades associated con controles Progress Bar](../WDChamp/1013137.md).

## Visión general de la control Imagen con enlace
<a name="vision_general_control_imagen_con_enlace_ELTTEXTE000182"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000017"></a>

El control Imagen con enlace se puede utilizar para:

- Crear una imagen con enlace en una página: en este caso, un código de clic servidor se asocia a la imagen.

- get las coordenadas del clic en una página. En este caso, las coordenadas se devuelven al servidor al hacer clic en la imagen.

- Definir las áreas en las que se puede hacer clic/áreas del mapa. Una acción puede ser associated con el Image y con cada área del Image.




Para obtener más información, consulte [Modo de funcionamiento de una imagen con enlace](../WDChamp/1013158.md)

## Crear un control Imagen con enlace
<a name="crear_control_imagen_con_enlace_ELTTEXTE000206"></a>
Para crear un control Imagen con enlace: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", amplíe "Imagen" y haga clic en "Image".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual del control. La información principal es la siguiente: 

- El botón [...] que se encuentra en la pestaña "General" sirve para seleccionar la imagen associated con el control Imagen. En función de la Image seleccionada, aparece una asistente que le permite:

	- **comprime automáticamente el Image** (para los Image cuyo tamaño supera los 20 KB). Esta opción también permite convertir la imagen en uno de los formatos adecuados para las páginas de INTERNET. Cuando esta opción no está seleccionada:
			- Si el Image utiliza un formato no adaptado a las páginas de INTERNET, el Image se convertirá cada vez que se genere la página, por lo que esta generación puede verse ralentizada.
			- Si la Image utiliza un formato adaptado a las páginas pero su tamaño es demasiado grande, el tiempo de carga de la página puede aumentar.
			Si realiza la conversión inmediatamente, puede seleccionar el formato y la calidad de Image. También puedes usar el formato ***progresivo JPEG***. Este formato muestra una imagen progresivamente mientras el navegador carga la página.  

	- **convierte automáticamente la imagen** en uno de los formatos adecuados para las páginas INTERNET. Si no se selecciona esta opción, la Image se convertirá cada vez que se genere la página, por lo que esta generación puede verse ralentizada. Si realiza la conversión inmediatamente, puede seleccionar el formato y la calidad de Image.

	- **definir la ubicación de la imagen**: puede copiar el Image al directorio del sitio (directorio _WEB) para optimizar el tiempo de generación de la página o dejar el Image en su ubicación (recomendado cuando el archivo se comparte entre varios colaboradores).




- **Seleccionar el tipo de imagen** (pestaña "General"): Para más detalles sobre el tipo de Image (estática, dinámica, etc.), véase [Seleccionar el tipo de imagen](../WDChamp/1013132.md).
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Las imágenes dinámicas no son compatibles.

- **Protección de las imágenes** (pestaña"Detalles"): Si esta opción está marcada, el menú contextual de Image no se mostrará en el navegador. Algunas de las acciones relacionadas con la Image (guardar, imprimir, etc.) no estarán disponibles. Esta opción también está disponible en [controles Imagen](../WDChamp/1013218.md) y [Controles en miniatura](../WDChamp/1013275.md).

- **Modo de visualización** (pestaña "General"): Para más información sobre el modo de visualización y transparencia de la imagen, consulte [Seleccionar el modo de visualización de la imagen](../WDChamp/1013134.md).

- **Transición** (pestaña"General"): Cuando se modifica el valor de la control Imagen, se puede seleccionar una transición (por ejemplo, Fade, Cover, Slide, etc.) para que la transición de la Image sea más suave. Para obtener más información, consulte [Animar una imagen](../WDChamp/9500117.md). 

- **automatic scroll** (ficha "General"): Esta opción está disponible para imágenes "dinámicas" o imágenes "de una base de datos".: ruta". Para obtener más información, consulte [Imagen con deslizamiento automático](../WDChamp/9500118.md).

- También puede asociar una acción a la imagen (pestaña "General"). Para obtener más información, consulte [Seleccionar la acción asociada a un control](../WDChamp/1013165.md).

## Sitio multilingüe
<a name="sitio_multilingue_ELTTEXTE000230"></a>
Si su sitio es multilingüe, se puede seleccionar una Image diferente para cada idioma. Esta opción es útil cuando las imágenes contienen texto, por ejemplo.

**Para gestionar imágenes multilingües**: 

1. Abra la description de la control Imagen con enlace.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=mutlti_langue_img.gif). La ventana de gestión de imágenes multilingües se abre. Por defecto, el mismo archivo es associated con todos los idiomas que soporta la página.

3. Para cada idioma, pulse el botón [...] y seleccione el Image correspondiente al idioma. En tiempo de ejecución, se mostrará la imagen correspondiente según el idioma seleccionado con la función [Nación](../WDLang1/3054014.md).
	**Atención**: La función multilingüe solo gestiona los archivos de imagen, no las características de visualización de la imagen.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000103"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten manipular a control Casilla de verificación mediante la programación. Para ello, utilice la Variable de la control Casilla de verificación en el código.

La Variable del control Casilla de verificación corresponde al nombre del control Casilla de verificación. Este Variable es un número entero.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Varias funciones de WLanguage se utilizan para añadir, insertar o borrar opciones en un control Casilla de verificación. Para obtener más información, consulte [Funciones de gestión de control Casilla de verificación](../WDLang1/1000021310.md).

## Inicializar las diferentes opciones de una control Casilla de verificación
<a name="inicializar_las_diferentes_opciones_una_control_casilla_verificacion_ELTTEXTE000138"></a>
Por Default, no se marca ninguna opción cuando se muestra un control Casilla de verificación.

**Para marcar una opción de un control Casilla de verificación**, puede utilizar:

- la asignación directa: 
	
	```txt
	<Check Box control>[Option index] = 1 
	
	<Check Box control>[Option index] = True
	```


- la [Valor](../Proprietes/2510130.md) property: 
	
	```txt
	<Check Box control>[Option index].Value = True
	
	
	<Check Box control>[Option index].Value = 1
	```



**Para desmarcar una opción de un control Casilla de verificación**, puede utilizar:

- la asignación directa: 
	
	```txt
	<Check Box control>[Option index] = 0 
	
	<Check Box control>[Option index] = False
	```


- la [Valor](../Proprietes/2510130.md) property. 
	
	```txt
	<Check Box control>[Option index].Value = 0 
	
	<Check Box control>[Option index].Value = False
	```



**Observación**: Si el control es associated con el elemento un archivo de datos, la asignación se realiza automáticamente mediante: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la función [.fromFile](../WDLang4/1000021387.md).

- Para obtener más información, consulte [Enlace de un control Casilla de verificación a un artículo](../WDChamp/1013152.md).




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Si se admite el estado indeterminado de una opción** (pestaña "Detalles" de la descripción del control o propiedad [TriState](../Proprietes/2510124.md)), para inicializar la opción en el modo indeterminado, utilice la siguiente sintaxis:

```txt
<Check Box control>[Option index] = -1
```

## Recuperar el valor de una opción de una control Casilla de verificación
<a name="recuperar_valor_una_opcion_una_control_casilla_verificacion_ELTTEXTE000219"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### En WINDEV y WINDEV Mobile
<a name="windev_windev_mobile_ELTPARAGRAPHE000244"></a>

Para recuperar el valor de una opción, utilice la siguiente sintaxis:

```txt
<Option_Value> = <Check Box control>[Option index]
```

En este caso, &lt;Valor_de_opción&gt; es un booleano set a:

- True (1) si la casilla está marcada,

- False (0) si la casilla no está marcada.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) -1 si el cuadro no está definido (si este estado está soportado por la opción).




**Observación**: Si el control está vinculado al elemento un archivo de datos, el valor introducido en el control se actualiza automáticamente en la fuente de datos mediante el [.ToFile](../WDLang4/1000021385.md). Para obtener más información, consulte [Enlace de un control Casilla de verificación a un artículo](../WDChamp/1013152.md).<a name="NOTE3_2"></a>

## Modificación de los colores de un control Casilla de verificación fila de fila
<a name="modificacion_los_colores_control_casilla_verificacion_fila_fila_ELTTEXTE000305"></a>
Para modificar los colores de un control Casilla de verificación fila de fila, utilice la siguiente sintaxis:

```txt
<Check Box control>.Caption = gPen(<Color option 1>) + <Caption option 1> + TAB + ...
gPen(<Color of option N>) + <Caption of option N>
```

&lt;Color de la opción&gt; corresponde a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

## Modificación de las características de los subtítulos
<a name="modificacion_las_caracteristicas_los_subtitulos_ELTTEXTE000347"></a>
El [Pie de foto](../Proprietes/2510053.md) Property le permite:

- cambiar el título de la control Casilla de verificación:
	
	```txt
	<Check Box control>.Caption = <Control caption>
	```


- cambiar el título de todas las opciones de la control Casilla de verificación:
	Las opciones deben estar separadas por el carácter TAB. Sólo puede especificar las primeras opciones.
	
	```txt
	<Check Box control>.Caption = <Option 1> + TAB + <Option 2> + TAB + <Option 3> ...
	```


- cambiar el título de una opción:
	
	```txt
	<Check Box control>[Option index].Caption = <New option caption>
	```


- recuperar el título de una opción:
	
	```txt
	<Option caption> = <Check Box control>[Option index].Caption
	```




<a name="NOTE5_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el control Casilla de verificación utiliza un modo de visualización con imágenes y/o subtítulos, usted puede: 

- obtener y fijar la imagen de la opción con la [Image](../Proprietes/2510034.md) property:
	
	```txt
	<Check Box control>[Option index].Image = <New image>
	```


- obtener y establecer el subcapítulo de la opción con el [SubCaption](../Proprietes/1000017232.md) property:
	
	```txt
	<Check Box control>[Option index].SubCaption = <New sub-caption>
	```


- obtener y fijar la altura de la opción con la [Altura](../Proprietes/2510050.md) property: 
	
	```txt
	<Check Box control>[Option index].Height = <New height in pixels>
	```

## Propiedades específicas de los controles de las casillas de verificación
<a name="propiedades_especificas_los_controles_las_casillas_verificacion_ELTTEXTE000420"></a>
Las siguientes propiedades se utilizan para gestionar las características de una control Casilla de verificación por programación.


|   |   |
| --- | --- |
| [Altura](../Proprietes/2510050.md) | Obtiene y establece la altura de la opción (para los controles de casillas de verificación que utilizan un modo de visualización con una Image y/o un subcapítulo). |
| [BackgroundColor](../Proprietes/2510022.md) | Obtiene y establece la Color de fondo de la control Casilla de verificación. |
| [Color](../Proprietes/2510071.md) | Obtiene y establece la Color de opciones en la control Casilla de verificación. |
| [Conde](../WDLang1/3025009.md) | Obtiene el número de opciones en una control Casilla de verificación. |
| [HorizontalAlignment](../Proprietes/2510023.md) | Obtiene y establece la alineación horizontal de una control. |
| [Image](../Proprietes/2510034.md) | Obtiene y establece el Image associated con una opción (si el control Casilla de verificación utiliza un modo de visualización con un Image). |
| [InitialValue](../Proprietes/2510128.md) | Obtiene el valor inicial de una control Casilla de verificación. |
| [NumberColumn](../Proprietes/2510083.md) | Obtiene el número de columnas en la control Casilla de verificación. |
| [Pie de foto](../Proprietes/2510053.md) | Obtiene y establece el título de control Casilla de verificación o el título de una opción en el control Casilla de verificación. |
| [SubCaption](../Proprietes/1000017232.md) | Obtiene y establece el subcapítulo de una opción (si el control Casilla de verificación utiliza un modo de visualización con subcapítulos). |
| [TriState](../Proprietes/2510124.md) | Permite gestionar los tres estados de una opción en una control Casilla de verificación (marcada, no marcada, no definida). |
| [VerticalAlignment](../Proprietes/2510054.md) | Obtiene y establece la alineación vertical de una control. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una casilla de verificación control, consulte [Propiedades associated con un control Casilla de verificación](../WDChamp/1013172.md).

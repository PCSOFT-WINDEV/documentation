### Creando un control HTML en un reporte
<a name="creando_control_html_reporte_ELTPARAGRAPHE000103"></a>

Para crear un HTML control en un reporte:

1. En la pestaña "Creación", en el grupo "Texto", haga clic en "HTML".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

Observación: A partir de la versión 22, el HTML control en un reporte corresponde a un control Estático HTML.

### Ventana y programación de description
<a name="ventana_programacion_description_ELTPARAGRAPHE000161"></a>

El HTML control es un control Estático HTML. Puede introducir el texto directamente en la pestaña "General" de la ventana del control description.. Este texto se convertirá automáticamente a HTML. 

El HTML control se utiliza para mostrar: 

- el contenido de un archivo HTML local. 

- el contenido HTML de un elemento. 




Para inicializar el HTML control mediante programación, se puede asignar con: 

- un código HTML directamente. Por ejemplo: 
	
	```wl
	HTM_Code = "'''Hello'''"
	```


- la ruta de un archivo HTML local. 
	
	```wl
	HTM_Code = fExeDir() + "\MyHTMLFile.htm"
	```



Observaciones: 

- Las imágenes PNG y SVG en el contenido HTML no son compatibles. 

- Si la hoja de estilo CSS corresponde a un archivo externo, se ignora. 

- Todos los archivos externos que deben ser encontrados en INTERNET son ignorados. 

- Los códigos JavaScript no se ejecutan.

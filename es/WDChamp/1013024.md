
## Enlazar un control Combo Box a un campo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del campo
<a name="caracteristicas_del_campo_ELTTEXTE000136"></a>
Un control Combo Box puede estar enlazado a un campo de un archivo de datos, consulta, etc.

Se pueden enlazar los siguientes tipos de campos a un control Combo Box:

- Texto, 

- Numérico,

- Fecha u Hora, 

- Botón de opción, List Box, Combo Box.




**El campo enlazado al control Combo Box contendrá el índice del elemento seleccionado en el control.**

**Observación**: Cuando un campo está enlazado a un control Combo Box, el contenido del campo es el mismo para todos los registros del archivo de datos. Por ejemplo, si el campo enlazado al control contiene las opciones "Pagado", "En curso" y "Pendiente", este contenido será el mismo para todos los registros. 

El contenido del control Combo Box puede ser: 

- introducido directamente en el editor de análisis,

- introducido en el editor de ventanas o páginas,

- asignado mediante programación (con la función [ListAdd](../WDLang1/3049004.md)).

## Manipular el campo y el control Combo Box
<a name="manipular_campo_control_combo_box_ELTTEXTE000160"></a>


### Para actualizar el control Combo Box con el valor almacenado en el campo: 
<a name="para_actualizar_control_combo_box_con_valor_almacenado_campo_ELTPARAGRAPHE000038"></a>

**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los controles enlazados a un campo en el archivo de datos se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [FileToScreen](../WDLang4/3044210.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE2_2"></a>


### Para actualizar el campo con el índice del elemento seleccionado en el control Combo Box: 
<a name="para_actualizar_campo_con_indice_del_elemento_seleccionado_control_combo_box_ELTPARAGRAPHE000066"></a>

**Primer caso: los controles de la ventana o página están enlazados a los campos de un archivo de datos**
Todos los campos de el archivo de datos enlazados a un control de la ventana o página se actualizarán con:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [ScreenToFile](../WDLang4/3044146.md) en WINDEV o WINDEV Mobile.




**Segundo caso: los controles de la ventana o página no están enlazados a los campos de un archivo de datos**
El valor de cada control debe asignarse a cada campo utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

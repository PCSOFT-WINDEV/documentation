


## Gestión de fuentes icónicas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las **fuentes icónicas** se utilizan para mejorar la presentación de un título, pie de foto o texto. De hecho, las imágenes (iconos) pueden incluirse en el texto de una leyenda HTML.

Todos los controles de WEBDEV que incluyen una leyenda tienen una leyenda en formato HTML de Default: estática, botón, ... Esta estática utiliza un editor HTML integrado. Este editor HTML permite personalizar el texto (fuente, tamaño, color, etc.) a través de las opciones del panel "Texto" que se encuentra en la cinta de opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion_des_polices_icones%20-%20HC%20N%B0001.gif)
 El editor HTML integrado también incluye un botón que permite añadir un Icon en el texto.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Añadiendo un Icon
<a name="anadiendo_icon_ELTTEXTE000110"></a>
Para añadir una Icon en un pie de foto HTML: 

1. Posicione el cursor del ratón en el texto HTML a modificar.

2. Haga clic en el botón **Íconos de fuente** ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion_des_polices_icones%20-%20HC%20N%B0001%201.gif). 

3. La siguiente ventana se abre.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion_des_polices_icones%20-%20HC%20N%B0002.gif&type=thumb)


4. Seleccione la familia de iconos en la lista de fuentes.

5. Haga doble clic en la Icon solicitada o seleccione la Icon y valide la ventana. La Icon se añade en el título HTML en la posición del cursor.




Atención: La configuración actual del proyecto debe manipular cadenas Unicode en tiempo de ejecución. Esta opción está disponible en la ventana de configuración de description.

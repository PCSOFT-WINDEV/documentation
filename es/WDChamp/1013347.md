## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000186"></a>
**Atención: No se deben confundir los términos "Control Página interna" con "Página interna": el "Control Página interna" es un tipo de control y la "Página interna" es un tipo de página**.

WEBDEV utiliza de forma predeterminada los siguientes eventos para los **"Controles Página interna"**:


| Evento | Condición de ejecución |
| --- | --- |
| Después de cargar la página interna<br>(Código servidor) | Se ejecuta al cambiar la página interna con la función [ChangeSourcePage](../WDLang2/1000024600.md). Este evento se ejecuta después del evento "Declaraciones globales" e "Inicialización" de la nueva página interna.<br><br>Este evento permite, por ejemplo, inicializar controles o restaurar valores de la página interna.<br><br>No se debe utilizar la función [ChangeSourcePage](../WDLang2/1000024600.md) en este evento. |
| Antes de descargar la página interna<br>(Código servidor) | Se ejecuta al cambiar la página interna con la función [ChangeSourcePage](../WDLang2/1000024600.md). Este evento se ejecuta antes del evento de cierre de la página interna inicial.<br><br>Este evento se utiliza, por ejemplo, para guardar los valores de la página interna.<br><br>No se debe utilizar la función [ChangeSourcePage](../WDLang2/1000024600.md) en este evento. |




WEBDEV utiliza de forma predeterminada los siguientes eventos para las **"Páginas internas"**:: 


| Evento | Condición de ejecución |
| --- | --- |
| Declaraciones globales<br>(Código servidor) | Declaración de variables globales a la página. Este es el primer código que se ejecuta al abrir la página.<br>Este código se ejecuta antes que el código de declaración de las variables globales de la página host (página en la que se incluye la página interna).<br>Se puede, por ejemplo, llamar a un procedimiento de la página interna desde el código de declaración de las variables globales de la página host. |
| Inicialización<br>(Código servidor) | Se ejecuta al abrir la página.<br>Los eventos se ejecutan en el siguiente orden:<br><br>- Inicialización de los controles de la página.<br><br>- Inicialización de la página.<br><br><br> |
| Asignación de la propiedad Value<br>(Código servidor) | Se ejecuta cada vez que la propiedad ***Value*** se asigna a la página interna (directamente o con la propiedad Value). Este código se utiliza, por ejemplo, para recuperar los parámetros pasados a la página interna. |
| Obtención de la propiedad Value<br>(Código servidor) | Se ejecuta cada vez que se recupera la propiedad ***Value*** de la página interna. Este código permite que la página interna devuelva un valor a la página que contiene el control Página interna. |
| Carga (onload) de la página interna <br>(Código navegador) | Código navegador que se ejecuta cuando la página interna se muestra en el navegador (por ejemplo, con las funciones [PageDisplay](../WDLang2/3058008.md) o [PageRefresh](../WDLang2/3058011.md)). |
| Solicitud para actualizar la visualización<br>(Código servidor) | Se utiliza para agrupar el código de actualización de los controles de la página: contenido de los controles, estado (visible, en gris, etc.), título a actualizar, etc.<br><br>Este evento puede recibir parámetros. Solo se debe declarar un procedimiento en el evento. Por ejemplo: <br><br><br><pre><code>// Request for refreshing the display<br>PROCEDURE AskForRefresh(RefreshType is Boolean)</code></pre><br><br><br>Este evento se ejecuta mediante solicitud previa con las siguientes funciones: <br><br>- [RequestRefreshUI](../WDLang1/1000023899.md) o [RequestRefreshUIParent](../WDLang1/1000023900.md): Ejecuta el evento al final del evento actual (ejecución asíncrona). <br><br>- [ExecuteRefreshUI](../WDLang1/1000023901.md) o [ExecuteRefreshUIParent](../WDLang1/1000023902.md): Ejecuta el evento inmediatamente. <br><br><br>Al llamar a estas funciones, se puede especificar el valor de los parámetros esperados por el evento. |
| Cierre<br>(Código servidor) | Se ejecuta al cerrar la página.<br>**Observación**: Reanudar la entrada en uno de los controles de la página en este código impide que la página se cierre. |

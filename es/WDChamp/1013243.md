


## Personalización de la interfaz de un control Tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000189"></a>
En una control Tabla, usted tiene la posibilidad de personalizar los siguientes elementos:

- [el color de las filas pares y de las filas impares](#NOTE2_1).

- [la banda de selección y la barra de flotación](#NOTE2_2).

- [el título de las columnas (imágenes, iconos de clasificación, lupa, filtro,...)](../WDChamp/1013330.md)

- [las barras de desplazamiento verticales y horizontales](#NOTE2_3).

- [las fronteras exteriores e interiores de tabla](#NOTE2_4).




**Observación**: Personalizar estos elementos le permite aplicar un estilo "Vista" a los ordenadores que no ejecutan Windows Vista. Las plantillas de tema suministradas con WINDEV contienen ejemplos de imágenes reutilizables y modificables.

## Personalización
<a name="personalizacion_ELTTEXTE000213"></a>


### Color de fondo de las filas pares e impares
<a name="color_fondo_las_filas_pares_impares_ELTPARAGRAPHE000046"></a>

Puedes diferenciar las filas en un control Tabla especificando un color específico para las filas pares e impares.

Para especificar el color de fondo de las filas:

1. Ve a la pestaña "Estilo" en la ventana de descripción de control Tabla ("Descripción" en el menú contextual).

2. Seleccione el elemento "Colores de filas pares" o "Colores de filas impares".

3. Seleccione el color de fondo que desea utilizar (cuadro combinado"Color de fondo").



<a name="NOTE2_2"></a>


### Imagen de banda de selección e imagen de la barra de flotación
<a name="imagen_banda_seleccion_imagen_barra_flotacion_ELTPARAGRAPHE000059"></a>Se puede usar una imagen para ver fácilmente el fila seleccionado y el fila suspendido..<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BandeauTable.gif)
 To usar una imagen para el banda de selección y para la barra de flotación:

1. Ve a la pestaña "Estilo" en la ventana de descripción de control Tabla ("Descripción" en el menú contextual).

2. Seleccione el elemento "fila seleccionado" o "fila cubierto".

3. Abra la lista desplegable "Border" y seleccione "Image (9-image mode)".

4. Haga clic en el botón "Flecha derecha" que se encuentra a la derecha del cuadro combinado "Borde. Se abre una ventana para configurar la imagen de la barra.

5. Seleccione la imagen a utilizar haciendo clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- La opción "Examinar" se utiliza para seleccionar un archivo de imagen accesible desde su ordenador.. 

	- El botón "Catálogo" se utiliza para elegir una imagen que se encuentra en el catálogo de imágenes de WINDEV.




6. Si la imagen seleccionada contiene varios estados, especifique este número de estados. 

7. Cierre la ventana para configurar la imagen de la barra.


**Observación**: Tiene la posibilidad de mostrar la banda de selección sólo en algunas columnas de tabla.. Para obtener más información, consulte [Características de las columnas tabla](../WDChamp/1013330.md).

### Imagen de barras de desplazamiento
<a name="imagen_barras_desplazamiento_ELTPARAGRAPHE000095"></a>

Se puede utilizar una imagen para representar la barra de desplazamiento vertical y la barra de desplazamiento horizontal.

Para utilizar una imagen para las barras de desplazamiento:

1. Ve a la pestaña "Estilo" en la ventana de descripción de control Tabla ("Descripción" en el menú contextual).

2. Seleccione el elemento "Scrollbar".

3. Abra la lista desplegable "Border" y seleccione "Image (9-image mode)".

4. Haga clic en el botón "Flecha derecha" que se encuentra a la derecha del cuadro combinado "Borde. Se abre una ventana para configurar la imagen de las barras de desplazamiento.

5. Seleccione las imágenes a utilizar haciendo clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- La opción "Examinar" se utiliza para seleccionar un archivo de imagen accesible desde su ordenador.. 

	- El botón "Catálogo" se utiliza para elegir una imagen que se encuentra en el catálogo de imágenes de WINDEV.




6. Estas imágenes deben respetar el siguiente formato:

	- Barra de desplazamiento vertical: Tamaño de la imagen: 60 x 128 píxeles. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ascenseur_V_Table.gif)


	- Barra de desplazamiento Horizontal: Tamaño de la imagen: 384 x 47 píxeles. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ascenseur_H_Table.gif)

7. Cierre la ventana para configurar la imagen de las barras de desplazamiento.

### Imagen de bordes de control Tabla
<a name="imagen_bordes_control_tabla_ELTPARAGRAPHE000124"></a>

Se puede utilizar una imagen específica para los bordes de control Tabla.

Utilizar una imagen para los bordes de control Tabla:

1. Vaya a la pestaña "Estilo" de la ventana de descripción del control (opción "Descripción" del menú contextual).

2. Seleccione el elemento "Borde exterior" o "Borde interior".

3. Abra la lista desplegable "Border" y seleccione "Image (9-image mode)".

4. Haga clic en el botón "Flecha derecha" que se encuentra a la derecha del cuadro combinado "Borde. Se abre una ventana para configurar la imagen del borde.

5. Seleccione la imagen a utilizar haciendo clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) : 

	- La opción "Examinar" se utiliza para seleccionar un archivo de imagen accesible desde su ordenador.. 

	- El botón "Catálogo" se utiliza para elegir una imagen que se encuentra en el catálogo de imágenes de WINDEV.


Para aumentar el grosor del borde, todo lo que tienes que hacer es aumentar su desbordamiento.

6. Si la imagen seleccionada contiene varios estados, especifique este número de estados. 

7. Cierre la ventana para configurar la imagen del borde.

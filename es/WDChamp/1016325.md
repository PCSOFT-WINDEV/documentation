


## control Agenda relacionado a un archivo de datos o a Variable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000183"></a>
Tiene la capacidad de crear controles de Organizador relacionado a los datos. Puedes crear:

- un **control Agenda basado en un archivo de datos**: el control Agenda será relacionado a un archivo de datos o a una consulta. En este caso, el archivo de datos contiene las características de los nombramientos. 

- un **control Agenda basado en un Variable**: el control Agenda será relacionado a un Variable (una instancia de clase en la mayoría de los casos). En este caso, la clase lista las ocupaciones de muelles con sus características. 




Esto facilita la programación de los controles del Organizador: no es necesario utilizar la [Funciones de Organizerxxx](../WDLang1/1000019021.md) directamente. 





## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000207"></a>


### Creación de una control Agenda basada en un archivo de datos
<a name="creacion_una_control_agenda_basada_archivo_datos_ELTPARAGRAPHE000034"></a>

Para crear una control Agenda basada en un archivo de datos, puede: 

- utilizar la asistente para crear la control Agenda. Para ello, seleccione "Mostrar datos de un archivo o de una consulta existente". Responder a las preguntas de la asistente y validar. 

- modificar la description de la control Agenda. 




Modificar las características de una control Agenda existente:

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Contenido", seleccione "Archivo" y seleccione la fuente de datos utilizada.. Esta fuente de datos puede ser: 

	- un archivo de datos definido en el análisis,

	- una consulta del proyecto, 

	- una consulta incluida en la ventana o en la página. 




3. Asociar los diferentes elementos de la fuente de datos con las características del Appointment. Los elementos precedidos de una estrella son obligatorios. 
	**Observación**: Si el archivo de datos asociado no contiene directamente el elemento correspondiente, puede set un enlace de varios archivos. 

4. Validar la ventana description de la control Agenda. La control Agenda se basa ahora en la un archivo de datos. 





### Creación de una control Agenda basada en una Variable
<a name="creacion_una_control_agenda_basada_una_variable_ELTPARAGRAPHE000055"></a>

Para crear una control Agenda basada en una Variable, puede: 

- utilizar la asistente para crear la control Agenda. Para ello, seleccione "Mostrar datos de un Variable existente".. Responder a las preguntas de la asistente y validar. 

- modificar la description de la control Agenda. 




Para modificar las características de una control Agenda existente: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Contenido", seleccione la opción "Por variable", y luego seleccione la fuente de datos utilizada (clase, etc.). 

3. Asociar los diferentes elementos (miembros de la clase por ejemplo) de la fuente de datos con las características de la Appointment. Los elementos precedidos de una estrella son obligatorios. 

4. Validar la ventana description de la control Agenda. El control Agenda se basa ahora en un Variable. 






## Gestión de una control Agenda basada en un archivo de datos o Variable
<a name="gestion_una_control_agenda_basada_archivo_datos_variable_ELTTEXTE000237"></a>


### control Agenda basada en un archivo de datos
<a name="control_agenda_basada_archivo_datos_ELTPARAGRAPHE000071"></a>

Los controles del organizador basados en un archivo de datos son muy fáciles de manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario utiliza el control Agenda a través del menú contextual del control, las adiciones, supresiones y modificaciones realizadas en las citas se aplicarán automáticamente a la fuente de datos relacionado al control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario gestiona las citas a través de una ventana específica, basta con modificar el archivo de datos relacionado por control Agenda. Los datos visualizados se actualizarán automáticamente. También es posible utilizar [OrganizadorMuestra](../WDLang1/1000019425.md) para actualización los datos en el control. 

- Si el control Agenda permite la modificación de la duración y/o el desplazamiento de Appointment s, las modificaciones realizadas en el Appointment se aplicarán automáticamente al el archivo de datos relacionado al control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El borrado realizado en el control se realiza al mismo tiempo en la fuente de datos relacionado.





### control Agenda basada en un Variable
<a name="control_agenda_basada_variable_ELTPARAGRAPHE000104"></a>

Los controles del organizador basados en una Variable son muy fáciles de manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario utiliza el control Agenda a través del menú contextual del control, las adiciones, supresiones y modificaciones realizadas en las citas se aplicarán automáticamente a la fuente de datos relacionado al control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario maneja las citas a través de una ventana específica, todo lo que tiene que hacer es modificar los elementos relacionado al control Agenda (miembros de la clase por ejemplo).. Los datos visualizados se actualizarán automáticamente. También es posible utilizar [OrganizadorMuestra](../WDLang1/1000019425.md) para actualización los datos en el control. 

- Si el control Agenda permite la modificación de la duración y/o el desplazamiento de Appointment s, las modificaciones realizadas en el Appointment se aplicarán automáticamente a la fuente de datos relacionado al control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El borrado realizado en el control se realiza al mismo tiempo en la fuente de datos relacionado.

## Presentación del control Lector PDF
<a name="presentacion_del_control_lector_pdf_ELTTEXTE000316"></a>
El control "Lector PDF" es un control avanzado que permite a los usuarios finales ver documentos PDF directamente en las aplicaciones, sin utilizar un software externo. No es necesario abrir otros lectores. Esto le permite controlar lo que ven los usuarios finales y hace que el apoyo al usuario sea mucho más simple.



El control Lector PDF puede incluirse directamente en una ventana de la aplicación (como cualquier otro control), o abrirse en una nueva ventana (como un lector estándar).

Este control permite: 

- Abrir archivos PDF,

- Buscar texto en archivos PDF,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Imprimir archivos PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Permitir a los usuarios finales rellenar formularios.

- etc.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La cinta de opciones de este control permite realizar las acciones más comunes. Para obtener más información, consulte [Lector de PDF control: usando la cinta](../WDChamp/1410087040.md). 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) También se pueden utilizar varios métodos abreviados de teclado: 

| Atajo | Efecto | Atajo | Efecto |
| --- | --- | --- | --- |
| Ctrl + A | Selecciona todo el contenido del archivo PDF | Ctrl + C | Copia la selección en el portapapeles. |
| Ctrl + F | Inicia una búsqueda en el archivo PDF. | Ctrl + 0 | Abre un archivo PDF. |
| Ctrl + P | Imprime el archivo PDF actual. |   |   |
| Ctrl + rueda del ratón | Cambia el zoom. | Ctrl + 0 | Establece el zoom en 100% |
| Ctrl + inicio | Va al principio del documento. | Ctrl + Fin | Va al final del documento. |
| Re Pág | Muestra la página anterior. | Av Pág | Muestra la página siguiente. |

## Crear un control Lector PDF
<a name="crear_control_lector_pdf_ELTTEXTE000340"></a>
**Para crear un control Lector PDF**:

1. Seleccione el control que desea crear en la cinta de opciones: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Ofimática" y seleccione "Lector PDF".




2. Haga clic en la ubicación en la que desea crear el control.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Características del control Lector PDF
<a name="caracteristicas_del_control_lector_pdf_ELTTEXTE000368"></a>
Las siguientes opciones están disponibles en la ventana de descripción del control Lector PDF: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_Description%20-%20HC%20N%B0001.gif)


- **Documento inicial**: Permite seleccionar el documento inicial que se muestra en el control Lector PDF (pestaña "General"). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Con cinta de opciones**: Determina si se muestra o no la cinta de opciones (pestaña "General"). Si se muestra la cinta: 

	- el área "VI fuente de la cinta" permite: 

		- utilizar la cinta de opciones predeterminada. En este caso, la opción "VI fuente de la cinta" corresponde a "Ninguna". 

		- personalizar la cinta de opciones predeterminada.
						Simplemente seleccione la ventana predefinida en la opción "VI fuente de la cinta". En este caso, la ventana "IW_WinDevAAF_DocumentReaderBar" se integra automáticamente en el proyecto. Esta ventana interna (disponible en inglés y francés) puede modificarse según sus necesidades. Esta ventana contiene el código necesario para gestionar el control Lector PDF. Le recomendamos comprobar los errores de UI y asegurarse de que la ventana funciona correctamente después de realizar cualquier cambio. 

		- utilizar una ventana interna para crear una cinta de opciones específica.
						Puede utilizar cualquier ventana interna del proyecto para crear la cinta.




- La función "Aplicar plantilla del proyecto a la cinta de opciones" aplica la plantilla del proyecto a la ventana interna que corresponde a la cinta de opciones y a todas las ventanas abiertas a través de la cinta.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Mostrar panel de miniaturas**: Permite especificar si el panel de miniaturas se muestra de forma predeterminada en el control Lector PDF. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Permitir cambiar de documento (Abrir, Dnd, etc.)**: Esta opción muestra u oculta los botones que permiten abrir un nuevo documento en la cinta de opciones. Si no se selecciona esta opción, el usuario final solo podrá manipular el archivo PDF definido por la aplicación. De lo contrario, el usuario final podrá manipular cualquier archivo PDF.

## Menú contextual
<a name="menu_contextual_ELTTEXTE000392"></a>
De forma predeterminada, el control Lector PDF incluye varias opciones en el menú contextual asociado. Para desactivar el menú contextual, utilice la propiedad [ContextMenu](../Proprietes/2510077.md) con la constante *contextMenuDisabled*.

<a name="NOTE4_1"></a>


## Limitaciones
<a name="limitaciones_ELTTEXTE000416"></a>


- Las siguientes funcionalidades no están disponibles en el control Lector PDF: 

	- Marcadores, 

	- Interacciones mediante JavaScript (por ejemplo, enlace oculto en una imagen, etc.). 




- El control Lector PDF no está disponible en Windows XP.

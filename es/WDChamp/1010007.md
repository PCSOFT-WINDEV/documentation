


## Manipular opciones de menú mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000166"></a>
Las opciones del menú pueden ser manejadas:

- desde el editor de ventanas. Para obtener más información, consulte [Manejo de las opciones de menú en el editor](../WDChamp/1010008.md).

- mediante programación.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000190"></a>
Las opciones se manipulan de la misma manera en los menús contextuales y en los menús principales.

Para manipular una opción de menú mediante programación, utilice la siguiente sintaxis:

- **"&lt;OptionName&gt;"** si la opción se maneja desde la ventana.

- **"&lt;NombreDeVentana&gt;.&lt;NombreDeMenú&gt;.&lt;NombreDeOpción&gt;"** si la opción se maneja desde otra ventana.




**Observación**: También tiene la posibilidad de manipular los menús contextuales a través de la programación (no sólo sus opciones). Para obtener más información, consulte [Menú contextual personalizado](../WDChamp/1010017.md).
<a name="NOTE2_2"></a>

<a name="funciones_wlanguage_ELTPARAGRAPHE000047"></a>WINDEV y WINDEV Mobile proponen varias funciones WLanguage específicas para las opciones de menú:



|   |   |
| --- | --- |
| [EnumMenu](../WDLang1/3038039.md) | Devuelve:<br><br>- el nombre de la enésima opción o subopción que se encuentra en el menú principal de la ventana. <br><br>- el nombre de la enésima opción o subopción que se encuentra en un menú contextual personalizado (creado con WINDEV o WINDEV Mobile).<br><br><br> |
| [MenuAdd](../WDLang1/3038048.md) | Agrega un menú emergente existente en el menú principal de la ventana actual. |
| [MenuAddMenu](../WDLang1/1000017266.md) | Agrega un nuevo menú a un menú desplegable o contextual de una ventana o página. |
| [MenuAddOption](../WDLang1/1000017262.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. |
| [MenuAddSeparator](../WDLang1/1000017270.md) | Añade un nuevo separador en un menú (desplegable o contextual). |
| [MenuClone](../WDLang1/1000017257.md) | Clona un menú o una opción de menú así como el código de associated. |
| [MenuDelete](../WDLang1/3038058.md) | Borra un menú o una opción de menú. |
| [MenúExistir](../WDLang1/1000019365.md) | Indica si existe una opción de menú en un menú.. |
| [MenúInserciónOpción](../WDLang1/1000017265.md) | Inserta una nueva opción en una posición específica. |
| [MenúInsertarMenú](../WDLang1/1000017267.md) | Inserta un menú antes que otro menú en una ventana. |
| [MenuInsertSeparator](../WDLang1/1000017271.md) | Inserta un separador en un menú. |
| [MenuInvisible](../WDLang1/3038022.md) | Hace una opción de menú Invisible (menú emergente o menú desplegable). |
| [MenuIsMarked](../WDLang1/3038008.md) | Se utiliza para saber si una marca de verificación aparece delante de una opción de menú (menú emergente o menú desplegable). |
| [MenuLabel](../WDLang1/3038009.md) | Obtiene o cambia la etiqueta de una opción de menú (menú contextual o desplegable). |
| [MenuMark](../WDLang1/3038007.md) | Posiciona la marca de verificación delante de la opción del menú (menú emergente o menú desplegable). |
| [MenuSelectMinus](../WDLang1/3038010.md) | Desactiva (hace gris) una opción de menú (menú emergente o menú desplegable). |
| [MenuSelectPlus](../WDLang1/3038005.md) | Habilita una opción de menú (menú emergente o menú desplegable). |
| [MenuState](../WDLang1/3038003.md) | Identifica el estado de una opción de menú (menú contextual o menú desplegable): activo, inactivo o Invisible. |
| [MenuUnMark](../WDLang1/3038002.md) | Elimina la marca de verificación que aparece delante de la opción del menú (menú emergente o menú desplegable). |

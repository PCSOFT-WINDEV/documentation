
## Vinculación de una control Enlace a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000081"></a>
Una control Enlace puede estar vinculada a un elemento de un archivo de datos, consulta, etc.

Los siguientes tipos de artículos pueden estar vinculados a una control Enlace:

- Texto.

- Fecha u hora.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Actualización de la control Enlace con los datos del artículo (texto
<a name="actualizacion_control_enlace_con_los_datos_del_articulo_texto_ELTTEXTE000105"></a>
**1er caso: los controles de la página están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo de datos relacionado) serán actualizados por [ArchivoPágina](../WDLang4/3044148.md).

**2º caso: los controles de la página no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Link control> = <Data file>.<Item>
```

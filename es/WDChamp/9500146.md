
## Control Imagen: los formatos soportados
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000150"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Los siguientes formatos de Image están soportados en Linux: 

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- Gráficos de red portátiles (\*.png)

- Mapa de bits portátil (\*.pbm)

- Mapa de grises portátil (\*.pgm)

- Portátil Pixmap (\*.ppm)

- Etiquetado en el formato de archivo Image (\*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- X11 Mapa de bits (\*.xbm)

- X11 Pixmap (\*.xpm)

- Gráficos vectoriales escalables (\*.svg)
	Observación: Para administrar este formato, debe instalar una biblioteca QT que admita el formato SVG.

- WINDEV Image s (generado por el editor de Image, \*.WDPic).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Los formatos de Image soportados son: 

- Gráficos de red portátiles (\*.png)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).






![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los formatos de Image soportados son: 

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)
	Atención: Gif animado no son compatibles.

- Grupo Mixto de Expertos en Fotografía (\*.jpeg)

- Gráficos de red portátiles (\*.png)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).
	Atención: Si hay que asignar un fichero ".WDPic" por programación a un control, este fichero debe haber sido añadido previamente en la lista de elementos del proyecto.

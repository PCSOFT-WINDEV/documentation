<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000325"></a>
Las características de los controles se pueden ver y/o cambiar en:

- [la ventana de descripción del control](#NOTE2_1).

- [el modificador](#NOTE3_1).




El editor de ventanas también permite:

- [cambiar el título del control](#NOTE4_1).

- [cambiar el nombre del control](#NOTE5_1).

- [cambiar la posición y dimensiones del control](#NOTE6_1).

- [cambiar la imagen asociada a un control](#NOTE7_1).

## Ventana de descripción del control
<a name="ventana_descripcion_del_control_ELTTEXTE000349"></a>


### Abrir la ventana de descripción de un control
<a name="abrir_ventana_descripcion_control_ELTPARAGRAPHE000040"></a>

Para abrir la ventana de descripción de un control: 

- haga doble clic en el control.

- seleccione la opción "Descripción" en el menú contextual.

- seleccione el control y presione Alt + Entrar.




Ejemplo: Ventana de descripción del control Botón: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenDescriptionChamp.gif)


**Observaciones**:

- La ventana de descripción puede abrirse para una selección de controles. Solo se mostrarán las características comunes a los controles seleccionados.

- Se pueden abrir varias ventanas de descripción al mismo tiempo. Cada ventana de descripción muestra las características de uno o más controles.



<a name="NOTE2_2"></a>


### Características por pestaña
<a name="caracteristicas_por_pestana_ELTPARAGRAPHE000059"></a>

| Pestaña | Características |
| --- | --- |
| **General** | Permite especificar el nombre y las opciones de visualización del control. Para obtener más información, consulte las [Diferentes máscaras de entrada](../WDChamp/1014003.md).<br><br>- la imagen de un control Imagen, las opciones de un control Botón de opción, los controles de una barra de herramientas, etc. Para obtener más información, consulte la ayuda en línea del tipo de control deseado.<br><br><br> |
| **UI** | Permite definir:<br><br>- el estado inicial del control al abrir la ventana.<br><br>- los parámetros del control (acceder al control con Tab, etc.).<br><br>- la acción del clic derecho en el control.<br><br>- la apariencia del cursor al pasar por encima del control.<br><br>- el menú contextual del control. Para obtener más información, consulte [Menú contextual](../WDChamp/1010011.md).<br><br>- el atajo de teclado del control.<br><br>- el tamaño del control.<br><br>- la posición del control en la ventana.<br><br>- el anclaje del control. Para obtener más información, consulte [Anchor](../Proprietes/2510001.md).<br><br>- el plano de la ventana al que pertenece el control. Para obtener más información, consulte [Plano de la ventana](../WDChamp/1013078.md).<br><br>- el grupo de controles al que pertenece el control. Para obtener más información, consulte [Grupo de controles](../WDChamp/1010016.md).<br><br><br> |
| **Detalles** | Permite definir los parámetros avanzados del control:<br><br>- la acción de arrastrar y soltar. Para obtener más información, consulte [Arrastrar - Soltar](../WDLang1/3030007.md).<br><br>- el uso de la validación automática (para controles Botón). Para obtener más información, consulte [Validación automática de ventanas](../WDChamp/1013072.md).<br><br>- los ajustes de entrada para controles Campo de entrada.<br><br>- parámetros específicos de los controles List Box, Combo Box, Tabla, etc.<br><br><br>Estos parámetros avanzados se agrupan en varias categorías:<br><br>- los parámetros que cambian la apariencia del control (opciones sobre fondo beige).<br><br>- los parámetros que cambian el modo de funcionamiento y la entrada del control (parámetros sobre fondo verde).<br><br>- los parámetros relativos al valor de retorno del control (parámetros sobre fondo azul).<br><br><br> |

| **Nota** | Permite describir el modo de funcionamiento detallado del control.<br>Esta información se imprimirá en la documentación técnica (ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_imprimer.gif) en la pestaña "Inicio", en el grupo "General").<br><br>Esta información puede obtenerse y establecerse mediante programación con las propiedades [Nota](../Proprietes/2510085.md) y [NoteTitle](../Proprietes/2510121.md). |

| **Estilo** | Permite:<br><br>- definir un estilo predefinido para el control.<br><br>- cambiar o crear un estilo predefinido para el tipo de control.<br><br>- cambiar la apariencia del control (color de fondo, borde, fuente, etc.).<br><br><br> |

## Uso del modificador
<a name="uso_del_modificador_ELTTEXTE000379"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000184"></a>

El modificador permite ver y/o cambiar desde el editor de ventanas las principales características de los siguientes elementos:

- los controles seleccionados.

- la ventana seleccionada.




Para obtener más información, consulte [Modificador](../Editeurs/2027006.md).
<a name="NOTE3_2"></a>


### Ver y/o cambiar las características de uno o más controles con el modificador
<a name="ver_cambiar_las_caracteristicas_uno_mas_controles_con_modificador_ELTPARAGRAPHE000199"></a>

Para ver y/o cambiar las características de uno o varios controles con el modificador: 

1. Seleccione los controles que desea modificar:

	- en la ventana directamente (consulte [Seleccionar uno o más controles](../Editeurs/2026007.md)).

	- mediante la lista desplegable del modificador. En este caso, solo se puede seleccionar un control a la vez. Los controladores de tamaño del control seleccionado se muestran automáticamente.
			Si se seleccionan varios controles, solo se muestra en el modificador la información común a los controles seleccionados.




2. Haga los cambios deseados. Estos cambios se aplican a todos los controles. Es posible:

	- seleccionar el nuevo valor en una lista desplegable.

	- seleccionar el nuevo valor en una ventana específica (nuevo color, por ejemplo).

	- cambiar el valor directamente.

## Cambiar el título del control
<a name="cambiar_titulo_del_control_ELTTEXTE000409"></a>
Para cambiar el título de un control directamente en la ventana: 

1. Seleccione el control que desea modificar.

2. Haga clic de nuevo en el control o pulse la tecla Entrar. El cursor se convierte en un cursor de texto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LibelleModifDirect.gif)


3. Introduzca el nuevo texto.

4. Confirme con la tecla Entrar.




**Observaciones**: 

- Al introducir el texto directamente, se comprueba la ortografía y se permite el texto RTF y los caracteres no occidentales.

- También es posible cambiar el texto de una columna en un control Tabla.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Lib_table_modifdirect.gif)

## Cambiar el nombre del control
<a name="cambiar_nombre_del_control_ELTTEXTE000433"></a>
Puede cambiar el nombre de un control directamente desde la barra de estado del editor de ventanas. 

Para cambiar el nombre del control:

1. Seleccione el control que desea modificar.

2. Escriba el nuevo nombre del control en la barra de estado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NomChamp.gif)

## Cambiar el tamaño y la posición del control
<a name="cambiar_tamano_posicion_del_control_ELTTEXTE000457"></a>
Puede cambiar el tamaño y la posición de un control directamente desde la barra de estado del editor de ventanas.

Para cambiar el tamaño y la posición del control:

2. Introduzca los nuevos valores del tamaño y la posición en la barra de estado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PositionDimensionChamp.gif)

## Cambiar la imagen asociada a un control
<a name="cambiar_imagen_asociada_control_ELTTEXTE000481"></a>


### Cambiar la imagen del control
<a name="cambiar_imagen_del_control_ELTPARAGRAPHE000270"></a>

Puede cambiar la imagen asociada a un control Imagen o Botón directamente en la ventana.

Para cambiar la imagen de un control directamente en el editor de ventanas:

1. Seleccione el control que desea modificar.

2. Presione Ctrl + Mayús + P. Un selector de archivos se abre.

3. Seleccione el archivo de imagen que desea asociar al control actual y valide.

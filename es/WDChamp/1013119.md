
## Características de una barra de herramientas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000110"></a>
Las principales características de una "Barra de herramientas" control se configuran en la pestaña "Detalles" y en la pestaña "UI" de la ventana description de control.

La pestaña "General" sirve para conocer y definir los controles que se encuentran en la barra de herramientas. 

La pestaña "UI" se utiliza para definir:

- el estado inicial de la barra de herramientas.

- los parámetros relativos al tamaño y la posición de los controles en la barra de herramientas.




La pestaña "Detalles" permite definir: 

- las posiciones autorizadas para la barra de herramientas.

- el espacio entre los controles que se encuentran en la barra de herramientas.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Posiciones autorizadas para la barra de herramientas
<a name="posiciones_autorizadas_para_barra_herramientas_ELTTEXTE000134"></a>
WINDEV le permite seleccionar las diferentes posiciones que el usuario podrá utilizar para la barra de herramientas en tiempo de ejecución.

Esta barra de herramientas puede ser:

- anclado a uno de los lados de la ventana.

- flotante, la barra de herramientas será una ventana independiente.




Esta información se especifica en la pestaña "Detalles" de la ventana description de control. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Parámetros relativos al tamaño y la posición de la barra de herramientas
<a name="parametros_relativos_tamano_posicion_barra_herramientas_ELTTEXTE000158"></a>
Varios parámetros permiten configurar el modo de funcionamiento de la barra de herramientas control. Estos parámetros se definen en la pestaña "UI" de la ventana description de control:

- **Desplaza los controles de la ventana automáticamente**: Esta opción le permite desplazar los controles que se encuentran en la ventana cuando cambia la posición de la barra de herramientas.. Si esta opción no está seleccionada, algunos controles de ventana pueden estar ocultos según la posición de la barra de herramientas.

- **Redimensiona el tamaño automáticamente**: Esta opción permite adaptar automáticamente el tamaño de la barra de herramientas a su contenido (cuando los controles de la barra de herramientas se hacen visibles o Invisible mediante programación, por ejemplo).

- **Recordar tamaño y posición**: Esta opción sirve para guardar el tamaño y la posición de la barra de herramientas cuando se cierra la ventana de associated. Durante la siguiente apertura de la ventana, la barra de herramientas se mostrará con este tamaño y posición.


## Control Flexbox description: Pestaña General
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
En la pestaña "General" de la ventana Flexbox control description, puede ver y modificar las opciones de los diferentes control s integrados en un contenedor Flexbox. 

Puede definir el anclaje de cada control para get el comportamiento deseado en el contenedor Flexbox. 

Observación: Estos ajustes también aparecen en las opciones de anclaje de control.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Opciones de la pestaña "General
<a name="opciones_pestana_general_ELTTEXTE000132"></a>
Para cada control del contenedor Flexbox, puede configurar los siguientes aspectos: 

- **Ancho**: Distribución del espacio libre horizontal positivo en "partes".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_General%20-%20HC%20N%B0001%201.gif)


	- **si el ancho del contenedor flexbox aumenta** , los controles se estirarán horizontalmente para llenar el espacio disponible (flex-grow). Los valores posibles son: 

	- 0: sin crecimiento

	- 1: los controles crecerán por igual hasta ocupar el espacio horizontal disponible. Esta es la opción Default.. 

	- otro valor: Los controles crecerán proporcionalmente utilizando este coeficiente para llenar el espacio horizontal disponible.

- **Alto**: Definition de alineación vertical y anclajes (align-self). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_General%20-%20HC%20N%B0002.gif)
 Los valores posibles son: 

	- indefinido, la control se estirará verticalmente (stretch)

	- auto, definido por la Flexbox control

	- al inicio (flex-start)

	- center

	- al final (flex-end)

	- alinear textos \* (baseline): Los elementos se alinean de modo que sus líneas de base de texto se alinean. 
			Observación: "\*" indica que esta opción se ignora en el editor. 


 Por Default, el control se estirará verticalmente (stretch).

- **Orden**: Los controles se muestran siguiendo el orden definido por este parámetro. Si no se especifica ningún valor (indefinido o 0), la control se sitúa a la derecha de la control anterior . Atención: Una control con una posición definida por un número ya no puede desplazarse en el editor: permanecerá en la posición set por ese número.

- **Anclajes CSS personalizados**: Esta opción permite introducir código CSS personalizado con los anclajes deseados. Este código CSS anula los parámetros definidos anteriormente. Por ejemplo: 
	
	```txt
	flex 20 1 100px;
	```
corresponde a:

	- flex-grow: 20

	- retracción flexible: 1

	- flex-basis: 100px

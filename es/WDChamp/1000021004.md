
## Manejo de controles de cinta mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000176"></a>
WINDEV le permite programar los controles de la cinta manipular, así como sus pestañas y grupos. 

El nombre de la control Cinta de opciones se utiliza para manipular la cinta de opciones y sus pestañas mediante programación. 

Los nombres de los grupos se utilizan para manipular grupos mediante programación. 

**Observación**: No necesitas programar cómo cambiar entre pestañas. 





## Manejo de la control Cinta de opciones y sus grupos
<a name="manejo_control_cinta_opciones_sus_grupos_ELTTEXTE000200"></a>


### Manipulación de una control Cinta de opciones
<a name="manipulacion_una_control_cinta_opciones_ELTPARAGRAPHE000021"></a>

**Para manipular una pestaña del control Cinta de opciones**, utilice la siguiente sintaxis: 


```txt
<Ribbon Name>[TabNumber].PropertyName = <Value>
```


Para encanecer la segunda pestaña de la control Cinta de opciones, puede utilizar la siguiente sintaxis: 


```wl
RIBBON_Menu[2].Grayed = True
```


La primera pestaña de la cinta está activa por Default. La pestaña activa puede cambiarse mediante programación: 

- utilizando la siguiente sintaxis: 
	
	```txt
	<Ribbon Name> = <Tab Number>
	```


- utilizando el [Valor](../Proprietes/2510130.md) Property. 




Para get el número de la ficha Active, puede: 

- utilizar una expresión de asignación simple: 
	
	```txt
	<Active Tab Num> = <Ribbon Name>
	```


- utilice la propiedad [Valor](../Proprietes/2510130.md). 




La [Pie de foto](../Proprietes/2510053.md) Property obtiene y fija el texto de la ficha Active. Por ejemplo: 


```wl
Trace(RIBBON_Menu[MyActiveTab]Caption)
```





### Contraer un control Cinta de opciones
<a name="contraer_control_cinta_opciones_ELTPARAGRAPHE000060"></a>

La propiedad [Colapsado](../Proprietes/2513014.md) colapsa (o no) un control Cinta de opciones. Si la control Cinta de opciones está colapsada, sólo se muestra el texto de las pestañas (como en un menú estándar). El contenido de la pestaña se muestra cuando el usuario hace clic en ella. Si el usuario vuelve a hacer clic en la pestaña (o en uno de los botones de la pestaña), el contenido de la pestaña deja de mostrarse.. 

Para forzar la desaparición de la pestaña en una acción del usuario (cuando el usuario selecciona una control Combo Box, por ejemplo), basta con utilizar [Modo de ampliación de la cinta de opciones](../WDLang1/1000020378.md). 




### Manejo de un grupo
<a name="manejo_grupo_ELTPARAGRAPHE000076"></a>

Para manipular un grupo se utiliza la siguiente sintaxis: 


```txt
<Ribbon Name>.<Groupe Name>
```


Por ejemplo: 


```wl
RIBBON_Menu.GRITEM_QuickAccess.Visible = False
```





### Forzar el cierre de un grupo
<a name="forzar_cierre_grupo_ELTPARAGRAPHE000088"></a>

En un control Cinta de opciones, los grupos pequeños muestran un Icon y una flecha que sirve para abrir una ventana emergente que contiene todos los controles.

Por Default, este popup está cerrado:

- cuando el usuario hace clic en un botón,

- cuando el usuario realiza una selección en una control Combo Box.




Para establecer una acción diferente que cierre el grupo (por ejemplo, introducir 3 caracteres), utilice [Agrupación RibbonCloseGrouping](../WDLang1/1000020136.md) en el Process deseado.





## Propiedades específicas de los controles de la cinta de opciones
<a name="propiedades_especificas_los_controles_cinta_opciones_ELTTEXTE000242"></a>
Las siguientes propiedades pueden ser utilizadas con una control Cinta de opciones: 


|   |   |
| --- | --- |
| [Colapsado](../Proprietes/2513014.md) | Se usa para colapsar (o no) una control Cinta de opciones. Si la control Cinta de opciones está colapsada, sólo se muestra el texto de las pestañas. Al hacer clic en el texto de la ficha, ésta se visualiza íntegramente con todos sus grupos. |
| [Conde](../Proprietes/2510097.md) | Obtiene el número de pestañas de una control Cinta de opciones. |
| [Desacomodado](../Proprietes/1000026078.md) | Permite saber si una ficha está desprendida de un control Cinta de opciones. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Cinta de opciones, consulte [Propiedades associated con controles Ribbon](../WDChamp/1000021003.md).

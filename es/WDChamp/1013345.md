## Presentación
<a name="presentacion_ELTTEXTE000070"></a>
Aquí están los conceptos principales para programar los controles del looper de navegación.

Varias funciones de WLanguage le permiten manejar los controles del Looper de navegación a través de la programación.

No dude en consultar la documentación sobre estas funciones. Para obtener más información, consulte: [Funciones de gestión de control Looper](../WDLang2/1000023945.md).

## Añadir o modificar una fila en una control Looper basada en un archivo de datos
<a name="anadir_modificar_una_fila_una_control_looper_basada_archivo_datos_ELTTEXTE000103"></a>
La adición y modificación de una fila en una control Looper (y de una Record en una el archivo de datos) consta de dos fases:

1. Agregue o modifique el registro en el archivo de datos (a través de una ventana o página de formulario, por ejemplo)

2. Redistribución del contenido de la navegación control Looper ([&lt;Pantalla](../WDLang2/1000023451.md)).




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Borrar una fila de una control Looper basada en un archivo de datos
<a name="borrar_una_fila_una_control_looper_basada_archivo_datos_ELTTEXTE000137"></a>
Para eliminar una fila de una control Looper basada en un archivo de datos:: 

1. Eliminar la Record de el archivo de datos (a través de [&lt;Fuente&gt;.Delete](../WDLang4/1000025071.md)).

2. Volver a mostrar el contenido de la navegación control Looper (a través de [&lt;Pantalla](../WDLang2/1000023451.md)).


**Importante**: [&lt;RecNum](../WDLang2/1000023521.md) devuelve el número del registro actual en un control Looper basado en un archivo de datos.


## Vincular una nota reposicionable control a un artículo
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000075"></a>
Una Nota Reposicionable control puede estar ligada a un artículo en un archivo de datos, consulta, etc.

El artículo vinculado a la Nota control será un artículo de Texto.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Manipulación del artículo y de la nota reposicionable control
<a name="manipulacion_del_articulo_nota_reposicionable_control_ELTTEXTE000099"></a>


### Para actualizar la control con los datos del artículo:
<a name="para_actualizar_control_con_los_datos_del_articulo_ELTPARAGRAPHE000019"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los controles vinculados a un elemento en el archivo de datos (o un archivo relacionado) serán actualizados por [FileToScreen](../WDLang4/3044210.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
Se debe asignar un valor a cada control utilizando la siguiente sintaxis:


```txt
<Control name> = <Data file>.<Item>
```

<a name="NOTE3_2"></a>


### Para actualizar el artículo con los datos de la Nota Reposicionable control
<a name="para_actualizar_articulo_con_los_datos_nota_reposicionable_control_ELTPARAGRAPHE000026"></a>

**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la ventana serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

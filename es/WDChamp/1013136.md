


## Manejo programático de los controles Image
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000305"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten manipular un control Imagen a través de la programación. Para hacerlo, usa la Variable de la control Imagen en el código.

La Variable de control Imagen corresponde al nombre de control Imagen. Esta es una variable de texto.

## Mostrando un archivo Image en un control Imagen
<a name="mostrando_archivo_image_control_imagen_ELTTEXTE000329"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### En una ventana
<a name="una_ventana_ELTPARAGRAPHE000022"></a>

Para mostrar un archivo de Image en un control Imagen, una simple tarea es suficiente.

Por ejemplo:

- asignación directa: 
	
	```wl
	IMG_MyImage = "C:\Temp\MyImage.bmp"
	```


- asignación utilizando una variable:
	
	```wl
	ImagePath = fCurrentDir() + "\MyImage.bmp"
	IMG_MyImage = ImagePath
	```

	Observación: Un control Imagen también puede ser inicializado desde un buffer Variable o desde un texto Variable que contenga el valor Image en formato binario.




**Observación**: si el control es associated con elemento un archivo de datos, los valores se asignan automáticamente mediante [FileToScreen](../WDLang4/3044210.md). Para obtener más información, consulte [Enlace de un control Imagen a un artículo](../WDChamp/1013135.md).

## Recuperando el nombre del archivo Image que aparece en el control Imagen
<a name="recuperando_nombre_del_archivo_image_que_aparece_control_imagen_ELTTEXTE000359"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### En una ventana
<a name="una_ventana_ELTPARAGRAPHE000099"></a>

**Para obtener el nombre de la imagen que se muestra en un control Imagen:**

- realice una asignación simple. Por ejemplo:
	
	```wl
	EDT_CurrentImageContent = IMG_Image1
	```


- recuperar parte del nombre, con [fExtractPath](../WDLang1/3036026.md). Por ejemplo:
	
	```wl
	NameImageFile = fExtractPath(IMG_Image1, fFile + fExtension)
	```


- mediante programación, utilice la propiedad [Valor](../Proprietes/2510130.md)




**Observación**: Para recuperar la imagen (o su nombre) en el artículo associated con el control Imagen: 

1. Si el control es associated con un elemento de texto, el valor introducido se recupera automáticamente a través de [ScreenToFile](../WDLang4/3044146.md). Usted también tiene la habilidad de usar la siguiente notación: NOMBRE DE CLIENTE = EDT_Edit1.

2. Si el control es associated con una nota binaria, utilice [HLinkMemo](../WDLang4/3044173.md) para asociar el archivo Image con el artículo.

3. Si la control Imagen fue asignada desde un buffer Variable o desde una cadena Variable, [Valor](../Proprietes/2510130.md) devuelve una cadena vacía.
	Para obtener más información, consulte [Enlace de un control Imagen a un artículo](../WDChamp/1013135.md).

## Recuperando la posición del ratón en un Image con un clic
<a name="recuperando_posicion_del_raton_image_con_clic_ELTTEXTE000389"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para recuperar la posición del ratón en un área de clic, se deben utilizar las siguientes funciones en el código de clic de la Image:


|   |   |
| --- | --- |
| [MouseXPos](../WDLang1/3071007.md) | Devuelve la coordenada horizontal del clic del ratón. |
| [MouseYPos](../WDLang1/3071008.md) | Devuelve la coordenada vertical del clic del ratón. |

## Borrar o reiniciar el contenido de un control Imagen
<a name="borrar_reiniciar_contenido_control_imagen_ELTTEXTE000413"></a>
Para borrar o reiniciar una control Imagen, todo lo que tiene que hacer es asignar una cadena vacía a la control Imagen. Por ejemplo: 


```wl
IMG_MyImage = ""
```

**Observación**: También tiene la posibilidad de utilizar [dStartDrawing](../WDLang1/3029009.md)/[dEndDrawing](../WDLang1/3029019.md) en el control Imagen.

## Propiedades específicas de los controles Image
<a name="propiedades_especificas_los_controles_image_ELTTEXTE000437"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las siguientes propiedades se utilizan para gestionar las características de una control Imagen por programación.


|   |   |
| --- | --- |
| [Animación](../Proprietes/2510006.md) | Obtiene el estado actual de la Image animada o inicia/detén la animación de una Image (por ejemplo, GIF animado). |
| [HorizontalAlignment](../Proprietes/2510023.md) | Obtiene y establece la alineación horizontal de la leyenda Image. |
| [ImageMode](../Proprietes/2510058.md) | Se utiliza para averiguar y modificar el modo de visualización de un Image en un control Imagen. |
| [InitialAnimation](../Proprietes/2510012.md) | Sirve para conocer el estado inicial de la animación en una control Imagen. |
| [VerticalAlignment](../Proprietes/2510054.md) | Obtiene y establece la alineación vertical de la leyenda de Image. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con los controles Image, consulte [Propiedades de control Imagen](../WDChamp/1013138.md).

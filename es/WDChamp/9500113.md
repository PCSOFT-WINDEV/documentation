


## Control Campo de entrada HTML: Barra de formato automática
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000143"></a>
En los controles Campo de entrada HTML, se puede mostrar automáticamente una barra de formato tan pronto como el control entra en modo de edición. De esta manera, las opciones de formato se gestionan automáticamente.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000167"></a>


### Activar la barra de formato para los controles HTML
<a name="activar_barra_formato_para_los_controles_html_ELTPARAGRAPHE000018"></a>

Para activar la barra de formato para los controles HTML:

1. Abra la descripción de un control Campo de entrada HTML. La control Campo de entrada es un "texto HTML" control. 

2. En la pestaña "General", indique si el control utiliza una barra de herramientas HTML: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_ConfigBarreWB%20-%20HC%20N%B0002.gif)
 Para seleccionar las opciones disponibles en la barra de herramientas, haga clic en "Ajustes de la barra de herramientas...".. 

3. Valide.



<a name="NOTE2_2"></a>


### Funcionalidad disponible en las FAA
<a name="funcionalidad_disponible_las_faa_ELTPARAGRAPHE000035"></a>

En un control Campo de entrada HTML, la barra de herramientas HTML siempre está visible, o aparece automáticamente (según la opción seleccionada en la pestaña "General" de la ventana de descripción del control). 
<a name="NOTE2_3"></a>


### Fuente predeterminada en la barra de herramientas HTML
<a name="fuente_predeterminada_barra_herramientas_html_ELTPARAGRAPHE000042"></a>

En tiempo de ejecución, la barra de herramientas HTML se inicializa con la fuente, el tamaño y los atributos de fuente especificados en la descripción del control. Esta fuente puede modificarse mediante programación con [las propiedades de gestión de fuentes](../Proprietes/2515045.md). 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Uso de la barra de formato HTML
<a name="uso_barra_formato_html_ELTTEXTE000203"></a>
Esta es la barra HTML predeterminada: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_Barre_WB%20-%20HC%20N%B0003.gif)


Esta barra permite elegir: 

- La fuente y el tamaño. 
	Observación: Estas opciones también están disponibles sin la barra de herramientas.

- La semántica a nivel de texto (div, h1, etc.).

- Las características de la fuente (negrita, subrayado, cursiva, tachado). 

- El color de la fuente.

- El color de resaltado.

- Las opciones de alineación del texto. 

- Las viñetas y la numeración. 

- Las sangrías derecha e izquierda. 

- El formato de subíndice o superíndice. 

- Las características de los enlaces. 

- Las características de las imágenes. 




También puede: 

- insertar tablas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_Barre_Tableau%20-%20HC%20N%B0004.gif)


- elija el estilo de la tabla insertada . <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_Barre_Tableau%20-%20HC%20N%B0006.gif&type=thumb)


- ver el código fuente HTML (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_Barre_WB%20-%20HC%20N%B0003%201.gif)).  




**Observación**: 
Los colores disponibles de forma predeterminada son los definidos por el tema del sitio. Simplemente haga clic en el color deseado. Esto permite obtener colores coherentes en el sitio.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HTML_Barre_WBCouleur%20-%20HC%20N%B0004.gif)


## Menú contextual de las columnas de casillas de verificación (AAF)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000106"></a>
La Tabla y el TreeView controles Tabla pueden proponer columnas Check Box que permiten al usuario seleccionar las opciones solicitadas. 

Dependiendo del número de filas de control, puede ser tedioso para el usuario marcar o desmarcar todas las filas. 

El menú emergente de las columnas Check Box propone varias opciones específicas que simplifican las selecciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Colonne_Interrupteur%20-%20HC%20N%B0002.gif)


- Marcar todo: Se usa para marcar todas las casillas de verificación que se encuentran en la ventana de tabla. 

- Desmarcar todo: Se usa para desmarcar todas las casillas de verificación de tabla. 

- Filtro.... Comprobado sólo: Se utiliza para visualizar únicamente las líneas verificadas en el tabla. 

- Filtro.... No sólo marcado: Se utiliza para visualizar únicamente las líneas no verificadas en el tabla. 




Observación: Estas opciones están disponibles para las columnas de casillas de verificación en: 

- Tabla y TreeView controles Tabla rellenadas programáticamente. 

- Tabla y TreeView controles Tabla con fuente de datos en memoria.

## Programación
<a name="programacion_ELTTEXTE000130"></a>


### "Opciones de"Marcar/Desmarcar
<a name="opciones_demarcardesmarcar_ELTPARAGRAPHE000031"></a>

- Para seleccionar todas las filas del control Tabla o deseleccionar todas las filas del control Tabla o Tabla TreeView, utilice la función [AAFExecute](../WDLang1/1000022099.md) con las constantes *aafCheckAll* y *aafUncheckAll*. En este caso, la función se aplica a la columna Casilla de selección. 

- Para no proponer las opciones que permiten seleccionar y deseleccionar todas las filas del control Tabla o Tabla TreeView, utilice la función [AAFDisable](../WDLang1/1000022018.md) con las constantes *aafCheckAll* y *aafUncheckAll*. En este caso, la función se aplica a la Tabla o al Treeview control Tabla. 

- Para cambiar el texto de las opciones utilizadas para marcar y desmarcar todas las filas de la Tabla o control Tabla TreeView, utilice [AAFChangeCaption](../WDLang1/1000022100.md) con las constantes *aafCheckAll* y *aafUncheckAll*. En este caso, la función se aplica a la Tabla o al Treeview control Tabla. 





### Opciones de filtrado
<a name="opciones_filtrado_ELTPARAGRAPHE000047"></a>

- Para desactivar las opciones de filtrado, utilice [AAFDisable](../WDLang1/1000022018.md) asociada a la constante **aafFilter** de la Tabla o control Tabla TreeView. Atención: En este caso, las opciones de filtrado están desactivadas para la control y sus columnas. 

- Para ejecutar el filtro en la columna Casilla de verificación que se encuentra en la Tabla o control Tabla TreeView, utilice [TableEnableFilter](../WDLang1/1000017149.md) asociado a la constante *filterEqual*. Ejemplo: 
	
	```wl
	// Filter on the unchecked columns
	TableEnableFilter(COL_NoName1, filterEqual, 0)
	```

	```wl
	// Filter on the checked columns
	TableEnableFilter(COL_NoName1, filterEqual, 1)
	```

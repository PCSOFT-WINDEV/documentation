## Presentación
<a name="presentacion_ELTTEXTE000204"></a>
Una ventana maximizada ocupa toda la pantalla del dispositivo móvil. En la asistente se pueden configurar varias opciones para la creación de ventanas. Para iniciar esta asistente: 

- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

- La ventana de creación de nuevos elementos se abre: haga clic en "Ventana", y luego en "Ventana".

- En la ventana de creación asistente, seleccione "Asistente":

## Características de las ventanas maximizadas
<a name="caracteristicas_las_ventanas_maximizadas_ELTTEXTE000228"></a>


### Barra de título
<a name="barra_titulo_ELTPARAGRAPHE000026"></a>

La barra de título se utiliza para mostrar el título de la ventana. Esta barra de título puede mostrarse (o no) mediante "Con barra de título" en la pestaña "Estilo" de la ventana description. 
<a name="NOTE2_2"></a>


### Action Bar
<a name="action_bar_ELTPARAGRAPHE000035"></a>

Android 3 le permite insertar un Action Bar en sus aplicaciones. 

Las "Barras de acciones" sustituyen a la barra de título de una ventana y sirven para mostrar diversa información:

- el ícono de la aplicación,

- el título de la ventana,

- algunas opciones de menú,

- ...


Para obtener más información, consulte [Action Bar de una ventana](../WDChamp/1000025001.md).
<a name="NOTE2_3"></a>


### Pestañas
<a name="pestanas_ELTPARAGRAPHE000054"></a>

Las pestañas se utilizan para mostrar información sobre los diferentes paneles. El usuario sólo tendrá que elegir la pestaña solicitada.

Para obtener más información sobre este control, consulte [Control Pestaña](../WDChamp/1013194.md).
<a name="NOTE2_4"></a>


### barra de desplazamiento
<a name="barra_desplazamiento_ELTPARAGRAPHE000066"></a>

Las barras de desplazamiento se mostrarán automáticamente si el tamaño de la ventana es superior a la resolución de la pantalla del dispositivo Android utilizado.

Esta opción puede elegirse en cualquier momento en la ventana description (pestaña "Detalles").

**Atención**: En el modo de pantalla completa, las barras de desplazamiento de automatic nunca se muestran en una ventana maximizada.
<a name="NOTE2_5"></a>


### Menú
<a name="menu_ELTPARAGRAPHE000077"></a>

En una aplicación para Android, este menú se encuentra en la parte inferior de las ventanas. Un menú desplegable permite al usuario acceder rápidamente a una de las funciones de la aplicación.

En la edición, los menús de las ventanas de Android se presentan como simples menús desplegables.

En tiempo de ejecución, estos menús se mostrarán de forma diferente según el número de opciones:

- Si el menú contiene menos de 6 opciones, se muestra como botones grandes en la parte inferior de la pantalla.

- Si el menú contiene más de 6 opciones, las 5 primeras se muestran como botones grandes y un sexto botón automatic ("Más") permite al usuario ver el resto del menú como un desplegable.




Los menús de las aplicaciones Android admiten un único nivel de submenús. Más allá de este límite, se produce un error fatal al ejecutar la aplicación.

La adición de un menú desplegable a una ventana o la eliminación de un menú desplegable de una ventana se puede hacer en cualquier momento:

- en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Menú principal" y seleccione "Agregar menú principal".

- en la pestaña "Estilo" de la ventana description ("Activar menú principal").

## Tamaño de una ventana maximizada
<a name="tamano_una_ventana_maximizada_ELTTEXTE000276"></a>
El tamaño utilizable de una ventana maximizada depende de los elementos que se tienen en cuenta en la ventana (barra de título Visible o no, ...).

WINDEV Mobile le permite configurar todos estos elementos en sus ventanas para Android: 

- La pestaña "Estilo" de la ventana description permite configurar: 

	- La visualización de la barra de título 

	- La visualización de la barra del sistema




- La pestaña "Detalles" de la ventana description permite configurar el tamaño de la ventana:

## Opciones adicionales
<a name="opciones_adicionales_ELTTEXTE000300"></a>
La pestaña "Detalles" también permite configurar la gestión de la orientación de la ventana. En efecto, cuando el usuario modifica la orientación de su dispositivo Android, la ventana puede: 

- seguir la orientación del dispositivo. En este caso, los anclajes le permiten definir una ventana que se puede utilizar en modo vertical y en modo horizontal.. 

- estar bloqueado en el modo de retrato.

- estar bloqueado en modo apaisado.




**Observación**: Cuando se cambia el modo de visualización (de retrato a paisaje o a la inversa), el nuevo tamaño de la ventana es representado por un Line punteado en el editor de la ventana. Todo lo que tiene que hacer es cambiar el tamaño de la ventana a través de las manijas y posicionar los controles.

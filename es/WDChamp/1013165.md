
## Escogiendo la acción associated con el enlace
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000164"></a>
Puede asociar una acción al enlace independientemente del tipo de enlace. WEBDEV ofrece las acciones más comunes en la pestaña general de la ventana de control description.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Btn_Action.gif)


Para acciones más específicas, haga clic en "Más opciones".

En cualquier caso, deberá definir el destino de esta acción.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las acciones disponibles
<a name="las_acciones_disponibles_ELTTEXTE000188"></a>


### Acciones predefinidas
<a name="acciones_predefinidas_ELTPARAGRAPHE000024"></a>

Varias acciones preestablecidas son propuestas por Default en un cuadro de lista desplegable. Estas acciones preestablecidas dependen de los elementos de la Page o del proyecto.

Por ejemplo, usted puede elegir:

- ejecutar el código de clic de una control Enlace (código del navegador o código del servidor y del navegador),

- mostrar una Page del sitio,

- mostrar una Page de otro sitio, 

- realizar una acción personalizada (enviar un correo electrónico, llamar a un número de Phone, conectarse a un servidor FTP, descargar un archivo, etc.). En este caso, se pueden especificar diferentes parámetros. 




**Atención**: Las acciones se realizan después del código de clic del enlace (código del navegador y del servidor)
<a name="NOTE2_2"></a>


### Acciones específicas
<a name="acciones_especificas_ELTPARAGRAPHE000040"></a>

El enlace "Más opciones" muestra más opciones. Puedes, por ejemplo:

- realizar una acción en otro control, 

- set la posición en uno de los anclajes Page, 

- compartir una red social,

- cambiar idioma actual

- enviar el valor de los controles Page a ... una Page o un guión. Por ejemplo, esta opción puede utilizarse para Process el contenido de un formulario en un sitio estático.




**Observación**: Todas estas operaciones también pueden realizarse en el código.
<a name="NOTE2_3"></a>


### Caso especial: compartir en las redes sociales
<a name="caso_especial_compartir_las_redes_sociales_ELTPARAGRAPHE000057"></a>

La acción "Compartir en una red social" permite crear un enlace para compartir la Page actual, el dominio actual o una URL determinada en una red social.. Esta acción también puede realizarse a través de la [Red Social](../WDChamp/1000028001.md) control. 

Observación: El uso de un botón, un enlace o un Image pulsable le permite configurar completamente el estilo utilizado (sin embargo, le aconsejamos que cumpla con los acuerdos de licencia de las diferentes redes sociales). El [Red Social](../WDChamp/1000028001.md) control dispone de ajustes avanzados (por ejemplo, mostrar el número de acciones, etc.). 

El tabla a continuación presenta la información utilizada para cada red social: 

| Red Social | Elemento compartido | Efecto |
| --- | --- | --- |
| **Facebook** | El dominio | Crea una Post en Facebook con un enlace al dominio de la página web actual.<br><br>Atención: Debe existir un índice Default Page en el servidor de alojamiento. |
|   | La Page | Crea una Post en Facebook con un enlace a la web actual Page. |
|   | Destino personalizado | La URL debe escribirse en la "URL de destino" control.<br><br>**Caso 1: URL de una página Facebook**: <br>Incrementa el contador de Page de Facebook que ha gustado.<br>**Caso 2: URL de una página o sitio cualquiera**: <br>Crea una Post de Facebook con un enlace a la Page o página web especificada. |
| **Gorjeo** | El dominio | Crea un Tweet con un enlace al dominio de la página web actual. |
|   | La Page | Crea un Tweet con un enlace a la actual Page. |
|   | Destino personalizado | La URL debe escribirse en la "URL de destino" control.<br><br>Crea un Tweet con un enlace a la URL especificada. |
| **LinkedIn** | El dominio | Crea un LinkedIn Post con un enlace al dominio.<br><br>Atención: Debe existir un índice Default Page en el servidor de alojamiento. |
|   | La Page | Crea una LinkedIn Post con un enlace a la web Page. |
|   | Destino personalizado | Crea una Post LinkedIn con un enlace personalizado.<br><br>La URL del enlace a enviar debe introducirse en la "URL de destino" control. |
| **Tumblr** | El dominio | Crea un Tumblr Post con un enlace al dominio de la página web actual. |
|   | La Page | Crea un Tumblr Post con un enlace a la web actual Page. |
|   | Destino personalizado | La URL debe escribirse en la "URL de destino" control.<br><br>Crea un Tumblr Post con un enlace a la URL especificada. |
| **WhatsApp** | La Page | Crea un mensaje de WhatsApp con un enlace a la actual Page. |
|   | Destino personalizado | La URL debe escribirse en la "URL de destino" control.<br><br>Crea un mensaje de WhatsApp con un enlace a la URL especificada. |


El campo "Destino" permite seleccionar el modo de visualización de la ventana de conexión y entrada en la red social.

## Destino de la acción
<a name="destino_accion_ELTTEXTE000323"></a>
Si no se elige ningún destino para la acción, ésta se realiza en el destino Default definido para el Page actual.

Los objetivos disponibles para una acción realizada en un enlace son:


|   |   |
| --- | --- |
| **Nuevo navegador (_blank)**:<br>Muestra el resultado en una nueva ventana del navegador. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_nouveau.gif)<br> |
| **Fotograma actual (_self)**:<br>Muestra el resultado en el mismo cuadro. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_FrameEnCours.gif)<br> |
| **Marco principal (_parent)**:<br>Muestra el resultado en el marco padre (caso de conjuntos de marcos anidados). | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_FrameParent.gif)<br> |
| **Navegador actual (_top)**:<br>Muestra el resultado en la Page actual del navegador (sin marcos). | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dest_NavEnCours.gif)<br> |
| **Otros marcos del conjunto de marcos** (si el Page pertenece a un conjunto de marcos):<br>Muestra el resultado en un marco específico. |   |
| **iFrame control de la página** (si la página contiene al menos un iFrame control ):<br>Muestra el resultado en una [iFrame control](../WDChamp/1013270.md). |   |

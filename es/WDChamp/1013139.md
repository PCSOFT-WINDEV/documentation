
## Dibujar gráficos en una control Imagen
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
WINDEV, WINDEV Mobile y WEBDEV proponen varias funciones para crear y manejar gráficos.

Se pueden crear varios tipos de gráficos: Gráficos de tarta, columna, Line y Scatter, ... Las opciones de visualización se utilizan para producir diferentes efectos visuales para cada tipo de gráfico.

Estos gráficos se pueden dibujar:

- en un archivo de Image,

- en un control Imagen. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una ventana, 

- en una variable de tipo [Image](../WDLang1/1000019650.md).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo dibujar?
<a name="¿como_dibujar_ELTTEXTE000133"></a>
Para dibujar un gráfico en un control Imagen, se deben cumplir algunas reglas:

1. Cree el gráfico con [grCreate](../WDLang3/3042005.md).

2. Inicialice los datos del gráfico con las funciones [grAddData](../WDLang3/3042023.md) y [grScatterAddDataXY](../WDLang3/3042032.md).

3. Elija una control Imagen como destino de la carta ([grDestinoControl](../WDLang3/3042020.md))

4. Dibuja el gráfico con [grDraw](../WDLang3/3042026.md).




Ejemplo:


```wl
grCreate("Chart1",grPie)
grAddData("Chart1",1,50)
grAddData("Chart1",1,90)
grAddData("Chart1",1,500)
grDestinationControl("Chart1", IMG_Image1)
grDraw("Chart1")
```

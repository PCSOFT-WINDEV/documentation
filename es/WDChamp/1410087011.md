
## Manejo de una nota reposicionable control mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000171"></a>
Una Nota Reposicionable control puede ser manejada a través de la programación. Para ello, utilice la Variable de la nota reposicionable control en el código. 

También se pueden utilizar funciones específicas para reposicionar notas de manipular.. Para obtener más información, consulte [Funciones Notas reposicionables](../WDLang1/1000022398.md). 

Esta página de ayuda explica cómo manipular un billete reposicionable control mediante programación.

<a name="NOTE2_1"></a>


## Mostrando una información en una Nota Reposicionable control
<a name="mostrando_una_informacion_una_nota_reposicionable_control_ELTTEXTE000195"></a>
Para mostrar una información en una Nota Reposicionable control, una simple asignación es suficiente.

Por ejemplo:

- asignación directa:
	
	```wl
	NOTE_CustomerNote = "Important customer"
	```


- asignación utilizando una variable:
	
	```wl
	NOTE_CustomerNote = Note_Customer
	```





**Observación**: Si el control está vinculado a un elemento un archivo de datos, el valor de la fuente de datos se actualiza automáticamente en el control mediante el [FileToScreen](../WDLang4/3044210.md).
También puede utilizar la siguiente sintaxis: "NOTE_CustomerNote = CUSTOMER.NOTE". Para obtener más información, consulte [Vincular una nota reposicionable control a un artículo](../WDChamp/1410087012.md).

**Atención**: La Nota Reposicionable control no incluye barra de desplazamiento. El texto se truncará si supera el tamaño de la note reposicionable. El tamaño de control se puede modificar (con las propiedades [Altura](../Proprietes/2510050.md) y [anchura](../Proprietes/2510059.md)).

## Recuperando el valor mostrado en una Nota Reposicionable control
<a name="recuperando_valor_mostrado_una_nota_reposicionable_control_ELTTEXTE000219"></a>
Para recuperar el contenido de una Nota Reposicionable control:

- realice una asignación simple. 
	Por ejemplo:
	
	```wl
	CurrentNoteContent = NOTE_CustomerNote
	```


- utilice la propiedad [Valor](../Proprietes/2510130.md).




**Observación**: Si el control está enlazado a un campo de un archivo de datos, el valor introducido en el control se actualiza automáticamente en la fuente de datos utilizando la función [ScreenToFile](../WDLang4/3044146.md).
También puede utilizar la siguiente sintaxis: "CUSTOMER.Note = NOTE_CustomerNote". Para obtener más información, consulte [Vincular una nota reposicionable control a un artículo](../WDChamp/1410087012.md).

## Forzar la exportación o importación de billetes
<a name="forzar_exportacion_importacion_billetes_ELTTEXTE000243"></a>
El usuario puede importar o exportar las notas (que se encuentran en la ventana o en la aplicación) en un fichero extension "wdnotes" a través del menú contextual de la ventana control o Nota. 

Tiene la posibilidad de forzar la exportación o importación de las notas mediante la programación con [AAFExecute](../WDLang1/1000022099.md): 

- la constante *aafExportNote* se utiliza para forzar la exportación de las notas que se encuentran en la ventana especificada. El usuario debe seleccionar el archivo que contiene las notas. 

- la constante *aafImportNote* se utiliza para forzar la importación de los billetes. El usuario debe seleccionar el archivo que contiene las notas. 




**Atención**: la exportación sólo puede realizarse si se ha modificado el contenido de la note. 

Consejo: Para importar y exportar todas las notas reposicionables que se encuentran en una ventana, utilice [NoteLoadAll](../WDLang1/1000022397.md) y [NoteSaveAll](../WDLang1/1000022396.md).

## Propiedades específicas de los mandos Reposicionables note
<a name="propiedades_especificas_los_mandos_reposicionables_note_ELTTEXTE000267"></a>
Las siguientes propiedades se utilizan para gestionar las características de una Nota Reposicionable control mediante programación.




|   |   |
| --- | --- |
| [AutomaticallyMinimized](../Proprietes/1000021963.md) | El **AutomáticamenteMinimizado** Property se utiliza para: <br><br>- Averigüe el tiempo de espera antes de que la Nota Reposicionable control se minimice automáticamente.<br><br>- Definir el tiempo de espera antes de que la Nota Reposicionable control se minimice automáticamente. <br><br><br> |
| [Minimized](../Proprietes/1000022005.md) | El **minimizado** Property se utiliza para: <br><br>- averiguar si un billete reposicionable se minimiza, <br><br>- minimizar o restaurar una nota reposicionable.<br><br><br> |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una nota reposicionable control, consulte [Propiedades associated con controles note reposicionables](../WDChamp/1410087010.md).

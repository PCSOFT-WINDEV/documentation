## Resumen de Control deslizante
<a name="resumen_control_deslizante_ELTTEXTE000134"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

Un deslizador control se utiliza para seleccionar un valor entre dos límites. Su modo de funcionamiento es similar al modo de funcionamiento de una barra de desplazamiento.. El valor actual se identifica con un cursor que el usuario mueve con el ratón.

WEBDEV propone dos tipos de deslizadores:

- **control deslizante**: Hecho de una regla y un cursor, este control es muy útil para permitir al usuario de la web introducir un valor en una interval determinada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Potentio_Lin_WB.gif)


- **control deslizante de rango**: Este deslizador es un deslizador lineal con 2 cursores. Permite al usuario de la web definir una interval de valores. Por ejemplo, en un sitio de negocios, esta control permite al usuario de la Web definir un rango de precios para filtrar los productos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Potentio_Intervalle_WB.gif)

## Creación de una Control deslizante
<a name="creacion_una_control_deslizante_ELTTEXTE000158"></a>
Para crear una Control deslizante: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Control deslizante" y seleccione el tipo de control deslizante que desea crear: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Potentio_Creation_WB.gif)


2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: WINDEV también propone un Control deslizante. Para obtener más información, consulte [Control deslizante](../WDChamp/1013241.md).

## Características de la Control deslizante
<a name="caracteristicas_control_deslizante_ELTTEXTE000182"></a>
La pestaña "General" del Control deslizante permite especificar las principales características del control: 

- Orientación: Tiene la posibilidad de utilizar un deslizador horizontal o un deslizador vertical. 

- Límite mínimo: Límite inferior para los valores del control deslizante. 

- Límite superior: Límite superior para los valores del control deslizante.  




**Sólo para la corredera lineal**: 

- Valor inicial: Posición inicial del cursor de la barra deslizante. 




**Sólo para el control deslizante de la gama**: 

- Valor inferior: Posición de inicio para el cursor inferior del control deslizante. 

- Valor superior: Posición de inicio para el cursor superior del control deslizante. 





### Orientación de la Control deslizante
<a name="orientacion_control_deslizante_ELTPARAGRAPHE000075"></a>

Una Control deslizante se puede orientar horizontal o verticalmente. 

Para modificar la orientación de un Control deslizante: 

1. Seleccione la Control deslizante en la página. 

2. Seleccione "Elegir un estilo WEBDEV" en el menú contextual de control. 

3. En la ventana que se muestra, seleccione un estilo de deslizador vertical. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WB_Style_Potentiometre.gif&type=thumb)


4. Validar.

## Resumen de control Cinta de opciones
<a name="resumen_control_cinta_opciones_ELTTEXTE000195"></a>
La control Cinta de opciones propone una característica adicional con respecto a los menús que se encuentran en sus aplicaciones. Se utiliza para presentar en una pequeña superficie un gran número de controles dispuestos lógicamente para realizar elecciones. 

Por ejemplo, los diferentes menús de la aplicación "Gestión de ventas" presentados en cinta: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Ruban1.gif)
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Ruban2.gif)
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Ruban3.gif)

En este ejemplo, los diferentes elementos de la control Cinta de opciones son botones pero se puede usar cualquier tipo de control. 

En tiempo de ejecución, más o menos elementos de la cinta se muestran según el tamaño de la ventana. Si no se pueden mostrar algunos elementos, se utiliza una pequeña flecha para "expandir la cinta" y para mostrar todos los elementos en una ventana emergente.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Ruban4.gif)
**Notas**: 

- La tecla TAB pulsada en un control Campo de entrada que se encuentra en el control Cinta de opciones se utiliza para ir al siguiente control (según el orden de tabulación). 

- La tecla ENTER pulsada en un control Campo de entrada que se encuentra en el control Cinta de opciones no tiene ningún efecto.

## Creación de una control Cinta de opciones
<a name="creacion_una_control_cinta_opciones_ELTTEXTE000219"></a>
Para crear una control Cinta de opciones:

1. En la pestaña "Creación", en el grupo "Contenedores", despliegue "Pestaña y asociados" y haga clic en "Cinta de opciones".
	Otra posibilidad: En la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Otras barras" y haga clic en "Cinta de opciones".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor. 

3. Definir las características de la control Cinta de opciones. Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

4. Inserte los controles deseados en las diferentes agrupaciones. Todos los tipos de controles pueden incluirse en una agrupación. 




Para transformar el menú contextual de una ventana en un control Cinta de opciones: 

1. Abrir la ventana que contiene el menú principal. 

2. En la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Menú principal a Cinta de opciones".


Atención: Para utilizar esta opción, la control Cinta de opciones no debe existir en la ventana.

## Características de control Cinta de opciones
<a name="caracteristicas_control_cinta_opciones_ELTTEXTE000243"></a>
La control Cinta de opciones incluye paneles y grupos. 

Cada panel es representado por un ícono y un título. 

Cada panel contiene uno o más grupos separados por una Line vertical. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_Volet_regroupement.gif)



### description de una control Cinta de opciones
<a name="description_una_control_cinta_opciones_ELTPARAGRAPHE000103"></a>

La ventana de la description de una control Cinta de opciones le permite: 

- definir los diferentes paneles de la pestaña control Cinta de opciones (pestaña"General"). Para cada panel, tiene la capacidad de definir: 

	- la leyenda del panel,

	- su estado inicial,

	- la Image que se muestra junto a la leyenda. 
			La posición de esta imagen puede ser modificada en la pestaña "Estilo" (elementos "Pie de foto del panel activo" y "Pie de foto de los paneles de Inactive").  

	- el tooltip del panel. 




- definir las diferentes agrupaciones que se muestran en cada panel (pestaña"General"). Para cada agrupación, tiene la capacidad de definir: 

	- el título de la agrupación. Esta leyenda se mostrará en la parte inferior de la agrupación cuando ésta se abra.. 

	- si el panel muestra un botón específico ("Con el botón (haga clic en Process)"). 
			En este caso, el clic Process de agrupación se ejecutará cuando el usuario haga clic en este botón. 
			Esto le permite mostrar un menú emergente o una ventana específica, por ejemplo, ....

	- la Image que se mostrará cuando se reduzca la agrupación. Las dimensiones de esta imagen deben ser set a 32 x 32. 







**Observaciones**: 

- La imagen utilizada para el dibujo de los paneles se puede personalizar en la ventana de control de description (pestaña "Detalles"). 

- En la ventana de control de description (pestaña "Estilo") hay varias opciones para personalizar los paneles y las agrupaciones. 





### Uso de los botones en una control Cinta de opciones
<a name="uso_los_botones_una_control_cinta_opciones_ELTPARAGRAPHE000131"></a>Si un control Botón está en el grupo de un control Cinta de opciones, la pestaña "UI" del control Botón permite configurar el comportamiento del botón cuando la cinta se reduce. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ruban_Champ_Bouton.gif)
Es posible: 

- evitar la reducción de la control. 

- reducirlo para que aparezca el botón Image y el texto. 

- redúzcalo para que aparezca el botón Image. 




Las dos últimas opciones son útiles para los botones que muestran una Icon y una etiqueta específicas. 

**Observación**: Tiene la posibilidad de utilizar [botones con flecha](../WDChamp/1013163.md) en un control Cinta de opciones, permitiendo por ejemplo clasificar las acciones para realizarlas jerárquicamente.

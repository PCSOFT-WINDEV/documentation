


## Control Lector PDF: Cómo utilizar la cinta de opciones y las pestañas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000347"></a>
El control Lector PDF incluye varias funcionalidades disponibles en: 

- La cinta de opciones. Hay varias pestañas disponibles: 

	- La pestaña "Vista", que permite realizar las operaciones más comunes en el archivo PDF. 

	- La pestaña "Buscar", que se utiliza para realizar búsquedas en el archivo PDF. 

	- La pestaña "Edición", que permite editar el contenido del archivo PDF. 




- El panel del editor, que muestra las miniaturas de las diferentes páginas. 




Observación: En la barra de desplazamiento de control Lector PDF se muestra un tooltip que permite ver el número de la página actual y el número total de páginas en cualquier momento..

<a name="NOTE2_1"></a>


## Cinta: Pestaña "Vista"
<a name="cinta_pestana_vista_ELTTEXTE000371"></a>
La pestaña "Vista" permite realizar las operaciones más comunes en un archivo PDF. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%201.gif&type=thumb)
Esta pestaña contiene los siguientes grupos: 

- **General**: Permite abrir, imprimir o guardar un documento PDF.

- **Portapapeles**: Permite gestionar el portapapeles (copia). 

- **Zoom**: Permite gestionar el zoom del documento PDF.

- **Modo de visualización**: Permite gestionar las opciones de visualización.  

- **Páginas**: Permite desplazarse por las diferentes páginas del documento PDF. 

- **Herramientas**: Permite set la acción del ratón.  





### Grupo "General"
<a name="grupo_general_ELTPARAGRAPHE000043"></a>

El grupo **General** permite abrir, imprimir o guardar un documento PDF. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%202.gif)
Es posible: 

- **Abrir un archivo PDF existente**. 

	- Al abrir un archivo PDF, se conservan los permisos del archivo (permisos para copiar, imprimir, etc.). 

	- Si el archivo PDF está protegido por una contraseña, el usuario debe introducir la contraseña correspondiente. 




- **Imprimir el archivo PDF actual**. La ventana de configuración de la impresión se abre (con una vista previa del documento). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Imprimer%20-%20HC%20N%B0001.gif&type=thumb)
Es posible configurar: 

	- las propiedades de la impresora: Nombre de la impresora, orientación, modo de impresión, número de copias. 

	- las páginas a imprimir: todas las páginas, la página actual o una selección de páginas (páginas separadas por "-" para una selección, páginas separadas por ',' para páginas independientes). 

	- el redimensionamiento de las páginas: reducir las páginas de gran tamaño, ajustar la página o utilizar el tamaño real. También puede elegir el origen del papel según el formato de la página. 


 También es posible:

- previsualizar las diferentes páginas a imprimir. 

- exportar como un correo electrónico con el archivo PDF adjunto (botón "Exportar"). 

- **Guardar el archivo PDF actual**. Esta opción permite guardar el archivo PDF con sus cambios (incluyendo la orientación de la página). Puede guardar el archivo actual o utilizar la opción "Guardar como". 
	**Observación**: Esta funcionalidad está disponible a partir de la versión 250064 (Update 2). Si utiliza la ventana predeterminada, el cambio se realiza automáticamente cuando se recompila el proyecto. Si personalizó la ventana interna, es necesario comparar las diferencias entre la ventana interna "IW_WindevFAA_PDFReaderRibbon" del componente WDAAF y la que se encuentra en el directorio "Programs\\Data\\Preset windows\\FAA". Para obtener más información, consulte [Comparador de proyectos](../Editeurs/2030024.md).    




Observación: Si el archivo PDF está firmado, la pestaña "Vista" muestra la información específica. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0004.gif)
 Para obtener más información, consulte [Firma de documentos](#NOTE4_4). 


### Grupo "Portapapeles"
<a name="grupo_portapapeles_ELTPARAGRAPHE000087"></a>

El grupo **Portapapeles** permite copiar el contenido de la selección. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%203.gif)
Este contenido se puede pegar en otra aplicación (bloc de notas, etc.). 

Observaciones: 

- Solo es posible copiar el contenido si el archivo PDF abierto permite la copia. 

- Para seleccionar una parte del documento PDF, debe estar en "Modo de selección". 

- La copia también puede realizarse a través del menú contextual de la selección. 





### Grupo "Zoom"
<a name="grupo_zoom_ELTPARAGRAPHE000100"></a>

El grupo **Zoom** permite establecer el zoom en un documento PDF. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%204.gif)
Es posible: 

- Mostrar el documento PDF con una escala de 100%. 

- Mostrar el documento PDF para que el ancho de página se muestre por completo.

- Mostrar el documento PDF en formato de página entera. 

- Personalizar el zoom. 





### Grupo "Modo de visualización"
<a name="grupo_modo_visualizacion_ELTPARAGRAPHE000112"></a>

El grupo **Modo de visualización** permite definir las opciones de visualización del documento PDF actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%205.gif)
Es posible: 

- Mostrar una sola página. 

- Mostrar dos páginas. En este caso, puede utilizar una página de portada. La primera página se mostrará a la derecha.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Imprimer%20-%20HC%20N%B0002.gif&type=thumb)



Observación: la visualización de una portada modifica automáticamente el zoom para que se muestre toda la página. 


### Grupo "Páginas"
<a name="grupo_paginas_ELTPARAGRAPHE000125"></a>

El grupo **Páginas** permite moverse dentro de un documento PDF. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0001%206.gif)
Es posible: 

- Mostrar u ocultar el panel de miniaturas. Este panel muestra las diferentes páginas del documento PDF como miniaturas y permite desplazarse entre ellas. 

- Ver el número total de páginas en el documento. 

- Ver el número de la página actual. 

- Desplazarse por las distintas páginas con los botones de flecha.  





### "Grupo "Herramientas
<a name="grupo_herramientas_ELTPARAGRAPHE000137"></a>

El grupo **Herramientas** le permite set la acción del mouse. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0002.gif)


- Si se selecciona la herramienta "Mover", el cursor se transforma en una mano Icon y permite desplazarse rápidamente por las diferentes páginas del fichero PDF. 

- Si la herramienta "Seleccionar" está seleccionada, el cursor permite seleccionar y resaltar texto. Dependiendo de los permisos del archivo, puede copiar el texto seleccionado. 

- Si la herramienta "OCR" está seleccionada, puede arrastrar sobre un área para ejecutar el OCR. Se analiza el contenido del área y se copia el texto correspondiente en el portapapeles.

## Cinta: Pestaña "Buscar"
<a name="cinta_pestana_buscar_ELTTEXTE000431"></a>
La pestaña "Buscar" permite realizar búsquedas en el documento PDF actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Ruban%20-%20HC%20N%B0003.gif)
Para iniciar la búsqueda, simplemente escriba el texto buscado en el área de búsqueda. Puede especificar las opciones de búsqueda: 

- **Palabra completa**: Busca solo la palabra especificada. No se buscarán las palabras que contengan la palabra especificada. Por ejemplo, si en la búsqueda se especifica la palabra "niño" con la opción "Solo palabras completas", no se encontrará la palabra "niños". 

- **Distinguir mayúsculas y minúsculas**: Distingue entre mayúsculas y minúsculas en el cuadro de búsqueda.  Por ejemplo, si en la búsqueda se especifica la palabra "niño" con la opción "Distinguir mayúsculas y minúsculas", no se encontrará la palabra "Niño". 


Puede utilizar las flechas para desplazarse por los diferentes resultados de la búsqueda. Si la búsqueda se completa, se muestra un mensaje.

## Cinta: Pestaña "Edición"
<a name="cinta_pestana_edicion_ELTTEXTE000455"></a>
La pestaña "Edición" permite editar el documento PDF actual.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%201.gif)


Esta pestaña contiene los siguientes grupos: 

- **Rotación**, que permite girar las páginas del documento PDF. 

- **Páginas**, que permite eliminar o importar páginas.

- **Insertar**, que permite agregar una marca de agua o firma.



<a name="NOTE4_2"></a>


### Grupo "Rotación"
<a name="grupo_rotacion_ELTPARAGRAPHE000182"></a>

El grupo Rotación permite girar las páginas en el documento PDF. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%202.gif)
 Puede girar todas las páginas del Document mostradas en el lector de PDF. 
Observaciones: 

- Para girar la Page actual , abra el menú contextual de la Page y seleccione la opción deseada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_LecteurPDF_Menu%20-%20HC%20N%B0001.gif)


- Puede guardar la Document con las rotaciones (botón "Guardar" del grupo "General").



<a name="NOTE4_3"></a>


### Grupo "Páginas"
<a name="grupo_paginas_ELTPARAGRAPHE000197"></a>

El grupo "Páginas" permite insertar o suprimir páginas en el PDF Document. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%203.gif)


Usted puede: 

- Borrar la Page actual . 
	Observación: La Page actual puede suprimirse desde su menú contextual (opción "Suprimir Page"). 

- Borrar una set de páginas. En este caso, se mostrará una ventana para indicar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0004.gif)


	- el número de Page (s) a eliminar (separados por un punto y coma),

- el rango de páginas a eliminar: indique simplemente la primera y la última Page que desea suprimir. 

- el rango de páginas que se deben conservar: en este caso, se borrarán todas las páginas EXCEPTO el rango especificado. 

- Importar páginas: La ventana de configuración de la importación le permite: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0005.gif&type=thumb)


	- seleccione el archivo PDF que contiene las páginas que se van a importar. 

- indicar las páginas que se van a importar. Puede importar una o varias páginas, un rango de páginas o todas las páginas. 

- indicar la posición de las páginas importadas: al principio o al final de la Document, o antes de la Page actual .  




Puede guardar la Document con los cambios realizados (botón "Guardar" del grupo "General").

### "Insertar grupo
<a name="insertar_grupo_ELTPARAGRAPHE000227"></a>

El grupo "Insertar" permite insertar una Watermark o una firma en el PDF Document. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ%20Lecteur%20PDF_Volet_modification%20-%20HC%20N%B0003%204.gif)


Usted puede: 

- Agregar una marca de agua. 

- Añadir una o varias firmas manuscritas. 

- Añadir una o más firmas digitales. 




**Agregar un Watermark**
Cuando se selecciona la opción "Añadir una Watermark", aparece una ventana que indica: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0001.gif)


- el posicionamiento de la Watermark.

- el texto de Watermark.

- el tipo de letra, el tamaño y la color del texto.
	Atención: el tipo de letra debe estar ya utilizado en el archivo PDF. De lo contrario, se utilizará automáticamente "Arial"..  

- las opciones de transparencia de Watermark.




**Agregar una o más firmas manuscritas**
Para incluir firmas manuscritas, amplíe "Firma manuscrita".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0002.gif)
Es posible: 

- Añade una nueva firma.

- Utilizar una firma existente.  

- Borrar una firma existente. 




Cuando se añade una firma, aparece una ventana específica. Esta ventana le permite: 

- Seleccione el archivo Image que contiene la firma.

- Firmar con el ratón o con un lápiz óptico. 
	En este caso, la color de la pluma no se puede cambiar. 

- Escriba la firma en un área de texto, eligiendo el tipo de letra, el tamaño, etc.




Cuando se selecciona una firma existente, ésta aparece en la página actual en el lector de PDF control. Se puede mover y posicionar.  
Esto le permite utilizar varias firmas manuscritas: una firma completa y las iniciales, por ejemplo. Las iniciales pueden colocarse en la parte inferior derecha de las diferentes páginas.

**Agregar una firma digital**
En este caso, se requiere un PDF que firme Certificate. 
Al añadir una firma digital, aparece una ventana específica. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0003.gif)
Esta ventana permite: 

- Seleccione la Certificate para firmar el archivo PDF. 

- Seleccione la firma manuscrita que se associated con la firma digital. 




Atención: 

- Una firma digital requiere una firma Certificate y una firma manuscrita. 

- Para que Acrobat reconozca un archivo PDF firmado, la organización que crea el Certificate debe estar en la lista de confianza aprobada por Adobe o en el programa de servicios Document certificados (CDS). 




Cuando se abre un archivo PDF firmado digitalmente, aparece un mensaje en la esquina superior izquierda del PDF Reader control. Los usuarios pueden ver las características de la firma haciendo clic en este mensaje.

**Guardar un Document que contiene firmas**
Puede guardar la Document con los cambios realizados (botón "Guardar" del grupo "General"). 

**Abriendo un Document que contiene firmas digitales**: 
Al abrir un archivo PDF firmado digitalmente, aparece una nueva zona en la pestaña "Ver", que indica que el Document está firmado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0004.gif)
 Los usuarios pueden hacer clic en el enlace "firma" para ver los detalles de la firma. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Lecteur_PDF_Signature%20-%20HC%20N%B0005.gif&type=thumb)
 En esta ventana, el usuario puede hacer clic en el enlace para ver los detalles de la Certificate.


## Manejo de una barra de desplazamiento control mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000140"></a>
WINDEV le permite manipular una barra de desplazamiento control a través de la programación. Para ello, usa la Variable de la barra de desplazamiento control en el código.

El Variable de la barra de desplazamiento control:

- corresponde al nombre de la barra de desplazamiento.

- se inicializa con la posición de la barra de desplazamiento.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperación de la posición del cursor
<a name="recuperacion_posicion_del_cursor_ELTTEXTE000164"></a>
Para recuperar la posición del cursor en una barra de desplazamiento control, utilice la siguiente sintaxis:


```txt
<Position> = <Scrollbar Name>
```


&lt;Posición&gt; es un entero Variable

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificar la posición del cursor
<a name="modificar_posicion_del_cursor_ELTTEXTE000188"></a>
Para modificar la posición del cursor en una barra de desplazamiento control, utilice la siguiente sintaxis:


```txt
<Scrollbar Name> = <value>
```


&lt;Valor&gt; es un valor que se encuentra entre el valor mínimo y máximo de la barra de desplazamiento. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

## Propiedades específicas de la gestión de una barra de desplazamiento control
<a name="propiedades_especificas_gestion_una_barra_desplazamiento_control_ELTTEXTE000212"></a>
Las siguientes propiedades se utilizan para gestionar las características de una barra de desplazamiento control mediante programación.


|   |   |
| --- | --- |
| [Desplazarse](../Proprietes/1000021574.md) | Se utiliza para averiguar y modificar el paso de desplazamiento de una barra de desplazamiento control. |
| [ScrollValue](../Proprietes/2510011.md) | Se utiliza para averiguar y modificar el valor de desplazamiento de una barra de desplazamiento control. |
| [MaxValue](../Proprietes/2510009.md) | Se utiliza para averiguar y modificar el límite máximo de una barra de desplazamiento control. |
| [MinValue](../Proprietes/2510008.md) | Se utiliza para averiguar y modificar el límite mínimo de una barra de desplazamiento control. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una barra de desplazamiento control, consulte [Propiedades associated con controles Scrollbar](../WDChamp/1013112.md).

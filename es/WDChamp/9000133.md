


## Ventana del iPhone: asistente de creación de ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000188"></a>
La asistente para la creación de ventanas de iPhone permite crear fácilmente una ventana que contenga todos los elementos principales de una ventana para una aplicación de iPhone. 

Estos diferentes elementos se pueden añadir en cualquier momento en la ventana. 

También puede configurar diferentes opciones en la ventana description. 

Al final de la asistente, se crea la ventana bajo el editor con las características especificadas. Además de la ventana, puede ver los siguientes elementos en el editor: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen%EAtre_iphone_Editeur%20-%20HC%20N%B0001.gif&type=thumb)


- la barra del sistema, 

- las zonas que corresponden al modo vertical u horizontal (representadas por guiones), 

- la ubicación de la barra de inicio (del iPhone X). Se recomienda no colocar ninguna control clicable (botón, etc.) en este lugar.

## Los diferentes pasos de la asistente
<a name="los_diferentes_pasos_asistente_ELTTEXTE000212"></a>


### Orientación de la ventana
<a name="orientacion_ventana_ELTPARAGRAPHE000031"></a>

Hay varias opciones disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_Orientation.gif)
 Estas opciones pueden modificarse en el editor, en la pestaña "Detalles" de la ventana description ("Orientación"). 

También tiene la capacidad de especificar si las barras de desplazamiento de automatic deben ser mostradas. Las barras de desplazamiento se mostrarán automáticamente si el tamaño de la ventana es mayor que la resolución de la pantalla del dispositivo utilizado.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_FenDetail.gif)



### Contenido de la ventana
<a name="contenido_ventana_ELTPARAGRAPHE000041"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_Contenu.gif)
Tiene la posibilidad de crear una ventana: 

- con **barra de navegación**. Esta barra permite al usuario cerrar una ventana mediante un botón "Cerrar" o "Atrás", por ejemplo.
	Para añadir una barra de navegación a una ventana: En la pestaña "Ventana", en el grupo "Barras y menús", amplíe "Otras barras", haga clic en "Para compatibilidad" y, a continuación, seleccione "Barra de navegación manual (Barra de navegación iOS").

- con un **looper o una zona multilínea**. Este tipo de ventana se utiliza para seleccionar una opción, configurar una o varias opciones, etc.

- con una **barra de pestañas**. En la mayoría de los casos, se utiliza una "barra de pestañas" para acceder a las distintas ventanas de la aplicación.
	Para añadir una barra de pestañas a una ventana: en la pestaña "Ventana", en el grupo "Barras y menús", amplíe "Otras barras", haga clic en "Para compatibilidad" y, a continuación, seleccione "Barra de pestañas manual (iOS Tab Bar").

- con una **barra de herramientas**. En la mayoría de los casos, la barra de herramientas se utiliza para realizar acciones en la ventana actual. 
	Para añadir una barra de herramientas a una ventana: En la pestaña "Ventana", en el grupo "Barras y menús", amplíe "Otras barras", haga clic en "Para compatibilidad" y, a continuación, seleccione "Barra de herramientas manual (iOS ToolBar").




**Diferencia entre una barra de pestañas y una barra de herramientas**: 

- Una barra de herramientas propone acciones en la vista actual.

- Una barra de pestañas corresponde a las pestañas.





### Configuración del looper o de la zona multilínea
<a name="configuracion_del_looper_zona_multilinea_ELTPARAGRAPHE000099"></a>

Si ha elegido crear una ventana con looper o con zona multilínea, los diferentes planos de asistente le permiten configurar este elemento. 

- Hay varios tipos de bucles preestablecidos o zonas multilínea disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_TypeZR.gif)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ass_typeZM.gif)


- El siguiente plano se utiliza para configurar los elementos mostrados en el looper o en la zona multilínea. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ass_ParamZR.gif)


- Para un looper, tienes la posibilidad de elegir cómo se llenará el looper (programación, enlace de archivo, enlace de Variable).




Cuando se valida la asistente, el looper o la zona multilínea seleccionada se crea automáticamente en la ventana. 

Usted tiene la capacidad de escribir el código utilizado para manipular y hacer que funcione.. 

**Para un looper** , los principales eventos WLanguage a modificar son los siguientes:

- Código de inicialización de control Looper: esta Event se utiliza, por ejemplo, para llenar el bucle. 

- Código de selección de filas: esta Event se utiliza para gestionar las acciones realizadas cuando el usuario hace clic en la flecha estándar. 

- Haga clic en el código de IMG_Disclosure Image: esta Event se utiliza para gestionar el clic en la bala personalizada. 




Para obtener más información, consulte: 

- [Looper](../WDChamp/1013099.md) control. 

- [Zona multilínea](../WDChamp/1000024001.md) control. 





### Barra de pestañas o barra de herramientas
<a name="barra_pestanas_barra_herramientas_ELTPARAGRAPHE000133"></a>

Si se ha seleccionado esta opción, especifique los parámetros de la barra: 

- Color, 

- Estilo, 

- Botones mostrados.




Al final de la asistente, la ventana se crea con la barra especificada. Todo lo que tienes que hacer es introducir el código de los botones.


## Manejar un grupo de controles a través de la programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000110"></a>
WINDEV, WEBDEV et WINDEV Mobile le permiten manipular un grupo de controles a través de la programación. Para ello, utilice la Variable correspondiente al nombre del grupo de controles del código.

La Variable del grupo de controles corresponde al nombre del grupo de controles.

Ejemplo: 

```wl
// Handle the GR_Review group of controls
// Makes the group of controls invisible
GR_Review.State = Invisible
```


La operación realizada sobre los grupos de controles es global para la ventana que se ejecuta: incluye los grupos de controles de la ventana, incluyendo los grupos que se encuentran en una ventana interna, página interna, modelo de controles o supercontrol. En este caso, el nombre del grupo de control no debe ir precedido del nombre del elemento que lo contiene (ventana interna, página interna, control modelo o supercontrol por ejemplo).

Ejemplo: 

```wl
// Handle the GR_Review group of controls 
// in the IW_User internal window
// Makes the group of controls invisible
GR_Review.State = Invisible
```


Observación: En el código, puede identificar los controles que componen un grupo de controles utilizando [EnumControl](../WDLang1/3025005.md).

## Propiedades específicas de los grupos de controles
<a name="propiedades_especificas_los_grupos_controles_ELTTEXTE000134"></a>
Las siguientes propiedades se utilizan para gestionar las características de un grupo de controles a través de la programación.



|   |   |
| --- | --- |
| [Conde](../Proprietes/2510097.md) | Permite conocer el número de controles de un grupo de controles. |
| [ContextMenu](../Proprietes/2510077.md) | Permite conocer y modificar el menú contextual associated con un grupo de controles. |
| [Estado](../Proprietes/2510039.md) | Permite conocer y modificar el estado de visualización de un grupo de mandos. |
| [Modified](../Proprietes/2510080.md) | Se utiliza para averiguar si un control perteneciente a un grupo de control s fue modificado por el usuario (con el teclado o con el ratón). |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con un grupo de controles, consulte [Propiedades associated con un grupo de controles](../WDChamp/1010018.md).

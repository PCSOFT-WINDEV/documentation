## Presentación
<a name="presentacion_ELTTEXTE000154"></a>
WINDEV Mobile le permite desarrollar aplicaciones para diferentes familias de productos:   

- Android, 

- iPhone y/o iPad,

- Windows universal.




Veamos los detalles de cada plataforma: 




- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Universal Windows 10 App**: Para las Universal Windows Apps, seleccione el dispositivo que se tendrá en cuenta para editar la aplicación. También puede indicar el directorio del SDK de Windows 10. 




Todas estas características definen la plataforma de tiempo de ejecución.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cuándo se tiene en cuenta la plataforma de tiempo de ejecución?
<a name="¿cuando_tiene_cuenta_plataforma_tiempo_ejecucion_ELTTEXTE000178"></a>
Se tiene en cuenta la plataforma de tiempo de ejecución seleccionada:

- **para crear el windows**: las dimensiones seleccionadas se propondrán automáticamente para las nuevas ventanas.

- **para generar la aplicación**. La aplicación generada corresponde a una plataforma específica.

- **al iniciar la aplicación en modo de prueba**. La plataforma seleccionada permite utilizar el simulador correspondiente. 




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## ¿Cómo modificar la plataforma de tiempo de ejecución?
<a name="¿como_modificar_plataforma_tiempo_ejecucion_ELTTEXTE000202"></a>
La plataforma de tiempo de ejecución puede ser modificada:

- al crear un proyecto.

- en la descripción del proyecto: 

	- En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

	- En la pestaña "Proyecto", haga clic en el botón "Descripción de la plataforma".







La plataforma seleccionada es válida para todo el proyecto actual.

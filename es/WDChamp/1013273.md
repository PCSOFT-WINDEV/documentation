
## Menú contextual de controles TreeView (AAF)
			



## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
Cada control TreeView que se encuentra en una ventana propone un menú emergente al usuario. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Arbre.gif)


Este menú, disponible sin programación, incluye varias acciones:

- copy (el elemento, el fila, la columna o todo). copy ing la columna se utiliza para copy todo el contenido de las ramas treeview al portapapeles. Los diferentes elementos están separados por caracteres CR (Carriage Return). 

- expandir o contraer todo: Esta opción permite expandir o contraer todos los elementos de la rama actual. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) exportar a Word, Excel, XML: Esta opción se utiliza para exportar el contenido de la control TreeView a un archivo en el formato especificado.. 




**Observaciones**: 

- El mismo menú contextual está disponible para la sección treeview de una Tabla TreeView. 

- Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

## Ajustes
<a name="ajustes_ELTTEXTE000142"></a>


### Configuración del menú automatic en un control TreeView
<a name="configuracion_del_menu_automatic_control_treeview_ELTPARAGRAPHE000033"></a>

El menú contextual de control TreeView puede desactivarse. Para configurar la visualización del menú contextual associated con el control TreeView, debe:

1. Mostrar la ventana description del control TreeView.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, aparecerá el menú automatic del control TreeView. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








**Observaciones**: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.

- [AAFDisable](../WDLang1/1000022018.md) se utiliza para eliminar algunas opciones del menú contextual AAF específicas de los controles TreeView. Por ejemplo, las constantes *aafTableToExcel*, *aafTableToWord* y *aafTableToXML* permiten desactivar las funcionalidades de exportación.

## Resumen de control Linear Looper
<a name="resumen_control_linear_looper_ELTTEXTE000172"></a>


### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

Una control Linear Looper se utiliza para mostrar un set de controles en una lista horizontal Scrolling.

El control Linear Looper se utiliza para mostrar armónicamente un menú con imágenes, por ejemplo.

La información mostrada en el control Linear Looper puede:

- definirse mediante programación: esto se denomina ***Looper lineal poblado programáticamente***.

- vienen de un archivo de datos o consulta: en este caso se denomina control ***Looper lineal archivo de datos***.

- vienen de un WLanguage Variable: en este caso se denomina control ***Looper lineal sobre fuente***.




**Observación**: La control Linear Looper comparte numerosas características con la control Looper. Su objetivo específico es mostrar un pequeño número de elementos para proponer una visión agradable.

<a name="NOTE2_1"></a>


## Atributos de una control Linear Looper
<a name="atributos_una_control_linear_looper_ELTTEXTE000196"></a>
Una control Looper lineal incluye:

- **controls**, que se repiten en cada fila.

- **atributos**. Se utiliza un atributo para definir la característica de control que se modificará en cada fila. Por ejemplo, si el valor y la Color de control de PRECIO debe cambiar en cada fila, tendrá que definir dos atributos diferentes para el mismo control.




Para averiguar o modificar la característica de una control (su valor, su Color, ...), todo lo que hay que hacer es manipular esta control o el atributo correspondiente por programación.

## Crear un control Looper lineal
<a name="crear_control_looper_lineal_ELTTEXTE000220"></a>
Para crear una control Linear Looper:

1. En la pestaña "Creación", en el grupo "Datos", despliegue "Looper" y haga clic en "Looper lineal".

2. Haga clic en la ubicación en la que desea crear el control. La creación de control Linear Looper asistente se inicia automáticamente.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. Se pueden ver dos tipos de información:

- la información relativa a toda la control Looper (todo lo que tiene que hacer es seleccionar el nombre de control Looper).

- la información relativa a cada atributo de control Looper (todo lo que tiene que hacer es seleccionar el nombre de un atributo en el control Looper).

## Vocabulario
<a name="vocabulario_ELTTEXTE000244"></a>
Para que manipular a control Linear Looper sea fácil, usted debe estar familiarizado con el siguiente vocabulario:

| denominar | Definition |
| --- | --- |
| Atributo | Un atributo corresponde a un valor modificable para cada elemento de la control Linear Looper.<br>Un atributo puede ser relacionado a un Property de un control que se encuentra en el control Linear Looper para hacer que este Property cambie en todas las columnas.<br><br>Para cada atributo, se debe definir:<br><br>- el nombre del atributo.<br><br>- el nombre de la control en el bucle lineal que es relacionado al atributo.<br><br>- el Property del control correspondiente al atributo (valor, Color, ...).<br><br><br> |
| Looper lineal basado en un archivo de datos | El control Linear Looper está basado en un archivo de datos o en una consulta. Un Record que se encuentra en el archivo de datos se muestra en cada fila. Existen dos tipos de Loopers lineales basados en un archivo de datos:<br><br>- ***Looper lineal con fuente de datos en memoria***: el contenido de la control Looper lineal se carga en la memoria y luego se muestra. El acceso a los diferentes elementos de la control Linear Looper es más rápido. Este modo está reservado para archivos de datos con menos de 100 000 registros.<br><br>- ***Looper lineal con acceso directo a la fuente de datos (sin límite)***: El control Linear Looper muestra el contenido del archivo de datos o consulta directamente. El registro actual corresponde al elemento seleccionado. La selección de un elemento en la control Looper desencadena un acceso al archivo de datos de la relacionado. Este tipo de control Linear Looper está reservado para archivos de datos con más de 100.000 registros.<br><br><br> |
| Looper lineal rellenado programáticamente | El control Linear Looper no está basado en un archivo de datos ni en una consulta. Los elementos que aparecen en la control Linear Looper son definidos por el desarrollador (mediante programación). |
| Looper lineal basado en un Variable | La control Linear Looper se basa en una Variable (global al proyecto o página; miembro de la clase, etc.). |

### Operaciones en AJAX
<a name="operaciones_ajax_ELTPARAGRAPHE000109"></a>

Los controles del looper lineal no pueden ser modificados en AJAX (adición o eliminación de elementos).

Sin embargo, su contenido puede ser leído desde una función o desde un Process en modo AJAX. Del mismo modo, los procesos de los controles visualizados por el control Linear Looper pueden estar en modo AJAX.

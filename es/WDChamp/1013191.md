## Panorama de la control indexada
<a name="panorama_control_indexada_ELTTEXTE000114"></a>
Un control es un control indexado si contiene varios control con el mismo nombre. Cada control tiene el mismo nombre pero un índice diferente. Este índice le permite manipular fácilmente la control deseada mediante programación.

**Observación**: Los controles indexados permiten gestionar fácilmente los elementos de texto Array definidos en la base de datos editor de análisis.

¿Qué controles pueden indexarse? Controles y columnas de edición en controles Tabla (numéricos, texto, moneda, fecha y hora).

**Atención**:

- Todas las columnas indexadas deben estar en la misma control Tabla.

- Los controles indexados sólo pueden utilizarse en el editor de ventanas. Los mandos indexados no pueden utilizarse en el editor de reportes.

## Creación de una control indexada
<a name="creacion_una_control_indexada_ELTTEXTE000138"></a>


### Crear una control Campo de entrada indexada 
<a name="crear_una_control_campo_entrada_indexada_ELTPARAGRAPHE000027"></a>

Para crear un índice control Campo de entrada.

1. Crear una nueva edición de control. 

2. Darle el nombre de un control Campo de entrada existente. Los controles están indexados.



<a name="NOTE2_2"></a>


### Para crear una columna indexada
<a name="para_crear_una_columna_indexada_ELTPARAGRAPHE000037"></a>

Todo lo que tiene que hacer es crear una nueva columna en una control Tabla con el mismo nombre que una columna existente.. 

- El nombre de ambas columnas debe terminar con un número (por ejemplo, MiColumna1).. 

- Al crear la segunda columna, WINDEV propone crear una control indexada . Si acepta, el índice aparece en la ventana description del control Tabla.



<a name="NOTE2_3"></a>


### Artículo control y array indexado
<a name="articulo_control_array_indexado_ELTPARAGRAPHE000047"></a>

Tras su creación, la control indexada puede ser relacionado a uno de los elementos de un elemento array.

## Notas
<a name="notas_ELTTEXTE000174"></a>


### Artículo control y array indexado
<a name="articulo_control_array_indexado_ELTPARAGRAPHE000056"></a>

Cuando se utiliza un elemento array, no es necesario que todos los controles indexados correspondientes al elemento estén en la misma ventana (al contrario que en WINDEV 5.5). Si el elemento array contiene 6 elementos, sólo se pueden encontrar tres elementos en la ventana.
<a name="NOTE3_2"></a>
En una ventana, se puede localizar una control indexada:

- en una ventana (primer plano).

- en un avión.

- en una pestaña.

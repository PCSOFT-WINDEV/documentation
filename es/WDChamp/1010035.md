
## Propiedades associated con las celdas de las barras de estado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes propiedades son associated con las celdas que se encuentran en las barras de estado:



|   |   |
| --- | --- |
| [AnchorRateWidth](../Proprietes/2510120.md) | El **AnchorRateWidth** Property se utiliza para:<br><br>- Averigua la tasa de anclaje cuando una columna control (o una columna tabla) se extiende sobre el eje horizontal.<br><br>- Modifica la tasa de anclaje cuando una columna control (o una columna tabla) se extiende en el eje horizontal.<br><br><br> |
| [BackgroundColor](../Proprietes/2510022.md) | La propiedad **BackgroundColor** permite establecer el color de fondo de un control, ventana, página o bloque de un reporte. |
| [Caption](../Proprietes/2510053.md) | **En una base de datos**, el **Caption** Property se utiliza para:<br><br>- Averigua la leyenda associated con un elemento de archivo. <br><br>- Averigua la leyenda associated con un archivo de datos. <br><br>- Descubrir y modificar la leyenda de conexión. <br><br><br><br><br>**En un reporte**, el **Caption** Property se utiliza para:<br><br>- Averigua el texto que aparece en una estática o control Enlace.<br><br>- Modificar el texto que se muestra en una estática o control Enlace.<br><br><br><br><br>**En una ventana o página**, el **Caption** Property también se utiliza para:<br><br>- Averiguar y modificar la leyenda de un control en una ventana o en un Page.<br><br>- Descubrir y modificar el título de una opción de menú.<br><br>- Averiguar y modificar el título de una ventana o Page.<br><br>- Descubra el contenido de un control Migas de pan.<br><br><br><br><br> |
| [Color](../Proprietes/2510071.md) | El **Color** Property se utiliza para get o cambiar la fuente Color para:<br><br>- una fuente Variable.<br><br>- un texto visualizado en un control Estático o en un control calculado de un reporte.<br><br>- un texto mostrado en una ventana control.<br><br>- un texto mostrado en una página control.<br><br><br> |
| [Font](../Proprietes/2510095.md) | La propiedad **Font** permite<br><br>- Obtener el identificador de la fuente utilizada en un control de un reporte, página o ventana.<br><br>- Cambiar la fuente utilizada en un control de un reporte, página o ventana.<br><br><br> |
| [FontBold](../Proprietes/2510103.md) | **FontBold** se utiliza para:<br><br>- Determina si el texto está en negrita.<br><br>- Cambiar el peso del texto.<br><br><br> |
| [FontCharset](../Proprietes/2510089.md) | El **FontCharset** Property se utiliza para:<br><br>- Averigüe el carácter set utilizado por las fuentes de pantalla de un control.<br><br>- Modificar el carácter set utilizado por las fuentes de pantalla de un control.<br><br><br> |
| [FontCondensed](../Proprietes/2510096.md) | El **FontCondensed** Property se utiliza para:<br><br>- Determinar si los caracteres de un texto están condensados.<br><br>- Condensa (o no) los caracteres de un texto.<br><br><br> |
| [FontExtended](../Proprietes/2510104.md) | El **FontExtended** Property se utiliza para:<br><br>- Averigua si el espacio entre los caracteres de un texto está aumentado.<br><br>- Aumentar el espacio entre los caracteres de un texto.<br><br><br> |
| [FontItalic](../Proprietes/2510102.md) | **FontItalic** se utiliza para:<br><br>- Determina si el texto está en cursiva.<br><br>- Aplicar o eliminar el formato de cursiva del texto.<br><br><br> |
| [FontLarge](../Proprietes/2510101.md) | **FontLarge** se utiliza para:<br><br>- Averiguar si los caracteres de un texto están ampliados o no.<br><br>- Ampliar (o no) los caracteres de un texto.<br><br><br> |
| [FontName](../Proprietes/2510100.md) | El **FontName** Property se utiliza para:<br><br>- Identificar el tipo de letra utilizado en un control.<br><br>- Cambiar el tipo de letra utilizado en un control.<br><br><br> |
| [FontSize](../Proprietes/2510072.md) | El **FontSize** Property se utiliza para:<br><br>- Identificar el tamaño de la fuente utilizada en un control.<br><br>- Cambiar el tamaño de la fuente utilizada en un control.<br><br><br> |
| [FontStrikeOut](../Proprietes/2510091.md) | El **FontStrikeOut** Property se utiliza para:<br><br>- Determinar si el texto mostrado está tachado.<br><br>- Aplicar el formato de tachado (o no) al texto mostrado en un control.<br><br><br> |
| [FontUnderlined](../Proprietes/2510099.md) | **FontUnderlined** se utiliza para:<br><br>- Averiguar si un texto está subrayado o no.<br><br>- Subrayar (o no) un texto.<br><br><br> |
| [FullName](../Proprietes/2514005.md) | El **FullName** Property se utiliza para averiguar el nombre completo:<br><br>- de una control.<br><br>- de una ventana.<br><br>- de una opción de menú.<br><br>- de una columna tabla.<br><br><br><br><br> |
| [HelpNumber](../Proprietes/2510141.md) | La propiedad **HelpNumber** permite obtener y modificar el número de ayuda asociado al control. |
| [HorizontalAlignment](../Proprietes/2510023.md) | El **Alineación Horizontal** Property se utiliza para get y cambiar la alineación horizontal de un control. |
| [Image](../Proprietes/2510034.md) | El **Image** Property obtiene y establece la imagen de fondo de un control o bloque. |
| [InitialValue](../Proprietes/2510128.md) | El **Valor inicial** Property se utiliza para get el valor inicial de un control en una ventana o página. |
| [InitialVisible](../Proprietes/2510070.md) | La propiedad **..InitialVisible** permite saber si:<br><br>- un control fue visible durante la primera visualización de la ventana o página asociada.<br><br>- una ventana era visible durante su primera visualización.<br><br><br> |
| [InitialWidth](../Proprietes/2510067.md) | El **AnchoInicial** Property se utiliza para:<br><br>- Averigua la anchura inicial de un control o ventana.<br><br>- Modificar el ancho de la base utilizada para anclar los controles en "width"..<br><br>- Averigua la anchura inicial de un reporte control.<br><br><br> |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [Type](../Proprietes/2510131.md) | La propiedad **Type** permite obtener:<br><br>- el tipo de elemento (archivo de datos o campo) definido mediante programación o en el editor de análisis, vista o consulta.<br><br>- el tipo de objeto.<br><br>- el tipo de datos que se almacena en un Variant.<br><br><br> |
| [Value](../Proprietes/2510130.md) | La propiedad **Value** permite:<br><br>- Averiguar y modificar el valor de un reporte control.<br><br>- Averiguar y modificar el valor de un control o el título de una ventana.<br><br>- Averiguar el identificador de una secuencia. <br><br>- Averiguar y modificar el valor de un control o el título de un Page.<br><br><br> |
| [Visible](../Proprietes/2510138.md) | El **Visible** Property se utiliza para:<br><br>- Averiguar si un elemento es Visible.<br><br>- Hacer visible un elemento o Invisible.<br><br><br> |
| [Width](../Proprietes/2510059.md) | El **Ancho** Property obtiene y modifica el ancho:<br><br>- de un control o bloque en un reporte.<br><br>- de un control (que se encuentra en una ventana), de una ventana, de una columna que se encuentra en un tabla o de una columna que se encuentra en un cuadro de lista (incluida la listview).<br><br>- de un control (que se encuentra en un Page), columna que se encuentra en un tabla o columna que se encuentra en un cuadro de lista. La control puede ser superpuesta o no.<br><br><br> |
| [WinEditAllowed](../Proprietes/1000021850.md) | El **WinEditAllowed** Property se utiliza para determinar y configurar las acciones permitidas en un control cuando la ventana pasa al modo "modificación del UI por el usuario final". |





**Recordatorio**: El [Mensaje](../Proprietes/2510078.md) Property permite obtener y cambiar el mensaje de ayuda associated con un control en una ventana.

**Observaciones**:

- La propiedad [StatusBar](../Proprietes/2510007.md) permite:

	- averiguar si la barra de estado de la ventana actual está habilitada.

	- activar la barra de estado de la ventana actual.




- El nombre de una celda creada por [StatusBarAddCell](../WDLang1/3021001.md) debe estar encerrado entre operadores de indirección para poder ser tratado por las propiedades de WLanguage.

## Presentación
<a name="presentacion_ELTTEXTE000377"></a>
La editor de reportes le permite:

- [Proponer varios modos de impresión durante la prueba reporte](#NOTE2_1).

- [Mostrar el nombre de los controles](#NOTE3_1).

- [Mostrar los márgenes de reporte](#NOTE3_2).

- [Mostrar el contenido de los archivos de datos en los controles creados actualmente](#NOTE3_3).

- [Mostrar la página de useful reporte](#NOTE3_4).

- [Mostrar el nombre de los bloques (completo o abreviado)](#NOTE4_1).

- [Mostrar el lenguaje associated con cada fuente](#NOTE5_1).

- [Encuadre los controles y los bloques](#NOTE6_1).

- [Definir la unidad de medida utilizada en la pantalla](#NOTE7_1). 




Estas características se aplican a:

- sólo la reporte actual,

- sólo la próxima apertura de los informes.


La mayoría de estas características son accesibles:

- en el panel "Pantalla",

- desde la ventana de gestión de las opciones de edición de editor de reportes. Para abrir esta ventana, vaya a la pestaña "Vista" y haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) del grupo "Opciones".

## Proponer varios modos de impresión durante la prueba reporte
<a name="proponer_varios_modos_impresion_durante_prueba_reporte_ELTTEXTE000401"></a>
Durante el test reporte, editor de reportes propone varios modos de impresión: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GO_Etat_choix.gif)


Para más detalles sobre estos diferentes modos, véase [Modos de impresión de un reporte](../WDChamp/1011031.md).

Observación: Puedes mostrar esta ventana de nuevo. Para obtener más información, consulte [No volver a mostrar un mensaje](../Editeurs/2020011.md).

### Mostrar el nombre de los controles
<a name="mostrar_nombre_los_controles_ELTPARAGRAPHE000084"></a>

Para mostrar u ocultar los nombres de los controles en el editor de reportes, en la pestaña "Nombres de los controles", en el grupo "Vista", marque "Vista" (o pulse F6).

Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NOMCHAMPETAT.GIF)


Otra solución:

1. En la pestaña "Vista", haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) del grupo "Opciones".

2. Si es necesario, mostrar la pestaña "Mostrar. 

3. En la opción "Nombre de los mandos" seleccione: 

	- "Pantalla" para mostrar el nombre de los controles,

	- "Desactivado" para no mostrar nada.

### Mostrar el contenido de los archivos de datos
<a name="mostrar_contenido_los_archivos_datos_ELTPARAGRAPHE000163"></a>

La editor de reportes le permite mostrar el contenido de los archivos de datos en los controles de un reporte actualmente creado.. Esta característica permite configurar el tamaño de un control de acuerdo con los datos que se pueden mostrar en el control.

Esta característica se llama "Live Data".. Para obtener más información, consulte [Live Data](../Editeurs/2021005.md). 
<a name="NOTE3_4"></a>


### Mostrando la página de useful
<a name="mostrando_pagina_useful_ELTPARAGRAPHE000175"></a>

La editor de reportes se utiliza para mostrar el reporte actual tal y como será cuando se imprima:

- los márgenes de reporte son Invisible,

- los nombres de los bloques de reporte se abrevian cuando se muestran.


Para obtener una vista previa de una reporte tal y como se imprimirá, pulse F5.

## Mostrar el lenguaje associated con cada fuente
<a name="mostrar_lenguaje_associated_con_cada_fuente_ELTTEXTE000491"></a>
Para modificar la fuente en los controles de reporte, debe utilizar:

- El cuadro combinado "Fuente" en la pestaña "Estilo" del control description.

- El cuadro combinado "Font" en la barra emergente de la editor de reportes.


Mediante Default, sólo se muestran los nombres de las fuentes instaladas en el ordenador de desarrollo en el cuadro combinado "Fuente".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_Langues%20-%20HC%20N%B0001.gif)


En Windows, la misma fuente puede ser associated con varios grupos de idiomas. Para llenar cada nombre de fuente con el lenguaje associated:

1. En el editor de reportes, vaya a la pestaña "Vista" y haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) del grupo "Opciones".

2. Abra la pestaña "Preferencias" si es necesario. 

3. Comprobar "Mostrar todos los idiomas disponibles".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_Langues%20-%20HC%20N%B0003.gif)





En la pestaña "Estilo" de una determinada control, la lista de fuentes será la siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_Langues%20-%20HC%20N%B0004.GIF)


**Observación**: Esta opción se tendrá en cuenta en la próxima apertura de reporte.

## Mostrando los controles y los bloqueos a través de un Border
<a name="mostrando_los_controles_los_bloqueos_traves_border_ELTTEXTE000515"></a>
El editor de reportes le permite ver rápidamente todos los controles y bloques que se encuentran en el reporte actual.. 

Es posible: 

- enmarcar los controles y bloques con un simple Line. Este Line aparece sólo en modo de edición. Esta línea no es Visible en tiempo de ejecución.
	Este Border se utiliza para ver fácilmente el área ocupada por los controles y bloques que se encuentran en un reporte (especialmente para los controles de Image).

- mostrar las esquinas de los controles y bloques. Este modo de vista, más ligero que un Line, permite ver rápidamente las diferentes áreas.

- no utilizar ningún modo de visualización específico. 


Para cambiar el modo de visualización en el reporte actual, vaya a la pestaña "Vista", grupo "Vista", y seleccione el modo de visualización deseado: 

- Esquinas

- Todos los bordes

- nono


La tecla F7 se utiliza para modificar dinámicamente el modo de visualización.

Otra solución:

1. En la pestaña "Modificación", haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) del grupo "Opciones".

2. Si es necesario, mostrar la pestaña "Mostrar. 

3. En la opción "Marcos de controles", seleccione, "Ninguno", "Solo las esquinas" o "Todos los bordes".

## Unidad de medida
<a name="unidad_medida_ELTTEXTE000539"></a>
Puede utilizar milímetros o pulgadas en el editor de reportes. 

Para elegir la unidad de medida utilizada en la editor de reportes: 

1. En la pestaña "Vista", haga clic en el lanzador de cuadros de diálogo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)) del grupo "Opciones".

2. Abra la pestaña "Preferencias" si es necesario. 

3. En el apartado "Unidad de medida", seleccione la unidad que se utilizará en la pantalla: milímetros o pulgadas.

4. Validar. La unidad de medida seleccionada se utilizará para los siguientes informes abiertos en la editor de reportes.

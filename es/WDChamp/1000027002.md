
## Eventos associated con un Control deslizante (WEBDEV)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Eventos utilizados de forma predeterminada
<a name="eventos_utilizados_forma_predeterminada_ELTTEXTE000078"></a>
WINDEV gestiona los siguientes eventos por Default (en el orden en que aparecen en el editor de código) para los controles Control deslizante lineales y para los controles Control deslizante de rango:

| Evento | Condición de ejecución |
| --- | --- |
| Inicialización | Se ejecuta al abrir la página. |
| Siempre que se modifique (código del navegador) | Ejecutar cuando se hace clic en el cursor de Control deslizante, y cuando el cursor se mueve. |
| Fin de la modificación (código del navegador) | Ejecutar al final del movimiento del cursor Control deslizante (al soltar el cursor). |
| Fin de la modificación (código del servidor) | Ejecutar al final del movimiento del cursor Control deslizante (al soltar el cursor). |

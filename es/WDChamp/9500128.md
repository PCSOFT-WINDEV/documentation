
## Guardar la entrada de datos (FAA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
Los controles Campo de entrada cuentan con un menú "Recordar" que permite a los usuarios establecer las opciones para guardar los valores introducidos en el control. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=M%E9moriser%20la%20saisie%20effectu%E9e_Frame%201.jpg)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=M%E9moriser%20la%20saisie%20effectu%E9e_Frame%202.jpg)


Las siguientes opciones estarán disponibles: 

- **Recordar último valor introducido**: Si el usuario selecciona esta opción, el último valor introducido se mostrará automáticamente en el control cada vez que se abra la ventana. 

- **Recordar XXX**: Si el usuario selecciona esta opción, el valor XXX se introducirá automáticamente en el control cada vez que se abra la ventana.

- **Recordar la fecha de hoy** (Controles de tipo Fecha únicamente): Si el usuario selecciona esta opción, la fecha actual se introducirá automáticamente en el control cada vez que se abra la ventana. 

- **Recordar historial de entradas**: Si el usuario selecciona esta opción, los valores introducidos previamente en el control aparecen automáticamente cuando se introducen los primeros caracteres.
	Observación: Esta opción solo está disponible si el control Campo de entrada está enlazado a un campo de un archivo de datos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Saisie_HistSaisie.gif)
Si el historial de entradas está activado, se muestra una opción adicional para borrar el historial. 




**Observación**: La opción "Recordar" está disponible en el menú contextual de diferentes tipos de controles. Esta permite implementar la persistencia de datos. Los datos almacenados se guardan en el equipo del usuario. Para obtener más información, consulte [Persistencia de datos](../WDChamp/9000019.md).

## Configuración del menú
<a name="configuracion_del_menu_ELTTEXTE000168"></a>


### Configuración del menú automático de historial de entradas en un control Campo de entrada
<a name="configuracion_del_menu_automatico_historial_entradas_control_campo_entrada_ELTPARAGRAPHE000038"></a>

El menú contextual del control Campo de entrada puede ser desactivado. Para configurar el menú contextual del control:

1. Abra la ventana de descripción del control.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contextual". Es posible: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, se mostrará el menú automático del control. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú FAA](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








**Observaciones**: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las FAA. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las FAA y restaurar el menú estándar de Windows para un control determinado.

- [AAFDisable](../WDLang1/1000022018.md) permite eliminar algunas opciones del menú contextual de las FAA específicas de los controles Campo de entrada.

	- para desactivar la opción "Recordar", simplemente utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafSaveValue*. 

	- para desactivar las opciones del historial de entradas, simplemente utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafHistoryInput*.






<a name="NOTE2_2"></a>


### Configurar el historial de entradas
<a name="configurar_historial_entradas_ELTPARAGRAPHE000090"></a>

En la pestaña "Detalles" de la descripción del control Campo de entrada, se puede especificar que: 

- el valor en el control debe ser guardado. 

- el historial de entradas debe estar activado. 


En este caso, las opciones del menú contextual del control actual se seleccionarán automáticamente.

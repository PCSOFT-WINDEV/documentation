
## eYe magnet: formato condicional de los controles
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
El "imán eYe" (también llamado formato condicional) consiste en modificar el aspecto visual de una control para atraer la atención del usuario hacia esta control. WINDEV permite modificar automáticamente este aspecto visual según una set de condiciones. 

Por ejemplo, un control para el que la entrada es obligatoria se mostrará en un Background rojo mientras su contenido permanezca vacío.

La función eYe magnet se aplica a los siguientes controles:

- Controles campo de entrada.

- controles Combo Box.

- Columnas de controles Tabla en modo de entrada.

- Columnas del TreeView controles Tabla en modo de entrada.

- Células de control Tabla dinámica.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000159"></a>
<a name="NOTE2_2"></a>


### Definición de un nuevo imán eYe
<a name="definicion_nuevo_iman_eye_ELTPARAGRAPHE000031"></a>

Definir un nuevo imán eYe (formato condicional):

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

2. En la pestaña " UI ", haga clic en el enlace "eYe magnet: &lt;Ningún formato&gt;" o en la descripción del eYe magnet actual (si ya se definió un eYe magnet). 
	Observación: Para las celdas de una control Tabla dinámica, en la pestaña "Contenido", haga clic en el valor deseado ("Valor a mostrar en la tabla"). En el área de características "Display", haga clic en el enlace "eYe magnet: &lt;Ningún formato&gt;" o en la descripción del eYe magnet actual (si ya se definió un eYe magnet). 

3. Si el control no tiene todavía formato, la ventana para crear un formato condicional se abre directamente. 
	Si ya existe uno o más formatos para el control, se muestra una ventana con la lista de los eYe magnets. 

4. Haga clic en el botón "Añadir un imán eYe" para abrir la ventana description de un imán eYe.




La ventana para describir las condiciones y acciones es la siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=eyemagnet_fen_edition.gif)


- Los botones ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=eyemagnet_btn_plus.gif) y ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=eyemagnet_btn_supprimer.gif) se utilizan para añadir o eliminar condiciones o acciones.

- Cuando todas las condiciones ocurren simultáneamente, todas las acciones de associated se realizan automáticamente.

- Se pueden definir varios imanes eYe en la misma control para realizar diferentes acciones según las condiciones.

- El imán eYe se aplica después de los procesos de la control (Entrada en edición, Siempre que se modifique, etc.). Sus efectos no pueden ser sobrecargados por la programación.



<a name="NOTE2_3"></a>


### Modificación de un imán eYe existente
<a name="modificacion_iman_eye_existente_ELTPARAGRAPHE000061"></a>

Para modificar un imán eYe existente:

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

2. En la pestaña "UI", haga clic en la descripción del eYe magnet para abrir la ventana de descripción de los eYe magnets.

3. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=eyemagnet_btn_modif.gif) para abrir la ventana de edición.

4. Modificar las características del imán eYe. Para obtener más información, consulte [Definición de un nuevo imán eYe](#NOTE2_2).



<a name="NOTE2_4"></a>


### Eliminación de un imán para los ojos
<a name="eliminacion_iman_para_los_ojos_ELTPARAGRAPHE000078"></a>

Para eliminar un imán eYe existente: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

2. En la pestaña "UI", haga clic en la descripción del eYe magnet para abrir la ventana de descripción de los eYe magnets.

3. Haga clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=eyemagnet_btn_supprimer.gif) del imán de eYe para borrar.

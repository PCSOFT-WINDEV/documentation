
## Vinculación de un control Planificador con un un archivo de datos o un Variable
			

<a name="NOTA1"></a>
<a name="NOTA1_1"></a>


## Panorama general
<a name="panorama_general_ELTTEXTE000139"></a>
Los nuevos controles del Programador pueden ser directamente relacionado a los datos. Puede crear: 

- a **control Planificador basado en un archivo de datos**: la control Planificador será de relacionado a un archivo de datos o la consulta. En este caso, el archivo de datos contiene las características de los nombramientos. 

- a **control Planificador basado en un Variable**: el control Planificador será relacionado a un Variable (normalmente una instancia de clase). En este caso, la clase lista las ocupaciones de muelles con sus características. 




Esto facilita la programación de los controles del Programador. 





## ¿Cómo?
<a name="¿como_ELTTEXTE000163"></a>


### Creación de una control Planificador basada en un archivo de datos
<a name="creacion_una_control_planificador_basada_archivo_datos_ELTPARAGRAPHE000020"></a>

Para crear una control Planificador basada en un archivo de datos, puede: 

- utilizar la creación de control Planificador asistente. Para ello, seleccione "Mostrar datos de un archivo o de una consulta existente". Responder a las preguntas de la asistente y validar. 

- cambiar la description de la control Planificador. 




Para cambiar las características de una control Planificador existente:

1. Abra la ventana de control description (seleccione "description" en el menú contextual). 

2. En la pestaña "Contenido", seleccione "Archivo" y, a continuación, seleccione la fuente de datos deseada. Esta fuente de datos puede ser: 

	- un archivo de datos definido en el análisis,

	- una consulta del proyecto,

	- una consulta incluida en la ventana o página. 




3. Vincular los diferentes elementos de la fuente de datos a las características de Appointment. Los elementos precedidos de una estrella son obligatorios. 
	Marca **Remark**: Si el archivo de datos asociado no contiene directamente el elemento correspondiente, puede set un enlace de varios archivos. 

4. Validar la ventana de description de la control Planificador. La control Planificador se basa ahora en la un archivo de datos. 







### Creación de una control Planificador basada en una Variable
<a name="creacion_una_control_planificador_basada_una_variable_ELTPARAGRAPHE000042"></a>

Para crear una control Planificador basada en una Variable, puede: 

- utilizar la creación de control Planificador asistente. Para ello, seleccione "Mostrar datos de un Variable existente".. Responder a las preguntas de la asistente y validar. 

- cambiar la description de la control Planificador. 




Para cambiar las características de una control Planificador existente:

1. Abra la ventana de control description (seleccione "description" en el menú contextual). 

2. En la pestaña "Contenido", seleccione "Variable", y luego seleccione la fuente de datos deseada (clase, etc.). 

3. Vincular los diferentes elementos (miembros de la clase, por ejemplo) de la fuente de datos a las características de Appointment. Los elementos precedidos de una estrella son obligatorios. 

4. Validar la ventana de description de la control Planificador. El control Planificador se basa ahora en un Variable. 








## Gestión de una control Planificador basada en un archivo de datos o Variable
<a name="gestion_una_control_planificador_basada_archivo_datos_variable_ELTTEXTE000193"></a>


### control Planificador basada en un archivo de datos
<a name="control_planificador_basada_archivo_datos_ELTPARAGRAPHE000059"></a>

Los controles del programador basados en un archivo de datos son muy fáciles de manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario maneja el planificador a través del menú contextual (para añadir, eliminar o modificar citas, por ejemplo), el archivo de datos de associated se actualiza automáticamente con los cambios realizados en control. 

- Si el usuario maneja las citas a través de una ventana o página específica, es necesario modificar directamente el archivo de datos relacionado al control Planificador. Los datos mostrados se actualizarán automáticamente. También es posible utilizar [ProgramadorVisualizar](../WDLang1/1000019426.md) para actualizar los datos que aparecen en control. 




Marca **Remark**: Si el control está vinculado a una fuente de datos, los datos borrados en el control se borran también de la fuente de datos. 


### control Planificador basada en un Variable
<a name="control_planificador_basada_variable_ELTPARAGRAPHE000077"></a>

Los controles del programador basados en una Variable son muy fáciles de manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si el usuario maneja el programador a través del menú contextual (para añadir, eliminar o modificar citas, por ejemplo), la fuente de datos se actualiza automáticamente con los cambios realizados en el control.

- Si el usuario maneja las citas a través de una ventana o página específica, es necesario modificar directamente los elementos relacionado a la control Planificador (miembros de la clase, por ejemplo). Los datos mostrados se actualizarán automáticamente. También es posible utilizar [ProgramadorVisualizar](../WDLang1/1000019426.md) para actualizar los datos que aparecen en control.




Marca **Remark**: Si el control está vinculado a una fuente de datos, los datos borrados en el control se borran también de la fuente de datos.

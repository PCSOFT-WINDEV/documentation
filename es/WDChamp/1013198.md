


## Manejo programático de los controles ListView
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000338"></a>
WINDEV le permite programar los controles de manipular ListView. Basta con utilizar la Variable en lugar de la control ListView en el código. Esta variable es de tipo numérico.

La Variable del control ListView:

- corresponde al nombre de la control ListView.

- se inicializa con el índice del elemento seleccionado en la control ListView.




**Observación**: Para manipular un elemento ListView no associated con cualquier elemento almacenado, utilice la siguiente sintaxis:


```txt
<ListView control> [ <Element index>]
```

## Inicialización de una control ListView
<a name="inicializacion_una_control_listview_ELTTEXTE000362"></a>


### Asignar un valor a un elemento
<a name="asignar_valor_elemento_ELTPARAGRAPHE000031"></a>

Para inicializar un elemento encontrado en una control ListView, utilice [ListAdd](../WDLang1/3049004.md).

```wl
ListAdd(LSV_ListView1, "Sun", "C:\IMG\Sun.JPG")
ListAdd(LSV_ListView1, Photo)
```

<a name="NOTE2_2"></a>


### Rellenar una control ListView
<a name="rellenar_una_control_listview_ELTPARAGRAPHE000049"></a>Para inicializar un control ListView, hay que utilizar un bucle de iteración para inicializar todos los elementos del control.

```txt
Index is int
FOR Index = <Start position> TO <End position>
	ListAdd(<ListView control>, <Value>, <Image>)
END
```



```wl
i is int
FOR i = 1 TO 52
	ListAdd(LSV_ListView1, NAME[i], Image[i])
END
```

## Selección de un elemento en una control ListView
<a name="seleccion_elemento_una_control_listview_ELTTEXTE000392"></a>


### Seleccionar el elemento de la fila &lt;Índice&gt;
<a name="seleccionar_elemento_fila__ELTPARAGRAPHE000073"></a>

Para seleccionar un elemento en la fila &lt;Index&gt; utilice: 

- la asignación directa: 
	
	```txt
	<ListView control> = Index
	```


- la función [ListSelectPlus](../WDLang1/3049012.md): 
	
	```txt
	ListSelectPlus(<ListView control>, <Index>)
	```



<a name="NOTE3_2"></a>


### Fijar la posición en el último elemento de la control ListView
<a name="fijar_posicion_ultimo_elemento_control_listview_ELTPARAGRAPHE000097"></a>

Para set la posición en el último elemento del control ListView, utilice: 

- la propiedad [Conde](../WDLang1/3025009.md): 
	
	```txt
	<ListView control>.Count
	```


- la función [ListSelectPlus](../WDLang1/3049012.md) con la propiedad [Conde](../WDLang1/3025009.md): 
	
	```txt
	ListSelectPlus(<ListView control>, <ListView control>.Count)
	```


- la función [ListCount](../WDLang1/3049014.md): 
	
	```txt
	ListCount(<ListView control>)
	```

## Recuperar el elemento seleccionado en el control ListView
<a name="recuperar_elemento_seleccionado_control_listview_ELTTEXTE000422"></a>


### Obtener el índice del elemento seleccionado
<a name="obtener_indice_del_elemento_seleccionado_ELTPARAGRAPHE000140"></a>

Para get el índice del elemento seleccionado, utilice: 

- el elemento directamente:
	
	```txt
	Index = <ListView control>
	```


- la función [ListSelect](../WDLang1/3049011.md): 
	
	```txt
	Index = ListSelect(<ListView control>)
	```




<a name="NOTE4_1a"></a>
**Observación**: Para los controles ListView de selección múltiple, se utiliza [ListSelect](../WDLang1/3049011.md) para recuperar los elementos seleccionados.

```wl
// Retrieve the elements of a multi-selection ListView control
Rank is int = 0  // selection number
Index is int = 0  // index of the selected element
LOOP
	Rank = Rank + 1
	Index = ListSelect(LSV_ListView1, Rank)
	IF Index = - 1 THEN BREAK
END
```

<a name="NOTE4_2"></a>


### Obtener el valor del elemento seleccionado
<a name="obtener_valor_del_elemento_seleccionado_ELTPARAGRAPHE000180"></a>

Para get el valor del elemento seleccionado, utilice una de las siguientes sintaxis: 

- Sintaxis 1:
	
	```txt
	Index is int = <ListView control>
	<Variable> = <ListView control>[Index]
	```


- Sintaxis 2:
	
	```txt
	<Variable> = <ListView control>[<ListView control>]
	```


- Sintaxis 3: Propiedad [DisplayedValue](../Proprietes/2510129.md)
	
	```txt
	<Variable> = <ListView control>.DisplayedValue
	```

	**Atención**: El valor almacenado puede ser diferente del valor que muestra. Para obtener el valor almacenado correspondiente a una de las filas del control ListView, utilice el [StoredValue](../Proprietes/2510127.md) Property.

## Propiedades específicas de los controles ListView
<a name="propiedades_especificas_los_controles_listview_ELTTEXTE000458"></a>
Las siguientes propiedades son específicas de los controles ListView:


|   |   |
| --- | --- |
| **All tipos de controles ListView** |
| [Memoria](../Proprietes/2510076.md) | Indica si la control ListView especificada se rellena mediante programación o se basa en un archivo de datos. |
| [ListViewMode](../Proprietes/2510079.md) | Establece el modo de visualización de la control ListView (modos ListView, List Box y panorama). |
| [Conde](../WDLang1/3025009.md) | Obtiene el número de filas de una control ListView. |
| [VerticalOrientation](../Proprietes/2510092.md) | Obtiene y establece la orientación de una control ListView. |
| [FillType](../Proprietes/1000017723.md) | Indica cómo se rellena un control ListView (de forma programada, desde un archivo de datos o desde un Variable). |
| [DisplayedValue](../Proprietes/2510129.md) | Obtiene el valor de la fila actual o el valor de una fila específica en la control ListView. |
| [Vacío](../Proprietes/2510126.md) | Indica si una control ListView está vacía. |
| **Control ListView rellenado mediante programación únicamente** |
| [clasificadas](../Proprietes/2510115.md) | Indica si un control ListView poblado programáticamente está ordenado o clasifica un control ListView (ordenación basada en las etiquetas de los elementos). |
| **Control ListView basado en un archivo de datos únicamente** |
| [BrowsedFile](../Proprietes/2510040.md) | Obtiene y establece la el archivo de datos o consulta utilizada para mostrar los registros en los controles de ListView. |
| [AutoBrowse](../Proprietes/2510093.md) | Indica si una control ListView se pasa en bucle de forma automática o programada. |
| [DisplayedItem](../Proprietes/2510123.md) | Obtiene y establece el elemento que corresponde al texto mostrado en una control ListView. |
| [VisualizarItemImagen](../Proprietes/2510109.md) | Obtiene y establece el elemento que corresponde a la Image mostrada en una control ListView. |
| [StoredItem](../Proprietes/2510110.md) | Obtiene y establece el elemento almacenado de una control ListView. |
| [BrowsedItem](../Proprietes/2510111.md) | Obtiene y establece el elemento utilizado para recorrer automáticamente los controles de ListView. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control ListView, consulte [Propiedades associated con controles ListView](../WDChamp/1013219.md).

## Creación de una control en una ventana
<a name="creacion_una_control_una_ventana_ELTTEXTE000641"></a>


### Creación de una control en una ventana
<a name="creacion_una_control_una_ventana_ELTPARAGRAPHE000047"></a>

Para crear una control en una ventana:

1. Ir a la pestaña "Creación" en el editor de ventanas. Esta pestaña contiene todos los tipos de controles que se pueden crear en una ventana. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen_Creation_champ.gif&type=thumb)
 Según el tipo de control a crear: 

	- haga clic en su Icon. 

	- si aparece una flecha junto al nombre de control, amplíe la lista y seleccione el tipo de control a crear.




2. Haga clic en la ubicación en la que desea crear el control.




**Observación**: Para crear una nueva columna en un control Tabla desde el editor de la ventana:

1. Seleccione el control Tabla.

2. Haga clic derecho para abrir el menú contextual: 

	- seleccione "Añadir una columna". 

	- seleccione el tipo de columna que desea crear.






<a name="NOTE2_2"></a>


### Creando un control a partir de un diccionario control
<a name="creando_control_partir_diccionario_control_ELTPARAGRAPHE000070"></a>

WINDEV permite crear una control a partir de una control que se encuentra en un diccionario (archivo ".DIC"). Un diccionario permite almacenar elementos (controles, ventanas,...) para un uso futuro en los nuevos proyectos.. Para obtener más información, consulte [Diccionario](../Editeurs/2023002.md).

Para crear un control a partir de un control encontrado en un diccionario:

1. Visualiza la pantalla de ["Panel"Diccionario](../Editeurs/2027014.md).

2. Abra el diccionario si es necesario ("Abrir" en el menú contextual).

3. Seleccione la inicial control que se encuentra en el diccionario (en la categoría "Ventana control s").

4. Realice un Drag and Drop de la control encontrada en el diccionario hacia su futura posición en la ventana.

5. Suscriba el control creado al diccionario si es necesario. Si el control creado se suscribe al diccionario, las modificaciones hechas al control que se encuentran en el diccionario se aplicarán automáticamente al control creado. Para obtener más información, consulte [Suscribir un elemento al diccionario](../Editeurs/2023001.md).

### Creación de una control a partir del contenido del portapapeles
<a name="creacion_una_control_partir_del_contenido_del_portapapeles_ELTPARAGRAPHE000094"></a>

El editor de ventanas se utiliza para crear varios tipos de control según el contenido del portapapeles:

| Contenido del portapapeles | Tipo de control creado |
| --- | --- |
| Texto, por ejemplo, copiado de un editor de texto. | Control Estático |
| Archivo Image copiado desde el explorador de Windows. | control Imagen |
| Image, copiado de un HTML Page por ejemplo. | control Imagen |
| Archivo description, copiado del editor de análisis o del ["Panel"Análisis](../Editeurs/2027012.md). | Tabla basada en un archivo de datos |
| Elemento de archivo, copiado de la página web de ["Panel"Análisis](../Editeurs/2027012.md). | Control creado por RAD según el tipo de control especificado en la pestaña "RAD" del elemento description. |



**To crear un control a partir del contenido del portapapeles**: 

1. copy el elemento al portapapeles: en la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar" (o Ctrl + C).

2. En el editor de ventanas, en la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar" (o Ctrl + V).




**Observación**: Puede elegir explícitamente el formato en el que se debe pegar el contenido del portapapeles en el editor de ventanas.: en la pestaña "Inicio", en el grupo "Edición", amplíe "Pegar" y seleccione "Pegado especial". Para obtener más información, consulte [Pegado especial](../Editeurs/2026001.md).

### Creando un control Imagen a partir de un archivo Image por medio de arrastrar y soltar
<a name="creando_control_imagen_partir_archivo_image_por_medio_arrastrar_soltar_ELTPARAGRAPHE000180"></a>

WINDEV permite crear una control Imagen arrastrando un archivo Image desde el explorador de Windows y soltándolo en una ventana de WINDEV.

Para crear una control Imagen a partir de un archivo Image:

1. Seleccione el archivo Image deseado en el Explorador de Windows.

2. Mantenga pulsado el botón izquierdo del ratón y mueva el archivo Image a una ventana abierta en el editor WINDEV.

3. Suelte el botón del ratón. Se crea automáticamente un control Imagen associated con el archivo Image.




**Observación**:
Los siguientes formatos de Image están soportados en Windows:

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- CD de fotos Kodak (\*.pcd)

- Pincel de pintura (\*.pcx)

- Formato Adobe Photoshop (\*.psd)

- TrueVision TARGA (\*.tga)

- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- Gráficos de red portátiles (\*.png)

- MetaFicheros de Windows (\*.emf;\*.wmf)

- Iconos (\*.ico \*;.icw)
	Observaciones: 

	- Pantalla HQ (High Quality) disponible.

	- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).

## Vinculación de un elemento control a un archivo de datos
<a name="vinculacion_elemento_control_archivo_datos_ELTTEXTE000714"></a>
Excluyendo los botones, las pestañas, los controles de forma y las barras de herramientas, todos los controles de la ventana pueden estar vinculados a un elemento de un archivo de datos.

Para vincular un elemento existente de control a un archivo de datos:

1. Seleccione el control deseado.

2. Haga clic con el botón derecho del ratón y seleccione "description" en el menú contextual, luego vaya a la pestaña "Binding" en la ventana de control de description.

3. Introduzca la información sobre el artículo encuadernado.




**Observación**: También tiene la capacidad de crear directamente:

- un control Campo de entrada de un artículo de un archivo de datos:

	1. En la ["Panel"Análisis](../Editeurs/2027012.md), seleccione el elemento que estará vinculado a la control que se va a crear.

	2. Arrastre y suelte el elemento en la posición en la que desea crear la edición ligada control.




- un control Tabla de un archivo de datos:

	1. En el ["Panel"Análisis](../Editeurs/2027012.md), seleccione el el archivo de datos que estará relacionado con el control Tabla a crear.

	2. Arrastre y suelte el elemento en la posición en la que desea crear la control Tabla vinculada. El asistente para crear un control Tabla se abre automáticamente.

## Seleccionar
<a name="seleccionar_ELTTEXTE000738"></a>


### Selección de uno o más controles en una ventana
<a name="seleccion_uno_mas_controles_una_ventana_ELTPARAGRAPHE000229"></a>

WINDEV le permite seleccionar uno o más controles de ventana con el teclado o con el ratón. Para obtener más información, consulte [Seleccionar uno o más controles](../Editeurs/2026007.md).

**Caso especial: Controles agrupados**

- Un clic realizado en una control agrupada selecciona todas las control agrupadas.

- Dos clics sucesivos realizados en un control agrupado selecciona este control solamente.



<a name="NOTE4_2"></a>


### Agrupación de controles en una ventana:
<a name="agrupacion_controles_una_ventana_ELTPARAGRAPHE000244"></a>

Se pueden agrupar varios controles. Esta agrupación simplifica el movimiento de varios controles, por ejemplo.

**Recordatorio**: Hay dos tipos de grupos disponibles:

- el control le dio a manipular varios control en el editor de la ventana.

- el [grupos de controles](../WDChamp/1010016.md) a manipular varios controles a través de la programación.




A controles de grupo:

1. Seleccione los mandos a agrupar (ver [Seleccionar uno o más controles](../Editeurs/2026007.md)).

2. En la pestaña "Modificación", en el grupo "Profundidad y alineación", haga clic en "Agrupar selección" (Ctrl + G).




Para disociar controles agrupados:

1. Seleccione una control que pertenezca al grupo. WINDEV selecciona automáticamente todos los controles agrupados.

2. En la pestaña "Modificación", en el grupo "Profundidad y alineación", amplíe "Agrupar selección" y seleccione "Disociar selección" (Ctrl + Shift + G).

## Copiar los controles de las ventanas
<a name="copiar_los_controles_las_ventanas_ELTTEXTE000776"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000311"></a>

Se pueden copiar uno o varios controles en la misma ventana o de una ventana a otra.

El description y el código de control copiado serán idénticos a los del control inicial. Sólo el número de ayuda de la control copiada (pestaña "Ayuda") se inicializará a 0.
<a name="NOTE5_2"></a>


### Copiar una control en la misma ventana
<a name="copiar_una_control_misma_ventana_ELTPARAGRAPHE000320"></a>

Para copiar una control en la misma ventana:

- **método 1**:

	- Selecciona el control a copy.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar".
			Observación: También puedes pulsar Ctrl + C.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar".
			Observación: También puedes pulsar Ctrl + V.

	- Posiciona la control con el ratón (el contorno de la control aparece en líneas punteadas y te ayuda a posicionar la nueva control).







- **método 2**:

	- Selecciona el control a copy.

	- Haga clic con el botón derecho del ratón en el control de copy y mantenga pulsado el botón del ratón.

	- Mueve la control con el ratón (el contorno de la control aparece en líneas punteadas y te ayuda a posicionar la control para pegar).

	- Suelte el botón del ratón.

	- Seleccione "copy aquí".

	- Un nuevo nombre se asigna automáticamente a la copia de control. Este nombre puede modificarse en el description de control copiado ("description" en el menú contextual de control).






<a name="NOTE5_3"></a>


### Copiar una ventana control a otra ventana
<a name="copiar_una_ventana_control_otra_ventana_ELTPARAGRAPHE000371"></a>

Para copiar una control de una ventana a otra ventana:


- **método 1**:

	- En la primera ventana, seleccione la opción control a copy.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar".
			Observación: También puedes pulsar Ctrl + C.

	- Mostrar la ventana de destino en el editor de ventanas.

	- En la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar".
			Observación: También puedes pulsar Ctrl + V.

	- Posiciona la control con el ratón (el contorno de la control aparece en líneas punteadas y te ayuda a posicionar la nueva control).







- **método 2**: (ambas ventanas deben ser mostradas en el editor)

	- En la primera ventana, seleccione la opción control a copy.

	- Mantenga pulsado el botón izquierdo del ratón y arrastre la control hasta la ventana de destino.

	- Suelte el botón del ratón una vez que el control esté posicionado en la ventana de destino (el Border punteado del control le permite posicionar fácilmente el control pegado).

## Mover
<a name="mover_ELTTEXTE000818"></a>


### Mover uno o más controles de ventana
<a name="mover_uno_mas_controles_ventana_ELTPARAGRAPHE000433"></a>

Para mover uno o más controles de ventana:

1. Seleccione los controles que desea mover.

2. Mantenga pulsado el botón izquierdo del ratón y mueva los controles con el ratón.. El verificador de interfaz en tiempo real se utiliza para alinear los controles (véase el siguiente párrafo).


**Observación**: Los controles seleccionados también se pueden mover con las teclas de flecha que se encuentran en el teclado. Para acelerar el movimiento, presione la tecla Ctrl junto con la tecla de flecha.
<a name="NOTE6_2"></a>


### Mover una columna de una control Tabla en el editor
<a name="mover_una_columna_una_control_tabla_editor_ELTPARAGRAPHE000445"></a>

Para mover una columna de una control Tabla en el editor:

- **método 1**:

	- Abra la ventana que contiene el programa control Tabla.

	- Seleccione la control Tabla y haga clic en la cabecera de la columna a mover.

	- Seleccione "Mover a la izquierda" o "Mover a la derecha" en el menú contextual.




- **método 2**:

	- Abra la ventana de descripción del control Tabla.

	- Seleccione la columna que desea desplazar.

	- Utilice las flechas situadas a la derecha del nombre de control Tabla para desplazar la columna.

	- Valide la ventana de descripción.

### Controles de bloqueo
<a name="controles_bloqueo_ELTPARAGRAPHE000464"></a>

En algunos casos, es posible que desee bloquear los controles en una posición fija.: en este caso, el control no puede ser modificado en su posición y el control no es seleccionado durante una selección realizada con el lazo. Esta característica es a menudo useful para los controles de diseño.

Para bloquear un control: 

1. Seleccione la control.

2. Utilice uno de los siguientes métodos:

	- en la pestaña "Modificación", en el grupo "Ayuda edición", seleccione la opción "Bloqueado".

	- presiona Alt + V.

### Cambiar el tamaño de los controles de la ventana
<a name="cambiar_tamano_los_controles_ventana_ELTPARAGRAPHE000511"></a>

Para cambiar el tamaño de una control en una ventana, tiene la posibilidad de utilizar: 

- el ratón (a través de los mangos de tamaño de la control) 
	Observación: En este caso, el redimensionamiento del control y el contenido es WYSIWYG. 

- la siguiente métodos abreviados de teclado en la control seleccionada: 

	- Alt flecha derecha: Amplíe la control desde la derecha. 

	- Alt flecha izquierda: Amplíe la control desde la izquierda.

	- Altar flecha hacia arriba: Amplíe la control desde la parte superior.  

	- Altar flecha abajo: Ampliar la control desde el fondo.

### Utilizar un control fuera de la ventana
<a name="utilizar_control_fuera_ventana_ELTPARAGRAPHE000532"></a>

En la mayoría de los casos, los controles pertenecen a una ventana. Por lo tanto, estos controles aparecen en la edición y en el tiempo de ejecución. Sin embargo, un control puede ser posicionado en la ventana del anfitrión de la ventana: se dice que el control está "fuera de la ventana". El control será visible en modo de edición e invisible en tiempo de ejecución. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenFenAccueil.gif)
 Un control "fuera de la ventana" se utiliza para:

- Mantener una control en una ventana sin mostrarla en tiempo de ejecución. Esto le permite utilizar el contenido del control mediante programación (como una variable).

- Mostrar nuevos controles en una ventana al cambiar el tamaño de la ventana.



<a name="NOTE6_7"></a>


### Cambiar el tamaño de la ventana
<a name="cambiar_tamano_ventana_ELTPARAGRAPHE000544"></a>

Cuando la ventana se redimensiona (por el usuario o mediante programación), los controles que se encuentran en la zona de ampliación de la ventana se muestran automáticamente.
Por ejemplo:

- En el editor, la ventana "login" es la siguiente: <br>![El botón Detalles se utiliza para ampliar la altura de la ventana.](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChpHorsFen1.gif)


- En el momento de la ejecución, una vez que se ha pulsado el botón "Detalles", la ventana de "inicio de sesión" es la siguiente: <br>![Los controles fuera de ventana se muestran automáticamente.](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChpHorsFen2.gif)





Para evitar que los controles "off-window" se muestren cuando se cambia el tamaño de la ventana, estos controles deben estar ubicados Above la ventana (parte superior de la pantalla de salpicadura).

### Creando un "fuera de la ventana" control
<a name="creando_fuera_ventana_control_ELTPARAGRAPHE000562"></a>

Para crear un "fuera de la ventana" control:

1. Ir a la pestaña "Creación" en el editor de ventanas. Esta pestaña contiene todos los tipos de controles que se pueden crear en una ventana. Dependiendo del tipo de control a crear: 

	- haga clic en su Icon. 

	- si aparece una flecha junto al nombre de control, amplíe la lista y seleccione el tipo de control a crear.




2. En la ventana de inicio, haz clic en el lugar donde debe crearse el control.

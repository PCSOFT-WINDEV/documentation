
## Vinculación de una OLE control a un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Características del artículo
<a name="caracteristicas_del_articulo_ELTTEXTE000065"></a>
Un OLE control se puede vincular a un elemento en un archivo de datos , consulta, etc.. Los elementos "Sonido, Image y binario" pueden vincularse a los controles OLE. La información almacenada corresponde al nombre del archivo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Actualización de la partida con los datos de la OLE control
<a name="actualizacion_partida_con_los_datos_ole_control_ELTTEXTE000089"></a>
**1er caso: los controles de la ventana están vinculados a elementos de un archivo de datos**
Todos los elementos de el archivo de datos (o de un archivo relacionado) que estén vinculados a un control en la ventana serán actualizados por [ScreenToFile](../WDLang4/3044146.md).

**2º caso: los controles de la ventana no están vinculados a elementos de un archivo de datos**
El valor de cada control debe asignarse a cada elemento utilizando la siguiente sintaxis:


```txt
<Data file>.<Item> = <Control name>
```

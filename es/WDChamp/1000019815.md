## Presentación
<a name="presentacion_ELTTEXTE000275"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten manipular a control Mapa mediante la programación. Para ello, utilice la Variable de la control Mapa en el código.

La Variable del control Mapa corresponde al nombre del control Mapa.

Esta ayuda Page explica cómo manipular a control Mapa a través de la programación. 

**Observación**: Varias funciones de WLanguage son específicas para la gestión de los controles de mapas. Para obtener más información, consulte [Funciones de control Mapa](../WDLang3/1000019955.md).

### Visualización de un mapa
<a name="visualizacion_mapa_ELTPARAGRAPHE000060"></a>

Al abrir una ventana que contiene un control Mapa, se muestra un mapa del mundo automáticamente: el mapa está centrado en Europa, con un nivel mínimo de zoom.

Puede mostrar la última posición: basta con seleccionar "Recordar última posición mostrada" en la pestaña "General" de la ventana control Mapa description. En este caso, se muestra la última posición con el mismo nivel de zoom. 

Para visualizar un mapa correspondiente a una ubicación específica, utilice: 

- la función [MapDisplayPosition](../WDLang3/1000019787.md): Esta función se puede utilizar con una posición específica ([geoPosition](../WDLang3/1000019191.md) Variable), o en una ciudad o un monumento. 
	Por ejemplo: 
	
	```wl
	// Centers the map displayed by the "MAP_Position" control 
	// on a geographical location 
	MyPosition is geoPosition
	MapDisplayPosition(MAP_Position, MyPosition)
	
	// Centers the map displayed by the "MAP_Position" control on a city 
	// from its name
	MapDisplayPosition(MAP_Position, "Montpellier, France")
	
	// Centers the map displayed by the "MAP_Position" control on a monument 
	// from its name
	MapDisplayPosition(MAP_Position, "Eiffel Tower")
	```


- un comando de asignación directa con una posición específica. En este caso, la posición se visualiza sin ninguna animación.. 
	Ejemplo: 
	
	```wl
	// Centers the map displayed by the "MAP_Position" control 
	// on a geographical location
	MyPosition is geoPosition
	MAP_Position = MyPosition
	```

## Las siguientes funciones pueden utilizarse con los controles de mapa
<a name="las_siguientes_funciones_pueden_utilizarse_con_los_controles_mapa_ELTTEXTE000347"></a>
Las funciones WLanguage que se pueden utilizar con los controles Map se pueden clasificar en varias categorías: 

- Las funciones utilizadas para gestionar las posiciones en el mapa: 
	


|   |   |
| --- | --- |
| [MapDisplayPosition](../WDLang3/1000019787.md) | Centra el mapa visualizado en una control Mapa en una posición geográfica o en un lugar concreto. |
| [MapGetPosition](../WDLang3/1000019790.md) | Devuelve la posición geográfica de un punto localizado en el centro del mapa que se está mostrando en un control Map. |
| [MapInfoPosition](../WDLang3/1000019951.md) | Devuelve las coordenadas (en píxeles) del punto correspondiente a una posición geográfica (latitud y longitud). |
| [MapInfoXY](../WDLang3/1000019950.md) | Devuelve la posición geográfica (latitud y longitud) correspondiente a un punto de una control Mapa. |

- Las funciones utilizadas para seguir los movimientos del dispositivo: 
	


|   |   |
| --- | --- |
| [MapEndOfMove](../WDLang3/1000019949.md) | Se detiene siguiendo el movimiento del dispositivo en una control Mapa. |
| [MapFollowMovement](../WDLang3/1000019953.md) | Muestra la ubicación actual del dispositivo en una control Mapa y actualiza la ubicación a medida que se mueve. |

- Las funciones utilizadas para gestionar los marcadores en el mapa. Estas funciones utilizan las variables de [Marcador](../WDLang3/1000019940.md) para definir las características de estos marcadores. 
	


|   |   |
| --- | --- |
| [MapAddMarker](../WDLang3/1000019794.md) | Añade una nueva Marker a un mapa mostrado en una control Mapa. |
| [MapDeleteAll](../WDLang3/1000019952.md) | Borra lo siguiente de una control Mapa: <br><br>	- todos los marcadores, <br><br>	- todas las formas,<br><br>	- todas las imágenes.<br><br><br> |
| [MapDeleteMarker](../WDLang3/1000019795.md) | Elimina una Marker de un mapa mostrado por una control Mapa. |
| [MapModifyMarker](../WDLang3/1000019796.md) | Modifica un marcador mostrado en un control Map568 |

- Las funciones utilizadas para gestionar los itinerarios en el mapa. 
	


|   |   |
| --- | --- |
| [MapAddItinerario](../WDLang3/1000020383.md) | Añade un itinerario a un control Mapa. |
| [MapDeleteItinerario](../WDLang3/1000020384.md) | Borra un itinerario de una control Mapa. |

- Las funciones utilizadas para manipular las opciones adicionales de Google API para la gestión de mapas: 
	


|   |   |
| --- | --- |
| [MapConfigureGglAPI](../WDLang3/1000021611.md) | Declara parámetros adicionales para el control Map que serán transmitidos a el Google API. |
| [MapExecuteJS](../WDLang3/1000021615.md) | Ejecuta el código nativo de Google Maps API en un control Mapa en JavaScript. |
| [MapGetJSObject](../WDLang3/1000021474.md) | Devuelve la instancia de la API de control Mapa. |

## Propiedades específicas de los controles del Mapa
<a name="propiedades_especificas_los_controles_del_mapa_ELTTEXTE000476"></a>
Las siguientes propiedades se utilizan para gestionar un control Mapa a través de la programación: 



|   |   |
| --- | --- |
| [MapMode](../Proprietes/1000019956.md) | Devuelve o modifica el modo de visualización del mapa en el control Mapa. |
| [Valor](../Proprietes/2510130.md) | Sirve para centrar el mapa en la posición correspondiente a la [geoPosition](../WDLang3/1000019191.md) especificada Variable. |
| [Zoom](../Proprietes/1000017201.md) | Devuelve o modifica el nivel de zoom utilizado para mostrar el mapa. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Mapa, consulte [Propiedades associated con controles de Mapas](../WDChamp/1000019814.md).

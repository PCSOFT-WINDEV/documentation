


## Cortando la ventana según un Image
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
Una Image permite definir fácilmente la forma de una ventana. WINDEV analiza este Image y utiliza la forma definida por el Image. Esta imagen se usará para especificar las secciones visibles y Invisible de la ventana.

Esto le permite crear interfaces excepcionales.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fenetredetouree.gif)

## Recorte de la ventana según la imagen de fondo
<a name="recorte_ventana_segun_imagen_fondo_ELTTEXTE000155"></a>


### Los pasos principales
<a name="los_pasos_principales_ELTPARAGRAPHE000021"></a>

WINDEV permite usar un Image directamente desde el editor para definir la forma de la ventana. Para ello:

1. Crear el Image que se utilizará como fondo Image de la ventana. La transparencia Color de este Image debe ser definida. Este color puede ser:

	- magenta.

	- la Color del primer píxel que se encuentra en la esquina superior izquierda de la Image.
			Para obtener más información, lea el siguiente párrafo.




2. En la description de la ventana:

	- marque "Clip the Window" en la pestaña "Image".. WINDEV propone adaptar automáticamente las opciones de estilo de la ventana para que get obtenga un mejor resultado.

	- especificar las opciones de transparencia correspondientes a la imagen de fondo utilizada (utilizar el magenta como transparencia Color por ejemplo).
			**Observación**: También tiene la posibilidad de configurar las opciones de transparencia de las ventanas. Para obtener más información, consulte [Efecto traslúcido de las ventanas](../Editeurs/9500103.md).




3. Validar. La ventana se recortará automáticamente en el editor.

## Creando la Image que define la forma de la ventana
<a name="creando_image_que_define_forma_ventana_ELTTEXTE000179"></a>
La Image que define la forma de la ventana se crea mediante una herramienta de dibujo o de edición de Image. Las diferentes áreas que conforman la forma de la ventana deben estar juntas: no debería haber "lagunas". 

En esta imagen, define el Color que se usará como transparencia Color cuando se recorte la ventana (por ejemplo, "Light Magenta"). Todas las áreas que contengan este color se harán Invisible cuando se muestre la ventana recortada.

**Atención**: Los bordes deben ser lo más nítidos posible (no Gradient, ...) para evitar los efectos de visualización inesperados cuando se prevea o se abra la ventana recortada. De hecho, un solo Color se utiliza para definir la transparencia de todo el Image.

## Uso de una ventana recortada
<a name="uso_una_ventana_recortada_ELTTEXTE000203"></a>
WINDEV permite:

- usar un menú desplegable en las ventanas recortadas. Para ello, especifica la posición del menú principal en la pestaña "Detalles" de la ventana de description.

- asociar una ayuda sensible al contexto con la ventana (a través del mecanismo de consejos de herramientas avanzadas). Para ello, cree un botón cuya acción predeterminada sea "Ayuda sensible al contexto".

- gestionar [el efecto translúcido de las ventanas](../Editeurs/9500103.md).

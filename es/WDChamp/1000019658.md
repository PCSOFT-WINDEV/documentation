
## Control Planificador: Personalizar citas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000164"></a>
En el control Planificador, el Appointment se muestra en un área específica. Usted tiene la capacidad de traducir y personalizar esta área: el color del título, el color de fondo de la Appointment, ...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Personalizar el aspecto de las citas
<a name="personalizar_aspecto_las_citas_ELTTEXTE000188"></a>
**Para cambiar el aspecto de las citas utilizadas por Default** , puede cambiar las opciones de estilo definidas para este elemento.  

1. Abre la ventana de control Planificador description. 

2. En la pestaña "Estilo", despliegue "Elemento" y seleccione "Appointment".. 




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Por ejemplo, para "Appointment", puede: 

- usa una sombra,

- usar un separador en puntos entre horas, 

- definir el título y las opciones de Background: 

	- fondo sólido, 

	- Gradient Background,

	- Fondo sólido, título multiLine y fondo Line. Esta opción se utiliza para mostrar el título en varias líneas (especialmente si la Appointment no es lo suficientemente ancha para mostrar el título completo)).




- definir las fuentes utilizadas, ....

### Personalización mediante ventanas internas
<a name="personalizacion_mediante_ventanas_internas_ELTPARAGRAPHE000052"></a>

**Para personalizar completamente las citas** (requerido para la traducción):

1. Abre la ventana de control Planificador description. 

2. En la pestaña "General", seleccione la ventana interna utilizada para las citas: 

	- "Ninguno": se utilizará el modo de visualización predeterminado. 

	- "Ventana predefinida": Si esta opción está seleccionada, el componente interno WDAAF se agrega inmediatamente al proyecto actual. 

		- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La ventana interna "IW_WinDevViewAPT" es automáticamente associated con la opción "Ventana interna para las citas".. 


 Observaciones: 

- Si el componente WDAAF ya existe, solo se agrega la ventana al componente.  

- Esta ventana también se utiliza para personalizar las citas en los controles del Organizador.  

- Cualquier ventana interna del proyecto. 

3. Si es necesario, valide la ventana de descripción del control. 




**Observaciones**: 

- La ventana interna personalizada está disponible en el panel "Explorador de proyectos", en la carpeta "Componentes internos".. Se puede editar en el editor de ventanas. 
	**Atención**: esta ventana interna sólo puede utilizarse para personalizar la visualización de las citas. Solo se ejecuta el evento "Declaraciones globales". No se ejecutan otros eventos. Por lo tanto, no se pueden agregar botones con un código de clic específico. 

- Si utiliza la ventana de preajustes propuestos, se aplicarán las opciones de estilo de la Appointment (seleccionadas en la etiqueta "Estilo").

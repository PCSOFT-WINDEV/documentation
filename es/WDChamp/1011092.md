
## control Tabla dinámica en un reporte
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000121"></a>
La editor de reportes le permite crear informes "Crosstab".. En este caso, existen dos posibilidades para una aplicación WINDEV: 

- la fuente de datos corresponde a varios archivos de datos: se creará un Crosstab reporte basado en una consulta. 

- la fuente de datos corresponde a una control Tabla dinámica presente en una ventana (ejemplo en esta página): el reporte creado contiene un control Tabla dinámica. 




Cuando se crea un reporte basado en un control Tabla dinámica de una ventana, el reporte contiene sólo un control Tabla dinámica. Este tipo de control sólo puede crearse en este tipo de reporte: no puede crearse independientemente de reporte. 

En el editor, este control incluye dos bloques: 

- El bloque "Cabecera" que muestra los títulos de las columnas de la tabla dinámica. 

- El bloque "Tabla Pivotante" que muestra los títulos de las filas y los datos de associated. 




Para más detalles sobre las diferentes fuentes de una "Crosstab" reporte, véase [Apuñalamiento cruzado reporte](../WDChamp/1011062.md).

## Creación de un Crosstab reporte basado en un control Tabla dinámica
<a name="creacion_crosstab_reporte_basado_control_tabla_dinamica_ELTTEXTE000145"></a>


### Crear un reporte
<a name="crear_reporte_ELTPARAGRAPHE000031"></a>

Para crear una "Tabla Pivotante" reporte: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

2. Aparece la ventana de creación: haga clic en "Reporte", y luego en "Reporte".

3. El asistente de creación de reportes se abre. 

4. En esta asistente, especifique:

	- el tipo de reporte a crear: "Tabla cruzada".

	- el control Tabla dinámica utilizado como fuente de datos.

	- la plantilla de la reporte (si es necesario).

	- el nombre y el título de la reporte. Este nombre (".WDE". archivo correspondiente a la reporte) se utilizará para identificar la reporte en sus programas.

## Características de una tabla dinámica reporte y una control Tabla dinámica
<a name="caracteristicas_una_tabla_dinamica_reporte_una_control_tabla_dinamica_ELTTEXTE000169"></a>


### Características de la tabla dinámica (editor Reporte)
<a name="caracteristicas_tabla_dinamica_editor_reporte_ELTPARAGRAPHE000052"></a>

Puede modificar las siguientes características de una tabla dinámica reporte: 

- La fuente de datos del reporte puede modificarse en la pestaña "Datos" de la ventana description del reporte ("Reporte description" en el menú contextual del reporte). 

- La pestaña "General" de la ventana de description de control Tabla dinámica se utiliza para especificar: 

	- **el modo de cálculo de la anchura de las celdas**: las opciones disponibles son: 

		- Utilizar ancho definido en edición:  el ancho de las celdas será el mismo que el definido en la editor de reportes.
						Observación: Las manijas se pueden usar para redimensionar el área de datos en la control Tabla dinámica que se encuentra en el reporte. 

		- Llene la página tanto como sea posible en ancho: las celdas se ampliarán para llenar el espacio disponible en la página.

		- Evitar celdas truncadas: las celdas se ampliarán para mostrar todos los datos si hay suficiente espacio en la página.




- **las dimensiones en expansión**: las opciones disponibles son: 

	- Como el control: el nivel de detalle mostrado en el reporte será el mismo que el mostrado en el control Tabla dinámica utilizado como fuente de datos.

	- Todo expandido: se mostrarán todos los datos encontrados en la tabla pivotante.

	- Todo contraído: solo se mostrará el primer nivel de detalles de la Tabla dinámica.

- La pestaña "Estilo" de la ventana de description del PVT control se utiliza para definir el estilo del control: la opción "Definir el estilo independientemente del control PVT impreso" se utiliza para definir un estilo específico para el control que se encuentra en el reporte. Si esta opción está seleccionada, puede especificar el estilo para las cabeceras de fila y para los datos.

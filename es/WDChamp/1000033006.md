## Presentación

Los principales AAFs afectan a:

## Menú contextual de control Hoja de cálculo
<a name="menu_contextual_control_hoja_calculo_ELTTEXTE000302"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000036"></a>

Cada control Hoja de cálculo que se encuentra en una ventana propone un menú al usuario.

Este menú, disponible sin programación, incluye varias acciones:

- Cancel/Restaurar la última acción. 

- Cortar/copy/Pegar: Estas opciones se pueden aplicar a una celda, una columna, un fila o una selección de celdas. 

- Cancel zoom: Esta opción se utiliza para Cancel el zoom realizado por Ctrl + Rueda de ratón por ejemplo. 

- Insertar: Esta opción se utiliza para insertar una fila o una columna. 

- suprimir: Esta opción se utiliza para borrar el contenido de una celda, fila o columna.. Atención: todo el contenido se borra inmediatamente (valor, fórmula, ....). 

- Guardar como: Esta opción se utiliza para guardar el contenido de control Hoja de cálculo en un nuevo archivo xlsx.. 




Observación: Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md). 


### Ajustes
<a name="ajustes_ELTPARAGRAPHE000055"></a>

El menú contextual de control Hoja de cálculo puede desactivarse. Para configurar la visualización del menú contextual associated con el control Hoja de cálculo, debe:

1. Abra la ventana control Hoja de cálculo description.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contextual". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, aparecerá el menú automatic de control Hoja de cálculo. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú FAA](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las FAA. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las FAA y restaurar el menú estándar de Windows para un control determinado.

- Algunas opciones específicas pueden ser desactivadas por [AAFDisable](../WDLang1/1000022018.md).

## Barra de herramientas y barra de fórmulas
<a name="barra_herramientas_barra_formulas_ELTTEXTE000332"></a>
La control Hoja de cálculo incluye una barra de herramientas y una barra de fórmulas. 


### Barra de herramientas
<a name="barra_herramientas_ELTPARAGRAPHE000104"></a>

La barra de herramientas permite al usuario final realizar las operaciones de formateo más comunes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Tableur_Barre_outils.gif)


- Cancel/Restaurar

- Nombre de la fuente

- Tamaño de fuente

- Zoom de la selección

- Formateo de la celda: Negrita, Cursiva, Subrayado, Tachado

- Frontera y color de la frontera

- Color de fondo

- Color del texto

- Alineación del texto: Alineado a la izquierda, Centrado, Alineado a la derecha

- Fusión de celdas

- Formato de celda




Usted puede: 

- mostrar (o no) esta barra de herramientas. 

	- En el editor: Opción "Con barra de herramientas" en la pestaña "Detalles" del control description. 

	- mediante programación: Propiedad [ToolbarVisible](../Proprietes/1000021333.md). 




- personalizar la barra de herramientas. Para obtener más información, consulte [Personalización de la barra de herramientas (control Hoja de cálculo)](../WDChamp/1000033001.md).





### Barra de fórmulas
<a name="barra_formulas_ELTPARAGRAPHE000136"></a>

La barra de la fórmula permite al usuario final identificar las celdas seleccionadas e introducir la fórmula o el valor associated con una celda. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Tableur_Barre_Formule.gif)
 Se utilizan dos botones para validar o Cancel la entrada. 

Usted puede: 

- mostrar (o no) esta barra de fórmulas. 

	- En el editor: Opción "Con barra de fórmulas" en la pestaña "Detalles" del control description. 

	- mediante programación: Propiedad [FormulaBarVisible](../Proprietes/1000021334.md). 




- personalizar la barra de fórmulas. Para obtener más información, consulte [Personalización de la barra de fórmulas (control Hoja de cálculo)](../WDChamp/1000033001.md).




Observación: Una fórmula empieza por "=".. Al teclear el signo "=", aparecen todas las funciones disponibles en la fórmula. Para utilizar una función, todo lo que tiene que hacer es seleccionarla.

## Introducción de valores y fórmulas
<a name="introduccion_valores_formulas_ELTTEXTE000368"></a>
La entrada de valores y fórmulas se realiza: 

- en la celda directamente. 

- en la barra de fórmulas de la celda actual si ésta es Visible. 




Observación: Si el control Hoja de cálculo permite la entrada de valores, el usuario final tendrá la posibilidad de añadir, modificar o eliminar valores en las celdas.. El usuario final también puede añadir filas y columnas. Para permitir la entrada de valores:  

- en el editor: Opción "Permitir la introducción de valores" en la pestaña "Detalles" del control description. 

- mediante programación: Propiedad [InputValue](../Proprietes/1000021336.md). 




Una fórmula siempre empieza por "=".. Una fórmula puede incluir: 

- una función, 

- una o más celdas procedentes de la hoja de trabajo actual o de otra hoja de trabajo. Para reference una celda en un cálculo, todo lo que tienes que hacer es hacer clic en la celda solicitada: el nombre de la celda se muestra en la barra de fórmulas.

## Gestionar las células
<a name="gestionar_las_celulas_ELTTEXTE000392"></a>
La control Hoja de cálculo le permite: 

- Seleccione una celda. 

- Seleccione un rango de celdas. 

- Seleccionar varias celdas. 


Las diferentes opciones de la barra de herramientas se pueden aplicar a estas celdas. 

También puede: 

- crear fórmulas que contengan celdas diferentes. 

- Células copy. Por ejemplo: 

	- Seleccionar una celda que contenga un valor o una fórmula. 

	- Seleccione el copy manipular (cuadrado en la parte inferior de la selección) y mueva este manipular: El valor inicial se copia en las celdas seleccionadas 




- realizar una copy incremental de un grupo de células. Por ejemplo: 

	- Seleccionar dos celdas verticales que contengan dos valores de una serie ("1" y "2" por ejemplo). 

	- Seleccione el copy manipular (cuadrado en la parte inferior de la selección) y mueva este manipular a la parte inferior: se actualizan los diferentes valores de las celdas seleccionadas.

## Gestión de hojas de trabajo
<a name="gestion_hojas_trabajo_ELTTEXTE000416"></a>
La control Hoja de cálculo puede ser multihoja de trabajo y gestionar las diferentes hojas de trabajo de un archivo xlsx.. 

En ese caso: 

- Las diferentes hojas de trabajo se muestran en la parte inferior izquierda de control Hoja de cálculo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Tableur_multifeuille.gif)
 El botón "+" sirve para añadir nuevas hojas de cálculo. 

- Un menú contextual permite a los usuarios finales gestionar las distintas hojas de cálculo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Tableur_multifeuilleMenu.gif)
Es posible: 

	- Insertar una hoja. 

	- Eliminar la hoja de trabajo actual. 

	- Renombrar la hoja de trabajo actual. 
			Observación: Las opciones de este menú se pueden configurar como el menú contextual de control Hoja de cálculo. 







Consejo: Si se muestran varias hojas de trabajo, puede desplazarse por las diferentes fichas correspondientes a las hojas de trabajo mediante los botones de flecha. 

- Para mostrar la pestaña de la primera hoja de trabajo, presione Ctrl + haga clic en la flecha izquierda. 

- Para mostrar la pestaña de la última hoja de cálculo, pulse Ctrl + clic en la flecha de la derecha.

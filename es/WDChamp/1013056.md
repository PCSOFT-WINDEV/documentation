## Visión general de la RTF control
<a name="vision_general_rtf_control_ELTTEXTE000185"></a>


### El RTF control sólo puede ser usado en un reporte
<a name="rtf_control_solo_puede_ser_usado_reporte_ELTPARAGRAPHE000011"></a>

El "RTF" (Rich Text Format) control se utiliza para mostrar una cadena de caracteres formateada en su reporte. Esta cadena de caracteres está en RTF, lo que significa que se utilizan etiquetas específicas para definir el formato de cada carácter o cadena de caracteres. Estas etiquetas no se muestran en la control.

Por ejemplo, un "RTF" control se utiliza para mostrar el siguiente texto:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RTFimage1.gif)


**To crear un "RTF" control:**

1. En la pestaña "Creación", en el grupo "Texto", haga clic en "RTF".

2. Haga clic en la ubicación en la que desea crear el control.




**Observaciones**: 

- **A partir de la versión 22, el control RTF de un reporte corresponde a un control Estático de tipo RTF.** 

- En una ventana, tiene la posibilidad de utilizar una control Campo de entrada RTF o una control estática RTF. Para más detalles, consulte [Control Campo de entrada](../WDChamp/1013145.md) o [Control Estático](../WDChamp/1013179.md).

## ¿Cómo modificar el texto de un control RTF en un reporte?
<a name="¿como_modificar_texto_control_rtf_reporte_ELTTEXTE000209"></a>
El texto de un control "RTF" puede modificarse:

- a través de la programación (con la [Pie de foto](../Proprietes/2510053.md) Property).
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: También puede utilizar [RTFLoad](../WDLang1/3025041.md) para asignar un valor al RTF control.

- en la editor de reportes. 




**Para modificar el texto del control RTF en el editor de reportes**: 

1. Seleccione la control. Los controladores de selección aparecen alrededor del control.

2. Haga clic de nuevo en el control. El cursor se convierte en un cursor de texto.

3. Introduzca el nuevo texto de la control.

4. La barra de herramientas de diseño le permite modificar (si es necesario) el formato del texto.

5. Al hacer clic fuera del control, este vuelve a ser de solo lectura.



<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000077"></a>

- El texto que se muestra en el RTF control puede contener el valor de un artículo impreso en el reporte.
	Para insertar el valor de un campo o control del reporte en el texto del control Estático, introduzca el nombre del campo o control entre [% y %].
	Por ejemplo, el texto de control puede corresponder a: El proveedor [%SupplierName%] vende el producto [%ProductName%] a [%PriceIOT%]. Los campos NombreProveedor, NombreProducto y PrecioConImpuestos serán reemplazados por su valor cuando se ejecute el reporte.

- El texto que aparece en el RTF control puede provenir de un elemento de texto que se encuentra en la fuente de datos de reporte.
	Si el control es relacionado a un artículo, el nombre del artículo asociado se muestra en el cuadro combinado "Artículo vinculado" de la pestaña "General" del control description.



<a name="NOTE2_3"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000115"></a>

Por Default: 

- El formato de los archivos RTF soportados por las impresiones WINDEV y WEBDEV es el mismo que el formato RTF soportado por "Wordpad" (hasta Windows XP).. Hay que tener en cuenta algunos límites adicionales cuando se exporta una reporte a una RTF Document (por ejemplo, no se admiten las tablas y las áreas de texto).

- Para imprimir una control RTF, el archivo "RICHED20.DLL" debe encontrarse en el ordenador actual. En la mayoría de los casos, el archivo "RICHED20.DLL" se encuentra en el directorio del sistema de Windows.

- Si el archivo RTF es generado por otro programa (Microsoft Word, etc.), se recomienda abrir el archivo en Wordpad para comprobar la compatibilidad del RTF. Wordpad proporcionará una vista previa de impresión más realista.

- En todos los casos, le recomendamos que vuelva a guardar el archivo en Wordpad para forzar el formato RTF.

- Las opciones de zoom del RTF control no son compatibles con la visualización de enlaces en el texto. Si los enlaces son añadidos por [AddLink](../WDLang1/1000019038.md), el zoom se forzará automáticamente a 100%.

- Las tablas y las imágenes que se encuentran en una Document RTF no se pueden recuperar en una control RTF.
	Para recuperar las tablas e imágenes de un documento RTF en un control RTF, marque "RTF con imágenes y tablas" (pestaña "General" en la ventana description de control). En este caso, la compatibilidad con Wordpad y la presencia de RICHED20.DLL ya no son necesarias.




Si utiliza [iParameter](../WDLang5/3046008.md) con el parámetro "RTFWINDOWS=NO" o con la constante *iRTFWithImagesAndTables*, podrá imprimir tablas e imágenes en los controles RTF. La compatibilidad con Wordpad y la presencia de RICHED20.DLL ya no son necesarias..


## Ejemplo de Flexbox: automatic control disposición
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
Los controles Flexbox permiten organizar los controles exactamente como se desee cuando se cambia el tamaño del navegador.

He aquí un ejemplo de configuración. 

A Page contiene 5 celdas con imágenes y texto. Estas 5 celdas están en un Flexbox control.

Con el Flexbox control, cuando el navegador se redimensiona horizontalmente, las 5 celdas se posicionan automáticamente según 3 reglas:

- las celdas deben disponerse en una o varias filas en función de la anchura disponible,

- todas las células deben permanecer justificadas,

- las celdas deben seguir teniendo el mismo tamaño.




Observación: Esta Page se basa en el ejemplo "WW_Flexbox".





## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000129"></a>
Para reorganizar las celdas en función del tamaño del navegador: 

1. Colocar las celdas control s en un Flexbox control: 3 celdas en la primera fila y 2 celdas en la segunda. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple1%20-%20HC%20N%B0001.gif&type=thumb)


2. Abrir el control Flexbox description. Las opciones a modificar se encuentran en la pestaña "Detalles" de la ventana description.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Flexbox_Exemple1%20-%20HC%20N%B0002.gif&type=thumb)
 Cada regla a aplicar corresponde a una opción: 

	- cuando se reduce la anchura del navegador, las celdas pasan a la siguiente fila: 
			Para aplicar esta regla, vaya a "Salto de línea (flex-wrap)" y elija "envolver".. 

	- las celdas deben ocupar toda la anchura disponible y estar espaciadas uniformemente: Para aplicar esta regla, vaya a "Alineación horizontal (justificar-contenido)" y elija "espacio-alrededor". 

	- las celdas deben tener el mismo tamaño: Para aplicar esta regla, el valor "Alineación vertical en cada línea (alinear-elementos)" se set a "flex-start".







Para obtener una interfaz limpia, el espaciado entre filas y columnas es set a 20 píxeles.

Las opciones de anclaje del control Flexbox son set de forma que ocupe todo el espacio horizontal disponible y se adapte verticalmente para ajustarse a su contenido.

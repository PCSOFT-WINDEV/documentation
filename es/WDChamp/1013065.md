
## control Looper se llena de forma programada
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000070"></a>
Una control Looper rellenada programáticamente es directamente relacionado a las listas almacenadas en la memoria. Gestiona automáticamente mecanismos avanzados para el manejo de listas.

Una control Looper rellenada programáticamente corresponde a varias listas utilizadas en paralelo. Por lo tanto:

- añadir o insertar un fila en el control Looper realiza una inserción en todos los atributos/controles.

- borrar un fila del control Looper realiza un borrado de todos los atributos/controles.




El contenido de la control Looper debe ser inicializado por programación.

Para obtener más información, consulte [El looper se rellena de forma programada: WLanguage](../WDChamp/1013079.md).

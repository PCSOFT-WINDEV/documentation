


## ¿Cómo llenar un control TreeView desde un archivo de datos? (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **El enlace de datos está disponible en los controles TreeView**: el control puede ser directamente relacionado a un archivo de datos o Variable. Para obtener más información, consulte [control TreeView relacionado a un archivo Variable o de datos](../WDChamp/1013126.md).Por Default, el control TreeView no puede mostrar automáticamente los datos procedentes del un archivo de datos.

Veamos algunos ejemplos utilizados para realizar esta operación.

Este algoritmo se utiliza para rellenar un Treeview control a partir de un archivo de datos:

```wl
// Browsing the data file
HReadFirst(FileName, KeyName)
WHILE HOut() = False
	// Add to the TreeView control
	TREE_TreeViewControlName.Add(MyRoot + TAB + Branch + TAB + Leaf)
	HReadNext(FileName, KeyName)
END
```


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000119"></a>


### Ejemplo 1
<a name="ejemplo_1_ELTPARAGRAPHE000142"></a>


```wl
// Browsing the data file
HReadFirst(Company, City)
WHILE HOut() = False
	// Add to the TreeView control
	TREE_TreeViewControlName.Add(Company.City + TAB + Company.CompanyName)
	HReadNext(Company, City)
END
```

<a name="NOTE2_2"></a>


### Ejemplo 2
<a name="ejemplo_2_ELTPARAGRAPHE000157"></a>


```wl
// Browsing the data file
HReadFirst(Customer, CustName)
WHILE HOut() = False
	// Add to the TreeView control
	TREE_TreeViewControlName.Add(Left(Customer.CustName, 1) + TAB + Customer.CustName)
	HReadNext(Customer, CustName)
END
```

<a name="NOTE2_3"></a>


### Ejemplo 3
<a name="ejemplo_3_ELTPARAGRAPHE000172"></a>


```wl
// Browsing the data file
FOR EACH Customer ON CustName
	// Add to the TreeView control
	TREE_TreeViewControlName.Add(Left(Customer.CustName, 1) + TAB + Customer.CustName)
END
```

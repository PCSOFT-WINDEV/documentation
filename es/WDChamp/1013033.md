
## Asociar un identificador a un elemento encontrado en una control TreeView
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Identificador de elemento
<a name="identificador_elemento_ELTTEXTE000091"></a>
WINDEV, WINDEV Mobile y WEBDEV le ofrecen la posibilidad de asociar un identificador a cada elemento que se encuentra en un control TreeView. Este identificador puede ser un string o un entero.

Este identificador puede utilizarse para almacenar una información específica (número del Record correspondiente al elemento seleccionado en el control TreeView por ejemplo).

**Observación**: También tienes la capacidad de asociar una Image con un elemento. Para obtener más información, consulte [Asociar una Image con los elementos](../WDChamp/1013034.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Asociar un identificador a un elemento
<a name="asociar_identificador_elemento_ELTTEXTE000115"></a>
Para asociar un identificador a un elemento encontrado en una control TreeView, este identificador debe ser especificado en las siguientes funciones de programación:



|   |   |
| --- | --- |
| [TreeAdd](../WDLang1/3018012.md) | Agrega un elemento "hoja" (y todos los nodos necesarios) en un control TreeView. |
| [TreeInsert](../WDLang1/3018023.md) | Inserta una hoja en una control TreeView en una posición específica. |
| [TreeModify](../WDLang1/3018013.md) | Modifica un elemento (nodo u hoja) en un control TreeView. |





Estas funciones también están disponibles en la sintaxis del prefijo: 



|   |   |
| --- | --- |
| [&lt;Insertar](../WDLang1/1000023761.md) | Inserta una hoja en una control TreeView en una posición específica. |
| [&lt;Modificar](../WDLang1/1000023763.md) | Modifica un elemento (nodo u hoja) en un control TreeView. |
| [&lt;Vista de árbol&gt;.add](../WDLang1/1000023422.md) | Agrega un elemento "hoja" (y todos los nodos necesarios) en un control TreeView. |





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Lectura del identificador del elemento
<a name="lectura_del_identificador_del_elemento_ELTTEXTE000179"></a>
Para averiguar el identificador del elemento, todo lo que tiene que hacer es utilizar la siguiente función:



|   |   |
| --- | --- |
| [TreeIdentifier](../WDLang1/3018004.md) | Devuelve el identificador associated con un elemento (nodo u hoja) en una control TreeView. |





Esta función también está disponible en la sintaxis del prefijo: 



|   |   |
| --- | --- |
| [&lt;Identificador](../WDLang1/1000023766.md) | Devuelve el identificador associated con un elemento (nodo u hoja) en una control TreeView. |

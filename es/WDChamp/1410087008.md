## Resumen de la nota reposicionable control
<a name="resumen_nota_reposicionable_control_ELTTEXTE000139"></a>
La Nota Reposicionable control le permite incluir la famosa etiqueta amarilla en sus aplicaciones. 

Este control también puede ser usado para mostrar información específica, importante para el usuario final. 

Existen varios tipos de usos. Por ejemplo: 

- utilizando la Nota Reposicionable control para mostrar un dato que se encuentra en la base de datos (comentarios sobre el tipo de entrega para cada cliente por ejemplo). 

- usando la Nota Reposicionable control para recordar al usuario una acción a realizar. En este caso, el texto de la note es fijo, no es relacionado a la base de datos. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Note_Go%20-%20HC%20N%B0001.gif)
Observación: El usuario final puede manipular las notas reposicionables, puede cerrarlas, borrarlas. También puede añadir algunos a través de la AAF (automatic Application Feature) para crear notas. Para obtener más información, consulte [note reposicionable (AAF)](../Editeurs/1000021820.md).

## Creación de una nota reposicionable control
<a name="creacion_una_nota_reposicionable_control_ELTTEXTE000163"></a>
Para crear una Nota Reposicionable control: 

1. En la pestaña "Creación", en el grupo "Otros controles", seleccione "Nota reposi.".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. 

**Observaciones**: 

- Cuando se encuentra una Nota Reposicionable control en una ventana, no aparece ninguna fecha en la parte inferior de la note. Si el usuario modifica el texto de la note, la fecha de modificación se añadirá automáticamente en la parte inferior de la note.  

- Para crear un control Nota reposicionable mediante programación, utilice la función [ControlCreate](../WDLang1/1000020876.md) con la constante *typRepositionableNote*.

## Características de la nota reposicionable control
<a name="caracteristicas_nota_reposicionable_control_ELTTEXTE000187"></a>
La ventana description de la Nota Reposicionable control permite: 

- Escriba el contenido de la note (pestaña "General"),

- Minimizar automáticamente el note (pestaña "Detalles"). Si esta opción está marcada, note se minimizará automáticamente 5 segundos después de abrir la ventana. Esta opción es equivalente a la [Automáticamente minimizado](../Proprietes/1000021963.md) Property. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Note_Go%20-%20HC%20N%B0002.gif)





**Consejo**: Para desactivar la capacidad de realizar una entrada en un control de Nota Reposicionable, simplemente marque "Inactive" en la pestaña "UI".. El usuario puede entrar en el control pero no puede realizar ninguna entrada.

## Modo de funcionamiento de la nota reposicionable Default control
<a name="modo_funcionamiento_nota_reposicionable_default_control_ELTTEXTE000211"></a>
Si existe una note reposicionable en una ventana, cuando ésta se abre, la note reposicionable se muestra y se minimiza en el lateral de la ventana después de 5 segundos. 

Si el usuario restablece la note, la note se abre permanentemente. 

La note reposicionable se puede mover sobre la ventana y alrededor de la misma. 

El texto pasa a editarse al pulsar el botón note. 

La note reposicionable desaparece al pulsar la cruz. Pulse Ctrl + Z para volver a mostrar la note borrada.

Un menú emergente permite al usuario manipular la Nota Reposicionable control. Para más detalles, consulte la ayuda de [AAF de Nota control](../Editeurs/1000021820.md).




## Manejo de una control Gráfico mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000134"></a>
WINDEV, WINDEV Mobile y WEBDEV le permiten manipular a control Gráfico mediante la programación. Para hacerlo, usa la Variable de la control Gráfico en el código.

El Variable del control Gráfico corresponde al nombre del control Gráfico. Esta variable es un número entero.

Se pueden utilizar varias funciones de WLanguage para manipular a control Gráfico. Para más detalles, consulte la página web de [funciones para la gestión de gráficos](../WDLang3/3042003.md).  



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manejo de la serie de una control Gráfico
<a name="manejo_serie_una_control_grafico_ELTTEXTE000158"></a>
Para manipular la serie de un control Gráfico, utilice la siguiente sintaxis:


```txt
<Chart name>[Series index].<Property name>
```

<a name="NOTE3_1"></a>


## Propiedades específicas de los gráficos
<a name="propiedades_especificas_los_graficos_ELTTEXTE000182"></a>
Las siguientes propiedades se utilizan para gestionar los controles de los gráficos a través de la programación.


|   |   |
| --- | --- |
| [AnimationDuration](../Proprietes/1000021069.md) | Permite conocer y modificar la duración de la animación de una control Gráfico. Este Property también puede ser usado para borrar la animación de un control Gráfico. |
| [Conde](../WDLang1/3025009.md) | Le permite conocer el número de series que se encuentran en un gráfico. |
| [Valor](../Proprietes/2510130.md) | Permite conocer el nombre de un control Gráfico. |
| [Vacío](../Proprietes/2510126.md) | Le permite saber si una control Gráfico no contiene ninguna serie. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Gráfico, consulte [Propiedades associated con un control Gráfico](../WDChamp/1013008.md).

Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una serie que se encuentra en una control Gráfico, consulte [Propiedades associated con la serie de un control Gráfico](../WDChamp/1013011.md).

## Observaciones
<a name="observaciones_ELTTEXTE000206"></a>
Un control Gráfico puede ser manejado en un solo Thread. Una control Gráfico sólo puede ser manejada en la el hilo que la creó, lo que significa que en la Thread principal de la ventana. 

La control Gráfico sólo puede ser manejada en el código que se ejecuta directa o indirectamente desde los procesos de los controles de ventana y ventana..

**Consecuencias**: Dos hilos no pueden ser manipular el mismo control Gráfico.

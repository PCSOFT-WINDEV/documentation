## Presentación
<a name="presentacion_ELTTEXTE000206"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000011"></a>

La control Organigrama se utiliza para representar elementos de forma jerárquica.. 
Por ejemplo: 

- los colaboradores de una empresa según su posición. 

- las diferentes ramas de un grupo con sus interdependencias. 

- los diferentes pasos de una fabricación de Process, ...

## Crear un control Organigrama
<a name="crear_control_organigrama_ELTTEXTE000230"></a>
Para crear una control Organigrama: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", haga clic en "Organigrama".

2. Haga clic en la ubicación en la que desea crear el control. La creación de control Organigrama asistente comienza. Seleccione las diferentes características de control y valide. El control se crea automáticamente.


Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## description de un organigrama

### Opciones de control
<a name="opciones_control_ELTPARAGRAPHE000053"></a>

El control Organigrama puede ser configurado en el editor a través de la ventana description (pestaña "General").

Los parámetros del organigrama son los siguientes: 

- **Orientación**: El organigrama se puede mostrar horizontalmente (de izquierda a derecha o de derecha a izquierda) o verticalmente (de arriba hacia abajo o de abajo hacia arriba).. 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=orga_orientation.gif)


- **Tipo de flecha**: La flecha puede apuntar a los elementos hijo, al elemento padre o en ambas direcciones. 

- **Ventana interna para los elementos**: Ventana interna utilizada para personalizar la apariencia de los elementos que se encuentran en el organigrama (ver abajo). 

- **Anchura y altura mínima de los elementos**: tamaño mínimo de los elementos que se encuentran en la control Organigrama. 

- **Mover sobre varias filas de**. Si el número de elementos es mayor que el número especificado, los elementos se mostrarán en varias líneas (o columnas dependiendo de la orientación de control).

### Personalización del área del elemento: Color, traducción, etc.
<a name="personalizacion_del_area_del_elemento_color_traduccion_etc_ELTPARAGRAPHE000082"></a>

En la control Organigrama, cada elemento se visualiza en un área específica. Usted tiene la capacidad de traducir y personalizar esta área: título Color, fondo Color del elemento, ...

Para personalizar la visualización de los elementos:

1. Abre la ventana de control Organigrama description. 

2. En la pestaña "General", seleccione la ventana interna utilizada para los elementos: 

	- "Ninguno": se utilizará la pantalla de Default. 

	- "Ventana predefinida": 
			Si se selecciona esta opción, el componente interno WDAAF se añade inmediatamente al proyecto actual. La ventana interna "IW_WinDevViewOrgElement" se associated automáticamente a la opción "Ventana interna para los elementos".. 
			Observación: Si el componente WDAAF ya existe, sólo se añade la ventana al componente.  

	- una ventana interna existente de su proyecto. 




3. Si es necesario, valide la ventana de control description. 




**Observaciones**: 

- La ventana interna personalizada está disponible en el panel "Project explorer", en la carpeta "Internal components". Se puede modificar en el editor de ventanas.

- Si está utilizando una ventana interna de su proyecto, esta ventana interna debe gestionar la visualización de las características del elemento en los diferentes controles de esta ventana interna.. Para más detalles, ver el código de declaración de las variables globales que se encuentran en la ventana interna de presets. 




Para obtener más información, consulte [Personalizar la visualización de una control Organigrama](../WDChamp/1000019822.md).

## Imprimir un organigrama
<a name="imprimir_organigrama_ELTTEXTE000294"></a>
Para imprimir una control Organigrama: 

- **el usuario final** puede seleccionar una opción en el menú contextual para guardar el organigrama como imagen (Función de la aplicación automatic). Entonces, tendrán la capacidad de imprimir este Image. 

- **a través de la programación**, [OrgSaveImage](../WDLang1/1000019972.md) se utiliza para crear el Image del organigrama en un archivo Image. A continuación, la aplicación puede imprimir la imagen a través de una reporte, enviarla por Email, etc.

## Características de la aplicación de la automatic (AAF) para una control Organigrama
<a name="caracteristicas_aplicacion_automatic_aaf_para_una_control_organigrama_ELTTEXTE000318"></a>
La control Organigrama incluye varios AAF (Características de la aplicación automatic). 

El usuario puede: 

- zoom mediante el acceso directo "Ctrl + rueda del ratón. 

- desplazarse por el organigrama con el ratón. 

- usar los "deslizadores de rango" disponibles en los lados de la control para hacer zoom y moverse. 

- guardar el organigrama en una imagen (para imprimirlo, por ejemplo) a través del menú contextual de control.

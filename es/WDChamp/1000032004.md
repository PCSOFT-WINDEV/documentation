
## Manejo de una control Área de texto enriquecido mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000127"></a>
Para manipular un control Área de texto enriquecido de forma programada, utilice el Variable del control Área de texto enriquecido directamente en el código.

El Variable del control Área de texto enriquecido corresponde al nombre del control. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Iniciando un control Área de texto enriquecido
<a name="iniciando_control_area_texto_enriquecido_ELTTEXTE000151"></a>
Para inicializar un control Área de texto enriquecido, puedes: 

- inicializar el control con el código HTML. Esto permite formatear el contenido de control mediante la programación.
	Por ejemplo: 
	
	```wl
	// Modify the Rich Text Area
	RTA_MODIFIABLE = "'''This section was " + ...
	 	"modified'''"
	```


- utilice [Valor](../Proprietes/2510130.md) o [CaptionHTML](../Proprietes/1000020922.md). 



<a name="NOTE2_3"></a>


### Modificar uno o más elementos en una control Área de texto enriquecido
<a name="modificar_uno_mas_elementos_una_control_area_texto_enriquecido_ELTPARAGRAPHE000037"></a>

Un control Área de texto enriquecido puede contener enlaces o imágenes. Para modificar la información relativa a un enlace o a una Image que se encuentra en la control, basta con manipular el elemento solicitado mediante la programación. 

Por ejemplo, un control Área de texto enriquecido contiene un enlace y un Image. La leyenda y la URL del enlace cambian, dependiendo de si el usuario de la web está conectado o no.. De manera similar, el Image que se muestra es diferente. En este caso, todo lo que tiene que hacer es manipular directamente los controles de Enlace y Image. 


```wl
// Modify the link and the image found in the RTA_Home control
IF bConnected = True THEN
	LINK_Connection.Caption = "Disconnect"
	LINK_Connection.URL = "www.mysite.fr"
	IMG_Logo = "User_Logo.gif"
ELSE
	LINK_Connection.Caption = "Connect"
	LINK_Connection.URL = "www.mysite.fr\Login.htm"
	IMG_Logo = "Logo.gif"
END
```

## Recuperar el valor de una control Área de texto enriquecido
<a name="recuperar_valor_una_control_area_texto_enriquecido_ELTTEXTE000181"></a>
Para recuperar el contenido de un control Área de texto enriquecido:

- realice una asignación simple. Por ejemplo:
	
	```wl
	CurrentContent = RTA_Overview
	```


- manipular el control Área de texto enriquecido directamente. Por ejemplo:
	
	```wl
	Trace(RTA_Overview)
	```


- utilizar las propiedades [Valor](../Proprietes/2510130.md) o [CaptionHTML](../Proprietes/1000020922.md).




Observación: El texto recuperado corresponde al código HTML del control.

## Propiedades específicas para la gestión de una control Área de texto enriquecido
<a name="propiedades_especificas_para_gestion_una_control_area_texto_enriquecido_ELTTEXTE000205"></a>
Las siguientes propiedades se utilizan para gestionar las características de una control Área de texto enriquecido a través de la programación: 



|   |   |
| --- | --- |
| [CaptionHTML](../Proprietes/1000020922.md) | El **CaptionHTML** Property se utiliza para manipular el texto de un control como código HTML. |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una control Área de texto enriquecido, consulte [Propiedades associated con un control Área de texto enriquecido](../WDChamp/1000032003.md).

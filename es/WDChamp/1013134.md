## Modos de visualización
<a name="modos_visualizacion_ELTTEXTE000234"></a>

## Transparencia
<a name="transparencia_ELTTEXTE000258"></a>
Al mostrar una Image, tiene la posibilidad de definir sus opciones de transparencia. Estas opciones se conservan incluso si la imagen se modifica mediante programación (con la propiedad [Image](../Proprietes/2510034.md)).

Las opciones posibles son las siguientes:

- **Magenta transparente**: El color magenta de la imagen será transparente cuando la imagen se muestre en el editor o en ejecución.
	


|   |   |
| --- | --- |
| <br>![Transparencia: ninguno](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_MAGENTA1.gif)<br> | <br>![Magenta transparente](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_MAGENTA2.gif)<br> |


	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El magenta no se puede utilizar para gestionar la transparencia.

- **Pixel superior izquierdo**: todos los pixeles con el mismo color que el pixel superior izquierdo de la imagen serán transparentes.
	




**Atención**: El uso de un fondo Transparent Image ralentiza significativamente la visualización de la ventana (tiempo multiplicado por 3 o 4). Se recomienda:

- evitar el uso de imágenes con un Transparent Background.

- para los Image s en formato GIF o PSD, asegúrese de que no se describe ninguna transparencia color en formato Image (a través de un programa de tratamiento Image, por ejemplo).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) La transparencia de la Image no puede ser manejada.

## Programación
<a name="programacion_ELTTEXTE000314"></a>
Las imágenes se pueden gestionar mediante programación: 

- a través de las siguientes propiedades:  


|   |   |
| --- | --- |
| [BackgroundImage](../Proprietes/2510056.md) | La propiedad **..BackgroundImage** permite conocer y modificar la imagen de fondo de un elemento: ventana, página, control o reporte. |
| [BackgroundImagePrinted](../Proprietes/1000021350.md) | La **BackgroundImagePrinted** se utiliza para: <br><br>	- Determina si el fondo Image de un reporte (formulario u otro) debe imprimirse.<br><br>	- Imprimir (o no) el fondo Image de una reporte. <br><br><br> |
| [BackgroundImageState](../Proprietes/2510057.md) | El **BackgroundImageState** Property permite get y cambiar el número de dibujos de la imagen associated con el Background de un botón. |
| [Image](../Proprietes/2510034.md) | El **Image** Property obtiene y establece la imagen de fondo de un control o bloque. |
| [ImageMode](../Proprietes/2510058.md) | El **Image Modo** Property se utiliza para get y cambiar el modo de visualización de un Image en un control Imagen, en la columna de un tabla de tipo Image o en el Background de un control Gráfico en una ventana. |

- utilizando las siguientes funciones:  


|   |   |
| --- | --- |
| [WINBackgroundImage](../WDLang1/3038038.md) | Modifica el fondo Image de una ventana y especifica el modo de visualización de este Image. |

## Presentación del control Barra de desplazamiento
<a name="presentacion_del_control_barra_desplazamiento_ELTTEXTE000135"></a>


### Control de ventana
<a name="control_ventana_ELTPARAGRAPHE000011"></a>

El control Barra de desplazamiento permite desplazarse por diferentes objetos (texto, títulos, etc.). El desplazamiento del objeto se realiza mediante programación. Una barra de desplazamiento horizontal permite realizar un desplazamiento horizontal y una barra de desplazamiento vertical permite realizar el desplazamiento de forma vertical.

## Crear un control Barra de desplazamiento
<a name="crear_control_barra_desplazamiento_ELTTEXTE000159"></a>
Para crear un control Barra de desplazamiento: 

1. En la pestaña "Creación", en el grupo "Ajustar", despliegue "Barra de desplazamiento" y haga clic en "Barra de desplazamiento".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. Así, podrá elegir la orientación de la barra de desplazamiento.



## Imagen de un control Barra de desplazamiento
<a name="imagen_control_barra_desplazamiento_ELTTEXTE000183"></a>
Un control Barra de desplazamiento se puede dibujar a partir de elementos del sistema o con una imagen.

Para utilizar una imagen:

1. Abra la ventana de descripción del control.

2. Seleccione la pestaña "Estilo".

3. Seleccione el elemento "Fondo e imagen".

4. Seleccione bordes de tipo "Imagen (escala de 9 sectores)".

5. Haga clic en el botón que se encuentra junto a la opción "Bordes". 

6. La ventana emergente permite elegir las dos imágenes a utilizar (barra de desplazamiento horizontal y vertical) mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif): 

	- La opción "Examinar" permite utilizar un archivo al que se puede acceder desde el equipo. 

	- La opción "Catálogo" permite seleccionar una imagen del catálogo de imágenes. El catálogo de imágenes propone varias imágenes de barras de desplazamiento.

## Vocabulario asociado al control Barra de desplazamiento
<a name="vocabulario_asociado_control_barra_desplazamiento_ELTTEXTE000207"></a>
Para manipular fácilmente una barra de desplazamiento, se deben conocer los siguientes conceptos:

| Nombre | Definición |
| --- | --- |
| Valor de desplazamiento | Grado en que se desplaza el cuadro de desplazamiento cuando el usuario hace clic en el área entre el cuadro de desplazamiento y las flechas de desplazamiento. |
| Repetición | Cuando el botón del ratón permanece presionado en una de las flechas de desplazamiento, el clic en las flechas se repite de forma automática. Puede configurar el intervalo de tiempo entre estas repeticiones. |
| Valor mínimo | Valor de la barra de desplazamiento cuando el cuadro de desplazamiento se encuentra al inicio de la barra (1 predeterminado). Este valor corresponde al límite mínimo. |
| Valor máximo | Valor de la barra de desplazamiento cuando el cuadro de desplazamiento se encuentra al final de la barra. Este valor corresponde al límite máximo. |

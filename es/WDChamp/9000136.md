
## Animación de ventanas móviles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
En las aplicaciones móviles, las diferentes pantallas de la aplicación se muestran a través de una pequeña animación: pantalla desde la derecha y desde arriba, ....

WINDEV Mobile le permite configurar este modo de visualización de sus ventanas en sus aplicaciones móviles.  

Observación: Para que la animación se ejecute, la ventana debe tener un Action Bar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000101"></a>
Para configurar la animación de una ventana:

1. Abrir la ventana para animar en el editor de ventanas.  

2. Abra la descripción de la ventana (opción "Descripción" del menú contextual).

3. En la pestaña "UI", seleccione la animación deseada: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Animation_fenetre_mobile%20-%20HC%20N%B0001.gif)


4. Valide.




**Observación**: En el modo automatic, se utiliza la animación del sistema **excepto** si la ventana se abre desde un control en un control Looper. En este caso, la ventana se abre de izquierda a derecha.

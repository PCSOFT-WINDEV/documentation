## Presentación del control Editor de imágenes
<a name="presentacion_del_control_editor_imagenes_ELTTEXTE000228"></a>
El control "Editor de imágenes" es un control completo que permite a los usuarios finales crear y editar imágenes en aplicaciones WINDEV y sitios WEBDEV. Las ventajas se pueden ver inmediatamente: ahorro de tiempo, interfaz homogénea, ningún otro software a instalar, controlar o administrar. 

**Esta página de ayuda explica cómo utilizar el control Editor de imágenes en una aplicación WINDEV. Para obtener más información sobre cómo utilizar este control en un sitio WEBDEV, consulte [Control Editor de imágenes (WEBDEV)](../WDChamp/1410087158.md).**

Este control permite: 

- Abrir, crear, modificar y guardar archivos de imagen,

- Anotar imágenes,

- Combinar imágenes,

- Girar y voltear imágenes,

- Introducir texto en una imagen,

- Imprimir imágenes, etc.




La cinta de opciones de este control permite realizar las acciones más comunes. Para obtener más información, consulte [Control Editor de imágenes: Opciones de la cinta y de las pestañas](../WDChamp/1410087035.md).

También se pueden utilizar varios métodos abreviados de teclado: 

| Atajo | Efecto | Atajo | Efecto |
| --- | --- | --- | --- |
| Ctrl + A | Selecciona la capa actual. | Ctrl + C | Copia la selección en el portapapeles. |
| Ctrl + N | Crea una nueva imagen. | Ctrl + 0 | Abre una imagen existente. |
| Ctrl + P | Imprime la imagen actual. | Ctrl + S | Guarda la imagen. |
| Ctrl + T | Selecciona la capa actual. | Ctrl + V | Pega el contenido del portapapeles. |
| Ctrl + X | Corta la selección y la copia en el portapapeles. | Ctrl + Y | Rehace la última acción. |
| Ctrl + Z | Deshace la última acción. | Ctrl + rueda del ratón | Cambia el zoom. |

Observaciones:

- El editor de imágenes admite GIF animados.  

- El editor de imágenes reconoce automáticamente los dibujos a mano alzada. Las líneas, círculos o rectángulos dibujados a mano alzada se reconocerán automáticamente y se redibujarán.

## Crear un control Editor de imágenes
<a name="crear_control_editor_imagenes_ELTTEXTE000252"></a>
**Para crear un control Editor de imágenes**:

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Ofimática" y haga clic en "Editor de imágenes". 

2. Haga clic en la ubicación en la que desea crear el control.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Características del control Editor de imágenes
<a name="caracteristicas_del_control_editor_imagenes_ELTTEXTE000280"></a>


### Ventana de descripción
<a name="ventana_descripcion_ELTPARAGRAPHE000129"></a>

La pestaña "General" de la ventana de descripción del control Editor de imágenes incluye las siguientes opciones: 

- **Con cinta de opciones**: Permite mostrar una cinta en la parte superior del control. Esta cinta contiene diferentes opciones para manipular la imagen que se muestra en el control. Puede: 

	- utilizar la cinta de opciones predeterminada. En este caso, la opción "VI fuente de la cinta" corresponde a "Ninguna". 

	- personalizar la cinta de opciones predeterminada.
			Simplemente seleccione la ventana predefinida en la opción "VI fuente de la cinta". En este caso, la ventana "IW_WinDevAAF_RibbonPIC" se integra automáticamente en el proyecto. Esta ventana interna (disponible en inglés y francés) puede modificarse según sus necesidades. Esta ventana contiene el código necesario para gestionar el control Editor de imágenes. Le recomendamos comprobar los errores de UI y asegurarse de que la ventana funciona correctamente después de realizar cualquier cambio. 

	- utilizar una ventana interna para crear una cinta de opciones específica.
			Puede utilizar cualquier ventana interna del proyecto para crear la cinta.




- Al utilizar la cinta de opciones, la opción "Aplicar plantilla del proyecto a la cinta de opciones" permite utilizar automáticamente la plantilla del proyecto en la cinta seleccionada. 

- **Permitir cambiar de documento (Abrir, Nuevo, etc.)**: Si esta opción está seleccionada, el usuario podrá cargar cualquier documento en el control Editor de imágenes. De lo contrario, las opciones "Abrir" y "Nuevo" no estarán disponibles. Solo se podrá editar la imagen del control.  





### Formatos admitidos por el control Editor de imágenes
<a name="formatos_admitidos_por_control_editor_imagenes_ELTPARAGRAPHE000168"></a>

El control Editor de imágenes admite los siguientes formatos (en lectura/escritura): 

- JPG, JPEG, JPE

- BMP

- GIF

- Imágenes GIF animadas

- PNG

- ICO

- TIF

- WDPIC

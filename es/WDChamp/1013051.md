## Descripción de una tabla
<a name="descripcion_una_tabla_ELTTEXTE000113"></a>
Para visualizar el tabla description, seleccione en el menú contextual la "Tabla description" de un elemento tabla (celda perteneciente a la cabecera, a los datos o al total de tabla). Aparece la siguiente ventana:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtatDescriptionTableau.gif&type=thumb)
 Esta ventana se utiliza para ver:

- en la sección superior, el nombre de la tabla así como las columnas que se encuentran en la tabla.

- en la sección inferior, las características de la tabla si se selecciona el nombre tabla, o las características de la columna seleccionada.

## Características de una tabla
<a name="caracteristicas_una_tabla_ELTTEXTE000137"></a>
Se visualizan tres fichas en la sección inferior si se selecciona el nombre de la tabla. Estas pestañas se utilizan para configurar toda la tabla. 

**"Ficha"General: Configuración de la tabla**
La pestaña "General" sirve para definir:

- Si las filas de la tabla son separables y/o tienen ajuste automático.

	- Si las filas se pueden romper, pueden "solaparse" en 2 páginas (si el texto mostrado en la columna es muy largo, por ejemplo).

	- Si las filas se redimensionan automáticamente, las celdas pueden expandirse para mostrar todo su contenido. Si una célula se expande, todas las células que se encuentran en la fila también se expandirán.




- La posición de la cabecera tabla (bloque para visualizar la cabecera) así como sus características.

- El tamaño y la posición de la tabla en la página.




** "Border" ficha: Configuración de las líneas de cuadrícula de tabla**
La pestaña "Border" se utiliza para definir las cuadrículas de tabla. Para una configuración personalizada, puede configurar las cuadrículas de cada celda. Para ello, muestra la description de cada célula de la tabla.

**"Pestaña"Estilo: Configuración de la tabla**
La pestaña "Estilo" sirve para ver y modificar el estilo de la tabla. Sólo se visualizan los valores comunes a todas las celdas de tabla.. Para una configuración personalizada, tiene la posibilidad de configurar el estilo de cada celda. Para ello, muestra la description de cada célula de la tabla.

## Características de las columnas tabla
<a name="caracteristicas_las_columnas_tabla_ELTTEXTE000161"></a>
Para mostrar las características de una columna, seleccione esta columna en la sección superior de la ventana de description. 

Para cada columna, se visualizan los siguientes elementos en la sección superior:  

- El ***leyenda de la columna***. Esta leyenda se mostrará en una Control estática, en la cabecera fila de la tabla. Esta leyenda puede dejarse vacía.

- La ***Item associated con la columna ***. El control correspondiente se mostrará en la fila de datos de la tabla. Este control puede asociarse a ninguna Item para ser llenado mediante la programación, por ejemplo. El tipo se actualiza automáticamente según la Item seleccionada.

- El ***tipo de datos*** en la columna.

- El *** cálculo associated con la columna***. Todo lo que tiene que hacer es seleccionar el tipo de cálculo realizado en las posiciones visualizadas en la columna. El correspondiente control calculado se mostrará en el fila total del tabla.




Se visualizan dos fichas en la sección inferior si se selecciona el nombre de la columna. Estas pestañas se utilizan para configurar las características de la columna: 

- **"Ficha"Columna: Configuración de las columnas**
	La pestaña "Columna" sirve para configurar las características de visualización de la columna.: tipo, máscara, ....

- ** "Link" ficha: Configurando la Link de la columna**
	La pestaña "Link" se utiliza para configurar el elemento de la fuente de datos (Variable o Item) relacionado a la columna tabla.

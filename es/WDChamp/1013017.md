


## Modificación de las características de los controles reporte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000505"></a>
Las características de los controles se pueden ver y/o cambiar en:

- la ventana de description de este control.

- el modificador. 




La editor de reportes también le permite:

- modificar la leyenda de control.

- modificar el nombre de control.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ventana de controles description de reporte
<a name="ventana_controles_description_reporte_ELTTEXTE000529"></a>
Para abrir la ventana de description de un control, utilice uno de los siguientes métodos:

- haga doble clic en la control (recomendado).

- haga clic con el botón derecho del ratón para abrir el menú contextual del control y seleccione "description".

- seleccione la control, vaya a la pestaña "Modificación" y haga clic en el lanzador de cuadros de diálogo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif) del grupo "Edición rápida" (Alt + Enter).

- en la pestaña "Vista", haga clic en el lanzador de cuadros de diálogo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif) del grupo "Opciones" (F12). A continuación, seleccione el control y haga clic en el botón "description".




**Observación**: La ventana de descripción puede abrirse para una selección de controles. Solo se mostrarán las características comunes a los controles seleccionados.
<a name="NOTE2_2"></a>

| lengüeta | Características |
| --- | --- |

Por ejemplo:<br><br>- el título de control (este título puede ser multilingüe).<br><br>- el tipo de información que se muestra en la control (textual, numérica, ...).<br><br>- la máscara de visualización usada para el control (la máscara puede ser multilingüe). <br><br>- el Image que se muestra en un control Imagen, el tipo de código de barras utilizado en un control Código de barras, el tipo de gráfico utilizado en un control Gráfico, etc.. Para obtener más información, consulte la ayuda en línea del tipo de control deseado.<br><br><br> |
| **UI** | Permite definir:<br><br>- los parámetros de impresión de control (Visible, rompible, ...). <br><br>- el tamaño del control.<br><br>- la posición de control en el reporte y en el bloque al que pertenece el control.<br><br>- el anclaje del control.<br><br>- el grupo de control de associated. <br><br>- el avión associated. <br><br><br> |
| **Enlace** | Se utiliza para definir el elemento o la Variable que se encuentra en la fuente de datos a la que el control es relacionado. ## Principales parámetros de impresión de los controles reporte
<a name="principales_parametros_impresion_los_controles_reporte_ELTTEXTE000559"></a>
Los parámetros de impresión de un control son los siguientes (definidos en la pestaña "UI" de la ventana description):

| paramétrico | description | Disponible para los siguientes controles |
| --- | --- | --- |
| ***Visible*** | La control se puede mostrar u ocultar. Esta característica puede modificarse de forma programada con el programa [Visible](../Proprietes/2510138.md) Property. | [Estática](../WDChamp/1013179.md), [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md), [Image](../WDChamp/1013218.md), [Código de barras](../WDChamp/1013010.md), [Control RTF](../WDChamp/1013056.md), [Control Predefinido](../WDChamp/1013058.md), [Gráfico](../WDChamp/1013006.md), [Forma](../WDChamp/1013102.md), [Reporte interna control](../WDChamp/1013044.md) |
| ***Cambio de tamaño automático*** | El tamaño del control definido en el editor de reportes puede ser ampliado según el contenido del control. Las control s que sigan a la control de "redimensionamiento automático" se desplazarán automáticamente. <br>**Observaciones**:<br><br>- Si un control no es un control de tamaño automático, su contenido puede ser truncado si el tamaño del control es demasiado pequeño.<br>	Para obtener la altura de un control de redimensionamiento automático cuando se amplía, utilice la función [FutureHeight](../Proprietes/2511061.md) Property.<br><br>- Si un control en un bloque se redimensiona automáticamente, el bloque associated se redimensionará automáticamente. Un bloque que contiene controles de redimensionamiento automático es de redimensionamiento automático. Un bloque que no contiene un auto-dimensionamiento control no es auto-dimensionado.<br><br>- Los controles de un pie de página Page no se pueden redimensionar automáticamente.<br><br><br>**Limite**: En una etiqueta reporte, los controles no pueden ser de tamaño automático. | [Estática](../WDChamp/1013179.md), [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md), [Forma](../WDChamp/1013102.md), [Código de barras](../WDChamp/1013010.md), [Reporte interna control](../WDChamp/1013044.md) |
| ***multilínea*** | El contenido de control puede ser mostrado en varias líneas. Esta característica puede modificarse mediante programación con el programa [Multilínea](../Proprietes/2510105.md) Property.<br>**Observaciones**:<br><br>- La altura de control debe ser suficiente para mostrar varias líneas.<br><br>- Para modificar la altura de control mediante programación, utilice [Altura](../Proprietes/2510050.md).<br><br>- Si se aplica un ángulo de orientación a un control multilínea, se ignoran los retornos de carro (CR) automáticos. Solo se tienen en cuenta los saltos de línea manuales.<br><br><br> | [Estática](../WDChamp/1013179.md), [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md), [Control Predefinido](../WDChamp/1013058.md) |
| ***Quebradizo*** | El contenido de un control rompible puede imprimirse en varios Page s si no hay espacio suficiente para imprimir todo el control en un Page.<br>**Observaciones**:<br><br>- Para que un control sea un control rompible, el bloque que contiene el control también debe ser rompible.<br><br>- Si no hay espacio suficiente en el Page actual para imprimir un control no rompible , el control completo se imprimirá en el siguiente Page.<br><br><br> | [Estática](../WDChamp/1013179.md), [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md), [Image](../WDChamp/1013218.md), [Código de barras](../WDChamp/1013010.md), [Control RTF](../WDChamp/1013056.md), [Control Predefinido](../WDChamp/1013058.md), [Gráfico](../WDChamp/1013006.md), [Forma](../WDChamp/1013102.md) |
| ***En blanco si es cero*** | El valor de control se muestra sólo si el valor de control difiere de 0.<br>Si el valor de control es igual a 0, control muestra una cadena vacía (""). | [Estática](../WDChamp/1013179.md), [Campo](../WDChamp/1013059.md) |
| ***Elipsis (truncar texto largo con '...')*** | La elipsis se utiliza para indicar que un título es demasiado largo para que se muestre en su totalidad.. Se añaden los siguientes caracteres al pie de foto: "...". Estos caracteres se pueden añadir al final de la leyenda o en el centro del texto (recomendado cuando se muestran las rutas de archivo). | [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md) |
| ***Se muestra cuando cambia el valor*** | El valor de control sólo se visualiza si el valor de la posición relacionado cambia al pasar al siguiente registro en la fuente de datos de reporte. | [Campo](../WDChamp/1013059.md), [Control Calculado](../WDChamp/1013212.md) |

| ***Utilizar información compartida en ejecución*** | El control mostrará la "información compartida" del elemento relacionado al control (y no el índice correspondiente a esta "información compartida").<br>Por ejemplo, la "información RAD" del elemento "Título" corresponde al siguiente cuadro combinado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=UtiliserInfosRad.gif)<br> Sólo los índices de "Información compartida" se almacenan en la base de datos. | [Campo](../WDChamp/1013059.md) |
| ***Líneas de cuadrícula horizontales/verticales*** | Las líneas de cuadrícula horizontales y/o verticales se pueden mostrar (o no) en un gráfico. | [Gráfico](../WDChamp/1013006.md) |

## Uso del modificador
<a name="uso_del_modificador_ELTTEXTE000583"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000400"></a>

El modificador se utiliza para ver y/o modificar desde el editor de reportes las características principales:

- los controles seleccionados.

- de la reporte o bloque seleccionado.




Para obtener más información, consulte [Modificador](../Editeurs/2027006.md).
<a name="NOTE4_2"></a>


### Ver y/o cambiar las características de uno o más controles con el modificador
<a name="ver_cambiar_las_caracteristicas_uno_mas_controles_con_modificador_ELTPARAGRAPHE000415"></a>

Para ver y/o modificar las características de uno o más controles con el modificador:

1. Seleccione los controles que desea modificar:

	- en la reporte directamente (ver [Seleccione uno o más controles](../Editeurs/2026007.md)).

	- mediante la lista desplegable del modificador. En este caso, solo se puede seleccionar un control a la vez. Los controladores de tamaño del control seleccionado se muestran automáticamente.
			Si se seleccionan varios controles, solo se muestra en el modificador la información común a los controles seleccionados.




2. Haga los cambios deseados. Estos cambios se aplican a todos los controles. Es posible:

	- seleccionar el nuevo valor en una lista desplegable.

	- seleccionar el nuevo valor en una pantalla (modificación de color, por ejemplo).

	- cambiar el valor directamente.

## Modificando la leyenda de control
<a name="modificando_leyenda_control_ELTTEXTE000613"></a>
Para modificar la leyenda de un control en el reporte actualmente creado:

1. Seleccione el control que desea modificar.

2. Haga clic de nuevo en el control o pulse la tecla Entrar. El cursor se convierte en un cursor de texto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LibelleModifDirectetat.gif)


3. Introduzca el nuevo texto.

4. Confirme con la tecla Entrar.

## Cambiar el nombre del control
<a name="cambiar_nombre_del_control_ELTTEXTE000637"></a>
El nombre de un control puede ser modificado desde la barra de estado del editor de reportes.

Para cambiar el nombre del control:

1. Seleccione el control que desea modificar.

2. Escriba el nuevo nombre del control en la barra de estado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NomChampModifEtat.gif)

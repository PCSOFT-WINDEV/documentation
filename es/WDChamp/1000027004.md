
## Manejo de una WEBDEV Control deslizante mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000169"></a>
WEBDEV le permite manipular a Control deslizante a través de la programación. Para hacerlo, usa la Variable de la Control deslizante en el código.

La Variable de la Control deslizante:

- corresponde al nombre del control deslizante.

- se inicializa con la posición del cursor del control deslizante.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperación de la posición del cursor
<a name="recuperacion_posicion_del_cursor_ELTTEXTE000193"></a>
**Para recuperar la posición de una corredera lineal**utilice la siguiente sintaxis:


```txt
<Position> = <Slider Name>
```


&lt;Posición&gt; es un entero Variable que corresponde a la posición del cursor. 

Observaciones: 

- Para un control deslizante de rango, &lt;Posición&gt; corresponde a la posición del cursor principal (cursor del valor inferior). 

- También puede utilizar la [Valor](../Proprietes/2510130.md) Property.




**Para recuperar las posiciones de un deslizador de rango**, debe utilizar [LowerValue](../Proprietes/1000020433.md) y [UpperValue](../Proprietes/1000020434.md): 


```txt
Lower Position> = <Slider Name>.LowerValue
Upper Position> = <Slider Name>.UpperValue
```


&lt;Posición inferior&gt; y &lt;Posición superior&gt; son variables de tipo integer.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificar la posición del cursor
<a name="modificar_posicion_del_cursor_ELTTEXTE000217"></a>
**Para modificar la posición de una corredera lineal**utilice la siguiente sintaxis:


```txt
<Slider Name> = <Value>
```


&lt;Valor&gt; es un valor incluido entre el valor mínimo y el valor máximo del control deslizante.. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

**Para modificar la posición de un control deslizante de rango**utilice la siguiente sintaxis:


```txt
<Slider Name>..LowerValue = <Position of Main Cursor>
<Slider Name>..UpperValue = <Position of Secondary Cursor>
```


&lt;La posición del cursor principal y la posición del cursor secundario son valores incluidos entre el valor mínimo y el valor máximo de la barra deslizante.. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

## Propiedades específicas de la gestión de un deslizador WEBDEV
<a name="propiedades_especificas_gestion_deslizador_webdev_ELTTEXTE000241"></a>
Las siguientes propiedades se utilizan para gestionar las características de un deslizador por programación.


|   |   |
| --- | --- |
| [LowerValue](../Proprietes/1000020433.md) | Permite conocer y modificar el valor inferior de un deslizador de rango. |
| [MaxValue](../Proprietes/2510009.md) | Permite conocer y modificar el límite superior de un control deslizante. |
| [MinValue](../Proprietes/2510008.md) | Se utiliza para averiguar y modificar el límite inferior de un control deslizante. |
| [UpperValue](../Proprietes/1000020434.md) | Permite conocer y modificar el valor superior de un deslizador de rango. |
| [Vertical](../Proprietes/2510133.md) | Permite saber si el deslizador estándar es horizontal o vertical. |

Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una Control deslizante, consulte [Propiedades associated con controles Control deslizante](../WDChamp/1013233.md).

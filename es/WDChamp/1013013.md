
## Manejo de una control Gráfico en un sitio WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
Cuando se ejecuta un control Gráfico en una página de un sitio WEBDEV, una barra de herramientas permite al usuario Web modificar el diseño del gráfico.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_BarreWeb_20.gif)

<a name="NOTE2_1"></a>


## Características de la barra de herramientas
<a name="caracteristicas_barra_herramientas_ELTTEXTE000165"></a>
La barra de herramientas de un control Gráfico le permite elegir el tipo de gráfico a mostrar. Es posible:

- mostrar un gráfico circular: 

	- Circular, 

	- Semicírculo, 

	- Anillo, 

	- Embudo.




- visualizar un gráfico de columnas: 

	- Columna agrupada, columna apilada, columna agrupada horizontal, columna apilada horizontal,

	- Cascada.




- mostrar el gráfico en el siguiente formato: 

	- línea, 

	- dispersión,

	- área, 

	- radar

	- burbujas.




- mostrar un gráfico de existencias: 

	- velas japonesas, 

	- BarCharts, 

	- MinMax. 







Esta barra de herramientas también le permite hacerlo:

- mostrar (o no) las líneas de grilla horizontales o verticales.

- mostrar (o no) la leyenda y configurar su posición. 

- manejar los efectos (alisado, Gradient, levantado, dentado y transparencia).

- cuando pase el ratón por encima de los valores, muestre un tooltip, un retículo o un retículo múltiple.

- mostrar (o no) las animaciones. 

- habilitar (o no) la interactividad del gráfico.

- guardar o imprimir el gráfico.

## Implementación de la barra de herramientas
<a name="implementacion_barra_herramientas_ELTTEXTE000189"></a>
Cuando se crea una control Gráfico, la barra de herramientas es mostrada en el modo automatic por Default. 

Para configurar la visualización de la barra de herramientas: 

1. Mostrar la ventana de description de control Gráfico. 

2. En la pestaña "Detalles", en el área "Apariencia y barra de herramientas", tiene la posibilidad de seleccionar: 

	- el modo de visualización de la barra de herramientas: nunca, automatic o siempre Visible. 

	- las opciones mostradas en la barra de herramientas (botón"..." a la derecha del modo de visualización). 




3. Valide la ventana de descripción del control. 




Observación: si has elegido una barra de herramientas que siempre es Visible, la barra de herramientas aparece en la página actualmente en edición.

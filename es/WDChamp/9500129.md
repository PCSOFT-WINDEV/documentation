


## Mostrar una contraseña (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
De manera predeterminada, los caracteres de contraseñas son reemplazados por puntos. 

Ahora es posible mostrar estos caracteres en un formato legible (es necesario tener en cuenta los riesgos de seguridad). 

A partir de la versión 18, los usuarios pueden hacer clic en un ícono para mostrar la contraseña en el control: la contraseña se mostrará mientras el botón izquierdo del ratón esté pulsado. Esto permite al usuario comprobar la contraseña.

## Un icono específico
<a name="icono_especifico_ELTTEXTE000135"></a>


### Implementación
<a name="implementacion_ELTPARAGRAPHE000040"></a>

A partir de la versión 18, este icono se incluye de forma predeterminada. 

Al igual que todas las opciones incluidas en las AAF, este ícono se puede configurar. Para obtener más información, consulte [Configurar los menús de las AAF](../Editeurs/2010040.md). 

Para evitar que el usuario muestre la contraseña en un control Campo de entrada, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafDisplayPassword*.

Atención: Cuando se asigna un valor a un control Campo de entrada de tipo "Contraseña" (con el operador "=" o la función [FileToScreen](../WDLang4/3044210.md), por ejemplo), el ícono de visualización de la contraseña no se muestra por razones de seguridad. 
Consejo: Para evitar este comportamiento, simplemente asigne el contenido con la función [SendKey](../WDLang1/3014014.md): 


```wl
// Assigns a password via SendKey
SendKey(GeneratePassword(18), Handle(EDT_Password))
```

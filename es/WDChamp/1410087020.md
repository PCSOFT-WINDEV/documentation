## Presentación del control Procesador de texto
<a name="presentacion_del_control_procesador_texto_ELTTEXTE000270"></a>
El control "Procesador de texto" es un control avanzado que permite al usuario final crear y editar documentos en formato docx directamente en una aplicación WINDEV. Las ventajas se pueden ver inmediatamente: ahorro de tiempo, interfaz homogénea, ningún otro software a instalar, controlar o administrar. 

Este control permite: 

- Abrir, crear, modificar y guardar archivos "docx",

- Convertir archivos docx a PDF o a HTML para enviar correos electrónicos,

- Manipular imágenes y tablas,

- Administrar las áreas de texto,

- Guardar documentos en bases de datos HFSQL,

- Use un corrector ortográfico,

- Realizar impresiones, ....




La cinta de opciones de este control permite realizar las acciones más comunes. Para obtener más información, consulte [Procesador de texto: usando la cinta](../WDChamp/1410087025.md). 

También se pueden utilizar varios métodos abreviados de teclado: 


| Atajo | Efecto | Atajo | Efecto |
| --- | --- | --- | --- |
| Ctrl + A | Selecciona todo el texto. | Ctrl + C | Copia la selección en el portapapeles. |
| Ctrl + F | Inicia la búsqueda. | Ctrl + G | Hace la selección audaz |
| Ctrl + H | Inicia la búsqueda/reemplazo | Ctrl + I | Italiza la selección |
| Ctrl + S | Salva la Document | Ctrl + U | Cambia la selección a subrayado |
| Ctrl + V | Pega el contenido del portapapeles. | Ctrl + X | Personalizar la selección y colocarla en el portapapeles. |
| Ctrl + Y | Rehace la última acción. | Ctrl + Z | Deshace la última acción. |
| Ctrl + rueda del ratón | Cambia el zoom. | Ctrl + 0 | Establece el zoom en 100% |
| Ctrl + inicio | Va al principio del documento. | Ctrl + Fin | Va al final del documento. |
| Re Pág | Muestra la página anterior. | Av Pág | Muestra la página siguiente. |

## Creación de una control Procesador de texto
<a name="creacion_una_control_procesador_texto_ELTTEXTE000294"></a>
**Para crear un control de tipo Procesador de texto**:

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Ofimática" y haga clic en "Procesador de texto". 

2. Haga clic en la ubicación en la que desea crear el control.




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción" en el menú contextual.

## Características de control Procesador de texto
<a name="caracteristicas_control_procesador_texto_ELTTEXTE000322"></a>
La pestaña "General" de la ventana de descripción del control Procesador de texto propone las siguientes opciones: 

- **Documento inicial**: Sirve para indicar la Document inicial que aparece en la control Procesador de texto.  

- **Con cinta de opciones**: Indica si la cinta debe mostrarse o no.. Si se muestra la cinta de opciones, se puede especificar con la opción "VI fuente de la cinta".
	Si **"VI fuente de la cinta"** corresponde a: 

	- "Ninguno": se utilizará la cinta propuesta por Default. 

	- "Ventana predefinida": la cinta propuesta por Default se puede personalizar.
			Sólo tiene que seleccionar la ventana de preselección de Default en "Ribbon source IW". En este caso, la ventana "IW_WinDevAAF_WPBar" se incluye automáticamente en su proyecto.. Esta ventana interna (disponible en inglés y francés) puede modificarse según sus necesidades. Esta ventana contiene todo el código necesario para la gestión de control Procesador de texto. Le recomendamos comprobar los errores de UI y asegurarse de que la ventana funciona correctamente después de realizar cualquier cambio. 

	- Ventana interna del proyecto: Permite utilizar una ventana interna para proponer una cinta específica..
			Cualquier ventana interna de su proyecto puede ser utilizada para crear esta barra.


 También tiene la capacidad de aplicar el proyecto tema-modelo. 

- **Permitir cambiar de documento (Abrir, Nuevo, etc.)**: Adapta la cinta para que el usuario pueda acceder a los documentos externos de manipular.. Esta opción se utiliza para visualizar los botones "Nuevo", "Abrir" y "Guardar" de la cinta de opciones.. Si se marca esta opción, la apertura y la copia de seguridad de Document deben ser realizadas por la aplicación. 
	Observación: Esta opción se ignora si la cinta está totalmente personalizada (con una ventana interna de aplicación, por ejemplo)..  

- **Limitar la edición para simplificar la exportación HTML**:  Adapta la cinta de opciones y la visualización de Document para que sólo se muestren las opciones tenidas en cuenta por la exportación de HTML. Si esta opción está marcada: 

	- las siguientes opciones no están disponibles: 

		- Gestión de cabeceras y pies de página,

		- Gestión del trazado de Page,

		- Gestión de las pausas de Page. 




- el Document se ve forzado a mostrarse en modo "Visualización para la web".

- **Activar [Revisar ortografía] de forma predeterminada**: Especifica si el corrector ortográfico debe estar habilitado. Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md). 

- **Detectar y definir el formato de los enlaces automáticamente**: Permite la detección en automatic de los enlaces (http) presentes en la Document que aparecen en el control. En este caso, los enlaces se detectan durante la entrada, después de pulsar Espacio, Tabulador o Enter. Si estos enlaces son válidos, se pueden abrir con Ctrl + Click.
	Observación: Esta opción está marcada por Default.  




**Atención**: Esta control no puede mostrar ni aceptar la introducción de caracteres de derecha a izquierda.

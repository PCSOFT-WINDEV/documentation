


## Descripción de un control Agenda (control de ventana)
			





## Describiendo un control Agenda
<a name="describiendo_control_agenda_ELTTEXTE000287"></a>
El control Agenda puede ser configurado en el editor a través de la ventana description (pestaña "Detalles").

Las opciones disponibles son las siguientes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Description%20Agenda_Fen%EAtre%20-%20HC%20N%B0001.gif&type=thumb)


- **Parámetros de la agenda**: 

	- **Modo de visualización y granularidades**: Por Default, se muestran las citas: 

		- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a lo largo de una semana de 5 días, con granularidad de movimiento y duración set a 30 minutos.


 Al hacer clic en las opciones de visualización, se abre una ventana que le permite definir las diferentes opciones de visualización: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Description%20Agenda_Fen%EAtre%20-%20HC%20N%B0002.gif&type=thumb)
 Es posible definir los siguientes elementos: 

- Modo de visualización de la control Agenda: Permite definir la duración del rango visualizado: Día, mes, semana, ... o número de días personalizado. 

- Granularidad de las citas: Corresponde a la precisión de la control Agenda. Las citas pueden comenzar en un múltiplo de XX minutos, horas, ... y pueden tener una duración múltiplo de XX minutos, horas, ... El botón de flecha se utiliza para introducir valores preestablecidos. El bloqueo se utiliza para enlazar (o no) los valores. 

- La opción "Recordar modo" permite guardar y volver a aplicar el modo de visualización elegido por el usuario. 

- **Máscara de visualización para la fecha**: 
	Permite definir el formato de la fecha que aparece en la parte superior de las columnas "día" de la control Agenda.

	- " Mostrar número de semana" se utiliza para especificar si el número de la semana debe ser mostrado o no. A continuación, es posible especificar su formato. En el formato, la cadena'%1' corresponde al número de la semana. Ejemplo de formato: "Wk. %1".

	- La opción "Formato ISO 8601" se utiliza para que las semanas se numeren según la norma ISO 8601

	- La opción "Mostrar mes si es necesario" permite especificar un nombre de mes abreviado en la primera fecha mostrada en la control (si la máscara de fecha no muestra el mes). 




- **Intervalo de tiempo**: 

	- La franja horaria de Visible es el intervalo de tiempo que se mostrará en control. 

	- La franja horaria de trabajo corresponde a la franja horaria para la que se puede definir una Appointment.  
			Es posible modificar estos valores haciendo clic en la flecha doble. 
			Para definir franjas horarias específicas, utilice [OrganizerWorkingHour](../WDLang1/1000022530.md).

	- La opción "Modificar intervalo al redimensionar" permite ampliar la franja horaria cuando se amplía el control. Por ejemplo, si la franja horaria va de las 9:00 a las 18:00, al ampliar el control, esta franja horaria de Visible puede pasar a ser de 6:00 a 23:00. En este caso, la altura de las filas permanece constante.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Características de la cita**: La opción "Superposición" permite permite definir la superposición de varias citas. Las citas se pueden mostrar una al lado de la otra o se pueden desplazar.

- **Otros parámetros**: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) "Permitir seleccionar períodos": Permite al usuario seleccionar un período con el ratón.

	- "Permitir introducir una cita directamente": Permite al usuario introducir directamente una Appointment en la control Agenda. 
			![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Por defecto, la Appointment entra en edición, y el usuario puede introducir el título de la Appointment. 
			
			Es posible mostrar una ventana para configurar las opciones de Appointment.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) "Permitir y mostrar citas de todo el día": Se utiliza para especificar que un Appointment puede corresponder a un día entero. En este caso, se muestra un área específica en la parte superior de la pantalla control Agenda. Esta zona se utiliza para introducir las citas diarias.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Agenda_Journee.gif&type=thumb)





- **Editar con el ratón (o tocar)**:

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) "Modificar duración": Puede permitir (o no) al usuario modificar la duración de Appointment. 

	- "Mover". Puede permitir (o no) al usuario mover un Appointment. 

	- "Permitir al usuario superponer elementos". Si se selecciona esta opción, el usuario podrá solapar las citas cuando las mueva.. 
			![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En este caso, se tendrán en cuenta las opciones definidas para el solapamiento de citas (offset o side by side).

### Menú contextual propuesto automáticamente
<a name="menu_contextual_propuesto_automaticamente_ELTPARAGRAPHE000157"></a>

El control Agenda puede mostrar un menú emergente específico que propone al usuario las características de automatic. Este menú es propuesto por las AAF (Funcionalidades Automáticas de la Aplicación). 

Este menú permite al usuario: 

- Modificar el modo de visualización de la control Agenda: visualización diaria, semanal, mensual o sólo del día actual. 

- Añadir, eliminar o modificar una Appointment al seleccionar un periodo de tiempo. 
	**Observaciones**: Si el control es relacionado a una fuente de datos, la eliminación realizada en el control se realiza al mismo tiempo en la fuente de datos del relacionado. 

- Imprimir el contenido de la control Agenda. 




Para obtener más información, consulte [automatic características del control Agenda](../WDChamp/1016327.md).

### Personalizando el área de Appointment
<a name="personalizando_area_appointment_ELTPARAGRAPHE000181"></a>

En el control Agenda, el Appointment se muestra en un área específica. Usted tiene la capacidad de traducir y personalizar esta área: el color del título, el color de fondo de la Appointment, ...

**To modificar el aspecto de los nombramientos propuestos por Default**, tiene la posibilidad de modificar las opciones de estilo definidas para este elemento.  

1. Visualizar la ventana description del control Agenda. 

2. En la pestaña "Estilo", seleccione el elemento "Citas" y modifique las características deseadas.
	Es posible: 

	- usa una sombra,

	- usar un separador en puntos entre horas,

	- definir el título y las opciones de Background: 

		- fondo sólido, 

		- Gradient Background,

		- Fondo sólido, título multiLine y fondo Line. Esta opción se utiliza para mostrar el título en varias líneas (especialmente si la Appointment no es lo suficientemente ancha como para mostrar completamente el título).




- definir las fuentes utilizadas, ....




**Para personalizar la visualización de las citas**: 

1. Visualizar la ventana description del control Agenda. 

2. En la pestaña "General", seleccione la ventana interna utilizada para las citas: 

	- "Ninguno": se utilizará la pantalla de Default. 

	- "Ventana predefinida": Si se selecciona esta opción, el componente interno WDAAF se añade inmediatamente al proyecto actual. 

		- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La ventana interna "IW_WinDevViewAPT" es automáticamente associated con la opción "Ventana interna para las citas".. 


 Observaciones: 

- Si el componente WDAAF ya existe, sólo se añade la ventana al componente.  

- Esta ventana también se utiliza para personalizar las citas en los controles del Programador. 

- Cualquier ventana interna de su proyecto. 

3. Si es necesario, valide la ventana de control description. 




**Observaciones**: 

- La ventana interna personalizada está disponible en el panel "Project explorer", en la carpeta "Internal components". Se puede modificar en el editor de ventanas. 
	**Atención**: esta ventana interna sólo se puede utilizar para personalizar el dibujo relativo a la visualización de las citas. Sólo se ejecutan las "Declaraciones Globales" Event. No se ejecutan otros eventos. Por lo tanto, no puede añadir botones con un código de clic específico. 

- Si se utiliza la ventana preestablecida sugerida, se aplican las opciones de estilo de la Appointment (seleccionadas en la etiqueta "Style").

- La ventana interna de Default sólo funciona en el modo de visualización Día/Semana. Esta ventana no funciona en el modo de visualización Mes.


## Eventos associated con los controles de un reporte
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000112"></a>
Los eventos associated con los controles de un reporte son los siguientes:

| Evento | Condición de ejecución |
| --- | --- |
| **Inicialización** | Ejecutar al inicializar la reporte. Al iniciar una reporte (incluso durante una prueba), los eventos se ejecutan en el siguiente orden:<br><br>- "Apertura" Event de la reporte.<br><br>- "Iniciando" Event de los bloques de reporte.<br><br>- "Iniciando" Event de los controles.<br><br><br> |
| **Antes de imprimir** | Corre antes de imprimir un control. |
| entrada de ** en** | Se ejecuta cuando el cursor de entrada entra en la control. Este Event sólo está disponible para los controles de edición utilizados en el visor de reporte. |
| **-Salida de** | Ejecutar al pasar de un control a otro (tecla TAB, seleccionar otro control con el ratón, etc.). Este Event sólo está disponible para los controles de edición utilizados en el visor de reporte. |
| **Modificación de** | Ejecutar cada vez que se modifique el contenido de la control (tecleando o borrando un carácter por ejemplo). Este Event sólo está disponible para los controles de edición utilizados en el visor de reporte. |
| **Haga clic en** | Corre al hacer clic en la control. El clic en el control está disponible en el visor de reporte solamente. Una estrella permite al usuario identificar fácilmente los controles en los que se puede hacer clic. |


Puede introducir el código WLanguage en estos eventos. Para más detalles sobre la secuencia de estos acontecimientos, véase [Secuencia de eventos associated con informes](../WDChamp/1011004.md).

## Abrir y/o modificar los eventos de un control
<a name="abrir_modificar_los_eventos_control_ELTTEXTE000136"></a>
Para abrir y/o modificar los eventos de una control, utilice uno de los siguientes métodos:

- seleccione la control y pulse F2.

- seleccionar la control y mostrar la pestaña "[Vista contextual del código](../Editeurs/2027033.md)".. 

- seleccionar "Código" de la el menú contextual de la control (clic con el botón derecho del ratón en la control).

- haga clic en el icono "Código" de control description.


## control Imagen: Modo de funcionamiento de una imagen con enlace
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
Puede definir las acciones de clic de una control Imagen (pestaña "General" de la ventana de descripción del control) para convertirla en control Imagen con enlace.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=image_clic_modeFoctionnemen.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Acción del clic: No llamar al servidor
<a name="accion_del_clic_llamar_servidor_ELTTEXTE000132"></a>
Si la acción de clic es "Sin llamada al servidor", la control Imagen será una simple imagen clicable, associated sólo con un código de clic del navegador. Puede asociar una acción a un clic en la Image: 

- en el editor, en el campo "Tipo. 

- de forma programada, sólo en el código del navegador. 




Sin embargo, no puede get las coordenadas del ratón ni definir las áreas clicables.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Acción del clic: Llamar al servidor
<a name="accion_del_clic_llamar_servidor_ELTTEXTE000156"></a>
Si la acción de clic es "Llamada del servidor", la imagen será clicable y se associated con un código de clic del servidor. 

Si "Durante la acción" se set a "Enviar coordenadas de clic", puede get las coordenadas de clic en el código del servidor (utilizando [ImageXPos](../WDLang2/3045003.md) y [ImagenYPos](../WDLang2/3045001.md), o [PageParameter](../WDLang2/3058026.md) con el parámetro "POS"). 

Para obtener más información, consulte [Recuperación de las coordenadas del clic](../WDChamp/1013130.md).

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Acción del clic: Áreas de clic definidas en edición (Mapa de imágenes)
<a name="accion_del_clic_areas_clic_definidas_edicion_mapa_imagenes_ELTTEXTE000180"></a>
Si la acción de clic es "Hacer clic en áreas definidas en modo edición (mapa Image)", el Image soportará áreas clicables. Este tipo de Image también se denomina Mapa Image.

Una acción puede ser associated con el clic en el Image y en cada área clicable definida.

Para obtener más información, consulte [Definir las áreas sobre las que se puede hacer clic](../WDChamp/1013157.md).

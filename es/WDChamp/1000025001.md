## Presentación del Action Bar
<a name="presentacion_del_action_bar_ELTTEXTE000231"></a>
Las "Barras de acciones" sustituyen a la barra de título de una ventana y sirven para mostrar diversa información:

- el ícono de la aplicación,

- el título de la ventana,

- algunas opciones de menú,

- el menú desplegable de la ventana.

## Crear y configurar un Action Bar
<a name="crear_configurar_action_bar_ELTTEXTE000255"></a>
De forma predeterminada, las ventanas se crean con un Action Bar. 

Puede configurar las barras de la ventana: 

1. Abra la ventana de descripción (opción "Descripción" del menú contextual). 

2. En la pestaña "Estilo", especifique el contenido de la barra superior: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Action_bar%20-%20HC%20N%B0001.gif)


	- Pantalla completa: no se muestra ninguna barra,

- Barra de sistema sola: Solo se muestra la barra de sistema. 

- Barra de sistema + Action Bar: Se muestra la barra de sistema y el Action Bar. 

- Solo Action Bar: Solo se muestra el Action Bar. 




Según el tipo de barra seleccionada, el botón "Estilo de barra" permite set las opciones de estilo utilizadas (color y fuente). 

Observaciones: 





### Estilo bar
<a name="estilo_bar_ELTPARAGRAPHE000116"></a>

El botón "Estilo de barra" permite definir las opciones de estilo de las barras seleccionadas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Action_bar%20-%20HC%20N%B0002.gif)


- Barra de sistema.

- Action Bar.

- Barra de navegación. 






**Si usa el Action Bar** , puede configurar: 

- el fondo color: puede ser una específica o automatic color. 

- el color de la fuente. 

- si se debe aplicar la fuente Color a los iconos Active. 


el fondo Image del Action Bar. 

- si la barra debe mostrarse sobre los controles. Si se marca la opción "Barra en modo de superposición", se modifica el origen de los control s y se puede utilizar el [Opacidad](../Proprietes/2514006.md) Property en el Action Bar control. La barra aparecerá sobre los controles. 




**Si usa la barra de navegación** , puede configurar los colores utilizados para los elementos activos y Inactive.




## Menú contextual de los controles del Organizador (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000243"></a>
Cada control Agenda que se encuentra en una ventana propone un menú al usuario.

Este menú, disponible sin programación, incluye varias acciones:

- Cancelar o rehacer una acción.

- Elección del modo de visualización del organizador: Día, Semana Completa, Semana Laboral, Mes.

- Mostrando los elementos de hoy.

- Gestión de las citas: Adición, eliminación, modificación.

- Copiando el contenido de Appointment.

- Impresión del organizador.

- Envío del organizador por Email. 




Las acciones realizadas en la control Agenda pueden ser: 

- deshacer mediante "Deshacer" en el menú contextual o mediante Ctrl + Z. 

- rehacer mediante "Rehacer" en el menú contextual o mediante Ctrl + Y.


Estas acciones corresponden principalmente al movimiento, cambio de tamaño, eliminación, ... de citas.
**Observación**: Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

## Ajustes
<a name="ajustes_ELTTEXTE000267"></a>


### Configurar el menú contextual en un control Agenda
<a name="configurar_menu_contextual_control_agenda_ELTPARAGRAPHE000046"></a>

El menú contextual de control Agenda se puede desactivar. Para configurar la visualización del menú contextual associated con el control Agenda, debe:

1. Mostrar la ventana description del control Agenda.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, se mostrará el menú automático del control Agenda. Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Menu_Agenda.gif)








Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.

- [AAFDisable](../WDLang1/1000022018.md) se utiliza para eliminar algunas opciones del menú contextual AAF específicas de los controles del Organizador. Se pueden utilizar diferentes constantes según la opción de deshabilitar: 

	- *aafPrintOrganizer*: se usa para borrar la opción de imprimir el organizador. Atención, esta constante también puede borrar la opción de impresión propuesta por los controles del Planificador..  

	- *aafSendFileByEmail*: se utiliza para eliminar la opción de envío del organizador por Email. Atención, esta constante también puede borrar la opción de envío de Email propuesta por los controles del Programador y por los controles de edición de la Ruta.

### "Opción"Imprimir
<a name="opcionimprimir_ELTPARAGRAPHE000097"></a>

La opción "Imprimir" se utiliza para imprimir el contenido de la control Agenda. Aparece una ventana específica que le permite configurar la impresión. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Imp_Planning.gif)


Esta ventana permite configurar: 

- la orientación de reporte.

- la vez que interval: día, semana, mes, ....

- el período a imprimir (set de fechas). 




**Observación**: En función de las opciones especificadas para la duración por Page y para el período, la impresión del organizador puede requerir más de un Page. Por ejemplo, un organizador que imprima el programa diario durante una semana imprimirá 1 Page por día. 

Con la tecla "Imprimir" se inicia la impresión. 

El contenido de la control Agenda se visualiza en el visor de reporte.

Independientemente de la orientación seleccionada, el contenido de la control Agenda se imprime íntegramente entre los límites especificados..   

Las características de impresión son las siguientes:

- El título de la impresión corresponde a la leyenda de la control Agenda. Si no se ha especificado ningún título para la control Agenda, reporte no muestra ningún título.

- La impresión respeta la Color utilizada en el control Agenda.

- La fecha de hoy se imprime en la parte inferior izquierda.

- El número de la Page actual y el número total de Page s se imprimen en la parte inferior derecha Corner.

- El documento se imprime utilizando la dirección de escritura especificada con la propiedad [TextDirection](../Proprietes/2513020.md) a nivel de ventana y/o control Agenda.




**Observaciones**: 

- La opción "Imprimir" del menú contextual de los controles del Organizador requiere proporcionar las librerías de impresión del WINDEV framework con el ejecutable.

- Para desactivar la opción "Imprimir" en el menú contextual de los controles del Organizador, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafPrintOrganizer*.

- Para personalizar la reporte asociada a una control Agenda específica, debe crear un nuevo Organizador reporte basado en esta control Agenda. Se associated automáticamente con el control Agenda y se imprime a través de la opción "Imprimir".

### Personalizar la ventana de configuración para imprimir el planificador
<a name="personalizar_ventana_configuracion_para_imprimir_planificador_ELTPARAGRAPHE000139"></a>

WINDEV le permite incluir en su aplicación la ventana de configuración para imprimir el organizador utilizado por la AAF.. Usted tendrá la posibilidad de personalizar:

- el aspecto de esta ventana (por ejemplo, modificación de plantilla),

- los idiomas soportados por esta ventana. Esta característica es muy useful a la hora de traducir esta ventana a los diferentes idiomas disponibles en la aplicación.  




Para incluir la ventana para configurar la impresión en su aplicación: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccione el subdirectorio que contiene los elementos a importar. La ventana de ajustes de impresión se encuentra en el subdirectorio "Programas\\Datos\\Ventanas de preajuste\\AAF" del directorio WINDEV instalación.

3. WINDEV hace una lista de los elementos del directorio. Este directorio contiene los elementos que corresponden a las AAF personalizables. La ventana para importar se llama "WinDevAaf_ConfigPrintingSchedulerOrganizer.wdw".

4. Validar. La ventana está incluida en el proyecto y se puede modificar. Esta ventana personalizada se tendrá en cuenta automáticamente al ejecutar la aplicación.




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario. 

- Si está utilizando un control Agenda en su proyecto, el AAF de impresión del control Agenda también utilizará esta ventana.

### Opción "Enviar por correo electrónico"
<a name="opcion_enviar_por_correo_electronico_ELTPARAGRAPHE000177"></a>

La opción "Enviar por Email" se utiliza para:: 

- imprimir el contenido de la control Agenda en un archivo "Organizador.pdf. 

- crear un nuevo mensaje en la aplicación de mensajería Default y adjuntar el archivo "Organizer.pdf" a este mensaje. 




El Procedure para imprimir el organizador en el archivo PDF es idéntico al Procedure de impresión realizado a través de la opción "Imprimir el organizador" (véase el párrafo anterior). 

El Organizador reporte generado no se visualiza en el visor de reporte sino que se imprime directamente en un archivo en formato PDF. 

**Observaciones**: 

- La opción "Enviar por Email" en el menú contextual de los controles del Organizador requiere proporcionar las bibliotecas del WINDEV framework para imprimir y generar archivos PDF con el ejecutable.

- Para desactivar la opción "Enviar por correo electrónico" en el menú contextual de los controles del Organizador, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafSendFileByEmail*. 
	Atención, esta constante también puede borrar la opción de envío de Email propuesta por los controles del Programador y por los controles de edición de la Ruta.

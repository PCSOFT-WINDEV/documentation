
## Definición de un nombre único para el archivo generado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000110"></a>
Al generar un archivo en el servidor Web (PDF, HTML, ...), debe definir un nombre de archivo único.

Se pueden utilizar varios métodos para definir un nombre de archivo único.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000134"></a>
Para definir un nombre de fichero unívoco, puede hacerlo:

- concatenar el resultado de [DateSys](../WDLang1/3027026.md) y [TimeSys](../WDLang1/3027019.md). 
	Por ejemplo:
	
	```wl
	AFile is string = fWebDir() + "\" + DateSys() + TimeSys() + ".htm"
	```


- utilizar un identificador único que se encuentra en la base de datos del sitio.
	Por ejemplo:
	
	```wl
	AFile is string = fWebDir() + "\" + Customer.Id + Order.Num + ".htm"
	```


- utilizar [fTempFile](../WDLang1/3036011.md).
	Por ejemplo:
	
	```wl
	AFile is string 
	AFile = fTempFile()
	AFile = fExtractPath(AFile, fFile)
	AFile = fWebDir() + "\" + AFile + ".htm"
	```




<a name="NOTE2_2"></a>


### No se debe hacer
<a name="debe_hacer_ELTPARAGRAPHE000050"></a>

Las siguientes funciones no proporcionan un nombre de fichero único:

- [GetIdentifier](../WDLang1/3013028.md).

- [Cuenta de conexión](../WDLang2/3012004.md).

- [Aleatorio](../WDLang1/3013037.md).

- ...

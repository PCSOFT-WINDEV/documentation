
## Manejo de un editor de imágenes control mediante programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000162"></a>
WINDEV le permite manipular un editor de imágenes control a través de la programación. Para ello, utilice directamente la variable del control en el código.

Los controles del editor Image también pueden manejarse de forma programada a través de [Funciones Picxxx](../WDLang1/1000023383.md). Estas funciones se utilizan para crear, guardar, abrir y cerrar un Image en un Image Editor control. 

Esta página de ayuda explica cómo manipular un editor de imágenes control a través de la programación. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Mostrando una imagen en un editor de imágenes control
<a name="mostrando_una_imagen_editor_imagenes_control_ELTTEXTE000186"></a>
Para mostrar un Image en un editor Image control, puedes: 

- usar una asignación (directa o por Variable) con ruta del archivo Image. 
	
	```wl
	// Direct assignment
	IE_MyImage = "C:\Temp\MyImage.bmp"
	// Assignment by variable
	ImagePath = fCurrentDir() + "\MyImage.bmp"
	IE_MyImage = ImagePath
	```


- utilizar una asignación con una [Image](../WDLang1/1000019650.md) Variable.
	
	```wl
	MyImage is Image
	// Load from a file
	MyImage = dLoadImage("image.png")
	// Assign the Image Editor control
	IE_MyImage = MyImage
	```





Observación: Para obtener la ruta del archivo de imagen que se muestra en un control del Editor de imágenes, utilice el [FilePath](../Proprietes/1000023104.md) Property.

## Borrar o reiniciar el contenido de un Image Editor control
<a name="borrar_reiniciar_contenido_image_editor_control_ELTTEXTE000210"></a>
Para borrar o reiniciar un Editor de Imagen control, todo lo que tienes que hacer es asignar una cadena vacía al Editor de Imagen control. Por ejemplo:

```wl
IE_MyImage = ""
```
Observación: También se puede utilizar la función [PicCerrar](../WDLang1/1000023382.md).

## Configuración de las opciones de Default en un editor de Image control
<a name="configuracion_las_opciones_default_editor_image_control_ELTTEXTE000234"></a>
Es posible: 

- definir la herramienta utilizada por defecto al abrir el control del Editor de imágenes con el [InitialTool](../Proprietes/1410088609.md) Property. 

- set las dimensiones Default al crear una nueva imagen en el Editor de Imágenes control con las propiedades [ImageInitialHeight](../Proprietes/1410088607.md) y [ImageInitialWidth](../Proprietes/1410088608.md).

## Manejar la imagen mostrada en el Editor de Imágenes control
<a name="manejar_imagen_mostrada_editor_imagenes_control_ELTTEXTE000258"></a>
A manipular El Image encontrado en la Image Editor control , le recomendamos que use una variable [Image](../WDLang1/1000019650.md). Entonces, todos los [funciones de dibujo](../WDLang1/3029033.md) pueden ser utilizados en este [Image](../WDLang1/1000019650.md) Variable. Para actualizar el control del Editor de imágenes, basta con asignar el control con el [Image](../WDLang1/1000019650.md) Variable. 

También es posible utilizar una Variable de tipo [WDPic](../WDLang1/1000023503.md). Este Variable se utiliza principalmente para gestionar la visualización de las capas en un archivo ".WDPic"..

## Propiedades WLanguage asociadas
<a name="propiedades_wlanguage_asociadas_ELTTEXTE000282"></a>


### Propiedades específicas de la gestión de los controles de Image Editor
<a name="propiedades_especificas_gestion_los_controles_image_editor_ELTPARAGRAPHE000106"></a>

Las siguientes propiedades se utilizan para gestionar las características de un Image Editor control a través de la programación: 



|   |   |
| --- | --- |
| [FilePath](../Proprietes/1000023104.md) | El **FilePath** Property se utiliza para identificar: <br><br>- el nombre del archivo xlsx associated con un control Hoja de cálculo.<br><br>- el nombre del archivo associated con un editor de imágenes control. <br><br>- el nombre del archivo PDF associated con un lector de PDF control. <br><br>- el nombre del archivo DOCX associated con un control Procesador de texto. <br><br>- el nombre del archivo wddiag asociado a un editor Diagram control. <br><br><br> |
| [ImageInitialHeight](../Proprietes/1410088607.md) | La propiedad **ImageInitialHeight** permite obtener o establecer la altura predeterminada de una imagen en un control Editor de imágenes. |
| [ImageInitialWidth](../Proprietes/1410088608.md) | La propiedad **ImageInitialWidth** permite obtener o establecer el ancho predeterminado de una imagen en un control Editor de imágenes. |
| [ImageType](../Proprietes/1000023516.md) | El **ImageType** Property se utiliza para get el tipo de Image que se está editando en un control del Editor de imágenes (Icon o Image). |
| [InitialTool](../Proprietes/1410088609.md) | La propiedad **InitialTool** permite obtener o cambiar la herramienta de dibujo seleccionada de forma predeterminada en un control Editor de imágenes. |
| [PaneVisible](../Proprietes/1000023515.md) | El **PaneVisible** Property se utiliza para: <br><br>- determina si el panel del control del Editor de imágenes es Visible, <br><br>- mostrar u ocultar el panel del Editor de imágenes control.<br><br><br> |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con un editor de imágenes control, consulte [Propiedades associated con los controles del editor Image](../WDChamp/1410087033.md).

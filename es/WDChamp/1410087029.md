## Interactuar con una Web Component control
<a name="interactuar_con_una_web_component_control_ELTTEXTE000127"></a>


### Productos asociados a la Web Component control
<a name="productos_asociados_web_component_control_ELTPARAGRAPHE000019"></a>

Se pueden utilizar varios métodos para interactuar con una Web Component control. 

Por ejemplo, tiene la capacidad de crear procedimientos de Servidor y/o Navegador relacionado al control de Web Component (pestaña "Detalles" del control description). 

Estos procedimientos permiten modificar los datos que se encuentran en la Web Component control asignando un valor procedente de la WEBDEV Page por ejemplo. 

El control Web Component puede llamar directamente a estos procedimientos desde el código HTML de Web Component. El editor de código HTML propone los diferentes procedimientos disponibles a través de la realización de las acciones. 

Por ejemplo, el siguiente código HTML se utiliza para llamar al código "Acción" de la Component durante la pulsación del botón "Pedido: 

```html
<button onclick="CMP_Test()" class="w3-aqua">
	Order
</button>
```


El editor de código HTML propone los diferentes procedimientos y acciones disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CompoWeb_code.gif)


También tiene la posibilidad de llamar a los procedimientos de la Web Component control desde el WEBDEV control s directamente. 




### Procesos para asignar y recuperar el valor Property
<a name="procesos_para_asignar_recuperar_valor_property_ELTPARAGRAPHE000040"></a>

Los procesos para asignar y recuperar el Valor correctamente se utilizan principalmente para inicializar el valor de la Web Component durante la carga de Page y para recuperar su valor. 











## Manejo de la Web Component control mediante programación
<a name="manejo_web_component_control_mediante_programacion_ELTTEXTE000157"></a>
Las siguientes propiedades son específicas de la Web Component control: 



|   |   |
| --- | --- |
| [CodeCSS](../Proprietes/1000022365.md) | El **CodeCSS** Property se utiliza para leer y cambiar el código CSS associated con un componente web control. |
| [CodeHTML](../Proprietes/1000022364.md) | El **CodeHTML** Property se utiliza para leer y cambiar el código HTML associated con un componente web control. |
| [CodeJS](../Proprietes/1000022366.md) | El **CodeJS** Property se utiliza para leer y modificar el código JavaScript associated con un componente web control. |





Para obtener una lista completa de las propiedades de WLanguage que se pueden utilizar con una Component Web control, consulte [Propiedades asociadas a los controles Web Component](../WDChamp/1410087028.md).

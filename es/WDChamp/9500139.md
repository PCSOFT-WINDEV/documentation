


## Colorear una fila, una columna o una celda que se encuentra en una control (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000128"></a>
Para simplificar la lectura en los control s utilizados para ver grandes volúmenes de datos, el usuario final tiene la capacidad de Color los datos que se encuentran en el control. 

Los datos visualizados en los siguientes controles pueden ser coloreados: 

- control Tabla: celda, fila, columna

- Control Tabla TreeView: celda, fila, columna

- control Tabla dinámica: celda o selección múltiple de celdas.  




**Observación**: Esta opción sólo está disponible para los controles rellenados mediante programación o con una fuente de datos en memoria. Esta opción no está disponible para los controles de tabla con acceso directo a la fuente de datos. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Colorear un elemento
<a name="colorear_elemento_ELTTEXTE000152"></a>
**To Color un fila, columna, celda, ...**: 

1. Seleccione el elemento a Color en el control. 

2. Seleccione "Fondo Color" en el menú contextual del elemento y seleccione el Color solicitado en la lista. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Couleur_Table.gif)



La opción "Color por defecto" se utiliza para restaurar el Color inicial. 

**Observación**: El Color seleccionado está disponible sólo cuando se muestra el control. Esta información no se almacenará al cerrar la ventana..

## Programación
<a name="programacion_ELTTEXTE000176"></a>


### Programando la Color de células, filas, ...
<a name="programando_color_celulas_filas_ELTPARAGRAPHE000041"></a>

Los Color s de un control Tabla (fondo Color de filas, celdas) pueden ser modificados mediante programación con las siguientes propiedades:

- [Color](../Proprietes/2510071.md): para modificar el texto Color.

- [BackgroundColor](../Proprietes/2510022.md): para modificar el fondo Color.


Para más detalles sobre las sintaxis a utilizar, véase [Definición de colores en los controles de la tabla](../WDChamp/1013231.md).


### Desactivación de los colores en los elementos de una control
<a name="desactivacion_los_colores_los_elementos_una_control_ELTPARAGRAPHE000058"></a>

Para desactivar los colores en los elementos de un control, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafChangeBackgroundColor*. No se propondrá la opción de menú. 

Por ejemplo: 


```wl
// Prevent from changing the background color
AAFDisable(TABLE_Customer, aafChangeBackgroundColor)
```



### Modificar el texto de la opción "Color de fondo" del menú contextual
<a name="modificar_texto_opcion_color_fondo_del_menu_contextual_ELTPARAGRAPHE000072"></a>

El título de la opción de menú que permite al usuario modificar el fondo Color es "Fondo Color". Para modificar este título, utilice la función [AAFChangeCaption](../WDLang1/1000022100.md) con la constante *aafChangeBackgroundColor*. 

Por ejemplo: 


```wl
// Prevent from changing the background color
AAFChangeCaption(TABLE_Customer, aafChangeBackgroundColor, "Choose a background color")
```

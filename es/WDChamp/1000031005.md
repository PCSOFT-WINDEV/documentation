


## Control Cuadro de mando : Funcionalidades Automáticas de la Aplicación (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000277"></a>
El control Cuadro de mando propone varias características de la aplicación automatic (AAF) que permiten al usuario beneficiarse de todas las capacidades del control. Estas características están disponibles para las aplicaciones Windows, los sitios WEBDEV y las aplicaciones móviles.

<a name="NOTE2_1"></a>

## AAFs en las aplicaciones Windows
<a name="aafs_las_aplicaciones_windows_ELTTEXTE000301"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000030"></a>

Por Default, el control Cuadro de mando propone al usuario: 

- Cambiar al modo de edición: el modo de edición permite a los usuarios configurar su cuadro de mando.

- Guarda el contenido de control Cuadro de mando en formato Image.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChampTdB_FAAsimple.gif)





Cuando el usuario pasa al modo "Editar", puede configurar los diferentes elementos de cuadro de mando:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChampTdB_FAAEdition.gif)
Es posible: 

- esconder un elemento,

- añadir un elemento de Invisible,

- restaurar la configuración inicial.  


El modo de edición también se puede utilizar para modificar la posición y el tamaño de los elementos.. 

<a name="NOTE2_2"></a>


### Características del menú contextual
<a name="caracteristicas_del_menu_contextual_ELTPARAGRAPHE000051"></a>

- Guardar como (imagen): Esta característica se utiliza para guardar el contenido de control Cuadro de mando en formato Image (para imprimirlo por ejemplo). 

- Modo de edición: Esta función permite a los usuarios pasar al modo de edición para poder configurar su cuadro de mando.



<a name="NOTE2_3"></a>


### Modo de edición en la control Cuadro de mando
<a name="modo_edicion_control_cuadro_mando_ELTPARAGRAPHE000059"></a>

Las siguientes opciones están disponibles en el menú contextual del modo Edición: 

- añadir: Esta opción abre un submenú que muestra la lista de widgets realizados por Invisible y que pueden ser añadidos. 

- Ocultar: Esta opción se utiliza para ocultar el widget actual. 

- Restaurar configuración inicial: Esta opción se utiliza para restaurar la cuadro de mando a su estado anterior (antes de las modificaciones realizadas en el modo de edición). Todas las modificaciones realizadas (tamaño, posición, ocultación, adición) se cancelan.. 




El modo de edición puede configurarse a través de la programación: 

- [DashMode](../WDLang1/1000020956.md) se utiliza para: 

	- saber si el modo de edición está activado,

	- activar o desactivar el modo de edición,

	- configurar las acciones disponibles en el modo de edición. Por ejemplo, puede permitir que los widgets se muevan y cambien de tamaño sólo.  




- El uso de [AAFDisable](../WDLang1/1000022018.md) con la constante *aafDashEditMode* desactiva el menú contextual utilizado para cambiar al modo de edición.

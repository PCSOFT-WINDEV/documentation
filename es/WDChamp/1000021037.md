
## Gráfico en tiempo real
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000134"></a>
Este tipo de gráfico se utiliza para representar un flujo de datos permanente en tiempo real.. El gráfico se mueve automáticamente según los datos recibidos en tiempo real. 

Un gráfico en tiempo real es una opción específica de los gráficos de área, de líneas, de columnas y de dispersión. 

Para que un gráfico se convierta en un gráfico en tiempo real, es necesario: 

- para especificar -en el description de control- que el gráfico utiliza un eje X cuyas marcas se basan en una escala de tiempo, así como para especificar el interval para un gráfico en tiempo real. 

- para añadir los datos en el gráfico mediante la programación a través de [grAddTimeData](../WDLang3/1000023532.md). Esta función permite actualizar el gráfico. 
	Observación: Se recomienda utilizar una automatic Procedure (timer) para actualizar regularmente los datos del gráfico.

## Características de los gráficos en tiempo real
<a name="caracteristicas_los_graficos_tiempo_real_ELTTEXTE000158"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe%20temps%20r%E9el%20-%20HC%20N%B0001.gif&type=thumb)


En este cuadro:

**1. Título del gráfico**. 
Las características del título del gráfico pueden definirse en la pestaña "Detalles" del control description ("Editar título del gráfico").

Puede definir el título del gráfico, su posición y su estilo.

**2. Título de los ejes**. 
El título de los ejes horizontales y verticales se muestra si el título se definió en la pestaña "General" de la control Gráfico description. Este parámetro también puede definirse mediante programación con [grAxisTitle](../WDLang3/3042057.md).

**3. Coordenada X**. 
En un gráfico en tiempo real, el eje X está espaciado de acuerdo a intervalos de tiempo. En la pestaña "General" de la control Gráfico description, se pueden definir los siguientes elementos: 

- Modo de visualización del tiempo: : Es posible: 

	- utilizar un indicador estándar: el tiempo transcurrido comienza en 0.  

	- para mostrar el tiempo transcurrido desde la última adición: la duración se recalcula en cada pantalla. 0 corresponde a la última adición hecha a la carta. 




- Intervalo visualizado para un gráfico en tiempo real (ms): Esta interval se utiliza para definir el tiempo que interval muestra en el gráfico. Si el tiempo interval es de 30000ms, el gráfico mostrará los datos sobre un interval de 30 segundos. Por defecto, esta vez interval es 0 (equivalente a 60s).  




**4. mito**. 
Las características de la leyenda pueden definirse en la pestaña "Detalles" del control description ("Editar leyenda del gráfico").

Puedes definir su posición y su estilo.

**5. Etiquetas de serie**. 
El título de la leyenda corresponde al título de las diferentes series mostradas en el gráfico (definido en la pestaña "Series").

**6. Color de serie**. 
Se puede definir un color para cada serie en la pestaña "Serie" del control description. Un automatic Color es usado por Default.

**7. Etiqueta de categoría**. 
Las etiquetas de las categorías se pueden definir en la pestaña "Serie" del control description.

**8. Líneas de cuadrícula**. 
Las cuadrículas horizontales y verticales pueden definirse en la pestaña "General" del control description.

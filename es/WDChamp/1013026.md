### Eventos soportados por WINDEV y WINDEV Mobile
<a name="eventos_soportados_por_windev_windev_mobile_ELTPARAGRAPHE000013"></a>

WINDEV y WINDEV Mobile gestionan los siguientes eventos por Default (orden de aparición en el editor de código):


| Event | Condición de ejecución |
| --- | --- |
| Inicialización | Corre cuando se abra la ventana. (\*)<br><br>**Caso especial: Combo Box archivo de datos rellenado automáticamente**:<br>Cuando se ejecuta este código, la control Combo Box aún no se ha rellenado.. Para forzar a la control a rellenar este código, use:<br><br>- la función [ListDisplay](../WDLang1/3049005.md), <br><br>- las propiedades para gestionar los controles de navegación de Combo Box: [BrowsedFile](../Proprietes/2510040.md), [BrowsedItem](../Proprietes/2510111.md) o [filtros](../Proprietes/2510042.md).<br><br><br> |
| Fin de inicialización | **Code se encuentra sólo para navegación llena automáticamente Controles Combo Box**: <br>Corre una vez que el control Combo Box control se haya llenado. Este código permite seleccionar un elemento en la control, o añadir o modificar un elemento. |
| partida | Ejecutar cuando la control gana el foco |
| salida | Ejecutar al pasar de un control a otro (tecla TAB, seleccionar otro control con el ratón, etc.) |
| Siempre que se modifique | **Code encontrado sólo para controles Combo Box editables**: <br>Ejecutar cuando el usuario introduce datos en un control Combo Box editable. |
| Selección de una fila | Ejecutar cuando el usuario selecciona un elemento en la fila o modifica el elemento seleccionado. |


(\*) Por defecto, la "Inicialización" Event de cada control se ejecuta según el orden en que se crearon las control en la ventana. Para modificar este orden de ejecución: 

1. En la pestaña "Ventana", en el grupo "Orden", haga clic en "Inicialización". 

2. En la ventana mostrada, utilice las flechas de la derecha para modificar el orden en el que se inicializan los controles.

## Eventos opcionales
<a name="eventos_opcionales_ELTTEXTE000276"></a>
Se pueden utilizar varios eventos opcionales.
Para añadir un evento opcional:

1. Seleccione la control deseada.

2. Mostrar la ventana del código de esta control (tecla F2).

3. Haga clic en el enlace "Añadir otras Event s a xxx" en la parte inferior del código de la ventana, después de la última Event.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Traitements_optionnels_WD_OK%20-%20HC%20N%B0001.gif)


4. Todos los eventos opcionales disponibles para el control se muestran. 

5. Marque la opción Event para agregar y validar. 
	Observación: Puede seleccionar varios eventos opcionales. 

6. El Event opcional seleccionado se agrega automáticamente a los Event s administrados por el control.




Para desactivar un Event opcional, simplemente realice las mismas operaciones para mostrar la lista de Event s opcionales. A continuación, simplemente desmarque los eventos opcionales para eliminar. 
Observación: Si el código desactivado contiene código WLanguage, este código se elimina automáticamente.Se puede gestionar, por ejemplo:

- el desplazamiento del cursor por encima del control,

- botón izquierdo del ratón abajo, arriba, con un doble clic,

- el botón derecho del ratón presionado, liberado o con doble clic 

- la rueda del ratón, etc.

### Eventos opcionales de una control Combo Box: abrir y cerrar los controles Combo Box
<a name="eventos_opcionales_una_control_combo_box_abrir_cerrar_los_controles_combo_box_ELTPARAGRAPHE000170"></a>

Dos eventos opcionales son associated con controles Combo Box:

| Event | Condición de ejecución |
| --- | --- |
| Abrir la lista desplegable | Ejecutar cuando el usuario muestra el contenido de la control Combo Box.<br>Se utiliza para llenar una control Combo Box dinámicamente cuando se abre. |
| Cerrar la lista desplegable | Ejecutar cuando el usuario cierra la lista desplegable de la control Combo Box.<br>Se utiliza para posicionar el cursor en otro control cuando se valida el control Combo Box. |
| ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Introducir un nuevo valor | Ejecutar cuando el usuario valida el nuevo valor introducido en la control Campo de entrada de la control Combo Box. <br>Se utiliza para comprobar el valor introducido, o añadirlo a un archivo de datos... |

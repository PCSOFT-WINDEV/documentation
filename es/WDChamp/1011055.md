## Las diferentes fuentes de datos
<a name="las_diferentes_fuentes_datos_ELTTEXTE000154"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten crear informes con el editor de reportes. Los informes se pueden crear con los datos procedentes:

- [un archivo de datos](../WDChamp/1011052.md).

- [el resultado de una consulta](../WDChamp/1011013.md).

- [una vista en un archivo de datos HFSQL](../WDChamp/1011064.md).

- [una zona de memoria](../WDChamp/1011061.md).

- [una control Tabla encontrada en una ventana o en una página](../WDChamp/1011012.md).

- [una control Looper encontrada en una ventana o en una página](../WDChamp/1011093.md).

- [el Variable](../WDChamp/1011058.md). 

- [una control Tabla TreeView encontrada en una ventana o en una página](../WDChamp/1011082.md).

- [una control Agenda encontrada en una ventana o en una página](../WDChamp/1011081.md). 

- [una control Planificador encontrada en una ventana o en una página](../WDChamp/1011080.md). 

- [un archivo de texto](../WDChamp/1011011.md).

- [una fuente de datos programada](../WDChamp/1011034.md).




Los editores de reporte también le permiten crear informes associated sin ninguna fuente de datos.|   |   |   |

## Notas
<a name="notas_ELTTEXTE000196"></a>
Si se selecciona una de las fuentes de datos mencionadas por Above, el editor de reportes gestiona completamente la programación de las operaciones de exploración realizadas en los registros para mostrarlos en el reporte.

Para programar y seleccionar los registros que se mostrarán en el reporte, elija un [reporte sobre fuente de datos programada](../WDChamp/1011034.md).

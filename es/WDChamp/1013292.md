


## Manejo de un gráfico en el editor
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
En el editor, un control Gráfico incluye diferentes secciones:

- ***el título del gráfico***. 
	Esta área sólo aparece si se ha definido un título para el gráfico.. 
	Para definir el título, independientemente del editor, seleccione la opción "Editar el título de la carta" en la pestaña "Detalles. 

- ***la leyenda de la carta***. 
	Esta área sólo aparece si se ha definido una leyenda para el gráfico.. 
	Para definir una leyenda, independientemente del editor, seleccione "Editar la leyenda de la carta" en la pestaña "Detalles".. 

- ***el gráfico***.




Cada pieza puede configurarse por separado.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000110"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_Zones.gif)
Las tres áreas de una control Gráfico se pueden tratar por separado. Cada área puede ser posicionada manualmente: Basta con seleccionar la zona deseada con el cursor y cambiar su ubicación.

Por ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_Zones1.gif)
 Haga doble clic en una zona para ver su description.. Para ver la leyenda description, haz doble clic en la leyenda en el editor.

**Observación**: El posicionamiento definido para el título y para la leyenda puede ser automatic o manual. Tan pronto como el área correspondiente se modifica o se mueve con el ratón, su posicionamiento se convierte en manual.. Para restablecer el posicionamiento inicial, basta con seleccionar un posicionamiento automatic para la zona.

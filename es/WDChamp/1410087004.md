
## Manejo de una control de cable a través de la programación
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
WINDEV le permite manipular un control de alambre a través de la programación. Para hacerlo, usa la Variable del alambre control en el código.

Para manipular una control por programación, WINDEV propone varias funciones y propiedades. 

En esta página de ayuda se explica cómo manipular un control por medio de la programación. 





## Manejo de la control de alambre a través de la programación
<a name="manejo_control_alambre_traves_programacion_ELTTEXTE000106"></a>


### Las funciones de WLanguage
<a name="las_funciones_wlanguage_ELTPARAGRAPHE000018"></a>

Las siguientes funciones WLanguage se utilizan para manipular a Wire control: 



|   |   |
| --- | --- |
| [AttachPointAdd](../WDLang1/1000021902.md) | Define un punto de unión para una de las dos extremidades de una control de alambre. |
| [AttachPointDeleteAll](../WDLang1/1000021903.md) | Borra todos los puntos de conexión de una control. |








### Propiedades específicas de la gestión de un control de alambre
<a name="propiedades_especificas_gestion_control_alambre_ELTPARAGRAPHE000026"></a>

Las siguientes propiedades se utilizan para gestionar las características de una control de alambre a través de la programación: 



|   |   |
| --- | --- |
| [AttachPointEnd](../Proprietes/1000021908.md) | El **AttachPointEnd** Property se utiliza para get y cambiar la posición del punto de fijación final de un control. |
| [AttachPointStart](../Proprietes/1000021907.md) | El **AttachPointStart** Property se utiliza para get y cambiar la posición del punto de fijación inicial de un control. |
| [EndControl](../Proprietes/1000021878.md) | El **EndControl** Property se utiliza para determinar o cambiar el final control de un cable control. |
| [MaxNbSegment](../Proprietes/1000021906.md) | El **MaxNbSegment** Property se utiliza para get y cambiar el número máximo de segmentos Line para dibujar un Alambre control. |
| [OriginWaypoint](../Proprietes/1000022161.md) | El **OriginWaypoint** Property se utiliza para obtener y definir el origen de los puntos cuyas coordenadas se especifican en el [WayPoint](../Proprietes/1000021957.md) Property. |
| [ShapeExtremityEnd](../Proprietes/1000021890.md) | El **ShapeExtremityEnd** Property se utiliza para identificar y cambiar el tipo de forma utilizada para el final de un cable control. |
| [ShapeExtremitySize](../Proprietes/1000021960.md) | El **ShapeExtremitySize** Property se utiliza para identificar y cambiar el tamaño de la forma al principio y al final de un Alambre control. |
| [ShapeExtremityStart](../Proprietes/1000021889.md) | El **ShapeExtremityStart** Property se utiliza para get y cambiar la forma del extremo de origen de un control. |
| [StartControl](../Proprietes/1000021877.md) | El **StartControl** Property se utiliza para determinar o cambiar el control de inicio de un control. |
| [WayPoint](../Proprietes/1000021957.md) | El **WayPoint** Property se utiliza para determinar y modificar los puntos por los que debe pasar un cable control. |





Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con un control de alambre, consulte [Propiedades associated con mandos Wire](../WDChamp/1410087003.md).

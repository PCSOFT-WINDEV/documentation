<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
El editor de reportes permite crear informes de "Hoja de cálculo".. 

En este caso, la reporte sólo contiene un control Hoja de cálculo. 

Observación: Este tipo de control se crea cuando se crea una hoja de cálculo reporte y también se puede crear independientemente de este reporte.

## Creación de una hoja de cálculo reporte
<a name="creacion_una_hoja_calculo_reporte_ELTTEXTE000165"></a>


### Crear un reporte
<a name="crear_reporte_ELTPARAGRAPHE000024"></a>

Para crear una "hoja de cálculo" reporte: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- Aparece la ventana de creación: haga clic en "Reporte", y luego en "Reporte". 

	- El asistente de creación de reportes se abre. 




2. Seleccione una "Hoja de cálculo" reporte. 

3. En el resto de la asistente, especifique:

	- la fuente de datos de la control Hoja de cálculo creada en el reporte. Esta fuente puede ser: 

		- Artículo de un archivo de datos,

		- una control Hoja de cálculo en una ventana. 

		- ninguna, si se asigna control Hoja de cálculo a través de la programación. 




- opciones de escalado para la control Hoja de cálculo en la reporte: 

	- Ninguna escala: La hoja de trabajo se imprime con sus dimensiones. La impresión se puede realizar en varias páginas de ancho y alto si es necesario.. 

	- Ajustar hoja de cálculo a las dimensiones del control: La hoja de trabajo se reducirá para que se muestre en su totalidad en el control Hoja de cálculo. 

	- Ajustar todas las columnas al ancho del control: Las columnas de la hoja de trabajo se reducirán o ampliarán para ajustarse al ancho del control. 

	- Ajustar todas las filas al alto del control: Las filas que se encuentran en la hoja de trabajo se reducirán o ampliarán para que todas las filas se muestren a la altura de control.

## Características de una hoja de cálculo reporte y de una control Hoja de cálculo
<a name="caracteristicas_una_hoja_calculo_reporte_una_control_hoja_calculo_ELTTEXTE000189"></a>
Se pueden modificar las siguientes características de una hoja de cálculo reporte: 

- Por Default, la hoja de cálculo reporte no es relacionado a una fuente de datos. La fuente de datos de la hoja de cálculo reporte puede modificarse en la pestaña "Datos" de la ventana reporte description ("Reporte description" en el menú emergente de reporte). 

- La pestaña "General" de la ventana control Hoja de cálculo description permite especificar las características de escalado del control Hoja de cálculo. 

- La pestaña "Contenido" de la ventana del control Hoja de cálculo description le permite vincular el control Hoja de cálculo a una fuente de datos: 

	- Memorándum de un archivo de datos o consulta, etc., ... 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) control Hoja de cálculo en una ventana. 

	- [xlsDocument](../WDLang5/1000017464.md) variable. 




- La pestaña "Estilo" de la ventana control Hoja de cálculo description permite especificar el estilo de la cuadrícula y el texto del encabezado que se muestra en el control Hoja de cálculo. 




Para obtener más información, consulte [control Hoja de cálculo en un reporte](../WDChamp/1000033009.md).

## Manejo de una control Hoja de cálculo en una reporte mediante programación
<a name="manejo_una_control_hoja_calculo_una_reporte_mediante_programacion_ELTTEXTE000213"></a>
Puede asociar la hoja de cálculo de un reporte con: 

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md). Simplemente usa el código: 
	
	```txt
	<Spreadsheet control> = <xlsDocument variable>
	```


- un archivo xls/xlsx. Simplemente usa el código: 
	
	```txt
	<Spreadsheet control> = <Path of xls/xlsx file>
	```





Atención: Las funciones de gestión de control Hoja de cálculo no se pueden utilizar en el control Hoja de cálculo de un reporte.

## Vista general de la barra de herramientas control
<a name="vista_general_barra_herramientas_control_ELTTEXTE000119"></a>
La barra de herramientas control es un tipo avanzado de control, relacionado a una ventana. Una barra de herramientas contiene varios controles (botón, control Campo de entrada, combo box,...) que permiten al usuario iniciar rápidamente algunas características de su aplicación.

Una barra de herramientas puede serlo:

- **Anclado**: la barra de herramientas está "unida" a uno de los bordes de la ventana. 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=champ%20barre%20outils%20-%20HC%20N%B0002.gif)


- **Flotante**: la barra de herramientas es una ventana independiente.
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=champ%20barre%20outils%20-%20HC%20N%B0001.gif)





El usuario puede cambiar el estado de la barra de herramientas con el ratón.

## Creación de una barra de herramientas control
<a name="creacion_una_barra_herramientas_control_ELTTEXTE000143"></a>


### Para crear una barra de herramientas control:
<a name="para_crear_una_barra_herramientas_control_ELTPARAGRAPHE000029"></a>

1. En la pestaña "Ventana", en el grupo "Barras y menús", amplíe "Otras barras" y seleccione "Barra de herramientas".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción" en el menú contextual. Las diferentes posiciones permitidas para la barra de herramientas (acoplada o flotante) se pueden configurar en la pestaña "Detalles".

**Atención: Ningún control debe ser colocado cerca de una barra de herramientas**: Si los controles están colocados cerca de la barra de herramientas creada actualmente (a la izquierda de una barra de herramientas horizontal o debajo de una barra de herramientas vertical), estos controles se moverán automáticamente (y por tanto podrían desaparecer) cuando el usuario modifique la posición de la barra de herramientas. ### Asociar controles con la barra de herramientas
<a name="asociar_controles_con_barra_herramientas_ELTPARAGRAPHE000053"></a>

**Para asociar controles existentes con la barra de herramientas**:
**Solución 1**: Arrastrar y soltar el control a la "Barra de herramientas" control.
**Solución 2**: En la control description (pestaña "General"), haz clic en el botón "Add" y selecciona el control que se añadirá a la barra de herramientas. La control seleccionada se crea automáticamente en la barra de herramientas.

**Para asociar nuevos controles a la barra de herramientas**:
**Solución 1**: Arrastrar y soltar el control a la "Barra de herramientas" control.
**Solución 2**: En la control description (pestaña "General"), haz clic en el botón "Add" y selecciona el tipo de control que se añadirá a la barra de herramientas. La control seleccionada se crea automáticamente en la barra de herramientas. Para modificarla, haga clic en el botón "Editar: se muestra la description del nuevo control.

**Observación**: El espacio entre los controles que se encuentran en la barra de herramientas se puede configurar en [la pestaña "Detalles" de la barra de herramientas description](../WDChamp/1013119.md).

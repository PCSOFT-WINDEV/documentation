## Presentación

### Control de página
<a name="control_pagina_ELTPARAGRAPHE000011"></a>

Los sitios web proponen varios efectos de transición entre los elementos: 

- imágenes cargadas progresivamente en la página. 

- imágenes en movimiento.

- Image apareciendo con efectos superpuestos. 




También tienes la capacidad de realizar un efecto "movimiento automatic Image".. Este efecto simula el movimiento de la luz de una cámara sobre una Image sin programar.

WEBDEV le permite incluir fácilmente estas soluciones en sus sitios Web.

## Trasladar una Image (o cualquier tipo de control)
<a name="trasladar_una_image_cualquier_tipo_control_ELTTEXTE000219"></a>
WEBDEV le permite dar fácilmente un efecto gráfico al mover un control. Para ello, utilice la función [AnimationPlayOnControlProperty](../WDLang1/1000017398.md). Esta función, utilizada en el código del navegador, permite definir la animación que se realizará al modificar un control Property. 

Por lo tanto, para desplazar una control, basta con utilizar las propiedades [X](../Proprietes/2510135.md) y [Y](../Proprietes/2510137.md) de la control. Por ejemplo: 


```wl
AnimationPlayOnControlProperty(IMG_Image1, "X", 0, 200, 50000, animDeceleration)
AnimationPlayOnControlProperty(IMG_Image1, "Y", 0, 200, 50000, animDeceleration)
```
También puede utilizar esta función con otras propiedades para crear efectos especiales.: [anchura](../Proprietes/2510059.md), [Altura](../Proprietes/2510050.md), [Color](../Proprietes/2510071.md), [BackgroundColor](../Proprietes/2510022.md), [Opacidad](../Proprietes/2514006.md), ...

## Efecto de apariencia de una Image por recorte o desvanecimiento
<a name="efecto_apariencia_una_image_por_recorte_desvanecimiento_ELTTEXTE000243"></a>
WEBDEV le permite fácilmente dar un efecto de apariencia a un Image. Este Image se puede encontrar en: 

- un control Imagen, 

- a control Miniatura. 




Se debe utilizar el siguiente método: 

1. Vaya a la pestaña "UI" del control description (Image o control Miniatura). 

2. En la opción "Transición", seleccione el modo de transición deseado: swipe, overlay, underlay, ...
	El modo de transición "Aleatorio" permite activar de forma aleatoria varios modos de transición en el mismo control.

3. A través de la programación, cambiar la imagen associated con la control. Por ejemplo, en el código del navegador: 
	
	```wl
	IMG_Image1.Value = "images\hotel\tradewinds.jpg"
	```


4. En el momento de la ejecución, la nueva imagen sustituye a la que se encuentra en la control con el efecto solicitado.

## Efecto del movimiento automatic Image
<a name="efecto_del_movimiento_automatic_image_ELTTEXTE000267"></a>
WEBDEV permite dar fácilmente un efecto de "movimiento de automatic en un Image".  

Se debe utilizar el siguiente método: 

1. Vaya a la pestaña "UI" del control description (Image o control Miniatura). 

2. En la opción "Transición", seleccione "Movimiento panorámico". 

3. En tiempo de ejecución, el Image se mueve sin ninguna programación. 




Este efecto simula el ligero desplazamiento de una cámara sobre una imagen. Se combinan tres efectos: 

- se deslizan en direcciones aleatorias,

- zoom ligero,

- para la secuencia de efectos. 




**Para utilizar este efecto en un set de imágenes**, esta opción debe combinarse con el [automatic desplazamiento de imágenes](../WDChamp/9500118.md). 

1. Vaya a la pestaña "UI" del control description (Image o control Miniatura). 

2. En la sección "Transiciones", despliegue "Transición" y seleccione "Movimiento panorámico". 

3. En la sección "Deslizamiento automático", seleccione la opción "Deslizar imágenes de un directorio o lista programada". Según el resultado deseado, también puede seleccionar las opciones "Iniciar automáticamente", "Examinar directorio de la imagen (Pestaña General)" o "Ordenar contenido del deslizamiento".
	 Para obtener más información, consulte [automatic desplazamiento de imágenes](../WDChamp/9500118.md). 

4. Valide.




**Atención**: El movimiento Image está optimizado para los navegadores que soportan el estándar HTML 5 (Chrome, Firefox, ...). De lo contrario, el movimiento puede no parecer suave.

## Presentación del control Barra de progreso

### Control de ventana o página dinámica
<a name="control_ventana_pagina_dinamica_ELTPARAGRAPHE000011"></a>

Existen diferentes tipos de barras de progreso:

- Barras de progreso lineales simples (horizontales o verticales).
	

- Barras de progreso lineales simétricas (horizontales o verticales): la barra de progreso está dividida en dos secciones (negativa y positiva) para mostrar el progreso.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=jauge_exemple_lineaire.gif)
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este tipo de barra de progreso no está disponible.

- Medidores radiales: una aguja se mueve para indicar el valor del medidor.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Medidores radiales "LCD": una "barra LCD" se llena para indicar el valor de la barra de progreso.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Jauge_exemple_circulaire.gif)

### Crear un control Barra de progreso en una ventana
<a name="crear_control_barra_progreso_una_ventana_ELTPARAGRAPHE000070"></a>

Para crear un control Barra de progreso: 

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue la opción "Barra de progreso" y seleccione una barra de progreso lineal o un medidor radial. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Jauge20.gif)

	Observación: la opción "Barra de progreso infinita" permite crear un control [Barra de progreso infinita](../WDChamp/1000034001.md). 

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Diferentes medidores radiales predefinidos vienen con el producto. Estos medidores radiales están disponibles en el panel "Controles predefinidos" del editor.

- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo están disponibles las barras de progreso lineales simples.

### Características de las barras de progreso en una ventana: ventana de 7 pestañas
<a name="caracteristicas_las_barras_progreso_una_ventana_ventana_7_pestanas_ELTPARAGRAPHE000143"></a>

La pestaña "General" sirve para configurar:

- El tipo de barra de progreso: lineal (horizontal o vertical) o radial (con aguja o LCD).

- Los valores de la barra de progreso. Estos valores también se pueden especificar mediante programación. Para obtener más información, consulte [Manipular una barra de progreso mediante programación](../WDChamp/1013141.md). 




La pestaña "Detalles" permite configurar:

- Las imágenes mostradas en la barra de progreso (para barras de progreso lineal)
	Puede utilizar el modo de escala de 9 sectores en las imágenes de las barras de progreso.

- Si la barra de progreso es simétrica.
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción no está disponible.

- Si la barra de progreso utiliza una imagen fija.

- Si se debe mostrar el porcentaje de progreso en el control Barra de progreso. Puede elegir ocultar el porcentaje si es cero. También puede definir la posición del porcentaje. Esta posición puede modificarse mediante programación con la propiedad [HorizontalAlignment](../Proprietes/2510023.md). El formato del porcentaje puede definirse mediante programación con la propiedad [InputMask](../Proprietes/2510033.md).
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción no está disponible.




La pestaña "Estilo" permite configurar los parámetros visuales de la barra de progreso (incluyendo las imágenes y su posición en los medidores radiales).

**Observación**: Esta información también está disponible para las barras de progreso que se muestran en los controles Tabla.




## Manipulación de un elemento encontrado en una control TreeView mediante programación (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
Esta página de ayuda explica cómo manipular los elementos de un control TreeView directamente o a través de las funciones de WLanguage.

Es posible:

- Manejar un elemento de una control TreeView,

- Inicialización de un elemento en una control TreeView,

- Selección de un elemento en una control TreeView,

- Recuperar el elemento seleccionado en el control TreeView,

- Usar las propiedades de WLanguage en un elemento de un Treeview control.

## Manipulación directa de los elementos de una control TreeView
<a name="manipulacion_directa_los_elementos_una_control_treeview_ELTTEXTE000161"></a>


### Funciones de WLanguage
<a name="funciones_wlanguage_ELTPARAGRAPHE000184"></a>En las funciones WLanguage para la gestión de los controles TreeView, el elemento de la control TreeView se gestiona a través de su ruta completo (desde la raíz de la control TreeView).. Esta ruta tiene el siguiente formato:


```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Los diferentes elementos que constituyen las cadenas de caracteres de la ruta están separados por caracteres TAB.
<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Operación directa
<a name="operacion_directa_ELTPARAGRAPHE000199"></a>

Para manipular un elemento de un control TreeView directamente (con las propiedades WLanguage por ejemplo), hay dos anotaciones disponibles:

- Operación realizada a través del índice del elemento:
	Todo lo que tienes que hacer es usar la siguiente notación: **&lt;control TreeView&gt; [ &lt;Índice de elemento&gt;]**.
	Atención: Los índices especificados sólo tienen en cuenta los elementos de Visible.
	**Ejemplo**: Modificando la Color del 5º elemento que aparece en una control TreeView:
	
	```wl
	TREE_MyTreeView[5].Color = LightRed
	```


- Operación realizada a través de la ruta completa del elemento:
	Todo lo que tienes que hacer es usar la siguiente notación: **&lt;control TreeView&gt; [ &lt;Elemento ruta&gt;]**.
	La ruta del elemento es un string con el siguiente formato: 
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```
**Ejemplo**: Modificando la Color del elemento "Piña"....
	
	```wl
	TREE_MyTreeView["Recipe" + TAB + "Dessert" + TAB + "Pineapple"].Color = LightRed
	```

### Asociar un tooltip a un elemento
<a name="asociar_tooltip_elemento_ELTPARAGRAPHE000293"></a>

Para asociar un tooltip con un elemento en un Treeview control, puedes usar:

- las funciones de WLanguage utilizadas para añadir o modificar un elemento de un control TreeView ([&lt;Vista de árbol&gt;.add](../WDLang1/1000023422.md), [&lt;Insertar](../WDLang1/1000023761.md), [&lt;Modificar](../WDLang1/1000023763.md)),

- la [Sugerencia sobre herramientas](../Proprietes/2510005.md) property en el elemento deseado.


Si no se especifica el texto del elemento tooltip, el texto mostrado será el texto del tooltip para el control TreeView (si existe).

**Recordatorio**: Por defecto, se muestra el tooltip associated con los diferentes controles. Para especificar si la información sobre herramientas debe mostrarse o no, utilice [ShowToolTip](../WDLang1/3025014.md).

### Seleccionar el elemento de la fila &lt;Índice&gt;
<a name="seleccionar_elemento_fila__ELTPARAGRAPHE000351"></a>

Para seleccionar el elemento en fila &lt;Index&gt;, utilice: 

- la asignación directa: 
	
	```txt
	<TreeView Control> = Index
	```

	Si se visualiza la fila en el control TreeView, se selecciona el elemento correspondiente.

- la función [&lt;Vista de árbol&gt;.SelectPlus](../WDLang1/1000023770.md): 
	
	```txt
	<TreeView control>.SelectPlus(<Full element name>)
	```

### Posicionamiento sobre el último elemento de la control TreeView
<a name="posicionamiento_sobre_ultimo_elemento_control_treeview_ELTPARAGRAPHE000382"></a>

Para fijar la posición en el último elemento de la control TreeView, utilice la [Conde](../Proprietes/2510097.md) Property: 


```txt
<TreeView control> = <TreeView control>.Count
```

## Recuperar el elemento seleccionado en el control TreeView
<a name="recuperar_elemento_seleccionado_control_treeview_ELTTEXTE000392"></a>
Para recuperar: 

- el índice del elemento seleccionado, utiliza la lectura directa del elemento: 
	
	```txt
	Index = <TreeView Control>
	```


- la ruta completa del elemento seleccionado, utilice [&lt;TreeView&gt;.Select](../WDLang1/1000023767.md): 
	
	```txt
	<Path> = <TreeView Control>.Select()
	```


- el identificador del elemento seleccionado, utilice [&lt;Identificador](../WDLang1/1000023766.md).

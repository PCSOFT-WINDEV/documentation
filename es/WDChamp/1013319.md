


## Galería Image control Looper
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000309"></a>
La "Galería de imágenes" control Looper permite gestionar la visualización de un set de Image s sin programar. Las imágenes pueden tener diferentes tamaños y proporciones: el control Galería de imágenes se encarga de ofrecer una visualización armoniosa. 

La navegación entre imágenes se gestiona automáticamente. El control muestra las imágenes encontradas en un directorio, o provenientes de una base de datos o definidas mediante programación. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Galería en filas
<a name="galeria_filas_ELTTEXTE000333"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000034"></a>

Una galería de Image en filas propone la misma altura para cada Image del mismo fila . Las filas no necesariamente tienen la misma altura: esta altura se define automáticamente según las imágenes encontradas en la fila.

La galería en filas adapta automáticamente el ancho de cada fila para respetar las proporciones de Image.

Ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_r%E9p%E9t%E9e_galerie%20-%20HC%20N%B0002.gif)


Se recomienda una galería en las filas control para mostrar sólo imágenes (sin texto abajo por ejemplo). 


### Creación de una galería en filas
<a name="creacion_una_galeria_filas_ELTPARAGRAPHE000046"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para crear una galería Image en filas a partir de un control preestablecido : 

1. En la pestaña "Creación", en el grupo "Datos", despliegue "Looper" y seleccione "Galería en filas". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_Repetee_Galerie_creationWD%20-%20HC%20N%B0001.gif)


2. Haga clic en la ubicación en la que desea crear el control. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_Repetee_Galerie_creationWD%20-%20HC%20N%B0002.gif)
 La Galería en filas control incluye: 

	- a "Galería en filas" control Looper. 

	- un control Imagen. 

	- una control Estático utilizada para mostrar información. 




3. La Galería creada control puede ser personalizada. Ningún código es associated con los diferentes elementos de la Galería control. 




Observación: También puede crear un control Looper estándar y configurarlo para get una galería en filas. 


### Características de la galería en filas
<a name="caracteristicas_galeria_filas_ELTPARAGRAPHE000110"></a>

- **Control de referencia**: Una galería en filas incluye varias filas. Cada fila contiene al menos un reference control. Este reference control corresponde a un control Imagen, un control Imagen clicable o un control Miniatura. Este control reference se utiliza para definir un tamaño reference (un tamaño mínimo). 
	Este control de referencia se define en la pestaña "Detalles" de la ventana control Looper description. 

- **Ventana emergente de desplazamiento**: Es posible abrir automáticamente una ventana emergente al hacer clic en el reference control: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la ventana control Looper description, en la pestaña "Detalles", marque "Mostrar una popup al hacer clic".. 




- En tiempo de ejecución, la altura de cada fila se ajusta automáticamente para ajustarse al contenido mostrado.. 
	Atención: el modo de visualización de control Imagen es muy importante. 

	- Si la fuente Image es más pequeña que la reference control Imagen, el modo de visualización de la Image se aplica en tiempo de ejecución. 

	- Si la altura de la imagen de origen es mayor que su anchura en relación con el reference control Imagen, la altura del fila se incrementará.

## Galería en columnas
<a name="galeria_columnas_ELTTEXTE000369"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000145"></a>

Una galería de Image en columnas propone el mismo ancho para cada fila de la misma columna. En cada columna, la altura de fila puede variar: esta altura se define automáticamente en función de las imágenes visualizadas.

Ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_r%E9p%E9t%E9e_galerie%20-%20HC%20N%B0001.gif)



### Creación de una galería en columnas
<a name="creacion_una_galeria_columnas_ELTPARAGRAPHE000153"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para crear una galería Image en columnas a partir de un preset control: 

1. En la pestaña "Creación", en el grupo "Datos", despliegue "Looper" y seleccione "Galería en columnas". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_Repetee_Galerie_creationWD%20-%20HC%20N%B0001%201.gif)


2. Haga clic en la ubicación en la que desea crear el control. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Zone_Repetee_Galerie_creationWD%20-%20HC%20N%B0003.gif)
 La galería de columnas control incluye: 

	- a "Galería en columnas" control Looper. 

	- un control Imagen. 

	- una control Estático utilizada para mostrar información.  




3. La Galería creada control puede ser personalizada. Ningún código es associated con los diferentes elementos de la Galería control. 





### Características de la galería en columnas
<a name="caracteristicas_galeria_columnas_ELTPARAGRAPHE000213"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Configuración del número de columnas** (pestaña "Detalles" de la descripción del control): Tiene la posibilidad de personalizar el modo de visualización de las columnas de la galería.: 

	- Ancho de columnas variable: Si la anchura de la lanzadera permite una columna adicional, el número de columnas no cambia. Sólo se modifica el ancho de las columnas. 

	- Ancho y número de columnas variables: Si la anchura de la cinta transportadora control permite una columna adicional, ésta se añadirá. Antes de agregar la columna, las filas se estiran horizontalmente. 

	- Número de columnas variable (a la izquierda):  Si la anchura de la cinta transportadora control permite una columna adicional, ésta se añadirá. Las imágenes están alineadas a la izquierda. Antes de añadir la columna, puede aparecer un área vacía a la derecha. 

	- Número de columnas variable (centradas):  Si la anchura de la cinta transportadora control permite una columna adicional, ésta se añadirá. Antes de añadir la columna, las columnas están centradas y puede aparecer un área vacía a cada lado de las columnas centradas.

	- Ancho y número de columnas fijos (a la izquierda): El número de columnas y el ancho de las columnas no se modifican. Las columnas están alineadas a la izquierda. 

	- Ancho y número de columnas fijos (centradas): El número de columnas y el ancho de las columnas no se modifican. Las columnas están centradas.




- **Control de referencia**: Una galería en columnas incluye varias filas. Cada fila contiene al menos un control Imagen. La "Galería en columnas" control Looper está asociada a este control Imagen (o control Imagen con enlace o control Miniatura). Este control se utiliza para definir un tamaño de reference. 
	Este control de referencia se define en la pestaña "Detalles" de la ventana control Looper description. 

- **Ventana emergente de desplazamiento**: Es posible abrir automáticamente una ventana emergente al hacer clic en el reference control: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la ventana control Looper description, en la pestaña "Detalles", marque "Mostrar una popup al hacer clic".. 




- **En tiempo de ejecución** , la altura de cada fila se ajusta automáticamente para adaptarse al contenido que se muestra. 
	Atención: el modo de visualización y las opciones de anclaje de control Imagen son muy importantes. 

	- Si el origen Image es menor que el reference control Imagen, las dimensiones del reference control se aplican al Image en tiempo de ejecución. 

	- Si el ancho de la imagen fuente es mayor que su altura en relación con la reference control Imagen, el ancho de la columna se incrementará.

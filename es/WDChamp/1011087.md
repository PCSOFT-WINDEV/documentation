<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
Una función de búsqueda está disponible automáticamente cuando se previsualiza una reporte, a través del panel "Buscar. Esta búsqueda se realiza en todas las páginas que se muestran en el visor de reporte.

Esta función está disponible para las impresiones realizadas con editor de reportes (o con "Reports & Queries") y para las impresiones programadas (realizadas con las funciones WLanguage).

Recordatorio: El visor reporte está disponible: 

- en la editor de reportes de WINDEV, WEBDEV, WINDEV Mobile. 

- en "Reports and Queries".

- al imprimir informes desde una aplicación WINDEV.

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000119"></a>
Para iniciar la búsqueda desde el visor de reporte: 

1. Mostrar el panel "Encontrar" del visor de reporte: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Apercu_Volet_Rechercher.gif)


2. Escriba las características de la búsqueda:

	- El texto buscado.

	- Si se debe buscar la palabra completa ("Whole word only").

	- Si la búsqueda debe distinguir entre mayúsculas y minúsculas (opción"Sensible a mayúsculas y minúsculas").




3. Haga clic en la lupa para iniciar la búsqueda. El elemento encontrado se ilumina en verde.
	Los botones de flecha "Siguiente" y "Anterior" se usan para moverse entre los elementos encontrados.




**Observación**

- Si el visor de reporte muestra un archivo RTF con imágenes, la búsqueda no funcionará.

- [iParameterViewer](../WDLang5/1000024634.md) permite desactivar el panel de búsqueda si es necesario (constante *iButtonSearch*).


## Variante de formato de texto
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000078"></a>
Puede personalizar los atributos del texto, como el espaciado entre caracteres, el interlineado y el tamaño de la fuente en una ventana.
Estos ajustes se pueden definir manualmente en la ventana de descripción del control Estático.
En la versión 28, una nueva ventana de previsualización muestra y sugiere combinaciones predefinidas de estos atributos.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000102"></a>
Para acceder a las variantes de formato de un texto en un control Estático: 

1. Seleccione el control Estático. 

2. Abra el menú contextual del control y seleccione "Variante de formato".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Variantes_mise_en_forme%20-%20HC%20N%B0002.gif&type=thumb)


3. La ventana que aparece muestra las variantes de formato disponibles. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Variantes_mise_en_forme%20-%20HC%20N%B0001.gif&type=thumb)
Esta ventana muestra una vista previa de las diferentes variantes aplicadas al texto. 

4. Seleccione la variante deseada y presione la tecla Entrar para confirmar. La variante seleccionada se aplica automáticamente al texto. 




Observaciones: 

- En WEBDEV, esta opción también está disponible para los controles Área de texto enriquecido. 

- Puede editar y navegar entre las variantes de formato sugeridas utilizando el teclado.

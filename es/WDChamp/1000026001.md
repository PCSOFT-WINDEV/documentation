## Resumen de control Línea de tiempo
<a name="resumen_control_linea_tiempo_ELTTEXTE000271"></a>
La control Línea de tiempo se utiliza para representar cronologías de eventos cercanos que pueden ser paralelos.. 

Este control se utiliza por ejemplo: 

- para realizar una edición de vídeo. 

- para representar eventos en un robot. 

- para realizar una animación. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación de una control Línea de tiempo
<a name="creacion_una_control_linea_tiempo_ELTTEXTE000295"></a>
Para crear una control Línea de tiempo:

1. En la pestaña "Creación", en el grupo "Controles gráficos", despliegue "Tiempo" y seleccione "Línea de tiempo".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor. 




Observación: Las dimensiones del control que se creó se optimizan para ocupar el espacio disponible en la ubicación indicada. Si el tamaño del control no es el indicado, pulse Ctrl+Z: el control volverá al tamaño predeterminado.

Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

## Características de control Línea de tiempo
<a name="caracteristicas_control_linea_tiempo_ELTTEXTE000323"></a>


### Opciones de control
<a name="opciones_control_ELTPARAGRAPHE000050"></a>

El control Línea de tiempo puede ser configurado en el editor a través de la ventana description (pestaña "Detalles").

Los parámetros de la control Línea de tiempo son los siguientes: 

- **Unidad de eventos**: Define la unidad utilizada para la control Línea de tiempo: segundo, milisegundo o microsegundo. Este parámetro se utiliza para definir:

	- la unidad de visualización utilizada por la control en tiempo de ejecución. 

	- la unidad utilizada para los diferentes valores en la ventana de description.




- **Máscara de visualización**: Permite definir el formato de la hora visualizada en la regla.

- **Rango visible**: Número de segundos visibles en el TimeLine.  

- **Rango total**: Número de segundos visualizados por el TimeLine. 

- **Características de los eventos**

	- **Superposición**: Gestionar la superposición de eventos. Los eventos se pueden mostrar uno al lado del otro o ligeramente desplazados.. 

	- **Granularidad del evento**: Precisión para la visualización de la control Línea de tiempo. Es posible definir: 

		- la granularidad al mover una unidad de tiempo de Event( 1 por Default). 

		- la granularidad en cuanto a la duración de un evento, lo que significa la duración mínima de una unidad de tiempo de Event( 1 por Default).




- **Características del cabezal de reproducción**: El cabezal de reproducción permite ver la posición en la control Línea de tiempo. Las opciones del cabezal de reproducción son las siguientes: 

	- **Mostrar cabezal**: El cabezal de reproducción se mostrará en el control. En este caso, puede ser modificado (o no) por el usuario. 

	- **Cabezal desplazable**: Si se selecciona esta opción, el usuario podrá mover el cabezal de reproducción en la control Línea de tiempo con el ratón. 




- **Permitir seleccionar períodos**: Permite al usuario seleccionar un período con el ratón. 

- **Permitir introducir un evento directamente**: Permite al usuario añadir un Event directamente. Para ello, haga clic en la control Línea de tiempo para crear la Event e introduzca su título con el teclado. 

- **Posición de la barra de tiempo**: La barra de tiempo se puede mostrar en la parte superior o inferior de la pantalla de control Línea de tiempo. 

- **Barra despl. horizontal**: La barra de desplazamiento horizontal se utiliza para desplazar el contenido de control Línea de tiempo. Puede ser automatic, siempre mostrado o nunca. 




**Las opciones de edición mediante el ratón son las siguientes**: 

- **Modificar duración**. Puede permitir que los usuarios cambien la duración de un Event. 

- **Mover**. Puede permitir a los usuarios mover un Event. 

- **Permitir al usuario cambiar de pista**. Puede permitir que los usuarios cambien la pista de la Event. 

- **Permitir al usuario superponer elementos**. Si esta opción está seleccionada, al mover eventos, el usuario podrá hacer que se superpongan.. En este caso, se tendrán en cuenta las opciones definidas para la superposición de eventos (compensación de uno al lado del otro).. 




**Las opciones de dimensión de las áreas son las siguientes**:

- **Ancho del encabezado de las pistas**: 

	- según la leyenda de la pista. 

	- fijo. 




- **Maltura mínima de una pista**.

### Menú contextual propuesto automáticamente
<a name="menu_contextual_propuesto_automaticamente_ELTPARAGRAPHE000165"></a>

El control Línea de tiempo puede mostrar un menú emergente específico que propone al usuario las características del automatic. Este menú es propuesto por las AAF (Funcionalidades Automáticas de la Aplicación). 

En cuanto se encuentra al menos una pista en la control Línea de tiempo, este menú permite al usuario: 

- Realiza un zoom en la control (acercar o alejar la selección). 

- Añadir, eliminar o modificar una Event al seleccionar un periodo de tiempo. 
	Observaciones: Si el control es relacionado a una fuente de datos, la eliminación realizada en el control se realiza al mismo tiempo en la fuente de datos del relacionado.

- Copiar el contenido de un Event. 




Para más detalles sobre las características de este menú contextual, véase [Características de la automatic de la control Línea de tiempo](../WDChamp/1000026006.md).

### Personalizar el área de Event
<a name="personalizar_area_event_ELTPARAGRAPHE000185"></a>

En el control Línea de tiempo, el Event se muestra en un área específica. Usted tiene la capacidad de traducir y personalizar esta área: el color del título, el color del fondo de la Event, ...

**To personalizar la visualización de eventos**: 

1. Abre la ventana de control Línea de tiempo description. 

2. En la pestaña "General", seleccione la ventana interna utilizada para los eventos: 

	- "Ninguno": se utilizará la pantalla de Default. 

	- "Ventana predefinida": se añade automáticamente una ventana de preajuste al proyecto. Por Default, esta ventana se llama "IW_WinDevViewAPT.wdw". 

	- "Dibujo simplificado": la Event se muestra sin título. 




3. Si se ha seleccionado "Ventana preestablecida", la ventana interna estará disponible en el explorador del proyecto. Se puede modificar en el editor de ventanas. 
	**Atención**: esta ventana interna sólo se puede utilizar para personalizar el dibujo relativo a la visualización de eventos. Sólo se ejecuta el código de las "Declaraciones globales" Event. No se realizan otros eventos de WLanguage. Por lo tanto, no puede añadir botones con un código de clic específico. 




**Observaciones**: 

- También tiene la posibilidad de utilizar cualquier ventana interna de su proyecto.. 

- El aspecto de los eventos propuestos por defecto puede ser configurado seleccionando "Evento" en la pestaña "Estilo" de la ventana de control de description.

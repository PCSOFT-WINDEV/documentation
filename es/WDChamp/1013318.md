


## Manipular controles List Box mediante programación (sintaxis con prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
WINDEV, WEBDEV y WINDEV Mobile permiten manipular controles List Box mediante programación. Para ello, utilice la variable del control List Box en el código. Esta variable es de tipo numérico.

La variable del control List Box:

- corresponde al nombre del control List Box.

- se inicializa con el índice del elemento seleccionado en el control.




Para manipular un elemento del control List Box, utilice la siguiente notación: 


```txt
<List Box control> [<Element index>]
```


**Observación**: Existen varias funciones WLanguage específicas para los controles List Box. Para obtener más información, consulte [funciones de gestión de controles List Box](../WDLang1/1000017419.md).

## Inicializar un control Combo Box
<a name="inicializar_control_combo_box_ELTTEXTE000148"></a>


### Inicializar un elemento
<a name="inicializar_elemento_ELTPARAGRAPHE000171"></a>

Para inicializar un elemento en un control List Box, utilice la función [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md).

**Observación**: Para modificar un elemento en un control List Box, utilice la siguiente sintaxis: 


```txt
<List Box control>[Index] = <Value>
```
Por ejemplo: 

```wl
// Add elements
LIST_LIST1.Add("MOORE")
LIST_LIST1.Add(Cust_Name)
// Modify elements
LIST_LIST1[3] = "MOORE"
```

<a name="NOTE2_2"></a>


### Rellenar un control List Box
<a name="rellenar_control_list_box_ELTPARAGRAPHE000197"></a>Para inicializar un control Combo Box, simplemente utilice un bucle de iteración para inicializar todos los elementos del control.

```txt
Index is int
FOR Index = <Start position> TO <End position>
	<List Box control>.Add(<Value>)
END
```
Ejemplo:

```wl
i is int
FOR i = 1 TO 52
	LIST_LIST1.Add(NAME[i])
END
```

## Seleccionar un elemento en un control List Box
<a name="seleccionar_elemento_control_list_box_ELTTEXTE000215"></a>


### Seleccionar una fila
<a name="seleccionar_una_fila_ELTPARAGRAPHE000238"></a>

Para seleccionar un elemento en la fila &lt;Index&gt; utilice:

- la asignación directa: 
	
	```txt
	<List Box control> = Index
	```


- la función [&lt;Caja de lista&gt;.SelectPlus](../WDLang1/1000017414.md): 
	
	```txt
	<List Box control>.SelectPlus(<Index>)
	```




<a name="NOTE3_2"></a>


### Fijar la posición en el último elemento
<a name="fijar_posicion_ultimo_elemento_ELTPARAGRAPHE000265"></a>

Para fijar la posición en el último elemento del control List Box, utilice:

- la propiedad [Count](../WDLang1/3025009.md): 
	
	```txt
	<List Box control>.Count
	```


- la función [&lt;Caja de lista&gt;.SelectPlus](../WDLang1/1000017414.md) con la propiedad [Count](../WDLang1/3025009.md): 
	
	```txt
	<List Box control>.SelectPlus(<List Box control>.Count)
	```


- la función [&lt;List Caja&gt;.Cuenta](../WDLang1/1000017334.md): 
	
	```txt
	<List Box control>.Count()
	```

## Obtener el elemento seleccionado en un control List Box
<a name="obtener_elemento_seleccionado_control_list_box_ELTTEXTE000306"></a>


### Obtener el índice del elemento seleccionado
<a name="obtener_indice_del_elemento_seleccionado_ELTPARAGRAPHE000329"></a>

Para obtener el índice del elemento seleccionado, utilice una de las siguientes sintaxis:

- leer directamente el elemento: 
	
	```txt
	Index = <List Box control>
	```


- la función [&lt;Seleccione](../WDLang1/1000017336.md): 
	
	```txt
	Index = <List Box control>.Select()
	```





**Observación**: En los controles List Box de selección múltiple, la función [&lt;Seleccione](../WDLang1/1000017336.md) permite obtener los elementos seleccionados.

```wl
// Retrieve the elements of a multi-selection List Box control
Rank is int = 0  // selection number
Index is int = 0  // index of the selected element
LOOP
	Rank = Rank + 1
	Index = LIST_LIST1.Select(Rank)
	IF Index = - 1 THEN BREAK
END
```

<a name="NOTE4_2"></a>


### Obtener el valor del elemento seleccionado
<a name="obtener_valor_del_elemento_seleccionado_ELTPARAGRAPHE000370"></a>

Para obtener el valor del elemento seleccionado, utilice una de las siguientes sintaxis:

- **Sintaxis 1**:
	
	```txt
	Index is int = <List Box control>
	<Variable> = <List Box control>[Index]
	```


- **Sintaxis 2**:
	
	```txt
	<Variable> = <List Box control>[<List Box control>]
	```


- **Sintaxis 3**: Propiedad [DisplayedValue](../Proprietes/2510129.md) (en código Servidor únicamente)
	
	```txt
	Variable> = <List Box control>[Index].DisplayedValue
	```





**Atención**: El valor almacenado puede ser diferente del valor que muestra. Para obtener el valor almacenado correspondiente a una de las filas del control List Box, utilice la propiedad [StoredValue](../Proprietes/2510127.md).

## Recorrer los elementos de un control List Box
<a name="recorrer_los_elementos_control_list_box_ELTTEXTE000393"></a>
La instrucción **FOR EACH** realiza diferentes tipos de iteraciones en los controles List Box: 

- Recorrer los elementos del control. Se utiliza la siguiente sintaxis: 
	
	```txt
	FOR EACH ROW [<Index> [, <Counter>]] OF <List Box control>
	  ...
	END
	```
 donde: 

	- &lt;Index&gt;: Variable opcional de tipo Integer. Por cada iteración, contiene el índice de la fila actual.

	- &lt;Counter&gt;: Variable opcional de tipo Integer. Esta variable cuenta el número de iteraciones.




- Recorrer los elementos seleccionados. Se utiliza la siguiente sintaxis: 
	
	```txt
	FOR EACH SELECTED ROW [<Index>, [<Counter>]] OF <List Box control>
	    ...
	END
	```
 donde: 

	- &lt;Index&gt;: Variable de tipo Integer. Por cada iteración, contiene el índice de la fila actual seleccionada.

	- &lt;Counter&gt;: Variable opcional de tipo Integer. Esta variable cuenta el número de iteraciones (número de filas seleccionadas, por ejemplo).







Para obtener más información, consulte [Instrucción FOR EACH/FOR ALL (recorrer controles)](../Motscles/1510017.md).

## Propiedades específicas de los controles List Box
<a name="propiedades_especificas_los_controles_list_box_ELTTEXTE000439"></a>
Las siguientes propiedades son específicas de los controles List Box:  



|   |   |
| --- | --- |
| **Todos los tipos de controles List Box** |
| [HorizontalAlignment](../Proprietes/2510023.md) | Obtiene y establece la alineación horizontal de los elementos en el control List Box. |
| [LineHeight](../Proprietes/2510041.md) | Obtiene y establece la altura de las filas en un control List Box. |
| [Memory](../Proprietes/2510076.md) | Indica si el control List Box especificado se rellena mediante programación o si está basado en un archivo de datos. |
| [Count](../WDLang1/3025009.md) | Obtiene el número de filas de un control List Box. |
| [FillType](../Proprietes/1000017723.md) | Indica cómo se rellena un control (mediante programación, desde un archivo de datos o desde una variable). Se aplica a los controles Tabla, List Box, ListView, Combo Box y Looper. |
| [DisplayedValue](../Proprietes/2510129.md) | Obtiene el valor de la fila actual o el valor de una fila específica en el control List Box. |
| [StoredValue](../Proprietes/2510127.md) | Obtiene el valor almacenado actualmente al seleccionar una fila en el control List Box. |
| [Empty](../Proprietes/2510126.md) | Indica si un control List Box está vacío. |
| **Control List Box rellenado mediante programación únicamente** |
| [InitialContent](../Proprietes/2510021.md) | Obtiene el contenido inicial de un control List Box rellenado mediante programación. |
| [Sorted](../Proprietes/2510115.md) | Indica si un control List Box rellenado mediante programación está ordenado, u ordena el control. |
| **Control List Box basado en un archivo de datos únicamente** |
| [BrowsedFile](../Proprietes/2510040.md) | Obtiene o establece el archivo de datos o consulta utilizado para mostrar registros en los controles Combo Box. |
| [AutoBrowse](../Proprietes/2510093.md) | Indica si un control List Box se recorre de forma automática o mediante programación. |
| [DisplayedItem](../Proprietes/2510123.md) | Obtiene y establece el campo que se muestra en un control List Box. |
| [StoredItem](../Proprietes/2510110.md) | Obtiene y establece el campo almacenado de un control List Box. |
| [BrowsedItem](../Proprietes/2510111.md) | Obtiene y establece el campo utilizado para recorrer automáticamente los controles List Box. |


Para obtener la lista completa de las propiedades WLanguage que pueden utilizarse con un control List Box, consulte [Propiedades asociadas a los controles List Box.](../WDChamp/1013182.md)

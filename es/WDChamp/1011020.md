## Vista general del bloque de carrocería
<a name="vista_general_del_bloque_carroceria_ELTTEXTE000080"></a>
El bloque **Body** corresponde al bloque principal de reporte. Este bloque se encuentra necesariamente en cada reporte. Este bloque se imprime tantas veces como el número de registros a imprimir.

Para los informes gratuitos, el bloque **Body** puede repetirse tantas veces como sea necesario para llenar la página ("Repetir hasta el final de la página" marcado en la pestaña "UI" de las características del bloque).

**Observación**: El **Cuerpo** el bloque no puede borrarse. Sin embargo, este bloque puede hacerse invisible (en la ventana description del bloque o a través del [Visible](../Proprietes/2510138.md) Property).

Para más detalles sobre las diferentes opciones disponibles en este tipo de bloque, véase [Descripción de los bloques de reporte](../WDChamp/1011049.md).

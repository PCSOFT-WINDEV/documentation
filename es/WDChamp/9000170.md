


## Responsive Web Design: Gestión de los anclajes
			





<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000172"></a>
En "Responsive Web Design", el tamaño de los controles evoluciona en función del modo de creación de la Page: 

- **Primero el escritorio** modo: el ancho de los controles se reduce automáticamente cuando se reduce el ancho del navegador. 

- **Móvil Primero** modo: el ancho de los controles se incrementa automáticamente cuando se incrementa el ancho del navegador.


Por Default, el tamaño de los controles es proporcional al tamaño del navegador. 

Para definir el redimensionamiento de los controles en función del tamaño del navegador, WEBDEV propone el mecanismo de anclaje, para la Page y para los controles. 

**Observación**: No dude en utilizar el manipular del navegador de edición para ver el comportamiento de los controles en el editor al redimensionar el navegador.. En efecto, esta manipular le ayuda a diseñar correctamente su Page. 
Consejo: también puede hacer clic en un rango de resolución en la barra de resolución para ver directamente la página en este rango de resolución.

## Las opciones disponibles
<a name="las_opciones_disponibles_ELTTEXTE000196"></a>


### Anclaje a nivel Page: las opciones
<a name="anclaje_nivel_page_las_opciones_ELTPARAGRAPHE000033"></a>

Definir los anclajes a nivel Page: 

1. Seleccione "Anclar" en el menú contextual de Page. 

2. Aparece la ventana description de anclajes Page. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RWD%20-%20Ancrages%20-%20HC%20N%B0001.gif)





Las opciones disponibles son: 

- **Anclaje horizontal**: Las opciones de anclaje horizontal son: 

	- **Ancho fijo**: Esta opción se utiliza para fijar la anchura de Page. Se utiliza la anchura especificada en la pestaña "UI" de la ventana de la página description. 

	- **Ampliar o reducir como el navegador (o su contenedor)**: Mediante esta opción, Page se reduce o amplía en función del tamaño del navegador. 




- **Anchura mínima**: Esta opción permite definir la anchura mínima de Page en cada rango de resolución. Esta anchura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles).

- **Ancho máximo**: Esta opción permite definir la anchura máxima de Page en cada rango de resolución. Esta anchura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles). 

- **Anclaje vertical**: Las opciones de anclaje vertical son: 

	- **Alto fijo**: Esta opción se utiliza para fijar la altura de Page. Se utiliza la altura especificada en la pestaña "UI" de la ventana de la página description. 

	- **Adaptar al contenido y reajustar controles**: Esta opción permite definir la altura de Page en función del contenido de Page. Por ejemplo, para un control Looper con un Infinite número de filas, el control y el Page se expanden verticalmente según el número de filas.




- **Alto mínimo**: Esta opción permite definir la altura mínima de Page. Sólo está disponible si se ha seleccionado "Ajustar al contenido y tirar de otros controles".. Esta altura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles). 





### Anclaje de los mandos Page: las opciones
<a name="anclaje_los_mandos_page_las_opciones_ELTPARAGRAPHE000056"></a>

Para definir los anclajes a nivel de control: 

1. Seleccione "Anclaje" en el menú contextual de control. 

2. Aparece el control de anclajes de la ventana description. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RWD%20-%20Ancrages%20-%20HC%20N%B0002.gif&type=thumb)





Las opciones disponibles son: 

- **Posición**: 

	- Izquierda: El espacio a la izquierda de control (en relación al control anterior o al lado de Page) se mantiene proporcionalmente al modificar la anchura de Page. 

	- Derecho: El espacio a la derecha de control (en relación al control anterior o al lado de Page) se mantiene proporcionalmente al modificar la anchura de Page. 

	- Centro: Esta opción centra la control en la Page. Esta opción puede ser usada para un solo control. Si se encuentran dos controles en el mismo Line, el centrado se ignora. 




- **Anclaje horizontal**: Las opciones de anclaje horizontal son: 

	- **Ancho fijo**: Esta opción permite definir la anchura de control independientemente del tamaño de Page. 

	- **Adaptar al contenido y reajustar controles**: Esta opción se utiliza para reducir el ancho del control al mínimo para que su contenido sea Visible.

	- **Ampliar o reducir como el navegador (o su contenedor)** (opción Default): Esta opción se utiliza para mantener las proporciones existentes en el editor cuando se estira o reduce la Page. Este comportamiento es similar al del Page en modo zonificación. 




- **Anchura mínima**: Esta opción se utiliza para definir la anchura mínima en cada rango de resolución. Esta anchura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles).

- **Ancho máximo**: Esta opción se utiliza para definir la anchura máxima en cada rango de resolución. Esta anchura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles). 

- **Anclaje vertical**: Las opciones de anclaje vertical son: 

	- **Alto fijo**: Esta opción permite definir la altura de control independientemente del tamaño de Page.  

	- **Adaptar al contenido y reajustar controles**: Esta opción se utiliza para reducir la altura de control al mínimo para que su contenido sea Visible. 




- **Alto mínimo**: Esta opción se utiliza para definir la altura mínima de control. Sólo está disponible si se ha seleccionado "Ajustar al contenido y tirar de otros controles".. Esta altura puede dejarse sin definir, corresponder al tamaño en modo de edición o corresponder a un valor (en píxeles). 

- **Cuando el contenido es más grande que el control**: 

	- Horizontalmente: Esta opción permite definir el comportamiento cuando el contenido sobrepasa el límite de control. Usted puede: 

		- Ampliar la control y empujar la control s hacia la derecha. 

		- Truncar contenido. 

		- Habilitar una barra de desplazamiento que siempre será Visible. 

		- Mostrar una barra de desplazamiento sólo si es necesario. 




- Verticalmente: Esta opción permite definir el comportamiento cuando el contenido sobrepasa el límite de control. Usted puede: 

	- Ampliar la control y empujar la control s debajo de ella. 

	- Truncar contenido. 

	- Habilitar una barra de desplazamiento que siempre será Visible. 

	- Mostrar una barra de desplazamiento sólo si es necesario.

## Visualización de anclas en Responsive Web Design
<a name="visualizacion_anclas_responsive_web_design_ELTTEXTE000226"></a>
Puede elegir ver los anclajes en el editor: en la pestaña "Vista", en el grupo "Mostrar (todos los controles)", haga clic en "Anclajes". 

Los diferentes símbolos utilizados son los siguientes: 

- Anclaje de "anchura fija".: ->x (símbolo de versiones anteriores: ->). 

- "Ampliar o reducir como el navegador" ancla: sin símbolo (símbolo en versiones anteriores: o->).

- "Ajustar al contenido y tirar de otros controles" ancla: o<-

- Talla mínima alcanzada: ||<-

- Tamaño máximo alcanzado: ->||

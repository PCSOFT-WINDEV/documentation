## Presentación
<a name="presentacion_ELTTEXTE000168"></a>
<a name="Print_modes"></a>
Existen varios modos de impresión disponibles para una reporte. Estos diferentes modos se presentan a continuación en la tabla:


|   |   |   |   |
| --- | --- | --- | --- |
|   | [Impresión durante la prueba de reporte en el editor](../WDChamp/1011027.md) | [Imprimir una reporte desde el visor de reporte](../WDChamp/1011085.md)<br>(Solo WINDEV) | Imprimir una reporte mediante programación ([iDestination](../WDLang5/3046074.md) y [iPrintReport](../WDLang5/3046032.md)) |
| Impresión en papel | X | X | X |
| Vista previa de impresión<br>Visor de reportes | X | - | X (sólo WINDEV) |
| Vista previa en miniatura |   |   | X (sólo WINDEV) |
| Archivo HTML (con hoja de estilo) | X | X | X |
| Archivo HTML (sin hoja de estilo) |   |   | X |
| Archivo RTF | X | X | X |
| Archivo XML |   | X | X |
| Archivo PDF | X | X | X |
| Archivo de texto |   |   | X |
| Envío de un fax |   |   | X |
| Fichero PCL |   |   | X |
| Archivo XLS |   |   | X |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **WINDEV Mobile**: Al ejecutar una prueba en el editor, todas las opciones están disponibles. Sin embargo, cuando se imprime un reporte de forma programada en una aplicación iOS, Android o Windows Universal, sólo se puede imprimir en archivos PDF. **Observación**: Por Default, el modo de impresión del reporte se solicita automáticamente durante la prueba de cada reporte en una ventana específica. Si no quieres ver esta ventana durante la prueba de la reporte, marca la opción "No mostrar de nuevo" en la ventana de selección del modo de impresión. Para obtener más información, consulte [No volver a mostrar un mensaje](../Editeurs/2020011.md).

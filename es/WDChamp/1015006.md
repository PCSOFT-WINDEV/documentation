


## Control Tabla TreeView basado en una variable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000163"></a>
Un control Tabla TreeView basado en una variable es un control enlazado a una variable o un conjunto de variables.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000187"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Crear un control Tabla TreeView basado en una variable
<a name="crear_control_tabla_treeview_basado_una_variable_ELTPARAGRAPHE000031"></a>

Para crear un control Tabla TreeView basado en una variable:

1. En la pestaña "Creación", en el grupo "Datos", despliegue "Tabla y List Box" y seleccione "Tabla TreeView".

2. Haga clic en la ubicación en la que desea crear el control.

3. Vaya a la pestaña "Contenido" de la ventana de descripción del control Tabla TreeView. Seleccione el modo de relleno: "Mediante variable". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TableH_Variable_Contenu.gif&type=thumb)
 En esta pestaña puede definir el origen de los datos del control Tabla TreeView. Un control Tabla TreeView basado en una variable está enlazado a diferentes fuentes de datos que permiten mostrar registros anidados.

4. Si es necesario, defina los enlaces de las diferentes columnas del control Tabla TreeView.



<a name="NOTE2_2"></a>


### Definir el origen de datos del control Tabla TreeView
<a name="definir_origen_datos_del_control_tabla_treeview_ELTPARAGRAPHE000082"></a>

Para definir el origen de datos del control Tabla TreeView:

1. Haga clic en el botón "Agregar". La fuente "&lt;Ninguno&gt;" se agrega automáticamente.

2. Defina las características de la fuente:

	- Fuente: corresponde a la variable sobre la que se va a iterar (por ejemplo, variables avanzadas como Google, RSS, etc.).

	- Navegación: corresponde a la variable utilizada para recorrer la fuente.

	- Mostrar: corresponde a la variable que aparece en la primera columna del control Tabla TreeView.
			




3. Si es necesario, defina la fuente de la relación del nivel anterior. Por ejemplo, si el primer nivel recorre la variable de tipo array garrCustomer, el segundo puede recorrer la variable garrCustomer.Order. 

4. Defina tantas fuentes como sea necesario.




La primera columna del control Tabla TreeView muestra los datos de las fuentes seleccionadas.
<a name="NOTE2_3"></a>


### Definir los enlaces de las diferentes columnas del control Tabla TreeView basado en una variable
<a name="definir_los_enlaces_las_diferentes_columnas_del_control_tabla_treeview_basado_una_variable_ELTPARAGRAPHE000104"></a>

De forma predeterminada, la primera columna del control Tabla TreeView (es decir, la columna que representa la jerarquía) muestra los elementos especificados en las fuentes de datos (pestaña "Contenido", valor que se va a "Mostrar" de las diferentes fuentes de datos).

El control Tabla TreeView también puede mostrar otros datos en sus otras columnas.

Para definir los enlaces de las diferentes columnas: 

1. Si es necesario, cree una nueva columna (botón "Nuevo" en la ventana de descripción del control Tabla TreeView) o seleccione una columna existente.

2. Vaya a la pestaña "Enlace" y defina el enlace de datos deseado.
	Observación: El enlace de datos incluye una de las variables fuente del control Tabla TreeView. Si se modifica uno de los elementos que aparecen en el control Tabla TreeView, los cambios se reflejan automáticamente en la variable enlazada.

3. Valide.

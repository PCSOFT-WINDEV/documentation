## Creación de una etiqueta reporte
<a name="creacion_una_etiqueta_reporte_ELTTEXTE000176"></a>


### Creación
<a name="creacion_ELTPARAGRAPHE000036"></a>

1. Para crear una "etiqueta" reporte, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido: 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Reporte", y luego en "Reporte".

	- El asistente de creación de reportes se abre. En la asistente, seleccione "Etiqueta".




2. En esta asistente, debe especificar:

	- la fuente de datos de reporte (fichero de datos, consulta, ...). Para obtener más información, consulte [Fuente de datos Reporte](../WDChamp/1011055.md).

	- la información específica para la creación de etiquetas.






<a name="NOTE2_2"></a>


### Información específica para la creación de etiquetas
<a name="informacion_especifica_para_creacion_etiquetas_ELTPARAGRAPHE000055"></a>

La información específica para la creación de etiquetas es la siguiente:

- el número de copias idénticas que se imprimirán para cada etiqueta.

- el formato de las etiquetas y/o el formato de la hoja de etiquetas. Este formato puede corresponder a:

	- a **formato personalizado**:

		1. Seleccione el elemento "&lt;Custom&gt;" en el cuadro combinado "Make".

		2. Especifique el formato de la hoja de etiquetas en la que se imprimirán las etiquetas.

		3. Especifique el formato de las etiquetas.

		4. Especifique el número de etiquetas que se imprimirán a lo ancho de Page.




- a **formato preestablecido**:

	1. Seleccione la marca de las etiquetas utilizadas en el cuadro combinado "Marca".

	2. Seleccione el modelo de etiquetas utilizado en el cuadro combinado "Modelo".

	3. Aparece la información específica de las etiquetas utilizadas (leyenda, tamaño de las etiquetas y número de etiquetas impresas a lo ancho de Page). 
						Observación: Seleccione la opción "Inch" para mostrar el tamaño de las etiquetas en pulgadas.



<a name="NOTE2_3"></a>


### Modificación de las características de las etiquetas
<a name="modificacion_las_caracteristicas_las_etiquetas_ELTPARAGRAPHE000079"></a>

Una vez creada la "Etiqueta" reporte, las características de las etiquetas pueden ser modificadas en cualquier momento.: en la pestaña "Modificación", en el grupo "Reporte", haga clic en "Formato y etiquetas".

## WLanguage propiedades associated con la etiqueta informes
<a name="wlanguage_propiedades_associated_con_etiqueta_informes_ELTTEXTE000212"></a>
Las siguientes propiedades son associated con los informes de "Etiqueta". Estas propiedades se utilizan para configurar dinámicamente (mediante programación) el formato y el modo de impresión de las etiquetas.

**Observación**: Estas propiedades sólo se pueden utilizar en los informes "Label.



|   |   |
| --- | --- |
| [HSpace](../Proprietes/2511067.md) | La función **HSpace** Property se utiliza para determinar y modificar el espaciado horizontal entre etiquetas en un reporte. Este espacio se definió al crear la etiqueta reporte. |
| [LabelHeight](../Proprietes/2511069.md) | El **LabelHeight** Property se utiliza para:<br><br>- Averiguar la altura de las etiquetas en una etiqueta reporte.<br><br>- Modificar la altura de las etiquetas en una etiqueta reporte.<br><br><br> |
| [LabelNumberToStart](../Proprietes/2511032.md) | El **LabelNumberToStart** Property se utiliza para:<br><br>- Averigüe el número de la etiqueta en la hoja de etiquetas desde la que se realizará la impresión..<br><br>- Modificar el número de la etiqueta desde la que se realizará la impresión en la hoja de etiquetas..<br><br><br> |
| [LabelWidth](../Proprietes/2511055.md) | El **LabelWidth** Property se utiliza para:<br><br>- Descubra el ancho de las etiquetas de una "Etiqueta" reporte.<br><br>- Modificar el ancho de las etiquetas en una "Etiqueta" reporte.<br><br><br> |
| [NbLabel](../Proprietes/2511050.md) | El **NbLabel** Property se utiliza para:<br><br>- Descubra el número de etiquetas impresas en todo el ancho de la página.<br><br>- Modificar el número de etiquetas impresas en todo el ancho de la página.<br><br><br> |
| [NbLabelCopy](../Proprietes/2511059.md) | El **NbLabelCopy** Property se utiliza para:<br><br>- get el número de copias de etiquetas idénticas.<br><br>- Cambiar el número de copias de etiquetas idénticas.<br><br><br> |
| [VSpace](../Proprietes/2511065.md) | El **VSpace** Property se utiliza para get y cambiar el espacio vertical entre las etiquetas en un reporte. Este espacio se definió al crear la etiqueta reporte. |

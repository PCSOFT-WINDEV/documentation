## Presentación del control TreeView
<a name="presentacion_del_control_treeview_ELTTEXTE000211"></a>


### Control de ventana o página dinámica
<a name="control_ventana_pagina_dinamica_ELTPARAGRAPHE000011"></a>

El control TreeView es una estructura jerárquica que permite simplificar la visualización de información representada en forma de vista de árbol.

Por lo general, el control TreeView se utiliza para representar la estructura jerárquica de un disco, directorio, etc.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Crear un control TreeView
<a name="crear_control_treeview_ELTTEXTE000235"></a>
Para crear un control TreeView: 

1. En la pestaña "Creación", en el grupo "Datos", haga clic en "TreeView".

2. Haga clic en la ventana o página en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En la pestaña "Estilo" se pueden definir diferentes características, por ejemplo: 

- color de las filas pares e impares.

- imagen de la fila sobrevolada.

- imagen de la fila seleccionada.

- imagen y configuración de los botones "+" y "-".

- estilo de la marca de verificación.



<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Menú contextual propuesto automáticamente
<a name="menu_contextual_propuesto_automaticamente_ELTPARAGRAPHE000060"></a>

El control TreeView puede mostrar un menú contextual específico que propone funcionalidades automáticas a los usuarios. Este menú es propuesto por las AAF (Funcionalidades Automáticas de la Aplicación). 

Este menú permite al usuario: 

- copiar el contenido de la fila o columna. 

- expandir o contraer todo: Esta opción permite expandir o contraer todos los elementos de la rama actual. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) exportar el contenido del control TreeView a Excel, Word, etc. 




Para obtener más información, consulte [Menú contextual de controles TreeView (AAF)](../WDChamp/1013273.md).

## Vocabulario asociado al control TreeView
<a name="vocabulario_asociado_control_treeview_ELTTEXTE000271"></a>
Para manipular un control TreeView, se deben conocer algunos conceptos: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)


| Nombre | Definición |
| --- | --- |
| Rama | Sección del control TreeView que define una ruta:<br><br>- de la raíz a una hoja<br><br>- de un nodo a otro nodo<br><br>- de un nodo a una hoja<br><br>- de la raíz a un nodo.<br><br><br> |
| Elemento | Nombre genérico que corresponde al contenido de un control TreeView. La raíz, el nodo y la hoja son elementos de un control TreeView |
| Hoja | Último elemento de la estructura jerárquica: no hay ningún nivel inferior. |
| Nodo | Nivel intermedio, puede tener niveles superiores (llamados "Padre") y niveles inferiores (llamados "Hijo"). <br>**Observación**: La raíz o las hojas corresponden a nodos específicos. |
| Raíz | Primer elemento del control TreeView. Toda la jerarquía se puede expandir a partir de este elemento. |

## Modos para rellenar un control TreeView
<a name="modos_para_rellenar_control_treeview_ELTTEXTE000295"></a>
Un control TreeView se puede rellenar: 

- mediante programación a través de diferentes eventos y [Funciones de gestión de controles TreeView](../WDLang1/3018025.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) mediante un enlace de datos. Para obtener más información, consulte [Control TreeView vinculado a un archivo de datos](../WDChamp/1013126.md).

- con datos de una variable.

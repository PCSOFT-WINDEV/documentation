<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000081"></a>
En algunos casos, la Image de fondo de una ventana puede contener un texto (el nombre de la aplicación, una etiqueta específica, etc.).

Si la aplicación está destinada a ser utilizada en diferentes idiomas, debe definirse una Image específica para cada idioma.

WINDEV y WINDEV Mobile le permiten tener en cuenta fácilmente estas diferentes imágenes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestión multilingüe
<a name="gestion_multilingue_ELTTEXTE000105"></a>
Se puede mostrar un fondo diferente Image de acuerdo con el lenguaje de ejecución del programa.

Para definir un fondo multilingüe Image: 

1. Abre la ventana description.

2. Mostrar la pestaña "Image".

3. Haga clic en el icono de ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) a la derecha del nombre de Image y seleccione "Multilingüe".

4. Elija la imagen apropiada para cada idioma de la ventana.
	**Atención**: Las características de visualización introducidas en esta ventana se aplicarán a todos los archivos de Image.

5. Valide. El texto "Valor Multilingüe" aparece en el campo de la imagen de fondo.

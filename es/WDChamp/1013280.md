## Presentación
<a name="presentacion_ELTTEXTE000186"></a>


### Control de página dinámica
<a name="control_pagina_dinamica_ELTPARAGRAPHE000011"></a>

La "Página Interna" control se utiliza para incluir una página (y su código) en un control de otra página.

En tiempo de ejecución, la página a fusionar ("Página interna") se fusionará dinámicamente con la página principal.

La "página interna" control le permite compartir dinámicamente la misma sección de la interfaz en uno o más sitios. La página a fusionar (página interna) también puede provenir de un componente.

**Atención: No se deben confundir los términos "Control Página interna" con "Página interna": la "página interna" control es un tipo de control mientras que la "página interna" es un tipo de página.**

## Creando una página interna y asociándola a un control
<a name="creando_una_pagina_interna_asociandola_control_ELTTEXTE000210"></a>


### Crear una página interna
<a name="crear_una_pagina_interna_ELTPARAGRAPHE000026"></a>

Para crear una página interna:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. Aparece la ventana de creación: haga clic en "Página", y luego en "Página". El asistente de creación de páginas se abre automáticamente.

2. Haga clic en "Página interna" y seleccione "En blanco".

3. Validar para crear la página interna.




Una página interna corresponde a un tipo específico de página. La página interna no contiene ningún Border, ninguna barra de título y ningún fondo Image. Se pretende que se muestre en una "página interna" control.

Este tipo de página también se denomina "Página de origen".
<a name="NOTE2_2"></a>


### Creando una "página interna" control
<a name="creando_una_pagina_interna_control_ELTPARAGRAPHE000043"></a>

Para crear una "página interna" control: 

1. En la pestaña "Creación", en el grupo "Contenedores", haga clic en "Página interna".

2. Haga clic en la página de destino, en la ubicación en la que desea crear el control. El control aparece en el editor.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual. Esto le permite seleccionar la página interna que se mostrará en la "Página Interna" control al inicializar la página objetivo.

## Observaciones
<a name="observaciones_ELTTEXTE000240"></a>


### Diferencias entre una página interna control y una página modelo
<a name="diferencias_entre_una_pagina_interna_control_una_pagina_modelo_ELTPARAGRAPHE000069"></a>

- Se recomienda usar una "página interna" control para el trabajo en equipo. Modificar una página interna no causa que todas las páginas que la usan se actualización (a diferencia de las plantillas).

- Se recomienda usar una "página interna" control si se deben agrupar los control s a mostrar en la página de destino. Las plantillas se utilizan para definir el esquema de la página. Con las plantillas, los controles que se encuentran en la modelo se pueden utilizar en cualquier parte de la página.



<a name="NOTE3_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000077"></a>

- No se puede utilizar ninguna página interna procedente de un componente en una página estática.. Sólo se permiten las páginas internas del proyecto.



<a name="NOTE3_3"></a>


### Refactorización
<a name="refactorizacion_ELTPARAGRAPHE000087"></a>

WEBDEV le permite crear automáticamente una página interna y una página interna control a partir de una selección de control s. 

1. Seleccione los controles que se deben utilizar en la página interna. 

2. En la pestaña "Modificación", en el grupo "Transformaciones", despliegue "Refactorización y permutaciones" y seleccione "Crear una página interna a partir de la selección".



<a name="NOTE3_4"></a>


### Creación de una página interna a partir de una página (y viceversa)
<a name="creacion_una_pagina_interna_partir_una_pagina_viceversa_ELTPARAGRAPHE000109"></a>

WEBDEV permite crear una página interna a partir de una página (y a la inversa).

Para crear una página interna a partir de la página actual, en la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar página como página interna".

Para crear una página a partir de la página interna actual, en la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar página interna como página".

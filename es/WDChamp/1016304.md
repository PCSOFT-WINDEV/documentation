
## Manejar el día en un control Calendar mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000136"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten manipular a través de la programación:

- A control Calendario. Para obtener más información, consulte [Manejo de una control Calendario por programación](../WDChamp/1016301.md).

- Un día en un control Calendario.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000160"></a>


### Programación
<a name="programacion_ELTPARAGRAPHE000051"></a>

Para manipular un día que se encuentra en un control Calendario, utilice la siguiente anotación:


```txt
<Name of Calendar control>[<Date to handle>]
```


Por ejemplo:


```wl
// Displays the date on a red background
CAL_Calendar1["20120701"].BackgroundColor = LightRed
```


En un control Calendario se pueden utilizar varias propiedades para manipular al día.

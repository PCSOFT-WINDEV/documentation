
## Validar una página de un control que no tiene acción en el servidor (control List Box, Combo Box, etc.)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
Por Default, la validación de un Page debe realizarse mediante un botón "Enviar los valores de los controles al servidor (submit)".. 

Para validar una Page al seleccionar un elemento en un Cuadro de lista, Combo Box, Casilla de verificación o control Botón de opción, utilice [PageSubmit](../WDLang2/3058020.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000135"></a>
[PageSubmit](../WDLang2/3058020.md) se utiliza para validar una página a través de un botón Visible u oculto cuyo código de clic del servidor se ejecuta. Los siguientes eventos se ejecutan cuando se llama a [PageSubmit](../WDLang2/3058020.md):

- Envío de los datos al servidor.

- El servidor hace clic en el código del botón.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000159"></a>
Para validar una página de una control sin acción (List Box, Combo Box, control Casilla de verificación, etc.): 

1. Crear un nuevo botón en Page: este botón dinámico puede tener cualquier tipo, puede ser Visible o no. Este botón permite iniciar la validación de la Page. Sólo se ejecutará el código de clic del servidor de este botón.

2. En el código de modificación de la control sin acción, utilice [PageSubmit](../WDLang2/3058020.md) y especifique:

	- la Page para validar si es necesario.

	- el nombre del botón utilizado para la validación (creado en el paso 1).

	- el fotograma que contiene el resultado de la acción del botón (por ejemplo, para un conjunto de fotogramas).







<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000183"></a>
La selección de un elemento en la control List Box llamado LIST_Continent se utiliza para mostrar el elemento seleccionado en un cuadro de diálogo.

El código de la control List Box es el siguiente:


```wl
// -- Initialization of LIST_CONTINENT
ListAdd(LIST_CONTINENT, "Africa" + CR + "America" + CR + "Asia" + CR + "Europe" + CR + ...
				"Oceania")

// -- Modification and exit (onchange) from LIST_CONTINENT
// Validates the page and starts running the server click code of BTN_Button1
PageSubmit("", BTN_Button1)
```


Código de BTN_Button1:


```wl
// -- Click on BTN_Bouton1 (server code)
Info("Selection: " + LIST_CONTINENT[LIST_CONTINENT])
```

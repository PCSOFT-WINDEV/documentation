
## Manejo de una Control deslizante mediante programación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000149"></a>
WINDEV y WINDEV Mobile le permiten manipular a Control deslizante mediante la programación. Para hacerlo, usa la Variable de la Control deslizante en el código.

La Variable de la Control deslizante:

- corresponde al nombre del control deslizante.

- se inicializa con la posición del cursor del control deslizante.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recuperación de la posición del cursor
<a name="recuperacion_posicion_del_cursor_ELTTEXTE000173"></a>
Para recuperar la posición de un control deslizante, utilice la siguiente sintaxis:


```txt
<Position> = <Slider Name>
```


&lt;Posición&gt; es un entero Variable

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificar la posición del cursor
<a name="modificar_posicion_del_cursor_ELTTEXTE000197"></a>
Para modificar la posición de un control deslizante, utilice la siguiente sintaxis:


```txt
<Slider Name> = <Value>
```


&lt;Valor&gt; es un valor incluido entre el valor mínimo y el valor máximo del control deslizante.. Estos valores se han definido en la ventana de control description o en el código con las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).

**Observación**: También puede utilizar la [Valor](../Proprietes/2510130.md) Property.

## Propiedades específicas de la gestión de un deslizador
<a name="propiedades_especificas_gestion_deslizador_ELTTEXTE000221"></a>
Las siguientes propiedades se utilizan para gestionar las características de un deslizador por programación.


|   |   |
| --- | --- |
| [BackgroundImage](../Proprietes/2510056.md) | Se utiliza para averiguar y modificar el fondo Image de un deslizador estándar. |
| [Image](../Proprietes/2510034.md) | Se utiliza para encontrar y modificar la Image de un cursor en un deslizador estándar. |
| [ImageState](../Proprietes/2510055.md) | Se utiliza para averiguar y modificar el número de dibujos que se encuentran en la Image associated con el cursor Image en un deslizador estándar. |
| [MaxValue](../Proprietes/2510009.md) | Permite conocer y modificar el límite superior de un control deslizante.. |
| [MinValue](../Proprietes/2510008.md) | Se utiliza para averiguar y modificar el límite inferior de un control deslizante.. |
| [Vertical](../Proprietes/2510133.md) | Permite saber si el deslizador estándar es horizontal o vertical. |


Para obtener una lista completa de las propiedades de WLanguage que pueden utilizarse con una Control deslizante, consulte [Propiedades associated con controles Control deslizante](../WDChamp/1013233.md).

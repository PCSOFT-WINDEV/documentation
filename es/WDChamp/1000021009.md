
## Las cartas de la"Cascada
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000143"></a>
El gráfico "Cascada" es un gráfico de barras utilizado para representar las adiciones positivas o negativas de varias categorías. 

Utilizado principalmente en finanzas, este gráfico se utiliza, por ejemplo, para ver la evolución de un valor a lo largo del tiempo o para ver el impacto de los diferentes componentes en este valor.. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Características de una carta de cascada
<a name="caracteristicas_una_carta_cascada_ELTTEXTE000167"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_Waterfall.gif)


En este cuadro: 

**1.** Valores correspondientes a un total de. Un total corresponde a la suma de todas las categorías anteriores desde el principio de la tabla.. El valor se muestra si se marca "Mostrar valores" en la pestaña "Detalles" del control Gráfico description.

**2.** Valores positivos de una categoría. El valor se muestra si se marca "Mostrar valores" en la pestaña "Detalles" del control Gráfico description.

**3.** Valores negativos de una categoría. El valor se muestra si se marca "Mostrar valores" en la pestaña "Detalles" del control Gráfico description.

**4.** Valor que corresponde a un subtotal: un subtotal corresponde a la suma de todas las categorías anteriores del último total o subtotal. El valor se muestra si "Mostrar los valores" está marcado en la pestaña "Detalles" de la description de la control Gráfico.

**5.** **cuadrículas**: Para visualizar las retículas, seleccione "Cuadrícula" y "Cuadrícula" en la pestaña "General" del control description.

**6.** **Título del gráfico**: 
Las características del título del gráfico pueden definirse en la pestaña "Detalles" del control description ("Editar título del gráfico"). 
Puede definir el título del gráfico, su posición y su estilo.

**7.** **Título de los ejes**: 
El título de los ejes horizontales y verticales se muestra si el título se definió en la pestaña "General" de la control Gráfico description. Este parámetro también puede definirse mediante programación con [grAxisTitle](../WDLang3/3042057.md).

**8.** Leyenda del valor (categoría). 

**9. Barras de desplazamiento utilizadas para realizar un zoom**. La capacidad de acercar un gráfico se puede definir en la pestaña "Detalles" del control description. Esta opción sólo está disponible en WINDEV.

**Otras características**: 

- Los gráficos de la"Cascada" se pueden mostrar con una animación. Para animar este tipo de gráfico, basta con marcar "Activar animaciones" en la pestaña "Detalles" del control description. 

- Para personalizar los colores de las categorías, utilice [grWaterfallColor](../WDLang3/1000020957.md).

- Para definir los tipos de categorías, utilice [grWaterfallCategoryType](../WDLang3/1000020930.md).

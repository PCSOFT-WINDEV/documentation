
## Ejecución del botón de automatic (Ventana)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
Para forzar una acción en una ventana, todo lo que tienes que hacer es definir un botón de esta ventana en el modo de "ejecución de automatic".. El principio es muy simple, si no se ha utilizado ningún botón durante un período de tiempo determinado, el proceso del botón "Ejecución automática" se ejecutará automáticamente.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fenetre-btn_Auto.gif)


En tiempo de ejecución, la duración se reduce automáticamente. Cuando llega a cero, el código del botón se ejecuta automáticamente. Este mecanismo se detiene automáticamente en los siguientes casos:

- cuando el usuario hace clic en un botón.

- si el usuario modifica un control.

- si el usuario pulsa cualquier tecla del teclado.




**Observación**: El usuario tiene la posibilidad de añadir un timer en el botón. Esta es una de las características de la aplicación automatic (AAF). Para obtener más información, consulte [timer en el botón](../Editeurs/9500100.md).

## Implementación

### Definición de un botón de ejecución de automatic
<a name="definicion_boton_ejecucion_automatic_ELTPARAGRAPHE000031"></a>

Se puede definir un botón de ejecución de automatic:

- en la ventana description (pestaña "Detalles"): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Btn_automatique.gif)


- en la control Botón description (pestaña "Detalles"): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Btn_automatique_btn.gif)



Un mensaje indica si se han definido varios botones de ejecución de automatic en la ventana.

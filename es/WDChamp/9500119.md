<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
Una característica "multitáctil" es una técnica que permite al usuario interactuar con un dispositivo a través de varios puntos Contact (los dedos se utilizan en la mayoría de los casos). 

Una de las aplicaciones más comunes de la función multitáctil se utiliza para imágenes manipular. Al reducirse el tamaño de la pantalla de un Phone, a menudo es necesario realizar un zoom y/o moverse en un Image. 

Los gestos Multi-Touch le permiten acercar una imagen separando dos dedos, por ejemplo. 

WINDEV Mobile le permite manejar el multitáctil en las aplicaciones Android y iPhone/iPad a través de: 

- [Opciones especiales disponibles para los controles Image](../WDChamp/9500131.md).

- [Funciones WLanguage específicas](#NOTE2_1).

## Multitáctil: Funciones WLanguage específicas
<a name="multitactil_funciones_wlanguage_especificas_ELTTEXTE000135"></a>
Se pueden utilizar varias funciones de WLanguage para gestionar el multitáctil en los siguientes procesos de control: 

- Presionado/botón izquierdo abajo (Process opcional)

- Soltado/botón izquierdo arriba (Process opcional)

- Toque/Click

- Doble toque/Doble clic con el botón izquierdo (Process opcional)

- Muévete (Process opcional). 




Estas funciones son: 



|   |   |
| --- | --- |
| [GestureCurrentPointer](../WDLang1/1000019751.md) | Devuelve el índice del puntero (dedo o lápiz óptico) que activó la llamada a un Process específico de la función multitáctil en un control Imagen. |
| [GestureNbPointer](../WDLang1/1000019750.md) | Devuelve el número de punteros en Contact con la pantalla. |
| [GesturePosX](../WDLang1/1000019741.md) | Devuelve la posición horizontal del puntero (dedo o lápiz). |
| [GesturePosY](../WDLang1/1000019745.md) | Devuelve la posición vertical del puntero (dedo o lápiz). |

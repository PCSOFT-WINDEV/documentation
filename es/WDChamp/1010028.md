
## Ventanas modales
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
La ventana que abre otra ventana se llama **ventana padre** ; la ventana abierta por la ventana principal se llama **ventana secundaria**. Cuando se utilizan ventanas modales, una ventana padre sólo puede tener una ventana hija.

Este modo no permite utilizar otras ventanas.

**En ventanas modales:**

- las ventanas padre e hijo no pueden tomar el foco de entrada al mismo tiempo. La ventana hija es la única que puede estar en modo de edición.

- para poder editar en la ventana padre, la ventana hija debe estar cerrada.

- la ventana principal no puede cerrarse mientras se abra la ventana secundaria.

- una vez que se cierra la ventana secundaria, la ventana principal se convierte en la ventana actual.

- si se minimiza la ventana infantil, todo el proyecto se minimiza con el Icon de la ventana actual (o con el Icon del proyecto si la ventana no tiene Icon).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Abrir una ventana
<a name="abrir_una_ventana_ELTTEXTE000159"></a>
La función [Abrir](../WDLang1/3038035.md) se utiliza para abrir ventanas modales.

La última ventana abierta recibe el foco. Las otras ventanas no pueden ser editadas mientras la última ventana permanezca abierta.

La ventana secundaria siempre se muestra en la parte superior de la ventana principal.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Minimizar una ventana
<a name="minimizar_una_ventana_ELTTEXTE000183"></a>
Si se minimiza la ventana, se minimiza todo el proyecto.. El Icon utilizado es el Icon de la ventana actual (o el Icon del proyecto si la ventana no tiene Icon).

Una ventana puede ser minimizada:

- con la función [Iconizar](../WDLang1/3038015.md)

- con la función [WinSize](../WDLang1/3038020.md).




Una ventana puede ser restaurada:

- con la función [restaura](../WDLang1/3038042.md)

- con la función [WinSize](../WDLang1/3038020.md).




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Ventana actual
<a name="ventana_actual_ELTTEXTE000207"></a>
La ventana actual es la que gana el foco.  Su barra de título es Active y el código para obtener el foco se ha ejecutado en ella.

La ventana actual es siempre la última ventana abierta.

No se puede realizar ninguna entrada en una ventana que no sea la ventana actual.

<a name="NOTE5"></a>
<a name="NOTE5_1"></a>


## Cerrar una ventana
<a name="cerrar_una_ventana_ELTTEXTE000231"></a>
Las ventanas se pueden cerrar con [Cerrar](../WDLang1/3038018.md). Esta función utilizada sin especificar el nombre de la ventana a cerrar cerrará la última ventana que se abrió.

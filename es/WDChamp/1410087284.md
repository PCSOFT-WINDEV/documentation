## Propiedades disponibles en una página
<a name="propiedades_disponibles_una_pagina_ELTTEXTE000092"></a>
**Propiedades disponibles en una página**

Las siguientes propiedades son associated con los controles Flexbox:



|   |   |
| --- | --- |
| [Alias](../Proprietes/2510004.md) | La propiedad **Alias** permite: <br><br>- obtener y cambiar el alias de una ventana (WINDEV y WINDEV Mobile).<br><br>- obtener y cambiar el alias de una pestaña dinámica (WINDEV).<br><br>- obtener el nombre interno HTML/JavaScript de un control (WEBDEV).<br><br><br> |
| [BackgroundColor](../Proprietes/2510022.md) | La propiedad **BackgroundColor** permite establecer el color de fondo de un control, ventana, página o bloque de un reporte. |
| [BackgroundImage](../Proprietes/2510056.md) | La propiedad **..BackgroundImage** permite conocer y modificar la imagen de fondo de un elemento: ventana, página, control o reporte. |
| [BrowsedFile](../Proprietes/2510040.md) | **BrowsedFile** Property obtiene y establece la fuente de datos utilizada para mostrar registros en controles basados en un archivo de datos . |
| [BrowsedItem](../Proprietes/2510111.md) | El **BrowsedItem** Property se utiliza para:<br><br>- get el elemento utilizado para recorrer automáticamente los controles de Tabla, Looper, Cuadro de Lista, ListView y Combo Box, supercontroles e informes basados en archivos de datos.<br><br>- Cambia el elemento utilizado para recorrer automáticamente los controles de Tabla, Looper, Cuadro de Lista, ListView y Combo Box, los supercontroles y los informes basados en archivos de datos. A continuación, se reinicia la iteración.<br><br><br> |
| [ContextMenu](../Proprietes/2510077.md) | El **ContextMenu** Property se utiliza para:<br><br>- get el menú contextual personalizado de una control, ventana o Page.<br><br>- Cambiar el menú contextual personalizado de una ventana control o Page (o asociar otro menú contextual).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Personaliza los menús de sistema de los mandos añadiendo opciones.<br><br><br> |
| [Extra](../Proprietes/1410089327.md) | La propiedad **Extra** permite:<br><br>- Obtener información adicional sobre un control, ventana o página.<br><br>- Editar información adicional de un control, ventana o página.<br><br><br> |
| [FullName](../Proprietes/2514005.md) | El **FullName** Property se utiliza para averiguar el nombre completo:<br><br>- de una control.<br><br>- de una ventana.<br><br>- de una opción de menú.<br><br>- de una columna tabla.<br><br><br><br><br> |
| [Group](../Proprietes/2510049.md) | La propiedad **Group** permite: <br><br>- averiguar si el control pertenece a un grupo de control s. Si el control pertenece a uno o varios grupos, se puede utilizar el **Grupo** Property para identificar estos grupos de control s.<br><br>- asocia una control con un grupo de control s. <br><br><br> |
| [Height](../Proprietes/2510050.md) | La **altura** Property se usa para get y cambiar la altura:<br><br>- de un control o bloque en un reporte.<br><br>- de una control (en una ventana), ventana, fila de una Tabla, Cuadro de Lista o control ListView, opción de un Botón de Radio o control Casilla de verificación, romper cabecera o pie de una Tabla o control Looper.<br><br>- de un control (en un Page), romper cabecera o pie de página en una Tabla o control Looper. La control puede ser superpuesta o no.<br><br>- de una celda de ruptura en una Tabla o control Looper.<br><br><br> |
| [HTMLAfter](../Proprietes/1000017002.md) | El **HTMLAfter** Property se utiliza para get y cambiar el código HTML insertado después del control. |
| [HTMLAttribute](../Proprietes/1410089223.md) | La propiedad **HTMLAttribute** lee y cambia los atributos HTML del control. |
| [HTMLBefore](../Proprietes/1000017051.md) | **HTMLBefore** se utiliza para averiguar y modificar el código HTML añadido antes del control. |
| [HTMLClass](../Proprietes/1000021530.md) | El **HTMLClass** Property se utiliza para leer y cambiar el valor del atributo HTML "class" del control. |
| [InitialVisible](../Proprietes/2510070.md) | La propiedad **..InitialVisible** permite saber si:<br><br>- un control fue visible durante la primera visualización de la ventana o página asociada.<br><br>- una ventana era visible durante su primera visualización.<br><br><br> |
| [MouseCursor](../Proprietes/2510026.md) | El **MouseCursor** Property se utiliza para identificar y cambiar el cursor hover definido para un control, ventana o página. |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [Note](../Proprietes/2510085.md) | La propiedad **Note** permite:<br><br>- Encuentra las notas associated con un control, una ventana, un Page o un reporte.<br><br>- Modifica las notas associated con un control, una ventana un Page o un reporte.<br><br><br> |
| [NoteTitle](../Proprietes/2510121.md) | La propiedad **NoteTitle** permite:<br><br>- get el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte. <br><br>- Cambiar el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte.<br><br><br> |
| [Opacity](../Proprietes/2514006.md) | El **Opacidad** Property se utiliza para get efinir el porcentaje de opacidad de una ventana, o de los controles de una ventana o Page. |
| [Process](../Proprietes/1000020878.md) | El **Process** Property se utiliza para modificar los Process es que serán ejecutados por los eventos disponibles en un control, ventana, Page, reporte, etc. |
| [Type](../Proprietes/2510131.md) | La propiedad **Type** permite obtener:<br><br>- el tipo de elemento (archivo de datos o campo) definido mediante programación o en el editor de análisis, vista o consulta.<br><br>- el tipo de objeto.<br><br>- el tipo de datos que se almacena en un Variant.<br><br><br> |
| [Visible](../Proprietes/2510138.md) | El **Visible** Property se utiliza para:<br><br>- Averiguar si un elemento es Visible.<br><br>- Hacer visible un elemento o Invisible.<br><br><br> |
| [Width](../Proprietes/2510059.md) | El **Ancho** Property obtiene y modifica el ancho:<br><br>- de un control o bloque en un reporte.<br><br>- de un control (que se encuentra en una ventana), de una ventana, de una columna que se encuentra en un tabla o de una columna que se encuentra en un cuadro de lista (incluida la listview).<br><br>- de un control (que se encuentra en un Page), columna que se encuentra en un tabla o columna que se encuentra en un cuadro de lista. La control puede ser superpuesta o no.<br><br><br> |
| [X](../Proprietes/2510135.md) | El **X** Property se utiliza para:<br><br>- get la coordenada X de una columna o ventana control, control Tabla (posición en el eje X).<br><br>- get la coordenada X de un control o ventana (posición en el eje X).<br><br><br> |
| [XInitial](../Proprietes/2510140.md) | El **XInitial** Property se utiliza para:<br><br>- get la coordenada X: <br><br>	- de un control (posición en el eje X) al abrir la ventana o la página.<br><br>	- de una ventana (posición en el eje X) al abrir la ventana. <br><br><br><br><br>- Cambiar la coordenada X de un control anclado "a la derecha" (posición en el eje X) al abrir la ventana.<br><br>- get la coordenada X de un control en relación con su bloque (posición en milímetros en el eje X) al abrir el reporte.<br><br><br> |
| [Y](../Proprietes/2510137.md) | El **Y** Property se utiliza para:<br><br>- Averigua la coordenada Y de un control o ventana (posición en el eje Y).<br><br>- Modificar la coordenada Y de un control o ventana.<br><br><br> |
| [YInitial](../Proprietes/2510087.md) | El **YInicial** Property se utiliza para:<br><br>- Averigua la coordenada Y: <br><br>	- de un control (posición en el eje Y) al abrir la ventana o la página.<br><br>	- de una ventana (posición en el eje Y) al abrir la ventana.<br><br><br><br><br>- Modifica la coordenada Y de un control anclado "al fondo" (posición en el eje vertical) al abrir la ventana.<br><br>- Averigua la coordenada Y de un control en relación con su bloque (posición en milímetros en el eje vertical) al abrir el reporte.<br><br><br> |

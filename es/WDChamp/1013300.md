## Presentación
<a name="presentacion_ELTTEXTE000139"></a>


### control de un sitio
<a name="control_sitio_ELTPARAGRAPHE000011"></a>

El control Mapa del sitio permite al usuario Web:

- tienen una visión global de las páginas que se encuentran en el sitio.

- acceder a una página específica directamente.




La control Mapa del sitio también se utiliza para mejorar la referenciación de un centro.

## Creación de una control Mapa del sitio
<a name="creacion_una_control_mapa_del_sitio_ELTTEXTE000163"></a>


### Crear un control Mapa del sitio
<a name="crear_control_mapa_del_sitio_ELTPARAGRAPHE000025"></a>

Para crear una control Mapa del sitio:

1. En la pestaña "Creación", en el grupo "Navegación", haga clic en "Mapa del sitio".

2. Haga clic en la ubicación en la que desea crear el control. El control aparece en el editor. Si se define un ruta de navegación para el centro actual, el control Mapa del sitio visualiza los datos correspondientes.




Para ver las características del control, seleccione la opción "Descripción en el menú contextual.
<a name="NOTE2_2"></a>


### Características de una control Mapa del sitio
<a name="caracteristicas_una_control_mapa_del_sitio_ELTPARAGRAPHE000049"></a>

La ventana description de un "Mapa del sitio" control le permite:

- Construya el mapa del sitio si es necesario (pestaña "General").
	Si su sitio no contiene "Mapa del sitio" control, tiene la posibilidad de editar y construir [el mapa del sitio ruta](../WDChamp/1013301.md).

- Especificar las diferentes secciones del mapa del sitio a mostrar (pestaña "General").
	De hecho, se pueden utilizar varios controles de "Mapa del sitio" en la misma página: tienes la capacidad de mostrarlas una al lado de la otra y de usar diferentes modos de presentación. Si no se selecciona ningún elemento, se mostrará todo el mapa del sitio en el control (limitado por el número de niveles de Visible).

- Especifique el número de columnas de cada nivel así como la sangría de cada nivel (pestaña "Detalles").

- Indica los estilos utilizados para cada nivel del mapa del sitio (pestaña "Estilo").

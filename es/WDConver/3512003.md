## Presentación

### Comando Modo Line
<a name="comando_modo_line_ELTPARAGRAPHE000011"></a>

El uso de WDConver en modo de línea de comandos consiste en ejecutar una conversión description creada y guardada con WDConver (archivo "WDV").

La siguiente sintaxis se usa para iniciar WDConver en el comando Line:


```txt
WDConver.EXE [-report][-error][-wizard]/SCRIPT=<WDVFile>
```


**Detalles de los parámetros**:

| Parámetros | Significado |
| --- | --- |
| [-reporte] | Si se especifica este parámetro, se visualizará un estado reporte al final de la ejecución del script.<br>Por Default, si no se especifica este parámetro, no se mostrará ningún estado reporte al final de la ejecución del guión. |
| [-error] | Si se especifica este parámetro, se mostrará un mensaje de error al final de la ejecución del script.<br>Por Default, si no se especifica este parámetro, no se mostrará ningún mensaje de error al final de la ejecución del guión. |
| [-asistente] | Si se especifica este parámetro, se iniciará la asistente para la conversión de datos.<br>Por Default, si no se especifica este parámetro, no se iniciará la asistente para la conversión de datos. |
| /SCRIPT=&lt;WDVFile&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;)) | Cadena de caracteres que contiene el nombre y la ruta completa del description de conversión a uso (archivo "WDV").<br>Este parámetro es **obligatorio** excepto si se especifica el parámetro "-asistente". |
| /WDD=&lt;Ruta del análisis&gt; | Cadena de caracteres que contiene el ruta del análisis WINDEV, WEBDEV o WINDEV Mobile. Esta ruta debe incluirse entre comillas si contiene espacios en blanco. |
| /PWD=&lt;Contraseña no cifrada del análisis&gt;. | Cadena de caracteres que contiene la contraseña del análisis WINDEV, WEBDEV o WINDEV Mobile. Esta contraseña debe ir acompañada de comillas si contiene espacios en blanco. |
| /WDD55=&lt;Ruta del análisis del WINDEV 5.5&gt; | Cadena de caracteres que contiene el ruta del análisis WINDEV 5.5. Esta ruta debe incluirse entre comillas si contiene espacios en blanco. |
| /PWD55=&lt;Contraseña no cifrada del análisis de WINDEV 5.5&gt;. | Cadena de caracteres que contiene la contraseña del análisis de WINDEV 5.5. Esta contraseña debe ir acompañada de comillas si contiene espacios en blanco. |
| /SRCDIR=&lt;Directorio de los archivos de datos de Hyper File 5.5 buscados&gt;&gt;Directorio de los archivos de datos de Hyper File 5.5 buscados | Cadena de caracteres que contiene la ruta de los archivos de datos de Hyper File 5.5 a convertir. |
| /DSTDIR=&lt;Directorio de archivos de datos convertidos&gt; | Cadena de caracteres que contiene el directorio donde se crearán los archivos de datos convertidos a HFSQL Classic |
| /FILE=&lt;Archivo de datos fuente&gt; | Cadena de caracteres que contiene:<br><br>- la ruta del archivo de datos Hyper File 5.5 para convertir<br><br>- el nombre de la tabla a convertir en el servidor.<br><br><br> |
| /NAME=&lt;Nombre lógico en el análisis WINDEV 5.5 del archivo a convertir&gt;&gt;. | Cadena de caracteres correspondiente al nombre lógico del archivo de datos para convertir en el análisis WINDEV, WEBDEV o WINDEV Mobile. |
| /FILEPWD55=&lt;Contraseña no cifrada del archivo de origen&gt;. | Cadena de caracteres correspondiente a la contraseña del archivo WINDEV 5.5 a convertir. |
| /FILEDST=&lt;Destino ruta del archivo convertido&gt;. | Cadena de caracteres correspondiente a la ruta del fichero convertido |
| /NOREP | Se usa para ignorar el archivo.REP en formato WINDEV 5.5. |
| /MODE=[ HF5 \ EXTERN \ XML \ TEXT] Tipo de archivos fuente | Define el formato de los archivos a importar:<br><br>- HF5: Archivos de datos Hyper File 5.5<br><br>- EXTERN: archivos/tablas externas (ejemplo: Base de datos C/S)<br><br>- XML: archivos XML<br><br>- TEXTO: archivos de texto<br><br><br> |
| /PROVIDER=&lt;Proveedor OLEDB o Conector Nativo&gt; | Cadena de caracteres que define el proveedor o el Conector Nativo (también conocido como Native Access) utilizado para acceder a los datos a convertir..<br>Para la Conectores Nativos, utilice los siguientes strings: WinDevOracle, WinDevSQLServer, WinDevInformix, etc. |
| /DATASOURCE=&lt;Nombre de la fuente (archivo o servidor)&gt;. | Cadena de caracteres. Este parámetro se utiliza cuando se convierte un archivo a través de un proveedor Conector Nativo o OLE DB., ...<br>Corresponde al nombre de la fuente de datos. |
| /USER=&lt;nombre de usuario&gt; | Cadena de caracteres. Corresponde al nombre del usuario de la fuente de datos. |
| /USERPWD=&lt;Contraseña de usuario no cifrada&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;&gt;)) | Cadena de caracteres. Corresponde a la contraseña del usuario de la fuente de datos. |
| /BASE DE DATOS=&lt;Nombre de la base de datos&gt;&gt;Nombre de la base de datos | Cadena de caracteres. Corresponde a la base de datos a la que se accede (si se encuentran varias en el servidor). |

**Observación**: El Above tabla presenta los diferentes parámetros que pueden ser usados para iniciar WDConver en el modo de comando Line. Para get estos parámetros al iniciar WDConver, utilice una de las siguientes sintaxis:

- WDConver.EXE /help

- WDConver.EXE /?




**Observación**: Los scripts creados con una versión anterior de WINDEV 7 no se pueden utilizar con WDConver versión 8 y posteriores.. Los scripts creados con WINDEV 7.5 se pueden utilizar con WDConver versión 8 y posteriores.

## Ejemplos
<a name="ejemplos_ELTTEXTE000265"></a>


### Ejemplo de uso de un script:
<a name="ejemplo_uso_script_ELTPARAGRAPHE000160"></a>

La siguiente línea de comandos se utiliza para ejecutar la conversión description llamada "C:\\MyDirectory\\Mydescription.WDV". Si se produce un error, se muestra un mensaje al final de la modificación de los archivos de datos de automatic (parámetro "-error"). La modificación de los archivos de datos de asistente para automatic no se iniciará.


```wl
ExeRun("C:\MyTools\WDConver.EXE " +...
	"-error/SCRIPT=C:\MyDirectory\MyDescription.WDV")
```

<a name="NOTE2_2"></a>


### Ejemplo de uso
<a name="ejemplo_uso_ELTPARAGRAPHE000169"></a>

El siguiente comando Line se utiliza para:

- Seleccione todos los archivos Hyper File 5 accesibles por el archivo ".REP" y en el directorio "C:\\data".

- Convierta estos archivos en HFSQL Classic en el directorio "C:\\dataclassic".





```wl
ExeRun("C:\MyTools\WDConver.EXE " +...
	"/WDD55=C:\WDProject\Project5\Project.WD5\Project.WDD" +... 
	"/WDD=""C:\My Projects\Project\Project.wdd" +...
	"/SRCDIR=""C:\data"" /DSTDIR=c:\dataclassic")
```

<a name="NOTE2_3"></a>


### Ejemplo de conversión de un fichero individual
<a name="ejemplo_conversion_fichero_individual_ELTPARAGRAPHE000181"></a>

El siguiente comando Line se utiliza para convertir un archivo específico:


```wl
ExeRun("C:\MyTools\WDConver.EXE " +... 
	"/WDD55=C:\WDProject\Project5\Project.WD5\Project.WDD " +...
	"/WDD=C:\My Projects\Project\Project.wdd " +...
	"/FILE=C:\data\exchange.fic /NAME=EXCHANGE " +...
	"/DSTDIR=c:\dataclassic /NOREP")
```

<a name="NOTE2_4"></a>


### Ejemplo de conversión a través de un Conector Nativo (o acceso nativo)
<a name="ejemplo_conversion_traves_conector_nativo_acceso_nativo_ELTPARAGRAPHE000190"></a>

El siguiente comando Line se usa para convertir el "DEMO.CUSTOMER" tabla a través del Conector Nativo Oracle. El nombre lógico de esta tabla es "CLIENTE" en el servidor "MARS".. El archivo HFSQL Classic se creará en el directorio "c:\\data".


```wl
ExeRun("C:\MyTools\WDConver.EXE " +...
	"/MODE=EXTERN /DATASOURCE=MARS "+...
	"/PROVIDER=WinDevOracle /USER=DEMO "+...
	"/USERPWD=DEMO "+...
	"/WDD=""C:\My Projects\test.ANA\test.wdd"""+...
	"/DSTDIR=c:\data /FILE=DEMO.CUSTOMER /NAME=CUSTOMER")
```

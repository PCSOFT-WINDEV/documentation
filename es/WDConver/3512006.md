## Presentación
<a name="presentacion_ELTTEXTE000215"></a>


### WDConver
<a name="wdconver_ELTPARAGRAPHE000011"></a>

WDConver es una herramienta utilizada para convertir a un archivo de datos HFSQL Classic los **datos** de un archivo:

- en formato Hyper File 5 o 4.

- en formato de texto.

- en formato XML.

- en una base de datos a la que se accede mediante un proveedor OLE DB o un controlador ODBC (Excel, Access, Oracle, SQL Server, etc.).

- en una base de datos a la que se accede a través de un conector nativo de WINDEV (SQL Server, Oracle o AS/400).




**Atención**: WDConver sólo se puede utilizar para convertir el **datos** encontrado en un archivo. Para realizar esta conversión, la **estructura** del archivo de datos debe haberse convertido previamente. Para convertir la estructura de un archivo de datos, en la pestaña "Análisis", en el grupo "Creación", amplíe "Importar" y seleccione "Importar archivo/descripciones tabla...". 

**Observación**: A través de la programación, las importaciones pueden ser realizadas por [HImportHF55](../WDLang4/3044055.md), [HImportText](../WDLang4/3044011.md), [HImportXML](../WDLang4/3044007.md), ...

**Ejemplo de uso**: Después de una importante actualización de los datos encontrados en un archivo externo, WDConnect se utiliza para reimportar los datos a una base de datos WINDEV.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000239"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000053"></a>

WDConver puede ser redistribuido. WDConver se puede instalar con las aplicaciones desarrolladas con WINDEV. El acuerdo de licencia de WINDEV se aplica en su totalidad.

Al crear el instalación Procedure, WDConver puede ser suministrado con sus aplicaciones.

**Recordatorio**: El procedimiento de instalación se puede crear:

- de la asistente: en la pestaña "Proyecto", en el grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación", 

- desde el editor de instalación: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

## Iniciar WDConver
<a name="iniciar_wdconver_ELTTEXTE000269"></a>
WDConver se puede iniciar:

- [**En modo interactivo**](../WDConver/3512002.md), ejecutando "WDConver.EXE" o desde la página web de editor de análisis: en la pestaña "Análisis", en el grupo "Creación", amplíe "Importar" y seleccione "Importar datos de otra base de datos".

- [**En el comando Line modo**](../WDConver/3512003.md)para realizar una conversión desde una aplicación WINDEV.

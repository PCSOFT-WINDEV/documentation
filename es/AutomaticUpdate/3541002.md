


## Configuración de la actualización de automatic
			



<a name="NOTE1_24"></a>
<a name="NOTE1_1_24"></a>


## Presentación
<a name="presentacion_ELTTEXTE000119"></a>
automatic Update es una aplicación que le permite automatic aliar o actualizar manualmente los diferentes productos PC SOFT que posee.

Puede configurar:

- El inicio de la aplicación. 

- El directorio de descarga de actualizaciones.

- El ancho de banda utilizado.

## Opciones de configuración
<a name="opciones_configuracion_ELTTEXTE000167"></a>


### General
<a name="general_ELTPARAGRAPHE000039"></a>

La opción "Iniciar en el inicio de Windows" le permite al aliado de automatic lanzar la actualización de automatic cuando Windows se inicia. Esta opción asegura que se realice una búsqueda de automatic en cada inicio. 

Observación: Una búsqueda de actualizaciones en automatic se realiza cada 24 horas. 
<a name="NOTE3_2_24"></a>


### Descargar
<a name="descargar_ELTPARAGRAPHE000048"></a>

Hay dos modos disponibles para descargar actualizaciones. Es posible: 

- Descargar actualizaciones automáticamente: en este caso, puede instalar la actualización del producto deseado directamente desde "automatic Update" (botón "Update"). 

- Se pide que se descargue: en este caso, un mensaje le informa de que hay una actualización disponible y le pide que la descargue. También puede descargar la actualización del producto deseado (mediante el botón "Descargar" que se encuentra en la fila del producto). También se muestra una Notification en la parte superior de la aplicación. Si hay varias actualizaciones disponibles, se muestra el enlace "Descargar todo".. 




También puede: 

- especificar un directorio de descarga actualizado. Por Default, las actualizaciones de productos PC SOFT se descargan en el siguiente directorio: "C:\\Datos del programa\\PC SOFT\\Actualización automática\\Paquetes". Para cambiar este directorio, simplemente especifique otro ruta. 
	Observación: Al instalar una actualización, el archivo descargado no se elimina automáticamente. 

- indicar un límite de tamaño de descarga. 

- especificar un límite de tiempo para mantener las descargas. 

- permitir la descarga de actualizaciones en el nivel de validación 1. En este caso, también estarán disponibles las actualizaciones que hayan completado sólo la primera fase de validación. Para más detalles sobre las diferentes fases de validación, véase [Versión de validación](https://windev.com/ts/download/validation-vi.htm).

<a name="ancho_banda_ELTPARAGRAPHE000070"></a>

### ¿Qué actualizaciones tengo que descargar?
<a name="¿que_actualizaciones_tengo_que_descargar_ELTPARAGRAPHE000035"></a>

Las actualizaciones detectadas por la aplicación "automatic Update" corresponden a los productos PC SOFT que utiliza. Las actualizaciones propuestas le permiten aprovechar rápidamente las últimas funciones disponibles para sus productos.
<a name="NOTE2_2"></a>


### ¿Qué datos recoge PC SOFT al buscar e instalar las actualizaciones?
<a name="¿que_datos_recoge_soft_buscar_instalar_las_actualizaciones_ELTPARAGRAPHE000042"></a>

PC SOFT no recopila ninguna información personal ni ningún detalle del producto (nombre, número de serie, etc.) al buscar e instalar actualizaciones. Sólo se conoce la IP Address de su ordenador.
<a name="NOTE2_3"></a>


### ¿Qué idiomas se tienen en cuenta?
<a name="¿que_idiomas_tienen_cuenta_ELTPARAGRAPHE000049"></a>

La actualización de automatic está disponible en diferentes idiomas. Las actualizaciones propuestas dependerán del idioma del producto instalado en su ordenador (versión internacional o francesa).

Esta opción no está disponible en esta versión.
<a name="NOTE2_4"></a>


### ¿Cuántas veces tengo que buscar actualizaciones de get la versión más reciente de mi producto?
<a name="¿cuantas_veces_tengo_que_buscar_actualizaciones_get_version_mas_reciente_producto_ELTPARAGRAPHE000058"></a>

Las actualizaciones que se pueden utilizar con la versión actual de su producto se proponen cada vez que se ejecuta la actualización de automatic.

Después de cada actualización, es aconsejable volver a realizar la búsqueda de actualizaciones para ver si hay nuevas actualizaciones disponibles.
<a name="NOTE2_5"></a>


### No deseo instalar actualizaciones de mis productos desde la Web. ¿Puedo descargar las actualizaciones en mi ordenador e instalarlas yo mismo?
<a name="deseo_instalar_actualizaciones_mis_productos_desde_web_¿puedo_descargar_las_actualizaciones_ordenador_instalarlas_mismo_ELTPARAGRAPHE000067"></a>

Sí, las actualizaciones pueden descargarse e instalarse localmente en su ordenador.. Todas las actualizaciones están disponibles en el sitio de [PC SOFT, Centro de descargas](https://www.windev.com/ts/download/index.html).
<a name="NOTE2_6"></a>


### Estoy usando un firewall. ¿Puedo buscar actualizaciones?
<a name="estoy_usando_firewall_¿puedo_buscar_actualizaciones_ELTPARAGRAPHE000077"></a>

Si utiliza un cortafuegos que solicita autorización para cada conexión, es necesario responder "Sí" para que la comprobación de la actualización pueda funcionar correctamente.

No podrás usar el mecanismo de actualización del editor si estás usando un cortafuegos silencioso.
<a name="NOTE2_7"></a>


### Estoy usando un proxy. ¿Puedo buscar actualizaciones?
<a name="estoy_usando_proxy_¿puedo_buscar_actualizaciones_ELTPARAGRAPHE000086"></a>

Si utiliza un proxy, podrá utilizar el mecanismo de actualización del editor. Este mecanismo utiliza la configuración del proxy del Explorador de INTERNET.
<a name="NOTE2_8"></a>


### ¿Cómo puedo desinstalar las actualizaciones?
<a name="¿como_puedo_desinstalar_las_actualizaciones_ELTPARAGRAPHE000093"></a>

No es posible borrar las actualizaciones de los diferentes productos de PC SOFT. Para eliminar las actualizaciones instaladas, debe desinstalar el producto del equipo y, a continuación, volver a instalarlo.




## Month (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **Mes** Property se utiliza para get y cambiar el mes de un elemento. Esta propiedad se puede utilizar en:

- a Fecha, hora y fecha Variable o elemento. Para obtener más información, consulte [Mes Property en los tipos de fecha, hora y dateTime](../Proprietes/2512052.md).  

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md). 

- una variable de tipo [hScheduling](../WDLang4/1000018853.md).

- una variable de tipo [Repetition](../WDLang1/1000019262.md).

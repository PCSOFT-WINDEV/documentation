
## UserID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **UserID** Property se utiliza para get el identificador del usuario. Esta propiedad se puede utilizar en:

- una variable de tipo [sfSessionInfo](../WDLang5/1000018416.md).

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md).

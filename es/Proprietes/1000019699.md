
## DefaultCell

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DefaultCell** Property se utiliza para set el estilo de las celdas en los documentos XLS. 

Esta propiedad se puede utilizar en:

- una variable de tipo [xlsColumn](../WDLang5/1000017471.md).

- una variable de tipo [xlsRow](../WDLang5/1000017325.md).




## VerticalAlignment (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>
La alineación vertical de una control consiste en definir la posición del texto en la control en relación con los bordes de la control. Esto le permite centrar el contenido de un control Campo de entrada en relación a los bordes del área de entrada.. 

Hay tres tipos de alineación disponibles: 

- Alineación con la parte superior<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cardage_Haut.gif)


- Alineación en el centro<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cadrage_Milieu.gif)


- Alineación en la parte inferior<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cadrage_Bas.gif)




<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**VerticalAlignment** Property se usa para get y cambiar la alineación vertical de un control .

La alineación inicial de la control se define al describir la control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the control is aligned to top, align it at bottom
IF STC_Static.VerticalAlignment = vaTop THEN
	STC_Static.VerticalAlignment = vaBottom
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la alineación vertical de una control

`<Current alignment> = <Control used>.VerticalAlignment`
---

**`<Current alignment>: (Constante de tipo Integer)`**

Alineación vertical actual de control. Los valores posibles son los siguientes:



|   |   |
| --- | --- |
| *vaBottom* | el texto se encuentra en la parte inferior de la control |
| *vaMiddle* | el texto se encuentra en el centro de la control |
| *vaTop* | el texto se encuentra en la parte superior de la control |




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificación de la alineación vertical de control

`<Control used>.VerticalAlignment = <New alignment>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<New alignment>: (Constante de tipo Integer)`**

Nueva alineación de la control. Los valores posibles son: 



|   |   |
| --- | --- |
| *vaBottom* | el texto se encuentra en la parte inferior de la control |
| *vaMiddle* | el texto se encuentra en el centro de la control |
| *vaTop* | el texto se encuentra en la parte superior de la control |

## Observaciones


### Alineación horizontal
<a name="alineacion_horizontal_ELTPARAGRAPHE000191"></a>

Para definir la alineación horizontal del control, utilice la función [HorizontalAlignment](../Proprietes/2510023.md) Property.
<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000201"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **VerticalAlignment** Property se usa para get y cambia la alineación vertical para los siguientes tipos de controles:

- Estática

- Casilla de verificación o botón de radio

- Image

- Columna en una Tabla o Tabla TreeView

- List Box

- Combo Box

- Código de barras

- Gráfico

- Control Valoración

- Organigrama


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte , el **VerticalAlignment** Property se usa para get para cambiar los tipos de alineación vertical y los siguientes tipos:

- Control Estático.

- Control Calculado.

- Control Predefinido.

- Control Gráfico.

- Grupo de controles: Esta Property sólo se aplicará a los controles que admitan la alineación vertical.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **VerticalAlignment** solo se puede utilizar en los siguientes controles:

- Estática.

- Image.

- Casilla de verificación.

- Botón de opción.

- Combo Box.

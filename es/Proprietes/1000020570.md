
## Exist (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Exist** Property se utiliza para determinar si un elemento existe. 

Esta propiedad se puede utilizar en:

- la arrays asociativa o sus elementos. Para obtener más información, consulte [Exist](../Proprietes/1000022167.md).

- una variable de tipo [Variant](../Motscles/1514019.md). 

- una variable de tipo [XMLNode](../WDLang5/1000018786.md). 

- una variable de tipo [JSON](../Motscles/1000023611.md). 

- una variable de tipo [YAML](../Motscles/1000024438.md). 

- una variable de tipo [htmlAttribute](../WDLang5/1000025912.md).

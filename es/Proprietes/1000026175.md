
## InvalidInputPreventExit (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InvalidInputPreventExit** Property se utiliza para determinar si es posible salir del control en caso de entrada no válida. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF EDT_Password.InvalidInputPreventExit = True THEN
	InvalidInputShowTooltip("Invalid input.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificación del modo de gestión de entradas no válidas

`<Result> = <Control used>.InvalidInputPreventExit`
---

**`<Result>: ()`**



- <u><u><u><u>True</u></u></u></u>: no es posible salir de la control si la entrada no es válida y/o si la control está vacía. 

- <u><u><u><u>False</u></u></u></u>: es posible dejar la control en cualquier caso. 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- a control Campo de entrada,

- una Combo Box editable.

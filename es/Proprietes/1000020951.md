


## CaptionWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CaptionWidth** Property se utiliza para get y cambiar el ancho reservado para el área de texto en una Edición o control Combo Box. 

**Observación**: Esta Property es útil cuando se crea una control con [ControlCreate](../WDLang1/1000020876.md). De hecho, durante la creación dinámica de un control, el ancho de la leyenda es set a 0 por Default. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enlarge the caption of edit control 
EDT_EDIT1.Width += 150
EDT_EDIT1.CaptionWidth += 150
EDT_EDIT1.Caption = CUSTOMER.NAME + "'s phone number"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtención de la anchura del área de texto

`<Result> = <Control used>.CaptionWidth`
---

**`<Result>: (Integro)`**

Anchura (en píxeles) del área de texto en la control. 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser un control Campo de entrada o un control Combo Box. 


<a name="SYNTAX2"></a>

### Modificación de la anchura del área de texto

`<Control used>.CaptionWidth = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser un control Campo de entrada o un control Combo Box. 

**`<New value>: (Integro)`**

Nuevo ancho (en píxeles) de la leyenda associated con el control. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La control debe usar un estilo con la leyenda a la izquierda. En caso contrario, el **CaptionWidth** Property no tiene ningún efecto.

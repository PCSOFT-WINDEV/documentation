


## ContextMenu (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ContextMenu** Property se utiliza para:

- get el menú contextual personalizado de una ventana control, o Page.

- Modificar el menú contextual personalizado de una ventana control, o Page (o asociar otro menú contextual).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Personaliza los menús de sistema de los controles añadiendo opciones.




**Recordatorio**: Un menú contextual es un menú que el usuario abre haciendo clic con el botón derecho del ratón.. Puede crear menús contextuales directamente en el editor de ventanas o páginas, en la pestaña UI de la ventana de control description. La flecha situada a la derecha del campo "Menú contextual" permite editar o crear un menú contextual en el editor. Para obtener más información, consulte: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Menús contextuales en una ventana](../WDChamp/1010011.md).




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) **Observación**: Algunos tipos de controles (Campo de entrada, Tabla, etc.) tienen un menú del sistema predeterminado (que se muestra si no hay ningún menú contextual personalizado asociado al control). Este menú del sistema también se puede desactivar.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Code used to modify the context menu associated with the button
// Popup1 and Popup2 are existing context menus
IF BTN_Button1.ContextMenu = "Popup1" THEN
	BTN_Button1.ContextMenu = "Popup2"
ELSE
	// Restores the default AAF context menu
	BTN_Button1.ContextMenu = ""
END
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Initialization code of edit control
// Adds the Popup1 menu after the system context menu of the control
// Popup1 is an existing context menu
MySelf.ContextMenu = "+Popup1"
```

<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el menú contextual de una ventana control o Page

`<Menu> = <Element>.ContextMenu`
---

**`<Menu>: (Cadena de caracteres)`**



- Nombre del menú contextual personalizado associated con el objeto,

- Cadena vacía ("") si no hay ningún menú contextual personalizado associated con el objeto.




**`<Element>: (control, ventana o nombre de Page)`**

Elemento del que debe obtenerse el nombre del menú contextual: 

- Nombre del control, 

- Nombre de ventana, 

- Nombre de la página.





<a name="SYNTAX2"></a>

### Modificación del menú contextual de una ventana control o Page

`<Element>.ContextMenu = <New menu>`
---

**`<Element>: (control, ventana o nombre de Page)`**

Nombre del elemento cuyo menú contextual debe modificarse: 

- Nombre del control, 

- Nombre de ventana, 

- Nombre de la página.




**`<New menu>: (Cadena de caracteres o constante)`**

Nombre del nuevo menú contextual personalizado associated con el objeto (cadena vacía para restaurar el menú contextual del sistema). 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para añadir un menú contextual al menú del sistema, debes:

- escriba el signo "+" delante del nombre del menú (para mostrar el nuevo menú después de las opciones del menú del sistema).

- escriba el signo "+" después del nombre del menú (para mostrar el nuevo menú antes de las opciones del menú del sistema).

- especifique una cadena vacía ("") para utilizar el menú contextual AAF Default.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) También tiene la posibilidad de utilizar las siguientes constantes:


|   |   |
| --- | --- |
| *contextMenuDisabled* | Permite desactivar la visualización de todos los menús contextuales (incluidos los menús contextuales del sistema o los menús contextuales de los controles de edición Moneda + Euro).<br><br> |
| *contextMenuSystem* | Se utiliza para desactivar la visualización de las opciones de menú específicas de los AAF (Características de la aplicación automatic). Sólo se muestra el menú contextual de Windows. <br>También permite acceder a las opciones Unicode disponibles para algunos sistemas Windows ("Leer de derecha a izquierda, Mostrar los caracteres Unicode control, ..."). |

## Observaciones


### Añadir opciones de menú al menú contextual
<a name="anadir_opciones_menu_menu_contextual_ELTPARAGRAPHE000142"></a>

El **ContextMenu** Property no puede usarse para acumular el contexto personalizado associated con un menus personalizado un hex6100730073006f006300690061007400650064400 con un menus personalizado de un control. Sin embargo, puede añadir opciones a un menú contextual existente utilizando [MenuAddMenu](../WDLang1/1000017266.md), [MenuAddOption](../WDLang1/1000017262.md), ... Para obtener más información, consulte [Funciones de gestión de menús](../WDLang1/3038016.md).
<a name="NOTE0_2"></a>


### Menú contextual y grupo de controles
<a name="menu_contextual_grupo_controles_ELTPARAGRAPHE000160"></a>

Un menú contextual puede ser associated con un grupo de controles: todos los controles pertenecientes al grupo tendrán el mismo menú contextual. En modo lectura,  corresponde siempre a una cadena vacía ("").<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Control Tabla horizontal
<a name="control_tabla_horizontal_ELTPARAGRAPHE000171"></a>

El menú contextual del sistema de la controles Tabla Horizontal no contiene las opciones de exportación.
<a name="NOTE0_4"></a>


### Modificación del menú contextual y almacenamiento de información sobre controles Tabla
<a name="modificacion_del_menu_contextual_almacenamiento_informacion_sobre_controles_tabla_ELTPARAGRAPHE000178"></a>

Al cambiar el menú contextual, la información persistente guardada anteriormente para la control Tabla (tamaño de las columnas, ...) ya no está disponible.. Esta información se reinicializa.
<a name="NOTE0_5"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000185"></a>

El **ContextMenu** Property se aplica solo a los controles en una ventana (incluidas las columnas en controles Tabla ). Esta Property no puede utilizarse con menús, etc.

El **ContextMenu** Property no se aplica a los siguientes controles:

- Estática

- Image no clickeable

- Barra de progreso

- Forma




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **ContextMenu** solo se aplica a los siguientes elementos:

- Botón.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Ventana.

- Columna de control Tabla.

- Supercontrol.

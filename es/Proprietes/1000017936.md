
## Format (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Format** permite conocer y modificar el formato de un elemento. Esta propiedad se puede utilizar en:

- los archivos de datos. Para obtener más información, consulte [Formato Property en archivos de datos](../Proprietes/2512045.md).

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md). 

- una variable de tipo [Icon](../WDLang1/1000020811.md).

- una variable de tipo [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md).

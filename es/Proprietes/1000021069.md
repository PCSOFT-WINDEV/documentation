


## AnimationDuration (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AnimationDuration** Property se utiliza para conocer y definir el tiempo que tarda una animación en completar un ciclo: 

- en una control Gráfico. Esta duración es set a 30 centésimas de segundo por Default.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Barra de progreso infinita. Esta duración es set a 200 centésimas de segundo por Default.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase timeout
CHART_MyChart.AnimationDuration = 2000
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la duración de la animación

`<Result> = <Control used>.AnimationDuration`
---

**`<Result>: (Integro)`**

Duración de la animación control, en centésimas de segundo. 

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- A control Gráfico. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) A control Barra de progreso infinita.





<a name="SYNTAX2"></a>

### Modificar la duración de la animación

`<Control used>.AnimationDuration = <New duration>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- A control Gráfico. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) A control Barra de progreso infinita.




**`<New duration>: (Entero o Duración)`**

Nueva duración para la animación de control en centésimas de segundo. Si esta nueva duración es set a 0, la animación no se realizará más. 
Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo Duration,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la duración en un formato legible (por ejemplo, "1cs" o "10 ms").

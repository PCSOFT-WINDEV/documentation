
## User (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **User** permite obtener y cambiar el usuario de una funcionalidad. Esta propiedad se puede utilizar en:

- a Variable de tipo [Conexión](../WDLang4/1514073.md). Para obtener más información, consulte [drogadicta](../Proprietes/2512107.md).

- a Variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md).

- una variable generada automáticamente para representar [un Webservice importado](../WDLang3/3086001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable generada automáticamente para representar [un servicio web REST importado](../WDLang3/1410087809.md).

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una variable de tipo [sshSession](../WDLang3/1000021157.md).

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [hScheduleMaterializedView](../WDLang4/1000021444.md).

- una variable de tipo [hdfsConnection](../WDLang4/1000021650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a Variable de tipo [mqttSession](../WDLang3/1000023023.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [memcachedConnection](../WDLang4/1000023339.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [saasVisit](../WDLang3/1000024560.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WBAdminAuth](../WDLang2/1410089328.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WBAdminAccount](../WDLang2/1410089382.md).

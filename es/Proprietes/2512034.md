
## SourceFile (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SourceFile** Property se utiliza para set las características de un enlace. Es posible:

- Averigüe el nombre del archivo de origen utilizado en el enlace.

- Cambiar el archivo de origen utilizado en el enlace.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FichierSource.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the source file
ProjectManagement.SourceFile = PROJMGT
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el archivo de origen del enlace

`<Data file> = <Link>.SourceFile`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo fuente del enlace.

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en el editor de análisis o a través del tipo de [Descripción de la relación](../WDLang4/1514039.md).  


<a name="SYNTAX2"></a>

### Cambiar el archivo de origen del enlace

`<Link>.SourceFile = <New source file>`
---

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en la editor de análisis o a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

**`<New source file>: (Cadena de caracteres)`**

Nombre del archivo fuente utilizado por el enlace. Fichero definido en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La [LinkedFile](../Proprietes/2512031.md) Property se utiliza para obtener y modificar el archivo relacionado del enlace.

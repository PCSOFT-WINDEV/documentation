


## CrosshairOpacity (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **CrosshairOpacity** Property se utiliza para determinar y cambiar la opacidad de la retícula en un control Gráfico. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the opacity of the crosshair
CHART_MyChart.CrosshairOpacity = 50
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir la opacidad del punto de mira para un control Gráfico

`<Result> = <Chart control>.CrosshairOpacity`
---

**`<Result>: (Integro)`**

Opacidad actual del punto de mira del gráfico (entre 0 y 255). Por defecto, la opacidad es de set a 255 (totalmente opaca). 

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificando la opacidad del retículo para un control Gráfico

`<Chart control>.CrosshairOpacity = <New opacity>`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se va a utilizar. 

**`<New opacity>: (Constante de tipo Integer)`**

Nueva opacidad de la cruz, entre 0 (Invisible) y 255 (completamente opaca).

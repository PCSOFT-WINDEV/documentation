
## State (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **Estado** Property se utiliza para get y cambiar el estado de un elemento. Esta propiedad se puede utilizar en:

- **controles, ventanas, páginas y reportes**. Para obtener más información, consulte [Estado Property en los controles](../Proprietes/2510039.md).

- una variable de tipo [ParallelTask](../WDLang1/1000021220.md).

- una variable de tipo [Thread](../WDLang1/1000024879.md).




## SelectedText (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Texto Seleccionado** Property se utiliza para get y cambiar el texto seleccionado en un control editable, control Combo Box editable o columna editable en un control Tabla.

Observación: Esta Property permite evitar el uso de [Cursor](../Proprietes/2510025.md) y [CursorEnd](../Proprietes/2510047.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF EDT_Edit1.SelectedText = "<E>" THEN
	EDT_Edit1.SelectedText = "<Enter>"
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el texto seleccionado

`<Text> = <Control used>.SelectedText`
---

**`<Text>: (Cadena de caracteres)`**

Texto seleccionado por el usuario en el control Campo de entrada.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- a control Campo de entrada,

- a Combo Box en modo edición,

- una columna de una control Tabla en modo edición.  





<a name="SYNTAX2"></a>

### Cambiar el texto seleccionado

`<Control used>.SelectedText = <Text to select>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- a control Campo de entrada,

- a Combo Box en modo edición,

- una columna de una control Tabla en modo edición.




**`<Text to select>: (Cadena de caracteres)`**

Texto que debe ser resaltado en el control. El texto reemplaza al texto que está actualmente seleccionado.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000056"></a>

- Si la control está en formato de texto enriquecido, el texto seleccionado no tiene formato.

- Si la control es una control de Unicode, la cadena devuelta es una cadena de Unicode.



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000064"></a>

El **SelectedText** Property está disponible para los siguientes controles:

- control Campo de entrada ,

- Combo Box editable,

- columna de texto de una control Tabla.

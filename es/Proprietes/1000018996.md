


## AutomaticTooltip (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **automatic Tooltip** se utiliza para activar o desactivar los tooltips automatic en: 

- Controles del cuadro de lista. 

- Controles de mesa.

- Controles de tabla TreeView. 

- Controles de TreeView.


**Recordatorio**: Los tooltips de automatic son tooltips que automatic ally aparecen tan pronto como se trunca el contenido de una celda o el contenido de un fila. Por lo tanto, el usuario puede ver el contenido completo del elemento sin tener que realizar ninguna operación específica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disables the automatic tooltip
TABLE_MyTable.AutomaticTooltip = False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si los consejos de la herramienta automatic están activados

`<Result> = <Control used>.AutomaticTooltip`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los tooltip están activados,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 


Observación: Las herramientas de automatic están habilitadas por Default.

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Tabla, 

- Control Tabla TreeView, 

- Control List Box, 

- Control TreeView.





<a name="SYNTAX2"></a>

### Activando o desactivando los consejos de la herramienta automatic

`<Control used>.AutomaticTooltip = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Tabla, 

- Control Tabla TreeView, 

- Control List Box, 

- Control TreeView.




**`<New value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar los tooltip automáticos,

- <u><u><u><u>False</u></u></u></u> para desactivarlos.




## Address (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Address** permite:

- Establecer una conexión entre el control Conferencia (y el flujo asociado) y el receptor. Esta propiedad equivale a [StreamConnect](../WDLang3/3039008.md).

- Obtener la dirección de la última solicitud de conexión saliente.




A partir de la versión 20, los controles Conferencia están disponibles en Windows Vista y versiones posteriores. Si desea utilizar el control Conferencia con una versión anterior de Windows: 

1. Abra la ventana de descripción del control. 

2. En la pestaña "UI", seleccione la opción "Utilizar las API TAPI 3.x / Windows XP
(para compatibilidad)".

3. Valide. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect stream to receiver
CONF_Conference1.Address = "123.101.7.56"
IF CONF_Conference1 <> 0 THEN Info("Communication established")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la dirección de la última conexión

`<Result> = <Conference control>.Address`
---

**`<Result>: (Cadena de caracteres)`**

Dirección de la última solicitud de conexión saliente. Esta dirección puede ser:

- Una dirección IP en formato XXX.XXX.XXX.XXX.XXXX (125.5.110.100, por ejemplo).

- Una dirección IP que contiene el nombre del servidor (pop3.freesbee.com, por ejemplo).




**`<Conference control>: (Nombre del control)`**

Nombre del control Conferencia asociado al flujo.  


<a name="SYNTAX2"></a>

### Conectar el flujo al receptor

`<Conference control>.Address = <Receiver address>`
---

**`<Conference control>: (Nombre del control)`**

Nombre del control Conferencia asociado al flujo.

**`<Receiver address>: (Cadena de caracteres)`**

Nueva dirección del receptor. Esta dirección puede ser:

- Una dirección IP en formato XXX.XXX.XXX.XXX.XXXX (125.5.110.100, por ejemplo).

- Una dirección IP que contiene el nombre del servidor (pop3.freesbee.com, por ejemplo). Se recomienda esta sintaxis.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para saber si la conexión se estableció correctamente:

- compruebe el valor de la variable [ErrorOccurred](../WDLang1/3087001.md).

- compruebe el valor del control Conferencia (propiedad [Value](../Proprietes/2510130.md)). Si &lt;Control Conferencia&gt;.Value = 0, la conexión no se estableció correctamente.

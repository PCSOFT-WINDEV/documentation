


## UndockablePane (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **UnlockablePane** Property se utiliza para: 

- Determinar si el usuario final puede mover los paneles de una Cinta o control Pestaña Dinámica fuera de la ventana. 

- Permitir o impedir que el usuario final mueva los paneles de una Cinta o control Pestaña Dinámica fuera de la ventana. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TAB_Dynamic.UndockablePane = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si todos los paneles de una Cinta o control Pestaña Dinámica pueden ser desacoplados

`<Result> = <Control used>.UndockablePane`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si todos los paneles del control se pueden desacoplar,

- <u><u><u><u>False</u></u></u></u> si los cristales del control no se pueden desacoplar. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Pestaña, 

- control Cinta de opciones.





<a name="SYNTAX2"></a>

### Permitir o impedir que se desacoplen todos los paneles de una cinta o de una control Pestaña dinámica

`<Control used>.UndockablePane = <Undockable or not>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Pestaña, 

- control Cinta de opciones.




**`<Undockable or not>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si todos los paneles del control deben ser inamovibles,

- <u><u><u><u>False</u></u></u></u> si los paneles del control no deben ser inamovibles. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el panel se puede desacoplar, se puede separar mediante Arrastrar y soltar desde el botón del panel fuera del área de botones.. 

- Un panel desanclado puede volver a anclarse arrastrándolo y soltándolo en la zona de botones de la pestaña o de control Cinta de opciones.

- Si la control Pestaña tiene anclajes horizontales o verticales, la ventana libre puede cambiar de tamaño. 

- Si la propiedad [WithClosingButton](../Proprietes/1000021374.md) tiene el valor <u><u><u><u>False</u></u></u></u>, la ventana libre no puede cerrarse.




## SearchValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SearchValue** Property se utiliza para get y cambiar el valor del campo de búsqueda en un Action Bar. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Event "Whenever modifying the search" of an Action Bar control
n = TableSearch(TABLE_Customer.COL_NAME, ACTB_ACTIONBAR.SearchValue)
IF n!= -1 THEN
	TABLE_Customer = n
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperando el valor de la búsqueda control

`<Result> = <Action Bar control>.SearchValue`
---

**`<Result>: (Cadena de caracteres)`**

Valor de la búsqueda control del Action Bar. 

**`<Action Bar control>: (Nombre del control)`**

Nombre del Action Bar control a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el valor de la búsqueda control

`<Action Bar control>.SearchValue = <Value>`
---

**`<Action Bar control>: (Nombre del control)`**

Nombre de la Action Bar control a utilizar.

**`<Value>: (Cadena de caracteres)`**

Nuevo valor para la búsqueda control.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La búsqueda control del Action Bar puede ser visualizada por [ActionBarSearchVisible](../WDLang1/1000021599.md).




## VisibleEndUser (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VisibleEndUser** Property se utiliza para:

- Averiguar si un artículo es Visible (o no) para los usuarios finales. Este punto se definió en la editor de análisis o a través de la programación.

- Definir si un elemento debe ser Visible a los usuarios finales (sólo cuando el elemento se describe a través de la programación).




Cuando el artículo es Visible para los usuarios finales:

- se puede acceder a este artículo en [Reports & Queries](../Presentation/3088003.md). Los usuarios finales podrán utilizar este elemento en sus propios informes y consultas personalizados.

- los valores de este elemento pueden modificarse en las diferentes [Ventanas HFSQL para la asistencia de automatic](../WDLang4/3044188.md) (si se produce un error de duplicación, por ejemplo). 




**Observación**: La visibilidad de un elemento se define en el editor al describir los elementos del archivo (opción"Visible to the end user").






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the visibility of the Orders.OrdDate item
OrdDate.VisibleEndUser = True
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si un artículo es Visible para el usuario final

`Result> = <Data file>.<Item>.VisibleEndUser`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el artículo es Visible a los usuarios finales, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado.  


<a name="SYNTAX2"></a>

### Definir la visibilidad de un elemento descrito mediante programación

`<Item>.VisibleEndUser = <Visibility>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Visibility>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el artículo debe ser Visible a los usuarios finales, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.

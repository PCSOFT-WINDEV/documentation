
## XMLSource (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **XMLSource** Property se utiliza para get el código fuente XML. Esta propiedad se puede utilizar con: 

- una variable de tipo [wsResponse](../WDLang3/1000020568.md). 

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

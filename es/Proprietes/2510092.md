


## VerticalOrientation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VerticalOrientation** Property se utiliza para:

- Descubra la orientación de una control ListView.

- Modificar la orientación de un control ListView.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the control orientation
LSV_MyListView.VerticalOrientation = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la orientación de una control ListView

`<Result> = <ListView control>.VerticalOrientation`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control está orientado verticalmente,

- <u><u><u><u>False</u></u></u></u> si el control está orientado horizontalmente.




**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.  


<a name="SYNTAX2"></a>

### Modificar la orientación de una control ListView

`<ListView control>.VerticalOrientation = <New orientation>`
---

**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.

**`<New orientation>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para orientar el control verticalmente,

- <u><u><u><u>False</u></u></u></u> para orientar el control horizontalmente.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **VerticalOrientation** sólo puede utilizarse en controles ListView.

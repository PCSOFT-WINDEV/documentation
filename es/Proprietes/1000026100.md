
## Start (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Start** se utiliza para obtener el punto inicial de un elemento o Event. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [EventTimeline](../WDLang1/1000020612.md). 

- una variable de tipo [edtHTMLSelection](../WDLang1/1000026093.md).

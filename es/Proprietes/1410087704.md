
## Camera (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Camera** permite identificar y cambiar la cámara utilizada (trasera o frontal). 

Observaciones: 

- Si se está grabando un video, no es posible cambiar la cámara. 

- Esta propiedad solo puede utilizarse en controles Cámara que no están en "Modo compatible con la versión 26". 

- Esta propiedad corresponde a la opción "Cámara" de la pestaña "General" de la ventana de descripción del control Cámara. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CAM_MyCamera.Camera = camBackCamera THEN
	CAM_MyCamera.Camera = camFrontCamera
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificar la cámara utilizada por un control Cámara

`<Result> = <Camera control>.Camera`
---

**`<Result>: (Constante de tipo Integer)`**

Cámara utilizada: 


|   |   |
| --- | --- |
| *camBackCamera* | Cámara trasera del dispositivo.   |
| *camFrontCamera* | Cámara frontal del dispositivo. |


 De forma predeterminada, se utiliza la cámara trasera. 

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.


<a name="SYNTAX2"></a>

### Cambiar la cámara utilizada por un control Cámara

`<Camera control>.Flash = <New camera>`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<New camera>: (Constante de tipo Integer)`**

Cámara utilizada: 


|   |   |
| --- | --- |
| *camBackCamera* | Cámara trasera del dispositivo.   |
| *camFrontCamera* | Cámara frontal del dispositivo. |


 De forma predeterminada, se utiliza la cámara trasera.

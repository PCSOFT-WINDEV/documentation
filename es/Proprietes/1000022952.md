
## IndentFirstLine (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **IndentFirstLine** Property se utiliza para get y cambiar la sangría del primer Line de un elemento. Esta propiedad se puede utilizar en:

- variables de tipo [docParagraph](../WDLang1/1000022483.md).

- variables de tipo [docNumberingLevel](../WDLang1/1000022799.md).

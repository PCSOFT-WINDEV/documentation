
## RightToLeft (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RightToLeft** Property se utiliza para get y cambiar la dirección de escritura de un elemento. Esta propiedad se puede utilizar en:

- a control, ventana o reporte. Para obtener más información, consulte la ayuda sobre la propiedad [RightToLeft](../Proprietes/1000017228.md). 

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).




## RecordSize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Record Size** se utiliza para obtener el tamaño (en bytes) de un Record en:

- un archivo de datos descrito en el editor de análisis,

- un archivo de datos describió a través de la programación,

- una vista HFSQL,

- a query.




Este tamaño se calcula ignorando las teclas compuestas.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the size of a record in the Order data file
ResSize = Order.RecordSize
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Size> = <Element used>.RecordSize`
---

**`<Size>: (Integro)`**

Tamaño de la Record (en bytes) mientras se ignoran las claves compuestas.

**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento para el que se solicita el tamaño de una Record:

- nombre de un archivo de datos descrito en el editor de análisis o a través de la programación,

- nombre de una vista HFSQL,

- nombre de una consulta.

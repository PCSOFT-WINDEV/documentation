


## Name55 (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Name55** permite manipular el nombre lógico de un archivo de datos en formato Hyper File 5.5 que no se ha migrado y que se encuentra en un análisis WINDEV 28. Es posible:

- Averigua el nombre lógico de un archivo de datos.

- Definir el nombre de un archivo de datos 5.5 creado mediante programación en un análisis WINDEV 28.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the logical name of the Customer data file
ResName55 = Customer.Name55
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre lógico de un archivo de datos en formato 5.5

`<Logical name> = <Data file>.Name55`
---

**`<Logical name>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos en formato 5.5.

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos en el análisis en formato WINDEV 28.  


<a name="SYNTAX2"></a>

### Definición del nombre lógico de un archivo de datos en formato 5.5 creado mediante programación

`<Data file>.Name55 = <New logical name>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos definido mediante programación a través del tipo [Expediente description](../WDLang4/1514065.md).

**`<New logical name>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos en formato 5.5.

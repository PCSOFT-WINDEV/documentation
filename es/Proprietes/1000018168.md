


## XAxisMin (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **XAxisMin** Property obtiene el valor mínimo del eje X en un control Gráfico. Esta propiedad se puede utilizar en el evento "Zoom" de un control Gráfico para identificar la sección mostrada.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe_axes2%201.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the lower bound on the X-axis
Info("The lower bound of the X-axis is " + CHART_MyChart.XAxisMin)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.XAxisMin`
---

**`<Result>: (Real)`**

Límite inferior del eje X.

**`<Chart control>: (Nombre del control)`**

Nombre del control Gráfico a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La propiedad **XAxisMin** solo se puede utilizar en controles Gráfico. Esta propiedad no se puede utilizar en los gráficos que se muestran en las ventanas o en los controles Imagen.

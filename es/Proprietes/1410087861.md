
## Unicode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Unicode** Property se utiliza para get y cambiar el formato Unicode de un elemento. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control. Para obtener más información, consulte [Unicode](../Proprietes/1000017001.md).

- una variable de tipo [pdfParameter](../WDLang6/1410087857.md).


## Enabled (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Enabled** activa o desactiva un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md).

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). 

- una variable de tipo [InfoVersion](../WDLang1/1000018905.md).

- una variable de tipo [gpwUser](../WDLang6/1000021088.md).

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [hScheduleMaterializedView](../WDLang4/1000021444.md).


## FontMinSize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **FontMinSize** se utiliza para obtener y cambiar el tamaño mínimo de la fuente utilizada en un control Estático con la opción "Reducir fuente" (pestaña "Detalles" de la ventana control Estático description). 

Recordatorio: Esta opción permite reducir el tamaño de la fuente para que se muestre todo el texto en el control Estático. La fuente mínima también se puede especificar en la pestaña "Detalles" de la descripción del control Estático. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Set minimum font size
STC_MyTitle.FontMinSize = 8
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Conocer el tamaño mínimo de la fuente utilizada en un control Estático

`<Result> = <Static control>.FontMinSize`
---

**`<Result>: (Integro)`**



- 0 si esta funcionalidad no está activada (valor predeterminado). En este caso, no se reduce la fuente. 

- Tamaño mínimo de la fuente. Este tamaño se expresa en la unidad especificada para el elemento (devuelta por la [SizeUnit](../Proprietes/1410088365.md) Property). 




**`<Static control>: (Nombre del control)`**

Nombre de la control Estático a utilizar.


<a name="SYNTAX2"></a>

### Modificar el tamaño mínimo de la fuente utilizada en un control Estático

`<Static control>.FontMinSize = <Font size>`
---

**`<Static control>: (Nombre del control)`**

Nombre de la control Estático a utilizar.

**`<Font size>: (Integro)`**



- 0 para desactivar la funcionalidad. En este caso, no se reduce la fuente. 

- Tamaño mínimo de la fuente. Este tamaño se expresa en la unidad especificada para el elemento (devuelta por la [SizeUnit](../Proprietes/1410088365.md) Property). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta propiedad no tiene ningún efecto en los controles Estático RTF.

- Si el texto es demasiado largo, se muestra una elipsis. 

- El tamaño máximo de la fuente se define con la propiedad [FontSize](../Proprietes/2510072.md). 

- El tamaño se expresa en la unidad especificada para el elemento (devuelto por la [SizeUnit](../Proprietes/1410088365.md) Property).

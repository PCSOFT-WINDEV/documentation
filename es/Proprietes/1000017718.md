


## ManageRTF (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**ManageRTF** se utiliza para configurar el modo de gestión del formato RTF en un índice de texto completo. Es posible:

- Averiguar si el texto de los elementos que se encuentran en un índice de texto completo puede contener RTF. El índice de texto completo se definió en el editor de análisis por medio de la programación.

- Definir si el texto de los elementos que se encuentran en un índice de texto completo puede contener RTF. El índice de texto completo se ha definido mediante programación.





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de gestión del RTF

`<Management mode> = <Data file>.<Full-text index>.ManageRTF`
---

**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto puede contener RTF,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si el texto puede contener RTF, las etiquetas RTF no se indexarán.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).  


<a name="SYNTAX2"></a>

### Definir el modo de gestión del RTF

`<Full-text index>.ManageRTF = <New mode>`
---

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido por el tipo [FullTextIndex Description](../WDLang4/1000017461.md).

**`<New mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto puede contener RTF,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si el texto puede contener RTF, las etiquetas RTF no se indexarán..

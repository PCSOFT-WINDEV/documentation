
## MaxLinkedCardinality (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxLinkedCardinality** Property se utiliza para set las características de un enlace. Es posible:

- Averiguar la cardinalidad máxima del elemento relacionado.

- Modificar la cardinalidad máxima del elemento relacionado.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cardinaliteReliee.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the maximum cardinality
LinkClubPerson.MaxLinkedCardinality = hInfiniteCardinality
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la cardinalidad máxima del elemento relacionado

`<Cardinality> = <Link>.MaxLinkedCardinality`
---

**`<Cardinality>: (Integro)`**

Cardinalidad máxima del elemento relacionado. La constante *hInfiniteCardinality* permite identificar la cardinalidad "n".

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en el editor de análisis o a través del tipo de [Descripción de la relación](../WDLang4/1514039.md).  


<a name="SYNTAX2"></a>

### Modificación de la cardinalidad máxima del elemento relacionado

`<Link>.MaxLinkedCardinality = <New cardinality>`
---

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en la editor de análisis o a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

**`<New cardinality>: (Integro)`**

Nueva cardinalidad máxima para el elemento relacionado. La constante *hInfiniteCardinality* permite definir la cardinalidad "n". Por defecto, cuando un enlace es set a través de la programación, la cardinalidad máxima del elemento relacionado es 1.


## Page (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Page** permite manipular una página. Esta propiedad se puede utilizar en:

- una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfDocument](../WDLang6/1000024909.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Página WEBDEV control. Esta Property le permite manipular la página cargada en la WEBDEV Página control, sin salir de la ventana. Para obtener más información, consulte [Page](../Proprietes/1410089371.md).

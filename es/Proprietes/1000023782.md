


## TokenEnabled (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **TokenEnabled** determina si un control Campo de entrada es de tipo "Tokens de texto". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF EDT_Email.TokenEnabled = True THEN 
	EDT_Email.TokenListSeparator = ";.:" + TAB
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si el control Campo de entrada es de tipo "Tokens de texto"

`<Result> = <Edit control>.TokenEnabled`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Campo de entrada es de tipo "Tokens de texto".

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Edit control>: (Nombre del control)`**

Nombre del control Campo de entrada utilizado. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando la propiedad **TokenEnabled** se establece como <u><u><u><u>True</u></u></u></u>, si el usuario introduce un carácter separador o un retorno de carro (CR), se agrega automáticamente un token y el usuario puede continuar introduciendo valores en el control.

Un token es un bloque de texto que permite al usuario distinguir los diferentes elementos introducidos en un control. Por ejemplo, los tokens se utilizan para identificar fácilmente los diferentes destinatarios de un correo electrónico.

El valor devuelto o almacenado por un control Campo de entrada de tipo "Tokens de texto" es una cadena separada por retornos de carro (CR).
Por ejemplo: paula@email1.com&lt;CR&gt;mar@email2.com

Los separadores que desencadenan la creación de nuevos tokens en un control Campo de entrada se definen mediante la propiedad [TokenListSeparator](../Proprietes/1000023783.md). 

Para permitir o no el uso de duplicados, utilice la propiedad [TokenAllowDuplicate](../Proprietes/1000023784.md). 

Puede añadir sugerencias de autocompletado de tokens con la función [AssistedInputAdd](../WDLang1/1000020043.md).

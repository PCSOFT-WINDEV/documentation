
## SourceKey (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SourceKey** Property se utiliza para set las características de un enlace. Es posible:

- Averigüe el elemento del archivo de datos de origen utilizado en el enlace.

- Modificar el elemento del archivo de datos de origen utilizado en el enlace.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cleSource.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the source key
LinkClubPerson.SourceKey = Club.ClubID
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la clave de origen de un enlace

`<Key> = <Link>.SourceKey`
---

**`<Key>: (Cadena de caracteres)`**

Nombre de la clave utilizada por el enlace en el archivo de datos de la fuente.

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en el editor de análisis o a través del tipo de [Descripción de la relación](../WDLang4/1514039.md).  


<a name="SYNTAX2"></a>

### Modificar la clave de origen de un enlace

`<Link>.SourceKey = <New key>`
---

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en la editor de análisis o a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

**`<New key>: (Cadena de caracteres)`**

Nombre de la nueva clave utilizada por el enlace en el archivo de datos de origen. Esta clave fue definida en el editor de análisis o a través del tipo de [Artículo description](../WDLang4/1514071.md).

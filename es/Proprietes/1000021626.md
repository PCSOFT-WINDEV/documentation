


## DynamicTab (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DynamicTab** Property se utiliza para: 

- Averigüe el tipo de control Pestaña (pestaña estática o pestaña dinámica). 

- Modificar el tipo de control Pestaña (pestaña estática o pestaña dinámica). 




**Recordatorio**: Un control Pestaña dinámico es un control Pestaña que soporta la adición, la eliminación y el movimiento de los paneles.

Observación: Esta Property sustituye a la [TabMDI](../Proprietes/1000021471.md) Property.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the tab is a dynamic tab
IF TAB_Tab1.DynamicTab = True THEN
	// Close all the panes
	TabCloseAll(TAB_Tab1)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si un control Pestaña es una ficha dinámica

`<Result> = <Tab control>.DynamicTab`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Pestaña es una pestaña dinámica, 

- <u><u><u><u>False</u></u></u></u> si el control Pestaña es una pestaña estática. 




**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificar el tipo de un control Pestaña

`<Tab control>.DynamicTab = <Type of Tab control>`
---

**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. 

**`<Type of Tab control>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para obtener una dinámica control Pestaña, 

- <u><u><u><u>False</u></u></u></u> para obtener un control Pestaña estático . 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### control Pestaña dinámica
<a name="control_pestana_dinamica_ELTPARAGRAPHE000065"></a>

Si la propiedad **DynamicTab** se establece como <u><u><u><u>True</u></u></u></u>: 

- las funciones de [TabXXX](../WDLang1/1000021391.md) pueden llamarse en control.

- el operador [] de la ficha debe llamarse con el alias del panel. 

- el usuario final puede: 

	- eliminar los paneles si [WithClosingButton](../Proprietes/1000021374.md) se establece en <u><u><u><u>True</u></u></u></u>. 

	- añadir paneles si [ConNuevoBotón](../Proprietes/1000021377.md) se establece en <u><u><u><u>True</u></u></u></u>.

	- modificar el orden de los paneles mediante arrastrar y soltar. 

	- desacoplar los paneles si [UndockablePane](../Proprietes/1000021382.md) se establece en <u><u><u><u>True</u></u></u></u>.

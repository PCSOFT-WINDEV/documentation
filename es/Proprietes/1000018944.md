
## Out (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Outside** Property se utiliza para saber si un Email fue leído correctamente en el servidor. 

Esta propiedad se puede utilizar en:

- la estructura [Email](../WDLang3/3032029.md).

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [pvtPosition](../WDLang1/1000020726.md).

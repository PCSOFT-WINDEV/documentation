


## EmptyIfZero (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **EmptyIfZero** Property se utiliza para: 

- Averigua el modo de gestión de valor cero en un control.  

- Modificar el modo de gestión de valor cero en un control. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_Edit1 = 0
IF CBOX_ALL = True THEN
	EDT_Edit1.EmptyIfZero = False // Displays "0,00"
ELSE
	EDT_Edit1.EmptyIfZero = True // Displays ""
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de gestión de valor cero

`<Result> = <Control used>.EmptyIfZero`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control está vacío cuando su valor corresponde a cero,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Campo de entrada numérico, 

- Columna Check Box en una tabla o en una Tabla TreeView.

- Combo box o combo box que muestra una tabla. 

- Control Estático.





<a name="SYNTAX2"></a>

### Modificación del modo de gestión de valor cero

`<Control used>.EmptyIfZero = <Management mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Campo de entrada numérico, 

- Columna Check Box en una tabla o en una Tabla TreeView.

- Combo box o combo box que muestra una tabla. 

- Control Estático.




**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control está vacío cuando su valor corresponde a cero,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **EmptyIfZero** puede utilizarse en: 

- controles de edición numérica. 
	Cuando una control Campo de entrada numérica muestra el valor 0, puedes mostrar el valor o dejar la control vacía. 
	La propiedad **EmptyIfZero** corresponde a la opción "Restablecer si es cero", en la pestaña "Detalles" de la ventana numérica control Campo de entrada description.

- Columnas de Check Box de sólo lectura en tablas y Tablas TreeView. 
	La casilla se puede mostrar (o no) si no está marcada. 
	La propiedad **EmptyIfZero** corresponde a la opción "Ocultar la marca de verificación si no está marcada (sólo lectura)", en la pestaña "Detalles" de la ventana de la columna de casillas de verificación description.

- Combo Box controla la visualización de una lista o tabla. 

- Controles estáticos.

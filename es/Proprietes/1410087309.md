*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FileSize** Property se utiliza para get el tamaño de un archivo. Esta propiedad está disponible en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoDatabaseInfo](../WDLang4/1000022412.md).

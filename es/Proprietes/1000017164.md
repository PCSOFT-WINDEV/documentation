


## Description (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **description** se utiliza para obtener y cambiar la página description. La description puede ser truncada cuando se muestra en el resultado de la búsqueda. La description también puede introducirse desde el editor Page.

En las páginas WEBDEV activas, la propiedad **description** se utiliza para introducir dinámicamente el description associated con el Page (basándose en el contenido del Page).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the description of the page according to its content
MyPage.Description = "Online boat sales"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar el description associated con un Page

`<Page description> = <Page used>.Description`
---

**`<Page description>: (Cadena de caracteres)`**

Frase utilizada para describir la Page.

**`<Page used>: (Nombre de la página)`**

Nombre de la Page a analizar.  


<a name="SYNTAX2"></a>

### Modificar el description associated con un Page

`<Page used>.Description = <Page description>`
---

**`<Page used>: (Nombre de la página)`**

Nombre de la página para la que debe definirse la description.

**`<Page description>: (Cadena de caracteres)`**

description que debe ser associated con el Page. Esta frase debe animar a los internautas a visitar la página web de Page. Esta cadena debe cumplir con las siguientes reglas:

- Sin carácter CR (Carriage Return)

- Ser breve (hasta 200 caracteres)






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### description associated con un Page
<a name="description_associated_con_page_ELTPARAGRAPHE000045"></a>

El description asociado a una página se escribe en el encabezado HTML de la página (&lt;META NAME="description" CONTENT="..."&gt; tag). Este description es usado por los motores de búsqueda: este valor se muestra como la página description en la lista de resultados de la búsqueda.

El description debe definirse en el código de inicialización del Page. Además, la descripción debe definirse en páginas a las que se puede acceder a través de una URL en el siguiente formato: "http://".

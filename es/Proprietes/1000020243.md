
## Label (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Label** permite obtener y definir la etiqueta de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [EmailAddress](../WDLang3/1000020244.md).

- una variable de tipo [SIPAddress](../WDLang3/1000020245.md).

- una variable de tipo [WebAddress](../WDLang3/1000020246.md).

- una variable de tipo [Event](../WDLang3/1000020248.md). 

- una variable de tipo [InstantMessaging](../WDLang3/1000020247.md). 

- una variable de tipo [Phone](../WDLang3/1000020241.md).




## PageEndStick (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El bloque **PageEndStick** Property se utiliza para gestionar la posición del bloque Page Footer. El bloque Page Footer puede ser:

- pegado al bloque anterior, sin que el pie de página se encuentre necesariamente en la parte inferior de la página impresa. El pie de página se puede utilizar para realizar cálculos y visualizarlos en la parte inferior del bloque que muestra el tabla.

- no pegado al bloque anterior, el pie de página se imprimirá como de costumbre (en la parte inferior de la página). Este es el modo de gestión de Default del pie de página.




El **PageEndStick** Property se utiliza para:

- Averigüe si el pie de página debe estar pegado al último bloque.

- Modificar el modo de gestión del bloque Pie de página.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out how the page footer is managed
IF MyReport.PageEndStick = True THEN
	Trace("Page footer stuck to the previous block")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de gestión del pie de página

`<Current management mode> = <Report used>.PageEndStick`
---

**`<Current management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el pie de página está pegado al bloque anterior,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte utilizada.  


<a name="SYNTAX2"></a>

### Modificación del modo de gestión del pie de página

`<Report used>.PageEndStick = <New management mode>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte utilizada.

**`<New management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el pie de página debe estar pegado al bloque anterior,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **PageEndStick** Property se puede utilizar en cualquier parte del código del reporte, tanto en modo de lectura como de escritura.

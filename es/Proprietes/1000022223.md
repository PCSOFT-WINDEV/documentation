
## AuthToken (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **AuthToken** puede corresponder a: 

- una variable de tipo [AuthToken](../WDLang3/1000022220.md). 

- una propiedad de una variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- una propiedad de una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md). 

- una propiedad de una variable de tipo [emailSMTPSession](../WDLang3/1000018765.md).




## Month (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Mes** Property se utiliza para:

- Recuperar el mes de una Variable de tipo [salir con](../Motscles/1514066.md) o [FechaHora](../Motscles/1514070.md).

- Modificar el mes de una Variable de tipo [salir con](../Motscles/1514066.md) o [FechaHora](../Motscles/1514070.md).

- Obtener el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora"). 
	

- Modificar el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora"). 
	




**Observación**: El **Month** Property se utiliza para cambiar fácilmente las fechas (añadir un mes, etc.).

**Recordatorio**: Los elementos de fecha se utilizan para gestionar:

- Fechas "Año - Mes - Día" (formato AAAAMMDD)

- "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" fechas y horas (formato AAAAMMDDHHmmSSCCC)

- "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" duraciones (formato +DHHMMSSCCC)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Operations on a Date variable
StartDate is Date = "20011212"
// Add 1 month to the date
StartDate.Month++
// Modify the month
StartDate.Month = 5
```


<a name="Example2"></a>

```wl
// Operations on an item
Work.StartDate = "20011212"
// Add 1 month to the date
Work.StartDate.Month++
// Modify the month
Work.StartDate.Month = 5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el mes en una Fecha o DateTime Variable

`<Result> = <Date>.Month`
---

**`<Result>: (Integro)`**

Mes a 2 dígitos.

**`<Date>: (Fecha o Fecha-Hora)`**

Nombre de la Variable de tipo [salir con](../Motscles/1514066.md) o [FechaHora](../Motscles/1514070.md) que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del mes en una fecha o fecha-hora Variable

`<Date>.Month = <New month>`
---

**`<Date>: (Fecha o Fecha-Hora)`**

Nombre de la Variable de tipo [salir con](../Motscles/1514066.md) o [FechaHora](../Motscles/1514070.md) que se va a utilizar.

**`<New month>: (Entero o cadena de caracteres)`**

Nuevo mes expresado en números (incluidos entre 1 y 12). Reemplaza el mes en la fecha especificada.  


<a name="SYNTAX3"></a>

### Conocer el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora")

`<Result> = <Data file>.<Item>.Month`
---

**`<Result>: (Integro)`**

Mes a 2 dígitos.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX4"></a>

### Modificar el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora")

`<Data file>.<item>.Month = <New month>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<New month>: (Entero o cadena de caracteres)`**

Nuevo mes expresado en números (incluidos entre 1 y 12). Reemplaza el mes en la fecha especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Intervalo de fechas
<a name="intervalo_fechas_ELTPARAGRAPHE000162"></a>

Los tipos [salir con](../Motscles/1514066.md) y [FechaHora](../Motscles/1514070.md) se utilizan para gestionar las fechas del 01/01/0001 al 31/12/9999.
<a name="NOTE0_2"></a>


### Gestión de los meses y años
<a name="gestion_los_meses_anos_ELTPARAGRAPHE000175"></a>

**Caso 1: Asignación directa**
Durante una asignación directa (por ejemplo, *MyWork.MyDate.Month = n*), el mes debe estar comprendido entre 01 y 12. Se produce un error de WLanguage si se especifica un mes no válido..
La notación *MiTrabajo.MiFecha.Mes = MiTrabajo.MiFecha.Mes + 5* puede generar un error en tiempo de ejecución. Por ejemplo, las siguientes líneas de código provocan un error:


```wl
// Code triggering the error
MyDate is Date = "20201126"	// 11/11/2020 
MyDate.Month = MyDate.Month + 2 	
// Triggers a WLanguage error because the month is equal to 13
 
// Correct code
MyDate is Date = "20201126"	// 11/11/2020
MyDate.Month += 2
```


```wl
// Code triggering the error
MyDate is Date = "20201126"	// 11/11/2020
MyDate1 is Date 
MyDate1.Month = MyDate.Month + 2	
// Triggers a WLanguage error because the month is equal to 13
 
// Correct code
MyDate1 = MyDate   // 11/26/2004 
MyDate1.Month += 2
```

**Caso 2: Operaciones en los meses**
Al realizar operaciones en las fechas, el cambio de año se gestiona automáticamente.. Por lo tanto, si el número de meses es superior a 12, el año se modifica automáticamente y el número de meses vuelve a empezar a partir de 1.
Por ejemplo:


```wl
StartDate is Date = "20201226"   // 26/12/2020
// Add 5 months to the date
StartDate.Month += 5        // StartDate is "20210526"
```

<a name="NOTE0_3"></a>


### Operadores disponibles para los meses
<a name="operadores_disponibles_para_los_meses_ELTPARAGRAPHE000198"></a>

Los siguientes operadores aritméticos pueden utilizarse con el **Month** Property:

- ++ y --

- += y - =



```wl
StartDate is Date = "20201126"   
StartDate.Month++               // Add 1 month to the date
StartDate.Month += 5       // Add 5 months to the date
StartDate.Month -= 5        // Subtract 5 months from the date
```

<a name="NOTE0_4"></a>


### Cálculo del último día del mes
<a name="calculo_del_ultimo_dia_del_mes_ELTPARAGRAPHE000212"></a>

Para obtener el último día de un mes, basta con asignar 31 a la [jornada](../Proprietes/2512043.md) Property de la fecha. El último día se calculará automáticamente según el mes especificado.


```wl
StartDate is Date = "20201126"   
StartDate.Day = 31    
// November does not have 31 days
// The day is automatically replaced with 30
```

<a name="NOTE0_5"></a>


### Cálculo de un vencimiento a 90 días de fin de mes
<a name="calculo_vencimiento_90_dias_fin_mes_ELTPARAGRAPHE000224"></a>

El [jornada](../Proprietes/2512043.md) Property puede utilizarse para calcular un vencimiento de 90 días a final de mes.


```wl
MyDate is Date = "20201126"   
MyDate.Day += 90    
MyDate.Day = 31
```

<a name="NOTE0_6"></a>


### Cálculo del final de un período de 30 días
<a name="calculo_del_final_periodo_30_dias_ELTPARAGRAPHE000236"></a>

Un período de 30 días corresponde a un período de un mes a partir de una fecha determinada. Las propiedades **Month** y [jornada](../Proprietes/2512043.md) pueden utilizarse para calcular fácilmente la fecha de finalización de un período de 30 días.


```wl
StartDate is Date = "20201126"   
EndDate is Date = StartDate
EndDate.Month++    
EndDate.Day--
```


## LinkedKey (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LinkedKey** Property se utiliza para set las características de un enlace. Es posible:

- Averigua el elemento del archivo de datos de relacionado usado en el enlace.

- Modificar el elemento del archivo de datos de relacionado utilizado en el enlace.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cleReliee.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the linked key
LinkClubPerson.LinkedKey = Person.ClubID
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la clave de relacionado del enlace

`<Key name> = <Link name>.LinkedKey`
---

**`<Key name>: (Cadena de caracteres)`**

Nombre de la clave utilizada por el enlace en el archivo de datos de relacionado.

**`<Link name>: (Cadena de caracteres)`**

Nombre del enlace definido en el editor de análisis o a través del tipo de [Descripción de la relación](../WDLang4/1514039.md).  


<a name="SYNTAX2"></a>

### Modificando la clave de relacionado del enlace

`<Link name>.LinkedKey = <New key>`
---

**`<Link name>: (Cadena de caracteres)`**

Nombre del enlace definido en la editor de análisis o a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

**`<New key>: (Cadena de caracteres)`**

Nombre de la nueva clave usada por el enlace en el archivo de datos de relacionado. Esta clave fue definida en el editor de análisis o a través del tipo de [Artículo description](../WDLang4/1514071.md).




## ScrollWithFinger (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ScrollWithFinger** Property se utiliza para: 

- Determinar si el contenido de un control se puede mover con el dedo ("dedo Scrolling").

- Activar o desactivar la respuesta táctil para una control. 




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta Property se corresponde con la opción "Mover con el dedo" disponible en la pestaña "Detalles" de los controles de List Box, Table y Looper.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_TouchInterface = True THEN
	TABLE_CustomerTable.ScrollWithFinger = True
ELSE
	TABLE_CustomerTable.ScrollWithFinger = False
END
```
<a name="Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si una control permite el movimiento de los dedos

`<Result> = <Control used>.ScrollWithFinger`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control permite el movimiento de los dedos, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- a control List Box.

- un control Tabla.

- un control Looper. 





<a name="SYNTAX2"></a>

### Activar o desactivar el movimiento de los dedos

`<Control used>.ScrollWithFinger = <Finger scrolling>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- a control List Box.

- un control Tabla.

- un control Looper. 




**`<Finger scrolling>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control permite el dedo Scrolling, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Límite
<a name="limite_ELTPARAGRAPHE000109"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **ScrollWithFinger** Property sólo se puede utilizar en los controles List Box, Table y Looper. 
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000133"></a>

En algunos sistemas, un desplazamiento con el dedo de automatic (suministrado por el sistema) puede ser activado incluso si el desplazamiento con el dedo está desactivado para el control.


## Element (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Element** permite manipular los elementos de un documento Procesador de texto. Esta propiedad se puede utilizar en:

- variables de tipo [docParagraph](../WDLang1/1000022483.md).

- variables de tipo [docFragment](../WDLang1/1000022482.md).

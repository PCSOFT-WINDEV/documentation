


## ubicación

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Posición** palabra clave puede corresponder a:

- la función **[ubicación](../WDLang1/3024027.md)**, que se utiliza para encontrar la posición de una cadena de caracteres dentro de otra cadena de caracteres.

- el **Position** Property, que puede ser usado en:

	- una variable de tipo [Address](../WDLang3/1000019944.md).

	- una variable de tipo [Etiqueta](../WDLang3/1000019940.md).

	- una variable de tipo [Watermark](../WDLang5/1000019922.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docTabulation](../WDLang1/1000022573.md). 

	- una variable de tipo [MapImage](../WDLang3/1000025523.md).

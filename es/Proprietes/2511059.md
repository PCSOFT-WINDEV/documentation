


## NbLabelCopy (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbLabelCopy** Property se utiliza para:

- get el número de copias de etiquetas idénticas.

- Cambiar el número de copias de etiquetas idénticas.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print 5 copies of each label found in "RPT_Report1"
RPT_Report1.NbLabelCopy = 5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el número de copias de etiquetas idénticas

`<Result> = <Report used>.NbLabelCopy`
---

**`<Result>: (Integro)`**

Número de copias de etiquetas idénticas.

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el número de copias de etiquetas idénticas

`<Report used>.NbLabelCopy = <Number of copies>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.

**`<Number of copies>: (Integro)`**

Número de copias de etiquetas idénticas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Cola de impresión
<a name="cola_impresion_ELTPARAGRAPHE000058"></a>

Utilizando el **NbLabelCopy** Property añade elementos a la cola de impresión de la impresora utilizada.

Por ejemplo, si las 5 etiquetas que se encuentran en un reporte se imprimen 10 veces cada una, la cola de impresión contendrá un reporte con 50 etiquetas.

Para evitar la sobrecarga de la cola de impresión, se recomienda imprimir varias veces el mismo reporte ([NbCopy](../Proprietes/2511042.md) Property).

En nuestro ejemplo anterior, la reporte se imprimirá 10 veces. La cola de impresión contendrá 10 informes con 5 etiquetas.
<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000076"></a>

El **NbLabelCopy** Property sólo se aplica a los informes de "Etiqueta".


## InternalPagePopup (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **InternalPagePopup** Property se utiliza para obtener y modificar el nombre de la página interna utilizada para personalizar el contenido de los popups de Marker en una control Mapa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MAP_Map.InternalPagePopup = "IPAGE_Popup"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Conseguir que la página interna utilizada para personalizar los popups de la Marker en la control Mapa

`<Result> = <Map control>.InternalPagePopup`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la página interna utilizada para el popup,

- Cadena vacía ("") si la ventana Popup utiliza una visualización predeterminada.




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar.   


<a name="SYNTAX2"></a>

### Modificando la página interna utilizada para personalizar los popups de la Marker en la control Mapa

`<Map control>.InternalPagePopup = <New internal page>`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<New internal page>: (Nombre de la página)`**



- Nombre de la página interna utilizada para el popup,

- Cadena vacía ("") si la ventana Popup utiliza una visualización predeterminada.










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se ha especificado un nombre de página interna para el **InternalPagePopup** Property, la página interna se cargará cada vez que se muestre una ventana emergente en el control Mapa. La variable [Marker](../WDLang3/1000019940.md) que corresponde a la ventana emergente mostrada se pasa a la Event "Declaraciones globales" de la página interna a abrir. Este parámetro es pasado por reference y es considerado como un global de Variable a la página.

- La página interna especificada se abre en el modo "sólo pantalla".: el usuario no puede interactuar con los controles de la página interna.




## VisibleRange (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VisibleRange** Property se utiliza para get y cambiar el ancho de la pista Control deslizante de rango. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RangeSlider_Etendue.gif)
 La gama Visible también se puede configurar: 

- en la ventana de description de la Control deslizante de rango (pestaña "General"). 

- por el usuario cuando modifica las extremidades de la caja de la barra de desplazamiento. En este caso, la "Modificación del rango" Event del rango Control deslizante se ejecuta automáticamente. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the properties of the Range Slider control
EDT_Value = Myself.Value
EDT_LOWER_BOUND = Myself.MinValue
EDT_UPPER_BOUND = Myself.MaxValue
EDT_TOTAL_RANGE = Myself.TotalRange
EDT_VISIBLE_RANGE = Myself.VisibleRange
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubriendo la gama Visible de la Control deslizante de rango

`<Result> = <Range Slider control>.VisibleRange`
---

**`<Result>: (Integro)`**

Visible de la gama Control deslizante.

**`<Range Slider control>: (Nombre del control)`**

Nombre de la Control deslizante de rango utilizada.


<a name="SYNTAX2"></a>

### Modificando la gama de Visible de Control deslizante de rango

`<Range Slider control>.VisibleRange = <New visible range>`
---

**`<Range Slider control>: (Nombre del control)`**

Nombre de la Control deslizante de rango utilizada.

**`<New visible range>: (Integro)`**

La nueva gama Visible de la Control deslizante de rango. 
**Observación**: Para una Gama horizontal Control deslizante, sólo se modifica el extremo derecho. Para un Range Control deslizante vertical, sólo se modifica la extremidad inferior.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se modifica el rango visible de un Rango Control deslizante mediante la propiedad **VisibleRange**, no se ejecuta la "modificación de rango" Event del Rango Control deslizante.

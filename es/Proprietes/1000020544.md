
## Compression (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Compresión** Property se utiliza para get y cambiar el modo de compresión de datos. 

Esta propiedad se puede utilizar en:

- una conexión (HFSQL Client/Server,....). Para obtener más información, consulte [Compresión](../Proprietes/1000017276.md).

- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md).

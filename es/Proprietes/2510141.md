


## HelpNumber (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **HelpNumber** permite obtener y modificar el número de ayuda asociado al control. Este número corresponde al control "Número de ayuda" de la pestaña "Ayuda" de la ventana del control description.

Este número de ayuda corresponde al número de la página de ayuda que se mostrará cuando el usuario abra la ayuda contextual del control (pulsando F1, o haciendo clic en "?" en la barra de menús y haciendo clic en el control).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF Mode = "Add" THEN
	// Display the help for "Add"
	BTN_Button1.HelpNumber = 12
ELSE
	// Display the help for "Modify"
	BTN_Button1.HelpNumber = 48
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de ayuda associated con un control

`<Help number> = <Control used>.HelpNumber`
---

**`<Help number>: (Integro)`**

Ayuda al número associated con el control.

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.  


<a name="SYNTAX2"></a>

### Modificando el número de ayuda associated con un control

`<Control used>.HelpNumber = <New help number>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.

**`<New help number>: (Integro)`**

Nuevo número de ayuda associated con el control.

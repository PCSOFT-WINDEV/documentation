


## HTMLAfter (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HTMLAfter** Property se utiliza para get y cambiar el código HTML insertado después del control.

**Observación**: El código HTML generado después del control puede ser introducido en la pestaña "Advanced" de la ventana description del control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the style of a button dynamically
BTN_DynButton.HTMLBefore = "<DIV style=""..."">"
BTN_DynButton.HTMLAfter = "</DIV>"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el contenido HTML insertado después de la control

`<HTML content> = <Control used>.HTMLAfter`
---

**`<HTML content>: (Cadena de caracteres)`**

Contenido HTML insertado después de la control.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando el contenido HTML insertado después de la control

`<Control used>.HTMLAfter = <HTML content>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<HTML content>: (Cadena de caracteres)`**

El contenido HTML insertado después de la control.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta Property está disponible en todos los controles tras los cuales se puede insertar código HTML. Este Property no está disponible para los siguientes controles:

- Control HTML,

- Control Tabla.

- Control Looper.

- Control Tabla TreeView.

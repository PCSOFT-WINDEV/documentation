


## RepeatableReadSupported (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RepeatableReadSupported** Property se utiliza para: 

- Averigüe si el archivo de datos (archivo definido en la editor de análisis o mediante programación) puede utilizarse en HFSQL Client/Server en una transacción que incluya un modo de aislamiento "Lectura repetida".

- Permitir el uso de una transacción que incluya un modo de aislamiento de "Lectura Repetida" para un archivo de datos (cuando el archivo se describe sólo a través de la programación).




Atención: si esta opción está activada, el archivo no es compatible con las versiones 18 y anteriores.

Esta propiedad corresponde a la opción "Permitir transacciones REPEATABLE READ (requiere HFSQL versión 19 o posterior)" de la pestaña "Detalles" de la ventana del fichero description en el análisis. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RepeatableReadSupport%E9%20-%20HC%20N%B0001.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
If Customer.RepeatableReadSupported = False THEN Info("Unable to use the Repeatable Read mode")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si el archivo puede ser usado en modo "LECTURA REPETIBLE".

`<Result> = <Data file>.RepeatableReadSupported`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo puede ser usado en modo "LECTURA REPETIBLE",

- <u><u><u><u>False</u></u></u></u> en caso contrario.  




**`<Data file>: (Nombre del archivo)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definir si el archivo puede ser usado en modo "LECTURA REPETIBLE".

`<Data file>.RepeatableReadSupported = <"REPEATABLE READ" Mode>`
---

**`<Data file>: (Nombre del archivo)`**

Nombre del archivo de datos utilizado. Este nombre se definió con el tipo [Expediente description](../WDLang4/1514065.md).

**`<"REPEATABLE READ" Mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo puede ser usado en modo "LECTURA REPETIBLE",

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## DisplayedItemImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DisplayedItemImage** Property se utiliza para:

- get el elemento que corresponde al Image mostrado en un control ListView.

- Cambia el elemento que corresponde a la Image que aparece en un control ListView.


Observación: Este artículo puede venir de un archivo de datos, una vista o una consulta.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the item displayed in the "LSV_CustomerList" control
LSV_CustomerList.DisplayedItemImage = Customer.Photo
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el artículo que corresponde a la Image mostrada

`<Displayed item> = <ListView control>.DisplayedItemImage`
---

**`<Displayed item>: (Cadena de caracteres)`**

Nombre del artículo correspondiente al Image que aparece en el control ListView.

**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.  


<a name="SYNTAX2"></a>

### Cambiar el elemento que corresponde a la Image mostrada

`<ListView control>.DisplayedItemImage = <New item>`
---

**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.

**`<New item>: (Cadena de caracteres)`**

Nombre del nuevo elemento que se usará para mostrar la Image de control ListView. Este elemento debe pertenecer a la el archivo de datos buscada por el control (devuelta o modificada por la [BrowsedFile](../Proprietes/2510040.md) Property).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Image y leyenda
<a name="image_leyenda_ELTPARAGRAPHE000048"></a>

El **DisplayedItemImage** Property se utiliza para get y cambiar sólo el elemento que corresponde a las imágenes mostradas en el control.

Para obtener y cambiar el elemento que corresponde a los subtítulos de los elementos mostrados en el control, utilice la función [DisplayedItem](../Proprietes/2510123.md) Property.
<a name="NOTE0_2"></a>


### Manejo de los controles ListView a través de las propiedades
<a name="manejo_los_controles_listview_traves_las_propiedades_ELTPARAGRAPHE000062"></a>

- Los registros mostrados en una navegación control ListView provienen de el archivo de datos o de la consulta especificada por el [BrowsedFile](../Proprietes/2510040.md) Property.

- El sentido de la ordenación se define mediante la [BrowsedItem](../Proprietes/2510111.md) Property.

- Los registros pueden filtrarse mediante la función [filtros](../Proprietes/2510042.md) Property o [HFilter](../WDLang4/3044100.md).



<a name="NOTE0_3"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000083"></a>

La propiedad **DisplayedItemImage** sólo se aplica a los controles ListView.


## AuthURL (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **URLAuth** Property se utiliza para get y definir la URL de autorización que se utilizará (primera URL de la especificación OAuth 2.0). Esta propiedad se puede utilizar en: 

- una variable de tipo [OAuth2Parameters](../WDLang3/1000022212.md). 

- una variable de tipo [gpwOAuthParameters](../WDLang6/1000024751.md).


## InitialTool (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **InitialTool** permite obtener o cambiar la herramienta de dibujo seleccionada de forma predeterminada en un control Editor de imágenes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IE_ImageEditor.InitialTool = toolArrow
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener la herramienta de dibujo inicial del control Editor de imágenes

`<Result> = <Image Editor control>.InitialTool`
---

**`<Result>: (Constante de tipo Integer)`**

Herramienta seleccionada de forma predeterminada en el control Editor de imágenes: 



|   |   |
| --- | --- |
| *toolArrow* | La herramienta flecha se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolBlur* | La herramienta desenfoque se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolCrop* | La herramienta recorte se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolEllipse* | La herramienta elipse se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolEraser* | La herramienta borrador se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolFill* | La herramienta relleno se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolLine* | La herramienta línea se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolOCR* | La herramienta OCR se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolPencil* | La herramienta lápiz se seleccionada de forma predeterminada en el control Editor de imágenes. |
| *toolRectangle* | La herramienta rectángulo se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolText* | La herramienta texto se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolWatermark* | La herramienta marca de agua se selecciona de forma predeterminada en el control Editor de imágenes. |



**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.


<a name="SYNTAX2"></a>

### Cambiar la herramienta de dibujo inicial en un control Editor de imágenes

`<Image Editor control>.InitialTool = <Default tool>`
---

**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.

**`<Default tool>: (Constante de tipo Integer)`**

Herramienta a seleccionar de forma predeterminada en el control Editor de imágenes: 



|   |   |
| --- | --- |
| *toolArrow* | La herramienta flecha se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolBlur* | La herramienta desenfoque se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolCrop* | La herramienta recorte se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolEllipse* | La herramienta elipse se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolEraser* | La herramienta borrador se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolFill* | La herramienta relleno se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolLine* | La herramienta línea se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolOCR* | La herramienta OCR se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolPencil* | La herramienta lápiz se seleccionada de forma predeterminada en el control Editor de imágenes. |
| *toolRectangle* | La herramienta rectángulo se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolText* | La herramienta texto se selecciona de forma predeterminada en el control Editor de imágenes. |
| *toolWatermark* | La herramienta marca de agua se selecciona de forma predeterminada en el control Editor de imágenes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para cambiar las características predeterminadas del control Editor de imágenes, también puede utilizar las propiedades [InitialWidth](../Proprietes/2510067.md) y [InitialHeight](../Proprietes/2510043.md) para establecer el tamaño de la imagen creada de forma predeterminada en el control Editor de imágenes.

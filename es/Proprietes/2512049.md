


## WDD55Password (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WDD55Password** permite especificar la contraseña del análisis 5.5 que contiene un archivo de datos Hyper File 5.5 utilizado en un análisis HFSQL Classic.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines the password associated with the analysis 
// containing the Customer data file
Customer.WDD55Password = "Password"
...
// Validate the file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Data file>.WDD55Password = <Password>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos definido a través del tipo [Expediente description](../WDLang4/1514065.md).

**`<Password>: (Cadena de caracteres)`**

Contraseña de WDD en formato 5.5.

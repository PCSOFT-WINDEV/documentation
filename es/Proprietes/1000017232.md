


## SubCaption (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SubCaption** permite obtener y cambiar: 

- el subtítulo secundario de una opción que se encuentra en un Botón de radio o control Casilla de verificación.
	**Observación**: El Botón de Radio o control Casilla de verificación debe soportar los subtítulos ("Estándar + subtítulos" o "Vista + subtítulos" control).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el subtítulo secundario de un elemento encontrado en una control TreeMap. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the secondary caption of a Radio Button control
RADIO_Choice[2].SubCaption = "To enable the advanced mode"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el subtítulo de un elemento

`<Sub-caption text> = <Control used>[<Element number>].SubCaption`
---

**`<Sub-caption text>: (Cadena de caracteres)`**

Corresponde a: 

- el subtítulo de la opción para un botón de radio o control Casilla de verificación. 

- la leyenda secundaria del elemento para una control TreeMap. 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede corresponder a un botón de radio control, un control Casilla de verificación o un control TreeMap. 

**`<Element number>: (Integro)`**

Corresponde a: 

- el número de la opción utilizada para un botón de radio o control Casilla de verificación. 

- el índice del elemento utilizado para una control TreeMap. 





<a name="SYNTAX2"></a>

### Modificar el subtítulo de un elemento

`<Control used>[<Element number>].SubCaption = <New sub-caption>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede corresponder a un botón de radio control, un control Casilla de verificación o un control TreeMap. 

**`<Element number>: (Integro)`**

Corresponde a: 

- el número de la opción utilizada para un botón de radio o control Casilla de verificación. 

- el índice del elemento utilizado para una control TreeMap. 




**`<New sub-caption>: (Cadena de caracteres)`**

Texto del subtítulo a visualizar para el elemento especificado.

## Observaciones


### Check Box y control Botón de opción
<a name="check_box_control_boton_opcion_ELTPARAGRAPHE000077"></a>

La altura del subtítulo se adapta automáticamente al tamaño del texto visualizado. Esta altura puede ser modificada por el [Altura](../Proprietes/2510050.md) Property.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Control TreeMap
<a name="control_treemap_ELTPARAGRAPHE000089"></a>

La leyenda secundaria de un elemento se mostrará sólo si hay espacio disponible..

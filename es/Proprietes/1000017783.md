


## Value (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **Valor** Property se utiliza para get y cambiar el valor de un elemento. Esta propiedad se puede utilizar en:

- una control encontrada en una ventana, una reporte o una Page. Para obtener más información, consulte [Valor Property en los controles](../Proprietes/2510130.md).  

- una variable de tipo [gglLocation](../WDLang5/1000017441.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [rssSource](../WDLang5/1000017803.md).

- una variable de tipo [rssIdentifier](../WDLang5/1000017797.md).

- una variable de tipo [rssCategory](../WDLang5/1000017782.md).

- una variable de tipo [xlsCell](../WDLang5/1000017472.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [xmlReader](../WDLang5/1000023392.md).

- una variable de tipo [Galleta](../WDLang3/1000019222.md).

- una variable de tipo [xmlAttribute](../WDLang5/1000018799.md).

- a Variable generada automáticamente durante el [importación de un servicio web](../WDLang3/3086001.md).

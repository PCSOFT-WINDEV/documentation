


## BackgroundImageState (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BackgroundImageState** Property permite obtener y cambiar el número de dibujos de la imagen associated con el Background de un control Botón.

**Observaciones**:

- Cada dibujo de la Image corresponde al estado de un botón. Los dibujos deben seguir un orden específico: Normal, Pressed, Grayed, Focus y Rollover.

- El número de dibujos que se encuentran en una imagen se define en el editor de la ventana, en la pestaña "General" del control description.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the image associated with the border of the "BTN_Help" button
BTN_Help.BackgroundImage = "C:\MyImages\Border.bmp"
// Initialize the number of drawings found in the image
// associated with the border of the "BTN_Help" button
BTN_Help.BackgroundImageState = 3
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de dibujos que se encuentran en la imagen associated con el Border de un control Botón

`<Current number of drawings> = <Button control>.BackgroundImageState`
---

**`<Current number of drawings>: (Integro)`**

Número de dibujos encontrados en la imagen associated con el Border del botón especificado. Este número está entre 1 y 5.

**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar.  


<a name="SYNTAX2"></a>

### Modificando el número de dibujos encontrados en la imagen associated con el Border de un control Botón

`<Button control>.BackgroundImageState = <New number of drawings>`
---

**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar.

**`<New number of drawings>: (Integro)`**

Nuevo número de dibujos encontrados en la imagen associated con el Border del botón especificado. Este número debe estar entre 1 y 5.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **BackgroundImageState** Property se aplica sólo a los mandos de los botones.

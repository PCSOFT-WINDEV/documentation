
## ClientCertificate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ClientCertificate** Property se utiliza para administrar certificados de cliente. Esta propiedad se puede utilizar en:

- a Variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- a Variable de tipo [restRequest](../WDLang3/1000021481.md). 

- a Variable de tipo [wsRequest](../WDLang3/1000020577.md). 

- a Variable de tipo [apiRequest](../WDLang3/1410088935.md).

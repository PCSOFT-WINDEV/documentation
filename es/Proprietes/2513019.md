
## BackgroundStyle (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **BackgroundStyle** permite obtener y modificar el estilo de fondo de las celdas de tablas y/o tablas jerárquicas. Esta Property sólo es efectiva si la fila, columna o celda utiliza un color de fondo especificado con la [BackgroundColor](../Proprietes/2510022.md) Property. Este Property puede ser usado para resaltar una célula específica en el tabla.

**Ejemplo**: La celda del "lunes" está coloreada en verde claro con el estilo "Tubo".
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=StyleFond.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Color the DAY cell of the current row
COL_Day[1].BackgroundColor = LightGreen
COL_Day[1].BackgroundStyle = styleTube
```

## Sintaxis

### Averiguar el estilo del fondo Color

`<Background style> = <Element to use>.BackgroundStyle`
---

**`<Background style>: (Constante de tipo Integer)`**

Se utiliza actualmente el estilo Background:


|   |   |
| --- | --- |
| *styleCrossHatch* | Sombreados como +++++ |
| *styleDefault* | Valor predeterminado. Colores sólidos. |
| *styleDiagonalCrossHatch* | Escotillas como xxxxx |
| *styleGradient* | Simple Gradient (hasta el fondo). |
| *styleHorizontalHatch* | Escotillas tales como |
| *styleLeftDiagonalHatch* | Escotillas como ////// |
| *styleRightDiagonalHatch* | Escotillas como _EASLASH_\\_ASLASH_\\_ASLASH_\\_ASLASH_ _ASLASH |
| *styleTube* | 2 gradientes del color principal, usado para get un efecto de tubo. |
| *styleVerticalHatch* | Escotillas como \\\\\ |



**`<Element to use>: (Cadena de caracteres)`**

Identifica la celda, fila o columna de una tabla o control Tabla TreeView que se debe gestionar..

## Observaciones




### Cómo funciona el **BackgroundStyle** Property
<a name="como_funciona_docparampagetitleshort_property_ELTPARAGRAPHE000143"></a>

El **BackgroundStyle** Property aplica los cambios a las celdas, filas o columnas sólo si el color de estos elementos se ha establecido con el [BackgroundColor](../Proprietes/2510022.md) Property.

**Uso de las escotillas**: El color de las escotillas está definido por el [Color](../Proprietes/2510071.md) Property. Por defecto, la Color de las trampillas corresponde a la Color del texto. Para utilizar una Color diferente para el texto, utilice [gPen](../WDLang1/3029039.md).

Ejemplo: 


```wl
TABLE_NoName1.Color = Black
TABLE_NoName1.BackgroundColor = LightYellow
FOR I = 1 TO 10
	TableAddLine(MySelf)
	TABLE_NoName1[i].BackgroundStyle = styleLeftDiagonalHatch
	TABLE_NoName1.COL_NoName1[i] = gPen(LightRed) + "Row " + i
END
```

<a name="NOTE0_2"></a>




### Límite
<a name="limite_ELTPARAGRAPHE000170"></a>

El **BackgroundStyle** Property sólo se puede utilizar en las celdas, filas y columnas de los controles Table y TreeView Table.

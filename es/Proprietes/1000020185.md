


## CaptionPeriodHeader (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CaptionPeriodHeader** Property se utiliza para get y cambiar el título del período de tiempo mostrado en la cabecera de un Planificador o Agenda control en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// "Before printing" event of the header block
SCH_MySchedule.CaptionPeriodHeader = ...
	" Week from " + DateToString(SCH_MySchedule.StartDatePeriodDisplayed) + ...
	" to " + DateToString(SCH_MySchedule.EndDatePeriodDisplayed)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el título del período

`<Period caption> = <Control used>.CaptionPeriodHeader`
---

**`<Period caption>: (Cadena de caracteres)`**

Pie de foto del período que aparece en el encabezado de control.  

**`<Control used>: (Nombre del control)`**

Nombre del Programador o control Agenda en la reporte.  


<a name="SYNTAX2"></a>

### Modificar el título del período

`<Control used>.CaptionPeriodHeader = <New caption text>`
---

**`<Control used>: (Nombre del control)`**

Nombre del Programador o control Agenda que se encuentra en el reporte.

**`<New caption text>: (Cadena de caracteres)`**

Nueva leyenda del período a mostrar en el encabezado de control. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Cuando se llama en el "Antes de imprimir" Event del bloque de cabecera, el **CaptionPeriodHeader** Property se utiliza para cambiar la leyenda Default con el fin de mostrarla en un idioma específico (por ejemplo : en polaco). 

- El **CaptionPeriodHeader** Property no tiene efecto en los informes del Programador con recursos mostrados en columnas, porque en este caso no hay subtítulos.

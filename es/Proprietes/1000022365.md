


## CodeCSS (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CodeCSS** Property se utiliza para leer y cambiar el código CSS associated con un componente web control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(CMP_MyWebComponent.CodeCSS)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubriendo el código CSS associated con el componente web control

`<Result> = <Web Component control>.CodeCSS`
---

**`<Result>: (Cadena de caracteres)`**

Código CSS associated con el componente web control. Este código CSS se escribirá en la página. 

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el código CSS associated con el componente web control

`<Web Component control>.CodeCSS = <CSS Code>`
---

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 

**`<CSS Code>: (Cadena de caracteres)`**

Nuevo código CSS associated con el control.

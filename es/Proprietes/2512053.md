


## NbArrayElement (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbArrayElement** Property se utiliza para configurar el número de elementos de un elemento array. Es posible:

- get el número de elementos de un elemento de array (definido en el editor de análisis o mediante programación)

- Definir el número de elementos de un elemento de array (cuando el elemento se describe mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the number of elements in the 
// Title array item of Customer data file
Title.NbArrayElement = 3
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el número de elementos de un elemento de array

`Number of elements> = <Data file>.<Item>.NbArrayElement`
---

**`<Number of elements>: (Integro)`**



- Número de elementos que se encuentran en el elemento array especificado,

- 1 si el artículo especificado no es un artículo de array.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo array utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Definir el número de elementos en un elemento de array

`<Item>.NbArrayElement = <New number of elements>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo array utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<New number of elements>: (Integro)`**



- Número de elementos que se encuentran en el elemento array especificado,

- 1 si el artículo especificado no es un artículo de array.

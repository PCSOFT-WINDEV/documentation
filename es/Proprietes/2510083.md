


## NumberColumn (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NúmeroColumna** Property se utiliza para:

- get el número de columnas en:

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control List Box multicolumna.

	- un control Tabla o Tabla TreeView. 

	- un control Looper.

	- un control Botón de opción.

	- un control Casilla de verificación. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Tabla dinámica.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un Combo Box control Tabla.

	- a control Cuadro de mando.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Hoja de cálculo. 




- set el número de columnas: 

	- en una control List Box multicolumna. El número de columnas en una control List Box multicolumna no puede ser inferior a 2.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Looper.







![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Recordatorio**: Para definir varias columnas en un control List Box (List Box o control ListView visualizado en modo estándar):

1. Mostrar la ventana de description de control ("description" de el menú contextual de control).

2. En la ficha "Detalles", especifique el número de columnas en la opción "Número de columnas".





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// --Process for resizing the window
// If the width of the window is modified:
// - the width of the List Box is modified
LIST_MyList.Width = MyWindow.Width - 30
// - the number of columns in the List Box control is modified
// according to the new window width
LIST_MyList.NumberColumn = LIST_MyList.Width/50
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el número de columnas en una control

`<Number of columns> = <Control used>.NumberColumn`
---

**`<Number of columns>: (Integro)`**



- Número de columnas que se encuentran en la control List Box multicolumna especificada.

- Número de columnas que se encuentran en la control Tabla especificada.

- Número de columnas que se encuentran en la casilla de verificación o control Botón de opción especificada.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Número de columnas que se encuentran en la control Tabla dinámica especificada.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Número de columnas que se encuentran en el Combo Box control Tabla especificado.

- Número de celdas presentes en el ancho del control Cuadro de mando especificado.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Número máximo de columnas que contienen datos (valor o fórmula) en un control Hoja de cálculo. 

- Número de columnas encontradas en la control Looper especificada. 

- 1 si el elemento especificado no corresponde a una control que pueda ser procesada.




**`<Control used>: (Nombre del control)`**

Nombre de control cuyo número de columnas se solicita.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Modificación del número de columnas en una control

`<Control used>.NumberColumn = <Number of columns>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control List Box multicolumna

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Looper.




**`<Number of columns>: (Integro)`**

Nuevo número de columnas. 

Para los list box, este número debe ser mayor que 1 (un control List Box multicolumna no se puede convertir en un list box estándar y viceversa)..

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000150"></a>

La propiedad **NumberColumn** solo se aplica a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles List Box multicolumna (Controles ListView que se muestran en el modo estándar).

- Tabla o TreeView controles Tabla. 

- Controles de los botones de radio.

- Controles de las casillas de verificación.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) controles Tabla dinámica.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Combo Box controles Tabla.

- Controles del tablero de instrumentos.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) controles Hoja de cálculo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de looper.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **NumberColumn** solo se aplica a los siguientes elementos:

- Controles de las casillas de verificación.

- Controles de los botones de radio.

- controles Tabla.

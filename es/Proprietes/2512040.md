


## Index (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Index** indica el índice físico (orden físico) de un elemento en un Record.

Observación: El artículo fue creado en la editor de análisis o mediante programación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info("The Name item is the " + Customer.Name.Index + ...
		"physical item of CUSTOMER file")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Data file>.<Item>.Index`
---

**`<Result>: (Integro)`**



- Índice del campo, 

- 0 para un elemento de "clave compuesta".




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado.


## RightIndent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RightIndent** Property se utiliza para identificar y modificar la sangría derecha de un elemento. Esta propiedad se puede utilizar en:

- controles de edición y controles estáticos. Para obtener más información, consulte [RightIndent](../Proprietes/1000021110.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) variables de tipo [docParagraph](../WDLang1/1000022483.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) variables de tipo [docNumberingLevel](../WDLang1/1000022799.md).




## StoredItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StoredItem** Property se utiliza para:

- Averiguar el elemento almacenado o la Variable almacenada de una Tabla, Looper, Cuadro de Lista o control Combo Box.

- Modificar el elemento almacenado o la Variable almacenada de una Tabla, Looper, Cuadro de Lista o control Combo Box.




**Observaciones**:

- El elemento almacenado se utiliza para los controles associated con un archivo de datos o la consulta. La Variable almacenada se utiliza para los controles asociados a una Variable.

- El elemento almacenado o la variable almacenada se define en el editor de ventanas o páginas, en la pestaña "Contenido" de la descripción del control.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RubriqueMemorisee.gif)










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the stored item of the "TABLE_CustomerTable" control
TABLE_CustomerTable.StoredItem = "CustomerName"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar el artículo almacenado o Variable de un control

`<Stored item/variable> = <Control used>.StoredItem`
---

**`<Stored item/variable>: (Cadena de caracteres)`**



- Nombre del artículo almacenado para el control especificado. Este elemento pertenece a la el archivo de datos en la que se basa el control (devuelto o modificado por la [BrowsedFile](../Proprietes/2510040.md) Property).

- Nombre del Variable almacenado para el control especificado. Este nombre tiene el siguiente formato ":&lt;prefijo&gt;&lt;Variable Nombre&gt;" o ":&lt;Variable Nombre&gt;" (el prefijo se utiliza sólo si el estilo de código está activado en el proyecto).
	Esta variable pertenece a la variable en la que se basa el control (devuelta o modificada con la [BrowsedFile](../Proprietes/2510040.md) Property).

- Vaciar la cadena ("") si ningún elemento almacenado/Variable es associated con el control.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla,

- Control List Box, 

- Control Combo Box.   





<a name="SYNTAX2"></a>

### Modificar el artículo almacenado/Variable de un control

`<Control used>.StoredItem = <New item/variable>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla,

- Control List Box, 

- Control Combo Box. 




**`<New item/variable>: (Cadena de caracteres)`**



- Nombre del nuevo elemento almacenado para el control especificado. Este elemento debe pertenecer al el archivo de datos en el que se basa el control (devuelto o modificado por el [BrowsedFile](../Proprietes/2510040.md) Property).

- Nombre del nuevo Variable almacenado para el control especificado. Este nombre tiene el siguiente formato ":&lt;prefijo&gt;&lt;Nombre de la variable&gt;" o ":&lt;Nombre de la variable&gt;". El prefijo debe especificarse sólo si el estilo de código está activado en el proyecto (tanto si se muestran los prefijos como si no).
	Esta variable pertenece a la variable en la que se basa el control (devuelta o modificada con la [BrowsedFile](../Proprietes/2510040.md) Property).

- Vaciar la cadena ("") si no hay ningún elemento almacenado/Variable debe ser associated con el control.

## Observaciones


### Valor Property
<a name="valor_property_ELTPARAGRAPHE000115"></a>

El **StoredItem** Property modifica el valor devuelto por el [Valor](../Proprietes/2510130.md) Property.

Pueden darse dos casos distintos: 

- El cuadro de lista o control Tabla se gestiona mediante programación O el **StoredItem** Property corresponde a una cadena vacía (""). El [Valor](../Proprietes/2510130.md) Property le permite:

	- Descubra la fila seleccionada. Si no se selecciona fila, el [Valor](../Proprietes/2510130.md) Property devuelve -1. Si el control List Box (o el control Tabla) es de selección múltiple, se devuelve el número del último fila seleccionado.

	- Desplace la selección especificando el número de la nueva fila a seleccionar. Se desmarcan las líneas que se seleccionaron previamente..
			**Para identificar el fila seleccionado**, utilice una de las siguientes sintaxis: 
			
		```txt
		ResRow = <List Box or Table control>.Value
		ResRow = <List Box or Table control>
		```





- El cuadro de lista o control Tabla se basa en un archivo de datos Y el **StoredItem** Property no es una cadena vacía (""). La propiedad [Valor](../Proprietes/2510130.md) se utiliza para identificar el elemento almacenado para el Record asociado al fila seleccionado. Si no se selecciona fila, el [Valor](../Proprietes/2510130.md) Property devuelve una cadena vacía ("").
	**Para conocer el elemento almacenado asociado a la fila seleccionada**, utilice una de las siguientes sintaxis:
	
	```txt
	ResStoredItem = <List Box or Table control>.Value
	ResStoredItem = <List Box or Table control>.StoredItem
	```




<a name="NOTE0_2"></a>


### DataBinding Property
<a name="databinding_property_ELTPARAGRAPHE000159"></a>

El **StoredItem** Property se utiliza a menudo en combinación con el [DataBinding](../Proprietes/2510060.md) Property. Esta asociación crea un enlace automatic entre dos archivos de datos cuando se llama a [FileToScreen](../WDLang4/3044210.md) y [ScreenToFile](../WDLang4/3044146.md).
<a name="NOTE0_3"></a>


### Control Tabla rellenado mediante programación
<a name="control_tabla_rellenado_mediante_programacion_ELTPARAGRAPHE000177"></a>

Si un control Tabla poblado programáticamente debe mostrar datos provenientes de un archivo de datos, una consulta o una vista:

- Cree una control Tabla poblada de forma programada en el editor de ventanas o en el editor de páginas.

- En el código, especifique el elemento a recorrer con la [BrowsedItem](../Proprietes/2510111.md) Property. Este elemento debe pertenecer a el archivo de datos, a la consulta o a la vista cuyos datos se mostrarán en la control Tabla poblada programáticamente.

- En el código, especifique el elemento que se va a almacenar con el **StoredItem** Property. Este elemento debe pertenecer a el archivo de datos, a la consulta o a la vista cuyos datos se mostrarán en la control Tabla poblada programáticamente.

- Utilice [FileToMemoryTable](../WDLang1/3074005.md) para inicializar la control Tabla con los datos deseados. Los datos se clasificarán según el elemento de búsqueda especificado.

- El [DisplayedItem](../Proprietes/2510123.md) Property se inicializará automáticamente.



<a name="NOTE0_4"></a>


### control List Box se llena de forma programada
<a name="control_list_box_llena_forma_programada_ELTPARAGRAPHE000201"></a>

Si un Cuadro de Lista o control Combo Box poblado programáticamente debe mostrar datos provenientes de un archivo de datos, una consulta o una vista:

- Cree una Caja de Listas o control Combo Box poblada programáticamente en el editor de ventanas o en el editor de páginas.

- Utilice [FileToMemoryList](../WDLang1/3049007.md) para inicializar la control con los datos deseados.

- Las propiedades [BrowsedFile](../Proprietes/2510040.md), [DisplayedItem](../Proprietes/2510123.md), [BrowsedItem](../Proprietes/2510111.md) y **StoredItem** se inicializarán automáticamente.



<a name="NOTE0_5"></a>


### Búsqueda y selección de un elemento en un cuadro de lista o control Combo Box basado en un archivo de datos
<a name="busqueda_seleccion_elemento_cuadro_lista_control_combo_box_basado_archivo_datos_ELTPARAGRAPHE000226"></a>Para buscar y/o seleccionar un elemento en una Caja de Listas o control Combo Box basada en un archivo de datos, le recomendamos que utilice la siguiente sintaxis:

```wl
LIST_OrderList.StoredItem = "CustomerID"
// Find and select the element whose CustomerID is "135"
LIST_OrderList = 135
// If the element sought is not found, LIST_OrderList will be set to -1
```

<a name="NOTE0_6"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000236"></a>

El **StoredItem** Property se aplica sólo a los siguientes controles:

- Control Tabla,

- Control List Box, 

- Control Combo Box,

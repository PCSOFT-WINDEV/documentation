


## Keywords (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Keywords** permite obtener y cambiar las palabras claves asociadas a una página. Las palabras clave también pueden ser associated con una página del editor de la página.

En las páginas activas de WEBDEV, el **Keywords** Property se utiliza para elegir las palabras clave de forma dinámica associated con la página (en función del contenido mostrado en la página).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the keywords of the page according to its content
// Adds new keywords to the existing ones
MyPage.Keywords = MyPage.Keywords + ", Boat, Sail, Engine"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar las palabras clave associated con una página

`<List of keywords> = <Page used>.Keywords`
---

**`<List of keywords>: (Cadena de caracteres)`**

Palabras clave asociadas a la página. Las diferentes palabras clave associated con la página están separadas por una coma (", ").

**`<Page used>: (Nombre de la página)`**

Nombre de la página a analizar.  


<a name="SYNTAX2"></a>

### Modificando las palabras clave associated con una página

`<Page used>.Keywords = <List of keywords>`
---

**`<Page used>: (Nombre de la página)`**

Nombre de la página para la que se deben definir las palabras clave.

**`<List of keywords>: (Cadena de caracteres)`**

Palabras clave para asociar con la página. Las diferentes palabras clave están separadas por una coma (","). Si las palabras clave son ya associated con la página, estas palabras clave serán reemplazadas. Esta cadena debe cumplir con las siguientes reglas:

- Sin carácter CR (Carriage Return)

- Sin espacios en blanco y sin comillas.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Palabras clave associated con una página
<a name="palabras_clave_associated_con_una_pagina_ELTPARAGRAPHE000045"></a>

Las palabras clave associated con una página se escriben en el encabezado HTML de la página (&lt;META NAME="Palabras clave" CONTENT="..."&gt; tab). Estas palabras clave son a menudo utilizadas por los motores de búsqueda para las páginas de reference.

Las palabras clave deben estar definidas en el código de inicialización de la página.. Además, para ser eficientes, las palabras clave deben estar definidas en páginas accesibles a través de una URL en el siguiente formato: "http://".

Algunos consejos: 

- Las 3 primeras palabras clave de la lista son muy importantes.

- No introduzca más de 20 palabras clave por página.

- No repita la misma palabra clave.

- Usar diferentes formas gramaticales (singular/plural, sustantivo/verbo).

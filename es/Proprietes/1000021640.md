


## CurrentWorksheet (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CurrentWorksheet** Property se utiliza para get y cambiar la hoja de trabajo actual en un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a worksheet
nWorksheet is int 
nWorksheet = SpreadsheetAddWorksheet(PSHEET_MySpreadsheet)
// Select the worksheet
PSHEET_MySpreadsheet.CurrentWorksheet = nWorksheet
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la posición de la hoja de trabajo actual

`<Result> = <Spreadsheet control>.CurrentWorksheet`
---

**`<Result>: (Integro)`**

Posición de la hoja de trabajo actual.

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


<a name="SYNTAX2"></a>

### Modificar la hoja de trabajo actual en un control Hoja de cálculo

`<Spreadsheet control>.CurrentWorksheet = <New value>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<New value>: (Integro)`**

Posición de la hoja de trabajo que se convertirá en la hoja de trabajo actual. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el control Hoja de cálculo no maneja las múltiples hojas de trabajo: 

	- la modificación de la hoja de trabajo actual no tiene ningún efecto,

	- la posición de la hoja de trabajo actual es siempre set a 1. 




- Las funciones de gestión de control Hoja de cálculo siempre manipular los datos en la hoja de trabajo actual. 

- Las sintaxis para acceder a las células siempre manipular la hoja de trabajo actual.




## DoubleClick (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DoubleClick** Property se utiliza para get y cambiar el nombre del control Botón que se ejecutará cuando se haga doble clic en un Cuadro de Lista, ListView, Tabla, Looper, Casilla de Selección, Botón de Radio o control TreeView. 

Por defecto, se puede definir el nombre de la control Botón ejecutada al hacer doble clic en el editor de ventanas, en el control description (pestaña "Detalles").


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the Button control associated with the double click on "LIST_Map"
LIST_Map.DoubleClick = "BTN_Wizard"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el control Botón associated al doble clic

`<Button control> = <Control used>.DoubleClick`
---

**`<Button control>: (Nombre del control)`**



- Nombre de la control Botón cuyo código se ejecutará al hacer doble clic en la control especificada. 

- Vaciar la cadena ("") si ningún control Botón es associated con el doble clic.




**`<Control used>: (Nombre del control)`**

Nombre del control (List Box, ListView, TreeView, Looper, Check Box, Radio Button o Table) para el que queremos saber el nombre del botón associated con el doble clic.  


<a name="SYNTAX2"></a>

### Cambie el control Botón associated con el doble clic

`<Control used>.DoubleClick = <New Button control>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control (List Box, ListView, TreeView, Looper, Check Box, Radio Button o Table) para el que queremos modificar el nombre del botón associated con el doble clic.

**`<New Button control>: (Nombre del control)`**



- Nombre de la control Botón cuyo código se ejecutará al hacer doble clic en la control especificada,

- Vaciar la cadena ("") si ningún control Botón es associated con el doble clic.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **DoubleClick** solo se puede utilizar en los siguientes controles:

- Controles de TreeView,

- controles List Box y ListView,

- Controles de looper,

- controles Tabla y Tabla TreeView, 

- Controles de las casillas de verificación,

- Controles de los botones de radio,

- Controles Calendar,

- Controles TreeMap,

- Controles del organizador,

- Controles del programador.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **DoubleClick** Property sólo puede utilizarse en los controles List Box y TreeView.

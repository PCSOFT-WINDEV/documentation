


## Items (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Items** permite configurar los campos de un índice de texto completo. Es posible:

- Averiguar los diferentes elementos de un índice de texto completo (definido en la editor de análisis o mediante programación).

- Definir los diferentes elementos de un índice de texto completo (cuando el índice de texto completo se describe mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the full-text index
MyFullTextIndex.MinimumLength = 4
MyFullTextIndex.Name = "FullTextIndex1"
MyFullTextIndex.Items = "LASTNAME+FIRSTNAME"
MyFullTextIndex.AccentSensitive = True
MyFullTextIndex.CaseSensitive = False
HDescribeFullTextIndex(AFile, MyFullTextIndex)
...
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la expresión de un índice de texto completo

`<Index expression> = <Data file>.<Full-text index>.Items`
---

**`<Index expression>: (Cadena de caracteres)`**

Nombre de los diferentes elementos incluidos en el índice de texto completo. Estos nombres de elementos están separados por el carácter "+".

**`<Data file>: (Nombre del control)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre se definió en el editor de análisis o con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).  


<a name="SYNTAX2"></a>

### Definición de la expresión de un índice de texto completo

`<Full-text index>.Items = <New expression>`
---

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre se definió con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).

**`<New expression>: (Cadena de caracteres)`**

Nombre de los diferentes elementos incluidos en el índice de texto completo. Estos nombres de elementos están separados por el carácter "+"..

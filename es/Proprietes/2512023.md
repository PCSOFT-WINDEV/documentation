


## MaxAutoID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **MaxAutoID** se utiliza para obtener el identificador máximo de automatic de un archivo de datos replicado. **Esta propiedad es de solo lectura**.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the maximum automatic identifier of the Customer data file 
ResMaxAutoID = Customer.MaxAutoID
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Data file>.MaxAutoID`
---

**`<Result>: (Integro)`**



- Valor máximo del identificador automatic definido para el archivo de datos,

- 0 si no se han definido los límites o si no se replica el archivo de datos.




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado.

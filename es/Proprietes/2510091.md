


## FontStrikeOut (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontStrikeOut** Property se utiliza para:

- Determine si el texto está tachado.

- Aplicar el formato de tachado (o no) al texto mostrado en una control.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Apply strikethrough formatting to the text displayed in "STC_StaticControl"
STC_StaticControl.FontStrikeOut = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el texto mostrado en una control está tachado

`<Result> = <Control used>.FontStrikeOut`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto está tachado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Aplicar el formato de tachado (o no) al texto mostrado en una control

`<Control used>.FontStrikeOut = <Struck/Not struck>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  

**`<Struck/Not struck>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto debe ser tachado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000103"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **FontStrikeOut** Property sólo puede utilizarse en:

- a control Estático.

- una control calculada .

- to URL control Enlace.

- un control preestablecido.


En una ventana, el **FontStrikeOut** Property no se puede utilizar en:

- a Control de número,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una Web control Cámara,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Conferencia.


Observación: Esta Property se aplica únicamente a los controles "ListView" en el modo de visualización "List Box".




## Multiselection (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Multiselection** obtiene y establece el modo de selección:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Tabla dinámica.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) en un control List Box.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control ListView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) en un control Looper.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) en un control Calendario.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the selection mode based on the value of the Check Box control
IF CBOX_MultiSel = True THEN
	TABLE_Table1.Multiselection = True
ELSE
	TABLE_Table1.Multiselection = False
END
```

## Sintaxis

### Obtener el modo de selección

`<Selection mode> = <Control used>.Multiselection`
---

**`<Selection mode>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control permite la selección múltiple,

- <u><u><u><u>False</u></u></u></u> si el control es de selección única

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Control List Box.

### Cambiar el modo de selección

`<Control used>.Multiselection = <New selection mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Tabla dinámica.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Control List Box.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control ListView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Control Looper.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Control Calendario.




**`<New selection mode>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control debe permitir la selección múltiple,

- <u><u><u><u>False</u></u></u></u> si el control debe ser de selección única. Si el control cambia a selección única y hay varios elementos seleccionados, solo se conserva el primer elemento seleccionado.  




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para los controles Tabla y Tabla TreeView: Solo cambia el modo de selección de las filas. El modo de selección de las columnas y celdas no cambia.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Limitaciones para el control Calendario
<a name="limitaciones_para_control_calendario_ELTPARAGRAPHE000187"></a>

En un control Calendario, la propiedad **Multiselection** solo puede utilizarse para pasar del modo de selección única al modo de selección múltiple. Si se selecciona el modo "Selección de período" en el editor (pestaña "Detalles" de la ventana de descripción del control), la propiedad **Multiselection** no tiene ningún efecto.
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Modo de funcionamiento del control Tabla dinámica
<a name="modo_funcionamiento_del_control_tabla_dinamica_ELTPARAGRAPHE000213"></a>

En un control Tabla dinámica, la propiedad **Multiselection** solo puede utilizarse para pasar del modo de selección única al modo de selección múltiple. El usuario podrá seleccionar varias celdas contiguas. 

Esta propiedad equivale a la opción "Permitir selección múltiple de filas, columnas y celdas" en la pestaña "UI" de la ventana de descripción del control. 
<a name="NOTE0_4"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000228"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La propiedad **Multiselection** no tiene ningún efecto:

- en controles Tabla editables sin selección: devuelve <u><u><u><u>False</u></u></u></u>.

- en controles Tabla de solo lectura sin selección: devuelve <u><u><u><u>False</u></u></u></u>.

- en controles Tabla "Compatibles con la versión 5.5".

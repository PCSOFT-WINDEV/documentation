


## Default (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **valor predeterminado** Property se utiliza para determinar si un parámetro se ha pasado a un valor Procedure con un valor personalizado o Default .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Call the procedure
MyProc("A")
```

```wl
// Declare the procedure
PROCEDURE MyProc(p1, po2 = "Y", po3 = "Z")
```

```wl
// In this example: 
MyParameters[1].Default // is set to False
p1.Default               // is set to False
MyParameters[2].Default // is set to True
po2.Default              // is set to True
MyParameters[3].Default // is set to True
po3.Default              // is set to True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Parameter used>.Default`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si no se especificó el parámetro al llamar al procedimiento (se utilizó el valor predeterminado),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Parameter used>: (Nombre del parámetro)`**

Nombre del parámetro utilizado. Este parámetro está identificado:

- utilizando la palabra clave **MyParameters** y especificando el índice del parámetro utilizado. Por ejemplo: MyParameters[1] para manipular el primer parámetro pasado al procedimiento. 

- utilizando el nombre del parámetro especificado en la declaración de la Procedure.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **Default** Property solo tiene en cuenta los parámetros pasados al Procedure actual. If multiple Procedure s are nested and pass the same parameter with or without the default value, the **Default** Property only takes into account the value of the parameter in the Procedure where the Property is used.

- La propiedad **Default** solo se puede usar en el Process en el que se declara el elemento.




## DisplayedItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DisplayedItem** Property se utiliza para:

- Descubra el artículo o la Variable que muestra el control.

- Modificar el elemento o la Variable mostrada por el control.




**Observaciones**:

- **En el reporte "Apertura" Event**, este Property se utiliza para modificar el elemento que aparece en un control cuando la fuente de datos del reporte se modifica mediante programación (por [SourceName](../Proprietes/2511035.md) por ejemplo).

- **Si esta propiedad se utiliza en el control "Antes de imprimir" Event**, el cambio se realizará la próxima vez que se imprima el control (de hecho, el valor del elemento ya se lee al ejecutar el control "Antes de imprimir" Event).

- El elemento mostrado puede provenir de un archivo de datos, una vista o una consulta.

- El elemento o variable almacenado se define en el editor de ventanas o páginas, en la pestaña "Contenido" del control description.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para una control ListView, este ítem corresponde a la leyenda que aparece en la control ListView.

### Averiguar el artículo/Variable mostrado por una Tabla, Cuadro de Lista o control Combo Box

`<Item/variable currently displayed> = <Control used>.DisplayedItem`
---

**`<Item/variable currently displayed>: (Cadena de caracteres)`**



- Nombre del elemento mostrado por la control especificada. Este elemento pertenece a la el archivo de datos buscada por el control (devuelta o modificada por la [BrowsedFile](../Proprietes/2510040.md) Property).

- Nombre de Variable mostrado por el control especificado. Este nombre tiene el siguiente formato: 
	
	```txt
	":<prefix><Variable name>"
	```
 o 
	```txt
	":<Variable name>"
	```
(el prefijo se aplica sólo si el estilo de código está habilitado en el proyecto).
	Esta variable pertenece a la variable buscada por el control (devuelta o modificada con la [BrowsedFile](../Proprietes/2510040.md) Property).

- Para un control Tabla, lista de los nombres de los elementos o variables mostrados por cada columna. Los nombres de los artículos/variables están separados por el carácter "|" (AltGr + 6 teclas).




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: Tabla, List Box, ListView o Combo Box.

### Modificación del elemento/Variable mostrado por una Tabla, Cuadro de Lista o control Combo Box

`<Control used>.DisplayedItem = <New item/variable>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: Tabla, List Box, ListView o Combo Box.  

**`<New item/variable>: (Cadena de caracteres)`**



- Nombre del artículo que se mostrará en la control especificada. Este elemento pertenece a la el archivo de datos buscada por el control (devuelta o modificada por la [BrowsedFile](../Proprietes/2510040.md) Property).

- Nombre de la Variable que se mostrará en la control especificada. Este nombre tiene el siguiente formato: 
	
	```txt
	":<prefix><Variable name>"
	```
 o 
	```txt
	":<Variable name>"
	```
El prefijo debe especificarse sólo si el estilo de código está activado en el proyecto (tanto si se muestran los prefijos como si no).
	Esta variable pertenece a la variable buscada por el control (devuelta o modificada con la [BrowsedFile](../Proprietes/2510040.md) Property).

- Para un control Tabla, lista de los nombres de los elementos o variables mostrados por cada columna. Los nombres de los artículos/variables están separados por el carácter "|" (AltGr + 6 teclas)..

### Averiguar el elemento que muestra una reporte control

`<Item currently displayed> = <Control used>.DisplayedItem`
---

**`<Item currently displayed>: (Cadena de caracteres)`**

Nombre del artículo associated con el control especificado. Para un enlace de varios archivos, el nombre del elemento tiene el siguiente formato: 

```txt
<Name File1>.<Item name> + TAB + <Name File2>.<Item name>
```
Esta partida corresponde a:

- un elemento que pertenece al archivo utilizado como fuente de datos para la reporte (devuelto o modificado por [SourceName](../Proprietes/2511035.md)).

- un elemento (o su alias) perteneciente a la consulta utilizada como fuente de datos de reporte.




**`<Control used>: (Nombre del control)`**

Nombre de reporte control a utilizar.

En un reporte, este control puede ser:

- un control Imagen, 

- a control Código de barras,

- un RTF control Estático,

- a control Estático,

- a control Casilla de verificación,

- una control calculada .

- a control Hoja de cálculo

`<Control used>.DisplayedItem = <New item>`

- un control Imagen, 

- a control Código de barras,

- a control Casilla de verificación,

**`<New item>: (Cadena de caracteres)`**

Nombre del nuevo elemento associated con el control especificado. Para un enlace de varios archivos, el nombre del elemento tiene el siguiente formato: 
```txt
<Name File1>.<Item name> + TAB + <Name File2>.<Item name>
```
Esta partida corresponde a:

- un elemento que pertenece al archivo utilizado como fuente de datos para la reporte (devuelto o modificado por [SourceName](../Proprietes/2511035.md)).

- un elemento (o su alias) perteneciente a la consulta utilizada como fuente de datos de reporte.

### Manejo de los controles Tabla, Looper, List Box, ListView y Combo Box basados en un archivo de datos utilizando propiedades
<a name="manejo_los_controles_tabla_looper_list_box_listview_combo_box_basados_archivo_datos_utilizando_propiedades_ELTPARAGRAPHE000209"></a>

- Los registros mostrados en un Cuadro de Lista, Tabla, Looper o control Combo Box basados en un archivo de datos provienen de el archivo de datos o de la consulta especificada con el [BrowsedFile](../Proprietes/2510040.md) Property.

- El sentido de la ordenación se define mediante la [BrowsedItem](../Proprietes/2510111.md) Property.

- Los registros pueden filtrarse mediante la función [filtros](../Proprietes/2510042.md) Property o [HFilter](../WDLang4/3044100.md).

### Modificación del elemento que se muestra en los controles de la tabla que se rellenan mediante programación, basados en un archivo de datos y en Variable
<a name="modificacion_del_elemento_que_muestra_los_controles_tabla_que_rellenan_mediante_programacion_basados_archivo_datos_variable_ELTPARAGRAPHE000232"></a>

Si el parámetro &lt;Nuevo campo/variable&gt; contiene:

- Un número de elementos mayor que el número de columnas en el control Tabla, sólo los primeros elementos (o variables) son associated con las columnas en el control Tabla.

- Un número de elementos inferior al número de columnas de la control Tabla, las columnas "adicionales" no se asociarán a ningún elemento (o Variable).




**Observación**: Para asociar una columna a un artículo o a una Variable, utilice [DataBinding](../Proprietes/2510060.md).

### Control Tabla rellenado mediante programación
<a name="control_tabla_rellenado_mediante_programacion_ELTPARAGRAPHE000249"></a>

Si un control Tabla poblado programáticamente debe mostrar datos provenientes de un archivo de datos, una consulta o una vista:

- Cree un control Tabla poblado de forma programada en el editor de ventanas.

- En su programa, especifique el elemento que será explorado por [BrowsedItem](../Proprietes/2510111.md). Este elemento debe estar en el archivo de datos, consulta o vista cuyos datos se mostrarán en la control Tabla poblada programáticamente.

- Utilice [FileToMemoryTable](../WDLang1/3074005.md) para inicializar la control Tabla con los datos deseados. Los datos se clasificarán según el elemento de búsqueda especificado.

- El **DisplayedItem** Property se inicializará automáticamente.

### control List Box se llena de forma programada
<a name="control_list_box_llena_forma_programada_ELTPARAGRAPHE000271"></a>

Si un Cuadro de Lista o control Combo Box poblado programáticamente debe mostrar datos de un archivo de datos, consulta o vista:

- Cree un cuadro de lista o control Combo Box en el editor de ventanas.

- Utilice [FileToMemoryList](../WDLang1/3049007.md) para rellenar el cuadro de lista o control Combo Box con los datos deseados.

- Las propiedades [BrowsedFile](../Proprietes/2510040.md), **DisplayedItem**, [BrowsedItem](../Proprietes/2510111.md) y [StoredItem](../Proprietes/2510110.md) se inicializarán automáticamente.

### Control ListView
<a name="control_listview_ELTPARAGRAPHE000298"></a>

En el caso de los ListView control s, el **DisplayedItem** Property se utiliza para get y cambiar sólo el elemento que corresponde a los subtítulos de los elementos mostrados en el control.

Para obtener y cambiar el elemento que corresponde a las imágenes que se muestran en el control, utilice la función [VisualizarItemImagen](../Proprietes/2510109.md) Property.

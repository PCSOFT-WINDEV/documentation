
## Body (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Body** Property se utiliza para identificar y modificar el cuerpo HTML de un control Display HTML o un editor HTML control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the description of the article in the Article data file
Item.Description = HTM_Browser.Body
HModify(Item)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificar el cuerpo HTML de una pantalla HTML control o de un editor HTML control

`<Result> = <Control used>.Body`
---

**`<Result>: (Cadena de caracteres)`**

Código HTML del cuerpo. 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML.

- Control Editor HTML.  





<a name="SYNTAX2"></a>

### Modificación del cuerpo HTML de una pantalla HTML control o de un editor HTML control

`<Control used>.Body = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML.

- Control Editor HTML. 




**`<New value>: (Cadena de caracteres)`**

Nuevo código de cuerpo HTML. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **Body** Property se utiliza para modificar sólo el cuerpo del código HTML en el control, sin modificar su cabecera. Esto permite mantener cargadas las hojas de estilo CSS, por ejemplo.

- Si el control está vinculado a un elemento de un archivo de datos, las funciones **Body** Property serán utilizadas por [FileToScreen](../WDLang4/3044210.md) y [ScreenToFile](../WDLang4/3044146.md).

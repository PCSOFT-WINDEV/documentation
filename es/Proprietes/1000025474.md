
## GDPR (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **GDPR** permite: 

- averigüe si el GDPR se aplica a un artículo de un archivo de datos. 

- definir si el GDPR debe aplicarse a un artículo (en el artículo description sólo en el código).




Recordatorio: Los elementos a los que se aplica el editor de análisis
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile file
...

// Describe the "NAME" item
MyItem is Item Description
MyItem.Name = "NAME"
MyItem.Type = hItemText
MyItem.Size = 40
MyItem.KeyType = hUniqueKey
MyItem.GDPR = True

// Validate the description of the "NAME" item
HDescribeItem(AFile, MyItem)

// Validate the description of AFile file
...
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el GDPR se aplica al artículo

`<Result> = <Data file>.<Item>.GDPR`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el GDPR se aplica al elemento asociado al Record,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  


Nombre lógico del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).


<a name="SYNTAX2"></a>

### Determinar si el GDPR debe aplicarse al artículo

`<Item>.GDPR = <Management mode>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el GDPR se aplica al artículo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

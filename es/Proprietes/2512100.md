


## Access (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Access** permite gestionar el tipo de acceso de una conexión (HFSQL Client/Server, OLE DB, etc.). Es posible:

- Obtener el tipo de acceso utilizado para una conexión (HFSQL Client/Server, OLE DB, etc.).

- Definir el tipo de acceso utilizado para la conexión (HFSQL Client/Server, OLE DB, etc.). En este caso, si la conexión está actualmente abierta, se cerrará y se abrirá de nuevo con los nuevos parámetros la primera vez que se acceda a un archivo de datos que utilice esta conexión.




**Observación**: La conexión puede corresponder a:

- una conexión definida en el editor de análisis,

- una conexión definida con las funciones [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- una conexión definida con una variable de tipo [Connection](../WDLang4/1514073.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
// Describe the connection
MyConnection.User = "USER"
MyConnection.Password = "PASSWORD"
MyConnection.Server = "MYSERVER" 
MyConnection.Database = "Database"
MyConnection.Provider = hAccessHFClientServer
MyConnection.Access = hOReadWrite
MyConnection.ExtendedInfo = "Extended information"
MyConnection.CursorOptions = hClientCursor

HOpenConnection(MyConnection)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el tipo de acceso de una conexión

`<Type of access> = <Connection>.Access`
---

**`<Type of access>: (Constante de tipo Integer)`**

Devuelve el tipo de acceso de la conexión: 


|   |   |
| --- | --- |
| *hORead* | Solo se puede acceder a los archivos de datos utilizados por la conexión en modo de solo lectura. Solo se pueden leer registros. No podrá agregar o eliminar registros.<br>![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png) Esta constante solo está disponible para conexiones OLE DB. |
| *hOReadWrite* | Se puede acceder a los archivos de datos utilizados por la conexión en modo de lectura y escritura. Se pueden leer y escribir nuevos registros. |



**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre se define:

- en el editor de análisis, 

- con [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- con el tipo [Connection](../WDLang4/1514073.md).





<a name="SYNTAX2"></a>

### Definir el tipo de acceso de una conexión

`<Connection>.Access = <Type of access>`
---

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre se define:

- en el editor de análisis, 

- con [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- con el tipo [Connection](../WDLang4/1514073.md).




**`<Type of access>: (Constante de tipo Integer)`**

Define el tipo de acceso de una conexión: 


|   |   |
| --- | --- |
| *hORead* | Solo se puede acceder a los archivos de datos utilizados por la conexión en modo de solo lectura. Solo se pueden leer registros. No podrá agregar o eliminar registros.<br>![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png) Esta constante solo está disponible para conexiones OLE DB. Se ignora en los demás casos. |
| *hOReadWrite* | Se puede acceder a los archivos de datos utilizados por la conexión en modo de lectura y escritura. Se pueden leer y escribir nuevos registros. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad [CtAccess](../Proprietes/2512015.md) define el tipo de acceso a un archivo de datos utilizado por una conexión. Esta propiedad se conserva por motivos de compatibilidad con versiones anteriores. Se recomienda utilizar la propiedad **Access**.

Atención: La propiedad **Access** solo se aplica a los tipos "Connection". La propiedad [CtAccess](../Proprietes/2512015.md) se aplica a los archivos de datos.

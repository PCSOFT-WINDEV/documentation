


## ODBCDatabaseName (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>


### ¿Qué es una consulta realizada en una conexión?
<a name="¿que_una_consulta_realizada_una_conexion_ELTPARAGRAPHE000010"></a>

Una [consulta realizada en una conexión](../WDChamp/1011066.md) es una consulta realizada sobre archivos de datos manejados a través de un controlador ODBC específico o de una Conector Nativo.

Cuando se crea la conexión, se puede definir: 

- el nombre de la conexión

- el nombre de la fuente de datos

- el nombre de usuario y la contraseña

- el nombre de la base de datos a la que se accede (si hay más de una). 


La editor de reportes permite imprimir los datos procedentes de una consulta realizada en una conexión.. Todos los parámetros de la conexión a utilizar pueden ser definidos en la editor de reportes o a través de la programación con las propiedades del WLanguage. 
<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**ODBCDatabaseName** se utiliza para:

- Averiguar el nombre de la base de datos utilizada para imprimir una reporte a partir de una "consulta realizada en una conexión".

- Especifique el nombre de la base de datos utilizada para imprimir una reporte a partir de una "consulta realizada en una conexión".




**Observación**: Esta Property debe utilizarse en el código de apertura de la reporte.

Este Property debe utilizarse sólo si varias bases de datos corresponden al mismo nombre de conexión.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Opening code of the report
RPT_Report1.ODBCConnectionName = "MyConnection"
RPT_Report1.ODBCSQLCode = "Select * from Products"
RPT_Report1.ODBCUserName = "MOORE"
RPT_Report1.ODBCPassword = "PSW"
RPT_Report1.ODBCDatabaseName = "MyDatabase"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre de la base de datos utilizada al imprimir un reporte basado en una "consulta realizada en la conexión"

`<Result> = <Report used>.ODBCDatabaseName`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la base de datos utilizada al imprimir la reporte basada en una consulta realizada en una conexión. Este nombre sólo está disponible si se han definido varias conexiones del mismo nombre correspondientes a bases de datos diferentes.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Especificar el nombre de la base de datos que se utilizará al imprimir un reporte basado en una "consulta realizada en una conexión".

`<Report used>.ODBCDatabaseName = <Database to use>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte a utilizar.

**`<Database to use>: (Cadena de caracteres)`**

Nombre de la base de datos utilizada para realizar la consulta base del reporte a través del driver ODBC o de un Conector Nativo. **Tsu Property debe utilizarse sólo si varias bases de datos corresponden al mismo nombre de conexión.**

## Observaciones


### Propiedades utilizadas para gestionar informes basados en archivos de datos a los que se accede a través de un controlador ODBC o un Conector Nativo
<a name="propiedades_utilizadas_para_gestionar_informes_basados_archivos_datos_los_que_accede_traves_controlador_odbc_conector_nativo_ELTPARAGRAPHE000068"></a>

Las siguientes propiedades se utilizan para gestionar informes basados en archivos de datos a los que se accede a través de un controlador ODBC o de un Conector Nativo:


|   |   |
| --- | --- |
| [ODBCConnectionName](../Proprietes/2511033.md) | Permite identificar y cambiar el nombre de la conexión utilizada para manipular una base de datos a la que se accede a través de un controlador ODBC o un Conector Nativo. |
| **ODBCDatabaseName** | Permite identificar y cambiar el nombre de la base de datos utilizada por la consulta ejecutada en una base de datos a la que se accede a través de un controlador ODBC o un Conector Nativo. |
| [ODBCPassword](../Proprietes/2511031.md) | Se utiliza para identificar y cambiar la contraseña associated con el usuario autorizado a acceder a una base de datos a través de un controlador ODBC o un Conector Nativo. |
| [ODBCSQLCode](../Proprietes/2511028.md) | Permite identificar y modificar el código SQL de la consulta ejecutada en una base de datos a la que se accede a través de un controlador ODBC o un Conector Nativo. |
| [ODBCUserName](../Proprietes/2511019.md) | Permite identificar y cambiar el nombre del usuario autorizado a acceder a una base de datos a través de un controlador ODBC o un Conector Nativo. |


<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000100"></a>

El **ODBCDatabaseName** Property se aplica únicamente a los informes basados en una "consulta realizada en una conexión".


## Hora (palabra clave)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Hora** palabra clave puede corresponder a: 

- una variable de tipo [Hora](../Motscles/1514068.md). 

- la propiedad **Time**. Esta Property se utiliza para averiguar y modificar la hora de un elemento. Esta propiedad se puede utilizar en:

	- a Fecha, hora y fecha Variable o elemento. Para obtener más información, consulte [Hora Property en artículos y tipos de Fecha, Hora y DateTime](../Proprietes/2512038.md). 

	- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

	- una variable de tipo [hScheduling](../WDLang4/1000018853.md).

	- una variable de tipo [Repetition](../WDLang1/1000019262.md).

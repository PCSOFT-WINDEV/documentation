
## Mensaje

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Mensaje** palabra clave puede corresponder a:

- la función **[Mensaje](../WDLang1/3021006.md)** que permite mostrar para mostrar un mensaje en la barra de estado. 

- el **[Mensaje](../Proprietes/2510078.md)** Property que se utiliza para conocer y modificar el mensaje de ayuda associated con un control o una opción de menú en una ventana. 

- la propiedad **Message** disponible en:

	- una variable de tipo [sfError](../WDLang5/1000018455.md). 

	- la estructura [Email](../WDLang3/3032029.md)

	- una variable de tipo [Email](../WDLang3/1000018713.md).

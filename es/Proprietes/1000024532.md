
## Reduced (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Reducido** Property se utiliza para determinar si un control Panel acoplable está expandido/colapsado, o para cambiar entre los dos estados.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click on a Button control 
// Code used to collapse or expand the Dockable Panel control 
DOPA_MyDockablePanel.Reduced = NOT DOPA_MyDockablePanel.Reduced
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Saber si un control Panel acoplable está contraído o expandido

`<Result> = <Dockable Panel control>.Reduced`
---

**`<Result>: ()`**



- <u><u><u><u>True</u></u></u></u> si el control Panel acoplable está contraído,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Dockable Panel control>: (Nombre del control)`**

Nombre del control Panel acoplable a utilizar.


<a name="SYNTAX2"></a>

### Contraer o expandir un control Panel acoplable

`<Dockable Panel control>.Reduced = <Collapsed/expanded>`
---

**`<Dockable Panel control>: (Nombre del control)`**

Nombre del control Panel acoplable a utilizar.

**`<Collapsed/expanded>: ()`**



- <u><u><u><u>True</u></u></u></u> si se debe contraer el control Panel acoplable,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el control Panel acoplable está desacoplado: 

- el **Reduced** Property siempre devolverá <u><u><u><u>False</u></u></u></u>,

- modificar el **Reduced** Property no tiene ningún efecto (el control Panel acoplable permanece expandido y el valor del **Reduced** Property seguirá siendo <u><u><u><u>False</u></u></u></u>).

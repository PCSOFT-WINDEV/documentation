


## CtUserName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtUserName** Property se utiliza para set el nombre de usuario que se utilizará al conectarse a un tabla específico a través de OLE DB. Es posible:

- get el nombre de usuario que se utilizará para conectarse a un tabla vía OLE DB (nombre utilizado en [HConnect](../WDLang4/3044263.md)). Este Property puede utilizarse para el un archivo de datos definido en el editor de análisis o mediante programación.

- Definir el nombre de usuario que se utilizará para conectarse a un tabla por OLE DB (cuando el el archivo de datos se describe mediante programación). En este caso, esta Property es equivalente a la [CtDescritoNombreDeUsuario](../Proprietes/2512022.md) Property. 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describió el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the name of the user
Customer.CtUserName = "Julia"
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre del usuario (usado en HConnect)

`<User> = <Data file>.CtUserName`
---

**`<User>: (Cadena de caracteres)`**

Nombre del usuario de la fuente OLE DB definido durante la conexión por [HConnect](../WDLang4/3044263.md).

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definición del nombre del usuario (utilizado en HConnect)

`<Data file>.CtUserName = <New username>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<New username>: (Cadena de caracteres)`**

Nombre del usuario de la fuente OLE DB definido durante la conexión por [HConnect](../WDLang4/3044263.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El [CtDescritoNombreDeUsuario](../Proprietes/2512022.md) Property se utiliza para get y cambiar el nombre del usuario especificado para la conexión OLE DB cuando se describe el archivo de datos.




## ScreenCastAllowed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ScreenCastAllowed** Property se utiliza para: 

- averiguar si el usuario puede emitir el contenido mostrado en la control Multimedia a un dispositivo externo (compatible con AirPlay),

- permitir o impedir que el usuario transmita el contenido mostrado en la control Multimedia a un dispositivo externo (compatible con AirPlay). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allows the video capture
MM_Multi.ScreenCastAllowed = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estado de gestión para compartir el video

`<Result> = <Multimedia control>.ScreenCastAllowed`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el vídeo se puede transmitir a un dispositivo externo (compatible con AirPlay), 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.


<a name="SYNTAX2"></a>

### Permitir o impedir la difusión del vídeo

`<Multimedia control>.ScreenCastAllowed = <Video stream>`
---

**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.

**`<Video stream>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para permitir la transmisión del vídeo a un dispositivo externo (compatible con AirPlay). Una Icon específica se muestra en el reproductor de vídeo. 

- <u><u><u><u>False</u></u></u></u> si el vídeo no debe transmitirse a un dispositivo externo (compatible con AirPlay).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Mediante Default, el contenido que se muestra en el control Multimedia puede transmitirse a un dispositivo externo (compatible con AirPlay). Aparece automáticamente una Icon que permite transmitir el vídeo a un dispositivo externo (compatible con AirPlay).

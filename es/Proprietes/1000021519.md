


## NumberRow (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NumberRow** Property devuelve el número máximo de filas que contienen datos en un control Hoja de cálculo. Estas filas contienen datos, una fórmula, un formato o una altura no estándar.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FOR I = 1 _TO_ PSHEET_MySpreadsheet.NumberRow
	FOR J = 1 _TO_ PSHEET_MySpreadsheet.NumberColumn
			Trace(PSHEET_MySpreadsheet[I, J])
		END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el número de filas en una control Hoja de cálculo

`<Number of rows> = <Spreadsheet control>.NumberRow`
---

**`<Number of rows>: (Integro)`**

Número máximo de líneas que contienen datos (valor o fórmula) en un control Hoja de cálculo. 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo para la que queremos get el número de columnas.




## GridlinesVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **GridlinesVisible** Property se utiliza para: 

- Determinar si las líneas de la cuadrícula son o no Visible en un control. 

- Mostrar u ocultar las líneas de la cuadrícula en una control. 




Este Property está disponible: 

- para los controles de la hoja de cálculo:

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una ventana.

	- en un reporte.




- para los controles del editor de Diagram.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Gridlines hidden] button
PSHEET_SpreadsheetControl.GridlinesVisible = False
```

```wl
// The CBOX_Options control includes the following options: 
// Library
// Styles
// Toolbar
// Grid
DIAGEDT_Diagram.LibraryPanelVisible = CBOX_Options[1]
DIAGEDT_Diagram.ModifierPanelVisible = CBOX_Options[2]
DIAGEDT_Diagram.ToolbarVisible = CBOX_Options[3]
DIAGEDT_Diagram.GridlinesVisible = CBOX_Options[4]
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si las líneas de cuadrícula son Visible en un control Hoja de cálculo

`<Result> = <Control used>.GridlinesVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las líneas de cuadrícula son Visible en el control,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Hoja de cálculo en una ventana,

- y control Hoja de cálculo y reporte.  

- un elemento de Diagram Editor en una página o ventana.





<a name="SYNTAX2"></a>

### Mostrar u ocultar las líneas de la cuadrícula en una control Hoja de cálculo

`<Control used>.GridlinesVisible = <Visibility>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Hoja de cálculo en una ventana,

- y control Hoja de cálculo y reporte.  

- un elemento de Diagram Editor en una página o ventana.




**`<Visibility>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las líneas de cuadrícula del control Hoja de cálculo deben ser visibles,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## AppointmentContinuous (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **CitaContinua** Property se utiliza para determinar o cambiar la forma en que se muestran las citas de varios días en un control Planificador.  

Recordatorio: Se puede visualizar una Appointment de varios días en una control Planificador: 

- como una única Appointment continua. 

- como un Appointment por día. 




Observación: Esta propiedad corresponde a la opción "Visualización continua si es de varios días" de la ventana control Planificador description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SCH_MySchedule.AppointmentContinuous = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir el modo de visualización de Appointment de varios días

`<Result> = <Scheduler control>.AppointmentContinuous`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de visualización: 


|   |   |
| --- | --- |
| *Default* | Las citas se visualizan según los parámetros de visualización definidos para la control Planificador. |
| *False* | Las citas se muestran cada día. |
| *True* | Las citas se muestran en modo continuo. |



**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.


<a name="SYNTAX2"></a>

### Cambio del modo de visualización de Appointment de varios días

`<Scheduler control>.AppointmentContinuous = <New display mode>`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<New display mode>: (Constante de tipo Integer)`**

Modo de visualización a utilizar: 


|   |   |
| --- | --- |
| *Default* | Las citas se visualizan según los parámetros de visualización definidos para la control Planificador. |
| *False* | Las citas se muestran cada día. |
| *True* | Las citas se muestran en modo continuo. |

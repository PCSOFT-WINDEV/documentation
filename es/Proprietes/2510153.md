


## CheckSpell (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **CheckSpell** se utiliza para activar o desactivar el corrector ortográfico automatic. 

Una Line roja aparece automáticamente bajo las palabras que no se encuentran en el diccionario: 

- en los controles de edición. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en las columnas de control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en los controles del procesador de textos.




**Observación**: Esta Property es equivalente a la opción "Revisar la ortografía" encontrada: 

- en la pestaña "Detalles" de la ventana de description de los controles de edición,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en la pestaña "Detalles" de la ventana de description de las columnas de control Tabla.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable the spell check in an edit control
EDT_Text.CheckSpell = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el corrector ortográfico está activado

`<Result> = <Control used>.CheckSpell`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el corrector ortográfico está activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Campo de entrada ,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de una control Tabla,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto.





<a name="SYNTAX2"></a>

### Activar o desactivar el corrector ortográfico

`<Control used>.CheckSpell = <Enable/Disable>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Campo de entrada ,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de una control Tabla,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto.




**`<Enable/Disable>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el corrector ortográfico debe estar activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

### Corrector ortográfico utilizado en los controles de edición y en las columnas de un control Tabla
<a name="corrector_ortografico_utilizado_los_controles_edicion_las_columnas_control_tabla_ELTPARAGRAPHE000114"></a>

En los controles de edición y las columnas de un control Tabla en una aplicación WINDEV, el **CheckSpell** Property utiliza: 

- el diccionario de OpenOffice 2.0. 
	Para obtener más información, consulte [Corregir la ortografía con OpenOffice](../Editeurs/1013146.md).

- o el verificador integrado.
	En ese caso: 

	- los diccionarios francés e inglés se copian automáticamente (así como su respectiva licencia) en el directorio ejecutable (opción configurable en el [proyecto description ventana, pestaña "Opciones".](../Editeurs/9500227.md)). 

	- los diccionarios de francés e inglés están cargados por Default. 

	- para utilizar otros diccionarios, deben ser associated con un idioma y cargados en memoria con [Carga de diccionario](../WDLang1/1000022958.md). 




- Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md).




[ConfigureSpellCheck](../WDLang1/3025052.md) se utiliza para: 

- elegir el corrector ortográfico utilizado. 

- activar el corrector ortográfico para todos los controles de edición de la aplicación. 

- configurar el idioma utilizado (si se utiliza el corrector ortográfico de OpenOffice). 




Para las palabras no reconocidas, el menú contextual Default de WINDEV propone las siguientes opciones:

- Sustitución de la palabra por una palabra válida (se proponen las 8 primeras sugerencias de OpenOffice o del inspector integrado).

- Añade la palabra al diccionario.

### Corrector ortográfico utilizado en los controles de procesamiento de textos
<a name="corrector_ortografico_utilizado_los_controles_procesamiento_textos_ELTPARAGRAPHE000167"></a>

En los controles de procesamiento de textos, el **CheckSpell** Property utiliza los diccionarios Hunspell para comprobar la ortografía.

Por Default, si su proyecto contiene un control Procesador de texto: 

- los diccionarios francés e inglés se copian automáticamente (así como su respectiva licencia) en el directorio ejecutable (opción configurable en el [proyecto description ventana, pestaña "Opciones".](../Editeurs/9500227.md)). 

- los diccionarios de francés e inglés están cargados por Default. 




Para utilizar otros diccionarios, deben ser associated con un idioma y deben ser cargados en memoria por [Carga de diccionario](../WDLang1/1000022958.md). 

Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md).

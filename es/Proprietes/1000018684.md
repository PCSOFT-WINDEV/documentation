
## CompressedSize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CompressedSize** Property se utiliza para determinar el tamaño comprimido de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [zipArchive](../WDLang3/1000018679.md).

- una variable de tipo [zipArchivedFile](../WDLang3/1000019044.md).

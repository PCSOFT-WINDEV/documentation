


## PageByPagePrint (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PageByPagePrint** Property se utiliza para:

- Pedir a un reporte que se imprima página por página. Cada página de la reporte se enviará por separado a la cola de impresión.. Esta opción se utiliza para reducir el tamaño del spooler de impresión, por ejemplo.

- Descubra el modo de impresión de la reporte (página por página o todas a la vez).




**Observación**: Para imprimir una reporte página por página, también puede marcar "Enviar cada página por separado a la impresora" en la pestaña "Opciones" de la ventana reporte description.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print the report page by page
RPT_ReportName.PageByPagePrint = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de impresión de la reporte

`<Result> = <Report used>.PageByPagePrint`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprime el reporte página por página,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Pidiendo que se imprima un reporte página por página

`<Report used>.PageByPagePrint = <Print mode>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte a utilizar.

**`<Print mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprime el reporte página por página,

- <u><u><u><u>False</u></u></u></u> si se imprime todo el reporte.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### ¿En qué evento WLanguage es posible usar el PageByPagePrint Property?
<a name="¿en_que_evento_wlanguage_posible_usar_docparampagetitleshort_property_ELTPARAGRAPHE000078"></a>

Se recomienda utilizar la propiedad **PageByPagePrint** en el reporte "Apertura" Event. En otros eventos asociados con la reporte, esta Property será ignorada.
<a name="NOTE0_2"></a>


### Límite: Calculó control que está usando el número total de páginas de reporte
<a name="limite_calculo_control_que_esta_usando_numero_total_paginas_reporte_ELTPARAGRAPHE000089"></a>

El reporte no debe imprimirse página por página si este reporte contiene un "Número de páginas" calculado control. De hecho, la reporte debe estar completamente ejecutada para calcular el número de páginas.. Si, no obstante, se solicita una impresión página por página, el número de páginas se sustituirá por "...".

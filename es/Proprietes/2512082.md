


## Secure (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**Secure** Property se utiliza para configurar el nivel de seguridad del cifrado de archivos de datos. El nivel de seguridad puede ser:

- definido por Default,

- alto. Atención: en este caso, no podrá recuperar la contraseña del archivo de datos.




Es posible:

- Averigua el nivel de seguridad para la encriptación (archivo de datos definido en la editor de análisis o por programación).

- Definir el nivel de seguridad para la encriptación (sólo cuando el archivo de datos se describe mediante programación).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the encryption security level of the Orders data file
Orders.Secure = True
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinación del nivel de seguridad del cifrado

`<Security level> = <Data file>.Secure`
---

**`<Security level>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nivel de seguridad es máximo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definición del nivel de seguridad de encriptación (archivo descrito mediante programación)

`<Data file>.Secure = <Security level>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Security level>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nivel de seguridad debe ser máximo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta propiedad corresponde a la opción "Activar alto nivel de seguridad " de la pestaña "Detalles" de la ventana del archivo de datos description.

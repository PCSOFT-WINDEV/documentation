


## InitialItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**InitialItem** se utiliza para averiguar el nombre inicial de un elemento encontrado en una consulta. Este nombre se conoce por el alias de este elemento.

**Observación**: El [InitialFile](../Proprietes/2512029.md) Property se utiliza para obtener el nombre original del archivo de datos associated a un elemento en una consulta. Este nombre se conoce por el alias de este elemento.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HExecuteSQLQuery("SampleQuery", "SELECT Customer.CustName AS MySuperCustomer FROM Customer")
Trace(SampleQuery.MySuperCustomer.InitialFile) // Customer
Trace(SampleQuery.MySuperCustomer.InitialItem) // CustName
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Result> = <Query>.<Item alias>.InitialItem`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre inicial del artículo associated con el alias del artículo,

- Vaciar la cadena (""") si el alias del ítem es un agregado o si corresponde a un contador.




**`<Query>: (Cadena de caracteres)`**

Nombre de la consulta que contiene el artículo. Este parámetro también puede corresponder al nombre de un archivo de datos HFSQL, al nombre de una vista HFSQL, etc.

**`<Item alias>: (Cadena de caracteres)`**

Nombre del elemento en el resultado de la consulta (o en el archivo de datos, vista, etc.).

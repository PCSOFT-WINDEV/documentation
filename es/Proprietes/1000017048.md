


## ByAddress (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ByAddress** Property se utiliza para determinar si un parámetro Procedure fue pasado por Address.

Si el parámetro fue pasado por dirección, su valor puede ser modificado en el código Procedure.

Observaciones: 

- Esta Property es idéntica a la [ByReference](../Proprietes/1000024379.md).

- Los conceptos "Paso de parámetros por reference" y "Paso de parámetros por Address" son idénticos.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Index is int = 1
Index2 is int = 3
Index3 is int = 4
// Before the call to the procedure, the variables have their default values
AddOne(Index, Index2, Index3)
// After the call to the procedure, the values of the variables have been changed
```

```wl
// -- Declare the procedure
PROCEDURE AddOne(*)
FOR I = 1 _TO_ MyParameters.Count
	IF MyParameters[I].ByAddress = True THEN
		MyParameters[I] += 1
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Parameter used>.ByAddress`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el parámetro se pasó por referencia,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Parameter used>: (Cadena de caracteres)`**

Nombre del parámetro utilizado. Este parámetro se define mediante la palabra clave **MyParameters**, especificando el índice del parámetro utilizado. Por ejemplo: MisParámetros[ 1] a manipular el primer parámetro pasado al Procedure.

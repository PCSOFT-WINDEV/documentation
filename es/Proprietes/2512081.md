
## Second (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Segundo** Property se utiliza para:

- Recupera el número de segundos de un [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.

- Modificar el número de segundos en un [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.

- Recuperar los segundos o el número de segundos de un elemento Date (en formato "Date and Time" o "Duration").
	

- Modificar los segundos o el número de segundos en un elemento Date (en formato "Date and Time" o "Duration").
	




**Observación**: El **Second** Property se utiliza para cambiar fácilmente los tiempos y las duraciones.

**Recordatorio**: Los elementos de fecha se utilizan para gestionar:

- Fechas "Año - Mes - Día" (formato AAAAMMDD)

- "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" fechas y horas (formato AAAAMMDDHHmmSSCCC)

- "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" duraciones (formato +DHHMMSSCCC)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Operations performed on a DateTime variable
StartTime is DateTime = "200112311230250"
// Add 50 seconds
StartTime.Second += 50// Modify the seconds
StartTime.Second = 50
```


<a name="Example2"></a>



```wl
// Operations performed on a DateTime item
Work.StartTime = "200112311230250"
// Add 50 seconds
Work.StartTime.Second += 50// Modify the seconds
Work.StartTime.Second = 50
```

<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de segundos de una Hora, FechaHora o Duración Variable

`<Result> = <Time>.Second`
---

**`<Result>: (Integro)`**

Número de segundos en 2 dígitos.

**`<Time>: (FechaHora o Duración)`**

Nombre de la Variable de tipo [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del número de segundos en una Hora, FechaHora o Duración Variable

`<Time>.Second = <New number of seconds>`
---

**`<Time>: (FechaHora o Duración)`**

Nombre de la Variable de tipo [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) que se va a utilizar.

**`<New number of seconds>: (Cadena de caracteres o entero)`**

Nuevo número de segundos (entre 1 y 59).  


<a name="SYNTAX3"></a>

### Averiguar el número de segundos en un elemento de la Fecha (en formato "Fecha y hora" o "Duración")

`<Result> = <Data file>.<Item>.Second`
---

**`<Result>: (Integro)`**

Número de segundos en 2 dígitos.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX4"></a>

### Modificación del número de segundos en un elemento de la Fecha (en formato "Fecha y hora" o "Duración")

`<Data file>.<Item>.Second = <New number of seconds>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<New number of seconds>: (Cadena de caracteres o entero)`**

Nuevo número de segundos (entre 1 y 59).

## Observaciones




### Gestión de los segundos
<a name="gestion_los_segundos_ELTPARAGRAPHE000177"></a>

- **Caso 1: Asignación directa**
	Durante una asignación directa (por ejemplo, *MyWork.MyDateTime.Second = n*), los segundos deben encontrarse entre 00 y 59. Se produce un error WLanguage si se especifica un número de segundos no válido..
	La notación *MiTrabajo.MiHora.Segundo= MiTrabajo.MiHora.Segundo + 5* puede generar un error de ejecución.

- **Caso 2: Operaciones en los segundos**
	Al realizar operaciones en elementos de tipo Date (en formato 'Fecha y hora' o 'Duración'), los segundos están comprendidos entre 00 y 59. La suma o la resta de segundos tiene en cuenta automáticamente los cambios de 59 segundos a 1 minuto.



<a name="NOTE0_2"></a>




### Operadores disponibles para los segundos
<a name="operadores_disponibles_para_los_segundos_ELTPARAGRAPHE000192"></a>

Los siguientes operadores aritméticos pueden utilizarse con el **Second** Property: 

- ++ y --

- += y -=





```wl
// Example on an item
Work.StartDateTime = "200411261324320"
Work.StartDateTime.Second++     // Add 1 second
Work.StartDateTime.Second+=5  // Add 5 seconds
Work.StartDateTime.Second-=5  // Subtract 5 seconds
```




## InitialValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Valor inicial** Property se utiliza para get el valor inicial de un control en una ventana o página. Este valor corresponde al valor inicial definido en el editor (pestaña "Contenido" de la ventana del control description).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the value of "EDT_Customer" with its initial value
EDT_Customer = EDT_Customer.InitialValue
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Initial value> = <Control used>.InitialValue`
---

**`<Initial value>: (Cualquier tipo)`**

Valor inicial de un control especificado.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### ActiveX control
<a name="activex_control_ELTPARAGRAPHE000077"></a>

En el caso de una control de ActiveX, el valor inicial corresponde al identificador COM (ProgID) del ActiveX seleccionado al crear la ventana.
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### control Gráfico
<a name="control_grafico_ELTPARAGRAPHE000100"></a>

En el caso de un control Gráfico, el valor inicial corresponde a la imagen mostrada en el control Background, seleccionada al crear la ventana o la página.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Control Cajón de navegación
<a name="control_cajon_navegacion_ELTPARAGRAPHE000109"></a>

Para un control Cajón de navegación, el **InitialValue** Property corresponde al estado de un control (expandido (1) o colapsado(0)) seleccionado cuando se creó el control.
<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Control Combo Box
<a name="control_combo_box_ELTPARAGRAPHE000120"></a>

Para un control Combo Box, se utiliza el **InitialValue** Property para volver a leer la cadena inicial asignada al elemento.

Por ejemplo, si se utiliza el siguiente código para añadir un elemento: 

```wl
ListAdd(COMBO_Combo1, gImage(nImageNum,i) + "Customer.CustomerName" + i + ...
	gStoredValue("Customer.CustomerName" + i))
```
se utiliza el siguiente código para volver a leer la cadena inicial: 

```wl
COMBO_Combo1[1].InitialValue
```


Observación: El resultado de [DisplayedValue](../Proprietes/2510129.md) no permite recuperar información sobre el contenido añadido por [gStoredValue](../WDLang1/1410088363.md). 
<a name="NOTE0_6"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000144"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **InitialValue** Property no se puede utilizar en:

- una ventana, 

- un control Estático,

- un control Botón, 

- a control Forma,

- una opción en un botón de opción o casilla de verificación,

- un fila en un list box, listview o tabla,

- una celda o una columna en un tabla,

- a menu,

- un panel de pestañas,

- un grupo de controles,

- un elemento en una control TreeView.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **InitialValue** Property se aplica únicamente a los siguientes elementos:

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control Tabla.

- Supercontrol.

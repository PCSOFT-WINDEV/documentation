
## DownloadProgress (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **DownloadProgress** se utiliza para obtener el progreso de la descarga Notification. Esta propiedad se puede utilizar en:

- una variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

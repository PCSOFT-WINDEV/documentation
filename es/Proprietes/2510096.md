


## FontCondensed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontCondensed** Property se utiliza para gestionar el espaciado de los caracteres. Es posible:

- Determina si los caracteres de un texto están condensados.

- Condense (o no) los caracteres.




WLanguage admite los siguientes tipos de espaciado de caracteres:

- Normal: el espacio entre los caracteres es Normal.

- Condensado: se reduce el espacio entre caracteres (**FontCondensed** Property).

- Extendido: se aumenta el espacio entre caracteres ([FontExtended](../Proprietes/2510104.md) property).

- Grande: el espacio entre caracteres se incrementa al máximo ([FontLarge](../Proprietes/2510101.md) property).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Condense the characters in the text displayed in "STC_StaticControl"
STC_StaticControl.FontCondensed = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si los caracteres del texto mostrado en una control están condensados

`<Result> = <Control used>.FontCondensed`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los caracteres del texto que se muestra están comprimidos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre de control (control Estático o control calculado) a utilizar.


<a name="SYNTAX2"></a>

### Condensar los caracteres del texto mostrado en una control

`<Control used>.FontCondensed = <Condensed/Not condensed>`
---

**`<Control used>: (Nombre del control)`**

Nombre de control (control Estático o control calculado) a utilizar.

**`<Condensed/Not condensed>: (booleano)`**



- <u><u><u><u>True</u></u></u></u>  si los caracteres del texto que se muestra deben estar comprimidos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Variable de tipo Font
<a name="variable_tipo_font_ELTPARAGRAPHE000081"></a>

Si hay que aplicar fuentes idénticas (mismo tamaño, atributos, etc.) a varios controles, se recomienda utilizar una fuente creada mediante programación.

Se puede crear un tipo de letra a partir de un [Variable de tipo Font](../Motscles/1514045.md).

Para definir las características de la fuente, utilice:

- la función [FontCreate](../WDLang1/3060002.md), 

- las propiedades [Orientación](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Color](../Proprietes/2510071.md), [Extendido](../Proprietes/2513002.md), [atrevido](../Proprietes/2513001.md), [Cursiva](../Proprietes/2513007.md), [Grande](../Proprietes/2513012.md), [denominar](../Proprietes/2510082.md), [Subrayado](../Proprietes/2513013.md) y [envergadura](../Proprietes/2510125.md).




Esta fuente se puede utilizar:

- Para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- Para escribir en los gráficos (con las funciones [grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md) y [grAxisTitleFont](../WDLang3/3042059.md)).

- Para imprimir (con la función [iFont](../WDLang5/3046014.md)).

- En los controles de una ventana (propiedad [tipo de letra](../Proprietes/2510095.md)).

- En los controles de un reporte (propiedad [tipo de letra](../Proprietes/2510095.md)).



<a name="NOTE0_2"></a>


### Fuentes utilizadas por las funciones de impresión
<a name="fuentes_utilizadas_por_las_funciones_impresion_ELTPARAGRAPHE000163"></a>

Para crear una fuente que pueda ser utilizada por el [funciones de impresión](../WDLang5/3046065.md) ([iPrintBlock](../WDLang5/3046030.md), [iPrintWord](../WDLang5/3046034.md), etc.), utilice

- la función [iCreateFont](../WDLang5/3046045.md).

- una variable de tipo [tipo de letra](../Motscles/1514045.md).



<a name="NOTE0_3"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000188"></a>

En un reporte, el **FontCondensed** Property sólo puede utilizarse en:

- a control Estático.

- to URL control Enlace.

- una calculada control.

- y preajuste control.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **FontCondensed** Property no se puede utilizar en:

- a Control de número,

- una Web control Cámara,

- a control Conferencia.


Observación: Esta Property se aplica únicamente a los controles "ListView" en el modo de visualización "List Box".




## NbMemoItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NbMemoItem** se utiliza para obtener el número total de elementos de memo en un Record de:

- un archivo de datos descrito en el editor de análisis,

- un archivo de datos describió a través de la programación,

- una vista HFSQL,

- a query.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the number of memo items found in a record of the Customer data file
ResNbMemoItem = Customer.NbMemoItem
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Number of items> = <Element used>.NbMemoItem`
---

**`<Number of items>: (Integro)`**

Número de elementos de memo encontrados en una Record del objeto especificado.

**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento para el cual quiere saber el número de elementos de memo para un Record:

- nombre de un archivo de datos descrito en el editor de análisis o a través de la programación,

- nombre de una vista HFSQL,

- nombre de una consulta.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estadísticas sobre un Record
<a name="estadisticas_sobre_record_ELTPARAGRAPHE000056"></a>

- La propiedad [NbItem](../Proprietes/2512055.md) se utiliza para obtener el número total de elementos de una Record.

- La propiedad [NbKeyItem](../Proprietes/2512056.md) se utiliza para obtener el número total de elementos clave en una Record.

- La propiedad **NbMemoItem** se utiliza para obtener el número total de elementos de memoria en un Record.

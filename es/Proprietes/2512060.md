


## KeyType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **KeyType** Property se utiliza para set las características de un elemento clave. Es posible:

- Determinar si un elemento es clave para un elemento definido en la editor de análisis o mediante programación

- Definir un elemento clave (cuando el elemento se describe mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the characteristics of the Orders.OrdID key
Orders.OrdID.KeyType = hUniqueKey
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si un elemento es clave

`<Characteristic> = <Data file>.<Item>.KeyType`
---

**`<Characteristic>: (Constante de tipo Integer)`**

Las características de una posición son:


|   |   |
| --- | --- |
| *hDuplicatesKey* | Clave con duplicados. Este elemento puede tener el mismo valor para varios registros de archivos de datos. |
| *hNotAKey* | El artículo no es una llave. |
| *hPrimaryKey* | Clave primaria. El valor asignado a este elemento no será nulo y sólo estará en el archivo de datos. |
| *hUniqueKey* | Clave única. El valor asignado a este artículo será único en el archivo de datos. Ningún archivo Record tendrá el mismo valor para este artículo. |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

### Definir una clave mediante la programación

`<Item>.KeyType = <Type of key>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

**`<Type of key>: (Constante de tipo Integer)`**

Las características de una clave pueden ser:


|   |   |
| --- | --- |
| *hDuplicatesKey* | Clave con duplicados. Este elemento puede tener el mismo valor para varios registros de archivos de datos. |
| *hNotAKey* | El artículo no es una llave. |
| *hPrimaryKey* | Clave primaria. El valor asignado a este elemento no será nulo y sólo estará en el archivo de datos. |
| *hUniqueKey* | Clave única. El valor asignado a este artículo será único en el archivo de datos. Ningún archivo Record tendrá el mismo valor para este artículo. |

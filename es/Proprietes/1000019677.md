
## Recurso

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Resource** se utiliza para gestionar el recurso asociado a un Appointment. 

Esta propiedad se puede utilizar en:

- un control Planificador. Para obtener más información, consulte [Recurso](../Proprietes/1000019308.md). 

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

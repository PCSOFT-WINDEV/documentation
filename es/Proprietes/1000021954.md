


## TimestampType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TimestampType** Property se utiliza para:

- Averiguar el tipo de marca de tiempo para un artículo. Este punto se definió en la editor de análisis o a través de la programación.

- Definir el tipo de marca de tiempo para un elemento de Fecha, Hora o DateTime (cuando se describe el elemento a través de la programación).




Recordatorio: No se puede asignar un valor a un elemento "Timestamp" mediante programación. Se le asignará automáticamente la fecha de creación o de última modificación de la Record.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
AnItem.Name = "CmdDate"
AnItem.TimestampType = htsCreation
AnItem.TimestampUTC = False
HDescribeItem(AFile, AnItem)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el tipo de marca de tiempo para un artículo

`<Result> = <Data file>.<Item>.TimestampType`
---

**`<Result>: (Constante de tipo Integer)`**

Escriba ot timestamp para el artículo:


|   |   |
| --- | --- |
| *htsCreation* | Sello de tiempo al crear la Record solamente. |
| *htsLastModification* | Sello de tiempo cuando se modifica la Record. |
| *htsNone* | No hay marca de tiempo. |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado.

### Definir el tipo de sello de tiempo para un elemento de Fecha, Hora o Fecha-Hora

`<Item>.TimestampType = <Type of timestamp>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Type of timestamp>: (Constante de tipo Integer)`**

Escriba ot timestamp para el artículo:


|   |   |
| --- | --- |
| *htsCreation* | El artículo es un elemento de marca de tiempo que se rellena sólo cuando se crea la Record. |
| *htsLastModification* | El elemento es un elemento de marca de tiempo que se rellena cada vez que se modifica la Record. |
| *htsNone* | La posición no es una posición de fecha y hora. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En el editor de análisis, tienes la capacidad de definir un elemento de marca de tiempo. 

- Para definir un elemento de la marca de tiempo a través de la programación: 

	- El elemento debe ser un elemento DateTime, Date u Time,

	- el tipo de sello de tiempo se especifica con el **TimestampType** property.

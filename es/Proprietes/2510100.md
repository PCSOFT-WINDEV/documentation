


## FontName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontName** Property se utiliza para:

- Identifique el tipo de letra utilizado en una control.

- Cambiar el tipo de letra utilizado en una control.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the font used in "EDT_CustomerName" control
EDT_CustomerName.FontName = "Arial"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificación del tipo de letra utilizado en una control

`<Result> = <Control used>.FontName`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la fuente utilizada en la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Cambiar el tipo de letra utilizado en una control

`<Control used>.FontName = <New font>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  

**`<New font>: (Cadena de caracteres)`**

La nueva fuente usada en la control especificada.

## Observaciones


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000084"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **FontName** Property sólo puede utilizarse en:

- to URL control Enlace.

- a control Estático.

- una control calculada .

- un control preestablecido.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **FontName** Property no se puede utilizar en:

- a Control de número,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una Web control Cámara,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Conferencia.


Observación: Esta Property se aplica únicamente a los controles "ListView" en el modo de visualización "List Box".![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **FontName** Property se aplica únicamente a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Combo Box.

- Control TreeView.

- Columna de un control Tabla.


## EndPosition (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **EndPosition** Property se utiliza para obtener la posición final de un elemento en un tratamiento de palabras Document. Esta propiedad se puede utilizar en:

- variables de tipo [docElement](../WDLang1/1000022484.md). 

- variables de tipo [docFragment](../WDLang1/1000022482.md)

- variables de tipo [docParagraph](../WDLang1/1000022483.md).


## Destination (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Destino** Property se utiliza para definir: 

- el destino de una copia de seguridad. Esta Property puede utilizarse en una Variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

- el archivo de resultados de una solicitud. Esta propiedad se puede utilizar en:

	- una variable de tipo [httpRequest](../WDLang3/1000021158.md). 

	- una variable de tipo [restRequest](../WDLang3/1000021481.md). 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [iOutputInfo](../WDLang5/1410088954.md).

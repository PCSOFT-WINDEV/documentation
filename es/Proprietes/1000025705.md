
## Line (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **Lineal** Property se utiliza para definir el estilo Line utilizado. Esta propiedad se puede utilizar en:

- una variable de tipo [Franja de orilla](../Motscles/15140100.md). 

- una variable de tipo [docElementTextArea](../WDLang1/1000023420.md).

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).

- una variable de tipo [MapImage](../WDLang3/1000025523.md).

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md).

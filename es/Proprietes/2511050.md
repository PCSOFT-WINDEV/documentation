


## NbLabel (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbLabel** Property se utiliza para:

- Averigüe el número de etiquetas impresas en todo el ancho de la página.

- Modificar el número de etiquetas impresas en todo el ancho de la página.




**Observación**: El número de etiquetas impresas a lo largo de una página puede ser conocido y/o modificado en el editor de reportes: en la pestaña "Modificación", en el grupo "Reporte", seleccione "Formato y etiquetas".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=nbetiquette.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increases the number of labels that will be printed across the page width
RPT_CustLabel.NbLabel = RPT_CustLabel.NbLabel + 2
// Is equivalent to: RPT_CustLabel.NbLabel += 2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de etiquetas impresas en el ancho de la página

`<Result> = <Report used>.NbLabel`
---

**`<Result>: (Integro)`**

Número de etiquetas que se imprimirán en todo el ancho de la página.

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.  


<a name="SYNTAX2"></a>

### Modificando el número de etiquetas impresas a lo largo del ancho de la página

`<Report used>.NbLabel = <New number>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.

**`<New number>: (Integro)`**

Nuevo número de etiquetas que se imprimirán en todo el ancho de la página.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modificación del número de etiquetas
<a name="modificacion_del_numero_etiquetas_ELTPARAGRAPHE000073"></a>

Cuando se modifica el número de etiquetas a través de la programación, las etiquetas no se redimensionan automáticamente. Por lo tanto, le aconsejamos que compruebe el tamaño de las etiquetas.
<a name="NOTE0_2"></a>


### Modificación de las dimensiones de las etiquetas
<a name="modificacion_las_dimensiones_las_etiquetas_ELTPARAGRAPHE000080"></a>

- Para determinar o modificar la altura de las etiquetas en una "Etiqueta" reporte, utilice el [LabelHeight](../Proprietes/2511069.md) Property.

- Para determinar o modificar el ancho de las etiquetas en una "Etiqueta" reporte, utilice el [LabelWidth](../Proprietes/2511055.md) Property.

- Para determinar o modificar el espaciado horizontal entre etiquetas en una "Etiqueta" reporte, utilice el [HSpace](../Proprietes/2511067.md) Property.

- Para determinar o modificar el espaciado vertical entre etiquetas en una "Etiqueta" reporte, utilice el [VSpace](../Proprietes/2511065.md) Property.



<a name="NOTE0_3"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000102"></a>

El **NbLabel** Property sólo se aplica a los informes de "Etiqueta".

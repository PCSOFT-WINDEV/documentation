
## MaxNbCard (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **MaxNbCard** obtiene y establece el número máximo de tarjetas en un control Kanban. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Maximum number of cards in LIST_To_validate
// No more cards can be added in the list
// once this number is reached. 
LIST_To_validate.MaxNbCard = 2
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el número máximo de tarjetas permitidas

`<Result> = <Kanban list>.MaxNbCard`
---

**`<Result>: (Entero)`**



- Número máximo de tarjetas que pueden mostrarse en la lista Kanban,

- 0, no se puede agregar ninguna tarjeta a la lista Kanban,

- -1, se puede mostrar un número ilimitado de tarjetas. 




**`<Kanban list>: (Nombre del control)`**

Control Lista Kanban a manipular. 


<a name="SYNTAX2"></a>

### Establecer el número máximo de tarjetas permitidas

`<Kanban list>.MaxNbCard = <Maximum value>`
---

**`<Kanban list>: (Nombre del control)`**

Control Lista Kanban a manipular. 

**`<Maximum value>: (Entero)`**



- Número máximo de tarjetas que pueden mostrarse en la lista Kanban,

- 0, no se puede agregar ninguna tarjeta a la lista Kanban,

- -1, se puede mostrar un número ilimitado de tarjetas. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Una vez alcanzado el número máximo de tarjetas, el usuario no podrá soltar más tarjetas en la lista Kanban. El pie de página de la lista (si existe) aparece en gris.

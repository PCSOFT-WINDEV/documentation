


## TestMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TestMode** Property determina si el anuncio control muestra un anuncio de prueba. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ADV_MyAd.TestMode = False _AND_ ADV_MyAd.AdLoaded = True THEN
	NbAdViewed = NbAdViewed + 1
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Ad control>.TestMode`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la publicidad que se muestra es de prueba. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.  




**`<Ad control>: (Nombre del control)`**

Nombre del anuncio control a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento del modo de prueba
<a name="modo_funcionamiento_del_modo_prueba_ELTPARAGRAPHE000040"></a>

AdMob no permite a los proveedores de aplicaciones hacer clic en sus propios anuncios o aumentar el número de anuncios mostrados. 

El modo de prueba del anuncio control se utiliza para mostrar "anuncios falsos" en los que el clic no provoca ningún ingreso.

El modo de prueba es automático: 

- activada al iniciar la aplicación en el emulador. 

- desactivado en tiempo de ejecución en la Phone o en la tableta.

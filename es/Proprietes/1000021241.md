
## DescriptionStatusCode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **description StatusCode** se utiliza para obtener el estado description de la respuesta a la solicitud. Esta propiedad se puede utilizar en:

- una variable de tipo [httpResponse](../WDLang3/1000021165.md). 

- una variable de tipo [restResponse](../WDLang3/1000021477.md).

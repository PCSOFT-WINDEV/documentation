


## Selected (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**Selected** se utiliza para:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Averigua si se selecciona una fila (en un Listado, ListView o control Tabla) y selecciona una fila (en un Listado, ListView o control Tabla). 
	**Observación**: A diferencia de [ListSelect](../WDLang1/3049011.md), este Property se utiliza para averiguar directamente si el fila que corresponde al índice especificado está seleccionado.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Averigüe si una columna o una celda de una control Tabla está seleccionada y seleccione una columna o una celda de una control Tabla (para los controles de tabla con fila o selección de columna)..   





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Selects row 5 in the CITY List Box control
LIST_City[5].Selected = True
```

## Sintaxis
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
### Averiguar si un elemento es seleccionado

`<Result> = <Control used>[<Row index>].Selected`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento está seleccionado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Nombre del List Box, ListView o control Tabla: para saber si se ha seleccionado la fila.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la columna: usado para averiguar si la celda está seleccionada. Si no se especifica el índice fila, permite determinar si la columna está seleccionada. 




**`<Row index>: (Integro)`**

Índice de la fila a utilizar.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
### Selección de un elemento

`<Control used>[<Row index>].Selected = <Selection>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Nombre del List Box, ListView o control Tabla: se utiliza para seleccionar un fila o no.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la columna: se usa para seleccionar una celda o no. Si no se especifica el índice fila, permite seleccionar o deseleccionar la columna tabla. 




**`<Row index>: (Integro)`**

Índice de la fila a utilizar.

**`<Selection>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento debe estar seleccionado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### control List Box de selección única y selección múltiple
<a name="control_list_box_seleccion_unica_seleccion_multiple_ELTPARAGRAPHE000150"></a>

En los controles de cuadro de lista de selección múltiple, el **Selected** Property se utiliza para:

- averiguar las líneas seleccionadas.

- seleccione varias líneas.




En los controles de cuadro de lista de selección única, cuando se selecciona un fila con la propiedad **Selected**, el fila previamente seleccionado se deselecciona automáticamente.


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Límites
<a name="limites_ELTPARAGRAPHE000172"></a>

El **Selected** Property se aplica únicamente a:

- Controles de mesa.

- Controles de List Box.

- Controles ListView.

- Columnas de la tabla.

- Celdas de tabla.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **Selected** Property se aplica sólo a los controles de los cuadros de lista.

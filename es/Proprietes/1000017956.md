
## Type (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Tipo** Property se utiliza para get y cambiar el tipo de un elemento. Esta propiedad se puede utilizar en:

- los mandos de una ventana o Page. Para obtener más información, consulte [Tipo Property](../Proprietes/2510131.md).

- una variable de tipo [gglPostalAddress](../WDLang5/1000017575.md).

- una variable de tipo [gglContact](../WDLang5/1000017434.md). 

- una variable de tipo [gglEmail](../WDLang5/1000017571.md).

- una variable de tipo [gglInstantMessaging](../WDLang5/1000017569.md).

- una variable de tipo [gglOrganization](../WDLang5/1000017756.md).

- una variable de tipo [gglPhoneNumber](../WDLang5/1000017573.md).

- una variable de tipo [rssEnclosure](../WDLang5/1000017789.md).

- una variable de tipo [xlsCell](../WDLang5/1000017472.md).

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).




- una Variable cuyo tipo se genera automáticamente durante la [importación de un servicio web](../WDLang3/3086001.md).




- una variable de tipo [Definition](../WDLang1/1000019540.md). 

- una variable de tipo [Repetition](../WDLang1/1000019262.md).

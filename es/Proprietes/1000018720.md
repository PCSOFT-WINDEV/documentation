
## Priority (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Prioridad** Property se utiliza para get y cambiar el nivel de prioridad de un elemento. 

Esta propiedad se puede utilizar en:

- la estructura [Email](../WDLang3/3032029.md).

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [notificationCategory](../WDLang3/1000023981.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Document](../WDLang1/1000022461.md).

- una variable de tipo [Thread](../WDLang1/1000024879.md).




## CursorEnd (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>


### Definition
<a name="definition_ELTPARAGRAPHE000010"></a>

El cursor del ratón permite determinar fácilmente dónde se realizará una acción. En el caso de un control en modo de edición, el cursor se convierte en una Line vertical parpadeante, lo que le permite identificar la ubicación del siguiente carácter escrito. 
Los caracteres seleccionados se muestran en colores invertidos. La posición del cursor corresponde al inicio de la zona que se muestra en colores invertidos. 

Por ejemplo: 


| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=position1.gif)<br> | El cursor de entrada se coloca al principio del área de entrada. Su posición es 1. |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Position_fin.gif)<br> | El cursor de entrada se coloca al final del área de entrada. Su posición es 7 en este ejemplo. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=position_sel.gif)<br> | El cursor de entrada se coloca al principio del área en "video invertido" (posición 3 en este ejemplo). La propiedad **Cursor** permite obtener y cambiar el final del área seleccionada. |
|   |   |


<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**CursorEnd** se utiliza para conocer y modificar la posición final de selección del cursor del ratón en uno de los siguientes controles:

- Control Campo de entrada.

- Combo Box con entrada.

- "Editar control" y "Editar Combo Box" en un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto. 




Observación: Para get el texto seleccionado en un control Campo de entrada, un Combo Box editable o una columna en un control Tabla, puede utilizar [Texto Seleccionado](../Proprietes/1000017193.md).



**Consejo**: Para seleccionar una sección de texto en una control Campo de entrada desde un botón:

1. La opción "Borrado automático" no debe estar habilitada en el control Campo de entrada (a menos que se deba borrar todo el contenido del control).

2. El botón debe ser un botón de "Interrupción" y contener el siguiente código, por ejemplo.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code of BTN_Selection button (Interrupt button)
// Select the entire text
EDT_Edit1.Cursor = 1
EDT_Edit1.CursorEnd = Length(EDT_Edit1) + 1
```
<a name="Example2"></a>

```wl
// Deselect the text
EDT_Edit1.CursorEnd = EDT_Edit1.Cursor
```
<a name="Example3"></a>

```wl
// Place cursor at the end of the text
EDT_Edit1.CursorEnd = Length(EDT_Edit1) + 1
EDT_Edit1.Cursor = EDT_Edit1.CursorEnd
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la posición del último personaje seleccionado

`<Position> = <Control used>.CursorEnd`
---

**`<Position>: (Integro)`**

Posición del último carácter seleccionado + 1. Si no se selecciona ningún carácter, las propiedades [Cursor](../Proprietes/2510025.md) y **CursorEnd** son equivalentes. Si la control no está en edición, la posición es 0.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=image17.gif)


**`<Control used>: (Nombre del control)`**

Nombre de la control en la que se realizó la selección.  


<a name="SYNTAX2"></a>

### Seleccionando una set de caracteres

`<Control used>.CursorEnd = <Number of selected characters>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control en la que debe realizarse la selección.

**`<Number of selected characters>: (Integro)`**

La cadena seleccionada se encuentra entre la posición del cursor y el valor pasado a la función **CursorEnd** Property. 
El valor mínimo posible es 1 (cursor a la izquierda). El mayor valor corresponde al número de caracteres que se encuentran en la control + 1. 
**Atención**: Si el contenido del **CursorEnd** Property se asigna al propio Property, el texto se deselecciona: no se muestra ningún carácter en el vídeo inverso.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Límite: Selección en vídeo inverso
<a name="limite_seleccion_video_inverso_ELTPARAGRAPHE000137"></a>

La selección en video inverso está disponible sólo si el control está siendo editado. Si el control no está en modo de edición, el **CursorEnd** Property devuelve 0.

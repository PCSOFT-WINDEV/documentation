
## Unalterable (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Inalterable** Property se utiliza para configurar la inalterabilidad de un archivo de datos: el archivo de datos no puede modificarse; sólo pueden hacerse adiciones.

Es posible:

- Averiguar si un archivo de datos es inalterable o no (archivo de datos definido en la editor de análisis o mediante programación).

- Definir las características relativas a la inalterabilidad de un archivo de datos (cuando se describe el archivo de datos sólo mediante programación).




Recordatorio: Hacer inalterables los archivos: 

- evita errores de programación impidiendo la modificación o supresión (mediante [HModify](../WDLang4/3044042.md)/[HDelete](../WDLang4/3044018.md), o mediante una consulta). 

- añade un hash del contenido de Record a cada Record. Este hash es un hash HMAC-SHA256, cuya clave se deriva de la contraseña de encriptación del archivo.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description

// Describe the "Customer" data file
AFile.Name = "Customer"
AFile.Type = hFileNormal
AFile.Unalterable = True

// Item description 
AnItem.Name = "Message"
AnItem.Type = hItemText
AnItem.Size = 40
HDescribeItem(AFile, AnItem)

// Validate the data file description 
HDescribeFile(AFile)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si un archivo de datos es inalterable o no

`<Result> = <Data file>.Unalterable`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos es inalterable,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).


<a name="SYNTAX2"></a>

### Definición de las características relativas a la inalterabilidad de un archivo de datos (archivo de datos definido mediante programación)

`<Data file>.Unalterable = <Unalterability>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Unalterability>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos debe ser inalterable,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

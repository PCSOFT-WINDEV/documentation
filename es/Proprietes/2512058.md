


## NbComponent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbComponent** Property se utiliza para get el número de elementos de una clave compuesta.

**Observación**: El artículo fue creado en la editor de análisis o mediante programación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the number of components in the Customer.LNameFName composite key
ResNbComp = Customer.LNameFName.NbComponent
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Data file>.<Item>.NbComponent`
---

**`<Result>: (Integro)`**



- Número de elementos en la clave compuesta,

- 1 si la llave no es una llave compuesta.




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición clave utilizada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Manejo de una clave compuesta
<a name="manejo_una_clave_compuesta_ELTPARAGRAPHE000052"></a>

La [Componente](../Proprietes/2512020.md) Property se utiliza para obtener y cambiar el nombre del elemento associated con cada componente de la clave.

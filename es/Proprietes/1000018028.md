
## NumberColumn (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NúmeroColumna** Property se utiliza para get y cambiar el número de columnas de un elemento. Esta propiedad se puede utilizar en:

- los controles de una ventana. Para obtener más información, consulte [NumberColumn en los controles de la ventana](../Proprietes/2510083.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docFormatMulticolumn](../WDLang1/1000023580.md).


## AnimationDirectionAlernate (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **AnimationDirectionAlernate** permite: 

- Comprobar si la animación en un control Barra de progreso infinita se reproduce primero hacia adelante y luego hacia atrás. 

- Configurar la animación en un control Barra de progreso infinita para que se reproduzca primero hacia delante y luego hacia atrás.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PROGBAR_Endless1.Image = "Wheel.png"
PROGBAR_Endless1.NbAnimationSteps = 36 
PROGBAR_Endless1.AnimationDirectionAlernate = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Comprobar si la animación se reproduce primero hacia adelante y luego hacia atrás

`<Result> = <Progress Bar control>.AnimationDirectionAlernate`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la animación en el control Barra de progreso se reproduce en ciclos alternos,

- <u><u><u><u>False</u></u></u></u> si la animación en el control Barra de progreso se reproduce solo hacia adelante. 




**`<Progress Bar control>: (Nombre del control)`**

Nombre del control Barra de progreso infinita a utilizar. 


<a name="SYNTAX2"></a>

### Configurar la animación para que se reproduzca en ciclos alternos o solo hacia adelante

`<Progress Bar control>.AnimationDirectionAlernate = <New value>`
---

**`<Progress Bar control>: (Nombre del control)`**

Nombre del control Barra de progreso infinita a utilizar. 

**`<New value>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> para reproducir la animación en ciclos alternos. 

- <u><u><u><u>False</u></u></u></u> para reproducir la animación solo hacia adelante.

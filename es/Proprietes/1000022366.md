


## CodeJS (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CodeJS** Property se utiliza para leer y modificar el código JavaScript associated con un componente web control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(CMP_MyWebComponent.CodeJS)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguación del código JavaScript associated con el componente web control

`<Result> = <Web Component control>.CodeJS`
---

**`<Result>: (Cadena de caracteres)`**

Código JavaScript associated con el componente web control. Este código JavaScript se escribirá en la página. 

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificación del código JavaScript associated con el componente web control

`<Web Component control>.CodeJS = <JavaScript code>`
---

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 

**`<JavaScript code>: (Cadena de caracteres)`**

Nuevo código JavaScript associated con el control.




## DndSource (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>
Algunos controles de WINDEV y WEBDEV pueden [gestionar automáticamente las operaciones de arrastrar y soltar](../WDLang1/3030012.md): simplemente seleccione la opción correspondiente en la pestaña "Detalles" de la descripción del control. Un control puede definirse como origen o destino de una operación de arrastrar y soltar.

Las operaciones de arrastrar y soltar están disponibles automáticamente: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) para los controles Campo de entrada, List Box, ListView y TreeView.


En un [operación de arrastrar y soltar establecida mediante programación](../WDLang1/3030013.md) , las propiedades **DndSource** y [DndTarget](../Proprietes/2510016.md) se utilizan para definir los controles de origen y destino de arrastrar y colocar.
<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DndSource** Property se usa para determinar y cambiar el comportamiento de arrastrar y soltar de un control (Lista, ListView , Looper, TreeView, etc.). Esta control será definida como fuente para Drag and Drop, lo que significa que el Drag and Drop se realizará desde esta control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Initialization of LIST_Source
// LIST_Source is not a target
LIST_Source.DndSource = dndInactive

// -- Initialization of LIST_Target
LIST_Target.DndTarget = dndProgram
DndEvent("Drop", "LIST_Target", dndDrop)
DndEvent("DragOver", "LIST_Target", dndDragOver)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el comportamiento de una fuente control para Drag and Drop

`<Drag and Drop> = <Control used>.DndSource`
---

**`<Drag and Drop>: (Constante)`**

Identifica el comportamiento de la fuente control para arrastrar y soltar:


|   |   |
| --- | --- |
| *dndAuto* | La operación de arrastrar y soltar se gestiona automáticamente |
| *dndInactive* | Arrastrar y soltar está desactivado para este control |
| *dndProgram* | El desarrollador establece la operación de arrastrar y soltar mediante programación |



**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando el comportamiento de una fuente control para Drag and Drop

`<Control used>.DndSource = <Drag and Drop>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Drag and Drop>: (Constante)`**

Identifica el comportamiento de la fuente control para arrastrar y soltar.


|   |   |
| --- | --- |
| *dndAuto* | La operación de arrastrar y soltar se gestiona automáticamente |
| *dndInactive* | Arrastrar y soltar está desactivado para este control |
| *dndProgram* | El desarrollador establece la operación de arrastrar y soltar mediante programación |

## Observaciones


### Operaciones de arrastrar y soltar gestionadas automáticamente o establecidas mediante programación
<a name="operaciones_arrastrar_soltar_gestionadas_automaticamente_establecidas_mediante_programacion_ELTPARAGRAPHE000105"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las operaciones de arrastrar y soltar automáticas están disponibles para controles Campo de entrada, List Box, ListView, TreeView y Tabla.

Las operaciones de arrastrar y soltar pueden establecerse mediante programación para los siguientes controles:

- Control Campo de entrada

- Lista Box y control ListView

- control TreeView

- control Tabla

- control Imagen

- Control Código de barras

- Control Botón

- Control Looper

- control Gráfico

- Control Tabla TreeView

- Control Calendario

- Control Agenda

- Control Planificador




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo están disponibles las operaciones de arrastrar y soltar automáticas.


<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000152"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La propiedad **DndSource** solo puede utilizarse con los siguientes controles:

- Control Campo de entrada, 

- Control List Box, 

- Control ListView, 

- Control TreeView,

- Control Looper, 

- Control Imagen, 

- Control Gráfico,

- Control Tabla,

- Control Tabla TreeView, 

- Control Calendario, 

- Control Agenda, 

- Control Planificador, 

- Control Botón, 

- Control Valoración.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **DndSource** solo se aplica a los controles Campo de entrada, List Box y TreeView.


## Reference (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **reference** Property se utiliza para: 

- obtener y cambiar el identificador único de uno o más correos reference relacionado a la conversación actual.

- averiguar y modificar el elemento a firmar. 




Esta propiedad se puede utilizar en:

- la estructura [Email](../WDLang3/3032029.md).

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [xmlSignature](../WDLang3/1000020736.md).

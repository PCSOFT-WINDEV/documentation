


## ID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ID** Property se utiliza para get y cambiar las identificaciones. Esta propiedad se puede utilizar en:

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [sfDeletedRecord](../WDLang5/1000018751.md).

- una variable de tipo [sfObject](../WDLang5/1000018439.md). 

- una variable de tipo [sfMergeResult](../WDLang5/1000018499.md).

- una variable de tipo [sfSeekUpdatedResult](../WDLang5/1000018561.md).

- una variable de tipo [sfUndeleteResult](../WDLang5/1000018526.md).

- una variable de tipo [sfSaveResult](../WDLang5/1000018460.md).

- una variable de tipo [sfDeleteResult](../WDLang5/1000018546.md).

- una variable de tipo [sfEmptyRecycleBinResult](../WDLang5/1000018467.md).

- y [Estructura de Email](../WDLang3/3032029.md).

- una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- una variable de tipo [GanttTask](../WDLang1/1000020881.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoGridFSFileInfo](../WDLang4/1000022417.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una variable de tipo [Contact](../WDLang3/1000020254.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [sysNotification](../WDLang1/1410087893.md).

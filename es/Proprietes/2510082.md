


## Name (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Name** permite obtener:

- el nombre de la fuente asociada a una Variable de tipo [tipo de letra](../Motscles/1514045.md).
	**Observación**: El nombre de la fuente debe corresponder a una de las fuentes instaladas en el ordenador actual.

- el nombre de un bloque control, reporte o reporte.

- el nombre de una control, grupo de control s o ventana.

- el nombre de un control, grupo de control s o Page.
	




El **Name** Property se utiliza para cambiar el nombre de la fuente asociada a un Variable.

La propiedad **Name** también puede utilizarse para manipular los nombres de diferentes objetos HFSQL. Es posible:

- Averigüe el nombre de una conexión.

- Averigüe el nombre de un objeto manejado indirectamente:

	- archivo de datos definido en el editor de análisis o mediante programación,

	- vista,

	- consulta previamente ejecutada con [HExecuteQuery](../WDLang4/3044080.md),

	- elemento definido en la editor de análisis o mediante programación,

	- enlace definido en el editor de análisis o mediante programación.




- Definir el nombre del elemento descrito:

	- cuando se describe un archivo a través de la programación,

	- cuando se describe un elemento a través de la programación,

	- cuando se describe un enlace a través de la programación,

	- cuando se describe un índice de texto completo mediante programación.







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Cuando se utiliza la palabra clave [MiInfantilVentana](../Motscles/1511021.md), el **Name** Property obtiene el nombre de la última ventana hija cerrada en una aplicación móvil.

## Sintaxis

### Averiguar el nombre del elemento especificado

`<Result> = <Element used>.Name`
---

**`<Result>: (Cadena de caracteres)`**

Nombre del elemento especificado.

**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento a utilizar.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)![HFSQL Classic](https://doc.pcsoft.fr/ext/images/us/HF.png)![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) 
### Modificar el nombre del elemento especificado

`<Element used>.Name = <New name>`
---

**`<Element used>: (Cadena de caracteres)`**

Nombre del objeto dinámico utilizado:

- archivo definido mediante programación a través del tipo [Expediente description](../WDLang4/1514065.md),

- elemento definido mediante programación a través del tipo [Artículo description](../WDLang4/1514071.md),

- enlace definido mediante programación a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

- índice de texto completo definido mediante programación a través del tipo [FullTextIndex Description](../WDLang4/1000017461.md).




***or***

Nombre de la variable de tipo [tipo de letra](../Motscles/1514045.md) que se utilizará.

**`<New name>: (Cadena de caracteres)`**

Nuevo nombre para el elemento especificado

### Alias de ventana
<a name="alias_ventana_ELTPARAGRAPHE000192"></a>

La misma ventana (ventana para niños o hermanos) se puede abrir varias veces. Para diferenciar e identificar las ventanas abiertas, se asigna un alias a cada ventana.

Para definir un alias, utilice [OpenSister](../WDLang1/3038045.md) o [OpenChild](../WDLang1/3038021.md). Este alias se utiliza para identificar ventanas con funciones que toman un nombre de ventana como parámetro ([CurrentTitle](../WDLang1/3038023.md), [NextTitle](../WDLang1/3038043.md), [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md), etc.) o para manipular un control en una ventana determinada.

Para obtener el alias de una ventana, utilice [CurrentWin](../WDLang1/3038033.md), [AnteriorGanar](../WDLang1/3038014.md) o el [Alias](../Proprietes/2510004.md) Property.

Si no se especifica ningún alias cuando se abre la ventana, WINDEV asigna automáticamente un alias a la ventana.

### Tipo de objeto
<a name="tipo_objeto_ELTPARAGRAPHE000248"></a>

El **Name** Property devuelve una cadena vacía ("") para:

- filas en los controles Tabla, Cuadro de Lista, ListView o Combo Box,

- células en controles Tabla.

### Nombre de la conexión
<a name="nombre_conexion_ELTPARAGRAPHE000262"></a>

Una conexión puede ser creada por un [Conexión](../WDLang4/1514073.md) Variable. El nombre de la conexión corresponde al nombre de la Variable. 

Si ya existe una conexión con el mismo nombre (en el análisis o descrita de antemano), el nombre de la conexión es definido por el motor HFSQL que añade un sufijo (un número) al nombre de la Variable.

En este caso, el **Name** Property devuelve el nombre de la conexión definida por el motor HFSQL.
<a name="NOTE0_6"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Averiguar el nombre de la última ventana secundaria que se cerró**
En las aplicaciones móviles, las ventanas se abren con [OpenMobileWindow](../WDLang1/1000021018.md) (o [OpenChild](../WDLang1/3038021.md)) en lugar de [entablar](../WDLang1/3038035.md). 

Puede obtener el nombre de la última ventana hija cerrada en la "Cerrar una ventana hija" Event de la ventana que la abrió (la que llamó a [OpenMobileWindow](../WDLang1/1000021018.md) o [OpenChild](../WDLang1/3038021.md)).

Simplemente utilice el **Name** Property en la palabra clave [MiInfantilVentana](../Motscles/1511021.md): 


```wl
// Close a child window
NameChildWindow = MyChildWindow.Name
```
**Observación**: La [ReturnedValue](../Proprietes/2510134.md) Property se utiliza para get el valor devuelto por la ventana hija cuando se cierra.

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000319"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La propiedad **Name** no se puede utilizar en:

- una opción de un control Botón de opción o Casilla de verificación,

- un elemento de un control TreeView.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Name** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- Control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de tabla.

- Supercontrol.

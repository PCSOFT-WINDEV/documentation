
## Icon

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Icon** puede corresponder a: 

- a la propiedad **Icon**. Esta Property se utiliza para get y cambiar la Icon de un elemento. Esta propiedad se puede utilizar en:

	- una ventana. Para obtener más información, consulte [Icono Property](../Proprietes/1000019344.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpDevice](../WDLang3/1000020903.md).




- una variable de tipo [Icon](../WDLang1/1000020811.md).

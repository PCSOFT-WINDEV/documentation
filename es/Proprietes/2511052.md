


## WidthInPixel (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **WidthInPixel** Property se utiliza para get el ancho en píxeles de un control en un reporte.

**Observación**: El borde de la control está incluido en su anchura. El [anchura](../Proprietes/2510059.md) Property se utiliza para get y cambiar el ancho de un control en milímetros.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the width of the "ITEM_CustomerName" control
ResWidth = ITEM_CustomerName.WidthInPixel
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Control used>.WidthInPixel`
---

**`<Result>: (Real)`**

El ancho (en píxeles) de la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre de la control (de una reporte) a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **WidthInPixel** Property no puede utilizarse en:

- una reporte entera,

- un bloque reporte,

- un grupo de controles,

- a control Casilla de verificación,

- a control Campo de entrada,

- un control Agenda, 

- un control Planificador, 

- a control Diagrama de Gantt,

- a control Firma.




## StartDate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StartDate** Property se utiliza para get y cambiar la fecha de inicio de un período de tiempo. Esta propiedad se puede utilizar en:

- Calendar, Controles del organizador y del programador. Para obtener más información, consulte [Fecha de inicio Property](../Proprietes/1000017426.md).

- una variable de tipo [saasSubscription](../WDLang3/1000019088.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md). 

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [GanttTask](../WDLang1/1000020881.md).

- una variable de tipo [hServerOperation](../WDLang4/1000022471.md).

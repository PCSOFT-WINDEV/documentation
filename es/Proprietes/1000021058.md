


## TitleHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TitleHeight** Property obtiene y establece: 

- la altura del título para las columnas de una Tabla o control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la altura de la barra de título de una control Panel acoplable. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la altura de la barra de título de una ventana con una barra de título personalizada. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Update the title and the height of the column title in a Table control   
COL_Column1.Title = "Result" + CR + "and Forecast" + CR + "for 2020"
TABLE_Number.TitleHeight = 50
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener la altura del título

`<Result> = <Element used>.TitleHeight`
---

**`<Result>: (Integro)`**



- Altura de los títulos de las columnas expresada en píxeles. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Altura de la barra de título de una control Panel acoplable expresada en píxeles.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Altura de la barra de título de la ventana expresada en píxeles.




**`<Element used>: (Nombre de control o ventana)`**

Nombre del control o ventana a utilizar: 

- Nombre de una mesa o control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de un control Panel acoplable.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de una ventana con una barra de título personalizada.





<a name="SYNTAX2"></a>

### Cambiar la altura del título

`<Element used>.TitleHeight = <New value>`
---

**`<Element used>: (Nombre de control o ventana)`**

Nombre del control o ventana a utilizar: 

- Nombre de una tabla o de un control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de un control Panel acoplable.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de una ventana con una barra de título personalizada.




**`<New value>: (Integro)`**

Nueva altura expresada en píxeles: 

- Altura de los títulos de las columnas. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Altura de la barra de título de una control Panel acoplable.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Altura de la barra de título de la ventana.


Si este parámetro se pone a 0, el título es Invisible.


## DisplayAsMarkdown (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **DisplayAsMarkdown** indica si un control Campo de entrada muestra o no su contenido en formato Markdown. 

Observación: Esta propiedad cambia el tipo del control Campo de entrada a "Markdown (solo lectura)". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Markdown preview]
EDT_Promo.DisplayAsMarkdown = True

// Click [Edit]
EDT_Promo.DisplayAsMarkdown = False
SetFocus(EDT_Promo)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Saber si el contenido de un control Campo de entrada se muestra en formato Markdown

`<Result> = <Edit control>.DisplayAsMarkdown`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Campo de entrada está en modo de solo lectura y muestra el contenido en formato Markdown. El desplazamiento y la selección están habilitados (excepto si el valor de la propiedad [State](../Proprietes/2510039.md) es *Grayed*). El valor de la propiedad [State](../Proprietes/2510039.md) del control no cambia. El control Campo de entrada es de tipo "Markdown (solo lectura)". 

- <u><u><u><u>False</u></u></u></u> si el control Campo de entrada está en modo de entrada.  




**`<Edit control>: (Nombre del control)`**

Nombre del control Campo de entrada utilizado. 


<a name="SYNTAX2"></a>

### Especificar si el contenido de un control Campo de entrada debe mostrarse o no en formato Markdown

`<Edit control>.DisplayAsMarkdown = <Management mode>`
---

**`<Edit control>: (Nombre del control)`**

Nombre del control Campo de entrada utilizado.    

**`<Management mode>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u>: el control Campo de entrada cambia al modo de solo lectura y muestra el contenido en formato Markdown. El desplazamiento y la selección están habilitados (excepto si el valor de la propiedad [State](../Proprietes/2510039.md) es *Grayed*). El valor de la propiedad [State](../Proprietes/2510039.md) del control no cambia.  El control Campo de entrada cambia al modo "Markdown (solo lectura)". 

- <u><u><u><u>False</u></u></u></u>: el control Campo de entrada vuelve al modo de entrada. También es necesario utilizar la propiedad [State](../Proprietes/2510039.md) para activarlo. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La opción "Admitir Markdown" en la pestaña "UI" de la ventana de descripción de los controles Campo de entrada solo se aplica al título, tooltip y tokens asociados al control. Esta propiedad no se aplica al contenido del control Campo de entrada. 

- Para mostrar datos en formato Markdown, el control Campo de entrada debe cumplir una de las siguientes condiciones: 

	- ser de tipo "Markdown (solo lectura)". 

	- tener la propiedad **DisplayAsMarkdown** establecida en <u><u><u><u>True</u></u></u></u>.


## Torch (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Torch** permite determinar y cambiar la forma en que un control Cámara utiliza la linterna. 

Observaciones: 

- Esta propiedad solo puede utilizarse en controles Cámara que no están en "Modo compatible con la versión 26". 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CAM_MyCamera.Torch = camTorchOn
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar el modo de la linterna de un control Cámara

`<Result> = <Camera control>.Torch`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de linterna utilizado por la cámara: 


|   |   |
| --- | --- |
| *camTorchAuto* | La linterna se enciende automáticamente según las condiciones de luz.  |
| *camTorchOff* | La linterna siempre está apagada. |
| *camTorchOn* | La linterna siempre está encendida. |


 De forma predeterminada, la linterna está desactivada. 

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.


<a name="SYNTAX2"></a>

### Cambiar el modo de la linterna en un control Cámara

`<Camera control>.Torch = <New mode>`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<New mode>: (Constante de tipo Integer)`**

Modo de linterna utilizado por la cámara: 


|   |   |
| --- | --- |
| *camTorchAuto* | La linterna se enciende automáticamente según las condiciones de luz.  |
| *camTorchOff* | La linterna siempre está apagada. |
| *camTorchOn* | La linterna siempre está encendida. |


 De forma predeterminada, la linterna está desactivada.

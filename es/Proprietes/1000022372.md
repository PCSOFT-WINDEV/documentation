


## WinLineDetails (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **WinLineDetails** Property permite get y cambiar el nombre de la ventana interna utilizada para mostrar los detalles de un fila en un control Tabla. 

Observación: La fila se amplía en altura para mostrar la ventana interna.. La ventana interna se muestra debajo de los datos encontrados en el fila. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on [With Details]
TABLE_Order.WinLineDetails = "IW_OrderDetails" 
TableExpand(TABLE_Order, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis

### Descubriendo la ventana interna utilizada para mostrar los detalles de la fila

`<Internal window> = <Table control>.WinLineDetails`
---

**`<Internal window>: (Nombre de la ventana interna)`**

Nombre de la ventana interna utilizada para mostrar los detalles de un control fila.

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular. 



### Modificando la ventana interna utilizada para mostrar los detalles del fila

`<Table control>.WinLineDetails = <Internal window>`
---

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular. 

**`<Internal window>: (Nombre de la ventana interna)`**

Nombre de la ventana interna que se usará para mostrar los detalles de un control Tabla fila.
El prototipo de esta ventana interna es el siguiente:

```txt
PROCEDURE MyWindow(<Parent table> is control, <Row number> is int)
```

Estos parámetros se rellenan cuando se selecciona de fila a Process en el control. 

- &lt;Tabla padre&gt;: Nombre de la control Tabla utilizada. 

- &lt;Número de la fila&gt;: Entero correspondiente al número de la fila cuyos detalles se muestran.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


- La altura del fila que muestra la ventana interna corresponde a la altura inicial del fila más la altura de la ventana interna en modo de edición ([InitialHeight](../Proprietes/2510043.md) Property).

- La ventana interna se descarga cuando el usuario colapsa una fila.

- Las filas pueden ampliarse y/o colapsarse mediante la programación con [TableExpand](../WDLang1/1000010007.md) y [TableCollapse](../WDLang1/1000010008.md).

- También puede especificar una ventana interna específica para una fila utilizando la siguiente sintaxis:
	
	```txt
	<Table control>[Row number].WinLineDetails = NameOfIWForTheRow
	```

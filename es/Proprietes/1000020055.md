


## ElementOrientation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ElementOrientation** permite obtener y cambiar la orientación de un control Organigrama. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_VERTICAL = True THEN 
	ORG_MyOrganizationChart.ElementOrientation = LeftToRight
ELSE
	ORG_MyOrganizationChart.ElementOrientation = TopToBottom
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la orientación de un control Organigrama

`<Result> = <Organization Chart control>.ElementOrientation`
---

**`<Result>: (Constante de tipo Integer)`**

La orientación actual de la control: 


|   |   |
| --- | --- |
| *BottomToTop* | El elemento raíz del organigrama está en la parte inferior de la control, y sus hijos están en la parte superior. |
| *LeftToRight* | El elemento raíz del organigrama está a la izquierda de la control, y sus hijos a la derecha. |
| *RightToLeft* | El elemento raíz del organigrama está a la derecha de la control, y sus hijos a la izquierda. |
| *TopToBottom* | El elemento raíz del organigrama se encuentra en la parte superior de control, y sus hijos en la parte inferior. |



**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar. 


<a name="SYNTAX2"></a>

### Modificar la orientación de una control Organigrama

`<Organization Chart control>.ElementOrientation = <Orientation>`
---

**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar. 

**`<Orientation>: (Constante de tipo Integer)`**

La nueva orientación de la control: 


|   |   |
| --- | --- |
| *BottomToTop* | El elemento raíz del organigrama está en la parte inferior de la control, y sus hijos están en la parte superior. |
| *LeftToRight* | El elemento raíz del organigrama está a la izquierda de la control, y sus hijos a la derecha. |
| *RightToLeft* | El elemento raíz del organigrama está a la derecha de la control, y sus hijos a la izquierda. |
| *TopToBottom* | El elemento raíz del organigrama se encuentra en la parte superior de control, y sus hijos en la parte inferior. |

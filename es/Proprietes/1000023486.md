
## Parent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Parent** Property se utiliza para get el elemento padre de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [docElement](../WDLang1/1000022484.md).

- una variable de tipo [docParagraph](../WDLang1/1000022483.md).

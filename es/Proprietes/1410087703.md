
## Flash (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Flash** determina y cambia el modo de flash en un control Cámara. 

Observaciones: 

- Esta propiedad solo puede utilizarse en controles Cámara que no están en "Modo compatible con la versión 26". 

- Esta propiedad corresponde a la opción "Flash" de la pestaña "General" de la ventana de descripción del control Cámara. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CAM_MyCamera.Flash = camFlashOff THEN
	CAM_MyCamera.Flash = camFlashAuto
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar el modo de flash en un control Cámara

`<Result> = <Camera control>.Flash`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de flash de la cámara: 


|   |   |
| --- | --- |
| *camFlashAuto* | El flash se dispara automáticamente según las condiciones de luz.  |
| *camFlashOff* | El flash siempre está desactivado. |
| *camFlashOn* | El flash siempre está activado. |


 De forma predeterminada, el Flash está desactivado. 

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.


<a name="SYNTAX2"></a>

### Cambiar el modo de flash en un control Cámara

`<Camera control>.Flash = <New mode>`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<New mode>: (Constante de tipo Integer)`**

Modo de flash de la cámara: 


|   |   |
| --- | --- |
| *camFlashAuto* | El flash se dispara automáticamente según las condiciones de luz.  |
| *camFlashOff* | El flash siempre está desactivado. |
| *camFlashOn* | El flash siempre está activado. |


 De forma predeterminada, el Flash está desactivado.

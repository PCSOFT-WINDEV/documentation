
## Opacity (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Opacidad** Property se utiliza para get y cambiar la opacidad de un elemento. Esta propiedad se puede utilizar en:

- los mandos de una ventana o Page. Para obtener más información, consulte [Opacidad Property en los controles](../Proprietes/2514006.md).

- una variable de tipo [gglPath](../WDLang5/1000017433.md). 

- una variable de tipo [Color](../WDLang1/1000019661.md).

- una variable de tipo [Watermark](../WDLang5/1000019922.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picText](../WDLang1/1000025178.md).

- una variable de tipo [MapImage](../WDLang3/1000025523.md).

- una variable de tipo [Marker](../WDLang3/1000019940.md).


## Color (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Color** Property se utiliza para get y cambiar el color de un elemento. Esta propiedad se puede utilizar en:

- los mandos de una ventana o Page. Para obtener más información, consulte [Color en los mandos de una ventana o Page](../Proprietes/2510071.md).

- una variable de tipo [gglPath](../WDLang5/1000017433.md).

- una variable de tipo [gglMarker](../WDLang5/1000017442.md).

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md). 

- una variable de tipo [Franja de orilla](../Motscles/15140100.md).

- una variable de tipo [Color](../WDLang1/1000019661.md).

- una variable de tipo [Background](../WDLang1/1000022038.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Token](../WDLang1/1000023942.md).

- una variable de tipo [MarkerImage](../WDLang3/1000025796.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

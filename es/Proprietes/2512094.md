


## WDD55 (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**WDD55** se utiliza para:

- Averiguar el nombre y la ruta completa del archivo WDD en formato Hyper File 5.5. Este archivo se utiliza para acceder a los archivos en formato Hyper File 5.5 que se encuentran en un análisis en formato HFSQL Classic.

- Definir el nombre y la ruta completa del archivo WDD en formato Hyper File 5.5. Este archivo se utiliza para acceder a los archivos en formato Hyper File 5.5 que se encuentran en un análisis en formato HFSQL Classic. Este Property es useful cuando se describe programando un archivo de datos en formato Hyper File 5.5 en un análisis en formato HFSQL Classic.




**Atención**: Este Property sólo puede utilizarse para los archivos de datos en formato Hyper File 5.5 no migrados y utilizados en un análisis en formato HFSQL Classic.

El nombre del archivo WDD en formato 5.5 puede ser recuperado en el description del archivo de datos en el análisis:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDD55%20-%20HC%20N%B0001.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the path of the Orders data file
ResPath = Orders.WDD55
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la ruta del archivo WDD en formato Hyper File 5.5

`<Result> = <Data file>.WDD55`
---

**`<Result>: (Cadena de caracteres)`**

Ruta del DMA correspondiente al análisis inicial en formato Hyper File 5.5. Si el directorio de tiempo de ejecución del análisis corresponde al directorio actual, ruta es relativo al directorio actual.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos en el formato Hyper File 5.5 utilizado.  


<a name="SYNTAX2"></a>

### Definiendo la ruta del archivo WDD en formato Hyper File 5.5

`<Data file>.WDD55 = <Path>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Nombre definido mediante el tipo [Expediente description](../WDLang4/1514065.md).

**`<Path>: (Cadena de caracteres)`**

Ruta del DMA correspondiente al análisis inicial en formato Hyper File 5.5.


## PageLayout (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **PageLayout** obtiene y establece el diseño: 

- del diagrama actual en el control Editor de diagramas. El diseño se define en una variable de tipo [diagLayout](../WDLang1/1410088610.md).

- de la hoja actual en un control Hoja de cálculo. El diseño se define en una variable de tipo [xlsLayout](../WDLang5/1000023139.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Current sheet [NoMargins]
xLayout is xlsLayout = PSHEET_SpreadsheetControl.PageLayout
xLayout.LeftMargin = 0
xLayout.RightMargin = 0
PSHEET_SpreadsheetControl.PageLayout = xLayout
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el diseño de un diagrama

`<Result> = <Diagram Editor control>.PageLayout`
---

**`<Result>: (Variable de tipo diagLayout)`**

Variable de tipo [diagLayout](../WDLang1/1410088610.md) que corresponde al diseño actual del diagrama.

**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado.


<a name="SYNTAX2"></a>

### Cambiar el diseño de un diagrama

`<Diagram Editor control>.PageLayout = <New layout>`
---

**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado.

**`<New layout>: (Variable de tipo diagLayout)`**

Nombre de la variable de tipo [diagLayout](../WDLang1/1410088610.md) que corresponde al nuevo diseño del diagrama.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>
Este diseño se utiliza al imprimir el diagrama mediante el control Editor de diagramas o con la función [iPrintDiagram](../WDLang5/1410088189.md).

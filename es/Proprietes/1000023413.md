


## MarginInterCell (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MarginInterCell** Property se utiliza para: 

- averiguar el margen entre filas o columnas en un control Disposición. 

- modificar el margen entre filas o columnas en un control Disposición. 
	



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// 16 pixels between each row of LAYOUT_Row
LAYOUT_Row.MarginInterCell = 16
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el margen entre las filas o columnas en una control Disposición

`<Result> = <Layout control>.MarginInterCell`
---

**`<Result>: (Integro)`**

Margen (en píxeles) entre las filas o columnas de la control Disposición especificada. 

**`<Layout control>: (Nombre del control)`**

Nombre de la control Disposición que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificar el margen entre las filas o columnas en una control Disposición

`<Layout control>.MarginInterCell = <New margin>`
---

**`<Layout control>: (Nombre del control)`**

Nombre de la control Disposición que se va a utilizar. 

**`<New margin>: (Integro)`**

Nuevo margen (en píxeles) entre las filas o columnas de la control Disposición especificada. Este valor debe ser positivo o nulo.




## SecurityHtml (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SecurityHtml** Property se utiliza para get y cambiar el estado del mecanismo de seguridad de los siguientes controles: 

- Controles de visualización HTML. 

- Controles del editor HTML.


El mecanismo de seguridad de este tipo de controles bloquea todo el contenido potencialmente peligroso (plug-ins, ActiveX, código JavaScript, etc.). 

**Observaciones**: 

- También puede gestionar el mecanismo de seguridad de: 

	- el control de visualización HTML a través de "Prohíbe la ejecución de scripts JavaScript" (pestaña "Detalles" de la ventana del control description). Si esta opción está marcada, el mecanismo de seguridad está activado.

	- el control del Editor HTML a través de "Permitir ejecución del código JavaScript en el documento editado" (pestaña "General" en la ventana del control description). El mecanismo de seguridad está activado si esta opción no está seleccionada.




- "HTML Display control" es el nuevo nombre del HTML Edit control de la versión 25 y anteriores. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve a document on Internet
HTTPRequest(gsURL)

// Enable security mechanism of HTML Display control
HTM_EditDoc.SecurityHtml = secHtmlMaxi

// Initialize the control with the document
HTM_EditDoc = HTTPGetResult()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el mecanismo de seguridad de una control HTML está activado

`<Result> = <Control used>.SecurityHtml`
---

**`<Result>: (Constante booleana o booleana)`**

Modo de seguridad actual: 



|   |   |
| --- | --- |
| *secHtmlMaxi* (o <u><u><u><u>True</u></u></u></u>)<br>(Valor predeterminado) | No se ejecuta ningún script (javascript, vbscript, etc.). No se muestra ni se ejecuta ningún ActiveX o plugin. |
| *secHtmlNone* (o <u><u><u><u>False</u></u></u></u>) | Los scripts se ejecutan. El ActiveX o los plugins se muestran y se ejecutan. <br><br>**Observación**: Los scripts no se ejecutarán en el control si el [Estado](../Proprietes/2510039.md) Property del control es set a *Inactive* o *Grayed*. |



**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML. 

- Control Editor HTML. 





<a name="SYNTAX2"></a>

### Activar/desactivar el mecanismo de seguridad de los controles de edición de HTML

`<Control used>.SecurityHtml = <Value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML. 

- Control Editor HTML. 




**`<Value>: (Constante booleana o booleana)`**

Modo de seguridad a aplicar: 



|   |   |
| --- | --- |
| *secHtmlMaxi* (o <u><u><u><u>True</u></u></u></u>)<br>(Valor predeterminado) | No se ejecuta ningún script (javascript, vbscript, etc.). No se muestra ni se ejecuta ningún ActiveX o plugin. |
| *secHtmlNone* (o <u><u><u><u>False</u></u></u></u>) | Los scripts se ejecutan. El ActiveX o los plugins se muestran y se ejecutan. <br><br>**Observación**: Los scripts no se ejecutarán en el control si el [Estado](../Proprietes/2510039.md) Property del control es set a *Inactive* o *Grayed*. |


## CaptionHTML (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CaptionHTML** Property se utiliza para manipular texto como código HTML. Esta propiedad se puede utilizar en:

- los mandos en una Page. Para obtener más información, consulte la ayuda sobre la propiedad [CaptionHTML](../Proprietes/1000020922.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

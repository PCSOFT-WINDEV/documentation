
## Extra (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Extra** permite:

- Obtener información adicional sobre un control, ventana o página.

- Editar información adicional de un control, ventana o página.






La información adicional está destinada a los desarrolladores y solo puede manipularse mediante programación. 

Esta propiedad permite almacenar información avanzada sin afectar a la ejecución de la aplicación. Por ejemplo, es posible almacenar un valor de cualquier tipo, un array, etc. También es posible agregar miembros a la propiedad **Extra**. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_Text.Extra.PreviousValue = EDT_City
EDT_Text.Extra.DateModified = DateSys()
```

```wl
MyControl.Extra = Value
MyControl.Extra.Member = Value
MyControl.Extra["Member"] = Value
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener información adicional sobre un elemento (control, ventana, página, etc.)

`<Information> = <Element used>.Extra`
---

**`<Information>: (Tipo de información)`**

Información asociada al elemento especificado.

**`<Element used>: (Nombre de control, ventana o página)`**

Nombre del elemento (control, ventana o página) que se va a utilizar.   


<a name="SYNTAX2"></a>

### Editar información adicional de un elemento (control, ventana, página, etc.)

`<Element used>.Extra = <New information>`
---

**`<Element used>: (Nombre de control, ventana o página)`**

Nombre del elemento (control, ventana o página) que se va a utilizar.  

**`<New information>: (Tipo de información)`**

Información asociada al elemento especificado.




## FontItalic (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**FontItalic** se utiliza para:

- Determine si el texto está en cursiva.

- Aplicar o eliminar el formato de cursiva del texto.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Write the text displayed in "STC_StaticControl" in italic
STC_StaticControl.FontItalic = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el texto mostrado en una control está en cursiva

`<Result> = <Control used>.FontItalic`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto está en cursiva,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Aplicar o eliminar el formato de cursiva del texto mostrado en una control

`<Control used>.FontItalic = <Italic/Not italic>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Italic/Not italic>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto debe estar en cursiva,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000097"></a>

En un reporte, el **FontItalic** Property sólo puede utilizarse en:

- un control Estático.

- to URL control Enlace.

- un control Calculado.

- un control Predefinido.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **FontItalic** Property no se puede utilizar en:

- a Control de número,

- una Web control Cámara,

- a control Conferencia.


Observación: Esta Property se aplica únicamente a los controles "ListView" en el modo de visualización "List Box".![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **FontItalic** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Combo Box.

- Control TreeView.

- Columna de un control Tabla.

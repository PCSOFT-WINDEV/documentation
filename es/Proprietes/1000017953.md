
## Size (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Tamaño** palabra clave puede corresponder a: 

- [envergadura](../WDLang1/3024031.md) se utiliza para averiguar el tamaño de una cadena de caracteres o de un buffer.

- El **Size** Property se utiliza para determinar y modificar el tamaño de un elemento. Esta propiedad se puede utilizar en:

	- los controles de una ventana o reporte. Para obtener más información, consulte la ayuda sobre la propiedad [envergadura](../Proprietes/2510125.md).

	- los elementos de texto y tiempo de un archivo de datos. Para obtener más información, consulte la ayuda sobre la propiedad [envergadura](../WDLang1/3024031.md).

	- una variable de tipo [tipo de letra](../Motscles/1514045.md). Para obtener más información, consulte la ayuda sobre la propiedad [envergadura](../Proprietes/2510125.md).

	- una variable de tipo [gglMarker](../WDLang5/1000017442.md). 

	- una variable de tipo [rssEnclosure](../WDLang5/1000017789.md).

	- una variable de tipo [sfExecuteQueryResult](../WDLang5/1000018558.md).

	- una variable de tipo [InfoVersion](../WDLang1/1000018905.md).

	- una variable de tipo [zipArchive](../WDLang3/1000018679.md).

	- una variable de tipo [zipArchivedFile](../WDLang3/1000019044.md).

	- una variable de tipo [Watermark](../WDLang5/1000019922.md).

	- una variable de tipo [hdfsStatus](../WDLang4/1000021973.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoGridFSFileInfo](../WDLang4/1000022417.md).

	- una variable de tipo [MarkerImage](../WDLang3/1000025796.md).




- una variable de tipo [SecurePassword](../Motscles/1000021064.md).

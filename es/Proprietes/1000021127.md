


## Synonym (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Synonym** define los sinónimos en un índice de texto completo definido mediante programación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyFullTextIndex is FullTextIndex Description
...
MyFullTextIndex.Synonym = "Car Vehicle Automobile" + CR + "House Villa Condominium"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Name of full-text index>.Synonym = <Synonyms>`
---

**`<Name of full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido por el tipo [FullTextIndex Description](../WDLang4/1000017461.md).

**`<Synonyms>: (Cadena de caracteres)`**

Lista de nombres considerados sinónimos. Los nombres están separados por un espacio, los sinónimos están separados por un Carriage Return (CR). 


```txt
"WordA SynonymA1 SynonymA2" + CR + "WordN SynonymN1 SynonymN2"
```

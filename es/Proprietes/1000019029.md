
## NumberExecution (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NúmeroEjecución** Property se utiliza para get y cambiar el número de ejecuciones de una tarea. Esta propiedad se puede utilizar en: 

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). 

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).


## WorkingDirectory (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WorkingDirectory** permite: 

- obtener o cambiar el directorio utilizado por el control Editor HTML para almacenar las imágenes y hojas CSS de una página HTML.

- obtener el directorio utilizado por el control de visualización HTML para almacenar las imágenes y hojas CSS de una página HTML.






**Observaciones**: 

- Control Editor HTML: cuando se utiliza el control, se crea automáticamente un directorio de trabajo. Este directorio se elimina cuando se cierra la ventana o la aplicación.

- Control de visualización HTML: cuando se utiliza la función [HTMLFromEmail](../WDLang1/1410087263.md), se crea automáticamente un directorio de trabajo para guardar las imágenes asociadas al correo electrónico. Este directorio se elimina cuando se cierra la ventana o la aplicación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copy an image to the directory and insert it into the document
fCopyFile("logo.png", CompleteDir(HTMEDT_NoName1.WorkingDirectory) + "logo.png")
HTMEDT_NoName1.Insert("<img src='logo.png'>")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el directorio utilizado por el control para almacenar archivos

`<Result> = <Control used>.WorkingDirectory`
---

**`<Result>: (Cadena de caracteres)`**



- Control Editor HTML: Directorio utilizado por el control Editor HTML para almacenar archivos (imágenes, hojas CSS, etc.). 

- Control de visualización HTML: Directorio utilizado por el control de visualización HTML. Si la URL asociada al control es un archivo, la propiedad **WorkingDirectory** devuelve el directorio correspondiente, de lo contrario devuelve una cadena vacía (""). 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Editor HTML, 

- Control de visualización HTML. 





<a name="SYNTAX2"></a>

### Cambiar el directorio utilizado por el control Editor HTML para almacenar archivos (imágenes, hojas CSS, etc.)

`<HTML Editor control>.WorkingDirectory = <New directory>`
---

**`<HTML Editor control>: (Nombre del control)`**



- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Nombre del control Editor HTML utilizado. 




**`<New directory>: (Cadena de caracteres)`**

Directorio utilizado para almacenar: 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) archivos del control Editor HTML (imágenes, hojas CSS, etc.).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las imágenes o las hojas de estilo CSS con una URL de archivo (es decir, que empieza por "file://") no pueden mostrarse de forma predeterminada al editar texto en el control Editor HTML. 

- Para mostrar imágenes u hojas de estilo CSS con una URL de archivo (es decir, que empiece por "file://"), estos archivos deben: 

	- estar ubicados en el directorio especificado con la propiedad **WorkingDirectory**.

	- ser relativos. Por ejemplo: &lt;img src='miLogotipo.png' ...




- Este directorio se inicializa con un directorio temporal. Este directorio se elimina cuando se cierra la ventana o la aplicación. Para los controles Editor HTML, puede cambiarse con la propiedad **WorkingDirectory**. En este caso, el directorio especificado no se elimina.

- La opción de la cinta del control Editor HTML que permite insertar imágenes en la página copia automáticamente las imágenes en este directorio.




## FontAppointmentTitle (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontAppointmentTitle** Property se utiliza para:

- Descubra las características de la fuente utilizada para el título de las citas en los informes Programador y Organizador. 

- Modifique las características de la fuente utilizada para el título de las citas en los informes Programador y Organizador.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Characteristics of font used for the SCH_Scheduler control in the RPT_Scheduler report
MyFont is Font

MyFont = SCH_SCHEDULER.FontAppointmentTitle
Trace(MyFont.Name)
Trace(MyFont.Bold)
Trace(MyFont.Underline)
Trace(MyFont.Size)
Trace(MyFont.Color)
```
<a name="Example2"></a>



```wl
// Define the fonts 
MyFont is Font
MyFont.Name = "Comic Sans MS"
MyFont.Size = 18
MyFont.Color = PastelBlue

MyFont2 is Font
MyFont2.Name = "Courier new"
MyFont2.Size = 14
MyFont2.Color = PastelGreen

// Modify the fonts used in the report
SCH_SCHEDULER.FontAppointmentTitle = MyFont
SCH_SCHEDULER.FontAppointmentContent = MyFont2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la fuente utilizada para el título de las citas

`<Result> = <Control used>.FontAppointmentTitle`
---

**`<Result>: (tipo de letra)`**

[tipo de letra](../Motscles/1514045.md) variable que contiene las características del tipo de letra utilizado para el título de los nombramientos. 

**`<Control used>: (Nombre del control)`**

Nombre del programador o control Agenda en un programador u organizador reporte.  


<a name="SYNTAX2"></a>

### Modificar la fuente utilizada para el título de los nombramientos

`<Control used>.FontAppointmentTitle = <New font>`
---

**`<Control used>: (Nombre del control)`**

Nombre del Programador o control Agenda que se encuentra en un Programador u Organizador reporte.

**`<New font>: (tipo de letra)`**

[tipo de letra](../Motscles/1514045.md) variable que contiene las características del tipo de letra que debe utilizarse para el título de los nombramientos.   



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **FontAppointmentTitle** Property se aplica sólo a los controles del Organizador y del Programador en los informes.

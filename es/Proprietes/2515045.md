


## Propiedades de gestión de fuentes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Estas propiedades pueden utilizarse en una Variable de tipo [tipo de letra](../Motscles/1514045.md). 



|   |   |
| --- | --- |
| [Bold](../Proprietes/2513001.md) | El **Bold** Property se utiliza para:<br><br>- Averigüe si una fuente Variable está en negrita.<br><br>- Modificar el atributo de negrita de una fuente Variable.<br><br><br> |
| [CharacterSpacing](../Proprietes/1410088367.md) | La propiedad **CharacterSpacing** obtiene y establece el espaciado entre caracteres. |
| [Charset](../Proprietes/2512008.md) | La propiedad **Charset** permite: <br><br>- Obtener el conjunto de caracteres utilizado por una variable de tipo Font.<br><br>- Cambiar el conjunto de caracteres utilizado por una variable de tipo Font.<br><br>- Obtener el conjunto de caracteres utilizado por un archivo de datos HFSQL. <br><br><br> |
| [Color](../Proprietes/2510071.md) | El **Color** Property se utiliza para get o cambiar la fuente Color para:<br><br>- una fuente Variable.<br><br>- un texto visualizado en un control Estático o en un control calculado de un reporte.<br><br>- un texto mostrado en una ventana control.<br><br>- un texto mostrado en una página control.<br><br><br> |
| [Condensed](../Proprietes/2513006.md) | El **Condensado** Property se utiliza para:<br><br>- Determinar si una fuente Variable está condensada.<br><br>- Condensar la fuente Variable.<br><br><br>**Este Property se mantiene por compatibilidad con versiones anteriores.** |
| [Extended](../Proprietes/2513002.md) | **Extended** se utiliza para:<br><br>- Averiguar si una fuente Variable se extiende.<br><br>- Extender (o no) una fuente Variable.<br><br><br>**Este Property se mantiene por compatibilidad con versiones anteriores.** |
| [Italic](../Proprietes/2513007.md) | La **Italic** Property se utiliza para:<br><br>- Averigüe si una fuente Variable está en cursiva.<br><br>- Modificar el atributo en cursiva de una fuente Variable.<br><br><br> |
| [Large](../Proprietes/2513012.md) | **Large** se utiliza para:<br><br>- Averiguar si una fuente Variable está agrandada.<br><br>- Ampliar (o no) una fuente Variable.<br><br><br>**Este Property se mantiene por compatibilidad con versiones anteriores.** |
| [LineSpacing](../Proprietes/1410088366.md) | La propiedad **LineSpacing** obtiene y establece el interlineado de las variables de tipo [Font](../Motscles/1514045.md). |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [Orientation](../Proprietes/1410086632.md) | La propiedad **Orientation** permite:<br><br>- Obtener el ángulo (en grados) de una variable de tipo Font.<br><br>- Modificar el ángulo (en grados) de una variable de tipo Font.<br><br><br> |
| [Size](../Proprietes/2510125.md) | El **Size** Property se utiliza para determinar y modificar el tamaño de un elemento. |
| [SizeUnit](../Proprietes/1410088365.md) | El **SizeUnit** Property obtiene y establece la unidad de tamaño de la fuente. |
| [StrikeOut](../Proprietes/2513005.md) | La propiedad **StrikeOut** permite:<br><br>- Determinar si el formato de tachado se aplica a una variable de tipo Font.<br><br>- Aplicar formato de tachado a una variable de tipo Font.<br><br><br> |
| [Underline](../Proprietes/2513013.md) | La propiedad **Underline** permite:<br><br>- Averigua si una fuente Variable está subrayada.<br><br>- Subraye (o no) una fuente Variable.<br><br><br> |

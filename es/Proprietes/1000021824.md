


## WinEditMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WinEditMode** permite conocer y configurar las acciones permitidas en una ventana y sus controles cuando la ventana pasa al modo "Permitir que el usuario final modifique la UI".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allows the size and position to be modified in the window 
// except for the OK button 
WIN_MyWindow.WinEditMode = weSelection + wePosition + weSize
BTN_OK.WinEditAllowed = weNone
WInEdActive(WIN_MyWindow, True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar las operaciones permitidas en una ventana y sus controles

`<Result> = <Window used>.WinEditMode`
---

**`<Result>: (Constante o combinación de constantes de tipo Integer)`**

Combinación de las siguientes constantes: 


|   |   |
| --- | --- |
| *weAddRepositionableNote* | El usuario final puede añadir una note reposicionable en la ventana.  |
| *weAddWire* | El usuario final puede añadir un cable entre varios controles. |
| *weChangeTooltip* | El usuario final puede modificar los consejos de herramientas de los controles. |
| *weChangeWire* | El usuario final puede modificar los cables entre los controles. |
| *weMoveControl* | El usuario final puede mover los controles. |
| *weMultiSelection* | El usuario final puede seleccionar varios controles al mismo tiempo. <br><br>Atención: Esta constante requiere el uso de la constante *weSelection*. |
| *wePosition* | El usuario final puede mover los controles. |
| *weResizeControl* | El usuario final puede cambiar el tamaño de los controles. |
| *weSelection* | El usuario final puede seleccionar los controles. |
| *weSize* | El usuario final puede modificar el tamaño de los controles.  |



**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana a utilizar. 


<a name="SYNTAX2"></a>

### Modificar las operaciones permitidas en una ventana y sus controles

`<Window used>.WinEditMode = <Authorized operations>`
---

**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana a utilizar.

**`<Authorized operations>: (Constante o combinación de constantes de tipo Integer)`**

Operaciones permitidas en la ventana y sus controles: 


|   |   |
| --- | --- |
| *weAddRepositionableNote* | El usuario final puede añadir una note reposicionable en la ventana.  |
| *weAddWire* | El usuario final puede añadir un cable entre varios controles. |
| *weAll* | El usuario final puede modificar todo en la ventana. <br><br>Esta opción no puede combinarse con ninguna de las otras opciones.. |
| *weChangeTooltip* | El usuario final puede modificar los consejos de herramientas de los controles. |
| *weChangeWire* | El usuario final puede modificar los cables entre los controles. |
| *weMoveControl* | El usuario final puede mover los controles. |
| *weMultiSelection* | El usuario final puede seleccionar varios controles al mismo tiempo. <br><br>Atención: Esta constante requiere el uso de la constante *weSelection*. |
| *weNone* | El usuario final no puede modificar o seleccionar nada en la ventana. Esta opción es la opción Default. <br><br>Esta opción no puede combinarse con ninguna de las otras opciones.. |
| *wePosition* | El usuario final puede mover los controles. |
| *weResizeControl* | El usuario final puede cambiar el tamaño de los controles. |
| *weSelection* | El usuario final puede seleccionar los controles. |
| *weSize* | El usuario final puede modificar el tamaño de los controles. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **WinEditMode** Property se utiliza para set permisos de modificación en una ventana y sus controles. Los permisos pueden configurarse en cada mando con el programa [WinEditPermitido](../Proprietes/1000021850.md) Property.

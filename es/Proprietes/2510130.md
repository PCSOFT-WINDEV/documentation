


## Value (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Value** permite:

- Averigua y modifica el valor de un reporte control.

- Averiguar y modificar el valor de una control o el título de una ventana.

- Averiguar el identificador de una secuencia. 

- Averiguar y modificar el valor de una control o el título de una Page.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Centrar el mapa que se encuentra en una control Mapa en una posición específica.

- Obtener el valor asociado a un valor de un Variable de tipo [Enumeración](../Motscles/1514067.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Averiguar el identificador o el manipular asociado a un control Contenedor nativo.




**Observación**: El **Value** Property se utiliza raramente y puede ser sustituido por una sintaxis equivalente (ver Observaciones).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the image file associated with the "IMG_Image" control
IMG_Image.Value = "C:\MyImages\Image.BMP"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el valor de un elemento

`<Result> = <Element used>.Value`
---

**`<Result>: (Cualquier tipo)`**

Este resultado depende del tipo de elemento.

**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el valor de un elemento

`<Element used>.Value = <New value>`
---

**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar.

**`<New value>: (Cualquier tipo)`**

Nuevo valor para el elemento especificado. Este parámetro depende del tipo de elemento especificado.

### Valor de control Casilla de verificación o valor de la opción en una control Casilla de verificación
<a name="valor_control_casilla_verificacion_valor_opcion_una_control_casilla_verificacion_ELTPARAGRAPHE000309"></a>

- Marcar o desmarcar una opción en un control Casilla de verificación.

```txt

ResChecked = <Check Box control>[<Option number>]

ResChecked = <Option name>.Value

ResChecked = <Option name>
```
Observación: Si no se especifica &lt;Número de opción&gt;, la primera opción es seleccionada por Default.

### Valor de List Box, ListView o control Tabla
<a name="valor_list_box_listview_control_tabla_ELTPARAGRAPHE000362"></a>

Se pueden presentar dos casos distintos.

- El cuadro de lista, ListView o control Tabla se rellena mediante programación O el [StoredItem](../Proprietes/2510110.md) Property corresponde a una cadena vacía ("").
	El **Value** Property se utiliza para:

	- Obtener la fila seleccionada. Si no se selecciona fila, el **Value** Property devuelve -1. Si el control permite la selección múltiple, se devuelve el número del último fila seleccionado.
			Para averiguar la fila seleccionada, utilice una de las siguientes sintaxis:
			
		```txt
		ResRow = <Control name>.Value
		
		ResRow = <Control name>
		```


	- Seleccione otra fila especificando el número de la nueva fila a seleccionar. Las filas seleccionadas previamente se deseleccionan.   




- El List Box, ListView, o controles Tabla se basan en un archivo de datos Y el [StoredItem](../Proprietes/2510110.md) Property no está vacío.
	La propiedad **Value** se utiliza para identificar el elemento almacenado para el Record asociado al fila seleccionado. Si no se selecciona fila, el **Value** Property devuelve una cadena vacía ("").
	Para averiguar el elemento almacenado associated con el fila seleccionado, utilice una de las siguientes sintaxis:
	
	```txt
	ResStoredItem = <Control name>.Value
	
	ResStoredItem = <Control name>.StoredItem
	```


**Caso especial: Valor de la celda en el fila** actual

- Averiguar el valor de la celda identificada por la columna especificada y por el fila actual en el control Tabla. Si no hay ninguna fila actual, se devuelve una cadena vacía ("").

- Modificar el valor de la celda identificada por la columna especificada y por el fila actual en el control Tabla.




&lt;Resultado&gt; es una cadena de caracteres.

Para averiguar el valor de la celda identificada por la columna especificada y por la fila actual en el control Tabla, utilice una de las siguientes sintaxis:


```txt
ResColumn = <Column name>.Value

ResColumn = <Column name>
```

El **Value** Property se utiliza para get el Address associated con la página HTML que se muestra en el control. Si el Page:

- es una Page construida en HTML directamente, el valor corresponde a "about:blank".

- es un Page de un sitio, el valor corresponde al Address del Page (www.windev.com por ejemplo).

- es un archivo HTML, el valor corresponde a la Address del archivo.


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **Value** Property devuelve la URL del sitio pero no puede devolver el código HTML del control. Este Property no puede ser usado para asociar un archivo htm local con el HTML control.

### Valor de control Mapa
<a name="valor_control_mapa_ELTPARAGRAPHE000691"></a>

El **Value** Property se utiliza para centrar el mapa (en un control Mapa) en la posición que corresponde a un [geoPosition](../WDLang3/1000019191.md) Variable. 

Ejemplo: 


```wl
// Centers the map displayed by the "MAP_Position" control on a geographical position
MyPosition is geoPosition

MAP_Position.Value = MyPosition
```
**Observaciones**: 

- La propiedad **Value** es equivalente a la función [MapDisplayPosition](../WDLang3/1000019787.md). La única diferencia es que: cuando se utiliza [MapDisplayPosition](../WDLang3/1000019787.md), se muestra una animación al fijar la posición en el lugar deseado.

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000851"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **Value** Property no se puede utilizar en:

- un reporte.

- un bloque de reporte.

- un control Casilla de verificación.

- a control Forma.

- un control Agenda.

- un control Planificador. 

- a control Diagrama de Gantt. 

- a control Firma.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **Value** Property no se puede utilizar en:

- una opción en un botón de radio.

- un panel de pestañas.

- un grupo de controles.


 
 
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Value** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- Control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de tabla.

- Supercontrol.

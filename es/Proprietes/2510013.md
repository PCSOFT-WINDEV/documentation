


## MemoryCurrency (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>


<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MemoriaMoneda** Property se utiliza para get y cambiar la moneda utilizada cuando se manipula un control "Moneda + Euro" de forma programada.

Observaciones: 

- El valor de la moneda introducido en un control de edición "Moneda + Euro" (o columna tabla) se introduce en la moneda definida por el [DisplayCurrency](../Proprietes/2510030.md) Property. 

- Si la moneda de visualización y la moneda de la memoria no son idénticas, la conversión se realiza automáticamente al guardar el Record asociado. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Changes the memory currency used by EDT_Edit1
EDT_Edit1.MemoryCurrency = eEUR   // Stored in Euros
EDT_Edit1 = 10   // Euro =>  Displays "65.60F"
EDT_Edit1.MemoryCurrency = eFRF    // Stored in Francs
EDT_Edit1 = 10   // Francs => displays "10.00F"
```

## Observaciones


### Moneda adicional
<a name="moneda_adicional_ELTPARAGRAPHE000319"></a>

También tiene la posibilidad de definir monedas personalizadas: permiten elegir una moneda adicional, creada mediante programación con [eInitCurrency](../WDLang1/3033003.md). Para elegir una de estas monedas, utilice los valores de 16 a 19:

- 16: Moneda personalizada #1

- 17: Moneda personalizada #2

- 18: Moneda personalizada #3

- 19: Moneda personalizada #4



<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000334"></a>

El **MemoryCurrency** Property se aplica cuando se manipulan los siguientes controles "Moneda + Euro" de forma programada:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) controles calculados y controles de artículos que se encuentran en los informes.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Controles de edición"Moneda + Euro".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) que se encuentran en los controles de la tabla "Moneda + Euro".


## SeriesColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SeriesColor** permite:

- Averigua la Color de una serie en un gráfico.

- Modificar la Color de una serie en un gráfico. 




**Observación**: Esta Property es equivalente a la [grSeriesColor](../WDLang3/3042013.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CHART_MyChart[2].SeriesColor = grColorAutomatic THEN
	CHART_MyChart[2].SeriesColor = LightRed
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la Color de una serie

`<Result> = <Series>.SeriesColor`
---

**`<Result>: (Entero o constante de tipo Integer)`**

Color de la serie especificada. Este color puede corresponder a:

- la constante ***grColorAutomatic*** (si el color se define automáticamente).

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).




**`<Series>: (Cadena de caracteres)`**

Serie cuyo Color se solicita. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.  


<a name="SYNTAX2"></a>

### Modificando la Color de una serie

`<Series>.SeriesColor = <New color>`
---

**`<Series>: (Cadena de caracteres)`**

Serie cuya Color debe ser modificada. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.

**`<New color>: (Entero o constante de tipo Integer)`**

Color de la serie especificada. Este color puede corresponder a:

- la constante ***grColorAutomatic*** (si el color debe definirse automáticamente).

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).

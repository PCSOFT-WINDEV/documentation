
## ReturnedValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ReturnedValue** Property se utiliza para manipular el valor de retorno. Esta propiedad se puede utilizar en:

- ventanas, páginas y controles. Para obtener más información, consulte [ReturnedValue](../Proprietes/2510134.md).

- una variable de tipo [ParallelTask](../WDLang1/1000021220.md).

- una variable de tipo [Thread](../WDLang1/1000024879.md).

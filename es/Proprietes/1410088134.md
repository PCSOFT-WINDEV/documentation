
## ColorFormat (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ColorFormat** Property se utiliza para gestionar el formato Color (Color o escala de grises). Esta propiedad está disponible en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Scanner](../WDLang6/1410087808.md), 

- una variable de tipo [pdfParameter](../WDLang6/1410087857.md).

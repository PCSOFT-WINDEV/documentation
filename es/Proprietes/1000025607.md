
## Compass (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Compass** permite: 

- mostrar u ocultar la brújula en una control Mapa,

- determinar si la brújula se muestra en una control Mapa. 




Atención: La brújula sólo se muestra cuando se gira el mapa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display compass
MAP_MyMap.Compass = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si la brújula se muestra en una control Mapa

`<Result> = <Map control>.Compass`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la brújula se muestra en la control Mapa, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Map control>: (Nombre del control)`**

Nombre de control Mapa utilizado. 


<a name="SYNTAX2"></a>

### Mostrando u ocultando la brújula en una control Mapa

`<Map control>.Compass = <Authorization>`
---

**`<Map control>: (Nombre del control)`**

Nombre de control Mapa utilizado. 

**`<Authorization>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la brújula debe ser mostrada en la control Mapa, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.

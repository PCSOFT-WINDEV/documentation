


## InitialState (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InitialState** Property se utiliza para get el estado inicial de un control. El estado inicial corresponde al estado definido durante el control description (pestaña "UI").


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtatInitial_Champ.gif)<br>Estado inicial de un control Campo de entrada (pestaña "UI") | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_init_champ_btn.gif)<br>I Estado inicial de control (botón por ejemplo) |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Table_initial.gif)<br>Estado inicial de un control Tabla (pestaña "UI") | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Etat_init_col.gif)<br>Estado inicial de la columna tabla (pestaña "UI")  |

Para saber si un control es visible, utilice la función [Visible](../Proprietes/2510138.md) Property.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF BTN_Close.InitialState = Grayed THEN
	BTN_Close.State = Active
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Initial status> = <Control>.InitialState`
---

**`<Initial status>: (Constante de tipo Integer)`**

Estado inicial de la control, definido en el editor.


|   |   |
| --- | --- |
| *Active* | El control está activo, Visible y en edición |
| *DisplayOnly o Inactive* | El control es de sólo lectura, no se permite ninguna entrada o acción<br>Para los controles de la Tabla, el usuario podrá seleccionar un tabla fila. También se puede seleccionar mediante programación con [TableSelectPlus](../WDLang1/3074031.md). |
| *Grayed* | El control es Grayed |
| *ReadOnlyNoSelection* | Sólo para los controles de tabla: El control es de sólo lectura, no se permite ninguna entrada.<br>En este caso, el usuario no podrá seleccionar un tabla fila con el ratón o con las flechas. Una fila sólo puede seleccionarse mediante programación con [TableSelectPlus](../WDLang1/3074031.md). Esta fila se seleccionará a través del banda de selección.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |



**`<Control>: (Nombre de control o grupo de control s)`**

Nombre de control o grupo de control s.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estado de un grupo de controles
<a name="estado_grupo_controles_ELTPARAGRAPHE000136"></a>

Si el **InitialState** Property se utiliza en un grupo de control s, el estado devuelto es el estado inicial del primer control del grupo.
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000145"></a>

- El **InitialState** Property no se puede utilizar en un menú o en una opción de menú.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **InitialState** Property no tiene ningún efecto sobre los controles HTML.

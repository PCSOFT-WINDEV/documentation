


## ParallaxRateY (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ParallaxRateY** Property se usa para determinar y especificar qué tan rápido se mueve un control cuando se mueve la barra de desplazamiento en la ventana, ventana interna o Page. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IMG_Product.ParallaxRateY = 500
IMG_Product.ParallaxRateHeight = 500
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la velocidad a la que se mueve la control al mover la barra de desplazamiento

`<Result> = <Control used>.ParallaxRateY`
---

**`<Result>: (Integro)`**

Velocidad a la que se mueve la control (incluida entre 0 y 1000). 

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. 


<a name="SYNTAX2"></a>

### Modificando la velocidad a la que se mueve la control al mover la barra de desplazamiento

`<Control used>.ParallaxRateY = <New speed>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. 

**`<New speed>: (Integro)`**

Velocidad a la que se mueve la control (incluida entre 0 y 1000). El valor predeterminado de este parámetro es 0. 

- Si este parámetro es set a 0, el control se desplaza como de costumbre al mover la barra de desplazamiento que se encuentra en la ventana o en el Page. 

- Si este parámetro es set a 500, el control se mueve la mitad de rápido que los otros control s, lo que se utiliza para realizar un "parallax scroll" (efecto de profundidad). 

- Si este parámetro se set a 1000, el control no se desplaza al mover la barra de desplazamiento que se encuentra en la ventana o en el Page. Esto le permite crear botones fijos, por ejemplo.

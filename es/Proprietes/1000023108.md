
## Minor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Menor** Property se utiliza para get y cambiar el número menor de una baliza. Esta propiedad se puede utilizar en: 

- una variable de tipo [beaconDetectionInfo](../WDLang3/1000023109.md). 

- una variable de tipo [beaconGroup](../WDLang3/1000023105.md).

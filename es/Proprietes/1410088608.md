
## ImageInitialWidth (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ImageInitialWidth** permite obtener o establecer el ancho predeterminado de una imagen en un control Editor de imágenes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IE_MyImageEditor.ImageInitialWidth = 800
IE_MyImageEditor.ImageInitialHeight = 600
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el ancho predeterminado de una imagen en el control Editor de imágenes

`<Result> = <Image Editor control>.ImageInitialWidth`
---

**`<Result>: (Entero)`**

Ancho predeterminado (en pixeles) de una imagen en el control Editor de imágenes. 

**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.


<a name="SYNTAX2"></a>

### Establecer el ancho predeterminado de una imagen en el control Editor de imágenes

`<Image Editor control>.ImageInitialWidth = <New width>`
---

**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.

**`<New width>: (Entero)`**

Nuevo ancho predeterminado (en pixeles) de la imagen en el control Editor de imágenes. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para cambiar las características predeterminadas del control Editor de imágenes, también es posible utilizar: 

- la propiedad [ImageInitialHeight](../Proprietes/1410088607.md) para establecer la altura predeterminada de la imagen creada en el control Editor de imágenes.

- la propiedad [InitialTool](../Proprietes/1410088609.md) para establecer la herramienta predeterminada del control Editor de imágenes.

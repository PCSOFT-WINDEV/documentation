


## Filter (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Filter** permite identificar y cambiar el filtro de los registros en: 

- controles Tabla basada en un archivo de datos,

- navegando por TreeView controles Tabla,

- Controles del cuadro de lista basados en un archivo de datos,

- navegando por los controles de Combo Box,

- Navegación Controles de looper. 




Este filtro (si existe) puede ser definido durante el control description (pestaña "Contenido").<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Filtre.gif)
 Este filtro (**Filtro** Property o filtro definido en el editor) tiene las siguientes características:

- Filtro de tipo "Comienza por".

- se aplica sólo a la posición de búsqueda.

- afecta al objeto utilizado (Tabla, Combo Box, List Box o control ListView)

- no afecta a el archivo de datos en el que se basa el objeto (excepto para el controles Tabla archivo de datos, véase más abajo).




Observaciones:

- Para utilizar un filtro en una clave compuesta, utilice [HFilter](../WDLang4/3044100.md).

- El **Filter** Property no puede utilizarse para identificar filtros definidos por el usuario (con la función [TableEnableFilter](../WDLang1/1000017149.md), por ejemplo).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// - Code whenever a Combo Box control is modified
// Update elements of the Combo Box control according to
// the letters entered in the edit control of the Combo Box
COMBO_Element.Filter = COMBO_Element
```
<a name="Example2"></a>

```wl
// Identifies the filter condition on the City search key
IF TABLE_Customer.Filter = "" THEN
	TABLE_Customer.Filter = "PARIS"
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar el filtro definido en el elemento de búsqueda

`<Filter value> = <Control used>.Filter`
---

**`<Filter value>: (Tipo de filtro)`**

Valor actual del filtro definido en el elemento de búsqueda. Este filtro es un filtro "Start with".

**`<Control used>: (Nombre del control)`**

Nombre de la control cuyo filtro actual se solicita: este filtro se define mediante la propiedad **Filter**, o en el editor de ventanas o páginas (en el control description). Esta control debe ser una: 

- navegar por la Caja de Listas o ListView,

- navegando por Combo Box,

- tabla de navegación o tabla TreeView,

- Looper basado en un archivo de datos.  





<a name="SYNTAX2"></a>

### Modificar el filtro definido en el elemento de búsqueda

`<Control used>.Filter = <New filter value>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control cuyo filtro actual será modificado. Esta control debe ser una: 

- navegar por la Caja de Listas o ListView,

- navegando por Combo Box,

- tabla de navegación o tabla TreeView,

- Looper basado en un archivo de datos.    




**`<New filter value>: (Tipo de filtro)`**



- Nuevo valor para el filtro definido en la clave de búsqueda,

- Cadena vacía (""") para ningún filtro.

<a name="NOTE0_1"></a>

## Observaciones


### Filtrar Property y controles Tabla a partir de un archivo de datos
<a name="filtrar_property_controles_tabla_partir_archivo_datos_ELTPARAGRAPHE000098"></a>

Para controles Tabla basada en un archivo de datos, la casilla "Utilizar la función HFilter para navegar por el archivo" se utiliza para mantener las opciones de búsqueda y ordenación en las columnas filtradas (lupa, etc.).

Si la opción "Utilizar la función HFilter para navegar por el archivo" está seleccionada:

- la Property no se aplica sólo a la control Tabla, sino también a la el archivo de datos en la que se basa la control. Por lo tanto, una exploración realizada en el archivo base de la control será filtrada.

- A el archivo de datos se le aplica automáticamente un filtro ([HFilter](../WDLang4/3044100.md)) en el que se basa control Tabla.

- Si se define un filtro en el archivo de datos ([HFilter](../WDLang4/3044100.md)) ANTES de visualizar el control Tabla ([TableDisplay](../WDLang1/3074003.md) por ejemplo), sólo se tendrá en cuenta el filtro del control Tabla. Entonces, el archivo de datos será navegado según el filtro del control Tabla.



<a name="NOTE0_2"></a>


### Manejo de los controles de la tabla de navegación, list box, ListView, Combo Box y looper a través de las propiedades
<a name="manejo_los_controles_tabla_navegacion_list_box_listview_combo_box_looper_traves_las_propiedades_ELTPARAGRAPHE000128"></a>

- Los registros mostrados en un cuadro de lista, tabla, Combo Box o control Looper basados en un archivo de datos provienen de el archivo de datos o de la consulta especificada con el [BrowsedFile](../Proprietes/2510040.md) Property.

- El sentido de la ordenación se define mediante la [BrowsedItem](../Proprietes/2510111.md) Property.

- Los registros pueden filtrarse mediante la función **Filter** Property o la función [HFilter](../WDLang4/3044100.md). **Atención**: el **Filter** Property sólo puede utilizarse para aplicar un filtro "Empieza por" en la clave de búsqueda.



<a name="NOTE0_3"></a>


### Modificación de las propiedades utilizadas para recorrer una tabla, un cuadro de lista, ListView, Combo Box o control Looper basados en un archivo de datos
<a name="modificacion_las_propiedades_utilizadas_para_recorrer_una_tabla_cuadro_lista_listview_combo_box_control_looper_basados_archivo_datos_ELTPARAGRAPHE000150"></a>

Para modificar varias propiedades ([BrowsedItem](../Proprietes/2510111.md), **Filter** y [BrowsedFile](../Proprietes/2510040.md)), se recomienda:

1. Detener la iteración asignando una cadena vacía ("") a la [BrowsedFile](../Proprietes/2510040.md) Property.

2. Modificar las propiedades relevantes.

3. Reinicia la iteración con la [BrowsedFile](../Proprietes/2510040.md) Property.


Este método se utiliza para optimizar las modificaciones realizadas. En efecto, cada una de las modificaciones realizadas individualmente reinicia la iteración. Estas operaciones pueden llegar a ser bastante lentas cuando se realizan en archivos grandes.
<a name="NOTE0_4"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000177"></a>

El **Filter** Property se aplica sólo a los controles de una ventana o página. Esta Property se aplica únicamente a los siguientes controles:

- Navegando por control List Box,

- Navegando por control ListView,

- Navegando por control Combo Box,

- Navegando por control Tabla,

- Navegando por control Looper,

- Navegando por control Tabla TreeView.

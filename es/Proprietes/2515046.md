


## ObserverOrientation (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ObserverOrientation** obtiene o establece el ángulo secundario de un control Carrusel. Este ángulo secundario hace que el carrusel se "incline":

- hacia adelante o hacia atrás para un control horizontal

- hacia la izquierda o hacia la derecha para un control vertical.




**Observación**: La orientación predeterminada se restablece si el valor de la propiedad [Vertical](../Proprietes/2510133.md) cambia en el control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CAR_Carousel1.ObserverOrientation += 15 // The carousel leans by 15°
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el ángulo secundario de un control Carrusel

`<Result> = <Carousel control>.ObserverOrientation`
---

**`<Result>: (Entero)`**

Valor del ángulo secundario (en grados, de 0 a 360). 

**`<Carousel control>: (Nombre del control)`**

Nombre del control Carrusel cuyo ángulo secundario se va a identificar.  


<a name="SYNTAX2"></a>

### Modificar el ángulo secundario de un control Carrusel

`<Carousel control>.ObserverOrientation = <New angle>`
---

**`<Carousel control>: (Nombre del control)`**

Nombre del control Carrusel cuyo ángulo secundario se va a modificar.

**`<New angle>: (Entero)`**

Valor del ángulo secundario (en grados, de 0 a 360).




## MarginHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MarginHeight** Property se utiliza para: 

- Averiguar el margen vertical entre los widgets que se encuentran en una control Cuadro de mando. 

- Modificar el margen vertical entre los Widgets encontrados en un control Cuadro de mando.




Observación: El margen vertical entre los widgets de una control Cuadro de mando se define en la pestaña "Detalles" de la ventana de descripción de la control Cuadro de mando. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify CBOX_WidgetSpacing
IF CBOX_WidgetSpacing THEN
	DASH_Main.MarginWidth = 20
	DASH_Main.MarginHeight = 20
ELSE
	DASH_Main.MarginWidth = 0
	DASH_Main.MarginHeight = 0
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el margen vertical entre los widgets

`<Result> = <Dashboard control>.MarginHeight`
---

**`<Result>: (Integro)`**

Margen vertical (en píxeles) entre los widgets que se encuentran en la control Cuadro de mando especificada. 

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificación del margen vertical entre los widgets

`<Dashboard control>.MarginHeight = <New margin>`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar. 

**`<New margin>: (Integro)`**

Nuevo margen vertical (en píxeles) entre los widgets que se encuentran en la control Cuadro de mando especificada. Este valor debe ser positivo o nulo. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Este margen no tiene en cuenta la Border definida en el estilo.

- Para cambiar el margen horizontal, utilice la función [MarginWidth](../Proprietes/1000021028.md) Property.




## MaxLineHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxLineHeight** Property se utiliza para get y cambiar la altura máxima en: 

- las filas de una control Tabla (caso de filas multilínea con redimensionamiento de automatic)

- las filas de una control Tabla TreeView. 

- las filas de una control Looper. 




**Observación**: La altura máxima de las filas de una control Tabla se establece en el editor de ventanas, en la pestaña "Detalles" del control description.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Indicates that the maximum height of a row 
// is used to display 2 lines of text.
TABLE_Table1.MaxLineHeight = 2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura máxima de las filas

`<Current maximum height> = <Control used>.MaxLineHeight`
---

**`<Current maximum height>: (Integro)`**

Altura máxima (en caracteres) de las filas para la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: control Tabla, control Tabla TreeView o control Looper. 


<a name="SYNTAX2"></a>

### Modificar la altura de las filas

`<Control used>.MaxLineHeight = <New maximum height>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: control Tabla, control Tabla TreeView o control Looper.

**`<New maximum height>: (Integro)`**

Nueva altura máxima (en caracteres): 

- de las filas de la tabla especificada o de la tabla Treeview control Tabla. 

- de las filas en el control Looper especificado. 


Sólo se modificarán las nuevas líneas visualizadas. Las líneas que ya se visualizan no se modificarán.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Default altura máxima
<a name="default_altura_maxima_ELTPARAGRAPHE000073"></a>

La altura máxima de un fila es set a 0 por Default. En este caso, la altura máxima de una fila corresponde a la altura de fila por defecto.
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000080"></a>

- Los valores posibles están entre 0 y +2147483647 (0x7FFFFFFF).

- El **MaxLineHeight** Property sólo puede utilizarse con los siguientes mandos:

	- Mesa.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) tabla jerárquica. 

	- Looper. 




- La altura de una fila modificada mediante programación con [Altura](../Proprietes/2510050.md) no se ve afectada por este máximo.

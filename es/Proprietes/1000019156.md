
## FreeDLL



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **FreeDLL** puede corresponder a: 

- la función **FreeDLL** que permite descargar una biblioteca de 32 bits (DLL) cargada previamente en memoria. Para obtener más información, consulte [FreeDLL](../WDLang1/3014009.md). 

- el **FreeDLL** Property disponible para un Variable de tipo [API description](../WDLang1/1000019149.md).

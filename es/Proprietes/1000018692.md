
## File (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **File** permite manipular los archivos a los que se aplica una funcionalidad. Esta propiedad se puede utilizar en: 

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). 

- una variable de tipo [zipArchive](../WDLang3/1000018679.md).

- una variable de tipo [hRSConfig](../WDLang4/1000020484.md).




Observación: La palabra clave **Archivo** también puede corresponder a un [Expediente description](../WDLang4/1514065.md).

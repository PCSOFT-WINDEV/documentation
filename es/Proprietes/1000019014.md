


## Note (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Note** obtener y cambiar el texto asociado a un elemento. Esta propiedad se puede utilizar en:

- Las ventanas, las páginas o los controles. Para obtener más información, consulte [Nota](../Proprietes/2510085.md).

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- una variable de tipo [Marker](../WDLang3/1000019940.md).

- una variable de tipo [websocketClient](../WDLang3/1000025952.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagShape](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [sysNotification](../WDLang1/1410087893.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [sysNotificationButton](../WDLang1/1410087947.md).

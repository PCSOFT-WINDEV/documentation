
## Movable (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Movible** Property se utiliza para:

- Determinar si una columna control Tabla puede ser movida por el usuario,

- Permitir y/o evitar que el usuario mueva una columna de control Tabla.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allow the "Name" column to be moved by the user
COL_Name.Movable = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si una columna control Tabla puede ser desplazada

`<Result> = <Column>.Movable`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la columna especificada puede ser movida por el usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Column>: (Nombre del control)`**

Nombre de la columna control Tabla a utilizar.  


<a name="SYNTAX2"></a>

### Permitir y/o evitar que el usuario mueva una columna de control Tabla

`<Column>.Movable = <Move allowed>`
---

**`<Column>: (Nombre del control)`**

Nombre de la columna de control Tabla a utilizar.

**`<Move allowed>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la columna especificada puede ser movida por el usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Por Default, el usuario puede mover una columna de control Tabla. Esta opción puede ser modificada en el editor de la ventana (pestaña "General" de la columna description).

- Cuando se marca "Movible" para una columna, el usuario puede modificar la posición de esta columna (a la derecha o a la izquierda). Para realizar esta acción, el usuario debe:

	- haga clic en el encabezado de esta columna y mantenga pulsado el botón izquierdo del ratón,

	- mover la columna a su nueva posición y soltar el botón izquierdo del ratón.


## spLength (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spLength** permite:

- Obtener la longitud del hash de salida. Esta propiedad puede utilizarse para campos de tipo "Contraseña" definidos en el editor de análisis o mediante programación. 

- Establecer la longitud del hash de salida (solo para campos de tipo "Contraseña" definidos mediante programación). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile file
...

// Describe the "Password" item
MyItem is Item Description
MyItem.Name = "Password"
MyItem.Type = hItemSecurePassword
MyItem.spDerivation = SP_PBKDF2
MyItem.spHash = SP_SHA2_256
MyItem.spIterationCount = 20000
MyItem.spLength = 256

// Validate the description of the "Password" item
HDescribeItem(AFile, MyItem)

// Validate the description of AFile file
...
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la longitud del hash de salida (campo "Contraseña")

`<Length> = <Data file>.<Item>.spLength`
---

**`<Length>: (Entero)`**

Longitud del hash de salida (en bits). La clave debe ser un múltiplo de 8, mayor de 160 bits y menor de 2048 bits. Se recomienda utilizar una clave de 256 o 512 bits. 

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Establecer la longitud del hash de salida (campo "Contraseña" definido mediante programación)

`<Item>.spLength = <Length>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Item Description](../WDLang4/1514071.md).

**`<Length>: (Entero)`**

Longitud del hash de salida (en bits). La clave debe ser un múltiplo de 8, mayor de 160 bits y menor de 2048 bits. Se recomienda utilizar una clave de 256 o 512 bits.

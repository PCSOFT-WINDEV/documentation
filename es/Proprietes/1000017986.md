
## PublicationDate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PublicationDate** Property se utiliza para get la fecha de último acceso a un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [gglCalendar](../WDLang5/1000017418.md). 

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [rssEntry](../WDLang5/1000017792.md).

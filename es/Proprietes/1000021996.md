


## ValuePerStar (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ValueByStar** Property se utiliza para get y cambiar el valor de una estrella en un control Valoración. 

Observación: Esta propiedad corresponde a la opción "Valor de una 'estrella'" disponible en la pestaña "General" de la ventana de description de la control Valoración. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// To use values from 0 to 100 in RATE_Rating1

// The control displays from 0 to 5 stars
RATE_Rating1.MaxValue = 5 
// The value of each star is set to 20
RATE_Rating1.ValuePerStar = 20 
// The values displayed in the control go from 0 to 100 (5 * 20)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el valor de una estrella en un control Valoración

`<Result> = <Rating control>.ValuePerStar`
---

**`<Result>: (Real)`**

Valor associated con una estrella. 

**`<Rating control>: (Nombre del control)`**

Nombre de la control Valoración a utilizar. 


<a name="SYNTAX2"></a>

### Modificar el valor de una estrella en un control Valoración

`<Rating control>.ValuePerStar = <New value>`
---

**`<Rating control>: (Nombre del control)`**

Nombre de la control Valoración a utilizar.

**`<New value>: (Real)`**

Nuevo valor associated con una estrella (o con cualquier otro logo gráfico usado en el control Valoración). Este valor debe ser superior a 0. De lo contrario, este valor se ignora. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Por defecto, el valor de una estrella (o cualquier otro logo) que se encuentra en la control Valoración es set a 1. 

- La propiedad [Valor](../Proprietes/2510130.md) del control Valoración corresponde al número de estrellas multiplicado por el valor de **ValuePerStar**. 

- El número máximo de estrellas encontradas en una control Valoración es devuelto por [MaxValue](../Proprietes/2510009.md).


## FontSize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontSize** Property se utiliza para gestionar el tamaño de la fuente utilizada para un elemento. Esta propiedad se puede utilizar en: 

- controles. Para obtener más información, consulte [FontSize](../Proprietes/2510072.md). 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfTextElement](../WDLang6/1000024556.md).

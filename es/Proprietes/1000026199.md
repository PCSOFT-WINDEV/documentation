
## Attachment (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Attachment** Property se utiliza para gestionar los anexos. Esta propiedad se puede utilizar en: 

- una variable de tipo [pdfDocument](../WDLang6/1000024909.md). 

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

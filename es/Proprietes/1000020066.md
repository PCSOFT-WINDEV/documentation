


## ElementWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ElementWidth** permite:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Descubra o modifique el ancho de los elementos de una control Organigrama.

- Descubra o modifique el ancho de las celdas de un control Cuadro de mando.




Observaciones: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El ancho mínimo de los elementos que se encuentran en una control Organigrama se define en la pestaña "General" de la ventana de descripción de la control Organigrama.

- El ancho de una celda que se encuentra en una control Cuadro de mando se define en la pestaña "Detalles" de la descripción de una control Cuadro de mando. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ORG_Main.ElementWidth = 200
ORG_Main.ElementHeight = 100

IF CBOX_Zoom = True THEN
	ORG_Main.ElementWidth = 400
	ORG_Main.ElementHeight = 200
END
```
<a name="Example2"></a>

```wl
// Reduce the height of all cells in the Dashboard control
IF DASH_Dashboard.ElementHeight > 50 THEN 
	DASH_Dashboard.ElementHeight = 50
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el ancho de los elementos o células

`<Result> = <Control used>.ElementWidth`
---

**`<Result>: (Integro)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Anchura (en píxeles) de los elementos que se encuentran en la control Organigrama especificada.

- Anchura (en píxeles) de las celdas que se encuentran en la control Cuadro de mando especificada.




**`<Control used>: (Nombre del control)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de control Organigrama a utilizar.

- Nombre de la control Cuadro de mando a utilizar.





<a name="SYNTAX2"></a>

### Modificando el ancho de los elementos o células

`<Control used>.ElementWidth = <New width>`
---

**`<Control used>: (Nombre del control)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de control Organigrama a utilizar. 

- Nombre de la control Cuadro de mando a utilizar.




**`<New width>: (Real)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nuevo ancho (en píxeles) de los elementos que se encuentran en la control Organigrama especificada. 

- Nuevo ancho (en píxeles) de las celdas encontradas en la control Cuadro de mando especificada.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El margen entre los elementos se añade a la anchura de los elementos.

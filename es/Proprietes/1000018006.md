
## Email (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Email** permite obtener y cambiar una dirección de correo electrónico. Esta propiedad se puede utilizar en:

- una variable de tipo [gglAuthor](../WDLang5/1000017746.md).

- una variable de tipo [gglConnection](../WDLang5/1000018009.md).

- una variable de tipo [gglPerson](../WDLang5/1000017446.md).

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md).

- una variable de tipo [gpwUser](../WDLang6/1000021088.md).

- una variable de tipo [saasUser](../WDLang3/1000019083.md). 

- una variable de tipo [OpenIDIdentity](../WDLang3/1000023567.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WBAdminAccount](../WDLang2/1410089382.md).




## PersistedCalculatedItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PersistedCalculatedItem** Property se utiliza para:

- Averiguar si el valor de un elemento calculado se almacena en el archivo de datos. Este punto se definió en la editor de análisis o a través de la programación.

- Definir si el valor de un elemento calculado se almacena en el archivo de datos (al describir el elemento mediante programación).




Observación: Los elementos calculados corresponden a los elementos de "Expresión" en la editor de análisis. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
AnItem.Name = "CmdDate"
AnItem.CalculatedItemExpression = "'Written by ' + CURRENT_USER + ' on ' + SYSDATE"
AnItem.PersistedCalculatedItem = True
HDescribeItem(AFile, AnItem)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el valor de un elemento calculado se almacena en el archivo de datos

`<Result> = <Data file>.<Item>.PersistedCalculatedItem`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor del elemento calculado se almacena en el archivo de datos.

- <u><u><u><u>False</u></u></u></u> si el valor del elemento vuelve a calcularse en cada operación de lectura.




**`<Data file>: (Nombre del archivo)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Nombre del campo)`**

Nombre del artículo utilizado.

### Indicando si el valor de un elemento calculado debe ser almacenado en el archivo de datos

`<Item>.PersistedCalculatedItem = <Value>`
---

**`<Item>: (Nombre del campo)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor del elemento calculado debe almacenarse en el archivo de datos.

- <u><u><u><u>False</u></u></u></u> si el valor del elemento debe ser volver a calcularse en cada operación de lectura.

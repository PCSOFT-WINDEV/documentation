
## LargeIcon (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LargeIcon** Property se utiliza para definir el Icon grande utilizado en las notificaciones. Este icono aparecerá a la izquierda del título y del mensaje de Notification.

Este Property se utiliza: 

- por una Variable de tipo [Notification](../WDLang3/1000019441.md). 

- por una Variable de tipo [notificationFormat](../WDLang3/1000021301.md).

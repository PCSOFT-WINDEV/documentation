
## WholeDay

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **WholeDay** Property se utiliza para establecer un Event para todo un día. Esta propiedad se puede utilizar en:

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

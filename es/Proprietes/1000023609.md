
## ProcedureTrace (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Procedure Trace** permite especificar el nombre del WLanguage Procedure utilizado para get las cabeceras y datos enviados. Esta propiedad se puede utilizar en:

- una variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

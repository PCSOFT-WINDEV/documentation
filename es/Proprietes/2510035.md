


## MinHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MinHeight** Property se utiliza para get y cambiar la altura mínima de un control o ventana.

La altura mínima de los mandos y ventanas es de 0 por Default. Sin embargo, algunos controles no pueden ser redimensionados por debajo del tamaño de su Border.

Recordatorio: La altura mínima de una control o ventana se utiliza durante las siguientes operaciones:

- Ventana redimensionada por el usuario.

- Manejo de anclas.

- Ventana redimensionada mediante programación (con la función [Altura](../Proprietes/2510050.md) Property o la función [WinSize](../WDLang1/3038020.md)).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the minimum height of a control
EDT_Edit1.MinHeight = 200
EDT_Edit1.Height = 150
// In fact, the height of the control will be equal to 200
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura mínima de un elemento

`<Current minimum height> = <Element used>.MinHeight`
---

**`<Current minimum height>: (Integro)`**

Altura mínima de un elemento específico (incluyendo la Border) expresada en píxeles.

**`<Element used>: (Nombre de control o ventana)`**

Nombre del elemento (ventana o control) cuya altura mínima se calculará.  


<a name="SYNTAX2"></a>

### Modificar la altura mínima de un elemento

`<Element used>.MinHeight = <New minimum height>`
---

**`<Element used>: (Nombre de control o ventana)`**

Nombre del elemento (ventana o control) cuya altura mínima será modificada

**`<New minimum height>: (Integro)`**

Nueva altura mínima del elemento (incluyendo la Border) expresada en píxeles.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Altura inicial (control y ventana)
<a name="altura_inicial_control_ventana_ELTPARAGRAPHE000092"></a>

La altura inicial (descrita en el editor) es devuelta por el [InitialHeight](../Proprietes/2510043.md) Property.

En la mayoría de los casos, la altura de un elemento depende de las propiedades [MaxHeight](../Proprietes/2510065.md) y **MinHeight**. Sin embargo, esta restricción ya no se aplica cuando la altura se modifica mediante programación (con la función [Altura](../Proprietes/2510050.md) Property o la función [WinSize](../WDLang1/3038020.md)).

### Límites
<a name="limites_ELTPARAGRAPHE000117"></a>

El **MinHeight** Property se aplica únicamente a los siguientes elementos:

- Botón.

- Estática.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Mesa.

- Supercontrol.


## Movable (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Movible** Property se utiliza para definir si un elemento puede ser movido. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una columna de una control Tabla. Para más detalles, consulte la ayuda del programa [Móviles](../Proprietes/2510032.md) Property.

- una variable de tipo [Marker](../WDLang3/1000019940.md).

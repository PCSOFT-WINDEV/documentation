


## VisibleInExportAndPrint (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VisibleInExportAndPrint** Property se utiliza para: 

- Determinar si una columna de una Tabla o control Tabla TreeView es Visible para su exportación e impresión.

- Modificar la visibilidad de una columna en una Tabla o control Tabla TreeView para exportar e imprimir.




Observación: Esta propiedad corresponde a la opción "Exportar e imprimir" de la pestaña "General" de la ventana description de una columna de una Tabla o control Tabla TreeView. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Print without details] button
TABLE_CUSTOMER.COL_Details.VisibleInExportAndPrint = False
iPrintReport(RPT_ReportOnCustomerTable)
TABLE_CUSTOMER.COL_Details.VisibleInExportAndPrint = Default
```
<a name="Example2"></a>

```wl
// Export to Excel:
// 1. The COL_Details column is exported
TABLE_MyTable.COL_Details.VisibleInExportAndPrint = False
TableToExcel(TABLE_MyTable, sFile, taNoTitle + taWithInvisibleColumns)
// 2. The COL_Details column is not exported
TABLE_MyTable.COL_Details.VisibleInExportAndPrint = False
TableToExcel(TABLE_MyTable, sFile, taNoTitle)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averigua si una columna de una tabla o control Tabla TreeView es Visible para exportar e imprimir

`<Result> = <Column>.VisibleInExportAndPrint`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de gestión de la columna: 


|   |   |
| --- | --- |
| *Default* | La columna se exporta o se imprime sólo si está en Visible en la pantalla. |
| *False* | La columna nunca se puede exportar o imprimir. <br>**Atención**: Cuando se exporta con una de las funciones de exportación ([TablaToExcel](../WDLang1/3074033.md), [TablaToText](../WDLang1/1000017061.md), [TableToWord](../WDLang1/3074036.md), [TableToXML](../WDLang1/3074037.md), [TablaToClipboard](../WDLang1/1000017062.md)), si se utiliza la constante *taWithInvisibleColumns*, la columna se seguirá exportando. |
| *True* | La columna puede ser exportada o impresa (incluso si se trata de Invisible en la pantalla). |



**`<Column>: (Nombre del control)`**

Nombre de la columna a utilizar. Esta columna puede ser una columna que se encuentra en una Tabla o control Tabla TreeView. 


<a name="SYNTAX2"></a>

### Modificar la visibilidad de una columna en una Tabla o control Tabla TreeView para exportar e imprimir

`<Column>.VisibleInExportAndPrint = <Visibility>`
---

**`<Column>: (Nombre del control)`**

Nombre de la columna a utilizar. Esta columna puede ser una columna que se encuentra en una Tabla o control Tabla TreeView. 

**`<Visibility>: (Constante de tipo Integer)`**

Modo de gestión de la columna: 


|   |   |
| --- | --- |
| *Default* | La columna se exporta o se imprime sólo si está en Visible en la pantalla. |
| *False* | La columna nunca se puede exportar o imprimir.<br>**Atención**: Cuando se exporta con una de las funciones de exportación ([TablaToExcel](../WDLang1/3074033.md), [TablaToText](../WDLang1/1000017061.md), [TableToWord](../WDLang1/3074036.md), [TableToXML](../WDLang1/3074037.md), [TablaToClipboard](../WDLang1/1000017062.md)), si se utiliza la constante *taWithInvisibleColumns*, la columna se seguirá exportando. |
| *True* | La columna puede ser exportada o impresa (incluso si se trata de Invisible en la pantalla). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **VisibleInExportAndPrint** Property se tiene en cuenta en los siguientes casos:  

- impresión realizada a través de la impresión AAF (automatic Application Feature): imprimir mediante la opción "Imprimir" del menú contextual de control Tabla.  

- impresión realizada a través de la exportación de AAFs en el menú contextual del control Tabla.

- exportar mediante las funciones de exportación ([TablaToExcel](../WDLang1/3074033.md), [TablaToText](../WDLang1/1000017061.md), [TableToWord](../WDLang1/3074036.md), [TableToXML](../WDLang1/3074037.md), [TablaToClipboard](../WDLang1/1000017062.md)).

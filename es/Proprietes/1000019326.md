
## Inherited (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Inherited** Property se utiliza para determinar si un elemento es heredado. Esta propiedad es utilizada por:

- una variable de tipo [Procedure Description](../WDLang1/1000019317.md).

- una variable de tipo [Property Description](../WDLang1/1000019324.md).

- una variable de tipo [Variable Description](../WDLang1/1000019560.md).

- una variable de tipo [Option Description](../WDLang1/1000023072.md). 

- una variable de tipo [Value description](../WDLang1/1000023061.md).

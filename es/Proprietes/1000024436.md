
## SignatureMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SignatureMode** Property se utiliza para get y cambiar las opciones de firma de un control Firma en un reporte. 

Observación: Esta propiedad corresponde a la opción "Modo de firma" de la pestaña "General" de la ventana control Firma description. Este control sólo está disponible en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SIG_MySignature.SignatureMode = iSignature_PKCS7 THEN
	bChorusMode = False
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de firma de una control Firma

`<Result> = <Signature control>.SignatureMode`
---

**`<Result>: (Constante de tipo Integer)`**

El modo de firma de la control especificada. Los valores posibles son: 



|   |   |
| --- | --- |
| *iSignature_CAdES_B* | CAdES Formato básico de firma electrónica. |
| *iSignature_CAdES_C* | CAdES C formato de firma electrónica. |
| *iSignature_CAdES_T* | El CAdES confía en el formato de firma electrónica de Token. |
| *iSignature_CAdES_X* | CAdES X Formato de firma electrónica tipo 2. |
| *iSignature_CAdES_XL* | CAdES XL Tipo 2 formato de firma electrónica. |
| *iSignature_PKCS7* | Formato de firma PKCS7. Este es un formato estándar de Adobe no aprobado por los estándares ETSI y Chorus.. |


Recordatorio: CAdES (CMS Advanced Electronic Signature) es un estándar que permite firmas "envueltas" o "despegadas".

**`<Signature control>: (Nombre del control)`**

Nombre de control Firma a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el modo de firma de una control Firma

`<Signature control>.SignatureMode = <New signature mode>`
---

**`<Signature control>: (Nombre del control)`**

Nombre de control Firma a utilizar.  

**`<New signature mode>: (Constante de tipo Integer)`**

El nuevo modo de firma de la control especificada. Los valores posibles son: 



|   |   |
| --- | --- |
| *iSignature_CAdES_B* | CAdES Formato básico de firma electrónica. |
| *iSignature_CAdES_C* | CAdES C formato de firma electrónica. |
| *iSignature_CAdES_T* | El CAdES confía en el formato de firma electrónica de Token. |
| *iSignature_CAdES_X* | CAdES X Formato de firma electrónica tipo 2. |
| *iSignature_CAdES_XL* | CAdES XL Tipo 2 formato de firma electrónica. |
| *iSignature_PKCS7* | Formato de firma PKCS7. Este es un formato estándar de Adobe no aprobado por los estándares ETSI y Chorus.. |


Recordatorio: CAdES (CMS Advanced Electronic Signature) es un estándar que permite firmas "envueltas" o "despegadas".

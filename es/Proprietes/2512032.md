


## LogFile (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LogFile** Property se utiliza para determinar si un archivo de datos es un archivo de registro.

**Recordatorio**: Un archivo de registro se utiliza para almacenar las diferentes operaciones realizadas en un archivo de registro.. Se crea un archivo de registro si se han definido opciones de registro para uno o más archivos de datos del análisis.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EXTERN SupplierJNL
HDeclareExternal("C:\WINDEV\Examples\SalesMgt\SupplierJNL.fic", "SupplierJNL")
IF SupplierJNL.LogFile = True THEN
	Info("You are handling a log file")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Data file>.LogFile`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo es un archivo de registro,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 


Observación: Los archivos de datos descritos en el análisis no son archivos de "registro".

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Puede corresponder a un nombre de alias de archivo declarado por [HDeclareExternal](../WDLang4/3044204.md).

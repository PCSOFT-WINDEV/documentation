


## RightToLeft (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RightToLeft** Property se utiliza para get la dirección de escritura real en un control, columna, ventana o reporte. 

La dirección de escritura puede modificarse con el [TextDirection](../Proprietes/2513020.md) Property. Si la dirección de escritura en el control es la misma dirección definida en las opciones del proyecto, el [TextDirection](../Proprietes/2513020.md) Property devuelve la constante *tdDefault*. En este caso, el **RightToLeft** Property se utiliza para get la dirección de escritura real

Este Property se aplica a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Una ventana o los controles de una ventana (a excepción de: Control de subíndice, Web control Cámara, Splitter, Calendar y Organizador).

- Un reporte o los siguientes controles reporte: 

	- Control Estático, 

	- Control Calculado, 

	- Texto RTF control,

	- Control Predefinido, 

	- Enlazar y editar control,

	- Control Gráfico.







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Atención: Esta Property está disponible sólo para informes.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// The writing direction used in the column is the default direction
TABLE_Table1.COL_Column2.TextDirection = tdDefault
// Find the default writing direction
IF TABLE_Table1.COL_Column2.RightToLeft = True THEN
	EDT_Edit1.TextDirection = tdRightToLeft
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la dirección de escritura que se utiliza realmente

`<Result> = <Element name>.RightToLeft`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la dirección de escritura utilizada es de derecha a izquierda,

- <u><u><u><u>False</u></u></u></u> si la dirección de escritura utilizada es de izquierda a derecha.




**`<Element name>: (Nombre de la ventana, reporte o control)`**

Nombre del elemento cuya dirección de escritura se solicita: 

- Nombre de la ventana, 

- Nombre de los controles de la ventana,

- Nombre de la reporte,

- Nombre de los controles reporte. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro corresponde al nombre del reporte o al nombre de un reporte control.

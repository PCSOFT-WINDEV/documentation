
## Caption (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Caption** Property se utiliza para get y cambiar el texto de un elemento. Esta propiedad se puede utilizar en:

- controles, ventanas, páginas e informes. Para obtener más información, consulte [Pie de foto](../Proprietes/2510053.md).

- una variable de tipo [gglPostalAddress](../WDLang5/1000017575.md).

- una variable de tipo [gglContact](../WDLang5/1000017434.md).

- una variable de tipo [gglEmail](../WDLang5/1000017571.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [gglLocation](../WDLang5/1000017441.md).

- una variable de tipo [gglInstantMessaging](../WDLang5/1000017569.md).

- una variable de tipo [gglPerson](../WDLang5/1000017446.md).

- una variable de tipo [gglOrganization](../WDLang5/1000017756.md).

- una variable de tipo [gglPhoneNumber](../WDLang5/1000017573.md).

- una variable de tipo [inAppProduct](../WDLang3/1000020868.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Token](../WDLang1/1000023942.md).

- una variable de tipo [MarkerImage](../WDLang3/1000025796.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [sysNotificationButton](../WDLang1/1410087947.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [sysNotificationInput](../WDLang1/1410088024.md).

- una variable de tipo [AssistedInput](../WDLang1/1410088983.md).

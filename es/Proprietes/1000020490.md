
## Server (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Server** permite gestionar los servidores. Esta propiedad se puede utilizar en:

- una conexión HFSQL. Para obtener más información, consulte [Servidor](../Proprietes/2512105.md).

- una variable de tipo [hRSConfig](../WDLang4/1000020484.md).

- una variable de tipo [hbConnection](../WDLang4/1000021685.md).

- una variable de tipo [memcachedConnection](../WDLang4/1000023339.md).

- una variable de tipo [redisConnection](../WDLang4/1000023527.md).

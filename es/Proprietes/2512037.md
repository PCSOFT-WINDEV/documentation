


## HugeFile (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HugeFile** Property se utiliza para configurar el tamaño máximo de un archivo de datos. Es posible:

- Determine si un archivo de datos está configurado para contener más de 2 GB de datos (archivo definido en el editor de análisis o mediante programación).

- Definir las características relativas al tamaño de un archivo de datos (cuando el fichero se describe mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the maximum size of the Customer data file
Customer.HugeFile = True
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el tamaño máximo de un archivo de datos

`<Result> = <Data file>.HugeFile`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos está configurado para contener más de 2 GB de datos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).


<a name="SYNTAX2"></a>

### Definición del tamaño máximo de del archivo de datos (definido mediante programación)

`<Data file>.HugeFile = <Size 2GB>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<Size 2GB>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos debe estar configurado para contener más de 2 GB de datos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Archivos de datos de más de 2 GB
<a name="archivos_datos_mas_2_ELTPARAGRAPHE000085"></a>

Los archivos de datos que superen los 2 GB sólo pueden crearse en los discos NTFS.. Además, el sistema operativo debe soportar los archivos que superen los 2 GB.

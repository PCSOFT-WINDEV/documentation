
## CollapsedImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CollapsedImage** Property se utiliza para: 

- descubrir y/o modificar la imagen por defecto associated con un fila colapsado en un control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) descubrir y/o modificar la imagen por defecto associated con un fila colapsado en un control TreeView.

- descubrir y/o modificar la imagen associated con un fila colapsado específico en un control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) conocer y/o modificar la imagen de una fila contraída en un elemento jerárquico de un reporte de tipo "Tabla TreeView". 




Esta Property se utiliza para definir: 

- la imagen que será usada por Default durante las próximas adiciones en el control Tabla TreeView. Esta Image será tenida en cuenta por [TableAdd](../WDLang1/3074017.md), [TableAddLine](../WDLang1/3074006.md), [TableAddChild](../WDLang1/1000010004.md), [TableInsert](../WDLang1/3074025.md), [TableInsertLine](../WDLang1/3074026.md), [TableInsertChild](../WDLang1/1000010005.md), etc.

- el Image colapsado para una de las filas encontradas en el control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la imagen que será usada por Default durante las próximas adiciones en el control TreeView. Esta Image será tenida en cuenta por [TreeAdd](../WDLang1/3018012.md) y [TreeInsert](../WDLang1/3018023.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) la Image colapsada de las filas en una "Tabla de Vista de Árbol" reporte.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the image used for a collapsed row
TVT_Recipe.CollapsedImage = "C:\MyImages\Image.BMP"
```


<a name="Example2"></a>



```wl
// Change the collapsed image associated with row 4
TVT_Recipe[4].CollapsedImage = "C:\MyImages\Image.BMP"
```


<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Change the collapsed image in the report
// --Before printing ITEM_TreeView
If MySelf.HierarchicalDepth = 1 THEN MySelf.CollapsedImage = "C:\Images\tool.png"
```







<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la imagen por defecto associated con un fila colapsado o la imagen associated con un fila específico

`<Result> = <Control used>.CollapsedImage`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre y ruta completo de la imagen por defecto associated con las filas colapsadas en el control. 

- Cadena vacía ("") si la imagen usada por Default no fue modificada.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView.




Para averiguar la imagen colapsada associated con un fila específico en un control Tabla TreeView, use la siguiente notación: 

```txt
<TreeView Table>[<Row number>]
```



<a name="SYNTAX2"></a>

### Modificando la imagen por defecto associated con un fila colapsado o la imagen associated con un fila específico

`<Control used>.CollapsedImage = <New image>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView.




Para modificar la imagen colapsada associated con un fila específico en un control Tabla TreeView, use la siguiente notación: 

```txt
<TreeView Table>[<Row number>]
```


**`<New image>: (Cadena de caracteres)`**

Nombre y ruta completo de la nueva imagen associated por defecto con un fila colapsado.  




<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Conocer la imagen de un elemento jerárquico contraído en un reporte "Tabla TreeView"

`<Result> = <TreeView item>.ExpandedImage`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre y ruta completa de la imagen asociada con un fila colapsado en el elemento TreeView. 

- Vaciar la cadena ("") si no se utiliza la Image.




**`<TreeView item>: (Cadena de caracteres)`**

Nombre del elemento TreeView a utilizar.


<a name="SYNTAX4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Modificando la Image de un elemento colapsado de TreeView en una "Tabla de TreeView" reporte

`<TreeView item>.ExpandedImage = <New image>`
---

**`<TreeView item>: (Cadena de caracteres)`**

Nombre del elemento TreeView a utilizar.  

**`<New image>: (Cadena de caracteres)`**

Nombre y ruta completa de la nueva imagen asociada con un fila colapsado en el elemento TreeView.

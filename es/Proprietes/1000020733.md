
## Method (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Method** especifica el método utilizado. Esta propiedad se puede utilizar en:

- una variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- una variable de tipo [gglOverride](../WDLang5/1000026204.md).

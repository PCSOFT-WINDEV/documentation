


## TotalNbRec (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TotalNbRec** Property se utiliza para:

- get el número de registros que se van a imprimir (por ejemplo, un reporte basado en un archivo, zona de memoria o consulta).

- Modificar el número de registros a imprimir (es decir, set un límite, por ejemplo, imprimir sólo los primeros 50 registros).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the maximum number of records to be printed
RPT_Customer.TotalNbRec = 25
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el número máximo de registros a imprimir

`<Result> = <Report used>.TotalNbRec`
---

**`<Result>: (Integro)`**



- Número de registros a imprimir,

- 0 si se deben imprimir todos los registros.




**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del número máximo de registros a imprimir

`<Report used>.TotalNbRec = <New maximum number>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte a utilizar.

**`<New maximum number>: (Integro)`**



- Nuevo número de registros a imprimir,

- 0 si todos los registros deben imprimirse.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Número de registros
<a name="numero_registros_ELTPARAGRAPHE000063"></a>

- El [NbRecRead](../Proprietes/2511041.md) Property devuelve el número de registros seleccionados por la fuente del reporte (consulta, vista, zona de memoria o control Tabla).

- El [NbPrintedRec](../Proprietes/2511058.md) Property devuelve el número de registros seleccionados tanto por la fuente del reporte COMO por el código de filtro (si lo hay).

- Las propiedades [NbRecRead](../Proprietes/2511041.md) y [NbPrintedRec](../Proprietes/2511058.md) devuelven el mismo valor para un reporte basado en un archivo: el número de registros en el archivo Y seleccionados por el código del filtro (si lo hay).

- El **TotalNbRec** Property se utiliza para determinar y modificar el número de registros a imprimir. El número de registros a imprimir estará comprendido entre 1 y el valor de la [NbPrintedRec](../Proprietes/2511058.md) Property.



<a name="NOTE0_2"></a>


### Tipo de reporte
<a name="tipo_reporte_ELTPARAGRAPHE000090"></a>

Si la reporte es relacionado to:

- ***un archivo HFSQL***, los registros a imprimir se seleccionan al principio del archivo de datos.

- ***a consulta***, el número máximo de registros a imprimir puede ser conocido y/o modificado en el description de opciones avanzadas de la consulta. Para mostrar las opciones avanzadas de la consulta relacionado a reporte: 

	- Seleccione "Editar la consulta" en el menú contextual de reporte. Se visualiza el editor de consultas. 

	- Seleccione "Consulta description" en el menú contextual de la consulta. 

	- En la ventana que aparece, haga clic en el botón "Avanzado".






<a name="NOTE0_3"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000104"></a>

El **TotalNbRec** Property no se puede utilizar en los informes sin una fuente de datos.

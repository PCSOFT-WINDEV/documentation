


## Language (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Language** permite establecer el idioma utilizado para comparar y ordenar campos Unicode (campos de tipo Unicode Text o Unicode Text Memo). Es posible:

- Averigua los parámetros de lenguaje usados en tiempo de ejecución para comparar y clasificar. Esta Property puede utilizarse para un elemento definido en la editor de análisis o definido mediante programación. Se tienen en cuenta las modificaciones realizadas por el [H.LanguageCreationUnicode](../WDLang4/3044382.md) Variable.

- Definir los parámetros del lenguaje utilizado para las comparaciones y para la ordenación (cuando el artículo se describe mediante programación). En este caso, esta Property equivale a la [LanguageDescribed](../Proprietes/1000017333.md) Property.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the language of the item
Trace("Language used in the data file: ") + Customer.Name.Language
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar los parámetros del lenguaje

`Language parameters> = <Data file>.<Item>.Language`
---

**`<Language settings>: (Cadena de caracteres)`**

Características del lenguaje utilizado para las comparaciones y para la ordenación, en el siguiente formato:

```txt
<Language identifier> + TAB + <Country identifier> + TAB + <Variant>
```

Si los parámetros de idioma se han definido en el análisis como: 

- "Definidos durante la creación", &lt;Parámetros del idioma&gt; corresponderán a la constante *hLanguageCreation* (o al valor "##CREATION##"+TAB+"##CREATION##"+TAB+"##CREATION##").

- "Orden binario", &lt;Parámetros del idioma&gt; corresponderá a la constante *hLanguageBinarySort* (o al valor "##BINARY##"+TAB+"##BINARY##"+TAB+"##BINARY##").

- "Orden rápido (multilingüe)", &lt;Parámetros del idioma&gt; corresponderá a la constante *hLanguageFastSort* (o al valor "##FASTSORT##"+TAB+"##FASTSORT##"+TAB+"##FASTSORT##").




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

### Definición de los parámetros del lenguaje (elemento descrito mediante programación)

`<Item>.Language = <Language settings>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Language settings>: (Cadena de caracteres)`**

Características del idioma que se utilizará para las comparaciones y para la ordenación, en el siguiente formato:

```txt
<Language identifier> + TAB + <Country identifier> + TAB + <Variant>
```

 


- Para utilizar un orden binario, &lt;Parámetros del idioma&gt; debe corresponder a la constante *hLanguageBinarySort* (o al valor "##BINARY##"+TAB+"##BINARY##"+TAB+"##BINARY##").

- Para utilizar un orden binario, &lt;Parámetros del idioma&gt; debe corresponder a la constante *hLanguageFastSort* (o al valor "##FASTSORT##"+TAB+"##FASTSORT##"+TAB+"##FASTSORT##").

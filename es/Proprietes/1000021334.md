
## FormulaBarVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FormulaBarVisible** Property se utiliza para: 

- Averigüe si se muestra una barra de fórmulas en un control Hoja de cálculo. 

- Visualizar (o no) una barra de fórmulas en un control Hoja de cálculo. 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Tableur_Barre_Formule.gif)


Recordatorio: 

- En un control Hoja de cálculo, la barra de fórmulas se utiliza para escribir fórmulas y nombres.

- Esta propiedad corresponde a la opción "Con barra de fórmulas" de la pestaña "Detalles" de la ventana control Hoja de cálculo description.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_FormulaBar THEN
	PSHEET_Spreadsheet.FormulaBarVisible = True
ELSE
	PSHEET_Spreadsheet.FormulaBarVisible = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averigüe si se muestra una barra de fórmulas en un control Hoja de cálculo

`<Result> = <Spreadsheet control>.FormulaBarVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la barra de fórmulas es visible, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 


<a name="SYNTAX2"></a>

### Visualizar (o no) una barra de fórmulas en un control Hoja de cálculo

`<Spreadsheet control>.FormulaBarVisible = <Display>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer visible la barra de fórmulas,

- <u><u><u><u>False</u></u></u></u> para hacer la barra de la fórmula Invisible. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Puede personalizar la barra de fórmulas en un control Hoja de cálculo. Para obtener más información, consulte [Personalización de la barra de fórmulas (control Hoja de cálculo)](../WDChamp/1000033001.md).

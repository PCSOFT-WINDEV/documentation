


## PhysicalName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PhysicalName** Property se utiliza para gestionar el nombre físico de los archivos de datos HFSQL. Esta Property tiene en cuenta los posibles cambios de nombre realizados por [HChangeName](../WDLang4/3044141.md). Es posible:

- Identificar el nombre físico de un archivo de datos definido en la editor de análisis o mediante programación.

- Definir un nombre físico para un archivo de datos (sólo cuando se describe el archivo de datos mediante programación).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the physical name of the Customer data file
ResPhysName = Customer.PhysicalName
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el nombre físico de un archivo de datos

`<Current name> = <Data file>.PhysicalName`
---

**`<Current name>: (Cadena de caracteres)`**

Nombre físico del archivo de datos (sin extensión). Este nombre tiene en cuenta los posibles cambios de nombre ([HChangeName](../WDLang4/3044141.md)).

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

### Definir un nombre físico para un archivo de datos creado mediante programación

`<Data file>.PhysicalName = <New name>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<New name>: (Cadena de caracteres)`**

Nombre físico del archivo de datos (sin extensión).

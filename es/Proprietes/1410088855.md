
## ActionAllowed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **AllowedAction** define qué acciones están permitidas o no en un elemento. Esta propiedad se puede utilizar en: 

- una variable de tipo [diagShape](../WDLang1/1410088082.md).

- una variable de tipo [diagPath](../WDLang1/1410088384.md), 

- una variable de tipo [diagGroup](../WDLang1/1410088127.md), 

- una variable de tipo [diagImage](../WDLang1/1410088414.md), 

- una variable de tipo [diagConnector](../WDLang1/1410088454.md), 

- una variable de tipo [diagOval](../WDLang1/1410088135.md), 

- una variable de tipo [diagPolygon](../WDLang1/1410088137.md), 

- una variable de tipo [diagRectangle](../WDLang1/1410088083.md), 

- una variable de tipo [kbCard](../WDLang1/1410089167.md).

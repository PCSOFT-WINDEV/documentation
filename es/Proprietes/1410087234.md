
## HeightWithAnimation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HeightWithAnimation** Property se utiliza para modificar la altura de una celda de un control Disposición con la animación "Resize cell" definida: 

- en la pestaña "Avanzado" de la ventana del proyecto description (botón "Animación control"). 

- en la pestaña "Estilo" de control o del proyecto. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the height of the 3rd cell with animation
LAYOUT_Data[3].HeightWithAnimation = LAYOUT_Data[3].Height + 50
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

### Modificar la altura de una celda de una control Disposición con una animación

`<Cell>.HeightWithAnimation = <New height>`
---

**`<Cell>: (Nombre del control)`**

Celda en un control Disposición. La sintaxis utilizada para designar la célula control Disposición es la siguiente: 

```txt
<Layout control>[<Cell number>]
```


**`<New height>: (Integro)`**

Nueva altura de control. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las propiedades de la animación (duración, por ejemplo) pueden especificarse en las opciones del proyecto y anularse en el estilo control. 

- Esta Property permite redimensionar el control Disposición con una animación.

- Esta Property sólo funciona en controles verticales de Layout (celdas una debajo de la otra).

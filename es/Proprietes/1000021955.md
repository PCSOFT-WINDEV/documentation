


## TimestampUTC (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TimestampUTC** Property se utiliza para:

- Averiguar si un elemento de marca de tiempo utilizará la hora UTC o la hora local. Este punto se definió en la editor de análisis o a través de la programación.

- Defina si un elemento de la marca de tiempo utilizará una hora UTC o una hora local (al describir el elemento por programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
AnItem.Name = "CmdDate"
AnItem.TimestampType = htsCreation
AnItem.TimestampUTC = False
HDescribeItem(AFile, AnItem)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si un elemento de marca de tiempo utilizará la hora UTC o la hora local

`<Result> = <Data file>.<Item>.TimestampUTC`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento de la marca de tiempo utiliza una hora UTC.

- <u><u><u><u>False</u></u></u></u> si el elemento de la marca de tiempo utiliza una hora local.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado.

### Indicando si un elemento de marca de tiempo usará una hora UTC

`<Item>.TimestampUTC = <Value>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

**`<Value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el campo de tipo marca de tiempo debe utilizar una hora UTC.

- <u><u><u><u>False</u></u></u></u> si el campo de tipo marca de tiempo debe utilizar una hora local.

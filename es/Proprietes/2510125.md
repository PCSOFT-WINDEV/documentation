


## Size (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Size** Property se utiliza para determinar y modificar el tamaño de un elemento. 

Esta Property puede ser usada para descubrir y modificar: 

- el tamaño (en picas) de una [tipo de letra](../Motscles/1514045.md) variable. 
	**Observación**: el nombre de la fuente debe corresponder a una de las fuentes instaladas en el ordenador actual. El tamaño de la letra es set a 12 por Default.

- el número máximo de caracteres que se encuentran en:

	- una ventana control (control Campo de entrada, columna tabla o cuadro combinado editable)

	- a página de control (control Campo de entrada)

	- a control Estático en un reporte.
			**Observación**: El número máximo de caracteres que se encuentran en un control se define en la pestaña "General" del control description.




- el tamaño máximo de un elemento de texto (cadena de caracteres, nota de texto, carácter) definido en la editor de análisis o mediante programación.

- el tamaño máximo de un elemento de Tiempo definido en la editor de análisis o mediante programación.




Esta Property también puede ser usada para descubrir: 

- el número de columnas de una control Tabla.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000002510125&name=size_property&product=WD">Ver ejemplos adicionales</a>

```wl
// Define the maximum number of characters found in the "DelivAddress" item
DelivAddress.Size = 50
...
// Define the maximum number of characters found in the "DelivTime" item
DelivTime.Size = 6 // HHMMSS format
...
// Validate the data file description
HDescribeFile(Customer)
```
<a name="Example2"></a>

```wl
// Modify the maximum number of characters found in the "EDT_Name" control
EDT_Name.Size = 25
```

## Sintaxis

### Averiguar el tamaño de una Fuente Variable, el número máximo de caracteres encontrados en una control

`<Result> = <Element to use>.Size`
---

**`<Result>: (Integro)`**



- Tamaño (en picas) de la fuente especificada.

- Número máximo de caracteres encontrados en la control especificada o 0 si la control especificada no tiene límite.

- Número de columnas tabla si el control especificado es un tabla.




**`<Element to use>: (Fuente opcional Variable o nombre de control)`**



- Nombre de la variable de tipo [tipo de letra](../Motscles/1514045.md) que se utilizará.

- Nombre del control a manipular  





<a name="SYNTAX2"></a>

### Modificar el tamaño de una fuente Variable, el número máximo de caracteres que se encuentran en un control

`<Element to use>.Size = <New size>`
---

**`<Element to use>: (Fuente opcional Variable o nombre de control)`**



- Nombre de la variable de tipo [tipo de letra](../Motscles/1514045.md) que se utilizará.

- Nombre del control a manipular.




**`<New size>: (Integro)`**



- Nuevo tamaño (en picas) para la fuente especificada.

- Número máximo de caracteres encontrados en la control especificada o 0 si la control especificada no tiene límite.




Observación: El número de columnas que se encuentran en una control Tabla no se puede modificar.  


<a name="SYNTAX3"></a>

### Averiguar el tamaño de un artículo

`<Result> = <Data file>.<Item>.Size`
---

**`<Result>: (Integro)`**



- Número máximo de caracteres soportados por el elemento especificado,

- 0 para las notas.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

***Si esta posición es una posición de texto***número de caracteres en el elemento.

***Si esta posición es una posición de tiempo***el número de caracteres se utiliza para identificar el formato del artículo:

| Número de caracteres<br>en el elemento de tiempo | Formato del elemento Tiempo |
| --- | --- |
| 2 | HH |
| 4 | HHMM |
| 6 | HHMMSS |
| 8 | HHMMSSCC |
| 9 | HHMMSSCCC |

### Definir el tamaño (número de caracteres) de un elemento descrito mediante programación

`<Item>.Size = <Number of characters>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

***Si esta posición es una posición de texto***número de caracteres en el elemento.

***Si esta posición es una posición de tiempo***el número de caracteres se utiliza para identificar el formato del artículo:

| Número de caracteres en el elemento Tiempo | Formato del elemento Tiempo |
| --- | --- |
| 2 | HH |
| 4 | HHMM |
| 6 | HHMMSS |
| 8 | HHMMSSCC |
| 9 | HHMMSSCCC |



**`<Number of characters>: (Integro)`**

Nuevo número máximo de caracteres soportados por el elemento especificado.

### Variable de tipo Font
<a name="variable_tipo_font_ELTPARAGRAPHE000237"></a>

Se puede crear una fuente a partir de una variable de tipo [Font](../Motscles/1514045.md).

Para definir las características de la fuente, utilice:

- la función [FontCreate](../WDLang1/3060002.md).
	

- las propiedades [Charset](../Proprietes/2512008.md), [Orientation](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Color](../Proprietes/2510071.md), [Bold](../Proprietes/2513001.md), [Italic](../Proprietes/2513007.md), [CharacterSpacing](../Proprietes/1410088367.md), [LineSpacing](../Proprietes/1410088366.md), [Name](../Proprietes/2510082.md), [Underline](../Proprietes/2513013.md), [Size](../Proprietes/2510125.md) y [SizeUnit](../Proprietes/1410088365.md).
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Solo se pueden utilizar las siguientes propiedades con el tipo **Font**: [Orientation](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Color](../Proprietes/2510071.md), [Bold](../Proprietes/2513001.md), [Italic](../Proprietes/2513007.md), [Name](../Proprietes/2510082.md), [Underline](../Proprietes/2513013.md), [Size](../Proprietes/2510125.md).
	




Esta fuente se puede utilizar:

- Para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- Para escribir en los gráficos (con las funciones [grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md) y [grAxisTitleFont](../WDLang3/3042059.md)).

- Para imprimir (con la función [iFont](../WDLang5/3046014.md)).

- En los controles de una ventana (propiedad [Font](../Motscles/1514045.md)).

- En los controles de un reporte (propiedad [Font](../Motscles/1514045.md)).





<a name="NOTE0_2"></a>


### Número de caracteres
<a name="numero_caracteres_ELTPARAGRAPHE000244"></a>

Si el número máximo de caracteres que se ha especificado es menor que el número de caracteres que se encuentra actualmente en el control:

- El contenido de la control no se modifica en la pantalla.

- El valor de la control incluye todos los caracteres mostrados en la control.



<a name="NOTE0_3"></a>


### Control Campo de entrada multilínea
<a name="control_campo_entrada_multilinea_ELTPARAGRAPHE000254"></a>

Si se modifica el número máximo de caracteres permitidos en un control Campo de entrada multilínea con el **Size** Property, se podrá asignar un texto que contenga más caracteres que el número que se definió. Por ejemplo:


```wl
// "EDT_Address" is a multiline edit control
EDT_Address.Size = 5
EDT_Address = "ABCDEFGHI" 
// "EDT_Address" contains "ABCDEFGHI"
```

<a name="NOTE0_4"></a>


### control con máscara de entrada
<a name="control_con_mascara_entrada_ELTPARAGRAPHE000265"></a>

El **Size** Property no debe utilizarse para modificar el número máximo de caracteres en los controles con máscara de entrada. En efecto, el número máximo de caracteres encontrados en la control depende de la máscara de entrada definida para esta control.
<a name="NOTE0_5"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000274"></a>

**En los archivos de datos HFSQL**, el **Size** Property se aplica sólo a los elementos de texto y tiempo.

**En un reporte**, el **Size** Property sólo puede utilizarse en controles estáticos.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **En una ventana**, el **Size** Property se aplica sólo a:

- controles de edición,

- (en tablas de sólo lectura),

- Columnas de tabla,

- cuadros combinados editables.






![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **Size** Property se aplica únicamente a los siguientes elementos:

- Control Campo de entrada.

- Combo Box.

- Mesa.

- Columna de tabla.

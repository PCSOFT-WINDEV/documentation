*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Procedure** puede corresponder a:

- la propiedad **Procedure** utilizada en: 

	- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

	- una variable de tipo [rssStream](../WDLang5/1000017795.md).

	- una variable de tipo [rssCloud](../WDLang5/1000017785.md).

	- una variable de tipo [Procedure Description](../WDLang1/1000019317.md).

	- una variable de tipo [Definition](../WDLang1/1000019540.md).

	- una variable de tipo [httpProgress](../WDLang3/1000021222.md). 

	- una variable de tipo [Descripción de ParallelTask](../WDLang1/1000021586.md).




- una variable de tipo [Procedure](../Motscles/1000018907.md).

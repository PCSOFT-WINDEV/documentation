


## AdjustmentMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AdjustmentMode** Property determina y cambia cómo se ajusta un control Hoja de cálculo en un reporte.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Initialization process of the PSHEET_Statistics control
PSHEET_Statistics = "FashionShow.xlsx"
PSHEET_Statistics.AdjustmentMode = adjustementModeRowsAndColumns
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el modo de ajuste de un control Hoja de cálculo en un reporte

`<Result> = <Spreadsheet control>.AdjustmentMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de ajuste de una control Hoja de cálculo: 


|   |   |
| --- | --- |
| *adjustementModeColumns* | Todas las columnas se ajustan al ancho de la control. |
| *adjustementModeRows* | Todas las filas están ajustadas a la altura de control. |
| *adjustmentModeNone* | No se realiza ningún tipo de escalado. |
| *adjustmentModeRowsAndColumns* | La hoja de trabajo se ajusta a las dimensiones de una control Hoja de cálculo. |



**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo a utilizar. 


<a name="SYNTAX2"></a>

### Modificar el modo de ajuste de un control Hoja de cálculo en un reporte

`<Spreadsheet control>.AdjustmentMode = <New Adjustment>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo a utilizar. 

**`<New Adjustment>: (Constante de tipo Integer)`**

Nuevo modo de ajuste para el control Hoja de cálculo: 


|   |   |
| --- | --- |
| *adjustementModeColumns* | Todas las columnas se ajustan al ancho de la control. |
| *adjustementModeRows* | Todas las filas están ajustadas a la altura de control. |
| *adjustmentModeNone* | No se realiza ningún tipo de escalado. |
| *adjustmentModeRowsAndColumns* | La hoja de trabajo se ajusta a las dimensiones de una control Hoja de cálculo. |

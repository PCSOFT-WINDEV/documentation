


## SwipeEnabled (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SwipeEnabled** Property activa o desactiva el soporte del gesto de deslizamiento: 

- en una control Ventana interna. 

- en una fila en una control Looper.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IWC_InternalWin.SwipeEnabled = False
```


<a name="Example2"></a>

```wl
LOOP_Product[2].SwipeEnabled = False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir el modo de gestión del gesto de deslizamiento

`<Result> = <Control used>.SwipeEnabled`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se permite el gesto de deslizar, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**



- Nombre de la control Ventana interna utilizada. 

- Fila del control Looper utilizado. En este caso, se utiliza la siguiente anotación: 
	
	```txt
	Looper_control[Row_Number]
	```






<a name="SYNTAX2"></a>

### Modificación del modo de gestión del gesto de deslizamiento

`<Control used>.SwipeEnabled = <Swipe management>`
---

**`<Control used>: (Nombre del control)`**



- Nombre de la control Ventana interna utilizada. 

- Fila del control Looper utilizado. En este caso, se utiliza la siguiente anotación: 
	
	```txt
	Looper_control[Row_Number]
	```





**`<Swipe management>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se permite el gesto de deslizar, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Pase en una ventana interna
<a name="pase_una_ventana_interna_ELTPARAGRAPHE000067"></a>

Por Default, el gesto de deslizar está habilitado: 

- cuando la opción "Permitir cambiar de contenido al deslizar" está seleccionada en un control Ventana interna (pestaña "UI" de la ventana de descripción del control),

- cuando más de una ventana interna fue añadida por [IWListAdd](../WDLang1/1000020558.md). 


En este caso, el usuario puede modificar la ventana interna que se muestra con un simple gesto del dedo. 

El **SwipeEnabled** Property se utiliza para desactivar y volver a activar la compatibilidad con el gesto de deslizamiento. Esto le permite desactivar temporalmente el gesto de deslizamiento para realizar una acción específica. 


### Pase un control Looper
<a name="pase_control_looper_ELTPARAGRAPHE000088"></a>

El gesto de deslizamiento es activado por Default en un control Looper cuando: 

- la opción "Deslizamiento de fila: El "deslizamiento personalizable" está activado (pestaña "Detalles" de la ventana de control de description),

- se definió una ventana interna para el pase personalizado.  


En este caso, el usuario puede mostrar la ventana interna con un simple gesto táctil en la fila de la control Looper. 

El **SwipeEnabled** Property se utiliza para desactivar y volver a activar la compatibilidad con el gesto de deslizamiento. Esto le permite desactivar temporalmente el gesto de deslizamiento para realizar una acción específica.


## NextApproverID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NextApproverID** Property se utiliza para gestionar los usuarios que realizarán la siguiente acción. Esta propiedad se puede utilizar en:

- una variable de tipo [sfProcessSubmitRequest](../WDLang5/1000018516.md).

- una variable de tipo [sfProcessWorkitemRequest](../WDLang5/1000018520.md).

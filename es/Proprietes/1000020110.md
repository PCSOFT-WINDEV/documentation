


## ResourceWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ResourceWidth** Property se utiliza para get y cambiar la anchura de los recursos en un control Planificador donde los recursos están dispuestos en columnas. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_Zoom = True THEN
	SCH_MyScheduler.ResourceWidth = 200
ELSE
	SCH_MyScheduler.ResourceWidth = 100
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la amplitud de los recursos

`<Result> = <Control used>.ResourceWidth`
---

**`<Result>: (Integro)`**



- Anchura de los recursos en píxeles, si la anchura se ha modificado mediante programación (con el **ResourceWidth**) Property).

- -1 en los siguientes casos: 

	- si la anchura de los recursos se calcula automáticamente (utilizando la información dada en la ventana de control description).

	- si los recursos se visualizan en fila. 







**`<Control used>: (Nombre del control)`**

Nombre de la control Planificador utilizada.  


<a name="SYNTAX2"></a>

### Modificar el ancho de los recursos

`<Control used>.ResourceWidth = <New width>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control Planificador utilizada.

**`<New width>: (Integro)`**



- Nuevo ancho para los recursos expresados en píxeles. 

- -1 para calcular automáticamente el ancho de los recursos a partir de la información dada en la ventana de description de control.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la anchura de los recursos se define con el **ResourceWidth** Property, los recursos no pueden ser "anclados" según el tamaño disponible. 

- Sólo se puede utilizar el valor -1. Los otros valores negativos se ignoran.

- La anchura de una Resource específica puede modificarse con la [anchura](../Proprietes/2510059.md) Property utilizando la siguiente sintaxis:
	
	```txt
	Schedule_Name.Resource["MyResource"].Width = <New value>
	```




## Visible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Visible** Property se utiliza para:

- Averiguar si un elemento es Visible.

- Hacer visible un elemento o Invisible.




Este Property se aplica a la:

- controles y grupos de controles que se encuentran en una ventana, en una página o en un reporte.

- Bloques reporte,

- opciones de menú.




**Observaciones**:

- En el editor de ventanas y en el editor de páginas, la visibilidad del control se define en la pestaña "UI" del control description.

- En el editor de reportes, la visibilidad del control se define en la pestaña "Detalles" del control description.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Make the "EDT_Company" control visible if it is invisible
IF EDT_Company.Visible = False THEN
	EDT_Company.Visible = True
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si un elemento es Visible

`<Result> = <Element used>.Visible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado es visible,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar.  


<a name="SYNTAX2"></a>

### Haciendo visible un control o un bloque de reporte/Invisible

`<Element used>.Visible = <Visible/Invisible>`
---

**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar.

**`<Visible/Invisible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado debe ser visible,

- <u><u><u><u>False</u></u></u></u> si el elemento especificado debe ser Invisible.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Atributos de control
<a name="atributos_control_ELTPARAGRAPHE000113"></a>

El **Visible** Property no modifica los demás atributos de estado del mando (selección sin entrada, activa o Grayed out)
<a name="NOTE0_2"></a>


### Grupo de controles
<a name="grupo_controles_ELTPARAGRAPHE000122"></a>

Si el objeto especificado es un grupo de controles, la visibilidad de todos los controles de ese grupo cambia.
<a name="NOTE0_3"></a>

### El uso de atajos según control establece
<a name="uso_atajos_segun_control_establece_ELTPARAGRAPHE000131"></a>

No se pueden utilizar atajos con una Invisible control. Por ejemplo, un botón de Invisible no reacciona cuando se utilizan atajos.

### Controles en las pestañas
<a name="controles_las_pestanas_ELTPARAGRAPHE000140"></a>

Si un control de un panel de pestañas es visible, pero el panel no se muestra, la propiedad **Visible** devolverá <u><u><u><u>False</u></u></u></u>.
<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Filas en una control Tabla
<a name="filas_una_control_tabla_ELTPARAGRAPHE000155"></a>Un fila que se encuentra en un control Tabla puede hacerse visible o Invisible. Por ejemplo:


```wl
// Makes the second row of the Table control invisible
TABLE_Table[2].Visible = False
```
Si la control Tabla utiliza barras con colores alternados, se respetará la alternancia de colores.

Observación: Si se utiliza la propiedad **Visible** en una celda tabla, se ocultarán todas las celdas que se encuentren en la columna correspondiente.

### Límites
<a name="limites_ELTPARAGRAPHE000199"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **Visible** Property no se aplica a los informes.![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **Visible** Property no puede utilizarse en:

- un fila de una caja de lista o control Combo Box,

- un menú de ventana

- un elemento en una control TreeView.




Si se utiliza la propiedad **Visible** en una celda tabla, se ocultarán todas las celdas que se encuentren en la columna correspondiente. ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **Visible** Property se aplica únicamente a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de un control Tabla.

- Supercontrol.


 
 
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **Visible** Property no se aplica a las ventanas.


## Timeout (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Timeout** Property se utiliza para get y cambiar el tiempo máximo de espera antes de que una solicitud se agote. Esta propiedad se puede utilizar en:

- el flujo cuando se utiliza un control de videoconferencia. Para obtener más información, consulte [Timeout](../Proprietes/2510052.md).

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- una variable de tipo [wsRequest](../WDLang3/1000020577.md).

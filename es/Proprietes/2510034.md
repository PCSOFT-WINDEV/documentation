


## Image (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Image** permite identificar y cambiar la imagen de fondo asociada a un elemento.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **Imagen** Property obtiene y establece el fondo Image un control o bloque.![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **Image** Property obtiene y fija el Image:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) mostrada en la barra de Icon de una ventana matriz de MDI (sólo WINDEV),

- associated con la leyenda de un botón,

- associated con un panel de pestañas o con una pestaña,

- associated con los paneles de la barra de herramientas o con el botón de un panel de la barra de herramientas,

- associated con el título de una opción del menú,

- se utiliza como cursor en una barra de progreso o en un control deslizante (excepto los controladores redondos),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) se utiliza como marca de verificación para las casillas de verificación y los botones de radio (sólo WINDEV),

- utilizado en las opciones de casillas de verificación y botones de radio en estilo Vista,

- utilizado en la Image de fondo de una control Calendario.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilizado en una control Valoración.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilizado para los marcadores de una control Mapa.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) se utiliza para una agrupación cuando la agrupación de una cinta está cerrada.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ...usado para representar una barra de progreso de Infinite.


**Observación**: Estas imágenes se definen en el editor description de la ventana control, o Page (pestaña "General" o "Detalles").

`<Result> = <Element used>.Image`

**`<Result>: (Cadena de caracteres)`**

**`<Element used>: (Nombre de control o bloque.)`**

Nombre del elemento (control o bloque) a utilizar.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Imagen associated con una reporte control o bloque
<a name="imagen_associated_con_una_reporte_control_bloque_ELTPARAGRAPHE000275"></a>

- La imagen asociada a una control Imagen puede conocerse y/o modificarse en la editor de reportes, en la pestaña "General" del control description.

- La imagen de fondo de un control o bloque puede ser conocida y/o modificada en la editor de reportes, en la pestaña "Estilo" de la ventana de la description. Todo lo que tienes que hacer es seleccionar el elemento "Image de fondo".



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Image escaneada en una reporte
<a name="image_escaneada_una_reporte_ELTPARAGRAPHE000285"></a>

El **Image** Property no puede utilizarse para cambiar la imagen escaneada de un reporte. De hecho, la Image escaneada de una reporte está acostumbrada:

- posicionar fácilmente los diferentes mandos en la reporte,

- ver el resultado en la presentación preliminar.



<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Image en una opción de menú
<a name="image_una_opcion_menu_ELTPARAGRAPHE000299"></a>

Si una Image es associated con una opción de menú marcada, la Image reemplaza la marca.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Image de una ficha o panel de fichas
<a name="image_una_ficha_panel_fichas_ELTPARAGRAPHE000308"></a>Para modificar la Image de una pestaña (que significa la Image de los botones de la pestaña), use el siguiente código:


```wl
TAB_Tab.Image = "Example.jpg"
```
Para cambiar la Image que aparece en el botón de la pestaña (junto al texto), basta con especificar el índice de la pestaña que se desea modificar:


```wl
TAB_Tab[1].Image = "Example.jpg"
```

<a name="NOTE0_5"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Imágenes de marcas de verificación en las casillas y botones de selección
<a name="imagenes_marcas_verificacion_las_casillas_botones_seleccion_ELTPARAGRAPHE000322"></a>

El **Image** Property cambia la marca de las casillas de verificación y los botones de opción. Esta propiedad corresponde a la opción "Imagen de marcas" que se encuentra en la pestaña "Detalles" del control description.

Si la Image de la marca de verificación no debe ser animada: la Image utilizada debe contener los dos o tres estados de la marca de verificación: Verificado, Desmarcado e Indefinido si es necesario.

Si la Image de la marca de verificación está animada, la Image utilizada debe ser una Image en formato GIF que contenga las siguientes Image s para cada estado principal (Verificado, No Verificado e Indefinido):

- Marca de verificación activada (sin animación)

- Marca de verificación pulsada (animación)

- Marca de verificación Grayed (sin animación)

- Marque esta opción (animación)




Ejemplo de Image utilizado para una marca de verificación animada:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Image_Interrup.gif)
 1. Imágenes del estado comprobado.

2. Imágenes para el estado no verificado.

3. Imágenes para el estado indefinido.

### Límites
<a name="limites_ELTPARAGRAPHE000367"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **Image** Property no se puede utilizar en:

- un reporte.

- un código de barras.

- un control Casilla de verificación.


Formatos autorizados de Image: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los siguientes formatos de Image pueden ser visualizados: todos los formatos descritos en la función [fGraphicFilter](../WDLang1/3036009.md).


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **Image** Property sólo puede utilizarse en:

- una ventana MDI,

- un botón, 

- una pestaña o un panel de pestañas,

- un panel de la barra de herramientas o un botón de panel,

- una opción de menú,

- una barra de progreso, 

- un deslizador (excepto los redondos),

- una casilla de verificación,

- un botón de radio,

- a control Calendario,

- a control Valoración.

- a control Mapa Marker.

- a control Barra de progreso infinita.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Image** solo se aplica a los siguientes elementos:

- Botón.

- Ventana.

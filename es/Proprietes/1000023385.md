
## FilePath (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FilePath** Property se utiliza para identificar el nombre del archivo associated con un control. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una oficina control encontrada en una ventana (control Hoja de cálculo, control Procesador de texto, Editor de imágenes control, Lector de PDF control). Para obtener más información, consulte [FilePath](../Proprietes/1000023104.md).

- una variable de tipo [Document](../WDLang1/1000022461.md).


## NullableIndex (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NullIableIndex** determina si un campo clave acepta valores "Null". Es posible:  

- Saber si un campo clave acepta valores "Null". 

- Permitir o no el uso de valores "Null" en un campo clave (solo al definir el archivo de datos mediante programación). 




Observación: Esta propiedad equivale a la opción "Permitir valores NULL en los índices (compatible solo con la versión 28 y posteriores)" en la pestaña "Detalles" de la ventana de descripción del archivo de datos, en el editor de análisis. 

**Atención: Los archivos con la opción "Permitir valores NULL en los índices" activada solo son compatibles con las aplicaciones de la versión 28 o posterior.**
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description

// Describe the "CUSTOMER" data file
AFile.Name = "CUSTOMER"
AFile.Type = hFileNormal
AFile.NullableIndex = True

// Item description 
AnItem.Name = "Message"
AnItem.Type = hItemText
AnItem.Size = 40
HDescribeItem(AFile, AnItem)

// Validate the data file description 
HDescribeFile(AFile)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Saber si un campo clave acepta valores "Null" 

`<Result> = <Data file>.NullableIndex`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos está configurado para aceptar valores "Null" en un campo clave,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).


<a name="SYNTAX2"></a>

### Permitir o no el uso de valores "Null" en un campo clave (archivo de datos definido mediante programación)

`<Data file>.NullableIndex = <NULL value supported>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió con el tipo [File Description](../WDLang4/1514065.md).

**`<NULL value supported>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos debe permitir el uso de valores "Null" en un campo clave, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Uso de valores "Null" en campos clave
<a name="uso_valores_null_campos_clave_ELTPARAGRAPHE000071"></a>

Si las claves de un archivo de datos permiten valores "Null": 

- Puede utilizar las funciones de búsqueda con el valor "Null". 

- Varias claves del archivo de datos pueden contener el valor "Null".

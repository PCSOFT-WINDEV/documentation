
## StartPosition (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StartPosition** Property se utiliza para obtener la posición inicial de un elemento en un Document de tratamiento de textos . Esta propiedad se puede utilizar en:

- variables de tipo [docElement](../WDLang1/1000022484.md). 

- variables de tipo [docFragment](../WDLang1/1000022482.md)

- variables de tipo [docParagraph](../WDLang1/1000022483.md).

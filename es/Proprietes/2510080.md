


## Modified (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Modificado** Property se utiliza para determinar si un control o grupo de control s ha sido modificado por el usuario (entrada de teclado o ratón).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the "EDT_CustomerName" control was modified by the user,
// changes are saved in the bound data file 
IF EDT_CustomerName.Modified = True THEN
	ScreenToFile()
	HModify(Customer)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Element used>.Modified`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si:

	- la control especificada fue modificada por el usuario,

	- la opción "Almacenar el último valor introducido" fue seleccionada por el usuario en el menú contextual de la control,

	- al menos una control perteneciente al grupo especificado de control s fue modificada por el usuario,

	- al menos una control perteneciente a la ventana especificada fue modificada por el usuario.




- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element used>: (Nombre de control o ventana)`**

Nombre del elemento (control, grupo de control s o ventana) a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Celda de la tabla
<a name="celda_tabla_ELTPARAGRAPHE000086"></a>

Para determinar si una celda tabla ha sido modificada, basta con utilizar **Modified** Property en el código de salida de la columna:


```txt
IF [<Column name>].Modified THEN ...
```

<a name="NOTE0_2"></a>


### Inicialización y reinicialización del **Modified** Property
<a name="inicializacion_reinicializacion_del_docparampagetitleshort_property_ELTPARAGRAPHE000097"></a>

La propiedad **Modified** se establece en <u><u><u><u>False</u></u></u></u> cuando se abre la ventana.

La propiedad **Modified** se restablece a <u><u><u><u>False</u></u></u></u> sólo mediante programación (*&lt;Nombre de objeto&gt;.Modificado = False*).

**Consejo**: Cuando se modifica un control mediante programación, se puede manipular directamente el **Modified** Property de la ventana. Un proceso de validación puede comprobar fácilmente si se ha realizado una modificación (en este caso, no es necesario evaluar el **Modified** Property en cada ventana control).
<a name="NOTE0_3"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000126"></a>

La propiedad **Modified** no se puede utilizar en:

- a control Forma,

- una opción de un control Casilla de verificación o Botón de opción,

- un menú,

- un elemento de un control TreeView, 

- un panel de pestañas,

- una Web control Cámara,

- a control Conferencia,

- a control Splitter.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Modified** solo se aplica a los siguientes elementos:

- Botón.

- Control Campo de entrada.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Ventana.

- Mesa.

- Columna de tabla.

- Supercontrol.

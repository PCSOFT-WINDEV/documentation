


## Directory (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Directorio** Property se utiliza para administrar el directorio físico de archivos HFSQL. Esta Property tiene en cuenta los posibles cambios de directorio realizados por [HChangeDir](../WDLang4/3044168.md) o [HSubstDir](../WDLang4/3044028.md). Es posible:

- Identificar el directorio físico de un archivo de datos definido en el editor de análisis o mediante programación.

- Definir un directorio físico para un archivo de datos (cuando el archivo se describe mediante programación).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the name and full path of the Customer data file
NameAndFullPath is String
NameAndFullPath = Customer.Directory + "\" + ...
	Customer.PhysicalName + Customer.Extension
// NameAndFullPath corresponds to "C:\MyFiles\Customer.Fic"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el directorio físico de un archivo de datos 

`<Current directory> = <Data file>.Directory`
---

**`<Current directory>: (Cadena de caracteres)`**

Nombre del directorio del archivo de datos. Este nombre tiene en cuenta los posibles cambios de directorio realizados por [HChangeDir](../WDLang4/3044168.md) o [HSubstDir](../WDLang4/3044028.md).

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definición de un directorio físico para un archivo de datos descrito mediante programación

`<Data file>.Directory = <New directory>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre se definió con el tipo [Expediente description](../WDLang4/1514065.md).

**`<New directory>: (Cadena de caracteres)`**

Nombre físico del directorio.

![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) El directorio del archivo de datos debe ser relativo a la base de datos encontrada en el servidor. Por ejemplo: ".\\Data"


## Login (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Login** Property se utiliza para get y cambiar el nombre de usuario. Esta propiedad se puede utilizar en:

- una variable de tipo [sfConnection](../WDLang5/1000017579.md). 

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md).

- una variable de tipo [saasUser](../WDLang3/1000019083.md).

- una variable de tipo [gpwUser](../WDLang6/1000021088.md).

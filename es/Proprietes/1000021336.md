


## InputValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InputValue** Property se utiliza para: 

- averiguar si el usuario final puede introducir o modificar valores en un control Hoja de cálculo. 

- permitir o impedir que el usuario final introduzca o modifique valores en un control Hoja de cálculo.




Observación: Esta propiedad corresponde a la opción "Permitir entrada de valores" de la pestaña "Detalles" de la ventana control Hoja de cálculo description.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_ReadOnly THEN
	PSHEET_Spreadsheet.InputValue = True
ELSE
	PSHEET_Spreadsheet.InputValue = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el usuario puede modificar o añadir valores

`<Result> = <Spreadsheet control>.InputValue`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario puede: 

	- añadir, modificar valores y cargar un archivo XLXS,

	- insertar o borrar columnas o filas. 




- <u><u><u><u>False</u></u></u></u> si el usuario no puede:

	- añadir, modificar valores y cargar un archivo XLXS. 

	- insertar o borrar columnas o filas.







**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


<a name="SYNTAX2"></a>

### Permitiendo al usuario modificar o añadir valores

`<Spreadsheet control>.InputValue = <Authorization>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Authorization>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para permitir al usuario:

	- añadir, modificar valores y cargar un archivo XLXS. 

	- insertar o borrar columnas o filas.




- <u><u><u><u>False</u></u></u></u> para no permitir al usuario:

	- añadir, modificar valores y cargar un archivo XLXS.

	- insertar o borrar columnas o filas.









<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el [Fórmula de entrada](../Proprietes/1000021337.md) Property es set a <u><u><u><u>False</u></u></u></u> y el **InputValue** Property es set a <u><u><u><u>True</u></u></u></u>, el usuario final podrá introducir y modificar valores en el control Hoja de cálculo.

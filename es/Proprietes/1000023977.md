
## Token

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Token** puede corresponder a:

- la propiedad [Token](../Proprietes/1000023976.md) que se utiliza para manipular los diferentes tokens asociados a un "Texto con tokens" control Campo de entrada.

- una variable [Token](../WDLang1/1000023942.md) que se utiliza para manipular un Token asociado a un "Texto con Token s" control Campo de entrada.

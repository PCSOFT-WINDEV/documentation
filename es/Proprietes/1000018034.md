
## Modified (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Modificado** Property permite saber si un elemento ha sido modificado. Esta propiedad se puede utilizar en:

- los controles de una ventana. Para obtener más información, consulte [Modificado en los controles de una ventana](../Proprietes/2510080.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Document](../WDLang1/1000022461.md).

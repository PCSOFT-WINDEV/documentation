


## CtDescribedOLEDBProvider (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtDescribedOLEDBProvider** Property se utiliza para obtener y set el nombre del proveedor OLE DB (también llamado servidor OLE DB). Es posible:

- Averiguar el nombre del proveedor de OLE DB utilizado para la conexión a un tabla de OLE DB.. Este proveedor se definió en el editor de análisis o cuando se describió el el archivo de datos a través de la programación.
	Atención: El **CtDescribedOLEDBProvider** Property ignora cualquier cambio realizado con [HConnect](../WDLang4/3044263.md).
	Este Property puede utilizarse para el un archivo de datos definido en el editor de análisis o mediante programación.

- Definir el nombre del proveedor OLE DB utilizado para la conexión a un tabla por OLE DB (cuando el archivo de datos se describió mediante programación). Esta Property es equivalente a la [CtOLEDBProvider](../Proprietes/2512033.md) Property. 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describe el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the name of the OLE DB provider
Customer.CtDescribedOLEDBProvider = hOledbOracle
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el proveedor de OLE DB utilizado

`<OLE DB provider> = <Data file>.CtDescribedOLEDBProvider`
---

**`<OLE DB provider>: (Cadena de caracteres)`**

Nombre del proveedor de OLE DB o una de las siguientes constantes:


|   |   |
| --- | --- |
| *hAccessHF7* | Pseudo-conexión a una base de datos HFSQL Classic. |
| *hAccessHFClientServer* | Conector Nativo a una base de datos HFSQL Client/Server. |
| *hNativeAccessAS400* | Conector Nativo AS/400 (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessDB2* | Conector Nativo DB2 (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessInformix* | Conector Nativo Informix (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessMariaDB* | Conector Nativo MariaDB (módulo opcional de WINDEV/WEBDEV) |
| *hNativeAccessMySQL* | Conector Nativo MySQL (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessOracle* | Conector Nativo Oracle (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessOracleLite* | Conector Nativo Oracle (módulo opcional de WINDEV Mobile). |
| *hNativeAccessPostgreSQL* | Conector Nativo PostgreSQL (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessProgress* | Conector Nativo Progress (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessSQLAzure* | Native SQL Azure Connector (módulo opcional de WINDEV/WEBDEV, suministrado con Conector Nativo SQL Server). |
| *hNativeAccessSQLite* | Conector SQLite nativo (suministrado con WINDEV/WEBDEV). |
| *hNativeAccessSQLServer* | Conector Nativo SQL Server (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessSQLServerMobile* | Conector Nativo SQL Server (módulo opcional de WINDEV Mobile). |
| *hNativeAccessSybase* | Conector Nativo Sybase (módulo opcional de WINDEV/WEBDEV). |
| *hNativeAccessXBase* | Conector nativo xBase (proporcionado con WINDEV/WEBDEV). |
| *hNativeAccessXML* | Conector XML nativo (suministrado con WINDEV/WEBDEV). |
| *hODBC* | Proveedor OLE DB para ODBC (permite acceder a una fuente de datos ODBC declarada en las fuentes de datos ODBC de Windows). |
| *hOledbAccess97* | Proveedor OLE DB para Access 97. |
| *hOledbAccess2000* | Proveedor OLE DB para Access 2000. |
| *hOledbAccess2007* | Proveedor OLE DB para Access 2007. |
| *hOledbAccess2010* | Proveedor OLE DB para Access 2010. |
| *hOledbDBase5* | Proveedor OLE DB para dBase 5. |
| *hOledbExcel2007* | Proveedor OLE DB para Excel 2007. |
| *hOledbExcel2000* | Proveedor OLE DB para Excel 2000. |
| *hOledbExcel97* | Proveedor OLE DB para Excel 97. |
| *hOledbLotus4* | Proveedor OLE DB para Lotus 4. |
| *hOledbOracle* | Proveedor OLE DB para Oracle. |
| *hOledbSQLServer* | Proveedor OLE DB para SQL Server. |


**Atención**: Para utilizar una conexión OLE DB, es necesario:

- instalar MDAC versión 2.6 o más reciente (instalación realizada por WINDEV y al crear la versión cliente),

- instalar el proveedor OLE DB que corresponde a la base de datos utilizada.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

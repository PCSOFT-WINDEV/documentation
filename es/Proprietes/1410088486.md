
## spAge (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spAge** permite obtener el tiempo transcurrido desde la última vez que se cambió la contraseña. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the last password change in the Authentication file
PasswordAge is Duration
PasswordAge = Authentication.PasswordItem.spAge
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Time elapsed> = <Data file>.<Item>.spAge`
---

**`<Time elapsed>: (Duración)`**

Tiempo transcurrido desde la última vez que se cambió la contraseña. 

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo de tipo "Contraseña" manipulado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).

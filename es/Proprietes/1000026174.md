
## MandatoryInput (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MandatoryInput** Property se utiliza para determinar si se requiere un control y para activar o desactivar la entrada de datos requerida. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF m_bCriticalAccount THEN
	EDT_Password.MandatoryInput = True
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si la entrada de datos requerida está activada o desactivada en un control

`<Result> = <Control used>.MandatoryInput`
---

**`<Result>: ()`**



- <u><u><u><u>True</u></u></u></u> si se requiere la entrada. En este caso, la control arrojará errores en los siguientes casos: 

	- Control Campo de entrada: Error si la control está vacía. 

	- Control Combo Box con entrada: Error si la control está vacía. 

	- control Combo Box no editable: Error si no se selecciona fila. 

	- Control Casilla de verificación: Error si no se marca ninguna opción. 

	- Control Botón de opción: Error si no se marca ninguna opción. 

	- Control Valoración: control marcada como con errores si no aparecen estrellas. 




- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- a control Campo de entrada,

- a control Casilla de verificación,

- a control Botón de opción,

- a control Combo Box,

- el control Valoración. 





<a name="SYNTAX2"></a>

### Activación o desactivación de la entrada requerida en una control

`<Control used>.MandatoryInput = <Required input>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- a control Campo de entrada,

- a control Casilla de verificación,

- a control Botón de opción,

- a control Combo Box,

- el control Valoración. 




**`<Required input>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se requiere la entrada. En este caso, la control arrojará errores en los siguientes casos: 

	- Control Campo de entrada: Error si la control está vacía. 

	- Control Combo Box con entrada: Error si la control está vacía. 

	- control Combo Box no editable: Error si no se selecciona fila. 

	- Control Casilla de verificación: Error si no se marca ninguna opción. 

	- Control Botón de opción: Error si no se marca ninguna opción. 

	- Control Valoración: control marcada como con errores si no aparecen estrellas. 




- <u><u><u><u>False</u></u></u></u> en caso contrario.

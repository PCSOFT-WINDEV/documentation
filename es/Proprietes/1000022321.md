


## FolderImageWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FolderImageWidth** Property se utiliza para get y cambiar el ancho de la carpeta Image en los controles TreeView y TreeView Table.

Esta Property corresponde a la opción: 

- "Ancho de la imagen de la carpeta" en la pestaña "Detalles" de la ventana control TreeView description. 

- "Ancho" de la sección "Imagen de las carpetas" que se encuentra en la pestaña "Detalles" de la ventana de description de los controles de la tabla TreeView. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the display of folders
TREE_MyTreeView.FolderImageWidth = 0

// Restore the display of folders
TREE_MyTreeView.FolderImageWidth = -1 // Default value
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el ancho de la Image correspondiente a la carpeta

`<Current width> = <Control used>.FolderImageWidth`
---

**`<Current width>: (Integro)`**



- Anchura actual de la Image mostrada (expresada en píxeles),

- 0 si no se muestra la Image,

- -1 si la Image se muestra con su anchura inicial. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView, 

- Control Tabla TreeView. 





<a name="SYNTAX2"></a>

### Modificando el ancho de la Image correspondiente a la carpeta

`<Control used>.FolderImageWidth = <New width>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView,

- Control Tabla TreeView. 




**`<New width>: (Integro)`**

Nueva anchura para la Image mostrada (expresada en píxeles)

Si este parámetro es set a: 

- 0 si no se muestra la Image,

- -1 si la Image se muestra con su anchura inicial.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar el ancho de la Image mostrada para un fila específico en un TreeView o control Tabla TreeView, use la siguiente sintaxis:

```txt
<Control_Name>[Row_Number].FolderImageWidth = Value
```

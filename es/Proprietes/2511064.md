


## HeightInPixel (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HeightInPixel** Property se utiliza para get la altura en píxeles de un control en un reporte.

**Observación**: La frontera de la control está incluida en su altura. El [Altura](../Proprietes/2510050.md) Property se utiliza para get y cambiar la altura de un control en milímetros.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the height of "ITEM_CustomerName"
ResHeight = ITEM_CustomerName.HeightInPixel
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Control used>.HeightInPixel`
---

**`<Result>: (Real)`**

Altura (en píxeles) de la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre de reporte control a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Altura
<a name="altura_ELTPARAGRAPHE000054"></a>

Una control debe estar totalmente contenida en el bloque al que pertenece.
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000061"></a>

El **HeightInPixel** Property no puede utilizarse en:

- una reporte entera,

- un bloque reporte,

- un grupo de controles,

- a control Casilla de verificación,

- a control Campo de entrada,

- un control Agenda, 

- un control Planificador, 

- a control Diagrama de Gantt,

- a control Firma.




## BackgroundColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **BackgroundColor** permite obtener y cambiar el color de fondo de: 

- un control.

- una ventana. 

- una página.

- un bloque de reporte.

- un texto en una opción de menú.




**Observación**: Los efectos de la propiedad **BackgroundColor** dependen del tipo de control (para saber más, consulte el párrafo "Observaciones").


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The background color is modified according to the displayed value
IF EDT_Total > 15000 THEN
	EDT_Total.BackgroundColor = LightRed
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar el color de fondo

`<Background color> = <Element used>.BackgroundColor`
---

**`<Background color>: (Entero o constante)`**

Color del elemento especificado. Corresponde a una de las siguientes opciones:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido](../WDLang5/3010002.md).




**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar: control, ventana, página o bloque de reporte.  


<a name="SYNTAX2"></a>

### Cambiar el color de fondo

`<Element used>.BackgroundColor = <New color>`
---

**`<Element used>: (Tipo de elemento)`**

Nombre del elemento a utilizar: control, ventana, página o bloque de reporte.

**`<New color>: (Entero o constante)`**

Nuevo color para el elemento especificado. Corresponde a una de las siguientes opciones:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido](../WDLang5/3010002.md).

### Color de fondo de un control Tabla y de sus celdas, columnas o filas
<a name="color_fondo_control_tabla_sus_celdas_columnas_filas_ELTPARAGRAPHE000262"></a>

Al utilizarse en un control Tabla, la propiedad **BackgroundColor** permite:

- Eliminar los colores específicos de las celdas y columnas mediante la siguiente sintaxis:
	
	```txt
	<Table control>.BackgroundColor = DefaultColor
	```


- Cambiar el color de fondo de todas las columnas del control Tabla (especialmente columnas y celdas sin color de fondo). Por ejemplo, el siguiente código aplica el color azul a todas las columnas:
	
	```txt
	<Table control>.BackgroundColor = DarkBlue
	```





Esta propiedad obtiene y establece el color de fondo de una celda en un control Tabla. Se debe utilizar la siguiente sintaxis para especificar la celda: 

```txt
<Table control>[RowNum][ColumnNum]
```


Ejemplo: El siguiente código cambia el color de fondo de la celda en la primera fila de la segunda columna.


```wl
TABLE_Table1[1][2].BackgroundColor = DarkBlue
```
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=table.gif)
**Atención**: Para conservar el color de fondo al seleccionar una fila de un control Tabla basado en un archivo de datos, utilice la propiedad **BackgroundColor** en el proceso "Selección de una fila".
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Para cambiar el estilo del color de fondo de una celda, columna o fila en un control Tabla o Tabla TreeView, utilice la propiedad [BackgroundStyle](../Proprietes/2513019.md).

### Color de fondo de un control Pestaña
<a name="color_fondo_control_pestana_ELTPARAGRAPHE000322"></a>

Para los controles Pestaña:

- Para las pestañas: 

	- La propiedad [Color](../Proprietes/2510071.md) obtiene y establece el color del texto de las pestañas.

	- Cuando se utiliza en las pestañas, la propiedad **BackgroundColor** obtiene y establece el color de fondo de las pestañas. 




- Para el control Pestaña: La propiedad **BackgroundColor** obtiene y establece el color de fondo del control Pestaña (color utilizado para las pestañas, los paneles de pestañas y el área detrás de las pestañas).

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000367"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Limitaciones en los controles de un reporte**
La propiedad **BackgroundColor** no tiene ningún efecto en:

- los reportes (en su totalidad).

- los controles Reporte interno.

- los controles Firma.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Limitaciones en los controles de ventanas WINDEV**
La propiedad **BackgroundColor** no tiene efecto en los siguientes controles:

- controles HTML,

- controles Cámara Web,

- controles deslizantes. 


 
 
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **BackgroundColor** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- Control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de control Tabla.

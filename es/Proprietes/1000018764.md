
## ServerAddress (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ServerAddress** Property obtiene y establece el servidor Address. Esta propiedad se puede utilizar en:

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).

- una variable de tipo [emailPOP3Session](../WDLang3/1000018759.md). 

- una variable de tipo [emailSMTPSession](../WDLang3/1000018765.md). 

- una variable de tipo [WBAdminAuth](../WDLang2/1410089328.md).




## StartVisibleRange (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StartVisibleRange** Property se utiliza para: 

- obtener la primera fecha u hora de Visible: 

	- en un control Planificador.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Línea de tiempo. 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) cambiar la primera hora de Visible en un control Línea de tiempo. 




Observación: La primera fecha u hora de Visible corresponde a la posición de la barra de desplazamiento en el control. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// TimeLine control on 2 minutes
TimelineChangeMode(TL_TimeLine, schModeSecond, 120)

// equivalent to:
TL_TimeLine.StartTotalRange = 0  // starts from 0 sec
TL_TimeLine.EndTotalRange = 120 // end at 120 sec 

// Make the first 30 seconds visible 
TL_TimeLine.StartVisibleRange = 0 // starts at 0 sec
TL_TimeLine.EndVisibleRange = 30 // size at 30 sec
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener la primera fecha u hora de Visible

`<Result> = <Control used>.StartVisibleRange`
---

**`<Result>: (Número entero o cadena de caracteres en formato Fecha/Hora)`**

Primer valor de Visible en el control. Este valor corresponde a: 

- un número entero para: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Línea de tiempo,

	- una control Planificador en modo segundo, milisegundo o microsegundo. 




- una fecha en formato YYYYMMDDDHHHSSCC para una control Planificador en modo día, semana o mes.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Planificador. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Línea de tiempo. 





<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Cambio del valor de la primera hora de Visible

`<TimeLine control>.StartVisibleRange = <New value>`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo a utilizar.  

**`<New value>: (Integro)`**

El primer valor que se hizo Visible en el control Línea de tiempo



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Todos los eventos representados en la control están entre [StartTotalRange](../Proprietes/1000020392.md) y [EndTotalRange](../Proprietes/1000020393.md). Esto corresponde al alcance total de la barra de desplazamiento o del deslizador de alcance associated con el control. 

- Todos los eventos Visible en la control están entre **StartVisibleRange** y [EndVisibleRange](../Proprietes/1000020395.md). Corresponde al tamaño de la pista de la barra de desplazamiento o del deslizador de rango associated con el control.

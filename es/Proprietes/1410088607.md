
## ImageInitialHeight (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ImageInitialHeight** permite obtener o establecer la altura predeterminada de una imagen en un control Editor de imágenes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IE_MyImageEditor.ImageInitialWidth = 800
IE_MyImageEditor.ImageInitialHeight = 600
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener la altura predeterminada de una imagen en el control Editor de imágenes

`<Result> = <Image Editor control>.ImageInitialHeight`
---

**`<Result>: (Entero)`**

Altura predeterminada (en pixeles) de una imagen en el control Editor de imágenes. 

**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.


<a name="SYNTAX2"></a>

### Establecer la altura predeterminada de una imagen en el control Editor de imágenes

`<Image Editor control>.ImageInitialHeight = <New height>`
---

**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.

**`<New height>: (Entero)`**

Nueva altura predeterminada (en pixeles) de una imagen en el control Editor de imágenes.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para cambiar las características predeterminadas del control Editor de imágenes, también es posible utilizar: 

- la propiedad [ImageInitialWidth](../Proprietes/1410088608.md) para establecer el ancho predeterminado de la imagen creada en el control Editor de imágenes.

- la propiedad [InitialTool](../Proprietes/1410088609.md) para establecer la herramienta predeterminada del control Editor de imágenes.




## NullIfEmpty (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NullIfEmpty** Property se utiliza para: 

- Averigua si el valor devuelto por la control es NULO si su contenido está vacío. 

- Configura el valor devuelto por la control si su contenido está vacío.




Observación: Esta propiedad equivale a la opción "Devolver NULL si está vacía" que se encuentra en la ventana de description de control (pestaña "Detalles").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_MinPrice.NullIfEmpty = True
EDT_MaxPrice.NullIfEmpty = True
REQ_Find.ParamMinPrice = EDT_MinPrice
REQ_Find.ParamMaxPrice = EDT_MaxPrice
HExecuteQuery(QRY_Find)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averigua si el valor devuelto por la control es NULO si su contenido está vacío

`<Result> = <Control used>.NullIfEmpty`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor devuelto por el control es NULL si el contenido está vacío,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a un control Campo de entrada o a un control Combo Box editable. 


<a name="SYNTAX2"></a>

### Configura el valor devuelto por la control si su contenido está vacío.

`<Control used>.NullIfEmpty = <New management mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a un control Campo de entrada o a un control Combo Box editable. 

**`<New management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor devuelto por el control es NULL si el contenido está vacío,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- este Property permite realizar fácilmente búsquedas multicriterio a través de una consulta con parámetros en los que los valores de los parámetros corresponden a los controles de edición. De hecho, los parámetros de consulta cuyo valor es NULL se ignoran. Para obtener más información, consulte [Uso de una consulta con parámetros](../Editeurs/2032032.md).  

- Este Property no tiene ningún efecto cuando se asigna un valor al control.

- Este Property está disponible para los controles de edición y para los controles editables de Combo Box.


## Year (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Year** permite:

- Obtener el año de una variable o campo de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md).

- Cambiar el año en una variable de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md).




Observación: La propiedad **Year** permite cambiar fechas fácilmente: agregar un año, etc.

**Recordatorio**: Los campos y variables de tipo Date permiten gestionar:

- fechas simples: "Año - Mes - Día" (formato AAAAMMDD).

- fechas y horas: "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" (formato AAAAMMDDHHmmSSCCC).

- duraciones: "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" (formato +DHHMMSSCCC).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartDate is Date = DateSys()
// Add 1 year to the date
StartDate.Year++
// Modify the years
StartDate.Year = 2003
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el año de una variable o campo de tipo Date o DateTime

`<Result> = <Date>.Year`
---

**`<Result>: (Entero)`**

Año de 4 dígitos.

**`<Date>: (Date o DateTime)`**

Nombre de la variable o campo de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md) que se va a utilizar. 




<a name="SYNTAX2"></a>

### Cambiar el año de una variable o campo de tipo Date o DateTime

`<Date>.Year = <New year>`
---

**`<Date>: (Date o DateTime)`**

Nombre de la variable o campo de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md) que se va a utilizar



**`<New year>: (Entero o cadena de caracteres)`**

Nuevo año en cifras (entre 1 y 9999). Reemplaza el año de la fecha especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Operadores disponibles para los años
<a name="operadores_disponibles_para_los_anos_ELTPARAGRAPHE000149"></a>

Los siguientes operadores aritméticos se pueden utilizar con la propiedad **Year**:

- ++ y --

- += y -=





```wl
StartDate is Date = "20011212"
StartDate.Year++  // Add 1 year to the date
StartDate.Year+=5 // Add 5 years to the date
StartDate.Year-=5 // Subtract 5 years to the date
```

<a name="NOTE0_2"></a>


### Años bisiestos
<a name="anos_bisiestos_ELTPARAGRAPHE000163"></a>

Si la fecha manipulada corresponde al 29 de febrero de un año bisiesto, al sumar o restar un año también se modifica el día. Por ejemplo:


```wl
StartDate is Date = "19960229"
// Add 1 year to the date
StartDate.Year++ 
// StartDate is "19970228"
```

<a name="NOTE0_3"></a>


### Cálculos de fechas
<a name="calculos_fechas_ELTPARAGRAPHE000172"></a>

El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999.

Los cálculos de fechas realizados con las [funciones WLanguage](../WDLang1/3027029.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.

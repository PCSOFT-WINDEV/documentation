
## RoughHeader (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RoughHeader** se utiliza para get la cabecera devuelta con la respuesta. Esta propiedad se puede utilizar en:

- una variable de tipo [httpResponse](../WDLang3/1000021165.md). 

- una variable de tipo [restResponse](../WDLang3/1000021477.md).




## DeleteButton (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **DeleteButton** se utiliza para determinar si debe haber un botón de borrado automatic en un texto de edición control, y para cambiar su posición. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables the delete button on EDT_Edit1
// The button is always visible
EDT_Edit1.DeleteButton = dbAlwaysVisible
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la posición del botón de borrado en la control Campo de entrada

`<Result> = <Edit control>.DeleteButton`
---

**`<Result>: (Constante de tipo Integer)`**

Posición del botón de borrado: 



|   |   |
| --- | --- |
| *dbAlwaysVisible* | El botón de borrado siempre se muestra. |
| *dbInEdit* | El botón de borrar se muestra cuando el control Campo de entrada está en modo de edición.. |
| *dbNone* | El botón de borrado no se muestra. |
| *dbOffEdit* | El botón de borrar se muestra cuando el control Campo de entrada no está en modo edición. |



**`<Edit control>: (Nombre del control)`**

Texto control Campo de entrada a utilizar. 


<a name="SYNTAX2"></a>

### Modificando la posición del botón de borrado en la control Campo de entrada

`<Edit control>.DeleteButton = <Delete button>`
---

**`<Edit control>: (Nombre del control)`**

Texto control Campo de entrada a utilizar. 

**`<Delete button>: (Constante de tipo Integer)`**

Posición del botón de borrado: 



|   |   |
| --- | --- |
| *dbAlwaysVisible* | El botón de borrado siempre se muestra. |
| *dbInEdit* | El botón de borrar se muestra cuando el control Campo de entrada está en modo de edición.. |
| *dbNone* | El botón de borrado no se muestra. |
| *dbOffEdit* | El botón de borrar se muestra cuando el control Campo de entrada no está en modo edición. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El botón de borrado sólo está disponible para los controles de edición de texto. No se puede utilizar en controles de edición multilínea, numéricos, de fecha y hora, etc.

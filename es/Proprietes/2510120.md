


## AnchorRateWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AnchorRateWidth** Property se utiliza para:

- Averigua la tasa de anclaje cuando un control (o una columna de tabla) se estira en el eje horizontal.

- Modificar la tasa de anclaje cuando una control (o una columna tabla) se estira en el eje horizontal.




La tasa de anclaje se utiliza para definir cómo se desplaza o redimensiona la control en relación con la evolución del tamaño de la ventana (o reporte) a la que está anclada. La tasa de anclaje corresponde al porcentaje en que se mueve el control en relación con la ventana (o reporte). Por lo tanto, si la tasa de anclaje es set a 0, el control no se redimensionará. Si es set a 1000, el control se redimensionará normalmente.

La tasa de anclaje horizontal (**AnchorRateWidth** Property) se aplica si el control está anclado horizontalmente (*anchorWidth* constante utilizada con el [Ancla](../Proprietes/2510001.md) Property).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The List Box control is enlarged by 50% of the new window size (500 for 1000).
LIST_ListPersons.AnchorRateWidth = 500
```

## Sintaxis

### Obtener la tasa de anclaje al ampliar una control horizontalmente

`<Result> = <Control used>.AnchorRateWidth`
---

**`<Result>: (Integro)`**

Tasa de anclaje incluida entre 0 (control sin estirar) y 1000 (ancla estándar)

**`<Control used>: (Nombre del control)`**

Nombre de control (o columna) a utilizar. Este control puede ser un control: 

- de una ventana.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) de una reporte.





<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
### Cambio de la tasa de anclaje al ampliar una control horizontalmente

`<Control used>.AnchorRateWidth = <Anchor rate>`
---

**`<Control used>: (Nombre del control)`**

Nombre de control (o columna) a utilizar.

**`<Anchor rate>: (Integro)`**

Nueva tasa de anclaje incluida entre 0 (control sin estirar) y 1000 (anclaje estándar)

## Observaciones


### Anclar un control
<a name="anclar_control_ELTPARAGRAPHE000109"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En una ventana, el [Ancla](../Proprietes/2510001.md) Property se utiliza para determinar cómo se redimensiona o se mueve un control cuando se redimensiona la ventana. Si el control es associated con un control Pestaña, el ancla se define en relación con la pestaña.
	Para obtener más información, consulte [Anclaje de controles en una ventana](../WDChamp/2026002.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El [Ancla](../Proprietes/2510001.md) Property no está disponible para las columnas de los controles de tabla o de tabla TreeView. Los anclajes de una columna se pueden especificar directamente con el botón **AnchorRateWidth** Property.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el [Ancla](../Proprietes/2510001.md) Property se utiliza para saber cómo se redimensiona o se mueve el control cuando se redimensiona el reporte. Este comportamiento fue definido en la editor de reportes. 
	Para obtener más información, consulte [Controles de anclaje en un reporte](../WDChamp/1013298.md). 



<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000147"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **AnchorRateWidth** Property se aplica a todos los controles de una ventana.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **AnchorRateWidth** Property se aplica a todos los controles de un reporte. En este caso, sólo se puede averiguar la tasa de anclaje utilizada.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **AnchorRateWidth** Property se aplica únicamente a los siguientes elementos:

- Botón.

- Control Estático.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Mesa.

- Supercontrol.

- Grupo de cintas.




## NoteTitle (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NoteTitle** permite:

- get el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte. 

- Modificar el título de las notas utilizadas en la documentación técnica de un control, ventana, página o reporte.




A menudo, esta Property permite al desarrollador almacenar información sin afectar a la ejecución de la aplicación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the "EDT_Prospect" control with the title of the notes
EDT_Prospect = EDT_Prospect.NoteTitle
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el título de las notas

`<Result> = <Element used>.NoteTitle`
---

**`<Result>: (Cadena de caracteres)`**

Título de las notas utilizadas en la documentación técnica del elemento especificado.

**`<Element used>: (control, ventana, página o nombre de reporte)`**

Nombre del elemento (control, ventana, página o reporte) a utilizar.  


<a name="SYNTAX2"></a>

### Cambiar el título de las notas

`<Element used>.NoteTitle = <New title>`
---

**`<Element used>: (control, ventana, página o nombre de reporte)`**

Nombre del elemento (control, ventana, página o reporte) a utilizar. 

**`<New title>: (Cadena de caracteres)`**

Nuevo título de las notas utilizadas en la documentación técnica del elemento especificado.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000090"></a>

- El título de las notas utilizadas en la documentación técnica de un control, ventana o página se define en la pestaña "Nota" del control, ventana, página o reporte description. Esta control permite describir brevemente el funcionamiento del elemento actual.

- También puede asociar una nota a la control. Esta nota también se utilizará en la documentación técnica. El [Nota](../Proprietes/2510085.md) Property se utiliza para cambiar las notas.





### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000099"></a>

En un reporte, el **NoteTitle** Property se aplica a todos los controles, bloques e informes.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **NoteTitle** Property no se puede utilizar en:

- a control Forma,

- una opción en una casilla de verificación o control Botón de opción,

- un fila en un control Tabla o un fila en un List Box o un control Combo Box,

- a menu,

- un panel de pestañas,

- un grupo de controles,

- un elemento en una control TreeView.

- un elemento de un Listview control.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **NoteTitle** sólo se aplica a los controles Combo Box en esta versión.




## BackgroundImage (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **..BackgroundImage** permite conocer y modificar la imagen de fondo de un elemento: ventana, página, control o reporte. 

**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo de una ventana o de un control se define en la ventana de descripción del elemento (pestaña "General" o "Imagen").

- La imagen de fondo de una reporte se define en la descripción de la reporte (pestaña "Fondo"). 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Modify the background image of "WIN_EditCustomer" window
WIN_EditCustomer..BackgroundImage = "C:\MyImages\Image.bmp"
```

```wl
// Before printing Page_header
// Modify the background image of report
// Use a different page found in a PDF
MyReport..BackgroundImagePrinted = True
MyReport..BackgroundImage = PDFExtractPage("Form.pdf", 2)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la imagen de fondo de un control, ventana o reporte

`<Current image> = <Element used>..BackgroundImage`
---

**`<Current image>: (Cadena de caracteres)`**

Nombre y ruta de la imagen de fondo. Esta imagen de fondo puede corresponder a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada a una ventana (excepto para las ventanas padre MDI).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada al área MDI de una ventana principal de MDI.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen asociada al borde de un botón.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada a una barra de progreso o deslizador.

- La imagen de fondo de una reporte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo de una control Forma.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo de un control de notas reposicionables.




**`<Element used>: (Nombre del control, ventana, página o reporte)`**

Nombre del elemento utilizado.  


<a name="SYNTAX2"></a>

### Modificar la imagen de fondo de un control, ventana o reporte

`<Element used>..BackgroundImage = <New image>`
---

**`<Element used>: (Nombre del control, ventana, página o reporte)`**

Nombre del elemento utilizado.

**`<New image>: (Cadena de caracteres)`**

Nombre y ruta (completo, relativo o UNC) de la nueva imagen de fondo. Esta imagen puede corresponder a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada a una ventana (excepto para las ventanas padre MDI).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada al área MDI de una ventana principal de MDI.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen asociada al borde de un botón.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo asociada a una barra de progreso o deslizador.

- La imagen de fondo de una reporte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo de una control Forma.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen de fondo de un control de notas reposicionables.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modificar la imagen
<a name="modificar_imagen_ELTPARAGRAPHE000166"></a>

Al cambiar la imagen, ésta se busca en la biblioteca de aplicaciones (archivo WDL) y luego en el disco en la ubicación especificada.

Observación: Si se intenta sustituir la imagen que se encuentra en la BDM por una imagen con el mismo nombre que se encuentra en el disco, no se realiza el cambio de imagen.
<a name="NOTE0_2"></a>


### Modificar la imagen de fondo en una reporte
<a name="modificar_imagen_fondo_una_reporte_ELTPARAGRAPHE000177"></a>

La propiedad **..BackgroundImage** está disponible en todos los tipos de reportes. **..FondoImagen** debe ser llamada antes de imprimir la página, por ejemplo en el caso "Antes de imprimir el bloque de encabezamiento de la página".

**En un reporte basado en un formulario**: 

- la propiedad **..BackgroundImage** reemplaza la imagen de fondo seleccionada en edición. Todos los tipos de imágenes soportadas en los informes están disponibles (PDF, SVG, JPG, PNG, etc.). 

- la imagen se guarda por página: por lo tanto, se puede especificar una imagen de fondo diferente para cada página impresa.

- cuando se asigna la propiedad, la nueva imagen de fondo se imprimirá **desde la página siguiente salta**.




**En un reporte no basado en un formulario**: 

- la imagen se dibuja en "homotético".

- esta propiedad tiene un efecto en la impresión real, en la vista previa de impresión y en la exportación de PDF. No tiene ningún efecto para los otros tipos de exportación (RTF, XLS, HTML, XML).




Observación: Se ignora la gestión de los márgenes especificados en la pestaña "Fondo" de la descripción de reporte ("Ignorar márgenes físicos de la impresora"): los márgenes físicos se ignoran sistemáticamente. 
<a name="NOTE0_3"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000201"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, la propiedad **..BackgroundImage** solo se aplica a los:

- windows, 

- botones,

- barras de progreso, 

- deslizadores, 

- reportes.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de forma en una ventana.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de notas reposicionables en una ventana.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **..BackgroundImage** solo se aplica a los siguientes elementos:

- Button.

- Ventana.

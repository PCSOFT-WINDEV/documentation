
## TimeZone (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TimeZone** Property se utiliza para set la zona horaria. Esta propiedad se puede utilizar en:

- una variable de tipo [gglCalendar](../WDLang5/1000017418.md). 

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md). 

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

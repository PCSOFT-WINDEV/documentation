
## URI

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave URI puede corresponder a: 

- una variable de tipo [URI](../WDLang3/1000024027.md). 

- el URI Property, utilizado para gestionar una ubicación en un archivo XML. Esta propiedad se puede utilizar en:

	- una variable de tipo [xmlNamespace](../WDLang5/1000018803.md). 

	- una variable de tipo [xmlSignatureReference](../WDLang3/1000020746.md).

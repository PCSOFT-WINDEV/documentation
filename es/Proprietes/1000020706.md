


## ProgressBar (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ProgressBar** Property se utiliza para identificar y cambiar el control Barra de progreso utilizado al calcular un control Tabla dinámica (funciones [PVTCcalcularTodos](../WDLang1/1000020539.md) y [PVTCalcularActualizar](../WDLang1/1000020534.md)). 
<a name="sample_code"></a>

## Ejemplo


```wl
PVT_Sales.ProgressBar = PROGBAR_PVTSales.Name
PVTCalculateAll(PVT_Sales)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificación de la barra de progreso utilizada para calcular una control Tabla dinámica

`<Result> = <Pivot Table control>.ProgressBar`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la control Barra de progreso utilizada. 

- Cadena vacía ("") si se usa la barra de progreso de automatic del control Tabla dinámica. 




**`<Pivot Table control>: (Nombre del control)`**

Nombre de control Tabla dinámica. 


<a name="SYNTAX2"></a>

### Cambio de la barra de progreso utilizada para el cálculo de una control Tabla dinámica

`<Pivot Table control>.ProgressBar = <New Progress Bar>`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.  

**`<New Progress Bar>: (Cadena de caracteres)`**

Nuevo nombre de la barra de progreso a utilizar. Este nombre puede corresponder a: 

- el nombre de un control Barra de progreso ([denominar](../Proprietes/2510082.md) Property utilizado en el control Barra de progreso). 

- una cadena vacía ("") si se debe utilizar la barra de progreso automática del control Tabla dinámica. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las barras de estado de la barra de progreso no se pueden utilizar. 

- Si el control Barra de progreso es Invisible, se hace visible automáticamente durante el cálculo y se convierte en Invisible al final del cálculo..


## IgnoreError (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **IgnoreError** obtiene y establece los diferentes errores de certificado ignorados por el control de visualización HTML y el control Página WEBDEV.

Esta propiedad permite ejecutar sitios web con certificados no conformes.
**Atención**: El uso de esta propiedad puede presentar riesgos de seguridad y solo debe utilizarse para mostrar sitios web legítimos en el control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTM_WebPage.IgnoreError = httpIgnoreExpiredCertificate + httpIgnoreInvalidCertificate
HTM_WebPage = "https://www.mysitewithaninvalidcertificate.com/login"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener los diferentes errores ignorados por el control de visualización HTML y el control Página WEBDEV

`<Result> = <Control used>.IgnoreError`
---

**`<Result>: (Constante o combinación de constantes de tipo Integer)`**

Errores de certificado ignorados: 


|   |   |
| --- | --- |
| *httpIgnoreExpiredCertificate* | La fecha del certificado se ignora |
| *httpIgnoreInvalidCertificate* | El certificado se ignora. |
| *httpIgnoreInvalidCertificateName* | El nombre del sitio especificado en el certificado se ignora. |
| *httpIgnoreRedirectToHTTP* | Se permite la redirección a un servidor no seguro. |



**`<Control used>: (Nombre del control)`**

Nombre del control de visualización HTML o control Página WEBDEV a manipular. 


<a name="SYNTAX2"></a>

### Establecer los diferentes errores que se deben ignorar en el control de visualización HTML y el control Página WEBDEV

`<Control used>.IgnoreError = <Error to ignore>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control de visualización HTML o control Página WEBDEV a manipular. 

**`<Error to ignore>: (Constante o combinación de constantes de tipo Integer)`**

Errores de certificado ignorados: 



|   |   |
| --- | --- |
| *httpIgnoreExpiredCertificate* | La fecha del certificado se ignora |
| *httpIgnoreInvalidCertificate* | El certificado se ignora. |
| *httpIgnoreInvalidCertificateName* | El nombre del sitio especificado en el certificado se ignora. |
| *httpIgnoreRedirectToHTTP* | Se permite la redirección a un servidor no seguro. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta propiedad está disponible para el control de visualización HTML y el control Página WEBDEV. Ambos tipos de controles se basan en Chromium Embedded Framework (CEF).

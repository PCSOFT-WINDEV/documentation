


## FilterProcedure (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **FilterProcedure** permite obtener y modificar el procedimiento utilizado para aplicar un filtro en un encabezado de fila o columna al calcular un control Tabla dinámica. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the content of a pivot table 
// for the stores whose group matches <"MyFilterProc"> 
// from January 1st, 2012
PVT_Statistics1.COL_InvoiceDate.MinValue = "20120101"
PVT_Statistics1.COL_StoreGroup.FilterProcedure = "MyFilterProc"
PVTCalculateAll(PVT_Statistics)

PROCEDURE MyFilterProc(sGroupName is string)
// Ignore the stores found in the "TEST" group
IF sGroupName = "TEST" THEN RESULT False
// Take all the other ones
RETURN True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el nombre del filtro Procedure

`<Result> = <Header to filter>.FilterProcedure`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de WLanguage Procedure usado para el filtro.

**`<Header to filter>: (Nombre del control)`**

Nombre de fila o cabecera de columna que se encuentra en el control Tabla dinámica filtrado. 


<a name="SYNTAX2"></a>

### Modificando el filtro Procedure associated con el encabezado

`<Header to filter>.FilterProcedure = <New procedure>`
---

**`<Header to filter>: (Nombre del control)`**

Nombre de la cabecera fila o de la cabecera de columna que se encuentra en el pivote filtrado tabla. 

**`<New procedure>: (Cadena de caracteres)`**



- Nombre de WLanguage Procedure usado para el filtro.

- Cadena vacía para desactivar el filtro.

<a name="NOTE0_1"></a>

## Observaciones


### Procedimiento de filtro
<a name="procedimiento_filtro_ELTPARAGRAPHE000042"></a>

El filtro Procedure se utiliza para elegir los valores que se tendrán en cuenta al calcular el control Tabla dinámica durante la llamada a [PVTCcalcularTodos](../WDLang1/1000020539.md) y [PVTCalcularActualizar](../WDLang1/1000020534.md).

El Procedure toma en parámetro el valor de la clave para el encabezado fila o el encabezado de la columna para el cual fue definido.

Este Procedure debe regresar: 

- <u><u><u><u>True</u></u></u></u> para aceptar el valor

- <u><u><u><u>False STRIMEULB</u></u></u></u> a rechazarla: el valor no se tendrá en cuenta en la control Tabla dinámica. 




Este Procedure se utiliza principalmente para acelerar el cálculo del pivote tabla reduciendo el volumen a Process.

**Atención**: No confundir con [filtro PVT](../WDLang1/1000020531.md) que se aplica al resultado calculado y que da una visión diferente del resultado mostrado.

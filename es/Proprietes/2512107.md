


## User (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **User** permite gestionar el usuario de una conexión (HFSQL Client/Server, OLE DB, etc.). Es posible:

- Identificar el usuario de una conexión (HFSQL Client/Server, OLE DB, etc.).

- Definir el usuario de una conexión (HFSQL Client/Server, OLE DB, ....). En este caso, si la conexión está actualmente abierta, se cerrará y se reabrirá con los nuevos parámetros durante el primer acceso a un archivo que utilice esta conexión.




**Observación**: La conexión puede corresponder a:

- una conexión descrita en la editor de análisis,

- una conexión descrita por [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- una conexión descrita por un [Conexión](../WDLang4/1514073.md) variable.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
// Describe the connection
MyConnection.User = "FP"
MyConnection.Password = "Test"
MyConnection.Server = "MYSERVER" 
MyConnection.Database = "Database"
MyConnection.Provider = hAccessHFClientServer
MyConnection.Access = hOReadWrite
MyConnection.ExtendedInfo = "Extended information"
MyConnection.CursorOptions = hClientCursor

HOpenConnection(MyConnection)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificación del usuario de una conexión

`<User name> = <Connection used>.User`
---

**`<User name>: (Cadena de caracteres)`**

Nombre del usuario de la conexión.

**`<Connection used>: (Nombre de la conexión)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por [Conexión](../WDLang4/1514073.md) variable.  





<a name="SYNTAX2"></a>

### Definición del usuario de una conexión

`<Connection used>.User = <New username>`
---

**`<Connection used>: (Nombre de la conexión)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por [Conexión](../WDLang4/1514073.md) variable.




**`<New username>: (Cadena de caracteres)`**

Nombre del usuario de la conexión.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
[CtUserName](../Proprietes/2512026.md) se utiliza para definir el usuario de una conexión. Este Property se mantiene para la compatibilidad retroactiva. Se recomienda utilizar el **User** Property.

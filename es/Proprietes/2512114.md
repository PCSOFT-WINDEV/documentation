


## NbDescribedThumbnail (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbDescribedThumbnail** Property se utiliza para get el número de miniaturas esperadas para un elemento. Este punto se definió en la editor de análisis o a través de la programación.

Observación: El número de miniaturas esperadas puede ser diferente del número de miniaturas realmente generadas (devueltas por el [NbPequeña imagen](../Proprietes/2512113.md) Property).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the number of generated thumbnails 
// and the number of expected thumbnails
// for the Customer.Photo item
NumberThumbnail is int
ExpectedNumberThumbnail is int
NumberThumbnail = Customer.Photo.NbThumbnail
ExpectedNumberThumbnail = Customer.Photo.NbDescribedThumbnail
// Same number?
IF NumberThumbnail = ExpectedNumberThumbnail THEN
	Info("All the thumbnails have been generated")
ELSE
	FOR Number = 1 TO ExpectedNumberThumbnail
		ErrorString += "Thumbnail " + Number + Customer.Photo.InfoThumbnailGeneration[n] + CR
	END
	Error("Generation error", ErrorString)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Expected number of thumbnails> = <Data file>.<Item>.NbDescribedThumbnail`
---

**`<Expected number of thumbnails>: (Integro)`**

Número de miniaturas solicitadas para el elemento especificado.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **NbDescribedThumbnail** Property está disponible:

- en HFSQL Classic: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: sólo para los artículos que se encuentran en un archivo de datos.


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **NbDescribedThumbnail** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.

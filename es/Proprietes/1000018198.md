
## Link (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Link** permite obtener y cambiar el enlace de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [rssEntry](../WDLang5/1000017792.md).

- una variable de tipo [rssImage](../WDLang5/1000017799.md).

- una variable de tipo [rssTextInput](../WDLang5/1000017802.md).  

- una variable de tipo [hdfsStatus](../WDLang4/1000021973.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElement](../WDLang1/1000022484.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementLink](../WDLang1/1000022970.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagWaypoint](../WDLang1/1410088548.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagSelection](../WDLang1/1410088172.md).

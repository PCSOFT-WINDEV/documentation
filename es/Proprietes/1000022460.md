


## VerticalSwipe (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VerticalSwipe** Property se utiliza para determinar y cambiar la dirección de deslizamiento en una ventana interna con soporte de gestos de deslizamiento.

Observación: Esta propiedad corresponde a la opción "Orientación del deslizamiento" de la pestaña "UI" de la ventana de descripción de un control Ventana interna. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Event: Changing the orientation of the window
// The following code is used to always swipe on the same axis
IW_MyInternalWin1.VerticalSwipe = NOT IW_MyInternalWin1.VerticalSwipe
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la dirección del golpe

`<Result> = <Internal Window control>.VerticalSwipe`
---

**`<Result>: (karaoke)`**



- <u><u><u><u>True</u></u></u></u> si el barrido se realiza verticalmente,

- <u><u><u><u>False</u></u></u></u> en caso contrario: el deslizamiento se realiza horizontalmente.  




**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna utilizada. 


<a name="SYNTAX2"></a>

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta Property tiene n efecto si la función de deslizamiento no está activada en la control Ventana interna. Para obtener más información, consulte [Cambiar el contenido de la ventana con un deslizamiento](../WDChamp/1013277.md).




## Count (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Count** devuelve el número de elementos en un conjunto determinado. Puedes get:

- el número de líneas en un List Box o control Combo Box.

- el número de filas de una control Tabla. Si hay un filtro en una columna, el **Count** Property devuelve el número de elementos mostrados. 

- el número de filas de una control Looper.

- el número de elementos de una control ListView.

- el número de filas de una columna control Tabla.

- el número de opciones en un botón de radio o control Casilla de verificación.

- el número de controles de un grupo de controles.

- el número de elementos de una control TreeView.

- el número de series de una control Gráfico.

- el número de elementos de una array, asociativa array o avanzada array (arrays utilizada en las variables gglxxx por ejemplo) así como el número de ocurrencias para un elemento de la array asociativa .

- el número de elementos en una cola, lista o pila WLanguage. 

- el número de subelementos indexados en una variante.

- el número de paneles de una control Pestaña.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de opciones de un menú.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de elementos de una control Organigrama.




**Observación**: La propiedad **Count** puede utilizarse en los controles Tabla, List Box, Looper o Combo Box basados en un archivo de datos o rellenados mediante programación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements found in the "TABLE_CustomerTable" control
ResCount = TABLE_CustomerTable.Count
```
<a name="Example2"></a>

```wl
// Number of elements in CustomerArray
CustomerArray is dynamic array
CustomerArray = new dynamic array of 4 by 7 int
Trace("Number of elements: " + CustomerArray.Count)
```

### control Tabla basada en un archivo de datos con barra de desplazamiento rebotante
<a name="control_tabla_basada_archivo_datos_con_barra_desplazamiento_rebotante_ELTPARAGRAPHE000167"></a>

En un control Tabla basado en un archivo de datos con una barra de desplazamiento continua, el **Count** Property devuelve el número de filas mostradas en el control Tabla y no el número de registros en el archivo de datos associated.

### Array y la asociación array
<a name="array_asociacion_array_ELTPARAGRAPHE000178"></a>

El **Count** Property se utiliza para:

- Averigua el número de elementos en una array o en una array asociativa. 
	Ejemplo:
	
	```wl
	// Associative array of integers 
	// indexed on strings and with duplicates
	aaIDPerCustomer is associative array of int
	// Add the identifier of customer "A"
	aaIDPerCustomer["A"] = 55
	// Add the identifier of customer "B"
	aaIDPerCustomer["B"] = 321
	// Add the identifier of customer "A"
	aaIDPerCustomer["A"] = 56
	
	Trace("Nb elements: " + aaIDPerCustomer.Count)
	// Displays 2 (ignores the duplicates)
	```


- Averiguar el número de ocurrencias para un elemento en una array asociativa. Esta Property es muy useful para una array asociativa con duplicados. 
	Ejemplo:
	
	```wl
	// Associative array of integers 
	// indexed on strings and with duplicates
	aaIDPerCustomer is associative array of int
	// Add the identifier of customer "A"
	aaIDPerCustomer["A"] = 55
	// Add the identifier of customer "B"
	aaIDPerCustomer["B"] = 321
	// Add the identifier of customer "A"
	aaIDPerCustomer["A"] = 56
	
	Trace("Nb elements: " + aaIDPerCust["A"].Count)
	// Displays 1
	```

### Límite
<a name="limite_ELTPARAGRAPHE000198"></a>

La propiedad **Count** solo se aplica a:

- controles List Box y ListView,

- controles Tabla y Tabla TreeView, 

- Controles de looper,

- controles Combo Box,

- Controles de TreeView,

- Controles de las casillas de verificación,

- Controles de los botones de radio,

- columnas de controles Tabla, columnas de TreeView controles Tabla,

- Pestañas de control,

- grupos de controles, 

- Controles de la carta,

- arrays, arrays asociativa,

- colas, pilas y listas,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles TreeMap.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) menús.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) controles Organigrama.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Count** solo se aplica a los siguientes elementos:

- Controles de las casillas de verificación.

- Controles de los botones de radio.

- Controles de List Box.

- controles Combo Box.

- Controles de TreeView.

- Pestañas de control.

- controles Tabla.

- columna de controles Tabla.

- colas, pilas y listas.

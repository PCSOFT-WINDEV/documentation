


## ResourceHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ResourceHeight** Property se utiliza para get y cambiar la altura de los recursos en un control Planificador donde los recursos están dispuestos en filas. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_Zoom = True THEN
	SCH_MySchedule.ResourceHeight = 200
ELSE
	SCH_MySchedule.ResourceHeight = 100
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la altura de los recursos

`<Result> = <Scheduler control>.ResourceHeight`
---

**`<Result>: (Integro)`**



- Altura de los recursos en píxeles, si la altura fue modificada a través de la programación (con el **ResourceHeight**) Property).

- -1 en los siguientes casos: 

	- si la altura de los recursos se calcula automáticamente (a través de la información dada en la ventana de description del control).

	- si los recursos se muestran en columnas. 







**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada.  


<a name="SYNTAX2"></a>

### Cambiar la altura de los recursos

`<Scheduler control>.ResourceHeight = <New height>`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada.

**`<New height>: (Integro)`**



- Nueva altura para los recursos expresados en píxeles. 

- -1 para calcular automáticamente la altura de los recursos a partir de la información dada en la ventana de description del control.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la altura de los recursos se define con el **ResourceHeight** Property, los recursos no pueden ser "anclados" según el tamaño disponible. 

- Sólo se puede utilizar el valor -1. Los otros valores negativos se ignoran.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La altura de un recurso específico puede modificarse con la [Altura](../Proprietes/2510050.md) Property utilizando la siguiente sintaxis:
	
	```txt
	Schedule_Name.Resource["MyResource"].Height = <New value>
	```

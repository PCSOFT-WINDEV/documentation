
## Virtual (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Virtual** Property se utiliza para determinar si un elemento es virtual. Esta propiedad es utilizada por:

- una variable de tipo [Procedure Description](../WDLang1/1000019317.md).

- una variable de tipo [Property Description](../WDLang1/1000019324.md).

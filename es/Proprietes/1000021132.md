
## Stemming (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Stemming** Property se utiliza para permitir el stemming y para definir el lenguaje utilizado para el stemming en un índice de texto completo definido mediante programación.

La derivación se utiliza para realizar búsquedas en un índice de texto completo teniendo en cuenta las variaciones de una palabra dada.. Esta característica no es interesante para las bases de datos de conocimiento y permite encontrar indistintamente palabras en singular o plural.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyFullTextIndex is FullTextIndex Description
...
MyFullTextIndex.Stemming = nationEnglish
```

<a name="XSYNTAX"></a>

## Observaciones


- La gestión de los sinónimos ([Sinónimo](../Proprietes/1000021127.md)) no debe estar activada si se habilita el stemming. 

- La gestión de los caracteres acentuados ([AccentSensitive](../Proprietes/2512077.md)) debe estar activada si se habilita el stemming. 

- Debido a excepciones gramaticales (verbos irregulares en inglés, verbos del tercer grupo en francés, etc.), es posible que algunas conjugaciones no se agrupen (por ejemplo, si se busca "fly" no se encontrarán los registros que contengan "flown") o que algunas formas plurales no se agrupen con su forma singular (pie/pies por ejemplo).




## KeepVisibleHierarchy (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **KeepVisibleHierarchy** Property se utiliza para:

- Averigüe qué elementos de TreeView tabla se imprimirán: imprimiendo toda la jerarquía o imprimiendo la jerarquía que es Visible en el control.

- Modificar el modo de impresión de la jerarquía: imprimiendo toda la jerarquía o imprimiendo la jerarquía que es Visible en el control.




Esta propiedad solo puede utilizarse en el código de apertura de un reporte "Reporte basado en Tabla TreeView".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Opening code of the report
// Prints the visible hierarchy only
RPT_Report1.KeepVisibleHierarchy = True
```


<a name="Example2"></a>

```wl
// -- Opening code of the report
// Prints the entire hierarchy
RPT_Report1.KeepVisibleHierarchy = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de impresión de la jerarquía

`<Result> = <Report used>.KeepVisibleHierarchy`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el reporte imprimirá solo las filas de la jerarquía que se muestran actualmente en el control Tabla TreeView,

- <u><u><u><u>False</u></u></u></u> si el reporte imprimirá todas las filas de jerarquía que se muestran en el control Tabla TreeView.




**`<Report used>: (Nombre del reporte)`**

Nombre de la "Reporte en la tabla TreeView" reporte a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el modo de impresión de la jerarquía

`<Report used>.KeepVisibleHierarchy = <Type of print>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Reporte en la tabla TreeView" reporte a utilizar.

**`<Type of print>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para imprimir solo las filas de la jerarquía que se muestran actualmente en el control Tabla TreeView,

- <u><u><u><u>False</u></u></u></u> para imprimir todas las filas de la jerarquía que se muestran en el control Tabla TreeView.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **KeepVisibleHierarchy** sólo se aplica a los informes "Reporte on TreeView Table" y sólo se puede utilizar en el código de apertura del reporte.

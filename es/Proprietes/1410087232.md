


## VisibleWithAnimation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **VisibleConAnimación** se utiliza para hacer visible o Invisible un control mediante la animación "Aparecer/Desaparecer. Esta animación se define: 

- en la pestaña "Avanzado" de la ventana del proyecto description (botón "Animación control"). 

- en la pestaña "Estilo" de control o del proyecto. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Makes the "Apply" Button control visible
BTN_Apply.VisibleWithAnimation = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

`<Control used>.VisibleWithAnimation = <Visible>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la célula control, grupo de control s o control Disposición a manipular. 

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer el control Visible con la animación "Aparecer/Desaparecer",

- <u><u><u><u>False</u></u></u></u> para hacer el control Invisible con la animación "Aparecer/Desaparecer. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las propiedades de la animación (duración, por ejemplo) pueden especificarse en las opciones del proyecto y anularse en el estilo control. 

- Esta Property sólo está disponible para los controles, grupos de controles y celdas control Disposición.

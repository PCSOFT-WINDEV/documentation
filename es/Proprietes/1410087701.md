
## TapToFocus (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **TapToFocus** permite: 

- Determinar si la función "pulsar para enfocar" está habilitada en un control Cámara. 

- Habilitar o deshabilitar la función "pulsar para enfocar" en un control Cámara. 


Si se habilita el modo de pulsar para enfocar, la cámara enfocará en el punto que el usuario haya pulsado. 

Observaciones: 

- Esta propiedad solo puede utilizarse en controles Cámara que no están en "Modo compatible con la versión 26". 

- Esta propiedad corresponde a la opción "Habilitar enfoque al pulsar" en la pestaña "General" de la ventana de descripción del control Cámara. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CAM_MyCamera.TapToFocus = True THEN
	ToastDisplay("Tap to focus.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si la función de pulsar para enfocar está habilitada en un control Cámara

`<Result> = <Camera control>.TapToFocus`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función pulsar para enfocar está habilitada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar. 


<a name="SYNTAX2"></a>

### Habilitar o deshabilitar la función de pulsar para enfocar en un control Cámara

`<Camera control>.TapToFocus = <Authorization>`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar. 

**`<Authorization>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> para habilitar la función de pulsar para enfocar,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

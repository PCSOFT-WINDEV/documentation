


## FilterWithBounds (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **FilterWithBounds** permite saber si se han especificado límites para el filtro establecido con la función [HFilter](../WDLang4/3044100.md) en un archivo de datos HFSQL, vista o consulta.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables the filter
HFilter(Customer, CustomerName, "D" + hValMin, "F" + hValMax, "City='Paris'")
IF Customer.FilterWithBounds = True THEN Trace("Bounds have been defined for the filter")
// Disables the filter
HDeactivateFilter(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Element used>.FilterWithBounds`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se han especificado límites para el filtro con la función [HFilter](../WDLang4/3044100.md),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element used>: (Cadena de caracteres)`**

Nombre del archivo de datos HFSQL, consulta o vista en la que se definió el filtro.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Valores de los límites
<a name="valores_los_limites_ELTPARAGRAPHE000070"></a>

Si se han definido límites, sus valores son devueltos por [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md).
<a name="NOTE0_2"></a>


### Recordatorio: hMinVal y hMaxVal utilizados en un filtro
<a name="recordatorio_hminval_hmaxval_utilizados_filtro_ELTPARAGRAPHE000083"></a>

Para implementar **un filtro genérico sobre un valor**debes hacerlo:

- introducir el límite mínimo con la constante ***hMinVal*** para darle el valor mínimo.

- introducir el límite máximo con la constante ***hMaxVal*** para darle el valor máximo.




Por ejemplo, para seleccionar los clientes cuyo apellido comienza con "Smith":


```wl
HFilter(Customer, Name, "Smith" + hMinVal, "Smith" + hMaxVal)
```
Se seleccionarán los clientes "Durá" y "Durán".

Para más detalles, consulte la ayuda sobre [HFilter](../WDLang4/3044100.md).




## InterpretAmpersand (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InterpretAmpersy** Property se utiliza para determinar y cambiar cómo se interpreta el carácter '&' en:

- la leyenda de una control Estático.

- elementos de un control List Box.

- elementos de un control Combo Box.

- los valores de una columna en una control Tabla.




**Observación**: Por Default, si una cadena de caracteres contiene el carácter "&", la letra que sigue al carácter "&" es considerada como el atajo. Para obtener más información, consulte [Atajo de teclado y tecla de acceso directo](../WDChamp/1014018.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The '&' character is not interpreted as a shortcut
LIST_BookList.InterpretAmpersand = False
// Add a book title
ListAdd(LIST_BookList, "War&Peace")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar cómo se interpreta el carácter "&"...

`<Result> = <Control used>.InterpretAmpersand`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la letra que sigue al carácter "&" se interpreta como una tecla de acceso directo,

- <u><u><u><u>False</u></u></u></u> si no se interpreta el carácter '&'.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando la forma en que se interpreta el carácter "&"...

`<Control used>.InterpretAmpersand = <New Interpretation Mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. 

**`<New Interpretation Mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la letra que sigue al carácter "&" debe ser interpretada como una tecla de acceso directo,

- <u><u><u><u>False</u></u></u></u> si el carácter '&' no debe ser interpretado.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Carácter'&' interpretado como un atajo
<a name="caracter_interpretado_como_atajo_ELTPARAGRAPHE000064"></a>

Si la letra que sigue al carácter '&' se considera como el atajo, para mostrar el carácter '&' en un pie de foto, se debe escribir dos veces. Por ejemplo:

- "War&Peace" se mostrará como "War<u><u><u><u>P</u></u></u></u>eace" (con Alt + P como tecla de acceso directo).

- "War&&Peace" se mostrará como "War&Peace".



<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000078"></a>

El **InterpretAmpersand** Property sólo puede utilizarse en:

- un control Estático.

- un control List Box.

- un control Combo Box.

- una columna en control Tabla.

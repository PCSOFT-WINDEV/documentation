


## NbCopy (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbCopy** Property se utiliza para determinar y modificar el número de copias reporte a imprimir.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observaciones**:

- El visor reporte permite al usuario final configurar directamente el número de copias a imprimir.

- Esta Property funciona durante una impresión realizada sólo en una impresora.

- Si el usuario especifica un número de copias ([iParameter](../WDLang5/3046008.md) o [iParameterReport](../WDLang5/3046077.md)) antes de imprimir el reporte, este número de copias tendrá prioridad sobre el especificado con el **NbCopy** Property.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print 10 copies of the "RPT_Customer" report
RPT_Customer.NbCopy = 10
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de copias idénticas para un reporte

`<Result> = <Report used>.NbCopy`
---

**`<Result>: (Integro)`**

Número de copias a imprimir.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el número de copias idénticas para un reporte

`<Report used>.NbCopy = <Number of copies>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte a utilizar.

**`<Number of copies>: (Integro)`**

Número de copias a imprimir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Permite al usuario definir el número de copias de reporte
<a name="permite_usuario_definir_numero_copias_reporte_ELTPARAGRAPHE000069"></a>

Dado que el **NbCopy** Property sólo puede utilizarse en el código del reporte, el número de copias debe pasarse como parámetro al reporte en el código de impresión.

Por ejemplo:

1. Código utilizado para imprimir la reporte.
	
	```wl
	iPrintReport(RPT_MyReport, 3)
	```


2. Código de apertura de reporte.
	
	```wl
	PROCEDURE RPT_MyReport(nNbCopies)
	RPT_MyReport.NbCopy = nNbCopies
	```




<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000087"></a>

- El **NbCopy** Property sólo se aplica a los informes (y no a los controles o bloques de un reporte).

- Esta Property se tiene en cuenta cuando la impresión se realiza en una impresora.

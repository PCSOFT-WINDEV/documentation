


## AttachPointEnd (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AttachPointEnd** Property se utiliza para get y cambiar la posición del punto de fijación final de un control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartAttachPointID is int
EndAttachPointID is int

WIRE_Link_wire is Control

WIRE_Link_wire <- ControlCreate("Link", typWire)


WIRE_Link_wire.StartControl = IMG_Start
WIRE_Link_wire.EndControl = IMG_End

StartAttachPointID = AttachPointAdd(IMG_Start, 100, 25))
EndAttachPointID = AttachPointAdd(IMG_End, 0, 75))

WIRE_Link_wire.AttachPointStart = StartAttachPointID
WIRE_Link_wire.AttachPointEnd = EndAttachPointID
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la posición del punto de unión de los extremos en un alambre control

`<Result> = <Wire control>.AttachPointEnd`
---

**`<Result>: (Entero o constante de tipo Integer)`**



- Identificador del punto de unión si el punto de unión fue personalizado. Este identificador corresponde al valor devuelto por [AttachPointAdd](../WDLang1/1000021902.md). 

- Constante correspondiente a la posición del punto de unión (si esta posición no fue personalizada). 
	


|   |   |
| --- | --- |
| *ptAutomatic*<br>(Valor predeterminado) | WINDEV selecciona automáticamente el mejor punto de enganche. |
| *ptBottomCenter* | El punto de fijación se sitúa en el centro del borde inferior de la control. |
| *ptTopCenter* | El punto de unión está situado en el centro del borde superior de la control. |
| *ptMiddleRight* | El punto de enganche se sitúa en el centro de la Border derecha del control. |
| *ptMiddleLeft* | El punto de enganche se sitúa en el centro de la Border izquierda del control. |






**`<Wire control>: (Nombre del control)`**

Nombre del alambre control que se utilizará. 


<a name="SYNTAX2"></a>

### Modificando la posición del punto de unión del extremo en un alambre control

`<Wire control>.AttachPointEnd = <New value>`
---

**`<Wire control>: (Nombre del control)`**

Nombre del alambre control a utilizar.

**`<New value>: (Entero o constante de tipo Integer)`**

Este valor puede corresponder a:

- el identificador del punto de enganche devuelto por [AttachPointAdd](../WDLang1/1000021902.md).

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *ptAutomatic*<br>(Valor predeterminado) | WINDEV selecciona automáticamente el mejor punto de enganche. |
| *ptBottomCenter* | El punto de fijación se sitúa en el centro del borde inferior de la control. |
| *ptTopCenter* | El punto de unión está situado en el centro del borde superior de la control. |
| *ptMiddleRight* | El punto de enganche se sitúa en el centro de la Border derecha del control. |
| *ptMiddleLeft* | El punto de enganche se sitúa en el centro de la Border izquierda del control. |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El punto de conexión inicial se puede personalizar con el programa [AttachPointStart](../Proprietes/1000021907.md) Property. 

- Si los puntos de anclaje de Default han sido borrados por [AttachPointDeleteAll](../WDLang1/1000021903.md), las constantes de posición ya no pueden ser utilizadas.

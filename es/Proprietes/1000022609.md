
## ContentType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ContentType** Property se utiliza para get y definir el contenido de un elemento. Este Property puede ser usado: 

- en una Variable de tipo [BarCode](../WDLang5/1000019451.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una Variable de tipo [mongoGridFSFileInfo](../WDLang4/1000022417.md).




## Legend (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Legend** permite:

- Descubra la etiqueta de una serie.

- Modificar la etiqueta de una serie.




**Observación**: Esta Property es equivalente a la [grSeriesLabel](../WDLang3/3042024.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Legend of series 2
CHART_MyChart[2].Legend = "Paris"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la etiqueta de una serie

`<Result> = <Series>.Legend`
---

**`<Result>: (Cadena de caracteres)`**

Etiqueta de la serie especificada.

**`<Series>: (Cadena de caracteres)`**

Serie cuya etiqueta se solicita. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.  


<a name="SYNTAX2"></a>

### Modificar la etiqueta de una serie

`<Series>.Legend = <New label>`
---

**`<Series>: (Cadena de caracteres)`**

Serie cuya etiqueta debe ser modificada. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.

**`<New label>: (Cadena de caracteres)`**

Etiqueta de la serie especificada.


## InitialVisible (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **..InitialVisible** permite saber si:

- un control era visible durante la primera visualización de la ventana o página asociada.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una ventana era visible durante su primera visualización.




**Observaciones**:

- La visibilidad inicial de un control o ventana corresponde a la definida en el editor (pestaña"Detalles" de la descripción del objeto).. Esta opción se utilizará cuando se inicialice el control o ventana.

- La visibilidad de un control o ventana puede ser modificada por [..Visible](../Proprietes/2510138.md) y [..State](../Proprietes/2510039.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Restores the "EDT_Company" control to its initial visibility status
EDT_Company..Visible = EDT_Company..InitialVisible
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Element used>..InitialVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el objeto especificado era visible cuando se mostró por primera vez,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element used>: (Nombre del control o nombre de la ventana)`**

Nombre del objeto (control o ventana) a utilizar. 

Para un control asociado a una pestaña, utilice la siguiente anotación: 

```txt
<Tab name>.<Control name>
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Atributos de control
<a name="atributos_control_ELTPARAGRAPHE000108"></a>

**..InitialVisible** no modifica los otros atributos de control (selección sin entrada, habilitada o en gris).


### Límites
<a name="limites_ELTPARAGRAPHE000113"></a>

La propiedad **..InitialVisible** no puede utilizarse en:

- una opción en una casilla de verificación o botón de opción,

- una fila en un tabla, en un list box o en un combo box,

- una célula tabla,

- un menú de ventana,

- un grupo de controles,

- un elemento en una control TreeView.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **..InitialVisible** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- Image control.

- Control Casilla de verificación.

- Control Botón de opción.

- Un control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control de una tabla.

- Columna de tabla.

- Supercontrol.

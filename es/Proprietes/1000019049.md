


## DayOfMonth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DayOfMonth** Property se utiliza para get y cambiar el número del día del mes para el que debe ejecutarse la tarea programada. 

Esta propiedad se puede utilizar en:

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [hScheduling](../WDLang4/1000018853.md).

- una variable de tipo [Repetition](../WDLang1/1000019262.md).

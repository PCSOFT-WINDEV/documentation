
## Geodesic (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Geodésico** Property se utiliza para indicar si el dibujo debe seguir la curvatura de la Tierra. Esta propiedad se puede utilizar en: 

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md).

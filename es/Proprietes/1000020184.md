


## EndDatePeriodDisplayed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **EndDatePeriodDisplayed** Property se utiliza para get la fecha de finalización del periodo de tiempo mostrado en un Planificador o control Agenda de un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// "Before printing" process of the header block
SCH_MySchedule.CaptionPeriodHeader = ...
	" Week from " + DateToString(SCH_MySchedule.StartDatePeriodDisplayed) + ...
	" to " + DateToString(SCH_MySchedule.EndDatePeriodDisplayed)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la fecha de finalización del período de tiempo mostrado en la reporte

`<End date> = <Control used>.EndDatePeriodDisplayed`
---

**`<End date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de finalización del período de tiempo que aparece en la reporte.  

**`<Control used>: (Nombre del control)`**

Nombre del Programador o control Agenda en la reporte.

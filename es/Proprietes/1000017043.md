


## Hint (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Hint** obtiene y establece el texto de ayuda que se muestra cuando el control Campo de entrada está vacío.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Indication.gif)
El texto de ayuda se muestra directamente en el control y desaparece en cuanto se introduce un carácter. El texto de ayuda puede definirse:

- en WINDEV y WINDEV Mobile, en la pestaña "Ayuda" de la ventana de descripción del control Campo de entrada,

- en WEBDEV, en la pestaña "Contenido" de la ventana de descripción del control.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_Edit1.Hint = "Type the customer name"
EDT_Password.Hint = "Password"
EDT_Date.Hint = "Exit date, 12/06/2007"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el texto de ayuda

`<Hint text> = <Control used>.Hint`
---

**`<Hint text>: (Cadena de caracteres)`**

Texto que se mostrará en el control especificado cuando se cumplan las siguientes condiciones:

- El control Campo de entrada (o el control de un Combo Box con entrada) está vacío.

- No se están introduciendo datos en el control.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: control Campo de entrada o Combo Box editable .  


<a name="SYNTAX2"></a>

### Cambiar el texto de ayuda

`<Control used>.Hint = <Hint text>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: control Campo de entrada o Combo Box editable.

**`<Hint text>: (Cadena de caracteres)`**

Texto que se mostrará en el control especificado cuando se cumplan las siguientes condiciones:

- El control Campo de entrada (o el control de un Combo Box con entrada) está vacío.

- No se están introduciendo datos en el control.


Si este parámetro es una cadena vacía (""), no se mostrará ningún texto de ayuda.

Este texto puede corresponder a una cadena gráfica (que utiliza las funciones [gBackground](../WDLang1/3029037.md), [gImage](../WDLang1/3029042.md), etc.).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Seleccione la opción "En blanco si es 0" si desea que el texto de ayuda se muestre en los controles Campo de entrada de tipo Numérico, Monetario y Monetario + Euro.

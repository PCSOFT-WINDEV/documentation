


## ToolbarVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ToolbarVisible** permite: 

- saber si la barra de herramientas o la cinta de opciones se muestra en un control.  

- mostrar u ocultar la barra de herramientas o la cinta de opciones en un control. 




En un control Hoja de cálculo, el **ToolbarVisible** Property se utiliza para: 

- saber si se muestra la barra de herramientas o la cinta de opciones. 

- mostrar u ocultar la barra de herramientas o la cinta de opciones.


En un control de Procesamiento de texto, Editor de imágenes, Lector de PDF, Editor de HTML o Editor de diagramas, el **ToolbarVisible** Property se utiliza para :  

- saber si la cinta se muestra. 

- mostrar una cinta.




Recordatorio:  Esta Property corresponde a la opción: 

- "Barra de herramientas" en la pestaña "Detalles" de la ventana de descripción del control Hoja de cálculo. Tiene la capacidad de mostrar una barra de herramientas o una cinta.. Esta propiedad se utiliza para mostrar u ocultar el elemento elegido en la ventana de description. Si se selecciona la opción "Sin barra", el **ToolbarVisible** Property se utilizará para mostrar u ocultar la barra de herramientas. 

- "Con cinta de opciones", en la pestaña "General" de la ventana description de los controles Procesador de textos, Editor de imágenes, Editor de HTML, Editor de Diagram o Lector de PDF.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_Toolbar THEN
	PSHEET_Spreadsheet.ToolbarVisible = True
ELSE
	PSHEET_Spreadsheet.ToolbarVisible = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si una barra de herramientas (o la cinta) se muestra en la control

`<Result> = <Control used>.ToolbarVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la barra de herramientas (o la cinta de opciones) es visible,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Hoja de cálculo. 

- Control Procesador de texto.

- Control Editor de imágenes.

- Control Lector PDF.

- Control Editor HTML.

- Control Editor de diagramas.





<a name="SYNTAX2"></a>

### Mostrar (o no) una barra de herramientas (o una cinta) en una control

`<Control used>.ToolbarVisible = <Display>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Hoja de cálculo. 

- Control Procesador de texto.

- Control Editor de imágenes.

- Control Lector PDF.

- Control Editor HTML.

- Control Editor de diagramas.




**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer visible la barra de herramientas (o la cinta de opciones),

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0_1"></a>

## Observaciones


- control Hoja de cálculo: Puede personalizar la barra de herramientas en un control Hoja de cálculo. Para obtener más información, consulte [Personalización de la barra de herramientas (control Hoja de cálculo)](../WDChamp/1000033001.md).

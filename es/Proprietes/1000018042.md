
## tipo de letra

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Fuente** palabra clave puede corresponder a: 

- la [tipo de letra](../Motscles/1514045.md) Variable, que se utiliza para definir las características de un tipo de letra. 

- la **Fuente** Property, que se utiliza para get y cambiar la fuente utilizada en los controles de una Page, ventana o reporte. Para obtener más información, consulte [Fuente en los controles](../Proprietes/2510095.md).

- el **Font** Property disponible para: 

	- una variable de tipo [xlsCell](../WDLang5/1000017472.md). 

	- una variable de tipo [Watermark](../WDLang5/1000019922.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picText](../WDLang1/1000025178.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

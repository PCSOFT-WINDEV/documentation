
## Alias (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Alias** permite obtener el alias de un elemento. Esta propiedad se puede utilizar en: 

- un elemento WINDEV, WEBDEV o WINDEV Mobile: 

	- para obtener y establecer el alias de una ventana (WINDEV). 

	- para obtener y establecer el alias de una pestaña dinámica (WINDEV)

	- para obtener el nombre interno HTML/JavaScript de un control (WEBDEV).


 Para obtener más información, consulte la ayuda sobre la propiedad [Alias](../Proprietes/2510004.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoGridFSFileInfo](../WDLang4/1000022417.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docStyle](../WDLang1/1000022486.md). 

- una variable de tipo [Option Description](../WDLang1/1000023072.md). 

- una variable de tipo [Value description](../WDLang1/1000023061.md). 

- una variable de tipo [YAML](../Motscles/1000024438.md).

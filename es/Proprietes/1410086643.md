
## VersionHTTP (Property)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## XXX
<a name="xxx_ELTTEXTE000075"></a>
**HTTPVersion** Property se utiliza para administrar la versión HTTP utilizada por el servidor. Esta propiedad se puede utilizar en:

- a Variable de tipo [httpRequest](../WDLang3/1000021158.md). 

- una variable generada automáticamente para representar [un Webservice importado](../WDLang3/3086001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable generada automáticamente para representar [un servicio web REST importado](../WDLang3/1410087809.md).

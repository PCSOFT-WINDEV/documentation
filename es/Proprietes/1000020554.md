
## WithZoom (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WithZoom** permite especificar y determinar si el usuario puede acercar y alejar la pantalla: 

- en un control de visualización HTML en aplicaciones Android, iPhone o iPad. 

- en un control Mapa en aplicaciones Android, iPhone o iPad.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable zoom on the HTM_NavigationPCS control
HTM_NavigationPCS.WithZoom = False
```


<a name="Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el zoom está habilitado en un control HTML o Mapa

`<Result> = <Control used>.WithZoom`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el zoom está habilitado en el control,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML.

- Control Mapa.





<a name="SYNTAX2"></a>

### Habilitar o deshabilitar el zoom en un control HTML o Mapa

`<Control used>.WithZoom = <Zoom>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control de visualización HTML.

- Control Mapa.




**`<Zoom>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe habilitar el zoom en el control,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el zoom está habilitado, el usuario podrá acercar y alejar con gestos táctiles.

- La propiedad no tiene efecto en modo simulador.




## DisplayEvenIfNull (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **DisplayEvenIfNull** permite: 

- Determinar si se muestra una fila o columna de un control Tabla dinámica, incluso si no contiene ningún valor. 

- Mostrar u ocultar una fila o columna en un control Tabla dinámica si no contiene ningún valor. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The Month column will be displayed even if it contains no value
PVT_Sales.COL_Month.DisplayEvenIfNull = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el modo de visualización actual de una fila o columna si está vacía

`<Result> = <Row or column name>.DisplayEvenIfNull`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fila o columna se muestra aunque esté vacía, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Row or column name>: (Nombre del control)`**

Nombre de la fila o columna en el control Tabla dinámica. 


<a name="SYNTAX2"></a>

### Cambiar el modo de visualización de una fila o columna vacía

`<Row or column name>.DisplayEvenIfNull = <New display mode>`
---

**`<Row or column name>: (Nombre del control)`**

Nombre de la fila o columna del control Tabla dinámica a utilizar. 

**`<New display mode>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe mostrar la fila o columna aunque esté vacía, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Después de modificar la propiedad, el control Tabla dinámica no se vuelve a calcular: solo se vuelve a mostrar. 

- Si la propiedad se inicializa en <u><u><u><u>True</u></u></u></u> para una fila o columna que corresponde a una fecha, se muestran todos los días (meses o trimestres) aunque no haya datos para la fecha.

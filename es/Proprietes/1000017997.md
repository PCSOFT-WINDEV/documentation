


## EndDate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **EndDate** Property se utiliza para get y cambiar la fecha de finalización de un período de tiempo. Esta propiedad se puede utilizar en:

- Calendar, Controles del organizador y del programador. Para obtener más información, consulte [Fecha de finalización Property](../Proprietes/1000017427.md).

- una variable de tipo [saasSubscription](../WDLang3/1000019088.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md). 

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [Repetition](../WDLang1/1000019262.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [GanttTask](../WDLang1/1000020881.md).




## Height (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **altura** Property se usa para get y cambiar la altura:

- de un control o bloque en un reporte.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) de una control (en una ventana), ventana, fila de una Tabla, Cuadro de Lista o control ListView, opción de un Botón de Radio o control Casilla de verificación, romper cabecera o pie de una Tabla o control Looper.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de una celda de ruptura en una Tabla o control Looper. 




También puede get la altura de una celda en un control Tabla.



**Observaciones**:

- El borde de la control o bloque está incluido en su altura. El [HeightInPixel](../Proprietes/2511064.md) Property se utiliza para get la altura de un control en píxeles.

- La altura de un control se define en el editor, en el control description. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increases the height of "EDT_CustomerName"
EDT_CustomerName.Height = EDT_CustomerName.Height + 15
// Is equivalent to: EDT_CustomerName.Height += 15
```
<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Modify the height of a window and control
WIN_CustomerForm.Height += 100	// Adds 100 pixels
EDT_Input1.Height += 10	// Adds 10 pixels
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura de un elemento

`<Result> = <Element used>.Height`
---

**`<Result>: (Real)`**

La altura del elemento especificado. Esta altura se expresa en píxeles en una ventana y/o en una Page, y en milímetros en una reporte.

**`<Element used>: (Nombre de control o bloque.)`**

Nombre del elemento (control o bloque) a utilizar.  


<a name="SYNTAX2"></a>

### Modificar la altura de un elemento

`<Element used>.Height = <New height>`
---

**`<Element used>: (Nombre de control o bloque.)`**

Nombre del elemento (control o bloque) a utilizar.

**`<New height>: (Real)`**

Nueva altura del elemento especificado. Esta altura se expresa en píxeles en una ventana y/o en una Page, y en milímetros en una reporte.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Altura en una reporte
<a name="altura_una_reporte_ELTPARAGRAPHE000129"></a>

- La altura de un bloque no rompible no puede exceder la altura disponible de Page (excluyendo la cabecera Page y el pie Page).

- La altura de un pie de página Page no puede modificarse. Sin embargo, la altura de una cabecera Page puede modificarse mediante programación.

- Una control debe estar totalmente contenida en el bloque al que pertenece.



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Altura de Fila
<a name="altura_fila_ELTPARAGRAPHE000140"></a>

La altura de cada fila en una Tabla, Cuadro de Lista o control ListView puede definirse mediante programación. Para navegar por List Box y controles Tabla, esta altura se mantiene sólo para la sección Visible de la pantalla

Ejemplo: Define la altura de fila 3 en "LIST_List1" a 200.


```wl
LIST_List1[3].Height = 200
```
Observación: Para una columna en un control Tabla , el **Height** Property obtiene la altura Visible de la columna 06060605. Esta propiedad es de solo lectura.

### Altura inicial (control y ventana)
<a name="altura_inicial_control_ventana_ELTPARAGRAPHE000156"></a>

La altura inicial (descrita en el editor) es devuelta por el [InitialHeight](../Proprietes/2510043.md) Property.

En la mayoría de los casos, la altura de un objeto depende de [MaxHeight](../Proprietes/2510065.md) y [MinHeight](../Proprietes/2510035.md). Sin embargo, esta restricción ya no se aplica cuando la altura se cambia a través de la programación (con el **Height** Property o la función [WinSize](../WDLang1/3038020.md) ).

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000190"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Limitaciones para informes**
La propiedad **Height** no se puede usar en un reporte o en un control Casilla de verificación .

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **Height** solo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- Control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de control Tabla.

- Supercontrol.


## HeaderVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **HeaderVisible** permite: 

- Determinar si la fila y las cabeceras de las columnas se muestran en una control Hoja de cálculo. 

- Mostrar u ocultar la fila y las cabeceras de las columnas en una control Hoja de cálculo.




Recordatorio: 

- En un control Hoja de cálculo, las cabeceras se utilizan para identificar fácilmente las celdas.

- Esta propiedad corresponde a la opción "Mostrar encabezados de filas y columnas" de la pestaña "Detalles" de la ventana control Hoja de cálculo description.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_Header THEN
	PSHEET_Spreadsheet.HeaderVisible = True
ELSE
	PSHEET_Spreadsheet.HeaderVisible = False
END
```

<a name="SYNTAX1"></a>

### Determinar si la fila y las cabeceras de las columnas son Visible

`<Result> = <Spreadsheet control>.HeaderVisible`
---

**`<Result>: (booleano)`**



- <u><u><u>True</u></u></u> si los encabezados son visibles,

- <u><u><u>False</u></u></u> en caso contrario.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


<a name="SYNTAX2"></a>

### Mostrar u ocultar la fila y las cabeceras de las columnas

`<Spreadsheet control>.HeaderVisible = <Header Status>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Header Status>: (booleano)`**



- <u><u><u>True</u></u></u> si el encabezado es visible,

- <u><u><u>False</u></u></u> en caso contrario.

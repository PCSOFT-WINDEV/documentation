


## AutoLineWrap (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AutoLineWrap** Property se utiliza para: 

- determinar si la función de envoltura automatic Line está activada en un multiLine Editar control, en una columna de una Tabla o control TreeView o en una cabecera fila de un control Tabla dinámica. 

- set el modo de envoltura Line en un multiLine Editar control, en una columna de una Tabla o control TreeView o en una cabecera fila de un control Tabla dinámica.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the "Automatic line wrap" Check Box
EDT_MultilineText.AutoLineWrap = CBOX_AutoWrap
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si la envoltura de Line está automáticamente soportada

`<Result> = <Control used>.AutoLineWrap`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el salto de línea automático está activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Campo de entrada multilínea, 

- Columna en una Tabla o control Tabla TreeView.

- control Tabla dinámica. 





<a name="SYNTAX2"></a>

### Ajuste del modo de envoltura de Line

`<Control used>.AutoLineWrap = <Line wrap management>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Campo de entrada multilínea, 

- Columna en una Tabla o control Tabla TreeView.

- control Tabla dinámica. 




**`<Line wrap management>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar el salto de línea automático en el control,

- <u><u><u><u>False</u></u></u></u> para desactivar el salto de línea automático en el control.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la propiedad **AutoLineWrap** se establece en <u><u><u><u>False</u></u></u></u>, las líneas que son muy largas pueden desplazarse horizontalmente (con o sin barra de desplazamiento visible, según la configuración del control Campo de entrada). 

- Si la propiedad **AutoLineWrap** se establece como <u><u><u><u>True</u></u></u></u>, las líneas que desbordan se dividen en varias líneas. 

- Esta Property no tiene ningún efecto sobre un RTF control, un HTML control o un control Campo de entrada no multilineal.

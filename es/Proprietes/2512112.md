


## MaxThumbnailWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxThumbnailWidth** Property se utiliza para:

- get la anchura máxima de una miniatura de un elemento de Image (nota binaria). Este punto se definió en la editor de análisis o a través de la programación.

- establecer la anchura máxima de una miniatura de un elemento de imagen (memo binario), en el elemento description sólo a través de la programación.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the Photo item
Photo is Item Description
...
// Define the maximum width of the first 
// thumbnail for the description of the Photo item
Photo.MaxThumbnailWidth[1] = 50
// equivalent to Photo.MaxThumbnailWidth = 50
...
// Validate the description of the file
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la anchura máxima de una miniatura de un artículo de Image (nota binaria)

`<Current maximum width> = <Data file>.<item>.MaxThumbnailWidth[<N>]`
---

**`<Current maximum width>: (Integro)`**

Ancho máximo (en píxeles) de la miniatura especificada.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<N>: (Entero opcional)`**

Número de la miniatura cuya anchura máxima se solicita. Si no se especifica este parámetro, se devuelve el ancho máximo de la primera miniatura..  


<a name="SYNTAX2"></a>

### Modificación de la anchura máxima de una miniatura en un artículo de Image (nota binaria)

`<item>.MaxThumbnailWidth[<N>] = <New maximum width>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<N>: (Entero opcional)`**

Número de la miniatura cuya anchura máxima debe modificarse. Si no se especifica este parámetro, se modifica el ancho máximo de la primera miniatura.

**`<New maximum width>: (Integro)`**

Nuevo ancho máximo (en píxeles) para la miniatura especificada. Esta anchura debe estar comprendida entre 1 y 65535.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **MaxThumbnailWidth** Property está disponible:

- en HFSQL Classic: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: sólo para los artículos que se encuentran en un archivo de datos.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **MaxThumbnailWidth** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.

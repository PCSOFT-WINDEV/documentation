


## Compression (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Compresión** Property se utiliza para gestionar la compresión de datos durante una conexión (HFSQL Client/Server, etc.). Es posible:

- Averigüe si la conexión transmite datos comprimidos.

- Definir si los datos transmitidos entre el servidor y el cliente deben ser comprimidos. En este caso, si la conexión está actualmente abierta, se cerrará y se reabrirá con los nuevos parámetros durante el primer acceso a un archivo que utilice esta conexión.




**Recordatorio**: La compresión de datos se utiliza para reducir el volumen de datos que fluyen a través de los medios de transmisión (esta característica es interesante para las transmisiones facturadas en función del volumen de datos transmitidos).. La compresión de datos es compatible con la versión 14 del servidor HFSQL. Para obtener más información, consulte [Marcos de compresión](../WDLang4/1000017306.md).

**Observación**: La conexión puede corresponder a:

- una conexión descrita en la editor de análisis,

- una conexión descrita por [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- una conexión descrita por un [Conexión](../WDLang4/1514073.md) variable.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
// Describe the connection
MyConnection.User = "USER"
MyConnection.Password = "PASSWORD"
MyConnection.Server = "MYSERVER" 
MyConnection.Database = "Test"
MyConnection.Provider = hAccessHFClientServer
MyConnection.Access = hOReadWrite
MyConnection.ExtendedInfo = "Extended information"
MyConnection.CursorOptions = hClientCursor
MyConnection.Compression = True

HOpenConnection(MyConnection)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la conexión transmite datos comprimidos

`<Result> = <Connection>.Compression`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los datos transmitidos están comprimidos,

- <u><u><u><u>False</u></u></u></u> si los datos transmitidos no están comprimidos.




**Observación**: La gestión de la compresión está disponible a partir de la versión 14 del servidor HFSQL.

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por el tipo [Conexión](../WDLang4/1514073.md).  





<a name="SYNTAX2"></a>

### Definir la compresión de datos para una conexión

`<Connection>.Compression = <Data compression>`
---

**`<Connection>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por el tipo [Conexión](../WDLang4/1514073.md).




**`<Data compression>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para comprimir los datos,

- <u><u><u><u>False</u></u></u></u> para no comprimir los datos.




**Observación**: La gestión de la compresión está disponible a partir de la versión 14 del servidor HFSQL.




## CtInitialCatalog (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtInitialCatalog** Property se utiliza para get la base de datos Default definida para el acceso. Tiene la capacidad de identificar la base de datos Default definida para la conexión por ejemplo.

Esta información fue especificada:

- cuando se creó la conexión en la editor de análisis,

- cuando la conexión fue definida en [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).




Este Property puede utilizarse para un archivo de datos o para una conexión definida en el editor de análisis o mediante programación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the database that will be used on the data source
// of "MyConnection"
Info(MyConnection.CtInitialCatalog)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Initial Catalog> = <File name/connection>.CtInitialCatalog`
---

**`<Initial Catalog>: (Cadena de caracteres)`**



- Base de datos que se utilizará en la fuente de datos de la conexión.

- Vacíe la cadena ("") si la conexión no requiere una base de datos específica.




**`<File name/connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre de la conexión utilizada. Este nombre se definió en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **CtInitialCatalog** Property se utiliza para get la base de datos Default de una conexión. Este Property se mantiene para la compatibilidad retroactiva. Se recomienda utilizar la [Base de datos](../Proprietes/2512101.md) Property para leer y modificar la base de datos de una conexión.




## PhoneNumber (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PhoneNumber** Property se utiliza para obtener y cambiar un número Phone. Esta propiedad se puede utilizar en:

- una variable de tipo [gglContact](../WDLang5/1000017434.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [gpwUser](../WDLang6/1000021088.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WBAdminAccount](../WDLang2/1410089382.md).

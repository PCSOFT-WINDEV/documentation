


## BackgroundColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BackgroundColor** Property se utiliza para obtener y cambiar el fondo color de un elemento. Esta propiedad se puede utilizar en:

- los mandos de una ventana o Page. Para obtener más información, consulte la ayuda sobre la propiedad [BackgroundColor](../Proprietes/2510022.md).

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [xlsCell](../WDLang5/1000017472.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [GanttTask](../WDLang1/1000020881.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docCell](../WDLang1/1000022927.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docFragment](../WDLang1/1000022482.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docParagraph](../WDLang1/1000022483.md) (características de formato).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Token](../WDLang1/1000023942.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Toast](../WDLang1/1000024638.md).

- una variable de tipo [MarkerImage](../WDLang3/1000025796.md).

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Diagrama](../WDLang1/1410088055.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

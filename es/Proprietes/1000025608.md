
## InternalWindowPopup (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InternalWindowPopup** Property se utiliza para obtener y modificar el nombre de la ventana interna utilizada para personalizar el contenido de las ventanas emergentes Marker en un control Mapa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MAP_Map.InternalWindowPopup = "IW_Popup"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la ventana interna utilizada para personalizar las ventanas emergentes de Marker en el control Mapa

`<Result> = <Map control>.InternalWindowPopup`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la ventana interna utilizada para la Popup,

- Cadena vacía ("") si la ventana Popup utiliza una visualización predeterminada.




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar.   


<a name="SYNTAX2"></a>

### Modificación de la ventana interna utilizada para personalizar las ventanas emergentes de Marker en el control Mapa

`<Map control>.InternalWindowPopup = <New internal window>`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<New internal window>: (Cadena de caracteres)`**



- Nombre de la ventana interna utilizada para la Popup,

- Cadena vacía ("") si la ventana Popup utiliza una visualización predeterminada.










<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se especificó un nombre de ventana interna para la propiedad **InternalWindowPopup**, la ventana interna se cargará cada vez que se muestre una ventana Popup en el control Mapa. La variable [Marker](../WDLang3/1000019940.md) que corresponde a la ventana Popup se pasa al evento "Declaraciones globales" de la ventana interna a abrir. Este parámetro se pasa por referencia y se considera como una variable global a la ventana.

- La ventana interna especificada se abre en modo de "solo visualización": el usuario no puede interactuar con los controles de la ventana interna.

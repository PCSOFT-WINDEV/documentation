


## BottomIndent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BottomIndent** Property obtiene y establece el espacio en la parte inferior del texto (margen inferior) en un control. 

Esta propiedad está disponible en: 

- Controles campo de entrada.

- Controles estáticos.




**Observación**: Esta Property equivale a fijar el margen inferior: 

- Para los controles estáticos: el margen inferior se define en la pestaña "Estilo" de la ventana del control description ("Bordes exteriores", "Márgenes", "Margen inferior").

- Para un control Edit, el margen inferior se define en la pestaña "Estilo" de la ventana del control description ("Área de entrada", "Márgenes", "Margen inferior").



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define a margin equal to 4 pixels at the top and bottom of Static control
STC_Date.BottomIndent = 4
STC_Date.TopIndent = 4
```
<a name="Example2"></a>

```wl
// Keep a 5-pixel margin at the top and bottom of the input area
EDT_CarName.BottomIndent = 5
EDT_CarName.TopIndent = 5
```



<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el valor del margen inferior

`<Result> = <Control used>.BottomIndent`
---

**`<Result>: (Integro)`**

Número de píxeles del margen inferior. La sangría es de set a 0 píxeles por Default (sin sangría).  

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Estático, 

- Edita control.





<a name="SYNTAX2"></a>

### Cambiar el valor del margen inferior

`<Control used>.BottomIndent = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Estático, 

- Edita control.




**`<New value>: (Integro)`**

Nuevo valor para el margen inferior (en píxeles). Este valor debe ser positivo.

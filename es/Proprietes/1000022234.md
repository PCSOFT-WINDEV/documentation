
## HorizontalAlignment (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HorizontalAlignment** Property se utiliza para get y cambiar la alineación horizontal de un control o texto. Esta propiedad se puede utilizar en: 

- los mandos de una ventana o Page. Para obtener más información, consulte [HorizontalAlignment](../Proprietes/2510023.md). 

- una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picText](../WDLang1/1000025178.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagText](../WDLang1/1410088393.md).

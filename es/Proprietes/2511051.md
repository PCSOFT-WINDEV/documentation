


## BottomMargin (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BottomMargin** Property se utiliza para:

- Descubra el margen inferior de una reporte.

- Modificar el margen inferior de una reporte.




**Atención**: El margen inferior de un reporte sólo puede ser modificado en el reporte "Opening" Event.

**Observación**: Los diferentes márgenes de una reporte se definen en el formato de la página ("Layout" en el menú contextual de la reporte).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MargeBasse%20-%20HC%20N%B0001.GIF)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the bottom margin of "RPT_Customer" report
RPT_Customer.BottomMargin = 25
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el margen inferior de una reporte

`<Result> = <Report used>.BottomMargin`
---

**`<Result>: (Real)`**

Margen inferior (en milímetros) de la reporte especificada.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el margen inferior de una reporte

`<Report used>.BottomMargin = <New bottom margin>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte a utilizar.

**`<New bottom margin>: (Real)`**

Nuevo margen inferior (en milímetros) para la reporte especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Margen físico/Margen lógico
<a name="margen_fisicomargen_logico_ELTPARAGRAPHE000063"></a>

Cada impresora incluye un margen físico donde no se permite la impresión. Las propiedades **BottomMargin**, [RightMargin](../Proprietes/2511053.md), [LeftMargin](../Proprietes/2511048.md) y [TopMargin](../Proprietes/2511043.md) se utilizan para definir márgenes de impresión "lógicos".

- **Si el margen físico es mayor que el margen lógico**: El margen lógico se ignora. La reporte será comprimida para ser impresa en una sola página.. Para evitar la compresión reporte, seleccione "Imprimir reporte tal cual" en la ventana reporte description (pestaña "Opciones").

- **Si el margen físico es menor que el margen lógico**: Se tiene en cuenta el margen lógico. El margen físico se ignora.




[iEndPrinting](../WDLang5/3046053.md) anula los márgenes.
<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000092"></a>

El **BottomMargin** Property sólo se aplica a los informes.




## Memory (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Memory** determina si el control especificado se rellena mediante programación o está basado en un archivo de datos.

La propiedad **Memory** se aplica a los controles List Box, ListView, Combo Box, Tabla, Taba TreeView o Looper (dependiendo del producto utilizado: WINDEV, WINDEV Mobile o WEBDEV).

**Recordatorio**: 

- Controles List Box, ListView, Combo Box, Table, TreeView Table y Looper **poblados programáticamente** manipular datos almacenados en memoria. 

- Los controles List Box, ListView, Combo Box, Tabla, Tabla TreeView y Looper **basados en un archivo de datos** manipulan datos de archivos de datos o consultas.




**Importante: Este Property se mantiene por compatibilidad retroactiva**. Se recomienda utilizar el [FillType](../Proprietes/1000017723.md) Property. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF LIST_List1.Memory = True THEN
	ListAdd(LIST_List1, "Paris")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Control used>.Memory`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el contenido del control proviene de las adiciones realizadas por la programación.

- <u><u><u><u>False</u></u></u></u> si el contenido del control proviene de un archivo de datos.




**`<Control used>: (Nombre del control)`**

Nombre del control (List Box, ListView, Combo Box, Looper, Table o TreeView tabla) a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Controles de cuadro de lista, ListView, Combo Box y tabla basados en un archivo de datos
<a name="controles_cuadro_lista_listview_combo_box_tabla_basados_archivo_datos_ELTPARAGRAPHE000083"></a>

Las siguientes propiedades definen los registros que se mostrarán.

- [BrowsedFile](../Proprietes/2510040.md): Nombre del archivo de datos de la base.

- [BrowsedItem](../Proprietes/2510111.md): Nombre del elemento utilizado para la navegación (la ordenación se realizará en este elemento).

- [filtros](../Proprietes/2510042.md): Si este filtro no está vacío, sólo se mostrarán los registros correspondientes al filtro.

- [AutoBuscar](../Proprietes/2510093.md): Indica si se utilizan los eventos de navegación associated con el elemento.



<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000103"></a>

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En Java, el **Memory** Property sólo se aplica a los controles de los cuadros de lista.




## MaxWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxWidth** Property se utiliza para get y cambiar el ancho máximo de una control o ventana.

Por Default, la anchura máxima:

- de un control es igual a 2 147 483 647 (0x7FFFFFFF). Este valor equivale a "sin límite de tamaño máximo".

- de una ventana es igual a 20 000. Sin embargo, el tamaño de las ventanas no debe exceder el tamaño de la pantalla.




**Recordatorio**: La anchura máxima de una control o ventana se utiliza durante las siguientes operaciones:

- Ventana redimensionada por el usuario.

- Manejo de anclas.




La anchura máxima de un control o ventana no se utiliza cuando la ventana se redimensiona mediante programación (con la función [anchura](../Proprietes/2510059.md) Property o [WinSize](../WDLang1/3038020.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the maximum width of a control
EDT_Edit1.MaxWidth = 200
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la anchura máxima de un elemento

`<Current maximum width> = <Element used>.MaxWidth`
---

**`<Current maximum width>: (Integro)`**

Ancho máximo del elemento especificado (incluyendo la Border) expresado en píxeles.

**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento (ventana o control) utilizado.  


<a name="SYNTAX2"></a>

### Modificar la anchura máxima de un elemento

`<Element used>.MaxWidth = <New maximum width>`
---

**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento (ventana o control) utilizado.

**`<New maximum width>: (Integro)`**

Nueva anchura máxima del elemento (incluyendo la Border) expresada en píxeles.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Ancho inicial (control y ventana)
<a name="ancho_inicial_control_ventana_ELTPARAGRAPHE000087"></a>

La anchura inicial (definida en el editor) es devuelta por la [InitialWidth](../Proprietes/2510067.md) Property.

En la mayoría de los casos, la anchura de un objeto depende de las propiedades **MaxWidth** y [MinWidth](../Proprietes/2510062.md). Sin embargo, esta restricción deja de aplicarse cuando se modifica la anchura mediante programación (con la función [anchura](../Proprietes/2510059.md) Property o la función [WinSize](../WDLang1/3038020.md)).

### Límites
<a name="limites_ELTPARAGRAPHE000112"></a>

El **MaxWidth** Property se aplica únicamente a los siguientes elementos:

- Botón.

- Control Estático.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Mesa.

- Supercontrol.

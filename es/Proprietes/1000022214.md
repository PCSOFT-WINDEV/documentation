
## ClientSecret (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ClientSecret** Property se utiliza para get el secreto del cliente definido cuando se declaró o registró la aplicación. Esta propiedad es utilizada por:

- una variable de tipo [gglConnection](../WDLang5/1000018009.md). 

- una variable de tipo [OAuth2Parameters](../WDLang3/1000022212.md). 

- una variable de tipo [OpenIDParameters](../WDLang3/1000023561.md).


## NumberAccessiblePages (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NúmeroDePáginasAccesibles** Property se utiliza para get el número de páginas cargadas actualmente en un lector de PDF o control Procesador de texto.

Al abrir un archivo PDF que contiene un gran número de páginas, el PDF control carga cada página sucesivamente: algunas páginas no son accesibles inmediatamente. Este Property devuelve el número de páginas PDF actualmente cargadas sin tener que esperar al final de la carga.

El comportamiento es idéntico para un archivo DOCX abierto por un control Procesador de texto. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the number of pages currently loaded in the PDF Reader control
Trace(PDF_Reader.NumberAccessiblePages)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Control used>.NumberAccessiblePages`
---

**`<Result>: (Integro)`**

Número de páginas actualmente cargadas en el lector de PDF control o en el control Procesador de texto. 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Lector PDF, 

- Control Procesador de texto. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El [NumberPage](../Proprietes/2510084.md) Property espera a que se cargue el archivo PDF o DOCX y devuelve el número total de páginas. Este Property está bloqueando.

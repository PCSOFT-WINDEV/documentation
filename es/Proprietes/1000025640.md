<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Polygon** puede corresponder a:

- una propiedad de una variable de tipo [ocrTextBlock](../WDLang6/1000025639.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [aiImageResult](../WDLang6/1410087965.md).

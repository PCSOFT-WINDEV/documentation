
## Comment (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Comment** permite gestionar los comentarios. Esta propiedad se puede utilizar en:

- una variable de tipo [sfProcessSubmitRequest](../WDLang5/1000018516.md).

- una variable de tipo [sfProcessWorkitemRequest](../WDLang5/1000018520.md).

- una variable de tipo [Cookie](../WDLang3/1000019222.md).

- una variable de tipo [InfoVersion](../WDLang1/1000018905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docProperty](../WDLang1/1000022945.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagShape](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagSelection](../WDLang1/1410088172.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoOperationsOption](../WDLang4/1410089271.md).




## PuntuaciónSensible (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**..PunctuationSensitive** se utiliza para configurar el modo de gestión de la puntuación (coma, punto, etc.) en las diferentes búsquedas realizadas en el elemento o en el índice de texto completo (por [HReadSeek](../WDLang4/3044050.md), [HReadSeekFirst](../WDLang4/3044036.md) y [HReadSeekLast](../WDLang4/3044034.md) por ejemplo).. Es posible:

- Averigüe si la puntuación se tiene en cuenta en las búsquedas realizadas sobre este elemento (definido en la editor de análisis o definido por la programación).

- Definir el modo de gestión de la puntuación de un elemento (cuando el elemento se describe por programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the management mode of punctuation for the Orders.Name item
Name..PunctuationSensitive = True
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la puntuación se tiene en cuenta

`<Result> = <Data file>.<Item>..PunctuationSensitive`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las búsquedas realizadas en este elemento son sensibles a la puntuación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres (con o sin comillas))`**

Nombre del archivo de datos manipulado. Este nombre se definió en la editor de análisis o por el tipo de [Descripción del archivo](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres (con o sin comillas))`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o por el tipo de [Descripción de elemento](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Definición del modo de gestión de la puntuación (elemento descrito por la programación)

`<Item>..PunctuationSensitive = <New management mode>`
---

**`<Item>: (Cadena de caracteres (con o sin comillas))`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo de [Descripción de elemento](../WDLang4/1514071.md).

**`<New management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la puntuación debe tenerse en cuenta para las búsquedas realizadas en este elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Búsqueda realizada sobre un elemento "sensible a la puntuación
<a name="busqueda_realizada_sobre_elemento_sensible_puntuacion_ELTPARAGRAPHE000099"></a>

Cuando se realiza una búsqueda en un elemento "sensible a la puntuación", sólo se encontrarán las frases con caracteres de puntuación coincidentes.. Por lo tanto, si el nombre del cliente es "Mr.Smith", la búsqueda se realiza en:

- "Sr. Smith" encontrará el registro correspondiente,

- "Sr. Smith" no encontrará ningún registro coincidente.



<a name="NOTE0_2"></a>


### Configurar las diferentes opciones de búsqueda de un elemento
<a name="configurar_las_diferentes_opciones_busqueda_elemento_ELTPARAGRAPHE000109"></a>

- Para manejar los caracteres acentuados durante las búsquedas, use [... AcentuaciónSensible](../Proprietes/2512077.md).

- Para gestionar el caso durante las búsquedas, utilice [..Sensible a mayúsculas y minúsculas](../Proprietes/2512085.md).

- Para gestionar la puntuación durante las búsquedas, utilice **..PuntuaciónSensible**.


## spDerivation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spDerivation** permite:

- Obtener la función de derivación de claves utilizada para el hash de un campo de tipo "Contraseña". Esta propiedad puede utilizarse para campos de tipo "Contraseña" definidos en el editor de análisis o mediante programación. 

- Establecer la función de derivación de claves utilizada para el hash (solo para los campos de tipo "Contraseña" definidos mediante programación). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile file
...

// Describe the "Password" item
MyItem is Item Description
MyItem.Name = "Password"
MyItem.Type = hItemSecurePassword
MyItem.spDerivation = SP_PBKDF2
MyItem.spHash = SP_SHA2_256
MyItem.spIterationCount = 20000
MyItem.spLength = 256

// Validate the description of the "Password" item
HDescribeItem(AFile, MyItem)

// Validate the description of AFile file
...
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la función de derivación de claves utilizada (campo "Contraseña")

`<Derivation function> = <Data file>.<Item>.spDerivation`
---

**`<Derivation function>: (Constante de tipo Integer)`**

Función de derivación de claves utilizada para el hash de un campo de tipo "Contraseña": 



|   |   |
| --- | --- |
| *SP_PBKDF2* | Se utiliza la función de derivación de claves basada en contraseña 2 (PBKDF2). |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Establecer la función de derivación de claves utilizada (campo "Contraseña" descrito mediante programación)

`<Item>.spDerivation = <Derivation function>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Item Description](../WDLang4/1514071.md).

**`<Derivation function>: (Constante de tipo Integer)`**

Función de derivación de claves utilizada para el hash de un campo de tipo "Contraseña": 



|   |   |
| --- | --- |
| *SP_PBKDF2* | Se utiliza la función de derivación de claves basada en contraseña 2 (PBKDF2). |

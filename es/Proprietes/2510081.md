


## NbLinesPerPage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**NbLinesPerPage** se utiliza para:

- get y cambiar el número máximo de filas por página en un control Tabla.

- get y cambiar el número máximo de filas por página en un control Looper. 
	Atención: si el control Looper tiene varias columnas, el **NbLinesPerPage** Property corresponde al número total máximo de ocurrencias mostradas en el control Looper (número de columnas multiplicado por el número de filas).




**Recordatorio**: El número de filas por página es el número máximo de filas de una Tabla o control Looper que se pueden mostrar en una página. Esta opción se utiliza para optimizar el tamaño de las páginas. Este número está definido en el control description:

- Para los controles del Looper: Pestaña "General" del control description.

- Para los controles de la mesa: Pestaña "Detalles" del control description.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays 20 rows of the Looper control on a page
LOOP_Looper1.NbLinesPerPage = 20
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número máximo de filas por página

`<Number of rows> = <Control used>.NbLinesPerPage`
---

**`<Number of rows>: (Integro)`**

Número máximo de líneas actualmente definidas.

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: control Tabla o control Looper.  


<a name="SYNTAX2"></a>

### Modificar el número máximo de filas por página

`<Control used>.NbLinesPerPage = <New number of rows>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: control Tabla o control Looper. 

**`<New number of rows>: (Integro)`**

Número máximo de filas por página. Este número no debe ser igual a 0.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- No es necesario llamar a [TableDisplay](../WDLang1/3074003.md) o [LooperDisplay](../WDLang2/3083001.md) para tener en cuenta el nuevo número máximo de filas por página.

- Esta Property no tiene efecto si el Looper y/o control Tabla está en modo Ajax.

- Esta Property no está disponible para los controles de Tabla TreeView.


## InitialAnimation (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **InitialAnimation** determina si la animación de un control Imagen, Botón, Estático o Carrusel está activada cuando se crea el control. El estado inicial de la animación se define en la ventana de descripción del control (pestaña "Detalles").
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=AnimationInitiale%20-%20HC%20N%B0001.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
// Reset the image to its initial state
IMG_Image1.Animation = IMG_Image1.InitialAnimation
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Status> = <Control used>.InitialAnimation`
---

**`<Status>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la animación está habilitada en el control Imagen, Botón, Carrusel o Estático cuando se crea,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 


De forma predeterminada, la animación se establece en <u><u><u><u>True</u></u></u></u>.

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede corresponder a: 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) un control Imagen, 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) un control Botón,

- un control Estático, 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) un control Carrusel. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Animar una imagen
<a name="animar_una_imagen_ELTPARAGRAPHE000062"></a>

La imagen a animar en el control Imagen puede ser:

- una imagen animada (GIF animado, por ejemplo).

- una imagen animada por WINDEV. En este caso, el tipo de animación puede definirse en la ventana de descripción del control Imagen (pestaña "Detalles") o mediante programación con la función [AnimationSetMode](../WDLang1/3017001.md).




Animar una imagen en un control Botón: Solo se puede asociar una imagen animada al control (GIF animado, por ejemplo).
<a name="NOTE0_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Animar texto en un control Estático
<a name="animar_texto_control_estatico_ELTPARAGRAPHE000079"></a>

Para animar el texto en un control Estático, debe elegir la animación:

- en la ventana de descripción del control Estático (pestaña "Detalles"),

- mediante programación con la función [AnimationSetMode](../WDLang1/3017001.md).



<a name="NOTE0_3"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000092"></a>

La propiedad **InitialAnimation** solo se aplica a: 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) controles Imagen, Botón, Estático, Barra de progreso y Carrusel en una ventana.

- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) controles Estático en una ventana.


## Orientation (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Orientation** obtiene y establece el ángulo de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [Font](../Motscles/1514045.md). Para obtener más información, consulte la ayuda sobre la propiedad [Orientation](../Proprietes/1410086632.md). 

- una variable de tipo [Marca de agua](../WDLang5/1000019922.md).

- una variable de tipo [Background](../WDLang1/1000022038.md).

- una variable de tipo [MapImage](../WDLang3/1000025523.md).

- una variable de tipo [diagShape](../WDLang1/1410088082.md).

- una variable de tipo [diagSelection](../WDLang1/1410088172.md).


## FontStrikeOut (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **FontStrikeOut** Property se utiliza para determinar si un texto está tachado. Esta propiedad se puede utilizar en: 

- los controles. Para obtener más información, consulte [FontStrikeOut](../Proprietes/2510091.md). 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md).

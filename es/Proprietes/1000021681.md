


## TransparencyMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TransparencyMode** Property se utiliza para get y cambiar el modo de transparencia de un Image.

Observación: El modo de transparencia se define en el editor a través de la opción "Transparencia" que se encuentra en la ventana de controles de description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the transparency mode of the image found in the "IMG_CarImage" control
IMG_CarImage.TransparencyMode = trsImage
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el modo de transparencia actual

`<Current transparency> = <Control used>.TransparencyMode`
---

**`<Current transparency>: (Constante de tipo Integer)`**

El actual modo de transparencia de la Image:


|   |   |
| --- | --- |
| *trsTopLeftCorner* | Los puntos con el mismo color que el del punto superior izquierdo de la imagen serán Transparent. |
| *trsImage*  | La transparencia de Image se mantiene. |
| *trsMagenta* | El color magenta que se encuentra en la imagen será Transparent durante la visualización. |



**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Imagen.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Columna Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Background de un control Gráfico.





<a name="SYNTAX2"></a>

### Modificando el modo de transparencia

`<Control used>.TransparencyMode = <New mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- control Imagen.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Columna Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Background de un control Gráfico.




**`<New mode>: (Constante de tipo Integer)`**

El nuevo modo de transparencia de la Image:


|   |   |
| --- | --- |
| *trsTopLeftCorner* | Los puntos con el mismo color que el del punto superior izquierdo de la imagen serán Transparent. |
| *trsImage*  | La transparencia de Image se mantiene. |
| *trsMagenta* | El color magenta que se encuentra en la imagen será Transparent durante la visualización. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Antecedentes Image de un control Gráfico
<a name="antecedentes_image_control_grafico_ELTPARAGRAPHE000097"></a>

Cuando se modifica el modo de transparencia de la Image de fondo que se encuentra en una control Gráfico, la pantalla del gráfico debe actualizarse (mediante [grDraw](../WDLang3/3042026.md), por ejemplo) para que aparezca el cambio de modo de transparencia.
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000107"></a>

El **TransparencyMode** Property se aplica sólo a los siguientes controles:

- control Imagen.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Columna Image.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Background de un control Gráfico.

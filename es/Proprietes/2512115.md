


## Thumbnail (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Thumbnail** Property se utiliza para: 

- get el valor de la miniatura de un artículo. Este punto se definió en la editor de análisis o a través de la programación.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Value of first thumbnail of Customer.Photo item
ThumbnailValue is string
ThumbnailValue = Customer.Photo.Thumbnail[1]
// equivalent to ThumbnailValue = Customer.Photo.Thumbnail

// Null value?
IF ThumbnailValue <> "" THEN
	Info("The value of this thumbnail is: " + ThumbnailValue)
ELSE
	ErrorString = Customer.Photo.InfoThumbnailGeneration[1]
	Error("Generation error", ErrorString)
END
```


<a name="Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el valor de una miniatura de un artículo

`Value> = <Data file>.<Item>.Thumbnail`
---

**`<Value>: (Cadena de caracteres)`**

Image de la miniatura especificada en formato de nota binaria.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

## Observaciones


### Imagen en miniatura del artículo
<a name="imagen_miniatura_del_articulo_ELTPARAGRAPHE000106"></a>

El **Thumbnail** Property está disponible:

- en modo HFSQL Classic y HFSQL Mobile: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: sólo para los artículos que se encuentran en un archivo de datos.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **Thumbnail** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.

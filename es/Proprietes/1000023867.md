
## RootNode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RootNode** Property se utiliza para obtener los nodos raíz de un HTML o XML Document. Esta propiedad se puede utilizar en:

- una variable de tipo [xmlDocument](../WDLang5/1000018782.md). 

- una variable de tipo [htmlDocument](../WDLang5/1000025910.md).

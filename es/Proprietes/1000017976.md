
## Text (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Texto** Property se utiliza para get y modificar el texto. Esta propiedad se puede utilizar en:

- los artículos de un archivo de datos: se utiliza para determinar si un elemento es un elemento de texto. Para obtener más información, consulte [Texto Property](../Proprietes/2512089.md).

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

- una variable de tipo [Watermark](../WDLang5/1000019922.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una variable de tipo [Mosaico](../WDLang6/1000020713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElement](../WDLang1/1000022484.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docFragment](../WDLang1/1000022482.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Toast](../WDLang1/1000024638.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfTextElement](../WDLang6/1000024556.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picText](../WDLang1/1000025178.md).

- una variable de tipo [ocrTextBlock](../WDLang6/1000025639.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagShape](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagSelection](../WDLang1/1410088172.md).

- una variable de tipo [kbCard](../WDLang1/1410089167.md).

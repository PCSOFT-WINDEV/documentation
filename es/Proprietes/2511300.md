
## Source (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Fuente** Property se utiliza para get y cambiar la fuente de un elemento. Esta propiedad se puede utilizar en:

- una control Gráfico o una conexión. Para obtener más información, consulte [Fuente Property en una carta o conexión](../Proprietes/2510147.md).

- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

- una variable de tipo [rssEntry](../WDLang5/1000017792.md). 

- la estructura [Email](../WDLang3/3032029.md)

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [OpenIDIdentity](../WDLang3/1000023567.md).

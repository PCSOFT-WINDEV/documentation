


## AlternativeText (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AlternativeText**Property se utiliza para get y editar el texto alternativo de los controles en una página.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the product image from the Product data file
IMG_Product = Product.Image
IMG_Product.AlternativeText = Product.Name
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubriendo el texto alternativo associated con el control

`<Result> = <Control used>.AlternativeText`
---

**`<Result>: (Cadena de caracteres)`**

Texto alternativo de control.

**`<Control used>: (Nombre del control)`**

Nombre de control cuyo texto alternativo se solicita. Este control puede ser: 

- un control Imagen o un Image clicable,

- el Captcha control,

- a control Miniatura,

- a control Código de barras,

- a control Gráfico. 





<a name="SYNTAX2"></a>

### Modificando el texto alternativo associated con el control

`<Control used>.AlternativeText = <Alternative text>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser: 

- un control Imagen o un Image clicable,

- el Captcha control,

- a control Miniatura,

- a control Código de barras,

- a control Gráfico. 




**`<Alternative text>: (Cadena de caracteres)`**

Nuevo texto alternativo del control. Este texto es un texto simple, sin etiquetas HTML. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta Property es equivalente a los controles [Pie de foto](../Proprietes/2510053.md) Property en Image, Image clicable, Captcha, Thumbnail, Bar Code y Chart.

- El texto alternativo de un Image (atributo "alt" en HTML) indica el texto mostrado por el navegador cuando el Image no está disponible. Este texto también se utiliza en lugar del Image para la accesibilidad de las páginas.

- El texto alternativo también puede ser definido en el editor en la ventana de description del control. Por ejemplo: 

	- en la pestaña "Ayuda" para los controles de Image, Código de Barras y Miniaturas,

	- en la pestaña "Detalles" de los controles del Captcha,

	- en la pestaña "UI" para los controles de los gráficos.




## NbReceived (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de parámetros que recibe realmente un Procedure.  

**Observación**: La propiedad [Conde](../Proprietes/2510097.md) devuelve el número de parámetros que se pueden utilizar en el Procedure.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Call the procedure
MyProc("A")
```

```wl
// Declare the procedure
PROCEDURE MyProc(p1, po2 = "Y", po3 = "Z")
Info(MyParameters.NbReceived) // Returns 1
Info(MyParameters.Count) // Returns 3
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <MyParameters>.NbReceived`
---

**`<Result>: (Integro)`**

El número de parámetros recibidos por el Procedure. Este número ignora los parámetros opcionales para los que no se especificó ningún valor durante la llamada.

**`<MyParameters>: (Palabra clave)`**

Palabra clave del lenguaje WLanguage utilizada para designar todos los parámetros de una Procedure.




## URL (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **URL** Property se utiliza para get y cambiar una URL. Esta propiedad se puede utilizar en:

- un enlace, un botón, una imagen clicable o un iFrame control. Para obtener más información, consulte [URL](../Proprietes/2510132.md).

- una variable de tipo [rssEnclosure](../WDLang5/1000017789.md).

- una variable de tipo [rssImage](../WDLang5/1000017799.md).

- una variable de tipo [rssSource](../WDLang5/1000017803.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpDevice](../WDLang3/1000020903.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpImage](../WDLang3/1000020905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpService](../WDLang3/1000020908.md).

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

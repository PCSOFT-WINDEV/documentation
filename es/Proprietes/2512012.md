


## CtDescribedPassword (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtDescribedPassword** Property permite set la contraseña utilizada para abrir el archivo de datos en una conexión OLE DB (archivo definido en el código solamente). 


**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describe el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Set the password to open the Customer data file
Customer.CtDescribedPassword = "Password"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Data file>.CtDescribedPassword = <Password>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Nombre definido mediante el tipo [Expediente description](../WDLang4/1514065.md).

**`<Password>: (Cadena de caracteres)`**

Contraseña del archivo que debe ser abierto por una conexión OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La [Contraseña](../Proprietes/2510086.md) Property se utiliza para set la contraseña associated con un usuario y una conexión.

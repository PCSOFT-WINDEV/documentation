


## ModificationDurationAPT (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ModificationDurationAPT** se utiliza para determinar y especificar si el usuario puede modificar la duración de un Appointment en un Programador o control Agenda. 

Esta propiedad equivale a la opción "Modificación de la duración" que se encuentra en la pestaña "Detalles" de la ventana description de los controles del Programador y el Organizador.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF gbModifMode = True THEN
	ORG_MyOrganizer.MovementAPT = True
	ORG_MyOrganizer.ModificationDurationAPT = True
	ORG_MyOrganizer.DirectInputAPT = True
	ORG_MyOrganizer.PeriodSelection = True
ELSE
	ORG_MyOrganizer.MovementAPT = False
	ORG_MyOrganizer.ModificationDurationAPT = False
	ORG_MyOrganizer.DirectInputAPT = False
	ORG_MyOrganizer.PeriodSelection = False
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si el usuario puede modificar la duración de un Appointment

`<Result> = <Control used>.ModificationDurationAPT`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario puede modificar la duración de las citas,

- <u><u><u><u>False</u></u></u></u> si el usuario no puede modificar la duración de las citas.




**`<Control used>: (Nombre del control)`**

Nombre del Organizador o control Planificador utilizado.  


<a name="SYNTAX2"></a>

### Permitir o impedir que el usuario cambie la duración de las citas

`<Control used>.ModificationDurationAPT = <Modifiable Duration>`
---

**`<Control used>: (Nombre del control)`**

Nombre del Organizador o control Planificador utilizado.

**`<Modifiable Duration>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para permitir al usuario modificar las citas,

- <u><u><u><u>False</u></u></u></u> para no permitir al usuario modificar las citas.

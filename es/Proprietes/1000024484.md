
## Index (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Index** permite obtener el índice de un elemento. Esta propiedad se puede utilizar en:

- un artículo Record. Para obtener más información, consulte la ayuda sobre la propiedad [Índice](../Proprietes/2512040.md).

- una variable de tipo [htmlNode](../WDLang5/1000026081.md).

- una variable de tipo [kbCard](../WDLang1/1410089167.md).




## TriState (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TriState** Property se utiliza para:

- Determine si una opción de casilla de verificación admite 3 estados (marcada, desmarcada o indefinida) o 2 estados (marcada o desmarcada).

- Determinar si las casillas de verificación de una control TreeView admiten 3 estados (marcada, no marcada o indefinida) o 2 estados (marcada o no marcada)

- Cambiar el tipo de una opción en una casilla de verificación: opción con 3 estados o con 2 estados.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Interrupteur_TroisEtat.gif)
**Observación**: Si la opción soporta los tres estados, puede tomar valor:

- 0: la opción no está marcada

- 1: la opción está marcada

- -1: la opción es indefinida (Grayed)




En tiempo de ejecución, si la opción admite tres estados, un primer clic realizado sobre una opción no marcada la marcará y un segundo clic le dará un valor no especificado.

**Atención**: Un error de WLanguage ocurre si una opción que soporta 2 estados toma -1 como valor. De manera similar, un error de WLanguage ocurre si una opción de 3 estados inicializada a -1 se cambia a una opción de 2 estados.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines an unspecified option
CBOX_CheckBox1[1].TriState = True
CBOX_CheckBox1[1] = -1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si un elemento admite tres estados

`Result> = <Element name>.TriState`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la opción admite tres estados, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Opción de la control Casilla de verificación. Debe utilizarse la siguiente sintaxis: &lt;Nombre de control Casilla de verificación&gt;[&lt;Número de opción&gt;]. 

- control TreeView. 





<a name="SYNTAX2"></a>

### Gestionar los tres estados de una opción en una control Casilla de verificación

`Check Box control>[<Option number>].TriState = <Management mode`
---

**`<Check Box control>: (Nombre del control)`**

Nombre de control Casilla de verificación a utilizar.

**`<Option number>: (Integro)`**

Número de la opción a utilizar.

**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para gestionar los tres estados,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **TriState** Property se aplica únicamente a: 

- las opciones de una control Casilla de verificación.

- los controles TreeView (en modo de sólo lectura).

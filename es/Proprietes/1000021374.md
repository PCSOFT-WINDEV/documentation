


## WithClosingButton (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WithClosingButton** permite: 

- Determinar si todas las pestañas de un control Pestaña dinámica tienen un botón Cerrar. 

- Mostrar un botón Cerrar en todas las pestañas de un control Pestaña dinámica. 




Recordatorio: El botón Cerrar (X) permite a los usuarios cerrar las pestañas. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TAB_DynamicTab.WithClosingButton = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si todas las pestañas de un control Pestaña dinámica tienen un botón Cerrar

`<Result> = <Tab control>.WithClosingButton`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si todas las pestañas de un control Pestaña dinámica tienen un botón Cerrar (X),

- <u><u><u><u>False</u></u></u></u> si las pestañas de un control Pestaña dinámica no tienen un botón Cerrar (X). 




**`<Tab control>: (Nombre del control)`**

Nombre del control Pestaña a utilizar. 


<a name="SYNTAX2"></a>

### Mostrar un botón Cerrar en todas las pestañas de un control Pestaña dinámica

`<Tab control>.WithClosingButton = <Close button>`
---

**`<Tab control>: (Nombre del control)`**

Nombre del control Pestaña a utilizar. 

**`<Close button>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si las pestañas de un control Pestaña dinámica deben mostrar un botón Cerrar (X),

- <u><u><u><u>False</u></u></u></u> si las pestañas de un control Pestaña dinámica no deben mostrar un botón Cerrar (X). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El evento "Cierre de una pestaña" se ejecuta cuando el usuario hace clic en el botón Cerrar. La pestaña se cierra si el evento no impide que el usuario la cierre (con la instrucción "RESULT False" o la función [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md)).

- Si las pestañas no tienen un botón Cerrar (X), las siguientes funcionalidades automáticas de la aplicación (AAF) se desactivan del menú contextual de las pestañas:

	- Cerrar,

	- Cerrar las demás pestañas,

	- Cerrar pestañas a la derecha,

	- Cerrar pestañas a la izquierda.




- El ícono de cierre "X" no puede eliminarse ni mostrarse en una sola pestaña: todas o ninguna de las pestañas muestran el ícono de cierre.

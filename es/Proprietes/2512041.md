


## ExtendedInfo (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ExtendedInfo** Property se utiliza para get información adicional:

- sobre un archivo de datos a la que se accede por una Conector Nativo o HFSQL Client/Server.

- sobre un artículo al que accede un Conector Nativo o HFSQL Client/Server.

- sobre una conexión. En este caso, esta Property equivale a la [CtExtendedInfo](../Proprietes/2512017.md) Property.




Esta información ampliada puede modificarse para un archivo, una conexión o un elemento definido mediante programación.

Para más detalles, consulte la documentación suministrada por el proveedor de la base de datos.


**Importante**: En este momento, esta Property sólo puede ser usada con archivos de datos (y elementos) del AS/400 y con conexiones de la HFSQL Client/Server. En las conexiones HFSQL Client/Server , solo el cifrado puede ser administrado por **ExtendedInfo** Property. 


Esta información se muestra en el editor de análisis:

- en la ventana de description de un archivo de datos.

- en la ventana de description de los artículos encontrados en un archivo de datos.

- en la ventana description de una conexión: en la pestaña "Análisis", en el grupo "Conexión", haga clic en "Conexiones".




Esta Property se puede utilizar:

- en modo lectura para un fichero o un elemento definido en la editor de análisis o mediante programación.

- en modo de escritura para un archivo de datos definida mediante programación o para una conexión.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get extended information about the Customer data file
Info(Customer.ExtendedInfo)
// Get extended information about the CustomerName item in the Customer data file
Info(Customer.CustomerName.ExtendedInfo)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la información extendida

`<Extended information> = <Data file/Item/Connection>.ExtendedInfo`
---

**`<Extended information>: (Cadena de caracteres)`**

Información ampliada sobre el elemento especificado. Los distintos datos se separan mediante punto y coma (';').

**`<Data file/Item/Connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre del elemento utilizado (formato: .). Este nombre se definió en el editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

- Nombre de la conexión utilizada. Esta conexión puede corresponder a:

	- una conexión definida en el editor de análisis,

	- una conexión definida con las funciones [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

	- una conexión definida con una Variable de tipo [Conexión](../WDLang4/1514073.md).  







<a name="SYNTAX2"></a>

### Definición de la información ampliada

`<Data file/Item/Connection>.ExtendedInfo = <Extended information>`
---

**`<Data file/Item/Connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Nombre definido mediante el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre del elemento utilizado (formato: .). Nombre definido mediante el tipo [Artículo description](../WDLang4/1514071.md).

- Nombre de la conexión utilizada. Esta conexión puede corresponder a:

	- una conexión definida en el editor de análisis,

	- una conexión definida con las funciones [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

	- una conexión definida con una Variable de tipo [Conexión](../WDLang4/1514073.md).







**`<Extended information>: (Cadena de caracteres)`**

Información ampliada sobre el elemento especificado. Los distintos datos deben ir separados por punto y coma (';'). 

Observación: La lista de información extendida reconocida por OLE DB y el Conectores Nativos se presenta en [Información opcional de conexión](../WDLang4/9000124.md).

### **ExtendedInfo** Property y conexión Variable
<a name="docparampagetitleshort_property_conexion_variable_ELTPARAGRAPHE000145"></a>

Changing the encryption information of an HFSQL Client/Server connection with the **ExtendedInfo** Property automatically updates the information returned by the [Cifrado](../Proprietes/2512102.md) Property (and vice versa).

Por lo tanto, la siguiente Line de código:


```wl
MyConnection.CryptMethod = hCryptRC5_16
```
modificará la opción ENCRYPTION en la cadena de información extendida.

Del mismo modo, la siguiente Line de código:


```wl
MyConnection.ExtendedInfo = "CRYPT=RC5_16"
```
modificará el [Cifrado](../Proprietes/2512102.md) Property.

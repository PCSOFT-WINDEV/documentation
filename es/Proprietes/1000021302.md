
## SecondaryMessage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SecondaryMessage** se utiliza para obtener y cambiar el mensaje secundario que se muestra debajo del mensaje primario del Notification.

Esta propiedad está disponible en: 

- variables de tipo [Notification](../WDLang3/1000019441.md). 

- variables de tipo [notificationFormat](../WDLang3/1000021301.md).

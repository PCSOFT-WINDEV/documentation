
## Ribbon (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Ribbon** se utiliza para acceder al control Cinta de opciones del control asociado (Procesador de textos, Lector de PDF, Hoja de cálculo, Editor de imágenes, Editor de HTML o Diagram Editor control s). Esto le permite hacer visible la control Cinta de opciones o Invisible. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Hide the "Print" button in the ribbon
MyRibbon is Control <- WP_Document.Ribbon
{MyRibbon.FullName + ".BTN_WDFAA_Print", indControl}..Visible = False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Control used>.Ribbon`
---

**`<Result>: (Control)`**

control Cinta de opciones associated con el control especificado. 

**`<Control used>: (Nombre del control)`**

Nombre de la control que tiene una cinta: 

- Procesador de texto, 

- Lector PDF, 

- Hoja de cálculo,

- Editor de imágenes,

- Editor HTML.

- Editor de diagramas.




## FutureHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FutureHeight** Property se utiliza para determinar la altura de un control de tamaño automático en un reporte después de ser ampliado. Este Property se utiliza principalmente para aumentar el tamaño de un control situado al lado de un control de tamaño automático (en una forma reporte por ejemplo).

Recordatorio: la altura actual de una control es devuelta por [Altura](../Proprietes/2510050.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the control is going to be resized before it is printed
IF MyControl.Height < MyControl.FutureHeight THEN
	// Resize its neighbor
	NeighborControl.Height = MyControl.FutureHeight
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Future height> = <Control used>.FutureHeight`
---

**`<Future height>: (Integro)`**

Altura de la control (incluida la Border) expresada en milímetros cuando se amplía esta control.

**`<Control used>: (Nombre del control)`**

Nombre de la control de tamaño automático a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Atención: modo de cálculo de la altura futura
<a name="atencion_modo_calculo_altura_futura_ELTPARAGRAPHE000052"></a>

La altura futura de un control de redimensionamiento automático se calcula en función del contenido del control en el momento en que se llama a **FutureHeight** Property. 
Por lo tanto, si se llama al Property en el Event "Antes de la impresión" del bloque de cuerpo, la futura altura de uno de los controles del bloque de cuerpo ignorará las asignaciones realizadas mediante programación en el control "Antes de la impresión" Event.
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000063"></a>

- Esta Property sólo se aplica a los controles de tamaño automático que se encuentran en los informes. Un control se ajusta automáticamente si se selecciona la opción "Ajuste automático" en la ventana de descripción del control (pestaña UI). 

- Esta Property no se aplica a los controles RTF.

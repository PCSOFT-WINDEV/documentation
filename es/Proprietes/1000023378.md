


## DisplayMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DisplayMode** Property obtiene y cambia: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el modo de visualización en una control Procesador de texto,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el modo de visualización en un lector PDF control,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el modo de visualización en un editor HTML control,

- el modo de visualización en un Kanban control. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Web mode] button
WP_Doc.DisplayMode = displayModeWeb
```



```wl
IF ScreenType() = stTablet THEN
	KANBAN_TASKS.DisplayMode = displayModeKanbanMultiList
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el modo de visualización de una Office control

`<Result> = <Control used>.DisplayMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de visualización actual del control: 



|   |   |
| --- | --- |
| *displayModeContinuous* | El espacio en blanco entre las páginas se oculta. <br><br>Observación: Esta constante solo está disponible para controles Editor HTML.  |
| *displayModeHtmlSourceCode* | El código fuente HTML se muestra en el control Editor HTML.   <br><br>Observación: Esta constante solo está disponible para controles Editor HTML. |
| *displayModeKanbanMultiList* | Se pueden visualizar simultáneamente varias listas en Kanban control. Esta constante corresponde a la opción "Listas múltiples" de la pestaña "Detalles" de la ventana del control kanban description.       <br><br>Observación: Esta constante sólo está disponible para los controles Kanban. |
| *displayModeKanbanSingleList* | Sólo se mostrará una lista en el Kanban control. Scrolling puede acceder a las demás listas en horizontal. Esta constante corresponde a la opción "Lista centrada desplazable" de la pestaña "Detalles" de la ventana del control Kanban description.   <br><br>Observación: Esta constante sólo está disponible para los controles Kanban. |
| *displayModeOnePage*<br>(Valor predeterminado) | El documento se muestra tal y como se imprimirá. <br><br>Observación: Esta constante no está disponible para controles Editor HTML. |
| *displayModeTwoPages* | Las páginas del documento se muestran una al lado de la otra.  <br><br>Observación: Esta constante no está disponible para controles Editor HTML. |
| *displayModeTwoPagesWithCover* | Las páginas del documento se muestran una al lado de la otra. La primera página se establece como portada. La primera página se muestra a la derecha. <br><br>Observación: Esta constante no está disponible para controles Procesador de texto o Editor HTML. |
| *displayModeWeb* | El documento se muestra como una página Web. <br><br>Observación: Esta constante no está disponible para controles Lector PDF. |



**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Procesador de texto.

- Control Lector PDF a utilizar. 

- Control Editor HTML.  

- Control kanban.





<a name="SYNTAX2"></a>

### Cambiar el modo de visualización de un Office control

`<Control used>.DisplayMode = <New mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Procesador de texto.

- Control Lector PDF a utilizar.

- Control Editor HTML.

- Control kanban.




**`<New mode>: (Constante de tipo Integer)`**

Modo de visualización a aplicar al control: 



|   |   |
| --- | --- |
| *displayModeContinuous* | El espacio en blanco entre las páginas se oculta. <br><br>Observación: Esta constante solo está disponible para controles Editor HTML.  |
| *displayModeHtmlSourceCode* | El código fuente HTML se muestra en el control Editor HTML.   <br><br>Observación: Esta constante solo está disponible para controles Editor HTML. |
| *displayModeKanbanMultiList* | Se pueden visualizar simultáneamente varias listas en Kanban control. Esta constante corresponde a la opción "Listas múltiples" de la pestaña "Detalles" de la ventana del control kanban description.       <br><br>Observación: Esta constante sólo está disponible para los controles Kanban. |
| *displayModeKanbanSingleList* | Sólo se mostrará una lista en el Kanban control. Scrolling puede acceder a las demás listas en horizontal. Esta constante corresponde a la opción "Lista centrada desplazable" de la pestaña "Detalles" de la ventana del control Kanban description.   <br><br>Observación: Esta constante sólo está disponible para los controles Kanban. |
| *displayModeOnePage*<br>(Valor predeterminado) | El documento se muestra tal y como se imprimirá. <br><br>Observación: Esta constante no está disponible para controles Editor HTML. |
| *displayModeTwoPages* | Las páginas del documento se muestran una al lado de la otra.  <br><br>Observación: Esta constante no está disponible para controles Editor HTML. |
| *displayModeTwoPagesWithCover* | Las páginas del documento se muestran una al lado de la otra. La primera página se establece como portada y se muestra a la derecha. <br><br>Observaciones: <br><br>- Esta constante cambia el modo de visualización, pero no cambia el zoom del documento en el control Lector PDF. Para cambiar el zoom, utilice la propiedad [Zoom](../Proprietes/1000017201.md). <br><br>- Esta constante no está disponible para controles Procesador de texto o Editor HTML. <br><br><br> |
| *displayModeWeb* | El documento se muestra como una página Web. <br><br>Observación: Esta constante no está disponible para controles Lector PDF. |




## Importance (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Importance** se utiliza para obtener y cambiar el nivel de importancia de un Appointment. Esta propiedad se puede utilizar en:

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).


## Timestamp (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Timestamp** Property se utiliza para gestionar la marca de tiempo de los valores. Esta propiedad está disponible en:

- una variable de tipo [hbScanResult](../WDLang4/1000021724.md).

- una variable de tipo [hbCell](../WDLang4/1000021707.md).

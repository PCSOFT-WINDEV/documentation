


## Collapsed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **colapsado** Property se utiliza para:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) get o definir el estado (colapsado o expandido) que se utilizará cuando se añadan filas a una control Tabla TreeView. 
	Esta Property no tiene ningún efecto inmediato pero es tenida en cuenta por [TableAdd](../WDLang1/3074017.md), [TableAddLine](../WDLang1/3074006.md), [TableAddChild](../WDLang1/1000010004.md), [TableInsert](../WDLang1/3074025.md), [TableInsertLine](../WDLang1/3074026.md), [TableInsertChild](../WDLang1/1000010005.md), ...
	El estatus ampliado es el estatus de Default. 

- obtener o set el estado (colapsado/expandido) para: 

	- los descansos de controles Tabla.

	- las roturas de los controles del looper.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) los encabezados y pies de página de los controles Tabla y Looper.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Cajón de navegación.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Cinta de opciones.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Tabla dinámica en una reporte.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Define the collapsed status for the next addition in the TreeView Table control
TVT_Recipe.Collapsed = True
// Insert a leaf ("Pancakes") at position 2 below the "Desserts" node
TableInsertChild(TVT_Recipe, "Desserts", 2, "Pancakes")
```


<a name="Example2"></a>

```wl
// -- Click code of a Button control etc 
// This code is used to collapse or expand the Table control break 
// for the current row according to its status 

TABLE_NestedBreaks.BreakHeader3.Collapsed = ...
	NOT TABLE_NestedBreaks.BreakHeader3.Collapsed
```


<a name="Example3"></a>

```wl
// -- Click code of a Button control etc 
// This code is used to collapse or expand the Looper control break 
// for the row according to its status 

LOOP_NestedBreaks.BRK_BreakHeader1.Collapsed = ...
	NOT LOOP_NestedBreaks.BRK_BreakHeader1.Collapsed
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificación del estado Colapsado/Expandido utilizado al añadir un nuevo elemento en una control o el estado Colapsado/Expandido de una control

`<Result> = <Control used>.Collapsed`
---

**`<Result>: (booleano)`**

**control Tabla TreeView y las interrupciones en controles Tabla**

- <u><u><u><u>True</u></u></u></u> si se utiliza el estado Contraído para agregar la próxima fila,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Control Cajón de navegación**

- <u><u><u><u>True STRIMEULB</u></u></u></u> si el control Cajón de navegación está colapsado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**Ruptura de un control Looper**

- <u><u><u><u>True</u></u></u></u> si la ruptura está colapsada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Control Cinta de opciones**

- <u><u><u><u>True</u></u></u></u> si el control Cinta de opciones está contraído. En este caso, sólo se muestran los títulos de los paneles (como en un menú estándar).. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificación del estado Colapsado/Expandido utilizado al añadir un nuevo elemento o del estado Colapsado/Expandido de una control

`<Control used>.Collapsed = <Collapsed/Expanded>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Collapsed/Expanded>: (booleano)`**

**control Tabla TreeView y las interrupciones en controles Tabla**

- <u><u><u><u>True</u></u></u></u> si debe utilizase el estado Contraído para agregar la próxima fila,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Control Cajón de navegación**

- <u><u><u><u>True STRIMEULB</u></u></u></u> para ocultar un control Cajón de navegación, 

- <u><u><u><u>False</u></u></u></u> para expandirla. 




**Ruptura de un control Looper**

- <u><u><u><u>True</u></u></u></u> para colapsar el descanso, 

- <u><u><u><u>False</u></u></u></u> para expandirla.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Control Cinta de opciones**

- <u><u><u><u>True</u></u></u></u> para contraer la cinta de opciones. 

- <u><u><u><u>False</u></u></u></u> para expandirla.

### Averiguar el estado de visualización de un control Tabla dinámica en un reporte

`<Result> = <Pivot Table control>.Collapsed`
---

**`<Result>: (Constante de tipo Integer)`**

Visualizar el status de las líneas y columnas que se encuentran en un control Tabla dinámica en un reporte: 


|   |   |
| --- | --- |
| *pvtAllCollapsed* | El control Tabla dinámica que se encuentra en el reporte muestra el primer nivel de la fuente control que se encuentra en la ventana. |
| *pvtAllExpanded* | El control Tabla dinámica que se encuentra en el reporte muestra todos los datos de la fuente control que se encuentra en la ventana: se desglosan todas las líneas y columnas. |
| *pvtLikeTheControl* | El control Tabla dinámica que se encuentra en el reporte es idéntico a la fuente control que se encuentra en la ventana: se respetan las líneas desglosadas o colapsadas. |



**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar. Este control se encuentra en un reporte.  


<a name="SYNTAX4"></a>

### Modificar el estado de visualización de un control Tabla dinámica en un reporte

`<Pivot Table control>.Collapsed = <Display status>`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar. Este control se encuentra en un reporte.  

**`<Display status>: (Constante de tipo Integer)`**

Visualizar el status de las líneas y columnas que se encuentran en un control Tabla dinámica en un reporte: 


|   |   |
| --- | --- |
| *pvtAllCollapsed* | El control Tabla dinámica que se encuentra en el reporte muestra el primer nivel de la fuente control que se encuentra en la ventana. |
| *pvtAllExpanded* | El control Tabla dinámica que se encuentra en el reporte muestra todos los datos de la fuente control que se encuentra en la ventana: se desglosan todas las líneas y columnas. |
| *pvtLikeTheControl* | El control Tabla dinámica que se encuentra en el reporte es idéntico a la fuente control que se encuentra en la ventana: se respetan las líneas desglosadas o colapsadas. |

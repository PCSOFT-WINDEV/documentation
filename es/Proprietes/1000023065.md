
## AssociatedValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AssociatedValue** Property se utiliza para obtener el valor associated. Esta propiedad se puede utilizar en:

- una variable de tipo [Option Description](../WDLang1/1000023072.md). 

- una variable de tipo [Value description](../WDLang1/1000023061.md).

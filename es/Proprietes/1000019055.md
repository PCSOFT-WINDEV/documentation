


## MultimediaReader (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MultimediaReader** Property se utiliza para get y cambiar el reproductor multimedia utilizado actualmente por un control Multimedia para reproducir archivos multimedia.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the video of the current product 
// and display it in the Multimedia control
VIDEO_Product.Value = "http://server/GetProductVideo.awp?ID=5"

// Select the best media player 
// (if the type of video cannot be deduced from the URL)
VIDEO_Product.MultimediaReader = MultimediaReaderFlash
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinación del reproductor multimedia utilizado actualmente por una control Multimedia

`<Result> = <Multimedia control>.MultimediaReader`
---

**`<Result>: (Constante de tipo Integer)`**

Los valores posibles son:


|   |   |
| --- | --- |
| *MultimediaReaderAutomatic* <br>(valor predeterminado) | El control Multimedia elige el mejor reproductor multimedia según la extensión del archivo. |
| *MultimediaReaderFlash* | El control Multimedia utilizará el reproductor MacroMedia Flash (adaptado para archivos .FLV, .MP4, etc.) |
| *MultimediaReaderNativeBrowser* | El control multimedia utilizará el reproductor nativo HTML5. |
| *MultimediaReaderQuicktime* | El control Multimedia utilizará el reproductor QuickTime de Apple (adaptado para archivos .MOV, .MP4, etc.) |
| *MultimediaReaderWindowsMedia* | El control Multimedia utilizará el reproductor de Windows Media (adaptado para archivos .WMV, .ASF, .AVI, etc.) |



**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar


<a name="SYNTAX2"></a>

### Modificación del reproductor multimedia utilizado actualmente por una control Multimedia

`<Multimedia control>.MultimediaReader = <Multimedia player>`
---

**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar

**`<Multimedia player>: (Constante de tipo Integer)`**

Tipo de reproductor multimedia que se va a utilizar: 


|   |   |
| --- | --- |
| *MultimediaReaderAutomatic* <br>(valor predeterminado) | El control Multimedia elige el mejor reproductor multimedia según la extensión del archivo. |
| *MultimediaReaderFlash* | El control Multimedia utilizará el reproductor MacroMedia Flash (adaptado para archivos .FLV, .MP4, etc.) |
| *MultimediaReaderNativeBrowser* | El control multimedia utilizará el reproductor nativo HTML5. |
| *MultimediaReaderQuicktime* | El control Multimedia utilizará el reproductor QuickTime de Apple (adaptado para archivos .MOV, .MP4, etc.) |
| *MultimediaReaderWindowsMedia* | El control Multimedia utilizará el reproductor de Windows Media (adaptado para archivos .WMV, .ASF, .AVI, etc.) |

## Observaciones


### Modo de funcionamiento del control de Multimedia en el modo automatic
<a name="modo_funcionamiento_del_control_multimedia_modo_automatic_ELTPARAGRAPHE000091"></a>

En modo automático (constante *MultimediaReaderAutomatic*), el reproductor define el tipo de vídeo en función de su extensión. Este modo no funciona si el archivo no tiene la extensión adecuada o si ruta es una URL a una página/script dinámica.




## HSpace (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **HSpace** Property se utiliza para determinar y modificar el espaciado horizontal entre etiquetas en un reporte. Este espacio se definió al crear la etiqueta reporte.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ESPACEH.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the horizontal spacing between the labels
RPT_LabelReport.HSpace += 10   // Adds 10 mm
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinación del espacio horizontal entre etiquetas

`<Current spacing> = <Label report>.HSpace`
---

**`<Current spacing>: (Real)`**

Espacio horizontal actual entre etiquetas. Este espacio se expresa en milímetros.

**`<Label report>: (Nombre del reporte)`**

Nombre de la etiqueta reporte.  


<a name="SYNTAX2"></a>

### Modificación del espacio horizontal entre etiquetas

`<Label report>.HSpace = <New spacing>`
---

**`<Label report>: (Nombre del reporte)`**

Nombre de la etiqueta reporte.

**`<New spacing>: (Real)`**

Nuevo espacio horizontal entre etiquetas. Este espacio se expresa en milímetros.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Altura y anchura de las etiquetas
<a name="altura_anchura_las_etiquetas_ELTPARAGRAPHE000056"></a>

Las propiedades [LabelWidth](../Proprietes/2511055.md) y [LabelHeight](../Proprietes/2511069.md) se utilizan para definir el tamaño de las etiquetas mediante programación.

El [VSpace](../Proprietes/2511065.md) Property se utiliza para definir el espacio vertical entre las etiquetas en un reporte.
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000074"></a>

Esta propiedad se puede utilizar en cualquier WLanguage Event associated con reporte.

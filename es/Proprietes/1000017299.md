
## InMilliseconds (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InMilliseconds** Property se utiliza para convertir una Duración Variable a milisegundos. Es posible:

- Averigüe el número de milisegundos correspondiente a una duración.

- Modificar el número de milisegundos correspondiente a una duración.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDuration is Duration
MyDuration.InHours = 210
Trace(MyDuration.InMilliseconds)  // 756000000
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de milisegundos correspondiente a una duración Variable

`<Result> = <Duration used>.InMilliseconds`
---

**`<Result>: (Integro)`**

Número de milisegundos correspondiente a la duración especificada.

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.  


<a name="SYNTAX2"></a>

### Modificando el número de milisegundos correspondiente a una duración Variable

`<Duration used>.InMilliseconds = <New duration>`
---

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.

**`<New duration>: (Integro)`**

Número de milisegundos correspondiente a la duración especificada.

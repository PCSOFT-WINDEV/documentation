
## ProgressBar (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
WLanguage propone:

- [ProgressBar](../WDLang1/3021008.md) que se utiliza para mostrar una barra de progreso.

- La **ProgressBar** Property, que puede utilizarse en: 

	- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

	- una variable de tipo [httpProgress](../WDLang3/1000021222.md).

	- a control Tabla dinámica. Para obtener más información, consulte [ProgressBar](../Proprietes/1000020706.md).

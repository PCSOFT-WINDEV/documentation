


## LeftIndent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LeftIndent** Property se utiliza para get y cambiar el espacio a la izquierda del texto en un control. 

Esta Property se aplica a los siguientes controles: 

- las columnas de Table y TreeView controles Tabla (entre el comienzo de la columna y el comienzo del texto).

- controles Campo de entrada.

- los controles Combo Box y List Box.

- los controles de los botones.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) los controles de la cinta.

- los controles de la pestaña.

- Controles estáticos.




Este espacio se expresa en píxeles.
<br>![control Tabla sin ranura izquierda](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RetraitGauche.gif)

<br>![control Tabla con sangría izquierda set a 20 píxeles](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RetraitGauche2.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Indents all the columns in a Table control by 20 pixels
TABLE_Table1.LeftIndent = 20
// Sets an additional indentation to the Day column of the current row
COL_Day.LeftIndent = 50
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la sangría izquierda de una control

`<Result> = <Control used>.LeftIndent`
---

**`<Result>: (Integro)`**

Número de píxeles utilizados para sangrar el texto. La sangría es de set a 0 píxeles por Default (sin sangría). 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- columna de una control Tabla.

- columna de una control Tabla TreeView.. 

- control Campo de entrada.

- Control List Box o Combo Box.

- Control Botón.

- control Cinta de opciones.

- Control Pestaña.

- Control Estático.





<a name="SYNTAX2"></a>

### Cambiar la sangría izquierda de una control

`<Control used>.LeftIndent = <New indent>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- columna de una control Tabla.

- columna de una control Tabla TreeView.. 

- control Campo de entrada.

- Control List Box o Combo Box.

- Control Botón.

- control Cinta de opciones.

- Control Pestaña.

- Control Estático.




**`<New indent>: (Integro)`**

Número de pixeles a utilizar para la sangría del texto en el control especificado. 
Observación: La sangría aplicada a toda la control Tabla: Si se aplica un guión específico a una de las columnas, el nuevo guión no se aplicará a esta columna..

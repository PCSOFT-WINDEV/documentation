


## MarginWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MarginWidth** Property se utiliza para: 

- Descubre el margen horizontal entre los widgets que se encuentran en un control Cuadro de mando. 

- Modificar el margen horizontal entre los widgets encontrados en un control Cuadro de mando.




Observación: El margen horizontal entre los widgets de un control Cuadro de mando se define en la pestaña "Detalles" de la ventana de descripción de control Cuadro de mando. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify CBOX_WidgetSpacing
IF CBOX_WidgetSpacing THEN
	DASH_Main.MarginWidth = 20
	DASH_Main.MarginHeight = 20
ELSE
	DASH_Main.MarginWidth = 0
	DASH_Main.MarginHeight = 0
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el margen horizontal entre los widgets

`<Result> = <Dashboard control>.MarginWidth`
---

**`<Result>: (Integro)`**

Margen horizontal (en píxeles) entre los widgets que se encuentran en el control Cuadro de mando especificado.. 

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el margen horizontal entre los widgets

`<Dashboard control>.MarginWidth = <New margin>`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar. 

**`<New margin>: (Integro)`**

Nuevo margen horizontal (en píxeles) entre los widgets que se encuentran en la control Cuadro de mando especificada. Este valor debe ser positivo o nulo. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Este margen no tiene en cuenta la Border definida en el estilo.

- Para cambiar el margen vertical, utilice la función [MarginHeight](../Proprietes/1000021029.md) Property.


## InitialValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Valor inicial** Property se utiliza para get y/o cambiar un valor inicial. Esta propiedad se puede utilizar en:

- los controles de una ventana o página. Para obtener más información, consulte [ InitialValue](../Proprietes/2510128.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md).

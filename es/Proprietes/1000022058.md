
## UUID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **UUID** Property se utiliza para get el identificador de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [btleCharacteristic](../WDLang3/1000022060.md).

- una variable de tipo [btleDescriptor](../WDLang3/1000022064.md). 

- una variable de tipo [btleService](../WDLang3/1000022057.md).

- una variable de tipo [beaconGroup](../WDLang3/1000023105.md).

- una variable de tipo [beaconDetectionInfo](../WDLang3/1000023109.md).

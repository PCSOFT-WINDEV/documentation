


## MemoryFormat (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000010"></a>

Los controles de edición de fecha y hora y los controles de Calendar utilizan dos formatos: 

- El formato del valor visualizado (llamado máscara de entrada): este formato se utiliza para mostrar la fecha o la hora en la control Campo de entrada.

- El formato del valor devuelto por el programa o transferido a una posición.. 


Si la máscara de entrada y el valor devuelto no son idénticos, la conversión se realiza automáticamente al guardar el Record asociado. 

El formato del valor devuelto se escribe en el editor de la ventana, en la pestaña "Detalles" de la ventana description de los controles de edición de la fecha o la hora.
<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MemoryFormat** Property se utiliza para get y cambiar el formato del valor devuelto para:

- los controles de edición de Fecha u Hora.

- las columnas de los controles de la tabla de fecha o de hora.

- Calendar controla.  

- Controles estáticos. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) los controles editables de Combo Box.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) las columnas de los controles de tabla visualizados por los controles Combo Box.

<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
EDT_Date1.MemoryFormat = "YYYMMDD"
Trace(EDT_Date1)  // Will display 20010702
EDT_Date1.MemoryFormat = "MM/DD/YYYY"
Trace(EDT_Date1) // Will display 07/02/2001
EDT_Time1.MemoryFormat = "HH:MM"
Trace(EDT_Time1)  // Will display "14:31"
EDT_Time1.MemoryFormat = "HHMMSS"
Trace(EDT_Time1) // Will display "143123"
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Stored format of a Combo Box control displaying a list
Trace(COMBO_ListEdit.MemoryFormat)
COMBO_ListEdit.MemoryFormat = "YYYMMDD"
Trace(COMBO_ListEdit.MemoryFormat)

// Combo Box displaying a table
// Stored format of a Date column in a Table control
Trace(COMBO_TableEdit.COL_DateCol.MemoryFormat)
```

## Sintaxis

### Averiguar el valor devuelto

`<Returned value> = <Control used>.MemoryFormat`
---

**`<Returned value>: (Cadena de caracteres)`**

Formato utilizado actualmente para el valor de retorno del control especificado.

Para los controles y columnas de edición de una control Tabla, se selecciona la opción "Tal cual" si este parámetro corresponde a una cadena vacía ("").: en este caso, el valor devuelto corresponde a la máscara de entrada (devuelta por la [InputMask](../Proprietes/2510033.md) property).

Para los controles Calendar se utiliza el formato Default ("YYYYMMDD") si este parámetro corresponde a una cadena vacía ("").

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- "Fecha" o "Hora" control Campo de entrada,

- Columna "Fecha" u "Hora" de un control Tabla,

- Control Calendario.

### Modificando el valor devuelto

`<Control used>.MemoryFormat = <New format>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- "Fecha" o "Hora" control Campo de entrada,

- Columna "Fecha" u "Hora" de un control Tabla,

- Control Calendario.






**`<New format>: (Cadena de caracteres)`**

Nuevo formato usado para el valor de retorno del control especificado.

Para los controles de edición y las columnas de una control Tabla, se selecciona la opción "Tal cual" si este parámetro corresponde a una cadena vacía.: en este caso, el valor devuelto corresponde a la máscara de entrada (devuelta por la [InputMask](../Proprietes/2510033.md) property).

Para los controles de Calendar, se utiliza el formato Default ("YYYYMMDD") si este parámetro corresponde a una cadena vacía ("").

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000132"></a>

El **MemoryFormat** Property se aplica únicamente a:

- Controles de edición de "Fecha" y "Hora".

- las columnas del control Tabla de tipo "Fecha" y "Hora".

- Controles Calendar.

- Controles estáticos. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) controles Combo Box editables.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de un control Tabla mostrado por los controles Combo Box.






![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **MemoryFormat** Property se aplica sólo a los controles de Edición.




## Image (palabra clave)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Image** puede corresponder a: 

- una variable de tipo [Image](../WDLang1/1000019650.md). 

- el **Imagen** Property utilizado para get y cambiar el Image de un elemento. Esta propiedad se puede utilizar en:

	- un control. Para obtener más información, consulte [Imagen Property en los controles](../Proprietes/2510034.md).

	- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

	- una variable de tipo [Marker](../WDLang3/1000019940.md).

	- una variable de tipo [Appointment](../WDLang1/1000019244.md).

	- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

	- una variable de tipo [rssStream](../WDLang5/1000017795.md).

	- una variable de tipo [OrgElement](../WDLang1/1000019713.md).

	- una variable de tipo [Icon](../WDLang1/1000020811.md).

	- una variable de tipo [Tiff](../WDLang1/1000021025.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [GanttTask](../WDLang1/1000020881.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpImage](../WDLang3/1000020905.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElement](../WDLang1/1000022484.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WDPic](../WDLang1/1000023503.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [BarCode](../WDLang5/1000019451.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

	- una variable de tipo [MapImage](../WDLang3/1000025523.md).


## Valid (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Valid** Property se utiliza para determinar si un elemento es válido. Esta propiedad se puede utilizar en:

- una variable de tipo [Image](../WDLang1/1000019650.md).

- una variable de tipo [AuthToken](../WDLang3/1000022220.md). 

- una variable de tipo [OpenIDIdentity](../WDLang3/1000023567.md).

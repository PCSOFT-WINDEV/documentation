


## StartDate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **StartDate** permite determinar y cambiar la fecha de inicio del período de tiempo seleccionado:

- en un control Calendario. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) en un control Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) en un control Planificador.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the selected period (Calendar control)
NbDays is int 
NbDays = DateDifference(CAL_Calendar1.StartDate, CAL_Calendar1.EndDate) + 1
Info("Selected time period: from " + ...
	DateToString(CAL_Calendar1.StartDate) + ...
	" to " + DateToString(CAL_Calendar1.EndDate) + ...
	" which means " + NbDays + " days")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar la fecha de inicio del período de tiempo

`<Start date> = <Control used>.StartDate`
---

**`<Start date>: (Cadena de caracteres o variable de tipo Date)`**



- Fecha de inicio del período de tiempo seleccionado en el control.  

	- En un control Calendario, esta fecha está en formato "AAAAMMDD".

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) En un control Agenda o Planificador, esta fecha está en formato "AAAAMMDDHHmmSSCCC". 




- Fecha del día seleccionada si se selecciona un día,

- Cadena vacía ("") si no se seleccionó ningún día.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado:

- Control Calendario.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Control Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Control Planificador.





<a name="SYNTAX2"></a>

### Cambiar la fecha de inicio del período de tiempo

`<Control used>.StartDate = <New date>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado:

- Control Calendario, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Control Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Control Planificador.




**`<New date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de inicio del período de tiempo a seleccionar en el control. 

- En un control Calendario, esta fecha está en formato "AAAAMMDD".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) En un control Agenda o Planificador, esta fecha está en formato "AAAAMMDDHHmmSSCCC".

## Observaciones


### Selección de período
<a name="seleccion_periodo_ELTPARAGRAPHE000097"></a>

Para seleccionar un período de tiempo : 

- en un control Calendario, la opción "       Selección de período" debe estar seleccionada en la pestaña "Detalles".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) en un control Agenda, la opción "Permitir seleccionar períodos" debe estar seleccionada en la pestaña "Detalles".

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) en un control Planificador, la opción "Permitir seleccionar períodos" debe estar seleccionada en la pestaña "Detalles".



<a name="NOTE0_2"></a>


### Cambiar un período de tiempo en un control Calendario, Agenda o Planificador
<a name="cambiar_periodo_tiempo_control_calendario_agenda_planificador_ELTPARAGRAPHE000124"></a>

**Para cambiar la fecha de inicio de un período de tiempo seleccionado**, especifique la fecha de inicio (propiedad **StartDate**) y la fecha de finalización (propiedad [EndDate](../Proprietes/1000017427.md)). Cuando se cambia la fecha de inicio: 

- la fecha de finalización se deselecciona automáticamente.

- se debe seleccionar la fecha de finalización (mediante programación o haciendo clic en una fecha del control).




**Para cambiar la fecha de finalización de un período de tiempo seleccionado**, utilice la propiedad [EndDate](../Proprietes/1000017427.md): el período de tiempo cambia automáticamente.  
<a name="NOTE0_3"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000144"></a>

- La fecha de finalización de un período de tiempo se puede identificar y cambiar con la propiedad [EndDate](../Proprietes/1000017427.md).

- La propiedad **StartDate** devuelve la fecha de inicio del período de tiempo en orden cronológico. Por lo tanto, si se seleccionó la fecha de finalización antes de la fecha de inicio, la propiedad **StartDate** devuelve la última fecha seleccionada.

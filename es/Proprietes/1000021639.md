


## MultiWorksheet (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **MultiWorksheet** permite: 

- Averigua si un control Hoja de cálculo maneja varias hojas de trabajo. 

- Cambiar el modo de gestión de las hojas de trabajo de una control Hoja de cálculo.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switches the Spreadsheet control to multi-worksheet mode to load an xlsx file
PSHEET_MySpreadsheet.MultiWorksheet = True
SpreadsheetLoad(PSHEET_MySpreadsheet, "FullDoc.xlsx")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el modo de gestión de las hojas de trabajo

`<Result> = <Spreadsheet control>.MultiWorksheet`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Hoja de cálculo maneja varias hojas de trabajo, 

- <u><u><u><u>False</u></u></u></u> si el control Hoja de cálculo maneja una sola hoja de trabajo.  




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


<a name="SYNTAX2"></a>

### Modificar el modo de gestión de las hojas de trabajo

`<Spreadsheet control>.MultiWorksheet = <New mode>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<New mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Hoja de cálculo debe manejar varias hojas de trabajo, 

- <u><u><u><u>False</u></u></u></u> si el control Hoja de cálculo debe manejar una sola hoja de trabajo.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si se activa el modo de gestión de múltiples hojas de trabajo: 

- Los botones para manejar las hojas de trabajo se muestran en la control. 

- Los documentos XLSX cargados y guardados gestionan todas las hojas de trabajo ([Hoja de cálculoCarga](../WDLang1/1000021340.md) y [SpreadsheetSave](../WDLang1/1000021341.md)).


Si el modo de gestión de múltiples hojas de trabajo no está activado: 

- [Hoja de cálculoCarga](../WDLang1/1000021340.md) carga una sola hoja de trabajo. 

- No se muestra ningún botón para la gestión de la hoja de cálculo.

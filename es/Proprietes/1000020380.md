


## RulerValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RulerValue** Property se utiliza para get o cambiar la posición del cabezal de reproducción en un control Línea de tiempo.

**Recordatorio**: El cabezal de reproducción de una control Línea de tiempo es una línea vertical en negrita que representa un punto específico en la control. El estilo del cabezal de reproducción se puede definir en la pestaña "Estilo" de la control Línea de tiempo ("Tiempo seleccionado").  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TimelineDeleteAll(TL_Robots)
Evt is EventTimeline

TimelineAddTrack(TL_Robots, "Robot 1")
Evt.Track = "Robot 1"
Evt.Title = "Start"
Evt.Start = 10
Evt.End = 150
Evt.BackgroundColor = LightGreen
TimelineAddEvent(TL_Robots, Evt)

Evt.Track = "Robot 1"	
Evt.Title = "Special process"
Evt.Start = 150
Evt.End = 450
Evt.BackgroundColor = PastelBlue
TimelineAddEvent(TL_Robots, Evt)

// Positions the playhead at a specific point
TL_Robots.RulerVisible = True
TL_Robots.RulerValue = 300

// -- Code for modifying the TimeLine control
// Displays the selected value in the static
STC_TimeLineRuler = TL_Robots.RulerValue
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el valor del cabezal de reproducción

`<Result> = <TimeLine control>.RulerValue`
---

**`<Result>: (Integro)`**

Posición actual del cabezal de reproducción. La unidad utilizada es idéntica a la utilizada para la visualización de la control Línea de tiempo (segundo, milisegundo de microsegundo).. 

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo a utilizar. 


<a name="SYNTAX2"></a>

### Modificar el valor del cabezal de reproducción

`<TimeLine control>.RulerValue = <New position>`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo a utilizar. 

**`<New position>: (Integro)`**

Nueva posición del cabezal de reproducción. La unidad utilizada es idéntica a la utilizada para la visualización de la control Línea de tiempo (segundo, milisegundo de microsegundo).. Este valor debe estar entre los valores de las propiedades [StartTotalRange](../Proprietes/1000020392.md) y [EndTotalRange](../Proprietes/1000020393.md), de lo contrario el cabezal de reproducción será Invisible. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El usuario puede mover el cabezal de reproducción: 

	- si la opción "Cabezal desplazable" está marcada en la ventana de descripción de control Línea de tiempo. 

	- si el valor de la propiedad [ReglaModificable](../Proprietes/1000020381.md) es <u><u><u><u>True</u></u></u></u>. 




- El usuario final puede cambiar el valor del cabezal de reproducción moviéndolo con el ratón. Esta acción desencadena la "Siempre que se modifica" Event del control Línea de tiempo. 

- La posición por defecto del cabezal de reproducción corresponde al valor de la [StartTotalRange](../Proprietes/1000020392.md) Property.

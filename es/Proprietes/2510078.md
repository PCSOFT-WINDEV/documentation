


## Message (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Message** permite:

- Encuentre el mensaje de ayuda associated con un control o con una opción de menú (en una ventana).

- Modificar el mensaje de ayuda associated con un control o con una opción de menú (en una ventana).




**Observaciones**:

- El mensaje de ayuda se muestra en la barra de estado de la ventana.

- El mensaje de ayuda se define en la pestaña "Ayuda" del control description (opción "Texto de la barra de estado"), con la función **Mensaje** Property o la función [Mensaje](../WDLang1/3021006.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the help message of the "EDT_ImageFile" Edit control
ImageFile.Message = "The image file must be a black && white image"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el mensaje de ayuda

`<Result> = <Element used>.Message`
---

**`<Result>: (Cadena de caracteres)`**



- Mensaje de ayuda para el objeto especificado,

- Vaciar la cadena ("") si no hay mensaje de ayuda es associated con el objeto.




**`<Element used>: (Cadena de caracteres)`**

Nombre del objeto (control o opción de menú) cuyo mensaje de ayuda se solicita.  


<a name="SYNTAX2"></a>

### Modificando el mensaje de ayuda

`<Element used>.Message = <New help message>`
---

**`<Element used>: (Cadena de caracteres)`**

Nombre del objeto (control u opción de menú) cuyo mensaje de ayuda debe ser modificado.

**`<New help message>: (Cadena de caracteres)`**

Nuevo mensaje de ayuda para el objeto especificado.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Visualización del mensaje de ayuda
<a name="visualizacion_del_mensaje_ayuda_ELTPARAGRAPHE000052"></a>

- **Para un control** (excepto para los botones control s): el mensaje de ayuda se muestra en la barra de estado cuando el control toma el foco.

- **Para un control Botón**: el mensaje de ayuda se muestra en la barra de estado cuando se pulsa el botón izquierdo del ratón.

- **Para una opción de menú**: el mensaje de ayuda se muestra en la barra de estado cuando se pasa el ratón por encima de la opción.



<a name="NOTE0_2"></a>


### y carácter
<a name="caracter_ELTPARAGRAPHE000061"></a>Para que aparezca el carácter "&" en un mensaje de ayuda, este carácter debe escribirse dos veces. En caso contrario, se subrayará el carácter que sigue al carácter "&" y no se mostrará el carácter "&".. Por ejemplo:


```wl
ImageFile.Message = "The image file must be a black && white image"
```

<a name="NOTE0_3"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000070"></a>

La propiedad **Message** no se puede utilizar en:

- un control Imagen, 

- a control Forma,

- una opción en una control Casilla de verificación,

- una opción en una control Botón de opción,

- una fila en una control List Box, control Combo Box o control Tabla,

- una celda de columna, 

- a tab,

- un grupo de controles,

- un elemento de un control TreeView.

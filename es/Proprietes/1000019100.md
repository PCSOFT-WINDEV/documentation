


## GranularityAppointment (Property)

*** ***
	


	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>16 (56)</B></B>, este Property se mantiene para la compatibilidad con el pasado. Le recomendamos que utilice [GranularityMovement](../Proprietes/1000019642.md) y [GranularityDuration](../Proprietes/1000019643.md).

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **GranularidadCitas** Property se utiliza para get y cambiar la precisión de la cuadrícula utilizada para definir las citas: 

- en un control Agenda. 

- en un control Planificador.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the grid in order to position an appointment every 15 minutes
ORG_MyOrganizer.GranularityAppointment = 15
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la precisión para posicionar las citas en la cuadrícula

`<Result> = <Control used>.GranularityAppointment`
---

**`<Result>: (Integro)`**

Precisión permitida por la cuadrícula de la control al posicionar las citas (en minutos). El valor por defecto es set a 30 minutos.
Esta precisión puede incluirse entre 1 (un minuto) y 240 (4 horas).

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: control Agenda o control Planificador.


<a name="SYNTAX2"></a>

### Modificando la precisión para posicionar las citas en la cuadrícula

`<Control used>.GranularityAppointment = <Value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: control Agenda o control Planificador.

**`<Value>: (Integro)`**

Precisión permitida por la cuadrícula de la control al posicionar las citas (en minutos).
Esta precisión puede incluirse entre 1 (un minuto) y 1440 (24 horas).

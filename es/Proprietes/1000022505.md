
## FontItalic (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **FontItalic** Property se utiliza para gestionar el estilo de cursiva en un texto. Esta propiedad se puede utilizar en: 

- controles. Para obtener más información, consulte [FontItalic](../Proprietes/2510102.md). 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md).

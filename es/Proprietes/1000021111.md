
## BitPerPixel (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BitPerPixel** Property se utiliza para get y cambiar el número de bits por píxel en un Image. Esta propiedad se puede utilizar en:

- una variable de tipo [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpImage](../WDLang3/1000020905.md).

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una variable de tipo [Icon](../WDLang1/1000020811.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Scanner](../WDLang6/1410087808.md).

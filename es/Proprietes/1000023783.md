


## TokenListSeparator (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TokenListSeparator** Property recupera o modifica los separadores que desencadenan la creación de nuevos Token al introducir información en un "Texto Token" control Campo de entrada. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF EDT_Email.TokenEnabled = True THEN
	EDT_Email.TokenListSeparator = ";.:"
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir los separadores que desencadenan la creación de fichas

`<Result> = <Edit control>.TokenListSeparator`
---

**`<Result>: (Cadena de caracteres)`**

Lista de separadores autorizados para la creación de Token.
Observación: La tecla Enter siempre corresponde a un separador de Token. 

**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada utilizada. 


<a name="SYNTAX2"></a>

### Modificando los separadores que desencadenan la creación de fichas

`<Edit control>.TokenListSeparator = <New separators>`
---

**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada utilizada. 

**`<New separators>: (Cadena de caracteres)`**

Lista de separadores autorizados para la creación de Token.
Observación: La tecla Enter siempre corresponde a un separador de Token.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se genera un error: 

	- si la control Campo de entrada no es una "ficha de texto" control Campo de entrada. 

	- si la propiedad [TokenEnabled](../Proprietes/1000023782.md) corresponde a <u><u><u><u>False</u></u></u></u>.




- Cada carácter especificado con el **TokenListSeparator** Property se procesa individualmente. Por ejemplo, si el **TokenListSeparator** Property se corresponde con ";:", al introducir ':' se crea un Token.

- La tecla Enter siempre desencadena la creación de un token, por lo que no es necesario añadirlo a la lista de separadores del **TokenListSeparator** Property.

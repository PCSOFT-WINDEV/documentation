


## GranularityMovement (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **GranularidadMovimiento** Property se utiliza para get y cambiar el tamaño de la cuadrícula para moverse:  

- citas en una control Agenda. 

- citas en una control Planificador.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) eventos en una control Línea de tiempo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) tareas en una columna del diagrama de Gantt.




**Observación**: Para obtener y cambiar el tamaño de la cuadrícula utilizada para la duración, llame a la [GranularityDuration](../Proprietes/1000019643.md) Property. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch the grid to 15 minutes 
SCH_Schedule1.GranularityMovement = 15
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Elementos móviles: Averiguar el tamaño de la red

`<Result> = <Control used>.GranularityMovement`
---

**`<Result>: (Integro)`**

La resolución de la cuadrícula utilizada en la control para el movimiento: 

- citas (en minutos). El valor por defecto es set a: 

	- 30 minutos en una control Agenda.

	- 24 horas en un control Planificador. 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) eventos en una control Línea de tiempo. El valor por defecto es set a 60 (segundos, milisegundos o microsegundos). La unidad depende de la unidad de pantalla utilizada para la control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de tareas en una columna del diagrama de Gantt. El valor por defecto es set a 1 hora.  




Para un Organizador o control Planificador, esta resolución puede incluirse entre: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para una control Planificador, entre 1 (un minuto) y 43200 (1 mes). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para una control Agenda: 1 (un minuto) y 1440 (24 horas).


Este valor debe ser un múltiplo de 60 (120, 240, 320, ....) o un submúltiplo de 60 (1, 2, 5, 10, 15, 30, ....).. 
Las citas o los eventos comienzan con un múltiplo de este valor.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Agenda.

- Control Planificador.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Línea de tiempo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Columna Diagrama de Gantt.





<a name="SYNTAX2"></a>

### Elementos móviles: Modificando el tamaño de la red

`<Control used>.GranularityMovement = <Value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Agenda.

- Control Planificador.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Línea de tiempo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Columna Diagrama de Gantt.




**`<Value>: (Integro)`**

Nueva resolución de la retícula utilizada en el control para mover: 

- citas en un Programador o control Agenda. Esta resolución se expresa en minutos. 




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) eventos en una control Línea de tiempo. La unidad depende de la unidad de pantalla utilizada para la control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de tareas en una columna del diagrama de Gantt. El valor por defecto es set a 1 hora.  


Este valor debe ser un múltiplo de 60 (120, 240, 320, ....) o un submúltiplo de 60 (1, 2, 5, 10, 15, 30, ....).. 
Las citas o los eventos se iniciarán con un múltiplo de este valor.

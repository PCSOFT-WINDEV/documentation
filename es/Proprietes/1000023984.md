
## Vibration (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Vibration** se utiliza para gestionar las vibraciones Notification. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Notification](../WDLang3/1000019441.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [notificationCategory](../WDLang3/1000023981.md).

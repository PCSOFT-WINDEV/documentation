


## SignatureType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SignatureType** Property se utiliza para get y definir el tipo de firma que se utilizará en un control Firma.

Observación: Esta propiedad corresponde a la opción "Tipo de firma" de la pestaña "General" de la ventana control Firma description. Este control sólo está disponible en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The Signature control uses a certification signature
SIG_MySignature.SignatureType = typeSignatureCertification
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el tipo de firma utilizado por la control Firma

`<Result> = <Signature control>.SignatureType`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de firma: 


|   |   |
| --- | --- |
| *typeSignatureApproval* | Firma de aprobación.<br>Una firma de aprobación corresponde a una marca aplicada por un tercero para indicar que aprueba el contenido del documento. |
| *typeSignatureCertification* | Firma de certificación.<br>Una firma de certificación corresponde a un sello aplicado al Document para garantizar su integridad. |



**`<Signature control>: (Nombre del control)`**

control Firma a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el tipo de firma usado por la control Firma

`<Signature control>.SignatureType = <New type>`
---

**`<Signature control>: (Nombre del control)`**

control Firma para usar.

**`<New type>: (Constante de tipo Integer)`**

Tipo de firma a utilizar: 


|   |   |
| --- | --- |
| *typeSignatureApproval* | Firma de aprobación.<br>Una firma de aprobación corresponde a una marca aplicada por un tercero para indicar que aprueba el contenido del documento. |
| *typeSignatureCertification* | Firma de certificación. <br>Una firma de certificación corresponde a un sello aplicado al Document para garantizar su integridad. |

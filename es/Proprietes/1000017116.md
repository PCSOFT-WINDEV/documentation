


## HandwrittenInput (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **HandwrittenInput** se utiliza para gestionar el lápiz óptico o el ratón Input( Windows XP Tablet PC o Windows Vista, respectivamente). Es posible:

- Averigua si un RTF Edit control permite la entrada de datos a mano.

- Permitir la introducción de datos manuscritos en un RTF Editar control.




Esta propiedad equivale a la opción "Entrada manuscrita (PC tablet)" de la pestaña "General" de la ventana Editar control description.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allow the handwritten input
EDT_RTFEdit.HandwrittenInput = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la entrada de escritura a mano está permitida por la control

`<Handwritten input> = <Edit control>.HandwrittenInput`
---

**`<Handwritten input>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la entrada manuscrita está permitida en el control Campo de entrada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada utilizada. Este control debe ser un control Campo de entrada de RTF.  


<a name="SYNTAX2"></a>

### Permitiendo (o no) la entrada manuscrita

`<Edit control>.HandwrittenInput = <Handwritten input>`
---

**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada utilizada. Este control debe ser un RTF control Campo de entrada.

**`<Handwritten input>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la entrada manuscrita debe estar permitida en el control Campo de entrada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## salir con

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Fecha** palabra clave puede corresponder a: 

- una variable de tipo [salir con](../Motscles/1514066.md). 

- el **Fecha** Property de un Variable de tipo [InfoVersion](../WDLang1/1000018905.md).

- el **Fecha** Property de un Variable de tipo [mongoGridFSFileInfo](../WDLang4/1000022417.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el **Fecha** Property de un Variable de tipo [saasVisit](../WDLang3/1000024560.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el **Fecha** Property de un Variable de tipo [sysNotification](../WDLang1/1410087893.md).

- el **Fecha** Property de un Variable de tipo [kbCard](../WDLang1/1410089167.md).

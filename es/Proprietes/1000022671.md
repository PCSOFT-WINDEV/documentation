
## DisplayOnly (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ReadOnly** Property se utiliza para determinar si un elemento es de sólo lectura. Esta propiedad se puede utilizar en:

- una variable de tipo [NFCTag](../WDLang3/1000020411.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Document](../WDLang1/1000022461.md).




## CtDescribedUserName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtDescribedUserName** Property se utiliza para set el nombre de usuario que se utilizará al conectarse a un tabla específico a través de OLE DB. Es posible:

- Averiguar el nombre de usuario que se utilizará para conectar un archivo de datos a un tabla por OLE DB (archivo de datos definido en el editor de análisis o mediante programación). Este nombre de usuario se especificó durante el description del archivo de datos.

- Definir el nombre de usuario que se utilizará para conectar un archivo de datos a un tabla por OLE DB (cuando se describe el archivo de datos mediante programación). 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por defecto, las características de la conexión OLE DB se definen en la editor de análisis durante la descripción del archivo de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the name of the user
Customer.CtDescribedUserName = "Julia"
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre del usuario

`<User> = <Data file>.CtDescribedUserName`
---

**`<User>: (Cadena de caracteres)`**

Nombre del usuario de la fuente OLE DB, definido durante el archivo description en el editor de análisis.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definir el nombre del usuario

`<Data file>.CtDescribedUserName = <New username>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<New username>: (Cadena de caracteres)`**

Nombre de usuario de la fuente OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El [CtUserName](../Proprietes/2512026.md) Property se utiliza para get y cambiar el nombre de usuario especificado para la conexión OLE DB cuando se utiliza [HConnect](../WDLang4/3044263.md).

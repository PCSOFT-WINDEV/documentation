
## Thickness (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Grosor** Property se utiliza para get y cambiar el grosor de un elemento. Esta propiedad se puede utilizar en:

- los controles de la ventana: 

	- las cartas de Line. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) los controles de Wire. 


 Para obtener más información, consulte [Espesor](../Proprietes/2510144.md).

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md). 

- una variable de tipo [gglPath](../WDLang5/1000017433.md).

- una variable de tipo [Franja de orilla](../Motscles/15140100.md).

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md).


## Wrapping (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Wrapping** Property se utiliza para get y cambiar las opciones de envoltura de un elemento. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementImage](../WDLang1/1000022696.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementTextArea](../WDLang1/1000023420.md).

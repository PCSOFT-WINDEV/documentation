


## NbDayDisplayed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbDayDisplayed** Property se utiliza para: 

- get y cambiar el número de días mostrados en un Organizador o control Planificador. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) get el número de días mostrados en un control Diagrama de Gantt en un reporte. 




**Observación**: Este número de días se utiliza únicamente para la visualización en modo "semana".


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays 2 days in the Scheduler control
SCH_Scheduler1.NbDayDisplayed = 2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de días que se muestran

`<Result> = <Control used>.NbDayDisplayed`
---

**`<Result>: (Integro)`**

Número de días que se muestran en el control. 

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Agenda.

- Control Planificador. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) control Diagrama de Gantt en un reporte





<a name="SYNTAX2"></a>

### Modificar el número de días mostrados

`<Control used>.NbDayDisplayed = <New Number of Days>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Agenda.

- Control Planificador. 




**`<New Number of Days>: (Integro)`**

Número de días que se mostrarán en el control. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Uso de la función OrganizerChangeMode**: 
	[OrganizerChangeMode](../WDLang1/1000019497.md) se utiliza para modificar el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). Si el número de días visualizados se configuró con la función **NbDayDisplayed** Property, [OrganizerChangeMode](../WDLang1/1000019497.md) sobrescribe este valor con el valor especificado en la llamada a la función. 

- **Uso de la función SchedulerChangeMode**: 
	[SchedulerChangeMode](../WDLang1/1000019440.md) se utiliza para modificar el rango de tiempo mostrado por un control Planificador (Día/Semana/Mes). Si el número de días visualizados se configuró con la función **NbDayDisplayed** Property, [SchedulerChangeMode](../WDLang1/1000019440.md) sobrescribe este valor con el valor especificado en la llamada a la función.

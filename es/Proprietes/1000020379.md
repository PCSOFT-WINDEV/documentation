


## RulerVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RulerVisible** Property se utiliza para: 

- Determinar si un cabezal de reproducción es Visible en un control Línea de tiempo. 

- Mostrar u ocultar un cabezal de reproducción en una control Línea de tiempo. 




**Recordatorio**: El cabezal de reproducción de una control Línea de tiempo es una línea vertical en negrita que representa un punto específico en la control. El estilo del cabezal de reproducción se puede definir en la pestaña "Estilo" de la control Línea de tiempo ("Tiempo seleccionado").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Set the playhead to 5 seconds
TL_TimeLine.RulerVisible = True
TL_TimeLine.RulerValue = 5
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si un cabezal de reproducción es Visible

`<Result> = <Control used>.RulerVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el cabezal de reproducción está Visible en el control, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre de control Línea de tiempo utilizado. 


<a name="SYNTAX2"></a>

### Mostrar u ocultar un cabezal de reproducción

`<Control used>.RulerVisible = <Visible/Invisible>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control Línea de tiempo utilizada.

**`<Visible/Invisible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el cabezal de reproducción debe ser Visible, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El usuario puede mover el cabezal de reproducción: 

	- si la opción "Cabezal desplazable" está marcada en la ventana de descripción de control Línea de tiempo. 

	- si el valor de la propiedad [ReglaModificable](../Proprietes/1000020381.md) es <u><u><u><u>True</u></u></u></u>.




- El cabezal de reproducción es Visible para el usuario: 

	- si la opción "Mostrar cabezal" está marcada en la ventana de descripción de control Línea de tiempo. 

	- si la propiedad **RulerVisible** se establece en <u><u><u><u>True</u></u></u></u>.




- La posición por defecto del cabezal de reproducción corresponde al valor de la [StartTotalRange](../Proprietes/1000020392.md) Property. 

- El usuario final puede cambiar el valor del cabezal de reproducción moviéndolo con el ratón. Esta acción activa el evento "Modificación" del control Línea de tiempo. 

- La nueva posición del cabezal de reproducción es devuelta por el [Valor de regla](../Proprietes/1000020380.md) Property.

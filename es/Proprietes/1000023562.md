
## RedirectionURL (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RedirectionURL** Property se utiliza para get la URL utilizada para la redirección del resultado. Esta propiedad es utilizada por:

- una variable de tipo [OAuth2Parameters](../WDLang3/1000022212.md). 

- una variable de tipo [OpenIDParameters](../WDLang3/1000023561.md).

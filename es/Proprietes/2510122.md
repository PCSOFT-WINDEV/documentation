


## TotalsEnabled (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TotalsEnabled** Property le permite:

- \* Identifica el método de cálculo utilizado para las columnas de una Tabla o control Tabla TreeView (automatic o personalizado),

- Prohibir o forzar temporalmente los cálculos de columnas (automatic o personalizados) en una Tabla o control Tabla TreeView.




Observaciones: 

- Para más detalles sobre la aplicación y programación de los cálculos en controles Tabla, véase: 

	- [Realización de cálculos automatic en las columnas de controles Tabla](../WDChamp/1013201.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Realización de cálculos personalizados en las columnas de controles Tabla](../WDChamp/1013331.md).




- Si la propiedad [DisplayEnabled](../Proprietes/2510015.md) se utiliza en el control Tabla y es set a <u><u><u>False</u></u></u>, los cálculos no se actualizan. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TABLE_TABLE1.TotalsEnabled = False // no more calculated totals
FOR I = 1 TO 50
	TableAdd(TABLE_TABLE1, MyCalculationRowContent(I))
	Multitask(-1) // the table is redisplayed without totals
END
TABLE_TABLE1.TotalsEnabled = True
Trace(" Total = " + COL_TOTALCOL[rowTotal])
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificación del método de cálculo utilizado para las columnas (automatic o personalizado)

`<Result> = <Control used>.TotalsEnabled`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los cálculos de las columnas (automáticos o personalizados) se realizan inmediatamente,

- <u><u><u><u>False</u></u></u></u> si no se realizan los cálculos de las columnas (automáticos o personalizados).




**`<Control used>: (Nombre del control)`**

Nombre de la tabla o control Tabla TreeView que contiene los cálculos de la columna (automatic o personalizado).  


<a name="SYNTAX2"></a>

### Cambio del método de cálculo de las columnas (automatic o personalizado)

`<Control used>.TotalsEnabled = <New method>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la tabla o control Tabla TreeView que contiene los cálculos de la columna (automatic o costumbre).

**`<New method>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para forzar la ejecución de cálculos de columnas (automáticos o personalizados) en el control,

- <u><u><u><u>False</u></u></u></u> no realizar los cálculos de columnas (automáticos o personalizados).

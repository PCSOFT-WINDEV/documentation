
## HTML (Property)

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HTML** Property se utiliza para configurar el uso de contenidos HTML. Esta propiedad se puede utilizar en: 

- un reporte. Para más detalles, consulte el documento [HTML](../Proprietes/2511040.md) Property.

- la estructura [Email](../WDLang3/3032029.md).

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una variable de tipo [edtHTMLSelection](../WDLang1/1000026093.md).


## filtros

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Filter** puede corresponder a: 

- la función [filtros](../WDLang1/1000024845.md), que devuelve un array que contiene los elementos de una fuente array para la que un Procedure devuelve <u><u><u>True</u></u></u>.

- la propiedad **Filter** permite obtener y cambiar las características de un filtro. 
	Esta propiedad se puede utilizar en:

	- Navegar por List Box, Combo Box y controles Tabla. Para obtener más información, consulte la ayuda sobre la propiedad [filtros](../Proprietes/2510042.md).

	- una variable de tipo [hbScanParameter](../WDLang4/1000021715.md).

	- una variable de tipo [mongoOperationUpdateOne](../WDLang4/1410089164.md).

	- una variable de tipo [mongoOperationUpdateAll](../WDLang4/1410089165.md).

	- una variable de tipo [mongoOperationReplaceOne](../WDLang4/1410089166.md).

	- una variable de tipo [mongoOperationDeleteAll](../WDLang4/1410089162.md).

	- una variable de tipo [mongoOperationDeleteOne](../WDLang4/1410089161.md).

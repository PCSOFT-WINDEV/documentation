


## MagnifierMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MagnifierMode** Property determina y cambia cómo se muestra la lupa en los encabezados de las columnas de una Tabla o control Tabla TreeView.

**Observación**: El modo de gestión de la lupa indica el modo de funcionamiento de la control Tabla cuando el usuario hace clic en la lupa en los títulos de las columnas.. Este modo de gestión también lo utiliza [TableInputSearch](../WDLang1/3074024.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the mode for using the magnifier
TABLE_Table1.MagnifierMode = filterContains
// Use the magnifier
TableInputSearch(TABLE_Table1.COL_Column1, "")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de gestión de la lupa en una control Tabla

`<Result> = <Control used>.MagnifierMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de gestión actual de la lupa en la control Tabla o en la columna especificada. Este parámetro puede corresponder a una de las siguientes constantes:


|   |   |
| --- | --- |
| *filterContains* | Al hacer clic en la lupa se iniciará un filtro "Contiene: sólo se mostrarán las líneas para las que el valor de columna "Contiene" los caracteres especificados. |
| *filterDifferent* | Al hacer clic en la lupa se inicia un filtro "Diferente a".: sólo se mostrarán las líneas para las que el valor de la columna "es diferente de" los caracteres especificados. |
| *filterDoesNotContain* | Al hacer clic en la lupa se iniciará un filtro "No contiene: sólo se mostrarán las líneas para las que el valor de la columna "no contiene" los caracteres especificados. |
| *filterDoesNotEndWith* | Al hacer clic en la lupa se iniciará un filtro "No termina con: sólo se mostrarán las líneas para las que el valor de la columna "no termina con" los caracteres especificados. |
| *filterDoesNotStartWith* | Al hacer clic en la lupa se iniciará un filtro "No empieza con: sólo se mostrarán las líneas para las que el valor de la columna "no empieza por" los caracteres especificados. |
| *filterEndsWith* | Al hacer clic en la lupa se inicia un filtro "Finalizar con: sólo se mostrarán las líneas para las que el valor de la columna "no termina con" los caracteres especificados. |
| *filterEqual* | Al hacer clic en la lupa se iniciará un filtro "Igual a: sólo se mostrarán las líneas para las que el valor de la columna "es igual a" los caracteres especificados. |
| *filterGreater* | Al hacer clic en la lupa se iniciará un filtro "Mayor que: sólo se mostrarán las líneas para las que el valor de la columna "es mayor que" los caracteres especificados. |
| *filterGreaterOrEqual* | Al hacer clic en la lupa se iniciará un filtro "Mayor que o igual a: sólo se mostrarán las filas en las que el valor de la columna "es mayor o igual que" los caracteres especificados. |
| *filterLess* | Al hacer clic en la lupa se iniciará un filtro "Menos que: sólo se mostrarán las líneas para las que el valor de la columna "es inferior a" los caracteres especificados. |
| *filterLessOrEqual* | Al hacer clic en la lupa se iniciará un filtro "Menos que o igual a: sólo se mostrarán las líneas en las que el valor de la columna "es inferior o igual a" los caracteres especificados. |
| *filterStartsWith* | Al hacer clic en la lupa se inicia un filtro "Starts with: sólo se mostrarán las filas en las que el valor de la columna "Empieza con" los caracteres especificados. |
| *SearchDefault* | Indica que la columna utilizará el modo de gestión de la lupa definida para la associated tabla. Esta constante sólo está disponible para las columnas tabla. |
| *SearchStartsWith* | Un clic en la lupa permite clasificar la tabla de acuerdo con la columna especificada y cambiar esta columna al modo de búsqueda.: sólo se mostrarán las líneas para las que el valor de la columna "empieza por" los caracteres especificados. |



**`<Control used>: (Nombre del control)`**

Nombre de la columna tabla, control Tabla o control Tabla TreeView a utilizar.  


<a name="SYNTAX2"></a>

### Modificando el modo de gestión de la lupa en una control Tabla

`<Control used>.MagnifierMode = <New mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la columna tabla, control Tabla o control Tabla TreeView a utilizar.

Si se realiza la modificación en el control Tabla, esta modificación afecta a todas las columnas con la constante *SearchDefault* (si es posible).

Si la modificación se realiza en una columna tabla, el efecto se limita a esta columna.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **MagnifierMode** Property sólo puede utilizarse en los controles de Tabla y Tabla TreeView, así como en las columnas que contienen.

El **MagnifierMode** Property no tiene ningún efecto:

- en la controles Tabla archivo de datos con "Acceso directo".

- en la controles Tabla archivo de datos que utilizan una barra de desplazamiento con un movimiento continuo.

- en las columnas que no muestran ninguna lupa y que no se pueden clasificar.

- en las columnas que no soportan el tipo de filtro solicitado (por ejemplo, un filtro de tipo *filterContains* en una columna numérica).

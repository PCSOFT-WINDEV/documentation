
## NavigationPaneVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NavigationPaneVisible** Property se utiliza para mostrar el panel de miniaturas en el PDF Reader control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the navigation pane
PDF_MyPDF.NavigationPaneVisible = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el panel de miniaturas se muestra en el lector de PDF control

`<Result> = <PDF Reader control>.NavigationPaneVisible`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el panel de miniaturas es visible,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.


<a name="SYNTAX2"></a>

### Modificando la visualización del panel de miniaturas en el lector de PDF control

`<PDF Reader control>.NavigationPaneVisible = <New value>`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<New value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el panel de miniaturas es visible,

- <u><u><u><u>False</u></u></u></u> si el panel de miniaturas está oculto.

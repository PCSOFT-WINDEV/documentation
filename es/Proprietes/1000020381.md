


## RulerModifiable (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **RulerModifiable** permite: 

- Averiguar si el usuario puede mover el cabezal de reproducción en una control Línea de tiempo. 

- Permitir o impedir que el usuario mueva el cabezal de reproducción en una control Línea de tiempo. 




**Recordatorio**: El cabezal de reproducción de una control Línea de tiempo es una línea vertical en negrita que representa un punto específico en la control. El estilo del cabezal de reproducción se puede definir en la pestaña "Estilo" de la control Línea de tiempo ("Tiempo seleccionado").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on the BTN_Modify button
TL_TimeLine.RulerModifiable = True

// Whenever the TimeLine control is modified
// (Code run whenever the playhead is moved by the user)
// Display the selected position in the Static control
STC_RulerPosition = TL_TimeLine.RulerValue
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el usuario puede mover el cabezal de reproducción

`<Result> = <TimeLine control>.RulerModifiable`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el cabezal de reproducción puede ser movido por el usuario, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo utilizado. 


<a name="SYNTAX2"></a>

### Permitir o impedir que el usuario mueva el cabezal de reproducción

`<TimeLine control>.RulerModifiable = <Modifiable or not>`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo utilizado. 

**`<Modifiable or not>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para permitir al usuario mover el cabezal de reproducción,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El usuario puede mover el cabezal de reproducción: 

	- si la opción "Cabezal desplazable" está marcada en la ventana de descripción de control Línea de tiempo. 

	- si la propiedad **RulerModifiable** se establece en <u><u><u><u>True</u></u></u></u>. 




- La posición por defecto del cabezal de reproducción corresponde al valor de la [StartTotalRange](../Proprietes/1000020392.md) Property. 

- El usuario final puede cambiar el valor del cabezal de reproducción moviéndolo con el ratón. Esta acción desencadena la "Siempre que se modifica" Event del control Línea de tiempo. 

- La nueva posición del cabezal de reproducción es devuelta por el [Valor de regla](../Proprietes/1000020380.md) Property.

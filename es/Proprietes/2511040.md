
## HTML (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HTML** Property se utiliza para: 

- averiguar si la reporte actual se imprime en modo HTML. Por lo tanto, algunos controles que no se requieren cuando se imprime en modo HTML pueden hacerse Invisible (número de página por ejemplo).

- gestionar el tipo HTML de una control Estático que se encuentra en una reporte.




**Observaciones**:

- La impresión en modo HTML la realiza [iDestination](../WDLang5/3046074.md).

- El **HTML** Property no puede utilizarse en el código de apertura del reporte. Esta Property puede ser utilizada en todos los demás códigos de los controles o bloques de la reporte y la reporte (código de inicialización de un bloque, por ejemplo).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Initialization code of end of document
// If the current report is printed in HTML mode
// The page number is not printed. 
// The page number is associated with the GR_GROUP1 group of controls
IF RPT_Report2.HTML = True THEN GR_GROUP1.Visible = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la actual reporte se imprime en modo HTML

`<Result> = <Report name>.HTML`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprimió el reporte en modo HTML, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Report name>: (Cadena de caracteres)`**

Nombre de la reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Averiguar si la control Estático está en formato HTML

`<Result> = <Control name>.HTML`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Estático es de tipo HTML, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control name>: (Cadena de caracteres)`**

Nombre de la control Estático que se va a utilizar.  




<a name="SYNTAX3"></a>

### Modificar el tipo de HTML de una control Estático

`<Control name>.HTML = <New type>`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de la control Estático que se va a utilizar. 

**`<New type>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Estático debe ser de tipo HTML, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **HTML** Property no puede utilizarse en el código de apertura del reporte.

- El **HTML** Property se aplica únicamente a: 

	- reportes. 

	- Controles de estática en los informes.

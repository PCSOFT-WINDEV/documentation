
## UseMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **modo de uso** Property se utiliza para determinar o cambiar cómo reacciona el lector de PDF control a la entrada del mouse.. El ratón se utiliza para seleccionar el texto en la Document o para moverse dentro de las páginas de Document.

El lector de PDF está en modo de selección por Default. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switches the control to move mode
PDF_MyPDF.UseMode = useModeMovement
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Cómo reacciona el lector de PDF control a la entrada del ratón

`<Result> = <PDF Reader control>.UseMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo actual para usar el PDF Document: 


|   |   |
| --- | --- |
| *useModeMovement* | El control está en modo de movimiento: <br><br>- el botón izquierdo del ratón le permite moverse dentro de la Document. <br><br>- el cursor del ratón se convierte en una mano. <br><br>- si aparece la cinta de opciones, se selecciona la opción "Mover" (panel "Visualización", grupo "Modo"). <br><br><br><br><br>Hasta la versión 27, esta constante se llamaba *modeUseMovement* . |
| *useModeOCR* | El control está en modo de selección: <br><br>- el botón izquierdo del ratón permite seleccionar los elementos de la Document. <br><br>- el cursor cambia a un lazo. <br><br>- si aparece la cinta de opciones, se selecciona la opción "OCR" (pestaña "Visualización", grupo "Modo"). <br><br><br> |
| *useModeSelection* | El control está en modo de selección: <br><br>- el botón izquierdo del ratón permite seleccionar los elementos de la Document. <br><br>- el cursor se convierte en un cursor de texto. <br><br>- si aparece la cinta de opciones, se selecciona la opción "Selección" (panel "Visualización", grupo "Modo"). <br><br><br><br><br>Hasta la versión 27, esta constante se llamaba *modeUseSelection* . |



**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular. 


<a name="SYNTAX2"></a>

### Cambiar la reacción del lector de PDF control a la entrada del ratón

`<PDF Reader control>.UseMode = <New mode>`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

| *useModeOCR* | El control está en modo de selección: <br><br>- el botón izquierdo del ratón permite seleccionar los elementos de la Document. <br><br>- el cursor cambia a un lazo. <br><br>- si aparece la cinta de opciones, se selecciona la opción "OCR" (pestaña "Visualización", grupo "Modo"). <br><br><br> |
| *useModeSelection* | El control está en modo de selección: <br><br>- el botón izquierdo del ratón permite seleccionar los elementos de la Document. <br><br>- el cursor se convierte en un cursor de texto. <br><br>- si aparece la cinta de opciones, se selecciona la opción "Selección" (panel "Visualización", grupo "Modo"). <br><br><br><br><br>Hasta la versión 27, esta constante se llamaba *modeUseSelection* . |

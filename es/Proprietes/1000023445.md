
## SearchAAF (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SearchAAF** permite: 

- averiguar si la búsqueda a través de AAF está habilitada en un control Combo Box no editable,

- permitir (o no) la búsqueda a través de AAF en un control Combo Box no editable. 




Observación: Esta propiedad equivale a la opción "Con búsqueda" de la pestaña "Detalles" de la ventana Combo Box description. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RechercheFAA%20-%20HC%20N%B0001.gif)
 Se muestra un control de búsqueda cuando se expande la Combo Box. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RechercheFAA%20-%20HC%20N%B0002.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on the [Enable the search] button
COMBO_Customer.SearchAAF = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si la búsqueda está habilitada en un control Combo Box no editable

`<Result> = <Control used>.SearchAAF`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la búsqueda está activada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre de control Combo Box no editable a utilizar.


<a name="SYNTAX2"></a>

### Habilitar (o no) la búsqueda en un control Combo Box no editable

`<Control used>.SearchAAF = <Activation/Deactivation>`
---

**`<Control used>: (Nombre del control)`**

Nombre de control Combo Box no editable a utilizar.

**`<Activation/Deactivation>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar la búsqueda,

- <u><u><u><u>False</u></u></u></u> para desactivar la búsqueda.

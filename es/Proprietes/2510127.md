


## StoredValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StoredValue** Property se utiliza para get el valor almacenado actualmente por: 

- una fila de un control List Box, ListView o Combo Box. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un "Texto Token" control Campo de entrada.

- a fila de a control Tabla. 


El valor almacenado actualmente puede corresponder a:

- **para los controles de List Box, ListView, y Combo Box basados en un archivo de datos**: el valor del elemento almacenado definido en el editor de la ventana (pestaña "Contenido" del control description): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=rubriquememorisee.gif)
. 

- **para el cuadro de lista, ListView, y Combo Box controles rellenados mediante programación**: el valor definido por [gStoredValue](../WDLang1/1410088363.md).
	Recordatorio: Para recuperar el valor definido por [gStoredValue](../WDLang1/1410088363.md):

	- utilice la función [gStoredValueEnable](../WDLang1/1410088364.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) marque "Devolver el valor especificado con la función gStoredValue" (pestaña "Detalles" del control description). 


 **Observación**: Esta Property puede utilizarse para recuperar los valores almacenados de un Cuadro de Lista de selección múltiple.

- **para "Texto Token" editar controles**: el valor de la Token. 

- **para un fila de un control Tabla poblado programáticamente**: el valor almacenado de la fila. Para este tipo de control, también se puede asignar el valor almacenado del fila. 

- una estructura del array, para un fila de un control Tabla basado en un Variable, poblado por un array de estructuras. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Values selected in a multi-selection List Box control
i is int = 1
iSelectedRow is int
iSelectedRow = ListSelect(LIST_Cedex, 1)
WHILE iSelectedRow <> -1
	Trace(LIST_Cedex[iSelectedRow].StoredValue)
	i++
	iSelectedRow = ListSelect(LIST_Cedex, i)
END
```


<a name="Example2"></a>

```wl
// Find out the stored value when selecting the 3rd element of the List Box control
Info("The stored value is " + LIST_Cedex[3].StoredValue)
```
<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Find out the value stored by the "Text token" edit control
Info("The stored value is " + EDT_Token.StoredValue)
// Returns for example: 
// TOKEN_ADDED+CR+TOKEN_ADDED2
```

## Sintaxis

### Obtención del valor almacenado de un cuadro de lista, ListView o control Combo Box

`<Stored value> = <Control used>[<Element index>].StoredValue`
---

**`<Stored value>: (Cadena de caracteres)`**

Valor que se almacenará si el usuario selecciona el elemento en la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre del List Box, ListView o control Combo Box a utilizar.

**`<Element index>: (Integro)`**

Índice del elemento de la control (List Box, ListView o Combo Box) cuyo valor almacenado se solicita.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Obtener el valor almacenado en un "Texto Token" control Campo de entrada

`<Stored value> = <Edit control>.StoredValue`
---

**`<Stored value>: (Cadena de caracteres)`**

Valor almacenado. Si la control tiene varios tokens, este parámetro tendrá la siguiente sintaxis: 

```txt
<Value of Token 1> + CR + ... + <Value of Token N>
```


**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" control Campo de entrada a utilizar. 


<a name="SYNTAX3"></a>

### Obtener el valor almacenado de una fila de una control Tabla

`<Stored value> = <Table control>[<Row index>].StoredValue`
---

**`<Stored value>: (Cadena de caracteres)`**

Valor almacenado para la fila. 

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular. 

**`<Row index>: (Integro)`**

Índice de la fila cuyo valor se solicita.

## Observaciones


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000143"></a>

El **StoredValue** Property se aplica únicamente a: 

- List Box, controles ListView y Combo Box.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) "Texto Token" controles de edición. 

- la controles Tabla.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **StoredValue** Property se aplica sólo a los controles List Box y Combo Box.

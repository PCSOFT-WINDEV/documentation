


## CaptionHTML (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CaptionHTML** Property se utiliza para manipular el texto de un control como código HTML.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LINK_GSC.CaptionHTML = "Display the <strong>general sales conditions</strong>"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperando el valor de la leyenda de control

`<Result> = <Control to use>.CaptionHTML`
---

**`<Result>: (Cadena de caracteres)`**

Pie de foto de control en formato HTML.

**`<Control to use>: (Nombre del control)`**

Nombre del control a manipular: Área de texto enriquecido, botón, etc.


<a name="SYNTAX2"></a>

### Modificando el valor de la leyenda de control

`<Control to use>.CaptionHTML = <New value>`
---

**`<Control to use>: (Nombre del control)`**

Nombre del control a manipular: Área de texto enriquecido, botón, etc.

**`<New value>: (Cadena de caracteres)`**

Nuevo pie de foto para el control. Este valor debe corresponder a un código HTML válido.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Estático y estático HTML
<a name="estatico_estatico_html_ELTPARAGRAPHE000049"></a>

Una control incluye un pie de foto o un pie de foto HTML: no puede incluir ambos al mismo tiempo. Al asignar un valor al **CaptionHTML** Property se borra el valor del texto asignado con el [Pie de foto](../Proprietes/2510053.md) Property (e inversamente).

Se produce un error si se accede a la [Pie de foto](../Proprietes/2510053.md) Property en modo de sólo lectura y el texto de la control está en HTML. Sin embargo, la lectura del **CaptionHTML** Property en un control cuyo título ha sido asignado mediante el [Pie de foto](../Proprietes/2510053.md) Property devuelve el valor del título en HTML. 

Tabla de resumen: 


|   |   |   |
| --- | --- | --- |
|   | Property asignado en el control: [Pie de foto](../Proprietes/2510053.md) | Property asignado en el control: **CaptionHTML** |
| Property leer: [Pie de foto](../Proprietes/2510053.md) | Devuelve el valor. | Error. |
| Property leer: **CaptionHTML** | Devuelve el valor codificado en HTML. | Devuelve el valor. |




### Validez del código HTML
<a name="validez_del_codigo_html_ELTPARAGRAPHE000096"></a>

No se realiza ninguna comprobación sobre la validez del código HTML. Si se asigna un valor HTML inválido en la control, es posible que el navegador no muestre correctamente la página. Debes prestarle atención:

- Las etiquetas (y atributos) no están correctamente cerradas o no son válidas para el estándar HTML de la página (HTML 4, HTML 5, ....).. 

- La codificación de caracteres especiales. Utilice la función [TextToHTML](../WDLang1/3025050.md).





### Limitación
<a name="limitacion_ELTPARAGRAPHE000107"></a>

Este Property está disponible para los siguientes controles: 

- Control Botón, 

- Control Enlace, 

- Control Área de texto enriquecido, 

- Control Campo de entrada, 

- Control Casilla de verificación, 

- Control Botón de opción, 

- Control List Box, 

- Control de visualización con formato, 

- Control Combo Box, 

- Opción de control Casilla de verificación,

- Opción de control Botón de opción,

- Opción de menú.

- Columna de tabla.




## GenerationNumber (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **GenerationNumber** permite obtener el número de generación del archivo de datos físico asociado al archivo de datos lógico especificado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the generation number of the Customer data file
ResGenerationNum = Customer.GenerationNumber
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Data file>.GenerationNumber`
---

**`<Result>: (Integro)`**

Número de generación del archivo de datos físicos actualmente associated con el archivo de datos lógicos especificado.

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado.

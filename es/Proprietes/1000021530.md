


## HTMLClass (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HTMLClass** Property se utiliza para leer y cambiar el valor del atributo HTML "class" del control. 

Observación: El valor del atributo HTML "class" se utiliza para seleccionar los estilos CSS aplicados a la control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Applies a CSS to style the control if an error occurs
// and systematically targets the control to apply an auto-completion JS
IF Position(EDT_Edit, "@") = -1 THEN
    EDT_Edit.HTMLClass = "InputInError ui-autocomplete"
ELSE
    EDT_Edit.HTMLClass = "InputValid ui-autocomplete"
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Leyendo el valor del atributo "clase"...

`<Result> = <Control used>.HTMLClass`
---

**`<Result>: (Cadena de caracteres)`**

Valor del atributo "class. Si este atributo contiene varios valores, los diferentes valores están separados por espacios. 

Observación: En tiempo de ejecución, el valor inicial del atributo "clase" corresponde a la concatenación de valores definidos durante la edición (con el carácter espacio como separador) en la ventana de control description.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.


<a name="SYNTAX2"></a>

### Modificar el valor del atributo "clase".

`<Control used>.HTMLClass = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. 

**`<New value>: (Cadena de caracteres)`**

Nuevo valor para el atributo "class. Si este atributo contiene varios valores, los diferentes valores están separados por espacios. 
Atención: 

- Este parámetro distingue entre mayúsculas y minúsculas. 

- El contenido existente del atributo "class" se sobrescribe con el nuevo valor. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- No se realiza ninguna verificación del valor. En HTML, todos los caracteres están permitidos. Si se asigna un valor inválido, el navegador probablemente mostrará la página con errores.

- El atributo "class" en HTML también incluye un contenido específico añadido por WEBDEV para definir el estilo y asegurarse de que la control funciona correctamente. A este contenido adicional no se puede acceder con el **HTMLClass** Property.

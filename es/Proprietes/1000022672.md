


## SelectionDirection (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SelectionDirection** Property se utiliza para determinar y cambiar la dirección de selección (de izquierda a derecha o de derecha a izquierda) en un control Procesador de texto. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
T is int = 30
FOR I = WP_MyDoc.Cursor _TO_ WP_MyDoc.Cursor + WP_MyDoc.SelectionLength
	f is docFragment = WP_MyDoc[i ON 1]
	f.Style.FontSize = T
	IF WP_MyDoc.SelectionDirection = 1 THEN
		T += 5
	ELSE
		T -= 5
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la dirección de la selección en un control Procesador de texto

`<Result> = <Word Processing control>.SelectionDirection`
---

**`<Result>: (Integro)`**

Dirección de selección:

- 1: La selección se realizó de acuerdo a la dirección de escritura (de izquierda a derecha). 

- 2: La selección se realizó en orden inverso al de la dirección de escritura (de derecha a izquierda).. 




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.


<a name="SYNTAX2"></a>

### Modificar la dirección de selección en un control Procesador de texto

`<Word Processing control>.SelectionDirection = <New value>`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<New value>: (Integro)`**

Nueva dirección de selección:

- 1: La selección se realizó de acuerdo a la dirección de escritura (de izquierda a derecha). 

- 2: La selección se realizó en orden inverso al de la dirección de escritura (de derecha a izquierda)..




## PageNumber (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **PageNumber** permite: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) get y cambiar el número de la página mostrada en un control Imagen (archivos Image "multipágina").

- get y cambia el número Page de un archivo PDF mostrado en un control Imagen de una ventana. 

- get y cambiar el número Page de un fichero PDF que aparece en el Background de un reporte. 




**Recordatorio**: Algunos formatos de Image (TIFF por ejemplo) pueden contener varios Image s en el mismo archivo. Cada Image corresponde a un fichero Page. Estos formatos se denominan formatos "multi-Page".

Observación: Para obtener y cambiar el número de la página que se muestra en un control de Procesamiento de Palabras o Lector de PDF, utilice la función [NumberDisplayedPage](../Proprietes/1000023479.md) Property. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
// Variables
i is int
// Modify the image file associated with the "IMG_MyImage" control in a window
IMG_MyImage = "MultipageFile.TIF"
// Successively display the different pages found in the image file
FOR I = 1 TO IMG_MyImage.NumberPage
	IMG_MyImage.PageNumber = I
	// Timeout for loading images
	Multitask(-100)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificar el número de la Page mostrada

`<Page number> = <Element used>.PageNumber`
---

**`<Page number>: (Integro)`**



- Número de la Page que se muestra actualmente en la control Imagen especificada o en el fondo de la reporte especificada .

- 1 si el formato del fichero Image no es "multi-Page" o si se visualiza el primer Page del fichero "multi-Page".

- 0 si el control especificado o reporte no es associated con un formato Image soportado.




**`<Element used>: (Nombre control o reporte)`**

Nombre de la control Imagen o reporte que se va a utilizar.  


<a name="SYNTAX2"></a>

### Cambiar el número de la Page mostrada

`<Element used>.PageNumber = <Page number>`
---

**`<Element used>: (Nombre control o reporte)`**

Nombre de control Imagen o reporte a utilizar.

**`<Page number>: (Integro)`**

Número de la Page que se mostrará en la control Imagen especificada o en el fondo de la reporte especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modificación de la Page visualizada en un control Imagen
<a name="modificacion_page_visualizada_control_imagen_ELTPARAGRAPHE000072"></a>

Cuando se asocia una imagen o un fichero PDF a un control (desde el editor de ventanas, a través de la [Image](../Proprietes/2510034.md) Property, etc.), la imagen mostrada por Default corresponde a la primera Page del fichero "multi-Page".

Para cambiar la página que aparece en la control Imagen, basta con utilizar la **PageNumber** Property.

**Atención**: El **PageNumber** Property sólo se aplica al archivo de imagen actualmente asociado al control Imagen.
<a name="NOTE0_2"></a>


### Modificación de la Page del fichero PDF visualizado en la Background de un reporte
<a name="modificacion_page_del_fichero_pdf_visualizado_background_reporte_ELTPARAGRAPHE000090"></a>

Si se utiliza un archivo PDF como formulario reporte, tiene la posibilidad de elegir (en el reporte description) el Page del archivo PDF que se mostrará (pestaña "Fondo", opción "# de PDF Page"). Para cambiar mediante programación la página que se muestra en el fondo de la reporte, basta con utilizar la **PageNumber** Property.

**Atención**: El **PageNumber** Property se aplica sólo al archivo de imagen utilizado actualmente como formulario reporte.
<a name="NOTE0_3"></a>


### Número de Page s encontrados en un fichero "multi-Page".
<a name="numero_page_encontrados_fichero_multipage_ELTPARAGRAPHE000103"></a>

Para obtener el número de páginas de un archivo de imagen "multipágina", utilice el programa [NumberPage](../Proprietes/2510084.md) Property.

### Dibujo en una control Imagen associated con una Image"multipágina".
<a name="dibujo_una_control_imagen_associated_con_una_imagemultipagina_ELTPARAGRAPHE000115"></a>

Si se realiza un dibujo (realizado con [funciones de dibujo](../WDLang1/3029033.md)) en el control Imagen, este dibujo se borrará automáticamente cuando se modifique la página visualizada con el **PageNumber** Property.

Este dibujo debe ser re-dibujado cada vez que se modifica la página mostrada en la imagen control. Para re-dibujar este dibujo, le recomendamos que use una Procedure.
<a name="NOTE0_5"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000129"></a>

La propiedad **PageNumber** solo se aplica a:

- Controles de Image en las ventanas.

- reportes.




## InitialHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InitialHeight** Property se utiliza para:

- **Findicar la altura de creación de un control o ventana.**
	La altura de creación corresponde a la altura definida en el editor de la ventana durante la description del objeto. Esta altura puede evolucionar y puede ser modificada. Lo devuelve la [Altura](../Proprietes/2510050.md) Property.

- **Modificar la altura de la base utilizada para el anclaje de los mandos "en altura".**
	Cuando un control está anclado, la altura base utilizada para calcular cómo se redimensiona el control según la ventana es la altura definida cuando se creó el control (devuelta por **InitialHeight** Property).
	Para modificar la altura de una control anclada en altura, hay que modificar también su altura inicial.

- **Conocer la altura de un control o bloque de un reporte al crearlos.**
	La altura de creación corresponde a la altura definida en el editor de reportes durante el description del objeto. Esta altura puede evolucionar y puede ser modificada. Esta nueva altura es devuelta por la [Altura](../Proprietes/2510050.md) Property.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increases the height of "STC_Title" by 30 millimeters
STC_Title.Height = STC_Title.InitialHeight + 30
```
<a name="Example2"></a>

```wl
// Restores the initial height of the control
EDT_Edit1.Height = EDT_Edit1.InitialHeight
// Enlarges TABLE_Table1 that is anchored in height by 30 pixels
TABLE_Table1.Height += 30
TABLE_Table1.InitialHeight += 30
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura de la creación

`<Creation height> = <Element used>.InitialHeight`
---

**`<Creation height>: (Integro)`**

Altura inicial del objeto especificado (incluyendo la Border). Esta altura se expresa en píxeles en una ventana y/o en una página, y en milímetros en una reporte.

**`<Element used>: (Nombre de control, ventana o bloque)`**

Nombre del elemento (control, ventana o bloque reporte) cuya altura inicial debe calcularse.

### Modificación de la altura inicial de una control anclada en altura

`<Control used>.InitialHeight = <New height>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control anclado cuya altura inicial debe ser modificada.

**`<New height>: (Integro)`**

Nueva altura inicial del control anclado (incluyendo el Border) expresada en píxeles.

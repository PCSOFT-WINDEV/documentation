


## CtDescribedDatabase (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **CtDescribedDatabase** permite manipular la fuente de datos OLE DB en una conexión OLE DB a una tabla específica. Es posible:

- get la fuente de datos OLE DB utilizada para conectarse a un tabla vía OLE DB. Esta fuente de datos se describió en el editor de análisis o cuando se describió el el archivo de datos a través de la programación.
	Atención: la base de datos **CtDescribedDatabase** Property ignora cualquier cambio realizado con [HConnect](../WDLang4/3044263.md).
	Este Property puede utilizarse para el un archivo de datos definido en el editor de análisis o mediante programación.

- Definir la fuente de datos OLE DB utilizada para conectarse a un tabla por OLE DB (cuando el archivo de datos se describe mediante programación). Esta Property es equivalente a la [CtDatabase](../Proprietes/2512047.md) Property. 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describe el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the OLE DB data source
ResOLEDBDatabase = Customer.CtDescribedDatabase
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la fuente de datos de la OLE DB

`<OLE DB data source> = <Data file>.CtDescribedDatabase`
---

**`<OLE DB data source>: (Cadena de caracteres)`**

ruta completa de la fuente de datos OLE DB, especificada durante el description del archivo.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definir la fuente de datos de la OLE DB

`<Data file>.CtDescribedDatabase = <OLE DB data source>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<OLE DB data source>: (Cadena de caracteres)`**

ruta completa de la fuente de datos OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La [CtDatabase](../Proprietes/2512047.md) Property se utiliza para get y cambiar la fuente de datos especificada para la conexión OLE DB.


## Definition (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Definition** Property se utiliza para definir un tipo. Esta propiedad es utilizada por:

- una variable de tipo [Property Description](../WDLang1/1000019324.md).

- una variable de tipo [Variable Description](../WDLang1/1000019560.md).

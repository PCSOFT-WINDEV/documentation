
## SystemBarColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SystemBarColor** Property se utiliza para get o cambiar la barra del sistema Color en las ventanas de pantalla completa en los dispositivos móviles.

Atención: la barra Color no se puede cambiar en los dispositivos Nexus.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWindow.SystemBarColor = PastelGreen
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la barra del sistema Color

`<Result> = <Window used>.SystemBarColor`
---

**`<Result>: (Entero o constante)`**

Color barra de sistema. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).




**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana a utilizar. 


<a name="SYNTAX2"></a>

### Cambio de la barra del sistema Color

`<Window used>.SystemBarColor = <New color>`
---

**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana a utilizar. 

**`<New color>: (Entero o constante)`**

Nueva Color para el elemento especificado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [y la preselección Color](../WDLang5/3010002.md), 

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *AutomaticColor* | La barra del sistema Color está adaptada al Action Bar (por Default). |
| *Transparent* | La barra del sistema es Transparent. |

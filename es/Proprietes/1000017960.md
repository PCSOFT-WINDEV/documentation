
## Address

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Address** puede corresponder a: 

- la propiedad **Address** que puede utilizarse en un control Conferencia. Para obtener más información, consulte [Address](../Proprietes/2510017.md).

- la propiedad **Address** que puede utilizarse en: 

	- una variable de tipo [gglPostalAddress](../WDLang5/1000017575.md).

	- una variable de tipo [gglContact](../WDLang5/1000017434.md).

	- una variable de tipo [gglEmail](../WDLang5/1000017571.md).

	- una variable de tipo [gglInstantMessaging](../WDLang5/1000017569.md).

	- una variable de tipo [COMObject](../WDLang1/1000018983.md).

	- una variable generada automáticamente para representar [un servicio web importado](../WDLang3/3086001.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable generada automáticamente para representar [un servicio REST importado](../WDLang3/1410087809.md).

	- una variable de tipo [API description](../WDLang1/1000019149.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una variable de tipo [sshSession](../WDLang3/1000021157.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) una variable de tipo [mqttSession](../WDLang3/1000023023.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WBAdminAccount](../WDLang2/1410089382.md).

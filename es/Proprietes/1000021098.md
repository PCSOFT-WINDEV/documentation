


## Pane (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Pane** permite: 

- Averigua las pestañas del panel associated con un control. 

- Definir las pestañas del panel associated con un control. 




**Observación**: Este Property funciona sólo en los paneles de pestañas estándar. Este Property no funciona en los paneles de pestañas dinámicas. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Associate the Validate button to panes 1, 2 and 4
BTN_Validate.Pane = "1,2,4"

// Find out the panes associated with a button
IF BTN_Validate.Pane [=] "2" THEN Info("The control is associated with pane 2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar las pestañas del panel associated con un control

`<Result> = <Control used>.Pane`
---

**`<Result>: (Entero o cadena de caracteres)`**



- Número de panel del panel associated con el control. El número de panel del primer panel es 1. 

- Cadena de caracteres que contiene la lista de paneles associated con el control, utilizando el siguiente formato: 
	
	```txt
	"<Number pane 1>,<Number pane 2>,...,<Number pane N>"
	```


- 0 si la control no pertenece a ningún panel. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.


<a name="SYNTAX2"></a>

### Definir las pestañas del panel associated con un control

`<Control used>.Pane = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<New value>: (Entero o cadena de caracteres)`**



- Número de panel del panel associated con el control. El número de panel del primer panel es 1. 

- Cadena de caracteres que contiene la lista de paneles associated con el control, utilizando el siguiente formato: 
	
	```txt
	"<Number pane 1>,<Number pane 2>,...,<Number pane N>"
	```







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el control no pertenece a ninguna pestaña, no puede ser associated con uno o más paneles.

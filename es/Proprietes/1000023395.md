
## NamespaceDeclared (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NamespaceDeclared** Property se utiliza para gestionar los espacios de nombres definidos. Esta propiedad se puede utilizar en:

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [xmlReader](../WDLang5/1000023392.md).

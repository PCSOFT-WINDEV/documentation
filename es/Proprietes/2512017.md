


## CtExtendedInfo (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtExtendedInfo** Property se utiliza para get la información adicional insertada en la cadena de conexión de la base de datos. Puede get la información adicional especificada para la conexión a un tabla vía OLE DB por ejemplo.

Esta información fue especificada:

- cuando se creó la conexión en la editor de análisis.

- cuando la conexión fue definida en [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).




Este Property puede utilizarse para un archivo de datos o para una conexión definida en el editor de análisis o mediante programación.

**Recordatorio**: Información adicional: Información insertada en la cadena para conectarse a la base de datos.

Para más detalles, consulte la documentación suministrada por el proveedor del acceso utilizado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the extended information about the connection used
// by the Customer data file
Info(Customer.CtExtendedInfo)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Extended information> = <Data file\connection>.CtExtendedInfo`
---

**`<Extended information>: (Cadena de caracteres)`**

Información insertada en la cadena para la conexión a la base de datos. 

**`<Data file\connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre de la conexión utilizada. Este nombre se definió en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

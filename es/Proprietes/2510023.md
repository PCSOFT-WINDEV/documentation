


## HorizontalAlignment (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>
La alineación horizontal de una control consiste en definir la posición del texto en la control en relación con los bordes de la control. Permite centrar el contenido de una control Estático en relación con los bordes de la control.

Hay varias alineaciones disponibles:

- alineación a la izquierda <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cadrage_gauche.gif)


- alineación a la derecha <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cadrage_Droite.gif)


- centrado <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cadrage_centre.gif)


- justificado



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Alineación Horizontal** Property se utiliza para get y cambiar la alineación horizontal de un control.

La alineación inicial del control se define en el editor durante el control description.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the control is left-aligned, align it to right
IF STC_Static1.HorizontalAlignment = haLeft THEN
	STC_Static1.HorizontalAlignment = haRight
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la alineación horizontal de una control

`<Current alignment>.HorizontalAlignment = <Control used>`
---

**`<Current alignment>: (Constante de tipo Integer)`**

La actual alineación horizontal de control. Los valores posibles son: 



|   |   |
| --- | --- |
| *haCenter* | El texto está centrado. |
| *haJustified* | El texto está justificado. |
| *haLeft* | El texto está alineado a la izquierda. |
| *haNotDefined* | Alineación definida por la herencia CSS.<br>Esta constante sólo está disponible en WEBDEV. |
| *haRight* | El texto está alineado a la derecha. |



**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificar la alineación horizontal de una control

`<Control used>.HorizontalAlignment = <New alignment>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. 

**`<New alignment>: (Constante de tipo Integer)`**

Nueva alineación horizontal de la control. Los valores posibles son: 



|   |   |
| --- | --- |
| *haCenter* | El texto está centrado. |
| *haJustified* | El texto está justificado. |
| *haLeft* | El texto está alineado a la izquierda. |
| *haNotDefined* | Alineación definida por la herencia CSS.<br>Esta constante sólo está disponible en WEBDEV. |
| *haRight* | El texto está alineado a la derecha. |

### Alineación horizontal y texto que no se muestra completamente en la control
<a name="alineacion_horizontal_texto_que_muestra_completamente_control_ELTPARAGRAPHE000154"></a>

La siguiente información se muestra si el texto no puede mostrarse completamente en la control:

- En el editor de reportes, se muestra la parte izquierda del texto.

- En el editor de ventanas:

	- en una control estática, se muestra la parte derecha del texto.

	- en una control Campo de entrada, se muestra la parte izquierda del texto.

### Limitación
<a name="limitacion_ELTPARAGRAPHE000177"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **HorizontalAlignment** Property se utiliza para get y cambiar la alineación horizontal para los siguientes tipos de controles:

- Control Estático

- Control Campo de entrada

- Combo Box (editable o no)

- Casilla de verificación / Botón de radio

- List Box

- ListView

- Image

- Columna en una tabla y columna en una Tabla TreeView

- Celda en una barra de estado

- Código de barras

- Gráfico

- Barra de progreso. La Property se utiliza para averiguar y modificar la alineación horizontal para el porcentaje de progreso mostrado en la barra de progreso.

- Control Valoración

- Organigrama


 
 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **HorizontalAlignment** Property se utiliza para get y cambiar la alineación horizontal para los siguientes tipos de controles:

- Control Estático.

- Control Calculado.

- Control Predefinido.

- Control Gráfico.

- Grupos de controles. Esta Property sólo se aplicará a los controles que admitan la alineación horizontal.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **HorizontalAlignment** Property se aplica sólo a los siguientes controles:

- Control Estático.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- Columna de la tabla (texto, cuadro combinado o casilla de verificación).

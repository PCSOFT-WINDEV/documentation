


## SourcePage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SourcePage** se utiliza para identificar la fuente de una página interna (es decir, la página interna mostrada por un control Internal Page ).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(IP_InternalPageControl1.SourcePage) // Displays IP_InternalPage2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Internal page used> = <Internal Page control>.SourcePage`
---

**`<Internal page used>: (Cadena de caracteres)`**

Nombre de la página interna utilizada por la "Página interna" control.

**`<Internal Page control>: (Nombre del control)`**

Nombre de la página interna control.




## SQLCode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SQLCode** permite obtener el código SQL de una consulta creada con el editor de consultas (archivo ".WDR"). Los comentarios introducidos en el editor de consultas son Visible en el código SQL.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Assigns the SQL code of the query to a string
MySQLCode is string
MySQLCode = QRY_QueryCalculation.SQLCode
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el código SQL de una consulta

`<SQL code> = <Query used>.SQLCode`
---

**`<SQL code>: (Cadena de caracteres)`**

Código SQL de la consulta especificada. Si el archivo ".WDR" contiene comentarios, éstos se encontrarán en el código SQL de la consulta (los comentarios comienzan por "--").

**`<Query used>: (Nombre de la consulta)`**

Nombre lógico de la consulta utilizada. Este nombre se definió en el editor de consultas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Formatear el resultado
<a name="formatear_resultado_ELTPARAGRAPHE000050"></a>

El código SQL de la consulta es idéntico al que se muestra en el editor de consultas a través de "Código SQL" en el menú contextual de la consulta: se mantienen los caracteres de espacio, las tabulaciones y los caracteres CR.
<a name="NOTE0_2"></a>


### Uso en los procedimientos almacenados en HFSQL
<a name="uso_los_procedimientos_almacenados_hfsql_ELTPARAGRAPHE000057"></a>

La propiedad **SQLCode** se puede utilizar en el código de una Procedure almacenada para get el código de una consulta almacenada en el servidor.

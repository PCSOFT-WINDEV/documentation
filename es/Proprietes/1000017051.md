


## HTMLBefore (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**HTMLBefore** se utiliza para averiguar y modificar el código HTML añadido antes del control.

**Observación**: El código HTML generado antes del control puede ser introducido en la pestaña "Advanced" de la ventana description del control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the style of a button dynamically
BTN_DynButton.HTMLBefore = "<DIV style=""..."">"
BTN_DynButton.HTMLAfter = "</DIV>"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el contenido HTML insertado antes de la control

`<HTML content> = <Control used>.HTMLBefore`
---

**`<HTML content>: (Cadena de caracteres)`**

Contenido insertado antes de la control.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando el contenido HTML insertado antes de la control

`<Control used>.HTMLBefore = <HTML content>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<HTML content>: (Cadena de caracteres)`**

Contenido insertado antes de la control.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta opción está disponible para todos los controles que incluyen un código HTML que se puede insertar antes de. Este Property no está disponible para los siguientes controles:

- Control HTML,

- Control Tabla.

- Control Looper.

- Control Tabla TreeView.


## MinZoom (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MinZoom** Property se utiliza para get el valor mínimo de zoom que se puede utilizar para el control Cámara (para la cámara en uso).

Observación: Esta propiedad solo puede utilizarse en controles Cámara que no están en "Modo compatible con la versión 26". 

Observación: 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the maximum and minimum values for a Slider control
// to adjust the zoom of the Camera control 
SLD_Zoom.MinValue = CAM_MyCamera.MinZoom
SLD_Zoom.MaxValue = CAM_MyCamera.MaxZoom
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Camera control>.MinZoom`
---

**`<Result>: (Real)`**

Zoom mínimo de la control Cámara. 

**`<Camera control>: (Nombre del control)`**

Nombre de la control Cámara utilizada.

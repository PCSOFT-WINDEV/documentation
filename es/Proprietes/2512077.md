


## AccentSensitive (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AccentSensitive** Property se utiliza para determinar si se tienen en cuenta los caracteres acentuados en las diferentes búsquedas realizadas sobre el ítem o índice de texto completo (con [HReadSeek](../WDLang4/3044050.md)[HReadSeekFirst](../WDLang4/3044036.md) , 04fhex00530, por [HReadSeekLast](../WDLang4/3044034.md) ). Es posible:

- Averigüe si los caracteres acentuados se tienen en cuenta en las búsquedas realizadas en este elemento (o en este índice de texto completo).. El artículo o el índice de texto completo se ha definido en la editor de análisis o mediante programación.

- Definir la gestión de los caracteres acentuados para un artículo o para un índice de texto completo (cuando el artículo o el índice de texto completo se describen mediante programación).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define how the accented characters are managed in Orders.Name
Name.AccentSensitive = True
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si se tienen en cuenta los caracteres acentuados

`<Result> = <Data file>.<Item>.AccentSensitive`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los acentos se tienen en cuenta en las búsquedas realizadas en este elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre se definió en el editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**



- Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

- Nombre del índice de texto completo utilizado. Este nombre se definió en el editor de análisis o con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).  





<a name="SYNTAX2"></a>

### Definición del modo de gestión de los caracteres acentuados (elemento descrito mediante programación)

`<Item>.AccentSensitive = <New management mode>`
---

**`<Item>: (Cadena de caracteres)`**



- Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

- Nombre del índice de texto completo utilizado. Este nombre se definió con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).




**`<New management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los acentos deben tenerse en cuenta en las búsquedas realizadas en este elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Búsqueda realizada en un elemento "sensible a los acentos
<a name="busqueda_realizada_elemento_sensible_los_acentos_ELTPARAGRAPHE000108"></a>

Cuando se realiza una búsqueda en un elemento "sensible a los acentos", sólo se encontrarán las palabras con caracteres acentuados coincidentes. Por lo tanto, si el nombre del cliente es "Renée", la búsqueda realizada en:

- "Renée" encontrará la Record, 

- "Renee" no encontrará la Record.



<a name="NOTE0_2"></a>


### Configurar las diferentes opciones de búsqueda de un elemento
<a name="configurar_las_diferentes_opciones_busqueda_elemento_ELTPARAGRAPHE000118"></a>

- Para realizar búsquedas sensibles o insensibles a los acentos, utilice **AccentSensitive** Property .

- Para que las búsquedas distingan entre mayúsculas y minúsculas o no, utilice el botón [Sensible a mayúsculas y minúsculas](../Proprietes/2512085.md) Property.

- Para que las búsquedas sean sensibles o insensibles a la puntuación, utilice el botón [PuntuaciónSensible](../Proprietes/2512087.md) Property. 



<a name="NOTE0_3"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000135"></a>

El **AccentSensitive** Property no debe usarse en un elemento numérico o en una clave compuesta. De lo contrario, un resultado incorrecto puede ser devuelto..

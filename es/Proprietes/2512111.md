


## InfoThumbnailGeneration (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InfoThumbnailGeneration** Property se utiliza para determinar si las miniaturas de un elemento se han generado con éxito.

Este punto se definió en la editor de análisis o a través de la programación.

**Observación**: Las miniaturas HFSQL de un elemento se generan automáticamente cuando se realizan las siguientes operaciones:

- escribiendo el correspondiente record ([HAdd](../WDLang4/3044147.md) o [HWrite](../WDLang4/3044092.md)),

- cambiar el valor del artículo ([HModify](../WDLang4/3044042.md)),

- [automatic modificación de los archivos de datos](../WDLang4/3044195.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate the first thumbnail for the description of the Photo item?
InfoGene is string
InfoGene = Product.Photo.InfoThumbnailGeneration[1]
// equivalent to InfoGene = Photo.InfoThumbnailGeneration
```


<a name="Example2"></a>

```wl
// Generate the Nth thumbnail of the Product.Image item?
InfoGene is string
InfoGene = Product.Image.InfoThumbnailGeneration[num]

IF InfoGene = "" THEN
	// Thumbnail number greater than the expected number of thumbnails?
	IF num > Product.Image.NbDescribedThumbnail THEN
		Error("This thumbnail does not exist")
	ELSE
		Info("The thumbnail was successfully generated")
	END
ELSE
	// Generation error
	Error("Generation error", InfoGene)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Generation information> = <Data file>.<Item>.InfoThumbnailGeneration[<N]`
---

**`<Generation information>: (Cadena de caracteres)`**



- Vacíe la cadena (""") si la miniatura especificada se ha generado correctamente,

- Mensaje de error de lo contrario.




También se devuelve una cadena vacía ("""):

- si &lt;N&gt; es mayor que el número de miniaturas descritas para el elemento. Para obtener el número de miniaturas de un artículo determinado, utilice la función [NbDescripción de la miniatura](../Proprietes/2512114.md) Property.

- si la actual Record fue escrita por un motor Hyper File 9 (o anterior) (ver las Notas).




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<N>: (Entero opcional)`**

Número de la miniatura utilizada. Si no se especifica este parámetro, se devuelve la información de generación de la primera miniatura..

## Observaciones


### Limitación
<a name="limitacion_ELTPARAGRAPHE000098"></a>

El **InfoThumbnailGeneration** Property está disponible:

- en HFSQL Classic: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: en los artículos encontrados de un archivo de datos solamente.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **InfoThumbnailGeneration** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.




## GUIDAnalysis (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**..GUIDAnalysis** devuelve la GUÍA del análisis que se utilizó para crear el archivo de datos. Este identificador se utiliza en el archivo.REP (ver [Gestión del archivo.REP](../WDLang4/3044186.md) para más detalles). 

**Observación**: La GUÍA del análisis también se puede encontrar en la editor de análisis (pestaña "Notas" de la descripción del archivo).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GUIDAnalyse%20-%20HC%20N%B0001.gif)







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(Customer..GUIDAnalysis)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Analysis GUID> = <Data file>..GUIDAnalysis`
---

**`<Analysis GUID>: (Cadena de caracteres)`**



- GUÍA del análisis que se utilizó para crear el archivo de datos. 

- Cadena vacía ("") si el archivo de datos fue creado por programación o si el archivo utilizado no es un archivo de datos en formato HFSQL Classic o Cliente/Servidor.




**`<Data file>: (Cadena de caracteres (con o sin comillas))`**

Nombre del archivo de datos para el que se solicita la GUÍA del análisis.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
[..GUIDFile](../Proprietes/2512099.md) se utiliza para encontrar la GUÍA del archivo. Esta GUÍA se definió en el análisis que se utilizó para crear el archivo de datos.


## Background

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave "Background" puede corresponder a: 

- el tipo [Background](../WDLang1/1000022038.md) Variable,

- el **Fondo** Property de un Variable de tipo [docParagraph](../WDLang1/1000022483.md). 

- el **Fondo** Property de un Variable de tipo [docElementTextArea](../WDLang1/1000023420.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el **Fondo** Property de un Variable de tipo [diagSelection](../WDLang1/1410088172.md).

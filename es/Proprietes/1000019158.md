
## Parameter (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Parameter** permite obtener y cambiar ciertos parámetros. Esta propiedad se puede utilizar en:

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [API description](../WDLang1/1000019149.md).

- una variable de tipo [Descripción de ParallelTask](../WDLang1/1000021586.md).

- una variable de tipo [Sintaxis description](../WDLang1/1410089154.md).

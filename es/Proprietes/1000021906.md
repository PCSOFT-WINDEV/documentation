
## MaxNbSegment (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxNbSegment** Property se utiliza para get y cambiar el número máximo de segmentos Line para dibujar un Alambre control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a waypoint to the wire so that it follows the desired path
Waypoint1, Waypoint2 are Points 

Waypoint1.X = 280
Waypoint1.Y = 54

Waypoint2.X = 320
Waypoint2.Y = 54
WIRE_MyWire.MaxNbSegment = 20

WIRE_MyWire = [Waypoint1, Waypoint2]
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el número máximo de segmentos de Line para un control de alambre

`<Result> = <Wire control>.MaxNbSegment`
---

**`<Result>: (Integro)`**



- Número máximo de segmentos de Line que componen el alambre control,

- 1 corresponde al dibujo de una sola Line entre dos controles. 




**`<Wire control>: (Nombre del control)`**

Nombre del alambre control a utilizar.


<a name="SYNTAX2"></a>

### Modificando el número máximo de segmentos de Line para un control de alambre

`<Wire control>.MaxNbSegment = <New maximum number>`
---

**`<Wire control>: (Nombre del control)`**

Nombre del alambre control a utilizar.

**`<New maximum number>: (Integro)`**



- Número máximo de segmentos de Line que componen el alambre control,

- 1 corresponde al dibujo de una sola Line entre dos controles. 




Observación: Si el Alambre control puede ser dibujado con menos segmentos, el dibujo se simplifica automáticamente.

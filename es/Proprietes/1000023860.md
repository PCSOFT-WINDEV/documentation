
## WorksheetName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **WorksheetName** Property se utiliza para obtener y cambiar el nombre de una hoja de trabajo en un XLS Document. Esta propiedad se puede utilizar en:

- A control Hoja de cálculo. Para más detalles, consulte la ayuda del programa [Nombre de la hoja de trabajo](../Proprietes/1000021646.md) Property.

- Una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

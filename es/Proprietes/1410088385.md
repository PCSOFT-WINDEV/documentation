
## spMatchesDescription (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spMatchesDescription** determina si los parámetros (función de derivación de claves, algoritmo hash, número de iteraciones y longitud de salida) del campo de tipo "Contraseña" en el archivo de datos coinciden con los parámetros descritos en el análisis o mediante programación. 

Estos parámetros pueden ser diferentes si se cambiaron en la descripción del archivo. La modificación automática de datos no recalcula las contraseñas (para ello, se necesita la contraseña del usuario). Esto puede dar lugar a un desajuste entre los parámetros reales de la contraseña y los parámetros deseados en la descripción del archivo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the parameters of PasswordItem in the Authentication data file
IF Authentication.PasswordItem.spMatchesDescription = False THEN
	Info("Please change your password.")
END
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Matching parameters> = <Data file>.<Item>.spMatchesDescription`
---

**`<Matching parameters>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si los parámetros de la contraseña segura descritos coinciden con los del archivo de datos, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).

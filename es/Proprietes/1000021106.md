


## StoreTheValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StoreTheValue** Property se utiliza para:

- Averiguar si un control mantiene su valor entre dos inicios de aplicación a través de la persistencia. 

- Modificar una control para que conserve (o no) su valor entre dos inicios de aplicación a través de la persistencia. 




Esta propiedad corresponde a la opción "Store the value" que se encuentra en la pestaña "Details" de la ventana de description de los siguientes controles: 

- Casilla de verificación/Botón de radio,

- Combo box/List box,

- Editar control,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Pestaña estática/Pestaña dinámica.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
c is Control
c <- ControlCreate("EDT_Date_1", typDate, 0, 0, 100, 20)
c.Caption = "Today's date"
c.StoreTheValue = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si un control mantiene su valor entre dos inicios de aplicación

`<Result> = <Control used>.StoreTheValue`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control guarda su valor entre dos inicios de aplicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser: 

- una casilla de verificación o control Botón de opción,

- una Combo Box o control List Box,

- a control Campo de entrada,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Pestaña estática o una control Pestaña dinámica.





<a name="SYNTAX2"></a>

### Modificar un control para que mantenga (o no) su valor entre dos inicios de aplicación

`<Control used>.StoreTheValue = <New management mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser: 

- una casilla de verificación o control Botón de opción,

- una Combo Box o control List Box,

- a control Campo de entrada,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Pestaña estática o una control Pestaña dinámica.




**`<New management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control debe guardar su valor entre dos inicios de la aplicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si esta propiedad se establece en <u><u><u><u>True</u></u></u></u> después de la fase de inicialización, el control no tomará el valor anterior.

- Este Property es useful para configurar los controles creados por [ControlCreate](../WDLang1/1000020876.md) y [ControlClone](../WDLang1/3025017.md). 

- Esta propiedad corresponde a la opción "Almacenar el valor" que se encuentra en el menú contextual de AAF (Características de la aplicación automatic). Cuando un control es creado por [ControlCreate](../WDLang1/1000020876.md), el menú de AAFs está disponible para el usuario final: la gestión de la persistencia puede añadirse a partir de entonces (excepto si la opción está desactivada por [AAFDisable](../WDLang1/1000022018.md).


## Title (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Título** Property se utiliza para get y cambiar el título de un elemento. Esta propiedad se puede utilizar en:

- ventanas, páginas o controles de gráficos. Para obtener más información, consulte [Título Property](../Proprietes/2510090.md).

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [gglCalendar](../WDLang5/1000017418.md). 

- una variable de tipo [gglDocument](../WDLang5/1000018001.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [gglOrganization](../WDLang5/1000017756.md).

- una variable de tipo [Notification](../WDLang3/1000019441.md).

- una variable de tipo [notificationFormat](../WDLang3/1000021301.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [rssChannel](../WDLang5/1000017769.md)

- una variable de tipo [rssEntry](../WDLang5/1000017792.md)

- una variable de tipo [rssImage](../WDLang5/1000017799.md)

- una variable de tipo [rssTextInput](../WDLang5/1000017802.md)

- una variable de tipo [OrgElement](../WDLang1/1000019713.md).

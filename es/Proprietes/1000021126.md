
## StopWord (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StopWord** Property se utiliza para definir las palabras de parada en un índice de texto completo definido mediante programación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyFullTextIndex is FullTextIndex Description
...
MyFullTextIndex.StopWord = "the at a an"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Full-text index>.StopWord = <Stop Words>`
---

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido por el tipo [FullTextIndex Description](../WDLang4/1000017461.md).

**`<Stop Words>: (Cadena de caracteres)`**

Lista de nombres considerados como palabras vacías. Las palabras pueden ser separadas por: 

- un espacio,

- una coma

- to semicolon,

- a period,

- un colon,

- un retorno de carro (CR),

- una tabulación (TAB).




## MapMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MapMode** Property determina y cambia el modo de visualización del mapa en el control Mapa. Este mapa se puede visualizar en modo mapa de carreteras o en modo satélite.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ModeCarte.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the display mode of the Map control
MAP_Position.MapMode = mapModeSatellite
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de visualización de una control Mapa

`<Result> = <Map control>.MapMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de visualización actual del mapa en la control Mapa. Este parámetro puede corresponder a una de las siguientes constantes:


|   |   |
| --- | --- |
| *mapModePlan* | El mapa que aparece en el control Mapa está en modo de mapa de carreteras. |
| *mapModeSatellite* | El mapa que aparece en el control Mapa está en modo satélite. |



**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del modo de visualización de una control Mapa

`<Map control>.MapMode = <New display mode>`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<New display mode>: (Constante de tipo Integer)`**

Nuevo modo de visualización del mapa en la control Mapa. Este parámetro puede corresponder a una de las siguientes constantes:


|   |   |
| --- | --- |
| *mapModePlan* | El mapa que aparece en el control Mapa está en modo de mapa de carreteras. |
| *mapModeSatellite* | El mapa que aparece en el control Mapa está en modo satélite. |

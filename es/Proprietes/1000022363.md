
## TitleColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TítuloColor** Property se utiliza para: 

- obtener el texto Color de un título de columna en una Tabla o control Tabla TreeView.

- set el texto Color del título de una o todas las columnas de una Tabla o control Tabla TreeView.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF bVIPCustomer THEN
	COL_Note.TitleColor = LightRed
ELSE
	COL_Note.TitleColor = DefaultColor
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el texto Color de un título de columna en una Tabla o control Tabla TreeView

`<Result> = <Element used>.TitleColor`
---

**`<Result>: (Constante de tipo Integer)`**

Color actualmente utilizada para el texto del título de la columna especificada. Para obtener más información, consulte [color predefinido de WLanguage](../WDLang5/3010002.md). 

**`<Element used>: (Nombre del control)`**



- Nombre de la columna a utilizar. 

- Nombre del control Tabla. En este caso, sólo se tiene en cuenta la primera columna de la control Tabla. 

- Nombre de la tabla TreeView control. En este caso, sólo se tiene en cuenta la primera columna de la control Tabla TreeView.





<a name="SYNTAX2"></a>

### Fijar el texto Color del título de una o todas las columnas de una Tabla o control Tabla TreeView

`<Element used>.TitleColor = <New color>`
---

**`<Element used>: (Nombre del control)`**



- Nombre de la columna a utilizar.

- Nombre de la control Tabla (a set el Color del título de todas las columnas). 

- Nombre de la control Tabla TreeView (a set el Color del título de todas las columnas). 




**`<New color>: (Entero o constante de tipo Integer)`**

Nuevo Color para el texto del título. Este parámetro puede corresponder a una de las siguientes opciones: 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

- la constante *ColorDefault*. En este caso, el título de la columna utilizará la Color común a todas las columnas (es decir, la Color definida en el estilo de la control Tabla).

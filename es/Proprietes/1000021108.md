
## ButtonType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ButtonType** permite obtener y cambiar el tipo de un control Botón: Cancelación, Interrupción, Validación, ....

Observación: Esta propiedad corresponde a la opción "Tipo" disponible en la pestaña "UI" de la ventana de description de una control Botón. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
B1 is Control
B1 <- ControlCreate("B1", typButton, 100, 50, 32, 32)
B1.ButtonType = btnInterruption
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el tipo de un botón

`<Result> = <Button control>.ButtonType`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de botón: 


|   |   |
| --- | --- |
| *btnCancel* | Botón Cancel. |
| *btnHelp* | Botón de ayuda. |
| *btnInterruption* | Botón de interrupción. |
| *btnNormal* | Botón reposo. |
| *btnValidate* | Botón Validar. |



**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar. 


<a name="SYNTAX2"></a>

### Cambiar el tipo de botón

`<Button control>.ButtonType = <New type>`
---

**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar. 

**`<New type>: (Constante de tipo Integer)`**

Tipo de botón: 


|   |   |
| --- | --- |
| *btnCancel* | Botón Cancel. |
| *btnHelp* | Botón de ayuda. |
| *btnInterruption* | Botón de interrupción. |
| *btnNormal* | Botón reposo. |
| *btnValidate* | Botón Validar. |

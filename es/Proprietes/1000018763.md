
## Option (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Option** permite obtener y cambiar ciertas opciones. Esta propiedad se puede utilizar en:

- a control Código de barras. Para obtener más información, consulte [Opción](../Proprietes/2510154.md).

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).

- una variable de tipo [emailPOP3Session](../WDLang3/1000018759.md).

- una variable de tipo [emailSMTPSession](../WDLang3/1000018765.md).

- una variable de tipo [hFileParameter](../WDLang4/1000018900.md). 

- una variable de tipo [API description](../WDLang1/1000019149.md).

- una variable de tipo [Descripción de ParallelTask](../WDLang1/1000021586.md).

- una variable de tipo [FullTextIndex Description](../WDLang4/1000017461.md).

- una variable de tipo [mqttSession](../WDLang3/1000023023.md). 

- una variable de tipo [mongoOperationAdd](../WDLang4/1410089163.md).

- una variable de tipo [mongoOperationUpdateOne](../WDLang4/1410089164.md).

- una variable de tipo [mongoOperationUpdateAll](../WDLang4/1410089165.md).

- una variable de tipo [mongoOperationReplaceOne](../WDLang4/1410089166.md).

- una variable de tipo [mongoOperationDeleteAll](../WDLang4/1410089162.md).

- una variable de tipo [mongoOperationDeleteOne](../WDLang4/1410089161.md).

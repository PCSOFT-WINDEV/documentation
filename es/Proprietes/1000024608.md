
## Visible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Visible** Property se utiliza para set la visibilidad de un elemento. Esta propiedad se puede utilizar en: 

- controles de una ventana, página o reporte. Para obtener más información, consulte [Visible](../Proprietes/2510138.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

- una variable de tipo [kbCard](../WDLang1/1410089167.md).




## Guest (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **Guest** Property se utiliza para get y modificar una lista de invitados. Esta propiedad se puede utilizar en:

- una variable de tipo [gglEvent](../WDLang5/1000017440.md)

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

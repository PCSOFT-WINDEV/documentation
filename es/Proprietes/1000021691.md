
## Attribute (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Attribute** permite gestionar atributos. Esta propiedad está disponible en:

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

- una variable de tipo [xmlReader](../WDLang5/1000023392.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [hbTableDescription](../WDLang4/1000021688.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [hbColumnDescription](../WDLang4/1000021694.md).

- una variable de tipo [xpathResult](../WDLang5/1000021967.md).

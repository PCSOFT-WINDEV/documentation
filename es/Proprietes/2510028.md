


## ToolTipTitle (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ToolTipTitle** obtiene y establece el texto que se muestra en el tooltip asociado al título de una columna del control Tabla. El tooltip solo aparece cuando se pasa el cursor por encima del título de la columna del control Tabla.

**Recordatorio**: Para obtener y cambiar el texto que se muestra en el tooltip de una fila del control Tabla, utilice la propiedad [ToolTip](../Proprietes/2510005.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Tooltip for the Student column
COL_Student.ToolTipTitle = "Student name"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el texto del tooltip asociado a un título de columna

`<Tooltip text> = <Column>.ToolTipTitle`
---

**`<Tooltip text>: (Cadena de caracteres)`**

Texto actual del tooltip asociado al título de la columna especificada.

**`<Column>: (Nombre del control)`**

Nombre de la columna del control Tabla a utilizar.  


<a name="SYNTAX2"></a>

### Cambiar el texto del tooltip asociado a un título de columna

`<Column>.ToolTipTitle = <New tooltip text>`
---

**`<Column>: (Nombre del control)`**

Nombre de la columna del control Tabla a utilizar. 

**`<New tooltip text>: (Cadena de caracteres)`**

Nuevo texto del tooltip asociado al título de la columna. No se mostrará ningún tooltip si el texto es una cadena vacía.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **ToolTipTitle** solo se aplica a las columnas de controles Tabla en una ventana o página.




## CodeHTML (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CodeHTML** Property se utiliza para leer y cambiar el código HTML associated con un componente web control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(CMP_MyWebComponent.CodeHTML)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubriendo el código HTML associated con el componente web control

`<Result> = <Web Component control>.CodeHTML`
---

**`<Result>: (Cadena de caracteres)`**

Código HTML associated con el componente web control. Este código HTML se escribirá en la página. 

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el código HTML associated con el componente web control

`<Web Component control>.CodeHTML = <HTML Code>`
---

**`<Web Component control>: (Nombre del control)`**

Nombre del componente web control que se va a utilizar. 

**`<HTML Code>: (Cadena de caracteres)`**

Nuevo código HTML associated con el control.

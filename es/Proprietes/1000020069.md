


## StoreSortAAF (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StoreSortAAF** Property se utiliza para determinar y especificar si la ordenación definida por el usuario en una Tabla o control Tabla TreeView se mantiene cuando una aplicación se cierra y se abre de nuevo. 

Este orden se define a través de los botones de clasificación de las columnas de tabla.

Si se mantiene el orden de clasificación, se volverá a aplicar automáticamente la próxima vez que se abra la ventana que contiene la tabla,. 

**Observación**: Esta propiedad equivale a la opción "Recordar las clasificaciones de los usuarios", en la pestaña "Detalles" de la ventana control Tabla description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify CBOX_StoreSort
IF CBOX_StoreSort = True THEN
	TABLE_PRICE.StoreSortAAF = True
ELSE
	TABLE_PRICE.StoreSortAAF = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si las opciones de clasificación están almacenadas

`<Result> = <Table used>.StoreSortAAF`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardan las opciones de orden definidas por el usuario en el control Tabla. 

- <u><u><u><u>False</u></u></u></u> en caso contrario (valor predeterminado). 




**`<Table used>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular. 


<a name="SYNTAX2"></a>

### Modificar la gestión de las opciones de clasificación

`<Table used>.StoreSortAAF = <Storage>`
---

**`<Table used>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular. 

**`<Storage>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para recordar las opciones de clasificación definidas por el usuario en el control Tabla entre 2 lanzamientos de la aplicación. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La información de clasificación se guarda con el resto de la información persistente (para más detalles, consulte la ayuda sobre [SaveParameter](../WDLang1/3025046.md)).




## BackgroundImagePrinted (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **BackgroundImagePrinted** se utiliza para: 

- Determinar si la Image de fondo de un reporte (formulario u otro) debe imprimirse.

- Imprimir (o no) el fondo Image de un reporte. 


**Observación**: Esta Property se guarda por página: esto le permite definir diferentes ajustes para cada página impresa. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyReport.BackgroundImage = PDFExtractPage("form.pdf", 2)
MyReport.BackgroundImagePrinted = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si la Image de fondo de un reporte (formulario u otro) debe imprimirse

`<Result> = <Report used>.BackgroundImagePrinted`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si debe imprimirse la imagen de fondo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte utilizada.  


<a name="SYNTAX2"></a>

### Imprimir (o no) el fondo Image de un reporte

`<Report used>.BackgroundImagePrinted = <Image print>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte utilizada.

**`<Image print>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si debe imprimirse la imagen de fondo,

- <u><u><u><u>False</u></u></u></u> si no debe imprimirse la imagen de fondo. 


Los nuevos ajustes se tendrán en cuenta a partir del siguiente salto de página. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **BackgroundImagePrinted** Property se guarda por página: esto le permite definir diferentes ajustes para cada página impresa. 

- Si los ajustes se modifican con el **BackgroundImagePrinted** Property, los nuevos ajustes se tendrán en cuenta a partir del siguiente salto de página.

- Este Property está disponible para todo tipo de informes. 

- Esta Property afecta a la imagen de fondo de los informes de formularios o a la especificada por la [BackgroundImage](../Proprietes/2510056.md) Property.

- Se tiene en cuenta la gestión de los márgenes especificados en la pestaña "Fondo" de la descripción de reporte ("Ignorar márgenes físicos de la impresora").

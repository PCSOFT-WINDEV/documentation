


## Encryption (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Encryption** permite gestionar el tipo de cifrado utilizado entre un equipo Cliente y un servidor HFSQL. Es posible:

- Averigua el tipo de encriptación usado para una conexión de HFSQL Client/Server.

- Definir el tipo de cifrado utilizado para la conexión HFSQL Client/Server. En este caso, si la conexión está actualmente abierta, se cerrará y se abrirá de nuevo con los nuevos parámetros la primera vez que se acceda a un archivo de datos que utilice esta conexión.




**Observación**: La conexión puede corresponder a:

- una conexión definida en el editor de análisis,

- una conexión definida con las funciones [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

- una conexión definida con una variable de tipo [Conexión](../WDLang4/1514073.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection
// Describe the connection
MyConnection.User = "USER"
MyConnection.Password = "PASSWORD"
MyConnection.Server = "MYSERVER" 
MyConnection.Database = "Database"
MyConnection.CryptMethod = hCryptRC5_16
MyConnection.Provider = hAccessHFClientServer
MyConnection.Access = hOReadWrite
MyConnection.ExtendedInfo = "Extended information"

HOpenConnection(MyConnection)
```

<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el tipo de encriptación de una conexión de HFSQL Client/Server

`<Type of encryption> = <Connection used>.Encryption`
---

**`<Type of encryption>: (Constante)`**

Devuelve el tipo de cifrado utilizado para la conexión.:  


|   |   |
| --- | --- |
| *hEncriptaciónRC5_16* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado optimizado RC5 de 16 rondas (128 bits).. |
| *hEncryptionAES128* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 128 bits con el modo de funcionamiento OFB.. |
| *hEncryptionAES192* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 192 bits con el modo de funcionamiento OFB.  |
| *hEncryptionAES256* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 256 bits con el modo de funcionamiento OFB..  |
| *hEncryptionNO* | No se realiza ninguna encriptación. |
| *hEncryptionStandard* | La conexión entre el servidor y los ordenadores cliente está cifrada mediante un algoritmo de cifrado optimizado de 128 bits. |



**`<Connection used>: (Nombre de la conexión)`**

Nombre de la conexión utilizada. Este nombre se define:

- en el editor de análisis, 

- con [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- con el tipo [Conexión](../WDLang4/1514073.md).  





<a name="SYNTAX2"></a>

### Definir el tipo de cifrado para una conexión

`<Connection used>.Encryption = <Type of encryption>`
---

**`<Connection used>: (Nombre de la conexión)`**

Nombre de la conexión utilizada. Este nombre se define:

- en el editor de análisis, 

- con [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- con el tipo [Conexión](../WDLang4/1514073.md).




**`<Type of encryption>: (Constante)`**

Define el tipo de cifrado utilizado para la conexión:  


|   |   |
| --- | --- |
| *hEncriptaciónRC5_16* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado RC5 optimizado de 16 rondas (128 bits). |
| *hEncryptionAES128* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 128 bits con el modo de funcionamiento OFB.. |
| *hEncryptionAES192* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 192 bits con el modo de funcionamiento OFB.  |
| *hEncryptionAES256* | La conexión entre el servidor y los ordenadores cliente se cifra mediante un algoritmo de cifrado AES de 256 bits con el modo de funcionamiento OFB..  |
| *hEncryptionNO* | No se realiza encriptación. |
| *hEncryptionStandard* | La conexión entre el servidor y los ordenadores cliente se encripta mediante un algoritmo de cifrado optimizado de 128 bits. |

### ExtendedInfo Property y conexión
<a name="extendedinfo_property_conexion_ELTPARAGRAPHE000182"></a>

Al cambiar la información de cifrado de una conexión HFSQL Client/Server con la propiedad [ExtendedInfo](../Proprietes/2512041.md), se actualiza automáticamente la información devuelta por la propiedad **Encryption** (y viceversa).

Por lo tanto, la siguiente Line de código:


```wl
MyConnection.CryptMethod = hCryptRC5_16
```
modificará la opción ENCRYPTION en la cadena de información extendida.

Del mismo modo, la siguiente Line de código:


```wl
MyConnection.ExtendedInfo = "CRYPT=RC5_16"
```
modificará el **Encryption** Property .

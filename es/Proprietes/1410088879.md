
## DisplayFormat (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **DisplayFormat** obtiene y establece el formato de visualización deseado para un texto determinado: texto sin formato, RTF o Markdown.

Este texto puede ser: 

- el título de un control Estático. 

- el título de un control Campo de entrada.

- el título de una columna de un control Tabla.  

- el texto de una opción de menú.  



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
STC_Info.DisplayFormat = dfMarkdown
STC_Info = "**Bold** and *italic* text"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el formato de visualización del texto de un control

`<Result> = <Control used>.DisplayFormat`
---

**`<Result>: (Constante de tipo Integer)`**

Formato esperado por el control Estático: 



|   |   |
| --- | --- |
| *dfMarkdown* | Texto Markdown. |
| *dfPlainText* | Texto sin formato específico. |
| *dfRTF* | Texto en formato RTF. |



**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede corresponder a: 

- el nombre de un control Estático,

- el nombre de un control Campo de entrada,

- el nombre de una columna de un control Tabla,

- el nombre de una opción de menú.  





<a name="SYNTAX2"></a>

### Establecer el formato de visualización del texto de un control

`<Control used>.DisplayFormat = <New format>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede corresponder a: 

- el nombre de un control Estático,

- el nombre de un control Campo de entrada,

- el nombre de una columna de un control Tabla,

- el nombre de una opción de menú.     




**`<New format>: (Constante de tipo Integer)`**

Nuevo formato esperado por el control Estático: 



|   |   |
| --- | --- |
| *dfMarkdown* | Texto Markdown. |
| *dfPlainText* | Texto sin formato específico. |
| *dfRTF* | Texto en formato RTF. |

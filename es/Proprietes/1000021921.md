


## ParallaxRateHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ParallaxRateHeight** Property se usa para determinar y especificar qué tan rápido se reduce un control cuando se mueve la barra de desplazamiento en la ventana, ventana interna o Page. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
IMG_Product.ParallaxRateY = 500
IMG_Product.ParallaxRateHeight = 500
```



<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la velocidad a la que se reduce el control cuando Scrolling la barra de desplazamiento

`<Result> = <Control used>.ParallaxRateHeight`
---

**`<Result>: (Integro)`**

Velocidad a la que se reduce la control (incluida entre 0 y 1000). 

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. 


<a name="SYNTAX2"></a>

### Modificando la velocidad a la que se reduce el control cuando Scrolling la barra de desplazamiento

`<Control used>.ParallaxRateHeight = <New speed>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. 

**`<New speed>: (Integro)`**

Velocidad a la que se reduce la control (incluida entre 0 y 1000). El valor predeterminado de este parámetro es 0. 

- Si este parámetro se set a 0, el control se mueve como de costumbre cuando Scrolling la ventana o el Page. 

- Si este parámetro es set a 1000, el control se aplasta hacia abajo como el desplazamiento realizado en la ventana o Page va a lo largo de.




## NbItem (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NbItem** se utiliza para obtener el número total de elementos (incluyendo las claves compuestas) en un Record de:

- un archivo de datos descrito en el editor de análisis,

- un archivo de datos describió a través de la programación,

- una vista HFSQL,

- a query.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the number of items in a record of the Customer data file
ResNbItem = Customer.NbItem
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Number of items> = <Element used>.NbItem`
---

**`<Number of items>: (Integro)`**

Número de elementos en una Record del elemento utilizado.

**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento para el que se quiere averiguar el número de elementos para un Record

- nombre de un archivo de datos descrito en el editor de análisis o a través de la programación,

- nombre de una vista HFSQL,

- nombre de una consulta.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La propiedad **NbItem** se utiliza para obtener el número total de elementos de un Record.

- La propiedad [NbKeyItem](../Proprietes/2512056.md) se utiliza para obtener el número total de elementos clave en una Record.

- La propiedad [NbMemoItem](../Proprietes/2512057.md) se utiliza para obtener el número total de partidas de notas en una Record.

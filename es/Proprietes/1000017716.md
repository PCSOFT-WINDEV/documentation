


## MinimumLength (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MinimumLength** Property se utiliza para configurar la longitud mínima de las palabras a indexar (índices de texto completo). Es posible:

- get la longitud mínima de las palabras a indexar en un índice de texto completo (definido en la editor de análisis o mediante programación).

- Definir la longitud mínima de las palabras a indexar en un índice de texto completo (descrito mediante programación).





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la longitud mínima de las palabras a indexar

`<Length> = <Data file>.<Full-text index>.MinimumLength`
---

**`<Length>: (Integro)`**



- Longitud mínima de las palabras a indexar,

- 0 para indexar todas las palabras sin longitud mínima.




**`<Data file>: (Nombre del control)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [FullTextIndex Description](../WDLang4/1000017461.md).  


<a name="SYNTAX2"></a>

### Definición de la longitud mínima de las palabras a indexar

`<Full-text index>.MinimumLength = <New Length>`
---

**`<Full-text index>: (Cadena de caracteres)`**

Nombre del índice de texto completo utilizado. Este nombre fue definido por el tipo [FullTextIndex Description](../WDLang4/1000017461.md).

**`<New Length>: (Integro)`**



- Longitud mínima de las palabras a indexar,

- 0 si todas las palabras deben estar indexadas.




## AutomaticErase (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AutomaticErase** Property se utiliza para:

- averigua si una control Campo de entrada está en modo "Borrar automatic".

- modificar la característica "borrar automatic" de una control Campo de entrada.




**Observación**: Si un control Campo de entrada está en modo "Borrado automático", el valor del control se selecciona automáticamente al entrar en él. Esta característica puede ser configurada en la pestaña "Detalles" de la ventana de description de control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Temporarily disables the automatic erase 
// to avoid selecting everything during the call to SetFocus

// Stores the current automatic erase
AutoErase is boolean 
AutoErase = EDT_Edit1.AutomaticErase
EDT_Edit1.AutomaticErase = False

// Gives focus back to the edit control
SetFocus(EDT_Edit1) 

// Restores the automatic erase
EDT_Edit1.AutomaticErase = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el modo "Borrado de automatic"

`<Current mode> = <Control used>.AutomaticErase`
---

**`<Current mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el modo "Borrado automático" está activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del elemento utilizado: 

- Control Campo de entrada, 

- Texto columna tabla.  





<a name="SYNTAX2"></a>

### Modificando el modo "Borrado de automatic"

`<Control to use>.AutomaticErase = <New mode>`
---

**`<Control to use>: (Nombre del control)`**

Nombre del elemento utilizado: 

- Control Campo de entrada, 

- Texto columna tabla.




**`<New mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar el modo "Borrado automático",

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## ConnectionTimeout (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ConnectionTimeout** Property se usa para administrar el tiempo de espera de la conexión. Esta propiedad está disponible en:

- una variable de tipo [sshSession](../WDLang3/1000021157.md).

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- a Variable de tipo [mqttSession](../WDLang3/1000023023.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a Variable de tipo [apiRequest](../WDLang3/1410088935.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [memcachedConnection](../WDLang4/1000023339.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [redisConnection](../WDLang4/1000023527.md).

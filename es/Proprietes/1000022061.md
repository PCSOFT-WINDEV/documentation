
## Property (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Property** Property se utiliza para get y cambiar las propiedades de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [Definition](../WDLang1/1000019540.md).

- una variable de tipo [btleCharacteristic](../WDLang3/1000022060.md).




## VisibleOutsideWindow (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **VisibleOutsideWindow** Property se utiliza para: 

- determinar si un botón es Visible aunque su posición (X,Y) esté fuera del Border de la ventana (total o parcialmente). 

- modificar la visibilidad de un "fuera de la ventana" control Botón.




**Atención**: Para utilizar el **VisibleOutsideWindow** Property, la ventana actual debe tener un Border recortado . De lo contrario, la Property no tiene ningún efecto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Place the OK Button control below the window
BTN_OK.Y = MyWindow.Height + 10
// Make the Button control visible 
BTN_OK.VisibleOutsideWindow = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si una control Botón es Visible fuera de la ventana

`<Result> = <Button control>.VisibleOutsideWindow`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Botón es Visible fuera de la ventana, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar.


<a name="SYNTAX2"></a>

### Modificar la visibilidad de una control Botón fuera de la ventana

`<Button control>.VisibleOutsideWindow = <Visibility>`
---

**`<Button control>: (Nombre del control)`**

Nombre de control Botón a utilizar

**`<Visibility>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Botón debe ser Visible fuera de la ventana, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La control Botón no será visible si se sitúa por encima o a la izquierda de la ventana (Y&lt;0 o X&lt;0 Property). Para hacerse visible o Invisible, la control Botón debe posicionarse debajo o a la derecha de la ventana. 

- El control Botón no será visible si el [Visible](../Proprietes/2510138.md) Property es set a <u><u><u><u>False</u></u></u></u>.


## Punto

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Point** se utiliza para gestionar un punto. Esta palabra clave puede corresponder a: 

- a property del [gglPath](../WDLang5/1000017433.md) variable. 

- una variable de tipo [Punto](../Motscles/1000018908.md). 

- un property de un variable de tipo [Polygon](../WDLang1/1000025624.md).

- un property de un variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- un property de un variable de tipo [MapPolyline](../WDLang3/1000025489.md).

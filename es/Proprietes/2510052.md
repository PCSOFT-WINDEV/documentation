


## Timeout (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Timeout** Property se utiliza para get y cambiar el tiempo después del cual se considera que la conexión ha "fallado". Este tiempo de espera se tiene en cuenta cuando se utiliza la función [StreamConnect](../WDLang3/3039008.md) y la [Address](../Proprietes/2510017.md) Property.

El tiempo de espera de Default es de set a 2 minutos por Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase timeout
MyStream.Timeout = 150000
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el tiempo de espera antes de que la conexión falle

`<Result> = <Stream name>.Timeout`
---

**`<Result>: (Integro)`**

Tiempo tras el cual se considera que la conexión ha "fallado". Este tiempo de espera se expresa en milisegundos. El tiempo de espera por defecto es de set a 2 minutos (o 120 000 milisegundos).

**`<Stream name>: (Cadena de caracteres)`**

Nombre del arroyo o nombre del control Conferencia associated con el arroyo.  


<a name="SYNTAX2"></a>

### Modificar el tiempo de espera antes de que la conexión falle

`<Stream name>.Timeout = <New timeout>`
---

**`<Stream name>: (Cadena de caracteres)`**

Nombre del arroyo o nombre del control Conferencia associated con el arroyo.

**`<New timeout>: (Entero o Duración)`**

Tiempo tras el cual se considera que la conexión ha "fallado". Este tiempo de espera se expresa en milisegundos. El tiempo de espera por defecto es de set a 2 minutos (o 120 000 milisegundos). Este parámetro puede corresponder a: 

- un número entero correspondiente al número de miles de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, "1cs" o "10 ms").

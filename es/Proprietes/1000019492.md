


## NullAllowed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NullAllowed** Property se utiliza para gestionar el valor NULL en los elementos de los archivos de datos HFSQL.

Este Property está acostumbrado:

- Averigua el modo de gestión del valor NULL de un artículo de un archivo de datos descrito en el editor de análisis.

- Definir el modo de gestión del valor NULL cuando se describe un elemento de un archivo de datos mediante programación.




Si el archivo de datos y el elemento admiten valores NULL, la propiedad [Null](../Proprietes/2512064.md) puede utilizarse para asignar NULL al elemento para el Record actual.

El **NullAllowed** Property es useful para evitar el uso del valor NULL para algunos elementos en un archivo de datos definidos a través de la programación (un archivo de datos que soporta el valor NULL).  






<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el artículo permite el uso del valor NULL

`Result> = <Data file>.<Item>.NullAllowed`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el artículo permite el uso del valor NULL, 

- <u><u><u><u>False</u></u></u></u> si el artículo no permite el uso del valor NULL.




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).


<a name="SYNTAX2"></a>

### Definir el modo de gestión del valor NULL para el artículo

`Item>.NullAllowed = <Management mode`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para administrar el valor NULL,

- <u><u><u><u>False</u></u></u></u> para no tener en cuenta el valor NULL.







## Observaciones


### ¿Cómo saber si un archivo de datos apoya la gestión del valor NULL?
<a name="¿como_saber_archivo_datos_apoya_gestion_del_valor_null_ELTPARAGRAPHE000094"></a>

Para determinar si un archivo de datos permite el uso del valor NULL, utilice el programa [NullSupported](../Proprietes/2512061.md) Property.

Atención:

- No es posible asignar un valor a la propiedad **NullAllowed** en un archivo de datos o elemento que no admite el valor NULL: se produce un error fatal.

- La lectura de la propiedad **NullAllowed** en un archivo de datos o elemento que no admite el valor NULL siempre devuelve <u><u><u><u>False</u></u></u></u>.



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000117"></a>

No se puede utilizar el valor NULL:

- en los artículos de array.

- en los identificadores de automatic.

- en las teclas compuestas.

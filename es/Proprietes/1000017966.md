


## Language (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Language** permite obtener y cambiar el idioma utilizado. Esta propiedad se puede utilizar en:

- elementos del archivo de datos a set el lenguaje utilizado para las comparaciones y ordenaciones. Para obtener más información, consulte [Idioma Property en los artículos](../Proprietes/1000017053.md).

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md).

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una variable de tipo [SpeechSynthesisVoice](../WDLang3/1000025986.md).

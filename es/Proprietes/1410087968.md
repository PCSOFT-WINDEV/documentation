
## ConfidenceLevel (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ConfidenceLevel** Property obtiene el nivel de confianza de un análisis de datos. Esta propiedad se puede utilizar en:

- una variable de tipo [ocrTextBlock](../WDLang6/1000025639.md).

- una variable de tipo [aiImageResult](../WDLang6/1410087965.md).

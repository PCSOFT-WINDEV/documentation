
## StoreFilterAAF (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StoreFilterAAF** Property se utiliza para determinar y especificar si los filtros set por el usuario en una Tabla o TreeView tabla control se guardan cuando una aplicación se cierra y se abre de nuevo.

Este filtro es set a través de los botones de filtrado en las columnas tabla (clic derecho en la lupa o embudo Icon). 

Si se guarda el filtro, la próxima vez que se abra la ventana que contiene la tabla, aparecerá un breve mensaje para indicar el filtro que se ha implementado, entonces el filtro se volverá a aplicar automáticamente. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MemoriseFiltreFAA.gif)


**Observación**: Esta propiedad equivale a la opción "Recordar filtros de usuario", en la pestaña "Detalles" de la ventana control Tabla description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify CBOX_StoreFilter
IF CBOX_StoreFilter = True THEN
	TABLE_PRICE.StoreFilterAAF = True
ELSE
	TABLE_PRICE.StoreFilterAAF = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si las opciones de filtro se almacenan

`<Result> = <Table control>.StoreFilterAAF`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardan las opciones de filtro definidas por el usuario en el control Tabla. 

- <u><u><u><u>False</u></u></u></u> en caso contrario (valor predeterminado). 




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular. 


<a name="SYNTAX2"></a>

### Modificación de la gestión de las opciones de filtro

`<Table control>.StoreFilterAAF = <Storage>`
---

**`<Table control>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular. 

**`<Storage>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardan las opciones de filtro definidas por el usuario en el control Tabla entre 2 inicios de la aplicación. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La información del filtro se guarda con el resto de la información persistente (para más detalles, consulte la ayuda sobre [SaveParameter](../WDLang1/3025046.md)).

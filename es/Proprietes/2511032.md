


## LabelNumberToStart (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LabelNumberToStart** Property se utiliza para:

- Averigüe el número de la etiqueta en la hoja de etiquetas desde la que se realizará la impresión.

- Modificar el número de la etiqueta desde la que se realizará la impresión en la hoja de etiquetas.




Esta Property le permite imprimir en una hoja de etiqueta en la que ya se ha realizado una impresión previa. La impresión comenzará a partir de la etiqueta cuyo número se especifica.

**Observación**: Esta Property debe utilizarse en el código de apertura de la reporte.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Opening code of the report
// Prints from the label #6 of the page
RPT_CustLabel.LabelNumberToStart = 6
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de la etiqueta a partir de la cual comenzará la impresión

`<Result> = <Report used>.LabelNumberToStart`
---

**`<Result>: (Integro)`**

Número de la etiqueta desde la que se iniciará la impresión. Este número se da de acuerdo con la hoja de etiquetas utilizada (las etiquetas están numeradas de izquierda a derecha y de arriba hacia abajo).. 
La impresión comienza en la etiqueta #1 de Default.

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.  


<a name="SYNTAX2"></a>

### Modificando el número de la etiqueta a partir de la cual se iniciará la impresión

`<Report used>.LabelNumberToStart = <New number>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la "Etiqueta" reporte a utilizar.

**`<New number>: (Integro)`**

Número de la etiqueta desde la que se iniciará la impresión. Este número debe indicarse de acuerdo con la hoja de etiquetas utilizada (las etiquetas están numeradas de izquierda a derecha y de arriba abajo)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000064"></a>

El **LabelNumberToStart** Property debe utilizarse en el proceso de apertura del reporte. Por ejemplo, puede pasar el número de la primera etiqueta a imprimir como parámetro a la reporte al iniciar la impresión.

Esta Property sólo se aplica a las impresiones realizadas en una hoja de etiquetas.

Esta Property no puede ser utilizada si la reporte contiene un encabezado y un pie de página.
<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000077"></a>

El **LabelNumberToStart** Property sólo se aplica a los informes de "Etiqueta".

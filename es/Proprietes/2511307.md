
## Minute (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Minuto** Property se utiliza para get y cambiar los minutos de un elemento. Esta propiedad se puede utilizar en:

- a Fecha, hora y fecha Variable o elemento. Para obtener más información, consulte [Minuta Property sobre los tipos de fecha, hora y dateTime](../Proprietes/2512050.md).  

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md). 

- una variable de tipo [hScheduling](../WDLang4/1000018853.md).

- una variable de tipo [Repetition](../WDLang1/1000019262.md).

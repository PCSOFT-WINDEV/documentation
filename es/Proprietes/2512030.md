


## Extension (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Extension** permite:

- get la extensión de un archivo de datos definida en el editor de análisis o mediante programación.

- definir la extensión de un archivo de datos (cuando el archivo se describe sólo a través de la programación).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the name and full path of the Customer data file
NameAndFullPath is string
NameAndFullPath = Customer.Directory + "\" + ...
	Customer.PhysicalName + Customer.Extension
// NameAndFullPath corresponds to "C:\MyFiles\Customer.Fic"
```
<a name="Example2"></a>

```wl
// Define the extension of the AFile file
AFile is File Description
AFile.Name = "CUSTOMER"
AFile.Extension = ".AAA"
// The created file will be named "CUSTOMER.AAA"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la extensión de un archivo de datos

`<File extension> = <Data file>.Extension`
---

**`<File extension>: (Cadena de caracteres)`**



- Extensión del archivo de datos utilizada (precedida por un punto). 

- Vaciar la cadena ("") si el archivo de datos no tiene extensión.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definiendo la extensión de un archivo de datos

`<Data file>.Extension = <New extension>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<New extension>: (Cadena de caracteres)`**

Extensión del archivo de datos utilizada (precedida por un punto).

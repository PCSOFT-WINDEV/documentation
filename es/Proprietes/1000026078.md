
## Undocked (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Undocked** Property se utiliza para determinar si un panel de una Ficha o control Cinta de opciones está desacoplado. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If pane 2 is undocked from the Ribbon control, make the following pane active
IF RIBBON_Menu[2].Undocked THEN
	RIBBON_Menu = 3
ELSE
	RIBBON_Menu = 2
END

// Determine if a pane is undocked from a Tab control
FOR i = 1 TO TAB_Option.Count
      IF TAB_Option[i].Undocked THEN RESULT True
END
RESULT False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si un panel está desacoplado

`<Result> = <Control used>.Undocked`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el panel está desacoplado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control used>: (Nombre del control)`**

Nombre del panel de la Ficha o control Cinta de opciones a utilizar. Se debe utilizar la siguiente sintaxis: 

```txt
<Control name>[Element index]
```
Por ejemplo: 
```txt
RIBBON_Menu[2]
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un panel está desacoplado si se encuentra en una ventana libre diferente a la ventana donde está la control.

- Para desacoplarse de una control Pestaña, el panel debe ser dinámico. 

- Para desacoplar un panel de un control Cinta de opciones, marque la opción "Pestañas desacoplables " en la ventana de control description.

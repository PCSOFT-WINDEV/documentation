
## Height (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Altura** Property se utiliza para get y cambiar la altura de un elemento. Esta propiedad se puede utilizar en:

- un elemento encontrado en una ventana, Page o reporte. Para obtener más información, consulte [Altura](../Proprietes/2510050.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [rssImage](../WDLang5/1000017799.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).  

- una variable de tipo [xlsRow](../WDLang5/1000017325.md).

- una variable de tipo [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpImage](../WDLang3/1000020905.md).

- una variable de tipo [Franja de orilla](../Motscles/15140100.md).

- una variable de tipo [Rectangle](../Motscles/1514079.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementImage](../WDLang1/1000022696.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docRow](../WDLang1/1000022925.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementTextArea](../WDLang1/1000023420.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WDPic](../WDLang1/1000023503.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfPage](../WDLang6/1000024904.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfTextElement](../WDLang6/1000024556.md).

- una variable de tipo [MapImage](../WDLang3/1000025523.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagSelection](../WDLang1/1410088172.md).

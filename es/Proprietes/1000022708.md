
## MaxSize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxSize** Property se utiliza para obtener y set un tamaño máximo. Esta propiedad está disponible en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoCollectionOption](../WDLang4/1000022413.md).

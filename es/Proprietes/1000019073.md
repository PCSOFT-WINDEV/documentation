
## Number (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Number** permite obtener el número de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [API description](../WDLang1/1000019149.md).

- una variable de tipo [gglPhoneNumber](../WDLang5/1000017573.md).  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docSection](../WDLang1/1000022685.md).

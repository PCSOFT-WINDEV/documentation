


## NbRecRead (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**NbRecRead** se utiliza para conocer el número de registros leídos: seleccionado tanto por la fuente de la reporte como por el código del filtro (si existe).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If more than 100 records have been read
IF RPT_CustomerReport.NbRecRead > 100 THEN
	// The print of "RPT_CustomerReport" is stopped
	iEndReport()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Report used>.NbRecRead`
---

**`<Result>: (Integro)`**

Número de registros leídos.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Número de registros
<a name="numero_registros_ELTPARAGRAPHE000047"></a>

- El **NbRecRead** Property devuelve el número de registros seleccionados por la fuente del reporte (consulta, vista, zona de memoria o control Tabla).

- El [NbPrintedRec](../Proprietes/2511058.md) Property devuelve el número de registros seleccionados tanto por la fuente del reporte COMO por el código de filtro (si lo hay).

- Las propiedades **NbRecRead** y [NbPrintedRec](../Proprietes/2511058.md) devuelven el mismo valor para un reporte basado en un archivo: el número de registros en el archivo Y seleccionados por el código del filtro (si lo hay).

- El [TotalNbRec](../Proprietes/2511047.md) Property se utiliza para determinar y modificar el número de registros a imprimir. El número de registros a imprimir se incluirá entre 1 y [NbPrintedRec](../Proprietes/2511058.md).



<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000073"></a>

El **NbRecRead** Property no se puede utilizar en los informes sin una fuente de datos.

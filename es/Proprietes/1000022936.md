
## SelectionIfReadOnly (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SelectionIfReadOnly** Property se utiliza para determinar y especificar si el contenido de un control de edición puede ser seleccionado cuando está en modo de sólo lectura.

**Observación**: Esta propiedad corresponde a la opción "Selección en sólo lectura" de la pestaña "Detalles" de la ventana Editar control description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_Date.SelectionIfReadOnly = False
EDT_Date.State = Inactive
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si se puede seleccionar el contenido de una control de edición cuando está en modo de sólo lectura

`<Result> = <Edit control>.SelectionIfReadOnly`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se puede seleccionar el contenido de un control de edición cuando está en modo de sólo lectura,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Edit control>: (Nombre del control)`**

Nombre de la Edición control a utilizar. 


<a name="SYNTAX2"></a>

### Indica si se puede seleccionar el contenido de una control de edición cuando está en modo de sólo lectura

`<Edit control>.SelectionIfReadOnly = <New value>`
---

**`<Edit control>: (Nombre del control)`**

Nombre de control Campo de entrada a utilizar.

**`<New value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se puede seleccionar el contenido de un control de edición cuando está en modo de sólo lectura,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Detalles
<a name="detalles_ELTPARAGRAPHE000057"></a>

- Por defecto, la propiedad **SelectionIfReadOnly** se establece como <u><u><u><u>True</u></u></u></u>. 

- Si la propiedad **SelectionIfReadOnly** se establece en <u><u><u><u>False</u></u></u></u> y si la Edición control es de sólo lectura, hacer clic en el control no tendrá ningún efecto.

- Si el control está en gris, se ignora el **SelectionIfReadOnly** Property. 

- Si el control está activado, se ignora el **SelectionIfReadOnly** Property.

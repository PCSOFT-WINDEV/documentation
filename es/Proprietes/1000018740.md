
## LeadID (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LeadID** Property se utiliza para gestionar los lead IDs. Esta propiedad se puede utilizar en:

- una variable de tipo [sfLeadConvertResult](../WDLang5/1000018739.md). 

- una variable de tipo [sfLeadConvert](../WDLang5/1000018742.md).

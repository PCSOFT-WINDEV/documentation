
## StoredValue (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **StoredValue** permite obtener y cambiar el valor almacenado de un elemento. Esta propiedad se puede utilizar en:

- las filas de los controles List Box y Tabla, así como en los controles Campo de entrada con tokens. Para obtener más información, consulte [StoredValue](../Proprietes/2510127.md).

- una variable de tipo [Token](../WDLang1/1000023942.md). 

- una variable de tipo [AssistedInput](../WDLang1/1410088983.md).

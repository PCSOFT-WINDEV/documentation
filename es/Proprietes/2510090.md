
## Title (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Title** permite:

- get el título de una columna control Tabla, de una ventana control Gráfico o de una ventana Page. 

- Cambiar el título de una columna control Tabla , una control Gráfico, una ventana o una Page.




En los informes, esta Property se utiliza para get y cambiar el título de un control Gráfico. 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Get the title of "WIN_InputForm" window
ResTitle = WIN_InputForm.Title
```


<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Add an image to the title of a column in a Table control
TABLE_MyTable.COL_MyColumn.Title = gImage("MyImage.BMP") + "My title"
```


<a name="Example3"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Display the date on the right of window title
MyWindow.Title = EOT + gAlignment("D", DateToSTring(Today(), ...
		"DDDD DD MMMM YYYY"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el título de un elemento

`<Result> = <Element name>.Title`
---

**`<Result>: (Cadena de caracteres)`**



- Título de la ventana especificada o Page.

- Título de la columna especificada.

- Título de la columna a la que pertenece la celda tabla especificada.

- Título de la control Gráfico especificada.




**`<Element name>: (Cadena de caracteres)`**

Nombre del elemento (columna, gráfico, celda tabla, ventana o Page) a utilizar. 
Una celda tabla no tiene nombre. Para recuperar el título de la columna a la que pertenece una celda tabla, utilice la siguiente sintaxis: 

```txt
<Table name>[Row number][Column number]
```

Por ejemplo: 

```txt
TABLE_Table1[1][2]
```



<a name="SYNTAX2"></a>

### Cambiar el título de un elemento

`<Element name>.Title = <New title>`
---

**`<Element name>: (Cadena de caracteres)`**

Nombre del elemento (columna, gráfico, celda tabla, ventana o Page) a utilizar. 
Una celda tabla no tiene nombre. Para recuperar el título de la columna a la que pertenece una celda tabla, utilice la siguiente sintaxis: 

```txt
<Table name>[Row number][Column number]
```

Por ejemplo: 

```txt
TABLE_Table1[1][2]
```


**`<New title>: (Cualquier tipo)`**



- Nuevo título para la ventana especificada o Page. Las funciones gráficas se pueden utilizar en este parámetro. 
	**Caso especial en Windows Vista (y posterior)**: Las ventanas que usan el estilo "Aero sistema" no pueden tener un título gráfico: se ignora la parte gráfica. Si "Aero" no está habilitado, el título del gráfico se mostrará como de costumbre.

- Nuevo título para la columna.

- Nuevo título para la control Gráfico.

<a name="NOTE0_1"></a>

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Caso de las ventanas con clip
<a name="caso_las_ventanas_con_clip_ELTPARAGRAPHE000147"></a>

El **Title** Property también puede utilizarse para cambiar el título de las ventanas recortadas. Sin embargo, este título no se muestra en la ventana.  

Si se utilizó una plantilla recortada, el título de la ventana se muestra en un control Estático. El contenido de este control es ignorado por el **Title** Property.
<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000160"></a>

Las propiedades **Title**, [Valor](../Proprietes/2510130.md) y [Pie de foto](../Proprietes/2510053.md) son equivalentes.

Para get el título de la ventana, utilice una de las siguientes sintaxis:


```txt
ResTitle = <Window name>.Title

ResTitle = <Window name>.Value

ResTitle = <Window name>.Caption

ResTitle = <Window name>
```


Para get el título de la columna, utilice una de las siguientes sintaxis:


```txt
ResTitle = <Column name>.Title

ResTitle = <Column name>.Caption
```


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para get el título de la columna que contiene una celda en un control Tabla, utilice cualquiera de las siguientes sintaxis:


```txt
ResTitle = <Table control name>[<Row number>][<Column number>].Title

ResTitle = <Table control name>[<Row number>][<Column number>].Caption
```

<a name="NOTE0_3"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000190"></a>

La propiedad **Title** solo se aplica a:

- ventanas y páginas,

- columnas en Table y TreeView controles Tabla,

- celdas en Table y TreeView controles Tabla,

- Controles de la carta,




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **Title** Property se aplica sólo a los controles de la Carta. 

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En Java, el **Title** Property sólo se aplica a los siguientes elementos:

- Ventana.

- Columna de un control Tabla.

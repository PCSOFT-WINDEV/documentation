
## Nulo (palabra clave)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave "Nulo" puede corresponder a: 

- la palabra clave Null. Para obtener más información, consulte [Nulo (Palabra reservada)](../Motscles/1511015.md). 

- la [Null](../Proprietes/2512064.md) property. Esta Property se utiliza para gestionar el valor NULL en los elementos de los archivos de datos HFSQL o en los elementos de los archivos de datos a los que accede Conectores Nativos.

- la constante nula. Esta constante corresponde a un entero inicializado a 0. Para obtener más información, consulte [Constantes de Notas adicionales](../WDLang1/3010001.md).




## DayBreakHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DayBreakHeight** Property se utiliza para get y cambiar la altura de las pausas entre los días en un control Planificador donde los días se organizan en filas, y los recursos en columnas. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_WithBreaks = True THEN
	SCH_MySchedule.DayBreakHeight = 25
ELSE
	SCH_MySchedule.DayBreakHeight = 0
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Conseguir la altura de los descansos entre días

`<Result> = <Scheduler control>.DayBreakHeight`
---

**`<Result>: (Integro)`**



- Altura de las rupturas en píxeles. 

- 0 si no hay ruptura. 




**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada.  


<a name="SYNTAX2"></a>

### Modificación de la altura de las pausas entre días

`<Scheduler control>.DayBreakHeight = <New height>`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada.

**`<New height>: (Integro)`**



- Nueva altura para las pausas entre los días expresadas en píxeles. 

- 0 para desactivar las pausas entre los días.

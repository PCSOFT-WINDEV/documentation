


## CtDescribedCaption (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtDescribedCaption** Property se utiliza para obtener y set el título de la conexión a la fuente de datos OLE DB. Es posible:

- Averiguar el título de la conexión a la fuente de datos OLE DB. Esta fuente de datos se describió en el editor de análisis o cuando se describió el el archivo de datos mediante programación.

- Definir el título de la conexión OLE DB (cuando el archivo de datos se describe mediante programación).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the caption of the connection to the OLE DB data source
ResCaptionOLEDBSource = Customer.CtDescribedCaption
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el título de la conexión

`<Caption> = <Data file>.CtDescribedCaption`
---

**`<Caption>: (Cadena de caracteres)`**

Leyenda de la conexión.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definiendo el título de la conexión

`<Data file>.CtDescribedCaption = <Caption>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<Caption>: (Cadena de caracteres)`**

Leyenda de la conexión.


## ToolTip (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ToolTip** Property se utiliza para get y cambiar el texto mostrado en el tooltip associated con una ventana o página control.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observaciones**:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **ToolTip** Property se utiliza para insertar imágenes y texto coloreado en el tooltip a través del [Funciones gXXX](../WDLang1/3029028.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **ToolTip** Property permite insertar contenido RTF en los tooltips.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los usuarios finales pueden modificar los datos de las herramientas en sus aplicaciones. Para obtener más información, consulte [Información sobre herramientas de control (AAF)](../Editeurs/1000021821.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El texto de la información sobre herramientas puede mostrarse en una información sobre herramientas estándar o personalizada. Para obtener más información, consulte [Información sobre herramientas personalizada](../Editeurs/9500233.md).




**Recordatorio**: Para escribir el mensaje que aparece en el tooltip de una control en el editor: 

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual). 

2. En la pestaña "Ayuda", en el área "Información sobre la herramienta", escriba el texto de la información sobre la herramienta. 
	Observaciones: 

	- En una aplicación multilingüe, el texto de la información sobre la herramienta se puede escribir en varios idiomas. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la opción "RTF" está seleccionada, el texto puede formatearse a través de las opciones de la barra de herramientas RTF. El formato se respetará en tiempo de ejecución.

## Sintaxis

### Averiguar el texto de la información de la herramienta

`<Tooltip text> = <Control used>.ToolTip`
---

**`<Tooltip text>: (Cadena de caracteres)`**

Texto actual del tooltip associated con el control.

**Observación**: Si el texto de la información sobre la herramienta está en RTF, el texto devuelto contendrá las etiquetas RTF. 

**`<Control used>: (Nombre del control)`**

Nombre de la control para la que se solicita el texto del tooltip.  


<a name="SYNTAX2"></a>

### Modificar el texto del tooltip

`<Control used>.ToolTip = <New tooltip text>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control para la que hay que modificar el texto de la información sobre herramientas.

**`<New tooltip text>: (Cadena de caracteres)`**

Nuevo texto para el tooltip associated con el control. Si este texto corresponde a una cadena vacía (""), no se mostrará ningún tooltip para esta control.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Este texto puede contener imágenes y puede ser formateado mediante el programa [Funciones gXXX de WLanguage](../WDLang1/3029028.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Este texto puede corresponder a un texto RTF (archivo RTF o texto con etiquetas RTF).

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Tooltip de una control Pestaña
<a name="tooltip_una_control_pestana_ELTPARAGRAPHE000161"></a>

Para los controles de pestañas, se puede definir un tooltip para cada uno de los paneles de pestañas.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Tooltip associated con un botón de radio y/o control Casilla de verificación
<a name="tooltip_associated_con_boton_radio_control_casilla_verificacion_ELTPARAGRAPHE000170"></a>

Se puede definir un tooltip para la control y para cada opción. Por ejemplo:


```wl
RADIO_MyRadioButton[1].ToolTip = "Option #1"
```

<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Tooltip associated con fila en un Treeview control
<a name="tooltip_associated_con_fila_treeview_control_ELTPARAGRAPHE000181"></a>

Para los controles TreeView, se puede definir un tooltip para cada uno de los elementos treeview ([TreeAdd](../WDLang1/3018012.md), [TreeInsert](../WDLang1/3018023.md) y [TreeModify](../WDLang1/3018013.md)).
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Tooltip associated con una control Tabla y con una columna en una control Tabla
<a name="tooltip_associated_con_una_control_tabla_con_una_columna_una_control_tabla_ELTPARAGRAPHE000199"></a>

El editor de ventanas permite definir un tooltip para una control Tabla y para cada una de sus columnas. La punta de la herramienta associated con la control Tabla se muestra si no se definió una punta de la herramienta para la columna que se encuentra en el cursor del ratón.

<a name="NOTE0_6"></a>

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000235"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **ToolTip** Property se aplica sólo a las ventanas y controles de una ventana (incluidas las columnas y celdas de un tabla). Esta Property no puede ser usada con los menús. 



![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **ToolTip** Property se aplica sólo a los siguientes controles:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Control Tabla.

- Columna de la tabla control (tipo texto, cuadro combinado o casilla de verificación).

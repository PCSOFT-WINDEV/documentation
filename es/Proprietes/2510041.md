


## LineHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LineHeight** Property se utiliza para get y cambiar la altura de las filas:

- en una control List Box,

- en un control ListView (sólo en modo de visualización estándar),

- en una control Tabla,

- en una control Combo Box,

- en una control Looper (altura fila).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Agenda.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Hoja de cálculo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control de Treeview.




**Observaciones**: 

- Control List Box o Combo Box: La altura de las filas se define en el editor de la ventana, en la pestaña "Estilo" del control description.

- Tabla, vista de árbol, looper o control Hoja de cálculo: La altura de las filas se define en el editor de la ventana, en la pestaña "Detalles" del control description.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the height of rows found in "TABLE_CustomerTable"
TABLE_CustomerTable.LineHeight = TABLE_CustomerTable.LineHeight + 100
// Equivalent to: TABLE_CustomerTable.LineHeight +=100
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura de las filas en un control

`<Current height> = <Control used>.LineHeight`
---

**`<Current height>: (Integro)`**

Altura (en píxeles) de las filas en la control especificada.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificar la altura de las filas en un control

`<Control used>.LineHeight = <New height>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<New height>: (Integro)`**

Nueva altura (en píxeles) de las filas en la control especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Controles del cuadro de lista de navegación, tabla y Combo Box
<a name="controles_del_cuadro_lista_navegacion_tabla_combo_box_ELTPARAGRAPHE000103"></a>

Cuando se modifica la altura de las filas en una Tabla de navegación, Cuadro de Lista o control Combo Box, la nueva altura se aplica sólo a las filas de Visible.
<a name="NOTE0_2"></a>


### Control List Box
<a name="control_list_box_ELTPARAGRAPHE000110"></a>

Si es necesario, se puede mostrar una barra de desplazamiento vertical en una control List Box cuando se modifica la altura de las filas.
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Control ListView
<a name="control_listview_ELTPARAGRAPHE000119"></a>

El **LineHeight** Property sólo se tiene en cuenta si el control ListView se muestra en modo estándar.
<a name="NOTE0_4"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000128"></a>

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **LineHeight** Property sólo se puede utilizar en los controles de Looper.![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **LineHeight** Property se aplica únicamente a: 

- Controles de List Box,

- Control Combo Box, 

- Controles de mesa.


## TimeUTC (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TimeUTC** Property se utiliza para obtener y set un Tiempo Universal Coordinado (UTC ). 

Esta propiedad se puede utilizar en: 

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [hScheduling](../WDLang4/1000018853.md).

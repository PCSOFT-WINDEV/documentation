


## CalendarButton (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Calendar Button** se utiliza para determinar y especificar si un control de edición de fecha tiene un botón para mostrar una ventana emergente Calendar.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BoutonCalendrier.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a Calendar button if necessary
IF EDT_Date.CalendarButton = False THEN 
	EDT_Date.CalendarButton = True
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si se muestra un botón de Calendar

`<Result> = <Edit control>.CalendarButton`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se muestra un botón Calendario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Edit control>: (Nombre del control)`**

Nombre de la fecha control Campo de entrada a utilizar.  


<a name="SYNTAX2"></a>

### Mostrando un botón de Calendar o no

`<Edit control>.CalendarButton = <Display>`
---

**`<Edit control>: (Nombre del control)`**

Nombre de la Fecha control Campo de entrada a utilizar.

**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si debe mostrarse un botón Calendario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

### Modo de funcionamiento de la control Calendario
<a name="modo_funcionamiento_control_calendario_ELTPARAGRAPHE000075"></a>

En tiempo de ejecución, los controles de edición de fecha proponen la opción "Calendar" por Default. Esta opción permite mostrar una Calendar utilizada para seleccionar una fecha.

Si la propiedad **CalendarButton** se utiliza para mostrar un botón Calendar en la fecha Editar control, la opción "Calendar" ya no está disponible en el menú contextual de la fecha control.

**Observación**: Para desactivar la visualización de la Calendar de un proyecto o de una control (mediante el botón o el menú contextual), utilice [AAFDisable](../WDLang1/1000022018.md).


## FullName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FullName** Property se utiliza para get el nombre completo de un elemento. Esta propiedad se puede utilizar en:

- una control, una ventana o una opción de menú. Para obtener más información, consulte [FullName](../Proprietes/2514005.md).

- una variable de tipo [sfUserInfo](../WDLang5/1000018423.md).

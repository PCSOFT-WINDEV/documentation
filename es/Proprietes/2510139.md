


## XImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **XImage** Property se utiliza para get y cambiar la coordenada X inicial de la sección Image que se mostrará en un control Imagen o en el fondo de un control Gráfico.

**Recordatorio**: El origen de la imagen corresponde al punto de la imagen que se mostrará en la esquina superior izquierda de la control. Por defecto, **XImage** = 0.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the origin of the image displayed in the "IMG_Car" control
SourceXCoord = IMG_Car.XImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la coordenada X de la sección Image

`<Result> = <Control used>.XImage`
---

**`<Result>: (Integro)`**

Coordenada X de la sección Image que se mostrará en el control (en píxeles).

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificación de la coordenada X de la sección Image

`<Control used>.XImage = <New X-coordinate>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<New X-coordinate>: (Integro)`**

Nueva coordenada X de la sección Image que se mostrará en la control (en píxeles). Este valor debe ser positivo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Definiendo la sección de Image
<a name="definiendo_seccion_image_ELTPARAGRAPHE000050"></a>

Un control Imagen se utiliza para mostrar una sección del Image que se encuentra en el archivo associated Image. Las siguientes propiedades se utilizan para definir la sección de Image que se mostrará en el control Imagen:

- **XImage**: utilizado para definir la coordenada X inicial de la sección de Image.

- [YImage](../Proprietes/2510046.md): utilizado para definir la coordenada Y inicial de la sección de Image.

- [ImageHeight](../Proprietes/2510044.md): utilizado para definir la altura de la sección de Image.

- [ImageWidth](../Proprietes/2510066.md): utilizado para definir el ancho de la sección de la Image.

<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000077"></a>

El **XImage** Property se aplica sólo a los controles Image y Chart en una ventana. Este Property no se aplica a los controles de imagen associated con un Image animado.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **XImage** Property se aplica sólo a los controles Image.

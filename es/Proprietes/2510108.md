


## RichEdit (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RichEdit** Property se utiliza para gestionar el tipo RTF (Rich Text Format): 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) de una ventana control.
	**Recordatorio**: El tipo RTF es una opción que se aplica a los controles de edición y a los controles estáticos. Esta opción se encuentra en la pestaña "Detalles" del control description.

- de una control Estático encontrada en un reporte.
	**Recordatorio**: El tipo RTF es una opción que se aplica a los controles estáticos. Esta opción se encuentra en la pestaña "General" del control description.




Es posible: 

- averiguar si la control está en RTF. 

- modificar el tipo de RTF de un control.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the "STC_Name" control is in RTF, it becomes visible
IF STC_Name.RichEdit = True THEN
	STC_Name.Visible = True
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la control está en RTF

`<Result> = <Control used>.RichEdit`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado es de tipo RTF,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Edita control o control Estático en una ventana. 

- control Estático en un reporte.





<a name="SYNTAX2"></a>

### Modificar el tipo de RTF de un control

`<Control used>.RichEdit = <Type>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Edita control o control Estático en una ventana. 

- control Estático en un reporte.




**`<Type>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado debe ser de tipo RTF,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Manejar el valor de un RTF control
<a name="manejar_valor_rtf_control_ELTPARAGRAPHE000102"></a>

- Si la asignación (o la operación de lectura) se realiza directamente o por [Valor](../Proprietes/2510130.md): el valor de la control RTF está en RTF (este valor contiene etiquetas RTF).

- Si la asignación (o la operación de lectura) es realizada por [RTFToText](../WDLang1/3025032.md): el valor de la control RTF no está en RTF (este valor no contiene ninguna etiqueta RTF).



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000116"></a>

En una ventana, el **RichEdit** Property se aplica sólo a los siguientes controles:

- controles de edición,

- Controles estáticos. 

- columna de texto tabla. 


En un reporte, el **RichEdit** Property se aplica sólo a los controles estáticos.

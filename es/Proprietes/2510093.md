


## AutoBrowse (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **AutoBrowse** se utiliza para determinar si la operación de navegación realizada en un Cuadro de Lista, Looper, Tabla o control Combo Box basada en un archivo de datos es automatic o programada.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is the browse performed in "TABLE_Customer" automatic or programmed?
ResBrowse = TABLE_Customer.AutoBrowse
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Control used>.AutoBrowse`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la navegación del elemento especificado es automática.

- <u><u><u><u>False</u></u></u></u> si la navegación del elemento especificado se realiza mediante programación.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: Caja de listas, Mesa, Looper o control Combo Box.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tipo de iteración
<a name="tipo_iteracion_ELTPARAGRAPHE000070"></a>

- ***automatic iteración***: el archivo de datos navegar y la visualización de los datos en el elemento son automatic. No se requiere programación específica.
	La clasificación se realiza en el elemento de búsqueda definido en la pestaña "Contenido" del elemento description. Si se especifica un filtro ("Filtro en el elemento de búsqueda" en la pestaña "Contenido" del elemento description), se utiliza este filtro. No se ejecuta el navegador Process para el elemento especificado.

- ***Iteración programada***: El desarrollador debe gestionar la búsqueda de archivos y la visualización de datos en el elemento.
	Se ejecutan los procesos "Leer el primer Record", "Leer el anterior Record", "Leer el siguiente Record" y "Leer el último Record".. Una iteración programada permite ordenar y filtrar los registros con precisión. Sin embargo, una iteración programada es más lenta y compleja que una iteración automatic.



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000082"></a>

El **AutoBrowse** Property se aplica únicamente a:

- los controles del List Box de navegación,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) controles Tabla archivo de datos,

- navegar por los controles de Combo Box,

- navegación Controles del looper,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) navegar por los controles de ListView.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La propiedad **AutoBrowse** sólo se aplica a controles Tabla.

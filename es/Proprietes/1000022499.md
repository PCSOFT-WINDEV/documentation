
## FontBold (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **FontBold** Property se utiliza para gestionar el estilo de negrita en un texto. Esta propiedad se puede utilizar en: 

- controles. Para obtener más información, consulte [FontBold](../Proprietes/2510103.md). 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md).




## Abbreviation (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Abbreviation** obtiene la abreviatura de un archivo de datos. La abreviatura se define en el editor de análisis al definir el archivo de datos. Esta propiedad es de solo lectura.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Abbreviation of the Order data file
ResAbbreviation = Order.Abbreviation
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Abbreviation of data file> = <Data file>.Abbreviation`
---

**`<Abbreviation of data file>: (Cadena de caracteres)`**

Dos letras que identifican la abreviatura del archivo de datos.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos a utilizar.

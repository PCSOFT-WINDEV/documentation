
## Identifier (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Identifier** permite obtener y cambiar el identificador de un elemento. Esta propiedad se puede utilizar en:

- flujos. Para obtener más información, consulte [Identificador Property en los flujos](../Proprietes/2510036.md).

- una variable de tipo [gglCalendar](../WDLang5/1000017418.md).

- una variable de tipo [gglContact](../WDLang5/1000017434.md).

- una variable de tipo [gglDocument](../WDLang5/1000018001.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [gglContactGroup](../WDLang5/1000017576.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

- una variable de tipo [emailAttach](../WDLang3/1000018752.md).

- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md). 

- una variable de tipo [hRSConfig](../WDLang4/1000020484.md).

- una variable de tipo [rssEntry](../WDLang5/1000017792.md). 

- una variable de tipo [inAppProduct](../WDLang3/1000020868.md).


## UpdateDate (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **UpdateDate** Property se utiliza para get la fecha de actualización de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [gglContactGroup](../WDLang5/1000017576.md).

- una variable de tipo [gglDocument](../WDLang5/1000018001.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).




## Propiedades HFSQL associated con las consultas y vistas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas propiedades se pueden utilizar en las consultas o vistas HFSQL.




|   |   |
| --- | --- |
| [DescribedName](../Proprietes/2512066.md) | El **DescribedName** Property se utiliza para manipular el nombre lógico de los archivos de datos HFSQL (ignorando los posibles cambios realizados por [HAlias](../WDLang4/3044176.md)). |
| [ExecutionCompleted](../Proprietes/2512117.md) | La propiedad **ExecutionCompleted** permite saber si se completó la ejecución de una consulta o vista (HFSQL Classic o Client/Server). |
| [FilterCondition](../Proprietes/2512096.md) | El **FilterCondition** Property permite conocer la condición de selección set por [HFilter](../WDLang4/3044100.md) en un archivo de datos, vista o consulta HFSQL. |
| [FilteredItem](../Proprietes/2512086.md) | El **FilteredItem** Property se utiliza para identificar el elemento que tiene un filtro implementado por [HFilter](../WDLang4/3044100.md). |
| [FilterWithBounds](../Proprietes/2512097.md) | La propiedad **FilterWithBounds** permite saber si se han especificado límites para el filtro establecido con la función [HFilter](../WDLang4/3044100.md) en un archivo de datos HFSQL, vista o consulta. |
| [MaxAutoID](../Proprietes/2512023.md) | La propiedad **MaxAutoID** se utiliza para obtener el identificador máximo de automatic de un archivo de datos replicado. |
| [MaxValue](../Proprietes/2510009.md) | El **MaxValue** Property se utiliza para:<br><br>- averiguar y modificar el límite superior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite superior definido para el filtro actual.<br><br>- averiguar el valor máximo de una serie de gráficos.<br><br>- averiguar y modificar el valor máximo para algunos tipos de controles.<br><br><br> |
| [MinAutoID](../Proprietes/2512039.md) | La propiedad **MinAutoID** se utiliza para obtener el identificador mínimo automatic de un archivo de datos replicado. |
| [MinValue](../Proprietes/2510008.md) | El **MinValue** Property se utiliza para:<br><br>- averiguar y modificar el límite inferior de un reporte basado en un archivo de datos. <br><br>- averiguar el límite inferior definido para el filtro actual.<br><br>- averiguar el valor mínimo de una serie de gráficos.<br><br>- averiguar y modificar el valor mínimo para algunos tipos de controles. <br><br><br> |
| [Name](../Proprietes/2510082.md) | La propiedad **Name** permite obtener:<br><br>- el nombre de la fuente asociada a una fuente Variable.<br><br>- el nombre de un bloque control, reporte o reporte.<br><br>- el nombre de una control, grupo de control s o ventana.<br><br>- el nombre de un control, grupo de control s o Page.<br><br><br> |
| [NbItem](../Proprietes/2512055.md) | La propiedad **NbItem** se utiliza para obtener el número total de elementos (incluyendo las claves compuestas) en un Record de:<br><br>- un archivo descrito en el editor de análisis,<br><br>- un archivo descrito mediante programación,<br><br>- una vista HFSQL,<br><br>- una consulta.<br><br><br> |
| [NbKeyItem](../Proprietes/2512056.md) | La propiedad **NbKeyItem** se utiliza para obtener el número total de elementos clave (incluyendo las claves compuestas) en un Record de:<br><br>- un archivo descrito en el editor de análisis,<br><br>- un archivo descrito mediante programación,<br><br>- una vista HFSQL,<br><br>- una consulta.<br><br><br> |
| [NbMemoItem](../Proprietes/2512057.md) | La propiedad **NbMemoItem** se utiliza para obtener el número total de elementos de memo en un Record de:<br><br>- un archivo descrito en el editor de análisis,<br><br>- un archivo descrito mediante programación,<br><br>- una vista HFSQL,<br><br>- una consulta.<br><br><br> |
| [RecordSize](../Proprietes/2512090.md) | La propiedad **Record Size** se utiliza para obtener el tamaño (en bytes) de un Record en:<br><br>- un archivo descrito en el editor de análisis,<br><br>- un archivo descrito mediante programación,<br><br>- una vista HFSQL,<br><br>- una consulta.<br><br><br> |
| [SQLCode](../Proprietes/1000017045.md) | La propiedad **SQLCode** permite obtener el código SQL de una consulta creada con el editor de consultas (archivo ".WDR"). |
| [Type](../Proprietes/2510131.md) | La propiedad **Type** permite obtener:<br><br>- el tipo de elemento (archivo de datos o campo) definido mediante programación o en el editor de análisis, vista o consulta.<br><br>- el tipo de objeto.<br><br>- el tipo de datos que se almacena en un Variant.<br><br><br> |

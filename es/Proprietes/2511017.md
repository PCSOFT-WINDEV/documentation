


## SelectionOnly (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SelectionOnly** Property se utiliza para:

- Averigua qué elementos de la Tabla o de la control Tabla TreeView se imprimirán: imprimir todos los registros o sólo los registros seleccionados.

- Cambiar el modo de impresión de las filas en la Tabla o control Tabla TreeView: imprimir todos los registros o sólo los registros seleccionados.




Este Property sólo puede utilizarse en el código de apertura de un "Reporte sobre tabla" o "Reporte sobre la vista de árbol tabla" reporte.

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta Property está disponible para la compatibilidad con versiones anteriores. Los informes sobre tabla no están disponibles.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Opening code of the report
// Print only the rows selected in the Table control
RPT_Report1.SelectionOnly = True
```
<a name="Example2"></a>

```wl
// -- Opening code of the report
// Print all rows of the Table control
RPT_Report1.SelectionOnly = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de impresión de las filas que se encuentran en una tabla o Treeview control Tabla

`<Result> = <Report used>.SelectionOnly`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el reporte imprimirá solo las filas seleccionadas del control Tabla o Tabla TreeView,

- <u><u><u><u>False</u></u></u></u> si el reporte imprimirá todas las filas del control.




**`<Report used>: (Nombre del reporte)`**

Nombre del "Reporte en tabla" o "Reporte en treeview tabla" reporte a usar.


<a name="SYNTAX2"></a>

### Cambiar el modo de impresión de las filas de una Tabla o control Tabla TreeView

`<Report used>.SelectionOnly = <Type of print>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre del "Reporte en tabla" o "Reporte en treeview tabla" reporte a usar.

**`<Type of print>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para imprimir solo las filas seleccionadas del control Tabla o Tabla TreeView,

- <u><u><u><u>False</u></u></u></u> para imprimir todas las filas del control.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **SelectionOnly** sólo se aplica a los informes "Reporte en tabla" y "Reporte en TreeView Table" y sólo puede utilizarse en el código de apertura del reporte.

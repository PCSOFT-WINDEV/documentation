


## MemZoneName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MemZoneName** Property se utiliza para:

- averiguar el nombre de la zona de memoria utilizada como fuente de datos para una reporte.

- modificar el nombre de la zona de memoria utilizada como fuente de datos para una reporte.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the name of the data source for "RPT_CustomerReport"
ResViewName = RPT_CustomerReport.MemZoneName
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre de la zona de memoria associated con un reporte

`<Result> = <Report used>.MemZoneName`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la zona de memoria utilizada como fuente de datos para la reporte especificada.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte associated con una zona de memoria a utilizar. Se produce un error de WLanguage si este parámetro no corresponde a una reporte associated con un área de memoria.  


<a name="SYNTAX2"></a>

### Modificar el nombre del área de memoria associated con un reporte

`<Report used>.MemZoneName = <Name of memory area>`
---

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar. Se produce un error de WLanguage si este parámetro no corresponde a una reporte associated con una zona de memoria.

**`<Name of memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria utilizada como fuente de datos para la reporte especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **MemZoneName** sólo puede utilizarse en un reporte associated con un área de memoria.


## Zoom (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Zoom** Property se utiliza para determinar y cambiar el zoom aplicado a un elemento. Esta propiedad se puede utilizar en: 

- una control o una ventana. Para obtener más información, consulte [Zoom](../Proprietes/1000017201.md). 

- una variable de tipo [pdfParameter](../WDLang6/1410087857.md).




## NbPrintedRec (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbPrintedRec** Property se utiliza para determinar el número de registros ya impresos.

**Observación**: El **NbPrintedRec** Property ignora los bloques repetidos hasta el pie de página.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If more than 5 records have been printed
IF RPT_CustomerReport.NbPrintedRec > 5 THEN
	// The print of "RPT_CustomerReport" is stopped
	iEndReport()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Report used>.NbPrintedRec`
---

**`<Result>: (Integro)`**

Número de registros ya impresos.

**`<Report used>: (Nombre del reporte)`**

Nombre de la reporte que se va a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Número de registros
<a name="numero_registros_ELTPARAGRAPHE000050"></a>

- El [NbRecRead](../Proprietes/2511041.md) Property devuelve el número de registros seleccionados por la fuente del reporte (consulta, vista, zona de memoria o control Tabla).

- El **NbPrintedRec** Property devuelve el número de registros seleccionados tanto por la fuente del reporte COMO por el código del filtro (si lo hay).
	Las propiedades [NbRecRead](../Proprietes/2511041.md) y **NbPrintedRec** devuelven el mismo valor para un reporte basado en un archivo: el número de registros en el archivo Y seleccionados por el código del filtro (si lo hay).

- El [TotalNbRec](../Proprietes/2511047.md) Property se utiliza para determinar y modificar el número de registros a imprimir. El número de registros a imprimir estará entre 1 y **NbPrintedRec**.



<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000076"></a>

El **NbPrintedRec** Property no se puede utilizar en los informes sin una fuente de datos.

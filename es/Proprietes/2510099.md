


## FontUnderlined (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**FontUnderlined** se utiliza para:

- Averigüe si un texto está subrayado o no.

- Subrayar (o no) un texto.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Underline the text displayed in "STC_StaticControl"
STC_StaticControl.FontUnderlined = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si el texto que aparece en una control está subrayado

`<Result> = <Control used>.FontUnderlined`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto está subrayado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando el atributo de subrayado para el texto que se muestra en un control

`<Control used>.FontUnderlined = <Underlined/Not underlined>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Underlined/Not underlined>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto debe estar subrayado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000119"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **FontUnderlined** Property sólo puede utilizarse en:

- un control Estático.

- un control Calculado.

- to URL control Enlace.

- un control Predefinido.


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **FontUnderlined** Property no se puede utilizar en:

- a Control de número,

- una Web control Cámara,

- a control Conferencia.Observación: Esta Property se aplica únicamente a los controles "ListView" en modo de visualización "List Box"..

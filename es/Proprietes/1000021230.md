


## NbAnimationSteps (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbAnimationSteps** Property se utiliza para get y especificar el número de dibujos en el Image que define la animación de un control Barra de progreso infinita personalizado . Esta Image fue especificada en el editor o a través de [Image](../Proprietes/2510034.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PROGBAR_Endless1.Image = "MyAnimation.png"
PROGBAR_Endless1.NbAnimationSteps = 10 // 10 images in "MyAnimation.png"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el número de dibujos que se encuentran en la animación

`<Result> = <Infinite Progress Bar control>.NbAnimationSteps`
---

**`<Result>: (Int)`**

Número de pasos definidos para animar la control Barra de progreso infinita. 

**`<Infinite Progress Bar control>: (Nombre del control)`**

Nombre de la control Barra de progreso infinita a utilizar. 


<a name="SYNTAX2"></a>

### Modificando el número de dibujos que componen la animación

`<Infinite Progress Bar control>.NbAnimationSteps = <New value>`
---

**`<Infinite Progress Bar control>: (Nombre del control)`**

Nombre de la control Barra de progreso infinita a utilizar. 

**`<New value>: (Int)`**

Número de pasos a realizar para animar la control Barra de progreso infinita. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Este Property no tiene ningún efecto si el control es nativo. El control debe ser un control Barra de progreso infinita personalizado. 

- La Image está cortada verticalmente en "bandas" para cada paso de la animación.

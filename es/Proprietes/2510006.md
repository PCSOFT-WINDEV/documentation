
## Animation (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Animation** permite animar una imagen. 

Es posible:

- Determinar si la imagen o el texto está animado.

- Iniciar o detener la animación de una imagen o un texto (GIFs animados, por ejemplo).




Esta propiedad se puede utilizar en los siguientes controles:

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Controles Imagen

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Controles Botón

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Controles Estático

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Controles Carrusel

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Controles Barra de progreso.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La propiedad **Animation** también anima el texto en un control Estático.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
// Initialize the Image control with Arrow.GIF (animated GIF)
IMG_Image1 = "C:\Temp\Arrow.gif"
// Start the animation
IMG_Image1.Animation = True
```


<a name="Example2"></a>


![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Initialize the Static control with the following text: "WINDEV is great"
STC_Caption1 = "WINDEV is great"
// Start the animation
STC_Static1.Animation = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si la imagen o el texto está animado

`<Status> = <Control used>.Animation`
---

**`<Status>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la imagen o el texto del control está animado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.  


<a name="SYNTAX2"></a>

### Iniciar o detener la animación

`<Control used>.Animation = <Animation mode>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.

**`<Animation mode>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> para comenzar la animación,

- <u><u><u><u>False</u></u></u></u> para detener la animación.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Animar una imagen
<a name="animar_una_imagen_ELTPARAGRAPHE000104"></a>

La imagen a animar en el **control Imagen** puede ser:

- una imagen animada (GIF animado, por ejemplo).

- una imagen animada por WINDEV. En este caso, el tipo de animación puede definirse en la ventana de descripción del control Imagen (pestaña "Detalles") o mediante programación con la función [AnimationSetMode](../WDLang1/3017001.md).




En un control Botón, solo se puede asociar una imagen animada al control (GIF animado, por ejemplo).
<a name="NOTE0_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Animar un texto
<a name="animar_texto_ELTPARAGRAPHE000121"></a>

Para animar el texto en un control Estático, debe elegir la animación:

- en la ventana de descripción del control Estático (pestaña "Detalles"),

- mediante programación con la función [AnimationSetMode](../WDLang1/3017001.md).



<a name="NOTE0_3"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Animar una barra de progreso
<a name="animar_una_barra_progreso_ELTPARAGRAPHE000136"></a>

En una barra de progreso, la animación inicia cuando cambia el valor del progreso: la posición no se fija inmediatamente en el nuevo valor, sino que aumenta (o disminuye) progresivamente para dar un efecto de animación.

Observación: este efecto está adaptado a las barras de progreso circulares.
<a name="NOTE0_4"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000145"></a>

De forma predeterminada, la animación se establece en <u><u><u><u>True</u></u></u></u>.

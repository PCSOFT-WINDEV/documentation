
## TitleHeader (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TitleHeader** Property se utiliza para: 

- get el título y el aspecto de la cabecera de una columna (Tabla o control Tabla TreeView). 

- get el número de cabeceras de columna (Tabla o control Tabla TreeView).

- get el título y el aspecto de la cabecera de una columna (Tabla o control Tabla TreeView).




Atención: Este Property no puede ser asignado o copiado directamente. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the title of a title-header on row 1
COL_DATE.TitleHeader[2].Caption = "New caption"

// Modifies the title of a title-header on row 2
COL_DATE.TitleHeader[1].Caption = "Sub-title"
// Modifies the font used for the title of a title-header on row 2
COL_DATE.TitleHeader[1].Font.Bold = True

// Modifies the color of sub-title on row 1
COL_DATE.TitleHeader[1].Color = LightRed
COL_DATE.TitleHeader[1].FontBold = True
FOR I = 1 TO COL_DATE.TitleHeader.Count 
	COL_DATE.TitleHeader[1].Color = LightRed
END
```

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperar el valor de una Property para el título de una columna

`<Result> = <Column>.TitleHeader[<Row number>].<Property>`
---

**`<Result>: (Tipo esperado)`**

Valor de Property.

**`<Column>: (Nombre del control)`**

Nombre de la columna tabla utilizada. 

**`<Row number>: (Integro)`**

Número de título fila a utilizar. La fila 1 corresponde al primer título de la columna fila a partir de la parte inferior.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Surentete_ligne.gif)


**`<Property>: (Propiedad WLanguage)`**

WLanguage Property que se usará para manipular el encabezamiento del título. Las propiedades disponibles son: 


|   |   |
| --- | --- |
| [BackgroundColor](../Proprietes/2510022.md) | Se utiliza para averiguar el fondo Color de la cabecera del título. |
| [Color](../Proprietes/2510071.md) | Se utiliza para averiguar el texto Color del encabezamiento del título. |
| [FontBold](../Proprietes/2510103.md) | Se usa para averiguar si el texto del encabezado es en negrita..  |
| [FontCondensed](../Proprietes/2510096.md) | Se utiliza para averiguar si el texto del encabezado del título está condensado.. |
| [FontExtended](../Proprietes/2510104.md) | Se utiliza para averiguar si el texto del encabezado del título se ha extendido..  |
| [FontItalic](../Proprietes/2510102.md) | Se usa para averiguar si el texto del encabezado es cursiva..  |
| [FontLarge](../Proprietes/2510101.md) | Se utiliza para averiguar si el texto de la cabecera es grande. |
| [FontName](../Proprietes/2510100.md) | Se usa para encontrar el nombre de la fuente usada para el texto de la cabecera del título. |
| [FontSize](../Proprietes/2510072.md) | Se usa para averiguar el tamaño del texto del encabezado del título. |
| [FontStrikeOut](../Proprietes/2510091.md) | Se utiliza para averiguar si el texto del título está tachado.. |
| [FontUnderlined](../Proprietes/2510099.md) | Permite saber si el texto de la cabecera está subrayado. |
| [Pie de foto](../Proprietes/2510053.md) | Se usa para encontrar el texto del encabezado del título. |
| [tipo de letra](../Proprietes/2510095.md) | Se usa para averiguar la fuente de la cabecera del título. <br><br>Observación: Las propiedades de las fuentes se pueden utilizar directamente. |

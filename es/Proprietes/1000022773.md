
## PageLayout (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PageLayout** Property establece las características de diseño de un Document. Esta propiedad se puede utilizar en: 

- el editor de Diagram control. Para obtener más información, consulte [Diseño](../Proprietes/1000023138.md). 

- una variable de tipo [docFragment](../WDLang1/1000022482.md). 

- una variable de tipo [docParagraph](../WDLang1/1000022483.md). 

- una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md). 

- una variable de tipo [diagLayout](../WDLang1/1410088610.md).




## InDays (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InDays** Property se utiliza para convertir una Duración Variable en días. Es posible:

- Averigüe el número de días correspondiente a una duración.

- Modificar el número de días correspondiente a una duración.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartDate is DateTime
EndDate is DateTime
MyDuration is Duration

StartDate = "200812251535"
EndDate = "200812271812"
MyDuration = EndDate - StartDate
Trace(MyDuration.InDays)   // 2.109027777778
Trace(MyDuration.InHours)  // 50.61666666667
Trace(MyDuration.InMinutes) // 3037
```


<a name="Example2"></a>



```wl
MyDuration is Duration
MyDuration.InDays = 1.5
Trace(MyDuration.InHours)  // 36
Trace(MyDuration.InMinutes) // 2160
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de días correspondientes a una duración Variable

`<Result> = <Duration used>.InDays`
---

**`<Result>: (Numeric)`**

Número de días correspondiente a la duración especificada.

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.  


<a name="SYNTAX2"></a>

### Modificar el número de días correspondientes a una duración Variable

`<Duration used>.InDays = <New duration>`
---

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.

**`<New duration>: (Real)`**

Número de días correspondientes a la duración especificada.


## Site (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Site** permite obtener el sitio SaaS asociado a:

- una variable de tipo [saasService](../WDLang3/1000019096.md), 

- una variable de tipo [saasPricing](../WDLang3/1000019091.md).

- una variable de tipo [saasVisit](../WDLang3/1000024560.md).

- una variable de tipo [WBAdminConnection](../WDLang2/1410089344.md).

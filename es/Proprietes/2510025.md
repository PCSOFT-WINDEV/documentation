


## Cursor (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>


### Definition
<a name="definition_ELTPARAGRAPHE000010"></a>

El cursor del ratón permite determinar fácilmente dónde se realizará una acción. En el caso de un control en modo de edición, el cursor se convierte en una Line vertical parpadeante, lo que le permite identificar la ubicación del siguiente carácter escrito. 

Los caracteres seleccionados están resaltados. La posición del cursor es set al principio de la sección resaltada. 

Por ejemplo: 


| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=position1.gif)<br> | El cursor de entrada se coloca al principio del área de entrada. Su posición es 1. |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Position_fin.gif)<br> | El cursor de entrada se coloca al final del área de entrada. Su posición es 7 en este ejemplo. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=position_sel.gif)<br> | El cursor de entrada se sitúa al principio de la sección resaltada (posición 3 en este ejemplo). El [CursorEnd](../Proprietes/2510047.md) Property se utiliza para get y cambiar el final del área seleccionada. |
|   |   |


<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Cursor** Property se utiliza para get y cambiar la posición del cursor del ratón en uno de los siguientes controles:

- Control Campo de entrada.

- Control Combo Box con entrada.

- "Editar control" y "Editar Combo Box" en un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Editor HTML.




**Observación**: Para get el texto seleccionado en un control Campo de entrada, un Combo Box editable o una columna en un control Tabla, puede utilizar [Texto Seleccionado](../Proprietes/1000017193.md).



**Consejo**: Para seleccionar una sección de texto en una control Campo de entrada de una control Botón:

1. La opción "Borrado automático" no debe estar habilitada en el control Campo de entrada (a menos que se deba borrar todo el contenido del control).

2. El botón debe ser un botón de "Interrupción" y contener, por ejemplo, el siguiente código:
	
	```wl
	// -- Click of BTN_Selection (Interrupt button)
	// Select the entire text
	EDT_Edit1.Cursor = 1
	EDT_Edit1.CursorEnd = Length(EDT_Edit1) + 1
	ScreenFirst(EDT_Edit1)
	```

```wl

```

## Sintaxis

### Averiguar la posición del cursor de entrada

`<Current value> = <Control used>.Cursor`
---

**`<Current value>: (Integro)`**

Posición actual del cursor de entrada en la control. Esta posición puede variar entre:

- 0 (el control no está en edición y no hay cursor de entrada).

- El número de caracteres que aparecen en la control + 1 (el cursor de entrada se encuentra al final del área de entrada).




**`<Control used>: (Nombre del control)`**

Nombre de la control en la que aparece el cursor de entrada: 

- Control Campo de entrada.

- Control Combo Box con entrada.

- "Editar control" y "Editar Combo Box" en un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Editor HTML.





<a name="SYNTAX2"></a>

### Modificar la posición del cursor de entrada

`<Control used>.Cursor = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control donde se debe modificar la posición del cursor de entrada: 

- Control Campo de entrada.

- Control Combo Box con entrada.

- "Editar control" y "Editar Combo Box" en un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Procesador de texto.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Editor HTML.




**`<New value>: (Integro)`**

Nueva posición del cursor de entrada, entre 1 y el número de caracteres que se muestran actualmente en la control + 1.

### Caso especial: Control RFT
<a name="caso_especial_control_rft_ELTPARAGRAPHE000172"></a>

El número de caracteres mostrados en un control RTF puede diferir si este control contiene caracteres CR (Carriage Return).

En efecto, el carácter CR corresponde a dos caracteres en un control no RTF mientras que corresponde a un carácter en un control RTF.
<a name="NOTE0_2"></a>


### Límite: Modificar la posición del cursor de entrada
<a name="limite_modificar_posicion_del_cursor_entrada_ELTPARAGRAPHE000181"></a>

La posición del cursor de entrada puede ser modificada sólo si el control está actualmente en edición. Si el control no está en modo de edición, el **Cursor** Property devuelve 0.

**Recordatorio**: Para forzar la entrada en una control, utilice [ScreenFirst](../WDLang1/3025039.md) o [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md).

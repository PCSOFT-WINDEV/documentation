
## WayPoint (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **WayPoint** permite manipular los puntos de paso de un conector. Esta propiedad se puede utilizar en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Conector. Para obtener más información, consulte la ayuda sobre la propiedad [WayPoint](../Proprietes/1000021957.md). 

- una variable de tipo [diagConnector](../WDLang1/1410088454.md).

- una variable de tipo [diagActionAllowed](../WDLang1/1410088840.md).




## Password (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Password** puede utilizarse: 

- **en los controles de edición** (o en el texto Columnas tabla).
	En este caso, **Password** se utiliza para: 

	- averigua si un control Campo de entrada es una "Contraseña" control

	- modificar (si es necesario) el tipo de contraseña de un control Campo de entrada. 


La opción "Contraseña" permite al usuario escribir un texto sin que este texto sea Visible en la pantalla: todos los caracteres introducidos se sustituyen por estrellas ("\*").
	**Observación**: El tipo "Password" es una opción que se aplica a los controles de edición. Esta opción se encuentra en la pestaña "General" del control description.

- **en las conexiones (HFSQL Client/Server, OLE DB, etc.)**
	En este caso, el **Password** Property establece la contraseña del usuario que utiliza la conexión. En este caso, si la conexión está actualmente abierta, se cerrará y volverá a abrirse con los nuevos parámetros durante el primer acceso a un archivo que esté utilizando esta conexión..
	**Observación**: La conexión puede corresponder a:

	- una conexión descrita en la editor de análisis.

	- una conexión descrita por [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).

	- una conexión descrita por un [Conexión](../WDLang4/1514073.md) variable.

```wl
MyConnection is Connection
// Describe the connection
MyConnection.User = "USER"
MyConnection.Password = "PASSWORD"
MyConnection.Server = "MYSERVER" 
MyConnection.Database = "Database"
MyConnection.Provider = hAccessHFClientServer
MyConnection.Access = hOReadWrite
MyConnection.ExtendedInfo = "Extended information"
MyConnection.CursorOptions = hClientCursor

HOpenConnection(MyConnection)
```

## Sintaxis

### Averiguar si un control Campo de entrada es una "Contraseña" control

`<Result> = <Edit control>.Password`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado es de tipo "Contraseña",

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada (o nombre de la columna Tabla) a utilizar.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Modificar el tipo de "Contraseña" de un control Campo de entrada

`<Edit control>.Password = <New type>`
---

**`<Edit control>: (Nombre del control)`**

Nombre de la control Campo de entrada (o nombre de la columna tabla) a utilizar.

**`<New type>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado debe ser de tipo "Contraseña",

- <u><u><u><u>False</u></u></u></u> en caso contrario.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png)![OLE DB](https://doc.pcsoft.fr/ext/images/us/OLEDB.png)![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) 
### Definición de la contraseña utilizada durante una conexión

`<Connection>.Password = <New password>`
---

**`<Connection>: (Cadena de caracteres)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por el tipo [Conexión](../WDLang4/1514073.md).




**`<New password>: (Cadena de caracteres)`**

Contraseña del usuario actual que se utilizará para la conexión especificada.

## Observaciones

### "Opción"Contraseña
<a name="opcioncontrasena_ELTPARAGRAPHE000166"></a>

La opción "Contraseña" sólo afecta a la visualización del contenido de control. El valor de la control corresponde a los caracteres tecleados por el usuario (y no a las estrellas ("\*")).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Límite
<a name="limite_ELTPARAGRAPHE000175"></a>

El **Password** Property se aplica únicamente a: 

- los controles de edición.

- las columnas de texto tabla.




## SelectionLength (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SelectionLength** Property se utiliza para get y cambiar la longitud de la selección realizada en un control Procesador de texto.

**Consejo**:  Este Property se utiliza para mostrar una selección en el control Procesador de texto. El punto de partida de la selección corresponde a la posición actual del cursor (que se puede manejar con la [Cursor](../Proprietes/2510025.md) Property).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switches the current selection in bold
IF WP_MyDoc.SelectionLength <> 0 THEN
	f is docFragment(WP_MyDoc, WP_MyDoc.Cursor, WP_MyDoc.SelectionLength)
	f.Style.FontBold = True
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la longitud de la selección

`<Result> = <Word Processing control>.SelectionLength`
---

**`<Result>: (Integro)`**

Número de caracteres encontrados en la selección. 

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 


<a name="SYNTAX2"></a>

### Modificar la longitud de selección

`<Word Processing control>.SelectionLength = <New Length>`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 

**`<New Length>: (Integro)`**

Número de caracteres a seleccionar. Si la longitud de selección es negativa o nula, sólo se muestra el simple cursor en la control.

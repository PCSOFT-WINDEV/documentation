


## HTMLEndPage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HTMLEndPage** Property se utiliza para get y cambiar el código HTML insertado al final de la página.

Observación: El código HTML insertado al final de la página también puede especificarse en la pestaña "Avanzado" de la ventana de la página description (pestaña "Html"). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a Google Analytics code
sGoogleCode is string = [
<script type="text/javascript">

var _gaq = _gaq || [];
_gaq.push(['_setAccount', 'UA-99999999-9']);
_gaq.push(['_trackPageview']);

(function() {
var ga = document.createElement('script'); ga.type = 'text/javascript'; ga.async = true;
ga.src = ('https:' == document.location.protocol? 
'https://ssl': 'http://www') + '.google-analytics.com/ga.js';
var s = document.getElementsByTagName('script')[0]; s.parentNode.insertBefore(ga, s);
})();

</script>
]

MyPage.HTMLEndPage = sGoogleCode
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el contenido HTML insertado al final de la página

`<HTML content> = <Page used>.HTMLEndPage`
---

**`<HTML content>: (Cadena de caracteres)`**

Contenido HTML insertado al final de la página.

**`<Page used>: (Nombre de la página)`**

Nombre de la página a utilizar.  


<a name="SYNTAX2"></a>

### Modificar el contenido HTML insertado al final de la página

`<Page used>.HTMLEndPage = <HTML content>`
---

**`<Page used>: (Nombre de la página)`**

Nombre de la página a utilizar.

**`<HTML content>: (Cadena de caracteres)`**

Contenido HTML que se debe insertar al final de la página.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El código HTML se inserta al final de la página, justo antes del cierre de la etiqueta "BODY".

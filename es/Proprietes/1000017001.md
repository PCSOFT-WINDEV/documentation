


## Unicode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Unicode** Property se utiliza para:

- Determina si el valor de una control es una cadena Unicode.

- Especificar si el valor de un control es una cadena de Unicode.




**Observación**: Esta Property es muy useful para gestionar los controles relacionado a un elemento HFSQL en formato Unicode.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the value of EDT_EmailMessage is in Unicode format
IF EDT_EmailMessage.Unicode = False THEN
	EDT_EmailMessage.Unicode = True
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el valor de la control está en formato Unicode

`<Result> = <Control used>.Unicode`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor del control está en formato Unicode,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Especificar si el valor de una control está en formato Unicode

`<Control used>.Unicode = <Unicode/Ansi>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Unicode/Ansi>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el valor del control debe estar en formato Unicode,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de conjuntos de caracteres no latinos
<a name="gestion_conjuntos_caracteres_latinos_ELTPARAGRAPHE000060"></a>

No hay necesidad de usar controles Unicode para manejar los idiomas que usan un carácter no latino set. Simplemente utilice [ChangeCharset](../WDLang1/3054001.md).
<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000070"></a>

El **Unicode** Property está disponible para los siguientes controles:

- Editar control,

- Control Combo Box, 

- Control Estático, 

- Control List Box, 

- Columna de control Tabla,

- Control ListView.

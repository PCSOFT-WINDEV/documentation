
## Sorted (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**Sorted** se utiliza para:

- Averigua si un control está clasificado. 

- Ordenar un control o no. 




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta Property está disponible para los controles List Box, ListView y Combo Box que se rellenan mediante programación.



**Observaciones**: 

- La opción **Sorted** se utiliza para clasificar los elementos en la Lista de Casillas o control Combo Box cuando se muestra. Si la control está ordenada, los elementos se añadirán ([ListAdd](../WDLang1/3049004.md)) teniendo en cuenta la ordenación (los elementos se insertarán realmente).

- Para los controles ListView, la clasificación se realiza según los títulos de los elementos.

- Esta opción se puede elegir en la ventana o en el editor de páginas del control description (pestaña "Detalles", opción "Lista ordenada").







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ListAdd(LIST_List1, "CC")
ListAdd(LIST_List1, "DD")
ListAdd(LIST_List1, "BB")
LIST_List1.Sorted = True
// The order in the List Box control becomes BB, CC, DD
ListAdd(LIST_List1, "AA") // Will be inserted at the beginning of the list
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averigua si un control está clasificado

`<Result> = <Control used>.Sorted`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado está ordenado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Nombre del List Box o control ListView.

- Nombre de la control Combo Box.





<a name="SYNTAX2"></a>

### Ordenar un control o no

`<Control used>.Sorted = <To sort>`
---

**`<Control used>: (Nombre del control)`**

Nombre de la control a clasificar: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre del List Box o control ListView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la control Combo Box.




**`<To sort>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado debe estar ordenado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000126"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **Sorted** Property se aplica sólo a los controles List Box, ListView y Combo Box rellenados mediante programación. El **Sorted** Property no se aplica a los controles Table List Box y Combo Box.

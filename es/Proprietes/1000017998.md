
## Content (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Contenido** Property se utiliza para get y cambiar el contenido de un elemento. Esta propiedad se puede utilizar en:

- Controles de List Box o Combo Box. Para obtener más información, consulte [Contenido Property en los controles](../Proprietes/1000017242.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md). 

- una variable de tipo [emailAttach](../WDLang3/1000018752.md).

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [zipArchivedFile](../WDLang3/1000019044.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [BarCode](../WDLang5/1000019451.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- una variable de tipo [Notification](../WDLang3/1000019441.md). 

- una variable de tipo [Record](../WDLang4/1000020962.md). 

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [httpResponse](../WDLang3/1000021165.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- una variable de tipo [restResponse](../WDLang3/1000021477.md).

- una variable de tipo [notificationFormat](../WDLang3/1000021301.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docCell](../WDLang1/1000022927.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementLink](../WDLang1/1000022970.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementTextArea](../WDLang1/1000023420.md).

- una variable de tipo [pdfAttachment](../WDLang6/1000024899.md).

- una variable de tipo [SecurePassword](../Motscles/1000021064.md).

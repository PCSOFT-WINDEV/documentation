


## Text (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Texto** Property se utiliza para determinar si un elemento es un elemento de texto. **Esta propiedad es de solo lectura**.

**Observación**: Los siguientes elementos se consideran como elementos de texto:

- cadena de caracteres,

- memo texto, 

- carácter.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is the Customer.Phone item a text item?
ResText = Customer.Phone.Text
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Result> = <Data file>.<Item>.Text`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si la posición es una posición de texto, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado.

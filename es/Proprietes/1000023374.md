
## Weight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Peso** Property se utiliza para get y cambiar el peso de un elemento. Esta propiedad se puede utilizar en:

- a control TreeMap. Para obtener más información, consulte [Grosor](../Proprietes/1000018904.md).

- una variable de tipo [memcachedConnection](../WDLang4/1000023339.md).

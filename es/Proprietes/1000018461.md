
## Success (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Éxito** Property indica si la operación fue exitosa. Esta propiedad se puede utilizar en:

- una variable de tipo [sfLeadConvertResult](../WDLang5/1000018739.md)

- una variable de tipo [sfDisconnectResult](../WDLang5/1000018560.md).

- una variable de tipo [sfExecuteProcessResult](../WDLang5/1000018559.md).

- una variable de tipo [sfMergeResult](../WDLang5/1000018499.md).

- una variable de tipo [sfUndeleteResult](../WDLang5/1000018526.md).

- una variable de tipo [sfSaveResult](../WDLang5/1000018460.md).

- una variable de tipo [sfDeleteResult](../WDLang5/1000018546.md).

- una variable de tipo [sfEmptyRecycleBinResult](../WDLang5/1000018467.md).

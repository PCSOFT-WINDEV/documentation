


## TopIndent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TopIndent** Property se utiliza para get y cambiar el espacio en la parte superior del texto (margen superior) en un control.

Esta propiedad está disponible en: 

- Controles campo de entrada.

- Controles estáticos.




**Observación**: Esta Property equivale a fijar el margen superior: 

- Para los controles estáticos: el margen superior se define en la pestaña "Estilo" de la ventana del control description ("Bordes exteriores", "Márgenes", "Margen superior").

- Para un control Edit, el margen superior se define en la pestaña "Estilo" de la ventana del control description ("Área de entrada", "Márgenes", "Margen superior").



<a name="Example1"></a>
<a name="1000022891_sample_code"></a>

## Ejemplo


```wl
// Define a margin equal to 4 pixels at the top and bottom of Static control
STC_Date.BottomIndent = 4
STC_Date.TopIndent = 4
```
<a name="1000022891_Example2"></a>

```wl
// Keep a 5-pixel margin at the top and bottom of the input area
EDT_CarName.BottomIndent = 5
EDT_CarName.TopIndent = 5
```



<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el valor del margen superior en una control

`<Result> = <Control used>.TopIndent`
---

**`<Result>: (Integro)`**

Número de píxeles utilizados para el margen en la parte superior del texto. La sangría es de set a 0 píxeles por Default (sin sangría).  

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Estático, 

- Edita control.





<a name="SYNTAX2"></a>

### Cambiar el valor del margen superior en una control

`<Control used>.TopIndent = <New value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Estático, 

- Edita control.




**`<New value>: (Integro)`**

Nuevo valor para el margen en la parte superior del texto (píxeles ien). Este valor debe ser positivo.

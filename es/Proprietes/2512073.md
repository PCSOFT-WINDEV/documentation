


## Replication (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Replication** Property se utiliza para:

- identificar el modo de replicación utilizado para un archivo de datos (definido en el editor de análisis o mediante programación)

- especificar el modo de replicación utilizado para un archivo de datos (cuando el archivo se describe mediante programación).




**Esta Property sólo puede ser usada en una réplica basada en un registro.**
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the data file is configured for replication (log-based replication)
IF Customer.Replication = hNotReplicated THEN Info("Unable to replicate")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de replicación

`<Current mode> = <Data file>.Replication`
---

**`<Current mode>: (Constante de tipo Integer)`**

Los modos de replicación son los siguientes:


|   |   |
| --- | --- |
| *hMasterReplica* | El archivo de datos es una réplica maestra (se utilizó [HCreateMasterReplica](../WDLang4/3044175.md) para crear la réplica maestra) |
| *hNotReplicated* | El archivo de datos no está configurado para una réplica basada en un registro. Para implementar la replicación basada en registros, seleccione "Replicación" en las características del archivo en la editor de análisis. |
| *hReplicated* | El archivo de datos está configurado para ser replicado. |
| *hSubscriberReplica* | El archivo de datos es una réplica de abonado (se ha utilizado [HCreateSubscriberReplica](../WDLang4/3044253.md) para crear la réplica de abonado) |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definiendo el modo de replicación

`<Data file>.Replication = <Replication mode>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<Replication mode>: (Constante de tipo Integer)`**

Los modos de replicación son los siguientes:


|   |   |
| --- | --- |
| *hNotReplicated* | El archivo de datos no está configurado para una réplica basada en un registro. |
| *hReplicated* | El archivo de datos está configurado para ser replicado. |

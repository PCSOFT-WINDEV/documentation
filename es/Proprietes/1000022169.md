
## MaxButtonWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxButtonWidth** Property se utiliza para obtener y set el ancho máximo de un botón en un panel de pestañas. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TAB_MyDynTab.MaxButtonWidth = 200
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el ancho máximo de un botón en un panel de pestañas.

`<Result> = <Tab control>.MaxButtonWidth`
---

**`<Result>: (Integro)`**



- 0, no se define ningún límite para la anchura máxima de un botón en un panel de pestañas. 

- Ancho máximo (en píxeles) de un botón en un panel de pestañas. 




**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. 


<a name="SYNTAX2"></a>

### Establecer la anchura máxima de un botón en un panel de pestañas.

`<Tab control>.MaxButtonWidth = <Width>`
---

**`<Tab control>: (Nombre del control)`**

Nombre de control Pestaña para usar.

**`<Width>: (Integro)`**



- Ancho máximo (en píxeles) de un botón en un panel de pestañas. 

- 0 para especificar que no hay límite. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **MaxButtonWidth** Property se utiliza para limitar el tamaño de los botones cuando el control Pestaña utiliza la opción "Tamaño del panel: ajustar al texto" (pestaña "Detalles" de la ventana de control description). 

- Caso de las pestañas dinámicas: El texto de la ficha se calcula automáticamente. El **MaxButtonWidth** Property permite evitar textos demasiado largos y botones demasiado grandes. 

- Si el texto de los botones supera el tamaño especificado, el texto se muestra con una elipsis al final ("...").

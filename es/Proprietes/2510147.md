


## Source (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Source** se puede utilizar en:

- **un control Gráfico**: en este caso, el **Fuente** Property se utiliza para get la fuente de la serie: Columna de la tabla, lista de valores, ...
	Observaciones:

	- [grSourceSeries](../WDLang3/3042067.md) se utiliza para definir la fuente de datos de las series.

	- [SourceDescription](../Proprietes/2510148.md) se utiliza para encontrar el nombre y/o el contenido de la fuente de datos.




- **una conexión**: en este caso, el **Fuente** Property se utiliza para:

	- get el origen de una conexión (HFSQL Client/Server, OLE DB, etc.).

	- Definir el origen de una conexión (HFSQL Client/Server, OLE DB, etc.). En este caso, si la conexión está actualmente abierta, se cerrará y se reabrirá con los nuevos parámetros durante el primer acceso a un archivo que utilice esta conexión.

## Sintaxis

### Obtener la fuente de datos de una serie

`<Result> = <Series>.Source`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de fuente de datos de la serie:  


|   |   |
| --- | --- |
| *grCallback* | La fuente de datos es una función de WLanguage. |
| *grItem* | La fuente de datos es una posición de fichero. |
| *grListBox* | La fuente de datos es una control List Box. |
| *grProgramming* | La fuente de datos es una lista de valores. |
| *grTable* | La fuente de datos es una array. |
| *grTableColumn* | La fuente de datos es una columna tabla. |



**`<Series>: (Cadena de caracteres)`**

Series a utilizar. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.

### Obtener el origen de una conexión

`<Data source> = <Connection used>.Source`
---

**`<Data source>: (Cadena de caracteres)`**



- Nombre del servidor (para HFSQL Client/Server, SQL Server, Oracle, MySQL, etc.)

- Nombre del archivo utilizado como fuente de datos (para Access, XML, xBase, etc.)




**`<Connection used>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por el tipo [Conexión](../WDLang4/1514073.md).

### Definir la fuente de datos de la conexión

`<Connection used>.Source = <Data source>`
---

**`<Connection used>: (Cadena de caracteres o variable de tipo Connection)`**

Nombre de la conexión utilizada. Este nombre fue definido:

- en el editor de análisis, 

- por [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md),

- por el tipo [Conexión](../WDLang4/1514073.md).




**`<Data source>: (Cadena de caracteres)`**



- Nombre o IP Address del servidor (para HFSQL Client/Server, SQL Server, Oracle, MySQL, etc.).

- Nombre del archivo utilizado como fuente de datos (para Access, XML, xBase, etc.).

### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000161"></a>

Las propiedades [Servidor](../Proprietes/2512105.md) y **Source** son equivalentes.

Estas dos propiedades se mantienen para que el código sea más fácil de entender. En una conexión (HFSQL Client/Server, SQL Server, Oracle, MySQL, etc.), el [Servidor](../Proprietes/2512105.md) Property es más apropiado porque es necesario especificar un nombre de servidor.

Sin embargo, en una conexión Access, XML o xBase, el **Source** Property es más apropiado porque es necesario especificar un archivo.




## FolderModeImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FolderModeImage** Property se utiliza para get y cambiar el modo de visualización de la carpeta Image en los controles TreeView y TreeView Table.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TVT_MyTreeViewTable.FolderImageWidth = 50
TVT_MyTreeViewTable.FolderModeImage = imgHWIS
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtención del modo de visualización de la carpeta Image

`<Current mode> = <Control used>.FolderModeImage`
---

**`<Current mode>: (Constante de tipo Integer)`**



- Modo de visualización actual de Image:
	


|   |   |
| --- | --- |
| *imgCentered* | centrado: Imagen centrada en la control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Centre_1.GIF)<br> |
| *imgH* | Homotético: Imagen ampliada o reducida (respetando las proporciones de la imagen inicial) para ocupar toda la superficie de control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Homo2.GIF)<br> |
| *imgHC* | Homotético centrado: Image ampliada o reducida (respetando las proporciones del Image inicial) para ocupar toda la superficie del control y centrada en el control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Homocentre.GIF)<br> |
| *imgHCE* | Homotético expandido centrado : Image ampliada o reducida (respetando las proporciones de la Image inicial) para que el lado más pequeño de la Image se muestre completamente en la control y para que la Image se centre en la control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HomoCentreEtendu.GIF)<br> |
| *imgHCWIS* | Homotético sin aumento: Si el Image puede ser mostrado completamente en el control, se muestra al 100% y centrado en el control. En caso contrario, la Image se reduce proporcionalmente para que se muestre por completo y se centre en la control. |
| *imgHE* | Homotético expandido: Imagen ampliada o reducida (respetando las proporciones de la imagen inicial) para que el lado más pequeño de la imagen se muestre por completo en la control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HomoEtendu.GIF)<br> |
| *imgHQ* | En modo homotético, usado para mejorar la calidad de Image cuando ésta se reduce en relación a su tamaño inicial.<br><br>Esta constante debe combinarse con las otras constantes. |
| *imgHWIS* | Homotético sin aumento: Si la imagen puede visualizarse en su totalidad en la control, se muestra en 100%. De lo contrario, la Image se reduce proporcionalmente para poder ser visualizada en su totalidad. |
| *imgNormal* | 100%: Imagen mostrada al 100% en la esquina superior izquierda de control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_100_1.GIF)<br> |
| *imgNoScale* | Permite no hacer zoom en la Image en el modo "Fuentes grandes".<br><br>Esta constante debe combinarse con una de las constantes utilizadas para definir el modo de visualización de Image. |
| *imgStretched* | Expandido: Imagen extendida por toda la superficie de la control.<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Etiree.GIF)<br> |
| *imgTiled* | Mosaico: Image se muestra al 100% y Tile d para ocupar toda la superficie de control (Tile).<br><br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Mosaique.GIF)<br> |






**`<Control used>: (Nombre del control)`**

Nombre del control a utilizar:

- Control TreeView, 

- Control Tabla TreeView.

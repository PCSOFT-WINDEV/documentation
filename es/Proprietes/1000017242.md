
## Content (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Content** permite leer o escribir todo el contenido de un control: 

- Cuadro de lista rellenado mediante programación,

- Combo Box Memoria, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Hoja de cálculo.


Esto permite leer o escribir todas las filas de la control en un solo Line de código.

**Observación**: En una Caja de Listas o control Combo Box basada en un archivo de datos, sólo se puede leer la lista de elementos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds elements to the current content of the List Box control
LIST_List1.Content = LIST_List1.Content + CR + "Tommy" + CR + "Emma"
```




<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
n is int
// Add the row into the Table control (the Combo Box column being the 2nd one)
n = TableAddLine(TABLE_Table1, "1", "c2")
// Modify the content of the Combo Box column for the added row 
// Note: Return values are used via gStoredValue, 
// therefore, this option must be enabled in the description of the column
TABLE_Table1.COL_ComboBox[n].Content = "computer 1" + gStoredValue("c1") + CR + ...
			"computer 2" + gStoredValue("c2")
```

<a name="SYNTAX1"></a>

## Sintaxis

### Obtención del contenido de una control Hoja de cálculo, o de un Cuadro de Listas o Combo Box rellenado mediante programación

`<Control Content> = <Control used>.Content`
---

**`<Control Content>: (Cadena de caracteres)`**

Lista de elementos para el contenido de la control. Los elementos están separados por caracteres CR.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para un control Hoja de cálculo, el contenido del control en formato binario. Este contenido puede ser guardado por [fSaveBuffer](../WDLang1/1000019412.md). 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Esta control debe ser: 

- a control List Box poblada programáticamente,

- a control Combo Box poblada programáticamente,

- una columna tabla de tipo "Lista de valores",

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) y control Hoja de cálculo.





<a name="SYNTAX2"></a>

### Modificación del contenido de un control Hoja de cálculo, o de un Cuadro de Listas o Combo Box poblado mediante programación

`<Control used>.Content = <Control Content>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Esta control debe ser una: 
Nombre del control a manipular. Esta control debe ser: 

- a control List Box poblada programáticamente,

- a control Combo Box poblada programáticamente,

- una columna tabla de tipo "Lista de valores",

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) y control Hoja de cálculo.




**`<Control Content>: (Cadena de caracteres)`**

Lista de elementos que aparecen en la control. Los elementos están separados por caracteres CR.  

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para un control Hoja de cálculo, el contenido binario para mostrar en el control. Utilice la función [fLoadBuffer](../WDLang1/1000019410.md).

## Observaciones
<a name="NOTE0_1"></a>
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### control Hoja de cálculo
<a name="control_hoja_calculo_ELTPARAGRAPHE000139"></a>

El **Content** Property se utiliza para get y modificar el contenido binario del control Hoja de cálculo. Por ejemplo: 

```wl
PSHEET_Show.Content = fLoadBuffer(fExeDir()+"\Fashion Showxls")
```


Esta Property es equivalente a la [Valor](../Proprietes/2510130.md) Property.


## Price (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Precio** Property se utiliza para gestionar el precio de un elemento. 

Esta propiedad se puede utilizar en:

- una variable de tipo [saasPricing](../WDLang3/1000019091.md).

- una variable de tipo [inAppProduct](../WDLang3/1000020868.md).

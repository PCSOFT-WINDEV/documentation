
## Author (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Author** obtiene y cambia el autor de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [gglEvent](../WDLang5/1000017440.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [rssEntry](../WDLang5/1000017792.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docProperty](../WDLang1/1000022945.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfDocument](../WDLang6/1000024909.md).

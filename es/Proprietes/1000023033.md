
## Selection (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Selección** Property se utiliza para get las características de la selección (o del cursor): 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Procesador de texto.
	Observación: Esta selección se encuentra en la sección que se está editando en la control (cuerpo, cabecera o pie de página). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Hoja de cálculo.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un editor HTML control

- en un editor de Diagram control.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Get the fragment selected by the user
MySelection is docFragment = WP_Doc.Selection.Fragment
// The user selection is displayed in bold
MySelection.Formatting.FontBold = True
```

```wl
// Get the first text section selected by the user
MySelection1 is docSection = WP_Doc.Selection.Section[1]
```



```wl
// Retrieve the selected elements in a Diagram Editor control
MySelection is diagSelection <- DIAGEDT_MyDiagram.Selection
IF MySelection.Shape.Count > 0 THEN
	// Get the shapes
	FOR EACH stShape OF MySelection.Shape	
		STC_SELECTION_INFO = stShape.Name + "has been selected." + 
		" [ " + stShape.X + ", " + stShape.Y + " - " + stShape.Width + 
		"x" + stShape.Height + " ]"	
	END
ELSE
	STC_SELECTION_INFO = "Click a shape in the diagram to get the selection"
END
```

## Sintaxis
<a name="SYNTAX1"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Recuperar las propiedades de la selección en un control Procesador de texto

`<Result> = <Word Processing control>.Selection.<Property>`
---

**`<Result>: (Tipo correspondiente a la Property utilizada)`**

Resultado esperado (véase "Observaciones").  

**`<Word Processing control>: (Nombre del control)`**

Nombre de control Procesador de texto.

**`<Property>: ()`**

Property para usar en la selección (ver Observaciones). 




<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Recuperar las propiedades de la selección en un control Hoja de cálculo

`<Result> = <Spreadsheet control>.Selection`
---

**`<Result>: (psheetSelection Variable)`**

[psheetSelection](../WDLang1/1000023498.md) variable describiendo la selección.   

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de control Hoja de cálculo. 




<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Recuperación de las propiedades de la selección en un editor HTML control

`<Result> = <HTML Editor control>.Selection`
---

**`<Result>: (edtHTMLSelection Variable)`**

[edtHTMLSelection](../WDLang1/1000026093.md) variable describiendo la selección.   

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML control.




<a name="SYNTAX4"></a>

### Recuperando las propiedades de la selección en un editor Diagram control

`<Result> = <Diagram Editor control>.Selection`
---

**`<Result>: (diagSelection Variable)`**

[diagSelection](../WDLang1/1410088172.md) variable describiendo la selección.   

**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Selección en una control Procesador de texto
<a name="seleccion_una_control_procesador_texto_ELTPARAGRAPHE000103"></a>

| Property | Explicaciones |
| --- | --- |
| Fragment | &lt;Resultado&gt; corresponde a una [docFragment](../WDLang1/1000022482.md) Variable que contiene las características del fragmento seleccionado. <br><br>Ejemplo: &lt;Resultado&gt; = &lt;Control Procesador de texto&gt;.Selection.Fragment |
| Sección[X] | Se utiliza para obtener la sección associated con la selección. <br><br>- X corresponde al número de sección de get.<br><br>- &lt;Resultado&gt; corresponde a una [docSection](../WDLang1/1000022685.md) Variable que contiene las características de la sección associated con la selección. <br><br><br><br><br>Ejemplo: &lt;Resultado&gt; = &lt;control Procesador de texto&gt;.Selección.Sección[1] |

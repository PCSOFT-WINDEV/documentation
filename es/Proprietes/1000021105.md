


## ShortcutKey (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ShortcutKey** obtiene y establece el método abreviado de teclado del control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// To be included in the declaration code of global variables of window
EXTERN "KEYCONST.wl"

BTN_BUTTON1.ShortcutKey = skControl + skShift + Asc("U") // Ctrl + Shift + U
BTN_BUTTON1.ShortcutKey = skControl + VK_F1		// Ctrl + F1
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubriendo el atajo de teclado associated con un control

`<Result> = <Control used>.ShortcutKey`
---

**`<Result>: (Integro)`**

Tecla de atajo associated con el control. 

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. 


<a name="SYNTAX2"></a>

### Modificando el atajo de teclado associated con un control

`<Control used>.ShortcutKey = <New shortcut>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. 

**`<New shortcut>: (Constante de tipo Integer)`**

Tecla de atajo associated con el control. Esta clave puede corresponder a uno de los siguientes elementos o a una combinación de los siguientes elementos: 

- Llaves de control: 
	


|   |   |
| --- | --- |
| *skAlt* | Tecla alt. |
| *skControl* | Tecla Ctrl. |
| *skShift* | Tecla mayús. |



- Llaves específicas de control: Constantes adicionales están disponibles en el archivo "KeyConst.wl. Estas constantes se utilizan para identificar todas las teclas del teclado. Este archivo está disponible en el directorio "\\Personal\\Extern" de WINDEV o WINDEV Mobile. Para obtener más información, consulte [observaciones](#NOTE0_1).

- Teclas de caracteres: Basta con utilizar el código ASCII del carácter (devuelto por [Asc](../WDLang1/3024004.md)).

## Observaciones


### Gestión de todas las teclas del teclado
<a name="gestion_todas_las_teclas_del_teclado_ELTPARAGRAPHE000065"></a>

Para gestionar todas las llaves con el **ShortcutKey** Property:

1. Incluya el archivo KeyConst.WL en su proyecto. Este archivo asocia una constante específica con cada tecla del teclado. Este archivo está disponible en el directorio \\Personal\\Externo de WINDEV. Para incluir este archivo en sus aplicaciones, debe utilizar la palabra clave [EXTERN](../Motscles/1511013.md) de la siguiente manera:
	
	```wl
	EXTERN "KEYCONST.wl"
	```


2. Utiliza las constantes en el **ShortcutKey** Property:
	
	```wl
	cControl is Control <- ControlCreate("BTN_Button", typButton, 6, 60, 160, 30, False)
	cControl.ShortcutKey = skAlt + skShift + VK_F10
	```





Observaciones: 

- Para gestionar las claves de los caracteres, basta con utilizar el código ASCII de ese carácter (devuelto por [Asc](../WDLang1/3024004.md)).

- Todas las constantes encontradas en el archivo KeyConst.WL comienzan con las letras VK_.. Estas constantes son mostradas por el completador de código en el editor de código desde el momento en que el archivo ha sido integrado con la palabra clave [EXTERN](../Motscles/1511013.md).

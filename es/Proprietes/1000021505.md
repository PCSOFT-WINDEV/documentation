


## CrosshairColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CrosshairColor** Property se utiliza para get y cambiar el Color de la retícula en un control Gráfico. 

Observación: Si la Color del retículo es automatic, esta Color se basa en la Color de la cuadrícula para la control Gráfico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Set the crosshair color to green
CHART_MyChart.CrosshairColor = LightGreen
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir la Color del retículo para una control Gráfico

`<Result> = <Chart control>.CrosshairColor`
---

**`<Result>: (Constante de tipo Integer)`**

La actual Color del retículo del gráfico. Este parámetro se corresponde con uno de los parámetros de [Constantes de Color](../WDLang5/3010002.md). 

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se va a utilizar. 


<a name="SYNTAX2"></a>

### Modificar la Color del retículo para una control Gráfico

`<Chart control>.CrosshairColor = <New color>`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se va a utilizar. 

**`<New color>: (Constante de tipo Integer)`**

La nueva Color del retículo del gráfico. Este parámetro se corresponde con uno de los parámetros de [Constantes de Color](../WDLang5/3010002.md).




## FontBold (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**FontBold** se utiliza para:

- Determine si el texto está en negrita.

- Cambiar el peso del texto.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Write the text displayed in "STC_StaticControl" in bold
STC_StaticControl.FontBold = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el texto mostrado en una control está en negrita

`<Result> = <Control used>.FontBold`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto que se muestra está en negrita,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Aplicar o eliminar el formato de negrita del texto mostrado en una control

`<Control used>.FontBold = <Bold/Not bold>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Bold/Not bold>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto que se muestra debe estar en negrita,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000098"></a>

En un reporte, el **FontBold** Property sólo puede utilizarse en:

- a control Estático.

- to URL control Enlace.

- una control calculada .

- un control preestablecido.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **FontBold** Property no se puede utilizar en:

- a Control de número,

- una Web control Cámara,

- a control Conferencia.


Observación: Esta Property se aplica únicamente a los controles "ListView" en el modo de visualización "List Box".

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En Java, el **FontBold** Property sólo se aplica a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Combo Box.

- Control TreeView.

- Columna de tabla.

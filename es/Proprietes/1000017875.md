


## MIMEType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MIMEType** Property se utiliza para get y cambiar el tipo MIME de un elemento. Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpImage](../WDLang3/1000020905.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [gglDocument](../WDLang5/1000018001.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [gglAttachment](../WDLang5/1000026182.md).

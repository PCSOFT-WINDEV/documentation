


## ListViewMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ListViewMode** Property se utiliza para:

- Descubra el modo de visualización de un control ListView. 

- Modificar el modo de visualización de un control ListView.




**Observación**: El modo de visualización de una control ListView también puede modificarse en el editor, en la ventana de control description (pestaña "Detalles").

**Recordatorio**: Un control ListView se puede mostrar: 

- en modo estándar: el control se muestra sin imágenes. 

- en modo ListView: el control se muestra con imágenes. 

- in panorama mode: la control se muestra con las imágenes como si se presentaran en una pared curva. Sólo se pueden visualizar 20 imágenes como máximo en este modo.  



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to standard mode
LSV_ListView1.ListViewMode = lvmSimpleList
```

<a name="SYNTAX"></a>

### Descubra el modo de visualización de un control ListView

`<Result> = <ListView control>.ListViewMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de visualización de control ListView: 


|   |   |
| --- | --- |
| *lvmListView* | control ListView se muestra en modo ListView. |
| *lvmPanorama* | control ListView se muestra en modo panorámico. |
| *lvmSimpleList* | control ListView se muestra en modo estándar. |



**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.  


<a name="SYNTAX4"></a>

### Modificando el modo de visualización de un control ListView

`<ListView control>.ListViewMode = <New display mode>`
---

**`<ListView control>: (Nombre del control)`**

Nombre de control ListView a utilizar.

**`<New display mode>: (Constante de tipo Integer)`**

Modo de visualización de control ListView: 


|   |   |
| --- | --- |
| *lvmListView* | control ListView se muestra en modo ListView. |
| *lvmPanorama* | control ListView se muestra en modo panorámico. |
| *lvmSimpleList* | control ListView se muestra en modo estándar. |




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La propiedad **ListViewMode** sólo puede utilizarse en controles ListView.

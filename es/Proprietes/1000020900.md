
## SerialNumber (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SerialNumber** Property se utiliza para gestionar los números de serie. Esta propiedad se puede utilizar en: 

- una variable de tipo [Certificate](../WDLang1/1000019287.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [upnpDevice](../WDLang3/1000020903.md).

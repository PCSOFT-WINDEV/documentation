
## DisplayedSection (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Sección visualizada** Property se utiliza para identificar y cambiar la parte del Image que se visualiza en un control Imagen.

Esta propiedad se utiliza principalmente para acercar una imagen mostrando solo una sección. Esta propiedad debe utilizarse en un control Imagen que permita hacer zoom. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save displayed section
rectSave is Rectangle
rectSave = IMG_Photo.DisplayedSection
// Restore displayed section
IMG_Photo.DisplayedSection = rectSave

// Zoom X2 on top right
rectX2 is Rectangle
rectX2.X = 1000
rectX2.Y = 0
rectX2.Width = 500
rectX2.Height = 500
IMG_Photo.DisplayedSection = rectX2

rectX2.DisplayedSection = O
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Conseguir que la sección Image aparezca en un control Imagen

`<Result> = <Image control>.DisplayedSection`
---

**`<Result>: (Variable de tipo Rectangle)`**

Variable de tipo [Rectangle](../Motscles/1514079.md) que corresponde a la sección de la imagen que se muestra.

Si el control Imagen no contiene una imagen (o contiene una imagen generada (funciones de dibujo)), el rectángulo devuelto corresponderá al control.  

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.


<a name="SYNTAX2"></a>

### Cambiar la sección de la Image que se muestra en una control Imagen

`<Image control>.DisplayedSection = <New section displayed>`
---

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<New section displayed>: (Variable de tipo Rectangle)`**



- Nombre de la variable de tipo [Rectangle](../Motscles/1514079.md) que corresponde a la sección de la imagen a manipular. 

- 0 para mostrar toda la imagen en el control Imagen. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El rectángulo corresponde a pixeles de la imagen. 

- Si la imagen tiene varios DPI, el rectángulo corresponde a un zoom 1x. 

- Al modificar la sección de la imagen que se muestra (sintaxis 2): 

	- El zoom se calcula según el tamaño del rectángulo especificado. 

	- Si el rectángulo no tiene la misma relación x/y que el control Imagen , se usa la relación más grande. 

	- Si el rectángulo está vacío, la acción no tiene ningún efecto.

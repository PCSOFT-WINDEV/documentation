


## RTFWithImagesAndTables (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El programa **RTFWithImagesAndTables** Property se utiliza para gestionar imágenes y tablas en controles estáticos RTF en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable the support for the images and tables 
// in the RTF of the STC_Description control
STC_Description.RTFWithImagesAndTables = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el modo de gestión de imágenes y tablas en RTF

`<Result> = <Static control>.RTFWithImagesAndTables`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las imágenes y las tablas son compatibles con el control Estático en RTF, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Static control>: (Nombre del control)`**

Nombre de la control Estático que se va a utilizar. Este control se encuentra en un reporte. 


<a name="SYNTAX2"></a>

### Modificando el modo de gestión de imágenes y tablas en RTF

`<Static control>.RTFWithImagesAndTables = <New mode>`
---

**`<Static control>: (Nombre del control)`**

Nombre de la control Estático que se va a utilizar. Este control se encuentra en un reporte.

**`<New mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para soportar las imágenes y tablas del control Estático en RTF, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Este Property le permite cambiar el modo de gestión de las imágenes y tablas en un control. También puede cambiar este modo de gestión para todos los controles utilizando [iParameter](../WDLang5/3046008.md) con la constante *iRTFWithImagesAndTables*.

- La función [iParameter](../WDLang5/3046008.md) con la constante *iRTFWithImagesAndTables* tiene prioridad sobre la propiedad **RTFWithImagesAndTables**. Por tanto, la propiedad **RTFWithImagesAndTables** se ignora si se llama a la función [iParameter](../WDLang5/3046008.md) con la constante *iRTFWithImagesAndTables*.


## CtAccess (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtAccess** Property se utiliza para set el tipo de acceso a el archivo de datos cuando se conecta a un tabla específico a través de OLE DB. Es posible:

- get el tipo de acceso a el archivo de datos utilizado para conectarse a un tabla vía OLE DB.
	El tipo de acceso puede corresponder a:

	- el tipo de acceso definido en el editor de análisis,

	- el tipo de acceso especificado en [HConnect](../WDLang4/3044263.md), [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).
			Este Property puede utilizarse para un archivo de datos o para una conexión definida en el editor de análisis o mediante programación.




- Definir el tipo de acceso a el archivo de datos utilizado para conectarse a una tabla vía OLE DB (cuando se describe programáticamente sólo el archivo de datos).
	En este caso, esta Property es equivalente a la [CtDescribedAccess](../Proprietes/2512048.md) Property. 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describió el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the type of access to the Customer data file
ResAccessType = Customer.CtAccess
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el tipo de acceso a el archivo de datos por una conexión

`<Type of access> = <Data file or Connection>.CtAccess`
---

**`<Type of access>: (Constante)`**

Devuelve el tipo de acceso a el archivo de datos.


|   |   |
| --- | --- |
| *hORead* | Se puede acceder al archivo de datos en modo de solo lectura. Solo se pueden leer registros. No se pueden agregar o eliminar. |
| *hOReadWrite* | Se puede acceder al archivo de datos en modo de lectura y escritura. Se pueden leer y escribir nuevos registros. |



**`<Data file or Connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre de la conexión utilizada. Este nombre se definió en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).  





<a name="SYNTAX2"></a>

### Definir el tipo de acceso al archivo por una conexión

`<Data file or Connection>.CtAccess = <Type of access>`
---

**`<Data file or Connection>: (Cadena de caracteres)`**



- Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

- Nombre de la conexión utilizada. Este nombre se definió en el editor de análisis o con las funciones [HDescribeConnection](../WDLang4/3044205.md) o [HOpenConnection](../WDLang4/3044107.md).




**`<Type of access>: (Constante)`**

Define el tipo de acceso a el archivo de datos.


|   |   |
| --- | --- |
| *hORead* | Se puede acceder al archivo de datos en modo de solo lectura. Solo se pueden leer registros. No se pueden agregar o eliminar. |
| *hOReadWrite* | Se puede acceder al archivo de datos en modo de lectura y escritura. Se pueden leer y escribir nuevos registros. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000110"></a>

El [CtDescribedAccess](../Proprietes/2512048.md) Property se utiliza para get y cambiar el tipo de acceso especificado para la conexión OLE DB cuando se describe el archivo de datos.
<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000120"></a>

El **CtAccess** Property se utiliza para definir el tipo de acceso a un archivo de datos que maneja una conexión. Este Property se mantiene para la compatibilidad retroactiva. Se recomienda utilizar el [Acceso](../Proprietes/2512100.md) Property. El [Acceso](../Proprietes/2512100.md) Property se utiliza para definir el tipo de acceso a los archivos de datos que maneja una conexión.

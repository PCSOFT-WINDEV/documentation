
## Error

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Error** permite gestionar los errores. Esta palabra clave puede corresponder a:

- la palabra clave "CASE ERROR" utilizada para [Gestión de errores automatic](../WDLang1/3034010.md). 

- la función **[Error](../WDLang1/3021013.md)**.

- la **Error** Property de un Variable de tipo [sfLeadConvertResult](../WDLang5/1000018739.md). 

- la **Error** Property de un Variable de tipo [sfDisconnectResult](../WDLang5/1000018560.md). 

- el **Error** Property de un Variable de tipo [sfMergeResult](../WDLang5/1000018499.md).

- el **Error** Property de un Variable de tipo [sfUndeleteResult](../WDLang5/1000018526.md).

- el **Error** Property de un Variable de tipo [sfSaveResult](../WDLang5/1000018460.md).

- el **Error** Property de un Variable de tipo [sfDeleteResult](../WDLang5/1000018546.md).

- el **Error** Property de un Variable de tipo [sfEmptyRecycleBinResult](../WDLang5/1000018467.md).

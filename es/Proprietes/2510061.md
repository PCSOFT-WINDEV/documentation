


## Driver (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Driver** permite obtener el controlador de captura de video al que está asociado el control Cámara.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info("The Camera control in this window is associated with the driver " + CAM_Camera1.Driver)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Camera control>.Driver`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena vacía ("") si el control de la cámara es associated sin controlador para la captura de vídeo. No se puede utilizar ninguna función de vídeo.

- Driver para la captura de vídeo associated con el control Cámara especificado. Este resultado tiene el siguiente formato: 
	
	```txt
	<Driver name> + TAB + <Driver version>
	```

	Donde: 

	- &lt;Nombre del controlador&gt; corresponde al nombre del controlador de vídeo,

	- &lt;La versión del controlador&gt; corresponde a la versión del controlador.







**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar

## Observaciones


### Gestión de los controladores para la captura de vídeo
<a name="gestion_los_controladores_para_captura_video_ELTPARAGRAPHE000040"></a>

[VideoSetDriver](../WDLang1/3079010.md) se utiliza para modificar el controlador para la captura de vídeo asociado al control Cámara y [VideoListDriver](../WDLang1/3079008.md) se utiliza para get la lista de controladores para la captura de vídeo instalados en el el equipo.

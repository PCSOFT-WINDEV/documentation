


## NumberPage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NumberPage** permite obtener: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de páginas que se encuentran en un archivo "multipágina" de Image. Esta Image se muestra en una control Imagen o en el fondo de una control Gráfico.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de páginas que se encuentran en un archivo PDF mostrado en una control Imagen.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de páginas que se encuentran en un archivo PDF que se muestra en un lector de PDF control.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el número de páginas encontradas en un archivo DOCX mostrado en una control Procesador de texto.




**Recordatorio**: Algunos formatos de Image (TIFF por ejemplo) pueden contener varios Image s en el mismo archivo. Cada Image corresponde a una página de archivo. Estos formatos se denominan formatos "multipágina".


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Modify the image file associated with the "IMG_MyImage" control
IMG_MyImage.Image = "MultipageFile.TIF"
// Successively display the different pages found in the image file
FOR I = 1 TO IMG_MyImage.NumberPage
	IMG_MyImage.PageNumber = I
	// Timeout for loading images
	Multitask(-100)
END
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Display page 23 in the PDF Reader control
IF PDF_MyPDFReader.NumberPage>23 THEN
	PDF_MyPDFReader.NumberDisplayedPage = 23
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Number of pages> = <Control used>.NumberPage`
---

**`<Number of pages>: (Integro)`**



- Número de páginas encontradas en el archivo associated con el control especificado.

- 1 si el formato del archivo Image no es "multipágina" o si el formato "multipágina" contiene una sola página.

- 0 si el control especificado no es associated con un formato Image soportado por WINDEV.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de control Imagen,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de control Gráfico,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre del lector de PDF control.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de control Procesador de texto.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modificar la página mostrada
<a name="modificar_pagina_mostrada_ELTPARAGRAPHE000087"></a>

Para modificar la página mostrada: 

- Para los controles Image y los controles Chart, utilice [PageNumber](../Proprietes/2510106.md).

- Para los controles del lector de PDF y los controles del procesador de textos, utilice [NumberDisplayedPage](../Proprietes/1000023479.md).



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Caso de archivos PDF y DOCX con un gran número de páginas
<a name="caso_archivos_pdf_docx_con_gran_numero_paginas_ELTPARAGRAPHE000109"></a>

Si el archivo PDF o DOCX abierto en el control contiene un gran número de páginas, el control carga cada página sucesivamente: algunas páginas no son accesibles inmediatamente. 

En ese caso: 

- **NumberPage** es un bloqueo property: espera a que se complete la operación de carga de archivos. La aplicación está bloqueada. 

- La [NúmeroPáginas accesibles](../Proprietes/1000023477.md) Property devuelve el número de páginas cargadas actualmente en el archivo PDF sin esperar a que se complete la carga Process.

- El [NumberDisplayedPage](../Proprietes/1000023479.md) Property devuelve el número de la página que se muestra actualmente en el control (al menos el 50% de la página). 



<a name="NOTE0_3"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000130"></a>

El **NumberPage** Property se aplica únicamente a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de Image en las ventanas,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de los gráficos en las ventanas,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de PDF Reader.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de procesamiento de textos.

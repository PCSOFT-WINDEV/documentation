


## Condensed (Property)

*** ***
	

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>27</B>, este Property se mantiene para la compatibilidad con el pasado. Se recomienda utilizar el nuevo [CharacterSpacing](../Proprietes/1410088367.md) Property para gestionar el espaciado de los caracteres. 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Condensado** Property se utiliza para:

- Determinar si una fuente Variable está condensada.

- Condensar la fuente Variable.




**Observación**: Los caracteres de la fuente estirada pueden ser:

- Normal: los personajes son Normal.

- Condensado: son más estrechos (**Condensed** property).

- Extendido: son más amplios ([Extendido](../Proprietes/2513002.md) property).

- Grande: los caracteres son lo más amplios posible ([Grande](../Proprietes/2513012.md) property).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the characteristics of a Font variable
Font1 is Font
Font1.Name = "Arial"
Font1.Size = 14
Font1.Condensed = True
// Apply the font to "MyControl"
MyControl.Font = Font1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cómo determinar si una fuente Variable está condensada

`<Result> = <Font>.Condensed`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fuente especificada está condensada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Font>: (Variable de tipo Font)`**

Nombre de la variable de tipo [tipo de letra](../Motscles/1514045.md) que se utilizará.  


<a name="SYNTAX2"></a>

### Condensar una fuente Variable

`<Font>.Condensed = <Condensed/Not condensed>`
---

**`<Font>: (Variable de tipo Font)`**

Nombre de la variable de tipo [tipo de letra](../Motscles/1514045.md) que se utilizará.

**`<Condensed/Not condensed>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fuente especificada debe ser condensada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Variable de tipo Font
<a name="variable_tipo_font_ELTPARAGRAPHE000092"></a>

Se puede crear una fuente a partir de una variable de tipo [tipo de letra](../Motscles/1514045.md).

Para definir las características de la fuente, utilice:

- la función [FontCreate](../WDLang1/3060002.md), 

- o las propiedades [Charset](../Proprietes/2512008.md), [Orientación](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Color](../Proprietes/2510071.md), [Extendido](../Proprietes/2513002.md), [atrevido](../Proprietes/2513001.md), [Cursiva](../Proprietes/2513007.md), [Grande](../Proprietes/2513012.md), [denominar](../Proprietes/2510082.md), [subrayar](../Proprietes/2513013.md) y [envergadura](../Proprietes/2510125.md).




Esta fuente se puede utilizar:

- Para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- Para escribir en los gráficos (con las funciones [grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md) y [grAxisTitleFont](../WDLang3/3042059.md)).

- Para imprimir (con la función [iFont](../WDLang5/3046014.md)).

- En los controles de una ventana (propiedad [tipo de letra](../Motscles/1514045.md)).

- En los controles de un reporte (propiedad [tipo de letra](../Motscles/1514045.md)).



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000175"></a>

Las propiedades que definen las opciones de estiramiento de los caracteres (propiedades **Condensed**, [Extendido](../Proprietes/2513002.md) y [Grande](../Proprietes/2513012.md) ) sólo se admiten si se utiliza el [tipo de letra](../Motscles/1514045.md) Variable especificado para la impresión (fuente de impresión o fuente de los controles en un reporte).

En los demás casos (fuente utilizada para los dibujos, los gráficos o los controles de ventana), estas características se ignoran.




## HTMLEdit (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **HTMLEdit** Property se utiliza para: 

- averiguar si una control Procesador de texto se muestra en modo optimizado para la edición de HTML,

- modificar una control Procesador de texto para mostrarla (o no) en modo optimizado para la edición HTML. 




Observación: Si la propiedad **HTMLEdit** está activada, la propiedad [DisplayMode](../Proprietes/1000023378.md) no puede ser modificada y devuelve la constante *displayModeWeb*. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Web mode] button
WP_MyDocument.HTMLEdit = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si una control Procesador de texto se muestra en modo optimizado para la edición de HTML

`<Result> = <Word Processing control>.HTMLEdit`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Procesador de texto se muestra en modo optimizado para la edición HTML,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.


<a name="SYNTAX2"></a>

### Modificación de una control Procesador de texto para mostrarla (o no) en modo optimizado para la edición HTML

`<Word Processing control>.HTMLEdit = <New value>`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<New value>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Procesador de texto se muestra en modo optimizado para la edición HTML,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

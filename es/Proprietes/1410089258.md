
## Document (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Document** puede corresponder a: 

- la variable de tipo [Document](../WDLang1/1000022461.md), que se utiliza para manipular documentos de procesadores de texto. 

- la propiedad **Document**. Esta propiedad se puede utilizar en:

	- una variable de tipo [mongoOperationAdd](../WDLang4/1410089163.md).

	- una variable de tipo [mongoOperationReplaceOne](../WDLang4/1410089166.md).

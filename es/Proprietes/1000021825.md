


## CalculatedDefaultValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CalculatedDefaultValue** Property se utiliza para:

- Averiguar la expresión SQL que define el valor Default de un elemento. Este punto se definió en la editor de análisis o a través de la programación.

- Definir la expresión SQL que devuelve el valor de Default de un elemento (elemento descrito sólo mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
AnItem.Name = "CmdDate"
AnItem.Type = hItemDateTime
AnItem.CalculatedDefaultValue = "SYSDATE"
HDescribeItem(AFile, AnItem)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la expresión SQL que define el valor Default de un elemento

`<Result> = <Data file>.<Item>.CalculatedDefaultValue`
---

**`<Result>: (Cadena de caracteres)`**

Expresión SQL que define el valor Default del artículo.

**`<Data file>: (Nombre del archivo)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Nombre del campo)`**

Nombre del artículo utilizado.


<a name="SYNTAX2"></a>

### Definición de la expresión SQL que devuelve el valor Default de un elemento descrito mediante programación

`<Item>.CalculatedDefaultValue = <SQL expression>`
---

**`<Item>: (Nombre del campo)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<SQL expression>: (Cadena de caracteres)`**

Expresión SQL que define el valor Default del artículo. Esta expresión no debe utilizar ningún otro elemento.

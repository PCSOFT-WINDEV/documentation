


## CompactToolbar (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CompactToolbar** Property se utiliza para:

- averiguar si las barras de herramientas que se encuentran en la ventana están compactadas.

- compactar (o no) las barras de herramientas.




La compresión de las barras de herramientas consiste en eliminar los caracteres de espacio cuando:

- una opción de la barra de herramientas se hace visible o Invisible. La barra de herramientas se redimensiona automáticamente cuando una opción se convierte en Invisible.

- una barra de herramientas se hace visible o Invisible. Las otras barras de herramientas se reposicionan automáticamente cuando una barra de herramientas se convierte en Invisible para evitar dejar espacio libre.

- una barra de herramientas está acoplada o desacoplada.




Las barras de herramientas no se compactan si el usuario las mueve mientras está acoplado.

**Observación**: Esta propiedad corresponde a la opción "Compactar barras de herramientas" de la pestaña "Detalles" de la ventana description.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WIN_MainWin.CompactToolbar = True
TBAR_Bar1.Visible = True
WIN_MainWin.CompactToolbar = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si las barras de herramientas están compactadas

`<Result> = <Window used>.CompactToolbar`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las barras de herramientas de la ventana están compactadas,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana utilizada.  


<a name="SYNTAX2"></a>

### Definir la compresión de las barras de herramientas

`<Window used>.CompactToolbar = <Compact>`
---

**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana utilizada.

**`<Compact>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las barras de herramientas de la ventana deben estar compactadas, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## Latitude (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Latitud** Property se utiliza para get y cambiar la latitud de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [gglCoordinate](../WDLang5/1000017489.md).

- una variable de tipo [gglMarker](../WDLang5/1000017442.md).

- una variable de tipo [geoPosition](../WDLang3/1000019191.md).

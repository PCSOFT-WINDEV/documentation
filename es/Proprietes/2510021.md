


## InitialContent (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **InitialContent** obtiene el contenido inicial:

- de un control List Box rellenado mediante programación.

- de un control Combo Box rellenado mediante programación.

- de una columna de tipo "Lista de valores" de un control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de un control Hoja de cálculo.




Este contenido inicial se define:

- en la descripción del control (pestaña "General" de los controles List Box y Combo Box o de la columna de tipo "Lista de valores").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) al inicializar el control Hoja de cálculo en el editor de ventanas.




Observación: El contenido inicial de controles List Box o Combo Box puede modificarse mediante programación si se encuentran en un control Looper. 

**Importante**: Un control Looper permite crear el atributo 'Contenido inicial' en un control Combo Box o List Box. Por lo tanto, el contenido del control List Box o Combo Box puede ser diferente en cada fila del Looper.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Clears the list and restores its initial content
ListDeleteAll(LIST_List1)
ListAdd(LIST_List1, LIST_List1.InitialContent)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el contenido inicial de un control

`<Initial content> = <Control used>.InitialContent`
---

**`<Initial content>: (Cadena de caracteres)`**

Lista de elementos introducidos en el editor de ventanas o páginas para el contenido inicial del objeto. Los elementos están separados por retornos de carro (caracteres CR).

**`<Control used>: (Nombre del control)`**

Nombre del control del que se quiere obtener el contenido inicial. Este control debe ser de tipo: 

- List Box rellenado mediante programación.

- Combo Box rellenado mediante programación, 

- Columna de tipo "Lista de valores",

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Hoja de cálculo inicializada en el editor de ventanas.





<a name="SYNTAX2"></a>

### Cambiar el contenido inicial de un control List Box o Combo Box, o de una columna en un control Looper

`<Attribute name>.InitialContent = <Initial content>`
---

**`<Attribute name>: (Cadena de caracteres)`**

Nombre del atributo vinculado al control cuyo contenido inicial debe modificarse. El control vinculado al atributo debe ser un List Box o Combo Box rellenado mediante programación o una columna de tipo "Lista de valores" contenida en un control Looper. Este parámetro tiene la siguiente forma: 

```txt
<Attribute name>[Row index]
```


**`<Initial content>: (Cadena de caracteres)`**

Lista de elementos que definen el contenido inicial del objeto. Los elementos están separados por retornos de carro (caracteres CR).

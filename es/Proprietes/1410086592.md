
## X (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **X** Property se utiliza para get la coordenada X de un elemento. Esta propiedad se puede utilizar en:

- controles, ventanas, páginas e informes. Para obtener más información, consulte [X](../Proprietes/2510135.md).

- una variable de tipo [Punto](../Motscles/1000018908.md).

- una variable de tipo [Rectangle](../Motscles/1514079.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementImage](../WDLang1/1000022696.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementTextArea](../WDLang1/1000023420.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfTextElement](../WDLang6/1000024556.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagHead](../WDLang1/1410088470.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagShape](../WDLang1/1410088082.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagPoint](../WDLang1/1410088140.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagWaypoint](../WDLang1/1410088548.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagConnectionPoint](../WDLang1/1410088457.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [diagSelection](../WDLang1/1410088172.md).


## FontName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FontName** Property se utiliza para gestionar el nombre de la fuente utilizada. Esta propiedad se puede utilizar en: 

- controles. Para obtener más información, consulte [FontName](../Proprietes/2510100.md). 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfTextElement](../WDLang6/1000024556.md).

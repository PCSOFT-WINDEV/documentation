
## ExpandedImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ExpandedImage** Property se utiliza para: 

- averigua y modifica la imagen por defecto associated con un fila expandido en un control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) descubrir y/o modificar la imagen por defecto associated con un fila expandido en un control TreeView.

- descubrir y modificar la imagen associated con un fila expandido específico en un control Tabla TreeView.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) conocer y/o modificar la imagen de una fila expandida en un elemento jerárquico de un reporte de tipo "Tabla TreeView". 




Esta Property se utiliza para definir: 

- la imagen que será usada por Default durante las próximas adiciones en el control Tabla TreeView. Esta Image será tenida en cuenta por [TableAdd](../WDLang1/3074017.md), [TableAddLine](../WDLang1/3074006.md), [TableAddChild](../WDLang1/1000010004.md), [TableInsert](../WDLang1/3074025.md), [TableInsertLine](../WDLang1/3074026.md), [TableInsertChild](../WDLang1/1000010005.md), etc.

- el Image ampliado para una de las filas que se encuentran en el control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la imagen que será usada por Default durante las próximas adiciones en el control TreeView. Esta Image será tenida en cuenta por [TreeAdd](../WDLang1/3018012.md) y [TreeInsert](../WDLang1/3018023.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) el Image expandido de las filas que se encuentra en un tabla "TreeView" reporte.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines the image used for an expanded row
TVT_Recipe.ExpandedImage = "C:\MyImages\Image.BMP"
```


<a name="Example2"></a>

```wl
// Changes the expanded image of row 4
TVT_Recipe[4].ExpandedImage = "C:\MyImages\Image.BMP"
```


<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Change the expanded image in the report
// --BeforePrinting ITEM_TreeView
If MySelf.HierarchicalDepth = 1 THEN MySelf.ExpandedImage = "C:\Images\tool.png"
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar la imagen por defecto associated con un fila expandido o la imagen associated con un fila específico

`<Result> = <Control used>.ExpandedImage`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre y ruta completo de la imagen por defecto associated con las filas expandidas del control.

- Cadena vacía ("") si la imagen usada por Default no fue modificada.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView.




Para encontrar la imagen expandida associated con un fila específico en un control Tabla TreeView, use la siguiente notación: 

```txt
<TreeView Table control>[<Row number>]
```



<a name="SYNTAX2"></a>

### Modificando la imagen por defecto associated con un fila expandido la imagen associated con un fila específico

`<Control used>.ExpandedImage = <New image>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control TreeView.




Para modificar la imagen expandida associated con un fila específico en un control Tabla TreeView, use la siguiente notación: 

```txt
<TreeView Table control>[<Row number>]
```


**`<New image>: (Cadena de caracteres)`**

Nombre y ruta completa de la nueva imagen associated por Default con las filas expandidas en un control Tabla TreeView.  




<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Descubrir el Image de un elemento ampliado de la "TreeView Table" reporte

`<Result> = <TreeView item>.ExpandedImage`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre y ruta completa de la imagen asociada a una fila ampliada en el elemento de la vista en árbol. 

- Vaciar la cadena ("") si no se utiliza la Image.




**`<TreeView item>: (Cadena de caracteres)`**

Nombre del elemento TreeView a utilizar.


<a name="SYNTAX4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Modificando la Image de un elemento ampliado de la "TreeView Table" reporte

`<TreeView item>.ExpandedImage = <New image>`
---

**`<TreeView item>: (Cadena de caracteres)`**

Nombre del elemento TreeView a utilizar.  

**`<New image>: (Cadena de caracteres)`**

Nombre y ruta completa de la nueva imagen asociada a una fila ampliada en el elemento de la vista en árbol.

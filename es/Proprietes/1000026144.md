
## DefaultFilePath (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DefaultFilePath** Property se utiliza para obtener y set el nombre completo del archivo (ruta y el nombre del archivo) propuesto por Default cuando un usuario selecciona la opción "Guardar como" en un procesador de textos o editor HTML control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WP_WPControl.DefaultFilePath = "C:\mydocs\doc.docx"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificando el nombre completo del archivo propuesto por Default

`<Result> = <Control used>.DefaultFilePath`
---

**`<Result>: (Cadena de caracteres)`**

Nombre completo del archivo (ruta y nombre del archivo) propuesto por Default cuando un usuario selecciona la opción "Guardar como".. 

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Procesador de texto. 

- Control Editor HTML. 





<a name="SYNTAX2"></a>

### Modificar el nombre completo del archivo propuesto por Default

`<Control used>.DefaultFilePath = <Default path>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Procesador de texto. 

- Control Editor HTML. 




**`<Default path>: (Cadena de caracteres)`**

Nombre completo del archivo (ruta y nombre del archivo) propuesto por Default cuando un usuario selecciona la opción "Guardar como"..

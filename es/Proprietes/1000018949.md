


## Count (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**Conde** palabra clave puede corresponder a: 

- la función **[Conde](../WDLang1/3025009.md)**. Esta función devuelve el número de apariciones de una control en una ventana, página o grupo de control s.

- el **Count** Property, que devuelve un número de elementos. Esta propiedad se puede utilizar en:

	- los mandos de una ventana o Page. Para obtener más información, consulte [Conde](../Proprietes/2510097.md).

	- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

	- una variable de tipo [JSON](../Motscles/1000023611.md). 

	- una variable de tipo [YAML](../Motscles/1000024438.md).

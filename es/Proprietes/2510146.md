


## AutoRefresh (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AutoRefresh** Property se utiliza para:

- Averigüe si los datos que se encuentran en la serie se recalculan automáticamente cada vez que se llama a [grDraw](../WDLang3/3042026.md).

- Modificar el modo para recalcular los datos encontrados en la serie.




Observación: Esta Property es equivalente a la [grAutoRefreshSeries](../WDLang3/3042066.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Automatically recalculate the chart for series 2
CHART_MyChart[2].AutoRefresh = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de cálculo de una serie

`<Result> = <Series used>.AutoRefresh`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los datos de la serie vuelven a calcularse automáticamente al llamar a la función [grDraw](../WDLang3/3042026.md),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Series used>: (Cadena de caracteres)`**

Series a utilizar. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.  


<a name="SYNTAX2"></a>

### Modificar el modo de cálculo de una serie

`<Series used>.AutoRefresh = <New mode>`
---

**`<Series used>: (Cadena de caracteres)`**

Series a utilizar. Utilice la siguiente notación: *&lt;Nombre del gráfico&gt;[&lt;Número de serie&gt;]*.

**`<New mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para que la serie vuelva a calcularse automáticamente cada vez que se llama a la función [grDraw](../WDLang3/3042026.md),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## ImageState (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ImageState** Property se utiliza para get y cambiar el número de dibujos en:

- La imagen associated con un botón.

- La Image del cursor en un deslizador.

- La imagen associated con el botón de un panel lateral.

- La imagen associated con el botón de un panel de pestañas.

- La imagen associated con el control Valoración.

- La imagen associated con un grupo en un control Cinta de opciones.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen associated con una opción de menú.




En los botones de radio y las casillas de verificación, el **ImageState** Property se utiliza para get el número de pasos de la animación. 

Para las imágenes en un formato distinto al GIF animado, la animación se define mediante un tablero de dibujo.. Los pasos de la animación corresponden al número de líneas de dibujo de la imagen set.  

**Observación**: Cada dibujo de Image corresponde a un estado:

- Para controles de botones, panel de la barra lateral, panel de pestañas, opción de menú, etc..: los dibujos deben seguir un orden específico: Normal, Pressed, Grayed, Focus y Rollover.

- Para los controles de Control deslizante: los dibujos deben seguir este orden: Normal y Prensado.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the image associated with the "BTN_Help" button
BTN_Help.Image = "C:\MyImages\Help.bmp"
// Initialize the number of drawings in the image associated with "BTN_Help"
BTN_Help.ImageState = 3
```

## Sintaxis

### Averiguar el número de dibujos que se encuentran en la imagen associated con un control

`<Current number of drawings> = <Control used>.ImageState`
---

**`<Current number of drawings>: (Integro)`**

Número de dibujos encontrados en:

- La imagen associated con el botón. Este número está entre 1 y 5.

- La Image del cursor en un deslizador. Este número está entre 1 y 2.

- La imagen associated con el botón de un panel lateral. Este número está entre 1 y 5.

- La imagen associated con el botón de un panel de pestañas. Este número está entre 1 y 5.

- La imagen associated con el control Valoración. Este número está entre 1 y 5.  

- La imagen associated con un grupo en un control Cinta de opciones. Este número está entre 1 y 5.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen associated con una opción de menú. Este número está entre 1 y 5.




Para los controles Casilla de verificación y Botón de opción, número de pasos de animación.

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificando el número de dibujos encontrados en la imagen associated con un control

`<Control used>.ImageState = <Number of Drawings>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Number of Drawings>: (Integro)`**

Nuevo número de dibujos encontrados en:

- La imagen associated con el botón. Este número debe estar entre 1 y 5.

- La Image del cursor en un deslizador. Este número debe estar entre 1 y 2.

- La imagen associated con el botón de un panel lateral. Este número debe estar entre 1 y 5.

- La imagen associated con el botón de un panel de pestañas. Este número está entre 1 y 5. 

- La imagen associated con el control Valoración. Este número debe estar entre 1 y 5.  

- La imagen associated con un grupo en un control Cinta de opciones. Este número debe estar entre 1 y 5.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La imagen associated con una opción de menú. Este número debe estar entre 1 y 5.




Para los controles Casilla de verificación y Botón de opción, nuevo número de pasos de animación.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **ImageState** Property se aplica únicamente a: 

- botones,

- deslizadores,

- paneles de la barra lateral,

- los paneles de pestañas,

- casillas de verificación y botones de radio,

- Controles de clasificación.

- grupos en una control Cinta de opciones.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) opciones de menú.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **ImageState** Property se aplica sólo a los botones.

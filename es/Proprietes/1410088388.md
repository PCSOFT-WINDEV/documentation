
## spIterationCount (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spIterationCount** permite:

- Obtener el número de iteraciones del algoritmo hash utilizado para un campo de tipo "Contraseña". Esta propiedad puede utilizarse para campos de tipo "Contraseña" definidos en el editor de análisis o mediante programación. 

- Establecer el número de iteraciones del algoritmo hash utilizado para un campo de tipo "Contraseña" (solo para campos de tipo "Contraseña" definidos mediante programación). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile file
...

// Describe the "Password" item
MyItem is Item Description
MyItem.Name = "Password"
MyItem.Type = hItemSecurePassword
MyItem.spDerivation = SP_PBKDF2
MyItem.spHash = SP_SHA2_256
MyItem.spIterationCount = 20000
MyItem.spLength = 256

// Validate the description of the "Password" item
HDescribeItem(AFile, MyItem)

// Validate the description of AFile file
...
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el número de iteraciones del algoritmo hash utilizado para un campo de tipo "Contraseña"

`<Iterations> = <Data file>.<Item>.spIterationCount`
---

**`<Iterations>: (Entero)`**

Número de iteraciones del algoritmo hash utilizado.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Establecer el número de iteraciones del algoritmo hash utilizado para un campo de tipo "Contraseña" (definido mediante programación)

`<Item>.spIterationCount = <Iterations>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Item Description](../WDLang4/1514071.md).

**`<Iterations>: (Entero)`**

Número de iteraciones del algoritmo hash utilizado. Se recomienda establecer un valor entre 10 000 y 30 000

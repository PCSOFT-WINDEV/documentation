
## DateModified (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DateModified** Property se utiliza para get la fecha de modificación de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [gglContact](../WDLang5/1000017434.md). 

- una variable de tipo [zipArchivedFile](../WDLang3/1000019044.md).

- una variable de tipo [hdfsStatus](../WDLang4/1000021973.md).

- una variable de tipo [pdfDocument](../WDLang6/1000024909.md).

- una variable de tipo [pdfAttachment](../WDLang6/1000024899.md).

- una variable de tipo [DiskFile](../WDLang1/1410088883.md).

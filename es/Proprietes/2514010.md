


## Propiedades de gestión de fechas y horas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes propiedades se utilizan para gestionar fechas y horas:



|   |   |
| --- | --- |
| [Date](../Proprietes/2512065.md) | El **Fecha** Property se utiliza para:<br><br>- Recuperar la fecha de un [DateTime](../Motscles/1514070.md) Variable.<br><br>- Modificar la fecha en un [DateTime](../Motscles/1514070.md) Variable.<br><br>- Recuperar la fecha de un elemento Date (en formato "Date and Time") .<br><br>- Modificar la fecha en un elemento Date (en formato "Date and Time")<br><br><br> |
| [Day](../Proprietes/2512043.md) | El **Day** Property se utiliza para:<br><br>- Obtiene el día de una [Fecha](../Motscles/1514066.md) o [Hora](../Motscles/1514070.md) Variable, u obtener el número de días de una [Duración](../Motscles/1514069.md) Variable.<br><br>- Cambiar el día de una [Fecha](../Motscles/1514066.md) o [Hora](../Motscles/1514070.md) Variable, o cambiar el número de días de una [Duración](../Motscles/1514069.md) Variable.<br><br>- get el día o el número de días de un elemento de fecha (en formato "Fecha", "Fecha y hora" o "Duración")<br><br>- Modificar el día o el número de días de un elemento Date (en formato "Date", "Date and Time" o "Duration").<br><br><br> |
| [Hour](../Proprietes/2512038.md) | La propiedad **Hour** permite:<br><br>- Obtiene la hora de un Variable de tipo [Hora](../Motscles/1514068.md) o [HoraFecha](../Motscles/1514070.md), o el número de horas de un Variable de tipo [Duración](../Motscles/1514069.md).<br><br>- Cambia la hora en un Variable de tipo [Hora](../Motscles/1514068.md) o [HoraFecha](../Motscles/1514070.md), o el número de horas en un Variable de tipo [Duración](../Motscles/1514069.md).<br><br>- Obtener la hora o el número de horas de un campo de tipo [Date](../Motscles/1514066.md) (en formato "Fecha y hora" o "Duración) o de un campo de tipo [Time](../Motscles/1514068.md).<br><br>- Modificar la hora o el número de horas de un campo de tipo [Date](../Motscles/1514066.md) (en formato "Fecha y hora" o "Duración") o de un campo de tipo [Time](../Motscles/1514068.md).<br><br><br> |
| [InDays](../Proprietes/1000017295.md) | El **InDays** Property se utiliza para convertir una Duración Variable a días. |
| [InHours](../Proprietes/1000017296.md) | El **InHours** Property se utiliza para convertir una Duración Variable en horas. |
| [InMilliseconds](../Proprietes/1000017299.md) | El **InMilliseconds** Property se utiliza para convertir una Duración Variable a milisegundos. |
| [InMinutes](../Proprietes/1000017297.md) | El **InMinutes** Property se utiliza para convertir una Duración Variable en minutos. |
| [InSeconds](../Proprietes/1000017298.md) | El **InSeconds** Property se utiliza para convertir una Duración Variable a segundos. |
| [Millisecond](../Proprietes/2512062.md) | El **Millisecond** Property se utiliza para:<br><br>- Recupera el número de milisegundos de una [Hora](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.<br><br>- Modificar el número de milisegundos en una [Hora](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.<br><br>- Recuperar los milisegundos o el número de milisegundos de un elemento Date (en formato "Date and Time" o "Duration")<br><br>- Modificar los milisegundos o el número de milisegundos de un elemento Date (formato"Date and Time" o"Duration")<br><br><br> |
| [Minute](../Proprietes/2512050.md) | El **Minute** Property se utiliza para:<br><br>- Recupera los minutos de una [Hora](../Motscles/1514068.md) o [HoraFecha](../Motscles/1514070.md) Variable, o el número de minutos de una [Duración](../Motscles/1514069.md).<br><br>- Cambiar los minutos en una [Hora](../Motscles/1514068.md) o [HoraFecha](../Motscles/1514070.md) Variable, o el número de minutos en una [Duración](../Motscles/1514069.md).<br><br>- Recuperar los minutos o el número de minutos de un elemento Fecha (en formato "Fecha y hora" o "Duración") o de un elemento Hora..<br><br>- Modificar los minutos o el número de minutos en un ítem Fecha (en formato "Fecha y hora" o "Duración") o en un ítem Hora..<br><br><br> |
| [Month](../Proprietes/2512052.md) | El **Mes** Property se utiliza para:<br><br>- Recuperar el mes de una [Fecha](../Motscles/1514066.md) o [Hora](../Motscles/1514070.md) Variable.<br><br>- Modificar el mes en una [Fecha](../Motscles/1514066.md) o [Hora](../Motscles/1514070.md) Variable.<br><br>- Obtener el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora")<br><br>- Modificar el mes de un campo de tipo Fecha (en formato "Fecha simple" o "Fecha y Hora")<br><br><br> |
| [Second](../Proprietes/2512081.md) | El **Segundo** Property se utiliza para:<br><br>- Recupera el número de segundos de una [Hora](../Motscles/1514068.md), [Fecha](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.<br><br>- Modificar el número de segundos de una [Hora](../Motscles/1514068.md), [Fecha](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) Variable.<br><br>- Recuperar los segundos o el número de segundos de un elemento Date (en formato "Date and Time" o "Duration")<br><br>- Modificar los segundos o el número de segundos en un elemento Date (en formato "Date and Time" o "Duration")<br><br><br> |
| [Time](../Proprietes/2512069.md) | El **Time** Property le permite:<br><br>- Recuperar la hora de un [DateTime](../Motscles/1514070.md) Variable.<br><br>- Modificar la hora en un [DateTime](../Motscles/1514070.md) Variable.<br><br>- Recuperar la hora de un elemento Date (en formato "Date and Time")<br><br>- Modificar la hora en un elemento Date (formato"Date and Time").<br><br><br> |
| [Year](../Proprietes/2512095.md) | La propiedad **Year** permite:<br><br>- Obtener el año de una variable o campo de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md).<br><br>- Cambiar el año en una variable o campo de tipo [Date](../Motscles/1514066.md) o [DateTime](../Motscles/1514070.md).<br><br><br> |

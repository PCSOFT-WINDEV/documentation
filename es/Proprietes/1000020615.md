<a name="Use"></a>
<a name="description"></a>
"FIN" puede corresponder a: 

- la etiqueta "END:", que se utiliza para definir un código que se ejecutará sistemáticamente al final de la ejecución de una Process. Para obtener más información, consulte [Etiqueta "END:".](../Motscles/1510014.md). 

- la palabra clave que indica el final de una sentencia compuesta. Para obtener más información, consulte [Declaraciones compuestas](../Motscles/1510006.md). 

- la palabra clave que indica el final de un Procedure interno. Para obtener más información, consulte [Procedimiento interno](../Motscles/1514075.md). 

- el **End** Property, que se utiliza para identificar el final de un elemento. Esta propiedad se puede utilizar en:

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [EventTimeline](../WDLang1/1000020612.md). 

	- una variable de tipo [edtHTMLSelection](../WDLang1/1000026093.md).


## TitleBackgroundColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **TitleBackgroundColor** permite obtener y cambiar el color de fondo del título de una columna de un control Tabla o Tabla dinámica. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF bVIPCustomer THEN
	COL_Note.TitleBackgroundColor = LightRed
ELSE
	COL_Note.TitleBackgroundColor = DefaultColor
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el fondo Color de un título de columna

`<Background color> = <Column used>.TitleBackgroundColor`
---

**`<Background color>: (Entero o constante)`**

Fondo Color del título de la columna. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).




**`<Column used>: (Nombre del control)`**

Nombre de una columna que se encuentra en una Tabla o control Tabla TreeView. 


<a name="SYNTAX2"></a>

### Modificando el fondo Color de un título de columna

`<Column used>.TitleBackgroundColor = <New background color>`
---

**`<Column used>: (Nombre del control)`**

Nombre de una columna que se encuentra en una Tabla o control Tabla TreeView. 

**`<New background color>: (Entero o constante)`**

Nuevo fondo Color para el título de la columna. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md), 

- la constante *ColorDefault*. En este caso, el fondo Color corresponde al fondo Color común a todos los títulos de columna (definido en el estilo control). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el fondo del título de la columna es una imagen, el fondo del título está sobrecargado con esta Color.

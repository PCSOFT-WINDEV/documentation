


## ScreenCastInProgress (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ScreenCastInProgress** Property se utiliza para determinar si el vídeo reproducido en un control Multimedia se está reproduciendo en un dispositivo externo (compatible con Air Play). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF MM_Multi.ScreenCastInProgress = True THEN
	// Code if the video is being played on an external device (compatible with Air Play)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Multimedia control>.ScreenCastInProgress`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el vídeo se está visualizando en un dispositivo externo, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.

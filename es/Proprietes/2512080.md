


## TextItemCompleted (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **TextItemCompleted** Property se utiliza para gestionar los elementos de texto en un archivo de datos HFSQL. Las posiciones de texto pueden ser:

- se rellenan automáticamente con caracteres espaciados (mismo modo de funcionamiento que el utilizado en WINDEV 5.5),

- no rellenado con espacios en blanco.




Es posible:

- Averigüe si un archivo de datos está configurado para rellenar los elementos de texto (archivo de datos definido en el editor de análisis o mediante programación).

- Definir las características de los elementos de texto (sólo cuando se describe el archivo de datos mediante programación).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Characteristics of text items in the Orders data file
Orders.TextItemCompleted = True
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si los elementos de texto se llenarán con caracteres de espacio

`<Filled items> = <Data file>.TextItemCompleted`
---

**`<Filled items>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos está configurado para completar los elementos de texto con espacios,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definición del modo de gestión de los elementos de texto (archivo creado mediante programación)

`<Data file>.TextItemCompleted = <Filled items>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<Filled items>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos debe estar configurado para completar los elementos de texto con espacios,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## CalculatorButton (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **CalculatorButton** permite determinar e indicar si un control Campo de entrada de tipo Numérico o Monetario tiene un botón para mostrar una calculadora emergente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BoutonCalculatrice%20-%20HC%20N%B0001.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on BTN_ENABLE_CALCULATOR
EDT_Price.CalculatorButton = True
EDT_Discount.CalculatorButton = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si se muestra un botón Calculadora

`<Result> = <Edit control>.CalculatorButton`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se muestra el botón Calculadora. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Edit control>: (Nombre del control)`**

Nombre del control Campo de entrada de tipo Numérico o Monetario a utilizar. 


<a name="SYNTAX2"></a>

### Mostrar un botón Calculadora

`<Edit control>.CalculatorButton = <Display>`
---

**`<Edit control>: (Nombre del control)`**

Nombre del control Campo de entrada de tipo Numérico o Monetario a utilizar. 

**`<Display>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si debe mostrarse el botón Calculadora. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento de la calculadora
<a name="modo_funcionamiento_calculadora_ELTPARAGRAPHE000056"></a>

En tiempo de ejecución, los controles Campos de entrada de tipo Numérico y Monetario incluyen la opción "Calculadora" en el menú contextual. Esta opción muestra una calculadora para realizar un cálculo. 

Si se utiliza la propiedad **CalculatorButton** para mostrar un botón Calculadora en un control Campo de entrada de tipo Numérico o Monetario, la opción "Calculadora" no aparece en el menú contextual del control. 

**Observación**: Para desactivar la calculadora (botón u opción del menú contextual) en un proyecto o control, utilice la función [AAFDisable](../WDLang1/1000022018.md).




## CalculatedItemExpression (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CalculatedItemExpression** Property se utiliza para:

- Averiguar la expresión SQL que define el valor de un ítem de expresión (o ítem calculado). Este punto se definió en la editor de análisis o a través de la programación.

- Definir la expresión SQL que da el valor de un elemento de expresión (cuando se describe el elemento a través de la programación).




![Conectores Nativos](https://doc.pcsoft.fr/ext/images/us/AN.png) Este Property está disponible para el conector nativo de MySQL. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AFile is File Description
AnItem is Item Description
AnItem.Name = "CmdDate"
AnItem.CalculatedItemExpression = "'Written by ' + CURRENT_USER + ' on ' + SYSDATE"
AnItem.PersistedCalculatedItem = True
HDescribeItem(AFile, AnItem)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Encontrar la expresión SQL que define el valor de un elemento calculado

`Result> = <Data file>.<Item>.CalculatedItemExpression`
---

**`<Result>: (Cadena de caracteres)`**

Expresión SQL que define el valor del elemento.

**`<Data file used>: (Nombre del archivo)`**

Nombre del archivo de datos utilizado.

**`<Item used>: (Nombre del artículo)`**

Nombre del artículo utilizado.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Definir la expresión SQL que devuelve el valor de un elemento calculado

`<Item used>.CalculatedItemExpression = <SQL expression>`
---

**`<Item used>: (Nombre del artículo)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<SQL expression>: (Cadena de caracteres)`**

Expresión SQL que define el valor del elemento.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Un ítem de expresión (o un ítem calculado) es un ítem cuyo contenido se calcula de acuerdo a la expresión SQL asociada y a los valores de los ítems en la Record actual. Para obtener más información, consulte [Editor de análisis : los diferentes tipos de artículos](../Editeurs/2011026.md).

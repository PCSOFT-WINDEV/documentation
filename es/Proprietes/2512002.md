
## MinLinkedCardinality (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MinLinkedCardinality** Property se utiliza para set las características de un enlace. Es posible:

- Descubra la cardinalidad mínima del artículo relacionado en el enlace.

- Modificar la cardinalidad mínima del elemento relacionado en el enlace.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cardinaliteReliee.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the minimum cardinality
LinkClubPerson.MinLinkedCardinality = 1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la cardinalidad mínima del artículo de relacionado

`<Cardinality> = <Link>.MinLinkedCardinality`
---

**`<Cardinality>: (Integro)`**

Cardinalidad mínima de la pieza relacionado.

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en el editor de análisis o a través del tipo de [Descripción de la relación](../WDLang4/1514039.md).  


<a name="SYNTAX2"></a>

### Modificar la cardinalidad mínima del artículo relacionado

`<Link>.MinLinkedCardinality = <New cardinality>`
---

**`<Link>: (Cadena de caracteres)`**

Nombre del enlace definido en la editor de análisis o a través del tipo [Descripción de la relación](../WDLang4/1514039.md).

**`<New cardinality>: (Integro)`**

Nueva cardinalidad mínima para el artículo relacionado. En Default, cuando se describe un enlace mediante programación, la cardinalidad mínima del elemento relacionado es igual a 0.


## Event

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Event** puede corresponder a:  

- la función **[Event](../WDLang1/3015011.md)**, que se utiliza para interceptar eventos de Windows. 

- el **Evento** Property de un Variable de tipo [gglCalendar](../WDLang5/1000017418.md).

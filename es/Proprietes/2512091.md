


## SortType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **SortType** Property se utiliza para:

- Averiguar la dirección de clasificación definida para un artículo o un componente de una clave compuesta.. Este punto se definió en la editor de análisis o a través de la programación.

- Definir la dirección de clasificación de un artículo o de un componente de una clave compuesta (cuando el artículo se describe mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the sort direction for the LastName component of the LastNameFirstName composite key
Res is int
Res = Customer.LastNameFirstName[1].SortType
SWITCH Res
	CASE hIncreasing: Info("Ascending sort on LastName")
	CASE hDecreasing: Info("Descending sort on LastName")
	CASE hNotSorted: Info("No sort on LastName")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la dirección de clasificación definida para un artículo

`<Result> = <Data file>.<Item>.SortType`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de tipo: 



|   |   |
| --- | --- |
| *hDecreasing* | Orden descendente |
| *hIncreasing* | Orden ascendente |
| *hNotSorted* | Posición no clasificada |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. 
Si el elemento es un componente de una clave compuesta, &lt;Nombre del elemento&gt; corresponde a: ** &lt;Nombre de la clave compuesta&gt;[&lt;Índice del componente&gt;]**.
Ejemplo: ApellidoNombre[ 1].

### Definir la dirección de clasificación de un artículo descrito mediante programación

`<Item>.SortType = <Sort direction>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md). 
Si el elemento es un componente de una clave compuesta, &lt;Nombre del elemento&gt; corresponde a: ** &lt;Nombre de la clave compuesta&gt;[&lt;Índice del componente&gt;]**.
Ejemplo: ApellidoNombre[ 1].

**`<Sort direction>: (Constante de tipo Integer)`**

Tipo de tipo: 



|   |   |
| --- | --- |
| *hDecreasing* | Orden descendente |
| *hIncreasing* | Orden ascendente |
| *hNotSorted* | Posición no clasificada |

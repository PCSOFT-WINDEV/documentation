


## MinWidth (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MinWidth** Property se utiliza para get y cambiar el ancho mínimo de una control o ventana.

El ancho mínimo de los controles y ventanas es set a 0 por Default. Sin embargo, algunos controles no pueden ser redimensionados por debajo del tamaño de su Border.

**Recordatorio**: La anchura mínima de una control o ventana se utiliza durante las siguientes operaciones:

- Ventana redimensionada por el usuario.

- Manejo de anclas.

- Ventana redimensionada mediante programación (con la función [anchura](../Proprietes/2510059.md) Property o la función [WinSize](../WDLang1/3038020.md)).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Resize a column (2 pixels)
COL_Column1.MinWidth = 2
COL_Column1.Width = 2
```


<a name="Example2"></a>

```wl
// Modify the minimum width of a control
EDT_Edit1.MinWidth = 200
EDT_Edit1.Width = 150
// In fact, the width of control will be equal to 200
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la anchura mínima de un elemento

`<Current minimum width> = <Element used>.MinWidth`
---

**`<Current minimum width>: (Integro)`**

Anchura mínima del elemento especificado (incluyendo la Border) expresada en píxeles.

**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento (ventana o control) utilizado.  


<a name="SYNTAX2"></a>

### Modificar la anchura mínima de un elemento

`<Element used>.MinWidth = <New minimum width>`
---

**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento (ventana o control) utilizado.

**`<New minimum width>: (Integro)`**

Nueva anchura mínima del elemento (incluyendo la Border) expresada en píxeles.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Ancho inicial (control y ventana)
<a name="ancho_inicial_control_ventana_ELTPARAGRAPHE000093"></a>

La anchura inicial (definida en el editor) es devuelta por la [InitialWidth](../Proprietes/2510067.md) Property.

En la mayoría de los casos, la anchura de un elemento depende de las propiedades [MaxWidth](../Proprietes/2510068.md) y **MinWidth**. Sin embargo, esta restricción deja de aplicarse cuando se modifica la anchura mediante programación (con la función [anchura](../Proprietes/2510059.md) Property o la función [WinSize](../WDLang1/3038020.md)).
<a name="NOTE0_2"></a>


### Ancho de una columna en un control Tabla
<a name="ancho_una_columna_control_tabla_ELTPARAGRAPHE000116"></a>

Cuando se modifica la anchura de una columna en un control Tabla (con el [anchura](../Proprietes/2510059.md) Property), si la anchura especificada es inferior a la anchura mínima, ésta debe modificarse (con el **MinWidth** Property) para reflejar el cambio.
<a name="NOTE0_4"></a>
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Límite
<a name="limite_ELTPARAGRAPHE000130"></a>

El **MinWidth** Property se aplica únicamente a los siguientes elementos:

- Botón.

- Control Estático.

- Control Campo de entrada.

- Image.

- Casilla de verificación.

- Botón de opción.

- List Box.

- Combo Box.

- TreeView.

- Pestaña.

- Mesa.

- Columna de tabla.

- Supercontrol.

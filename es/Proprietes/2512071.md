


## PasswordProtected (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **PasswordProtected** Property se utiliza para determinar si un archivo de datos está protegido por contraseña.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out whether the Customer data file is password-protected 
ResProtection = Customer.PasswordProtected
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Protected> = <Data file>.PasswordProtected`
---

**`<Protected>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo de datos está protegido por una contraseña, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este archivo de datos ha sido descrito en la editor de análisis.




## TabMDI (Property)

*** ***
	

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20 (57)</B>, se recomienda utilizar el [DynamicTab](../Proprietes/1000021626.md) Property. 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **TabMDI** Property se utiliza para: 

- get el tipo de control Pestaña (pestaña simple o pestaña dinámica - MDI). 

- Cambiar el tipo de control Pestaña (pestaña simple o pestaña dinámica - MDI). 




**Recordatorio**: Una control Pestaña dinámica permite añadir, suprimir y desplazar paneles.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the tab is an MDI tab
IF TAB_Tab1.TabMDI = True THEN
	// Close all the panes
	TabCloseAll(TAB_Tab1)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si un control Pestaña es una ficha MDI (dinámica)

`<Result> = <Tab control>.TabMDI`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Pestaña es una pestaña MDI (dinámica), 

- <u><u><u><u>False</u></u></u></u> si el control Pestaña es una simple pestaña. 




**`<Tab control>: (Nombre del control)`**

Nombre del control Pestaña a utilizar. 


<a name="SYNTAX2"></a>

### Cambiar el tipo de control Pestaña

`<Tab control>.TabMDI = <Type of Tab control>`
---

**`<Tab control>: (Nombre del control)`**

Nombre del control Pestaña a utilizar. 

**`<Type of Tab control>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para obtener un MDI control Pestaña (dinámico), 

- <u><u><u><u>False</u></u></u></u> para obtener un simple control Pestaña. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### MDI control Pestaña
<a name="mdi_control_pestana_ELTPARAGRAPHE000066"></a>

Si la propiedad **TabMDI** se establece como <u><u><u><u>True</u></u></u></u>: 

- las funciones de [TabXXX](../WDLang1/1000021391.md) pueden llamarse en control.

- el operador ][ de la ficha debe llamarse con el alias del panel. 

- el usuario final puede: 

	- eliminar los paneles si [WithClosingButton](../Proprietes/1000021374.md) se establece en <u><u><u><u>True</u></u></u></u>. 

	- añadir paneles si [WithNewButton](../Proprietes/1000021377.md) se establece en <u><u><u><u>True</u></u></u></u>.

	- modificar el orden de los paneles mediante arrastrar y soltar. 

	- desacoplar los paneles si [UndockablePane](../Proprietes/1000021382.md) se establece en <u><u><u><u>True</u></u></u></u>.

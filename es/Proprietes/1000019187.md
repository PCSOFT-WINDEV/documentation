


## SecondaryAxis (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **SecondaryAxis** permite: 

- determinar si una serie del gráfico está en el eje secundario. 

- establecer la posición de una serie del gráfico en el eje secundario.




**Observación**: Esta propiedad equivale a [grSeriesSecondaryAxis](../WDLang3/1000019188.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Colors series 2 in CHART_MyChart if it is placed on the secondary axis
IF CHART_MyChart[2].SecondaryAxis = True THEN
	CHART_MyChart[2].SeriesColor = LightRed
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si una serie está en el eje secundario

`<Result> = <Series used>.SecondaryAxis`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la serie se encuentra en el eje secundario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Series used>: (Nombre del control)`**

Serie a utilizar. Utilice la siguiente notación de sintaxis: 

```txt
<Chart name>[<Series number>]
```



<a name="SYNTAX2"></a>

### Cambiar la posición de la serie

`<Series used>.SecondaryAxis = <New position>`
---

**`<Series used>: (Nombre del control)`**

Serie cuya posición debe ser modificada. Utilice la siguiente notación de sintaxis: 

```txt
<Chart name>[<Series number>]
```


**`<New position>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> para establecer la posición de la serie en el eje secundario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta propiedad solo está disponible para gráficos con ejes. No se puede utilizar en gráficos circulares o radiales, por ejemplo.

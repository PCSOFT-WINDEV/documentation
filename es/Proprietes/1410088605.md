
## CheckMarkStyle (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CheckMarkStyle** Property obtiene y establece el check Icon de una opción en un menú principal o contextual. Esta Icon sólo aparece si la opción de menú está marcada. 

Con esta Property, puede obtener grupos de opciones con un comportamiento similar al de los botones de radio en los menús de Windows 11. Esta Property puede utilizarse tanto si el menú utiliza el lenguaje de diseño de Windows 11 como si no. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OPTION_Top.CheckMarkStyle = csCircle
OPTION_Middle.CheckMarkStyle = csCircle 
OPTION_Bottom.CheckMarkStyle = csCircle 
OPTION_Middle.Checked = True
```

<a name="XSYNTAX"></a>

## Sintaxis

### Obtención del check Icon de un menú de opciones

`<Result> = <Element to use>.CheckMarkStyle`
---

**`<Result>: (Constante de tipo Integer)`**

Estilo de la marca de verificación: 



|   |   |
| --- | --- |
| *csCircle* | Cheque redondo Icon. |
| *csDefault* | Marca de verificación Default. |



**`<Element to use>: (Nombre del control)`**

Nombre de la opción de menú.


<a name="SYNTAX2"></a>

### Modificación de la comprobación Icon de un menú de opciones

`<Element to use>.CheckMarkStyle = <new value>`
---

**`<Element to use>: (Nombre del control)`**

Nombre de la opción de menú.

**`<new value>: ()`**

Estilo que se aplicará al cheque Icon: 


|   |   |
| --- | --- |
| *csCircle* | Cheque redondo Icon. |
| *csDefault* | Marca de verificación Default. |

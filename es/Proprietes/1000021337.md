


## InputFormula (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La **InputFormula** Property se utiliza para: 

- averiguar si el usuario final puede introducir o modificar fórmulas en un control Hoja de cálculo. 

- permitir o impedir que el usuario final introduzca o modifique fórmulas en un control Hoja de cálculo.




Esta Property impide que el usuario final ejecute código..

Observación: Esta propiedad corresponde a la opción "Permitir entrada de fórmulas" de la pestaña "Detalles" de la ventana control Hoja de cálculo description.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CBOX_ReadOnly THEN
	PSHEET_Spreadsheet.InputFormula = True
ELSE
	PSHEET_Spreadsheet.InputFormula = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar si el usuario puede modificar o añadir fórmulas

`<Result> = <Spreadsheet control>.InputFormula`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario puede modificar y agregar fórmulas o cambiar un archivo XLSX, 

- <u><u><u><u>False</u></u></u></u> si el usuario no puede modificar ni agregar fórmulas o cambiar un archivo XLSX.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


<a name="SYNTAX2"></a>

### Permitiendo al usuario modificar o añadir fórmulas

`<Spreadsheet control>.InputFormula = <Authorization>`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Authorization>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para permitir al usuario cambiar/añadir fórmulas o cargar un archivo XLSX. 

- <u><u><u><u>False</u></u></u></u> para evitar que el usuario cambie/añada fórmulas o cargue un archivo XLSX.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el **InputFormula** Property es set a <u><u><u><u>False</u></u></u></u> y el [InputValue](../Proprietes/1000021336.md) Property es set a <u><u><u><u>True</u></u></u></u>, el usuario final podrá introducir y modificar valores en el control Hoja de cálculo.

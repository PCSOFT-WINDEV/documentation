
## HTMLAttribute (Propiedad)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **HTMLAttribute** lee y cambia los atributos HTML del control.

**Observaciones**: 

- Los atributos HTML del control cambian su comportamiento para almacenar información adicional específica (atributos "data-"), agregar información de accesibilidad (atributos "aria-"), etc.
	En tiempo de ejecución, los valores iniciales de la propiedad son los atributos definidos en modo de edición.

- En el editor, los atributos HTML se definen en la pestaña "Avanzado" de la ventana de descripción del control. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disables the spelling checker on the control
EDT_Edit.HTMLAttribute["spellcheck"] = "false"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Leer el valor del atributo HTML

`<Result> = <Control>.HTMLAttribute[<Attribute>]`
---

**`<Result>: (Tipo esperado)`**

Valor del atributo.

**`<Control>: (Nombre del control)`**

Nombre del control utilizado. 

**`<Attribute>: (Cadena de caracteres)`**

Nombre del atributo cuyo valor se desea leer.


<a name="SYNTAX2"></a>

### Cambiar el valor del atributo HTML

`<Control>.HTMLAttribute[<Attribute>] = <Value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. 

**`<Attribute>: (Cadena de caracteres)`**

Nombre del atributo cuyo valor se desea cambiar.

**`<Control>: (Nombre del control)`**

Nombre del control utilizado. 

**`<Value>: (Tipo del atributo)`**

Nuevo valor del atributo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los nombres y valores se codifican correctamente antes de escribirse en la página.

- Si se agrega un atributo en el editor de WEBDEV (para asegurarse de que el control funciona correctamente): ambos se incluirán en el código HTML: tendrá prioridad el que se definió con la propiedad **HTMLAttribute**. Sin embargo, el código HTML resultante no se validará y es posible que la página no se muestre correctamente en el navegador.

- Los nombres y valores no se comprueban. Si se utiliza un nombre o valor no válido, es posible que la página no se muestre correctamente en el navegador.

- Para eliminar un atributo, utilice la siguiente sintaxis: 
	
	```txt
	<Control>.HTMLAttribute[<Attribute name>].Delete()
	```


- Para definir un atributo sin valor, utilice la siguiente sintaxis: 
	```txt
	<Control>.HTMLAttribute[<Attribute name>] = Null
	```

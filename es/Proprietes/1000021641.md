


## NbWorksheet (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **NbWorksheet** permite obtener el número de hojas de trabajo con un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Browse the wooksheets found in PSHEET_MySpreadsheet
FOR I = 1 TO PSHEET_MySpreadsheet.NbWorksheet
	PSHEET_MySpreadsheet.CurrentWorksheet = I
	PSHEET_MySpreadsheet["A1"] = "On the worksheet " + I
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.NbWorksheet`
---

**`<Result>: (Integro)`**

Número de hojas de trabajo encontradas en el control Hoja de cálculo. 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El control Hoja de cálculo debe estar configurado para soportar múltiples hojas de trabajo, si es necesario: 

	- En el editor: En la pestaña "Detalles" de la ventana de control de description, marque "Permitir múltiples hojas de trabajo". 

	- A través de la programación, utilice el [MultiWorksheet](../Proprietes/1000021639.md) Property.




- Las hojas de trabajo pueden ser creadas: 

	- por el usuario final,

	- con [Hoja de cálculoAñadir Hoja de trabajo](../WDLang1/1000021642.md) o [SpreadsheetInsertWorksheet](../WDLang1/1000021648.md),

	- cargando un archivo xlsx que contiene varias hojas de trabajo ([Hoja de cálculoCarga](../WDLang1/1000021340.md)).

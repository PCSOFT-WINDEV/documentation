


## AutomaticTextColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **automatic TextColor** se utiliza para identificar y cambiar el Color que se utilizará para el texto de los elementos que utilizan el "automatic" Color. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to reverse video 
WIN_Overview.BackgroundColor = Black
WIN_Overview.AutomaticTextColor = White
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Descubrir el Color asociado al automatic Color

`<Result> = <Element used>.AutomaticTextColor`
---

**`<Result>: (Entero o constante de tipo Integer)`**

El color asociado con el automatic Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).




**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento a utilizar: ventana, ventana interna, supercontrol, control Pestaña, control Zona multilínea, control Cinta de opciones. 


<a name="SYNTAX2"></a>

### Modificando el Color asociado al automatic Color

`<Element used>.AutomaticTextColor = <New color>`
---

**`<Element used>: (El nombre de la ventana o el nombre de control)`**

Nombre del elemento a utilizar: ventana, ventana interna, supercontrol, control Pestaña, control Zona multilínea, control Cinta de opciones. 

**`<New color>: (Entero o constante de tipo Integer)`**

El color asociado con el automatic Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [y la preselección Color](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Al modificar el color automático, todos los elementos que usan el color "automático" en su estilo también se modifican. Los elementos que no utilizan esta Color no se modifican.

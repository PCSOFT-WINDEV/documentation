


## ExecutionCompleted (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ExecutionCompleted** permite saber si se completó la ejecución de una consulta o vista (HFSQL Classic o Client/Server).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF MyQuery.ExecutionCompleted = True THEN
	Info("The number of records found in the view is: " + HNbRec(hStateActive))
ELSE
	Info("The number of records already retrieved is: " + HNbRec(hNonBlocking))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Element used>.ExecutionCompleted`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la ejecución de la vista o consulta ha finalizado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Element used>: (Cadena de caracteres)`**

Nombre de la consulta o vista a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La propiedad **ExecutionCompleted** sólo puede utilizarse en vistas y consultas HFSQL Classic y Cliente/Servidor. 

- Para todos los demás elementos (archivos de datos, consultas que utilizan Conectores Nativos, etc.), el **ExecutionCompleted** Property devuelve siempre <u><u><u><u>True</u></u></u></u>.

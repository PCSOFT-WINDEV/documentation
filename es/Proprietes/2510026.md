


## MouseCursor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MouseCursor** Property se utiliza para identificar y cambiar el cursor hover definido para un control, ventana o página.

**Recordatorio**: Es posible definir:

- el cursor hover utilizado por un control o ventana en la pestaña "UI" del control o ventana description.

- el cursor de desplazamiento utilizado por un control de página o página en la pestaña "Detalles" de la ventana de control o página description.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the cursor of the BTN_Validate button
BTN_Validate.MouseCursor = curHelp
```

### Cursores preestablecidos
<a name="cursores_preestablecidos_ELTPARAGRAPHE000585"></a>

Los cursores preestablecidos corresponden a los cursores de Windows y pueden ser modificados a través del panel de control (o los temas de escritorio). Las ilustraciones corresponden a los valores de Default y pueden cambiar según las preferencias del usuario.

### Límites
<a name="limites_ELTPARAGRAPHE000629"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El **MouseCursor** Property se aplica únicamente a:

- Controles de las ventanas.

- Ventanas.

- Modelos de controles.




Esta Property no se puede usar con:

- Menús , 

- Filas de una tabla o control Tabla TreeView,

- Columnas de una tabla o control Tabla TreeView.

- Controles del mapa. 


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **MouseCursor** Property se aplica únicamente a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Control Tabla.

- Columna de control Tabla.

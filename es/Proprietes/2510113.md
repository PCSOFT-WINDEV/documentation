


## AnchorRateBottom (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **AnchorRateBottom** Property se utiliza para:

- Averiguar la tasa de anclaje cuando una control se mueve en el eje vertical

- Modificar la velocidad de anclaje cuando una control se mueve en el eje vertical.




La tasa de anclaje se utiliza para definir cómo se desplaza o redimensiona la control en relación con la evolución del tamaño de la ventana (o reporte) a la que está anclada. La tasa de anclaje corresponde al porcentaje en que se mueve el control en relación con la ventana (o reporte). Por lo tanto, si la tasa de anclaje es set a 0, el control no se redimensionará. Si es set a 1000, el control se redimensionará normalmente.

La tasa de anclaje al fondo (**AnchorRateBottom** Property) se aplica si el control está anclado al fondo (*anchorBottom* constante utilizada con el [Ancla](../Proprietes/2510001.md) Property).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The List Box control will be resized by 50% relative to the window size
LIST_ListPersons.AnchorRateBottom = 500
```

## Sintaxis

### Averiguar la velocidad de anclaje cuando un control anclado se mueve hacia el fondo

`<Result> = <Control used>.AnchorRateBottom`
---

**`<Result>: (Integro)`**

La tasa de anclaje incluida entre 0 (control no movido) y 1000 (ancla Normal).

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular. Este control puede ser un control: 

- de una ventana.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) de una reporte.





<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
### Modificar la velocidad de anclaje cuando un control anclado se mueve hacia el fondo

`<Control used>.AnchorRateBottom = <Anchor rate>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Anchor rate>: (Integro)`**

Nueva tasa de anclaje incluida entre 0 (control no movido) y 1000 (ancla Normal).

## Observaciones


### Anclar un control
<a name="anclar_control_ELTPARAGRAPHE000104"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En una ventana, el [Ancla](../Proprietes/2510001.md) Property se utiliza para determinar cómo se redimensiona o se mueve un control cuando se redimensiona la ventana. Si el control es associated con una pestaña, el ancla se define en relación con la pestaña.
	Para obtener más información, consulte [Anclaje de controles en una ventana](../WDChamp/2026002.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En un reporte, el [Ancla](../Proprietes/2510001.md) Property se utiliza para saber cómo se redimensiona o se mueve el control cuando se redimensiona el reporte. Este comportamiento fue definido en la editor de reportes. 
	Para obtener más información, consulte [Controles de anclaje en un reporte](../WDChamp/1013298.md).

### Límites
<a name="limites_ELTPARAGRAPHE000134"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **AnchorRateBottom** Property se aplica a todos los controles de una ventana.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El **AnchorRateBottom** Property se aplica a todos los controles de un reporte. En este caso, sólo se puede averiguar la tasa de anclaje utilizada.. 

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **AnchorRateBottom** Property se aplica sólo a los siguientes controles:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Control Tabla.

- Supercontrol.

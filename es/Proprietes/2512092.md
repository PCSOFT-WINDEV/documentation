


## DefaultValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DefaultValue** Property se utiliza para:

- Averiguar el valor Default de un artículo. Este punto se definió en la editor de análisis o a través de la programación.

- Definir el valor de Default de un artículo (artículo descrito sólo mediante programación).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the default value of the Orders.Title item
Title.DefaultValue = 1
...
// Validate the data file description
HDescribeFile(Orders)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el valor Default de un artículo

`<Result> = <Data file>.<Item>.DefaultValue`
---

**`<Result>: (Tipo de elemento)`**

Valor predeterminado del campo.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado.

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado.

### Definir el valor de Default de un elemento descrito mediante programación

`<Item>.DefaultValue = <Value>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Artículo description](../WDLang4/1514071.md).

**`<Value>: (Tipo de elemento)`**

Valor predeterminado del campo.

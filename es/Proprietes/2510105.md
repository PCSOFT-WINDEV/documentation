


## Multiline (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
En una ventana o página, el **Multiline** Property se utiliza para:

- determinar si una control Campo de entrada admite texto multilínea o hacerla compatible con la multilínea.

- determinar si una tabla control admite texto multilínea o hacer que una columna tabla sea compatible con multilínea. Para reflejar este cambio, actualización el control Tabla (con [TableDisplay](../WDLang1/3074003.md), por ejemplo) o rellena de nuevo el control Tabla.




En un reporte, el **Multiline** Property se utiliza para:

- determinar si una control admite texto multilínea

- modificar la opción "Multilínea" de una control.




**Observaciones**: 

- La opción "Multilínea" se utiliza para mostrar el contenido de una control en varias líneas sin truncar su contenido.

- En los informes, la opción multilínea se aplica a los controles Static, Item, Calculated y Link. Esta opción se encuentra en la pestaña "Detalles" del control description.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Does the "EDT_Address" control support multiline text?
ResMultiline = EDT_Address.Multiline
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si una control admite texto multilínea

`<Result> = <Control used>.Multiline`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado admite texto multilínea,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.  


<a name="SYNTAX2"></a>

### Modificación de la opción "Multilínea" de una control

`<Control used>.Multiline = <Is multiline?>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

**`<Is multiline?>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control especificado debe admitir texto multilínea,

- <u><u><u><u>False</u></u></u></u> si el control especificado no admite texto multilínea.

## Observaciones

### Controles multilínea en una reporte
<a name="controles_multilinea_una_reporte_ELTPARAGRAPHE000108"></a>

Si se aplica un ángulo de orientación a un control multilínea en un reporte, los retornos de carro del automatic son ignorados. Solo se tienen en cuenta los saltos de línea manuales.
<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000115"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En un reporte, el **Multiline** Property se aplica sólo a:

- Controles de estática,

- Controles de artículos,

- Controles calculados,

- controles Enlace.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, el **Multiline** Property se aplica sólo a:

- Controles campo de entrada.

- columnas en el controles Tabla.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **Multiline** Property sólo puede utilizarse con mandos de edición.


## Replication (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Replication** Property se utiliza para identificar y cambiar el modo de replicación. Esta propiedad se puede utilizar en:

- un archivo de datos. Para obtener más información, consulte [Replicación](../Proprietes/2512073.md).

- una variable de tipo [hServerOperation](../WDLang4/1000022471.md).

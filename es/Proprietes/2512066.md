


## DescribedName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DescribedName** Property se utiliza para manipular el nombre lógico de los archivos de datos HFSQL (ignorando los posibles cambios realizados por [HAlias](../WDLang4/3044176.md)). Es posible:

- Averigüe el nombre lógico de un elemento:

	- archivo de datos definido en el editor de análisis o mediante programación,

	- ver, 

	- interrogar




- Definir el nombre lógico de un archivo de datos (cuando el fichero se describe mediante programación).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the logical name of the Customer data file
ResName = Customer.DescribedName
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre lógico de un elemento

`<Logical name> = <Element>.DescribedName`
---

**`<Logical name>: (Cadena de caracteres)`**

Nombre lógico del elemento:

- archivo de datos definido en el editor de análisis o mediante programación,

- ver, 

- consulta.




**`<Element>: (Cadena de caracteres)`**

Elemento utilizado (por ejemplo, indirectamente).

### Definición del nombre lógico de un archivo de datos creado mediante programación

`<Data file>.DescribedName = <New name>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre se definió con el tipo [Expediente description](../WDLang4/1514065.md).

**`<New name>: (Cadena de caracteres)`**

Nombre lógico del elemento.

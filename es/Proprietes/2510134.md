


## ReturnedValue (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ReturnedValue** Property se utiliza para:

- get y cambiar el valor devuelto por una ventana, página o reporte.

- get y cambiar el valor devuelto por una opción en un control Botón de opción. 




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Recordatorio**: Una ventana abierta por [entablar](../WDLang1/3038035.md) puede devolver un resultado al cerrarse. Este valor es devuelto:

- por la palabra clave [Devuelve](../Motscles/1510007.md) en el código de cierre de la ventana,

- mediante la función [Cerrar](../WDLang1/3038018.md),
	**Observación**: [Cerrar](../WDLang1/3038018.md) devuelve múltiples valores.

- por el **ReturnedValue** property utilizado antes de cerrar la ventana.


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Se utiliza en la palabra clave [MiInfantilVentana](../Motscles/1511021.md), el **ReturnedValue** Property se utiliza para get el valor devuelto por una ventana hija en una aplicación móvil.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Recordatorio**: Una reporte impresa por [iPrintReport](../WDLang5/3046032.md) puede devolver un resultado al final del trabajo de impresión. Este valor es devuelto:

- por la palabra clave [Devuelve](../Motscles/1510007.md) en el código de cierre de reporte,

- por la propiedad **ReturnedValue** utilizada antes de cerrar el reporte.

<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Code used to open the "WIN_EditPassword" window
MyPassword = Open(WIN_EditPassword)
IF MyPassword <> "" THEN
	Info("Wrong password")
END

// --------------------------------------------------------------------------------
// -- Click code on the "OK" Button control in "WIN_EditPassword"
// The user types his password in the "EDT_Password" control
// and validates the window
Close()

// --------------------------------------------------------------------------------

// -- Closing code of the window
WIN_EditPassword.ReturnedValue = EDT_Password
```
<a name="Example2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Event "Closing a child window of WIN_List_Customers"
// -------------------------------------------------------------
// Refreshes the list of customers
LooperDisplay(LOOP_Customers, taReExecuteQuery)

// For any return (except for the deletion)
IF MyChildWindow.ReturnedValue <> -1 THEN
	nIndex is int
	// Finds the customer row
	nIndex = LooperSearch(ATT_CustomerNum, Customer.CustomerNum)
	IF nIndex > 0 THEN
		// Positions on the customer row
		LooperPosition(LOOP_Customers, nIndex)
	END
END
```

## Sintaxis

### Averiguar el valor devuelto

`<Result> = <Element used>.ReturnedValue`
---

**`<Result>: (Cualquier tipo)`**

Valor que será devuelto:

- al cerrar la ventana, página o reporte actual.

- si la opción está seleccionada en el botón de radio.




**`<Element used>: (Nombre de la ventana, página, reporte, opción)`**

Nombre del elemento utilizado:

- Nombre de la ventana, página, reporte o reporte interno utilizado. Esta ventana, página o reporte debe abrirse.

- Nombre de la opción en el botón de radio. Este nombre tiene el siguiente formato: &lt;control Botón de opción&gt;[&lt;Número de opción&gt;].  





<a name="SYNTAX2"></a>

### Modificando el valor devuelto

`<Element used>.ReturnedValue = <New value>`
---

**`<Element used>: (Nombre de la ventana, página, reporte, opción)`**

Nombre del elemento utilizado:

- Nombre de la ventana, página o reporte utilizada. Esta ventana, página o reporte debe abrirse.

- Nombre de la opción en el botón de radio. Este nombre tiene el siguiente formato: &lt;control Botón de opción&gt;[&lt;Número de opción&gt;].




**`<New value>: (Cualquier tipo)`**

Nuevo valor a devolver. Se pueden utilizar los siguientes tipos:

- estructura

- estructura dinámica

- class

- tipo avanzado

- array

- asociativa array

- queue

- stack

- listar

## Observaciones
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Descubrir el valor devuelto por una ventana secundaria
<a name="descubrir_valor_devuelto_por_una_ventana_secundaria_ELTPARAGRAPHE000204"></a>

En las aplicaciones móviles, las ventanas se abren con [OpenMobileWindow](../WDLang1/1000021018.md) (o [OpenChild](../WDLang1/3038021.md)) en lugar de [entablar](../WDLang1/3038035.md). 

Puede obtener el valor devuelto por una ventana hija en el Event "Cerrar una ventana hija" de la ventana que la abrió (la que llamó a [OpenMobileWindow](../WDLang1/1000021018.md) o [OpenChild](../WDLang1/3038021.md)).

Por ejemplo:

- En el código de cierre de la ventana hija, se utiliza [Cerrar](../WDLang1/3038018.md) para devolver un parámetro: 
	
	```wl
	Close("", MyParameter)
	```


- El nombre de la ventana hija, así como el valor devuelto, puede recuperarse en el Event "Cerrar una ventana hija" de la ventana que llama, utilizando la palabra clave [MiInfantilVentana](../Motscles/1511021.md): 
	
	```wl
	// Close a child window
	NameChildWindow = MyChildWindow.Name
	ValueChildWindow = MyChildWindow.ReturnedValue
	```

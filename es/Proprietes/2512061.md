


## NullSupported (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NullSupported** Property se utiliza para gestionar el valor NULL en los archivos de datos HFSQL.

Este Property está acostumbrado:

- Determine cómo se maneja el valor NULL para un archivo de datos descrito en el documento editor de análisis.

- Definir que el valor NULL se manejará cuando se describa un archivo de datos a través de la programación.




**Observación**: Si el archivo de datos y el elemento admiten valores NULL (propiedad [NullAllowed](../Proprietes/1000019492.md)), la propiedad [Null](../Proprietes/2512064.md) puede utilizarse para asignar NULL al elemento para el Record actual.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the management mode of the NULL value in the Customer data file
Customer.NullSupported = True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si los datos admiten el valor NULL

`<Result> = <Data file>.NullSupported`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo admite el valor NULL, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

### Definir cómo se tratará el valor NULL

`<Data file>.NullSupported = <Management mode>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre lógico del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<Management mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para administrar el valor NULL,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Caso especial: Ver y consultar
<a name="caso_especial_ver_consultar_ELTPARAGRAPHE000097"></a>

Cuando se utiliza en una vista o consulta, la propiedad **NullSupported** devuelve <u><u><u><u>True</u></u></u></u> si al menos uno de los archivos fuente admite el valor NULL.


### Casos de error
<a name="casos_error_ELTPARAGRAPHE000108"></a>

El acceso de escritura al [Null](../Proprietes/2512064.md) Property falla si el **NullSupported** Property es set a <u><u><u><u>False</u></u></u></u>.

El error 70409 ("Ha llamado a la función [HModify](../WDLang4/3044042.md). El archivo &lt;XXXX&gt; no soporta el acceso a la propiedad Null. La gestión de Null debe estar habilitada en la descripción del archivo".) ocurre si se cumplen las siguientes condiciones:

- un elemento encontrado en una consulta o en una vista es set a NULL,

- este ítem viene de un archivo que no soporta el valor NULL

- [HModify](../WDLang4/3044042.md) se utiliza en la consulta o en la vista.

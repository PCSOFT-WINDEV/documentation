
## Empty (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Empty** Property se utiliza para determinar si un elemento está vacío. Esta propiedad se puede utilizar en:

- los siguientes elementos: 

	- Controles de looper, controles de gráficos, etc.

	- arrays, colas, pilas. 


 Para obtener más información, consulte [Vacío](../Proprietes/2510126.md).

- una variable de tipo [xmlAttribute](../WDLang5/1000018799.md).

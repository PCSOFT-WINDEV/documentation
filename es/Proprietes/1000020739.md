
## Key (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Key** Property se utiliza para manipular llaves:

- en una variable de tipo [xmlSignature](../WDLang3/1000020736.md). 

- en una variable de tipo [memcachedResult](../WDLang4/1000023359.md).

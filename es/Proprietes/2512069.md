
## Time (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Time** Property le permite:

- Recuperar la hora de una [FechaHora](../Motscles/1514070.md) Variable.

- Modificar la hora en una [FechaHora](../Motscles/1514070.md) Variable.

- Recuperar la hora de un elemento Date (en formato "Date and Time").
	

- Modificar la hora en un elemento Date (formato"Date and Time").
	




**Recordatorio**: Los elementos de fecha se utilizan para gestionar:

- Fechas "Año - Mes - Día" (formato AAAAMMDD)

- "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" fechas y horas (formato AAAAMMDDHHmmSSCCC)

- "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" duraciones (formato +DHHMMSSCCC)







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Operations performed on a DateTime variable
StartDate is DateTime = "200112121515"
EndTime is Time = "1530"
// Change the time
StartDate.Time = "153050"// Modify the Time part
StartDate.Time = EndTime
```


<a name="Example2"></a>





```wl
// Operations performed on a DateTime item
Work.StartDate = "200112121515"
Work.EndTime = "1530"
// Change the time
Work.StartDate.Time = "153050"// Modify the Time part
Work.StartDate.Time = Work.EndTime
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la hora en un DateTime Variable

`<Result> = <DateTime variable>.Time`
---

**`<Result>: (Cadena de caracteres o tiempo)`**

Tiempo extraído del [FechaHora](../Motscles/1514070.md) Variable.

**`<DateTime variable>: (FechaHora)`**

Nombre de la variable de tipo [FechaHora](../Motscles/1514070.md) que se utilizará.


<a name="SYNTAX2"></a>

### Modificación de la hora en un DateTime Variable

`<DateTime variable>.Time = <New time>`
---

**`<DateTime variable>: (FechaHora)`**

Nombre de la variable de tipo [FechaHora](../Motscles/1514070.md) que se utilizará.

**`<New time>: (Cadena de caracteres o tiempo o entero)`**

Un nuevo tiempo que reemplaza al tiempo del Variable usado.

El formato de hora puede ser:

- Integro: número de centésimas de segundo pasadas desde medianoche "00:00:00:00:00"

- HHMM

- HHMMSS

- HHMMSSCC

- HHMMSSCCC




(HH: horas, MM: minutos, SS: segundos, CC: centésimas de segundo, CCC: milésimas de segundo).


<a name="SYNTAX3"></a>

### Averiguar la hora en un elemento de la Fecha (en formato "Fecha y hora")

`<Result> = <Data file>.<Item>.Time`
---

**`<Result>: (Cadena de caracteres o tiempo)`**

Tiempo extraído del [FechaHora](../Motscles/1514070.md) Variable.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX4"></a>

### Modificación de la hora en un elemento de la Fecha (en formato "Fecha y hora")

`<Data file>.<item>.Time = <New time>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<New time>: (Cadena de caracteres o tiempo o entero)`**

Nueva hora que sustituye a la hora del artículo utilizado. El formato de hora puede ser:

- Integro: número de centésimas de segundo pasadas desde medianoche "00:00:00:00:00"

- HHMM

- HHMMSS

- HHMMSSCC

- HHMMSSCCC




(HH: horas, MM: minutos, SS: segundos, CC: centésimas de segundo, CCC: milésimas de segundo).

## Observaciones




### Intervalo de fechas
<a name="intervalo_fechas_ELTPARAGRAPHE000175"></a>

Los elementos "Fecha" (en formato "Fecha y hora") se utilizan para gestionar las fechas encontradas entre el 01/01/0001 a las 00:00 y el 31/12/9999 a las 23:59.

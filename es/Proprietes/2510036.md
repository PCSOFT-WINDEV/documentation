


## Identifier (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Identifier** permite obtener y cambiar el nombre del equipo actual que se muestra a los participantes que reciben el flujo de audio o video. Este nombre puede ser un número, por ejemplo.

El identificador de la corriente es una cadena vacía de Default.

Observación: En la recepción, este nombre será devuelto por [StreamInfo](../WDLang3/3039009.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the name of the identifier
MyStream.Identifier = "JuliaPC"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención del identificador de flujo

`<Result> = <Stream name>.Identifier`
---

**`<Result>: (Cadena de caracteres)`**

Nombre con el que el ordenador actual aparece a sus corresponsales.

**`<Stream name>: (Cadena de caracteres)`**

Nombre del arroyo o nombre del control Conferencia associated con el arroyo.  


<a name="SYNTAX2"></a>

### Modificar el identificador de la corriente

`<Stream name>.Identifier = <New Name>`
---

**`<Stream name>: (Cadena de caracteres)`**

Nombre del arroyo o nombre del control Conferencia associated con el arroyo.

**`<New Name>: (Cadena de caracteres)`**

Nombre con el que el ordenador actual aparece a sus corresponsales.

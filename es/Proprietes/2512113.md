


## NbThumbnail (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **NbThumbnail** Property se utiliza para get el número de miniaturas generadas para un elemento. Este punto se definió en la editor de análisis o a través de la programación.

**Observación**: Las miniaturas HFSQL de un elemento se generan automáticamente:

- al escribir la correspondiente record ([HAdd](../WDLang4/3044147.md) o [HWrite](../WDLang4/3044092.md)),

- al modificar el valor del artículo ([HModify](../WDLang4/3044042.md)),

- durante la [automatic modificación de los archivos de datos](../WDLang4/3044195.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the number of generated thumbnails 
// and the number of expected thumbnails
// for the Customer.Photo item
NumberThumbnail is int
ExpectedNumberThumbnail is int
NumberThumbnail = Customer.Photo.NbThumbnail
ExpectedNumberThumbnail = Customer.Photo.NbDescribedThumbnail
// Same number?
IF NumberThumbnail = ExpectedNumberThumbnail THEN
	Info("All the thumbnails have been generated")
ELSE
	FOR Number = 1 TO ExpectedNumberThumbnail
		ErrorString += "Thumbnail " + Number + Customer.Photo.InfoThumbnailGeneration[n] + CR
	END
	Error("Generation error", ErrorString)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Number of generated thumbnails> = <Data file>.<item>.NbThumbnail`
---

**`<Number of generated thumbnails>: (Integro)`**

Número de miniaturas generadas para el elemento especificado.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **NbThumbnail** Property está disponible:

- en modo HFSQL Classic y HFSQL Mobile: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: sólo para los artículos que se encuentran en un archivo de datos.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **NbThumbnail** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.

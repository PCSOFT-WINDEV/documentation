


## MaskTitleTime (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaskTitleTime** Property se utiliza para get o cambiar la máscara utilizada para el título de las columnas de tiempo: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Agenda.

- en un control Planificador. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Línea de tiempo. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SCH_Scheduler.MaskTitleTime = "HH:MM:SS" // Displays "01:00:00"
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
TL_TimeLine.MaskTitleTime = "SS'CC" // Displays "01'23"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar la máscara de los títulos de las columnas de tiempo

`<Result> = <Control used>.MaskTitleTime`
---

**`<Result>: (Cadena de caracteres)`**

Máscara de visualización actual para los títulos de las columnas de tiempo. Esta máscara corresponde a: 

- " " (cadena vacía) si se usa la máscara de Default. 

- la máscara de visualización utilizada (véanse las observaciones). 




**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Agenda.

- Control Planificador.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Línea de tiempo. 





<a name="SYNTAX2"></a>

### Modificando la máscara de los títulos de las columnas de tiempo

`<Control used>.MaskTitleTime = <New mask>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Agenda.

- Control Planificador.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Control Línea de tiempo.




**`<New mask>: (Cadena de caracteres)`**

Nueva máscara de visualización para los títulos de las columnas de tiempo. Esta máscara puede corresponder a: 

- una máscara preestablecida o una máscara personalizada (ver Comentarios).

- "" (cadena vacía) para usar la máscara de visualización de Default. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Formatos de visualización disponibles
<a name="formatos_visualizacion_disponibles_ELTPARAGRAPHE000086"></a>

El formato de visualización puede corresponder a: 

- una máscara de preselección: 

	- 'HH:MM'

	- 'HH:MM:SS'

	- 'HH:MM:SS:CC'

	- 'HH:MM:SS:CCC'

	- 'HH:MM AM'




- una máscara personalizada. En este caso, tiene la posibilidad de utilizar los siguientes elementos: 

	- HH' (hora)

	- MM" (minutos)

	- SS" (segundos)

	- CC" o "CCC" (centésimas de segundo con 2 ó 3 caracteres)

	- AM" (para las horas en formato AM/PM).

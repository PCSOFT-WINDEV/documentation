
## HighlightColor (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **HighlightColor** Property se utiliza para set el resalte color de un elemento. Esta propiedad se puede utilizar en: 

- una variable de tipo [docFormatting](../WDLang1/1000022817.md). 

- una variable de tipo [docElementText](../WDLang1/1000022493.md) (características de formato). 

- una variable de tipo [docFragment](../WDLang1/1000022482.md) (características de formato).

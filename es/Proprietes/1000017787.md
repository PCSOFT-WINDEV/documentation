


## Path (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Path** Property se utiliza para get y cambiar un ruta. Esta propiedad se puede utilizar en:

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [rssCloud](../WDLang5/1000017785.md).

- una variable de tipo [Cookie](../WDLang3/1000019222.md).

- una variable de tipo [hdfsStatus](../WDLang4/1000021973.md).

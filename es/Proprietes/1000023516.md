
## ImageType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ImageType** Property se utiliza para get el tipo de Image que se está editando en un control del Editor de imágenes (Icon o Image).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(IE_Editor.ImageType)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Image Editor control>.ImageType`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de Image utilizada: 


|   |   |
| --- | --- |
| *picIcon* | Imagen de tipo Icon. |
| *picImage* | Image de tipo Image. |



**`<Image Editor control>: (Nombre del control)`**

Nombre del control Editor de imágenes utilizado.

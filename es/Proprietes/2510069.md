


## RightClick (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RightClick** Property se utiliza para get y cambiar el nombre del control Botón que se ejecutará cuando se haga clic con el botón derecho del ratón sobre un objeto.

Por defecto, se puede definir el nombre de la control Botón ejecutada al hacer clic con el botón derecho del ratón en el editor de ventanas, en el control description (pestaña "Detalles").

**Observación**: Si se especifican tanto "Clic derecho" como "Menú emergente" para la misma control, sólo se ejecutará la acción definida para el clic derecho.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the button associated with the right click of the BTN_New control
BTN_New.RightClick = "Btn_Wizard"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificando la control Botón associated con el clic derecho

`<Button control> = <Control used>.RightClick`
---

**`<Button control>: (Cadena de caracteres)`**



- Nombre de la control Botón cuyo código se ejecutará al hacer clic con el botón derecho en la control especificada,

- Vaciar la cadena ("") si no hay control Botón es associated con el clic derecho.




**`<Control used>: (Nombre del control)`**

Nombre del control para el que se solicita el nombre de la control Botón associated con el clic derecho.  


<a name="SYNTAX2"></a>

### Modificando la control Botón associated con el clic derecho

`<Control used>.RightClick = <New Button control>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control para el que se debe modificar el nombre del botón associated con el botón derecho.

**`<New Button control>: (Nombre del control)`**



- Nombre de la control Botón cuyo código se ejecutará al hacer clic con el botón derecho en la control especificada,

- Vaciar la cadena ("") si no hay un botón es associated con el botón derecho.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **RightClick** Property se aplica sólo a las ventanas y controles de una ventana (incluidas las columnas tabla). Este Property no puede ser usado con los menús, etc.

Este Property no está disponible para los siguientes controles:

- ActiveX control,

- Control Forma, 

- control Imagen sin clic,

- Control Barra de progreso, 

- Control Estático.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El **RightClick** Property se aplica únicamente a los siguientes elementos:

- Control Botón.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Ventana.

- Control Tabla.

- Columna de tabla.

- Supercontrol.

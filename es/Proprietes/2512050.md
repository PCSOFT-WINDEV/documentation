


## Minute (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Minute** Property se utiliza para:

- Recuperar los minutos de un [momento](../Motscles/1514068.md) o [FechaHora](../Motscles/1514070.md) Variable, o el número de minutos de un [Duración](../Motscles/1514069.md) Variable.

- Modificar los minutos en un [momento](../Motscles/1514068.md) o [FechaHora](../Motscles/1514070.md) Variable, o el número de minutos en un [Duración](../Motscles/1514069.md) Variable.

- Recuperar los minutos o el número de minutos de un elemento Fecha (en formato "Fecha y hora" o "Duración") o de un elemento Hora.. 
	

- Modificar los minutos o el número de minutos en un ítem Fecha (en formato "Fecha y hora" o "Duración") o en un ítem Hora.. 
	




**Observación**: El **Minute** Property se utiliza para cambiar fácilmente los tiempos y las duraciones (añadir una hora, etc).

**Recordatorio**: Los elementos de fecha se utilizan para gestionar:

- Fechas "Año - Mes - Día" (formato AAAAMMDD)

- "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" fechas y horas (formato AAAAMMDDHHmmSSCCC)

- "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" duraciones (formato +DHHMMSSCCC)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartTime is Time = "1230"
// Add 50 minutes
StartTime.Minute += 50
// StartTime corresponds to 13:20

// Modify the minutes
StartTime.Minute = 10
```


<a name="Example2"></a>

```wl
Work.StartTime = "1230"
// Add 50 minutes
Work.StartTime.Minute += 50
// StartTime corresponds to 13:20

// Modify the minutes
Work.StartTime.Minute = 10
```

## Sintaxis

### Averiguar el número de minutos de una Hora, FechaHora o Duración Variable

`<Result> = <Time>.Minute`
---

**`<Result>: (Integro)`**

Número de minutos en 2 dígitos.

**`<Time>: (Hora, fecha, hora o duración)`**

Nombre de la Variable de tipo [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del número de minutos en una Hora, FechaHora o Duración Variable

`<Time>.Minute = <New number of minutes>`
---

**`<Time>: (Hora, fecha, hora o duración)`**

Nombre de la Variable de tipo [momento](../Motscles/1514068.md), [FechaHora](../Motscles/1514070.md) o [Duración](../Motscles/1514069.md) que se va a utilizar.

**`<New number of minutes>: (Cadena de caracteres o entero)`**

Nuevo número de minutos (incluido entre 0 y 59). Reemplaza el número de minutos de la variable manipulada.  


<a name="SYNTAX3"></a>

### Averiguar el número de minutos en un elemento de Hora o Fecha (en formato de "Fecha y Hora" o "Duración")

`<Result> = <Data file>.<Item>.Minute`
---

**`<Result>: (Integro)`**

Número de minutos en 2 dígitos.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX4"></a>

### Modificación del número de minutos en un elemento de Hora o Fecha (en formato de "Fecha y Hora" o "Duración")

`<Data file>.<item>.Minute = <New number of minutes>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<New number of minutes>: (Cadena de caracteres o entero)`**

Nuevo número de minutos (incluido entre 0 y 59). Reemplaza el número de minutos en el elemento especificado..

## Observaciones


### Gestión de los minutos y las horas
<a name="gestion_los_minutos_las_horas_ELTPARAGRAPHE000167"></a>

**Caso 1: Asignación directa**
Durante una asignación directa (por ejemplo, *MyTime.Minute = n*), los minutos deben estar comprendidos entre 00 y 59. Se produce un error de WLanguage si se especifica un número de minutos no válido..
La sintaxis *MiHora.Minuto = MiHora.Minuto + 5* puede generar un error en tiempo de ejecución.

** Caso 2: Operaciones en el acta**
Cuando se realizan operaciones en las variables [momento](../Motscles/1514068.md) y [FechaHora](../Motscles/1514070.md), las horas van de 0 a 23, y los minutos van de 0 a 59. La suma o resta de minutos tiene en cuenta automáticamente los cambios de 59 minutos a la hora siguiente.

Por lo tanto, si se añaden 50 minutos a:

- una variable de tipo [momento](../Motscles/1514068.md) correspondiente a las 23:35, el resultado será 0:25.

- una variable de tipo [FechaHora](../Motscles/1514070.md), se modificará la hora y el número de minutos (se añadirá una hora).



```wl
// Example on a variable
StartTime is Time = "2335"   
StartTime.Minute += 50
// StartTime corresponds to 0:25 
 
StartTime is DateTime = "202011262335"
StartTime.Minute += 50	
// StratTime corresponds to 27/11/2020 at 0:25
```

<a name="NOTE0_2"></a>


### Operadores disponibles para los minutos
<a name="operadores_disponibles_para_los_minutos_ELTPARAGRAPHE000201"></a>

Los siguientes operadores aritméticos pueden utilizarse con el **Minute** Property: 

- ++ y --

- += y -=



```wl
// Example on an item
Work.StartTime = "1320"
Work.StartTime.Minute++     // Add 1 minute
Work.StartTime.Minute+=5     // Add 5 minutes
Work.StartTime.Minute-=5	// Subtract 5 minutes
```




## Description (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **description** se utiliza para obtener y cambiar el description de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [gglCalendar](../WDLang5/1000017418.md).

- una variable de tipo [hBackupDescription](../WDLang4/1000017455.md). 

- una variable de tipo [hScheduledOptimization](../WDLang4/1000018854.md).

- una variable de tipo [hScheduledTask](../WDLang4/1000017448.md).

- una variable de tipo [Marker](../WDLang3/1000019940.md).

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [rssImage](../WDLang5/1000017799.md). 

- una variable de tipo [rssEntry](../WDLang5/1000017792.md).

- una variable de tipo [rssTextInput](../WDLang5/1000017802.md)

- una variable de tipo [hRSConfig](../WDLang4/1000020484.md).

- una variable de tipo [inAppProduct](../WDLang3/1000020868.md).

- una variable de tipo [hScheduleMaterializedView](../WDLang4/1000021444.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [notificationCategory](../WDLang3/1000023981.md).

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).

- una variable de tipo [MapImage](../WDLang3/1000025523.md).

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md).

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md).

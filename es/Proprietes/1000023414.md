


## RowDetailsBinding (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **RowDetailsBinding** Property permite identificar y modificar el enlace de datos utilizado para mostrar los detalles de un fila en un control Tabla de navegación . 

**Recordatorio**:  Los detalles de fila son un "sub-tabla" que se muestra debajo de fila cuando el usuario hace clic en el [+] Icon junto al fila o hace doble clic en el fila en el control Tabla. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the orders when clicking the row displaying the customer
TABLE_CUSTOMER.RowDetailsBinding = "CUSTOMERID:ORDER.CUSTOMERID"
```



```wl
// Displays the city name when clicking the row displaying the customer
TABLE_CUSTOMER.RowDetailsBinding = "ZC:CITY.ZC:NAME"
```



```wl
// Displays the orders as well as the city and state when clicking the row displaying a customer 
TABLE_CUSTOMER.RowDetailsBinding = [
CUSTOMERID:ORDER.CUSTOMERID
ZC:CITY.ZC:NAME,STATE
]
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificación del enlace de datos utilizado para mostrar los detalles de una fila en una control Tabla de navegación

`<Result> = <Browsing Table control>.RowDetailsBinding`
---

**`<Result>: (Constante o cadena de caracteres)`**

Vinculación actual: 



|   |   |
| --- | --- |
| *lldNone* | No se ha definido ninguna vinculación. <br>No se visualiza ningún detalle en la navegación control Tabla al hacer clic en el signo "[+]" o al hacer doble clic en el fila. |
| *lldAutomatic* | Las vinculaciones definidas en el análisis se utilizan para encontrar los archivos de datos vinculados para mostrar. <br>Si se vinculan varios archivos de datos en el análisis, el usuario podrá elegir el archivo a mostrar en los detalles a través de un menú contextual. |
| Cadena de caracteres | La encuadernación es personalizada. La ventana detallada muestra los datos especificados. <br>Esta cadena tiene el siguiente formato: <br><br><pre><code>ITEM:FILEA.ITEMA:DispItem1,DispItem2,DispItem3<CR><br>ITEM2:FILEB.ITEMB</code></pre><br>donde: <br><br>- ITEM,ITEM2 son los elementos que se encuentran en el archivo de búsqueda principal de control Tabla,<br><br>- FILEA,FILEB son los archivos de datos vinculados que se mostrarán en detalle,<br><br>- ITEMA, ITEMB son los elementos vinculados a los elementos ITEM, ITEM2 en estos archivos,<br><br>- DispItem1, DispItem2, DispItem3 son los elementos encontrados en FILEA que se mostrarán en detalle. Si este elemento no se llena, se mostrarán todos los elementos de análisis de Visible.<br><br><br><br><br>Por ejemplo, el Line: <br><br><pre><code>OrderRef:QRY_OrderLine.ParamOrderRef:VISUAL,REF,<br>CAPTION,UNITPRICE,QUANTITY,TOTALBT</code></pre><br> vincula el elemento OrderRef de la control Tabla con el elemento ParamOrderRef de la consulta QRY_OrderLine. <br>Los elementos que se muestran en la ventana detallada son los siguientes: VISUAL, REF, CAPTION, UNITPRICE, QUANTITY, TOTALBT.. |



**`<Browsing Table control>: (Nombre del control)`**

Nombre de la navegación control Tabla a utilizar.  


<a name="SYNTAX2"></a>

### Modificación del enlace de datos utilizado para mostrar los detalles de una fila en una control Tabla de navegación

`<Browsing Table control>.RowDetailsBinding = <New value>`
---

**`<Browsing Table control>: (Nombre del control)`**

Nombre de la navegación control Tabla a utilizar.  

**`<New value>: (Constante o cadena de caracteres)`**

| --- | --- |

| Cadena de caracteres | La encuadernación es personalizada. La ventana detallada muestra los datos especificados. <br>Esta cadena tiene el siguiente formato: <br><br><pre><code>ITEM:FILEA.ITEMA:DispItem1,DispItem2,DispItem3<CR><br>ITEM2:FILEB.ITEMB</code></pre><br>donde: <br><br>- ITEM,ITEM2 son los elementos que se encuentran en el archivo de búsqueda principal de control Tabla,<br><br>- FILEA,FILEB son los archivos de datos vinculados que se mostrarán en detalle,<br><br>- ITEMA, ITEMB son los elementos vinculados a los elementos ITEM, ITEM2 en estos archivos,<br><br>- DispItem1, DispItem2, DispItem3 son los elementos encontrados en FILEA que se mostrarán en detalle.


## Version (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Version** permite obtener y modificar la versión de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [rssStream](../WDLang5/1000017795.md).

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

- una variable de tipo [xmlDocument](../WDLang5/1000018782.md).

- una variable de tipo [hbReading](../WDLang4/1000021701.md).

- una variable de tipo [pdfParameter](../WDLang6/1410087857.md).

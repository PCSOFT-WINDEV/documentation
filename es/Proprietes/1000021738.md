
## EndTimestamp (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **EndTimestamp** Property se utiliza para set la fecha y hora de finalización de una acción. Está disponible para los siguientes tipos de variables:

- [hbReading](../WDLang4/1000021701.md)

- [hbScanParameter](../WDLang4/1000021715.md)

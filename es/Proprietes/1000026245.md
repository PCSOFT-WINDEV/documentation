
## InvalidInputMessage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InvalidInputMessage** Property se utiliza para identificar y set el mensaje que se mostrará si el control contiene datos no válidos (máscara de entrada o valor fuera de los límites). Si la Property está vacía, se utilizará una cadena Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_ProductRef.InvalidInputMessage = "Please use the standard product reference format"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificación del mensaje que aparece si el control contiene datos no válidos

`<Result> = <Control used>.InvalidInputMessage`
---

**`<Result>: (Cadena de caracteres)`**



- Mensaje de error que aparece si el control contiene datos no válidos,

- Cadena vacía si se utiliza el mensaje de error de Default ("Entrada no válida"). 




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Campo de entrada. 

- Control Combo Box. 





<a name="SYNTAX2"></a>

### Cambiar el mensaje que aparece si la entrada no es válida

`<Control used>.InvalidInputMessage = <Message>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Control Campo de entrada. 

- Control Combo Box. 




**`<Message>: (Cadena de caracteres)`**



- Mensaje de error que aparece si el control contiene datos no válidos,

- Cadena vacía si se utiliza el mensaje de error de Default ("Entrada no válida").




## ToolTip (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ToolTip** Property se utiliza para get y cambiar el texto de un tooltip. Esta propiedad se puede utilizar en:

- los mandos de una ventana o Page. Para obtener más información, consulte [ToolTip](../Proprietes/2510005.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [GanttTask](../WDLang1/1000020881.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docElementLink](../WDLang1/1000022970.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Token](../WDLang1/1000023942.md).

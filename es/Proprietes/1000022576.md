
## LeftIndent (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **LeftIndent** Property se utiliza para get y cambiar la sangría izquierda de un elemento. Esta propiedad se puede utilizar en:

- controles de edición y controles estáticos. Para obtener más información, consulte [LeftIndent](../Proprietes/2513018.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) variables de tipo [docParagraph](../WDLang1/1000022483.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) variables de tipo [docNumberingLevel](../WDLang1/1000022799.md).


## spHash (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **spHash** permite:

- Obtener el algoritmo utilizado para aplicar sal a un campo de tipo "Contraseña". Esta propiedad puede utilizarse para campos de tipo "Contraseña" definidos en el editor de análisis o mediante programación. 

- Establecer el algoritmo utilizado para aplicar la sal (solo para campos de tipo "Contraseña" descritos mediante programación). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile file
...

// Describe the "Password" item
MyItem is Item Description
MyItem.Name = "Password"
MyItem.Type = hItemSecurePassword
MyItem.spDerivation = SP_PBKDF2
MyItem.spHash = SP_SHA2_256
MyItem.spIterationCount = 20000
MyItem.spLength = 256

// Validate the description of the "Password" item
HDescribeItem(AFile, MyItem)

// Validate the description of AFile file
...
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el algoritmo utilizado para aplicar sal a un campo de tipo "Contraseña"

`<Salting algorithm> = <Data file>.<Item>.spHash`
---

**`<Salting algorithm>: (Constante de tipo Integer)`**

Algoritmo utilizado para aplicar sal a un campo de tipo "Contraseña": 



|   |   |
| --- | --- |
| *SP_SHA2_256* | Se utiliza el Algoritmo de Hash Seguro 256 (SHA-256). |
| *SP_SHA2_512* | Se utiliza el Algoritmo de Hash Seguro 512 (SHA-512). |
| *SP_SHA3_256* | Se utiliza el Algoritmo de Hash Seguro 3-256 (SHA-3 256). |
| *SP_SHA3_512* | Se utiliza el Algoritmo de Hash Seguro 3-512 (SHA-3 512). |



**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo utilizado. Este nombre se definió en el editor de análisis o con el tipo [File Description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió en el editor de análisis o con el tipo [Item Description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Establecer el algoritmo utilizado para aplicar sal a un campo de tipo "Contraseña" (descrito mediante programación)

`<Item>.spHash = <Salting algorithm>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del campo utilizado. Este nombre se definió con el tipo [Item Description](../WDLang4/1514071.md).

**`<Salting algorithm>: (Constante de tipo Integer)`**

Algoritmo utilizado para aplicar sal a un campo de tipo "Contraseña": 



|   |   |
| --- | --- |
| *SP_SHA2_256* | Se utiliza el Algoritmo de Hash Seguro 256 (SHA-256). |
| *SP_SHA2_512* | Se utiliza el Algoritmo de Hash Seguro 512 (SHA-512). |
| *SP_SHA3_256* | Se utiliza el Algoritmo de Hash Seguro 3-256 (SHA-3 256). |
| *SP_SHA3_512* | Se utiliza el Algoritmo de Hash Seguro 3-512 (SHA-3 512). |




## Category (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Category** permite obtener y cambiar la categoría de un elemento. Esta propiedad se puede utilizar en:

- una variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [EventTimeline](../WDLang1/1000020612.md).

- una variable de tipo [Appointment](../WDLang1/1000019244.md).

- una variable de tipo [rssChannel](../WDLang5/1000017769.md).

- una variable de tipo [rssEntry](../WDLang5/1000017792.md).

- la estructura [Email](../WDLang3/3032029.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docProperty](../WDLang1/1000022945.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [Notification](../WDLang3/1000019441.md).


## Permission (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Permission** se utiliza para obtener y modificar Permission s. 

Esta palabra clave puede corresponder a: 

- el **Permission** Property que se puede utilizar en:

	- una variable de tipo [btleCharacteristic](../WDLang3/1000022060.md).

	- una variable de tipo [btleDescriptor](../WDLang3/1000022064.md). 

	- una variable de tipo [fbSession](../WDLang5/1000021851.md).


## PageBorderVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **PageBorderVisible** permite: 

- determinar si los bordes de página se muestran en un control Editor de diagramas.  

- mostrar u ocultar los bordes de página en un control Editor de diagramas.




Observación: Al imprimir el diagrama, los bordes permiten ver claramente las páginas necesarias para el trabajo de impresión. Puede establecer el tamaño de la página con:  

- la propiedad "PageLayout" del control Editor de diagramas,

- la propiedad "PageLayout" de la variable de tipo [Diagram](../WDLang1/1410088055.md).  



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the page borders and gridlines
DIAGEDT_Diagram.PageBorderVisible = True
DIAGEDT_Diagram.GridlinesVisible = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Determinar si se muestran los bordes de página

`<Result> = <Diagram Editor control>.PageBorderVisible`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se muestran los bordes de página,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado. 


<a name="SYNTAX2"></a>

### Mostrar u ocultar los bordes de página

`<Diagram Editor control>.PageBorderVisible = <Display>`
---

**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado. 

**`<Display>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> para que se muestren los bordes de página,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## BarVisible (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BarVisible** Property se utiliza para gestionar el banda de selección en una columna de una Tabla o control Tabla TreeView. Es posible:

- Averigua si el banda de selección es Visible o no en la columna.

- Mostrar u ocultar el banda de selección en la columna.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Do not display the selection bar on columns 1 and 5
COL_Column1.BarVisible = False
COL_Column5.BarVisible = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averigua si el banda de selección es Visible en una columna

`<Visible/Invisible> = <Column used>.BarVisible`
---

**`<Visible/Invisible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el banda de selección es Visible,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Column used>: (Nombre del control)`**

Nombre de la columna (Tabla o TreeView Table) a utilizar.  


<a name="SYNTAX2"></a>

### Hacer visible la banda de selección o Invisible en una columna

`<Column used>.BarVisible = <Visible/Invisible>`
---

**`<Column used>: (Nombre del control)`**

Nombre de la columna (Tabla o TreeView Table) a utilizar.

**`<Visible/Invisible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la banda de selección debe ser visible en la columna,

- <u><u><u><u>False</u></u></u></u> para hacer visible la banda de selección en la columna.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El **BarVisible** Property se puede utilizar en las columnas de los controles Tabla y Tabla TreeView.

- El **BarVisible** Property se puede utilizar en los controles de la tabla de selección múltiple: la banda de selección es visible/Invisible en la columna especificada para todas las filas seleccionadas.

- El **BarVisible** Property puede utilizarse en barras de selección associated con un Color o un Image.


## Header

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Header** permite obtener y cambiar un encabezado. Esta propiedad se puede utilizar en:

- a Variable de tipo [Email](../WDLang3/1000018713.md). 

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [httpResponse](../WDLang3/1000021165.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- una variable de tipo [restResponse](../WDLang3/1000021477.md).

- a Variable de tipo [apiRequest](../WDLang3/1410088935.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [psheetSelection](../WDLang1/1000023498.md).


## ContentType (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **ContentType** Property se utiliza para get y cambiar el tipo de un mensaje. Esta propiedad se puede utilizar en:

- una variable de tipo [emailAttach](../WDLang3/1000018752.md).

- una variable de tipo [httpRequest](../WDLang3/1000021158.md).

- una variable de tipo [httpResponse](../WDLang3/1000021165.md).

- una variable de tipo [restRequest](../WDLang3/1000021481.md).

- una variable de tipo [restResponse](../WDLang3/1000021477.md).

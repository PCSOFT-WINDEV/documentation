
## CriticalSection

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **CriticalSection** puede corresponder a:

- la función [CriticalSection](../WDLang1/1000024373.md), que se utiliza para habilitar una sección crítica con nombre o sobre variable en una sentencia [USO DE ... EN](../Motscles/1510025.md).

- \* el tipo de variable [CriticalSection](../WDLang1/1000023972.md), que permite definir secciones críticas para limitar la ejecución simultánea de un código (procedure, línea de código, etc.) **a un thread** en un momento dado de una aplicación.

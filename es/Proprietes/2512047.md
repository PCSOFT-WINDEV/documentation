


## CtDatabase (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CtDatabase** Property se utiliza para set la fuente de datos OLE DB cuando se conecta a un tabla específico a través de OLE DB. Es posible:

- get la fuente de datos OLE DB utilizada para conectarse a un tabla vía OLE DB.
	Esta fuente puede corresponder a:

	- la fuente de datos definida en el editor de análisis,

	- la fuente de datos especificada en [HConnect](../WDLang4/3044263.md) (si se ha utilizado esta función).
			Este Property puede utilizarse para el un archivo de datos definido en el editor de análisis o mediante programación.




- Definir la fuente de datos OLE DB utilizada para conectarse a un tabla por OLE DB (cuando el archivo de datos se describe mediante programación).
	En este caso, esta Property es equivalente a la [CtDescribedDatabase](../Proprietes/2512016.md) Property. 




**Recordatorio**: [HConnect](../WDLang4/3044263.md) se utiliza para redefinir uno o varios parámetros de la conexión OLE DB a un tabla. Por Default, se definen las características de la conexión OLE DB:

- en la editor de análisis durante la descripción del archivo de datos,

- cuando se describió el archivo de datos a través de la programación.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the OLE DB data source
ResOLEDBDatabase = Client.CtDatabase
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averigua la fuente de datos de la OLE DB

`<OLE DB data source> = <Data file>.CtDatabase`
---

**`<OLE DB data source>: (Cadena de caracteres)`**

ruta completa de la fuente de datos OLE DB.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).  


<a name="SYNTAX2"></a>

### Definir la fuente de datos de la OLE DB

`<Data file>.CtDatabase = <OLE DB data source>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido por el tipo [Expediente description](../WDLang4/1514065.md).

**`<OLE DB data source>: (Cadena de caracteres)`**

ruta completa de la fuente de datos OLE DB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El [CtDescribedDatabase](../Proprietes/2512016.md) Property se utiliza para get y cambiar la fuente de datos especificada para la conexión OLE DB cuando se describe el archivo de datos.

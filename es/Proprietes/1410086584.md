
## Compilar

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **Compile** puede corresponder a:

- la función **[Compilar](../WDLang1/3013015.md)** que se utiliza para compilar dinámicamente un procedure cuyo código fuente se suministra.. 

- la declaración **&lt;COMPILAR SI&gt;** que se utiliza para Compile( o no) un código WLanguage según una condición (principalmente utilizado para compilar un código según una plataforma o a un tipo de configuración.

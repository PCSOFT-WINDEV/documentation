
## FileName (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **FileName** Property obtiene el nombre de un fichero (fichero WDPic o fichero generado para un trabajo de impresión). Esta propiedad se puede utilizar en:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [iOutputInfo](../WDLang5/1410088954.md).

- una variable de tipo [WDPic](../WDLang1/1000023503.md).

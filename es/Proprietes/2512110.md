


## MaxThumbnailHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**MaxThumbnailHeight** se utiliza para:

- get la altura máxima de una miniatura de un elemento de Image (nota binaria). Este punto se definió en la editor de análisis o a través de la programación.

- establecer la altura máxima de una miniatura de un elemento de imagen (memo binario), en el elemento description a través de la programación solamente.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the Photo item
Photo is Item Description
...
// Define the maximum height of the first 
// thumbnail for the description of the Photo item
Photo.MaxThumbnailHeight[1] = 50
// equivalent to Photo.MaxThumbnailHeight = 50
...
// Validate the data file description
HDescribeFile(Customer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la altura máxima de una miniatura de un artículo de Image (nota binaria)

`<Current maximum height> = <Data file>.<item>.MaxThumbnailHeight[<N>]`
---

**`<Current maximum height>: (Integro)`**

Altura máxima (en píxeles) de la miniatura especificada.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<N>: (Entero opcional)`**

Número de la miniatura cuya altura máxima se solicita. Si no se especifica este parámetro, se devuelve la altura máxima de la primera miniatura..  


<a name="SYNTAX2"></a>

### Modificar la altura máxima de una miniatura en un artículo de Image (nota binaria)

`<item>.MaxThumbnailHeight[<N>] = <New maximum height>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre de la posición de nota binaria utilizada. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<N>: (Entero opcional)`**

Número de la miniatura cuya altura máxima debe modificarse. Si no se especifica este parámetro, se modifica la altura máxima de la primera miniatura.

**`<New maximum height>: (Integro)`**

Nueva altura máxima (en píxeles) para la miniatura especificada. Esta altura debe estar comprendida entre 1 y 65535.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El **MaxThumbnailHeight** Property está disponible:

- en HFSQL Classic: en los artículos de un archivo de datos, ver o consultar.

- en modo HFSQL Client/Server: sólo para los artículos que se encuentran en un archivo de datos.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El **MaxThumbnailHeight** Property sólo está disponible en los elementos del archivo de datos en HFSQL Client/Server.

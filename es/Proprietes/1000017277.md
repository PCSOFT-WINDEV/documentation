


## KeySize (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **KeySize** Property se utiliza para set las características de un elemento clave de la nota de texto. Es posible:

- get el tamaño de la parte indexada de la nota para un elemento definido en la editor de análisis o mediante programación.

- Definir el tamaño de la parte indexada de la nota (cuando el artículo se describe mediante programación).   





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyMemoIndex is Item Description
MyMemoIndex.Name = "Comments"
MyMemoIndex.Type = hItemTextMemo
MyMemoIndex.KeyType = hUniqueKey
MyMemoIndex.SortType = hIncreasing
MyMemoIndex.AccentSensitive = False
MyMemoIndex.CaseSensitive = False
MyMemoIndex.PunctuationSensitive = False
MyMemoIndex.KeySize = 50
HDescribeItem(Book, MyMemoIndex)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el tamaño de la parte indexada de la nota

`<Size> = <Data file>.<Item>.KeySize`
---

**`<Size>: (Integro)`**

Número de caracteres que se indexarán en la nota de texto.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Definir el tamaño de la parte indexada del memorándum

`<Item>.KeySize = <Indexed size>`
---

**`<Item>: (Artículo description Variable)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Indexed size>: (Integro)`**

Tamaño de la parte indexada de la nota, expresado en número de caracteres.


## InSeconds (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **InSeconds** Property se utiliza para convertir una Duración Variable a segundos. Es posible:

- Averigüe el número de segundos correspondiente a una duración.

- Modificar el número de segundos correspondiente a una duración.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDuration is Duration
MyDuration.InHours = 210
Trace(MyDuration.InSeconds)  // 756000
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar el número de segundos correspondientes a una duración Variable

`<Result> = <Duration used>.InSeconds`
---

**`<Result>: (Real)`**

Número de segundos correspondiente a la duración especificada.

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.  


<a name="SYNTAX2"></a>

### Modificando el número de segundos correspondientes a una duración Variable

`<Duration used>.InSeconds = <New duration>`
---

**`<Duration used>: (Duración)`**

Nombre de la variable de tipo [Duración](../Motscles/1514069.md) que se utilizará.

**`<New duration>: (Real)`**

Número de segundos correspondiente a la duración especificada.

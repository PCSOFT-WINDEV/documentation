
## ScrollValue (Propiedad)

*** ***
	



<a name="XOverwiew"></a>
<a name="Presentación"></a>
En un control Barra de desplazamiento, el valor de desplazamiento es el grado en que se desplaza el cuadro de desplazamiento cuando el usuario hace clic en el área entre el cuadro y una de las flechas de desplazamiento. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=avancerapide.gif)


En un Control de número, esta propiedad corresponde a los incrementos en los que cambia el valor cuando el usuario hace clic en una de las flechas. 
<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **ScrollValue** obtiene y establece el valor de desplazamiento. Es posible:

- Obtener el valor de desplazamiento para una barra de desplazamiento y control deslizante de rango, o el valor de incremento para un control de número.

- Establecer el valor de desplazamiento para una barra de desplazamiento y control deslizante de rango, o el valor de incremento para un control de número.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the scroll value
SCROLL_Scrollbar1.ScrollValue = 10
// Simulate the scroll value of a page
SCROLL_Scrollbar1 += SCROLL_Scrollbar1.ScrollValue
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el valor de desplazamiento

`<Current scroll value> = <Control used>.ScrollValue`
---

**`<Current scroll value>: (Entero)`**

Grado en que se desplaza el cuadro de desplazamiento cuando el usuario hace clic en el fondo de una barra de desplazamiento o control deslizante de rango, o en las flechas de un control de número.

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Barra de desplazamiento,

- Control de número,

- Control deslizante de rango.





<a name="SYNTAX2"></a>

### Establecer el valor de desplazamiento

`<Control used>.ScrollValue = <New scroll value>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Barra de desplazamiento,

- Control de número,

- Control deslizante de rango.




**`<New scroll value>: (Entero)`**

Nuevo valor en que se desplaza el cuadro de desplazamiento cuando el usuario hace clic en el fondo de una barra de desplazamiento o control deslizante de rango, o en las flechas de un control de número.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Valor de desplazamiento y límites
<a name="valor_desplazamiento_limites_ELTPARAGRAPHE000075"></a>

Si las propiedades [MinValue](../Proprietes/2510008.md) y [MaxValue](../Proprietes/2510009.md) son menores que **ScrollValue**, la propiedad **ScrollValue** toma automáticamente el siguiente valor: (MaxValue - MinValue).
<a name="NOTE0_2"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000093"></a>

La propiedad **ScrollValue** solo se aplica a los controles de una ventana: 

- Barra de desplazamiento. 

- Control de número.

- Control deslizante de rango.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) En Linux, esta propiedad solo está disponible para controles Barra de desplazamiento.

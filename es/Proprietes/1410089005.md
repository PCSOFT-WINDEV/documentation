
## EditMode (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **EditMode** obtiene y establece el modo de edición del control Editor de diagramas: selección o dibujo a mano alzada. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables freehand drawing
DIAGEDT_MyDiagram.EditMode = diagModeFreehandDrawing
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el modo de edición actual en el control Editor de diagramas

`<Result> = <Diagram Editor control>.EditMode`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de edición actual en el control: 



|   |   |
| --- | --- |
| *diagModeFreehandDrawing* | El control está en modo de dibujo a mano alzada. |
| *diagModeSelection* | El control está en modo de selección. |



**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado. 


<a name="SYNTAX2"></a>

### Cambiar el modo de edición actual en el control Editor de diagramas

`<Diagram Editor control>.EditMode = <New editing mode>`
---

**`<Diagram Editor control>: (Nombre del control)`**

Nombre del control Editor de diagramas utilizado. 

**`<New editing mode>: (Constante de tipo Integer)`**

Nuevo modo de edición en el control: 



|   |   |
| --- | --- |
| *diagModeFreehandDrawing* | El control está en modo de dibujo a mano alzada. |
| *diagModeSelection* | El control está en modo de selección. |

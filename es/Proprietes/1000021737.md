
## StartTimestamp (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **StartTimestamp** Property se utiliza para set la fecha y hora de inicio de una acción. Está disponible para los siguientes tipos de variables:

- [hbReading](../WDLang4/1000021701.md),

- [hbScanParameter](../WDLang4/1000021715.md).

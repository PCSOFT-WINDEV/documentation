
## Pushed (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Pulsado** Property se utiliza para get y cambiar el estado de un botón de dos estados (pulsado o liberado).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Press BTN_Button1
BTN_Button1.Pushed = True

// Modify window size
IF BTN_Detail.Pushed = True THEN
	WIN_Window1.Height = 400
ELSE
	WIN_Window1.Width = 200
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención del estado de un control Botón encendido/apagado

`<Result> = <Button control>.Pushed`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se presiona el control Botón, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Button control>: (Nombre del control)`**

Nombre de la control Botón de encendido/apagado ("Botón On/Off" marcado en la pestaña "UI" del control description).  


<a name="SYNTAX2"></a>

### Modificar el estado de un control Botón encendido/apagado

`<Button control>.Pushed = <New state>`
---

**`<Button control>: (Nombre del control)`**

Nombre de la control Botón de encendido/apagado ("Botón On/Off" marcado en la pestaña "UI" del control description).

**`<New state>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Botón debe ser presionado, 

- <u><u><u><u>False</u></u></u></u> si el control Botón debe estar en su estado Normal.

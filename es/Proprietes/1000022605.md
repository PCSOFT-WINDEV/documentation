
## Database (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Database** permite obtener y cambiar la base de datos asociada a una conexión. Esta propiedad se puede utilizar en:

- una conexión. Para obtener más información, consulte la ayuda sobre la propiedad [Base de datos](../Proprietes/2512101.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoConnection](../WDLang4/1000022409.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [mongoGridFS](../WDLang4/1000022416.md).


## BackgroundImage (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BackgroundImage** Property se utiliza para identificar y modificar el fondo color de un elemento. Esta propiedad se puede utilizar en:

- una ventana o una control (botón, barra de progreso, deslizador, HTML control Campo de entrada). Para obtener más información, consulte [BackgroundImage](../Proprietes/2510056.md).

- una variable de tipo [OrgElement](../WDLang1/1000019713.md).

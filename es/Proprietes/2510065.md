


## MaxHeight (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **MaxHeight** Property se utiliza para get y cambiar la altura máxima de un control o ventana.

Por Default, la altura máxima:

- de un control es igual a 2 147 483 647 (0x7FFFFFFF). Este valor equivale a "sin límite de tamaño máximo".

- de una ventana es igual a 20 000. Sin embargo, el tamaño de las ventanas no debe exceder el tamaño de la pantalla.




**Recordatorio**: La altura máxima de una control o ventana se utiliza durante las siguientes operaciones:

- Ventana redimensionada por el usuario.

- Manejo de anclas.




La altura máxima de un control o de una ventana no se utiliza cuando la ventana se redimensiona mediante programación (con la función [Altura](../Proprietes/2510050.md) Property o la función [WinSize](../WDLang1/3038020.md)).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the maximum height of a control
EDT_Edit1.MaxHeight = 200
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la altura máxima de un elemento

`<Current maximum height> = <Element used>.MaxHeight`
---

**`<Current maximum height>: (Integro)`**

Altura máxima del elemento especificado (incluyendo los bordes) en píxeles.

**`<Element used>: (Nombre de control o ventana)`**

Nombre del elemento (ventana o control) cuya altura máxima se calculará.  


<a name="SYNTAX2"></a>

### Modificar la altura máxima de un elemento

`<Element used>.MaxHeight = <New maximum height>`
---

**`<Element used>: (Nombre de control o ventana)`**

Nombre del elemento (ventana o control) cuya altura máxima será modificada.

**`<New maximum height>: (Integro)`**

Nueva altura máxima del elemento (incluidos los bordes) en píxeles.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Altura inicial (control y ventana)
<a name="altura_inicial_control_ventana_ELTPARAGRAPHE000097"></a>

La altura inicial (descrita en el editor) es devuelta por el [InitialHeight](../Proprietes/2510043.md) Property.

En la mayoría de los casos, la altura del objeto depende de las propiedades **MaxHeight** y [MinHeight](../Proprietes/2510035.md). Sin embargo, esta restricción ya no se aplica cuando la altura se modifica mediante programación (con la función [Altura](../Proprietes/2510050.md) Property o la función [WinSize](../WDLang1/3038020.md)).

### Límites
<a name="limites_ELTPARAGRAPHE000122"></a>

El **MaxHeight** Property se aplica únicamente a los siguientes elementos:

- Control Botón.

- Control Estático.

- Control Campo de entrada.

- control Imagen.

- Control Casilla de verificación.

- Control Botón de opción.

- Control List Box.

- Control Combo Box.

- Control TreeView.

- Control Pestaña.

- Control Tabla.

- Supercontroles.

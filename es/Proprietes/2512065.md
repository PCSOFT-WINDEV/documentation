


## Date (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Fecha** Property se utiliza para:

- Recuperar la fecha de una [FechaHora](../Motscles/1514070.md) Variable.

- Modificar la fecha en una [FechaHora](../Motscles/1514070.md) Variable.

- Recuperar la fecha de un elemento Date (en formato "Date and Time").
	

- Modificar la fecha en un elemento Date (en formato "Date and Time").
	




**Recordatorio**: Los elementos de fecha se utilizan para gestionar:

- Fechas "Año - Mes - Día" (formato AAAAMMDD)

- "Año - Mes - Día - Horas - Minutos - Segundos - Milisegundos" fechas y horas (formato AAAAMMDDHHmmSSCCC)

- "Número de días - Número de horas - Número de minutos - Número de segundos - Número de milisegundos" duraciones (formato +DHHMMSSCCC)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Operations performed on a DateTime variable
StartDate is DateTime = "200112121515"
EndDate is Date = "20011209"
// Change the date
StartDate.Date = "20011225"
// Modify the Date part
StartDate.Date = EndDate
```


<a name="Example2"></a>

```wl
// Operations performed on a DateTime item
MyWork.StartDate = "200112121515"
MyWork.EndDate = "20011209"
// Change the date
MyWork.StartDate.Date = "20011225"
// Modify the Date part
MyWork.StartDate.Date = MyWork.EndDate
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la fecha en un DateTime Variable

`<Result> = <DateTime>.Date`
---

**`<Result>: (salir con)`**

Fecha extraída de la [FechaHora](../Motscles/1514070.md) Variable.

**`<DateTime>: (FechaHora)`**

Nombre de la variable de tipo [FechaHora](../Motscles/1514070.md) que se utilizará.  


<a name="SYNTAX2"></a>

### Modificación de la fecha en un DateTime Variable

`<DateTime>.Date = <New date>`
---

**`<DateTime>: (Fecha o Fecha-Hora)`**

Nombre de la variable de tipo [FechaHora](../Motscles/1514070.md) que se utilizará.

**`<New date>: (Cadena de caracteres o fecha o número entero)`**

Nueva fecha en el siguiente formato:

- AAAAMMDD

- Integro: número de días transcurridos entre el 1 de enero de 1800 y la fecha especificada.




Reemplaza la fecha en la Variable especificada.  


<a name="SYNTAX3"></a>

### Averiguar la fecha en un elemento de la Fecha (formato de Fecha y Hora)

`<Result> = <Data file>.<Item>.Date`
---

**`<Result>: (salir con)`**

Fecha extraída del artículo.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX4"></a>

### Modificación de la fecha en una Variable (formato de fecha y hora)

`<Data file>.<item>.Date = <New date>`
---

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).

**`<New date>: (Cadena de caracteres o fecha o número entero)`**

Nueva fecha en el siguiente formato:

- AAAAMMDD

- Integro: número de días transcurridos entre el 1 de enero de 1800 y la fecha especificada.




Reemplaza la fecha en la fecha especificada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los tipos [salir con](../Motscles/1514066.md) y [FechaHora](../Motscles/1514070.md) se utilizan para gestionar las fechas del 01/01/0001 al 31/12/9999.

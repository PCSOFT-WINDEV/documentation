


## CompactOption (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **CompactOption** Property se utiliza para:

- averiguar si las opciones de una control (Radio Button, Check Box, Dashboard) están en modo compactado.

- cambiar las opciones de una control (Botón de radio, Casilla de verificación, Tablero) al modo compactado.




En modo compactado, las opciones de una control se acercan: 

- cuando se hace una opción Invisible. 

- cuando se elimina una opción.




Si las opciones no están compactadas, mantienen su posición inicial.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
CBOX_Options.CompactOption = True
// Deletes option #3 from a Check Box control
CheckBoxDelete(CBOX_Options, 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si las opciones de una control están compactadas

`<Result> = <Control used>.CompactOption`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las opciones están compactadas,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Casilla de verificación,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Botón de opción,

- a control Cuadro de mando.





<a name="SYNTAX2"></a>

### Definir si las opciones que se encuentran en una control deben ser compactadas

`<Control used>.CompactOption = <Compact>`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado. Este control puede ser: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Casilla de verificación,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control Botón de opción,

- a control Cuadro de mando.




**`<Compact>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las opciones del control deben estar compactadas,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta propiedad está disponible en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de las casillas de verificación. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el botón de radio controla. 

- los controles del Dashboard.




**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Controles de la casilla de verificación y del botón de opción: Esta propiedad corresponde a la opción "Compactar opciones invisibles" de la pestaña "Detalles" del control description.

- Control Cuadro de mando : Esta propiedad corresponde a la opción "Compactar Widgets" de la pestaña "Detalles" del control description.

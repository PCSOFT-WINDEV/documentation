
## GUIDFile (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **GUIDFile** devuelve el GUID del archivo de datos definido en el análisis que permitió crear el archivo de datos. Este identificador se utiliza en el archivo.REP (para más detalles, véase [Gestión de archivos .REP](../WDLang4/3044186.md)).

**Observación**: La GUÍA del archivo de datos también se puede encontrar en el editor de análisis (ficha "Notas" en el description del archivo de datos).




<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GUIDAnalyse%20-%20HC%20N%B0001.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// GUID of "Customer" data file
Trace(Customer.GUIDFile)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<File GUID> = <Data file>.GUIDFile`
---

**`<File GUID>: (Cadena de caracteres)`**



- GUÍA del archivo definido en el análisis que se utilizó para crear el archivo de datos. 

- Cadena vacía ("") si el archivo de datos fue creado por programación o si el archivo de datos utilizado no es un archivo en formato HFSQL Classic o Cliente/Servidor.




**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos para el que se solicita la GUÍA.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El [GUIDAnalysis](../Proprietes/2512098.md) Property devuelve el GUID del análisis que se utilizó para crear el el archivo de datos.


## Namespace (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La propiedad **Namespace** permite gestionar espacios de nombres. Esta propiedad se puede utilizar en:

- una variable de tipo [xmlAttribute](../WDLang5/1000018799.md).

- una variable de tipo [xmlNode](../WDLang5/1000018786.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [xmlReader](../WDLang5/1000023392.md).

- una variable de tipo [memcachedConnection](../WDLang4/1000023339.md).

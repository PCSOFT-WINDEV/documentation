
## Border (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Border** Property se utiliza para get y cambiar los bordes de un elemento. Esta propiedad se puede utilizar en:

- el borde de una reporte control. Para obtener más información, consulte [Franja de orilla](../Proprietes/1000018157.md).

- una variable de tipo [xlsCell](../WDLang5/1000017472.md). 

- una variable de tipo [gglMapParameter](../WDLang5/1000017505.md).

- una variable de tipo [OrgElement](../WDLang1/1000019713.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [pdfSignature](../WDLang6/1000022255.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docParagraph](../WDLang1/1000022483.md) (características de formato).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [docNumberingLevel](../WDLang1/1000022799.md).

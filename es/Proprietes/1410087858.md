
## Quality (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **Calidad** Property se utiliza para obtener y set el nivel de calidad de un elemento. Esta propiedad se puede utilizar en: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una variable de tipo [SpeechSynthesisVoice](../WDLang3/1000025986.md). 

- una variable de tipo [pdfParameter](../WDLang6/1410087857.md).

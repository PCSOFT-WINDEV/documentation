
## DecimalPart (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **DecimalPart** Property se utiliza para configurar elementos numéricos. Es posible:

- Averigüe el número de dígitos en la parte decimal de un elemento numérico.

- Defina el número de dígitos en la parte decimal de un elemento numérico.




**Observación**: La [IntegerPart](../Proprietes/2512118.md) Property se utiliza para definir y get el número de dígitos en la parte entera de un elemento numérico.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Describe the AFile data file
...

// Describe the "Amount" item
AnItem.Name = "AMOUNT"
AnItem.Type = hItemNumeric
AnItem.IntegerPart = 15
AnItem.DecimalPart = 7

// Validate the description of the "Amount" item
HDescribeItem(AFile, AnItem)

// Validate the description of the AFile data file
...
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el número de dígitos en la parte decimal de un elemento numérico

`Result> = <Data file>.<Item>.DecimalPart`
---

**`<Result>: (Integro)`**

Número de dígitos que se tienen en cuenta en la parte decimal de la posición.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Expediente description](../WDLang4/1514065.md).

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido en la editor de análisis o con el tipo [Artículo description](../WDLang4/1514071.md).  


<a name="SYNTAX2"></a>

### Definir el número de dígitos en la parte decimal de un elemento numérico

`<Item>.DecimalPart = <Number of digits>`
---

**`<Item>: (Cadena de caracteres)`**

Nombre del artículo utilizado. Este nombre fue definido por el tipo [Artículo description](../WDLang4/1514071.md).

**`<Number of digits>: (Integro)`**

Número de dígitos a tener en cuenta para la parte decimal de un ítem Numérico.

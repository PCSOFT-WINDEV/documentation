
## BackgroundPattern (Property)

*** ***
	



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **BackgroundPattern** Property se utiliza para get y cambiar el patrón Background de un elemento. Esta propiedad se puede utilizar en:

- variables de tipo [docCell](../WDLang1/1000022927.md).

- variables de tipo [docFragment](../WDLang1/1000022482.md).

- variables de tipo [docParagraph](../WDLang1/1000022483.md).

- variables de tipo [docNumberingLevel](../WDLang1/1000022799.md).

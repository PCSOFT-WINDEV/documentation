


## Merge (Property)

*** ***
	



<a name="XOverwiew"></a>
<a name="Overview"></a>
La fusión de células se utiliza para fusionar varias células encontradas en una Tabla (o Tabla de Vista de Árbol) control en una sola: los bordes internos de las celdas desaparecen y el texto se coloca correctamente en la nueva celda. 

La combinación de celdas o títulos de columna se utiliza para crear controles que son más fáciles de ver (para los controles de tabla utilizados para mostrar estadísticas, por ejemplo).. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fusion.gif)

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La función **Merge** Property se utiliza para fusionar las celdas de una Tabla o control Tabla TreeView. Es posible:

- averiguar si los títulos de las celdas o de las columnas están fusionados.

- fusionar los títulos de las celdas o columnas.




**Importante**: Cuando una celda se combina con la siguiente, el contenido de la celda se esparce sobre la siguiente celda y el contenido de la siguiente celda no se muestra.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a specific text
TableAdd(TABLE_Customer, "List of customers in California")
// Merges the columns to display the text on all the columns
COL_LastName[1].Merge = mergeNextCell
COL_FirstName[1].Merge = mergeNextCell
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el modo de fusión

`<Current merge mode> = <Element used>.Merge`
---

**`<Current merge mode>: (Constante de tipo Integer)`**

Modo de fusión utilizado en el elemento:


|   |   |
| --- | --- |
| *mergeNextCell* | Fusionar la celda especificada con la celda siguiente (en la misma fila). |
| *mergeNextLine* | Fusionar la celda especificada con la celda encontrada en la siguiente fila. |
| *mergeNextTitle* | Fusionar el título de la columna con el título de la columna siguiente. |
| *mergeNone* | No se realiza ninguna fusión. |



**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento utilizado:

- Celda: &lt;Nombre de columna&gt; [ &lt;Índice Fila&gt;]

- Título de una columna: &lt;Nombre de la columna  





<a name="SYNTAX2"></a>

### Fusionando las células o los títulos de las columnas

`<Element used>.Merge = <Merge mode>`
---

**`<Element used>: (Cadena de caracteres)`**

Nombre del elemento utilizado:

- Celda: &lt;Nombre de columna&gt; [ &lt;Índice Fila&gt;]

- Título de una columna: &lt;Nombre de la columna&gt;




**`<Merge mode>: (Constante de tipo Integer)`**

Uno de los modos de fusión disponibles:


|   |   |
| --- | --- |
| *mergeNextCell* | Fusionar la celda especificada con la celda siguiente (en la misma fila). |
| *mergeNextLine* | Fusionar la celda especificada con la celda encontrada en la siguiente fila.<br>Atención: debe existir la siguiente fila, de lo contrario la operación de fusión no tiene ningún efecto. |
| *mergeNextTitle* | Fusionar el título de la columna con el título de la columna siguiente. |
| *mergeNone* | No se realiza ninguna fusión. |


En todos los casos, el contenido de las celdas no se combina.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Operaciones de fusión en cadena
<a name="operaciones_fusion_cadena_ELTPARAGRAPHE000112"></a>

Si la columna fusionada es la última columna de Visible, la operación de fusión no tiene efecto.
<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000119"></a>

- Las operaciones de fusión se cancelan cuando se realiza un borrado, una inserción o una clasificación en la control: deben ser recreados.

- La operación de fusión está disponible para las celdas y columnas que se encuentran en los controles de Tabla y Vista de Árbol.

- Las constantes para la fusión de las células horizontal y verticalmente no pueden ser associated.

- La selección de una fila que contiene células fusionadas puede producir efectos inesperados.

- Al exportar un control Tabla (a través del menú contextual o con [TablaToExcel](../WDLang1/3074033.md), [TablaToClipboard](../WDLang1/1000017062.md), [TablaToText](../WDLang1/1000017061.md), [TableToWord](../WDLang1/3074036.md), [TableToXML](../WDLang1/3074037.md)), se ignora la fusión de columnas.

- El **Merge** Property sólo se puede utilizar en los controles de Tabla y Tabla TreeView.

- Tiene la capacidad de fusionar un título de columna y mantener las columnas no fusionadas (para mostrar los datos en valor y en porcentaje, por ejemplo)..
	Si el título de la columna contiene el título principal y el título de las subcolumnas ("Contactos" en el primer Line, "Nb" y "%" en el segundo Line por ejemplo):

	- puede que tenga que modificar el título para que los títulos de las subcolumnas estén alineados (adición de caracteres de espacio, por ejemplo).

	- al redimensionar las subcolumnas, el título de las subcolumnas no se redimensionará.

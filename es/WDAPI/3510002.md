## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
WDAPI simplifica el uso de las APIs de Windows en una aplicación WINDEV. WDAPI genera el código y las estructuras de WLanguage que permiten el uso de un determinado Windows API.

WDAPI NO Document el modo de funcionamiento de cada Windows API del sistema!

Sin embargo, WDAPI le ofrece la posibilidad de ejecutar directamente un sitio o una Document (de su elección) correspondiente a la Document ación de las API de Windows. Si se selecciona una función, una estructura o una constante en WDAPI, puede visualizar la página de ayuda correspondiente.

**Observación**: La lista de APIs de Windows suministradas con WDAPI NO es exhaustiva.

**Recordatorio**: WDAPI se puede iniciar desde WDTool: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDTool".

## Obtener ayuda sobre una característica de Windows API

### Configuración de la ayuda de WDAPI a get sobre una función, estructura o constante del API de Windows
<a name="configuracion_ayuda_wdapi_get_sobre_una_funcion_estructura_constante_del_api_windows_ELTPARAGRAPHE000038"></a>

Para configurar WDAPI a la ayuda de get sobre una función, estructura o constante del API de Windows: 

1. Seleccione el panel "Sitios favoritos".

2. Especifique la Address correspondiente al sitio que propone la ayuda sobre el elemento solicitado (función, estructura o constante). Es posible:

	- Seleccione una Address encontrada en la lista.

	- Añadir una nueva Address. Para ello, introduzca la nueva Address y haga clic en "Añadir".







La Address también puede corresponder a:

- un INTERNET Address (por ejemplo: http://www.myfavoritesite.com).

- una ruta completa a una Document (por ejemplo: C:\\Mis documentos\\APIInfo.doc).

- una aplicación (por ejemplo: AASLASH_\\APIStation\\Herramientas\\ApiPlus.exe). 




**Este Address puede abrir directamente el sitio con la ayuda del elemento seleccionado (función, constante o estructura).**

Si el nombre del elemento se utiliza en la Address inicial, el nombre de la función, constante o estructura debe sustituirse por la palabra clave [%FUNCTION%] en la Address inicial. 
Por ejemplo:


```txt
http://www.apifunctions.com/found.asp?function=[%FUNCTION%]
```


Si la Address inicial utiliza la primera letra del elemento (índice alfabético, por ejemplo), la letra debe sustituirse por la palabra clave [%STARTFUNCT%] en la Address inicial.
Por ejemplo:


```txt
http://www.apifunctions.com/found.asp?function=[%STARTFUNCT%]
```

### Obtener ayuda sobre una función, estructura o constante de Windows API
<a name="obtener_ayuda_sobre_una_funcion_estructura_constante_windows_api_ELTPARAGRAPHE000083"></a>

Para la ayuda de get sobre una función, estructura o constante del API de Windows: 

1. Seleccione una función (o una estructura o una Variable) en WDAPI.

2. Haga clic en el enlace "Iniciar el enlace a mi página de ayuda de API".


El sitio seleccionado en el panel "Sitios preferidos" se inicia automáticamente.

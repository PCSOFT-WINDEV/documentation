


## WDAPI: Convertir una declaración de función o estructura C
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
WDAPI se utiliza para convertir una declaración de la función o estructura C en una declaración de la función o estructura WLanguage.. Le permite recuperar fácilmente un código C existente y convertirlo en WLanguage.

**Recordatorio**: WDAPI se puede iniciar desde WDTool: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDTool".

## Convertir una declaración de la función o estructura C en una declaración de la estructura WLanguage
<a name="convertir_una_declaracion_funcion_estructura_una_declaracion_estructura_wlanguage_ELTTEXTE000132"></a>


### Convertir una declaración de la función o estructura C en una declaración de la estructura WLanguage
<a name="convertir_una_declaracion_funcion_estructura_una_declaracion_estructura_wlanguage_ELTPARAGRAPHE000032"></a>

Para convertir una declaración de la función o estructura C en una declaración de la estructura WLanguage:

1. Seleccione el panel "Convertir C en WLanguage".

2. Para convertir una declaración de función C, introduzca el código de esta función en "Prototipo de la función en C"..
	Para convertir una declaración de estructura C, introduzca el código de esta estructura en "Estructura en C".

3. Haga clic en la flecha (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fleche.gif)) que se encuentra debajo del correspondiente control.

4. El código WLanguage correspondiente se genera automáticamente. Este código puede ser copiado en su programa WLanguage (ver el siguiente párrafo).


**Rare case:** Algunas características específicas de relacionado a una estructura o a una función del API de Windows pueden no ser interpretadas correctamente por WDAPI. En este caso, la conversión debe ser realizada por usted mismo.
<a name="NOTE2_2"></a>


### Copiar el código de declaración de la estructura C convertida a WLanguage
<a name="copiar_codigo_declaracion_estructura_convertida_wlanguage_ELTPARAGRAPHE000050"></a>

A copy el código de declaración de la estructura C convertido en WLanguage:

1. Convertir el código.

2. Haga clic en el botón "Copiar" associated con el código de declaración de la sintaxis o estructura de WLanguage.

3. En su proyecto WINDEV, vaya al Process o Event deseado:

	- inicialización del proyecto (recomendado).

	- declaración de variables globales de la ventana.

	- declaración de variables globales de la set de los procedimientos en los que se llama a la función API.




4. En el editor de código, seleccione "Pegar" en el menú contextual (o pulse Ctrl + V).

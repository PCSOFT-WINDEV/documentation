## Presentación
<a name="presentacion_ELTTEXTE000143"></a>
WDAPI simplifica el uso de las APIs de Windows en una aplicación WINDEV. WDAPI genera el código WLanguage y las estructuras necesarias para utilizar un determinado Windows API.

**Atención**: WDAPI no se puede utilizar para encontrar un API de acuerdo con una característica determinada.. Para utilizar WDAPI, debe conocer el nombre de la API a utilizar. 
WDAPI NO Document el uso de cada Windows API del sistema! Sin embargo, WDAPI le ofrece la posibilidad de iniciar directamente un sitio o un Document (elegido por usted mismo) correspondiente a la Document ación de los API s de Windows.

WDAPI no le enseñará a utilizar las 1.600 APIs de Windows, pero le ayudará enormemente en el paso de generación del código.. De hecho, la función API que se seleccionó en WDAPI puede utilizarse en su código.

**Observación**: La lista de APIs de Windows suministradas con WDAPI NO es exhaustiva.

## Iniciar WDAPI
<a name="iniciar_wdapi_ELTTEXTE000167"></a>


### estreno
<a name="estreno_ELTPARAGRAPHE000026"></a>

WDAPI puede iniciarse utilizando uno de los siguientes métodos:

- ejecutando el archivo "WDAPI.EXE" que se encuentra en el directorio de instalación de WINDEV.

- desde el centro de mantenimiento de WINDEV: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDTool".



<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000048"></a>

WDAPI no se puede distribuir con sus aplicaciones. WDAPI no se puede instalar con las aplicaciones desarrolladas con WINDEV.

Se aplican los términos de licencia de uso de WINDEV.

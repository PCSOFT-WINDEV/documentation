## ¿Cómo instalar WDTRAD?
<a name="¿como_instalar_wdtrad_ELTTEXTE000084"></a>
WDTRAD se instala automáticamente junto con WDMSG y/o WDINT.

Para instalar solo WDTRAD, instale y ejecute "InstallWDTRAD.EXE" (ubicado en el paquete de instalación de WDMSG o WDINT).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Observaciones
<a name="observaciones_ELTTEXTE000108"></a>


- El diccionario proporcionado con WDTRAD se instala de forma predeterminada. Para utilizar un diccionario personalizado: 

	1. Utilice WDTRAD para crear y personalizar su diccionario. 

	2. Recupere el diccionario copiando el subdirectorio "Diccionario" del directorio WDTRAD instalación.

	3. Pegue este subdirectorio en el directorio WDTRAD instalación deseado. 




- Si los mensajes de su aplicación son traducidos por una empresa externa, no olvide proporcionar los archivos que contienen los mensajes a traducir: 

	- un archivo de texto, 

	- el archivo de datos HFSQL (archivos .fic, .ndx y .mmo),

	- el archivo wdmsg.




- Una vez finalizada la traducción, se deben recuperar los archivos que contienen los mensajes traducidos. Estos archivos deben ser verificados por WDMSG o WDINT.

- Debe especificar el idioma utilizado la primera vez que inicie WDTRAD. Este idioma puede ser cambiado en cualquier momento en WDTRAD ("?... Idioma").

## Presentación
<a name="presentacion_ELTTEXTE000153"></a>
Las opciones del diccionario WDTRAD se pueden configurar a través de "Editar... Propiedades del diccionario". La ventana que aparece incluye varias pestañas no acoplables. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pestaña "General": Opciones disponibles
<a name="pestana_general_opciones_disponibles_ELTTEXTE000177"></a>
La pestaña "General" sirve para: 

- Estadísticas de traducción de get para el diccionario actual

- realizar operaciones de mantenimiento. 





### Estadísticas de traducción
<a name="estadisticas_traduccion_ELTPARAGRAPHE000023"></a>

Para get las estadísticas de traducción, haga clic en el enlace "Generar estadísticas del diccionario".. Estas estadísticas devuelven el número de recursos que se encuentran en el diccionario. 


### Mantenimiento
<a name="mantenimiento_ELTPARAGRAPHE000032"></a>

Las operaciones de mantenimiento son las siguientes: 

- **Configuración del idioma**: Puede configurar las características de cada idioma: teclado utilizado, símbolos. WDTRAD utiliza la configuración de símbolos cuando se reconstruye una traducción a partir de una traducción existente.. 
	Para configurar las características de un idioma: 

	1. Seleccione el idioma deseado. 

	2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=icone_Proprietes.gif). Las pestañas "Palabra clave" y "Símbolos" sirven para configurar las características específicas del idioma seleccionado.. 




- **Reindexar el diccionario**: El diccionario puede ser reindexado para cada idioma. Esta operación optimiza el acceso a los datos del diccionario. Esta operación de reindexación puede tardar unos minutos. 
	Para reindexar un idioma: 

	1. Seleccione el idioma deseado. 

	2. Haga clic en el botón "Reindex" y valide.

## Ficha "Terminologías personalizadas": Opciones disponibles
<a name="ficha_terminologias_personalizadas_opciones_disponibles_ELTTEXTE000213"></a>
La pestaña "Terminología" permite utilizar las traducciones de set para determinados términos. Las herramientas de traducción automática (por ejemplo, DeepL) utilizarán estas traducciones cuando se encuentren los términos especificados. 

La opción "Sincronizar con Amazon" permite almacenar estas terminologías en Amazon Translate.

## Pestaña "Buscar": Opciones disponibles
<a name="pestana_buscar_opciones_disponibles_ELTTEXTE000237"></a>
La pestaña "Búsqueda" se utiliza para realizar una búsqueda específica en el diccionario. 

Para ello: 

1. Seleccione el idioma de origen y el idioma de destino.

2. Introduzca el texto que desea encontrar en el idioma deseado. 

3. Haga clic en "Buscar". En función de la [opciones especificadas](../WDTrad/3518021.md), los resultados se enumeran y pueden modificarse. 




Observación: El menú contextual de los resultados permite añadir una entrada al diccionario ("Añadir una entrada al diccionario").

## Pestaña "Buscar/Sustituir": Opciones disponibles
<a name="pestana_buscarsustituir_opciones_disponibles_ELTTEXTE000261"></a>
En la pestaña "Buscar / Reemplazar" puede encontrar y reemplazar recursos y traducciones en el diccionario. 

Para ello: 

1. Seleccione el idioma de búsqueda.

2. Introduzca el texto de búsqueda en el idioma especificado. 
	Observación: Puede utilizar los caracteres \* y?. El carácter ~ permite excluir un término de la búsqueda. 

3. Especifique las opciones de búsqueda (mayúsculas/minúsculas, empieza por, etc.).

4. Haga clic en "Buscar". Los resultados se enumeran según las opciones especificadas. 

5. Introduzca el texto de sustitución. 

6. Realice la sustitución: 

	- Haga clic en "Reemplazar" para sustituir la Resource seleccionada. 

	- Haga clic en "Reemplazar todo" para sustituir todos los recursos de la lista.

## Pestaña "Recién añadidos": Opciones disponibles
<a name="pestana_recien_anadidos_opciones_disponibles_ELTTEXTE000285"></a>
La pestaña "Añadido recientemente" muestra los últimos recursos añadidos al diccionario. Puede seleccionar los idiomas deseados y el autor de la traducción. Se muestran los últimos 100 recursos.

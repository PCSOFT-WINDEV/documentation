
## Centro de Seguimiento de Proyectos: Uso de etiquetas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000177"></a>
Las etiquetas se utilizan para asociar las palabras clave a los incidentes en el sistema Centro de Seguimiento de Proyectos. El uso de etiquetas permite filtrar, clasificar o incluso generar estadísticas sobre las distintas incidencias de associated con un proyecto, una necesidad, ... 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000201"></a>


### Gestión de etiquetas
<a name="gestion_etiquetas_ELTPARAGRAPHE000016"></a>

Puede gestionar las etiquetas en la pestaña "Incidencias" de Centro de Seguimiento de Proyectos. 

Para abrir la ventana de gestión de etiquetas en el programa Centro de Seguimiento de Proyectos:

1. Abrir la pestaña "Incidencias". 

2. En la cinta, en el grupo "Gestión", haga clic en "Etiquetas". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0001%201.gif&type=thumb)


3. Se muestra la ventana de gestión de etiquetas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0003.gif&type=thumb)
Esta ventana permite: 

	- Etiquetas set: crear, modificar, etc.

	- ver la distribución de los incidentes por etiqueta. 








### Crear, editar y eliminar una etiqueta
<a name="crear_editar_eliminar_una_etiqueta_ELTPARAGRAPHE000033"></a>

**Para crear una etiqueta**: 

1. Abrir la pestaña "Incidencias". 

2. En la cinta, en el grupo "Gestión", haga clic en "Etiquetas". 

3. En la pestaña "Lista de etiquetas", haga clic en "+".. Se muestra la ventana de creación de etiquetas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0002.gif)
 En esta ventana, debe indicar: 

	- El nombre de la etiqueta (obligatorio). Este nombre se utilizará para identificar la etiqueta. El nombre no debe contener espacios. 

	- La etiqueta description (opcional): Explica las condiciones de uso de la etiqueta. 

	- La visibilidad de la etiqueta: 

		- Si la etiqueta es pública, todos los contribuyentes podrán verla. 

		- Si la etiqueta es privada, sólo el propietario podrá verla. 




- La etiqueta Color: Permite identificar rápidamente los elementos associated con una etiqueta. Este parámetro es obligatorio.  

4. Validar. La nueva etiqueta aparece inmediatamente en la lista de etiquetas. 




**Para editar una etiqueta**: 

1. Abrir la pestaña "Incidencias". 

2. En la cinta, en el grupo "Gestión", haga clic en "Etiquetas". 

3. En la pestaña "Lista de etiquetas", seleccione una etiqueta y haga clic en el botón de modificación: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0003%201.gif). Aparece la ventana de la etiqueta description. 

4. En esta ventana, puede modificar : 

	- Las diferentes características de una etiqueta privada. La etiqueta puede cambiarse a pública. También puede cambiar el propietario de una etiqueta privada (precaución: en este caso, la etiqueta ya no aparecerá en la lista de etiquetas). 

	- Las diferentes características de una etiqueta pública. Sin embargo, una etiqueta pública no puede cambiarse por una etiqueta privada. 




5. Validar. Los cambios se aplican automáticamente.  




**Para borrar una etiqueta**: 

1. Abrir la pestaña "Incidencias". 

2. En la cinta, en el grupo "Gestión", haga clic en "Etiquetas". 

3. En la pestaña "Lista de etiquetas", seleccione una etiqueta y haga clic en "-". 

4. Se solicita una confirmación y se elimina la etiqueta. Si la etiqueta es associated con uno o más incidentes, la asociación también se eliminará. 







### Asociar una etiqueta a un incidente
<a name="asociar_una_etiqueta_incidente_ELTPARAGRAPHE000074"></a>

Existen múltiples formas de asociar una etiqueta a un incidente: 

- desde la ventana de detalles del incidente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0004.gif&type=thumb)


- de la lista de incidencias. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0001.gif&type=thumb)


- desde la ventana de creación de incidentes en WDBal. 
	Observación: Las etiquetas se proponen automáticamente, en función del texto del mensaje correspondiente al incidente.




Observación: Todas las adiciones o supresiones de una etiqueta pública se registran en el historial de incidentes. 




### Filtrado de incidentes mediante etiquetas
<a name="filtrado_incidentes_mediante_etiquetas_ELTPARAGRAPHE000092"></a>

Para filtrar los incidentes con una etiqueta determinada: 

1. Abrir la pestaña "Incidencias". 

2. En la sección "Filtro" Above la lista de incidencias: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0006.gif)


	- Indique la(s) etiqueta(s) que debe(n) tenerse en cuenta. 

- Indique el filtro a utilizar haciendo clic en el enlace.

3. El resultado se muestra inmediatamente. 







### Ver la distribución de los incidentes por etiqueta
<a name="ver_distribucion_los_incidentes_por_etiqueta_ELTPARAGRAPHE000106"></a>

Para ver la distribución de los incidentes por etiqueta: 

1. Abrir la pestaña "Incidencias". 

2. En la cinta, en el grupo "Gestión", haga clic en "Etiquetas". 

3. En la ventana que aparece, vaya a la pestaña "Distribución de incidentes por etiqueta".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0005.gif&type=thumb)
 Esta ventana muestra la distribución de los incidentes por etiqueta, según los criterios de clasificación. Un menú contextual permite realizar las siguientes operaciones: 

	- Ocultar etiqueta. 

	- Modificar la etiqueta. 

	- Mostrar la lista de incidentes. En este caso, se abrirá una nueva pestaña de "Incidentes" en el Centro de Seguimiento de Proyectos, que contendrá únicamente los incidentes que tengan la etiqueta seleccionada. 





Esta ventana permite filtrar las etiquetas mostradas: 

- Asignar etiquetas,

- Autor

- set de incidentes,

- Tipo de incidente,

- Estado, 

- Etiquetas privadas o públicas,

- Incidentes sin etiquetas.

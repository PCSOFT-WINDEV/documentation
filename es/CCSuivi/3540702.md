## ¿Qué es un proyecto?
<a name="¿que_proyecto_ELTTEXTE000163"></a>
Un proyecto corresponde a un proyecto de trabajo. Este proyecto involucra a una o más personas (llamadas colaboradores).

Cada proyecto es con associated:

- colaboradores, 

- tareas a realizar.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Un proyecto de trabajo y una [SPOOL DE SOLICITUDES](../CCSuivi/3540707.md) se crean automáticamente cuando se incluye la opción "Sugerencias, problemas..." de la [Menú automatic (? menú)](../Editeurs/9000070.md) en la aplicación WINDEV. Este proyecto de trabajo y esta bobina de solicitudes son automáticamente associated con la correspondiente aplicación WINDEV.

### Crear un proyecto
<a name="crear_proyecto_ELTPARAGRAPHE000060"></a>

Un proyecto puede ser creado por un colaborador o por un administrador (por Default, el nombre de usuario y la contraseña corresponden a "ADMIN").

Para crear un proyecto: 

1. Despliegue ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Proyectos" para mostrar la lista de proyectos.

2. Haga clic en el botón "+".

3. Introduzca la información solicitada: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ajout_Prj_general.gif)


	- ***denominar***: Nombre del proyecto que se utilizará en los diferentes Centros control para identificar el proyecto.

- ***description***: Breve descripción del proyecto.

- ***Versión***, para los proyectos que requieren varias versiones.

- ***Tipo de aplicación***: Esta información es useful para un proyecto relacionado a un proyecto WINDEV, WEBDEV o WINDEV Mobile. De hecho, esta opción le permite identificar fácilmente el soporte de la aplicación final.. Para los demás proyectos, basta con seleccionar "Notas adicionales";.

- ***Localización*** (para proyectos asociados a proyectos WINDEV, WEBDEV o WINDEV Mobile): Permite abrir rápidamente el proyecto desde el Centro control. El proyecto se puede encontrar en la SCM o se puede acceder a él desde la el equipo (se debe especificar la ruta del proyecto). Esta información es opcional.

- ***Grupo***:  Utilizado para agrupar los proyectos. Todo lo que tienes que hacer es seleccionar el grupo de proyectos associated con el proyecto. Si el grupo solicitado no existe, el botón "+" se utiliza para crear un nuevo grupo de proyectos.. Un grupo de proyectos puede ser creado por los administradores solamente.   

- ***Ruta***, para proyectos asociados a proyectos WINDEV, WEBDEV o WINDEV Mobile: Esta información se utiliza para abrir rápidamente el proyecto desde el Centro control. El proyecto no se modifica directamente.

- ***Bobina***: Lista de tareas futuras associated con el proyecto. Si es necesario, se puede crear un nuevo spool.

- ***Color*** asociado al proyecto: Esta Color servirá para identificar fácilmente el proyecto en el calendario o en la hoja de ruta.

4. Cuando se valida el proyecto, se muestra la lista de colaboradores de associated con este proyecto (véase el párrafo siguiente).



<a name="NOTE2_3"></a>


### Asociar a los colaboradores a un proyecto
<a name="asociar_los_colaboradores_proyecto_ELTPARAGRAPHE000087"></a>

Asociar colaboradores a un proyecto:

1. Despliegue ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Proyectos" para mostrar la lista de proyectos.

2. Seleccione el proyecto solicitado.

3. Haga clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Projet%20-%20HC%20N%B0001%201.gif).

4. Se muestra la lista de colaboradores disponibles. Seleccione el colaborador y transfiéralo a la lista de colaboradores associated con el proyecto. Para cada contribuyente, usted tiene la capacidad de definir si este contribuyente es:

	- Jefe de proyectos. En este caso, el colaborador podrá añadir y eliminar tareas, asignar tareas a otros colaboradores, ....

	- Gerente de desarrollo.

	- Gerente de pruebas.

	- Gestor de documentación.

	- Gestor de bases de datos.

	- Gestor de copias de seguridad.




5. Observaciones:

	- Los equipos de colaboradores también pueden seleccionarse directamente en la pestaña "Equipos".

	- Si los contribuyentes solicitados no se encuentran en la lista, debe [crear un contribuyente](../CCSuivi/3540703.md).




6. Se pueden definir uno o varios jefes de proyecto. Para ello, haga clic en la casilla "Manag" junto al nombre. Atención: un gestor de proyectos no es necesariamente un administrador (y a la inversa).

7. Valide.


**Observación**: Sólo el director del proyecto o un administrador de los Centros control puede asociar a los colaboradores con un proyecto.

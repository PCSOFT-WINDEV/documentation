


## Mantener los archivos de los centros de control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000110"></a>
Al utilizar una base de datos, es posible que tenga que realizar operaciones de mantenimiento en los archivos de datos.. Estas operaciones se utilizan para optimizar el tiempo de acceso a los datos o para liberar las transacciones bloqueadas, por ejemplo.

**¿Cuándo deben utilizarse estas opciones?**
Cuando se produce un problema técnico (corte de energía, etc.), al poner en marcha los Centros de Control, un mensaje puede proponer a Cancel las operaciones bloqueadas o realizar una operación de re-indexación.
Observación: Si la base de datos de los centros de control es bastante importante, se debería realizar una operación de re-indexación de forma regular.

**Atención: Estas operaciones son operaciones de bloqueo: no se puede acceder a los datos de los Centros control durante la operación**.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000134"></a>
Para iniciar el mantenimiento de los archivos que se encuentran en los Centros control: 

1. Seleccione:

	- en la Centro de Seguimiento de Proyectos: expanda ![Menú de Centro de Seguimiento de Proyectos](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Mantenimiento".

	- en WINDEV, WEBDEV o WINDEV Mobile: en la pestaña "Herramientas", en el grupo "Centros de Control", haga clic en "Mantenimiento base de datos ".




2. En la ventana que se visualiza, especifique:

	- la base de datos correspondiente.

	- la operación a realizar:

		- Reindex the database: opción recomendada para las grandes bases de datos

		- Cancel locked transactions: opción recomendada si un mensaje indica que las transacciones están bloqueadas cuando se inicia un Centro control.

		- Migrar repositorio al formato más reciente: opción recomendada para aprovechar todas las nuevas características de la versión actual.

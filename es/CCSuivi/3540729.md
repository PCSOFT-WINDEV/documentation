
## Centro de Seguimiento de Proyectos: Incidencias nuevas/arregladas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000078"></a>
Para mejorar el seguimiento del proyecto, la Centro de Seguimiento de Proyectos muestra la evolución de las incidencias nuevas y fijas en un periodo de tiempo determinado, por uno o varios contribuyentes.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Tendencias de visualización
<a name="tendencias_visualizacion_ELTTEXTE000102"></a>
Para ver la tendencia de las incidencias nuevas y fijas en la Centro de Seguimiento de Proyectos:

1. Abrir la pestaña "Incidencias". 

2. En la cinta de opciones, en el grupo "Gestión", haga clic en "Tendencia".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Tag%20-%20HC%20N%B0001%202.gif&type=thumb)


3. Aparece la ventana de estadísticas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ccsuivi-tendance.gif&type=thumb)
 Esta ventana muestra el número de errores creados y corregidos, con tendencia a lo largo de un periodo de tiempo, según múltiples criterios: 

	- el modo de vista: 

		- "Desarrollo": Incidentes creados o arreglados durante el período de tiempo determinado y asignados a los contribuyentes seleccionados. 

		- "Prueba": Incidentes creados o arreglados durante el período de tiempo dado y cuyos contribuyentes son también los autores. 




- los contribuyentes incluidos en la tendencia. Si no se selecciona ningún colaborador, se incluirán todos los colaboradores. 

- la set de incidentes, si es necesario. Si no se selecciona set, se incluirán todas las incidencias.  

- el período de tiempo que se tiene en cuenta.

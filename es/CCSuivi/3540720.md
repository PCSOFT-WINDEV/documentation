## ¿Cómo crear la base de datos de los Centros control?
<a name="¿como_crear_base_datos_los_centros_control_ELTTEXTE000220"></a>


### Instalación
<a name="instalacion_ELTPARAGRAPHE000022"></a>

Al instalar WINDEV, WEBDEV o WINDEV Mobile, el programa instalación propone:

- crear la base de datos de los Centros de Control. Mediante Default, esta base de datos se creará en el directorio especificado (en el subdirectorio "\\Personal\\Centros control") cuando se inicie WINDEV, WEBDEV o WINDEV Mobile por primera vez. Esta base de datos estará en formato HFSQL Classic por Default.

- compartir una base de datos existente de los Centros de Control.




Si ha elegido los parámetros Default, la base de datos de los Centros control se crea localmente cuando se inicia WINDEV, WEBDEV o WINDEV Mobile por primera vez.
<a name="NOTE2_2"></a>


### Trabajar solo o en equipo
<a name="trabajar_solo_equipo_ELTPARAGRAPHE000034"></a>

**Si está trabajando en su propio**, puede mantener la base de datos de los Centros control localmente. Sin embargo, si se utilizan varios productos, se recomienda compartir esta base de datos local entre los diferentes productos.

**Si usted trabaja en el equipo**, le aconsejamos que comparta la base de datos de los Centros control. Es posible:

- utilizar una base de datos compartida de HFSQL Classic.

- utilizar una base de datos de HFSQL Client/Server.




El tipo de base de datos se configura en las opciones de WINDEV, WEBDEV o WINDEV Mobile: 

- en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile". 

- en la pestaña "Usuario", haga clic en "Parámetros de los Centros de Control".



<a name="NOTE2_3"></a>


### Base de datos HFSQL Classic compartida
<a name="base_datos_hfsql_classic_compartida_ELTPARAGRAPHE000067"></a>

**Para utilizar una base de datos compartida HFSQL Classic,** debes:

- Crear un directorio compartido accesible para todos los desarrolladores. Este directorio debe ser accesible en lectura/escritura para todos los colaboradores.

- Especifique la ruta de este directorio en las opciones de configuración de los Centros control. Todos los desarrolladores pertenecientes al equipo deben tener la misma configuración.




**Recordatorio**: El tipo de base de datos se configura en las opciones de WINDEV, WEBDEV o WINDEV Mobile: 

- en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile". 

- en la pestaña "Usuario", haga clic en "Parámetros de los Centros de Control".




La base de datos se creará automáticamente durante el primer uso. Los diferentes colaboradores tendrán que volver a introducir su nombre de usuario y contraseña si es necesario.

Observación: No se puede recuperar la información introducida por los diferentes promotores en los Centros control utilizados localmente.
<a name="NOTE2_4"></a>


### Base de datos HFSQL Client/Server
<a name="base_datos_hfsql_clientserver_ELTPARAGRAPHE000102"></a>

**Para usar una base de datos de HFSQL Client/Server**, debe:

- Instalar un servidor de HFSQL Client/Server.
	Para instalar HFSQL Client/Server, inicie el paquete de su producto instalación y haga clic en "Instalación de HFSQL Client/Server".
	**Observación**: En WINDEV, el programa instalación de un servidor HFSQL Client/Server está disponible en el subdirectorio "\\Install\\HFSQL server" del directorio instalación de su producto.

- Especifique las características de la base de datos en las opciones de configuración de los Centros control:

	- el nombre del servidor o la IP Address.

	- el nombre de la base de datos HFSQL Client/Server a utilizar. Esta base de datos debe estar vacía (sin archivos) o no encontrada (en este caso, se creará automáticamente).

	- el puerto que se utilizará para la conexión.







Todos los desarrolladores pertenecientes al equipo deben tener la misma configuración.

**Recordatorio**: El tipo de base de datos se configura en las opciones de WINDEV, WEBDEV o WINDEV Mobile: 

- en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile". 

- en la pestaña "Usuario", haga clic en "Parámetros de los Centros de Control".




**Observación**: Si está utilizando una base de datos compartida de HFSQL Classic, esta base de datos se puede migrar a HFSQL Client/Server. Para obtener más información, consulte [Utilización de la base de datos de los Centros control en modo Cliente/Servidor](../CCSuivi/3540712.md).

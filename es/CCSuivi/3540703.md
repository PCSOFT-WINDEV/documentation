<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
Los colaboradores son las personas que participan en uno o varios proyectos.

Para gestionar a los colaboradores, puedes:

- crear un contribuyente.

- modificar un colaborador (por ejemplo, para concederle derechos de administrador)

- borrando a un contribuyente.




Estas operaciones sólo puede realizarlas un administrador (el nombre de usuario y la contraseña de Default son "ADMIN").

Los colaboradores se gestionan desde la lista de colaboradores, visualizada a través de "Gestión... Gestionar los colaboradores"..
Para manipular a los colaboradores, despliegue ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Colaboradores".

## Gestión de colaboradores
<a name="gestion_colaboradores_ELTTEXTE000155"></a>


### Añadir un colaborador
<a name="anadir_colaborador_ELTPARAGRAPHE000032"></a>

Para añadir un colaborador:

1. Amplíe ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Colaboradores" para ver la lista de colaboradores.

2. Haga clic en el botón "Nuevo contribuyente".

3. Especifique la información solicitada en la nueva ventana del contribuyente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ajout_Intervenant.gif&type=thumb)
 La información principal es la siguiente:

	- Nombre de usuario y contraseña: permite al colaborador conectarse cuando se inicia el Centro control.

	- "Administrator" rights are granted to this contributor: permite conceder derechos de administrador a un colaborador. Sólo un administrador puede acceder a esta opción. Esta opción es Grayed para los demás usuarios.

	- tipo de contribuyente: es un colaborador permanente, puede trabajar fuera de los locales o procede de una empresa externa. 




4. Valide.




**Observación**: Una vez que un usuario ha iniciado sesión en la ventana de inicio de sesión de WINDEV, WEBDEV o WINDEV Mobile, puede añadirse automáticamente.
<a name="NOTE2_2"></a>


### Modificación de un colaborador
<a name="modificacion_colaborador_ELTPARAGRAPHE000057"></a>

Para modificar a un colaborador:

1. Amplíe ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Colaboradores" para ver la lista de colaboradores.

2. Seleccione el contribuidor que desea modificar.

3. Haga clic en el botón "Mostrar el contribuyente".

4. Modificar la información solicitada.

5. Valide.




**Observación**: Sólo un administrador o el colaborador puede modificar un ticket.
<a name="NOTE2_3"></a>


### Eliminación de un colaborador
<a name="eliminacion_colaborador_ELTPARAGRAPHE000074"></a>

To delete a contributor:

1. Amplíe ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif) y seleccione "Colaboradores" para ver la lista de colaboradores.

2. Seleccione el contribuyente que desea eliminar.

3. Haga clic en el botón "Eliminar el contribuyente".




**Observación**: **Solo un administrador puede eliminar a un colaborador**.

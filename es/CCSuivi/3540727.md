
## Centro de Seguimiento de Proyectos: Seguimiento del tiempo
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
El tiempo invertido puede agregarse a una lsita de: 

- las tareas de un requisito,

- múltiples requisitos, etc.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000120"></a>
Para controlar el tiempo dedicado a una tarea: 

1. Abra o cree una pestaña de "Tareas". 

2. En la cinta de opciones, en el grupo "Gestión", haga clic en "Seguimiento del tiempo". 

3. En la ventana que aparece, haga clic en "Agregar tareas". A continuación, elija las tareas que desea incluir: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCSuivi_TimeTracking%20-%20HC%20N%B0001.gif&type=thumb)


	- Tareas de una lista de requisitos: en este caso, seleccione la lista de requisitos mediante el botón "...".

- Tareas de un proyecto: en este caso, seleccione un proyecto mediante el botón "...". 

- Tareas de un requisito: en este caso, seleccione la lista de requisitos y, a continuación, el requisito deseado mediante el botón "...". 

- Selección de tareas: en este caso, seleccione las tareas deseadas mediante el botón "...". 

- Tareas de uno o varios colaboradores: en este caso, seleccione los colaboradores mediante el botón "...". 

- El tipo de tareas a tener en cuenta. 

4. La lista de requisitos y tareas seleccionadas se muestra en la ventana, con un cálculo del tiempo empleado en cada tarea. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCSuivi_TimeTracking%20-%20HC%20N%B0002.gif&type=thumb)





La ventana de seguimiento del tiempo permite: 

- mostrar el resultado proponiendo hacer el cálculo sobre un período determinado (opción "Calculate the time spent over a given period"). 

- ver los detalles del tiempo dedicado a las tareas de un requisito (botón "Expand / collapse all"). 

- imprimir o exportar la lista de tareas. 




Observación: No se almacenan datos. Los datos se eliminan cada vez que se abre o se cierra la ventana de seguimiento del tiempo.

## ReplicSynchro y ReplicAdmin
<a name="replicsynchro_replicadmin_ELTTEXTE000066"></a>
ReplicSynchro y ReplicAdmin se utilizan para sincronizar los datos de la aplicación:

- [ReplicSincro](../ReplicSynchro/3526551.md) sólo se puede utilizar para realizar la sincronización. Esta herramienta debe utilizarse para realizar una simple sincronización de datos.

- [ReplicAdmin](../ReplicSynchro/3526552.md) se utiliza para configurar las sincronizaciones y realizar la sincronización. Es el administrador de replicación.

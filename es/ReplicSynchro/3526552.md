## ReplicSynchro y ReplicAdmin
<a name="replicsynchro_replicadmin_ELTTEXTE000102"></a>
ReplicSynchro y ReplicAdmin se utilizan para sincronizar los datos de la aplicación:

- **ReplicSynchro** solo se puede usar para realizar la sincronización. Esta herramienta debe utilizarse para realizar una simple sincronización de datos.

- **ReplicAdmin** se utiliza para configurar las sincronizaciones y realizar la sincronización.. Es el administrador de replicación.

## ReplicAdmin
<a name="replicadmin_ELTTEXTE000126"></a>


### Iniciando ReplicAdmin
<a name="iniciando_replicadmin_ELTPARAGRAPHE000021"></a>

ReplicAdmin puede ejecutarse directamente o desde ReplicSynchro.
<a name="NOTE2_2"></a>


### Uso de ReplicAdmin
<a name="uso_replicadmin_ELTPARAGRAPHE000028"></a>

ReplicAdmin lista las diferentes sincronizaciones disponibles en el ordenador actual, con la fecha de la última sincronización.

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Replication.gif) Icon se utiliza para iniciar la replicación seleccionada.

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Connexion_directe.gif) / ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Connexion_directe1.gif) Icon se utiliza para pasar de online a offline.

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Cr%E9er_cpt.gif) Icon se utiliza para añadir una nueva réplica.

La ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Supprimer.gif) Icon se utiliza para eliminar la réplica seleccionada de la lista.
<a name="NOTE2_3"></a>


### Condición de uso
<a name="condicion_uso_ELTPARAGRAPHE000053"></a>

ReplicAdmin puede ser redistribuido.

## ReplicSynchro y ReplicAdmin
<a name="replicsynchro_replicadmin_ELTTEXTE000158"></a>
ReplicSynchro y ReplicAdmin se utilizan para sincronizar los datos de la aplicación:

- **ReplicSynchro** sólo se puede utilizar para realizar la sincronización. Esta herramienta debe utilizarse para realizar una simple sincronización de datos.

- **ReplicAdmin** se utiliza para configurar las sincronizaciones y realizar la sincronización. Es el administrador de replicación.

## ReplicSincro
<a name="replicsincro_ELTTEXTE000182"></a>


### Iniciando ReplicSynchro
<a name="iniciando_replicsynchro_ELTPARAGRAPHE000021"></a>

ReplicSynchro se puede ejecutar a través de "Iniciar... Herramientas PC SOFT... ReplicSynchro".
<a name="NOTE2_2"></a>


### Uso de ReplicSynchro
<a name="uso_replicsynchro_ELTPARAGRAPHE000028"></a>

ReplicSynchro lista las diferentes replicaciones disponibles en el ordenador actual, con la fecha de la última sincronización.

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Replication.gif) Icon se utiliza para iniciar la replicación seleccionada.

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Admin.gif) Icon se utiliza para iniciar el administrador de replicación.
<a name="NOTE2_3"></a>

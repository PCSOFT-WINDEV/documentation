
## Administrador SaaS
			



## Presentación
<a name="presentacion_ELTTEXTE000094"></a>
El administrador de SaaS es un sitio Web utilizado para configurar y gestionar los sitios SaaS desplegados en un servidor de aplicaciones WEBDEV.

Sus principales características son:

- [Administrando los sitios SaaS associated con un servidor de aplicaciones](../SaaSAdminWeb/1600003.md).

- [Gestionar los servicios y fijar los precios de associated con los sitios](../SaaSAdminWeb/1600006.md).

- [Gestión de las cuentas de los clientes y gestión de sus usuarios y sus suscripciones](../SaaSAdminWeb/1600005.md).

- Ver los informes utilizados para la facturación.




Para obtener más información, consulte:

- [SaaS](../WDLang3/9500213.md).

- [Desarrollo de aplicaciones SaaS con WEBDEV](../WDLang3/9500214.md).








## Uso
<a name="uso_ELTTEXTE000118"></a>
Para acceder al administrador SaaS, debe conectarse al siguiente Address:


```txt
http://<server>/WDAdminSaaS270
```
o


```txt
http://<server>/WD270AWP/WD270Awp.exe/CONNECT/WDAdminSaaS270
```


donde &lt;servidor&gt; representa el Address (nombre o IP Address) del Servidor de Aplicación WEBDEV.

Para utilizar el sitio del administrador de SaaS, debe autenticarse con un nombre de usuario y una contraseña.

De forma predeterminada, el nombre de usuario "**admin**" (con la contraseña **admin**) puede utilizarse una vez instalado el Servidor de Aplicaciones WEBDEV.

**Observación**: La contraseña de la cuenta "admin" debe modificarse lo antes posible para evitar el acceso de personas no autorizadas.. De hecho, la cuenta "admin" tiene todos los derechos de configuración SaaS en el servidor de aplicaciones.

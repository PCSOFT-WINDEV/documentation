


## Extracción de mensajes añadidos o modificados desde la última extracción
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000218"></a>
Un proyecto ya fue creado y traducido.

Ha añadido nuevos elementos (ventanas, páginas, informes, etc.) y/o ha modificado elementos existentes. 
En este caso, no es necesario extraer todos los mensajes.: simplemente extraer los mensajes que corresponden a nuevos elementos y/o elementos modificados. 
Puede extraer los mensajes añadidos o modificados: el nuevo archivo de extracción contendrá todos los mensajes añadidos y modificados desde la última extracción.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000242"></a>


### Extracción de mensajes añadidos o modificados desde la última extracción
<a name="extraccion_mensajes_anadidos_modificados_desde_ultima_extraccion_ELTPARAGRAPHE000026"></a>

Para extraer mensajes añadidos o modificados desde la última extracción:

1. Abra su proyecto en WINDEV, WEBDEV o WINDEV Mobile: en la pestaña "Inicio", en el grupo "General", despliegue "Abrir" y seleccione "Abrir un proyecto".

2. Iniciar WDMSG: en la pestaña "Proyecto", en el grupo "Traducir", despliegue "Traducir" y seleccione "Extraer mensajes".

3. Seleccione los idiomas de los mensajes a extraer. Estas lenguas corresponden a las lenguas ya disponibles en el proyecto, para ser traducidas a otras lenguas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant%20WDMSG%20-%20HC%20N%B0011.gif&type=thumb)
 Las lenguas propuestas son las definidas en la pestaña "Lenguas" del proyecto description. 
	Observaciones:

	- También se propone el nombre de idiomas personalizados. 

	- Para abrir la descripción del proyecto, en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

	- Para seleccionar varios idiomas de extracción, utilice Ctrl y/o Mayúsculas.




4. Pase a la etapa siguiente del asistente.

5. Si ha seleccionado varios idiomas, especifique el idioma de reference, es decir, el idioma utilizado durante la fase de desarrollo. Los mensajes de este idioma reference deben ser traducidos a otros idiomas. 

6. Pase a la etapa siguiente del asistente.  

7. Seleccione los diferentes elementos (ventanas, páginas, informes...) cuyos mensajes (leyenda, mensaje de código...) deben ser extraídos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant%20WDMSG%20-%20HC%20N%B0012.gif&type=thumb)
 Remark: Si los elementos de su proyecto están agrupados por carpetas personalizadas, los elementos propuestos por WDMSG también se agruparán por carpetas personalizadas.. 

8. Pase a la etapa siguiente.

9. Seleccione el tipo de archivo a generar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant%20WDMSG%20-%20HC%20N%B0003.gif&type=thumb)


	- Archivo WDMSG. Este tipo de archivo sólo puede utilizarse para traducir texto con WDTRAD. 

- Archivo HFSQL. Este tipo de archivo sólo puede utilizarse para traducir texto con WDTRAD.  

- Archivo de texto. Este tipo de archivo se puede utilizar para realizar una traducción con WDTRAD o con otro software de traducción.. 

10. Pase a la etapa siguiente.

11. Seleccione los parámetros del archivo de resultados, creado por la extracción (ver [Detalles de los parámetros](#NOTE2_2)).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant%20WDMSG%20-%20HC%20N%B0013.gif&type=thumb)


12. Pase a la etapa siguiente.

13. Seleccione la opción "Extraer solo los mensajes modificados o agregados".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant%20WDMSG%20-%20HC%20N%B0014.gif&type=thumb)


14. Especifique el archivo de extracción inicial (mensajes no traducidos ni corregidos) y el nuevo archivo de extracción que sólo contendrá los mensajes nuevos o modificados..
	El archivo de extracción se crea en el directorio seleccionado (directorio de proyecto de Default).
	Este archivo puede traducirse con un software de traducción específico (sólo archivo de texto) o con [WDTRAD](../WDTrad/3518010.md), la herramienta de traducción incluida en WDMSG. 

15. Valide.

### Parámetros de extracción
<a name="parametros_extraccion_ELTPARAGRAPHE000114"></a>

Los parámetros de extracción comunes a todos los tipos de archivos de extracción son los siguientes:

- **Keep the hotkey indicator ("&") in the extracted messages**: este parámetro permite incluir el carácter "&" en las leyendas de las control para especificar la tecla de acceso directo de control.
	Recordatorio: Esta tecla de acceso directo se utiliza para posicionarse directamente en el control en tiempo de ejecución pulsando Alt + Letra. En la leyenda de control, cualquier letra precedida por el carácter "&" se convierte automáticamente en la "tecla caliente" de control.

- **Transforma las cadenas codificadas en cadenas multilingües**: En el código se encuentran dos tipos de cadenas de caracteres:

	- *cadenas de caracteres multilingües*. Junto a estas cadenas aparece una Icon:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MessageMultilangue.gif)


	- *cadenas no multilingües ("codificadas")*. Estas cadenas van entre comillas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MessageNonMultilangue.gif)

- **Convertir a formato OEM (ANSI por defecto)**: si esta opción está activada, el archivo de extracción utilizará caracteres OEM (carácter DOS set, utilizable con un software DOS). En caso contrario, se utilizarán caracteres ANSI (carácter Windows set, utilizable con un software Windows).




También puede definir parámetros específicos para su propio software de traducción (véase el siguiente párrafo).
<a name="NOTE2_3"></a>


### Parámetros de extracción para un software de traducción
<a name="parametros_extraccion_para_software_traduccion_ELTPARAGRAPHE000137"></a>

Los siguientes parámetros no son necesarios si utiliza [WDTRAD](../WDTrad/3518010.md) para traducir los mensajes que se encuentran en su proyecto.

Si utiliza otro programa de traducción, debe definir parámetros específicos (estos parámetros pueden diferir según el programa utilizado).. Los parámetros específicos del software de traducción utilizado son los siguientes:

- **Agregar un delimitador para las secciones de texto que no se deben traducir**: con algunos programas de traducción, secciones de texto pueden ser ignoradas en el Document para traducir. Por ejemplo, los comentarios generados por WDMSG pueden ser ignorados. Para ello, las secciones de texto a ignorar deben estar encerradas en delimitadores (específicos del software).
	Si se activa esta opción, todas las secciones de texto específicas del formato del archivo de extracción se encerrarán en delimitadores (SSTRX de Default) y no se traducirán.
	Para más detalles, consulte la documentación sobre el software de traducción.

- **Agregar un espacio antes del delimitador**: con algún software de traducción, se debe añadir un espacio antes del delimitador. 
	Si esta opción está activada, se añade un carácter de espacio antes del delimitador.

- **Agregar un espacio después del delimitador**: con algún software de traducción, se debe añadir un espacio después del delimitador. 
	Si esta opción está activada, se añade un carácter de espacio después del delimitador.

- **Agregar información adicional como comentario**: si esta opción está activada, las líneas de comment se añadirán al archivo de extracción creado. Estas líneas se utilizan para averiguar el origen de los mensajes (mensajes associated con un control, una ventana, ...).

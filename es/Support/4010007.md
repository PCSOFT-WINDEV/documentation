## Enviando un Email al Soporte Técnico
<a name="enviando_email_soporte_tecnico_ELTTEXTE000093"></a>
Para enviar un Email al Soporte Técnico, debe utilizar necesariamente la herramienta denominada TechSupportRequest que se utiliza para normalizar el formato de los Email s transmitidos al Soporte Técnico Gratuito y para acortar el tiempo de respuesta.

Para obtener más información, consulte [¿Cómo ponerse en contacto con el Soporte Técnico Gratuito](../Support/4010005.md).

## Envío de una carpeta al soporte técnico
<a name="envio_una_carpeta_soporte_tecnico_ELTTEXTE000117"></a>
Si un representante de Soporte Técnico le pide que envíe algunos elementos, siga estos pasos:

- <u><u><u><u>aislar</u></u></u></u> el problema, 

- no enviar todo un proyecto, sino *únicamente* las ventanas o páginas necesarias,

- proporcionan una versión modificable de estos elementos,

- describen las operaciones que se deben realizar con estos elementos.




Cada conjunto de elementos debe incluir una description clara para conocer el propósito de la solicitud, e incluir la información necesaria para que la Process la solicite (serie detallada de acciones para reproducir el problema).

El número de serie del producto debe estar claramente indicado en todos los documentos enviados al Soporte Técnico Gratuito.

**Importante**: Desafortunadamente, las solicitudes sin una description no pueden ser procesadas.

*En cualquier caso, por favor, evite enviar la solicitud completa!*

## Crear una copia de seguridad para el Soporte Técnico
<a name="crear_una_copia_seguridad_para_soporte_tecnico_ELTTEXTE000141"></a>
Para enviar elementos al Soporte Técnico a través de medios físicos o a través de la INTERNET, utilice la herramienta de respaldo de la aplicación incluida en WINDEV: en la pestaña "Proyecto", en el grupo "Otras acciones", despliegue "Copia seg./Restaurar" y seleccione "Copia de seguridad del proyecto".

Compruebe si todos los elementos de aplicación necesarios para reproducir el problema están incluidos en la copia de seguridad.

Si es necesario, desmarque los elementos de aplicación que no son necesarios para reproducir el problema.

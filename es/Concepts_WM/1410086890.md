<a name="NOTE1_1"></a>
WINDEV Mobile cubre todo el ciclo de desarrollo de las aplicaciones:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P10-Cycle%20de%20d%E9veloppement%20d'une%20application.gif)
**Detalles de las diferentes etapas**:

**Etapa de diseño**: Una aplicación puede diseñarse a partir de requisitos específicos, un modelado UML de los procesos o incluso de archivos de datos existentes.

**Etapa de desarrollo**: El proyecto y el análisis se crean a través de asistentes específicos. La aplicación puede desarrollarse en modo RAD (Rapid Application Development) con generación automática de código e interfaz; o puede resultar de la creación manual de los diferentes elementos del proyecto.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P11-Cycle%20de%20d%E9veloppement%20d'une%20application-phases.gif)
**Etapa de pruebas y generación**: WINDEV Mobile incluye varias herramientas de prueba automatizadas para construir aplicaciones fiables y asegurarse de que no haya regresiones entre las etapas de desarrollo.

**Etapa de implementación**: Una aplicación WINDEV Mobile puede implementarse de diferentes maneras: directamente en el dispositivo móvil, mediante una descarga, a través de Play Store, MarketPlace, App Store o Windows Store. En cualquier caso, los archivos de datos HFSQL (si los hay) se actualizarán automáticamente en función de los cambios realizados en la aplicación.

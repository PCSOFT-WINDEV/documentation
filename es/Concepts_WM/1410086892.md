<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
La interfaz de usuario (UI) de una aplicación se compone principalmente de ventanas.

WINDEV Mobile incluye un editor de ventanas avanzado que permite crear fácil y rápidamente diferentes tipos de UI para todo tipo de plataformas. 

Las siguientes características permiten crear fácilmente aplicaciones intuitivas y fáciles de usar:

- controles potentes.

- anclajes para que la UI se adapte automáticamente al tamaño de la pantalla. Este mecanismo es útil para el desarrollo de aplicaciones móviles porque permite adaptarse a diferentes resoluciones de pantalla.

- un sistema de compilación de la UI con detección de errores (títulos vacíos, texto sin traducir, superposicionamiento, etc.).

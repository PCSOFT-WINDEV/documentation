
## 1. Proyecto y análisis

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El desarrollo de una **Aplicación** móvil (Android, iOS, etc.) con WINDEV Mobile se basa en dos elementos principales: el Proyecto y el Análisis.

Un **Proyecto WINDEV Mobile** es un conjunto de elementos (ventanas, controles, clases, componentes, etc.) que, en conjunto, permiten crear una aplicación.

Un **análisis WINDEV Mobile** es un esquema de la base de datos utilizada por la aplicación. El análisis agrupa las diferentes tablas (también llamadas archivos de datos) utilizadas por la aplicación.

Una aplicación se construye a partir de un proyecto.

En la mayoría de los casos, un proyecto está asociado a un análisis.

Un análisis puede estar asociado a uno o más proyectos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P9-Projet-et-Analyse.png)


## 1. Cuadro de mando del proyecto

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El cuadro de mando del proyecto es un elemento esencial para la gestión de proyectos WINDEV Mobile. Muestra una visión global y detallada del progreso del proyecto.

El cuadro de mando incluye varios indicadores sobre el contenido del proyecto: 

- estadísticas del proyecto, 

- incidentes,

- tareas,

- estado de las pruebas automatizadas,

- resultado de diferentes auditorías,

- lista de elementos extraídos del SCM (Administrador de Código Fuente),

- resultado de los planes de acción (integración continua), etc.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Tableau%20de%20bord.gif&type=thumb)





Los elementos del cuadro de mando se muestran como Widgets. Estos Widgets se pueden configurar, mover, activar, desactivar, etc. También es posible agregar nuevos indicadores. 

La configuración del cuadro de mando se guarda para cada usuario. La configuración del cuadro de mando es la misma para todos los proyectos de un mismo usuario.

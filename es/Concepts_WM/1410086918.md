
## 4. HFSQL Classic

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
HFSQL Classic es el formato de base de datos proporcionado con WINDEV Mobile. Este formato es compatible con WINDEV, WINDEV Mobile y WEBDEV.

Se trata de un sistema de gestión de bases de datos relaciones (DBMS) de libre distribución.

Este formato se puede utilizar en iOS, Android y aplicaciones de la plataforma universal de Windows.

Este formato es idéntico al de HFSQL Classic en WINDEV y WEBDEV (archivo ".WDD", archivos de datos, etc.).

Sin embargo, dado que el tamaño disponible en los dispositivos móviles es limitado, HFSQL Classic no admite las siguientes operaciones:

- transacciones.

- registros.

- bloqueos para archivos y registros.

- archivos en formato Hyper File 5.5.

## Manipular una base de datos HFSQL Classic desde un dispositivo móvil
<a name="manipular_una_base_datos_hfsql_classic_desde_dispositivo_movil_ELTTEXTE000135"></a>
Una base de datos HFSQL Classic es un conjunto de archivos ".FIC", ".NDX" y ".MMO".

## Instalación mediante un archivo APK (Android)
<a name="instalacion_mediante_archivo_apk_android_ELTTEXTE000189"></a>
Esta instalación consiste en:

- generar la aplicación en el equipo de desarrollo.

- firmar el archivo APK digitalmente.
	 Nota: se puede usar una clave autofirmada.

- copiar el archivo APK en un dispositivo Android.

- ejecutar el archivo APK en el dispositivo Android. Esta acción inicia la instalación de la aplicación.




Para ejecutar la aplicación, haga clic en el ícono correspondiente en el menú "Todos los programas".

<a name="NOTE2_1"></a>


## Instalación mediante Google Play (Android)
<a name="instalacion_mediante_google_play_android_ELTTEXTE000213"></a>
Google Play es una aplicación de Google. Google Play propone aplicaciones Android que pueden comprarse o descargarse a través de una interfaz única en los dispositivos Android.

Esta instalación consiste en:

- generar la aplicación en el equipo de desarrollo.

- firmar el archivo APK digitalmente. Para publicar aplicaciones en Google Play, se recomienda utilizar una clave real firmada por una autoridad de confianza reconocida.

- subir el archivo APK a la página Web de Google Play. 
	Nota: debe registrarse previamente en el sitio.


Los usuarios solo tienen que instalar la aplicación deseada directamente desde Google Play.

Para ejecutar la aplicación, selecciónela entre las aplicaciones instaladas.

<a name="NOTE3_1"></a>


## Instalación mediante App Store (iOS)
<a name="instalacion_mediante_app_store_ios_ELTTEXTE000237"></a>
App Store es una aplicación de Apple. App Store propone aplicaciones para iPhone y iPad que pueden comprarse o descargarse a través de una interfaz única en los dispositivos de Apple.

Esta instalación consiste en:

- generar la aplicación en el equipo de desarrollo.

- preparar las imágenes necesarias para la implementación. 

- compilar la aplicación en Xcode. 

- guardar la aplicación en "iTunes Connect". 

- compilar la aplicación en Xcode. 

- subir la aplicación a App Store. 


Los usuarios solo tienen que instalar la aplicación deseada directamente desde App Store.

Para ejecutar la aplicación, selecciónela entre las aplicaciones instaladas.

## Instalación mediante una red ad hoc o una red interna (iOS)
<a name="instalacion_mediante_una_red_hoc_una_red_interna_ios_ELTTEXTE000261"></a>
Este tipo de implementación permite distribuir la aplicación en un dispositivo iPhone o iPad. 

La mayoría de las operaciones se realizan en la Web, luego en el "Centro para desarrolladores de Apple" y después en el compilador Xcode en un Mac.

Dependiendo de su licencia de desarrollo de iOS, puede utilizar uno de los siguientes modos de instalación: 

- una red ad hoc, para instalar la aplicación en varios dispositivos (hasta 100).

- una red interna, para instalar una aplicación en todos los dispositivos que pertenecen a una empresa. La aplicación se distribuirá a través de un servidor Web privado de la empresa o a través de iTunes.

## Instalación mediante la tienda privada de iOS y Android
<a name="instalacion_mediante_tienda_privada_ios_android_ELTTEXTE000285"></a>
Este tipo de implementación permite distribuir la aplicación en dispositivos iPhone, iPad o Android. 

Este tipo de instalación consiste en: 

- Configurar el servidor que alojará el programa de instalación de la aplicación. 

- Habilitar la tienda privada en la aplicación. 

- Generar la aplicación. 

- Subir la aplicación a la tienda privada. 


Los usuarios solo tienen que instalar la aplicación deseada mediante el enlace o el código de barras correspondiente directamente desde sus dispositivos.

## Instalación mediante Microsoft Store (aplicación universal de Windows)
<a name="instalacion_mediante_microsoft_store_aplicacion_universal_windows_ELTTEXTE000309"></a>
Microsoft Store es una aplicación de Microsoft. Cuenta con aplicaciones universales de Windows que pueden comprarse o descargarse a través de una interfaz única en dispositivos Windows.

Esta instalación consiste en:

- generar la aplicación en el equipo de desarrollo.

- subir la aplicación al sitio Web de Microsoft Store. 
	Nota: debe registrarse previamente en el sitio.


Los usuarios solo tienen que instalar la aplicación deseada directamente desde Microsoft Store (dispositivos con Windows 10, tabletas, PC o teléfonos).

Para ejecutar la aplicación, selecciónela entre las aplicaciones instaladas.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile incluye varias funciones de comunicación para diferentes campos.

Estas funciones permiten:

- gestionar correos electrónicos (protocolos POP3, IMAP y SMTP),

- transferir archivos a través de FTP,

- realizar solicitudes HTTP y HTTPS,

- usar la telefonía,

- realizar solicitudes SOAP (para acceder a Webservices SOAP, J2EE o .Net),

- gestionar SMS,

- utilizar sockets TCP (con uso automático del protocolo SSL) y sockets UDP,

- usar sockets Infrarrojos o Bluetooth.




La disponibilidad de estas funciones depende de las características del dispositivo que ejecuta la aplicación.


## ¿Cómo descubrir WINDEV Mobile?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile es un Entorno de Desarrollo Integrado completo para aplicaciones Android, iPhone, iPad y Universal Windows 10 (teléfono, tabletas y PC). Incluye todas las herramientas necesarias para diseñar y desarrollar aplicaciones. 

**Para aprender a utilizar WINDEV Mobile de forma rápida y eficaz, le recomendamos seguir los siguientes pasos**:


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif) | Leer los "Conceptos" (en línea o en la documentación física).<br>Esta guía presenta los principales conceptos para crear una aplicación WINDEV Mobile. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.gif) | Seguir el "Tutorial" (tutorial en línea + ejercicios)<br>El tutorial proporciona un primer enfoque "práctico" de WINDEV Mobile. Podrá familiarizarse con los editores de WINDEV Mobile. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif) | Probar los ejemplos<br>Pruebe los diferentes ejemplos de WINDEV Mobile en los campos que más le interesan.  |


La ayuda en línea, disponible en [https://ayuda.windev.es](https://ayuda.windev.es) o instalada con WINDEV Mobile, permite encontrar rápidamente la sintaxis de una función WLanguage, obtener ayuda sobre la interfaz, etc. Para cada tema de programación, encontrará una descripción de la funcionalidad asociada y la lista de funciones WLanguage correspondientes.

Observación: Si hay diferencias entre los manuales de usuario y la ayuda en línea, siga las instrucciones de la ayuda en línea.

**Esperamos que disfrute desarrollar con WINDEV Mobile**.

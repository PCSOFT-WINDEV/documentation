<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los datos manipulados por una aplicación WINDEV Mobile pueden provenir de:

- una base de datos HFSQL Classic o HFSQL Client/Server (archivos ".Fic").

- archivos ".INI".

- archivos de texto (ANSI o UNICODE).

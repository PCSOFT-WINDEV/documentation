


## 8. Componente externo
			



| [Página anterior](../Concepts_WM/1410086909.md) | [Tabla de contenido](../Concepts_WM/1410086964.md) | [Página siguiente](../Concepts_WM/1410086910.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un componente externo es un conjunto de elementos de WINDEV Mobile: ventanas, reportes, análisis, etc. Este conjunto de elementos realiza una operación específica. Por ejemplo, un componente externo puede corresponder a una de las siguientes acciones:

- Envío de mensajes de texto,

- Envío de correos electrónicos,

- ...




Un componente externo de WINDEV Mobile puede distribuirse a otros desarrolladores de WINDEV Mobile (gratis o pago). Los desarrolladores podrán incluir fácilmente las funcionalidades del componente externo en las aplicaciones. El componente externo se incluirá en la aplicación y se distribuirá con la aplicación.

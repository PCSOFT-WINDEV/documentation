
## 3. Configuración de proyecto

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las configuraciones de proyecto permiten adaptar el mismo proyecto a distintas plataformas.

A partir del mismo proyecto, es posible crear:

- aplicaciones con diferentes elementos, nombres, etc.

- diferentes componentes externos o internos,

- aplicaciones multiplataforma.




Puede trabajar en una configuración específica en cualquier momento: los elementos que no pertenecen a la configuración aparecen en gris en el panel "Explorador de proyectos".
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P31-Configuration%20du%20projet.gif)


La generación múltiple permite generar todas las configuraciones del proyecto (o algunas de ellas) en una sola operación.

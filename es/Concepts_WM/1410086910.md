<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile le permite generar muchos otros tipos de proyectos.


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0003%201.gif) | **Aplicaciones Android**<br><br>WINDEV Mobile permite generar aplicaciones para Android. Estas aplicaciones se pueden ejecutar en smartphones, tabletas y dispositivos ultraportátiles que utilicen este sistema operativo (versión 5.0 y posterior). Estas aplicaciones se pueden publicar en Play Store. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0003%202.gif) | **Aplicaciones iOS**<br><br>WINDEV Mobile permite generar aplicaciones para iOS (a partir de iOS 9). Estas aplicaciones se pueden ejecutar en iPhones y iPads. Además, se pueden publicar en Apple Store. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%202.gif) | **Universal Windows 10 App**<br><br>WINDEV Mobile permite generar aplicaciones para tabletas y teléfonos con Windows 10.<br>Estas aplicaciones pueden iniciarse en modo de mosaico en tabletas Windows 10. Además, se pueden publicar en Windows Store. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%201.gif) | **Webservices SOAP o REST**<br><br>Un Webservice (también llamado servicio Web XML) puede generarse a partir de un proyecto WINDEV Mobile. Un Webservice utiliza un conjunto de funciones (uno o varios conjuntos de procedimientos) y los hace accesibles a través de la Web (o de una red privada) mediante los protocolos HTTP, SOAP y REST.<br><br>Observación: Un Webservice debe implementarse en un WEBDEV Application Server para ser utilizado. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mode%20generation%20Mobile%20-%20HC%20N%B0001%203.gif) | **Componentes externos**<br><br>Los componentes externos son fragmentos de aplicaciones que permiten compartir una o varias funcionalidades específicas entre varias aplicaciones. Un componente generado en WINDEV también puede utilizarse en un proyecto WEBDEV o WINDEV Mobile. |


## 5. HFSQL Client/Server

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una aplicación WINDEV Mobile también puede utilizar una base de datos HFSQL en modo Client/Server.

Las características del modo Client/Server son las siguientes:

- Una aplicación HFSQL Client/Server puede ejecutarse en diferentes dispositivos móviles (equipos cliente).

- Los archivos de datos se encuentran en un servidor. Solo el servidor accede físicamente a los archivos de datos.

- Todos los procesos (consultas, lectura y escritura de registros en un archivo de datos, etc.) se realizan en el servidor.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WM%20HF%20Client%20serveur.gif)


HFSQL Client/Server puede utilizarse en iPhone, iPad, Android y la plataforma universal de Windows.

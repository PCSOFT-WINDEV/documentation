


## 12. Modelos de controles

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile permite crear modelos de controles. Un modelo de controles es un conjunto de controles que se pueden reutilizar en varias ventanas.

Los cambios realizados en un modelo de controles se aplican automáticamente a todas las ventanas que lo utilizan.

Un modelo de controles permite:

- agrupar un conjunto de controles para un propósito específico.

- hacer que los controles sean independientes de la ventana principal.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Modele%20champ%201.gif&type=thumb)
Vista de un modelo de controles.
Un contorno naranja aparece alrededor del modelo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Modele%20champ%202.gif&type=thumb)
 Modelo en una ventana.
Los elementos del modelo están delimitados por una línea azul e identificados por un cuadrado amarillo.
Un modelo de controles se puede crear:

- mediante el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_ICO_Cr%E9er_cpt.gif) disponible en los botones de acceso rápido. En la ventana "Nuevo", haga clic en "Ventana", y luego en "Modelo de controles".

- a partir de los controles de la ventana (seleccione los controles y haga clic derecho, en el menú contextual que aparece, seleccione la opción "Refactorización .. Crear un modelo de controles a partir de la selección").




Para crear una ventana basada en un modelo de controles, cree un control de tipo "Modelo de controles".

Las características de los elementos se pueden disociar del modelo. Por ejemplo, disociar la posición de un control del modelo para moverlo a otro lugar, pero manteniendo otros cambios (código, estilo, etc.). Esto se conoce como **herencia de controles**. En este caso, los elementos se identifican por un cuadrado azul.

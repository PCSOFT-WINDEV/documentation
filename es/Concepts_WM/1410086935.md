|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LOGO_Android.gif)<br> | <br><br>- **Instalación mediante un archivo APK**. Esta instalación se ejecuta en el dispositivo Android.<br><br>- **Instalación mediante Google Play**.<br><br>- **Instalación mediante una tienda privada**. <br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Logo_IOS.gif)<br> | <br><br>- **Compilación en Xcode en Mac**. La compilación en Xcode permite crear la aplicación ejecutable en el iPhone o iPad.<br><br>- **Instalación**: <br><br>	- **mediante App Store**, <br><br>	- **mediante una red interna**, <br><br>	- **mediante una red ad hoc**,<br><br>	- **mediante una tienda privada**. <br><br><br><br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=logo_Windows_10.gif)<br> | <br><br>- **Instalación mediante Microsoft Store**.<br><br>- **Instalación directa en un teléfono**.<br><br>- **Instalación en una tableta**.<br><br><br> |

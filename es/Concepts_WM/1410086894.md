
## 6. Layouts

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los anclajes permiten cambiar fácilmente la orientación de una aplicación móvil o adaptar una aplicación de un teléfono a una tableta. 

Sin embargo, en algunos casos, la interfaz debe modificarse completamente: la posición de los controles debe ser diferente. 

Los layouts permiten definir varias vistas de una ventana en el mismo proyecto sin duplicar la ventana. Esto permite definir:

- una vista específica para el modo vertical,

- una vista específica para el modo horizontal,

- una vista específica para teléfono,

- una vista específica para tableta, etc.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Agencements.gif)


**Para crear un layout**, en la pestaña "Ventana", en el grupo "Layouts", despliegue "Layouts" y seleccione "Agregar layouts". El asistente de creación de layouts se abre y le permite crear el número necesario de layouts. Una vez creados los layouts, puede cambiar la posición de los controles en uno o varios layouts para obtener la interfaz deseada. 

**En tiempo de ejecución**, el layout inicial se selecciona automáticamente en función de la plataforma, la orientación y el tamaño de la pantalla. El layout cambia automáticamente cuando el dispositivo pasa del modo vertical al horizontal.

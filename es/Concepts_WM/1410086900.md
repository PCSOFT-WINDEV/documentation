<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile incluye un editor de reportes que permite crear e imprimir fácilmente todo tipo de reportes.

Los reportes permiten resumir y sintetizar datos.

Es posible:

- agrupar datos.

- ordenar datos en función de diferentes criterios.

- realizar cálculos (medias, estadísticas, etc.) y crear gráficos.




El siguiente diagrama ilustra el concepto de reporte:

- los datos a imprimir provienen de una fuente de datos (archivo de datos descrito en un análisis, vista HFSQL, consulta, área de memoria o archivo de texto).

- el reporte agrupa, ordena y define el formato de los datos.

- el reporte puede ejecutarse como un archivo PDF. 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P24-Etats.gif)


**Otros modos de impresión**

WINDEV Mobile también permite imprimir usando funciones WLanguage (funciones iXXX).

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV Mobile le permite gestionar las reglas de negocio. Una regla de negocio se usa para definir un modo de funcionamiento específico o determinar un proceso en particular. Por ejemplo: el cálculo de un tipo de IVA específico, las reglas para cambiar el estatus de un cliente, la fórmula para calcular los gastos de envío, una comisión comercial, una tasa de descuento, un coeficiente de depreciación, etc.

Una regla de negocio puede ser simple o compleja.

Las reglas de negocio pueden provenir de un pliego de condiciones (que corresponden a los requisitos).

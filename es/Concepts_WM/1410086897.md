
## 10. Ventana interna
			

| [Página anterior](../Concepts_WM/1410086896.md) | [Tabla de contenido](../Concepts_WM/1410086964.md) | [Página siguiente](../Concepts_WM/1410086898.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El control Ventana interna permite incluir una ventana (y su código) en otra ventana. En tiempo de ejecución, la ventana interna se combinará dinámicamente con la ventana principal.


### 1. Crear una ventana interna
<a name="1_crear_una_ventana_interna_ELTPARAGRAPHE000015"></a>




|   |   |
| --- | --- |
| Una ventana interna puede crearse mediante el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_ICO_Cr%E9er_cpt.gif) en los botones de acceso rápido. En la ventana de creación de nuevos elementos, haga clic en "Ventana", y luego en "Ventana interna".<br><br>Una ventana interna es una ventana específica que no tiene barra de título ni menús.<br>En esta ventana se pueden utilizar todo tipo de controles. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fen%EAtre%20interne%201.gif&type=thumb)<br> |




### 2. Utilizar una ventana interna.
<a name="2_utilizar_una_ventana_interna_ELTPARAGRAPHE000030"></a>




|   |   |
| --- | --- |
| Para utilizar una ventana interna, es necesario:<br><br>- crear un control Ventana interna.<br><br>- seleccionar la ventana interna a utilizar en la descripción del control y validar.<br><br><br> | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fen%EAtre%20interne%202.gif&type=thumb)<br> |

Observaciones:

- La ventana interna utilizada en el control "Ventana interna" puede modificarse mediante programación.

- Limitaciones: El área de fondo es rectangular y no se pueden realizar sobrescrituras. Para realizar sobrescrituras, se recomienda utilizar modelos de controles.

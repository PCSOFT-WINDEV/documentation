
## 1. Ventanas generadas mediante la metodología RAD

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El RAD (Desarrollo Rápido de Aplicaciones) permite crear ventanas a partir de:

- el análisis relacionado con el proyecto,

- plantillas.




Las ventanas generadas contienen el código fuente necesario. Estas ventanas se pueden probar inmediatamente con los datos que se encuentran en el equipo de desarrollo.

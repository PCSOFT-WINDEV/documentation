## Presentación
<a name="presentacion_ELTTEXTE000100"></a>
WDDIXIO es una herramienta que le ayuda a traducir el texto de una aplicación. 

WDDIXIO simplifica la entrada y traducción de mensajes, y reduce el riesgo de errores.

WDDIXIO permite:

- proponer traducciones para el mensaje a traducir.

- traducir las expresiones de manera estandarizada.




Se puede acceder a WDDIXIO desde las diferentes ventanas de descripción de WINDEV, WEBDEV y WINDEV Mobile (botón "Traducir" ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Combo_des_saisie_btn_multilangue.GIF)).
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDDixio%20-%20HC%20N%B0001.gif)


**Observaciones**:

- WDDIXIO incluye un diccionario francés-inglés con más de 6000 traducciones. Este diccionario se puede enriquecer a través de WDTRAD.

- Para obtener más información sobre WDDIXIO, consulte [Instalación de WDDIXIO](../WDDixio/3514005.md).

## ¿Cómo traducir los mensajes de un proyecto con WDDIXIO?
<a name="¿como_traducir_los_mensajes_proyecto_con_wddixio_ELTTEXTE000124"></a>
Para traducir los diferentes mensajes de un proyecto: 

1. Configure WDDIXIO en su IDE. 

2. Configure las opciones de WDDIXIO. 

3. Traduzca los diferentes elementos. 

4. Recupere las traducciones. 


Para obtener más información, consulte [Utilizar WDDIXIO](../WDDixio/3514006.md).

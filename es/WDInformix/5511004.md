## Principales observaciones
<a name="principales_observaciones_ELTTEXTE000242"></a>


### Elementos Unicode
<a name="elementos_unicode_ELTPARAGRAPHE000022"></a>

Los elementos Unicode no son compatibles con Native Informix Connector. Le aconsejamos que utilice elementos de texto en formato Ansi y que realice la Conversion( si es necesario) con [UnicodeToAnsi](../WDLang1/3024025.md).

[HCreation](../WDLang4/3044255.md) mostrará un mensaje de error al crear archivos que contengan elementos en formato Unicode.
<a name="NOTE2_2"></a>


### Configuración
<a name="configuracion_ELTPARAGRAPHE000036"></a>

En los parámetros de configuración de la capa de clientes de Informix, el parámetro "DELIMIDENT" debe ser set a "YES".

Esta configuración es necesaria si los nombres de las tablas (o columnas de la tabla) contienen caracteres acentuados (por ejemplo, "ÓrdenesProductos"
<a name="NOTE2_3"></a>


### Búsquedas, filtros y consultas
<a name="busquedas_filtros_consultas_ELTPARAGRAPHE000045"></a>

No se puede realizar ninguna búsqueda ni definir ningún filtro en un elemento utilizado en la cláusula HAVING (**HReadSeekxx** y [HFilter](../WDLang4/3044100.md)).

En efecto, con Informix, un artículo no puede utilizarse tanto en la cláusula HAVING como en la cláusula WHERE.
<a name="NOTE2_4"></a>




### Gestión de las fechas, las horas y la hora de la fecha
<a name="gestion_las_fechas_las_horas_hora_fecha_ELTPARAGRAPHE000058"></a>

Native Informix Connector utiliza las siguientes funciones para formatear los valores de los tipos de fecha, hora y hora en las solicitudes enviadas al servidor:

- salir con: MDY(MM,DD,YYYYY)

- momento: DATETIME(HH:MM) HORA A MINUTO

- Datetime: DATETIME(YYYY-MM-DD HH:MM:SS.ccm) AÑO A LA FRACCIÓN




**Caso especial para los servidores Informix versión 7.23 o anterior**
Las funciones MDY no son soportadas por estos servidores. Para evitar este límite, Native Informix Connector puede intercambiar las fechas como cadenas de caracteres en formato Y4MD.. En este caso, debe definir en el ordenador cliente el entorno Variable DBDATE a Y4MD con Setnet32.

**Observación**: Si el Native Informix Connector detecta que la versión del cliente es igual a 7.23 (o anterior), comprueba en el registro si el valor de la cadena DBDATE está definido a Y4MD en la siguiente ubicación si el valor Default_USER_SETTING está definido a CU:


```txt
HKEY_CURRENT_USER\Software\Informix\Environment
```


de lo contrario, en el siguiente lugar:


```txt
HKEY_LOCAL_MACHINE\Software\Informix\Environment
```

## Características específicas de Native Informix Connector
<a name="caracteristicas_especificas_native_informix_connector_ELTTEXTE000284"></a>


### Opciones del análisis
<a name="opciones_del_analisis_ELTPARAGRAPHE000083"></a>

- Las opciones de análisis relativas a los índices de texto completo no están soportadas por Conector Nativo.

- Las opciones de análisis relativas a la indexación de los memorandos de texto de Partial no están soportadas por el Conector Nativo.



<a name="NOTE3_2"></a>


### Registros idénticos
<a name="registros_identicos_ELTPARAGRAPHE000091"></a>

El Native Informix Connector no puede diferenciar entre dos registros idénticos (es decir, con valores idénticos para todos los artículos, excluyendo los artículos individuales).

Cuando se modifica o elimina una Record, se modifican o eliminan todas las Record idénticas.
<a name="NOTE3_3"></a>


### Forzar el identificador automatic
<a name="forzar_identificador_automatic_ELTPARAGRAPHE000100"></a>

La constante *hForceIDAuto* no puede utilizarse con [HModify](../WDLang4/3044042.md). Esta constante sólo puede utilizarse con [HAdd](../WDLang4/3044147.md). 
<a name="NOTE3_4"></a>


### INSERTAR consultas e identificador de automatic
<a name="insertar_consultas_identificador_automatic_ELTPARAGRAPHE000113"></a>

Los identificadores automatic no son compatibles con las consultas INSERT (creadas mediante programación o en el editor de consultas). Para utilizar un identificador automatic, llame a [HAdd](../WDLang4/3044147.md) en el tabla.
<a name="NOTE3_5"></a>


### Bloqueos
<a name="bloqueos_ELTPARAGRAPHE000123"></a>

Los bloqueos NO PUEDEN ser utilizados con las funciones HFSQL en esta versión.
<a name="NOTE3_6"></a>


### Tipos no soportados
<a name="tipos_soportados_ELTPARAGRAPHE000130"></a>

Los elementos ROW, MULTISET y LIST no son compatibles con el Native Informix Connector.
<a name="NOTE3_7"></a>


### Restaurar memos
<a name="restaurar_memos_ELTPARAGRAPHE000137"></a>

Los memos no se restauran cuando se utiliza [HRestorePosition](../WDLang4/3044137.md) en las consultas y en los informes.

**Consecuencia**: En un tabla "Browsing control" basado en una consulta, los memos ya no aparecen: 

- al hacer clic en una fila de la control Tabla,

- al mover la barra de desplazamiento en la control Tabla.



<a name="NOTE3_8"></a>


### acoplamientos
<a name="acoplamientos_ELTPARAGRAPHE000152"></a>

- [HListLink](../WDLang4/3044138.md) no es compatible con esta versión.

- Los enlaces entre tablas no están soportados en esta versión.



<a name="NOTE3_9"></a>


### Alias y caracteres acentuados
<a name="alias_caracteres_acentuados_ELTPARAGRAPHE000162"></a>

En esta versión, los alias (para nombres de tablas y/o columnas tabla) que contienen un carácter acentuado no son soportados.
<a name="NOTE3_10"></a>


### Búsqueda genérica
<a name="busqueda_generica_ELTPARAGRAPHE000169"></a>

Una búsqueda genérica realizada en una cadena de caracteres que contiene el carácter "\*" " puede fallar. El controlador de Informix muestra un "Error general".

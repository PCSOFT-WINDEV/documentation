
## Conector Natif Informix para WINDEV y WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000166"></a>
WINDEV y WEBDEV proponen un Conector Nativo opcional para Informix (también llamado Native Access to Informix). Este Conector Nativo le permite acceder a una base de datos Informix desde un programa WLanguage sin utilizar ningún controlador externo.

Beneficios de usar el Native Informix Connector:

- Las funciones HReadxxx (HReadNext, HModify, etc.) pueden utilizarse en las bases de datos de Informix. Esto significa que una base de datos Informix y una base de datos HFSQL pueden ser manejadas con los mismos comandos. Se simplifica la adaptación de una aplicación existente a bases de datos externas.. 

- El RAD permite generar el código con las funciones Hxxx (HReadNext, ...) o con las funciones SQLxxx. 

- Acceso directo a una base de datos Informix sin capa intermedia: MDAC, OLE DB y ODBC son inútiles. 

- WDMAP (el visor de datos) se puede utilizar en sus bases de datos Informix (a través de la capa de cliente de 32 bits).. 

- El Centro HFSQL control puede utilizarse en sus bases de datos Informix (CC28 HF.exe para las capas de cliente de 32 bits y CC28 HF64.exe para las capas de cliente de 64 bits). Esto le permite realizar consultas directamente y ver los datos de Informix.. 

- La instalación es sencilla: No se requiere una capa de OLE DB u ODBC en la computadora del usuario. 

- Todas las versiones de las bases de datos de Informix son compatibles

- etc.


El uso de esta Conector Nativo es muy recomendable para el desarrollo de aplicaciones Cliente/Servidor (que acceden a bases de datos Informix) con WINDEV o WEBDEV.. Siéntase libre de Contact nosotros para cualquier información adicional.

**Atención**: 

- Esta versión del Conector Nativo Informix funciona con WINDEV y WEBDEV 28.

- Esta versión del Native Informix Connector no funciona con los siguientes productos: Informix Standard Edition (SE), Informix OnLine e Informix C-ISAM.

## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000190"></a>


### Condiciones para utilizar el conector Informix nativo
<a name="condiciones_para_utilizar_conector_informix_nativo_ELTPARAGRAPHE000043"></a>

**En 32 bits**: 

- Debe instalarse la capa de cliente de Informix 3.34.0.11061 (versión comprobada por nuestros servicios).

- El **ICLIT09B.DLL** DLL (versión comprobada por nuestros servicios) debe ser accesible:

	- del ordenador de desarrollo.

	- de la computadora implementación.







**En 64 bits**: 

- Capa de cliente de Informix 4.10.00.00.15364 (versión probada por nuestros servicios) instalada.

- El **ICLIT09B.DLL** DLL (versión comprobada por nuestros servicios) debe ser accesible:

	- del ordenador de desarrollo.

	- de la computadora implementación.







La **ICLIT09B.DLL** DLL es buscada por Default. Si esta DLL no está disponible:

1. Cree un archivo "WDIFMX.INI" en el directorio de Windows (C:\\WINDOWS por ejemplo).

2. Añada la siguiente información en el archivo "WDIFMX.INI":





```txt
[CLIENT]
DRIVER=<DLL name>
```


donde &lt;DLL nombre&gt; corresponde a la ruta completa de la DLL Informix. Esta DLL también puede corresponder al controlador ODBC utilizado para acceder a Informix.
<a name="NOTE3_2"></a>


### Los archivos necesarios
<a name="los_archivos_necesarios_ELTPARAGRAPHE000075"></a>

Los siguientes archivos son necesarios para utilizar el Native Informix Connector:

- ***ICLIT09B.DLL*** (versión comprobada por nuestros servicios): Esta DLL es instalada por la capa de cliente de Informix. Consulte el acuerdo de licencia de Informix para obtener más detalles sobre las condiciones de distribución de esta DLL.

- ***wd28 0ifmx.dll***: Necesario para la Conector Nativo a una base de datos Informix en modo de 32 bits. Este archivo se suministra con Conector Nativo para Informix.

- ***wd28 0ifmx64.dll***: Necesario para la Conector Nativo a una base de datos Informix en modo de 64 bits. Este archivo se suministra con Conector Nativo para Informix.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000220"></a>


### Uso del conector Informix nativo en sus aplicaciones WINDEV y sitios WEBDEV
<a name="uso_del_conector_informix_nativo_sus_aplicaciones_windev_sitios_webdev_ELTPARAGRAPHE000094"></a>

Para utilizar Informix de forma nativa, debe programar:

- con las funciones "SQL" de WLanguage (véase [Programación con funciones SQL](../WDInformix/5511006.md)).

- con las funciones "HFSQL" de WLanguage (véase [Programación con funciones HFSQL](../WDInformix/5511001.md)).




**Observaciones**: 

- Para obtener más información, consulte [las características específicas relativas al uso del Native Informix Connector](../WDInformix/5511004.md).

- El Conector Nativo Informix tiene en cuenta la opción "Crear automáticamente los archivos de datos cuando se abran" (en la pestaña "Archivos" de la descripción del proyecto) .

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El desarrollo de un **sitio Web** con WEBDEV se basa en dos elementos principales: el Proyecto y el Análisis.

Un **Proyecto WEBDEV** es un conjunto de elementos (páginas, reportes, controles, clases, componentes, etc.) que, en conjunto, permiten crear un sitio Web.

Un **análisis WEBDEV** es un esquema de la base de datos utilizada por la aplicación. El análisis contiene los diferentes archivos de datos (tablas) utilizados por el sitio.

Un sitio se construye a partir de un proyecto.

En la mayoría de los casos, un proyecto está asociado a un análisis.

Un análisis puede estar asociado a uno o más proyectos.

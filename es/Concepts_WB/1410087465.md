<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
**1. Mejore el estilo de la página de inicio.**

La página de inicio es un elemento importante del sitio. En algunos sitios, es solo una página de conexión, pero en la mayoría de sitios de Internet, es la página principal para atraer visitantes (y futuros clientes del sitio). Es primordial prestar atención a su contenido y presentación. Le recomendamos leer más documentación sobre cómo crear sitios Web fáciles de usar.

**2. Contáctenos**

Cuando los usuarios visitan un sitio, deben poder ponerse en contacto con la persona o la empresa responsable del sitio: proporcione la información que permitirá a los usuarios ponerse en contacto con su usted: nombre, dirección postal, dirección de correo electrónico, etc.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20Contact%20-%20HC%20N%B0001.gif&type=thumb)
Esta página muestra un mapa con la ubicación de la empresa o tienda, los horarios de apertura, etc.

**3. ¿El contenido de la página es muy extenso? Utilice opciones que le permitan al usuario volver al principio de la página**.

Si el contenido de sus páginas es demasiado largo, utilice botones o enlaces para volver al principio de la página sin tener que utilizar la barra de desplazamiento.

**4. Establezca el estilo y la alineación de los botones**.

Asegúrese de que los botones utilicen el mismo estilo, y verifique la alineación. El editor de páginas incluye un mecanismo de ajuste y alineación para simplificar esta tarea.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20styles%20et%20boutons%20-%20HC%20N%B0001.gif&type=thumb)


**5. Evite el uso de framesets**.

Aunque los framesets parecen ser más eficientes al principio, conllevan algunos problemas específicos: incluyen varias páginas. Por lo tanto, el posicionamiento en buscadores (SEO) es más difícil. Además, la navegación por el sitio Web es menos intuitiva para los usuarios (especialmente con el botón "Atrás"). Hoy en día, no se recomienda el uso de framesets para desarrollar sitios Web.

**6. ¿Su sitio contiene listas con más de 20 elementos? Distribuya la lista de elementos en varias páginas**.

Puede utilizar un control Paginación para navegar entre las páginas sin necesidad de escribir código. Esto hará que la interfaz sea mucho más clara. Su sitio Web será más rápido y fácil de usar.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20liste%20d'%E9l%E9ments%20-%20HC%20N%B0001.gif&type=thumb)


**7. Utilice imágenes al 100% o en modo homotético centrado**.

Para mantener las proporciones de las imágenes, le recomendamos elegir:

- imágenes al 100% (para imágenes estáticas y dinámicas o si la imagen tiene el mismo tamaño que el control).

- imágenes en modo homotético centrado (para imágenes generadas o imágenes de diferentes tamaños de una base de datos): el tamaño de la imagen se adaptará al área definida para la imagen. Las proporciones se mantendrán.

**8. ¿Su sitio utiliza imágenes? Asegúrese de que se muestran correctamente en las páginas**.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20affichage%20images%20-%20HC%20N%B0001.gif&type=thumb)


Si se produce un problema de visualización, compruebe que:

- la imagen existe,

- la imagen está guarda en un formato reconocido por el navegador (GIF o JPG, por ejemplo),

- la imagen está almacena en el subdirectorio &lt;ProjectName&gt;_WEB del proyecto.




**9. ¿El usuario debe introducir una cantidad? Utilice un control Combo Box para simplificar la entrada de cantidades**. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20combo%20-%20HC%20N%B0001.gif&type=thumb)


**10. ¿Su sitio Web incluye un catálogo de productos y un sistema de gestión de pedidos?**

Especifique la información del producto (nombre, referencia, precio, etc.), incluyendo un botón o enlace para pedir el producto, añadirlo a una selección personalizada, al carrito, etc.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_10%20conseils%20d'ergonomie%20%20%20liste%20d'%E9l%E9ments%20-%20HC%20N%B0001%201.gif&type=thumb)

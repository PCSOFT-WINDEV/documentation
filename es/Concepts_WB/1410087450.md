
## 11. Componente externo

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un componente externo es un conjunto de elementos WEBDEV: páginas, reportes, análisis, etc. Este conjunto de elementos realiza una operación específica. Por ejemplo, un componente externo puede corresponder a una de las siguientes acciones:

- Envío de fax, 

- Envío de correos electrónicos,

- ...




Un componente externo de WEBDEV puede distribuirse a otros desarrolladores de WEBDEV (de forma gratuita o paga). Los desarrolladores podrán integrar fácilmente en sus sitios Web la funcionalidad incluida en el componente externo, sin tener acceso al código fuente correspondiente (si este no se distribuyó con el componente). El componente externo se incluirá en el sitio y se distribuirá con él.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P3-Composant%20Externe.gif)

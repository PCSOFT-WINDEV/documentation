
## 8. Asociar los controles a los datos

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una página puede mostrar información de:

- una base de datos: los controles están enlazados directamente a los campos de los archivos de datos o consultas disponibles en la base de datos

- variables en el código de la aplicación (variables globales a la página o al proyecto o parámetros pasados a la página).




Para mostrar esta información en una página, los controles de la página deben estar enlazados a:

- los diferentes campos de la base de datos.

- las variables WLanguage disponibles.




Existen varios métodos para mostrar y actualizar valores de una fuente de datos:

- El enlace entre un control y un campo o variable puede definirse en el editor de páginas, en la descripción del control (pestaña "Enlace").

- La función [PageToFile](../WDLang4/3044022.md) actualiza el registro o la variable con los datos de la página.

- La función [FileToPage](../WDLang4/3044148.md) actualiza los datos de la página con la información guardada en el archivo de datos o variable.


## 9. Consultas

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las consultas permiten solicitar información de una base de datos para ver, insertar, cambiar o eliminar datos. La estructura de la consulta define los datos manipulados.

Una consulta puede solicitar información de uno o más archivos de datos.

El editor de consultas permite crear consultas fácilmente sin tener que programar.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=les_requetes.gif)


**Observación**: En el código, las consultas se pueden manipular de la misma manera que los archivos de datos. Las consultas se pueden asociar a los controles de la UI (un control Tabla, por ejemplo) para mostrar los datos deseados.

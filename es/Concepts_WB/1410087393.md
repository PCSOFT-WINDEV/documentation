
## 6. Centro de Control de Alojamiento

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Centro de Alojamiento control, destinado principalmente a las empresas de alojamiento y a los Webmasters, le ayuda a alojar los sitios WEBDEV.

El Centro control gestiona las cuentas WEBDEV, así como la cuenta del servidor web y los derechos del sistema operativo.

Se propone una elección de Default: le permite instalar fácilmente un servidor, sin conocimientos específicos.

La configuración de los siguientes elementos está centralizada:

- Cuentas Windows,

- Grupos de usuarios de FTP,

- Grupos de clientes de WEBDEV,

- Directorio de Inicio,

- FTP alias,

- Cuentas WEBDEV, 

- Data directories,

- Sitios Web virtuales, ....




El Centro de Control de Alojamiento de WEBDEV funciona en el entorno Windows con el servidor web IIS (Servidor de Información INTERNET, todas las versiones a partir de la versión 5).


## 4. ¿Servidor dedicado o compartido?

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las empresas de alojamiento proponen dos tipos de servidores para instalar sus sitios WEBDEV:

- **servidor compartido**:  
	servidor cuyos recursos son compartidos por varios clientes (cliente = empresa alojada). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8-Serveur%20d%E9di%E9%20ou%20mutualis%E9-serveur%20mutualis%E9.gif)

	**Características**:
	La empresa de alojamiento proporciona un servidor a varios clientes.
	La empresa de alojamiento:

	- configura el servidor FTP (requerido sólo para despliegues FTP).

	- define el número máximo de conexiones para todos los sitios dinámicos que pertenecen a un mismo cliente (a través del gestor de cuentas WEBDEV).
			La instalación y las actualizaciones de un sitio dinámico o estático se pueden realizar por FTP.




- **servidor dedicado**: el servidor está reservado para el cliente. La empresa de alojamiento proporciona el acceso a INTERNET (IP Address, cable, etc.).
	La computadora es alquilada por la compañía de hospedaje, o proporcionada por el cliente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8-Serveur%20d%E9di%E9%20ou%20mutualis%E9-serveur%20d%E9di%E9.gif)

	**Características**:
	La empresa de alojamiento proporciona el acceso a INTERNET (IP Address, cable, etc.). 
	El cliente debe:

	- proporcionar el servidor (o alquilarlo),

	- configurar el equipo (sistema, etc.),

	- instalar el servidor web,

	- instalar y configurar sitios WEBDEV (a través de HTTP, FTP o DVD-ROM).

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El RAD (Desarrollo Rápido de Aplicaciones) permite crear páginas a partir de:

- el análisis relacionado con el proyecto,

- modelos de RAD estándar o personalizados,

- plantillas.




**En la generación RAD**, las páginas generadas contienen todo el código necesario para su funcionamiento. Estas páginas se pueden probar inmediatamente con los datos que se encuentran en el equipo de desarrollo.

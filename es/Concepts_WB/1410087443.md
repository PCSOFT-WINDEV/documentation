
## 4. WEBDEV, WINDEV y WINDEV Mobile: 100% compatibles

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los nuevos proyectos suelen ser multiplataforma.

Por ejemplo, en el caso de un sistema ERP en Windows, es muy probable que, además de la aplicación principal, haya agentes de ventas que utilicen terminales PDA o Smartphones, tiendas con dispositivos móviles para gestionar inventarios y sitios de Internet e Intranet.

**Todos los elementos, excepto la UI (páginas y ventanas), son 100% compatibles y pueden compartirse fácilmente entre proyectos WINDEV, WEBDEV y WINDEV Mobile**.

Así, es posible compartir conjuntos de procedimientos o clases, por ejemplo, entre varios proyectos.

Un proyecto puede abrirse en cualquier IDE, independientemente del que se haya utilizado para crearlo.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Format_compatible%20WB%20-%20HC%20N%B0001.gif)
Al abrir un proyecto en un IDE diferente, aparece un asistente que permite crear una configuración de proyecto específica para el IDE utilizado.

Por ejemplo, si abre un proyecto WINDEV en WEBDEV, podrá crear una configuración de proyecto llamada "Sitio", para agrupar todos los elementos necesarios para el sitio WEBDEV.

También puede ver los elementos de cada plataforma desde cada entorno. Un proyecto en WEBDEV muestra las miniaturas de las ventanas de WINDEV y WINDEV Mobile, por ejemplo. Al hacer clic en una ventana WINDEV desde el editor de proyectos de WEBDEV se abre la ventana WINDEV (WINDEV debe estar instalado en el equipo).

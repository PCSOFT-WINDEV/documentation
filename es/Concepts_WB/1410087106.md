
## 4. Diferentes tipos de páginas

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Se pueden utilizar diferentes tipos de páginas en los sitios creados con WEBDEV:

- **Páginas estáticas**. Este tipo de páginas muestra datos fijos.

- **Páginas dinámicas WEBDEV en modo Sesión**. Estas páginas muestran dinámicamente la información que se encuentra en una base de datos. El contenido de la página depende del registro que se muestra.  Cuando se muestra una página dinámica en modo sesión, el contexto de la página asociada se crea automáticamente en el servidor. El contexto de página contiene principalmente las variables globales y las variables de posicionamiento en los archivos de datos.

- **Páginas dinámicas WEBDEV AWP (Active WEBDEV Page)**. Estas páginas muestran dinámicamente los datos que se encuentran en una base de datos. El contenido de la página depende del registro que se muestra.
	Una página AWP es una página dinámica WEBDEV sin contexto persistente en el servidor. El contexto de página AWP es temporal. Se crea en una sesión temporal. 

- **Páginas dinámicas PHP**. Estas páginas muestran dinámicamente los datos que se encuentran en una base de datos. Solo pueden ser usadas en un sitio PHP.

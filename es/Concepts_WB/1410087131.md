


## 20. Modelos de controles

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV permite crear modelos de controles. Un modelo de controles es un conjunto de controles que se pueden volver a utilizar en varias páginas.

Los cambios realizados en un modelo de controles se aplican automáticamente a todas las páginas que utilizan dicho modelo.

Un modelo de controles permite:

- agrupar un conjunto de controles para un propósito específico.

- hacer que los controles sean independientes de la página que los contiene.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mod%E8les%20de%20champs%20WB%20-%20HC%20N%B0001.gif&type=thumb)
Vista de un modelo de controles.
El modelo está rodeado por una línea naranja en el editor.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Mod%E8les%20de%20champs%20WB-%20HC%20N%B0002.gif&type=thumb)
Modelo utilizado en una página.
Los elementos del modelo tienen un borde rojo y están identificados con un cuadrado amarillo.

Para utilizar un modelo de controles en una página, cree un control de tipo "Modelo de controles".

**Observación**: La programación asociada a los elementos del modelo se puede realizar directamente en el modelo.

Las características de los elementos se pueden disociar del modelo. Por ejemplo, disociar la posición de un control del modelo para moverlo a otro lugar, pero manteniendo otros cambios (código, estilo, etc.). Esto se conoce como **herencia de controles**. En este caso, los elementos se identifican por un cuadrado azul.

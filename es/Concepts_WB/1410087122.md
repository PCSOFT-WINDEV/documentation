
## 11. Página Popup

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV permite crear páginas emergentes (Popup). Las páginas emergentes permiten interactuar con el usuario de una manera sencilla.

Una página popup se asocia a una página. Una página puede tener varias páginas popup.

Al usar páginas popup se puede manipular la ventana de diálogo en el editor como un elemento independiente de la página, aún si esta está integrada en la página popup.


## 1. ¿Qué es un sitio Web?
			

| [Tabla de contenido](../Concepts_WB/1410087102.md) | [Página siguiente](../Concepts_WB/1410087104.md) |
| --- | --- |



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un sitio Web es un conjunto de páginas HTML (HyperText Markup Language) almacenadas en un servidor Web. Estas páginas HTML están organizadas con un propósito específico (presentar una empresa, vender productos, etc.). 

Un sitio está destinado a ser utilizado por los usuarios de Internet. Los usuarios acceden al sitio a través de un navegador.

WEBDEV permite crear fácilmente sitios Web con o sin datos.

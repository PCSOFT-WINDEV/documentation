<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Veamos las principales diferencias entre WEBDEV y WINDEV:

- WEBDEV permite crear páginas mientras que WINDEV crea ventanas.

- WEBDEV acepta diferentes tipos de código: un código WLanguage ejecutado en el servidor, un código WLanguage ejecutado en el navegador y un código JavaScript ejecutado en el navegador. 
	Cuando se crea una página PHP, aparece un código adicional: un código PHP ejecutado en el servidor.

- Nuevas funciones WLanguage específicas para la Web están disponibles.

- Algunas funciones WLanguage que no son útiles para la Web no están disponibles.

- Algunas funciones WLanguage solo se pueden utilizar en un proceso/evento que se pueda ejecutar en código Servidor.

- Algunas funciones WLanguage solo se pueden utilizar en un proceso/evento que se pueda ejecutar en código Navegador.

- Hay nuevos tipos de controles diseñados específicamente para aplicaciones Web:

	- Control de visualización con formato, 

	- Control Applet Java, 

	- Control Banner deslizante, 

	- Control Barra de navegación, 

	- Control Captcha, 

	- Control Celda, 

	- Control Migas de pan, 

	- Control Componente Web, 

	- Control IFrame, 

	- Control Enlace, 

	- Control Línea, 

	- Control Página interna,

	- Control Esquina doblada, 

	- Control Mapa del sitio, 

	- Control Popup, 

	- Control Paginación, 

	- Control Red social, 

	- Control SilverLight, 

	- Control Tabla HTML, 

	- Control Upload, 

	- Control Miniatura, 

	- Control Área de texto enriquecido.




- Algunos tipos de controles que no se utilizan en Internet no están disponibles: Barra de desplazamiento, ActiveX, Objeto OLE, Control de número, etc.

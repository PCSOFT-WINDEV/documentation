
## 10. Modelos de páginas

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV permite crear modelos de páginas. Un modelo contiene elementos comunes a un conjunto de páginas del sitio

Los cambios realizados en un modelo de páginas se aplican automáticamente a todas las páginas que utilizan dicho modelo.

Un modelo de páginas permite utilizar el tema gráfico definido para un sitio.

Para crear una página basada en un modelo, seleccione el modelo que se utilizará al crear la página.

**Observación**: La programación asociada a los elementos del modelo se puede realizar directamente en el modelo.

Las características de los elementos se pueden disociar del modelo. Por ejemplo, disociar la posición de un control del modelo para moverlo a otro lugar, pero manteniendo otros cambios (código, estilo, etc.). Esto se conoce como **herencia**. En este caso, los elementos se identifican por un cuadrado azul.

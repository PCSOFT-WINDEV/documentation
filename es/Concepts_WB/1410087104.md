
## 2. Sitio estático o dinámico

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Se pueden crear varios tipos de sitios:

- sitios estáticos, 

- sitios dinámicos.




La siguiente tabla presenta las principales diferencias entre estos tipos de sitios:

| Sitio estático | Sitio dinámico |
| --- | --- |
| El contenido de las páginas del sitio es fijo, se define una sola vez.<br><br>Un sitio estático no puede interactuar con datos. | Los datos que se muestran en las páginas cambian. En la mayoría de los casos, las páginas permiten:<br><br>- realizar procesos y/o cálculos por programación.<br><br>- mostrar datos almacenados en una base de datos.<br><br>- mostrar imágenes y texto interactivo.<br><br><br><br><br>WEBDEV propone varios tipos de sitios dinámicos:<br><br>- Sitio WEBDEV en modo Sesión.<br><br>- Sitio WEBDEV en modo AWP.<br><br>- Sitio PHP.<br><br><br> |
| **Sitio estático WEBDEV**: no se requiere el WEBDEV Application Server. | **Sitio dinámico WEBDEV**: se requiere el WEBDEV Application Server o el lenguaje PHP. |


**Observación**: Un sitio dinámico puede contener una sección estática (presentación de la empresa, etc.).

WEBDEV permite crear:

- páginas estáticas,

- páginas dinámicas.

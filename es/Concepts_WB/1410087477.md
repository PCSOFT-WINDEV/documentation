
## 19. Envío de correos electrónicos

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
En un sitio web, los correos electrónicos permiten:

- simplificar la comunicación entre los usuarios: enviar correos electrónicos al administrador del sitio web, utilizar una lista de distribución, etc.

- validar un pedido: en un sitio web de ventas, se envía un correo electrónico a los clientes para validar los pedidos.

- transferir datos de un equipo a otro: los nuevos pedidos de una tienda virtual se transfieren por correo electrónico a la sede de la empresa, etc.




WEBVED incluye dos métodos para enviar correos electrónicos desde un sitio web.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## 1. Enviar correos electrónicos desde el equipo del usuario
<a name="1_enviar_correos_electronicos_desde_equipo_del_usuario_ELTTEXTE000149"></a>
**Utilice este método cuando el usuario deba enviar un correo electrónico a una dirección específica: webmaster del sitio, soporte técnico, departamento de ventas, etc.**

La función [EmailOpenMail](../WDLang2/3032021.md) abre la aplicación de correo electrónico del usuario. Algunos parámetros pueden establecerse de forma predeterminada: dirección de correo electrónico del destinatario, etc. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4-Envoyer%20des%20emails.gif)
Este método permite utilizar el programa de correo electrónico del usuario: no se requiere ningún proceso específico en el sitio WEBDEV.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## 2. Enviar correos electrónicos desde el servidor (solo páginas en modo Sesión o AWP)
<a name="2_enviar_correos_electronicos_desde_servidor_solo_paginas_modo_sesion_awp_ELTTEXTE000173"></a>
En este caso, el servidor gestiona los correos electrónicos.

Los correos electrónicos se pueden enviar:

- en el sitio WEBDEV directamente (por ejemplo, para una transferencia de datos por correo electrónico, o para un correo electrónico de confirmación de pedidos);

- mediante la cola de correo electrónico (proporcionado con WEBDEV).

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4-Envoi%20de%20l'email%20depuis%20le%20serveur.gif)
Utilice este método para personalizar el envío de correos electrónicos:

- comprobar el contenido de los correos electrónicos enviados por los usuarios (sitios de tipo "lista de distribución"),

- comprobar la validez de la dirección del destinatario,

- cifrar la información enviada por correo electrónico,

- validar un proceso, etc.

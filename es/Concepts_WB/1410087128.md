
## 17. WLanguage: un lenguaje simple y robusto

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WLanguage es el lenguaje de programación utilizado en WEBDEV. Este lenguaje permite programar fácilmente todos los procesos Web deseados sin saber HTML, JavaScript o PHP.

WLanguage está compuesto por comandos simples, cercanos al lenguaje cotidiano, y está disponible en inglés y francés. Su uso es intuitivo.

Veamos un ejemplo que ilustra el uso de WLanguage: comprobar la entrada de una dirección:

**En WLanguage**, solo se requieren unas pocas líneas de código:

```wl
IF NoSpace(ADDRESS) = "" THEN
      Error("Specify your address") 
      SetFocusAndReturnToUserInput(ADDRESS)
END
```

**En JavaScript** el proceso equivalente es el siguiente:

```txt
<SCRIPT LANGUAGE=Javascript>
function EXE_NoSpace(szString, nPart)
{
var nFirst = 0;
var nLast = szString.length-1;
if (nPart & 1) while (szString.charAt(nFirst)==" ") nFirst++;
if (nPart & 2) while (szString.charAt(nLast)==" ") nLast--;
return szString.substring(nFirst,nLast+1);
}
// Click (onclick) of BUTTON_OK
function _VALIDORD_BUTTON_OK_CLI()
{
if((EXE_NoSpace(ADDRESS,3)==""))
{
alert("Specify your address");
return;
}
}
</SCRIPT>
```


El código WLanguage es más fácil de escribir, entender y mantener. Es mucho más fiable. Los cambios son más fáciles de hacer.

**Utilice WLanguage en todos sus procesos**.

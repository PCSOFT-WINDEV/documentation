
## 9. Página interna

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El control Página interna se utiliza para incluir una página (y su código) en otra página. En tiempo de ejecución, la página interna se fusionará dinámicamente con la página principal.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## 1. Crear una página interna
<a name="1_crear_una_pagina_interna_ELTTEXTE000136"></a>
Una página interna es una página específica que no contiene barra de título ni menú.

En esta página se pueden utilizar todo tipo de controles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Page%20interne%20-%20HC%20N%B0001.gif&type=thumb)
En el ejemplo, se utiliza una página interna para gestionar un carrito de compras.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## 2. Usar una página interna
<a name="2_usar_una_pagina_interna_ELTTEXTE000160"></a>
Para usar una página interna, es necesario:

- crear un control Página interna.

- en la descripción del control, seleccione la página interna a utilizar y valide.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Page%20interne%20-%20HC%20N%B0002.gif&type=thumb)



**Observaciones**:

- La página interna utilizada en el control Página interna no se puede modificar mediante programación.

- El área de fondo es rectangular y no se pueden realizar sobrescrituras. Para realizar sobrescrituras, se recomienda utilizar modelos de controles.

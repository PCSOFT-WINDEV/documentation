
## 17. Gestión de tareas

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Centro de Seguimiento de Proyectos permite que los colaboradores de un proyecto gestionen su calendario de tareas. Estas tareas pueden estar relacionadas con ciertos requisitos y corresponder a diferentes proyectos. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Suivi%20de%20projet%20-%20HC%20N%B0001.gif&type=thumb)


Una vez definidas las tareas del proyecto, podrá seguir el progreso de las mismas en el Centro de Seguimiento de Proyectos. El tiempo dedicado a una tarea se calcula automáticamente, no requiere ninguna acción específica y no genera ninguna restricción particular.

Al abrir el proyecto, el Centro de Seguimiento de Proyectos solicita o indica la tarea actual. Una vez que haya completado una tarea, indique que la tarea está terminada y especifique la nueva tarea.

Una tarea puede estar relacionada con un elemento del proyecto (ventana, reporte, etc.). Cuando se abre el elemento, se cuenta el tiempo que se pasa en él y se almacena en el Centro de Seguimiento de Proyectos. A la inversa, el elemento que corresponde a la tarea deseada puede abrirse automáticamente desde la lista de tareas.
Cada desarrollador puede ver su propia lista de tareas en el panel "Centro de Seguimiento de Proyectos" del editor.

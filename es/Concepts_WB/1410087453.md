## Presentación
<a name="presentacion_ELTTEXTE000186"></a>
WEBDEV le permite generar sitios PHP sin siquiera conocer el lenguaje PHP. El sitio puede ser desarrollado enteramente en WLanguage, como cualquier sitio estándar de WEBDEV. Sin embargo, tienes la capacidad de introducir tu propio código PHP (código de servidor).
<a name="NOTE1_2"></a>


### ¿Qué es un sitio PHP?
<a name="¿que_sitio_php_ELTPARAGRAPHE000017"></a>

Un sitio PHP está compilado en PHP. Si este sitio utiliza una base de datos, se utilizará un controlador ODBC (MyODBC, ODBC para Oracle, etc.) o un Conector Nativo de MySQL (también llamado Native Access) para acceder a esta base de datos.

El controlador ODBC es específico para cada base de datos (se puede encontrar en el sitio del editor de la base de datos).

Un sitio PHP incluye páginas ".php" y requiere un motor PHP para ser ejecutado.

Para Compile( en PHP) un proyecto PHP con WEBDEV, un motor PHP debe ser instalado en el ordenador de desarrollo. Varios motores PHP están disponibles en INTERNET. Debes descargar uno e instalarlo. Por ejemplo, puede utilizar EasyPHP (http://www.easyphp.org/). Este motor se usa sólo por ejemplo, puedes usar cualquier otro motor PHP.



- Base de datos MySQL requerida.



<a name="NOTE1_3"></a>


### ¿Por qué desarrollar un sitio PHP con WEBDEV?
<a name="¿por_que_desarrollar_sitio_php_con_webdev_ELTPARAGRAPHE000036"></a>

Para algunos sitios (especialmente los pequeños), puede ser que useful desarrolle estos sitios en PHP para que sean alojados sin costo adicional. Ahora tienes la capacidad de hacerlo con WEBDEV.

**Observación**: El número de funciones WLanguage que se pueden utilizar en un proyecto PHP es limitado en esta versión. Este número de funciones aumentará en las próximas versiones.

## Características de un proyecto PHP
<a name="caracteristicas_proyecto_php_ELTTEXTE000265"></a>


### Creación de un proyecto PHP
<a name="creacion_proyecto_php_ELTPARAGRAPHE000049"></a>

El método para crear un proyecto PHP es el mismo que el método para crear cualquier proyecto WEBDEV:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Cr%E9er_cpt.gif) en los botones de acceso rápido de WEBDEV. Aparece la ventana de creación: haga clic en "Proyecto". 

2. El asistente de creación de proyectos se abre.

3. En uno de los primeros pasos de la asistente, puede elegir cómo se generará su proyecto. Para ello, seleccione "PHP" y siga la asistente.
	Tan pronto como el proyecto se configura como un proyecto PHP, todas las herramientas disponibles en WEBDEV se configuran automáticamente para el proyecto PHP:

	- El RAD PHP que genera el código PHP de un proyecto para varias características de WEBDEV.

	- El compilador indica las funciones o controles que no pueden ser usados en PHP. Un mensaje en color púrpura aparece en el panel "Errores de compilación".

	- El despliegue del sitio PHP se propone a través de WDDeploy.





**Observación**: Le aconsejamos crear un nuevo proyecto WEBDEV para desarrollar un sitio PHP en lugar de transformar un sitio WEBDEV en PHP.
<a name="NOTE2_2"></a>


### PHP RAD
<a name="php_rad_ELTPARAGRAPHE000072"></a>

Como para cualquier proyecto associated con un análisis, el RAD puede ser usado para generar las páginas de su sitio.

El RAD tiene en cuenta todas las características específicas de la generación de PHP (controles, eventos, funciones que pueden utilizarse en un sitio PHP generado a partir de WEBDEV).
<a name="NOTE2_3"></a>


### Probar un sitio PHP en el editor
<a name="probar_sitio_php_editor_ELTPARAGRAPHE000083"></a>

Durante la prueba de un sitio PHP, el sitio se compila en PHP y se inicia en su navegador. Puede ver las páginas PHP generadas por WEBDEV.
Recordatorio: Se requiere un motor PHP en la computadora de desarrollo.

La compilación de un sitio PHP se utiliza para:

- ...descubrir los errores de programación de WLanguage.

- ver los controles, eventos, funciones WLangage no disponibles en la generación de PHP. Estos errores aparecen en púrpura en el panel "Errores de compilación".





<a name="NOTE2_4"></a>


### Desplegando un sitio PHP
<a name="desplegando_sitio_php_ELTPARAGRAPHE000109"></a>

El método para desplegar un sitio PHP es idéntico al método para desplegar un sitio estático. Al crear la instalación Procedure, WDDeploy le ayuda a implementar su sitio PHP.

Para más información, consulte la ayuda en línea.


## 15. Repetir un grupo de controles: loopers

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los controles Looper permiten repetir un conjunto de controles en una página Web. En cada fila del Looper se puede mostrar diferente información. Los registros de una base de datos pueden mostrarse en cada fila. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_R%E9p%E9ter%20un%20groupe%20de%20champs%20%20%20ZR%20-%20HC%20N%B0001.gif&type=thumb)
Nuevo control Looper en el editor de WEBDEV
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_R%E9p%E9ter%20un%20groupe%20de%20champs%20%20%20ZR%20-%20HC%20N%B0002.gif&type=thumb)
Control Looper en el navegador

Las características de los controles se pueden modificar en cada fila del control Looper: título, color del texto, etc.

Los controles se pueden repetir:

- en una columna,

- en varias columnas. En este caso, se trata de un "Looper horizontal".




Al crear un control Looper, se puede relacionar directamente con un archivo de datos o consulta. Los atributos del control Looper se definen automáticamente. No hay nada por programar.


## 9. Supervise sus sitios, servidores, etc.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
La mayoría de los sitios dependen de muchos factores externos: Sitio web, servidor HFSQL, red, ... Para optimizar la gestión de los incidentes, WEBDEV permite utilizar un robot de vigilancia.

El motor de monitoreo se compone de tres archivos ejecutables que se ejecutan en diferentes equipos. Este motor permite ejecutar diferentes pruebas: Las pruebas de INTERNET, las pruebas de red, ...

Si se produce un problema durante una prueba, el motor de monitoreo le envía un mensaje utilizando uno de los siguientes métodos:

- Mensaje enviado a través de la aplicación de mensajería de PC SOFT (WDBAL).

- Email

- A través de un programa de terceros.Además, una alerta sonora puede ser set en el monitor.


## 5. Configuración de proyecto

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las configuraciones de proyecto permiten adaptar el mismo proyecto a distintas plataformas.

A partir del mismo proyecto, es posible crear:

- sitios con diferentes elementos, nombres, etc.

- diferentes componentes, 

- un Webservice.




Puede trabajar en una configuración específica en cualquier momento: los elementos que no pertenecen a la configuración aparecerán en gris en el editor de proyectos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P3-configuration%20de%20projet.gif)
La generación múltiple permite generar todas las configuraciones del proyecto (o algunas de ellas) en una sola operación.


## 18. Orden de ejecución del código de los botones/enlaces
			

| [Página anterior](../Concepts_WB/1410087128.md) | [Tabla de contenido](../Concepts_WB/1410087102.md) | [Página siguiente](../Concepts_WB/1410087130.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Existen dos códigos de clic para los botones/enlaces:

- **un código Clic (navegador)**, que puede escribirse en WLanguage (o JavaScript) en el editor de código. Este código se incluye en la página HTML y se ejecuta en el equipo del usuario.

- ** un código de clic Servidor**, que puede escribirse en WLanguage (solo en páginas dinámicas). Este código se ejecuta en el servidor.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_Ordre%20d'execution%20des%20codes%20des%20boutons%20-%20HC%20N%B0001.gif&type=thumb)
La tabla que se muestra a continuación presenta los diferentes códigos que se ejecutan (según el tipo de botón/enlace) y la acción asociada.

El aspecto y el destino del botón/enlace no afectan al orden de ejecución de los códigos.

Consejo: Póngase en el lugar del usuario para entender y recordar el orden de ejecución de los códigos Servidor y Navegador: el usuario hace clic en el botón del navegador, por lo tanto, el clic Navegador debe ejecutarse primero.

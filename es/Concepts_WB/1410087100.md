
## ¿Cómo descubrir WEBDEV?

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV es un entorno de desarrollo integrado completo que viene con todas las herramientas necesarias para diseñar y desarrollar sitios Web. 

**Para aprender a utilizar WEBDEV de forma rápida y eficaz, le recomendamos que proceda en el siguiente orden**:


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif) | Leer los "Conceptos" (en línea o en la documentación física).<br> <br>Esta guía presenta los principales conceptos necesarios para crear un sitio WEBDEV. Algunos conceptos incluyen una sección "Práctica" que presenta diferentes funcionalidades del editor. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.gif) | "Tutorial" (tutorial en línea + ejercicios).<br>El tutorial permite abordar WEBDEV de manera práctica. Le ayuda a familiarizarse con los principales editores de WEBDEV. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif) | Probar los ejemplos.<br>Pruebe los diferentes ejemplos proporcionados con WEBDEV en los campos que más le interesan (comercio electrónico, directorio, programación, etc.). |


La ayuda en línea, disponible en [https://ayuda.windev.es](https://ayuda.windev.es) o instalada con WEBDEV, permite encontrar rápidamente la sintaxis de una función WLanguage, obtener ayuda sobre la interfaz, etc. Para cada tema de programación, encontrará una descripción de la funcionalidad asociada y la lista de funciones WLanguage correspondientes.



**Esperamos que disfrute desarrollar con WEBDEV.**

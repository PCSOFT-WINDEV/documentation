
## 4. Activación de un sitio WEBDEV dinámico

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Su sitio WEBDEV dinámico (en modo Sesión o AWP) puede ser accedido tan pronto como esté en línea. La puesta en línea de un sitio WEBDEV consiste en activar el sitio.

Se pueden utilizar varios métodos para activar un sitio WEBDEV:

- al realizar la instalación por HTTP: se puede utilizar una casilla de verificación para activar automáticamente el sitio después de ser instalado.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P7_D%E9ploiement%20-%20HC%20N%B0008%202.gif&type=thumb)





- con el administrador remoto.

- con el administrador de WEBDEV que se encuentra en el ordenador de implementación (opción "Bloqueado" en la pestaña "Sitios").<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P7_Administrateur%20WB%20-%20HC%20N%B0002%202.gif&type=thumb)

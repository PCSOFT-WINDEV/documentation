<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una aplicación WEBDEV HFSQL puede estar en modo Classic (HFSQL Classic) o Client/Server.

Las características del modo Classic son las siguientes:

- Un sitio que utiliza HFSQL Classic se ejecuta en diferentes navegadores. Por cada sitio que se ejecuta, se inicia una sesión en el servidor.

- Los archivos de datos se encuentran en un directorio (en el servidor Web o en otro equipo) al que pueden acceder las sesiones de los sitios WEBDEV. Cada sesión accede físicamente a los archivos de datos.

- Los procesos (consultas, leer o agregar registros en un archivo de datos, etc.) son realizados por cada sesión.


## 15. Centro de Seguimiento de Proyectos

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WEBDEV incluye un Centro de Seguimiento de Proyectos para gestionar proyectos de desarrollo. El Centro de Seguimiento de Proyectos permite:

- Gestionar los requisitos del proyecto,

- Seguir el avance del proyecto (calendario de tareas a realizar),

- Gestionar los errores y cambios realizados en el proyecto.




El Centro de Seguimiento de Proyectos utiliza una base de datos HFSQL Classic o Client/Server: la base de datos del Centro de Control. Esta base de datos es compartida por diferentes herramientas disponibles en WEBDEV: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P3-Les%20centres%20de%20controle.gif)


Observación: Al instalar WEBDEV, el programa de instalación le pide:

- crear la base de datos de los Centros de Control. Esta base de datos se creará automáticamente en formato HFSQL Classic en el directorio especificado.

- compartir una base de datos existente de los Centros de Control.

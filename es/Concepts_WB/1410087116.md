Leyenda:


|   |   |   |
| --- | --- | --- |
| **.WWH: Formato fuente de las páginas**<br>Este archivo contiene la descripción completa de la página (control, código Navegador y código Servidor). Este archivo es utilizado por el editor y permanece en el equipo de desarrollo.<br><br> |   | **.WDL: Biblioteca del proyecto**<br>La biblioteca contiene el código Servidor de las páginas del proyecto y se genera al implementar el sitio WEBDEV (en modo Sesión o AWP). Este archivo se encuentra en el servidor.<br><br> |
| **.HTML genérico**<br>Este archivo contiene la descripción completa de la página (control, código Navegador y código Servidor). Algunos parámetros de esta página son genéricos y se completarán dinámicamente cuando se muestre la página.<br><br> |   | **.AWP: Página AWP**<br>Este archivo se genera a partir del archivo WWH y contiene la descripción de los controles, el código Servidor y el código Navegador. Este archivo se encuentra en el servidor.<br><br> |
| **.HTM: Página HTML del sitio**<br>Este archivo se genera a partir del archivo WWH y contiene la descripción de los controles en HTML y el código Navegador (si el código se escribió en WLanguage, se convierte automáticamente a JavaScript). Este archivo se encuentra en el servidor.<br><br> |   | **.PHP: Página PHP**<br>Este archivo se genera a partir del archivo WWH y contiene la descripción de los controles, el código Servidor y el código Navegador en formato PHP. Este archivo se encuentra en el servidor.<br><br> |

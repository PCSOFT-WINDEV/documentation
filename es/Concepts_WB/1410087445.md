
## 6. Generación múltiple

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las configuraciones de proyecto permiten definir fácilmente cómo se generará el proyecto. Se pueden definir varios sitios, bibliotecas o componentes para el mismo proyecto.

Para generar el resultado de cada configuración, puede seleccionar cada configuración una por una y generar el elemento correspondiente.

Existe un método más rápido: **la generación múltiple**. Las configuraciones a generar se seleccionan en una sola operación y el resultado se obtiene de forma inmediata.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P3_WB%20-%20Generation%20multiple%20-%20HC%20N%B0001.gif)
Para iniciar una generación múltiple, en la pestaña "Proyecto", en el grupo "Generación", despliegue "Generar configuración" y seleccione "Generación múltiple".

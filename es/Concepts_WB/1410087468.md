<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El aspecto gráfico de un sitio Web es un elemento primordial. Con WEBDEV, es muy fácil crear un sitio atractivo con un estilo profesional a primera vista. Para ello, se pueden utilizar varias herramientas: temas, paletas y modelos de páginas predefinidos.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Tema
<a name="tema_ELTTEXTE000188"></a>
El tema de un sitio se puede seleccionar en la ventana de descripción del proyecto (en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción"). Los temas se pueden configurar en la pestaña "Tema". WEBDEV incluye varios temas.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Ambiance_prj_rewali%20-%20HC%20N%B0001.gif&type=thumb)


Un tema contiene:

- una hoja de estilo con estilos WEBDEV.

- un directorio de imágenes.

- una paleta de colores predeterminada.

- un conjunto de reportes que se pueden utilizar como base para construir nuevos reportes del proyecto.




Los temas se pueden intercambiar en cualquier momento para modificar el aspecto gráfico de un sitio. Al cambiar el tema, WEBDEV adapta automáticamente el aspecto visual de los controles del sitio.

El tema de una página puede definirse en tres niveles diferentes (en orden descendente de prioridad):

- En el proyecto.

- En el modelo de páginas del que la página hereda propiedades.

- En la misma página.




El tema general debe definirse a nivel del proyecto. Si un conjunto de páginas (por ejemplo, las páginas de "Promoción" de un sitio de comercio electrónico) debe utilizar un tema específico, puede redefinir el tema del proyecto como un modelo de páginas. Si una página en particular debe tener un tema diferente, puede elegir un tema solo para esa página.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Paletas de colores
<a name="paletas_colores_ELTTEXTE000212"></a>
Las paletas de colores definen un conjunto de colores que están disponibles en todos los selectores de color del proyecto. Además, estas paletas permiten personalizar fácilmente el aspecto visual de un sitio.

Los colores también se aplican a los estilos definidos en el tema del proyecto para cambiar fácil y rápidamente el aspecto visual de un proyecto.

La paleta de colores de una página se puede definir en tres niveles diferentes (en orden descendente de prioridad):

- En el proyecto.

- En el modelo de páginas del que la página hereda propiedades.

- En la misma página.




La paleta general debe definirse a nivel del proyecto. Si un conjunto de páginas (por ejemplo, las páginas de "Administración" de un sitio) debe utilizar una paleta específica, puede redefinir la paleta del proyecto como un modelo de páginas. Finalmente, si una página específica debe tener una paleta diferente, puede elegir la paleta solo para esa página.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modelos de páginas predefinidos
<a name="modelos_paginas_predefinidos_ELTTEXTE000236"></a>
Los modelos de páginas predefinidos proporcionan estructuras de páginas que corresponden a una amplia variedad de sitios y promueven la reutilización y armonía gráfica entre los proyectos.

WEBDEV proporciona varios modelos.

El mismo modelo predefinido se puede agregar varias veces a un proyecto para usarlo con diferentes temas o paletas de colores.

Una vez que se agrega el modelo predefinido al proyecto, puede abrirse en el editor y modificarse como cualquier otro modelo de páginas.

Un modelo de páginas predefinido también se puede utilizar como base para crear otro modelo de páginas.

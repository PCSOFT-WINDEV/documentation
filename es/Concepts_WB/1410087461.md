<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Groupware Usuario permite gestionar usuarios y permisos en un sitio WEBDEV fácilmente.
Existen dos tipos de usuarios:

- usuario simple, que utiliza directamente el sitio.

- supervisor: configura usuarios y permisos.

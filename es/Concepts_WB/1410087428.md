<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
HFSQL Cluster es una extensión del modelo de base de datos HFSQL Client/Server. En un clúster de bases de datos, los servidores HFSQL contienen una copia de las bases de datos y se sincronizan en tiempo real.

- La carga de lectura puede equilibrarse entre varios servidores.

- Puede realizar cambios en la configuración física sin interrupciones en los equipos cliente.

- Si uno de los servidores se cae, el cliente es redirigido automáticamente a otro servidor.

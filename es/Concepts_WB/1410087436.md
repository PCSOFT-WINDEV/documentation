<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los controles de una página pueden estar vinculados a un archivo de datos o consulta. Los controles también pueden vincularse a una consulta definida al crear el control.

En este caso, la consulta se integra en la página. Se encuentra en el archivo WWH que corresponde a la página. Si el archivo WWH se copia (a otro proyecto, por ejemplo), las consultas incrustadas utilizadas por la página también se copiarán.

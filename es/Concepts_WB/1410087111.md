## Usuario de internet
<a name="usuario_internet_ELTTEXTE000143"></a>



|   |
| --- |
| <br><br>- un equipo: PC, Mac, Unix, Smartphone, etc.<br><br><br> |
| <br><br>- un navegador: Microsoft Edge, FireFox, Chrome, etc.<br><br><br> |
| <br><br>- un acceso a Internet (o Intranet).<br><br><br> |


No se necesita descargar ningún módulo. Este modo de funcionamiento es rápido e inmediato.




| Servidor | Equipo de desarrollo |
| --- | --- |
| <br><br>- **1 servidor**: PC<br><br><br> | <br><br>- **1 equipo**: PC<br><br><br> |
| <br><br>- **1 sistema operativo del servidor** (Windows o Linux en una versión soportada por el editor)<br><br><br> | <br><br>- **1 sistema operativo**:<br>	Windows Vista o posterior.<br><br><br> |
| <br><br>- **1 Servidor Web**:<br>	IIS, Apache, etc.<br><br><br> | <br><br>- **1 Servidor Web**:<br>	IIS, Apache, etc.<br><br><br> |
| <br><br>- **1 WEBDEV Application Server**<br><br><br>o<br><br>- **1 Motor PHP**<br>	(versión 5.3 o posterior)<br><br><br> |   |
|   | <br><br>- **1 versión de WEBDEV para desarrollo**<br><br><br> |
| <br><br>- **el sitio en sí mismo**<br><br><br> | <br><br>- **el sitio que se está desarrollando**<br><br><br> |
| <br><br>- **los datos** \*<br>	(solo sitio dinámico)<br><br><br> | <br><br>- **los datos** (opcional) \*<br><br><br> |
|   | <br><br>- **al menos un navegador**:<br>	Microsoft Edge, Firefox, Chrome, etc.<br><br><br> |


\* Los datos pueden estar en otro equipo conectado a la red.

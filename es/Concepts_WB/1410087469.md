
## 11. Elegir la acción de un botón/enlace

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Dependiendo de la operación a realizar en los controles de la página, puede elegir entre varias acciones de un botón/enlace.

La acción se puede elegir en la descripción del control (pestaña "General").<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Op%E9ration%20sur%20les%20champs%20-%20HC%20N%B0001.gif)


Veamos algunos ejemplos de usos "estándar" de los botones/enlaces en una página:

- **Enviar datos de la página actual al servidor para ser procesados**
	Ejemplo: Formulario de registro: el enlace permite comprobar la validez de la información introducida y mostrar una página de confirmación.
	Para realizar este tipo de proceso, utilice un enlace con la acción "Enviar valores al servidor". Los datos de la página se envían al servidor para ser procesados.




- **Realizar un proceso en el servidor sin recuperar los datos de la página actual**
	Ejemplo: Página de menú con botones: cada botón abre una página del sitio.
	Para realizar este tipo de proceso, utilice un botón con la acción "No enviar nada al servidor".




- **Realizar un proceso en código Navegador**
	Ejemplo: Realizar un cálculo simple localmente, una verificación de entrada, entre otros, ...
	Para realizar este tipo de proceso, utilice un botón con la acción "No enviar nada al servidor".




- **Mostrar una página externa al sitio WEBDEV**
	Ejemplo: Iniciar un sitio de un motor de búsqueda mediante un botón.
	Para realizar este tipo de proceso, utilice un botón con la acción "No enviar nada al servidor".




- **Reiniciar los controles de la página**
	Ejemplo: Enlace para borrar el contenido de los controles de la página actual.
	Para realizar este tipo de proceso, utilice un enlace con la acción, "Eliminar contenido de los controles en el navegador".

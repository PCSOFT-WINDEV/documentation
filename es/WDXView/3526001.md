<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000132"></a>
Esta página de ayuda presenta las diferentes características de WDXView:

- Averigua las propiedades, métodos, eventos y constantes associated con un ActiveX.

- Compruebe un ActiveX y sus características.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Descubrir las propiedades
<a name="descubrir_las_propiedades_ELTTEXTE000156"></a>
Para mostrar información sobre un ActiveX:

1. Ejecutar WDXView

2. Seleccione un ActiveX en la lista desplegable "ActiveX. Este cuadro combinado enumera todos los controles ActiveX que se encuentran en el equipo. 
	**Observación**: La búsqueda Icon (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=recherche.jpg)) permite seleccionar el ActiveX mediante palabras clave.
	Se muestra la información sobre el ActiveX.

3. Elija el elemento cuya description se solicita. Este elemento puede corresponder a:

	- un método, 

	- un evento, 

	- un Property,

	- a constant.







**Observaciones**:

- La description de las características del ActiveX corresponde a la información suministrada por el editor. Dependiendo del ActiveX, esta información puede ser más o menos detallada.   

- Un ActiveX no incluye necesariamente eventos, métodos, propiedades o constantes.

## Comprobación de un ActiveX y sus características
<a name="comprobacion_activex_sus_caracteristicas_ELTTEXTE000184"></a>




### Comprobación de un ActiveX y sus características
<a name="comprobacion_activex_sus_caracteristicas_ELTPARAGRAPHE000050"></a>

Para comprobar un ActiveX:

1. En WDXView, seleccione el ActiveX solicitado.

2. Haga clic en el botón "Vista previa. El ActiveX seleccionado se ejecuta automáticamente. 




Para probar un método o una Event del ActiveX seleccionado:

- haga doble clic en el nombre del método o Event para probar,

- seleccione "Correr" de el menú contextual del método o Event.




Si el método espera parámetros, una ventana de diálogo le permite introducir estos parámetros. Si se deben especificar varios parámetros, estos parámetros deben estar separados por punto y coma.
<a name="NOTE3_2"></a>




### Modificar un Property de un ActiveX
<a name="modificar_property_activex_ELTPARAGRAPHE000068"></a>

Para modificar un Property de un ActiveX:

- haga doble clic en el nombre de la Property,

- seleccione "Modificar" de el menú contextual de la Property.




Especifique el nuevo valor de la Property y valídelo.
<a name="NOTE3_3"></a>




### Notas
<a name="notas_ELTPARAGRAPHE000081"></a>

Solo se pueden modificar las propiedades **Int**, **Long**, **Short**, **Real**, **Double**, **Date**, **Boolean** y **String**.

Solo se pueden ejecutar las funciones con parámetros que devuelvan un resultado de tipo **Int**, **Long**, **Short**, **Real**, **Double**, **Date**, **Boolean** y **String**.

## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
El Centro de Usuario control se utiliza para gestionar las aplicaciones desplegadas en la red de la empresa. Desde este centro control, los usuarios podrán instalar sus aplicaciones dedicadas.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Iniciando el Centro de Usuario control
<a name="iniciando_centro_usuario_control_ELTTEXTE000135"></a>


### Iniciando el Centro de Usuario control
<a name="iniciando_centro_usuario_control_ELTPARAGRAPHE000018"></a>

Para iniciar el Centro control del usuario:

- en la pestaña "Herramientas", en el grupo "Centros de Control", haga clic en "Control Usuario".

- ejecutar "CCUser.EXE" directamente. 



<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000040"></a>

El Centro de Usuarios de control puede ser redistribuido.

También se puede utilizar en red. Para ello, copy los siguientes archivos en un directorio de la red:

- CCUSER.EXE,

- CCDEV.WDK,

- WDOutil.WDK, 

- CCxxxAdminExe.WDK,

- el WINDEV framework (wdxxx\*.dll).

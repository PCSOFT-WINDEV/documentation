


## Centro de Control del Usuario: Aplicaciones de red disponibles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000112"></a>
El Centro control del usuario se utiliza para listar las aplicaciones para las que se creó un instalación de red.

Al crear la red instalación de la aplicación, tiene la posibilidad de elegir si la reference instalación de la aplicación será reference d en el Centro control del usuario.

En la asistente para la creación de la instalación, en el paso llamado "Red instalación: Comprobando las versiones", el botón "Avanzado..." permite definir si el instalación debe proponer a reference (o no) la aplicación en el Centro control del usuario.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_User_Option_Install.gif)
 Esta opción está marcada por Default. Desmarque la opción si no quiere que el instalador proponga a reference la aplicación.

Al instalar la versión reference de la aplicación, una opción le permite reference la aplicación en el Centro control del usuario.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_User_Option_Install2.gif)

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000136"></a>


### Listado de las aplicaciones de red disponibles
<a name="listado_las_aplicaciones_red_disponibles_ELTPARAGRAPHE000027"></a>

La lista de aplicaciones de red disponibles se actualiza automáticamente cuando se inicia el Centro control para usuarios.

Esta lista aparece en "Aplicaciones de red disponibles" (esquina superior izquierda en la ventana principal del Centro control).
<a name="NOTE2_2"></a>


### Instalación de una aplicación de red
<a name="instalacion_una_aplicacion_red_ELTPARAGRAPHE000036"></a>

Para cada aplicación de red que se encuentre en el Centro control, todo lo que tiene que hacer es hacer doble clic en la aplicación seleccionada para instalarla.
<a name="NOTE2_3"></a>


### Ubicación de las aplicaciones de red disponibles
<a name="ubicacion_las_aplicaciones_red_disponibles_ELTPARAGRAPHE000043"></a>

Un archivo de parámetros globales permite al Centro control del usuario enumerar las configuraciones de red disponibles. Este archivo se encuentra en la carpeta \\AdminExe\\ de la carpeta compartida de los Centros control. Este archivo se llama "apps.ini".

Se puede configurar la ubicación de la carpeta compartida de los Centros control:

- en el ordenador de desarrollo: 

	- Abrir las opciones generales de WINDEV: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV". 

	- Haga clic en el botón "Parámetros de los Centros control".




- en un centro de producción: durante el paso denominado "Referenciar la aplicación en el Centro de Control de Usuario" (paso visible arriba), al instalar la versión reference de la aplicación en el servidor.

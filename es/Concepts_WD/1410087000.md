
## 21. Gestionar las reglas de negocio

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV le permite gestionar las reglas de negocio. Una regla de negocio se usa para definir un modo de funcionamiento específico o determinar un proceso en particular. Por ejemplo: el cálculo de un tipo de IVA específico, las reglas para cambiar el estatus de un cliente, la fórmula para calcular los gastos de envío, una comisión comercial, una tasa de descuento, un coeficiente de depreciación, etc.

Una regla de negocio puede ser simple o compleja.

Las reglas de negocio pueden provenir de:

- un pliego de condiciones (que corresponden a ciertos requisitos).

- sugerencias hechas por los usuarios de la aplicación.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2-Gestion%20des%20r%E8gles%20m%E9tier.gif)


Durante el desarrollo, las reglas de negocio definidas para el proyecto se muestran directamente en el panel "Reglas de negocio" del entorno de desarrollo. Este panel muestra el número de elementos del proyecto a los que se aplican las reglas de negocio y el porcentaje de reglas que se aplican actualmente.


## 5. Ventanas modales, no modales o MDI

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las aplicaciones creadas con WINDEV pueden utilizar varios tipos de ventanas (y mezclar cada uno de estos tipos).
<a name="NOTE1_2"></a>


### Ventanas modales
<a name="ventanas_modales_ELTPARAGRAPHE000017"></a>

Una ventana modal se abre en primer plano de la aplicación y bloquea el acceso a las demás ventanas que ya están abiertas (si las hay). En la mayoría de los casos, las ventanas modales se utilizan como:

- ventanas principales de una aplicación,

- cuadros de diálogo cuya validación desencadena una acción.


En WINDEV, cuando se abre una ventana modal, aparece una sombra gris en las ventanas que se muestran en el fondo: esta característica se llama Oscurecimiento de Ventanas Inactivas (OVI).

### Ventanas no modales
<a name="ventanas_modales_ELTPARAGRAPHE000036"></a>

Una ventana no modal se abre en paralelo con otras ventanas de la aplicación. Es posible acceder a cada ventana. En la mayoría de los casos, las ventanas no modales se utilizan:

- como paletas de herramientas,

- como cuadros de diálogo que afectan a la ventana principal cuando se modifican. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fen%EAtres%20non%20modales%20-%20HC%20N%B0001.gif&type=thumb)



En este ejemplo, la ventana de descripción de una silla abre la ventana de descripción de otra silla. Se puede acceder a todas las ventanas abiertas.
<a name="NOTE1_4"></a>


### Ventanas MDI
<a name="ventanas_mdi_ELTPARAGRAPHE000051"></a>

En una aplicación MDI, todas las ventanas de la aplicación se muestran en la misma ventana principal: la ventana padre MDI. Existen dos tipos de ventanas en las aplicaciones MDI: 


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fenetres%20MDI%20-%20HC%20N%B0001.gif&type=thumb)<br> | **Ventana padre MDI**: Las demás ventanas de la aplicación se muestran en esta ventana. Esta ventana permite al usuario manipular el menú principal de la aplicación. |
| **Ventana hija MDI**: Ventana de la aplicación que se muestra en la ventana principal. El usuario podrá escribir información en estas ventanas. |

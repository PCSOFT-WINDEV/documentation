<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Groupware Usuario permite administrar fácilmente usuarios y permisos en una aplicación WINDEV.

Existen dos tipos de usuarios:

- usuario simple: utiliza la aplicación directamente.

- supervisor: configura usuarios y permisos.

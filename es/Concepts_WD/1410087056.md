<a name="NOTE1_1"></a>
El editor de análisis permite definir la estructura de los archivos de datos.

Los archivos se crean físicamente en función de la información introducida en el editor de análisis.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=creation_fichier_HFSQL.gif)
**Observación**: Este esquema solo presenta los principales archivos creados. Se pueden crear otros archivos específicos si el archivo de datos utiliza registros, transacciones o replicaciones.

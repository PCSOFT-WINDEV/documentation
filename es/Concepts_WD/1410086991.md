
## 12. Componente externo

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un componente externo es un conjunto de elementos de WINDEV : ventanas, reportes, análisis, etc. Este conjunto de elementos realiza una operación específica. Por ejemplo, un componente externo puede corresponder a una de las siguientes acciones:

- Envío de mensajes de texto,

- Envío de correos electrónicos,

- etc.




Un componente externo de WINDEV puede distribuirse a otros desarrolladores de WINDEV (de forma gratuita o paga). Los desarrolladores podrán incluir fácilmente las funcionalidades del componente externo en las aplicaciones. El componente externo se incluirá en la aplicación y se distribuirá con la aplicación.

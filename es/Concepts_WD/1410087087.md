
## 5. Webservices

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV permite utilizar Webservices (o servicios Web XML) fácilmente. Un Webservice es una aplicación disponible en la Web. WINDEV le permite acceder a los Webservices SOAP (Simple Object Access Protocol) y REST (Representational State Transfer).

Cualquier aplicación WINDEV puede utilizar un Webservice existente. 

Para utilizar un Webservice SOAP, solo necesita obtener la dirección del Webservice e importar su descripción a la aplicación. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-Les%20Webservices.gif)


Para utilizar un Webservice REST, solo necesita obtener las características del Webservice y utilizar la función [RESTSend](../WDLang3/1000021476.md). 

**Observación**: también puede proporcionar uno o más conjuntos de procedimientos de una aplicación WINDEV como un Webservice SOAP o REST. Para ello, es necesario: 

- crear una configuración de proyecto específica,

- para un Webservice REST, crear una descripción del Webservice REST,

- generar el Webservice,

- implementarlo en un WEBDEV Application Server.

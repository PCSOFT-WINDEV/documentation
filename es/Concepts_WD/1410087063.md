
## 1. RAD (Desarrollo rápido de aplicaciones)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El RAD (Desarrollo Rápido de Aplicaciones) permite crear ventanas a partir de:

- el análisis relacionado con el proyecto,

- modelos de RAD estándar o personalizados,

- plantillas.




Con la metodología RAD, las ventanas se generan con todo el código necesario para funcionar correctamente. Estas ventanas se pueden probar inmediatamente con los datos que se encuentran en el equipo de desarrollo.


## 8. Asociar los controles a los datos

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una ventana puede mostrar información de:

- una base de datos: Los controles están enlazados directamente a los campos de los archivos de datos o consultas disponibles en la base de datos.

- variables en el código WLanguage de la aplicación (variables globales a la ventana o al proyecto o parámetros pasados a la ventana).


Para mostrar información en una ventana, los controles de la ventana deben estar enlazados a:

- los diferentes campos de la base de datos.

- las variables WLanguage disponibles.




Existen varios métodos para mostrar y actualizar valores de una fuente de datos:

- El enlace entre un control y un campo o variable puede definirse en el editor de ventanas, en la descripción del control (pestaña "Enlace").

- La función [ScreenToFile](../WDLang4/3044146.md) actualiza el registro o la variable con los valores de la UI.

- La función [FileToScreen](../WDLang4/3044210.md) actualiza los datos de la UI con los valores del archivo de datos o de la variable.


## 6. Ventana interna
			

| [Página anterior](../Concepts_WD/1410086971.md) | [Tabla de contenido](../Concepts_WD/1410087098.md) | [Página siguiente](../Concepts_WD/1410086973.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El control Ventana interna permite incluir una ventana (y su código) en otra ventana. En tiempo de ejecución, la ventana interna se combinará dinámicamente con la ventana principal.


|   |   |
| --- | --- |
| **1. Crear una ventana interna**.<br><br>Para crear una ventana interna:<br><br>- Haga clic en ![Crear un elemento](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Nouveau.gif) en los botones de acceso rápido. <br><br>- La ventana de creación de nuevos elementos se abre: haga clic en "Ventana", y luego en "Ventana interna".<br><br><br> |
| Una ventana interna es una ventana específica que no tiene barra de título ni menús.<br>En esta ventana se pueden utilizar todo tipo de controles.<br><br>En el ejemplo, se utiliza una ventana interna para seleccionar archivos. | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fen%EAtre%20interne%20-%20HC%20N%B0002.gif)<br> |




|   |   |
| --- | --- |
| **2. Utilizar una ventana interna**. |
| Para utilizar una ventana interna:<br><br>- Cree un control Ventana interna.<br><br>- En la ventana de descripción del control, seleccione la ventana interna a utilizar.<br><br>- Valide la ventana de descripción del control.<br><br><br> | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Fen%EAtre%20interne%20-%20HC%20N%B0001.gif&type=thumb)<br> |
| **Observaciones**:<br><br>- La ventana interna utilizada en el control "Ventana interna" puede modificarse mediante programación.<br><br>- Limitaciones: El área de fondo es rectangular y no se pueden realizar sobrescrituras. Para realizar sobrescrituras, se recomienda utilizar modelos de controles.<br><br><br> |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los controles de una ventana pueden estar vinculados a un archivo de datos o consulta existente, ... Los controles también pueden vincularse a una consulta definida al crear el control.

En este caso, la consulta está integrada en una ventana. Se encuentra en el archivo WDW que corresponde a la ventana. Si el archivo WDW se copia (a otro proyecto, por ejemplo), las consultas incrustadas utilizadas por la ventana también se copiarán.

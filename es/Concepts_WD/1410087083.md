<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## POP3 o IMAP y SMTP
<a name="pop3_imap_smtp_ELTTEXTE000134"></a>
Estos son los protocolos estándar de Internet para administrar correos electrónicos. Están estandarizados por los documentos RFC 1939, 3501 y 5321. Estos protocolos le permiten comunicarse directamente con el servidor, disponible en su proveedor de servicios de internet (ISP). 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-Gestion%20des%20emails%20Protocoles%20POP3%20ou%20IMAP%20et%20SMTP.gif)


- POP3 (Post Office Protocol) es un protocolo para leer correos electrónicos.

- IMAP (Internet Message Access Protocol) es un protocolo para leer correos electrónicos a distancia.

- SMTP (Simple Mail Transfer Protocol) es un protocolo para enviar correos electrónicos.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## "Simple Mail API (también conocido como SMAPI o Simple MAPI)"
<a name="simple_mail_api_tambien_conocido_como_smapi_simple_mapi_ELTTEXTE000158"></a>

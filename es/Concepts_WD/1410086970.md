<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
La interfaz de usuario (UI) de una aplicación se compone principalmente de ventanas.

WINDEV incluye un editor de ventanas avanzado que le permite desarrollar de forma fácil y rápida diferentes tipos de UI.

Un gran número de funcionalidades le permiten crear aplicaciones intuitivas y fáciles de usar:

- controles potentes,

- un mecanismo de anclajes que permite a la UI adaptarse automáticamente al tamaño de la pantalla,

- un sistema de compilación de la UI con detección de errores (títulos vacíos, texto sin traducir, superposicionamientos, etc.), 

- etc.


## 8. Optimizar una aplicación
			

| [Página anterior](../Concepts_WD/1410087069.md) | [Tabla de contenido](../Concepts_WD/1410087098.md) | [Página siguiente](../Concepts_WD/1410087071.md) |

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV incluye varios métodos para optimizar las aplicaciones:

- **El analizador de rendimiento**, que permite identificar rápidamente los procesos que consumen tiempo. Así podrá optimizar el proceso deseado.





|   |   |
| --- | --- |
|   |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Analyseur%20performances%20-%20HC%20N%B0002.gif&type=thumb)<br> | Analiza el rendimiento de la aplicación durante una prueba y muestra los procesos más largos para optimizar el código fuente.<br>Para iniciar el analizador de rendimiento, en la pestaña "Proyecto", en el grupo "Auditoría y rendimiento", despliegue "Analizar rendimiento" y seleccione "Analizar rendimiento". |




- **La auditoría estática del proyecto**, que señala los posibles problemas de programación (grupos vacíos, elementos huérfanos, etc.), así como los archivos de gran tamaño que pueden provocar tiempos de carga excesivamente largos.





|   |   |
| --- | --- |
|   |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Audit%20statique%20-%20HC%20N%B0003.gif&type=thumb)<br> | Reporte de auditoría, análisis de la biblioteca (para detectar archivos grandes) y análisis del paquete de instalación. <br><br>Para iniciar la auditoría estática, vaya a la pestaña "Proyecto", grupo "Auditoría y rendimiento", despliegue "Auditoría estática" y seleccione "Iniciar auditoría estática". |




- **La auditoría dinámica del proyecto**, que detecta la pérdida de recursos, los errores no procesados y la lentitud de la aplicación en su entorno de producción.





|   |   |
| --- | --- |
|   |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Audit%20dynamique%20-%20HC%20N%B0002.gif&type=thumb)<br> | Analiza el rendimiento de la aplicación durante una prueba y muestra los procesos que contienen excepciones y errores que no se gestionan correctamente. <br><br>Para iniciar la auditoría dinámica, en la pestaña "Proyecto", despliegue "Modo de prueba" y seleccione "Depurar proyecto con la auditoría activada". |




## 6. Generación múltiple

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las configuraciones de proyecto permiten definir fácilmente las plataformas de destino del proyecto. Es posible definir varios ejecutables, componentes y bibliotecas para el mismo proyecto.

Puede seleccionar cada configuración de forma individual para generar el programa correspondiente.

Existe un método más rápido: la **generación múltiple**. Puede seleccionar al mismo tiempo todas las configuraciones que desea generar para obtener los programas correspondientes de forma inmediata.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_WD_Generation_multiple%20-%20HC%20N%B0001.gif)


Para iniciar una generación múltiple, en la pestaña "Proyecto", en el grupo "Generación", haga clic en "Generación múltiple".


## 13. Imprimir: Reportes

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Un reporte proporciona una vista personalizada de los datos: datos introducidos en la base de datos, de un archivo de texto, de un control Tabla, etc, ... 

Los reportes pueden mostrarse en la pantalla, guardarse en un archivo HTML, imprimirse, etc.

Los reportes permiten resumir y sintetizar datos.

Es posible:

- agrupar datos.

- ordenar datos en función de diferentes criterios.

- realizar cálculos (medias, estadísticas) o incluso crear gráficos.




El siguiente diagrama ilustra el concepto de reporte:

- los datos a imprimir provienen de una fuente de datos (archivo de datos descrito en un análisis, vista HFSQL, consulta, área de memoria o archivo de texto).

- el reporte agrupa, ordena y define el formato de los datos.

- el reporte puede ejecutarse en la pantalla, guardarse como un archivo (HTML, RTF, etc.). o enviarse a la impresora.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1-Edition-Les-Etats.gif)

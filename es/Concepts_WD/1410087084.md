
## 2. Transferencia de archivos mediante FTP

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
FTP (File Transfer Protocol) es un protocolo para transferir archivos de un sitio a otro. Este protocolo permite intercambiar archivos a través de TCP/IP o Internet.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000132"></a>
Para manipular archivos en un servidor FTP desde una aplicación WINDEV, debe cumplir las siguientes normas:

1. Conectarse a un servidor FTP (función [FTPConnect](../WDLang3/3040016.md)). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-Transfert%20de%20fichiers%20par%20FTP.gif)


2. Enviar y recibir archivos (según el modo de conexión). En el caso de una conexión pasiva, la aplicación WINDEV debe solicitar la autorización del servidor FTP antes de realizar cualquier operación en los archivos del servidor. También es posible:

	- obtener las características de los archivos ubicados en un servidor FTP: atributos, tamaño, etc.

	- manipular archivos en un servidor FTP: crear, eliminar...

	- obtener la lista de los archivos de un directorio ubicado en el servidor FTP ejecutando un procedimiento que inicie un proceso por cada archivo encontrado.
			<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-Transfert%20de%20fichiers%20par%20FTP-2.gif)





3. Cerrar la conexión con el servidor (función [FTPDisconnect](../WDLang3/3040011.md)). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-Transfert%20de%20fichiers%20par%20FTP-3.gif)

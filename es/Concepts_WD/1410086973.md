
## 7. Modelos de ventanas

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV le permite crear modelos de ventanas. Estos modelos contienen los elementos gráficos comunes a todas las ventanas de la aplicación.

Los cambios realizados en un modelo de ventanas se aplican automáticamente a todas las ventanas que lo utilizan.

Un modelo de ventanas conserva el tema gráfico definido para una aplicación.

Vista de un modelo de ventanas.
El modelo está rodeado por una línea naranja
en el editor.

Uso de un modelo en varias ventanas.
Los elementos del modelo están identificados por un cuadro amarillo.

**Observación**: La programación asociada a los elementos del modelo se puede realizar directamente en el modelo.

Las características de los elementos se pueden disociar del modelo. Por ejemplo, disociar la posición de un control del modelo para moverlo a otro lugar, pero manteniendo otros cambios (código, estilo, etc.). Esto se conoce como **herencia de controles**. En este caso, los elementos se identifican por un cuadrado azul.


## 4. WINDEV y la telefonía

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
WINDEV permite gestionar fácilmente las llamadas telefónicas entrantes y salientes a través de ciertas funciones WLanguage específicas. Estas funciones gestionan el buzón de voz, un contestador automático (u otro) directamente en una aplicación WINDEV. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P6-WinDev%20et%20la%20t%E9l%E9phonie.gif)


**Configuración necesaria**:
Para utilizar las funcionalidades de telefonía, debe tener:

- un módem.
	Para guardar y reproducir grabaciones, este módem debe incluir soporte de voz.
	Para identificar a la persona que llama, el módem debe ser compatible con el identificador de llamadas (Caller ID).

- el protocolo Tapi 3.1.

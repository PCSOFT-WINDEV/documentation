<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Centro de Seguimiento de Proyectos permite que el administrador de proyectos administre los proyectos de desarrollo. Para ello, es necesario:

- definir los diferentes colaboradores del proyecto.

- definir los requisitos (y los elementos asociados).




Cada desarrollador realiza las diferentes tareas que se le asignan.

El director de proyectos puede seguir el estado de avance del proyecto en cualquier momento.


## 1. Proyecto y Análisis

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El desarrollo de una **Aplicación Windows** con WINDEV se basa en dos elementos principales: el Proyecto y el Análisis.

Un **Proyecto WINDEV** es un conjunto de elementos (ventanas, reportes, controles, clases, componentes, etc.) que, en conjunto, permiten crear una aplicación Windows.

Un **análisis WINDEV** es un esquema de la base de datos utilizada por la aplicación. El análisis agrupa las diferentes tablas (también llamadas archivos de datos) utilizadas por la aplicación.

Una aplicación se construye a partir de un proyecto.

En la mayoría de los casos, un proyecto está asociado a un análisis.

Un análisis puede estar asociado a uno o más proyectos.

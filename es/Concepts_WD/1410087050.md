
## 1. Análisis: estructura de la base de datos

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Cuando un proyecto WINDEV, WEBDEV o WINDEV Mobile utiliza archivos de datos, debe estar asociado a un análisis. Un análisis permite describir las estructuras de los datos (archivos de datos, campos, etc.) utilizados en el proyecto.

El editor de análisis permite crear un análisis fácilmente.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P47-Analyse%20Structure%20de%20la%20base%20de%20donn%E9es.gif)


El análisis de un proyecto WINDEV, WEBDEV o WINDEV Mobile corresponde al MLD (Modelo Lógico de Datos). El análisis contiene toda la estructura y organización de los datos: los datos se agrupan por archivos (llamados archivos de datos). Cada archivo contiene múltiples datos, llamados campos.

En el análisis, la descripción de un archivo de datos se puede relacionar con un tipo de archivo (HFSQL Classic, HFSQL Client/Server, SQLite, etc.).

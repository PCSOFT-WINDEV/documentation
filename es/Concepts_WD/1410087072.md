
## 10. UMC: Macro-Código de Usuario

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El Macro-Código de Usuario (UMC) permite que los usuarios finales introduzcan su propio código WLanguage para agregarlo a una aplicación existente.

Veamos el siguiente caso:
Una ventana con un formulario de cliente requiere una dirección de correo electrónico pero no comprueba su formato. El usuario quiere comprobar si el carácter "@" está en la dirección.

- en la descripción del proyecto,

- al crear el ejecutable.

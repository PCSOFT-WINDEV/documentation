
## 5. HFSQL Cient/Server

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
HFSQL Client/Server es un poderoso sistema de gestión de bases de datos relacionales (RDBMS) en modo Cliente/Servidor.

Las características de HFSQL Client/Server son las siguientes:

- Una aplicación HFSQL Client/Server puede ejecutarse en diferentes equipos cliente que se comunican con el servidor de base de datos a través de la red.

- Los archivos de datos se encuentran en un servidor. Solo el servidor accede físicamente a los archivos de datos.

- Todos los procesos (consultas, lectura y escritura de registros en un archivo de datos, etc.) se realizan en el servidor.

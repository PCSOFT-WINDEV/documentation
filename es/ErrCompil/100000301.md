


## Error 301: Identificador no reconocido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000108"></a>
Está usando un nombre de control, constante o Variable que no existe o que no fue declarado de antemano.

**Atención**: Este error puede aparecer como un error o como una advertencia según el modo de gestión de los identificadores desconocidos elegidos en el proyecto description: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Mostrar la pestaña "Compilación".

3. El modo de compilación de identificadores desconocidos puede ser: 

	- Modo estricto: desencadena un error de compilación. 

	- Modo permisivo: realiza una declaración implícita y lanza un aviso.







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000132"></a>
Compruebe la ortografía del objeto utilizado y modifique su nombre si es necesario.

**Recordatorio**: El colorido sintáctico que se encuentra en el editor de código le permite saber si está utilizando un identificador conocido o no.: si el identificador es desconocido, su nombre se muestra en rojo.

Si la ortografía es correcta, compruebe la existencia del control, variable o Procedure y cree este control, variable o Procedure si es necesario.

**Consejo**: Para crear una Procedure directamente a partir de su nombre, posicione el cursor sobre el nombre y pulse la tecla F4.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000156"></a>


### Error encontrado en la ortografía de un Variable
<a name="error_encontrado_ortografia_variable_ELTPARAGRAPHE000052"></a>**Código que desencadena el error**


```wl
MyProject is string
...
MProject = fCurrentDir() + ProjectName
```
**Posible corrección**
Revisa la ortografía de la Variable.


```wl
MyProject is string
...
MyProject = fCurrentDir() + ProjectName
```


## Error 112: Una array sólo puede tener un máximo de dimensiones X
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Se definió una array (dinámica o no) con más de 10 dimensiones.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Reducir el número de dimensiones de array. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000131"></a>


### Array declarada con 13 dimensiones. 
<a name="array_declarada_con_13_dimensiones_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Array13 is array of 3 by 3 by 3 by 3 by 3 by 3 by 3 by 3 by 3 
		by 3 by 3 by 3 by 3 int
```


**Posible corrección**

Reducir el tamaño para respetar el valor máximo autorizado. 


```wl
Array13 is array of 3 by 3 by 3 by 3 by 3 by 3 by 3 by 3 by 3 int
```

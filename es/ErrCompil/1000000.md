
## Error 1037: Este miembro ya existe en la estructura
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Está declarando una estructura que utiliza dos miembros con el mismo nombre. Este error puede ser causado por un error tipográfico en el nombre de uno de los miembros de la estructura, por ejemplo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Renombrar o borrar el miembro extra.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Declarar una estructura que utiliza dos miembros con el mismo nombre
<a name="declarar_una_estructura_que_utiliza_dos_miembros_con_mismo_nombre_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Coord is Structure 
	PosX is int
	PosX is int
END
```


 
**Posible corrección**
Cambiar el nombre de uno de los miembros.


```wl
Coord is Structure 
	PosX is int
	PosY is int
END
```


## Error 507: El operador&apos;++&apos; no está permitido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está tratando de usar el operador aritmético ++ con un Variable que no soporta este operador.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Modifique su código y utilice el operador o la función apropiada.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Uso del operador aritmético ++ en una cadena de caracteres
<a name="uso_del_operador_aritmetico_una_cadena_caracteres_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
MyString is string = "A"
MyString++
```




**Posible corrección**

Usa el operador += para añadir un valor a la cadena de MyString Variable.


```wl
MyString is string = "A"
MyString+=MyString
```


## Error 533: La expresión de un { } debe ser una cadena
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000099"></a>
Estás usando un operador indirecto en un valor o en un Variable que no es una cadena.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000123"></a>
Comprobar y modificar el tipo de la Variable en la que se utiliza el operador indirecto. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000147"></a>


### Indirección sobre un valor numérico
<a name="indireccion_sobre_valor_numerico_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
{5}..Note = "Note"
```




**Posible corrección**

Sustituir el valor numérico por una cadena de caracteres. 


```wl
{"MyControl"}..Note = "Note"
```

<a name="NOTE3_2"></a>


### Indirección en un Variable numérico
<a name="indireccion_variable_numerico_ELTPARAGRAPHE000041"></a>

**Código que desencadena el error** 


```wl
MyControl is real = 5
{MyControl}..Note = "Note"
```




**Posible corrección**

Sustituir el valor numérico por una cadena de caracteres. 


```wl
MyControl is string = "Button1"
{MyControl}..Note = "Note"
```

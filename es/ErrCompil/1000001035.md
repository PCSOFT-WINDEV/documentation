
## Error 1035: Una clase utilizada en una estructura debe tener un constructor sin parámetros.....
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
Está utilizando una clase en una estructura. Esta clase posee un constructor que espera parámetros obligatorios.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Dos posibilidades:

- Definir parámetros opcionales en el constructor de la clase.

- No usar ningún parámetro en el constructor de la clase pero crear un método de inicialización. Los parámetros se pasarán al método de inicialización de la clase.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000134"></a>


### Usando una clase con parámetros obligatorios en la estructura Definition
<a name="usando_una_clase_con_parametros_obligatorios_estructura_definition_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
ObjectSize is structure
	Height is int
	Width is int
	Object_Position is Class1(3,4)
END
```




**Posible corrección**

Utilizar un método de inicialización.


```wl
ObjectSize is structure
	Height is int
	Width is int
	Object_Position is Class1()
END
 
ObjectSize1 is ObjectSize
ObjectSize1.Position_object:Class1:Initialization(3,4)
```

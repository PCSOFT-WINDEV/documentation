
## Error 1045: Una clase no puede declararse fuera del código de declaración de una clase.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
El código actual corresponde a una declaración de clase. Este código se encuentra en un Event que no es de la clase "Declaración" Event.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Compruebe si la clase fue creada en su proyecto y copie el código de declaración de la clase en la clase "Declaración" Event.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000131"></a>


### Código para la declaración de clase utilizado en el código del botón
<a name="codigo_para_declaracion_clase_utilizado_codigo_del_boton_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
// -- Click on BTN_Button1
AFile is class 
	Name is string
	Extension is string
	Directory is string
END
```




**Posible corrección**

Crear la clase AFile y copiar el código para declarar la clase AFile en la clase "Declaración" Event.


```wl
// -- Declaration of AFile class
AFile is class 
	Name is string 
	Extension is string
	Directory is string
END
```


## Error 1022: El acceso al método de la clase está restringido por una herencia protegida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000068"></a>
Está intentando acceder a un método de una clase que se deriva de una herencia protegida.. No se puede acceder a este método desde un código externo a esta clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000092"></a>
Dos posibilidades: 

- Acceder a este método a partir de un método de la clase derivada o a partir de un método de una de sus propias clases derivadas.. 

- Transformar la herencia protegida en herencia pública: el método será accesible desde el código de las clases derivadas así como desde un código externo a la clase.

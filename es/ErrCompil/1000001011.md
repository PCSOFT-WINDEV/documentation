
## Error 1011: Para acceder al miembro global de la clase, utilice ClassName::MemberName
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está intentando acceder a un miembro global utilizando la sintaxis utilizada para acceder a un miembro de la clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Utilice la sintaxis adecuada que le permita acceder a un miembro global de la clase en el código de la clase:

```txt
ClassName::NameGlobalMember
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Inicialización de un miembro global en el código de declaración de la clase
<a name="inicializacion_miembro_global_codigo_declaracion_clase_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
// Initializing a global member
Drawing:pLast = NULL
```




**Posible corrección**

Utilice la sintaxis apropiada.


```wl
// Initializing a global member
Drawing::pLast = NULL
```

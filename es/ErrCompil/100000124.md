
## Advertencia 124: Siguiendo a una Variable global
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Se declaró un Variable: ya existe una Variable global con el mismo nombre en la ventana o proyecto actual.

Si mantiene este nombre, la variable "global" ya no será visible en la actual Process: se usará el Variable local en su lugar.

Atención: Si se redefine un Variable global al proyecto en el código de inicialización de la ventana, el nuevo Variable se utilizará en todos los procesos de la ventana y los controles de la ventana.

**Recordatorio**: Alcance de variables

- Las variables globales declaradas en el código de inicialización del proyecto son Visible en todo el proyecto. 

- Las variables globales declaradas en el código de apertura de las ventanas son Visible en todos los procesos de la ventana y los controles de la ventana. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Consejo
<a name="consejo_ELTTEXTE000097"></a>
Evite utilizar los mismos identificadores para diferentes variables.

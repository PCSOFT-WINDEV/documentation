
## Error 1010: El miembro de la clase es constante
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000091"></a>
Está intentando modificar a un miembro constante de la clase.

**Recordatorio**: Un miembro constante es accesible en modo de sólo lectura. Un miembro puede serlo:

- **Constante protegida**: El valor del miembro puede ser leído y modificado desde dentro de la clase y el valor del miembro puede ser leído desde una clase derivada.. El acceso al miembro será prohibido desde cualquier otra sección del código.

- **Constante pública**: El valor de este miembro puede ser leído y modificado desde dentro de la clase o desde una clase derivada.. Sólo podrás leer el valor del miembro desde cualquier otra sección del código.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000115"></a>
Dos posibilidades:

- Sustituir el miembro constante por un **miembro público**: el acceso al miembro no será restringido.

- No manipular el miembro constante en modo de lectura/escritura.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000139"></a>


### Usando un miembro constante en el código de clic de un botón
<a name="usando_miembro_constante_codigo_clic_boton_ELTPARAGRAPHE000033"></a>

**Código que desencadena el error** 


```wl
// -- Declare the class
Drawing is class
	PUBLIC CONSTANT
	BaseColor is int	 // Color
END
 
// -- Click code of a button 
Drawing::BaseColor = LightRed
...
```




**Posible corrección**

Usar un miembro público.


```wl
// -- Declare the class
Drawing is class
	PUBLIC
	BaseColor is int	 // Color
END
 
// -- Click code of a button 
Drawing::BaseColor = LightRed
...
```

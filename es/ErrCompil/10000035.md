
## Error 35: Continuar&apos; está disponible sólo en los bucles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
La palabra clave CONTINUAR se usó en una sentencia que no era un bucle.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Utilice la palabra clave CONTINUAR en una instrucción de bucle o elimine esta palabra clave.

**Observación**: En una sentencia de bucle, la palabra clave CONTINUE se utiliza para finalizar inmediatamente la iteración actual.. El código que se encuentra después de la palabra clave CONTINUAR no se ejecutará.. La siguiente iteración se ejecuta automáticamente.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000134"></a>


### Usando la palabra clave CONTINUAR en un Procedure
<a name="usando_palabra_clave_continuar_procedure_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
PROCEDURE Compare(Number1, Number2)
IF Number1 > Number2 THEN 
	RETURN
ELSE
	CONTINUE
END
```




**Posible corrección**

Eliminar la palabra clave CONTINUAR.


```wl
PROCEDURE Compare(Number1, Number2)
IF Number1 > Number2 THEN 
	RETURN
ELSE
	Info(Number2 + "is greater than " + Number1)
END
```

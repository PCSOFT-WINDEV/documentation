
## Error 502: Tipo incompatible
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000081"></a>
Está intentando realizar una operación aritmética o una operación lógica entre dos expresiones de tipo incompatible.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000105"></a>
Este tipo de operación no es posible. Encuentre las funciones de WLanguage que le permiten realizar esta operación.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000129"></a>


### El siguiente código intenta realizar una resta entre dos cadenas de caracteres
<a name="siguiente_codigo_intenta_realizar_una_resta_entre_dos_cadenas_caracteres_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
string1 is string
string2 is string
Trace(string1-string2)
```




**Posible corrección**

No se puede realizar ninguna sustracción de cadenas de caracteres en WLanguage.

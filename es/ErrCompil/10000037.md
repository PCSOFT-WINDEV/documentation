
## Información 37: Optimice su bucle FOR
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
El código de su bucle FOR utiliza un contador cuyo tipo es:

- 2 bytes int

- 4 bytes reales




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Utilice un contador entero o un contador real.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000134"></a>


### Uso de un contador entero de 2 bytes
<a name="uso_contador_entero_2_bytes_ELTPARAGRAPHE000028"></a>

**Código que desencadena el mensaje de información** 


```wl
I is 2-byte int
FOR I = 1 TO 5
	ListAdd(LIST_List1,"Element "+I)
END
```




**Posible corrección**

Usar un contador entero.


```wl
I is int
FOR I = 1 TO 5
	ListAdd(LIST_List1,"Element "+I)
END
```


## Error 1055: Ancestro&apos; sólo puede utilizarse en el código de una clase derivada de una clase
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000070"></a>
Está intentando usar la palabra clave'Antepasado' en el código de una clase no derivada..


 
**Recordatorio**: La palabra clave "Ancestro" se utiliza para especificar, en el código de una clase derivada, la clase ancestral associated con el método o con el miembro a utilizar.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000094"></a>
No utilice la palabra clave"Antepasado".

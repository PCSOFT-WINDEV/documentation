
## Error 1060: No se puede llamar al método sin parámetro: add ()
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está intentando utilizar un método sin especificar los paréntesis.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Añada los paréntesis y los parámetros esperados por el método (si es necesario).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Usando el método Destruir de la clase Dibujo
<a name="usando_metodo_destruir_clase_dibujo_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Drawing::Destroy
```


 
**Posible corrección**


Añada los corchetes.


```wl
Drawing::Destroy()
```

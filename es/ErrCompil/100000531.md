
## Error 531: Los subíndices negativos o nulos no están permitidos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000092"></a>
Está utilizando los operadores para la extracción de cadenas ([[ o ]]). La extracción índice para una cadena WLanguage siempre comienza desde 1: el índice del primer personaje es set a 1.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000116"></a>
Modificar el valor especificado para la extracción índice.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000140"></a>


### Extracción del primer carácter de la cadena 
<a name="extraccion_del_primer_caracter_cadena_ELTPARAGRAPHE000035"></a>

**Código que desencadena el error**


```wl
sString is string = "WEBDEV is great"
Trace(sString[[0]])
```


**Posible corrección**

Reemplazar 0 por 1 (primer carácter de la cadena).


```wl
sString is string = "WEBDEV is great"
Trace(sString[[1]])
```

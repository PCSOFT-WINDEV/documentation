
## Error 106: Redefinición de una Variable global
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Usted está tratando de declarar una Variable global. El nombre de este Variable ya ha sido utilizado por otro Variable global encontrado en el proyecto o en la ventana actual.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Renombrar una de las variables o borrar la declaración inútil.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Se utilizan dos variables globales con el mismo nombre pero con diferentes tipos
<a name="utilizan_dos_variables_globales_con_mismo_nombre_pero_con_diferentes_tipos_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
// -- Initialization code of project
Global
	b is int
 
	b is string
```




**Posible corrección**

Renombrar el segundo Variable.


```wl
// -- Initialization code of project
Global
	b is int
 
	c is string
```

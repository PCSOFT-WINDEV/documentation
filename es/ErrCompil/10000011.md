
## Error 11: La etiqueta es desconocida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Esta Line de código utiliza la sentencia GOTO. La etiqueta utilizada en la declaración GOTO no estaba definida en el código actual.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Verifique si la etiqueta existe y utilice su nombre en la sentencia GOTO (la coloración sintáctica le permite verificar la existencia del nombre de la etiqueta).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Se ha cometido un error tipográfico en nombre de la etiqueta
<a name="cometido_error_tipografico_nombre_etiqueta_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res,"Process OK")
IF Res = -1 THEN GOTO ERRRITE
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERRWRITE: 
Info("Unable to write into " + FileName)
```


**Posible corrección**

Compruebe el nombre de las etiquetas y corrija el error tipográfico. En este ejemplo, ERRWITE se sustituye por ERRWRITE.


```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res,"Process OK")
IF Res = -1 THEN GOTO ERRWRITE
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERRWRITE: 
Info("Unable to write into " + FileName)
```

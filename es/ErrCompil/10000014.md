
## Error 14: Sentencia ELSE sin IF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000082"></a>
Está utilizando la palabra clave ELSE sin utilizar la sentencia condicional correspondiente IF.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000106"></a>
Compruebe el código de su estado de cuenta condicional.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000130"></a>


### No se ha introducido el principio de la expresión condicional
<a name="introducido_principio_expresion_condicional_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
EDT_Lesson = "French"
ELSE
	EDT_Lesson = "English"
END
```


**Posibles correcciones**
Completar la declaración condicional.


```wl
IF TableSelect(TABLE_Lesson) = 1 THEN
	EDT_Lesson = "French"
ELSE
	EDT_Lesson = "English"
END
```

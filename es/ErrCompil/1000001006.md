
## Error 1006: El miembro no es global: uso: para acceder a él
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000096"></a>
Está intentando acceder a un miembro de la clase con la sintaxis utilizada para acceder a un miembro global..


Se está utilizando la siguiente sintaxis:



```wl
ClassName::MemberName
```


Esta sintaxis no puede utilizarse para acceder a un miembro no global.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000120"></a>
Utilice la sintaxis adecuada para acceder a un miembro no global de la clase:


```wl
ClassName:MemberName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000144"></a>


### Usando el miembro BaseColor de la clase Dibujo
<a name="usando_miembro_basecolor_clase_dibujo_ELTPARAGRAPHE000036"></a>



**Código que desencadena el error**


```wl
Drawing::BaseColor
```



**Posible corrección**

Utilice la sintaxis correcta.


```wl
Drawing:BaseColor
```

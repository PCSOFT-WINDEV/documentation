
## Error 1049: Las directivas de acceso no están permitidas en un Variable compuesto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está intentando usar algunas directivas de acceso (: PÚBLICO, PRIVADO, PROTEGIDO) durante la Definition de un compuesto Variable. 

No se puede restringir el acceso a un miembro de un Variable compuesto: todos los miembros de un compuesto Variable son públicos.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Eliminar la palabra clave correspondiente a una directiva de acceso: PÚBLICOS, PRIVADOS Y PROTEGIDOS.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Definición de una estructura
<a name="definicion_una_estructura_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
Coord is composed of
	 PRIVATE
	 PosX is int
	 PosY is int
END
```




**Posible corrección**

Borrar la palabra clave PRIVADA.


```wl
Coord is composed of
	PosX is int
	PosY is int
END
```

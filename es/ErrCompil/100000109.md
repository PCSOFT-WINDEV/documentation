
## Error 109: La cadena fija es demasiado larga
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está utilizando una cadena fija cuyo tamaño excede el tamaño máximo autorizado.

Recordatorio: El tamaño máximo de una cadena fija no puede superar los 32 KB (32767 bytes).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Modifique el tamaño de su cadena fija según el valor máximo autorizado.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### La cadena fija es demasiado larga
<a name="cadena_fija_demasiado_larga_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
B is string fixed of 40000
```




**Posible corrección**

Reducir el tamaño para respetar el valor máximo autorizado.


```wl
B is string fixed of 30000
```


## Error 705: El Property no existe en este tipo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Estás tratando de dirigir un Property. Este Property no puede ser usado en el tipo especificado.

**Observación**: La terminación automatic automatic ally propone las propiedades correspondientes al tipo de objeto utilizado. Si el Property no se encuentra en esta lista, no puede ser usado en este tipo de objeto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Vea la ayuda en línea para encontrar el Property solicitado y el tipo en el que se puede aplicar este Property.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Usar el nombre Property en un Variable booleano
<a name="usar_nombre_property_variable_booleano_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
b is boolean
b..Name = "C"
```




**Posible corrección**

Utilice otra Property, otra función, etc.

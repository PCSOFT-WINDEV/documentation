
## Error 1017: El método no es global: uso: para acceder a él
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000091"></a>
Está intentando acceder a un método de la clase utilizando la sintaxis utilizada para acceder a un método global.

Se está utilizando la siguiente sintaxis:

```txt
ClassName::MethodName
```


Esta sintaxis no se puede utilizar para acceder a un método no global.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000115"></a>
Usar la sintaxis apropiada para acceder a un método (no global) de la clase: 

```txt
ClassName:MethodName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000139"></a>


### Uso del método no global Visualización de la clase Dibujo
<a name="uso_del_metodo_global_visualizacion_clase_dibujo_ELTPARAGRAPHE000033"></a>

**Código que desencadena el error** 


```wl
Drawing::Display()
```




**Posible corrección**

Utilice la sintaxis correcta.


```wl
Drawing:Display()
```


## Error 13: La etiqueta ya existe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000082"></a>
El código utiliza la sentencia GOTO con etiquetas de ramificación. Una etiqueta con el mismo nombre ya existe en este código.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000106"></a>
No se pueden utilizar varias etiquetas con el mismo nombre en el mismo código. Cambie el nombre de sus etiquetas para utilizar nombres diferentes para todas las etiquetas definidas en este código.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000130"></a>


### Debido a un error tipográfico, dos etiquetas con el mismo nombre se encuentran en el código actual
<a name="debido_error_tipografico_dos_etiquetas_con_mismo_nombre_encuentran_codigo_actual_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res,"Process OK")
IF Res = -1 THEN GOTO ERRWRITE
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERROPEN: 
Info("Unable to write into " + FileName)
```


**Posibles correcciones**

Modificar el nombre de la etiqueta duplicada: el segundo ERROPEN debe ser sustituido por ERRWRITE.


```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res,"Process OK")
IF Res = -1 THEN GOTO ERRWRITE
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERRWRITE: 
Info("Unable to write into " + FileName)
```

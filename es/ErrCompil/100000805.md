
## Error 805: No hay sintaxis sin parámetro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está usando una función WLanguage (o Procedure) sin indicar los corchetes. Esta función no acepta sintaxis sin parámetro y, por lo tanto, sin paréntesis.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Completa la llamada a la función o a la Procedure especificando los parámetros obligatorios.

**Observación**: Consulte la ayuda en línea para conocer la sintaxis detallada de esta función.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Uso de la función Ln
<a name="uso_funcion_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
Res = Ln
```




**Posible corrección**

Completar la sintaxis.


```wl
Res = Ln(5)
```


## Error 4: Se requiere un espacio después del carácter &apos;:&apos;.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
Esta Line de código se utiliza, por ejemplo, para asignar o acceder a un miembro de una clase en un bucle SWITCH. El carácter ':' está acostumbrado sucesivamente:

- especifique la condición (CASO: ),

- especifique el miembro de la clase.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Añadir un espacio entre los dos "colones".. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000134"></a>


### Asignación de un miembro de la clase actual
<a name="asignacion_miembro_clase_actual_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
SWITCH NbDir
	CASE 1: i=1
	CASE 2: i=2
	CASE 5::b=4
END
```




**Posible corrección**

Insertar un espacio entre los caracteres **::**. 


```wl
SWITCH NbDir
	CASE 1: i=1
	CASE 2: i=2
	CASE 5:: b=4
END
```


## Error 1033: Las directivas de acceso no están permitidas en una estructura
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está intentando usar algunas directivas de acceso (: PÚBLICO, PRIVADO, PROTEGIDO) durante la Definition una estructura. El acceso a un miembro de una estructura no puede ser restringido: todos los miembros de una estructura son públicos.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Eliminar la palabra clave correspondiente a una directiva de acceso: PÚBLICOS, PRIVADOS Y PROTEGIDOS.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Definición de una estructura
<a name="definicion_una_estructura_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Coord is structure 
	 PRIVATE
	 PosX is int
	 PosY is int
END
```




**Posible corrección**

Borrar la palabra clave PRIVADA.


```wl
Coord is structure 
	PosX is int
	PosY is int
END
```


## Error 1005: Para acceder explícitamente a un elemento global de la clase&apos;%1&apos;, utilice la sintaxis
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000089"></a>
Está intentando utilizar un elemento global (miembro, método o constante) de la clase base de la clase derivada.. 

La sintaxis utilizada es incorrecta. Se está utilizando la siguiente sintaxis:

```txt
:NameBaseClass:ElementName
```


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000113"></a>
Para utilizar un elemento global (miembro, método o constante) de la clase base a partir de una clase derivada, el elemento debe ir precedido por el nombre de la clase base..

```txt
NameBaseClass:ElementName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000137"></a>


### Usando el último miembro global definido en la clase Drawing a partir de un método de la clase derivada CIRCLE
<a name="usando_ultimo_miembro_global_definido_clase_drawing_partir_metodo_clase_derivada_circle_ELTPARAGRAPHE000031"></a>

**Código que desencadena el error** 


```wl
:Drawing::plast
```




**Posible corrección**

Borrar los dos puntos (: carácter) delante del nombre de la clase base.


```wl
Drawing::pLast
```

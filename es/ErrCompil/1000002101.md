
## Información 2101: El&apos;Short int&apos; ya no existe: use &apos;unsigned 1-byte int&apos; (sin signo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000082"></a>
Se está utilizando un ***entero corto***. Este tipo de datos ya no existe en WINDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000106"></a>
Reemplace el tipo Short Integer por el tipo ***Unsigned 1-byte integer***.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000130"></a>


### Usando un entero corto
<a name="usando_entero_corto_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
MyData is short int
```


**Posible corrección**

Utilice el tipo "entero de 1 byte sin signo".


```wl
MyData is unsigned 1-byte int
```

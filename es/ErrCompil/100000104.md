
## Error 104: Dynamic&apos; sólo se puede utilizar para las clases y los objetos de automatización
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000089"></a>
En este código Line, la palabra clave **Dynamic** se utiliza para definir un objeto que no puede ser dinámico.

**Recordatorio**: El **Dinámico** sólo se puede utilizar con los siguientes tipos de palabras clave:

- class

- tabla

- estructura

- automation object




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000113"></a>
Elimine la palabra calve **dynamic**.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000137"></a>


### Un entero se declara como entero dinámico
<a name="entero_declara_como_entero_dinamico_ELTPARAGRAPHE000032"></a>

**Código que desencadena el error** 


```wl
NbOrder is dynamic int
```


**Posible corrección**

Elimine la palabra clave **Dynamic**.


```wl
NbOrder is int
```


## Error 802: Sintaxis no válida para esta función
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está intentando utilizar una función WLanguage con una sintaxis incorrecta.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Revise las sintaxis disponibles en la ayuda de onLine y modifique su código Line.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Se pasan demasiados parámetros a la función llamada Random
<a name="pasan_demasiados_parametros_funcion_llamada_random_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Random(5, 7, 8)
```




 
**Posible corrección**
Compruebe la sintaxis en la ayuda online y pase 2 parámetros a la función.


```wl
Random(5, 7)
```

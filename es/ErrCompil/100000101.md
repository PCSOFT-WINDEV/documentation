
## Error 101: No se permite la declaración de Variable global
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000090"></a>
Usted está tratando de declarar una variable global en un Process donde este tipo de declaración no está permitida.


**Recordatorio**: Las variables globales deben declararse en:

- El código de inicialización del proyecto para ser utilizado en todos los procesos del proyecto (código de proyecto, código de ventanas y código de controles).

- El código de apertura de una ventana para ser utilizado en todos los procesos de la ventana (código de la ventana y código de los controles que se encuentran en la ventana).

- El código de carga de la página para ser utilizado en todos los procesos del navegador de la página (código de la página y código de los controles encontrados en la página).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000114"></a>
Declare su Variable global en el código apropiado.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000138"></a>


### Un Variable global se declara en el código de clic de un botón que se encuentra en la ventana "WIN_Menu".
<a name="variable_global_declara_codigo_clic_boton_que_encuentra_ventana_winmenu_ELTPARAGRAPHE000032"></a>

**Código que desencadena el error** 


```wl
// -- Click code of BTN_Display
Global
	Result_Read is boolean
```




**Posible corrección**

Declarar la variable global en las "Declaraciones globales" Process de la ventana.


```wl
// -- Global declarations of WIN_MENU
Global
	Result_Read is boolean
```

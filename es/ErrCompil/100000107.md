
## Error 107: Redefinición del parámetro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000107"></a>
Está utilizando el mismo nombre:

- para dos parámetros en la misma Procedure,

- para un parámetro y una variable local del Procedure.




**Recordatorio**:

- Los parámetros de un Procedure se consideran variables locales de este Procedure.

- Los parámetros del código de apertura de la ventana (lo que significa en la Procedure declarada en el código de apertura de la ventana) se consideran como variables globales a toda la ventana.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000131"></a>
Cambiar el nombre de uno de los parámetros o borrar la segunda declaración (que es inútil).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000155"></a>


### Un parámetro de la Procedure y una variable local de la Procedure tienen el mismo nombre
<a name="parametro_procedure_una_variable_local_procedure_tienen_mismo_nombre_ELTPARAGRAPHE000033"></a>

**Código que desencadena el error**


```wl
PROCEDURE CalcPrice(Price, VATRate)
 
Price is real
```




**Posible corrección**

Cambie el nombre de la Variable o el parámetro.


```wl
PROCEDURE CalcPrice(Price, VATRate)
 
UnitPrice is real
```

<a name="NOTE3_2"></a>


### Varios parámetros de la Procedure tienen el mismo nombre
<a name="varios_parametros_procedure_tienen_mismo_nombre_ELTPARAGRAPHE000049"></a>**Código que desencadena el error**


```wl
PROCEDURE CalcPrice(Price, VATRate, Price)
```


 
**Posible corrección**

Eliminar el parámetro inútil.


```wl
PROCEDURE CalcPrice(Price, VATRate)
```

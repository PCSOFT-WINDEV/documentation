
## Error 111: La cuerda de Pascal %1 es demasiado larga
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
Está utilizando una cadena Pascal cuyo tamaño excede el tamaño máximo autorizado.


**Recordatorio**: El tamaño máximo de una cadena de Pascal es de 255 caracteres.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Modifique el tamaño de su cadena Pascal según el valor máximo autorizado.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000134"></a>


### La cadena fija es demasiado larga
<a name="cadena_fija_demasiado_larga_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
B is Pascal string of 700
```




**Posible corrección**

Reducir el tamaño para respetar el valor máximo autorizado.


```wl
B is Pascal string of 200
```

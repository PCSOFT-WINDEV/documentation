
## Error 703: El Property no existe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
El Property utilizado no existe en WLanguage. Este error es probablemente causado por un error tipográfico.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Compruebe el nombre de la Property y modifique su código en consecuencia.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Usando la Property DE VALOR
<a name="usando_property_valor_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
BTN_Button1..Valu
```




**Posible corrección**

El nombre de la Property es Valor.


```wl
BTN_Button1..Value
```

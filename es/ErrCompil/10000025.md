
## Error 25: No hay declaración de THEN es associated con este IF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000097"></a>
Está utilizando una expresión condicional (IF). Esta declaración debe incluir la palabra clave ENTONCES.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000121"></a>
Compruebe el código de su sentencia condicional y añada la sentencia THEN si es necesario.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000145"></a>


### Usando una declaración IF en un Line
<a name="usando_una_declaracion_line_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
IF TABLE_Lesson..Empty 
	IMG_Draw..State = Grayed
```


**Posible corrección**
Añada la palabra clave ENTONCES entre la condición y la acción a realizar si la condición es correcta.


```wl
IF TABLE_Lesson..Empty THEN IMG_Draw..State = Grayed
```

<a name="NOTE3_2"></a>


### Uso de una sentencia IF en varias líneas
<a name="uso_una_sentencia_varias_lineas_ELTPARAGRAPHE000040"></a>

**Código que desencadena el error**


```wl
IF TABLE_Lesson..Empty 
	IMG_Draw..State = Grayed
END
```


**Posible corrección**
Añada la palabra clave ENTONCES entre la condición y la acción a realizar si la condición es correcta.


```wl
IF TABLE_Lesson..Empty THEN
	IMG_Draw..State = Grayed
END
```

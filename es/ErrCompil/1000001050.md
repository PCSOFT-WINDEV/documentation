
## Error 1050: Este miembro ya existe en el compuesto Variable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Usted está declarando un compuesto Variable que utiliza dos miembros con el mismo nombre. Este error puede ser causado por una errata en el nombre de un miembro del compuesto Variable por ejemplo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Renombrar o borrar el miembro extra.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Declarando una Variable compuesta que utiliza dos miembros con el mismo nombre
<a name="declarando_una_variable_compuesta_que_utiliza_dos_miembros_con_mismo_nombre_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Coord is composed of 
	PosX is int
	PosY is int
	PosX is int
END
```


 
**Posible corrección**
Borrar uno de los miembros.


```wl
Coord is composed of 
	PosX is int
	PosY is int
END
```

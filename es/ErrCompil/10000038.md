
## Error 38: Falta la sentencia END que marca el final de la declaración de estructura.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Estás declarando una estructura. Falta la palabra clave END que marca el final de la declaración de estructura.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Indique el final de la declaración de estructura utilizando la palabra clave END.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Declaración de estructura incompleta
<a name="declaracion_estructura_incompleta_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
MyClass is structure
	Student is string
	Level is int
	Year is Date
```




**Posible corrección**

Especifique el final de la estructura con la palabra clave END.


```wl
MyClass is structure
	Student is string
	Level is int
	Year is Date
END
```

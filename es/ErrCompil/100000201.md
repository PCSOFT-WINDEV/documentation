
## Error 201: El método ya tiene un atributo virtual
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Usted está declarando un método virtual: la palabra clave "VIRTUAL" se utilizó dos veces durante la declaración del método virtual.

**Recordatorio**: Según Default, un método no es virtual cuando se crea.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Eliminar la palabra clave extra "VIRTUAL".

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Declarar un método virtual
<a name="declarar_metodo_virtual_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
VIRTUAL PRIVATE VIRTUAL PROCEDURE Calculate_StockMarket()
```


**Posible corrección**

Elimina la palabra clave VIRTUAL extra.


```wl
VIRTUAL PRIVATE PROCEDURE Calculate_StockMarket()
```


## Error 123: La constante ya existe
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000099"></a>
Está intentando declarar una constante que ya fue declarada en el proyecto o en la ventana actual.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000123"></a>
Renombrar una de las constantes o borrar la declaración inútil.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000147"></a>


### Se utilizan dos constantes en la misma Process con el mismo valor
<a name="utilizan_dos_constantes_misma_process_con_mismo_valor_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
// -- Initialization code of project
constant B=5
 
Constant
	B = 5
END
```




**Posible corrección**

Una de las declaraciones es inútil. Borrar la constante extra.


```wl
// -- Initialization code of project
constant B=5
```

<a name="NOTE3_2"></a>


### Una constante se define en el código del proyecto y luego en el código de inicialización de una ventana.
<a name="una_constante_define_codigo_del_proyecto_luego_codigo_inicializacion_una_ventana_ELTPARAGRAPHE000041"></a>

**Código que desencadena el error** 


```wl
// -- Initialization code of project
constant B = 5
 
// -- Global declarations of the window
Constant
	B = 7
END
```




**Posible corrección**

Renombrar la segunda declaración.


```wl
// -- Initialization code of project
constant B = 5
 
// -- Global declarations of the window
Constant
	D = 7
END
```

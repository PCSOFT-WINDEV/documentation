
## Error 1029: El acceso a la clase base está restringido por una herencia protegida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000068"></a>
Intenta acceder a una clase base de una herencia. Esta clase fue derivada por una herencia protegida: no se puede acceder a esta clase desde un código externo a la clase o desde un código de una de sus clases derivadas.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000092"></a>
Dos posibilidades: 

- Acceda a la clase base desde un método de la clase derivada.

- Utilizar una herencia pública: la clase será accesible desde un código externo a la clase, así como desde un código de sus clases derivadas.

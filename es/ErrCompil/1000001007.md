
## Error 1007: El miembro es global: utilice el operador::.: para acceder a él
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000091"></a>
Está intentando acceder a un miembro global de la clase utilizando la sintaxis que le permite acceder a un miembro no global.

Se está utilizando la siguiente sintaxis:

```txt
ClassName:MemberName
```


Esta sintaxis no se puede utilizar para acceder a un miembro global.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000115"></a>
Usar la sintaxis apropiada para acceder a un miembro global de la clase: 

```txt
ClassName::MemberName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000139"></a>


### Usando el último miembro global de la clase Dibujo
<a name="usando_ultimo_miembro_global_clase_dibujo_ELTPARAGRAPHE000033"></a>

**Código que desencadena el error** 


```wl
Drawing::pLast
```




**Posible corrección**

Corregir la sintaxis utilizada.


```wl
Drawing::pLast
```

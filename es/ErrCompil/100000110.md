
## Error 110: La cadena ASCIIZ es demasiado larga
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Está utilizando una cadena ASCIIZ cuyo tamaño excede el tamaño máximo autorizado.

**Recordatorio**: El tamaño máximo de una cadena ASCIIZ no puede superar los 64 KB (65535 bytes).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000109"></a>
Modifique el tamaño de su cadena ASCIIZ según el valor máximo autorizado.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### La cadena fija es demasiado larga
<a name="cadena_fija_demasiado_larga_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
B is ASCIIZ string of 70000
```




**Posible corrección**

Reducir el tamaño para respetar el valor máximo autorizado.


```wl
B is ASCIIZ string of 60000
```

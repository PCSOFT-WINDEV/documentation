
## Error 2109: Está utilizando la función ObjectInfo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000162"></a>
Está intentando utilizar el ***ObjectInfo*** función. Esta función ya no existe en WINDEV. Utilice una de las funciones de sustitución.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000186"></a>
Reemplace la función ***ObjectInfo*** por una de las nuevas funciones del WLanguage:



|   | description | Función WLanguage correspondiente |
| --- | --- | --- |
| **ObjectInfo(xx,0)** | Lista de elementos que se encuentran en una ventana, en un grupo de controles, en una pestaña, etc. | [EnumControl](../WDLang1/3025005.md) |
| **ObjectInfo(xx,1)** |
| **ObjectInfo(2)** | Nombre del elemento que contiene la control (nombre de la ventana, nombre de la control Tabla, supercontrol, etc.) | [ParentObject](../WDLang1/3025031.md) |
| **ObjectInfo(4)** | Nombre de la pestaña que contiene el control | [ControlTab](../WDLang1/3025028.md) |
| **ObjectInfo(11)** | El nombre de la control que se encuentra en el aire con el cursor del ratón | [ControlOver](../WDLang1/3025021.md) |
| **InformaciónObjeto(15,4)** | Manipulación de elementos de control Combo Box (área de entrada o lista del control Combo Box) | [Handle](../WDLang1/3015005.md) |
| **ObjectInfo(22)** | Configura el tiempo de espera para abrir una ventana de sistema (abierta por funciones como Error, Info, etc.) | [DelayBeforeClosing](../WDLang1/3038040.md) |
| **ObjectInfo(26)** | Nombre de una columna según su índice | [TableEnumColumn](../WDLang1/3074012.md) |
| **ObjectInfo(34)** | Modificación de los colores en un tooltip | [ToolTipColor](../WDLang1/3025015.md) |
| **ObjectInfo(35)** | Fija una columna de una control Tabla | [TablaFixColumn](../WDLang1/3074014.md) |
| **ObjectInfo(36)** | Mueve una columna en una control Tabla | [TableMoveColumn](../WDLang1/3074011.md) |
| **ObjectInfo(39)** | Lista de columnas ordenadas | [TableSortedColumn](../WDLang1/3074010.md) |
| **ObjectInfo(43)** | Combo Box de llenado automático: permite (o no) recuperar el valor definido por gLink en un cuadro de lista o control Combo Box | [gLinkActivate](../WDLang1/3029036.md), [gStoredValueEnable](../WDLang1/1410088364.md) |

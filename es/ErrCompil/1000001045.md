
## Error 1046: Parámetro incorrecto para el constructor de la clase
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Está intentando definir un objeto de una clase. El número de parámetros no corresponde al número de parámetros esperados por la clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Compruebe los parámetros esperados por el constructor de la clase y modifique su código.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Pasar parámetros a un constructor
<a name="pasar_parametros_constructor_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
// -- Click code on BTN_CreateCircle
pDrawing is Circle dynamic = new Circle(gpDrawing, x, y, width)
```

```wl
// -- Constructor of "Circle" class
PROCEDURE Circle::Constructor(pt, aa, bb, cc, dd)
```




**Posible corrección**

Pasar el número esperado de parámetros.


```wl
// -- Click code on BTN_CreateCircle
pDrawing is Circle dynamic = new Circle(gpDrawing, x, y, width, height)
 
// -- Constructor of "Circle" class
PROCEDURE Circle::Constructor(pt, aa, bb, cc, dd)
```


## Error 2: Línea incompleta
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000120"></a>
La actual Line de código está incompleta: la actual declaración de WLanguage no terminó. Los casos más comunes son:

- La declaración utilizada no está completa: por ejemplo, falta un corchete.

- El código Line está escrito sobre el siguiente Line.




**Recordatorio**: En el editor de códigos, ir a la siguiente Line (con la tecla ENTER) se considera como un separador de declaraciones.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000144"></a>
Compruebe la sintaxis de la función y complete la Line.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000168"></a>


### La sintaxis de la función es incompleta: falta el soporte de cierre
<a name="sintaxis_funcion_incompleta_falta_soporte_cierre_ELTPARAGRAPHE000030"></a>

**Código que desencadena el error** 


```wl
HReadFirst(Customer, Name
```




**Posible corrección**

Añade el soporte de cierre al final de la Line.


```wl
HReadFirst(Customer, Name)
```

<a name="NOTE3_2"></a>


### Expresión de cálculo incorrecta
<a name="expresion_calculo_incorrecta_ELTPARAGRAPHE000046"></a>

**Código que desencadena el error** 


```wl
MyVariable = 3 +
```




**Posible corrección**

Complete la Line de código si es necesario.


```wl
MyVariable = 3 + 3
```

<a name="NOTE3_3"></a>


### La Line de código se escribe sobre la siguiente Line
<a name="line_codigo_escribe_sobre_siguiente_line_ELTPARAGRAPHE000062"></a>

**Código que desencadena el error** 


```wl
// Add rows in a Table control populated programmatically
TableAdd(TABLE_TABLE1, CUSTOMER.CONTACTNAME + TAB + CUSTOMER.COMPANYNAME +
	TAB + CUSTOMER.CUSTOMERCITY
```




**Posible corrección**
Utilice "..." al final de la Line para indicar que la Line actual de código continúa sobre la siguiente Line.


```wl
// Add rows in a Table control populated programmatically
TableAdd(TABLE_TABLE1, CUSTOMER.CONTACTNAME + TAB + CUSTOMER.COMPANYNAME + ...
	TAB + CUSTOMER.CUSTOMERCITY)
```

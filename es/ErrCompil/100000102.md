
## Error 102: Las constantes se pueden definir en el código de inicialización del proyecto, ....
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000093"></a>
Está tratando de declarar una constante en una Process donde este tipo de declaración no está permitida.

**Recordatorio**: Las constantes deben ser declaradas en:

- El código de inicialización del proyecto para ser utilizado en todos los procesos del proyecto (código de proyecto, código de ventanas y código de controles).

- El código de apertura de una ventana o página para ser utilizado en todos los procesos de la ventana o página (códigos de ventana/página y códigos de los controles que se encuentran en la ventana/página).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000117"></a>
Declare su constante en el código apropiado.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000141"></a>


### Se declara una constante en el código de clic de un botón que se encuentra en la ventana "WIN_Menu".
<a name="declara_una_constante_codigo_clic_boton_que_encuentra_ventana_winmenu_ELTPARAGRAPHE000030"></a>

**Código que desencadena el error** 


```wl
// -- Click code of BTN_DISPLAY
Constant 
	VATRate = 22.6
END
```


**Posibles correcciones**
Declarar la constante en el código de apertura de la ventana.


```wl
// -- Opening code of WIN_Menu
Constant
	VATRate = 22.6
END
```


Declarar la constante en el código de declaración de las variables globales de la página.


```wl
// -- Declaration code of global variables of PAGE_Menu
Constant 
	VATRate = 22.6
END
```

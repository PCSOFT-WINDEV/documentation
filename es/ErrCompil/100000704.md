
## Error 704: El Property no existe en
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Estás tratando de dirigir un Property. Este Property no puede ser usado en el tipo de control especificado o Variable.

**Observación**: La terminación automatic automatic ally propone las propiedades correspondientes al tipo de objeto utilizado. Si el Property no se encuentra en esta lista, no puede ser usado en este tipo de objeto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Vea la ayuda en línea para encontrar el Property solicitado y el tipo de control sobre el que se puede aplicar este Property.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Usando la abreviatura Property en un control
<a name="usando_abreviatura_property_control_ELTPARAGRAPHE000027"></a>

**Código que desencadena el error** 


```wl
Trace(BTN_Button1..Abbreviation)
```




**Posible corrección**

Usa otra Property.


## Error 202: El método ya tiene un atributo de acceso (público, protegido o privado)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000088"></a>
Está utilizando un método de clase. Se han asignado varios niveles de acceso a este método. Un método sólo puede tener un único nivel de acceso.

**Recordatorio**: un método puede tener uno de los siguientes 3 niveles de acceso: 

- reservado: acceso permitido dentro del código de la clase.

- Protegido: acceso permitido dentro del código de la clase y de las clases derivadas.

- colectividad: acceso sin restricciones (modo de acceso Default).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000112"></a>
Modifique su código para tener un solo nivel de acceso.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000136"></a>


### Declaración de un método de clase
<a name="declaracion_metodo_clase_ELTPARAGRAPHE000031"></a>

**Código que desencadena el error** 


```txt
PROTECTED PUBLIC METHOD CalculateStockMarket(Share)
```


**Posible corrección**

Especifique un nivel de acceso único para este método.


```txt
PUBLIC METHOD CalculateStockMarket(Share)
```


## Error 701: El Property es de sólo lectura
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000081"></a>
Esta línea de código trata de asignar un valor a un Property de sólo lectura.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000105"></a>
Modifique su código para usar esta Property en sólo lectura en vez de en lectura/escritura.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000129"></a>


### Usando la abreviatura Property
<a name="usando_abreviatura_property_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
CUSTOMER..Abbreviation = "AB"
```




**Posible corrección**

Borre el código Line.

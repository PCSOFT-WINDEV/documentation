
## Error 532: El tamaño de la cadena a extraer debe ser estrictamente positivo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000092"></a>
Está utilizando los operadores para la extracción de cadenas ([[ o ]]). El tamaño de la cadena a extraer debe ser positivo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000116"></a>
Modificar el valor del tamaño de la cadena a extraer.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000140"></a>


### Extracción de una parte de un string
<a name="extraccion_una_parte_string_ELTPARAGRAPHE000035"></a>

**Código que desencadena el error**


```wl
sString is string ="WEBDEV is great"
Trace(sString[[1 on 0]])
```


**Posible corrección**

Modificar el valor extraído para que el tamaño de la cadena extraída sea positivo..


```wl
sString is string ="WEBDEV is great"
Trace(sString[[1 on 5]])
```

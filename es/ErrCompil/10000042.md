
## Error 42: Etiqueta fuera del alcance de GOTO
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Una etiqueta GOTO se utiliza en un bucle FOR. Sin embargo, la expresión GOTO correspondiente no se encuentra en la expresión FOR.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>


- Incluir la declaración GOTO y la etiqueta correspondiente en la declaración FOR. 

- Elimine la etiqueta de la instrucción GOTO del bucle FOR.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### Acceso a una etiqueta declarada en el código de una sentencia de bucle
<a name="acceso_una_etiqueta_declarada_codigo_una_sentencia_bucle_ELTPARAGRAPHE000026"></a>

**Código que desencadena el error** 


```wl
I is int
MyArray is array of 10 int
...
GOTO WRITE2
 
FOR I = 1 TO 10
	MyArray[I] = 0
 
	WRITE2: 
	MyArray[I-1]=I
 
END
```




**Posible corrección**

Borrar la etiqueta.


```wl
I is int
MyArray is array of 10 int
...
 
FOR I = 1 TO 10
	MyArray[I] = 0
 
	MyArray[I-1]=I
 
END
```

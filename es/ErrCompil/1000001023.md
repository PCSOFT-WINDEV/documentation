
## Error 1014: El acceso al miembro de la clase está restringido por una herencia privada.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000071"></a>
Está intentando acceder a un miembro de un grupo previamente derivado de una herencia privada. No se puede acceder a este miembro desde:

- Un código externo a la clase derivada (o externo a sus propias clases derivadas).

- Un código de las clases derivadas de la primera clase derivada de una herencia privada.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000095"></a>
Dos posibilidades: 

- Acceder a este miembro desde un método de la clase derivada. 

- Transformar la herencia privada en herencia pública: el miembro será accesible desde el código de las clases derivadas así como desde un código externo a la clase.

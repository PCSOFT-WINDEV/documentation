
## Advertencia 127: Está usando una fuente de datos local de la Process
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Esta línea de código declara una Variable local (tipo "fuente de datos"). Esta variable será destruida al final de la Process.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Consejo
<a name="consejo_ELTTEXTE000107"></a>
Para utilizar una "Fuente de datos" Variable en todos los procesos de su aplicación, este Variable debe ser definido como Variable global.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000131"></a>


### Uso de una fuente de datos local en la ventana "WIN_Management
<a name="uso_una_fuente_datos_local_ventana_winmanagement_ELTPARAGRAPHE000025"></a>

**Código que desencadena el aviso**


```wl
MyCustomerFile is Data Source
```



**Posible corrección**
Definición de la fuente de datos global en la ventana.


```wl
// -- Global declarations of WIN_Management window
GLOBAL
	MyCustomerFile is Data Source
```

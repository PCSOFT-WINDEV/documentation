
## Error 1048: La herencia no está permitida en un Variable compuesto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Estás tratando de definir una herencia en el Definition de un Variable compuesto. La herencia no está permitida en un Variable compuesto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Borrar la declaración de la herencia. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Herencia en un Variable compuesto
<a name="herencia_variable_compuesto_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Coordinate is composed of
	x is real
	y is real
END
Coodinate1 is composed of
	a Coordinate object
	x1 is real
	y1 is real
END
```




**Posible corrección**

Borrar la herencia. 


```wl
Coordinate is composed of
	x is real
	y is real
END
Coodinate1 is composed of
	x1 is real
	y1 is real
END
```

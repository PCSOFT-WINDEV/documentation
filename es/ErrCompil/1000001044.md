
## Error 1044: La clase ya se deriva de esta clase
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Estás tratando de redefinir una herencia. La misma herencia se describe dos veces en el mismo código.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Borrar la sintaxis adicional. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Declarar una clase derivada
<a name="declarar_una_clase_derivada_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Circle is class
	is Drawing object
	is Drawing object
END
```




**Posible corrección**

Suprimir la declaración adicional. 


```wl
Circle is class
	is Drawing object
END
```

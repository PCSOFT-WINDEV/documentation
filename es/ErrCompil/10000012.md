
## Advertencia 12: La etiqueta no ha sido utilizada
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000085"></a>
Esta Line de código utiliza la sentencia GOTO con etiquetas de ramificación. Esta etiqueta está definida en la actual Process pero no es usada por ninguna declaración GOTO.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Consejo
<a name="consejo_ELTTEXTE000109"></a>
Compruebe si esta etiqueta es realmente inútil:

- Si la etiqueta es inútil, bórrela.

- Si la etiqueta es useful, corrija la declaración GOTO para usar la etiqueta adecuada.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000133"></a>


### Debido a un error tipográfico, la etiqueta no es utilizada por ningún GOTO
<a name="debido_error_tipografico_etiqueta_utilizada_por_ningun_goto_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error**


```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res, "Process OK")
IF Res = -1 THEN GOTO ERROPEN
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERRWRITE: 
Info("Unable to write into " + FileName)
```


**Posibles correcciones**

Modificar el código de la declaración GOTO y sustituir la etiqueta duplicada por la etiqueta correspondiente..
 

```wl
Res = fOpen(FileName, foWrite)
IF Res = -1 THEN GOTO ERROPEN
Res = fWrite(Res,"Process OK")
IF Res = -1 THEN GOTO ERRWRITE
...
RETURN
 
ERROPEN: 
Info("The " + FileName + " file cannot be opened. Check its existence.")
 
ERRWRITE: 
Info("Unable to write into " + FileName)
```

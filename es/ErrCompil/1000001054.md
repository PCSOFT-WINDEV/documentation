
## Error 1054: Ancestro" no puede utilizarse fuera del código de una clase.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está intentando acceder a un miembro o a un método de la clase base de una clase derivada utilizando la palabra clave'Ancestro'.. Esta palabra clave sólo se puede utilizar en el código de una clase derivada.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Para utilizar explícitamente un método o un miembro de la clase base de un código externo a la clase, especifique el nombre de la clase ancestral a utilizar.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Código que desencadena el error
<a name="codigo_que_desencadena_error_ELTPARAGRAPHE000025"></a>


```wl
// -- Button click code 
Ancestor::pLast
```




 
**Posible corrección**
Especifique el nombre de la clase. 


```wl
// -- Button click code 
Drawing::pLast
```

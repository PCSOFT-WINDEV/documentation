
## Error 530: Operador -- no permitido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000082"></a>
Estás tratando de usar el -- operador en un Variable que no soporta este operador.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000106"></a>
Modifique su código y utilice el operador o la función apropiada.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000130"></a>


### Usando el operador -- en una cadena de caracteres
<a name="usando_operador_una_cadena_caracteres_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
MyString is string = "ABCD"
MyString--
```


**Posible corrección**

Use la función de la izquierda para quitar la letra D del MyString Variable.


```wl
MyString is string = "ABCD"
MyString = Left(MyString, 3)
```

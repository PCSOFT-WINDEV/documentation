
## Error 303: El ambiguo nombre de control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Está tratando de manipular una columna de una control Tabla (basada en un archivo de datos o poblada programáticamente). En la ventana actual, ya hay una columna con el mismo nombre en varios controles de Tabla.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Especifique el nombre completo de la columna anteponiendo al nombre de la columna el nombre de la control Tabla.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>
La columna "COL_PriceIOT" está en los controles de la tabla "TABLE_Order" y "TABLE_OrderLine".

**Código que desencadena el error** 


```wl
// -- Click code of "BTN_Calculate" button
COL_PriceIOT = COL_PriceBT*COL_VAT
```


 
**Posible corrección**

Prefija el nombre de la columna con el nombre de la control Tabla.


```wl
// -- Click code of "BTN_Calculate" button
TABLE_OrderLine.COL_PriceIOT = COL_PriceBT*COL_VAT
```


## Error 103: El tipo es desconocido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000102"></a>
Este tipo es desconocido. Probablemente ha cometido un error tipográfico en nombre de un tipo simple, una clase o una estructura.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000126"></a>
Compruebe el tipo utilizado. Si está utilizando una clase, verifique si esta clase está incluida en el proyecto..


**Observación**: Los tipos reconocidos están coloreados en NEGRO (por Default) por la coloración sintáctica.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000150"></a>


### Usando un tipo simple desconocido
<a name="usando_tipo_simple_desconocido_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
Turnover is in
```




**Posible corrección**

Corregir el tipo.


```wl
Turnover is int
```

<a name="NOTE3_2"></a>


### Declaración de estructura
<a name="declaracion_estructura_ELTPARAGRAPHE000044"></a>

**Código que desencadena el error** 


```wl
STPoint is structure
	x, y are int
END
 
APoint is STPont
```




**Posible corrección**

Corregir el nombre de la estructura.


```wl
STPoint is structure
	x, y are int
END
 
APoint is STPoint
```

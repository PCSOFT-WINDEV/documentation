
## Error 31: No se encuentra el archivo a incluir. Compruebe el nombre y el ruta....
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000090"></a>
Esta Line de código utiliza la palabra clave EXTERN con un nombre de archivo. El archivo no se encuentra porque:

- El nombre del archivo es incorrecto (por ejemplo, error ortográfico en el nombre del archivo).

- El ruta utilizado para encontrar el archivo es incorrecto (el ruta especificado no existe, por ejemplo).




**Recordatorio**: La palabra clave EXTERN se utiliza para insertar un archivo en el código WLanguage. Por Default, si no se especifica ruta para el archivo, el archivo se busca en el directorio del proyecto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000114"></a>
Compruebe el ruta y el nombre del archivo a incluir en el código.

**Observación**: El subdirectorio "Personal\\WLanguage" de WINDEV, WEBDEV o WINDEV Mobile puede utilizarse para almacenar los archivos a incluir. En este caso, el ruta es automatic: el archivo se busca automáticamente en este directorio.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000138"></a>


### Usando un nombre de archivo que no existe en el directorio del proyecto
<a name="usando_nombre_archivo_que_existe_directorio_del_proyecto_ELTPARAGRAPHE000032"></a>

**Código que desencadena el error** 


```wl
EXTERN "Constants.txt"
```




**Posible corrección**

Crear el fichero o modificar el acceso ruta al fichero.


```wl
EXTERN "C:\Constants\Constants.txt"
```

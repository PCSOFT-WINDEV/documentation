
## Error 1016: El método de la clase es privado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000075"></a>
Está intentando acceder a un método privado de la clase desde un código que no sea un método de esta clase.

**Recordatorio**: Un método privado sólo puede utilizarse a partir de un método de la clase o de un método de una clase derivado de esta clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000099"></a>
Dos posibilidades:

- Reemplace el método privado por:

	- A **colectividad** método: el acceso al método no será restringido.

	- A **protegido** método: el acceso a este método se realizará desde una clase derivada.




- No manipular este método privado de uno de los códigos de proyecto.

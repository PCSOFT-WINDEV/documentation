
## Error 1032: La herencia no está permitida en una estructura
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está tratando de definir una herencia en la Definition de una estructura. La herencia no está permitida en una estructura.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Borrar la sintaxis de herencia. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Intento de herencia en una estructura
<a name="intento_herencia_una_estructura_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
Coord is structure 
	PosX is int
	PosY is int
END
 
ObjectSize is structure
   a coord object
	Height is int
	Width is int
END
```




**Posible corrección**

Borrar la sintaxis de herencia. 


```wl
Coord is structure 
	PosX is int
	PosY is int
END
 
ObjectSize is structure
	Height is int
	Width is int
END
```

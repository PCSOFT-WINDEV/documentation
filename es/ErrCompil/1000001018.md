
## Error 1018: El método es global: utilice el operador::.: para acceder a él
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000093"></a>
Está intentando acceder a un método global de la clase utilizando la sintaxis utilizada para acceder a un método no global.

Se está utilizando la siguiente sintaxis:


```txt
ClassName:MethodName
```


Esta sintaxis no se puede utilizar para acceder a un método global.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000117"></a>
Utilice la sintaxis apropiada para acceder a un método global de la clase:


```txt
ClassName::MethodName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000141"></a>


### Utilizando el método global Destruir de la clase Dibujo
<a name="utilizando_metodo_global_destruir_clase_dibujo_ELTPARAGRAPHE000035"></a>

**Código que desencadena el error** 


```wl
Drawing::Destroy()
```




**Posible corrección**

Corregir la sintaxis utilizada: 


```wl
Drawing::Destroy()
```

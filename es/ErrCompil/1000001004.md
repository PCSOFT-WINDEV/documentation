
## Error 1004: Para acceder explícitamente a un elemento de la clase base&apos;%1&apos;, utilice la sintaxis
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000088"></a>
Intentar utilizar un elemento (miembro, método o constante) de la clase base de la clase derivada.

La sintaxis utilizada es incorrecta. Se está utilizando la siguiente sintaxis:

```txt
:NameBaseClass:ElementName
```


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000112"></a>
Para utilizar un elemento (miembro, método o constante) de la clase base a partir de una clase derivada, el elemento debe ir precedido por el nombre de la clase base..

```txt
NameBaseClass:ElementName
```


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000136"></a>


### Usando el miembro BaseColor de la clase Drawing en la clase derivada Round
<a name="usando_miembro_basecolor_clase_drawing_clase_derivada_round_ELTPARAGRAPHE000031"></a>

**Código que desencadena el error** 


```wl
:Drawing:BaseColor = LightRed
```


**Posible corrección**

Eliminar los dos puntos (: carácter) delante del nombre de la clase base.


```wl
Drawing:BaseColor = LightRed
```

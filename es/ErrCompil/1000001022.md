
## Error 1013: El acceso al miembro de la clase está restringido por una herencia protegida.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000068"></a>
Está intentando acceder a un miembro de una clase previamente derivada por herencia protegida. No se puede acceder a este miembro desde un código externo a la clase derivada (o desde un código externo a sus propias clases derivadas).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000092"></a>
Dos posibilidades: 

- Acceder a este miembro desde un método de la clase derivada o desde un método de una de sus propias clases derivadas.. 

- Transformar la herencia protegida en herencia pública: el miembro será accesible desde el código de las clases derivadas así como desde un código externo a la clase.

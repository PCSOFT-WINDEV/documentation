
## Error 1015: El método de la clase está protegido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000070"></a>
Está intentando acceder a un método protegido de la clase desde un código distinto a un método de esta clase o clase derivada.

**Recordatorio**: Un método protegido sólo puede utilizarse a partir de un método de la clase o de un método de una clase derivado de esta clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000094"></a>
Dos posibilidades:

- Sustituya el método protegido por un **colectividad** método: el acceso al método no será restringido.

- No manipular este método protegido de uno de los códigos del proyecto.

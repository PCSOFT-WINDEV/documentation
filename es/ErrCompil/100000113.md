
## Error 113: Array no dinámico sin definición
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000093"></a>
Estás definiendo un array Variable sin especificar los elementos de este array.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000117"></a>
Rellene la sintaxis de la declaración array especificando las dimensiones de array y el tipo de elementos array..

**Observación**: Para utilizar una array dinámica, añada la palabra clave DINÁMICA durante la declaración de la array.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000141"></a>


### Declarando un array
<a name="declarando_array_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
Arr_TO is array
```




**Posibles correcciones**

Especifique las dimensiones de la array.


```wl
Arr_TO is array of 10 by 10 reals
```

Use la palabra clave DINÁMICA para indicar que la array es una array dinámica. Las dimensiones serán definidas en el código por [Dimensión](../WDLang1/3013022.md).


```wl
Arr_TO is dynamic array
```

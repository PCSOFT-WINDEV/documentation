
## Error 108: El tamaño de la Variable no está permitido
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000111"></a>
Estás usando un Variable entero o un Variable real especificando un tamaño específico. Este tamaño no está disponible para este tipo de Variable.

Los tamaños disponibles para el tipo entero son:

- 1 byte

- 2 bytes

- 4 bytes

- 8 octetos




Los tamaños disponibles para el tipo real son:

- 4 bytes reales

- 8 bytes reales




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000135"></a>
Utilice uno de los tamaños disponibles para el entero Variable o para el verdadero Variable.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000159"></a>


### Un entero Variable utiliza un tamaño que no se reconoce
<a name="entero_variable_utiliza_tamano_que_reconoce_ELTPARAGRAPHE000037"></a>

**Código que desencadena el error** 


```wl
MyInteger is 3-byte int
```




**Posible corrección**

Utilice un tamaño reconocido por WLanguage.


```wl
MyInteger is 2-byte int
```

<a name="NOTE3_2"></a>


### Un Variable real utiliza un tamaño que no se reconoce
<a name="variable_real_utiliza_tamano_que_reconoce_ELTPARAGRAPHE000053"></a>

**Código que desencadena el error** 


```wl
Price is 2-byte real
```




**Posible corrección**

Utilice un tamaño reconocido por WLanguage.


```wl
Price is 4-byte real
```


## Error 120: Redefinición de la estructura
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000099"></a>
Está intentando declarar dos estructuras con el mismo nombre en la misma Process.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000123"></a>
Cambie el nombre de una de las estructuras o elimine la declaración inútil.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000147"></a>


### Dos estructuras con el mismo nombre se declaran en la misma Process
<a name="dos_estructuras_con_mismo_nombre_declaran_misma_process_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error**


```wl
Point_Coord is structure
	x,y are int
END
 
...
 
Point_Coord is structure
	x,y are int
END
```




**Posible corrección**

Borrar la segunda estructura (declaración inútil).


```wl
Point_Coord is structure
	x,y are int
END
```

<a name="NOTE3_2"></a>


### En la misma Process se encuentran dos estructuras con el mismo nombre pero con miembros diferentes
<a name="misma_process_encuentran_dos_estructuras_con_mismo_nombre_pero_con_miembros_diferentes_ELTPARAGRAPHE000041"></a>

**Código que desencadena el error**


```wl
Point_Coord is structure
	x,y are int
END
 
...
 
Point_Coord is structure
	Coord_x,Coord_y are real
END
```




**Posible corrección**

Suprimir la segunda declaración (que es inútil).


```wl
Point_Coord is structure
	x,y are int
END
 
...
 
Image_Point_Coord is structure
	Coord_x,Coord_y are real
END
```

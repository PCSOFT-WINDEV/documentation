
## Error 39: Falta la declaración END que marca el final de la declaración del compuesto Variable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Usted está declarando un compuesto Variable. Falta la palabra clave END que marca el final de la declaración del compuesto Variable. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Indique el final de la declaración del compuesto Variable usando la palabra clave FIN. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Declaración incompleta de un compuesto Variable
<a name="declaracion_incompleta_compuesto_variable_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
ProductRef is composed of
	FamilyCode is fixed string on 8
	ProductCode is fixed string on 10
```




**Posible corrección**

Añade la palabra clave END para terminar la declaración del compuesto Variable. 


```wl
ProductRef is composed of
	FamilyCode is fixed string on 8
	ProductCode is fixed string on 10
END
```

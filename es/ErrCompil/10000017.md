
## Error 17: OTHER CASE debe ser la última declaración utilizada en SWITCH
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000084"></a>
Está utilizando una sentencia condicional como SWITCH ... MAY/MIN. La declaración "OTRO CASO" debe ser el último CASO utilizado.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000108"></a>
Mueva el bloque de instrucciones comenzando con "OTHER CASE" al final de la instrucción conditional SWITCH.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000132"></a>


### La declaración OTHER CASE no es la última declaración
<a name="declaracion_other_case_ultima_declaracion_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
SWITCH ProductType
	CASE "1"
	    Message("Food product")
	OTHER CASE
	    Message("Exceptional bargain")
	CASE "2"
	    Message("Cleaning product")
END
```


 
**Posible corrección**
Mover el bloque de sentencia que comienza con "OTHER CASE" al final de la sentencia conditional SWITCH.


```wl
SWITCH ProductType
	CASE "1"
	    Message("Food product")
	CASE "2"
	    Message("Cleaning product")
	OTHER CASE
	    Message("Exceptional bargain")
END
```

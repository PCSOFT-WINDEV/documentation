
## Error 3: La cuerda no tiene final "
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000118"></a>
Esta Line de código contiene comillas para delimitar una cadena de caracteres. Estas citas han sido "abiertas" pero no han sido cerradas antes del final del código Line.

**Recordatorio**: En el editor de códigos, ir a la siguiente Line (con la tecla ENTER) se considera como un separador de declaraciones.

Para especificar que el código actual Line continúa en el próximo Line, "..." debe ser usado al final del Line.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000142"></a>
Revisa tu código Line y termina la cadena con comillas.

**Observación**: La coloración sintáctica le ayuda a ver el final de la cadena de caracteres.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000166"></a>


### Mostrar un mensaje
<a name="mostrar_mensaje_ELTPARAGRAPHE000031"></a>Código que desencadena el error:


```txt
Info("You cannot perform this operation)
```


**Posible corrección**

Añada la cita final al final de la cadena de caracteres.


```wl
Info("You cannot perform this operation")
```

<a name="NOTE3_2"></a>


### Mostrando un mensaje que contiene una información almacenada en un Variable
<a name="mostrando_mensaje_que_contiene_una_informacion_almacenada_variable_ELTPARAGRAPHE000046"></a>Código que desencadena el error:


```txt
Info("Unable to save the file named + FileName)
```


**Posible corrección**

Añade una cita para separar el final de la cadena de caracteres de la Variable.


```wl
Info("Unable to save the file named" + FileName)
```

<a name="NOTE3_3"></a>


### Visualización de un mensaje escrito sobre varias líneas cole
<a name="visualizacion_mensaje_escrito_sobre_varias_lineas_cole_ELTPARAGRAPHE000061"></a>

Código que desencadena el error:


```txt
Info("You cannot perform this operation
	you have no sufficient rights")
```


**Posible corrección**

Añade la cita final al final de la Line así como "+ ..." para especificar que la cadena de caracteres continúa en el siguiente Line.


```wl
Info("You cannot perform this operation"+...
	"you have no sufficient rights")
```

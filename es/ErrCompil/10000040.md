
## Error 40: Falta la sentencia END que marca el final de la declaración de clase.
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está declarando una clase en el código de declaración de su clase. La declaración de clase no termina con la palabra clave END. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Usó la palabra clave END para terminar la declaración de su clase. 

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Declarar una clase sin la palabra clave END
<a name="declarar_una_clase_sin_palabra_clave_end_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
// -- Declaration of "File" class
File is class 
	Name is string 
	Extension is string
	Directory is string
```




**Posible corrección**

Usar la palabra clave END.


```wl
// -- Declaration of "File" class
File is class 
	Name is string 
	Extension is string
	Directory is string
END
```


## Error 403: La posición no pertenece al fichero
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000087"></a>
Usted está manejando un elemento de un archivo de datos usando una sintaxis como: 

&lt;NombreDeArchivo&gt;.&lt;NombreDeArchivo>.

El elemento especificado no existe en el archivo de datos especificado.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000111"></a>
Compruebe la ortografía del artículo.

Verificar si la posición existe. Cree la posición si es necesario.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000135"></a>


### El artículo PriceIOT no existe en el fichero de pedido
<a name="articulo_priceiot_existe_fichero_pedido_ELTPARAGRAPHE000031"></a>

**Código que desencadena el error**


```wl
Orders.PriceIOT = PriceBT*VAT
```




**Posible corrección**

Crea el artículo en el archivo de datos o usa otro artículo del archivo de datos.

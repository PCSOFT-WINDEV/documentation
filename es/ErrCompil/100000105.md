
## Error 105: Redefinición de la Variable local
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000099"></a>
Está tratando de declarar una variable local que ya fue declarada en la actual Process.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000123"></a>
Renombrar una de las variables o borrar la declaración inútil.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplos
<a name="ejemplos_ELTTEXTE000147"></a>


### Se utilizan dos variables locales con el mismo nombre pero con diferentes tipos
<a name="utilizan_dos_variables_locales_con_mismo_nombre_pero_con_diferentes_tipos_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
b is int
 
b is string
```




**Posible corrección**

Renombrar el segundo Variable.


```wl
b is int
 
c is string
```

<a name="NOTE3_2"></a>


### Dos variables locales tienen el mismo nombre y el mismo tipo
<a name="dos_variables_locales_tienen_mismo_nombre_mismo_tipo_ELTPARAGRAPHE000041"></a>

**Código que desencadena el error** 


```wl
I is int
FOR I = 1 TO 10
...
END
...
 
I is int
FOR I = 1 TO 10
...
END
```




**Posible corrección**

Suprimir la segunda declaración (que es inútil).


```wl
I is int
FOR I = 1 TO 10
...
END
...
 
FOR I = 1 TO 10
...
END
```


## Error 809: El *carácter no se puede utilizar como parámetro obligatorio en una función
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Está intentando utilizar el \*carácter para reemplazar un parámetro en una función WLanguage. Este parámetro es obligatorio y no puede ser sustituido por el \*carácter.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Sustituya el \*carácter por el valor del parámetro.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Uso del \*carácter en la función ListAdd
<a name="uso_del_starcaracter_funcion_listadd_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
ListAdd(*, "Vince")
```




**Posible corrección**

Utilice el nombre de la lista.


```wl
ListAdd(LIST_List1, "Vince")
```

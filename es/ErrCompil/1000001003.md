
## Error 1003: El elemento no existe en la clase
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000086"></a>
Está intentando acceder a un miembro, un método o una constante que no estaba definida en la clase.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000110"></a>
Dos soluciones: 

- Compruebe la ortografía del objeto utilizado.

- Cree el objeto en la clase.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000134"></a>


### Usar un método mal escrito
<a name="usar_metodo_mal_escrito_ELTPARAGRAPHE000028"></a>

**Código que desencadena el error** 


```wl
pDrawing is dynamic Drawing = new Drawing(gpDrawing,x,y,width,height)
pDrawing:Disp()
```




**Posible corrección**

El método Disp no existe: el nombre del método es Display. 


```wl
pDrawing is dynamic Drawing = new Drawing(gpDrawing,x,y,width,height)
pDrawing:Display()
```

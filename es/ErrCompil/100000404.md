
## Error 404: El archivo para la navegación HFSQL no puede ser una expresión de este tipo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Motivo
<a name="motivo_ELTTEXTE000083"></a>
Este código utiliza la sentencia FOR EACH. El nombre del archivo de datos que se espera en esta declaración no corresponde al nombre de un archivo de datos válido.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Corrección
<a name="correccion_ELTTEXTE000107"></a>
Revise el código de la declaración FOR EACH y use el nombre de un archivo de datos HyperFileSQL.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000131"></a>


### Usando un número entero en un bucle FOR EACH
<a name="usando_numero_entero_bucle_for_each_ELTPARAGRAPHE000025"></a>

**Código que desencadena el error** 


```wl
B is int
FOR EACH B
	ListAdd(LIST_List1, CustomerName)
END
```




**Posible corrección**

Usa el nombre de un archivo de datos en lugar de un Variable escalar.


```wl
FOR EACH CUSTOMER
	ListAdd(LIST_List1, CustomerName)
END
```

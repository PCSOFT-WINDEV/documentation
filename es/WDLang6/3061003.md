
## BurnerCancel( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la actual quema de Process. Esta quemadura Process fue iniciada por [BurnerSave](../WDLang6/3061006.md).

**Observación**: Al cancelar la grabación de Process, la compilación actual se borra automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Burn the CD 
BurnerSave()
...
// Cancel the current burn process
ResCancel is boolean
ResCancel = BurnerCancel()
// Request for cancelation taken into account?
IF ResCancel = True THEN
	// Status of burner
	IF BurnerStatus() = burnerInactive THEN
		Info("The burn process was canceled")
	ELSE
		Info("The cancelation request was taken into account but it was not performed yet")
	END
ELSE
	// Error details
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerCancel()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la solicitud de cancelación de la quema Process fue tomada en cuenta. Para saber si la anulación se ha realizado realmente, utilice [Estado del quemador](../WDLang6/3061007.md).

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

## Observaciones




### Recopilación
<a name="recopilacion_ELTPARAGRAPHE000052"></a>

Una compilación corresponde a la set de archivos que deben ser grabados en un CD o DVD. Esta recopilación se tendrá en cuenta la próxima vez que se llame a [BurnerSave](../WDLang6/3061006.md).

Una compilación se crea automáticamente durante la primera llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o a [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar ([BurnerMediaTipo](../WDLang6/3061013.md)).

- al seleccionar el quemador default ([QuemadorSeleccionar](../WDLang6/3061012.md)).

- después de llamar a la función [BurnerSave](../WDLang6/3061006.md) (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>).

- al cancelar la grabación (función **BurnerCancel**).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.




## ScanProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para obtener y cambiar las propiedades de una Scanner.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyScanner is Scanner
// Get the grayscale value 
// from which a pixel will be converted
// to black or white when scanned
WIA_IPS_THRESHOLD is int = 6159
Threshold is int = ScanProperty(MyScanner, WIA_IPS_THRESHOLD)
```



```wl
MyScanner is Scanner
let ScannerCapability = ScanProperty(MyScanner, scanPropertyCapability) 
IF ScannerCapability & scanCapabilityFeeder THEN
	Info("The scanner has a feeder")
END
IF ScannerCapability & scanCapabilityFlatbed THEN
	Info("The scanner has a flatbed")
END
IF ScannerCapability & scanCapabilityDuplex THEN
	Info("The scanner supports duplex scanning")
END
```

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener las propiedades de una Scanner

`<Result> = ScanProperty(<Scanner> , <Property>)`
---

**`<Result>: (Tipo de la Property)`**

Valor de Property. 

Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se convierte en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.

**`<Scanner>: (Escáner Variable)`**

Nombre de la variable [Scanner](../WDLang6/1410087808.md) inicializada con la información del escáner del que queremos obtener un Property.

**`<Property>: (Integro)`**

Property cuyos valores se solicitan. 
Esta propiedad puede corresponder a: 

- una de las siguientes constantes del archivo WiaDef.h en la API de Windows: WIA_DIP_xxx, WIA_DPA_xxx, WIA_DPS_xxx, WIA_IPA_xxx, o WIA_IPS_xxx.
	Atención: Las propiedades admitidas varían en función del controlador utilizado. La lista de propiedades se puede encontrar en [https://github.com/tpn/winddk-8.1/blob/master/Include/um/WiaDef.h](https://github.com/tpn/winddk-8.1/blob/master/Include/um/WiaDef.h).

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *scanPropertyCapability* | Capacidad de Scanner. En este caso, &lt;Resultado&gt; puede ser cualquier combinación de las siguientes constantes.: <br><br>	- *scanCapabilityFeeder*: La Scanner tiene un alimentador.<br><br>	- *scanCapabilityFlatbed*: La Scanner tiene una cama plana.<br><br>	- *scanCapabilityDuplex*: El Scanner admite el escaneo a doble cara.<br><br><br> |
| *scanPropertyPaperSize* | Tamaño de página actualmente set para escanear. ### Cambiar las propiedades de una Scanner

---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la propiedad,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

**`<Scanner>: (Escáner Variable)`**

Nombre de la variable [Scanner](../WDLang6/1410087808.md) inicializada con la información del escáner para el que queremos cambiar un Property.

**`<Property>: (Integro)`**


## SAP Variable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000124"></a>
El SAP Variable se utiliza para configurar los parámetros de la conexión con el servidor SAP. Estos parámetros serán utilizados por [SAPConnect](../WDLang6/1000017120.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## SAP Variable
<a name="sap_variable_ELTTEXTE000148"></a>
El SAP Variable contiene los siguientes elementos:


|   |   |
| --- | --- |
| Conexión SAP | Cadena de caracteres<br><br>description de la conexión SAP. Para más detalles, consulte la documentación específica sobre SAP.<br>Ejemplo:<br><br>Conexión = "TYPE=3 ASHOST=MINISAP.test.com SYSNR=00 <br>CLIENTE=000 USUARIO=BCUSER PASSWD=minisap"<br><br><br><br>Observación: si se especifica este parámetro, se ignoran las otras variables SAP. |
| Contraseña SAP | Contraseña del usuario que se conecta. |
| Idioma SAP | Cadena de caracteres<br><br>Define el idioma utilizado para la conexión (por ejemplo,"en" para inglés). |
| Número de cliente SAP | Cadena de caracteres<br><br>Número del cliente que se conecta. |
| Número de sistema SAP | Integro<br><br>Número del sistema SAP. |
| SAP.Address | Cadena de caracteres<br><br>Address del sistema SAP. |
| usuario de SAP | Cadena de caracteres<br><br>Nombre del usuario que se conecta. |


<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ejemplo
<a name="ejemplo_ELTTEXTE000172"></a>


### Conexión a un servidor SAP
<a name="conexion_servidor_sap_ELTPARAGRAPHE000075"></a>


```wl
SAP.Address = "MINISAP.test.com"
SAP.SystemNumber = "0"
SAP.ClientNumber = "000"
SAP.User = "BCUSER"
SAP.Password = "MINISAP"
SAP.Language = ""
IDSAP is int = SAPConnect()
```

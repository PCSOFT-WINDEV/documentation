## Presentación
<a name="presentacion_ELTTEXTE000148"></a>
PDF es uno de los formatos más utilizados para transmitir y almacenar documentos electrónicos.. 

Se pueden utilizar varios métodos para crear o administrar archivos PDF: 

- Impresión en archivos PDF: Se pueden crear archivos PDF al imprimir tanto en editor de reportes como en WLanguage.. 
	Para obtener más información, consulte [Impresión en PDF](../WDChamp/1011091.md). 

- Visualización de archivos PDF a través del lector de PDF control. 
	Para obtener más información, consulte [Control Lector PDF](../WDChamp/1410087036.md). 

- [Operaciones sencillas en archivos PDF en WLanguage](#NOTE2_1): Comprobar la protección, extraer una Page, fusionar PDF, etc.

- [Manejo del contenido de un archivo PDF en WLanguage](#NOTE3_1)utilizando diferentes tipos de variables.

## Manejo del contenido de un archivo PDF en WLanguage
<a name="manejo_del_contenido_archivo_pdf_wlanguage_ELTTEXTE000257"></a>
Para manipular más simplemente el contenido de un archivo PDF, WLanguage ofrece diferentes tipos de variables: 



|   |   |
| --- | --- |
| [pdfAttachment](../WDLang6/1000024899.md) | El tipo **pdfAttachment** permite definir las características avanzadas de un archivo adjunto en un documento PDF. |
| [pdfDocument](../WDLang6/1000024909.md) | El tipo **pdfDocument** obtiene todas las características de un archivo PDF: autor, fecha de creación, contenido de Page, etc. |
| [pdfPage](../WDLang6/1000024904.md) | El tipo **pdfPage** permite definir las características avanzadas de una página de un documento PDF cargado en memoria. |
| [pdfParameter](../WDLang6/1410087857.md) | El tipo **pdfParameter** define las características del archivo PDF que se va a generar. |
| [pdfSignature](../WDLang6/1000022255.md) | El tipo **pdfSignature** permite definir las características avanzadas de la firma que se aplicará a un archivo PDF al firmar un PDF existente (mediante la función [PDFSign](../WDLang6/1000022241.md)).  |
| [pdfTextElement](../WDLang6/1000024556.md) | El tipo **pdfTextElement** permite conocer las características de un elemento de texto en una página PDF. |





A [pdfDocument](../WDLang6/1000024909.md) Variable permite cargar el contenido de un archivo PDF en la memoria. Por ejemplo: 


```wl
MyPDF is pdfDocument = PDFOpen("test.pdf")

MySecondPDF is pdfDocument = "C:\temp\examplePDF.pdf"
```


A partir de esta variable, es posible acceder a las páginas del PDF Document y para cada página, el texto del PDF Document. Por ejemplo: 


```wl
MyPDF is pdfDocument = "test.pdf"
 
FOR EACH DocumentPage OF MyPDF.Page
	FOR EACH PageElement OF DocumentPage.TextElement
		Trace(PageElement.Text)
	END
END
```


Con este tipo de variables, es posible eliminar o añadir páginas a un Document mediante las funciones de gestión de array ([añadir](../WDLang1/1000017348.md), [suprimir](../WDLang1/1000017354.md), ... ) directamente en la array de [pdfPage](../WDLang6/1000024904.md) Variable s contenida en la [pdfDocument](../WDLang6/1000024909.md) Variable.

Ejemplo: 

```wl
MyPDFDoc is pdfDocument = PDFOpen("C:\temp\Myguide.pdf")
Trace(MyPDFDoc.PDFVersion)
Trace(MyPDFDoc.Author)
Trace(MyPDFDoc.Page.Count)
Trace(MyPDFDoc.Page[1].Width)
Trace(MyPDFDoc.Page[36].Height)

FOR EACH ATextElement OF MyPDFDoc.Page[2].TextElement
	Trace(ATextElement.Text + " X = " + 
	ATextElement.X + " Y = " + ATextElement.Y + " L = " + 
	ATextElement.Width + " H = " + ATextElement.Height)
END

MyPDFDoc2 is PDFDocument = "C:\temp\plan.pdf"
// Add pages
Add(MyPDFDoc2.Page, MyPDFDoc.Page[5])
// Insert pages
Insert(MyPDFDoc2.Page, 1, MyPDFDoc.Page[32])
// Save modified PDF
PDFSave(MyPDFDoc2, "MyPDFDoc2.pdf")
ShellExecute("MyPDFDoc2.pdf")
```

Las funciones específicas del PDF permiten gestionar las variables de [pdfDocument](../WDLang6/1000024909.md): 



|   |   |
| --- | --- |
| [PDFAddImage](../WDLang6/1410087847.md) | Añade una Image a una Page en un archivo PDF existente. |
| [PDFAddWatermark](../WDLang6/1410087843.md) | Añade una Watermark a todas o algunas páginas de un archivo PDF existente. |
| [PDFClose](../WDLang6/1000025127.md) | Cierra el archivo PDF cargado en memoria en una variable de tipo [pdfDocument](../WDLang6/1000024909.md). |
| [PDFGetSignature](../WDLang6/1410088120.md) | Obtiene la firma de un PDF Document como un buffer. |
| [PDFGetSignatureDate](../WDLang6/1410088121.md) | Obtiene la fecha de una de las firmas de un PDF Document. |
| [PDFNbSignatures](../WDLang6/1410088119.md) | Obtiene el número de firmas en un PDF Document. |
| [PDFOpen](../WDLang6/1000025032.md) | Abre un documento PDF y lo carga en una variable de tipo [pdfDocument](../WDLang6/1000024909.md). |
| [PDFSave](../WDLang6/1000024864.md) | Guarda el contenido de una variable de tipo [pdfDocument](../WDLang6/1000024909.md) en un archivo ".PDF". |





**Observación: Vinculación con el lector de PDF control**: 

- A [pdfDocument](../WDLang6/1000024909.md) Variable se le puede asignar un lector de PDF control. 

- Un documento PDF de un control PDF Reader puede ser recuperado en un [pdfDocument](../WDLang6/1000024909.md) Variable. 
	Ejemplo: 
	
	```wl
	MyPDFDoc is pdfDocument
	MyPDFDoc = PDF_PDFReader
	```

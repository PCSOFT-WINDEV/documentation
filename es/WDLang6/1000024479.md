
## gpwListOAuthParameters (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la lista de parámetros OAuth definidos para el Groupware Usuario, es decir, la lista de proveedores (Google, Facebook, etc.) a los que es posible conectarse a través del Groupware Usuario.. Estos parámetros se definieron a través del sitio de administración del Groupware Usuario. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrOAuthParam is array of gpwOAuthParameters
arrOAuthParam = gpwListOAuthParameters()
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END

Parameters is gpwOAuthParameters = arrOAuthParam[1]
sCode is string = gpwAuthLogin(Parameters)
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END

User is gpwUser
sToken is string
(sToken, User) = gpwRemoteConnect(Parameters, sCode)
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwListOAuthParameters()`
---

**`<Result>: (Array of gpwOAuthParameters variables)`**

Array de variables de tipo [gpwOAuthParameters](../WDLang6/1000024751.md) que corresponden a la descripción de los parámetros OAuth.


## gpwEncryptPassword (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la contraseña cifrada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF gpwEncryptedPasswords() = True THEN
	GPU_UserPWD.EncryptedPassword = gpwEncryptPassword(sPassword)
ELSE
	GPU_User.Password = sPassword
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwEncryptPassword(<Password>)`
---

**`<Result>: (Cadena de caracteres)`**

Contraseña cifrada. 

**`<Password>: (Cadena de caracteres)`**

Contraseña en formato legible.




## Funciones de gestión de íconos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para administrar los azulejos que se encuentran en las aplicaciones de las aplicaciones de Windows Store: 



|   |   |
| --- | --- |
| [TileAdd](../WDLang6/1000020719.md) | Añade una nueva personalización de Tile para la aplicación actual de Windows Store Apps. |
| [TileDeleteAll](../WDLang6/1000020720.md) | Elimina todas las modificaciones hechas a la aplicación Tile. |
| [TileDisplayBadge](../WDLang6/1000020721.md) | Muestra una insignia (Notification o Image numérica) en la Tile de una aplicación universal de Windows 10. |

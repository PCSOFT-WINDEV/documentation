


## &lt;Variable Scanner&gt;.ToPDF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un documento escaneado en un archivo PDF.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyScanner is Scanner

// Scan all the pages in the feeder to a PDF file
MyScanner.ToPDF("File.PDF")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Scanner>.ToPDF([<Number of pages>] [, <PDF parameters>] , <PDF file>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el documento,

- <u><u><u><u>False</u></u></u></u> si no se guardó el documento. La función [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.




**`<Scanner>: (Variable de tipo Scanner)`**

Nombre de la variable [Scanner](../WDLang6/1410087808.md) que corresponde al escáner a utilizar.

**`<Number of pages>: (Entero opcional)`**



- Número de páginas a escanear,

- Constante *scanNumberPagesAll* para escanear todas las páginas del alimentador (valor predeterminado).




**`<PDF parameters>: (Variable de tipo pdfParameter)`**

Nombre de la variable [pdfParameter](../WDLang6/1410087857.md) con las características del archivo PDF a crear.

**`<PDF file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo (hasta 260 caracteres). La extensión del archivo es opcional. Se puede utilizar una ruta de tipo UNC.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sin un alimentador, para escanear varias páginas y agregarlas a un solo archivo PDF, debe escanear y agregar cada página a un archivo PDF y luego utilizar la función [PDFMerge](../WDLang6/1000019712.md).

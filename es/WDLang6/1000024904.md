
## pdfPage (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfPage** permite definir las características avanzadas de una página de un documento PDF cargado en memoria. Las características de esta página se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "test.pdf"

FOR EACH DocumentPage OF MyPDF.Page
	Trace(DocumentPage.Height)
END

PDFSave(MyPDF, "passwordtest.pdf")

// It is also possible to use prefix syntax
// MyPDF.Save("passwordtest.pdf")
```



```wl
MyPDFDoc is pdfDocument = PDFOpen("C:\temp\Myguide.pdf")
Trace(MyPDFDoc.PDFVersion)
Trace(MyPDFDoc.Author)
Trace(MyPDFDoc.Page.Count)
Trace(MyPDFDoc.Page[1].Width)
Trace(MyPDFDoc.Page[36].Height)

FOR EACH ATextElement OF MyPDFDoc.Page[2].TextElement
	Trace(ATextElement.Text + " X = " + 
	ATextElement.X + " Y = " + ATextElement.Y + " L = " + 
	ATextElement.Width + " H = " + ATextElement.Height)
END

MyPDFDoc2 is PDFDocument = "C:\temp\plan.pdf"
// Add pages
Add(MyPDFDoc2.Page, MyPDFDoc.Page[5])
// Insert pages
Insert(MyPDFDoc2.Page, 1, MyPDFDoc.Page[32])
// Save modified PDF
PDFSave(MyPDFDoc2, "MyPDFDoc2.pdf")
ShellExecute("MyPDFDoc2.pdf")
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo pdfPage
<a name="propiedades_especificas_descripcion_variables_tipo_pdfpage_ELTPARAGRAPHE000055"></a>

Las siguientes propiedades se pueden utilizar para manipular una página de un documento PDF:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Height | Real | Alto de la página en milímetros. <br><br>Si se modifica esta altura, la página se redimensiona automáticamente y los elementos se reposicionan. <br><br> |
| Orientation | Constante de tipo String | Permite conocer y modificar la orientación de la página: <br><br>- *Orient0*: Orientación predeterminada.<br><br>- *Orient90*: Página orientada a 90°.<br><br>- *Orient180*: Página orientada a 180°.<br><br>- *Orient270*: Página orientada a 270°.<br><br><br> |
| TextElement | Array de [pdfTextElement](../WDLang6/1000024556.md) | Información sobre el texto de la página. |
| Width | Real | Ancho de la página en milímetros. <br><br>Si se modifica este ancho, la página se redimensiona automáticamente y los elementos se reposicionan. <br><br> |


## gpwRemoteConnect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conectar un usuario remoto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrOAuthParam is array of gpwOAuthParameters
arrOAuthParam = gpwListOAuthParameters()
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END

Parameters is gpwOAuthParameters = arrOAuthParam[1]
sCode is string = gpwAuthLogin(Parameters)
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END

User is gpwUser
sToken is string
(sToken, User) = gpwRemoteConnect(Parameters, sCode)
IF ErrorOccurred THEN
	Error(ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Conectar un usuario remoto identificado con un nombre de usuario y contraseña

`(<Token>, <User>) = gpwRemoteConnect(<Login> , <Password> [, <Code> [, <Trust key>]])`
---

**`<Token>: (Cadena de caracteres)`**

Token de acceso. Este token debe utilizarse para conectarse a la aplicación (3ª sintaxis de la función **gpwRemoteConnect**). 

**`<User>: (Variable de tipo gpwUser)`**

Variable de tipo [gpwUser](../WDLang6/1000021088.md) que contiene la información del usuario conectado. 

**`<Login>: (Cadena de caracteres)`**

Inicie sesión o inicie sesión en Email. 

**`<Password>: (Cadena de caracteres)`**

Contraseña de usuario. 

**`<Code>: (Cadena de caracteres)`**

Código de 6 dígitos devuelto por la aplicación de autenticación de dos factores (Google Authenticator o Miccrosoft Authenticator, por ejemplo). 

**`<Trust key>: (Buffer)`**

Clave de confianza utilizada si el usuario marcó la opción "Dispositivo de confianza" para la autenticación de dos factores.  


<a name="SYNTAX2"></a>

### Conectar un usuario remoto a través de OAuth

`(<Token>, <User>) = gpwRemoteConnect(<Parameters> , <Code>)`
---

**`<Token>: (Cadena de caracteres)`**

Token de acceso. Este token debe utilizarse para conectarse a la aplicación (3ª sintaxis de la función **gpwRemoteConnect**). 

**`<User>: (Variable de tipo gpwUser)`**

Variable de tipo [gpwUser](../WDLang6/1000021088.md) que contiene la información del usuario conectado. 

**`<Parameters>: (Variable de tipo gpwOAuthParameters)`**

Nombre de la variable de tipo [gpwOAuthParameters](../WDLang6/1000024751.md) que corresponde a los parámetros de la conexión OAuth de la aplicación. 

**`<Code>: (Buffer)`**

Código de identificación del usuario devuelto por la función [gpwAuthLogin](../WDLang6/1000023597.md). 


<a name="SYNTAX3"></a>

### Volver a conectar un usuario mediante el token de acceso

`<User> = gpwRemoteConnect([<Token>])`
---

**`<User>: (Variable de tipo gpwUser)`**

Variable de tipo [gpwUser](../WDLang6/1000021088.md) que contiene la información del usuario conectado. 

**`<Token>: (Cadena de caracteres opcional)`**

Token de acceso proporcionado por una de las sintaxis anteriores de la función **gpwRemoteConnect**.

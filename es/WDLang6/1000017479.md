
## PDFToText( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae el texto encontrado en un archivo PDF.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString is string
MyString = PDFToText("C:\Temp\MyDocument.pdf")
// Display in a multiline edit control
EDT_EditMultiText = MyString
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PDFToText(<PDF file> [, <Pages to extract> [, <Password> [, <Options>]]])`
---

**`<Result>: (Cadena de caracteres)`**

Texto del archivo PDF.

**`<PDF file>: (Cadena de caracteres)`**

Nombre y ruta del archivo PDF a analizar.

**`<Pages to extract>: (Cadena de caracteres opcional)`**

Rango de páginas en las que se debe extraer el texto. El formato utilizado es idéntico al utilizado en las cajas de impresión estándar.: números Page individuales del rango de Page s separados por punto y coma. Por ejemplo, "1;3;4;6-10;12" significa que se procesarán las páginas 1, 3, 4, 6 a 10 y 12.

Si no se especifica este parámetro o si corresponde a una cadena vacía (""), se extraen todas las páginas.

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña necesaria para abrir el archivo si el archivo PDF tiene contraseña protected.  

**`<Options>: (Constante de tipo Integer)`**

Modo de división de texto: 


|   |   |
| --- | --- |
| *pttCompatible* | Divida el texto PDF utilizando el algoritmo de las versiones 24 y anteriores.. |
| *pttDefault*<br>(Valor predeterminado) | Divida el texto PDF utilizando un algoritmo optimizado. Esta división puede ser diferente de las versiones anteriores. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Conversión de PDF a texto
<a name="conversion_pdf_texto_ELTPARAGRAPHE000068"></a>

- El formato de la Document se pierde cuando se realiza la conversión de PDF a texto.

- El texto se extrae en el orden o apariencia de los comandos PDF y se escribe secuencialmente en la cadena de resultados.. Se mantiene la organización del texto en párrafos y bloques (así como los caracteres CR)..  

- Los caracteres Unicode no se devuelven.

- Los datos encontrados en un formulario PDF no se extraen (estos datos no se almacenan en el archivo PDF)..  



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000078"></a>

- [PDFIsProtected](../WDLang6/1000017491.md) se utiliza para averiguar si se necesita una contraseña para abrir un archivo PDF.

- [PDFNumberOfPages](../WDLang6/1000017480.md) devuelve el número total de páginas encontradas en un archivo PDF.

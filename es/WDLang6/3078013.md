


## TwainSelectSource( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un cuadro de diálogo estándar utilizado para listar los diferentes dispositivos Twain disponibles en el ordenador actual.. El usuario tendrá la posibilidad de elegir el dispositivo Twain utilizado por Default. Este dispositivo será utilizado por todas las siguientes funciones de Twain.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TwainSelectSource.gif)
**Notas**: 

- [TwainFuente actual](../WDLang6/3078015.md) permite conocer el dispositivo Twain utilizado por Default mediante las funciones Twain.

- En 64 bits, sólo se enumeran los dispositivos que incluyen un controlador de 64 bits.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a standard dialog box used to list the different
// Twain devices available on the current computer
IF TwainSelectSource() = False THEN
	Error("The user did not choose any Twain device")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TwainSelectSource()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario seleccionó un dispositivo Twain,

- <u><u><u><u>False</u></u></u></u> en caso contrario. El mensaje de error correspondiente es devuelto por [ErrorInfo](../WDLang1/3013008.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Cambiando el dispositivo Default Twain:
<a name="cambiando_dispositivo_default_twain_ELTPARAGRAPHE000047"></a>

Para cambiar el dispositivo Twain usado por Default, puedes:

- utilice la función [TwainFuente actual](../WDLang6/3078015.md).

- mostrar la lista de los dispositivos Twain instalados con la función **TwainSelectSource**. Entonces, el usuario podrá elegir el dispositivo Twain que será utilizado por Default.





### Modo de funcionamiento en las aplicaciones de 64 bits
<a name="modo_funcionamiento_las_aplicaciones_64_bits_ELTPARAGRAPHE000181"></a>

En una aplicación de 64 bits, esta función utilizará automáticamente el controlador Twain de 64 bits.. No se puede utilizar el TWAIN si este controlador no está instalado..

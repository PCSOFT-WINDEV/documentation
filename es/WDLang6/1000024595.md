
## gpwAddUserToGroup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un usuario a un grupo de la base de datos del Groupware Usuario.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gpwAddUserToGroup("Premium")
IF ErrorOccurred = True THEN
       // Full message of the main error
        Trace(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwAddUserToGroup(<Group> [, <Move>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario se agregó al grupo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.  




**`<Group>: (Cadena de caracteres)`**

Nombre del grupo al que debe asociarse el usuario actual. 

**`<Move>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si el usuario debe moverse al grupo especificado. En este caso, el usuario se asociará solo a este grupo. Se eliminará de todos los demás grupos. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si el usuario debe asociarse al grupo especificado y seguir estando asociado a los otros grupos.




## Funciones de gestión de ejecutables en modo Consola
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar diferentes flujos de ejecutables en modo "consola": 



|   |   |
| --- | --- |
| [ConsoleError](../WDLang6/1000024023.md) | Escribe una cadena de caracteres en el flujo de salida de errores estándar "stderr". |
| [ConsoleRead](../WDLang6/1000024024.md) | Lee una cadena en el flujo de entrada estándar "stdin". |
| [ConsoleWrite](../WDLang6/1000024022.md) | Escribe una cadena de caracteres en el flujo de salida estándar "stdout" (también llamado "consola"). |

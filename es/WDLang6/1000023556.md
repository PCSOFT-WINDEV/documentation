
## gpwDefault (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor que corresponde a "Default" según el framework utilizado. 

Observación: Dado que el groupware y sus archivos son multilingües, es posible recuperar el valor de la constante que representa la palabra "Default".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCstDEfauLt is string
sCstDefault = gpwDefault()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwDefault()`
---

**`<Result>: (Character string)`**

El valor de la palabra "Default" según el framework que se utiliza: 

- "Default" si el framework está en inglés,

- "DEFAUT" si el framework está en francés.


## gpwUser (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gpwUser** permite describir y modificar las características de un usuario del Groupware Usuario WEBDEV o WINDEV. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de las variables gpwUser
<a name="propiedades_especificas_las_variables_gpwuser_ELTPARAGRAPHE000055"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a **gpwUser** Variable LE:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Contraseña | Cadena de caracteres | Contraseña de usuario.<br><br>Esta contraseña no se recupera cuando se utiliza [gpwGetUserInfo](../WDLang6/3041004.md) para get la información del usuario. |
| ContraseñaPara entrar | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si los usuarios deben introducir su contraseña la primera vez que se conectan,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| Derechos de acceso | Constante de tipo Integer | Derechos de acceso de los usuarios: <br><br>- *gpwRightsAddApplication*: Derecho a añadir una aplicación en Tienda privada. <br><br>- *gpwRightsNone*: No hay derechos de adición. <br><br><br> |
| Email | Cadena de caracteres | Dirección de correo electrónico del usuario. |
| FirstName | Cadena de caracteres | Nombre del usuario. |
| Habilitado | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el usuario está activado,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| IDExternal | Cadena de caracteres | Identificador del usuario en una base de datos externa al groupware. Esta Property puede utilizarse, por ejemplo, para vincular el usuario del programa de grupo del archivo de usuario de la aplicación que utiliza el programa de grupo haciendo referencia a su identificador. <br><br> |
| LastName | Cadena de caracteres | Apellido del usuario. |
| Nombre de usuario | Cadena de caracteres | Nombre de usuario. |
| Número de teléfono | Cadena de caracteres | Número de Phone del usuario. |
| Supervisor | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el usuario es supervisor,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| TOTPKey | Cadena de caracteres | Clave TOTP relacionado al usuario para la autenticación de dos factores. |
| TrustList | Buffer | Si la autenticación de dos factores está activada, el usuario puede indicar que el ordenador actual es un ordenador de confianza. Esta Property enumera todos los dispositivos, ordenadores y navegadores de confianza definidos por el usuario. <br><br>Observación: Los usuarios pueden eliminar los ordenadores de confianza en el perfil de usuario de la Agrupación de Usuarios. |
| TwoFactorAuthentication | Integro | Modo de gestión de la autenticación de dos factores: <br><br>- 0: Autenticación de dos factores desactivada. La autenticación de dos factores está desactivada para el usuario y no es necesaria.. <br><br>- 1: Autenticación de dos factores activada. La autenticación de dos factores está activada para el usuario. <br><br>- 2: La autenticación de dos factores debe estar activada. La autenticación de dos factores es necesaria, pero está desactivada.. <br><br><br> |

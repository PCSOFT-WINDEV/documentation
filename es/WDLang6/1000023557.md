
## gpwNone( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor de la palabra "NINGUNO" según el framework que se está utilizando.. La palabra "NINGUNO" se utiliza cuando un usuario no pertenece a un grupo. 

Observación: Dado que el groupware y sus archivos son multilingües, es posible recuperar el valor de la constante que representa la palabra "NINGUNO".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCstNone is string
sCstNone = gpwNone()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwNone()`
---

**`<Result>: (Cadena de caracteres)`**

Valor de la palabra "NINGUNO" según el framework utilizado:   

- "&lt;NONE&gt;" si el framework está en inglés

- "&lt;AUCUN&gt;" si el framework está en francés.

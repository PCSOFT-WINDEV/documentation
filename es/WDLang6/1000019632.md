
## apLog( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una información en un archivo "log" durante la ejecución de un plan de acción.. El nombre del archivo de registro tiene el siguiente formato: "&lt;Nombre del plan de acción&gt;.log" y se guarda en el directorio del plan de acción. El contenido de este fichero corresponde al estado reporte de la ejecución del plan de acción.. 

**Atención**: Esta función sólo se puede ejecutar: 

- del código de un plan de acción. 

- de un WLanguage Procedure que se encuentra en un proyecto o en una configuración de "Plan de Acción".. 




**Observación**: Esta función es una función utilizada para manipular, el editor de WINDEV, WEBDEV o WINDEV Mobile.. Por lo tanto, está disponible en los tres productos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable the "StaticAudit" action
IF paramAudit = False THEN
	apDisableAction(StaticAudit)
	apLog("** Static audit disabled ***")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`apLog(<Information>)`
---

**`<Information>: (Cadena de caracteres)`**

Información que debe guardarse en el archivo "log" del plan de acción.

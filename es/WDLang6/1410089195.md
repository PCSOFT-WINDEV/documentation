
## pdfBookmark (Tipo de variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfBookmark** permite definir todas las características avanzadas de un marcador en un documento PDF. Las características del marcador se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "test.pdf"
FOR EACH Bookmark OF MyPDF
	Trace(Bookmark.Title)
END
```

<a name="PROPS"></a>

## Properties


### Propiedades específicas de variables pdfBookmark
<a name="propiedades_especificas_variables_pdfbookmark_ELTPARAGRAPHE000025"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable **pdfBookmark**:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| PageNumber | Integer | Número de la página que contiene el marcador.<br><br>**Esta propiedad es de solo lectura**. |
| ChildBookmark | Array de **pdfBookmark** | Array de marcadores hijos.<br><br>**Esta propiedad es de solo lectura**. |
| Title | Character string | Título del marcador. <br><br>**Esta propiedad es de solo lectura**. |

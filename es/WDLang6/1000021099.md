
## gpwConnectUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta a un usuario con el programa de grupo de usuarios.

## Sintaxis
<a name="SYNTAX1"></a>

### Conectar a un usuario con el programa de grupo de usuarios

`<Result> = gpwConnectUser(<Login> , <Password>)`
---

**`<Result>: (Constante de tipo Integer)`**

Resultado de la conexión: 


|   |   |
| --- | --- |
| *gpwError* | Se produjo un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). |
| *gpwInvalidPassword* | La contraseña del usuario es desconocida. |
| *gpwOK* | El usuario estaba conectado. |
| *gpwUnknownUser* | El usuario es desconocido: no se encuentra en la base de datos de groupware usuario. |



**`<Login>: (Cadena de caracteres)`**

Nombre de usuario. 

**`<Password>: (Cadena de caracteres)`**

Contraseña associated con el nombre de usuario. 


<a name="SYNTAX2"></a>

### Conectar a un usuario identificado con el Groupware de usuarios a través de un correo electrónico Address

`<Result> = gpwConnectUser(<User's email address>)`
---

**`<Result>: (Constante de tipo Integer)`**

Resultado de la conexión: 


|   |   |
| --- | --- |
| *gpwError* | Se produjo un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). |
| *gpwInvalidPassword* | La contraseña del usuario es desconocida. |
| *gpwOK* | El usuario estaba conectado. |
| *gpwUnknownUser* | El usuario es desconocido: no se encuentra en la base de datos de groupware usuario. |



**`<User's email address>: (Cadena de caracteres)`**

Correo electrónico del usuario recuperado en una autenticación OpenID (**Email** Property del [OpenIDIdentity](../WDLang3/1000023567.md) Variable).




## Funciones de bloqueo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para bloquear una aplicación:



|   |   |
| --- | --- |
| [LockAppOff](../WDLang6/1000017012.md) | Desbloquea una aplicación que fue bloqueada automáticamente o por [LockAppPassword](../WDLang6/1000017011.md). |
| [LockAppOn](../WDLang6/1000017011.md) | Bloquea una aplicación WINDEV. |
| [LockAppPassword](../WDLang6/1000017010.md) | Especifica una contraseña específica para desbloquear la aplicación actual (aplicación bloqueada automáticamente o por [LockAppOn](../WDLang6/1000017011.md)). |
| [LockAppTimeout](../WDLang6/1000017006.md) | Devuelve o modifica el tiempo de inactividad antes de que se bloquee una aplicación WINDEV. |
| [ValidatePassword](../WDLang6/1000017009.md) | Indica si la contraseña especificada es válida para el usuario actual. |

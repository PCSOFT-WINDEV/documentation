
## PDFPassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Protege y encripta un PDF Document con una contraseña. Se crea un nuevo Document PDF encriptado con la contraseña especificada. Esta contraseña se solicitará al abrir el archivo PDF.  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF NOT PDFPassword("clear.pdf", "encrypt.pdf", iProtectionDefault, "password") THEN 
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFPassword(<Source PDF file> , <Result PDF file> , <Options> , <Password> [, <Protection password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se aplica la protección,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error. 




**`<Source PDF file>: (Cadena de caracteres)`**

Nombre y ruta completo del archivo PDF para proteger. 
Observación: Este archivo PDF no debe ser ya contraseña-protected. 

**`<Result PDF file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo PDF protected.

**`<Options>: (Constante o combinación de constantes de tipo Integer)`**

Opciones de protección disponibles: 



|   |   |
| --- | --- |
| *iProtectionComment* | Impide agregar o editar comentarios. |
| *iProtectionDefault* | Protección de Default: el archivo pdf está encriptado con la contraseña sin ninguna protección adicional. |
| *iProtectionModification* | El contenido del archivo PDF no se puede modificar. |
| *iProtectionPrinting* | El archivo PDF no se puede imprimir. |
| *iProtectionSelection* | El contenido del archivo PDF no se puede copiar o extraer del archivo. |



**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para abrir el archivo PDF.  Esta contraseña permite descifrar el archivo PDF para leerlo respetando los permisos set en el parámetro .

Esta contraseña no puede contener caracteres Unicode. 

**`<Protection password>: (Cadena de caracteres opcional)`**

Contraseña utilizada para permitir la modificación de las opciones de protección para el archivo PDF. Si se utiliza esta contraseña, no se tendrán en cuenta las opciones de protección especificadas en los parámetros . 

Esta contraseña no puede contener caracteres Unicode.

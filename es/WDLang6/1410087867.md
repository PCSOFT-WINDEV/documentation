
## Funciones de escáner (sintaxis con prefijo)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de escáner son las siguientes (sintaxis con prefijo):



|   |   |
| --- | --- |
| [&lt;Scanner Variable&gt;.Capacidad](../WDLang6/1410088721.md) | Se utiliza para determinar si una Scanner tiene una capacidad determinada. |
| [&lt;Scanner Variable&gt;.ToImage](../WDLang6/1410087864.md) | Recupera una Document de un escáner en una Variable de tipo [Image](../WDLang1/1000019650.md). |
| [&lt;Variable Scanner&gt;.Property](../WDLang6/1410087863.md) | Permite obtener y cambiar las características de un escáner. |
| [&lt;Variable Scanner&gt;.ToPDF](../WDLang6/1410087866.md) | Guarda un documento escaneado en un archivo PDF. |
| [&lt;Variable Scanner&gt;.WithUserInterface](../WDLang6/1410087860.md) | Guarda un documento de un escáner en uno o más archivos, utilizando la interfaz de usuario del escáner. |

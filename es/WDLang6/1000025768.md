
## ocrOption (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **ocrOption** permite definir las opciones generales del OCR (Reconocimiento óptico de caracteres) nativo. Las características de estas opciones se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creation of options
MyOCROptions is ocrOption

// Detect only numbers when reading an image
MyOCROptions.OnlyDigit = True

OCRParameter(MyOCROptions)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo ocrOption
<a name="propiedades_especificas_descripcion_variables_tipo_ocroption_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades permiten manipular las opciones avanzadas del OCR nativo:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Accuracy | Constante de tipo Integer |   |
| ListWord | Array of strings |   |
| OnlyDigit | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> si el OCR debe devolver solo números arábigos,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br> |
| WithSpellingChecker | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para activar el corrector ortográfico,<br><br>- <u><u><u><u>False</u></u></u></u> para desactivarlo. <br><br><br> |

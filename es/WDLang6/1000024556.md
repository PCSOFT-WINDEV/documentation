
## pdfTextElement (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfTextElement** permite conocer las características de un elemento de texto en una página PDF. Las características este elemento se pueden obtener mediante diferentes propiedades WLanguage. Por ejemplo, es posible conocer la posición del texto y su contenido, para realizar búsquedas.

Atención: Todas las propiedades de la variable **pdfTextElement** son de solo lectura porque el texto de un archivo PDF no puede modificarse. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "test.pdf" 

FOR EACH DocumentPage OF MyPDF.Page
	FOR EACH PageElement OF DocumentPage.TextElement
		Trace(PageElement.Text)
	END
END
```





<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo pdfTextElement
<a name="propiedades_especificas_las_variables_tipo_pdftextelement_ELTPARAGRAPHE000057"></a>

Las siguientes propiedades se pueden utilizar para manipular un elemento de texto del archivo PDF:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| FontName | Character string | Nombre de la fuente utilizada para el elemento de texto.  <br><br>**Esta propiedad es de solo lectura**.<br><br> |
| FontSize | Real | Tamaño de la fuente utilizada para el elemento de texto (en punto tipográfico, 72PPP).  <br><br>**Esta propiedad es de solo lectura**.<br><br> |
| Height | Real | Altura del rectángulo que indica la posición y el tamaño del elemento en la página (en milímetros). <br><br>**Esta propiedad es de solo lectura**. |
| Text | Character string | Texto del elemento.<br><br>**Esta propiedad es de solo lectura**. |
| Width | Real | Ancho del rectángulo que indica la posición y el tamaño del elemento en la página (en milímetros). <br><br>**Esta propiedad es de solo lectura**. |
| X | Real | Posición X del rectángulo que indica la posición y el tamaño del elemento en la página (en milímetros). <br><br>**Esta propiedad es de solo lectura**. |
| Y | Real | Posición Y del rectángulo que indica la posición y el tamaño del elemento en la página (en milímetros).  <br><br>**Esta propiedad es de solo lectura**. |


## Grabación de un CD/DVD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000132"></a>
Varias funciones de WLanguage se utilizan para grabar los CDs o DVDs. Es posible crear: 

- CDs/DVDs de datos.

- CD de audio.




**Observación**: Sólo puede grabar archivos en un CD o DVD. No se puede copy un CD o DVD directamente.


### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000019"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La grabación de CD está disponible para Windows XP y versiones posteriores. La grabación de DVDs está disponible para Windows Vista y para Windows XP cuando se utiliza la actualización KB932716 para Windows XP (esta actualización debe descargarse desde el sitio de Microsoft e instalarse manualmente porque no se tiene en cuenta en el mecanismo de Live Update)..

## ¿Cómo grabar un CD/DVD?
<a name="¿como_grabar_cddvd_ELTTEXTE000162"></a>
Para grabar un CD/DVD:

1. Definir (si es necesario) el quemador a utilizar: 
	


|   |   |
| --- | --- |
| [BurnerList](../WDLang6/3061009.md) | Returns the list of burners installed on the current computer. |
| [BurnerPath](../WDLang6/3061004.md) | Devuelve el ruta del quemador actual. |
| [BurnerProperty](../WDLang6/3061011.md) | Permite conocer y modificar las propiedades del quemador actual. |
| [BurnerSelect](../WDLang6/3061012.md) | Permite seleccionar el quemador Default. |
| [BurnerStatus](../WDLang6/3061007.md) | Devuelve el estado del quemador actual. |

2. Abra (si es necesario) la puerta del quemador actual ([BurnerEject](../WDLang6/3061014.md)).
	**Observación**: Para saber si el quemador contiene un soporte, utilice [BurnerMediaFondo](../WDLang6/1000019910.md). 

3. Especifique (si es necesario) el tipo de CD/DVD que desea crear:
	


|   |   |
| --- | --- |
| [BurnerMediaInfo](../WDLang6/3061008.md) | Recupera las características del CD/DVD que se encuentra en la grabadora actual. |
| [BurnerMediaType](../WDLang6/3061013.md) | Permite conocer y modificar el formato del CD o DVD a grabar. |

4. Borrar (si es necesario) los archivos ya encontrados en el CD/DVD a través de [BurnerErase](../WDLang6/3061005.md). Un procedimiento de WLanguage ("callback") se llama regularmente durante este borrado Process. Esta Procedure se utiliza para gestionar los diferentes eventos que ocurren.

5. Especifique (si es necesario) el nombre que se le dará al CD/DVD a grabar mediante [BurnerVolumeName](../WDLang6/3061019.md).

6. Seleccione los archivos que desea grabar en el CD/DVD:
	


|   |   |
| --- | --- |
| [BurnerAddDirectory](../WDLang6/3061002.md) | Añade todos los archivos encontrados en un directorio a la compilación. |
| [BurnerAddFile](../WDLang6/3061001.md) | Añade un archivo a la compilación. |

7. Recuperar (si es necesario) las características de la compilación ([Información de BurnerCompilationInfo](../WDLang6/3061018.md).

8. Grabar los archivos seleccionados en el CD/DVD con [BurnerSave](../WDLang6/3061006.md). Un procedimiento de WLanguage ("callback") se llama regularmente durante esta quemadura Process. Esta Procedure se utiliza para gestionar los diferentes eventos que ocurren.

9. Cancel (si es necesario) la operación de quemado en curso ([QuemadorCancelar](../WDLang6/3061003.md).



<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000080"></a>

- Las funciones de grabación se pueden utilizar para grabar CDs de varias sesiones.

- Los DVDs de varias sesiones no se pueden grabar. Sin embargo, el DVD puede copiarse en el disco duro y luego volver a grabarse con el nuevo contenido.

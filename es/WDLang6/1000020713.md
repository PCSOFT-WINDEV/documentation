
## Azulejo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Tile** permite definir las características avanzadas de un ícono para una aplicación en modo Universal Windows 10 App. Las características de esta Tile pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyTile is Tile
MyTile.Type = tileWide
MyTile.Model = tileTemplateWideImageTop
Add(MyTile..Images, "photo.jpg")
MyTile.Text = "Important"
TileDeleteAll()
TileAdd(MyTile)
```

<a name="NOTE0_2"></a>


### Visualización de texto y/o imágenes
<a name="visualizacion_texto_imagenes_ELTPARAGRAPHE000146"></a>

- Si la propiedad **..Image** está vacía, solo se muestra el texto (y eventualmente el título). 

- Por el contrario, si el título y el texto están vacíos, sólo se muestra la Image.



<a name="NOTE0_3"></a>


### Baldosas cuadradas y rectangulares
<a name="baldosas_cuadradas_rectangulares_ELTPARAGRAPHE000154"></a>

Las baldosas pueden ser cuadradas y anchas. El usuario elige el modo de visualización que más le convenga. Para que la Tile se muestre correctamente en los dos modos, estos dos modos deben ser manejados. 

En este caso, se debe usar Tile Add para agregar los dos modos de visualización del Tile. 

Por ejemplo: 


```wl
// Tile in rectangular mode
MyTile is Tile
MyTile.Type = tileWide
MyTile.Model = tileTemplateWideImageTop
Add(MyTile..Images, "photo.jpg")
MyTile.Text = "My little text"
TileAdd(MyTile)

// Tile in square mode
MyTile.Type = tileSquare
MyTuile.Model = tileTemplateScrolling
TileAdd(MyTile)
```

<a name="NOTE0_4"></a>


### Notas
<a name="notas_ELTPARAGRAPHE000167"></a>

- Se pueden desplazar hasta 5 azulejos diferentes en una cola. 

- Los Tile s pueden ser actualizados incluso si la aplicación está cerrada especificando un URL en el XML description de Tile. 

- Tiene la posibilidad de añadir insignias en la parte inferior de la Tile con [TileDisplayBadge](../WDLang6/1000020721.md).

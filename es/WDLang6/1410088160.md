


## &lt;pdfDocument Variable&gt;.Guardar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el contenido de una variable de tipo [pdfDocument](../WDLang6/1000024909.md) en un archivo ".PDF". 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "Test.pdf"

FOR EACH DocumentPage OF MyPDF.Page
	Trace(DocumentPage.Height)
END

MyPDF.Save("passwordtest.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <PDF to save>.Save(<Name and path of the PDF file> [, <Quality>] [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<PDF to save>: (Variable de tipo pdfDocument)`**

Nombre de la variable [pdfDocument](../WDLang6/1000024909.md) que corresponde al documento PDF a guardar.

**`<Name and path of the PDF file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo PDF a crear.

**`<Quality>: (Constante opcional de tipo Integer)`**

Configura el nivel de compresión de colores y escala de grises. 


|   |   |
| --- | --- |
| *iAutomaticQuality* | Nivel de compresión calculado automáticamente para generar un archivo PDF de mejor calidad (especialmente para las imágenes).<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |
| *iAverageQuality* | Nivel medio de compresión.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |
| *iHighQuality* | El bajo nivel de compresión utilizado para get una alta calidad para el PDF generado.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |
| *iLowQuality* | Alto nivel de compresión.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |
| *iMaximumQuality* | Nivel de compresión mínimo a get una calidad máxima. Esta constante se recomienda al crear un archivo PDF destinado a la impresión.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |
| *iMinimumQuality* | Nivel de compresión máximo. La calidad será mínima. Esta constante se puede utilizar para crear un archivo PDF destinado a ser visto en la pantalla.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |



**`<Password>: (Cadena de caracteres opcional)`**

Nueva contraseña para abrir el archivo PDF. 

- Si este parámetro corresponde a una cadena vacía (""), el documento PDF no tendrá ninguna contraseña. 

- Si no se especifica este parámetro, el documento PDF conservará su contraseña, si la tiene.

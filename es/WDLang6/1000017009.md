
## ValidatePassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la contraseña especificada es válida para el usuario actual.. La contraseña puede ser:

- la contraseña de autenticación de Windows

- la contraseña del grupo de usuarios de WINDEV/WEBDEV.

- la contraseña utilizada para bloquear las aplicaciones (contraseña especificada con [Contraseña LockAppPassword](../WDLang6/1000017010.md)).




![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta función sólo está disponible para comprobar si la contraseña es válida para el User Groupware. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ValidatePassword("MyPassword") = True THEN
	Info("User validated")
ELSE
	info("Invalid password")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ValidatePassword(<Password> [, <Authentication>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la contraseña es correcta,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Password>: (Cadena de caracteres)`**

Contraseña para validar para el usuario actual.

**`<Authentication>: (Constante opcional de tipo Integer)`**

Método de autenticación de contraseña:



|   |   |
| --- | --- |
| *pwdCustom* | La autenticación se realiza según la contraseña especificada en [Contraseña LockAppPassword](../WDLang6/1000017010.md).<br><br> |
| *pwdGpw* | Autenticación realizada por el grupo de usuarios de WINDEV/WEBDEV. |
| *pwdWindows* | Autenticación realizada según la contraseña de Windows.<br> |


Si no se especifica &lt;Autenticación&gt;:

- Si se ha utilizado [Contraseña LockAppPassword](../WDLang6/1000017010.md), se comprueba esta contraseña.

- Si el groupware usuario está activada, la contraseña se comprueba en relación con el groupware usuario.

- De lo contrario, la contraseña se coteja con la cuenta de Windows..

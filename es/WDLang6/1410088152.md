
## Funciones de detección de objetos en imágenes (AI)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten detectar objetos en las imágenes: 



|   |   |
| --- | --- |
| [AIDetect](../WDLang6/1410087942.md) | Detecta diferentes elementos en una imagen: objeto, rostros, etc, ... Esta detección se realiza mediante la inteligencia artificial (IA). |
| [AIDetectContour](../WDLang6/1410089109.md) | Detecta los contornos de las formas en una imagen. |
| [AIDetectDrawing](../WDLang6/1410089108.md) | Clasifica los dibujos a mano alzada, incluidos círculos, cuadrados y triángulos. |
| [AIDetectModel](../WDLang6/1410087982.md) | Ejecuta un modelo de red neuronal entrenado en una imagen. Esta función utiliza la inteligencia artificial para detectar objetos en una imagen. |



Estas funciones utilizan los siguientes tipos de variables: 



|   |   |
| --- | --- |
| [aiImageResult](../WDLang6/1410087965.md) | El tipo **aiImageResult** obtiene el resultado de la función [AIDetect](../WDLang6/1410087942.md). |
| [aiNeuralNetworkModel](../WDLang6/1410088095.md) | El tipo **aiNeuralNetworkModel** define las características avanzadas de una red neuronal utilizada por la función [AIDetectModel](../WDLang6/1410087982.md). |

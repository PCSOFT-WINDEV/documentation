
## Funciones de gestión del reconocimiento óptico de caracteres (OCR) integrado
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la tecnología OCR integrada en WINDEV, WEBDEV y WINDEV Mobile:



|   |   |
| --- | --- |
| [OCRAddLanguage](../WDLang6/1000025728.md) | Agrega un idioma al OCR nativo. |
| [OCRDeleteAllLanguages](../WDLang6/1000025729.md) | Elimina todos los idiomas del OCR nativo. |
| [OCRDetectTextArea](../WDLang6/1000025704.md) | Devuelve todas las áreas de texto de una imagen sin leer el texto. |
| [OCRExtractText](../WDLang6/1000025699.md) | Lee el texto de una imagen. |
| [OCRExtractTextBlock](../WDLang6/1000025703.md) | Devuelve todos los bloques de texto de una imagen. |
| [OCRLoadLanguage](../WDLang6/1000025790.md) | Agrega un idioma al OCR nativo. |
| [OCRModelPathDirectory](../WDLang6/1000025700.md) | Permite establecer el directorio que contiene los modelos de idioma para el sistema de OCR nativo. |
| [OCRParameter](../WDLang6/1000025701.md) | Inicializa las opciones generales del OCR nativo (Reconocimiento óptico de caracteres). |


## ScanGet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica las características de un escáner y las almacena en una variable de tipo [Scanner](../WDLang6/1410087808.md). Esta variable se puede utilizar para realizar operaciones de escaneo.

Observación: Para obtener la lista de escáneres conectados al equipo actual, utilice la función [ScanList](../WDLang6/1410087807.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the characteristics of a scanner
MyScanner is Scanner 
MyScanner = ScanGet("hp scanjet 5590")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScanGet(<Scanner name>)`
---

**`<Result>: (Variable de tipo Scanner)`**

Variable de tipo [Scanner](../WDLang6/1410087808.md) inicializada con las características de &lt;Scanner name&gt;. 

Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.

**`<Scanner name>: (Cadena de caracteres)`**

Nombre del escáner cuyas características se van a obtener. Este nombre debe ser uno de los nombres devueltos por la función [ScanList](../WDLang6/1410087807.md).


## apDisableAction( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva la ejecución de una acción que se encuentra en un plan de acción.. 

Esta función sólo se puede utilizar en los procesos de un plan de acción.. Por ejemplo, algunas acciones pueden ser desactivadas en el código de inicialización del plan de acción de acuerdo con los parámetros. 

**Observación**: Esta función es una función utilizada para manipular, el editor de WINDEV, WEBDEV o WINDEV Mobile.. Por lo tanto, está disponible en los tres productos. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable the "StaticAudit" action
IF paramAudit = False THEN
	apDisableAction(StaticAudit)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`apDisableAction(<Action Name>)`
---

**`<Action Name>: (Cadena de caracteres)`**

Nombre de la acción a desactivar. La acción desactivada no se ejecutará y, por lo tanto, no provocará ningún error.

**Recordatorio**: El nombre de la acción se define en la ventana de acción de description.

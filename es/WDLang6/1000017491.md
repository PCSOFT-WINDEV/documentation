
## PDFIsProtected( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si el archivo PDF requiere una contraseña para ser leído.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks whether the PDF file is protected
IF PDFIsProtected("C:\Temp\MyDocument.pdf") = True THEN
	Info("Password-protected PDF file")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PDFIsProtected(<PDF file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se requiere una contraseña para abrir el archivo PDF,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<PDF file>: (Cadena de caracteres)`**

Nombre y ruta del archivo PDF a analizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo se tiene en cuenta la contraseña necesaria para abrir el archivo.. Si el archivo PDF solo requiere una contraseña para cambiar la configuración de protección, no se considera como protected ( **PDFIsProtected** devolverá <u><u><u><u>False</u></u></u></u> ).

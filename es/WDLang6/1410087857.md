
## pdfParameter (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfParameter** define las características del archivo PDF que se va a generar. Las características de este archivo PDF se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display two full pages with scrolling and cover page
Params is pdfParameter
Params.Quality = iHighQuality
Params.Zoom = zoomFullWidth
Params.PageDisplayMode = dispTwoPage + dispEnableScrolling + dispWithCoverPage
iParameterPDF(Params)
iDestination(iGenericPDF)
iPrintReport(MyReport)

ShellExecute(iLastFile())
```



```wl
// Display a page with scrolling and set zoom to content
param is pdfParameter
param.Zoom = zoomFitVisible
param.PageDisplayMode = dispSinglePage + dispEnableScrolling
iParameterPDF(param)
iDestination(iGenericPDF, gsPDFName)
iPrintReport(RPT_Database)

ShellExecute(gsPDFName)
```



```wl
// Set page zoom to 100% and enable scrolling
param is pdfParameter
param.Zoom = 100
param.PageDisplayMode = dispEnableScrolling
iParameterPDF(param)
iDestination(iGenericPDF, gsPDFName)
iPrintReport(RPT_Database)

ShellExecute(gsPDFName)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo pdfParameter
<a name="propiedades_especificas_las_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000035"></a>

Las siguientes propiedades pueden utilizarse para configurar un archivo PDF:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Bookmark | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> si se van a incluir marcadores en el archivo PDF. Por cada página del archivo PDF se define un marcador automáticamente. Cada marcador tiene un nombre con el siguiente formato "Página &lt;número de página&gt;" (por ejemplo, "Página 1").<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| ColorFormat | Constante de tipo Integer | Gestión del color en el archivo PDF: <br><br>- *pdfColor*: Crear un archivo PDF a color. <br><br>- *pdfGrayscale*: Crear un archivo PDF en blanco y negro. <br><br><br> |
| ModificationPassword | Character string | Contraseña utilizada para permitir a un usuario cambiar las propiedades del archivo PDF. |
| OpeningPassword | Character string | Contraseña necesaria para abrir el archivo PDF. |
| PageDisplayMode | Constante de tipo Integer | Modo de visualización de la página en el lector de PDF. Corresponde a una combinación de las siguientes constantes: <br><br>- *dispWithCoverPage*: Establece la primera página del archivo PDF como portada. La primera página se muestra a la derecha cuando la visualización de la página está en modo de 2 páginas. <br><br>- *dispEnableScrolling*: Visualización con desplazamiento progresivo: Muestra las páginas en una columna vertical ininterrumpida que ocupa el ancho de una página. El desplazamiento se puede activar tanto en el modo de visualización de 1 página como en el de 2 páginas. <br><br>- *dispDefault* (Valor predeterminado): Visualización predeterminada. <br><br>- *dispTwoPage*: Muestra dos páginas una al lado de la otra en el lector de PDF. <br><br>- *dispSinglePage*: Muestra una página del archivo PDF a la vez en el lector de PDF. No se muestran los márgenes. <br><br><br> |
| Protection | Constantes de tipo Integer | Opciones de protección del archivo PDF generado. Corresponde a la combinación de las siguientes constantes: <br><br>- Opciones de formato: <br><br>	- *iPDFA* o *iPDFA1b*: PDF en formato PDF/A-1b. Todos los elementos necesarios para mostrar o imprimir el archivo PDF se incluyen en el archivo generado. Por lo tanto, el archivo generado es más grande. <br>			Observación: Este formato tiene algunas limitaciones: <br><br>		- Transparencia de imágenes no soportada. <br><br>		- Fuentes OpenType no soportadas. <br><br>		- Uso de fuentes incrustadas.<br><br><br><br><br>- *iPDFA3b*: PDF en formato PDF/A-3b. Todos los elementos necesarios para mostrar o imprimir el archivo PDF se incluyen en el archivo generado. Por lo tanto, el archivo generado es más grande. Puede agregar diferentes tipos de archivos al archivo PDF, como XML, CSV, procesadores de texto, hojas de cálculo, etc. <br>			Observación: Este formato tiene algunas limitaciones: <br><br>	- Transparencia de imágenes no soportada. <br><br>	- Fuentes OpenType no soportadas. <br><br>	- Uso de fuentes incrustadas.<br><br>- Opciones de protección: <br><br>	- *iProtectionNone*: No hay protección para el archivo PDF.<br><br>	- *iProtectionComment*: Impide agregar o editar comentarios.<br><br>	- *iProtectionPrinting*: El archivo PDF no se puede imprimir.<br><br>	- *iProtectionModification*: El contenido del archivo PDF no se puede modificar.<br><br>	- *iProtectionSelection*: El contenido del archivo PDF no se puede copiar o extraer del archivo.<br><br><br><br><br><br> |
| Quality | Constante de tipo Integer | Calidad de compresión de colores y escala de grises: <br><br>- *iAutomaticQuality*: Calidad calculada automáticamente. <br><br>- *iHighQuality*: Compresión para alta calidad.<br><br>- *iLowQuality*: Compresión para baja calidad.<br><br>- *iMaximumQuality*: Compresión para calidad máxima. <br><br>- *iMinimumQuality*: Compresión para calidad mínima.<br><br>- *iAverageQuality*: Compresión para calidad media.<br><br><br> |
| Unicode | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> si el archivo PDF contiene caracteres Unicode, <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br><br><br> |
| Version | Constante de tipo Integer | Versión del PDF generado: <br><br>- *iPDF14*: Archivo generado usando PDF 1.4 (predeterminado). <br><br>- *iPDF15*: Archivo generado usando PDF 1.5<br><br>- *iPDF16*: Archivo generado usando PDF 1.6<br><br>- *iPDF17*: Archivo generado usando PDF 1.7<br><br><br> |
| Zoom | Entero o constante de tipo Integer | Zoom predeterminado al abrir el archivo PDF: <br><br>- Valor del zoom entre 25% y 6400%,<br><br>- *zoomFitVisible*: Zoom para mostrar la página de manera que el texto y las imágenes ocupen todo el ancho del visor. <br><br>- *zoomDefault*: Zoom predeterminado. <br><br>- *zoomFullPage*: Zoom para mostrar toda la página. <br><br>- *zoomFullHeight*: Zoom para mostrar todo el espacio vertical de la página. <br><br>- *zoomFullWidth*: Zoom para mostrar todo el espacio horizontal de la página. <br><br>- *zoomActualSize*: Zoom al 100%. <br><br><br> |

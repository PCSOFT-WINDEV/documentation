
## SAPConnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta un sistema SAP R/3 a partir de la información proporcionada en el [Estructura SAP](../WDLang6/1000017184.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SAP.Address = "MINISAP.test.com"
SAP.SystemNumber = "0"
SAP.ClientNumber = "000"
SAP.User = "BCUSER"
SAP.Password = "MINISAP"
SAP.Language = ""
IDSAP is int = SAPConnect()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SAPConnect()`
---

**`<Result>: (Integro)`**



- Identificador de la conexión,

- -1 si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **SAPConnect** devuelve -1 solo si el servidor no está disponible. Si se ha producido un error (inicio de sesión erróneo, contraseña errónea o cuenta de cliente no encontrada), SAP acepta la conexión y devuelve un error durante una llamada a un Procedure.

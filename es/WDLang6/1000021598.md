
## ActionPlanConfigure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura un plan de acción en la fábrica de software. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates the instance of action plan for compiling my project 
ArrParameter is associative array of Variants
ArrParameter["project"] = "MyProject"
ActionPlanConfigure("FacCoord", "admin", "fackey", "MyLibrary" + TAB + ...
	"PlanCompilProject", "PlanCompilMyProject", ArrParameter)
```



```wl
// Creates the action plan instance with a parameter to open a project from the SCM
// The "Open a project from the SCM" action takes the name of the project 
// in the "paramOpenProject" parameter
ArrParameter is associative array of Variants
ArrParameter["paramOpenProject"] = "CS:\\TEST:4900\SCM"+CR+"SCM:\WINDEV Projects\My_Project"
ActionPlanConfigure("FacCoord", "admin", "MyLibrary", "My_Instance_PLAN_TEST", ...
	ArrParameter, 30s, apStatusCompletedFailure + apStatusCompletedOK, "MG")
```





<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ActionPlanConfigure(<Coordinator> , <User> , <Password> , <Name of action plan> , <Instance of action plan> [, <Parameters> [, <Timeout> [, <Required information> [, <Recipients> [, <Labels>]]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se configuró el plan de acción,

- <u><u><u><u>False</u></u></u></u> si no se configuró el plan de acción. El mensaje de error correspondiente es devuelto por [ErrorInfo](../WDLang1/3013008.md).




**`<Coordinator>: (Cadena de caracteres)`**

Nombre de el equipo utilizado para coordinar la ejecución de los planes de acción.

**`<User>: (Cadena de caracteres)`**

Nombre de usuario. Este usuario debe poder ejecutar el plan de acción en el coordinador.

**`<Password>: (Cadena de caracteres)`**

Contraseña de usuario.

**`<Name of action plan>: (Cadena de caracteres)`**

Nombre del plan de acción a configurar. 
Para configurar un plan de acción de una biblioteca determinada, el nombre del plan de acción debe tener el siguiente formato: 

```txt
<Library name> + TAB + <Name of action plan>
```


Observaciones: 

- El nombre de la biblioteca puede contener (o no) la extensión. 

- Si no se especifica el nombre de la biblioteca, se configurará el primer plan de acción encontrado que corresponda al nombre especificado.




**`<Instance of action plan>: (Cadena de caracteres)`**

Nombre de la instancia del plan de acción creado o modificado con los parámetros especificados.

**`<Parameters>: (Asociación array)`**

Nombre de la asociación Array Variable que contiene los parámetros del plan de acción. Esta array asociativa está indexada en el nombre de los parámetros.

**`<Timeout>: (Entero o duración opcional)`**

Duración máxima para ejecutar la instancia del plan de acción en segundos. Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en claro ('1s' por ejemplo). 




**`<Required information>: (Constante opcional de tipo Integer)`**

Indica en qué caso un mensaje debe ser enviado por la aplicación de mensajería interna (WDBal).. Corresponde a una de las siguientes constantes:


|   |   |
| --- | --- |
| *apStatusCompletedFailure* | La ejecución del plano finalizó con error. |
| *apStatusCompletedOK* | La ejecución del plano finalizó sin errores. |



**`<Recipients>: (Cadena de caracteres opcional)`**

Lista de personas a informar a través de la mensajería de WDBal. Los nombres de los destinatarios están separados por caracteres CR (Carriage Return). Este parámetro tiene el siguiente formato: 

```txt
"Name recipient 1" + CR + ... + CR + "Name recipient N"
```


**`<Labels>: (Array de cuerdas)`**

Lista de etiquetas que necesita un programa para poder ejecutar el plan de acción set. 
Recordatorio: Puede asignar una o varias etiquetas a un programa. Este parámetro puede utilizarse para ejecutar un plan de acción sólo en algunos programas.

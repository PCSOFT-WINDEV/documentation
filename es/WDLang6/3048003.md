
## JavaExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia una aplicación Java o un applet Java invocando el método "Main" de una clase Java dada.

**Atención**:

- La ruta de la clase a ejecutar debe ser especificada por [JavaLoad](../WDLang6/3048001.md).

- La ejecución de la clase Java debe implementar el siguiente método: "Static public void main(String[] args)".



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003048003&name=javaexecute_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Full paths where the classes (*.class) will be sought
// Note: Specify the names of folders/directories and the names of .jar or .zip
// (and not just the name of .class)
JavaLoad("D:\sources\AppJava1\classes.Zip;D:\sources\Example\JavaTest")
JavaExecute("Application", "MyString")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`JavaExecute(<Class name> [, <Parameter>])`
---

**`<Class name>: (Cadena de caracteres)`**

Nombre y directorio de la clase a ejecutar. Este parámetro distingue entre mayúsculas y minúsculas: debe corresponder exactamente al nombre especificado en la clase.

Si la clase está contenida en uno o varios bultos, deberá indicarse el ruta completo de esta clase con el carácter "/" como separador.. Ejemplo: "MyPackage/MySubPackage/MyClass".

**`<Parameter>: (Cadena de caracteres opcional)`**

Parámetro que debe ser enviado a la aplicación Java.

## Observaciones




### Llamar a JavaLoad
<a name="llamar_javaload_ELTPARAGRAPHE000050"></a>

- Una única llamada a [JavaLoad](../WDLang6/3048001.md) debe realizarse en una aplicación WINDEV o en un sitio WEBDEV.
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) se puede llamar a [JavaLoad](../WDLang6/3048001.md) tantas veces como sea necesario.

- Se requiere una de las siguientes versiones de la máquina virtual SUN Java: versión 1.1, 1.2 (o sub-versión), 1.3 (o sub-versión), 1.4 o 6.

### Error
<a name="error_ELTPARAGRAPHE000070"></a>

Si se produce un error al ejecutar la clase especificada, el mensaje de error correspondiente se muestra en un cuadro de mensaje y se detiene la aplicación WINDEV (o el sitio WEBDEV).


## ConsoleRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee una cadena en el flujo de entrada estándar "stdin".

Observación: Cuando se llama la función, la consola espera a que el usuario introduzca el texto.. Cuando el usuario introduce el carácter ' '\\n(usando la tecla Enter), la cadena introducida es devuelta por la función.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConsoleWrite("Please enter your first name: ")

let sFirstName = ConsoleRead()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ConsoleRead()`
---

**`<Result>: (Cadena de caracteres)`**

Caracteres introducidos en el flujo de entrada estándar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Atención: En Windows, el ejecutable generado debe estar en modo consola para permitir al usuario introducir texto.

## Componente:
wd280std.dll

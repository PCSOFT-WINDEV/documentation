
## gpwAddUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un usuario a la base de datos de User Groupware.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Values of new user
clUser is gpwUser
clUser.Email = EDT_Email
clUser.Login = EDT_Login
clUser.Password = EDT_Password
clUser.LastName = EDT_LastName
clUser.FirstName = EDT_FirstName
clUser.PhoneNumber = EDT_Phone

// Add the user
nRes is int = gpwAddUser(clUser)
SWITCH nRes
	CASE gpwOk
		Info("ok")

	CASE gpwActivationByEmail
		Info("Activation by email")

	OTHER CASE
		Info("Unable to add the user." + CR + ErrorInfo())
		RETURN
END

// Activate the user
SWITCH gpwActivateUser(clUser..Login)
	// Activation by email
	CASE gpwActivationByEmail
		PopupDisplay(POPUP_ValidateSubscription)
	// Immediate activation
	CASE gpwActivationImmediate
		// Connects the user directly
		gpwConnectUser(clUser.Login, clUser.Password)
		// Displays the confirmation popup
		PopupDisplay(POPUP_ConfirmSubscription)
	OTHER CASE
		Info("Unable to activate the user." + CR + HErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwAddUser(<User>)`
---

**`<Result>: (Constante booleana o entera)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Corresponde a una booleana: 

- <u><u><u><u>True</u></u></u></u> si se agregó el usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<User>: (Variable de tipo gpwUser)`**

Nombre de la [gpwUser](../WDLang6/1000021088.md) Variable que contiene la información del nuevo usuario. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Dependiendo del modo de validación de los nuevos usuarios configurados en la página de gestión, la validación de los usuarios puede realizarse de forma inmediata o puede ser realizada por Email. [gpwActivateUser](../WDLang6/1000021096.md) activa el nuevo usuario.

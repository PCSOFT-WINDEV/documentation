
## LockAppPassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica una contraseña específica para desbloquear la aplicación actual (aplicación bloqueada automáticamente o por [LockAppOn](../WDLang6/1000017011.md)). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Password sought in the User file
LockAppPassword(User.Password)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`LockAppPassword(<Password>)`
---

**`<Password>: (Cadena de caracteres)`**

Contraseña que se utilizará para desbloquear la aplicación.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Aplicación bloqueada
<a name="aplicacion_bloqueada_ELTPARAGRAPHE000035"></a>

Cuando la aplicación está bloqueada, todas las ventanas de aplicación se minimizan. Cuando el usuario intenta restaurar la aplicación, aparece la siguiente ventana:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Verrouappli.gif)


El usuario debe escribir una contraseña. Esta contraseña puede corresponder a:

- la contraseña de Windows del ordenador actual (por Default),

- la contraseña de usuario definida en el groupware usuario (si el groupware usuario es usada por su aplicación),

- una contraseña específica, definida con la función **LockAppPassword**.

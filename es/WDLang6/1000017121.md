
## SAPDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desconecta un sistema SAP R/3.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SAP.Address = "MINISAP.test.com"
SAP.SystemNumber = "0"
SAP.ClientNumber = "000"
SAP.User = "BCUSER"
SAP.Password = "MINISAP"
SAP.Language = ""
IDSAP is int = SAPConnect()
...
SAPDisconnect(IDSAP)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SAPDisconnect(<Connection identifier>)`
---

**`<Connection identifier>: (Integro)`**

Identificador de la conexión, devuelto por la función [SAPConnect](../WDLang6/1000017120.md).

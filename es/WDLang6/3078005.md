


## TwainListSource( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de controladores para los dispositivos Twain conectados al ordenador actual.

**Observación**: [TwainFuente actual](../WDLang6/3078015.md) permite conocer el dispositivo Twain utilizado por Default mediante las funciones Twain.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the list of drivers for the Twain devices 
// found on the current computer in LIST_DeviceDrivers
ListAdd(LIST_DeviceDrivers, TwainListSource())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TwainListSource()`
---

**`<Result>: (Cadena de caracteres)`**

Devuelve la lista de controladores para los dispositivos Twain instalados en el ordenador actual, en el siguiente formato:

```txt
<Name of driver 1> + CR + <Name of driver N>
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Visualización del resultado en un List Box o control Combo Box
<a name="visualizacion_del_resultado_list_box_control_combo_box_ELTPARAGRAPHE000037"></a>

&lt;Resultado&gt; puede asignarse directamente a un control List Box o Combo Box mediante la función [ListAdd](../WDLang1/3049004.md). Cada Line contendrá el nombre del controlador correspondiente a cada dispositivo Twain instalado en el ordenador actual.
<a name="NOTE0_2"></a>


### Requisitos previos en el servidor de 64 bits de Windows 2008
<a name="requisitos_previos_servidor_64_bits_windows_2008_ELTPARAGRAPHE000047"></a>

Para una aplicación de 32 bits que debe realizar una operación de digitalización en un servidor de 64 bits Windows 2008, se requiere la función "Experiencia de usuario" de Windows.. Para añadir esta función:

1. Seleccione las características del administrador de servidores. 

2. Haz clic en "Añadir una característica".. 

3. Marque "Experiencia de usuario" (o "Experiencia de escritorio" para Windows en inglés). 

4. El servidor se reinicia al final de instalación.





### Modo de funcionamiento en las aplicaciones de 64 bits
<a name="modo_funcionamiento_las_aplicaciones_64_bits_ELTPARAGRAPHE000184"></a>

En una aplicación de 64 bits, esta función utilizará automáticamente el controlador Twain de 64 bits.. No se puede utilizar el TWAIN si este controlador no está instalado..

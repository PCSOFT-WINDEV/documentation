
## OCRParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa las opciones generales del OCR nativo (Reconocimiento óptico de caracteres). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creation of options
MyOCROptions is ocrOption

// Detect only numbers when reading an image
MyOCROptions.OnlyDigit = True

OCRParameter(MyOCROptions)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OCRParameter(<Advanced options>)`
---

**`<Advanced options>: (Variable de tipo ocrOption)`**

Nombre de la variable de tipo [ocrOption](../WDLang6/1000025768.md) que contiene los parámetros avanzados de gestión del OCR.

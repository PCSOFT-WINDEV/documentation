
## gpwSupervisor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de usuario de la cuenta "Supervisor" según el framework que se está utilizando.. Esta cuenta se crea automáticamente cuando el programa de grupo es set.. 

Observación: Dado que el groupware y sus archivos son multilingües, es posible recuperar el valor de la constante que representa el nombre de la cuenta "Supervisor".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF sCurrentAccount = gpwSupervisor() THEN
	// Program below the code to run 
	// when the supervisor is the current account 
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwSupervisor()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de usuario del supervisor según el framework utilizado. Corresponde a: 

- "Supervisor" si el framework está en inglés,

- "Supervisar" si el framework está en francés,


## ConsoleError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe una cadena de caracteres en el flujo de salida de errores estándar "stderr".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConsoleError("Error 404")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ConsoleError(<Message>)`
---

**`<Message>: (Cadena de caracteres)`**

Cadena para escribir. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Atención: en Windows, el ejecutable generado debe estar en modo consola para poder ver la cadena escrita.

<a name="XComponent"></a>

## Componente:
wd280std.dll


## gpwCheckUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia y la validez de la contraseña de un usuario de el groupware usuario.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nRes is int = gpwCheckUser(Login, Password)
// If the login failed
IF nRes <> gpwOK THEN
        SWITCH nRes
                CASE gpwError: Error("Error while initializing the groupware.")
                CASE gpwUnknownUser: Error("Unknown user.")
                CASE gpwInvalidPassword: Error("Invalid password")
        END
        EndProgram()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwCheckUser(<Login> , <Password>)`
---

**`<Result>: (Integro)`**

Resultado de la verificación: 


|   |   |
| --- | --- |
| *gpwError* | Se produjo un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). |
| *gpwInvalidPassword* | La contraseña de usuario no es válida. No corresponde a la contraseña de usuario descrita en la base de datos de groupware usuario.  |
| *gpwOK* | El usuario está conectado. |
| *gpwUnknownUser* | El usuario es desconocido: no se encuentra en la base de datos de groupware usuario. |



**`<Login>: (Cadena de caracteres)`**

Nombre de usuario. 

**`<Password>: (Cadena de caracteres)`**

Contraseña associated con el nombre de usuario.

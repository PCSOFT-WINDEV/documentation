
## gpwCheckPassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba una contraseña (cifrada o no).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF gpwCheckPassword(sLogin, sPassword) THEN
	RESULT gpwOk
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwCheckPassword(<Login> , <Password>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la contraseña corresponde al usuario, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Login>: (Cadena de caracteres)`**

Nombre de usuario. 

**`<Password>: (Cadena de caracteres)`**

Contraseña associated con el nombre de usuario.




## gpwGetUserInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera información sobre el usuario actual (aplicación que está utilizando el User Groupware).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the first name of user
FirstName is string 
FirstName = gpwGetUserInfo(gpwInfoFirstName)

// Retrieve the list of groups for the user
sGroupList is string = gpwGetUserInfo(gpwInfoGroup)

FOR EACH STRING sGroup OF sGroupList SEPARATED BY TAB
        Trace(sGroup)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gpwGetUserInfo(<Information>)`
---

**`<Result>: (Cadena, booleano o gpwUser Variable)`**



- Información solicitada.

- Variable de tipo [gpwUser](../WDLang6/1000021088.md), si se ha utilizado la constante *gpwFullDetails*.




**`<Information>: (Constante)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *gpwFullDetails* | Recupera un Variable de tipo [gpwUser](../WDLang6/1000021088.md) que contiene toda la información sobre el usuario actual.<br>Observación: la propiedad **Password** de la variable no se especifica. |
| *gpwInfoFirstName* | Nombre del usuario |
| *gpwInfoGroup* | Nombre del grupo para el usuario o lista de grupos separados por pestañas. |
| *gpwInfoLastName* | Apellido del usuario |
| *gpwInfoLogin* | Login del usuario o constante correspondiente al tipo de usuario: <br><br>- *gpwSupervisor*: Supervisor de la solicitud<br><br>- *gpwVisitante*: Visitante.<br><br><br> |
| *gpwInfoSupervisor* | Permite saber si el usuario es un administrador (&lt;Resultado&gt; se establecerá en <u><u><u><u>True</u></u></u></u>) o no (&lt;Resultado&gt; se establecerá en <u><u><u><u>False</u></u></u></u>). |
| *gpwInfoTwoFactorAuthentication* | Determina si la autenticación de dos factores está activada o desactivada (&lt;Resultado&gt; set a <u><u><u><u>True</u></u></u></u> o <u><u><u><u>False</u></u></u></u>, respectivamente) para el usuario conectado. Esta información se proporciona para cada aplicación. |

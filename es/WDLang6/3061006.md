


## BurnerSave( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Graba los archivos encontrados en la compilación en el CD o DVD.

Durante el proceso de quemado, **BurnerSave** llama regularmente a un lenguaje WL específico Procedure. Este procedimiento se utiliza para manejar los diferentes eventos que ocurren durante la quemadura Process.

**Observaciones**:

- Después de llamar a **BurnerSave**, la compilación actual se borra automáticamente.

- Esta función no está bloqueando. Para saber si la quema se ha completado, utilice el lenguaje WLanguage Procedure llamado regularmente por **BurnerSave**.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003061006&name=burnersave_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Actual CD burning
// The burner door will be automatically opened at the end of burn process
ResBurn is boolean 
ResBurn = BurnerSave(burnerAutoEject, ProcBurnCD)
// To view the entire example, click the "Example" link
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerSave(<Burn options> [, <WLanguage procedure> [, <Pointer>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició la grabación de los archivos del CD o DVD,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).
	Para saber si la quema es completa, pruebe la Event que se encuentra en el &lt;Nombre de la Procedure&gt; Procedure.




Observación: Un CD o DVD debe encontrarse en la grabadora.. Este CD/DVD debe contener suficiente espacio para grabar una nueva sesión.

**`<Burn options>: (Constante o combinación de constantes de tipo Integer)`**

Opción para grabar archivos en el CD o DVD:


|   |   |
| --- | --- |
| *burnerAutoEject* | Abre la puerta del quemador al final del quemado Process. |
| *burnerFinalize* | Finaliza los medios de comunicación: no se puede realizar ninguna otra operación de escritura en el medio. <br>Observación: Sin embargo, los medios reescribibles pueden ser borrados y reutilizados. |
| *burnerNoEject* | No abre la puerta del quemador al final de la quemadura Process. |
| *burnerSimulate* | Simula la grabación de CD o DVD. Todos los pasos de la grabación de Process se llevan a cabo, excepto la propia grabación de los archivos en el CD/DVD. |


Por defecto, los archivos se queman realmente en el CD/DVD (sin simulación) y la puerta de la grabadora no se abre al final de la grabación Process.

**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

Nombre del procedimiento WLanguage ("callback") llamado regularmente durante la quemadura Process. Esta Procedure se utiliza para gestionar los diferentes eventos que ocurren.
Este procedimiento tiene el siguiente formato:


```txt
PROCEDURE <Procedure name>(<Event encountered>, <Event message>, 
	<Procedure pointer>, <Parameter 1>, <Parameter 2>)
```
Los parámetros de este Procedure son opcionales.
No hay necesidad de pasarle parámetros a este Procedure. De hecho, estos parámetros se rellenan automáticamente cada vez que se procesa un archivo.

**`<Pointer>: (Entero opcional)`**

Puntero pasado al procedimiento &lt;Nombre de procedimiento&gt;.

## Observaciones


### Procedure se utiliza para gestionar los diferentes eventos que se producen
<a name="procedure_utiliza_para_gestionar_los_diferentes_eventos_que_producen_ELTPARAGRAPHE000094"></a>

La función **BurnerSave** llama regularmente al procedimiento &lt;Nombre de procedimiento&gt;. Este procedimiento se utiliza para manejar los diferentes eventos que ocurren durante la quemadura Process. Este Procedure es un Procedure local o global.

Esta Procedure debe ser declarada de la siguiente manera:

```txt
PROCEDURE <Procedure name>(<Event encountered>, <Event message>, 
	<Procedure pointer>, <Parameter 1>, <Parameter 2>)
```


- &lt;Event encontrada&gt; es una constante que indica el tipo de Event.

- &lt;Mensaje Event&gt; es una cadena de caracteres que contiene el mensaje del Event encontrado.

- &lt;Puntero Procedure&gt; es un entero cuyo valor es el que se pasa al parámetro &lt;Puntero&gt; de **BurnerSave**. Si no se especifica &lt;Pointer&gt;, &lt;Pointer&gt; es set a 0.

- &lt;Parámetros 1&gt; y &lt;Parámetros 2&gt; son números enteros que contienen un valor específico de la Event que ocurrió.




Los diferentes valores son los siguientes:

| Event | &lt;Evento encontrado&gt; | &lt;Mensaje evento&gt; | &lt;Parámetro 1&gt; | &lt;Parámetro 2&gt; |
| --- | --- | --- | --- | --- |
| Cancelar la quemadura Process | *burnerNotificationCancel* | "" | 0 | 0 |
| Error detectado | *burnerNotificationError* | Mensaje de error | Código de error que se ha producido | 0 |
| Cerrar el CD/DVD | *burnerNotificationCloseCD* | "" | Duración (en segundos) del cierre del CD/DVD | 0 |
| Quemar Process completado | *burnerNotificationEndBurning* | Mensaje de error si &lt;Parámetro 1&gt; es diferente de 0, cadena vacía (""") en caso contrario | Código de retorno, 0 si no hay error | 0 |
| Quemar un bloque | *burnerNotificationBurnBlock* | "" | Rango del bloque actual | Número total de bloques a quemar |
| Grabar una pista | *burnerNotificationBurnTrack* | "" | Rango de la pista actual | Número total de pistas a grabar |
| Prepárate para quemar | *burnerNotificationPrepareBurning* | "" | Duración (en segundos) de la preparación para la grabación | 0 |
| Otro Event | ** | Mensaje de Event | Rango del paso actual | Número total de pasos a realizar |


Para obtener el valor de &lt;puntero Procedure&gt;, su valor debe asignarse al valor de &lt;puntero&gt; en la Procedure con [Transferencia](../WDLang1/3014015.md).

**Observación**: Los parámetros de este Procedure son opcionales: tiene la capacidad de especificar la Event y el mensaje por ejemplo.
<a name="NOTE0_2"></a>


### Sesiones y pistas
<a name="sesiones_pistas_ELTPARAGRAPHE000215"></a>

Los CDs/DVDs se dividen en sesiones y pistas. Una sesión corresponde a todos los archivos grabados al mismo tiempo durante una sola operación.. Una pista corresponde a un archivo. Una sesión puede contener una o más pistas.

Cada nueva operación de grabación en el mismo CD/DVD genera una nueva sesión en el CD/DVD.

**Observación**: Para un CD/DVD de audio, algunos lectores de CD/DVD sólo pueden leer las pistas de la primera sesión.
<a name="NOTE0_3"></a>


### Recopilación
<a name="recopilacion_ELTPARAGRAPHE000226"></a>

Una compilación corresponde a la set de archivos que deben ser grabados en un CD o DVD. Esta compilación se tendrá en cuenta la próxima vez que se llame a **BurnerSave**.

Una compilación se crea automáticamente durante la primera llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o a [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar ([BurnerMediaTipo](../WDLang6/3061013.md)).

- al seleccionar el quemador default ([QuemadorSeleccionar](../WDLang6/3061012.md)).

- después de llamar a la función **BurnerSave** (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>).

- al cancelar la grabación process ([QuemadorCancelar](../WDLang6/3061003.md)).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.



<a name="NOTE0_4"></a>


### CD/DVD de audio grabado
<a name="cddvd_audio_grabado_ELTPARAGRAPHE000266"></a>

No puede grabar una compilación en un CD/DVD de audio que ya esté grabado.




## Funciones de gestión de grupos de usuarios
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para personalizar la gestión del Grupo de Usuarios.



|   |   |
| --- | --- |
| [gpwActivateUser](../WDLang6/1000021096.md) | Valida un nuevo usuario en la base de datos de User Groupware activando el usuario. |
| [gpwAddAuthLogin](../WDLang6/1000023576.md) | Agrega un botón (Facebook, Google, Microsoft, etc.) de conexión en la ventana o página de inicio de sesión. |
| [gpwAddUser](../WDLang6/1000021094.md) | Añade un usuario a la base de datos de User Groupware. |
| [gpwAddUserToGroup](../WDLang6/1000024595.md) | Agrega un usuario a un grupo de la base de datos del Groupware Usuario. |
| [gpwAuthLogin](../WDLang6/1000023597.md) | Le permite conectarse al User Groupware utilizando la autenticación con una cuenta como Facebook, Google, Microsoft, etc. |
| [gpwBiometricAuthentication](../WDLang6/1000024596.md) | Activa la autenticación biométrica para el usuario conectado o recupera el estado actual del usuario conectado. |
| [gpwCheckPassword](../WDLang6/1000021618.md) | Comprueba una contraseña (cifrada o no). |
| [gpwCheckUser](../WDLang6/1000021104.md) | Comprueba la existencia y la validez de la contraseña de un usuario de el groupware usuario. |
| [gpwConnectUser](../WDLang6/1000021099.md) | Conecta a un usuario con el programa de grupo de usuarios. |
| [gpwDefault](../WDLang6/1000023556.md) | Devuelve el valor que corresponde a "Default" según el framework utilizado. |
| [gpwDisconnectUser](../WDLang6/1000021100.md) | Desconecta al usuario actual del Groupware Usuario. |
| [gpwEncryptedPasswords](../WDLang6/1000021616.md) | Se utiliza para averiguar si las contraseñas de el groupware usuario están encriptadas. |
| [gpwEncryptPassword](../WDLang6/1000021617.md) | Devuelve la contraseña cifrada. |
| [gpwEnumControl](../WDLang6/1000020536.md) | Enumera los controles que se encuentran en una ventana o Page. |
| [gpwFindUser](../WDLang6/1000021087.md) | Recupera las características de un usuario del groupware usuario.. |
| [gpwForgottenPassword](../WDLang6/1000021097.md) | Ejecuta una acción para recuperar o cambiar la contraseña de un usuario. |
| [gpwGetFileInfo](../WDLang6/3041008.md) | Recupera el directorio de los archivos de datos que se encuentran en el Grupo de Usuarios. |
| [gpwGetLDAPInfo](../WDLang6/1000017063.md) | Recupera la configuración del servidor LDAP utilizado para el User Groupware.  |
| [gpwGetRights](../WDLang6/1000020728.md) | Se utiliza para get los derechos concedidos a un control, una ventana, una página o un reporte. |
| [gpwGetUserInfo](../WDLang6/3041004.md) | Recupera información sobre el usuario actual (aplicación que está utilizando el User Groupware). |
| [gpwImportLDAPUsers](../WDLang6/1000017058.md) | Importa los usuarios almacenados en un directorio LDAP al archivo de usuarios de groupware usuario. |
| [gpwInitAnalysis](../WDLang6/3041005.md) | Iniciar el análisis de groupware usuario. |
| [gpwInLDAPMode](../WDLang6/1000017056.md) | Permite saber si la identificación de los usuarios en el groupware usuario se hará a través de un directorio LDAP. |
| [gpwListOAuthParameters](../WDLang6/1000024479.md) | Recupera la lista de parámetros OAuth definidos para el Groupware Usuario, es decir, la lista de proveedores (Google, Facebook, etc.) a los que es posible conectarse a través del Groupware Usuario.. Estos parámetros se definieron a través del sitio de administración del Groupware Usuario. |
| [gpwLogin](../WDLang6/3041001.md) | Conecta a un usuario con una aplicación que está utilizando el User Groupware. |
| [gpwMode](../WDLang6/1000020566.md) | Devuelve el modo de ejecución del Grupo de Usuarios. |
| [gpwModifyUser](../WDLang6/1000021095.md) | Modifica un usuario en la base de datos de groupware usuario. |
| [gpwNone](../WDLang6/1000023557.md) | Devuelve el valor de la palabra "NINGUNO" según el framework utilizado. |
| [gpwOpen](../WDLang6/3041002.md) | Abre la ventana de inicio de sesión de User Groupware o Page. |
| [gpwOpenConfiguration](../WDLang6/3041009.md) | Abre la ventana o página de configuración de User Groupware. |
| [gpwOpenConnection](../WDLang6/3041007.md) | Abre una conexión a una base de datos específica Cliente/Servidor para gestionar los archivos de groupware usuario en modo Cliente/Servidor. |
| [gpwOpenLDAPSession](../WDLang6/1000017055.md) | Informa al User Groupware que la autenticación de los usuarios se hará a través de un directorio LDAP. |
| [gpwOpenSite](../WDLang6/1000020821.md) | Permite iniciar el sitio cuando se utiliza el Groupware Usuario. |
| [gpwRegisteredUsers](../WDLang6/1000023559.md) | Devuelve el valor que corresponde a "Usuarios registrados" según el framework utilizado. |
| [gpwRemoteConnect](../WDLang6/1000024855.md) | Permite conectarse a la aplicación a través del Webservice del Groupware Usuario. |
| [gpwResetPassword](../WDLang6/1000024854.md) | Restablece la contraseña de un usuario registrado. |
| [gpwSupervisor](../WDLang6/1000023555.md) | Devuelve el nombre de usuario de la cuenta "Supervisor" según el framework utilizado. |
| [gpwVisitor](../WDLang6/1000023558.md) | Devuelve el valor que corresponde a la palabra "Visitante" según el framework utilizado. |

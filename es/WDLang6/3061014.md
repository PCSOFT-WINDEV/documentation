
## BurnerEject( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre o cierra la puerta del quemador actual. Este quemador puede ser seleccionado por [QuemadorSeleccionar](../WDLang6/3061012.md).

**Observación**: Para conocer la lista de quemadores disponibles desde el ordenador actual, utilice [Lista de quemadores](../WDLang6/3061009.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opens the burner door
ResOpen is boolean 
ResOpen = BurnerEject() 
// Error occurred? 
IF ResOpen = False THEN 
	// Display the error 
	Error(ErrorInfo(errMessage)) 
ELSE 
	Info("Insert the CD or DVD into the burner") 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerEject([<Opening>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación solicitada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Opening>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado) si la bandeja de la unidad debe abrirse,

- <u><u><u><u>False STRIMEULB</u></u></u></u> si la puerta del quemador debe estar cerrada.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Abrir y cerrar la puerta del quemador
<a name="abrir_cerrar_puerta_del_quemador_ELTPARAGRAPHE000059"></a>Si la puerta del quemador ya está abierta, el siguiente código Line no tendrá efecto: 


```wl
BurnerEject(True)
```


Del mismo modo, si la puerta del quemador ya está cerrada, el siguiente código Line no tendrá efecto: 


```wl
BurnerEject(False)
```


Algunos quemadores ignoran la petición de cerrar la puerta. En este caso, la función **BurnerEject** devuelve <u><u><u><u>True</u></u></u></u> aunque la bandeja de la unidad permanezca abierta.
<a name="NOTE0_2"></a>

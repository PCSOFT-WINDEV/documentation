
## BurnerProperty( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer y modificar las propiedades del quemador actual.. Este quemador puede ser seleccionado por [QuemadorSeleccionar](../WDLang6/3061012.md).

**Observación**: Para conocer la lista de quemadores disponibles desde el ordenador actual, utilice [Lista de quemadores](../WDLang6/3061009.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Burner speed
ResSpeed is int 
ResSpeed = BurnerProperty(burnerSpeed)

Info("The write speed is: " + ResSpeed + "x")
```


<a name="Example2"></a>



```wl
// Management of "SmartBurn"
BurnerProperty(burnerSmartBurn, 1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar las propiedades de los quemadores

`<Result> = BurnerProperty(<Burner property>)`
---

**`<Result>: (Integro)`**

Valor de Property solicitado.

**`<Burner property>: (Constante de tipo Integer)`**

Solicitó a Property:


|   |   |
| --- | --- |
| *burnerMaxSpeed* | Velocidad máxima de escritura |
| *burnerSecondsBetweenTracks* | Número de segundos entre pistas. |
| *burnerSmartBurn* | Gestión de SmartBurn:<br><br>- 0: SmartBurn no es compatible.<br><br>- 1: SmartBurn es compatible.<br><br><br> |
| *burnerSpeed* | Velocidad de escritura actual |




<a name="SYNTAX2"></a>

### Modificando las propiedades del quemador

`<Result> = BurnerProperty(<Burner property> , <New value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la propiedad,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Burner property>: (Constante de tipo Integer)`**

Property para modificar:


|   |   |
| --- | --- |
| *burnerSecondsBetweenTracks* | Número de segundos entre pistas. |
| *burnerSmartBurn* | Gestión de SmartBurn:<br><br>- 0: SmartBurn no es compatible.<br><br>- 1: SmartBurn es compatible.<br><br><br> |
| *burnerSpeed* | Velocidad de escritura. |



**`<New value>: (Integro)`**

Nuevo valor para el Property solicitado.

<a name="NOTE0_1"></a>

## Observaciones


### SmartBurn
<a name="smartburn_ELTPARAGRAPHE000101"></a>

La tecnología Smart Burn (Smart Monitoring And Adapting Recording Technology for Burning) se utiliza para optimizar la velocidad de grabación en función de la calidad del CD y para evitar errores tales como "buffer underruns".

Esta tecnología está disponible para algunos tipos de quemadores.
<a name="NOTE0_2"></a>


### Configuración necesaria 
<a name="configuracion_necesaria_ELTPARAGRAPHE000110"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La grabación de CD está disponible para Windows XP y versiones posteriores. La grabación de DVDs está disponible para Windows Vista y para Windows XP cuando se utiliza la actualización KB932716 para Windows XP (esta actualización debe descargarse desde el sitio de Microsoft e instalarse manualmente porque no se tiene en cuenta en el mecanismo de Live Update)..

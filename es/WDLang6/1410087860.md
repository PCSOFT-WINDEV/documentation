


## &lt;Variable Scanner&gt;.WithUserInterface (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un documento de un escáner en uno o más archivos, utilizando la interfaz de usuario del escáner.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyScanner is Scanner
// Scanner with user interface
arrFile is array of string = MyScanner.WithUserInterface()
// Display paths of scanned files
FOR EACH FileName OF arrFile
	Trace(FileName)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Scanner>.WithUserInterface([<File path>] [, <Options>])`
---

**`<Result>: (Array)`**

Array de cadenas que contiene las rutas de los archivos escaneados. 
Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.

**`<Scanner>: (Variable de tipo Scanner)`**

Nombre de la variable de tipo [Scanner](../WDLang6/1410087808.md) inicializada con los datos del escáner utilizado. Solo se tendrá en cuenta el nombre del escáner. Los demás parámetros del escáner estarán determinados por las opciones elegidas por el usuario en la interfaz de usuario del escáner.

**`<File path>: (Cadena de caracteres opcional)`**

Nombre y ruta completa (o relativa) del archivo (hasta 260 caracteres). La extensión del archivo no se tendrá en cuenta ya que dependerá del tipo de archivo elegido por el usuario. Se puede utilizar una ruta UNC.
El nombre de cada archivo escaneado se indexará para no sobrescribir un archivo existente.
Si no se especifica la ruta, los archivos se generan en el directorio actual con el prefijo "scan" (por ejemplo, si el usuario seleccionó el formato bmp, se generará un archivo "scan.bmp").

**`<Options>: (Combinación de constantes opcionales de tipo Integer)`**

Opciones de escaneo con la interfaz de usuario:


|   |   |
| --- | --- |
| *scanUIOptionDefault*<br> (Valor predeterminado) | Utiliza las opciones predeterminadas. |
| *scanUIOptionSingleImage* | Restringe la interfaz del usuario para escanear un solo archivo.<br><br>Observación: Algunos escáneres no admiten esta opción. |
| *scanUIOptionSystem* | Muestra la interfaz de usuario del sistema, si está disponible, en lugar de la interfaz de usuario específica del escáner. |

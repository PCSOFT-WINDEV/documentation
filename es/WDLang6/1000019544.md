


## Funciones de gestión de planes de acción (fábrica de software)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los planes de acción de la fábrica de software:



|   |   |
| --- | --- |
| [ActionPlanConfigure](../WDLang6/1000021598.md) | Configura un plan de acción en la fábrica de software. |
| [ActionPlanExecute](../WDLang6/1000021332.md) | Ejecuta una instancia del plan de acción en la fábrica de software. |
| [ActionPlanInfo](../WDLang6/1000021829.md) | Recupera la información relativa a la ejecución de un plan de acción. |
| [apDisableAction](../WDLang6/1000019620.md) | Desactiva la ejecución de una acción que se encuentra en un plan de acción. |
| [apExecute](../WDLang6/1000019457.md) | Ejecuta un plan de acción. |
| [apFullProjectName](../WDLang6/1000019640.md) | Recupera el nombre físico completo de un proyecto abierto durante la ejecución de un plan de acción.. |
| [apLog](../WDLang6/1000019632.md) | Guarda una información en un archivo "log" durante la ejecución de un plan de acción. |

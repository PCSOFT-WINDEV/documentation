
## PDFNumberOfPages( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número total de páginas que se encuentran en un archivo PDF.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbPages is int
NbPages = PDFNumberOfPages("C:\temp\MyDocument.pdf")
Info("Number of pages in the PDF: " + NbPages)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PDFNumberOfPages(<PDF file> [, <Password>])`
---

**`<Result>: (Integro)`**

Número total de páginas en el archivo PDF.

**`<PDF file>: (Cadena de caracteres)`**

Nombre y ruta del archivo PDF a analizar.

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña necesaria para abrir el archivo si el archivo PDF tiene contraseña protected.

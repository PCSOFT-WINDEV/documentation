
## BurnerMediaFound( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si el quemador está vacío o no.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MediaFound is boolean
MediaFound = BurnerMediaFound()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BurnerMediaFound()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se encuentra un medio en el quemador. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.

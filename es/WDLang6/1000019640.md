
## apFullProjectName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el nombre físico completo de un proyecto abierto durante la ejecución de un plan de acción.. 

**Atención**: Esta función sólo se puede ejecutar:

- del código de un plan de acción.

- de un WLanguage Procedure que se encuentra en un proyecto o en una configuración de "Plan de Acción".


WINDEV, WEBDEV o WINDEV Mobile deben haber sido iniciados (por una acción previa). 

**Observación**: Esta función es una función utilizada para manipular, el editor de WINDEV, WEBDEV o WINDEV Mobile.. Por lo tanto, está disponible en los tres productos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Writes the project name into the log
apLog(apFullProjectName())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = apFullProjectName()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre completo del proyecto actualmente procesado.

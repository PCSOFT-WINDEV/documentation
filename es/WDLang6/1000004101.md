
## CDEject( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre o cierra la puerta del reproductor de CD/DVD seleccionado en el ordenador actual.

**Observación**: Para conocer la lista de reproductores de CD/DVD disponibles en el ordenador actual, utilice [Lista de CD](../WDLang6/1000004102.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opens the door of drive E:
ResOpen is boolean 
ResOpen = CDEject("E:")
// Error occurred?
IF ResOpen = False THEN
	// Display the error
	Error(ErrorInfo(errMessage))
ELSE
	Info("Insert the CD into the player")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CDEject(<Player> [, <Opening>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación solicitada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Player>: (Cadena de caracteres)`**

Jugador cuya puerta debe estar abierta o cerrada

**`<Opening>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si la bandeja de la unidad debe abrirse,

- <u><u><u><u>False</u></u></u></u> si la bandeja de la unidad debe cerrarse.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Abrir y cerrar la puerta del reproductor
<a name="abrir_cerrar_puerta_del_reproductor_ELTPARAGRAPHE000058"></a>

- Si la bandeja de la unidad ya está abierta, **CDEject**(&lt;Unidad&gt;,True) no tendrá ningún efecto. Si la bandeja de la unidad está cerrada, la función **CDEject**(&lt;Unidad&gt;, False) tampoco tendrá ningún efecto.

- Algunos jugadores pueden ignorar la petición de cerrar la puerta. En este caso, la función **CDEject** devuelve <u><u><u><u>True</u></u></u></u> aunque la bandeja de la unidad permanezca abierta.

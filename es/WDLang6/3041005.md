
## gpwInitAnalysis (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Iniciar el análisis de groupware usuario.

**Observación**: Esta función se refiere a la Agrupación de Usuarios utilizada en modo compatible (se mantiene por compatibilidad con la versión 18). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the analysis of user groupware
IF gpwInitAnalysis() = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gpwInitAnalysis()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la inicialización se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.

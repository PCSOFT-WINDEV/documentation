
## aiNeuralNetworkModel (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **aiNeuralNetworkModel** define las características avanzadas de una red neuronal utilizada por la función [AIDetectModel](../WDLang6/1410087982.md). Las características de esta red neuronal se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Image declaration
MyImage is Image
MyImage = IMG_Test

// Declare a model
MyAINNModel is aiNeuralNetworkModel

MyAINNModel.Configuration = "MyModel.cfg"
MyAINNModel.TrainedWeights = "MyModel.weights.pb"
MyAINNModel.PixelScaleFactor = 1.0

// Caution: the X and Y dimensions depend on the model. 
// If the specified dimensions do not match the model,
// AIDetectModel returns an error. 
MyAINNModel.XDimension = 300 
MyAINNModel.YDimension = 300 

MyAINNModel.AverageIntensityR = 104
MyAINNModel.AverageIntensityG = 117
MyAINNModel.AverageIntensityB = 113
MyAINNModel.RGBColor = True

myMatrixArray is array of 1 array of 1 by 1 by 200 by 7 real

// Run model
myMatrixArray = AIDetectModel(MyAINNModel, MyImage)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo aiNeuralNetworkModel
<a name="propiedades_especificas_las_variables_tipo_aineuralnetworkmodel_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden utilizarse para manipular un modelo de red neuronal:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| AverageIntensityB | Integro | Intensidad media del color azul (B) de los datos de entrenamiento. <br><br>Esta propiedad es opcional. |
| AverageIntensityG | Integro | Intensidad media del color verde (G) de los datos de entrenamiento. <br><br>Esta propiedad es opcional. |
| AverageIntensityR | Integro | Intensidad media del color rojo (R) de los datos de entrenamiento. <br><br>Esta propiedad es opcional. |
| Configuración | Cadena de caracteres | Ruta completa del archivo que contiene la configuración del modelo.<br><br>Observación: Los tipos de configuración posibles son: <br><br>- Caffe: \*.prototxt<br><br>- Tensorflow: \*.pbtxt<br><br>- Darknet: \*.cfg<br><br><br> |
| OutputLayerName | Cadena de caracteres | Nombre de la capa de la red neuronal de salida. |
| PixelScaleFactor | Real | Se utiliza escalar los valores de los pixeles. El valor predeterminado es 1,0 (sin escalado). |
| RGBColor | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si las imágenes proporcionadas están codificadas en formato RGB. En este caso, la conversión a la codificación BGR se hará automáticamente. <br><br>- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si las imágenes proporcionadas están codificadas en formato BGR. <br><br><br><br><br>Esta propiedad es opcional. |
| TrainedWeights | Character string | Ruta completa del archivo que contiene los pesos entrenados del modelo. <br><br>Observación: Los posibles tipos de pesos son: <br><br>- Caffe: \*.caffemodel<br><br>- Tensorflow: \*.pb<br><br>- Darknet: \*.weights<br><br>- Open Neural Network Exchange (ONNX): \*.onnx<br><br><br> |
| XDimension | Integro | Tamaño de la imagen requerido por el modelo: Valor de X. <br><br>De forma predeterminada, esta propiedad corresponde al ancho de la imagen. |
| YDimension | Integro | Tamaño de la imagen requerido por el modelo: Valor de Y. <br><br>De forma predeterminada, esta propiedad corresponde a la altura de la imagen. |

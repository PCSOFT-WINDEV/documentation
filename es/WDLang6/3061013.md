
## BurnerMediaType( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer y modificar el formato del CD o DVD a grabar. El CD o DVD es un CD o DVD de datos de Default.

**Observación**: La compilación actual se elimina automáticamente cuando se modifica el tipo de CD o DVD a grabar.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Burn a data CD 
BurnerMediaType(mediaType_CD_ROM_XA)
...
// Add the files 
BurnerAddFile("C:\MyDirectory\MyFileA.TXT", "\MyFiles\FileA.TXT") 
BurnerAddFile("C:\MyDirectory\MyFileB.TXT", "\MyFiles\FileB.TXT")
...
// Burn the CD 
BurnerSave("MyCD")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificar el tipo de CD o DVD

`<Result> = BurnerMediaType(<Type of CD or DVD>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se tuvo en cuenta la modificación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Type of CD or DVD>: (Constante de tipo Integer)`**

Tipo de CD o DVD:


|   |   |
| --- | --- |
| *mediaType_CD_ROM_XA* | CD de datos. <br>El CD se creará en modo "Track at Once.<br>Los datos se escribirán con las especificaciones de Joliet e ISO 9660..<br>Se pueden añadir otros archivos y los archivos existentes se pueden actualizar posteriormente. |
| *mediaType_CDDA_CDROM* | Audio CD. <br>El CD se creará en modo "Pista a la vez" (pausa de tamaño fijo entre pistas).<br>No se puede añadir ningún otro archivo de audio o datos a partir de entonces. |
| *mediaType_Datos* | CD o DVD de datos. <br>El CD o DVD se creará en modo "Pista a la vez.<br>Los datos se escribirán con las especificaciones de Joliet e ISO 9660..<br>En un CD solamente, se pueden agregar otros archivos o actualizar los archivos existentes posteriormente. |


**Observación**: No se puede grabar ningún DVD de audio.

## Observaciones


### Recopilación
<a name="recopilacion_ELTPARAGRAPHE000115"></a>

Una compilación corresponde a la set de archivos que deben ser grabados en un CD o DVD. Esta recopilación se tendrá en cuenta la próxima vez que se llame a [BurnerSave](../WDLang6/3061006.md).

Una compilación se crea automáticamente durante la primera llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o a [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar (**BurnerMediaType**).

- al seleccionar el quemador default ([QuemadorSeleccionar](../WDLang6/3061012.md)).

- después de llamar a la función [BurnerSave](../WDLang6/3061006.md) (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>).

- al cancelar la grabación process ([QuemadorCancelar](../WDLang6/3061003.md)).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.

### Presencia de un CD en la grabadora
<a name="presencia_grabadora_ELTPARAGRAPHE000158"></a>

Para saber el tipo de CD, el CD debe estar insertado en la grabadora. Para modificar el tipo de CD, el CD insertado en la grabadora debe tener suficiente espacio para grabar una nueva sesión..  
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### CD y DVD con varias sesiones
<a name="dvd_con_varias_sesiones_ELTPARAGRAPHE000167"></a>

Los CDs de datos pueden ser CDs de varias sesiones: tiene la posibilidad de añadir otros archivos o de modificar los archivos existentes posteriormente.

Esta funcionalidad no está disponible:

- Para los CDs de audio

- Para los DVDs de datos.




**Consejo**: Para añadir datos a un DVD, por ejemplo:

1. copy el contenido del DVD al disco duro.

2. Borra el DVD.

3. Vuelva a grabar el DVD con el nuevo contenido.

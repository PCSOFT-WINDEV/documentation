
## OCRModelPathDirectory (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite establecer el directorio que contiene los modelos de idioma para el sistema de OCR nativo.

Atención: Todos los modelos de idioma deben estar en el mismo directorio.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OCRModelPathDirectory("C:\OCR\ModelPath")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = OCRModelPathDirectory([<Directory>])`
---

**`<Result>: (Cadena de caracteres)`**

Ruta del directorio de los modelos de idioma. 

Si no se especifica el parámetro &lt;Directorio&gt;, &lt;Resultado&gt; corresponderá al directorio actual. 

**`<Directory>: (Cadena de caracteres opcional)`**

Nueva ruta del directorio ruta de los modelos de idioma. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El producto incluye tres idiomas: Español, inglés y francés. Si se utiliza uno de estos idiomas, no es necesario especificar la ruta del directorio de los modelos de idioma. Los modelos de idioma correspondientes se almacenan automáticamente en el mismo directorio que el ejecutable. Si se utilizan otros modelos personalizados, todos los modelos se deben almacenar en el mismo directorio.

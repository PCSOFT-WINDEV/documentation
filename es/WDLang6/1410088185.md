
## Python: Uso en WLanguage

## Presentación
<a name="presentacion_ELTTEXTE000074"></a>
WLanguage permite el uso de funciones Python en modo nativo (para obtener más información, consulte [Python: Llamadas nativas](../WDLang6/1410088133.md)). También es posible ejecutar funciones Python a través de funciones WLanguage. 

Hay varias funciones disponibles: 



|   |   |
| --- | --- |
| [PythonAddPath](../WDLang6/1410088169.md) | Añade el ruta a un directorio o el ruta al entorno virtual para buscar módulos de Python. |
| [PythonExecute](../WDLang6/1410088102.md) | Ejecuta una función de un módulo de Python. |
| [PythonInitialize](../WDLang6/1410088100.md) | Indica la ubicación de la DLL de Python a cargar. |
| [PythonTerminate](../WDLang6/1410088101.md) | Descarga la DLL de Python y libera la memoria. |






### Atención
<a name="atencion_ELTPARAGRAPHE000127"></a>

- Los módulos ".py" deben estar en el disco, en la carpeta del ejecutable implementado, o en la ubicación especificada con la función [PythonAddPath](../WDLang6/1410088169.md). Los módulos ".py" no se pueden encontrar si están integrados en la biblioteca del ejecutable.

- Python distingue entre mayúsculas y minúsculas. Los nombres de los módulos y de las funciones deben escribirse respetando las mayúsculas y minúsculas.

## Ejemplo de código
<a name="ejemplo_codigo_ELTTEXTE000136"></a>

```wl
// Initialize the Python interpreter
PythonInitialize(PYTHON_INSTALL_PATH + "\Python39\python3.dll")
// Declare the import paths of the modules used
PythonAddPath("D:\source.test\pythonProject\venv\Lib\site-packages")
PythonAddPath("D:\source.test\pythonProject")
// Solve 5x³+x²-3x+2=0
sEquation is string = "5*x**3+x**2-3*x+2"
sListRoot is string = PythonExecute("main", "solver", sEquation)
nNbRoot is int = StringCount(sListRoot, ",") + 1
Trace("The ""[%sEquation%]"" equation admits [%nNbRoot%] roots:")
FOR n = 1 _TO_ nNbRoot
	Trace("r[%n%] = [%ExtractString(sListRoot, n, "", "")%]")
END
```

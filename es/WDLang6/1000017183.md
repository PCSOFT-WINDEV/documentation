


## Funciones SAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones permiten utilizar las funciones BAPI de un servidor SAP.:  



|   |   |
| --- | --- |
| [SAPConnect](../WDLang6/1000017120.md) | Conecta un sistema SAP R/3 a partir de la información proporcionada en la [Estructura SAP](../WDLang6/1000017184.md). |
| [SAPDisconnect](../WDLang6/1000017121.md) | Disconnects a SAP R/3 system. |
| [SAPExecute](../WDLang6/1000017122.md) | Llama a una función sincrónica o BAPI en un sistema SAP. |

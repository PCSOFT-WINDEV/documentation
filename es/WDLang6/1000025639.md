
## OCRTextBlock (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **ocrTextBlock** permite definir e identificar las características avanzadas de un bloque de texto leído por el OCR nativo. Las características del bloque de texto se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyOCRTextBlock is OCRTextBlock
arrTextBlocks is array of ocrTextBlock 
arrTextBlocks = OCRExtractTextBlock(MyImage)
Trace("Blocks extracted successfully")
FOR i = 1 _TO_ arrTextBlocks.Count
	MyOCRTextBlock = arrTextBlocks[i]
	let sString = MyOCRTextBlock.Text	
	let nLevel = MyOCRTextBlock.ConfidenceLevel
	Trace(sString)
	Trace("Confidence level: " + nLevel)
	Trace("----------------------------------------")
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo ocrTextBlock
<a name="propiedades_especificas_descripcion_variables_tipo_ocrtextblock_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades permiten manipular un bloque de texto leído por el OCR nativo:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ConfidenceLevel | Integer | Nivel de confianza. |
| Polygon | Variable de tipo [Polygon](../WDLang1/1000025624.md) | Polígono del contorno del texto. <br><br>Observación: Para el OCR, el polígono debe ser de 4 puntos. |
| Text | Character string | Texto contenido en el polígono. |

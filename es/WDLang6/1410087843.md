


## PDFAddWatermark (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Watermark a todas o algunas páginas de un archivo PDF existente.

Observación: Esta función puede utilizarse varias veces en la misma Document para añadir diferentes marcas de agua. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWatermark is Watermark
MyWatermark.Text = "Confidential"
MyWatermark.Font.Name = "Arial"
MyWatermark.Position = iCenterH + iBottom

MyPDF is pdfDocument = fExeDir() + [fSep()] + "RPT_NoName1.pdf"

// Add watermark to PDF document
PDFAddWatermark(MyPDF, MyWaterMark)

MyWatermark.Text = "Confidential - Do not distribute"
// Add watermark to one page of the PDF document
PDFAddWatermark(MyPDF.Page[1], MyWaterMark)

PDFSave(MyPDF, fExeDir() + [fSep()] + "RPT_NoName1_With_Watermark.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadir una Watermark a una página de un PDF Document

`PDFAddWatermark(<Page> , <Watermark>)`
---

**`<Page>: (pdfPage Variable)`**

Nombre de la variable [pdfPage](../WDLang6/1000024904.md) que corresponde a la página del archivo PDF a la que debe añadirse el Watermark. 

**`<Watermark>: (Watermark Variable)`**

Nombre de la variable [Watermark](../WDLang5/1000019922.md) correspondiente al Watermark que se añadirá al Page. 


<a name="SYNTAX2"></a>

### Añadir una Watermark a un PDF Document

`PDFAddWatermark(<Document> , <Watermark>)`
---

**`<Document>: (Variable de tipo pdfDocument)`**

Nombre de la variable [pdfDocument](../WDLang6/1000024909.md) que corresponde al PDF Document a utilizar. 

**`<Watermark>: (Watermark Variable)`**

Nombre de la variable [Watermark](../WDLang5/1000019922.md) que corresponde al Watermark que se añadirá en todas las páginas del PDF Document. 


<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Añadir una Watermark a la Document mostrada en un lector de PDF control

`PDFAddWatermark(<PDF Reader control> , <Watermark>)`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular

**`<Watermark>: (Watermark Variable)`**

Nombre de la variable [Watermark](../WDLang5/1000019922.md) que corresponde al Watermark que se añadirá en todas las páginas del PDF Document.

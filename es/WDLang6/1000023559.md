
## gpwRegisteredUsers( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor que corresponde a "Usuarios registrados" según el framework que se está utilizando.. El término "Usuarios registrados" se utiliza cuando un usuario está registrado en el groupware.

Observación: Dado que el groupware y sus archivos son multilingües, es posible recuperar el valor de la constante que representa el término "Usuarios registrados".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCstgpwRegisteredUsers is string
sCstgpwRegisteredUsers = gpwRegisteredUsers()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwRegisteredUsers()`
---

**`<Result>: (Cadena de caracteres)`**

Valor de "Usuarios registrados" según el framework utilizado: 

- "Usuarios registrados" si el framework está en inglés,

- "Utilisateurs enregistrés" si el framework está en francés.

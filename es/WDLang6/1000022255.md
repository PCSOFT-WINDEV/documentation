
## pdfSignature (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfSignature** permite definir las características avanzadas de una firma que se aplicará a un archivo PDF: 

- al firmar un PDF existente (vía [PDFSign](../WDLang6/1000022241.md)). 

- durante una impresión (vía [iPrintSignature](../WDLang5/1000021690.md). 


Las características de esta firma pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySignature is pdfSignature
MySignature.Certificate = CertificateLoad("test.pfx", "pwd")[1]
MySignature.Caption = "Signed by test"
MySignature.X = 10
MySignature.Y = 10
MySignature.Width = 150
MySignature.Height = 50
PDFSign("MyPDFFile.pdf", MySignature)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables pdfSignature
<a name="propiedades_especificas_las_variables_pdfsignature_ELTPARAGRAPHE000040"></a>

Las siguientes propiedades pueden ser utilizadas para manipular la firma de un archivo PDF:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Altura | Real | Altura del rectángulo de firma en la Page (en milímetros). Esta altura debe ser superior a 0. |
| anchura | Real | Anchura del rectángulo de firma en la Page (en milímetros). Esta anchura debe ser superior a 0. |
| BackgroundColor | Constante de tipo Integer | Fondo color utilizado para la zona de la firma. Este parámetro puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br><br><br><br> |
| Cepillo de fondo | Constante de tipo Integer | Patrón utilizado para rellenar el borde de la firma: <br><br>- *iCrossed*: Cepillo cruzado.<br><br>- *iDiagonalCrossed*: Cepillo cruzado en diagonal.<br><br>- *iDownwardDiagonal*: Cepillo diagonal descendente.<br><br>- *iUpwardDiagonal*: Cepillo diagonal ascendente.<br><br>- *iHorizontal*: Cepillo horizontal.<br><br>- *iSolid* (valor predeterminado): Cepillo sólido.<br><br>- *iVertical*: Cepillo vertical.<br><br><br> |
| Certificate | Variable de tipo [Certificate](../WDLang1/1000019287.md) | Certificate solía firmar el PDF. |
| Franja de orilla | Variable de tipo [Franja de orilla](../Motscles/15140100.md) | description de borde de firma. |
| HorizontalAlignment | Constante de tipo Integer | Alineación horizontal del texto de la firma: <br><br>- *iCenterH*: Texto centrado horizontalmente.<br><br>- *iLeft* (valor predeterminado): Texto alineado a la izquierda.<br><br>- *iRight*: Texto alineado a la derecha.<br><br><br> |
| Image | Variable de tipo [Image](../WDLang1/1000019650.md) | Firma Image. <br><br>Para hacer invisible una firma, este Property y el **Caption** Property no deben especificarse. |
| Opacidad | Integro | Opacidad del fondo Color en la zona de la firma. Valor incluido entre 0 y 100: <br><br>- 0: Fondo transparente, <br><br>- 100: Opaco Background. <br><br><br> |
| Page | Integro | Número de la Page donde aparece la firma. Esta Property es set a 0 por defecto (lo que significa que la firma aparece al final de Document). |
| Pie de foto | Cadena de caracteres | Texto de la firma. <br><br>Para hacer una firma Invisible , no se debe especificar este Property y la **Imagen** Property. La propiedad **Caption** también puede ser una cadena vacía.  |
| VerticalAlignment | Constante de tipo Integer | Alineación vertical del texto de la firma: <br><br>- *iBottom*: Texto alineado en la parte inferior.<br><br>- *iCenterV*: Texto centrado verticalmente.<br><br>- *iTop* (valor predeterminado): Texto alineado en la parte superior<br><br><br> |
| X | Real | Coordenada X de la posición del área de firma en la Page (en milímetros) |
| Y | Real | Coordenada Y de la posición del área de firma en la Page (en milímetros) |


<a name="NOTE0_2"></a>


### Firma visible o Invisible
<a name="firma_visible_invisible_ELTPARAGRAPHE000183"></a>

Si no se especifican las propiedades **Título** e **Imagen** , la firma será Invisible: el Document será firmado y una barra azul será mostrada por Adobe Reader pero la firma no será visible en el Document. 

En caso contrario, la firma aparecerá en el documento con los parámetros especificados por las propiedades del tipo **pdfSignature** .

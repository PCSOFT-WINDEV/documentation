
## ScanSelect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un cuadro de diálogo estándar con los escáneres disponibles. El usuario puede seleccionar el escáner deseado: la información correspondiente se obtendrá y asignará a una variable de tipo [Scanner](../WDLang6/1410087808.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Scanner selection by the user
MyScanner is Scanner = ScanSelect()
// No scanner connected to the computer or operation canceled by the user?
IF ErrorOccurred _OR_ MyScanner.Name = "" THEN
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScanSelect()`
---

**`<Result>: (Variable de tipo Scanner)`**

Variable de tipo [Scanner](../WDLang6/1410087808.md) inicializada con los datos del escáner seleccionado por el usuario en el cuadro de diálogo estándar. 
Si se produce un error, o si el usuario no seleccionó un escáner en el cuadro de diálogo, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El comportamiento de la función varía según el número de escáneres conectados al equipo actual:

- Si no hay ningún escáner conectado al equipo actual, no se muestra el cuadro de diálogo estándar y la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.

- Si hay al menos un escáner disponible, aparecerá un cuadro de diálogo estándar con los diferentes escáneres disponibles en el equipo actual. El usuario podrá seleccionar el escáner a utilizar.

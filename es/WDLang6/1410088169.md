
## PythonAddPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade el ruta a un directorio o el ruta al entorno virtual para buscar módulos de Python. 

Observación: Mediante Default, los módulos de Python se buscan desde el directorio de ejecución. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF PythonInitialize("C:\Programs\Python\Python39\python39.dll") THEN
	PythonAddPath("C:\tmp\python-modules\")
	PythonExecute("test_module", "id", 5)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PythonAddPath(<Module path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha tenido en cuenta el ruta, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Module path>: (Cadena de caracteres)`**

Ruta donde se encuentran los módulos de Python. 

Atención: se ignoran los subdirectorios.


## gpwEnumControl( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los controles que se encuentran en una ventana o Page. 

**Observaciones**: 

- Esta función es utilizada por el groupware usuario para enumerar los controles cuyo estado (Invisible, gris, inactivo) puede ser modificado por el groupware usuario.

- **Esta función se puede usar incluso si el User Groupware no está habilitado en el proyecto**. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PROCEDURE EnumChildControl(sParentName is string)

// Browse the controls
nIndex is int = 1
sNameChildControl is string
sControlInfo is string = gpwEnumControl(sParentName, nIndex)
WHILE sControlInfo <> ""
	Trace(sControlInfo)
	// Retrieve the full name of the child control
	sNameChildControl = ExtractString(sControlInfo, 1, TAB)
	Trace(sNameChildControl)
	// Next control
	nIndex++
	sControlInfo = gpwEnumControl(sParentName, nIndex)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwEnumControl(<Parent element> , <Index>)`
---

**`<Result>: (Cadena de caracteres)`**



- Características de control encontradas. Estas características tienen el siguiente formato: 
	
	```txt
	<Full control name> + TAB + <Full parent name> + TAB + ...
	<Type> + TAB + <Plain type> + TAB + <Caption>
	```

	donde: 

	- : El nombre completo del control encontrado. 

	- : Nombre completo del padre de control (cadena vacía para una ventana).

	- &lt;Tipo&gt;: identificador de tipo control ([escribir a máquina](../Proprietes/2510131.md)). 

	- : Tipo legible de control (por ejemplo, "Botón", "Barra de desplazamiento", ...). 

	- &lt;Título&gt;: Título del control. 




- "" (cadena vacía) si la enumeración ha terminado. 




**`<Parent element>: (Cadena de caracteres)`**

Nombre del objeto "padre" (que contiene el control). Este parámetro corresponde al nombre de una ventana o Page. 

**`<Index>: (Integro)`**

Índice del control para enumerar. Cada control es associated con un número. Este número corresponde al orden en que se han creado los controles..

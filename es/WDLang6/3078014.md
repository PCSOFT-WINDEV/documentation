


## TwainScanArea( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para definir o reiniciar el área a escanear.

**Observación**: Solo se debe llamar a la función **TwainScanArea** si el dispositivo Twain utilizado corresponde a un escáner. En los demás casos, se ignora la función **TwainScanArea**.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the area to scan
TwainScanArea(0, 0, 150, 280)
// Display the document found in the scanner in the "IMG_MyImage" control
// Only the area to scan will be displayed
TwainToControl(IMG_MyImage, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Reinicialización de la zona a escanear

`TwainScanArea()`
---


<a name="SYNTAX2"></a>

### Definir el área a escanear

`TwainScanArea(<X1> , <Y1> , <X2> , <Y2>)`
---

**`<X1>: (Integro)`**

Coordenada X (en milímetros) de la Corner superior izquierda del área a escanear.

**`<Y1>: (Integro)`**

Coordenada Y (en milímetros) de la Corner superior izquierda del área a escanear.

**`<X2>: (Integro)`**

Coordenada X (en milímetros) de la Corner inferior derecha del área a escanear.

**`<Y2>: (Integro)`**

Coordenada Y (en milímetros) de la Corner inferior derecha del área a escanear.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento en las aplicaciones de 64 bits
<a name="modo_funcionamiento_las_aplicaciones_64_bits_ELTPARAGRAPHE000167"></a>

En una aplicación de 64 bits, esta función utilizará automáticamente el controlador Twain de 64 bits.. No se puede utilizar el TWAIN si este controlador no está instalado..

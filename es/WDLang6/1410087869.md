
## Escanear un documento (mediante un controlador WIA)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
Windows Image Acquisition (WIA) es un modelo de controlador e interfaz de programación de aplicaciones (API) de Windows que permite a las aplicaciones escanear documentos utilizando un escáner. Este estándar está reemplazando cada vez más a Twain. 

WINDEV incluye funciones para escanear documentos utilizando escáneres con controladores WIA. 

Las funciones de escáner permiten:

- escanear el documento mediante programación.

- iniciar la interfaz del escáner (o la interfaz del sistema correspondiente). Esta interfaz permite guardar el documento, cambiar la configuración predeterminada del dispositivo, etc., ... Las acciones disponibles dependen del dispositivo utilizado.




Observación: Se admite WIA 2. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000116"></a>
El principio es el siguiente: 

1. Defina una variable de tipo [Scanner](../WDLang6/1410087808.md). Esta variable contiene las características del escáner utilizado. 

2. Obtener las características del escáner mediante las siguientes funciones: 


|   |   |
| --- | --- |
| [ScanGet](../WDLang6/1410087836.md) | Identifica las características de un escáner y las almacena en una variable de tipo [Scanner](../WDLang6/1410087808.md). |
| [ScanList](../WDLang6/1410087807.md) | Devuelve la lista de los escáneres conectados al equipo actual. |
| [ScanSelect](../WDLang6/1410087835.md) | Abre un cuadro de diálogo estándar con los escáneres disponibles. |

3. Escanear el documento utilizando las siguientes funciones: 


|   |   |
| --- | --- |
| [ScanToImage](../WDLang6/1410087838.md) | Recupera una Document de un escáner en una Variable de tipo [Image](../WDLang1/1000019650.md). |
| [ScanToPDF](../WDLang6/1410087839.md) | Guarda un documento escaneado en un archivo PDF. |
| [ScanWithUserInterface](../WDLang6/1410087840.md) | Guarda un documento de un escáner en uno o más archivos, utilizando la interfaz de usuario del escáner. |




Observaciones: 

- El tipo [Scanner](../WDLang6/1410087808.md) está disponible en modo de lectura y escritura. Esto permite cambiar uno o más parámetros entre dos escaneos. Las constantes *scanxxxDefault* permite conservar el último valor especificado para las diferentes propiedades. 

- La función [ScanProperty](../WDLang6/1410087853.md) permite obtener y cambiar las diferentes propiedades del escáner utilizadas por el controlador (propiedades de la API de Windows).

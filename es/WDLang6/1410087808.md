
## Scanner (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Scanner** manipula la información de un escáner. Las características del escáner se pueden definir y cambiar mediante diferentes propiedades WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Scan an image
MyScanner is Scanner = ScanSelect()
IMG_Scan = ScanToImage(MyScanner)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo Scanner
<a name="propiedades_especificas_las_variables_tipo_scanner_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades permiten manipular las características de un escáner:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Area | [Rectangle](../Motscles/1514079.md) | Área a escanear (en milésimas de milímetro). De forma predeterminada, se utiliza un rectángulo con coordenadas y dimensiones cero (0,0,0,0) para escanear toda la página.<br><br>**Observación**: La función ScanProperty tiene en cuenta esta área si se utiliza la constante **scanPaperSizeCustom**. |
| BitPerPixel | Integer | <br><br>- Número de bits por pixel, <br><br>- Constante *scanBitsPerPixelDefault* (valor predeterminado) si el valor de la propiedad no debe cambiar en el siguiente escaneo. La propiedad utilizará el valor anterior. <br><br><br> |
| Brightness | Integer | <br><br>- Brillo, entre -1000 y 1000,<br><br>- Constante *scanBrightnessDefault* (valor predeterminado) si el valor de la propiedad no debe cambiar en el siguiente escaneo. La propiedad utilizará el valor anterior. <br><br><br> |
| ColorFormat | Constante de tipo Integer | Formato de color de la imagen:<br><br>- *scanColorFormatAuto*: Formato de color detectado automáticamente por el dispositivo (valor predeterminado).<br><br>- *scanColorFormatRGB*: Imagen RGB (24 bits por pixel).<br><br>- *scanColorFormatGrayscale*: Imagen en escala de grises.<br><br>- *scanColorFormatBlackWhite*: Imagen en blanco y negro.<br><br><br> |
| Contraste | Integer | <br><br>- Contraste, entre -1000 y 1000,<br><br>- Constante *scanContrastDefault* (valor predeterminado) si el valor de la propiedad no debe cambiarse en el próximo escaneo. La propiedad utilizará el valor anterior.<br><br><br> |
| Duplex | Constante de tipo Integer | Opciones del modo de doble cara: <br><br>- Constante *scanDuplexDefault* (valor predeterminado) si la propiedad no debe cambiar en el siguiente escaneo. La propiedad utilizará el valor anterior.<br><br>- *scanDuplexYes*: escaneo a dos caras.<br><br>- *scanDuplexNo*: escaneo a una cara. <br><br><br> |
| Height | Integer | <br><br>- Altura máxima (en milésimas de milímetro),<br><br>- Constante *scanHeightDefault* (valor predeterminado).<br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Name | Character string | <br><br>- Nombre del escáner,<br><br>- Cadena vacía si no se ha inicializado la variable (valor predeterminado).<br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| ProcedureProgress | Procedure | <br><br>- Nombre del procedimiento WLanguage al que se llama al escanear una página,<br><br>- Cadena vacía para no utilizar un procedimiento (valor predeterminado). <br><br><br><br><br>Este procedimiento tiene el siguiente formato: <br><br><pre><code><Procedure name>(<Page number>, <br>	<Progress percentage>)</code></pre><br>donde: <br><br>- &lt;Page number&gt;: Número de la página escaneada.<br><br>- &lt;Progress percentage&gt;	: Porcentaje de progreso del escaneo de la página (entre 0 y 100)<br><br><br>Si el procedimiento devuelve <u><u><u><u>False</u></u></u></u>, el escaneo se interrumpe. |
| Resolución | Integer | <br><br>- Resolución de la imagen en pixeles por pulgada,<br><br>- Constante *scanResolutionDefault* (valor predeterminado) si el valor de la propiedad no debe cambiar en el siguiente escaneo. La propiedad utilizará el valor anterior.<br><br><br> |
| ScanFrom | Constante de tipo Integer | Indica el origen del escaneo: <br><br>- *scanFromAuto*: los documentos se escanearán desde el alimentador si hay un alimentador y hay páginas en él. En caso contrario, los documentos se escanearán desde el cristal (valor predeterminado). <br><br>- *scanFromFeeder*: el escaneo se realiza desde el alimentador de documentos. <br><br>- *scanFromFlatBed*: el escaneo se realiza desde el cristal. <br><br><br> |
| Width | Integer | <br><br>- Ancho máximo (en milésimas de milímetro),<br><br>- Constante *scanWidthDefault* (valor predeterminado).<br><br><br><br><br>**Esta propiedad es de solo lectura**. |

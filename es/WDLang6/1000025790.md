
## OCRLoadLanguage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un idioma al OCR nativo. 

Observaciones: 

- La función **OCRLoadLanguage** permite agregar un modelo ".traineddata".

- En iOS, utilice la función [OCRAddLanguage](../WDLang6/1000025728.md) para agregar un idioma. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyModelName is string = "eng"
OCRLoadLanguage(MyModelName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Agregar un idioma

`OCRLoadLanguage(<Model file>)`
---

**`<Model file>: (Cadena de caracteres)`**

Nombre corto del modelo a agregar.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Agregar varios idiomas

`OCRLoadLanguage(<Array of model files>)`
---

**`<Array of model files>: (Array de cadenas de caracteres)`**

Array que contiene los nombres cortos de los modelos a agregar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El modelo que contiene el idioma debe estar ubicado en el directorio del ejecutable. 

- Los modelos en español, inglés y francés están integrados en el producto. Los nombres cortos de estos modelos son: "spa", "eng" y "fra". Se seleccionan de acuerdo con el idioma actual de la aplicación.  

- La extensión del archivo modelo debe ser ".traineddata". Esta extensión no es necesaria al llamar a la función: solo se requiere el nombre corto. 

- Hay otros modelos disponibles en el sitio de TESSERACT:

	- [https://github.com/tesseract-OCR/tessdata_best](https://github.com/tesseract-OCR/tessdata_best)

	- [https://github.com/tesseract-OCR/tessdata](https://github.com/tesseract-OCR/tessdata)

	- [https://github.com/tesseract-OCR/tessdata_fast](https://github.com/tesseract-OCR/tessdata_fast)


 La lista de idiomas está disponible en: [https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files-in-different-versions.md](https://github.com/tesseract-ocr/tessdoc/blob/master/Data-Files-in-different-versions.md)

- Se pueden combinar varios idiomas: 

	- deben agregarse uno a uno (sintaxis 1).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) pueden agregarse en una sola operación mediante un array (sintaxis 2).

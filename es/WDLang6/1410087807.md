
## ScanList (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de los escáneres conectados al equipo actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the list of scanners of the current computer in LIST_Scanner
ListAdd(LIST_Scanner, ScanList())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScanList()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de escáneres conectados al equipo actual, con el siguiente formato:

```txt
<Scanner name 1> + CR + ... <Scanner name N>
```


donde: &lt;Scanner name 1&gt; ... &lt;Scanner name N&gt; corresponden a los nombres de los escáneres conectados.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Dos escáneres no pueden tener el mismo nombre: el sistema añade automáticamente un "#&lt;número&gt;" si ya existe un escáner con el mismo nombre.




## PDFMergeWithPassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fusiona varios archivos PDF existentes en un solo archivo PDF. Las contraseñas de los diferentes archivos PDF son compatibles. Los archivos se fusionan uno tras otro.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PDFMergeWithPassword("C:\PDF Doc\pdf_merge.pdf", ...
	"C:\PDF Doc\ReportQuery.pdf", "pwd", "C:\PDF Doc\Concepts.pdf", "pwd2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFMergeWithPassword(<Result PDF file> , <PDF file 1> , <PDF Password 1> , <PDF file 2> , <PDF Password 2> [, <PDF file 3> , <PDF password 3> [... [, <PDF file N> , <PDF Password N>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se generó el archivo de resultado,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Result PDF file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo PDF a crear. Este archivo contendrá los archivos PDF que se han fusionado.

**`<PDF file 1>: (Cadena de caracteres)`**

Nombre y ruta completa del primer archivo PDF a fusionar.

**`<PDF Password 1>: (Cadena de caracteres)`**



- Contraseña para abrir el primer archivo PDF, 

- Cadena vacía ("") si el fichero no tiene contraseña.




**`<PDF file 2>: (Cadena de caracteres)`**

Nombre y ruta completa del segundo archivo PDF a fusionar.

**`<PDF Password 2>: (Cadena de caracteres)`**



- Contraseña para abrir el segundo archivo PDF, 

- Cadena vacía ("") si el fichero no tiene contraseña.




**`<PDF file 3>: (Cadena de caracteres opcional)`**

Nombre y ruta completa del 3er archivo PDF a fusionar.

**`<PDF password 3>: (Cadena de caracteres opcional)`**



- Contraseña para abrir el tercer archivo PDF.

- Cadena vacía ("") si el fichero no tiene contraseña.






**`<PDF file N>: (Cadena de caracteres opcional)`**

Nombre y ruta completa del archivo PDF Nth a fusionar.

**`<PDF Password N>: (Cadena de caracteres opcional)`**



- Contraseña para abrir el archivo PDF Nth, 

- Cadena vacía ("") si el fichero no tiene contraseña.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los archivos PDF protected por una contraseña de administrador no se pueden fusionar. 

- Hasta la versión 18: 

	- Esta función sólo genera archivos PDF en la versión 1.4. 

	- La fusión no es compatible con los archivos PDF de la versión 1.5 y posteriores.. 




- A partir de la versión 19, los archivos PDF de la versión 1.5 son compatibles con WINDEV y WEBDEV. 

- Los marcadores que se encuentran en los archivos PDF no se guardan mediante la operación de fusión.

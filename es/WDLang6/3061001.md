
## BurnerAddFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un archivo a la compilación. Cuando se hayan añadido todos los archivos solicitados a la compilación, se podrá grabar el CD/DVD ([BurnerSave](../WDLang6/3061006.md)).

**Observación**: Para añadir a la compilación todos los archivos encontrados en un directorio, utilice [BurnerAddDirectory](../WDLang6/3061002.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003061001&name=burneraddfile_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Burn a data CD 
BurnerMediaType(MediaType_CD_ROM_XA) 
// Add the files 
ResAdd is boolean 
ResAdd = BurnerAddFile("C:\MyDirectory\MyFileA.TXT", "\MyFiles\FileA.TXT") 
ResAdd = BurnerAddFile("C:\MyDirectory\MyFileB.TXT", "\MyFiles\FileB.TXT") 
// To view the entire example, click the "Example" link
```


<a name="Example2"></a>



```wl
// Burn an audio CD
BurnerMediaType(mediaType_CDDA_CDROM) 
// Add the files 
ResAdd is boolean 
ResAdd = BurnerAddFile("C:\MyMusic\MySongA.WAV") 
ResAdd = BurnerAddFile("C:\MyMusic\MySongB.WAV") 
// To view the entire example, click the "Example" link
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerAddFile(<File to add> [, <File on CD or DVD>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó el archivo a la compilación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**Observación**: Un CD o DVD debe encontrarse en la grabadora.. Este CD o DVD debe tener suficiente espacio para grabar una nueva sesión.

**`<File to add>: (Cadena de caracteres)`**

Nombre y ruta del archivo que se añadirá a la compilación (hasta 260 caracteres). La ruta puede ser una ruta completa, relativa o UNC.
Para una compilación de audio, estos archivos deben corresponder a archivos ".WAV" (44100 Hz, 16 bits).



**`<File on CD or DVD>: (Cadena de caracteres opcional)`**

Nombre y ruta del archivo en el CD o DVD. Esta ruta debe comenzar con un carácter de barra invertida ("\\").
Por Default, el archivo será añadido en la raíz del CD/DVD.
Este parámetro se ignora para una compilación de audio. De hecho, el archivo se añadirá automáticamente en la raíz del CD/DVD..

<a name="NOTE0_1"></a>

## Observaciones




### Recopilación
<a name="recopilacion_ELTPARAGRAPHE000070"></a>

Una compilación corresponde a la set de archivos que deben ser grabados en un CD o DVD. Esta recopilación se tendrá en cuenta la próxima vez que se llame a [BurnerSave](../WDLang6/3061006.md).

Una compilación se crea automáticamente al llamar por primera vez a la función **BurnerAddFile** o [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar ([BurnerMediaTipo](../WDLang6/3061013.md)).

- al seleccionar el quemador default ([QuemadorSeleccionar](../WDLang6/3061012.md)).

- después de llamar a la función [BurnerSave](../WDLang6/3061006.md) (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>).

- al cancelar la grabación process ([QuemadorCancelar](../WDLang6/3061003.md)).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.



<a name="NOTE0_2"></a>




### Añadir varios archivos idénticos
<a name="anadir_varios_archivos_identicos_ELTPARAGRAPHE000112"></a>

Si el mismo archivo se añade varias veces a la compilación:

- para un CD/DVD de audio: este archivo se encontrará en el CD/DVD tantas veces como se agregó.

- para un CD/DVD de datos: sólo el último archivo añadido (mismo nombre, mismo directorio en el CD/DVD) se encontrará en el CD/DVD.



<a name="NOTE0_3"></a>


## LockAppOff( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desbloquea una aplicación que estaba bloqueada automáticamente o por [LockAppOn](../WDLang6/1000017011.md).

**Observación**: Esta función se utiliza principalmente en la ventana que desbloquea la aplicación.. También se puede utilizar para desbloquear una aplicación por programación (por ejemplo, mediante un atajo de teclado específico).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LockAppOff()
```

<a name="SYNTAX1"></a>

## Sintaxis

`LockAppOff()`

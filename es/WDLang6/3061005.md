


## BurnerErase( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los archivos encontrados en un CD o DVD regrabable. Este CD (o DVD) debe ser insertado en la grabadora actual.

Al borrar los archivos del CD o DVD, la función **BurnerErase** llama a un procedimiento específico de WLanguage. Este procedimiento se utiliza para manejar los diferentes eventos que ocurren durante el borrado de Process.

**Observaciones**:

- El quemador puede ser seleccionado por [QuemadorSeleccionar](../WDLang6/3061012.md).

- Esta función no está bloqueando. Para saber si el borrado se ha completado, utilice el lenguaje WLanguage Procedure llamado regularmente por **BurnerErase**.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003061005&name=burnererase_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Entirely erase the files found on CD
// The burner door will automatically opened at the end of erase process
ResErase is boolean 
ResErase = BurnerErase(burnerFullErasing + burnerAutoEject, ProcCDErase)
// To view the entire example, click the "Example" link
```

## Sintaxis

`<Result> = BurnerErase([<Erase options> [, <WLanguage procedure> [, <Pointer>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició el proceso para borrar los archivos del CD/DVD,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


Para determinar si la eliminación ha terminado, pruebe el evento encontrado en &lt;Procedure name&gt;.

**`<Erase options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Opciones para borrar archivos del CD/DVD:


|   |   |
| --- | --- |
| *burnerAutoEject* | Abre la puerta del quemador al final de borrar Process. |
| *burnerFastErasing* | Borrado rápido: los archivos no se borran completamente del CD/DVD. Sólo se borrarán las referencias al contenido del CD.<br><br>Este método es más rápido pero los archivos pueden ser restaurados por un software específico.. Este método no se recomienda si los archivos encontrados en el CD/DVD contienen información confidencial. |
| *burnerFullErasing* | Borrado completo: los archivos se borran completamente del CD/DVD.<br><br>Este método puede tardar bastante tiempo, pero los archivos eliminados no se pueden restaurar. |
| *burnerNoEject* | No abras la puerta del quemador al final del borrado de Process. |

Por defecto, el borrado de Process es rápido y la puerta del quemador no se abre automáticamente al final del borrado de Process.

**`<WLanguage procedure>: (Nombre de la Procedure opcional)`**

Nombre de WLanguage Procedure ("callback") llamado regularmente durante la operación de borrado. Esta Procedure se utiliza para gestionar los diferentes eventos que ocurren.
Este procedimiento tiene el siguiente formato:


```txt
PROCEDURE <Procedure name>(<Event encountered>, 
	<Event message>, <Procedure pointer>, <Parameter>)
```
Los parámetros de este Procedure son opcionales.
No hay necesidad de pasarle parámetros a este Procedure. De hecho, estos parámetros se rellenan automáticamente cada vez que se procesa un archivo.

**`<Pointer>: (Entero opcional)`**

Puntero pasado al procedimiento &lt;Nombre de procedimiento&gt;.

## Observaciones


### Procedure se utiliza para gestionar los diferentes eventos que se producen
<a name="procedure_utiliza_para_gestionar_los_diferentes_eventos_que_producen_ELTPARAGRAPHE000092"></a>

La función **BurnerErase** llama regularmente al procedimiento &lt;Nombre de procedimiento&gt;. Este procedimiento se utiliza para manejar los diferentes eventos que ocurren durante el borrado de Process. Este Procedure es un Procedure local o global.

Esta Procedure debe ser declarada de la siguiente manera:


```txt
PROCEDURE <Procedure name>(<Event encountered>, <Event message>, 
		<Procedure pointer>, <Parameter>)
```


- &lt;Event encontrada&gt; es una constante que indica el tipo de Event.

- &lt;Mensaje Event&gt; es una cadena de caracteres que contiene el mensaje del Event encontrado.

- &lt;Puntero Procedure&gt; es un entero cuyo valor es el que se pasa al parámetro &lt;Puntero&gt; de **BurnerErase**. Si no se especifica &lt;Pointer&gt;, &lt;Pointer&gt; es set a 0.

- &lt;Parámetro&gt; es un número entero que contiene un valor específico para la Event encontrada.
	Los diferentes valores son los siguientes:
	

| Event | &lt;Evento encontrado&gt; | &lt;Mensaje evento&gt; | &lt;Parámetro&gt; |
| --- | --- | --- | --- |
| Cancele el borrado de Process | *burnerNotificationCancel* | "" | 0 |
| Borrado (sólo disponible en Vista o Windows XP + KB032716) | *burnerNotificationErasing* | "" | En este caso, se gestionan dos parámetros: <br>***Parámetro 1***: Número de segundos desde el inicio de la operación de borrado. <br>***Parámetro 2***: Duración total estimada de la operación (puede cambiar durante la operación de borrado)<br>Atención: Esta información es devuelta por el quemador y puede no ser válida. |
| Error encontrado | *burnerNotificationError* | Mensaje de error | Código de error que se ha producido |
| Operación de borrado completada | *burnerNotificationEndErasing* | Mensaje de error si &lt;Parámetro&gt; es diferente de 0, cadena vacía (""") en caso contrario | Código de retorno, 0 si no hay error |
| Otro Event |   | Mensaje de Event | 0 |


	Para obtener el valor de &lt;puntero Procedure&gt;, su valor debe asignarse al valor de &lt;puntero&gt; en la Procedure con [Transferencia](../WDLang1/3014015.md).




Observación: Los parámetros de este Procedure son opcionales: tiene la capacidad de especificar la Event y el mensaje por ejemplo.

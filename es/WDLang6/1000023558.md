
## gpwVisitor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor que corresponde a la palabra "Visitante" según el framework utilizado. Un visitante es cualquier persona que no sea un supervisor.

Observación: Dado que el groupware y sus archivos son multilingües, es posible recuperar el valor de la constante que representa la palabra "visitante".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCstVisitor is string
sCstVisitor = gpwVisitor()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwVisitor()`
---

**`<Result>: (Cadena de caracteres)`**

Valor de la palabra "visitante" según la framework utilizada: 

- "Visitante (no conectado)" si el framework está en inglés,

- "Visiteur (non connecté)" si el framework está en francés.

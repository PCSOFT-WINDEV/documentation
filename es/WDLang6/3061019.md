
## BurnerVolumeName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar o modificar el nombre del CD o DVD que se va a grabar.

**Observación**: Esta función debe llamarse antes de añadir archivos a la compilación ([QuemadorAñadirArchivo](../WDLang6/3061001.md) o [BurnerAddDirectory](../WDLang6/3061002.md)). De lo contrario, el nombre del CD será ignorado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of CD
ResName is int = BurnerVolumeName("My CD")
// Error occurred?
IF ResName = False THEN
	// Display the error
	Error(ErrorInfo(errMessage))
ELSE
	// Add the files
	BurnerAddDirectory("C:\MyMusic\*.WAV")
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre del CD o DVD a grabar

`<Result> = BurnerVolumeName()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre del CD o DVD a grabar.  


<a name="SYNTAX2"></a>

### Modificar el nombre del CD o DVD a grabar

`<Result> = BurnerVolumeName(<Name of CD or DVD>)`
---

**`<Result>: (Integro)`**



- <u><u><u><u>True</u></u></u></u> si se tuvo en cuenta el nombre del CD o DVD,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Name of CD or DVD>: (Cadena de caracteres)`**

Nombre que se debe dar al CD o DVD para grabar. Este nombre se mostrará en el selector de archivos para identificar el CD o el DVD..

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Presencia de un CD/DVD en la grabadora
<a name="presencia_cddvd_grabadora_ELTPARAGRAPHE000056"></a>

Para encontrar el nombre de un volumen, debe encontrar un CD o DVD en la grabadora.. Para modificar el nombre del volumen, el CD/DVD insertado en la grabadora debe tener suficiente espacio para grabar una nueva sesión..  
<a name="NOTE0_2"></a>




### Configuración necesaria 
<a name="configuracion_necesaria_ELTPARAGRAPHE000064"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La grabación de CD está disponible para Windows XP y versiones posteriores. La grabación de DVDs está disponible para Windows Vista y para Windows XP cuando se utiliza la actualización KB932716 para Windows XP (esta actualización debe descargarse desde el sitio de Microsoft e instalarse manualmente porque no se tiene en cuenta en el mecanismo de Live Update)..

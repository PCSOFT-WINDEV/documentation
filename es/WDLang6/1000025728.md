
## OCRAddLanguage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un idioma al OCR nativo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add American English
MyModelName is string = "en_US"
OCRAddLanguage(MyModelName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OCRAddLanguage(<ISO Language>)`
---

**`<ISO Language>: (Cadena de caracteres)`**

Idioma que se agregará en formato ISO. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para plataformas diferentes de iOS, utilice la función [OCRLoadLanguage](../WDLang6/1000025790.md).

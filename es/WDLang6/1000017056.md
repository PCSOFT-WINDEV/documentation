
## gpwInLDAPMode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la identificación de los usuarios en el User Groupware se hará a través de un directorio LDAP.  
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gpwInLDAPMode()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el groupware usuario está utilizando un directorio LDAP para autenticar a los usuarios, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) &lt;Resultado&gt; siempre se establece en <u><u><u><u>False</u></u></u></u>.



<a name="XComponent"></a>

## Componente:
wd280gpu.dll


## Constantes utilizadas por las funciones de fábrica
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por las funciones de gestión de la fábrica de software: 



| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **apStatusDisconnected** | El robot utilizado para ejecutar el plan de acción ya no responde. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apStatusPending** | El plan de acción está a la espera de ser ejecutado. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apStatusInProgress** | El plan de acción se está ejecutando actualmente. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apStatusStartup** | El coordinador va a iniciar el plan de acción. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apStatusCompletedFailure** | La ejecución del plano finalizó con error. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apStatusCompletedOK** | La ejecución del plano finalizó sin errores. | [ActionPlanExecute](../WDLang6/1000021332.md) |
| **apInfoCancelationInProgress** | Información solicitada sobre el plan de acción: Cancelación en curso. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoProgress** | Información solicitada sobre el plan de acción: Porcentaje de progreso. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoStart** | Información solicitada sobre el plan de acción: Fecha y hora de inicio del plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoStatus** | Información solicitada sobre el plan de acción: Estado del plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoEnd** | Información solicitada sobre el plan de acción: Fecha y hora de finalización del plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoLog** | Información solicitada sobre el plan de acción: Contenido del registro de tiempo de ejecución del plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoComputer** | Información solicitada sobre el plan de acción: Nombre de el equipo que dirige el plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |
| **apInfoUser** | Información solicitada sobre el plan de acción: Nombre del usuario que inició el plan de acción. | [ActionPlanInfo](../WDLang6/1000021829.md) |

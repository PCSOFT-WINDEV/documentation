


## Funciones de archivos PDF (sintaxis prefijada)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>

 
WLanguage incluye distintos tipos de variables para los archivos PDF de manipular:  



|   |   |
| --- | --- |
| [pdfAttachment](../WDLang6/1000024899.md) | El tipo **pdfAttachment** permite definir las características avanzadas de un archivo adjunto en un documento PDF. |
| [pdfDocument](../WDLang6/1000024909.md) | El tipo **pdfDocument** obtiene todas las características de un archivo PDF: autor, fecha de creación, contenido de Page, etc. |
| [pdfPage](../WDLang6/1000024904.md) | El tipo **pdfPage** permite definir las características avanzadas de una página de un documento PDF cargado en memoria. |
| [pdfParameter](../WDLang6/1410087857.md) | El tipo **pdfParameter** define las características del archivo PDF que se va a generar. |
| [pdfSignature](../WDLang6/1000022255.md) | El tipo **pdfSignature** permite definir las características avanzadas de la firma que se aplicará a un archivo PDF al firmar un PDF existente (mediante la función [PDFSign](../WDLang6/1000022241.md)).  |
| [pdfTextElement](../WDLang6/1000024556.md) | El tipo **pdfTextElement** permite conocer las características de un elemento de texto en una página PDF. |






Las siguientes funciones se utilizan para manipular archivos PDF, utilizando la sintaxis de prefijo: 



|   |   |
| --- | --- |
| [&lt;pdfDocument Variable&gt;.Guardar](../WDLang6/1410088160.md) | Guarda el contenido de una variable de tipo [pdfDocument](../WDLang6/1000024909.md) en un archivo ".PDF". |
| [.GetSignature](../WDLang6/1410088159.md) | Obtiene la firma de un PDF Document como un buffer. |
| [.NbSignatures](../WDLang6/1410088154.md) | Obtiene el número de firmas en un PDF Document. |
| [AddWatermark](../WDLang6/1410087872.md) | Añade una Watermark a todas las páginas de un archivo PDF existente. |
| [GetSignatureDate](../WDLang6/1410088157.md) | Obtiene la fecha de una de las firmas de un PDF Document. |
| [.AddImage](../WDLang6/1410087875.md) | Añade una Image a una Page en un archivo PDF existente. |
| [AddWatermark](../WDLang6/1410087871.md) | Añade una Watermark a una Page de un archivo PDF existente. |







Observación: Las siguientes funciones se pueden utilizar para manipular archivos PDF directamente: 



|   |   |
| --- | --- |
| [PDFExtractPage](../WDLang6/1000021279.md) | Extrae una página de un archivo PDF en una Variable de tipo [Image](../WDLang1/1000019650.md). |
| [PDFIsProtected](../WDLang6/1000017491.md) | Comprueba si el archivo PDF requiere una contraseña para ser leído. |
| [PDFMerge](../WDLang6/1000019712.md) | Fusiona varios archivos PDF existentes en un solo archivo PDF. Los archivos se fusionan uno tras otro. Las rutas de los archivos PDF se pueden encontrar en un array.  |
| [PDFMergeWithPassword](../WDLang6/1000020126.md) | Combina varios archivos PDF existentes y con contraseña- protected en un solo archivo PDF. |
| [PDFNumberOfPages](../WDLang6/1000017480.md) | Devuelve el número total de páginas que se encuentran en un archivo PDF. |
| [PDFPassword](../WDLang6/1000024229.md) | Protege un PDF Document con una contraseña. |
| [PDFSign](../WDLang6/1000022241.md) | Aplica una firma de aprobación a un PDF existente. |
| [PDFToText](../WDLang6/1000017479.md) | Extrae el texto encontrado en un archivo PDF. |

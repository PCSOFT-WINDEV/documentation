
## BurnerCompilationInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las características de la compilación actual.

**Observación**: Una compilación corresponde al conjunto de archivos que se grabarán en un CD o DVD. Esta recopilación se tendrá en cuenta la próxima vez que se llame a [BurnerSave](../WDLang6/3061006.md).
Una compilación se crea automáticamente durante la primera llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o a [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar ([BurnerMediaTipo](../WDLang6/3061013.md)).

- al seleccionar el quemador default ([QuemadorSeleccionar](../WDLang6/3061012.md)).

- después de llamar a la función [BurnerSave](../WDLang6/3061006.md) (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u>True</u></u></u>).

- al cancelar la grabación process ([QuemadorCancelar](../WDLang6/3061003.md)).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Characteristics of compilation
ResNbTracks, ResBlocks, ResBlockSize are int
ResNbTracks = BurnerCompilationInfo(compilationNumberAudioTracks)
ResBlocks = BurnerCompilationInfo(compilationBlocksUsed)
ResBlockSize = BurnerCompilationInfo(compilationBlockSize)
 
Info("The compilation contains: " + ResNbTracks + " tracks ", ...
	"and it is using " + ResBlocks + " blocks of " + ResBlockSize + " bytes each.")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerCompilationInfo(<Type of characteristic>)`
---

**`<Result>: (Constante de tipo Integer)`**



- Característica solicitada,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 


Esta función no se puede utilizar si no hay ningún CD insertado en la grabadora.

**`<Type of characteristic>: (Constante de tipo Integer)`**

Tipo de característica a recuperar:


|   |   |
| --- | --- |
| *compilationBlockSize* | Tamaño del bloque (en bytes).<br>El tamaño del bloque depende del tipo de compilación (datos o audio). |
| *compilationBlocksUsed* | Número de bloques encontrados en la compilación actual.<br>Este número se actualiza después de cada llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o [BurnerAddDirectory](../WDLang6/3061002.md). |
| *compilationNumberAudioTracks* | <br><br>- Número de pistas de audio encontradas en la compilación actual.<br><br>- 0 si la compilación actual es una compilación de datos.<br><br><br>Este número se actualiza después de cada llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o [BurnerAddDirectory](../WDLang6/3061002.md). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### taco
<a name="taco_ELTPARAGRAPHE000108"></a>

- El tamaño del CD se expresa en bloques.

- Un bloque corresponde a 2048 bytes en un CD de datos y a 2352 bytes en un CD de audio.

- Un segundo en el CD corresponde a 75 bloques.


## TileDeleteAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las modificaciones hechas a la aplicación Tile. La aplicación Tile será el Tile definido en la edición. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TileDeleteAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TileDeleteAll()`

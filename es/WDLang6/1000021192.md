


## TwainProperty( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite get o cambiar las propiedades del dispositivo Twain actual. Este dispositivo Twain puede ser seleccionado por [TwainSelectSource](../WDLang6/3078013.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the scanner resolution
nResolution is int = TwainProperty(TwainResolution)
```
<a name="Example2"></a>

```wl
// Modify the resolution to 200 dpi
bApply is boolean = TwainProperty(TwainResolution, 200)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar las propiedades del actual dispositivo Twain

`<Result> = TwainProperty(<Property>)`
---

**`<Result>: (Integro)`**



- Valor de la Property especificada .

- 0 si el dispositivo TWAIN no soporta la modificación de la Property solicitada o el valor de la Property solicitada.

## Observaciones


### Propiedades específicas
<a name="propiedades_especificas_ELTPARAGRAPHE000365"></a>

Tiene la capacidad de recuperar o modificar el valor de un Property no figura en las constantes de Twain mencionadas en Above. En este caso, el parámetro &lt;Property&gt; corresponderá al valor de la constante CAP_xxx o ICAP_xxx de TWAIN.H, asociada a la Property solicitada.


### Modo de funcionamiento en las aplicaciones de 64 bits
<a name="modo_funcionamiento_las_aplicaciones_64_bits_ELTPARAGRAPHE000497"></a>

En una aplicación de 64 bits, esta función utilizará automáticamente el controlador Twain de 64 bits.. No se puede utilizar el TWAIN si este controlador no está instalado..

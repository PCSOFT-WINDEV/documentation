


## PDFAddImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Image a una Page en un archivo PDF existente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument
...
MyImage is Image = fExeDir() + [fSep()] + "Signature.pdf"
PDFAddImage(MyPDF.Page[1], MyImage, 10, 10, 150, 50, drHomotheticCentered)
PDFSave(MyPDF, fExeDir() + [fSep()] + "RPT_NoName1_With_Image.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PDFAddImage(<Page used> , <Image to add> , <X> , <Y> , <Width> , <Height> [, <Display mode>])`
---

**`<Page used>: (pdfPage Variable)`**

Nombre de la [pdfPage](../WDLang6/1000024904.md) Variable que corresponde a la página a la que hay que añadir el Image.

**`<Image to add>: (Control, Variable o nombre de cadena)`**

Image a añadir al fichero PDF Page. Esta imagen puede corresponder a: 

- una variable de tipo [Image](../WDLang1/1000019650.md), 

- el nombre y la ruta del Image que se añadirá al Page,

- un campo de tipo Image memo,

- un control Imagen. 




**`<X>: (Real)`**

Coordenada X de la Image en la Page (en milímetros).

**`<Y>: (Real)`**

Coordenada Y de la Image en la Page (en milímetros).

**`<Width>: (Real)`**

Anchura de Image (en milímetros).

**`<Height>: (Real)`**

Altura de Image (en milímetros).

**`<Display mode>: (Opcional Constante entera o combinación de constantes)`**

Modo de visualización de la imagen: 


|   |   |
| --- | --- |
| *drHighQuality* | Esta constante puede combinarse con las opciones de redimensionamiento homotético.<br><br>En modo homotético, esta constante permite mejorar la calidad de la imagen cuando esta se reduce con respecto a su tamaño inicial |
| *drHomothetic* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse. Ocupa:<br><br>- todo el alto especificado.<br><br>- todo el ancho especificado.<br><br><br> |
| *drHomotheticCentered* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse y se centra. Ocupa:<br><br>- todo el alto especificado: la imagen se centra horizontalmente.<br><br>- todo el ancho especificado: la imagen se centra verticalmente.<br><br><br> |
| *drHomotheticExtended* | La imagen se amplía o reduce para que el lado más pequeño de la imagen se muestre completamente. |
| *drStretched*<br>(Valor predeterminado) | La Image se estira si es necesario para ajustarse a la anchura y la altura especificadas. |
| *drTiled* | La imagen se repite para ocupar el nuevo ancho y alto especificados. |

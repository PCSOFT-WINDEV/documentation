
## gpwGetLDAPInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la configuración del servidor LDAP utilizado para el User Groupware.  
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gpwGetLDAPInfo(<Type of information>)`
---

**`<Result>: (Tipo de información solicitada)`**

Información solicitada.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Corresponde siempre a 0 o cadena vacía (""").

**`<Type of information>: (Constante de tipo Integer)`**

Información por recuperar:


|   |   |
| --- | --- |
| *gpwLDAPDNBase* | DN base (cadena de caracteres en formato "DC=xxx, DC=zzzz, ...."). |
| *gpwLDAPNoFile* | Modo para utilizar el directorio LDAP. <u><u><u><u>True</u></u></u></u> si el groupware usuario se inicia "sin archivo de datos", sólo en modo de autentificación. |
| *gpwLDAPPort* | Puerto del servidor LDAP. |
| *gpwLDAPServer* | Nombre del servidor LDAP. |
| *gpwLDAPVersion* | Número de la versión LDAP. |





<a name="XComponent"></a>

## Componente:
wd280gpu.dll

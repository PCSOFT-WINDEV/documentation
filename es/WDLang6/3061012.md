
## BurnerSelect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite seleccionar el quemador de Default. Este quemador será utilizado por Default por todos los [funciones de quemador](../WDLang6/3061015.md).

**Observación**:

- Por Default, el quemador utilizado corresponde al primer quemador devuelto por [Lista de quemadores](../WDLang6/3061009.md).

- La compilación actual se borra automáticamente cuando se selecciona el quemador Default.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the burners installed
Burner_List is string
Burner_List = BurnerList()
IF Burner_List ~= "" THEN
	Error("No burner was detected on the computer")
	RETURN
END
// Fill the TABLE_Burner table with the list of burners
ABurner, PathOfBurner are strings

FOR EACH STRING ABurner OF Burner_List SEPARATED BY CR
	// Select the burner
	IF BurnerSelect(ExtractString(ABurner, 1)) = False THEN
		Error("Failure selecting the burner", ErrorInfo())
		RETURN
	END
	// Path of current burner
	PathOfBurner = BurnerPath()
	// Add into the table
	TableAddLine(TABLE_Burner, PathOfBurner, ABurner)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BurnerSelect(<Burner>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se ha seleccionado el quemador,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Burner>: (Entero o cadena de caracteres)`**

Índice o identificador del quemador, devuelto por [Lista de quemadores](../WDLang6/3061009.md).

## Observaciones




### Recopilación
<a name="recopilacion_ELTPARAGRAPHE000062"></a>

Los [funciones de quemador](../WDLang6/3061015.md) están disponibles para Windows XP y posteriores.

Una compilación corresponde a la set de archivos que deben ser grabados en un CD o DVD. Esta recopilación se tendrá en cuenta la próxima vez que se llame a [BurnerSave](../WDLang6/3061006.md).

Una compilación se crea automáticamente durante la primera llamada a [QuemadorAñadirArchivo](../WDLang6/3061001.md) o a [BurnerAddDirectory](../WDLang6/3061002.md). Se puede crear una sola compilación a la vez.

Una compilación se elimina automáticamente:

- al cambiar el tipo de CD/DVD a grabar ([BurnerMediaTipo](../WDLang6/3061013.md)).

- al seleccionar el grabador utilizado de forma predeterminada (función **BurnerSelect**).

- después de llamar a la función [BurnerSave](../WDLang6/3061006.md) (solo si se inició la grabación, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>).

- al cancelar la grabación process ([QuemadorCancelar](../WDLang6/3061003.md)).

- al cerrar la aplicación WINDEV o el sitio WEBDEV actual.

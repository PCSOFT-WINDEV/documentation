


## Funciones Java
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones le permiten ejecutar fácilmente algún código de Java: 



|   |   |
| --- | --- |
| [JavaExecute](../WDLang6/3048003.md) | Inicia una aplicación Java o un applet Java invocando el método "Main" de una clase Java dada. |
| [JavaExecuteFunction](../WDLang6/3048004.md) | Inicia una función estática específica de una clase determinada de Java. |
| [JavaLoad](../WDLang6/3048001.md) | Especifica la ubicación de las clases de usuario necesarias para ejecutar la aplicación Java o el applet Java. |

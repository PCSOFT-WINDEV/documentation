
## pdfDocument (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **pdfDocument** obtiene todas las características de un archivo PDF: autor, fecha de creación, contenido de Page, etc.. Algunas características del PDF Document pueden modificarse mediante distintas propiedades de WLanguage. Es posible añadir o eliminar páginas. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "test.pdf"

FOR EACH DocumentPage OF MyPDF.Page
	Trace(DocumentPage.Height)
END

PDFSave(MyPDF, "passwordtest.pdf")

// It is also possible to use prefix syntax
// MyPDF.Save("passwordtest.pdf")
```



```wl
MyPDFDoc is pdfDocument = PDFOpen("C:\temp\Myguide.pdf")
Trace(MyPDFDoc.PDFVersion)
Trace(MyPDFDoc.Author)
Trace(MyPDFDoc.Page.Count)
Trace(MyPDFDoc.Page[1].Width)
Trace(MyPDFDoc.Page[36].Height)

FOR EACH ATextElement OF MyPDFDoc.Page[2].TextElement
	Trace(ATextElement.Text + " X = " + 
	ATextElement.X + " Y = " + ATextElement.Y + " L = " + 
	ATextElement.Width + " H = " + ATextElement.Height)
END

MyPDFDoc2 is PDFDocument = "C:\temp\plan.pdf"
// Add pages
Add(MyPDFDoc2.Page, MyPDFDoc.Page[5])
// Insert pages
Insert(MyPDFDoc2.Page, 1, MyPDFDoc.Page[32])
// Save modified PDF
PDFSave(MyPDFDoc2, "MyPDFDoc2.pdf")
ShellExecute("MyPDFDoc2.pdf")
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo **pdfDocument**
<a name="propiedades_especificas_las_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000065"></a>

Las siguientes propiedades permiten manipular un documento PDF:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Adjunto | Array de [pdfAttachment](../WDLang6/1000024899.md) | Archivos adjuntos del documento PDF. <br><br>Se pueden eliminar o agregar archivos adjuntos a un documento utilizando las funciones para manipular un array ([añadir](../WDLang1/1000017348.md), [suprimir](../WDLang1/1000017354.md), ... en el array de variables de tipo [pdfAttachment](../WDLang6/1000024899.md)). <br><br><br><br> |
| Autor | Cadena de caracteres | Autor del documento (solo si esta información está disponible en el documento PDF). <br><br>**Esta propiedad es de solo lectura**. |
| DateCreated | FechaHora | Fecha y hora de creación del documento (solo si esta información está disponible en el documento PDF). <br><br>**Esta propiedad es de solo lectura**.<br><br> |
| DateModified | FechaHora | Fecha y hora de la última modificación del documento (solo si esta información está disponible en el documento PDF). <br><br>**Esta propiedad es de solo lectura**.<br><br>Esta propiedad corresponde a "ModificationDate" en las versiones anteriores. |
| Generador | Cadena de caracteres | Nombre del programa que generó el documento (solo si esta información está disponible en el documento PDF). <br><br>**Esta propiedad es de solo lectura**. |
| marcador | Array de [pdfBookmark](../WDLang6/1410089195.md) | Marcadores del PDF Document. <br><br> |
| Page | Array de [pdfPage](../WDLang6/1000024904.md) | Páginas del documento PDF. <br><br>Se pueden eliminar o agregar páginas a un documento utilizando las funciones para manipular un array ([añadir](../WDLang1/1000017348.md), [suprimir](../WDLang1/1000017354.md), ... en el array de variables de tipo [pdfPage](../WDLang6/1000024904.md)). |
| Palabra clave | Cadena de caracteres | Palabra clave del documento (solo si esta información está disponible en el documento PDF). Si se especifican varias palabras clave, el separador utilizado depende del generador. <br><br>**Esta propiedad es de solo lectura**. |
| PDFVersion | Cadena de caracteres | Versión del estándar PDF utilizado en el documento PDF. <br><br>**Esta propiedad es de solo lectura**. |
| Título | Cadena de caracteres | Título del documento (solo si esta información está disponible en el documento PDF). <br><br>**Esta propiedad es de solo lectura**. |
| Valor | Cadena de caracteres o memoria intermedia | Documento PDF. Esta propiedad puede corresponder a: <br><br>- la ruta del documento PDF,<br><br>- el buffer que contiene el documento PDF. <br><br><br> |


<a name="NOTE0_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000218"></a>

- Para inicializar la variable de tipo **pdfDocument**, asigne una cadena o (buffer) que contenga un archivo PDF. Después puede modificar el documento (agregar archivos adjuntos o definir una contraseña, por ejemplo), y guardarlo de nuevo llamando a la función [PDFSave](../WDLang6/1000024864.md). 

- Puede eliminar o agregar páginas a un documento utilizando funciones de arrays ([añadir](../WDLang1/1000017348.md), [suprimir](../WDLang1/1000017354.md), etc, ... en el array de variables de tipo [pdfPage](../WDLang6/1000024904.md)). También es posible agregar o eliminar archivos adjuntos.

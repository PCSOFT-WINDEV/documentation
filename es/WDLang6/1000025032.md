
## PDFOpen (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un documento PDF y lo carga en una variable de tipo [pdfDocument](../WDLang6/1000024909.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = PDFOpen("test.pdf")

IF ErrorOccurred() THEN
   Error(ErrorInfo())
   RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFOpen(<Document source> [, <Password>])`
---

**`<Result>: (Variable de tipo pdfDocument)`**

Variable de tipo [pdfDocument](../WDLang6/1000024909.md) que corresponde al documento PDF especificado. 

Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.

**`<Document source>: (Tipo de fuente)`**

Corresponde a uno de los siguientes elementos: 

- Ruta completa o relativa del archivo PDF a abrir (hasta 260 caracteres).

- Nombre de la variable de tipo Buffer que contiene el documento PDF. 

- Nombre del elemento memo que contiene el documento PDF. Este parámetro tiene la siguiente forma: &lt;Archivo de datos&gt; &lt;Elemento&gt;.




**`<Password>: (Cadena de caracteres opcional)`**

Contraseña para abrir el archivo PDF (si el archivo está protegido con una contraseña).

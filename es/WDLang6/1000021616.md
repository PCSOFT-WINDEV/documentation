
## gpwEncryptedPasswords( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sirve para saber si las contraseñas de la Agrupación de Usuarios están encriptadas. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF gpwEncryptedPasswords() THEN
	GPU_UserPWD.EncryptedPassword = gpwEncryptPassword(sLogin, sPassword)
ELSE
	GPU_User.Password = sPassword
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwEncryptedPasswords()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las contraseñas de el groupware usuario están cifradas,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

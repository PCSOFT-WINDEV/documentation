
## gpwModifyUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un usuario en la base de datos de groupware usuario.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Values of new user
clUser is gpwUser
clUser.Email = EDT_Email
clUser.Login = EDT_Login
clUser.Password = EDT_Password
clUser.LastName = EDT_LastName
clUser.FirstName = EDT_FirstName
clUser.PhoneNumber = EDT_Phone

// Modify the user
IF gpwModifyUser(clUser) = False
	Info("Unable to modify the user." + CR + HErrorInfo())
	RETURN 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwModifyUser(<User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<User>: (Variable de tipo gpwUser)`**

Nombre de la [gpwUser](../WDLang6/1000021088.md) Variable que contiene la información a modificar.

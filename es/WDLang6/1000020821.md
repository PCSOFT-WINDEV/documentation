
## gpwOpenSite (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite iniciar el sitio cuando se utiliza el Groupware Usuario. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the site
gpwOpenSite()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gpwOpenSite()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició el sitio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Componente:
wd280gpu.dll


## LockAppTimeout( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el tiempo de inactividad antes de que se bloquee una aplicación WINDEV.. 

**Observación**: El tiempo de inactividad corresponde al tiempo transcurrido desde la última acción realizada por el usuario en la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Lock the application if it is not used during 15 minutes 
LockAppTimeout(15)

// Application does not lock anymore
LockAppTimeout(-1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir el tiempo de inactividad

`<Result> = LockAppTimeout()`
---

**`<Result>: (Integro)`**



- Tiempo de inactividad (expresado en minutos) antes de bloquear la aplicación.

- 0 (valor de Default) significa que la función "Bloqueo de la aplicación" no estaba activada.  





<a name="SYNTAX2"></a>

### Modificando el tiempo de inactividad

`LockAppTimeout(<New timeout>)`
---

**`<New timeout>: (Entero o Duración)`**



- Tiempo de inactividad antes de bloquear la aplicación (expresado en minutos). 

- Si este valor es set a 0, la función está desactivada (la aplicación nunca se bloqueará). 


Este parámetro puede corresponder a: 

- un número entero correspondiente al número de minutos,

- una variable de tipo Duration,

- la indicación directa de la duración (1 minuto por ejemplo).




## apExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un plan de acción. 

**Atención**: Esta función sólo se puede ejecutar: 

- del código de un plan de acción. 

- de un WLanguage Procedure que se encuentra en un proyecto de "Plan de Acción" (o configuración). 




**Observación**: Esta función es una función utilizada para manipular, el editor de WINDEV, WEBDEV o WINDEV Mobile.. Por lo tanto, está disponible en los tres productos. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Run an action plan that expects 2 parameters: 
// - the runtime directory of WINDEV
// - the WINDEV version to use
apExecute(PLAN_MyPlan, "D:\WINDEV23", "23")

// Other possibility: specify the parameters before running the function
PLAN_MyPlan.DirWD  = "D:\WINDEV23"
PLAN_MyPlan.Version = "23"
apExecute(PLAN_MyPlan)
```
<a name="Example2"></a>

```wl
// This example explains how to pass tow information to the same parameter. 
// For example, we want to open a project in the SCM. 
// For the parameters of this action plan, we must specify: 
// 1 - The SCM database used, 
// 2 - The project in the SCM. 
// The CR that will be used to separate the information in the parameter.
apExecute(PLAN_MyPlan, "CS:\\HFServer\MySCMDatabase" + CR + "SCM:\WINDEV projects\MySCMProject")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`apExecute(<Action plan> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Action plan>: (Cadena de caracteres)`**

Nombre del plan de acción a ejecutar. Este plan de acción se ejecuta íntegramente antes de la siguiente declaración. 
En caso de error: 

- La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.

- La ejecución del plan de acción se detiene por error. 

- Se ejecuta el código "Si se produce un error" del plan de acción de llamada.




**`<Parameter 1>: (Tipo de parámetro)`**

Primer parámetro que se pasará al plan de acción. 



**`<Parameter N>: (Tipo de parámetro)`**

Parámetro N que se pasará al plan de acción.

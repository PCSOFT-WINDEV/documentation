


## gpwOpenConfiguration( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana o página de configuración de User Groupware.

Esta función se tiene en cuenta únicamente si el usuario actual es un **supervisor** y si está conectado al Groupware Usuario (con la función [gpwLogin](../WDLang6/3041001.md) o desde la página abierta con la función [gpwOpen](../WDLang6/3041002.md)).

**Observación**: Esta función está disponible independientemente del modo de integración de groupware usuario en el sitio ("groupware usuario automático" o "groupware usuario personalizado" en la pestaña "Integración" de la ventana description de groupware usuario).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open User Groupware configuration page 
gpwOpenConfiguration()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gpwOpenConfiguration()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió la página de configuración,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

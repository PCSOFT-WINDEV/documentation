


## ScanCapability (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Determina si un escáner tiene una funcionalidad determinada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyScanner is Scanner

// Indicates whether the scanner supports duplex scanning
Duplex is boolean = ScanCapability(MyScanner, scanCapabilityDuplex)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScanCapability(<Scanner> , <Capacity>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el escáner tiene la funcionalidad &lt;Capability&gt;, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.




**`<Scanner>: (Variable de tipo Scanner)`**

Nombre de la variable de tipo [Scanner](../WDLang6/1410087808.md) inicializada con la información del escáner del que se va a determinar la funcionalidad.

**`<Capacity>: (Constante de tipo Integer)`**

Información sobre la funcionalidad del escáner:  



|   |   |
| --- | --- |
| *scanCapabilityDuplex* | Determina si el escáner admite el escaneo a doble cara. |
| *scanCapabilityFeeder* | Determina si el escáner tiene un alimentador. |
| *scanCapabilityFlatbed* | Determina si el escáner tiene un cristal. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
También es posible obtener las funcionalidades de un escáner con la función [ScanProperty](../WDLang6/1410087853.md). Esta función devuelve una combinación de constantes que corresponden a las funcionalidades del escáner. La función **ScanCapability** simplifica el código que indica si el escáner tiene una funcionalidad determinada.




## ScanToImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un documento de un escáner en una Variable de tipo [Imagen](../WDLang1/1000019650.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyScanner is Scanner 
MyScanner = ScanSelect()

// Display the document in the scanner in an Image control
IMG_MyImage = ScanToImage(MyScanner)
IF ErrorOccurred THEN
	Error("The document was not scanned", ErrorInfo())
END

// Scan all pages in the feeder
ScanToImage(MyScanner, ProcessPageScan)
// Process a scanned page
INTERNAL PROCEDURE ProcessPageScan(nPageNum is int, ImageScan is Image)
	dSaveImageJPEG(ImageScan, "page" + nPageNum + ".jpg")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Escanear una sola página

`<Result> = ScanToImage(<Scanner>)`
---

**`<Result>: (Imagen Variable)`**

[Imagen](../WDLang1/1000019650.md) Variable recuperado del Scanner. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.

**`<Scanner>: (Variable de tipo Scanner)`**

Nombre del [Escanear](../WDLang6/1410087808.md) Variable inicializado con los datos del Scanner que se va a utilizar.


<a name="SYNTAX2"></a>

### Escaneo de varias páginas (con alimentador)

`<Result> = ScanToImage(<Scanner> [, <Number of pages>] , <End of page scan>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la exploración se ha completado con éxito,

- <u><u><u><u>False</u></u></u></u> si se produce un error durante la exploración Process. La función [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.




**`<Scanner>: (Variable de tipo Scanner)`**

Nombre del [escanear](../WDLang6/1410087808.md) Variable inicializado con los datos del Scanner que se va a utilizar.

**`<Number of pages>: (Entero opcional)`**



- Número de páginas a escanear,

- Constante *scanNumberPagesAll* para escanear todas las páginas del alimentador (valor predeterminado).




**`<End of page scan>: (Nombre de Procedure)`**

Nombre de la WLanguage Procedure ("callback") llamada una vez que se ha escaneado una página. Este procedimiento tiene el siguiente formato: 

```txt
<Procedure name>(<Page number>, <Image>)
```
donde: 

- &lt;Page number&gt;: Número de la página escaneada. 

- &lt;Image&gt;: [Imagen](../WDLang1/1000019650.md) Variable que contiene la página escaneada. Esta variable sólo es válida cuando se llama a la Procedure: por lo tanto, debe ser copiado o guardado en un archivo (no debe ser referenciado).


Si el procedimiento devuelve <u><u><u><u>False</u></u></u></u>, el escaneo se interrumpe.




## <PDF reader>AddWatermark (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Watermark a todas las páginas de un archivo PDF en un lector de PDF control.

Observación: Esta función puede utilizarse varias veces en la misma Document para añadir diferentes marcas de agua. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWatermark is Watermark
MyWatermark.Text = "Confidential"
MyWatermark.Font.Name = "Arial"
MyWatermark.Position = iCenterH + iBottom

MyPDF is pdfDocument = fExeDir() + [fSep()] + "RPT_NoName1.pdf"

// Add watermark to PDF Reader control
PDF_MyReader.AddWatermark(MyWatermark)
```

<a name="XSYNTAX"></a>

`<PDF Reader control>.AddWatermark(<Watermark>)`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular

**`<Watermark>: (Watermark Variable)`**

Nombre de la variable [Watermark](../WDLang5/1000019922.md) que corresponde al Watermark que se añadirá en todas las páginas del PDF Document.

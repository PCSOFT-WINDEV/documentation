## Presentación
<a name="presentacion_ELTTEXTE000292"></a>
WINDEV es un entorno completo de desarrollo integrado de DevOps que permite desarrollar aplicaciones Windows (y Linux) en diferentes campos: comercial, industrial, médico, entre otros, ... Las aplicaciones pueden acceder a información almacenada en las bases de datos.

A lo largo de este tutorial, aprenderá a crear aplicaciones (con o sin base de datos) y a enriquecerlas utilizando las diferentes funcionalidades de WINDEV.

<a name="NOTE3_1"></a>


## Iniciar WINDEV
<a name="iniciar_windev_ELTTEXTE000316"></a>


- Abra WINDEV 28.

	- **Si no ha iniciado WINDEV 28 antes**, aparecerá un asistente de bienvenida:

		- Si ha trabajado con una versión anterior de WINDEV, este asistente le permitirá recuperar las configuraciones existentes.

		- Si es un nuevo usuario, el asistente le ayudará a configurar el entorno. Podrá elegir la configuración de pantalla y configurar los Centros de Control. 




- **Si ya ha iniciado WINDEV 28**, inicie sesión, si es necesario. El entorno de desarrollo se abre. La página de inicio aparece en pantalla. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bienvenue_WD_defaut.jpg&type=thumb)

- Veamos más de cerca el entorno de desarrollo de WINDEV:

	- En la sección "Part 1 - Lesson 1.1. Discover WINDEV", haga doble clic en "Full application - Answers".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bienvenue%202%20WD%20-%20HC%20N%B0005.jpg&type=thumb)

## Entorno de desarrollo
<a name="entorno_desarrollo_ELTTEXTE000340"></a>


### Editor
<a name="editor_ELTPARAGRAPHE000072"></a>

El entorno de desarrollo de WINDEV incluye una interfaz de usuario (UI) específica y varios editores que permiten crear los diferentes elementos de las aplicaciones.

Por ejemplo, el editor de ventanas permite crear ventanas, el editor de reportes permite crear reportes, etc.

- Para explorar WINDEV, abra la ventana "Product form": 

	1. Presione Ctrl + E. 

	2. En la ventana que aparece, introduzca el nombre de la ventana que desea abrir: WIN_Product_form. 

	3. Valide. La ventana aparece en el editor. 







Todos los editores utilizan el mismo entorno: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WD%20-%20HC%20N%B0002.jpg&type=thumb)


1. **Menú de los editores** que se muestra como una cinta de opciones (veremos cómo se utiliza en el siguiente párrafo).

2. **Editor actual** (en este caso, editor de ventanas). El editor muestra el elemento que se está creando o modificando en modo WYSIWYG (What You See Is What You Get).

3. **Paneles**. La UI de WINDEV incluye varios paneles horizontales y verticales que le permiten acceder rápidamente a diferentes tipos de información. Por ejemplo: 

	- el panel "Explorador de proyectos" (a la izquierda) muestra una lista de todos los elementos del proyecto por categoría.

	- el panel de búsqueda (en la parte inferior) permite realizar búsquedas en todo el proyecto y sus elementos.


Los paneles se pueden ocultar pulsando Ctrl + W.

4. **Pestañas de documentos abiertos**. Estas pestañas permiten ver rápidamente todos los elementos abiertos. Al hacer clic en un elemento, este se muestra en el editor correspondiente.

### Cinta de opciones en detalle
<a name="cinta_opciones_detalle_ELTPARAGRAPHE000100"></a>

Las opciones de WINDEV están disponibles en la cinta de opciones. La cinta contiene pestañas en las que se agrupan las diferentes opciones de los editores.

**Elementos de la cinta de opciones**
La cinta de opciones incluye tres áreas:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WD%20-%20HC%20N%B0003.jpg&type=thumb)


- el área de botones, a la izquierda (1).

- el área de pestañas, en la parte superior (2).

- el área de opciones (3).


Veamos estas áreas en detalle. 

**Área de botones (1)**


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WD%20-%20HC%20N%B0003%201.jpg)<br> | El área de botones agrupa los **botones de acceso rápido**. Estos botones permiten realizar las operaciones más frecuentes, comunes a todos los editores: guardar, abrir, crear, etc.<br><br>Puede abrir la ventana "Acerca de" desde el logotipo del IDE.  |

**Área de pestañas (2)**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WD%20-%20HC%20N%B0003%202.jpg)
Las diferentes pestañas de la cinta permiten acceder a las opciones de los editores del proyecto actual. Existen varios tipos de pestañas:

- la pestaña actual: El texto de la pestaña se muestra en negrita y está subrayado por una línea amarilla.

- las pestañas contextuales, específicas del elemento actual: El nombre de la pestaña se muestra en amarillo (o azul en el tema claro).

- las pestañas disponibles: El nombre de la pestaña se muestra en blanco (o en negro en el tema claro).




**Área de opciones (3)**<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Environnement%20WD%20-%20HC%20N%B0003%203.jpg&type=thumb)
Las opciones de la cinta cambian dependiendo de la pestaña seleccionada. Existen varios tipos de opciones:

- Opciones a seleccionar.

- Botones para hacer clic.

- Botones desplegables que muestran otras opciones. Existen dos tipos de botones desplegables:

	- botones desplegables para desplegar un menú

	- botones divididos que permiten desplegar un menú (haciendo clic en la flecha), o realizar una acción predeterminada (haciendo clic en el ícono del botón).





Las opciones están organizadas por grupos. Cada grupo de opciones tiene un nombre y un iniciador de cuadro de diálogo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Regroupement_GAF.jpg). Este botón permite realizar una acción específica según el grupo actual: abrir la descripción del elemento actual, abrir la ayuda, etc, ... 

En este tutorial, para hacer referencia a una opción de menú, utilizaremos los términos "pestañas" y "grupos". Por ejemplo:

Para abrir la ayuda, vaya a la pestaña "Inicio", grupo "Ayuda en línea" y haga clic en "Ayuda".

### Opciones del entorno
<a name="opciones_del_entorno_ELTPARAGRAPHE000166"></a>

Puede cambiar varias opciones del entorno: 

- Tema: De forma predeterminada, el entorno utiliza un tema oscuro. Sin embargo, hay varios temas disponibles: 

	- Tema claro, cinta de opciones gris. En este modo, la barra de menú no tiene colores: está en gris. 

	- Tema gris. En este modo, el entorno y las ventanas de la interfaz se muestran sobre un fondo gris claro. 

	- Tema oscuro. En este modo, el entorno y las ventanas de la interfaz se muestran sobre un fondo negro o gris oscuro. 




- Idioma: De forma predeterminada, el entorno se muestra en el idioma elegido durante la instalación. El entorno puede mostrarse en español, inglés o francés.

- Panel: Algunos paneles se muestran de forma predeterminada en el entorno ("Explorador de proyectos", etc.). Puede elegir qué paneles se muestran y fijar su posición.  




Todas estas opciones se encuentran en las opciones del entorno. Para cambiar estas opciones, vaya a la pestaña "Inicio", grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV". Puede cambiar varias opciones del entorno en las diferentes pestañas de la ventana que aparece. 
Tenga en cuenta que para cambiar ciertas opciones puede ser necesario reiniciar WINDEV.

## Preguntas/Respuestas
<a name="preguntasrespuestas_ELTTEXTE000258"></a>


### ¿Cómo ver el elemento al que pertenece el código WLanguage que aparece en el editor de código?
<a name="¿como_ver_elemento_que_pertenece_codigo_wlanguage_que_aparece_editor_codigo_ELTPARAGRAPHE000025"></a>

Para ver el elemento que corresponde al código WLanguage actual, en la pestaña "Código", en el grupo "Navegación", haga clic en "Ir al elemento" (Ctrl + Alt + F2). Enseguida, aparece la ventana que contiene el elemento deseado.
<a name="NOTE2_2"></a>


### ¿Cómo imprimir el código WLanguage que se muestra?
<a name="¿como_imprimir_codigo_wlanguage_que_muestra_ELTPARAGRAPHE000046"></a>

Para imprimir código WLanguage directamente, haga clic en el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Impression_GAF.jpg) en la pestaña "Inicio" o presione Ctrl + P.
<a name="NOTE2_3"></a>


### ¿Cómo "Buscar y/o reemplazar"?
<a name="¿como_buscar_reemplazar_ELTPARAGRAPHE000057"></a>

Para buscar y reemplazar (en el código, interfaz, etc.), vaya a la pestaña "Inicio", grupo "Buscar" o al panel "Buscar - Reemplazar" (Ctrl + F): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Rechercher%20Remplacer%20-%20HC%20N%B01.jpg&type=thumb)


A continuación, puede definir las diferentes características de la búsqueda.
<a name="NOTE2_4"></a>


### ¿Qué significan los signos "+" y "-" en el editor de código?
<a name="¿que_significan_los_signos_editor_codigo_ELTPARAGRAPHE000077"></a>

El editor de código permite expandir o contraer el código WLanguage. Esta característica es útil si los procesos utilizan muchas instrucciones estructuradas (bucles, condiciones, navegaciones, etc.). 

Para contraer un conjunto de líneas de código, en la pestaña "Contraer", en el grupo "Código contraíble ", despliegue "Vista" y seleccione la opción "Contraer todo" (o presione las teclas Ctrl + Shift + \*). 

Solo los comentarios permanecen visibles. Al pasar el cursor sobre un comentario, aparece un globo de ayuda (tooltip) con el código WLanguage asociado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_QR%20-%20Code%20r%E9pli%E9%20-%20HC%20N%B01.jpg)
Presione Ctrl + \* para expandir todo el código WLanguage. Haga clic en los símbolos "-" o "+" para contraer o expandir la sección de código correspondiente.
<a name="NOTE2_5"></a>


### ¿Cómo puedo saber quién escribió una línea de código?
<a name="¿como_puedo_saber_quien_escribio_una_linea_codigo_ELTPARAGRAPHE000112"></a>

Presione F6 para ver la información de cada línea de código (nombre y fecha de creación/modificación).
<a name="NOTE2_6"></a>


### ¿Es posible obtener el número de una línea de código?
<a name="¿es_posible_obtener_numero_una_linea_codigo_ELTPARAGRAPHE000121"></a>

En el editor de código, para mostrar u ocultar los números de las líneas de código, en la pestaña "Vista", en el grupo "Ayuda edición", haga clic en "Mostrar números de línea" (o presione: Ctrl + Shift + G). 
<a name="NOTE2_7"></a>


### ¿Cómo se puede obtener fácilmente la sintaxis o ayuda de una función WLanguage?
<a name="¿como_puede_obtener_facilmente_sintaxis_ayuda_una_funcion_wlanguage_ELTPARAGRAPHE000142"></a>

Al introducir una función WLanguage en el editor de código:

- el nombre de cada parámetro se muestra en un tooltip en la línea de código que se está introduciendo. Pase el cursor por encima del nombre del parámetro actual para ver los detalles.
	Si hay varias sintaxis disponibles, es posible cambiar de una sintaxis a otra utilizando Ctrl + flecha arriba o flecha abajo.

- la sintaxis de la función WLanguage se muestra en la barra de estado del editor.




En la ayuda que se muestra, los parámetros entre [ y ] son opcionales.

Para las funciones WLanguage que utilizan nombres de archivos de datos, controles, ventanas o reportes, la entrada predictiva también muestra la lista de los elementos del proyecto que corresponden al parámetro de la función que se está introduciendo.

Cada función y propiedad WLanguage está asociada a una página de ayuda. Puede abrir directamente las páginas de ayuda desde el editor presionando F1 en el nombre de la función o propiedad deseada.
<a name="NOTE2_8"></a>


### ¿Cuáles son los métodos abreviados de teclado útiles en el editor de código? 
<a name="¿cuales_son_los_metodos_abreviados_teclado_utiles_editor_codigo_ELTPARAGRAPHE000168"></a>



- Para ver los eventos de un control, método, clase, procedimiento o bloque de reporte, presione F2 en el nombre del objeto deseado.

- Ctrl + F2 permite volver al proceso o evento inicial. 
	Para pasar de un proceso/evento a otro, presione F2 repetidamente. Para volver al proceso o evento inicial, presione Ctrl + F2 el mismo número de veces. 

- Ctrl + L borra la línea actual.

- Ctrl + D duplica la línea actual (o las líneas seleccionadas) en la línea de abajo.

- Tab y Shift + Tab permiten aumentar y disminuir la sangría de las líneas seleccionadas.

- "Ctrl + /" convierte las líneas seleccionadas en comentarios y "Ctrl + Shift + /" elimina los comentarios (Atención: tecla "/" del teclado numérico).

- Ctrl + R vuelve a ajustar la sangría del código mostrado automáticamente.

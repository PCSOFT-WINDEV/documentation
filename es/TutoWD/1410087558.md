## Crear un proyecto
<a name="crear_proyecto_ELTTEXTE000289"></a>


- Para crear el proyecto:

	1. Inicie WINDEV. 

	2. Vaya a la página de inicio de WINDEV (Ctrl + &lt;). 

	3. En la página de inicio, haga clic en "Crear un proyecto" y seleccione "Aplicación Windows o Linux". El asistente de creación de proyectos se abre. Las diferentes etapas del asistente le ayudarán a crear el proyecto. La información especificada en el asistente se podrá modificar más adelante.
			|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=astuce.png) | Consejo | Consejo: También puede crear un proyecto con la siguiente opción:<br><br>		1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Cr%E9er1_GAF.jpg) en los botones de acceso rápido.<br><br>		2. La ventana de creación de nuevos elementos se abre: haga clic en "Proyecto".<br><br><br> |





	4. La primera etapa del asistente permite seleccionar el sistema operativo para el que está destinado el proyecto. Seleccione "Plataforma Windows" y vaya a la etapa siguiente del asistente (botón "Siguiente").  

	5. En esta lección, vamos a crear un proyecto "En blanco". Seleccione "Crear un proyecto en blanco" y vaya a la siguiente etapa del asistente. 

	6. WINDEV le pedirá que introduzca la descripción de la plataforma. Conserve las opciones predeterminadas ("Ejecutable Windows de 64 bits" y "Ejecutable con ventanas"). Pase a la etapa siguiente del asistente.

	7. El asistente le pide que introduzca el nombre, la ubicación y la descripción del proyecto. En nuestro caso, este proyecto se llamará "sqldatabase".  

	8. Las diferentes etapas del asistente aparecen en el menú de la izquierda. Puede hacer clic en la etapa que desee. Dado que las demás etapas de la "Descripción" no son estrictamente necesarias, puede hacer clic directamente en "Normas". 

	9. Esta etapa permite definir el estilo de código. No cambie las opciones sugeridas. Pase a la etapa siguiente. 

	10. Esta etapa permite definir el tema gráfico. Seleccione "Elegant". Pase a la etapa siguiente.

	11. Haga clic en la etapa "Base de datos". Esta etapa le permite definir una base de datos.

	12. Seleccione "Sí, utilizar una base de datos existente". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Import%20OLEDB%20-%20HC%20N%B0001.jpg&type=thumb)
Pase a la etapa siguiente.

	13. Seleccione la opción "Importar la descripción desde una base de datos existente". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Import%20OLEDB%20-%20HC%20N%B0002.jpg&type=thumb)


	14. El asistente de creación de análisis se abre.

## Crear un análisis
<a name="crear_analisis_ELTTEXTE000331"></a>


- Las etapas para crear el análisis son las siguientes:

	1. Indique el nombre y el directorio del análisis. De forma predeterminada, el nombre del análisis corresponde al nombre del proyecto y el directorio del análisis es un directorio ".ana" en el directorio del proyecto. Conserve los parámetros predeterminados. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Import%20OLEDB%20-%20HC%20N%B0003.jpg&type=thumb)
Pase a la etapa siguiente del asistente. 

	2. A continuación, puede elegir los tipos de bases de datos que utiliza el proyecto. Seleccione OLE DB (o la base de datos SQL utilizada si tiene el Conector Nativo correspondiente). 
			Observación: Los Conectores Nativos se muestran primero. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Import%20OLEDB%20-%20HC%20N%B0004.jpg&type=thumb)
Pase a la etapa siguiente del asistente.

	3. Seleccione la fuente OLE DB que corresponde al tipo de su base de datos.
			Pase a la etapa siguiente del asistente.

	4. En las siguientes etapas, especifique los parámetros de la conexión OLE DB (las etapas son las mismas si eligió un Conector Nativo):

		- el nombre del servidor de la base de datos. 

		- el nombre de usuario y la contraseña. 

		- el nombre de la base de datos si es necesario. 

		- el nombre de la conexión (el título asociado a la conexión es opcional). 


 Pase a la etapa siguiente.

5. Indique el modo de acceso a los datos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Import%20OLEDB%20-%20HC%20N%B0007.jpg&type=thumb)
Es posible:

	- acceder directamente a los datos en su formato actual. Solo la descripción de las tablas se importa al análisis. No se requiere ninguna conversión.

	- convertir los datos a HFSQL Classic.

## Generación del análisis
<a name="generacion_del_analisis_ELTTEXTE000355"></a>
La generación del análisis consiste en validar los cambios hechos en dicho análisis (crear archivos de datos, agregar o eliminar campos, etc.) y aplicarlos a todo el proyecto (páginas, controles enlazados, reportes, etc.).

Si se han realizado cambios en el análisis, aparece un cuadro de diálogo para generar el análisis cuando se cierra el editor.

También puede generar el análisis manualmente. Veamos cómo realizar esta acción. 

- Para generar el análisis:

	1. En editor de análisis, en la pestaña "Análisis", en el grupo "Análisis", haga clic en "Generación".

	2. La generación del análisis se inicia automáticamente.







Cuando se genera el análisis, los datos pueden manipularse como datos HFSQL. No hay ninguna diferencia. Es posible:

- Crear una aplicación completa utilizando el RAD (Rapid Application Development). 

- Crear una aplicación completa y personalizada. Para obtener más información, consulte ["Parte 4"](../TutoWD/1410087532.md).

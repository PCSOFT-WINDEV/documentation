**¡Ha completado el tutorial!**

Este tutorial cubrió una serie de temas, pero no todas las funcionalidades de WINDEV, ¡faltan muchas por descubrir!

Ahora está familiarizado con los principales conceptos.



También puede consultar los ejemplos proporcionados con WINDEV: algunos son simples y solo abordan un tema, mientras que otros pueden ser más complejos. Estos ejemplos le mostrarán las diferentes facetas de WINDEV. El código fuente también permite profundizar en el aprendizaje.

Aún quedan muchos temas por descubrir, ¡cientos, o incluso miles!. 

WINDEV ofrece un gran número de funcionalidades que no se abordaron en este tutorial:

- funciones HTTP

- controles de ofimática (Editor de imágenes, Procesador de texto, Hoja de cálculo, etc.)

- creación de plantillas

- reportes anidados, consultas con parámetros

- compilación dinámica, llamadas a las DLL y más...




Para más información, consulte la ayuda en línea.

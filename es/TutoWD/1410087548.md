## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
Esta parte presenta varias funcionalidades avanzadas. No es esencial para su conocimiento de WINDEV, pero le permitirá descubrir algunas funcionalidades avanzadas. 

Las diferentes lecciones de esta parte se basan en ejemplos proporcionados con WINDEV.

WINDEV ofrece diferentes tipos de ejemplos: 

- **ejemplos completos**: estos ejemplos corresponden a una aplicación completa que utiliza una o varias funcionalidades.

- **ejemplos didácticos**: estos ejemplos corresponden a una aplicación pequeña que permite descubrir una funcionalidad.

- **ejemplos unitarios**: estos ejemplos corresponden a una ventana que permite probar rápidamente el uso de una función.



<a name="NOTE2_2"></a>


### Ejemplo práctico
<a name="ejemplo_practico_ELTPARAGRAPHE000037"></a>

Para simplificar la presentación de las diferentes funcionalidades, utilizaremos ejemplos unitarios o didácticos proporcionados con WINDEV.

- Para abrir un ejemplo completo, didáctico o unitario desde la página principal de WINDEV:

	1. Vaya a la página de inicio (Ctrl + &lt;). 

	2. En la página de inicio, haga clic en "Abrir un ejemplo". La lista de ejemplos completos, didácticos y unitarios proporcionados con WINDEV aparece. Estos ejemplos están agrupados por tipo de ejemplo (completo, didáctico, etc.). 

	3. Puede introducir una palabra clave en la barra de búsqueda (por ejemplo, "Errors"). Solo se muestran los ejemplos que contienen esta palabra. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P7_Acc%E8s%20exemples%20-%20HC%20N%B0001.jpg&type=thumb)
Los íconos a la derecha del nombre del ejemplo indican la plataforma para la que se creó el ejemplo. 
			Observación: Los botones a la derecha de la barra de búsqueda permiten filtrar los ejemplos según la(s) plataforma(s) deseada(s). 

	4. Para abrir un ejemplo, haga doble clic en el nombre del ejemplo. 

	5. Para los ejemplos completos y didácticos, el proyecto actual se cierra automáticamente y luego se abre el proyecto de ejemplo. 

	6. Para los ejemplos unitarios, la ventana correspondiente se abre en el proyecto actual. 







**Observación**: Si modifica un ejemplo completo o didáctico, la próxima vez que lo abra, WINDEV le preguntará automáticamente con qué versión del ejemplo desea trabajar: 

- la versión modificada. En este caso, encontrará todos sus cambios. 

- la versión original del ejemplo. En este caso, se elimina la versión modificada y se obtienen todas las funciones del ejemplo original.

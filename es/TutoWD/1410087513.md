## Crear una ventana: introducir y mostrar de datos
<a name="crear_una_ventana_introducir_mostrar_datos_ELTTEXTE000589"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000060"></a>

Crearemos la siguiente ventana: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_WD%20Ma%20premi%E8re%20fen%EAtre%20-%20Go%20-%20HC%20N%B01.jpg)
Esta ventana permite que el usuario introduzca su nombre y muestra un mensaje de bienvenida con el botón "Display".

Esta acción puede parecer demasiado básica, pero le recomendamos crear esta ventana. Le sorprenderá lo intuitivo y fácil que es el editor de WINDEV. Esta ventana le permitirá descubrir conceptos fundamentales para el resto del tutorial.
<a name="NOTE3_2"></a>


### Crear una ventana
<a name="crear_una_ventana_ELTPARAGRAPHE000071"></a>

- Para crear la ventana:

	1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Creer1_GAF.jpg) en los botones de acceso rápido del menú de WINDEV:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_Ma%20premi%E8re%20fen%EAtre%20-%20cr%E9ation%20fen%EAtre%20-%20HC%20N%B003.jpg)


	2. La ventana de creación de nuevos elementos se abre. Esta ventana permite crear los elementos del proyecto.

3. Haga clic en "Ventana", y luego en "Ventana". El asistente de creación de ventanas se abre.

4. Seleccione "En blanco" en la lista de ventanas "estándar" que se muestran a la izquierda.   <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_WD%20Ma%20premi%E8re%20fen%EAtre%20-%20cr%E9ation%20fen%EAtre%20-%20HC%20N%B001.jpg)

			

5. Valide el asistente de creación de ventanas (botón "OK"). La ventana se crea automáticamente en el editor. La ventana para guardar el elemento se abre. Esta ventana permite especificar: 

	- El título del elemento. El título se mostrará en la barra de título de la ventana. 

	- El nombre del elemento. El nombre corresponde al nombre lógico del elemento. Este nombre se utilizará para manipular el elemento mediante programación. 

	- La ubicación del elemento. La ubicación corresponde a la carpeta en la que se guarda el elemento.

### Introducir y visualizar un valor
<a name="introducir_visualizar_valor_ELTPARAGRAPHE000121"></a>

Para introducir y visualizar un valor, debe crear:

- un control en el que los usuarios introducirán el nombre. Este tipo de control es un "control Campo de entrada".

- un control Botón para que se muestre el nombre introducido.




- Para crear el control Campo de entrada:

	1. En la pestaña "Creación", en el grupo "Controles frecuentes", despliegue "Entrada" (haga clic en la flecha): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_WD%20Popup%20nouveau%20champ.jpg)
Enseguida, aparece una lista de controles Campo de entrada predefinidos.
			|   |   |   |
| --- | --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=note.png) | Nota | Esta lista muestra diferentes controles Campo de entrada que se pueden utilizar directamente. Estos controles también están disponibles en los controles Smart. Los diferentes controles se pueden probar directamente: haga clic en el ícono "GO" que aparece al pasar el cursor sobre el control. |





	2. En el campo de búsqueda que aparece en la parte superior de la lista de controles predefinidos, introduzca "Entrada de texto simple". El control correspondiente aparece en la lista. 

	3. Haga clic en el control "Entrada de texto simple". El control seguirá el cursor del ratón.

	4. Mueva el ratón a la ubicación en la que se va a crear el control (por ejemplo, en la parte superior de la ventana). Para soltar el control en la ventana, haga clic de nuevo.

	5. Haga clic derecho en el control que acaba de crear. El menú contextual del control se abre. Seleccione la opción "Descripción" en el menú contextual. La ventana de descripción del control Campo de entrada se abre inmediatamente.

	6. Defina la siguiente información del control:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_WD%20D%E9tails%20champ%20de%20saisie.jpg&type=thumb)


		- El nombre del control: "EDT_FirstName" (1).

- El título del control: "First name" (2).

- El tipo de control: Texto (3).

7. Valide la ventana de descripción del control (botón "OK"). El control aparece en el editor de ventanas.




El título del control se trunca en el editor. 



- Para que el control se muestre correctamente:

	1. Seleccione el control (haca clic él).

	2. Haga clic derecho para abrir el menú contextual y seleccione la opción "Adaptar tamaño".

	3. El control se redimensiona inmediatamente y muestra todo el texto.


 **Observación**: También puede redimensionar manualmente el área de entrada del control:

1. Seleccione el control.

2. Ubique el cursor en uno de los controladores de tamaño de la derecha del control (cuadrado pequeño). El cursor se convierte en una flecha.

3. Mantenga pulsado el botón izquierdo del ratón para cambiar el tamaño del control.

4. Suelte el botón del ratón una vez que haya cambiado el tamaño del control.




- Para crear el control Botón "Display":

	1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Bouton_GAF.jpg).

	2. Ubique el control en la ventana. Haga clic en la ubicación en la que desea crear el control Botón (por ejemplo, a la derecha del control Campo de entrada).

	3. Haga clic en el control que acaba de crear. El texto que se muestra en el control pasa al modo de edición.
			|   |   |   |

	4. Introduzca el texto "Display". El nombre del control Botón cambia automáticamente a "BTN_Display".

	5. El nombre del control se muestra:

		- en el tooltip que aparece al pasar el cursor por encima del control:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P1_WD%20Nom%20bouton%20au%20survol.jpg)

- Vamos a mostrar el texto escrito en un cuadro de diálogo (una mini ventana propuesta por el sistema). Para ello, utilizaremos nuestra primera función WLanguage: [Info](../WDLang1/3021011.md).

	2. Haga clic derecho para abrir el menú contextual del control.

	3. Seleccione "Código". Esta opción abre el editor de código de WINDEV. Este editor permite escribir todas las instrucciones WLanguage.

## Conclusión
<a name="conclusion_ELTTEXTE000865"></a>
WINDEV permite crear una aplicación fácilmente. ¿Pero qué es una **Aplicación**?

Una **aplicación** es una herramienta que permite realizar tareas y acciones automáticamente. Una aplicación contiene un ejecutable (o un conjunto de programas ejecutables), bibliotecas, archivos de datos, etc. 

Un **ejecutable** es un archivo compuesto por elementos que el usuario puede manipular directamente (ventanas, vistas de reportes, etc.). Es lo que inicia el usuario final de una aplicación.

Para crear un ejecutable, WINDEV propone crear un **proyecto**. Un proyecto vincula y organiza los diferentes elementos del programa. El ejecutable se creará a partir del proyecto.

Si la aplicación utiliza datos, WINDEV le permite definir la estructura de la base de datos mediante el **análisis**. El análisis de WINDEV contiene la estructura de los archivos de datos (también conocidos como "Tablas" en otras bases de datos). Estos archivos de datos contendrán los datos de la aplicación. Vamos a ver cómo crear un análisis en la [parte 3](../TutoWD/1410087520.md) de este tutorial.

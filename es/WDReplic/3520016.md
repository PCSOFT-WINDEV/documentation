## Presentación
<a name="presentacion_ELTTEXTE000208"></a>


### WDReplic
<a name="wdreplic_ELTPARAGRAPHE000011"></a>

WDReplic es una herramienta utilizada para gestionar la replicación de datos entre varios ordenadores que utilizan la misma aplicación.

**Recordatorio**: La replicación es la operación que permite mantener actualizadas bases de datos remotas de estructuras idénticas en las que se realizan varias operaciones. Las operaciones realizadas en cada una de las bases de datos se aplican a todas las demás bases de datos.

**Atención**: Para utilizar WDReplic, la base de datos de la aplicación debe estar configurada para soportar la replicación. Contact su proveedor de aplicaciones para más detalles (o vea la ayuda en línea de WINDEV).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Características de WDReplic
<a name="caracteristicas_wdreplic_ELTTEXTE000232"></a>
WDReplic le ofrece la posibilidad de gestionar completamente la replicación de datos.

WDReplic está acostumbrado a:

- Implementar una replicación.

- Ejecute una replicación.

- Programe una réplica.

- Edite una réplica.

## ¿Cuándo y cómo debo usar WDReplic?
<a name="¿cuando_como_debo_usar_wdreplic_ELTTEXTE000256"></a>
Se debe utilizar WDReplic:

- [Para configurar la replicación](../WDReplic/3520019.md).

- [Para ejecutar la replicación](../WDReplic/3520002.md).




**Atención**: El uso de WDReplic asume que la aplicación fue configurada para soportar la replicación (modificación del análisis requerido).
<a name="NOTE3_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000052"></a>

WDReplic puede ser redistribuido. WDReplic puede instalarse con las aplicaciones desarrolladas con WINDEV o WEBDEV. El acuerdo de licencia de WINDEV o WEBDEV se aplica en su totalidad.

Al crear el instalación Procedure, se puede añadir WDReplic a los módulos instalados con sus aplicaciones.

**Recordatorio**: El procedimiento de instalación se puede crear: 

- mediante el asistente. Para abrir el asistente, vaya a la pestaña "Proyecto", grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación". 

- desde el editor de instalación. Para abrir el editor de instalación, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

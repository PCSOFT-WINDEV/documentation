
## WDAdminHF: Gestión de los archivos de datos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
El administrador remoto HFSQL propone una gestión completa de las bases de datos HFSQL Client/Server. 

**Para mostrar las características de un archivo de datos HFSQL Client/Server**: 

1. Conectarse al servidor HFSQL solicitado a través del administrador remoto HFSQL. 

2. Seleccionar"Bases de datos". 

3. Seleccione la base de datos solicitada y haga clic en el enlace "Acceder a la base de datos seleccionada".. 

4. Las opciones mostradas son las opciones disponibles en la base de datos actual. Haga clic en "Archivos".. 

5. Haga clic en el nombre del archivo de datos para gestionar. Se muestran las opciones disponibles. 






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Detalles de las opciones disponibles
<a name="detalles_las_opciones_disponibles_ELTTEXTE000167"></a>


### Conexiones
<a name="conexiones_ELTPARAGRAPHE000027"></a>

En esta opción se enumeran los diferentes usuarios conectados al archivo de datos actual.

La lista de conexiones de Active contiene la siguiente información:

- el nombre de usuario, 

- El nombre el equipo,

- el nombre de la aplicación que accede a la base de datos, etc.




Usted tiene la capacidad de administrar las conexiones desde esta página: permitir o prohibir las conexiones, desconectar a los usuarios o enviar un mensaje a los usuarios.


### description
<a name="description_ELTPARAGRAPHE000040"></a>

Esta opción se utiliza para mostrar la description de los diferentes artículos en el archivo de datos (nombre del artículo, tamaño, tipo, etc.).


### Propiedades
<a name="propiedades_ELTPARAGRAPHE000045"></a>

Esta opción se utiliza para mostrar las propiedades de del archivo de datos (nombre físico, formato, protección, etc.)


### Buscar
<a name="buscar_ELTPARAGRAPHE000050"></a>

Esta opción se utiliza para realizar una búsqueda en el archivo de datos. Tiene la posibilidad de introducir las condiciones de búsqueda de forma explícita.


### consulta SQL
<a name="consulta_sql_ELTPARAGRAPHE000055"></a>

Esta opción se utiliza para realizar una búsqueda en el archivo de datos. Tiene la posibilidad de introducir directamente el código SQL de la consulta a ejecutar.


### Derechos
<a name="derechos_ELTPARAGRAPHE000060"></a>

Esta opción se utiliza para gestionar los derechos sobre el archivo de datos para el usuario o grupo seleccionado.


### Bloqueos
<a name="bloqueos_ELTPARAGRAPHE000065"></a>

Esta opción se utiliza para gestionar los bloqueos.

Se muestra la siguiente información:

- el número de la Record cerrada,

- el nombre del usuario que está bloqueando,

- el nombre del ordenador de cierre,

- la IP Address del ordenador de bloqueo,

- el nombre de la aplicación de cierre,

- la fecha de inicio del bloqueo,

- la duración del bloqueo.





### Registro
<a name="registro_ELTPARAGRAPHE000080"></a>

Esta opción se utiliza para ver el contenido de los registros y para realizar una búsqueda en los registros.

El archivo debe estar registrado.




### Mantenimiento
<a name="mantenimiento_ELTPARAGRAPHE000088"></a>

Esta opción se utiliza para liberar un Record o todas las transacciones.

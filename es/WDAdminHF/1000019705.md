
## WDAdminHF: Tarea programada
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000119"></a>
El administrador remoto de HFSQL se utiliza para definir las tareas programadas. Estas tareas programadas pueden corresponder a: 

- la llamada a un Procedure almacenado,

- una copia de seguridad. 




La llamada a un Procedure almacenado se utiliza para ejecutar el código de su elección, por ejemplo el mantenimiento personalizado, la edición de estadísticas, ...

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000149"></a>


### Crear una tarea programada
<a name="crear_una_tarea_programada_ELTPARAGRAPHE000024"></a>

Para crear una tarea programada:

1. Conectarse al servidor HFSQL solicitado. 

2. Seleccione "Tareas programadas".

3. Haga clic en el botón "Crear. Comienza la asistente para crear una tarea programada.

4. Especifique las opciones de programación: 

	- mes de ejecución: cada mes, algunos meses

	- día de ejecución: todos los días, algunos días de la semana o algunos días del mes

	- tiempo de ejecución: a una hora fija o durante el día de forma regular




5. Indique el contenido de la tarea programada: 

	- Procedimiento almacenado: especificar el nombre de la base de datos y el nombre del conjunto que contiene el Procedure almacenado, y luego seleccionar el Procedure.
			Atención: Para que un Procedure almacenado iniciado desde una tarea programada pueda acceder a los datos, debe utilizar [HDeclareExternal](../WDLang4/3044204.md). Esta función se utiliza para declarar las fuentes de datos que se utilizarán en los procesos de la Procedure almacenada. De hecho, cuando se inicia un Procedure almacenado desde el administrador remoto de HFSQL, no hay un análisis actual y los datos de HFSQL Client/Server no son accesibles inmediatamente.
			Si no se utiliza [HDeclareExternal](../WDLang4/3044204.md) en el código del Procedure almacenado, el Procedure provocará un error fatal, notificado en el registro de eventos del sistema.
			Observación: cuando un Procedure almacenado es iniciado por [HExecuteProcedure](../WDLang4/3044358.md) desde una aplicación WINDEV, WEBDEV o WINDEV Mobile, utiliza el análisis de la aplicación y puede utilizar directamente los datos de HFSQL Client/Server.

	- Copia de seguridad:

		- Especifique los elementos que se van a guardar: el contenido del servidor, una base de datos específica o una selección de bases de datos.

		- Especifique si la copia de seguridad debe contener los archivos de índice (archivos de índice con extensión NDX y archivos de índice de texto completo con extensión FTX).. Si no se guardan los archivos índice, el tamaño de la copia de seguridad se reducirá, pero el tiempo de restauración será mayor.. Observación: si los archivos de índice no se guardan, las contraseñas de los archivos deben especificarse durante la operación de restauración para permitir su reindexación.

		- Especifique el directorio de la copia de seguridad. Si este nombre corresponde a un ruta relativo, la copia de seguridad se realizará en el subdirectorio "Backup" del servidor HFSQL. El asistente se utiliza para definir un subdirectorio de "copia de seguridad" ("Backup" de Default). La fecha y la hora (en formato "Año\\Mes\\Día\\Tiempo-Minuto") se pueden añadir al nombre del directorio para identificar fácilmente la copia de seguridad..
						Observación: Las copias de seguridad se realizan en un subdirectorio numerado ("copia de seguridad\\Copia de seguridad\\1", por ejemplo). Por lo tanto, si se realizan varias copias de seguridad, los archivos correspondientes no serán sobrescritos por una nueva copia de seguridad.




6. Dé una description a su tarea programada y valídela.

7. La tarea programada se muestra automáticamente en el administrador remoto de HFSQL.





### Eliminar una tarea programada
<a name="eliminar_una_tarea_programada_ELTPARAGRAPHE000065"></a>

Para eliminar una tarea programada:

1. Conectarse al servidor HFSQL solicitado. 

2. Seleccione "Tareas programadas".

3. Seleccione la tarea programada que desea eliminar.

4. Haga clic en el botón "Eliminar".


## WDAdminHF: Gestión del servidor
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000158"></a>
Durante la conexión al administrador remoto de HFSQL, debe especificar el servidor HFSQL que se tendrá en cuenta. A continuación, se enumeran las diferentes opciones disponibles en el servidor seleccionado. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Detalles de las opciones disponibles
<a name="detalles_las_opciones_disponibles_ELTTEXTE000188"></a>


### Bases de datos
<a name="bases_datos_ELTPARAGRAPHE000019"></a>

Esta opción se utiliza para seleccionar una de las bases de datos que se encuentran en el servidor. Esto le permite acceder a todas las opciones de [gestión de la base de datos](../WDAdminHF/1000019702.md). 


### Conexiones
<a name="conexiones_ELTPARAGRAPHE000027"></a>

Esta opción enumera los diferentes usuarios conectados a las diferentes bases de datos que se encuentran en el servidor.

La lista de conexiones de Active contiene la siguiente información:

- el nombre de la base de datos,

- el nombre de usuario, 

- El nombre el equipo,

- el nombre de la aplicación que accede a la base de datos, ....




Usted tiene la capacidad de administrar las conexiones desde esta página: permitir o prohibir las conexiones, desconectar a los usuarios o enviar un mensaje a los usuarios.


### Transacciones
<a name="transacciones_ELTPARAGRAPHE000041"></a>

Esta opción se utiliza para gestionar las transacciones actuales o las transacciones interrumpidas en el servidor.


### Usuarios
<a name="usuarios_ELTPARAGRAPHE000046"></a>

Esta opción se utiliza para conocer y gestionar los usuarios del servidor. Tiene la posibilidad de crear, modificar o eliminar un usuario. 

Al crear o modificar un usuario, el usuario puede ser un superusuario: los usuarios definidos como "superusuarios" pueden realizar todas las acciones en el servidor, las bases de datos y los archivos de datos.


### Grupos
<a name="grupos_ELTPARAGRAPHE000053"></a>

Esta opción sirve para conocer y gestionar los grupos de usuarios del servidor. Tiene la posibilidad de crear, modificar o eliminar un grupo de usuarios.. 

También tiene la posibilidad de asociar a los usuarios con un grupo. 


### Derechos
<a name="derechos_ELTPARAGRAPHE000060"></a>

Esta opción se utiliza para gestionar los derechos en el servidor para el usuario o grupo seleccionado.


### Información del sistema
<a name="informacion_del_sistema_ELTPARAGRAPHE000065"></a>

Esta opción se utiliza para averiguar: 

- la plataforma del servidor.

- la versión de la plataforma.

- la arquitectura del motor.

- el procesador del servidor.

- la memoria disponible.

- la versión de los elementos en el servidor.





### Configuración del servidor
<a name="configuracion_del_servidor_ELTPARAGRAPHE000077"></a>

Esta opción se utiliza para configurar el servidor. Se tienen en cuenta los siguientes elementos:

- Ubicación de las bases de datos en el servidor (opción"General")

- Directorio de registros y copias de seguridad (opción"Registros")

- Archivo de registro (opción"Logs"): tiene la capacidad de especificar el directorio de los archivos de registro y habilitar algunas opciones de copia de seguridad. También puede especificar el tamaño máximo del archivo de registro.

- Parámetros del sistema (opción"Sistema"): puerto de escucha, cuenta de usuario del servidor, puerto para depuración remota, ....

- Tamaño de la caché de índice (opción"Caches"). 
	Observación: en 32 bits, el tamaño de la caché de índice está limitado a 1 GB.

- Estadísticas de actividad (opción"Estadísticas de actividad"): Las estadísticas de actividad se utilizan para medir la carga de trabajo del servidor. Usted tiene la capacidad de habilitar las estadísticas de actividad y especificar el directorio de respaldo.





### Registros
<a name="registros_ELTPARAGRAPHE000091"></a>

Esta opción se utiliza para ver los registros del servidor directamente. Esta opción sólo está disponible si los registros están habilitados en el servidor.

También tiene la posibilidad de importar los registros. 


### copias
<a name="copias_ELTPARAGRAPHE000098"></a>

Esta opción se utiliza para realizar y restaurar copias de seguridad del servidor y/o bases de datos.. Para obtener más información, consulte [Realización de copias de seguridad con el administrador remoto de HFSQL](../WDAdminHF/1000019704.md). 


### Tareas programadas
<a name="tareas_programadas_ELTPARAGRAPHE000106"></a>

Esta opción se utiliza para crear, modificar y eliminar las tareas programadas que se encuentran en el servidor.. Para obtener más información, consulte [Gestión de las tareas programadas con el administrador remoto de HFSQL](../WDAdminHF/1000019705.md).

## Principales observaciones
<a name="principales_observaciones_ELTTEXTE000208"></a>


### Versiones necesarias
<a name="versiones_necesarias_ELTPARAGRAPHE000018"></a>

**Atención**: Esta versión del Conector Nativo Progress funciona con WINDEV 28 y/o WEBDEV 28.
<a name="NOTE2_2"></a>


### Uso de los comandos de WLanguage en las consultas SQL
<a name="uso_los_comandos_wlanguage_las_consultas_sql_ELTPARAGRAPHE000029"></a>

Los comandos de WLanguage no pueden utilizarse en las consultas SQL. 
<a name="NOTE2_3"></a>


### Elementos Unicode
<a name="elementos_unicode_ELTPARAGRAPHE000036"></a>

Los elementos Unicode no son compatibles con el Conector de Progreso Nativo. Le aconsejamos que utilice elementos de texto en formato Ansi y que realice la Conversion( si es necesario) con [UnicodeToAnsi](../WDLang1/3024025.md).

[HCreation](../WDLang4/3044255.md) mostrará un mensaje de error cuando se creen archivos de datos que contengan elementos en formato Unicode.
<a name="NOTE2_4"></a>


### Requerido tabla
<a name="requerido_tabla_ELTPARAGRAPHE000050"></a>

El "sysprogress.systables" tabla debe ser accesible en modo de lectura para el usuario conectado.. De lo contrario, se muestra una Error("Versión de progreso no soportada").
<a name="NOTE2_5"></a>


### Actualizar la consulta de una nota
<a name="actualizar_consulta_una_nota_ELTPARAGRAPHE000057"></a>

El progreso no le permite realizar una consulta de actualización en un elemento del memorándum (LVARBINARIO). Sin embargo, WINDEV y WEBDEV le ofrecen la posibilidad de actualizar la nota mediante [HModify](../WDLang4/3044042.md) (el Record se borra y luego se añade).
<a name="NOTE2_6"></a>


### Inserción de más de 100 registros con una nota
<a name="insercion_mas_100_registros_con_una_nota_ELTPARAGRAPHE000067"></a>

Insertar más de 100 registros que contienen una posición de nota desencadena el error #9799. Este error es reference d al lado de Progress (reference P26935).

Para evitar este error, cierra la conexión antes del 100º Record, y luego vuelve a abrirla.
<a name="NOTE2_7"></a>


### Memo de texto y memo binario
<a name="memo_texto_memo_binario_ELTPARAGRAPHE000076"></a>

En una base de datos Progress, todos los memos son memos binarios. Sin embargo, se puede guardar un texto en los memos binarios.

## Características específicas del Conector de Progreso Nativo
<a name="caracteristicas_especificas_del_conector_progreso_nativo_ELTTEXTE000268"></a>


### Opciones de análisis
<a name="opciones_analisis_ELTPARAGRAPHE000085"></a>

- Las opciones de análisis relativas a los índices de texto completo no están soportadas por Conector Nativo.

- Las opciones de análisis relativas a la indexación de los memorandos de texto de Partial no están soportadas por el Conector Nativo.



<a name="NOTE3_2"></a>




### Consultas
<a name="consultas_ELTPARAGRAPHE000094"></a>

No se puede realizar ninguna búsqueda, filtro o clasificación (ORDER BY) en las columnas de agregados (Min, Max, etc.).
<a name="NOTE3_3"></a>


### Memo
<a name="memo_ELTPARAGRAPHE000101"></a>

- Los memos que excedan los 2 MB no pueden ser insertados.

- no se puede utilizar [SQLGetMemo](../WDLang4/3072020.md) o [SQLGetTextMemo](../WDLang4/3072012.md) para recuperar una nota en una cadena de texto. Se debe utilizar la función [HExtractMemo](../WDLang4/3044072.md). Este problema se solucionará en una próxima versión.



<a name="NOTE3_4"></a>


### Cierre de la conexión y de la transacción
<a name="cierre_conexion_transaccion_ELTPARAGRAPHE000118"></a>

Si la transacción no ha finalizado, puede producirse un fallo de protección general al cerrar la conexión.. Este problema se solucionará en una próxima versión.
<a name="NOTE3_5"></a>


### Caracteres acentuados en los alias
<a name="caracteres_acentuados_los_alias_ELTPARAGRAPHE000125"></a>

Si un alias contiene caracteres acentuados, debe hacerlo:

- encerrar el alias entre comillas dobles,

- utilizar la constante *hQueryWithoutCorrection* al ejecutar una consulta que utiliza este alias.



<a name="NOTE3_6"></a>


### Bloqueo HFSQL
<a name="bloqueo_hfsql_ELTPARAGRAPHE000135"></a>

Las cerraduras Record NO PUEDEN ser usadas con las funciones HFSQL en esta versión.

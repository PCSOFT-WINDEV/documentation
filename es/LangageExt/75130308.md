
## Tabla,Pantalla (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza la pantalla del tabla. Los controles calculados se recalculan. El tabla se reposiciona según el archivo de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Display,ProductTab,10");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Table,Display,ProductTab,10');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Table,Display,ProductTab,10")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Display(<Table name> , <Subscript>)`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la tabla a utilizar.

**`<Subscript>: (Cadena de caracteres)`**

Índice del elemento a mostrar o carácter que indica la acción a realizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Visualización de una memoria tabla
<a name="visualizacion_una_memoria_tabla_ELTPARAGRAPHE000044"></a>

- Si se especifica &lt;Índice&gt;, **Table,Display** vuelve a mostrar la fila cuyo índice ha sido especificado.

- Si &lt;Índice&gt;=P o &lt;Índice&gt;=T o &lt;Índice&gt;=\*, **Table,Display** vuelve a mostrar los elementos que se muestran actualmente.

- **Table, Display** se utiliza para volver a calcular los controles calculados.



<a name="NOTE0_2"></a>




### Visualización en un tabla de navegación
<a name="visualizacion_tabla_navegacion_ELTPARAGRAPHE000054"></a>

- Si se especifica &lt;Índice&gt;, **Table,Display** vuelve a mostrar la fila cuyo índice ha sido especificado.

- Si &lt;Índice&gt;=P, **Table,Display** vuelve a mostrar los elementos que se muestran actualmente en la tabla. Esto se utiliza para actualización la pantalla (después de una operación de escritura realizada en el archivo por Table,Save por ejemplo)

- Si &lt;Índice&gt;=T, **Table,Display** vuelve a mostrar el tabla del elemento actual en el fichero. Esto le permite visualizar la tabla desde una posición determinada.




Por ejemplo, para mostrar los registros correspondientes a una búsqueda.

1. La búsqueda se realiza (HReadSeek).

2. El resultado se muestra por medio de la tabla, la pantalla.

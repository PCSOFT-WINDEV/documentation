
## Pantalla, Pantalla a Archivo (Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa los elementos del archivo con el valor de los controles de la ventana vinculados a estos elementos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Screen,ScreenToFile");
HAdd("CUSTOMER");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Screen,ScreenToFile');
HAdd('CUSTOMER');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Screen,ScreenToFile")
call HAdd("CUSTOMER")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Screen,ScreenToFile()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Screen,ScreenToFile** no escribe directamente en el archivo. Asigna las variables del archivo.




- Para modificar la Record en el archivo, debe ejecutar HAdd o HModify según el caso.

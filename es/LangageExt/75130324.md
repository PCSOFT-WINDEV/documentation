
## Windows,TCurrent (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de la ventana actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Windows,TCurrent,Title of the current window");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Windows,TCurrent,Title of the current window');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Windows,TCurrent,Title of the current window")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Windows,TCurrent(<Window title>)`
---

**`<Window title>: (Cadena de caracteres)`**

 Título mostrado en la barra de título de la ventana actual.


## WDEnd (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza un programa que está utilizando WINDEV.
Observación: **WDEnd** es obligatorio en todos los programas de WINDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
WDEnd();
```


<a name="Example2"></a>



```txt
// In Pascal
WDEnd;
```


<a name="Example3"></a>





```txt
// In VB
call WDEnd
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WDEnd()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no es llamada por CALLWD.

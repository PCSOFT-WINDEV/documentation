
## Tabla,Select,? (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del elemento seleccionado en una control Tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Select,?,ProdTable");
strcpy(Subscript,WDInt);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Select,?,ProdTable');
Subscript:=WdInt;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Select,?,ProdTable")
subscript=WdInt
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Select,?(<Table control>)`
---

**`<Table control>: (Nombre del control)`**

Nombre del control Tabla a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- WdInt contiene la índice del elemento seleccionado.

- En el caso de una control Tabla rellenada programáticamente, el índice es relativo a toda la control Tabla.

- Para una control Tabla basada en un archivo de datos, el índice es relativo a los elementos visualizados.


## nWDSetCallbackSiguiente
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000076"></a>
set el "callback" llamado cuando se usan las palabras clave WDKEY y RETURNWDKEY para que se abra la siguiente ventana (sólo en C, Pascal y Visual Basic). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000100"></a>

```wl
Int      nWDSetCallBackNext(pfnWDCALLBACK pfnCallBack, DWORD dwUserParam);
```


**Detalles de los parámetros**

**pfnWDCALLBACK**: Address

Address de la función CallBack

**dwUserParam**:

Parámetro de usuario pasado a la función "Callback. Este parámetro se utiliza para dar la dirección de una estructura de datos asociada con el actual Process por ejemplo.

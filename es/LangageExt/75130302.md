
## Proyecto (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre y ejecuta un proyecto WINDEV. Se muestra la primera ventana del proyecto y la entrada se activa automáticamente en esta ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Project,Account");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Project,Account');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Project,Account")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Project(<Project name>)`
---

**`<Project name>: (Cadena de caracteres)`**

Nombre del proyecto a iniciar. Cuando se ejecuta el proyecto, se carga la hoja de estilo de la biblioteca del proyecto.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Importante: **Project** debe ser llamado después de [Biblioteca,Disco](../LangageExt/7513024.md) y antes de HInit.

- Si se ha seleccionado una primera ventana en la lista de proyectos, esta ventana será abierta por **Project**. Para evitar abrir una ventana, seleccione &lt;Ninguna&gt; como primera ventana en la lista de ventanas de proyectos.

- Un programa escrito en C o en Pascal sólo puede contener la función para iniciar el proyecto.

- WdKey contiene "\*E\*" si se ha producido un error durante la apertura.

- Si una ventana llama a algunas variables o procedimientos globales, el proyecto debe ser ejecutado por el Proyecto antes de abrir la ventana, de lo contrario las variables o procedimientos globales y la hoja de estilo no serán reconocidos.

- Varios proyectos pueden ser llamados en el mismo programa. Para ello, el proyecto actual debe ser cerrado antes de abrir otro. Para cerrar el proyecto actual, utilice la siguiente sintaxis:

	- Ejemplo en C: 
			
		```txt
		HInit(&stArrCtn[0],NBCTN);
		CALLWD("LIBRA,,DISK,MYLIBRARY");
		CALLWD("PROJECT,MyProject");
		```


	- Ejemplo en Pascal: 
			
		```txt
		HInit;
		CALLWD('PROJECT, LIBRARY,DISK,MYLIBRARY');
		CALLWD('PROJECT, MyProject');
		```


	- Ejemplo en VB: 
			
		```txt
		call HInit
		call CALLWD("PROJECT, BLIBRARY,DISK,MYLIBRARY ")
		call CALLWD("PROJECT, MyProject ")
		```

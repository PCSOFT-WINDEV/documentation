
## Pantalla,Columna (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de la columna actual subscriptada en un tabla de navegación o en una memoria tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Screen,Column");
ColNum=WDInt;
```


<a name="Example2"></a>





```txt
// In Pascal
CallWD('Screen,Column');
ColNum:=WdInt;
```


<a name="Example3"></a>





```txt
// In VB
call CallWd("Screen,Column")
ColNum=WDInt
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Screen,Column()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Screen,Column** devuelve el número de la columna seleccionada en WdInt

- ** Screen, Column** sólo se puede utilizar en las columnas suscritas de las tablas de navegación y memoria.


## Estilo (Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga una hoja de estilo.

**Observación**: Un proyecto se adjunta a la hoja de estilo estándar o a una hoja de estilo específica del proyecto. Al cargar el proyecto, se carga la hoja de estilo adjunta al proyecto. **Style** se utiliza para cargar una hoja de estilo distinta a la de Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("STYLE,PROJECT1.WDS");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('STYLE,PROJECT1.WDS');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("STYLE,PROJECT1.WDS")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Style(<Sheet name>)`
---

**`<Sheet name>: (Cadena de caracteres)`**

Nombre de la nueva hoja de estilo a utilizar. El nombre de una hoja de estilo es el nombre del proyecto que la creó con la extensión WDS. La hoja de estilo estándar se llama WD.STY.

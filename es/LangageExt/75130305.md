
## WavInit (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa el intérprete de sonido. Esta función debe ejecutarse una vez en cada programa. Permite calibrar la velocidad de ejecución de los sonidos.. Esta función puede durar unos minutos.
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SoundInit([<Sound Initialization>])`
---

**`<Sound Initialization>: (Character string (optional))`**

 Nombre del archivo de inicialización que se utilizará para inicializar el intérprete de sonido.

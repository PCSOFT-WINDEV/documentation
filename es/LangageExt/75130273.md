
## Cerrar (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la ventana actual y restaura el contexto que existía antes de que se abriera la ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Close");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Close');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Close")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Close()`

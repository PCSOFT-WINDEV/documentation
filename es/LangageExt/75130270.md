
## Espacio (Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Rellena el contenido de WdString con caracteres de espacio a la derecha.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Space,Yes");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Space,Yes');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Space,Yes")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Space(<Indicator>)`
---

**`<Indicator>: (Cadena de caracteres)`**

Puede ser set a "Sí" o "No":

- "Sí": llena WdString con caracteres de espacio.

- "No": no llena WdString con caracteres de espacio.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función se comporta como una palanca: después de la llamada a Option,Space,Yes, WdString se llenará automáticamente con caracteres de espacio hasta que la llamada a Option,Space,No

- Si Option,Space,No es llamado, WdString no será llenado por caracteres de espacio.

- Por Default, WdString no se llenará de caracteres espaciales ("No").

- Si se llama a Opción,Espacio,Sí, entonces Pantalla,get devuelve en WdString el contenido del control rellenado con caracteres de espacio (longitud de WdString = longitud del control).
	**Atención**: WdString puede contener hasta 200 caracteres.

- Si se llama Option,Space,Yes, entonces Screen,Current devuelve en WdString el contenido del control lleno de caracteres espaciales a get 10 caracteres.

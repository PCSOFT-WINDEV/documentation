
## Test (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra cada llamada de función y cada status reporte devuelto por la función.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Test,Yes");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Test,Yes');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Test,Yes")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Test(<Indicator>)`
---

**`<Indicator>: (Cadena de caracteres)`**

Puede ser set a:

- "Sí" para cambiar al modo de prueba

- "no" para inhibir el modo de prueba.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para ejecutar la siguiente función, se debe pulsar una tecla.

- Para Cancel el modo de prueba, todo lo que tienes que hacer es presionar la tecla Esc (equivalente a Opción,Prueba,No).


## nWDOpenWDL: Abrir una biblioteca
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000072"></a>
Abre una biblioteca creada por WINDEV (archivo con extensión WDL). Este archivo contiene las ventanas, el código y el análisis de la aplicación WINDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000096"></a>

```wl
Int      nWDOpenWDL(LPCTSTR pszFullWDLName);
```


**Detalles de los parámetros**

**pszFullWDLName:** Cadena de caracteres

Nombre y ruta completa de la librería WINDEV (.WDL) a abrir.

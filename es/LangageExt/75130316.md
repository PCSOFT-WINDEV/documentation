
## Tabla,Posición (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciona un elemento dado o devuelve el elemento encontrado en la primera fila de una memoria tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Position,NameTable,10");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Position,NameTable,10');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Position,NameTable,10")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Position(<Table name> [, <Subscript>])`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la tabla a utilizar.

**`<Subscript>: (Cadena de caracteres (opcional))`**

Índice del elemento en el tabla.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Table, Position** no puede utilizarse con tablas basadas en archivos de datos.

- Si no se especifica índice, WdInt contiene el índice correspondiente al primer fila visualizado.

- Si se especifica la índice, el elemento correspondiente se coloca en la parte superior de la tabla.

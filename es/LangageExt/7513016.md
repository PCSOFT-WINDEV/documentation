## Sintaxis
<a name="sintaxis_ELTTEXTE000096"></a>

```wl
Int      nWDInit(BOOL bDisplayError);
```


**Detalles del parámetro**

**bDisplayError**: *Boolean*

- True para mostrar la ventana de error estándar de WLanguage si se produce un error de programación (error de sintaxis, por ejemplo).

- False para recuperar el error y personalizar la gestión de errores.


## Nation( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el idioma de visualización para los títulos, subtítulos y mensajes.... de las ventanas y los controles.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
French=5
Spanish=7;
//LANGUAGE is a radio button
CALLWD("SCREEN,GET, LANGUAGE");
If WdInt==1 CALLWD("NATION,%d",French);
If WdInt==2 { //Spanish
	CALLWD("NATION,%d",Spanish);
	CALLWD("LIBRARY,ERROR,SPANISH.WDM");
If WdInt>0 CALLWD("ERROR,WDM file not found")
```


<a name="Example2"></a>



```txt
// In Pascal
French:=5; Spanish:=7;
{LANGUAGE is a radio button}
CALLWD('SCREEN,GET, LANGUAGE');
If WdInt=1 THEN {French}
	CALLWD('NATION,'+ConvertString(French));
End;
If WdInt=2 THEN {Spanish}
Begin
	CALLWD('NATION,'+ ConvertString(Spanish));
	CALLWD('LIBRARY, ERROR,SPANISH.WDM');
	If WdInt>0 THEN CALLWD('ERROR,WDM file not found');
End;
```


<a name="Example3"></a>



```txt
// In VB
French=5
Spanish=7
'LANGUAGE is a radio button
CALL CALLWD("SCREEN,GET, LANGUAGE")
If WdInt=1 THEN 
	CALL CALLWD("NATION,"+STR$(French))
End If

If WdInt=2 THEN
	CALL CALLWD("NATION,"+STR$(Spanish))
	CALL CALLWD("LIBRARY,ERROR,SPANISH.WDM")
	If WdInt>0 THEN
		CALL CALLWD("ERROR,WDM file not found")
	End if
End if
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Nation(<Language Number>)`
---

**`<Language Number>: (Integro)`**

 Número del idioma en el que se ejecutará el proyecto. Este valor se define en el editor al seleccionar los idiomas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Después de llamar a la función **Nation**, se mostrarán los títulos, textos y mensajes al cargar la ventana... en el idioma solicitado.

- **Nation** puede utilizarse para desarrollar aplicaciones internacionales que se distribuyan en varios idiomas


## nWDExecuteEx: Ejecutar una función WLanguage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000077"></a>
Ejecuta una función WLanguage por compilación dinámica, sin formato de comando.

**Observación**: esta función se comporta como [nWDExecute](../LangageExt/7513015.md) pero no interpreta la cadena de caracteres que contiene el comando. Esto permite una ejecución más rápida de comandos no formateados (comandos que no contienen %s, %d, ...).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000101"></a>

```wl
Int      nWDExecuteEx(LPCTSTR pszCommand);
```


**Detalles de los parámetros**

**pszComando**: 

WLanguage para ejecutar el comando

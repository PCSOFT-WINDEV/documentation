
## nWDGetHFContexto: Recuperación del contexto HyperFileSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000066"></a>
Recuperación del contexto HyperFileSQL (sólo en C, Pascal y Visual Basic). Esta función se utiliza cuando se inicializa un proyecto que utiliza un análisis HyperFileSQL.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000090"></a>

```wl
Int      nWDGetHFContext(void** ppHFContext);
```

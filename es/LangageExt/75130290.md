
## Lista,Seleccionar,? (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el elemento seleccionado en la lista.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("List,Select,?,ProdList,1");
strcpy(Selection,WDString);
strcpy(Subscript,WDInt);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('List,Select,?,ProdList,1');
Selection:=WdString;
Subscript:=WdInt;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("List,Select,?,ProdList,1")
Selection=WdString
Subscript=WdInt
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Select,?(<List name> , <Rank>)`
---

**`<List name>: (Cadena de caracteres)`**

Nombre de la lista a utilizar.

**`<Rank>: (Cadena de caracteres)`**

Rango del elemento seleccionado (lista de selección múltiple): 1 a get el primer elemento seleccionado, 2 a get el segundo, ...  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- WdInt contiene la índice del elemento seleccionado.

- WdString contiene el valor del elemento seleccionado.

- Si no se ha realizado ninguna selección, WdInt es igual a -1.

- Si se utiliza esta función en los cuadros combinados editables, el valor devuelto por WdInt es inconsistente.

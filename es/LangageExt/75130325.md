
## Windows,Título (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de la siguiente ventana (o cuadro de diálogo) mostrada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Windows,TNext,Title of the next window");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Windows,TNext,Title of the next window');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Windows,TNext,Title of the next window")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Windows,Title(<Window title>)`
---

**`<Window title>: (Cadena de caracteres)`**

 Título que se mostrará en la barra de título de la siguiente ventana.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Windows, Title** y [Windows,TNext](../LangageExt/75130326.md) son equivalentes.

- Si no se utiliza la ventana **Windows, Title**, la siguiente ventana que se abrirá tiene el título que se definió durante la ventana description o el mismo título que la ventana actual (para las ventanas estándar "Info", "YesNo", "Error" y "Confirmar").

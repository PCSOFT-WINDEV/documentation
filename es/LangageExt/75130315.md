
## Tabla,Count( Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos de una control Tabla. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Count,ProdTable");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Count,ProdTable');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Table,Count,ProdTable")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Count(<Table control>)`
---

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En el caso de una control Tabla poblada programáticamente, WdInt contiene el número de elementos de la control Tabla.

- En el caso de una control Tabla basada en un archivo de datos, WdInt contiene el número de elementos mostrados en la control Tabla.

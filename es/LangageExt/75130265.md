
## Pantalla,Input( Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la entrada en los controles de una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
while (strcmp(Key,"ESC")!=0) {
	CALLWD("Screen,Input");
	strcpy(Key, WDKey);
```


<a name="Example2"></a>





```txt
// In Pascal
While Key<>'ESC' DO
Begin
	CALLWD('Screen,Input');
	Key:=WDKey;
	...
End;
```


<a name="Example3"></a>



```txt
// In VB
While Key<>"ESC"
	call CALLWD("Screen,Input")
	Key=WdKey
	...
WEnd
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Screen,Input(</CR> , </NONLOCKING>)`
---

**`</CR>: ()`**

Le permite detectar si se ha producido un error.. Si se especifica este parámetro, WdKey contiene "\*E\*" si se produce un error.

**`</NONLOCKING>: ()`**

Le permite salir inmediatamente de la entrada sin tener que esperar un resultado.. Si se especifica este parámetro: "\*N\*" se devuelve en WdKey si no se ha realizado ninguna acción.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al final de la entrada, la WdKey Variable contiene el estado reporte de la entrada modificado en WLanguage por ReturnWdKey o WdKey=&lt;Key Code

- Al final de la entrada del menú, la WdKey Variable contiene "\*M\*" y la WdString contiene la secuencia de atajos para las opciones seleccionadas.


## Uso (Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana especificada cerrando todas las ventanas abiertas de antemano.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Use,Main.WDW");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Use,Main.WDW');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Use,Main.WDW")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Use(<Window name> [, <X> [, <Y> [, </CR>]]])`
---

**`<Window name>: (Cadena de caracteres)`**

 Nombre de la ventana a abrir. Este parámetro incluye el ruta, el nombre y la extensión de la ventana (la extensión de las ventanas WINDEV es ".WDW" de Default).

**`<X>: (Entero (opcional))`**

 Abscisa (en pixeles) de la ventana (con respecto a la esquina superior izquierda de la pantalla o ventana padre).

**`<Y>: (Entero (opcional))`**

Mostrar la coordenada y (en píxeles) de la ventana (en relación con la Corner superior izquierda de la pantalla).

**`</CR>: (Parámetro (opcional))`**

Le permite detectar si se ha producido un error..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se especifica el parámetro /CR, el código de estado "E\*" se devuelve en WdKey en caso de error.

- Para modificar el título de la ventana, debe utilizar Windows,TNext o Windows,TCurrent.

- No se debe llamar a la función **Use** durante una prueba interactiva. De lo contrario, aparecerá un mensaje de error. No se puede pasar ningún parámetro a la ventana a través de esta función.

- Si una ventana llama a procedimientos globales, el proyecto debe ser ejecutado por Proyecto antes de abrir la ventana, de lo contrario los procedimientos globales y la hoja de estilo no serán reconocidos.

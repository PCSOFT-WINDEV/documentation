
## Opción, Prueba (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
En una ventana de información, muestra cada llamada a la función y cada status reporte devuelto por la función.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Option,Test,Yes");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Option,Test,Yes');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Option,Test,Yes")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Option,Test(<Indicator>)`
---

**`<Indicator>: (Cadena de caracteres)`**

Puede ser set a:

- "Sí" para cambiar al modo de prueba

- "no" para inhibir el modo de prueba.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para ejecutar la siguiente función de aplicación, se debe pulsar una tecla.

- Para cancelar el modo de prueba, solo debe pulsar la tecla Esc (equivalente a **Option,Test,No**).

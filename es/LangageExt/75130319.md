
## Tabla,Seleccionar,+ (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona un elemento en una control Tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Select,+,ProdTable,10");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Select,+,ProdTable,10');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Select,+,ProdTable,10")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Select,+(<Table control> , <Index>)`
---

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.

**`<Index>: (Cadena de caracteres)`**

Número de la fila a seleccionar. Si este parámetro es inferior a 0, el control Tabla basado en el un archivo de datos "sube" para mostrar el Record deseado. Si el índice es mayor que el número de filas mostradas, la control Tabla se "desplaza hacia abajo" para mostrar la Record deseada. Sin embargo, le recomendamos que utilice [Tabla, Visualización](../LangageExt/75130308.md) precedida del posicionamiento en el archivo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En el caso de una control Tabla rellenada programáticamente, el índice es relativo a toda la control Tabla.

- Para una control Tabla basada en un archivo de datos, el índice es relativo a los elementos visualizados.


## Pantalla,Archivo a pantalla (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa los controles de ventana con los valores de los ítems de fichero relacionado a los controles.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
HReadFirst("CUSTOMER","NAME")
if!HOut() {
	CALLWD("Screen,ScreenToFile");
else
	CALLWD("Error,Empty file");
}
```


<a name="Example2"></a>



```txt
// In Pascal
HReadFirst("CUSTOMER","NAME");
If not HOut() then CALLWD('Screen,'FileToScreen')
else CALLWD('ERROR,Empty file');
```


<a name="Example3"></a>



```txt
// In VB
if not HOut() then Call CALLWD("Screen,FileToScreen")
else call CALLWD("ERROR,Empty file")
End If
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Screen,FileToScreen()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Pantalla, FileToScreen** no lee el archivo. Los controles se inicializan con el Record cargado en la memoria. Un Record debe estar necesariamente cargado en la memoria antes de llamar a esta función.

- Para una relación directa, **Screen, FileToScreen** transfiere el valor de los elementos del registro cargado en memoria a los controles relacionados, sin realizar ninguna operación de lectura.

- En caso de enlace con la búsqueda en un archivo relacionado, la búsqueda se realiza en el archivo relacionado antes de transferir el valor de los elementos al control.

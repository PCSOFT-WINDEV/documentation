
## Lista,Posición (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciona el primer elemento mostrado en la lista o devuelve la índice del primer elemento mostrado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
//position element 10 at the top of the list
CALLWD("List,Position,NameList,10");
```


<a name="Example2"></a>



```txt
// In Pascal
(* position element 10 at the top of the list *)
APPELWD('List,Position,NameList,10');
```


<a name="Example3"></a>



```txt
// In VB
'position element 10 at the top of the list
call CALLWD("List,Position,NameList,10")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Position(<List name> [, <Subscript>])`
---

**`<List name>: (Cadena de caracteres)`**

Nombre de la control List Box a modificar.

**`<Subscript>: (Cadena de caracteres (opcional))`**

Índice del elemento a posicionar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si no se especifica índice, WdInt contiene el índice del elemento actual.

- Si se especifica el índice, el elemento correspondiente se coloca en la parte superior de la lista.

- La lista, la posición no selecciona ningún elemento. La función utilizada para seleccionar un elemento es Lista, Seleccionar, +.

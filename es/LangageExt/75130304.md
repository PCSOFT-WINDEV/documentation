
## HourGlass( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un puntero del ratón en forma de reloj de arena.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Hourglass,Yes");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Hourglass,Yes');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Hourglass,Yes")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Hourglass(<Indicator>)`
---

**`<Indicator>: (Cadena de caracteres)`**

Puede ser set a:

- "Sí" para mostrar el puntero en forma de reloj de arena

- "No" para restaurar el puntero inicial del ratón.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Reloj de Arena, No es llamado por Default.

- El reloj de arena recupera automáticamente su forma inicial durante la llamada a [Pantalla,Entrada](../LangageExt/75130265.md).


## Lista,Insertar (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un elemento en el índice dado de una lista desplegable o expandida.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD('List,Insert,ProdList,2=%s",CODE);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('List,Insert,ProdList,2="+CODE);
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("List,Insert,ProdList,2="+CODE)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Insert(<List name> , <Subscript> , <Value>)`
---

**`<List name>: (Cadena de caracteres)`**

Nombre de la control List Box a modificar.

**`<Subscript>: (Cadena de caracteres)`**

Subíndice del elemento en la lista donde el elemento debe ser insertado.

**`<Value>: (Cadena de caracteres)`**

Valor a insertar en la lista.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la lista está ordenada, List,Insert posiciona el elemento en la posición actual sin ordenar la lista.. Para una lista ordenada, le aconsejamos que utilice Listar, Añadir en lugar de Listar, Insertar.

- Si la índice especificada es mayor que el número de elementos de la lista, List,Insert es equivalente a List,Add.


## xxWDGetReturnValue
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000125"></a>
Permite recuperar el valor de retorno tras una llamada a [nWDEvalorar](../LangageExt/7513017.md), [nWDExecute](../LangageExt/7513015.md) o [nWDExecuteEx](../LangageExt/7513020.md).

Cada una de las funciones que se presentan a continuación devuelve un tipo diferente. Para averiguar el tipo que será devuelto por el código de WLanguage:

- véase la documentación sobre la función WLanguage,

- utilizar [wWDGetReturnValueType](../LangageExt/7513008.md)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000149"></a>


BOOL bWDGetReturnValue(void);




Devuelve el valor si se ha producido un error: Si el valor no es booleano, la función devuelve False.<a name="NOTE2_2"></a>


Int nWDGetReturnValue(void);




Devuelve el valor si se ha producido un error: Si el valor no es un número entero, la función devuelve 0.<a name="NOTE2_3"></a>


WDINT64 i64WDGetReturnValue(void);




Devuelve el valor si se ha producido un error: Si el valor no es un entero de 8 bytes, la función devuelve 0.<a name="NOTE2_4"></a>


Doble dWDGetReturnValue(void);




Devuelve el valor si se ha producido un error: Si el valor no es un flotador, la función devuelve 0,0.<a name="NOTE2_5"></a>


Doble dWDGetReturnValue(void);




Devuelve el valor si se ha producido un error: Si el valor no es un flotador, la función devuelve 0,0.

## Notas
<a name="notas_ELTTEXTE000197"></a>


### Valor devuelto
<a name="valor_devuelto_ELTPARAGRAPHE000076"></a>

Se devolverá un valor por defecto si el tipo del valor de retorno no corresponde a una de las funciones mencionadas de Above. Este valor de Default se especifica para cada función (valor de retorno si se produce un error)

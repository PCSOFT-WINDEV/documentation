
## Tabla,Búsqueda (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un valor en una control Tabla rellenada programáticamente. Es una búsqueda por coincidencia exacta.

**Observación**: Esta función no debe utilizarse para una control Tabla basada en un archivo de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Search,Prod,000002");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Table,Search,Prod,000002');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Table,Search,Prod,000002")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Search(<Column name> , <Sought value>)`
---

**`<Column name>: (Cadena de caracteres)`**

Nombre de la columna donde se realizará la búsqueda.

**`<Sought value>: (Cadena de caracteres)`**

Valor buscado en la columna. Este valor debe tener el mismo tamaño que el valor de la columna buscada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**TABLE, Search** devuelve en WdInt:

- el índice del elemento tabla que coincide con la búsqueda,

- -1 si no se encuentra la cadena buscada.


## nWDEvalorar: Evaluación del valor de un elemento
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000080"></a>
Evaluación del valor de un elemento: control, Variable, HyperFileSQL item (por ejemplo: Control.de.ventanas, Variable, archivo.elemento, ...).

**Observación**: no se permite el acceso a los elementos subdivididos (ejemplo: Tabla[ 1]).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000104"></a>

```wl
Int      nWDEvaluate(LPCTSTR pszElement);
```


**Detalles de los parámetros**

**pszElement:**

Nombre del elemento a evaluar

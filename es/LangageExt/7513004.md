## Principio
<a name="principio_ELTTEXTE000251"></a>
A continuación encontrará la description de las funciones accesibles en lenguaje C para interconectar sus módulos WINDEV dentro de un código fuente C.

Los parámetros esperados por cada comando se presentan en detalle, así como las características específicas.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Constantes
<a name="constantes_ELTTEXTE000275"></a>
Las funciones que devuelven una **int** indicando el resultado del comando, utilice las siguientes constantes (definidas en wdextern.h):

|   Name |   Value | Significado |
| --- | --- | --- |
| WDERROR_OK | 0 | Éxito |
| WDERROR_VERSION | 1 | Número de versión incompatible |
| WDERROR_DLLLOAD | 2 | Problema al cargar la DLL |
| WDERROR_WDLLOAD | 3 | Problema al cargar la BDM |
| WDERROR_DLLLCPL | 4 | DLL de compilación no encontrada |
| WDERROR_CPLERROR | 5 | ¿error de compilación? |
| WDERROR_MEMORIA | 6 | No hay memoria disponible |
| WDERROR_TYPE | 8 | Tipo no convertible |
| WDERROR_INIT | 9 | Idioma externo no inicializado |
| WDERROR_EJECUCIÓN | 10 | Error de tiempo de ejecución |
| WDERROR_ERROR | 11 | Error anterior no procesado |
| WDERROR_COMMAND | 12 | Comando de nWDExecute demasiado tiempo |
| WDERROR_INITPROJECT | 13 | El código de inicialización del proyecto ya se ha ejecutado |
| WDERROR_UNKNOWNELEMENT | 14 | Elemento desconocido |

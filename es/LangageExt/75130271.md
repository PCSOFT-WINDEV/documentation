
## Execute( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta el código de un control o el código de un Procedure en WLanguage.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
// Run the code of the DRAW procedure of the current window
CALLWD("Execute,DRAW.PRO");
// Run the code of the validation button
CALLWD("Execute,Validate.CLI");
```


<a name="Example2"></a>



```txt
// In Pascal
(* Run the code of the DRAW procedure of the current window *)
CALLWD('Execute,DRAW.PRO');
(* Run the code of the validation button *)
CALLWD('Execute,Validate.CLI');
```


<a name="Example3"></a>



```txt
// In VB
' Run the DRAW procedure of the current window
call CALLWD("Execute,DRAW.PRO")
' Run the code of the validation button
call CALLWD("Execute,VALIDATE.CLI")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Ejecutando el código de un control

`Execute(<Control name> , <Type of Event>)`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de control cuyo código se ejecutará.

**`<Type of Event>: (Cadena de caracteres)`**

Corresponde al código de la Process en WLanguage que debe ser ejecutado:

- "INI" para el código de inicialización

- "ENT" para el código de entrada de un control

- "SOR" para el código de salida de un control

- "CLI" para el código de clic en un botón

- "MOD" para el código de modificación de un control

- "LEN" para el código de entrada de un tabla fila

- "LSO" para el código de salida de un tabla fila

- "SUI" para el código Process del siguiente en un tabla

- "PRE" para el código Process del anterior en un tabla

- "AFF" para el código de visualización de una tabla.  





<a name="SYNTAX2"></a>

### Ejecutando el código de un Procedure

`Execute(<Procedure Name + ".PRO">)`
---

**`<Procedure Name + ".PRO">: (Cadena de caracteres)`**

Nombre de la Procedure en WLanguage que debe ser ejecutada.

## Observaciones




### Atención
<a name="atencion_ELTPARAGRAPHE000059"></a>

No se puede pasar ningún parámetro con la función **Execute**.

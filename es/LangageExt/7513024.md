
## Biblioteca,Disco (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la biblioteca de proyectos que está almacenada en el disco.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Library,Disk,Account.WDL");
CALLWD("Project,Account");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Library,Disk,Account.WDL');
CALLWD('Project,Account');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Library,Disk,Account.WDL")
call CALLWD("Project,Account")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Library,Disk(<Library name> [, <"/CR">])`
---

**`<Library name>: (Cadena de caracteres)`**

Nombre de la biblioteca con la extensión y el acceso ruta si es posible.

**`<"/CR">: (Cadena de caracteres (opcional))`**

Código de estado opcional devuelto por **Biblioteca,Disco**.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Biblioteca, Disco** debe ser llamado antes de HInit y Project.
	Una biblioteca es creada por el editor de WINDEV y su extensión es la Biblioteca Digital Mundial.. Se puede abrir una sola biblioteca en un momento dado.
	La hoja de estilo del proyecto es cargada por la función:
	Proyecto,&lt;nombre_del_proyecto&gt;&gt; Proyecto

- **Biblioteca,Disco** se llama con el parámetro "/CR":

	- Si se produce un error al abrir la biblioteca:

		- El código de estado "\*E\*" se devuelve en la WdKey Variable (WDKey en C)

		- WdInt (WDEntier en C) contiene un archivo no existente, una biblioteca no válida y una memoria insuficiente.




- Si no se ha producido ningún error al abrir la biblioteca:

	- WdInt (WDInt en C) contiene 0

	- WdKey (WDKey en C) devuelve una cadena vacía.

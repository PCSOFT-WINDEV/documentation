
## Info( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una ventana de diálogo que contiene un logotipo de información y el mensaje pasado en el parámetro.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Info,Hello");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Info,Hello');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Info,Hello")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Info(<Message>)`
---

**`<Message>: (Cadena de caracteres)`**

Mensaje a mostrar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La ventana se cerrará cuando el usuario valide el botón "OK".

- Por Default, la ventana tiene el mismo título que la ventana actual. Para modificar el título, debe usar Windows,TNext.

- El tamaño máximo del mensaje no puede exceder los 200 caracteres.

- Para visualizar el mensaje en varias líneas, utilice la siguiente sintaxis:

	- Ejemplo en C: 
			
		```txt
		CALLWD("Info,  \n ");
		```


	- Ejemplo en Pascal:
			
		```txt
		CALLWD('Info, <Text1>'+Chr(10)+'<Text2>');
		```


	- Ejemplo en VB: 
			
		```txt
		call CALLWD("Info, "+CHR(10)+"")
		```


## HsMenú, Etiqueta (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de una opción de menú.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("HSMENU,LABEL,%s,%s",Option,NewOption);
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('HSMENU,LABEL,'+Option+','+NewOption);
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("HSMENU,LABEL,"+Option+","+NewOption)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HsMenu,Label(<Option identifier> , <New caption>)`
---

**`<Option identifier>: (Cadena de caracteres)`**

Número de la opción de modificar. El número de la opción es el asignado durante el menú description. Está disponible en el editor del menú y en la documentación del proyecto.

**`<New caption>: (Cadena de caracteres)`**

Nueva leyenda para la opción. Este parámetro debe contener el carácter "&" (delante del atajo de la opción).

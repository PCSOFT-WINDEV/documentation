
## Proyecto nWDInit: Código de inicialización del proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000072"></a>
Ejecute el código de inicialización del proyecto.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000096"></a>

```wl
Int      nWDInitProject(LPCTSTR pszProjectName)
```


**Detalles de los parámetros**

**pszProjectName:** *Cadena de caracteres*

Nombre del proyecto cuyo código de inicialización debe ejecutarse. Atención: se puede ejecutar un único código de inicialización del proyecto en una aplicación.

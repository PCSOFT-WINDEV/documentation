
## Menú Hs,Seleccionar (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona o deselecciona, hace que Invisible, marca o desmarca una opción que se encuentra en el menú actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("HsMenu,Select,+,FS");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('HsMenu,Select,+,FS');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("HsMenu,Select,+,FS")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HsMenu,Select(<Code> [, <Option>])`
---

**`<Code>: (Cadena de caracteres)`**

Carácter que puede tomar por valor:

- "+" para seleccionar una opción

- "-" para desmarcar una opción

- "I" para hacer una opción Invisible

- "M" para marcar una opción

- "D" para desmarcar una opción.




**`<Option>: (Cadena de caracteres (opcional))`**

Atajo utilizado para acceder a la opción o al número de opción.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El acceso a la opción se realiza según los accesos directos de la opción.

- Al teclear el menú, las opciones deseleccionadas no pueden ser validadas, serán Grayed.

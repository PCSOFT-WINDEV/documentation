
## Tabla,Borrar (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un elemento de un tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Delete,ProdTable,3");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Delete,ProdTable,3');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Delete,ProdTable,3")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Delete(<Table name> [, <Subscript>])`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la tabla a utilizar.

**`<Subscript>: (Cadena de caracteres (opcional))`**

Índice del elemento a eliminar. Los siguientes elementos se mueven hacia arriba por un rango.

Si no se especifica índice, se borra el elemento seleccionado.

Para una tabla de memoria, la índice es relativa a toda la tabla.

Para una tabla de navegación, la índice es relativa a los elementos vistos.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para una tabla de navegación, la Record se borra directamente del archivo.

- Para un borrado encadenado, los borrados deben realizarse desde el último elemento a borrar.. De hecho, los elementos que se encuentran después de cada eliminación se mueven hacia la parte superior.

- **Table,Delete**,&lt;Nombre de la tabla&gt;,\* elimina todos los elementos de una tabla memoria

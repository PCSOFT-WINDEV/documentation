
## Tabla,get (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el valor de un elemento dado o el valor del elemento seleccionado en una tabla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Get,ProdTable");
strcpy(Value,WDString);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Get,ProdTable');
Value:=WdString;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Get,ProdTable")
Value=WdString
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Get(<Table name> [, <Subscript>])`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la columna tabla.

**`<Subscript>: (Cadena de caracteres opcional)`**

Subíndice del elemento en la tabla.

- Si no se especifica este parámetro, **Table, Get** devuelve el valor del elemento actual en tabla .

- Si se especifica este parámetro, **Table,Get** devuelve el valor del elemento cuyo índice se especifica.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El valor del elemento recuperado está contenido en WdString. El elemento índice está contenido en WdInt.

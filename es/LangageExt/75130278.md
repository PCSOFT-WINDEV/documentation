
## Menú HsMenu,Nombre (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un menú en la ventana que se abre. El menú es el menú de otra ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("HsMenu,Name,Menu3");
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('HsMenu,Name,Menu3');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("HsMenu,Name,Menu3")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HsMenu,Name(<Name of Menu Window>)`
---

**`<Name of Menu Window>: (Cadena de caracteres)`**

Nombre de la ventana que contiene el menú (sin la extensión).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **HsMenú, Nombre** le permite usar en una ventana, un menú que fue creado en otra ventana.

- **HsMenu, Name** sólo puede ser usado en las ventanas que pueden incluir un menú (ventana con un "Border delgado").

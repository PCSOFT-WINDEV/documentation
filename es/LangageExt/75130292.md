
## Lista,Seleccionar,- (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un elemento de una lista desplegable o de una lista ampliada.. Esta función se utiliza principalmente para los cuadros de lista de selección múltiple de manipular.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("List,Select,-,ProdList,10");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('List,Select,-,ProdList,10');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("List,Select,-,ProdList,10")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Select,-(<List name> , <Subscript>)`
---

**`<List name>: (Cadena de caracteres)`**

Nombre de la lista a utilizar.

**`<Subscript>: (Cadena de caracteres)`**

Índice del elemento a desmarcar. Si &lt;Índice&gt;=\*, se desmarcan todos los elementos de la lista.

**Observación**: **List,Select,-** no tiene efecto si &lt;Índice&gt; es mayor que el número de elementos de la lista..

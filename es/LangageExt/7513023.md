
## Beep( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para emitir un pitido en el altavoz de el equipo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Beep");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Beep');
```


<a name="Example3"></a>





```txt
// In VB
Call CALLWD("Beep")
```

<a name="SYNTAX1"></a>

## Sintaxis


## Lista,Count((Lengua externa))



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos encontrados en una lista.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD('List,Count,ProdList');
NbElem=WDInt;
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('List,Count,ProdList');
NbElem:=WDInt;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("List,Count,ProdList")
NbElem=WDInt
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Count(<List name>)`
---

**`<List name>: (Cadena de caracteres)`**

Nombre de control List Box a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
WdInt contiene el número de elementos que se encuentran en la lista.


## Error( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una ventana de diálogo que contiene un logotipo de error y el mensaje pasado en el parámetro.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Error,Error reading the file %s",FileName);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Error,Error reading the file '+FileName);
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Error,Error reading the file "+FileName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Error(<Message>)`
---

**`<Message>: (Cadena de caracteres)`**

Mensaje a mostrar (hasta 200 caracteres).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La ventana se cerrará cuando el usuario valide el botón "OK".

- Por Default, el título de la ventana es el título de la ventana actual. Para modificar el título, debe utilizar [Windows,TNext](../LangageExt/75130326.md).

- Para visualizar el mensaje en varias líneas, utilice la siguiente sintaxis:

	- Ejemplo en C:
			
		```txt
		CALLWD("Error,  \n ");
		```


	- Ejemplo en Pascal:
			
		```txt
		CALLWD('Error, '+Chr(10)+ ' ');
		```


	- Ejemplo en VB:
			
		```txt
		call CALLWD("Error, "+CHR(10)+"")
		```


## Biblioteca,Error( Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga el archivo que contiene los mensajes de error manejados por WD??EXE.DLL en el idioma deseado. Por defecto, si no se llama a ninguna función **Library, Error**, los mensajes de error se muestran en francés.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("LIBRARY,ERROR,GERMAN.WDM");
if WdInt>0 {
	CALLWD("ERROR, File of messages not found");
}
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('LIBRARY, ERROR, GERMAN.WDM');
if WdInt>0 then CALLWD('ERROR,File of messages not found');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("LIBRARY, ERROR, GERMAN.WDM")
if WdInt>0 then
	call CALLWD("ERROR,File of messages not found")
end if
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Library,Error(<File name>)`
---

**`<File name>: (Cadena de caracteres)`**

Nombre y ruta del archivo a cargar. Este archivo se suministra con WINDEV o es creado por WDINT. Su extensión es.WDM. WdInt es mayor que 0 cuando no se encontró &lt;Nombre de archivo&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La biblioteca **L, Error** no puede ser usada para cargar un archivo que no haya sido creado por WDINT.

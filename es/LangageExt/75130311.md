
## Tabla,Col (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice actual de la columna.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table, Col, TabName");
RowNum=WdInt;
```


<a name="Example2"></a>





```txt
// In Pascal
CALLWD('Table, Col, TabName');
RowNum=WdInt;
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Table, Col, TabName")
RowNum=WdInt
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Col(<Table name>)`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la tabla a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
WdInt contiene el número de la fila actual.

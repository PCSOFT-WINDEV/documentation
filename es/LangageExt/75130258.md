
## Pantalla,Coordenadas (Lenguaje externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las coordenadas de la control en relación con el borde superior izquierdo de la ventana, así como el tamaño físico de una control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
// Retrieves the position and size of the image
CALLWD("Screen,Coordinate,IMAGE");
strcpy(Coord,WDString);
```




<a name="Example2"></a>



```txt
// In Pascal
// Retrieves the position and size of the image
CALLWD('Screen,Coordinate,IMAGE');
Coord:=WdString;
```


<a name="Example3"></a>



```txt
// In VB
// Retrieves the position and size of the image
call CALLWD("Screen,Coordinate,IMAGE")
Coord=WdString
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Screen,Coordinate(<Control name> [, <Subscript>])`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de la control cuyas coordenadas deben ser recuperadas.

**`<Subscript>: (Cadena de caracteres (opcional))`**

Índice del control.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las coordenadas se devuelven en WdString en el siguiente formato: ",:,"

Los parámetros (,: y ) se expresan en píxeles. El tamaño de la control tiene en cuenta la frontera exterior de la control.

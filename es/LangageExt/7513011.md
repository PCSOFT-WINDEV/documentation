
## nWDSetParentNext
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Uso
<a name="uso_ELTTEXTE000074"></a>
set el padre de la siguiente ventana a abrir.

Por ejemplo, esta función permite adjuntar una ventana WINDEV a una ventana abierta desde el idioma externo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Sintaxis
<a name="sintaxis_ELTTEXTE000098"></a>

```wl
Int      nWDSetParentNext(HWND hWnd);
```


**Detalles de los parámetros**

**hWnd:** Handle

Manipulación de la ventana principal de la siguiente ventana a abrir. Para asociar una ventana WINDEV a una ventana en lenguaje externo, este parámetro será el manipular de la ventana en lenguaje externo.


## HsMenu,Mark (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Determina si una opción de menú está marcada o no.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
// Is the option corresponding to the LC path marked?
CALLWD("HsMenu,Mark,LC");
if WDInt==1 CALLWD("INFO, The option is marked");
// Is the option #15 marked?
CALLWD("HsMenu,Mark,15");
if WDInt==0 CALLWD("INFO, The option is not marked");
```


<a name="Example2"></a>





```txt
// In Pascal
(* Is the option corresponding to the LC path marked? *)
CALLWD('HsMenu,Mark,LC');
if WDInt=1 then CALLWD('INFO, The option est marked');
(* Is the option #15 marked? *)
CALLWD('HsMenu,Mark,15');
if WDInt=0 then CALLWD('INFO, The option is not marked');
```


<a name="Example3"></a>





```txt
// In VB
' Is the option corresponding to the LC path marked?
call CALLWD("HsMenu,Mark,LC")
if WDInt=1 then  
	call CALLWD("INFO, The option is marked")
end if
' Is the option #15 marked?
call CALLWD("HsMenu,Mark,15")
if WDInt=0 then  
	call CALLWD("INFO, The option is not marked")
end if
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HsMenu,Mark(<Option>)`
---

**`<Option>: (Cadena de caracteres)`**

Este parámetro puede ser:

- un número que corresponde a la índice de la opción, 

- una cadena de caracteres que corresponde a la ruta de acceso de la opción (accesos directos)  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**HsMenu,Mark** devuelve el estado de las marcas de opción en WdInt:

- 0 si la opción no está marcada

- 1 si la opción está marcada

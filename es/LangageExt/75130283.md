
## Listar, Añadir (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un elemento a un Combo Box o control List Box.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD('List,Add,ProdList=%s",CODE);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('List,Add,ProdList='+CODE);
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("List,Add,ProdList="+CODE)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`List,Add(<List to handle> [, <Value>])`
---

**`<List to handle>: (Nombre del control)`**

Nombre de la Caja de Listas o Combo Box que se va a modificar.

**`<Value>: (Cadena de caracteres (opcional))`**

Valor a agregar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el cuadro de lista o control Combo Box no está ordenado, el elemento se añade al final de la lista.

- Si el cuadro de lista o la control Combo Box están ordenados, el valor se añade en el lugar correspondiente dentro del orden de clasificación.

- En el caso de un cuadro de lista ordenado o control Combo Box, se debe utilizar **Lista,Añadir** en lugar de ** Lista,Insertar**.

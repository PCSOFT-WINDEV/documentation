
## Tabla,Guardar (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica o añade un Record en el archivo de datos relacionado al tabla.

**Observación**: Esta función sólo debe utilizarse con las tablas de navegación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Save,ProdTable");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Save,ProdTable');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Table,Save,ProdTable")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Save(<Table name>)`
---

**`<Table name>: (Cadena de caracteres)`**

Nombre de la tabla de navegación a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Table,Save** inicializa las variables de los elementos del archivo con los valores de los controles de la fila seleccionada de la tabla. El Record está escrito en el archivo de datos.

- **Table, Save** no gestiona los elementos de los ficheros relacionado.. HAdd o HModify debe ser usado para escribir en el archivo relacionado.

- Para modificar el archivo de datos relacionado a un tabla directamente, no debe usar HAdd (o HModify): debe utilizar [Tabla,Añadir](../LangageExt/75130309.md), [Tabla,Insertar](../LangageExt/75130313.md) o [Tabla,Modificar](../LangageExt/75130314.md) y [Tabla,Guardar](../LangageExt/75130312.md).

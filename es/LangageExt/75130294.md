
## Mensaje (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje en la barra de estado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Message,Enter the requested information");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Message,Enter the requested information');
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Message,Enter the requested information")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Message(<Message Content>)`
---

**`<Message Content>: (Cadena de caracteres)`**

Mensaje a mostrar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El mensaje permanece en pantalla hasta el siguiente mensaje que aparece en la barra.

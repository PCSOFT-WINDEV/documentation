
## Tabla, Añadir (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un elemento a una control Tabla.

**Observación**: Para añadir un elemento en una posición específica, debe utilizar [Tabla,Insertar](../LangageExt/75130313.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Table,Add,ProductTable=%s",Row);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Table,Add,ProductTable='+Row);
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Table,Add,ProductTable="+Row)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Table,Add(<Table control> , <Row value>)`
---

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.

**`<Row value>: (Cadena de caracteres)`**

Valor de la fila a añadir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Añadir filas a un control Tabla poblado de forma programática
<a name="anadir_filas_control_tabla_poblado_forma_programatica_ELTPARAGRAPHE000048"></a>

&lt;Valor de la Fila&gt; se añade al último fila del control Tabla poblado programáticamente, aunque no se muestre.

Después de llamar a la función **Table,Add**, la tabla no está en modo de edición: para introducir valores en el control Tabla, debe llamar al [Pantalla,Entrada](../LangageExt/75130265.md).
<a name="NOTE0_2"></a>




### Añadir registros a una control Tabla basada en un archivo de datos
<a name="anadir_registros_una_control_tabla_basada_archivo_datos_ELTPARAGRAPHE000061"></a>

&lt;Valor de fila&gt; se agrega a la primera fila disponible de la tabla o, si no hay ninguna fila disponible, a la última fila visible de la tabla.

El archivo no se modifica: Llame a [Tabla,Guardar](../LangageExt/75130312.md) para modificar el archivo.

Después de llamar a la función **Table,Add**, la tabla no está en modo de edición: para introducir valores en el control Tabla, debe llamar al [Pantalla,Entrada](../LangageExt/75130265.md).

En una tabla basada en un archivo de datos, debe llamarse a [Tabla,Guardar](../LangageExt/75130312.md) o [Tabla,Borrar](../LangageExt/75130320.md) después de **Table,Add** para poder llamar de nuevo a **Table,Add** (o [Tabla,Insertar](../LangageExt/75130313.md)). De lo contrario, el contenido de la fila se pierde.

Si la control Tabla contiene varias columnas, los valores de inicialización deben estar separados por el valor del carácter [ TAB] (código binario 9):

- Ejemplo en C:
	
	```txt
	CALLWD("Table,Add,CustomerTab=SMITH\tJohn");
	```


- Ejemplo en Pascal:
	
	```txt
	CALLWD('Table,Add,CustomerTab=SMITH'+#9+'John');
	```


- Ejemplo en VB:
	
	```txt
	call CALLWD("Table,Add,CustomerTab=SMITH"+CHR$(9)+"John")
	```

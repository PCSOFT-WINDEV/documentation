
## Sí,No (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un cuadro de diálogo que incluye un logotipo, un mensaje y los botones "Sí" y "No".. El usuario debe validar uno de los botones ("Sí" o "No") para cerrar la ventana.

Si la respuesta es "Sí", WdString contiene "O" y WdInt contiene 1.

Si la respuesta es "No", WdString contiene "N" y WdInt contiene 0.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("YesNo,Do you want to exit from the program");
if WDInt=1 Exit_Program();
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('YesNo,Do you want to exit from the program');
if WdInt=1 then Exit_Program;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("YesNo,Do you want to exit from the program")
if WdInt=1 then gosub Exit_Program
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Yes,No(<Message>)`
---

**`<Message>: (Cadena de caracteres)`**

Mensaje a mostrar (hasta 200 caracteres).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para visualizar el mensaje en varias líneas, utilice la siguiente sintaxis:

Ejemplo en C:


```txt
CALLWD("YesNo,  \n ");
```


Ejemplo en Pascal:


```txt
CALLWD('YesNo, '+Chr(10)+' ');
```


Ejemplo en VB:


```txt
call CALLWD("YesNo, "+CHR(10)+"")
```

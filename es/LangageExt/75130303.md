
## get (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el valor de un control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Get,Name");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Get,Name');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("Get,Name")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Get(<Control name> [, <Subscript>])`
---

**`<Control name>: (Cadena de caracteres)`**

 Nombre de la control a recuperar.

**`<Subscript>: (Cadena de caracteres (opcional))`**

Subíndice para una control subíndice .

<a name="NOTE0_1"></a>

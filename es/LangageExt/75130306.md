
## WavPlay (Idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reproduce el sonido solicitado. El intérprete de sonido debe ser inicializado por [SoundInit](../LangageExt/75130305.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("WavPlay,DING.WAV");
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('WavPlay,DING.WAV');
```


<a name="Example3"></a>





```txt
// In VB
call CALLWD("WavPlay,DING.WAV")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WavPlay(<Sound Name>)`
---

**`<Sound Name>: (Cadena de caracteres)`**

Nombre del archivo de sonido a reproducir. Este archivo es un archivo ".WAV" o ".VOC"..

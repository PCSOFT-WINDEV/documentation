
## Opción, versión (idioma externo)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra el número de versión de WINDEV wdxxxwm.dll.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```txt
// In C
CALLWD("Option,Version");
strcpy(Version,WDString);
```


<a name="Example2"></a>



```txt
// In Pascal
CALLWD('Option,Version');
Version:=WdString;
```


<a name="Example3"></a>



```txt
// In VB
call CALLWD("Option,Version")
Version=WdString
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Option,Version()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El número de versión está contenido en WdString en el siguiente formato: 9.99X (por ejemplo: 1.05A).

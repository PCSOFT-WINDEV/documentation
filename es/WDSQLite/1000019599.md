
## Conector Nativo SQLite
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
WINDEV y WEBDEV 28 incluyen un conector SQLite nativo. Esta Conector Nativo permite acceder a una base de datos SQLite desde un programa WLanguage sin utilizar un controlador externo.

SQLite es un sistema de gestión de archivos indexados. 

El Conector Nativo to SQLite está disponible: 

- en Windows de 32 y 64 bits.

- en Linux de 32 y 64 bits. 






<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000141"></a>


### Condiciones de uso del Native SQLite Connector
<a name="condiciones_uso_del_native_sqlite_connector_ELTPARAGRAPHE000033"></a>

- SQLite versión 3



<a name="NOTE3_2"></a>


### Archivos necesarios
<a name="archivos_necesarios_ELTPARAGRAPHE000040"></a>

El uso del conector SQLite nativo requiere el archivo wd28 0sqlite.dll. Este archivo es necesario para la conexión nativa a una base de datos SQLite.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000171"></a>


### Usando el Native SQLite Connector en sus aplicaciones WINDEV o sitios WEBDEV
<a name="usando_native_sqlite_connector_sus_aplicaciones_windev_sitios_webdev_ELTPARAGRAPHE000051"></a>

Para usar SQLite en modo nativo, debe programar:

- con las funciones "SQL" de WLanguage (véase [Programación con funciones SQL](../WDSQLite/1000019634.md)).

- con las funciones "HFSQL" de WLanguage (véase [Programación con funciones HFSQL](../WDSQLite/1000019624.md)).




**Observaciones**: 

- No se olvide de leer el documento [limitaciones del conector nativo SQLite](../WDSQLite/1000019635.md).

- Para desplegar sitios que utilicen el Conector SQLite Nativo, este último debe estar instalado en el servidor (Servidor de Aplicaciones WEBDEV).

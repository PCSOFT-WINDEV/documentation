


## Controles Combo Box y List Box: Funcionalidades automáticas (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000236"></a>
El Cuadro de la Lista y los controles de la Combo Box pueden beneficiarse de varios AAF (Características de la aplicación de la automatic): 

- **Para el control Combo Box**: 

	- [Clasificación de los elementos](#NOTE2_1).

	- [Almacenamiento del valor seleccionado](#NOTE3_1). 

	- [Búsqueda en la lista ampliada](#NOTE3B_1).




- **Para el control List Box**: 

	- [Clasificación de los elementos](#NOTE2_1). 

	- [Almacenamiento del valor seleccionado](#NOTE2_1). 

	- [Copiar la fila](#NOTE4_1).

	- [Copiar todas las líneas](#NOTE3_1).

## Opción de orden
<a name="opcion_orden_ELTTEXTE000260"></a>


### Condiciones necesarias
<a name="condiciones_necesarias_ELTPARAGRAPHE000050"></a>

Esta característica es muy useful si el cuadro de lista es bastante largo y si el desarrollador olvidó ordenar el contenido.

Para que esta característica esté disponible, deben cumplirse las siguientes condiciones:

- El List Box o control Combo Box debe ser llenado por programación o por un archivo cargado en memoria.

- El List Box o control Combo Box no debe estar ya clasificado.





### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000060"></a>

Para usar la función automatic para ordenar un Cuadro de Lista o control Combo Box, selecciona "Ordenar" en el el menú contextual del control.


### Implementación
<a name="implementacion_ELTPARAGRAPHE000065"></a>

La opción de clasificación en los controles Combo Box y List Box se implementa automáticamente desde la versión 15.

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Para obtener más información, consulte [Configuración de los menús de AAF](../Editeurs/2010040.md).

**A través de la programación**:

- [AAFDisable](../WDLang1/1000022018.md) con la constante *aafSortListOrCombo* elimina la opción de ordenación del menú contextual del Cuadro de Lista especificado o control Combo Box.

- [AAFChangeCaption](../WDLang1/1000022100.md) se utiliza para modificar el título de la opción de ordenación.

- el [clasificadas](../Proprietes/2510115.md) Property determina si la Caja de Listas o el control Combo Box están ordenados.

## Guardar valores
<a name="guardar_valores_ELTTEXTE000296"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000094"></a>

Para guardar el valor seleccionado en un Cuadro de Lista o control Combo Box, seleccione "Recordar valor" en el menú contextual del control.


### Implementación
<a name="implementacion_ELTPARAGRAPHE000099"></a>

La opción "Recordar valor" está disponible automáticamente en los controles Combo Box y List Box. Esta opción permite gestionar la [persistencia de los datos](../WDChamp/9000019.md). 

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Para obtener más información, consulte [Configuración de los menús de AAF](../Editeurs/2010040.md).

**Mediante programación**: [AAFDisable](../WDLang1/1000022018.md) (o [DisableAAF](../WDLang1/3013061.md)) con la constante *aafSaveValue* elimina la opción "Recordar valor" del menú contextual del Cuadro de Lista especificado o control Combo Box.

**Observación**: También puede guardar el valor a través de la opción "Recordar valor" de la ventana del control description (pestaña "Detalles"). Esta opción se utiliza para guardar el valor incluso si la opción de menú no fue activada por el usuario. La opción de menú tiene prioridad si fue habilitada por el usuario.

## Búsqueda en la lista ampliada
<a name="busqueda_lista_ampliada_ELTTEXTE000326"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000127"></a>

Para buscar en la lista ampliada de la control Combo Box, simplemente: 

- desglose la lista de la control Combo Box,

- empiece a introducir el texto a buscar. Aparece un área de entrada que le permite introducir el elemento deseado.. 





### Implementación
<a name="implementacion_ELTPARAGRAPHE000135"></a>

La búsqueda en las listas ampliadas está disponible automáticamente en los controles de Combo Box. 

**Mediante programación**: [AAFDisable](../WDLang1/1000022018.md) con la constante *aafFindInCombo* desactiva la búsqueda del control Combo Box especificado.

**Observación**: También es posible ofrecer un cuadro de búsqueda por defecto en la lista de una control Combo Box a través de "Con búsqueda", en la ventana de control description (pestaña "Detalles"). Esta opción también puede ser activada por el [SearchAAF](../Proprietes/1000023445.md) Property.

## Copiar la fila o todas las líneas
<a name="copiar_fila_todas_las_lineas_ELTTEXTE000356"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000159"></a>

Para copy el contenido del fila seleccionado en un control List Box, seleccione "copy fila" en el menú contextual. El contenido de fila se copia en el portapapeles. 

Para copy el contenido de todas las filas de una control List Box, seleccione "copy todo" en el menú contextual. El contenido de todas las filas se copia en el portapapeles. 


### Implementación
<a name="implementacion_ELTPARAGRAPHE000168"></a>

Las opciones "copy fila" y "copy todos" están disponibles automáticamente en los controles del cuadro de lista. 

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Para obtener más información, consulte [Configuración de los menús de AAF](../Editeurs/2010040.md).

**Mediante programación**: 

- [AAFDisable](../WDLang1/1000022018.md) associated con:

	- la constante *aafCopy* se utiliza para desactivar el copy de un fila. 

	- la constante *aafCopyAll* se utiliza para desactivar el copy de todas las filas.  




- [AAFChangeCaption](../WDLang1/1000022100.md) se utiliza para modificar el texto de las opciones del menú ("copy fila" y "copy todo").

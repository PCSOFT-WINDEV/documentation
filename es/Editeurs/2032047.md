


## Crear una consulta de eliminación 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000153"></a>
<a name="Delete_Query"></a>
Una **consulta de eliminación** (instrucción SQL [BORRAR](../Editeurs/2034001.md)) permite eliminar registros de un archivo de una base de datos.

Para realizar este borrado, las características principales de la consulta de borrado son las siguientes:

- el archivo de datos de la que se borrarán los datos.

- las condiciones de selección de los registros a borrar (clientes que viven en Seattle, productos cuyo precio es superior a 40 dólares, etc.).




Para más detalles sobre las condiciones de selección de la consulta, véase [Condiciones de selección de una consulta](../Editeurs/2032019.md).

Por ejemplo, elimine los pedidos realizados antes del 1 de enero de 2010:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_RequeteSuppression.gif)
 El editor de consultas genera automáticamente el código SQL de esta consulta. 

**Para ver el código SQL de una consulta**: 

1. Abrir el menú contextual del gráfico de consulta.

2. Seleccione "Código SQL".




También tiene la posibilidad de presionar la tecla F2. 


```sql
DELETE FROM 
	ORDERS
WHERE 
	ORDERS.OrderDate < '20100101'
```

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000195"></a>
Para crear una nueva consulta de borrado:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Consulta".

	- El asistente de creación de consultas se abre.




2. Especifique que desea crear una consulta de borrado con la opción asistente (Borrar).

3. Si su proyecto es relacionado a ningún análisis, especifique el análisis al que se adjuntará la consulta.

4. Pase a la etapa siguiente del asistente.

5. Seleccione el archivo de datos de la que se borrarán los datos.
	Pase a la etapa siguiente.

6. Defina las condiciones de selección haciendo clic en la columna "Condición" de las posiciones relevantes.. Para más detalles sobre las condiciones de selección de la consulta, véase [Condiciones de selección de una consulta](../Editeurs/2032019.md).
	Pase a la etapa siguiente.

7. Especifique (si es necesario) la secuencia de las condiciones de selección con los operadores condicionales AND y OR AND:

	- *Condicional Y operador*: los registros seleccionados coincidirán con ambas condiciones. Por ejemplo, los clientes cuyo nombre comienza con la letra "A" Y que viven en Denver.

	- *Operador condicional O*: los registros seleccionados coincidirán con una de las dos condiciones o con ambas condiciones. Por ejemplo, la consulta selecciona los clientes que viven en Seattle O en Denver.
			Pase a la etapa siguiente.




8. Escriba el nombre de la consulta (nombre del archivo ".WDR" correspondiente a la consulta). Este nombre se utilizará para identificar la consulta en sus programas.

9. Especifique el título de la consulta describiendo brevemente el propósito de la consulta. Este epígrafe se utilizará en la documentación del proyecto.

10. Validar la consulta description. La representación gráfica de la consulta se muestra automáticamente en el editor de consultas.

## Notas
<a name="notas_ELTTEXTE000219"></a>


- Para modificar todas las características de una consulta: en la pestaña "Consulta", en el grupo "Edición", haga clic en "Descripción". Para obtener más información, consulte [Definir una consulta](../Editeurs/2032038.md).

- Para generar el código SQL correspondiente a esta consulta, seleccione "Código SQL" en el menú contextual de la consulta. Si se realizan cambios en la consulta description, estos cambios se reflejarán automáticamente en el código SQL correspondiente.

- También tiene la posibilidad de crear una consulta a partir de uno o más ficheros de datos descritos en la editor de análisis: Para ello, arrastre los archivos de datos seleccionados en el panel "Análisis" y suéltelos en el editor de consultas.

- La prueba de la consulta creada se puede ejecutar desde el editor de consultas. Para obtener más información, consulte [Ejecutar la prueba de una consulta](../Editeurs/2019011.md).

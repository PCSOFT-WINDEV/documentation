


## Importar elementos de DreamWeaver a un sitio WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000166"></a>
WEBDEV le permite importar elementos de Dreamweaver a su sitio WEBDEV. Usted tiene la capacidad de importar: 

- [una biblioteca de Dreamweaver](#NOTE2_1): la biblioteca se convertirá automáticamente en control modelo. 

- [a Dreamweaver página modelo](#NOTE2_2): la página de Dreamweaver modelo se convertirá automáticamente en la página WEBDEV modelo. 

- [una página de Dreamweaver](#NOTE2_3): la página se importa con sus bibliotecas y plantillas. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000190"></a>


### Importar una biblioteca de Dreamweaver
<a name="importar_una_biblioteca_dreamweaver_ELTPARAGRAPHE000028"></a>

Para importar una biblioteca de Dreamweaver a un sitio WEBDEV:

1. Abra el proyecto WEBDEV en el que debe importarse la página.

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Una biblioteca Dreamweaver".

3. Especifique el ruta o URL de la biblioteca a importar (archivo".lbi"). Se muestra la vista previa.

4. Seleccione el modo de importación. 

5. Validar. La biblioteca se incluye automáticamente en el proyecto actual de WEBDEV como control modelo.



<a name="NOTE2_2"></a>


### Importar una página de Dreamweaver modelo
<a name="importar_una_pagina_dreamweaver_modelo_ELTPARAGRAPHE000053"></a>

Para importar una página HTML a un sitio WEBDEV:

1. Abra el proyecto WEBDEV en el que debe importarse la página.

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Un modelo de páginas Dreamweaver".

3. Especifique la ruta o URL de la modelo que desea importar. Se muestra la vista previa.

4. Seleccione el modo de importación. 

5. Validar. La página modelo se incluye automáticamente en el proyecto WEBDEV actual.. Si el modelo utiliza una biblioteca o un "template", estos elementos de detectan automáticamente. 

6. Si las plantillas de control o las plantillas de página ya se encuentran en el proyecto, estos elementos son automáticamente reutilizados por el modelo importado. 

7. Si las plantillas de control o las plantillas de las páginas no se encuentran en el proyecto: 

	- Si las fuentes de las bibliotecas y plantillas están disponibles, la importación de estos elementos se propone automáticamente.. 

	- Si las fuentes de las bibliotecas y plantillas no están disponibles o si se ha rechazado la importación, se realiza un análisis inverso de la página.: las plantillas de página y plantillas de control necesarias se crean automáticamente y associated con la página.






<a name="NOTE2_3"></a>


### Importar una página de Dreamweaver
<a name="importar_una_pagina_dreamweaver_ELTPARAGRAPHE000083"></a>

Para importar una página HTML a un sitio WEBDEV:

1. Abra el proyecto WEBDEV en el que debe importarse la página.

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Una página HTML".
	**Observación**: También tiene la posibilidad de arrastrar y soltar la página al editor de páginas de WEBDEV.

3. Especifique el ruta o la URL de la página HTML a importar. Se muestra la vista previa de la página HTML a importar.

4. Seleccione el modo de importación. Los estilos y las imágenes associated con la página importada pueden ser manejados según dos métodos:

	- Si se selecciona "Importar una página independiente al proyecto":

		- se crea una hoja de estilo WEBDEV (archivo ".WWS") y automáticamente associated con la página importada. Esta hoja de estilo contiene todos los estilos CSS de la página inicial.

		- las diferentes imágenes de associated con la página importada serán copiadas a un directorio específico (subdirectorio "&lt;PageName&gt;" del directorio del proyecto Default).




- Si se selecciona "Importar varias páginas del mismo sitio al proyecto":

	- los estilos de la página importada se añaden automáticamente a la hoja de estilo associated con el proyecto.

	- las diferentes imágenes associated con la página importada se copiarán en el directorio "&lt;Nombre del proyecto&gt;_WEB" del proyecto.

5. Validar. La página HTML se incluye automáticamente en el proyecto WEBDEV actual.. Si la página utiliza una biblioteca o un "modelo", estos elementos se detectan automáticamente. 

6. Si las plantillas de control o las plantillas de la página ya se encuentran en el proyecto, estos elementos son automáticamente reutilizados por la página importada. 

7. Si las plantillas de control o las plantillas de las páginas no se encuentran en el proyecto: 

	- Si la fuente de las bibliotecas y plantillas está disponible, la importación de estos elementos se propone automáticamente.. 

	- Si el origen de las librerías y plantillas no está disponible o si se ha rechazado la importación, se realiza un análisis inverso de la página.: las plantillas de página y plantillas de control necesarias se crean automáticamente y associated con la página.

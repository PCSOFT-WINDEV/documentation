## Presentación
<a name="presentacion_ELTTEXTE000204"></a>
La ventana del proyecto description le permite configurar varios elementos de su proyecto. Esta ventana incluye las siguientes pestañas:

- La pestaña ["Proyecto"](../Editeurs/2030046.md), que permite introducir información general sobre el proyecto. 

- La pestaña "Análisis", que permite asociar un análisis al proyecto y/o a un modelo UML.. 

- La pestaña ["Archivos"](../Editeurs/2030052.md), que permite introducir información sobre los archivos de datos utilizados por la aplicación (sólo si el proyecto está vinculado a un análisis).

- La pestaña "Datos en vivo", que permite configurar cómo se muestran los datos de los ficheros de datos en los distintos editores.

- La pestaña ["Idiomas"](../Editeurs/9500219.md), que se utiliza para configurar los diferentes idiomas utilizados por el proyecto, así como las opciones que se utilizarán para cada idioma.

- La pestaña **"Estilo"** , que le permite elegir el plantilla , personalizar la vista previa de impresión y seleccionar una hoja de estilo. También puedes configurar el tamaño de la pantalla.

- La pestaña "SCM", que sirve para gestionar el trabajo en equipo (SCM o GIT).

- La pestaña ["Opciones"](../Editeurs/9500227.md), que permite configurar las opciones de actualización, copia de seguridad y sincronización de modelo. Esta pestaña también se utiliza para habilitar el modo estricto y para definir el estilo de código.

- La pestaña ["Avanzado"](../Editeurs/9500236.md), que permite configurar cómo utilizar "Reports & Queries", las AAF y cómo bloquear la aplicación.

- La pestaña ["Recopilación"](../Editeurs/9500218.md), que permite configurar las opciones de compilación.

- La pestaña ["Telemetría"](../Editeurs/9500223.md), que permite configurar las opciones de telemetría del proyecto.

## Ficha Estilo
<a name="ficha_estilo_ELTTEXTE000234"></a>


### Plantilla del proyecto
<a name="plantilla_del_proyecto_ELTPARAGRAPHE000054"></a>

 La pestaña "Estilo" de la ventana description del proyecto permite identificar y definir el plantilla utilizado por el proyecto. Una vista previa muestra la plantilla en uso.

Observación: Si se modifica la plantilla a través de la ventana del proyecto description, esta modificación sólo se tiene en cuenta para las nuevas ventanas. Para aplicar una plantilla a las ventanas existentes, vaya a la pestaña "Proyecto", grupo "Otras acciones", y haga clic en "Aplicar una plantilla al proyecto".. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La opción " Ventanas del sistema Info, YesNo, Confirm y Dialog (configuraciones WINDEV) " permite personalizar las ventanas de información del sistema, utilizando las AAF o integrando las ventanas correspondientes directamente en el proyecto. Para obtener más información, consulte [Personalización de las ventanas de información del sistema (o cuadros de diálogo)](../WDLang1/3021003.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La opción "Aplicar tema de la plantilla a los componentes integrados en el proyecto y a las ventanas de las FAA" permite utilizar el tema del proyecto plantilla en todos los elementos incluidos en el proyecto.: componentes, ventanas específicas de AAF, etc.



![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La opción "Tooltip" se utiliza para definir cómo mostrar los tooltips en su aplicación. En Default, el modo de visualización es "Estándar".. Puede personalizar la información sobre herramientas de la aplicación. Para obtener más información, consulte [Tooltips personalizados](../Editeurs/9500233.md). 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Visor de reportes
<a name="visor_reportes_ELTPARAGRAPHE000140"></a>

El visor reporte permite a los usuarios finales de una aplicación WINDEV visualizar los informes antes de imprimirlos. Este visor de reporte se puede personalizar para que coincida con el aspecto de su aplicación. Para obtener más información, consulte [Personalizar el visor de reporte (vista previa)](../WDChamp/2024002.md). 


### Hojas de estilo
<a name="hojas_estilo_ELTPARAGRAPHE000148"></a>

Esta opción permite seleccionar la(s) hoja(s) de estilo associated con el proyecto. 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Tamaño de la ventana al crearla
<a name="tamano_ventana_crearla_ELTPARAGRAPHE000155"></a>

Esta opción permite seleccionar el tamaño que propondrá Default al crear ventanas en el editor de ventanas.

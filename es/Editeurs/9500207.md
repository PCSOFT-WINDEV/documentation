


## Editor de análisis : Describir una conexión
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000168"></a>
Cualquier programa desarrollado con WINDEV, WEBDEV o WINDEV Mobile puede acceder fácilmente:

- una base de datos HFSQL.

- una base de datos que no sea HFSQL.




En cualquier caso, todo lo que tiene que hacer es describir la estructura de la base de datos en el análisis.

Las conexiones se usan para enlazar la description de un archivo de datos con los datos reales. Por ejemplo, una conexión se crea automáticamente cuando se importa un archivo perteneciente a una base de datos externa en editor de análisis.

Se crean las conexiones:

- al importar un archivo de datos que pertenece a una base de datos existente. Para obtener más información, consulte [Manipular una base de datos externa](../Editeurs/2011025.md).

- desde el editor de análisis: en la pestaña "Análisis", en el grupo "Conexión", haga clic en "Conexiones".

## Crear una conexión
<a name="crear_una_conexion_ELTTEXTE000198"></a>


### Creación de una conexión (Editor de análisis)
<a name="creacion_una_conexion_editor_analisis_ELTPARAGRAPHE000047"></a>

Para crear una conexión en el editor de análisis:

1. En la pestaña "Análisis", en el grupo "Conexión", haga clic en "Nueva conexión".

2. En la creación de la conexión asistente, seleccione el tipo de conexión que desea crear:

	- HFSQLDrive (solución de pago). 

	- PCSCloud (solución de pago).

	- HFSQL Client/Server.

	- HFSQL Classic.

	- una base de datos externa.




3. **Al crear una conexión para HFSQLDrive**, especifique las características de su base de datos HFSQLDrive en los diferentes pasos del asistente (para más detalles sobre HFDSQLDrive, véase [HFSQLDrive, su base de datos en la nube en 3 clics](https://pcscloud-drive.net/UK/HFSQLDrive/index.awp)):

	- el nombre del servidor o la IP Address

	- el número de puerto del servidor.

	- el nombre de usuario y la contraseña.

	- el nombre de la base de datos.

	- el nombre de la conexión.




4. **Al crear una conexión para PCSCloud**, especifique las características de su cuenta de PCSCloud en los diferentes pasos del asistente (para más detalles sobre PCSCloud, consulte [Nube para bases de datos HFSQL](https://pcscloud.net/UK/cloud_applications.awp)):

	- el nombre del servidor o la IP Address

	- el número de puerto del servidor.

	- el nombre de usuario y la contraseña.

	- el nombre de la base de datos.

	- el nombre de la conexión.




5. **Al crear una conexión para HFSQL Client/Server**, indique las diferentes etapas del asistente:

	- el nombre del servidor o la IP Address

	- el número de puerto del servidor.

	- el nombre de usuario y la contraseña.

	- el nombre de la base de datos.

	- el nombre de la conexión.




6. **Al crear una conexión para HFSQL Classic**, especifique lo siguiente en los diferentes pasos del asistente:

	- el nombre de la conexión.

	- el directorio de archivos de datos de Default. 




7. **Al crear una conexión a una base de datos externa**, indique las diferentes etapas del asistente:

	- el tipo de acceso utilizado para la conexión

	- el nombre o la IP Address del servidor o el archivo ruta.

	- el nombre de usuario y la contraseña.

	- el nombre de la base de datos.

	- el nombre de la conexión.







Una vez creada, la conexión puede asociarse a un archivo de datos en la ventana el archivo de datos description

## Beneficio de las conexiones de HFSQL Classic
<a name="beneficio_las_conexiones_hfsql_classic_ELTTEXTE000222"></a>


- El uso de una conexión para los archivos de datos HFSQL Classic facilita el cambio de conexiones. Para utilizar una base de datos HFSQL Client/Server o una base de datos Oracle, por ejemplo, basta con cambiar la conexión definida en el análisis (con [HChangeConnection](../WDLang4/3044150.md), por ejemplo).

- Se recomienda utilizar una conexión HFSQL Classic cuando se utilice la replicación universal asistida

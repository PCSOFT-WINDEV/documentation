<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000275"></a>
Al instalar la aplicación, usted tiene la capacidad de:

- [sugieren guardar los elementos sustituidos por la instalación](#NOTE2_1). 

- [mostrar un resumen de las opciones elegidas para instalación](#NOTE3_1).

- [visualizar el paso con la aplicación directa instalación sugerencia](#NOTE4_1).

- [realizar una instalación silenciosa](#NOTE4_1_Silent). 

- [mostrar un contrato de licencia](#NOTE5_1).

- [mostrar un archivo "Léame](#NOTE6_1).




Por Default: 

- el resumen de las opciones elegidas para la instalación se mostrará antes de iniciar la instalación.

- el asistente sugiere la aplicación directa instalación. 




Estos parámetros están configurados en WDInst (editor de instalación). Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

## Mostrando un resumen de las opciones elegidas para instalación
<a name="mostrando_resumen_las_opciones_elegidas_para_instalacion_ELTTEXTE000329"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000085"></a>

Antes de iniciar la instalación, tiene la posibilidad de mostrar un resumen de las opciones seleccionadas por el usuario. Este resumen contiene la siguiente información:

- el directorio de instalación de la aplicación.

- el tipo de instalación (completo, mínimo o personalizado).

- la lista de componentes instalados (para una instalación personalizada).

- los detalles de las modificaciones realizadas en los archivos ".INI" y en el archivo "AutoExec.BAT" (si se realizan modificaciones).

- los detalles de las modificaciones realizadas en las claves del registro (si se realizan modificaciones).



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Info_affiches_install%20-%20HC%20N%B0004.gif&type=thumb)

<a name="NOTE3_2"></a>


### Mostrando un resumen de las opciones elegidas para instalación
<a name="mostrando_resumen_las_opciones_elegidas_para_instalacion_ELTPARAGRAPHE000100"></a>

Para mostrar un resumen de las opciones elegidas para instalación:

1. Seleccione "Configurar ajustes ... Asistente".

2. Seleccione "Mostrar la página de resumen antes de iniciar la instalación".

## Sugerir una rápida instalación
<a name="sugerir_una_rapida_instalacion_ELTTEXTE000359"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000117"></a>

El programa instalación puede sugerir: 

- una rápida (o directa) instalación: Si el usuario elige esta opción, la aplicación instalación se realiza automáticamente con las opciones de Default. 

- una instalación personalizada : El usuario puede configurar las diferentes opciones de instalación. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Info_affiches_install%20-%20HC%20N%B0002.gif&type=thumb)


Observación: por Default, se sugiere una rápida instalación. 
<a name="NOTE4_2"></a>


### Sugerir una rápida instalación
<a name="sugerir_una_rapida_instalacion_ELTPARAGRAPHE000131"></a>

Para configurar una instalación rápida:

1. Seleccione "Configurar ajustes ... Asistente".

2. Seleccione "Mostrar la página que propone la instalación directa con un solo clic".




Observación: Si esta opción no está marcada, el asistente no mostrará el paso con la sugerencia de un clic instalación.

## Realizando una silenciosa instalación
<a name="realizando_una_silenciosa_instalacion_ELTTEXTE000389"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000149"></a>

Las configuraciones de la versión de reference y la versión cliente pueden realizarse en modo silencioso. En este caso, la instalación se hará sin ninguna intervención del usuario. Las opciones definidas por Default serán utilizadas automáticamente.
<a name="NOTE4_2_Silent"></a>


### Sugerir (o no) una instalación silenciosa 
<a name="sugerir_una_instalacion_silenciosa_ELTPARAGRAPHE000156"></a>

Para configurar la silenciosa instalación:

1. Seleccione "Configurar ajustes ... Asistente".

2. Marque la opción: 

	- **Instalación cliente silenciosa**: la aplicación instalación para el usuario final no requiere ninguna acción por parte del usuario. 

	- **Implementación silenciosa de la versión de referencia ** (solo disponible en una red instalación con actualización automatic ): la versión reference de la red instalación se instalará sin intervención del usuario. 







Observación: Si estas opciones están desmarcadas, la asistente mostrará todas las opciones necesarias para instalar la aplicación.

## Visualización de un contrato de licencia
<a name="visualizacion_contrato_licencia_ELTTEXTE000419"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000180"></a>

La instalación de su aplicación puede comenzar con la validación de un acuerdo de licencia. El usuario deberá aceptar los términos del acuerdo de licencia antes de poder continuar.

El archivo de la licencia será mostrado por la instalación asistente. Puede ser:

- en formato de texto (".TXT").

- en formato RTF (".RTF" o ".WRI", por ejemplo).



<a name="NOTE5_2"></a>


### Mostrando un acuerdo de licencia al comienzo de instalación
<a name="mostrando_acuerdo_licencia_comienzo_instalacion_ELTPARAGRAPHE000192"></a>

Mostrando el acuerdo de licencia al principio de instalación: 

1. Seleccione "Configurar ajustes ... Asistente".

2. Seleccione un fichero de licencia para cada idioma soportado por el programa instalación (botón "Seleccionar"). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Info_affiches_install%20-%20HC%20N%B0001%201.gif&type=thumb)

## Visualización de un archivo "Léame
<a name="visualizacion_archivo_leame_ELTTEXTE000449"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000205"></a>

Puede pedir al usuario que abra un archivo "Léame" una vez que instalación haya finalizado.. Por ejemplo, este archivo puede proporcionar información sobre las limitaciones del programa, ayudar al usuario a familiarizarse con él, etc.

Al final de instalación, el usuario tendrá que marcar "Mostrar el archivo "Léame"" para leer este archivo.

Este archivo puede ser:

- en formato de texto (".TXT"),

- en formato Word (".DOC"),

- en formato HTML (".HTM" o ".HTML"),

- en formato RTF (".RTF" o ".WRI", por ejemplo),

- en cualquier formato reconocido por la el equipo donde se realiza la instalación.



<a name="NOTE6_2"></a>


### Visualización de un archivo "Léame
<a name="visualizacion_archivo_leame_ELTPARAGRAPHE000222"></a>

Para visualizar un archivo "Léame: 

1. Seleccione "Parámetros de instalación" .. "Fin de la instalación".

2. Seleccione un archivo "Léame" para cada idioma soportado por el programa instalación (botón "Seleccionar"). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options%20install_WDInst%20-%20HC%20N%B0005%201.gif&type=thumb)

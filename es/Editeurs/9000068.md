


## Limpieza del proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
Después de meses o años de desarrollo y mantenimiento, el directorio de su proyecto a menudo contiene varios archivos que ya no se utilizan pero que usted no se atreve a borrar.

Pruebe archivos y ventanas, imágenes inútiles, ... ¡Es hora de limpiar!

Una asistente de WINDEV/WEBDEV/WINDEV Mobile detecta automáticamente estos elementos no utilizados y los elimina del proyecto, sin riesgo alguno.. Los elementos eliminados del proyecto serán archivados (en formato ZIP o en un directorio de backup) para que puedan ser restaurados posteriormente si es necesario....

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000121"></a>
Limpiar el directorio del proyecto:

1. En la pestaña "", en el grupo "", despliegue "" y seleccione "Limpiar directorio del proyecto". El proyecto de limpieza de asistente comienza. Pase a la etapa siguiente.

2. Seleccione los directorios del proyecto en los que se debe realizar la búsqueda de archivos no utilizados.. Los posibles directorios son los siguientes:

	- directorio raíz del proyecto.

	- Directorio"Groupware" del proyecto.

	- de los directorios de las copias de seguridad de los proyectos.

	- "Java" si el proyecto actual puede ser utilizado para generar un archivo Java.




3. Elija el tipo de archivos que se tendrán en cuenta. La asistente enumera todos los archivos encontrados.

4. Seleccione los archivos que se van a archivar.

5. Los archivos se pueden archivar:

	- en un archivo comprimido (en formato ZIP): seleccione "Sí: Cree un archivo ZIP con los archivos".

	- en un directorio de respaldo: seleccione "No: Mueva los archivos a un directorio de respaldo".




6. Especifique el nombre del archivo comprimido que se creará o el directorio al que se moverán los archivos a archivar.. Marque "Añadir la fecha de hoy al nombre del archivo/directorio" para prefijar automáticamente el nombre del archivo o directorio con la fecha de hoy.

7. Validar para archivar los archivos no utilizados.

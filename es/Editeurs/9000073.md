## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
El editor de código propone varias herramientas utilizadas para corregir el código:

- el verificador de automatic de los elementos del proyecto,

- el corrector ortográfico para los comentarios y las cadenas de caracteres. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Comprobador de automatic
<a name="comprobador_automatic_ELTTEXTE000175"></a>
Cuando no se reconoce el nombre de una Variable o de un elemento, el menú contextual del editor de código propone una lista de posibles correcciones. Esta lista presenta el nombre de las variables y/o elementos de proyecto que pueden corresponder.

Simplemente seleccione el nombre deseado.

## Corrector ortográfico
<a name="corrector_ortografico_ELTTEXTE000199"></a>
El corrector ortográfico del editor de código detecta los errores tipográficos:

- los comentarios, 

- las cadenas utilizadas en el código (cadenas simples, mensajes multilingües, diálogos y cadenas multilingües).




Las funciones y comandos de WLanguage se reconocen como ortográficamente correctos.

**¿Qué corrector ortográfico se utiliza?**
El corrector ortográfico utilizado es el que se proporciona con WINDEV/WEBDEV/WINDEV Mobile. Este corrector ortográfico se puede utilizar: 

- para comentarios (en francés o inglés, dependiendo del idioma utilizado para el código).

- para cuerdas multilingües en francés e inglés.


Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md).

El corrector ortográfico está habilitado por Default. 

**Cuerdas multilingües**
El diccionario utilizado es el del idioma mostrado. Para cambiar el idioma, en la pestaña "Vista", en el grupo "Idioma mostrado" despliegue "Idioma mostrado". El corrector ortográfico es una opción global para el usuario.

**Observación**: Por Default, el corrector ortográfico sólo está disponible en inglés y francés. Para utilizar el corrector ortográfico para otros idiomas en cadenas multilingües, basta con añadir los archivos que corresponden a los diccionarios Hunspell deseados en: "\\Programas\\Datos\\Dic". 
Estos diccionarios se pueden encontrar fácilmente en Internet. Los diccionarios contienen un archivo ".dic" y ".aff" con el mismo nombre que deben estar en la misma ubicación.

**Atención**: El corrector ortográfico está desactivado si la cadena multilingüe utiliza un color que difiere del de las cadenas.

**Palabras añadidas al diccionario**
Las palabras que se agregan al diccionario se encuentran en el archivo "C:\\Users\\&lt;User&gt;\\AppData\\Roaming\\PC SOFT\\perso.dic".
<a name="NOTE3_2"></a>


### Desactivar el corrector ortográfico
<a name="desactivar_corrector_ortografico_ELTPARAGRAPHE000087"></a>

Para desactivar el corrector ortográfico:

1. En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. En la pestaña "Código", desmarque las ubicaciones (comentarios o cadenas) en las que no se debe utilizar el corrector ortográfico (opción "Activar el corrector ortográfico").

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000218"></a>
Un sistema de ayuda incluye varias páginas. Las páginas son los principales elementos de un sistema de ayuda. Cuando los usuarios abren la ayuda, son dirigidos a una página de ayuda.

Se pueden implementar varios tipos de páginas de ayuda en un archivo de ayuda, veamos los más comunes:

- **páginas de ayuda sensibles al contexto**: el contenido de la ayuda Page proporciona información sobre el uso de una ventana específica. Generalmente, este tipo de página de ayuda se abre a través de la ayuda Icon ("?") de la ventana.

- **Páginas de ayuda "Cómo hacer"**: estas páginas de ayuda describen paso a paso las diferentes acciones necesarias para realizar una operación.

- **Páginas de ayuda "conceptuales"**: estas páginas de ayuda presentan un concepto o una idea y definen los términos utilizados.




Hay [enlaces](../Editeurs/2010023.md) entre las diferentes páginas. Estos enlaces permiten a los usuarios navegar por el sistema de ayuda.

**Observación**: Para generar automáticamente las páginas de ayuda correspondientes a los distintos elementos del proyecto (ventanas e informes), en la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Generar páginas de ayuda del proyecto".

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000242"></a>


### Crear una ayuda en blanco Page
<a name="crear_una_ayuda_blanco_page_ELTPARAGRAPHE000043"></a>

Para crear una ayuda en blanco Page: 

- Método 1: 

	- Seleccione una Page en el resumen. 

	- En la pestaña "Modificación", en el grupo "Tabla de contenido", despliegue "Nuevo" y seleccione "Nuevo espacio en blanco Page".

	- Se crea automáticamente una nueva Page en blanco que puede modificarse.




- Método 2: Seleccione "Nuevo .. Nuevo espacio en blanco Page" en el menú contextual del resumen. Se crea automáticamente una nueva Page en blanco que puede modificarse.

- 3º método: 

	- En la pestaña "Modificación", en el grupo "Tabla de contenido", despliegue "Nuevo" y seleccione "Nuevo Page".

	- En el asistente, seleccione "Crear un Page en blanco".. 

	- Especifique las características de la Page y valídelas.




- 4º método: Seleccione "Nueva .. Nueva Page" en el menú contextual del resumen. En el asistente, seleccione "Crear un Page en blanco".. Especifique las características de la Page y valídelas.



<a name="NOTE2_2"></a>


### Creando una página de ayuda a partir de un Document existente
<a name="creando_una_pagina_ayuda_partir_document_existente_ELTPARAGRAPHE000087"></a>

Una asistente permite crear una página de ayuda a partir de una Document existente . En esta asistente, seleccione:

- el nombre y la ruta del archivo TXT, HTM o RTF a importar. 

- el nombre de la ayuda Page a crear.




**Atención**: Si el Document importado contiene varios Page s, todos los Page s se importarán en un único Page de ayuda.

Para iniciar la importación de asistente, utilice uno de los siguientes métodos:

- en la pestaña " Modificación ", en el grupo " Tabla de contenido ", expanda " Nuevo " y seleccione "Nuevo Page ". En el asistente, seleccione "Importar un Document existente".

- seleccione "Nueva .. Nueva Page" en el menú contextual del resumen. En el asistente, seleccione "Importar un Document existente".



<a name="NOTE2_3"></a>


### Suprimir una ayuda Page
<a name="suprimir_una_ayuda_page_ELTPARAGRAPHE000116"></a>

Para suprimir una ayuda Page: 

1. Seleccione la ayuda Page en el resumen.

2. Seleccione "Borrar la Page" en el menú contextual del resumen (o pulse la tecla Supr).

## Modificación de las páginas de ayuda
<a name="modificacion_las_paginas_ayuda_ELTTEXTE000278"></a>


### Secciones de una ayuda Page
<a name="secciones_una_ayuda_page_ELTPARAGRAPHE000128"></a>Una ayuda Page incluye varias secciones:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur-Aide_page.gif)
 Todas estas secciones se pueden modificar.
<a name="NOTE3_2"></a>


### En una ayuda Page, puede: 
<a name="una_ayuda_page_puede_ELTPARAGRAPHE000137"></a>

- modificar el texto que se encuentra en la cabecera o en la Page. Basta con introducir el texto en el editor de ayuda. 
	Observación: un corrector ortográfico es Active en la ayuda Page. 

- suprimir la cabecera de Page (pestaña "Apariencia" de [propiedades de la página](../Editeurs/2010024.md)).

- modificar el título del Page que aparece en el sumario (el título del Page se modifica en el [propiedades de la página](../Editeurs/2010024.md)).

- configurar el modo de visualización de la Page, modificando la ventana modelo associated con esta Page por ejemplo. Para obtener más información, consulte [Definir una ventana de ayuda modelo](../Editeurs/2010006.md).

- añadir imágenes (associated con enlaces o no). Para obtener más información, consulte [Insertar una Image en una ayuda Page](../Editeurs/2010026.md).

- añadir vídeos. Para obtener más información, consulte [Insertar un vídeo en una ayuda Page](../Editeurs/2010026.md).

- añadir enlaces. Para obtener más información, consulte [Creación de enlaces entre las páginas de ayuda](../Editeurs/2010023.md).

- asociar las palabras clave del índice con la ayuda de Page. Para obtener más información, consulte [Asociar las palabras clave del índice a una ayuda Page](../Editeurs/2010009.md).

## Presentación
<a name="presentacion_ELTTEXTE000150"></a>
Realizar una búsqueda en el proyecto actual es una operación común: buscar un título, buscar un nombre de control, ... Esta búsqueda se puede realizar:

- a través del panel "Explorador de proyectos",

- a través de la ["Panel "Buscar/Reemplazar](../Editeurs/2027013.md),




También puede ser útil realizar una búsqueda en diferentes proyectos (por ejemplo, para buscar una Procedure utilizada en otro proyecto). Esta búsqueda se puede realizar en cualquier tipo de proyecto (WINDEV, WEBDEV o WINDEV Mobile).

La búsqueda multiproyecto le permite realizar una búsqueda en:

- proyectos específicos, 

- todos los proyectos encontrados en un directorio, 

- todos los proyectos encontrados en un disco o en un servidor.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000174"></a>


### Realización de una búsqueda multiproyecto
<a name="realizacion_una_busqueda_multiproyecto_ELTPARAGRAPHE000041"></a>

Para realizar una búsqueda multiproyecto:

1. Seleccione "Taller... Búsqueda multiproyecto...". Aparece la ventana para configurar los directorios de búsqueda.

2. Especifique los directorios, discos o proyectos afectados por la búsqueda:

	- el botón "Directorio" permite seleccionar los proyectos que se encuentran en un directorio o en un disco. Todos los proyectos WINDEV, WEBDEV o WINDEV Mobile que se encuentren en esta ubicación serán tenidos en cuenta.. La búsqueda también se realiza en los subdirectorios de los lugares especificados. Se puede especificar un horario UNC ruta.

	- el botón "Proyecto" permite añadir un proyecto específico (WINDEV, WEBDEV o WINDEV Mobile). Se pueden añadir diferentes tipos de proyectos (un proyecto WINDEV y un proyecto WINDEV Mobile, por ejemplo).




3. Validar. El panel "Buscar - Reemplazar" se muestra automáticamente y se activa en el editor (si es necesario).

4. En el panel "Buscar - Reemplazar", introduzca:

	- la palabra o la expresión a encontrar, 

	- los criterios de búsqueda (contiene, palabra entera, ....).




5. Iniciar la búsqueda (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_rech.gif)). Si se encuentran elementos, la reference de estos elementos se muestra en el panel "Buscar - Reemplazar".




Observación: La búsqueda multiproyecto también puede realizarse desde el panel "Buscar - Reemplazar".. Para ello, seleccione la opción "En " para configurar los directorios de búsqueda.<a name="NOTE2_2"></a>


### Gestión del resultado de la búsqueda
<a name="gestion_del_resultado_busqueda_ELTPARAGRAPHE000066"></a>

El resultado de la búsqueda multiproyecto se muestra en el panel "Buscar - Reemplazar".. Los resultados se agrupan por proyectos.

La siguiente información se especifica para cada elemento encontrado:

- su ubicación, 

- su valor.




El menú contextual del resultado de la búsqueda se utiliza para:

- copy la lista de elementos encontrados en el portapapeles.

- imprimir la lista de elementos encontrados.




Para acceder directamente a uno de los elementos encontrados, haga doble clic en la ubicación del elemento.

**Observación**: Si el elemento encontrado pertenece a un proyecto distinto al proyecto actual:

- Si el proyecto se desarrolló con el mismo producto, el proyecto se abre en una nueva instancia del producto.

- si el proyecto se desarrolló con un producto diferente, el proyecto se abre en el producto associated.

### Criterios de búsqueda
<a name="criterios_busqueda_ELTPARAGRAPHE000094"></a>

Independientemente del tipo de búsqueda realizada, puede especificar una combinación de los siguientes criterios de búsqueda:

- Contiene: la expresión buscada debe contener la expresión especificada.

- Comienza por: la expresión buscada comienza con la expresión especificada.

- Palabra completa: la expresión buscada corresponde exactamente a la expresión especificada.

- Ignorar la puntuación: la búsqueda ignora la puntuación (coma, punto, etc.).

- Ignorar las casas: la búsqueda ignora las mayúsculas y minúsculas.

- Ignorar los caracteres acentuados: la búsqueda ignora los caracteres acentuados.




## Despliegue de una aplicación a través de App Store
			



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000228"></a>
El despliegue en la App Store es el modo más común de implementación. 

Se utiliza cuando la aplicación está destinada al público en general public. En este modo, las instalaciones y actualizaciones están totalmente gestionadas por la App Store de Apple.

**Beneficios**:

- instalación simple para el usuario.

- Actualizaciones de automatic.


**Inconvenientes**:

- Aplicación accesible para cualquier usuario.

- Apple comprueba la aplicación automática y manualmente antes de su publicación y puede rechazar o eliminar una aplicación según sus propios criterios.

## Generación de la aplicación
<a name="generacion_aplicacion_ELTTEXTE000276"></a>
Le aconsejamos que lea: 

- la lista de condiciones previas especificadas por Apple: [https://developer.apple.com/appstore/resources/submission/index.html](https://developer.apple.com/appstore/resources/submission/index.html)

- la ayuda Page [Principio para el desarrollo de una aplicación iPhone/iPad](../Editeurs/1000019863.md). 




Atención: Es necesario: 

- Proporcione 3 imágenes PNG que serán usadas como Icon para su aplicación. Estas imágenes deben tener las siguientes dimensiones: 57x57, 72x72 y 114x114. 
	Consejo: Crea un Image de 512x512 e incluye este Image en tu catálogo Image personalizado. Al generar la aplicación, seleccione esta Image en el catálogo: WINDEV Mobile construirá las 3 imágenes a las dimensiones adecuadas.

- Proporcionar 4 imágenes en formato PNG que se utilizarán como pantalla de inicio, con las siguientes dimensiones: 320x480, 640x960, 768x1004 y 1024x748




Para generar la aplicación en WINDEV Mobile, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Generation_iOS_WM_bl.gif) en los botones de acceso rápido.

En la pantalla siguiente, inserte el App Id suministrado por Apple: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Applestore_dep1.gif&type=thumb)


Para encontrar el Bundle, consulte la siguiente Page: [https://developer.apple.com/membercenter/index.action](https://developer.apple.com/membercenter/index.action)

Una vez conectado, haga clic en la opción "Certificados, identificadores y perfiles"..<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=23_IOS_Deploiement1.gif)
 Para crear la licencia de distribuidor:

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=23_IOS_Deploiement2.gif)


- La asistente para crear un nuevo perfil comienza. Seleccione: 

	- el tipo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=23_IOS_Deploiement3.gif)


	- el identificador: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=23_IOS_Deploiement4.gif)

## Preparando la aplicación para implementación en la App Store
<a name="preparando_aplicacion_para_implementacion_app_store_ELTTEXTE000300"></a>


### Imágenes necesarias
<a name="imagenes_necesarias_ELTPARAGRAPHE000109"></a>

Las siguientes imágenes son necesarias para implementar la aplicación:

- Aplicación Icon en JPG/PNG (RGB) (512x512 o 1204x1024)

- Captura de pantalla de la aplicación para el iPhone en JPG/PNG (RGB) 72DPI (960x640, 960x600, 640x960, 640x920, 640x1136, 640x1096, 1136x640, 1136x600)

- Captura de pantalla de la aplicación para iPad en JPG/PNG (RGB) (1024x768, 1024x748, 768x1024, 768x1004, 2048x1536, 2048x1496, 1536x2048 o 1536x2008)

- Si la aplicación es associated con una aplicación Apple Watch, captura de pantalla de la aplicación para Apple Watch en JPG/PNG (RGB) (312x390)





### Conexión iTunes
<a name="conexion_itunes_ELTPARAGRAPHE000121"></a>

Antes de implementar la aplicación, debe conectarse a Apple iTunes Connect para preparar la aplicación que se cargará desde Mac..  

Las etapas son las siguientes: 

- Conéctese a la siguiente Address: [http://itunesconnect.apple.com](http://itunesconnect.apple.com)

- Haz clic en "Mis aplicaciones".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=itunes_connect1.gif)


- Añada su aplicación mediante el botón "Nueva aplicación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=itunes_connect2.gif)


- Rellene la información solicitada: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=itunes_connect3.gif)


	- Nombre de la solicitud,

- Idioma de la aplicación, ....

- Pulse el botón "Crear" y especifique la información solicitada. 




Tu aplicación ya es reconocida por "iTunes Connect".

## Implementación en el App Store
<a name="implementacion_app_store_ELTTEXTE000330"></a>


### Compilación en Mac
<a name="compilacion_mac_ELTPARAGRAPHE000148"></a>

Para desplegar en el App Store:

- Genere su aplicación en WINDEV Mobile. Para obtener más información, consulte [Generación de una aplicación iOS](../Editeurs/1000019908.md). 

- Al final de la generación, copy tu directorio "YourApplication.xcode.gen" en tu MAC.

- Inicie Xcode y conéctese con su cuenta de Apple. Los archivos necesarios se descargan automáticamente. 

- Cargue su proyecto. Compila tu proyecto. 

- Cuando la aplicación se compila, un archivo debe ser construido a través de "Producto .. Archivo". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=XCode_archive1.gif)

	Observación: Si la opción "Archivo" es Grayed out, anule la selección del dispositivo de destino especificado y seleccione "Ninguno/Cualquiera".. 

- El organizador se inicia automáticamente en la pestaña "Archivo".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=xcode_archive2.gif)


- El botón "Cargar en App Store" se utiliza para enviar la aplicación a App Store.. 







### Implementación con iTunes Connect
<a name="implementacion_con_itunes_connect_ELTPARAGRAPHE000169"></a>

En iTunes Connect: 

- Crear (si es necesario) una versión de la aplicación. El número de versión de la aplicación debe corresponder al número de versión especificado al crear la generación de la aplicación en WINDEV Mobile (o el número de versión de WINDEV Mobile debe comenzar con el mismo número).. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=itunes_connect4.gif)


- Especifique la información relativa a la versión. 
	Observación: Algunas opciones se utilizan para gestionar la publicación automáticamente (o no) una vez que la aplicación ha sido validada. 

- Especifique la compilación a utilizar. Debe encontrar la compilación que fue enviada previsiblemente por Xcode. 
	Observación: el tiempo de Process puede ser bastante largo (varios minutos). 

- Pulse el botón "Enviar para validación".. La aplicación cambia al paso de revisión de automatic. Dependiendo de sus opciones de implementación, recibirá un Email durante la validación por parte de Apple y la aplicación puede ser (o será) desplegada.

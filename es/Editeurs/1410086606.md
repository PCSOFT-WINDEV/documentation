## Funciones del grupo "Selección
<a name="funciones_del_grupo_seleccion_ELTTEXTE000359"></a>
El editor Image incluye varias herramientas de selección: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0001%205.gif)


- **Rectángulo**: Se utiliza para seleccionar un área rectangular en la capa actual con el ratón.. El área se define manteniendo pulsado el botón izquierdo. Soltando el botón izquierdo del ratón se termina el área de selección.. El área está representada por un rectángulo en Line punteado.

- **Lazo**: Se usa para seleccionar cualquier área en la capa actual con el ratón. El punto de inicio se define con el botón izquierdo del ratón. El área se define manteniendo pulsado el botón izquierdo. Soltando el botón izquierdo del ratón se termina el área de selección.. El área está representada por un contorno de color. A continuación, esta zona se puede desplazar con la herramienta de desplazamiento.

- **Varita mágica**: Se utiliza para seleccionar un área coloreada sin tener que dibujar su contorno.. Todo lo que tiene que hacer es especificar la tolerancia para las selecciones realizadas con esta herramienta. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_Image_baguette%20-%20HC%20N%B0001.gif)
 Una tolerancia baja significa que la varilla seleccionará menos colores mientras que una tolerancia alta significa que la varilla cubrirá una amplia gama de colores. 

- **Mover y cambiar el tamaño** (Ctrl T): Permite: 

	- mover el contenido de la capa o área seleccionada con el ratón. El movimiento se realiza manteniendo pulsado el botón izquierdo del ratón. Soltando el botón izquierdo del ratón se valida el movimiento.

	- cambiar el tamaño del contenido de la capa o del área seleccionada con el ratón. El redimensionamiento se realiza con el ratón, utilizando las manijas del área. Si se mantiene pulsada la tecla Shift mientras se redimensiona, se mantienen las proporciones.

	- girar la capa o el área seleccionada con el ratón. Simplemente aleje el ratón de las manijas de área para mostrar un cursor de rotación (flecha doble curva). A continuación, mantenga el botón izquierdo del ratón pulsado cuando gire el objeto en la dirección deseada. Soltando el botón izquierdo del ratón se valida la rotación.

## Funciones del grupo "General

Las opciones del grupo "General" permiten manipular el Image que aparece en el editor: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0001%204.gif)
Es posible: 

- **Recortar el Image**: Esta opción convierte el cursor del ratón en una herramienta específica que selecciona el área que se desea conservar. El Image se recorta automáticamente. 

- **Redimensionar la imagen**: En la ventana que se muestra, especifique el nuevo tamaño o el porcentaje de ampliación/reducción.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0002.gif)

- **Modificar el área de trabajo**: En la ventana que se muestra, introduzca: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0003.gif)


	- Nuevo tamaño del área de trabajo (absoluto o porcentual). 

- El modo de ampliación Image. 

- **Rotación**: Permite girar la capa actual (rotación preestablecida o personalizada). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0004.gif)


- **Simetría**: Permite realizar una simetría de la capa actual (horizontal y/o vertical). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0005.gif)

## Funciones del grupo "Herramientas
<a name="funciones_del_grupo_herramientas_ELTTEXTE000431"></a>
El editor de Image propone varias herramientas para manejar el Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0001%203.gif)


- **Lápiz**: Se utiliza para dibujar una curva continua en la capa actual. El dibujo comienza en cuanto se pulsa el botón izquierdo del ratón y sigue el movimiento del ratón.. El grosor y color de la línea se pueden definir en los ajustes de la pestaña "Imagen", sección "Colores".

- **Borrador**: Se utiliza para borrar una sección de Image en la capa actual. El grosor del borrador se define en los ajustes del panel "Imágenes", sección "Colores".

- **Llenar**: Rellena un área de la capa actual. Esta área puede ser la capa completa, un área definida por una selección, ... El relleno color puede definirse en los ajustes del panel "Imágenes", sección "color s".. 

- **Elipse**: Se utiliza para dibujar una elipse en la capa actual. Con un clic, puede definir el punto de partida del área en la que se dibujará la elipse.. La elipse se dibujará a medida que mueva el ratón. El punto final se define soltando el botón izquierdo del ratón. El grosor Line y Color, así como el fondo Color pueden definirse en el panel "Imágenes", sección "Color y Line"..
	Observación: para obtener un círculo perfecto, pulse Mayúsculas mientras dibuja una elipse. 

- **Rectángulo**: Se usa para dibujar un rectángulo en la capa actual. Con un clic, puede definir el punto de partida del rectángulo. El rectángulo se dibujará a medida que mueva el ratón. El punto final se define soltando el botón izquierdo del ratón. El grosor Line y Color, así como el fondo Color pueden definirse en el panel "Imágenes", sección "Color y Line"..
	Observación: para obtener un cuadrado perfecto, pulse Mayúsculas mientras dibuja un rectángulo. 

- **Flecha**: Se usa para dibujar una flecha con el ratón sobre la capa actual. El punto de inicio se define con el botón izquierdo del ratón. La Line sigue el movimiento del ratón. El punto final (punta de flecha) se define soltando el botón del ratón. El grosor de la línea y color y las características de la flecha pueden definirse en el panel "Imágenes", sección "color s"..
	Observación: a get una flecha perfectamente horizontal o vertical, presiona Shift mientras dibujas la flecha.

- **Texto**: Se utiliza para definir un área de texto en la Image. El área de texto se define con el botón izquierdo del ratón. La fuente del texto, el tamaño, la alineación, color y los atributos se pueden definir en la barra de menú que se encuentra Above el área de texto.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0001.gif)


- **Marca de agua**: Permite añadir una Watermark a la Image. El área de Watermark se define con el botón izquierdo del ratón. Aparece una ventana que permite definir los diferentes ajustes de Watermark: fuente, Color, alineación, ángulo, texto, ... <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Champ_Editeur_img_Filigrane%20-%20HC%20N%B0001.gif)
 La opción "Vista previa en la imagen" permite visualizar el Watermark en la imagen actual, sin cerrar la ventana de configuración del Watermark..  

- **Line**: Se utiliza para dibujar una Line con el ratón en la capa actual. El punto de inicio se define con el botón izquierdo del ratón. La Line sigue el movimiento del ratón. El punto final se define soltando el botón izquierdo del ratón. El grosor y color de la línea se pueden definir en los ajustes de la pestaña "Imagen", sección "Colores".
	Observación: a get un Line perfectamente horizontal o vertical, presiona Shift mientras dibujas el Line. 

- **Blur**: Realiza una Blur gaussiana sobre la selección. La ventana que aparece le permite especificar el radio. También puede previsualizar la Image antes de aplicar la gaussiana Blur. 

- **Captura**: Permite capturar un área de la pantalla con un lazo. La captura de pantalla se añade a la Image actual en una nueva capa.

## Funciones del grupo "Transformación
<a name="funciones_del_grupo_transformacion_ELTTEXTE000455"></a>
El editor Image permite realizar varias transformaciones en el Image actual: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_Image_baguette%20-%20HC%20N%B0003.gif)


- **Icon**: Se utiliza para cambiar la imagen actual en Icon. Los diferentes formatos Icon disponibles aparecen a la izquierda del editor. Puede añadir formatos específicos (botón "+"), suprimir formatos inútiles, regenerar los iconos redimensionando la fuente Image. 

- **Multi DPI**: Se utiliza para crear una set de imágenes correspondientes a los diferentes DPIs . Los diferentes PPP disponibles aparecen a la izquierda del editor. Puede añadir DPI específicos (botón "+"), suprimir DPI inútiles, regenerar los Image s redimensionando el Image de origen . La opción **"Exportar DPI"** presente en la cinta permite generar los archivos correspondientes. El nombre de estas imágenes está formateado de la siguiente manera: 
	
	```txt
	omImage@dpi99x.Extension
	```
Donde 99x corresponde al coeficiente multiplicador en relación con el editor DPI.
	El editor opera en 160 DPI. Por ejemplo, para un "Logo.png" Image, el set de Image s será:

	- "Logo@dpi1x.png" para los 160 DPI.

	- "Logo@dpi1_5x.png" para los 240 PPP. 

	- "Logo@dpi2x.png" para los 320 DPI.

	- "Logo@dpi3x.png" para los 480 PPP. 

	- "Logo@dpi4x.png" para los 640 PPP. 




- **Borde**: Se utiliza para añadir una frontera alrededor de la Image. Usted puede: 

	- usar un borde preestablecido: negro simple, Round(ndroid), etc.

	- personalizar un borde preestablecido.

	- usar un borde personalizado. 




- **5 estados**: Permite generar (o regenerar) automáticamente una imagen de 5 estados a partir de una imagen simple. El tamaño de la Image se multiplica por 5 en anchura. Los 5 estados corresponden a: Normal, Presionado, Grayed, Seleccionado y Hover.

## Funciones del grupo "Varios
<a name="funciones_del_grupo_varios_ELTTEXTE000479"></a>
El editor Image incluye funciones avanzadas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_image_ruban%20-%20HC%20N%B0001%202.gif)


- **OCR**: Reconocimiento óptico de caracteres (OCR) en el Image mostrado en el editor Image. Basta con hacer clic y arrastrar sobre la zona a analizar. El texto se copia automáticamente en el portapapeles. 

- **Escanear**: Digitaliza una imagen utilizando una Scanner. La ventana que aparece le permite seleccionar: 

	- la deseada Scanner. 

	- la fuente (cama plana, alimentador, etc.). 

	- el modo Color (Color, escala de grises, negro y White, etc.). 

	- la resolución. 


 Puede obtener una vista previa de la Document e iniciar la exploración mediante los botones "Vista previa" y "Exploración", respectivamente. El resultado de la exploración se muestra en el editor de Image. El resultado de la exploración sustituye a la Image mostrada anteriormente

## Efectos y ajustes
<a name="efectos_ajustes_ELTTEXTE000503"></a>
El editor Image permite definir varios ajustes en el Image actual mediante las opciones del grupo "Efectos y ajustes".: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_Image_baguette%20-%20HC%20N%B0004.gif)


- **Ligereza de contraste**: Permite definir el contraste y la luminosidad de la Image. El contraste está incluido entre -100 y +100. La ligereza se incluye entre -255 y +255. Estos ajustes se pueden ver en tiempo real en el Image (opción "Vista previa en Image ").

- **Color transparente**: Se usa para seleccionar el color que se convertirá en Transparent en el Image. Esta configuración se puede ver en tiempo real en Image (opción "Vista previa en Image "). La pipeta sirve para seleccionar directamente una color en su Image. 

- **Ajustes HSL**: Permite modificar el Tono, la Saturación y la Luminosidad de la Image. El tono se incluye entre -180 y +180. La saturación se incluye entre -100 y +100. La ligereza se incluye entre -255 y +255. Estos ajustes se pueden ver en tiempo real en Image (opción "Vista previa en Image "). 

- **Transformando colores**: Permite sustituir una color por otra en la Image. Esta configuración se puede ver en tiempo real en Image (opción "Vista previa en Image ").

- **Sombra**: Se utiliza para aplicar un efecto de sombra en los elementos de Image en relación con el Image Background. Los efectos de sombra aumentan cada vez que se hace clic en la opción "Sombra".. La fuente de luz utilizada para generar la sombra proviene de la Corner superior izquierda.

- **aureola**: Se utiliza para definir una sombra alrededor de los elementos de Image (equivalente a una sombra en todos los lados). Puedes definir el grosor del halo. Esta configuración se puede ver en tiempo real en Image (opción "Vista previa en Image ").

- **Blanco y negro**: Se usa para convertir la imagen en Color en negro y White (escala de grises).

## Capas
<a name="capas_ELTTEXTE000527"></a>
El editor de Image le permite manejar las capas, lo que significa diferentes niveles en el Image. 

Estas capas se pueden gestionar: 

- a través de la cinta. 

- a través del panel "Imágenes. 




En la cinta, las opciones son las siguientes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_Image_baguette%20-%20HC%20N%B0005.gif)


- Nueva capa: Se utiliza para crear una capa en la Image. Esta capa puede ser creada: 

	- en blanco, 

	- de un Image cargado desde un directorio,

	- de un Image cargado de un catálogo de Image. 


La capa creada puede manejarse en el panel "Imágenes", sección "Capas".

- Duplicado: Duplica las capas seleccionadas en el panel "Imágenes", sección "Capas".

- Aplanar: Fusiona todas las capas seleccionadas. Es posible seleccionar una lista de capas para fusionar utilizando la multiselección en la lista de capas. Si una de las capas seleccionadas no es Visible (símbolo de ojo gris), se muestra un mensaje de error que indica que las capas que no son Visible no se pueden fusionar.




En la sección "Capas" del panel "Imágenes" se pueden realizar las siguientes operaciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002.gif)


- Hacer visible una capa o Invisible (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%201.gif)),

- Añadir una nueva capa (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%202.gif)),

- Eliminar la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%203.gif)),

- Duplicar la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%204.gif)),

- Mover la capa seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%205.gif)),

- Aplanar la Image (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_calque%20-%20HC%20N%B0002%206.gif)). 




Cada capa incluye un menú contextual que permite manipular Rla (duplicar, fusionar, mover, etc.). 

Observación: Ctrl + clic permite seleccionar la capa más apropiada, es decir, la capa que tiene contenido de Image bajo el cursor del ratón. 

La opción "Visible/Invisible mediante programación" permite manipular la capa en el código: Esto le permite mostrar la imagen en un control Imagen mientras especifica la capa que debe hacerse visible o Invisible ([ImageLayerVisible](../WDLang2/1000023504.md)).
Atención: En cuanto se activa la opción "Visible/Invisible mediante programación" para las capas, el archivo ".wdpic" debe tratarse mediante un [WDPic](../WDLang1/1000023503.md) Variable.

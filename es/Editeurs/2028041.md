<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
WINDEV incluye una asistente para generar un programa de instalación:

- para instalar una aplicación WINDEV.

- para instalar otros archivos. Instalar golosinas, por ejemplo (imágenes, vídeos,...).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Instalación de una aplicación WINDEV
<a name="instalacion_una_aplicacion_windev_ELTTEXTE000141"></a>
La creación del programa instalación asistente permite crear el programa instalación de una aplicación WINDEV. Por Default, si la asistente no pide el nombre de la aplicación a instalar, la aplicación a instalar corresponde al proyecto abierto en WINDEV.

Si la asistente pide el nombre de la aplicación a instalar, todo lo que tiene que hacer es especificar el proyecto (archivo ".WPP") a instalar.

El directorio base de la aplicación seleccionada es recuperado automáticamente por la asistente. Este directorio corresponde al directorio "EXE" del proyecto a instalar. Por Default, todos los archivos encontrados en este directorio excepto los archivos de datos (".FIC", ".NDX" y ".MMO") serán seleccionados.

## Instalación de otros archivos
<a name="instalacion_otros_archivos_ELTTEXTE000165"></a>
La creación del programa instalación asistente (iniciado desde WDInst, el editor de instalación) permite crear un programa instalación associated sin ninguna aplicación. Esta opción permite proporcionar nuevos archivos de vídeo o nuevos archivos de Image a los usuarios por ejemplo (en una instalación independiente de la instalación de la aplicación principal).

**Generar un programa instalación associated sin aplicación**:

1. En WINDEV, cierre el proyecto actual: en la pestaña "Inicio", en el grupo "General", despliegue "Cerrar" y seleccione "Cerrar proyecto".

2. Abra el editor de instalación WDInst: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

3. Seleccione "Archivo... Nuevo".

4. No especifique ningún proyecto (archivo".WDP").

5. Haga clic en el botón "Avanzado".

6. Seleccione el directorio de base.
	Este directorio se utiliza para definir la estructura de los directorios a crear en los ordenadores de los usuarios..
	Por ejemplo, si el directorio base es "C:\\MyFiles" y si los archivos a instalar se encuentran en "C:\\MyFiles\\MyImages\\BackgroundImages", estos archivos se instalarán en el subdirectorio "MyImages\\BackgroundImages" del directorio de instalación por Default.

7. Seleccione los archivos a instalar en los siguientes pasos de la asistente.

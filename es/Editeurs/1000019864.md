
## Despliegue de la aplicación para iPhone/iPad
			



## Condición previa
<a name="condicion_previa_ELTTEXTE000165"></a>
Para poder probar y/o implementar la aplicación en un dispositivo (iPhone o iPad), debe inscribirse en el **Apple Developer Program**. Es necesario comprar una membresía para inscribirse. Puede ver la lista de programas en [https://developer.apple.com/programs/enroll/](https://developer.apple.com/programs/enroll/)

Inicialmente, Apple ofrecía tres programas distintos:

- Programa para desarrolladores de iOS - Individual,

- Programa para desarrolladores de iOS - Empresa,

- Programa empresarial para desarrolladores de iOS.




Apple lo simplificó en marzo de 2021. Ahora, hay 2 programas: 

- Individual. 

- Organización. 




Este programa incluye un certificado de desarrollador que permite firmar aplicaciones para compilarlas y distribuirlas. El certificado tiene un costo. El certificado de desarrollador es necesario incluso para una instalación destinada a realizar pruebas (debug) en un dispositivo. 

Estas son las características de cada programa: 

- **Individual**: Este programa permite los siguientes tipos de distribución:

	- App Store,

	- Apple Business Manager y Apple School Manager (se requiere la validación de Apple para su distribución en el App Store, sin dejar de ser privada),

	- Ad Hoc (limitado a 100 dispositivos).




- **Organización**: Este tipo de programa permite la distribución fuera de la Apple Store, por lo tanto sin la validación de la aplicación por parte de los equipos de Apple. 
	Para distribuir una aplicación en una tienda privada (ver [MDM con WDBaaS](../Editeurs/1410086652.md)), debe utilizar inscribirse en este tipo de programa. A continuación, puede distribuir su aplicación en un número ilimitado de dispositivos Apple.








## Ejecución de la prueba de la aplicación en el dispositivo
<a name="ejecucion_prueba_aplicacion_dispositivo_ELTTEXTE000189"></a>
La prueba en el dispositivo está disponible independientemente del tipo de registro. 

Todo lo que tienes que hacer es conectar tu dispositivo a tu MAC y compilarlo para generar el programa para el dispositivo solicitado.. A continuación, el programa se instalará en el dispositivo. Se ejecutará de forma independiente. En este caso, la aplicación se despliega en [Modo ad hoc](../Editeurs/1000019884.md) en el dispositivo. 

El programa también puede ser instalado a través de Wi-Fi o de la red (In-House implementación).





## Despliegue de la aplicación en el dispositivo
<a name="despliegue_aplicacion_dispositivo_ELTTEXTE000213"></a>
Existen tres métodos para implementar una aplicación en un dispositivo (iPhone o iPad).

- **A través del App Store**: 
	Este tipo permite distribuir aplicaciones a través del App Store sin restricciones. La aplicación incluirá la firma relacionada con el certificado, pero no estará relacionada con un solo dispositivo.

- **A través de una red privada (antes implementación)**: 
	Este tipo permite distribuir aplicaciones a través de un servidor Web a un grupo de usuarios dentro de una empresa. Para este tipo de distribución, es necesario ser miembro de la Organización. La aplicación incluirá la firma relacionada con el certificado, pero no estará relacionada con un solo dispositivo.

- **A través de una red ad hoc**: 
	Este tipo de distribución permite instalar las aplicaciones en una red Ad Hoc que contiene hasta 100 dispositivos (iPhone y iPad). 
	La aplicación debe ser recompilada para el dispositivo de destino incluyendo el Certificate para la firma así como el identificador único del dispositivo

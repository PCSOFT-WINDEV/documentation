
## Preparar teléfono para probar una aplicación Android
			



<a name="NOTE0_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000094"></a>
WINDEV Mobile le permite desarrollar aplicaciones Android. Para ejecutar la prueba de estas aplicaciones en un Android Phone, hay que realizar algunas operaciones: 

- [Instalando los controladores ADB del Phone en la computadora de desarrollo](#NOTE1_1).

- [Activación de la depuración USB](#NOTE2_1).




<a name="NOTE1_1"></a>


## Instalación de los controladores ADB
<a name="instalacion_los_controladores_adb_ELTTEXTE000118"></a>
Cuando se establece la primera conexión entre el Phone y la computadora de desarrollo, Windows encuentra automáticamente los controladores correspondientes. 

1. Especifique que la ubicación se indicará manualmente. 

2. Los controladores se encuentran en el directorio instalación del SDK (directorio "usb_driver"). Si no se encuentra este directorio, utilice sdk manager para instalar estos controladores. Para obtener más información, consulte [https://developer.android.com/studio/run/oem-usb.html](https://developer.android.com/studio/run/oem-usb.html)




<a name="NOTE2_1"></a>


## Activación de la depuración USB
<a name="activacion_depuracion_usb_ELTTEXTE000142"></a>
La depuración USB debe estar activada para realizar pruebas en el teléfono. Si no se realiza esta operación, el Phone puede no ser detectado por WINDEV Mobile. 

**Para habilitar la depuración USB**: 

1. En el teléfono, vaya a "Configuración". 

2. Seleccione "Acerca del dispositivo". 

3. Presione varias veces la opción "Número de compilación" para activar el modo de desarrollador. 

4. Suba un nivel. 

5. La entrada "Opciones para desarrolladores" aparece. Seleccione esta opción. 

6. Seleccione "Depuración por USB". 


**Atención**: Las acciones a realizar pueden cambiar según la versión del teléfono y el fabricante. Por ejemplo, en un teléfono Samsung Galaxy, hay que "presionar" varias veces el "Número de versión" en la opción "Acerca del dispositivo" para activar las "Opciones para desarrolladores". En cualquier caso, puede realizar una búsqueda en Google con "depuración por USB &lt;nombre del dispositivo&gt;" para adaptar el modo de funcionamiento al dispositivo utilizado. 

Observación: En algunos dispositivos, debe seleccionar "Modo de almacenamiento masivo (MSC)" para que el dispositivo sea detectado..

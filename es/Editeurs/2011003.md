## Presentación
<a name="presentacion_ELTTEXTE000190"></a>
Cuando se manejan análisis que contienen varias descripciones de archivos de datos, el gráfico de análisis es difícil de leer.. Para gestionar estos grandes análisis, WINDEV, WEBDEV y WINDEV Mobile proponen Custom-Folders en el análisis. 

Las carpetas personalizadas se utilizan para agrupar varias descripciones de archivos de datos. 

Estas "carpetas personalizadas" se utilizan para simplificar: 

- la búsqueda y la visualización de la description de un archivo de datos encontrada en un gran análisis. 

- la operación realizada en los grupos de archivos de datos a través de la programación. 
	Por ejemplo, puede cambiar la conexión de todos los archivos de datos de un Carpeta Personal con [HChangeConnection](../WDLang4/3044150.md): para ello, especifique el nombre de la Carpeta Personal que debe manejarse en la función WLanguage (véase la lista de funciones disponibles a continuación). 
	Esta función está disponible para los archivos de datos HFSQL Classic, los archivos de datos HFSQL Client/Server y los archivos de datos manejados por Native Access. 




Estas agrupaciones se definen en el ["Panel"Análisis](../Editeurs/2027012.md) o en el gráfico de análisis. Estas carpetas personalizadas no se crean físicamente.

El siguiente ejemplo agrupa los ficheros de datos según su categoría: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GroupeAnalyses.gif)


**Limitaciones**:

- Una Carpeta Personal puede estar vacía y no contener ningún archivo de datos description.

- No es obligatorio asociar la descripción de la un archivo de datos a una Carpeta Personal.

- Una Carpeta Personal no puede pertenecer a otra Carpeta Personal.

- Dos carpetas personalizadas no pueden tener el mismo nombre.

- Una Carpeta Personal no puede tener el mismo nombre que una un archivo de datos description.

## Manejo de las carpetas personalizadas en el análisis
<a name="manejo_las_carpetas_personalizadas_analisis_ELTTEXTE000220"></a>
Las diferentes operaciones que se pueden realizar en las Custom-Folders del análisis son las siguientes:

- crear una Carpeta Personal.

- renombrar una Carpeta Personal.

- borrar Carpeta Personal.

- asociar un archivo description con una Carpeta Personal.

- disociar un archivo description de un Carpeta Personal.


Para manipular las carpetas personalizadas en el análisis, el panel "Análisis" debe ser mostrado. Para visualizar el panel "Análisis", vaya a la pestaña "Inicio", al grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y luego "Análisis".
<a name="NOTE2_2"></a>


### Manejar una Carpeta Personal
<a name="manejar_una_carpeta_personal_ELTPARAGRAPHE000075"></a>

Para crear un Carpeta Personal:

1. En el panel "Análisis" (o en el gráfico de análisis), seleccione "Nueva Carpeta Personal" en el menú contextual (clic derecho). Se crea automáticamente una nueva Carpeta Personal.

2. Introduzca el nombre de esta Carpeta Personal y Confirm( Introduzca la clave).




Para renombrar una Carpeta Personal:

1. Seleccione la Carpeta Personal para renombrarla en el panel "Análisis".

2. Abra el menú contextual (haga clic con el botón derecho) y seleccione "Renombrar" (o pulse F2).

3. Introduzca el nuevo nombre de esta Carpeta Personal y Confirm( Introduzca la clave).




Para borrar en Carpeta Personal:

1. Seleccione la Carpeta Personal a borrar (en el panel "Análisis" o en el gráfico de análisis).

2. Abra el menú contextual (clic derecho) y seleccione "Eliminar".. 

3. La Carpeta Personal es eliminada. Los archivos de datos de associated con esta Carpeta Personal se disocian automáticamente de esta Carpeta Personal.




Para asociar un archivo description con una Carpeta Personal:

1. Seleccione el archivo description que debe ser associated con el Carpeta Personal.

2. Arrastrar y soltar la description en la Carpeta Personal deseada. La description es automáticamente associated con la Carpeta Personal.
	**Observación**: Si la description ya pertenecía a una Carpeta Personal, la description sólo se asocia a la nueva Carpeta Personal.




Para disociar un archivo description de un Carpeta Personal:

1. Seleccione el archivo description para disociar en el panel "Análisis".

2. Haga clic derecho para abrir el menú contextual y seleccione "Retirar de la Carpeta personal". 

3. El archivo description se disocia automáticamente del Carpeta Personal.

## Acciones que se pueden realizar en una Carpeta Personal
<a name="acciones_que_pueden_realizar_una_carpeta_personal_ELTTEXTE000250"></a>


### Personalizar la representación gráfica de la Carpeta Personal
<a name="personalizar_representacion_grafica_carpeta_personal_ELTPARAGRAPHE000112"></a>

La personalización de una Carpeta Personal consiste en mostrar una imagen de fondo específica en la Carpeta Personal.

Para personalizar la representación gráfica de una Carpeta Personal:

1. Seleccione la Carpeta Personal en el gráfico de análisis. 

2. Haga clic derecho para abrir el menú contextual y seleccione la opción "Descripción".

3. Seleccione la opción "Representación de la Carpeta Personal en el gráfico del análisis" y elija la imagen deseada. Esta Image se mostrará en la representación gráfica de la Carpeta Personal.

4. Valide.



<a name="NOTE3_2"></a>


### Imprimir el contenido de la Carpeta Personal
<a name="imprimir_contenido_carpeta_personal_ELTPARAGRAPHE000130"></a>

Tiene la posibilidad de imprimir la documentación de los elementos que se encuentran en la Carpeta Personal: 

1. Seleccione la Carpeta Personal en el panel "Análisis".. 

2. Abra el menú contextual (clic derecho) y seleccione "Imprimir".

## Manejo de grupos/carpetas personalizadas mediante programación
<a name="manejo_gruposcarpetas_personalizadas_mediante_programacion_ELTTEXTE000280"></a>
Se pueden utilizar varias funciones de WLanguage para las carpetas personalizadas de manipular a través de la programación: 



|   |   |
| --- | --- |
| [HChangeConnection](../WDLang4/3044150.md) | Cambia dinámicamente la conexión asociada a un archivo de datos. |
| [HChangeDir](../WDLang4/3044168.md) | Cambia la ruta de acceso a un archivo de datos (es decir, el directorio en el que se manipulará el archivo). |
| [HChangeLocation](../WDLang4/3044167.md) | Modifica el modo de búsqueda de los archivos de datos físicos (.FIC, .NDX, etc.). |
| [HClose](../WDLang4/3044073.md) | Cierra un archivo de datos o todos los archivos de datos abiertos por el usuario actual: todos los archivos de datos físicos correspondientes se cierran para el usuario actual. |
| [HCreationIfNotFound](../WDLang4/3044254.md) | Realiza las siguientes operaciones: <br><br>- Si el archivo no existe, crea un archivo de datos vacío (archivo de extensión ".FIC") con el archivo de índice y el archivo memo, si es necesario. La función equivale a [HCreation](../WDLang4/3044255.md). <br><br>- Si el archivo existe, abre el archivo.<br><br><br> |
| [HFlush](../WDLang4/3044004.md) | Obliga al sistema operativo del equipo en el que se encuentran los archivos de datos a escribir los datos directamente en el disco. |
| [HListFile](../WDLang4/3044045.md) | Devuelve la lista de archivos:<br><br>- encontrado en el análisis actual o en un análisis específico reconocido por el motor HFSQL. Los archivos definidos por [HDeclare](../WDLang4/3044203.md), [HDeclareExternal](../WDLang4/3044204.md) y [HDescribeFile](../WDLang4/3044206.md) se tienen en cuenta.<br><br>- disponible en una conexión a través de un acceso nativo o a través de un controlador OLE DB.<br><br>- para un grupo de archivos definidos en la editor de análisis.<br><br><br> |
| [HOpen](../WDLang4/3044120.md) | Abre un archivo de datos. |
| [HPass](../WDLang4/3044108.md) | Define la contraseña usada para crear o abrir un archivo de datos. |

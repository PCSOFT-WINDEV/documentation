


## SEO de un sitio WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000227"></a>
Para que su sitio a los visitantes de get, los usuarios de la Web deben ser capaces de encontrarlo. La forma más fácil de encontrar un sitio es utilizar un motor de búsqueda. El usuario de la Web introduce palabras clave y el motor de búsqueda propone todos los sitios que corresponden a los criterios de búsqueda.

Por lo tanto, su sitio debe ser optimizado en los motores de búsqueda.

Para ayudarle, WEBDEV propone:

- **para optimizar cada estática y Active WEBDEV Page**: se puede definir una description y una set de palabras clave para cada página. Estas palabras clave permitirán a los usuarios acceder directamente a la página. También puede elegir no optimizar una página.

- **Una optimización de los motores de búsqueda asistente**, que indica las mejoras que se pueden hacer en cada página para propósitos de SEO.

- **incluir o usar una página específica como página de inicio**. Esta solución permite optimizar las páginas dinámicas, por ejemplo, asociando palabras clave con la página de inicio.. La página de inicio contiene todas las palabras clave que permitirán a los usuarios encontrar su sitio. 




**Observación**: Para mejorar el SEO de sus páginas, utilice [Páginas de Active WEBDEV](../WDChamp/3539009.md). Para optimizar un sitio dinámico (en modo de sesión), use:

- una página de inicio.

- una sección del sitio en modo estático. El sitio estático se utilizará para el seo e iniciará el sitio en modo sesión.

- una sección del sitio en modo AWP. El sitio AWP se utilizará para realizar el SEO y para iniciar el sitio en modo de sesión.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000251"></a>


### Utilizar (o no) una página estática o AWP para el SEO
<a name="utilizar_una_pagina_estatica_awp_para_seo_ELTPARAGRAPHE000035"></a>

Para optimizar una página estática o AWP de del sitio:

1. Abra la ventana de descripción de la página.

2. En la pestaña "Detalles", una sección específica permite configurar el SEO de la página. Es posible:

	- Desactivar el SEO para la página actual.

	- Permitir el SEO para la página actual asociando expresiones y palabras clave (Botón "Editar expresiones y palabras clave").






<a name="NOTE2_2"></a>


### Optimizar e introducir las palabras clave de una página
<a name="optimizar_introducir_las_palabras_clave_una_pagina_ELTPARAGRAPHE000052"></a>

Para optimizar una página estática o AWP de del sitio:

1. Abra la ventana de descripción de la página.

2. En la pestaña "Detalles", una sección específica permite configurar el SEO de la página. En esta sección, haga clic en "Editar expresiones y palabras clave".

3. **En la pestaña "description" de la ventana SEO**, escriba la página description. En la mayoría de los casos, esta descripción aparece cuando se muestra el resultado de una búsqueda realizada con un motor de búsqueda.. Consejos:

	- Utilice oraciones cortas, limite el número de palabras (hasta 200 caracteres)

	- La descripción debe incitar a los usuarios a visitar la página.
			**Observación**: La descripción puede modificarse dinámicamente utilizando la propiedad [description](../Proprietes/1000017164.md).
			El description asociado a una página se escribe en el encabezado HTML de la página (&lt;META NAME="description" CONTENT="..."&gt; tag). 




4. **En la pestaña "Palabras clave" de la ventana de SEO**, escriba las palabras clave associated con la página. Estas palabras clave (o expresiones) se utilizarán para optimizar la página: si el usuario introduce una de estas palabras clave, el motor de búsqueda propondrá la página correspondiente. Las palabras clave se deben introducir de la siguiente manera: escriba la primera palabra clave y pulse la tecla Entrar para escribir la siguiente.
	Consejos:

	- Las tres primeras palabras clave son las más importantes. Utilice palabras clave comunes y pertinentes.

	- Las palabras clave no deben repetirse.

	- Utilice variantes (singular/plural, sustantivo/verbo, etc.).
			Observación: Estas palabras clave pueden modificarse dinámicamente usando la propiedad [Palabras clave](../Proprietes/1000017163.md).




5. Valide las ventanas de SEO y de descripción de la página.

### Utilizar el asistente de SEO
<a name="utilizar_asistente_seo_ELTPARAGRAPHE000090"></a>

Para ayudarle a mejorar el SEO de sus sitios y páginas, WEBDEV propone un asistente de SEO. Este asistente se puede utilizar para una página específica o para todas las páginas del proyecto. El asistente analiza el contenido de las páginas del sitio e indica las mejoras que se pueden realizar para maximizar el posicionamiento en los motores de búsqueda.

Ejemplos de optimizaciones detectadas por el asistente:

- Se debe especificar el título de la página.

- Se deben especificar al menos tres palabras clave para la página.

- Se debe especificar la descripción de la página.

- El texto alternativo se debe introducir para todos los controles que lo propongan (si es posible, usando las palabras clave definidas para la página).

- ...




Para iniciar la asistente de SEO en una página: 

1. Abra la página deseada en el editor.

2. En la pestaña "Página", en el grupo "SEO", haga clic en "Optimizar".

3. El asistente SEO aparece en pantalla. Haga doble clic en una sugerencia para realizar la optimización correspondiente.. El botón "Actualizar" permite actualizar la lista de sugerencias.




Iniciar la asistente para optimizar todas las páginas del sitio: 

1. En la pestaña "Proyecto", en el grupo "Web", despliegue "SEO" y seleccione "Asistente SEO".

2. El asistente SEO aparece en pantalla. Haga doble clic en una sugerencia para realizar la optimización correspondiente.. El botón "Actualizar" permite actualizar la lista de sugerencias. Los resultados se pueden ordenar por página o por tipo de consejo.




**Observación**: El siguiente error puede ser mostrado por el SEO asistente: "El IDM de la palabra clave XXXX es set a yy%". El IDM corresponde al índice de densidad para las palabras clave. Es la relación entre el número de apariciones de la palabra clave en la página y el número total de palabras en la página..

### Incluir una página de inicio en su sitio web
<a name="incluir_una_pagina_inicio_sitio_web_ELTPARAGRAPHE000142"></a>

Si no hay una página de inicio de associated con su proyecto, puede hacerlo:

- elegir una página de inicio entre las páginas estáticas de su sitio.

- crear una nueva página estática que se convertirá en la página de inicio del sitio.




Para más detalles sobre la página de inicio de su sitio, consulte [Creación de una página de inicio para su sitio web](../WDChamp/2030010.md).

### Incluir un sitio en los motores de búsqueda
<a name="incluir_sitio_los_motores_busqueda_ELTPARAGRAPHE000157"></a>

Existen dos tipos de motores de búsqueda:

- **los motores de búsqueda "automatic":**
	Estos buscadores se basan en programas automáticos de análisis de contenidos.
	**¿Cómo puedo optimizar mi sitio Web?**
	Un enlace de tipo "Optimice su sitio" suele estar disponible en este tipo de motores de búsqueda. Simplemente especifique la dirección del sitio y el correo electrónico del Webmaster. El sitio será monitoreado automáticamente (según las palabras clave y el contenido del sitio) y optimizado.

- **El "Libro de Phone" buscadores:**
	Estos motores de búsqueda optimizan sitios que han sido controlados previamente de forma manual.
	**¿Cómo puedo optimizar mi sitio Web?**
	Un enlace de tipo "Optimice su sitio" suele estar disponible en este tipo de motores de búsqueda. 
	En la mayoría de los casos, el SEO se realiza siguiendo las siguientes etapas: 

	1. Elegir una categoría para el sitio (ocio, cultura, etc.). 

	2. Responder un cuestionario sobre el sitio para optimizarlo: dirección de Internet del sitio, correo electrónico del WebMaster, descripción del sitio, etc.

	3. Una vez enviado el cuestionario, el sitio será evaluado por una persona de la compañía que administra el directorio. Esta persona posicionará el sitio si considera que el contenido es pertinente.

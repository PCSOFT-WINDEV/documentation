
## Detección instantánea de código modificado/código actual
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000080"></a>
En el editor de código: 

- Las líneas de código modificadas son inmediatamente Visible (1).

- La Event en la que se posiciona el cursor puede ser fácilmente localizada (2). 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CodeModifie.GIF)


(1): **Las líneas de código modificadas**
Sólo se tienen en cuenta las modificaciones de código realizadas desde la apertura de la etapa de corriente:

- se han guardado las líneas de código precedidas por una barra azul.

- las líneas de código precedidas por una barra naranja aún no se han guardado.




Cuando el elemento se cierra, las barras de color desaparecen automáticamente.

(2): **Corriente Event**
La barra vertical a la izquierda del código cambia de color para identificar fácilmente la WLanguage Event en la que se encuentra el cursor. useful en clave grande Page.

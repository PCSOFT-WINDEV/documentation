## Creación de las diferentes ventanas de patrones
<a name="creacion_las_diferentes_ventanas_patrones_ELTTEXTE000195"></a>


### Los ficheros a crear
<a name="los_ficheros_crear_ELTPARAGRAPHE000039"></a>

Recordatorio : Para construir un patrón RAD utilizado para crear un proyecto completo, se requieren los siguientes archivos:

- Un formulario para RADFileA.

- Un formulario para RADFileB. Un cuadro combinado y un botón Vision Plus (o un cuadro combinado emergente) deben permitirle seleccionar un elemento de RADFileA.

- Un formulario para RADFileC. Un cuadro combinado y un botón Vision Plus (o un cuadro combinado emergente) deben permitirle seleccionar un elemento de RADFileB.

- Una tabla para RADFileA.

- Una tabla para RADFileB.

- Un tabla para RADFileC.

- Una ventana de relación RADFileB/RADFileD.




Estos diferentes elementos deben contener botones de enlace. También tiene la posibilidad de proporcionar el menú principal de la aplicación.
<a name="NOTE2_2"></a>


### Diferencia entre Móvil y Smartphone
<a name="diferencia_entre_movil_smartphone_ELTPARAGRAPHE000056"></a>

Crear una aplicación para Móvil y una aplicación para Smartphone son dos cosas diferentes. En un Smartphone, la pantalla es más pequeña y un menú desplegable le permite validar (o no) las pantallas.

Esta diferencia también se aplica a los patrones RAD. Esta página de ayuda explica cómo crear una ventana de formulario para Móvil y para Smartphone.

## Menús
<a name="menus_ELTTEXTE000231"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000076"></a>

En un patrón WINDEV Mobile, el menú principal de la aplicación se define en una ventana específica. El menú se muestra como un menú desplegable. Las diferentes opciones de menú se utilizan para abrir:

- las ventanas de formulario y las ventanas de tablas definidas en el patrón

- otras ventanas que se encuentran en el patrón (una ventana para configurar los correos electrónicos, ....).




**Importante : no olvide definir la primera ventana del proyecto en su patrón RAD. En la mayoría de los casos, esta primera ventana corresponde a la ventana del menú.**
<a name="NOTE3_2"></a>


### Creación del menú principal del patrón WINDEV Mobile y Smartphone
<a name="creacion_del_menu_principal_del_patron_windev_mobile_smartphone_ELTPARAGRAPHE000088"></a>

Para crear el menú principal del patrón WINDEV Mobile y Smartphone : 

1. Cree (si es necesario) la ventana donde se mostrará el menú.

2. Crear el menú : en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Menú principal" y seleccione "Agregar menú principal".

3. Mostrar la ventana description de la opción "Menú":

	- Esta opción debe llamarse : "MAINMENU_XXX" (donde XXX es el nombre de su elección), por ejemplo "MAINMENU_File".

	- Especifique el título de la opción (por ejemplo,"Archivo").







Las opciones de menú utilizadas para las opciones específicas del RAD deben tener un nombre específico : "MENUFIC_". El menú con este nombre se duplicará tantas veces como sea necesario para los diferentes archivos del análisis que requieran una entrada de menú.. El código associated con las opciones también se duplicará y se adaptará (si es necesario) al archivo utilizado.

El código puede ser, por ejemplo:


```wl
Open(WIN_Table_RADFileA)
```


Las otras opciones de menú (utilizadas para iniciar una característica incluida en el patrón, por ejemplo) pueden tener cualquier nombre. Se generarán automáticamente.

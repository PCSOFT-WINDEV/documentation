


## timer en el botón (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000138"></a>
Una ventana puede permanecer abierta indefinidamente, esperando a que el usuario haga clic en. En algunos casos, si el usuario no finaliza la acción actual (modificar información sobre un cliente, etc.), la información correspondiente puede estar bloqueada para los demás usuarios.. 

Un botón de "Validación automática" permite validar (o cancelar) automáticamente una acción tras un período de tiempo determinado (entre 1 segundo y 15 minutos). 

Esta característica es también muy useful para las ventanas de confirmación. 

**Recordatorio**: Un botón de timer on también puede ser incluido automáticamente en la aplicación. Para obtener más información, consulte [Validación de automatic](../WDChamp/1013072.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000162"></a>


### Configurando el timer a través de la AAF
<a name="configurando_timer_traves_aaf_ELTPARAGRAPHE000029"></a>

Para configurar el temporizador en uno de los botones de la aplicación:

1. Haga clic derecho en el botón para abrir el menú contextual y seleccione "Validación automática...". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Minuterie%20sur%20bouton_Etape%202.jpg)


2. Seleccione el tiempo para la validación automática:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Minuterie%20sur%20bouton_Etape%203.jpg)

	

3. El botón con el temporizador se activará automáticamente cada vez que se utilice la ventana. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Minuterie%20sur%20bouton_Etape%204.jpg)





**Observación**: Puede quitar el temporizador de un botón en cualquier momento seleccionando la opción "Cancelar validación automática" en el menú contextual del botón. 
<a name="NOTE2_2"></a>


### Personalizar la ventana de configuración del botón timer
<a name="personalizar_ventana_configuracion_del_boton_timer_ELTPARAGRAPHE000038"></a>

WINDEV le permite incluir en su aplicación la ventana de configuración del botón de tiempo utilizado por la AAF.. Usted tendrá la posibilidad de personalizar:

- el aspecto de esta ventana (por ejemplo, modificación de plantilla).

- los idiomas soportados por esta ventana. Esta característica es muy useful a la hora de traducir esta ventana a los diferentes idiomas disponibles en la aplicación.  




**To incluir la ventana para configurar el timer en su aplicación**:

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccione el subdirectorio que contiene los elementos a importar. La ventana de configuración del botón timer está en el subdirectorio "Programas\\Datos\\Ventanas preestablecidas\\AAF" del directorio WINDEV instalación.

3. WINDEV hace una lista de los elementos del directorio. Este directorio contiene los elementos que corresponden a las AAF personalizables. La ventana para importar se llama "WINDEVAaf_TimerWindow.wdw".

4. Validar. La ventana está incluida en el proyecto y se puede modificar. Esta ventana personalizada se tendrá en cuenta automáticamente al ejecutar la aplicación.




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario. 



<a name="NOTE2_3"></a>


### Desactivando la capacidad de implementar un botón de timer
<a name="desactivando_capacidad_implementar_boton_timer_ELTPARAGRAPHE000073"></a>

Para evitar que el usuario cree un botón de temporizador, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafAutomaticValidation*.
<a name="NOTE2_4"></a>


### El botón timer en las ventanas del sistema
<a name="boton_timer_las_ventanas_del_sistema_ELTPARAGRAPHE000083"></a>

Para proponer un botón de temporizador configurable en las ventanas del sistema, se recomienda marcar "Personalizar ventanas del sistema (Info, YesNo, Confirm, Dialog)" en la pestaña "Estilo" del proyecto description.

**Observación**: Si el texto de la ventana del sistema difiere del texto que aparece al definir el botón timer, no se propondrá el timer.

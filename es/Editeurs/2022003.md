


## Posicionamiento automático
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
Cuando se mueve un control en una ventana, en una página o en un reporte, las guías se muestran automatic ally por el verificador de interfaz en tiempo real (también llamado posicionamiento automatic). Los objetos se ajustan a las guías, lo que permite alinear el control desplazado con los demás control de la ventana, el Page o el reporte.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CorrecteurIHM.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000133"></a>


### Utilizar el posicionamiento automatic en una ventana o en un Page
<a name="utilizar_posicionamiento_automatic_una_ventana_page_ELTPARAGRAPHE000019"></a>

Para utilizar el posicionamiento automatic en una ventana o en un Page: 

1. En la pestaña " Vista", haga clic en el lanzador de cuadros de diálogo del grupo " Opciones".: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif)

2. En la pestaña " Magnetismo", en "Posicionamiento automatic", marque "Activado" y "Mostrar reglas".. 




Para evitar usar la posición de la automatic en una ventana, todo lo que tienes que hacer es desmarcar estas opciones.
<a name="NOTE2_2"></a>


### Usando el posicionamiento de automatic en un reporte
<a name="usando_posicionamiento_automatic_reporte_ELTPARAGRAPHE000044"></a>Para usar la posición de automatic en un reporte: 
En la pestaña "Vista", en el grupo "Ayuda edición", seleccione la opción "Posicionamiento automático". 

- Si se marca esta opción, se activa el posicionamiento de automatic. 

- Si esta opción no está marcada, la posición de la automatic está desactivada.

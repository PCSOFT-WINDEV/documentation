
## Creación de un ejecutable: Idioma del ejecutable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
Al crear el ejecutable, tiene la posibilidad de elegir los idiomas que se incluirán en el ejecutable.. Esta opción le permite generar un ejecutable diferente para cada idioma.

**Atención**: si elige generar el ejecutable para un solo idioma, no olvide modificar (si es necesario) las opciones del menú que permiten cambiar el idioma.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Idiomas del ejecutable
<a name="idiomas_del_ejecutable_ELTTEXTE000101"></a>
El lenguaje actual en WINDEV es propuesto por Default.

Usted tiene la capacidad de seleccionar:

- Uno de los idiomas utilizados por el proyecto,

- Todos los idiomas del proyecto, 

- El idioma actual en el editor.




**Observación**: La selección de un número limitado de idiomas se utiliza para reducir el tamaño de la biblioteca de aplicaciones (BDM).. Las configuraciones del proyecto también se pueden utilizar para gestionar los idiomas. Puede utilizar una configuración de proyecto para generar una biblioteca FR + US a partir de un proyecto que contenga, por ejemplo, 5 idiomas.

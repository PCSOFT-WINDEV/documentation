


## Compilar y probar el sistema de ayuda
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000187"></a>
Para crear un archivo basado en la información introducida en el editor de ayuda, es necesario compilar la ayuda. Se utiliza un compilador específico para cada formato de ayuda (HTML, Ayuda HTML o PDF). Estos compiladores se suministran con WINDEV, WEBDEV y WINDEV Mobile.

La compilación permite crear el archivo de ayuda final a partir de todos los elementos disponibles: imágenes, palabras clave, páginas de ayuda, etc.

La compilación debe realizarse:

- para crear el archivo de ayuda (CHM, PDF o páginas HTML).

- cada vez que se modifique el sistema de ayuda (modificación de una palabra clave Page, etc.).




Una vez completada la compilación, se puede probar el archivo de ayuda.

**Atención**: Antes de compilar la ayuda, asegúrese de que los compiladores proporcionados con WINDEV, WEBDEV y WINDEV Mobile están referenciados en el archivo [opciones del editor de ayuda](../Editeurs/2010020.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000211"></a>


### Definir la página predeterminada
<a name="definir_pagina_predeterminada_ELTPARAGRAPHE000032"></a>

Antes de compilar el sistema de ayuda, debe definir la página predeterminada. Esta página será la primera que se mostrará cuando se abra la ayuda.

Para definir la página predeterminada del sistema de ayuda: 

1. Vaya a las características del sistema de ayuda: en la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Descripción".

2. En la pestaña "General", seleccione la página predeterminada.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=aide_compilation%20-%20HC%20N%B0001.gif)




<a name="NOTE2_2"></a>


### Definir las opciones de compilación
<a name="definir_las_opciones_compilacion_ELTPARAGRAPHE000057"></a>

Mediante Default, la ayuda se compila en el formato elegido al crear el sistema de ayuda.

Para elegir el formato del sistema de ayuda, debe definir las opciones de compilación:

1. En la pestaña "Modificación", en el grupo "General", haga clic en "Opciones".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=aide_compilation%20-%20HC%20N%B0004%201.gif)


2. Seleccione el tipo de ayuda que desea generar y especifique las opciones de generación si es necesario. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=aide_compilation%20-%20HC%20N%B0004.gif)
Para obtener más información, consulte [Opciones de compilación](../Editeurs/2010015.md).





<a name="NOTE2_3"></a>


### Compilar un archivo de ayuda
<a name="compilar_archivo_ayuda_ELTPARAGRAPHE000091"></a>

Para compilar un archivo de ayuda: 

1. En la pestaña "Modificación", en el grupo "General", haga clic en "Compilar". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=aide_compilation%20-%20HC%20N%B0004%202.gif)
 Comienza la compilación.

2. Una vez completada la compilación:

	- Si el compilador no encontró ningún error, el archivo de ayuda se abre automáticamente.

	- Si el compilador detecta errores, se muestra el informe de la compilación. El informe muestra:

		- las advertencias: estas advertencias indican que hay enlaces inexistentes en el archivo de ayuda, por ejemplo.

		- las operaciones realizadas por el compilador, así como los errores y problemas encontrados. El botón "Imprimir" permite imprimir el informe.







Para probar el sistema de ayuda, haga clic en "Probar".
<a name="NOTE2_4"></a>


### Prueba del sistema de ayuda
<a name="prueba_del_sistema_ayuda_ELTPARAGRAPHE000123"></a>

El sistema de ayuda se puede probar mediante uno de los siguientes métodos:

- al compilar la ayuda (ver párrafo anterior). 

- en la pestaña "Modificación", en el grupo "General", haga clic en "Probar". Esta opción abre el último archivo de ayuda creado. Si se realizaron cambios desde la última compilación, estos no se tienen en cuenta.

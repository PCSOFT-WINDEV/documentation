
## Archivos y elementos de datos de zombie
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
Durante la vida de una aplicación, los items viven y mueren. Antes de morir, pueden convertirse en zombis.
Lo que significa que el item aún existe en la descripción de datos, que se puede usar en el código de la aplicación existente, pero que el item ya no se debe usar cuando se escribe un nuevo código.
Un elemento de zombie ocupa espacio y por lo tanto debe ser eliminado.
Los artículos de zombie se encuentran a menudo en el código muerto.
En la versión 23, tienes la capacidad de identificar los elementos de zombie en el análisis; serán señalados cuando se usen.

## Definir un archivo o un elemento como "zombie"
<a name="definir_archivo_elemento_como_zombie_ELTTEXTE000165"></a>


### Definir un archivo como "zombie"
<a name="definir_archivo_como_zombie_ELTPARAGRAPHE000024"></a>

Para definir un archivo zombie: 

1. En el editor de análisis, seleccione el archivo de datos para usar. 

2. Visualizar el archivo description ("description del archivo de datos" en el menú contextual). 

3. En la pestaña "Opciones: 

	- Seleccione la opción "Archivo obsoleto o zombi". 

	- Especifique una comment si es necesario. Este comment se mostrará en el error de compilación generado al usar el archivo zombie. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0001.gif)





4. Cierre la ventana de description del archivo de datos. 

5. En el gráfico de análisis, el archivo zombie se identifica con el ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0002%201.gif) Icon: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0002.gif)


6. En el panel "Análisis", el archivo zombie es Grayed y está tachado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0003.gif)


7. Generar el análisis para tener en cuenta las modificaciones. En el error de compilación, un nuevo error indica los usos del archivo zombie. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0003%201.gif)





Observación: También tiene la capacidad de realizar la operación inversa. 


### Definir un elemento como "zombi"
<a name="definir_elemento_como_zombi_ELTPARAGRAPHE000062"></a>

Para definir un archivo zombie: 

1. En el editor de análisis, seleccione el archivo de datos para usar. 

2. Visualizar la description de artículos ("description de artículos" en el menú contextual). 

3. En la pestaña "Avanzado: 

	- Seleccione la opción "Campo zombi (obsoleto)". 

	- Especifique una comment si es necesario. Este comment se mostrará en el error de compilación generado al usar el elemento zombie. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0004.gif&type=thumb)





4. Validar. 

5. En el gráfico del análisis, el elemento zombi está tachado en la representación del archivo de datos: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0005.gif&type=thumb)


6. En el panel "Análisis", el elemento zombie es Grayed fuera y tachado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_zombie%20-%20HC%20N%B0005%201.gif)


7. Generar el análisis para tener en cuenta las modificaciones. En el error de compilación, un nuevo error indica que se utiliza el elemento zombie. 




Observación: También tiene la capacidad de realizar la operación inversa.

## Manejar un archivo o artículo de zombie
<a name="manejar_archivo_articulo_zombie_ELTTEXTE000195"></a>
En los distintos editores que manipular ficheros y elementos de datos (editor de ventanas, editor de reportes, editor de consultas, etc.), los ficheros y elementos zombie pueden identificarse fácilmente porque están Grayed y tachados. 

Además, se utiliza un error de compilación para identificar fácilmente los elementos de zombie utilizados.

## Usando los patrones RAD
<a name="usando_los_patrones_rad_ELTTEXTE000150"></a>


### Los archivos de patrones
<a name="los_archivos_patrones_ELTPARAGRAPHE000029"></a>

Al generar el patrón RAD, se genera un archivo llamado &lt;Nombre del patrón&gt;.Patrón en el subdirectorio EXE del proyecto. Para ser utilizado en los diferentes productos, este archivo debe ser copiado:

- en el subdirectorio "\\Programas\\Datos\\Modelos\\RAD\\WD\\Patrones" para un patrón WINDEV.

- en el subdirectorio "\\Programs\\Data\\Models\\RAD\\WB\\Patterns" para un patrón WEBDEV.




Tan pronto como se copian estos archivos, los nuevos patrones RAD están disponibles en el producto correspondiente como RAD de aplicación completa (y como ventana o página RAD/RID si es necesario).
<a name="NOTE2_2"></a>


### Generación de un RAD de aplicación completa
<a name="generacion_rad_aplicacion_completa_ELTPARAGRAPHE000041"></a>

Para generar un RAD de Aplicación Completa:

1. En la pestaña "Proyecto", en el grupo "Generación", haga clic en "RAD Aplicación completa".

2. Seleccione el patrón a utilizar y el plantilla a aplicar si es necesario.

3. Siga los diferentes pasos de la asistente.


**Observación**: Si el asistente para la creación de la aplicación completa RAD se utiliza varias veces en el mismo proyecto, tiene la posibilidad de generar sólo los elementos correspondientes a los archivos añadidos al análisis. Se propone automáticamente una actualización del RAD cada vez que se modifica el análisis.: se añaden los elementos necesarios a la aplicación generada por el RAD.
<a name="NOTE2_3"></a>


### Generar una ventana RAD/RID o una página RAD/RID
<a name="generar_una_ventana_radrid_una_pagina_radrid_ELTPARAGRAPHE000066"></a>

Para generar una ventana o página RAD/RID:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. Aparece la ventana de creación: haga clic en "Ventana" y luego en "Ventana" o haga clic en "Página" y luego en "Página".

2. Para generar una ventana o página RAD, seleccione el patrón a utilizar en la pestaña "RAD" (así como el plantilla a utilizar si es necesario).

3. Para generar una ventana o página de RID, seleccione el patrón a utilizar en la pestaña "RID" (así como el plantilla a utilizar si es necesario)..
	Observación: las ventanas o páginas de RID utilizan los patrones RAD: el código encontrado en los patrones RAD no se genera.

4. Siga los diferentes pasos de la asistente.


**Observación**: En WEBDEV, sólo las páginas dinámicas utilizan los patrones RAD.

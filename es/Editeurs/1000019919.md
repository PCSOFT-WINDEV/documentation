


## Desarrollo de una aplicación para Apple Watch

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
Una aplicación Apple Watch es una aplicación destinada a un reloj asociado a un dispositivo iOS (Phone, tableta, ...).

Una aplicación del Apple Watch siempre es associated con una aplicación de iOS con la que puede intercambiar datos. 

Se deben realizar varios pasos para desarrollar una aplicación de Apple Watch: 

- [Creación y desarrollo del proyecto en WINDEV Mobile](#NOTE2_1). 

- [Generación de la aplicación](../Editeurs/1000019908.md). 

- [Compilación en Xcode](../Editeurs/1000019878.md). 




Esta ayuda Page presenta la creación de un proyecto Apple Watch y su desarrollo. El proyecto se genera y compila de la misma manera que una aplicación estándar de iOS.

<a name="NOTE2_1"></a>


## Creación de un proyecto para Apple Watch con WINDEV Mobile
<a name="creacion_proyecto_para_apple_watch_con_windev_mobile_ELTTEXTE000168"></a>
Para crear un proyecto de Apple Watch con WINDEV Mobile: 

1. Cree un proyecto para iOS (para obtener más información, consulte [Desarrollar una aplicación para iPhone/iPad](../Editeurs/1000019818.md)).

2. Cree una configuración de Apple Watch:

	- Abra el asistente para crear nueva configuración: en la pestaña "Proyecto", en el grupo "Configuración del proyecto", despliegue "Nueva configuración" y seleccione "Nueva configuración (mediante un asistente)". 
			Para obtener más información, consulte [Configuración del proyecto](../Editeurs/9000030.md).

	- En la etapa "Información general" del asistente, seleccione "Aplicación Apple Watch".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cr%E9erAppliAppleWatch%20-%20HC%20N%B0001.gif&type=thumb)


	- En la etapa "Aplicación iOS asociada" del asistente, seleccione la configuración iOS del proyecto asociado a la configuración de Apple Watch.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cr%E9erAppliAppleWatch%20-%20HC%20N%B0002.gif&type=thumb)

## Desarrollo de una aplicación para Apple Watch con WINDEV Mobile
<a name="desarrollo_una_aplicacion_para_apple_watch_con_windev_mobile_ELTTEXTE000192"></a>
El método para desarrollar una aplicación de Apple Watch y su aplicación associated iOS es idéntico al método para desarrollar una aplicación estándar.

**Para la aplicación iOS**, se crean los diferentes elementos de la aplicación. El proyecto se desarrolla igual que un proyecto WINDEV Mobile. Sin embargo, hay algunos aspectos específicos del desarrollo de iOS. Para obtener más información, consulte [Características específicas de aplicaciones iOS](../Editeurs/1000019865.md).

**Para la aplicación Apple Watch** , puede:

- crear ventanas. Estas ventanas están en formato Apple Watch.

- agregar controles a las ventanas. Sólo los controles Static, Button y Image están disponibles en una aplicación Apple Watch.

- utilizar funciones, propiedades y tipos WLanguage. La ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_AWatch.gif) Icon se utiliza para saber si la función está disponible en el Apple Watch.




**La aplicación iOS y Apple Watch** pueden intercambiar datos utilizando las siguientes funciones:


|   |   |
| --- | --- |
| [awProcedureGetAndSendInfo](../WDLang3/1000022118.md) | Especifica el WLanguage Procedure que se llamará cuando se reciba información procedente de [awSendAndGetInfo](../WDLang3/1000022116.md). |
| [awProcedureGetInfo](../WDLang3/1000022119.md) | Especifica el WLanguage Procedure que se llamará cuando se reciba información procedente de [awSendInfo](../WDLang3/1000022117.md). |
| [awSendAndGetInfo](../WDLang3/1000022116.md) | Envía datos de un iOS Phone a un Apple Watch (o al contrario) mientras espera una respuesta. |
| [awSendInfo](../WDLang3/1000022117.md) | Envía una información de un iOS Phone a un Apple Watch (o a la inversa). |





Cuando el desarrollo se complete, puedes generar el proyecto Xcode usado para compilar la aplicación Apple Watch y su aplicación associated iOS en MAC: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Generation_iOS_WM_bl.gif) en los botones de acceso rápido. 

2. Si es necesario, seleccione la primera ventana que aparece en las diferentes plataformas (iPhone, iPad y Apple Watch).

3. El asistente de generación se abre. Para obtener más información sobre la generación de proyectos en Xcode, consulte [Generación de la aplicación](../Editeurs/1000019908.md).

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000142"></a>
El control Casilla de verificación puede utilizar automáticamente varias AAF (Funcionalidades Automáticas de la Aplicación): 

- [Marcar todo](#NOTE2_1).

- [Desmarcar todo](#NOTE2_1).

- [Recordar el valor seleccionado](#NOTE3_1).

## Comprobar todo/Desproteger todo
<a name="comprobar_tododesproteger_todo_ELTTEXTE000166"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000034"></a>

La opción "Marcar todo" se utiliza para marcar todas las opciones que se encuentran en la casilla de verificación. 

La opción "Desmarcar todo" se utiliza para desmarcar todas las opciones que se encuentran en la casilla de verificación. 


### Implementación
<a name="implementacion_ELTPARAGRAPHE000041"></a>

Las opciones "Check all" y "Uncheck all" se implementan automáticamente en los controles de la casilla de verificación a partir de la versión 20. Estas opciones están disponibles para todos los tipos de controles Check Box (estándar, Vista,...).  

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Para obtener más información, consulte [Configuración de los menús de AAF](../Editeurs/2010040.md).

**Mediante programación**: 

- [AAFDisable](../WDLang1/1000022018.md) con la constante *aafCheckBoxCheckAll* permite eliminar las opciones utilizadas para marcar o desmarcar todo del menú contextual del control Casilla de verificación especificado.

- [AAFExecute](../WDLang1/1000022099.md) asociada a la constante *aafCheckBoxCheckAll* se utiliza para comprobar todas las opciones encontradas en el control Casilla de verificación especificado.

- [AAFChangeCaption](../WDLang1/1000022100.md) permite modificar el texto del menú contextual utilizado para comprobar todas las opciones de un control Casilla de verificación.

## Guardar valores
<a name="guardar_valores_ELTTEXTE000196"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000069"></a>

Para guardar el valor seleccionado en un Cuadro de Lista o control Combo Box, seleccione "Recordar valor" en el menú contextual del control.


### Implementación
<a name="implementacion_ELTPARAGRAPHE000074"></a>

La opción "Recordar valor" está disponible automáticamente en los controles Combo Box y List Box. Esta opción permite gestionar la [persistencia de los datos](../WDChamp/9000019.md). 

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Para obtener más información, consulte [Configuración de los menús de AAF](../Editeurs/2010040.md).

**Mediante programación**: [AAFDisable](../WDLang1/1000022018.md) con la constante *aafSaveValue* se utiliza para eliminar la opción "Recordar valor" del menú contextual del control Casilla de verificación especificado.

**Observación**: También puede guardar el valor a través de la opción "Recordar valor" en la ventana de control de description (pestaña "Detalles"). Esta opción se utiliza para guardar el valor incluso si la opción de menú no fue activada por el usuario. La opción de menú tiene prioridad si fue habilitada por el usuario.

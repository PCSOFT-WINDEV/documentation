
## Gestionar automáticamente las URLs cortas
			

<a name="NOTE0_0"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
Para los sitios WEBDEV dinámicos estándar (con contexto), el acceso al sitio puede realizarse: 

- via "standard" URL:
	
	```txt
	http://www.mysite.com/WD260AWP/WD260AWP.EXE/CONNECT/ProjectName
	```


- a través de una URL corta. Un URL corto es más fácil de recordar y hace que Transparent sea la versión de WEBDEV servidor de aplicaciones utilizada.
	
	```txt
	http://www.mysite.com/ProjectName
	```





**Los dos tipos de URL están disponibles automáticamente al instalar un sitio WEBDEV dinámico estándar.**

**Atención**: Las URLs cortas sólo funcionan con los siguientes servidores Web: 

- Apache 2.2 y posterior,

- IIS 7 y posteriores. 




**Observación**: El mecanismo de URLs cortas sólo funciona para los sitios dinámicos estándar. Para los sitios AWP, el nombre del servidor de aplicaciones puede ocultarse en la URL de las páginas de un sitio AWP a través del [mecanismo para la reescritura de URLs](../Editeurs/2030054.md).

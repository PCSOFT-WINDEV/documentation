## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
El editor de consultas le permite optimizar la velocidad de ejecución de una o más consultas.

La optimización de una consulta se basa en el siguiente principio: una asistente define las claves compuestas a modificar y/o añadir en el análisis associated con el proyecto.|   |   |   |

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000165"></a>


### Optimización de la velocidad de ejecución de la consulta actual
<a name="optimizacion_velocidad_ejecucion_consulta_actual_ELTPARAGRAPHE000033"></a>

Optimizar la velocidad de ejecución de la consulta actual: 

1. En la pestaña "Consulta", en el grupo "Analizar", haga clic en "Optimizar la consulta".

2. Se muestra la ventana de optimización de consultas y propone las diferentes modificaciones que se pueden realizar en el análisis.



<a name="NOTE2_2"></a>


### Optimización de la velocidad de ejecución de una o más consultas
<a name="optimizacion_velocidad_ejecucion_una_mas_consultas_ELTPARAGRAPHE000045"></a>

Para optimizar la velocidad de ejecución de una o más consultas: 

1. En la pestaña "Consulta", en el grupo "Analizar", haga clic en "Optimizar la consulta".

2. Se muestra la ventana de optimización de consultas y propone las diferentes modificaciones que se pueden realizar en el análisis.




**Atención**: La adición de un gran número de claves compuestas al análisis aumenta el tamaño del archivo índice y puede ralentizar el acceso a los archivos de datos.

**Observación**: Esta opción sólo está disponible si se cumplen las siguientes condiciones:

- los Datos en Vivo están habilitados: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". Seleccione la pestaña "Live Data" y marque "Enable the Live Data".. Para obtener más información, consulte [Live Data](../Editeurs/2021005.md).

- los archivos de datos se encuentran en el directorio de prueba del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". Seleccione la pestaña "Archivos". Para obtener más información, consulte [Probar un proyecto](../Editeurs/2019017.md).



<a name="NOTE2_3"></a>


### Optimización y claves compuestas
<a name="optimizacion_claves_compuestas_ELTPARAGRAPHE000070"></a>

Para optimizar sus consultas, el editor propone la creación de claves compuestas. El orden de los componentes en la clave compuesta no es importante.

No hay necesidad de crear una nueva clave que simplemente revierta el orden de los componentes que se encuentran en una clave existente.

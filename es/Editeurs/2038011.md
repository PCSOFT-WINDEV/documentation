


## Barra de herramientas y menú de SCM
			





## Menú SCM
<a name="menu_scm_ELTTEXTE000165"></a>
Las operaciones sobre la SCM pueden realizarse desde la pestaña SCM de WINDEV, WEBDEV o WINDEV Mobile.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0002.gif&type=thumb)
 Esta pestaña contiene diferentes áreas: 

- "Repositorio".

- "Todos los elementos". 

- "Elemento actual". 

- "Proyecto". 

- "Otras acciones". 




Veamos las opciones de estas diferentes áreas. 


### " SCM" zona
<a name="scm_zona_ELTPARAGRAPHE000021"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0001.gif)
La opción "Gestionar" se utiliza para: 

- Abra el administrador del SCM. Para obtener más información, consulte [Administrador del SCM](../Editeurs/2038008.md). 

- Crear un repositorio. Para obtener más información, consulte [Crear un repositorio](../Editeurs/2038005.md). 

- Definir una política de facturación. Para obtener más información, consulte [SCM: Política de reintegración](../Editeurs/2038024.md).





### "Área"Todos los elementos
<a name="areatodos_los_elementos_ELTPARAGRAPHE000040"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0002%201.gif)
Esta área se utiliza para gestionar todos los elementos del proyecto. 

Es posible: 

- **Obtener todo**: Utilizado para recuperar todas las nuevas versiones de los elementos del proyecto que se encuentran en el repositorio de SCM.. No se proponen los elementos actualmente comprobados por el usuario.

- **Reintegrar todo**: Se utiliza para registrar todos los elementos del proyecto en el repositorio de SCM.. Algunos elementos pueden permanecer en la caja.




La opción "Estado de los elementos" se utiliza para: 

- Conocer el estado de los elementos: Se muestra la lista de los elementos del proyecto, con la fecha de la última salida y su autor para cada elemento..  

- Actualizar el estado de los elementos: Se utiliza para actualizar la lista de elementos del proyecto y el estado de cada elemento. 





### "Área"Elemento de corriente
<a name="areaelemento_corriente_ELTPARAGRAPHE000056"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0002%202.gif)
Esta área se utiliza para gestionar el elemento actual en el editor. 

Es posible: 

- **Extraer**: Se utiliza para comprobar el archivo correspondiente al elemento actual desde el repositorio SCM.. La última versión del elemento se copia en el ordenador local.. Hay varios modos de pago disponibles. Para obtener más información, consulte [Comprobación de un elemento](../Editeurs/2038006.md).

- **Reintegrar**: Permite registrar el fichero correspondiente al elemento actual en el repositorio de SCM.. El elemento se copia en el repositorio de SCM. Se pone a disposición de los demás usuarios de SCM. Para obtener más información, consulte [Comprobación de un elemento de nuevo en](../Editeurs/2038007.md).

- **Undo la caja**: Le permite a Cancel la última salida.. Las modificaciones se cancelan y el archivo vuelve a "Sólo lectura".

- **Obtener la última versión**: Permite conocer la última versión del elemento actual en el editor.. Esta última versión es la que se encuentra en el repositorio de SCM. 

- **Comparar**: 

	- **Ver mis modificaciones desde la extracción**: Permite visualizar las modificaciones que se han realizado en la unidad actual desde que se verificó.

	- **Comparar con la versión del repositorio**: Compara el elemento actual (disponible localmente) con la versión anterior en el repositorio SCM.. Esta comparison se puede realizar incluso si el elemento no ha sido comprobado. Permite comprobar las modificaciones realizadas antes de volver a facturar el elemento.




- **Add**: Permite añadir al repositorio de SCM el elemento que se encuentra actualmente en el editor.. Si se utiliza esta opción desde el editor de análisis, el análisis se añade al repositorio. Para obtener más información, consulte [Añadir un elemento al repositorio](../Editeurs/2038009.md).

- **Propiedades**: Permite visualizar las propiedades del elemento actual (las diferentes cajas, las acciones,...).

- **Historial**: Usó a get la lista de modificaciones hechas al elemento y quién las realizó. Puedes get una versión anterior, guardarla, buscar las diferencias (entre 2 versiones por ejemplo) o fusionar la versión SCM con tu versión.





### "Área"Proyecto
<a name="areaproyecto_ELTPARAGRAPHE000085"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0002%203.gif)
Esta área se utiliza para gestionar el proyecto actual en el editor. 

Es posible: 

- **Agregar proyecto**: Permite añadir el proyecto actualmente abierto en el editor en el repositorio de SCM.. 

- **Importar desde el SCM**: Se utiliza para importar al proyecto actual un elemento que se encuentra en la SCM (un componente o una ventana que se encuentra en otro proyecto, por ejemplo).. Puedes compartir el elemento o importar un copy.

- Gestionar el modo **Maestro/Invitado**: esta opción se utiliza para gestionar las opciones de gestión de los proyectos maestro e invitado. Para obtener más información, consulte [Trabajar con la Administrador de Código Fuente](../Editeurs/2038004.md).

- Mostrar **la historia del proyecto**: Permite a get la lista de modificaciones realizadas en todo el proyecto y en el archivo del proyecto (archivo WDP, WWP o WPP). Tienes la posibilidad de get una versión anterior, guardarla, encontrar las diferencias (entre 2 versiones por ejemplo), etiquetar la versión de proyecto, ... 

- Gestionar las sucursales de **-**: Esta opción permite: 

	- Crear una rama. 

	- Recuperar las modificaciones de una rama. 
			Para obtener más información, consulte [SCM: Gestión de sucursales](../Editeurs/2038016.md). 








### "Área"Otras acciones
<a name="areaotras_acciones_ELTPARAGRAPHE000110"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Barre_outils%20-%20HC%20N%B0002%204.gif)
Esta área se utiliza para gestionar la hoja de estilo del proyecto y el trabajo remoto.. 

Es posible: 

- Revisa o revisa la hoja de estilo del proyecto. 

- Desconectar para un uso móvil y luego volver a conectar y sincronizar.

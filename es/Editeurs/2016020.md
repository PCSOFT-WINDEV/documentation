## Presentación
<a name="presentacion_ELTTEXTE000138"></a>
Los skins se utilizan para estandarizar el aspecto visual de un sitio y le permiten cambiar fácilmente de estilo.

Un tema contiene:

- una hoja de estilo con estilos WEBDEV.

- un directorio de imágenes.

- una paleta de colores predeterminada.

- un conjunto de reportes que se pueden utilizar como base para construir nuevos reportes del proyecto.




Echemos un vistazo a estos elementos: 

- La hoja de estilo: 
	**Recordatorio**: Una hoja de estilo incluye estilos WEBDEV. Cada estilo WEBDEV agrupa: 

	- uno o más estilos CSS,

	- valores de las propiedades para el control.
			Un estilo de WEBDEV se aplica a un tipo específico de control.
			Los estilos de una tema tienen nombres estandarizados. Por lo tanto, las pieles pueden intercambiarse entre sí.




- Las imágenes: 
	Las imágenes suministradas con la tema se utilizan para definir el aspecto visual de las celdas, tablas, controles de pestañas, etc.

- El Color Palette: 
	El Color Palette por defecto de un tema puede ser sustituido por cualquier otro Color Palette. Para obtener más información, consulte: [Paletas de colores](../Editeurs/2016021.md).






## Secuencia de aplicación de las pieles
<a name="secuencia_aplicacion_las_pieles_ELTTEXTE000162"></a>
El tema de una página puede definirse en tres niveles diferentes (en orden descendente de prioridad):

- En el proyecto.

- En el modelo de páginas del que la página hereda propiedades.

- En la misma página.




El tema general debe definirse a nivel del proyecto. 

Si un set de páginas debe utilizar un tema específico (las páginas de "Promoción" de un sitio, por ejemplo), tiene la posibilidad de reemplazar el proyecto tema: todo lo que tienes que hacer es redefinirlo para una página modelo. 

Si una página en particular debe tener un tema diferente, este tema puede ser elegido para la propia página.



## Uso de las pieles
<a name="uso_las_pieles_ELTTEXTE000186"></a>


### Elección de la tema de un proyecto
<a name="eleccion_tema_proyecto_ELTPARAGRAPHE000055"></a>

Elegir la tema de un proyecto:

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Selecciona la pestaña "Estilos de Tema".

3. Elija un tema entre los propuestos (así como su color Palette) y valide. 





### Modificación de la tema de una página o de la página modelo
<a name="modificacion_tema_una_pagina_pagina_modelo_ELTPARAGRAPHE000076"></a>

Para cambiar la tema de una página o la página modelo: 

1. Seleccione la opción "Descripción" en el menú contextual de la página.

2. Seleccione la pestaña "Estilo".

3. Haga clic en el botón de navegación para seleccionar una tema.

4. Si el tema seleccionado es "Ninguno", se utilizará el tema del proyecto.






## Actualizando temas
<a name="actualizando_temas_ELTTEXTE000216"></a>
Si un tema utilizado por un proyecto ha sido actualizado (es decir, modificado), WEBDEV propone actualizar este proyecto cuando se abra. 

Tiene la posibilidad de no seleccionar esta nueva versión del tema, ya que el tema anterior se almacena directamente en cada proyecto..

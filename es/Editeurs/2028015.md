


## WDInst: incluyendo un "autorun" en una instalación en CD
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
Un "autorun" puede ser ejecutado (o no) cuando el CD de instalación se inserta en la unidad de la computadora de instalación. Este "autoejecución" se utiliza para arrancar automáticamente:

- el programa instalación.

- una ventana de menú específica. El editor WDInst de instalación le permite configurar esta ventana.
	Este menú puede iniciar el programa instalación, mostrar un archivo "Léeme", abrir un sitio de INTERNET, ...




Por Default, no se ejecuta ningún "autorun" al insertar el CD.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000184"></a>


### Gestión de la ejecución automática
<a name="gestion_ejecucion_automatica_ELTPARAGRAPHE000025"></a>

La gestión de un "autorun" se realiza en WDInst (el editor de instalación). 

Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

La creación del programa instalación asistente no permite un "autorun".
<a name="NOTE2_2"></a>


### Incluyendo un "autorun" en el CD de instalación
<a name="incluyendo_autorun_instalacion_ELTPARAGRAPHE000048"></a>

Incluir un "autorun" en el CD de instalación:

1. En WDInst, seleccione "Setup parameters .. Advanced parameters".

2. Seleccione el tipo de "autorun" que desea incluir (opción "Acción realizada al insertar el CD"). Es posible: 

	- iniciar el instalador automáticamente.

	- ejecutar un menú de ejecución de automatic. 







**Observación**: Default no incluye ningún "autorun".
<a name="NOTE2_3"></a>


### Configurar el menú de "autoejecución".
<a name="configurar_menu_autoejecucion_ELTPARAGRAPHE000067"></a>

Configurar el menú de "autoejecución":

1. En WDInst, seleccione "Setup parameters .. Advanced parameters".

2. Seleccione "Iniciar un menú de ejecución de automatic". 

3. Haga clic en el botón "Parámetros".

4. Configure las diferentes opciones del menú en las pestañas de configuración:

	- **Pestaña General**: Se utiliza para definir:

		- el título de la ventana. Este título se mostrará en la barra de título de la ventana. Escriba el nombre de la aplicación: "WDScooter.

		- el gran título y su Color. Este título será la primera Line que se muestre en la ventana. Especifique el propósito de la aplicación: "Gestión de inventario".
						**Atención**: El número máximo de caracteres está limitado. Tomando como referencia las letras "i" y "m", el título principal puede contener entre 15 y 60 caracteres.

		- el subtítulo y su Color. Este título se mostrará debajo del título grande. Escriba un mensaje de bienvenida: "Bienvenido al programa instalación".
						**Atención**: El número máximo de caracteres está limitado. Tomando como referencia las letras "i" y "m", el subtítulo puede contener entre 30 y 100 caracteres.

		- la ventana Background. Este parámetro puede corresponder a:

			- un *sólido Color*. Seleccione el fondo Color.

			- un *Image*. Esta Image puede ser embaldosada o estirada. Seleccione la imagen que desea utilizar.




- **Ficha Botones**: La ventana del menú puede contener hasta 8 **pulsadores**.

	1. Para cada botón, especifique:

		- los ***pie de foto***. Esta leyenda se centrará en el botón.

		- el ***description***. Esta description se mostrará en la sección izquierda de la ventana cuando el botón correspondiente sea pasado por el cursor del ratón.

		- el comando ***y*** se ejecuta o el ***Document*** se abre con este botón. Cada botón puede:

			- iniciar un archivo ejecutable. Por ejemplo: "Install.EXE". Para ser iniciado, este archivo debe estar disponible en el CD o debe encontrarse en uno de los directorios definidos en el PATH de la computadora de instalación.

			- mostrar un Document. Por ejemplo: "Léeme.DOC". Para poder ser visualizado, este archivo debe estar disponible en el CD o debe encontrarse en uno de los directorios definidos en el PATH de la computadora de instalación.

			- abrir un sitio INTERNET. Por ejemplo: "http://www.windev.es". Para abrir este sitio, el ordenador instalación debe estar equipado con una conexión permanente a INTERNET.

			- iniciar un explorador. "E:\\MyApp" por ejemplo. Para ello, especifique en la columna "Command/Document" el directorio en el que debe abrirse el explorador.




El botón "Test" se utiliza para comprobar los parámetros elegidos para el programa de ejecución automatic del CD. El menú se muestra con los botones especificados, las imágenes y los colores configurados..

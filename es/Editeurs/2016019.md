


## Modelos de páginas predefinidos
			





## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
Las plantillas de páginas preestablecidas proporcionan algunas estructuras de páginas correspondientes a una amplia gama de sitios y también mejoran la reutilización y la armonía gráfica entre los proyectos.

Con WEBDEV se suministran varias plantillas y con [el LST](../Presentation/3084010.md) se suministran regularmente nuevas plantillas.



## Uso de las plantillas de página predefinidas
<a name="uso_las_plantillas_pagina_predefinidas_ELTTEXTE000113"></a>
Para utilizar una página preestablecida modelo:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido.

2. Aparece la ventana de creación: haga clic en "Página", y luego en "Página".

3. En la ventana para crear una nueva página, seleccione un modelo en la sección "Utilizar un modelo preestablecido".




WEBDEV añade automáticamente dos elementos al proyecto:

- una página modelo que corresponde a la modelo seleccionada y que utiliza el proyecto tema y el color Palette.

- una página en blanco que utiliza la modelo.





### Observaciones
<a name="observaciones_ELTPARAGRAPHE000031"></a>

- Si el modelo preestablecido ya se utiliza en el proyecto, se añadirá un nuevo copy. Para que varias páginas se basen en la misma configuración prefijada modelo, debe hacerlo:

	1. Crear una primera página con el método Above mencionado para importar el modelo preestablecido en el proyecto.

	2. Al crear las otras páginas, elija la modelo añadida al proyecto. Esta modelo se propondrá en la sección "Crear una página en blanco o una página asociada a un proyecto modelo".




- El mismo modelo preestablecido puede ser añadido varias veces en un proyecto para que pueda ser usado con diferentes pieles o diferentes paletas Color por ejemplo.

- Una vez que se agrega el modelo predefinido al proyecto, puede abrirse en el editor y modificarse como cualquier otro modelo de páginas. Para más información, véase [Modelos de páginas](../WDChamp/1012015.md).

- Un modelo de páginas predefinido también se puede utilizar como base para crear otro modelo de páginas.

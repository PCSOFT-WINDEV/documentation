


## Ventana description de una consulta (insertar, actualizar o borrar) o ventana description de una consulta en código SQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000083"></a>
Se utiliza la ventana description de una consulta (insertar, actualizar o borrar) o la ventana description de una consulta en código SQL:

- averiguar y/o modificar el nombre y el título de la consulta.

- especificar si el motor HFSQL debe verificar o no los comandos SQL utilizados (opción "Ejecutar con hQueryWithoutCorrection en modo de prueba").

- especifique la conexión utilizada durante la prueba de la consulta.

- averiguar y/o modificar los valores para insertar o modificar en el resultado de la consulta (insertar consulta o actualizar consulta).

- averiguar y/o modificar las condiciones de selección (actualizar consulta o borrar consulta).

- averiguar y/o modificar la secuencia de condiciones de selección (actualizar consulta o borrar consulta).

- ...

	- Aparece la ventana de creación: haga clic en "Consultas".




- de una consulta que se encuentra en el editor de la consulta ("Consulta description" de el menú contextual de la consulta).


## Configuración de la replicación universal asistida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000133"></a>
Si su aplicación utiliza la replicación universal asistida a través de un archivo .WER, WDInst propone configurar la instalación para instalar las diferentes herramientas necesarias para este tipo de replicación.

La réplica universal asistida se configura en "Parámetros de configuración .. Configurar la réplica universal asistida".

Los diferentes pasos de esta configuración son los siguientes:

- [Modo de funcionamiento de la réplica](#NOTE2_1).

- [Ubicación de la base de datos maestra](#NOTE3_1).

- [Ubicación de la base de datos maestra en modo interactivo](#NOTE4_1).

- [Ubicación de la base de datos local](#NOTE5_1).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Modo de funcionamiento de la réplica
<a name="modo_funcionamiento_replica_ELTTEXTE000157"></a>
Una aplicación WINDEV puede hacerlo:

- No usar la replicación.

- Utilice la replicación y opere siempre en modo online o replicado.

- Utilice la replicación y opere siempre en modo replicado.




Esta información puede ser modificada (o no) durante la instalación.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Ubicación de la base de datos maestra
<a name="ubicacion_base_datos_maestra_ELTTEXTE000181"></a>
La base de datos maestra puede serlo:

- descrito en un archivo.WER. Para ello, seleccione el archivo.WER correspondiente mediante el botón ".WER file. El archivo.WER puede ser editado directamente.

- accesible a través de un servidor de replicación. La información sobre el servidor de replicación debe introducirse mediante el botón "Servidor. Esta información afecta a la Address del servidor, el nombre de usuario y la contraseña. También es posible:

	- "Publicar los parámetros de la replicación en el servidor". Esta opción hace que la replicación sea inmediatamente utilizable.

	- "Permitir que estos parámetros se modifiquen durante la instalación": el usuario de la aplicación podrá modificar los parámetros del servidor de replicación.

	- "Utilice estos valores incluso si han sido personalizados durante una actualización anterior".




- configurado durante la instalación solamente. Los diferentes parámetros se preguntarán al usuario que está instalando la aplicación.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Ubicación de la base de datos maestra en modo interactivo
<a name="ubicacion_base_datos_maestra_modo_interactivo_ELTTEXTE000205"></a>
Si la aplicación ofrece la posibilidad de utilizar el modo en línea, debe definir la base de datos maestra que se utilizará para el modo en línea.. El botón "Master database" sirve para definir las características de las conexiones utilizadas por la aplicación.

Si la aplicación utiliza una base de datos HFSQL Classic, se crea automáticamente una conexión.

Es posible: 

- Utilizar las conexiones definidas en el análisis: Los parámetros de las conexiones definidas en el análisis se utilizarán para acceder a la base de datos maestra.

- Utilice las conexiones definidas en el archivo.WER: Los parámetros de las conexiones definidas en el archivo.WER se utilizarán para acceder a la base de datos maestra.

- Definir los parámetros de conexión ahora: Los parámetros de conexión se definen en el siguiente paso de la asistente.

- Solicitar los parámetros de conexión durante la instalación: Los parámetros de las conexiones son definidos por la persona que instala la aplicación.

## Ubicación de la base de datos local
<a name="ubicacion_base_datos_local_ELTTEXTE000229"></a>
Por Default, la base de datos local de la aplicación se crea en el directorio "Documents and Settings\\All Users\\Application Data\\&lt;Application Name&gt;".. El botón "Base de datos local" sirve para modificar este directorio.

Observación: Este directorio está pensado para funcionar con todo tipo de sistemas (incluyendo Windows Vista o posterior).. Les aconsejamos que no modifiquen esta ruta.

## Presentación
<a name="presentacion_ELTTEXTE000085"></a>
.NET es la plataforma de desarrollo de software de Microsoft. Esta plataforma .NET simplifica el desarrollo de aplicaciones de objetos distribuidos cuyos módulos se comunican a través de INTERNET.

Esta ayuda explica globalmente cómo usar la tecnología.NET en WINDEV. Consulte una documentación específica para obtener más detalles.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

### Creación de conjuntos.NET
<a name="creacion_conjuntosnet_ELTPARAGRAPHE000022"></a>

Usted tiene la capacidad de crear programas.NET (o ensamblajes.NET) desde una aplicación WINDEV.

Estos ensamblajes.NET se crean directamente a partir de clases WINDEV (con todas las ventajas del editor de código, del editor de documentación, etc.).
<a name="NOTE2_2"></a>


### Uso de ensamblajes.NET existentes
<a name="uso_ensamblajesnet_existentes_ELTPARAGRAPHE000031"></a>

Puede utilizar ensamblajes .NET (no creados con WINDEV) en una aplicación WINDEV o en un sitio WEBDEV:

- en una nueva aplicación: para ello, especifique el uso de ensamblajes .NET en la creación del proyecto asistente.

- en una aplicación existente: para hacerlo, seleccione los ensamblajes.NET que desea utilizar.

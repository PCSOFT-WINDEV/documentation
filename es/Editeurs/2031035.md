## ¿Qué páginas deben crearse?
<a name="¿que_paginas_deben_crearse_ELTTEXTE000269"></a>


### Los ficheros a crear
<a name="los_ficheros_crear_ELTPARAGRAPHE000043"></a>

Recordatorio: Para construir un patrón RAD utilizado para crear un proyecto completo, se requieren los siguientes archivos:

- Un formulario para RADFileA.

- Un formulario para RADFileB. Un botón Vision Plus (o un cuadro combinado emergente) debe permitirle seleccionar un elemento de RADFileA.

- Un formulario para RADFileC. Un botón Vision Plus (o un cuadro combinado emergente) debe permitirle seleccionar un elemento de RADFileB.

- Una tabla para RADFileA.

- Una tabla para RADFileB.

- Un tabla para RADFileC.

- Una página de relación RADFileB/RADFileD.




El menú principal del sitio puede ser incluido en el modelo associated con las páginas.

## Las páginas de Visión Plus
<a name="las_paginas_vision_plus_ELTTEXTE000299"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000071"></a>

Las páginas de Visión Plus están acostumbradas a:

- seleccionar un elemento de RADFileA de un formulario en RADFileB.

- seleccionar un elemento de RADFileB de un formulario en RADFileC.




Ejemplo: Selección de un cliente a partir de un pedido.

Los siguientes elementos le permiten utilizar una visión más página:

- en la página del formulario: un cuadro combinado y un botón.

- una página de Visión Plus. Esta página puede ser mostrada por el botón como una caja de diálogo, por ejemplo. Este caso se presenta en esta página de ayuda.



<a name="NOTE3_2"></a>


### Creación de la página Visión Plus
<a name="creacion_pagina_vision_plus_ELTPARAGRAPHE000088"></a>

La página Visión Plus contiene:

- una tabla.

- un botón Validar (para validar la selección del elemento) y un botón Cancel.




Para crear una página de Visión Plus: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- Aparece la ventana de creación: haga clic en "Página", y luego en "Página".

	- En la asistente que comienza, seleccione "Blank" y valide.




2. En la ventana de description, introduzca el nombre de la página. Este nombre debe contener necesariamente el nombre del archivo mostrado en la tabla (algunos ejemplos: PAGE_Vision_RADFileA, PAGE_Visio_RADFileA, ....).
	Observación: Si desea indicar el nombre del archivo actual al usuario, utilice el nombre del archivo (RADFileA por ejemplo) en una leyenda. El término RADFileX será reemplazado automáticamente por el título del archivo descrito en el análisis.

3. Guardar la página.

4. Crear un control Tabla en la página: en la pestaña "Creación", en el grupo "Datos", despliegue "Tabla" y seleccione "Tabla".

5. En la creación de tabla asistente, especifique las siguientes características:

	- Seleccione "Mostrar los datos de un fichero o de una consulta".

	- Seleccione el fichero base de la tabla (RADFileA por ejemplo).

	- Conserve el artículo propuesto así como el artículo de búsqueda.

	- Validar el nombre de la tabla.




6. Posicione la control Tabla en la página y defina su tamaño. 

7. Crear y posicionar los diferentes botones de su página: Validar y Cancel.

8. Caso de la ***Botón Validar***.
	Este botón se utilizará para validar la selección en la tabla. El código del botón Validar es el siguiente: 
	
	```wl
	// Close the page with True in return value
	PageCloseDialog(True)
	```

	Observación: la página se muestra como un cuadro de diálogo (PageDisplayDialog). El valor de True puede ser devuelto a la página de llamada.

9. Caso de los botones ***Cancel***
	Este botón se utiliza para Cancel la selección.
	
	```wl
	// Close the page with False in return value
	PageCloseDialog(False)
	```


10. Código de inicialización de la página
	
	```wl
	// Position on the current record
	TableDisplay(TABLE_RADFileB, taCurrentFirst)
	
	// Default return value
	MyPage..ReturnedValue = False
	```

### Creación de los elementos necesarios en la página del formulario
<a name="creacion_los_elementos_necesarios_pagina_del_formulario_ELTPARAGRAPHE000147"></a>

Las páginas de Visión Plus se pueden utilizar desde las siguientes páginas:

- Formulario en RADFileB

- Formulario en RADFileC


En estas páginas se debe crear un cuadro combinado y un botón. Estos elementos se pueden apilar sobre el supercontrol correspondiente a la forma.

Para crear el cuadro combinado: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en "Combo Box", y luego haga clic en la página.

2. En el cuadro combinado de creación de asistente, seleccione las siguientes opciones:

	- "Mostrar datos provenientes de un archivo o de una consulta".

	- Archivo de datos: RADFileA para un formulario en RADFileB, RADFileB para un formulario en RADFileC.

	- Conserve las opciones predeterminadas. El elemento almacenado debe ser el identificador del archivo.

	- El cuadro combinado es relacionado a ningún elemento.

	- el nombre del control es "COMBO_IDRADArchivoX" (COMBO_ es el prefijo definido en el estilo de código: no es necesario si no se utiliza un estilo de código). Por ejemplo, para el formulario RADFileB, el cuadro combinado se llamará COMBO_IDRADFileA.




3. Validar. Se crea la control Combo Box.




Para crear el botón: 

1. En la pestaña "Creación", en el grupo "Controles frecuentes", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_bouton.gif), y luego haga clic en la página.

2. En la ventana de control de description, especifique el nombre del botón. Esta control tiene un nombre con el siguiente formato "BTN_VISION_RADFileX" (BTN_ es el prefijo definido en el estilo de código: no es necesario si no se utiliza un estilo de código). Por ejemplo, para el formulario RADFileB, el botón se llamará BTN_VISION_RADFileA.

3. El código de clic del botón es el siguiente:
	
	```wl
	// Open the vision page of RADFileB
	IF PageDisplayDialog(PAGE_Vision_RADFileB) = True THEN
		// Refreshes the content of the RADFileB combo box
		ListDisplay(COMBO_IDRADFileB)
		// Assign and position the combo box
		COMBO_IDRADFileB = RADFileB.IDRADFileB
	END
	```



Este código se utiliza para abrir una página tipo cuadro de diálogo para seleccionar los registros.

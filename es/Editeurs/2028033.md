


## WDInst: configurar las características de los archivos a instalar
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000199"></a>
WDInst le permite configurar las características de los archivos a instalar. Es posible:

- Especifique el directorio de destino de uno o más archivos.

- Especificar los ordenadores en los que debe instalarse cada archivo (red instalación).

- copy y comprimir uno o más archivos.




Predeterminado:

- El directorio de destino de los archivos corresponde al directorio de instalación elegido por el usuario durante la instalación.

- Los archivos de datos (".FIC", ".NDX" y ".MMO") se instalan en el servidor y todos los demás archivos se instalan en los ordenadores del usuario.

- Los archivos se comprimen cuando se genera el programa instalación y se instalan en el ordenador de instalación.




Las características de los archivos a instalar están configuradas en WDInst. Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

A diferencia del instalación creación del programa asistente , WDInst ofrece una gestión avanzada de las características de los archivos a instalar.

## Especificar el directorio de destino de uno o más archivos a instalar
<a name="especificar_directorio_destino_uno_mas_archivos_instalar_ELTTEXTE000223"></a>
El directorio de destino se especificó cuando se agregó el archivo a la lista de archivos a instalar.

Para modificar el directorio de destino de uno o más archivos:

1. Seleccione el o los archivos deseados en la ventana "Lista de archivos instalación"..
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Mostrar la pestaña "Path" de la ventana "File description".

3. Especificar el directorio de destino de los archivos seleccionados ("Directorio"). Se pueden utilizar las siguientes palabras clave:

	- **&lt;SetupDir&gt;&gt;SetupDir**: Directorio de instalación elegido por el usuario durante la instalación.

	- **Configuración de la unidad&gt; &lt;SetupDrive**: del directorio de instalación elegido por el usuario durante la instalación.

	- **&lt;Idioma&gt;**: número de la lengua instalación.

	- **&lt;Archivo de configuración&gt;&gt;.**: El directorio de archivos de datos de instalación. WDInst se usa para configurar este directorio. Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

	- **&lt;DirUserData&gt;**: directorio de datos para el usuario conectado (especialmente para una instalación realizada en un ordenador con Windows Vista o posterior). 

	- **&lt;DirCommonData&gt;**: directorio de datos comunes a los usuarios (especialmente para una instalación realizada en un ordenador con Windows Vista o posterior). 

	- **&lt;Usuario&gt;**: Nombre de usuario

	- **&lt;srXXX&gt;**: Constantes de [SysDir](../WDLang1/3073026.md). 
			Por ejemplo: "&lt;SetupDir&gt;\\MyDirectory".




4. Si se selecciona un solo archivo, este archivo puede ser renombrado durante su instalación. Simplemente especifique el nuevo nombre del archivo ("Archivo").




**Observación**: El directorio de destino de un fichero también puede modificarse en la ventana "Lista de ficheros instalación". Para ello, haga clic en la columna "Directorio de destino" e introduzca el nuevo directorio de destino.

## Especificando los ordenadores en los que debe instalarse cada archivo (red instalación)
<a name="especificando_los_ordenadores_los_que_debe_instalarse_cada_archivo_red_instalacion_ELTTEXTE000247"></a>


### Si usted está realizando una red instalación
<a name="usted_esta_realizando_una_red_instalacion_ELTPARAGRAPHE000079"></a>

- Sólo en el servidor (archivos de datos compartidos por todos los usuarios, por ejemplo).

- Sólo en los ordenadores de los usuarios (el ejecutable de la aplicación, por ejemplo).

- Tanto en el servidor como en las computadoras de los usuarios (un archivo ".INI" utilizado para configurar la aplicación instalada en el servidor con los parámetros de Default y las aplicaciones instaladas en las computadoras de los usuarios con los parámetros específicos de los usuarios, por ejemplo).




**Para especificar los equipos en los que se deben instalar uno o más archivos:**

1. Seleccione el o los archivos deseados en la ventana "Lista de archivos instalación"..
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Vaya a la pestaña "Opción" de la ventana "Archivo description".

3. Para instalar los archivos seleccionados:

	- En el servidor, marque "Este archivo está instalado en el servidor".

	- En los ordenadores de los usuarios, marque "Este archivo está instalado en el cliente".

	- En el servidor y en los ordenadores de los usuarios, compruebe "Este archivo está instalado en el servidor" y "Este archivo está instalado en el cliente".

## Copiar y comprimir uno o más archivos
<a name="copiar_comprimir_uno_mas_archivos_ELTTEXTE000271"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000102"></a>

Por Default, cada archivo seleccionado para la instalación será:

- Comprimido cuando se genera el programa instalación.

- Instalado en la computadora de instalación.




WDInst le permite modificar este modo de funcionamiento de Default.

Por ejemplo, su instalación utiliza un archivo "autorun". Esta "[iniciar automáticamente](../Editeurs/2028015.md)" inicia un programa para presentar las diferentes características de su aplicación.

En este ejemplo, WDInst le permite:

- ***Evite comprimir*** los ficheros utilizados por este programa de visión general. Este programa se puede ejecutar directamente desde el CD. Este programa se instalará con la aplicación.

- ***Evite comprimir e instalar*** los ficheros utilizados por el programa de resumen. Este programa se puede ejecutar directamente desde el CD. Este programa **no lo hará** se instalará con la aplicación.



<a name="NOTE4_2"></a>


### Evitar la compresión de uno o más archivos al generar el programa instalación
<a name="evitar_compresion_uno_mas_archivos_generar_programa_instalacion_ELTPARAGRAPHE000124"></a>

Para evitar la compresión de uno o más archivos al generar el programa instalación: 

1. En la ventana "Lista de archivos instalación", seleccione los archivos que no deben ser comprimidos.
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Vaya a la pestaña "Opción" de la ventana "Archivo description".

3. Marque "No comprimir el archivo".



<a name="NOTE4_3"></a>


### Evitar comprimir e instalar uno o más archivos en la computadora de instalación
<a name="evitar_comprimir_instalar_uno_mas_archivos_computadora_instalacion_ELTPARAGRAPHE000137"></a>

Para evitar comprimir e instalar uno o más archivos en el ordenador de instalación: 

1. Seleccione el o los archivos deseados en la ventana "Lista de archivos instalación"..
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Vaya a la pestaña "Opción" de la ventana "Archivo description".

3. Marque "copy en la raíz del CD o del último disquete".




Estos archivos se copiarán:

- En la raíz del CD si el medio instalación corresponde a un CD.

- En la raíz del último disquete si el medio instalación corresponde a los disquetes.

- En el directorio especificado si el medio instalación corresponde a un solo directorio.

- En el último subdirectorio creado si el medio instalación corresponde a un directorio cortado al tamaño de un disquete.

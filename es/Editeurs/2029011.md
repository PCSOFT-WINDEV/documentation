## Atajos de teclado
<a name="atajos_teclado_ELTTEXTE000157"></a>
Los principales atajos que se pueden utilizar en el editor de consultas son los siguientes:


|   |   |
| --- | --- |
| Mostrar la lista de clases en el panel "Project explorer" (y en el panel "Code" si se muestra). | Mayúsculas + F8 |
| Mostrar la lista de procedimientos globales en el panel "Código | Ctrl + F8 |
| Mostrar el código SQL en el editor de código | F2 |
| Ayuda | F1 |
| Deshacer la última acción | Ctrl + Z |
| Detener el modo de prueba (cuando se ejecuta la prueba del proyecto o consulta) | Mayúsculas + F5 |
| Pegar contenido del portapapeles | Ctrl + V o Mayúsculas + Entradas |
| copy el elemento seleccionado | Ctrl + C o Ctrl + Ins |
| Cortar el elemento seleccionado | Ctrl + X o Alt + Del |
| Edite el código associated con la consulta seleccionada en el panel "Vista de código sensible al contexto". | Ctrl + Mayúsculas + F2 |
| Guardar | Ctrl + S |
| Guardar como | Ctrl + Mayúsculas + S |
| Guardar todo | Alt + S |
| Cerrar todos | Ctrl + Mayúsculas + W |
| Imprimir la documentación relativa a la consulta actual | Ctrl + P |
| Ocultar todo (cristales acoplables) | Ctrl + W |
| Nuevo (Crear un elemento) | Ctrl + N |
| Open (Abrir un elemento) | Ctrl + O |
| Abrir un proyecto | Ctrl + Mayúsculas + O |
| Restaurar (Deshacer la última acción) | Ctrl + Y o Ctrl + Mayúsculas + Z |
| Seleccionar todo | Ctrl + A |
| Borrar la selección | Supr |
| Ejecutar la prueba de la consulta actual | F9 |
| Zoom +/- | Ctrl + rueda del ratón |

Atención: + significa que las teclas deben pulsarse simultáneamente.

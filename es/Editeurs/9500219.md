## Presentación
<a name="presentacion_ELTTEXTE000205"></a>
La ventana del proyecto description le permite configurar varios elementos de su proyecto. Esta ventana incluye las siguientes pestañas:

- La pestaña ["Proyecto"](../Editeurs/2030046.md), que permite introducir información general sobre el proyecto

- La pestaña "Análisis", que permite asociar un análisis al proyecto y/o a un modelo UML.

- La pestaña ["Archivos"](../Editeurs/2030052.md), que permite introducir información sobre los archivos de datos utilizados por la aplicación (sólo si el proyecto está vinculado a un análisis).

- La pestaña "Datos en vivo", que permite configurar cómo se muestran los datos de los ficheros de datos en los distintos editores.

- **La pestaña "Idiomas"** , que se utiliza para configurar los diferentes idiomas utilizados por el proyecto, así como las opciones a utilizar para cada idioma.

- La pestaña ["Estilo"](../Editeurs/9500234.md), que permite personalizar la vista previa de impresión y elegir el tema y la hoja de estilo personalizada de plantilla. También puedes configurar el tamaño de la pantalla.

- La pestaña "SCM", que sirve para gestionar el trabajo en equipo (SCM o GIT).

- La pestaña ["Opciones"](../Editeurs/9500227.md), que permite configurar las opciones de actualización, copia de seguridad y sincronización de modelo. Esta pestaña también se utiliza para habilitar el modo estricto y para definir el estilo de código.

- La pestaña ["Avanzado"](../Editeurs/9500236.md), que permite configurar cómo utilizar "Reports & Queries", las AAF y cómo bloquear la aplicación.

- La pestaña ["Recopilación"](../Editeurs/9500218.md), que permite configurar las opciones de compilación.

- La pestaña ["Telemetría"](../Editeurs/9500223.md), que permite configurar las opciones de telemetría del proyecto.

## "Pestaña "Idiomas
<a name="pestana_idiomas_ELTTEXTE000229"></a>


### Idiomas del proyecto
<a name="idiomas_del_proyecto_ELTPARAGRAPHE000057"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Projet_multilangue%20-%20HC%20N%B0001.gif&type=thumb)


Esta pestaña se utiliza para gestionar las características de los idiomas del proyecto. 

Para añadir un nuevo idioma al proyecto: 

1. Haga clic en "Agregar". 

2. En la ventana que se muestra, seleccione los diferentes idiomas soportados por el proyecto.
	Observaciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ProjetMultilangue2%20-%20HC%20N%B0001.gif&type=thumb)


	- Si el idioma solicitado no se muestra en la lista, tiene la posibilidad de seleccionar un idioma adicional (los idiomas adicionales se muestran en la parte inferior de la ventana).. 

- Para nombrar un idioma adicional, haga clic en el idioma deseado y escriba su nombre en el área "Nombre de los idiomas personalizados".. 

3. Validar. Los diferentes idiomas se añaden a la lista de idiomas del proyecto.




Observaciones: 

- Los idiomas seleccionados se propondrán al introducir la información sobre los diferentes elementos que pueden ser traducidos (leyendas de los control, opciones de menú, mensaje de ayuda associated con un control, ...). El orden de visualización de estos idiomas es idéntico al de la lista de idiomas del proyecto.. 
	Este orden puede modificarse mediante los botones de flecha que se encuentran debajo de la lista de idiomas del proyecto.

- Al cambiar el idioma del proyecto, los cambios se tendrán en cuenta automáticamente:

	- cualquier nuevo objeto creado en los diferentes editores (Ventana, Reporte, control, ...).

	- cualquier elemento abierto en los diferentes editores (Window, Reporte, ....).




- El lenguaje Default (identificado con el número "1") corresponde al lenguaje utilizado por Default en tiempo de ejecución. Para modificarlo, utilice el menú contextual del idioma que utilizará Default.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La opción **" Cambiar el idioma inmediatamente después de llamar a la función Nation de WLanguage "** le permite cambiar directamente el idioma utilizado por el ejecutable después de usar [Nación](../WDLang1/3054014.md) 02300 (en este caso, [aprovechamiento](../WDLang1/3038044.md) ya no es necesario para aplicar el nuevo idioma). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La opción **" Seleccionar automáticamente el idioma del sistema al iniciar la aplicación "** permite utilizar el idioma del sistema del usuario por Default.

### Opciones lingüísticas
<a name="opciones_linguisticas_ELTPARAGRAPHE000120"></a>

La visualización de los datos numéricos encontrados en el proyecto (número, moneda, fecha,...) puede configurarse en cada idioma soportado por el proyecto.

Los parámetros utilizados de forma predeterminada son los definidos en las opciones de idioma de Windows (disponibles en el panel de control de Windows).

Para configurar estas opciones:

1. Seleccione el idioma deseado en la parte izquierda de la pantalla. 

2. Seleccione la pestaña correspondiente al elemento a configurar (número, divisa, ...). 

3. Modificar los parámetros deseados y validar. 




**Observación**: Se utiliza un botón para reiniciar los parámetros con los valores de Default.

**Al crear un control Campo de entrada o una columna en un control Tabla** que muestra datos numéricos (número, moneda, fecha, hora, duración, etc.), la máscara de entrada utilizada será automáticamente la máscara definida en las opciones de idioma del proyecto. Esta opción está disponible en ventanas y páginas.

**En tiempo de ejecución** , cuando una columna control Campo de entrada o tabla tiene una máscara "Definida por el proyecto", la máscara de entrada/visualización se adaptará automáticamente según las opciones seleccionadas en el proyecto para el idioma que se muestra en tiempo de ejecución.

**Observación**: La opción "Notas adicionales" permite definir: 

- la dirección de la escritura.

- el personaje que usó set.

- el archivo de traducción de WDM.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Se admiten las siguientes opciones de idioma:

- Separador decimal (número y moneda)

- Separador de miles (número y moneda)

- Símbolo de moneda

- Formato de hora

- Formato de fecha

- Día y mes

- Unidad para el tamaño del archivo

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000116"></a>
Para mostrar una Page de ayuda específica desde su aplicación WINDEV, debe especificarse un identificador para cada Page que se encuentre en su sistema de ayuda. 

Este identificador es único. Se utilizará:

- en la descripción de los controles (pestaña "Ayuda").

- mediante programación con la función [WHelp](../WDLang1/3013053.md).




Se pueden definir los identificadores de las páginas de ayuda:

- en la description del Page. 
	**Observación**: Este método no le permite tener una vista global de todos los identificadores utilizados en el sistema de ayuda.

- en la ventana de resumen de identificadores Page: en la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Identificadores de la página de ayuda".

## Ventana de gestión de los identificadores Page
<a name="ventana_gestion_los_identificadores_page_ELTTEXTE000140"></a>
Esta ventana muestra el identificador associated con cada Page. Es posible:

- modificar el identificador de las páginas de la lista de páginas.

- usar una de las funciones para la numeración de automatic:

	- **"Num Auto"**: se utiliza para numerar las páginas sin identificador. El identificador se calcula automáticamente según el identificador mayor utilizado.

	- **"Borrar todo"**: se utiliza para poner a cero todos los identificadores de las páginas de ayuda.

	- **"Duplicados"**: se utiliza para encontrar los identificadores duplicados.

	- **"Imprimir"**: se utiliza para imprimir la lista de páginas e identificadores de associated.

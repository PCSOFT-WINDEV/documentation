


## Traducir el código
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000128"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten traducir automáticamente su código WLanguage del inglés al francés y viceversa.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000152"></a>


### Traducir automáticamente el código de WLanguage al francés
<a name="traducir_automaticamente_codigo_wlanguage_frances_ELTPARAGRAPHE000018"></a>

Traducir automáticamente el código de WLanguage al francés:

1. Desde una ventana de código: en la pestaña "Código", en el grupo "Idiomas", amplíe "Convertir código" y seleccione "Traducir al francés".

2. Especifique el alcance de la traducción:

	- **Código actual**: sólo se traduce el código que se está editando (donde se encuentra el cursor).

	- **Elemento actual**: se traducen todos los códigos associated con el archivo actual en el editor de código (por ejemplo, todos los códigos de la ventana, controles y menús associated con la ventana).

	- **Todo el código que aparece en la ventana del código Active**: todos los códigos mostrados en la ventana de códigos actual serán traducidos.

	- **Todo el proyecto**: todos los códigos de todos los elementos del proyecto.




3. Validar. La traducción está terminada.




**Observaciones**: 

- Para traducir el código del francés al inglés, en la pestaña "Código", en el grupo "Idiomas", despliegue "Convertir código" y seleccione "Traducir al inglés". Los otros pasos son idénticos.

- Si se seleccionan varias líneas de código, sólo se traducirán las líneas seleccionadas.



<a name="NOTE2_2"></a>


### Traducir el código durante una operación de copy/pegar
<a name="traducir_codigo_durante_una_operacion_copypegar_ELTPARAGRAPHE000063"></a>

Traducir un código de idioma WLanguage durante una operación de copy/pegado (entre dos proyectos WINDEV que utilizan idiomas diferentes, por ejemplo):

1. Abrir las opciones del editor de código: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. En la pestaña "Código", marque "Convertir código a" y seleccione el idioma de traducción.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Coller-Code.gif)

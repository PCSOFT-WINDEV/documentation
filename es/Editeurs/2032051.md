
## Consulta basada en la consulta
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
Una consulta de consulta es una consulta realizada a partir de elementos que provienen de otra consulta.. La consulta inicial debe haberse ejecutado antes de la consulta basada en la consulta.

Este tipo de consulta está disponible:

- mediante programación, con [HExecuteQuery](../WDLang4/3044080.md) y [HExecuteSQLQuery](../WDLang4/3044084.md).

- en el editor de consultas.




**Atención**: No confundir "Consulta basada en la consulta" y "Subconsulta".. Para obtener más información, consulte [Sub-query](../Editeurs/2034009.md).

**Limitación:**Las consultas basadas en consultas no están disponibles para las bases de datos externas (a través de OLE DB, ODBC o Native Access).. De hecho, la base de datos no conoce la consulta de origen a partir de la cual se crea la segunda consulta.

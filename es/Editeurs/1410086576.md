
## Desarrollo de aplicaciones en el modo Universal Windows 10 App
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000139"></a>
WINDEV y WINDEV Mobile permiten desarrollar aplicaciones para tabletas de Windows 10 y para Phone 10 y Phone s de Windows 10 (aplicaciones de aplicación universal de Windows 10). 

Las aplicaciones universales de Windows 10 se desarrollan con WINDEV (o WINDEV Mobile) de la misma manera que las aplicaciones de Windows, salvo algunas diferencias. La prueba de la aplicación también se puede ejecutar en el ordenador de desarrollo en Windows 10.

**Observación**:
 
Para generar y comprobar aplicaciones para las aplicaciones de Windows Store, debe estar instalado el SDK de Windows 10.. Para obtener más información, consulte [Aplicaciones universales de Windows 10 App: Configuración necesaria](../Editeurs/1410086574.md). 
**ATENCIÓN**: El kit de desarrollo de software de Windows 10 sólo se puede instalar en Windows 8 y versiones posteriores.

## Creación de un proyecto para Universal Windows 10 App con WINDEV o WINDEV Mobile
<a name="creacion_proyecto_para_universal_windows_10_app_con_windev_windev_mobile_ELTTEXTE000163"></a>
Para crear un proyecto de Universal Windows 10 App con WINDEV o WINDEV Mobile: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

2. Aparece la ventana de creación: haga clic en "Proyecto". 

3. El asistente de creación de proyectos se abre. Elija el nombre y la ubicación de su proyecto. Tienes la capacidad de entrar en un corto description del proyecto. Pase a la etapa siguiente. 

4. Añadir (si es necesario) documentos de diseño al proyecto. Pase a la etapa siguiente. 

5. En el paso "description - Tipo de proyecto", seleccione "Universal Windows 10 App". Pase a la etapa siguiente. 

6. Especifique la ubicación del SDK de Windows 10 necesario para generar aplicaciones.. Si el SDK fue instalado, la instalación ruta es propuesta automáticamente ("C:\\Archivos de Programa (x86)\\Kits de Windows\\10\\bin" por ejemplo). 
	Para obtener más información, consulte [Instalación del kit de desarrollo de software de Windows 10](../Editeurs/1410086574.md).

7. Pase a la etapa siguiente. 

8. Seleccione el tipo de dispositivo para el que se crea la aplicación (Phone, tableta, PC). 

9. El resto de la asistente se realiza como para cualquier otra plataforma. Para obtener más información, consulte [Crear un proyecto WINDEV Mobile](../Editeurs/2030043.md).

### Análisis de un proyecto de Universal Windows 10 App
<a name="analisis_proyecto_universal_windows_10_app_ELTPARAGRAPHE000059"></a>

Un proyecto de aplicación universal de Windows 10 puede ser associated con un análisis. Los archivos de datos pueden estar en formato HFSQL Classic y HFSQL Client/Server. 







## Probar un proyecto de aplicación universal de Windows 10
<a name="probar_proyecto_aplicacion_universal_windows_10_ELTTEXTE000199"></a>
Existen varios modos de prueba disponibles para las aplicaciones de Universal Windows 10 App:

- El proyecto Go en el simulador de WINDEV o WINDEV Mobile. 

- La ejecución del proyecto en Windows 10. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=W8_test.gif)



### Prueba del proyecto (Go) en el simulador directamente
<a name="prueba_del_proyecto_simulador_directamente_ELTPARAGRAPHE000076"></a>

Esta prueba inicia una ventana de prueba del producto en el que se ejecuta la aplicación Universal Windows 10 App.. Esta prueba le permite usar el depurador. Sin embargo, se pueden notar varias diferencias en comparación con una ejecución real. De hecho, la simulación es utilizando el PC y no el dispositivo Windows 10 en modo Tile.. 




### Ejecute la prueba de la aplicación en tiempo real: ejecutando el proyecto en modo Tile
<a name="ejecute_prueba_aplicacion_tiempo_real_ejecutando_proyecto_modo_tile_ELTPARAGRAPHE000082"></a>

La opción "Ejecutar la prueba de aplicación en real" le permite ejecutar su proyecto en el modo Tile, desde Windows 10 directamente. La aplicación Universal Windows 10 App debe ser generada. 

- Si la aplicación no fue generada, WINDEV o WINDEV Mobile propone generar la aplicación. 

- Si la aplicación ya estaba generada, se genera automáticamente con los parámetros de última generación.. 


Para obtener más información, consulte [Generación de una aplicación de Windows 10 App Universal](../Editeurs/1410086577.md).




## WDInst: configurando la interfaz del programa instalación 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000441"></a>
El editor de instalación WDInst propone un programa estándar de instalación: 

- no hay papel tapiz, 

- no se visualiza ninguna diapositiva,

- plantilla estándar del instalador.  




Para hacer la instalación de su aplicación más fácil de usar, WDInst le permite:

- [Personalizar el estilo del programa instalación](#NOTE2_1).

- [Personalizar la plantilla de la creación del programa instalación asistente](#NOTE3_1). 

- [Mostrar imágenes (llamadas "diapositivas") a intervalos regulares de tiempo al instalar los archivos en el ordenador instalación.](#NOTE4_1).

- [Usar un programa personalizado de instalación (WDSetup personalizado)](../Editeurs/2028106.md).




**Observación**: La personalización del estilo y de la plantilla así como la selección de las diapositivas puede diferir según el lenguaje de ejecución del programa instalación.

La interfaz del programa instalación está configurada en WDInst: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

La creación del programa instalación asistente también se puede utilizar para personalizar la interfaz de usuario y el plantilla del programa instalación.

## Personalizar la interfaz del programa instalación
<a name="personalizar_interfaz_del_programa_instalacion_ELTTEXTE000465"></a>
Para hacer la instalación de su aplicación más fácil de usar, WDInst le permite personalizar la interfaz del programa instalación.

Para personalizar la interfaz del programa instalación: 

1. Seleccione "Configuración ... Estilo".

2. Seleccione el idioma para el que debe personalizarse la interfaz.

3. Configurar la interfaz del programa instalación.
	Observación: Para que se muestre una imagen de fondo, seleccione la opción "Mostrar imagen de fondo de la instalación".

## Personalizar la plantilla de la asistente
<a name="personalizar_plantilla_asistente_ELTTEXTE000489"></a>
El editor WDInst de instalación le permite elegir el estilo de las ventanas de instalación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Installation_WD_defaut%20-%20HC%20N%B0001%201.gif)


Para personalizar la plantilla del asistente: 

1. Seleccione "Configurar ajustes ... Asistente".

2. haga clic en "Asistente plantilla". 

3. Seleccione el fondo Image. Es posible utilizar: 




- ninguna imagen (Default),

- a específico 708 x 460 Image.

## Visualización de diapositivas a intervalos de tiempo regulares
<a name="visualizacion_diapositivas_intervalos_tiempo_regulares_ELTTEXTE000513"></a>
El editor WDInst de instalación le da la posibilidad de mostrar imágenes (llamadas diapositivas) cuando se instalan archivos en los ordenadores de los usuarios.

**Observaciones**:

- Si se muestra un logotipo en el fondo de pantalla, las diapositivas reemplazarán este logotipo.

- Cuanto más pequeños sean los archivos "diapositivas", más rápida será la instalación. Para reducir el tamaño de las diapositivas, le recomendamos que utilice formatos Image comprimidos (".GIF" o ".JPEG", por ejemplo).



<a name="NOTE4_2"></a>


### Diapositivas Scrolling
<a name="diapositivas_scrolling_ELTPARAGRAPHE000299"></a>

Para hacer que las diapositivas se desplacen a intervalos de tiempo regulares: 

1. Seleccione "Parámetros de configuración ... Diapositivas".

2. Seleccionar el idioma para el que se deben configurar las diapositivas.

3. Seleccione las diapositivas (ficheros Image) que desea visualizar (botón "Añadir").

4. Especificar la duración mínima de visualización (en segundos) de cada diapositiva. 
	Para optimizar la velocidad de instalación, las diapositivas se cargan entre dos operaciones de instalación (instalando dos archivos diferentes por ejemplo). La duración de la visualización puede ser superior a la duración especificada.

5. Modifique (si es necesario) el orden en el que se muestran las diapositivas (botones "Mover hacia arriba" y "Mover hacia abajo").

6. Seleccione la opción "Mostrar diapositivas en bucle" para mostrar las diapositivas durante toda la instalación. 
	Si esta opción está desmarcada y la instalación no se completa después de la última diapositiva, esta última permanecerá en pantalla hasta el final de la instalación.


## Importar estilos de un tema de WordPress con WEBDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000152"></a>
WordPress ofrece numerosas plantillas para crear sitios web originales. WEBDEV le permite importar fácilmente estas plantillas. 

Al importar los estilos de una plantilla de Wordpress, WEBDEV analiza el contenido de la modelo, extrae y genera los estilos utilizados. WEBDEV también importa las imágenes de las páginas de ejemplo. La importación crea la hoja de estilo basada en la plantilla y permite la importación de imágenes utilizadas en los ejemplos proporcionados con la plantilla.

Es posible importar varias plantillas en el mismo proyecto: cada hoja de estilo permanece independiente de las demás.

## Condición previa
<a name="condicion_previa_ELTTEXTE000176"></a>
**Antes de importar un WordPress modelo**, compruebe los siguientes puntos: 

- El paquete redistribuible de Visual C++ 2013 debe ser instalado en el equipo. 
	A la fecha de redacción de esta Document, este paquete puede descargarse del sitio de Microsoft ([https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads](https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads)). 

- El contenido del archivo del servidor MySQL debe ser copiado al directorio WEBDEV WordPress: 

	1. Descargar MySQL Server 5.X 32 o 64 bits como archivo ZIP (por ejemplo: [https://dev.mysql.com/downloads/mysql/5.7.html](https://dev.mysql.com/downloads/mysql/5.7.html)). 

	2. Crear una carpeta "mysql" en la carpeta [personal elements](../Editeurs/2020015.md) de WEBDEV. Esta carpeta será: 

		- " si WEBDEV está instalado en un directorio único,

		- "&lt;Usuario&gt;\\AppData\\Roaming\\PC SOFT\\28.0\\Wordpress" si se instala WEBDEV siguiendo el estándar de Windows.




3. Extraer el contenido del archivo en el directorio "\\WordPress\\mysql\\".. Las carpetas "bin", "lib" (...) deben estar en la raíz de la carpeta "\\WordPress\\mysql\\".
			Ejemplos de directorios: 

	- si WEBDEV está instalado en un directorio único:
						"C:\\&lt;carpeta WEBDEV 28&gt;\\Personal\\WordPress\\mysql\\bin\\mysqld.exe"

	- si WEBDEV se instala siguiendo el estándar de Windows:
						"C:\\Usuarios\\&lt;Usuario&gt;\\AppData\\Roaming\\PC SOFT\\28.0\\WordPress\\mysql\\bin\\mysqld.exe"

- Gestión de los puertos: Se utilizan los puertos TCP locales 3399 y 8099.. Los dispositivos de seguridad de los cortafuegos deben permitir a WEBDEV comunicarse localmente a través de estos puertos (localhost).




**Si es necesario, la importación de temas de WordPress asistente le pedirá que realice estas operaciones**.

Atención: Antes de importar un tema de WordPress, por favor lee su licencia para asegurarte de que puedes importarlo.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000200"></a>
**Para importar un tema de WordPress**: 

1. Abrir el proyecto WEBDEV en el que se importará el tema de WordPress. 

2. En la pestaña "Proyecto", en el grupo "Proyecto", expanda "Importar" y seleccione "Estilos de un tema de WordPress". 

3. El asistente de importación se abre automáticamente. 
	Observación: Si no se cumplen los requisitos previos, la asistente no propondrá importar el tema. 

4. Selecciona el archivo '.zip' que contiene el tema de WordPress que quieres importar y haz clic en "Finalizar". 

5. La importación comienza y puede tardar unos minutos. 

6. Al final de la importación, WEBDEV propone abrir una página de ejemplo en la que el estilo importado se aplica en los controles principales de una página de WEBDEV. 




Los estilos del tema de WordPress se integran en el proyecto a través de una hoja de estilo específica. 

**Para elegir un estilo de tema de WordPress para uno de sus controles**: 

1. Seleccione la control en la página. 

2. Abre el menú contextual y selecciona "Elegir un estilo de WEBDEV". 

3. Aparece la ventana de selección de estilo. 

4. Para ver sólo los estilos del tema de WordPress, seleccione su nombre en la lista "Estilos del proyecto. 

5. Seleccione el estilo deseado y valide.

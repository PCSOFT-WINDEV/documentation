### Principio para generar un sitio PHP con WEBDEV
<a name="principio_para_generar_sitio_php_con_webdev_ELTPARAGRAPHE000011"></a>

WEBDEV le permite generar un sitio PHP sin siquiera conocer el lenguaje PHP.. El sitio está desarrollado en WLanguage como cualquier otro sitio WEBDEV.

También tiene la posibilidad de introducir directamente su código PHP.

Un análisis sintáctico y léxico está ahora disponible para el código PHP (y para el código Javascript).

Al generar las páginas, las páginas WEBDEV y las funciones WLanguage se convierten automáticamente en páginas PHP.



Se pueden utilizar las siguientes bases de datos

- HFSQL, 

- MySQL, 

- PostgreSQL,

- Oracle,

- MariaDB.

## Las funciones disponibles en PHP
<a name="las_funciones_disponibles_php_ELTTEXTE000244"></a>
Las siguientes características están actualmente disponibles en PHP:  


|   |   |   |   |
| --- | --- | --- | --- |
| Funciones matemáticas | Gestión de cadenas de caracteres | GalletaXXX | Gestión de fechas y horarios |
| Funciones de dibujo | Funciones de EuroXXX | EmailSendMessage | InXXXMode |
| ExcepciónXXX | Gestión de archivos externos | ListaXXX | PáginaXXX |
| SQLxxx | SysEnvironment | SysDir | SysVersion |
| URLEncode | SubirXXX | Funciones HyperFileSQL | Funciones financieras |
| Funciones FTP | TablaXXX | LooperXXX | HTTPxxx |
| MatXXX | MemXXX | NavegadorXXX | EnchufeXXXX |
| StatXXX | ArrayXXX | XMLxxx |   |


Esta lista evoluciona muy rápidamente. Se recomienda revisar la documentación de cada función. El logotipo de ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=menu_btn_php_1.gif) indica si la función está disponible en PHP, así como sus limitaciones.

### Modo Ajax y PHP
<a name="modo_ajax_php_ELTPARAGRAPHE000120"></a>

El modo Ajax está disponible en PHP. Usted tiene la capacidad de usar las tablas y los loopers de Ajax.

## Presentación
<a name="presentacion_ELTTEXTE000099"></a>
En el panel "Fragmentos de código" se enumeran los fragmentos de código actualmente disponibles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BriqueCode_Gestionnaire16.gif)
 Se utiliza para

- Añade un fragmento de código,

- Modificar un fragmento de código,

- Usa un fragmento de código,

- Borrar un fragmento de código.




Para obtener más información, consulte [Gestión de fragmentos de código](../Editeurs/9000066.md).

<a name="NOTE2_1"></a>


## Visualización del panel "Fragmentos de código"
<a name="visualizacion_del_panel_fragmentos_codigo_ELTTEXTE000123"></a>
**Para mostrar el panel "Fragmentos de código "**, vaya a la pestaña "Inicio", al grupo "Entorno", expanda "Paneles", seleccione "Paneles" y, a continuación, seleccione "Fragmentos de código".

Para más detalles sobre la gestión de los paneles, véase [Manipulación de los cristales acoplables](../Editeurs/2027001.md).

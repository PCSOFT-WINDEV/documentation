## Creación de las diferentes ventanas de patrones
<a name="creacion_las_diferentes_ventanas_patrones_ELTTEXTE000239"></a>


### Los ficheros a crear
<a name="los_ficheros_crear_ELTPARAGRAPHE000042"></a>

Recordatorio: Para construir un patrón RAD utilizado para crear un proyecto completo, se requieren los siguientes archivos:

- Un formulario para RADFileA.

- Un formulario para RADFileB. Un cuadro combinado y un botón Vision Plus (o un cuadro combinado emergente) deben permitirle seleccionar un elemento de RADFileA.

- Un formulario para RADFileC. Un cuadro combinado y un botón Vision Plus (o un cuadro combinado emergente) deben permitirle seleccionar un elemento de RADFileB.

- Una tabla para RADFileA.

- Una tabla para RADFileB.

- Un tabla para RADFileC.

- Una ventana de relación RADFileB/RADFileD.




Estos diferentes elementos deben contener botones de enlace. También tiene la posibilidad de proporcionar el menú principal de la aplicación.

## Menús
<a name="menus_ELTTEXTE000269"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000070"></a>

Cualquier ventana de su patrón puede incluir un menú. Hay varios tipos de menús disponibles:

- un menú desplegable que se encuentra en la primera ventana de la aplicación.

- un menú en un control Carrusel.

- un menú en formato de botón.




Estos menús se usan para abrir:

- las ventanas de formulario y las ventanas de tablas definidas en el patrón

- otras ventanas que se encuentran en el patrón (una ventana para configurar los correos electrónicos, ....).




Los siguientes párrafos explican cómo crear un menú desplegable, un menú de carrusel y un menú de botones.

**Importante: no olvide definir la primera ventana del proyecto en su patrón RAD. En la mayoría de los casos, esta primera ventana corresponde a la ventana del menú.**
<a name="NOTE3_2"></a>


### Menú desplegable
<a name="menu_desplegable_ELTPARAGRAPHE000090"></a>

En la mayoría de los casos, el menú desplegable se incluye en una ventana específica, pero puede incluirse en cualquier ventana de su patrón.

Para crear un menú desplegable: 

1. Cree (si es necesario) la ventana donde se mostrará el menú.

2. Crear el menú: en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Menú principal" y seleccione "Agregar menú principal".

3. Las opciones de menú utilizadas para las opciones específicas del RAD deben tener un nombre específico: "MENUFIC_". El menú con este nombre se duplicará tantas veces como sea necesario para los diferentes archivos del análisis que requieran una entrada de menú.. El código associated con las opciones también se duplicará y se adaptará (si es necesario) al archivo utilizado.




Las otras opciones de menú (utilizadas para iniciar una característica incluida en el patrón, por ejemplo) pueden tener cualquier nombre. Se generarán automáticamente.

**Ejemplo**: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Pattern_Menu_deroulant.gif)


Observaciones:

- Tiene la posibilidad de incluir separadores en el menú.

- Una imagen puede ser associated con las opciones del menú. Este Image no será personalizado por el usuario.



<a name="NOTE3_3"></a>


### Menú del carrusel
<a name="menu_del_carrusel_ELTPARAGRAPHE000125"></a>

Para optimizar el aspecto de sus aplicaciones, tiene la posibilidad de utilizar un menú en formato control Carrusel. Las imágenes se desplazan, permitiendo al usuario elegir una opción específica.

Para crear un menú de carrusel: 

1. Cree (si es necesario) la ventana donde se mostrará el menú.

2. Crear un control Carrusel: en la pestaña "Creación", en el grupo "Otros controles", haga clic en "Carrusel".

3. El control Carrusel utilizado debe llamarse "MENUCAR_".

4. El código para seleccionar una Line en el menú del carrusel puede ser:



```wl
SWITCH Myself
	CASE 1: 
		//RADSTART
		Open(WIN_Table_RADFileA)
		//RADEND
	CASE 2: 
		Open(WIN_Table_RADFileB)
	CASE 3: 
		Open(WIN_Table_RADFileC)
END
```
En este ejemplo, los comentarios "//RADSTART" y "//RADEND" son obligatorios.: se utilizan para encerrar el código que se duplicará para cada archivo procesado por RAD.
<a name="NOTE3_4"></a>


### Creación de un menú de botones
<a name="creacion_menu_botones_ELTPARAGRAPHE000154"></a>

Para optimizar la apariencia de sus aplicaciones, puede utilizar un menú en el formato de una set de controles de botón. Puede definir la forma y la disposición de los botones en su menú.

Para crear un menú de botones: 

1. Cree (si es necesario) la ventana donde se mostrará el menú.

2. Crea los controles de los botones de tu menú.

3. El botón de control del menú debe llamarse "MENUCTRFIL_", con el código associated.




Por ejemplo: MENUCTRFIL_RADFileA, con "RADFileA management" como pie de foto, y como código.: 

```wl
Open(WIN_Table_RADFileA)
```
**4.** Puede crear botones adicionales para set su posición por adelantado (si se utilizan muchos archivos durante la generación, por ejemplo). Estos botones se llaman MENUCTR_X y no tienen código associated (MENUCTR_1 por ejemplo). Si no se crean estos botones adicionales, los botones del menú se apilarán.

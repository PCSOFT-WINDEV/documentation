


## Ventana de selección rápida
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
¿Busca un elemento del proyecto? ¿Sólo conoce parte de su nombre? Pulse Ctrl + E para abrir la ventana de selección rápida.

Lo encontrará extremadamente useful.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000099"></a>
El acceso directo Ctrl + E se puede utilizar en cualquier parte del editor (editor de ventanas, editor de páginas, editor de reportes, ....). Este atajo abre automáticamente la ventana de selección rápida.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ctrl_E.gif)
 Esta ventana enumera todos los elementos del proyecto, y sirve para seleccionar y visualizar el elemento deseado en el editor correspondiente. Hay una vista previa para cada elemento, lo que le permite ver rápidamente si el elemento propuesto se corresponde con lo que está buscando.

**¿Buscas un elemento específico** que contenga la palabra "Archivo"? Simplemente especifique "Archivo" en el área de búsqueda. Se listarán todos los elementos que contengan esta palabra.

**Está buscando sólo los elementos que contienen las palabras "WIN" y "Test" ¿**? Simplemente introduzca "WIN\*Test" o "WIN Test" en el área de búsqueda. El carácter "\*" puede corresponder a cualquier personaje o a un set de personajes.

La opción "Buscar también los subelementos (controles, procedimientos,...)" se utiliza para habilitar o no la búsqueda en los subelementos del proyecto.. Si esta opción está activada, la búsqueda se realizará también en los controles, en los procedimientos, ... Dependiendo del número de elementos encontrados en el proyecto, esta búsqueda puede ser bastante larga.

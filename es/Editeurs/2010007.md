


## Generación automática de índices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000121"></a>
La generación del índice automatic sugiere palabras clave de índice basadas en el contenido de su ayuda Page.

Estas palabras clave aún deben comprobarse. Sin embargo, esta función busca automáticamente las palabras clave existentes en la ayuda Page según criterios específicos

Atención: las palabras clave de índice se ignoran en los archivos de ayuda PDF.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000145"></a>
Para iniciar la generación del índice automatic: 

1. Seleccione la ayuda Page que desea utilizar.

2. Seleccione "Índice automatic" en el menú contextual de la ayuda Page.

3. En la ventana que se muestra, seleccione las opciones necesarias para buscar automáticamente las palabras clave. Consulte el párrafo siguiente para obtener más detalles.

4. Haga clic en "Buscar" para iniciar la búsqueda.

5. Seleccione las palabras clave que se associated con el Page. Aparece una marca de verificación junto a cada palabra clave seleccionada.

6. Añada nuevas palabras clave si es necesario (botón "Añadir").

7. Validar. Las palabras clave seleccionadas se associated automáticamente con la Page actual .


**Observación**: La ventana de generación de palabras clave automatic también puede abrirse desde las propiedades de Page: 

1. En la pestaña "Modificación", en el grupo "Tabla de contenido", haga clic en "Descripción". 

2. En la pestaña "Índice", haga clic en "Índice auto...".

## Opciones para seleccionar palabras clave
<a name="opciones_para_seleccionar_palabras_clave_ELTTEXTE000169"></a>
Las opciones para seleccionar las palabras clave son las siguientes:

- **Mostrar palabras clave en mayúsculas**: Si se marca esta opción, todas las palabras clave sugeridas se escribirán automáticamente en mayúsculas en el índice.

- Filtros seleccionados: 

	- **Palabras en mayúsculas**: Si se incluye este filtro en la búsqueda, todas las palabras en mayúsculas que se encuentren en la ayuda Page se sugerirán automáticamente como palabras clave. Si se excluye este filtro de la búsqueda, no se propondrá ninguna de las palabras en mayúsculas que se encuentran en la Page.

	- **Palabras en minúsculas**: Si se incluye este filtro en la búsqueda, todas las palabras en minúsculas que se encuentren en la ayuda Page se propondrán automáticamente como palabras clave.. Si se excluye este filtro de la búsqueda, no se propondrá ninguna de las palabras en minúsculas que se encuentran en la Page.

	- **Palabras con mayúsculas y minúsculas**: Si se incluye este filtro en la búsqueda, todas las palabras de la ayuda Page con mayúsculas y minúsculas se propondrán automáticamente como palabras clave. Si se excluye este filtro de la búsqueda, no se sugerirá ninguna de las palabras con mayúsculas y minúsculas.

	- **Palabras de menos de tres letras.**: Si se incluye este filtro en la búsqueda, todas las palabras de menos de 3 letras de la ayuda Page se sugerirán automáticamente como palabras clave. Si se excluye este filtro de la búsqueda, no se propondrá ninguna palabra de menos de 3 letras.

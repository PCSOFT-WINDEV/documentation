## Configuración del servidor para los sitios WEBDEV
<a name="configuracion_del_servidor_para_los_sitios_webdev_ELTTEXTE000117"></a>
Con Domino, se debe crear un servidor virtual para cada sitio WEBDEV dinámico alojado.. 

Para configurar el servidor:

1. Iniciar "Administrador de Dominó".

2. Conectarse a un servidor web. Para ello, en la pestaña "Configuración del servidor" seleccione "WEB" y el servidor web que desea configurar.

3. Haga doble clic en el primer servidor (Domino Server).

4. Para crear un servidor virtual:

	- En el menú "Crear WEB (R5)", seleccione la opción "Virtual Server".

	- En el cuadro de diálogo que se muestra, seleccione la opción "Virtual Host".




5. Para configurar este servidor virtual:

	- Haga doble clic en él.

	- Los parámetros a definir se encuentran en la pestaña "Basics" y en la pestaña "Mapping".




6. Los parámetros a configurar son los siguientes:





|   |   |
| --- | --- |
| Nombre de host | Address del sitio web (DNS Address o IP Address) |
| Página principal de Default | Nombre de la página de inicio del sitio (Home.htm por Default) |
| URL de inicio | URL que se ejecutará si se introduce la dirección del sitio (por ejemplo, nombre de la página de inicio) |
| HTML directory | ruta completa del directorio _WEB para el sitio |
| Directorio Icon | ruta completa del directorio _WEB para el sitio |
| Ruta de URL de Icon | Nombre del directorio virtual del sitio (/&lt;Site Name&gt;_WEB) |
| directorio CGI | Ruta del directorio AWP (se encuentra en el subdirectorio Programas del directorio de instalación de WEBDEV). |
| CGI Url ruta | /WD28 0AWP |

## Observación
<a name="observacion_ELTTEXTE000141"></a>


### Gestión de las páginas AWP (Páginas Web de Active)
<a name="gestion_las_paginas_awp_paginas_web_active_ELTPARAGRAPHE000068"></a>

Los sitios WEBDEV que utilizan páginas WEBDEV en formato AWP no pueden ser utilizados en los servidores de Domino.

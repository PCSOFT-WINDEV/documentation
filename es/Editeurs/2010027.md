## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
En cualquier momento, puede modificar las características de un Image insertado en una página de ayuda en las propiedades de Image. La ventana de propiedades de Image te permite:

- ver y seleccionar el archivo Image.

- definir las zonas clicables de la Image.

### Definir las áreas sobre las que se puede hacer clic
<a name="definir_las_areas_sobre_las_que_puede_hacer_clic_ELTPARAGRAPHE000045"></a>

Las áreas clicables son las áreas en las que el usuario podrá hacer clic. Cuando se cierran estas áreas, el cursor del ratón se convierte en una mano, indicando que el usuario puede hacer clic.

Una misma Image puede contener varias zonas clicables.

**Para crear un área en la que se puede hacer clic**:

1. Haz clic en el botón "Nuevo".

2. Definir el enlace associated con la zona clicable. Este enlace puede mostrar una Page o un ancla encontrada en el sistema de ayuda actual, una Page encontrada en un sitio INTERNET o una Page encontrada en otro sistema de ayuda.. Para obtener más información sobre las relaciones, consulte [Propiedades del enlace](../Editeurs/2010022.md). 

3. Defina el área de clic con el ratón o ajuste los valores en "Parámetros del área de clic "..
	Observación: Para que la zona clicable cubra toda la Image, haga clic en "Imagen completa".. 




**Para eliminar un área en la que se puede hacer clic**:

1. Seleccione el área en la que se puede hacer clic para eliminarla.

2. Haz clic en "Eliminar".

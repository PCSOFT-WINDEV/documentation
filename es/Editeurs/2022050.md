


## Enlace de datos: Crear controles desde el panel "Explorador de proyectos"
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
El enlace de datos es un proceso que vincula las variables en la memoria con:

- los controles de una ventana.

- los controles de una página.

- un reporte (origen de datos).




Al crear un control (Tabla, List Box, etc.), el asistente le pide que lo enlace a una variable del proyecto (variable global o variable utilizada en la ventana o página actual).

También puede crear un control y enlazarlo a una variable arrastrando la variable desde el panel "Explorador de proyectos" y soltándola en el editor.

Al crear un reporte, el asistente le pide que lo enlace a una variable. Para obtener más información, consulte [Reportes basados en variables](../WDChamp/1011058.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000141"></a>


### Crear un control a partir de una variable desde el panel "Explorador de proyectos"
<a name="crear_control_partir_una_variable_desde_panel_explorador_proyectos_ELTPARAGRAPHE000035"></a>

Para crear un control a partir de una variable desde el panel "Explorador de proyectos": 

1. Abra el panel "Explorador de proyectos" si es necesario: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Paneles", haga clic en "Paneles" y, a continuación, seleccione "Explorador de proyectos".

2. Haga clic en el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_synchro_explorateur.gif) para seleccionar en el panel "Explorador de proyectos" el elemento que se muestra actualmente en el editor. Los detalles de las variables disponibles en la ventana o página se muestran en el panel "Explorador de proyectos".

3. Arrastre la variable deseada y suéltela en la ventana o página actual:

	- un control Campo de entrada se crea automáticamente para una variable simple, un miembro de clase, un miembro de estructura o un miembro avanzado.

	- un control Tabla se crea automáticamente para una variable de tipo Array.

	- control Campo de entrada se crea para cada miembro simple de variables de clase, variables de estructura o variables avanzadas. Para los miembros de un tipo avanzado, es necesario arrastrar el miembro y soltarlo en el editor por separado.






<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000064"></a>

- Las variables deben arrastrarse desde el panel "Explorador de proyectos".  

- En el caso de las clases, los nuevos controles se enlazan a una instancia de clase (es decir, a una variable de tipo Class). Para crear estos controles, se debe seleccionar una variable de tipo Class en las variables de la ventana o página, y no una clase de la sección "Clases" del panel "Explorador de proyectos".

- La sección "Clases" del panel "Explorador de proyectos" muestra las declaraciones de clase disponibles, y no las instancias de clase.

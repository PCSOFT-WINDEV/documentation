## Presentación
<a name="presentacion_ELTTEXTE000115"></a>
Al crear un enlace, puede especificar simplemente el destino. Mediante Default, el enlace abre el Page especificado en la ayuda actual Page.

Sin embargo, puede definir el tipo de enlace y el destino en las propiedades del enlace.

**Para abrir las propiedades del enlace**:

1. Sitúe el cursor del ratón sobre el enlace deseado.

2. Seleccione "Propiedades del enlace" en el menú contextual.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=proprieteLien.gif)





Observación: Los enlaces están disponibles en todos los modos de generación del sistema de ayuda.

## Propiedades del enlace
<a name="propiedades_del_enlace_ELTTEXTE000139"></a>
En esta ventana se muestra la siguiente información:

- **Texto del enlace**: Enlace al pie de foto. La leyenda aparecerá subrayada en la ayuda Page. El subrayado informa al usuario de que hay un enlace disponible.

- **Hipervínculo a**: Tipo de enlace. Se utiliza un enlace para abrir:

	- otra Page del sistema de ayuda.

	- a INTERNET Page.

	- un ancla de una Page.

	- un enlace hacia una Page encontrada en otro archivo de ayuda. Atención: el archivo WDH correspondiente al segundo sistema de ayuda debe encontrarse en el directorio del proyecto.




- **Destino Page**: Page que debe abrirse mediante el enlace. Esta opción corresponde a:

	- una de las páginas que se encuentran en el sistema de ayuda actual.

	- una INTERNET Address (www.windev.com por ejemplo).
			

	- un ancla en una de las páginas que se encuentran en el sistema de ayuda actual.

	- a Page encontrada en otro sistema de ayuda que se encuentra en el directorio del proyecto.




- **Modo de apertura**: Modo de apertura de la Page abierta por el enlace. Las opciones disponibles son:

	- *Ventana actual*: La página usa la misma description que la ventana de ayuda actual.

	- *Ventana emergente* (abrir una ayuda Page solamente): La ayuda Page se muestra como una ventana emergente (tooltip). 
			**Atención**: En este caso, la ayuda Page no debe tener superficie fija.

	- *Ventana específica*: La ayuda Page se abrirá en la ventana de ayuda seleccionada.

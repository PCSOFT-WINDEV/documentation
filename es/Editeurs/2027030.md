## Presentación
<a name="presentacion_ELTTEXTE000136"></a>
El panel "Depurador" muestra las opciones del depurador cuando se depura el proyecto o un elemento del proyecto (opción "Depurador").
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels.gif&type=thumb)


Las diferentes opciones del panel son las siguientes: 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Debug_IconeContinuerTest.gif): Continuar (F5).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%202.gif): Finalizar prueba (Mayús + F5).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%205.gif): Ejecutar paso a paso entrando en las funciones (F7).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%203.gif): Ejecutar paso a paso saltando las funciones (F8).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%206.gif): Ejecutar hasta el final de la función (Alt + Fin).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%208.gif): Ejecutar hasta el cursor. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%204.gif): Punto de interrupción (Ctrl + B).

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%207.gif): Eliminar todos los puntos de interrupción.

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%209.gif): Lista de puntos de interrupción. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2010.gif): Mostrar estado de las fuentes de datos. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2013.gif): Mostrar elementos del sistema. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2012.gif): Mostrar información sobre el archivo de volcado. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2011.gif): Mostrar expresiones automáticas. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Debug_IconeVariableLocale.gif): Mostrar variables locales. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Debug_IconeVariableGlobale.gif): Mostrar variables globales. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2015.gif): Mostrar miembros. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Debug_IconeCpteRenduHF.gif): Mostrar informe de HFSQL. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Debug_IconeCpteRenduWL.gif): Mostrar informe de WLanguage.

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%201.gif): Activar Analizador de Rendimiento en Tiempo Real. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2014.gif): Mostrar ventana de salida del depurador.

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Partie2_PileDesAppels%2016.gif): Mostrar en formato hexadecimal. 




Para obtener más información, consulte [Depurador](../Editeurs/2019001.md).

## Abrir el panel "Depurador"
<a name="abrir_panel_depurador_ELTTEXTE000160"></a>
**Para abrir el panel "Depurador"** , vaya a la pestaña "Inicio", grupo "Entorno", despliegue "Paneles", seleccione "Paneles", y a continuación, haga clic en "Depurador".

Observación: Este panel se abre automáticamente al depurar una ventana, una página o el proyecto.

Para obtener más información sobre los paneles, consulte [Manipular paneles acoplables](../Editeurs/2027001.md).

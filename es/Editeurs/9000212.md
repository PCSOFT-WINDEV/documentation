


## Búsqueda rápida en el código actual
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000114"></a>
Al escribir código, los desarrolladores a menudo necesitan hacer búsquedas rápidas en el código actual.
El entorno ya incluye una opción de búsqueda (Ctrl + F).
Sin embargo, hasta ahora era necesario utilizar el ratón y configurar las opciones de búsqueda. 
En la versión 28, se introdujo un nuevo modo de búsqueda simple para el código actual.
Las teclas Alt+ F3 abren el cuadro de búsqueda.
Las instancias se resaltan en el código a medida que se escribe. El número de instancias encontradas se muestra en el editor.

<a name="NOTE2_1"></a>


## ¿Cómo hacer una búsqueda rápida?
<a name="¿como_hacer_una_busqueda_rapida_ELTTEXTE000138"></a>
Para realizar una búsqueda rápida en el código: 

1. Abra un proyecto en el editor y vaya al código de un elemento. 

2. Presione Alt + F3 para abrir el cuadro de búsqueda rápida: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0001.gif)


3. Para pasar al modo "Buscar y Reemplazar", presione Ctrl + R cuando el cuadro de búsqueda tenga el foco. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002.gif)

	Observación: Para salir del modo de reemplazo, haga clic en el ícono de cierre o presione Esc. 




**Funcionamiento del cuadro de búsqueda**

- No se pueden abrir varios cuadros de búsqueda rápida. 

- El cuadro se cierra con el ícono de cierre o con la tecla Esc. 

- Puede navegar entre los resultados haciendo clic en las flechas ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%201.gif) o presionando F3 o Mayús + F3.  

- Si hay una sola línea seleccionada al abrir el cuadro de búsqueda, la selección se pega automáticamente en el cuadro de búsqueda y se inicia la búsqueda. 

- La búsqueda inicia automáticamente cuando el usuario empieza a escribir en el cuadro de búsqueda. 

- Las opciones de filtro disponibles son las siguientes: 

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%205.gif) Distinguir mayúsculas y acentos. 

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%206.gif) Buscar palabras completas. 

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%207.gif) Buscar usando expresiones regulares. 




- Para reemplazar un elemento: 

	- Presione Ctrl + R para pasar al modo de reemplazo. El cursor aparece automáticamente en el campo de entrada del texto de reemplazo. 

	- Presione Alt + R (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%203.gif)) para reemplazar el elemento actual y seleccionar el siguiente. 

	- Presione Entrar (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%202.gif)) para reemplazar todos los elementos encontrados.




- Presione Ctrl + F (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recherche_rapide_code%20-%20HC%20N%B0002%204.gif)) para abrir el panel de búsqueda.


## Generar un diagrama de clases 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
La generación crea las clases WINDEV (archivos ".WDC") que corresponden a las clases creadas en el editor de UML.

Es posible:

- generar una sola clase.

- generar todas las clases del diagrama.




Un diagrama de clases también se puede generar mediante ingeniería inversa. Para obtener más información, consulte [Generar un diagrama de clases mediante ingeniería inversa](../Editeurs/2035009.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000128"></a>


### Generar una sola clase
<a name="generar_una_sola_clase_ELTPARAGRAPHE000028"></a>

Para generar una sola clase:

1. Seleccione la clase que desea generar.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeGenerationclasse.gif). La ventana de generación de UML se abre.

3. La clase actual está:

	- en la tabla de arriba si es la primera vez que se genera la clase. La clase se creará automáticamente al realizar la generación UML.

	- en la tabla de abajo si ya se creó la clase. La clase se volverá a crear automáticamente al realizar la generación UML.




4. Valide la generación de la clase UML.



<a name="NOTE2_2"></a>


### Generar todas las clases del diagrama
<a name="generar_todas_las_clases_del_diagrama_ELTPARAGRAPHE000045"></a>

Para generar todas las clases del diagrama:

1. Seleccione la opción "Generar modelo UML" en el menú contextual del editor (o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=UML_Ico_generer.gif)). La ventana de generación de UML se abre. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GenerationDiagrammeclasse.gif)


2. Esta ventana muestra:

	- en la tabla de arriba, las clases que nunca se han generado. Estas clases se crearán automáticamente al realizar la generación UML.

	- en tabla de abajo, las clases ya se han creado. Estas clases se volverán a crear automáticamente al realizar la generación UML.




3. Valide la generación del diagrama de clases UML.

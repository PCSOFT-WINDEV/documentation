


## Instalación de WINDEV, WEBDEV o WINDEV Mobile de 64 bits

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
Esta ayuda Page presenta la configuración necesaria para instalar WINDEV, WEBDEV o WINDEV Mobile en modo 64 bits.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Por qué instalar una versión de 64 bits?
<a name="¿por_que_instalar_una_version_64_bits_ELTTEXTE000126"></a>
Se recomienda instalar la versión de 64 bits de WINDEV, WEBDEV o WINDEV Mobile (excepto si sus proyectos utilizan una función no disponible, véase la lista siguiente). Se recomienda esta instalación si se trata de proyectos que incluyen un gran número de elementos (más de 1500). 

Las limitaciones para instalar una versión de 64 bits son las siguientes: 

- Los conectores nativos de Progress y Sybase no están disponibles en el modo de 64 bits. 

- Las conexiones desde el editor de análisis se establecen en modo de 64 bits. Si se realiza una conexión hacia una base de datos externa, el cliente o el controlador ODBC o el proveedor OLE DB de esta base de datos también debe estar instalado en modo de 64 bits.

- Los análisis anteriores de Hyper File 5.5 no se pueden importar. 

- El antiguo RAD compatible con la versión 11 no está disponible.

<a name="NOTE3_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000150"></a>
La siguiente configuración es necesaria para el ordenador de desarrollo: 

- 4 GB o RAM (se recomiendan 8 GB o más).

- 2 GB de espacio en el disco (4 GB para una instalación completa : el software incluye cientos de ejemplos).

- ejecutando Windows de 64 bits.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Notas
<a name="notas_ELTTEXTE000174"></a>
El formato de los proyectos y de los elementos del proyecto es independiente del modo de compilación de los editores.. Todos los elementos fuente (proyectos, ventanas, informes, clases...) pueden ser abiertos por un editor de 32 bits y luego por un editor de 64 bits (y viceversa).

**No se debe instalar una versión de 32 bits y una versión de 64 bits del mismo software en la misma computadora de desarrollo** .

Sin embargo, si la misma configuración de desarrollo alberga una versión de 32 bits y una versión de 64 bits de WINDEV, WEBDEV o WINDEV Mobile:

- las dos versiones deben ser instaladas en carpetas diferentes como propone Default. 

- las dos versiones no deben ejecutarse simultáneamente.

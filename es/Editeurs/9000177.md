


## Primera ventana del proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000177"></a>
La primera ventana del proyecto corresponde a la primera ventana que se abrirá al iniciar la aplicación WINDEV o WINDEV Mobile.. 

Esta ventana es también la primera que se abrirá durante la prueba del proyecto en el ordenador de desarrollo.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo definir la primera ventana del proyecto?
<a name="¿como_definir_primera_ventana_del_proyecto_ELTTEXTE000201"></a>
WINDEV propone varios métodos para definir y elegir la primera ventana de proyecto:

- [A través del panel "Explorador de proyectos](#NOTE2_2).

- [En la configuración description](#NOTE2_3).

- [Durante la primera prueba del proyecto](#NOTE2_4). 




**Observación**: La primera ventana de proyecto también se solicita durante la prueba del proyecto, si no se definió ninguna ventana. Para obtener más información, consulte [Probar un proyecto](../Editeurs/2019003.md).
<a name="NOTE2_2"></a>


### Definición de la primera ventana de proyecto en el panel "Explorador de proyectos
<a name="definicion_primera_ventana_proyecto_panel_explorador_proyectos_ELTPARAGRAPHE000040"></a>

Para definir la primera ventana del proyecto:

1. En el panel "Explorador de proyectos", seleccione la ventana que se definirá como primera ventana. 

2. Abra el menú contextual de la ventana (haga clic con el botón derecho del ratón) y seleccione "Primera ventana de proyecto".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Premi%E8re_fen_prjWD%20-%20HC%20N%B0002.gif)

	La ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Premi%E8re_fen_prjWD%20-%20HC%20N%B0001.gif) Icon se utiliza para identificar la primera ventana de proyecto en el panel "Explorador de proyectos".






**Observación**: Si el proyecto incluye varias configuraciones, se puede definir una primera ventana para cada una de ellas. Para ello: 

1. Seleccione la configuración de proyecto deseada en el explorador de proyectos. 

2. Definir la primera ventana de proyecto en el menú contextual del explorador del proyecto.



<a name="NOTE2_3"></a>


### Definiendo la primera ventana en la configuración description
<a name="definiendo_primera_ventana_configuracion_description_ELTPARAGRAPHE000071"></a>

Para definir la primera ventana del proyecto description: 

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. En la pestaña "General", haga clic en "Descripción de la configuración". 

3. La ventana de description de la configuración actual se muestra. 

4. En la pestaña "General", la lista "Primera ventana de proyecto" permite seleccionar la primera ventana de proyecto. 





<a name="NOTE2_4"></a>


### Definición de la primera ventana durante la prueba del proyecto
<a name="definicion_primera_ventana_durante_prueba_del_proyecto_ELTPARAGRAPHE000103"></a>

Para probar un proyecto desde el editor: 

1. En la pestaña "Proyecto", en el grupo "Modo de prueba", despliegue "Modo de prueba" y seleccione "Depurar proyecto" (Ctrl + F9). También puede hacer clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_GO_Projet_WD.gif) en los botones de acceso rápido.

2. Si no se ha definido la primera ventana de proyecto, se muestra una ventana que le permite seleccionar la primera ventana de proyecto.. 
	

3. El editor se minimiza automáticamente y el proyecto se ejecuta.

## Resumen
<a name="resumen_ELTTEXTE000201"></a>
En una aplicación WINDEV, los usuarios tienen a su disposición diferentes tipos de ayuda: 

- Tooltip.

- Barra de estado.

- Ventana de ayuda.

- Tooltip avanzado.

- Página de ayuda. 

- Archivo CHM.





### Tooltip
<a name="tooltip_ELTPARAGRAPHE000021"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BULLEAIDE.GIF)


- **Nivel de ayuda**: Ayuda breve (pocas palabras)

- **Cómo crearlo**:

	- Ventana de descripción del control, pestaña "Ayuda"

	- Programación: Propiedad [ToolTip](../Proprietes/2510005.md)




- **Cómo abrir la ayuda**: Se inicia automáticamente cuando el cursor del ratón pasa por encima de la control.

- **Para más detalles**: [Ayuda de usuario](../Editeurs/2010012.md)





### Tooltip personalizado
<a name="tooltip_personalizado_ELTPARAGRAPHE000038"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0005.gif)


- **Nivel de ayuda**: Ayuda contextual.

- **como crearlo**: 

	- Editor de proyectos, para definir la apariencia.  

	- Ventana Control description, pestaña "Ayuda" para definir el contenido




- **Cómo abrir la ayuda**: Se inicia automáticamente cuando el cursor del ratón pasa por encima de la control.

- **Para más detalles**: [Tooltips personalizados](../Editeurs/9500233.md)





### Barra de estado
<a name="barra_estado_ELTPARAGRAPHE000053"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BARREMSG.GIF)


- **Nivel de ayuda**: Ayuda simple (frase, hora, barra de progreso, etc.)

- **Cómo crearlo**:

	- Ventana de descripción del control, pestaña "Ayuda"

	- Programación: Función [Mensaje](../WDLang1/3021006.md) o propiedad [Mensaje](../WDLang1/3021006.md)




- **Cómo abrir la ayuda**: Iniciado automáticamente (la ventana debe tener una barra de estado). 
	La información sobre un control determinado se muestra cuando el control recibe el foco.

- **Para más detalles**: [Ayuda de usuario](../Editeurs/2010012.md)





### Ventana de ayuda
<a name="ventana_ayuda_ELTPARAGRAPHE000075"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fenetre_Aide.gif)


- **Nivel de ayuda**: Mensaje de ayuda que puede contener varias frases.

- **como crearlo**: Ventana de descripción del control, pestaña "Ayuda"

- **Cómo abrir la ayuda**: Inicio automático (tecla F1).

- **Para más detalles**: [Ayuda de usuario](../Editeurs/2010012.md)





### Página de ayuda
<a name="pagina_ayuda_ELTPARAGRAPHE000087"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P0_Aide_Browser.jpg&type=thumb)


- **Nivel de ayuda**: Ayuda avanzada (páginas de ayuda, enlaces, resumen, imágenes, etc.).

- **como crearlo**: Editor de ayuda: Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. La ventana de creación de nuevos elementos se abre: haga clic en "Arquitectura" y luego en "Ayuda".

- **Cómo abrir la ayuda**: Puesta en marcha mediante programación ([WHelp](../WDLang1/3013053.md))

- **Como abrir la ayuda contextual de una ventana o control**: 

	1. Página de ayuda contextual de un control:

		- Especifique el número de la página que se mostrará en la ventana de descripción del control.

		- Utilice las funciones [WHelp](../WDLang1/3013053.md) y [NumHelp](../WDLang1/3013027.md).




2. Página de ayuda contextual de una ventana:

	- Utilice la función [WHelp](../WDLang1/3013053.md) y especifique el número de la página a mostrar.

- **Para obtener más información, consulte:** [Creación de la ayuda (CHM)](../Editeurs/2010002.md)

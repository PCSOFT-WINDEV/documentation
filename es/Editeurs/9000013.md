


## Ejecutar la prueba de un proyecto Java y compilarlo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
Cuando su aplicación Java esté desarrollada, puede ejecutar la prueba del proyecto Java y compilarlo antes de crear el archivo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Prueba de proyecto Java en el editor
<a name="prueba_proyecto_java_editor_ELTTEXTE000155"></a>
Durante la prueba de un proyecto desde el editor, el proyecto se compila e inicia en Windows. El proyecto no está compilado en Java.

Recordatorio: Para iniciar la prueba de un proyecto, en la pestaña "", en el grupo "Modo de prueba" , despliegue "Modo de prueba" y seleccione "Depurar proyecto".

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Compilación de un proyecto Java en WINDEV
<a name="compilacion_proyecto_java_windev_ELTTEXTE000179"></a>


### Compilación de un proyecto Java
<a name="compilacion_proyecto_java_ELTPARAGRAPHE000031"></a>

La compilación de un proyecto Java está acostumbrada a:

- identificar los errores de programación encontrados en el código de WLanguage.

- controles de pantalla, procesos, funciones WLanguage no disponibles en Java.




Estos errores aparecen en color púrpura en el panel "Errores de compilación".

Para obtener más información, consulte [Limitaciones de esta versión](../Editeurs/9000056.md). 

**Observación**: El número de funciones WLanguage que se pueden utilizar en un proyecto Java es limitado en esta versión.. Este número de funciones aumentará en las próximas versiones.

Para compilar un proyecto Java, en la pestaña "", en el grupo "", despliegue "" y seleccione "Recompilar proyecto".
<a name="NOTE3_2"></a>


### Compilación de un ejecutable WINDEV en Java
<a name="compilacion_ejecutable_windev_java_ELTPARAGRAPHE000064"></a>

Se puede crear una aplicación Java a partir de un proyecto WINDEV existente.

Para ello, se debe añadir una nueva configuración al proyecto: 

1. En la pestaña "", en el grupo "", haga clic en "".

2. Crear una nueva configuración (botón"Nueva configuración"). Elija la generación "Java".

3. Seleccione los elementos de proyecto existentes que deben incluirse en esta configuración.. Estos elementos también se incluirán en el archivo o en el applet que se generará.

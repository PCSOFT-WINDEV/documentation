### Principio para generar una aplicación Linux con WINDEV
<a name="principio_para_generar_una_aplicacion_linux_con_windev_ELTPARAGRAPHE000011"></a>

WINDEV le permite generar fácilmente una aplicación Linux. El método para desarrollar la aplicación es idéntico al método para desarrollar cualquier otra aplicación WINDEV.

Cuando se compila el proyecto WINDEV, las ventanas y las funciones de WLanguage se convierten automáticamente a Linux.

**Observaciones**: 

- A partir de la versión 27, las siguientes distribuciones de Linux son compatibles:

	- Debian 9 o posterior

	- Fedora 25 or later

	- Ubuntu 18.04 o superior (versión mínima de GCC: 5.1)




A partir de la versión 27, las siguientes distribuciones de Linux dejan de ser compatibles: 

- OpenSUSE:Leap:15.0. (La versión oficial es openSUSE Leap 15.2, que es compatible).

- Las aplicaciones Linux desarrolladas con WINDEV funcionan en los dispositivos equipados con un procesador Intel o con un procesador compatible con Intel (los procesadores ARM no son compatibles).

- A partir de la versión 28, sólo puede crear proyectos y ejecutables **64-bit** Linux.

## Formatos Image
<a name="formatos_image_ELTTEXTE000259"></a>
Los siguientes formatos de Image están soportados por las aplicaciones WINDEV de Linux: 

- Mapa de bits (\*.BMP)

- Formato de intercambio de gráficos (\*.gif)

- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

- Gráficos de red portátiles (\*.png)

- Mapa de bits portátil (\*.pbm)

- Mapa de grises portátil (\*.pgm)

- Portátil Pixmap (\*.ppm)

- Etiquetado en el formato de archivo Image (\*.Tiff). 
	Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

- X11 Mapa de bits (\*.xbm)

- X11 Pixmap (\*.xpm)

- Gráficos vectoriales escalables (\*.svg)
	Observación: Para administrar este formato, debe instalar una biblioteca QT que admita el formato SVG.

## Notas y limitaciones
<a name="notas_limitaciones_ELTTEXTE000289"></a>


### Usando un ejecutable de Linux
<a name="usando_ejecutable_linux_ELTPARAGRAPHE000058"></a>

Para utilizar un ejecutable de Linux, la biblioteca QT (Core y GUI) versión 5 (o posterior) debe estar instalada. 

**En Ubuntu y Debian**esta librería puede ser instalada con los siguientes comandos: 


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```
**En Ubuntu 20**, esta biblioteca se puede instalar con los siguientes comandos: 


```txt
sudo apt-get install qt5-default
```
**En mandriva 2010 Spring**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
urpmi libqtgui5
```


Observación: Para beneficiarse de la gestión de imágenes SVG, la biblioteca QT instalada debe ser compatible con el formato SVG.


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000087"></a>

Actualmente se soportan los siguientes controles/ventanas: 

- cristalera

- Control Estático

- Control Botón

- control Imagen

- control Gráfico

- Control List Box

- control Tabla (incluidas las tablas basadas en un archivo de datos)

- Editar control

- Control Barra de desplazamiento




Las siguientes características están disponibles: 

- Anclaje y redimensionamiento

- Funciones de dibujo

- Propiedades de la base: X, Y, Anchura, Altura, etc.

- Clasificar en columnas tabla

- Planos

- Menús (menús contextuales y menús AAF en las mesas).
	**Atención**: Los archivos creados por las opciones "Exportar a Excel" están siempre en formato XLSX (incluso si la extensión es XLS).. 




Limitaciones: 

- Barra de estado de las ventanas

- Gestión de cadenas gráficas (gImage, ....)

- Animaciones

- HTML y RTF control Campo de entrada

- Control de edición con el botón Calendar

- Ícono del ejecutable

- Columnas de tabla de los siguientes tipos: contenedor y cuadro combinado

- Acciones predefinidas




Atención: En los ejecutables de Linux, el método de autenticación para los servicios web "auNegotiate" no es compatible.


## Objetivo C: Llamando a un WLanguage Procedure
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
Una Procedure global en WLanguage puede ser llamada directamente desde el código de una Procedure global en el Objetivo C. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000116"></a>
Llamar a un WLanguage Procedure directamente desde el código de un Procedure global en el Objetivo C: 

1. Escriba el código de WLanguage Procedure. El WLanguage Procedure debe cumplir con algunas reglas: 

	- Tener un nombre aceptable en el Objetivo C. 

	- El tipo de parámetros de Procedure y el tipo de valor de retorno deben ser tipos soportados: booleano, entero, real, cadena, buffer. 

	- El atributo "ObjC" debe añadirse a la lengua WLanguage Procedure. 




2. A continuación, basta con llamar a Procedure desde el código Objective C. 




**Ejemplo**: Procedure para sumar 1 al número especificado:  

- Código WLanguage: 
	
	```wl
	PROCEDURE nWL_AddOne(n is int), ObjC: int
	RESULT n+1
	```


- Objetivo Código C: 
	
	```txt
	int nObjC_ResultInt(int i)
	{
	return nWL_AddOne(i);
	}
	```





**Ejemplo**: Procedure para visualizar texto: 

- Código WLanguage: 
	
	```wl
	PROCEDURE MyProcedure(psMessage is string), ObjC: string
	RETURN psMessage
	```


- Objetivo Código C: 
	
	```txt
	void ios_Call_Proc()
	{
	MyProcedure(@"My text");
	}
	```

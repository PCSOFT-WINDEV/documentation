
## Propiedades del modelado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000116"></a>
Las propiedades del modelado agrupan todas las opciones del modelado actualmente editado.

Estas propiedades se agrupan en tres pestañas:

- General

- Actores

- Conceptos




Para mostrar las propiedades del modelado, seleccione "Modelado description" de el menú contextual del modelado.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Detalles de las propiedades
<a name="detalles_las_propiedades_ELTTEXTE000140"></a>


### Propiedades generales
<a name="propiedades_generales_ELTPARAGRAPHE000026"></a>

La pestaña "General" contiene:

- **El título del modelado**. Si se selecciona "Insertar automáticamente un bloque de título", el título del modelo se muestra en el bloque de título y se actualiza automáticamente.

- **Tel nivel de automatización Default**. Este es el nivel de automatización usado por Default al crear un nuevo elemento en el editor de modelado.
	Usted tiene la capacidad de ocultar el nivel de automatización de los elementos en el modelado.

- **TLa inserción de un bloque de título en los submodelos**. Si se selecciona esta opción, se insertará automáticamente un bloque de título en los nuevos submodelos.

- **La distribución de los enlaces**. Esta opción permite especificar si los enlaces deben ser mostrados (o no) en la Border de los elementos.

- **La huella**. Esta opción permite especificar las características utilizadas para imprimir la documentación (cabecera y pie de página).

- **El tema gráfico**. Esta opción permite seleccionar la el tema gráfico que se utilizará para el modelado. Este libro de estilo corresponde a la hoja de estilo associated con el modelado. Para obtener más información, consulte [Estilos de un modelaje](../Editeurs/2010105.md).



<a name="NOTE2_2"></a>


### Actores
<a name="actores_ELTPARAGRAPHE000045"></a>

La lista de actores corresponde a la lista de personas que pueden participar en el modelado actualmente creado (como gestor, desarrollador o usuario).

Esta lista de actores puede corresponder a:

- individuos introducidos en el editor de modelos (haga clic en el botón "+" para añadir un actor).

- personas que participan en el groupware para desarrolladores. Estos son los contribuyentes descritos en la Centro de Seguimiento de Proyectos. Es posible:

	- Importe los contribuyentes (uno o todos) definidos en el sistema Centro de Seguimiento de Proyectos.

	- Actualizar la lista de actores con los contribuyentes que se encuentran en la Centro de Seguimiento de Proyectos.

	- Actualizar todo.






<a name="NOTE2_3"></a>


### Conceptos
<a name="conceptos_ELTPARAGRAPHE000061"></a>

Los conceptos se utilizan para resaltar los elementos de modelado importantes que se reutilizan en varias ubicaciones. Cuando se destaca un concepto, todos los elementos associated con este concepto están bordeados por el Color del concepto.

Esta pestaña se utiliza para ver y crear los diferentes conceptos de modelado.

**To crea un nuevo concepto,** hace clic en el botón "+" e introduce las características del concepto en el tabla.

El botón "Más conceptos" permite analizar el modelado actual y proponer nuevos conceptos.

## Creación del ejecutable
<a name="creacion_del_ejecutable_ELTTEXTE000125"></a>
El ejecutable de una aplicación que soporta el User Groupware es un ejecutable multi-instancia.

Para crear un ejecutable que soporte el groupware usuario: 

1. Inicie el asistente de generación de ejecutables mediante uno de los siguientes métodos: 

	- En la pestaña "Proyecto", en el grupo "Generación", haga clic en "Generar". 

	- Despliegue ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_generation_exe.gif) en los botones de acceso rápido y seleccione "Generar ejecutable Windows de 32 bits (.exe)". 




2. En la creación del ejecutable asistente (opción "Datos y groupware" en la parte izquierda de la pantalla) aparece un paso específico para el Groupware de usuario. Este paso establece el Default ruta para los archivos de User Groupware. Este directorio corresponde por Default a un subdirectorio del directorio ejecutable (directorio EXE\\GPW_ ).
	Observación: La ruta de los archivos de datos que se encuentran en el groupware también se puede especificar al instalar la aplicación.

3. Finalice el asistente. 




Observaciones:

- Si se han configurado archivos de datos del Groupware Usuario para un cliente, deben seleccionarse al preparar la instalación. Por lo tanto, es necesario personalizar la lista de archivos de la instalación.

- Si no se han configurado los archivos de datos del groupware, el Supervisor será el único usuario en la aplicación la primera vez que se inicie.

## Instalar una aplicación
<a name="instalar_una_aplicacion_ELTTEXTE000149"></a>
Los programas de instalación de las aplicaciones que utilizan el User Groupware se crean como cualquier otra aplicación.: en la pestaña "Proyecto", en el grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación".

Puede establecer la ruta de los archivos del groupware en el asistente de creación de la instalación (en la etapa "Opciones de la base de datos", seleccione "Establecer la ubicación de los archivos del groupware durante la instalación").

En este caso, cuando se instala la aplicación, aparece una pantalla específica para el Grupo de Usuarios. Esta pantalla permite a la persona que instala la aplicación:

- configurar la ubicación de los archivos de datos específicos del User Groupware. 

- activar historial de conexiones.

- habilitar la autenticación de usuarios a través de un directorio LDAP. En este caso, tendrás que introducir la Address del servidor, el número de puerto y la versión LDAP utilizada.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GPU_Exe_install%20-%20HC%20N%B0001.gif)
Observaciones:

- Si se han configurado archivos de datos del Groupware Usuario para un cliente, deben seleccionarse al preparar la instalación. Por lo tanto, es necesario personalizar la lista de archivos de la instalación.

- Si no se han configurado los archivos de groupware, sólo existirá el usuario "Supervisor" durante el primer inicio de la aplicación.




**Consejo**: Si no configura los diferentes niveles de uso de groupware usuario para sus clientes, le recomendamos que les proporcione un Document con la lista de todos los controles que se encuentran en sus ventanas o páginas con el fin de get una configuración optimizada.

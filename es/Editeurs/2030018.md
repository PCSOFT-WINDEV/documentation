


## Duplicar un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
Duplicar un proyecto le permite tener dos proyectos idénticos con el mismo nombre pero que se encuentran en directorios diferentes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000120"></a>
Para duplicar el proyecto actual:

1. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Exportar .. A un directorio".

2. Seleccione los archivos que desea exportar. Para duplicar el proyecto, seleccione todos los archivos. 
	Atención: Esta operación no le permite duplicar los archivos de análisis o los archivos de copia de seguridad.

3. Especifique el directorio al que deben exportarse los archivos.

4. Validar la duplicación del proyecto.

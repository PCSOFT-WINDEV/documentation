## Crear una consulta con parámetros
<a name="crear_una_consulta_con_parametros_ELTTEXTE000100"></a>
Independientemente del tipo de consulta (Seleccionar, Insertar, Borrar, Actualizar) e independientemente de su modo de creación (con la asistente o por SQL directamente), tiene la posibilidad de utilizar parámetros. El nombre del parámetro se especifica al crear la consulta.

Para crear una consulta con parámetros:

1. Crear una consulta. Para más detalles sobre cómo crear una consulta, véase [¿Cómo crear una consulta?](../Editeurs/2032059.md).

2. Cuando [definir las condiciones de selección](../Editeurs/2032019.md):

	- especifique que la condición de selección corresponde a un parámetro,

	- dar el nombre de este parámetro.

## Prueba de una consulta con parámetros
<a name="prueba_una_consulta_con_parametros_ELTTEXTE000124"></a>
Para probar una consulta con parámetros:

1. Abra la consulta con parámetros en el editor de consultas.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_Go_Fenetre_WD_bl.gif) en los botones de acceso rápido.

3. Especifique el valor de los parámetros que se utilizarán para la prueba de consulta. En esta ventana, tiene la posibilidad de seleccionar un valor existente a través de ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OrdreAffichage.gif). Aparece el siguiente menú: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_MenuVisuValeur.gif)


4. Es posible:

	- seleccionar uno de los valores posibles entre los valores encontrados en el archivo

	- ver el archivo completo.
			**Atención**: Esta opción sólo está disponible si está activada la opción Datos en vivo.. Para obtener más información, consulte [Live Data](../Editeurs/2021005.md).




5. Validar. El resultado de la consulta se muestra de acuerdo con los valores especificados.




**Observación**: Si no se ha seleccionado ningún parámetro (parámetro no seleccionado), se ignorarán las condiciones que utilizan los parámetros.

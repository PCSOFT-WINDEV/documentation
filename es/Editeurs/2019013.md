


## Ejecución de la prueba de un proyecto estático
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
La prueba de todo el proyecto permite simular la puesta en marcha del sitio. Esto permite probar todo el sitio, incluso si su desarrollo aún no está completo.

**Observación**: La prueba de un proyecto se puede ejecutar independientemente del elemento actual en el editor.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000175"></a>


### Probar un sitio estático (o la parte estática de un sitio)
<a name="probar_sitio_estatico_parte_estatica_sitio_ELTPARAGRAPHE000020"></a>

Para ejecutar la prueba de un sitio estático desde el editor:

1. En la pestaña "Proyecto", en el grupo "Modo de prueba", despliegue "Modo de prueba" y seleccione "Depurar proyecto desde la página de inicio".

2. El editor se minimiza automáticamente y el proyecto se ejecuta en el navegador Default.

3. La página de inicio aparece en pantalla. 




Al ejecutar la prueba del proyecto, se ejecutan los siguientes procesos antes de abrir la primera página:

- cargar la primera página (código del navegador).

- entrando en el primer control de la primera página (si este control tiene una entrada Process) (código de navegador).




**Observación**: Si su sitio no tiene página de inicio, WEBDEV le propone definir una página de inicio para su sitio.. Para obtener más información, consulte [Página de inicio](../WDChamp/2030010.md).

**Atención**: Para ejecutar la prueba de un sitio estático, debe iniciarse el servidor Web instalado en el ordenador de desarrollo.. 
<a name="NOTE2_2"></a>


### Detener la prueba del proyecto
<a name="detener_prueba_del_proyecto_ELTPARAGRAPHE000056"></a>

Se pueden utilizar varios métodos para detener la prueba:

- 1er método: Cerrar el navegador en el que se ejecuta la prueba. WEBDEV muestra el editor abierto cuando se inició la prueba.

- 2º método: Vuelva al editor con la barra de tareas (o Alt + Tab) y confirme que la prueba debe ser detenida.. WEBDEV muestra el editor abierto cuando se inició la prueba.

## Notas
<a name="notas_ELTTEXTE000205"></a>


### Navegador de prueba
<a name="navegador_prueba_ELTPARAGRAPHE000068"></a>

Para seleccionar el navegador de pruebas, en la pestaña "Proyecto", en el grupo "Modo de prueba", despliegue "Modo de prueba" y seleccione "Navegador de prueba".

El navegador Default está definido en las opciones de WEBDEV: 

- En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WEBDEV". 

- Mostrar la pestaña "Web".


Para obtener más información, consulte [Opciones de WEBDEV](../Editeurs/2036001.md).

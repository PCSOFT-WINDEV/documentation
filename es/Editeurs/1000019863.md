## Condición previa
<a name="condicion_previa_ELTTEXTE000108"></a>
Para desarrollar una aplicación WINDEV Mobile para iPhone/iPad, necesita: 

- 1 PC 

- 1 Mac

- 1 iPhone y/o iPad (opcional)


**¿Por qué un PC?**
WINDEV Mobile 28 es una aplicación sólo para Windows. 
La aplicación iOS se creará, desarrollará y optimizará en el PC. 
**Consejo**: No es necesario tener un PC. Puede instalar un software de virtualización como "Parallels Desktop" en un Mac para ejecutar Windows en Mac. En este caso, WINDEV Mobile puede ejecutarse en Mac. 

**Entonces, ¿por qué una Mac?**
Una vez finalizado el desarrollo, deberá compilar la aplicación en un Mac utilizando Xcode 13 o posterior.

**¿Cómo descargar Xcode?**
Para instalar Xcode, debe tener una cuenta de iTunes (y por lo tanto, un ID de Apple). Xcode se puede descargar de forma gratuita desde el Mac App Store (válido al publicar este documento). 

Observación: También puede inscribirse en el **iOS and Mac Developer Program** como desarrollador de aplicaciones para Mac, iPhone y iPad. Debe inscribirse para implementar sus aplicaciones. Así, podrá descargar una versión de Xcode. Xcode se puede descargar de forma gratuita (válido al publicar este documento).

Sitio web: [http://developer.apple.com/xcode/](http://developer.apple.com/xcode/)




## Gestión de una Component interna
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000159"></a>
Un componente interno es un conjunto de elementos de un proyecto. Este conjunto se utiliza para:

- **Organizar un proyecto**: puede crear componentes internos para agrupar los elementos de un proyecto, por funcionalidad, por ejemplo.

- **Compartir elementos entre varios proyectos**: Los elementos de un componente interno pueden ser compartidos entre varios proyectos a través del SCM. Para obtener más información, consulte [Compartir un componente interno](../Editeurs/2014101.md).




El editor permite manipular un Component interno desde el panel "Explorador de proyectos". Los elementos de un Component interno se ven en una estructura jerárquica en el explorador de proyectos: ventanas, informes, páginas,...

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Posibles operaciones
<a name="posibles_operaciones_ELTTEXTE000183"></a>


### Visualización de la description de un Component interno
<a name="visualizacion_description_component_interno_ELTPARAGRAPHE000026"></a>

Para visualizar la description de un Component interno:

1. Seleccione el componente interno en el panel "Explorador de proyectos".

2. Seleccione "description" en el menú contextual de reporte.




Para más detalles sobre la información mostrada, consulte [Ventana de descripción del componente interno](../Editeurs/2014103.md).
<a name="NOTE3_2"></a>


### Añadir un elemento en la Component interna
<a name="anadir_elemento_component_interna_ELTPARAGRAPHE000041"></a>

Se puede añadir un elemento en un Component interno:

- a través de la description de la Component interna (pestaña "Elementos").

- a través del panel "Explorador de proyectos. Para ello, arrastre y suelte un elemento del proyecto a la carpeta interna de Component.




El archivo correspondiente se traslada físicamente al directorio de Component interno. Por defecto, el elemento incluido en la Component interna es "Público". Para convertir este elemento en "Privado", seleccione "Cambiar a privado" en el menú contextual del elemento.
<a name="NOTE3_3"></a>


### Borrar un elemento de la Component interna
<a name="borrar_elemento_component_interna_ELTPARAGRAPHE000053"></a>

Un elemento puede ser eliminado de la lista interna de Component:

- a través de la description de la Component interna (pestaña "Elementos").

- a través del panel "Explorador de proyectos: una operación de Drag and Drop realizada sobre un elemento de la Component interna a la categoría de proyecto correspondiente permite "eliminar" el elemento de la Component interna. El elemento se convierte en associated con el proyecto.




**Atención**: Si se elimina un elemento de una Component interna mediante la tecla Suprimir en un proyecto que no utiliza la SCM, este elemento dejará de formar parte del proyecto. Para volver a integrar este elemento de Component interno en el proyecto, habrá que abrirlo y añadirlo al proyecto manualmente.
<a name="NOTE3_5"></a>


### Modificación de los elementos públicos y privados de un Component interno
<a name="modificacion_los_elementos_publicos_privados_component_interno_ELTPARAGRAPHE000065"></a>

Se puede realizar la modificación de los elementos públicos y privados que se encuentran en una Component interna:

- En el panel "Explorador de proyectos", seleccione el elemento y, a continuación, seleccione "Cambiar a privado" o "Cambiar a público" en el menú contextual. Si el elemento se cambia a privado, se muestra un cuadrado rojo en el elemento en el panel "Explorador de proyectos".

- A partir de la descripción del componente interno (pestaña "Elementos").



<a name="NOTE3_6"></a>


### Borrar un Component interno
<a name="borrar_component_interno_ELTPARAGRAPHE000075"></a>

Un Component interno puede ser eliminado del proyecto mediante la tecla Suprimir.

**Atención**: Si se elimina un Component interno de un proyecto que no utiliza el SCM, los elementos del Component interno dejarán de formar parte del proyecto. Para volver a incluir los elementos de la Component interna en el proyecto, habrá que abrir los elementos de la Component interna y añadirlos al proyecto manualmente.

Para reintegrar el componente interno en el proyecto, en la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Un componente interno .. Desde un archivo".
<a name="NOTE3_7"></a>


### Listado de los componentes internos del proyecto
<a name="listado_los_componentes_internos_del_proyecto_ELTPARAGRAPHE000098"></a>

Para listar los componentes internos de un proyecto:

1. En el explorador del proyecto: 

	- Seleccionar la carpeta "Componentes internos. 

	- Seleccione "Gestionar los componentes internos del proyecto" en el menú contextual.




2. La ventana que aparece enumera todos los componentes internos utilizados por el proyecto. Desde esta ventana, puedes hacerlo:

	- Muestra la description del Component interno seleccionado.

	- Actualizar la Component interna (durante una acción a través de SCM).

	- Borrar un Component interno .

	- Importar una Component interna encontrada en la SCM.

	- Importar los archivos de un Component interno al proyecto. Esta opción permite copiar los archivos de un Component interno en el proyecto actual. Los archivos se copian y no se comparten. Esta opción es útil cuando se borra una Component por error del proyecto.

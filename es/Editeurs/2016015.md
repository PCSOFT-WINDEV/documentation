


## Hoja de estilo personalizada
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000181"></a>
Cuando se crea un proyecto, se crea una hoja de estilo personalizada en blanco y associated con el proyecto actual. Esta hoja de estilo puede ser modificada desde el editor de ventanas y contendrá los diferentes estilos creados por el desarrollador.

Esta hoja de estilo se crea en el directorio del proyecto y se llama "&lt;ProjectName>.STY".

Otra hoja de estilo personalizada puede ser associated con el proyecto en cualquier momento.

Esta página explica cómo hacerlo:

- [Asociar un proyecto con una hoja de estilo personalizada](#NOTE2).

- [Crear, modificar o eliminar un estilo en una hoja de estilo personalizada](#NOTE3).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo asociar un proyecto con una hoja de estilo personalizada?
<a name="¿como_asociar_proyecto_con_una_hoja_estilo_personalizada_ELTTEXTE000205"></a>
Para asociar el proyecto actual con una hoja de estilo personalizada: 

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Visualizar la pestaña "Estilo".

3. Selecciona la hoja de estilo personalizada (archivo "STY") que debe ser associated con el proyecto en "Style sheet".




Observaciones:

- La misma hoja de estilo personalizada puede ser associated con varios proyectos.

- Cuando se abre una ventana que usa estilos de la hoja de estilo del proyecto anterior, se puede:

	- *no cambiar nada*. Los controles que usan los estilos de la hoja de estilo anterior no se modificarán.. Le aconsejamos que asocie estos controles con los estilos de la nueva hoja de estilo.
			Los estilos de la hoja de estilo anterior se pueden añadir a la nueva hoja de estilo personalizada durante la siguiente apertura de la ventana.

	- *añadir los estilos encontrados en la hoja de estilo anterior* a la nueva hoja de estilo personalizada.

## ¿Cómo crear
<a name="¿como_crear_ELTTEXTE000229"></a>


### Creación de un estilo para un tipo de control en la hoja de estilo personalizada del proyecto actual
<a name="creacion_estilo_para_tipo_control_hoja_estilo_personalizada_del_proyecto_actual_ELTPARAGRAPHE000064"></a>

Para crear un estilo para un tipo de control en la hoja de estilo personalizada del proyecto actual: 

1. Crear una control de este tipo en una ventana de su proyecto a través del panel "Creación".. Por ejemplo, para crear un nuevo estilo de botón, crea un botón control.

2. Muestra la ventana para editar las hojas de estilo ("Elige un estilo" de el menú contextual de la control).

3. Haga clic en el botón "Crear. Aparece la ventana del estilo description.

4. Especificar las opciones de estilo deseadas (pestañas "General" y "Estilo") y validar. Este nuevo estilo se añade automáticamente a la hoja de estilo personalizada del proyecto actual.. Este estilo se puede aplicar a los controles del mismo tipo en todos los proyectos que utilizan esta hoja de estilo.




Observación: También puedes crear un nuevo estilo desde la ventana de description de un control:

1. Vaya a la pestaña "Estilo" de la ventana de descripción del control (opción "Descripción" del menú contextual).

2. Modificar las opciones de estilo deseadas y validar (botón "OK" o "Aplicar"). Se muestra la ventana para modificar el estilo de control.

3. Seleccione la casilla "Añadir este estilo al proyecto", introduzca el nombre de este nuevo estilo y valide (botón "OK").



<a name="NOTE3_2"></a>


### Modificación de un estilo para un tipo de control en la hoja de estilo personalizada del proyecto actual
<a name="modificacion_estilo_para_tipo_control_hoja_estilo_personalizada_del_proyecto_actual_ELTPARAGRAPHE000082"></a>

Para modificar un estilo para un tipo de control en la hoja de estilo personalizada del proyecto actual:

1. Seleccione una control de este tipo en una de las ventanas de su proyecto. Por ejemplo, para modificar el estilo de un botón, seleccione un botón control.

2. Muestra la ventana de edición de las hojas de estilo ("Elige un estilo" de el menú contextual).

3. Seleccione el estilo que desea modificar.

4. Haga clic en el botón "Modify". Aparece la ventana del estilo description.
	**Observación**: El botón "Modificar" es Grayed si el estilo seleccionado pertenece a una hoja de estilo no modificable (hoja de estilo de plantilla por ejemplo).

5. Realice los cambios deseados (pestañas "General" y "Estilo") y valídelos ("OK").

6. Aplicar las modificaciones (botón"Aplicar"). Las modificaciones se aplican automáticamente a la control actual y a las control que utilizan este estilo en las ventanas abiertas en el editor de ventanas.




Observaciones:

- Para aplicar los cambios realizados en todos los controles que utilizan este estilo en el proyecto actual, vaya a la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Recompilar y sincronizar" y seleccione "Actualizar estilo y plantilla".

- También puedes modificar un estilo desde la description de un control (pestaña "Estilo").

- Para modificar el estilo de varios controles del mismo tipo, seleccione los controles (manteniendo la tecla Ctrl pulsada, por ejemplo) antes de abrir la ventana utilizada para editar las hojas de estilo.



<a name="NOTE3_3"></a>


### Eliminar un estilo para un tipo de control de la hoja de estilo personalizada del proyecto actual
<a name="eliminar_estilo_para_tipo_control_hoja_estilo_personalizada_del_proyecto_actual_ELTPARAGRAPHE000116"></a>

Para eliminar un estilo para un tipo de control de la hoja de estilo personalizada del proyecto actual:

1. Seleccione una control de este tipo en una de las ventanas de su proyecto. Por ejemplo, para borrar un estilo de botón, seleccione un botón control.

2. Muestra la ventana de edición de las hojas de estilo ("Elige un estilo" de el menú contextual).

3. Seleccione el estilo que desea eliminar.

4. Haga clic en el botón "Eliminar" y confirme la eliminación del estilo..
	**Observación**: El botón "Borrar" es Grayed si el estilo seleccionado pertenece a una hoja de estilo no modificable (hoja de estilo de plantilla por ejemplo).

5. Seleccione el nuevo estilo del control y aplíquelo con el botón "Aplicar".




Observaciones:

- Si otros controles de proyecto utilizan el estilo eliminado, le aconsejamos que seleccione "No action" al abrir las ventanas y que asocie estos controles a un estilo diferente.. Para obtener más información, consulte [Cambiando el estilo associated con un control](../Editeurs/2016006.md).

- También puedes eliminar un estilo desde la description de un control (pestaña "Estilo").

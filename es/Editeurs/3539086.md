## Configuración del servidor para el protocolo CGI
<a name="configuracion_del_servidor_para_protocolo_cgi_ELTTEXTE000136"></a>
Para configurar el servidor web: 

1. Editar el archivo "httpd.conf" en un editor de texto.
	Este archivo se encuentra en el subdirectorio "/Conf" del directorio instalación del servidor Apache.

2. Añada el alias Definition para el protocolo CGI (WD28 0AWP.EXE) al archivo "httpd.conf" con las siguientes líneas:
	
	```txt
	ScriptAlias /<Alias>/ "Engine directory"
	<Directory "Engine directory">
	Require all granted
	</Directory>
	```


	- **Ejemplo: Ordenador de desarrollo**: el motor está en el directorio "C:\\WEBDEV\\PROGRAMS\\ENGINE\\WINDOWS\\AWP" y el nombre del alias es WD28 0AWP:
			
		```txt
		ScriptAlias /WD280AWP/ "C:/WEBDEV/PROGRAMS/ENGINE/WINDOWS/AWP/"
		<Directory "C:/WEBDEV/PROGRAMS/ENGINE/WINDOWS/AWP/">
		Require all granted
		</Directory>
		```


	- **Ejemplo: Despliegue del equipo**: el motor está en el directorio "C:\\WEBDEV\\ENGINE\\WINDOWS\\AWP" y el nombre del alias es WD28 0AWP:
			
		```txt
		ScriptAlias /WD280AWP/ "C:/WEBDEV/ENGINE/WINDOWS/AWP/"
		<Directory "C:/WEBDEV/ENGINE/WINDOWS/AWP/">
		Require all granted
		</Directory>
		```





3. Reinicie el servidor Apache.




**Observaciones**:

- los caracteres \\se convierten en /

- no olvide la final / en la ruta.

- debe respetarse la sensibilidad a las mayúsculas y minúsculas.

## Configurar el servidor para los sitios WEBDEV
<a name="configurar_servidor_para_los_sitios_webdev_ELTTEXTE000160"></a>
Para configurar el servidor web:

1. Editar el archivo "httpd.conf" en un editor de texto. Este archivo se encuentra en el subdirectorio "/Conf" del directorio instalación del servidor Apache.

2. Añadir en el archivo "httpd.conf" la Definition de alias para las imágenes de cada sitio WEBDEV:
	
	```txt
	;alias for the images, js, static files, css...
	Alias /<Name of Image alias>/ "Directory of application images"
	<Directory "Directory of application images">
	Require all granted
	</Directory>
	```


3. Añada (si es necesario) la información utilizada para gestionar las URL cortas en el archivo "httpd.conf":
	
	```txt
	;for the management of short URLs
	<ProjectName Files>
		Require all granted
		ForceType application/WEBDEV28-awp 
	</Files>
	```


4. Ejemplo del sitio WEBDEMO.
	
	```txt
	;alias for the images, js, static files, css...
	Alias /WEBDEMO_WEB/ "C:/WEBDEMO/WEBDEMO_WEB/"
	<Directory "C:/WEBDEMO/WEBDEMO_WEB/">
	Order allow,deny
	Allow from all
	</Directory>
	;to manage short URLs
	<WEBDEMO files>
		Require all granted
		ForceType application/WEBDEV28-awp 
	</Files>
	```


5. Reinicie el servidor Apache.



<a name="NOTE2_2"></a>


### Atención
<a name="atencion_ELTPARAGRAPHE000062"></a>

- Con el servidor Apache, el nombre del alias es "sensible a mayúsculas y minúsculas".. Debe cumplir con el caso definido para el alias en las páginas HTML que hacen referencia al directorio de imágenes.. 

- Se debe definir un alias de imágenes para cada sitio WEBDEV.

## Configuración del servidor para los sitios que utilizan Active WEBDEV Pages
<a name="configuracion_del_servidor_para_los_sitios_que_utilizan_active_webdev_pages_ELTTEXTE000190"></a>
Para configurar el servidor web: 

1. Editar el archivo "httpd.conf" en un editor de texto. Este archivo se encuentra en el subdirectorio "/Conf" del directorio instalación del servidor Apache.

2. Añade la siguiente Line en la sección "AddType":
	
	```txt
	AddType application/WEBDEV28-awp .awp
	AddType application/WEBDEV28-awws .awws
	```


3. Añade la siguiente Line a la sección "Acción":

	- En Windows: 
			
		```txt
		Action application/WEBDEV28-awp /WD280AWP/WD280AWP.exe virtual
		Action application/WEBDEV28-awws /WD280AWP/WD280AWP.exe virtual
		```


	- En Linux: 
			
		```txt
		Action application/WEBDEV28-awp /WD280AWP/WD280AWP virtual
		Action application/WEBDEV28-awws /WD280AWP/WD280AWP virtual
		```








Observación: en este Line, WD28 0AWP corresponde al alias del motor.

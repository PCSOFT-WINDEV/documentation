
## Creación de un archivo de ayuda
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
WINDEV, WEBDEV y WINDEV Mobile permiten crear un archivo de ayuda y asociarlo a una aplicación o sitio web. Esta ayuda en línea puede ser iniciada por el usuario:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) desde una ventana de la aplicación (botón de ayuda, por ejemplo).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) desde la opción "? .. Ayuda".




El sistema de ayuda associated con la aplicación se crea en un editor dedicado: el editor de ayuda.

Están disponibles los siguientes formatos de ayuda:

- **HTML**: ayuda en formato HTML. El sistema de ayuda es un sitio web estático con un resumen y búsqueda de texto completo. 

- **CHM**: Ayuda en formato Html compilado.

- **PDF**: Archivo de ayuda en PDF.

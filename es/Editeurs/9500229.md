<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
La galería del proyecto Image permite ver todos los Image s de un proyecto en un solo clic. 

Esta galería ofrece varias características: 

- Se le notifica si no se utiliza una Image en uno de los directorios del proyecto. Si es necesario, se puede borrar.

- Las imágenes duplicadas están señaladas: 

	- imágenes estrictamente idénticas,

	- imágenes que son binarias idénticas pero con un nombre diferente, o almacenadas en subdirectorios diferentes. 


 Eliminar imágenes permite reducir el tamaño de los proyectos.

- Las imágenes que no están integradas en el SCM también están señaladas.

## Funcionalidades disponibles
<a name="funcionalidades_disponibles_ELTTEXTE000185"></a>


### Visualización de imágenes
<a name="visualizacion_imagenes_ELTPARAGRAPHE000049"></a>

La galería del proyecto Image le permite: 

- Listar y ver las imágenes ubicadas en las carpetas del proyecto actual.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Galerie_projet%20-%20HC%20N%B0001.gif&type=thumb)
 Puede seleccionar sólo la carpeta deseada, buscar un Image o clasificar los Image que se muestran. También puede ver las imágenes en una lista con miniaturas o en una lista detallada utilizando los iconos de ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Galerie_projet%20-%20HC%20N%B0001%201.gif). 





### Manejo de archivos
<a name="manejo_archivos_ELTPARAGRAPHE000060"></a>

La galería del proyecto Image le permite:

- Renombrar, duplicar o eliminar el archivo seleccionado a través de las opciones del grupo "Archivos" de la cinta de opciones.. 

- Imprime la Image seleccionada (Crtl + P). La ventana de previsualización y configuración se abre automáticamente. 

- Mostrar imagen. Esta opción se utiliza para mostrar la Image en tamaño real en el área de visualización de archivos. 

- Editar Image. Esta opción se utiliza para editar el Image seleccionado en el editor Image del producto utilizado. 
	Atención: Esta opción cierra la galería del proyecto Image. 





### Seleccione los archivos
<a name="seleccione_los_archivos_ELTPARAGRAPHE000072"></a>

La galería del proyecto Image permite seleccionar los Image s mostrados a través de varios filtros: 

- La opción "5 estados" se utiliza para mostrar sólo imágenes de 5 estados ubicadas en los directorios del proyecto.. 

- La opción "Multi DPI" se utiliza para listar sólo los archivos Image correspondientes a la gestión multi DPI. 

- La opción "Fuera del SCM" permite listar solo las imágenes que no tienen historial (registros) en el SCM.

- La opción "No utilizado" se utiliza para listar sólo las imágenes que no son utilizadas por el proyecto.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Galerie_projet%20-%20HC%20N%B0004.gif&type=thumb)


- La opción "Duplicar" se utiliza para listar imágenes potencialmente duplicadas almacenadas en los diferentes directorios del proyecto.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Galerie_projet%20-%20HC%20N%B0003.gif&type=thumb)






### Limpieza del proyecto
<a name="limpieza_del_proyecto_ELTPARAGRAPHE000085"></a>

La opción "Limpiar" se utiliza para iniciar la limpieza asistente para eliminar los archivos no utilizados. Para obtener más información, consulte [Limpieza del proyecto](../Editeurs/9000068.md).

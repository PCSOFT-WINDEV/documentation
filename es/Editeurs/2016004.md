


## Estilos CSS (Cascading Style Sheet)
			





## Presentación
<a name="presentacion_ELTTEXTE000118"></a>


### Los estilos CSS son los estilos estándar usados en INTERNET.
<a name="los_estilos_css_son_los_estilos_estandar_usados_internet_ELTPARAGRAPHE000008"></a>

Un estilo CSS se utiliza para definir algunas características de estilo de un control. Los estilos CSS están incluidos y son utilizados por los estilos WEBDEV. 

WEBDEV le permite administrar los estilos CSS. Es posible: 

- Crear un estilo CSS.

- Modificar un estilo CSS.

- Asociar un estilo CSS con un control.

- overload el estilo CSS associated con un control.

## Gestión de los estilos CSS
<a name="gestion_los_estilos_css_ELTTEXTE000142"></a>


### Creación y modificación de un estilo CSS
<a name="creacion_modificacion_estilo_css_ELTPARAGRAPHE000024"></a>

La creación y la modificación de un estilo CSS se puede realizar: 

- de la ventana de description de los estilos de control. 

- desde la ventana de gestión de estilos CSS. Para obtener más información, consulte [Edición de las hojas de estilo CSS](../Editeurs/2016001.md). 





### Creación y modificación desde la ventana de description del estilo control
<a name="creacion_modificacion_desde_ventana_description_del_estilo_control_ELTPARAGRAPHE000035"></a>

La ventana description de control s se utiliza para gestionar el estilo de este control (pestaña "Estilo"). La pestaña "Estilo" gestiona varios elementos. Los elementos que pueden utilizar un estilo CSS se identifican mediante "(CSS)" al final de su título. Dependiendo del tipo de control, se pueden utilizar varios estilos CSS para los diferentes elementos del control. Por ejemplo, para un control List Box: 

- se utiliza un estilo CSS para la leyenda de control

- se utiliza un estilo CSS para los elementos que se encuentran en la control.




El estilo CSS associated con un elemento control puede ser local o no.

**Si el estilo es local**, significa que este estilo es associated con el control actual (y que ya no depende de las modificaciones hechas a la hoja de estilo del proyecto). Este estilo no será el definido en la hoja de estilo CSS: este estilo se definirá en el archivo HTM de la página.
Se debe evitar el uso de estilos que sean "locales a un control" porque esta opción aumenta el tamaño de las páginas HTML generadas (y por lo tanto el tiempo necesario para cargar las páginas en el navegador del usuario de la Web).

**Si el estilo CSS corresponde al nombre de un estilo CSS**, significa que el control utiliza un estilo CSS definido en la hoja de estilo CSS. Si se modifica el estilo CSS, se actualizarán todos los controles que usen este estilo CSS.. 

Para editar las características del estilo CSS, seleccione la opción "Editar" del botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletCatalogueimage%201.gif). Es posible: 

- editar el estilo local

- editar el estilo CSS utilizado. 

- crear un nuevo estilo CSS y asociarlo con el control. 




Para obtener más información, consulte [description ventana de un estilo CSS](../Editeurs/2016007.md). 


### Asociar un estilo CSS con un control
<a name="asociar_estilo_css_con_control_ELTPARAGRAPHE000064"></a>

Para asociar un estilo CSS a un control, basta con especificar, en la ventana description del estilo, el estilo CSS que debe utilizarse para el elemento solicitado. 


### Sobrecargando el estilo CSS associated con un control
<a name="sobrecargando_estilo_css_associated_con_control_ELTPARAGRAPHE000069"></a>

**Atención**: Una modificación realizada en la sección inferior de las características de estilo no modifica el estilo CSS associated con el control: esta modificación redefine el estilo CSS localmente. Sólo se sobrecargan las características del estilo CSS de la etapa de corriente.. Esta modificación no se aplica a todos los controles que utilizan el estilo CSS.

## Presentación
<a name="presentacion_ELTTEXTE000156"></a>
Los sitios web pueden visualizarse en muchos dispositivos (Android, iPhone, etc.), pero estos dispositivos presentan pantallas más pequeñas que los ordenadores de sobremesa.

Puede ajustar cómo se muestran los elementos en los dispositivos móviles mediante diseños, zonificación, etc.

WEBDEV también le permite definir el modo de visualización de cada Page del sitio en los dispositivos móviles. Puedes crear sitios específicos para dispositivos Apple (iPad o iPhone).

Observación: WINDEV Mobile permite crear aplicaciones para iPhone y iPad. Para obtener más información, consulte [Desarrollar una aplicación para iPhone o iPad](../Editeurs/1000019818.md).

### Características específicas de programación
<a name="caracteristicas_especificas_programacion_ELTPARAGRAPHE000086"></a>

Un sitio para iPhone se desarrolla de la misma forma que un sitio estándar. Sin embargo, algunas características de programación son específicas de iPhone.

**Se pueden iniciar llamadas telefónicas directamente desde un sitio web**: en la URL del enlace que permite hacer la llamada telefónica, especifique "tel:" y el número de teléfono.

Por ejemplo:


```wl
// Phone
MAP_PHONE.URL = "phone:" + CUSTOMER.Phone
```


También se puede mostrar la dirección del cliente en el mapa de Google directamente desde el formulario del cliente: en la URL del enlace, indique "http://maps.google.com/maps?q=" seguido de la dirección del cliente. 
Por ejemplo:


```wl
// Geo-localization
MAP_TGMAP.URL = "http://maps.google.com/maps?q=" + ...
	URLEncode(Customer.Address + " " + Customer.City + ", " + Customer.Country)
```

### Consejos
<a name="consejos_ELTPARAGRAPHE000105"></a>

- Utilice modelos de páginas específicos de iPhone.

- Reduzca al máximo el tamaño de las imágenes del sitio. Las imágenes grandes, las animaciones Flash o los videos pueden ralentizar el tiempo de carga de la página.

- Defina anclajes para gestionar las diferentes orientaciones del iPhone.

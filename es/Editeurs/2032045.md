


## Consulta con parámetros 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
<a name="Qry_Param"></a>
Una consulta con parámetros es una consulta para la que las posiciones de clasificación y los criterios de selección no están fijados.. Los valores de los parámetros se inicializan mediante la programación. En el momento de la ejecución, el resultado dependerá del valor dado a los parámetros.  

Por ejemplo, se puede utilizar una consulta con parámetros para seleccionar los pedidos realizados por un cliente específico.. El usuario sólo tendrá que seleccionar el nombre del cliente. La consulta utilizará este nombre como parámetro.

**Para incluir una consulta con parámetros en una aplicación WINDEV, en una aplicación WINDEV Mobile o en un sitio WEBDEV**, debe:

- Crear una consulta (en el editor de consultas o mediante programación). 

- Crear una condición y/o una clasificación.  

- Definir el parámetro. 

- [Modificar la aplicación para utilizar la consulta con parámetros](../Editeurs/2032032.md).




La consulta con parámetros se puede utilizar con: 

- una ventana o página control (Tabla, Cuadro de Lista o control Combo Box),

- un reporte, 

- mediante programación.

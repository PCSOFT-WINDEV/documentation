


## Incluir archivos JavaScript o una web Resource
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000201"></a>
WEBDEV permite incluir archivos JavaScript o recursos externos en un proyecto WEBDEV. Estos archivos pueden:

- incluirse a nivel de proyecto o a nivel de Page.

- encontrarse en el directorio _WEB del sitio o ser un archivo externo al sitio

- estar en un formato específico (ISO, UTF-8, etc.).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000225"></a>


### Incluir archivos JavaScript (.js)
<a name="incluir_archivos_javascript_ELTPARAGRAPHE000022"></a>

Tiene la posibilidad de incluir uno o varios archivos JavaScript en un sitio o en una Page (\*.js). Estos archivos serán utilizados por las páginas HTML generadas. Estos archivos deben estar en el directorio "_WEB" del sitio.

Para incluir un archivo JavaScript en una aplicación Page: 

1. Abra la descripción de la página: en la pestaña "Página", en el grupo "Descripción", haga clic en "Descripción".

2. Seleccione la pestaña "Detalles".

3. En "Recursos externos", puede añadir recursos JavaScript: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeAjouterPropriete.gif) y seleccione el archivo JavaScript ruta.




4. Seleccione (si es necesario) la encoding deseada . Si la encoding corresponde a "", la encoding se detecta automáticamente. 

5. Valide.




Para incluir un archivo JavaScript en el proyecto: 

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Seleccione la pestaña "Avanzado".

3. Haga clic en "Archivos adicionales...".

4. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeAjouterPropriete.gif) y seleccione el archivo JavaScript ruta.

5. Seleccione (si es necesario) la encoding deseada . Si la encoding corresponde a "", la encoding se detecta automáticamente. 

6. Valide.


<a name="NOTE2_2"></a>


### Incluir recursos externos en el sitio
<a name="incluir_recursos_externos_sitio_ELTPARAGRAPHE000086"></a>

La inclusión de recursos externos en el sitio se utiliza para incluir los archivos de interfaz (interfaz de programación) propuestos por los servicios externos (Web API) en la WEB.. Esto le da la posibilidad de interactuar con estos servicios externos en el código del navegador.

Para incluir una Resource externa en una Page:

1. Abra la descripción de la página: en la pestaña "Página", en el grupo "Descripción", haga clic en "Descripción".

2. Seleccione la pestaña "Avanzado".

3. Seleccione la pestaña "Javascript".

4. Haga clic en "Añadir una web Resource" y especifique la dirección de la web Resource deseada.

5. Seleccione (si es necesario) la encoding deseada . Si la encoding corresponde a "", la encoding se detecta automáticamente. 

6. Valide.




Incluir un Resource externo en el proyecto: 

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Seleccione la pestaña "Avanzado".

3. Haga clic en "Archivos adicionales...".

4. Haga clic en el botón "Agregar un recurso Web" e indique la dirección del recurso Web. 

5. Seleccione (si es necesario) la encoding deseada . Si la encoding corresponde a "", la encoding se detecta automáticamente. 

6. Valide.


<a name="NOTE2_3"></a>


### automatic detección de encoding
<a name="automatic_deteccion_encoding_ELTPARAGRAPHE000147"></a>

Si la encoding corresponde a "", la encoding se detecta automáticamente:

- caso de un Web reference: el servidor es interrogado. La encoding es suministrada por el servidor.

- caso de un archivo reference: si el fichero está en UTF8, la encoding pasa automáticamente a UTF8.

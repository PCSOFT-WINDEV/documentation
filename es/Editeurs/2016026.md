
## WINDEV Mobile Generador de plantillas de tema: Personalización de los elementos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000447"></a>
El generador WINDEV Mobile de plantillas tema se utiliza para generar los archivos necesarios para crear un WINDEV Mobile plantilla personalizado. 

Tiene la posibilidad de personalizar los elementos generados: 

- [Imágenes](#NOTE2_1): estas imágenes se utilizarán en el estilo de los controles. 

- [Modelos de ventanas](#NOTE3_1). 

- [Modelos de reportes](#NOTE4_1). 




Esta página de ayuda presenta las principales imágenes utilizadas así como las modificaciones que se pueden realizar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Imágenes de plantilla
<a name="imagenes_plantilla_ELTTEXTE000471"></a>


### Consejos
<a name="consejos_ELTPARAGRAPHE000031"></a>

- La extensión de los archivos de Image puede ser modificada. Se pueden utilizar los siguientes formatos: JPG, PNG y GIF.

- Para que una imagen no sea usada en un control, todo lo que tienes que hacer es hacer que Transparent y set tengan un tamaño de 1x1 pixel. En este caso, el Image debe estar en formato PNG. Por lo tanto, el Image no se generará al generar el plantilla.

- Para que las imágenes se muestren correctamente en sus plantillas de tema, no modifique las dimensiones de los archivos.

## Modelos de ventanas
<a name="modelos_ventanas_ELTTEXTE000501"></a>
Las plantillas de ventanas se encuentran en el directorio "Template" del proyecto fuente de plantilla. 

La ventana modelo disponible es Template.wdt, modelo general de sus ventanas. Este modelo se utiliza para definir el fondo de sus ventanas (fondo Color, fondo Image, ...). 

Para personalizar las plantillas de ventanas, todo lo que tiene que hacer es abrirlas en WINDEV Mobile. Le aconsejamos que abra estos elementos en WINDEV Mobile **off project** (Arrastre y suelte el elemento a WINDEV Mobile por ejemplo).. 

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Modelos de reportes
<a name="modelos_reportes_ELTTEXTE000525"></a>
Las plantillas reporte se encuentran en el directorio "Template" del proyecto fuente de plantilla. Las plantillas reporte disponibles son: 

- Plantilla_Informes.wte: modelo general para informes. 

- Plantilla_Etiqueta.wte: Plantilla utilizada para los informes de Label. 

- Plantilla_Mailing.wte: Modelo utilizado para los informes de Mailing. 




Para personalizar las plantillas de reporte, todo lo que tiene que hacer es abrirlas en WINDEV Mobile. Le aconsejamos que abra estos elementos en WINDEV Mobile **off project** (Arrastre y suelte el elemento a WINDEV Mobile por ejemplo)..

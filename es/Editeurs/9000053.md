


## Generación múltiple
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
Las configuraciones de proyecto permiten definir fácilmente cómo se generará el proyecto. Para el mismo proyecto, tiene la posibilidad de definir los ejecutables, librerías, Webservices, ensamblajes de red,...

Para generar el resultado de cada configuración, se puede seleccionar la configuración solicitada en el panel "Explorador del proyecto" y generar el programa correspondiente (la generación de Icon se adapta automáticamente a la configuración seleccionada en los botones de acceso rápido). 

Existe un método más rápido: **la generación múltiple**. Puede seleccionar al mismo tiempo todas las configuraciones que desea generar para obtener los programas correspondientes de forma inmediata.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo realizar una generación múltiple?
<a name="¿como_realizar_una_generacion_multiple_ELTTEXTE000135"></a>
Para generar los resultados de varias configuraciones: 

1. En la pestaña "Proyecto", en el grupo "Generación", haga clic en "Generación múltiple".

2. En la ventana que se muestra, seleccione la configuración que se generará.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P2_WD_Generation_multiple%20-%20HC%20N%B0001.gif)


3. Validar. Se realiza la generación de las diferentes configuraciones.



<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000044"></a>

- Si ya se ha realizado una generación para una configuración, se tienen en cuenta los últimos parámetros de esta generación.

- Si no se realizó ninguna generación para una configuración, se tienen en cuenta los parámetros de Default.

- Configuración associated con un componente: el componente debe haberse creado para que se genere mediante la generación múltiple.

- Tiene la posibilidad de utilizar diferentes directorios de generación para cada configuración.. En este caso, cada directorio contendrá los archivos generados por una única configuración. No hay riesgo de sobrescribir archivos. Todos los archivos pertenecientes a una configuración pueden ser recuperados fácilmente. Para obtener más información, consulte [Configuración del proyecto](../Editeurs/9000030.md).

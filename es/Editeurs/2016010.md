## Presentación
<a name="presentacion_ELTTEXTE000224"></a>
Una hoja de estilo es un archivo ".STY. Una hoja de estilo agrupa:

- el estilo de los controles de la ventana (fondo Color, fuente, Border, ...),

- la información relativa al aspecto de los controles de ventana (fondo Image, dimensión, ...).




Las hojas de estilo personalizadas se pueden manejar desde la interfaz del editor de ventanas. Para obtener más información, consulte [Hoja de estilo personalizada](../Editeurs/2016015.md).

El manejo de los estilos a través de los archivos ".STY" es una opción avanzada de WINDEV.

Desde el archivo ".STY", puede hacerlo:

- [ver todos los estilos encontrados en un archivo ".STY](#NOTE2).

- [modificar un estilo en un archivo ".STY](#NOTE3).

- [borrar un estilo de un archivo ".STY](#NOTE3).

- [importar un estilo a un archivo ".STY](#NOTE4).

- [definir el estilo Default para un tipo de control](#NOTE5).

## ¿Cómo ver el contenido de una hoja de estilo (archivo.STY)?
<a name="¿como_ver_contenido_una_hoja_estilo_archivosty_ELTTEXTE000248"></a>
Para ver el contenido de una hoja de estilo, todo lo que tiene que hacer es abrir el correspondiente archivo ".STY" en WINDEV. Los estilos que se encuentran en la hoja de estilo se enumeran uno a Above el otro.

**Observación**: Las hojas de estilo de las plantillas de tema se encuentran en el directorio "Personal\\Template" del directorio WINDEV instalación.

## ¿Cómo modificar o borrar un estilo en una hoja de estilo (archivo.STY)?
<a name="¿como_modificar_borrar_estilo_una_hoja_estilo_archivosty_ELTTEXTE000272"></a>


### Modificar un estilo en una hoja de estilo (archivo".STY")
<a name="modificar_estilo_una_hoja_estilo_archivosty_ELTPARAGRAPHE000055"></a>

Para modificar un estilo en una hoja de estilo (archivo".STY"): 

1. Abrir la hoja de estilo (archivo ".STY") que contiene el estilo a modificar.

2. Seleccione el estilo que desea modificar.

3. Haz doble clic en el estilo (o selecciona "Tema-modelo description" de el menú contextual).

4. Haga los cambios deseados y aplíquelos.

5. Guarde la hoja de estilo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif)).




Observaciones:

- Estas modificaciones se aplicarán a los controles que utilizan este estilo al abrir el proyecto y/o las ventanas.

- Si había un proyecto abierto cuando se modificó la hoja de estilo (archivo ".STY"), vaya a la pestaña "Proyecto", grupo "Proyecto", amplíe "Recompilar y sincronizar" y seleccione "Actualizar estilo y plantilla" para tener en cuenta los cambios realizados en el proyecto actual.




**Atención**: Al actualizar WINDEV, las modificaciones realizadas en las hojas de estilo de los modelos de tema pueden sobrescribirse. Si se realizan modificaciones en las hojas de estilo de las plantillas de tema, le aconsejamos que haga una copia de seguridad de copy de los archivos modificados.
<a name="NOTE3_2"></a>


### Eliminar un estilo de una hoja de estilo (archivo".STY")
<a name="eliminar_estilo_una_hoja_estilo_archivosty_ELTPARAGRAPHE000089"></a>

Para eliminar un estilo de una hoja de estilo (archivo".STY"): 

1. Abra la hoja de estilo (archivo".STY") que contiene el estilo que desea eliminar.

2. Seleccione el estilo que desea eliminar.

3. Selecciona "Borrar" de el menú contextual (o pulsa la tecla Del).

4. Guarde la hoja de estilo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif)).




Observaciones:

- Cuando se abre un proyecto y/o una ventana que utiliza el estilo eliminado, se puede:

	- no cambiar nada. Los controles que utilizan el estilo eliminado no se modificarán. Le aconsejamos que asocie estos controles con otros estilos.
			El estilo eliminado se puede añadir a la hoja de estilo personalizada del proyecto durante la próxima apertura.

	- añadir el estilo eliminado a la hoja de estilo personalizada del proyecto.




- Si un proyecto se abrió cuando se eliminó un estilo de una hoja de estilo: 

	- cerrar este proyecto: en la pestaña "Inicio", en el grupo "General", despliegue "Cerrar" y seleccione "Cerrar proyecto".

	- reabrir este proyecto para tener en cuenta la supresión del estilo en el proyecto actual: en la pestaña "Inicio", en el grupo "General", despliegue "Abrir" y seleccione "Abrir un proyecto".

## ¿Cómo importar un estilo a una hoja de estilo (archivo".STY")?
<a name="¿como_importar_estilo_una_hoja_estilo_archivosty_ELTTEXTE000302"></a>
Para importar un estilo a una hoja de estilo (archivo".STY"): 

1. Abra la hoja de estilo (archivo".STY") que contiene el estilo que desea exportar.

2. Seleccione el estilo que desea exportar.

3. copy el estilo seleccionado en el portapapeles (Ctrl + C).

4. Abra la hoja de estilo (archivo".STY") en la que se importará el estilo.

5. Pegue el contenido del portapapeles (Ctrl + V).

6. Guarde la hoja de estilo (archivo ".STY") en la que se importó el estilo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif)). El estilo importado está ahora disponible en ambas hojas de estilo.
	Observación: Si el estilo importado utiliza archivos específicos (por ejemplo, el archivo Image que se muestra en el fondo del control), estos archivos se copian en el directorio de la hoja de estilo de importación.




**Atención**: Al actualizar WINDEV, los estilos importados en las hojas de estilo de las plantillas de tema pueden sobrescribirse. Si se realizan modificaciones en las hojas de estilo de las plantillas de tema, le aconsejamos que haga una copia de seguridad de copy de los archivos modificados.

## ¿Cómo definir el estilo Default para un tipo de control?
<a name="¿como_definir_estilo_default_para_tipo_control_ELTTEXTE000326"></a>
Para definir el estilo usado por Default para un tipo de control en el proyecto actual: 

1. Abra la hoja de estilo (archivo".STY") utilizada por el proyecto actual.

2. Seleccione el estilo a utilizar para un tipo de control. Por ejemplo, hay dos estilos disponibles para un "Botón" control: "StyleBtnA" y "StyleBtnB". Para usar el "StyleBtnA" de Default, selecciona el estilo "StyleBtnA".

3. Haga doble clic en el estilo (o seleccione "Tema-modelo description" en el menú contextual). Aparece la ventana del estilo description.

4. Vaya a la pestaña "Detalles".

5. Marque "Por Default" y valide.

6. Guarde la hoja de estilo (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif)).




Observación: Si su proyecto es associated con varias hojas de estilo de plantillas de tema, el estilo usado por Default será seleccionado de acuerdo con el siguiente orden:

- el estilo definido por Default en la hoja de estilo personalizada.

- el estilo definido por defecto en la hoja de estilo de la plantilla associated con la ventana actual.

- el estilo definido por Default en la hoja de estilo de Default de WINDEV.

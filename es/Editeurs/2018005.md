## Análisis de groupware usuario
<a name="analisis_groupware_usuario_ELTTEXTE000530"></a>
El análisis de groupware usuario es el siguiente:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=AnalyseGPW.gif)




**Archivos de datos utilizados para identificar a los usuarios**: estos archivos pueden ser comunes a varias aplicaciones:


|   |   |
| --- | --- |
| ***Configuración del GPWUser*** | Archivo de grupos de usuarios. |
| ***GPWConnectionHistoria*** | Archivo que contiene el historial de conexiones. Este archivo sólo se encuentra si se ha solicitado el historial de conexiones. |
| ***gpwUser*** | Fichero de usuarios. Contiene todas las características de los usuarios (login, apellido, nombre, contraseña,...) |

****

****

**Archivos de datos utilizados para la gestión de los derechos**: estos archivos son específicos de la aplicación. No se pueden compartir con otra aplicación.


|   |   |
| --- | --- |
| ***GPWConfiguración*** | Archivo que contiene la description de las diferentes configuraciones disponibles. Cada configuración es associated con un grupo de usuarios. |
| ***GPWElement*** | Archivo que contiene los diferentes elementos de la aplicación a configurar (controles de ventanas, opciones de menú,...) |
| ***GPWElementConfiguración*** | Status de cada elemento de aplicación para cada configuración. Por ejemplo, permite almacenar si el "menú de estadísticas" debe ser Grayed para el usuario "asistente". |

### Observación
<a name="observacion_ELTPARAGRAPHE000120"></a>

Para todas las llaves y para todos los componentes de las llaves compuestas:

- La dirección de navegación es ascendente.

- Las opciones de búsqueda son: Sensible al mayúsculas y minúsculas, Sensible a los caracteres acentuados, Sensible a los espacios y a la puntuación.

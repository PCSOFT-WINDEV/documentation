
## ¿Qué es una aplicación?
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000072"></a>
Una **application** es una set de objetos (ventanas, controles, ...) organizados con un propósito específico (vigilar las ventas de productos por ejemplo). El usuario utiliza una aplicación iniciando un archivo "Executable" (por ejemplo, "MonitorSales.exe").

WINDEV le permite crear una aplicación mediante la función **proyecto**.

Un proyecto es una set lógica de objetos y procesos cuyo propósito es lograr un objetivo determinado. El proyecto gestiona:

- las relaciones entre los objetos.

- las relaciones entre los objetos y los ficheros de datos (descritas en el análisis relacionado para el proyecto).




Un proyecto solo puede estar relacionado con un análisis.

Un mismo análisis puede asociarse a varios proyectos.




## Atajo de teclado en un botón (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000147"></a>
¿Quién no ha querido definir sus propios atajos de teclado, para acceder de forma fácil y rápida a las funciones más utilizadas de una aplicación? Las aplicaciones WINDEV ofrecen esta funcionalidad a todos los usuarios. Puede crear y modificar los atajos de teclado de sus aplicaciones WINDEV. 

El atajo de teclado se puede crear si no existe. 

El atajo de teclado de Default puede ser modificado si no le conviene. En este caso, puedes usar el atajo de Default o el nuevo.

Recordatorio: El atajo de teclado de un botón se define en la pestaña "UI" de la ventana control Botón description (opción "Atajo de teclado").

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000171"></a>


### Configuración del método abreviado de teclado
<a name="configuracion_del_metodo_abreviado_teclado_ELTPARAGRAPHE000026"></a>

Para configurar el atajo de teclado de un botón:

1. Haga clic con el botón derecho del ratón para abrir el menú contextual y seleccione "Elegir un atajo de teclado...".

2. En la ventana que se muestra, pulse las teclas deseadas. Aparecen las teclas seleccionadas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Choix%20d'un%20raccourci.jpg)
Valide. 

3. El acceso directo especificado se puede utilizar siempre que se utilicen la ventana y el botón.




**Observaciones**: 

- Si el acceso directo seleccionado ya se utiliza en la ventana, un mensaje indica que el acceso directo seleccionado no se puede utilizar.

- El acceso directo personalizado se muestra automáticamente en el tooltip del botón. Si ya se muestra un texto en el tooltip, el acceso directo se muestra al final del texto.

- Si el botón contiene un título, el acceso directo personalizado se muestra automáticamente al final del título del botón.

- Si se selecciona el atajo "&lt;Ninguno&gt;", se restaurará el comportamiento del botón Default (previsto por el desarrollador). 



<a name="NOTE2_2"></a>


### Modificación de un método abreviado de teclado personalizado
<a name="modificacion_metodo_abreviado_teclado_personalizado_ELTPARAGRAPHE000050"></a>

Para modificar un método abreviado de teclado personalizado:

1. Seleccione "Modificar acceso directo..." en el menú contextual del botón.

2. En la ventana que se muestra, especifique las teclas de acceso directo que desea utilizar.

3. Validar la modificación.


Observación: Si se selecciona el atajo "&lt;Ninguno&gt;", se restaurará el comportamiento del botón Default (previsto por el desarrollador).
<a name="NOTE2_3"></a>


### Desactivar la capacidad de crear accesos directos personalizados
<a name="desactivar_capacidad_crear_accesos_directos_personalizados_ELTPARAGRAPHE000063"></a>

Para evitar que el usuario cree accesos directos personalizados, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafAddShortcut*.
<a name="NOTE2_4"></a>


### Modificar la ventana para elegir las teclas de acceso directo
<a name="modificar_ventana_para_elegir_las_teclas_acceso_directo_ELTPARAGRAPHE000073"></a>

Por Default, la ventana para elegir las teclas de atajo está disponible en inglés y en francés. 

Para modificar esta ventana o traducirla, simplemente inclúyala en el proyecto: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccionar el directorio "Programas\\Datos\\Ventanas de preselección\\AAF".. 

3. Seleccionar la ventana "WinDevAaf_BtnShortcut.wdw. 

4. Valide la importación. La ventana está incluida en su proyecto y puede ser traducida o modificada.. 




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario.

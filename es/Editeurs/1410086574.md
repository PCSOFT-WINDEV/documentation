## Presentación
<a name="presentacion_ELTTEXTE000101"></a>
Para desarrollar aplicaciones universales de Windows 10 con WINDEV o WINDEV Mobile, debe instalar: 

- **[el SDK de Windows 10 de Microsoft](#NOTE2_1)**. El SDK para Windows 10 (Software Development Kit) es un set de archivos y aplicaciones distribuidos por Microsoft para compilar aplicaciones para el sistema operativo Windows 10. 

- **Visual Studio 2010 Paquete Redistribuible**. Este paquete puede descargarse del sitio web de Microsoft ([https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads](https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads)) (enlace válido en el momento de escribir esta página). 




**Atención**: 

- Para utilizar la Plataforma Universal Windows API s (UWP API), debe estar instalado Visual Studio (versión 2015 o posterior).. 

- La creación de un proyecto de Plataforma Universal de Windows con WINDEV o WINDEV Mobile está disponible independientemente del sistema de la computadora de desarrollo. Para generar y desplegar la aplicación, el sistema que se encuentra en la computadora de desarrollo debe ser Windows 8 (o posterior).

## SDK de Windows 10
<a name="sdk_windows_10_ELTTEXTE000125"></a>


### ¿Dónde encontrar el SDK para Windows 10?
<a name="¿donde_encontrar_sdk_para_windows_10_ELTPARAGRAPHE000031"></a>

El SDK para Windows 10 puede ser descargado del sitio de Microsoft: 

- [https://developer.microsoft.com/en-us/windows/downloads/sdk-archive/](https://developer.microsoft.com/en-us/windows/downloads/sdk-archive/)(versión en EE.UU.).

- [https://developer.microsoft.com/fr-fr/windows/downloads/sdk-archive/](https://developer.microsoft.com/fr-fr/windows/downloads/sdk-archive/)(versión FR).




**Importante**: 

- Este Address es válido en el momento de la publicación de esta documentación.

- Le recomendamos que compruebe si hay actualizaciones disponibles para el SDK de forma regular.. 



<a name="NOTE2_2"></a>


### ¿Cómo instalar el SDK para Windows 10?
<a name="¿como_instalar_sdk_para_windows_10_ELTPARAGRAPHE000050"></a>

Una vez que el archivo descargado se ha ejecutado, se instalan todas las herramientas necesarias para desarrollar una aplicación "Universal Windows 10 App" para Windows 10. 

Observación: La ruta del directorio instalación del SDK debe ser especificada en el asistente para la creación del proyecto.

## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
WINDEV puede ser usado para generar ventanas para una aplicación de Java.

La aplicación Java puede beneficiarse de la facilidad de uso, las pruebas automatizadas y la disposición de las ventanas de WINDEV. Las ventanas y los controles se utilizan en el "modo" Java, a través de clases y métodos especializados para cada ventana y control.

El "Javadoc" se suministra (en el directorio "Help\\Javadoc" de WINDEV).

Más específicamente, usted tiene la capacidad de generar el proyecto Eclipse. De hecho, la llamada al entorno WINDEV puede incluirse en los menús de Eclipse. Con WINDEV se suministra un plug-in.

## Generando las fuentes de Java del proyecto
<a name="generando_las_fuentes_java_del_proyecto_ELTTEXTE000184"></a>
Al generar las fuentes de Java del proyecto, se generan los siguientes elementos:

- un archivo (.jar) que contiene los elementos del proyecto y el código de WLanguage.

- un archivo fuente (.Java) para cada ventana de proyecto. Estos archivos se generan en el directorio src del directorio EXE.




Si se selecciona "Generar un proyecto Eclipse", también se generan los archivos.project y.classpath.
<a name="NOTE2_2"></a>


### Generando las fuentes de Java del proyecto
<a name="generando_las_fuentes_java_del_proyecto_ELTPARAGRAPHE000036"></a>

Generar las fuentes de Java del proyecto: 

1. Dependiendo de la configuración actual del proyecto: 

	- Si la configuración actual del proyecto corresponde al archivo de Java, en el panel "Proyecto", en el grupo "Generación", haga clic en "Generar".

	- Si la configuración actual del proyecto no se corresponde con el archivo de Java, utilice los botones de acceso rápido: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_generation_java.gif)





2. Seleccione "Usar las ventanas de proyecto de sus propios archivos de fuente de Java". Marque "Generar el proyecto Eclipse" si es necesario.. Pase a la pantalla siguiente.

3. Pase a la pantalla siguiente.

4. Especifique el nombre del archivo y su description. Pase a la pantalla siguiente.

5. Especificar si el WINDEV framework para Java debe ser incluido en el archivo. Pase a la pantalla siguiente.

6. Especifique los otros archivos que deben incluirse en el archivo ".jar. Puede seleccionar los controladores JDBC utilizados para acceder a una base de datos, por ejemplo. Pase a la pantalla siguiente.

7. Especifique:

	- el comando del compilador de Java para usar. Para personalizar este parámetro (si se instalan varios compiladores en el equipo, por ejemplo), seleccione "Custom".. En esta pantalla, también puede especificar el comando Line que será utilizado (vea la documentación sobre el compilador para más detalles).

	- el comando para crear el archivo comprimido. La opción "Personalizado" permite definir la herramienta que se utilizará para crear el archivo.. También puede especificar el comando Line que se utilizará (para más detalles, consulte la documentación del compilador) y el contenido del manifiesto del archivo. 

	- el comando para incluir otros archivos. La opción "Personalizar" permite definir la herramienta que se utilizará para extraer el archivo así como su comando Line.




8. Se crea el archivo ".jar. El archivo correspondiente fue creado en el directorio "EXE" del proyecto. Este archivo será utilizado por el proyecto Eclipse.




**Observación**: Para iniciar la aplicación Java una vez que se hayan generado las fuentes de Java, debe importar el proyecto a Eclipse y ejecutarlo desde Eclipse, o compilar y ejecutar el contenido del directorio scr que se encuentra en la raíz del directorio EXE.

## Uso del plug-in de Eclipse
<a name="uso_del_plugin_eclipse_ELTTEXTE000214"></a>


### Instalación del plug-in "Eclipse
<a name="instalacion_del_plugin_eclipse_ELTPARAGRAPHE000071"></a>

Para instalar el plug-in "Eclipse":

1. Ejecute el archivo WDEclipsePlugin.zip que se encuentra en el directorio "\\Program\\Data" de WINDEV.

2. Descomprime los archivos de WDEclipsePlugin.zip en el directorio instalación del software "Eclipse".

3. Reiniciar el software "Eclipse.  

4. En el menú "WINDEV .... Preferencias", especifique la ruta del ejecutable correspondiente a WINDEV. Esto le da la posibilidad de abrir directamente sus proyectos WINDEV desde el software Eclipse.



<a name="NOTE3_2"></a>


### Usando un proyecto WINDEV con Eclipse
<a name="usando_proyecto_windev_con_eclipse_ELTPARAGRAPHE000083"></a>

Utilizar un proyecto WINDEV con Eclipse:

1. En Eclipse, seleccione "Archivo... Importar... Proyecto existente en el espacio de trabajo".

2. Haga clic en "Siguiente".

3. Seleccione el directorio "Exe" del proyecto WINDEV a importar (una vez generados los archivos necesarios).

4. Seleccione el proyecto que desea importar y validar.




En el software Eclipse, el proyecto aparece en la vista "Explorador de paquetes".

Desde el "Explorador de Paquetes", tiene la capacidad de iniciar directamente el proyecto WINDEV a través de el menú contextual del proyecto.

- Casilla de verificación/Botón de radio

- jugarreta

- Image

- Tabla

- Combo Box

- lengüeta

- TreeView

## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
WINDEV, WEBDEV y WINDEV Mobile proponen diferentes modos de uso del entorno de desarrollo.

Las siguientes opciones se proponen cuando se inicia el producto por primera vez:

- **Recuperar la configuración del entorno de una versión anterior**. Esta opción le permite restaurar fácilmente todos los ajustes realizados en una versión anterior instalada en su ordenador.

- **Entorno simplificado**. Esta opción se recomienda para descubrir WINDEV, WEBDEV o WINDEV Mobile. Algunas características avanzadas no se muestran en el entorno (SCM, Centros control, Reglas de Negocios, ...). Estas funciones se pueden volver a activar en cualquier momento si es necesario.

- **Entorno completo**. Esta opción le permite utilizar un entorno exhaustivo. Algunas de las opciones que rara vez se utilizaban en las versiones anteriores se sustituyen por nuevas funciones.




También tiene la posibilidad de elegir la configuración de la pantalla:

- ***pantalla individual***: los diferentes paneles se mostrarán en la misma pantalla que el entorno de desarrollo.

- ***multipantalla***: el entorno de desarrollo se mostrará en una pantalla y los diferentes paneles se agruparán en la segunda pantalla.

- ***pantalla ancha***: el espacio libre a la derecha de la pantalla está optimizado para liberar la parte inferior del editor.  




El entorno se puede cambiar o personalizar en cualquier momento.

## Modificación del entorno de desarrollo
<a name="modificacion_del_entorno_desarrollo_ELTTEXTE000165"></a>
Cuando se utiliza WINDEV, WEBDEV o WINDEV Mobile, el modo de entorno seleccionado puede identificarse fácilmente: se muestra debajo del logotipo de la versión. Por ejemplo:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Env_mode_dev.gif)


Cambiar el entorno:

- haga clic en el modo actual.

- seleccione "Herramientas .. Opciones .... Opciones del entorno".




Aparece la ventana para configurar el entorno.

<a name="NOTE3_1"></a>


## Opciones del entorno
<a name="opciones_del_entorno_ELTTEXTE000189"></a>
La ventana para configurar el entorno está acostumbrada a:

- en la ficha "Entorno", defina las características disponibles.

- en la pestaña "Paneles", seleccione las ventanas acoplables que desea mostrar.




El botón "Reiniciar el asistente para configurar el entorno" permite reiniciar la configuración que el asistente utilizó al iniciar el producto por primera vez.


### Colores de las barras de herramientas
<a name="colores_las_barras_herramientas_ELTPARAGRAPHE000060"></a>

La Color de las barras de herramientas puede ser configurada. Para hacerlo, haga clic en el pequeño Icon color que se encuentra en la Corner de la ventana del entorno de desarrollo (debajo del logotipo de la versión). Una ventana emergente permite configurar la luminosidad del fondo de las barras de herramientas así como la Color principal.

### "Pestaña"Paneles
<a name="pestanapaneles_ELTPARAGRAPHE000074"></a>

- **Ventanas acoplables**: Seleccione las casillas de verificación correspondientes a los elementos que deben visualizarse en el entorno: paneles, barra de documentos abiertos, Centros [modificador](../Editeurs/2027006.md), [Vista previa de la posición](../Editeurs/2027002.md), control, ... Para obtener más información, consulte [Manipulación de los cristales](../Editeurs/2027001.md).

- **Notas adicionales**: Permite mostrar u ocultar la **barra de pestañas** y establecer el modo [Ojo de pez](../Editeurs/9000067.md):

	- **Nunca**: el modo "Ojo de pez" nunca estará activado.

	- **Cuando el zoom se incluye entre un 25% y un 75%.**: el modo "Ojo de pez" se activará en función del valor del zoom.

	- **Cuando el zoom es inferior al 100%.**: el modo "Ojo de pez" se activará en función del valor del zoom.

	- **Cuando se presionan Ctrl y Mayúsculas**: el modo "Ojo de pez" se activará al pulsar una combinación de teclas específica.

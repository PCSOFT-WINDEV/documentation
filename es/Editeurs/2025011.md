
## Ejecutable: Respaldo del proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000084"></a>
Al final de la creación ejecutable asistente, WINDEV propone hacer automáticamente una copia de seguridad de las fuentes de su proyecto.

Esta opción le permite guardar automáticamente su proyecto cada vez que se implemente una versión principal en los equipos de los usuarios.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000108"></a>


### Etapas a seguir
<a name="etapas_seguir_ELTPARAGRAPHE000020"></a>

Al crear el ejecutable, si selecciona "Sí: Inicie el proyecto de respaldo asistente", la asistente se iniciará automáticamente al final de la creación del ejecutable.

Esta asistente le permite:

1. Seleccione los directorios del proyecto que se incluirán en la copia de seguridad (análisis, exe,...)

2. Seleccione el tipo de copia de seguridad que desea realizar:

	- Archivo de copia de seguridad comprimido (formato zip). En este caso, debe especificarse el nombre del archivo de copia de seguridad.

	- Copia de seguridad de copy. En este caso, se debe especificar el directorio de copia de seguridad.






<a name="NOTE2_2"></a>


### Recordatorio
<a name="recordatorio_ELTPARAGRAPHE000035"></a>

Le aconsejamos que realice una copia de seguridad en un disco que no sea el de trabajo para evitar problemas físicos.

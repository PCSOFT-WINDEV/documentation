
## Android y Java: llamar a los procedimientos de WLanguage
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000079"></a>
Un Procedure global en WLanguage puede ser llamado directamente desde el código de un Procedure global en Java. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000103"></a>
Para llamar a un Procedure WLanguage directamente desde el código de un Procedure global en Java: 

1. Escriba el código de WLanguage Procedure. El WLanguage Procedure debe cumplir con algunas reglas: 

	- El Procedure y el conjunto de Procedure s al que pertenece deben tener un nombre válido en Java. 

	- El tipo de parámetros de Procedure y el tipo de valor de retorno deben ser tipos soportados: booleano, entero, entero de 8 bytes, real, cadena. Le aconsejamos que pase los parámetros por valor a través de la palabra clave "LOCAL"..  

	- El atributo "Java" debe añadirse al lenguaje WLanguage Procedure. 


 Ejemplo: 
	
	```wl
	PROCEDURE nWL_AddOne(n is int), Java: int
	RESULT n+1
	```


2. A continuación, basta con llamar a la Procedure desde el código Java. El nombre de la Procedure debe ir precedido del nombre del conjunto de Procedure s (también se puede añadir la importación para evitar el uso de prefijos).  
	Por ejemplo: 
	
	```txt
	int nReturnInt(int i)
	{
		return MySet.nWL_AddOne(i);
	}
	```

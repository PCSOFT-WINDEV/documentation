## Presentación
<a name="presentacion_ELTTEXTE000107"></a>
Para simplificar la lectura de las ventanas, el usuario final puede ampliar el contenido de una ventana con el ratón..<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD-Zoom%20Fenetre-6.jpg)
**Para acercarse o alejarse en una ventana**, el usuario debe utilizar la tecla Ctrl y la rueda del ratón.

**Para restablecer el nivel de zoom**, basta con seleccionar "zoom Cancel" en el menú contextual de la ventana (o pulsar Ctrl + 0).  

Observación: Si es necesario, pueden aparecer barras de desplazamiento durante el zoom..

## Programación
<a name="programacion_ELTTEXTE000131"></a>


### Programación del zoom
<a name="programacion_del_zoom_ELTPARAGRAPHE000024"></a>

Para programar el zoom que se aplicará a la ventana, utilice [Zoom](../Proprietes/1000017201.md) en la ventana. 

Por ejemplo: 


```wl
// Zoom in 150%
WIN_Customer..Zoom = 150
```
Si el zoom se set mediante programación, el menú contextual de la ventana permite Cancel el zoom. 


### Desactivar el zoom
<a name="desactivar_zoom_ELTPARAGRAPHE000037"></a>

Para desactivar el zoom en una ventana, utilice la función [AAFDisable](../WDLang1/1000022018.md) con la constante *aafZoom*.


### Cancelar el zoom
<a name="cancelar_zoom_ELTPARAGRAPHE000045"></a>

Para Cancel el zoom de una ventana (y restaurar el nivel de zoom inicial), basta con utilizar [AAFExecute](../WDLang1/1000022099.md) con la constante *aafZoom*.


### Modificar la leyenda de la opción utilizada para Cancel el zoom (mediante programación)
<a name="modificar_leyenda_opcion_utilizada_para_cancel_zoom_mediante_programacion_ELTPARAGRAPHE000053"></a>

Para Cancel el zoom, el usuario final puede abrir el menú contextual de la ventana y seleccionar "Cancel zoom". 

Para modificar el texto de esta opción, basta con utilizar [AAFChangeCaption](../WDLang1/1000022100.md) con la constante *aafZoom*.

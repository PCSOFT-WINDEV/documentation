## Estructura del instalador
<a name="estructura_del_instalador_ELTTEXTE000253"></a>
La pestaña "WINDEV framework" sirve para especificar el modo de gestión del instalador framework. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options_avancees_installeur%20-%20HC%20N%B0001.gif)
 Puede especificar las opciones framework instalación: 

- **copy el WINDEV framework con el instalación**. En este caso, el tamaño del programa de instalación puede ser bastante importante. 
	Si se selecciona esta opción: 

	- la pestaña "Personalizar el programa de instalación" permite elegir (si es necesario) el instalador que se utilizará (véase [Personalización del programa de instalación](#NOTE3_1)). 

	- la pestaña "DLL del instalador" se utiliza para ver las DLL que se tienen en cuenta y para añadir o eliminar algunas si es necesario. Esta opción puede ser necesaria cuando se utiliza un instalador personalizado. 




- **Descarga el WINDEV framework desde un servidor FTP**. En este caso, el usuario debe tener acceso a INTERNET. 
	Si se selecciona esta opción, la pestaña "Parámetros FTP" permite definir las opciones del servidor FTP utilizado (véase [Configuración del servidor FTP](#NOTE2_3)). 

- **Utilizar el framework WINDEV que se encuentra en el equipo del usuario**. En este caso, la framework se descargará sólo si el WINDEV framework no se encuentra en el ordenador de la instalación. 




Esta pestaña también se puede utilizar para limpiar las estructuras anteriores. La opción "Borrar los marcos anteriores encontrados en el directorio de destino" se utiliza para borrar todas las versiones anteriores de la WINDEV framework que se encuentran en los directorios de la instalación.
<a name="NOTE2_3"></a>


### Configurar el servidor FTP (modo'Download the WINDEV framework from an FTP server')
<a name="configurar_servidor_ftp_mododownload_the_windev_framework_from_ftp_server_ELTPARAGRAPHE000068"></a>

La pestaña "Parámetros FTP" sirve para: 

- Configurar el servidor FTP utilizado para la descarga de framework

- Traducir los mensajes de la herramienta de descarga. 




**Servidor FTP:**
 
Por Default, el servidor FTP de PC SOFT se utiliza para descargar el WINDEV framework. 

Si se utiliza un FTP privado, tiene la posibilidad de configurar: 

- el Address del servidor FTP,

- puerto de conexión,

- el modo de conexión: conexión pasiva, conexión anónima, ....




**Traducción**: 
El botón "Traducir" sirve para traducir o modificar los mensajes mostrados por la herramienta de descarga de DLLs..

## Personalización del programa de instalación
<a name="personalizacion_del_programa_instalacion_ELTTEXTE000283"></a>
La pestaña "Personalización del instalador" permite seleccionar el instalador que se va a utilizar. 

Para una instalación autónoma, el instalador utilizado por el usuario final puede ser personalizado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options_avancees_installeur%20-%20HC%20N%B0002.gif)


Para una instalación con actualización en red, tienes la posibilidad de personalizar: 

- el instalador utilizado durante el cliente instalación. 

- el instalador utilizado cuando se despliega la versión reference. 




En cualquier caso, usted puede: 

- Utilice el instalador estándar (elección de Default). 
	El instalador estándar es el que se proporciona con WINDEV para instalar las aplicaciones WINDEV. 

- Utilizar un instalador personalizado. 
	El programa estándar de instalación (llamado "WDSetup") se suministra con los ejemplos de WINDEV. Para personalizar el programa de instalación, todo lo que tienes que hacer es modificar el proyecto "WDSetup".
	 Luego, todo lo que tiene que hacer es especificar el directorio del instalador a utilizar. 




**Observación**: Para personalizar el programa instalación (en WINDEV): 

1. Abrir el proyecto "WDSetup. 
	Por Default, este proyecto se encuentra en el subdirectorio "Ejemplos" del directorio WINDEV instalación.
	Observación: Este ejemplo también está disponible en los "Ejemplos completos" que se muestran en la página de inicio.

2. Personalizar este proyecto (modificar el diseño de las ventanas, gestionar idiomas adicionales, ....).

3. Crear un nuevo ejecutable: haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_generer.gif) en los botones de acceso rápido.


**Observación**: El nombre de este ejecutable no importa.

## Gestión del UAC para el instalador
<a name="gestion_del_uac_para_instalador_ELTTEXTE000307"></a>
Por Default, la instalación de una aplicación WINDEV activa la UAC. 

La pestaña "UAC" sirve para modificar este modo de funcionamiento.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options_avancees_installeur%20-%20HC%20N%B0003.gif)


- para las configuraciones individuales: la UAC está desactivada para la instalación en el ordenador del usuario final. 

- para las configuraciones de la red con la actualización de automatic. En este caso, el UAC puede ser desactivado: 

	- para el instalador del usuario final,

	- para el instalador de la versión reference,

	- para ambos. 







Atención: La opción "El instalador NO requiere derechos de administrador" tiene algunas restricciones: no se pueden instalar archivos en los directorios del sistema (Archivos de programa, Windows...), no se pueden modificar los archivos y parámetros del sistema (win.ini, ruta, etc.). 

La instalación fracasará si los derechos no son suficientes. Siempre que sea posible, la instalación asistente guiará al usuario en sus elecciones para que pueda instalar la aplicación. 

Observación: La implementación del UAC para el instalador también se puede configurar en el paso ["Directorio de instalación"](../Editeurs/2028047.md) de la creación de asistente para instalación.

## Plug-in a utilizar
<a name="plugin_utilizar_ELTTEXTE000331"></a>
En un programa de instalación, puede que quieras personalizar algunas acciones, añadir pasos a la asistente, añadir nuevos procesos, etc.

Tiene la posibilidad de crear un plug-in para modificar el programa estándar de instalación. Este plugin se crea a partir del ejemplo "WD WDSETUP Plugin" proporcionado con WINDEV. Para obtener más información, consulte [Personalizar el programa instalación (Plug-in)](../Editeurs/2028109.md). 

La pestaña "Plug-in" se utiliza para especificar la ruta del plug-in que se incluirá en el programa estándar de la instalación.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options_avancees_installeur%20-%20HC%20N%B0004.gif)

## Fuentes a instalar
<a name="fuentes_instalar_ELTTEXTE000355"></a>
En un programa de instalación, es posible que tenga que instalar algunas fuentes de caracteres utilizadas por la aplicación. 
Para ello, añada los archivos TTF u OTF correspondientes a las fuentes a instalar. 

Observación: No olvides comprobar si tienes los derechos para distribuir estas fuentes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Options_avancees_installeur%20-%20HC%20N%B0005.gif)

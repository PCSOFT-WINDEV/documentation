
## Pestañas de documentos abiertos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000081"></a>
Las pestañas de Document permiten navegar entre los distintos elementos abiertos en los diferentes editores. Tanto el código como la UI aparecen en estas pestañas.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_document.gif)


Al pasar el ratón por encima de una pestaña, se abre una vista previa en miniatura.: ventana, reporte, consulta, código.......
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_Document_visu.gif)


Cada Document puede cerrarse pulsando el botón cerrar Icon en la pestaña o en la miniatura de la vista previa.. También puedes pulsar la rueda del ratón: el elemento sobre el que se pasa el ratón se cierra. 

Cada pestaña tiene un menú contextual con varias opciones. La opción "Cerrar todo menos esto" es absolutamente esencial. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barre_document_Fermer.gif)


Si las pestañas desbordan el espacio disponible, se vuelven desplazables: basta con utilizar la rueda del ratón para desplazarse por los diferentes elementos. También puedes arrastrar y soltar pestañas. 

La lista de todos los elementos abiertos también puede mostrarse en un menú. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=barre_document_Menu.gif)


Observación: El menú contextual de la barra de documentos abiertos (no de una pestaña individual) permite abrir elementos cerrados recientemente (opción "Cerrado recientemente"). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Recent_Ouvert.gif)

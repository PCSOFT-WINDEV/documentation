


## WDInst: opciones avanzadas para la red instalación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000295"></a>
Las opciones avanzadas de la red instalación se utilizan para..:

- configurar las opciones para el control remoto

- modificar las condiciones de aplicación de la actualización

- mostrar la ayuda sobre las nuevas características al usuario durante la red instalación.

- reference la aplicación en los centros de control. 




Por Default:

- no se muestra ayuda para las nuevas funciones.

- las opciones para el control remoto son las siguientes:

	- tiempo interval entre cada control remoto: 5 minutos.

	- mensaje para prohibir la conexión: "Esta aplicación fue bloqueada por el administrador para una actualización y no está disponible en este momento."

	- mensaje de cierre inminente: "Esta aplicación se detendrá para su actualización. Guarde sus modificaciones y salga de la aplicación".

	- mensaje de cierre forzado: "PRECAUCIÓN: La aplicación se detendrá para actualizarla".

	- mensaje para reanudar la instalación: "La actualización anterior de la aplicación no se terminó correctamente y la aplicación no puede ser ejecutada. ¿Quiere realizar la actualización de nuevo?".




- la actualización se instala automáticamente si el usuario no realiza ninguna operación durante 60 segundos.

## Configurando las opciones avanzadas de la red instalación
<a name="configurando_las_opciones_avanzadas_red_instalacion_ELTTEXTE000319"></a>
Las opciones avanzadas de la red instalación están configuradas en WDInst: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

Estos ajustes también pueden ser definidos a través de la creación del programa instalación asistente.

**Para acceder a las opciones avanzadas de la red instalación**:

1. Seleccione "Parámetros de configuración ... Asistente".

2. Seleccione la opción "Instalación de red con actualización automática". Los parámetros de la red instalación aparecen. 

3. Haga clic en el botón "Opciones avanzadas".

## Configuración de las opciones para la comprobación remota
<a name="configuracion_las_opciones_para_comprobacion_remota_ELTTEXTE000343"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000066"></a>

El control remoto de una aplicación WINDEV permite que el administrador de una aplicación de red gestione el cierre de la aplicación automatic para todos los usuarios actuales.

Para obtener más información, consulte [Principio de control remoto](../WDLang1/3013006.md).

**Observación**: La configuración de las opciones de control puede ser modificada por el gestor de la aplicación desde el [programa para la comprobación remota de los ejecutables de WINDEV](../WDAdminExe/2028052.md) (WDAMINEXE) instalado en el servidor.
<a name="NOTE3_2"></a>
Las opciones de control remoto se configuran en la pestaña "Parámetros control" de las opciones avanzadas de asistente. 

**Configurar las opciones para la comprobación remota:**


1. Especifica el tiempo interval (en minutos) entre cada cheque.

2. Introduzca los diferentes mensajes que aparecen en el ordenador del usuario final: 

	- el mensaje utilizado para prohibir la conexión. 
			Este mensaje se mostrará si los usuarios intentan conectarse a la aplicación mientras la opción "Conexión prohibida" está seleccionada en "WDADMINEXE".

	- el mensaje de cierre inminente.
			Este mensaje aparecerá si los usuarios están conectados a la aplicación y si la opción "Aviso de parada inminente" está seleccionada en "WDADMINEXE".

	- el mensaje para el apagado forzado.
			Este mensaje aparecerá si los usuarios están conectados a la aplicación y si la opción "Parada forzada" está seleccionada en "WDADMINEXE".

	- el mensaje para reanudar instalación.
			Este mensaje aparecerá si los usuarios intentan volver a conectarse a la aplicación mientras que la instalación de la última actualización ha fallado.







Observación: Si no se introduce ningún mensaje, aparecerá el siguiente mensaje: "La actualización anterior de la aplicación no finalizó correctamente y la aplicación no se puede ejecutar. ¿Desea reiniciar la actualización?".
<a name="NOTE3_3"></a>


### Importante: cambiar las opciones de control en WDADMINEXE
<a name="importante_cambiar_las_opciones_control_wdadminexe_ELTPARAGRAPHE000109"></a>

Si las opciones de control remoto se modifican directamente en el programa de administración "WDADMINEXE", los nuevos valores set en el administrador son conservados por Default. El enlace "Restaurar valores de Default" en WDADMINEXE permite restaurar los valores especificados en el último instalación.

## Modificación de las condiciones de aplicación de la actualización
<a name="modificacion_las_condiciones_aplicacion_actualizacion_ELTTEXTE000379"></a>
Para informar a los usuarios de que una actualización de la aplicación está disponible, se muestra la siguiente ventana cuando se inicia la aplicación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenMiseAJour.gif)
 Esta ventana permite instalar o no la actualización.

Las diferentes opciones para la validación de automatic son las siguientes:

- ***No validar las actualizaciones automáticamente***. 
	El usuario deberá validar la ventana.
	Si la actualización es obligatoria, el usuario deberá instalarla antes de poder utilizar la aplicación.. 
	Si la actualización es opcional, el usuario podrá instalar la actualización o iniciar la aplicación.

- ***Apply la actualización automáticamente si se propone*** (opción Default). 
	Un clic en el botón "Actualizar" se realizará al final del tiempo de espera para la "validación de automatic".
	Durante este tiempo de espera, el usuario tiene la posibilidad de Cancel la actualización o de iniciar la aplicación si la actualización no es obligatoria.

- ***Aplicar la actualización sólo si es obligatoria***. 
	Si la actualización es obligatoria, se hará un clic en el botón "Actualizar" al final del tiempo de espera para la "validación de automatic". Durante este tiempo de espera, el usuario tiene la posibilidad de Cancel la actualización. 
	Si la actualización es opcional, se hará un clic en el botón "Ignorar" al final del tiempo de espera para la "validación de automatic". Durante este tiempo de espera, el usuario tiene la posibilidad de instalar la actualización.

- ***Cancelar el inicio de la aplicación si es obligatorio realizar una actualización***. 
	Al finalizar el tiempo de espera para la "validación de automatic", se hará un clic en el botón "Cancelar"..
	Durante este tiempo de espera, el usuario tiene la posibilidad de instalar la actualización o de iniciar la aplicación.




**Observación**: Independientemente del idioma de ejecución del programa instalación, esta ventana será mostrada en francés por Default. Para mostrar esta ventana en un idioma que no sea el francés, debe traducir el proyecto WDSetup.
<a name="NOTE4_2"></a>
Las condiciones para aplicar la actualización se configuran en la pestaña "Actualización de automatic" de las opciones avanzadas de la asistente. 

**Para modificar las condiciones de aplicación de la actualización**: 

1. Seleccione la opción para la validación de automatic (ver Above).

2. Especificar el tiempo de espera (en segundos) después del cual la ventana de información será validada o cerrada (60 segundos por Default). 
	Dependiendo de la opción de "actualización de automatic":

	- la actualización se instalará automáticamente.

	- la aplicación se iniciará sin modificaciones.

	- se cancelará el inicio de la aplicación.







Este tiempo de espera se ignora si la opción "No validar las actualizaciones automáticamente" está marcada.

El botón "Parámetros para comprobar la presencia de actualizaciones" permite configurar: 

- **la frecuencia con la que se comprueba la disponibilidad de las actualizaciones**. 
	Por Default, la disponibilidad de una actualización en el servidor se comprueba cada vez que se inicia la aplicación. Esta frecuencia de comprobación se puede configurar. Se puede realizar: 

	- Cada vez que se inicia la aplicación. 

	- Cada X días. 




- **las interacciones con el usuario al comprobar la disponibilidad de una actualización**. 
	Cada vez que una aplicación es lanzada (por defecto), una aplicación instalada por medio de una "instalación red" se verifica si se encuentra una actualización disponible en el servidor de referencia. 

	- Si una versión más reciente está disponible en el servidor, la actualización de la aplicación se propone al usuario final.. 

	- Si no se encuentra la conexión de red al abrir la aplicación, se muestra un mensaje de advertencia por parte de Default. 
			Este mensaje puede ser inútil para una aplicación utilizada en un dispositivo móvil, conectada de forma intermitente.. En este caso, simplemente seleccione "Comprobación silenciosa".: No informar al usuario". 







Tiene la capacidad de permitir que estas opciones se modifiquen durante la instalación de la versión de la reference.

## Visualización de la ayuda sobre las nuevas funciones
<a name="visualizacion_ayuda_sobre_las_nuevas_funciones_ELTTEXTE000409"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000198"></a>

La ayuda sobre las nuevas funcionalidades se utiliza para indicar al usuario todas las modificaciones realizadas en cada actualización de la aplicación.

Se mostrará esta ayuda:

- cuando la actualización está instalada en los equipos del usuario (botón"Detalles").

- en la pestaña "Configuración" de [programa para la comprobación remota de los ejecutables de WINDEV](../WDAdminExe/2028052.md) (WDADMINEXE).




Las nuevas características de la aplicación pueden:

- en la ventana de opciones avanzadas de la red instalación.

- se guardará en un archivo en formato texto o RTF.

- corresponden al archivo de ayuda generado automáticamente durante la creación del ejecutable. Para obtener más información, consulte [Ayuda sobre las nuevas funcionalidades](../Editeurs/2028084.md). 




La ayuda sobre las nuevas características se configura en la pestaña "Ayuda para las nuevas características" de las opciones avanzadas de la asistente. 
<a name="NOTE5_2"></a>


### Introducción directa de las nuevas funciones
<a name="introduccion_directa_las_nuevas_funciones_ELTPARAGRAPHE000224"></a>

Introducir directamente las nuevas características (las últimas modificaciones realizadas por ejemplo) en la ventana de opciones avanzadas de la red instalación:

1. Marque "Texto".

2. Introduzca el texto deseado.

3. Para insertar automáticamente:

	- el número de versión actual, haga clic en el botón "Versión".

	- fecha de hoy, haga clic en el botón "Fecha".






<a name="NOTE5_3"></a>


### Recuperación del historial guardado en un archivo
<a name="recuperacion_del_historial_guardado_archivo_ELTPARAGRAPHE000238"></a>

Para recuperar el historial guardado en un archivo:

1. Marque "Archivo".

2. Seleccionar el archivo que contiene el historial ("Path of history file"). Este archivo puede estar en formato de texto (".TXT") o en formato RTF (".RTF" o ".WRI" por ejemplo).




**Observación**: Para ver directamente el archivo seleccionado, haga clic en el botón "Vista previa. Se mostrará el archivo:

- en el Bloc de notas (archivo en formato texto).

- en Word (archivo en formato RTF).

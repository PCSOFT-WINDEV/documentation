


## Importar una ventana WINDEV a un sitio WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
Las ventanas Windows desarrolladas con WINDEV (versión 7.5 y posteriores) pueden ser importadas a un sitio WEBDEV.. Las ventanas se convertirán automáticamente en páginas WEBDEV. Se recuperan los controles y el código de la ventana.

La importación de ventanas WINDEV permite ahorrar mucho tiempo al migrar una aplicación Windows a un sitio Internet o INTRANET.

**Atención**: Las páginas resultantes de la importación pueden tener un aspecto un poco diferente de las ventanas iniciales.. A veces, su interfaz y su código deben ser revisados.

**Observaciones**: 

- También tienes la capacidad de copy la ventana controla en una página directamente. Para obtener más información, consulte [copy entre los editores](../Editeurs/9000137.md).

- A partir de la versión 24, la webificación asistente permite transformar una aplicación WINDEV en un sitio web. Para obtener más información, consulte [Transforme sus aplicaciones WINDEV existentes en sitios web](../Editeurs/9500225.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000137"></a>


### Importar una ventana WINDEV a un sitio WEBDEV
<a name="importar_una_ventana_windev_sitio_webdev_ELTPARAGRAPHE000037"></a>

Para importar una ventana WINDEV a un sitio WEBDEV:

1. Abra el proyecto WEBDEV en el que debe importarse la ventana WINDEV.

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV  o WINDEV Mobile".

3. Seleccione la ventana WINDEV para importar (archivo".WDW"). La ventana WINDEV se convierte automáticamente en una página WEBDEV.



<a name="NOTE2_2"></a>


### Importación de los elementos WINDEV en la página WEBDEV
<a name="importacion_los_elementos_windev_pagina_webdev_ELTPARAGRAPHE000060"></a>

Los siguientes elementos no se importan a la página WEBDEV creada:

- Controles WINDEV que no existen en WEBDEV (Control deslizante, control Barra de progreso, etc.).

- Funciones de WLanguage, propiedades, palabras clave no disponibles en WEBDEV (función **OpenWindow**, palabra clave **MyWindow**, etc.).

- Plantillas de WINDEV tema. Si la ventana importada es associated con un plantilla, sólo se importarán las imágenes associated con la página y sus controles.

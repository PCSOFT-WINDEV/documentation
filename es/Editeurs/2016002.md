<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000088"></a>
Esta ventana le permite describir el estilo WEBDEV que se utilizará para los controles de su proyecto WEBDEV.

Esta ventana se utiliza para definir los estilos CSS y las opciones adicionales que definen el estilo WEBDEV.

Esta página de ayuda presenta las diferentes opciones que se encuentran en la pestaña "General" y en la pestaña "Estilo".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pestaña General
<a name="pestana_general_ELTTEXTE000112"></a>
Esta pestaña se utiliza para definir el nombre del estilo.

Para un estilo associated con un control Botón, también puede especificar:

- La aparición del botón (Image fijo o Image con rollover, ...).

- Las imágenes associated con el botón.

- Las dimensiones del botón.

<a name="NOTE3_1"></a>


## Ficha Estilo
<a name="ficha_estilo_ELTTEXTE000136"></a>
Esta pestaña se utiliza para definir los diferentes estilos de CSS utilizados por el estilo WEBDEV. Para obtener más información, consulte [Estilos CSS](../Editeurs/2016004.md).




## Editor de reportes: opciones y métodos abreviados de teclado
			



<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000245"></a>
Los editores de reporte proponen un menú en forma de cinta que agrupa todas sus opciones. Los siguientes paneles de cinta son específicos de la editor de reportes: 

- Creación: se utiliza para crear todos los controles que se encuentran en los informes. 

- Modificación: para modificar las características de la reporte. 

- Alineación: para gestionar las opciones de alineación de los elementos que se encuentran en el reporte. 

- Mostrar: se utiliza para especificar las opciones de visualización de los controles reporte y reporte en el editor..

## Métodos abreviados de teclado
<a name="metodos_abreviados_teclado_ELTTEXTE000269"></a>
Los principales atajos que se pueden utilizar en el editor de reportes son los siguientes:

| Acción | Acceso directo |
| --- | --- |
| Habilitar/deshabilitar los datos en vivo | F8 |
| Mostrar la description de la control o de la celda | F4 |
| Mostrar la lista de controles que se encuentran en el reporte actual | F12 |
| Mostrar la lista de clases en el panel "Explorador del proyecto" (y en el panel "Código" si se muestra). | Mayúsculas + F8 |
| Mostrar la lista de procedimientos globales en el panel "Explorador de proyectos" (y en el panel "Código" si se muestra). | Ctrl + F8 |
| Visualizar el nombre completo de los bloques / Visualizar el nombre corto de los bloques | F10 |
| Mostrar/Ocultar el borde de los controles | F5 |
| Mostrar/Ocultar el nombre de los controles | F6 |
| Mostrar/Ocultar los márgenes de reporte | F11 |
| Mostrar/Ocultar reglas | Ctrl + R |
| Ayuda | F1 |
| Deshacer la última acción | Ctrl + Z |
| Detener el modo de prueba (durante la prueba del proyecto o reporte) | Mayúsculas + F5 |
| Pegar contenido del portapapeles | Ctrl + V o Mayúsculas + Entradas |
| copy el estilo de elemento seleccionado | Ctrl + Mayúsculas + C |
| copy el elemento seleccionado | Ctrl + C o Ctrl + Ins |
| Cortar el elemento seleccionado | Ctrl + X o Alt + Del |
| Mueva la control o columna seleccionada a la derecha | Flecha hacia la derecha |
| Mueva la control o columna seleccionada a la izquierda | Flecha hacia la izquierda |
| Visualizar el código associated con el elemento seleccionado en el panel "Vista de código sensible al contexto". | Ctrl + Mayúsculas + F2 |
| Editar el código de selección | F2 |
| Grabe la reporte | Ctrl + S |
| Guardar como | Ctrl + Mayúsculas + S |
| Guardar todo | Alt + S |
| Cerrar todos los elementos abiertos | Ctrl + Mayúsculas + W |
| Imprimir la documentación de la reporte actual | Ctrl + P |
| Ocultar los cristales horizontales | Ctrl + Q |
| Ocultar todo (cristales acoplables) | Ctrl + W |
| Asigne el atributo de negrita a la control seleccionada | Ctrl + G |
| Asigne el atributo de la cursiva al control seleccionado | Ctrl + I |
| Nuevo (Crear un elemento) | Ctrl + N |
| Open (Abrir un elemento) | Ctrl + O |
| Abrir un proyecto | Ctrl + Mayúsculas + O |
| Restaurar (Deshacer la última acción) | Ctrl + Y o Ctrl + Mayúsculas + Z |
| Subraye el control seleccionado | Ctrl + U |
| Borrar la selección | Supr |
| Rastrear el proyecto | Alt + F9 |
| Probar proyecto | Ctrl + F9 |
| Ejecutar la prueba de la reporte actual | F9 |
| Mostrar reporte en 3D | Alt + D |
| Zoom en 100 % | Ctrl + Alt + 0 |
| Zoom +/- | Ctrl + rueda del ratón |

Atención: El signo "+" significa que las teclas deben pulsarse simultáneamente

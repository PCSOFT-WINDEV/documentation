
## Probar un sitio WEBDEV en un dispositivo móvil
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
Cada vez es más habitual que los desarrolladores prueben los sitios web en dispositivos móviles. WEBDEV permite probar proyectos y páginas directamente en un dispositivo móvil. 

Esto permite garantizar que las páginas son compatibles con dispositivos móviles.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000137"></a>


### Condición previa
<a name="condicion_previa_ELTPARAGRAPHE000018"></a>

- El dispositivo móvil y el ordenador de desarrollo deben estar conectados a la misma red (Wi-Fi, por ejemplo).

- El servidor web debe permitir varias conexiones simultáneas. Este modo de prueba no está disponible si utiliza el servidor IIS Express integrado.. 



<a name="NOTE2_2"></a>


### Ejecución de la prueba
<a name="ejecucion_prueba_ELTPARAGRAPHE000026"></a>

Tanto los proyectos como las páginas pueden probarse en dispositivos móviles.. 

**Para probar un proyecto en un dispositivo móvil**: 

1. Desplegar ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_GO_Projet_WB_bl.gif) Icon en los botones de acceso rápido. 

2. Seleccione "Probar navegador .. Phone (a través de un código &QR)"

3. Ejecutar la prueba del proyecto haciendo clic en el "GO" Icon.

4. Aparece una ventana con un código QR. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Agencement_simple_1%20-%20HC%20N%B0016.jpg)


5. Escanear el código QR con el dispositivo. 

6. La primera Page del proyecto aparece en el dispositivo. 




**Para probar WEBDEV Page que se muestra en el editor en un dispositivo móvil**: 

1. Desplegar ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Go_Page_bl.gif) Icon en los botones de acceso rápido. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GO_PAGE_WB_deroule.gif)


2. Seleccione "Prueba desde una Phone (acceso mediante un código QR)".. 

3. Aparece una ventana con un código QR.   <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P8_Agencement_simple_1%20-%20HC%20N%B0016.jpg)


4. Escanear el código QR con el dispositivo. 

5. La Page actual se muestra en el aparato. 


Observación: Las pruebas posteriores se ejecutarán en el dispositivo (a través del código QR) hasta que seleccione un modo de prueba diferente. 




### Observaciones
<a name="observaciones_ELTPARAGRAPHE000058"></a>

- Los puntos de interrupción (código del servidor) son Active en el dispositivo. 

- Puede establecer puntos de interrupción después de iniciar la prueba. 

- El modo de depuración estándar está disponible (ejecución paso a paso, posibilidad de ver variables, etc.). 

- Puede cambiar el navegador de prueba de Default en las opciones de WEBDEV. Para obtener más información, consulte [Opciones generales de WINDEV, WEBDEV y WINDEV Mobile](../Editeurs/2036001.md).




## Configuración del entorno
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000216"></a>
Una configuración de entorno corresponde a una copia de seguridad del entorno de trabajo en los editores de WINDEV, WEBDEV y WINDEV Mobile.: se guardan los diferentes paneles, paletas y barras de herramientas (así como su posición). Esta configuración puede ser restaurada en cualquier momento. Se pueden almacenar hasta 4 configuraciones por usuario/desarrollador.

**Consejo**: Las configuraciones del entorno pronto se vuelven indispensables cuando se utilizan varias pantallas. Los elementos que se encuentran en la interfaz de WINDEV, WEBDEV y WINDEV Mobile se pueden posicionar en cualquier pantalla.. La posición de cada elemento en las diferentes pantallas se guarda y restaura automáticamente.

**Observación**: Para evitar una modificación inesperada del entorno de trabajo, usted tiene la capacidad de bloquear el entorno..

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000240"></a>


### Guardar la configuración actual
<a name="guardar_configuracion_actual_ELTPARAGRAPHE000024"></a>

La configuración de trabajo actual puede guardarse en cualquier momento en una de las cuatro configuraciones disponibles:

1. Utilice uno de los siguientes métodos:

	- utilice el atajo Ctrl + Alt + Número de configuración (1 a 4).

	- en la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Panes" y luego "Guardar entorno ... Guardar en ".




2. En la ventana que se muestra, especifique el nombre de su configuración. Esta configuración se puede visualizar en cualquier momento, pulsando Ctrl + Número de configuración (1 a 4)

3. Validar. Se muestra el nombre de la configuración actual.




**Observación**: Las configuraciones del entorno se almacenan en un archivo .INI que se encuentra en el directorio "\\Personal" del directorio instalación del producto utilizado. Este archivo tiene el siguiente formato: "ConfigVoletEnv_&lt;Usuario&gt;.INI" donde &lt;Usuario&gt; es reemplazado por el nombre de usuario.
<a name="NOTE2_2"></a>


### Visualización de una configuración existente
<a name="visualizacion_una_configuracion_existente_ELTPARAGRAPHE000052"></a>

Una configuración existente se puede visualizar en cualquier momento:

1. Utilice uno de los siguientes métodos:

	- utilice el acceso directo Ctrl + Número de configuración (1 a 4).

	- en la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Guardar entorno .. &lt;nombre de la configuración&gt;".




2. Se muestra la configuración de entorno deseada.



<a name="NOTE2_3"></a>


### Exportación de una configuración
<a name="exportacion_una_configuracion_ELTPARAGRAPHE000077"></a>

Las configuraciones del entorno associated con un usuario pueden ser exportadas. Estas configuraciones pueden ser fácilmente importadas (por otro usuario o en otro producto, por ejemplo).

Para exportar las configuraciones:

1. En la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Guardar entorno... Configuración".

2. Haga clic en el botón "Exportar".

3. Introduzca la ubicación y el nombre del archivo de configuración.

4. Validar. Se crea el archivo INI que contiene todas las configuraciones actuales del entorno.



<a name="NOTE2_4"></a>


### Importar una configuración
<a name="importar_una_configuracion_ELTPARAGRAPHE000103"></a>

La importación de una o más configuraciones permite armonizar las interfaces de trabajo para WINDEV, WEBDEV y WINDEV Mobile.. Sus configuraciones pueden ser creadas en uno de estos tres productos, luego pueden ser exportadas e importadas a los otros dos productos.: las barras de herramientas y los paneles comunes a los diferentes productos se mostrarán de la misma manera.

Para importar una configuración:

1. En la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Guardar entorno... Configuración".

2. Haga clic en el botón "Importar".

3. Seleccione el archivo INI que contiene las configuraciones que desea importar.

4. Validar. En la ventana que se muestra, especifique la posición de cada configuración a importar en las configuraciones actuales.

5. Validar: las configuraciones se importan automáticamente al producto actual. Se pueden usar de inmediato.



<a name="NOTE2_5"></a>


### Borrar una configuración
<a name="borrar_una_configuracion_ELTPARAGRAPHE000130"></a>

Para borrar una configuración:

1. En la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Guardar entorno... Configuración".

2. Seleccione la configuración que desea eliminar.

3. Haga clic en el botón "Eliminar".



<a name="NOTE2_6"></a>


### Bloqueo del entorno
<a name="bloqueo_del_entorno_ELTPARAGRAPHE000155"></a>

Para bloquear el entorno, vaya a la pestaña "Inicio", al grupo "Entorno", expanda "Paneles", seleccione "Paneles", y luego seleccione "Bloquear entorno". Los paneles y las barras de herramientas ya no se pueden mover ni redimensionar.

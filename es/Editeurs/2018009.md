## Probar la aplicación
<a name="probar_aplicacion_ELTTEXTE000133"></a>
Durante la prueba de una aplicación que soporta el groupware usuario, la primera ventana o página que se muestra es la ventana o página de inicio de sesión (independientemente de la primera ventana o página definida en su aplicación).

De forma predeterminada se crea un solo usuario: el supervisor. Para iniciar sesión como supervisor, introduzca la siguiente información en la ventana o página de inicio de sesión:

- denominar: SUPERVISOR

- Contraseña: SUPERVISOR




Puede probar la aplicación, o configurar el Groupware Usuario.

**Observaciones**:

- Si no quiere que set la primera ventana/página de su aplicación como ventana/página de inicio de sesión, seleccione "Inicio manual" en el [Opciones de grupos de usuarios](../Editeurs/2018004.md). Utilice la función [gpwOpen](../WDLang6/3041002.md) para abrir la ventana de inicio de sesión.

- Las ventanas o páginas de gestión de el groupware usuario se ejecutan antes del código de inicialización del proyecto.

- La primera ventana o página de la aplicación definida en el proyecto, se iniciará después de la ventana o página de inicio de sesión (cuando el usuario no es el supervisor).

- Para evitar iniciar el groupware usuario cuando se ejecutan las pruebas de aplicación, desmarque "Taller .. Configurar el groupware usuario". Si el groupware usuario se habilita de nuevo más tarde, los archivos de datos utilizados anteriormente por el groupware usuario no se borrarán.

## Creación del ejecutable
<a name="creacion_del_ejecutable_ELTTEXTE000157"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) El ejecutable de una aplicación que soporta el User Groupware es un ejecutable multi-instancia.

Al crear el ejecutable que gestiona el groupware usuario ("Workshop .. Executable .. Generate the Windows executable"), se muestra una pantalla específica de el groupware usuario en los diferentes pasos de creación del ejecutable.

**Permite definir la ubicación de los archivos comunes** del User Groupware (archivos que contienen la información sobre los usuarios). Estos archivos pueden ser comunes a varias aplicaciones. En este caso, debe especificar el ruta de los ficheros de datos a utilizar. 
Por Default, este directorio corresponde a un subdirectorio del directorio de los ejecutables (EXE\\GPW_&lt;Nombre del proyecto&gt;_ directorio GPW).

Observación: La ruta de los archivos de datos comunes también se puede especificar al instalar la aplicación.

**Para obtener más información, consulte [Los archivos de datos de groupware usuario](../Editeurs/2018002.md).**

## Instalar una aplicación
<a name="instalar_una_aplicacion_ELTTEXTE000181"></a>
Para instalar una aplicación que utilice el programa de grupo de usuarios, seleccione "Taller .. Crear instalación Procedure".

En la creación de instalación asistente, se puede personalizar el ruta de los archivos de groupware ("Personalizar el ruta de los archivos de groupware durante instalación" en "Parámetros de la base de datos").

En este caso, cuando se instala la aplicación, aparece una pantalla específica para el Grupo de Usuarios. Esta pantalla permite a la persona que instala la aplicación:

- configurar la ubicación de los archivos de datos específicos del User Groupware. Para obtener más información, consulte [Los archivos de el groupware usuario](../Editeurs/2018002.md).

- activar historial de conexiones.

- habilitar la autenticación de usuarios a través de un directorio LDAP. En este caso, tendrás que introducir la Address del servidor, el número de puerto y la versión LDAP utilizada.




Observaciones:

- Si se han configurado archivos de datos del Groupware Usuario para un cliente, deben seleccionarse al preparar la instalación. Por lo tanto, es necesario personalizar la lista de archivos de la instalación.

- Si no se han configurado los archivos de datos del groupware, el Supervisor será el único usuario en la aplicación la primera vez que se inicie.




**Consejo**: Si no configura los diferentes niveles de uso de groupware usuario para sus clientes, le recomendamos que les proporcione un Document con la lista de todos los controles que se encuentran en sus ventanas o páginas con el fin de get una configuración optimizada.


## Copiar controles desde el editor de ventanas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000606"></a>
Los controles se pueden copiar desde el editor de ventanas: 

- al editor de páginas

- a la editor de reportes.

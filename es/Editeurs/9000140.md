
## Copiar controles de la editor de reportes
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000206"></a>
Los controles pueden copiarse desde el editor de reportes: 

- al editor de ventanas.

- al editor de páginas. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Conversión de la editor de reportes
<a name="conversion_editor_reportes_ELTTEXTE000230"></a>


### copy de la editor de reportes al editor de la ventana
<a name="copy_editor_reportes_editor_ventana_ELTPARAGRAPHE000020"></a>

Los siguientes controles son apoyados durante este copy: 

| control encontrada en el editor de reportes | control copió en el editor de la ventana |
| --- | --- |
| Estática | pulsador |
| Campo | Estática |
| Image | Image |
| Código de barras | Código de barras |
| RTF | Estática |
| HTML | Estática |
| Preestablecido | Estática |
| Gráfico | Gráfico |
| Casilla de verificación | Casilla de verificación |
| vincular | pulsador |
| corregir | corregir |
| Planificador | Planificador |
| Cálculo | NA |
| Forma | NA |
| Reporte interna | NA |

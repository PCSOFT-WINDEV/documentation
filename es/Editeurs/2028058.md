## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
El programa de creación de instalación asistente analiza y selecciona automáticamente los archivos que se instalarán con el sitio.

Durante una instalación por FTP, pulse el botón "Editar la lista de ficheros" para ver la lista de ficheros tenidos en cuenta. 

Al crear un paquete implementación o la instalación por medios físicos, la lista de archivos se propone automáticamente. 

Observación: Durante una instalación o actualización por FTP, tiene la posibilidad de comparar los archivos instalados en el servidor con los que se encuentran en la computadora de desarrollo.

## Lista de archivos a implementar
<a name="lista_archivos_implementar_ELTTEXTE000161"></a>


### Lista de archivos a instalar (instalación en un servidor de aplicaciones remoto)
<a name="lista_archivos_instalar_instalacion_servidor_aplicaciones_remoto_ELTPARAGRAPHE000024"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fichier_install_New.gif)
Por Default, la asistente selecciona todos los elementos que requiere el sitio dinámico de WEBDEV para funcionar:

- bibliotecas (archivos ".WDL")

- imágenes y páginas HTML estáticas (directorio _WEB del proyecto)

- hojas de estilo (archivos CSS)

- páginas HTML dinámicas (contenido de directorios para idiomas de proyecto)

- ...




Para cada archivo (o grupo de archivos), tienes la capacidad de especificar a través de el menú contextual si el archivo (o el grupo de archivos) debe ser: 

- actualizado. 

- nunca se modifica en el servidor (opción "Ninguna acción"). No se realizará ninguna acción sobre los archivos seleccionados. 

- borrado del servidor.

- nunca se puso al día sobre el servidor.


También tiene la posibilidad de pulsar los botones correspondientes ("Actualizar", "Ninguna acción", "Excluir de las actualizaciones" y "Eliminar").

El botón "Comparar" sirve para comparar el fichero local y el remoto: tamaño del archivo, fecha y hora de modificación.

**Observaciones**: 

- Para una actualización, sólo los nuevos archivos (o los archivos más recientes que los instalados anteriormente) están marcados para implementación.

- Usted tiene la capacidad de filtrar los archivos mostrados en esta ventana: archivos para actualizar solamente, archivos para borrar, excluir o sin ninguna acción que realizar.




**Caso de archivos de datos**:

- Durante una primera instalación, los archivos de datos son propuestos automáticamente por Default.

- Durante una actualización, los archivos de datos son propuestos por Default sólo si no existen en el servidor (nuevos archivos de datos por ejemplo).

- Si los archivos de datos se añaden manualmente a los archivos a instalar (botón "Añadir archivos adicionales"), estos archivos se incluirán automáticamente en la actualización cada vez que se realice una actualización. Sólo se instalarán si estos archivos son más recientes que los existentes.

- Para evitar la instalación de los archivos de datos, selecciónelos y haga clic en el botón "Excluir definitivamente los archivos seleccionados".




Durante un implementación en varios servidores simultáneamente, tienes la posibilidad de seleccionar los archivos a desplegar en cada servidor. El directorio de destino de cada archivo es común a todos los servidores.

### Seleccionando los archivos a instalar (paquete implementación o medios físicos)
<a name="seleccionando_los_archivos_instalar_paquete_implementacion_medios_fisicos_ELTPARAGRAPHE000065"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fichier_install_pack.gif)
Por Default, la asistente selecciona todos los elementos que requiere el sitio dinámico de WEBDEV para funcionar:

- bibliotecas (archivos ".WDL")

- imágenes y páginas HTML estáticas (directorio _WEB del proyecto)

- hojas de estilo (archivos CSS)

- páginas HTML dinámicas (contenido de directorios para idiomas de proyecto)

- ...




Esta lista corresponde a la lista de archivos que se instalarán. Es posible:

- eliminar archivos de esta lista (botón "Excluir los archivos"). Los archivos excluidos ya no se propondrán durante la próxima actualización del sitio.. 

- añadir archivos a esta lista (botón "Añadir archivos"). Estos archivos se propondrán automáticamente durante la actualización del sitio.

- modificar el directorio implementación. Todo lo que tienes que hacer es modificar el directorio de destino en el tabla. 




La asistente también puede utilizarse para desplegar el sitio en todos los idiomas (por Default) o sólo para algunos idiomas. Si se elige esta opción, un popup permite seleccionar los idiomas deseados.

**Caso de archivos de datos**:

- Por Default , asistente proporciona los archivos de datos.

- Si los archivos de datos se añaden manualmente a los archivos a instalar (botón "Añadir archivos"), estos archivos se incluirán automáticamente en la actualización cada vez que se realice una actualización.. Sólo se instalarán si estos archivos son más recientes que los existentes.

- Si no desea instalarlos, sólo tiene que seleccionarlos y hacer clic en "Excluir archivos".

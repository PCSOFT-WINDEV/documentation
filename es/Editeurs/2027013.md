<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
El panel "Buscar - Reemplazar" está acostumbrado a:

- Encontrar una expresión específica en todo el proyecto.

- Reemplazar las expresiones encontradas por otra expresión. 

- Encontrar una expresión específica en varios proyectos.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Rechercher_Volet1.gif&type=thumb)
Para obtener más información, consulte [Buscar en los editores](../Editeurs/9000150.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualización del panel "Buscar - Reemplazar
<a name="visualizacion_del_panel_buscar_reemplazar_ELTTEXTE000120"></a>
**Para mostrar el panel "Buscar - Reemplazar "**, vaya a la pestaña "Inicio", grupo "Entorno", expanda "Paneles", seleccione "Paneles", y luego seleccione "Buscar - Reemplazar".

Para más detalles sobre los paneles acoplables, véase [Manejo de paneles acoplables](../Editeurs/2027001.md). 

Observación: El panel "Buscar - Reemplazar" puede colocarse horizontal o verticalmente.

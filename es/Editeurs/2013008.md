


## Marcas de posicionamiento en el código
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000140"></a>
Las marcas de posicionamiento se utilizan para "etiquetar" líneas de código específicas. Estas marcas se utilizan para navegar rápidamente por el código de una marca de posicionamiento a otra. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000164"></a>


### Manipulación de las marcas de posicionamiento
<a name="manipulacion_las_marcas_posicionamiento_ELTPARAGRAPHE000018"></a>

Las siguientes opciones se utilizan para manipular las marcas de posicionamiento:

- **Añadir o eliminar una marca de posicionamiento en un Line de código**: en la pestaña "Código", en el grupo "Marcadores", amplíe "Marcador", o pulse Ctrl + F7. 

- **Ir a la siguiente marca**: en la pestaña "Código", en el grupo "Marcadores", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_suivante.gif) o pulse F7. 

- **Ir a la marca anterior**: en la pestaña "Código", en el grupo "Marcadores", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_precedente.gif), o pulse Shift + F7. 

- **Borrar todas las marcas de código**: en la pestaña "Código", en el grupo "Marcadores", haga clic en "Marcador" y seleccione "Eliminar todas las marcas".



<a name="NOTE2_2"></a>


### Posicionamiento de marcas en un resultado de búsqueda
<a name="posicionamiento_marcas_resultado_busqueda_ELTPARAGRAPHE000074"></a>

Para posicionar marcas en el resultado de una búsqueda:

1. Visualiza la pantalla de ["Panel"Buscar - Reemplazar](../Editeurs/2027013.md).

2. En la lista "Buscar", introduzca la expresión a buscar.

3. Especifique los elementos en los que se debe realizar la búsqueda (lista"En"). Tienes la capacidad de realizar una búsqueda de Default o una búsqueda avanzada.

4. Especifique los criterios de búsqueda ("Contiene", "Empieza con", "Ignora el caso", ....).

5. Haga clic en la ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_rech_marq.gif) Icon para marcar todos los resultados de la búsqueda.

6. Presione "Enter" para iniciar la búsqueda. Si se encuentran elementos, la reference de estos elementos se muestra en el panel "Encontrar - Reemplazar".. Para acceder directamente a uno de los elementos encontrados, haga doble clic en el elemento reference.


Para más detalles, consulte la página de ayuda denominada [Buscar/Reemplazar](../Editeurs/2021003.md).

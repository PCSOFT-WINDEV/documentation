


## Corrector ortográfico en la interfaz
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000127"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten utilizar un corrector ortográfico en sus interfaces. El corrector ortográfico está disponible en el editor de ventanas, páginas y reportes.

Hasta la versión 23, el corrector ortográfico utilizado es el de OpenOffice. 

A partir de la versión 24, el corrector ortográfico está integrado en WINDEV, WEBDEV y WINDEV Mobile. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000151"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000022"></a>

El corrector ortográfico detecta errores gramaticales en el texto introducido en los diferentes editores:

- Texto que se muestra directamente en el editor (título de un control, por ejemplo).

- Texto introducido en las ventanas de descripción del editor.




El corrector ortográfico está integrado en WINDEV, WEBDEV y WINDEV Mobile. De forma predeterminada, la revisión ortográfica se propone en inglés y francés. Para utilizar otros idiomas, agregue los archivos correspondientes a los diccionarios Hunspell deseados en el directorio "\\Programs\\Data\\Dic". 

Estos diccionarios se pueden encontrar fácilmente en Internet. Los diccionarios contienen un archivo ".dic" y ".aff" con el mismo nombre que deben estar en la misma ubicación. Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md). 
Observación: Los archivos ".dic" y ".aff" deben nombrarse de la siguiente manera &lt;Abreviatura del idioma&gt;-&lt;Abreviatura del subidioma&lt;. Por ejemplo: 

- fr-FR.dic / fr-FR.aff

- de_DE.fic / de_DE.aff

- nl_NL.dic / nl_NL.aff


Puede comprobar si se cargó el diccionario utilizando la función [DictionaryLoad](../WDLang1/1000022958.md). 
<a name="NOTE2_2"></a>


### Activar y desactivar el corrector ortográfico
<a name="activar_desactivar_corrector_ortografico_ELTPARAGRAPHE000050"></a>

Para activar o desactivar el corrector ortográfico:

1. Abra las opciones generales del entorno: En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile".

2. En la pestaña "General", seleccione o deseleccione la opción "Desactivar corrector ortográfico".




**Observaciones**: 

- El corrector ortográfico también se puede utilizar en el editor de código. Para obtener más información, consulte [Correctores en el editor de código](../Editeurs/9000073.md).

- El corrector ortográfico se detiene automáticamente si más del 50% de las palabras son incorrectas. En este caso, el corrector considera que el idioma no es conocido o que se trata de un texto que contiene referencias, nombres propios y, por lo tanto, que este texto no debe corregirse. 

- Las palabras que se agregan al diccionario se encuentran en el archivo "C:\\Users\\&lt;User&gt;\\AppData\\Roaming\\PC SOFT\\perso.dic".

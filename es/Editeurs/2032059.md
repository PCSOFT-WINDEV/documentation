## Presentación
<a name="presentacion_ELTTEXTE000152"></a>
Se pueden utilizar dos métodos para crear una consulta:

- creando la consulta con una asistente o con la ventana de description, y generando el código SQL automáticamente.

- escribir el código SQL. Para obtener más información, consulte [¿Cómo crear una consulta en código SQL?](../Editeurs/2032060.md)




El editor de consultas le permite crear varios tipos de consultas:

- **consulta de selección**: este tipo de consulta le permite ver una selección de registros de uno o más archivos de una base de datos. Este es el tipo de consulta más común.
	Para obtener más información, consulte [¿Cómo crear una consulta selecta?](../Editeurs/2032062.md)

- **consulta de inserción**: este tipo de consulta se utiliza para añadir registros a un archivo de base de datos.
	Para obtener más información, consulte [¿Cómo crear una consulta de inserción?](../Editeurs/2032058.md)

- **consulta de actualización**: este tipo de consulta se utiliza para actualizar los registros encontrados en un archivo de base de datos.
	Para obtener más información, consulte [¿Cómo crear una consulta de actualización?](../Editeurs/2032057.md)

- **delete query**: este tipo de consulta se utiliza para eliminar registros de un archivo de base de datos.
	Para obtener más información, consulte [¿Cómo crear una consulta de borrado?](../Editeurs/2032047.md)

- **combinando query**: este tipo de consulta se utiliza para combinar consultas selectas.
	Para obtener más información, consulte [¿Cómo crear una consulta combinada?](../Editeurs/2032035.md)

**Observación**: Todos los tipos de consultas se pueden basar en los artículos que pertenecen:

- a la un archivo de datos descrita en el análisis.

- una consulta que se creó previamente con el editor de consultas.




Las características de la consulta se pueden modificar en cualquier momento, incluso después de haberla creado. Para obtener más información, consulte [Definir una consulta](../Editeurs/2032038.md).

## Notas
<a name="notas_ELTTEXTE000194"></a>


### Insertar consulta, actualizar consulta y eliminar consulta:
<a name="insertar_consulta_actualizar_consulta_eliminar_consulta_ELTPARAGRAPHE000066"></a>

La capacidad de crear consultas de inserción, actualización y eliminación es compatible con el estándar SQL.

Utilizando los comandos nativos de WINDEV ([HReadFirst](../WDLang4/3044051.md), [HReadNext](../WDLang4/3044037.md), [HAdd](../WDLang4/3044147.md), [HModify](../WDLang4/3044042.md), [HDelete](../WDLang4/3044018.md), ... se adapta mucho mejor a la programación de este tipo de Process.

SQL es un lenguaje de consulta utilizado para interrogar datos, pero no es adecuado para modificaciones personalizadas de estos datos.. Para añadir, modificar o eliminar registros en un archivo de base de datos, la potencia y simplicidad de WINDEV lo convierten en la solución ideal y recomendada.|   |   |   |

## Principio
<a name="principio_ELTTEXTE000236"></a>
Para crear una consulta, debe hacerlo:

- especificar el método utilizado para crear la consulta (asistente, ventana description o código SQL).

- especifique el tipo de consulta (insertar, actualizar, ...).

- especificar (si es necesario) el análisis WINDEV, WINDEV Mobile o WEBDEV que contiene la description de los archivos de datos en los que se basará la consulta.

- seleccione los elementos que desea utilizar.

- introduzca las características de la consulta (condiciones de selección, clasificación,...).

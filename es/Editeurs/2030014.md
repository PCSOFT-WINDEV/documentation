


## Hacer una copia de seguridad de un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000116"></a>
Es posible hacer regularmente una copia de seguridad de los diferentes elementos de un proyecto.

Durante esta copia de seguridad, puedes crear:

- un fichero de archivo (extensión ZIP). Este archivo contiene todos los elementos de proyecto que se han grabado.

- un simple copy del proyecto.




El proyecto guardado puede ser restaurado en cualquier momento.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000140"></a>


### Para hacer una copia de seguridad del proyecto actual:
<a name="para_hacer_una_copia_seguridad_del_proyecto_actual_ELTPARAGRAPHE000025"></a>

1. En la pestaña "Proyecto", en el grupo "Otras acciones", despliegue "Copia seg./Restaurar" y seleccione "Copia de seguridad del proyecto". Comienza el archivo de asistente.

2. Seleccione las categorías de los archivos de proyecto que desea guardar.

3. Seleccione el tipo de copia de seguridad que desea realizar:

	- ***Archivo de copia de seguridad comprimido (zip)***. En este caso, especifique el nombre del directorio y archivo de copia de seguridad. Este nombre puede ser un nombre personalizado o puede contener la fecha de la copia de seguridad.. Atención: Si la copia de seguridad se realiza en el directorio de versiones y si ya se ha guardado una versión con el mismo nombre, se sobrescribirá el archivo comprimido correspondiente.

	- ***Copia de seguridad***. En este caso, especifique el nombre del directorio de copia de seguridad. Este nombre puede ser un nombre personalizado o puede contener la fecha de la copia de seguridad.




4. Valide el asistente. El archivo (o la copy del proyecto) se crea en el directorio especificado.

## Notas
<a name="notas_ELTTEXTE000164"></a>


### restituir
<a name="restituir_ELTPARAGRAPHE000052"></a>

Para restaurar una copia de seguridad o un archivo de respaldo, en la pestaña "Proyecto", en el grupo "Otras acciones", expanda "Copia seg./Restaurar" y seleccione "Restaurar un proyecto". Para obtener más información, consulte [Restaurar un proyecto](../Editeurs/2030038.md).




## Elementos de temas
			





## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
Los elementos tema se utilizan para cambiar rápidamente el aspecto visual de un proyecto sin tener que retocar cada página y cada modelo.

Los elementos de tema incluyen tres elementos:

- **[las plantillas de página predefinidas](../Editeurs/2016019.md)**: se componen de controles (principalmente [células](../WDChamp/1013096.md)) y definen un diseño default para las páginas del proyecto.

- **[el tema](../Editeurs/2016020.md)**: está hecho de una hoja de estilo estandarizada que puede ser intercambiada con la de cualquier otra tema. Una tema se aplica a una página, a una página modelo o a todo el proyecto.

- **[el Color Palette](../Editeurs/2016021.md)**: está hecha de una set de colores armoniosos cuyos nombres están estandarizados para que puedan ser intercambiados. Se aplica a los estilos de la tema para redefinir los tonos.






## Utilización de los elementos de tema
<a name="utilizacion_los_elementos_tema_ELTTEXTE000132"></a>


### Uso en un nuevo proyecto
<a name="uso_nuevo_proyecto_ELTPARAGRAPHE000030"></a>

Al crear un nuevo proyecto, la creación de proyectos asistente permite elegir una tema y una Color Palette. Estos serán associated con el proyecto y utilizados por Default en todas las páginas.

La tema de un proyecto puede ser modificada en cualquier momento en el editor. Para ello:

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Selecciona la pestaña "Estilos de Tema".

3. Elija el nuevo tema y su color Palette.

4. Valide la ventana de descripción del proyecto. 




Al crear las páginas, podrá elegir las plantillas de página preestablecidas que desea utilizar.


### Uso en un proyecto existente
<a name="uso_proyecto_existente_ELTPARAGRAPHE000056"></a>

Un proyecto existente puede ser fácilmente adaptado para utilizar los elementos de tema. Esta operación se presenta en [Adaptación de un proyecto para utilizar las pieles](../Editeurs/2016018.md).

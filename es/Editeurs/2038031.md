<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
Administrador de Código Fuente ofrece una interfaz gráfica para la edición de ramas con el fin de optimizar la gestión de las ramas de sus proyectos.. Esta visualización proporciona una visión sintética de la vida del proyecto.

<a name="NOTE2_1"></a>


## Ventana de gestión de sucursales
<a name="ventana_gestion_sucursales_ELTTEXTE000128"></a>


### Visualización de la ventana para la gestión de sucursales
<a name="visualizacion_ventana_para_gestion_sucursales_ELTPARAGRAPHE000018"></a>

Para abrir la ventana de gestión de sucursales, en la pestaña "SCM", en el grupo "Proyecto", despliegue "Ramas" y seleccione "Gestionar sucursales".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Visu_Branche%20-%20HC%20N%B0001.gif)
Observación: Si la opción "Mostrar todas las revisiones" no está marcada, no se mostrarán las revisiones intermedias, lo que ofrece una visión aún más concisa.. 


### Opciones disponibles en la ventana de gestión de sucursales
<a name="opciones_disponibles_ventana_gestion_sucursales_ELTPARAGRAPHE000037"></a>El menú contextual del círculo de la columna "Rama" permite realizar varias operaciones en la rama seleccionada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Visu_Branche%20-%20HC%20N%B0002.gif)
Es posible: 

- **Cambiar la rama Color**: Esta opción le permite cambiar la color associated con la rama. En el selector color que aparece, seleccione el color deseado y confirme. 
	Observación: Esta color también se utiliza en el editor cuando se trabaja en una rama. 

- **Cambiar la rama description**: Esta opción permite cambiar el description associated con la rama. En la ventana que se muestra, especifique el nuevo description y valide. 
	Observación: Este description también se utiliza en el editor cuando se trabaja en una sucursal. 

- **Crear una rama a partir de esta revisión**: Esta opción le permite crear ramas posteriormente. La creación de la rama asistente comienza inmediatamente. Para obtener más información, consulte [Creación de una sucursal](../Editeurs/2038029.md). 

- **Fusionar con esta rama**: Esta opción permite recuperar las modificaciones realizadas en la rama. Para obtener más información, consulte [Recuperar las modificaciones realizadas en una rama](../Editeurs/2038030.md). 

- **Eliminar esta rama**: Esta opción permite eliminar la rama seleccionada (por ejemplo, si esta ya no se utiliza. En este caso, la SCM le pide que confirme la eliminación de la rama (esta acción no puede deshacerse).

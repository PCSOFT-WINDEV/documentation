## Presentación
<a name="presentacion_ELTTEXTE000087"></a>
Una aplicación que funciona con el groupware usuario puede utilizarse con una base de datos manejada por un Conector Nativo.

Para integrar un Conector Nativo a una aplicación que utiliza el User Groupware, este último debe estar configurado para utilizar los archivos de datos del servidor. Estas operaciones se presentan en esta página de ayuda.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000111"></a>
Para utilizar los archivos de User Groupware con un Conector Nativo: 

1. El Grupo de Usuarios no debe utilizar el modo "automatic start" en su aplicación.
	Asegúrese de que "automatic ejecutar" está desmarcado en la ventana de configuración de el groupware usuario ("Taller .. Configurar el groupware usuario").

2. En el código de inicialización del proyecto, utilice [gpwOpenConnection](../WDLang6/3041007.md) para describir la conexión a la base de datos manejada por un Conector Nativo.  
	**Atención**: las contraseñas definidas para los archivos de groupware usuario se ignoran.

3. Utilizar [gpwOpen](../WDLang6/3041002.md) sin especificar las rutas de los archivos de datos. El concepto de directorio no existe en este caso.

## Presentación
<a name="presentacion_ELTTEXTE000240"></a>
En cuanto se puede utilizar una Image (en un control, ventana, reporte, ...), la ventana de elemento description propone el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) para seleccionar la Image. Este botón muestra diferentes opciones. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0002.gif)
 La opción "Catálogo" le permite seleccionar un Image entre los numerosos Image s proporcionados en el catálogo "Producto" Image de WINDEV, WEBDEV y WINDEV Mobile.. 

Este catálogo de Image se utiliza para agrupar:

- Clipart: imágenes con 1 o más estados que se pueden utilizar para imágenes, botones, ...

- Splash Screen, pantalla Splash Screen avanzada,

- Formas, 

- Animaciones GIF, 

- Banner, Fondos de ventanas,

- Powered by, 

- Frontera,

- Balas, botones gráficos,

- Imágenes personalizadas, 

- Imágenes utilizadas para personalizar el estilo de los diferentes controles (pestañas, columnas tabla,...)




**Observación**: También tienes la posibilidad de acceder directamente al catálogo de Image:

- desde el editor de código. En efecto, al teclear funciones que manipular Image s, la opción "" es propuesta automáticamente por la entrada asistida.

- a través del panel ["Catálogo de imágenes"](../Editeurs/2027019.md).

## El catálogo de Image

### Modo de visualización del catálogo de Image
<a name="modo_visualizacion_del_catalogo_image_ELTPARAGRAPHE000046"></a>

El catálogo de Image se puede mostrar de 2 maneras: 

- **modo simplificado**: este modo se selecciona automáticamente cuando se visualiza el catálogo desde la opción "Catálogo Image" de un control Imagen. Este modo simplificado muestra sólo la búsqueda control y las imágenes correspondientes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Catalogue_Image_bis%20-%20HC%20N%B0002.gif&type=thumb)


- **modo completo**: este modo se selecciona automáticamente cuando se visualiza el catálogo desde la opción "Catálogo de imágenes" de una control específica : Icon, bordes de un botón, ... Este modo permite seleccionar un tema y la categoría de la Image deseada, ... <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Catalogue_Image_bis%20-%20HC%20N%B0003.gif&type=thumb)



Utilice la manipular (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Catalogue_Image_bis%20-%20HC%20N%B0004.gif)) para cambiar entre el modo simplificado y el completo.. 

Observación: para crear aplicaciones con el lenguaje de diseño de Windows 11, utilice el tema "Windows 11" del catálogo Image.

## Cómo usar las Image que se encuentran en el catálogo de Image
<a name="como_usar_las_image_que_encuentran_catalogo_image_ELTTEXTE000288"></a>


### Realizando una búsqueda en el catálogo de Image
<a name="realizando_una_busqueda_catalogo_image_ELTPARAGRAPHE000068"></a>

Hay varios modos de búsqueda disponibles dependiendo del modo de visualización. 

**En modo simplificado** , simplemente ingrese la palabra clave de búsqueda e inicie la búsqueda usando la lupa. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Catalogue_image.gif&type=thumb)


Para buscar **en modo completo**:  

1. Especifique los parámetros de búsqueda:

	- **palabra clave**: La palabra clave se utiliza para realizar una búsqueda entre los cliparts. La palabra clave se busca entre las palabras clave associated con los cliparts. El nombre del archivo buscado también se puede escribir directamente.

	- **tema**: La elección de un tema permite visualizar todos los elementos gráficos que utilizan el mismo estilo gráfico. Hay varios temas disponibles. 

	- **tipo image**: Imágenes prediseñadas, Pantalla de salpicadura, ....




2. Si ha introducido una palabra clave, inicie la búsqueda con el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=catalogue_image%201.gif). Se muestran las imágenes correspondientes a los criterios seleccionados.



<a name="NOTE2_3"></a>


### Asociando una Image encontrada en el catálogo con el elemento actual
<a name="asociando_una_image_encontrada_catalogo_con_elemento_actual_ELTPARAGRAPHE000089"></a>

Asociar una Image encontrada en el catálogo con el elemento actual:

1. En la ventana description del elemento actual, haga clic en el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) y seleccione "Catálogo".. Observación: Las imágenes propuestas por el catálogo de imágenes están adaptadas al actual control. Por ejemplo, si se muestra el catálogo de imágenes de la description de un borde de botón, sólo las imágenes de bordes de botón serán propuestas por Default.

2. Busque el Image solicitado si es necesario.

3. Seleccionar imagen. 

4. Haga doble clic en esta Image. 
	

5. La ventana para configurar el Image se muestra: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Catalogue_image_Detail.gif&type=thumb)


6. Especifique: 

	- El nombre de Image para generar

	- El formato a generar: bmp, gif, jpg, png, svg, ico, ...
			El formato svg está ahora disponible para la mayoría de los cliparts.

	- El número de estados de Image (para las cliparts).

	- Las características de la Image generada : 

		- colores: Es posible:  

			- utilizar los colores Default disponibles en el catálogo Image. 

			- utilizar colores HSL. En este caso, puede cambiar el Tono, la Saturación y la Luminosidad de la Image. 

			- colorear la Image (especialmente para las Image monocromáticas). Un selector de color permite definir la color y el umbral utilizados (saturación).  




- rotación,

- simetría, 

- fondo color y nivel de transparencia del fondo color,

- Opacidad de Image: los píxeles con un valor inferior al nivel definido serán completamente Transparent.

- El tamaño de Image.

- Si quieres generar una animación (sólo si el elemento actual corresponde al fondo Image de un botón).

- Si quieres almacenar los parámetros para el próximo Image seleccionado del catálogo.

7. Validar. La imagen se copia automáticamente en el directorio del proyecto y associated con el elemento actual.




El botón "Editar imagen" también le permite abrir y editar la Image en una nueva vista del editor. Puede desplazar esta vista y aprovechar todas las opciones del editor Image.

### Visualización de imágenes personalizadas
<a name="visualizacion_imagenes_personalizadas_ELTPARAGRAPHE000145"></a>

Para mostrar imágenes personalizadas:

1. Abra el catálogo de Image en modo completo si es necesario.

2. Haga clic en "Imágenes personalizadas".. 

3. Para agregar un Image , haga clic en "Agregar".

4. En la pestaña "Image", seleccione el fichero Image solicitado. Especifique el nombre de la imagen y su description.

5. En la pestaña "Palabras clave", especifique las palabras clave relacionado al Image. Estas palabras clave pueden ser elegidas entre las existentes o pueden ser creadas.

6. Validar. Su Image se agrega automáticamente a los Image s personalizados. Se tendrá en cuenta en las próximas búsquedas.




**Observaciones**:

- También es posible modificar las características de un Image personalizado o eliminar un Image personalizado.

- Las imágenes personalizadas están referenciadas en la base de datos del Centro control. Estas imágenes se conservarán al cambiar de ordenador o al cambiar la versión de WINDEV, WEBDEV o WINDEV Mobile.. Para obtener más información, consulte [Base de datos de los centros control](../CCSuivi/3540720.md).

## Imágenes encontradas en el catálogo
<a name="imagenes_encontradas_catalogo_ELTTEXTE000324"></a>


### Formato de las imágenes
<a name="formato_las_imagenes_ELTPARAGRAPHE000171"></a>

Las imágenes que se encuentran en el catálogo están disponibles en diferentes formatos: GIF, BMP, PNG, SVG, etc.


<a name="NOTE3_2"></a>


### Los botones gráficos
<a name="los_botones_graficos_ELTPARAGRAPHE000184"></a>

Seleccionando los botones gráficos, el catálogo de Image presenta todos los botones gráficos disponibles. Estos botones están clasificados por color. Se proponen varias formas para cada color.
<a name="NOTE3_3"></a>


### Las balas
<a name="las_balas_ELTPARAGRAPHE000191"></a>

Seleccionando las balas, el catálogo de Image presenta una gama de balas que pueden ser utilizadas en sus aplicaciones.

## Presentación
<a name="presentacion_ELTTEXTE000125"></a>
El RAD (Rapid Application Development) es una tecnología utilizada para crear fácilmente una aplicación completa (o sitio).

El RAD se basa en patrones de programación preestablecidos utilizados para generar la aplicación solicitada.. También tiene la posibilidad de crear sus propios "patrones" para generar automáticamente una aplicación personalizada.: se elige el código a generar así como el posicionamiento de los controles.

El RAD puede iniciarse de la siguiente manera: en la pestaña "Proyecto", en el grupo "Generación", haga clic en "RAD Aplicación completa". Una vez elegido el patrón, simplemente siga los diferentes pasos de la asistente.

### Aplicación WINDEV
<a name="aplicacion_windev_ELTPARAGRAPHE000036"></a>

Los siguientes patrones RAD son propuestos por Default:

- **RAD simple**:
	Este patrón se utiliza para generar una aplicación completa. Esta aplicación contiene formularios, tablas y bucles. El código utilizado es lo más simple posible. Se puede aplicar cualquier libro de estilo.

- **patrón de Antakara**:
	Este patrón se utiliza para crear una aplicación que propone algunas características avanzadas como una lista de tareas, un Calendar, ... Esta aplicación también propone un menú hecho de imágenes animadas.

- **Patrón puro**:
	Este patrón se utiliza para crear una aplicación con una interfaz adecuada para las pantallas grandes.

- **Patrón Jet'Tames**:
	Este patrón se utiliza para crear una aplicación que propone una característica específica para el envío de correos electrónicos.. El menú de la aplicación utiliza un carrusel de imágenes.

- **Compatible con 11 patrones RAD**:
	Este patrón se utiliza para generar una aplicación correspondiente a las aplicaciones generadas por la versión 11.
	Atención: Este tipo de RAD no está disponible en 64 bits.

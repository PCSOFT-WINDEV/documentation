## Presentación
<a name="presentacion_ELTTEXTE000115"></a>
Las funciones de la aplicación automatic (AAF) le permiten añadir funciones a su aplicación sin tener que escribir un solo Line de código. Todo lo que tiene que hacer es comprobar las opciones relevantes para beneficiarse de estas características.

Algunos ejemplos de AAF:

- [Añade un menú automatic en tablas para el usuario](../Editeurs/9500102.md).

- [Generar una ayuda para su aplicación](../Editeurs/2010030.md).

- [Use los botones de timer](../Editeurs/9500100.md).

- [Realizar búsquedas multi-control](../Editeurs/9500101.md).

- [Proponga utilizar un corrector ortográfico (OpenOffice) en los controles de edición y en las columnas tabla.](../Editeurs/1013146.md).

- [Crear métodos abreviados de teclado personalizada](../Editeurs/9500108.md).

- [Ordenar los list boxes y los combo boxes](../Editeurs/9500115.md).

- [Ir a un número de Line en un control Campo de entrada](../Editeurs/9500120.md).

- [copy y pegar el contenido de los controles de la ventana](../Editeurs/9500116.md).


**ayudar a los usuarios**
Utilice la función [AAFOpenHelp](../WDLang1/1410089447.md) para que los usuarios puedan acceder a la documentación sobre las funciones de automatic en su aplicación (ordenación y búsqueda en tablas, opciones de gráficos, etc.).. El sitio [https://aaf.windev.com/](https://aaf.windev.com/) presenta las funcionalidades de automatic disponibles en sus aplicaciones: redimensionar ventanas, opciones de tabla, opciones de gráficos, etc.). .

## Programación
<a name="programacion_ELTTEXTE000139"></a>


### Funciones de gestión de automatic Características
<a name="funciones_gestion_automatic_caracteristicas_ELTPARAGRAPHE000064"></a>

Varias funciones de WLanguage permiten gestionar las características de automatic:




|   |   |
| --- | --- |
| [AAFChangeCaption](../WDLang1/1000022100.md) | Personaliza el texto de una opción de menú para los menús contextuales correspondientes a una AAF (característica de la aplicación automatic). |
| [AAFConfigure](../WDLang1/1000022108.md) | Permite especificar el comportamiento de algunas características de la aplicación automatic (AAF). Por ejemplo, se puede configurar la opción de exportación a Excel utilizada por [ el menú de tablas de automatic](../Editeurs/9500102.md). |
| [AAFContextMenuToString](../WDLang1/1000022467.md) | Devuelve la lista de opciones del menú contextual que realmente muestra el control. |
| [AAFDisable](../WDLang1/1000022018.md) | Desactiva una función de aplicación de automatic (AAF) en una ventana de control o en la aplicación actual. |
| [AAFEnable](../WDLang1/1000024767.md) | Vuelve a activar una Funcionalidad Automática de la Aplicación (AAF) desactivada previamente con la función [AAFDisable](../WDLang1/1000022018.md). |
| [AAFExecute](../WDLang1/1000022099.md) | Ejecuta una función de aplicación de automatic (AAF) en una control o ventana. Esta función se puede utilizar para iniciar el corrector ortográfico en un control Campo de entrada, por ejemplo. |
| [AAFExportParameter](../WDLang1/1000024375.md) | Guarda todas las personalizaciones hechas a través de AAF (automatic Application Features) en un archivo. |
| [AAFExportSecurity](../WDLang1/1000022481.md) | Asegura la exportación de datos confidenciales en los AAF (Automatic Application Features) que se utilizan para exportar, imprimir o guardar datos.. |
| [AAFImportParameter](../WDLang1/1000024377.md) | Carga todas las personalizaciones realizadas a través de AAF (automatic Applications Features) que han sido previamente guardadas en un archivo. |
| [AAFOpenHelp](../WDLang1/1410089447.md) | Abre el sitio web de ayuda AAF (automatic Application Features). |
| [AAFPossible](../WDLang1/1000022791.md) | Indica si se puede ejecutar una AAF (característica de la aplicación automatic) en un control específico. |

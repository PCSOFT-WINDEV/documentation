
## Abrir un archivo de ayuda (CHM, HTML o PDF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
Hay varias formas de abrir un archivo de ayuda:

- [abriendo **una ayuda completa**](#NOTE2_1): usado para abrir el archivo de ayuda (muestra el resumen). 
	En la mayoría de los casos, esta operación se realiza desde una opción de menú (como "... Ayuda general").

- [abriendo **la ayuda para una ventana o un control en la aplicación**](#NOTE3_1): se utiliza para abrir el archivo de ayuda en un Page específico . Esta página presenta las diferentes opciones disponibles en la ventana o control. 
	En la mayoría de los casos, esta operación se realiza mediante la tecla F1 o mediante un botón de ayuda específico de la ventana.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Abrir una ayuda completa
<a name="abrir_una_ayuda_completa_ELTTEXTE000150"></a>
La ayuda la abre [WHelp](../WDLang1/3013053.md). Sólo tiene que especificar el nombre del archivo de ayuda que desea abrir. El archivo de ayuda se abre en la primera Page de la ayuda definida durante la compilación.

Por ejemplo:


```wl
// Open a CHM help file
WHelp("MyChmHelp.chm")
// Opening the help in a multilingual application
WHelp("MyHelp" + Nation() + ".chm")
```

## Abrir la ayuda de una ventana o control
<a name="abrir_ayuda_una_ventana_control_ELTTEXTE000174"></a>


### Abrir la ayuda de una ventana
<a name="abrir_ayuda_una_ventana_ELTPARAGRAPHE000045"></a>

Para abrir la ayuda Page de una ventana, utilice [WHelp](../WDLang1/3013053.md) y especifique el identificador del Page deseado . Este identificador se definió al crear el sistema de ayuda (en el [características de la página](../Editeurs/2010024.md)).

**Observación**: Para crear automáticamente el botón de ayuda que permite visualizar la Page correspondiente a una ventana, arrastre y suelte la Page de ayuda pertinente en la ventana solicitada.

El siguiente ejemplo abre la ayuda Page de la ventana "WIN_Configuration_Imp".. El identificador de esta página de ayuda es set a 5.


```wl
// Open a CHM help file
WHelp("MyChmHelp.chm", 5)
```

<a name="NOTE3_2"></a>


### Abrir la ayuda de una control
<a name="abrir_ayuda_una_control_ELTPARAGRAPHE000064"></a>

Para abrir la página de ayuda de un control:

1. Especifique el número de la página asociada a la control en el control description (pestaña "Ayuda", "Número de ayuda" control).

2. Utilice [WHelp](../WDLang1/3013053.md) associated con [NumHelp](../WDLang1/3013027.md) para abrir la ayuda Page (mediante un botón de ayuda, por ejemplo).




El siguiente ejemplo presenta el código de un botón de ayuda. Este botón abre la página de ayuda associated con el control que se está editando.


```wl
// -- Code of Help button
WHelp("MyHelp.chm", NumHelp(ControlCurrent()))
```

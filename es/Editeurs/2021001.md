


## Copiar el estilo de un control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000130"></a>
Copiar el estilo de un control consiste en copiar las características de un estilo control a otro control. Sólo se copia el estilo de control (no su contenido).

Esta copy se puede realizar entre:

- Dos mandos del mismo tipo.

- Dos mandos de diferentes tipos. En este caso, sólo se copian las características de estilo comunes a los dos controles.




El estilo copy puede realizarse entre dos controles pertenecientes a la misma ventana (a la misma página o al mismo reporte) o entre dos controles pertenecientes a ventanas diferentes (a páginas diferentes o a informes diferentes).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000154"></a>


### Copiar el estilo de un control en una ventana o en una página
<a name="copiar_estilo_control_una_ventana_una_pagina_ELTPARAGRAPHE000025"></a>

A copy un estilo entre dos controles:

1. Seleccione el control que contiene el estilo a copy.

2. En la pestaña "Ventana", en el grupo "Estilo y plantillas", haga clic en "Copiar estilo" (Ctrl + Shift + C).
	El cursor del ratón se convierte en pincel: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeCopieStyleCurseur.gif)

3. Seleccione la control a la que debe copiarse el estilo. El tipo de este control puede diferir del tipo de control cuyo estilo se copia.




**Observación**: Para aplicar el estilo de un control a varios control s, mantenga la tecla Ctrl abajo cuando copie el estilo en los control s de destino.


### Copiar el estilo de un control en un reporte
<a name="copiar_estilo_control_reporte_ELTPARAGRAPHE000051"></a>

A copy un estilo entre dos controles:

1. Seleccione el control que contiene el estilo a copy.

2. En la pestaña "Modificación", en el grupo "Edición rápida", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_pinceau_Etat.gif), Ctrl + Shift + C).
	El cursor del ratón se convierte en pincel: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeCopieStyleCurseur.gif)

3. Seleccione la control a la que debe copiarse el estilo. El tipo de este control puede diferir del tipo de control cuyo estilo se copia.




**Observación**: Para aplicar el estilo de un control a varios control s, mantenga la tecla Ctrl abajo cuando copie el estilo en los control s de destino.

## Notas
<a name="notas_ELTTEXTE000184"></a>


### Estilo de un control tenido en cuenta para el estilo copy
<a name="estilo_control_tenido_cuenta_para_estilo_copy_ELTPARAGRAPHE000079"></a>

El estilo de un control corresponde a las características especificadas en la pestaña "Estilo" de la ventana del control description.
Las imágenes que se muestran en las control (imagen de fondo de un botón por ejemplo) no se copian junto con el estilo control. La ruta de la imagen debe ser especificada en la description de los controles.

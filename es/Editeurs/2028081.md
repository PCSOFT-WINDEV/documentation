


## Instalar un servidor de HFSQL Client/Server con una aplicación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000158"></a>
Al crear el programa instalación de una aplicación WINDEV usando los archivos de datos de HFSQL Client/Server, el asistente le permite:

- utilizar los parámetros de conexión definidos en el análisis o mediante programación. En este caso, no se solicita información adicional.

- definir parámetros de conexión al instalar la aplicación. La instalación asistente propone varias opciones:

	- permiten al usuario final instalar un servidor de HFSQL Client/Server.

	- configurar la conexión a los archivos HFSQL Client/Server.

	- instalar los archivos de datos de HFSQL Client/Server.







Estas opciones aparecen en el paso "Parámetros de la base de datos" de la creación de instalación asistente.


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Instalar un servidor y elegir un tipo de servidor
<a name="instalar_servidor_elegir_tipo_servidor_ELTTEXTE000188"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000028"></a>

Durante su instalación, una aplicación WINDEV (aplicación monousuario o multiusuario) puede permitir al usuario final instalar un servidor de HFSQL Client/Server. Para ello, la aplicación WINDEV debe utilizar los archivos de datos de HFSQL Client/Server en el description de su análisis.

Para instalar un servidor HFSQL con su aplicación: 

1. Marque "Incluir la instalación de un servidor HFSQL" en el paso "Parámetros de la base de datos" de la instalación asistente.

2. Ve al siguiente avión.

3. La instalación asistente le da la posibilidad de permitir al usuario final que se instale:

	- un servidor Windows HFSQL Client/Server.

	- un servidor Linux HFSQL Client/Server.
			Puede proponer instalar uno de estos servidores (o ninguno).




4. Para realizar estas configuraciones, el programa instalación debe ser suministrado con los paquetes instalación de los diferentes servidores. Con el botón "Advanced" se selecciona el paquete necesario. El último paquete de instalación disponible también se puede descargar de la página web de PC SOFT.

## Configuración de la conexión
<a name="configuracion_conexion_ELTTEXTE000212"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000049"></a>

Si su aplicación utiliza la conexión a un servidor de HFSQL Client/Server definido en su análisis, esta conexión puede ser configurada automáticamente al instalar la aplicación final.

Para configurar esta conexión: 

1. Seleccione la conexión deseada.

2. Haga clic en el botón "Parámetro".



<a name="NOTE3_2"></a>


### Configuración
<a name="configuracion_ELTPARAGRAPHE000061"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=INstall_cs.gif)
Las principales opciones de configuración son las siguientes:

- Si se permite al usuario (o no) introducir los parámetros de la conexión durante la instalación.

- Si la conexión debe utilizar las opciones del análisis o los valores especificados en la asistente. En este caso, debe especificar los valores de la conexión a utilizar.

- Si deben utilizarse los parámetros del servidor que se instaló previamente.




**Observaciones**:

- Si la instalación debe crear una nueva base de datos, el usuario especificado en los parámetros de la instalación debe ser un "superusuario". De hecho, incluso si el usuario tiene los derechos para crear la base de datos, no se le permitirá crear archivos para esta base de datos.

- Si el usuario configura una conexión durante la instalación, esta conexión se almacenará en el archivo "&lt;Nombre ejecutable&gt;".WX". Este archivo corresponde a un archivo oculto en formato de texto. El acceso a su contenido lo realiza [INIRead](../WDLang1/3013043.md).

## Instalación de los archivos de datos
<a name="instalacion_los_archivos_datos_ELTTEXTE000242"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000083"></a>

Al crear la aplicación del programa instalación, también puede desplegar los archivos de datos de HFSQL Client/Server en el servidor deseado.
<a name="NOTE4_2"></a>


### Configuración
<a name="configuracion_ELTPARAGRAPHE000090"></a>

Para cada archivo de la aplicación, el botón "Propiedades" permite especificar:

- la ubicación actual del archivo de datos para instalar.

- la conexión que debe tener associated con este archivo

- si el índice debe copiarse o no. Si no se copia el índice, se reconstruirá automáticamente durante la instalación. Esta opción se utiliza para limitar el tamaño del programa instalación cuando el archivo índice es bastante grande.

## Instalación de la aplicación
<a name="instalacion_aplicacion_ELTTEXTE000272"></a>
Cuando la aplicación se instala en el ordenador del usuario final, dependiendo de las opciones seleccionadas, la asistente permite al usuario instalar un servidor de HFSQL Client/Server en un ordenador Windows o Linux. Esta asistente le ofrece la posibilidad de configurar la conexión utilizada.  

Al final de instalación, se crea un archivo ".wx" al lado del ejecutable.

La estructura de este fichero se da a título informativo. Esta estructura puede ser modificada en una próxima versión. Este archivo no debe ser modificado manualmente.

Este archivo contiene la siguiente información:

```txt
[CONNECT]
NBCX = Number of defined connections
NOM_x = Name of the connection number x
BASE_x = Type of database used (always corresponds to hAccessHFClientServer)
ADRESSE_x = Address of server used (name or IP address of the server)
PORT_x = Connection port
DATABASE_x = Name of the database used
LOGIN_x = Connection login
PWD_x = Encrypted password
ACCESS_x = Type of access (hOReadWrite or hORead)
EXTINFO_x = Extended information
```

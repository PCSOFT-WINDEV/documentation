## Presentación
<a name="presentacion_ELTTEXTE000203"></a>
Las "Custom-Folders" simplifican la organización de los proyectos, permitiéndole agrupar los elementos de su aplicación por temas. Hay disponibles carpetas personalizadas: 

- para cada categoría de elementos en el panel "Explorador del proyecto".: ventanas, informes, ....

- a nivel de proyecto. 




**Observaciones**:

- También puede crear carpetas personalizadas en la editor de análisis. Para obtener más información, consulte [Carpetas personalizadas en el análisis](../Editeurs/2011003.md).

- También tiene la posibilidad de organizar un gran proyecto a través del sistema [componentes internos](../Editeurs/2014100.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000227"></a>


### Crear una Carpeta Personal
<a name="crear_una_carpeta_personal_ELTPARAGRAPHE000032"></a>

Para crear un Carpeta Personal:

1. Si es necesario, muestre el panel "Explorador de proyectos".

2. Seleccione la carpeta en la que se creará el Carpeta Personal: Ventanas, Páginas,...

3. Seleccione "Nueva Carpeta Personal" en el menú contextual.

4. Introduce el nombre de la Carpeta Personal y valida. El Carpeta Personal aparece en el panel "Explorador del proyecto" como una carpeta amarilla. Este Carpeta Personal puede ser personalizado. Para obtener más información, consulte [Acciones que se pueden realizar en una Carpeta Personal](#NOTE3_1).




**Observaciones**:

- Si estás usando SCM, debes ser maestro en el proyecto para crear un Carpeta Personal: la Carpeta Personal será transportada durante la próxima sincronización. De lo contrario, el Carpeta Personal creado desaparecerá durante la próxima sincronización.

- Las carpetas personalizadas están disponibles en los conjuntos de procedimientos.



<a name="NOTE2_2"></a>


### Abrir un Carpeta Personal
<a name="abrir_carpeta_personal_ELTPARAGRAPHE000054"></a>

Para abrir un Carpeta Personal, haga doble clic en su nombre en el panel "Explorador del proyecto".
<a name="NOTE2_4"></a>


### Asociando los elementos del proyecto con un Carpeta Personal
<a name="asociando_los_elementos_del_proyecto_con_carpeta_personal_ELTPARAGRAPHE000061"></a>

**To asociar un elemento del proyecto con un Carpeta Personal**: 

1. Seleccione el elemento deseado en el panel "Explorador de proyectos. 

2. Arrastrar y soltar el elemento a la Carpeta Personal solicitada. 




**To asociar el mismo elemento con varias carpetas personalizadas**, mantener pulsada la tecla Ctrl durante el arrastrar y soltar al Carpeta Personal.

**Observación**: Al guardar un nuevo elemento, se muestra la siguiente ventana: 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PersoDossier.GIF)

Esta ventana se utiliza para especificar la carpeta personalizada en la que el elemento actual será associated. 
Para que esta ventana se muestre automáticamente cada vez que se guarda un nuevo elemento, marque "Preguntar la ubicación de los nuevos elementos en el proyecto" en la pestaña "Opciones" de la descripción del proyecto (pestaña "Proyecto", grupo "Proyecto", "Descripción").

El elemento del proyecto **A puede ser associated con varias carpetas personalizadas.**

**Observación**: **Carpeta Personal y SCM**: 
Si estás usando el SCM, debes ser maestro en el proyecto para asociar un nuevo elemento con un Carpeta Personal. En este caso, para recuperar el elemento en la Carpeta Personal, los otros desarrolladores deben: 

- ir a la pestaña "SCM", grupo "Proyecto", ampliar "Maestro/Invitado". 

- seleccione "get la versión maestra (completa)". En todos los demás casos, el elemento será recuperado pero no será incluido en el Carpeta Personal.

- Un huésped puede incluir un elemento en un Carpeta Personal. Sin embargo, los otros desarrolladores no pueden recuperar el elemento en el mismo Carpeta Personal: sólo recuperarán el elemento.



<a name="NOTE2_5"></a>


### Disociar un elemento de una Carpeta Personal
<a name="disociar_elemento_una_carpeta_personal_ELTPARAGRAPHE000120"></a>

Para disociar un elemento de una Carpeta Personal: 

1. En el panel "Explorador de proyectos", seleccione el elemento a disociar. 

2. Arrastre y suelte el elemento que se encuentra en la Carpeta Personal al nombre del proyecto.



<a name="NOTE2_6"></a>


### Renombrar un Carpeta Personal
<a name="renombrar_carpeta_personal_ELTPARAGRAPHE000130"></a>

Para renombrar a un Carpeta Personal: 

1. En el panel "Explorador del proyecto", seleccione el Carpeta Personal para renombrar. 

2. Seleccione la opción "Descripción" en el menú contextual. 

3. En la ventana que aparece, modifique el nombre de Carpeta Personal y valide.

## Acciones que se pueden realizar en una Carpeta Personal
<a name="acciones_que_pueden_realizar_una_carpeta_personal_ELTTEXTE000275"></a>


### Imprimir el contenido de un Carpeta Personal
<a name="imprimir_contenido_carpeta_personal_ELTPARAGRAPHE000143"></a>

Puede imprimir la documentación de los elementos que se encuentran en un Carpeta Personal: 

1. Seleccione la Carpeta Personal en el panel "Explorador del proyecto".. 

2. Seleccione "Imprimir la documentación de Carpeta Personal" en el menú contextual.

3. Comienza la asistente para generar la documentación de los elementos en la Carpeta Personal.

## Componente
<a name="componente_ELTTEXTE000299"></a>
Cuando se añade un componente al proyecto actual, los elementos utilizados por este componente se agrupan en un Carpeta Personal.


## Lista de tareas 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000115"></a>
Cada proyecto puede ser associated con una lista de tareas. Esta lista se utiliza para informar a los desarrolladores sobre las modificaciones que deben hacerse al proyecto.

Estas modificaciones pueden afectar:

- una Process en el código de todo el proyecto,

- un objeto (ventana, reporte, ....) para modificar o crear, etc.


**Observaciones**: 

- **Esta lista de tareas no es relacionado al Centro de Seguimiento de Proyectos.** 

- Para gestionar las tareas y un calendario completo, le recomendamos que utilice el programa Centro de Seguimiento de Proyectos. Para obtener más información, consulte [¿Cómo gestiono mis tareas y mi agenda?](../CCSuivi/3540721.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000139"></a>


### Visualización de la hoja de ruta
<a name="visualizacion_hoja_ruta_ELTPARAGRAPHE000031"></a>

Para ver la lista de tareas, visualice la pantalla ["Panel"Lista de tareas](../Editeurs/2027015.md).
<a name="NOTE2_2"></a>


### Añadir una tarea
<a name="anadir_una_tarea_ELTPARAGRAPHE000041"></a>

Se pueden utilizar dos métodos para pedir una tarea, dependiendo de la naturaleza de la tarea. 

**Caso 1: Añadir una tarea específica a la programación**: Introduzca directamente en el código, la descripción de la tarea anterior de "//Por hacer". Por ejemplo:


```wl
// To do: Management of bar codes
```


Observación: Para posicionarse en el código en la ubicación específica que contiene la tarea a realizar, todo lo que tiene que hacer es hacer doble clic en el título de la tarea en el panel "Lista de tareas".

**Caso 2: Añadir una tarea a través del panel "Lista de tareas**: Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_Ajout_Tache.gif) en el ["Panel"Lista de tareas](../Editeurs/2027015.md). Cada tarea puede:

- Ser asignado a uno de los desarrolladores del proyecto.

- Corresponden a un nivel de prioridad específico (prioridad alta, media o baja). Esto le da la capacidad de ordenar las tareas por nivel de prioridad.



<a name="NOTE2_3"></a>


### Manejo de las tareas
<a name="manejo_las_tareas_ELTPARAGRAPHE000064"></a>

Para manipular las tareas, usa el menú contextual del panel "Lista de tareas".

**Observación**: La visualización de las tareas en el panel "Lista de tareas" no está sincronizada con el código. Por ejemplo, si el comment Line "// Hacer: Gestión de códigos de barras" se borra, la tarea correspondiente en el panel "Lista de tareas" no se borra automáticamente.

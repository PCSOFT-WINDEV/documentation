### ¿Qué es un metattipo?
<a name="¿que_metattipo_ELTPARAGRAPHE000011"></a>

Un metatítulo es una combinación de características de un elemento y una control vinculada.

Por ejemplo, un tipo de metadatos "Fax" contendrá el tipo y la longitud del elemento, la máscara de entrada, la alineación, ....

A continuación, esta información se utilizará en todos los módulos, para cada acción. Por lo tanto, la definición del análisis se simplifica, así como el proceso para la generación de RAD, la creación automatic de un control , ...

Es posible:

- [utilizar un metatype propuesto en el editor de análisis](#NOTE2_1).

- [crear sus propios metadatos](#NOTE2_2).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000162"></a>


### Utilizando un metatype propuesto por el editor de análisis
<a name="utilizando_metatype_propuesto_por_editor_analisis_ELTPARAGRAPHE000033"></a>

Para crear un elemento a partir de un metatype en el editor de análisis:

1. En la editor de análisis, mostrar la description de los elementos de archivo de datos deseados (doble clic en el archivo de datos).

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MLD_Metatype_ico.gif).

3. La siguiente ventana se abre:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MLD_Metatype.gif&type=thumb)


4. Especifique una palabra clave en el cuadro combinado y haga clic en el botón "Buscar. Una vez que se encuentre el metatype solicitado, haga clic en su Image. Se visualizan sus características.

5. Validar. La posición correspondiente se crea automáticamente.




Observación: Al crear el elemento, también puede seleccionar "Metatype" en la lista de nombres de elementos disponibles.
<a name="NOTE2_2"></a>


### Creando un metatype que será propuesto por el editor de análisis
<a name="creando_metatype_que_sera_propuesto_por_editor_analisis_ELTPARAGRAPHE000051"></a>

Para crear un metatype en el editor de análisis:

1. En la editor de análisis, mostrar la description de los elementos de archivo de datos deseados (doble clic en el archivo de datos).

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MLD_Metatype_ico.gif).

3. Se muestra la ventana de gestión de metatodos. 

4. Haga clic en el botón "Nuevo" (debajo de la opción "Tipos de metadatos de usuario") para crear un tipo de metadatos.

5. En la ventana que se visualiza, especifique:

	- en la pestaña "General: 

		- Nombre del tipo de metadatos.

		- description del metatype.

		- Image del tipo de metástasis.




- en la pestaña "Parámetros: 

	- el tipo y el subtipo del elemento (esta información corresponde a las opciones seleccionadas para el elemento base del metato).

	- el valor de Default.

	- la información compartida (esta información corresponde a las opciones seleccionadas para el elemento base del metattipo).

6. Validar la creación del metattipo. El tipo de metadatos se guarda en un archivo en formato XML. Por Default, este archivo se guarda en el subdirectorio "\\Personal\\Metatype" de WINDEV, WEBDEV o WINDEV Mobile. Este metatype puede entonces ser importado en el editor de análisis de otro ordenador.



<a name="NOTE2_3"></a>


### Importar un metatype en el editor de análisis
<a name="importar_metatype_editor_analisis_ELTPARAGRAPHE000081"></a>

Para importar un metatype en el editor de análisis:

1. En el editor de análisis, seleccione "Análisis... Importar un metatype de usuario".
	Observación: Para acceder a esta opción, utilice el menú principal accesible a través del logotipo de la versión: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_logo_menu17.gif)


2. Seleccione el archivo XML correspondiente al tipo de metadatos a importar. Este archivo XML se creó al crear el metattipo.

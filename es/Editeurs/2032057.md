


## Crear una consulta de actualización 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000168"></a>
<a name="Update_query"></a>
Una consulta de actualización (sentencia SQL [UPDATE](../Editeurs/2034001.md)) se utiliza para modificar los registros de un archivo de base de datos.

Para realizar esta modificación, las principales características de la consulta son las siguientes:

- el archivo de datos en el que los datos serán modificados.

- los elementos en los que se modificarán los valores.

- las condiciones de selección de los registros (clientes que viven en Seattle, productos cuyo precio es superior a 40 dólares, ....).




Para más detalles sobre las condiciones de selección de la consulta, véase [Condiciones de selección de una consulta](../Editeurs/2032019.md).

Por ejemplo, el cliente #45762 se ha casado. Su título y apellido han sido modificados. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_RequeteModification.gif)
 El editor de consultas genera automáticamente el código SQL de esta consulta. 

**Para ver el código SQL de una consulta**: 

1. Abrir el menú contextual del gráfico de consulta.

2. Seleccione "Código SQL".




También tiene la posibilidad de presionar la tecla F2. 

En nuestro ejemplo:


```sql
UPDATE 
	CUSTOMER
SET
	Title = 'Mrs',	
	CustomerName = 'VAUVERT'
WHERE 
	Customer.CustomerNum = 45762
```

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000210"></a>
Para crear una nueva consulta de actualización:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Consulta".

	- El asistente de creación de consultas se abre.




2. Especifique que desea crear una consulta de actualización (opción"Actualizar").

3. Si su proyecto es relacionado a ningún análisis, especifique el análisis al que se adjuntará la consulta.
	Pase a la etapa siguiente del asistente.

4. Seleccione el archivo de datos en el que los datos serán modificados.
	Pase a la etapa siguiente del asistente.

5. Para cada elemento cuyo valor deba modificarse, haga clic en la columna "Valor/Parámetro".. Una ventana permite seleccionar el valor o el parámetro a modificar.

6. Especifique el valor o el parámetro que contiene el valor a modificar. Los valores a modificar pueden corresponder a:

	- el valor de Default definido en el análisis.

	- un valor que debe especificarse. Especifique este valor.

	- un valor contenido en un parámetro. Indique el nombre del parámetro.

	- un valor nulo.




7. Valide su elección.

8. Repetir las operaciones 5 a 7 para cada elemento que contenga un valor que deba modificarse.

9. Definir las condiciones de selección. Haga clic en la columna "Condición" de los elementos relevantes. Para más detalles sobre las condiciones de selección de la consulta, véase [Condiciones de selección de una consulta](../Editeurs/2032019.md).
	Pase a la etapa siguiente del asistente.

10. Especifique (si es necesario) la secuencia de las condiciones de selección con los operadores condicionales AND y OR AND:

	- *Condicional Y operador*: los registros seleccionados coincidirán con ambas condiciones. Por ejemplo, los clientes cuyo nombre comienza con la letra "A" Y que viven en Denver.

	- *Operador condicional O*: los registros seleccionados coincidirán con una de las dos condiciones o con ambas condiciones. Por ejemplo, la consulta selecciona los clientes que viven en Seattle O en Denver.
			Pase a la etapa siguiente del asistente.




11. Escriba el nombre de la consulta (nombre del archivo ".WDR" correspondiente a la consulta). Este nombre se utilizará para identificar la consulta en sus programas.

12. Especifique el título de la consulta describiendo brevemente el propósito de la consulta. Este epígrafe se utilizará en la documentación del proyecto..
	Observación: El nombre de la consulta se define automáticamente a partir de la leyenda escrita.

13. Validar la consulta description. La representación gráfica de la consulta se muestra automáticamente en el editor de consultas.

## Notas
<a name="notas_ELTTEXTE000234"></a>


- Todas las características de una consulta pueden ser modificadas: en la pestaña "Consulta", en el grupo "Edición", haga clic en "Descripción". Para obtener más información, consulte [Definir una consulta](../Editeurs/2032038.md).

- Para generar el código SQL correspondiente a esta consulta, seleccione "Código SQL" en el menú contextual de la consulta. Si se realizan cambios en la consulta description, estos cambios se reflejarán automáticamente en el código SQL correspondiente.

- También tiene la posibilidad de crear una consulta a partir de uno o más ficheros de datos descritos en la editor de análisis: Para ello, arrastre los archivos de datos seleccionados en el panel "Análisis" y suéltelos en el editor de consultas.

- La prueba de la consulta creada se puede ejecutar desde el editor de consultas. Para obtener más información, consulte [Ejecutar la prueba de una consulta](../Editeurs/2019011.md).

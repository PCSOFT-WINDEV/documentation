## Presentación
<a name="presentacion_ELTTEXTE000194"></a>
Un modelado flexible incluye diferentes elementos relacionado juntos. Cada elemento tiene sus propias características.

Para ver las características de un elemento, seleccione "description" en el menú contextual de dicho elemento. La información visualizada depende del elemento seleccionado.

Hay dos pestañas disponibles:

- la pestaña "General" que presenta las características generales del elemento.

- la pestaña "Estilo" que se utiliza para definir las opciones de estilo del elemento.

## Opciones generales
<a name="opciones_generales_ELTTEXTE000218"></a>


### Pie de foto/Nombre
<a name="pie_fotonombre_ELTPARAGRAPHE000025"></a>

Leyenda mostrada en el elemento.

**Para modificar este título en el editor** , debe:

- Seleccione el elemento

- Haga clic en el título: el título está en edición y puede ser modificado.




**Para configurar el diseño del título o su posición** , seleccione " color /Estilo/Fuente" en el menú contextual del elemento.
<a name="NOTE2_2"></a>


### Image (sólo el elemento Image)
<a name="image_solo_elemento_image_ELTPARAGRAPHE000039"></a>

Ruta de la Image que se mostrará en el elemento Image. El botón "Catálogo" también le permite seleccionar una de las imágenes propuestas por la [catálogo de imágenes](../Editeurs/2020005.md) de WINDEV, WEBDEV, WINDEV Mobile. En este caso, el Image seleccionado se copiará automáticamente en el directorio del proyecto actual.
<a name="NOTE2_3"></a>


### vincular
<a name="vincular_ELTPARAGRAPHE000049"></a>

Ruta del fichero relacionado al elemento. El archivo relacionado puede ser un elemento del proyecto actual o de otro archivo.

**Para vincular un elemento del modelado flexible a un elemento del proyecto** , debe:

- arrastre el elemento en el proyecto (desde el panel "Proyecto", por ejemplo) y suéltelo sobre el elemento deseado en el modelado.

- "Arrastrar y soltar" el elemento desde el proyecto hacia el editor de modelado. En este caso, se crea un nuevo elemento en el modelado.




**Para modificar el vínculo entre el elemento del modelado y un elemento del proyecto**:

1. Compruebe si el panel "Explorador de proyectos" aparece en su entorno: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Paneles", seleccione "Paneles" y, a continuación, "Explorador de proyectos".

2. Abra la description del elemento de modelación.

3. Haz clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Cible.gif) y mantén pulsado el botón izquierdo del ratón.

4. Mueva el ratón hasta el nuevo elemento del proyecto que será relacionado en el panel "Explorador de proyectos".

5. Suelte el botón del ratón cuando se encuentre Above el elemento deseado. El nuevo enlace aparece en la ventana de description del elemento de modelación.




**Observación**: Esta operación no se aplica a los elementos "Ventanas", "Informes" y "Component" del modelado flexible. En este caso, para modificar el enlace, el elemento debe ser eliminado y recreado.

**Si el elemento de modelado está relacionado con un elemento del proyecto**, el elemento de modelado contiene el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=modelisation_liaison.gif). Esta flecha hacia abajo indica que se estableció un enlace: haciendo doble clic en el elemento se abre directamente el objeto relacionado. También puede seleccionar "Abre el elemento relacionado" en el menú contextual del elemento.

En los distintos editores, puede identificar los elementos de un modelado. Un botón específico se muestra en el description del elemento del proyecto cuando este elemento es relacionado a una modelación.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_lien_fen.gif)
 Si sólo hay un modelado relacionado al elemento, un clic en el botón abrirá el modelado.

Si hay varios modelos de relacionado al elemento, al hacer clic en el botón le permitirá seleccionar el modelo a abrir.
<a name="NOTE2_4"></a>


### Nivel de automatización
<a name="nivel_automatizacion_ELTPARAGRAPHE000095"></a>

El nivel de automatización se utiliza para averiguar y visualizar el nivel de automatización del elemento actual.

Si se marca "Nivel de automatización", el nivel de automatización se mostrará en formato Icon en el modelado.

Los niveles de automatización disponibles son:

- Proceso informático automatizado.

- Proceso informático con intervención manual.

- Proceso manual.



<a name="NOTE2_5"></a>


### Actores
<a name="actores_ELTPARAGRAPHE000110"></a>

Los actores son las personas que participan en la tarea descrita por el elemento.

Para configurar los actores que se encuentran en las diferentes listas, haga clic en el botón "Editar los actores".. Para obtener más información, consulte [Propiedades del modelado](../Editeurs/2010103.md).

- **Gerente**: Responsable del elemento descrito.

- **Desarrollador**: Persona responsable del desarrollo del elemento.

- **Usuario**: Persona que utiliza la aplicación, por ejemplo.




Se pueden seleccionar varias personas para cada tipo de responsabilidad.

Si se selecciona a una sola persona, tiene la posibilidad de obtener sus datos personales (dirección, número de Phone, Email, ...).

La casilla "Mostrar" permite especificar si la información introducida debe mostrarse (o no) en el modelado.

## Opciones de estilo
<a name="opciones_estilo_ELTTEXTE000266"></a>
Las opciones de estilo del elemento seleccionado se utilizan para definir todas las características de estilo del elemento seleccionado.. Usted tiene la capacidad de seleccionar:

- el tipo de letra utilizado, su tamaño, sus atributos (negrita, cursiva, ...), su color, su alineación.

- el fondo color del elemento, el color de su borde, su estilo y su grosor.




Cuando se modifica el estilo de un elemento, se puede:

- Crear un nuevo estilo en el modelado. En este caso, el estilo puede ser reutilizado por otro elemento.

- Modificar sólo el elemento actual.




Para obtener más información, consulte [Estilos de un modelaje](../Editeurs/2010105.md).

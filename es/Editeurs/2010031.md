## Resaltar los elementos seleccionados
<a name="resaltar_los_elementos_seleccionados_ELTTEXTE000411"></a>
En el editor de códigos, siempre que el cursor se posiciona en un identificador (variable, Procedure, archivo de datos, etc.), todos los usos de este identificador se resaltan en el código actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Surbrillance_Element.gif)


Del mismo modo, si el cursor del ratón se posiciona sobre una sentencia de bloque (IF, THEN, ELSE, FOR, END, LOOP, WHEN EXCEPTION ON, etc.), se resaltan los diferentes elementos de este bloque. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Surbrillance_bloc.gif)


Observación: Cuando las palabras clave secundarias (ELSE, END, etc.) se cierran con el cursor del ratón, la condición asociada se muestra en un tooltip. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_BulleSi.gif)


Finalmente, el actual Line puede ser fácilmente identificado a través de puntos usados para resaltar el actual Line. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_surlignage_ligne.gif)


Todas estas opciones pueden ser modificadas si es necesario. 

**Para modificar las opciones de resaltado**:

1. Mostrar la ventana de configuración del editor de código: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. En la pestaña "Código", seleccione las siguientes opciones si es necesario: 

	- "Resaltar los bloques de código (IF/THEN/END, paréntesis, etc)".

	- "Resaltar la línea actual".

	- "Destacar la declaración y los usos de la Variable".




3. Validar. 




**Observación**: Todos los colores utilizados para resaltar los elementos pueden configurarse en la pestaña "Colores" de la ventana de configuración del editor de código..

## Código de búsqueda
<a name="codigo_busqueda_ELTTEXTE000435"></a>
**Tpara encontrar un elemento en el editor de código**, todo lo que tienes que hacer es usar el panel "Encontrar/Reemplazar", activado mediante Ctrl + F.

**Para realizar una búsqueda inmediata en un elemento encontrado en el editor de código**todo lo que tiene que hacer es presionar Ctrl F3 en un elemento del editor de código (se muestra el panel "Buscar/Reemplazar"). 

Presionando Ctrl + F3: 

- llena la búsqueda control con la palabra sobre la que se posiciona el cursor. 

- desencadena la búsqueda. 

- resalta en el código todas las ocurrencias encontradas y las posiciones en la primera ocurrencia. 




Las teclas F3 y Mayúsculas + F3 le permiten navegar de una ocurrencia a otra a través del teclado.. 


### Buscar en una selección
<a name="buscar_una_seleccion_ELTPARAGRAPHE000109"></a>

El editor de código le permite realizar una búsqueda (y un reemplazo) en una selección. 

Si una selección que supera una línea es Active, la búsqueda se realiza automáticamente en esta selección. 

Al realizar una búsqueda en una selección, la palabra buscada se resalta en el código. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Recherche_Selection.gif)
 F3 y Shift + F3 permiten pasar de una ocurrencia a otra.  


### Encontrar y acceder a un Event
<a name="encontrar_acceder_event_ELTPARAGRAPHE000120"></a>

El editor de código permite encontrar y acceder a una Event del elemento actual: ventana, página, reporte, ... 

1. En la pestaña "Código", en el grupo "Navegación", escriba el evento que desea buscar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Recherche_Traitement.gif)
Observación: Para mostrar automáticamente la lista de eventos, utilice Alt + C.

2. Seleccione la Event deseada en la lista. Se muestra automáticamente en el editor de códigos.

## Selección en el editor de código
<a name="seleccion_editor_codigo_ELTTEXTE000471"></a>


### Selección de líneas de código
<a name="seleccion_lineas_codigo_ELTPARAGRAPHE000142"></a>

Para realizar una selección simple: 

- Un clic posiciona el cursor. 

- Con dos clics se selecciona la palabra. 

- Con tres clics se selecciona la Line. 

- Con cuatro clics se selecciona todo el texto. 




El resaltado de sintaxis se mantiene cuando se selecciona el texto. 

Observación: Para seleccionar la palabra en la posición del cursor, también puede pulsar Ctrl + Shift + E. 


### Selección rectangular
<a name="seleccion_rectangular_ELTPARAGRAPHE000158"></a>

Puede seleccionar un área de texto rectangular horizontal o vertical en el editor de código. Esta zona se puede copiar/cortar y pegar. 

Para realizar una selección rectangular:

1. Sitúe el cursor al principio del área para seleccionar. 

2. Mantenga pulsada la tecla Alt. 

3. Presione el botón izquierdo del ratón y seleccione el área deseada. 




Observación: Esta selección también se puede hacer con el teclado: 

1. Sitúe el cursor al principio del área a seleccionar.

2. Utilice las teclas Alt + Shift + Flechas (arriba, abajo, derecha, izquierda) para seleccionar el área deseada. 




El área se puede copiar/cortar a través del menú contextual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Selection_rectangle.gif)
**Observación**: Cuando el área se pega en otra área rectangular, el texto copiado mantiene su formato. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Selection_rectangle_2.gif)



### Selección por nivel de bloques
<a name="seleccion_por_nivel_bloques_ELTPARAGRAPHE000181"></a>

Tiene la capacidad de realizar una selección por nivel de bloques de código. 

Permite realizar una selección por nivel de bloques:

1. Posicionar el cursor en el bloque para seleccionar. 

2. Presione Ctrl + Alt + Flecha arriba para seleccionar el bloque encontrado Above el cursor. Repita esta operación tantas veces como sea necesario.. 

3. Presione Ctrl + Alt + Flecha abajo para deseleccionar el bloque encontrado Above el cursor. Repita esta operación tantas veces como sea necesario.

## Manejo de una o más líneas de código
<a name="manejo_una_mas_lineas_codigo_ELTTEXTE000507"></a>


### Selección de líneas de código
<a name="seleccion_lineas_codigo_ELTPARAGRAPHE000196"></a>

- Un clic posiciona el cursor. 

- Con dos clics se selecciona la palabra. 

- Con tres clics se selecciona la Line. 

- Con cuatro clics se selecciona todo el texto. 





### Mover líneas de código
<a name="mover_lineas_codigo_ELTPARAGRAPHE000204"></a>

Un área seleccionada (una o más líneas de código) puede ser movida directamente usando la combinación de teclas Alt + Flecha arriba o Alt + Flecha abajo.


### Duplicación de líneas de código
<a name="duplicacion_lineas_codigo_ELTPARAGRAPHE000209"></a>

La duplicación de Line s de código (Ctrl + D) opera sobre un solo Line o sobre una selección de Line s de código. 


### Eliminar partes de líneas de código
<a name="eliminar_partes_lineas_codigo_ELTPARAGRAPHE000216"></a>

A eliminar: 

- el código antes del cursor (en la misma Line): Ctrl + Shift + Backspace

- el código después del cursor (en la misma Line): Ctrl + Shift + Del





### Ir a la siguiente Line
<a name="siguiente_line_ELTPARAGRAPHE000224"></a>

Cuando esté escribiendo código y llegue al final de la Line, puede utilizar: 

- la tecla Enter,

- la tecla de flecha derecha.





### Información sobre las líneas de código seleccionadas
<a name="informacion_sobre_las_lineas_codigo_seleccionadas_ELTPARAGRAPHE000236"></a>

Cuando selecciona una o más líneas de código, la barra de estado del editor de código muestra la siguiente información: 

```txt
SEL <Selected characters>|<Selected lines>
```

Por ejemplo, cuando se seleccionan 3 líneas de código, se puede ver lo siguiente: 

```txt
50|3
```



### Invertir asignaciones
<a name="invertir_asignaciones_ELTPARAGRAPHE000249"></a>

Para anular una asignación: 

1. Seleccione la Line de código que contiene la asignación. 

2. Pulse Ctrl + Alt + = para invertir las asignaciones. 




Por ejemplo, el siguiente código: 

```wl
nx is int
ny is int

nx = ny
```
 cambia a

```wl
nx is int
ny is int

ny = nx
```

## Añadir comillas/corchetes o paréntesis automáticamente
<a name="anadir_comillascorchetes_parentesis_automaticamente_ELTTEXTE000567"></a>
Para añadir automáticamente comillas, corchetes o paréntesis: 

1. Seleccione una o más palabras en el editor de código. 

2. Pulse la tecla de presupuesto, de corchete o de paréntesis. 

3. El texto seleccionado se encierra automáticamente entre 2 citas, 2 corchetes o 2 paréntesis.

## Cambiar un texto a mayúsculas/minúsculas
<a name="cambiar_texto_mayusculasminusculas_ELTTEXTE000591"></a>
Para cambiar automáticamente un texto encontrado en el editor de código a caracteres en mayúsculas o minúsculas: 

1. Seleccione una o más palabras en el editor de código. 

2. Pulse la siguiente combinación de teclas: 

	- Ctrl + M: para cambiar el texto seleccionado a minúsculas.

	- Ctrl + Mayúsculas + M: para cambiar el texto seleccionado a mayúsculas.

## Mover procedimientos o métodos
<a name="mover_procedimientos_metodos_ELTTEXTE000639"></a>
Los procedimientos o los métodos son mostrados en su orden de creación por Default. 
Para mover una Procedure o un método: 

1. Posicione el cursor al principio del código para moverlo.. 

2. Seleccione el código de Procedure o del método: utilice el ratón o pulse Ctrl + A por ejemplo. 

3. Cortar la Procedure o el método (Ctrl + X). 

4. Coloca el cursor del ratón en la nueva posición de Procedure o método.

5. Pegue la Procedure o el método (Ctrl + V).




La Procedure o método fue movido.

## Creando una Procedure con el teclado
<a name="creando_una_procedure_con_teclado_ELTTEXTE000663"></a>
Para crear una Procedure global, puedes: 

- usar el explorador de proyectos. 

- usar el panel "Código" del editor (botón "Nuevo"). 

- escribir el prototipo de la Procedure para crear al final de una Procedure existente.




Para crear una Procedure local, puedes: 

- usar el panel del "Proyecto Explorador". 

- usar el panel "Código" del editor (botón "Nuevo"). 

- teclee el prototipo de la Procedure para crear en las "Declaraciones Globales" Event de la ventana o página.




Para obtener más información, consulte: 

- [Procedimiento local](../Motscles/1513009.md).

- [Procedimiento interno](../Motscles/1514075.md).

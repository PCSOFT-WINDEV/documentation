## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
El glosario enumera todas las abreviaturas utilizadas en el editor de código. Cada abreviatura le permite acceder a una función, un procedimiento, un Property, ...

El glosario se puede personalizar por completo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000113"></a>


### Uso del glosario
<a name="uso_del_glosario_ELTPARAGRAPHE000027"></a>

Utilizar el glosario:

1. En la ventana de código, introduzca una de las abreviaturas del glosario.

	- si la entrada asistida está habilitada: el nombre de la función correspondiente se muestra automáticamente en la lista debajo de los caracteres escritos.

	- si la entrada asistida no está habilitada, pulsar F5: el nombre de la función correspondiente se muestra automáticamente en la lista debajo de los caracteres escritos.




2. Seleccionar y validar el texto correspondiente a la abreviatura introducida. La función, la Procedure o la Property associated con la abreviatura se escribe automáticamente en la posición del cursor.



<a name="NOTE2_2"></a>


### Gestión de un glosario
<a name="gestion_glosario_ELTPARAGRAPHE000040"></a>

Para conocer y/o modificar el contenido del glosario, seleccione "Código... Glosario" (Mayúsculas + F5).

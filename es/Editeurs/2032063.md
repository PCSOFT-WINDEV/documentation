<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000192"></a>
El editor de consultas permite crear consultas automáticamente en los archivos de datos. Crear una consulta es simple e intuitivo. La consulta se genera automáticamente (en código SQL optimizado) y puede probarse inmediatamente.

Sin problemas de programación: Las ventanas, páginas, tablas, cuadros combinados e informes encontrados en su proyecto pueden basarse en consultas.

## Diferencias entre el editor de consultas estándar (incluido en WINDEV) y la versión de usuario del editor de consultas
<a name="diferencias_entre_editor_consultas_estandar_incluido_windev_version_usuario_del_editor_consultas_ELTTEXTE000216"></a>
Existen dos versiones del editor de consultas:

- **el editor de consultas estándar incluido en el producto.**
	Esta versión se suministra con WINDEV, WEBDEV o WINDEV Mobile. Esta versión sólo se puede utilizar al desarrollar una aplicación WINDEV o WINDEV Mobile, o un sitio WEBDEV.

- **la versión de usuario del editor de consultas** (también llamado **Informes y consultas**). Este editor sólo se suministra con WINDEV.
	Esta versión es independiente de WINDEV y puede ser utilizada por los usuarios finales..
	Algunas características del editor de consultas no están disponibles en esta versión. El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=er.png) Icon indica las diferencias entre Informes y Consultas y el editor de consultas estándar.



<a name="NOTE2_2"></a>


### Editor de consultas estándar
<a name="editor_consultas_estandar_ELTPARAGRAPHE000036"></a>

El editor de consultas estándar permite crear consultas y ejecutarlas desde el editor o desde una aplicación desarrollada con WINDEV, WINDEV Mobile o WEBDEV.

El dongle de WINDEV, WINDEV Mobile o WEBDEV es necesario para utilizar el editor de consultas.. Debe estar conectado al puerto paralelo de su ordenador cuando utilice el editor.

Sin embargo, el usuario final no necesita un dongle (para ejecutar una consulta desde una aplicación WINDEV, WINDEV Mobile o WEBDEV).

El editor de consultas integrado incluye varios ejemplos. En estos ejemplos, descubrirá las principales consultas que se pueden utilizar en sus aplicaciones.
Para conocer la lista de ejemplos de utilización de informes: 

- Mostrar la página de inicio (Ctrl + <).

- Escriba la palabra clave "Reporte".

- Se visualiza la lista de ejemplos para utilizar informes.




Compruebe estos ejemplos, le enseñarán mucho y podrá utilizarlos cuando cree sus propias consultas!

### Versión de usuario del editor de consultas (Informes y Consultas - sólo WINDEV)
<a name="version_usuario_del_editor_consultas_informes_consultas_solo_windev_ELTPARAGRAPHE000059"></a>

Reports & Queries le permite crear y ejecutar consultas desde el editor (WINDEV no es necesario).

Las consultas creadas se pueden ejecutar desde el editor de consultas o desde una aplicación WINDEV (si esta aplicación soporta las consultas creadas por Reports & Queries).

No se requiere ningún dongle para utilizar Reports & Queries.

Algunas operaciones son necesarias para crear nuevas consultas o para modificar las consultas existentes con Reports & Queries. Para obtener más información, consulte [Distribución de informes y consultas](../Presentation/3088002.md).

### Documentación
<a name="documentacion_ELTPARAGRAPHE000075"></a>

Los informes y consultas y el editor de consultas estándar se suministran con la misma ayuda en línea.. 

La ayuda en línea está destinada a ayudarle a manipular el editor. Se propone la ayuda en línea:

- en el formato de una ayuda local. Los archivos necesarios se instalan en el ordenador de desarrollo.

- en el formato de una ayuda de INTERNET. Se puede acceder a esta ayuda desde cualquier ordenador equipado con un acceso INTERNET. Esta ayuda se actualiza periódicamente.




**Para acceder a la ayuda en línea**utilice uno de los siguientes métodos:

- en la pestaña "Inicio", en el grupo "Ayuda en línea", despliegue "Ayuda".

- pulse F1.




**Para modificar el tipo de ayuda en línea utilizada**:

1. En la pestaña "Herramientas", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de XXX".

2. En la pestaña "Ayuda" seleccione el modo de acceso a la base de datos de ayuda : "Use la base de datos de ayuda de INTERNET" o "Use la base de datos de ayuda local" así como el idioma preferido.

3. Valide.

### Instalación
<a name="instalacion_ELTPARAGRAPHE000122"></a>

- **Instalación del editor de consultas estándar**: El editor de consultas se instala automáticamente con WINDEV, WEBDEV o WINDEV Mobile.

- **Instalación de informes y consultas**: Para instalar Reports & Queries, siga las instrucciones dadas al instalar la aplicación WINDEV.




**Configuración mínima requerida para Reportes y Consultas**
Los informes y consultas deben instalarse en un ordenador con las siguientes características:

- 512 MB de RAM (se recomienda 1 GB o más).

- 800 MB de espacio en disco.

- Windows Vista, 7, 8 o posterior.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000229"></a>


### Creación del sistema de ayuda de una aplicación o sitio web
<a name="creacion_del_sistema_ayuda_una_aplicacion_sitio_web_ELTPARAGRAPHE000042"></a>

Para crear el sistema de ayuda de una aplicación o sitio web: 

1. Abra el proyecto en WINDEV, WEBDEV o WINDEV Mobile.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- En la ventana que aparece, haga clic en "Arquitectura" y luego en "Ayuda".

	- La creación del sistema de ayuda asistente se inicia automáticamente.




3. Vaya al siguiente paso de la asistente y elija el tipo de sistema de ayuda que desea generar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cr%E9er_l_aide%20-%20HC%20N%B0001.gif&type=thumb)


	- HTML: el sistema de ayuda será un sitio estático con un resumen y búsqueda de texto completo. 

- CHM: el sistema de ayuda utilizará el formato Microsoft HTML Help 1.3. 

- PDF: el sistema de ayuda será un PDF Document con un resumen y marcadores. 

4. Pase a la etapa siguiente. 

5. Especifique el nombre del sistema de ayuda. Pase a la etapa siguiente del asistente. 

6. Seleccione las ventanas de aplicación para las que debe generarse una ayuda Page.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cr%E9er_l_aide%20-%20HC%20N%B0002.gif&type=thumb)

	Si elige generar un archivo de ayuda CHM, puede seleccionar "Generar automáticamente el botón de ayuda en cada ventana" para crear automáticamente un botón de ayuda en cada ventana seleccionada. Este botón abrirá automáticamente la ayuda correspondiente Page. 

7. Pase a la etapa siguiente. 

8. Seleccione los informes de aplicación para los que debe generarse una ayuda Page. Pase a la etapa siguiente del asistente. 

9. Validar. El asistente crea el set de páginas de ayuda y le pide que genere el sistema de ayuda en el formato deseado. 

10. Grabe la ayuda: se crea un archivo WDH en su proyecto.




El sistema de ayuda se crea y aparece en el editor de ayuda. Cada Page puede ahora modificarse, suprimirse, ... en el editor de ayuda.

**Observación**: Una vez creado su sistema de ayuda, utilice uno de los siguientes métodos para abrir este sistema de ayuda en el editor de ayuda:

- en la pestaña "Inicio", en el grupo "General", despliegue "Abrir", seleccione "Abrir" y, a continuación, seleccione el archivo de ayuda correspondiente (archivo WDH).

- seleccione la carpeta "Ayudas" en el panel "Explorador de proyectos" y haga doble clic en el nombre del sistema de ayuda.

### Ver las propiedades del sistema de ayuda
<a name="ver_las_propiedades_del_sistema_ayuda_ELTPARAGRAPHE000132"></a>

Las propiedades del sistema de ayuda permiten definir:

- el nombre lógico del sistema de ayuda, 

- la Default Page que muestra la ayuda, 

- los idiomas soportados por el sistema de ayuda, 

- ...


Para ver las propiedades del sistema de ayuda, vaya a la pestaña "Modificación", grupo "Sistema de ayuda" y haga clic en "Descripción".

Para obtener más información, consulte [Propiedades del sistema de ayuda](../Editeurs/2010016.md).

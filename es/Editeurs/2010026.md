


## Insertar una Image o un vídeo en una ayuda Page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000200"></a>
Puede ser useful utilizar imágenes o vídeos en las páginas de ayuda para explicar las acciones a realizar, indicar qué botones hay que utilizar, etc.

Además de las imágenes "normales", puede insertar imágenes en las que se pueda hacer clic en las páginas de ayuda: cuando el usuario hace clic en la Image (o en una sección de la Image), se puede mostrar una ayuda específica Page (como una Page o como una ventana emergente).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestión de imágenes
<a name="gestion_imagenes_ELTTEXTE000224"></a>


### Insertar un Image
<a name="insertar_image_ELTPARAGRAPHE000020"></a>

Para insertar una Image en una ayuda Page:

1. En la página de ayuda, sitúe el cursor donde desee insertar la Image.

2. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Imagen". 

3. En el explorador de archivos, seleccione la Image que desea insertar en la ayuda Page y confirme. 

4. Si es necesario, confirme que desea copiar la Image en el directorio del proyecto. 

5. La imagen de la ventana de description aparece:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_image.gif&type=thumb)


6. Si es necesario, defina las zonas en las que se puede hacer clic en Image:

	- Haz clic en el botón "Nuevo".

	- Especifique el enlace associated con el área de clicks. Para obtener más información, consulte [Propiedades de un enlace](../Editeurs/2010022.md).

	- Defina el área clicable con el ratón o cambie los valores en "Parámetros del área de clic".


Para más detalles sobre esta ventana, véase [Propiedades de la imagen](../Editeurs/2010027.md).

7. Validar. La Image se incluye en la ayuda Page. Se puede cambiar el tamaño con las manijas de selección.




Observación: También puede seleccionar la opción "Insertar una imagen" en el menú contextual de la página de ayuda. 


### Insertar una imagen prediseñada
<a name="insertar_una_imagen_predisenada_ELTPARAGRAPHE000063"></a>

Para insertar imágenes prediseñadas en una ayuda Page:

1. En la ayuda Page, sitúe el cursor donde desee insertar la imagen prediseñada.

2. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Catálogo".

3. El catálogo Image se abre, permitiéndole seleccionar el Image a visualizar. 

4. Confirmar la Image seleccionada en el catálogo. 

5. La imagen de la ventana de description aparece:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_image.gif&type=thumb)


6. Si es necesario, defina las zonas en las que se puede hacer clic en Image:

	- Haz clic en el botón "Nuevo".

	- Especifique el enlace associated con el área de clicks. Para obtener más información, consulte [Propiedades del enlace](../Editeurs/2010022.md).

	- Defina el área clicable con el ratón o cambie los valores en "Parámetros del área de clic".


Para más detalles sobre esta ventana, véase [Propiedades de la imagen](../Editeurs/2010027.md).

7. Validar. La Image se incluye en la ayuda Page. Se puede cambiar el tamaño con las manijas de selección.



<a name="NOTE2_2"></a>


### Propiedades de la imagen
<a name="propiedades_imagen_ELTPARAGRAPHE000106"></a>

Para ver las propiedades de una Image:

1. Seleccionar imagen.

2. Seleccione "Propiedades de Image" en el menú contextual. 




Para obtener más información, consulte [Propiedades de la imagen](../Editeurs/2010027.md).

## Gestión de vídeos
<a name="gestion_videos_ELTTEXTE000260"></a>


### Insertar un vídeo
<a name="insertar_video_ELTPARAGRAPHE000125"></a>

Para insertar un vídeo en una ayuda Page:

1. En la ayuda Page, sitúe el cursor donde desee insertar el vídeo.

2. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Video". 

3. En la ventana que aparece, seleccione el vídeo que desea insertar en la ayuda Page y confirme. 
	Observación: El archivo de vídeo ruta puede corresponder a un vídeo accesible desde el ordenador actual (archivo mp4 o WebM) o al Address de un vídeo (YouTube u otro). 

4. Si es necesario, confirme que desea copy el vídeo al directorio del proyecto. 

5. El vídeo está integrado en la ayuda Page. Se puede cambiar el tamaño con las manijas de selección.




Observaciones: 

- Los vídeos sólo están disponibles para las páginas de ayuda HTM y PDF. 

- También puede seleccionar "Insertar un vídeo" en el menú contextual de la ayuda Page. 

- Puede cambiar el vídeo a través de las propiedades del vídeo (opción "Propiedades del vídeo" del menú contextual).

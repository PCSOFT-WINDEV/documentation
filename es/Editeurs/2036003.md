


## Traducción con Google Translate
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten soportar hasta 64 idiomas en el mismo proyecto.: ventana, reporte, WLanguage, etc.

Puedes traducir contenidos a posteriori con WDMSG o en los editores directamente.

Para mejorar el flujo de trabajo, los editores pueden utilizar Google Translate.

Atención: Por supuesto, debes cumplir las normas de uso del servicio de Google, que están sujetas a cambios con el tiempo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Implementación
<a name="implementacion_ELTTEXTE000168"></a>


### Uso de Google Translation en los editores
<a name="uso_google_translation_los_editores_ELTPARAGRAPHE000024"></a>

Para utilizar Google Translation en los editores: 

1. Mostrar las opciones generales del software utilizado: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile". 

2. Vaya a la pestaña "Traducción". 

3. Seleccione el motor "Google Translation. Especifique la clave API necesaria para utilizar la Traducción de Google. 

4. Validar. 



<a name="NOTE2_2"></a>


### Obtención de la clave API necesaria para utilizar la Traducción de Google
<a name="obtencion_clave_api_necesaria_para_utilizar_traduccion_google_ELTPARAGRAPHE000048"></a>

Para get la clave API necesaria para utilizar la traducción de Google: 

1. Debes tener una cuenta de Google. 

2. Accede a la consola API de Google para habilitar el servicio "Traducir API" ([https://code.google.com/apis/console](https://code.google.com/apis/console), enlace válido en el momento de redactar este Document). 

3. Aparece una nueva opción ("Acceso API"). Haga clic en esta opción. 

4. El "API Key" se genera y se muestra en la página. Esta clave debe introducirse en la ventana de opciones de WINDEV, WEBDEV y WINDEV Mobile.. 




Observaciones: 

- Los datos de facturación deben estar debidamente cumplimentados para la cuenta de Google.. De hecho, Google Translation no es un servicio gratuito. Si no se especifica esta información, se producirá un error al intentar realizar una traducción.. 

- Para más detalles sobre cómo obtener la clave API y cómo utilizar Google Translate, consulte [https://developers.google.com/translate/v2/faq](https://developers.google.com/translate/v2/faq)

- La versión de la traducción que ha utilizado API es la v2.

## Uso de Google Translation en los editores
<a name="uso_google_translation_los_editores_ELTTEXTE000198"></a>
Cuando la traducción a través de Google Translation está habilitada, la mayoría de los editores le permiten utilizar este modo de traducción. 

**Para usar Google Translation en los editores**: 

1. Una Icon (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Projet_multilangue%20-%20HC%20N%B0003%201.gif)) aparece en las diferentes áreas que se pueden traducir con Google Translate. 

2. Haga clic en esta Icon para ver las opciones de traducción del texto actual: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_trad_menu.gif&type=thumb)


3. Todo lo que tienes que hacer es seleccionar la opción correspondiente a la traducción a realizar.




Para una ventana de description, también puede realizar las siguientes operaciones: 

1. Haga clic en la sección que contiene el texto a traducir. 

2. Pulse el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BtnTraduire.GIF) en la parte inferior de la ventana. 

3. Aparece la ventana de traducción.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Google_trad_fen.gif&type=thumb)
 En esta ventana: 

	- Seleccione el idioma de referencia. 

	- Haga clic en el enlace "Solicitar traducción" para get la traducción a través de Google Translation en el idioma deseado. 




4. Seleccione las traducciones que deben aplicarse a la ventana description (marque las traducciones o haga clic en los botones "Todas" y "Ninguna").. 

5. Validar la ventana de traducción. La información se aplica automáticamente a la ventana de description.

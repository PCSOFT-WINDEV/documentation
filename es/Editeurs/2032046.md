


## Seleccione la ventana de consulta description: Uso 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
Veamos algunas operaciones específicas de la ventana de description de una consulta seleccionada:

- modificar el orden de visualización en la lista de elementos de consulta.

- modificar los valores mostrados en el resultado de la consulta.

- utilizar elementos procedentes de una consulta existente.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Consejos y trucos
<a name="consejos_trucos_ELTTEXTE000142"></a>


### Modificar el orden de visualización de los elementos de consulta de la lista
<a name="modificar_orden_visualizacion_los_elementos_consulta_lista_ELTPARAGRAPHE000022"></a>

Los elementos de la consulta aparecen en la sección central de la ventana de description ("Consultar description" en el menú contextual de la consulta).

El orden de visualización de los elementos de consulta en la lista corresponde a:

- el orden de visualización de los elementos en el resultado de la consulta: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OrdreAffichage.gif) (opción seleccionada por default).

- la secuencia de ordenaciones en el resultado de la consulta: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OrdreTri.gif).




**Observación**: Para modificar el orden de visualización y/o el orden de clasificación de los elementos en el resultado de la consulta, utilice los botones de flecha que se encuentran a la derecha de la lista de elementos.
<a name="NOTE2_3"></a>


### Modificar los valores mostrados en el resultado de la consulta
<a name="modificar_los_valores_mostrados_resultado_consulta_ELTPARAGRAPHE000040"></a>

Usted tiene la capacidad de mostrar:

- el año a partir de una posición de fecha.

- el mes a partir de una posición de fecha.

- el estado de un artículo que contiene un código postal.




Modificar los valores mostrados en el resultado de la consulta:

1. Seleccione un elemento de fecha o un elemento que contenga un código postal.

2. Haga clic en el botón "Año, estado".

3. Select:

	- "Año" para mostrar el año en el resultado de la consulta.

	- "Estado" para mostrar el estado en el resultado de la consulta.




4. Valide.



<a name="NOTE2_4"></a>


### Uso de los elementos procedentes de una consulta existente
<a name="uso_los_elementos_procedentes_una_consulta_existente_ELTPARAGRAPHE000064"></a>

Los elementos que se pueden utilizar en una consulta pertenecen a:

- un archivo del análisis associated con la aplicación actual.

- una consulta de la aplicación actual.




El método para seleccionar un elemento perteneciente a un fichero de análisis es idéntico al método para seleccionar un elemento perteneciente a una consulta.


## Tienda privada para aplicaciones WINDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000222"></a>
Las versiones 27 y posteriores incluyen una Tienda privada. Esta tienda permite agrupar las aplicaciones utilizadas dentro de una empresa. 
Tienda privada puede ser accesible sólo desde dentro de la organización, o ser de acceso público. . 
Un portal con autenticación opcional permite que los usuarios autorizados instalen aplicaciones. 
Esto permite estandarizar y simplificar la distribución de aplicaciones dentro de una empresa. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Cómo instalar y utilizar una Tienda privada
<a name="como_instalar_utilizar_una_tienda_privada_ELTTEXTE000246"></a>
Para set la Tienda privada de una aplicación WINDEV: 

1. [Etapa 1: Configure su servidor si es necesario](#NOTE2_2).

2. [Etapa 2: Genere su solicitud](#NOTE2_3).

3. [Etapa 3: Instalar la aplicación en el Tienda privada](#NOTE2_4). 



<a name="NOTE2_2"></a>


### Etapa 1: Configure el servidor
<a name="etapa_1_configure_servidor_ELTPARAGRAPHE000037"></a>

Pueden presentarse dos casos: 

- **Si utiliza una plataforma PCSCloud**: los elementos necesarios se instalan de forma predeterminada en su plataforma. 
	Recordatorio: PCSCloud es una solución paga lista para usar. Para obtener más información, consulte [PCSCloud](https://pcscloud.net/UK/cloud_development.awp).

- **Si no utiliza una plataforma PCSCloud**: instale los siguientes elementos: 

	- un servidor HFSQL,

	- el servidor reservado, incluidos los módulos para el Groupware Usuario de dispositivos móviles (WDBaaS).


 Para obtener más información, consulte [Configuración de un servidor BaaS (Groupware Usuario y Tienda privada)](../Editeurs/1000021311.md).



<a name="NOTE2_3"></a>


### Etapa 2: Generar la aplicación instalación
<a name="etapa_2_generar_aplicacion_instalacion_ELTPARAGRAPHE000059"></a>

Las configuraciones de aplicación para Tienda privada pueden generarse del mismo modo que las configuraciones de aplicación estándar.. Sólo algunas opciones son diferentes. 

Para generar el instalación de su aplicación en el Tienda privada: 

1. En la pestaña "Proyecto", en el grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación".

2. El asistente que aparece le permite crear el ejecutable y el instalación de su aplicación. 

3. Si aún no ha creado el ejecutable, siga los pasos correspondientes. 

4. Haga clic en "Configurar" y seleccione "Configurar en una Tienda privada".. Pase a la etapa siguiente. 

5. Introduzca los parámetros de la aplicación Tienda privada: 

	- Nombre de la solicitud,

	- Visibilidad de la aplicación (privada o public): 

		- Aplicación public en la Tienda privada: Cualquier usuario conectado a Tienda privada puede descargar e instalar la aplicación. 

		- Aplicación private en la Tienda privada: sólo los usuarios autentificados pueden acceder a la aplicación del programa instalación. 




- description y la imagen associated con la aplicación. 

6. Pase a la etapa siguiente. 

7. Indicar las propiedades específicas del ejecutable de referencia, así como la description de la versión para la Tienda privada. 

8. Pase a la etapa siguiente. 

9. Los siguientes pasos son los mismos que para generar la instalación de una aplicación estándar. Para obtener más información, consulte [Asistente de instalación: Los pasos para crear la instalación de una aplicación WINDEV](../Editeurs/2028062.md). 

10. Al final del asistente, puede elegir cómo generar el programa instalación. Para las aplicaciones desplegadas en Tienda privada, sólo se puede generar un ejecutable autoextraíble.

11. Haga clic en "Finalizar". El programa de instalación se genera.



<a name="NOTE2_4"></a>


### Etapa 3: Instalar la aplicación en el Tienda privada
<a name="etapa_3_instalar_aplicacion_tienda_privada_ELTPARAGRAPHE000102"></a>

Una vez generado el programa de aplicación instalación, se puede ejecutar: 

- directamente ("Prueba"). 

- ejecutando el programa "exe", situado en el subdirectorio "Install_xxx" del directorio del proyecto.. 




Al instalar la aplicación: 

1. Si es necesario, seleccione el idioma instalación. 

2. Especifique las opciones de conexión de Tienda privada: 

	- El Address del servidor BaaS utilizado. 

	- Las credenciales de acceso del supervisor. 


 Esta información fue proporcionada durante la instalación del servidor reservado. 

- Si está utilizando una plataforma PCSCloud, el servidor Address es el nombre de su plataforma.

- Se puede acceder al sitio de administración de Tienda privada en http(s):///WDBaaS. Para conectarse por primera vez, utilice "admin" como nombre de usuario e introduzca la contraseña especificada en el archivo instalación.

3. Especifique los parámetros de la aplicación instalación: 

	- Firma de la instalación

	- Integración de un archivo de configuración personalizado. 

	- Transición de un instalación en la red a un instalación en el Tienda privada: Indique si la red instalación de su aplicación debe estar disponible en la Tienda privada. En este caso, las versiones antiguas instaladas en los ordenadores de los usuarios se actualizarán automáticamente para utilizar Tienda privada. 




4. Validar. La aplicación se instala en el servidor Tienda privada. Puede compartir el enlace Tienda privada con los usuarios de su aplicación.

### Utilización y configuración de Tienda privada
<a name="utilizacion_configuracion_tienda_privada_ELTPARAGRAPHE000134"></a>

Para verificar y configurar su aplicación en Tienda privada: 

1. Abrir Tienda privada: Se puede acceder al sitio de administración de Tienda privada en http(s):///WDBaaS. 

2. Se mostrarán las aplicaciones desplegadas con visibilidad set a "Público" (visibilidad set en la creación instalación asistente). 

3. Para configurar las opciones del Tienda privada, haga clic en "Conectarse".. 

4. Para conectarse por primera vez, utilice "admin" como nombre de usuario, e introduzca la contraseña especificada en el instalación ("admin" por Default). Una vez conectado, puede cambiar la contraseña si es necesario.. 

5. Aparece el catálogo de aplicaciones. Haga clic en "Administración".. 

6. El sitio de administración contiene: 

	- Las opciones generales de Tienda privada, a las que se accede a través de un menú situado en la parte superior de Page. 

	- Las diferentes aplicaciones en el Tienda privada, a la izquierda. puede hacer clic en el nombre de una aplicación para ver sus opciones. 







**1. Opciones generales de la Tienda privada.**
Las opciones generales de Tienda privada son: 

- Gestión de usuarios y grupos. Puede añadir usuarios y set un usuario específico como administrador (supervisor). Una vez creado el usuario, puede editar su perfil e indicar si está autorizado a añadir aplicaciones Windows a Tienda privada. Esta opción es particularmente useful para los Almacenes private de prueba o para los Almacenes private específicos de los equipos de desarrollo. 

- Configuración SMTP: Esta opción permite configurar el servidor SMTP para enviar correos electrónicos a los usuarios. 

- Opciones: Este Page contiene todas las opciones de personalización y configuración del Tienda privada: 

	- Etiqueta Tienda privada, que aparece en el banner del sitio (por Default, "Catálogo de aplicaciones"). 

	- Agregar aplicaciones Windows: Permite o impide que los usuarios añadan aplicaciones a Tienda privada (también puede configurar esta opción para cada usuario).. 

	- Creación de una cuenta: Especifica los derechos para crear cuentas: todos los usuarios, ninguno o sólo algunos usuarios. 

	- Modo de activación de los usuarios: automatic o por Email. 

	- Verificación en dos pasos: Obligatorio o habilitado sólo por el usuario. Puede crear un mensaje que pida al usuario que Contact a un administrador si no puede get el código. 

	- Claves reCAPTCHA: Permite especificar los detalles de reCAPTCHA v3 (disponibles en la consola de administración de Google).. 

	- Token restablecer: Se proporciona una conexión Token cuando se instala el Servidor de Aplicaciones. Puede volver a generar esta Token en caso de pérdida.   







**2. Opciones de la aplicación.**
Las opciones de la aplicación Windows le permiten: 

- Obtener el número de versión reference de la aplicación y la versión mínima requerida. También puede determinar si la aplicación se ha descargado. 

- Transformar una aplicación public en una aplicación private. Permite definir los derechos de descarga: 

	- Descarga disponible para todos. 

	- Descarga disponible sólo para usuarios conectados. 

	- Descarga disponible sólo para algunos usuarios o grupos.

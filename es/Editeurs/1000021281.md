## ¿Cómo desconectarse del Groupware Usuario?
<a name="¿como_desconectarse_del_groupware_usuario_ELTTEXTE000101"></a>
Objetivo: Crear un botón para cerrar la sesión del Groupware Usuario y volver a la página de inicio de sesión.

Solución: Para cerrar la sesión, utilice la función [gpwDisconnectUser](../WDLang6/1000021100.md). 

- En modo integrado, la desconexión conserva los permisos del usuario no conectado.

- En modo no integrado, para conservar los permisos:

	- Cree un usuario "no conectado", por ejemplo. 

	- Defina los permisos. 

	- En el código, utilice: 
			
		```wl
		gpwDisconnectUser()
		gpwLogin("not connected", "password")
		// Refresh the current page
		PageDisplay()
		```




## Selección de uno o más controles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000181"></a>
Se pueden seleccionar uno o más controles en una ventana, en una página o en un reporte mediante el teclado o el ratón.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo se selecciona un control?
<a name="¿como_selecciona_control_ELTTEXTE000205"></a>
Para seleccionar una control, sólo tienes que hacer clic en la control para seleccionarla.

El control seleccionado se muestra con asas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Selection_champs_poignees.gif)


El nombre del control seleccionado aparece en la barra de estado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NomChampBarreEtat.gif)

## ¿Cómo seleccionar varios controles?
<a name="¿como_seleccionar_varios_controles_ELTTEXTE000229"></a>


### Selección de varios controles con la tecla Ctrl
<a name="seleccion_varios_controles_con_tecla_ctrl_ELTPARAGRAPHE000033"></a>

Para seleccionar varios controles al mismo tiempo con la tecla Ctrl:

1. Presione la tecla Ctrl y manténgala presionada.

2. Seleccione el control s con el mouse (haga clic en cada control con el mouse).

3. Suelte la tecla Ctrl. Los controles seleccionados se muestran con manijas. 




**Observaciones**: 

- Para añadir un control a una selección existente o para borrar un control de una selección existente, mantenga pulsada la tecla Ctrl y haga clic en el control que debe añadirse o borrarse.

- Cuando se muestra la ventana de 7 pestañas ("Descripción" de el menú contextual de la selección), sólo se muestran las opciones comunes a los controles seleccionados.




- Los diferentes tipos de mangos se presentan en las Notas.



<a name="NOTE3_2"></a>


### Selección de varios controles con el lazo del ratón
<a name="seleccion_varios_controles_con_lazo_del_raton_ELTPARAGRAPHE000053"></a>

El lazo del ratón corresponde a un área de selección. Esta área está representada por un rectángulo punteado. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LassoSouris.gif)


Para seleccionar varios controles al mismo tiempo:

1. Haz clic al lado de la primera control para seleccionar y mantén el botón izquierdo del ratón pulsado.

2. Mueva el cursor del ratón para seleccionar todos los controles: el área de selección está representada por un rectángulo punteado (el lazo del ratón).

3. Suelte el botón del ratón cuando se realice la selección.. 




**Observaciones**:

- Por Default, todos los controles con una sección incluida en el área del lazo serán seleccionados. Para seleccionar sólo los controles que están totalmente incluidos en el área de lazo, mantenga pulsada la tecla Mayúsculas durante los pasos de selección.

- Para añadir un control a una selección existente o para borrar un control de una selección existente, mantenga pulsada la tecla Ctrl y haga clic en el control que debe añadirse o borrarse.

- Cuando se muestra la ventana description ("description" de el menú contextual de la selección), sólo se muestran las opciones comunes a los controles seleccionados.

## ¿Cómo Cancel la selección de varios controles?
<a name="¿como_cancel_seleccion_varios_controles_ELTTEXTE000259"></a>
A Cancel la selección de varios controles, puede:

- haga clic en un control no seleccionado,

- haga clic en la Background de la ventana, página o reporte,

- haga clic en la barra de título de la ventana, página o reporte.

## Notas
<a name="notas_ELTTEXTE000283"></a>


### Manijas de controles
<a name="manijas_controles_ELTPARAGRAPHE000091"></a>

Cuando se acaba de crear una control o cuando se selecciona, se muestra con asas. Las asas están acostumbradas:

- ver el tamaño de un control,

- modificar el tamaño de un control.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PoigneesNoires.gif)
 <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PoigneeGriseBlanche.gif)
 
Los colores de los mangos se utilizan para identificar las acciones que se pueden realizar:

- Las manijas **negra** se usan para ver y cambiar el tamaño de un control.

- El **White maneja** sólo puede ser usado para ver un control. Por ejemplo, cuando se seleccionan varios controles, los mangos de White indican que los controles seleccionados no pueden ser redimensionados.
	Este tipo de tiradores también se muestra cuando una ventana está en modo de sólo lectura para el desarrollador.: los controles no pueden modificarse.

- El **gray maneja** indica el primer control seleccionado durante una selección múltiple. Este control será el reference control.



<a name="NOTE5_2"></a>


### Caso especial: seleccionando un control superpuesto
<a name="caso_especial_seleccionando_control_superpuesto_ELTPARAGRAPHE000114"></a>

Los controles se pueden superponer en una ventana o en una página.

Para seleccionar fácilmente un control superpuesto:

- **1er método**: Haga clic en el hexadecimal superpuesto control s sucesivamente hasta que seleccione el control solicitado.
	**Observación**: El nombre del control seleccionado aparece en la barra de estado.

- **2º método**:

	- Haga clic en los controles superponibles mientras mantiene pulsada la tecla Mayúsculas. El nombre de los controles superpuestos se muestra en un menú emergente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=champSuperpose.gif)


	- Seleccione la control en el menú contextual.



<a name="NOTE5_3"></a>


### Caso especial: seleccionando un control que pertenece a los control s agrupados
<a name="caso_especial_seleccionando_control_que_pertenece_los_control_agrupados_ELTPARAGRAPHE000132"></a>

Cuando se selecciona un control que pertenece a un set de control agrupados, todos los control agrupados se seleccionan automáticamente.

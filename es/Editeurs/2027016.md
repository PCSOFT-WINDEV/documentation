## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
El panel "Análisis UML" muestra los diferentes diagramas UML (diagrama de clase, diagrama de implementación, ...) del proyecto actual.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletAnalyseUML.gif)
 TEl menú emergente (clic derecho) del panel "Análisis UML" le permite:

- Crear un nuevo modelo UML. El modelo UML asociado al proyecto puede contener varios diagramas UML.

- Abrir un modelo UML.

- Abra el modelo UML del proyecto actual.

- Muestra la descripción de los diagramas.

- Abrir y cerrar los diagramas.

- Renombrar los diagramas.

- Borrar los diagramas.




Si no hay ningún modelo UML asociado al proyecto actual, el panel "Análisis UML" propone crear un nuevo modelo.




## Configuración de los menús de la AAF (Características de la aplicación automatic)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000220"></a>
WINDEV incluye varias funciones de aplicación de automatic (AAF). Estas AAF permiten a los usuarios finales imprimir y exportar a Excel el contenido de una control Tabla, mostrar una Calendar al introducir una fecha, etc.. 

En la mayoría de los casos, los menús contextuales específicos se relacionado a las AAF. Estos menús contextuales aparecen en las ventanas, los controles Edición, los controles Image, etc.

Puede definir fácilmente las distintas opciones del menú contextual en el editor.

También se puede añadir un menú contextual personalizado al menú contextual del AAF. Este menú contextual puede colocarse **antes de** o **después de** el menú contextual AAF.

**Observación**: Se muestra un menú AAF para los siguientes controles: 

- En WINDEV: Organizador, Botón, Editar control, Cuadro combinado, Diagrama de Gantt, Ventana, Gráfico, Image, Casilla de verificación, Cuadro de lista, Organigrama, Programador, Botón de radio, Tabla, Pivote tabla, TimeLine y Looper.

- En WINDEV Mobile: Casilla combinada, Casilla de verificación, Casilla de lista, Botón de radio, Tabla y Looper.

## Configuración de los menús contextuales de los AAF
<a name="configuracion_los_menus_contextuales_los_aaf_ELTTEXTE000244"></a>


### Configuración a nivel de proyecto
<a name="configuracion_nivel_proyecto_ELTPARAGRAPHE000029"></a>

Con Default, todas las opciones de los menús contextuales de AAF están disponibles para todos los tipos de controles de un proyecto.

Para ocultar algunas opciones del menú contextual de AAF para algunos tipos de controles:

1. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Visualizar:

	- la pestaña "Avanzado" en WINDEV.

	- la pestaña "Estilo" en WINDEV Mobile.




3. Haga clic en "Menú de las FAA". La ventana que aparece permite configurar las opciones que propondrá Default en el menú contextual AAF para cada tipo de control.

4. Para cada tipo de control, seleccione las opciones a mostrar.

5. Validar la ventana de ajustes y el proyecto description.




**Observación**: El menú AAF puede configurarse para los siguientes controles: 

- En WINDEV: Organizador, Botón, Editar control, Cuadro combinado, Ventana, Image, Casilla de verificación, Cuadro de lista, Programador, Botón de radio, Tabla, Pivote tabla, TimeLine y Looper.

- En WINDEV Mobile: Casilla combinada, Casilla de verificación, Casilla de lista, Botón de radio, Tabla y Looper.



<a name="NOTE2_2"></a>


### Configuración a nivel de control
<a name="configuracion_nivel_control_ELTPARAGRAPHE000068"></a>

Se puede elegir un menú contextual para cada tipo de control. El menú contextual "Sistema" es propuesto por Default. Este menú contextual "Sistema" contiene las diferentes opciones del AAF.

Para configurar el menú contextual de los controles: 

1. Abra la ventana de descripción del control.

2. Vaya a la pestaña "UI".

3. El cuadro combinado "Menú contextual" permite configurar el menú contextual:

	- el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Fleche.gif) se utiliza para crear un nuevo menú contextual.

	- al que está acostumbrado el cuadro combinado:

		- mostrar el menú contextual AAF y set las opciones que se mostrarán para el control actual.

		- mostrar un menú contextual específico. Este menú contextual puede aparecer antes o después, o incluso sustituir al menú contextual del AAF




4. Valide.




**Observación**: El menú AAF puede configurarse para los siguientes controles: Botón, Editar control, Cuadro combinado, Ventana, Image, Casilla de verificación, Cuadro de lista, Organigrama, Botón de radio, Tabla, Pivote tabla, Línea de tiempo, Looper y Gráfico. En el caso de la control Gráfico, el menú contextual puede mostrarse u ocultarse y puede aparecer un menú personalizado antes o después de. Para configurar las opciones que aparecerán en el menú contextual de un gráfico, utilice [grMenuOpción](../WDLang3/3042042.md).
<a name="NOTE2_3"></a>


### Desactivar el menú contextual del AAF
<a name="desactivar_menu_contextual_del_aaf_ELTPARAGRAPHE000100"></a>

Para desactivar el menú contextual del AAF:

- en el editor de ventanas: en la descripción del control (pestaña "UI"), deseleccione la opción "Mostrar el menú de las FAA (Sistema)" en la opción "Menú Cont.".

- utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

## ¿Cómo traducir los menús contextuales de AAF?
<a name="¿como_traducir_los_menus_contextuales_aaf_ELTTEXTE000280"></a>
En una aplicación multilingüe, los menús contextuales AAF deben traducirse, así como todos los subtítulos y mensajes de la aplicación. WDINT es necesario para traducir este tipo de menú.

**Recordatorio**: WDINT es una herramienta adicional. Esta herramienta crea un archivo específico (extensión.WDM) que contiene todas las traducciones de los recursos encontrados en WINDEV/WEBDEV/WINDEV Mobile. Esta herramienta se suministra con WDMSG para extraer los subtítulos para traducirlos y reintegrarlos a su aplicación.. Póngase en contacto con el Departamento de ventas de PC SOFT para más información sobre las condiciones de uso de este producto. Por Default, todos los recursos en inglés y francés se distribuyen con WDINT.
<a name="NOTE3_2"></a>


### Traducción de los menús contextuales de Default
<a name="traduccion_los_menus_contextuales_default_ELTPARAGRAPHE000135"></a>

Para traducir los menús contextuales de Default: 

1. Instale WDINT si aún no lo ha hecho.

2. Iniciar WINDEV. En la pestaña "Proyecto", en el grupo "Traducir", despliegue "Traducir" y seleccione "WDInt - Traducción del framework WINDEV". 

3. En el WDINT asistente, especifique el idioma de los ficheros a extraer así como su nombre, y el ruta del fichero a generar. Este archivo contendrá los mensajes a traducir en el idioma de extracción seleccionado.. 

4. Traducir recursos con WDTRAD (incluido con WDINT).

5. Reintegrar el archivo de traducción. 

6. Generar el archivo WDM que corresponde a la lengua deseada. 

7. copy el archivo WDM a su directorio de aplicaciones.

8. En WINDEV, abre la ventana del proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

9. En la pestaña "Idiomas", especifique el archivo WDM que contiene las traducciones deseadas (pestaña "Varios").

10. Valide la ventana de descripción del proyecto. 

11. Cuando la aplicación se ejecute en el idioma de destino, los menús contextuales aparecerán en ese idioma.

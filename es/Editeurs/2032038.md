## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
<a name="Query_description"></a>
Una vez creada una consulta, todas sus características pueden ser modificadas en cualquier momento:

- archivos y/o elementos utilizados,

- elementos mostrados en el resultado de la consulta,

- condiciones de selección, 

- tipo de registros,

- ...




Sólo el tipo de consulta no puede modificarse. De hecho, se tienen en cuenta demasiados parámetros específicos. Para cambiar el tipo de consulta, todo lo que tiene que hacer es crear una nueva consulta.

## Ventana de descripción
<a name="ventana_descripcion_ELTTEXTE000126"></a>
Las principales características de una consulta se pueden acceder en la ventana description de la consulta. El editor de la consulta propone tres ventanas description:

- [Ventana de description de una consulta seleccionada](../Editeurs/2032049.md).

- [Ventana de description de una consulta de inserción, actualizar consulta, borrar consulta](../Editeurs/2032036.md)o en código SQL.

- [Ventana description de una consulta combinada](../Editeurs/2032031.md).


Para mostrar la ventana de description de una consulta:

- haga doble clic en la representación gráfica de la consulta.

- seleccione "Consulta description" en el menú contextual del editor de consultas.

- en la pestaña "Consulta", en el grupo "Edición", haga clic en "Descripción".

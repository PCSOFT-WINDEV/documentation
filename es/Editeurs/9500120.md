## Presentación

### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000011"></a>

Se pueden utilizar varios métodos para posicionar automáticamente el cursor de edición en un Line específico en un control Campo de entrada:

- Seleccione "Ir a Line..." en el menú contextual de control. 

- Presione Ctrl + G.

- Corre [AAFExecute](../WDLang1/1000022099.md):
	
	```txt
	AAFExecute(<Edit control>, aafGotoLine)
	```
La siguiente ventana se abre:

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Atteindre_Ligne.gif)
 El usuario puede escribir el número de la Line solicitada y validar.

### Modificando la ventana de selección de Line
<a name="modificando_ventana_seleccion_line_ELTPARAGRAPHE000033"></a>

Por Default, la ventana de selección de Line está disponible en inglés y en francés. 

Para modificar esta ventana o traducirla, simplemente inclúyala en el proyecto: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV y dependencias".

2. Seleccionar el directorio "Programas\\Datos\\Ventanas de preselección\\AAF".. 

3. Seleccione la ventana "WinDevAaf_ReachLineWindow.wdw".. 

4. Valide la importación. La ventana está incluida en su proyecto y puede ser traducida o modificada.. 




Observaciones:

- No se debe cambiar el nombre de la ventana.

- Si agrega esta ventana a la aplicación, es posible que no obtenga las mejoras realizadas en ella durante las diferentes actualizaciones. No olvide comprobar si esta ventana ha evolucionado.

- Atención: Para modificar la ventana, el archivo correspondiente no debe ser de solo lectura. Modifique las características del archivo en Windows si es necesario.

## Presentación
<a name="presentacion_ELTTEXTE000109"></a>


### Antes de implementar su sitio WEBDEV dinámico, le recomendamos que ejecute las pruebas finales desde un servidor de pruebas.
<a name="antes_implementar_sitio_webdev_dinamico_recomendamos_que_ejecute_las_pruebas_finales_desde_servidor_pruebas_ELTPARAGRAPHE000011"></a>

ATENCIÓN: El motor implementación para la prueba NO DEBE ser instalado en un ordenador donde esté instalada la versión de Desarrollo WEBDEV o la versión de Despliegue WEBDEV.

Las limitaciones del motor de implementación para la prueba en comparación con el motor real de implementación son las siguientes:

- El número máximo de conexiones (para todas las aplicaciones WEBDEV instaladas en el equipo) no puede exceder de 10.

- El motor de implementación para la prueba NO funciona en modo TEST (lo que significa que este motor no funciona con el editor de WEBDEV)

- El administrador de WEBDEV tiene la siguiente mención: "Motor de prueba".

- En la barra de título del navegador se muestra una mención que indica que la aplicación se utiliza con un motor de prueba.: "Motor de prueba; cualquier uso que no sea la prueba puntual es ilegal"




**Elementos que deben instalarse en el "Servidor de prueba".**

- Su sitio dinámico de WEBDEV (instalación por FTP o por medios físicos). Para obtener más información, consulte [Sitio instalación](../Editeurs/2028039.md).

- Un navegador INTERNET.

- El motor "Test" de WEBDEV. Este motor está disponible en el paquete instalación de WEBDEV "Versión de Desarrollo".

- Un servidor web (el servidor Apache se proporciona con WEBDEV "Versión de Desarrollo").

- El protocolo TCP/IP.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000133"></a>


### Probar un sitio dinámico en un servidor de prueba
<a name="probar_sitio_dinamico_servidor_prueba_ELTPARAGRAPHE000038"></a>

Se pueden utilizar varios métodos para probar el proyecto en un servidor de pruebas:

- Método 1:

	1. Iniciar el administrador de WEBDEV: en la pestaña "Herramientas", en el grupo "Web utilities", haga clic en "WDAdmin". 

	2. Haga clic en el botón "Página de prueba" (pestaña "Avanzado").




- 2º método: (recomendado)
	Conéctese al sitio introduciendo lo siguiente INTERNET Address: 
	
	```txt
	http://<IP address of test server>/WD280AWP/WD280AWP.EXE/CONNECT/<Project name>
	```




## Borrar patches
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
Para modificar rápidamente el modo de funcionamiento de una aplicación, puede utilizar [parches de aplicación](../Editeurs/2030031.md). Al instalar la aplicación, puede borrar las bibliotecas de parches existentes. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Eliminar bibliotecas de patch
<a name="eliminar_bibliotecas_patch_ELTTEXTE000110"></a>
Las librerías de parches están configuradas en el editor de instalación WDInst. Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

Para borrar las bibliotecas de parches: 

1. Seleccione "Parámetros de configuración .. Borrar parches". 

2. Mediante el botón "Añadir", seleccione las bibliotecas de parches que deben eliminarse del ordenador cliente..  

3. Validar.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000087"></a>


### Eliminar un proyecto de Administrador de Código Fuente
<a name="eliminar_proyecto_administrador_codigo_fuente_ELTPARAGRAPHE000011"></a>

Para eliminar un proyecto de Administrador de Código Fuente:

1. Abrir proyecto.

2. Abrir el proyecto description: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

3. En la pestaña "SCM", selecciona "Sin groupware de desarrollador y sin SCM" (o "Trabajo en equipo utilizando el groupware de desarrollador" según tus necesidades).

4. Valide.




**Observación**: Si selecciona de nuevo la opción "Utilizar el SCM (solo o en equipo)", el botón "Parámetros" propone diferentes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GDS_Enlever%20-%20HC%20N%B0001.gif)


- Agregar el proyecto a un repositorio: Abre la asistente para importar un proyecto a un repositorio

- Volver a conectar el proyecto con su repositorio: Vuelve a conectar el proyecto con el depósito correspondiente.

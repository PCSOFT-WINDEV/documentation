


## Oscurecimiento de ventanas inactivas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000239"></a>
¿Su aplicación abre varias ventanas al mismo tiempo y no sabe cuál utilizar?

Las ventanas inactivas pueden oscurecerse automáticamente. Así, podrá saber qué ventana utilizar.

Esta funcionalidad se conoce como Oscurecimiento de Ventanas Inactivas (OVI).

Por ejemplo: <br>![Modifica la información relativa a un cliente
y haga clic en el botón "Validar".](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Grisage%20automatique%20des%20fenetres%20-%20HC%20N%B0001.gif)<br>![Una ventana le pide que confirme o cancele los cambios.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000263"></a>


### OVI para todas las ventanas de la aplicación
<a name="ovi_para_todas_las_ventanas_aplicacion_ELTPARAGRAPHE000038"></a>

Por Default, la función OVI está habilitada para todas las ventanas de la aplicación. Cuando se crea un nuevo proyecto, esta opción es marcada por Default.

Para habilitar la función OVI en todas las ventanas de la aplicación:

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. En la pestaña "Avanzado" seleccione la opción "Activar OVI (Oscurecimiento de Ventanas Inactivas) para oscurecer las ventanas cuando están inactivas". 

3. Valide.




**Observaciones**:

- Si la opción "Activar OVI (Oscurecimiento de Ventanas Inactivas) para oscurecer las ventanas cuando están inactivas" está desactivada en el proyecto, el OVI se desactivará en tiempo de ejecución, incluso en las ventanas en las que esta opción está seleccionada.

- Si la opción "Activar OVI (Oscurecimiento de Ventanas Inactivas) para oscurecer las ventanas cuando están inactivas" está activada en el proyecto, el OVI se activará en tiempo de ejecución, incluso en las ventanas en las que esta opción está seleccionada.



<a name="NOTE2_2"></a>


### OVI en una ventana específica
<a name="ovi_una_ventana_especifica_ELTPARAGRAPHE000080"></a>

Por Default, la función OVI está habilitada para todas las ventanas de la aplicación. Al crear una nueva ventana, la función OVI está habilitada para esta ventana.

Para habilitar la función OVI en una ventana específica:

1. Abre la ventana para la que debe estar activada la función OVI.

2. Vaya a la pestaña "UI" de la ventana description ("description" en el menú contextual).

3. Seleccione "Oscurecer ventana cuando se vuelve inaccesible (O.V.I.)" y valide.




**Observación**: Esta función puede activarse o desactivarse:

- por el usuario final a través de "Gris la ventana si es Inactive (OVI)" desde el menú contextual de la ventana.

- mediante programación con la función [WinActivateDDW](../WDLang1/3038056.md).



<a name="NOTE2_3"></a>


### Programación
<a name="programacion_ELTPARAGRAPHE000105"></a>

Por Default, la función OVI está habilitada para todas las ventanas de la aplicación.

Se pueden utilizar varias funciones de WLanguage para configurar la función OVI en una aplicación:


|   |   |
| --- | --- |
| [AAFDisable](../WDLang1/1000022018.md) | Utilizada con a la constante *aafRateDDW*, desactiva la posibilidad de que el usuario final modifique el nivel de OVI. Las opciones de menú correspondientes no se vuelven a mostrar y los atajos Ctrl + Alt + signo "+" o Ctrl + Alt + signo "-" del teclado número no funcionan más. |
| [WinActivateDDW](../WDLang1/3038056.md) | Activa o desactiva la función OVI. |
| [WinDisableEffect](../WDLang1/1000017172.md) | Desactiva los efectos visuales del motor gráfico de WINDEV. Esta función permite que una aplicación se ejecute en un ordenador con capacidades gráficas limitadas o que se ejecute en modo TSE si el ancho de banda es limitado.. Se puede desactivar el OVI mediante la constante *effectDDW*. |
| [WinForceDDW](../WDLang1/1000019890.md) | Fuerza la función OVI en una ventana, incluso si esta ventana no es inaccesible. |
| [WinInactiveEffect](../WDLang1/3038055.md) | Activa o desactiva la función OVI. |
| [WinRateDDW](../WDLang1/3038057.md) | Permite conocer y modificar el nivel de gris de las ventanas en las que se aplica la función OVI. |

## Función OVI en los ordenadores de los usuarios
<a name="funcion_ovi_los_ordenadores_los_usuarios_ELTTEXTE000299"></a>


### Configuración de los ordenadores del usuario final
<a name="configuracion_los_ordenadores_del_usuario_final_ELTPARAGRAPHE000151"></a>

Dependiendo de la configuración de los ordenadores del usuario final, la función OVI puede no tener ningún efecto.

De hecho, la función OVI se ve afectada por las características de la tarjeta de vídeo, la velocidad del procesador, ....

Si se produce un problema, pida a los usuarios finales que actualicen el controlador de su tarjeta de vídeo.

**Observación**: Para funcionar, los ordenadores de los usuarios finales deben:

- tienen un modo de visualización superior a 256 colores.

- estar en modo de 32 ó 64 bits.

- estar ejecutando Windows 2000 o posterior.

- utilizar el nivel máximo de "Aceleración de hardware" para la tarjeta de vídeo (propiedades de visualización avanzadas de el equipo).

### Opciones de menú para configurar la OVI
<a name="opciones_menu_para_configurar_ovi_ELTPARAGRAPHE000171"></a>Varias opciones permiten a los usuarios configurar la función OVI. Estas opciones aparecen en el menú contextual de la ventana (clic derecho en la ventana Background).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GFI_menu_ctx.gif)
Las opciones son las siguientes: 

- **Oscurecer ventana si está inactiva (OVI)**: Esta opción se usa para poner en gris (o no) las ventanas de Inactive. Esta opción es válida sólo para la ventana actual.

- **Aumentar la intensidad del OVI**: Esta opción se utiliza para mostrar una barra de progreso que indica la tasa actual de gris para las ventanas de Inactive. Puedes aumentar esta tasa presionando Ctrl + Alt + '+' en el teclado numérico. Esta tasa de gris será válida para toda la aplicación.

- **Disminuir la intensidad del OVI**: Esta opción se utiliza para mostrar una barra de progreso que indica la tasa actual de gris para las ventanas de Inactive. Puede disminuir esta tasa presionando Ctrl + Alt + "-" en el teclado numérico. Esta tasa de gris será válida para toda la aplicación.




Para dejar de mostrar la barra de progreso, simplemente presione Esc o Enter.**Observación**: Para evitar que los usuarios modifiquen la tasa de OVI, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafRateDDW*.


### Límite
<a name="limite_ELTPARAGRAPHE000190"></a>

Los controles HTML, Xaml, Cube, Carousel (.Net version) y ActiveX no son atenuados por la característica OVI.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000231"></a>


### Crear un diagrama de casos de uso
<a name="crear_diagrama_casos_uso_ELTPARAGRAPHE000039"></a>

Para crear un diagrama de casos de uso:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Arquitectura", y luego en "UML".

	- El asistente de creación de diagramas UML se abre.




2. Seleccione "Diagrama de casos de uso".

3. Especifique el modelo UML en el que se insertará el diagrama.

4. Elija editar el diagrama directamente.

5. Valide el nuevo diagrama de casos de uso. El nuevo modelo aparece en el editor.



<a name="NOTE2_2"></a>


### Crear un caso de uso
<a name="crear_caso_uso_ELTPARAGRAPHE000061"></a>

Para crear un caso de uso: 

1. En la pestaña "Diagrama de casos de uso", en el grupo "Creación", haga clic en "Caso de uso". Un nuevo elemento de caso de uso aparece en el modelo UML.

2. Haga clic derecho en el nuevo caso de uso y seleccione "Descripción" en el menú contextual. La ventana de descripción del caso de uso se abre.

3. Especifique:

	- el nombre del caso de uso. Este nombre se muestra en el editor de UML.

	- el título que describe el objetivo del caso de uso. Este título aparecerá en la documentación técnica del editor de UML.




4. Valide la descripción del caso de uso.



<a name="NOTE2_3"></a>


### Crear un actor
<a name="crear_actor_ELTPARAGRAPHE000088"></a>

Para crear un actor: 

1. En la pestaña "Diagrama de casos de uso", en el grupo "Creación", haga clic en "Actor".

2. Haga clic derecho en el nuevo actor y seleccione "Descripción" en el menú contextual. La ventana de descripción del actor se abre.

3. Especifique:

	- el nombre del actor. Este nombre se muestra en el actor en el editor de UML.

	- el título que describe el objetivo del actor.




4. Valide la descripción del actor.



<a name="NOTE2_4"></a>


### Crear una relación
<a name="crear_una_relacion_ELTPARAGRAPHE000115"></a>

Para crear una relación:

1. Según el tipo de relación a crear, en la pestaña "Diagrama de casos de uso", en el grupo "Creación":

	- haga clic en "Generalización" para crear una generalización.

	- haga clic en "Asociación" para crear una asociación.

	- haga clic en "Dependencia" para crear una dependencia.




2. Especifique la información solicitada.

3. Valide la descripción de la relación.



<a name="NOTE2_5"></a>


### Crear un paquete
<a name="crear_paquete_ELTPARAGRAPHE000138"></a>

Para crear un paquete: 

1. En la pestaña "Diagrama de casos de uso", en el grupo "Creación", haga clic en "Paquete".

2. Haga clic derecho en el nuevo paquete y seleccione "Descripción" en el menú contextual. La ventana de descripción del paquete se abre.

3. Especifique:

	- el nombre del paquete. Este nombre se muestra en el paquete en el editor de UML.

	- el título que describe el objetivo del paquete.




4. Valide la descripción del paquete.

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000163"></a>
WINDEV, WEBDEV y WINDEV Mobile incluyen un editor de imágenes. 

Una cinta de opciones y un panel contienen las principales funcionalidades del editor. 

Observación: Esta página de ayuda explica cómo iniciar el editor de imágenes. Para obtener más información sobre las funcionalidades disponibles, consulte [Editor de imágenes: Funcionalidades](../Editeurs/1410086606.md).

<a name="NOTE2_1"></a>


## Abrir el editor de imágenes
<a name="abrir_editor_imagenes_ELTTEXTE000187"></a>
El editor de imágenes se puede abrir: 

- para mostrar una imagen o ícono existente,

- para crear una nueva imagen o un nuevo ícono. 



<a name="NOTE2_2"></a>


### Mostrar una imagen existente
<a name="mostrar_una_imagen_existente_ELTPARAGRAPHE000033"></a>

Para mostrar una imagen existente, puede: 

- utilizar el panel "Imágenes" que se encuentra en el editor de WINDEV, WEBDEV o WINDEV Mobile. Este panel muestra la lista de imágenes ubicadas en el directorio del proyecto actual (o en el directorio _WEB para un proyecto WEBDEV). También puede abrir una imagen ubicada en el disco. Para obtener más información, consulte [Panel "Imágenes"](../Editeurs/2027036.md). 

- utilizar el menú contextual del control que muestra la imagen: 

	- Abra el menú contextual del control (por ejemplo, control Imagen o Botón). 

	- Seleccione la opción "Editar imagen".




- utilizar la ventana de descripción del control: En la pestaña "General", despliegue ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) a la derecha de la ruta de imagen y seleccione "Editar imagen".




La imagen seleccionada aparece en el editor de imágenes y puede editarse. 
<a name="NOTE2_3"></a>


### Crear una imagen (o un ícono)
<a name="crear_una_imagen_icono_ELTPARAGRAPHE000054"></a>

Para crear una imagen en el editor de imágenes, puede: 

- utilizar el panel "Imágenes" que se encuentra en el editor de WINDEV, WEBDEV o WINDEV Mobile. Para obtener más información, consulte [Panel "Imágenes"](../Editeurs/2027036.md).

- utilizar la ventana de creación de nuevos elementos: 

	1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Nouveau.gif) en los botones de acceso rápido. 

	2. En la ventana que aparece, haga clic en "Taller" y luego en "Imagen". 

	3. Una ventana para definir la imagen a crear aparece. Es posible crear: 

		- una imagen en blanco o elegir una imagen del catálogo de imágenes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_nouveau%20-%20HC%20N%B0001.gif)


		- un ícono en blanco o elegir un ícono del catálogo de imágenes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_nouveau%20-%20HC%20N%B0002.gif)

4. Valide. Si es necesario, seleccione la imagen de base y sus características en el catálogo de imágenes. 

5. Una nueva imagen aparece en el editor de imágenes.

## Guardar una imagen a través del editor de imágenes
<a name="guardar_una_imagen_traves_del_editor_imagenes_ELTTEXTE000223"></a>


### Guardar una imagen editada
<a name="guardar_una_imagen_editada_ELTPARAGRAPHE000082"></a>

Una imagen editada en el editor de imágenes puede guardarse mediante el ícono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Enregistrer.gif) de la cinta de opciones. 

Pueden darse varios casos: 

- Si la imagen no tiene capas (imagen simple editada directamente o imagen acoplada), se guarda automáticamente en el formato de origen con el mismo nombre. Por lo tanto, la imagen existente se actualiza directamente en las ventanas, reportes, etc. que utilizan la imagen. 

- Si la imagen tiene al menos una capa, puede guardarse en varios formatos: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Editeur_img_sauver%20-%20HC%20N%B0001.gif)


	- **Imagen solo en formato PNG**. Las capas se perderán.

- **Imagen en formato PNG y WDPIC**. El nombre del archivo de imagen WDPIC tiene el siguiente formato: "nombredeimagendeorigen.extensiondeorigen.wdpic". La imagen WDPIC contiene la descripción de todas las capas y cambios realizados en la imagen de origen. La imagen original no cambia. Si intenta cargar una imagen que tiene un archivo .wdpic equivalente, el archivo .wdpic se carga automáticamente en el editor de imágenes.

- **Imagen solo en formato WDPIC**. 
			En este caso, la imagen de origen no cambia. 
			La imagen WDPIC puede asociarse a los diferentes elementos de un proyecto, como cualquier otro formato de imagen.




Observación: Una vez seleccionado el formato, los futuros cambios se guardan en el mismo formato. 


### Guardar una nueva imagen o ícono
<a name="guardar_una_nueva_imagen_icono_ELTPARAGRAPHE000105"></a>

Si creó una imagen o ícono en el editor de imágenes, puede guardarlo haciendo clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_Enregistrer.gif) en la cinta de opciones. Una ventana aparece y le pide que guarde: 

- la imagen en formato "wdpic" de forma predeterminada. 

- el ícono en formato "ico" de forma predeterminada. 




Para cambiar el formato en el que se guardará la imagen, haga clic en "..." y seleccione el formato deseado. Los siguientes formatos están disponibles: WDPIC, TIF, ICO, PNG, JPEG, BMP y GIF. 

Observación: Una vez seleccionado el formato, los futuros cambios se guardan en el mismo formato.

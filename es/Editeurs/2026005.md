## Presentación
<a name="presentacion_ELTTEXTE000184"></a>
¿Tu ventana contiene controles superpuestos?

Quieres averiguar la posición de orden Z de tus controles: ¿qué controles se muestran en primer plano?

La pantalla bidimensional pronto se vuelve inadecuada para este tipo de operación.

WINDEV ofrece una vista espacial de una ventana al permitirle ver la ventana en tres dimensiones. La posición de orden Z de los diferentes controles de la ventana aparece inmediatamente. Una control seleccionada en este modo de visualización permanece seleccionada cuando el modo vuelve a las dos dimensiones.

Usted tiene la posibilidad de elegir el mejor ángulo de visión mediante el ratón y las teclas del teclado.

**Atención**: se requiere una configuración específica para utilizar esta función.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000232"></a>
Para mostrar sus ventanas en 3 dimensiones:

1. Visualizar la ventana en el editor.

2. En la pestaña "Vista", en el grupo "Opciones", haga clic en "3D".

3. La ventana se muestra en 3D.




**Observación**: Para mostrar una vista tridimensional de su ventana y animar esta vista, presione Alt + Mayúsculas.

El ratón o el teclado pueden utilizarse para optimizar la vista de la ventana.

| Se mueve con el ratón |   |
| --- | --- |
| Rotación (botón izquierdo del ratón) | Haga clic dentro de la vista de la ventana y mueva el ratón mientras mantiene pulsado el botón izquierdo. |
| Reubicación (botón derecho del ratón) | Haga clic dentro de la vista de la ventana y mueva el ratón mientras mantiene pulsado el botón derecho. |
| **Se mueve con el teclado** |   |
| rotación | Utilice las teclas de flecha, así como las teclas + y - del teclado numérico.<br><br>También puede utilizar los números del teclado numérico. |
| Traslación | Mantenga pulsada la tecla Mayúsculas y utilice las teclas de flechas del teclado numérico. |
| Zoom | Pulse la siguiente combinación de teclas:<br><br>- Mayúsculas + para acercar la imagen<br><br>- Mayúsculas - para alejar la imagen.<br><br><br> |
| Modificar la separación entre los mandos (modificaciones entre los diferentes planos de visualización) | / (en el teclado numérico) para reducir el espacio entre los controles.<br>"\*" (en el teclado numérico) para aumentar el espacio entre los mandos |
| Reinicialización de la pantalla | Tecla 0 en el teclado numérico: reinicialización completa<br>5 en el teclado numérico: reinicialización manteniendo el espacio entre los controles. |
| Animación de la vista en 3 dimensiones | Tecla bloq despl.<br>Clave Esc de Cancel |

**Observación**: Para desactivar la vista tridimensional, pulse la tecla Esc.

## Notas
<a name="notas_ELTTEXTE000256"></a>


### Posibles acciones al mostrar una ventana en 3D
<a name="posibles_acciones_mostrar_una_ventana_3d_ELTPARAGRAPHE000116"></a>

En el modo tridimensional, sólo algunas acciones están disponibles para los controles de ventana. Puedes, por ejemplo:

- seleccione una control,

- mostrar el control description,

- mostrar el código fuente de la control.



<a name="NOTE4_2"></a>


### Condiciones para mostrar una ventana en 3 dimensiones
<a name="condiciones_para_mostrar_una_ventana_3_dimensiones_ELTPARAGRAPHE000127"></a>

Para mostrar sus ventanas en 3 dimensiones, el ordenador actual debe estar equipado con:

- DirectX versión 8.0.A (o posterior).

- Una tarjeta aceleradora 3D con controladores actualizados.

- Un modo de visualización en 16 bits o 32 bits.




Si utiliza WINDEV o WINDEV Mobile en modo de 64 bits, para implementar la visualización en 3 dimensiones, debe descargar e instalar el paquete desde la siguiente Address: [https://www.microsoft.com/fr-fr/download/details.aspx?id=35](https://www.microsoft.com/fr-fr/download/details.aspx?id=35)

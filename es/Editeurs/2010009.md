
## Creación de palabras clave de índice
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000154"></a>
Las palabras clave Índice permiten a los usuarios encontrar fácilmente una ayuda Page. Cada ayuda Page puede ser associated con una o varias palabras clave y cada palabra clave puede ser associated con una o varias Page s.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PageAide_index2.gif)


Atención: las palabras clave de índice se ignoran en los archivos de ayuda PDF.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000178"></a>


### Creación de una palabra clave de índice (sin asociarla a una Page)
<a name="creacion_una_palabra_clave_indice_sin_asociarla_una_page_ELTPARAGRAPHE000021"></a>

Para crear una palabra clave de índice:

1. Abra la lista de palabras clave del índice en el editor de ayuda ("Índice" pestaña ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_Ico_Index.gif)).

2. En la lista de palabras clave, seleccione "Nueva palabra clave" en el menú contextual. Las palabras "Nueva entrada de índice" aparecen en edición.

3. Introduzca la palabra clave y valide.




**Observaciones**: Para set una palabra clave como "subpalabra clave", basta con arrastrarla y soltarla sobre la palabra clave principal.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Motcle.gif)

<a name="NOTE2_2"></a>


### Crear una palabra clave de índice y asociarla a una Page
<a name="crear_una_palabra_clave_indice_asociarla_una_page_ELTPARAGRAPHE000038"></a>

Puede crear una palabra clave de índice y asociarla directamente a un Page, desde las propiedades de Page.

1. Abra las propiedades de Page ("Propiedades de Page" en el menú contextual de Page).

2. Vaya a la pestaña "Índice".

3. Haga clic en "Añadir".. Se abre la ventana de gestión de palabras clave de índice.




Esta ventana le permite: 

- crear las palabras clave y subpalabras clave del sistema de ayuda. 

- asociar las palabras y subpalabras clave existentes a la Page actual . 




**Para crear una palabra clave o palabra clave secundaria**:

1. Introduzca la palabra clave o subpalabra clave. 

2. Usted puede: 

	- Haga clic en "Crear clave".: la palabra clave se creará y se añadirá a la lista de palabras clave por orden alfabético. 

	- Seleccione la palabra clave principal en la lista y haga clic en "Subclave".: la palabra clave se añadirá como subentrada de la palabra clave seleccionada. 







Para asociar una palabra clave o una subpalabra clave a la Page actual : 
 Seleccione la palabra clave o palabra clave secundaria en la lista "Palabras clave disponibles para este sistema de ayuda" y haga clic en el botón ">". 

1. La palabra clave o una subpalabra clave aparece en la lista de palabras clave associated con el Page. 

	- Valide la ventana.







**Observación**: Las palabras clave del índice también pueden generarse automáticamente a partir del contenido de Page. Para obtener más información, consulte [Generación automática de índices](../Editeurs/2010007.md).
<a name="NOTE2_3"></a>


### Asociar una palabra clave de índice existente a una Page
<a name="asociar_una_palabra_clave_indice_existente_una_page_ELTPARAGRAPHE000074"></a>

Para asociar una palabra clave de índice a una Page:

**1ra solución**:

1. Abra la ayuda Page en el editor de ayuda.

2. Abra las propiedades de Page ("Propiedades de Page" en el menú contextual de Page).

3. En la pestaña "Índice", pulse el botón "Añadir. Seleccione la palabra clave deseada en "Palabras clave disponibles para este sistema de ayuda" y haga clic en "->". 




**2da solución**:

1. Abra la lista de palabras clave del índice en el editor de ayuda ("Índice" pestaña ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_Ico_Index.gif)).

2. Seleccione la entrada de índice deseada.

3. Haga clic con el botón derecho del ratón en la palabra clave y seleccione "Propiedades de la palabra clave".. Para obtener más información, consulte [Propiedades de las palabras clave](../Editeurs/2010029.md).

4. En la pestaña "Asociación", pulse el botón "Añadir" y seleccione la ayuda deseada Page.



<a name="NOTE2_4"></a>


### Eliminación de una palabra clave de índice
<a name="eliminacion_una_palabra_clave_indice_ELTPARAGRAPHE000099"></a>

Para eliminar una palabra clave de índice:

1. Abra la lista de palabras clave del índice en el editor de ayuda ("Índice" pestaña ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_Ico_Index.gif)).

2. Seleccione la entrada de índice deseada.

3. Seleccione "Eliminar palabra clave" en el menú contextual. Todas las páginas associated con esta palabra clave ya no serán associated con esta palabra clave.

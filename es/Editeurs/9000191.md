## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
WINDEV incluye varios componentes de utilidad que le permiten acceder directamente a los datos de manipular. Puede integrarlos directamente en sus aplicaciones. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000155"></a>
Para integrar una utilidad Component en su proyecto: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar". Seleccione "Una Component externa " y después "Utilidad Component o ejemplo".. 

2. Aparece una ventana con la lista de componentes de utilidad disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Gestion%20des%20composants%2021%20-%20HC%20N%B0001.gif)


3. Seleccione la Component que desea importar a su proyecto. 

4. Validar. La Component es importada. Aparece en el panel "Explorador de proyectos.

## Los diferentes componentes disponibles
<a name="los_diferentes_componentes_disponibles_ELTTEXTE000179"></a>
Los diferentes Component s se entregan con su documentación para utilizar las funcionalidades del Component. 
**ExternalAccess** es un Component que permite consultar los datos de los Centros control. Puede conectarse a la base de datos de control Center (HFSQL Classic, HFSQL Client/Server, [PCSCloud](https://pcscloud.net/UK/cloud_development.awp) o SCMDrive) para exportar los datos sobre los proyectos, los colaboradores, las tareas, los incidentes y los requisitos como archivos JSON. 

Para más detalles sobre los Centros control, véase [Centros de Control](../Editeurs/9000043.md). 
**CCMenu** le permite administrar algunas de las funciones disponibles en el menú *?*. 

Para obtener más información, consulte [Menú "?"](../Editeurs/9000070.md). 
**CCHCEdit** le permite anotar imágenes. 
**CCFeedBack** le permite administrar los comentarios de los usuarios. Para obtener más información, consulte [Comentarios Component](../Editeurs/2014013.md). 
**Robot Component s** corresponde a la gestión Component del robot de vigilancia. 

Para obtener más información, consulte [Presentación del motor de monitoreo](../RobotMonitor/3541100.md).
El componente **TipOfTheDay** permite administrar los consejos del día.

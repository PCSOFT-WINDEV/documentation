
## Configuración de la modificación de los archivos de datos de automatic 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000122"></a>
La modificación automática de los ficheros de datos actualiza la description de ficheros de datos en los ordenadores de los usuarios.

Para obtener más información, consulte [Modificación automática de archivos de datos](../WDLang4/3044195.md).

<a name="NOTE2_1"></a>


## Configurando la modificación de datos de la automatic con la asistente
<a name="configurando_modificacion_datos_automatic_con_asistente_ELTTEXTE000152"></a>
Para configurar la modificación de datos automatic con asistente, seleccione la aplicación (archivo .exe) para la que desea ejecutar la modificación.

Si los archivos de datos son protected con contraseña, o si están en una base de datos HFSQL Client/Server, tendrá que especificar las contraseñas: 

1. Haga clic en el botón "Avanzado".

2. Seleccione el análisis en formato WINDEV (archivo ".WDD") associated con la aplicación.

3. Si es necesario, especifique lo siguiente para cada archivo del análisis:

	- la contraseña del fichero (si existe) en el análisis. 
			**Observación**: Para cambiar esta contraseña, seleccione "Cambiar contraseña" e introduzca la nueva contraseña.

	- si la aplicación utiliza un análisis WINDEV 5.5 migrado, especifíquelo:

		- el análisis fuente en formato WINDEV 5.5 (archivo ".WDA").

		- el nombre del archivo en el análisis en formato WINDEV 5.5.

		- la contraseña del archivo (si existe) en el análisis en formato WINDEV 5.5.







Esta información (excepto las contraseñas) se muestra automáticamente.

**Observaciones**:

- Durante la actualización, los archivos de datos que se encuentran automáticamente son:

	- los archivos que se encuentran en el directorio instalación de la aplicación.

	- los archivos que figuran en el archivo ".REP. Para más detalles, consulte la ayuda sobre el programa [gestión de archivos .REP](../WDLang4/3044186.md).




- asistente propone realizar una modificación "silenciosa" de los datos de automatic: no se mostrará ninguna ventana de configuración al usuario final. 

- Para que el usuario pueda configurar la modificación de automatic, puede utilizar WDInst para incluir el botón "Opciones avanzadas" en el programa instalación.

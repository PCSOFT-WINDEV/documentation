<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000100"></a>
El minimapa es una opción que permite navegar rápidamente por el editor de código. 

En la parte derecha (o izquierda) del editor de código se muestra una vista previa reducida del código.

La sección que corresponde al código que se muestra está claramente representada. Y el movimiento de la pequeña ventana de vista previa se sincroniza con el contenido del editor de código.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ascenseur_visuel_code%20-%20HC%20N%B0002.gif&type=thumb)
 Este modo mejora la precisión, la velocidad y la comodidad cuando Scrolling. También proporciona una visión general de todos los eventos y procesos. Por ejemplo, se muestran todos los procedimientos encontrados en una set de procedimientos o todos los métodos de una clase.

El minimapa indica: 

- errores de compilación. Las líneas que contienen un error de compilación están resaltadas en rojo. 

- resultados de la búsqueda. Las líneas que contienen un elemento de búsqueda están resaltadas en naranja.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000124"></a>
Para gestionar el minimapa en el editor de código: 

1. Abrir las opciones del editor de código: En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. En la ventana que aparece, seleccione la pestaña "Código. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ascenseur_visuel_code%20-%20HC%20N%B0001.gif&type=thumb)


3. Seleccione el tipo de barra de desplazamiento deseado: 

	- Barra de desplazamiento estándar.

	- Minimapa a la izquierda. 

	- Minimapa a la derecha. 




4. Validar.

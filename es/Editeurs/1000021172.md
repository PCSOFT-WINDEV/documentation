


## Groupware Usuario: Aplicaciones multilingües
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
Si su aplicación es multilingüe, la función multilingüe debe gestionarse también en las ventanas/páginas de groupware usuario. Por Default, las ventanas/páginas de groupware usuario soportan el inglés, el americano y el francés.

Esta página de ayuda explica cómo hacerlo:

- [Configurar el User Groupware para utilizarlo en una aplicación multilingüe](#NOTE2_1). 

- [Traducir las diferentes ventanas o páginas de groupware usuario](#NOTE3_1).

- [Ejecutar la aplicación en el idioma solicitado](#NOTE4_1).




Para obtener más información, consulte [Creación de una aplicación multilingüe](../WDLang1/3054008.md).

## Configurar el programa de grupo de usuarios para una aplicación multilingüe
<a name="configurar_programa_grupo_usuarios_para_una_aplicacion_multilingue_ELTTEXTE000170"></a>
Para incluir las ventanas o páginas del User Groupware en una aplicación multilingüe:

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Groupware Usuario". 

2. Habilitar el programa de grupo de usuarios. 

3. Dependiendo de los idiomas soportados, seleccione el modo de integración: 

	- Para una aplicación en inglés/francés, en la pestaña "Opciones", seleccione "Sin personalización". 

	- Para una aplicación que admita otros idiomas, en la pestaña "Opciones", seleccione "Personalización completa (modo experto)". 




4. Dependiendo del modo de inicio de la aplicación, defina cómo se iniciará el groupware: 

	- Si la aplicación debe utilizar el lenguaje de ejecución, en la pestaña "General", seleccione "Inicio de automatic". 

	- Si la aplicación debe pedir al usuario que seleccione el idioma de ejecución, en la pestaña "General", seleccione "Inicio manual"




5. Validar.

## Traducir los elementos del programa de grupo de usuarios
<a name="traducir_los_elementos_del_programa_grupo_usuarios_ELTTEXTE000194"></a>
Por Default, los elementos (ventana, páginas, mensajes, ...) que se encuentran en el groupware usuario están disponibles en inglés y en francés.

Para traducir estos elementos a otros idiomas, todo lo que tienes que hacer es traducir los diferentes subtítulos, mensajes, ... de estas ventanas o páginas (utilizando [WDMSG](../WDMsg/3518040.md) si es necesario, para extraer los mensajes a traducir). WDMSG es una herramienta opcional para WINDEV, WEBDEV y WINDEV Mobile que se utiliza para extraer y reintegrar todos los mensajes a traducir.
Observación: No olvides comprobar las cadenas multilingües.

## Ejecutar la aplicación en el idioma solicitado
<a name="ejecutar_aplicacion_idioma_solicitado_ELTTEXTE000218"></a>
Pueden presentarse dos casos: 

- La aplicación se inicia automáticamente en el lenguaje Default. El idioma puede ser cambiado a partir de entonces. 
	En este caso, los elementos del groupware también se ejecutarán en el lenguaje Default. 

- La aplicación pide al usuario que elija el idioma de ejecución. En este caso, se debe: 

	1. Pida al usuario el idioma en el que debe ejecutarse la aplicación (por ejemplo, a través de una ventana o página específica).

	2. Configurar el idioma de ejecución de la aplicación con [Nación](../WDLang1/3054014.md).

	3. Abrir la primera ventana o página de gestión de la Agrupación de Usuarios con [gpwOpen](../WDLang6/3041002.md).







Ejemplo de inicio personalizado: 


```wl
// Initialization code of project
// Open the window for language selection
nLanguage is int = Open(WIN_Select_Language)
// Configure the language for the application
Nation(nLanguage)
// Start the first window of user groupware
gpwOpen()
```

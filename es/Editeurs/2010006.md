


## Definir una ventana de ayuda modelo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000176"></a>
WINDEV permite crear plantillas de ventanas de ayuda. Estas plantillas se utilizan para agrupar todas las características de la ventana de ayuda en un único lugar: Características HTML, página Color, etc.

Para utilizar todas las características de una página de ayuda modelo, basta con asociar el modelo a la página de ayuda (pestaña "Diseño" de la página description).

<a name="NOTE2_1"></a>


## Crear un modelo
<a name="crear_modelo_ELTTEXTE000200"></a>


### ¿Cómo crear una ventana de ayuda de modelo? 
<a name="¿como_crear_una_ventana_ayuda_modelo_ELTPARAGRAPHE000020"></a>

Crear una nueva ventana de ayuda de modelo:

1. En la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Ventanas de ayuda".

2. Pulse el botón "Nuevo" y especifique el nombre de la ventana Definition.

3. Seleccione las opciones de la modelo a través de las diferentes pestañas:

	- Paneles mostrados y botones del sistema de ayuda (pestaña "Común").

	- Características específicas de las páginas de ayuda compiladas en formato HTML (pestaña "Ayuda HTML").

	- Los colores utilizados en la Page (pestaña "Maquetación").

## "Ficha "Común
<a name="ficha_comun_ELTTEXTE000224"></a>
Las opciones comunes de una Page modelo se utilizan para definir:

- el **nombre lógico** de modelo .

- el **título de la ventana**del modelo. Si se especifica este título, éste se mostrará independientemente del título definido en las características de la ayuda Page.

- los **paneles** que se pueden mostrar desde el Page .

- los **botones** propuestos en la barra de herramientas: Estos botones permitirán al usuario: 

	- ver las Page visitadas anteriormente . 

	- ver todas las opciones de ayuda (fuentes, posibilidad de hacer que la ayuda sea siempre Visible, etc.).

	- imprimir la ayuda actual Page.

## "HTML Help" tab
<a name="html_help_tab_ELTTEXTE000248"></a>
Las opciones específicas del sistema de ayuda compilado en formato HTML (archivo CHM) son las siguientes:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modele_HTML.gif)


- **Mostrar el panel de favoritos**: El panel de favoritos permite reference las páginas de ayuda más consultadas por el usuario.

- **Opciones de diseño**:

	- ***Panel lateral oculto por defecto***: Oculta automáticamente todas las pestañas (resumen, índice, búsqueda, etc.): en la ventana sólo aparecen la barra de herramientas y la Page. Un botón en la barra de herramientas muestra las diferentes pestañas.

	- ***Sincronización automática de la tabla de contenidos***: Sincroniza automáticamente el resumen con la Page actual en la ayuda. Si se hace referencia a esta página en el resumen, éste se amplía y se selecciona la Icon correspondiente.

	- ***Ocultar el panel lateral cuando pierde el enfoque***: Oculta automáticamente todas las pestañas cuando el archivo de ayuda pierde el enfoque.. Las pestañas se vuelven a mostrar tan pronto como la ayuda recupera el enfoque.




- **Botones disponibles en la barra de herramientas**: 

	- ***Ocultar/Mostrar el panel lateral***: Este botón muestra u oculta las pestañas (resumen, búsqueda, índice, etc.).

	- ***página de inicio***: Este botón permite visualizar la Default Page del sistema de ayuda.

	- ***Próximo***: Este botón permite visualizar la Page siguiente (según el orden definido).

	- ***Sincronizar el tabla de contenidos***: Este botón permite sincronizar la tabla de contenidos con la ayuda actual Page. 

	- ***Detener***: Este botón detiene la carga de la ayuda Page. 

	- ***Actualizar***: Este botón vuelve a mostrar la ayuda actual Page. 

	- ***Enlace directo 1/Enlace directo 2***: Este botón abre un sitio web, una ayuda específica Page, etc.. Las propiedades de estos enlaces deben ser ingresadas (caption y URL).

## "Pestaña "Diseño
<a name="pestana_diseno_ELTTEXTE000272"></a>
La pestaña "Diseño" permite definir las opciones de color utilizadas en las páginas de ayuda.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modele_Apparence.gif)


- **Área de encabezado**: 
	Permite definir: 

	- el fondo color de la zona de cabecera.

	- el fondo que Image usó para esta área.

	- el estilo del texto que se muestra en esta área.




- **Área principal**:
	Se utiliza para definir el fondo Color del área de texto y/o el fondo Image utilizado para esta área.

- **Posicionamiento**:
	Permite definir las coordenadas de la ventana de ayuda (en píxeles). La asistente se utiliza para simplificar la introducción de coordenadas:

	- haga clic en "Asistente".. Aparece una ventana.

	- modificar el tamaño y la posición de esta ventana y validar. El tamaño y las coordenadas de esta ventana se recuperan automáticamente.

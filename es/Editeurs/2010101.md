## Presentación
<a name="presentacion_ELTTEXTE000169"></a>
Para crear un nuevo modelo flexible, puede: 

- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- Aparece la ventana de creación: haga clic en "Arquitectura" y luego en "Modelado flexible".

	- Comienza la asistente para crear un modelo flexible.




- Seleccione "Nuevo modelado flexible" en el menú contextual del panel "Explorador de proyectos".. [La asistente para crear un nuevo modelo](../Editeurs/2010101.md) comienza.

## Creación de modelos flexibles asistente
<a name="creacion_modelos_flexibles_asistente_ELTTEXTE000193"></a>
La asistente para la creación de un modelo flexible le permite:

- especifique el título del modelo flexible.

- elegir el contenido inicial del modelado.

	- ***vacíos***: Modelado en blanco

	- ***Simple***: Modelado que utiliza una base de datos

	- ***Con componentes***: Modelado que utiliza una base de datos y componentes.


Este contenido puede ser personalizado a partir de entonces.

- elegir las propiedades avanzadas del modelado. Estas propiedades afectan:

	- la inserción de bloques de títulos de automatic: utilizado para configurar la presencia de automatic de un bloque de título en el modelado principal y en los submodelos. Esta opción puede ser modificada en cualquier momento en el editor.

	- el uso de automatic de los contribuyentes que se encuentran en el Centro de Seguimiento de Proyectos en la lista de actores modelo. Esta opción puede ser modificada en cualquier momento en el editor.





La description del modelo se ha completado. El modelo se muestra en el editor.

## Los diferentes elementos que se pueden insertar en un modelo
<a name="los_diferentes_elementos_que_pueden_insertar_modelo_ELTTEXTE000217"></a>
El editor de modelado le permite insertar varios tipos de elementos en su modelado. Estos elementos están disponibles en el panel "Creación" del editor de modelos.

- Inicio (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Commencement_i.gif)): Se usa para ver el comienzo del modelado flexible.

- Final (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Termine_ico.gif)): Se usa para ver el final del modelado flexible.

- Decisión/Esquema (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Choix_ico.gif)): Se utiliza para ver una opción en el modelado flexible.

- Process (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Trait_Ico.gif)): Se utiliza para ver un Process.

- Solicitud (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Appli_ico.gif)): Se usa para ver una aplicación.

- Archivo de datos (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_HF_ico.gif)): Se utiliza para ver un archivo de datos.

- Image (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Image_ico.gif)): Solía incluir un Image (Image o Image personalizado que se encuentra en el catálogo de Image) en el modelado flexible. Esta imagen tiene las mismas características que una Process.

- Bloque del título (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Modelisation_Cart_ico.gif)): Se utiliza para incluir un bloque de título en el modelado flexible. Se utiliza para dar un título a un modelo flexible o a un sub-modelo, por ejemplo.


También tiene la posibilidad de insertar formas simples (círculo, elipse, rectángulo, rectángulo redondeado) y de unir todos estos elementos.
<a name="NOTE3_2"></a>


### Descripción de los elementos
<a name="descripcion_los_elementos_ELTPARAGRAPHE000082"></a>

Cada elemento de un modelado flexible puede ser configurado a través de su ventana description ("description" de el menú contextual del elemento).

Para obtener más información, consulte [Descripción de un elemento](../Editeurs/2010102.md).
<a name="NOTE3_3"></a>


### Añadir un elemento de proyecto a un modelo flexible
<a name="anadir_elemento_proyecto_modelo_flexible_ELTPARAGRAPHE000094"></a>

Si realiza un modelado flexible de un proyecto WINDEV, WEBDEV o WINDEV Mobile, los elementos que se encuentran en su proyecto se pueden insertar directamente en el modelado.. Para ello, arrastre y suelte un elemento del proyecto (que se encuentra en el panel "Explorador de proyectos" o en el panel "Análisis", por ejemplo) hacia el modelo.

Cuando el elemento se inserta en el modelado flexible, ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=modelisation_liaison.gif) indica que el elemento es relacionado a un objeto del proyecto. Para abrir este objeto directamente, puede hacerlo:

- haga doble clic en la flecha.

- seleccione "Abrir el elemento relacionado" de el menú contextual.



<a name="NOTE3_4"></a>


### Submodelado
<a name="submodelado_ELTPARAGRAPHE000108"></a>

Los elementos "Application", "Process" y "Choice" pueden utilizarse para entrar en un "submodelado".. Por ejemplo, el description del ERP (Enterprise Resource Planning) de una empresa puede incluir los siguientes submodelos: Gestión de clientes, gestión de presupuestos, marketing, producción, contabilidad,...

Para obtener más información, consulte [Gestión de los sub-modelos](../Editeurs/2010104.md).
<a name="NOTE3_5"></a>


### Gestión de la altitud de los objetos
<a name="gestion_altitud_los_objetos_ELTPARAGRAPHE000120"></a>

Todos los elementos que se encuentran en el editor de modelos se pueden apilar. Un elemento se puede encontrar en el primer plano o en la Background, un elemento puede ser movido un nivel arriba o abajo. Esto le permite posicionar una Image en la Background del modelado por ejemplo.

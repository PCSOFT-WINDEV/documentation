
## MVP: Ejemplo simple de archivo único: Tabla + Archivo enlazado
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
Veamos un ejemplo simple e ilustrado sobre el uso de MVP para crear una ventana de Tabla con su formulario relacionado. Este ejemplo utiliza un solo archivo. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000119"></a>
A diferencia del modo de programación clásico, el control Tabla es relacionado a un Variable en lugar de un archivo de datos. 

Del mismo modo, los controles de formulario no son relacionado para los ítems sino para las variables. 

Se trata de utilizar el concepto "Binding. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IHM-et-Binding.gif)



### La ventana Tabla
<a name="ventana_tabla_ELTPARAGRAPHE000025"></a>

En este ejemplo, la ventana "Tabla" le permite ver una lista de contactos.

Las posibles acciones son: 

- Crear una Contact. 

- Borrar un Contact. 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Exemple-Gestion-Contacts-01.gif)



### La ventana Formulario
<a name="ventana_formulario_ELTPARAGRAPHE000037"></a>

En este ejemplo, la ventana "Formulario" permite crear y modificar un contacto.

Las posibles acciones son: 

- Validar la entrada. 

- Cancel la entrada. 



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Exemple-Gestion-Contacts-02.gif)

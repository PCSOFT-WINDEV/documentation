
## Estilo control: Características de las fuentes
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000085"></a>
Puede cambiar muchas opciones de estilo en la pestaña "Estilo" de la ventana de control description. Puede cambiar las características del tipo de letra utilizado en los pies de foto de control, en los campos de entrada o incluso en los elementos de una lista.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Características de las fuentes
<a name="caracteristicas_las_fuentes_ELTTEXTE000109"></a>
Para cualquier control que muestre texto, puede configurar los siguientes elementos en la pestaña "Estilo" de la ventana description: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Stylle_surcharge%202.gif)


- **Fuente**: Familia de fuentes a utilizar. 

- **Posición**: determina la posición del texto con respecto a la control. 

- **Color de la fuente**: Permite elegir el tipo de letra Color. 

- **Tamaño**: Tamaño de fuente. Puede elegir la unidad deseada: Píxel CSS, punto o punto WINDEV. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Style_Unit%E9_interligne%20-%20HC%20N%B0002%201.gif)


- Opciones de formato: Negrita, Cursiva, Subrayado, Tachado. 

- Opciones avanzadas de formato (botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Style_Unit%E9_interligne%20-%20HC%20N%B0001%201.gif)): En la ventana que aparece, puede set: 

	- el interlineado ([LineSpacing](../Proprietes/1410088366.md) Property),

	- el espaciado entre caracteres ([CharacterSpacing](../Proprietes/1410088367.md) Property),

	- el ángulo, sólo para controles estáticos. 




- **Sombra**: Permite aplicar una sombra al texto y elegir la dirección de la sombra. Puede set la sombra Color mediante la opción "Sombra col."..

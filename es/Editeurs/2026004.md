


## Manejo de las vistas de una ventana 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
Para facilitar la manipular de las ventanas grandes (que no pueden mostrarse por completo en el editor, por ejemplo), WINDEV y WINDEV Mobile le permiten manipular varias vistas de la misma ventana. 

Cada vista puede mostrar una sección diferente de la misma ventana. 

Todas las modificaciones realizadas en una de las vistas se aplicarán automáticamente a todas las vistas abiertas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Vue_Fen.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000110"></a>
Para abrir una nueva vista de la ventana actual en el editor, en la pestaña "Vista", en el grupo "Opciones", haga clic en "Duplicar vista".

Para cerrar una de las vistas, basta con cerrar la ventana correspondiente. 

**Observación**: Como las vistas se basan en el mismo archivo, un solo elemento abierto aparece en la barra de elementos abiertos.

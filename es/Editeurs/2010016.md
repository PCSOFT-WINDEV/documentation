


## Propiedades del sistema de ayuda
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
Las propiedades del sistema de ayuda permiten definir:

- el nombre de la ayuda ([Pestaña "General"](#NOTE2_1)).

- la página que muestra Default ([Pestaña "General"](#NOTE2_1)).

- el título de la ayuda ([Pestaña "General"](#NOTE2_1)).

- las lenguas soportadas por el sistema de ayuda (["Pestaña "Idiomas](#NOTE3_1)).




Para ver las propiedades del sistema de ayuda, vaya a la pestaña "Modificación", grupo "Sistema de ayuda" y haga clic en "Descripción".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pestaña "General"
<a name="pestana_general_ELTTEXTE000159"></a>
En la pestaña "General", tiene la posibilidad de definir:

- **el nombre lógico del archivo de ayuda**. Este nombre se usará para manipular el archivo de ayuda en WINDEV. En Default, este nombre corresponde al nombre físico del fichero de ayuda. Este nombre se utiliza en la [opciones de generación](../Editeurs/2010015.md). 

- **el Default Page del archivo de ayuda**. Esta página será la primera página que muestre Default. Seleccione esta Page entre las Page s que se encuentran en su sistema de ayuda.

- **el titulo de la ayuda**. Este título se mostrará en la barra de título del archivo de ayuda.

## "Pestaña "Idiomas
<a name="pestana_idiomas_ELTTEXTE000183"></a>
La pestaña "Idiomas" permite definir los distintos idiomas que admite el sistema de ayuda. De hecho, varios idiomas pueden ser soportados por un sistema de ayuda. Para obtener más información, consulte [Creación de un sistema de ayuda multilingüe](../Editeurs/2010025.md).

**Recordatorio**: Por Default, su sistema de ayuda y el proyecto apoyan los mismos idiomas. Para añadir o eliminar idiomas:

1. Abrir la ventana del sistema de ayuda description: 

	- en la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Descripción". 

	- vaya a la pestaña "Idiomas".




2. Seleccione los idiomas que serán soportados por su sistema de ayuda. El botón "Sincronizar con el proyecto" permite tener en cuenta automáticamente todas las lenguas soportadas por el proyecto.

## Presentación
<a name="presentacion_ELTTEXTE000136"></a>
El administrador de SCM (Administrador de Código Fuente) le permite gestionar las ramas de los diferentes proyectos que se encuentran en SCM.

Una rama es una copy del proyecto realizada en un momento dado. Por ejemplo, se puede crear una rama para estabilizar una versión enviada a los usuarios finales.. Los desarrolladores pueden trabajar tanto en la rama como en los elementos programados para la próxima versión.. Para obtener más información, consulte [Gestión de sucursales](../Editeurs/2038016.md).

**Observación**: El administrador de SCM también se puede utilizar para realizar operaciones de mantenimiento en la base de datos., ... Para obtener más información, consulte [Vista general del administrador de SCM](../Editeurs/2038008.md).

## Gestión de sucursales
<a name="gestion_sucursales_ELTTEXTE000160"></a>


### Creación de una sucursal
<a name="creacion_una_sucursal_ELTPARAGRAPHE000028"></a>

Para crear una sucursal:

1. Seleccione el proyecto solicitado en el administrador de SCM.

2. En la pestaña "Directorios y proyectos", en el grupo "Ramas", haga clic en "Crear una Rama".


El asistente de creación de ramas se abre. Para obtener más información, consulte [Creación de una sucursal](../Editeurs/2038016.md).
<a name="NOTE2_2"></a>


### Recuperar las modificaciones realizadas en una rama
<a name="recuperar_las_modificaciones_realizadas_una_rama_ELTPARAGRAPHE000055"></a>

Para recuperar las modificaciones realizadas en una rama: 

1. Seleccione el proyecto solicitado en el administrador de SCM.

2. En la pestaña "Directorios y proyectos", en el grupo "Ramas", haga clic en "Obtener cambios".


La asistente para recuperar las modificaciones hechas a una rama comienza. Para obtener más información, consulte [Recuperación de las modificaciones realizadas en una rama](../Editeurs/2038016.md).
<a name="NOTE2_3"></a>


### Borrar una sucursal
<a name="borrar_una_sucursal_ELTPARAGRAPHE000082"></a>

Para eliminar una rama: 

1. Seleccione la sucursal deseada en el administrador de SCM.

2. Seleccione "Eliminar" en el menú contextual de la rama. 


ATENCIÓN: No se recomienda eliminar una sucursal de SCM. ¡Esta operación no se puede deshacer.

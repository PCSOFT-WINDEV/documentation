### Generar una aplicación Java con WINDEV
<a name="generar_una_aplicacion_java_con_windev_ELTPARAGRAPHE000011"></a>

WINDEV le permite generar una aplicación Java sin siquiera conocer el lenguaje Java. El método para desarrollar la aplicación es idéntico al método para desarrollar cualquier otra aplicación WINDEV.

Cuando se compila el proyecto WINDEV, las ventanas y las funciones WLanguage se convierten automáticamente en Java.

**Observación**: Le aconsejamos que cree un nuevo proyecto WINDEV para desarrollar una aplicación Java en lugar de transformar una aplicación WINDEV en Java.

## Crear un proyecto Java
<a name="crear_proyecto_java_ELTTEXTE000167"></a>


### Creación del proyecto
<a name="creacion_del_proyecto_ELTPARAGRAPHE000024"></a>

Un proyecto Java se crea como cualquier otro proyecto WINDEV. 

Para crear un proyecto: 

- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

- Aparece la ventana de creación: haga clic en "Proyecto".

- El asistente de creación de proyectos se abre.




En uno de los primeros pasos de la asistente, puede elegir cómo se generará su proyecto. Para ello, seleccione "Archivo Java" y continúe.

Una vez que su proyecto es creado, usted desarrolla en WLanguage como para cualquier otra aplicación WINDEV.

**Observación**: El número de funciones WLanguage que pueden utilizarse en un proyecto Java está limitado en esta versión. Este número de funciones aumentará en las próximas versiones. Para obtener más información, consulte [Funciones disponibles en Java](../Editeurs/9000056.md). 
<a name="NOTE2_2"></a>


### Modificar el tipo de generación del proyecto
<a name="modificar_tipo_generacion_del_proyecto_ELTPARAGRAPHE000048"></a>

De forma predeterminada, un proyecto WINDEV para Java corresponde a una configuración de proyecto específica. El tipo de generación del proyecto puede ser modificado en cualquier momento.. Solo debe crear una nueva configuración de proyecto.

Para gestionar las configuraciones del proyecto, vaya a la pestaña "Proyecto", al grupo "Configuración del proyecto" y haga clic en "Administrar configuraciones".. Para obtener más información, consulte [Configuraciones del proyecto](../Editeurs/9000030.md).
<a name="NOTE2_3"></a>


### Características de un proyecto de Java
<a name="caracteristicas_proyecto_java_ELTPARAGRAPHE000072"></a>

Una vez que el proyecto ha sido configurado como un proyecto Java, todas las herramientas disponibles en WINDEV se configuran automáticamente para el proyecto Java:

- La compilación del proyecto indica si su proyecto contiene funciones o controles que no pueden ser utilizados en Java.

- La creación del archivo genera un archivo ".jar" en el directorio EXE del proyecto.

- El módulo instalación de la aplicación sólo gestiona los archivos necesarios para desplegar la aplicación Java.




**Observación**: Le aconsejamos que cree un nuevo proyecto WINDEV para desarrollar una aplicación Java en lugar de transformar una aplicación WINDEV en Java.
<a name="NOTE2_4"></a>


### RAD disponible en Java
<a name="rad_disponible_java_ELTPARAGRAPHE000085"></a>

En un proyecto de Java que maneja archivos de datos, tienes el abomotu de usar el Full il Application RAD o el Window RAD (Simple RAD).

Para utilizar un patrón RAD distinto del Simple RAD, el código generado tendrá que ser adaptado para operar en una aplicación Java.

Para obtener más información, consulte [Manejo de una base de datos de una aplicación de Java](../Editeurs/9000008.md).

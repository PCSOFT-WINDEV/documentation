
## Actualización de automatic (Aplicaciones de la red)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
Durante una red instalación, un reference instalación se encuentra en uno o más servidores reference. Los usuarios instalan su aplicación desde el servidor de reference.

**Esta etapa del asistente permite gestionar las actualizaciones automáticas de las aplicaciones desde el servidor de referencia**.

Estos parámetros pueden modificarse cuando la aplicación se instala en el servidor de reference (para ello, compruebe "Estos parámetros pueden modificarse durante el implementación en el servidor"). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Frecuencia para verificar la disponibilidad de las actualizaciones
<a name="frecuencia_para_verificar_disponibilidad_las_actualizaciones_ELTTEXTE000110"></a>
Por Default, la disponibilidad de una actualización en el servidor se comprueba cada vez que se inicia la aplicación. Esta frecuencia de comprobación se puede configurar. Se puede realizar: 

- Cada vez que se inicia la aplicación. 

- Cada X días.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Interacciones con el usuario al verificar la disponibilidad de las actualizaciones
<a name="interacciones_con_usuario_verificar_disponibilidad_las_actualizaciones_ELTTEXTE000134"></a>
Cada vez que una aplicación es lanzada (por defecto), una aplicación instalada por medio de una "instalación red" se verifica si se encuentra una actualización disponible en el servidor de referencia. 

- Si una versión más reciente está disponible en el servidor, la actualización de la aplicación se propone al usuario final.. 

- Si al abrir la aplicación no hay conexión de red, Default muestra un mensaje de advertencia. 
	Este mensaje puede ser inútil para una aplicación utilizada en un dispositivo móvil, conectada de forma intermitente.. En este caso, simplemente seleccione "Comprobación silenciosa".: No informar al usuario". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_silencieuse.gif)

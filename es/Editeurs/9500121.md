


## AAF: Opciones de edición sencillas en un control Campo de entrada
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000073"></a>
El menú emergente de un control Campo de entrada (clic con el botón derecho del ratón) se utiliza para realizar operaciones de edición sencillas:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_MenuCtxChSaisie.gif)
 Estas sencillas operaciones son las siguientes: 

- "Deshacer": Cancela la última acción realizada. Esta operación también se puede realizar pulsando Ctrl + Z. 

- "Cortar": Recupera el contenido de la control y lo copia en el portapapeles. El control está vacío. Esta operación también se puede realizar pulsando Ctrl + X. 

- "Copiar": Recupera el contenido de la control y lo copia en el portapapeles. El control no está autorizado. Esta operación también se puede realizar pulsando Ctrl + C. 

- "Pegar": Rellena el control con el contenido del portapapeles. Esta operación también se puede realizar pulsando Ctrl + V. 

- "Pegar sin formato": Rellena el control con el contenido del portapapeles sin conservar el formato del texto. Esta operación también se puede realizar pulsando Ctrl + Mayúsculas + V. 

- "Eliminar": Despeja la control. Esta operación también puede realizarse pulsando la tecla [Supr.].

- "Seleccionar todo": Selecciona el contenido de control. También es posible seleccionar el contenido de control pulsando Ctrl + A o realizar tres clics sucesivos en el control.




## WDInst: agrupando los archivos a instalar
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000169"></a>
Por Default, al generar el programa instalación, los archivos a instalar se comprimen en un solo archivo (".WDZ") para ser distribuidos fácilmente.

Cuando se ejecuta el programa instalación, estos archivos se descomprimen automáticamente, se extraen del archivo y se copian al ordenador de instalación. También tiene la posibilidad de no comprimir los archivos a instalar (ver [Copiar y comprimir uno o más archivos](../Editeurs/2028033.md)).

El editor WDInst de instalación le permite agrupar los archivos a instalar (archivos agrupados por tipo por ejemplo). En este caso:

- La generación del editor de instalación crea tantos archivos como el número de grupos. El nombre de los archivos de almacenamiento creados corresponde a "&lt;GroupName>.WDZ".

- Los archivos que no pertenecen a ningún grupo serán archivados en el archivo "__WDInst.WDZ".




Ningún grupo de archivos es creado por Default.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Los grupos
<a name="los_grupos_ELTTEXTE000193"></a>


### Beneficios de los grupos
<a name="beneficios_los_grupos_ELTPARAGRAPHE000030"></a>

- Optimizando la generación del programa instalación para las actualizaciones. 
	Si un grupo no contiene ningún archivo modificado desde la última generación, el archivo correspondiente (archivo".WDZ") no se volverá a crear.

- Organización de los archivos a instalar (crear un grupo para los archivos Image y otro grupo para los archivos de vídeo por ejemplo).

- Uso simplificado de [componentes](../Editeurs/2028032.md) (componente "Multimedia" asociado a los grupos "imagen" y "video", por ejemplo).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000217"></a>


### Administrar grupos
<a name="administrar_grupos_ELTPARAGRAPHE000046"></a>

Los grupos se manejan en el editor de instalación WDInst. Para abrir este editor, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

El programa de creación de instalación asistente no puede ser usado para manejar grupos de archivos.
<a name="NOTE3_2"></a>


### Creación de un nuevo grupo de archivos
<a name="creacion_nuevo_grupo_archivos_ELTPARAGRAPHE000067"></a>

La creación de un grupo se realiza asociando uno de los archivos a instalar a un nuevo grupo.

Para crear un grupo y asociarlo a un archivo:

1. Seleccione el archivo que debe ser associated con el grupo en la ventana "Lista de archivos instalación".

2. En la ventana "File description", muestra la pestaña "Option".

3. Haga clic en el botón "Nuevo".

4. Escriba el nombre del grupo de archivos y valide.



<a name="NOTE3_3"></a>


### Asociar uno o más archivos a un grupo de archivos
<a name="asociar_uno_mas_archivos_grupo_archivos_ELTPARAGRAPHE000081"></a>

Para asociar uno o más archivos a un grupo de archivos: 

1. Seleccione los archivos que desea asociar en la ventana "Lista de archivos de instalación".
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Vaya a la pestaña "Opción" de la ventana "Archivo description".

3. Seleccione el grupo de archivos associated ("Grupo").



<a name="NOTE3_4"></a>


### Asociar uno o más archivos nuevos a un grupo de archivos
<a name="asociar_uno_mas_archivos_nuevos_grupo_archivos_ELTPARAGRAPHE000094"></a>

1. Seleccione "Editar... Añadir archivos".

2. Seleccione los archivos que se agregarán. Para obtener más información, consulte [Añadir uno o más archivos](../Editeurs/2028066.md).

3. Seleccione el grupo de archivos associated haciendo clic en el botón "Otro".

4. Validar la adición de archivos.



<a name="NOTE3_5"></a>


### Disociar uno o más archivos de un grupo de archivos
<a name="disociar_uno_mas_archivos_grupo_archivos_ELTPARAGRAPHE000107"></a>

Para disociar uno o más archivos de un grupo de archivos: 

1. Seleccione los archivos que desea disociar en la ventana "Lista de archivos de instalación". 
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Vaya a la pestaña "Opción" de la ventana "Archivo description".

3. Seleccione "&lt;Ningún grupo&gt;" en el cuadro combinado "Grupo".



<a name="NOTE3_6"></a>


### Viendo los archivos associated con un grupo de archivos
<a name="viendo_los_archivos_associated_con_grupo_archivos_ELTPARAGRAPHE000120"></a>

Para visualizar los archivos associated con un grupo de archivos, en la ventana "Lista de archivos instalación", seleccione el grupo solicitado en el cuadro combinado "Grupo visualizado".




## Verificación de un elemento en la Administrador de Código Fuente (SCM)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
La comprobación de un elemento en el repositorio se utiliza para poner el elemento modificado a disposición de todos los usuarios de Administrador de Código Fuente. El elemento que fue modificado localmente se copia en el proyecto reference en el servidor de SCM.

**Observación**: Los elementos extraídos del Administrador de Código Fuente aparecen rodeados por una línea roja en el editor de proyectos.

**Para registrar un elemento en el repositorio**: 

1. Seleccione este elemento en el panel "Explorador de proyectos".

2. Seleccione "Check in" en el menú contextual.




Otra posibilidad: en la pestaña "SCM", en el grupo "Elemento actual", haga clic en "Reintegrar".

**Importante**: La hoja de estilo es un elemento específico del proyecto. Para reintegrar la hoja de estilo, en la pestaña "SCM", en el grupo "Otras acciones", despliegue "Hoja de estilo" y seleccione "Reintegrar hoja de estilo del proyecto".

## Comprobación de un elemento de nuevo en
<a name="comprobacion_elemento_nuevo_ELTTEXTE000142"></a>
La siguiente pantalla aparece cuando se vuelve a comprobar un elemento en la SCM:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Int%E9gration%20dans%20le%20GDS%20-%20HC%20N%B0010.gif)
Es posible:

- En la pestaña "Tareas", especifique que el elemento registrado corresponde a la ejecución de una tarea o a la finalización de una tarea.

- En la pestaña "Incidentes", especifique que el elemento registrado corrige un incidente.

- En la pestaña "comment", introduzca un comment sobre la operación de facturación. Por ejemplo, puede indicar que la ventana propone una nueva opción de menú.

- Descubra las modificaciones realizadas en el elemento (botón "Mis modificaciones").

- Ver las propiedades del elemento (botón "Propiedades").




Puede reintegrar los cambios realizados en el elemento y mantener el elemento extraído (opción "Conservar elemento extraído").

Cuando un elemento se registra en el Administrador de Código Fuente, está bordeado por un Line negro en el editor del proyecto.

**Observación**: si el elemento registrado no fue modificado, una pantalla específica es mostrada por la SCM.

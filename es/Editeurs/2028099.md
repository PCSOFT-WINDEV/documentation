


## Múltiples instalación del mismo sitio en el mismo servidor
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000093"></a>
WEBDEV le permite instalar automáticamente varias copias del mismo sitio en el mismo servidor, sin tener que renombrar el proyecto.

Un proveedor puede, por ejemplo, instalar el mismo sitio para diferentes clientes en el mismo servidor.

Los archivos de datos pueden ser comunes o estar separados por sitio.

La cuenta WEBDEV puede ser común o diferente, con configuraciones comunes o diferentes.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000117"></a>
Para realizar una instalación múltiple del mismo sitio dinámico de WEBDEV en el mismo servidor, simplemente inicie el sitio instalación asistente: en la pestaña "Proyecto", en el grupo "Generación", despliegue "Implementar sitio" y seleccione "Generar e implementar sitio".

Uno de los pasos de la asistente permite definir el nombre de la implementación del sitio. Esto le permite instalar en el mismo servidor varios sitios con diferentes nombres de implementación. Utilizar la función [ProjectInfo](../WDLang1/3064004.md) con la constante *piNameDeployedSite* permite conocer el nombre real del sitio

Para obtener más información, consulte [Instalación de un sitio dinámico](../Editeurs/2028039.md).


## Entorno del editor del proyecto 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EnvEdProjet.gif)


Los principales elementos del editor del proyecto son los siguientes:


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif)<br> | **Cinta de opciones**<br>Permite un acceso rápido a las principales características del editor de proyectos. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.gif)<br> | **Cuadro de mando del proyecto**<br>Cuadro de mandos que muestra el estado del proyecto actual y permite acceder a todos sus elementos. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif)<br> | **[Acristalamientos verticales acoplables](../Editeurs/2027001.md)**<br>Muestra:<br><br>- los accesos directos a las principales funciones disponibles en el editor actual, <br><br>- las características más comunes, <br><br>- los elementos (ventanas, informes, controles, etc.) disponibles en el proyecto actual.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE4.gif)<br> | **Ventana de descripción**<br>Permite visualizar y/o modificar los ajustes del proyecto. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle5.gif)<br> | **[Abrir documentos](../Editeurs/2029012.md)**<br>Muestra un botón para cada Document abierta en el editor. Para mostrar el elemento solicitado en primer plano, pulse el "botón" correspondiente. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE6.gif)<br> | **[Cristales horizontales acoplables](../Editeurs/2027001.md)**<br>Los cristales solían hacerlo:<br><br>- mostrar los elementos del código (lista de clases, procedimientos locales o globales, ["Panel "Código](../Editeurs/2027009.md)), <br><br>- mostrar los errores de compilación (errores encontrados en el código WLanguage, ["Panel "Errores de compilación](../Editeurs/2027031.md)), <br><br>- realizar búsquedas ([" Panel Buscar - Reemplazar](../Editeurs/2027013.md)), <br><br>- mostrar el depurador durante la prueba de un elemento (["Depurador" pan](../Editeurs/2027030.md)), <br><br>- ...<br><br><br> |

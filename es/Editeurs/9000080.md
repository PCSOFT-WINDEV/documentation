## Presentación
<a name="presentacion_ELTTEXTE000103"></a>
Para ayudarle a gestionar la navegación de los usuarios de la Web en sus sitios, una función avanzada para analizar las dependencias y las imágenes huérfanas está disponible en WEBDEV. Esta característica completa a [el análisis del código muerto y los elementos huérfanos](../Editeurs/9000075.md).

Se le notificará inmediatamente de enlaces rotos reales o potenciales.. De hecho, las cejas no están fijas en modo dinámico.

También se notifican las imágenes no utilizadas y las páginas huérfanas.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000127"></a>


### Análisis de dependencias e imágenes huérfanas
<a name="analisis_dependencias_imagenes_huerfanas_ELTPARAGRAPHE000025"></a>

Para analizar las dependencias y las imágenes huérfanas, en la pestaña "Proyecto", en el grupo "Auditoría y rendimiento", amplíe "Auditoría estática" y seleccione "Analizar dependencias".

La ventana de resultados de este análisis enumera:

- **las imágenes huérfanas** (que no son utilizados por ninguna página). Es posible:

	- Borrar las imágenes seleccionadas del disco.

	- Archive las imágenes seleccionadas en un archivo.zip y elimínelas.

	- Ignorar las imágenes seleccionadas. En este caso, las imágenes ignoradas pueden mostrarse haciendo clic en "Ignorar imágenes huérfanas".




- **las dependencias**, es decir, los archivos relacionados con otros archivos. Para cada elemento del proyecto WEBDEV (columna"Elemento"), se listan todas las dependencias (columna"Archivo").. Tienes la habilidad de ver:

	- todas las dependencias,

	- sólo las dependencias relativas a las páginas HTML,

	- sólo las dependencias relativas a las imágenes,

	- sólo las dependencias relativas a los archivos JavaScript,

	- otras dependencias.







Estas dependencias se pueden visualizar para el proyecto, las páginas del proyecto, los estilos o los informes.

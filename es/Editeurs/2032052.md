## Presentación
<a name="presentacion_ELTTEXTE000134"></a>
Una consulta se utiliza para interrogar una base de datos con el fin de ver, insertar, modificar o eliminar datos. La estructura de consulta define los datos manipulados.

Una consulta puede interrogar uno o más archivos de datos.

El editor de consultas permite crear consultas fácilmente sin tener que programar.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RequeteRepresentation.gif)
**Por ejemplo**, en la aplicación de Gestión de ventas, desea obtener:

- los pedidos realizados en marzo de 2015. Se debe crear una consulta para seleccionar los pedidos realizados entre el 01/03/2015 y el 31/03/2015 en el archivo ORDERS.
	La consulta QRY_March2015 interroga el archivo ORDERS. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_ExempleUnFichier.gif)


- el número de pedidos realizados por clientes en París (Francia). Se debe crear una consulta que interrogará:

	- el archivo CUSTOMER para seleccionar los clientes que viven en París (Francia). 

	- el archivo ORDERS para ver el número de pedidos.







La consulta QRY_OrdersCustomerParisFrance interroga dos archivos de datos (CUSTOMER y ORDERS). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_ExemplePlusieursFic.gif)
**Observación**: Para obtener más información sobre cómo manipular una consulta en un proceso WLanguage, consulte [Hacer referencia a un elemento en un proceso WLanguage](../WDChamp/1014007.md).

## Características de una consulta
<a name="caracteristicas_una_consulta_ELTTEXTE000158"></a>
Los elementos y características de una consulta son los siguientes:

- [campos](../Editeurs/2032004.md),

- [combinaciones](../Editeurs/2032018.md),

- [condiciones de selección](../Editeurs/2032019.md),

- [serie de condiciones de selección](../Editeurs/2032055.md),

- [operaciones en una consulta](../Editeurs/2032020.md),

- [criterio de ordenación de los registros](../Editeurs/2032005.md),

- [posibilidad de ver los primeros/últimos n registros de una consulta](../Editeurs/2032022.md),

- [posibilidad de editar los valores que se muestran en el resultado de la consulta](../Editeurs/2032016.md),

- [posibilidad de ver los registros de un rango de tiempo determinado](../Editeurs/2032029.md).




Las características de la consulta se pueden modificar en cualquier momento, incluso después de haberla creado. Para obtener más información, consulte [Definir una consulta](../Editeurs/2032038.md).
<a name="NOTE2_2"></a>


### Crear una consulta desde el editor de reportes
<a name="crear_una_consulta_desde_editor_reportes_ELTPARAGRAPHE000078"></a>

Al crear un reporte, se puede definir una consulta. Esta consulta es conocida como **consulta incrustada**.

Para obtener más información, consulte [Modo de creación de consultas](../Editeurs/2032040.md).

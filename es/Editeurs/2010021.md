


## Selección de las palabras clave associated con un Page
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
La ventana de selección de palabras clave permite seleccionar las palabras clave que se associated con una ayuda Page. Esta ventana también se puede utilizar para crear las palabras clave y subpalabras clave necesarias.

Los elementos creados, aunque no sean associated con la ayuda Page, se mostrarán en la lista de palabras clave del editor de ayuda.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Motcle.gif)
 Para abrir la ventana de selección de palabras clave Page: 

- Abra la ayuda Propiedades Page ("Propiedades Page" en el menú contextual). 

- En la pestaña "Índice", pulse el botón "Añadir".




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ventana de gestión de palabras clave
<a name="ventana_gestion_palabras_clave_ELTTEXTE000116"></a>
Esta ventana presenta las diferentes palabras y subpalabras clave ya encontradas en el índice.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SaisieMotcle.gif&type=thumb)


**Para seleccionar una palabra clave y asociarla con la ayuda actual Page**:

1. Haga clic en la palabra clave (o subpalabra clave) deseada. Desglose la estructura de árbol si es necesario.

2. Haga clic en el botón "->".




**Para crear una nueva palabra clave**:

1. Escriba la palabra clave que desea crear en el área "Añadir una clave o una subclave".

2. Haga clic en el botón "Crear clave".

3. La nueva clave se muestra en la estructura de árbol de las palabras clave disponibles. Seleccione esta palabra y haga clic en el botón "->" para asociarla a la página de ayuda actual.




**Para crear una nueva palabra clave secundaria**:

1. Seleccione la palabra clave "padre" de la subpalabra clave que desea crear.

2. Escriba la palabra clave que desea crear en el área "Añadir una clave o una subclave".

3. Pulse el botón "Subclave. La nueva clave se muestra en la estructura de árbol de las palabras clave disponibles. Seleccione esta palabra y haga clic en el botón "->" para asociarla a la página de ayuda actual.


**Observación**: Los símbolos + y - indican si la palabra clave contiene subpalabras clave colapsadas (+) o si la estructura de árbol de la palabra clave está expandida (-).

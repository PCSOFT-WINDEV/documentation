## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
WEBDEV le permite gestionar todos los estilos WEBDEV utilizados en un proyecto. Los estilos de WEBDEV pueden ser vistos para cada tipo de control.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Edición de la hoja de estilo WEBDEV de un proyecto
<a name="edicion_hoja_estilo_webdev_proyecto_ELTTEXTE000121"></a>
Para editar los estilos WEBDEV de un proyecto:

1. Selecciona un tipo de control para el que quieras averiguar los estilos de WEBDEV disponibles (por ejemplo, una control Botón, para ver los diferentes estilos de WEBDEV associated con ella). 

2. En la pestaña "Modificación", en el grupo "Estilo", haga clic en "Estilos".

3. Se muestra la lista de estilos WEBDEV del proyecto que pueden ser associated con el control seleccionado (control Botón en nuestro ejemplo).




Si se han definido uno o más estilos de WEBDEV en el proyecto para un tipo específico de control, no se puede definir ningún estilo CSS de Default para este tipo de control. El primer estilo de WEBDEV encontrado en la lista corresponderá al estilo Default para este tipo de control. El estilo Default no puede ser modificado.

**Observación**: La casilla de verificación "Local" devuelve la lista de estilos asociados a la hoja de estilo asociada a la página (pestaña "Estilo" de la página description).

Esta ventana permite:

- Modificar el estilo seleccionado. Si el estilo no es un estilo local, esta modificación se aplicará a todos los controles que usen este estilo.

- Crea un nuevo estilo.

- Duplicar estilo seleccionado.

- Eliminar el estilo seleccionado: en este caso, el estilo associated con el control será el mismo pero será local al control.




La ventana de edición de estilo incluye las características de estilo correspondientes a la control para la que se definió el estilo. Para obtener más información, consulte [Editar ventana de estilos WEBDEV](../Editeurs/2016002.md).

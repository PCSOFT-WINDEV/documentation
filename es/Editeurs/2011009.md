## Crear un campo
<a name="crear_campo_ELTTEXTE000238"></a>
Para crear un elemento: 

1. Haga doble clic en el archivo de datos en el que se creará el campo. Se muestra la ventana description de los elementos del archivo.

2. Haga clic en la primera fila vacía en la tabla de elementos o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%201.GIF).

3. En el tabla, especifique: 

	- el nombre del artículo,

	- el título del artículo,

	- el tipo de artículo y su tamaño si es necesario,

	- si el artículo se refiere a datos personales (marca de verificación GDPR). Para obtener más información, consulte [Definición de los elementos "Datos personales](../Editeurs/1410086744.md).


 Observación: Para obtener más información, consulte [Los diferentes tipos de artículos](../Editeurs/2011026.md).

4. En la parte derecha de la ventana, indique los detalles del nuevo elemento (tipo de clave, orden de clasificación, formato, etc.).

5. Valide la descripción del campo.

## Gestión de una posición
<a name="gestion_una_posicion_ELTTEXTE000262"></a>


### Duplicar/Copiar un elemento
<a name="duplicarcopiar_elemento_ELTPARAGRAPHE000068"></a>

Cuando se crea un artículo cuyo nombre es idéntico al nombre de un artículo existente, WINDEV, WEBDEV y WINDEV Mobile proponen automáticamente utilizar la description del artículo inicial.

Un artículo también puede duplicarse: 

- en el mismo archivo.

- en un archivo diferente.  




Para duplicar un artículo en el mismo expediente, directamente desde la ventana description del artículo: 

1. Seleccione el elemento que desea duplicar en la ventana description de un archivo de datos

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%209.gif). El artículo se duplica automáticamente. Su nombre es el nombre del artículo original acompañado de un número.




Para duplicar un elemento en otro fichero: 

1. Seleccione el elemento que desea duplicar en la ventana description de un archivo de datos

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%2010.gif). 

3. En la ventana que aparece, indique el fichero description en el que se duplica automáticamente el elemento. 

4. Validar. El elemento aparece en la description del fichero seleccionado.



<a name="NOTE3_2"></a>


### Borrar un elemento
<a name="borrar_elemento_ELTPARAGRAPHE000098"></a>

Para borrar un elemento:

1. Seleccione el elemento a eliminar en la description de elementos de archivo.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%202.gif). El artículo se borra automáticamente de el archivo de datos.




**Observaciones**: 

- Cuando se crea un elemento cuyo nombre es idéntico al nombre de un elemento previamente borrado, WINDEV, WEBDEV y WINDEV Mobile proponen automáticamente restaurar la description del elemento borrado.

- La eliminación de un elemento desencadena la eliminación de datos durante la siguiente modificación de datos de automatic, incluso si se crea un nuevo elemento con el mismo nombre.



<a name="NOTE3_3"></a>


### Cambiar el nombre de un elemento
<a name="cambiar_nombre_elemento_ELTPARAGRAPHE000115"></a>

Para cambiar el nombre de un elemento:

1. Seleccione el elemento a renombrar en la description de elementos de archivo.

2. Escriba el nombre del nuevo elemento.

3. Valide los cambios. El elemento se renombra automáticamente.



<a name="NOTE3_4"></a>


### Utilizar un diccionario
<a name="utilizar_diccionario_ELTPARAGRAPHE000126"></a>

Un elemento puede ser incluido en un diccionario para ser reutilizado en una nueva aplicación, por ejemplo. Para añadir un elemento a un diccionario, basta con arrastrarlo desde la editor de análisis y soltarlo en el diccionario (en la ["Panel"Diccionario](../Editeurs/2027014.md)).

Cuando se incluye en un diccionario, el artículo description puede suscribirse a las modificaciones. Cualquier modificación realizada en el elemento description se aplicará al objeto de diccionario y a todos los usos de este objeto.

Para obtener más información, consulte [Diccionario](../Editeurs/2023002.md).

## Clasificando los elementos que aparecen en el archivo de datos description
<a name="clasificando_los_elementos_que_aparecen_archivo_datos_description_ELTTEXTE000304"></a>
Los elementos de los archivos de datos pueden visualizarse de acuerdo con los siguientes criterios: 

- orden lógico. 

- orden físico. 

- por orden alfabético.




Para modificar el orden de visualización de los elementos: 

1. Haga doble clic en el archivo de datos. Se muestra la ventana description de los elementos del archivo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001.gif&type=thumb)


2. Los botones que se encuentran en la sección "Display" sirven para ordenar los elementos que se muestran en esta ventana: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%203.gif)


	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%204.gif) clasifica por orden alfabético. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%205.gif) clasifica los artículos e índices en orden físico. 

- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0002.gif) clasifica los elementos en orden lógico. 

3. El embudo de la columna "Nombre" (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Manipulations_rubriques%20-%20HC%20N%B0001%206.gif)) permite realizar una búsqueda: basta con hacer clic en el embudo y empezar a escribir la palabra deseada. Se mostrarán todos los elementos coincidentes.

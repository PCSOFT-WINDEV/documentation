


## Restauración de una copia de seguridad de un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000087"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten guardar regularmente los diferentes elementos de su proyecto en un archivo (extensión ZIP).. Este archivo contiene todos los elementos de proyecto que se han grabado. Para obtener más información, consulte [Haciendo una copia de seguridad del proyecto](../Editeurs/2030014.md). 

El proyecto archivado puede ser restaurado en cualquier momento. Los proyectos se restauran desde WINDEV, WEBDEV o WINDEV Mobile.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000111"></a>


### Para restaurar un archivo de proyecto:
<a name="para_restaurar_archivo_proyecto_ELTPARAGRAPHE000023"></a>

1. En la pestaña "Proyecto", en el grupo "Otras acciones", amplíe "Copia seg./Restaurar" y seleccione "Restaurar un proyecto". Se muestra la asistente para restaurar un proyecto.

2. Especifique el archivo de almacenamiento que contiene el proyecto que desea restaurar.

3. Seleccione el directorio para restaurar el proyecto.

4. Validar. El proyecto se restaura en el directorio especificado.

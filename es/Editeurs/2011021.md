
## ¿Qué es una base de datos? 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000071"></a>
Una **base de datos** es una colección organizada de datos para un fin determinado (por ejemplo, gestionar pedidos). Estos datos se organizan y almacenan en una base de datos para que los usuarios puedan usarlos fácilmente.

Todos los días usamos bases de datos: una simple libreta de direcciones es una base de datos.

La libreta de direcciones permite almacenar nombres, direcciones, números de teléfono, etc. El índice alfabético permite encontrar una persona fácilmente.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=base-de-d1.gif)


En una base de datos computarizada, encontramos los mismos conceptos que en la libreta de direcciones:

- la información almacenada corresponde a los **campos**.

- los grupos de información corresponden a los **archivos de datos**,

- los índices se crean a partir de campos de ordenación (denominados **claves**).

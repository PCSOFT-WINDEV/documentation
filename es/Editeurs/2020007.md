## Presentación
<a name="presentacion_ELTTEXTE000064"></a>
Algunas características de WINDEV utilizan el GDI+ framework.

Esta framework específica le permite beneficiarse de todos los recursos gráficos del sistema.

Esta framework contiene una biblioteca específica: gdiplus.dll.

Esta biblioteca se suministra con los sistemas recientes (posteriores a Windows 2000).

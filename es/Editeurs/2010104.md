
## Submodelado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000133"></a>
En un modelado flexible, los elementos "Application", "Process" y "Choice" pueden utilizarse para introducir un "submodelado".

Por ejemplo, el description del ERP (Enterprise Resource Planning) de una empresa puede incluir los siguientes submodelos: Gestión de clientes, gestión de presupuestos, marketing, producción, contabilidad,...

## Manejo del sub-modelo
<a name="manejo_del_submodelo_ELTTEXTE000157"></a>


### Creación de un sub-modelo
<a name="creacion_submodelo_ELTPARAGRAPHE000028"></a>

Para crear un sub-modelo a partir de un elemento:

1. Seleccione el elemento que debe ser relacionado a un sub-modelo (Aplicación, Process u Opción).

2. Seleccione "Añadir un sub-modelo" de el menú contextual del elemento.

3. El sub-modelado se crea y se abre automáticamente en el editor.




**Observación: también es posible asociar un elemento a un submodelado existente.** Para hacerlo, debe:

1. Seleccione el elemento que debe ser relacionado para el sub-modelado.

2. Seleccione "Seleccionar un sub-modelo existente" de el menú contextual del elemento.

3. Seleccione el sub-modelo deseado. Este sub-modelo puede pertenecer al modelado actual o a otro modelado.

4. Valide.

5. El elemento aparece con el símbolo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=modelisation_sous_vide.gif).



<a name="NOTE2_2"></a>


### Entrando en un sub-modelo
<a name="entrando_submodelo_ELTPARAGRAPHE000049"></a>

Para entrar en el sub-modelado de un elemento:

1. Seleccione el elemento deseado.

2. Seleccione "Entrar en el sub-modelo" de el menú contextual del elemento.

3. El sub-modelado se muestra automáticamente.




**Observación**: **si el elemento es relacionado para un sub-modelado existente**, usted tiene la capacidad de mostrar este sub-modelado:

1. Seleccione el elemento deseado.

2. Seleccione "Seguir el enlace al sub-modelo" de el menú contextual del elemento.

3. El sub-modelado se muestra automáticamente.




Un sub-modelo puede tener varios modelos "padres".
<a name="NOTE2_3"></a>


### Volviendo al nivel anterior de modelado
<a name="volviendo_nivel_anterior_modelado_ELTPARAGRAPHE000068"></a>

Para volver al nivel anterior, puede utilizar:

- la barra de título del sub-modelo. Los diferentes modelos están representados en el siguiente formato: "Modelado1&gt;Sub-Modelado2&gt;Sub-Modelado3". Se utiliza un clic en el modelado solicitado para visualizarlo.

- los diferentes paneles a la izquierda del editor. Cada panel corresponde a un modelado o a un sub-modelado. Se utiliza un clic en el modelado solicitado para visualizarlo.

- "Back to parent" de el menú contextual del sub-modelo.



<a name="NOTE2_4"></a>


### Borrar un sub-modelo
<a name="borrar_submodelo_ELTPARAGRAPHE000079"></a>

Para borrar un sub-modelo:

1. Seleccione el elemento deseado.

2. Seleccione "Borrar el submodelo" en el menú contextual. 




Observación: Si el elemento es relacionado a un sub-modelo existente, seleccione "Borrar el enlace al sub-modelo".

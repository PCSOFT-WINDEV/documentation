


## Opciones del editor de ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000294"></a>
Para configurar las opciones generales del editor de ventanas: 

1. En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de ventanas". 

2. Aparece la ventana de configuración. Las posibles modificaciones se agrupan en 4 temas: 

	- Magnetismo.

	- Clic.

	- Opciones.

	- Controles.

<a name="NOTE2_1"></a>


## Magnetismo
<a name="magnetismo_ELTTEXTE000318"></a>


### Configuración de la rejilla
<a name="configuracion_rejilla_ELTPARAGRAPHE000038"></a>

La rejilla de encaje a presión simplifica el posicionamiento y la alineación de los mandos de las ventanas.

El tamaño de la cuadrícula se expresa en píxeles.

Opciones recomendadas para la red:

- Rejilla de Invisible.

- Cuadrícula activa.

- Distancia vertical de la rejilla: 12 píxeles.

- Distancia horizontal de la rejilla: 12 píxeles.




Si el espaciado de la cuadrícula es set a 1, la cuadrícula no será Visible.

Para obtener más información, consulte [Cuadrícula magnética](../Editeurs/2022002.md).


### Posicionamiento automático
<a name="posicionamiento_automatico_ELTPARAGRAPHE000059"></a>

Cuando se mueve un control en una ventana, las guías son mostradas automatic LEMENTE por el verificador de interfaz en tiempo real (también llamado posicionamiento automatic). Los objetos se ajustan a las guías, lo que le permite alinear el control movido con los demás controles de la ventana.

Usted tiene la capacidad de habilitar (o no) la posición y las reglas.. 

Para obtener más información, consulte [Posicionamiento automático](../Editeurs/2022003.md). 


### Configuración de las reglas
<a name="configuracion_las_reglas_ELTPARAGRAPHE000071"></a>

Al crear o modificar una ventana en el editor de ventanas, tiene la posibilidad de mostrar las reglas de alineación. 

Estas reglas tienen guías magnéticas: cuando un control se acerca a una guía, esta lo "atrae" automáticamente. Esta función permite posicionar, alinear o redimensionar fácilmente los controles en una ventana.

Tiene la posibilidad de hacer visibles (o no) las reglas y las guías y de configurar la Color de las guías. 

Para obtener más información, consulte [Reglas de alineación (Editor)](../Editeurs/2022001.md).

## Haga clic
<a name="haga_clic_ELTTEXTE000354"></a>


### Acción de doble clic
<a name="accion_doble_clic_ELTPARAGRAPHE000089"></a>

Por defecto, al hacer doble clic en un control se abre la ventana de description. 

Puedes cambiar este comportamiento de Default. 

El doble clic puede: 

- **Editar descripción**: Abra la ventana de control description (opción Default).

- **Abrir código**: Abre la ventana del código associated con el control. 

- **Seleccionar un estilo**: Abre la ventana que se utiliza para elegir el estilo de control. 





### Un solo clic
<a name="solo_clic_ELTPARAGRAPHE000114"></a>

Un solo clic en una control realiza una acción específica. Un solo clic en la control le permite:

- Editar el texto del control (opción Default). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LibelleModifDirect.gif)


- Cambiar tamaño del control después de editar el texto con un solo clic. 

- Editar el nombre del control después de cambiar el texto con un solo clic, si es posible (opción Default).

## Opciones
<a name="opciones_ELTTEXTE000384"></a>


### Por cada selección
<a name="por_cada_seleccion_ELTPARAGRAPHE000128"></a>

Cada vez que se selecciona un control en la ventana, tienes la posibilidad de: 

- mostrar las propiedades de la control seleccionada en el panel "Modificador. Para obtener más información, consulte [El panel "Modificador](../Editeurs/2027006.md). 

- mostrar los procesos del elemento seleccionado en el panel "Vista de código contextual. Para obtener más información, consulte [Panel "Vista de código sensible al contexto](../Editeurs/2027033.md). 





### Posición de la ventana WINDEV en la pantalla de inicio
<a name="posicion_ventana_windev_pantalla_inicio_ELTPARAGRAPHE000142"></a>

Por Default, la ventana del WINDEV está centrada en la ventana de la casa. También se puede colocar en la parte superior izquierda.

## Controles
<a name="controles_ELTTEXTE000420"></a>


### Lista de fuentes
<a name="lista_fuentes_ELTPARAGRAPHE000163"></a>

Para modificar el tipo de letra de los controles de ventana, utilice el cuadro combinado "Fuente" que se encuentra en la pestaña "Estilo" del control description.

En Default, sólo se muestran en el cuadro combinado "Fuente" los nombres de las fuentes instaladas en el ordenador de desarrollo.

En Windows, la misma fuente puede ser associated con varios grupos de idiomas. Para rellenar cada nombre de fuente con el idioma associated, sólo tiene que marcar "Mostrar todos los idiomas disponibles".

Observación: Esta opción se tendrá en cuenta durante la próxima apertura de la ventana.


### Otras opciones
<a name="otras_opciones_ELTPARAGRAPHE000174"></a>

Las otras opciones disponibles son las siguientes: 

- **Tecla de acceso rápido automática al crear un control**: Esta opción se utiliza para crear automáticamente un atajo cuando se crea el control. Este acceso directo puede modificarse posteriormente. Para obtener más información, consulte [Atajo de teclado y tecla de acceso directo](../WDChamp/1014018.md). 

- **Mostrar un símbolo de representación en los controles Imagen en edición**: Esta opción se utiliza para mostrar (o no) una cruz en los controles de Image. 
	<br>![Image con retícula](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Image_Mire%20-%20HC%20N%B0001.gif)<br>![Image sin retícula](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Image_Mire%20-%20HC%20N%B0001%201.gif)

	Esta opción también puede ser desactivada control por control. Para obtener más información, consulte [Control Imagen](../WDChamp/1013218.md). 

- **Calcular anclajes al copiar/pegar**: Esta opción se utiliza para calcular automáticamente las mejores opciones de anclaje para un control cuando se realiza un copy/paste en una ventana. 

- **Forzar vista previa de las ventanas internas al 100%**: Se usa para forzar la vista previa de las ventanas internas al 100% en los controles de la ventana interna.. Por lo tanto, la ventana interna se visualiza correctamente y el tamaño de la control Ventana interna se puede adaptar en el editor.. Para obtener más información, consulte [La control Ventana interna](../WDChamp/1013255.md). 

- **Resaltar esquinas al pasar el cursor sobre el control**: Se utiliza para ver los límites de la control durante el vuelco. 

- **Mostrar un error de compilación de UI si la ventana no se ha actualizado con la plantilla**: Se utiliza para mostrar (o no) un error específico de la UI si la plantilla no está udpada. 

- **Adaptar zoom al DPI de la pantalla al abrir las ventanas**: Permite optimizar el tamaño de visualización de la ventana en el editor.. 

- **Mostrar sombras de ventanas (en el editor)**: Se utiliza para mostrar (o no) una sombra bajo el contorno de las ventanas que se están editando en el editor de ventanas. 

- **Desactivar la auditoría de UI para todos los proyectos**: Desactiva la auditoría de la interfaz de usuario en todos los proyectos de Default. Se puede volver a activar sólo para los proyectos deseados. 

- **Tamaño predeterminado de controles Campo de entrada**: Se utiliza para definir el tamaño de la Default (en número de caracteres) de la control Campo de entrada cuando se crean.

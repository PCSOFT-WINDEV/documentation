


## Generar un diagrama de clases mediante ingeniería inversa
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000079"></a>
WINDEV, WINDEV Mobile y WEBDEV permiten crear el diagrama de clases de una aplicación aplicando ingeniería inversa a las clases (archivos ".WDC") de la aplicación. Las clases del proyecto se analizan y el diagrama de clases se crea con base en la información del análisis.

**Observación**: Para crear un diagrama de clases mediante ingeniería inversa, la aplicación WINDEV debe contener clases WINDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000103"></a>
Para crear un diagrama de clases mediante ingeniería inversa:

1. Abra el proyecto en el que se debe crear el diagrama de clases.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido.

	- La ventana de creación de nuevos elementos se abre: haga clic en "Arquitectura", y luego en "UML".

	- El asistente de creación de diagramas UML se abre.




3. Seleccione "Diagrama de clases".

4. Especifique el modelo UML en el que se insertará el diagrama.

5. Elija generar el diagrama de clases mediante ingeniería inversa.

6. Valide la creación del diagrama de clases. El nuevo diagrama de clases aparece en el editor.




## AAF: Controles de edición de la ruta del archivo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000144"></a>
Los controles de edición que utilizan la máscara de entrada "File name and full ruta" o "File name without ruta" proponen varios AAFs (Automatic Application Features): 

- [un menú emergente específico utilizado para manipular el archivo](#NOTE2_1). 

- [una vista previa del archivo en formato de miniatura (para un archivo Image o PDF ruta)](#NOTE3_1).  




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Un menú emergente específico
<a name="menu_emergente_especifico_ELTTEXTE000168"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000025"></a>

Se propone automáticamente un menú emergente para los controles de edición utilizados para introducir un nombre de archivo y ruta. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=faa_Chemin_de_fichier%20-%20HC%20N%B0002.gif)


Este menú emergente contiene las siguientes opciones específicas de los archivos: 

- "Navegar": Esta opción se utiliza para abrir el selector de archivos de Windows y seleccionar el archivo solicitado.. 

- "Abrir": Esta opción se utiliza para abrir el archivo en el software associated en el ordenador actual. 

- "Abrir con": Esta opción se utiliza para abrir el archivo en uno de los programas propuestos por el sistema.. 

- "Abrir la ubicación del archivo": Esta opción se utiliza para abrir el explorador de archivos en el directorio que contiene el archivo. 

- "Print": Esta opción se utiliza para imprimir el archivo. 

- "Enviar por correo electrónico": Esta opción se utiliza para enviar el archivo por Email. El software de mensajería utilizado se abre automáticamente y el archivo se añade como archivo adjunto.. 




Observación: La opción "Examinar" también está disponible para los controles de edición cuyo tipo es "Directorio ruta".


### Implementación
<a name="implementacion_ELTPARAGRAPHE000046"></a>

El menú emergente para los archivos y las rutas de los archivos se implementa automáticamente desde la versión 17. 

Como todos los menús popup propuestos por las AAF, tienes la posibilidad de configurar las opciones de Default propuestas por el control. Ver [Configuración de los menús de AAF](../Editeurs/2010040.md) para más detalles. 

Por programación: 

- [AAFExecute](../WDLang1/1000022099.md) ([EjecutarAAF](../WDLang1/3013060.md)) associated con constantes específicas se utiliza para ejecutar directamente las opciones de el menú contextual.

- [AAFDisable](../WDLang1/1000022018.md) (o [DisableAAF](../WDLang1/3013061.md)) associated con constantes específicas se utiliza para eliminar algunas opciones de el menú contextual del control Campo de entrada especificado.

## Miniaturas para previsualizar las imágenes o los archivos PDF
<a name="miniaturas_para_previsualizar_las_imagenes_los_archivos_pdf_ELTTEXTE000198"></a>


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000075"></a>

Cuando se introduce o se pasa el cursor por un archivo ruta, si este archivo es una imagen o un PDF Document, se muestra automáticamente una miniatura redimensionable. 

Por lo tanto, el usuario puede comprobar si la imagen adecuada o Document fue seleccionada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=faa_Chemin_de_fichier%20-%20HC%20N%B0001.gif)
 El tamaño de la miniatura se calcula automáticamente. 

Observación: Si es necesario, el usuario puede desactivar la visualización de las miniaturas a través de la opción "Desactivar la vista previa de Image" que se encuentra en el menú contextual del archivo ruta. 


### Implementación
<a name="implementacion_ELTPARAGRAPHE000087"></a>

La visualización de los archivos Image y los archivos PDF en una miniatura se implementa automáticamente a partir de la versión 17. 

En la programación, la constante [AAFDisable](../WDLang1/1000022018.md) (o [DisableAAF](../WDLang1/3013061.md)) asociada a la constante *aafDisplayThumbnail* desactiva la visualización automática de la miniatura del archivo para el control Campo de entrada especificado.

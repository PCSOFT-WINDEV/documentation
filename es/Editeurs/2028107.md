
## Implementación en la nube para aplicaciones PC SOFT
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000128"></a>
PC SOFT le permite alojar sus sitios en CLOUD para aplicaciones de PC SOFT.. La instalación se realiza fácilmente, directamente desde el entorno. 

Una vez que su sitio está desarrollado, usted tiene la posibilidad de instalarlo directamente en su plataforma que se encuentra en el CLOUD. 

Para obtener más información, consulte [http://www.pcscloud.net](http://www.pcscloud.net).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Etapas a seguir
<a name="etapas_seguir_ELTTEXTE000152"></a>
Los pasos para instalar un sitio WEBDEV en una plataforma PC SOFT CLOUD son los siguientes:

1. [Creación de una cuenta CLOUD si es necesario](#NOTE3).

2. [Introducción de los parámetros de cuenta](#NOTE3_2).

3. [Selección de los archivos a instalar](../Editeurs/2028058.md). 

4. [Modificación automática](../Editeurs/2028048.md). 

5. [Parámetros del sitio](../Editeurs/2028051.md). 

6. Transferencia de archivos y instalación. 
	Los mensajes de error pueden ser mostrados en diferentes pasos de la instalación remota. Para obtener más información, consulte [Errores durante una instalación por FTP](../Editeurs/2028057.md).

7. Cuando la instalación se completa, la asistente provee el comando Line usado para iniciar el sitio. Este Line puede ser copiado (si es necesario) en el portapapeles a través de el menú contextual.




Observación: Los parámetros de la implementación especificados en la asistente son almacenados y propuestos de nuevo durante la próxima implementación.





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Creación de una cuenta CLOUD
<a name="creacion_una_cuenta_cloud_ELTTEXTE000176"></a>
Se puede crear una cuenta CLOUD directamente en el sitio CLOUD ([http://www.pcscloud.net](http://www.pcscloud.net)). 

Atención: El uso de la NUBE está sujeto a un contrato especial y a un precio.. 

<a name="NOTE3_2"></a>


## Parámetros de la cuenta
<a name="parametros_cuenta_ELTTEXTE000200"></a>
Una vez creada la cuenta CLOUD: 

1. Debe especificar los parámetros de su cuenta de PC SOFT CLOUD: 

	- el usuario

	- la contraseña




2. Validar para iniciar la instalación del sitio. 




El botón "Parámetros avanzados" se utiliza para definir los parámetros de implementación: 

- Parámetros FTP.

- HTTP/HTTPS parameters.

- Parámetros de proxy HTTP/HTTPS.

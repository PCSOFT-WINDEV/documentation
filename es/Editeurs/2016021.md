## Secuencia de aplicación de las paletas de Color
<a name="secuencia_aplicacion_las_paletas_color_ELTTEXTE000283"></a>
La paleta de colores de una página se puede definir en tres niveles diferentes (en orden descendente de prioridad):

- En el proyecto.

- En el modelo de páginas del que la página hereda propiedades.

- En la misma página.




Por lo tanto, la Palette general debe ser definida a nivel de proyecto. Si un conjunto de páginas (por ejemplo, las páginas de "Administración" de un sitio) debe utilizar una paleta específica, puede redefinir la paleta del proyecto como un modelo de páginas. Finalmente, si una página específica debe tener una paleta diferente, puede elegir la paleta solo para esa página.

WEBDEV permite: 

- [Definir la Palette utilizada](#NOTE3_1). 

- [Crear una nueva paleta](#NOTE4_2).

- [Modificar un Palette existente](#NOTE4_3).

- [Duplicar una paleta](#NOTE4_4).




Observación: se puede acceder a los colores de las paletas mediante programación ([Paleta de colores](../WDLang2/1000019905.md)). Esto le permite crear un código genérico, independiente de la tema y la Palette utilizadas.

## Definiendo el Color Palette utilizado
<a name="definiendo_color_palette_utilizado_ELTTEXTE000307"></a>
Las paletas de Color definen un set de Color que se proponen en todos los recolectores de Color del proyecto.

Los colores también se aplican a los estilos definidos en el tema del proyecto para cambiar fácil y rápidamente el aspecto visual de un proyecto.

Cuando se crea un proyecto WEBDEV, la asistente permite elegir la Color Palette que se utilizará. Esta elección puede ser modificada en cualquier momento. 

**To modificar el color Palette utilizado por un proyecto**:

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Selecciona la pestaña "Pieles".

3. Elija un Palette entre los Palette s preestablecidos.

4. Valide la ventana de descripción del proyecto. 




**To modificar el Color Palette utilizado por una página modelo o por una página**:

1. Abra la ventana de descripción de la página: en la pestaña "Página", en el grupo "Descripción", haga clic en "Descripción".

2. Seleccione la pestaña "Estilo".

3. Haga clic en el botón de búsqueda para seleccionar un Color Palette.

4. Si el Palette seleccionado es "Como el proyecto", se utilizará el Palette del proyecto.




**Observación**: El color Palette usado por un proyecto (nuevo Palette o Palette modificado) se copia en el directorio del proyecto. Cuando el proyecto se utiliza en SCM, el Color Palette también se incluye en los elementos gestionados por el SCM.

## Gestionar las paletas de Color
<a name="gestionar_las_paletas_color_ELTTEXTE000331"></a>
Para aumentar las posibilidades de las paletas, WEBDEV le permite:  

- Crear una nueva paleta. 

- Modificar un Palette existente. 

- Duplicar una paleta. 

- Modificar un elemento encontrado en la lista de los últimos elementos utilizados.




Para gestionar las paletas: 

- seleccione "Crear una nueva paleta" en la pestaña "Skins" del proyecto description (para abrir el proyecto description, vaya a la pestaña "Proyecto", grupo "Proyecto", y haga clic en "Descripción"). 

- en la pestaña "Página", en el grupo "Tema y estilos", despliegue "Paleta" y haga clic en el enlace "Crear una nueva paleta".




El editor de Palette aparece. Este editor se utiliza para: 

- [Crear una nueva paleta](#NOTE4_2). 

- [Modificar un Palette existente.](#NOTE4_3)

- [Duplicando un Palette.](#NOTE4_4)



<a name="NOTE4_2"></a>


### Generando/Creando una Palette
<a name="generandocreando_una_palette_ELTPARAGRAPHE000155"></a>

En el editor de Palette: 

1. Haga clic en la opción "Generar una paleta". 

2. Especificar el nombre de la Palette. 

3. Selecciona la base Color de la paleta a través del círculo cromático y el gráfico Color. 

4. Validar. 

5. Las características de los diferentes colores que se encuentran en la Palette se muestran. Es posible: 

	- modificar cada paleta Color individualmente. Se muestra una vista previa en la parte superior del editor. 

	- guardar las modificaciones realizadas. 

	- recargar las últimas modificaciones guardadas (se pierden las modificaciones realizadas desde la última copia de seguridad). 

	- seleccione una Color con el cuentagotas.




6. Validar la ventana. La nueva paleta se añade automáticamente a la lista de paletas disponibles en el proyecto description.



<a name="NOTE4_3"></a>


### Modificación de un Palette
<a name="modificacion_palette_ELTPARAGRAPHE000176"></a>

En el editor de Palette: 

1. Haga clic en la opción "Modificar una paleta". 

2. Seleccione el Palette para modificar. Si este Palette es un Palette suministrado por WEBDEV, este Palette se duplica automáticamente: especificar su nuevo nombre. 

3. Validar. 

4. Las características de los diferentes colores que se encuentran en la Palette se muestran. Es posible: 

	- modificar cada paleta Color individualmente. Se muestra una vista previa en la parte superior del editor. 

	- guardar las modificaciones realizadas. 

	- recargar las últimas modificaciones guardadas (las modificaciones realizadas desde la última copia de seguridad se pierden).

	- seleccione una Color con el cuentagotas.




5. Validar la ventana. La Palette se modifica automáticamente.



<a name="NOTE4_4"></a>


### Duplicar una paleta
<a name="duplicar_una_paleta_ELTPARAGRAPHE000196"></a>

En el editor de Palette: 

1. Haga clic en "Duplicar a Palette".. 

2. Selecciona el Palette para duplicar. 

3. Validar. 

4. Especificar el nombre del nuevo Palette (creado a partir del Palette seleccionado). 

5. Las características de los diferentes colores que se encuentran en la Palette se muestran. Es posible: 

	- modificar cada paleta Color individualmente. Se muestra una vista previa en la parte superior del editor. 

	- guardar las modificaciones realizadas. 

	- recargar las últimas modificaciones guardadas (se pierden las modificaciones realizadas desde la última copia de seguridad). 

	- seleccione una Color con el cuentagotas.




6. Validar la ventana. La nueva paleta se añade automáticamente a la lista de paletas disponibles en el proyecto description.

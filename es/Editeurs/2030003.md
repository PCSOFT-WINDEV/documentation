<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000515"></a>
Al crear un proyecto, se crea automáticamente un directorio específico para este proyecto.. Este directorio corresponde al directorio principal del proyecto. Los diferentes elementos del proyecto se almacenan en los subdirectorios de este directorio principal. Cada uno de estos subdirectorios tiene un papel específico.

Por ejemplo, los directorios del proyecto "Contabilidad" son los siguientes. Este proyecto contiene todos los subdirectorios posibles. Dependiendo de la configuración de su proyecto, algunos directorios pueden no existir.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=RepertoireProjetWD.gif)
**Observación**: Una copia de seguridad del proyecto se puede realizar en cualquier momento: en la pestaña "Proyecto", en el grupo "Otras acciones", despliegue "Copia seg./Restaurar" y seleccione "Copia de seguridad del proyecto". Para obtener más información, consulte [Guardar un proyecto](../Editeurs/2030014.md).

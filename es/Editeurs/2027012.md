## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
Aparece el panel "Análisis":

- los archivos de datos y los elementos del análisis actual. Las claves están en negrita y las claves únicas están en negrita y subrayadas. 

- los procedimientos almacenados, las consultas almacenadas y los disparadores en el análisis actual.

- las vistas SQL y las vistas materializadas en el análisis actual.

- las conexiones descritas en el análisis actual.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletAnalyse.gif&type=thumb)
El menú contextual (botón derecho del ratón) del panel "Análisis" sirve para:

- Abrir análisis.

- Crear: 

	- un archivo de datos.

	- una vista SQL o materializada.

	- el Carpeta Personal. Para obtener más información, consulte [Carpetas personalizadas en el análisis](../Editeurs/2011003.md).

	- una set de procedimientos almacenados. 

	- una consulta almacenada.

	- un detonante.

	- una enumeración.

	- una combinación.




- Mostrar el análisis description. 

- Mostrar la description de los archivos.

- Vea la description de los archivos.

- Gestionar conexiones. 

- Generar el análisis.

- Imprimir la documentación del análisis.




**Atención**: La información que se muestra corresponde al análisis que se describe actualmente en el documento editor de análisis. Esta información puede diferir de la mostrada en el ["Panel "Explorador de proyectos](../Editeurs/2027029.md) (que muestra la información del último análisis generado).

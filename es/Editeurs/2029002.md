
## Medio ambiente de la editor de análisis 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
El entorno de la editor de análisis es el siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EnvEditeurAnalyses.gif&type=thumb)


Los principales elementos de la WINDEV editor de análisis son los siguientes:


|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif) | **Menú del editor**<br>Acceso rápido a las principales funciones de editor de análisis. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle2.gif)<br> | **Gráfico de análisis**<br>Las barras de desplazamiento le permiten desplazarse por los elementos del gráfico (análisis con varios archivos de datos). También tiene la posibilidad de ampliar el gráfico para ver mejor los elementos del análisis (Ctrl + rueda del ratón). |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle3.gif)<br> | **[Panel Análisis](../Editeurs/2027012.md)**<br>Lista los diferentes ficheros de datos presentes en el análisis actual. Basta con desplegar el archivo de datos deseado para ver los nombres y tipos de sus elementos. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle4.gif)<br> | **Archivo de datos description ventana**<br>Permite:<br><br>- ver y/o modificar las características del archivo de datos.<br><br>- abrir la ventana description de los elementos de el archivo de datos.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle5.gif)<br> | **[Documentos abiertos](../Editeurs/2029012.md)**<br>Muestra un botón para cada Document abierta en el editor. Para mostrar el elemento solicitado en primer plano, pulse el "botón" correspondiente. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle6.gif)<br> | **ventana del navegador**<br>Permite ver todos los elementos que se encuentran en la carpeta abierta. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle7.gif)<br> | **Descripción de los campos**<br>Abre la description de los elementos del fichero de datos actual. |

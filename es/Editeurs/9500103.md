<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000100"></a>
Dependiendo del estilo de sus aplicaciones, la Border de sus ventanas puede ser translúcida: tendrás la posibilidad de ver en transparencia las ventanas que se encuentran abajo o el fondo Image de tu escritorio de Windows.

En la el menú contextual de ventanas, "Disable the translucent effect for the window borders" permite al usuario deshabilitar el efecto translúcido en las ventanas. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MenuCtxFenetre.gif)

## Activando o desactivando el efecto Transparent de las ventanas
<a name="activando_desactivando_efecto_transparent_las_ventanas_ELTTEXTE000124"></a>


### Habilitando el efecto Transparent en sus ventanas
<a name="habilitando_efecto_transparent_sus_ventanas_ELTPARAGRAPHE000021"></a>

Para activar el efecto Transparent en sus ventanas:

1. Mostrar la pestaña "Imagen" de la ventana description.

2. Especifique el fondo Image utilizado por la ventana.

3. Clipar la ventana ("Clip the window").  

4. Seleccione las opciones de transparencia de Vista y Blur. Los márgenes que se tendrán en cuenta para el efecto "Blur" se pueden definir a través del botón "Márgenes de la zona Blur".

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=flou_vista.gif)
**Observación**: En Vista, la opción "System transparency in Vista (Aero)" permite al sistema gestionar la transparencia si el modo de visualización utilizado es el modo "Aero. Recomendamos que use esta opción si está usando las plantillas de ActivVista y ActivVista Lite tema. Esta opción no se recomienda si está usando otras plantillas de tema.
<a name="NOTE2_2"></a>


### Programación
<a name="programacion_ELTPARAGRAPHE000036"></a>

Para activar (o no) el efecto translúcido en los bordes de las ventanas, usa [Estilo de dibujo](../WDLang1/3014030.md).

Para desactivar el efecto translúcido en los bordes de las ventanas, usa [WinDisableEffect](../WDLang1/1000017172.md).
<a name="NOTE2_3"></a>


### Ejemplos
<a name="ejemplos_ELTPARAGRAPHE000051"></a>

Las plantillas ActivVista y ActivVista Lite tema utilizan el efecto translúcido de las ventanas.




## Probar una Page estática 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000142"></a>
La prueba de una sola Page permite ejecutar la Page actual en WEBDEV.

Esto le permite realizar la prueba de su proyecto a partir de una Page determinada o comprobar el modo de funcionamiento de una Page en cuanto esté totalmente desarrollada.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000166"></a>


### Ejecución de la prueba de estática actual Page
<a name="ejecucion_prueba_estatica_actual_page_ELTPARAGRAPHE000020"></a>

Para probar la Page estática actual desde el editor, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Go_Page_bl.gif) en los botones de acceso rápido (o pulse F9). El editor se minimiza automáticamente y la página se muestra en el navegador Default.

**Consejo**: Para seleccionar el navegador utilizado al ejecutar la prueba Page, haga clic en la prueba Icon del Page. La flecha de la derecha despliega un menú que permite elegir el navegador que se utilizará para la prueba.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=GO_PAGE_WB_deroule.gif)


También puede seleccionar el navegador Default en la página web de [Opciones de WEBDEV](../Editeurs/2036001.md).

**Cuando se inicia una prueba estática Page** , se ejecutan los siguientes eventos:

- "Cargar" la Page actual (código del navegador). 

- "Entrada en el control " de la primera control de la página actual (si esta control ) ( Event ) (BRIENTO) ( control ) (BRIENTO).




Durante la prueba, se pueden ejecutar todas las funciones de Page. Tendrá la posibilidad de abrir otras páginas, por ejemplo.
Observación: La última prueba realizada (ventana, consulta, reporte, ...) puede volver a ejecutarse desde cualquier elemento del proyecto a través de "IR . Reejecutar la última prueba" que se encuentra en los botones de acceso rápido. 
<a name="NOTE2_2"></a>


### Detención de la prueba Page
<a name="detencion_prueba_page_ELTPARAGRAPHE000047"></a>

Existen varios métodos para detener la prueba:

- 1er método: Cerrar el sitio cuya prueba se está ejecutando actualmente. WEBDEV muestra el editor abierto cuando se inició la prueba.

- 2º método: Vuelve al editor con la barra de tareas (o presiona Alt + Tab) y confirma el final de la prueba. WEBDEV muestra el editor abierto cuando se inició la prueba.

## Notas
<a name="notas_ELTTEXTE000196"></a>


### Navegador de prueba
<a name="navegador_prueba_ELTPARAGRAPHE000059"></a>

Para seleccionar el navegador de prueba, en la pestaña "Proyecto", en el grupo "Modo de prueba", despliegue "Modo de prueba" y seleccione "Navegador de prueba".

El navegador Default está definido en las opciones de WEBDEV: 

1. En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WEBDEV". 

2. En la ventana que aparece, seleccione la pestaña "Web".. 


Para obtener más información, consulte [Opciones de WEBDEV](../Editeurs/2036001.md).

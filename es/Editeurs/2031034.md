Los diferentes pasos para crear un patrón WEBDEV RAD son los siguientes:

1. [Creación del proyecto "RAD Pattern".](../Editeurs/2031021.md)

2. Creación de las diferentes páginas del patrón.

	- Formar páginas.

	- Página de la tabla.

	- Página de relaciones.

	- Botones y página de Visión Plus.

	- **Menús y página de inicio.**

4. [Usando el patrón RAD.](../Editeurs/2031024.md)

## ¿Qué páginas deben crearse?
<a name="¿que_paginas_deben_crearse_ELTTEXTE000179"></a>


### Los ficheros a crear
<a name="los_ficheros_crear_ELTPARAGRAPHE000032"></a>

Recordatorio: Para construir un patrón RAD utilizado para crear un proyecto completo, se requieren los siguientes archivos:

- Un formulario para RADFileA.

- Un formulario para RADFileB. Un botón Vision Plus (o un cuadro combinado emergente) debe permitirle seleccionar un elemento de RADFileA.

- Un formulario para RADFileC. Un botón Vision Plus (o un cuadro combinado emergente) debe permitirle seleccionar un elemento de RADFileB.

- Una tabla para RADFileA.

- Una tabla para RADFileB.

- Un tabla para RADFileC.

- Una página de relación RADFileB/RADFileD.




El menú principal del sitio puede ser incluido en el modelo associated con las páginas.

## Menús
<a name="menus_ELTTEXTE000209"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000060"></a>

En un sitio WEBDEV, el menú se muestra en todas las páginas del sitio. Por lo tanto, le aconsejamos que cree este menú en la página modelo, ya que todas las páginas del sitio están asociadas a modelo.

El menú de su sitio puede ser horizontal o vertical.

Este menú se utilizará para abrir:

- las páginas del formulario y las páginas de tabla definidas en el modelo

- las otras páginas encontradas en el patrón.




Los siguientes párrafos explican cómo crear un menú desplegable.

**Importante: no olvide crear una primera página de proyecto en su patrón RAD.**Esta primera página puede ser utilizada para mostrar el menú del sitio y puede ser la página utilizada para gestionar la devolución desde las páginas Formulario o Tabla.. En nuestro ejemplo, esta primera página se llama "PAGE_Home". Esta primera página es una página dinámica (como todas las páginas del patrón).
<a name="NOTE3_2"></a>


### Creación de un menú desplegable
<a name="creacion_menu_desplegable_ELTPARAGRAPHE000078"></a>

En la mayoría de los casos, el menú desplegable está incluido en el modelo. 

Para crear un menú desplegable: 

1. Cree (si es necesario) la página o el modelo que muestra el menú.

2. Crear el menú: en la pestaña "Creación", en el grupo "Navegación", haga clic en "Menú".

3. Las opciones de menú utilizadas para las opciones específicas del RAD deben tener un nombre específico: "MENUFIC_XXX" (MENUFIC_RADFileA por ejemplo). Las opciones de menú con este nombre se duplicarán para cada archivo del análisis.




El código associated con las opciones también se duplicará y se adaptará (si es necesario) al archivo utilizado.

Las otras opciones de menú (utilizadas para iniciar una característica incluida en el patrón, por ejemplo) pueden tener cualquier nombre. Se generarán automáticamente.

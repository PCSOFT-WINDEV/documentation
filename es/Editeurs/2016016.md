


## Importar una hoja de estilo CSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
WEBDEV le permite importar una hoja de estilo CSS a un proyecto WEBDEV.

Esto te permite reutilizar los estilos creados con otra herramienta e incluirlos en tu sitio WEBDEV.

**Observación**: Al importar un HTML existente Page, WEBDEV propone importar también los estilos CSS associated con este Page. Para obtener más información, consulte [Importar un HTML Page existente](../Editeurs/2030029.md). 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Importar una hoja de estilo existente
<a name="importar_una_hoja_estilo_existente_ELTTEXTE000119"></a>
Para importar una hoja de estilo CSS existente:

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Una hoja de estilo CSS".

2. El asistente de importación se abre automáticamente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Site%20Internet%20-%20Import%20CSS%20-%20HC%20N%B0001%201.jpg)


3. Seleccione el modo de gestión de estilo. Pase a la etapa siguiente. 

4. Seleccione el archivo CSS que contiene los estilos que desea importar (botón "..."). 
	Se muestra la lista de estilos CSS encontrados en el archivo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P4_Site%20Internet%20-%20Import%20CSS%20-%20HC%20N%B0002%201.jpg)


5. Seleccione los estilos que desea importar en el proyecto (los estilos marcados serán importados).

6. Validar. Los estilos se incluyen automáticamente en la hoja de estilo associated con el proyecto.

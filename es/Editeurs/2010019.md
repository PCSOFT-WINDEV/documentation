## Modo de búsqueda
<a name="modo_busqueda_ELTTEXTE000096"></a>
El modo de búsqueda completa le permite hacer que la pestaña "Buscar" esté disponible en sus archivos de ayuda.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_Rechercher.gif)


Esta pestaña permite al usuario construir una base de datos de palabras a partir del contenido de las páginas de ayuda.. Esta base de datos de palabras se puede utilizar para buscar todas las páginas de ayuda utilizando una palabra específica. Esta base de datos de palabras (pequeñas o grandes) se almacena en un archivo con la extensión FTS.

WINDEV le da la posibilidad de configurar la base de datos de palabras que se crearán.. Si la opción "Utilizar el modo de búsqueda completa" está marcada, puedes:

- **Incluya las páginas Invisible que se encuentran en el tabla de contenidos**: si esta opción está seleccionada, la búsqueda se realizará en todas las páginas del sistema de ayuda; de lo contrario, se realizará en las páginas que se encuentran en el resumen.

- **Permitir la búsqueda por frase**: si esta opción está seleccionada, el usuario podrá introducir una frase para realizar su búsqueda. Sólo se propondrán las páginas que contengan esta frase.

- **Mostrar las frases encontradas**: si esta opción está seleccionada, cuando se busca una frase, esta frase se resaltará en las páginas que coincidan con la búsqueda.

- **Permitir la búsqueda por similitud**: si esta opción está seleccionada, todas las palabras con la misma raíz serán seleccionadas por la búsqueda.




**Observación**: Las opciones seleccionadas serán las opciones utilizadas por Default para crear la base de datos de palabras. Estas opciones pueden ser modificadas por el usuario en cualquier momento.

## Comprimir el archivo de ayuda
<a name="comprimir_archivo_ayuda_ELTTEXTE000120"></a>
Esta opción se utiliza para comprimir (o no) el archivo de ayuda. La compresión del archivo de ayuda se utiliza para optimizar la cantidad de espacio ocupado por el archivo HLP.. Se recomienda esta compresión.

## Información del cuadro "Acerca de
<a name="informacion_del_cuadro_acerca_ELTTEXTE000168"></a>

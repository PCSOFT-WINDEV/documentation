## Presentación
<a name="presentacion_ELTTEXTE000155"></a>
WEBDEV le permite "absorber" las páginas HTML existentes. La página se importa en el proyecto WEBDEV y cada elemento de la página se convierte automáticamente en un elemento WEBDEV (control Estático, HTML control, Enlace, etc.).

El principio de esta característica se basa principalmente en la recuperación del estilo de una página HTML.

Observación: Para importar una página de DremWeaver o elementos de DremWeaver, consulte [Importar elementos de DreamWeaver a un sitio WEBDEV](../Editeurs/2030059.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000179"></a>


### Importar una página HTML a un sitio WEBDEV
<a name="importar_una_pagina_html_sitio_webdev_ELTPARAGRAPHE000031"></a>

Para importar una página HTML a un sitio WEBDEV:

1. Abra el proyecto WEBDEV en el que debe importarse la página.

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Una página HTML".

3. Especifique el ruta o la URL de la página HTML a importar. Se muestra la vista previa de la página HTML a importar.
	**Observación**: También tiene la posibilidad de iniciar la importación de la página mediante un arrastrar y soltar del archivo HTM en el editor de páginas de WEBDEV.

4. Seleccione el modo de importación. Los estilos y las imágenes associated con la página importada pueden ser manejados según dos métodos:

	- Si se selecciona "Importar una página independiente al proyecto":

		- se crea una hoja de estilo WEBDEV (archivo ".WWS") y automáticamente associated con la página importada. Esta hoja de estilo contiene todos los estilos CSS de la página inicial.

		- las diferentes imágenes de associated con la página importada serán copiadas a un directorio específico (subdirectorio "&lt;PageName&gt;" del directorio del proyecto Default).




- Si se selecciona "Importar varias páginas del mismo sitio al proyecto":

	- los estilos de la página importada se añaden automáticamente a la hoja de estilo associated con el proyecto.

	- las diferentes imágenes associated con la página importada se copiarán en el directorio "&lt;Nombre del proyecto&gt;_WEB" del proyecto.

5. Validar. La página HTML se incluye automáticamente en el proyecto WEBDEV actual.

	- Si se identifican estilos idénticos en la página HTML a importar, sólo se crea un tipo en la página importada.. 

	- Si se detecta un estilo usado en una página HTML ya importada, este mismo estilo será reutilizado.. 





**Observación**: Cuando se importa una página HTML, recomendamos que no se realice ninguna operación de "copy-pegar".

### Elementos importados
<a name="elementos_importados_ELTPARAGRAPHE000078"></a>

El tabla a continuación presenta las principales transformaciones realizadas entre la página HTML inicial y la página WEBDEV creada:

| Fuente de la página HTML | Página WEBDEV creada |
| --- | --- |
| vincular | control Enlace associated con la acción del enlace de la fuente |
| Aplicativo Java | El Applet Java controla a associated con el Applet Java de origen |
| &lt;OBJECT&gt; y &lt;BLINK&gt; tags | HTML control con el contenido de las etiquetas de origen |
| Image | control Imagen associated con la fuente Image |
| Estática | control Estático o control Estático HTML con el contenido de la estática inicial. |


**Observación**: Las áreas del mapa no se importan.

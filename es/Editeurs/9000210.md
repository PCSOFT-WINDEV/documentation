<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000328"></a>
Markdown es un lenguaje de marcado ligero que permite aplicar formato al texto. Por su sencillez y practicidad, Markdown es fácil de leer, aprender y recordar.

Todos los controles de WINDEV y WINDEV Mobile soportan Markdown. Solo tiene que activar la opción correspondiente e introducir el texto en formato Markdown.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000352"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Habilitar Markdown en un control 
<a name="habilitar_markdown_control_ELTPARAGRAPHE000020"></a>

Abra la ventana de descripción del control deseado, vaya a la pestaña "UI" y active la opción "Admitir Markdown".

Esta opción solo se aplica a:

- títulos, 

- tooltips, 

- tokens asociados a un control Campo de entrada.
	Observación: Esta propiedad no se aplica al contenido del control Campo de entrada.




Observación: Markdown también es compatible con los controles Estático del editor de reportes. 


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Control Campo de entrada de tipo Markdown
<a name="control_campo_entrada_tipo_markdown_ELTPARAGRAPHE000038"></a>

En la versión 28 Update 2, se introdujo un nuevo tipo para el control Campo de entrada: "Markdown (solo lectura)".

Los caracteres de formato Markdown del texto asignado al control serán interpretados tanto en el editor como en tiempo de ejecución.

Este modo ofrece varias ventajas, en comparación con un control Estátco:

- si el contenido con formato es demasiado grande, aparecerá automáticamente una barra de desplazamiento que permitirá al usuario desplazarse por el contenido.

- puede seleccionar una parte del texto que se muestra (para copiarlo, por ejemplo).





### Convertir Markdown
<a name="convertir_markdown_ELTPARAGRAPHE000050"></a>WLanguage incluye varias funciones para convertir contenido Markdown a otros formatos:



|   |   |
| --- | --- |
| [MarkdownToDOCX](../WDLang1/1410089153.md) | Convierte una cadena Markdown en un archivo DOCX. |
| [MarkdownToHTML](../WDLang1/1410088874.md) | Convierte una cadena Markwdown en un buffer HTML (codificado en UTF-8). |
| [MarkdownToPDF](../WDLang1/1410089119.md) | Convierte una cadena (o buffer) Markdown en un buffer que contiene un documento PDF. |
| [MarkdownToText](../WDLang1/1410089001.md) | Convierte una cadena Markdown en texto sin formato (sin etiquetas). |





Al editar documentos que contienen texto Markdown (actas de reuniones, informes de intervención, etc.), estas funciones le permiten:

- mostrar el contenido en un sitio web (HTML),

- enviar el documento por correo electrónico (PDF),

- ...

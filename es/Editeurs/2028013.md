


## WDInst: Configuración del ordenador de configuración
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000345"></a>
Al crear el programa de instalación, WDInst le permite configurar el equipo donde el programa será instalado. Es posible:

- Proponga ejecutar un programa al final de la configuración (con o sin línea de comandos).

- Modificar el registro:

	- modificar el identificador utilizado para desinstalar la aplicación,

	- modificar o crear entradas.




- Modificar o crear el archivo "AutoExec.BAT".

- Modificar o crear los archivos ".INI".




Por defecto, el programa de instalación propone iniciar el programa al final de la instalación y no se realizará ninguna modificación en los ordenadores de los usuarios.

El ordenador de configuración está configurado en el editor de configuración WDInst. Para abrir WDInst, en la pestaña "", en el grupo "", haga clic en "".

La asistente para crear el programa de instalación también puede ser usada para configurar el ordenador de instalación.

**Observación**: Solicitar al usuario que confirme las modificaciones realizadas en el sistema durante la instalación de la aplicación:

1. Inicie WDInst si es necesario: en la pestaña "", en el grupo "", haga clic en "".

2. Seleccione "Parámetros de configuración... Configuración del sistema".

3. Seleccione la opción "Solicitar confirmación antes de modificar".

## Proponer la ejecución de un programa
<a name="proponer_ejecucion_programa_ELTTEXTE000369"></a>
Por defecto, el programa de instalación propone ejecutar el ejecutable de la aplicación al final de la instalación. La opción "Iniciar el programa" está marcada.. El ejecutable se ejecuta automáticamente al hacer clic en el botón "Listo".

Se utiliza el editor de configuración WDInst:

- Especifique el ejecutable para ejecutar y asociar parámetros con este ejecutable. Se puede ejecutar un único ejecutable (archivo".EXE") que se encuentra en la lista de archivos a instalar.

- No propongo ejecutar el ejecutable de la aplicación al final de la configuración.




**Observación**: Si se realizan modificaciones en el archivo "AutoExec.Bat", no se debe ejecutar ningún programa al final de la configuración. De hecho, las modificaciones realizadas en este archivo serán tenidas en cuenta cuando se reinicie el equipo.
<a name="NOTE2_2"></a>


### Especificar el ejecutable a ejecutar y asociar parámetros con este ejecutable
<a name="especificar_ejecutable_ejecutar_asociar_parametros_con_este_ejecutable_ELTPARAGRAPHE000078"></a>

Para especificar el ejecutable que se ejecutará y asociar parámetros a este ejecutable: 

1. Seleccione "Parámetros de configuración ... Fin de la configuración".

2. Introducir el ejecutable que debe ejecutarse al final del setup de aplicaciones (opción"Línea de comandos").
	El ejecutable a ejecutar corresponde a:

	- El nombre y la ruta completa del archivo ejecutable. Se pueden utilizar las siguientes palabras clave:

		- **- &lt;SetupDir&gt;&gt;SetupDir**: directorio de instalación elegido por el usuario durante la instalación.

		- **- Configuración de la unidad&gt; &lt;SetupDrive**: del directorio de instalación elegido por el usuario durante la instalación.

		- **- &lt;Archivo de configuración&gt;&gt;.**: directorio de instalación de archivos de datos. WDInst se usa para configurar este directorio. 
						Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

		- **- &lt;WinDir&gt;&gt;Dirección del viento**: Directorio Windows de ordenadores de usuario




- Los parámetros de tiempo de ejecución del programa instalado (si es necesario).
			Por ejemplo: "&lt;SetupDir&gt;\\MyApp.EXE /Param".



<a name="NOTE2_3"></a>


### No propongo ejecutar el ejecutable de la aplicación al final de la configuración
<a name="propongo_ejecutar_ejecutable_aplicacion_final_configuracion_ELTPARAGRAPHE000105"></a>

Para evitar proponer que se ejecute el ejecutable de la aplicación al final de la configuración: 

1. Seleccione "Parámetros de configuración ... Fin de la configuración".

2. Borrar el contenido de la opción "Línea de comandos".

## Modificar el identificador utilizado para desinstalar la aplicación
<a name="modificar_identificador_utilizado_para_desinstalar_aplicacion_ELTTEXTE000405"></a>
Si ha elegido permitir que su aplicación se desinstale, se creará automáticamente un identificador específico en el registro.. Este identificador puede ser modificado a través del editor de configuración WDInst.

Por defecto, este identificador es "&lt;ApplicationNameWithoutSpace&gt;".. Por ejemplo, "WDCustomer" para la aplicación "WD Customer".. Este identificador puede (o no) contener espacios en blanco.

Este identificador se creará en la clave:

- "HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Uninstall" en Windows 2000 y NT

- "HKEY_LOCAL_MACHINE\\Software\\Microsoft\\Windows\\CurrentVersion\\Uninstall" en Windows 98 y Me.



<a name="NOTE3_2"></a>


### Modificar el identificador utilizado para desinstalar la aplicación
<a name="modificar_identificador_utilizado_para_desinstalar_aplicacion_ELTPARAGRAPHE000129"></a>

Para modificar el identificador utilizado para desinstalar la aplicación: 

1. Seleccione "Parámetros de configuración ... Fin de la configuración".

2. Seleccione el nuevo identificador de la aplicación.




**Observación**: Este identificador puede (o no) contener espacios en blanco.

## Modificación o creación del archivo "AutoExec.BAT
<a name="modificacion_creacion_del_archivo_autoexecbat_ELTTEXTE000435"></a>
Tiene la posibilidad de modificar o crear el archivo "AutoExec.BAT" en los ordenadores de configuración.. Este archivo se modificará o creará (si es necesario) cuando se reinicie el equipo. En este caso, se propone automáticamente un reinicio al final del setup de aplicaciones.

El archivo "AutoExec.BAT" se utiliza para:

- añadir el directorio del ejecutable de la aplicación en el PATH,

- inicie la aplicación cuando se inicie el ordenador de instalación,

- etc.




**Recordatorio**:

- El archivo "AutoExec.BAT" se ejecuta automáticamente cuando se inicia el equipo.

- El PATH lista los directorios en los que se buscan prioritariamente los ejecutables (archivos".EXE") y las bibliotecas (archivos"DLL").



<a name="NOTE4_2"></a>


### Modificación o creación del archivo "AutoExec.BAT
<a name="modificacion_creacion_del_archivo_autoexecbat_ELTPARAGRAPHE000159"></a>

Para modificar o crear el archivo "AutoExec.BAT: 

1. Seleccione "Parámetros de configuración... Configuración del sistema".

2. Introduzca la línea de comandos para añadir o modificar en el archivo "AutoExec.BAT".

3. Especifique la ubicación de la "línea de comandos" correspondiente en el archivo "AutoExec.BAT" (cuadro combinado"Ubicación"):

	- ***Al principio***. Corresponde a la primera "línea de comandos" del archivo "AutoExec.BAT".

	- ***Al final***. Corresponde a la última "línea de comandos" del archivo "AutoExec.BAT".

	- ***Después del PATH***. Al posicionar una "línea de comandos" después del PATH, el ejecutable y/o las librerías utilizadas en esta línea de comandos se buscarán en los directorios del PATH.




4. Si la misma línea de comandos ya existe en el archivo "AutoExec.BAT", puede hacerlo:

	- Reemplazar la línea de comandos existente. Seleccione el parámetro "Reemplazar" (cuadro combinado"Tipo").

	- Agregar la línea de comandos. Seleccionar el parámetro "Añadir" (combo box "Tipo").







**Observación**: Para añadir directamente el directorio de instalación de la aplicación a la ruta, utilice los botones:

- "Al inicio". El directorio de instalación de la aplicación será el directorio **primeramente** del PATH.

- "Al final". El directorio de instalación de la aplicación será el directorio **por última vez** directorio de PATH.

## Modificar o crear los archivos ".INI
<a name="modificar_crear_los_archivos_ini_ELTTEXTE000465"></a>
Al instalar una aplicación WINDEV, tiene la posibilidad de modificar o crear los archivos ".INI" en los ordenadores de instalación.. Estos archivos se asociarán con la aplicación instalada.

Los archivos ".INI" están acostumbrados a:

- Personalizar la aplicación según el usuario conectado

- Definir los parámetros por defecto de la aplicación




Entonces, los archivos ".INI" pueden ser leídos y modificados desde la aplicación directamente ([INIRead](../WDLang1/3013043.md) y [INIWrite](../WDLang1/3013039.md)).
<a name="NOTE5_2"></a>
Por ejemplo, para guardar el lenguaje de ejecución del programa de instalación en el archivo "MyApp.INI" (que se encuentra en el directorio de instalación de la aplicación):

- "Archivo ".INI: &lt;Dirección de instalación&gt;\\MyApp.INI

- Sección: IDIOMA

- Configura: INSTALAR_IDIOMA

- Valor: &lt;Idioma&gt;




La palabra clave **&lt;Language&gt;** se reemplazará automáticamente durante la instalación por el número del idioma de ejecución del programa de instalación (este número puede utilizarse con la función [Nación](../WDLang1/3054014.md)).

En este ejemplo, si el número de idioma de ejecución es 3, el siguiente parámetro se añadirá al archivo "MyApp.INI":


```txt
[LANGUAGE]
LANGUAGE_INSTALL=3
```

<a name="NOTE5_3"></a>


### Modificar o crear archivos ".INI" durante la instalación
<a name="modificar_crear_archivos_ini_durante_instalacion_ELTPARAGRAPHE000222"></a>

Para modificar o crear archivos ".INI" durante la instalación: 

1. Seleccione "Parámetros de configuración... Configuración del sistema".

2. Especifique el archivo ".INI" correspondiente. Este archivo se creará si es necesario. Puede indicar:

	- El nombre y la ruta completa del fichero.
			Se pueden utilizar las siguientes palabras clave:

		- **&lt;SetupDir&gt;&gt;SetupDir**: directorio de instalación elegido por el usuario durante la instalación.

		- **Configuración de la unidad&gt; &lt;SetupDrive**: del directorio de instalación elegido por el usuario durante la instalación.

		- **&lt;Archivo de configuración&gt;&gt;.**: directorio de instalación de archivos de datos. WDInst se usa para configurar este directorio. Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

		- **&lt;WinDir&gt;&gt;Dirección del viento**: directorio de Windows en los ordenadores de los usuarios.
						Por ejemplo: "&lt;SetupDir&gt;\\MyDirectory\\MyFile.INI".




- El nombre del archivo. El archivo se buscará o creará en el directorio de Windows. Por ejemplo: "MyFile.INI".

- Ningún archivo. Se modificará el archivo "WIN.INI".

3. Especifique la sección que desea modificar o crear.

4. Especifique el parámetro que desea modificar o crear.

5. Especifique el valor del parámetro.
	Para recuperar el número del idioma de tiempo de ejecución para el programa de instalación, utilice la función **&lt;Idioma&gt;&gt;Lenguaje** palabra clave.

## Modificación o creación de entradas en el registro
<a name="modificacion_creacion_entradas_registro_ELTTEXTE000501"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000255"></a>

Al instalar una aplicación WINDEV, tiene la posibilidad de modificar o crear entradas en el registro.

El registro se lo permite:

- Personalizar la aplicación según el usuario conectado

- Definir los parámetros por defecto de la aplicación




Entonces el registro puede ser leído y modificado desde la aplicación directamente ([RegistrySetValue](../WDLang1/3065006.md), [RegistryQueryValue](../WDLang1/3065012.md) , ...).
<a name="NOTE6_2"></a>
Por ejemplo, para guardar el idioma de ejecución del programa de instalación en el registro, escriba:

- Clave: HKEY_LOCAL_MACHINE\\Software\\MyCompany\\MyApp\\1.0\\

- denominar: Idioma

- Valor: &lt;Idioma&gt;




La palabra clave **&lt;Language&gt;** se reemplazará automáticamente durante la instalación por el número del idioma de ejecución del programa de instalación (este número puede utilizarse con la función [Nación](../WDLang1/3054014.md)).
<a name="NOTE6_3"></a>


### Modificar o crear entradas en el registro durante la instalación
<a name="modificar_crear_entradas_registro_durante_instalacion_ELTPARAGRAPHE000289"></a>

Para modificar o crear entradas en el registro durante la instalación: 

1. Seleccione "Parámetros de configuración... Configuración del sistema".

2. Especifique la clave relevante (ruta completa). Esta clave se creará si es necesario.

3. Especifique el nombre del valor a modificar. Este valor se creará si es necesario.

4. Especifique el nuevo valor. 
	Para recuperar el número del idioma de tiempo de ejecución para el programa de instalación, utilice la función **&lt;Idioma&gt;&gt;Lenguaje** palabra clave.

## Opciones del menú contextual en las ventanas
<a name="opciones_del_menu_contextual_las_ventanas_ELTTEXTE000223"></a>


### Personalizar interfaz
<a name="personalizar_interfaz_ELTPARAGRAPHE000026"></a>

Desea modificar la interfaz de sus ventanas, cambiar la posición de los controles, añadir consejos de herramientas, ... Todo lo que tiene que hacer es activar el modo "Modificación. Este modo te permite mover controles, cambiar su tamaño, cambiar la color de los controles, añadir un tooltip, ...

Ejemplo: Falta un dato de herramienta en el formulario del cliente en el número de Phone. Para añadir este tooltip: 

- Pasar al modo de modificación ("Personalizar la interfaz .. Pasar al modo de modificación" en el menú contextual de la ventana).

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Bulle%20d_aide_2%20-%20HC%20N%B0001.gif)


- Seleccione la control y haga clic en "Editar información sobre la herramienta" en el menú contextual de la control.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Bulle%20d_aide_2%20-%20HC%20N%B0002.gif)


- El texto de la información sobre la herramienta se muestra en la modificación. Tienes la capacidad de escribir el texto solicitado.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Bulle%20d_aide%20-%20HC%20N%B0002.gif)


- Cuando se escribe el texto, haga clic en la ventana Background. 

- Cierre el modo de modificación ("Fin del modo de edición" en el menú contextual de la ventana).





### Notas reposicionables
<a name="notas_reposicionables_ELTPARAGRAPHE000052"></a>

¿Rara vez usas una función de tu aplicación y nunca recuerdas cuándo marcar una opción?
Añada una note reposicionable en su escaparate. Esta note se mostrará automáticamente al abrir la ventana y se reducirá al cabo de unos segundos.
Para crear una note reposicionable: 

1. Haga clic con el botón derecho del ratón para abrir el menú contextual de la ventana. 

2. Seleccione "Reposicionable note s .. Crear un note reposicionable ". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable%20-%20HC%20N%B0003.gif)


3. La note reposicionable aparece en la ventana actual. Puede colocarlo en cualquier lugar de la ventana y escribir el texto de la note.
	Observación: Puede pegar texto en la note (Ctrl + V). 




Por Default, el reposicionable note tiene las siguientes características: 

- note texto en rojo. 

- Tamaño medio para el texto note. 

- Fondo color de nota color ed en amarillo. 

- El día de la creación aparece en la note (Hoy). Esta fecha se irá ajustando con el tiempo (por ejemplo: hace 3 días, etc.). 

- Reducción de automatic: Cada vez que se visualiza la ventana, la nota aparece durante 10 segundos y luego se reduce automáticamente (en la parte superior derecha Corner de la ventana).


El menú contextual de la note reposicionable permite configurar esta note.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable%20-%20HC%20N%B0002.gif)
 Las opciones disponibles son las siguientes: 

- **Modificar texto de la nota**: El cursor de entrada aparece al final del texto de note para su modificación.
	Observación: Para modificar el texto de la note, también tiene la posibilidad de pulsar directamente la note. 

- **Minimizar**: Esta opción se utiliza para minimizar la note. La nota se posiciona como una miniatura en la parte superior derecha de la ventana de Corner.
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable1%20-%20HC%20N%B0001.gif&type=thumb)
 Un clic en el cuadrado Icon sirve para restaurar el note a su dimensión inicial. 

- **Minimizar automáticamente**: Si se selecciona esta opción, la note se mostrará al abrir la ventana y se reducirá automáticamente después de 5 segundos (si no se ha modificado). 

- **Color de fondo**: Esta opción permite personalizar el fondo color del note. 

- **Texto color**: Esta opción se utiliza para personalizar el texto Color. 

- **Tamaño del texto**: Esta opción se utiliza para personalizar el tamaño del texto. 

- **Eliminar nota**: Esta opción se utiliza para eliminar la note. En este caso, la note no se mostrará más. El borrado se puede deshacer pulsando Ctrl + Z. 

- **Exportar notas de esta ventana**: Esta opción se utiliza para crear un archivo "wdnote" en el directorio especificado. El nombre del archivo propuesto por Default corresponde al nombre de la ventana. Esta opción se utiliza para guardar las notas associated con la ventana. Las notas exportadas no se borran de la ventana. 

- **Exportar todas las notas de la aplicación**: Esta opción se utiliza para crear un archivo "wdnote" en el directorio especificado. El nombre del archivo propuesto por Default corresponde al nombre de la aplicación. Esta opción se utiliza, por ejemplo, para guardar las notas de la aplicación.. Las notas exportadas no se eliminan de la aplicación. 

- **Importar notas**: Esta opción se utiliza para importar un archivo "wdnote" en la aplicación. Esta ventana puede contener tanto las notas de una ventana como las de la aplicación.. Las notas que se encuentran tanto en la aplicación como en el archivo no se modificarán, las notas que se encuentran sólo en el archivo se añadirán a la aplicación..





### Forma copy
<a name="forma_copy_ELTPARAGRAPHE000066"></a>

Al escribir varias copias de un formulario una tras otra, parte de la información que se debe escribir puede ser la misma para cada instancia del formulario.. Por ejemplo, cuando se teclea un conjunto de artículos, el nombre y la reference de los artículos cambian cada vez pero el nombre del proveedor puede permanecer igual durante varias entradas.

Para agilizar las operaciones, puede pedir a la ventana actual que copy la información introducida en los diferentes controles en el portapapeles para pegarla cada vez que se realice una nueva entrada.Para copy los datos de la ventana actual al portapapeles, utilice:

- La opción "Formulario copy" en el menú contextual de la ventana.

- El atajo Ctrl + Alt + C.




Para pegar los datos del portapapeles en la ventana actual, utilice:

- La opción "Pegar formulario" en el menú contextual de la ventana.

- El atajo Ctrl + Alt + V.





### Oscurecer ventana si está inactiva (OVI)
<a name="oscurecer_ventana_esta_inactiva_ovi_ELTPARAGRAPHE000082"></a>

¿Su aplicación abre varias ventanas al mismo tiempo y no sabe cuál utilizar?

Las ventanas inactivas pueden oscurecerse automáticamente. Así, podrá saber qué ventana utilizar.

Esta funcionalidad se conoce como Oscurecimiento de Ventanas Inactivas (OVI).

Por ejemplo: <br>![Modifica la información relativa a un cliente
y haga clic en el botón "Validar".](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Grisage%20automatique%20des%20fenetres%20-%20HC%20N%B0001.gif)<br>![Una ventana le pide que confirme o cancele los cambios.
La ventana principal (en el fondo) deja de estar disponible.
Se oscurece automáticamente. 
Esto le permite identificar inmediatamente la ventana que se debe utilizar.](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Grisage%20automatique%20des%20fenetres%20-%20HC%20N%B0002.gif)
Se pueden utilizar varias opciones para atenuar la ventana actual. 
Las opciones son las siguientes: 

- **Oscurecer ventana si está inactiva (OVI)**: Esta opción se usa para poner en gris (o no) las ventanas de Inactive. Esta opción es válida sólo para la ventana actual.

- **Aumentar la intensidad del OVI**: Esta opción se utiliza para mostrar una barra de progreso que indica la tasa actual de gris para las ventanas de Inactive. Puedes aumentar esta tasa presionando Ctrl + Alt + '+' en el teclado numérico. Esta tasa de gris será válida para toda la aplicación.

- **Disminuir la intensidad del OVI**: Esta opción se utiliza para mostrar una barra de progreso que indica la tasa actual de gris para las ventanas de Inactive. Puede disminuir esta tasa presionando Ctrl + Alt + "-" en el teclado numérico. Esta tasa de gris será válida para toda la aplicación.




Para dejar de mostrar la barra de progreso, simplemente presione Esc o Enter.


### Almacenar el tamaño y la posición de las ventanas
<a name="almacenar_tamano_posicion_las_ventanas_ELTPARAGRAPHE000091"></a>

Por Default, se muestra una ventana según la posición definida por el desarrollador de la aplicación. Además, el tamaño de la ventana corresponde al tamaño definido por el desarrollador. 

Si esta función está activada, el último tamaño y la última posición de la ventana se guardan automáticamente al cerrar la ventana.. La ventana se mostrará en la misma posición durante su próxima apertura.

Esta función puede activarse o desactivarse a través de "Recordar el tamaño y la posición de la ventana" en el menú contextual de la ventana (que aparece al hacer clic con el botón derecho del ratón en la ventana).

Observación: Una vez que el tamaño y la posición de la ventana han sido modificados, tienes la capacidad de restaurar su tamaño y posición de Default. Basta con seleccionar "Restaurar el tamaño y la posición de Default" en el menú contextual de la ventana (haciendo clic con el botón derecho del ratón).


### Animación de ventanas
<a name="animacion_ventanas_ELTPARAGRAPHE000104"></a>

¿Sus ventanas se ven opacas cuando se abren y se cierran? Alégrenlos: con animaciones!
Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_AnimationFenetre.gif)
 Esta función puede desactivarse fácilmente mediante la opción "Desactivar animaciones de la ventana" del menú contextual de la ventana (haciendo clic con el botón derecho del ratón en la ventana).

**Consejo para el desarrollador**: *para animar las ventanas de su aplicación, configure las diferentes animaciones (botón "Animaciones de ventanas") en la pestaña "Avanzado" del proyecto description*.


### Redimensionar automáticamente sus ventanas según su contenido
<a name="redimensionar_automaticamente_sus_ventanas_segun_contenido_ELTPARAGRAPHE000117"></a>

¿Sus ventanas se muestran en un formato minúsculo mientras que el tamaño de su pantalla es mucho mayor? El tamaño de sus ventanas se puede adaptar automáticamente a su contenido!
Por ejemplo:<br>![Before : la ventana es muy pequeña.](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_FenAdapteTailleAvant.gif)<br>![After : la ventana se redimensiona automáticamente en función de su contenido y del espacio disponible.](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_FenAdapteTailleApres.gif)
 Esta función puede activarse o desactivarse fácilmente a través de la opción "Adaptar el tamaño de la ventana" del menú contextual del botón (clic derecho en la ventana).


### Efecto traslúcido de las ventanas
<a name="efecto_traslucido_las_ventanas_ELTPARAGRAPHE000131"></a>

Dependiendo del estilo de sus aplicaciones, el borde de sus ventanas puede ser translúcido.: tendrás la posibilidad de ver en transparencia las ventanas que se encuentran abajo o el fondo Image de tu escritorio de Windows.
La opción "Desactivar el efecto translúcido de los bordes de las ventanas" se utiliza para desactivar este efecto de transparencia en sus ventanas.

**Consejo para el desarrollador**: *para habilitar (o no) el efecto de transparencia de sus ventanas, marque "Transparencia y efecto difuso de Vista" en la pestaña "Imagen" de la ventana description y configure el área sombreada de su ventana*.


### Buscar en las ventanas
<a name="buscar_las_ventanas_ELTPARAGRAPHE000142"></a>

¿Quieres encontrar una palabra en los controles de edición de tu ventana?
Nada es más fácil. La opción "Buscar" permite realizar una búsqueda en todos los controles de su ventana (controles de edición, filas de tabla, etc.).. También es posible presionar directamente Ctrl + F.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_recherche.gif)
 En la ventana que aparece, introduzca la palabra o expresión a buscar, así como las características de la búsqueda. Haga clic en el botón "Siguiente" para iniciar la búsqueda. La palabra buscada se seleccionará automáticamente si se encuentra.

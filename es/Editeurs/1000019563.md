
## Fábrica de software
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
El concepto de fábrica de software se utiliza para automatizar el Process para crear aplicaciones. 

La tarea de compilación del código fuente se asigna a un robot en una máquina de compilación. 

La reporte producida por el robot permite al jefe de proyecto conocer el nivel de estabilidad del proyecto y a los programadores corregir inmediatamente los posibles errores de compilación, en tiempos reducidos.. 

El robot también puede ejecutar las auditorías y las pruebas automatizadas. 

El propósito de la fábrica de software es también automatizar todas las tareas manuales hasta la Process para crear la aplicación instalación. 

Una vez obtenido el producto acabado, la fábrica de software se utiliza para automatizar la limpieza, la organización y el backup de los elementos de la aplicación.. 

Para obtener más información, consulte [Principio e implementación de la fábrica de software](../Editeurs/1000019568.md).

## Integración continua
<a name="integracion_continua_ELTTEXTE000116"></a>
La integración continua es una nueva característica propuesta por la fábrica de software. 

La integración continua es una set de prácticas que consisten en comprobar que, por cada modificación realizada en el código fuente, no hay regresión en la aplicación actualmente desarrollada. 

Esta verificación es realizada en tiempo real, cada noche en la mayoría de casos. Cada mañana, cada desarrollador obtiene las modificaciones hechas durante el día. 

Se define y programa un plan de acción que se inicia cada noche y en todos los proyectos.: 

- una recuperación de todos los elementos del SCM, 

- una recuperación de todos los componentes externos, 

- la generación de todos los ejecutables, componentes, ...

- haciendo algunas pruebas, ....




Los robots devuelven algunos informes de estado que se utilizan para averiguar el estado de la versión del producto..

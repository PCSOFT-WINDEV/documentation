
## Ventana con flecha (iPad)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000198"></a>
El principio de las ventanas con flecha es sencillo: cuando el usuario hace clic en un botón, la ventana se mostrará contra el botón de llamada con una flecha de dirección entre el botón y la ventana. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=iPad_Fen_fleche.gif)


**Observaciones**: 

- Este tipo de ventana sólo está disponible para las aplicaciones de iPad. Si se selecciona "Ventana con flecha" para una aplicación de iPhone, la ventana no se maximizará y no tendrá flecha.

- La ventana con flecha utiliza el estilo del sistema: el borde de la ventana y el dibujo de la flecha son suministrados por el sistema iOS.

## Creación de una ventana con flecha
<a name="creacion_una_ventana_con_flecha_ELTTEXTE000222"></a>


### Creación de una ventana con flecha
<a name="creacion_una_ventana_con_flecha_ELTPARAGRAPHE000028"></a>

Para crear una ventana con flecha: 

1. Cree una nueva ventana: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Ventana", y luego en "Ventana". 

	- En asistente para la creación de ventanas, seleccione "Espacio en blanco no maximizado con flecha" y valide. 




2. La ventana con flecha se muestra en el editor. El borde y la flecha se muestran en el editor, pero no se pueden modificar.. Por el contrario, el tamaño de la ventana puede ser modificado. 



<a name="NOTE2_2"></a>


### Barra de título
<a name="barra_titulo_ELTPARAGRAPHE000049"></a>

Por Default, la ventana creada con la flecha no contiene ninguna barra de título. 

Para mostrar una barra de título: 

1. Abra la descripción de la ventana (opción "Descripción" del menú contextual). 

2. En la pestaña "Estilo", seleccione la opción "Con una barra de título". 

3. Validar. 




El contenido que aparece en la barra de título corresponde al título de la ventana introducido en la pestaña "General" de la ventana description de la ventana con flecha. 

**Observación**: Cambiar el título mediante programación ([Título](../Proprietes/2510090.md)) sólo es posible en las "Declaraciones globales" Event de la ventana. 
<a name="NOTE2_3"></a>


### Cambiar una ventana maximizada por una ventana con una flecha
<a name="cambiar_una_ventana_maximizada_por_una_ventana_con_una_flecha_ELTPARAGRAPHE000073"></a>

Transformar una ventana maximizada en ventana con una flecha: 

1. Abra la descripción de la ventana (opción "Descripción" del menú contextual). 

2. En la pestaña "Estilo": 

	- Seleccione "Ventana no maximizada/recortada". 

	- En los parámetros, seleccione: 

		- "Ventana con flecha". 

		- "Con una barra de título" para insertar automáticamente una barra de título en la ventana. 




3. Validar.

## Programación de una ventana con flecha
<a name="programacion_una_ventana_con_flecha_ELTTEXTE000258"></a>


### Abrir una ventana con una flecha
<a name="abrir_una_ventana_con_una_flecha_ELTPARAGRAPHE000104"></a>

Para abrir una ventana con una flecha desde un botón de la aplicación: 

- En el editor, utilice la acción rápida "Abrir" associated con el botón. 

	- Mostrar la ventana de description de control Botón. 

	- En la pestaña "General", en el área "Acción", seleccione la ventana con la flecha para abrirla. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=iPad_Fen_fleche_ouvrir.gif)





- A través de la programación, utilice [OpenChild](../WDLang1/3038021.md) en el código de clic del botón. 
	 Observación: No se puede llamar a [OpenChild](../WDLang1/3038021.md) varias veces sucesivamente desde la ventana principal. Por el contrario, una ventana padre puede abrir una ventana hija que puede abrir otra ventana hija..  



<a name="NOTE3_2"></a>


### Cerrar una ventana con una flecha
<a name="cerrar_una_ventana_con_una_flecha_ELTPARAGRAPHE000126"></a>

Se puede cerrar una ventana con flecha: 

- a través de un botón que contiene [Cerrar](../WDLang1/3038018.md). 

- cuando el usuario hace clic fuera de la ventana. En este caso, se ejecuta la Event "Cerrar" de la ventana. 



<a name="NOTE3_3"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000139"></a>

- Tiene la posibilidad de pasar parámetros a una ventana con una flecha. Para obtener más información, consulte [Ventana con parámetros](../WDChamp/1010026.md).

- La ventana con flecha puede devolver un valor que se recuperará en la ventana principal (en la "Cerrar una ventana hija" Event). Para obtener más información, consulte la ayuda sobre la función [Cerrar](../WDLang1/3038018.md).

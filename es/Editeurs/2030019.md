


## Exportación de elementos a un directorio
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
Todos los elementos manejados por un proyecto (ventana, reporte, clases, set de procedimientos, consultas, ...) pueden ser transferidos de un proyecto a otro. La exportación le permite a copy los elementos del proyecto actual en otro directorio (y por lo tanto en otro proyecto).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000133"></a>


### Exportar elementos del proyecto actual
<a name="exportar_elementos_del_proyecto_actual_ELTPARAGRAPHE000018"></a>

Para exportar elementos del proyecto actual: 

1. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Exportar .. A un directorio".

2. Seleccione el directorio donde se deben exportar los ficheros de datos a. Aparece una marca de verificación delante de los archivos seleccionados.

3. En la lista de elementos WINDEV que se muestra, seleccione los elementos que desea exportar.. 
	Observación: Esta lista contiene el nombre del archivo, la description del archivo (si existe) y la ubicación actual del archivo.

4. Validar.
	Los archivos seleccionados se exportan automáticamente al directorio del proyecto especificado.




**Observación**: Para incluir los elementos exportados en los componentes de un proyecto, utilice el programa [lista de componentes del proyecto](../Editeurs/2030033.md).

## Notas
<a name="notas_ELTTEXTE000157"></a>


### Dependencia de los archivos
<a name="dependencia_los_archivos_ELTPARAGRAPHE000053"></a>

Los elementos se exportan con sus dependencias: todos los archivos relacionado al archivo exportado también se exportarán. Por ejemplo, una ventana se exporta automáticamente con su fondo Image.
<a name="NOTE3_2"></a>


### Estimación del tamaño de los archivos exportados
<a name="estimacion_del_tamano_los_archivos_exportados_ELTPARAGRAPHE000060"></a>

El botón "Calcular" sirve para estimar el tamaño de los archivos exportados.. Este tamaño tiene en cuenta el tamaño del elemento a exportar y el tamaño de los archivos en función de este elemento.

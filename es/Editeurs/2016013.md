<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
Una hoja de estilo (extensión".STY") contiene:

- el estilo de los controles de la ventana (fondo Color, fuente, Border, ...),

- la información relativa al aspecto de los controles de ventana (fondo Image, dimensión, ...).




Cuando se crea un proyecto, varias hojas de estilo son automáticamente associated con el proyecto:

- **una hoja de estilo Default WINDEV**. 
	Esta hoja de estilo contiene varios estilos de Default para todos los tipos de controles. Esta aplicación se encuentra en el directorio "\\Programas\\Datos" del directorio instalación de WINDEV.

- **una [hoja de estilo personalizada](../Editeurs/2016015.md)**.
	Al crear un proyecto, esta hoja de estilo vacía es automáticamente associated con el proyecto actual y se llama "&lt;NombreDeProyecto&gt;.STY". Esta hoja de estilo se crea en el directorio del proyecto. 
	Esta hoja de estilo contiene: 

	- información de estilo sobre la ventana (imagen de fondo, fondo Color). 

	- información de estilo sobre las opciones de menú. 

	- el estilo de los controles (y algunas propiedades de los controles como la alineación). 
			Esta hoja de estilo es totalmente modificable y personalizable.




- **una hoja de estilo tema-modelo** si el proyecto es associated con un plantilla.
	Esta hoja de estilo contiene los estilos propuestos por la plantilla del proyecto para todo tipo de controles. 
	**Observación**: La hoja de estilo tema-modelo asociada al proyecto depende del plantilla del proyecto. Si el plantilla es modificado del proyecto, el tema - modelo hoja de estilos associated será cambiado con el proyecto. Para cambiar la plantilla del proyecto, se puede utilizar uno de los siguientes métodos: 

	- en la pestaña "Estilo" de la descripción del proyecto, en la sección "Plantilla del proyecto", seleccione la plantilla deseada. 

	- en la pestaña "Proyecto", en el grupo "Otras acciones", opción "Aplicar una plantilla al proyecto". Seleccione la plantilla deseada. 


 Para obtener más información, consulte [Plantilla](../Editeurs/2016011.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000175"></a>


### Operaciones que se pueden realizar en los estilos
<a name="operaciones_que_pueden_realizar_los_estilos_ELTPARAGRAPHE000065"></a>

La mayoría de las operaciones realizadas en los estilos se realizan en la hoja de estilo personalizada. El editor de ventanas propone una interfaz específica que le permite realizar estas operaciones.

Desde la interfaz del editor de ventanas, puede hacerlo:

- [cambiar la hoja de estilo personalizada associated con el proyecto](../Editeurs/2016015.md).

- [cambiar el estilo associated por un control](../Editeurs/2016006.md).

- [modificar el estilo de una control](../Editeurs/2016006.md).

- [crear, modificar o eliminar un estilo en una hoja de estilo personalizada](../Editeurs/2016015.md).




Para una gestión avanzada de las hojas de estilo (personalizando las hojas de estilo tema-modelo por ejemplo), WINDEV permite a manipular el archivo ".STY" en el editor de ventanas.

Desde el archivo ".STY", puede hacerlo:

- [modificar o borrar un estilo en una hoja de estilo](../Editeurs/2016010.md).

- [ver el contenido de una hoja de estilo](../Editeurs/2016010.md).

- [importar un estilo en una hoja de estilo](../Editeurs/2016010.md).

- [definir el estilo Default para un tipo de control](../Editeurs/2016010.md).




Observación: También tiene la posibilidad de comparar dos hojas de estilo. Para obtener más información, consulte [Comparador de proyectos](../Editeurs/2030024.md).

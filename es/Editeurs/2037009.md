## Configuración del servidor para el protocolo CGI (AWP)
<a name="configuracion_del_servidor_para_protocolo_cgi_awp_ELTTEXTE000093"></a>
Para configurar el servidor web: 

1. Editar el archivo "srm.cnf" en un editor de texto. Este archivo se encuentra en el subdirectorio "/Conf" del directorio de configuración del servidor FrontPage.

2. Añade en el archivo "srm.cnf" el Definition del alias del protocolo CGI (WD28 0AWP.EXE) a través del siguiente Line:





```txt
WinScriptAlias /<Alias>/ "Engine directory"
```


Por ejemplo, si el protocolo CGI se encuentra en el directorio "C:\\WEBDEV\\PROGRAMS\\ENGINE\\WINDOWS\\AWP" y si el nombre del alias es WD28 0AWP:


```txt
WinScriptAlias /WD220AWP/ C:/WEBDEV/PROGRAMS/ENGINE/WINDOWS/AWP/
```


donde "C:/WEBDEV/PROGRAMAS/ENGINE/WINDOWS/AWP": es el directorio de instalación del motor WEBDEV (WD28 0AWP.EXE)

**Observaciones**: 

- los caracteres "_ASLASH_" se convierten en "/

- no olvidar el carácter final '/' en la ruta.

## Configuración del servidor para los sitios WEBDEV
<a name="configuracion_del_servidor_para_los_sitios_webdev_ELTTEXTE000117"></a>
Para configurar el servidor Web, añadir en el archivo "srm.cnf" el Definition del alias para las imágenes de cada sitio WEBDEV:


```txt
Alias /<Name Image Alias>/ "Directory of site images"
```


Ejemplo para el sitio WEBDEMO.


```txt
Alias /WEBDEMO_WEB/ "C:/WEBDEMO/WEBDEMO_WEB/"
```

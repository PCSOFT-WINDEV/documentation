


## Opciones del editor de ayuda
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
Las opciones del editor de ayuda permiten definir:

- los diferentes compiladores de ayuda utilizados (pestaña "General").

- la comprobación realizada en el sistema de ayuda cuando se abre (pestaña "General").

- la calidad de las imágenes generadas durante la compilación (pestaña "Imágenes").

- las imágenes insertadas en una Page (pestaña "Imágenes").




Para ver las opciones del editor de ayuda, en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de ayuda".

<a name="NOTE2_1"></a>


## Opciones de la pestaña "General
<a name="opciones_pestana_general_ELTTEXTE000155"></a>
La pestaña "General" incluye las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=option_editeur_aide%20-%20HC%20N%B0001.gif&type=thumb)


- Compilador de ayuda: WINDEV incluye dos compiladores de ayuda:

	- **compilador WinHelp**: este compilador corresponde al programa **hcrtf.exe**. Este programa está incluido en el subdirectorio "Programas" del directorio de instalación de WINDEV.

	- **Compilador de ayuda HTML**: este compilador corresponde al programa **hhc.exe**. Este programa está incluido en el subdirectorio "Programas" del directorio de instalación de WINDEV.


 **Observación**: Para compilar y probar los archivos de ayuda HLP o CHM, deben especificarse las rutas de acceso a ambos compiladores.

- Integridad del sistema de ayuda: Si se produce un problema al abrir el sistema de ayuda, tiene la posibilidad de comprobar (y corregir) el sistema de ayuda tan pronto como se abra.. Simplemente seleccione la opción "Verificar la integridad del sistema de ayuda durante la carga".
	**Atención**: Esta operación puede ser bastante larga.. Se debe utilizar antes de abrir un sistema de ayuda que cause problemas, y luego se debe desmarcar.

## Opciones de la pestaña "Imágenes
<a name="opciones_pestana_imagenes_ELTTEXTE000179"></a>
La pestaña "Imágenes" incluye las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=option_editeur_aide%20-%20HC%20N%B0002.gif&type=thumb)


- Calidad de las imágenes JPEG (sólo en formato CHM): Cuando se crea el sistema de ayuda en el editor de ayuda, se pueden insertar todo tipo de imágenes en las páginas de ayuda.. Sin embargo, al compilar en formato CHM, todas las imágenes utilizadas se convertirán automáticamente al formato JPEG.. De hecho, este formato comprimido se utiliza para optimizar el tamaño de sus archivos de ayuda.
	El editor de ayuda de WINDEV le permite elegir la calidad de las imágenes JPEG creadas durante la compilación.
	Debe utilizar un índice de calidad incluido entre 50 y 70 %.
	**Atención**: Cuanto más alta sea la calidad de Image, más claros serán los Image s, pero más grande será el archivo CHM compilado.

- Gestión de la inserción de una Image en una Page de ayuda : Hay varias opciones disponibles cuando se inserta un Image en una ayuda Page: 

	- **Preguntar para cada imagen si debe ser copiada en el directorio del proyecto**.

	- **Siempre copiar las imágenes en el directorio del proyecto**: esta opción se utiliza para almacenar todas las imágenes utilizadas por el archivo de ayuda en el directorio del proyecto.

	- **Nunca copiar las imágenes en el directorio del proyecto**:  todas las imágenes utilizadas por el archivo de ayuda permanecen en su directorio de origen. Si estas imágenes ya no están disponibles cuando se recopile el archivo de ayuda, se mostrará un cuadrado White en el archivo de ayuda.

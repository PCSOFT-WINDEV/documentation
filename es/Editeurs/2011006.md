## Presentación
<a name="presentacion_ELTTEXTE000216"></a>
Las principales operaciones que se pueden realizar en un LDM son las siguientes:

- [Duplicar/Copiar un MLD](#NOTE2_1).

- [Eliminar un MLD](#NOTE2_2).

- [Cambiar el nombre de un MLD](#NOTE2_3).

## Manipulación en LDM
<a name="manipulacion_ldm_ELTTEXTE000240"></a>


### Duplicar/Copiar un MLD
<a name="duplicarcopiar_mld_ELTPARAGRAPHE000039"></a>

Duplicar un MLD le permite tener dos MLD iguales con nombres diferentes.

Para duplicar/copy un LDM:

1. Abrir el análisis para duplicarlo en el editor de análisis.

2. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar como". Se muestra el análisis description.

3. Modificar el nombre y el directorio del análisis.

4. Validar las modificaciones:

	- Todos los archivos utilizados para describir el análisis duplicado se copian automáticamente en el directorio especificado.

	- El análisis duplicado es automáticamente associated con el proyecto actual.







**Observación**: Los subdirectorios de respaldo del análisis de la fuente no se copian. El nuevo análisis no contiene ninguna copia de seguridad.
<a name="NOTE2_2"></a>


### Eliminar un MLD
<a name="eliminar_mld_ELTPARAGRAPHE000070"></a>

Para eliminar un LDM:

1. Abrir explorador de Windows.

2. Borrar el directorio completo del análisis (&lt;Nombre del proyecto&gt;.Directorio ANA).




**Observación**: Si este análisis fue usado por un proyecto, este proyecto debe ser associated con otro análisis cuando se abra.
<a name="NOTE2_3"></a>


### Cambiar el nombre de un MLD
<a name="cambiar_nombre_mld_ELTPARAGRAPHE000082"></a>

Para renombrar un LDM:

1. Abre el LDM para cambiar el nombre en la editor de análisis.

2. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar como". Se muestra el análisis description.

3. Introduzca el nuevo nombre para el LDM.

4. Validar la modificación.

5. Todos los archivos utilizados para describir el análisis renombrado se copian automáticamente en el directorio especificado.

6. El análisis renombrado no es automáticamente associated con el proyecto actual. Asociar (si es necesario) el análisis renombrado con el proyecto actual.. Para más detalles, véase "Asociar un LDM a un proyecto existente".

7. En el explorador de Windows, elimine (si es necesario) el directorio de análisis de origen.



<a name="NOTE2_4"></a>


### Asociar un LDM a un proyecto existente
<a name="asociar_ldm_proyecto_existente_ELTPARAGRAPHE000109"></a>

Asociar un LDM con un proyecto existente:

1. Abra el proyecto para asociarlo.

2. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". Se abre el proyecto description.

3. En la pestaña "Análisis", especifique el nombre y ruta del LDM que debe ser associated con el proyecto. .




**Observación**: Si los datos del análisis de origen se tratan en el proyecto, la asociación de un nuevo análisis puede provocar errores (error duplicado, error de visualización,...).
<a name="NOTE2_5"></a>


### Factor de zoom
<a name="factor_zoom_ELTPARAGRAPHE000134"></a>

Se puede especificar un factor de zoom para reducir o ampliar la visualización del LDM actual.

Para especificar un factor de zoom:

- **método 1**:

	- En la pestaña "Vista", en el grupo "Ayuda edición", seleccione el factor de zoom que desea utilizar.

	- Introduzca el factor de zoom en la barra de estado del editor (abajo a la derecha).




<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ZoomBarreMessage.gif)


- **método 2**:

	- Presione la tecla Ctrl.

	- Modifique el factor de zoom con la rueda del ratón mientras mantiene pulsada la tecla Ctrl.






<a name="NOTE2_6"></a>


### Mover la pantalla LDM
<a name="mover_pantalla_ldm_ELTPARAGRAPHE000163"></a>

Para mover la pantalla de un LDM:

1. Abra el LDM en la editor de análisis.

2. Presione la tecla Alt.

3. Modifique la posición del LDM manteniendo presionado el botón izquierdo del ratón y la tecla Alt.

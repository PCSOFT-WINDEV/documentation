
## Generador de plantillas WINDEV tema: Personalización de los elementos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000494"></a>
El generador de plantillas de WINDEV tema se utiliza para generar los archivos necesarios para crear un plantilla personalizado para los proyectos WINDEV. 

Tiene la posibilidad de personalizar los elementos generados: 

- [Imágenes](#NOTE2_1): estas imágenes se utilizarán en el estilo de los controles. 

- [Modelos de ventanas](#NOTE3_1). 

- [Modelos de reportes](#NOTE4_1). 




Esta página de ayuda presenta las principales imágenes utilizadas así como las modificaciones que se pueden realizar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Imágenes de plantilla
<a name="imagenes_plantilla_ELTTEXTE000518"></a>


### Consejos
<a name="consejos_ELTPARAGRAPHE000030"></a>

- La extensión de los archivos de Image puede ser modificada. Se pueden utilizar los siguientes formatos: JPG, PNG y GIF.

- Para que una imagen no sea usada en un control, todo lo que tienes que hacer es hacer que Transparent y set tengan un tamaño de 1x1 pixel. En este caso, el Image debe estar en formato PNG. Por lo tanto, el Image no se generará al generar el plantilla.

- Para que las imágenes se muestren correctamente en sus plantillas de tema, no modifique las dimensiones de los archivos.

## Modelos de ventanas
<a name="modelos_ventanas_ELTTEXTE000548"></a>
Las plantillas de ventanas se encuentran en el directorio "Template" del proyecto fuente de plantilla. Las plantillas de ventana disponibles son: 

- Plantilla.wdt : modelo general de sus ventanas. Este modelo se utiliza para definir el fondo de sus ventanas (fondo Color, fondo Image, ...). 

- Menú_plantilla.wdt: Modelo utilizado para las ventanas de menú. 

- Template_MessBox.wdt: Modelo utilizado para las ventanas del Message Box. 

- Plantilla_Popup.wdt: Plantilla utilizada para las ventanas emergentes. 

- Template_Preview.wdt: Modelo utilizado para la presentación preliminar. 




Para personalizar las plantillas de ventana, todo lo que tiene que hacer es abrirlas en WINDEV. Le aconsejamos que abra estos elementos en WINDEV **fuera de proyecto** (Arrastre y suelte el elemento a WINDEV por ejemplo).

## Modelos de reportes
<a name="modelos_reportes_ELTTEXTE000572"></a>
Las plantillas reporte se encuentran en el directorio "Template" del proyecto fuente de plantilla. Las plantillas reporte disponibles son: 

- Plantilla_Informes.wte: modelo general para informes. 

- Plantilla_Etiqueta.wte: Plantilla utilizada para los informes de Label. 

- Plantilla_Mailing.wte: Modelo utilizado para los informes de Mailing. 




Para personalizar las plantillas de reporte, todo lo que tiene que hacer es abrirlas en WINDEV. Le aconsejamos que abra estos elementos en WINDEV **fuera de proyecto** (Arrastre y suelte el elemento a WINDEV por ejemplo).

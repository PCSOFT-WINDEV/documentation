


## Seleccione la ventana de consulta description 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000124"></a>
La ventana de description de una consulta seleccionada incluye varias secciones. Esta ventana de description se utiliza para:

- modificar los elementos utilizados en la consulta.

- gestionar las condiciones de selección.

- ver la consulta en código SQL o en lenguaje natural.

- gestionar los registros "duplicados".

- ...


Se muestra esta ventana de description:

- al crear una consulta Select. 
	Recordatorio: crear una consulta: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Consultas".




- de una consulta Select encontrada en el editor de consultas ("Consulta description" de el menú contextual de la consulta).

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
El panel "Mensajería" se utiliza para intercambiar mensajes entre los empleados de una empresa.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletMessagerie.gif)
 Para más detalles, véase [Mensajería](../Editeurs/2027023.md).

**Observación**: La aplicación de mensajería tiene en cuenta a los colaboradores registrados en el groupware de desarrolladores y/o en los diferentes Centros control. Para obtener más información, consulte [Base de datos de los centros control](../CCSuivi/3540720.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualización del panel "Mensajería
<a name="visualizacion_del_panel_mensajeria_ELTTEXTE000116"></a>
**Para visualizar el panel "Mensajería "**, vaya a la pestaña "Inicio", al grupo "Entorno", expanda "Paneles", seleccione "Paneles" y, a continuación, seleccione "Mensajería".

Para más detalles sobre la gestión de los paneles, véase [Manejo de paneles acoplables](../Editeurs/2027001.md).

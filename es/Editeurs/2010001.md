
## Crear, modificar, borrar un estilo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000150"></a>
WINDEV, WEBDEV y WINDEV Mobile permiten manipular R los estilos definidos para un sistema de ayuda.

Para editar los estilos disponibles en el sistema de ayuda actual: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Editar estilos". Esta opción está disponible cuando el sistema de ayuda se muestra en el editor.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Edition_Styles.gif)


Esta ventana presenta los diferentes estilos que se encuentran en el sistema de ayuda, así como las características de cada estilo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Añadir un nuevo estilo
<a name="anadir_nuevo_estilo_ELTTEXTE000174"></a>
Para añadir un nuevo estilo: 

1. Abre la ventana de description de estilo: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Editar estilos".

2. Haz clic en el botón "Nuevo".

3. Introduzca el nombre del nuevo estilo y sus características en la ventana de propiedades de estilo.

4. Validar la ventana para las propiedades de estilo. El estilo se muestra en la ventana de estilo description.

5. Validar el estilo de la ventana de description.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Modificar un estilo
<a name="modificar_estilo_ELTTEXTE000198"></a>
Para modificar un estilo: 

1. Abre la ventana de description de estilo: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Editar estilos".

2. Seleccione el estilo que desea modificar.

3. Haga clic en el botón "Modify".

4. Introduzca las características del estilo en la ventana de propiedades de estilo.

5. Validar la ventana para las propiedades de estilo. El estilo se muestra en la ventana de estilo description.

6. Validar el estilo de la ventana de description.




<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Eliminar un estilo
<a name="eliminar_estilo_ELTTEXTE000222"></a>
Para eliminar un estilo: 

1. Abre la ventana de description de estilo: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Editar estilos".

2. Seleccione el estilo que desea eliminar.

3. Haga clic en el botón "Eliminar". Se borra el estilo. Los párrafos associated con este estilo ya no son associated con un estilo (el cuadro combinado "Estilo" está vacío).

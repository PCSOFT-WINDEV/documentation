


## Configurando la interfaz del programa instalación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000175"></a>
Para hacer la instalación de su aplicación más fácil de usar, el editor de instalación le permite personalizar la interfaz del programa instalación. Estas opciones están disponibles en los pasos relativos a la plantilla del instalador. 

Hay dos tipos de personalización disponibles para la interfaz:

- [personalizando la interfaz para una instalación estándar o para una red instalación](#NOTE2_1).

- [personalizando la interfaz para una instalación compacta](#NOTE3_1). 




**Observación**: La personalización de la interfaz puede diferir según el lenguaje de ejecución del programa instalación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Personalizar la interfaz del programa instalación (estándar instalación o red instalación)
<a name="personalizar_interfaz_del_programa_instalacion_estandar_instalacion_red_instalacion_ELTTEXTE000199"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000029"></a>

Por defecto, el fondo del programa instalación corresponde a un gradiente de azul.

Para personalizar tu programa de instalación, puedes:

- introduzca un título para la ventana de fondo.

- seleccione una imagen lateral para la instalación asistente (un logo).

- mostrar una leyenda o un logotipo en la ventana de fondo.

- seleccione un color de degradado o una imagen para el fondo.




**Observación**: WINDEV se suministra con:

- dos imágenes de fondo preestablecidas.

- dos imágenes laterales preestablecidas.




Estas imágenes están disponibles en el subdirectorio "Programas\\Datos\\Plantillas\\WDInst" del directorio instalación de WINDEV.
<a name="NOTE2_2"></a>


### Para personalizar la interfaz del programa instalación:
<a name="para_personalizar_interfaz_del_programa_instalacion_ELTPARAGRAPHE000050"></a>

1. Seleccione el idioma para el que debe personalizarse la interfaz.

2. Escriba el título de la ventana de fondo ( ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PUCE1Bis.GIF) ). Este título se mostrará en la barra de título de la ventana. Especifique el propósito de la aplicación: "Gestión de inventario".

3. Haga clic en el botón "Logo".

4. Configure la interfaz del programa instalación.




**Por defecto, el instalación asistente no se muestra en una ventana de fondo. Esto le permite sólo modificar la imagen lateral de la asistente.**

Para obtener todas las opciones, seleccione "Mostrar imagen de fondo de instalación".

## Personalizar la interfaz del programa instalación (compacto instalación)
<a name="personalizar_interfaz_del_programa_instalacion_compacto_instalacion_ELTTEXTE000229"></a>
Por defecto, la interfaz de la compacta instalación es la siguiente:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=install_compacte.gif)
 La creación de instalación asistente le permite personalizar y traducir:

- los textos de esta interfaz

- los mensajes de error visualizados.



<a name="NOTE3_2"></a>


### Para personalizar y traducir la interfaz de la compacta instalación:
<a name="para_personalizar_traducir_interfaz_compacta_instalacion_ELTPARAGRAPHE000125"></a>

1. Haga clic en el botón "Layout" de la asistente.

2. Introduzca los nuevos mensajes en el idioma seleccionado.




## Consulta y proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una consulta no necesariamente debe estar vinculada a un proyecto. Sin embargo, es necesario que las consultas estén integradas en un proyecto para poder llamarlas desde un proceso del proyecto.

Por otro lado, las consultas deben estar vinculadas a un análisis WINDEV, WEBDEV o WINDEV Mobile. Una consulta puede manipular datos que provienen de un análisis distinto al análisis inicial. En este caso, la estructura de los archivos y elementos utilizados por la consulta debe ser idéntica.

El análisis WINDEV, WEBDEV o WINDEV Mobile contiene la descripción de los archivos que serán interrogados por la consulta.

Si se modifica el análisis, se realiza una sincronización automática entre el análisis y las consultas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_et_projet.gif)
Al crear la consulta:

- si un proyecto asociado a un análisis está abierto, la consulta se vinculará automáticamente al análisis del proyecto.

- si no hay ningún proyecto abierto, se debe especificar el análisis utilizado en el asistente de creación de consultas.

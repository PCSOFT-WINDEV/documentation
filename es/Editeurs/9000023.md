


## Código colapsado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000254"></a>
El código está colapsado para mejorar la legibilidad, al mostrar sólo los comentarios delante de las líneas de código.

Puede contraer un código que ya es conocido y verificado.

Colapsar este código se utiliza para liberar espacio en el editor de código. El código se vuelve más claro y legible.

Se pueden colapsar varios tipos de líneas de código:

- las líneas de código correspondientes a una estructura de código específica: las sentencias condicionales (IF, CASE, WHEN EXCEPTION), las declaraciones de variables globales, constantes, estructuras, ....

- las líneas de código precedidas por un comment.

- las líneas de código específicas, definidas por el desarrollador.

- las líneas de código correspondientes a una cadena de varias líneas. 

- las líneas de código de un Process.




También puede declarar que los bloques de código personalizados colapsen.

Ningún código es colapsado por Default.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000278"></a>


### Colapsar un código existente
<a name="colapsar_codigo_existente_ELTPARAGRAPHE000043"></a>

Un código que puede ser colapsado está precedido por el signo "-".

Para contraer un código existente, puede hacerlo:

- haga clic en el "-" Icon que se encuentra delante de la declaración. Sólo la línea de comment que precede a la declaración seguirá siendo Visible.

- mostrar el menú contextual del "-" Icon y seleccionar "Colapso".




También es posible:

1. Mueva el cursor de edición al código para contraerlo.

2. Utilice uno de los siguientes métodos:

	- en la pestaña "Vista", en el grupo "Código contraíble ", expanda "Contraer" y seleccione "Contraer".

	- pulsar Ctrl + la tecla "-" en el teclado numérico







Cuando el cursor del ratón pasa por encima de un código colapsado, un tooltip muestra una sección del código colapsado.

**Observación**: Para contraer todas las líneas de código colapsables en la Process actual, en la pestaña "Vista", en el grupo "Código contraíble ", expanda "Contraer" y seleccione "Contraer todo".
<a name="NOTE2_2"></a>


### Expansión de un código
<a name="expansion_codigo_ELTPARAGRAPHE000091"></a>

Un código que se puede ampliar va precedido del signo "+".

Para desglosar un código, puede hacerlo:

- haga clic en el "+" de Icon que se encuentra delante del código para expandirlo.

- mostrar el menú contextual del "+" Icon y seleccionar "Expandir".




También es posible:

1. Mueva el cursor de edición al código para expandirlo.

2. Utilice uno de los siguientes métodos:

	- en la pestaña "Vista", en el grupo "Código contraíble ", amplíe "Contraer" y seleccione "Ampliar".

	- pulsar Ctrl + la tecla "+" en el teclado numérico







**Observación**: Para expandir todas las líneas de código colapsadas en el Process actual, en la pestaña "Vista", en el grupo "Código contraíble ", expanda "Contraer" y seleccione "Expandir todo".
<a name="NOTE2_3"></a>


### Definiendo una set de líneas de código para colapsar/expandir
<a name="definiendo_una_set_lineas_codigo_para_colapsarexpandir_ELTPARAGRAPHE000137"></a>

Para definir específicamente las líneas de código a colapsar/desplegar:

1. En el editor de código, seleccione las diferentes líneas de código que pueden ser colapsadas/expandidas.

2. En la pestaña "Vista", en el grupo "Código contraíble ", amplíe "Contraer" y seleccione "Contraer selección".


El código se colapsa inmediatamente y se muestra el "+" de Icon delante del código.

**Observación**: Para expandir permanentemente esta set de líneas de código:

1. Sitúe el cursor del ratón sobre este código.

2. En la pestaña "Vista", en el grupo "Código contraíble ", expanda "Contraer" y seleccione "Expandir permanentemente".



<a name="NOTE2_4"></a>


### Declarar un bloque de código personalizado
<a name="declarar_bloque_codigo_personalizado_ELTPARAGRAPHE000180"></a>Para declarar un bloque de código personalizado, utilice la siguiente sintaxis: 


```txt
<BLOCK>

...

<END>
```


donde 


```txt
<BLOCK Comment always displayed>

...

<END>
```


Estos bloques de código se utilizan, por ejemplo, para resaltar los diferentes pasos de un algoritmo o para agrupar las declaraciones de variables o miembros..




## Pegado especial 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000166"></a>
El **Pasta especial** le permite elegir explícitamente el formato en el que se pegará el contenido del portapapeles en el editor de ventanas o páginas.. 

Por ejemplo, si usted copy un archivo de datos encontró en el análisis, el comando **Special Paste** le permite:

- Crear un tabla que muestre los registros encontrados en el archivo de datos copiado.

- copy el nombre del archivo de datos en una control Estático.

- Mostrar la representación gráfica de un archivo de datos en un control Imagen y crear un archivo Bitmap (extensión "BMP").




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000190"></a>
Para utilizar el comando Pegar especial:

1. Seleccione el elemento a pegar. 

2. copy el elemento a pegar: 

	- en WINDEV, WEBDEV o WINDEV Mobile: en la pestaña "Inicio", en el grupo "Edición", haga clic en "Copiar". 

	- "Editar... copy" en el software utilizado. 

	- Ctrl + C.




3. Muestra la ventana o página donde se va a pegar el elemento.

4. En la pestaña "Inicio", en el grupo "Edición", haga clic en "Pegar" y seleccione "Pegado especial".

5. Seleccione el formato en el que se pegará el elemento copiado (ver más abajo).

6. Validar la operación de pegado.

## Notas
<a name="notas_ELTTEXTE000214"></a>
Los diferentes formatos propuestos por WINDEV, WEBDEV y WINDEV Mobile para la pasta especial son:

- **archivo de Windows**: Se han copiado uno o más archivos en el portapapeles. Si se selecciona esta opción, se crearán uno o más controles en la ventana actual.. El tipo de los controles creados depende del tipo de archivos copiados. 
	Por ejemplo, si se copia un archivo Image del explorador, se creará un control Imagen que muestre el Image en la ventana actual. El control Imagen es associated con el archivo de imagen copiado (fuente ruta).

- **Fichero o artículo de análisis**: Uno o más archivos de datos o elementos encontrados en el análisis se han copiado en el portapapeles.. Si se selecciona esta opción, se crearán uno o más controles en la ventana actual.. 
	Por ejemplo, si se copia la descripción del archivo de datos "Cliente", se creará una tabla con la lista de los clientes en la ventana actual.

- **Grupo de objetos móviles WINDEV, WEBDEV o WINDEV**: Uno o más elementos de WINDEV, WEBDEV o WINDEV Mobile (consulta, control, ...) han sido copiados en el portapapeles. Si se selecciona esta opción, se crearán uno o más controles en la ventana o página actual.. 
	Por ejemplo, si se copia un botón que se encuentra en el diccionario WINDEV, este botón se creará en la ventana actual.

- **Image**: Una o más imágenes han sido copiadas en el portapapeles. Si se selecciona esta opción:

	- Se crearán uno o más controles en la ventana actual. Estos controles mostrarán las imágenes copiadas. Las imágenes copiadas **no serán automáticamente associated** con el Color Palette.

	- Se crearán uno o más archivos de mapa de bits (extensión".BMP"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.
			Por ejemplo, si se copia un Image mostrado en un software de procesamiento de Image ("Photoshop" por ejemplo):

		- se creará en la ventana actual un control Imagen que muestre el Image copiado,

		- se creará un archivo Bitmap (extensión"BMP"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.




- **Image con Palette**: Una o más imágenes han sido copiadas en el portapapeles. Si se selecciona esta opción:

	- Se crearán uno o más controles en la ventana actual. Estos controles mostrarán las imágenes copiadas. Las imágenes copiadas **se asociarán automáticamente a** con el color Palette.

	- Se crearán uno o más archivos de mapa de bits (extensión".BMP"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.
			Por ejemplo, si se copia un Image mostrado en un software de procesamiento de Image ("Photoshop" por ejemplo):

		- se creará en la ventana actual un control Imagen que muestre el Image copiado,

		- se creará un archivo Bitmap (extensión"BMP"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.




- **Image (improved metafile)**: Una o más imágenes han sido copiadas en el portapapeles. Si se selecciona esta opción:

	- Se crearán uno o más controles en la ventana actual.

	- Se crearán uno o más archivos de metafile (extensión".EMF"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.
			Por ejemplo, si se copia un Image visualizado en un software de procesamiento Image (por ejemplo, "Visio"):

		- se creará en la ventana actual un control Imagen que muestre el Image copiado,

		- se creará un archivo metafile (extensión".EMF"). Se utiliza un selector de archivos para especificar el nombre de archivo y ruta.




- **Text**: Se ha copiado un texto en el portapapeles. Si se selecciona esta opción, se creará una control Estático multilínea en la ventana actual.. Este control mostrará el texto copiado ** sin el formato**.
	Por ejemplo, si se ha copiado un texto escrito en Word, se creará un control Estático multilínea en la ventana actual.. Este control mostrará el texto copiado ** sin el formato**.

- **RTF**: Un texto formateado fue copiado al portapapeles. Si se selecciona esta opción, se creará una control Estático multilínea en RTF en la ventana actual. Este control mostrará el texto copiado ** con formato**.
	Por ejemplo, si el texto escrito en negrita y en rojo en Word fue copiado, se creará una multilínea control Estático en RTF en la ventana actual. Este control mostrará el texto copiado ** con formato** (en negrita y rojo).

- **Selección del editor de ventanas**: Uno o más elementos encontrados en el editor de ventanas (control Campo de entrada, botón, ventana, ....) han sido copiados en el portapapeles.. Si se selecciona esta opción, los elementos copiados se crearán en el editor de ventanas.. 
	Por ejemplo, si se ha copiado una selección de controles, estos controles seleccionados se crearán en la ventana actual..
	**Observación**: Esta opción corresponde a un "Copiar/pegar" estándar de controles.

- **Selección del editor de páginas**: Uno o más elementos encontrados en el editor de páginas (control Campo de entrada, botón, ....) han sido copiados en el portapapeles.. Si se selecciona esta opción, los elementos copiados se crearán en el editor de páginas. 
	Por ejemplo, si se ha copiado una selección de controles, estos controles seleccionados se crearán en la página actual..
	**Observación**: Esta opción corresponde a un "Copiar/pegar" estándar de controles.

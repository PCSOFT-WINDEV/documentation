## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
Los estilos WEBDEV son estilos avanzados, más fáciles de usar que los estilos CSS.

Un estilo de WEBDEV agrupa un set de estilos CSS y características para un control.

Por ejemplo, para un botón, el estilo WEBDEV agrupa:

- El aspecto del botón,

- Las imágenes associated con el botón,

- El estilo de CSS associated con el botón de pie de foto.




Un control puede ser con associated:

- uno o más estilos CSS. Los estilos CSS utilizados por los control se especifican en los diferentes elementos de la pestaña "Style" de la ventana description de control. 

- un estilo WEBDEV. El estilo de WEBDEV utilizado se selecciona en la pestaña "Estilo" de la ventana de description de la control. 




**Observación**: Un estilo de WEBDEV puede ser modificado desde la ventana de description del control. Esta modificación puede ser: 

- local a la control. En este caso, la redefinición del estilo se realiza sólo para la control. 

- aplicado a todos los controles que utilizan el estilo WEBDEV.

## Estilos WEBDEV
<a name="estilos_webdev_ELTTEXTE000147"></a>


### Creación y modificación de un estilo WEBDEV
<a name="creacion_modificacion_estilo_webdev_ELTPARAGRAPHE000036"></a>

La creación y la modificación de un estilo WEBDEV puede ser realizada: 

- de la ventana de description de los estilos de control. 

- desde la ventana de gestión de estilos WEBDEV. Para obtener más información, consulte [Edición de las hojas de estilo WEBDEV](../Editeurs/2016014.md). 





### Creación y modificación desde la ventana de description de control
<a name="creacion_modificacion_desde_ventana_description_control_ELTPARAGRAPHE000047"></a>

La ventana description de control s se utiliza para gestionar el estilo de este control (pestaña "Estilo"). La pestaña "Estilo" gestiona varios elementos. Los elementos que pueden utilizar un estilo CSS se identifican mediante "(CSS)" al final de su título. 
Los otros elementos son administrados únicamente por el estilo WEBDEV.

En la pestaña "Estilo" aparece una ventana a la derecha de la ventana de la pestaña 7.. Esta ventana permite: 

- **encuentra el nombre del estilo de WEBDEV associated con el control**. Si no hay un estilo de WEBDEV es associated con el control, "Aspecto libre: no se muestra el estilo relacionado WEBDEV".. 

- **averiguar si las características de estilo del control se redefinen en relación con el estilo WEBDEV**. En este caso, existen algunas "sobrecargas" de estilo para este control. Estas sobrecargas pueden ser listadas y borradas si es necesario.. 

- **get una visión general del estilo inicial** del control. 




Esta ventana también se puede utilizar para realizar algunas acciones. Es posible: 

- **Seleccionar un estilo**. Esta opción se utiliza para seleccionar un estilo de WEBDEV y asociarlo con el actual control. En este caso, el control se actualizará automáticamente cuando se actualice el estilo associated. 

- **Agregar este estilo al proyecto**. Esta opción se utiliza para crear un nuevo estilo de WEBDEV en el proyecto a partir de todas las características definidas actualmente para el actual control. El control será automáticamente associated con este nuevo estilo WEBDEV. Luego, todo lo que tienes que hacer es especificar el nombre del nuevo estilo. 

- **Actualizar estilo en el proyecto**. Esta opción aparece cuando se modifica el estilo de la control. Esta opción se utiliza para tener en cuenta estas modificaciones en todos los controles que utilizan el estilo WEBDEV actual.. El estilo WEBDEV se actualiza. 

- **Disociar estilo**. Esta opción se utiliza para eliminar el vínculo entre el estilo de WEBDEV y el estilo de la control. Si el estilo WEBDEV evoluciona, estas evoluciones no serán tenidas en cuenta por la control.

### Edición de los estilos WEBDEV
<a name="edicion_los_estilos_webdev_ELTPARAGRAPHE000069"></a>

WEBDEV permite editar los estilos de WEBDEV que se encuentran en la hoja de estilo associated con el proyecto.

Para obtener más información, consulte [Edición de los estilos WEBDEV](../Editeurs/2016014.md).

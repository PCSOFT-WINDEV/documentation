


## Renombrar un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
Renombrar un proyecto se utiliza para modificar el nombre del proyecto. Todos los elementos relacionado al proyecto antes de renombrarlo se mantienen.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000144"></a>
Para renombrar el proyecto actual:

1. Abrir el proyecto cuadro de mando. 

2. En la pestaña "Proyecto", en el grupo "Otras acciones", despliegue "Más acciones" y seleccione "Renombrar proyecto".

3. Especifique el nuevo nombre del proyecto y valide. 




El nombre del proyecto se modifica automáticamente. Se conservan los elementos relacionado al proyecto.

**Observaciones**:

- Sólo se modifican el nombre del proyecto (archivo WDP, WWP o WPP) y el nombre del directorio de compilación (.cpl).. 

- Los siguientes elementos no se modifican cuando se cambia el nombre del proyecto:  

	- el nombre del directorio del proyecto. 

	- el nombre del análisis (si existe).




- Si el ejecutable se generó antes de la operación de renombrar, su nombre no se modifica durante la siguiente generación del ejecutable..


## Auditorías
			



## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
Las auditorías proporcionan un conjunto de funcionalidades para mejorar automáticamente la calidad y el rendimiento del proyecto, y para supervisar las condiciones en las que se ejecuta.

Existen varios tipos de auditorías:

- Auditoría estática.

- Auditoría UI.

- Auditoría dinámica.

- Telemetría. 




Estos diferentes tipos de auditorías deben utilizarse conjuntamente para sacarles el máximo partido.





## Auditoría estática
<a name="auditoria_estatica_ELTTEXTE000141"></a>
La auditoría estática realiza un análisis detallado de un proyecto y sus elementos.

Esta auditoría se realiza desde el editor del proyecto.

Consulte la página de ayuda de associated para saber cómo funciona: [Auditoría estática](../Editeurs/1014501.md).



## Auditoría UI
<a name="auditoria_ELTTEXTE000165"></a>
La auditoría de UI detecta las características obsoletas utilizadas en sus aplicaciones. 

A partir de la versión 27, esta auditoría es activada por Default cuando se abre un proyecto. 

Esta auditoría analiza directamente las ventanas y las páginas. 

Consulte la página de ayuda de associated para saber cómo funciona: [Auditoría UI](../Editeurs/9000207.md).

Observación: Los problemas detectados por la auditoría UI también se muestran en la auditoría estática. 



## Auditoría dinámica
<a name="auditoria_dinamica_ELTTEXTE000189"></a>
La auditoría dinámica analiza el comportamiento de un proyecto durante su ejecución.

Esta auditoría puede realizarse en modo de prueba o en el entorno de producción.

Consulte la página de ayuda de associated para saber cómo funciona: [Auditoría dinámica](../Editeurs/1014502.md).





## Telemetría
<a name="telemetria_ELTTEXTE000213"></a>
La telemetría recoge y agrega datos sobre el uso de sus aplicaciones. 

Además, muestra un reporte de las aplicaciones implementadas, para así poder mejorarlas. 

Consulte la página de ayuda de associated para saber cómo funciona: [Presentación de la telemetría](../WDLang1/1410086580.md).

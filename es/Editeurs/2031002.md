El RAD (Rapid Application Development) se utiliza para crear rápidamente ventanas a partir de patrones preestablecidos.. Por Default, 10 patrones de ventanas RAD se suministran con WINDEV.

Cada patrón corresponde a un archivo Script (extensión .MDL). Este archivo contiene:

- La description del patrón RAD, mostrada al iniciar la creación de la ventana asistente

- Los diferentes pasos de la asistente

- La description de la ventana generada y su código

- La description de los controles generados y su código

## Los archivos de patrones
<a name="los_archivos_patrones_ELTTEXTE000276"></a>
Todos los archivos utilizados por el RAD (ventana o proyecto RAD) están disponibles en el subdirectorio "_ASLASH_ Personal\\Templates\\RAD" de WINDEV.

Para cada patrón RAD, este directorio contiene:

- **Un archivo BMP "RAD_WIN_xx"**:
	Image correspondiente al Icon para el tipo de la ventana generada, XX corresponde al número del patrón RAD.

- **Un archivo MDL** que contiene el script para el patrón RAD.

- Los archivos de las diferentes clases WINDEV (.WDC) utilizadas por los patrones del Object RAD

## Modificar o crear un patrón RAD
<a name="modificar_crear_patron_rad_ELTTEXTE000300"></a>


### Modificación de un patrón RAD existente
<a name="modificacion_patron_rad_existente_ELTPARAGRAPHE000147"></a>

Para modificar un patrón RAD existente: 

1. En el directorio "\\Personal\\Templates\\RAD" de WINDEV, crea una copia de seguridad copy de todos tus archivos de guión (extensión mdl).

2. Abra el archivo de patrones que debe modificarse en el bloc de notas (notepad.exe) de Windows.

3. Modificar el código del script según sus necesidades. Para más detalles sobre el código de la secuencia de comandos, véase [La estructura de los patrones RAD](../Editeurs/2031001.md).

4. Guardar los cambios

5. Reiniciar WINDEV. Las modificaciones realizadas en el fichero modelo se tendrán en cuenta automáticamente.



<a name="NOTE3_2"></a>


### Creación de un nuevo patrón RAD a partir de un patrón existente
<a name="creacion_nuevo_patron_rad_partir_patron_existente_ELTPARAGRAPHE000163"></a>

Para crear un nuevo patrón RAD a partir de un patrón existente: 

1. En el directorio "\\Personal\\Templates\\RAD" de WINDEV, crea una copia de seguridad copy de todos tus archivos de guión (extensión mdl).

2. Cambie el nombre del archivo "copy de..." del archivo de patrones que se usará como base para su nuevo patrón.

3. Abra este archivo de patrón en el Bloc de notas (notepad.exe) de Windows.

4. Modifique el siguiente código de acuerdo a sus requerimientos:





```txt
#############################
# name of the pattern
#############################
Name=Form with linked form
Title=Form and form
Number=11
#############################
# description of the pattern
#############################
Description=Beginning
Form with Linked Form pattern.
This pattern is suited for the files linked by a simple link.
Description=End
```





|   |   |
| --- | --- |
| denominar | Nombre del patrón que aparece en la ventana de creación asistente. |
| Título | Título de la ventana generada. |
| Número | Número del patrón RAD. Este número debe ser modificado para crear un nuevo patrón RAD. |
| description = Comienzo<br>description = End | description del patrón mostrado en la ventana de creación asistente. |

**Observación**: el código de los archivos MDL no está sangrado y no puede ser sangrado.

**5.** Modificar el código del script según sus necesidades. Para más detalles sobre el código del script, véase [La estructura de los patrones RAD](../Editeurs/2031001.md).
**6.** Grabar las modificaciones.
**7.** Crear (si es necesario) las imágenes associated con el nuevo patrón RAD (RAD_Win_XX.BMP donde xx corresponde al número del patrón).
**8.** Reiniciar WINDEV. El nuevo patrón se muestra cuando se inicia la creación de la ventana asistente.

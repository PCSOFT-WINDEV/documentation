


## WDInst : lenguajes apoyados por la instalación 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000176"></a>
Si sus aplicaciones deben ser instaladas en varios países, el editor de instalación WDInst le da la posibilidad de generar un instalación multilingüe.

Los lenguajes soportados por el programa instalación serán propuestos al usuario al instalar la aplicación.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Installation_WD_defaut%20-%20HC%20N%B0002.gif)

## Configurando una instalación multilingüe
<a name="configurando_una_instalacion_multilingue_ELTTEXTE000200"></a>
La configuración de una instalación multilingüe se realiza en el editor de instalación WDInst. Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

El programa de creación de instalación asistente también puede ser utilizado para configurar un instalación multilingüe.

Los idiomas "francés" y "americano" son propuestos por Default al usuario cuando se instala la aplicación.. Para mantener este modo de gestión, se deben configurar los siguientes elementos para estos dos idiomas:

- una licencia, si una licencia se muestra al principio de la aplicación instalación.

- un archivo "Léame", si aparece un archivo "Léame" al final de la aplicación instalación.

- un diseño.

- diapositivas (sólo WDInst).

- los títulos de los atajos y los grupos de programas (sólo WDInst).

## Gestionar varios idiomas en el programa instalación
<a name="gestionar_varios_idiomas_programa_instalacion_ELTTEXTE000224"></a>
Para apoyar a varios idiomas en el programa de instalación: 

1. Selecciona "Configuración ... Estilo".. Las lenguas ya admitidas por el programa instalación figuran en la zona "Lenguas disponibles durante instalación".

2. Pulse el botón "Añadir" para añadir nuevas lenguas.
	**Observación**: Una vez añadido un nuevo idioma, se deben configurar las características específicas de este idioma (opciones de diseño, diapositivas,...). Estos idiomas también deben ser compatibles con el proyecto "WDSetup".. Para obtener más información, consulte [WDSetup: Gestión de idiomas adicionales](../Editeurs/2028037.md). 

3. Haga clic en "Suprimir" para suprimir la lengua seleccionada en "Lenguas disponibles para instalación"..
	**Atención**: Se perderán todas las características definidas para el idioma eliminado (opciones de diseño, diapositivas,...).

## Configurando un programa multilingüe de instalación
<a name="configurando_programa_multilingue_instalacion_ELTTEXTE000248"></a>
Una vez seleccionados los diferentes idiomas que soporta el programa instalación, debe:

- [seleccionar una licencia específica para cada idioma](../Editeurs/2028053.md) si es necesario.

- [seleccione un archivo "Léame" específico para cada idioma](../Editeurs/2028053.md) si es necesario..

- [configurar un diseño específico para cada idioma](../Editeurs/2028030.md).

- [seleccionar las diapositivas específicas para cada idioma](../Editeurs/2028030.md).

- [entrar en el título de los accesos directos](../Editeurs/2028004.md) (creado durante la instalación) [en todos los idiomas disponibles y especifique el grupo de programas en el que deben crearse los accesos directos](../Editeurs/2028004.md).
	El pie de foto de los atajos se creará en el lenguaje de ejecución del programa instalación.

- personalizar el proyecto "WDSetup" si el programa instalación debe soportar idiomas distintos del americano y el francés (idiomas propuestos por Default).

- Si el instalación incluye un desinstalador, traduzca el [opciones del desinstalador](../Editeurs/2028087.md) en los diferentes idiomas. 




Para que el programa instalación admita idiomas adicionales, el proyecto "WDSetup" debe personalizarse antes de crear el programa instalación. Este proyecto (suministrado con los ejemplos completos de WINDEV) corresponde al programa estándar instalación de las aplicaciones WINDEV. Para obtener más información, consulte [Usando un programa personalizado de instalación](../Editeurs/2028037.md).

### Aplicación instalada en la red
<a name="aplicacion_instalada_red_ELTPARAGRAPHE000104"></a>

Al actualizar una aplicación instalada en la red, se muestra una ventana cuando se inicia la aplicación desde un ordenador de usuario final. Esta ventana informa al usuario de que existe una versión actualizada de la aplicación.. Esta ventana permite instalar o no la actualización.

## Inicio de la aplicación multilingüe
<a name="inicio_aplicacion_multilingue_ELTTEXTE000278"></a>
Por defecto, una aplicación multilingüe se ejecuta en el idioma principal del proyecto ("Idioma principal" en la pestaña "Idiomas" del proyecto description).

**Para iniciar la aplicación en el idioma de tiempo de ejecución del programa instalación**: 

- recuperar el idioma de ejecución del programa instalación y guardar este idioma en un archivo ".INI" o en el registro del ordenador instalación. Esta operación puede realizarse automáticamente durante la instalación. Para obtener más información, consulte [Gestión del registro y de los archivos ".INI" en instalación](../Editeurs/2028013.md).

- utilizar [Nación](../WDLang1/3054014.md) para cambiar el idioma que utiliza la aplicación en tiempo de ejecución.




Para obtener más información, consulte [Desarrollo de aplicaciones multilingües](../WDLang1/3054008.md).

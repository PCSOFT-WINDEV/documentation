<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
La modificación del automatic de los archivos de datos permite actualizar el description de los archivos de datos en el servidor Web.

Para obtener más información, consulte [Modificación automática de archivos de datos](../WDLang4/3044195.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración de la modificación de datos de automatic
<a name="configuracion_modificacion_datos_automatic_ELTTEXTE000119"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=mofiauto.gif&type=thumb)
Para realizar la modificación de automatic de los archivos de datos:

- en formato HFSQL Classic, marque "Incluir el automatic automatic de los ficheros en el instalación" y compruebe el análisis associated con el sitio.

- en formato HFSQL Client/Server, marque "Realizar la modificación de los ficheros de datos HFSQL Client/Server".. El botón "Conexiones" permite describir los parámetros de las conexiones HFSQL Client/Server correspondientes.

- en formato Hyper File 5.5 a HFSQL Classic, marque "Realizar la modificación automatic de los ficheros de datos Hyper File 5.5 y la conversión a HFSQL Classic".



<a name="NOTE2_2"></a>


### Gestión de las contraseñas de los archivos de datos de HFSQL Classic
<a name="gestion_las_contrasenas_los_archivos_datos_hfsql_classic_ELTPARAGRAPHE000033"></a>

Si sus ficheros de datos HFSQL Classic tienen contraseña protected, haga clic en el botón "Contraseñas".

Para cada archivo de análisis, especifique (si es necesario):

- la contraseña del archivo (si existe) en el análisis en formato HFSQL Classic. 
	**Observación**: Para modificar esta contraseña, marque "Modificar la contraseña" y escriba dos veces la nueva contraseña.

- si el sitio utiliza un análisis Hyper File 5.5 migrado:

	- el análisis de la fuente en formato Hyper File 5.5 (archivo ".WDA").

	- el nombre del archivo en el análisis en formato Hyper File 5.5.

	- la contraseña del archivo (si existe) en el análisis en formato Hyper File 5.5.







**Observación**: Esta información (excepto las contraseñas) se muestra automáticamente.

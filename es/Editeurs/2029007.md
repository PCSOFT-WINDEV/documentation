
## Entorno del editor de código 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EnvEdCode.gif&type=thumb)


Los principales elementos del editor de código son los siguientes:


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif)<br> | **Cinta de opciones**<br>Da acceso rápido a las principales características del editor de código. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.gif)<br> | **Panel de cinta específico del editor de código**<br>Se utiliza para seleccionar rápidamente los códigos a visualizar. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif)<br> | **Ventanas de código**<br>Se pueden abrir varias ventanas de código al mismo tiempo. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE4.gif)<br> | **Punto de interrupción para el depurador**<br>La ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeStop.gif) Icon señala un punto de ruptura. Al ejecutar la prueba del proyecto, el depurador se iniciará automáticamente cuando se ejecute el Line precedido por este Icon. Para añadir o eliminar un punto de interrupción, sólo tienes que hacer clic delante del código Line. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle5.gif)<br> | **Error**<br>Un texto subrayado en rojo indica que se ha producido un error en el código. Los detalles del error están disponibles:<br><br>- cuando la Line de código es pasada por el cursor del ratón: un tooltip muestra el texto del error.<br><br>- en el panel "Errores de compilación".<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE6.gif)<br> | **Entrada asistida**<br>Lista desplegable que propone los diferentes valores posibles para cada parámetro de la función que se está escribiendo.<br><br>Para obtener más información, consulte [Autocompletado](../Editeurs/2013001.md).<br><br>La opción &lt;Xxx wizard&gt; indica que un asistente permite introducir los diferentes parámetros de la función respondiendo a algunas preguntas sencillas. Para iniciar el asistente, seleccione la opción &lt;xxx wizard&gt;. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE7.gif)<br> | **set de paneles**<br>Los siguientes paneles son "específicos" del editor de código:<br><br>- Panel "Errores de compilación" que muestra los detalles de los errores,<br><br>- Panel "Vista contextual del código" utilizado para ver el código del elemento actual en el editor.<br><br>- Panel "Depurador" que muestra las opciones del depurador cuando éste se inicia.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE8.gif)<br> | **Minimapa**<br>Una vista previa reducida del código se muestra en la parte derecha (o izquierda) del editor de código.<br>La sección que corresponde al código que se muestra está claramente representada. El movimiento de la pequeña ventana de visualización también mueve el contenido del editor de código. |

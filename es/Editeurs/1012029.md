## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
El concepto de Mashup consiste en agregar datos procedentes de diferentes programas informáticos, diferentes servicios, diferentes fuentes de datos, ....

Esta agregación es sencilla; en la mayoría de los casos, se realiza en una página de un sitio INTERNET.

El propósito de Mashup es permitir una interoperabilidad real, un enlace entre diferentes software. Es más fácil crear aplicaciones compuestas.

## Mashup específico de WEBDEV
<a name="mashup_especifico_webdev_ELTTEXTE000157"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Para simplificar el mashup en los sitios desarrollados en WEBDEV, usted debe ser capaz de interactuar con los diferentes servicios propuestos por los sitios en la Web.

WEBDEV le ofrece la posibilidad de:

- realizar un mashup a nivel de servidor usando las funciones de WLanguage directamente.

- realizar un mashup a nivel de "navegador" (sin ejecutar código en el WEBDEV servidor de aplicaciones):

	- utilizando APIs Web.

	- incluyendo componentes evolucionados en la página (sección de páginas incluyendo los procesos) para interactuar directamente con ellos.







No hay carga de trabajo adicional en el servidor según las operaciones realizadas por el usuario Web. La programación se realiza directamente en WLanguage: el código Javascript ya no es necesario.

Los intercambios de datos propuestos por estos servicios pueden estructurarse en XML o directamente en JSON.

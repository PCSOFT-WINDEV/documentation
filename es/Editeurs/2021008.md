


## Notas personalizadas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000103"></a>
Las notas personalizadas te permiten escribir cualquier idea que quieras recordar, notas que no debes olvidar, .... Puede introducir tantos recordatorios como desee en cualquiera de los elementos de su proyecto.. Incluso puede pegar sus notas personalizadas en el escritorio de Windows.

Un note personalizado puede contener texto (texto libre o texto RTF), imágenes, sonido, ...

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000127"></a>


### Añadir una Nota Personal
<a name="anadir_una_nota_personal_ELTPARAGRAPHE000020"></a>

Para añadir una costumbre-note:

1. Seleccione "Nueva Nota Personal" en el menú contextual (clic derecho del ratón).

2. La asistente para crear una nueva Nota Personal comienza. Seleccione el tipo de información incluida en el nuevo Nota Personal:

	- texto simple.

	- texto formateado.

	- documento escaneado.

	- archivo de texto, hoja de cálculo Excel, Image, etc.

	- grabación de voz.




3. Según el tipo de Nota Personal, introduce el contenido del Nota Personal, selecciona el archivo esperado o Record el memorándum de voz.

4. Da un título a esta nueva Nota Personal y valida. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Perso_notes%20-%20HC%20N%B0001.gif&type=thumb)




<a name="NOTE2_2"></a>


### Asociar una Nota Personal a un elemento de proyecto
<a name="asociar_una_nota_personal_elemento_proyecto_ELTPARAGRAPHE000039"></a>

Asociar una Nota Personal a un elemento de proyecto:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Perso_notes%20-%20HC%20N%B0003.GIF) (clavija) en la barra de título de Nota Personal. La Icon se convierte en un empujón hacia fuera: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Perso_notes%20-%20HC%20N%B0002.GIF).

2. Abre el elemento del proyecto al que el Nota Personal debe ser associated.

3. Haga clic de nuevo en la chincheta para asociar la Nota Personal al elemento actual.



<a name="NOTE2_3"></a>


### Borrar un Nota Personal
<a name="borrar_nota_personal_ELTPARAGRAPHE000054"></a>

Para borrar una Nota Personal, simplemente haz clic en la "X" de la Nota Personal (arriba a la derecha) y valida el cuadro de diálogo.

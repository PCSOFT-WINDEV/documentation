
## Property de la entidad/asociación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
En la metodología MERISE, las propiedades corresponden a las características de las entidades.

El editor de análisis le permite crear propiedades fácilmente.

Durante la [generación de LDM a partir de CDM](../Editeurs/2011020.md), las propiedades se convertirán automáticamente en [elementos](../Editeurs/2011009.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación de una Property
<a name="creacion_una_property_ELTTEXTE000133"></a>
Para describir un Property:

1. Haga doble clic en la entidad o la asociación en la que debe crearse el Property. Se muestra la ventana de description de las propiedades.

2. Haga clic en la primera fila vacía en la tabla de propiedades o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeAjouterPropriete.gif)

3. Especifica el nombre, el título, el tipo, el tamaño y las notas de Property.

4. Para una entidad Property, marque "Identificador" si el Property es el identificador de la entidad.

5. Valide la propiedad description.

## Las principales operaciones que se pueden realizar en un Property
<a name="las_principales_operaciones_que_pueden_realizar_property_ELTTEXTE000157"></a>




### Borrar un Property
<a name="borrar_property_ELTPARAGRAPHE000044"></a>

Para borrar un Property:

1. Seleccione la propiedad a eliminar en la description de las propiedades de la entidad o de las propiedades de la asociación.

2. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeSupprimerPropriete.gif). El Property se borra automáticamente de la entidad o de la asociación.



<a name="NOTE3_2"></a>




### Renombrar un Property
<a name="renombrar_property_ELTPARAGRAPHE000057"></a>

Para renombrar a un Property:

1. Seleccione la propiedad a renombrar en la description de las propiedades de la entidad o de las propiedades de la asociación.

2. Introduce el nuevo nombre de Property y valida la modificación. El Property es renombrado automáticamente.

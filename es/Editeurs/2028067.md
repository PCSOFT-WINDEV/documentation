


## WDInst : generando el programa instalación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000224"></a>
La generación del programa instalación se utiliza para crear el programa instalación según las características especificadas en el editor instalación WDInst. Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

El programa instalación se generará en un directorio para ser copiado en diferentes medios.

Por Default, el programa instalación se genera en el subdirectorio "Install" de su aplicación.

Para generar el programa instalación, seleccione "Archivo .. Generar" de WDInst.

## Apoyo al programa instalación
<a name="apoyo_programa_instalacion_ELTTEXTE000248"></a>


### Apoyo al programa instalación
<a name="apoyo_programa_instalacion_ELTPARAGRAPHE000036"></a>

Los medios disponibles para el programa instalación son los siguientes:

- ***Directorio único***
	El programa instalación se crea en un solo directorio.

- ***Ejecutable autoextraíble***
	El programa instalación está creado como un ejecutable autoextraíble. Cuando este ejecutable se ejecuta en los ordenadores de los usuarios, el programa instalación se descomprime e inicia automáticamente.
	Esta solución se recomienda si el programa instalación se distribuye a través de INTERNET.

- ***Soporte con autoejecución***
	El programa instalación se crea en un solo directorio para ser copiado en un CD-ROM, DVD-ROM, llave USB, ... 
	En este caso, tienes la capacidad de introducir los parámetros para la ejecución de automatic. Para obtener más información, consulte [Incluyendo una ejecución automática](../Editeurs/2028015.md). 

- ***Múltiples directorios***
	Los archivos del programa instalación están agrupados en varios subdirectorios. El tamaño de los ficheros contenidos en cada subdirectorio no supera el tamaño especificado (1,44 MB por Default). Entonces, el contenido de cada subdirectorio puede ser copiado.




Para ayudarle a elegir el medio adaptado a su modo instalación, consulte el apartado [Resumen tabla para la instalación de una aplicación WINDEV](../Editeurs/2028005.md).

Al crear los medios de comunicación de instalación, puedes:

- **Vaciar el directorio antes de generar la instalación**: Si esta opción está marcada, el contenido del directorio donde se genera el programa instalación se borrará antes de generar un nuevo instalación. Esta opción le permite eliminar cualquier archivo inútil.

- **Proteger la instalación con una contraseña**: Esta opción le permite especificar la contraseña que se le pedirá al instalar la aplicación.

- **Firmar la instalación con el certificado**: Para garantizar la integridad del paquete de la instalación, tienes la posibilidad de firmar la instalación. Todo lo que tienes que hacer es seleccionar la Certificate para usar. Los certificados disponibles en el ordenador actual, en los certificados personales almacenados o en los certificados de Windows, se proponen automáticamente.. 




Observación: Para un instalación a través de un Tienda privada, sólo está disponible el "ejecutable autoextraíble"..

## Generando el programa instalación
<a name="generando_programa_instalacion_ELTTEXTE000272"></a>


### Generando el programa instalación
<a name="generando_programa_instalacion_ELTPARAGRAPHE000094"></a>

Para generar el programa instalación:

1. Seleccione "Archivo... Generar".

2. Seleccione los medios para el programa de instalación. Los diferentes medios de comunicación se presentan en el párrafo Above. Para ayudarle a elegir el medio adaptado a su modo instalación, consulte el apartado [Resumen tabla para la instalación de una aplicación WINDEV](../Editeurs/2028005.md).

3. Cada vez que se genera el programa instalación, el editor de instalación WDInst crea un archivo temporal ".WXF". Este archivo contiene la description del programa instalación. Este archivo puede utilizarse para ejecutar la prueba de un programa instalación personalizado ([proyecto personalizado "WDSetup](../Editeurs/2028037.md)). Para mantener este archivo, marque "Keep the WDX file for checking a custom WDSetup" (pestaña"Advanced") 
	Para obtener más información, consulte [Ficheros WXF y WDF](#NOTE4_1).

4. Seleccione las características del programa de instalación de acuerdo con el medio seleccionado de instalación: 
	

| Medio de instalación | Características |
| --- | --- |
| <br><br>	- Directorio único<br><br><br> | <br><br>	- Modificar (si es necesario) el directorio en el que se generará el programa instalación.<br><br><br> |
| <br><br>	- Ejecutable autoextraíble<br><br><br> | <br><br>	- Modificar (si es necesario) el directorio en el que se generará el ejecutable autoextraíble.<br><br>	- Introduzca el nombre del ejecutable a crear (sin extension).<br><br><br> |
| <br><br>	- Soporte con autoejecución<br><br><br> | <br><br>	- Modificar (si es necesario) el directorio en el que se generará el programa instalación.<br><br><br>Especifique (si es necesario) los parámetros para la ejecución de automatic:<br><br>	- Ícono del CD.<br><br>	- Acción realizada al insertar el CD. Para obtener más información, consulte [automatic ejecución de CD](../Editeurs/2028015.md).<br><br><br> |
| <br><br>	- Directorios múltiples<br><br><br> | <br><br>	- Modificar (si es necesario) el directorio base para generar el programa instalación.<br><br>	- Seleccione el tamaño máximo de todos los archivos que se encuentran en cada subdirectorio. Los subdirectorios se llamarán "DISCO &lt;Directorio#".<br><br><br> |



5. Validar la generación del programa instalación.

## Ficheros WXF y WDF
<a name="ficheros_wxf_wdf_ELTTEXTE000296"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000152"></a>

Cada vez que se genera el programa instalación, WDInst crea dos archivos que contienen el description del programa de desarrollo:

- **un archivo ".WDF**. Este archivo se utiliza para editar la description del programa instalación en WDInst. Por Default, este archivo se llama "&lt;NombreDeAplicación&gt;.WDF" y se guarda en el directorio de la aplicación en el ordenador de desarrollo.

- **un archivo ".WXF**. Este archivo se utiliza para comprobar un programa personalizado de instalación (proyecto personalizado "WDSetup"). Por Default, este archivo se comprime y se archiva en el archivo "INSTALL.WDZ", luego se borra de la computadora de generación.




Si está usando un programa personalizado de instalación (proyecto personalizado "WDSetup"), debe mantener el archivo ".WXF" para poder ejecutar la prueba de este programa personalizado de instalación. De hecho, el programa personalizado instalación recupera del archivo ".WXF" todas las opciones del programa instalación que han sido especificadas en WDInst (el editor de instalación).

En este caso, este archivo se llamará "&lt;ApplicationName&gt;.WXF" y se guardará en el directorio del proyecto a instalar.

**Observación**: Para obtener más información, consulte [Uso de una configuración WDS personalizada](../Editeurs/2028037.md).
<a name="NOTE4_2"></a>


### Guardar el archivo ".WXF
<a name="guardar_archivo_wxf_ELTPARAGRAPHE000171"></a>

Para conservar el archivo ".WXF", seleccione la opción "Guardar el archivo WXF para probar un WDSetup personalizado" al generar el programa de instalación (en WDInst, en la opción "Archivo .. Generar", vaya a la pestaña "Avanzado").

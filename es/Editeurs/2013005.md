
## Estadísticas de código
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
Las estadísticas de código afectan:

- el número total de líneas en la Process correspondiente ,

- la tasa de comentarios,

- el número de líneas por Process,

- el número de líneas de código con y sin comentarios,

- el número de líneas de comment.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000129"></a>


### Visualización de las estadísticas
<a name="visualizacion_las_estadisticas_ELTPARAGRAPHE000024"></a>

Estas estadísticas se pueden visualizar:

- **para el objeto actualmente editado en el editor de código (ventana, reporte, ....)**
	Las estadísticas se refieren a los diferentes procesos del elemento actual y sus controles. 
	Utilice "Código .... Estadísticas ... Objeto". Aparece la ventana de configuración. Haga clic en la tecla "Indicación" para calcular las estadísticas.

- **para el proyecto actualmente editado en el editor de código**. 
	Las estadísticas visualizadas se refieren a todo el proyecto: cada ventana, reporte, Procedure global ... se analiza. 
	Usar "Código .. Estadísticas ... Proyecto". Aparece la ventana de configuración. Haga clic en la tecla "Indicación" para calcular las estadísticas.




En la ventana para configurar las estadísticas, especifique:

- el porcentaje máximo de comentarios a tener en cuenta.

- el número mínimo de líneas de código a tener en cuenta.




Observación: Para acceder a estas opciones, utilice el menú principal accesible a través del logotipo de la versión: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_logo_menu17.gif)

### Impresión de las estadísticas
<a name="impresion_las_estadisticas_ELTPARAGRAPHE000052"></a>

El código estadístico se imprime desde la ventana de configuración de las estadísticas.. Se imprimen los siguientes elementos:

- el número total de líneas en la Process correspondiente ,

- la tasa de comentarios,

- el número de líneas por Process.

## Mostrar/Ocultar eventos vacíos de WLanguage
<a name="mostrarocultar_eventos_vacios_wlanguage_ELTTEXTE000197"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000028"></a>

El editor de código permite visualizar el código de una selección múltiple (controles, procedimientos globales, métodos de clase, etc.). Cada elemento tiene un número de eventos de WLanguage.

Mostrar todos estos eventos puede afectar a la legibilidad, a pesar de que la mayoría de ellos están vacíos.

Por Default, los eventos vacíos no se muestran.
<a name="NOTE2_2"></a>


### Mostrar eventos vacíos de WLanguage
<a name="mostrar_eventos_vacios_wlanguage_ELTPARAGRAPHE000039"></a>

Para mostrar los eventos vacíos de WLanguage:

1. Vaya al editor de código.

2. En la pestaña "Vista", en el grupo "Ayuda edición", marque o desmarque "Mostrar procesos vacíos". Si ya se muestran eventos vacíos de WLanguage, se ocultan automáticamente (y viceversa).




También puedes pulsar Ctrl + F6.

## Mostrar/ocultar los eventos WLanguage de las plantillas anuladas
<a name="mostrarocultar_los_eventos_wlanguage_las_plantillas_anuladas_ELTTEXTE000227"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000065"></a>

Si el elemento que se muestra en el editor de código es associated con un modelo, puede mostrar los eventos WLanguage de las plantillas anuladas.
<a name="NOTE3_2"></a>


### Visualización de los eventos WLanguage de las plantillas anuladas
<a name="visualizacion_los_eventos_wlanguage_las_plantillas_anuladas_ELTPARAGRAPHE000072"></a>

Para mostrar los eventos WLanguage de las plantillas anuladas:

1. Vaya al editor de código.

2. En la pestaña "Vista", en el grupo "Ayuda edición", marque o desmarque "Mostrar procesos de los modelos". Si estos eventos ya se muestran, se ocultan automáticamente (y viceversa).




También es posible utilizar la combinación de teclas Mayúsculas + F6.

## Mostrando/ocultando los números de Line
<a name="mostrandoocultando_los_numeros_line_ELTTEXTE000257"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000098"></a>

El editor de códigos le permite mostrar un número de Line delante de cada código Line.

Los números de línea no son mostrados por Default.
<a name="NOTE4_2"></a>


### Mostrando los números de Line
<a name="mostrando_los_numeros_line_ELTPARAGRAPHE000107"></a>

Para mostrar los números de Line:

1. Vaya al editor de código.

2. En la pestaña "Vista", en el grupo "Ayuda edición", haga clic en "Mostrar números de línea". Si los números de Line ya aparecen, se ocultan automáticamente (y viceversa).




También es posible utilizar la combinación de teclas Ctrl + Mayúsculas + G.




## Creación de un ejecutable: Opciones generales 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000170"></a>
Las principales opciones de ejecución son las siguientes:

- **el nombre del ejecutable** (archivo".EXE"). El nombre "&lt;Nombre del proyecto&gt;.EXE" es propuesto por Default.

- **el Icon del programa ejecutable** (archivo "ICO"). Este Icon puede ser elegido en el catálogo Icon de WINDEV (botón "Catálogo"').

- **la pantalla de bienvenida del programa ejecutable**. Esta Image se mostrará en tiempo de ejecución cuando se inicialice la aplicación y cuando se carguen las diferentes bibliotecas WINDEV/WINDEV Mobile.




Estas opciones generales se definen en la creación ejecutable asistente.

## Icon del programa ejecutable
<a name="icon_del_programa_ejecutable_ELTTEXTE000194"></a>
Para asociar un Icon con el programa ejecutable, puedes:

- usa un Icon que se encuentra en el catálogo Icon de WINDEV.

- elige un archivo Icon (.ICO) en tu ordenador.


Esta elección se realiza a través del botón desplegable de "Nombre del icono" control:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0002%202.gif)


Cuando elija un archivo Icon (.ICO) en su computadora, asegúrese de eso:

- el archivo del icono es un archivo en formato ICO de 16-Color (formato de Windows 3.x). Estos iconos no tienen características específicas.

- el archivo Icon es un archivo en formato 256 ICO (formato Win32, usado en Windows 98, Me y 2000). Estos iconos no tienen características específicas.

- el archivo de iconos es un archivo que utiliza un **grupo de iconos** de diferentes tamaños y diferentes resoluciones Color. Los grupos de iconos (también llamados iconos en formato Windows XP) deben cumplir con el siguiente estándar:

	- los iconos deben haber sido creados por una herramienta que organiza los iconos aumentando el tamaño y aumentando la resolución de Color.

### Pantalla splash de una aplicación WINDEV
<a name="pantalla_splash_una_aplicacion_windev_ELTPARAGRAPHE000072"></a>

La "pantalla de inicio" se mostrará en tiempo de ejecución cuando se inicialice la aplicación y cuando se carguen las bibliotecas WINDEV. Al crear el ejecutable, puede utilizar:

- a **pantalla de inicio estándar**: se mostrará una imagen durante la inicialización. Este Image puede ser elegido en el catálogo Image suministrado con WINDEV.
	Los formatos de Image soportados son:

	- Mapa de bits (\*.BMP)

	- Formato de intercambio de gráficos (\*.gif)

	- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

	- Pincel de pintura (\*.pcx)

	- Formato Adobe Photoshop (\*.psd)

	- TrueVision TARGA (\*.tga)

	- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
			Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

	- Gráficos de red portátiles (\*.png)

	- MetaFicheros de Windows (\*.emf;\*.wmf)

	- Iconos (\*.ico \*;.icw)

	- Cursores (\*.cur)


 Observación: Si la imagen no está en formato BMP, la librería wdxxxpnt.dll debe ser associated con el programa ejecutable.
	Este tipo de pantalla de salpicadura se recomienda si la inicialización Process de la aplicación es bastante larga.
	Es posible:

- ***configurar la información que se muestra automáticamente*** en este Image (botón "Avanzado"). Esta información es la siguiente:

	- *Número de versión del ejecutable*: muestra (o no) el número de versión del ejecutable. Puede configurar el tamaño, la posición, la fuente utilizada para este texto.

	- *Título de la solicitud*: muestra (o no) el título de la aplicación. Por Default, este título corresponde al nombre del proyecto. Puede configurar el tamaño, la posición, la fuente utilizada para este texto.

	- *Mensaje visualizado*: se utiliza para configurar las características de los mensajes mostrados a través de la programación. Estos mensajes pueden mostrarse mediante [Mensaje](../WDLang1/3021006.md) en el código de inicialización del proyecto. Puede, por ejemplo, mostrar un mensaje que indique los diferentes pasos de inicialización.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SplashImage.gif&type=thumb)

- a **pantalla de bienvenida con texto animado**: el texto especificado (hasta 20 caracteres) se mostrará en una animación. Para ver la animación, haga clic en el botón "Mostrar la animación. Esta animación se puede configurar a través del botón "Opciones. Es posible:

	- elegir el fondo Color de la pantalla de salpicadura.

	- mostrar la pantalla de bienvenida a pantalla completa o no.

	- mostrar la pantalla de bienvenida durante el primer inicio de la aplicación en el día.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SplashAnimation.gif)





- **sin pantalla de bienvenida**: no se mostrará ninguna Image durante la inicialización.




**Observación**: El uso de una pantalla de bienvenida puede aumentar significativamente el tamaño del ejecutable.. Por ejemplo, el uso de una pantalla de bienvenida con texto animado puede aumentar el tamaño del ejecutable en 400 KB.

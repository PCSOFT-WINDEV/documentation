
## Espaciar una selección de controles
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
Para crear interfaces modernas, puede aumentar el espacio entre los controles de las ventanas. Con WINDEV y WINDEV Mobile, solo necesita indicar el espaciado deseado y los controles se desplazarán automáticamente.  

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000128"></a>
Para aumentar el espacio entre los controles: 

1. Seleccione los controles deseados. 

2. En la pestaña "Alineación", en el grupo "Otras alineaciones", despliegue "Ampliar
Espaciar" y seleccione "Espaciar la selección". 

3. En la ventana que aparece, defina la configuración para aumentar el espacio entre los controles seleccionados: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aerer_la_selection%20-%20HC%20N%B0003.gif)
 Puede: 

	- Especificar el valor del espaciado horizontal y/o vertical. El ícono <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aerer_la_selection%20-%20HC%20N%B0003%201.gif)
 permite cambiar ambos valores al mismo tiempo.  

	- Especificar la tolerancia deseada, si es necesario. La tolerancia permite detectar con precisión los controles que deben espaciarse, aunque se superpongan o no estén bien alineados. 


 La opción <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aerer_la_selection%20-%20HC%20N%B0003%202.gif)
 permite ver los cambios en tiempo real en la ventana. 

4. Valide. Los ajustes se aplican a los controles seleccionados. 




**Observaciones**: 

- Se espacio se agrega a la derecha y abajo. 

- Atención: La ventana no cambia de tamaño.

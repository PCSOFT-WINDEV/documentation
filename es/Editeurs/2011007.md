## Manejando a un archivo de datos
<a name="manejando_archivo_datos_ELTTEXTE000213"></a>


### Duplicación/Copia un archivo de datos
<a name="duplicacioncopia_archivo_datos_ELTPARAGRAPHE000034"></a>

Duplicando a un archivo de datos se usan para get dos archivos de datos idénticos con nombres diferentes.

Para duplicar/copy un archivo de datos:

1. Selecciona el archivo de datos para duplicar.

2. En la pestaña "Inicio", en el grupo "Edición", haga clic en "Duplicar". 
	Observación: También tiene la posibilidad de seleccionar "Duplicar" en el menú contextual del archivo. 

3. Especifique la información solicitada y valide. El archivo de datos duplicado se crea automáticamente en el análisis actual (LDM).



<a name="NOTE2_2"></a>


### Borrar un archivo de datos
<a name="borrar_archivo_datos_ELTPARAGRAPHE000061"></a>

Para eliminar a un archivo de datos:

1. Selecciona el archivo de datos para borrar.

2. Haga clic con el botón derecho del ratón para abrir el menú contextual del archivo de datos.

3. Seleccione"Eliminar".. El archivo de datos se borra automáticamente del análisis (LDM). Si los enlaces apuntaban a este archivo de datos, estos enlaces también se eliminan.




**Observación**: Al crear un archivo de datos cuyo nombre es idéntico al nombre de un archivo de datos que fue borrado anteriormente, WINDEV, WEBDEV y WINDEV Mobile proponen restaurar el description del archivo de datos borrado.
<a name="NOTE2_3"></a>


### Cambio de nombre de un archivo de datos
<a name="cambio_nombre_archivo_datos_ELTPARAGRAPHE000074"></a>

Para cambiar el nombre de un archivo de datos:

1. Selecciona el archivo de datos para cambiar el nombre.

2. Abra el archivo description: 

	- En la pestaña "Análisis", en el grupo "Elemento actual", haga clic en "Descripción".

	- Abrir el menú contextual del archivo (clic derecho) y seleccionar "description del archivo de datos". 




3. En la ventana de description, modifique el nombre del archivo de datos (y su nombre en el disco si es necesario). 

4. Validar. El archivo de datos se renombra automáticamente.



<a name="NOTE2_4"></a>


### Utilizar un diccionario
<a name="utilizar_diccionario_ELTPARAGRAPHE000101"></a>

Se puede incluir un archivo de datos en un diccionario para ser reutilizado en una nueva aplicación, por ejemplo. El archivo de datos se incluye en un diccionario simplemente arrastrándolo desde el editor de análisis y soltándolo en el diccionario (en el ["Panel"Diccionario](../Editeurs/2027014.md)).

Cuando se incluye en el diccionario, el description del archivo de datos puede suscribirse a las modificaciones. Cualquier modificación hecha en la description del archivo de datos se aplicará al objeto de diccionario y a todos los usos de este objeto.

Para obtener más información, consulte [diccionario](../Editeurs/2023002.md).

### La creación del correspondiente archivo de datos vacío
<a name="creacion_del_correspondiente_archivo_datos_vacio_ELTPARAGRAPHE000118"></a>

Para crear el correspondiente archivo de datos físicos vacío:

1. Seleccione el archivo de datos para el cual debe crearse un archivo físico.

2. Abra el menú contextual del archivo (clic derecho) y seleccione "Crear archivo de datos".

3. Selecciona el directorio donde se creará y valida el archivo de datos.




Los siguientes archivos se crean automáticamente en el directorio especificado:

- "&lt;Archivo de datos&gt;.FIC" (archivo de datos).

- "&lt;Nombre del archivo de datos&gt;.MMO" (archivo de notas).

- "&lt;Nombre del archivo de datos&gt;.NDX" (archivo de índice).




Si el archivo de datos se registra, se creará automáticamente un subdirectorio "JNL" que contendrá los diferentes archivos registrados.

Si el archivo de datos contiene un índice de texto completo, se creará un archivo "&lt;Nombre del archivo&gt;.FTX".. 

**Observaciones**:

- Esta opción equivale a llamar a [HCreationIfNotFound](../WDLang4/3044254.md).

- Esta opción será ignorada si el archivo de datos ya existe en el directorio especificado.

## Presentación
<a name="presentacion_ELTTEXTE000162"></a>
Cada control Tabla que se encuentra en una ventana propone un menú al usuario.

Este menú, disponible sin programación, incluye varias acciones:

- Exportación a Word o a OpenOffice (dependiendo del software instalado en el ordenador del usuario).

- Exportación a Excel o a OpenOffice Calc (dependiendo del software instalado en el ordenador del usuario).

- Exportar a XML.

- copy la fila o la tabla.

- Cree un gráfico para representar los datos.

- Imprimir la tabla (ver [AAF para la impresión de un tabla](../WDChamp/9500132.md)). 

- Seleccione varias columnas tabla:




Para simplificar el menú, sólo las primeras 32 columnas son Visible en el menú. Además, sólo se proponen las columnas con un título.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Menu.gif)
 
Ejemplo de gráfico generado automáticamente por el menú contextual de un control Tabla:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Livre_Table_Graphe%20-%20HC%20N%B0002.gif)
 Este menú permite al usuario realizar fácilmente varios procesos no necesariamente previstos en la aplicación:

- Exportación a un archivo externo de Excel, Word o XML.

- Posibilidad de mostrar u ocultar las columnas (para limitar el tamaño del menú, sólo se muestran las primeras 32 columnas).

- Generar automáticamente un gráfico a partir de los datos encontrados en un tabla. 
	Observación: no puede seleccionar las columnas que se incluirán en el gráfico. Para seleccionar algunas columnas específicas de tabla, le aconsejamos que cree un nuevo gráfico a partir del control Tabla que aparece en una ventana.




Observaciones:

- Puede agregar opciones a este menú mediante programación. Para ello, utilice la propiedad [ContextMenu](../Proprietes/2510077.md).

- Al exportar una tabla mediante el menú contextual, se ignora la fusión de columnas.

- Tabla horizontal: El menú de AAFs no contiene las opciones utilizadas para realizar una exportación a un archivo externo de Excel, Word o XML.

### Configurando el menú de automatic en un control Tabla
<a name="configurando_menu_automatic_control_tabla_ELTPARAGRAPHE000053"></a>

El menú contextual de una control Tabla puede desactivarse. Para configurar la visualización del menú contextual de control Tabla, debe:

1. Abra la ventana de descripción del control Tabla.

2. Seleccione la pestaña "UI".

3. Elija el menú deseado en la opción "Menú contex.". Usted puede: 

	- "Mostrar el menú de las FAA (Sistema)": En este caso, el menú contextual que aparecerá será el menú control Tabla automatic (Menú de las AAF). Para obtener más información sobre cómo configurar este menú, consulte [Configuración del menú AAF](../Editeurs/2010040.md).

	- "Agregar un menú contextual": Si esta opción está seleccionada, puede seleccionar el menú contextual personalizado que se mostrará. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Menu2.gif)






Observaciones: 

- Si ambas opciones están seleccionadas, el menú contextual personalizado puede agregarse antes o después del menú de las AAF. 

- Si no se selecciona ninguna opción, no se seleccionará ningún menú contextual. La opción &lt;Desactivado&gt; se mostrará en la ventana de descripción. 

- La propiedad [ContextMenu](../Proprietes/2510077.md) también permite eliminar el menú contextual específico de las AAF y restaurar el menú estándar de Windows para un control determinado.

### Configurando la opción "Seleccionar columnas...".
<a name="configurando_opcion_seleccionar_columnas_ELTPARAGRAPHE000095"></a>

En algunos casos, es posible que no desee ofrecer al usuario la posibilidad de ocultar las columnas.. Para ello, se puede activar o desactivar la opción "Seleccionar columnas...":

1. Abra la ventana de descripción del control Tabla.

2. Seleccione la pestaña "Detalles".

3. Marque o desmarque el menú "'Seleccionar columnas...'".

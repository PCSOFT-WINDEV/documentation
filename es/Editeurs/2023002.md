## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
El diccionario es un archivo que puede contener los elementos del proyecto: description de archivos de datos, artículos, ventanas, etc.. El diccionario es una forma conveniente de almacenar los elementos creados para que puedan ser reutilizados en nuevos proyectos.

Cuando se reutiliza un elemento del diccionario, este elemento puede "**suscribirse**"a las modificaciones. Cada vez que se modifica el objeto encontrado en el diccionario, esta modificación puede aplicarse a todos los objetos del suscriptor que pueden "seguir" la evolución del diccionario.

Un diccionario puede ser associated con un usuario: se llama el **diccionario del usuario**. En este caso, puede visualizarse de forma permanente en el panel "Dictionary".. 

**También se puede utilizar un diccionario para simplificar el trabajo en equipo.**: un diccionario compartido entre varios desarrolladores puede contener procedimientos, botones de preselección. Estos objetos son compartidos y pueden ser accedidos por todos los desarrolladores.

La gestión del groupware de desarrolladores en el diccionario y en sus elementos se utiliza para restringir el acceso al diccionario (o a sus objetos) para algunos desarrolladores.

## Principio
<a name="principio_ELTTEXTE000137"></a>
El diccionario es un archivo que contiene una set de elementos: procedimientos, ventanas, controles, ... Un diccionario se puede manejar desde el panel "Diccionario".

Cuando manejas un diccionario, tienes la habilidad de hacerlo:

- Crear un nuevo diccionario o abrir uno existente.

- Añadir elementos (ventanas, informes,...) al diccionario.

- Elementos de copy del diccionario.




Para obtener más información, consulte [Manejo de un diccionario](../Editeurs/2023003.md).

Cuando se añade un elemento mediante arrastrar y soltar desde el diccionario o al diccionario, se propone una suscripción. Si la suscripción es aceptada:

- Cada vez que se abre el elemento de proyecto, WINDEV, WEBDEV y WINDEV Mobile comprueban si este elemento está actualizado en relación con el elemento que se encuentra en el diccionario y proponen actualizarlo si es necesario.

- Cada vez que se cierra un elemento modificado, WINDEV, WEBDEV y WINDEV Mobile proponen actualizar el objeto encontrado en el diccionario.




El concepto de suscripción se utiliza para aplicar fácilmente las modificaciones realizadas a un objeto de abonado a todos los demás objetos de abonado.. Para obtener más información, consulte [Suscribir un elemento al diccionario](../Editeurs/2023001.md).

## Compatibilidad con WINDEV 5.5
<a name="compatibilidad_con_windev_55_ELTTEXTE000161"></a>


### Atención
<a name="atencion_ELTPARAGRAPHE000057"></a>

**Para migrar un diccionario 5.5 existente**le aconsejamos que lo haga:

- Cree una ventana en blanco.

- Añada todos los elementos del diccionario en la ventana.

- Migrar la ventana.

- Vuelva a crear el diccionario en WINDEV 28.

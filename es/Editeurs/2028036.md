## ¿Cómo dar al usuario la posibilidad de iniciar la instalación en varios idiomas?
<a name="¿como_dar_usuario_posibilidad_iniciar_instalacion_varios_idiomas_ELTTEXTE000142"></a>
Todo lo que tienes que hacer es seleccionar los diferentes idiomas que se propondrán al usuario al iniciar el programa instalación.

El programa instalación se ejecutará en el idioma seleccionado por el usuario.

Sólo el francés y el inglés son gestionados automáticamente por WINDEV.. Para que el programa instalación soporte idiomas adicionales, debe personalizar el proyecto "WDSetup" antes de crear el programa instalación. Este proyecto (suministrado con los ejemplos del WINDEV) corresponde al programa estándar instalación de las aplicaciones del WINDEV.

Para obtener más información, consulte [WDSetup: gestión de idiomas adicionales](../Editeurs/2028037.md).

**Para todos los idiomas seleccionados, tendrá la posibilidad de especificar**:

- una licencia, si una licencia se muestra al principio de la aplicación instalación.

- un archivo "Léame", si un archivo "Léame" se muestra al final de la aplicación instalación.

- una interfaz del programa instalación específica para cada idioma.




**Aplicación instalada en la red**
Al actualizar una aplicación instalada en la red, se muestra una ventana cuando se inicia la aplicación desde un ordenador de usuario final. Esta ventana informa al usuario de que existe una versión actualizada de la aplicación.. Esta ventana permite al usuario instalar (o no) la actualización. Esta ventana también se encuentra en el proyecto WDSetup.

## Inicio de la aplicación multilingüe
<a name="inicio_aplicacion_multilingue_ELTTEXTE000190"></a>
Por defecto, una aplicación multilingüe se ejecuta en el idioma principal del proyecto ("Idioma principal" en la pestaña "Idiomas" del proyecto description).

Para iniciar la aplicación en el lenguaje de ejecución del programa instalación:

1. Recuperar el lenguaje de ejecución del programa instalación y guardar este lenguaje en un archivo ".INI" o en el registro del ordenador de la instalación. Esta operación puede realizarse automáticamente durante la instalación. Para obtener más información, consulte [Gestionar el registro y los archivos ".INI" durante instalación](../Editeurs/2028047.md).

2. Utilice [Nación](../WDLang1/3054014.md) para cambiar el idioma que utiliza la aplicación en tiempo de ejecución.




Para obtener más información, consulte [Desarrollo de aplicaciones multilingües](../WDLang1/3054008.md).

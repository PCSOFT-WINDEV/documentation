


## WDInst: gestión de los archivos a instalar 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000295"></a>
El editor de instalación WDInst se utiliza para gestionar los archivos a instalar:

- [Ver la lista de archivos](#NOTE2_1),

- [Agregar uno o más archivos](#NOTE3_1),

- [Eliminar uno o más archivos](#NOTE4_1).




Los archivos a instalar se gestionan en el editor de instalación WDInst. Para abrir WDInst, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

A diferencia de la asistente para la creación del programa instalación, WDInst propone una gestión avanzada de los archivos a instalar.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualización de la lista de archivos a instalar
<a name="visualizacion_lista_archivos_instalar_ELTTEXTE000319"></a>
Los ficheros a instalar pueden consultarse en la ventana "Lista de ficheros instalación" de WDInst.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Parametrer_caract_installation_fichier%20-%20HC%20N%B0001%201.gif&type=thumb)

<a name="NOTE2_2"></a>
En esta lista se muestran dos tipos de archivos:

- ***archivos individuales***. El archivo se define por su nombre completo (nombre y extension) en el ordenador de desarrollo. Este archivo se incluirá en el programa instalación (si se encuentra en el directorio de origen durante la generación).

- ***Archivos genéricos***. El nombre del archivo contiene caracteres comodín (\*,?). Todos los archivos encontrados en el directorio de origen y correspondientes a este filtro se incluirán en el programa instalación durante la generación.




Por ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TypeFichier.gif)
 Para ver los archivos asociados con un [grupo de archivos](../Editeurs/2028010.md) , seleccione el grupo deseado en el cuadro combinado "Grupo mostrado".
<a name="NOTE2_3"></a>


### Ordenar la lista de archivos a instalar
<a name="ordenar_lista_archivos_instalar_ELTPARAGRAPHE000065"></a>

Para ordenar la lista de archivos a instalar, haga clic en el encabezado de la columna correspondiente al criterio de ordenación (nombre del archivo, directorio de origen, directorio de destino, ....).
<a name="NOTE2_4"></a>


### Modificar el directorio de destino de un archivo que se encuentra en la lista de archivos a instalar
<a name="modificar_directorio_destino_archivo_que_encuentra_lista_archivos_instalar_ELTPARAGRAPHE000072"></a>

El directorio de destino corresponde al directorio en el que se instalará el archivo.

Para definir el directorio de destino, se pueden utilizar las siguientes palabras clave:

- ****: Directorio de instalación elegido por el usuario durante la instalación.

- ****: Drive del directorio instalación elegido por el usuario durante el proceso instalación.

- ****: El directorio de archivos de datos de instalación. WDInst se usa para configurar este directorio. Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

- ****: Directorio Windows de ordenadores de usuario

- ****: directorio del sistema.

- ****: directorio de referencia para las configuraciones con actualización de automatic solamente.




Por ejemplo: "\\MiDirectorio".

Para modificar el directorio de destino de un archivo que se encuentra en la lista de archivos a instalar:

1. Seleccione el archivo deseado.

2. Haga clic en la columna "Directorio de destino".

3. Introduzca el nuevo directorio de destino.

## Instalación de uno o más archivos adicionales
<a name="instalacion_uno_mas_archivos_adicionales_ELTTEXTE000367"></a>


### Por Default , el instalación selecciona automáticamente todos los archivos que se encuentran en el directorio "EXE" del proyecto.
<a name="por_default_instalacion_selecciona_automaticamente_todos_los_archivos_que_encuentran_directorio_exe_del_proyecto_ELTPARAGRAPHE000135"></a>

El editor de instalación WDInst le permite añadir:

- Ficheros individuales.

- Ficheros genéricos.




Este tipo de archivos se han presentado en el párrafo anterior.
<a name="NOTE3_2"></a>


### Añadir uno o más ficheros individuales
<a name="anadir_uno_mas_ficheros_individuales_ELTPARAGRAPHE000147"></a>

Para añadir uno o más archivos individuales:

1. Seleccione "Editar .. Añadir archivos".. Se abre la ventana para añadir archivos.

2. Seleccione la Drive y el directorio donde se encontrarán los ficheros a añadir. Se muestra la lista de los archivos encontrados en el directorio.

3. Para filtrar los archivos que se muestran según un criterio de búsqueda, introduzca el criterio de búsqueda en la opción "Filtro" (por ejemplo, "\*.DLL" o "\*cliente\*") y haga clic en el botón "Filtrar". 
	**Observación**: Se permiten los caracteres comodín (\*,?).

4. Marque "uno o más archivos".

5. Seleccione los archivos que se agregarán. 
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

6. Especifique el directorio de destino de los archivos a instalar.
	Para definir el directorio de destino, se pueden utilizar las siguientes palabras clave:

	- ****: Directorio de instalación elegido por el usuario durante la instalación.

	- ****: Drive del directorio instalación elegido por el usuario durante el proceso instalación.

	- ****: El directorio de archivos de datos de instalación. WDInst se usa para configurar este directorio. Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

	- ****: Directorio Windows de ordenadores de usuario

	- ****: directorio del sistema.

	- ****: directorio de referencia para las configuraciones con actualización de automatic solamente.
			Por ejemplo: "\\MiDirectorio".




7. Para asociar los ficheros a instalar a una [grupo de archivos](../Editeurs/2028010.md), haga clic en el botón "Otros" y especifique el grupo de ficheros a asociar.

8. Validar la adición de archivos.


<a name="NOTE3_3"></a>


### Añadir uno o más ficheros correspondientes a un fichero genérico
<a name="anadir_uno_mas_ficheros_correspondientes_fichero_generico_ELTPARAGRAPHE000198"></a>

Para añadir uno o más ficheros correspondientes a un fichero genérico:

1. Seleccione "Editar .. Añadir archivos".. Se abre la ventana para añadir archivos.

2. Seleccione la Drive y el directorio donde se encontrarán los ficheros a añadir.

3. Marque "un archivo genérico".

4. Introduzca el archivo genérico ("\*.BMP" por ejemplo).
	**Observación**: Se permiten los caracteres comodín (\*,?).

5. Para seleccionar los ficheros situados en los subdirectorios del directorio especificado, marque "Añadir recursivamente todos los ficheros correspondientes".

6. Especifique el directorio de destino de los archivos a instalar.
	Para definir el directorio de destino, se pueden utilizar las siguientes palabras clave:

	- ****: Directorio de instalación elegido por el usuario durante la instalación.

	- ****: Drive del directorio instalación elegido por el usuario durante el proceso instalación.

	- ****: El directorio de archivos de datos de instalación. WDInst se usa para configurar este directorio. Para obtener más información, consulte [Configuración del directorio de archivos de datos](../Editeurs/2028017.md).

	- ****: directorio de Windows en los ordenadores de los usuarios.
			Por ejemplo: "\\MiDirectorio".




7. Para asociar los ficheros a instalar a una [grupo de archivos](../Editeurs/2028010.md), haga clic en el botón "Otros" y especifique el grupo de ficheros a asociar.

8. Validar la adición de archivos.

## Eliminar uno o más archivos para instalar
<a name="eliminar_uno_mas_archivos_para_instalar_ELTTEXTE000403"></a>
Para eliminar uno o más archivos a instalar:

1. Seleccione los archivos que desea eliminar en la ventana "Lista de archivos instalación"..
	Para seleccionar varios archivos simultáneamente, mantenga pulsada la tecla Ctrl o Mayúsculas al seleccionar archivos.

2. Seleccione "Editar .. Borrar" (o pulse la tecla Supr).

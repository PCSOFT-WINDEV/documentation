### Configuración necesaria para la instalación de WINDEV
<a name="configuracion_necesaria_para_instalacion_windev_ELTPARAGRAPHE000022"></a>

Distinguimos entre la configuración del ordenador de desarrollo (su ordenador) y la configuración necesaria para que funcionen las aplicaciones desarrolladas por WINDEV (los ordenadores del usuario final).

**Computadora de desarrollo**:

- Pantalla: tamaño mínimo: 1920 x 1080

- 4 GB de RAM (se recomiendan 8 GB o más).

- 4 GB de espacio en el disco (10 GB para una instalación completa : el software incluye cientos de ejemplos).

- con Windows Vista o 7 (32 o 64 bits) o posterior. 

- Atención: Para instalar WINDEV, debe estar conectado como administrador del ordenador. 




**Computadora del usuario**:

- 2 GB de RAM.

- con Windows Vista o 7 (32 o 64 bits) o posterior. Se requiere un procesador SSE2.




Observará que el código fuente de WINDEV es idéntico independientemente de la versión de Windows en la que se ejecuten los programas.

Observación: Cuando el ordenador de desarrollo tiene una pantalla de alta resolución, el tamaño de la fuente en los editores WINDEV puede ser demasiado pequeño para ser legible. Este es el caso de una retina de Macbook en 2.560 x 1.600 o de los ordenadores Windows en 3.000 x 1.800.. Para aumentar el tamaño de todas las fuentes: 

1. Abra las opciones generales de WINDEV (pestaña "Inicio", grupo "Entorno", "Opciones"). 

2. Vaya a la pestaña "General. 

3. En "Gestión del DPI", seleccione "Dejar que Windows gestione los PPP".

4. Los ajustes de DPI se aplicarán la próxima vez que se inicie WINDEV Mobile.


Para obtener más información, consulte [Opciones WINDEV](../Editeurs/2036001.md).

## Instalación
<a name="instalacion_ELTTEXTE000427"></a>
La instalación es muy fácil, simplemente siga las instrucciones incluidas en el paquete de la instalación.

Puede instalar WEBDEV y WINDEV Mobile por separado, o integrarlos en WINDEV si ya está instalado. Hasta la versión 25, esta integración sólo se permite en modo de 64 bits. A partir de la versión 26, puede integrar WEBDEV y WINDEV Mobile en WINDEV en modo de 64 y 32 bits.

Para facilitar las cosas, instalación Procedure crea automáticamente el grupo de programas "WINDEV", "WEBDEV" o "WINDEV Mobile" en "Inicio ... Programas" e instala los archivos necesarios.

El WINDEV, WEBDEV y WINDEV Mobile instalación asistente propone ayudar a mejorar el producto permitiendo la recolección de información sobre cómo se utiliza el producto. Esta recopilación de datos opcional y anónima permite que PC SOFT mejore sus productos.

## Dongle
<a name="dongle_ELTTEXTE000481"></a>
El dongle suministrado con su producto es específico para el entorno de desarrollo.

Este dongle no es necesario para los usuarios finales o para la empresa de alojamiento de sus sitios INTERNET.

El principio es simple, el dongle debe estar enchufado cuando se utiliza el producto.

Hay dos tipos de llaves disponibles, la llave que va en el puerto paralelo de su ordenador (la más común) o la llave USB.

Para asegurarse de que la chaveta de ajuste funciona correctamente, es necesario:

- Inserte la llave en el puerto paralelo de su ordenador cuando esté apagado.

- conecte su impresora al dongle

- enciende la el equipo y la impresora

- iniciar un trabajo de impresión




Si la Document se imprime, su dongle está bien conectado.

Para la llave USB, conéctela a un puerto USB disponible e inicie WINDEV, WEBDEV o WINDEV Mobile.

**Observación**: Si el dongle está apagado o parpadea, consulte las siguientes preguntas frecuentes: [Instalar el controlador necesario para reconocer el dongle de WINDEV, WEBDEV o WINDEV MOBILE](https://faq.pcsoft.fr/faqread.awp?idfaq=3519&lang=es).

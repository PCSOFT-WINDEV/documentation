


## Groupware Usuario y archivos de datos Cliente/Servidor
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
Una aplicación que utiliza el Groupware Usuario puede ser utilizada con una base de datos HFSQL en modo Client/Server.

Para que una aplicación que utiliza el Groupware Usuario funcione en modo Client/Server:

- convierta la aplicación existente a Client/Server. Para obtener más información, consulte [Client/Server](../WDLang4/3044306.md).

- configure el Groupware Usuario para utilizar los archivos de datos que se encuentran en el servidor. Esta página de ayuda muestra cómo hacerlo.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000161"></a>


### Modo automático
<a name="modo_automatico_ELTPARAGRAPHE000026"></a>

Para convertir los archivos de datos del Groupware Usuario al modo Client/Server: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Groupware Usuario". 

2. En la ventana de descripción de la configuración del proyecto actual, en la sección "Groupware Usuario: Desactivado", haga clic en "Cambiar". 

3. En la ventana que aparece, active el Groupware Usuario.

4. Seleccione el "Groupware Usuario estándar". 

5. En la pestaña "Archivos" seleccione la opción "HFSQL Client/Server".

6. Haga clic en la opción "Haga clic aquí para introducir los parámetros del servidor". Especifique:

	- el nombre del servidor.

	- el número de puerto.

	- el nombre de usuario utilizado para la base de datos.

	- la contraseña asociada.

	- el nombre de la base de datos.

	- los directorios de los archivos de datos.




7. Si hay archivos de datos específicos del Groupware Usuario, cópielos en el servidor.



<a name="NOTE2_2"></a>


### Modo manual
<a name="modo_manual_ELTPARAGRAPHE000060"></a>

Para convertir los archivos de datos del Groupware Usuario al modo Client/Server: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Groupware Usuario".

2. En la ventana de descripción de la configuración actual del proyecto, en la opción "Groupware Usuario", haga clic en "Cambiar". 

3. En la pestaña "General", seleccione "Inicio manual (mediante programación)". 
	El Groupware Usuario no debe utilizar el modo "Inicio automático" en su aplicación.

4. Seleccione las diferentes opciones del Groupware Usuario y valide la ventana. 

5. En el código de inicialización del proyecto, utilice la función [gpwOpenConnection](../WDLang6/3041007.md) para definir la conexión a la base de datos HFSQL Client/Server.

6. Si hay archivos de datos específicos del Groupware Usuario, cópielos en el servidor.

7. Utilice la función [gpwOpen](../WDLang6/3041002.md) para especificar la ubicación de los archivos de datos. El directorio especificado debe ser relativo al directorio del servidor.

## Acciones aplicadas a los elementos de la consulta
<a name="acciones_aplicadas_los_elementos_consulta_ELTTEXTE000135"></a>
Esta sección de la ventana de description de una consulta se utiliza para definir las diferentes acciones que se aplicarán a los elementos de la consulta.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DescrReqPartie3.gif)



|   |   |
| --- | --- |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.GIF) | Elemento al que se aplicarán las acciones. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.GIF) | Permite:<br><br>- mostrar (o no) los valores del elemento seleccionado en el resultado de la consulta.<br><br>- renombrar el elemento seleccionado en el resultado de la consulta (crear un alias).<br><br><br> |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.GIF) | Clasifica los valores del artículo seleccionado en el resultado de la consulta. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE4.GIF) | Añade un nuevo elemento al resultado de la consulta. Esta partida corresponde a:<br><br>- el número de registros encontrados en un grupo de registros.<br><br>- el número de registros encontrados en un archivo de datos.<br><br><br> |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE5.GIF) | Añade un nuevo elemento al resultado de la consulta. Esta posición corresponde a la suma de un grupo de registros. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE6.GIF) | Añade un nuevo elemento al resultado de la consulta. Esta partida corresponde a:<br><br>- la media para un grupo de registros.<br><br>- el valor mínimo para un grupo de registros.<br><br>- el valor máximo para un grupo de registros.<br><br><br> |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE7.GIF) | Modifica los valores mostrados en el resultado de la consulta. Usted tiene la capacidad de mostrar:<br><br>- el año a partir de una posición de fecha.<br><br>- el mes a partir de una posición de fecha.<br><br><br> |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE8.GIF) | Define las condiciones de selección. Es posible:<br><br>- aplicar una condición de selección a la posición seleccionada.<br><br>- modificar y/o borrar una condición existente.<br><br>- modificar la secuencia de las condiciones existentes.<br><br><br> |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE9.gif) | Selecciona los registros correspondientes a un período determinado. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE10.GIF) | Selecciona los primeros n registros o los últimos n registros mostrados en el resultado de la consulta. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE11.gif) | Elimina (o no) los registros "duplicados" del resultado de la consulta. |
| ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE12.gif) | Permite:<br><br>- especifique el tipo de base de datos utilizada (HFSQL, Oracle u otra).<br><br>- gestionar las uniones (añadir, modificar, eliminar, reiniciar).<br><br>- comprobar (o no) la sintaxis SQL utilizada (sólo en el modo de prueba).<br><br>- especifique la opción de tiempo de ejecución (optimizando la memoria utilizada, comprobando el formato de archivo, ....).<br><br>- especificar si la consulta es Visible en "Reports & Queries".<br><br><br> |


## WDInst: Instalando una aplicación de HFSQL Client/Server
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
Al crear el programa instalación de una aplicación WINDEV que utiliza los archivos de datos de HFSQL Client/Server, WDInst le permite:

- instalar un servidor de HFSQL Client/Server si es necesario.

- configurar las conexiones utilizadas. Es posible:

	- utilizar los parámetros de conexión definidos en el análisis o por programación.

	- definir parámetros de conexión al instalar la aplicación.




- copy los archivos de datos HFSQL en el servidor durante la instalación






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Instalar un servidor y elegir un tipo de servidor
<a name="instalar_servidor_elegir_tipo_servidor_ELTTEXTE000190"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000026"></a>

Durante su instalación, una aplicación WINDEV (aplicación monousuario o multiusuario) puede permitir al usuario final instalar un servidor de HFSQL Client/Server. Para ello, la aplicación WINDEV debe utilizar los archivos de datos de HFSQL Client/Server en el description de su análisis.

WDInst permite que el usuario final lo instale:

- un servidor Windows HFSQL Client/Server.

- un servidor Linux HFSQL Client/Server.




Puede proponer instalar uno de estos servidores (o ninguno).

Para configurar esta opción desde el WDInst, seleccione "Parámetros de los archivos instalación .. HFSQL" y seleccione la pestaña "HFSQL C/S".
<a name="NOTE2_2"></a>


### Parámetros avanzados
<a name="parametros_avanzados_ELTPARAGRAPHE000042"></a>

**Atención**: Para realizar estas configuraciones, el programa instalación debe ser suministrado con los paquetes instalación de los diferentes servidores. Con el botón "Advanced" se selecciona el paquete necesario. El último paquete de instalación disponible también se puede descargar de la página web de PC SOFT.

## Configuración de la conexión
<a name="configuracion_conexion_ELTTEXTE000220"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000051"></a>

Si su aplicación utiliza la conexión a un servidor de HFSQL Client/Server definido en su análisis, esta conexión puede ser configurada automáticamente al instalar la aplicación final.

Para configurar esta opción desde el WDInst, seleccione "Parámetros de los archivos instalación .. HFSQL" y seleccione la pestaña "Conexiones"..

**Para configurar estas conexiones**: 

1. Seleccione "Editar la ubicación de los archivos de datos".

2. Seleccione la conexión deseada.

3. Haga clic en el botón "Parámetro. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=INstall_cs.gif)





Las principales opciones de configuración son las siguientes:

- Si se permite al usuario (o no) introducir los parámetros de la conexión durante la instalación.

- Si la conexión debe utilizar las opciones del análisis o los valores especificados en la asistente. En este caso, debe especificar los valores de la conexión a utilizar.

- Si deben utilizarse los parámetros del servidor que se instaló previamente.




**Observaciones**:

- Si la instalación debe crear una nueva base de datos, el usuario especificado en los parámetros de la instalación debe ser un "superusuario". De hecho, incluso si el usuario tiene los derechos para crear la base de datos, no se le permitirá crear los archivos de datos para esta base de datos.

- Si el usuario configura una conexión durante la instalación, esta conexión se almacenará en el archivo "&lt;Nombre ejecutable&gt;".WX". Este archivo corresponde a un archivo oculto en formato de texto. El acceso a su contenido lo realiza [INIRead](../WDLang1/3013043.md).

## Instalación de los archivos de datos
<a name="instalacion_los_archivos_datos_ELTTEXTE000244"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000083"></a>

Al crear la aplicación del programa instalación, también puede desplegar los archivos de datos de HFSQL Client/Server en el servidor deseado.

Para configurar esta opción desde el WDInst, seleccione "Parámetros de los archivos instalación .. HFSQL" y seleccione la pestaña "HFSQL C/S".
<a name="NOTE4_2"></a>


### Configuración
<a name="configuracion_ELTPARAGRAPHE000092"></a>

Para cada archivo de datos de la aplicación, el botón "Propiedades" permite especificar:

- la ubicación actual del archivo de datos para instalar.

- la conexión que debe ser associated con este archivo de datos. 

- si el índice debe copiarse o no. Si no se copia el índice, se reconstruirá automáticamente durante la instalación. Esta opción se utiliza para limitar el tamaño del programa instalación cuando el archivo índice es bastante grande.

## Instalación de la aplicación
<a name="instalacion_aplicacion_ELTTEXTE000274"></a>
Cuando la aplicación se instala en el ordenador del usuario final, dependiendo de las opciones seleccionadas, la asistente permite al usuario instalar un servidor de HFSQL Client/Server en un ordenador Windows o Linux y configurar la conexión utilizada.  

Al final de instalación, se crea un archivo ".wx" al lado del ejecutable.

La estructura de este fichero se da a título informativo. Esta estructura puede ser modificada en una próxima versión. Este archivo no debe ser modificado manualmente.

Este archivo contiene la siguiente información:


```txt
[CONNECT]
NBCX = Number of defined connections
NOM_x = Name of the connection number x
BASE_x = Type of database used (always corresponds to hAccessHFClientServer)
ADRESSE_x = Address of server used (name or IP address of the server)
PORT_x = Connection port
DATABASE_x = Name of the database used
LOGIN_x = Connection login
PWD_x = Encrypted password
ACCESS_x = Type of access (hOReadWrite or hORead)
EXTINFO_x = Extended information
```




## Importación de un proyecto WINDEV/WINDEV Mobile y sus elementos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000233"></a>
Un proyecto WINDEV y todos sus elementos compatibles con WINDEV Mobile (análisis, ventanas, clases, conjuntos de procedimientos, consultas, etc.) pueden abrirse en WINDEV Mobile.. Asimismo, un proyecto WINDEV Mobile puede abrirse en WINDEV.

Un proyecto WINDEV (o WINDEV Mobile) y todos sus elementos compatibles con WEBDEV (análisis, clases, conjuntos de procedimientos, consultas, etc.) pueden ser importados a un nuevo proyecto WEBDEV.. 
Asimismo, un proyecto WEBDEV puede ser importado a un nuevo proyecto WINDEV.. Para obtener más información, consulte [Importar un proyecto WEBDEV y sus elementos](../Editeurs/2030041.md).

### Importación de elementos WINDEV Mobile (ventanas, informes, etc.) en un proyecto WINDEV
<a name="importacion_elementos_windev_mobile_ventanas_informes_etc_proyecto_windev_ELTPARAGRAPHE000104"></a>

Para importar elementos de WINDEV Mobile (ventanas, informes, etc.) a un proyecto WINDEV:

1. Abrir el proyecto WINDEV donde se debe realizar la importación.  

2. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV  o WINDEV Mobile". El asistente de importación se abre automáticamente.

3. Seleccione el proyecto WINDEV Mobile que contiene los elementos a importar o seleccione directamente los archivos WINDEV Mobile a importar.. Pase a la etapa siguiente.

4. Seleccione los elementos que desea importar.

5. Si el proyecto WINDEV Mobile contiene elementos que pueden ser compartidos con WINDEV (clases, conjuntos de procedimientos, informes, consultas), la importación de asistente le permite:

	- comparten estos elementos. Estos elementos no se copiarán en el proyecto WINDEV pero serán accesibles desde el proyecto.

	- copy estos elementos al proyecto WINDEV. Estos elementos evolucionarán de forma independiente en cada proyecto.




6. Validar. La importación se realiza automáticamente.

## Notas
<a name="notas_ELTTEXTE000317"></a>


### Códigos de gestión de proyectos compartidos entre WINDEV, WINDEV Mobile y WEBDEV
<a name="codigos_gestion_proyectos_compartidos_entre_windev_windev_mobile_webdev_ELTPARAGRAPHE000168"></a>

Si los proyectos comparten clases o conjuntos de procedimientos, los elementos que se encuentran en el proyecto fuente se manejan directamente.. Se utiliza un único código fuente.

Puede que tengas que adaptar el código para que funcione en diferentes plataformas. Por ejemplo, se recomienda utilizar las funciones InXXXmode ([InAndroidMode](../WDLang1/1000018990.md), [IniOSMode](../WDLang1/1000019765.md), etc.) y/o la sentencia &lt;IF ExecutionTarget=...&gt; para diferenciar el código que se ejecutará en un PC, un sitio web o un Phone. 



<a name="NOTE4_2"></a>


### Dependencia de los archivos
<a name="dependencia_los_archivos_ELTPARAGRAPHE000184"></a>

Los elementos se importan junto con sus dependencias: todos los archivos relacionado al elemento importado también se importan. Las imágenes se convierten automáticamente a un formato compatible con WINDEV, WEBDEV o WINDEV Mobile.

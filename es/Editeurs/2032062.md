


## Crear una consulta de selección 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000241"></a>
<a name="Select_query"></a>
Una consulta ***select*** (sentencia SQL [SELECCIONE](../Editeurs/2034001.md)) se utiliza para ver una selección de registros de una base de datos. Este es el tipo de consulta más común.

Para realizar esta selección, las principales características de la consulta son las siguientes:

- el tipo de unión entre los ficheros de datos utilizados (si se utilizan varios ficheros de datos),

- las condiciones de selección de los registros (clientes que viven en Seattle, productos cuyo precio es superior a 40 dólares, ....),

- la agrupación de los valores de los artículos (agrupación de los productos por tipo, de los clientes por edad, etc.),

- las operaciones realizadas sobre el contenido de los artículos (importe total de los pedidos de un cliente, importe medio gastado por cliente,...),

- el tipo de resultado de la consulta (resultado ordenado por código postal, por precio, ....).




Por ejemplo, desea visualizar la cantidad total de pedidos por cliente que vive en Londres.. Solo se considerarán los clientes con un total de pedidos superior a 3000 dólares. El resultado se clasificará por importe total en orden ascendente. Para realizar esta consulta, el resultado se agrupará automáticamente por nombre de cliente y por ciudad.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_RequeteSelection.gif)
 El editor de consultas genera automáticamente el código SQL de esta consulta. 

**Para ver el código SQL de una consulta**: 

1. Abrir el menú contextual del gráfico de consulta.

2. Seleccione "Código SQL".




También tiene la posibilidad de presionar la tecla F2. 


```sql
SELECT 
	CUSTOMER.CustomerName AS CustomerName,	
	CUSTOMER.CITY AS CITY,	
	SUM(ORDERS.TotalIOT) AS the_sum_TotalIOT
FROM 
	CUSTOMER,	
	ORDERS
WHERE 
	CUSTOMER.CustomerNum = ORDERS.CustomerNum
	AND
	(
		CUSTOMER.CITY = 'London'
	)
GROUP BY 
	CUSTOMER.CustomerName,	
	CUSTOMER.CITY
HAVING 
	SUM(ORDERS.TotalIOT) > 3000
ORDER BY 
	the_sum_TotalIOT ASC
```


Para más detalles sobre la description y el uso de una consulta Select, véase: 

- [Seleccione la ventana de consulta description](../Editeurs/2032049.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000265"></a>
Se pueden utilizar dos métodos para crear una consulta de selección:

- creación a través de la ventana de description de una consulta selectiva.

- creación a través de una asistente simplificada.



<a name="NOTE2_2"></a>


### Crear una nueva consulta selectiva con la ventana description de una consulta selectiva
<a name="crear_una_nueva_consulta_selectiva_con_ventana_description_una_consulta_selectiva_ELTPARAGRAPHE000060"></a>

Para crear una nueva consulta selectiva con la ventana description de una consulta selectiva:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- En la ventana que se muestra, haga clic en "Query".

	- El asistente de creación de consultas se abre.




2. Seleccione crear una consulta de selección (opción "Selección (SELECT)"). 
	Pase a la etapa siguiente del asistente.

3. Si su proyecto es relacionado a ningún análisis, especifique el análisis al que se adjuntará la consulta. Valide.

4. La ventana de descripción de la consulta se abre.

5. Escriba el nombre de la consulta (nombre del archivo ".WDR" correspondiente a la consulta). Este nombre se utilizará para identificar la consulta en sus programas.

6. Especifique el título de la consulta describiendo brevemente el propósito de la consulta. Este epígrafe se utilizará en la documentación del proyecto.

7. En [la sección izquierda](../Editeurs/2032011.md), seleccione los diferentes elementos utilizados por la consulta.
	Observación: Para simplificar la selección de los artículos, éstos pueden ser mostrados por el archivo de datos de associated, por consulta de associated o en orden alfabético.
	Para más detalles sobre los elementos de una consulta, véase [Elementos que participan en una consulta](../Editeurs/2032004.md).

8. En [la sección central](../Editeurs/2032011.md), seleccione la consulta en la que debe realizarse una acción (condición de selección, ordenación, etc.).

9. En [la sección correcta](../Editeurs/2032039.md), defina las acciones que deben realizarse sobre el elemento seleccionado (condición de selección, ordenación, etc.).
	Para obtener más información, consulte:

	- [condiciones de selección de una consulta](../Editeurs/2032019.md),

	- [operaciones en una consulta](../Editeurs/2032020.md),

	- [ordenación de los registros de una consulta](../Editeurs/2032005.md).




10. Eliminar (si es necesario) los registros "duplicados" del resultado de la consulta (botón "Eliminar duplicados"). 
	En este caso, cada Record encontrada en el resultado de la consulta será única. Por ejemplo, si se utiliza una consulta para mostrar a todos los clientes que hicieron al menos un pedido, los clientes que hicieron varios pedidos aparecerán una sola vez en el resultado de la consulta.

11. Validar la ventana. La representación gráfica de la consulta se muestra automáticamente.

12. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y haga clic en "Guardar" para guardar la consulta.



<a name="NOTE2_3"></a>


### Creando una nueva consulta Select con la asistente simplificada
<a name="creando_una_nueva_consulta_select_con_asistente_simplificada_ELTPARAGRAPHE000129"></a>

Para crear una nueva consulta de selección con la asistente simplificada:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. En la ventana que aparece, haga clic en "Query" (consulta). El asistente de creación de consultas se abre.

2. Especifique que desea crear una consulta Select (opción "Select"), marque "Usar la asistente simplificada".
	Pase a la etapa siguiente del asistente.

3. Si su proyecto es relacionado a ningún análisis, especifique el análisis al que se adjuntará la consulta.
	Pase a la etapa siguiente del asistente.

4. Seleccione los elementos utilizados por la consulta.
	Observación: Para simplificar la selección de los artículos, éstos pueden ser mostrados por el archivo de datos de associated, por consulta de associated o en orden alfabético.
	Para más detalles sobre los elementos de una consulta, véase [Elementos que participan en una consulta](../Editeurs/2032004.md).
	Pase a la etapa siguiente del asistente.

5. Seleccione el orden en el que se verán los artículos. Pase a la etapa siguiente.

6. Especificar si se deben describir o no las condiciones de selección.. 
	Pase a la etapa siguiente.
	Si se deben definir condiciones de selección:

	- seleccione la posición para la que debe añadirse una condición de selección.

	- seleccionar los criterios de selección (igual a, mayor que, ...) de la condición. Esta condición se puede aplicar a un valor dado, a un parámetro, a los registros seleccionados por otra consulta, ...
			Repita este paso para todas las condiciones de selección que desee definir.




7. Especificar si los elementos seleccionados por la consulta deben ser clasificados o no.
	En este caso, seleccione la posición de clasificación y especifique el orden de clasificación.

8. Escriba el nombre de la consulta (nombre del archivo ".WDR" correspondiente a la consulta). Este nombre se utilizará para identificar la consulta en sus programas.

9. Especifique el título de la consulta describiendo brevemente el propósito de la consulta. Este epígrafe se utilizará en la documentación del proyecto..
	Observación: El nombre de la consulta se define automáticamente a partir de la leyenda escrita.

10. Validar la consulta description. La representación gráfica de la consulta se muestra automáticamente en el editor de consultas.

## Notas
<a name="notas_ELTTEXTE000301"></a>


- Para modificar las características de una consulta, en la pestaña "Consulta", en el grupo "Edición", haga clic en "Descripción". Para obtener más información, consulte [Definir una consulta](../Editeurs/2032038.md).

- Para generar el código SQL correspondiente a esta consulta, seleccione "Código SQL" en el menú contextual de la consulta. Si se realizan cambios en la consulta description, estos cambios se reflejarán automáticamente en el código SQL correspondiente.

- También tiene la posibilidad de crear una consulta a partir de uno o más ficheros de datos descritos en la editor de análisis: Para ello, arrastre los archivos de datos seleccionados en el panel "Análisis" y suéltelos en el editor de consultas.

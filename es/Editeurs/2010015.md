## Presentación
<a name="presentacion_ELTTEXTE000107"></a>
Las opciones de compilación permiten definir la configuración de generación del sistema de ayuda. Hay varios ajustes:

- Nombre del fichero de ayuda a generar.

- Compilador a utilizar.

- Ajustes de compilación para el compilador seleccionado.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Nombre del archivo de ayuda generado
<a name="nombre_del_archivo_ayuda_generado_ELTTEXTE000131"></a>
El nombre del archivo de ayuda generado por WINDEV puede ser diferente del nombre del sistema de ayuda. Por ejemplo, en una aplicación multilingüe, se crearán y generarán diferentes archivos de ayuda a partir del mismo sistema de ayuda WINDEV (archivo WDH).

En este caso, el nombre de un archivo de ayuda puede contener varios elementos:

- **el nombre base del archivo de ayuda**. El nombre especificado en las propiedades del sistema de ayuda es el sugerido por Default: en la pestaña "Modificación", en el grupo "Sistema de ayuda", haga clic en "Descripción". Puede modificarse en los ajustes de generación. 

- **el sufijo de los archivos generados**. Este sufijo puede corresponder a: 

	- Sin sufijo: el nombre del archivo generado es el nombre base del archivo de ayuda. 

	- Número de la lengua utilizada en los ficheros. Este número es utilizado por la función [Nación](../WDLang1/3054014.md).

	- El identificador de idioma (US para inglés).

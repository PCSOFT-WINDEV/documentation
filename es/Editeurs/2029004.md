


## Entorno de la editor de reportes 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtatEditeur.gif&type=thumb)
Los elementos importantes de editor de reportes son los siguientes:


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif)<br> | **Cinta de opciones**<br>Permite un acceso rápido a las principales características de editor de reportes. Esta cinta incluye paneles, grupos y opciones. Para obtener más información, consulte [Un menú en forma de cinta](../Editeurs/9000149.md). |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle2.gif)<br> | [**Reglas de alineación**](../Editeurs/2022001.md)<br>Se utiliza para alinear los controles en la reporte. Para mostrar las reglas, presione Ctrl + R. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif)<br> | **Reporte editado actualmente**<br>Las barras de desplazamiento se utilizan para desplazarse por los informes grandes. También puedes hacer un zoom en la reporte para visualizar con más precisión los elementos de la reporte (Ctrl + rueda del ratón). |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE4.gif)<br> | [**Paneles verticales**](../Editeurs/2027001.md)<br>Muestra:<br><br>- los elementos (archivos, variables, informes, controles, ...) disponibles en el proyecto actual (["Panel "Explorador de proyectos](../Editeurs/2027029.md)).<br><br>- los elementos description de reporte a través del [modificador](../Editeurs/2027006.md).<br><br><br>\   \   \   \<br>\ --- \ --- \ --- \<br>\ ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ER.png) \ Importante \ En Reports & Queries sólo están disponibles los paneles ["Explorador de proyectos"](../Editeurs/2027029.md) y ["Modifier"](../Editeurs/2027006.md) . \<br><br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle5.gif)<br> | **Ventana de descripción**<br>Le permite ver y/o modificar:<br><br>- Las características del control seleccionado (s).<br><br>- las características de la reporte actual.<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE6.gif)<br> | **ventana de código**<br>Muestra los procesos associated con el reporte, con sus bloques y controles. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE7.gif)<br> | [**Documentos abiertos**](../Editeurs/2029012.md)<br>Muestra un botón para cada Document abierta en el editor. Para mostrar el elemento solicitado en primer plano, pulse el "botón" correspondiente. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE8.gif)<br> | [**Paneles horizontales**](../Editeurs/2027001.md)<br>Muestra:<br><br>- errores de compilación (en el código WLanguage) o errores UI (["Panel "Errores de compilación](../Editeurs/2027031.md)), <br><br>- la herramienta de búsqueda ([" Panel Buscar - Reemplazar](../Editeurs/2027013.md)), <br><br>- el depurador durante la prueba de un elemento (["Depurador" pan](../Editeurs/2027030.md)), <br><br>- ...<br><br><br> |

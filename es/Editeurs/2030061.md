


## Favicon: Definir el sitio Icon
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000098"></a>
WEBDEV le permite definir fácilmente el sitio Icon (también llamado FavIcon). Este icono se muestra en acceso directo en el escritorio, en las listas de favoritos del navegador, en la cabecera de la barra de Address, ...<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Favicone.gif)


<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000122"></a>
Para asociar un Favicon a su sitio Address:

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. En la pestaña "Proyecto", el control "Ícono del sitio" sirve para introducir el Image utilizado como favicon. 

3. Ampliar el ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Menu_Image_Editeur%20-%20HC%20N%B0001.gif) Icon: 

	- La opción "Examinar" permite seleccionar un archivo Image que se encuentre en el disco. 

	- La opción "Catálogo" para seleccionar un Image en el catálogo de Image suministrado con WEBDEV.




4. Valide la ventana de descripción. 




**Limitaciones**:

- El tamaño de Image debe ser igual a 16x16 o 32x32. 

- Le recomendamos que utilice una Image en formato .ICO para que esta Image se visualice correctamente en todos los navegadores que admiten la visualización de un Favicon. 

- Evite utilizar los formatos ".PNG", ".GIF", ".JPG" porque estos formatos no son compatibles con algunos navegadores.. Sólo los navegadores más recientes pueden mostrarlos.

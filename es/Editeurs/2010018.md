
## Opciones de compilación (formato chm)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Permitiendo la descompilación
<a name="permitiendo_descompilacion_ELTTEXTE000101"></a>
Si esta opción está marcada, el archivo de ayuda en formato CHM puede ser descompilado (por la herramienta apropiada), permitiéndole a get todas las páginas HTML así como los archivos utilizados por estas páginas (imágenes por ejemplo).

La descompilación de un sistema de ayuda es una forma fácil de get las diferentes páginas encontradas en un sistema de ayuda del archivo compilado sin tener las fuentes del sistema de ayuda.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Recopilación de la información para la búsqueda de texto completo
<a name="recopilacion_informacion_para_busqueda_texto_completo_ELTTEXTE000125"></a>
Esta opción le permite proponer un modo de búsqueda avanzada (pestaña"Buscar" de la ayuda).

Para proponer este modo, se crea automáticamente una lista de palabras utilizadas en toda la página del sistema de ayuda durante la compilación.

Para que esta opción esté disponible en los sistemas de ayuda creados con WINDEV, debe tener el compilador de ayuda completo.

**Observación: Búsqueda avanzada:** En este modo, el texto escrito por el usuario se buscará en todas las páginas de ayuda del archivo.. Cuando se valida la búsqueda, se muestra la lista de páginas que contienen el texto.. En esta lista sólo se muestran los títulos de las páginas.. Todo lo que tiene que hacer es seleccionar la página para mostrarla.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Creación de un índice binario
<a name="creacion_indice_binario_ELTTEXTE000149"></a>
Si esta opción está seleccionada, se crea un índice binario. Este índice se compilará directamente en el archivo de ayuda de CHM. Por lo tanto, el tamaño ocupado por el índice se reducirá.

Esta opción es necesaria si desea que su índice se clasifique en orden alfabético.

<a name="NOTE4"></a>
<a name="NOTE4_1"></a>


## Crear una tabla de contenido binario
<a name="crear_una_tabla_contenido_binario_ELTTEXTE000173"></a>
Esta opción se recomienda para los sistemas de ayuda grandes para que el resumen se cargue más rápido cuando se abre el archivo de ayuda.. También reduce el tiempo necesario para abrir una página de ayuda desde el resumen.

<a name="NOTE5"></a>
<a name="NOTE5_1"></a>


## Observación
<a name="observacion_ELTTEXTE000197"></a>
Por razones de seguridad, Windows ahora sólo permite que los archivos CHM se ejecuten localmente. No puede iniciar un archivo CHM que se encuentre en la red.

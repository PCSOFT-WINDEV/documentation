


## Importación de elementos en un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000110"></a>
Todos los elementos manejados por un proyecto (ventana, reporte, clases, set de procedimientos, consultas, ...) pueden ser transferidos de un proyecto a otro, para el mismo producto. La importación de elementos se utiliza para añadir elementos procedentes de otro proyecto al proyecto actual.. 

Por ejemplo, las ventanas WINDEV pueden ser importadas desde un proyecto WINDEV al proyecto WINDEV actual.. 

Observación: También tiene la posibilidad de realizar importaciones entre proyectos que pertenecen a productos diferentes.. Para obtener más información, consulte: 

- [Importar un proyecto WINDEV](../Editeurs/2030002.md)

- [Importación de ventanas WINDEV](../Editeurs/2030001.md)




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000134"></a>
Para importar elementos de un proyecto al proyecto actual (ambos proyectos utilizan el mismo producto): 

1. En la pestaña "Proyecto", en el grupo "Proyecto", amplíe "Importar" y seleccione "XXX elementos y sus dependencias" (donde XXX representa el nombre del producto actual, WINDEV, WEBDEV o WINDEV Mobile).

2. Seleccione el directorio que contiene los archivos a importar. La lista de elementos se muestra automáticamente. Esta lista incluye:

	- el nombre del archivo,

	- la description del archivo (si existe).




3. Seleccione los elementos a importar. Aparece una marca de verificación delante de los archivos seleccionados.

4. Validar. Los archivos seleccionados se importan automáticamente al directorio actual del proyecto y se incluyen en la lista de elementos del proyecto.

<a name="NOTE3_1"></a>


## Notas
<a name="notas_ELTTEXTE000158"></a>


### Dependencia de los archivos
<a name="dependencia_los_archivos_ELTPARAGRAPHE000054"></a>

Los elementos se importan junto con sus dependencias: todos los archivos relacionado al elemento importado también se importan. Por ejemplo, la ventana y su fondo Image se importan automáticamente.
<a name="NOTE3_2"></a>


### Estimación del tamaño de los archivos importados
<a name="estimacion_del_tamano_los_archivos_importados_ELTPARAGRAPHE000061"></a>

El botón "Calcular" se utiliza para estimar el tamaño de los archivos importados.. Este tamaño tiene en cuenta el tamaño del elemento a importar y el tamaño de los archivos que dependen de este elemento.

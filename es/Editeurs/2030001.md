


## Importar una ventana WINDEV a un proyecto móvil
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000121"></a>
Las ventanas WINDEV que no tienen el mismo formato en WINDEV Mobile, WINDEV Mobile le ofrece la posibilidad de importar una ventana WINDEV a un proyecto WINDEV Mobile.

**Recordatorio**: Los análisis, consultas, conjuntos de procedimientos, informes, clases tienen el mismo formato en WINDEV y en WINDEV Mobile.. Estos elementos pueden ser compartidos entre un proyecto WINDEV estándar y un proyecto WINDEV Mobile.. Para obtener más información, consulte [Compartir elementos WINDEV](../WDLang3/3023045.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000145"></a>
Para importar una ventana WINDEV en un proyecto WINDEV Mobile:

1. En la pestaña "Proyecto", en el grupo "Proyecto", despliegue "Importar" y seleccione "Elementos WINDEV".

2. Seleccione el archivo de ventana (extensión WDW) que desea importar.

3. La ventana seleccionada se abre automáticamente en WINDEV Mobile. Para incluir esta ventana en el proyecto actual, todo lo que tiene que hacer es guardarla. La ventana se graba en formato WPW.




**Observación**: El tamaño de la ventana importada corresponde a su tamaño inicial. Es necesario comprobar la UI y el tamaño de la ventana para adaptarlos a la plataforma (Android, iOS, etc.).

## Notas
<a name="notas_ELTTEXTE000169"></a>


### Operaciones realizadas durante la importación
<a name="operaciones_realizadas_durante_importacion_ELTPARAGRAPHE000048"></a>

- Los controles que no existen en WINDEV Mobile se borran automáticamente (barra de herramientas, ...).

- La lista de errores que se han producido (por ejemplo, funciones que no existen en WINDEV Mobile) se muestra en el panel "Errores de compilación".



<a name="NOTE3_2"></a>


### Dependencia de los archivos
<a name="dependencia_los_archivos_ELTPARAGRAPHE000056"></a>

Las ventanas se importan con sus dependencias: todos los archivos relacionado a la ventana importada también se importan. Las imágenes se convierten automáticamente a un formato compatible con WINDEV Mobile. Por ejemplo, el fondo Image de una ventana se importa automáticamente y se convierte al formato GIF.
<a name="NOTE3_3"></a>


### Diferencias al ejecutar los eventos de una ventana
<a name="diferencias_ejecutar_los_eventos_una_ventana_ELTPARAGRAPHE000063"></a>

Las diferencias entre los sistemas operativos de Windows y los sistemas para dispositivos móviles dan como resultado una gestión diferente de los eventos associated con las ventanas. El orden de ejecución de los eventos de WLanguage associated con la ventana es diferente en WINDEV y en WINDEV Mobile.

Al importar una ventana WINDEV, si su código se basa en el orden en que se ejecutan los eventos WLanguage associated con la ventana, compruebe si la aplicación tiene el comportamiento esperado (en modo de prueba, simulador o en tiempo de ejecución).
<a name="NOTE3_4"></a>

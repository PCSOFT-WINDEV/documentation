


## Fábrica de software: Opciones de administrador
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000139"></a>
Se pueden configurar varias opciones en el administrador de la Fábrica de Software: 

- Autenticación

- Coordinador

- Robots




Para configurar estas opciones: 

1. Inicie el administrador de la Fábrica de Software: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDFAdmin".

2. En la ventana que se muestra, especifique el nombre del coordinador al que el administrador debe conectarse.. 

3. El administrador de la Fábrica de Software inicia. 

4. Haga clic en la pestaña "Opciones" para configurar el administrador de la Fábrica de Software.

## Autenticación - Parámetros de los Centros control
<a name="autenticacion_parametros_los_centros_control_ELTTEXTE000163"></a>
Los parámetros de autenticación se utilizan para definir la base de datos de las cuentas utilizadas por la Fábrica de Software. La Fábrica de Software utiliza las cuentas de los Centros control. Debe especificar las opciones para conectarse a esta base de datos. Esta base de datos puede ser: 

- una base de datos en modo compartido. Basta con especificar el directorio compartido que contiene los archivos de datos de la base de datos del Centro control. 

- una base de datos HFSQL Client/Server. Todo lo que tienes que hacer es especificar el servidor y su puerto así como la base de datos utilizada. 
	Observación: el botón "Test" se utiliza para comprobar los parámetros especificados. 

- una plataforma en la nube. Todo lo que tienes que hacer es especificar el nombre de la plataforma utilizada. 
	Observación: Tienes la capacidad de crear directamente una plataforma de la Nube si es necesario.




La opción "Solicitar contraseña al iniciar" se utiliza para forzar la autentificación del usuario cuando el administrador de la Fábrica de Software inicia.

## Coordinador
<a name="coordinador_ELTTEXTE000187"></a>
Configurar el coordinador le permite: 

- Elija el coordinador.

- Definir las opciones del coordinador (enlace "Opciones del coordinador"). 




Las opciones del coordinador afectan: 

- La base de datos de los Centros control sirve para almacenar los planes de acción y guardar sus informes de situación. 

- El establecimiento de la historia de los resultados. Se puede definir el número de resultados almacenados para cada plan de acción. 

- El establecimiento del método para enviar los informes de estado de los planes de acción (mensajería interna de los Centros de Control, Email, etc.). 

- La configuración de los receptores en caso de fallo del robot.

## Robots
<a name="robots_ELTTEXTE000211"></a>
Las siguientes opciones afectan a los robots: 

- **Modify the priority order of robots**: Cuando se dispone de varias computadoras, se puede seleccionar una computadora específica para ejecutar los planes de acción (la computadora más rápida, por ejemplo). 
	Recordatorio: Puedes añadir robots en la pestaña "Robots" del administrador de la Fábrica de Software.

- **Gestión de los cuadros de diálogo "No mostrar de nuevo"**: Esta opción permite volver a activar los cuadros de diálogo previamente desactivados (marcando la casilla "No mostrar de nuevo", por ejemplo).

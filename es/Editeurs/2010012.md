
## Tipos de sistemas de ayuda en una aplicación WINDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000175"></a>
Existen varias soluciones para proporcionar ayuda a los usuarios:

- [tooltips](#NOTE2_1),

- [información mostrada en la barra de estado](#NOTE3_1),

- [ventanas de ayuda estándar](#NOTE4_1),

- [archivo de ayuda HLP o CHM](#NOTE5_1).




Los tres primeros tipos de sistemas de ayuda se crean directamente en el editor de ventanas, en la descripción de cada control (pestaña "Ayuda"). Estos sistemas de ayuda tienen una presentación simple y no pueden vincularse entre sí.

El cuarto tipo de sistema de ayuda se crea en el editor de ayuda. Es un sistema de ayuda más estructurado, que permite guiar a los usuarios paso a paso.

## Tooltips
<a name="tooltips_ELTTEXTE000199"></a>
Los tooltips o globos de ayuda son rectángulos de color amarillo claro que se muestran cuando se pasa el cursor sobre un control. Un tooltip contiene un mensaje de ayuda sobre el control por el que se pasa el cursor. Un tooltip puede informar al usuario sobre la unidad monetaria prevista al escribir un precio, por ejemplo.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=BULLEAIDE.GIF)


De forma predeterminada, los tooltips se habilitan automáticamente en una aplicación WINDEV. 

Para escribir el mensaje que se mostrará en el tooltip de un control: 

1. Abra la ventana de descripción del control (seleccione "Descripción" del menú contextual). 

2. En la pestaña "Ayuda", en el área "Tooltip", escriba el texto de ayuda. 

3. Validar. 




**Observaciones**: 

- En una aplicación multilingüe, puede introducir el texto del tooltip en varios idiomas, en la ventana description de control. 

- Si la opción "RTF" está seleccionada, se puede definir el formato del texto mediante las opciones de la barra de herramientas RTF. El formato se mantendrá en tiempo de ejecución. 

- El texto del tooltip puede modificarse mediante programación con la propiedad [ToolTip](../Proprietes/2510005.md). 

- La información sobre herramientas aplica automáticamente un margen alrededor del texto: por lo tanto, el texto del tooltip es más legible.

- Los usuarios finales pueden modificar los tooltips en las aplicaciones. Para obtener más información, consulte [Tooltips de controles (FAA)](../Editeurs/1000021821.md). 




Para más detalles sobre cómo personalizar la información sobre herramientas, consulte [Tooltips personalizados](../Editeurs/9500233.md).

## Barra de estado
<a name="barra_estado_ELTTEXTE000223"></a>
Puede mostrar información en la barra de estado de una ventana. Hay dos casos:

- El mensaje está relacionado con un control. El mensaje se puede introducir en la pestaña "Ayuda" de la ventana de descripción de los controles. El mensaje aparecerá cuando el usuario seleccione el control.

- El mensaje está relacionado con un proceso definido mediante programación. Utilice la función [Mensaje](../WDLang1/3021006.md) para definir el mensaje que se mostrará.



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Barremsg.gif)


Para obtener más información sobre las barras de estado, consulte [Barra de estado de las ventanas](../WDChamp/1010038.md).

## Ventanas de ayuda estándar
<a name="ventanas_ayuda_estandar_ELTTEXTE000247"></a>
Las ventanas de ayuda estándar son ventanas "Windows" estándar utilizadas para mostrar un texto explicativo asociado a un control.

El título y el texto de este tipo de ventana pueden introducirse en la pestaña "Ayuda" de la ventana de descripción del control. No se pueden utilizar imágenes en estas ventanas.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fenetre_Aide.gif)


Si se asoció este tipo de ventana a un control, la ventana se abrirá automáticamente al presionar F1.

## Archivo de ayuda CHM
<a name="archivo_ayuda_chm_ELTTEXTE000271"></a>
Un sistema de ayuda incluye un resumen, palabras clave (índice) y páginas. Cuando los usuarios abren la ayuda, son dirigidos a una página de ayuda. Las páginas de ayuda pueden contener imágenes, enlaces a otras páginas, etc

En WINDEV, puede crear archivos de ayuda CHM: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_CHM.gif)


Hay varias formas de poner en marcha esta ayuda:

- ayuda completa: el archivo de ayuda se muestra directamente.

- ayuda completa contextual: el archivo de ayuda se abre en una página específica que corresponde a la ventana o control actual.




Para obtener más información sobre cómo crear un archivo de ayuda, consulte [Creación de un archivo de ayuda](../Editeurs/2010004.md).

Para más detalles sobre las distintas formas de abrir un archivo CHM, consulte [Inicio de la ayuda](../Editeurs/2010017.md).

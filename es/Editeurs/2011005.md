


## Entidad de un MDL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000143"></a>
Para agrupar la información según el método MERISE, se deben crear entidades.

El editor de análisis le permite crear fácilmente entidades en un CDM (Conceptual Data Model).

Cuando [generar el MDL a partir del MDL](../Editeurs/2011020.md), las entidades se convertirán automáticamente en [archivos](../Editeurs/2011010.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Crear una entidad
<a name="crear_una_entidad_ELTTEXTE000167"></a>
Para crear una entidad:

1. En la pestaña "MCD", en el grupo "Creación", despliegue "Nuevo" y seleccione "Nueva entidad". La descripción de la entidad se muestra.

2. Especifique:

	- los **nombre de la entidad**. Este nombre será el nombre lógico del fichero creado cuando se genere el LDM a partir del CDM.

	- los **leyenda de la entidad** que describe brevemente el tema de la entidad.

	- si un **"Propiedad"Identificador automático** debe definirse en la entidad. El valor de esta propiedad es único para cada registro y se calcula automáticamente.

	- los **comentario** que presentan el modo de funcionamiento de la entidad.




3. La entidad se convierte en la entidad actual. Las propiedades de esta entidad se pueden describir automáticamente.

## Operaciones que se pueden realizar en una entidad
<a name="operaciones_que_pueden_realizar_una_entidad_ELTTEXTE000191"></a>


### Duplicar/Copiar una entidad
<a name="duplicarcopiar_una_entidad_ELTPARAGRAPHE000056"></a>

Para duplicar/copiar una entidad:

1. Seleccione la entidad a duplicar.

2. Presione Ctrl + C y luego Ctrl + V para copiar la entidad.

3. En la ventana de duplicación, especifique la información solicitada y valide. La entidad duplicada se crea automáticamente en el MDL actual.



<a name="NOTE3_2"></a>


### Borrar una entidad
<a name="borrar_una_entidad_ELTPARAGRAPHE000067"></a>

Para eliminar una entidad:

1. Seleccione la entidad que desea eliminar.

2. Abrir la el menú contextual de la entidad seleccionada (clic con el botón derecho del ratón).

3. Seleccione"Eliminar".. La entidad se elimina automáticamente del MDL. Si las relaciones apuntaban a esta entidad, estas relaciones también se borran.



<a name="NOTE3_3"></a>


### Renombrar una entidad
<a name="renombrar_una_entidad_ELTPARAGRAPHE000078"></a>

Para renombrar una entidad:

1. Seleccione la entidad a renombrar.

2. En la pestaña "MCD", en el grupo "Elemento actual", haga clic en "Descripción".

3. Modificar el nombre de la entidad y validar la descripción de la entidad. La entidad se renombra automáticamente.

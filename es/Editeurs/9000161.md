
## Desenmascarar a los editores
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000101"></a>
¿Quién no ha querido nunca ver el código y la UI al mismo tiempo, sin estar limitado por el entorno? WINDEV, WEBDEV y WINDEV Mobile le permiten "separar" los editores del entorno de la ventana principal.

Un caso común es la UI simultánea y la edición de código. El editor de código se desacoplará del entorno y se mostrará junto a él, en una segunda pantalla, por ejemplo.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000125"></a>


### Separar una ventana del entorno
<a name="separar_una_ventana_del_entorno_ELTPARAGRAPHE000021"></a>

Para separar una ventana del entorno: 

1. Seleccione el botón correspondiente al elemento a separar. Este botón se encuentra en la barra de documentos abiertos en el editor. 

2. Mover el botón fuera del entorno. 

3. Liberación. El elemento aparece en una nueva ventana, independiente del entorno. 




Esta nueva ventana puede ser: 

- en una nueva pantalla,

- redimensionado,

- ...


Si se abre una ventana de código desde el elemento separado, esta ventana será mostrada por Default en el mismo entorno que el elemento separado. 


### Volver a conectar una ventana al medio ambiente
<a name="volver_conectar_una_ventana_medio_ambiente_ELTPARAGRAPHE000038"></a>

Para volver a conectar un elemento al entorno: 

- Método 1: 

	1. Seleccione el botón correspondiente al elemento que desea volver a adjuntar. Este botón se encuentra en la barra de Document en la ventana independiente. 

	2. Seleccione "Adjuntar a la ventana principal" en el menú contextual. 
			Observación: Si la ventana independiente contiene varios elementos, tiene la posibilidad de adjuntar un solo elemento a la ventana principal.. 




- Método 2: 

	1. Seleccione la ventana independiente. 

	2. Mueve la ventana independiente hacia la barra de Document abierta en la ventana principal del editor. 

	3. Cuando el editor cambie a Color, suelte el botón del ratón: la ventana independiente se vuelve a conectar automáticamente al entorno.  
			Observación: Si la ventana independiente contiene varios elementos, todos los elementos se vuelven a unir a la ventana principal..

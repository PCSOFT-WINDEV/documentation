## Presentación
<a name="presentacion_ELTTEXTE000113"></a>
El panel "Catálogo Image" es un catálogo Image sensible al contexto. Este panel le permite acceder a las imágenes más qUI ckly para asociarlas a los controles de sus interfaces (UI).

Cuando se edita un control que requiere un Image (control Imagen, control Botón, ...), el catálogo Image sensible al contexto encuentra automáticamente los Image s que pueden ser utilizados (basados en la leyenda control, por ejemplo).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=voletCatalogueimage.gif)

## Utilización del panel "Catálogo Image
<a name="utilizacion_del_panel_catalogo_image_ELTTEXTE000161"></a>


### Búsqueda de imágenes
<a name="busqueda_imagenes_ELTPARAGRAPHE000045"></a>

Existen diferentes métodos de búsqueda de imágenes:

- Puede introducir una palabra clave directamente en el panel: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletCatalogueimage%203.gif). 
	Puede seleccionar un tema mediante el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletCatalogueimage%201.gif). La búsqueda se inicia automáticamente al pulsar Enter.

- Cuando se selecciona una control, se utiliza la leyenda control como palabra clave de búsqueda.



<a name="NOTE3_2"></a>


### Asignar una imagen a una control
<a name="asignar_una_imagen_una_control_ELTPARAGRAPHE000061"></a>

Se pueden utilizar varios métodos para asignar una imagen del catálogo de imágenes a una control:

- Cuando arrastra un Image desde el catálogo y lo suelta en la ventana, Page o reporte: se crea automáticamente un nuevo control Imagen con el Image seleccionado.

- Cuando arrastra una imagen desde el catálogo y la suelta en la control de la ventana o Page: las características de la control se modifican automáticamente.

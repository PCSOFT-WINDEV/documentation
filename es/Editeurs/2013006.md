## Presentación
<a name="presentacion_ELTTEXTE000119"></a>
Por Default, se abre una ventana de código único para cada elemento abierto en el editor

Por ejemplo, una ventana de código muestra el código de un botón que se encuentra en una ventana. Si se abre una nueva ventana de código para mostrar el código de un control que se encuentra en esta ventana, se utiliza la ventana que muestra el código del botón.

Esto se llama reciclaje de ventanas de código (modo Default).

Para utilizar varias ventanas de código para mostrar el código de varios objetos del mismo elemento, simplemente Cancel el reciclaje de las ventanas de código.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000143"></a>


### Implementar el reciclaje de ventanas de código
<a name="implementar_reciclaje_ventanas_codigo_ELTPARAGRAPHE000024"></a>

Reciclar las ventanas de código de un archivo WINDEV o WEBDEV (window, reporte, ....):

1. Visualizar uno de los códigos del elemento solicitado.

2. En la pestaña "Vista", en el grupo "Ayuda edición", seleccione la opción "Reutilizar misma ventana de código".



<a name="NOTE2_2"></a>


### Utilizar varias ventanas de código para el mismo elemento
<a name="utilizar_varias_ventanas_codigo_para_mismo_elemento_ELTPARAGRAPHE000046"></a>

Para visualizar varias ventanas de código para un elemento WINDEV, WINDEV Mobile o WEBDEV (window, reporte, ....):

1. Visualizar uno de los códigos del elemento solicitado.

2. En la pestaña "Vista", en el grupo "Ayuda edición", desmarque "Reutilizar misma ventana de código".

## Notas
<a name="notas_ELTTEXTE000173"></a>


### Utilización de varias ventanas de código para el mismo elemento WINDEV/WINDEV Mobile/WEBDEV (ventana, reporte, etc.)
<a name="utilizacion_varias_ventanas_codigo_para_mismo_elemento_windevwindev_mobilewebdev_ventana_reporte_etc_ELTPARAGRAPHE000070"></a>

Si el código del mismo elemento (código de un botón, código de una control, ...) es solicitado varias veces, la ventana del código correspondiente se vuelve a activar.

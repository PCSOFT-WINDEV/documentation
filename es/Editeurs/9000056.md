### Generar una aplicación Java con WINDEV
<a name="generar_una_aplicacion_java_con_windev_ELTPARAGRAPHE000011"></a>

WINDEV le permite generar una aplicación Java sin siquiera conocer el lenguaje Java. El método para desarrollar la aplicación es idéntico al método para desarrollar cualquier otra aplicación WINDEV.

Cuando se compila el proyecto WINDEV, las ventanas y las funciones WLanguage se convierten automáticamente en Java.

## Características específicas
<a name="caracteristicas_especificas_ELTTEXTE000251"></a>
Los siguientes elementos no están disponibles en la generación de Java:

- los componentes. 
	Observación: Los componentes internos están disponibles en Java. Si el componente interno incluye un análisis, este análisis será ignorado durante la ejecución.

- el uso de los servicios web

- el cuadro de diálogo Automatización OLE

- el uso y la generación de componentes de red

- la herencia múltiple

- la compilación dinámica

- la declaración "GOTO

- la sintaxis utilizada para especificar el nombre de una clase base delante de un método
	Ejemplo: MyBaseClass::MiMétodoVirtual

- el idioma externo

- reportes. Sin embargo, la impresión está disponible en WLanguage.
	Tiene la posibilidad de imprimir en formato PDF en WLanguage.

- las ventanas"popup

- las variables locales no soportan las indirecciones

- no se pueden encontrar varias estructuras con el mismo nombre en el mismo proyecto (incluso si están definidas en clases diferentes). 

- la gestión del formato RTF

- el UMC (User Macro-Code)




Las plantillas de tema de la versión 10 (y anteriores) están soportadas por las aplicaciones de Java.

Los cursores personalizados de ratón ".CUR" o ".ANI" no están soportados en Java.

**Estilos**: las siguientes funciones no están disponibles:

- Cortar las ventanas.

- Sombra para controles estáticos.

- Transparencia y animación de ventanas.

- Es posible que algunas fuentes específicas de Windows no sean compatibles con Java (se utilizará una fuente Default).

- Atributos de la fuente: subrayado y tachado.

- Cambiar el tamaño de los cuadros combinados.

- Uso de la plantilla del proyecto en las ventanas del sistema.

- Los botones no pueden tener texto multilínea.


Los estilos de Vista están disponibles en Java.

### Formatos Image
<a name="formatos_image_ELTPARAGRAPHE000152"></a>

Los siguientes formatos de Image están soportados en Java: JPEG, PNG, GIF, BMP.

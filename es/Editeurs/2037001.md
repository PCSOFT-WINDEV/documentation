
## Averigua la cuenta que ejecuta el protocolo AWP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## ¿Cómo proceder en IIS 5 o 6?
<a name="¿como_proceder_iis_5_6_ELTTEXTE000098"></a>
Para identificar la cuenta de Windows que ejecuta el protocolo AWP:

1. Abra el administrador de servicios de información de INTERNET (IIS) ("Iniciar .. Ejecutar .. .. inetmgr.exe" en Windows NT o Windows 2000).

2. Mostrar las propiedades del servicio "WWW".

3. Haga clic en la pestaña "Seguridad de directorios".

4. Mostrar los métodos de autenticación para el acceso anónimo (botón"Editar").




La cuenta de Windows corresponde al nombre de usuario utilizado para iniciar una sesión anónima (botón"Modificar").
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=iis5_compte_awp.gif)

## ¿Cómo proceder en IIS 7?
<a name="¿como_proceder_iis_7_ELTTEXTE000122"></a>
Para identificar la cuenta de Windows que ejecuta el protocolo AWP:

1. Abra el administrador de servicios de información de INTERNET (IIS) ("Iniciar .. Ejecutar .. .. inetmgr.exe" en Windows Vista).

2. Seleccione "Sitio Web de Default".

3. Seleccione "Autenticación" en la lista que se muestra.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=iis7_Authentification.gif)


4. En la pantalla que aparece, seleccione "Autenticación anónima".

5. Haz clic en el botón "Editar..." de la derecha.

6. Se muestra el nombre del usuario anónimo. Puede ser modificado si es necesario. Valide.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IIS7_Authentification2.gif)

## ¿Cómo proceder en Apache?
<a name="¿como_proceder_apache_ELTTEXTE000146"></a>
Para identificar la cuenta de Windows que ejecuta el protocolo AWP:

1. Inicie el servicio del panel de control.

2. Muestra las propiedades del servicio "Apache".

3. Se muestra la cuenta de Windows correspondiente:

	- la cuenta del sistema

	- otra cuenta de Windows.

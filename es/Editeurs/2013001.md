


## Autocompletado 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000200"></a>
El autocompletado está disponible para los comandos de WLanguage, los procedimientos, las variables, los archivos de datos, los elementos de los archivos de datos, las ventanas, los controles, los tipos de variables, etc.

Esta función de autocompletar reduce el riesgo de errores de escritura y ahorra mucho tiempo: no hay necesidad de buscar el nombre exacto de un Variable o control global, ¡se sugiere automáticamente!

La terminación de automatic está acostumbrada:

- ***completar la entrada actual*** proponiendo una lista de diferentes valores (nombre de la función, nombre de la propiedad, etc.). Todos los valores propuestos en la lista contienen la palabra actualmente escrita. 
	El contenido de la lista se actualiza cada vez que se realiza una modificación..
	Por ejemplo, al escribir "OKC", todas las funciones, controles, constantes y fragmentos de código, ... que contengan estos caracteres son sugeridos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=completion.gif)
 **Observación**: La función de autocompletar también está disponible cuando se modifica una función ya introducida en el editor de código. 
	Ejemplo: si desea reemplazar " **HReadFirst** " con " **HReadSeekFirst** ": cuando empiezas a escribir "Seek" entre "Read" y "First", la función de autocompletar muestra automáticamente "HReadSeekFirst". Al seleccionar la nueva función, la antigua se sustituye automáticamente.  

- ***proponer solo el valor de los parámetros posibles*** según la función WLanguage utilizada.
	Por ejemplo, al utilizar [HRead](../WDLang4/3044047.md), el primer parámetro esperado corresponde al nombre de un archivo de datos. La finalización de automatic sólo propondrá los archivos de datos del análisis associated con el proyecto actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=completion2.gif)


- ***seleccionar las combinaciones de constantes para un parámetro***. 
	Para ello, seleccione las opciones deseadas (con las teclas de flecha) y pulse la barra espaciadora para confirmar.. 
	Una vez confirmadas, todas las opciones seleccionadas se integran automáticamente en el código.
	Observación: Las constantes se separan en áreas de diferentes colores que se utilizan para identificar las constantes que pueden combinarse (o no) entre sí (dos constantes que se encuentran en un área de colores con un selector no pueden seleccionarse al mismo tiempo).. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Completion_cst.gif)


- ***iniciar los asistentes de código***. Estos asistentes le permiten introducir los diferentes parámetros de la función respondiendo a preguntas sencillas. 
	La opción &lt;xxx wizard&gt; indica que hay un asistente disponible. Para iniciar el asistente, seleccione esta opción.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=AssistantCode.gif)


- ***proponer los diferentes atributos de extensión disponibles*** en la línea de código actual.
	Un atributo de extensión corresponde a una directiva pasada al compilador. Estos atributos son poderosos y prácticos. Para obtener la lista de atributos de extensión disponibles en el elemento actual, simplemente escriba "&lt;".  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Completion_Attributs%20-%20HC%20N%B0001.gif)

	Para más detalles sobre los atributos disponibles de extension, véase [Lista de atributos de extension](../Motscles/1511025.md). 




**Observaciones**: 

- Para seleccionar una de las sugerencias: 

	1. Desplácese por la lista con las teclas de flecha. 

	2. Confirmar la selección: 

		- con la tecla Intro.

		- con la tecla Tab.




- Las listas desplegables se pueden cambiar de tamaño.

- **Si su procedimiento o función utiliza un elemento del proyecto (ventana, página, reporte, etc.)**, puede obtener la lista de los elementos deseados en las sugerencias de código. Para ello, añada un atributo con una extension específica en la declaración de los parámetros de la función o de la Procedure. Para obtener más información, consulte [Parámetros de una función o Procedure](../Motscles/1513001.md). 

- **Si está utilizando una variable de fuente de datos** , todo lo que tiene que hacer es agregar un atributo con un extension específico para asociarlo con un archivo de datos existente (o consulta). En este caso, la finalización de automatic le permitirá acceder a los artículos directamente. Para obtener más información, consulte [Variable de tipo Data Source](../WDLang4/1514053.md). 

- El editor de código también proporciona sugerencias de parámetros Input( asistidas, etc.). Para obtener más información, consulte [Editor de código: Entrada predictiva](../Editeurs/2013009.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000224"></a>


### Permitiendo la finalización de automatic
<a name="permitiendo_finalizacion_automatic_ELTPARAGRAPHE000093"></a>

Para permitir la finalización de automatic en su proyecto:

1. Abrir las opciones del editor de código: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. Seleccione la opción "Activar entrada predictiva tras una inactividad de" e indique el tiempo tras el cual debe proponerse el autocompletado.




**Observación**: Para desactivar la finalización de automatic, desmarque esta opción.

## Observaciones
<a name="observaciones_ELTTEXTE000248"></a>


### El lenguaje del código
<a name="lenguaje_del_codigo_ELTPARAGRAPHE000123"></a>

La lista de palabras clave en idiomas extranjeros propuesta por la finalización de automatic está disponible en inglés o en francés.

Para modificar el lenguaje de códigos usado por la finalización de automatic:

1. Abrir las opciones del editor de código: en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones del editor de código".

2. Seleccione el idioma del código en el cuadro combinado "Idioma código: ".



<a name="NOTE3_2"></a>


### La finalización de automatic sobre el tipo de variables
<a name="finalizacion_automatic_sobre_tipo_variables_ELTPARAGRAPHE000151"></a>

Cuando se declara un nuevo Variable, los diferentes tipos se proponen automáticamente. Simplemente escribe: "MiVariable es ".

## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
La selección de registros realizada por consulta no se muestra necesariamente en un reporte, en una ventana, ... Esta selección también se puede utilizar en la programación.

La mayoría de las funciones HFSQL (navegar, buscar, leer,...) se pueden utilizar tanto en los archivos de datos como en el resultado de la consulta.. Para utilizar estas funciones en una consulta, es necesario ejecutar primero la consulta (con [HExecuteQuery](../WDLang4/3044080.md)).

Por ejemplo, el *Cliente34* selecciona todos los clientes cuyo código postal comienza por 34. El *PrintCustomer* Procedure se utiliza para imprimir una factura proforma para cada cliente. El siguiente código se utiliza para navegar por el resultado de la consulta de *Customer34* e imprimir una factura para todos los clientes seleccionados.


```wl
// Run the query
ResExecute = HExecuteQuery(Customer34)
// Error while running the query?
IF ResExecute = True THEN
	// Browse the result of the query
	FOR EACH Customer34
	   	// Print the invoice for each record
		PrintCustomer(Customer34.CustomerLName, Customer34.CustomerFName, ...
				Customer34.CustomerAddress, Customer34.CustomerZipCode, ...
				Customer34.CustomerCity)
	END
ELSE
	 // Display the error message
	 Error("The query was not run.")
END
```

## Contraseña
<a name="contrasena_ELTTEXTE000116"></a>


- **Contraseña del análisis**: Si la consulta está asociada a un análisis protegido por contraseña en tiempo de ejecución, esta contraseña debe especificarse al abrir el análisis ([HOpenAnalysis](../WDLang4/3044106.md)) en la inicialización Process del proyecto.

- **Contraseña del archivo**: Si la consulta se basa en uno o varios archivos protegidos por contraseña, especifique las contraseñas con [HPass](../WDLang4/3044108.md) antes de ejecutar la consulta (con [HExecuteQuery](../WDLang4/3044080.md)).

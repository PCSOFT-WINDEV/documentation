<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000121"></a>
El panel "Imágenes" le permite ver y listar todas las imágenes en el directorio y subdirectorios del proyecto.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_images%20-%20HC%20N%B0001.gif)
 Desde el panel "Imágenes", puede: 

- abrir un Image en el editor de Image,

- crear un Image en el editor de Image,

- borrar un Image. 




Observación: El panel "Image s" se muestra automáticamente en el editor de Image. En este caso, este panel se utiliza para: 

- ver la vista previa y definir el zoom de la Image que se muestra en el editor de Image,

- mostrar los colores y configurar las diferentes herramientas utilizadas (grosor de la Line, opacidad, etc.),

- gestionar las capas. 


Para obtener más información, consulte [Editor de imágenes](../Editeurs/1410086603.md).

## Visualización del panel "Imágenes
<a name="visualizacion_del_panel_imagenes_ELTTEXTE000145"></a>
**Para visualizar el panel "Imágenes "**, vaya a la pestaña "Inicio", al grupo "Entorno", expanda "Paneles", seleccione "Paneles", y luego seleccione "Imágenes". 

Para más detalles sobre la gestión de los paneles, véase [Manejo de paneles acoplables](../Editeurs/2027001.md).

## Uso del panel "Imágenes
<a name="uso_del_panel_imagenes_ELTTEXTE000169"></a>
El panel "Imágenes" se utiliza para: 

- abrir, en el editor Image, un Image específico que se encuentra en el disco (en un directorio distinto al del proyecto actual). Para ello, haga clic en el icono ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_images%20-%20HC%20N%B0001%202.gif) y seleccione el archivo que se abrirá en el editor Image. 

- crear una nueva image en el editor de image (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_images%20-%20HC%20N%B0001%201.gif)). 

- manipular una imagen a través de el menú contextual: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_images%20-%20HC%20N%B0002.gif)


	- Modificar el Image: El Image se abre en el editor de Image para ser modificado. 

- Abrir ubicación: Se utiliza para mostrar el explorador de archivos de Windows en el directorio que contiene el archivo Image. 

- suprimir: Se utiliza para borrar el archivo Image. 




Observación: Cuando se mueve el Image, la punta de la herramienta indica el ruta actual del Image.

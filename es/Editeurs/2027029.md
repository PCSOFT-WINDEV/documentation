## Presentación
<a name="presentacion_ELTTEXTE000339"></a>
El panel "Explorador del proyecto" se utiliza para simplificar el entorno de desarrollo de WINDEV, WEBDEV y WINDEV Mobile.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletExplorateurProjet.gif)
 Este panel se utiliza para:

- enumerar los elementos del proyecto.

- enumerar los elementos de análisis.

- enumerar los procedimientos, las clases, etc.

- lista de descripciones externas.

- buscar en el proyecto actual.

## Características del panel del "Proyecto Explorador"
<a name="caracteristicas_del_panel_del_proyecto_explorador_ELTTEXTE000363"></a>


### Operaciones generales en el panel del "Proyecto Explorador".
<a name="operaciones_generales_panel_del_proyecto_explorador_ELTPARAGRAPHE000026"></a>

El panel "Explorador de proyectos" muestra los elementos del proyecto y, si es necesario, los "subelementos" de associated. 

Tienes la capacidad de arrastrar y soltar los elementos seleccionados hacia el entorno. Por ejemplo, puedes arrastrar y soltar un Variable global en una ventana para crear un control Campo de entrada.

También puedes arrastrar y soltar elementos en el panel "Explorador del proyecto"...: por ejemplo, un elemento puede ser trasladado a una carpeta personalizada o a un componente interno.

![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_synchro_explorateur.gif) permite posicionar el banda de selección en el panel "Explorador de proyectos" sobre el elemento actual del editor.
<a name="NOTE2_2"></a>


### Gestionar el proyecto y sus elementos
<a name="gestionar_proyecto_sus_elementos_ELTPARAGRAPHE000040"></a>

Todos los elementos del proyecto están agrupados en el panel "Explorador del proyecto".. Un doble clic sobre el nombre del elemento permite visualizarlo en el entorno.

**El menú contextual del proyecto** (clic con el botón derecho del ratón sobre la "P" Icon) permite realizar las operaciones más habituales sobre el proyecto:

- Mostrar el proyecto description,

- Recompilar el proyecto,

- Crear un nuevo elemento, 

- Enumere todos los elementos del proyecto,

- ...




Cada elemento del proyecto incluye su propio menú emergente que propone todas las características disponibles para este elemento. Por ejemplo, en una ventana o página, puedes: 

- abrir el elemento en el editor correspondiente,

- probar elemento, 

- duplicar el elemento,

- ver los códigos del elemento, etc.




**El elemento "Configuración"** se utiliza para gestionar el [configuraciones de proyecto](../Editeurs/9000030.md). El menú emergente se utiliza para añadir una nueva configuración o para gestionar las configuraciones existentes. Los elementos que no pertenecen a la configuración seleccionada son Grayed en el panel "Explorador del proyecto".

**Para las ventanas e informes**, el panel "Explorador del proyecto" enumera los controles en el elemento seleccionado, así como las variables locales y los procedimientos locales.

**Para las consultas,** el panel "Explorador del proyecto" enumera los elementos de resultado de la consulta así como sus parámetros.

**La categoría "Procedimientos "** contiene los conjuntos de procedimientos del proyecto o los de los componentes situados en el proyecto. Las constantes, las variables globales y los procedimientos se enumeran para cada set de procedimientos. El panel "Explorador del proyecto" también permite crear procedimientos o conjuntos de procedimientos.

**Para las clases**, el panel "Explorador del proyecto" lista los miembros y métodos. El panel "Explorador del proyecto" también permite crear clases o métodos.

**Para los componentes internos**, el panel "Explorador del proyecto" enumera los componentes internos y sus elementos. Los elementos se enumeran en una estructura de árbol, agrupados por tipo (ventanas, informes, etc.). 
Sólo se muestran las ramas que contienen elementos. 

El panel "Explorador del proyecto" también enumera las variables globales del proyecto y los archivos y elementos de análisis.
<a name="NOTE2_3"></a>


### Modelado inverso y referencias cruzadas
<a name="modelado_inverso_referencias_cruzadas_ELTPARAGRAPHE000085"></a>

El menú emergente de los elementos de proyecto se utiliza para realizar un modelado inverso y/o para mostrar las referencias cruzadas.
<a name="NOTE2_4"></a>


### Gestionar el análisis
<a name="gestionar_analisis_ELTPARAGRAPHE000092"></a>

El panel "Explorador del proyecto" muestra el análisis del proyecto actual, es decir, los archivos de datos y sus elementos. El análisis visualizado corresponde al último análisis generado que puede ser utilizado en la aplicación.

**Recordatorio**: El panel "Análisis" muestra el análisis que se está editando y se utiliza para manipular la description de los archivos o elementos de datos.

El editor de análisis puede abrirse desde el panel "Explorador de proyectos", a través del menú contextual de los archivos de datos ("Abrir análisis"). El editor de análisis se abre con el archivo de datos seleccionado en el panel "Explorador del proyecto".. Este archivo de datos también se selecciona en el panel ["Análisis"](../Editeurs/2027012.md).. 

Para crear los controles correspondientes, todo lo que tiene que hacer es arrastrar y soltar los elementos hacia una ventana, página o reporte.

**Observación**: el panel "Explorador del proyecto" muestra una punta de herramienta al pasar el cursor sobre una tecla compuesta. Esta información sobre herramientas contiene los componentes de la clave compuesta. El signo más o menos que se encuentra delante del nombre del componente indica la dirección de clasificación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Volet_Clecomp.gif)

### Realice una búsqueda con el panel "Explorador del proyecto"...
<a name="realice_una_busqueda_con_panel_explorador_del_proyecto_ELTPARAGRAPHE000115"></a>

Para realizar una búsqueda con el panel "Explorador del proyecto":

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Rech_Loupe.gif) Icon para iniciar la búsqueda. 

2. Aparece la ventana de búsqueda de elementos del proyecto.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CTRL_E.gif)


3. Introduzca el nombre (o parte del nombre) del elemento que está buscando. La búsqueda se realiza en todos los nombres de los elementos del proyecto. El resultado se muestra en la ventana de búsqueda. Haciendo doble clic en el elemento se abre en el editor correspondiente. 

4. Para una búsqueda más exhaustiva, consulte "Buscar también los subelementos
(controles, procedimientos, etc.)": el resultado de la búsqueda tiene en cuenta: 

	- todas las referencias a la palabra buscada en el código de WLanguage.

	- todas las referencias a la palabra buscada en el contenido de control s (leyenda de control, notas, etc.).

### Categoría "Otros".: incluidos los elementos del ejecutable, la biblioteca, etc.
<a name="categoria_otros_incluidos_los_elementos_del_ejecutable_biblioteca_etc_ELTPARAGRAPHE000137"></a>

Puede incluir archivos (archivos de datos HFSQL, imágenes, sonidos, etc.) directamente en el ejecutable o archivo generado. Esto le permite utilizar recursos que no deben estar Visible en el disco del usuario final. 

Para ello, simplemente incluya estos recursos en el proyecto a través de la carpeta "Otros", en el panel "Explorador del proyecto".. 

Observación: Los recursos de la categoría "Otros" incluidos en la biblioteca ejecutable (archivo WDL) pueden extraerse con [fExtractResource](../WDLang1/1000019509.md). 

Las siguientes características se utilizan para simplificar las operaciones realizadas en la carpeta "Other: 

- El "arrastrar y soltar" de los elementos a incluir en la carpeta "Otro" se puede realizar desde el explorador de archivos de Windows.. 

- Un menú emergente en el menú "Otros" se utiliza para añadir o eliminar elementos.. 

- Una carpeta "Other" también está disponible en cada componente interno, lo que le permite incluir elementos en los componentes internos compartidos.. 

- La función utilizada para realizar una búsqueda en el proyecto también navega por la carpeta "Other" del proyecto.. 



<a name="NOTE2_6b"></a>


### "Categoría "Descripciones externas: incluir descripciones externas en el proyecto
<a name="categoria_descripciones_externas_incluir_descripciones_externas_proyecto_ELTPARAGRAPHE000162"></a>

La categoría "Descripciones externas" contiene las diferentes descripciones importadas en el proyecto: 

- Archivos XML, JSON, YAML. Para obtener más información, consulte: 

	- [xlsDocument](../WDLang5/1000017464.md)

	- [YAML](../Motscles/1000024438.md)

	- [JSON](../Motscles/1000023611.md)




- Archivo XSD. Para obtener más información, consulte [Gestión del XSD](../WDLang5/1000019170.md). 

- Módulo Python. Para obtener más información, consulte [Python: Llamadas nativas](../WDLang6/1410088133.md). 




El menú contextual de esta categoría permite importar o actualizar estas descripciones.

### Configurar la pantalla en el panel "Explorador del proyecto"
<a name="configurar_pantalla_panel_explorador_del_proyecto_ELTPARAGRAPHE000190"></a>

Puede configurar las opciones de visualización y clasificación utilizadas en el panel "Explorador del proyecto".: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_propri%E9t%E9_Explorateur.gif).

2. En la ventana que aparece, seleccione las opciones deseadas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WX_Explo_prj_Option.gif)





Las opciones de visualización son las siguientes: 

- Opciones de visualización en el panel "Explorador del proyecto":

	- **Ordenar por orden alfabético**: Todos los elementos del proyecto se muestran en su carpeta ordenados alfabéticamente (opción Default).

	- **Ordenar por fecha de modificación**: Todos los elementos del proyecto se muestran en su carpeta ordenados en orden descendente según su última fecha de modificación.




- Procedimientos y métodos

	- **Ordenar por orden alfabético**: Los procedimientos y los métodos se muestran en orden ascendente según su nombre (opción Default).

	- **Ordenar por fecha de modificación**: Los procedimientos y los métodos se muestran en orden descendente según su última fecha de modificación.

	- **Ordenar por visibilidad (pública, protegida, privada)**: Los procedimientos y los métodos se clasifican y se muestran según su atributo de visibilidad: Público, Protegido y Privado.

	- **Ordenar por grupo de colores**: Los procedimientos y los métodos se clasifican y se muestran de acuerdo con el Color que se les asignó.




- Clases

	- **Mostrar elementos de las clases base**: Esta opción se utiliza para mostrar (o no) los elementos pertenecientes a las clases base (en caso de herencia o override por ejemplo.

	- **Mostrar solo elementos públicos**: Esta opción se utiliza para mostrar los elementos públicos que se encuentran en las clases (miembros, métodos).




- Campos del análisis

	- **Ordenar por orden lógico**: Los elementos de cada archivo de datos se muestran en el orden en que han sido creados al escribir el archivo description.

	- **Ordenar por orden alfabético**: Los elementos de cada archivo de datos se muestran en orden ascendente según su nombre.




- Sincronización : 

	- **Sincronizar automáticamente el explorador con el elemento que está siendo editado**: Esta opción se utiliza para seleccionar automáticamente el elemento que se está editando en el panel "Explorador del proyecto".. Por ejemplo, si se muestra una ventana en el editor de ventanas, su nombre se selecciona automáticamente en el panel "Explorador del proyecto"..

## Mostrando el panel del "Proyecto Explorador"
<a name="mostrando_panel_del_proyecto_explorador_ELTTEXTE000429"></a>
**To mostrar el panel "Explorador del proyecto"**: en la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Explorador de proyectos".

Para obtener más información sobre los paneles, consulte [Manipular paneles acoplables](../Editeurs/2027001.md).

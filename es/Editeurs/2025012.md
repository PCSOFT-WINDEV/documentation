<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000109"></a>
La generación de una biblioteca consiste en seleccionar los elementos que maneja esta aplicación (ventanas, informes, clases, conjuntos de procedimientos, consultas, archivos de datos...) así como sus dependencias (por ejemplo, las imágenes utilizadas en las ventanas). Para más detalles, consulte la ayuda sobre el programa [libraries](../Editeurs/2030022.md).

Esta pantalla se muestra al crear el ejecutable si ha elegido utilizar la biblioteca principal de la aplicación.

**Observación**: Para crear el ejecutable, utilice uno de los siguientes métodos: 

- En la pestaña "Proyecto", en el grupo "Generación", haga clic en "Generar". 

- Expanda ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_generation_exe.gif) en los botones de acceso rápido y seleccione "Generar ejecutable (exe) Windows de 32 bits" o "Generar ejecutable (exe) Windows de 64 bits".




La creación del ejecutable asistente permite definir el modo de funcionamiento del ejecutable y los modos de utilización de la biblioteca.

La pantalla "Modo de funcionamiento del ejecutable" sirve para indicar si se utiliza una biblioteca. Si ha optado por utilizar una biblioteca (biblioteca externa o integrada), la pantalla "Generar una biblioteca" le permite seleccionar los elementos que deben incluirse en la biblioteca.

Recordatorio: 

- Biblioteca incluida en el ejecutable: Esta es la opción más común. El archivo WDP correspondiente a la librería se incluye en el ejecutable. 

- Biblioteca utilizada externamente. En este caso, el archivo de la biblioteca es un archivo WDL generado junto al ejecutable.

## Generando una biblioteca: elección de los elementos de la biblioteca
<a name="generando_una_biblioteca_eleccion_los_elementos_biblioteca_ELTTEXTE000133"></a>
La biblioteca y el proyecto actual tendrán el mismo nombre. Es posible:

- **Modificar el nombre de la biblioteca.**: simplemente modificar el nombre directamente.... 

- **Agregar elementos** (botón "Agregar"): 
	Se puede añadir cualquier tipo de archivo a la lista de elementos insertados en la biblioteca.: imágenes, archivos de texto, etc.

- **Eliminar elementos**:  (botón "Eliminar")
	Los archivos correspondientes y sus dependencias no se incluirán en la biblioteca.

- **Cree la biblioteca a partir de una biblioteca existente description** (botón "WDU"). 
	Al crear una biblioteca WINDEV o WINDEV Mobile, se crea automáticamente un archivo ".WDU" con el mismo nombre. Este archivo contiene todas las referencias de los elementos incluidos en la biblioteca. Todo lo que tiene que hacer es seleccionar el archivo WDU que contiene los elementos a tener en cuenta.

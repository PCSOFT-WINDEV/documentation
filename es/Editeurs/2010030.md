


## Creando una ayuda de automatic
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000102"></a>
Las funciones de la aplicación automatic (AAF) le permiten añadir funciones a su aplicación sin tener que escribir un solo Line de código. Todo lo que tiene que hacer es comprobar las opciones relevantes para beneficiarse de estas características.

Usted tiene la capacidad de generar una ayuda para su aplicación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000126"></a>
WINDEV incluye un editor de ayuda que le permite generar un sistema de ayuda completo para su aplicación.

El editor de ayuda WINDEV genera un esqueleto de ayuda para ayudarle a construir un sistema de ayuda.. Se describen los aspectos principales de sus aplicaciones.

Encontrarás páginas de presentación:

- **las funcionalidades del control Tabla**: búsqueda, columnas ordenables, menús contextuales, etc.. 
	Esta página se genera automáticamente si su aplicación incluye al menos una ventana con un control Tabla y si elige Document este tipo de ventana.

- **las características de la ventana del visor reporte**.
	Esta página se genera automáticamente si su aplicación incluye al menos una reporte y si decide documentar al menos una reporte.




Esta generación de automatic aumenta el valor de sus aplicaciones facilitando a sus usuarios el aprendizaje de su uso.

**Observación**: En la versión actual, sólo se documentan automáticamente la utilización de controles Tabla y la ventana de visualización de reporte. En las próximas versiones se incluirán otras páginas generadas automáticamente.

## Creación de una ayuda
<a name="creacion_una_ayuda_ELTTEXTE000150"></a>
Las páginas de ayuda de automatic se generan al crear la ayuda. Estas páginas se incluyen de acuerdo a las elecciones hechas en la asistente para la creación de ayuda. Por ejemplo, la página description del visor de reporte sólo se incluirá si eliges documentar al menos un reporte.

Para generar una ayuda:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. La ventana de creación de nuevos elementos se abre: haga clic en "Arquitectura" y luego en "Ayuda".

2. Una asistente te ayuda a crear el sistema de ayuda. Especificar la información solicitada hasta llegar al avión utilizado para seleccionar las ventanas a Document.

3. Si seleccionas al menos una ventana que contenga un control Tabla, se genera automáticamente una página de ayuda para explicar las características de este control.

4. En el siguiente paso, puede seleccionar los informes a Document. Seleccionando al menos una reporte, se genera automáticamente la página de ayuda para la ventana del visor de reporte.

5. Finalice el asistente: se genera su archivo de ayuda.




Observará que se ha incluido una sección "Varios" en el resumen del archivo de ayuda. Esta sección contiene las páginas de ayuda que se generaron automáticamente.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fonctionnalites_auto_Aide2.gif)

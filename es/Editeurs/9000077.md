## Presentación
<a name="presentacion_ELTTEXTE000228"></a>
WINDEV le permite crear programas batch en Linux (aplicaciones en modo consola, sin GUI).

Estos programas pueden ser útiles para automatizar el archivo copy en un servidor Linux por ejemplo, ...

## Creación de un proyecto Linux
<a name="creacion_proyecto_linux_ELTTEXTE000252"></a>


### Creación de un proyecto Linux
<a name="creacion_proyecto_linux_ELTPARAGRAPHE000027"></a>

Un proyecto WINDEV ejecutado en Linux es un proyecto sin interfaz: en la mayoría de los casos, contiene conjuntos de procedimientos, clases, ... que se inician desde el código de inicialización del proyecto.

Parte de la información se puede mostrar en modo consola en el ordenador en tiempo de ejecución: para ello, utilice [dbgStandardOutput](../WDLang1/3014020.md) y especifique el texto a mostrar.

Para crear un ejecutable de Linux, puede hacerlo:

- crear un ejecutable de Linux directamente.

- crear una configuración de proyecto específica para Linux.



<a name="NOTE2_2"></a>


### Creación de un ejecutable de Linux directamente
<a name="creacion_ejecutable_linux_directamente_ELTPARAGRAPHE000044"></a>

Una vez que su proyecto contiene todos los elementos necesarios para que su ejecutable de Linux funcione, el ejecutable de Linux puede ser creado directamente:

1. Seleccione "Taller .. Executable .. Genera el ejecutable de Linux (consola)".

2. La asistente para crear un ejecutable de Linux se inicia.

3. Escriba el nombre del ejecutable. El mismo nombre se le da automáticamente a la biblioteca associated con el ejecutable.

4. Especifique las bibliotecas framework necesarias para la ejecución.

5. Seleccione (si es necesario) los componentes utilizados por el ejecutable.

6. Especifique el mensaje de error que se mostrará en la consola cuando se produzca un error de ejecución.

7. Especifique si el ejecutable debe guardarse en el Centro de reutilización.

8. Especifique si se debe realizar una copia de seguridad del proyecto.

9. Validar la creación del ejecutable.




Al crear el ejecutable:

- El directorio EXE del proyecto contiene todos los archivos necesarios para ejecutar el programa Linux: (\*.exe), biblioteca (\*.wdl), Linux framework (archivos \*.so). 

- La configuración del proyecto actual se configura automáticamente para generar un ejecutable de Linux.

### Creación de una configuración de proyecto específica para el ejecutable de Linux
<a name="creacion_una_configuracion_proyecto_especifica_para_ejecutable_linux_ELTPARAGRAPHE000068"></a>

Para crear un ejecutable de Linux, WINDEV también le permite crear una configuración de proyecto específica para Linux. En este caso, todos los elementos necesarios para crear un ejecutable de Linux se pueden agrupar en esta configuración de proyecto. Para generar el ejecutable de Linux, todo lo que tiene que hacer es seleccionar la configuración correspondiente en el editor y luego crear el ejecutable.

**Para crear una nueva configuración de proyecto para un ejecutable de Linux:**

1. Visualizar el gestor de configuración ("Proyecto .. Configuraciones de proyecto... Gestionar las configuraciones").

2. Haga clic en el botón "Nueva configuración".

3. En la asistente, especifique el tipo de generación: "Ejecutable Linux".

4. Seleccione los elementos que deben incluirse en esta configuración. 
	**Atención**: Los ejecutables de Linux operan en modo consola y por lo tanto no muestran ninguna ventana.

5. Seleccione (si es necesario) los componentes utilizados por la configuración del proyecto.

6. Valide.




**Para seleccionar la configuración del proyecto para el ejecutable de Linux:**

- Visualizar el gestor de configuración ("Proyecto .. Configuraciones de proyecto... Gestión de configuración") y seleccionar la configuración deseada.

- Mostrar el panel "Explorador de proyectos" ("Mostrar... Barras de herramientas... Paneles... Explorador de proyectos") y seleccionar la configuración deseada en este panel.




**Observación**: Cuando la configuración del proyecto específica de Linux se muestra en el editor, la pestaña "Errores de compilación" muestra los errores de compilación específicos de la compilación del ejecutable de Linux.

**Para empezar a crear el ejecutable de Linux:**

1. Seleccione "Taller .. Executable .. Genera el ejecutable de Linux (consola)".

2. La asistente para crear un ejecutable de Linux se inicia.

3. Escriba el nombre del ejecutable. El mismo nombre se le da automáticamente a la biblioteca associated con el ejecutable.

4. Especifique el mensaje de error que se mostrará en la consola cuando se produzca un error de ejecución.

5. Especifique si el ejecutable tendrá en cuenta las bibliotecas de parches.

6. Especifique los idiomas soportados por el ejecutable.

7. Indique los elementos que desea incluir en la biblioteca. 

8. Seleccione (si es necesario) los componentes utilizados por el ejecutable.

9. Especifique la información sobre la versión ejecutable.

10. Seleccione la distribución de Linux que desea usar.

11. Especifique si el ejecutable debe guardarse en el Centro de reutilización.

12. Validar la creación del ejecutable.

## Ejecutando la prueba del ejecutable de Linux y distribuyéndolo
<a name="ejecutando_prueba_del_ejecutable_linux_distribuyendolo_ELTTEXTE000288"></a>
Para ejecutar la prueba del ejecutable de Linux creado con WINDEV y distribuirlo: 

1. Conéctese al ordenador Linux a través de la aplicación "WinScp".

2. copy los archivos \*.exe, \*.WDL y \*.SO que se encuentran en el directorio EXE del proyecto al ordenador Linux.

3. Cambie los atributos del programa para darle los derechos de ejecución (por ejemplo: chmod + x myapp).

4. Ejecute el programa (por ejemplo, para ejecutar el ejecutable desde el directorio actual: ./myapp).

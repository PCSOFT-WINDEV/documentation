
## Historia de las versiones
			



## Presentación
<a name="presentacion_ELTTEXTE000108"></a>
La historia de las versiones de configuración es una característica avanzada de [la configuración de red](../Editeurs/2028085.md). 

Esta característica se utiliza para mantener en el servidor de reference una o más versiones anteriores de la aplicación. Esto permite al usuario volver a una versión anterior si se produce un problema.. 

El historial de versiones también se utiliza para instalar en el servidor la siguiente versión actualizada de una aplicación sin enviarla a todos los clientes.. Esto se utiliza para distribuir una versión a los "probadores beta" antes de que esté disponible para todos.

**Observación**: El historial de versiones sólo se aplica a las configuraciones en la red local.. El historial de versiones no está disponible para las configuraciones de red por IP.





## Configurar el historial de versiones
<a name="configurar_historial_versiones_ELTTEXTE000132"></a>
El historial de la versión se configura al instalar la versión reference de la aplicación.

Durante la puesta en marcha se visualiza el siguiente plano:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_Historique_version.gif)


**Para habilitar el historial de versiones**Todo lo que tiene que hacer es marcar la casilla correspondiente.

Se pueden configurar dos parámetros:

- El número de versiones que se almacenarán (además de la versión actual). Si este parámetro es igual a 0, el número de versiones almacenadas es ilimitado.

- El tamaño global de la historia de las versiones (en MB). Si este parámetro es igual a 0, el espacio en disco utilizado para almacenar el historial de versiones será ilimitado.. El tamaño aproximado de una versión se especifica a título informativo.




Si se especifican los dos parámetros límite, se utiliza el más bajo de los dos límites.





## Gestión del historial de versiones
<a name="gestion_del_historial_versiones_ELTTEXTE000156"></a>
La historia de las versiones se gestiona a través del Centro de Aplicaciones de Red de Control (WDAdminExe.exe). El Centro de Control de Aplicaciones de Red está instalado en el subdirectorio INSTALL de la configuración de reference de la aplicación.

La pestaña para gestionar las versiones es la siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_HVersion_adm.gif)


1: Versión actualmente visualizada por los ordenadores cliente.

2: Lista de versiones encontradas en la configuración de reference. La ayuda para el historial de la versión seleccionada se propone (si está disponible) en la sección derecha de la ventana.

3: Se pueden realizar las siguientes operaciones para cada versión:

- **Prohibir una versión**: La versión seleccionada será prohibida. Los usuarios ya no pueden volver a esta versión.. El ordenador cliente desplegado en esta versión se verá obligado a actualizarse cuando se inicie la aplicación la próxima vez.

- **Permitir una versión**: La versión seleccionada será permitida. Un usuario puede elegir cambiar la versión actual instalada en el equipo mediante el menú "?" (consulte [El menú](../Editeurs/9000070.md) para obtener más detalles). Esta opción permite al usuario volver a la versión anterior si se produce un problema (si esta versión anterior funcionaba correctamente).

- **Forzar la versión**: La versión seleccionada será forzada en todos los ordenadores cliente cuando se inicie la aplicación la próxima vez, incluso si se instala una versión más reciente en estos ordenadores.. Esta opción se utiliza para hacer que la versión instalada en los ordenadores cliente "avance" o para hacer que los probadores beta "regresen" a una versión "oficial".

- **Liberar la versión**: Esta opción se utiliza para Cancel los efectos de "Forzar la versión".




4: Se usa para habilitar una versión. Al actualizar la versión de reference, tienes la posibilidad de solicitar la activación de la versión al asistente de instalación.

5: Se utiliza para borrar una versión anterior.

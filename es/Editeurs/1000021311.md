
## Configuración de un servidor BaaS (Groupware Usuario y Tienda privada)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000166"></a>
Varias funciones de WINDEV, WEBDEV y WINDEV Mobile utilizan un servidor BaaS: 

- Groupware de usuario en modo BaaS (WINDEV, WEBDEV, WINDEV Mobile),

- Tienda privada para aplicaciones WINDEV y WINDEV Mobile. 




El servidor BaaS utilizado por estas funciones puede ser: 

- una plataforma de desarrollo **PCCLOUD** (solución llave en mano de pago). Para obtener más información, consulte [PCSCloud](https://pcscloud.net/UK/cloud_development.awp). 

- un servidor personalizado. 




Esta ayuda Page explica los diferentes pasos para configurar un servidor BaaS: 

- [Etapa 1: Instalación de un servidor HFSQL](#NOTE2_1). 

- [Etapa 2: Instalación del servidor BaaS reservado](#NOTE3_1).




Observaciones: 

- En el equipo debe estar instalado un servidor web (IIS, Apache u otro) en el que se instalará el servidor reservado. El servidor web debe ser accesible con autenticación básica. 

- En el caso de las aplicaciones móviles (Tienda privada o User Groupware), el servidor web debe: 

	- cumplir con el protocolo HTTPS con un Certificate emitido por una autoridad de confianza reconocida para dispositivos móviles iOS y Android,

	- permitir un tamaño máximo del contenido de las respuestas HTTPS superior al tamaño de las instalaciones (IPA y APK).




- El servidor HFSQL y el servidor reservado se pueden instalar en dos equipos diferentes. 

- El servidor reservado puede instalarse en un servidor Web que también aloje un servidor de aplicaciones WEBDEV.

## Etapa 1: Instalación del servidor HFSQL
<a name="etapa_1_instalacion_del_servidor_hfsql_ELTTEXTE000190"></a>
El programa de configuración del servidor HFSQL está disponible en el paquete de configuración WINDEV o WINDEV Mobile. Si no posee este paquete de instalación, la instalación del servidor HFSQL también se puede encontrar en nuestro sitio ([http://www.windev.es](https://www.windev.com)).

Para instalar el servidor HFSQL desde el paquete de instalación WINDEV/WINDEV Mobile:

1. Inicie el programa de instalación de WINDEV/WINDEV Mobile.

2. Seleccione "Configuración del servidor HFSQL".

3. A continuación, seleccione "Instalar o actualizar un servidor HFSQL".

4. Acepte el acuerdo de licencia.

5. Elija la plataforma ("Para Windows en este ordenador").

6. Si los servidores HFSQL ya están instalados en el ordenador actual, seleccione "Instalar un nuevo servidor".

7. Seleccione el directorio de instalación y especifique el nombre del servidor y el puerto. Por defecto, el número de puerto utilizado es 4900. 

8. Especificar el nombre y la contraseña del usuario con la gestión que será creada por Default en el servidor HFSQL. Por Default, este usuario es "Admin" y no tiene contraseña.

9. Instalar (si es necesario) el Centro de Control HFSQL si éste no se encuentra o no es accesible desde su ordenador. 

10. El asistente propone configurar el envío de notificaciones para identificar las disfunciones del servidor en tiempo real.. Seleccione la opción deseada. 

11. El asistente propone habilitar la autenticación a través del directorio Active. Habilite esta opción si es necesario.

12. Validar (o no) la participación en la mejora del producto permitiéndonos recopilar información sobre el uso del producto.. Esta recogida opcional y anónima permite a PC SOFT mejorar las características del producto.

13. La configuración se realiza. Por Default (si no ha cambiado la configuración del administrador), para conectarse al servidor en modo de administrador, debe utilizar el usuario "Admin" sin contraseña.
	Observación: Por razones de seguridad, no olvide cambiar la contraseña de administrador.

## Etapa 2: Servidor reservado instalación (WDBaaS)
<a name="etapa_2_servidor_reservado_instalacion_wdbaas_ELTTEXTE000237"></a>
El servidor BaaS reservado es un servidor de aplicaciones WEBDEV reservado para el User Groupware y el Tienda privada. La configuración del servidor reservado es común a User Groupware y a Tienda privada. 

Para instalar el servidor de Grupos de Usuarios Reservados/Tienda privada: 

1. Inicie el servidor reservado instalación (WX28 0PACKRSRV.exe, situado en el subdirectorio "Install\\WDServeurR" del directorio de instalación de WINDEV o WINDEV Mobile).

2. Valide la licencia. 

3. Seleccione el módulo a instalar: " Instalar módulos para el groupware usuario y el Tienda privada". Pase a la etapa siguiente.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_serveur_store%20-%20HC%20N%B0001.gif)


4. En caso necesario, seleccione la versión del servidor de aplicaciones reservado que desea instalar (32 o 64 bits). Pase a la etapa siguiente.

5. Seleccione el directorio de instalación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_serveur_store%20-%20HC%20N%B0002.gif)
Pase a la etapa siguiente.

6. Especifique las características del servidor HFSQL utilizado para los módulos WDBaaS (User Groupware/Tienda privada). 
	Observación: Este servidor HFSQL ha sido instalado previamente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_serveur_store%20-%20HC%20N%B0003.gif)
 Estas características son: 

	- Nombre del servidor HFSQL, 

	- Puerto. 

	- Nombre y contraseña del administrador del servidor HFSQL. Esta información es necesaria para configurar automáticamente la base de datos WDBaaS y el usuario associated. 




7. Pruebe la conexión para verificar los parámetros. 

8. Especificar los parámetros específicos de la base de datos WDBaaS utilizada por el User Groupware/Tienda privada: 

	- el nombre de la base de datos dedicada: WDBAAS por Default. 

	- el nombre de usuario y la contraseña de la base de datos. 




9. Especifique la contraseña para los archivos de datos WDBaaS (existentes o por crear).

10. Especifique la contraseña del administrador del sitio WDBaaS (Usuario: Admin).

11. Esta base de datos y el usuario se crearán automáticamente. 

12. Pase a la etapa siguiente.

13. especificar si se requiere una conexión SSL para los sitios de administración remota. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Install_serveur_store%20-%20HC%20N%B0004.gif)


14. Pase a la etapa siguiente.

15. El resumen de instalación se muestra. 

16. Pase a la etapa siguiente.   

17. La asistente comprueba si hay un servidor Web en el ordenador de la instalación. 

	- Si no hay ningún servidor web instalado en el equipo, el asistente le pide que instale y configure un servidor para poder utilizar el servidor reservado para el User Groupware y/o Tienda privada. 

	- Si existe un servidor web en el equipo, el asistente lo configura para que pueda utilizar el servidor reservado para el User Groupware y/o el Tienda privada. 




18. Al final de la instalación, el asistente muestra un resumen: 
	
	```txt
	WDBaaS for the User Groupware and the Private Store has been installed.
	WDBaaS server connection token to be provided to the environment development: 
	WDXxxxxxxxxxxxxxxxxxxxxx
	
	The HFSQL connection parameters are: 
	- Server: xxx
	- Database: xxx
	- User xxx - Password: xxx
	
	File password: xxx
	```

	**Copie y conserve esta información**. Debe utilizarse el Token para conectarse al servidor WDBaaS: 

	- en la ventana de configuración de User Groupware en su proyecto WINDEV.  

	- en la ventana User Groupware / Tienda privada settings de su proyecto WINDEV Mobile.

	- en la ventana de configuración de User Groupware en su proyecto WEBDEV.  


 **Importante**: La configuración del servidor reservado se almacena en el siguiente archivo: "C:\\ProgramData\\WDBaaS\\GPUConnection.Default.xml"
	Este archivo debe añadirse al plan de copias de seguridad del servidor. Este archivo tendrá que ser restaurado si el servidor es reinstalado o duplicado en una nueva máquina física o virtual con la misma configuración.

19. Finalice el asistente. Puede iniciar directamente el sitio de administración WDBaaS de Tienda privada. 




Para más detalles sobre el uso de la Tienda privada, véase: 

- [Tienda privada para aplicaciones WINDEV](../Editeurs/9000203.md).

- [Tienda privada para aplicaciones móviles](../Editeurs/1410086652.md).




Observación: Si el nombre de la base de datos es incorrecto, puede reiniciar el instalación del servidor reservado para especificar el nombre correcto de la base de datos y conservar los identificadores.

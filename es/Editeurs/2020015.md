


## Instalación de WINDEV/WEBDEV/WINDEV Mobile: Cumplir con el estándar de directorios
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000179"></a>
Las diferentes versiones de Windows proponen un estándar en cuanto a la ubicación de algunos archivos y el uso de directorios.

A partir de la versión 14, la instalación de WINDEV, WEBDEV y WINDEV Mobile cumple con esta norma. Durante la instalación, puedes:

- Instalar el producto seleccionado en un solo directorio. Este modo instalación corresponde al "Estándar" instalación.

- Instale el producto cumpliendo con el estándar de Windows.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>

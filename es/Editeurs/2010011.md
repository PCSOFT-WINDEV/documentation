<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000092"></a>
El editor de ayuda le permite definir estilos de párrafos. Estos estilos contienen todas las características de un párrafo:

- características de la fuente (nombre, tamaño, peso, subrayado, ....).

- características del párrafo (sangrías, viñetas, alineación, espaciado, ....).




Una vez definidos, estos estilos pueden utilizarse para todas las páginas del sistema de ayuda.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo gestionar los estilos?
<a name="¿como_gestionar_los_estilos_ELTTEXTE000116"></a>
WINDEV le da la capacidad de:

- crear, modificar o eliminar un estilo.

- aplicar un estilo a un párrafo.
	Para aplicar un estilo a un párrafo, seleccione el párrafo y seleccione el estilo correspondiente en el cuadro combinado de estilos:
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Combo_style.gif)


- disociar un estilo: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Disociar estilos".
	Esta opción le permite a Cancel la asociación entre un estilo y un párrafo. En este caso, el cuadro combinado de estilos está vacío para este párrafo. Entonces, este párrafo puede ser associated con un estilo.


## Transformar un proyecto WINDEV en móvil: una auditoría detallada
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000174"></a>
Transformar las aplicaciones WINDEV en móviles es muy fácil.

WINDEV Mobile incluye todos los elementos de los proyectos WINDEV: ventanas, consultas, código,, ... todo sin salir del editor de WINDEV (por supuesto, es necesario tener una licencia de WINDEV Mobile).

Claro, las ventanas deben cambiar de tamaño para adaptarse a las pantallas de teléfonos y tabletas.

¡Y también hay adaptaciones de código para funciones no soportadas en teléfonos móviles, por ejemplo la ausencia de un ratón en una tableta!

Para ayudarle, una asistente analiza el código de su proyecto e indica con precisión las líneas de código a modificar.

## ¿Cómo transformar un proyecto WINDEV en móvil?
<a name="¿como_transformar_proyecto_windev_movil_ELTTEXTE000198"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000025"></a>

Para transformar un proyecto WINDEV en móvil, simplemente: 

- Abrir el proyecto WINDEV en WINDEV Mobile. WINDEV Mobile propondrá entonces crear una configuración de proyecto móvil. 

- Si utiliza WINDEV y una versión incorporada de WINDEV Mobile, simplemente cree una configuración específica para la plataforma deseada.. 




La auditoría de transformación móvil le ofrece todos los consejos necesarios para transformar sin problemas una aplicación WINDEV en Mobile para iOS y Android. 


### Transformar un proyecto WINDEV en móvil (WINDEV con WINDEV Mobile incluido)
<a name="transformar_proyecto_windev_movil_windev_con_windev_mobile_incluido_ELTPARAGRAPHE000035"></a>

Para empezar a transformar un proyecto WINDEV en móvil: 

1. Abrir el proyecto en WINDEV. 

2. En la pestaña "Proyecto", en el grupo "Auditoría y rendimiento", amplíe "Auditoría estática" y seleccione "Auditoría móvil".. 

3. La asistente para la transformación en móvil se abre. 

4. Esta asistente propone crear una configuración de proyecto específica para móviles (Android o iOS). Pase a la etapa siguiente. 

5. Seleccione el tipo de dispositivo. Pase a la etapa siguiente.

6. La asistente enumera las ventanas WINDEV de su proyecto. 

7. Seleccione las ventanas a duplicar en la nueva configuración móvil. Cualquier modificación necesaria de UI se hará en la configuración del móvil (así como la modificación necesaria del código). 

8. Pase a la etapa siguiente. 

9. Seleccione los elementos (que no sean ventanas) a incluir en la configuración móvil. Estos elementos serán compartidos. 

10. Pase a la etapa siguiente. 

11. Especificar el tema gráfico utilizada para la configuración del proyecto. 

12. Pase a la etapa siguiente.

13. Valide el asistente. 

14. La auditoría móvil se inicia automáticamente. 




Observaciones: 

- El nombre de las ventanas duplicadas va seguido de "_MÓVIL".. 

- Puedes copy/pegar código o elementos entre las ventanas de WINDEV y WINDEV Mobile en cualquier momento.

- Windows sólo se puede duplicar cuando se crea la configuración del proyecto móvil por primera vez..  





### Cargando un proyecto WINDEV en WINDEV Mobile
<a name="cargando_proyecto_windev_windev_mobile_ELTPARAGRAPHE000073"></a>

Para empezar a transformar un proyecto WINDEV en móvil: 

1. Abrir el proyecto WINDEV en WINDEV Mobile. 

2. El asistente de creación de nuevas configuraciones de proyecto se abre. 

3. Seleccione la plataforma de la nueva configuración del proyecto (iOS o Android, por ejemplo). Pase a la etapa siguiente. 

4. Seleccione el tipo de dispositivo. Ir a la pantalla siguiente. 

5. La asistente enumera las ventanas WINDEV de su proyecto. 

6. Seleccione las ventanas a duplicar en la nueva configuración móvil. Cualquier modificación necesaria de UI se hará en la configuración del móvil (así como la modificación necesaria del código). 

7. Pase a la etapa siguiente.

8. Seleccione los elementos (que no sean ventanas) a incluir en la configuración móvil. Estos elementos serán compartidos. 

9. Pase a la etapa siguiente. 

10. Especificar el tema gráfico utilizada para la configuración del proyecto.

11. Pase a la etapa siguiente.  

12. Valide el asistente. 

13. La auditoría móvil se inicia automáticamente. 




Observaciones: 

- El nombre de las ventanas duplicadas va seguido de "_MÓVIL".. 

- Puedes copy/pegar código o elementos entre las ventanas de WINDEV y WINDEV Mobile en cualquier momento.

- Windows sólo se puede duplicar cuando se crea la configuración del proyecto móvil por primera vez.. 







### Siga los consejos de la auditoría móvil
<a name="siga_los_consejos_auditoria_movil_ELTPARAGRAPHE000099"></a>

La auditoría móvil se inicia automáticamente al final de la creación de una configuración móvil y puede iniciarse en cualquier momento a través del menú WINDEV o WINDEV Mobile.: en la pestaña "Proyecto", en el grupo "Auditoría y rendimiento", amplíe "Auditoría estática" y seleccione "Auditoría móvil".

Esta fiscalización permite subsanar algunos errores de compilación proponiendo soluciones. 

Todos los errores de compilación encontrados también se muestran en el panel de errores.. 

Es necesario corregir todos estos errores para que la aplicación se transforme completamente en móvil.. Algunos errores pueden estar relacionados con procesos o funciones no disponibles en teléfonos móviles, funciones WLanguage no disponibles, etc.

Atención: En algunos casos, pueden ser necesarios nuevos desarrollos para adaptar el comportamiento de la aplicación al entorno móvil.. 




### Terminar la transformación en móvil
<a name="terminar_transformacion_movil_ELTPARAGRAPHE000125"></a>

Cuando se corrigen todos los errores de compilación y auditoría móvil y se realizan las adaptaciones de gestión, se completa la transformación móvil..

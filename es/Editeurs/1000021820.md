## Notas reposicionables: Funcionalidades disponibles
<a name="notas_reposicionables_funcionalidades_disponibles_ELTTEXTE000252"></a>


### Creación de una note reposicionable
<a name="creacion_una_note_reposicionable_ELTPARAGRAPHE000019"></a>

Para crear una note reposicionable: 

1. Haga clic con el botón derecho del ratón para abrir el menú contextual de la ventana. 

2. Seleccione "Reposicionable note s .. Crear un note reposicionable ". <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable%20-%20HC%20N%B0003.gif)


3. La note reposicionable aparece en la ventana actual. Puede colocarlo en cualquier lugar de la ventana y escribir el texto de la note.
	Observación: Puede pegar texto en la note (Ctrl + V). 




Por Default, el reposicionable note tiene las siguientes características: 

- note texto en rojo. 

- Tamaño medio para el texto note. 

- Fondo color de nota color ed en amarillo. 

- El día de la creación aparece en la note (Hoy). Esta fecha se irá ajustando con el tiempo (por ejemplo: hace 3 días, etc.). 

- Reducción de automatic: Cada vez que se visualiza la ventana, la nota aparece durante 10 segundos y luego se reduce automáticamente (en la parte superior derecha Corner de la ventana). 





### Configuración de una note reposicionable
<a name="configuracion_una_note_reposicionable_ELTPARAGRAPHE000041"></a>

El menú contextual de la note reposicionable permite configurar esta note.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable%20-%20HC%20N%B0002.gif)
 Las opciones disponibles son las siguientes: 

- **Modificar texto de la nota**: El cursor de entrada aparece al final del texto de note para su modificación.
	Observación: Para modificar el texto de la note, también tiene la posibilidad de pulsar directamente la note. 

- **Minimizar**: Esta opción se utiliza para minimizar la note. La nota se posiciona como una miniatura en la parte superior derecha de la ventana de Corner.
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Note_Repostionnable1%20-%20HC%20N%B0001.gif&type=thumb)
 Un clic en el cuadrado Icon sirve para restaurar el note a su dimensión inicial. 

- **Minimizar automáticamente**: Si se selecciona esta opción, la note se mostrará al abrir la ventana y se reducirá automáticamente después de 5 segundos (si no se ha modificado). 

- **Color de fondo**: Esta opción permite personalizar el fondo color del note. 

- **Texto color**: Esta opción se utiliza para personalizar el texto Color. 

- **Tamaño del texto**: Esta opción se utiliza para personalizar el tamaño del texto. 

- **Eliminar nota**: Esta opción se utiliza para eliminar la note. En este caso, la note no se mostrará más. El borrado se puede deshacer pulsando Ctrl + Z. 

- **Exportar notas de esta ventana**: Esta opción se utiliza para crear un archivo "wdnote" en el directorio especificado. El nombre del archivo propuesto por Default corresponde al nombre de la ventana. Esta opción se utiliza para guardar las notas associated con la ventana. Las notas exportadas no se borran de la ventana. 

- **Exportar todas las notas de la aplicación**: Esta opción se utiliza para crear un archivo "wdnote" en el directorio especificado. El nombre del archivo propuesto por Default corresponde al nombre de la aplicación. Esta opción se utiliza, por ejemplo, para guardar las notas de la aplicación.. Las notas exportadas no se eliminan de la aplicación. 

- **Importar notas**: Esta opción se utiliza para importar un archivo "wdnote" en la aplicación. Esta ventana puede contener tanto las notas de una ventana como las de la aplicación.. Las notas que se encuentran tanto en la aplicación como en el archivo no se modificarán, las notas que se encuentran sólo en el archivo se añadirán a la aplicación..

## Implementación y configuración
<a name="implementacion_configuracion_ELTTEXTE000282"></a>


### Activación de las notas reposicionables
<a name="activacion_las_notas_reposicionables_ELTPARAGRAPHE000070"></a>

Por defecto, las notas reposicionables están disponibles para los usuarios de aplicaciones compiladas con la versión 21 (y posteriores), para todas las ventanas que utilizan el menú contextual AAF (automatic Application Features). Esta característica es relacionado a la modificación de UI por el usuario final. 

**Para habilitar las notas reposicionables en el proyecto**: 

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

2. Asegúrate de que el usuario final pueda modificar la UI y que las notas estén disponibles: 

	1. En la pestaña "Avanzado", haga clic en "Permitir que el usuario final modifique la UI" o en "Opciones" para ver las opciones. 

	2. En la ventana que aparece: 

		- comprobar si se utilizan las opciones de Default,

		- si se definen opciones específicas, hay que comprobar las opciones de gestión de la note reposicionable. 




3. Asegúrese de que el menú contextual del proyecto "Notas reposicionables" está activado a nivel de proyecto: 

	1. En la pestaña "avanzado", haga clic en "Menú de las FAA". 

	2. En las ventanas que se abren, seleccione el elemento "Ventana: la opción "Notas reposicionables" debe estar activada. 







**Para habilitar las notas reposicionables en una ventana**: 

1. Abra la ventana de descripción (opción "Descripción" del menú contextual). 

2. Asegúrate de que el usuario final pueda modificar la UI y que las notas estén disponibles: 

	1. En la pestaña "Detalles", haga clic en "Permitir que el usuario final modifique la UI" o en "Opciones" para ver las opciones. 

	2. En la ventana que aparece: 

		- comprobar si se utilizan las opciones de Default,

		- si se definen opciones específicas, hay que comprobar las opciones de gestión de la note reposicionable. 




3. Asegúrese de que el menú contextual del proyecto "Notas reposicionables" está activado a nivel de ventana: 

	1. En la pestaña "UI", despliegue la opción "Menú Cont". Marque "Mostrar menú AAF" y, a continuación, haga clic en "Configuración".

	2. En la ventana que se abre, la opción "Notas reposicionables" debe estar activada.. 








### Desactivar las notas reposicionables
<a name="desactivar_las_notas_reposicionables_ELTPARAGRAPHE000134"></a>

Puede desactivar la posibilidad de utilizar notas reposicionables: 

- para todo el proyecto, desactivando las notas reposicionables en el menú AAF: 

	1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".

	2. En la pestaña "avanzado", haga clic en "Menú de las FAA". 

	3. En las ventanas que se abren, seleccione el elemento "Window" y desactive la opción "Repositionable notes".. 




- para una ventana, desactivando las notas reposicionables en el menú AAF: 

	1. Abre la ventana description. 

	2. En la pestaña "UI", vaya a las opciones del menú contextual y haga clic en "Parámetros". 

	3. En las ventanas que se abren, desactive la opción "Notas reposicionables. 







Observación: También tiene la posibilidad de desactivar las opciones relativas a las notas reposicionables en la ventana de configuración de la modificación de UI por el usuario final, para todo el proyecto o para las ventanas. 


### Activación y desactivación mediante programación
<a name="activacion_desactivacion_mediante_programacion_ELTPARAGRAPHE000168"></a>

Para desactivar las notas reposicionables, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafRepositionableNote*. 

Para deshabilitar o habilitar las notas reposicionables en una ventana a través de la programación, también puede utilizar la propiedad [WinEditMode](../Proprietes/1000021824.md) con la constante *weAddRepositionableNote*:

```wl
MyWindow.WinEditMode = weAddRepositionableNote
```
Esta Property obtiene y establece todas las acciones permitidas a los usuarios finales cuando la opción "Permitir a los usuarios finales modificar la UI" está activada..

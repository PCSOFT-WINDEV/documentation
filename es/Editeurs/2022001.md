


## Reglas de alineación (Editor)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000203"></a>
Al crear o modificar una ventana en el editor de ventanas, tiene la posibilidad de mostrar las reglas de alineación. 
Estas reglas de alineación también pueden visualizarse en el editor Page y en editor de reportes.

Estas reglas tienen guías magnéticas: cuando un control se acerca a una guía, esta lo "atrae" automáticamente. Esta función le permite posicionar, alinear o redimensionar fácilmente los controles que se encuentran en la ventana, Page o reporte.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Regles.gif)

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000227"></a>


### Visualización de las reglas
<a name="visualizacion_las_reglas_ELTPARAGRAPHE000028"></a>

Para que se muestren las reglas, en la pestaña "Vista", en el grupo "Ayuda edición", seleccione "Reglas".

Existen dos tipos de guías:

- Las guías, utilizadas para alinear y ordenar los controles dentro de las ventanas, páginas o informes.

- Las guías de borde, utilizadas para definir un borde de idéntico tamaño a cada lado de la ventana, Page o reporte.




**Observación**: Para mostrar u ocultar las guías en la ventana, Page o reporte, seleccione o deseleccione "Mostrar guías" en el menú contextual de las reglas.

**Guías habilitadas**
Ya sea que se trate de Visible o no, las guías siempre están habilitadas: al mover un control hacia un marcador con el ratón, este se acopla automáticamente a la guía correspondiente.

Una vez que la control se acopla a la guía, todos los control acoplados a esta guía se desplazan automáticamente cuando se mueve la guía.

Nota: Un control movido con las teclas del teclado hacia una Marker no siempre está unido a la guía correspondiente. De hecho, esta operación se realiza con el píxel más cercano.
<a name="NOTE2_2"></a>


### Configuración de las reglas
<a name="configuracion_las_reglas_ELTPARAGRAPHE000062"></a>

Las reglas también pueden configurarse desde las opciones de edición del editor de ventanas, el editor Page o editor de reportes. Para ello, pulse F12 (pestaña "Magnetismo").

Esta opción permite:

- hacer a los gobernantes Visible o no.

- hacer las guías Visible o no.

- elija la color de las guías.

## Manejo de los marcadores
<a name="manejo_los_marcadores_ELTTEXTE000257"></a>
Los marcadores se pueden manipular fácilmente con el ratón. Cuando el cursor pasa por encima de un marcador, este se transforma en una flecha doble.
<a name="NOTE3_2"></a>


### Creación de una Marker
<a name="creacion_una_marker_ELTPARAGRAPHE000082"></a>

Para crear un Marker:

- 1er método: Haga clic en una de las reglas. La Marker se crea en la marca más cercana.

- 2º método: Haga clic derecho en la regla y seleccione "Agregar un marcador" en el menú contextual. La Marker se crea en la marca más cercana.



<a name="NOTE3_3"></a>


### Mover un Marker
<a name="mover_marker_ELTPARAGRAPHE000092"></a>

Para mover un Marker:

1. Haga clic en la Marker que se encuentra en la regla. El cursor se convierte en una flecha doble.

2. Mueve el cursor a la nueva posición de Marker.



<a name="NOTE3_4"></a>


### Borrar marcadores
<a name="borrar_marcadores_ELTPARAGRAPHE000102"></a>

- Para suprimir un Marker, seleccione "Suprimir el Marker" del el menú contextual (botón derecho del ratón) de un Marker.

- Para borrar todos los marcadores de una regla, seleccione "Borrar todo" en el menú contextual de una regla.



<a name="NOTE3_5"></a>


### Copiar los marcadores y las guías
<a name="copiar_los_marcadores_las_guias_ELTPARAGRAPHE000110"></a>

A copy los marcadores y las guías:

1. Seleccione "Guías copy" en el menú emergente de la regla.

2. Abra la ventana Page o reporte en la que deben copiarse los marcadores y las guías.

3. Visualizar (si es necesario) las reglas en esta ventana, Page o reporte: en la pestaña "Vista", en el grupo "Ayuda edición", seleccione la opción "Reglas".

4. Seleccione "Pegar" en el menú emergente de la regla.

## Caso especial: Los marcadores de frontera
<a name="caso_especial_los_marcadores_frontera_ELTTEXTE000305"></a>


### Definition y mover
<a name="definition_mover_ELTPARAGRAPHE000136"></a>

Los marcadores de borde se utilizan para definir un "margen" en la ventana actual (Page o reporte). Los marcadores de borde están situados en los extremos de las reglas.

Para mover el margen izquierdo o derecho, apunte a un borde Marker en la regla horizontal. Cuando el puntero se convierta en una flecha doble, mueva el margen.

Para mover el margen superior o inferior, apunte a un borde Marker en la regla vertical. Cuando el puntero se convierta en una flecha doble, mueva el margen.
<a name="NOTE4_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000147"></a>

- **Los marcadores de borde no se pueden eliminar**. Si los bordes no son necesarios en una ventana o en un reporte, los marcadores de borde pueden reducirse a su tamaño mínimo.

- **Por Default , los marcadores de borde que se encuentran en los lados opuestos están atados**: cuando se mueve una frontera Marker en un lado, la frontera Marker en el lado opuesto se mueve en consecuencia.
	Para modificar sólo uno de los marcadores de borde en el editor de ventanas, debe hacerlo:

	1. Haga clic en la frontera Marker: el cursor del ratón se convierte en una flecha doble.

	2. Mantenga pulsada la tecla Ctrl.

	3. Mueve la frontera Marker con el ratón.

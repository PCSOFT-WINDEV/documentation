


## Editor de modelado: Gestión de los estilos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Visión general y principio
<a name="vision_general_principio_ELTTEXTE000127"></a>
El editor de modelado propone una gestión avanzada de los estilos. Esta gestión de estilos le permite definir y utilizar fácilmente estilos predefinidos para los objetos de un modelado flexible.

**El principio es sencillo**: 

1. **El tema gráfico**: Tan pronto como se modifica el estilo de un elemento en un modelado flexible, un libro de estilo es automáticamente relacionado al modelado flexible. Este libro de estilo es una especie de "hoja de estilo" para el modelado flexible. El archivo associated con el tema gráfico es un archivo llamado "Charter_&lt;Nombre del modelo&gt;.wsm". Este archivo se guarda junto con el modelado flexible y puede ser utilizado por varios modelos.

2. **Durante la primera modificación del estilo de un elemento, hay varias soluciones disponibles**: 

	- **Crear un nuevo estilo en el modelado**: Esta opción le permite añadir un estilo en el tema gráfico. Todos los elementos del mismo tipo utilizarán este nuevo estilo.

	- **Modificar únicamente este elemento**: Esta opción permite modificar sólo el elemento seleccionado. Por lo tanto, este elemento no es relacionado a el tema gráfico.




3. **Existen varias soluciones al modificar el estilo de un elemento del tema gráfico**: 

	- **overload el estilo del elemental**: En este caso, sólo se modificará la unidad actual.. El elemento siempre será relacionado al estilo definido en el tema gráfico y las evoluciones del estilo associated con el elemento se tendrán en cuenta automáticamente.

	- **Crear un nuevo estilo**: Para ello, especifique el nombre del nuevo estilo a crear en el tema gráfico.

	- **Modificar el estilo en el tema gráfico**: En este caso, la modificación se aplicará a todos los elementos que utilicen el mismo estilo.

	- **Disociar el elemento**: En este caso, las modificaciones de estilo se aplicarán al elemento actual solamente.. Este elemento ya no será relacionado al estilo y no tendrá en cuenta los cambios de estilo.




4. Si varios estilos son associated con el mismo tipo de elemento, el estilo del elemento puede ser seleccionado a través de "Elegir un estilo..." de el menú contextual del elemento.

## Manejando los estilos y el tema gráfico
<a name="manejando_los_estilos_tema_grafico_ELTTEXTE000151"></a>


### Asociar un libro de estilo con un modelo
<a name="asociar_libro_estilo_con_modelo_ELTPARAGRAPHE000033"></a>

Asociar un libro de estilo con un modelo: 

1. En la pestaña "Modelado ", en el grupo "General", despliegue "Tema gráfico" y seleccione "Seleccionar un tema gráfico".

2. En la modelación description, seleccione el tema gráfico para usar.



<a name="NOTE2_2"></a>


### Asociar un estilo a un elemento
<a name="asociar_estilo_elemento_ELTPARAGRAPHE000055"></a>

Para asociar un estilo con un elemento: 

1. Seleccione el elemento.

2. Selecciona "Elegir un estilo..." de el menú contextual.

3. Selecciona el estilo deseado y valídalo.



<a name="NOTE2_3"></a>


### Actualizar los estilos de una modelación según el libro de estilo seleccionado
<a name="actualizar_los_estilos_una_modelacion_segun_libro_estilo_seleccionado_ELTPARAGRAPHE000066"></a>

Para actualizar los estilos de un modelado según el libro de estilo seleccionado, vaya a la pestaña "Modelado ", grupo "General", expanda "Tema gráfico" y seleccione "Actualizar estilos desde el tema gráfico".

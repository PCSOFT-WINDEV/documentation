
## Buscar/Reemplazar en el proyecto actual
			






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>18</B></B>, la búsqueda en un proyecto o en varios proyectos ha evolucionado. Para obtener más información, consulte [Buscar en los editores](../Editeurs/9000150.md).



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000163"></a>
La funcionalidad de "Buscar/Reemplazar" permite:

- Encuentra una expresión específica en todo el proyecto actual.

- Reemplazar las expresiones encontradas en el proyecto curernt por otra expresión.




Observación: también se puede realizar una búsqueda en varios proyectos. Para obtener más información, consulte [Búsqueda de varios proyectos](../Editeurs/2021009.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000187"></a>


### Realizar una búsqueda en el proyecto actual
<a name="realizar_una_busqueda_proyecto_actual_ELTPARAGRAPHE000033"></a>

Para realizar una búsqueda en el proyecto actual:

1. Visualiza la pantalla de [" Panel Buscar - Reemplazar](../Editeurs/2027013.md).

2. En la lista "Buscar", introduzca la expresión que desea buscar.

3. Especifique los elementos en los que debe realizarse la búsqueda (lista "En"). Tienes la capacidad de realizar una búsqueda de Default o una búsqueda avanzada.

4. Especifique los criterios de búsqueda ("Contiene", "Empieza por", "Distingue mayúsculas de minúsculas", ...).

5. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_rech.gif) Icon para iniciar la búsqueda. Si se encuentran elementos, se muestra la reference de estos elementos en el panel "Buscar - Reemplazar".. Para acceder directamente a uno de los elementos encontrados, haga doble clic en el elemento reference.




**Observación**: La búsqueda no tiene en cuenta las características del análisis ni las características del modelo UML associated con el proyecto.
<a name="NOTE3_2"></a>


### Realizar un "Buscar/Reemplazar" en el proyecto actual
<a name="realizar_buscarreemplazar_proyecto_actual_ELTPARAGRAPHE000053"></a>

Para realizar un "Buscar/Reemplazar" en el proyecto actual:

1. Visualiza la pantalla de [" Panel Buscar - Reemplazar](../Editeurs/2027013.md).

2. En la lista "Buscar", introduzca la expresión que desea buscar.

3. Introduzca la expresión que desea sustituir en la lista "Sustituir".

4. Especifique los elementos en los que debe realizarse la búsqueda (lista "En"). Tienes la capacidad de realizar una búsqueda de Default o una búsqueda avanzada.

5. Especifique los criterios de búsqueda ("Contiene", "Empieza por", "Distingue mayúsculas de minúsculas", ...).

6. Se utilizan varios iconos:

	- Reemplazar la expresión seleccionada.

	- Reemplazar la última expresión encontrada o la expresión seleccionada y buscar la siguiente expresión.

	- Reemplazar la última expresión encontrada o la expresión seleccionada y buscar la expresión anterior.

	- Reemplazar todas las expresiones encontradas.







**Observación**: La búsqueda no tiene en cuenta las características del análisis ni las características del modelo UML associated con el proyecto.

## Notas
<a name="notas_ELTTEXTE000217"></a>


### Tipo de búsqueda
<a name="tipo_busqueda_ELTPARAGRAPHE000079"></a>

Existen varios tipos de búsqueda disponibles:

- **Búsqueda predeterminada**: La búsqueda de Default puede realizarse en el código fuente del elemento Active, o en todo el elemento Active (código fuente, mensajes de código, controles, elementos, dependencias y opciones multilingües), o en todo el proyecto. Las opciones de la búsqueda Default se seleccionan en el panel "Buscar - Reemplazar".

- **Búsqueda avanzada**: La búsqueda avanzada permite especificar los elementos de proyecto en los que se debe realizar la búsqueda, así como el alcance de la búsqueda (las opciones del elemento en el que se debe realizar la búsqueda).. Para realizar una búsqueda avanzada, seleccione  en la opción "En" del panel "Buscar - Reemplazar".


<a name="NOTE4_2"></a>


### Criterios de búsqueda
<a name="criterios_busqueda_ELTPARAGRAPHE000091"></a>

Independientemente del tipo de búsqueda realizada, puede especificar una combinación de los siguientes criterios de búsqueda:

- Contiene: la expresión buscada debe contener la expresión especificada.

- Comienza por: la expresión buscada comienza con la expresión especificada.

- Palabra completa: la expresión buscada corresponde exactamente a la expresión especificada.

- Distinguir mayúsculas y minúsculas: la búsqueda tiene en cuenta los caracteres en mayúsculas y minúsculas.

- Respetar los caracteres acentuados: la búsqueda tiene en cuenta los caracteres acentuados.

- Ignorar comentarios: la búsqueda ignora los comentarios (los comentarios del código que empiezan por //). 



<a name="NOTE4_3"></a>


### Marcar el resultado de una búsqueda
<a name="marcar_resultado_una_busqueda_ELTPARAGRAPHE000105"></a>

Usted tiene la capacidad de marcar el resultado de una búsqueda: todos los elementos que se encuentran en el código corresponderán a una marca de posicionamiento.

Para set una marca en un resultado de búsqueda, basta con hacer clic en el ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ico_rech_marq.gif) Icon antes de iniciar la búsqueda.

Para más detalles sobre las marcas, véase [Marcas de posicionamiento](../Editeurs/2013008.md).


## Entorno del editor de ventanas 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EnvEdFen.gif)


Los elementos principales del editor de ventanas de WINDEV son los siguientes:


|   |   |
| --- | --- |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE1.gif)<br> | **Cinta de opciones**<br>Da acceso rápido a las principales características del editor de ventanas. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE2.gif)<br> | [**Reglas de alineación**](../Editeurs/2022001.md)<br>Se utiliza para alinear los controles en la ventana. Para mostrar las reglas, presione Ctrl + R. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE3.gif)<br> | [Pestañas de documentos abiertos](../Editeurs/2029012.md)<br>Cada pestaña corresponde a una Document abierta . Al hacer clic en una pestaña, se abre el elemento en el editor correspondiente. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE4.gif)<br> | **Seleccionado control**<br>control actualmente modificada. Las asas negras permiten identificar fácilmente la control seleccionada .<br><br>Para obtener más información, consulte [Seleccione uno o más controles](../Editeurs/2026007.md). |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Cercle5.gif)<br> | [**Anclaje**](../WDChamp/2026002.md)<br>Las flechas rojas indican el anclaje de la control.<br><br>Recordatorio: El anclaje se utiliza para gestionar la posición de la control cuando se cambia el tamaño de la ventana. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE6.gif)<br> | **[Paneles acoplables](../Editeurs/2027001.md)**<br>Muestra:<br><br>- los accesos directos a las principales funciones disponibles en el editor actual (["Panel "Comandos](../Editeurs/2027010.md)).<br><br>- las características más comunes (["Panel "Comandos](../Editeurs/2027010.md)).<br><br>- los elementos (ventanas, informes, controles, etc.) disponibles en el proyecto actual (["Panel "Explorador de proyectos](../Editeurs/2027029.md)).<br><br>- ...<br><br><br> |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE7.gif)<br> | **Ventana de descripción**<br>Permite visualizar y/o modificar las características:<br><br>- los controles seleccionados.<br><br>- la ventana seleccionada.<br><br><br>Esta información también aparece en el modificador (["Panel "Modificar](../Editeurs/2027006.md)). |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CERCLE8.gif)<br> | **[Paneles acoplables](../Editeurs/2027001.md)**<br>Los cristales solían hacerlo:<br><br>- mostrar los elementos del código (lista de clases, procedimientos locales o globales, ["Panel "Código](../Editeurs/2027009.md)), <br><br>- mostrar los errores de compilación (errores encontrados en el código WLanguage, ["Panel "Errores de compilación](../Editeurs/2027031.md)), <br><br>- realizar búsquedas ([" Panel Buscar - Reemplazar](../Editeurs/2027013.md)), <br><br>- mostrar el depurador durante la prueba de un elemento (["Depurador" pan](../Editeurs/2027009.md)), <br><br>- ...<br><br><br> |

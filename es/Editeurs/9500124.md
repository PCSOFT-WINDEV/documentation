


## automatic Calendar para los controles de edición de fecha (AAF)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
El Date control Campo de entrada propone automáticamente una opción "Calendar" en el menú que aparece.. Esta opción se utiliza para mostrar una Calendar y para seleccionar una fecha. El usuario tiene la posibilidad de hacer clic en una fecha para que esta fecha se seleccione y se muestre en el control Campo de entrada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FAA_Calendrier.gif)

## A automatic Calendar
<a name="automatic_calendar_ELTTEXTE000135"></a>


### Implementación
<a name="implementacion_ELTPARAGRAPHE000019"></a>

La opción "Calendario" del menú contextual de los controles de edición de fechas es automatic. 

También tienes la capacidad de mostrar un botón "Calendar" a la derecha del área de entrada del control. Para ello, marque "Mostrar un botón calendario" en la pestaña "General" de control Campo de entrada. El botón "Editar calendario..." permite personalizar el estilo de la Calendar. 

Mediante programación: 

- la propiedad [BotónCalendario](../Proprietes/1000017275.md) se utiliza para determinar e indicar si una fecha control Campo de entrada tiene un botón para mostrar una ventana emergente Calendar.

- la función [AAFExecute](../WDLang1/1000022099.md) utilizada con la constante *aafCalendar* permite mostrar el calendario debajo del control Campo de entrada especificado.

- la función [AAFDisable](../WDLang1/1000022018.md) utilizada con la constante *aafCalendar* elimina la opción "Calendario" en el menú contextual del control Campo de entrada especificado.

- [AAFChangeCaption](../WDLang1/1000022100.md) se utiliza para: 

	- modificar la leyenda de la opción de menú "Calendario" (constante *aafCalendar*).

	- modificar el título del botón "Hoy" en la ventana del calendario (constante *aafToday*). 








### Traduciendo el Calendar
<a name="traduciendo_calendar_ELTPARAGRAPHE000057"></a>

El Calendar se suministra en inglés y en francés. 

Algunos parámetros pueden ser traducidos. Para obtener más información, consulte [Control Calendario](../WDChamp/1016300.md).

## Presentación

Las acciones preestablecidas permiten definir el código de una Event en un solo clic, sin tener que escribir un solo Line de código.

Por ejemplo, en WINDEV:

*Inicialización de un componente, los parámetros se especifican en la interfaz*

Se proponen las acciones preestablecidas:

- en los controles de los botones (en el Event "Click").

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en las opciones del menú.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000181"></a>


### Definición de una acción preestablecida
<a name="definicion_una_accion_preestablecida_ELTPARAGRAPHE000041"></a>

Para definir una acción preestablecida:

1. Abra la ventana de descripción del control (seleccione "Descripción" en el menú contextual).

2. En la pestaña "General", defina la acción deseada: 

	1. ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Seleccione la acción a ejecutar en el cuadro combinado "Acción".. 
			Las acciones más comunes aparecen en esta lista. Para seleccionar una acción más específica, haga clic en "Todas las acciones" u "Otras acciones".





Las acciones propuestas dependen de los elementos que se encuentran en el proyecto actual (ventanas, páginas, componente, Procedure, ...).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Observación**: Para elegir la acción preestablecida, también tienes la posibilidad de seleccionar "Acción preestablecida..." de la el menú contextual de la control.

## Nota
<a name="nota_ELTTEXTE000205"></a>


### La información que aparece en el editor de código
<a name="informacion_que_aparece_editor_codigo_ELTPARAGRAPHE000070"></a>

Cuando se utiliza una acción preestablecida, se especifica una información que indica esta acción preestablecida en el editor de código.

Por ejemplo:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ActionPredefInfo.GIF)


Al hacer clic en la acción preestablecida que aparece en el editor de código se abre: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la ventana de selección de acciones preestablecidas.


Por lo tanto, la acción preestablecida puede ser modificada desde el editor de código directamente.

**Atención**: Cualquier código WLanguage introducido en el editor de código después de la llamada a la acción preestablecida no se ejecutará.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Para incluir una acción preestablecida en un código WLanguage, utilice [ExecutePresetAction](../WDLang1/3014028.md).
<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Ejecutar una acción preestablecida mediante programación
<a name="ejecutar_una_accion_preestablecida_mediante_programacion_ELTPARAGRAPHE000105"></a>

WINDEV también propone ejecutar una acción preestablecida utilizando la función [ExecutePresetAction](../WDLang1/3014028.md).

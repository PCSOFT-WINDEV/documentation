


## Reparando el proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
En algunos casos, es posible reparar el proyecto a manipular un proyecto que causa problemas. Para reparar el proyecto, en la pestaña "Proyecto", en el grupo "Proyecto", amplíe "Recompilar y sincronizar" y seleccione "Reparar proyecto".

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cuáles son las operaciones que se realizan al reparar un proyecto?
<a name="¿cuales_son_las_operaciones_que_realizan_reparar_proyecto_ELTTEXTE000106"></a>
Las siguientes operaciones se realizan cuando se repara un proyecto:

- Abrir y guardar todos los elementos del proyecto. Todos los elementos del proyecto se guardan de nuevo en el formato de la versión utilizada actualmente.

- [Recopilación del proyecto](../Editeurs/9500205.md).

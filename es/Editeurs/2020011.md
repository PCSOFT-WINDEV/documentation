


## No volver a mostrar un mensaje
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000116"></a>
El entorno de WINDEV, WEBDEV y WINDEV Mobile a menudo muestra ventanas de advertencia.

Muy importante cuando usted descubre una característica, estas ventanas pueden rápidamente volverse inútiles.

La opción "No mostrar de nuevo" aparece ahora en muchas ventanas, permitiéndole desactivar la ventana de advertencia actual.

WINDEV, WEBDEV y WINDEV Mobile proponen una gestión centralizada de las ventanas de alerta comunes a todos los proyectos.: esto le permite recuperar las ventanas que ya no se muestran y volver a habilitarlas si es necesario.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000140"></a>


### No volver a mostrar un mensaje
<a name="volver_mostrar_mensaje_ELTPARAGRAPHE000024"></a>

Para no mostrar más un mensaje, simplemente marque "No mostrar de nuevo" en el mensaje.

Si se espera una respuesta por el mensaje, la respuesta seleccionada se almacenará automáticamente y será utilizada por Default.
<a name="NOTE2_2"></a>


### Viendo los elementos marcados con "No mostrar de nuevo" y volviendo a activar un mensaje
<a name="viendo_los_elementos_marcados_con_mostrar_nuevo_volviendo_activar_mensaje_ELTPARAGRAPHE000033"></a>

La lista de mensajes marcados con "No mostrar de nuevo" puede ser vista en cualquier momento:

1. En la pestaña "Inicio", en el grupo "Entorno", amplíe "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile".

2. Seleccione la pestaña "No mostrar de nuevo". La lista corresponde a los mensajes que no se mostrarán de nuevo. El nombre del editor afectado, el título del mensaje y la respuesta seleccionada (si es necesario) se muestran para cada mensaje.




Para volver a habilitar un mensaje, desmarque el mensaje solicitado en la lista y valídelo.

Observaciones:

- Esta lista sólo muestra los elementos marcados con "No mostrar de nuevo" que son comunes a todos los proyectos. Cuando una opción es específica de un proyecto, debe estar configurada en el proyecto.. Ejemplo: la ventana que pregunta si la prueba debe ser detenida cuando se vuelve al editor: esta configuración es específica para cada proyecto.

- La ventana de acceso no se tiene en cuenta entre los elementos marcados con "No mostrar de nuevo". Esta ventana se configura en las opciones generales del producto. Para visualizar las opciones generales, vaya a la pestaña "Inicio", grupo "Entorno", amplíe "Opciones" y seleccione "Opciones generales de WINDEV/WEBDEV/WINDEV Mobile".




## Live Data: ver los datos en los editores
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000302"></a>
El "Live Data" consiste en mostrar el contenido de los archivos de datos en los controles creados en los editores del producto utilizado (editor de ventanas, editor de páginas, editor de reportes, etc ...). Esta característica permite configurar el tamaño de un control de acuerdo con los datos que se pueden mostrar en el control.

**Atención**: Para utilizar los "Datos en tiempo real", debe hacerlo:

- asociar un análisis al proyecto,

- manipular los ficheros de datos existentes.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo implementar Live Data?
<a name="¿como_implementar_live_data_ELTTEXTE000332"></a>


### Implementación de "Live Data" en el proyecto actual (editor de ventanas, editor de páginas y editor de reportes)
<a name="implementacion_live_data_proyecto_actual_editor_ventanas_editor_paginas_editor_reportes_ELTPARAGRAPHE000025"></a>

Para implementar "Live Data" en el proyecto actual (editor de ventanas, editor de páginas y editor de reportes): 

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". La ventana de descripción del proyecto se abre.

2. Visualizar la pestaña "Live Data".

3. Seleccione "Activar Live Data".

4. Para cada archivo de datos, especifique si es necesario:

	- el tipo de archivo de datos: Esta opción permite seleccionar el tipo de del archivo de datos que se utilizará en la vista previa. De hecho, el archivo utilizado puede ser cualquier archivo de datos: HFSQL Classic, HFSQL Client/Server, Oracle, SQL Server, ...: todo lo que tiene que hacer es seleccionar la conexión correspondiente.
			Se pueden utilizar diferentes tipos de archivos de datos al mismo tiempo.

	- el directorio: El directorio puede corresponder a:

		- "Localización automática ": la localización del archivo de datos se realiza automáticamente. También puede hacer clic en "Localización automática ".

		- "Directorio de modo de prueba": los archivos de datos utilizados son los definidos para el modo Test (el directorio está definido en la pestaña "Files" del proyecto description.

		- "Directorio definido en análisis": los ficheros de datos utilizados son los definidos en el análisis.

		- "Navegar": le permite seleccionar el archivo de datos deseado.




- la contraseña del archivo de datos.  

5. Valide.


**Observación**: En este caso, Live Data se implementará durante la próxima apertura de ventanas, páginas y/o informes.
<a name="NOTE2_2"></a>

### Implementación de Live Data sobre el proyecto actual en Reports & Queries
<a name="implementacion_live_data_sobre_proyecto_actual_reports_queries_ELTPARAGRAPHE000073"></a>

Para implementar Live Data en el proyecto actual en Reports & Queries: 

1. Seleccione "Opciones de los archivos de proyecto".. La ventana de descripción se abre.

2. Seleccione "Activar Live Data".

3. Para cada archivo de datos, especifique si es necesario:

	- el tipo de archivo de datos: Esta opción permite seleccionar el tipo de del archivo de datos que se utilizará en la vista previa. De hecho, el archivo utilizado puede ser cualquier archivo de datos: HFSQL Classic, HFSQL Client/Server, Oracle, SQL Server, ...: todo lo que tiene que hacer es seleccionar la conexión correspondiente.
			Se pueden utilizar diferentes tipos de archivos de datos al mismo tiempo.

	- el directorio: El directorio puede corresponder a:

		- "Localización automática ": la localización del archivo de datos se realiza automáticamente. También puede hacer clic en "Localización de automatic".

		- "Directorio de modo de prueba": los archivos de datos utilizados son los definidos para el modo Test (el directorio está definido en la pestaña "Files" del proyecto description).

		- "Directorio definido en análisis": los ficheros de datos utilizados son los definidos en el análisis.

		- "Navegar": le permite seleccionar el archivo de datos deseado.




- la contraseña del archivo de datos.  

4. Valide.


**Observación**: En este caso, la implementación de Live Data se tendrá en cuenta en la próxima apertura de informes y consultas modificables en "Reports & Queries".

## ¿Cómo desactivar los datos en vivo?
<a name="¿como_desactivar_los_datos_vivo_ELTTEXTE000362"></a>


### Desactivar "Live Data" en el proyecto actual (editor de ventanas, editor de páginas y editor de reportes)
<a name="desactivar_live_data_proyecto_actual_editor_ventanas_editor_paginas_editor_reportes_ELTPARAGRAPHE000104"></a>

Desactivar "Live Data" en el proyecto actual (editor de ventanas, editor de páginas y editor de reportes):

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". La ventana de descripción del proyecto se abre.

2. Visualizar la pestaña "Live Data".

3. Deseleccione la opción "Activar Live Data" y valide.


**Observación**: En este caso, los datos en vivo se desactivarán durante la próxima apertura de ventanas, páginas y/o informes.
<a name="NOTE3_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Desactivando Live Data en el proyecto actual en Reports & Queries
<a name="desactivando_live_data_proyecto_actual_reports_queries_ELTPARAGRAPHE000135"></a>

Para desactivar "Live Data" en el proyecto actual en Reports & Queries: 

1. Seleccione "Opciones de los archivos de proyecto".. La ventana de descripción se abre.

2. Deseleccione la opción "Activar Live Data" y valide.


**Observación**: En este caso, Live Data se desactivará durante la próxima apertura de informes y consultas.

## ¿Cómo cambiar el directorio de los archivos de datos utilizados para Live Data?
<a name="¿como_cambiar_directorio_los_archivos_datos_utilizados_para_live_data_ELTTEXTE000392"></a>


### Cambiar el directorio de los ficheros de datos utilizados para "Live Data".
<a name="cambiar_directorio_los_ficheros_datos_utilizados_para_live_data_ELTPARAGRAPHE000153"></a>

Para cambiar el directorio de los ficheros de datos utilizados para "Live Data":

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". La ventana de descripción del proyecto se abre.

2. Visualizar la pestaña "Live Data".

3. Seleccione "Activar Live Data".

4. Para cada archivo de datos, especifique la ubicación del archivo. El botón "Localización automática " permite buscar automáticamente los archivos. Esta búsqueda puede llevar bastante tiempo.

5. Valide.

## ¿Cómo especificar la contraseña de los archivos de datos utilizados para Live Data?
<a name="¿como_especificar_contrasena_los_archivos_datos_utilizados_para_live_data_ELTTEXTE000422"></a>


### Especificación de la contraseña de los archivos de datos utilizados para "Live Data".
<a name="especificacion_contrasena_los_archivos_datos_utilizados_para_live_data_ELTPARAGRAPHE000210"></a>Para especificar la contraseña de los archivos de datos utilizados para "Live Data".: 
Si los archivos de datos utilizados para "Live Data" están protegidos por contraseña, la contraseña se puede especificar en las opciones del proyecto:

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". La ventana de descripción del proyecto se abre.

2. Visualizar la pestaña "Live Data".

3. Seleccione "Activar Live Data".

4. Para cada archivo de datos, especifique la contraseña correspondiente y valídela.


**Observación**: Si no se especifica la contraseña para un archivo protegido por contraseña, no se mostrarán los datos que se encuentran en este archivo.
<a name="NOTE5_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)

<a name="especificacion_del_directorio_los_ficheros_datos_utilizados_para_live_data_reports_queries_ELTPARAGRAPHE000243"></a>

Si los archivos de datos utilizados para "Live Data" están protegidos por contraseña, se puede especificar esta contraseña:

1. Seleccione "Opciones de los archivos de proyecto".. La ventana de descripción se abre.

2. Seleccione "Activar Live Data".

3. Para cada archivo de datos, especifique la contraseña correspondiente y valídela.


**Observación**: Si no se especifica la contraseña para un archivo protegido por contraseña, no se mostrarán los datos que se encuentran en este archivo.

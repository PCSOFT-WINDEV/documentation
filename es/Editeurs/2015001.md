## Presentación
<a name="presentacion_ELTTEXTE000263"></a>
Puede imprimir diferentes documentaciones técnicas que describen todos los elementos del proyecto (ventanas, Page, informes, ficheros de datos, artículos, ensamblajes, Custom-Folders, etc.).

Para imprimir la documentación, en la pestaña "![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_imprimer.gif)", en el grupo "Inicio" despliegue el ícono General:

- la opción "Imprimir la documentación técnica del proyecto" abre la ventana correspondiente de asistente. Puede elegir el tipo de información que se va a imprimir. La documentación técnica puede contener todas las características del proyecto y sus elementos.

- la opción "Imprimir. En este caso, la documentación técnica contendrá únicamente los detalles del elemento actual (ventana, reporte, consulta, Carpeta Personal, proyecto, etc.)..
	Si se utiliza la opción "Imprimir" en un elemento simple (el código de un elemento, por ejemplo), sólo se imprimirá este elemento. 




**Observaciones**: 

- Si se va a cambiar el formato de impresión de la documentación técnica (de A4 a A3), puede adaptar las páginas de la documentación al nuevo formato. Para obtener más información, consulte [Área de impresión](../Editeurs/2015001.md).

- La documentación técnica no puede imprimirse en "Reports & Queries".

- En versiones anteriores, la documentación técnica se denominaba "documentación de programación".. 

- Al imprimir el código WLanguage desde el editor de código, aparece un cuadro de diálogo que permite: 

	- seleccionar la impresora que se va a utilizar

	- elegir la orientación del papel. 

	- cambiar el tamaño de las áreas de impresión según el formato utilizado por la impresora, si es necesario.

## Documentación técnica del proyecto
<a name="documentacion_tecnica_del_proyecto_ELTTEXTE000287"></a>


### Tipo de información a imprimir
<a name="tipo_informacion_imprimir_ELTPARAGRAPHE000051"></a>

Existen diferentes tipos de documentación técnica:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dossier%20du%20projet%20-%20HC%20N%B0001%201.gif)


- General: Contiene información básica sobre los elementos del proyecto.

- codificar: Contiene los procesos de todos los objetos y procedimientos del proyecto.

- Complete: Contiene toda la información del proyecto. Este tipo de documentación técnica puede ser muy extensa. 

- A medida: Corresponde a una selección de información a imprimir. 



<a name="NOTE2B_2"></a>


### Contenido de la documentación técnica
<a name="contenido_documentacion_tecnica_ELTPARAGRAPHE000064"></a>

La documentación técnica **completa** del proyecto puede contener varias partes:

- Proyecto description (elementos encontrados en el proyecto, código del proyecto, etc.).

- description de componentes internos del proyecto.

- Análisis description (gráfico de análisis, diccionario de datos, etc.).

- description de ventanas y/o páginas (Image de ventanas, lista de controles en cada ventana, etc.).

- description de plantillas de ventanas o plantillas Page

- description de informes (Image de informes, lista de controles en cada reporte, etc.).

- description de plantillas de reporte.

- description de consultas (Image de consultas, código SQL correspondiente, etc.).

- description de clases (código de clases, etc.).

- description de modelos UML (UML Diagram, lista de relaciones en el Diagram, etc.).

- description de modelización flexible (Diagram, lista de aplicaciones, lista de componentes, etc.)

- description de conjuntos de procedimientos (código, mensajes multilingües, Custom-Notes, etc.).

- Descripción de las pruebas de automatic (código, resumen de resultados).

- description de normas empresariales (description, archivos adjuntos, enlaces, etc.).




**Al imprimir la documentación general**, solo se imprime la información principal sobre el proyecto, ventanas y/o páginas, modelos e informes..  

**Al imprimir una documentación técnica personalizada** , puede elegir la información a imprimir en cada parte (botón "Editar"). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dossier%20du%20projet%20-%20HC%20N%B0002.gif)

Por ejemplo, para las consultas, puede imprimir:

- la representación gráfica de las consultas,

- detalles generales de las consultas (nombre lógico y físico, tipo, análisis associated con las consultas, etc.),

- información específica sobre el resultado de las consultas (elementos de resultado, parámetros de consulta, criterios de ordenación, etc.),

- detalles de la consulta (condiciones de selección, grupos, etc.),

- el código SQL correspondiente a las consultas.

- Notas personalizadas. 

- reglas de negocio.




Independientemente del tipo de documentación técnica elegida, también puede seleccionar los diferentes elementos del proyecto que se van a incluir. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dossier%20du%20projet%20-%20HC%20N%B0001%202.gif)
 El botón "Modificar" permite seleccionar una o varias ventanas, por ejemplo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dossier%20du%20projet%20-%20HC%20N%B0006.gif)
 En la documentación sólo aparecerá información sobre las ventanas seleccionadas.

**Observación**: Si el proyecto actual contiene ventanas WINDEV, ventanas Mobile y páginas WEBDEV, puede imprimir información sobre todos los elementos del proyecto, independientemente del producto desde el que se imprima la documentación.

**Atención**: Si hay representaciones gráficas que imprimir (análisis, diagramas UML, etc.), debe configurar las áreas de impresión. Para obtener más información, consulte [Áreas de impresión](#NOTE4_1).

### Generación de la documentación técnica
<a name="generacion_documentacion_tecnica_ELTPARAGRAPHE000140"></a>

La documentación técnica puede imprimirse en la impresora seleccionada. La documentación técnica se muestra en el visor de reporte. Usted puede: 

- iniciar el trabajo de impresión,

- exportar la documentación técnica en formato PDF (a través de la pestaña "Exportar").

## Documentación técnica del elemento actual
<a name="documentacion_tecnica_del_elemento_actual_ELTTEXTE000329"></a>


### Contenido de la documentación técnica
<a name="contenido_documentacion_tecnica_ELTPARAGRAPHE000152"></a>El contenido de esta documentación técnica se puede configurar: sólo la información seleccionada se presentará en detalle.<br>![Documentación técnica de una ventana](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Dossier%20du%20projet%20-%20HC%20N%B0007.gif)
 Por ejemplo, si el elemento actual es una ventana, puede imprimir uno o varios de los siguientes detalles:

- información general sobre la ventana (nombres lógicos y físicos, título, etc.),

- la ventana Image,

- detalles de los controles de la ventana (nombre y leyenda, características, etc.),

- detalles de los menús de las ventanas (nombre de las distintas opciones, accesos directos a las opciones, etc.),

- el código de la ventana,

- el código de los controles que se encuentra en la ventana,

- los procedimientos de las ventanas,

- mensajes multilingües de la ventana.

- etc.




**Observación**: Si el elemento actual es una ventana de códigos, la opción "Imprimir" se utiliza para imprimir los diferentes códigos que aparecen en esta ventana.

## Área de impresión
<a name="area_impresion_ELTTEXTE000359"></a>


### Especificación de las áreas de impresión
<a name="especificacion_las_areas_impresion_ELTPARAGRAPHE000186"></a>

Si hay que imprimir representaciones gráficas (LDM, diagramas UML, etc.), las zonas de impresión deben ser set antes de imprimir la documentación técnica.

Para especificar las áreas de impresión de la representación gráfica actual:

1. En la pestaña "Vista", en el grupo "Edición", seleccione la opción "Áreas de impresión". Una serie de bordes que representan las áreas imprimibles en la documentación técnica aparecerán en el editor actual.

2. Reducir la visualización de la representación gráfica para ver el gráfico completo: 

	- en la pestaña "Vista", modifique el valor del zoom.

	- pulse Ctrl + rueda del ratón.




3. Con el ratón:

	- mover los bordes al lugar deseado manteniendo el botón del ratón pulsado (el cursor del ratón es Black).

	- seleccione el número de páginas en las que se debe imprimir la representación gráfica (el cursor se convierte en una flecha doble Norte-Oeste/Sureste).

## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
Los Centros control proporcionan una visión global de un área determinada, ya sea en fase de desarrollo, en un emplazamiento operativo o en el mantenimiento o mejora de una aplicación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Los Centros control dedicados a la gestión de proyectos
<a name="los_centros_control_dedicados_gestion_proyectos_ELTTEXTE000150"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000018"></a>

Puede supervisar los proyectos en WINDEV, WEBDEV y WINDEV Mobile a través de [Centro de Seguimiento de Proyectos](../CCSuivi/3540701.md). 

La Centro de Seguimiento de Proyectos permite gestionar los diferentes eventos durante la vida de un proyecto: Requisitos, Tareas, Agenda personal, Incidentes, ... Desde este centro, los colaboradores pueden indicar las tareas pendientes o realizadas, las incidencias solucionadas, ... Este centro también se utiliza para recuperar los comentarios de los usuarios sobre las aplicaciones implementadas.

## Los centros redistribuibles de control
<a name="los_centros_redistribuibles_control_ELTTEXTE000180"></a>


### Con WINDEV, WEBDEV y WINDEV Mobile se proporcionan varios Centros control redistribuibles:
<a name="con_windev_webdev_windev_mobile_proporcionan_varios_centros_control_redistribuibles_ELTPARAGRAPHE000046"></a>

- **[Centro de motores control y bases de datos HFSQL (red y Cliente/Servidor)](../CCHF/3540001.md)**
	Este control Center (también llamado HFSQL control Center) se utiliza para gestionar las diferentes bases de datos HFSQL (Red, Móvil y principalmente Cliente/Servidor) que se encuentran en la empresa. Este Centro control puede utilizarse para desarrollar una aplicación. Se puede redistribuir junto con las aplicaciones de HFSQL Client/Server.

- **[Centro de Control del Usuario](../CCUser/3540201.md)**
	Este Centro control se utiliza para gestionar las aplicaciones desplegadas en la red de la empresa. Desde este Centro control, los usuarios podrán instalar sus aplicaciones dedicadas. El Centro de Usuarios de control puede ser redistribuido.

- **[Centro de Control de Aplicaciones de Red](../WDAdminExe/2028052.md)**: El Centro control de aplicaciones WINDEV ofrece al gestor de una aplicación en la red una visión completa de la misma.: actualizaciones, usuarios, actualización con notificaciones push, ... Este Centro control es redistribuible. Se instala automáticamente con las aplicaciones en las que está activada la opción "Live update"..

## Centros específicos control
<a name="centros_especificos_control_ELTTEXTE000204"></a>
WINDEV, WEBDEV y WINDEV Mobile incluyen varios Centros control específicos: 

- **[Centro de Control de Alojamiento](../WDAdminDeploiement/3539029.md)**: El Hosting control Center permite alojar fácilmente sitios WEBDEV. Esta herramienta está destinada principalmente a los proveedores de alojamiento y a los Webmasters.

- **[Centro de Control Android](../CCAndroid/1410086630.md)**: El Centro de Control Android le permite administrar fácilmente sus dispositivos Android durante la fase de desarrollo. Este Centro control se incluye con WINDEV Mobile, y no es redistribuible.

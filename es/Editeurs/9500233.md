
## Tooltips personalizados
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000223"></a>
Los tooltips son muy useful para los usuarios de la aplicación. 

A partir de la versión 26, la información sobre las herramientas se puede personalizar: utilice información sobre herramientas personalizadas para mostrar sus mensajes. Existen varios tipos de información sobre herramientas: 

- Información sobre herramientas estándar: 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0001.gif)


- Plantilla tooltips: 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0003.gif)


- Información sobre herramientas con estilo personalizado: 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0005.gif)


- Información sobre herramientas personalizada a través de una ventana de aplicación.  




**Recordatorio**: El texto que aparece en la información sobre la herramienta puede ser: 

- Introducido en el editor: 

	1. Abra la ventana de descripción del control (seleccione "Descripción" del menú contextual). 

	2. En la pestaña "Ayuda", en el área "Tooltip", escriba el texto de ayuda. 
			Observaciones: 

		- En una aplicación multilingüe, el texto del tooltip se puede escribir en varios idiomas. 

		- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la opción "RTF" está seleccionada, se puede definir el formato del texto mediante las opciones de la barra de herramientas RTF. El formato se mantendrá en tiempo de ejecución. 




- Especificado a través de la [ToolTip](../Proprietes/2510005.md) Property. 




Observaciones: 

- La información sobre herramientas personalizada se utiliza para todas las informaciones sobre herramientas de la aplicación: 

	- Tooltips de los controles. 

	- Información de herramientas automatic para los control s de tabla y lista (se muestra si el texto no está completo en el control, por ejemplo). 




- Los tooltips personalizados no se utilizan en los siguientes casos: 

	- control Gráfico tooltips. 

	- Información sobre la barra de desplazamiento.  




- Si sus usuarios personalizan el texto de la información sobre herramientas a través del AAf, el texto personalizado se mostrará en la información sobre herramientas personalizada.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000247"></a>


### Personalización de la información sobre herramientas
<a name="personalizacion_informacion_sobre_herramientas_ELTPARAGRAPHE000061"></a>

Para personalizar la información sobre herramientas en su aplicación: 

1. Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

2. En la pestaña "Estilo", especifique el tipo de información sobre herramientas que se utilizará en el proyecto: 
	<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0002.gif&type=thumb)


	- Estándar: se utilizan los tooltips estándar con una Background amarilla. 

- Definido por la plantilla: se utilizan los tooltips de la plantilla utilizados por el proyecto. 

- A medida: el editor de estilo de información sobre herramientas se abre y muestra varias opciones de personalización: 
			<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Bulles_aide_personnalisees%20-%20HC%20N%B0004.gif)
 Puede personalizar: 

	- El texto mostrado: tipo de letra, tamaño, sombra, Color, etc.

	- Los bordes de la información sobre herramientas: márgenes, fondo Color, tipo de borde, ... 

3. Valide la ventana de descripción del proyecto. Todos los tooltips mostrados por su aplicación utilizarán el aspecto seleccionado. 




Observación: La información sobre herramientas puede mostrarse con una animación. Para activar esta animación: 

1. Abra la ventana de descripción del proyecto:  en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

2. En la pestaña "avanzado", haga clic en "Animación de controles". 

3. En "Tooltip", seleccione y configure la animación deseada. 

4. Validar. 

5. Valide la ventana de descripción del proyecto. 





### Caso especial: Utilizar una ventana de aplicación para personalizar la información sobre herramientas
<a name="caso_especial_utilizar_una_ventana_aplicacion_para_personalizar_informacion_sobre_herramientas_ELTPARAGRAPHE000119"></a>

WINDEV permite utilizar una ventana de la aplicación para mostrar el contenido de los tooltips. 

En ese caso: 

- Sólo se ejecuta el código de declaración de las variables globales de la ventana. 

- El código de declaración de las variables globales de una ventana debe contener el siguiente código: 
	
	```txt
	PROCEDURE DeclarationWindowGlobalVariables(TooltipText, SourceControl)
	STC_TOOLTIP = TooltipText
	```
 donde: 

	- TooltipText es el texto que se mostrará en el tooltip. Este texto debe aparecer en una control Estático en la ventana (por ejemplo, STC_TOOLTIP). 

	- Sourcecontrol es un parámetro opcional que contiene el nombre de la control para la que se muestra la información sobre herramientas. 




- La opción "Adaptar tamaño según el contenido" (pestaña "General" de la ventana description) debe estar marcada. 

- La ventana debe utilizar una imagen de fondo con la opción de recorte activada ("Recortar ventana (utiliza el modo de transparencia de la imagen de fondo)" en la pestaña "Imagen de la ventana de description). 

- Para mostrar texto con efectos en la información sobre herramientas, puede, por ejemplo: 

	- utilizar RTF. 

	- utilizar las funciones de cadena gráfica (Para más detalles, véase [Funciones de gestión de cadenas gráficas](../WDLang1/3029028.md)). 


 Esta información debe especificarse en el tooltip del control (por ejemplo, en la pestaña "Ayuda" de la ventana control description, o a través del [ToolTip](../Proprietes/2510005.md) Property del control). 

- El panel "Modificador" contiene opciones específicas para las ventanas que se muestran en modo de información sobre herramientas. Las opciones son las siguientes: 

	- "Modo tooltip": Activa o desactiva el modo de visualización de la información sobre herramientas. Si se activa este modo, se tendrán en cuenta las demás opciones del modo de información sobre herramientas. 

	- "modo Tooltip: X offset": Define el desplazamiento X de la ventana (mostrada como información sobre herramientas) en relación con el cursor del ratón. El cursor del ratón corresponde al punto (0,0). 

	- "modo Tooltip: Y offset": Define el desplazamiento Y de la ventana (mostrada como información sobre herramientas) en relación con el cursor del ratón. El cursor del ratón corresponde al punto (0,0). 

	- "modo Tooltip: Punto caliente": Define el punto de la ventana (mostrado como información sobre herramientas) que se mostrará en la posición del cursor del ratón. Esta opción permite crear información sobre herramientas similar a un bocadillo..

## Funciones y propiedades de WLanguage
<a name="funciones_propiedades_wlanguage_ELTTEXTE000277"></a>
Se pueden utilizar varias funciones y propiedades de WLanguage para configurar los tooltips.

Las siguientes funciones y propiedades se pueden utilizar en las descripciones de herramientas personalizadas:

- Funciones de WLanguage: 
	


|   |   |
| --- | --- |
| [ShowToolTip](../WDLang1/3025014.md) | Habilita o deshabilita el mecanismo de visualización de la información de las herramientas de control. |
| [ToolTipDelay](../WDLang1/3025051.md) | Se usa para modificar:<br><br>	- el tiempo de espera antes de mostrar los tooltips.<br><br>	- la duración de visualización de las indicaciones de las herramientas.<br><br><br> |
| [TooltipMaxWidth](../WDLang1/1000022257.md) | Modifica el ancho máximo de la información sobre herramientas. |

- Propiedades de WLanguage: 
	


|   |   |
| --- | --- |
| [AutomaticTooltip](../Proprietes/1000018996.md) | La propiedad **automatic Tooltip** se utiliza para habilitar o deshabilitar los tooltips automatic en los controles List Box, Table y TreeView Table. |
| [ToolTip](../Proprietes/2510005.md) | El **ToolTip** Property se utiliza para get y cambiar el texto mostrado en el tooltip associated con una ventana o página control. |




## Etapa 6: Configuración del Groupware Usuario
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000081"></a>
Si el sitio está configurado para usar el groupware usuario, los parámetros de la base de datos del groupware pueden ser configurados durante la instalación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configurar el sitio
<a name="configurar_sitio_ELTTEXTE000105"></a><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=wwinst_param_gpu_new.gif)
Si el sitio utiliza el programa de grupo de usuarios: 

- versión 18 y posteriores: especifique el directorio de ubicación del archivo de derechos.

- versión compatible: especifique el directorio de ubicación del archivo de derechos y usuarios.




Estos archivos se pueden colocar en el directorio de datos del sitio o en un directorio específico.
<a name="NOTE3_2"></a>


### Observación
<a name="observacion_ELTPARAGRAPHE000029"></a>

Las trayectorias pueden ser relativas o absolutas.

- Si se utilizan algunas rutas relativas, se considerarán relativas al directorio de datos del sitio.

- Los caminos absolutos son caminos en el servidor de implementación.

## Presentación
<a name="presentacion_ELTTEXTE000122"></a>
Si su aplicación es multilingüe, la función multilingüe debe gestionarse también en las ventanas/páginas de groupware usuario. Por Default, las ventanas/páginas de groupware usuario soportan el inglés/americano y el francés.

Esta página de ayuda explica cómo hacerlo:

- Configurar el User Groupware para utilizarlo en una aplicación multilingüe. 

- Traducir las diferentes ventanas o páginas de groupware usuario.

- Ejecute la aplicación en el idioma solicitado.




Para obtener más información, consulte [Creación de una aplicación multilingüe](../WDLang1/3054008.md).

## Configurar el programa de grupo de usuarios para la función multilingüe
<a name="configurar_programa_grupo_usuarios_para_funcion_multilingue_ELTTEXTE000146"></a>
Para incluir las ventanas o páginas del User Groupware en una aplicación multilingüe:

1. Seleccione "Taller .. Configurar el"User Groupware".

2. Asegúrese de que las siguientes opciones estén seleccionadas:

	- Permitir la gestión de groupware usuario

	- Integración personalizada




3. Compruebe si está marcado "Marcha manual".

## Traducir las ventanas o páginas de groupware usuario
<a name="traducir_las_ventanas_paginas_groupware_usuario_ELTTEXTE000170"></a>


### Traducir las ventanas o páginas de las aplicaciones multilingües
<a name="traducir_las_ventanas_paginas_las_aplicaciones_multilingues_ELTPARAGRAPHE000049"></a>

Las ventanas o páginas de groupware usuario están disponibles en inglés y en francés por Default.

Para traducir estas ventanas o páginas a otros idiomas, todo lo que tienes que hacer es traducir los diferentes títulos, ... de estas ventanas o páginas (utilizando [WDMSG](../WDMsg/3518040.md) si es necesario, para extraer los mensajes a traducir). WDMSG es una herramienta opcional de WINDEV, WEBDEV o WINDEV Mobile utilizada para comprobar y comprobar todos los mensajes a traducir.

## Ejecutar la aplicación en el idioma solicitado
<a name="ejecutar_aplicacion_idioma_solicitado_ELTTEXTE000194"></a>
Para ejecutar la aplicación multilingüe mientras se gestiona el User Groupware, debe: 

1. Pregunte si es necesario) el idioma en el que se debe ejecutar la aplicación (por ejemplo, a través de una ventana o página específica.

2. Configurar el idioma de ejecución de la aplicación con [Nación](../WDLang1/3054014.md).

3. Abrir la primera ventana o página de gestión de la Agrupación de Usuarios con [gpwOpen](../WDLang6/3041002.md).





```wl
// Initialization code of project
// open the window for language selection
nLanguage is int = Open(WIN_Select_Language)
// Configure the language for the application
Nation(nLanguage)
// Start the first window of user groupware
gpwOpen()
```

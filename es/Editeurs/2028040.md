## Seleccionar soporte
<a name="seleccionar_soporte_ELTTEXTE000117"></a>
Al crear el programa instalación a través de la asistente, tiene la posibilidad de elegir las opciones relativas al apoyo de la instalación.

El programa instalación se genera en un directorio para ser copiado en diferentes soportes.

Por Default, el programa instalación se genera en el subdirectorio "Install" de su aplicación.

Los diferentes soportes son los siguientes:

- ***Múltiples directorios***(no disponible para las configuraciones compactas)
	Los archivos del programa instalación están agrupados en varios subdirectorios. El tamaño de los archivos encontrados en cada subdirectorio no excede el tamaño de un disquete (1.44 MB por Default). A continuación, el contenido de cada subdirectorio puede copiarse en un disquete.

- ***Directorio único***
	El programa instalación se crea en un solo directorio.

- ***Ejecutable autoextraíble***
	El programa instalación está creado como un ejecutable autoextraíble. Cuando este ejecutable está en las computadoras de los usuarios, el programa instalación se descomprime automáticamente y se inicia.
	Esta solución se recomienda si el programa instalación se distribuye a través de INTERNET.

- ***Soporte con autoejecución***
	El programa instalación se crea en un solo directorio para ser copiado en un CD.




Observación: Si se marca **Borrar el directorio antes de generar el instalación**, el directorio de generación del instalación se borrará antes de generar el nuevo programa instalación. Esta opción permite eliminar los archivos inútiles del directorio de generación del programa instalación.

## Seguridad
<a name="seguridad_ELTTEXTE000141"></a>
Se proponen varias opciones para asegurar la instalación: 

- **Protege la instalación con la contraseña**: Tiene la posibilidad de proteger la instalación con una contraseña (no disponible para las configuraciones compactas). En este caso, debe especificar la contraseña que desea utilizar. Esta contraseña será solicitada cuando se ejecute el programa instalación.

- **Firme la instalación con la Certificate**: Para garantizar la integridad del paquete de la instalación, tienes la posibilidad de firmar la instalación. Todo lo que tienes que hacer es seleccionar la Certificate para usar. La asistente propone automáticamente los certificados disponibles en el ordenador actual, en el almacén personal de certificados de Windows.

## Parámetros avanzados
<a name="parametros_avanzados_ELTTEXTE000165"></a>
Los parámetros avanzados sirven para modificar los diferentes parámetros (directorio utilizado, tamaño de un disquete, etc.) del soporte seleccionado.

Para ayudarle a elegir el soporte adaptado a su modalidad de instalación, consulte el documento [Resumen tabla para la instalación de una aplicación WINDEV](../Editeurs/2028005.md).

El botón "Opciones avanzadas del instalador" permite definir: 

- las opciones para la gestión de la framework del instalador (con personalización del instalador). 

- las opciones para el UAC del instalador. 




Para obtener más información, consulte [Opciones avanzadas de instalación](../Editeurs/2028108.md).

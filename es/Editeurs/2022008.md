
## Ampliar una ventana
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
Las interfaces de usuario modernas suelen utilizar ventanas más grandes y fuentes de mayor tamaño en los distintos controles. 

A partir de la versión 28, puede aumentar automáticamente el tamaño de letra de todos los controles de una selección, ventana o proyecto. Esto evita tener que cambiar la plantilla y redimensionar los controles uno por uno. 

Cuando se cambia el tamaño de la fuente, todos los controles se amplían y desplazan automáticamente para conservar el diseño actual. La ventana también se amplía, si es necesario. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000135"></a>
Para ampliar una ventana: 

1. Abra la ventana deseada en el editor. 

2. En la pestaña "Alineación", en el grupo "Otras alineaciones", despliegue "Ampliar
Espaciar" y seleccione "Ampliar". 

3. En la ventana que aparece, especifique las opciones de ampliación: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Agrandir_la_fen%EAtre%20-%20HC%20N%B0002.gif)
 Puede: 

	- especificar el factor de escala. 

	- especificar si la fuente y los estilos deben ampliarse. En este caso, puede: 

		- Crear sobrescrituras de estilo: el estilo original se sobrescribirá para aplicar los cambios. 

		- Crear y utilizar una plantilla ampliada: en este caso, se genera una nueva plantilla basada en la plantilla actual. Por ejemplo, si la ventana utiliza la plantilla "Phoenix", con un factor de escala de 25%, se creará una plantilla "Phoenix (x1.25)".  




4. Valide. Las opciones elegidas se aplican. 




Observaciones: 

- Puede aplicar cambios de tamaño a la selección, a la ventana, o a todo el proyecto. 

- Si elige crear una plantilla "ampliada", podrá aplicarla a otros elementos del proyecto. 

- Si se actualiza la plantilla original, la plantilla "ampliada" se actualizará automáticamente. 

- Si la ventana contiene un modelo de ventanas, también deberá ampliar el modelo. También puede ampliar elementos de todo el proyecto (el modelo se ampliará automáticamente con los demás elementos del proyecto). 

- Los modelos de controles no se amplían.

## Presentación
<a name="presentacion_ELTTEXTE000417"></a>
Esta ventana le permite describir un estilo CSS que se utilizará para los controles de su proyecto WEBDEV. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Sty.gif&type=thumb)


Esta ventana permite:

- **Modificar las características de un estilo existente**. Todo lo que tiene que hacer es seleccionar el estilo, el estado y las características del estilo seleccionado, y luego validar.
	**Observaciones**: 

	- Si el estilo no es "local" para la control, estas modificaciones se aplicarán a todas las control que utilicen el estilo modificado.

	- Si el estilo es "local a la control", sólo se modificará el estilo de la control. Sin embargo, el estilo se incluirá en la página HTML (consecuencia: la carga de la página es más lenta).




- **Crear un nuevo estilo**: Haga clic en [ Create], introduzca el nombre y las características del estilo y valide.

- **Cambiar el nombre de un estilo**: Haga clic en el botón [ Rename] y especifique el nuevo nombre del estilo. 
	**Atención**: esta opción no se recomienda. En efecto, esta opción anula todos los vínculos entre los controles que utilizan el estilo y el estilo renombrado. Para los controles que estaban usando el estilo antes de ser renombrado, este estilo será cambiado a un estilo "Local".




Esta página de ayuda presenta las diferentes opciones disponibles en la pestaña "Texto" y en la pestaña "Background".. La vista previa se utiliza para ver dinámicamente las características del estilo.

## Ficha de texto
<a name="ficha_texto_ELTTEXTE000465"></a>


### Fuentes
<a name="fuentes_ELTPARAGRAPHE000222"></a>

Seleccione una de las familias de fuentes disponibles.

Una familia tipográfica es una set organizada de fuentes. Si el primer tipo de letra no está disponible en la el equipo del usuario de la web, se busca el segundo tipo de letra. Si esta fuente está disponible, se utiliza; en caso contrario, se busca la siguiente fuente.

El botón [...] se utiliza para crear nuevas familias de fuentes, así como para duplicar y modificar las familias existentes. 

Para obtener más información, consulte [Familia de la fuente](../Editeurs/2016022.md). 
<a name="NOTE3_2"></a>


### envergadura
<a name="envergadura_ELTPARAGRAPHE000244"></a>

Seleccione el tamaño de la fuente. Los tamaños disponibles son:

- Un tamaño definido, especificado en una unidad concreta (píxeles, centímetros, milímetros, pulgadas, punto, pica, porcentaje, ...).

- Un tamaño relativo a la fuente actual (cuando se muestra la página):

- xx-small: Tamaño extremadamente pequeño

- x-small: Tamaño extra pequeño

- small: Tamaño pequeño

- medio: Tamaño estándar

- x-large: Tamaño grande

- xx-large: Tamaño extra grande

- -1: el tamaño más pequeño

- +1: el tamaño más grande
	



<a name="NOTE3_3"></a>


### Decoración
<a name="decoracion_ELTPARAGRAPHE000263"></a>

Opciones adicionales para formatear la fuente. Estas opciones sirven para subrayar los enlaces o tacharlos, por ejemplo.

- nono: sin decoración adicional

- Tachado: el texto es tachado: a Line se dibuja a través del texto. Ejemplo: a

- subrayar: el texto está subrayado. Ejemplo: <u><u><u><u>B</u></u></u></u>

- Parpadeo: el texto parpadea (esta opción produce un efecto de parpadeo sólo en el navegador Netscape)

- Resaltado: una línea se dibuja Above el texto

### Otros parámetros de la fuente
<a name="otros_parametros_fuente_ELTPARAGRAPHE000279"></a>


|   |   |
| --- | --- |
| Color | Seleccione la Color de la fuente. |
| Peso | Grosor del carácter. Usted tiene la capacidad de seleccionar:<br><br>- un peso preestablecido (Normal, negrita, pesado, ligero)<br><br>- un espesor (de 100 a 900)<br><br><br> |
| sangrado | Guión para la primera Line. Especifica un valor que puede ser expresado en píxeles, centímetros, milímetros, pulgadas, punto, pica, porcentaje, ... |
| Estilo | Estilo de los personajes: Normal, cursiva, oblicua |
| Espacio entre caracteres | Espacio entre las letras. Especifica un valor que puede ser expresado en píxeles, centímetros, milímetros, pulgadas, punto, pica, porcentaje, etc. |
| Variant | Modo de visualización de los caracteres: estándar o minúsculas. |
| Interlineado | Espacio entre las líneas. Especifica un valor que puede ser expresado en píxeles, centímetros, milímetros, pulgadas, punto, pica, porcentaje, etc. |
| May/Min | Formato del personaje: 1ª letra en mayúscula, mayúscula, minúscula o ninguna (por Default) |
| Separación silábica | Gestión de la separación de palabras. Puede utilizar el guión estándar o cortar las palabras si es necesario. |
| Alineación | Presentación del texto: alineado a la izquierda, alineado a la derecha o justificado (alineado a la izquierda y a la derecha). |
| Alineación vertical | Alineación del texto en altura en relación con el área disponible. |

## Ficha de Background
<a name="ficha_background_ELTTEXTE000513"></a>
Las siguientes características pueden modificarse en la pestaña "Background":


|   |   |
| --- | --- |
| Color | Seleccione el fondo Color del control. |
| Image | Imagen mostrada en el control Background. El Image seleccionado debe encontrarse en el directorio Image del proyecto (&lt;NombredelProyecto>_WEB). |
| Mosaico | repetition de la imagen en el control. La Image se puede alicatar:<br><br>- en azulejos (opción repetida)<br><br>- horizontalmente (opción X repetida)<br><br>- verticalmente (opción Y repetida)<br><br><br> |
| Posición fija | Si esta opción está marcada, la Image no se desplazará cuando se utilice la barra de desplazamiento. |
| Posición horizontal y vertical | Posición de la Image en relación con la control. El valor puede expresarse en píxeles, centímetros, milímetros, pulgadas, punto, pica, porcentaje, ... |

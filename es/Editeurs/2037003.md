## Configurando el servidor para el protocolo CGI (AWP)
<a name="configurando_servidor_para_protocolo_cgi_awp_ELTTEXTE000098"></a>
Para configurar el servidor web:

1. Muestra la ventana de configuración del servidor.

2. Haga clic en "Configuración virtual de Default...".

3. Seleccione la pestaña "CGI estándar".

4. Defina el alias para el protocolo CGI:

	- Virtual: /WD28 0AWP

	- Current: C:\\WEBDEV\\PROGRAMS\\ENGINE\\WINDOWS\\AWP (nombre del directorio instalación del protocolo CGI "WD28 0AWP.EXE")

	- Haga clic en el botón "AGREGAR

## Configuración del servidor para los sitios WEBDEV
<a name="configuracion_del_servidor_para_los_sitios_webdev_ELTTEXTE000122"></a>
Para configurar el servidor web: 

1. Muestra la ventana de configuración del servidor.

2. Haga clic en "Configuración virtual de Default...".

3. Seleccione la pestaña "CGI estándar".

4. Para cada aplicación WEBDEV, defina el alias de las imágenes de la aplicación:

	- Haga clic en la pestaña "Alias

		- Virtual: /&lt;Nombre_del_proyecto&gt;_WEB (MyApp_WEB por ejemplo)

		- Current: nombre del directorio para las imágenes de la aplicación  
						(por ejemplo, C:\\WDPROJECT\\MyApp\\MyApp_WEB)




- Haga clic en el botón "AGREGAR

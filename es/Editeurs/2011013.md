


## Relaciones en un MDL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
Las relaciones se utilizan para vincular entidades entre sí o para vincular asociaciones con entidades.

La editor de análisis permite crear fácilmente relaciones.

Cuando [generar el MDL a partir del MDL](../Editeurs/2011020.md), las relaciones se convertirán automáticamente en [enlaces](../Editeurs/2011008.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Crear una relación
<a name="crear_una_relacion_ELTTEXTE000147"></a>
Para crear una relación:

1. En la pestaña "MCD", en el grupo "Creación", despliegue "Nuevo" y seleccione "Trazar una relación". El puntero se convierte en un lápiz.

2. Seleccione:

	- las dos entidades a vincular.

	- a la entidad y a la asociación a vincular.




3. Definir los puntos cardinales de la relación.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Definición de las cardenalidades
<a name="definicion_las_cardenalidades_ELTTEXTE000171"></a>
Definir los cardenales:

1. Seleccione la relación cuyas cardinalidades se deben definir.

2. En la pestaña "MCD", en el grupo "Elemento actual", haga clic en "Descripción".

3. Escoge la cardinalidad entre las propuestas.

4. Validar la description de los cardenales.




Para más detalles sobre las cardinalidades, véase [Cardinalidades](../Editeurs/2011014.md).

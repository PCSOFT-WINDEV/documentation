


## Ventana del iPad: Asistente de creación de ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000214"></a>
La asistente para la creación de ventanas para el iPad permite crear fácilmente una ventana que contenga todos los elementos principales de una ventana para una aplicación del iPad.

Estos diferentes elementos se pueden añadir en cualquier momento en la ventana. 

También puede configurar diferentes opciones en la ventana description. 

Al final de la asistente, se crea la ventana bajo el editor con las características especificadas. Además de la ventana, puede ver los siguientes elementos en el editor: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fen%EAtre_iphone_Editeur%20-%20HC%20N%B0002.gif&type=thumb)


- la barra del sistema, 

- las zonas que corresponden al modo vertical u horizontal (representadas por guiones).

<a name="NOTE3_1"></a>


## Los diferentes pasos de la asistente
<a name="los_diferentes_pasos_asistente_ELTTEXTE000238"></a>


### Orientación de la ventana
<a name="orientacion_ventana_ELTPARAGRAPHE000028"></a>

Hay varias opciones disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_Orientation.gif)
 Estas opciones pueden modificarse en el editor, en la pestaña "Detalles" de la ventana description ("Orientación"). 

También tiene la capacidad de especificar si las barras de desplazamiento de automatic deben ser mostradas. Las barras de desplazamiento se mostrarán automáticamente si el tamaño de la ventana es mayor que la resolución de la pantalla del dispositivo utilizado.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_FenDetail.gif)



### Contenido de la ventana: simple o doble
<a name="contenido_ventana_simple_doble_ELTPARAGRAPHE000038"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=iPad_Simple_double.gif)
Es posible crear: 

- una ventana **sencilla**: los datos se muestran en toda la ventana. 
	En este caso, los pasos siguientes de la asistente le permitirán definir el contenido de la ventana. 

- una ventana **doble**: la ventana se divide en dos secciones. Este tipo de interfaz le permite implementar un menú o una selección de opciones en la sección izquierda, mientras que los detalles se muestran a la derecha.. 
	En este caso, los pasos siguientes de la asistente permiten definir: 

	- el contenido de la sección izquierda. 

	- el contenido de la sección derecha. Este contenido puede ser: 

		- estática (definida en el editor): la asistente propone definir el contenido de la sección. 

		- una ventana interna. En este caso, se crea automáticamente una control Ventana interna en la ventana. 









## Definición del contenido
<a name="definicion_del_contenido_ELTTEXTE000268"></a>


### ¿Cuándo se debe definir el contenido?
<a name="¿cuando_debe_definir_contenido_ELTPARAGRAPHE000058"></a>

La asistente permite definir el contenido de la ventana en los siguientes casos: 

- una simple ventana.

- sección izquierda de una ventana doble. 

- sección derecha de una ventana doble con contenido estático.





### Contenido de la ventana
<a name="contenido_ventana_ELTPARAGRAPHE000067"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_Contenu.gif)
Tiene la posibilidad de crear una ventana: 

- con un **Action Bar**. Esta control se utiliza para mostrar diversas informaciones: el ícono de la aplicación, el título de la ventana, algunas opciones de menú y el menú desplegable de la ventana, ...
	Para añadir una "Action Bar" en una ventana: en la pestaña "Ventana", en el grupo "Barras y menús", haga clic en "Otras barras" y seleccione "Action Bar".

- con un **looper o una zona multilínea**. Este tipo de ventana se utiliza para configurar una o más opciones, para seleccionar una opción, ....

- con una **barra de pestañas**. En la mayoría de los casos, se utiliza una "barra de pestañas" para acceder a las distintas ventanas de la aplicación.
	Para añadir una barra de pestañas a una ventana: en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Otras barras" y seleccione "Barra de pestañas manual (iOS Tab Bar)".

- con una **barra de herramientas**. En la mayoría de los casos, la barra de herramientas se utiliza para realizar acciones en la ventana actual. 
	Para añadir una barra de herramientas a una ventana: en la pestaña "Ventana", en el grupo "Barras y menús", despliegue "Otras barras" y seleccione "Toobar (iOS ToolBar)".




**Diferencia entre una barra de pestañas y una barra de herramientas**: 

- Una barra de herramientas propone acciones en la vista actual.

- Una barra de pestañas corresponde a las pestañas.





### Configuración del looper o de la zona multilínea
<a name="configuracion_del_looper_zona_multilinea_ELTPARAGRAPHE000125"></a>

Si ha elegido crear una ventana con looper o con zona multilínea, los diferentes planos de asistente le permiten configurar este elemento. 

- Hay varios tipos de bucles preestablecidos o zonas multilínea disponibles: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_Ass_TypeZR.gif)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ass_typeZM.gif)


- El siguiente paso se utiliza para configurar los elementos mostrados en el looper o en la zona multilínea. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IOS_ass_ParamZR.gif)


- Para un looper, tienes la posibilidad de elegir cómo se llenará el looper (programación, enlace de archivo, enlace de Variable).




Cuando se valida la asistente, el looper o la zona multilínea seleccionada se crea automáticamente en la ventana. 

Usted tiene la capacidad de escribir el código utilizado para manipular y hacer que funcione.. 

**Para un looper** , se deben modificar los siguientes eventos:

- Código de inicialización del looper: esta Event se utiliza, por ejemplo, para llenar el bucle. 

- Código de selección de filas: esta Event se utiliza para gestionar las acciones realizadas cuando el usuario hace clic en la flecha estándar. 

- Haga clic en el código de IMG_Disclosure Image: esta Event se utiliza para gestionar el clic en la bala personalizada. 




Para obtener más información, consulte: 

- [Looper](../WDChamp/1013099.md) control. 

- [Zona multilínea](../WDChamp/1000024001.md) control. 





### Barra de pestañas o barra de herramientas
<a name="barra_pestanas_barra_herramientas_ELTPARAGRAPHE000159"></a>

Si se ha seleccionado esta opción, especifique los parámetros de la barra: 

- Color

- Estilo

- Botones mostrados




Al final de la asistente, la ventana se crea con la barra especificada. Todo lo que tienes que hacer es introducir el código de los botones.


## Análisis asociado a un proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000070"></a>
Cuando un proyecto WINDEV, WEBDEV o WINDEV Mobile utiliza archivos de datos, debe estar asociado a un análisis. Un análisis permite describir las estructuras de datos (archivos de datos, elementos/campos, etc.) utilizadas en el proyecto.

El editor de análisis permite crear un análisis fácilmente.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=P47-Analyse%20Structure%20de%20la%20base%20de%20donn%E9es.gif)


Existen dos métodos para crear un análisis:

- ** Método 1**: Crear el análisis directamente (es decir, un Modelo Lógico de Datos (MLD)).

- ** Método 2**: Crear un Modelo Conceptual de Datos (MCD) y luego, crear el análisis automáticamente a partir del MCD.




**Observación**: En una aplicación WINDEV, WEBDEV o WINDEV Mobile, los términos "MLD" y "Analysis" se utilizan indistintamente para definir la estructura de la base de datos asociada a un proyecto.

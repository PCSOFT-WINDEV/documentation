


## set de fuentes (sitio WEBDEV)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000175"></a>
El navegador soporta las fuentes más comunes de Default: Arial, Verdana, ...

WEBDEV le ofrece la posibilidad de utilizar fuentes personalizadas que difieren de las propuestas por Default. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000199"></a>


### Gestión de fuentes
<a name="gestion_fuentes_ELTPARAGRAPHE000020"></a>

Al crear o modificar un estilo, tiene la posibilidad de seleccionar una fuente entre las fuentes estándar.

También tiene la posibilidad de seleccionar nuevas fuentes. Estas fuentes pueden provenir de: 

- el ordenador de desarrollo (estas fuentes han sido instaladas en su ordenador en Windows). 

- una operación de descarga (por ejemplo, descarga de fuentes de Google).



<a name="NOTE2_2"></a>


### Creación de una set de fuentes
<a name="creacion_una_set_fuentes_ELTPARAGRAPHE000032"></a>

Para añadir una set de fuentes: 

1. Abra la ventana de gestión de estilos (pestaña "Estilo" del control description). 

2. En el área "Fuentes", haga clic en el botón "... ".: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_1.gif)


3. Aparece la ventana para la gestión de fuentes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_2.gif)
 Esta ventana presenta los conjuntos de fuentes existentes. 

4. Haga clic en "Nuevo" para crear una nueva set de fuentes. 

5. En el asistente que se inicia, debe seleccionar los diferentes tipos de letra que componen el set de fuentes. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_3.gif)
Es posible: 

	- seleccionar una fuente descargable. Esta fuente se descargará automáticamente en el ordenador del usuario..  

	- seleccionar una fuente del sistema, que sólo se mostrará si está instalada en el ordenador del usuario. Si se selecciona esta opción, deberá seleccionar la fuente deseada.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_4.gif)


	- seleccionar una fuente genérica, que estará disponible para todos los navegadores. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_5.gif)
Observación: Este tipo de fuente es obligatorio en una set de fuentes. De este modo, el texto se mostrará independientemente del ordenador del usuario.. 

6. Seleccione el tipo de letra deseado y, a continuación, el tipo de letra que desea añadir a la set de fuentes. 

7. Aparece la ventana con los diferentes tipos de letra de la set. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_6.gif)
Es posible: 

	- Añadir una nueva fuente (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_6%203.gif)). La asistente comienza de nuevo. 

	- Borrar la fuente seleccionada (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_6%202.gif)). 

	- Cambiar el orden de las fuentes (con las flechas ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB_6%201.gif)). 




8. Validar la ventana de creación de la set de fuentes. 

9. Validar la ventana para la gestión de los conjuntos de fuentes. La set de fuentes seleccionada se aplica al estilo actual. 



<a name="NOTE2_3"></a>


### Importar una fuente
<a name="importar_una_fuente_ELTPARAGRAPHE000075"></a>

Para importar una nueva fuente: 

1. En la ventana de gestión de juegos de fuentes, haga clic en "Nuevo".. 

2. En la asistente que se inicia, seleccione "Fuente descargable".. Pase a la etapa siguiente del asistente. 

3. En la ventana que aparece, haga clic en "Importar una fuente".. 

4. La importación de la fuente asistente comienza. 

5. Seleccione el modo para importar la fuente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB%20-%20HC%20N%B0007.gif)


	- Importar una fuente de Google Web Fonts: 

	- La asistente propone la lista de fuentes de Google para importar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_perso_WB%20-%20HC%20N%B0008.gif)


**Observaciones**: 

- Asegúrese de que los archivos de las fuentes utilizados estén libres de derechos o que posee una licencia que le permite distribuirlas. 

- Si importa una fuente y crea una set de fuentes, los archivos correspondientes a esta fuente se copian en el subdirectorio "&lt;Nombre_del_proyecto&gt;_WEB\\Styles\\fonts".. Por lo tanto, se tendrán en cuenta automáticamente a la hora de instalar el sitio.




**Atención: set de fuentes que contienen fuentes importadas:** 
Antes de utilizar una fuente descargada, debe integrarse en una set de fuentes similares. 

Si la set de fuentes contiene sólo la fuente descargable, puede darse el siguiente caso: si la fuente no se encuentra en el navegador: 

- la fuente Times es mostrada por Default mientras se descarga la fuente solicitada. 

- si la fuente no se puede descargar, se sigue mostrando la fuente Times. 




Para evitar la visualización de una fuente no deseada, es aconsejable crear una set de fuentes que contenga los siguientes elementos: 

- las fuentes para descargar,

- una o más fuentes de sustitución correspondientes a las fuentes de los sistemas principales,

- una o más fuentes correspondientes a las fuentes más comunes,

- las familias tipográficas del tipo associated. La lista de tipos de familias de fuentes es la siguiente: 

	- "cursiva": estilo manuscrito,

	- "fantasía": como Impact,

	- "monoespacio" como Courier,

	- "sans-serif" como Arial,

	- "Serif" como el Times.





Si no se encuentra la primera fuente, se utilizará la primera que se encuentre en la lista y en el equipo. 

**Ejemplo**: Se define la siguiente set de fuentes: "Open Sans, Segoe UI, Arial, Helvetica, sans-serif". Si no se encuentra el tipo de letra "Open Sans", se utilizará el tipo de letra "Segoe UI", de lo contrario se utilizará el tipo de letra Arial o Helvetica. Estas fuentes son bien conocidas para tener alternativas correctas. Si no se encuentran estas dos fuentes, se usará la primera fuente disponible sin serifa.

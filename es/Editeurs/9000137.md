
## copy entre los editores
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten realizar copias (o arrastrar y soltar) de controles entre los productos y sus editores.. 

Tienes la habilidad de controlar el copy: 

- [del editor de ventanas al editor de páginas](../Editeurs/9000138.md). 

- [desde el editor de ventanas a editor de reportes](../Editeurs/9000138.md). 

- [del editor de páginas al editor de ventanas](../Editeurs/9000139.md). 

- [desde el editor de páginas a la página de editor de reportes](../Editeurs/9000139.md). 

- [desde editor de reportes al editor de ventanas](../Editeurs/9000140.md). 

- [de la editor de reportes al editor de páginas](../Editeurs/9000140.md). 




Durante esta copia, los controles que se encuentran en los diferentes editores se mantendrán así como el código associated. 

En algunos casos, los controles se transformarán durante el copy para adaptarse mejor al editor de destino.

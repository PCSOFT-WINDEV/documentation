## Presentación
<a name="presentacion_ELTTEXTE000179"></a>
El siguiente mecanismo permite a los desarrolladores escribir código WLanguage (funciones, sentencias, etc.) más fácilmente en el editor de código:

- **Visualización de la sintaxis de la función que se está introduciendo mediante diferentes tooltips**. Esta ayuda se muestra cuando se escribe el nombre de la función por completo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=saisie_assist%201.gif)
 Si hay varias sintaxis disponibles para la función actual, se muestran. 
	Para modificar la sintaxis de Active, presione Ctrl + Abajo o Ctrl + Arriba.

- **Tooltip con el description del parámetro que se pasa por encima**. Aparece este tooltip: 

	- cuando el cursor del ratón se encuentra en la posición de un parámetro a escribir (el cursor del ratón pasa por encima de la sintaxis en el tooltip).

	- si se utiliza Alt + F1 en un parámetro que ya está escrito. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=saisie_assist.gif&type=thumb)





- **Tooltip con el description del resultado**. Aparece este tooltip: 

	- cuando el cursor del ratón se encuentra en el nombre de la función (al escribir una nueva función). 

	- si se pulsa Alt + F1 en el nombre de una función que ya está escrita. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=saisie_assist_result.gif&type=thumb)





- **Tooltip con las sintaxis disponibles en las declaraciones de WLanguage**. Esta ayuda aparece cuando se introduce el primer término de la sentencia WLanguage. En algunos casos, la asistente también sugiere sintaxis similares. Por ejemplo: 

	- Palabra clave WHILE: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_a_la_saisie_2%20-%20HC%20N%B0001.gif)


	- Palabra clave IF: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Aide_a_la_saisie_2%20-%20HC%20N%B0002.gif)

## Modo de funcionamiento
<a name="modo_funcionamiento_ELTTEXTE000203"></a>


- La entrada asistida está disponible en cualquier momento en todos los proyectos. 

- La Icon que se encuentra en el tooltip para describir el parámetro actual indica el tipo de parámetro esperado. Para más detalles sobre estos iconos y su significado, consulte la sección "Notas" de esta Page.

- Los diferentes tooltips también se muestran cuando se escribe la llamada a los procedimientos o métodos. Para más detalles, consulte la ayuda sobre [la documentación de procedimientos y métodos de automatic](../Motscles/1513006.md).

- La sintaxis actual de la función WLanguage se muestra siempre en la barra de estado.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SyntaxeFonction.gif&type=thumb)


- El editor de código también muestra las opciones de finalización del código al introducir los parámetros de la función, por ejemplo. Para obtener más información, consulte [Autocompletado](../Editeurs/2013001.md).


## Gestión de los usuarios de una aplicación instalada en red
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000099"></a>
La gestión de los usuarios le permite informarse desde el servidor:

- los usuarios actualmente conectados a la aplicación,

- para cada usuario conectado:

	- el equipo solía iniciar la aplicación,

	- la versión de la aplicación utilizada.







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Características del usuario
<a name="caracteristicas_del_usuario_ELTTEXTE000123"></a>
Para ver las características de los usuarios de una aplicación en el servidor:

1. Inicie la aplicación WDADMINEXE que se encuentra en el subdirectorio INSTALL del directorio instalación del servidor.

2. Mostrar la pestaña "Usuarios.

Para obtener más información, consulte [Comprobación remota de los ejecutables de WINDEV](../WDAdminExe/2028052.md).

## Notas
<a name="notas_ELTTEXTE000182"></a>


### Si decide no incluir la gestión de usuarios
<a name="decide_incluir_gestion_usuarios_ELTPARAGRAPHE000043"></a>

Si la gestión de los usuarios no está incluida en el programa instalación, WDADMINEXE seguirá instalado en el servidor. Sin embargo:

- los archivos LOKxxx.tmp no se crearán,

- la pestaña "Usuarios" será Grayed,

- la remota control de la aplicación estará disponible. El control remoto le permite desconectar a los usuarios en cualquier momento.. Para obtener más información, consulte [Comprobación remota de los ejecutables de WINDEV](../WDAdminExe/2028052.md).

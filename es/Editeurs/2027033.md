<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000086"></a>
El panel "Vista de código sensible al contexto" permite ver y editar el código del elemento (ventana, página, reporte, control, etc) seleccionado en el editor.

Las características del código mostrado en el panel corresponden a las características del editor de código.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=VoletCodectx.gif)
 El menú emergente contiene las opciones de el menú contextual del editor de código.

## Visualización del panel "Vista de código sensible al contexto"
<a name="visualizacion_del_panel_vista_codigo_sensible_contexto_ELTTEXTE000110"></a>
**Para mostrar el panel "Vista de código sensible al contexto "**, vaya a la pestaña "Inicio", al grupo "Entorno", expanda "Paneles", seleccione "Paneles" y, a continuación, seleccione "Vista de código sensible al contexto".

Para más detalles sobre la gestión de los paneles, véase [Manipulación de los cristales acoplables](../Editeurs/2027001.md).




## Creación de anclas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000151"></a>
Los anclajes sirven para definir puntos concretos en una ayuda Page.

Entonces, podrá posicionarse directamente sobre este punto en la ayuda Page a partir de un enlace por ejemplo.

Las anclas son muy useful en largas páginas de ayuda. Permiten al lector moverse fácilmente dentro de la Page.

Se puede crear un ancla:

- en una palabra o en un grupo de palabras.

- en un Image.




Para ver fácilmente un ancla en el editor de ayuda de WINDEV, el texto correspondiente al ancla se ilumina en verde claro.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000175"></a>


### Creación de un ancla
<a name="creacion_ancla_ELTPARAGRAPHE000029"></a>

Para crear un ancla:

1. Mostrar la ayuda Page que debe contener el ancla en el editor de ayuda.

2. Seleccione el grupo de palabras para el que se debe crear el ancla.

3. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Ancla". 
	Observación: También tiene la posibilidad de seleccionar "Insertar un ancla" en el menú contextual.

4. En la ventana que aparece, especifique el nombre del anclaje. Este nombre se usa para identificar el ancla cuando se crean enlaces hacia este ancla.. El texto seleccionado es propuesto por Default.

5. Valide.



<a name="NOTE2_2"></a>


### Creación de un enlace hacia un ancla
<a name="creacion_enlace_hacia_ancla_ELTPARAGRAPHE000056"></a>

Para crear un enlace hacia un ancla:

1. Mostrar la ayuda Page que contendrá el enlace en el editor de ayuda.

2. Seleccione el grupo de palabras para el que debe crearse el enlace.

3. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Enlace". 
	Observación: También puede seleccionar "Insertar un enlace" en el menú contextual. Aparece la ventana de propiedades del enlace.

4. Seleccione las siguientes características:

	- Hipervínculo a: "Enlace a un ancla". 

	- Página de destino: seleccione la Page deseada y ancle




5. Valide.



<a name="NOTE2_3"></a>


### Obtener el nombre de un ancla
<a name="obtener_nombre_ancla_ELTPARAGRAPHE000086"></a>

Para mostrar el nombre de un ancla:

1. Posicione el cursor del ratón sobre el texto del ancla (resaltado en verde claro).

2. Haga doble clic en el ancla o seleccione "Propiedades de un ancla" en el menú contextual. El nombre del ancla puede ser modificado.



<a name="NOTE2_4"></a>


### Eliminación de un ancla
<a name="eliminacion_ancla_ELTPARAGRAPHE000096"></a>

Para borrar un ancla:

1. Abra la ayuda Page que contiene el ancla en el editor de ayuda. 

2. Posicione el cursor del ratón sobre la palabra o sobre la Image correspondiente al ancla.

3. Seleccione "Eliminar un ancla" en el menú contextual. El ancla se borra automáticamente.




**Atención**: Los enlaces que apuntan a este ancla no se eliminan. Pueden producirse errores durante la compilación si no se modifican los enlaces.


## Traspaso de las diferencias de un elemento a otro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000189"></a>
El comparador del proyecto se utiliza para comparar dos elementos y para listar las diferencias encontradas.

Para obtener más información, consulte [Comparador de proyectos](../Editeurs/2030024.md).

Una vez que se han enumerado las diferencias, se puede pasar de un elemento a otro:

- las diferencias de código.

- las diferencias de características (tamaño y posición de una ventana o página, por ejemplo).

- los controles que se encuentran en uno de los dos elementos solamente (ventana, página o reporte).

- los procedimientos locales que se encuentran sólo en uno de los dos elementos (ventana, página o reporte).

- los procedimientos globales que se encuentran en uno de los dos conjuntos de procedimientos solamente.

- los métodos observados únicamente en una de las dos clases.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000213"></a>


### Arrastrar las diferencias de código de un elemento a otro
<a name="arrastrar_las_diferencias_codigo_elemento_otro_ELTPARAGRAPHE000040"></a>

Trasladar las diferencias de código de un elemento a otro:

1. Seleccione las líneas de código que desea arrastrar.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar el código seleccionado en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar el código seleccionado en la ventana izquierda a la ventana derecha.





**Limitación:** Los procesos associated con el proyecto no pueden ser llevados a cabo.
<a name="NOTE2_2"></a>


### Transferencia de las diferencias de características de un elemento a otro
<a name="transferencia_las_diferencias_caracteristicas_elemento_otro_ELTPARAGRAPHE000057"></a>

Trasladar las diferencias de características de un elemento a otro:

1. Seleccione la Line de la característica a llevar.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar la característica seleccionada en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar la característica seleccionada en la ventana izquierda a la ventana derecha.





**Limitación:** El nombre lógico de un elemento no se puede arrastrar.
<a name="NOTE2_3"></a>


### Llevar los controles que se encuentran en uno de los dos elementos solamente (ventana, página o reporte)
<a name="llevar_los_controles_que_encuentran_uno_los_dos_elementos_solamente_ventana_pagina_reporte_ELTPARAGRAPHE000074"></a>

Para llevar a cabo los controles que se encuentran en uno de los dos elementos solamente (ventana, página o reporte):

1. Selecciona la control para llevarla a cabo.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar el control seleccionado en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar el control seleccionado en la ventana izquierda a la ventana derecha.





**Limitación:** Los siguientes elementos no pueden ser transferidos:

- las columnas de una tabla. Sin embargo, una control Tabla puede ser prorrogada.

- los atributos de un looper. Sin embargo, un looper puede ser transportado.

- un control perteneciente a otro control (supercontrol, looper control, célula, ventana interna, ...). En este caso, el "contenedor" control también debe ser transportado. 
	Sin embargo, al traspasar una control que contenga otras control s (supercontrol s, control, celda, ventana interna, etc.), tiene la posibilidad de traspasar algunas de las control s que se encuentran en esta control.

- los menús emergentes y/o sus opciones de menú.



<a name="NOTE2_4"></a>


### Llevar a cabo los procedimientos locales que se encuentran en uno de los dos elementos solamente (ventana, página o reporte).
<a name="llevar_cabo_los_procedimientos_locales_que_encuentran_uno_los_dos_elementos_solamente_ventana_pagina_reporte_ELTPARAGRAPHE000098"></a>

Para llevar a cabo los procedimientos locales que se encuentran en uno de los dos elementos solamente (ventana, página o reporte):

1. Selecciona la Procedure local para llevarla a cabo.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar el procedure local seleccionado en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar el procedure local seleccionado en la ventana izquierda a la ventana derecha.






<a name="NOTE2_5"></a>


### Llevar a cabo los procedimientos globales que se encuentran en uno de los dos conjuntos de procedimientos solamente.
<a name="llevar_cabo_los_procedimientos_globales_que_encuentran_uno_los_dos_conjuntos_procedimientos_solamente_ELTPARAGRAPHE000113"></a>

Llevar a cabo los procedimientos globales que se encuentran en uno de los dos conjuntos de procedimientos solamente:

1. Selecciona el Procedure global para llevar a cabo.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar el procedure global seleccionado en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar el procedure global seleccionado en la ventana izquierda a la ventana derecha.





**Limitación**: Los conjuntos de procedimientos no se pueden propagar. Para llevar a cabo una Procedure global, el correspondiente conjunto de Procedure s debe encontrarse en los dos proyectos que se comparan.
<a name="NOTE2_6"></a>


### Prórroga de los métodos encontrados en una sola de las dos clases
<a name="prorroga_los_metodos_encontrados_una_sola_las_dos_clases_ELTPARAGRAPHE000130"></a>

Trasladar los métodos encontrados en una de las dos clases solamente:

1. Seleccione el método de transferencia.

2. Clic:

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%201.GIF) para trasladar el método seleccionado en la ventana derecha a la ventana izquierda.

	- ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Merge%20code%20element%20-%20HC%20N%B0001%202.GIF) para trasladar el método seleccionado en la ventana izquierda a la ventana derecha.





**Limitación:** Una clase no puede ser prorrogada. Para trasladar un método, la clase correspondiente debe encontrarse en los dos proyectos que se comparan.

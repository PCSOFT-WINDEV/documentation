<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
La ventana description de una consulta combinada consta de varias secciones. Esta ventana de description se utiliza para:

- cambiar las consultas utilizadas en la consulta.

- modificar el orden de ejecución de la consulta.
	Recordatorio: Los elementos que aparecen en el resultado de la consulta combinada corresponden a los elementos de la primera consulta realizada.

- ver la consulta en código SQL o en lenguaje natural.

- gestionar los registros "duplicados".

- ordenar los registros.


Se muestra esta ventana de description:

- al crear una consulta Select. 
	Recordatorio: Para crear una consulta: 

	- Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- Aparece la ventana de creación: haga clic en "Consultas".




- a partir de una consulta combinada mostrada en el editor de consultas ("Consulta description" en el menú contextual de la consulta).

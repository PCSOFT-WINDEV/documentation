


## Sincronizar el análisis con bases de datos externas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000116"></a>
El análisis debe estar sincronizado con una base de datos externa si se utiliza:

- archivos de datos de una base de datos externa (Oracle, SQL Server, etc.).

- archivos de datos cuyos datos no se han convertido al formato HFSQL Classic.




Esta sincronización se utiliza para tener en cuenta las posibles modificaciones realizadas en la estructura de los ficheros de datos que se encuentran en la base de datos externa.

Es posible:

- forzar la sincronización desde la editor de análisis.

- programe un chequeo automatic para los cambios hechos en la base de datos externa al abrir el análisis.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000146"></a>


### Forzar la sincronización
<a name="forzar_sincronizacion_ELTPARAGRAPHE000030"></a>

Para forzar la sincronización: 

1. Abra el análisis que contiene los archivos de datos externos en el editor de análisis.

2. En la pestaña "Análisis", en el grupo "Análisis", despliegue "Sincronización " y seleccione "Actualizar análisis a partir de las bases de datos externas". La sincronización de la base de datos externa asistente comienza.

3. Seleccione las bases de datos utilizadas en el análisis que desea comprobar.

4. Especifique la clase de cheque y valide. Tienes la capacidad de comprobarlo:

	- los ficheros de datos (tablas) comunes a la base de datos externa y al análisis.

	- todos los ficheros de datos (tablas) que se encuentran en la base de datos externa.




5. Se muestra la lista de archivos de datos que no son consistentes con el análisis actual.. El análisis actual puede actualizarse. Seleccione las modificaciones que desea aplicar.

6. Programar (si es necesario) una verificación regular de la estructura de las bases de datos externas.. Esta comprobación se realizará automáticamente cuando se abra el análisis.. Especifique (si es necesario) la frecuencia de esta verificación.

7. Valide.



<a name="NOTE2_2"></a>


### Programación de una sincronización regular (cuando se abre el análisis)
<a name="programacion_una_sincronizacion_regular_cuando_abre_analisis_ELTPARAGRAPHE000060"></a>

1. Abra el análisis que contiene los archivos externos en el editor de análisis.

2. Abra la ventana de descripción del análisis: 

	- hacer clic con el botón derecho del ratón para abrir el menú contextual del esquema de análisis. 

	- seleccione "Análisis description".




3. En la pestaña "Opciones", seleccione la frecuencia de sincronización:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Synchro.gif)


4. Valide.

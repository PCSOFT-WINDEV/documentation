<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000119"></a>
Para cada elemento del proyecto (ventanas, informes,...), se calcula el tiempo total invertido:

- para todos los colaboradores del proyecto,

- para cada colaborador del proyecto.




Esta gestión de tiempo le permite ejecutar estadísticas sobre la duración del desarrollo de un proyecto y averiguar fácilmente cuáles son las partes de su proyecto que más tiempo consumen.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000143"></a>


### Ver el resumen tabla
<a name="ver_resumen_tabla_ELTPARAGRAPHE000023"></a>

Para ver el resumen tabla, vaya a la pestaña "Proyecto", grupo "Otras acciones", amplíe "Gestión del tiempo" y seleccione "Tiempo dedicado a los elementos del proyecto".
<a name="NOTE2_2"></a>


### Activar o desactivar la gestión del tiempo
<a name="activar_desactivar_gestion_del_tiempo_ELTPARAGRAPHE000042"></a>

Para activar o desactivar esta gestión de tiempo: 

- Abra la ventana de descripción del proyecto: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

- Vaya a la pestaña "Avanzado". 

- Marcar/Desmarcar "Habilitar la función para el recuento del tiempo empleado en los elementos del proyecto".

- Valide.



<a name="NOTE2_3"></a>


### Cálculo de duraciones
<a name="calculo_duraciones_ELTPARAGRAPHE000066"></a>

Si se abren varios elementos, sólo se utiliza el elemento actual al calcular la duración.

Si el editor actual es Inactive (otra aplicación tiene el foco por ejemplo), el cálculo de la duración se detiene automáticamente.

Si un elemento permanece abierto durante menos de 60 segundos o si permanece Inactive durante menos de 10 segundos, estas duraciones son ignoradas.

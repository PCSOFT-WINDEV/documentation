


## Edición de la hoja de estilo CSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
WEBDEV le ofrece la posibilidad de gestionar todos los estilos CSS utilizados en un proyecto.. 

Es posible:

- editar la hoja de estilo CSS de un proyecto.

- definir los estilos CSS de Default. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Edición de la hoja de estilo CSS de un proyecto
<a name="edicion_hoja_estilo_css_proyecto_ELTTEXTE000165"></a>
Editar la hoja de estilo CSS de un proyecto:

1. En la pestaña "Página", en el grupo "Tema y estilos", despliegue "Estilos" y seleccione "Editar estilos CSS".

2. Se muestra la lista de estilos CSS encontrados en el proyecto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Edition-StyleCSS.gif&type=thumb)





Esta ventana permite:

- Modificar el estilo seleccionado. Si el estilo no es un estilo local, esta modificación se aplicará a todos los controles que usen este estilo.

- Crea un nuevo estilo.

- Duplicar estilo seleccionado.

- Eliminar el estilo seleccionado: en este caso, el estilo associated con el control será el mismo pero será local al control.




**Observación**: La casilla de verificación "Local" devuelve la lista de estilos asociados a la hoja de estilo asociada a la página (pestaña "Estilo" de la página description).

## Definiendo los estilos CSS de Default
<a name="definiendo_los_estilos_css_default_ELTTEXTE000189"></a>
Los estilos CSS de Default son los estilos CSS que utilizará Default al crear un nuevo control.

Para un tipo determinado de control, si no se encuentra un estilo de WEBDEV en el proyecto, tienes la posibilidad de definir uno o más estilos CSS de Default (dependiendo del tipo de control).

**To definir los estilos CSS de Default de un proyecto**:

1. En la pestaña "Página", en el grupo "Tema y estilos", despliegue "Estilos" y seleccione "Definir estilos CSS predeterminados". Se muestra la lista de estilos CSS encontrados en el proyecto, así como la lista de controles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Styles_CSS_Defaut.gif&type=thumb)


2. Seleccione un estilo CSS. Dependiendo del estilo seleccionado:

	- En la lista de controles, los tipos de controles con fondo rojo son aquellos para los que otro estilo CSS fue definido por Default.

	- En la lista de controles, los tipos de controles con fondo azul son aquellos para los que el estilo CSS seleccionado fue definido por Default.

	- En la lista de controles, las líneas associated con el estilo actual se muestran en verde.




3. Seleccione los controles a los que este estilo debe ser associated por Default (la lista de controles es de selección múltiple).
	**Atención**:

	- Seleccione los otros tipos que utilizan este estilo. De lo contrario, las otras asociaciones de este estilo se perderán.

	- El botón "Validar" valida la selección.




4. Haga clic en el botón [ Validar].




**Observación**: También puede modificar, crear, eliminar o duplicar un estilo de la lista.

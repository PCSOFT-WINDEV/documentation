


## Crear una consulta de inserción 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000164"></a>
<a name="Insert_query"></a>
Una **consulta de inserción** (instrucción SQL [INSERTAR](../Editeurs/2034001.md)) permite agregar registros a un archivo de una base de datos.

La siguiente información es necesaria para realizar una consulta de inserción:

- el archivo de datos en el que se insertarán los datos.

- los elementos en los que se insertarán los nuevos valores.




Por ejemplo, la creación de un nuevo cliente en el archivo de datos "Cliente": <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Requete_RequeteInsertion.gif)
 El editor de consultas genera automáticamente el código SQL de esta consulta. 

**Para ver el código SQL de una consulta**: 

1. Abrir el menú contextual del gráfico de consulta.

2. Seleccione "Código SQL".




También tiene la posibilidad de presionar la tecla F2. 

En nuestro ejemplo:


```sql
INSERT INTO CUSTOMER
	( 
		Title,	
		CustomerName,	
		Company,	
		Address1,	
		PostalCode,	
		City
	)
VALUES
	( 
		'Miss',	
		'Montgomery',	
		'UNISOFT',	
		'Accacia Avenue',	
		'EC4',	
		'London'
	 )
```

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000206"></a>


### Crear una consulta de inserción
<a name="crear_una_consulta_insercion_ELTPARAGRAPHE000047"></a>

Para crear una nueva consulta de inserción:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- La ventana de creación de nuevos elementos se abre: haga clic en "Consulta".

	- El asistente de creación de consultas se abre.




2. Especifique que desea crear una consulta de inserción (opción "Insertar").<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ass_creation_req_insertion%20-%20HC%20N%B0001.gif&type=thumb)


3. Si su proyecto es relacionado a ningún análisis, especifique el análisis al que se adjuntará la consulta.
	Pase a la etapa siguiente del asistente.

4. Seleccione el archivo de datos en el que se insertarán los datos.
	Pase a la etapa siguiente.

5. Los nuevos valores se insertarán en los diferentes elementos del archivo de datos. Para cada elemento pertinente: 

	- Haga clic en la columna "Valor/Parámetro".. 

	- En la ventana emergente que aparece, especifique el valor o el parámetro que contiene el valor a insertar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ass_creation_req_insertion%20-%20HC%20N%B0005.gif&type=thumb)
 Los nuevos valores a insertar pueden ser:

		- el valor de Default definido en el análisis.

		- un valor que debe especificarse. Especifique este valor.

		- un valor contenido en un parámetro. Indique el nombre del parámetro.

		- un valor nulo.




- Valide su elección.

6. Repita los pasos anteriores para cada elemento en el que desee insertar un valor. 
	Observación: el botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ass_creation_req_insertion%20-%20HC%20N%B0004.gif) genera automáticamente un parámetro para cada elemento sin valor.

7. Pase a la etapa siguiente.

8. Escriba el nombre de la consulta (nombre del archivo ".WDR" correspondiente a la consulta). Este nombre se utilizará para identificar la consulta en sus programas.

9. Especifique el título de la consulta describiendo brevemente el propósito de la consulta. Este epígrafe se utilizará en la documentación del proyecto..
	Observación: El nombre de la consulta se define automáticamente a partir de la leyenda escrita.

10. Validar la consulta description. La representación gráfica de la consulta se muestra automáticamente en el editor de consultas.

## Notas
<a name="notas_ELTTEXTE000230"></a>


- Para modificar las características de una consulta: en la pestaña "Consulta", en el grupo "Edición", haga clic en "Descripción". Para obtener más información, consulte [Descripción de una consulta](../Editeurs/2032038.md).

- Para generar el código SQL correspondiente a esta consulta, seleccione "Código SQL" en el menú contextual de la consulta. Si se realizan cambios en la consulta description, estos cambios se reflejarán automáticamente en el código SQL correspondiente.

- También tiene la posibilidad de crear una consulta a partir de uno o más ficheros de datos descritos en la editor de análisis: Para ello, arrastre los archivos de datos seleccionados en el panel "Análisis" y suéltelos en el editor de consultas.

- La prueba de la consulta creada se puede ejecutar desde el editor de consultas. Para obtener más información, consulte [Probar una consulta](../Editeurs/2019011.md).

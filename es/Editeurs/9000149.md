## Presentación
<a name="presentacion_ELTTEXTE000154"></a>
<a name="Presentation_Ribbon"></a>
Las diferentes opciones de los editores WINDEV, WEBDEV y WINDEV Mobile se presentan a través de una cinta. La cinta contiene pestañas en las que se agrupan las diferentes opciones de los editores.

La cinta incluye varias áreas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Environnement%20WD%20-%20HC%20N%B0003.gif)


1. El área de botones de acceso rápido.

2. El área del panel.

3. El área de grupos de opciones.

## Los botones de acceso rápido
<a name="los_botones_acceso_rapido_ELTTEXTE000178"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000026"></a>

Los botones de acceso rápido se utilizan para realizar las operaciones más habituales, comunes a todos los editores: guardar, abrir, crear, etc.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Environnement%20WD%20-%20HC%20N%B0003%201.gif)
 Los 3 botones de la parte superior de esta zona son botones específicos: 

- el logotipo de la versión se utiliza para mostrar: 

	- la ventana "Acerca de...".

	- los menús personalizados.

	- los menús desplegables de las versiones anteriores (versión 17 y anteriores).




- los 2 iconos siguientes se utilizan para restaurar los menús y las barras de herramientas de versiones anteriores (versión 17 y anteriores).





### Personalizar botones de acceso rápido
<a name="personalizar_botones_acceso_rapido_ELTPARAGRAPHE000040"></a>

Para personalizar los botones de acceso rápido: 

1. Haga clic con el botón derecho en el área de botones de acceso rápido. 

2. Seleccione "Personalizar los botones de acceso rápido". La siguiente ventana se abre: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_personnaliser.gif&type=thumb)
 En esta pantalla: 

	- La sección de la izquierda corresponde a la lista de todas las opciones disponibles en el producto. 

	- La sección derecha corresponde a los 9 botones del área de acceso rápido. 




3. Sólo 9 botones pueden estar en el área de acceso rápido. Para añadir uno o más botones, se deben eliminar algunos botones de esta área. Los botones de flecha se utilizan para manipular, quitar o reposicionar las opciones. 

4. Validar. Los cambios se aplican automáticamente.

## Pestañas
<a name="pestanas_ELTTEXTE000208"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000059"></a>

Las diferentes pestañas de la cinta de opciones permiten acceder a las opciones de los editores. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Environnement%20WD%20-%20HC%20N%B0003%202.gif)
 Hay varios tipos de fichas disponibles: 

- la pestaña actual: La pestaña aparece en gris claro y bajo el nombre de la pestaña aparece una Line de color.

- las pestañas contextuales, específicas del elemento actual: Estas pestañas están relacionadas con el editor actual o con el elemento seleccionado. Se muestra el nombre de la pestaña: 

	- en amarillo para WINDEV, 

	- en naranja para WINDEV Mobile, 

	- en azul para WEBDEV.




- las otras pestañas disponibles: El nombre de la pestaña aparece en White.





### Encontrar una opción en el menú
<a name="encontrar_una_opcion_menu_ELTPARAGRAPHE000074"></a>

¿Necesita encontrar una opción específica? Utilice la búsqueda Icon en la barra de pestañas: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_Recherche.gif)


1. Haga clic en la búsqueda Icon: una búsqueda control está habilitada. 

2. Tan pronto como se escribe una palabra, el editor propone automáticamente las posibles opciones associated con esta palabra. En el ejemplo siguiente, la palabra buscada es "cuadro de mando".: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_recherche_option.gif)


3. Seleccione una de las respuestas propuestas y se le posicionará inmediatamente en las opciones del menú. Estas opciones están bordeadas por un borde amarillo sobre un amarillo claro Background.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Ruban_selection_option.gif)

## Las opciones
<a name="las_opciones_ELTTEXTE000238"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000090"></a>

Las opciones se agrupan en las pestañas.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Environnement%20WD%20-%20HC%20N%B0003%203.gif)
 Las opciones cambian en función de la pestaña seleccionada. 

Existen varios tipos de opciones: 

- Opciones a seleccionar.

- Botones para hacer clic.

- Botones desplegables que muestran otras opciones. Existen dos tipos de botones desplegables: 

	- botones desplegables para desplegar un menú.

	- botones divididos que permiten desplegar un menú (haciendo clic en la flecha), o realizar una acción predeterminada (haciendo clic en el ícono del botón).







Las opciones están organizadas por grupos. Cada grupo de opciones tiene un nombre y un iniciador de cuadro de diálogo: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_regroup.gif). Este botón permite realizar una acción específica según el grupo actual: abrir la description del elemento actual, abrir la ayuda, ...

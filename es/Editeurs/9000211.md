
## Panel de dispositivos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000125"></a>
WINDEV Mobile permite crear aplicaciones que se adaptan a todos los dispositivos utilizando anclajes en los controles.

Para casos concretos, puede ser necesario crear un layout para un teléfono o tableta determinados.

¿Cómo determinar si una ventana se mostrará como se espera en un dispositivo específico?

¿Cómo determinar si el comportamiento de los anclajes actuales es el esperado tanto en modo vertical como horizontal en el mismo dispositivo?

Para resolver esta y muchas otras dudas, puede utilizar el Panel de dispositivos de WINDEV Mobile.

Este panel permite simular el comportamiento de las ventanas (anclajes y layouts) en un conjunto determinado de dispositivos.

## Abrir y utilizar el panel de dispositivos
<a name="abrir_utilizar_panel_dispositivos_ELTTEXTE000149"></a>


### Abrir el panel de dispositivos
<a name="abrir_panel_dispositivos_ELTPARAGRAPHE000026"></a>

Para abrir el panel de dispositivos: 

1. Abra una ventana en el editor. 

2. En la pestaña "Ventana", en el grupo "Vista previa", haga clic en "Panel de dispositivos". 

3. El panel de dispositivos se abre y muestra la ventana actual en diferentes miniaturas. Cada miniatura representa la ventana actual en diferentes dispositivos y en diferentes configuraciones del proyecto. Si la ventana contiene layouts, los diferentes layouts se muestran en el panel de dispositivos.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Panel_des_appareils%20-%20HC%20N%B0001.gif)





Observación: Puede mantener abierto el "Panel de dispositivos": este se actualiza en tiempo real con los cambios realizados en el editor. Los cambios o las nuevas ventanas abiertas se muestran en el editor.  


### Menú de la ventana
<a name="menu_ventana_ELTPARAGRAPHE000046"></a>

En el menú del panel de visualización (que se muestra al hacer clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Panel_des_appareils%20-%20HC%20N%B0001%201.gif)), puede: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Panel_des_appareils%20-%20HC%20N%B0005.gif)


- Agregar un dispositivo. La lista de dispositivos disponibles se abre. Solo tiene que seleccionar el dispositivo deseado. 

- Actualizar las miniaturas. 

- Definir las opciones de la actualización automática de miniaturas. 

- Definir el modo de visualización de los layouts. Es posible: 

	- utilizar el layout que se mostrará en tiempo de ejecución. 

	- utilizar el layout que se muestra en el editor. 




- Definir el tamaño de las miniaturas. 





### Menú contextual de las miniaturas
<a name="menu_contextual_las_miniaturas_ELTPARAGRAPHE000063"></a>

Puede ampliar las miniaturas con un solo clic. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Panel_des_appareils%20-%20HC%20N%B0002.gif)
Esto permite ver los detalles de la ventana que se muestra en un dispositivo específico. 

El menú contextual de cada miniatura permite: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Panel_des_appareils%20-%20HC%20N%B0004.gif)


- Cambiar entre el modo vertical y horizontal. 

- Agregar un dispositivo en modo vertical u horizontal. 

- Agregar un dispositivo. La lista de dispositivos disponibles se abre. Solo tiene que seleccionar el dispositivo deseado. 

- Actualizar las miniaturas. 

- Probar la ventana con el dispositivo seleccionado. Cuando se ejecuta una prueba (mediante la opción "GO"), aparece la ventana con el cuerpo del dispositivo y la resolución de pantalla. 

- Crear el layout que corresponde a la miniatura. 

- Eliminar la miniatura.

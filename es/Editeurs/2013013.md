<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000078"></a>
Los diferentes editores le permiten ver directamente el código de un elemento pulsando la tecla F2.

**Por ejemplo**, pulsando la tecla F2 en una ventana control se utiliza para iniciar el editor de códigos para mostrar el código de este botón.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Code_Histo_Navigation.gif)


La tecla F2 también le permite moverse dentro de los diferentes procedimientos anidados en el editor de código.

Para volver al elemento anterior, simplemente presione Ctrl + F2

**Por ejemplo**, pulsando la tecla F2 en una ventana control se utiliza para iniciar el editor de código para mostrar el código de este botón. Presionando Ctrl + F2 se cierra la ventana de código y se vuelve al botón inicial.

También se puede realizar una navegación completa: 

- usando los botones Siguiente y Anterior que se encuentran en la barra de herramientas de este código. 

- vía Alt + Flecha izquierda y Alt + Flecha derecha. Alt + Inicio y Alt + Posición final al principio y al final de la historia. 




Estas opciones de navegación en el código también tienen en cuenta el posicionamiento del cursor en los diferentes procesos.

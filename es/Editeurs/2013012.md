


## Incluyendo el código
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
Desea incluir sus líneas de código en una condición, en una estructura, ....

Nada más fácil, todo lo que tiene que hacer es seleccionar "Incluir en" en el editor de código.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000113"></a>
Para insertar varias líneas de código existentes en una condición, un bucle o una Variable:

1. En el editor de códigos, seleccione las líneas de código deseadas.

2. En la pestaña "Código", en el grupo "Edición", despliegue "Rodear con" y seleccione el tipo de código que debe enmarcar las líneas seleccionadas:

	- Condiciones tales como IF .... ENTONCES, SI ... THEN ... SI NO, CAMBIE, ....

	- Sentencias de bucle: PARA, PARA CADA UNO, BUCLE, ....

	- Tipo de variable: Estructura, compuesto Variable, Cuerda multilínea.
			**Observación**: También puede seleccionar "Rodear con" en el menú contextual del editor de código.




3. El código se incluye inmediatamente entre las declaraciones o condiciones.

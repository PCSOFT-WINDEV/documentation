


## Elementos del proyecto 
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000126"></a>
El proyecto incluye ventanas, informes, consultas, archivos de ayuda, conjuntos de procedimientos, ....

Al crear un elemento en un proyecto, este elemento se incluye automáticamente en la lista de elementos del proyecto actual.. Las dependencias de este elemento (imágenes, etc.) también se incluyen automáticamente en la lista de elementos del proyecto.

Los elementos que se encuentran en la lista de elementos del proyecto se propondrán automáticamente para ser incluidos en la biblioteca de la aplicación (si se crea la biblioteca).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000150"></a>
**Para visualizar la lista de elementos de un proyecto**, vaya a la pestaña "Proyecto", grupo "Proyecto", amplíe "Lista de elementos" y seleccione "Lista de elementos del proyecto".

Esta opción permite:

- **Añadir elementos pertenecientes a otros proyectos**: 
	Todos los tipos de archivos se pueden añadir a la lista de elementos del proyecto: imágenes, archivos de texto, etc., ...
	Observación: Los archivos correspondientes y sus dependencias formarán parte del proyecto, pero los archivos correspondientes no se moverán al directorio de su proyecto.

- **Borrar elementos de su proyecto**:
	Observación: Los archivos correspondientes y sus dependencias ya no formarán parte del proyecto, pero los archivos correspondientes no se eliminarán físicamente.

- **Regenera la lista de elementos encontrados en tu proyecto**:
	Esta opción le permite reconstruir la lista de elementos del proyecto a partir de los elementos que se encuentran en el directorio especificado.. Esta opción se utiliza cuando se exportan elementos WINDEV al directorio de un segundo proyecto, por ejemplo. Para tener en cuenta los nuevos elementos en el segundo proyecto, todo lo que tienes que hacer es regenerar la lista de elementos.

- **Reubicar los elementos**: 
	Esta opción permite modificar el directorio del fichero físico correspondiente al elemento seleccionado..

## Notas
<a name="notas_ELTTEXTE000174"></a>


### Consejos: Adición de elementos a una biblioteca
<a name="consejos_adicion_elementos_una_biblioteca_ELTPARAGRAPHE000058"></a>

Para añadir elementos a una biblioteca, le aconsejamos que añada estos elementos a la lista de elementos del proyecto. Por lo tanto, para utilizar una Image específica en una o varias ventanas de su proyecto mediante programación, esta Image debe estar incluida en la lista de elementos del proyecto (no es necesario incluirla en una ventana).
<a name="NOTE3_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Incluir elementos en el ejecutable: "Otro" en el panel "Explorador del proyecto"**
Los archivos (archivos de datos HFSQL, imágenes, sonidos, etc.) pueden incluirse directamente en el ejecutable generado. Esto le permite utilizar recursos que no deben estar Visible en el disco del usuario final. 

Para ello, simplemente incluya estos recursos en el proyecto a través de la carpeta "Otros", en el panel "Explorador del proyecto".. 
Las siguientes características se utilizan para simplificar las operaciones realizadas en la carpeta "Other: 

- El "arrastrar y soltar" de los elementos a incluir en la carpeta "Otro" se puede realizar desde el explorador de archivos de Windows.. 

- Un menú emergente en el menú "Otros" se utiliza para añadir o eliminar elementos.. 

- Una carpeta "Other" también está disponible en cada componente interno, lo que le permite incluir elementos en los componentes internos compartidos.. 

- La función utilizada para realizar una búsqueda en el proyecto también navega por la carpeta "Other" del proyecto..




## Asociar documentos al proyecto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000216"></a>
Para describir un proyecto en detalle, puede asociar documentos con él.

Estos documentos pueden corresponder a archivos de texto, imágenes, documentos escaneados, ...

La lista de documentos associated con el proyecto se muestra en el [documentación del proyecto](../Editeurs/2015001.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Asociar un Document al proyecto
<a name="asociar_document_proyecto_ELTTEXTE000240"></a>
Tienes la capacidad de asociar un Document con un proyecto:

- en el [asistente de creación de proyectos](#NOTE2_2).

- de la [cuadro de mando](#NOTE2_3).

- del panel [Explorador de proyectos](../Editeurs/2027029.md).



<a name="NOTE2_2"></a>


### Asociar una Document en la creación del proyecto asistente
<a name="asociar_una_document_creacion_del_proyecto_asistente_ELTPARAGRAPHE000043"></a>

Asociar una Document en la creación del proyecto asistente:

1. Vaya a "description - Documentos adjuntos".

2. Para añadir una Document existente, haz clic en el botón "Añadir una Document existente" y selecciona el archivo a añadir. Este archivo se puede copiar (o no) en el subdirectorio "Documentos" del proyecto actual.

3. Para añadir un Document escaneado:

	- posicionar el documento a escanear en una Scanner accesible desde el ordenador actual.

	- haga clic en "Escanear un documento...".

	- si hay varios Scanner s accesibles desde el ordenador actual, seleccione el Scanner que se utilizará.

	- especificar el nombre del Document escaneado. Por defecto, este Document se guarda en el subdirectorio "Document s" del directorio actual.




4. Los documentos associated con el proyecto están listados.

5. Para eliminar una Document de esta lista, seleccione la Document que desea eliminar y haga clic en "Retirar de la lista".




**Observación**: En este paso del asistente, puede crear un esqueleto de la carpeta para las especificaciones del proyecto ("Generar un esquema para el documento de especificaciones"). Se crea automáticamente un archivo "&lt;Nombre del proyecto&gt;.RTF" en el subdirectorio "Documentos" del proyecto actual. Este archivo contiene el esqueleto de las especificaciones del proyecto (objetivo, visión general, resumen de la tarea, ...) y puede ser modificado en el siguiente paso.
<a name="NOTE2_3"></a>


### Asociar una Document a la cuadro de mando
<a name="asociar_una_document_cuadro_mando_ELTPARAGRAPHE000075"></a>

Asociar una Document de la cuadro de mando:

1. Mostrar la cuadro de mando: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_tableaubord.gif).

2. Mostrar el widget "Documentos" si es necesario: 

	- Haga clic con el botón derecho del ratón fuera de los widgets para abrir el menú contextual de cuadro de mando. 

	- Seleccionar la opción "Modo de edición. 

	- Vuelva a abrir el menú contextual de la cuadro de mando. 

	- Seleccione "Añadir... Documentos". 

	- Vuelva a abrir el menú contextual de la cuadro de mando y seleccione la opción "Modo de edición".




3. Haga clic en el botón "+" del widget "Documentos. 

	- Para añadir una Document existente, seleccione "Seleccionar una Document existente" y luego seleccione el archivo a añadir. Este archivo se puede copiar (o no) en el subdirectorio "Documentos" del proyecto actual.

	- Para añadir un Document escaneado:

		- Coloque el documento a escanear en una Scanner accesible desde el ordenador actual.

		- Seleccione la opción "Escanear un documento".

		- Si hay varios Scanner s accesibles desde el ordenador actual, seleccione el Scanner que se utilizará.

		- Especifique el nombre de la Document escaneada. Por defecto, este Document se guarda en el subdirectorio "Document s" del directorio actual.







Para borrar una Document de la cuadro de mando: 

1. Selecciona el Document a borrar en el widget "Document s".. 

2. Abra el menú contextual de Document y seleccione "Eliminar del proyecto" o "Eliminar del disco".

## Viendo los documentos associated con el proyecto
<a name="viendo_los_documentos_associated_con_proyecto_ELTTEXTE000276"></a>
Tiene la posibilidad de ver los documentos associated con el proyecto:

- de la [cuadro de mando](../Editeurs/9500221.md).

- del panel [Explorador de proyectos](../Editeurs/2027029.md).



<a name="NOTE3_2"></a>


### Viendo una Document desde la cuadro de mando
<a name="viendo_una_document_desde_cuadro_mando_ELTPARAGRAPHE000131"></a>

Para ver una Document de la cuadro de mando:

1. Mostrar la cuadro de mando: en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_tableaubord.gif).

2. En el widget "Document s", abra el menú contextual de la Document que desea ver y seleccione "Abrir" (o haga clic en el archivo para verlo directamente).



<a name="NOTE3_3"></a>


### Viendo una Document desde el panel del "Proyecto Explorador"...
<a name="viendo_una_document_desde_panel_del_proyecto_explorador_ELTPARAGRAPHE000151"></a>

Para ver una Document desde el panel del "Proyecto Explorador":

1. Mostrar el panel "Explorador de proyectos: en la pestaña "Inicio", en el grupo "Entorno", amplíe "Paneles", seleccione "Paneles" y, a continuación, seleccione "Explorador de proyectos".

2. Haga clic en el elemento "Documentos. Se muestra la lista de documentos associated con el proyecto.

3. Seleccione el Document que desea ver y seleccione "Abrir" en el menú contextual (o haga doble clic en el archivo para verlo directamente).




## Creación de un sitio WEBDEV: Creación de la biblioteca
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
Una biblioteca (archivo con extensión WDL) se utiliza para agrupar los recursos del proyecto (páginas, consultas,...) en una biblioteca..
WEBDEV propone dos métodos para crear una biblioteca:

- Método 1: Si hay una configuración de proyecto "Biblioteca" en el proyecto, simplemente seleccione esta configuración de proyecto y genérela: en la pestaña "Proyecto", en el grupo "Generación", despliegue "Generar configuración" y seleccione "Generar configuración actual".

- Método 2: Generación de la biblioteca automatic cuando se despliega un sitio WEBDEV dinámico (caso más común).

## Creación de una biblioteca al implementar un sitio dinámico
<a name="creacion_una_biblioteca_implementar_sitio_dinamico_ELTTEXTE000147"></a>


### La creación de la biblioteca cuando se crea la instalación
<a name="creacion_biblioteca_cuando_crea_instalacion_ELTPARAGRAPHE000035"></a>

1. En la pestaña "Proyecto", en el grupo "Generación", despliegue "Implementar sitio" y seleccione "Generar e implementar sitio".

2. En la asistente, seleccione los elementos a incluir en la biblioteca. La biblioteca y el proyecto actual tendrán el mismo nombre. Es posible:

	- **Añadir elementos** (botón"Añadir"): 
			Se puede añadir cualquier tipo de archivo a la lista de elementos insertados en la biblioteca.: imágenes, archivos de texto, etc.

	- **Quitar elementos** (Botón"Quitar"): 
			Los archivos correspondientes y sus dependencias serán ignorados en la biblioteca.

	- **Crear la biblioteca a partir de una biblioteca existente description** (botón "WDU"): 
			Al crear una biblioteca WEBDEV, se crea automáticamente un archivo ".WDU" con el mismo nombre. Este archivo contiene todas las referencias de los elementos incluidos en la biblioteca. Todo lo que tiene que hacer es seleccionar el archivo WDU que contiene los elementos a tener en cuenta.




3. Si su proyecto utiliza componentes, el siguiente paso consiste en incluir estos componentes en la biblioteca. Por lo tanto, los archivos de los componentes no son directamente Visible en los archivos instalados.

4. En el siguiente paso, especifique los idiomas que se incluirán en la biblioteca. Esta opción se utiliza para reducir el tamaño de la biblioteca, por ejemplo. Sólo el lenguaje actual es propuesto por Default. Usted tiene la capacidad de seleccionar:

	- Uno de los idiomas utilizados por el proyecto.

	- Todos los idiomas del proyecto.

	- El idioma actual en el editor.




5. En el paso siguiente, especifique las características de la biblioteca. Esta información es la siguiente:

	- el nombre de la empresa. Por ejemplo, el nombre del editor.

	- una breve description del propósito de la aplicación. Por ejemplo: "Gestión de inventario".

	- el número de versión del sitio. Este número se utiliza para identificar las diferentes versiones de las bibliotecas creadas para el mismo sitio.. Este número de versión puede incrementarse cada vez que se vuelva a crear la biblioteca ("Incrementar automáticamente la versión en cada creación").




6. La biblioteca se genera automáticamente.

7. Siga los diferentes pasos de [la creación de instalación asistente](../Editeurs/2028039.md).

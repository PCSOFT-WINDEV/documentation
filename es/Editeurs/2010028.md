<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
Para editar los estilos disponibles en el sistema de ayuda actual: en la pestaña "Modificación", en el grupo "Estilos", haga clic en "Editar estilos". Se muestra la lista de estilos disponibles.

Al crear o modificar un estilo, se muestra la ventana de propiedades de estilo.. Esta ventana incluye 2 pestañas:

- Fuente description,

- Párrafo description.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Fuente description
<a name="fuente_description_ELTTEXTE000128"></a>
La ventana de la fuente description es la siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Style_Paragraphe.gif)


Esta ventana le permite seleccionar:

- el nombre de la fuente utilizada, 

- el tamaño de la fuente, 

- la color de la fuente (se puede elegir una color personalizada), 

- el estilo de la fuente.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Apartado description
<a name="apartado_description_ELTTEXTE000152"></a>
El párrafo de la ventana de description es el siguiente: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Style_Police.gif)
 
Esta ventana le permite seleccionar:

- los márgenes izquierdo y derecho del párrafo, 

- una viñeta al principio del párrafo,

- el espacio entre las líneas de párrafo.

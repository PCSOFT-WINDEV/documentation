## Compilación de proyectos
<a name="compilacion_proyectos_ELTTEXTE000121"></a>
Al desarrollar un proyecto, el proyecto se recopila de forma regular. Esta compilación se utiliza para comprobar el código introducido en el editor de código. Si se producen errores, los posibles problemas y sus soluciones se muestran en el panel "Errores de compilación".

Se pueden dar varios tipos de información:

- Errores de código (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_erreur.GIF)).

- Advertencias (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_warning.GIF)).

- Información (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_info.GIF)).

- Errores de UI (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Erreur_IHM.gif)).




**Observación**: En un ordenador equipado con varios procesadores o con varios "núcleos" (dual core, ....), la compilación de proyectos se realiza en paralelo y utiliza toda la potencia disponible.. El paso de compilación es acelerado por 30%.

## Los diferentes tipos de posibles errores
<a name="los_diferentes_tipos_posibles_errores_ELTTEXTE000145"></a>


### Errores
<a name="errores_ELTPARAGRAPHE000035"></a>

Los errores (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_erreur.GIF)) señalan un problema mayor (error de programación, ...). La prueba o ejecución del proyecto no funcionará.

Los errores son siempre Visible en el editor de código y deben ser corregidos para que el programa funcione.
<a name="NOTE2_2"></a>


### Advertencias
<a name="advertencias_ELTPARAGRAPHE000046"></a>

Las advertencias (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_warning.GIF)) señalan casos de programación que pueden causar comportamientos inesperados.

Las advertencias son mostradas por Default en el editor de código pero no impiden que el programa se ejecute.

Las advertencias pueden convertirse a veces en verdaderos problemas de programación
<a name="NOTE2_3"></a>


### Información
<a name="informacion_ELTPARAGRAPHE000059"></a>

La información (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_err_info.GIF)) explica las opciones del compilador o presenta consejos para mejorar el código.

La información puede ser mostrada para ayudarle a entender por qué un código tiene un comportamiento inesperado.
<a name="NOTE2_4"></a>


### Errores de UI
<a name="errores_ELTPARAGRAPHE000070"></a>

Los errores de compilación de UI (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Erreur_IHM.gif)) le informan de los posibles problemas detectados en sus interfaces: imágenes que no existen, texto truncado, etc.

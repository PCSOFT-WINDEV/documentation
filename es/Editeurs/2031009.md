


## Tipo de ventanas/páginas creadas por RID
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
El RID (Rapid Interface Design) permite crear ventanas o páginas a partir de:

- el análisis relacionado con el proyecto.

- el patrón RID seleccionado.

- plantillas.




Las ventanas/páginas generadas sólo contienen los controles relacionado para los ítems de análisis. El código completo requerido para que estas ventanas/páginas funcionen debe ser escrito por el desarrollador.. Puede introducir directamente su código personalizado.

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000144"></a>


### Creación de una ventana/página de RID
<a name="creacion_una_ventanapagina_rid_ELTPARAGRAPHE000024"></a>

Para crear una ventana/página de RID:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_nouveau.gif) en los botones de acceso rápido. 

	- Aparece la ventana de creación: haga clic en "Ventana" (o "Página") y luego en "Ventana" (o "Página").

	- El asistente se abre.




2. Visualizar la ficha "R.I.D.".

3. Seleccione el patrón RID que desea utilizar.

4. Seleccione el tipo de ventana/página que desea crear.



<a name="NOTE2_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Los diferentes tipos de ventanas del RID
<a name="los_diferentes_tipos_ventanas_del_rid_ELTPARAGRAPHE000043"></a>

Los diferentes tipos de ventanas propuestos por el RID son los siguientes:

- Ventana de formulario simple.

- Ventana simple de tabla.

- Tabla con ventana de formulario detallada.




Para obtener más información, consulte [Tipos de ventanas RAD](../Editeurs/2031003.md).




## Asociación en un MDL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000147"></a>
En un **MDL** (Conceptual Data Model), las asociaciones se utilizan para enlazar las entidades entre sí.

El editor de análisis le permite crear fácilmente asociaciones.

Cuando [generar el MDL a partir del MDL](../Editeurs/2011020.md), las asociaciones se convertirán automáticamente en [archivos](../Editeurs/2011010.md) o [enlaces](../Editeurs/2011008.md) (según las cardinalidades especificadas).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación de una asociación
<a name="creacion_una_asociacion_ELTTEXTE000171"></a>


### Creación de una asociación
<a name="creacion_una_asociacion_ELTPARAGRAPHE000031"></a>

Crear una asociación:

1. En la pestaña "MCD", en el grupo "Creación", despliegue "Nuevo" y seleccione "Nueva asociación".

2. Especifique:

	- los **nombre de la asociación**. Cuando se genere el LDM a partir del MDL, este nombre será el nombre lógico del archivo de datos creado si esta asociación contiene al menos un Property o pertenece a un enlace complejo.

	- los **pie de foto de la asociación** que resume el propósito de la asociación.

	- los **comentario** que presentan el modo de funcionamiento de la asociación.




3. La asociación se convierte en la asociación actual. Las propiedades de esta asociación se pueden describir automáticamente.

## Operaciones realizadas en una asociación
<a name="operaciones_realizadas_una_asociacion_ELTTEXTE000195"></a>


### Duplicar una asociación
<a name="duplicar_una_asociacion_ELTPARAGRAPHE000060"></a>

Para duplicar/copy una asociación:

1. Seleccione la asociación que desea duplicar.

2. Presione Ctrl + C y luego Ctrl + V para la asociación copy.

3. En la ventana de duplicación, especifique la información solicitada y valide. La asociación duplicada se crea automáticamente en el MDL actual.



<a name="NOTE3_2"></a>


### Eliminación de una asociación
<a name="eliminacion_una_asociacion_ELTPARAGRAPHE000071"></a>

Para eliminar una asociación:

1. Seleccione la asociación que desea eliminar.

2. Abrir la el menú contextual de la asociación seleccionada (clic con el botón derecho del ratón).

3. Seleccione"Eliminar".. La asociación se borra automáticamente del MDL.. Si las relaciones apuntaban a esta asociación, estas relaciones también se borran.



<a name="NOTE3_3"></a>


### Renombrar una asociación
<a name="renombrar_una_asociacion_ELTPARAGRAPHE000082"></a>

Para cambiar el nombre de una asociación:

1. Seleccione la asociación que desea renombrar.

2. En la pestaña "MCD", en el grupo "Elemento actual", haga clic en "Descripción".

3. Modificar el nombre de la asociación y validar la asociación description. La asociación se renombra automáticamente.

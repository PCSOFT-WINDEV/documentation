<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
En algunos casos, puede aparecer un mensaje de error al abrir el análisis relacionado a un proyecto. Los mensajes de error pueden ser, por ejemplo:

- El archivo buscado es desconocido.

- Archivo no encontrado en deffic.




El análisis puede ser reparado directamente.

**Observación**: Le aconsejamos que realice una copia de seguridad del análisis antes de repararlo.

<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000121"></a>


### Reparación de un análisis
<a name="reparacion_analisis_ELTPARAGRAPHE000025"></a>

Para reparar un análisis:

1. En la pestaña "Análisis", en el grupo "Análisis", despliegue "Mantenimiento" y seleccione "Reparar análisis".

2. Seleccione el análisis a reparar. Tienes la capacidad de reparar:

	- El análisis actual.

	- Un análisis específico. Para ello, seleccione el archivo ".wda" correspondiente al análisis a reparar.




3. Seleccione el modo de reparación:

	- **Reparación rápida**: los archivos de datos correspondientes a la description del análisis se reindexan, el análisis se sincroniza con su representación gráfica y se actualiza.

	- **Reparación completa**: Reparación rápida mientras se comprueba la consistencia del análisis. Usted tiene la capacidad de elegir las acciones a realizar.




4. Valide. Se repara el análisis.

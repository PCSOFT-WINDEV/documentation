


## Generación de scripts SQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
El contenido de un análisis se puede exportar en el formato de scripts SQL para una base de datos específica (SQL Server, Oracle,...). Este script genera las restricciones de integridad, los índices, ....

Este script puede ser modificado y ejecutado para crear tablas en un servidor. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000113"></a>
Generar el script SQL correspondiente a un análisis (o a un archivo):

1. En la pestaña "Análisis", en el grupo "Análisis", despliegue "Generación" y seleccione "Generar script SQL". El asistente se abre.

2. Seleccione las tablas/archivos para los que se debe generar el script SQL.

3. Seleccione el tipo de base de datos para la que se debe generar el script SQL. Se enumeran todas las bases de datos disponibles.

4. Especifique las opciones para generar el código SQL:

	- Generar DROP TABLE (código para borrar las tablas existentes).

	- Generar los índices.

	- Generar las restricciones de integridad.




5. Se muestra el script generado. Este script puede ser guardado en un archivo específico para ser modificado y luego ejecutado.

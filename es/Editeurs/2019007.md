


## Probar un proyecto dinámico desde el administrador
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000133"></a>
La ejecución de la prueba desde el administrador de WEBDEV (WDAdmin) se utiliza para probar: 

- las características del sitio.

- las características específicas de la Web (uso de cookies, ....).




**Observación**: El administrador de WEBDEV sólo puede ser utilizado para ejecutar la prueba de sitios dinámicos (o la prueba de la parte dinámica de sitios "estáticos + dinámicos").

Ejecutar la prueba desde el administrador de WEBDEV equivale a iniciar el sitio dinámico desde un ordenador remoto.

Antes de implementar un sitio WEBDEV dinámico, le recomendamos que ejecute la prueba de este sitio al menos una vez desde el administrador de WEBDEV.

Esta página de ayuda explica cómo ejecutar la prueba de un proyecto desde el administrador de WEBDEV. La prueba de un proyecto dinámico también se puede ejecutar desde el editor WEBDEV. Para obtener más información, consulte:

- [Probar un proyecto dinámico desde el editor WEBDEV](../Editeurs/2019009.md),

- [Las diferencias entre probar un proyecto desde el editor y desde el administrador](../Editeurs/2019008.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000157"></a>


### Probar un sitio dinámico (o la prueba de la parte dinámica de un sitio) desde el administrador
<a name="probar_sitio_dinamico_prueba_parte_dinamica_sitio_desde_administrador_ELTPARAGRAPHE000036"></a>

Para ejecutar la prueba desde el administrador de WEBDEV:

1. Inicie el administrador: en la pestaña "Herramientas", en el grupo "Web utilities", haga clic en "WDAdmin".

2. En la pestaña "Avanzado" haga clic en el botón "Página de prueba".



<a name="NOTE2_2"></a>


### Detener la prueba del proyecto
<a name="detener_prueba_del_proyecto_ELTPARAGRAPHE000058"></a>

Para detener la prueba de un proyecto:

1. Visualice el administrador de WEBDEV (para ello, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_WWAdmin.gif) que se encuentra en la barra de tareas).

2. En la pestaña "Conexiones" haga clic en el botón "Desconectar".

## Notas
<a name="notas_ELTTEXTE000187"></a>


### Navegador de prueba
<a name="navegador_prueba_ELTPARAGRAPHE000072"></a>

El navegador de prueba es el navegador definido por Default en el ordenador actual. 
<a name="NOTE3_2"></a>


### Diferencias al ejecutar la prueba de un sitio desde el editor y desde el administrador
<a name="diferencias_ejecutar_prueba_sitio_desde_editor_desde_administrador_ELTPARAGRAPHE000079"></a>

Se observan varias diferencias entre estos dos métodos de prueba.

Para obtener más información, consulte: [Diferencias entre los diferentes modos de prueba de un proyecto dinámico](../Editeurs/2019008.md).

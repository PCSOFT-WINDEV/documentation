
## Propiedades de una página de ayuda
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
Las características de una ayuda Page sirven para definir:

- el título y el nombre de Page ([Pestaña "General"](#NOTE2_1)).

- las palabras clave del índice associated con la ayuda Page ([Pestaña " Índice "](#NOTE3_1)).

- el aspecto de las distintas zonas en la ayuda Page (["Pestaña "Diseño](#NOTE4_1)).




Para ver las propiedades de una ayuda Page, seleccione "Propiedades Page" en el menú contextual de una ayuda Page.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Pestaña "General"
<a name="pestana_general_ELTTEXTE000170"></a>
 
Las opciones generales de una ayuda Page sirven para definir:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PageAide_General.gif&type=thumb)


- el **nombre lógico** de la ayuda Page .

- el **título del Page**. Este título aparecerá en el sumario (si la Page aparece en el sumario). Este título se puede introducir en los diferentes idiomas soportados por el sistema de ayuda.

- el **número de ayuda** del Page: Este número sirve para identificar la ayuda Page. Esto le permite abrir la ayuda sobre un Page ([WHelp](../WDLang1/3013053.md)) específico. Para obtener más información, consulte [Abrir la ayuda](../Editeurs/2010017.md).

- la **visibilidad Page en el tabla de contenidos**: 

	- si se marca esta opción, la Page aparecerá en la posición especificada en el resumen de ayuda.

	- si esta opción no está marcada, la Page sólo aparecerá en el resumen mostrado en el editor de ayuda. En tiempo de ejecución, la leyenda Page no aparecerá en el resumen.


 Atención: esta opción sólo se tiene en cuenta para la generación de un archivo chm. 




Para simplificar la gestión de los números de ayuda en su sistema de ayuda, consulte [Ayuda identificador Page](../Editeurs/2010008.md).

Para obtener más información, consulte: [Cómo crear un sistema de ayuda multilingüe](../Editeurs/2010025.md).

## Pestaña " Índice "
<a name="pestana_indice_ELTTEXTE000194"></a>
La pestaña Índice le permite:

- asociar la ayuda actual Page a un índice existente o a una nueva palabra clave de índice (botón "Añadir").

- disociar la ayuda Page de una de las palabras clave del índice (botón "Suprimir").

- generar un índice automatic a partir del contenido Page (botón "Autoíndice ...").



<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PageAide_Index.gif&type=thumb)


Para obtener más información, consulte [Asociar las palabras clave del índice a una ayuda Page](../Editeurs/2010009.md).

## "Pestaña "Diseño
<a name="pestana_diseno_ELTTEXTE000218"></a>
La pestaña "Diseño" permite asociar la página de ayuda actual a una ventana de ayuda description de: simplemente seleccione la description deseada . Para crear una nueva description, haga clic en "Definiciones".. Para obtener más información, consulte [Definición de una ventana modelo](../Editeurs/2010006.md).

Para crear una cabecera Page, marque "Esta página contiene un encabezado" y especifique la altura en píxeles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=zoneEntete_description.gif)


Se aplicarán automáticamente las características de la cabecera especificadas en la ventana de ayuda description.

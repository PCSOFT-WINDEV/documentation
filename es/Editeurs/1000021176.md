


## Groupware Usuario y archivos a los que se accede mediante un Conector Nativo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000104"></a>
Una aplicación que incluye el Groupware Usuario se puede utilizar con una base de datos a la que se accede mediante un Conector Nativo (también llamado 'Acceso Nativo').

Para que una aplicación que utiliza el Groupware Usuario funcione con un Conector Nativo, se debe configurar el Groupware para que utilice los archivos de datos del servidor. Esta página de ayuda muestra cómo hacerlo.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000128"></a>
Para utilizar los archivos del Groupware Usuario con un Conector Nativo: 

1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Groupware Usuario". 

2. En la ventana de descripción de la configuración del proyecto actual, en la sección "Groupware Usuario: Desactivado", haga clic en "Cambiar". 

3. En la ventana que aparece, active el Groupware Usuario.

4. En la pestaña "General", seleccione: 

	- Groupware Usuario estándar. 

	- Inicio manual (mediante programación). 
			El Groupware Usuario no debe utilizar el modo "Inicio automático" en su aplicación.




5. Seleccione las diferentes opciones del Groupware Usuario y valide la ventana. 
	Para obtener más información sobre las opciones disponibles, consulte [Groupware Usuario: Opciones y ajustes](../Editeurs/1000021153.md)

6. En el código de inicialización del proyecto, utilice la función [gpwOpenConnection](../WDLang6/3041007.md) para definir la conexión a la base de datos mediante un Conector Nativo.  
	**Atención**: las contraseñas de los archivos de datos del Groupware Usuario no se tienen en cuenta.

7. Utilice la función [gpwOpen](../WDLang6/3041002.md) sin especificar la ruta de los archivos de datos. El concepto de directorio no existe en este caso.

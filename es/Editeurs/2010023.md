## Presentación
<a name="presentacion_ELTTEXTE000177"></a>
Los enlaces ayudan a los usuarios a navegar por el sistema de ayuda. Se puede establecer un vínculo:

- en una palabra o en un grupo de palabras.

- en un Image.




Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=PageAide_Lien.gif)


WINDEV le permite crear un enlace a:

- una ayuda Page del sistema de ayuda actual.

- un ancla en una ayuda Page del sistema de ayuda actual.

- una ayuda Page de otro sistema de ayuda del proyecto.

- una web Page.




Las características de los enlaces se especifican en el documento [Propiedades del enlace](../Editeurs/2010022.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000201"></a>


### Crear un enlace de texto
<a name="crear_enlace_texto_ELTPARAGRAPHE000036"></a>

Para crear un enlace:

1. Mostrar la ayuda Page que contendrá el enlace en el editor de ayuda.

2. Seleccione el grupo de palabras para el que debe crearse el enlace.

3. En la pestaña "Modificación", en el grupo "Edición", haga clic en "Enlace". También puede seleccionar "Insertar un enlace" en el menú contextual. Aparece la ventana de propiedades del enlace.

4. Definir las características del enlace. Para obtener más información, consulte [Propiedades del enlace](../Editeurs/2010022.md).

5. Valide.





<a name="NOTE2_2"></a>


### Propiedades de un enlace de texto
<a name="propiedades_enlace_texto_ELTPARAGRAPHE000070"></a>

Una vez creado, el enlace se utiliza para mostrar la ayuda Page especificada en la ventana de ayuda actual: la nueva ayuda Page sustituye a la ayuda Page que se muestra actualmente.

Estas características pueden modificarse a través de [las propiedades del enlace](../Editeurs/2010022.md). Por ejemplo, la ayuda Page puede mostrarse como una "ventana emergente".

Para abrir las propiedades del enlace:

1. Sitúe el cursor del ratón sobre el texto del enlace.

2. Haga doble clic en el texto del enlace o seleccione "Propiedades del enlace" en el menú contextual.



<a name="NOTE2_3"></a>


### Crear un enlace Image
<a name="crear_enlace_image_ELTPARAGRAPHE000087"></a>

Las imágenes insertadas en una ayuda Page pueden contener uno o varios enlaces a otros Page s. Estas imágenes se convierten en **imágenes en las que se puede hacer clic.**. Estos enlaces pueden ser creados cuando se inserta el Image en la página de ayuda o más tarde a través de las propiedades de un Image.

Para obtener más información, consulte [Insertar una Image en una ayuda Page](../Editeurs/2010026.md).
<a name="NOTE2_4"></a>


### Eliminar un enlace de texto
<a name="eliminar_enlace_texto_ELTPARAGRAPHE000099"></a>

Para eliminar un enlace:

1. Abra la ayuda Page que contiene el enlace en el editor de ayuda.

2. Sitúe el cursor del ratón sobre la palabra o sobre la Image que contiene el enlace.

3. Seleccione "Eliminar enlace" en el menú contextual. El enlace se elimina automáticamente.



<a name="NOTE2_5"></a>


### Suprimir todos los enlaces de texto de una Page
<a name="suprimir_todos_los_enlaces_texto_una_page_ELTPARAGRAPHE000110"></a>

Para borrar todos los enlaces de una Page:

1. Abra la ayuda Page en el editor de ayuda.

2. Sitúe el cursor del ratón en la pantalla Page.

3. Seleccione "Eliminar todos los enlaces" en el menú contextual.



<a name="NOTE2_6"></a>


### Suprimir uno o varios enlaces Image
<a name="suprimir_uno_varios_enlaces_image_ELTPARAGRAPHE000121"></a>

Para suprimir los enlaces definidos en una Image:

1. Seleccione el Image sobre el que se puede hacer clic.

2. Seleccione "Propiedades de Image" en el menú contextual.

3. Seleccione el enlace que desea eliminar y haga clic en el botón "Eliminar".. Para obtener más información, consulte [Propiedades de la imagen](../Editeurs/2010027.md).

## Presentación
<a name="presentacion_ELTTEXTE000123"></a>
El panel "Hashtags de código" muestra la lista de hashtags que se encuentran en el código WLanguage. 

Los hashtags permiten identificar una línea de código para indicar: 

- una acción pendiente,

- información sobre la finalidad del código,

- una corrección de errores,

- etc.




**Ejemplos**: Ejemplos de hashtags en el código: 


```wl
// #Example Hashtag example
// #TODO Finish testing the code
```


**Reglas** 
Los hashtags de código deben seguir las siguientes reglas: 

- Deben utilizarse en los comentarios. Por ejemplo: 
	```wl
	// #MyTag
	or
	/* #MyTag
	```


- Los hashtags no deben ir precedidos de ningún carácter (excepto la marca de comentario, un espacio o un tabulador). 

- Los hashtags deben empezar por una letra y solo pueden contener caracteres alfanuméricos o un guion bajo. No se pueden utilizar espacios en el nombre del hashtag.

- Los hashtags no distinguen entre mayúsculas y minúsculas. 




**Observaciones**: 

- Puede crear hashtags de código en ventanas internas, páginas internas, modelos de controles, modelos de páginas o ventanas, componentes internos, etc.

- Puede crear hashtags en código WLanguage Navegador. 





### Contenido del panel "Hashtags de código"
<a name="contenido_del_panel_hashtags_codigo_ELTPARAGRAPHE000038"></a>

El panel "Hashtags de código" muestra una lista de todos los hashtags que se encuentran en los eventos WLanguage del proyecto. De forma predeterminada, se muestra el nombre del hashtag seguido del nombre del elemento que lo contiene.

Al hacer doble clic en un hashtag se muestra el código WLanguage asociado. 

Los diferentes botones del panel "Hashtags de código" permiten: 

- navegar entre los hashtags (siguiente, anterior)

- navegar entre los elementos de un grupo (cuando hay varios hashtags asociados al mismo elemento). 

- cambiar el modo de visualización: puede elegir uno los siguientes modos de visualización: 

	- Hashtags agrupados por nombre (por ejemplo, los hashtags "#todo" de todos los elementos del proyecto). 

	- Hashtags agrupados por elemento (por ejemplo, todos los hashtags de la ventana WIN_Menu). 




- buscar los diferentes hashtags.

## Abrir el panel "hashtags de código"
<a name="abrir_panel_hashtags_codigo_ELTTEXTE000153"></a>
**Para abrir el panel "Hashtags de código"**, vaya a la pestaña "Inicio", en el grupo "Entorno", despliegue "Paneles", seleccione "Paneles", y luego seleccione "Hashtags de código".

Para obtener más información sobre los paneles, consulte [Manipular paneles acoplables](../Editeurs/2027001.md).

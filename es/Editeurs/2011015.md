## Principales operaciones que se pueden realizar en un MDL
<a name="principales_operaciones_que_pueden_realizar_mdl_ELTTEXTE000202"></a>


### Borrar un MDL
<a name="borrar_mdl_ELTPARAGRAPHE000034"></a>

Para eliminar un MDL:

1. Abrir explorador de Windows.

2. Eliminar todo el directorio CDM (&lt;CDM Name&gt;.ANA directory).



<a name="NOTE2_2"></a>


### Renombrar un MDL
<a name="renombrar_mdl_ELTPARAGRAPHE000044"></a>

Para renombrar un MDL:

1. Abra el CDM que debe ser renombrado en la editor de análisis.

2. En la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar como".

3. Introduzca el nuevo nombre del MDL y valide la modificación.. El MDL renombrado se convierte en el MDL actual del proyecto.



<a name="NOTE2_3"></a>


### Verificación de un MDL
<a name="verificacion_mdl_ELTPARAGRAPHE000063"></a>

Para comprobar un MDL:

1. En la pestaña "MCD", en el grupo "Análisis", haga clic en "Verificar MCD".

2. Aparece una ventana con una lista de los errores y/o el aviso.. Estos errores/advertencias son desencadenados por:

	- las inconsistencias encontradas en el MDL.

	- el estándar especificado por el usuario en la pestaña "Detalles" de la descripción del MCD ("Información del modelo" del menú contextual del editor de análisis).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=OptionsMCD.gif&type=thumb)





3. Para imprimir esta lista, haga clic en el botón "Imprimir".




**Observación**: Cuando [generar el MDL a partir del MDL](../Editeurs/2011020.md), el MDL se comprueba automáticamente. Si se produce un problema, se muestra una ventana con una lista de los errores y/o advertencias.

- Si sólo se muestran advertencias, la generación de LDM a partir de CDM puede continuar haciendo clic en el botón "Cerrar".

- Si se muestra al menos un error, se detiene la generación de LDM a partir de CDM.. El botón "Cerrar" sirve para volver al editor y corregir el MDL.



<a name="NOTE2_4"></a>


### Modificar la visualización de un MDL
<a name="modificar_visualizacion_mdl_ELTPARAGRAPHE000098"></a>

Se puede especificar un factor de zoom para reducir o ampliar la visualización del MDL actual.

Para especificar un factor de zoom:

- **método 1**:

	- En la pestaña "Vista", en el grupo "Ayuda edición", seleccione el factor de zoom que desea utilizar.

	- Introduzca el factor de zoom en la barra de estado del editor (abajo a la derecha).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ZoomBarreMessage.gif)





- **método 2**:

	- Presione la tecla Ctrl.

	- Modifique el factor de zoom con la rueda del ratón mientras mantiene pulsada la tecla Ctrl.






<a name="NOTE2_5"></a>


### Mover la pantalla de un MDL
<a name="mover_pantalla_mdl_ELTPARAGRAPHE000125"></a>

Para mover la pantalla de un MDL:

1. Abrir el MDL en la editor de análisis.

2. Presione la tecla Alt.

3. Modifique la posición del MDL manteniendo presionado el botón izquierdo del ratón y la tecla Alt.

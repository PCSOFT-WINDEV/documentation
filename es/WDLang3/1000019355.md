
## WiFiConnectionInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la información solicitada sobre la conexión Wi-Fi actual del dispositivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Info(StringBuild("Your current connection speed is %1 Mbps", ...
	WiFiConnectionInfo(wifiSpeed)))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WiFiConnectionInfo([<Type of information>])`
---

**`<Result>: (Cadena de caracteres)`**



- Información solicitada. Si se solicitan varios tipos de información, cada tipo de información está separado por un carácter CR (Carriage Return). 
	El orden en el que se devuelve la información (si se solicita) es el siguiente: *wifiSSID* + *wifiBSSID* + *wifiIPAddress* + *wifiMacAddress* + *wifiSpeed* + *wifiIDNetwork*

- Vaciar la cadena (""") si el dispositivo no está conectado a un punto de acceso Wi-Fi.. La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Type of information>: (Constante opcional de tipo Integer)`**

Información solicitada: 


|   |   |
| --- | --- |
| *wifiAll*<br>(Valor predeterminado) | Información completa sobre la conexión (corresponde a *wifiSSID* + *wifiBSSID* + *wifiIPAddress* + *wifiMacAddress* + *wifiSpeed* + *wifiIDNetwork*).<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El valor devuelto para las constantes *wifiIPAddress* y *wifiMacAddress* corresponde a una cadena vacía. |
| *wifiBSSID* | BSSID (Basic Service set Identifier) del punto de acceso de la conexión. |
| *wifiIDNetwork* | <br><br>- Identificador de la red a la que está conectado el dispositivo,<br><br>- Vacíe la cadena ("") si el dispositivo está conectado a una red no identificada.<br><br><br> |
| *wifiIPAddress* | IP Address associated con la conexión.<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Constante no disponible |
| *wifiMacAddress* | Wi-Fi Mac Address del dispositivo.<br><br>Atención: Desde la versión 6 del SDK de Android, el Mac que ha devuelto Address tiene el siguiente formato "02:00:00:00:00".<br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Constante no disponible |
| *wifiSpeed* | Velocidad del enlace Wi-Fi (expresada en Mbps). |
| *wifiSSID* | SSID (Service set Identifier) de la red Wi-Fi a la que está conectado el dispositivo. |


## Constantes de Permission
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para gestionar los permisos:

| Constante | Detalles | Función |
| --- | --- | --- |
| **permCamera** | Permission para acceder a la(s) cámara(s) del dispositivo. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permWriteContact** | Permission para modificar los contactos. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permWriteExternalStorage** | Permission para escribir en la memoria externa. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permRecordAudio** | Permiso para los flujos de audio de Record. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permSendSMS** | Permission para el envío de SMS. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permManageExternalStorage** | Permission para gestionar el almacenamiento externo. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permReadContact** | Permission para leer los contactos. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permReadPhoneState** | Permission para acceder a la información de Phone. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permLocation** | Permission para acceder a la localización del dispositivo. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permBackgroundLocation** | Permission para acceder a la ubicación del dispositivo cuando la aplicación se ejecuta en el Background. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |
| **permFineLocation** | Permission para acceder a la ubicación precisa del dispositivo. | [PermissionRequest](../WDLang3/1410087423.md), <br>[PermissionList](../WDLang3/1410087429.md) |

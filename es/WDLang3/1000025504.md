
## MapCircle (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **MapCircle** se utiliza para definir las características de un círculo que se dibujará en el mapa que se muestra en una control Mapa. Las características de este círculo pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a circle to the Map control
Circle is MapCircle
Circle.Center = position
Circle.Radius = 50
Circle.LineColor = LightRed
Circle.ActionClick = ClickProc
MapAddShape(MAP_MyMap, Circle)

INTERNAL PROCEDURE ClickProc(c is MapCircle)
	ToastDisplay(StringBuild("Click on the <%1> circle.", c.Name))
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables MapCircle
<a name="propiedades_especificas_las_variables_mapcircle_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un círculo:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionClick | Nombre de Procedure | Nombre de la WLanguage Procedure que se llama cuando el usuario hace clic en el círculo que aparece en la control Mapa. Este procedimiento tiene el siguiente formato: <br><br><pre><code>PROCEDURE <Procedure name>(<Circle>)</code></pre><br>donde &lt;Círculo&gt; es una **MapCircle** Variable que corresponde al círculo en el que el usuario ha pulsado. <br><br>Si no se especifica esta Property (o corresponde a una cadena vacía ("")), no se realizará ninguna acción al hacer clic.<br><br>Observación: se recomienda utilizar procedimientos globales. Si la Procedure es local, su nombre debe ir precedido del nombre completo del objeto al que pertenece la Procedure (ejemplo para una ventana local Procedure: "WIN_Window1.MyProcedure").<br><br> |
| BackgroundColor | [Color](../WDLang1/1000019661.md), entero, Constante entera | Color de fondo del círculo. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br><br><br>Observación: Para cambiar la opacidad del relleno, puedes escribir:<br><br><pre><code>MyMapCircle.BackgroundColor.Opacity = 50</code></pre><br> |
| Centro | [geoPosition](../WDLang3/1000019191.md) | Centro del círculo. |
| denominar | Cadena de caracteres | Nombre de la forma (aquí, nombre del círculo). Se asigna un nombre automáticamente cuando se crea la Variable. Puede ser modificado, pero tiene que ser único si se añade la forma a la control Mapa. Este nombre puede utilizarse para borrar el círculo con [MapDeleteShape](../WDLang3/1000025510.md). |
| description | Cadena de caracteres | description de la forma, utilizada para almacenar información adicional, por ejemplo. |
| Espesor | Integro | El grosor de la Line utilizado para el círculo fuera de la Line. <br>El grosor es de set a 2 por Default. |
| Line | Constante de tipo Integer | El estilo de Line usado para el círculo fuera deLine: <br><br>- *LineSolid* (valor Default): Línea continua. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitcontinu.gif)<br><br>- *LineDotted*: lunares. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitpointille.gif)<br><br>- *LineDash*: Guiones. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitTiret.gif)<br><br>- *LineDotAndDash*: Puntos y guiones. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitmixte.gif)<br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) No disponible. |
| LineColor | [Color](../WDLang1/1000019661.md), entero, Constante entera | Círculo de la Color. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br> |
| Radio | Real | Radio del círculo en metros. El radio de Default es de 100 metros. |
| ZOrder | Integro | El orden Z del círculo en relación con las otras formas de la control Mapa. Le permite manejar las superposiciones. <br><br>Una forma con un orden Z alto se dibujará sobre una forma con un orden Z más bajo. <br><br>Este Property es set a 0 por Default. <br><br> |

| [MapModifyShape](../WDLang3/1000025511.md) | Modifica una forma dibujada en el mapa que se muestra en una control Mapa. |


## Constantes iCloud
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las constantes iCloud son las siguientes:

| Constante | Detalles | Función |
| --- | --- | --- |
| **ickvAccountChange** | Causa del evento: El usuario utilizó una cuenta diferente. | [iCloudKeyValueWatch](../WDLang3/1410088760.md) |
| **ickvSynchronizationFailed** | Causa del evento: Error al intentar sincronizar el dispositivo con el servidor iCloud de Apple. | [iCloudKeyValueWatch](../WDLang3/1410088760.md) |
| **ickvServerChange** | Causa del evento: Se modificaron las claves desde el servidor iCloud de Apple. | [iCloudKeyValueWatch](../WDLang3/1410088760.md) |
| **ickvInitialSynchronizationChange** | Causa del evento: Sincronización inicial con el servidor iCloud de Apple. | [iCloudKeyValueWatch](../WDLang3/1410088760.md) |
| **ickvQuotaViolation** | Causa del evento: Se alcanzó la cuota máxima (número de claves, espacio de almacenamiento, etc.). | [iCloudKeyValueWatch](../WDLang3/1410088760.md) |

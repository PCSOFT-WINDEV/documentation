


## ContactFirst( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la primera Contact en un libro de Lotus Notes o Outlook Address.
**Atención**: Esta función **no se puede utilizar** para leer el primer contacto encontrado en la libreta Address de una mensajería Outlook Express.

Al leer un Contact, los datos de este Contact se actualizan en el [Estructura de Contact](../WDLang3/1000006111.md). Para recuperar los datos de la Contact actual, basta con recuperar los valores de la [Estructura de Contact](../WDLang3/1000006111.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartSession("Password", "MarsServer", "email/julia.nsf", ...
		"C:\Lotus Notes\julia.id")
// Read the first contact
ContactFirst(SessionID)
// Add the contacts found into a List Box control
WHILE mContact.Out = False
	ListAdd(LIST_ContactList, mContact.Name + TAB + mContact.FirstName)
	ContactNext(SessionID)
END
```


<a name="Example2"></a>

```wl
// Start an Outlook session
SessionID is int
SessionID = OutlookStartSession("Profile")
// Read the first contact
ContactFirst(SessionID)
// Add the contacts found into a List Box control
WHILE mContact.Out = False
	ListAdd(LIST_ContactList, mContact.Name + TAB + mContact.FirstName)
	ContactNext(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ContactFirst(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó el primer contacto,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

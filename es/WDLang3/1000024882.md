


## grComparativeSeriesPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija la posición (izquierda o derecha) de una serie en un gráfico de tipo Histograma de comparación. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grComparativeSeriesPosition(CHART_NoName1, 1, grAtRight)
grComparativeSeriesPosition(CHART_NoName1, 2, grAtLeft)

grSeriesColor(CHART_NoName1, 1, GetColor())
grSeriesColor(CHART_NoName1, 2, GetColor())

rRight is real
rLeft is real

nRadius is int = 250

FOR n = 1 TO 360
	rRight = nRadius + nRadius * Cos(n)
	rLeft = nRadius + nRadius * Sin(n + 180)
	grAddData(CHART_NoName1, 1, n, rRight)
	grAddData(CHART_NoName1, 2, n, rLeft)
	grCategoryLabel(CHART_NoName, 1, n, n)
END

grDraw(CHART_NoName1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grComparativeSeriesPosition(<Chart name> , <Series number> , <Position>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico del editor de ventanas.




**`<Series number>: (Entero)`**

Número de la serie a modificar. 

**`<Position>: (Entero)`**

Posición de la serie manipulada en el gráfico:


|   |   |
| --- | --- |
| *grAtLeft* | Serie mostrada a la izquierda del gráfico. |
| *grAtRight* | Serie mostrada a la derecha del gráfico. |

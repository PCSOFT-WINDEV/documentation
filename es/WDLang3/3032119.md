
## LDAPReplaceAttributeValue( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reemplaza un valor específico de un atributo por un nuevo valor.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies a specific value of an attribute 
// sValue2 must be replaced with sNewValue
LDAPStartModify("SessionName")
LDAPReplaceAttributeValue("SessionName", "AttributeName", sValue2, sNewValue)
LDAPValidateModify("SessionName", "DN")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPReplaceAttributeValue(<LDAP session> , <Attribute name> , <Former attribute value> , <New attribute value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error. La modificación se realizará durante la llamada a [LDAPValidateModify](../WDLang3/3032117.md).




**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP utilizada.

**`<Attribute name>: (Cadena de caracteres)`**

Nombre del atributo a utilizar.

**`<Former attribute value>: (Cadena de caracteres ANSI)`**

Valor del atributo que se debe reemplazar.

**`<New attribute value>: (Cadena de caracteres ANSI)`**

Nuevo valor del atributo que sustituye a todos los &lt;Valores de atributo anteriores&gt;..

## Observaciones




### Diferencias entre LDAPReplaceAttribute y LDAPReplaceAttributeValue
<a name="diferencias_entre_ldapreplaceattribute_docparampagetitleshort_ELTPARAGRAPHE000053"></a>

[Atributo LDAPReplaceAttribute](../WDLang3/3032113.md) se utiliza para sustituir todos los valores de un atributo por un único valor. Un atributo de varios valores (varios valores para este atributo) se convertirá en un atributo de un solo valor (un solo valor para este atributo).

**LDAPReplaceAttributeValue** sustituye el valor específico de un atributo por un nuevo valor. Un atributo de varios valores sigue siendo un atributo de varios valores.
<a name="NOTE0_2"></a>




### Modificación de un objeto existente en un servidor LDAP
<a name="modificacion_objeto_existente_servidor_ldap_ELTPARAGRAPHE000068"></a>

Para modificar un objeto existente en un servidor LDAP, debe definir las diferentes modificaciones a realizar.

Para optimizar la velocidad de modificación de los objetos, las modificaciones se definen localmente: se establece una comunicación única con el servidor al validar la modificación. Los pasos son:

1. Modificación de inicio ([LDAPStartModificar](../WDLang3/3032112.md))

2. Operaciones de modificación:

	- Añadir atributos y sus valores [Atributo LDAPAddAttribute](../WDLang3/3032110.md)).

	- Borrar atributos ([LDAPDeleteAttribute](../WDLang3/3032115.md)) o sus valores ([LDAPDeleteAttributeValue](../WDLang3/3032116.md)).

	- Sustituir los valores de un atributo ([Atributo LDAPReplaceAttribute](../WDLang3/3032113.md) o **LDAPReplaceAttributeValue**)

	- Renombrar atributos ([LDAPRenameAttribute](../WDLang3/3032120.md))




3. Validar las modificaciones en el servidor ([LDAPValidateModify](../WDLang3/3032117.md)). Las modificaciones se realizan en el servidor según el orden utilizado en el código de WLanguage.

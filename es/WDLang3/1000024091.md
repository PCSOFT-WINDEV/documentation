


## &lt;AutoRefreshCategoryLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o define si las etiquetas de las categorías se actualizan cada vez que se dibuja el gráfico.. En este caso, el contenido de la serie se recalcula automáticamente cada vez que se llama a [&lt;Dibujo](../WDLang3/1000024062.md). Las etiquetas de las categorías se actualizan si es necesario.

**Ejemplo de uso**: una serie en un gráfico se utiliza como serie reference: esta serie no se actualiza automáticamente.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control
IF CHART_MyChart.AutoRefreshCategoryLabel() = True THEN
	CHART_MyChart.Draw()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificando el modo utilizado para actualización las etiquetas de categoría

`<Result> = <Chart control>.AutoRefreshCategoryLabel()`
---

**`<Result>: (booleano)`**

Modo actual para actualizar las etiquetas de las categorías: 

- <u><u><u><u>True STRIMEULB</u></u></u></u> si las etiquetas se actualizan automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).  



<a name="SYNTAX2"></a>

### Definiendo el modo a actualización las etiquetas de la categoría

`<Chart control>.AutoRefreshCategoryLabel(<Automatic refresh>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).  


**`<Automatic refresh>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para una actualización automática,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si las etiquetas de la categoría están en modo automatic, las etiquetas de la categoría serán automatic ally recalculadas durante la siguiente llamada a [&lt;Dibujo](../WDLang3/1000024062.md).

- Si las etiquetas de categoría no están en modo automatic, las etiquetas de categoría se recalcularán durante la siguiente llamada a [&lt;Tabla&gt;.SourceCategoryLabel](../WDLang3/1000024138.md).

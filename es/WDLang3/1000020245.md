
## SIPAddress (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **SIPAddress** se utiliza para definir las características avanzadas de un Address SIP. SIP (Session Initiation Protocol) es un protocolo estandarizado utilizado para establecer comunicaciones multimedia (sonido e imágenes).. Se utiliza principalmente para VoIP (voz sobre IP).

Las características de este SIP Address se pueden definir y modificar mediante varias propiedades de WLanguage. 

Este tipo de Variable es usado por los [Contact](../WDLang3/1000020254.md) Variable s.

**Observación**: Consulte [Declarar una variable](../Motscles/1514032.md) para obtener más detalles sobre la declaración de este tipo de variables y el uso de propiedades WLanguage.

<a name="NOTE0_1"></a>

## Observaciones


### Propiedades específicas de la description de las variables SIPAddress
<a name="propiedades_especificas_description_las_variables_sipaddress_ELTPARAGRAPHE000038"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a SIP Address:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Address | Cadena de caracteres | SORBO DE Address. |
| Etiqueta | Cadena de caracteres | Etiqueta para los tipos de direcciones SIP personalizadas (opcional)<br>Si **..Type** no corresponde a la constante *sipCustom*, el valor de este Property será ignorado. |
| escribir a máquina | Constante de tipo Integer | Tipo de SIP Address: <br><br>- *sipOther*: otras clases<br><br>- *sipWork*: profesional SIP Address<br><br>- *sipHome*: personal SIP Address<br><br>- *sipCustom*: tipo personalizado que corresponde al valor de la propiedad **..Label**.<br><br><br> |


**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;..&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;



<a name="NOTE0_2"></a>


### Reinicio
<a name="reinicio_ELTPARAGRAPHE000083"></a>

Para reiniciar el contenido de un **SIPAddress** Variable, tienes la posibilidad de usar [VariableReset](../WDLang1/3013069.md).

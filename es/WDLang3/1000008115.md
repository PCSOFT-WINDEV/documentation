
## GroupFirst( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee el primer grupo de contactos que se encuentra en el libro de Address de un Lotus Notes o de un mensaje de Outlook.

Cuando se lee un grupo de contactos, los datos que se encuentran en este grupo se actualizan en el [Estructura del grupo](../WDLang3/1000008120.md). Para recuperar los datos del grupo actual de contactos, basta con recuperar los valores de la [Estructura del grupo](../WDLang3/1000008120.md).

**Atención**: Esta función **no permite** leer el primer grupo de contactos que se encuentra en la libreta de direcciones de una mensajería Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
	"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Read the first group of contacts
GroupFirst(SessionID)
// Add the contact groups found into a List Box control
WHILE mGroup.Out = False
	ListAdd(LIST_ContactGroups, mGroup.Name + TAB + mGroup.Description)
	GroupNext(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = GroupFirst(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó el primer grupo de contactos,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Grupo actual de contactos
<a name="grupo_actual_contactos_ELTPARAGRAPHE000066"></a>

El mGroup.Out Variable se utiliza para averiguar si hay un grupo de contactos actual.
<a name="NOTE0_2"></a>




### Versión de Outlook utilizada
<a name="version_outlook_utilizada_ELTPARAGRAPHE000074"></a>

Los grupos de contactos no son compatibles con Outlook 97. Por lo tanto, no se puede encontrar ningún grupo de contactos en esta versión de Outlook.

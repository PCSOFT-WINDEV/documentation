


## MapDeleteMarker (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una Marker de un mapa mostrado por una control Mapa. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete a marker
MyMarker is Marker
MyMarker.Name = "Destination"
MyMarker.Position.Latitude = 43.613708 
MyMarker.Position.Longitude = 3.876972

IF CBOX_DisplayMyDestination = True THEN
	MapAddMarker(MAP_Position, MyMarker)
ELSE
	MapDeleteMarker(MAP_Position, MyMarker)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapDeleteMarker([<Map control> [, <Marker>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el marcador,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre de control opcional)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, se utilizará la control Mapa a la que pertenece la actual Event.

**`<Marker>: (Marcador Variable o cadena de caracteres opcional)`**

Marker para borrar. Este parámetro puede corresponder a: 

- una variable [Marker](../WDLang3/1000019940.md) correspondiente al marker que se va a suprimir. 

- una cadena de caracteres correspondiente al nombre de la Marker para borrar. El nombre corresponde a la propiedad **Name** de la variable de tipo [Marker](../WDLang3/1000019940.md).




Si no se especifica este parámetro, se borrarán todos los marcadores del mapa (equivalente a [MapDeleteAll](../WDLang3/1000019952.md)). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Sólo una Marker previamente añadida a la control Mapa puede ser eliminada. 

- Para añadir una nueva Marker a una control Mapa, utilice [MapAddMarker](../WDLang3/1000019794.md).

- Para cambiar una Marker en una control Mapa, utilice [MapModifyMarker](../WDLang3/1000019796.md).


## SaaSCheckService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba los derechos de acceso del usuario a un servicio del sitio SaaS.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SaaSConnect("login", "password") = True THEN
	IF SaaSCheckService("CRM") = False THEN
		Info("You are not allowed to use the CRM service")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Comprobación de los derechos de acceso a un servicio SaaS

`<Result> = SaaSCheckService([<SaaS service>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario puede utilizar el servicio especificado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario (o si no hay ningún usuario conectado). [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.




**`<SaaS service>: (Cadena de caracteres opcional)`**

Nombre del servicio SaaS cuyos derechos deben comprobarse. Este nombre se definió en la interfaz para la gestión de SaaS.
Si no se especifica este parámetro, la función devuelve <u><u><u><u>True</u></u></u></u> si se permite al usuario usar al menos un servicio del sitio SaaS.


<a name="SYNTAX2"></a>

### Comprobación de los derechos de acceso a una set de servicios SaaS

`<Result> = SaaSCheckService(<Names of the SaaS services>)`
---

**`<Result>: (Array)`**

Array de variables booleanas que indica si el servicio especificado está disponible para el usuario: 

- <u><u><u><u>True</u></u></u></u> si el usuario puede utilizar el servicio especificado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario (o si no hay ningún usuario conectado). [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.




**`<Names of the SaaS services>: (Array)`**

Nombre de la array que contiene los nombres de los distintos servicios SaaS cuyos derechos deben comprobarse. Estos nombres se definen en la interfaz de administración de SaaS.

## Observaciones


### Acceso restringido a los servicios de un sitio SaaS
<a name="acceso_restringido_los_servicios_sitio_saas_ELTPARAGRAPHE000071"></a>

Cuando los clientes se suscriben a un plan determinado, adquieren licencias para utilizar un servicio. A continuación, proporcionan estas licencias a los usuarios finales.
Por lo tanto, es posible que algunos usuarios no tengan acceso a un servicio porque el administrador no haya activado su licencia.

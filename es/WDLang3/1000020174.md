
## SpeechSynthesisStop( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene todas las operaciones de lectura actuales en el motor para la síntesis de voz.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SpeechSynthesisInProgress() = True THEN
	SpeechSynthesisStop()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpeechSynthesisStop()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Sólo se detienen las operaciones de lectura iniciadas por la aplicación.. Para saber si una operación de lectura está en curso en el motor de síntesis de voz, utilice [SpeechSynthesisInProgress](../WDLang3/1000020173.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Funcionalidad de aplicación requerida**
	Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.
	Característica necesaria: Micrófono: Esta función permite a las aplicaciones realizar grabaciones de audio.

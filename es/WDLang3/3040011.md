


## FTPDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la conexión entre el ordenador actual y el servidor FTP (File Transfer Protocol). Esta conexión fue establecida por [FTPConnect](../WDLang3/3040016.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Connection to an FTP server
nConnection is int 
nConnection = FTPConnect("192.108.10.2")
...
// Disconnection
FTPDisconnect(nConnection)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FTPDisconnect(<Connection identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles.




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [FTPConnect](../WDLang3/3040016.md).





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tiempo de desconexión
<a name="tiempo_desconexion_ELTPARAGRAPHE000078"></a>

De forma predeterminada, todas las funciones FTP fallan si no se recibe ninguna respuesta del servidor FTP después de 20 segundos. Este tiempo de espera se puede modificar al conectarse al servidor FTP con la función [FTPConnect](../WDLang3/3040016.md).




## <Chart>.RBMaximumValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece el valor máximo en un gráfico de barras radiales. Este valor es set a 100 por Default, por lo que los datos se expresan en porcentaje. Este valor puede modificarse y los porcentajes se recalculan automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.RBMaximumValue(1212)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.RBMaximumValue(<Maximum value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (en la ventana o editor de reportes).

**`<Maximum value>: (Integro)`**

Nuevo valor máximo a aplicar. Este valor debe ser positivo.
Este valor es set a 100 por Default, por lo que los datos se expresan en porcentaje. Cuando se cambia este valor, los datos del gráfico se recalculan automáticamente.

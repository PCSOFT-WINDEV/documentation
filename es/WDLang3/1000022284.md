


## &lt;emailOutlookSession Variable&gt;.CloseSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Cierra la sesión de Outlook.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
MySession is emailOutlookSession

MySession..Name = "OutlookProfile"
MySession.StartSession()
...
MySession.CloseSession()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Outlook session>.Sesión cerrada()`
---

**`<Outlook session>: (emailOutlookSession variable)`**

Nombre de la [emailOutlookSession](../WDLang3/1000018767.md) Variable correspondiente a la sesión Email a utilizar.

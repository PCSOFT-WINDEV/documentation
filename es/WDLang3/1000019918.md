


## EmailCopy (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia un Email encontrado en un directorio a otro directorio de un servidor IMAP. 

**Observación**: Esta función no está disponible para las sesiones POP3, Outlook y Notes.. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyEmail is Email
MySession is emailIMAPSession
...
// Start the session
IF EmailStartSession(MySession) = True THEN
	// Read the first email
	IF EmailReadFirst(MySession, MyEmail) = True THEN
		// Copies the email into another folder
		EmailCopy(MySession, MyEmail, "Inbox/Read")
	END
	EmailCloseSession(MySession)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Copiar un correo electrónico encontrado en un Variable de correo electrónico

`<Result> = EmailCopy(<Email> , <Documentation>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se copió la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable que corresponde al Email a copiar. Este correo es associated con una sesión IMAP. 

**`<Documentation>: (Cadena de caracteres)`**

La carpeta de destino del Email. Esta carpeta debe existir. La lista de carpetas es devuelta por [EmailListFolder](../WDLang3/3032133.md). 


<a name="SYNTAX2"></a>

### Copiar un correo electrónico encontrado en un Variable de correo electrónico mientras se especifica la sesión IMAP

`<Result> = EmailCopy(<Session> , <Email> , <Documentation>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se copió la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Session>: (Variable de tipo emailIMAPSession)`**

Nombre de la sesión de [emailIMAPSession](../WDLang3/1000018957.md) correspondiente a la sesión a la que pertenece el Email. 

**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable que corresponde al Email a copiar. 

**`<Documentation>: (Cadena de caracteres)`**

La carpeta de destino del Email. Esta carpeta debe existir. La lista de carpetas es devuelta por [EmailListFolder](../WDLang3/3032133.md).

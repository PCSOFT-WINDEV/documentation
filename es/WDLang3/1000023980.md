
## NotifDeleteCategory( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una categoría de Notification de la solicitud.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all unimportant notification categories from the application
arrCategory is array of notificationCategory = NotifListCategory()
FOR EACH Category OF arrCategory
	IF Category.Priority = notifLowPriority THEN
		NotifDeleteCategory(Category.Identifier)
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NotifDeleteCategory(<Identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la categoría,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si el borrado falla, utilice [ErrorInfo](../WDLang1/3013008.md) para get más detalles sobre el error.




**`<Identifier>: (Cadena de caracteres)`**

Identificador de la categoría Notification para eliminar. El identificador de categoría distingue entre mayúsculas y minúsculas y acentos.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Cuando se elimina una categoría, se pierden los cambios realizados por el usuario final en la configuración de la aplicación Notification.

- Para listar las categorías existentes de la aplicación, utilice [NotifListCategory](../WDLang3/1000023979.md).

- **NotifDeleteCategory** sólo está disponible para los dispositivos que funcionan con Android 8.0 o posterior (nivel 26 de API). 
	Si se utiliza la función con una versión anterior del sistema, se produce un error fatal. 
	Para determinar la versión de Android en la que se está ejecutando la aplicación, utilice la función [SysAndroidVersion](../WDLang1/1000019785.md).

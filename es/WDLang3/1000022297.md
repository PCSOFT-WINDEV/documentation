


## &lt;emailIMAPSession Variable&gt;.NbMensaje (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de mensajes entrantes en el servidor Email (protocolo IMAP).
Este número de mensajes evoluciona en función de los Email s entrantes o de los Email s efectivamente borrados del servidor Email (a través de un comando "expunge", borrar la papelera de un WebMail por ejemplo). También tiene la posibilidad de forzar la destrucción de todos los correos electrónicos eliminados a través de [&lt;emailIMAPSession Variable&gt;.Expunge](../WDLang3/1000022279.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySession is EmailIMAPSession

nbMSG is int = MySession.NbMessage()
FOR I = 1 TO nbMSG
	EmailReadMessage(MySession, I)
	IF Email.Out <> True THEN
		CU.NAME = Email.Sender
		CU.MESSAGE_TXT = Email.Message
		...
	ELSE
		Info("This message was deleted")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <IMAP session>.NbMessage()`
---

**`<Result>: (Integro)`**



- Número de mensajes recibidos en el servidor en la sesión actual de Email,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<IMAP session>: (Variable de tipo emailIMAPSession)`**

Nombre de la [emailIMAPSession](../WDLang3/1000018957.md) Variable correspondiente a la sesión Email a utilizar.

<a name="NOTE0_1"></a>

## Observaciones


### Mensajes eliminados (protocolo IMAP)
<a name="mensajes_eliminados_protocolo_imap_ELTPARAGRAPHE000290"></a>

Si un mensaje fue borrado por [EmailDeleteMessage](../WDLang3/3032027.md), este mensaje es realmente borrado de la sesión: 

- durante el siguiente comando "borrar" (por ejemplo, borrar la papelera de un WebMail).

- al utilizar [&lt;emailIMAPSession Variable&gt;.Expunge](../WDLang3/1000022279.md) para forzar la destrucción de todos los correos electrónicos eliminados.

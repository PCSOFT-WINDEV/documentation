


## grDestinationImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define una imagen Variable como destino de una carta. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is Image
MyImage.Width = 100
MyImage.Height = 100
grCreate("MyChart", grPie)

// Defines the destination of the chart drawing
grDestinationImage("MyChart", MyImage)
// Adds the data
grAddData("MyChart", 1, 10)
...
// Draws the chart
grDraw("MyChart")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grDestinationImage(<Chart name> , <Name of Image variable>)`
---

**`<Chart name>: (Cadena de caracteres)`**

Nombre del gráfico a utilizar, definido por [grCreate](../WDLang3/3042005.md).

**`<Name of Image variable>: (Imagen Variable)`**

Nombre de la [Image](../WDLang1/1000019650.md) Variable donde se dibujará el gráfico. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso y límites
<a name="condiciones_uso_limites_ELTPARAGRAPHE000046"></a>

- El gráfico se dibujará en el [Image](../WDLang1/1000019650.md) Variable especificado. El gráfico se dibujará en un White Background. El dibujo anterior contenido en la [Image](../WDLang1/1000019650.md) Variable será borrado. 

- Esta función no se puede utilizar con los controles Gráficos.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta función no puede ser usada para get el Image de un gráfico 3D. El gráfico se convertirá automáticamente a 2D. 



<a name="NOTE0_2"></a>


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000063"></a>

Para dibujar un gráfico:

- crear el gráfico ([grCreate](../WDLang3/3042005.md)),

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- definir el destino de la carta con las siguientes funciones: 

	- [grDestinoBMP](../WDLang3/3042018.md), 

	- [grDestinoControl](../WDLang3/3042020.md), 

	- [grDestinoEMF](../WDLang3/3042017.md), 

	- [grDestinoWnd](../WDLang3/3042031.md),

	- **grDestinationImage**,

	- [grDestinoWMF](../WDLang3/3042019.md). 




- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).

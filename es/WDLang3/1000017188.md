
## HTTPResetForm( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un formulario HTTP que se está editando actualmente.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPCreateForm("FORM")
...
HTTPResetForm("FORM")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HTTPResetForm(<Form name>)`
---

**`<Form name>: (Cadena de caracteres)`**

Nombre del formulario HTTP a borrar. Este nombre se especificó al crear el formulario HTTP ([HTTPCreateForm](../WDLang3/1000017119.md)). Se produce un error si el formulario no existe.




## &lt;Mapa&gt;.GetJSObject (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la instancia API de un control Mapa. Esto le permite manipular la control Mapa en un código JavaScript directamente. 

En combinación con [MapConfigureGglAPI](../WDLang3/1000021611.md), esta función permite gestionar parámetros adicionales no propuestos por defecto por control Mapa. 

**Observación**: En WINDEV, esta función se puede utilizar en el código JavaScript ejecutado por [&lt;Mapa&gt;.ExecuteJS](../WDLang3/1000022384.md) para manipular el control Mapa de la aplicación WINDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Server code
MapConfigureGglAPI("libraries=geometry,places","language=fr")
```

```wl
// Browser code
MyMap is object dynamic
MyMap = MAP_Browser.GetJSObject()
FindAddress(MyMap)
```

```txt
// JavaScript function that uses the dynamic object
FindAddress(MyMap)
{
	var infowindow;
	// Rome: 41.873144, 12.480391
	var pyrmont = new google.maps.LatLng(41.873144, 12.480391);
	var request = {
		location: pyrmont,
		radius: 500,
		types: ['store']
		};
	infowindow = new google.maps.InfoWindow();
	var service = new google.maps.places.PlacesService(MyMap);
	service.nearbySearch(request, ReturnAddress_WB);
	
}
```

<a name="SYNTAX1"></a>

`<Map control>.GetJSObject()`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

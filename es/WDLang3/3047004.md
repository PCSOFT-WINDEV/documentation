
## J2EE constants
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por las funciones J2EE:

| Constante | description | Detalles | Utilizado por las funciones |
| --- | --- | --- | --- |
| **J2EEAnyURIType** | Tipo de parámetro | URL Address | [J2EERun](../WDLang3/3047007.md) |
| **J2EEBase64BinaryType** | Tipo de parámetro | Binario | [J2EERun](../WDLang3/3047007.md) |
| **J2EEBooleanType** | Tipo de parámetro | booleano | [J2EERun](../WDLang3/3047007.md) |
| **J2EEByteType** | Tipo de parámetro | Byte | [J2EERun](../WDLang3/3047007.md) |
| **J2EEDecimalType** | Tipo de parámetro | Cualquier número sin límite | [J2EERun](../WDLang3/3047007.md) |
| **J2EEDoubleType** | Tipo de parámetro | Doblemente real | [J2EERun](../WDLang3/3047007.md) |
| **J2EEErrActor** | Tipo de error a devolver | URL del servidor que activó la Error( útil si la Procedure es ejecutada por varios servidores). | [J2EEError](../WDLang3/3047010.md) |
| **J2EEErrCode** | Tipo de error a devolver | Error de la última corrida de J2EE Procedure:<br><br>- "VersionMismatch": La versión J2EE utilizada en el ordenador actual difiere de la utilizada en el servidor.<br><br>- "Debe ser de las afueras.": El servidor no entiende el parámetro especificado en el encabezado (solo si el parámetro &lt;Ejecución del procedimiento&gt; del encabezado se inicializa en <u><u><u><u>True</u></u></u></u>).<br><br>- "Cliente": Configuración inválida en relación con la ejecución de la Procedure.<br><br>- "Servidor": El servidor no ejecutó el Procedure.<br><br><br>**Observación**: Esta cadena de caracteres se puede rellenar con información adicional. Se utiliza el siguiente formato: "&lt;Error&gt;.&lt;Información&gt;". Por ejemplo: "Autenticación de cliente" si el error se debe a la autenticación del cliente. | [J2EEError](../WDLang3/3047010.md) |
| **J2EEErrDetails** | Tipo de error a devolver | Detalles del error. Este mensaje está en formato XML. | [J2EEError](../WDLang3/3047010.md) |
| **J2EEErrMessage** | Tipo de error a devolver | Mensaje de error devuelto por el servidor en la última ejecución de J2EE Procedure. | [J2EEError](../WDLang3/3047010.md) |
| **J2EEFloatType** | Tipo de parámetro | Real | [J2EERun](../WDLang3/3047007.md) |
| **J2EEHexBinaryType** | Tipo de parámetro | Hexadecimal binary | [J2EERun](../WDLang3/3047007.md) |
| **J2EEHTTPResult** | Tipo de resultado | Resultado (en formato HTTP) de la última ejecución de J2EE Procedure | [J2EEGetResultado](../WDLang3/3047003.md) |
| **J2EEIntType** | Tipo de parámetro | Entero limitado | [J2EERun](../WDLang3/3047007.md) |
| **J2EELongType** | Tipo de parámetro | Long integer | [J2EERun](../WDLang3/3047007.md) |
| **J2EENonNegativeIntegerType** | Tipo de parámetro | Entero no negativo | [J2EERun](../WDLang3/3047007.md) |
| **J2EENormalizedStringType** | Tipo de parámetro | Cadena de caracteres sin caracteres CR o TAB | [J2EERun](../WDLang3/3047007.md) |
| **J2EEPositiveIntegerType** | Tipo de parámetro | Entero positivo (excepto 0) | [J2EERun](../WDLang3/3047007.md) |
| **J2EEResult** | Tipo de resultado | Resultado de la última ejecución de J2EE Procedure | [J2EEGetResultado](../WDLang3/3047003.md) |
| **J2EEShortType** | Tipo de parámetro | Short integer | [J2EERun](../WDLang3/3047007.md) |
| **J2EEStringType** | Tipo de parámetro | Cadena de caracteres | [J2EERun](../WDLang3/3047007.md) |
| **J2EETypeNegativoNúmero entero** | Tipo de parámetro | Entero negativo (excepto 0) | [J2EERun](../WDLang3/3047007.md) |
| **J2EETypeNonPositiveInteger (Entero no positivo)** | Tipo de parámetro | Entero no positivo | [J2EERun](../WDLang3/3047007.md) |
| **J2EETypeNúmero entero** | Tipo de parámetro | Número entero ilimitado | [J2EERun](../WDLang3/3047007.md) |
| **J2EETypeUnsignedLong** | Tipo de parámetro | Entero largo sin signo | [J2EERun](../WDLang3/3047007.md) |
| **J2EEUnsignedByteType** | Tipo de parámetro | Byte no firmado | [J2EERun](../WDLang3/3047007.md) |
| **J2EEUnsignedIntType** | Tipo de parámetro | Entero sin signo | [J2EERun](../WDLang3/3047007.md) |
| **J2EEUnsignedShortType** | Tipo de parámetro | Entero corto sin firmar | [J2EERun](../WDLang3/3047007.md) |
| **J2EEXMLResult** | Tipo de resultado | Resultado (en formato XML) de la última ejecución de J2EE Procedure | [J2EEGetResultado](../WDLang3/3047003.md) |


## FaxInbox( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los faxes pendientes o los faxes entrantes.

**Observación**: Esta función sólo se puede utilizar en Windows 2000 o XP.. Desde Windows Vista, la gestión de faxes sólo está disponible con Windows Vista Professional y Windows Vista Edition Inclusive.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectID is int
ConnectID = FaxConnect()
...
ResSendFax is boolean
ResSendFax = FaxSend(ConnectID, "MyFax", "Message.txt", "0102030405")
...
ListReception is string 
ListReception = FaxInbox(ConnectID)
...
FaxDisconnect(ConnectID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxInbox(<Connection identifier>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de identificadores de fax separados por caracteres CR (Carriage Return). Este identificador le permite manipular el fax mediante las funciones de gestión de fax.. Los identificadores de fax corresponden a todos los faxes entrantes.

**`<Connection identifier>: (Integro)`**

Identificador de la conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md).

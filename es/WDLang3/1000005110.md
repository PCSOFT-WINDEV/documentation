
## NotesSave( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el actual Document en Lotus Notes. Esta Document fue modificada por [NotasModificarItem](../WDLang3/1000005113.md), [NotasDeleteItem](../WDLang3/1000005116.md) o [NotasAñadirAdjunto](../WDLang3/1000005119.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", ...
			"MarsServer", "email\julia.nsf", ...
			"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
...
// Read the first document 
NotesDocumentFirst(SessionID)
// Modify the subject of the document
NotesModifyItem(SessionID, "Subject", "My new subject")
// Save the modifications
NotesSave(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesSave(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el documento,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 


Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).


## tapiKeyPressed( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el historial de las teclas pulsadas en el teclado Phone desde la última llamada a **tapiKeyPressed**. El historial de teclas pulsadas se reinicia automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ListKeys is string 
ListKeys = tapiKeyPressed()
IF Position(ListKeys, "*") <> 0 THEN
	Trace("The * key was used")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiKeyPressed([<Call identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de teclas pulsadas desde la última llamada a la función **tapiKeyPressed**.

**Atención**: Esta lista corresponde a la historia de las teclas pulsadas a ambos lados del Phone Line (lado entrante y lado saliente).

**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). La llamada actual se gestiona si no se especifica este parámetro.

## Observaciones




### ¿Cómo saber si se ha pulsado una tecla?
<a name="¿como_saber_pulsado_una_tecla_ELTPARAGRAPHE000043"></a>

**Para saber si se ha pulsado una tecla**, utilice la constante *tapiMonitoredDigit* en el lenguaje WL Procedure llamado por [tapiDial](../WDLang3/3076013.md) o [tapiEscuchar](../WDLang3/3076029.md).

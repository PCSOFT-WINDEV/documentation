


## Funciones de gestión de citas (Outlook, Lotus Notes, dispositivo móvil)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones para gestionar las ocupaciones de muelles se utilizan para: 

- gestionar los contactos de la agenda de una mensajería Lotus Notes o Outlook. 









|   |   |
| --- | --- |
| [AppointmentAdd](../WDLang3/1000011001.md) | Añade una Appointment: <br><br>- en un Calendar de Lotus Notes o Outlook. <br><br>- en un Calendar en un dispositivo móvil (Android o iOS). <br><br><br> |
| [AppointmentCreate](../WDLang3/1000020691.md) | Muestra la ventana de creación de la aplicación nativa Appointment para la gestión de los Appointment que se encuentran en el dispositivo Android. |
| [AppointmentDelete](../WDLang3/1000011008.md) | Elimina: <br><br>- la cita actual desde la Calendar de una mensajería de Lotus Notes o Outlook. <br><br>- un Appointment del Calendar encontrado en un dispositivo móvil (Android/iOS).<br><br><br> |
| [AppointmentDisplay](../WDLang3/1000020692.md) | Muestra una Appointment en la aplicación nativa de gestión de Appointment s que se encuentra en el dispositivo móvil (Android o iOS). |
| [AppointmentFirst](../WDLang3/1000011005.md) | Lee la primera Appointment encontrada en la Calendar de una mensajería de Lotus Notes o Outlook. |
| [AppointmentLast](../WDLang3/1000011002.md) | Lee la última Appointment en un Calendar de Lotus Notes o Outlook . |
| [AppointmentList](../WDLang3/1000020690.md) | Enumera las citas encontradas en el dispositivo móvil (Android/iOS) y que corresponden a los criterios especificados. |
| [AppointmentListCalendar](../WDLang3/1000020661.md) | Lista los calendarios disponibles en el dispositivo móvil (Android/iOS). |
| [AppointmentModify](../WDLang3/1000011003.md) | Modifica la Appointment actual: <br><br>- en un Calendar de Lotus Notes o Outlook.<br><br>- en el Calendar encontrado en un dispositivo móvil (Android/iOS). <br><br><br> |
| [AppointmentNext](../WDLang3/1000011007.md) | Lee el Appointment que se encuentra después del Appointment actual en el Calendar de una mensajería de Lotus Notes o Outlook. |
| [AppointmentPrevious](../WDLang3/1000011004.md) | Lee la Appointment que se encuentra antes de la actual Appointment en la Calendar de un mensaje de Lotus Notes o de Outlook. |
| [AppointmentRead](../WDLang3/1000011011.md) | Lee una cita que se encuentra en el calendario de un mensaje de Lotus Notes o Outlook. |
| [AppointmentReset](../WDLang3/1000011006.md) | Reinicia: <br><br>- la estructura [Appointment](../WDLang3/1000011010.md).<br><br>- la [Appointment](../WDLang1/1000019244.md) Variable.<br><br><br> |
| [CitaCalendario del aseo](../WDLang3/1000021828.md) | Exporta un Appointment o una lista de Appointment s en formato iCalendar. |
| [iCalendarToAppointment](../WDLang3/1000021827.md) | Lista las citas descritas en una cadena de caracteres de formato iCalendar. |

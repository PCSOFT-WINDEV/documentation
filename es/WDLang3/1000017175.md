


## Funciones OBEX
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones OBEX son las siguientes: 



|   |   |
| --- | --- |
| [OBEXConnect](../WDLang3/1000017153.md) | Se utiliza para conectarse a un dispositivo compatible con el protocolo OBEX.. <br>Recordatorio: El protocolo OBEX es un protocolo para el intercambio de datos por infrarrojos y Bluetooth. |
| [OBEXDisconnect](../WDLang3/1000017154.md) | Se utiliza para desconectarse de un dispositivo que soporta el protocolo OBEX. |
| [OBEXGetFile](../WDLang3/1000017152.md) | Recupera un archivo de un dispositivo que soporta el protocolo OBEX. |
| [OBEXGetVCard](../WDLang3/1000017156.md) | Recupera una VCard de un dispositivo que soporta el protocolo OBEX. |
| [OBEXListFile](../WDLang3/1000017148.md) | Enumera los archivos compartidos por un dispositivo que utiliza el protocolo OBEX. |
| [OBEXSendFile](../WDLang3/1000017151.md) | Envía un archivo a un dispositivo compatible con el protocolo OBEX. |
| [OBEXSendVCard](../WDLang3/1000017155.md) | Envía un archivo VCard (tarjeta de visita virtual) a un dispositivo compatible con el protocolo OBEX. |

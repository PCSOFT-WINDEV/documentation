
## tapiCallIsOver( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la llamada entrante o saliente ha finalizado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is the current call ended?
IF tapiCallIsOver() = True THEN
	// Hang up the line
	tapiHangUp()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiCallIsOver([<Call identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la llamada ha terminado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). Si no se especifica este parámetro, se gestiona la llamada saliente o entrante actual..




## HTTPCookieReset (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las cookies (globalmente o para un dominio) almacenadas.: 

- por las llamadas a [HTTPRequest](../WDLang3/3043007.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) por las llamadas a [HTTPSend](../WDLang3/1000021183.md).


**Observación**: Las cookies leídas y enviadas por [HTTPRequest](../WDLang3/3043007.md) sólo se almacenan en la memoria de la aplicación. No están relacionados con las cookies de los navegadores Web instalados en el equipo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Request on the Google.com site as if it was the very first call
HTTPCookieReset(".google.com")
HTTPRequest("www.google.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borra las cookies almacenadas por httpRequest

`HTTPCookieReset([<Domain>])`
---

**`<Domain>: (Cadena de caracteres opcional)`**

Dominio para el que se borrarán las cookies. Si no se especifica este parámetro (si no se especifica ningún dominio), se borrarán todas las cookies.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Borra las cookies almacenadas por HTTPSend

`HTTPCookieReset(<HTTP request> [, <Domain>])`
---

**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre del [httpRequest](../WDLang3/1000021158.md) Variable para el que se borrarán las cookies.  

**`<Domain>: (Cadena de caracteres opcional)`**

Dominio para el que se borrarán las cookies. Si no se especifica este parámetro (si no se especifica ningún dominio), se borrarán todas las cookies.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un dominio corresponde al final de Address del servidor HTTP (".windev.com" por ejemplo).

- El uso de la función **HTTPCookieReset** afectará la próxima llamada a la función [HTTPRequest](../WDLang3/3043007.md) o [HTTPSend](../WDLang3/1000021183.md).

- La función **HTTPCookieReset** equivale a la función [HTTPCookieReplace](../WDLang3/1000019231.md) con un array vacío.

- **Atención**: **HTTPCookieReset** no borra las cookies de INTERNET Explorer.

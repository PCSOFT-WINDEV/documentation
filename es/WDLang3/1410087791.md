


## grRBRotationDirection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la dirección de rotación en un gráfico de barras radiales. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grRBRotationDirection(CHART_Chart1, grAnticlockwise)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grRBRotationDirection(<Chart name> , <Rotation>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Rotation>: (Constante de tipo Integer)`**

Dirección de rotación en el gráfico: 


|   |   |
| --- | --- |
| *grAnticlockwise* | Los datos del gráfico de la barra radial se mostrarán en sentido contrario a las agujas del reloj. |
| *grClockwise* | Los datos de la barra radial se mostrarán en el sentido de las agujas del reloj. |




## grSunburstAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a una carta de Sunburst.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Montpellier", 320000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Paris", 789000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France", 1109000) 
grSunburstAddData(CHART_Chart, "Europe" + TAB + "Belgium", 900000)
grSunburstAddData(CHART_Chart, "Asia", 250000)
grDraw(CHART_Chart)
```

## Sintaxis
<a name="SYNTAX1"></a>

`grSunburstAddData(<Chart name> , <Section path> , <Value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección que se añadirá al gráfico. Este parámetro tiene el siguiente formato:

```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```
 Esta ruta define la posición de los datos especificando sus diferentes niveles jerárquicos. 
Por ejemplo, si la ruta contiene "Europa + TAB + Francia + TAB + Montpellier", los datos se añadirán en el tercer nivel de la tabla debajo de las categorías "Europa\\Francia\\Montpellier".

**`<Value>: (Real)`**

Valor de la parte añadida. Este valor no debe ser negativo.

## Observaciones


### gráfico de proyección solar
<a name="grafico_proyeccion_solar_ELTPARAGRAPHE000102"></a>

- Si un nivel no contiene su propio valor, su valor se calculará sumando los valores de los subniveles.

- Si la suma de los datos para los subniveles de un nivel específico excede el valor de este nivel, se devolverá un error al dibujar el gráfico.

- La Color de los subniveles se calcula automáticamente: este Color tiene el mismo tono que el Color del nivel principal pero más pálido.



<a name="NOTE0_2"></a>


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000113"></a>

[grAddData](../WDLang3/3042023.md) se utiliza para añadir datos en un gráfico (circular, de columnas, ...)

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [grStockAddData](../WDLang3/3042009.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [grHMAddData](../WDLang3/1000021628.md).

- Para los gráficos de dispersión, se recomienda utilizar [grScatterAddDataXY](../WDLang3/3042032.md).

- Para los gráficos de superficie, se recomienda utilizar [grSurfaceAddData](../WDLang3/1000020425.md).




Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [grIncreaseData](../WDLang3/3042030.md).


## BTActivate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la gestión de Bluetooth en el dispositivo. Para conocer el estado de activación del Bluetooth en el dispositivo o recibir una notificación cuando el estado cambie, utilice [BTStatus](../WDLang3/1000019360.md).

**Observación**:  Las funciones de Bluetooth no están disponibles en el emulador de Android. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF BTStatus() = btDisabled THEN
	BTActivate(True)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BTActivate([<Activation of Bluetooth>])`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de activación del Bluetooth después de la modificación. Los valores posibles son: 


|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |
| *btError* | No se pudo recuperar el estado de activación del Bluetooth. |



**`<Activation of Bluetooth>: (Booleano opcional)`**

Nuevo estado de activación de Bluetooth: 

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): Habilita el Bluetooth en el dispositivo. 

- <u><u><u><u>False</u></u></u></u>: Desactiva el Bluetooth del dispositivo.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000077"></a>

La llamada a esta función modifica los permisos requeridos por la aplicación.

Permisos necesarios:

- BLUETOOTH: esta Permission permite que las aplicaciones se conecten a los servicios Bluetooth. 

- BLUETOOTH_ADMIN: esta Permission permite a las aplicaciones descubrir y conectar los servicios Bluetooth.

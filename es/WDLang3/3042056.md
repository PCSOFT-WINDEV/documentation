


## grTitle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la leyenda y/o la posición del título del gráfico.

Ejemplo de título en un gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)


El gráfico utilizado puede ser:

- un gráfico creado mediante programación ([grCreate](../WDLang3/3042005.md)).

- un control Gráfico en el editor de ventanas, editor de páginas o editor de reportes.




**Observación**: Un gráfico no tiene título de Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the caption and the position of title in "MyChart"
grTitle("MyChart", "Turnover projection", grAtBottom)
```
<a name="Example2"></a>

```wl
// Modify the caption and the position of title in CHART_MyChart
grTitle(CHART_MyChart, "Turnover projection", grAtTop)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grTitle(<Chart name> [, <Title> [, <Position>]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Title>: (Cadena de caracteres opcional)`**

Título del gráfico.

**`<Position>: (Constante opcional)`**

Indica la posición del título:


|   |   |
| --- | --- |
| *grAtBottom* | Título mostrado en la parte inferior del gráfico. |
| *grAtTop* | Título mostrado en la parte superior del gráfico. |
| *grNone* | Sin título. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Visualización del título
<a name="visualizacion_del_titulo_ELTPARAGRAPHE000167"></a>

Las modificaciones realizadas con la función **grTitle** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).
<a name="NOTE0_2"></a>


### Posicionamiento manual del título: grTitle sin efecto
<a name="posicionamiento_manual_del_titulo_docparampagetitleshort_sin_efecto_ELTPARAGRAPHE000181"></a>

**grTitle** no tiene efecto si el título del gráfico se coloca manualmente ("Posicionamiento manual" en la ventana de título del gráfico).
<a name="NOTE0_3"></a>


### Las lenguas que utilizan un carácter no latino set
<a name="las_lenguas_que_utilizan_caracter_latino_set_ELTPARAGRAPHE000198"></a>

**grTitle** utilizará el carácter no latino seleccionado set sólo si la configuración del proyecto associated utiliza cadenas UNICODE en tiempo de ejecución.

Para comprobar la opción seleccionada en la configuración actual: 

1. En la pestaña "Proyecto", en el grupo "Configuración del proyecto", haga clic en "Configuración actual". 

2. Ir a la pestaña "Unicode": la opción "Utilizar cadenas UNICODE en ejecución" debe estar seleccionada. 

3. Validar si es necesario.

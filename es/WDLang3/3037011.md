
## Estructura de estado de FaxCompleteStatus
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000362"></a>
La estructura FaxCompleteStatus se rellena automáticamente durante la llamada a [FaxStatus](../WDLang3/3037001.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Detalles de la estructura de FaxCompleteStatus
<a name="detalles_estructura_faxcompletestatus_ELTTEXTE000386"></a>



|   |   |
| --- | --- |
| **FaxCompleteStatus.ScheduleDateTime** | FechaHora<br><br>Fecha y hora de envío del fax. |
| **FaxCompleteStatus.SenderDept** | Cadena de caracteres<br><br>Departamento de remitente. |
| **FaxCompleteStatus.InError** | booleano<br><br>True: Especifica si se ha cancelado el fax (incluida la cancelación de usuario). |
| **FaxCompleteStatus.Sent ByCurrentConnection** | booleano<br><br>True: Especifica si se envía un fax a través de la conexión especificada en [FaxStatus](../WDLang3/3037001.md) (&lt;Identificador de conexión&gt;). |
| **FaxCompleteStatus.Status** | Constante de tipo Integer<br><br>Estado del fax a nivel de dispositivo:<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxStatusCanceled* \ Se cancela la transmisión del fax. \<br>\ *FaxStatusCall* \ El fax se está enviando. \<br>\ *FaxStatusLocked* \ El número de destinatario está bloqueado. \<br>\ *FaxStatusDisconnected* \ Desconexión durante la transmisión. \<br>\ *FaxStatusSend* \ El fax se está enviando actualmente. \<br>\ *FaxStatusSent* \ El fax fue enviado. \<br>\ *FaxStatusError* \ No se le puede dar a get el estado del fax. \<br>\ *FaxStatusFatalError* \ Error fatal. \<br>\ *FaxStatusBadNumber* \ Número de fax equivocado. \<br>\ *FaxStatusServerPending* \ El dispositivo está disponible. \<br>\ *FaxStatusOffline* \ El dispositivo está desconectado y no está disponible. \<br>\ *FaxStatusInactive* \ El envío de faxes está desactivado (pausa entre dos llamadas, por ejemplo). \<br>\ *FaxStatusUnavailable* \ El fax está ocupado (utilizado por otra aplicación) \<br>\ *FaxStatusInit* \ La transmisión está actualmente inicializada \<br>\ *FaxStatusBusy* \ El destinatario está ocupado \<br>\ *FaxStatusNotFaxCall* \ El dispositivo recibió una llamada que no era una llamada de fax (llamada Phone por ejemplo) \<br>\ *FaxStatusNoDialTone* \ No hay tono de marcado \<br>\ *FaxStatusHandled* \ El fax es manejado y transmitido por el administrador de fax. \<br>\ *FaxStatusReceiving* \ El dispositivo está recibiendo actualmente un fax \<br>\ *FaxStatusAnswered* \ El destinatario respondió \<br>\ *FaxStatusDelayed* \ La llamada se retrasa \<br>\ *FaxStatusRouting* \ El dispositivo está enrutando un fax entrante. \<br>\ *FaxStatusNoAnswer* \ El destinatario no responde \<br>\ *FaxStatusRing* \ El fax del destinatario está sonando \<br><br> |
| **FaxCompleteStatus.QueueStatus** | Entero.<br><br>Estado del fax en el gestor de fax. Puede corresponder a una combinación de las siguientes constantes:<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxQueueStatusFailed* \ El fax no se ha enviado \<br>\ *FaxQueueStatusPending* \ El fax está pendiente \<br>\ *FaxQueueStatusInProgress* \ El fax se está enviando actualmente \<br>\ *FaxQueueStatusRetriesExceeded* \ Se ha alcanzado el número máximo de intentos y no se enviará el fax. \<br>\ *FaxQueueStatusNoLine* \ No hay Line disponible para enviar el fax. Se enviará cuando un Line esté disponible. \<br>\ *FaxQueueStatusPaused* \ El fax está en pausa. \<br>\ *FaxQueueStatusRetrying* \ El fax no se envió y se hará un nuevo intento más tarde. \<br>\ *FaxQueueStatusDeleting* \ El fax está actualmente borrado. \<br><br> |
| **FaxCompleteStatus.Identifier** | Integro<br><br>Identificador numérico del fax. |
| **FaxCompleteStatus.StatusCaption** | Cadena de caracteres<br><br>description del estado del fax. |
| **FaxCompleteStatus.PageCount** | Entero.<br><br>Número total de páginas a transmitir. |
| **FaxCompleteStatus.RecipientName** | Cadena de caracteres<br><br>Nombre del destinatario. |
| **FaxCompleteStatus.DocumentName** | Cadena de caracteres<br><br>Nombre del fax. |
| **FaxCompleteStatus.SenderName** | Cadena de caracteres<br><br>Nombre del remitente. |
| **FaxCompleteStatus.DeviceName** | Cadena de caracteres<br><br>Nombre del dispositivo (módem) utilizado para la transmisión. |
| **FaxCompleteStatus.RecipientNumber** | Cadena de caracteres<br><br>Número de destinatarios. |
| **FaxCompleteStatus.CurrentPage** | Entero.<br><br>Número de la página que se está transmitiendo actualmente. |
| **FaxCompleteStatus.ScheduleAction** | Constante de tipo Integer.<br><br>Se utiliza para programar el fax saliente:<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxScheduleActionDiscountPeriod* \ El fax se enviará en horas no pico. \<br>\ *FaxScheduleActionSpecificTime* \ El fax se enviará a la hora especificada en FaxCompleteStatus.ScheduleDateTime. \<br>\ *FaxScheduleActionNow* \ El fax se enviará lo antes posible. \<br><br> |
| **FaxCompleteStatus.Owner** | Cadena.<br><br>Nombre del usuario que emitió el fax. |
| **FaxCompleteStatus.SenderCompany** | Cadena.<br><br>Empresa de remitente. |
| **FaxCompleteStatus.SizeInBytes** | Integro<br><br>Tamaño de la Document transmitida (en bytes). |
| **FaxCompleteStatus.Finalizado** | Booleano.<br><br>Especifica si la transmisión está terminada (lo que significa finalizada o cancelada). |
| **FaxCompleteStatus.Type** | Constante de tipo Integer.<br><br>Tipo de fax:<br><br><br>\   \   \<br>\ --- \ --- \<br>\ *FaxTypeFailReceive* \ El fax aún no se ha enrutado porque no se ha recibido. \<br>\ *FaxTypeSend* \ Fax sent. \<br>\ *FaxTypeRouting* \ El fax no ha sido enrutado. Se volverá a intentar más tarde. \<br>\ *FaxTypeUnknown* \ Tipo desconocido porque el fax aún no está programado. \<br>\ *FaxTypeReceive* \ Fax recibido. \<br><br> |

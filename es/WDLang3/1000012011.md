
## TaskRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee una tarea en la lista de tareas de un mensaje de Lotus Notes o de Outlook.. Para ser leída, esta tarea debe haber sido leída previamente ([TaskLast](../WDLang3/1000012002.md), [TareaAnterior](../WDLang3/1000012004.md), [TaskFirst](../WDLang3/1000012005.md) o [TaskNext](../WDLang3/1000012007.md)), añadida ([Añadir tarea](../WDLang3/1000012001.md)) o modificada ([TaskModify](../WDLang3/1000012003.md)).

**Atención**: Esta función **no se puede utilizar** para leer una tarea en la lista de tareas de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf", ...
					"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Read the first task
TaskFirst(SessionID)
...
// Read this same task again
TaskRead(SessionID, mTask.ID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TaskRead(<Session identifier> , <Task identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la tarea,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

**`<Task identifier>: (Cadena de caracteres)`**

Identificador de la tarea a leer. Este identificador se encuentra en el mTask.ID Variable.

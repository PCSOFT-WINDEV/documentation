


## grScatterSeriesColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la Color de una serie en un gráfico de "Dispersión". 

**Recordatorio**: Por defecto, los colores de las series (o los colores de las secciones) se seleccionan según un orden preset entre un set de 13 colores.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a data into the "Scatter" chart named "MyChart".
// This value (10, 20) is added at the end of the first series.
grScatterAddDataXY("MyChart", 1, 10, 20)
// The series 1 of "MyChart" is colored in yellow
grScatterSeriesColor("MyChart", 1, LightYellow)
```
<a name="Example2"></a>

```wl
// Adds a data into the CHART_MyChart control ("Scatter" chart).
// This value (20, 20) is added at the end of the first series.
grScatterAddDataXY(CHART_MyChart, 1, 20, 20)
// The series 1 of CHART_MyChart is colored in green
grScatterSeriesColor(CHART_MyChart, 1, LightGreen)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grScatterSeriesColor(<Chart name> , <Series number> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de serie a utilizar.

**`<Color>: (Entero o constante)`**

Color de la serie especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los datos definidos con la función **grScatterSeriesColor** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- Para visualizar los gradientes de Color, utilice [grGradient](../WDLang3/3042029.md).

- La función **grScatterSeriesColor** es equivalente a la función [grSeriesColor](../WDLang3/3042013.md). 
	La sintaxis de [grSeriesColor](../WDLang3/3042013.md) utilizada con los gráficos de "dispersión" es:
	
	```txt
	grSeriesColor(<Chart name>, 2*<Index>-1, <Color>)
	```

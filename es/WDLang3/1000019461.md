
## SaaSConnectedUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el usuario actualmente conectado a través de [SaaSConnect](../WDLang3/1000018959.md). Esta función se utiliza para encontrar al usuario conectado a un sitio SaaS a través de una página de conexión de automatic.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
U is saasUser 
U = SaaSConnectedUser()
Info("Hello" + U.FirstName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSConnectedUser()`
---

**`<Result>: (saasUser Variable)`**

[saasUser](../WDLang3/1000019083.md) variable que contiene las características del usuario conectado.

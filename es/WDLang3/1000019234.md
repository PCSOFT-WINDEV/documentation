


## HTTPCookieWrite (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade o modifica un Cookie en una petición HTTP. 

Hay dos tipos de uso disponibles: 

- Si se ejecuta la solicitud con la función [HTTPRequest](../WDLang3/3043007.md), se debe llamar a la función HTTPCookieWrite antes de la función [HTTPRequest](../WDLang3/3043007.md) para establecer el valor de las cookies que se enviarán al servidor HTTP.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la solicitud se ejecuta con [HTTPSend](../WDLang3/1000021183.md), **HTTPCookieWrite** debe ejecutarse en el httpRequest Variable (sintaxis 2). 


**Observación**: Las cookies leídas y enviadas por [HTTPRequest](../WDLang3/3043007.md) sólo se almacenan en la memoria de la aplicación. No están relacionados con las cookies de los navegadores Web instalados en el equipo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sets the value of an identification cookie with a stored value
HTTPCookieWrite("domain.com", "ID", LoadParameter("ID"))

// Performs an HTTP request
HTTPRequest("www.MySite.com?Login")

// Retrieves and stores the value of the cookie returned by the server
sID is string = HTTPCookieRead(".mysite.com", "ID")
SaveParameter("ID", sID)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Solicitud dirigida por httpRequest

`HTTPCookieWrite(<Domain> , <Name> , <Value>)`
---

**`<Domain>: (Cadena de caracteres)`**

Dominio de validez de Cookie.

El Cookie se enviará a todas las solicitudes HTTP cuyo dominio termine con el dominio de validez de Cookie.

Por ejemplo, un Cookie cuyo dominio de validez es ".midominio.fr" será enviado durante las siguientes solicitudes:

- http://mydomain.fr

- http://www.mydomain.fr/index.html

- http://server.mydomain.fr/Page.awp


No se enviará en las siguientes solicitudes:

- http://www.mydomain.com

- http://otherdomain.fr




**`<Name>: (Cadena de caracteres)`**

Nombre de Cookie.

Este nombre no distingue entre mayúsculas y minúsculas.

**`<Value>: (Cadena de caracteres)`**

Valor de Cookie.

Observaciones:

- El valor de un Cookie está limitado a 4KB. Más allá de este tamaño, el servidor es libre de rechazar el Cookie.

- Los caracteres de control (TAB, CR, etc.) así como el punto y coma y los ceros binarios no están permitidos.

## Observaciones


- Si la gestión de las cookies no fue habilitada por [HTTPCookieManage](../WDLang3/1000019218.md), las cookies no se envían al servidor: 

	- durante la llamada a [HTTPRequest](../WDLang3/3043007.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) durante la llamada a [HTTPSend](../WDLang3/1000021183.md).




- La Cookie se crea si no existe.


## ModbusConnectRTU( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta un cliente a un esclavo Modbus a través de un puerto serie.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates a session and connects to the Modbus slave through the COM4 serial port
Session is modbusSession
Session = ModbusConnectRTU("COM4",1)

// If the connection fails, display the error.
IF ErrorOccurred = True THEN
	Trace(ErrorInfo())
ELSE
	// Write a value to register 0x42 in the slave.
	ModbusWriteRegister(Session, 0x42,0xCAFE)
END

// Release serial port
ModbusDisconnect(Session)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Session> = ModbusConnectRTU(<Serial port> [, <Slave ID> [, <Rate> [, <Parity> [, <Nb data Bits> [, <Nb stop Bits>]]]]])`
---

**`<Session>: (modbusSession Variable)`**

[modbusSession](../WDLang3/1000025924.md) Variable utilizado para comunicarse con el esclavo Modbus.

**`<Serial port>: (Cadena de caracteres)`**

Nombre del puerto en serie que se va a utilizar: COM1, COM2, COM3, ... COM256.

**`<Slave ID>: (Entero opcional)`**

Identificador de esclavo, entre 0 y 255. 
Por defecto, este parámetro es set a 0 (difusión Address). 
Observación: También se puede especificar el valor 0 si el ID de esclavo se va a especificar posteriormente a través del **SlaveID** Property del [modbusSession](../WDLang3/1000025924.md) Variable.

**`<Rate>: (Entero opcional)`**

Velocidad de transferencia de datos (expresada en baudios o bits por segundo) que se utilizará para la conexión en serie. Por defecto, este parámetro es set a 9600 baudios.

**`<Parity>: (Entero opcional)`**

Paridad de transferencia en el puerto:



|   |   |
| --- | --- |
| 0 | No parity |
| 1 | Paridad par |
| 2 | Paridad impar |

Observación: No hay paridad Default. 

**`<Nb data Bits>: (Entero opcional)`**

Número de bits de datos en el puerto serie: 5, 6, 7 u 8.

**`<Nb stop Bits>: (Entero opcional)`**

Número de bits de parada en el puerto serie: 



|   |   |
| --- | --- |
| 0 <br>(Valor predeterminado) | 1 bit de parada |
| 1 | 1.5 bits de parada |
| 2 | 2 bits de parada |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Después de llamar a **ModbusConnectRTU**, se recomienda comprobar el [ErrorOccurred](../WDLang1/3087001.md) Variable para asegurarse de que no hay errores.

- El identificador del esclavo está en el [modbusSession](../WDLang3/1000025924.md) Variable. Si una operación debe realizarse en varios esclavos diferentes, puede cambiar el identificador del esclavo en la [modbusSession](../WDLang3/1000025924.md) Variable para evitar la reapertura de la sesión.

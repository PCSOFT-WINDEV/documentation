


## grSeriesLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la etiqueta de una serie de datos en un gráfico.

Ejemplo de etiqueta de serie:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 Para mostrar la etiqueta, utilice [grLegend](../WDLang3/3042047.md).

**Recordatorio**: Por Default, un gráfico no tiene etiqueta. Para más detalles sobre la gestión de leyendas, etiquetas, ... consulte [Gestión de leyendas en los gráficos](../WDChamp/3042069.md). 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the "Sales" label of second data series in "MyChart"
grSeriesLabel("MyChart", 2, "Sales")
```
<a name="Example2"></a>

```wl
// Initialize the "Sales" label of second data series in CHART_MyChart
grSeriesLabel(CHART_MyChart, 2, "Sales")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grSeriesLabel(<Chart name> , <Series number> , <Label>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie para la que se inicializará una etiqueta.

**`<Label>: (Cadena de caracteres)`**

Leyenda de la etiqueta a inicializar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Etiquetas de la serie
<a name="etiquetas_serie_ELTPARAGRAPHE000139"></a>Las etiquetas de la serie set corresponden a la leyenda de la carta.
Las etiquetas de la serie <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 no pueden utilizarse en los gráficos circulares.

Las etiquetas de serie se definen con **grSeriesLabel**. 

Los datos definidos con la función **grSeriesLabel** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

La fuente de la leyenda puede cambiarse con [grLegendFont](../WDLang3/3042037.md).  

[grLegend](../WDLang3/3042047.md) indica la presencia y la posición de la leyenda (es decir, todas las etiquetas de las series).

**Observación**: Para definir las etiquetas de las series en los gráficos de "dispersión", le recomendamos que utilice [grScatterSeriesLabel](../WDLang3/3042038.md).
<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000176"></a>

La función **grSeriesLabel** es equivalente a la función [Leyenda](../Proprietes/2510143.md).

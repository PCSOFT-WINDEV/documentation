


## grScatterGetDataXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un dato de un gráfico"Scatter".. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display data in the "CHART_MyChart" control ("Scatter" chart). 
// This data is the first X-coordinate of first series.
Info(grScatterGetDataXY(CHART_MyChart, 1, 1, grXValue))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = grScatterGetDataXY(<Chart name> , <Series number> , <Index> , <Value to retrieve>)`
---

**`<Result>: (Real)`**

Valor recuperado.

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md), 

- el nombre de la control Gráfico (en la ventana, página o editor de reportes).




**`<Series number>: (Integro)`**

Número de la serie para la que se recuperará un valor. 
Si la serie especificada no existe, el valor recuperado es 0.

**`<Index>: (Integro)`**

Índice de los datos de la serie cuyo valor se recuperará.
Si el dato correspondiente al índice especificado no existe en la serie, el valor recuperado es 0.

**`<Value to retrieve>: (Constante de tipo Integer)`**

Indica el valor a recuperar:


|   |   |
| --- | --- |
| *grXValue* | Valor X de la serie. |
| *grYValue* | Valor Y de la serie. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [grStockGetData](../WDLang3/1000021174.md).

- Para recuperar los datos de un gráfico de superficie, se recomienda utilizar [grSurfaceGetData](../ErrCompil/1000001036.md).

- Para recuperar los datos de otro tipo de gráfico (circular, de columnas, Line, etc.), le recomendamos que utilice [grGetDatos](../WDLang3/1000021171.md).

- Para recuperar los datos de un gráfico Sunburst, se recomienda utilizar [grSunburstGetData](../WDLang3/1000021287.md). 

- Para recuperar los datos de un gráfico Heatmap, se recomienda utilizar [grHMGetData](../WDLang3/1000021632.md).

- Le recomendamos utilizar la función **grScatterGetDataXY** para recuperar datos en gráficos de "dispersión". También se puede utilizar la función [grGetDatos](../WDLang3/1000021171.md).
	La siguiente sintaxis:
	
	```wl
	grScatterGetDataXY(CHART_MyChart, i, 1, grXValue)
	grScatterGetDataXY(CHART_MyChart, i, 1, grYValue)
	```
 es equivalente a la sintaxis: 
	
	```wl
	grGetData(CHART_MyChart, i*2-1, 1)
	grGetData(CHART_MyChart, i*2, 1)
	```





- Por Default, un gráfico creado por [grCreate](../WDLang3/3042005.md) no contiene datos.

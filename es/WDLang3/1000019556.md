
## ContactCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana de creación de Contact de la aplicación nativa de gestión de Contact que se encuentra en el dispositivo (Android, iPhone o iPad, Universal Windows).

## Sintaxis
<a name="SYNTAX1"></a>

### Creación de un contacto e inicialización del contacto Variable

`<Result> = ContactCreate([<Contact>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó un contacto,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

- <u><u><u><u>True</u></u></u></u> si se muestra la interfaz de usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Contact>: (Contacte con Variable)`**

Nombre de la [Contact](../WDLang3/1000020254.md) Variable que contiene los datos del nuevo Contact.

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000085"></a>

- Si el usuario crea una nueva Contact, la [Estructura de Contact](../WDLang3/1000006111.md) o la [Contact](../WDLang3/1000020254.md) Variable se rellena automáticamente con la información relativa a esta nueva Contact.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **ContactCreate** es una función no bloqueante. 

- **ContactCreate** puede utilizarse en el emulador. No se puede utilizar en el simulador.



<a name="NOTE0_2"></a>


### Creación de una Contact (sin utilizar la ventana de creación)
<a name="creacion_una_contact_sin_utilizar_ventana_creacion_ELTPARAGRAPHE000113"></a>

Para crear una nueva Contact de forma programada (sin abrir ninguna ventana): 

1. Describa el contacto utilizando la [Contact](../WDLang3/1000020254.md) Variable. 

2. Utilice la función [ContactAdd](../WDLang3/1000006101.md).

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000323"></a>

Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.

Característica necesaria: Contactos

Esta característica permite a las aplicaciones acceder a los contactos.


## geoTrackingStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el estado del sistema para el seguimiento de posición.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
If geoTrackingStatus() = geoTrackingDisabled THEN
	RETURN
END

// process depending on the active position tracking
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = geoTrackingStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del sistema de seguimiento: 



|   |   |
| --- | --- |
| *geoTrackingDisabled*  | El sistema de localización está desactivado. |
| *geoTrackingEnabled* | El sistema de seguimiento de la ubicación está activado. |
| *geoTrackingError* | Error al recuperar el estado de activación del sistema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). |

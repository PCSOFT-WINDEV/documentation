


## Funciones de telefonía
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones le permiten fácilmente manipular todas las características de telefonía de un módem desde una aplicación WINDEV o WINDEV Mobile:



|   |   |
| --- | --- |
| [iOSAppGroupDirectory](../WDLang3/1410088694.md) | Devuelve la ruta del contenedor utilizado para compartir archivos entre varias aplicaciones que pertenecen al mismo grupo de aplicaciones de iOS. |
| [iOSExcludeFileFromiCloud](../WDLang3/1000021779.md) | Excluye un archivo de la copia de seguridad de iCloud. |
| [tapiAnswerCall](../WDLang3/3076026.md) | Contesta una llamada entrante que fue detectada. |
| [tapiBlindTransfer](../WDLang3/3076011.md) | Realiza una transferencia "a ciegas". |
| [tapiCallDuring](../WDLang3/3076004.md) | Devuelve la duración de la llamada (diferencia entre la fecha y hora de inicio de la llamada y la fecha y hora de finalización de la misma). |
| [tapiCalledID](../WDLang3/3076025.md) | Se utiliza para averiguar el número de Phone llamado. |
| [tapiCallEnd](../WDLang3/3076002.md) | Devuelve la fecha y hora de finalización de la llamada. |
| [tapiCallerID](../WDLang3/3076010.md) | Se utiliza para averiguar el número de Phone que llama (el que llama). |
| [tapiCallIsBusy](../WDLang3/3076017.md) | Se usa para averiguar si el número al que se llama está ocupado. |
| [tapiCallIsOver](../WDLang3/3076003.md) | Permite saber si la llamada entrante o saliente ha finalizado. |
| [tapiCallIsWaiting](../WDLang3/3076033.md) | Se utiliza para saber si la llamada está en espera. |
| [tapiCallStart](../WDLang3/3076006.md) | Devuelve la fecha y hora de inicio de la llamada (llamada entrante o saliente). |
| [tapiCapability](../WDLang3/3076014.md) | Devuelve las características de un dispositivo de telefonía. |
| [tapiCompleteTransfer](../WDLang3/3076021.md) | Realiza una llamada Transfer( con capacidad para recuperar la llamada). |
| [tapiDevice](../WDLang3/3076032.md) | Selecciona el dispositivo TAPI que se utilizará durante las siguientes operaciones de telefonía: <br><br>- [tapiDial](../WDLang3/3076013.md),<br><br>- [tapiListen](../WDLang3/3076029.md).<br><br><br> |
| [tapiDeviceList](../WDLang3/3076024.md) | Enumera los dispositivos compatibles con TAPI 2.0 y TAPI 3.1 instalados en el ordenador actual. |
| [tapiDial](../WDLang3/3076013.md) | Marca un número de Phone para una voz Line. |
| [tapiError](../WDLang3/3076020.md) | Indica si la última función tapixxx ([funciones para gestionar las llamadas a Phone](../WDLang3/3076005.md)) al módulo TAPI ha devuelto un error. |
| [tapiHangUp](../WDLang3/3076030.md) | Cuelga un Phone Line iniciado con [tapiDial](../WDLang3/3076013.md) o [tapiAnswerCall](../WDLang3/3076026.md). |
| [tapiHold](../WDLang3/3076034.md) | Pone una llamada en espera. |
| [tapiKeyPressed](../WDLang3/3076027.md) | Devuelve el historial de las teclas pulsadas en el teclado Phone desde la última llamada a tapiKeyPressed. |
| [tapiLineDial](../WDLang3/3076012.md) | Marca un número de Phone para una voz Line y elige el dispositivo a utilizar. |
| [tapiListen](../WDLang3/3076029.md) | Inicia la detección de llamadas (también llamado "call monitoring service"). |
| [tapiNoAnswer](../WDLang3/3076008.md) | Se utiliza para averiguar si se ha dado una respuesta a la llamada. |
| [tapiOrigin](../WDLang3/3076028.md) | Permite determinar el tipo de llamada (entrante o saliente). |
| [tapiPlay](../WDLang3/3076023.md) | Reproduce un archivo de sonido (.WAV) para la Line especificada . |
| [tapiRecord](../WDLang3/3076019.md) | Records the current communication as a".WAV" file. |
| [tapiSendKey](../WDLang3/3076018.md) | Permite simular el uso de las teclas de Phone. |
| [tapiStop](../WDLang3/3076007.md) | Deja de leer un mensaje pregrabado ([tapiPlay](../WDLang3/3076023.md)). |
| [tapiStopCallDetection](../WDLang3/3076022.md) | Detiene la detección de llamada especificada que fue iniciada por [tapiListen](../WDLang3/3076029.md). |
| [tapiUnhold](../WDLang3/3076035.md) | Recibe una llamada en espera. |







![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Estas funciones son compatibles con el protocolo TAPI 2.0.

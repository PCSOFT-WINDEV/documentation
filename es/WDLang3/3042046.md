


## grLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica varios parámetros de una leyenda adicional en un gráfico.

Ejemplo de leyendas adicionales: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grLibell%E9%20-%20HC%20N%B0001.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
// Display the caption for the labels of data categories in "MyChart"
grLabel("MyChart", grShowLabel, True)
```
<a name="Example2"></a>

```wl
// Chart control
// Display the caption for the percentages of data categories in "CHART_MyChart"
grLabel(CHART_Chart, grShowPercent, True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grLabel(<Chart name> , <Additional legend> , <Display>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Additional legend>: (Constante)`**

Leyenda adicional que se mostrará con el gráfico:

- ***con un gráfico circular***: 
	


|   |   |
| --- | --- |
| *grShowLabel* | Etiqueta de categorías de datos. |
| *grShowPercent* | Porcentaje de categorías de datos. |
| *grShowValue* | Valor de las categorías de datos. |






- ***con un gráfico de columnas***:
	


|   |   |
| --- | --- |
| *grShowValue* | Valor de las categorías de datos. |






- ***con un gráfico de columnas apiladas***: 
	


|   |   |
| --- | --- |
| *grShowValueIfNull* | Etiqueta de categorías de datos, incluso para los valores nulos en el gráfico de columnas apiladas. |






**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe mostrar la leyenda adicional,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Leyenda del gráfico
<a name="leyenda_del_grafico_ELTPARAGRAPHE000173"></a>

El título del gráfico puede incluir tres elementos:

- las etiquetas de las categorías.

- el valor de las categorías.

- el porcentaje de categorías.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grLibell%E9%20-%20HC%20N%B0001.gif)
Los títulos de los gráficos sólo pueden mostrarse en los gráficos circulares, de columnas y de columnas apiladas.

**grLabel** muestra los diferentes elementos de la leyenda del gráfico. Se visualizan diferentes elementos según el tipo de gráfico.

Las etiquetas de las categorías mostradas corresponden a las definidas por [grCategoryLabel](../WDLang3/3042015.md).

El valor de las categorías depende de los datos del gráfico.

El porcentaje de categorías se calcula automáticamente.

La pantalla de información puede ser formateada por [grMask](../WDLang3/3042034.md).

Las modificaciones realizadas con la función **grLabel** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

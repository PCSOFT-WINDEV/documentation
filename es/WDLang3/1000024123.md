


## &lt;Gráfico&gt;.levantado (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un gráfico con efecto de relieve. Basta con especificar la profundidad del efecto. Todos los tipos de gráficos se pueden mostrar con un efecto elevado.

Ejemplo de gráfico con efecto en relieve: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Relief.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the depth of raised effect for CHART_MyChart
CHART_MyChart.Raised(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.Raised(<Raised>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada en la ventana, página o editor de reportes.

**`<Raised>: (Integro)`**

Profundidad del efecto de aumento (en píxeles). Le aconsejamos que no utilice un efecto de alto relieve para que el gráfico se muestre correctamente..
Si este parámetro es igual a 0, el gráfico se muestra en dos dimensiones (sin efecto en relieve)..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Todos los tipos de gráficos se pueden mostrar con un efecto elevado.

- Los datos definidos con la función **&lt;Gráfico&gt;.levantado** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).




## &lt;Gráfico&gt;.ScatterSeriesColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la Color de una serie en un gráfico de "Dispersión". 

**Recordatorio**: Por defecto, los colores de las series (o los colores de las secciones) se seleccionan según un orden preset entre un set de 13 colores.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a data into the CHART_MyChart control ("Scatter" chart).
// This value (20, 20) is added at the end of the first series.
CHART_MyChart.ScatterAddDataXY(1, 20, 20)
// The series 1 of CHART_MyChart is colored in green
CHART_MyChart.ScatterSeriesColor(1, LightGreen)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.ScatterSeriesColor(<Series number> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de serie a utilizar.

**`<Color>: (Entero o constante)`**

Color de la serie especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los datos definidos con la función **&lt;Gráfico&gt;.ScatterSeriesColor** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- Para visualizar los gradientes de Color, utilice [&lt;Tabla&gt;.Gradient](../WDLang3/1000024061.md).

- La función **&lt;Gráfico&gt;.ScatterSeriesColor** es equivalente a la función [&lt;Gráfico&gt;.SeriesColor](../WDLang3/1000023599.md). 
	La sintaxis de [&lt;Gráfico&gt;.SeriesColor](../WDLang3/1000023599.md) utilizada con los gráficos de "dispersión" es:
	
	```txt
	<Chart control>.SeriesColor(2*<Index>-1, <Color>)
	```


## HTTPAddFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un archivo al formulario HTTP. Este formulario ha sido creado por [HTTPCreateForm](../WDLang3/1000017119.md) y será enviado por [HTTPSendForm](../WDLang3/1000017190.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPCreateForm("FORM")
HTTPAddParameter("FORM", "Price", "3400")
HTTPAddFile("FORM", "Photo", "C:\Temp\MyPhoto.BMP")
...
HTTPSendForm("FORM", "www.mysite.com")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HTTPAddFile(<Form name> , <Parameter name> , <File name> [, <Type of file>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó el archivo al formulario HTTP, 

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Form name>: (Cadena de caracteres)`**

Nombre del formulario HTTP a utilizar. Este nombre se definió con [HTTPCreateForm](../WDLang3/1000017119.md). Se produce un error si el formulario no existe.

**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro a añadir al formulario. Este parámetro será relacionado al archivo. Si el nombre del parámetro ya existe en el formulario, se sobrescribe su valor.

**`<File name>: (Cadena de caracteres)`**

ruta completa del archivo que se añadirá al formulario. Este archivo debe ser accesible desde el ordenador actual. De hecho, el archivo se abre inmediatamente en modo de sólo lectura y se bloquea en modo de escritura.. El archivo se leerá cuando se ejecute [HTTPSendForm](../WDLang3/1000017190.md).

**`<Type of file>: (Cadena de caracteres)`**

description del tipo MIME del archivo. Este parámetro corresponde a la cabecera "Content-Type".

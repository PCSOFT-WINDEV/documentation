
## CloseSession( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sesión con Lotus Notes o Outlook.

**Recordatorio**: Una sesión se inicia con [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [OutlookStartSession](../WDLang3/3032123.md) o [NotasOpenConnection](../WDLang3/1000005102.md).

Una sesión que no finalice se cerrará automáticamente al final del programa WINDEV o de la sesión WEBDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a session 
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"email/julia.nsf", "C:\Lotus Notes\julia.id")
// If a session is started, close this session
IF SessionID <> 0 THEN
	CloseSession(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`CloseSession(<Session identifier>)`
---

**`<Session identifier>: (Integro)`**

Identificador de la sesión actual. Este valor es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [OutlookStartSession](../WDLang3/3032123.md) o [NotasOpenConnection](../WDLang3/1000005102.md).

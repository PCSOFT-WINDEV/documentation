
## KioskDisable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva el modo kiosco para la aplicación actual. Los botones del sistema se muestran de nuevo en la interfaz. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disable kiosk mode for the current application
KioskDisable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`KioskDisable()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000029"></a>

La función [InKioskMode](../WDLang1/1000024427.md) permite saber si la aplicación está en modo kiosco.




## Funciones SMS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para enviar y leer SMS fácilmente: 



|   |   |
| --- | --- |
| [SMSDelete](../WDLang3/3068008.md) | Borra un mensaje SMS de la memoria del Android Phone. Esta función ya no debe utilizarse. |
| [SMSFirst](../WDLang3/3068006.md) | Lee el primer mensaje SMS almacenado en el Android Phone. |
| [SMSNbMessage](../WDLang3/3068009.md) | Devuelve el número de mensajes SMS almacenados en un Android Phone. |
| [SMSNext](../WDLang3/3068007.md) | Lee el siguiente mensaje SMS almacenado en el Android Phone. |
| [SMSReset](../WDLang3/3068001.md) | Restablece todas las variables de [Estructura SMS](../WDLang3/3068003.md). |
| [SMSRunApp](../WDLang3/1000020864.md) | Inicia la aplicación nativa de envío de SMS que se encuentra en el dispositivo (Android o iPhone/iPad). |
| [SMSSend](../WDLang3/3068002.md) | Envía un SMS. |

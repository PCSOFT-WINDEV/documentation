<a name="Use"></a>
<a name="description"></a>
Envía un mensaje desde el servidor a todos los ordenadores cliente conectados.. El servidor es un servidor FTP (File Transfer Protocol) o un servidor RPC (Remote Procedure Call) alimentado por WINDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003056007&name=netclientsendmessage_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
NetClientSendMessage(ServerID, "Received", 0, 0)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`NetClientSendMessage(<Server identifier> , <Event> [, <Parameter 1> [, <Parameter 2>]])`
---

**`<Server identifier>: (Integro)`**

Identificador del servidor utilizado (devuelto por [NetStartServer](../WDLang3/3056013.md)).

**`<Event>: (Cadena de caracteres enteros o Ansi)`**

Identificador del Windows Event enviado a las ventanas de los ordenadores clientes (número de Windows Event o cadena de caracteres específicos).

**`<Parameter 1>: (Entero opcional)`**

Parámetro asociado a la Event.

**`<Parameter 2>: (Entero opcional)`**

Parámetro asociado a la Event.

### Recuperación del mensaje en los ordenadores clientes

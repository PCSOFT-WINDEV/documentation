
## InstantMessaging (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **InstantMessaging** permite definir las características avanzadas de la mensajería instantánea de un contacto (Google Talk, Skype, MSN, etc.). 

Las características de esta mensajería pueden ser definidas y modificadas por varias propiedades de WLanguage. 

Este tipo de Variable es utilizado por los [Contact](../WDLang3/1000020254.md) Variable s.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Propiedades específicas de la descripción de las variables InstantMessaging
<a name="propiedades_especificas_descripcion_las_variables_instantmessaging_ELTPARAGRAPHE000038"></a>

Se pueden utilizar las siguientes propiedades:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Address | Cadena de caracteres | Address de Contact en la mensajería instantánea correspondiente. |
| escribir a máquina | Constante de tipo Integer | Tipo de Address: <br><br>- *imOther*: otros tipos<br><br>- *imWork*: Address profesional de mensajería instantánea<br><br>- *imHome*: Address personal de mensajería instantánea<br><br>- *imCustom*: tipo personalizado que corresponde al valor de la propiedad **Label**.<br><br><br> |
| Etiqueta | Cadena de caracteres | Etiqueta para la mensajería personalizada (opcional). <br>Si la propiedad **Type** no corresponde a la constante *imCustom*, se ignorará el valor de la propiedad. |
| Protocolo | Constante de tipo Integer |   |


<a name="NOTE0_2"></a>


### Reinicio
<a name="reinicio_ELTPARAGRAPHE000106"></a>

Puede utilizar [VariableReset](../WDLang1/3013069.md) para restablecer el contenido de una variable de tipo **InstantMessaging**.

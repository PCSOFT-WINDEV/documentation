
## LDAPDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para desconectarse de un servidor LDAP específico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
LDAPDisconnect("MyLDAPSession")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPDisconnect(<LDAP session>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cierra la conexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP a interrumpir.


## geoGetAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera una dirección o una lista de direcciones correspondientes a la description de un lugar geográfico.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the address of a monument
MyAddress is Address 
MyAddress = geoGetAddress("Eiffel Tower")
IF ErrorOccurred = False THEN
	Info(MyAddress.Country)
END
```

<a name="SYNTAX1"></a>
<a name="SYNTAX2"></a>

### Recuperación de una lista de direcciones (modo asíncrono)

`geoGetAddress(<Description> , <WLanguage procedure> [, <Number of addresses>])`
---

**`<Description>: (Cadena de caracteres)`**

Ubicación cuyo Address debe ser recuperado. Este string puede corresponder a: 

- la ubicación especificada como ciudad y Country. Ejemplo: "París, Francia". 
	Observación: La ubicación debe especificarse en el idioma del teléfono. 

- la ubicación expresada por la posición ("&lt;latitud&gt;, &lt;longitud&gt;"). Ejemplo: "43.639942,3.837672".




**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLanguage Procedure ("callback") llamada para el lugar con la Address que debe ser recuperada. Para obtener más información sobre este procedimiento, consulte [Parámetros del procedimiento utilizado por la función geoGetAddress](../WDLang3/1000025033.md). 

**`<Number of addresses>: (Entero opcional)`**

Número máximo de direcciones devueltas en la lista. Si no se especifica este parámetro, el número máximo de direcciones será set a 10.

## Observaciones


### Gestión de varias direcciones (sintaxis 1)
<a name="gestion_varias_direcciones_sintaxis_1_ELTPARAGRAPHE000155"></a>

Si varias direcciones corresponden a la misma description: 




Para obtener la lista de direcciones que coinciden con la description, utilice la segunda o tercera sintaxis. Puede utilizar [OpenMobileWindow](../WDLang1/1000021018.md) o [OpenAsynchronous](../WDLang1/1410087804.md) para abrir una ventana de selección.

## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
Modbus es un protocolo de comunicación "de campo" (industrial y/o de entornos restringidos) creado en 1979. Este protocolo se utiliza ampliamente en entornos industriales para control programmable logic control lers (PLCs). Es conocido por su versatilidad, robustez, simplicidad y el gran número de dispositivos compatibles.

Una red Modbus se compone de al menos 1 maestro (o cliente) que pedirá a los esclavos (o servidores) que realicen una acción. En el Modbus, sólo hay dos acciones básicas, que tienen varias variantes: consulta para leer datos, consulta para escribir datos.

## Modbus y WINDEV
<a name="modbus_windev_ELTTEXTE000113"></a>
WINDEV soporta dos métodos de comunicación con esclavos Modbus (o servidores).

- **Modbus RTU** usando líneas serie (puerto serie, puerto USB convertido en puerto serie, etc.). Los esclavos que usan RS-232 pueden conectarse directamente al puerto serie del PC. Sin embargo, para RS-422 o RS-485, se requiere un convertidor (en casi todos los casos, los puertos seriales de las computadoras sólo soportan RS-232). Para conectarse en este modo, basta con utilizar [ModbusConnectRTU](../WDLang3/1000025929.md). 

- **Modbus TCP** mediante un enlace Ethernet. Para conectarse en este modo, basta con utilizar [ModbusConnectTCP](../WDLang3/1000025930.md). 




WLanguage ofrece funciones de lectura: 



|   |   |
| --- | --- |
| [ModbusReadBit](../WDLang3/1000025935.md) | Lee uno o más bits de un esclavo Modbus, empezando por una Address específica. |
| [ModbusReadFloat](../WDLang3/1000025936.md) | Lee uno o más flotadores de una sola precisión de un esclavo Modbus, a partir de una Address específica. |
| [ModbusReadRegister](../WDLang3/1000025937.md) | Lee uno o más registros de un esclavo Modbus, comenzando en una Address específica. |





y funciones de escritura: 



|   |   |
| --- | --- |
| [ModbusWriteBit](../WDLang3/1000025932.md) | Escribe uno o varios bits a una Address específica en un esclavo Modbus. |
| [ModbusWriteFloat](../WDLang3/1000025933.md) | Escribe uno o más flotadores de precisión simple a un esclavo Modbus, comenzando en una Address específica. |
| [ModbusWriteRegister](../WDLang3/1000025934.md) | Escribe uno o más registros (valores de 16 bits) a un esclavo Modbus, a partir de una Address específica. |





[ModbusDisconnect](../WDLang3/1000025931.md) se utiliza para cerrar la sesión. 
ATENCIÓN: Dado que las operaciones de escritura permiten a los PLCs control, se recomienda probar los diferentes comandos antes de la fase de producción para asegurarse de que los PLCs realizan las operaciones requeridas.

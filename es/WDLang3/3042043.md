


## grOrientación (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la orientación de los ejes del gráfico. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reverses the X-axis and the Y-axis
// Chart created through programming
grOrientation("MyChart", grYCoordinate, grAtLeft, grAtBottom)
```
<a name="Example2"></a>

```wl
// Reverses the X-axis and the Y-axis
// Chart control
grOrientation(CHART_MyChart, grYCoordinate, grAtLeft, grAtBottom)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grOrientation(<Chart name> [, <First axis> [, <Orientation of horizontal axis> [, <Orientation of vertical axis>]]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<First axis>: (Constante opcional)`**

Tipo de cotas a visualizar en el eje horizontal:


|   |   |
| --- | --- |
| *grXCoordinate*<br>(valor predeterminado) | Las coordenadas X se muestran en el eje horizontal. |
| *grYCoordinate* | Las coordenadas Y se muestran en el eje vertical. |



**`<Orientation of horizontal axis>: (Constante opcional)`**

Orientación del eje horizontal:


|   |   |
| --- | --- |
| *grAtLeft* | El eje horizontal está orientado a la izquierda (los valores negativos están a la derecha y los positivos a la izquierda). |
| *grAtRight*<br>(valor predeterminado) | El eje horizontal está orientado a la derecha (los valores negativos están a la izquierda y los positivos a la derecha). |



**`<Orientation of vertical axis>: (Constante opcional)`**

Orientación del eje vertical:


|   |   |
| --- | --- |
| *grAtBottom* | El eje vertical está orientado hacia abajo (los valores negativos están arriba y los positivos abajo). |
| *grAtTop*<br>(valor predeterminado) | El eje vertical está orientado hacia arriba (los valores negativos están en la parte inferior y los positivos en la parte superior). |

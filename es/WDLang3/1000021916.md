


## grSunburstPullOut (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Saca una sección (o todas las secciones) de una carta de Sunburst.. 

**Observación**: Cada sección de la carta puede ser extraída independientemente de las otras.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add data
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Montpellier", 320000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Paris", 789000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France", 1109000) 
grSunburstAddData(CHART_Chart, "Europe" + TAB + "Belgium", 900000)
grSunburstAddData(CHART_Chart, "Asia", 250000)
grDraw(CHART_Chart)
// Pull out a section by 20 pixels in "CHART_Chart"
grSunburstPullOut(CHART_Chart, "Europe" + TAB + "France" + TAB + "Montpellier", 20)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Sacar una sección

`grSunburstPullOut(<Chart name> , <Section path> , <Offset>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección a extraer en la tabla. Este parámetro tiene el siguiente formato:


```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```


**`<Offset>: (Integro)`**

Desplazamiento de la sección especificada (en píxeles).  


<a name="SYNTAX2"></a>

### Sacando todas las secciones

`grSunburstPullOut(<Chart name> , <Offset>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Offset>: (Integro)`**

Desplazamiento de diferentes secciones (en píxeles).

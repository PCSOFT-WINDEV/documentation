


## UPNPListDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los dispositivos UPNnP disponibles en la red. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
arrDevices is array of upnpDevice
arrDevices = UPNPListDevice("upnp:rootdevice", 51300)
IF ErrorOccurred THEN
	Info(ErrorInfo(errFullDetails))
ELSE
	InitDeviceList()
END
```
<a name="Example2"></a>

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UPNPListDevice(<Type> , <Port> [, <Address> [, <Timeout>]])`
---

**`<Result>: (Array de las variables de la upnpDevice)`**

Nombre de la variable array de [upnpDevice](../WDLang3/1000020903.md) que contiene la lista de dispositivos encontrados en la red. 

**`<Type>: (Cadena de caracteres)`**

Tipo de dispositivos a listar. 

**`<Port>: (Integro)`**

Número de puerto utilizado para la búsqueda.

**`<Address>: (Cadena de caracteres opcional)`**

Address en la que se crea el zócalo de búsqueda.

**`<Timeout>: (Entero opcional)`**

Tiempo máximo de espera expresado en segundos. Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 


Este parámetro corresponde a 5 segundos de Default.

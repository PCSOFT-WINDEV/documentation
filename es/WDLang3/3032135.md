


## EmailChangeFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la carpeta actual en el software de mensajería (Outlook, software de mensajería que utiliza el protocolo IMAP, ....).   

Esta carpeta se encuentra en la rama "Carpetas personales. 

Atención: Las "Carpetas genéricas" no son compatibles. 

**Recordatorio**: Esta función no se puede utilizar para modificar la carpeta actual del software de mensajería Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032135&name=emailchangefolder_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// IMAP protocol
IF EmailStartIMAPSession(EDT_USER, EDT_PASSWORD, ...
		"imap.mydomain.fr") = True THEN
	EmailChangeFolder(EDT_USER, "Personal folders\Inbox\Work") 
	...
ELSE
	Error("Unable to establish the connection")
END
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Start the session
SessionID = EmailStartOutlookSession("MyProfile")
...
// Change the current folder
EmailChangeFolder(SessionID, "Personal folders\Inbox\Work")
```

## Sintaxis

`<Result> = EmailChangeFolder(<Session> , <Folder path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, la ruta especificada no existe). Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Cadena de caracteres, entero, o emailOutlookSession o emailIMAPSession Variable)`**

Identificador del actual período de sesiones de Email. Este identificador puede ser:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un número entero devuelto por [EmailStartOutlookSession](../WDLang3/3032128.md) o [OutlookStartSession](../WDLang3/3032123.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [emailOutlookSession](../WDLang3/1000018767.md).

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).

- el nombre de usuario pasado a [EmailStartIMAPSession](../WDLang3/1000019189.md).




**`<Folder path>: (Cadena Unicode)`**

Ruta de la nueva carpeta actual.

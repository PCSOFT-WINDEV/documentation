
## upnpDevice (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **upnpDevice** permite definir las características avanzadas de un dispositivo UPnP. Las características del dispositivo uPnP pueden definirse y modificarse mediante diferentes propiedades del lenguaje WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDevice is upnpDevice 
MyDevice = UPNPSearchDevice(CST_UPNP_UUID_SONY_BRAVIA, 51300)
IF ErrorOccurred THEN
	ErrorInfo(errFullDetails)
ELSE
	Add(m_arrDevice, MyDevice)
END
```

### Propiedades específicas de la descripción de variables de tipo upnpDevice
<a name="propiedades_especificas_descripcion_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un dispositivo UPnP:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Código de producto | Cadena de caracteres | Código de producto (UPC). |
| denominar | Cadena de caracteres | Nombre del dispositivo. |
| escribir a máquina | Cadena de caracteres | URI que contiene el tipo de dispositivo. |
| Icon | Array de [upnpImage](../WDLang3/1000020905.md) | Lista de diferentes iconos disponibles. |
| ModeloDescripción | Cadena de caracteres | description de las características del dispositivo. |
| Nombre del modelo | Cadena de caracteres | Nombre del modelo. |
| Nombre del productor | Cadena de caracteres | Nombre del fabricante. |
| NombreUnico | Cadena de caracteres | Nombre único del dispositivo (UDN). |
| Número de modelo | Cadena de caracteres | Número de modelo. |
| Número de serie | Cadena de caracteres | Número de serie del dispositivo. |
| PresentaciónURL | Cadena de caracteres | URL de la página web que se puede usar para control el dispositivo. |
| ProductorURL | Cadena de caracteres | Enlace a la página web del fabricante. |
| Servicio | Array de [upnpService](../WDLang3/1000020908.md) | Lista de los diferentes servicios UPnP disponibles. |
| URL | Cadena de caracteres | URL que contiene el XML para el dispositivo description. |
| URL del modelo | Cadena de caracteres | URL de una Web Page que contiene la información específica del modelo. |

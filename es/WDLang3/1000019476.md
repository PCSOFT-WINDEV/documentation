


## Funciones de gestión telefónica
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el Phone:



|   |   |
| --- | --- |
| [AndroidActivityResultProcedure](../WDLang3/1410088881.md) | Permite especificar el procedimiento a llamar para obtener el resultado devuelto por una actividad de terceros. |
| [AppleAdvertisingIdentifier](../WDLang3/1410087936.md) | Devuelve el identificador de publicidad del dispositivo (solo iOS). |
| [AppleRequestTrackingAuthorizationAsynchronous](../WDLang3/1410087935.md) | Muestra la solicitud de Permission para el seguimiento de la actividad del usuario. |
| [AppleSignIn](../WDLang3/1000025404.md) | Permite realizar una conexión con la opción "Iniciar Sesión con Apple" en dispositivos con iOS 13 y posteriores. |
| [AppleTrackingAuthorizationStatus](../WDLang3/1410087934.md) | Permite saber si se concedió permiso a una aplicación para rastrear la actividad del usuario utilizando sus datos (solo para iOS). |
| [BrightnessAuto](../WDLang3/1000020182.md) | Devuelve o modifica el estado de activación del ajuste automatic del brillo del dispositivo. |
| [BrightnessGet](../WDLang3/1000019810.md) | Devuelve la configuración actual del brillo de la pantalla del dispositivo. |
| [BrightnessModify](../WDLang3/1000020170.md) | Modifica la configuración del brillo de la pantalla del dispositivo. |
| [ChangeTheme](../WDLang3/1000025266.md) | Permite cambiar mediante programación el tema (oscuro o claro) utilizado por la aplicación móvil. |
| [CheckUserIdentity](../WDLang3/1000021348.md) | Permite comprobar la identidad real del usuario actual a través de la comprobación biométrica. |
| [DeepLinkProcedure](../WDLang3/1410089051.md) | Define el procedimiento WLanguage que se ejecuta al hacer clic en un enlace profundo. |
| [GetTheme](../WDLang3/1000025320.md) | Permite conocer el tema actual de una aplicación móvil. |
| [iOSApplicationOpenURLProcedure](../WDLang3/1410089346.md) | Permite especificar el procedimiento al que se llamará cuando la aplicación necesite abrir un recurso identificado por una URL. |
| [LEDSwitchOff](../WDLang3/1000019447.md) | Apaga el LED (diodo emisor de luz) del aparato. |
| [LEDSwitchOn](../WDLang3/1000019446.md) | Enciende el LED (diodo emisor de luz) del aparato. |
| [MobileNetworkConnectionInfo](../WDLang3/1000020176.md) | Devuelve información sobre la conexión actual a los datos a través de la red móvil. |
| [MobileNetworkStatus](../WDLang3/1000020175.md) | Devuelve el estado actual de la conexión a los datos de la red móvil o solicita ser notificado cuando cambia el estado de la conexión a los datos. |
| [NotifAdd](../WDLang3/1000019442.md) | Añade una Notification en la barra de sistema del dispositivo (Android) o en el centro Notification (iOS). |
| [NotifDelete](../WDLang3/1000019444.md) | Borra una Notification que aparece en la barra de sistema del dispositivo. |
| [NotifDeleteCategory](../WDLang3/1000023980.md) | Elimina una categoría de Notification de la solicitud. |
| [NotifListCategory](../WDLang3/1000023979.md) | Enumera las categorías de Notification de la aplicación. |
| [NotifModify](../WDLang3/1000019443.md) | Modifica una Notification que aparece actualmente en la barra de sistema del dispositivo. |
| [NotifPushDisable](../WDLang3/1000020817.md) | Desactiva la gestión de las notificaciones push para una aplicación WINDEV Mobile (Android o IOS). |
| [NotifPushEnable](../WDLang3/1000020816.md) | Permite la gestión de las notificaciones push en una aplicación WINDEV Mobile (Android o iOS). |
| [NotifPushListInvalidIdentifiers](../WDLang3/1000020854.md) | Devuelve la lista de identificadores no válidos en el servicio APN de Apple.. |
| [NotifPushProcedure](../WDLang3/1000020818.md) | Especifica el procedimiento WLanguage llamado cuando una aplicación WINDEV Mobile (Android o iOS) recibe un push Notification. |
| [NotifPushSend](../WDLang3/1000020819.md) | Envía un push Notification a un dispositivo móvil (iOS o Android). |
| [PowerStatus](../WDLang3/3067007.md) | Utiliza para get diversas informaciones sobre la batería (principal o secundaria) del dispositivo en el que se ejecuta la aplicación. |
| [RecorderAction](../WDLang3/1000014001.md) | Ejecuta una acción en la grabadora o dictáfono que se encuentra en el dispositivo actual. |
| [ScreenSize](../WDLang3/1000022211.md) | Devuelve el tamaño (en pulgadas) de la diagonal de la pantalla del dispositivo donde se ejecuta la aplicación. |
| [ScreenType](../WDLang3/1000022210.md) | Devuelve la clase de pantalla en la que se ejecuta la aplicación.: Phone, tableta, computadora o reloj. |
| [SIPVisible](../WDLang3/3071013.md) | Muestra el teclado. |
| [tapiDialerCall](../WDLang3/1000019235.md) | Abre la aplicación de telefonía por defecto (marcador) que se encuentra en la Phone y marca el número especificado. |
| [tapiDialerDisplay](../WDLang3/1000019236.md) | Abre la aplicación de telefonía Default (marcador) y muestra el número especificado. No se realiza ninguna llamada. |
| [VibrationStop](../WDLang3/1000019449.md) | Detiene las vibraciones del dispositivo (incluso si han sido provocadas por otra aplicación). |
| [VibrationTrigger](../WDLang3/1000019448.md) | Dispara las vibraciones del dispositivo. |
| [VolumeGet](../WDLang3/1000019807.md) | Recupera el ajuste actual del volumen de una fuente de sonido. |
| [VolumeModify](../WDLang3/1000020169.md) | Modifica el ajuste del volumen de una fuente de sonido. |
| [VolumePhysicalButton](../WDLang3/1000020168.md) | Identifica o modifica la fuente de sonido asociada a los botones físicos utilizados para set el volumen del dispositivo. |

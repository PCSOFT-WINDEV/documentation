
## Recordatorio (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Reminder** se utiliza para describir y manipular un recordatorio para informar al usuario de un Appointment.

Este tipo de Variable es utilizado por el [Appointment](../WDLang1/1000019244.md) Variable s. 

Las características de Reminder pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a new reminder by notification an hour before the beginning of the appointment
reminderAPT is Reminder
reminderAPT.Type = reminderNotification
reminderAPT.Delay = 60
Add(apt.Reminder, reminderAPT)
AppointmentModify(apt)

// Adds a new reminder by email one week before the beginning of the appointment
reminderAPT is Reminder
reminderAPT.Type = reminderEmail
reminderAPT.Delay = 7 * 24 * 60 
Add(apt.Reminder, reminderAPT)
AppointmentModify(apt)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables Reminder
<a name="propiedades_especificas_description_las_variables_reminder_ELTPARAGRAPHE000043"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a Reminder:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| delay | Entero o Duración | Tiempo de espera en minutos antes del inicio de la Appointment para la que se debe realizar el Reminder. Esta propiedad puede corresponder a: <br><br>- un número entero correspondiente al número de minutos,<br><br>- una variable de tipo Duration,<br><br>- la duración en un formato legible (por ejemplo, 1 minuto). <br><br><br> |
| escribir a máquina | Constante de tipo Integer | El método que se utilizará para realizar la Reminder: <br><br>- *reminderDefault*: El método Default. Este método depende de la configuración del dispositivo. <br><br>- *reminderEmail*: enviar una Email. Este tipo de Reminder sólo está disponible si la cuenta asociada al Appointment Calendar soporta este tipo de Reminder y si esta cuenta está configurada.<br><br>- *reminderNotification*: enviar un Notification. <br><br>- *reminderSMS*: enviar a SMS. Este tipo de Reminder sólo está disponible si la cuenta asociada al Appointment Calendar soporta este tipo de Reminder y si esta cuenta está configurada.<br><br><br><br><br> |

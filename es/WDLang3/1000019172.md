## Descripción general del protocolo IMAP
<a name="descripcion_general_del_protocolo_imap_ELTTEXTE000172"></a>
<a name="NOTE1_1"></a>
**El protocolo IMAP** es un protocolo estándar para la gestión de correos electrónicos. A diferencia del protocolo POP3, el principio para utilizar el protocolo IMAP consiste en dejar los mensajes en el servidor de mensajería para poder leerlos desde varios clientes.. Incluye un indicador específico utilizado para averiguar si se ha leído un mensaje.. Varios servidores de mensajería soportan tanto el protocolo IMAP como el protocolo POP3.

## Uso del protocolo IMAP

<a name="NOTE2_1"></a>


### Recepción de mensajes utilizando el protocolo IMAP
<a name="recepcion_mensajes_utilizando_protocolo_imap_ELTPARAGRAPHE000048"></a>

Para recibir correos electrónicos utilizando el protocolo IMAP

1. Inicie una sesión IMAP utilizando un [emailIMAPSession](../WDLang3/1000018957.md) Variable y [EmailStartSession](../WDLang3/3032028.md). 
	**Ejemplo para iniciar una sesión usando un emailIMAPSession Variable:**
	
	```wl
	// Start the messaging session
	MySession is emailIMAPSession
	MySession.ServerAddress = "pop.mycompany.us"
	MySession.Name = "user"
	MySession.Password = "secret"
	EmailStartSession(MySession)
	```


2. Lea los mensajes en el servidor de mensajería:

	- con la función [EmailGetAll](../WDLang3/1000018727.md).

	- con un bucle como:
			**Ejemplo que está usando un correo electrónico Variable:**
			
		```wl
		MyMessage is Email
		EmailReadFirst(MySession,MyMessage)
		WHILE NOT MyMessage.Out
			// Place the process of the message read...
			...
			
			EmailReadNext(MySession, MyMessage)
		END
		```

			**Ejemplo que utiliza la estructura Email:**
			
		```wl
		EmailReadFirst(MySession)
		WHILE NOT Email.Out
			// Place the process of the message read...
			...
			
			EmailReadNext(MySession)
		END
		```


	- o con un bucle como (las palabras clave "NOT READ" son opcionales y le permiten navegar sólo por los mensajes no leídos o por todos los mensajes):
			
		```wl
		MyMessage is Email
		FOR EACH MyMessage NOT READ OF MySession
			// Place the process of the message read...
			...
		END
		```





3. Cierre la sesión IMAP con [EmailCloseSession](../WDLang3/3032006.md).

### Conexión con el proveedor de servicios
<a name="conexion_con_proveedor_servicios_ELTPARAGRAPHE000110"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Se necesita una conexión INTERNET para gestionar los correos electrónicos. Pueden ocurrir varios casos:

- El usuario utiliza un módem para conectarse a INTERNET: [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md) establece la conexión con INTERNET.

- El usuario utiliza una conexión directa a INTERNET (cable o ADSL): no se requiere ninguna operación específica.

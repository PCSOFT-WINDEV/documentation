
## tapiCapability( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las características de un dispositivo de telefonía. Le permite saber si este dispositivo es compatible con todos los tipos de llamadas entrantes, las llamadas de voz y los faxes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF BinaryAND(tapiCapability("LPT2"), tapiOptionMediaModeVoice) = tapiOptionMediaModeVoice THEN 
	Info("Voice supported")
END

IF BinaryAND(tapiCapability("LPT2"), tapiOptionMediaModeFax) = tapiOptionMediaModeFax THEN
	Info("Fax supported")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiCapability(<Device name>)`
---

**`<Result>: (Entero o constante de tipo integer)`**



- 0 si se produce un error, 

- una o más de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *tapiOptionMediaModeFax* | Dispositivo que acepta los faxes |
| *tapiOptionMediaModeUnknown* | Dispositivo que acepta las llamadas entrantes de tipo desconocido. |
| *tapiOptionMediaModeVoice* | Dispositivo que acepta las llamadas de voz.<br>Observación: Los módems de fax que permiten la marcación automática pueden devolver esta constante. |






Para identificar fácilmente el resultado, utilice [BinaryAND](../WDLang1/3050010.md).

**`<Device name>: (Cadena de caracteres)`**

Nombre del dispositivo a utilizar. La lista de dispositivos de telefonía es devuelta por [tapiDeviceList](../WDLang3/3076024.md).

## Observaciones


### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000179"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.

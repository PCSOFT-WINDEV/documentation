
## HTTPCertificate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el cliente Certificate usado por Default para identificarse en un servidor. Este Certificate es utilizado por las funciones SOAP y por las funciones HTTP de la siguiente manera. 

Durante la llamada a una función HTTP o a una función SOAP, se deben realizar los siguientes pasos si el sitio solicita una certificación de cliente: 

- **Si un Certificate es fijo**, se busca. Si existe el Certificate, se utiliza. 

- **Si un Certificate no es fijo o si el Certificate fijo fue rechazado por el sitio**, una ventana para la selección de Certificate le permite seleccionar el Certificate a utilizar. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CertifList is string 
Certif is string

CertifList = HTTPListCertificate()
Certif = ExtractString(ExtractString(CertifList, 1, CR), 3, TAB)

HTTPCertificate(Certif)
IF NOT HTTPRequest("https://testapubuntu.mycompany.com") THEN
	Error(ErrorInfo(errFullDetails))
ELSE
	Info("OK")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HTTPCertificate(<Certificate>)`
---

**`<Certificate>: (Cadena de caracteres)`**

Corresponde a: 

- el nombre de la Certificate para usar. Si se encuentran dos Certificate con el mismo nombre en la tienda de Certificate, uno de ellos será elegido al azar por el sistema.

- la huella del pulgar de la Certificate para usar. Sólo la huella del pulgar puede identificar una Certificate. 


El nombre de Certificate (y la huella digital) se puede recuperar con [Certificado HTTPList](../WDLang3/1000018890.md). 

Si este parámetro corresponde a una cadena vacía (""), el Certificate por defecto se reinicia a "&lt;None&gt;". 

**Atención**: **HTTPCertificate** no comprueba si existe el certificate.

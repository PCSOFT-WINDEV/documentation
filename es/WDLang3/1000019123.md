
## SaaSAdminDeleteUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un usuario de SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todos los usuarios de todos los clientes. Un administrador de cuentas de cliente sólo puede eliminar usuarios de su propia cuenta de cliente.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the usernamed Mark (from the Hybrid client account)
MyUser is saasUser

MyUser = SaaSAdminFindUser("mark@hybrid.com")

IF SaaSAdminDeleteUser(MyUser) = True THEN
	Info("User deleted")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminDeleteUser(<User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<User>: (saasUser Variable o cadena de caracteres)`**

Usuario SaaS (o inicio de sesión del usuario SaaS) para eliminar de la base de datos SaaS.

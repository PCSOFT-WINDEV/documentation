
## HTTPSimulateNetwork( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Simula una red lenta al limitar la velocidad de descarga y carga.. 

Observación: Esta simulación sólo se tiene en cuenta en el tipo [HTTPRequest](../WDLang3/1000021158.md) y cuando se utiliza [HTTPSend](../WDLang3/1000021183.md) y [HTTPSendForm](../WDLang3/1000017190.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPSimulateNetwork(1024, 1024)
res is httpResponse = HTTPSend("http://www.windev.com")
HTTPSimulateNetwork()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTTPSimulateNetwork([<Maximum Download Speed> [, <Maximum Upload Speed>]])`
---

**`<Maximum Download Speed>: (Entero opcional)`**



- Velocidad máxima a simular para la recepción de datos, expresada en kilobytes por segundo.. Esta velocidad se da sólo a título informativo. 

- 0 (valor de Default) significa que esta velocidad no está limitada.  




**`<Maximum Upload Speed>: (Entero opcional)`**



- Velocidad máxima a simular para el envío de datos, expresada en kilobytes por segundo.. Esta velocidad se da sólo a título informativo. 

- 0 (valor de Default) significa que esta velocidad no está limitada.

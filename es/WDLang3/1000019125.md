
## SaaSAdminListUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los usuarios de la base de datos de SaaS Webservice.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todos los usuarios de todos los clientes. Los gestores de cuentas de clientes sólo pueden listar usuarios en sus propias cuentas de clientes. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Count the number of users
arrUser is array of saasUser 
arrUser = SaaSAdminListUser()

Trace("The SaaS database contains" + arrUser.Count + "users")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminListUser([<Client account>])`
---

**`<Result>: (Array de las variables de la saasUser)`**

Enumera los usuarios registrados en la base de datos SaaS.

**`<Client account>: (saasClient Variable o cadena de caracteres opcional)`**

Cuenta de cliente a utilizar. Si se especifica este parámetro, sólo se listan los usuarios associated con esta cuenta de cliente, de lo contrario se listan todos los usuarios de la base de datos SaaS.

Este parámetro también puede ser una cadena de caracteres correspondiente al nombre o valor del [Referencia externa](../WDLang3/1000019072.md) Property para la cuenta de cliente deseada.


## Constantes NFC
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes constantes son utilizadas por las funciones NFC.

| Constante | Detalles | Función |
| --- | --- | --- |
| **nfcEnabled** | El sensor NFC está habilitado. | [NFCStatus](../WDLang3/1000020415.md) |
| **nfcDisabled** | El sensor NFC está desactivado. | [NFCStatus](../WDLang3/1000020415.md) |
| **nfcErrDisabled** | NFC no está activado en el dispositivo. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrCancellation** | La acción fue cancelada por el usuario. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrDetection** | La etiqueta NFC ha salido del control de detección durante la escritura de Process. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrEmulator** | Función llamada desde el emulador. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrFormat** | El formato de los datos a escribir no es válido. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrUnknown** | Error desconocido. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrUnavailable** | El sensor NFC no existe o no está disponible. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrReadOnly** | La etiqueta NFC está protegida en modo escritura. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrNDEF** | La etiqueta NFC no puede formatearse con el formato NDEF (NFC Data Exchange Format). | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrProtection** | La etiqueta NFC no puede protegerse en modo escritura (modo no soportado por la etiqueta). | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcErrMaxSize** | El tamaño de los datos a escribir excede el tamaño máximo de los datos que pueden almacenarse en la etiqueta NFC. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcUnknown** | Tipo de etiqueta:  tipo desconocido. Valor de Default si no se realizó ninguna operación de lectura con la etiqueta. | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcUnavailable** | El dispositivo no está equipado con un sensor NFC o éste no está disponible. | [NFCStatus](../WDLang3/1000020415.md) |
| **nfcMedia** | Tipo de datos: Media data. Puede corresponder a cualquier Resource identificado por un tipo de MIME (imagen, video, Document, etc.). | Variable de tipo [nfcData](../WDLang3/1000020406.md) |
| **nfcMifareClassic** | Tipo de etiqueta: NFC Mifare Classic | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcOK** | La operación se ha completado. | [NFCWriteTag](../WDLang3/1000020417.md) |
| **nfcText** | Tipo de datos: Datos de texto. | Variable de tipo [nfcData](../WDLang3/1000020406.md) |
| **nfcType1** | Tipo de etiqueta: Foro NFC Tipo 1 | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcType2** | Tipo de etiqueta: Foro NFC Tipo 2 | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcType3** | Tipo de etiqueta: Foro NFC Tipo 3 | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcType4** | Tipo de etiqueta: Foro NFC Tipo 4 | Variable de tipo [nfcTag](../WDLang3/1000020411.md) |
| **nfcURI** | Tipo de datos: Datos de URI (Uniform Resource Identifier). Puede corresponder a:<br><br>- la url de una página web (http://www.windev.com),<br><br>- un FTP Address (ftp://ftp.windev.com/),<br><br>- un número de Phone (tel:0123456789),<br><br>- un correo electrónico Address (mailto:adresse@mail.com), etc.<br><br><br> | Variable de tipo [nfcData](../WDLang3/1000020406.md) |
| **nfcEmpty** | Tipo de datos: Tipo de vacío | Variable de tipo [nfcData](../WDLang3/1000020406.md) |




## &lt;Tabla&gt;.SurfaceMesh (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la malla en una tabla de Superficie. 

**Observación**: La malla también puede activarse/desactivarse para un gráfico de superficie desde la ventana description del control Gráfico ("Mostrar una malla en la superficie" en la pestaña "General" del control description). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.SurfaceMesh(True, DarkBlue)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceMesh(<Mesh> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).

**`<Mesh>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar la malla. 

- <u><u><u><u>False</u></u></u></u> para desactivar la malla.




**`<Color>: (Entero o constante de tipo Integer)`**

Color de la malla. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).




## &lt;emailOutlookSession Variable&gt;.NbMensaje (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de mensajes entrantes en Outlook. Este número de mensajes evoluciona en función de los correos electrónicos recibidos o eliminados por [EmailDeleteMessage](../WDLang3/3032027.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyOutlookSession is EmailOutlookSession

nbMSG is int = MyOutlookSession.NbMessage()
FOR I = 1 TO nbMSG
	EmailReadMessage(MyOutlookSession, I)
	IF Email.Out <> True THEN
		CU.NAME = Email.Sender
		CU.MESSAGE_TXT = Email.Message
		...
	ELSE
		Info("This message was deleted")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Outlook session>.NbMessage()`
---

**`<Result>: (Integro)`**



- Número de mensajes en la bandeja de entrada de Outlook al momento de llamar a la función **&lt;emailOutlookSession Variable&gt;.NbMensaje**,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Outlook session>: (Variable de tipo emailOutlookSession)`**

Nombre de la [emailOutlookSession](../WDLang3/1000018767.md) Variable correspondiente a la sesión Email a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Mensajes eliminados (mensajería de Outlook)
<a name="mensajes_eliminados_mensajeria_outlook_ELTPARAGRAPHE000292"></a>

Si un mensaje fue borrado por [EmailDeleteMessage](../WDLang3/3032027.md) en la misma sesión de correo electrónico, **&lt;emailOutlookSession Variable&gt;.NbMensaje** devuelve el número real de mensajes en la carpeta de entrada.

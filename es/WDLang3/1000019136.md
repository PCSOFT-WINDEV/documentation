
## SaaSAdminDeleteSubscription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una suscripción SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is saasSite 
P is saasPricing 
C is saasClient 
Sub is saasSubscription

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")

// Find the subscription to delete
Sub = SaaSAdminFindSubscription(P, C)

// The Hybrid client does no longer subscribe to the Promo 2010 of Forum
SaaSAdminDeleteSubscription(Sub)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminación de una suscripción (mediante variables saasxxx)

`<Result> = SaaSAdminDeleteSubscription(<Subscription>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Subscription>: (saasSubscription Variable)`**

Nombre de la [saasSubscription](../WDLang3/1000019088.md) Variable correspondiente a la suscripción que se va a eliminar de la base de datos SaaS.


<a name="SYNTAX2"></a>

### Suprimir una suscripción identificada con el nombre de associated pricing

`<Result> = SaaSAdminDeleteSubscription(<Pricing> , <SaaS site> , <Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre del precio al que se ha suscrito el cliente.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS al que pertenece el precio.

**`<Client account>: (Cadena de caracteres)`**

Nombre de la cuenta del cliente (o reference externo).


## OBEXDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para desconectarse de un dispositivo que soporta el protocolo OBEX.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// OBEX disconnection
IF OBEXDisconnect(nObexID) = True THEN
	Info("Disconnection successful")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OBEXDisconnect(<Identifier of OBEX connection>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la desconexión correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Identifier of OBEX connection>: (Integro)`**

Identificador de la conexión OBEX para cerrar. Este identificador es devuelto por [OBEXConnect](../WDLang3/1000017153.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo las pilas de Microsoft bluetooth son soportadas por esta función. Para get la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

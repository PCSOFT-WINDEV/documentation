
## TaskDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina la tarea actual de la lista de tareas de un mensaje de Lotus Notes o Outlook.

**Atención**: Esta función **no se puede utilizar** para eliminar la tarea actual de la lista de tareas de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf",...
					"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Read the first task
TaskFirst(SessionID)
// Delete this task
TaskDelete(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TaskDelete(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

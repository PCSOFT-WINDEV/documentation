
## SIPVisible( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el teclado.






<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SIPVisible(<Visible/Invisible>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el teclado actual era visible antes de llamar a esta función,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Visible/Invisible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer visible el teclado actual,

- <u><u><u><u>False</u></u></u></u> para que el teclado actual sea Invisible.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### "Activar teclado al entrar datos"
<a name="activar_teclado_entrar_datos_ELTPARAGRAPHE000092"></a>

La opción "Activar teclado al entrar datos" (pestaña "Detalles" de la ventana control Campo de entrada description) convierte automáticamente el teclado actual en Visible cuando el control está en modo de edición.
<a name="NOTE0_3"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000107"></a>

**Atención**: La función **SIPVisible** no devuelve ningún código de error. Para saber si esta función ha generado un error, utilice la función [ErrorOccurred](../WDLang1/3087001.md) Variable. Si se produce un error, puedes get los detalles del mismo con [ErrorInfo](../WDLang1/3013008.md).

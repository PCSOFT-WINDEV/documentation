


## grHMAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a un gráfico de Heatmap.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grHMAddData(CHART_Chart1, 1, 1, 10)
grHMAddData(CHART_Chart1, 1, 2, 30)
grHMAddData(CHART_Chart1, 2, 1, 20.5)
grHMAddData(CHART_Chart1, 2, 2, 5)
```

```wl
// Define the colors
grHMColor(CHART_HMap, -50, DarkBlue)
grHMColor(CHART_HMap, +50, LightRed)
FOR i = 1 TO 500
	grHMAddData(CHART_HMap, Random(-100, 100), Random(-100, 100), -100)
	grHMAddData(CHART_HMap, Random(-100, 100), Random(-100, 100), 100)
END
grDraw(CHART_HMap)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grHMAddData(<Chart name> , <X> , <Y> , <Value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<X>: (Integro)`**

Posición de los datos en el eje X.

**`<Y>: (Integro)`**

Posición de los datos en el eje Y.

**`<Value>: (Entero o real)`**

Valor a agregar. Si ya se encuentra un valor en la posición (X, Y) especificada, el valor existente se sustituye por el nuevo valor. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los datos definidos con la función **grHMAddData** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

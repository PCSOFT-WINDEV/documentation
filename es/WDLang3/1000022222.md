


## &lt;btleDevice Variable&gt;.GetAService (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el servicio de un dispositivo Bluetooth de baja energía correspondiente al identificador especificado (UUID).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDevice is btleDevice
...
MyService is btleService = MyDevice.GetAService("ACC54-DAD4D5-DADD")
IF ErrorOccurred THEN
	Error("The service does not exist.")
	RETURN
END
...
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Device>.GetAService(<Identifier>)`
---

**`<Result>: (btleService Variable)`**



- [btleService](../WDLang3/1000022057.md) variable correspondiente al servicio encontrado. 

- Si ningún servicio corresponde con el identificador especificado, el valor de la variable [ErrorOccurred](../WDLang1/3087001.md) es <u><u><u><u>True</u></u></u></u>.




**`<Device>: (btleDevice Variable)`**

Nombre del [btleDevice](../WDLang3/1000022054.md) Variable que corresponde al dispositivo Bluetooth utilizado.

**`<Identifier>: (Cadena de caracteres)`**

Identificador (UUID) del servicio a recuperar. 

Atención: Si varios servicios corresponden al mismo identificador (UUID), sólo se recupera el primer servicio..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Varios servicios del mismo dispositivo Bluetooth pueden corresponder al mismo identificador (UUID). Para recuperar todos los servicios correspondientes a un mismo identificador, utilice [&lt;btleDevice Variable&gt;.GetAllServices](../WDLang3/1000022216.md).

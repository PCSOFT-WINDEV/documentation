
## SaaSAdminAddUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un usuario de SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todos los usuarios de todos los clientes. Los gestores de cuentas de clientes sólo pueden añadir usuarios a sus propias cuentas de clientes. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the client account
C is saasClient dynamic 
C = SaaSAdminFindClient("Hybrid")

// Add a new user
SAM is saasUser 
SAM.Login = "sam@hybrid.com"
SAM.Password = "secret"
SaaSAdminAddUser(SAM, C)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadiendo un usuario (a través de las variables saasxxx)

`<Result> = SaaSAdminAddUser(<User> , <Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<User>: (saasUser Variable)`**

Nombre de la variable [saasUser](../WDLang3/1000019083.md) correspondiente al description del nuevo usuario.

**`<Client account>: (saasClient Variable o cadena de caracteres)`**

Cuenta de cliente a la que se conectará el usuario.

Si este parámetro es una cadena de caracteres, puede representar el nombre o el [Referencia externa](../WDLang3/1000019072.md) Property de la cuenta del cliente.


<a name="SYNTAX2"></a>

### Añadir un usuario identificado por su nombre

`<Result> = SaaSAdminAddUser(<Username> , <Password> , <Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Username>: (Cadena de caracteres)`**

Nuevo nombre de usuario.

**`<Password>: (Cadena de caracteres)`**

Contraseña del nuevo usuario.

**`<Client account>: (Cadena de caracteres)`**

Nombre o reference externo de la cuenta del cliente a la que el usuario será relacionado.

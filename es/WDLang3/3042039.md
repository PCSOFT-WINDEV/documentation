


## grOrigin (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las marcas de inicio y final en uno de los ejes del gráfico.. 

**Observación**: La función **grOrigin** permite mostrar solo una parte del gráfico, reduciendo los valores que se muestran en uno de los ejes.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the start and end marks on the horizontal axis
// of "MyChart". The mark will go from 5 to 12.
grOrigin("MyChart", 5, 12, grXCoordinate)
grDraw("MyChart")
```
<a name="Example2"></a>

```wl
// Modify the start and end marks on the vertical axis of "CHART_Chart" control
// The mark will go from 500 to 1000
// Restore the default values

// Initialize the window
gnYMin is int
gnYMax is int

// Button for modifying the mark
gnYMin = CHART_Chart..YAxisMin
gnYMax = CHART_Chart..YAxisMax
grOrigin(CHART_Chart, 500, 1000, grYCoordinate)
grDraw(CHART_Chart)

// Button for restoring the mark
grOrigin(CHART_Chart, gnYMin, gnYMax, grYCoordinate)
grDraw(CHART_Chart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grOrigin(<Chart name> , <Minimum mark> [, <Maximum mark> [, <Type of axis>]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Minimum mark>: (Real)`**

Marca de inicio en el eje especificado.

Este parámetro también puede corresponder a una duración, una fecha, una hora o una fecha-hora.. 

**`<Maximum mark>: (Real opcional)`**

Marca final en el eje especificado. Si no se especifica este parámetro, la &lt;Marca de graduación máxima&gt; corresponde al valor máximo de los datos del gráfico.

Este parámetro también puede corresponder a una duración, una fecha, una hora o una fecha-hora.. 

**`<Type of axis>: (Constante opcional)`**

Eje para el que se deben modificar las marcas:



|   |   |
| --- | --- |
| *grSecondaryYCoordinate* | Eje Y secundario (eje vertical). |
| *grXCoordinate*<br>(valor predeterminado) | Eje X (eje horizontal). |
| *grYCoordinate* | Eje Y (eje vertical). |
| *grZCoordinate* | Eje Z (sólo tablas de superficie).<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |

<a name="NOTE0_1"></a>

## Observaciones


### Marcas en los ejes
<a name="marcas_los_ejes_ELTPARAGRAPHE000185"></a>

- Eje X (eje horizontal): los parámetros &lt;Marca de graduación mínima&gt; y &lt;Marca de graduación máxima&gt; corresponden al índice mínimo y máximo de los datos en las series mostradas.

- Eje Y (eje vertical): los parámetros &lt;Marca de graduación mínima&gt; y &lt;Marca de graduación máxima&gt; corresponden al valor mínimo y máximo de los datos.

- Límites: 100 marcas de Above y 100 marcas por debajo del eje X.

- Para modificar la frecuencia de las marcas en el eje horizontal o vertical de un gráfico, utilice [grInterval](../WDLang3/3042027.md).

- Los datos definidos con la función **grOrigin** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).



<a name="NOTE0_2"></a>


### Tipos de gráficos
<a name="tipos_graficos_ELTPARAGRAPHE000208"></a>

Las marcas en los ejes no tienen ningún efecto en los diagramas de tarta. 
<a name="NOTE0_3"></a>


### Restaurando los valores de Default
<a name="restaurando_los_valores_default_ELTPARAGRAPHE000217"></a>Para restaurar la visualización del gráfico antes de modificar el origen, se deben guardar los valores mínimo y máximo de los ejes.. Estos valores pueden conocerse a través de las siguientes propiedades: 



|   |   |
| --- | --- |
| [XAxisMax](../Proprietes/1000018169.md) | La propiedad **XAxisMax** obtiene el valor máximo del eje X en un control Gráfico. Esta propiedad se puede utilizar en el evento "Zoom" de un control Gráfico para identificar la sección mostrada. |
| [XAxisMin](../Proprietes/1000018168.md) | El **XAxisMin** Property obtiene el valor mínimo del eje X en un control Gráfico. Esta propiedad se puede utilizar en el evento "Zoom" de un control Gráfico para identificar la sección mostrada. |
| [YAxisMax](../Proprietes/1000018171.md) | El **YAxisMax** Property obtiene el valor máximo del eje Y en un control Gráfico. Esta propiedad se puede utilizar en el evento "Zoom" de un control Gráfico para identificar la sección mostrada. |
| [YAxisMin](../Proprietes/1000018170.md) | El **YAxisMin** Property obtiene el valor mínimo del eje Y en un control Gráfico. |
| [ZAxisMax](../Proprietes/1000020626.md) | El **ZAxisMax** Property obtiene el valor máximo del eje Z en un control Gráfico (gráfico de superficie). |
| [ZAxisMin](../Proprietes/1000020662.md) | El **ZAxisMin** Property obtiene el valor mínimo del eje Z en un control Gráfico (gráfico de superficie). |



![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: [grZoom](../WDLang3/1000021478.md) también puede utilizarse para ver una sección del gráfico. No obstante, el usuario final puede restaurar la configuración inicial a través del menú contextual del gráfico ("Zoom inicial").

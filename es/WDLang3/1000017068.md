
## SNMPCloseSession( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sesión SNMP.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SessionID is int
// Start a session
SessionID = SNMPStartSession("TestDoc", "public")
IF SessionID = -1 THEN Error("Error while opening the session " + ErrorInfo())
...
SNMPCloseSession(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SNMPCloseSession(<Identifier of SNMP session>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la sesión se cerró correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Identifier of SNMP session>: (Integro)`**

Identificador de la sesión SNMP a cerrar. Este identificador fue devuelto por [Sesión de inicio SNMPS](../WDLang3/1000017067.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se cierra la aplicación (o cuando se cierra la sesión de un sitio dinámico), las sesiones SNMP que aún están abiertas se cierran automáticamente.

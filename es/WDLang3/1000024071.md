


## &lt;Espesor de línea (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera y/o modifica el valor del grosor de la Line en un gráfico de Line.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control
// Modifies the line thickness for all the series of the chart
CHART_MyChart.LineThickness(grAllSeries,4)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperando el grosor de la Line (carta de Line)

`<Result> = <Chart control>.LineThickness(<Series number>)`
---

**`<Result>: (Integro)`**

Grosor de la línea.

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes). Esta tabla debe ser una tabla de Line.


**`<Series number>: (Integro)`**

Número de la serie para la que se solicita el espesor de Line.


<a name="SYNTAX2"></a>

### Modificando el grosor de la Line (carta de Line)

`<Chart control>.LineThickness(<Series number> , <New thickness>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes). Esta tabla debe ser una tabla de Line.


**`<Series number>: (Integro)`**



- Número de la serie para la que debe modificarse el espesor de Line. 

- Constante *grAllSeries* para cambiar el grosor de todas las series (incluidas las nuevas series).




**`<New thickness>: (Integro)`**

El nuevo grosor de la Line. Este parámetro puede corresponder a un valor superior a 0. El grosor de la línea es set a 1 de Default.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Unidades que se tienen en cuenta para el espesor
<a name="unidades_que_tienen_cuenta_para_espesor_ELTPARAGRAPHE000306"></a>

El espesor de la Line corresponde a valores mayores o iguales a 0.

- Si el grosor corresponde a 0, el grosor será de 1 píxel en la pantalla (lo que significa 1 pica al imprimir).

- Si el grosor es superior a 1, el grosor será de x píxeles en la pantalla, lo que significa x picas al imprimir.



<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000319"></a>

- Si se utiliza la función **&lt;Espesor de línea** en un gráfico diferente al gráfico de líneas, no se devuelve ningún error. La información se almacena y se utilizará cuando se modifique el tipo de gráfico (por [&lt;Gráfico&gt;.tipo](../WDLang3/1000024168.md) o con el menú contextual del gráfico).

- Equivalencia: la función **&lt;Espesor de línea** es equivalente a la función [Espesor](../Proprietes/2510144.md).

- Los datos definidos con la función **&lt;Espesor de línea** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).




## &lt;Mapa&gt;.ChangeStyle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el estilo del mapa que se muestra en el control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Apply "Aubergine" style to the map
MAP_MyMap.ChangeStyle(mcsAubergine)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.ChangeStyle(<Style> [, <Configuration file path>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudiera aplicar el nuevo estilo, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<Style>: (Integro)`**

Modifica el estilo aplicado al mapa que se muestra en la control Mapa. Este parámetro puede tomar uno de los siguientes valores:


|   |   |
| --- | --- |
| *mcsAubergine* | Estilo "berenjena".. |
| *mcsAutomatic* | El estilo se aplicará de acuerdo con el modo de luz u oscuridad de la aplicación:<br><br>- Tema claro: se utilizará el estilo "estándar",<br><br>- Tema oscuro: se usará el estilo "nocturno".<br><br><br>Si el tema de la aplicación cambia mientras se muestra la control Mapa, el estilo cambiará automáticamente. |
| *mcsCustom* | Estilo personalizado. en este caso, el &lt;fichero de configuración ruta&gt; debe ser especificado. |
| *mcsDark* | Estilo "oscuro". |
| *mcsNight* | Estilo "Noche". |
| *mcsRetro* | Estilo "Retro". |
| *mcsSilver* | Estilo "Plateado". |
| *mcsStandard* | Estilo "estándar". |



**`<Configuration file path>: (Cadena de caracteres opcional)`**

Ruta del archivo JSON que describe el estilo personalizado que se aplicará al mapa. Este archivo debe ser un archivo JSON válido codificado en UTF-8. Para obtener más información sobre la configuración de estilos personalizados, véase: [https://mapstyle.withgoogle.com/](https://mapstyle.withgoogle.com/)

Este parámetro se tendrá en cuenta sólo si el parámetro &lt;Estilo&gt; corresponde a la constante *mcsCustom*.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Por defecto, la control Mapa utiliza el estilo de la automatic.

- Para recuperar el estilo utilizado por el mapa que aparece en el control Mapa, utilice [&lt;Mapa&gt;.GetStyle](../WDLang3/1000025798.md).

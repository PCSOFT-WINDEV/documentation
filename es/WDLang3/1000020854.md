
## NotifPushListInvalidIdentifiers( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	 **Atención**: A partir del 31 de marzo de 2021, Apple dejará de dar soporte al protocolo binario heredado de APNs. Esta función no debe utilizarse después de esta fecha. 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de identificadores no válidos para el servicio APN de Apple.

**Observación**: Se recomienda llamar esta función una o dos veces al día (mediante una tarea programada, por ejemplo).. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCertificate is Certificate = CertificateLoad("applecertif.cer")

Let MyInvalidIdentifiers = NotifPushListInvalidIdentifiers(myCertificate, ...
		typeServiceIOSTest)
For I = 1 _TO_ MyInvalidIdentifiers.Count
	// Delete MyInvalidIdentifiers[i] from the server
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NotifPushListInvalidIdentifiers(<Certificate> , <Server type>)`
---

**`<Result>: (Array de los amortiguadores)`**

Nombre de la array de los búferes a los que se devuelve la lista de identificadores no válidos. 

**`<Certificate>: (Variable de tipo Certificate)`**

Nombre de la [Certificate](../WDLang1/1000019287.md) Variable correspondiente al parámetro de identificación de la aplicación en el servicio APN. 
Esta variable representa la Certificate (obtenida de Apple) para las notificaciones push de la aplicación. Esta Certificate puede haber sido cargada por [CertificadoCarga](../WDLang1/1000020851.md). 

**`<Server type>: (Constante de tipo Integer)`**

Tipo de servidor al que se enviará el Notification: 


|   |   |
| --- | --- |
| *typeServiceIOSRelease* | Servidor de lanzamiento de IOS (APN). |
| *typeServiceIOSTest* | Servidor de pruebas IOS (APN). |

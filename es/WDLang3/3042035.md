


## grMenú (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite elegir el tipo de menú contextual que muestra un gráfico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the popup menu in popup format of "MyChart", created through programming
grMenu("MyChart", grMenuForCompatibility)
```
<a name="Example2"></a>

```wl
// Enables the popup menu in window format for the CHART_MyChart control
grMenu(CHART_MyChart, grMenuWindow)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grMenu(<Chart name> , <Type of menu>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico del editor de ventanas.




**`<Type of menu>: (Constante de tipo Integer)`**

Tipo de menú a mostrar: 


|   |   |
| --- | --- |
| *grMenuForCompatibility* | Aparece un menú desplegable. Esta opción se utiliza para mostrar el menú de gráficos que se encuentran en las versiones anteriores a la versión 23..  <br><br>Observación: Esta constante corresponde al valor de True utilizado en las versiones anteriores. |
| *grMenuNone* | No se muestra ningún menú emergente. <br><br>Observación: Esta constante corresponde al valor de False utilizado en las versiones anteriores. |
| *grMenuWindow* | Aparece un menú emergente en formato de ventana. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para modificar el texto de las opciones del menú contextual conservado por compatibilidad, utilice [grMenuOpción](../WDLang3/3042042.md).

- Los datos definidos con la función **grMenú** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

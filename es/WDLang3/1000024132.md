


## &lt;Gráfico&gt;.SunburstAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a una carta de Sunburst.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart.SunburstAddData("Europe" + TAB + "France" + TAB + "Montpellier", 320000)
CHART_Chart.SunburstAddData("Europe" + TAB + "France" + TAB + "Paris", 789000)
CHART_Chart.SunburstAddData("Europe" + TAB + "France", 1109000) 
CHART_Chart.SunburstAddData("Europe" + TAB + "Belgium", 900000)
CHART_Chart.SunburstAddData("Asia", 250000)
CHART_Chart.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SunburstAddData(<Section path> , <Value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).



**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección que se añadirá al gráfico. Este parámetro tiene el siguiente formato:
 Esta ruta define la posición de los datos especificando sus diferentes niveles jerárquicos. 
Por ejemplo, si la ruta contiene "Europa + TAB + Francia + TAB + Montpellier", los datos se añadirán en el tercer nivel de la tabla debajo de las categorías "Europa\\Francia\\Montpellier".

**`<Value>: (Real)`**

Valor de la parte añadida. Este valor no debe ser negativo.

<a name="NOTE0_1"></a>

## Observaciones


### gráfico de proyección solar
<a name="grafico_proyeccion_solar_ELTPARAGRAPHE000225"></a>

- Si un nivel no contiene su propio valor, su valor se calculará sumando los valores de los subniveles.

- Si la suma de los datos para los subniveles de un nivel específico excede el valor de este nivel, se devolverá un error al dibujar el gráfico.

- La Color de los subniveles se calcula automáticamente: este Color tiene el mismo tono que el Color del nivel principal pero más pálido.



<a name="NOTE0_2"></a>


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000237"></a>

[.AddData](../WDLang3/1000023533.md) se utiliza para añadir datos en un gráfico (circular, de columnas, ...)

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [.StockAddData](../WDLang3/1000023565.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [&lt;Gráfico&gt;.HMAddData](../WDLang3/1000024079.md).

- Para los gráficos de dispersión, se recomienda utilizar [.ScatterAddDataXY](../WDLang3/1000024105.md).

- Para los gráficos de superficie, se recomienda utilizar [grSurfaceAddData](../WDLang3/1000024143.md).




Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [.IncreaseData](../WDLang3/1000024086.md).




## Funciones FTP estándar
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar el FTP:



|   |   |
| --- | --- |
| [FTPAttribute](../WDLang3/3040006.md) | Identifica los atributos de un archivo que se encuentra en un servidor FTP (File Transfer Protocol). |
| [FTPCommand](../WDLang3/3040017.md) | Envía un comando FTP específico a un servidor. |
| [FTPConfigure](../WDLang3/1000020947.md) | Modifica el comportamiento de las funciones FTP estándar. |
| [FTPConnect](../WDLang3/3040016.md) | Conecta el ordenador actual a un servidor FTP (File Transfer Protocol). |
| [FTPCurrentDir](../WDLang3/3040003.md) | Identifica o modifica el directorio actual en un servidor FTP (File Transfer Protocol). |
| [FTPDate](../WDLang3/3040015.md) | Devuelve las diferentes fechas de un archivo (fecha de creación, modificación o acceso) ubicado en un servidor FTP (File Transfer Protocol). |
| [FTPDateTime](../WDLang3/1000020946.md) | Devuelve las diferentes fechas y horas (creación, modificación o acceso) de associated con un archivo encontrado en un servidor FTP (File Transfer Protocol). |
| [FTPDeleteFile](../WDLang3/3040010.md) | Elimina un archivo de un servidor FTP (File Transfer Protocol). |
| [FTPDisconnect](../WDLang3/3040011.md) | Cierra la conexión entre el ordenador actual y el servidor FTP (File Transfer Protocol). |
| [FTPFileExist](../WDLang3/1000020948.md) | Comprueba la existencia de un archivo en un servidor FTP.. |
| [FTPGet](../WDLang3/3040002.md) | Transfiere un archivo o directorio desde un servidor FTP (File Transfer Protocol) al ordenador actual. |
| [FTPGetFile](../WDLang3/1000024597.md) | Transfiere un archivo de un servidor FTP (File Transfer Protocol) al equipo actual. |
| [FTPListFile](../WDLang3/3040018.md) | Enumera los archivos (y/o los directorios) que se encuentran en un directorio de un servidor FTP y devuelve el número de archivos (y/o directorios) listados. |
| [FTPMakeDir](../WDLang3/3040001.md) | Crea un directorio en un servidor FTP (File Transfer Protocol). |
| [FTPName](../WDLang3/3040019.md) | Devuelve el nombre del último archivo al que se ha accedido mediante una función FTP (File Transfer Protocol). |
| [FTPProxy](../WDLang3/1000019366.md) | Especifica si las funciones de comunicación que utilizan el protocolo FTP deben pasar por un proxy para ejecutar sus peticiones. |
| [FTPRemoveDir](../WDLang3/3040008.md) | Elimina un directorio y su contenido (archivos y subdirectorios) de un servidor FTP (File Transfer Protocol). |
| [FTPRenameFile](../WDLang3/3040005.md) | Renombra o mueve un archivo encontrado en un servidor FTP (File Transfer Protocol). |
| [FTPSend](../WDLang3/3040014.md) | Transfiere un archivo o directorio a un servidor FTP (File Transfer Protocol). |
| [FTPSize](../WDLang3/3040007.md) | Devuelve el tamaño (en bytes) de un archivo encontrado en un servidor FTP (File Transfer Protocol). |
| [FTPTime](../WDLang3/3040013.md) | Devuelve los diferentes tiempos (creación, modificación o acceso) de associated con un archivo encontrado en un servidor FTP (File Transfer Protocol). |

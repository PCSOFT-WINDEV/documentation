
## SaaSAdminIsConnected( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si se ha abierto una sesión de gestión SaaS.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SaaSAdminIsConnected() = False THEN
	Error("You must connect as administrator.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminIsConnected([<Required level>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> en los siguientes casos: 

	- El Administrador del SaaS está conectado y no se especifica ningún parámetro. 

	- El administrador SaaS está conectado y se especifica la constante *saasAdministrator*.

	- Se conecta un gestor de cuentas y se especifica la constante *saasAccountManager*.

	- Una u otra está conectada y se especifican ambas constantes. 




- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Required level>: (Constante opcional o combinación de constantes opcionales de tipo Integer)`**

Nivel necesario para la conexión: 


|   |   |
| --- | --- |
| *saasAccountManager* | Nivel requerido: Gerente de cuentas senior. |
| *saasAdministrator* <br>(valor predeterminado) | Nivel requerido: Administrador. |

Estas dos constantes se pueden combinar.

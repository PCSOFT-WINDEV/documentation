
## sInExitQueue( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el número de bytes en espera de ser transferidos en el búfer de salida del puerto serie o puerto paralelo especificado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The COM2 port is opened
// Send a message to the output buffer of COM2
sWrite(2, MyMessage)
// Wait for the end of the write operation
LOOP
	IF sInExitQueue(2) = 0 THEN BREAK
END
Info("End of write operation")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = sInExitQueue(<Port number>)`
---

**`<Result>: (Integro)`**

Número de bytes en espera de ser transferidos. 

**`<Port number>: (Integro)`**



- Número del puerto serie correspondiente: 1, 2, 3, ... 32 para COM1, COM2, COM3, ... COM32.

- Número del puerto paralelo correspondiente: -1, -2, -3 o -4 para LPT1, LPT2, LPT3 o LPT4.

- Número del puerto de infrarrojos correspondiente: 1, 2, 3, ... 32 para IR1, IR2, IR3, ... IR32.

- Número de puerto devuelto por [sOpen](../WDLang3/3062013.md) (si esta función fue llamada con un nombre de puerto).

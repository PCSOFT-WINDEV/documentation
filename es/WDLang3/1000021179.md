


## grSuperficieConseguirDatos (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera los datos de un gráfico de superficie. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display a data in the "CHART_MyChart" control. 
// This data is the X-coordinate of the first point.
Info(grSurfaceGetData(CHART_MyChart, 1, grXCoordinate))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = grSurfaceGetData(<Chart name> , <Index> , <Value to retrieve>)`
---

**`<Result>: (Real)`**

Valor recuperado.

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. 
Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md), 

- el nombre de la control Gráfico (en la ventana, página o editor de reportes).




**`<Index>: (Integro)`**

Índice de datos definido por el orden de adición en el gráfico. 
Si el dato correspondiente al índice especificado no existe, el valor recuperado es 0.

**`<Value to retrieve>: (Integro)`**

Indica el valor a recuperar:


|   |   |
| --- | --- |
| *grXCoordinate* | Coordenada X del punto a recuperar. |
| *grYCoordinate* | Coordenada Y del punto a recuperar. |
| *grZCoordinate* | Coordenada Z del punto a recuperar. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Recordatorio: Para los datos de una carta de superficie, el plano XY constituye la base de la superficie, mientras que Z indica la altura del punto.

- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [grStockGetData](../WDLang3/1000021174.md).

- Para recuperar los datos de un gráfico de dispersión, se recomienda utilizar [grScatterGetDataXY](../WDLang3/1000021175.md).

- Para recuperar un dato de un gráfico, utilice [grGetDatos](../WDLang3/1000021171.md).


## SaaSIsConnected( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define si la conexión con el servicio web SaaS todavía está establecida.. 

**Recordatorio**: La conexión con el servicio web SaaS fue establecida por [SaaSConnect](../WDLang3/1000018959.md). Una vez establecida la conexión, el Webservice SaaS prueba periódicamente la validez de la conexión llamando automáticamente a la función **SaaSIsConnected** cada 4 minutos.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SaaSIsConnected() = False THEN
	IF SaaSConnect("login", "password") = True THEN
		Info("Connection re-established")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSIsConnected()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la conexión sigue establecida,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000209"></a>
La estructura SMS es una estructura preestablecida de WLanguage (no se requiere declaración).. Esta estructura se utiliza para crear y leer un SMS.

**Observación**: Para poner a cero todas las variables de la estructura SMS, utilice [SMSReset](../WDLang3/3068001.md).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) La estructura SMS se utiliza para generar códigos QR (al imprimir un código de barras, por ejemplo).

## Las variables de la estructura del SMS
<a name="las_variables_estructura_del_sms_ELTTEXTE000233"></a>
La estructura contiene los siguientes miembros:


|   |   |
| --- | --- |
| **Adjunto** | Cadena de caracteres correspondiente a la ruta del archivo adjunto para asociar al mensaje. Si se especifica esta Variable, el mensaje enviado será un MMS. <br><br>**Atención**: Para ser enviado, el archivo adjunto debe estar ubicado en la memoria compartida del dispositivo (por ejemplo, la tarjeta SD). <br><br>**Observación**: Esta Variable sólo se utiliza cuando se envía un SMS con [SMSRunApp](../WDLang3/1000020864.md). Se ignora en todos los demás casos. <br><br> |
| **CountryPrefix** | Cadena de caracteres que contiene el prefijo internacional (33 de Default para Francia).<br>Si el número del destinatario empieza por "0" y si se especifica un prefijo nacional, "0" se sustituirá por este prefijo.<br>Si no se especifica el prefijo nacional, utilice un número en formato internacional. Por ejemplo, 1612345678.<br><br> |
| **mensaje** | Cadena de caracteres que contiene el mensaje saliente o el mensaje entrante (hasta 160 caracteres).<br><br> |
| **Number** | String de caracteres que contiene el número de destinatario o el número de remitente. |
| **NúmeroTipo** | Indica el tipo de número utilizado:<br><br>- ***smsInternationalNumber*** (valor de Default): se puede acceder a estos números desde cualquier lugar y están en formato 06.xx.xx.xx.xx.xx<br><br>- ***smsNationalNumber***: números cortos, accesibles dentro de la Country.<br><br><br><br><br> |
| **ReceiveDate** | Fecha y hora en que se recibió el SMS.<br><br> |
| **Reintentar** | Booleano (<u><u><u><u>True</u></u></u></u> predeterminado)<br>Indica si el mensaje debe enviarse regularmente si no se recibe.<br><br> |
| **Subscript** | Número entero correspondiente al índice del SMS entrante.<br><br> |

## Diferencias entre el modo GO y el tiempo de ejecución
<a name="diferencias_entre_modo_tiempo_ejecucion_ELTTEXTE000257"></a>
En **GO mode** (simulación en el ordenador de desarrollo), se produce un error de WLanguage cuando se utiliza un Variable de la estructura del SMS.

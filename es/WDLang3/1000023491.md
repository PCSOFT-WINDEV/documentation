


## grScatter3DMarkCaption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para especificar el título de la marca para un gráfico de dispersión 3D. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grScatter3DMarkCaption(CHART_MyChart, grXCoordinate, 1, "January")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grScatter3DMarkCaption(<Chart name> , <Mark axis> , <Value> , <Caption>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico de dispersión 3D a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas).




**`<Mark axis>: (Constante de tipo Integer)`**

Tipo de eje en el que se visualizarán las marcas:


|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |
| *grZCoordinate* | Eje Z. |



**`<Value>: (Real)`**

Valor de la marca para la que se modificará la leyenda.  

**`<Caption>: (Cadena de caracteres)`**

Nueva leyenda para la marca. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de marcas
<a name="gestion_marcas_ELTPARAGRAPHE000080"></a>

Para que se muestre el título de la marca: 

- debe existir una marca con el valor especificado. De lo contrario, no se mostrará el título de la marca.

- la marca debe tener un valor. El título no se mostrará si una marca no tiene valor.




**Recordatorio**: Las marcas de una gráfica de dispersión 3D se pueden definir: 

- en el editor, en la ventana control Gráfico description (pestaña "General"). 

- mediante programación con la función [grInterval](../WDLang3/3042027.md).

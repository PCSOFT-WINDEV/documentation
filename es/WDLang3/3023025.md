## Entorno de desarrollo móvil
<a name="entorno_desarrollo_movil_ELTTEXTE000104"></a>
WINDEV Mobile es un entorno de desarrollo para PC que le permite desarrollar: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) aplicaciones en el modo de aplicación universal de Windows 10.




**Observaciones**: 

- Las aplicaciones generadas funcionan únicamente en dispositivos móviles (equipados con procesadores compatibles con ARM, XScale, Samsung y ARM).. Estas aplicaciones no funcionan en el PC.

- A partir de la versión 27, ya no es posible desarrollar aplicaciones para Windows Mobile. Windows Mobile, Windows CE y Windows Embedded ya no reciben soporte de Microsoft, pero se siguen utilizando en el sector industrial. 
	Si desea seguir desarrollando para Windows Mobile, debe utilizar la versión 26. La versión 28 le da acceso a la versión 26, que se puede descargar [aquí](http://windev.com/dl.htm).

## Funciones de WLanguage
<a name="funciones_wlanguage_ELTTEXTE000137"></a>
La mayoría de funciones WLanguage disponibles en WINDEV también están disponibles en WINDEV Mobile. Estas funciones son comunes a ambos entornos de desarrollo.

Sólo las funciones específicas de Windows no están disponibles en WINDEV Mobile.

## Base de datos
<a name="base_datos_ELTTEXTE000161"></a>
Los datos manipulados por una aplicación WINDEV Mobile pueden provenir de:

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) una base de datos HFSQL Classic o HFSQL Client/Server.

- archivos ".INI".

- archivos de texto (ANSI o UNICODE).

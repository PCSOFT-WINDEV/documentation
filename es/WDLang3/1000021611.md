


## MapConfigureGglAPI (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Declara parámetros adicionales para el control Map que serán transmitidos a el Google API. Esta función se utiliza principalmente para manipular las bibliotecas adicionales de funciones, no disponibles en el control Mapa por defecto. Estas funciones estarán disponibles en el código del navegador en el objeto JavaScript nativo devuelto por [MapGetJSObject](../WDLang3/1000021474.md).
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Atención: Esta función debe utilizarse en el código de inicialización del proyecto.. La función no tendrá ningún efecto si se utiliza en otra Process.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MapConfigureGglAPI("libraries=geometry,places","language=fr")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapConfigureGglAPI(<Parameter 1> [, <Parameter 2> [... [, <Parameter N>]]])`
---

**`<Parameter 1>: (Cadena de caracteres)`**

Primer parámetro transmitido a la API. El parámetro tiene el siguiente formato: Nombre=Valor.

**`<Parameter 2>: (Cadena de caracteres opcional)`**

Segundo parámetro opcional transmitido a la API. El parámetro tiene el siguiente formato: Nombre=Valor.



**`<Parameter N>: (Cadena de caracteres opcional)`**

Último parámetro opcional transmitido a la API. El parámetro tiene el siguiente formato: Nombre=Valor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La configuración es global al proyecto. Cada llamada a la función reemplaza los parámetros definidos de antemano.

- Si se llama a la función sin parámetro, los parámetros definidos de antemano se borran y no se transmite ningún parámetro al API.

- El control Mapa puede no funcionar si el valor transmitido no es válido.

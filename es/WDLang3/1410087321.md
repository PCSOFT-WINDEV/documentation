
## DownloadRemove( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela una descarga eliminándola del gestor de descargas, y borra los archivos existentes.

Observación: Las funciones de descarga no tendrán efecto en el simulador.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Remove the download identified by nId
DownloadRemove(nId)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DownloadRemove(<Identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el archivo, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Identifier>: (Integro)`**

Identificador de la descarga (devuelto por [DownloadAdd](../WDLang3/1410087319.md)).

<a name="NOTE0_1"></a>

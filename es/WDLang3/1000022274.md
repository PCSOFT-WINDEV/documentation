


## &lt;emailOutlookSession Variable&gt;.AddFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>


Agrega una carpeta al software de mensajería de Outlook. Esta carpeta se crea en la rama "Carpetas personales".

**Recordatorio**: Esta función no se puede utilizar para añadir una carpeta al software de mensajería Outlook Express.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the Outlook session
SessionID is EmailOutlookSession
...
// Create a folder in the current folder 
SessionID.AddFolder(SessionID.CurrentFolder() + "\Work")
// Create a folder in the custom folder
SessionID.AddFolder("Personal folders\Inbox\MyOwnWork")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.AddFolder(<Folder path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailOutlookSession)`**

Nombre de la Variable de tipo [emailOutlookSession](../WDLang3/1000018767.md) que corresponde a la sesión actual de Email.

**`<Folder path>: (Cadena Unicode)`**

Ruta de la carpeta a crear. Esta ruta debe comenzar necesariamente con "Carpetas personales".

Si esta ruta contiene varias ramas que no existen, se crean todas las ramas.

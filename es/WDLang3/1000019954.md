
## geoRunApp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la aplicación nativa de mapas del dispositivo para mostrar: 

- una ubicación geográfica específica,

- una ruta entre dos lugares.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts the maps application to display the route
// between the current location of the device and another location
MyPosition is geoPosition = GPSGetPosition()
MyAddress is Address = geoGetAddress("Eiffel Tower, Paris, France")
IF ErrorOccurred = False THEN
	geoRunApp(myPosition, MyAddress.Position)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Mostrando una ubicación geográfica

`<Result> = geoRunApp(<Position> [, <Zoom> [, <Display mode>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudo iniciar la aplicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Position>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable correspondiente a la ubicación geográfica (latitud y longitud) que se desea mostrar en la aplicación de mapas.

**`<Zoom>: (Entero opcional)`**

Nivel de zoom que debe aplicarse al mapa mostrado en la aplicación de mapas. Este valor debe estar comprendido entre 1 (se visualiza toda la Tierra) y 23. Si no se especifica este parámetro, se aplicará el zoom máximo.

**`<Display mode>: (Constante opcional de tipo Integer)`**

Modo de visualización de la ubicación en la aplicación: 


|   |   |
| --- | --- |
| *geoMap2D* <br>(valor predeterminado) | La ubicación se muestra en un mapa bidimensional. |
| *geoView360* | La ubicación se muestra como una Image con una vista de 360°. En este modo, la ubicación especificada debe corresponder a un lugar que figure en el StreetView de Google. |




<a name="SYNTAX2"></a>

### Visualización de una ruta entre dos lugares

`<Result> = geoRunApp(<Start location> , <End location>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudo iniciar la aplicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Start location>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable correspondiente a la ubicación geográfica (latitud y longitud) del punto de inicio de la ruta.

**`<End location>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable correspondiente a la ubicación geográfica (latitud y longitud) del punto final de la ruta.


<a name="SYNTAX3"></a>

### Visualización de una Marker en la aplicación de mapas

`<Result> = geoRunApp(<Position> , <Marker text>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudo iniciar la aplicación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Position>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable correspondiente a la ubicación geográfica (latitud y longitud) de la Marker.

**`<Marker text>: (Cadena de caracteres)`**

Texto para asociar a la Marker

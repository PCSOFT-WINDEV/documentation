
## &lt;Mapa&gt;.AddImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva Image al mapa que se muestra en una control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an image to the Map control
ImageOnMap is MapImage
ImageOnMap.Position = MyPosition
ImageOnMap.Image = "image.png"
ImageOnMap.ActionClick = ClickProc
MAP_MyMap.AddImage(ImageOnMap)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.AddImage(<Image>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudiera añadir la Image,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<Image>: (MapImage Variable)`**

Nombre de la [MapImage](../WDLang3/1000025523.md) Variable que corresponde a la Image a añadir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la Image ya ha sido añadida a la control Mapa o si es inválida (por ejemplo.: ancho no especificado), se mostrará un error fatal. 

- Para borrar una Image dibujada en una control Mapa, utilice [&lt;Mapa&gt;.BorrarImagen](../WDLang3/1000025829.md). 

- Para modificar una Image dibujada en una control Mapa, utilice [&lt;Mapa&gt;.ModificarImagen](../WDLang3/1000025818.md).

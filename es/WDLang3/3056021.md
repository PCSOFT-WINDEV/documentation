
## NetCloseRemoteAccess( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la Line previamente abierta por [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AccessReport is int = NetOpenRemoteAccess("My connection")
IF AccessReport = 0 THEN
	// Process
	...
	NetCloseRemoteAccess()
ELSE
	Error("Error during the connection", NetMsgError(AccessReport))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetCloseRemoteAccess([<Access name>])`
---

**`<Result>: (Integro)`**



- 0 si se ha realizado el cierre,

- un código de error de lo contrario. Este código de error puede utilizarse con [NetMsgError](../WDLang3/3056019.md) para get los detalles del error y el mensaje correspondiente.




**`<Access name>: (Cadena de caracteres opcional)`**

Nombre de la conexión "Acceso remoto a la red" definida en Windows. La lista de accesos remotos instalados en un ordenador es devuelta por [NetListRemoteAcceso remoto](../WDLang3/3056027.md).

Si se especifica este parámetro, el acceso remoto se cierra, independientemente de su modo de apertura (por [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md) o no).

Si no se especifica este parámetro, se cerrará la última conexión abierta por [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md) con &lt;Cierre de automatic&gt; establecida en <u><u><u><u>True </u></u></u></u>.




## grTimeLevelMax (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija el número de niveles para mostrar en la escala temporal de un gráfico, así como el nivel máximo.

Observación: Esta función puede utilizarse con [grTimeLevelMask](../WDLang3/1000023017.md), que define la máscara de un nivel en la escala temporal del gráfico.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The maximum number ol fevels to display is set to 2
grTimeLevelMax(CHART_MyChart, 2)

// Forbids the display of years for CHART_MyChart
grTimeLevelMax(CHART_MyChart, 3, grtlMonth)
```

```wl
// Set the time scale to display several values per month
// but displaying only the months in the scale 
// (days are not displayed)
grTimeLevelMask(CHART_MyChart, grtlMonth, "Mmm")
grTimeLevelMax(CHART_MyChart, 1, grtlMonth)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grTimeLevelMax(<Chart name> , <Number of levels> [, <Maximum level>])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en el editor de ventanas o páginas.




**`<Number of levels>: (Integro)`**

Número máximo de niveles mostrados en el gráfico. 

**`<Maximum level>: (Constante opcional de tipo Integer)`**

Nivel máximo a visualizar: 


|   |   |
| --- | --- |
| *grtlDay* | Nivel máximo visualizado: días. |
| *grtlHour* | Nivel máximo visualizado: horas. |
| *grtlMillisecond* | Nivel máximo visualizado: milisegundos. |
| *grtlMinute* | Nivel máximo visualizado: minutos. |
| *grtlMonth* | Nivel máximo visualizado: meses. |
| *grtlSecond* | Nivel máximo visualizado: segundos. |
| *grtlYear* | Nivel máximo visualizado: años. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se muestra una escala de tiempo en los datos de DateTime, el diseñador de gráficos dibuja varios niveles correspondientes a las diferentes unidades de tiempo visualizadas. 

- El nivel mínimo a visualizar se define en función del rango de datos.

- El nivel máximo se define utilizando dos parámetros:

	- El número máximo de niveles a mostrar en la carta (por Default: 3). 

	- El nivel máximo permitido para la visualización (por Default: año).





La función **grTimeLevelMax** permite modificar estos parámetros. Esto le permite no overload la carta o no mostrar algunos niveles sin relevancia para los datos mostrados.

Por ejemplo: 

- para evitar mostrar los años, todo lo que tienes que hacer es set el nivel máximo con la constante *grtlMonth*.

- para evitar mostrar más de dos niveles de tiempo en la carta, todo lo que tienes que hacer es set el parámetro &lt;Número de niveles&gt; a 2.


## SocketCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un zócalo. Un socket es un recurso utilizado por las aplicaciones para comunicarse de un equipo a otro, independientemente del tipo de red.

**Observaciones**:

- Esta función debe llamarse en el servidor utilizado para la comunicación.. Si el ordenador cliente utiliza una aplicación WINDEV para conectarse a este socket, esta aplicación tendrá que utilizar [SocketConnect](../WDLang3/3070016.md).

- **SSL sockets**: Para crear un socket SSL, utilice [SocketCreateSSL](../WDLang3/1000017041.md).

- **UDP sockets**: Para crear un socket UDP, utilice [SocketCreateUDP](../WDLang3/3070019.md).

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003070011&name=socketcreate_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
IF SocketCreate("Server", 8000) = False THEN
	Error("creation error" + ErrorInfo(errMessage))
END
```

## Sintaxis

`<Result> = SocketCreate(<Socket name> , <Port number> [, <Address>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).




**`<Socket name>: (Cadena de caracteres)`**

Nombre que se le dará al zócalo. Este nombre será utilizado por todas las funciones de los zócalos. Atención: el nombre del socket distingue entre mayúsculas y minúsculas.

**`<Port number>: (Integro)`**

Número de puerto que se le asignará al zócalo. Este número debe incluirse entre 1 y 65000. Este número se especificará cuando los ordenadores cliente estén conectados al servidor..
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Debe ser "root" para crear un socket en un puerto menor o igual a 1024 (ser "root" significa ser el usuario que tiene todos los permisos del sistema).. 

**`<Address>: (Cadena de caracteres opcional)`**

IP Address en el que el enchufe debe ser creado en el ordenador actual. Si el ordenador actual es associated con varios IP Address es (o con varias tarjetas de red), este parámetro le permite elegir el IP Address que será associated con el zócalo.

Esta IP Address puede tener el siguiente formato:

- IP Address en formato XXX.XXX.XXX.XXX (125.5.110.100 por ejemplo).

- IP Address que contiene el nombre del servidor (www.windev.com por ejemplo). Se recomienda esta sintaxis.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El IP Address también es devuelto por [NetIPAddress](../WDLang3/3056023.md).

## Observaciones


### Cómo recrear un zócalo
<a name="como_recrear_zocalo_ELTPARAGRAPHE000109"></a>

Si [SocketClose](../WDLang3/3070015.md) no cierra correctamente un socket (reiniciando el equipo, por ejemplo), la recreación del socket puede tardar bastante tiempo. De hecho, tendrá que esperar hasta que el sistema libere el zócalo.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Reutilizar un Address que ya se utiliza
<a name="reutilizar_address_que_utiliza_ELTPARAGRAPHE000121"></a>

Por defecto, **SocketCreate** permite reutilizar un Address ya utilizado (por la propia aplicación o por otra aplicación). Para prohibir este modo de funcionamiento, escriba la siguiente Line de código:


```wl
Socket.Option = SocketNoReuseAddress
```


**Observación**: Por lo tanto, esta opción le permite saber si ya se utiliza un Address o un puerto. En este caso, la función **SocketCreate** devuelve un error.
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Conexión a un puerto de infrarrojos
<a name="conexion_puerto_infrarrojos_ELTPARAGRAPHE000138"></a>

Para conectar a una toma que utilice un puerto de infrarrojos:

1. Cree un socket utilizando el puerto de infrarrojos del servidor ([SocketCreateInfrarrojo](../WDLang3/3070012.md) en una aplicación WINDEV).

2. En la aplicación cliente (aplicación WINDEV), utilice [SocketConnectInfrarrojo](../WDLang3/3070018.md) para conectarse a este socket.

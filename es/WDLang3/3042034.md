


## grMask (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Formatea los datos mostrados por [grLabel](../WDLang3/3042046.md) o por [grTooltip](../WDLang3/3042010.md)



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grMask(CHART_MyChart, grMaskPercent, "99.99%")
grLabel(CHART_MyChart, grShowPercent, True)
grDraw(CHART_MyChart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grMask(<Chart name> , <Type of mask> , <Mask value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Type of mask>: (Constante numérica)`**

Máscara usada:


|   |   |
| --- | --- |
| *grMaskAxis* | La máscara especificada se utiliza para formatear los valores mostrados en los ejes del gráfico..<br>**Ejemplo**: Muestra el valor en la moneda actual.: el valor de la máscara es "+99,999,999,999,999.999.99 $". |
| *grMaskPercent* | La máscara especificada permite definir el formato de los porcentajes que se muestran en el gráfico (función grLabel(&lt;Nombre del gráfico&gt;, grShowPercent, True))<br>**Atención**: el símbolo % debe encontrarse en &lt;Mask Value&gt;..<br>**Ejemplo**: "99,99%" muestra el porcentaje con 2 dígitos después del punto decimal. |
| *grMaskTimeAxis* | La máscara especificada se utiliza para formatear las fechas u horas que se muestran en los ejes del gráfico..<br>**Ejemplo**: Muestra la fecha en formato "MM/DD/AAAA: el valor de la máscara es "MM/DD/YYYYY"..<br><br><pre><code>grMask("MyChart", grMaskTimeAxis, "MM/DD/YYYY")</code></pre><br> Esta constante es soportada por los tipos de gráficos que pueden incluir un eje temporal: Columna, Columna horizontal, Line y Área.<br> |
| *grMaskValue* | La máscara especificada permite definir el formato de los valores que se muestran en el gráfico (función grLabel(&lt;Nombre del gráfico&gt;, grShowValue, True))<br>**Ejemplo**: Muestra el valor en la moneda actual.: el valor de la máscara es "+99,999,999,999,999.999.99 $". |



**`<Mask value>: (Cadena de caracteres)`**

description de la máscara a utilizar.

- Para la constante *grMaskValue*: Vaciar la cadena ("", valor de Default) para desactivar la máscara de visualización.

- Para la constante *grMaskPercent* : "99%": Valor de Default.

- Para la constante *grMaskTimeAxis*: 
	

	- Si el valor a visualizar es una fecha, debe estar en formato "AAAAMMDD".. Las máscaras admitidas son idénticas a las utilizadas por [DateToString](../WDLang1/3027025.md).

	- Si el valor a visualizar es una hora, debe estar en formato "HHMMSSCC".. Las máscaras admitidas son idénticas a las utilizadas por [TimeToString](../WDLang1/3027023.md).





Las máscaras se definen mediante caracteres: 9", ".", ", ",", "+", "-" y espacios en blanco. Por ejemplo: "99.999,99". Para las máscaras de moneda, utilice hasta 17 dígitos significativos para la parte entera y 6 para la parte decimal.. Por ejemplo: "99 999 999 999 999 999,999999".

**Observaciones**:

- Para que el número se llene de ceros a la izquierda, debe utilizarse el carácter "0" a la izquierda de la máscara.. Por ejemplo: "099.999,99".

- Para forzar los signos, utilice el carácter "+" a la izquierda. Por ejemplo: "+9999".  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función sólo afecta a los gráficos que muestran valores o porcentajes.

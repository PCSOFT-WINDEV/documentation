


## grCreate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un tipo específico de gráfico.

**Observaciones**:

- Se recomienda crear un control Gráfico directamente en lugar de usar **grCreate**. De hecho, la control Gráfico ofrece muchas más características.

- Esta función sólo puede utilizarse para crear un gráfico mediante programación. Esta función no puede utilizarse en el editor de ventanas, el editor Page o editor de reportes.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a column chart 
// (and save the parameters modified by the user)
grCreate("MyChart", grColumn)
```

## Observaciones


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000397"></a>

Para crear y dibujar un gráfico:

- cree el gráfico (función **grCreate**), 

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- definir el destino de la carta con las siguientes funciones: 

	- [grDestinoBMP](../WDLang3/3042018.md), 

	- [grDestinoControl](../WDLang3/3042020.md), 

	- [grDestinoEMF](../WDLang3/3042017.md), 

	- [grDestinoWnd](../WDLang3/3042031.md),

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinoWMF](../WDLang3/3042019.md). 




- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).



<a name="NOTE0_2"></a>


### Dibujar varios gráficos al mismo tiempo
<a name="dibujar_varios_graficos_mismo_tiempo_ELTPARAGRAPHE000444"></a>

No pueden existir varios gráficos con el mismo nombre al mismo tiempo.

Para dibujar varios gráficos al mismo tiempo (en una o más ventanas, por ejemplo), se deben especificar diferentes nombres de gráficos.
<a name="NOTE0_3"></a>


### Gráficos e hilos
<a name="graficos_hilos_ELTPARAGRAPHE000453"></a>

Un solo gráfico puede ser manejado en un Thread.

Consecuencias:

- Dos hilos no pueden hacer manipular el mismo gráfico.

- Un gráfico creado en una Thread se destruye automáticamente al final de esta Thread. El gráfico existe en el subproceso que contiene la llamada a la función **grCreate**.



<a name="NOTE0_4"></a>


### Valores de Default de un gráfico creado mediante programación
<a name="valores_default_grafico_creado_mediante_programacion_ELTPARAGRAPHE000467"></a>

Los valores de Default de un gráfico son los siguientes:

- Menú contextual habilitado (configurable por [grMenu](../WDLang3/3042035.md) y [grMenuOpción](../WDLang3/3042042.md)).
	

- Marcas automatic en los ejes (configurables por [grInterval](../WDLang3/3042027.md)).

- Los colores se seleccionan según un orden preset entre un set de 13 colores (configurable por [grSeriesColor](../WDLang3/3042013.md)).

- Los gradientes color están disponibles (pueden ser configurados por [grGradient](../WDLang3/3042029.md)).

- Tamaño del archivo (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).

- Tamaño de la ventana (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).
	

- No hay etiqueta para la serie (puede ser configurada por [grSeriesLabel](../WDLang3/3042024.md)).

- No hay leyenda para los elementos (puede ser configurada por [grCategoryLabel](../WDLang3/3042015.md) y [grLabel](../WDLang3/3042046.md)).

- Sin leyenda (puede ser configurada por [grLegend](../WDLang3/3042047.md)).

- No hay título para el gráfico (puede ser configurado por [grTitle](../WDLang3/3042056.md)).

- No hay título para los ejes (puede ser configurado por [grAxisTitle](../WDLang3/3042057.md)).

- No hay líneas de cuadrícula (puede ser configurado por [grGridlines](../WDLang3/3042052.md)).

- Sin datos (los elementos pueden ser añadidos por [grAddData](../WDLang3/3042023.md)).

- Sin destino (puede ser configurado por [grDestinoWMF](../WDLang3/3042019.md), [grDestinoControl](../WDLang3/3042020.md), [grDestinoEMF](../WDLang3/3042017.md), [grDestinoWnd](../WDLang3/3042031.md) y [grDestinoBMP](../WDLang3/3042018.md)).

- No hay visualización de gráficos (para dibujar el gráfico, utilice [grDraw](../WDLang3/3042026.md)).

- Sin efecto de elevación (configurable por [grRaised](../WDLang3/3042051.md)).

- Para los gráficos de tarta, el ángulo inicial es igual a 0 (puede ser configurado por [grPieStartAngle](../WDLang3/3042022.md)).

- Para los gráficos de barras y los gráficos MinMax, el espacio entre dos categorías es igual a 10 píxeles (puede ser configurado por [grColumnSpacing](../WDLang3/3042014.md)).

- Para los gráficos de dispersión, los puntos son relacionado y su tamaño es igual a 1 (puede ser configurado por [grScatterLinkPoint](../WDLang3/3042040.md) y [grScatterPointSize](../WDLang3/3042041.md)).

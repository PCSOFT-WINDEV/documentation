## Tipos de variables associated con la gestión de los correos electrónicos
<a name="tipos_variables_associated_con_gestion_los_correos_electronicos_ELTTEXTE000516"></a>



|   |   |
| --- | --- |
| [Correo electrónico](../WDLang3/1000018713.md) | El tipo **Email** permite manipular el contenido de un correo electrónico así como la información asociada. |
| [emailAttach](../WDLang3/1000018752.md) | El tipo **emailAttach** se usa para manipular un archivo adjunto. |
| [emailIMAPSession](../WDLang3/1000018957.md) | El tipo **emailIMAPSession** se utiliza para acceder a un servidor de mensajería utilizando el protocolo IMAP (INTERNET Access Message Protocol) con el fin de recuperar los mensajes que se encuentran en este servidor. |
| [emailNotesSession](../WDLang3/1000018768.md) | El tipo **emailNotesSession** permite acceder a un servidor de mensajería Lotus Notes para identificar los mensajes que se encuentran en este servidor y enviar mensajes. |
| [emailOutlookSession](../WDLang3/1000018767.md) | El tipo **emailOutlookSession** se utiliza para manipular el software de mensajería de Outlook (correos electrónicos, contactos, grupos de contactos, tareas, citas y carpetas). |
| [emailPOP3Session](../WDLang3/1000018759.md) | El tipo **emailPOP3Session** se utiliza para acceder a un servidor de mensajería utilizando el protocolo POP3 (Post Office Protocol) con el fin de recuperar los mensajes que se encuentran en este servidor. |
| [emailSMTPSession](../WDLang3/1000018765.md) | El tipo **emailSMTPSession** permite acceder a un servidor de mensajería utilizando el protocolo SMTP (Simple Mail Transfer Protocol) para enviar correos electrónicos. |

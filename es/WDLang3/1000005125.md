
## NotesNbAttachment( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de archivos adjuntos a un elemento del actual Document en Lotus Notes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the connection
ConnectionID is int
ConnectionID = NotesOpenConnection("Password", "MarsServer", ...
		"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Specify the Lotus Notes database to use
DatabaseID is int
DatabaseID = NotesOpenDatabase(ConnectionID, "julia.nsf")
...
// Read the first document
NotesDocumentFirst(DatabaseID)
...
// Number of files attached to the "MyLotusNotesItem" item in the current Notes document?
NbFiles is int 
NbFiles = NotesNbAttachment(DatabaseID, "MyLotusNotesItem")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesNbAttachment(<Session identifier> , <Lotus Notes item>)`
---

**`<Result>: (Integro)`**

Número de archivos adjuntos al elemento especificado en las actuales Notas Document. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<Lotus Notes item>: (Cadena de caracteres)`**

Nombre del elemento Lotus Notes (Texto enriquecido) a utilizar.

Para conocer el nombre y el tipo de los elementos de las notas actuales Document, utilice [NotesListItem](../WDLang3/1000005112.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Artículo de un Notas Document
<a name="articulo_notas_document_ELTPARAGRAPHE000057"></a>

Cada elemento de un Document de notas puede contener hasta 256 archivos adjuntos.

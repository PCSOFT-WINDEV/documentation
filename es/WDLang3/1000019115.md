
## SaaSAdminModifyClient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una cuenta de cliente SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador del SaaS podrá modificar todas las cuentas de los clientes. Un gestor de cuenta de cliente sólo tendrá acceso a su propia cuenta de cliente. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the client name from "Hybrid" to "Hybrid Corp."
S is saasClient 
S = SaaSAdminFindClient("Hybrid")
S.Name = "Hybrid corp."

IF SaaSAdminModifyClient(S) = True THEN
	Info("The client account was successfully modified.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminModifyClient(<Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Client account>: (Variable de tipo saasClient)`**

Nombre del [saasClient](../WDLang3/1000019072.md) Variable correspondiente a la cuenta del Cliente SaaS que debe ser modificado en la base de datos SaaS.

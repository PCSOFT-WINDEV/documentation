
## SaaSAdminSubscriptionDeallocateLicense( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela una licencia de suscripción para un usuario de SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las licencias. Un gestor de cuenta de cliente sólo tendrá acceso a las licencias de sus usuarios.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
P is dynamic saasPricing 
C is saasClient dynamic 
Sub is saasSubscription dynamic 
U is saasUser dynamic 

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")
Sub = SaaSAdminFindSubscription(P, C)
U = SaaSAdminFindUser("mark@hybrid.com")

// The usernamed Mark is no longer allowed to benefit from the promotion 2010
SaaSAdminSubscriptionDeallocateLicense(Sub, U)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminación de una licencia mediante el uso de la suscripción

`<Result> = SaaSAdminSubscriptionDeallocateLicense(<Subscription> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Subscription>: (saasSubscription Variable)`**

Nombre de la [saasSubscription](../WDLang3/1000019088.md) Variable correspondiente a la suscripción que debe ser modificada en la base de datos SaaS.

**`<User>: (saasUser Variable)`**

Nombre de [saasUser](../WDLang3/1000019083.md) Variable que identifica al usuario al que se le retirará la licencia. Este usuario ya no puede utilizar la suscripción.


<a name="SYNTAX2"></a>

### Eliminar una licencia utilizando el precio

`<Result> = SaaSAdminSubscriptionDeallocateLicense(<Pricing> , <SaaS site> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre del precio relevante.

Debe existir una suscripción relacionado a esta tarificación para la cuenta de cliente del usuario.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio al que pertenece el precio.

**`<User>: (Cadena de caracteres)`**

Login del usuario al que se le quitará la licencia. Este usuario ya no puede utilizar la suscripción.

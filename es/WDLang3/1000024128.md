


## &lt;SaveWMF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un gráfico que fue dibujado de antemano en formato WMF..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Saves the chart named CHART_MyChart in the "C:\Charts\Chart.WMF" file
CHART_MyChart.SaveWMF("C:\Charts\Chart.WMF")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.SaveWMF(<WMF file name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el gráfico,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<WMF file name>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo WMF a crear. La extensión del archivo es obligatoria. Este archivo WMF se crea al llamar a la función **&lt;SaveWMF**. El archivo se sobrescribe y se vuelve a crear si ya existía..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Guardar un gráfico
<a name="guardar_grafico_ELTPARAGRAPHE000200"></a>

- Para guardar un gráfico, éste debe haber sido dibujado previamente ([&lt;Dibujo](../WDLang3/1000024062.md)). Las modificaciones realizadas desde la última llamada a [&lt;Dibujo](../WDLang3/1000024062.md) serán ignoradas.

- El tamaño por defecto de la imagen es set a 639 x 479. Estas dimensiones se tienen en cuenta si la carta nunca fue dibujada o si no es Visible. 



<a name="NOTE0_2"></a>


### Copia de seguridad en formato WMF
<a name="copia_seguridad_formato_wmf_ELTPARAGRAPHE000230"></a>

- Este archivo puede ser leído por una herramienta que reconoce el formato WMF.

- El tamaño del archivo es idéntico al tamaño del destino (tamaño de la ventana, control o archivo).

- Le recomendamos que previsualice el gráfico antes de guardarlo en un archivo.



<a name="NOTE0_3"></a>


### Diferencias entre grDestinationWMF y &lt;SaveWMF
<a name="diferencias_entre_grdestinationwmf_docparampagetitleshort_ELTPARAGRAPHE000242"></a>

- [grDestinationWMF](../WDLang3/3042019.md) se utiliza para definir el destino del siguiente dibujo del gráfico. Cuando se utiliza [&lt;Dibujo](../WDLang3/1000024062.md), el gráfico se dibujará en el archivo especificado y se tendrán en cuenta todos los cambios realizados desde la última vez que se dibujó el gráfico.

- La función **&lt;SaveWMF** permite guardar el gráfico directamente en un archivo en formato WMF. La Image guardada corresponde al gráfico durante la última llamada a [&lt;Dibujo](../WDLang3/1000024062.md). Los cambios realizados en el gráfico desde la última vez que se llamó a [&lt;Dibujo](../WDLang3/1000024062.md) serán ignorados.

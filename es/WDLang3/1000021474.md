


## MapGetJSObject (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la instancia API de un control Mapa. Esto le permite manipular la control Mapa en un código JavaScript directamente. 

En combinación con [MapConfigureGglAPI](../WDLang3/1000021611.md), esta función permite gestionar parámetros adicionales no propuestos por defecto por control Mapa. 

**Observación**: En WINDEV, esta función se puede utilizar en el código JavaScript ejecutado por [MapExecuteJS](../WDLang3/1000021615.md) para manipular el control Mapa de la aplicación WINDEV.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Server code
MapConfigureGglAPI("libraries=geometry,places","language=fr")
```

```wl
// Browser code
MyMap is object dynamic
MyMap = MapGetJSObject(MAP_Browser)
FindAddress(MyMap)
```

```txt
// JavaScript function that uses the dynamic object
FindAddress(MyMap)
{
	var infowindow;
	// Rome: 41.873144, 12.480391
	var pyrmont = new google.maps.LatLng(41.873144, 12.480391);
	var request = {
		location: pyrmont,
		radius: 500,
		types: ['store']
		};
	infowindow = new google.maps.InfoWindow();
	var service = new google.maps.places.PlacesService(MyMap);
	service.nearbySearch(request, ReturnAddress_WB);
	
}
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapGetJSObject([<Map control>])`
---

**`<Result>: (Objeto dinámico)`**

Nombre del objeto dinámico Variable que contiene la instancia del API de control Mapa. 

**`<Map control>: (Nombre de control opcional)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, se utilizará la control Mapa a la que pertenece la actual Event.

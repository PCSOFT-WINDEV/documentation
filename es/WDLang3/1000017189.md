
## HTTPAddParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un parámetro a un formulario HTTP. Este formulario ha sido creado por [HTTPCreateForm](../WDLang3/1000017119.md) y será enviado por [HTTPSendForm](../WDLang3/1000017190.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPCreateForm("FORM")
HTTPAddParameter("FORM", "Price", "3400") 
...
HTTPSendForm("FORM", "www.mysite.com")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HTTPAddParameter(<Form name> , <Parameter name> , <Parameter value>)`
---

**`<Form name>: (Cadena de caracteres)`**

Nombre del formulario HTTP a utilizar. Este nombre se definió con [HTTPCreateForm](../WDLang3/1000017119.md). Se produce un error si el formulario no existe.

**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro a añadir al formulario. Si el parámetro ya existe en el formulario, se sobrescribe su valor.

**`<Parameter value>: (Cadena Ansi o Unicode, o Buffer)`**

Valor que debe darse al parámetro.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para añadir un archivo al formulario, utilice [HTTPAddFile](../WDLang3/1000017191.md).




## grRealTimeInterval (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece el tiempo interval Visible en un gráfico de tiempo con visualización en tiempo real. 

Ejemplo: En este ejemplo, el tiempo de interval es de 10 segundos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe%20temps%20r%E9el%20-%20HC%20N%B0001%201.gif)
Observación: Esta función sólo se aplica a los gráficos con escala de tiempo, cuyos datos se añaden a través de [grAddTimeData](../WDLang3/1000023532.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FOR I = 1 TO 10
	d is Duration
	d.Minute = I
	grAddTimeData(CHART_MyChart, 1, d, 2 + Sin(i*36))
END

d is Duration = 2min
grRealTimeInterval(CHART_MyChart, d)
grDraw(CHART_NoName1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grRealTimeInterval(<Chart control> , <Interval>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se va a utilizar. 

**`<Interval>: (Duración)`**

Tiempo interval Visible en el gráfico (expresado en milisegundos). Por Default, este valor es de 60 segundos. Si este parámetro corresponde a 0, se utiliza el valor de Default.




## &lt;Tabla&gt;.SurfaceAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a un gráfico de superficie. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
x, y, z are real
wImage is Image

// Load the photo found in memory in an Image variable
wImage = "sunset.jpg"


// Scan the image and all the points
// to define the rate of lightness for the point
// via ColorLightness
FOR I = 1 _TO_ wImage.Width STEP 4
	FOR j = 1 _TO_ wImage.Height STEP 4
		x = i
		y = j
		z = ColorLightness(dPixelColor(wImage, x, y))
		CHART_Surface.SurfaceAddData(x, y, z)
	END
END

// Draw the chart in the Chart control
CHART_Surface.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceAddData(<X> , <Y> , <Z>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico (se encuentra en el editor de ventanas o en el editor de reportes).

**`<X>: (Real)`**

Coordenada X del punto a añadir.

**`<Y>: (Real)`**

Coordenada Y del punto a añadir.

**`<Z>: (Real)`**

Cota Z del punto a sumar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gráfico de superficie
<a name="grafico_superficie_ELTPARAGRAPHE000200"></a>

Para los datos de una carta de superficie, el plano XY constituye la base de la superficie, mientras que Z indica la altura del punto.
<a name="NOTE0_2"></a>


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000210"></a>

[.AddData](../WDLang3/1000023533.md) se utiliza para añadir datos en un gráfico (de tarta, de columnas, ...).

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [.StockAddData](../WDLang3/1000023565.md).

- Para los gráficos de dispersión, se recomienda utilizar [.ScatterAddDataXY](../WDLang3/1000024105.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [&lt;Gráfico&gt;.HMAddData](../WDLang3/1000024079.md).

- Para las cartas Sunburst, se recomienda utilizar [&lt;Gráfico&gt;.SunburstAddData](../WDLang3/1000024132.md). 




Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [.IncreaseData](../WDLang3/1000024086.md).

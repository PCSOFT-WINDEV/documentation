
## SaaSWriteSiteParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una información específica para un sitio SaaS en la configuración de una cuenta de cliente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Customize the logo
SaaSWriteSiteParameter("logo", "./logo_PCSOFT.png")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSWriteSiteParameter(<Parameter name> , <Value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el parámetro,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


Observación: El usuario conectado debe tener los derechos para escribir esta información (derechos de gestión).

**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro que será modificado en el sitio actual.

Los parámetros del sitio deben haber sido definidos en el [Administrador SaaS](../SaaSAdminWeb/1600000.md) al configurar el sitio.

**`<Value>: (Cadena de caracteres)`**

Nuevo valor para el parámetro.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los parámetros de un sitio son definidos por el administrador del sistema SaaS al configurar el sitio en el sistema [Administrador SaaS](../SaaSAdminWeb/1600000.md). Pueden tener valores de Default.

Los parámetros del sitio están pensados para permitir la personalización de la interfaz gráfica de usuario o el comportamiento de los sitios SaaS.. Para más detalles sobre su uso, consulte "[Desarrollo de un sitio SaaS](../WDLang3/9500214.md)".

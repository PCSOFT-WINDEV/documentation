
## Procedimiento WLanguage llamado por la función iOSApplicationOpenURLProcedure



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento WLanguage ("callback") llamado por la función [iOSApplicationOpenURLProcedure](../WDLang3/1410089346.md) cuando la aplicación debe abrir un recurso identificado por una URL. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iOSApplicationOpenURLProcedure(DisplayURL)

INTERNAL PROCEDURE DisplayURL(uURL is URI)
	IF WinStatus(WIN_NoName1) <> NotFound THEN
		WIN_NoName1.STC_Result = uURL
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`iOSApplicationOpenURLProcedure_Callback(<URL>)`
---

**`<URL>: (Variable de tipo URI)`**

Variable de tipo [URI](../WDLang3/1000024027.md) que corresponde a la URL que debe abrirse.




## &lt;Tabla&gt;.AxisTitleFont (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la fuente utilizada para los títulos de los ejes del gráfico.

Ejemplo de títulos para los ejes: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titreaxe.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the font used for
// the title of axes in "CHART_MyChart".
MyFont is Font
MyFont.Name = "Arial"
MyFont.Size = 12
MyFont.Bold = True
CHART_MyChart.AxisTitleFont(MyFont)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Definir la fuente utilizada para el título de los ejes del gráfico

`<Chart control>.AxisTitleFont(<Font used>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada en la ventana, página o editor de reportes.


**`<Font used>: (tipo de letra)`**

Fuente a utilizar. Este parámetro corresponde a una variable de tipo [tipo de letra](../Motscles/1514045.md). Las características de la fuente se definieron mediante la función [FontCreate](../WDLang1/3060002.md) o las [propiedades de fuentes](../Proprietes/2515045.md).


<a name="SYNTAX2"></a>

### Definir la fuente utilizada para el título de los ejes del gráfico (sintaxis conservada para compatibilidad retroactiva)

`<Chart control>.AxisTitleFont(<Font identifier>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada en la ventana, página o editor de reportes.


**`<Font identifier>: (Integro)`**

Identificador de fuente, definido por [grCreateFont](../WDLang3/3042001.md). Un error de WLanguage ocurre si esta fuente no existe.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Título del gráfico
<a name="titulo_del_grafico_ELTPARAGRAPHE000288"></a>El título de los ejes del gráfico se puede mostrar para los diferentes ejes.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)
 El título de los ejes del gráfico está definido por [&lt;Gráfico&gt;.AxisTitle](../WDLang3/1000024153.md). Esta función también puede utilizarse para set la posición del título.

La fuente utilizada para el título del gráfico se puede modificar con la función **&lt;Tabla&gt;.AxisTitleFont**.
<a name="NOTE0_2"></a>


### Inicialización de fuentes
<a name="inicializacion_fuentes_ELTPARAGRAPHE000309"></a>

La fuente Default utilizada para el título de los ejes tiene las siguientes características:

- Nombre de la fuente: Arial, 

- envergadura: 14,

- Estilo: Normal, 

- Color: Black .


## GroupAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un grupo de contactos en el libro Address de un Lotus Notes o un mensaje de Outlook. La [Estructura del grupo](../WDLang3/1000008120.md) se utiliza para definir los datos de este grupo de contactos.

**Atención**: Esta función **no permite** agregar un grupo de contactos a la libreta de direcciones de una mensajería Outlook Express. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Initialize the variables of the Group structure
GroupReset()
// Fill the variables for the new group
mGroup.Name = "My friends"
mGroup.Description = "Group of friends"
mGroup.NbContact = 3
mGroup.Contact[1] = "Peter"
mGroup.Contact[2] = "Paul"
mGroup.Contact[3] = "Jack"
// Add the group
IF GroupAdd(SessionID) = True THEN
	Info("Group of contacts successfully added")
ELSE
	Error("Error during the addition", ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = GroupAdd(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los grupos de contactos no son compatibles con Outlook 97. Por lo tanto, no se puede añadir ningún grupo de contactos en esta versión de Outlook.

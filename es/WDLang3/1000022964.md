


## grScatter3DAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un nuevo punto para mostrar en un gráfico de dispersión 3D. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a point with the color and shape of series
grScatter3DAddData(CHART_Name3DScatter, 1, 10, 10, 10)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grScatter3DAddData(<Chart name> , <Series number> , <X value> , <Y value> , <Z value> [, <Color> [, <Type>]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas).




**`<Series number>: (Integro)`**

Número de la serie a la que se añadirá un valor. Si la serie especificada no existe, el diseñador de la carta crea tantas series como sean necesarias para get una matriz consistente. Todas las series creadas se inicializan a 0.

**`<X value>: (Real)`**

Coordenada horizontal (coordenada X) del punto a añadir.

**`<Y value>: (Real)`**

Coordenada vertical (coordenada Y) del punto a añadir. 

**`<Z value>: (Real)`**

Coordenada en altitud del punto a sumar. 

**`<Color>: (Entero o constante de tipo Integer)`**

Punto Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [y la preselección Color](../WDLang5/3010002.md).

- la constante *ColorUndefined* (valor por defecto) para aplicar la serie Color. 




**`<Type>: (Constante opcional de tipo Integer)`**

Tipo de punto:


|   |   |
| --- | --- |
| *grPointCircle* | Punto en forma de círculo. |
| *grPointCross* | Punta en forma de cruz. |
| *grPointCube* | Punto en forma de cubo (3D). |
| *grPointDiamond* | Punta en forma de diamante. |
| *grPointPyramid* | Punta en forma de pirámide (3D). |
| *grPointSphere* | Punto en forma de esfera (3D). |
| *grPointSquare* | Punto en forma de cuadrado. |
| *grPointTriangle* | Punta en forma de triángulo. |


Si no se especifica este parámetro, el tipo de punto corresponde al tipo de punto definido para la serie. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El tamaño del punto se basa en el tamaño mínimo de la marca del gráfico.




## tapiBlindTransfer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Realiza una transferencia "a ciegas. No hay capacidad para recuperar el Line.
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiBlindTransfer(<Call identifier> , <Transfer number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se transfirió la llamada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Call identifier>: (Integro)`**

Identificador de llamada a transferencia. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md).

**`<Transfer number>: (Cadena de caracteres)`**

Número de Phone a marcar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Limitación
<a name="limitacion_ELTPARAGRAPHE000039"></a>

No se puede llamar a la función **tapiBlindTransfer** en el procedimiento de detección de llamadas (definido con la función [tapiEscuchar](../WDLang3/3076029.md)).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Transferencia de una llamada con capacidad para recuperar la comunicación
<a name="transferencia_una_llamada_con_capacidad_para_recuperar_comunicacion_ELTPARAGRAPHE000053"></a>

Para realizar una transferencia de llamadas con posibilidad de recuperar la llamada inicial (si el número de transferencia se niega a coger la comunicación, por ejemplo), utilice [tapiCompletoTransferencia](../WDLang3/3076021.md).
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.

Atención: la transferencia de llamadas no está disponible para todos los dispositivos. La transferencia de llamadas se tiene en cuenta para una centralita telefónica.

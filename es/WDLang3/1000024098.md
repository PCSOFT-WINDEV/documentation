


## &lt;Scatter3DAltitudeColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define los diferentes colores utilizados según los niveles de altitud de una serie que se encuentran en un gráfico de dispersión 3D.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_My3DScatter.Scatter3DAltitudeColor(1, LightRed, 100)
CHART_My3DScatter.Scatter3DAltitudeColor(1, PastelBlue, 0)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.grScatter3DAltitudeColor(<Series number> , <Color> , <Altitude>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en el editor de ventanas).

**`<Series number>: (Integro)`**

Número de la serie a utilizar.

**`<Color>: (Entero o constante de tipo Integer)`**

Color para la altitud especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).




**`<Altitude>: (Real)`**

Altitud para la cual se aplicará el Color.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si se especifican los colores de altitud, los puntos encontrados a esta altitud se colorearán en consecuencia.. 

Para los puntos que se encuentran entre dos colores, se aplicará una Gradient lineal.

Para los puntos que se encuentran fuera de los límites de Color, se aplicará el Color de serie.

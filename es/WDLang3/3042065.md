


## grSaveParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda los parámetros de un gráfico en formato de cadena. Estos parámetros pueden ser restaurados por [grLoadParámetro](../WDLang3/3042064.md).

**Recordatorio**: Por Default, la configuración de un gráfico se guarda automáticamente al cerrar la aplicación. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the parameters of a chart created through programming
INIWrite("Parameters", "MyChart", "Param", grSaveParameter("MyChart"))

// Restore the parameters of a chart created through programming
grLoadParameter("MyChart", INIRead("Parameters", "MyChart", "Param"))
```
<a name="Example2"></a>

```wl
// Save the parameters of a Chart control
INIWrite("Parameters", "CHART_Chart", "Param", grSaveParameter(CHART_Chart))

// Restore the parameters of a Chart control
grLoadParameter(CHART_Chart, INIRead("Parameters", "CHART_Chart", "Param"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grSaveParameter(<Chart name>)`
---

**`<Result>: (Cadena de caracteres ANSI)`**



- Ajustes del gráfico. Esta cadena puede guardarse para ser restaurada posteriormente por [grLoadParámetro](../WDLang3/3042064.md).

- Cadena vacía ("") si se produce un error.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se guardarán los siguientes parámetros de la tabla: 

- el tipo de gráfico.

- el título del gráfico y su posición.

- la posición de la leyenda.

- el espacio entre dos barras para los gráficos de columnas.

- la profundidad de los gráficos de columnas.

- las características de la Gradient.

- la orientación de los ejes.

- las líneas de la cuadrícula.

- los parámetros 3D para los gráficos en 3 dimensiones.

- las características de las fuentes utilizadas.

- las características del alisamiento.

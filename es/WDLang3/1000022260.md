


## grAxisPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la posición donde se dibujará uno de los ejes del gráfico (coordenadas X o Y). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grAxisPosition(CHART_MyChart, grXCoordinate, grMin)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grAxisPosition(<Chart name> , <Axis> , <Position>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en la ventana, página o editor de reportes).






**`<Axis>: (Constante de tipo Integer)`**

Eje a posición: 


|   |   |
| --- | --- |
| *grXCoordinate* | Eje de coordenadas X (eje X). |
| *grYCoordinate* | Eje de coordenadas Y (eje Y). |



**`<Position>: (Constante de tipo Integer)`**

Posición en la que se debe colocar el eje. 


|   |   |
| --- | --- |
| *grMax* | El eje se coloca en la marca máxima del gráfico. |
| *grMin* | El eje se coloca en la marca mínima del gráfico. |
| *grZero* | El eje se sitúa en la marca 0. <br><br>Esta constante corresponde a la visualización de las cartas de Default. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


- Esta función se puede utilizar en todos los tipos de gráficos que muestran un eje.

- Esta función se utiliza para modificar el eje primario de las coordenadas Y. El eje secundario de coordenadas Y se sitúa siempre en sentido contrario al eje primario de coordenadas Y.

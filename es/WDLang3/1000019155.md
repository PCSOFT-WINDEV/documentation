


## <zipArchive Variable>FileSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño de uno de los ficheros encontrados en el archivo (en formato CAB, RAR, WDZ, ZIP, 7z, TAR o TGZ (TAR.GZ)):

- el tamaño de un archivo comprimido.

- el tamaño del archivo antes de la compresión.




Al extraer un archivo de un archivo comprimido, le recomendamos que compare el tamaño sin comprimir del archivo con el espacio disponible en el disco.

Recordatorio: 

- ![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo están disponibles los archivos ZIP, TAR y TGZ (TAR.GZ).

- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sólo están disponibles los archivos ZIP.

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyArchive is zipArchive
// Open an archive
ResOpenArchive = MyArchive.Open("C:\MyArchives\Archive.zip")

// Size of "File.txt"
IF ResOpenArchive = 0 THEN
	ResFileSize = MyArchive.FileSize("File.txt")
	...
END
```

## Sintaxis

### Averiguar el tamaño de un fichero identificado por su índice

`<Result> = <Archive>.FileSize(<File index> [, <Type of size>])`
---

**`<Result>: (Integro)`**

Tamaño del archivo especificado.

**`<Archive>: (Variable de tipo zipArchive)`**

Nombre de una Variable de tipo [zipArchive](../WDLang3/1000018679.md) correspondiente al archivo a utilizar.

**`<File index>: (Integro)`**

Índice del archivo cuyo tamaño debe ser calculado. [.FindFile](../WDLang3/1000026330.md) devuelve este índice.

**`<Type of size>: (Booleano opcional)`**

Tipo de cálculo para el tamaño del archivo:

- <u><u><u><u>True</u></u></u></u>: Tamaño del archivo comprimido.

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): Tamaño del archivo antes de la compresión.




<a name="SYNTAX2"></a>

### Averiguar el tamaño de un archivo identificado por su ruta

`<Result> = <Archive>.FileSize(<File path> [, <Type of size>])`
---

**`<Result>: (Integro)`**

Tamaño del archivo especificado.

**`<Archive>: (Variable de tipo zipArchive)`**

Nombre de una Variable de tipo [zipArchive](../WDLang3/1000018679.md) correspondiente al archivo a utilizar.

**`<File path>: (Integro)`**

ruta almacenada del archivo cuyo tamaño debe ser calculado.

**`<Type of size>: (Booleano opcional)`**

Tipo de cálculo para el tamaño del archivo:

- <u><u><u><u>True</u></u></u></u>: Tamaño del archivo comprimido.

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): Tamaño del archivo antes de la compresión.

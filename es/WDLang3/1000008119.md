


## Contact agrupa las funciones de gestión
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los grupos de contactos que se encuentran en el libro de Address de Lotus Notes, Domino o Outlook:



|   |   |
| --- | --- |
| [GroupAdd](../WDLang3/1000008111.md) | Añade un grupo de contactos en el libro de Address de una mensajería de Lotus Notes o de Outlook. |
| [GroupDelete](../WDLang3/1000008118.md) | Borra el grupo actual de contactos del libro Address de un Lotus Notes o de un mensaje de Outlook. |
| [GroupFirst](../WDLang3/1000008115.md) | Lee el primer grupo de contactos que se encuentra en el libro de Address de un Lotus Notes o de un mensaje de Outlook. |
| [GroupLast](../WDLang3/1000008112.md) | Lee el último grupo de contactos de una agenda de Lotus Notes o Outlook Address. |
| [GroupModify](../WDLang3/1000008113.md) | Modifica el grupo actual de contactos en el libro de Address de una mensajería de Lotus Notes o de Outlook. |
| [GroupNext](../WDLang3/1000008117.md) | Lee el grupo de contactos que se encuentra después del grupo actual en el libro de Address de una mensajería de Lotus Notes o de Outlook. |
| [GroupPrevious](../WDLang3/1000008114.md) | Lee el grupo de contactos anterior al grupo actual en una libreta de Lotus Notes o Outlook Address. |
| [GroupRead](../WDLang3/1000008121.md) | Lee un grupo de contactos que se encuentran en la libreta de Address de una mensajería de Lotus Notes o Outlook. |
| [GroupReset](../WDLang3/1000008116.md) | Reinicia el [Estructura del grupo](../WDLang3/1000008120.md). |

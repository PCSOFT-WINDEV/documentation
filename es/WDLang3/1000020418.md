
## NFCSendTag( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía una etiqueta NFC a otro dispositivo. 

**Observación**: Las funciones NFC no se pueden utilizar en el simulador ni en el emulador.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sends an URL to another device 
tag is nfcTag
url is nfcData
url.Type = nfcURI
url.Content = "www.pcsoft.com"
Add(tag.Data, url)
IF NFCSendTag(tag) = False THEN
	Error("Unable to write NFC tag.", ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NFCSendTag(<Tag>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó en envío,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Tag>: (nfcTag Variable)`**

Nombre de la [nfcTag](../WDLang3/1000020411.md) Variable que describe los datos a enviar.

<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000056"></a>

- Las etiquetas NFC son enviadas por la función Android Beam del dispositivo. Esta característica debe estar habilitada en el dispositivo transmisor y en el dispositivo receptor.. 
	La función Android Beam está activada en la ventana de configuración de las comunicaciones inalámbricas. 

- La llamada a **NFCSendTag** abre automáticamente una ventana para pedir a los usuarios que acerquen sus dispositivos a otro dispositivo con NFC. Esta ventana se cerrará automáticamente: 

	- cuando el envío termine,

	- si el usuario cancela el envío pulsando la tecla Atrás del dispositivo.
			Observación: Para personalizar el mensaje que se muestra en la ventana, utilice la función [NextTitle](../WDLang1/3038043.md) antes de llamar a la función **NFCSendTag**. 




- En el dispositivo de destino, la etiqueta enviada puede leerse como una etiqueta NFC estándar: 

	- mediante una aplicación desarrollada con WINDEV Mobile utilizando [NFCReadTag](../WDLang3/1000020416.md),

	- por una aplicación externa compatible.






<a name="NOTE0_2"></a>


### Lectura del NFC enviado
<a name="lectura_del_nfc_enviado_ELTPARAGRAPHE000085"></a>

Para que un dispositivo pueda leer la etiqueta NFC enviada por **NFCSendTag**, debe admitir: 

- el protocolo NDEF Push (com.android.npp),

- el protocolo SNEP (Simple NDEF Exchange Protocol). 


Si está ejecutando la versión 4.0 o posterior, debe soportar los dos protocolos.
<a name="NOTE0_3"></a>


### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000099"></a>

La llamada a esta función modifica los permisos requeridos por la aplicación. 

Permisos necesarios:  

- NFC: Permite a las aplicaciones realizar operaciones de entrada/salida a través de la tecnología NFC.

- VIBRATOS: Se utiliza para acceder a la función del sensor.

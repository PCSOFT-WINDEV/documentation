
## OutlookStartSession( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite acceder a los datos gestionados por el software de mensajería de Outlook (correos electrónicos, contactos, grupos de contactos, tareas, citas y carpetas).. Esta función es equivalente a la función [EmailStartOutlookSession](../WDLang3/3032128.md). Para cerrar la sesión, debe utilizar [Sesión cerrada](../WDLang3/3032124.md) o [EmailCloseSession](../WDLang3/3032006.md).

**Atención**: Esta función no se puede utilizar para acceder a los datos que maneja el software de mensajería Outlook Express.

**Observación**: Cuando se utiliza **OutlookStartSession**, el directorio actual es modificado automáticamente por Outlook.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032123&name=outlookstartsession_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Start the session
IF OutlookStartSession("MyProfile") <> 0 THEN
	Info("Session started")
ELSE
	Error(ErrorInfo())
END
```

## Sintaxis

`<Result> = OutlookStartSession(<Profile> [, <Start options>])`
---

**`<Result>: (Integro)`**



- Identificador de sesión. 

- 0 o <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Profile>: (Cadena de caracteres)`**

Identifica el perfil de conexión.



- El perfil Default es devuelto por [Perfil de OutlookDefault](../WDLang3/1000019347.md). 

- La lista de perfiles disponibles es devuelta por [Perfil de OutlookList](../WDLang3/1000019346.md). 




Si este parámetro corresponde a una cadena vacía ("""):

- si hay varios perfiles definidos en el ordenador actual, **OutlookStartSession** muestra una ventana con todos los perfiles entre los que el usuario puede elegir el suyo.

- si se define un único perfil en el ordenador actual, **OutlookStartSession** selecciona automáticamente este perfil.




**`<Start options>: (Constante opcional de tipo Integer)`**

Modo para iniciar la sesión de Outlook: 


|   |   |
| --- | --- |
| *outlookAPTWithContact*<br>(Valor predeterminado) | Las funciones de AppointmentXXX listarán los contactos asociados a las citas en el Variable de mAppointment.Participant (ver [Estructura de Appointment](../WDLang3/1000011010.md)). |
| *outlookAPTWithParticipant* | Las funciones de NombramientoXXX listarán a los participantes associated con los nombramientos en las siguientes variables:<br><br>- Nombramiento del Participante<br><br>- Nombramiento del Participante<br><br>- Nombramiento del ParticipanteBcc<br><br><br>Para obtener más información, consulte [Estructura de Appointment](../WDLang3/1000011010.md). |

## Observaciones




### Creando un perfil
<a name="creando_perfil_ELTPARAGRAPHE000104"></a>

Para iniciar una nueva sesión con Outlook, se debe definir un "perfil".. Este "perfil" se define en la configuración de la conexión de INTERNET con Outlook.

Para crear un perfil:

1. Inicie Outlook.

2. En la ventana "Elegir un perfil", haga clic en [ New].

3. Seleccione el servicio "Microsoft Exchange Server".

4. Dar un nombre al perfil.

5. Introduzca el nombre del servidor Microsoft Exchange.



<a name="NOTE0_2"></a>




### Directorio actual
<a name="directorio_actual_ELTPARAGRAPHE000126"></a>

**Atención**: La función **OutlookStartSession** modifica la ruta del directorio actual (devuelta por la función [fCurrentDir](../WDLang1/3036022.md)).

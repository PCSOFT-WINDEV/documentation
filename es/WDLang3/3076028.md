
## tapiOrigin( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite determinar el tipo de llamada (entrante o saliente).
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiOrigin([<Call identifier>])`
---

**`<Result>: (Constante de tipo Integer)`**

Origen de la llamada.


|   |   |
| --- | --- |
| *tapiOriginConference* | Llamada en modo conferencia (varias personas). |
| *tapiOriginExternal* | Llamada procedente del exterior. <br>El uso de este Variable depende del tipo de dispositivo utilizado, ... |
| *tapiOriginInbound* | Llamada entrante: comunicación desde el exterior. <br>El uso de este Variable depende del tipo de dispositivo utilizado, ... |
| *tapiOriginInternal* | Llamada interna a PABX |
| *tapiOriginOutbound* | Llamada saliente: comunicación hacia el exterior. |
| *tapiOriginUnavailable* | El origen de la llamada no está definido. <br>El uso de este Variable depende del tipo de dispositivo utilizado, ... |
| *tapiOriginUnknown* | El origen de la llamada no está definido. <br>El uso de este Variable depende del tipo de dispositivo utilizado, ... |



**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md).

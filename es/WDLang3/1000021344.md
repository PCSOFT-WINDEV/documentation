## Presentación
<a name="presentacion_ELTTEXTE000323"></a>
WINDEV, WEBDEV y WINDEV Mobile incluyen un control Gráfico (que se puede utilizar en el editor de ventanas, en el editor de páginas y en editor de reportes), así como varias funciones utilizadas para crear gráficos manipular..
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Uso de gráficos en WEBDEV Linux y en WINDEV Linux**
Para utilizar los gráficos en Linux, es necesario instalar la biblioteca QT (Core y GUI) versión 5 (o posterior). 

**En Ubuntu**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```

## ¿Cómo programar los gráficos?
<a name="¿como_programar_los_graficos_ELTTEXTE000347"></a>


### Creación de gráficos mediante programación
<a name="creacion_graficos_mediante_programacion_ELTPARAGRAPHE000070"></a>

WINDEV, WEBDEV y WINDEV Mobile incluyen varias funciones para manejar gráficos. Estas funciones permiten crear los gráficos, modificarlos, cambiar las opciones de visualización....

Las siguientes operaciones se utilizan para crear los gráficos: 

1. Cree el gráfico con [grCreate](../WDLang3/3042005.md).

2. Inicialice los datos del gráfico con las siguientes funciones:

	- [grAddData](../WDLang3/3042023.md).

	- [grScatterAddDataXY](../WDLang3/3042032.md) para los gráficos Scatter.

	- [grStockAddData](../WDLang3/3042009.md) para los gráficos de acciones.

	- [grSuperficieAñadirDatos](../WDLang3/1000020425.md) para las cartas de Superficie. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [grSunburstAddData](../WDLang3/1000021285.md) para las cartas de Sunburst.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [grHMAddData](../WDLang3/1000021628.md) para los gráficos de Heatmap.




3. Seleccione el destino de visualización del gráfico con las siguientes funciones:

	- [grDestinoControl](../WDLang3/3042020.md): para visualizar el gráfico en un control Imagen

	- [grDestinoWnd](../WDLang3/3042031.md): mostrar el gráfico en una ventana independiente
			

	- [grDestinationBMP](../WDLang3/3042018.md), [grDestinationEMF](../WDLang3/3042017.md) o [grDestinationWMF](../WDLang3/3042019.md) para guardar el gráfico en archivos en formato BMP, EMF o WMF respectivamente. 

	- [grDestinationImage](../WDLang3/1000020952.md): para guardar el gráfico en una [Image](../WDLang1/1000019650.md) Variable directamente. 




4. Dibuja el gráfico con [grDraw](../WDLang3/3042026.md).




Se utilizan varias funciones para configurar y modificar la apariencia del gráfico. Tiene la posibilidad de mostrar una leyenda, el título de los ejes, ... Para más detalles, consulte la lista de [funciones para la gestión de gráficos](../WDLang3/3042003.md).

### Gráficos e hilos
<a name="graficos_hilos_ELTPARAGRAPHE000157"></a>

Un solo gráfico puede ser manejado en un Thread.

Consecuencias:

- Dos hilos no pueden hacer manipular el mismo gráfico.

- Un gráfico creado en una Thread se destruye automáticamente al final de esta Thread. El gráfico existe en el hilo que contiene la llamada a [grCreate](../WDLang3/3042005.md).

### Valores Default de un gráfico (en una ventana o en un Page)
<a name="valores_default_grafico_una_ventana_page_ELTPARAGRAPHE000176"></a>

Los valores de Default de un gráfico son los siguientes:

- Menú contextual habilitado (configurable por [grMenu](../WDLang3/3042035.md) y [grMenuOpción](../WDLang3/3042042.md)).

- Marcas automatic en los ejes (configurables por [grInterval](../WDLang3/3042027.md)).

- Los colores se seleccionan según un orden preset entre un set de 13 colores (configurable por [grSeriesColor](../WDLang3/3042013.md)).

- Los gradientes color están disponibles (pueden ser configurados por [grGradient](../WDLang3/3042029.md)).

- Al pasar el ratón por un gráfico, se muestra automáticamente una información sobre la herramienta ([grTooltip](../WDLang3/3042010.md)).

- Tamaño del archivo (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).

- Tamaño de la ventana (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).

- No hay etiqueta para la serie (puede ser configurada por [grSeriesLabel](../WDLang3/3042024.md)).

- No hay leyenda para los elementos (puede ser configurada por [grCategoryLabel](../WDLang3/3042015.md) y [grLabel](../WDLang3/3042046.md)).

- Sin leyenda (puede ser configurada por [grLegend](../WDLang3/3042047.md)).

- No hay título para el gráfico (puede ser configurado por [grTitle](../WDLang3/3042056.md)).

- No hay título para los ejes (puede ser configurado por [grAxisTitle](../WDLang3/3042057.md)).

- No hay líneas de cuadrícula (puede ser configurado por [grGridlines](../WDLang3/3042052.md)).

- Sin datos (los elementos pueden ser añadidos por [grAddData](../WDLang3/3042023.md)).

- Sin destino (puede ser configurado por [grDestinoControl](../WDLang3/3042020.md), [grDestinoWnd](../WDLang3/3042031.md) y [grDestinationWMF](../WDLang3/3042019.md)).

- No hay visualización de gráficos (para dibujar el gráfico, utilice [grDraw](../WDLang3/3042026.md)).

- Sin efecto de elevación (configurable por [grRaised](../WDLang3/3042051.md)).

- Para los gráficos de tarta, el ángulo inicial es igual a 0 (puede ser configurado por [grPieStartAngle](../WDLang3/3042022.md)).

- Para los gráficos de barras y los gráficos MinMax, el espacio entre dos categorías es igual a 10 píxeles (puede ser configurado por [grColumnSpacing](../WDLang3/3042014.md)).

- Para los gráficos de dispersión, los puntos son relacionado y su tamaño es igual a 1 (puede ser configurado por [grScatterLinkPoint](../WDLang3/3042040.md) y [grScatterPointSize](../WDLang3/3042041.md)).

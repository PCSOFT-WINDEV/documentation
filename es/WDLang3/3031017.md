
## DotNetVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de versión de .Net Framework disponible en el ordenador actual.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the version of the .NET Framework
Info("Version of .NET Framework: " + DotNetVersion())
```


<a name="Example2"></a>



```wl
// Is the .NET 3.0 Framework installed on the current computer?
IF DotNetVersion(dnvInstalled, "3.0") = True THEN
	Info("The .NET 3.0 Framework is installed")
ELSE
	Info("The .NET 3.0 Framework is not installed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DotNetVersion([<Requested information> [, <Parameter>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Versión de .NET Framework,

- Vacíe la cadena (""") si .NET Framework no está disponible.




**`<Requested information>: (Constante opcional de tipo Integer)`**

Versión de .NET Framework para devolver:


|   |   |
| --- | --- |
| *dnvCurrent*<br>(Valor predeterminado) | Devuelve la versión actual instalada (es decir, la versión más reciente). |
| *dnvInstalled* | Devuelve True si la versión solicitada en &lt;Parámetro&gt; está instalada en el ordenador actual. |
| *dnvList* | Devuelve la lista de versiones instaladas en el ordenador actual.. Las diferentes versiones están separadas por caracteres CR. |



**`<Parameter>: (Variante opcional)`**

Si &lt;Información solicitada&gt; está configurada en dnvInstalled, &lt;Parámetro&gt; es una cadena de caracteres que contiene la versión cuya prueba debe ejecutarse ("3.0" por ejemplo).. La versión a probar se compara con las diferentes versiones instaladas en el equipo. La comparison entre los números de versión es un "Comienza con" comparison.


## upnpImage (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **upnpImage** permite definir las características avanzadas de la imagen de un dispositivo UPnP. Las características de esta UPnP Image pueden definirse y modificarse mediante diferentes propiedades del WLanguage. 

Este tipo de Variable es utilizado por el [upnpDevice](../WDLang3/1000020903.md) Variable s. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables upnpImage
<a name="propiedades_especificas_description_las_variables_upnpimage_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un Image de un dispositivo UPnP:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Altura | Integro | Alto de la imagen en pixeles. |
| anchura | Integro | Ancho de la imagen en pixeles. |
| BitPerPixel | Integro | Número de bits por pixel. |
| Image | Variable de tipo [Image](../WDLang1/1000019650.md) | Image |
| MIMEType | Cadena de caracteres | MIME tipo de Image. |
| URL | Cadena de caracteres | URL de la imagen. |

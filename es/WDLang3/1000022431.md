


## &lt;Mapa&gt;.ItinerarioDistancia (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la distancia de un itinerario en una control Mapa. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sID is string = MAP_MapControl.AddItinerary(["Place de la Comédie, Montpellier", ...
		"Avenue des Champs Elysées, Paris"], itineraryCar)
rDist is real
rDist = MAP_MapControl.ItineraryDistance(sID)
STC_DETAILS = "Distance: " + rDist / 1000.0 + " km"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.ItineraryDistance(<Itinerary ID>)`
---

**`<Result>: (Real)`**



- Distancia del itinerario expresada en metros. 

- 0 si se produce un problema (ID no válido, por ejemplo).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<Itinerary ID>: (Cadena de caracteres)`**

Identificador del itinerario a utilizar. Esta identificación es el resultado de la función [&lt;Mapa&gt;.AddItinerario](../WDLang3/1000022420.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **&lt;Mapa&gt;.ItinerarioDistancia** no tiene efecto para los itinerarios añadidos con [&lt;Mapa&gt;.AddItinerario](../WDLang3/1000022420.md) y la constante *itineraryManual*).




## btleDescriptor (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **btleDescriptor** se utiliza para manipular el descriptor de una característica de baja energía de Bluetooth. 

Este descriptor describe los datos intercambiados entre dos dispositivos compatibles con Bluetooth LE: unidad de valor, rango de valores, ... Se identifica mediante un string unívoco (UUID).

Las características de este descriptor pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMyService is btleService
...
cMyCharacteristic is btleCharacteristic <- cMyService[1]
cMyDesc is btleDescriptor <- cMyCharacteristic.Descriptor[1]
```

### Propiedades específicas de las variables btleDescriptor
<a name="propiedades_especificas_las_variables_btledescriptor_ELTPARAGRAPHE000041"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un descriptor:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Permission | Integro | Permisos associated con el descriptor.<br><br>Esta propiedad es de solo lectura. |
| UUID | Cadena de caracteres | Identificador del descriptor. <br><br>Esta propiedad es de solo lectura. |




## &lt;Gráfico&gt;.DonutHolePorcentaje (función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>27</B>, esta función se mantiene por compatibilidad (seguirá siendo soportada en la próxima versión). Se recomienda utilizar la función [<Chart>.HolePercentage](../WDLang3/1410088090.md). 
</DIV>

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el porcentaje correspondiente al radio del agujero en una tabla de donuts.. 


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The hole will represent 50% of the total radius of the donut
CHART_MyChart.DonutHolePercentage(50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.DonutHolePercentage(<Percentage>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Percentage>: (Integro)`**

Porcentaje representado por el agujero del donut (entre 0 y 70).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los ajustes definidos con **&lt;Gráfico&gt;.DonutHolePorcentaje** se tendrán en cuenta la próxima vez que se llame a [&lt;Dibujo](../WDLang3/1000024062.md).

- Esta función sólo se puede utilizar con las cartas de donuts.

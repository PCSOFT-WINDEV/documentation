


## grSuperficieBorrarColorAltitudColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los colores de altitud especificados por [grSurfaceAltitudeColor](../WDLang3/1000020427.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grSurfaceDeleteAltitudeColor(CHART_Chart1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSurfaceDeleteAltitudeColor(<Chart name> [, <Altitude>])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Altitude>: (Real opcional)`**

La altitud de Color que será eliminada. Si no se especifica este parámetro, se borran todos los colores de altitud.: el gráfico está usando la Color de la primera serie.

## Componente:


## SMSDelete( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>25</B>, esta función ya no debe utilizarse. A partir de Android 4.4, ya no se permite el borrado de SMS mediante programación. El uso de esta función provoca un error fatal en la aplicación.  
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un mensaje SMS de la memoria del Android Phone. 




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SMSDelete(<Location> , <SMS index>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el SMS,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.






**`<Location>: (Constante de tipo Integer)`**

Ubicación del SMS a eliminar:


|   |   |
| --- | --- |
| *smsMemory* | SMS almacenados en la memoria del dispositivo. |
| *smsSIM* | SMS almacenado en la tarjeta SIM<br> |



**`<SMS index>: (Integro)`**

Índice del mensaje SMS que se desea borrar. Este índice es devuelto por el SMS.Índice Variable. Para obtener más información, consulte [Estructura del SMS](../WDLang3/3068003.md).

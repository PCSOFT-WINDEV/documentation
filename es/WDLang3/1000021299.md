


## MapLicenseGgl( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la clave de licencia "Google Maps API" que se utilizará en los controles de mapa (de WINDEV o WEBDEV).. Esta tecla puede ser necesaria para utilizar las funciones "Google Maps API for work".. Para más detalles, consulte la licencia "Google Maps API".

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Importante**: Se debe llamar a la función **MapLicenseGgl** **antes** de abrir la ventana que contiene el control Mapa. Si el control Carta se encuentra en la primera ventana del proyecto, se debe llamar a la función **MapLicenseGgl** en el código de inicialización del proyecto.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the Google key
MapLicenseGgl("ABCDEF12345789abcdef")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapLicenseGgl(<Google key>)`
---

**`<Google key>: (Cadena de caracteres)`**

Clave suministrada por Google.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El uso de un control Mapa está sujeto a términos y condiciones específicos. Compruebe la licencia de Google sobre el uso de los servicios de mapping.

| --- |
| **Atención: Los términos de servicio de Google Maps cambiaron el 22 de junio de 2016**. De ahora en adelante, **se debe especificar** una clave de licencia para utilizar este servicio. Es necesario: <br><br>	1. Generar la clave en la consola de desarrollo de Google ([https://console.developers.google.com](https://console.developers.google.com)). Se deben realizar las siguientes acciones (válidas en la fecha de publicación de esta página): <br><br>		- Utilice una cuenta de Google para conectarse a la consola de desarrollo. <br><br>		- Crear un proyecto de Google si es necesario. <br><br>		- Acceda al "API manager" del proyecto.<br><br>		- En el elemento "Google Maps API", haga clic en el enlace "Google Maps JavaScript API".<br><br>		- Haga clic "Habilitar" para activar la API de Google Maps JavaScript. <br><br>		- En el punto "Identificadores", ampliar el botón "Crear identificadores. <br><br>		- Seleccione "Clave de API". <br><br>		- Seleccione "Clave de API". <br><br>			- Para un sitio WEBDEV, seleccione "Tecla de navegador".. Se recomienda especificar el dominio que utilizará la clave para supervisar las consultas y evitar cualquier tipo de fraude.<br><br>			- Para una aplicación WINDEV, seleccione "Server key".. Se recomienda especificar la dirección que utilizará la clave para supervisar las consultas y evitar cualquier tipo de fraude.<br><br><br><br><br>	- Valide la creación de la clave y copie la clave creada por Google. <br><br>	2. Especifique la clave utilizada en la aplicación o el sitio con **MapLicenseGgl**.<br><br><br> |



- Esta función le permite especificar la licencia de uso para que el proveedor de Google Maps pueda proporcionar la aplicación.: 

	- **AA partir del 22 de junio de 2016**, acceso a las funciones "Google Maps JavaScript API. 

	- acceso a las características ampliadas de "Mapas API para el trabajo" (mayor número de consultas, seguimiento, cuota,...).




- La licencia se aplica a todos los mapas de la aplicación y se utilizará para todos los nuevos controles de mapa creados después de esta llamada.. Le aconsejamos que llame a esta función en el código de inicialización del proyecto.




## EmailCloseSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sesión para la gestión de Email según el modo seleccionado:

- correos electrónicos gestionados por el protocolo SMTP/POP3/IMAP: la función **EmailCloseSession** permite cerrar la sesión SMTP/POP3/IMAP. Si se inició una sesión POP3, una sesión SMTP y una sesión IMAP, todas las sesiones se cierran.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) correos electrónicos gestionados a través del cliente MS Exchange (Outlook): **EmailCloseSession** cierra la sesión de MS Exchange. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) correos electrónicos gestionados por Lotus Notes: la función **EmailCloseSession** permite cerrar una sesión con Lotus Notes. 




Una sesión de Email que no haya terminado se cerrará automáticamente al final del programa WINDEV o WINDEV Mobile o al final de la sesión de WEBDEV.
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sólo se admiten las sesiones POP3/SMTP.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start an email session via the SMTP/POP3 protocol
IF EmailStartSession(USER, PASSWORD, "pop3.providername.com", ...
		"smtp.providername.com") = True THEN
	UserName = USER
ELSE
	UserName = ""
	Error("Unable to establish the connection")
END
// If a session is started, close this session 
IF UserName <> "" THEN
	EmailCloseSession(UserName)
	UserName = ""
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cerrar una sesión de Email que utiliza el protocolo POP3, SMTP o IMAP

`EmailCloseSession(<Session>)`
---

**`<Session>: (Cadena de caracteres, o emailPOP3Session, emailSMTPSession o emailIMAPSession Variable)`**

Identifica la sesión de usuario. Corresponde a:

- el nombre de usuario que se utilizó para iniciar la sesión de email ([EmailStartSession](../WDLang3/3032028.md), [EmailStartSMTPSession](../WDLang3/3032025.md) o [EmailStartPOP3Session](../WDLang3/3032022.md)).

- una variable de tipo [emailPOP3Session](../WDLang3/1000018759.md).

- una variable de tipo [emailSMTPSession](../WDLang3/1000018765.md).

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).

### Cierre de una sesión de MS Exchange (Outlook) o Lotus Notes Email

`EmailCloseSession(<Session>)`
---

**`<Session>: (Entero, o emailOutlookSession o emailNotesSession Variable)`**

Identificador del actual período de sesiones de Email. Este valor puede ser:

- un número entero devuelto por [EmailStartSession](../WDLang3/3032028.md) o [EmailStartNotesSession](../WDLang3/3032121.md).

- una variable de tipo [emailOutlookSession](../WDLang3/1000018767.md).

- una variable de tipo [emailNotesSession](../WDLang3/1000018768.md).

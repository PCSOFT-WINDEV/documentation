


## grRotación (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- Girar un gráfico alrededor de uno de sus ejes para ver mejor los datos.. 

- Averiguar el ángulo de rotación de un gráfico en uno de sus ejes. 




El gráfico utilizado puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Performs a slight rotation of 35 degrees
// to highlight the raised effect 
// on a Surface chart
grRotation(CHART_Surface, grAxisX, -35)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Girar un gráfico

`grRotation(<Chart name> , <Axis> , <Angle>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Esta tabla puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.




Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Axis>: (Constante de tipo Integer)`**

Eje de rotación: 


|   |   |
| --- | --- |
| *grAxisX* | Rotación en el eje X. |
| *grAxisY* | Rotación en el eje Y. |
| *grAxisZ* | Rotación en el eje Z. |



**`<Angle>: (Real)`**

Ángulo de rotación en grados. 


<a name="SYNTAX2"></a>

### Averiguar el ángulo de rotación de un gráfico

`<Result> = grRotation(<Chart name> , <Rotation axis>)`
---

**`<Result>: (Real)`**

Ángulo de rotación en grados. 

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Esta tabla puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.




Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Rotation axis>: (Constante de tipo Integer)`**

Eje de rotación: 


|   |   |
| --- | --- |
| *grAxisX* | Rotación en el eje X. |
| *grAxisY* | Rotación en el eje Y. |
| *grAxisZ* | Rotación en el eje Z. |

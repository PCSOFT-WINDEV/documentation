
## DownloadInfo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **DownloadInfo** se utiliza para leer los diferentes detalles sobre una descarga en curso recuperados por [DownloadGetInfo](../WDLang3/1410087320.md). Las características de esta descarga pueden identificarse y modificarse utilizando diferentes propiedades del lenguaje WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Procedure to call at the end of the download
PROCEDURE DownloadCompleted(InfoDownload is DownloadInfo)
IF InfoDownload.Status = downloadStatusSuccessful THEN
    Info(InfoDownload.Title + " has been downloaded successfully."
    RESULT False
END


// ---------------
// Create a DownloadParameter variable 
// to store the download information
ParameterDownload is DownloadParameter

// Specify the parameters of the download
ParameterDownload.Source = ...
	"https://windev.com/storage/en_US/img/visual/25/logo-pcsoft.png"
ParameterDownload.Title = "PC Soft Logo"
ParameterDownload.Description = "PC Soft company logo"
ParameterDownload.Destination = SysDirExternalStorage(1, sseAppDownload)
ParameterDownload.MobileNetworkAllowed = True
ParameterDownload.MeteredNetworkAllowed = False
ParameterDownload.RequiresCharging = True
ParameterDownload.NotificationMode = downloadNotifProgressAndCompletion
// Procedure to call at the end of the download
ParameterDownload.ProcedureEnd = "DownloadCompleted"

// The download is added to the list of downloads to be started
// (The download manager will start the download as soon as possible).
nID is int = DownloadAdd(ParameterDownload)

telInfo is DownloadInfo = DownloadGetInfo(nID)
	
Info(telInfo.Title)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables de DownloadInfo
<a name="propiedades_especificas_descripcion_las_variables_downloadinfo_ELTPARAGRAPHE000032"></a>

Las siguientes propiedades pueden utilizarse para get los detalles de una descarga:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| description | Cadena de caracteres | Descripción de la descarga que aparece en la Notification (si está activada). |
| Destino | Variable de tipo [URI](../WDLang3/1000024027.md) | Address física donde se almacenará el archivo. |
| Estado | Constante de tipo Integer | Estado de la descarga: <br><br>- *downloadStatusFailed*: La descarga falló. El **StatusDetails** Property puede utilizarse para get más detalles.<br><br>- *downloadStatusPending*: Pendiente de descarga.<br><br>- *downloadStatusRunning*: La descarga está en curso.<br><br>- *downloadStatusPaused*: Descarga en pausa. El **StatusDetails** Property puede utilizarse para get más detalles.<br><br>- *downloadStatusSuccessful*: Descargar con éxito.<br><br><br> |
| FileSize | Entero de 8 bytes | Tamaño del archivo en bytes. |
| Fuente | Variable de tipo [URI](../WDLang3/1000024027.md) | http/https Address del archivo que se está descargando. |
| Identificador | Integro | Identificador de la descarga en el gestor de descargas. |
| Progress | Entero de 8 bytes | Número de bytes descargados. |
| StatusDetails | Integro | Detalles de la situación. <br><br>Si el **Estado** Property se establece en *downloadStatusFailed*, el **EstadoDetalles** Property puede utilizarse para get el código de error RFC 2616, si se recibió. <br><br>En caso contrario, corresponderá a una de las siguientes constantes: <br><br>- *downloadErrorInsufficientSpace*: Espacio de almacenamiento insuficiente.<br><br>- *downloadErrorFileAlreadyExists*: La URI apunta a un archivo existente.<br><br>- *downloadErrorUnknown*: Error desconocido.<br><br>- *downloadErrorStorage*: Error de almacenamiento.<br><br>- *downloadPausedWaitingToRetry*: La descarga se detuvo debido a un error de red. El sistema esperará antes de volver a intentarlo.<br><br>- *downloadPausedWaitingForWifi*: La descarga se detiene hasta que se encuentra una red Wi-Fi.<br><br>- *downloadPausedWaitingForNetwork*: La descarga se detiene hasta que se encuentra una red.<br><br>- *downloadPausedUnknown*: La descarga se detiene por una razón desconocida. <br><br><br> |
| Título | Cadena de caracteres | Título de la descarga que aparece en la Notification (si está activada). |

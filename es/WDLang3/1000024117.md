


## &lt;Tabla&gt;.LabelFont (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la fuente utilizada para las etiquetas de las categorías de un gráfico.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="sample_code"></a>

## Ejemplo

<a name="Example1"></a>
<a name="3042012_Example1"></a>

```wl
// Modify the font of labels for the axes of Chart control: CHART_Chart

// Create a specific font
fntLabel is font
fntLabel.Name = "Verdana"
fntLabel.Size = 10
fntLabel.Bold = True

// Change the font
CHART_Chart.LabelFont(fntLabel)
CHART_Chart.Draw()
```

<a name="SYNTAX1"></a>

## Sintaxis

### Definir la fuente utilizada para las etiquetas de los gráficos

`<Chart control>.LabelFont(<Font used>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se encuentra en el editor de reportes.


**`<Font used>: (tipo de letra)`**

Fuente a utilizar. Este parámetro corresponde a una variable de tipo [tipo de letra](../Motscles/1514045.md). Las características de la fuente se definieron mediante la función [FontCreate](../WDLang1/3060002.md) o las [propiedades de fuentes](../Proprietes/2515045.md).


<a name="SYNTAX2"></a>

### Definir la fuente utilizada para las etiquetas de los gráficos (se mantiene la sintaxis para compatibilidad con versiones anteriores)

`<Chart control>.LabelFont(<Font identifier>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico que se encuentra en el editor de reportes.


**`<Font identifier>: (Integro)`**

Identificador de fuente, definido por [grCreateFont](../WDLang3/3042001.md). Un error de WLanguage ocurre si esta fuente no existe.

## Observaciones


### Manejo de las etiquetas de las categorías
<a name="manejo_las_etiquetas_las_categorias_ELTPARAGRAPHE000296"></a>

Se visualizan las etiquetas de categoría:

- por debajo de las marcas horizontales para los gráficos de Line y de acciones (Candlestick y BarCharts),

- debajo del eje X para los gráficos de Columna y MinMax.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)
Las etiquetas de categoría no pueden utilizarse en los gráficos de dispersión.

Las etiquetas de las categorías son definidas por [.CategoryLabel](../WDLang3/1000024072.md).

La fuente de la etiqueta se puede modificar con la función **&lt;Tabla&gt;.LabelFont**.

Las etiquetas de categorías definidas con la función [.CategoryLabel](../WDLang3/1000024072.md) pueden mostrarse en el texto del gráfico Circular (constante *grShowLabel* utilizada con la función [&lt;Tabla&gt;.etiqueta](../WDLang3/1000024165.md)).<a name="NOTE0_2"></a>


### Inicialización de fuentes
<a name="inicializacion_fuentes_ELTPARAGRAPHE000337"></a>

La fuente Default utilizada para las etiquetas tiene las siguientes características:

- Nombre fuente: Arial

- envergadura: 14

- Estilo: Normal

- Color: Black



<a name="NOTE0_3"></a>


### Orientación de la fuente
<a name="orientacion_fuente_ELTPARAGRAPHE000352"></a>

Para las etiquetas de un gráfico, la "orientación" Property de la fuente es ignorada. 

Para cambiar la orientación de la fuente en las etiquetas, la description de la control Gráfico debe ser modificada en la ventana de 7-tab. 

Por Default, la orientación es elegida automáticamente. Esta orientación tiene en cuenta el tamaño y el número de etiquetas a mostrar.

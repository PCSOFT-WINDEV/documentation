


## Funciones BTLE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para manipular los dispositivos Bluetooth de baja energía: 



|   |   |
| --- | --- |
| [BTLECaracterísticaCambio de valor](../WDLang3/1000022071.md) | Habilita o deshabilita la detección del cambio de valor de una característica de bajo consumo de energía de Bluetooth. |
| [BTLECharacteristicReadValue](../WDLang3/1000022069.md) | Lee el valor de una característica en un dispositivo Bluetooth de bajo consumo. |
| [BTLECharacteristicWriteValue](../WDLang3/1000022070.md) | Escribe un valor en una característica de un dispositivo Bluetooth Low Energy. |
| [BTLEConnect](../WDLang3/1000021884.md) | Se utiliza para conectarse a un dispositivo Bluetooth de bajo consumo. |
| [BTLEConnectionStatus](../WDLang3/1000022068.md) | Devuelve el estado actual de la conexión a un dispositivo Bluetooth de bajo consumo o pide que se le notifique cuando cambia el estado de la conexión. |
| [BTLEDescriptorReadValue](../WDLang3/1000022072.md) | Lee el valor de un descriptor en un dispositivo Bluetooth de bajo consumo. |
| [BTLEDescriptorWriteValue](../WDLang3/1000022073.md) | Escribe el valor de un descriptor en un dispositivo Bluetooth Low Energy. |
| [BTLEDisconnect](../WDLang3/1000021883.md) | Se utiliza para desconectarse de un dispositivo Bluetooth de bajo consumo. |
| [BTLEGetACaracterística](../WDLang3/1000022113.md) | Recupera la característica de un servicio Bluetooth de baja energía correspondiente al identificador especificado (UUID). |
| [BTLEGetADescriptor](../WDLang3/1000022112.md) | Recupera el descriptor de una característica de baja energía de Bluetooth correspondiente al identificador especificado (UUID). |
| [BTLEGetAllCaracterísticas](../WDLang3/1000022115.md) | Recupera todas las características (con el mismo identificador) associated con un servicio BLE. |
| [BTLEGetAllServices](../WDLang3/1000022114.md) | Recupera todos los servicios correspondientes a un UUID. |
| [BTLEGetAService](../WDLang3/1000022111.md) | Recupera el servicio de un dispositivo Bluetooth de baja energía correspondiente al identificador especificado (UUID). |
| [BTLEInitialize](../WDLang3/1000022110.md) | Inicializa Bluetooth Low Energy (necesario sólo en Windows e iOS) |
| [BTLEInitializeDevice](../WDLang3/1410087620.md) | Inicializa un dispositivo Bluetooth Low Energy en Windows. |
| [BTLEListDevice](../WDLang3/1000021882.md) | Devuelve la lista de dispositivos Bluetooth Low Energy. |
| [BTLEStatus](../WDLang3/1000021885.md) | Devuelve el estado actual de activación de Bluetooth en el dispositivo o pide que se le notifique cuando cambie el estado de activación. |



Los siguientes tipos de variables se utilizan para manipular los dispositivos Bluetooth de baja energía:   



|   |   |
| --- | --- |
| [btleCharacteristic](../WDLang3/1000022060.md) | El tipo **btleCharacteristic** se utiliza para manipular una característica de un servicio de Bluetooth de baja energía. |
| [btleDescriptor](../WDLang3/1000022064.md) | El tipo **btleDescriptor** se utiliza para manipular el descriptor de una característica de baja energía de Bluetooth |
| [btleDevice](../WDLang3/1000022054.md) | El tipo **btleDevice** le permite manipular un dispositivo que utiliza la tecnología Bluetooth Low Energy. |
| [btleService](../WDLang3/1000022057.md) | El tipo **btleService** se utiliza para manipular un servicio de Bluetooth de baja energía ("Servicio Bluetooth GATT"). |


## LEDSwitchOn( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enciende el LED (diodo emisor de luz) del aparato.. Este LED puede ser apagado por [LEDSwitchOff](../WDLang3/1000019447.md). 

Atención: 

- El dispositivo debe estar equipado con un LED.

- Esta función no tiene ningún efecto en el simulador o en el emulador.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch on the diode in red for 1 second
LEDSwitchOn(LightRed, 1000)

// Turn off LED
LEDSwitchOff()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LEDSwitchOn(<Color> [, <Duration> [, <Blinking>]])`
---

**`<Color>: (Integro)`**

Color del LED. Este color puede corresponder a: 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).


**Atención**: Algunos dispositivos no permiten modificar el LED Color o no proponen todos los Color s. Se utilizará una Color predeterminada (o una Color similar) si el dispositivo no soporta la Color especificada.

**`<Duration>: (Entero o duración opcional)`**

Duración (en milisegundos) durante la cual debe encenderse el diodo. 

- Si este parámetro es set a 0, el diodo se encenderá hasta la llamada a [LEDSwitchOff](../WDLang3/1000019447.md) o hasta que se cierre la aplicación. 

- Si no se especifica este parámetro, el diodo se encenderá durante 3 segundos.


Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 




**`<Blinking>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si el LED debe parpadear, 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si el LED debe permanecer fijo. La frecuencia de parpadeo puede cambiar de un dispositivo a otro.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento de los LEDs
<a name="modo_funcionamiento_los_leds_ELTPARAGRAPHE000087"></a>

El modo de funcionamiento de los LEDs difiere según el tipo de dispositivo utilizado: 

- Algunos dispositivos no permiten modificar el LED Color o no proponen todos los Color s. Se utilizará una Color predeterminada (o una Color similar) si el dispositivo no soporta la Color especificada.

- En algunos dispositivos, el LED sólo se enciende cuando la pantalla está apagada.

- En algunos dispositivos, cuando son relacionado al PC por USB o conectado, el LED se utiliza como luz de advertencia de carga. En este caso, la función **LEDSwitchOn** no tiene ningún efecto.

- La iluminación del LED se interrumpirá si el usuario amplía la ventana de notificaciones en el dispositivo.

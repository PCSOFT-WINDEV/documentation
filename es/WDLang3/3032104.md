
## LDAPListAttribute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los atributos de objeto en un servidor LDAP.

**Observación**: Esta función se utiliza para recuperar localmente todo el objeto LDAP (atributos y valores de atributos).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032104&name=ldaplistattribute_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Find the attributes
sListAttributes is string
sListAttributes = LDAPListAttribute("MySession", "DC=pcs,DC=local")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPListAttribute(<LDAP session> , <Object DN>)`
---

**`<Result>: (Cadena de caracteres)`**



- Lista de atributos,

- Cadena vacía ("") si ningún atributo es associated con el objeto. Los diferentes atributos están separados por caracteres "CR" (Carriage Return). 


Si se produce un error, el valor de &lt;Resultado&gt; es una cadena vacía ("") y la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.

**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP utilizada.

**`<Object DN>: (Cadena de caracteres)`**

Nombre Distinguido (DN) del objeto cuyos atributos se buscan.

<a name="NOTE0_1"></a>

## Observaciones




### Servidor LDAP y objeto LDAP local
<a name="servidor_ldap_objeto_ldap_local_ELTPARAGRAPHE000051"></a>

El servidor LDAP contiene objetos LDAP.

El objeto LDAP local es un objeto LDAP que se obtiene del servidor.
<a name="NOTE0_2"></a>




### Actualización del objeto LDAP local
<a name="actualizacion_del_objeto_ldap_local_ELTPARAGRAPHE000061"></a>

Cuando se ejecuta una función LDAP de lectura  (**LDAPListAttribute**, [LDAPNbValor](../WDLang3/3032106.md), [Valor LDAPValor](../WDLang3/3032107.md)), el objeto del servidor se devuelve completamente al equipo local para limitar la comunicación con el servidor. Por lo tanto, si estas funciones de lectura se llaman varias veces con el mismo DN, la comunicación con el servidor se establece en la primera llamada.. Las otras llamadas leerán el objeto local.

**LDAPListAttribute** actualiza el objeto LDAP local sólo si el DN del objeto pasado como parámetro es diferente del último DN pasado como parámetro a una de las siguientes funciones: **LDAPListAttribute**, [LDAPNbValor](../WDLang3/3032106.md), [Valor LDAPValor](../WDLang3/3032107.md).

**LDAPListAttribute** no actualiza el objeto LDAP local si el DN pasado como parámetro es el mismo que el último DN pasado como parámetro a una de las siguientes funciones: **LDAPListAttribute**, [LDAPNbValor](../WDLang3/3032106.md), [Valor LDAPValor](../WDLang3/3032107.md). Sin embargo, si se han utilizado [LDAPValidateAdd](../WDLang3/3032111.md) y [LDAPValidateModify](../WDLang3/3032117.md) con este mismo DN, el objeto local se actualiza automáticamente.


## FaxDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un fax del administrador de trabajos de impresión y lo cancela.. Entonces, [FaxStatus](../WDLang3/3037001.md) ya no puede ser utilizada en este fax.

**Observación**: Esta función se puede utilizar en Windows 2000, XP o posterior.. En Windows Vista (y posteriores), la gestión de faxes está disponible con Windows Professional y Windows Integral Edition.
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxDelete(<Connection identifier> , <Fax name or Fax identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se elimina el fax,

- <u><u><u><u>False</u></u></u></u> si se produce un error (si el nombre del fax no existe, por ejemplo).




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md).

**`<Fax name or Fax identifier>: (Cadena de caracteres)`**

Corresponde a:

- el nombre utilizado para identificar el fax saliente. Este nombre se definió con [FaxSend](../WDLang3/3037006.md).

- el identificador del fax (devuelto por [FaxOutbox](../WDLang3/3037008.md) o [Bandeja de entrada de fax](../WDLang3/3037009.md))






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función también se puede aplicar a los faxes que se transmiten actualmente. En este caso, la transmisión se cancela.

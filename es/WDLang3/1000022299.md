


## &lt;emailIMAPSession Variable&gt;.RemoveFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una carpeta en un servidor IMAP.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// IMAP protocol
MySession is emailIMAPSession
...
// Delete a folder from the current folder 
MySession.RemoveFolder(MySession.CurrentFolder() + "\Work")

// Delete a folder from the custom folder
MySession.RemoveFolder("Personal folders\Inbox\MyOwnWork")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.DeleteFolder(<Folder path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailIMAPSession)`**

Nombre de la [emailIMAPSession](../WDLang3/1000018957.md) Variable correspondiente a la sesión Email a utilizar.

**`<Folder path>: (Cadena Unicode)`**

Ruta de la carpeta a eliminar.

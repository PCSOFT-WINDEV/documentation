
## geoPosition (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **geoPosition** se utiliza para las posiciones geográficas de manipular a través de funciones de geolocalización WLanguage. Una posición geográfica se define por una latitud, una longitud y una fecha de medición.. También puede incluir información sobre la altitud, la velocidad, la dirección y/o la precisión de la medición.

El tipo **geoPosition** permite definir las características avanzadas de una ubicación geográfica. Las características de esta posición pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
pos1 is geoPosition
pos1.Latitude = 1.442951
pos1.Longitude = 43.604363

pos2 is geoPosition
pos2.Latitude = 2.505874
pos2.Longitude = 50.458744

InfoBuild("The distance between the two positions is %1 meters", geoDistance(pos1, pos2))
```

## Observaciones


### Propiedades específicas de las variables geoPosition
<a name="propiedades_especificas_las_variables_geoposition_ELTPARAGRAPHE000073"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **geoPosition**:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| AccuracyValid | booleano | <u><u><u><u>True</u></u></u></u> si la ubicación actual contiene información de precisión válida.<br><br>La propiedad **Accuracy** se considera válida en los siguientes casos:<br><br>- fue asignado por programación.<br><br>- fue inicializado por el proveedor de la localización.<br><br><br>**AccuracyValid** es accesible en modo de sólo lectura. |
| Altitud | Real | Altitud de la posición (en metros).<br><br>Esta Property se especifica sólo cuando está soportada por el proveedor de la localización. |
| AltitudeValid | booleano | <u><u><u><u>True</u></u></u></u> si la ubicación actual contiene información de altitud válida.<br><br>**Altitude** se considera válida en los siguientes casos:<br><br>- fue asignado por programación.<br><br>- fue inicializado por el proveedor de la localización.<br><br><br>**AltitudeValid** es accesible en modo de sólo lectura. |
| Dirección | Real | Dirección de movimiento del dispositivo (en grados al Este de True Norte).<br><br>Observaciones: <br><br>- El valor de **Direction** se tiene en cuenta a menos que el dispositivo se mueva demasiado despacio.<br><br>- Esta Property se especifica sólo cuando está soportada por el proveedor de la localización.<br><br><br> |
| DirectionValid | booleano | <u><u><u><u>True</u></u></u></u> si la ubicación actual contiene información de dirección válida.<br><br>**Direction** se considera válida en los siguientes casos:<br><br>- fue asignado por programación.<br><br>- fue inicializado por el proveedor de la localización.<br><br><br>**DirectionValid** es accesible en modo de sólo lectura. |
| Latitud | Real | Latitud de la posición (en grados).<br><br>Los valores posibles van desde -90° (Polo Sur) a +90° (Polo Norte). |
| Longitude | Real | Longitud de la posición (en grados).<br><br>Los posibles valores van desde -180° (longitud oeste) a +180° (longitud este) en relación con el meridiano de Greenwich. |
| MeasurementDate | Variable de tipo DateTime o cadena de caracteres | Fecha y hora de la medición. Esta fecha y esta hora corresponden a la hora local del dispositivo.. Para get la hora en formato UTC, utilice [DateTimeLocalToUTC](../WDLang1/3027036.md).<br><br> |
| PositionValid | booleano | <u><u><u><u>True</u></u></u></u> si se ha especificado la latitud y/o la longitud para la ubicación actual. <br><br>**PositionValid** es accesible en modo de sólo lectura. |
| Precisión | Real | Precisión de la medición (en metros).<br><br>Esta Property se especifica sólo cuando está soportada por el proveedor de la localización.<br><br> |
| SpeedValid | booleano | <u><u><u><u>True</u></u></u></u> si la ubicación actual contiene información de velocidad válida.<br><br>**Speed** se considera válida en los siguientes casos:<br><br>- fue asignado por programación.<br><br>- fue inicializado por el proveedor de la localización.<br><br><br>**SpeedValid** es accesible en modo de sólo lectura. |
| Velocidad | Real | Velocidad de avance del dispositivo móvil (en metros/segundo). <br><br>Esta Property se especifica sólo cuando está soportada por el proveedor de la localización. |

### Funciones que utilizan variables geoPosition
<a name="funciones_que_utilizan_variables_geoposition_ELTPARAGRAPHE000236"></a>




|   |   |
| --- | --- |
| [geoAzimuth](../WDLang3/1000019212.md) | Devuelve la dirección representada por un lugar en relación con otro. |
| [geoDistance](../WDLang3/1000019211.md) | Devuelve la distancia entre los dos lugares. Esta distancia está expresada en metros por Default. |
| [geoRunApp](../WDLang3/1000019954.md) | Inicia la aplicación nativa de mapas del dispositivo para mostrar: <br><br>- una ubicación geográfica específica, <br><br>- una ruta entre dos lugares.<br><br><br> |
| [GPSGetPosition](../WDLang3/1000019205.md) | Recupera información sobre la posición actual del dispositivo. |
| [GPSLastPosition](../WDLang3/1000019239.md) | Recupera la información sobre la última posición conocida del dispositivo. |
| [MapGetPosition](../WDLang3/1000019790.md) | Devuelve la posición geográfica de un punto localizado en el centro del mapa que se está mostrando en un control Map. |
| [MapInfoXY](../WDLang3/1000019950.md) | Devuelve la posición geográfica (latitud y longitud) correspondiente a un punto de una control Mapa. |

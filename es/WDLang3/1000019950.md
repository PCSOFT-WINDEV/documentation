


## MapInfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición geográfica (latitud y longitud) correspondiente a un punto de una control Mapa.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a marker at the click location in a Map control 
// (code to include in the click code of control)
MyLocation is geoPosition
MyPosition = MapInfoXY(MAP_Position, MouseXPos(), MouseYPos())
IF ErrorOccurred = False THEN
	MyMarker is Marker
	MyMarker.Position = MyLocation
	MapAddMarker(MAP_Position, MyMarker)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapInfoXY(<Map control> , <X> , <Y>)`
---

**`<Result>: (geoPosition Variable)`**

[geoPosition](../WDLang3/1000019191.md) variable correspondiente a la posición geográfica del punto en el mapa. 
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<X>: (Integro)`**

Coordenada X (en píxeles) del punto a estudiar. Esta coordenada es relativa a la esquina superior izquierda de la control. Si esta coordenada es negativa, se buscará la posición para un punto que no se muestra en el mapa.. 

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) del punto a estudiar. Esta coordenada es relativa a la esquina superior izquierda de la control. Si esta coordenada es negativa, se buscará la posición para un punto que no se muestra en el mapa.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar las coordenadas de un punto de la control Mapa en el que se ejecutó una Event (por ejemplo, haga clic): 

	- utilizar las funciones [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md). 




- El cálculo de la posición tiene en cuenta el nivel de zoom en la control Mapa durante la llamada a la función. Cuanto más alto sea el nivel de zoom, más precisa será la posición.

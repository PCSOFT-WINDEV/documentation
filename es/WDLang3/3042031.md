


## grDestinationWnd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define una ventana como destino del gráfico. Durante la siguiente llamada a [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en una ventana independiente. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define a window whose title is "Purchases" as destination of "MyChart"
grDestinationWnd("MyChart", "", "Purchase")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDestinationWnd(<Chart name> , <"Parent" window> , <Window title>)`
---

**`<Chart name>: (Cadena de caracteres)`**

Nombre del gráfico a utilizar, definido por [grCreate](../WDLang3/3042005.md).

**`<"Parent" window>: (Cadena de caracteres)`**

Nombre de la ventana "padre" de la ventana carta. Si este parámetro corresponde a una cadena vacía ("""), la ventana del gráfico se colocará delante de la ventana actual.

**`<Window title>: (Cadena de caracteres)`**

Título de la nueva ventana donde se dibujará el gráfico durante la llamada a [grDraw](../WDLang3/3042026.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso y límites
<a name="condiciones_uso_limites_ELTPARAGRAPHE000043"></a>

- El gráfico se dibuja en una ventana independiente. Esta ventana se abrirá (durante la llamada a [grDraw](../WDLang3/3042026.md)) delante de la ventana actual. La posición de esta ventana puede ser definida por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md). 

- Esta función no se puede utilizar con los controles gráficos. 

- Esta función no se puede utilizar en la editor de reportes.



<a name="NOTE0_2"></a>


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000058"></a>

Para dibujar un gráfico:

- crear el gráfico ([grCreate](../WDLang3/3042005.md)),

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- definir el destino de la carta con las siguientes funciones: 

	- [grDestinationBMP](../WDLang3/3042018.md), 

	- [grDestinoControl](../WDLang3/3042020.md), 

	- [grDestinationEMF](../WDLang3/3042017.md), 

	- **grDestinationWnd**,

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinationWMF](../WDLang3/3042019.md). 




- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).


Durante la siguiente llamada a [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en una ventana independiente.
<a name="NOTE0_3"></a>


### Cambio de destino
<a name="cambio_destino_ELTPARAGRAPHE000111"></a>

Si el gráfico se mostró previamente en un control, este gráfico se borrará del control durante la siguiente llamada al [grDraw](../WDLang3/3042026.md).

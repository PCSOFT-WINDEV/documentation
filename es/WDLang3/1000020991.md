


## &lt;upnpDevice Variable&gt;.ModifyMute (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el estado'Mute' de un dispositivo UPnP.

**Atención**: Para utilizar esta función, el dispositivo debe incluir un servicio "RenderingControl"..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrDevice is array of upnpDevices 
arrDevice = UPNPListDevice(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
FOR EACH Device of arrDevice
	Device.ModifyMute(True)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <UPnP device>.ModifyMute(<New status>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el modo de silencio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<UPnP device>: (Variable de tipo upnpDevice)`**

Nombre de [upnpDevice](../WDLang3/1000020903.md) Variable correspondiente al dispositivo UPnP a utilizar.

**`<New status>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el dispositivo debe estar en silencio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

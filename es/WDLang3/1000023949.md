


## &lt;xsdEntity Variable&gt;.AddAttribute (Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un atributo a un automatic Variable generado por la importación de un Webservice.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
xsdJob is DefinedReportJob
xsdJob.AddAttribute("xsi:type", xsdJob..Type)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Variable>.AddAttribute(<Name> , <Value>)`
---

**`<Variable>: (Variable generada por la importación de un Webservice)`**

automatic Variable del Servicio Web para modificar.

**`<Name>: (Cadena Unicode)`**

Nombre del atributo a añadir.

**`<Value>: (Cadena Unicode)`**

Valor del atributo a añadir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000195"></a>

Cuando se importa un Webservice a un proyecto, WINDEV, WEBDEV y WINDEV Mobile generan automáticamente los tipos de variables que permiten a manipular el flujo XML utilizado por el protocolo del Webservice.. 

**&lt;xsdEntity Variable&gt;.AddAttribute** añade un atributo que no estaría declarado en el WSDL (y por tanto no se generaría automáticamente) a un Variable de uno de los tipos importados.

**Observación**: La función **&lt;xsdEntity Variable&gt;.AddAttribute** es una función avanzada. En la mayoría de los casos, el WSDL declara todos los elementos necesarios para el uso de los Webservices.

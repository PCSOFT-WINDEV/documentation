


## <Chart>.RBAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade datos a un gráfico de barras radiales.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.RBAddData("Montpellier", 10)
CHART_Chart1.RBAddData("France", 50)
CHART_Chart1.RBAddData("Europe", 80)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.RBAddData(<Category> , <Value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (en la ventana o editor de reportes).

**`<Category>: (Cadena de caracteres)`**

Nombre de la categoría associated con los datos.

**`<Value>: (Entero o real)`**

Valor añadido. 
Por Default, el valor se considera como un porcentaje. Si no, puede especificar el valor "máximo" con [.RBMaximumValue](../WDLang3/1410088094.md) (el porcentaje se calculará automáticamente a partir de este valor).

## Observaciones


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000190"></a>

[.AddData](../WDLang3/1000023533.md) se utiliza para añadir datos en un gráfico (de tarta, de columnas, ...).

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [.StockAddData](../WDLang3/1000023565.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [&lt;Gráfico&gt;.HMAddData](../WDLang3/1000024079.md).

- Para los gráficos de dispersión, se recomienda utilizar [.ScatterAddDataXY](../WDLang3/1000024105.md).

- Para las cartas Sunburst, se recomienda utilizar [&lt;Gráfico&gt;.SunburstAddData](../WDLang3/1000024132.md). 

- Para los gráficos de superficie, se recomienda utilizar [&lt;Tabla&gt;.SurfaceAddData](../WDLang3/1000024143.md).




Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [.IncreaseData](../WDLang3/1000024086.md).


## FTPCurrentDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o modifica el directorio actual en un servidor FTP (File Transfer Protocol).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003040003&name=ftpcurrentdir_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Identify the current directory on the FTP server
ResCurrentDir = FTPCurrentDir(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el directorio actual

`<Result> = FTPCurrentDir(<Connection identifier>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre absoluto del directorio actual en el servidor FTP,

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [FTPConnect](../WDLang3/3040016.md).




<a name="SYNTAX2"></a>

### Modificar el directorio actual

`<Result> = FTPCurrentDir(<Connection identifier> , <New current directory>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el directorio actual fue modificado,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [FTPConnect](../WDLang3/3040016.md).

**`<New current directory>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del nuevo directorio actual. Las secciones de la ruta están separadas por barras oblicuas ("/").

## Observaciones




### Timeout
<a name="timeout_ELTPARAGRAPHE000093"></a>

De forma predeterminada, todas las funciones FTP fallan si no se recibe ninguna respuesta del servidor FTP después de 20 segundos. Este tiempo de espera se puede modificar al conectarse al servidor FTP con la función [FTPConnect](../WDLang3/3040016.md).
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>




### Permisos de usuarios
<a name="permisos_usuarios_ELTPARAGRAPHE000122"></a>

Sólo un usuario que tenga derechos de escritura en el servidor FTP puede manipular los archivos o directorios encontrados en un servidor FTP.. En la mayoría de los casos, no se conceden derechos de escritura a los usuarios "anónimos".




## &lt;emailIMAPSession Variable&gt;.CloseSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la sesión de IMAP. 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
MySession is emailIMAPSession

MySession.ServerAddress = "imap.mycompany.us"
MySession.Name = "user"
MySession.Password = EDT_Password

MySession.StartSession()
...
MySession.CloseSession()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<IMAP session>.Sesión cerrada()`
---

**`<IMAP session>: (emailIMAPSession variable)`**

Nombre de la [emailIMAPSession](../WDLang3/1000018957.md) Variable correspondiente a la sesión Email a utilizar.

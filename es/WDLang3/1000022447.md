


## &lt;Mapa&gt;.InfoPosición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las coordenadas (en píxeles) del punto correspondiente a una posición geográfica (latitud y longitud.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the point in the map corresponding to the current user position
MyLocation is geoPosition
MyLocation.Latitude = 1.442951
MyLocation.Longitude = 43.604363
Trace(MAP_Position.InfoPosition(MyLocation))
```


<a name="Example2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
// Returns the point in the map corresponding to the current user position
MyLocation is geoPosition
MyLocation = GPSGetPosition()
IF ErrorOccurred = False THEN
	RESULT MAP_Position.InfoPosition(MyLocation)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.InfoPosition(<Position>)`
---

**`<Result>: (Cadena de caracteres)`**

Coordenadas del punto correspondiente a la posición geográfica. Las coordenadas se devuelven en el siguiente formato: 

```txt
<X> + TAB + <Y>
```


**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<Position>: (geoPosition Variable)`**

Nombre de una Variable de tipo [geoPosition](../WDLang3/1000019191.md) correspondiente a la posición geográfica con las coordenadas deseadas en el mapa.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las coordenadas devueltas por la función **&lt;Mapa&gt;.InfoPosición** pueden corresponder a un punto que no es visible en el control Mapa (coordenadas negativas, por ejemplo).

- El cálculo de la posición tiene en cuenta el nivel de zoom en la control Mapa durante la llamada a la función. Cuanto más alto sea el nivel de zoom, más precisas serán las coordenadas.

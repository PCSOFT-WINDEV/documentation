


## La estructura de la tarea
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000236"></a>
La estructura de tareas es una estructura preestablecida de WLanguage (no se requiere ninguna declaración).. Esta estructura está acostumbrada:

- crear o modificar una tarea.

- recuperar el contenido de una tarea.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las diferentes variables de la estructura de la tarea son las siguientes
<a name="las_diferentes_variables_estructura_tarea_son_las_siguientes_ELTTEXTE000260"></a>
La estructura contiene los siguientes miembros:


|   |   |
| --- | --- |
| **mFecha límite de la tarea** | FechaHora.<br>Fecha y hora de vencimiento de la tarea. |
| **mTarea. Repita** | booleano. <br>Indica si la tarea es una tarea periódica. |
| **mTareaParticipanteBcc** | Array de cadenas de caracteres. <br>Nombre de los participantes ocultos y de Invisible en la tarea. <br><br>La siguiente sintaxis permite acceder a los diferentes participantes ocultos y de Invisible de la tarea: <br>mTask.ParticipantBcc[&lt;Número de participante&gt;]<br>Donde &lt;Participante num&gt; corresponde al número del participante oculto y de Invisible solicitado. <br><br>El número de participantes ocultos y Invisible a la tarea es devuelto por el mTask.NbParticipantBcc Variable. |
| **mTareaParticipanteCc** | Array de cadenas de caracteres. <br>Nombre de los participantes ocultos en la tarea. <br><br>La siguiente sintaxis le permite acceder a los diferentes participantes ocultos de la tarea: <br>mTask.ParticipantCc[&lt;Número de participante&gt;]<br>Donde &lt;Participante num&gt; corresponde al número del participante oculto solicitado. <br><br>El número de participantes ocultos en la tarea es devuelto por el mTask.NbParticipantCc Variable. |
| **mTask.AutoReminder** | booleano.<br>Indica si se debe hacer un automatic Reminder para la tarea. Este Reminder se hará en &lt;mTask.Reminder Date... |
| **mTask.Category** | Cadena de caracteres.<br>set de palabras clave separadas por un punto y coma y utilizadas para definir las categorías relacionado a la tarea. |
| **mTask.EndDate** | salir con.<br>Fecha de finalización de la tarea. |
| **mTask.ID** | Cadena de caracteres.<br>Identificador único de la tarea. |
| **mTask.Importance** | Constante de tipo Integer.<br>Nivel de importancia para la tarea:<br><br>- *taskImportanceLow*<br><br>- *taskImportanceHigh*<br><br>- *taskImportanceUndefined*<br><br>- *taskImportanceMedium*<br><br><br> |
| **mTask.NbParticipanteBcc** | Integro.<br>Número de participantes ocultos y de Invisible en la tarea. |
| **mTask.Notes** | Cadena de caracteres.<br>Notas sobre la tarea. |
| **mTask.Out** | booleano. <br>Indica si hay una tarea actual. |
| **mTask.recordatorioFecha** | FechaHora.<br>Fecha y hora en que la tarea Reminder aparecerá automáticamente.<br>Esta variable se tiene en cuenta sólo si se debe hacer un automatic Reminder (mTask.AutoReminder). |
| **mTask.StartDate** | salir con.<br>Fecha de inicio de la tarea. |
| **mTask.Status** | Constante de tipo Integer.<br>Estado de la tarea:<br><br>- *taskStateCanceled*<br><br>- *taskStatePaused*<br><br>- *taskStateInProgress*<br><br>- *taskStateLate*<br><br>- *taskStateNotStarted*<br><br>- *taskStateCompleted*<br><br><br> |
| **mTask.Subject** | Cadena de caracteres.<br>Tema de la tarea. |
| **NbParticipante en la tarea** | Integro.<br>Número de participantes en la tarea. |
| **NbParticipanteCc de mTask.Nb** | Integro.<br>Número de participantes ocultos en la tarea. |
| **Participante en la tarea** | Array de cadenas de caracteres. <br>Nombre de los participantes en la tarea.<br><br>La siguiente sintaxis le permite acceder a los diferentes participantes de la tarea: <br>mTask.Participant[&lt;Número de participante&gt;]<br>Donde &lt;Participante num&gt; corresponde al número del participante solicitado. <br><br>El número de participantes en la tarea es devuelto por el mTask.NbParticipante Variable. |


## tapiCallEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha y hora de finalización de la llamada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DateTimeEndCall is DateTime
DateTimeEndCall = tapiCallEnd()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiCallEnd([<Call identifier>])`
---

**`<Result>: (FechaHora)`**

Fecha y hora de finalización de la llamada.

**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). La llamada actual se gestiona si no se especifica este parámetro.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Disponibilidad de información
<a name="disponibilidad_informacion_ELTPARAGRAPHE000037"></a>

La información está disponible:

- en cualquier momento para la última llamada saliente

- hasta el final de la llamada para las llamadas entrantes. Si esta información debe ser almacenada, debe ser guardada antes del final de la llamada.. La información estará disponible hasta que el estado del procedimiento WLanguage sea la constante *tapiLineDisconnected*.



<a name="NOTE0_2"></a>


### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000174"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.

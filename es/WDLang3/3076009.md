


## WINDEV y la telefonía
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
WINDEV permite gestionar fácilmente las llamadas telefónicas entrantes y salientes a través de ciertas funciones WLanguage específicas.

Estas funciones le permiten gestionar un buzón de voz, un contestador automático (u otros) directamente desde una aplicación WINDEV.





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000121"></a>


### Condición previa
<a name="condicion_previa_ELTPARAGRAPHE000027"></a>

Para utilizar las funcionalidades de telefonía, debe tener:

- un módem.

	- Para guardar y reproducir grabaciones, este módem debe incluir soporte de voz.

	- Para identificar a la persona que llama, el módem debe ser compatible con el identificador de llamadas (Caller ID).




- el protocolo Tapi 2.0 o Tapi 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.



<a name="NOTE2_3"></a>


### Comprobación de la configuración
<a name="comprobacion_configuracion_ELTPARAGRAPHE000040"></a>

Para comprobar la configuración de su ordenador, le recomendamos que utilice **WDTelDiagno.exe**, suministrado con WINDEV. Esta herramienta se utiliza para listar las líneas TAPI disponibles y para especificar la línea que será utilizada por Default por las funciones de telefonía. Esto le permite detectar rápidamente los conflictos que pueden producirse entre las diferentes líneas TAPI.

**Observación**: Las líneas de TAPI difieren de las líneas estándar de Phone.

Antes de utilizar las funciones de telefonía, puede definir mediante programación el dispositivo TAPI en el que deben detectarse o marcarse las llamadas. Utilice las funciones:



|   |   |
| --- | --- |
| [tapiCapability](../WDLang3/3076014.md) | Devuelve las características de un dispositivo de telefonía. |
| [tapiDevice](../WDLang3/3076032.md) | Selecciona el dispositivo TAPI que se utilizará durante las siguientes operaciones de telefonía: <br><br>- [tapiDial](../WDLang3/3076013.md),<br><br>- [tapiListen](../WDLang3/3076029.md).<br><br><br> |
| [tapiDeviceList](../WDLang3/3076024.md) | Enumera los dispositivos compatibles con TAPI 2.0 y TAPI 3.1 instalados en el ordenador actual. |

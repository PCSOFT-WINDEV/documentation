


## &lt;websocketClient Variable&gt;.Send (Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía un mensaje del servidor WebSocket al cliente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve list of clients
arrClients is array of websocketClient = WebSocketListConnectedClient()
// To send a message to each client
FOR EACH oClient OF arrClients
	oClient.Send(bufMessage)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`Sends(<Client> , <Message>)`
---

**`<Client>: (websocketClient Variable)`**

Nombre de la [websocketClient](../WDLang3/1000025952.md) Variable que corresponde al cliente al que se enviará el mensaje.

**`<Message>: (Cadena de caracteres)`**

Mensaje para enviar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Esta función es asincrónica**: simplemente solicita el envío del mensaje al cliente. En realidad, el mensaje puede enviarse más tarde (después de la ejecución de **&lt;websocketClient Variable&gt;.Send**). La función **&lt;websocketClient Variable&gt;.Send** nunca devuelve errores (por ejemplo, si el cliente se desconectó).

- El protocolo WebSocket soporta dos formatos de mensajes: binario o cadena (UTF-8 codificado). **&lt;websocketClient Variable&gt;.Send** elige automáticamente el formato en función del tipo de parámetro recibido.

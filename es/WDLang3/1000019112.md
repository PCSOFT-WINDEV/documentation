


## EmailChangeStatus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el estado de una Email en un servidor de mensajería.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch all emails of an IMAP session to read
MySession is emailIMAPSession
...
FOR EACH EmailMessage NOT READ OF MySession
	EmailChangeStatus(MySession, EmailMessage, emailStatusRead)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EmailChangeStatus(<Session> , <Email> , <New status>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el estado en el servidor,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Session>: (emailOutlookSession o emailIMAPSession Variable)`**

Identifica la sesión de usuario. Corresponde a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [emailOutlookSession](../WDLang3/1000018767.md).

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).




**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable correspondiente al Email a utilizar.

**`<New status>: (Constante de tipo Integer)`**

Estado a asignar a la Email, los valores posibles son:


|   |   |
| --- | --- |
| *emailStatusNotRead* | Indica que el mensaje no se ha leído. |
| *emailStatusRead* | Indica que se ha leído el mensaje. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no está disponible: 

- para las sesiones de Notes ([emailNotesSession](../WDLang3/1000018768.md) Variable),

- para las sesiones POP3 ([emailPOP3Session](../WDLang3/1000018759.md) Variable),

- para las sesiones SMTP ([emailSMTPSession](../WDLang3/1000018765.md) Variable).

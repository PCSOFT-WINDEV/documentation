
## BTAcceptConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si una radio Bluetooth acepta (o no) las peticiones de conexión procedentes de los dispositivos.

**Observación**: Una radio Bluetooth corresponde a un transmisor Bluetooth. Este transmisor puede corresponder a una llave USB Bluetooth o puede ser un elemento incluido en un ordenador (por ejemplo, un portátil).




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Sintaxis que puede ser usada en Windows

`<Result> = BTAcceptConnection(<Radio identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la radio Bluetooth acepta las solicitudes de conexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Radio identifier>: (Integro)`**

Identificador de la radio Bluetooth a utilizar. Este identificador es devuelto por [BTListRadio](../WDLang3/1000017125.md).


## AppointmentDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una Appointment en la aplicación nativa de gestión de Appointment s que se encuentra en el dispositivo móvil (Android o iOS).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF NOT AppointmentDisplay(apt) THEN
	Error("Unable to display the appointment.", ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppointmentDisplay(<Appointment>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró la descripción de la cita,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Appointment>: (Appointment Variable)`**

Nombre de la variable [Appointment](../WDLang1/1000019244.md) correspondiente a la cita cuyo description se mostrará. Este Appointment (listado por [AppointmentList](../WDLang3/1000020690.md)) debe existir y estar incluido en el Calendar del dispositivo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000058"></a>

- La función **AppointmentDisplay** devuelve el control a la aplicación tan pronto como se muestra la descripción de la cita en la aplicación nativa.

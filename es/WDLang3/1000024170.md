


## &lt;Chart&gt;.DeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un gráfico dibujado y creado por [grCreate](../WDLang3/3042005.md) y destruye este gráfico. La destrucción de un gráfico permite liberar el espacio de memoria ocupado por la matriz utilizada por este gráfico.

**Observación**: Todos los gráficos utilizados en una aplicación se destruyen automáticamente al cerrar la ventana o la aplicación.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Destroy the chart named "MyChart"
CHART_MyChart.DeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.DeleteAll()`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar en la ventana, página o editor de reportes.

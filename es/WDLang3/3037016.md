


## Configuración del ordenador "servidor de fax
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000287"></a>
WINDEV y WEBDEV le permiten gestionar faxes a través del servidor de fax del sistema, disponible en Windows (2000, XP y posteriores).

Es posible:

- [enviar faxes](../WDLang3/3037002.md).

- [manipular los faxes entrantes y salientes](../WDLang3/3037017.md). 




Esta página de ayuda presenta la configuración del servidor de fax necesaria para realizar estas operaciones.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración del ordenador "servidor de fax
<a name="configuracion_del_ordenador_servidor_fax_ELTTEXTE000311"></a>


### Configuración del ordenador actual
<a name="configuracion_del_ordenador_actual_ELTPARAGRAPHE000029"></a>

Los siguientes elementos son necesarios para enviar faxes:

- Windows XP, 2000 o Vista (Vista Professional o Vista Full Version)

- un modem configurado en el equipo (servidor en WEBDEV)

- debe iniciarse el servicio de fax estándar de Windows (en el servidor de WEBDEV)




**Observación**: el servidor de fax puede ser un ordenador distinto del ordenador actual. En este caso, el servidor de fax debe estar instalado en un ordenador "Server 2003" y compartido.

**1. Compruebe la configuración del módem**

Abra el panel de control de Windows ("Inicio .. Configuración .. Panel de control") y seleccione "Phone y opciones de módem". Compruebe si hay un módem instalado en el equipo.

**2. Instalar un servidor de fax estándar**

Windows XP, 2000 y Vista se suministran con un servidor de fax.

Para instalar este servidor de fax en Windows XP o 2000:

- Seleccione la opción "Agregar o quitar programas" del panel de control de Windows (opción "Inicio .. Configuración .. Panel de control").

- Seleccione la opción "Agregar componentes de Windows".




Para instalar este servidor de fax en Windows Vista:

- Seleccione "Programas y características" en el panel control de Windows ("Inicio .. Configuración .. Panel control").

- Seleccione "Activar o desactivar las características de Windows" y, a continuación, "Fax y digitalización de Windows".




**3. Compruebe si el servicio de fax estándar de Windows está en modo entrante.**

Para más detalles, consulte los siguientes párrafos. 
<a name="NOTE2_2"></a>


### Configuración del servidor de fax en Windows 2000
<a name="configuracion_del_servidor_fax_windows_2000_ELTPARAGRAPHE000064"></a>

En Windows 2000, el servidor de fax se llama "Servicio de fax".

Para configurar el servidor de fax:

1. Abra el panel de control de Windows ("Inicio .. Configuración .. Panel de control").

2. Haga doble clic en "Servicio de fax".. Si no se muestra esta opción, instale el servidor de fax de Windows 2000 (véase Above).

3. Seleccione la pestaña "Opciones avanzadas" y haga clic en el botón "Abrir la consola para la gestión del servicio de fax".

4. La opción "Dispositivo" lista los módems y faxes instalados en el equipo.

5. Haga doble clic en su módem/fax.

6. En la pestaña "General", active la función que le permite enviar faxes desde una aplicación WINDEV o desde un sitio WEBDEV.

7. Valide.


Observación: Para crear un archivo de portada (.cov), seleccione la pestaña "Portada. 
<a name="NOTE2_3"></a>


### Configuración del servidor de fax en Windows XP
<a name="configuracion_del_servidor_fax_windows_ELTPARAGRAPHE000083"></a>

Para configurar el servidor de fax: 

1. Seleccione "Iniciar.... Impresoras y faxes".

2. Haga doble clic en la opción "Fax. Se inicia la configuración de la asistente para el fax.

3. Seleccione "Herramientas... Configuración".

4. Valide los parámetros de Default hasta llegar al plano "Configuración para la recepción y el envío de faxes".

5. Habilite la función que le permite enviar faxes desde una aplicación WINDEV.

6. Validar. Se inicia el servidor de fax.


Observación: Para crear un archivo de cobertura (.Cov), seleccione la opción "Herramientas .. Página de cobertura personal".
<a name="NOTE2_4"></a>


### Configuración del servidor de fax en Windows Vista
<a name="configuracion_del_servidor_fax_windows_vista_ELTPARAGRAPHE000099"></a>

Para configurar el servidor de fax: 

1. Seleccione "Iniciar... Todos los programas... Fax de Windows y digitalización".. La asistente para la configuración del fax se inicia.  

2. En la parte inferior del panel izquierdo, seleccione "Servicio de fax".

3. Seleccione "Herramientas.... Cuentas de fax" y haga clic en el botón "Añadir".

4. En la asistente, siga los diferentes pasos para conectarse a un fax-módem.

5. Para activar la función de envío de faxes, seleccione "Herramientas... Ajustes de fax" y marque "Permitir que este dispositivo envíe faxes".

6. Validar. Se inicia el servidor de fax.

## Configurar el servidor de fax mediante programación
<a name="configurar_servidor_fax_mediante_programacion_ELTTEXTE000353"></a>


### Opciones del servidor de fax
<a name="opciones_del_servidor_fax_ELTPARAGRAPHE000115"></a>

El servidor de fax estándar de Windows propone varias opciones que se pueden configurar directamente desde la interfaz del servidor de fax. Estas diferentes opciones se pueden configurar mediante programación a través del registro. Usted tiene la capacidad de configurar y descubrirlo:

- información sobre el envío de faxes (Windows 2000): 
	La clave de registro correspondiente es HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax:
	


|   |   |
| --- | --- |
| Número de intentos | reintentos |
| Número de minutos entre dos intentos | Reintentar delay |
| Número de días antes de eliminar los faxes no enviados | Días sucios |
| Archivar los faxes salientes | ArchiveOutgoingFax |
| Directorio de faxes salientes | ArchivoDirectorio |
| Imprimir el encabezado superior | Creación de marca |
| Prohibir las portadas personalizadas | ServerCoverPageOnly |
| Hora de inicio de la tarifa económica | InicioCheapTime |
| Momento en que finaliza la tarifa económica | StopCheapTime |



- información sobre el remitente del fax (Windows 2000): 
	La clave de registro correspondiente es HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax\\UserInfo:
	


|   |   |
| --- | --- |
| Número del remitente de fax | Número de fax |
| Buzón Address | Buzón |
| Empresa | Empresa |
| Título | Título |
| Nombre completo | FullName |
| Departamento | Departamento |
| Negocios Address | Ofimática |
| Teléfono fijo | Teléfono de casa |
| Negocios Phone | Teléfono de oficina |
| Muestra el monitor de estado al enviar y recibir faxes. | Notificación visual |
| Monitor de estado siempre en la parte superior | AlwaysOnTop |
| Sonido Notification | SonidoNotificación |
| Código de facturación | Código de facturación |

### Consejos
<a name="consejos_ELTPARAGRAPHE000221"></a>

Para crear un servidor de fax y gestionar la rellamada de faxes, se deben especificar las siguientes opciones en el administrador de fax:

- No hay reintento de envío de faxes: 
	
	```wl
	RegistrySetValue("HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Fax","Retries",0)
	```


- Tiempo de espera set a 0 mn entre dos intentos: 
	
	```wl
	RegistrySetValue("HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Fax","Retry Delay",0)
	```


- 0 día para guardar los faxes no enviados: 
	
	```wl
	RegistrySetValue("HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Fax","Dirty Days",0)
	```





Es posible que un fax cuyo estado cambió de ***FaxStatusInit*** O ***FaxStatusCall*** a ***FaxStatusInactive*** se haya intentado enviar una vez. Este fax será reenviado (no se reenviará automáticamente porque el valor de la opción "No reintentar enviar faxes" es <u><u><u><u>True</u></u></u></u>).

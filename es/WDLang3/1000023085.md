


## &lt;mqttSession Variable&gt;.Desconectar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite desconectarse de un servidor MQTT (también llamado "Broker").


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyMQTTSession is mqttSession
...

MyMQTTSession.Disconnect()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<MQTT session>.Disconnects()`
---

**`<MQTT session>: (mqttSession variable)`**

Nombre de [mqttSession](../WDLang3/1000023023.md) Variable correspondiente a los parámetros de la conexión a finalizar.

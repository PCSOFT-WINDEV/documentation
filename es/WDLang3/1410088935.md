
## apiRequest (Tipo de solicitud)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **apiRequest** se utiliza para definir todas las características avanzadas de una solicitud ejecutada a través de un servicio web REST importado utilizando la documentación de OpenAPI. Estas características pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Properties


### Propiedades específicas de las variables apiRequest
<a name="propiedades_especificas_las_variables_apirequest_ELTPARAGRAPHE000026"></a>

Las siguientes propiedades se pueden utilizar para las variables manipular **apiRequest**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ClientCertificate | Cadena o buffer | Corresponde a: <br><br>- una cadena de caracteres con un acceso ruta al archivo .p12 que contiene el Certificate para adjuntar a la solicitud. El Certificate se cargará automáticamente teniendo en cuenta:<br><br>	- el Certificate en la biblioteca ejecutable (si se ha integrado en la aplicación),<br><br>	- el Certificate en el lugar especificado del disco (si el Certificate no ha sido integrado en la biblioteca ejecutable).<br><br><br><br><br>- un buffer con el Certificate ([fLoadBuffer](../WDLang1/1000019410.md)).<br><br><br><br><br>Si esta propiedad corresponde a una cadena vacía (""), la Certificate por defecto se restablece a "". |
| Cabecera | array asociativa de cadenas de caracteres | Clave/Valor set de cabeceras para enviar.<br><br>Ejemplo de uso de esta Property: <br><br><br><pre><code>cMyRequest.Header["Authorization"] = ...<br>	" WSSE profile=""UsernameToken"""<br>cMyRequest.Header["X-WSSE"] = ...<br>	"UsernameToken Username=blahblah "<br>cMyRequest.Header["Cache-Control"] = " no-cache"</code></pre><br> |
| ClientCertificatePassword | Cadena de caracteres | Contraseña asociada al cliente Certificate (cadena vacía de Default) |
| ConnectionTimeout | Entero o Duración | Tiempo máximo de espera de la conexión expresado en milisegundos (20 segundos por Default, lo que significa 20000 milisegundos). Esta propiedad puede corresponder a: <br><br>- un número entero que corresponde al número de milisegundos,<br><br>- una variable de tipo Duration,<br><br>- la duración en un formato legible (por ejemplo, 20 s o 20000 ms). <br><br><br><br><br>La conexión falla si no puede establecerse dentro de este periodo. |

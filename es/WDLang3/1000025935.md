
## ModbusReadBit (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee uno o más bits de un esclavo Modbus, empezando por una Address específica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a session and connect to a Modbus server
Session is modbusSession
Session = ModbusConnectTCP("192.168.1.7")

arrResponse is array of 5 int

// If the connection fails, display the error
IF ErrorOccurred = True THEN
	Trace(ErrorInfo())
ELSE
	// Read a sequence of 5 bits in read-write mode starting at address 0x42
	arrResponse = ModbusReadBit(Session, 0x42, 5, False)
	// Display the error, if any
	IF ErrorOccurred = True THEN
		Trace(ErrorInfo())	
	END
END

// Disconnection
ModbusDisconnect(Session)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Leyendo 1 bit de un esclavo Modbus

`<Result> = ModbusReadBit(<Session> , <Address> , <Read-only>)`
---

**`<Result>: (booleano)`**

Valor del bit leído (<u><u><u><u>True</u></u></u></u> o <u><u><u><u>False</u></u></u></u>).

**`<Session>: (modbusSession Variable)`**

Nombre del [modbusSession](../WDLang3/1000025924.md) Variable que corresponde a la sesión de comunicación con el esclavo Modbus.

**`<Address>: (Integro)`**

Address del bit en el esclavo (entre 0 y 65.535).

**`<Read-only>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los bits a leer son parte de los datos de sólo lectura (código de función Modbus 2 (Read Discrete Inputs)). 

- <u><u><u><u>False</u></u></u></u> si los bits a leer son parte de los datos de lectura-escritura (código de función Modbus 1 (Read Coils)).





<a name="SYNTAX2"></a>

### La lectura de múltiples bits de un esclavo Modbus

`<Result> = ModbusReadBit(<Session> , <Address> , <Number of bits> , <Read-only>)`
---

**`<Result>: (Array)`**

Array de enteros que contiene el valor de los bits leídos (<u><u><u><u>True</u></u></u></u> o <u><u><u><u>False</u></u></u></u>).

**`<Session>: (modbusSession Variable)`**

Nombre del [modbusSession](../WDLang3/1000025924.md) Variable que corresponde a la sesión de comunicación con el esclavo Modbus.

**`<Address>: (Integro)`**

Comenzando Address del bit en el esclavo (entre 0 y 65.535).

**`<Number of bits>: (Integro)`**

Número de bits a ser leídos (a ser enviados por el esclavo).

**`<Read-only>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los bits a leer son parte de los datos de sólo lectura (código de función Modbus 2 (Read Discrete Inputs)). 

- <u><u><u><u>False</u></u></u></u> si los bits a leer son parte de los datos de lectura-escritura (código de función Modbus 1 (Read Coils)).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función bloquea la aplicación hasta que el esclavo responda al mensaje o hasta que se supere el tiempo de espera de la sesión. 

- Después de llamar a **ModbusReadBit**, se recomienda comprobar el [ErrorOccurred](../WDLang1/3087001.md) Variable para asegurarse de que no hay errores.

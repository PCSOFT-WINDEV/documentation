
## SpeechSynthesisInProgress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para saber si se está realizando una operación de lectura en el motor de síntesis de voz.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SpeechSynthesisInProgress() = True THEN
	SpeechSynthesisStop()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpeechSynthesisInProgress()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si hay una lectura en curso,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Sólo se tienen en cuenta las operaciones de lectura iniciadas por la aplicación.

- Para detener todas las operaciones de lectura en curso, utilice [SpeechSynthesisStop](../WDLang3/1000020174.md).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Funcionalidad de aplicación requerida**
	Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.
	Característica necesaria: Micrófono: Esta función permite a las aplicaciones realizar grabaciones de audio.

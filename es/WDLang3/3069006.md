


## Funciones SOAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar el protocolo SOAP:



|   |   |
| --- | --- |
| [SOAPAddAttribute](../WDLang3/1000019241.md) | Añade un atributo a un automatic Variable generado por la importación de un Webservice. |
| [SOAPAddHeader](../WDLang3/3069001.md) | Añade un encabezado a la siguiente función SOAP de un servicio web a ejecutar. |
| [SOAPAddSAMLAssertion](../WDLang3/1000020762.md) | Añade una aserción SAML a una solicitud de servicio web. |
| [SOAPAddXMLSignature](../WDLang3/1000020754.md) | Añade una firma XML a una solicitud de servicio web. |
| [SOAPEncodeResult](../WDLang3/3069003.md) | Permite configurar el formato del valor devuelto por el servidor WINDEV SOAP al programa cliente SOAP. |
| [SOAPError](../WDLang3/3069005.md) | Devuelve el error del último Jabón Procedure que falló. |
| [SOAPGetHeader](../WDLang3/1000019237.md) | Recupera un encabezado en el mensaje SOAP recibido tras una llamada a un servicio web.. |
| [SOAPGetResult](../WDLang3/3069002.md) | Devuelve el resultado del último Jabón Procedure que se ejecutó con éxito. |
| [SOAPPrepare](../WDLang3/1000019238.md) | Construye la petición SOAP para llamar a una función de servicio web. |
| [SOAPRun](../WDLang3/3069013.md) | Ejecuta un Procedure en un servidor SOAP. |
| [SOAPRunXML](../WDLang3/3069014.md) | Ejecuta un Procedure en un servidor SOAP. |
| [SOAPToHTTP](../WDLang3/1000023920.md) | Construye la petición HTTP para llamar a una función de un servicio web.. Esta petición puede ser modificada para adaptarla a las expectativas del Webservice, si es necesario.. A continuación, puede enviarse utilizando [HTTPSend](../WDLang3/1000021183.md). |

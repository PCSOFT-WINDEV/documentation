


## Funciones de gestión domótica
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el protocolo X10:



|   |   |
| --- | --- |
| [DomoClose](../WDLang3/1000003106.md) | Cierra la comunicación abierta por [DomoOpen](../WDLang3/1000003102.md). |
| [DomoOpen](../WDLang3/1000003102.md) | Se utiliza para establecer una conexión con una interfaz X10. |
| [DomoSend](../WDLang3/1000003104.md) | Envía un comando a uno o más dispositivos eléctricos compatibles con el protocolo X10. |
| [DomoStatus](../WDLang3/1000003105.md) | Permite saber si un dispositivo eléctrico (compatible con el protocolo X10) está encendido o apagado. |
| [DomoSwitchOff](../WDLang3/1000003103.md) | Apaga uno o más dispositivos eléctricos compatibles con el protocolo X10. |
| [DomoSwitchOn](../WDLang3/1000003101.md) | Enciende uno o más dispositivos eléctricos compatibles con el protocolo X10. |

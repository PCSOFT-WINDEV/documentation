
## Notificación Push: Configuración Android (Firebase)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000106"></a>
A partir de la actualización 22 de WINDEV Mobile (versión 220052), las notificaciones Push en Android utilizan la plataforma Firebase (en lugar de Google Cloud Messaging). 

Para utilizar la plataforma Firebase, debes tener una cuenta de Google y debes realizar una configuración específica. 

Esta página de ayuda presenta las diferentes operaciones a realizar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Etapas a seguir
<a name="etapas_seguir_ELTTEXTE000130"></a>
Los diferentes pasos son los siguientes: 

**1. Creación de un proyecto Firebase**

El proyecto Firebase se crea desde la consola de la Firebase, accesible desde la siguiente Address: [https://console.firebase.google.com/](https://console.firebase.google.com/)
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase1.gif)


Observación: También tiene la posibilidad de importar un proyecto de Google API (utilizado para enviar notificaciones push por GCM) desde esta consola.

**2. Añadir Firebase a la aplicación Android**

- Haga clic en el botón "Añadir Firebase a su aplicación Android".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase2.gif)


- Rellene el nombre del paquete de la aplicación con el definido en la generación de Android asistente. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase3.gif)


- Haga clic en el botón "Añadir una aplicación. 

- Haga clic en "Continuar" y luego en "Listo".




**3. Descargue el archivo "google-services.JSON".**  

- Mostrar los parámetros de su aplicación: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase4.gif)


- En la sección relativa a la aplicación Android, descargue el archivo "google-services.JSON".. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase5.gif)



Este archivo debe ser suministrado en la asistente para generar la aplicación: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Push_Android_firebase%20-%20HC%20N%B0001.gif)


**4. Recuperación de la clave del servidor**. Esta tecla es necesaria para enviar notificaciones Push. 

- Haga clic en la pestaña "Mensajería en la nube. 

- copy el identificador de la clave del servidor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Notif_FireBase6.gif)
 Este número de clave debe darse como parámetro a [NotifPushSend](../WDLang3/1000020819.md).

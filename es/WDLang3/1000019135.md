
## SaaSAdminModifySubscription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una suscripción SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todas las suscripciones. Un gestor de cuentas de cliente sólo puede modificar una suscripción por cuenta propia. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
P is dynamic saasPricing 
C is saasClient dynamic 
Sub is saasSubscription dynamic

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")

Sub = SaaSAdminFindSubscription(P, C)
Sub.NbLicenses = 3

// The Hybrid client now contains 3 user licenses for the Promo 2010 of Forum
SaaSAdminModifySubscription(Sub, P, C)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminModifySubscription(<SaaS subscription>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS subscription>: (saasSubscription Variable)`**

Nombre de la [saasSubscription](../WDLang3/1000019088.md) Variable correspondiente a la suscripción que debe ser modificada en la base de datos SaaS.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cualquier modo de modificación de una suscripción se presenta en los informes de la siguiente manera:

- Fin de la suscripción,

- Inicio de una nueva suscripción (con el mismo nombre) con las nuevas características.


Esto es necesario para calcular automáticamente los precios en función del número de licencias y de la duración de uso.

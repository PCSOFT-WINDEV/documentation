


## Funciones LDAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de LDAP son las siguientes:



|   |   |
| --- | --- |
| [LDAPAddAttribute](../WDLang3/3032110.md) | Añade un atributo o añade un nuevo valor a un atributo existente. |
| [LDAPConnect](../WDLang3/3032100.md) | Se utiliza para conectarse a un servidor LDAP a través de las características definidas en la [LDAPSession structure](../WDLang3/3032101.md). |
| [LDAPDelete](../WDLang3/3032114.md) | Elimina un objeto del servidor LDAP. |
| [LDAPDeleteAttribute](../WDLang3/3032115.md) | Elimina un atributo así como sus valores. |
| [LDAPDeleteAttributeValue](../WDLang3/3032116.md) | Elimina un valor de un atributo especificado. |
| [LDAPDisconnect](../WDLang3/3032102.md) | Se utiliza para desconectarse de un servidor LDAP específico. |
| [LDAPFind](../WDLang3/3032108.md) | Realiza una búsqueda en un servidor LDAP. |
| [LDAPListAttribute](../WDLang3/3032104.md) | Enumera los atributos de objeto en un servidor LDAP. |
| [LDAPListChildren](../WDLang3/3032105.md) | Enumera los hijos de un objeto en un servidor LDAP. |
| [LDAPMode](../WDLang3/3032126.md) | Modifica el modo operativo de una sesión LDAP. |
| [LDAPNbValue](../WDLang3/3032106.md) | Devuelve el número de valores de un atributo. |
| [LDAPRenameAttribute](../WDLang3/3032120.md) | Renombra un atributo. |
| [LDAPReplaceAttribute](../WDLang3/3032113.md) | Reemplaza todos los valores de un atributo por un valor específico. |
| [LDAPReplaceAttributeValue](../WDLang3/3032119.md) | Reemplaza un valor específico de un atributo por un nuevo valor. |
| [LDAPReset](../WDLang3/3032103.md) | Reinicia la estructura [LDAPSession](../WDLang3/3032101.md) con los valores de Default. |
| [LDAPStartAdd](../WDLang3/3032109.md) | Inicializa la adición de un nuevo objeto en un servidor LDAP. |
| [LDAPStartModify](../WDLang3/3032112.md) | Inicia la modificación de un objeto existente en un servidor LDAP. |
| [LDAPValidateAdd](../WDLang3/3032111.md) | Valida la adición de un nuevo objeto a un servidor LDAP. |
| [LDAPValidateModify](../WDLang3/3032117.md) | Valida la modificación de un objeto existente en un servidor LDAP. |
| [LDAPValue](../WDLang3/3032107.md) | Devuelve el valor correspondiente a la índice pasada como parámetro para el atributo especificado. |





Otras funciones de gestión de relacionado a LDAP: 



|   |   |
| --- | --- |
| [gpwGetLDAPInfo](../WDLang6/1000017063.md) | Recupera la configuración del servidor LDAP utilizado para el User Groupware.  |
| [gpwImportLDAPUsers](../WDLang6/1000017058.md) | Importa los usuarios almacenados en un directorio LDAP al archivo de usuarios de groupware usuario. |
| [gpwInLDAPMode](../WDLang6/1000017056.md) | Permite saber si la identificación de los usuarios en el groupware usuario se hará a través de un directorio LDAP. |
| [gpwOpenLDAPSession](../WDLang6/1000017055.md) | Informa al User Groupware que la autenticación de los usuarios se hará a través de un directorio LDAP. |

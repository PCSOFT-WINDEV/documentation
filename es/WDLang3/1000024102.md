


## &lt;Scatter3DLinkPoint (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vincula (o no) los puntos encontrados en un gráfico "3D Scatter".

**Recordatorio**: Los puntos encontrados en un gráfico de "dispersión 3D" son relacionado de Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_My3DScatter.Scatter3DLinkPoint(1, LineSolid)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.Scatter3DLinkPoint(<Series number> [, <Type of line>])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar que se encuentra en el editor de ventanas.

**`<Series number>: (Integro)`**

Número de la serie a utilizar.

**`<Type of line>: (Constante opcional de tipo Integer)`**

Tipo de Line para dibujar entre los puntos del gráfico: 



|   |   |
| --- | --- |
| *LineDash* | Line con guiones (------). |
| *LineDotAndDash* | Line mezcla puntos y guiones (.-.-.-.-.-). |
| *LineDotted* | Línea de puntos (........). |
| *LineNone* | No se dibuja ningún Line. |
| *LineSolid* <br>(Valor predeterminado) | Línea continua. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los datos definidos con la función **&lt;Scatter3DLinkPoint** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- La función **&lt;Scatter3DLinkPoint** se utiliza únicamente para los gráficos de "dispersión 3D". Esta función no tiene ningún efecto con las otras clases de gráficos.

- Los puntos son relacionado en el orden en que han sido añadidos

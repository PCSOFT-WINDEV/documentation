## Compartir diferentes elementos de WINDEV
<a name="compartir_diferentes_elementos_windev_ELTTEXTE000067"></a>
Los mismos elementos WINDEV (análisis, informes, consultas, etc.) pueden ser utilizados en una aplicación WINDEV y en una aplicación WINDEV Mobile:

- **Ventanas**: Windows tiene un formato diferente en WINDEV (archivo ".WDW") y WINDEV Mobile (archivo ".WPW"). En efecto, las ventanas no tienen el mismo tamaño, las mismas características, las mismas características y el mismo enfoque de usuario., ... WINDEV Mobile le permite importar ventanas WINDEV a un proyecto WINDEV Mobile.

- **Ayuda**: Este tipo de elemento no existe en Android ni en iOS.

- **Análisis, informes, consultas, clases, set de procedimientos, ...**: Estos elementos pueden ser compartidos entre una aplicación WINDEV y una aplicación WINDEV Mobile..


## TwoFactorAuthenticationCheckCode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Verifica el código dado por una aplicación de autenticación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF TwoFactorAuthenticationCheckCode(nCode, sTOTPKey) = False THEN
	Error("Invalid code.")
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TwoFactorAuthenticationCheckCode(<Code> , <TOTP key>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código es válido,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Code>: (Entero o cadena de caracteres)`**

Código de 6 dígitos generado por la aplicación de autenticación.



**`<TOTP key>: (Cadena de caracteres)`**

Cadena ANSI que contiene la clave TOTP del usuario a autenticar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Periodo de validez
<a name="periodo_validez_ELTPARAGRAPHE000051"></a>

- El período de validez del código es ligeramente más largo que el período mostrado por la aplicación de autenticación. Por ejemplo, cuando Google Authenticator muestra un nuevo código, el código anterior sigue siendo válido durante 1 minuto.

- **TwoFactorAuthenticationCheckCode** comprueba los códigos generados 1 minuto antes y 1 minuto después de la solicitud de verificación.

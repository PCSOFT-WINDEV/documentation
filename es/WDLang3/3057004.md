
## OLECopy( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia el objeto encontrado en un OLE control en el portapapeles de Windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a Paint Brush object
OLECreateEmpty(OLE_OLE1, "PBrush")
// The object is copied into the clipboard
OLECopy(OLE_OLE1)
// The object is pasted and saved in another control
OLEPaste(OLE_OLE2)
OLESave(OLE_OLE2, "OBJECT.OLE")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OLECopy(<OLE control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha copiado el objeto, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<OLE control>: (Nombre del control)`**

Nombre de la OLE control que contiene el objeto OLE de copy. Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control al que pertenece el actual Event.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Sólo se admite la versión 1 de OLE.

- Se sobrescribe el contenido del portapapeles. El contenido del portapapeles puede ser en formato de texto, gráfico o binario.


## Creación de un widget Android
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000121"></a>
Los widgets son pequeñas aplicaciones que pueden aparecer en la pantalla de inicio de los teléfonos o tabletas. Estos widgets se utilizan para mostrar información actualizada periódicamente y proponen interacciones sencillas con el usuario. 

Con WINDEV Mobile, puede crear widgets desde sus aplicaciones Android a través de las ventanas Widget. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Creación y despliegue de un widget Android
<a name="creacion_despliegue_widget_android_ELTTEXTE000145"></a>
El principio es muy sencillo: 

1. Se crea un proyecto Android. Para obtener más información, consulte [Desarrollo de aplicaciones para Android](../Editeurs/9000099.md). 

2. En su proyecto Android, usted crea ventanas de widgets. Estas ventanas contienen todas las funciones que necesita su widget. 
	Observaciones: 

	- Sólo hay 3 tipos de controles disponibles en las ventanas de widgets: [Controles Botón](../WDChamp/1013076.md), [Controles estáticos](../WDChamp/1013179.md) y [controles Imagen](../WDChamp/1013218.md).

	- Los estilos definidos para los controles de un widget en modo de edición se ignorarán en tiempo de ejecución. Sólo puede set el fondo Color de los controles y el fondo Image de los controles Botón.

	- Las ventanas de widgets presentan algunas características específicas. Para obtener más información, consulte [Ventanas de widgets](../WDChamp/9000143.md). 




3. Genere su aplicación Android. Para obtener más información, consulte [Generar una aplicación Android](../Editeurs/9000109.md). 

4. Desplegar el archivo de apk generado. 

5. Los widgets encontrados en la aplicación aparecerán en la lista de widgets disponibles y se podrán instalar en el escritorio. 





### Características específicas de los widgets
<a name="caracteristicas_especificas_los_widgets_ELTPARAGRAPHE000051"></a>

Los widgets de una aplicación generada con WINDEV Mobile se ejecutan en una Process diferente de la propia aplicación. 

Sin embargo, todos los widgets de una misma aplicación se ejecutan en la misma Process y, por lo tanto, compartirán el mismo contexto de ejecución (variables globales, navegación HFSQL, ...). 

El usuario tiene la posibilidad de añadir en la pantalla de inicio tantas instancias del mismo widget como sea necesario. Cada una de estas instancias corresponderá a la misma instancia de ventana: los cambios realizados en la ventana se aplicarán automáticamente a todas las instancias mostradas del widget correspondiente.

### Funciones para la gestión de widgets
<a name="funciones_para_gestion_widgets_ELTPARAGRAPHE000064"></a>Las siguientes funciones se utilizan para gestionar características específicas de los widgets de Android: 



|   |   |
| --- | --- |
| [InWidgetMode](../WDLang1/1000020649.md) | Indica si el código se ejecuta en modo Widget Android. |
| [WidgetDisplay](../WDLang3/1000020647.md) | Actualiza una ventana de widgets. |
| [WidgetRunApp](../WDLang3/1000020648.md) | Inicia la aplicación a la que pertenece un widget. |

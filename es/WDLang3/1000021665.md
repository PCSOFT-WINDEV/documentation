


## Funciones HTTP (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular páginas de INTERNET y enviar formularios HTTP (sintaxis de prefijo), ...:



|   |   |
| --- | --- |
| [&lt;httpRequest Variable&gt;.GetCookie](../WDLang3/1000021627.md) | Recupera las cookies leídas por una llamada a [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md) para un dominio HTTP. |
| [&lt;httpRequest Variable&gt;.ManageCookie](../WDLang3/1000021608.md) | Habilita o deshabilita la administración de cookies en una solicitud HTTP ejecutada por [httpRequest](../WDLang3/3043007.md). |
| [&lt;httpRequest Variable&gt;.ReadCookie](../WDLang3/1000021621.md) | Devuelve el valor de un Cookie recibido tras una petición HTTP. |
| [&lt;httpRequest Variable&gt;.ReemplazarCookie](../WDLang3/1000021623.md) | Reemplaza todas las cookies almacenadas por [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md) para un dominio. |
| [&lt;httpRequest Variable&gt;.ResetCookie](../WDLang3/1000021622.md) | Borra todas las cookies (globalmente o para un dominio) almacenadas por las llamadas a [httpRequest](../WDLang3/3043007.md). |
| [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md) | Envía una solicitud HTTP y espera la respuesta del servidor HTTP. |
| [&lt;httpRequest Variable&gt;.WriteCookie](../WDLang3/1000021607.md) | Añade o modifica una Cookie en una petición HTTP. |
| [&lt;httpResponse Variable&gt;.GetCookie](../WDLang3/1000021633.md) | Recupera las cookies leídas en la [httpResponse](../WDLang3/1000021165.md) Variable (que contiene el resultado de [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md)). |

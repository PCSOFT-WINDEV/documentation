


## grScatterLinkPoint (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vincula (o no) los puntos que se encuentran en un gráfico "Scatter".

Ejemplo de un gráfico con puntos relacionado: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TraitRelie.gif)


**Recordatorio**: Los puntos que se encuentran en un gráfico de "Dispersión" son relacionado de Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Unlink the points found in the "Scatter" chart named "MyChart"
grScatterLinkPoint("MyChart", False)
```
<a name="Example2"></a>

```wl
// Unlink the points found in the "Scatter" chart named CHART_MyChart
grScatterLinkPoint(CHART_MyChart, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Vinculando o desvinculando los puntos encontrados en un gráfico de dispersión

`grScatterLinkPoint(<Chart name> , <Bind>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Bind>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si los puntos deben ser relacionado entre ellos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  





<a name="SYNTAX2"></a>

### Vinculando o desvinculando los puntos encontrados en una serie de gráficos de dispersión

`grScatterLinkPoint(<Chart name> , <Series number> , <Bind>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie a utilizar. 

**`<Bind>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si los puntos deben ser relacionado entre ellos,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En las cartas "Scatter", un punto se define por sus coordenadas (coordenada X y coordenada Y).

- Los datos definidos con la función **grScatterLinkPoint** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- La función **grScatterLinkPoint** solo se utiliza para gráficos de "dispersión". Esta función no tiene ningún efecto con las otras clases de gráficos.

- Los puntos son relacionado en el orden en que han sido añadidos

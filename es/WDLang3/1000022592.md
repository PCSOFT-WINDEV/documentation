
## SaaSAdminConfigureSMTP( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la sesión SMTP associated con la cuenta de administrador de una sesión SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
e is emailSMTPSession
e..ServerAddress = "smtp.myserver.fr"
// ...
SaaSAdminConfigureSMTP(e)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminConfigureSMTP(<SMTP session> [, <Recipient> [, <Sender>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la configuración de la sesión SMTP se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SMTP session>: (Variable de tipo emailSMTPSession)`**

Nombre de la [emailSMTPSession](../WDLang3/1000018765.md) Variable que contiene los detalles de la configuración de la sesión SMTP (servidor, nombre de usuario, contraseña, etc.).

**`<Recipient>: (Cadena de caracteres opcional)`**

Correo electrónico Address del destinatario en carbono ciego copy. Este parámetro se ignora si no se especifica o si corresponde a una cadena vacía ("""). 

**`<Sender>: (Cadena de caracteres opcional)`**

Correo electrónico del remitente Address. Este parámetro se ignora si no se especifica o si corresponde a una cadena vacía ("""). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Necesidad de configuración de SMTP
<a name="necesidad_configuracion_smtp_ELTPARAGRAPHE000059"></a>

Para gestionar las contraseñas olvidadas (enviar un Email que contenga el código de seguridad, [SaaSSendSecurityCode](../WDLang3/1000022528.md)) se necesita la configuración de la sesión SMTP utilizada: 

- a través de **SaaaaaaSAdminConfigureSMTP**. 

- a través de la interfaz del administrador de SaaS.


## FaxDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una conexión con un servidor de fax (establecida previamente por [FaxConnect](../WDLang3/3037005.md)).

**Observación**: Esta función se puede utilizar en Windows 2000, XP o posterior.. En Windows Vista (y posteriores), la gestión de faxes está disponible con Windows Professional y Windows Integral Edition.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectID is int
ConnectID = FaxConnect()
...
IF FaxDisconnect(ConnectID) = False THEN 
	Error("The disconnection failed", ErrorInfo(errSystemMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxDisconnect(<Connection identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> si se produce un error. El mensaje de error correspondiente es devuelto por [ErrorInfo](../WDLang1/3013008.md).




**`<Connection identifier>: (Integro)`**

Identificador de la conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md).

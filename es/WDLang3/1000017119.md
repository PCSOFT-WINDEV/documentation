
## HTTPCreateForm( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un formulario HTTP. Este formulario puede ser enviado por [HTTPSendForm](../WDLang3/1000017190.md) y puede ser tratado por las funciones de gestión de formularios HTTP.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPCreateForm("FORM")
...
HTTPSendForm("FORM", "www.mysite.com")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HTTPCreateForm(<Form name>)`
---

**`<Form name>: (Cadena de caracteres)`**

Nombre del formulario HTTP. Este nombre se utiliza para manipular el formulario HTTP a través de la programación (con [HTTPAddParameter](../WDLang3/1000017189.md) por ejemplo). El formulario se borra si ya existe.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Liberar recursos
<a name="liberar_recursos_ELTPARAGRAPHE000074"></a>

La declaración del formulario se puede cancelar con [HTTPCancelForm](../WDLang3/1000017123.md). Esta función se utiliza para liberar los recursos asignados.. Los recursos asignados se liberan automáticamente:

- al final de la thread actual (o al final de la aplicación si es la thread principal).

- durante la llamada a [HTTPSendForm](../WDLang3/1000017190.md).

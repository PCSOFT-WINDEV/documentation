


## grHMCaptionX (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de una categoría en el eje X de un gráfico de Mapa de Calor.

Nota: Por Default, la leyenda de una categoría corresponde al valor numérico de la categoría.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grHMCaptionX(CHART_HMap, 1, "X1")
grHMCaptionX(CHART_HMap, 2, "X2")
```

```wl
// Caption of axes
FOR EACH ELEMENT nIndex, nTemperature OF gaaTemperature
	grHMCaptionX(CHART_Chart, nIndex, "" + nTemperature)
END
FOR EACH ELEMENT nIndex, nWind OF gaaWind
	grHMCaptionY(CHART_Chart, nIndex, "" + nWind)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grHMCaptionX(<Chart name> , <X> , <Caption>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<X>: (Integro)`**

Posición de la categoría en el eje X.

**`<Caption>: (Cadena de caracteres)`**

Nuevo título para la categoría. Si este parámetro corresponde a una cadena vacía ("""), se muestra el valor numérico de la categoría.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La leyenda definida por **grHMCaptionX** se tendrá en cuenta durante la siguiente llamada a [grDraw](../WDLang3/3042026.md).

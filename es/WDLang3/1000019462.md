
## SaaSChangePassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia la contraseña del usuario conectado. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SaaSConnect("login", "password") = True THEN
	IF SaaSChangePassword("M[]t2P4zz") = True THEN
		Info("Your password was successfully changed")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSChangePassword(<Password>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cambió la contraseña,

- <u><u><u><u>False</u></u></u></u> en caso contrario (o si no hay ningún usuario conectado). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Password>: (Cadena de caracteres)`**

Nueva contraseña.

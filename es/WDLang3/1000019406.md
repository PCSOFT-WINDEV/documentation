## Presentación
<a name="presentacion_ELTTEXTE000067"></a>
El DNS (Domain Name Server) es un servicio utilizado para establecer una correspondencia entre un Address IP y un nombre de dominio. 

Se pueden utilizar varias funciones de WLanguage para manipular fácilmente las direcciones IP y los dominios en un servidor DNS.. 



## Funciones WLanguage
<a name="funciones_wlanguage_ELTTEXTE000091"></a>
Las siguientes funciones se utilizan para gestionar los servidores DNS: 



|   |   |
| --- | --- |
| [DNSAddAddress](../WDLang3/1000019400.md) | Añade una relación IP/Nombre al servidor DNS. |
| [DNSDeleteAddress](../WDLang3/1000019402.md) | Elimina una relación Nombre/IP Address de un servidor DNS. |
| [DNSListAddress](../WDLang3/1000019401.md) | Enumera las direcciones IP de associated con un nombre en el servidor DNS. |


## Dirección (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Address** se utiliza para manipular el Address de un lugar o persona. Una Address se define por una Address postal (calle, ciudad, estado, código postal, Country) y/o una posición geográfica (latitud y longitud). Las características de esta Address pueden definirse y modificarse a través de diferentes propiedades de la WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyAddress is Address
MyAddress.City = "Paris"
MyAddress.Country = "France"

// Centers the map displayed by the "MAP_Position" control from an address
MapDisplayPosition(MAP_Position, MyAddress)
```

## Observaciones


### Propiedades específicas de las variables Address
<a name="propiedades_especificas_las_variables_address_ELTPARAGRAPHE000059"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **Address**:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| AdminArea | Cadena de caracteres | Región asociada a la Address (estado para una Address en los EE.UU., terreno para una Address en Alemania, etc.). <br><br>Esta propiedad es opcional. |
| Calle | Cadena de caracteres | Nombre de la calle, calle o avenida. <br><br>Esta propiedad es opcional. |
| Ciudad | Cadena de caracteres | Nombre de la ciudad. <br><br>Esta propiedad es opcional. |
| Country | Cadena de caracteres | Nombre del país. Esta propiedad es opcional. |
|   | Constante de tipo Integer | Tipo de Address: <br><br>- *addressOther*: otro tipo de Address.<br><br>- *addressWork*: profesional Address (oficina).<br><br>- *addressHome*: casa Address. <br><br>- *addressCustom*: tipo personalizado que corresponde al valor de la propiedad **Label**.<br><br><br><br><br>Este Property se utiliza principalmente para gestionar los contactos. |
|   | Cadena de caracteres | Etiqueta para los tipos de direcciones personalizadas (opcional).<br>Si **Type** no corresponde a la constante *addressCustom*, el valor de este Property será ignorado.<br><br>Este Property se utiliza principalmente para gestionar los contactos. |
| ubicación | Variable de tipo [geoPosition](../WDLang3/1000019191.md) | Posición geográfica de la Address: latitud y longitud. <br><br>Esta propiedad es opcional. <br><br>Si no se especifica la posición geográfica, el **PositionValid** Property de la variable [geoPosition](../WDLang3/1000019191.md) se establece en <u><u><u><u>False</u></u></u></u>. |
| ZipCode | Cadena de caracteres | Código postal de la Address. <br><br>Esta propiedad es opcional. |

### Usando las variables de Address
<a name="usando_las_variables_address_ELTPARAGRAPHE000161"></a>Las variables de **Address** son utilizadas por las siguientes funciones: 



|   |   |
| --- | --- |
| [geoGetAddress](../WDLang3/1000019797.md) | Recupera una dirección o una lista de direcciones correspondientes a la description de un lugar geográfico. |
| [MapAddItinerario](../WDLang3/1000020383.md) | Añade un itinerario a un control Mapa. |
| [MapDisplayPosition](../WDLang3/1000019787.md) | Centra el mapa visualizado en una control Mapa en una posición geográfica o en un lugar concreto. |





Las variables de **Address** son utilizadas por los siguientes tipos de variables: 



|   |   |
| --- | --- |
| [Contacto](../WDLang3/1000020254.md) | El tipo **Contact** se utiliza para definir las características avanzadas de un contacto que se encuentra en el libro de Phone. |

### Reinicio
<a name="reinicio_ELTPARAGRAPHE000173"></a>

Utilice la función [VariableReset](../WDLang1/3013069.md) para restablecer el contenido de una variable de tipo **Address**.

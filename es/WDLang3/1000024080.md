


## &lt;HMColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica los colores de los diferentes valores en un gráfico Heatmap.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the colors
CHART_HMap.HMColor(-50, DarkBlue)
CHART_HMap.HMColor(+50, LightRed)
FOR i = 1 TO 500
	CHART_HMap.HMAddData(Random(-100, 100), Random(-100, 100), -100)
	CHART_HMap.HMAddData(Random(-100, 100), Random(-100, 100), 100)
END
CHART_NoName1.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.HMColor(<Level> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).


**`<Level>: (Real o constante)`**

Nivel a partir del cual se aplicará el Color. Corresponde a: 

- un valor real,

- la constante *grHMColorEmptyValue*. En este caso, la Color se aplicará a los datos fuera de nivel o sin valor. Este color es Transparent de Default.




**`<Color>: (Entero o constante de tipo Integer)`**

Color que se entregará al nivel. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  

- la constante *AutomaticColor*. En este caso, se borra el nivel.

## Observaciones


### Gestión de colores
<a name="gestion_colores_ELTPARAGRAPHE000225"></a>

Los colores se definen por nivel. Para los valores encontrados entre 2 niveles, se aplica un Gradient lineal por Default.

Este comportamiento puede modificarse con la función [&lt;Parámetro](../WDLang3/1000024115.md) y la constante *grHMDisplayColor*. Se utiliza la siguiente sintaxis:

```txt
<Chart control>.Parameter(grHMDisplayColor, <Display type>)
```



 donde &lt;Tipo de pantalla&gt; corresponde a una de las siguientes constantes:



|   |   |
| --- | --- |
| *grColorGradient*<br>(Valor predeterminado) | Se aplica un degradado lineal para los valores intermedios. |
| *grLowerLevel* | La visualización de los colores se realiza por niveles. Todos los valores menores o iguales al nivel tomarán la Color dada por [&lt;HMColor](../WDLang3/1000024080.md). |
| *grHigherLevel* | La visualización de los colores se realiza por niveles. Todos los valores mayores o iguales al nivel tomarán la Color dada por [&lt;HMColor](../WDLang3/1000024080.md). |


Para las facetas fuera de los límites de color o sin valor, se aplicará el color especificado por *grHMColorEmptyValue*. Este color es Transparent de Default.

Observación: Los colores definidos con la función **&lt;HMColor** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

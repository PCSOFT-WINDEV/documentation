
## BTOpenDeviceProperties( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana de propiedades de un dispositivo Bluetooth.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nDeviceNumber is int
// The devices are stored in a Table control
nDeviceNumber = TABLE_Device.COL_ID[TABLE_Device]
// Opens the window for the properties of the selected device
BTOpenDeviceProperties(nDeviceNumber)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTOpenDeviceProperties(<Device identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró la ventana de propiedades del dispositivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Device identifier>: (Integro)`**

Identificador del dispositivo Bluetooth a utilizar. Este identificador es devuelto por [BTListDevice](../WDLang3/1000017132.md) o [Dispositivo BTSelectDevice](../WDLang3/1000017135.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo la pila Microsoft Bluetooth es compatible con esta función. Para conocer la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).  

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

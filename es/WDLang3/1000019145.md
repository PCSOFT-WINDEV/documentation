
## SaaSAdminSiteConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la conexión a la base de datos común o a la base de datos reference del sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
C is Connection

S = SaaSAdminFindSite("Forum")
C = SaaSAdminSiteConnection(S)

// Display the reference database of "Forum" SaaS site
Info(C.Database)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminSiteConnection(<SaaS site>)`
---

**`<Result>: (Variable de tipo Connection)`**

[Conexión](../WDLang4/1514073.md) variable que contiene los parámetros de conexión a la base de datos.

- Si el sitio no utiliza ninguna base de datos, la conexión devuelta estará vacía.

- Si el sitio utiliza una base de datos común a todas las cuentas de los clientes, la información para conectarse a esta base de datos será devuelta.

- Si el sitio utiliza una base de datos específica para cada cuenta de cliente, la información para conectarse a la base de datos de Default será devuelta.




**`<SaaS site>: (saasSite Variable o cadena de caracteres)`**

Nombre de la [saasSite](../WDLang3/1000019068.md) Variable (o nombre del sitio) que corresponde al sitio SaaS para el que se debe recuperar la conexión.

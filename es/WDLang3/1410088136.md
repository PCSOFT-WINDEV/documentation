
## App Tracking Transparency - Seguimiento de la actividad
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000079"></a>
App Tracking Transparency permite que las aplicaciones de iOS pidan permiso a los usuarios para rastrear su actividad en aplicaciones móviles y sitios web de terceros con fines publicitarios o de intercambio de datos. 

En iOS 14.5, las aplicaciones deben solicitar permiso a los usuarios para rastrear su actividad. 

El "seguimiento de la actividad" se produce cuando los datos que identifican al usuario o al dispositivo del usuario se combinan con datos similares de otras aplicaciones (o sitios web) con fines publicitarios (publicidad dirigida o campañas publicitarias) o cuando la información recopilada se comparte con corredores de datos. 

Si el usuario permite el seguimiento de la actividad, será posible rastrear o acceder al identificador de publicidad del dispositivo (IDFA). La aplicación podrá mostrar anuncios personalizados en otras aplicaciones o sitios web. Si el usuario no otorga el permiso, se reducirá la segmentación de los anuncios.

## ¿Cómo gestionar la transparencia de seguimiento de aplicaciones?
<a name="¿como_gestionar_transparencia_seguimiento_aplicaciones_ELTTEXTE000103"></a>
WINDEV Mobile permite: 

- abrir la ventana para solicitar el permiso de rastreo de la actividad. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=AppleATT.gif)
Esta ventana se puede personalizar como cualquier otra solicitud de autorización, especificando un mensaje personalizado en la etapa "Permisos" del asistente de generación de la aplicación iOS. Solo debe introducir un texto específico para el permiso "Tracking Usage Description". 

- comprobar si se ha otorgado el permiso de rastreo de la actividad.

- obtener el identificador de publicidad. Este identificador identifica de forma exclusiva el dispositivo utilizado. 




WINDEV Mobile incluye las siguientes funciones para gestionar la transparencia de seguimiento de aplicaciones: 



|   |   |
| --- | --- |
| [AppleAdvertisingIdentifier](../WDLang3/1410087936.md) | Devuelve el identificador de publicidad del dispositivo (solo iOS). |
| [AppleRequestTrackingAuthorizationAsynchronous](../WDLang3/1410087935.md) | Muestra la solicitud de Permission para el seguimiento de la actividad del usuario. |
| [AppleTrackingAuthorizationStatus](../WDLang3/1410087934.md) | Permite saber si se concedió permiso a una aplicación para rastrear la actividad del usuario utilizando sus datos (solo para iOS). |

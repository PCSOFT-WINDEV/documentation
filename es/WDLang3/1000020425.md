


## grSurfaceAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a un gráfico de superficie. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000020629&name=grsurfaceadddata_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
x, y, z are real
wImage is Image

// Load the photo found in memory in an Image variable
wImage = "sunset.jpg" 


// Scan the image and all the points
// to define the rate of lightness for the point
// via ColorLightness
FOR I = 1 _TO_ wImage.Width STEP 4
	FOR j = 1 _TO_ wImage.Height STEP 4
		x = i
		y = j
		z = ColorLightness(dPixelColor(wImage, x, y))
		grSurfaceAddData(CHART_Surface, x, y, z)
	END
END

// Draw the chart in the Chart control
grDraw(CHART_Surface)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSurfaceAddData(<Chart name> , <X> , <Y> , <Z>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<X>: (Real)`**

Coordenada X del punto a añadir. 

**`<Y>: (Real)`**

Coordenada Y del punto a añadir. 

**`<Z>: (Real)`**

Cota Z del punto a sumar.

## Observaciones


### Gráfico de superficie
<a name="grafico_superficie_ELTPARAGRAPHE000075"></a>

Para los datos de una carta de superficie, el plano XY constituye la base de la superficie, mientras que Z indica la altura del punto.
<a name="NOTE0_2"></a>


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000084"></a>

[grAddData](../WDLang3/3042023.md) se utiliza para añadir datos en un gráfico (de tarta, de columnas, ...).

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [grStockAddData](../WDLang3/3042009.md).

- Para los gráficos de dispersión, se recomienda utilizar [grScatterAddDataXY](../WDLang3/3042032.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [grHMAddData](../WDLang3/1000021628.md).

- Para las cartas Sunburst, se recomienda utilizar [grSunburstAddData](../WDLang3/1000021285.md). 




Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [grIncreaseData](../WDLang3/3042030.md).

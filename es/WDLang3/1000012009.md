


## Funciones de gestión de tareas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las tareas que se encuentran en la lista de tareas de una mensajería de Lotus Notes o Outlook:



|   |   |
| --- | --- |
| [TaskAdd](../WDLang3/1000012001.md) | Añade una tarea a la lista de tareas de un mensaje de Lotus Notes o de Outlook. |
| [TaskDelete](../WDLang3/1000012008.md) | Elimina la tarea actual de la lista de tareas de un mensaje de Lotus Notes o Outlook. |
| [TaskFirst](../WDLang3/1000012005.md) | Lee la primera tarea encontrada en la lista de tareas de un mensaje de Lotus Notes o Outlook. |
| [TaskLast](../WDLang3/1000012002.md) | Lee la última tarea de una lista de tareas de Lotus Notes o Outlook. |
| [TaskModify](../WDLang3/1000012003.md) | Modifica la tarea actual en una lista de tareas de Lotus Notes o Outlook. |
| [TaskNext](../WDLang3/1000012007.md) | Lee la tarea que se encuentra después de la tarea actual en la lista de tareas de una mensajería de Lotus Notes o Outlook. |
| [TaskPrevious](../WDLang3/1000012004.md) | Lee la tarea que se encuentra antes de la tarea actual en la lista de tareas de una mensajería de Lotus Notes o Outlook. |
| [TaskRead](../WDLang3/1000012011.md) | Lee una tarea de la lista de tareas de un mensaje de Lotus Notes o de Outlook. |
| [TaskReset](../WDLang3/1000012006.md) | Reinicia la [Estructura de la tarea](../WDLang3/1000012010.md). |

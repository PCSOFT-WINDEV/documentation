


## grScatterAddDataXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un dato a un gráfico "Scatter. 

**Recordatorio**: Por Default, un gráfico no contiene datos.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a data into the "Scatter" chart named "MyChart".
// This value (10, 20) is added at the end of the first series.
grScatterAddDataXY("MyChart", 1, 10, 20)
```
<a name="Example2"></a>

```wl
// Adds a data into the CHART_MyChart control ("Scatter" chart).
// This value (20, 10) is added at the end of the first series.
grScatterAddDataXY(CHART_MyChart, 1, 20, 10)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grScatterAddDataXY(<Chart name> , <Series number> [, <Index>] , <X value> , <Y value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie en la que se debe inicializar un punto. Si la serie especificada no existe, el diseñador de la carta crea tantas series como sean necesarias para get una matriz consistente. Todas las series creadas se inicializan a 0.

**`<Index>: (Entero opcional)`**

Índice de los datos a inicializar en la serie. Si no se especifica este parámetro, el valor se añade al final de la serie.

**`<X value>: (Real)`**

Coordenada horizontal (coordenada X) del punto a añadir.

**`<Y value>: (Real)`**

Coordenada vertical (coordenada Y) del punto a añadir.

<a name="NOTE0_1"></a>

## Observaciones


### Agregar datos a un gráfico de dispersión
<a name="agregar_datos_grafico_dispersion_ELTPARAGRAPHE000137"></a>

Los datos definidos con la función **grScatterAddDataXY** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).
<a name="NOTE0_2"></a>


### Agregar datos a un gráfico
<a name="agregar_datos_grafico_ELTPARAGRAPHE000151"></a>

[grAddData](../WDLang3/3042023.md) se utiliza para añadir datos en un gráfico (circular, de columnas, ...)

Dependiendo del tipo de gráfico, también se pueden utilizar funciones específicas para añadir datos: 

- Para los gráficos de acciones, se recomienda utilizar [grStockAddData](../WDLang3/3042009.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [grHMAddData](../WDLang3/1000021628.md).

- Para las cartas Sunburst, se recomienda utilizar [grSunburstAddData](../WDLang3/1000021285.md). 

- Para los gráficos de superficie, se recomienda utilizar [grSuperficieAñadirDatos](../WDLang3/1000020425.md).


Recordatorio: Para añadir un valor a los datos de un gráfico, utilice [grIncreaseData](../WDLang3/3042030.md).
<a name="NOTE0_3"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000192"></a>

Le recomendamos utilizar la función **grScatterAddDataXY** para agregar datos en gráficos de "dispersión 3D". Sin embargo, también puede utilizar [grAddData](../WDLang3/3042023.md).

La siguiente sintaxis:

```wl
grScatterAddDataXY(CHART_MyChart, i, 1, 10, 20)
```
 est es equivalente a la sintaxis:

```wl
grAddData(CHART_MyChart, i*2-1, 1, 10)
grAddData(CHART_MyChart, i*2, 1, 20)
```

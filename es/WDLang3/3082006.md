


## Funciones de archivo (gestión de ZIP, WDZ, CAB, RAR, 7z, TAR y TGZ (TAR.GZ))
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>










Las siguientes funciones de WLanguage se utilizan para crear y gestionar archivos:




|   |   |
| --- | --- |
| [zipAddDirectory](../WDLang3/3082014.md) | Añade todos los archivos encontrados en un directorio y en sus subdirectorios a un archivo y los comprime. |
| [zipAddFile](../WDLang3/3082008.md) | Añade un archivo (de cualquier tipo) a un archivo comprimido en formato CAB, ZIP, WDZ y 7z, TAR o TGZ (TAR.GZ) y lo comprime. |
| [zipAddFileList](../WDLang3/3082038.md) | Añade una lista de archivos (de cualquier tipo) a un archivo comprimido en formato ZIP, CAB, WDZ o 7z y lo comprime. |
| [zipChangePath](../WDLang3/3082027.md) | Modifica el ruta almacenado de un archivo en un archivo (en formato ZIP, WDZ o 7z). |
| [zipClose](../WDLang3/3082005.md) | Cierra un archivo. |
| [zipCompressionLevel](../WDLang3/3082054.md) | Cambia el nivel de compresión utilizado al crear un archivo comprimido en formato ZIP o en formato 7z. |
| [zipCreate](../WDLang3/3082003.md) | Crea un archivo. |
| [zipCreateExe](../WDLang3/3082013.md) | Crea un archivo autoextraíble como un ejecutable de 32 bits (.EXE). |
| [zipCurrentFile](../WDLang3/3082026.md) | Devuelve el nombre del fichero archivado que actualmente gestionan las funciones para añadir y extraer ficheros. |
| [zipDeleteAll](../WDLang3/3082051.md) | Elimina todos los archivos de un archivo comprimido. |
| [zipDeleteFile](../WDLang3/3082041.md) | Elimina un archivo de un archivo comprimido. |
| [zipDeleteFileList](../WDLang3/3082049.md) | Elimina una lista de archivos de un archivo comprimido. |
| [zipExist](../WDLang3/1000017233.md) | Se utiliza para averiguar si existe un archivo. |
| [zipExtractAll](../WDLang3/3082011.md) | Extrae todos los archivos de un archivo comprimido y los descomprime. |
| [zipExtractDirectory](../WDLang3/1000025518.md) | Extrae y descomprime automáticamente los archivos en un directorio en el árbol de archivos de un archivo a una ubicación física. |
| [zipExtractFile](../WDLang3/3082019.md) | Extrae un archivo de un archivo comprimido y lo descomprime automáticamente a una ubicación física o en memoria. |
| [zipExtractFileList](../WDLang3/1000017234.md) | Extrae y descomprime una lista de archivos encontrados en un archivo a una ubicación física. |
| [zipExtractPath](../WDLang3/3082023.md) | Devuelve el ruta de un fichero encontrado en el archivo. |
| [zipFileSize](../WDLang3/3082017.md) | Devuelve el tamaño de uno de los ficheros encontrados en el archivo (en formato CAB, RAR, WDZ, ZIP, 7z, TAR o TGZ (TAR.GZ)):<br><br>- el tamaño comprimido del archivo.<br><br>- el tamaño del archivo antes de la compresión.<br><br><br> |
| [zipFilter](../WDLang3/1000026293.md) | Devuelve la lista de formatos de archivo soportados por la plataforma actual, en el formato esperado por el filtro de [fSelect](../WDLang1/3036053.md). |
| [zipFindFile](../WDLang3/3082002.md) | Encuentra un archivo en un archivo (en formato CAB, ZIP, RAR, WDZ o 7z). |
| [zipInfoFile](../WDLang3/3082040.md) | Devuelve las características de un fichero encontrado en un archivo:<br><br>- El nombre del archivo y su ruta almacenado..<br><br>- El tamaño inicial del archivo.<br><br>- El tamaño comprimido del archivo.<br><br>- La fecha y hora de creación del archivo.<br><br>- La fecha y hora de la modificación del archivo.<br><br>- La fecha y la hora de acceso al archivo.<br><br>- Los atributos del archivo.<br><br>- El modo de encriptación del archivo.<br><br><br> |
| [zipIsMulti](../WDLang3/3082021.md) | Devuelve la clase de archivo: archivo de una sola parte o archivo de varias partes. |
| [zipListFile](../WDLang3/3082031.md) | Devuelve la lista de ficheros encontrados en un archivo. |
| [zipMerge](../WDLang3/3082046.md) | Fusiona los diferentes subarchivos que se encuentran en un archivo de varias partes para crear un archivo de una sola parte. |
| [zipMsgError](../WDLang3/3082034.md) | Devuelve el mensaje associated con un código de error en un archivo. |
| [zipNbFile](../WDLang3/3082035.md) | Devuelve el número de archivos encontrados en un archivo (en formato CAB, RAR, ZIP, WDZ o 7z). |
| [zipNbPart](../WDLang3/3082029.md) | Devuelve el número de subarchivos (partes) encontrados en un archivo en formato CAB, RAR, ZIP, WDZ, 7z, TAR o TGZ (TAR.GZ). |
| [zipNbPartNeeded](../WDLang3/3082052.md) | Devuelve el número de piezas de un tamaño determinado necesarias para dividir un archivo de una sola pieza. |
| [zipOpen](../WDLang3/3082044.md) | Abre un archivo existente. |
| [zipOpenCAB](../WDLang3/1000017250.md) | Abre un archivo existente en formato CAB. |
| [zipOpenRAR](../WDLang3/1000017294.md) | Abre un archivo existente en formato RAR. |
| [zipPassword](../WDLang3/3082053.md) | Define la contraseña utilizada para añadir archivos al archivo comprimido especificado y para extraer archivos del archivo comprimido especificado (formato RAR, ZIP y WDZ). |
| [zipSelectFile](../WDLang3/1000026152.md) | Selecciona los archivos de un archivo. |
| [zipSize](../WDLang3/3082012.md) | Devuelve el tamaño del archivo comprimido (en formato ZIP, RAR, CAB, WDZ o 7z):<br><br>- el tamaño de todos los archivos comprimidos que se encuentran en el archivo.<br><br>- el tamaño de todos los archivos encontrados en el archivo comprimido antes de la compresión.<br><br><br> |
| [zipSplit](../WDLang3/3082020.md) | Divide un archivo en varios subarchivos de un tamaño determinado. |



**Observación**: Las funciones de archivo pueden utilizarse en una Thread sin bloquear las otras Thread s durante su ejecución.

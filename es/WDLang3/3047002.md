


## Funciones J2EE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones J2EE son las siguientes:



|   |   |
| --- | --- |
| [J2EEAddHeader](../WDLang3/3047001.md) | Añade un encabezado a la próxima Procedure que se ejecutará en el servidor. |
| [J2EEError](../WDLang3/3047010.md) | Devuelve el error del último J2EE Procedure que falló. |
| [J2EEGetResult](../WDLang3/3047003.md) | Devuelve el resultado del último J2EE Procedure que fue ejecutado con éxito. |
| [J2EERun](../WDLang3/3047007.md) | Ejecuta un Procedure en un servidor de servicios web J2EE XML. |
| [J2EERunXML](../WDLang3/3047008.md) | Ejecuta un Procedure en un servidor de servicios web J2EE XML. |

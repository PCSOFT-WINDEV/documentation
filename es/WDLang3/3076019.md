


## tapiRecord( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Registra la comunicación actual como un archivo".WAV".. Esta función se puede utilizar para crear un mensaje de voz para un contestador automático.. La grabación se detiene automáticamente después de la duración especificada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The answerphone is started
// The message of the caller is currently recorded
// The maximum duration is set to 60 seconds
tapiRecord("M_" + Today() + Now() + ".wav", 60, nCallID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiRecord(<Name of WAV file> , <Maximum duration> [, <Call identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la grabación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Name of WAV file>: (Cadena de caracteres)`**

Nombre y ruta del archivo ".WAV" a crear. Se debe especificar la extensión del archivo.

**`<Maximum duration>: (Entero o Duración)`**

Duración máxima de la grabación (en segundos). Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 




**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). La llamada actual se gestiona si no se especifica este parámetro.


## HTTPCancelForm( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la declaración de un formulario HTTP y libera todos sus recursos. Este formulario ha sido creado por [HTTPCreateForm](../WDLang3/1000017119.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTTPCreateForm("FORM")
...
HTTPSendForm("FORM", "www.mysite.com")

CASE ERROR: 
HTTPCancelForm("FORM")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HTTPCancelForm(<Form name>)`
---

**`<Form name>: (Cadena de caracteres)`**

Nombre del formulario HTTP. Este nombre se especificó al crear el formulario HTTP ([HTTPCreateForm](../WDLang3/1000017119.md)).

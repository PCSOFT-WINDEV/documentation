


## grAutoRefreshCategoryLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o define si las etiquetas de las categorías se actualizan cada vez que se dibuja el gráfico.. En este caso, el contenido de la serie se recalcula automáticamente cada vez que se llama a [grDraw](../WDLang3/3042026.md). Las etiquetas de las categorías se actualizan si es necesario.

**Ejemplo de uso**: una serie en un gráfico se utiliza como serie reference: esta serie no se actualiza automáticamente.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
IF grAutoRefreshCategoryLabel("MyChart") = True THEN
	grDraw("MyChart")
END
```
<a name="Example2"></a>

```wl
// Chart control
IF grAutoRefreshCategoryLabel(CHART_MyChart) = True THEN
	grDraw(CHART_MyChart)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificando el modo utilizado para actualización las etiquetas de categoría

`<Result> = grAutoRefreshCategoryLabel(<Chart name>)`
---

**`<Result>: (booleano)`**

Modo actual para actualizar las etiquetas de las categorías: 

- <u><u><u><u>True STRIMEULB</u></u></u></u> si las etiquetas se actualizan automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.  





<a name="SYNTAX2"></a>

### Definiendo el modo a actualización las etiquetas de la categoría

`grAutoRefreshCategoryLabel(<Chart name> , <Automatic refresh>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Automatic refresh>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para una actualización automática,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si las etiquetas de la categoría están en modo automatic, las etiquetas de la categoría serán automatic ally recalculadas durante la siguiente llamada a [grDraw](../WDLang3/3042026.md).

- Si las etiquetas de categoría no están en modo automatic, las etiquetas de categoría se recalcularán durante la siguiente llamada a [grSourceCategoryLabel](../WDLang3/3042072.md).

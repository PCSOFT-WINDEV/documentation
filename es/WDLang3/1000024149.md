


## &lt;Tabla&gt;.SurfaceDeleteAltitudeColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los colores de altitud especificados por [&lt;Tabla&gt;.SurfaceAltitudeColor](../WDLang3/1000024144.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.SurfaceDeleteAltitudeColor()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceDeleteAltitudeColor([<Altitude>])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).

**`<Altitude>: (Real opcional)`**

La altitud de Color que será eliminada. Si no se especifica este parámetro, se borran todos los colores de altitud.: el gráfico está usando la Color de la primera serie.





<a name="XComponent"></a>

## Componente:
wd280grf.dll

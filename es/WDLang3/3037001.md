
## FaxStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de un fax enviado por [FaxSend](../WDLang3/3037006.md). **FaxStatus** rellena el [Estructura de FaxCompleteStatus](../WDLang3/3037011.md) que contiene las características del fax.

**Observación**: Esta función se puede utilizar en Windows 2000, XP o posterior.

**Recordatorio**: En Windows Vista (y posteriores), la gestión de faxes está disponible con Windows Professional y Windows Integral Edition.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectID is int
ConnectID = FaxConnect()
...
FaxName is string = "MyFax"
FaxID = FaxSend(ConnectID, FaxName, "Message.txt", "0102030405")
...
IF FaxStatus(ConnectID, FaxName) = FaxStatusSent THEN
	Info("The fax was sent")
ELSE
	Info("Status of the fax: " + FaxCompleteStatus.StatusCaption)
END
...
FaxDisconnect(ConnectID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxStatus(<Connection identifier> , <Fax name or Fax identifier>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del fax especificado:


|   |   |
| --- | --- |
| *FaxStatusAnswered* | El destinatario respondió. |
| *FaxStatusBadNumber* | Número de fax equivocado. |
| *FaxStatusBusy* | El destinatario está ocupado. |
| *FaxStatusCall* | El fax se está enviando. |
| *FaxStatusCanceled* | Se cancela la transmisión del fax. |
| *FaxStatusDelayed* | La llamada se ha retrasado. |
| *FaxStatusDisconnected* | Desconexión durante la transmisión. |
| *FaxStatusError* | No se le puede dar a get el estado del fax. |
| *FaxStatusFatalError* | Error fatal. |
| *FaxStatusHandled* | El fax es manejado y transmitido por el administrador de fax. |
| *FaxStatusInactive* | El envío de faxes está desactivado (pausa entre dos llamadas, por ejemplo). |
| *FaxStatusInit* | La transmisión está actualmente inicializada. |
| *FaxStatusLocked* | El número de destinatario está bloqueado. |
| *FaxStatusNoAnswer* | El destinatario no responde. |
| *FaxStatusNoDialTone* | No hay tono de llamada. |
| *FaxStatusNotFaxCall* | El dispositivo recibió una llamada que no era una llamada de fax (llamada de Phone por ejemplo). |
| *FaxStatusOffline* | El dispositivo está desconectado y no está disponible. |
| *FaxStatusReceiving* | El dispositivo está recibiendo un fax. |
| *FaxStatusRing* | El fax del destinatario está sonando. |
| *FaxStatusRouting* | El dispositivo está enrutando un fax entrante. |
| *FaxStatusSend* | El fax se está enviando actualmente. |
| *FaxStatusSent* | El fax fue enviado. |
| *FaxStatusServerPending* | El dispositivo está disponible. |
| *FaxStatusUnavailable* | El fax está ocupado (utilizado por otra aplicación). |



**`<Connection identifier>: (Integro)`**

Identificador de la conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md). Este identificador debe coincidir con el de la conexión utilizada para enviar el fax.

**`<Fax name or Fax identifier>: (Cadena de caracteres)`**

Corresponde a:

- el nombre utilizado para identificar el fax saliente. Este nombre se definió con [FaxSend](../WDLang3/3037006.md).

- el identificador del fax (devuelto por [Bandeja de entrada de fax](../WDLang3/3037009.md) o [FaxOutbox](../WDLang3/3037008.md))

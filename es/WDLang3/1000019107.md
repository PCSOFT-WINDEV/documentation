
## SaaSAdminAddService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un servicio SaaS en la description de un sitio.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add the "Post a message" service into the "Forum" site
S is dynamic saasSite 
MessageService is saasService

S = SaaSAdminFindSite("Forum")
MessageService.Name = "Post a message"

SaaSAdminAddService(MessageService, S)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadiendo un servicio SaaS (a través de las variables saasxxx)

`<Result> = SaaSAdminAddService(<SaaS service> , <SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS service>: (saasService Variable)`**

Nombre de la variable [saasService](../WDLang3/1000019096.md) que contiene el description del servicio a añadir.

**`<SaaS site>: (Variable de tipo saasSite)`**

Nombre de [saasSite](../WDLang3/1000019068.md) Variable correspondiente al sitio SaaS que recibe el nuevo servicio.


<a name="SYNTAX2"></a>

### Añadiendo un nuevo servicio SaaS identificado por su nombre

`<Result> = SaaSAdminAddService(<SaaS service> , <SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS service>: (Cadena de caracteres)`**

Nombre del nuevo servicio SaaS.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS que recibe el nuevo servicio.

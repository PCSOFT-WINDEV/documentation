


## zipCompressionLevel( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el nivel de compresión utilizado al crear un archivo comprimido en formato ZIP o en formato 7z.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
NumZipErr is int
MyArchive is zipArchive

// Create the archive named "MyArchive"
NumZipErr = zipCreate(MyArchive, "C:\MyDocuments\Archive.zip")
IF NumZipErr <> 0 THEN
	Error(zipMsgError(NumZipErr))
	RETURN
END

// Add a file and use the default compression level (6)
NumZipErr = zipAddFile(MyArchive, "C:\MyImages\Image1.jpg", zipNone)
IF NumZipErr <> 0 THEN
	Error(zipMsgError(NumZipErr) + "On the file " + zipCurrentFile(MyArchive))
END

// Change the compression level
// The next file will be stored (no compression)
zipCompressionLevel(MyArchive, 0)

// Add a file
NumZipErr = zipAddFile(MyArchive, "C:\MyImages\Image2.jpg", zipNone)
IF NumZipErr <> 0 THEN
	Error(zipMsgError(NumZipErr) + "On the file " + zipCurrentFile(MyArchive))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = zipCompressionLevel(<Archive> , <Compression level>)`
---

**`<Result>: (Integro)`**



- 0 si se ha cambiado el nivel de compresión,

- Un código de error (valor superior a 0) en caso contrario (ver Notas).




**`<Archive>: (Cadena de caracteres o zipArchive Variable)`**

Nombre del archivo que se va a utilizar.
Este nombre puede corresponder a:

- un nombre de archivo definido con [zipOpen](../WDLang3/3082044.md) o [zipCreate](../WDLang3/3082003.md).

- el nombre de una variable de tipo [zipArchive](../WDLang3/1000018679.md). 
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las variables [zipArchive](../WDLang3/1000018679.md) no están disponibles en Java.




**`<Compression level>: (Integro)`**

Valor incluido entre 0 (almacenamiento sin compresión) y 9 (compresión máxima). 
El nivel de compresión por defecto es set a 6.

## Observaciones


### Códigos de error
<a name="codigos_error_ELTPARAGRAPHE000123"></a>

Se devuelven los siguientes códigos de error:

- 15: Archivo en formato CAB.

- 27: Archivo en formato WDZ.

- 28: Nivel de compresión no válido.




El mensaje correspondiente al código de error es devuelto por [zipMsgError](../WDLang3/3082034.md).
<a name="NOTE0_2"></a>


### Nivel de compresión y tiempo de adición
<a name="nivel_compresion_tiempo_adicion_ELTPARAGRAPHE000139"></a>

**Atención**: Un alto nivel de compresión significa que la adición de un archivo al archivo comprimido llevará más tiempo.
<a name="NOTE0_3"></a>


### Notas
<a name="notas_ELTPARAGRAPHE000146"></a>

El nivel de compresión se define mediante la llamada a la función para todos los ficheros que se añadirán posteriormente al archivo (el nivel de compresión para los ficheros ya encontrados en el archivo sigue siendo el mismo).

**Atención**: No se puede cambiar el nivel de compresión de un archivo encontrado en un archivo comprimido existente.

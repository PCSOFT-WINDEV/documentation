
## SaaSAdminConfigureCurrency( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la moneda utilizada en los informes de conexión del sistema SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to Pounds
SaaSAdminConfigureCurrency("£")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminConfigureCurrency(<Currency>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la configuración,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Currency>: (Cadena de caracteres)`**

Caracter que simboliza la moneda que se debe utilizar en los informes de conexión del sistema SaaS.

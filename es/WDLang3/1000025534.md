


## MapDeleteImagen (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una Image dibujada en el mapa que se muestra en una control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete an image from the Map control
MapDeleteImage(MAP_MyMap, Image)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borrar una imagen identificada por la Variable asociada

`<Result> = MapDeleteImage(<Map control> [, <Image>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la Image fue borrada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Image>: (Opcional MapImage Variable)`**

Nombre de la [MapImage](../WDLang3/1000025523.md) Variable que corresponde al Image que se va a eliminar. Si no se especifica este parámetro, se borrarán todas las imágenes del mapa.


<a name="SYNTAX2"></a>

### Suprimir una Image identificada por su nombre

`<Result> = MapDeleteImage(<Map control> , <Image name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la Image fue borrada, 

- <u><u><u><u>false</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Image name>: (Cadena de caracteres)`**

El nombre de la Image será borrado. Este nombre corresponde a la propiedad **Name** de la variable de tipo [MapImage](../WDLang3/1000025523.md) que se va a eliminar. Si no se especifica este parámetro, se borrarán todas las imágenes del mapa.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para añadir una nueva Image que se dibuje en un mapa, utilice [MapAddImage](../WDLang3/1000025532.md).

- Para modificar una Image dibujada en una control Mapa, utilice [MapModifyImage](../WDLang3/1000025533.md).

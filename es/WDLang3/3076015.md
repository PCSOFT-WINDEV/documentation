
## Constantes de telefonía
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Detalles | Utilizado por la función |
| --- | --- | --- |
| **tapiCallInformation** | La información adicional (presentación del número) está disponible. En la mayoría de los casos, esta información estará disponible después del primer anillo. | [tapiEscuchar](../WDLang3/3076029.md) |
| **tapiPlayLock** | **tapiPlay** es una función de bloqueo: el código continúa ejecutándose sólo cuando se reproduce el sonido. Si no se especifica este parámetro, la función devuelve un resultado tan pronto como el sonido comienza a sonar. | [tapiPlay](../WDLang3/3076023.md) |
| **tapiPlayLoop** | El sonido se reproduce en bucle.<br>Si no se especifica este parámetro, el sonido se reproduce una sola vez. | [tapiPlay](../WDLang3/3076023.md) |
| **tapiLineProceeding** | Estado de la Line: La llamada se marca: búsqueda del corresponsal | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineConnected** | Estado de la Line: El Line está conectado | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineDialing** | Estado de la Line: Marcación en curso | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineBusy** | Estado de la Line: El Line está actualmente ocupado | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineDisconnected** | Estado de la Line: El corresponsal ha colgado | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineRingBack** | Estado de la Line: Timbre en curso | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiLineDialTone** | Estado de la Line: El Line recibe un tono de llamada | [tapiDial](../WDLang3/3076013.md), [tapiLineDial](../WDLang3/3076012.md), <br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiNewCall** | Nueva llamada detectada en espera de respuesta o de rechazo. | [tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOptionDefault** | Detección de llamadas en un módem fax. En la mayoría de los casos, las funciones para manejar llamadas no son soportadas por los módems. | [tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOptionMonitor** | Detecta las llamadas (para calcular estadísticas, por ejemplo), incluso si son gestionadas por otra aplicación.. No se podrá realizar ninguna operación durante la llamada. | [tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOptionMediaModeFax** | Uso de un dispositivo que admita llamadas entrantes de fax | [tapiCapacidad](../WDLang3/3076014.md)<br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOptionMediaModeUnknown** | Uso de un dispositivo que admite llamadas entrantes desconocidas | [tapiCapacidad](../WDLang3/3076014.md)<br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOptionMediaModeVoice** | Uso de un dispositivo compatible con llamadas entrantes de voz | [tapiCapacidad](../WDLang3/3076014.md)<br>[tapiEscuchar](../WDLang3/3076029.md) |
| **tapiOriginConference** | Llamada en modo conferencia (varias personas) | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiOriginInbound** | Llamada entrante: comunicación desde el exterior. El uso de este Variable depende del tipo de dispositivo utilizado, ... | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiOriginExternal** | El origen de la llamada no está definido. El uso de este Variable depende del tipo de dispositivo utilizado, ... | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiOriginUnavailable** | El origen de la llamada no está definido. El uso de este Variable depende del tipo de dispositivo utilizado, ... | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiOriginInternal** | Llamada interna a PABX | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiOriginOutbound** | Llamada saliente: comunicación hacia el exterior. | [tapiOrigen](../WDLang3/3076028.md) |
| **tapiMonitoredDigit** | Se ha utilizado una llave de Phone. | [tapiEscuchar](../WDLang3/3076029.md) |




## Gestión de los faxes entrantes o salientes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000130"></a>
Para enviar y recibir faxes directamente desde una aplicación WINDEV o un sitio WEBDEV, varias funciones le permiten enviar faxes manipular a través del servidor de fax disponible en Windows XP y Windows 2000. En Windows Vista, la gestión de faxes está disponible en Windows Vista Professional Edition y Windows Vista Ultimate Edition.

Recordatorio: los faxes también se pueden enviar desde una aplicación WINDEV o desde un sitio WEBDEV. Para obtener más información, consulte [Envío de fax](../WDLang3/3037002.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo enviar faxes manipular?
<a name="¿como_enviar_faxes_manipular_ELTTEXTE000154"></a>
Para manipular los faxes entrantes o salientes de una aplicación WINDEV o de un sitio WEBDEV, usted debe:

- [configurar el "Servidor de fax".](../WDLang3/3037016.md).

- crear la aplicación WINDEV/WEBDEV. 

- [configurar (si es necesario) las opciones del servidor de fax a través de la programación](../WDLang3/3037016.md).




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Programación del uso de faxes
<a name="programacion_del_uso_faxes_ELTTEXTE000178"></a>
Para utilizar los faxes entrantes o salientes de una aplicación WINDEV, debe: 

1. Leer el contenido de la bandeja de entrada ([Bandeja de entrada de fax](../WDLang3/3037009.md)) o el contenido de la bandeja de salida ([FaxOutbox](../WDLang3/3037008.md)) de los faxes. Estas funciones se utilizan para recuperar los identificadores numéricos de los faxes entrantes o pendientes.

2. Los identificadores de fax le permiten hacerlo:

	- Conocer el estado del fax y sus características (estructura [FaxStatus](../WDLang3/3037001.md) y [Estado de FaxCompleteStatus](../WDLang3/3037011.md)).

	- Maneje el fax con las siguientes funciones:
			


|   |   |
| --- | --- |
| [FaxDelete](../WDLang3/3037014.md) | Elimina un fax del administrador de trabajos de impresión y lo cancela.. Entonces, [FaxStatus](../WDLang3/3037001.md) ya no puede ser utilizada en este fax. |
| [FaxReiniciar](../WDLang3/3037012.md) | Reiniciar un fax en el administrador de trabajos de impresión. Esta función sólo se aplica a los faxes pendientes. |
| [FaxResume](../WDLang3/3037013.md) | Vuelve a activar un fax en pausa en el administrador de trabajos de impresión (fax pendiente). |
| [Pausa fax](../WDLang3/3037015.md) | Pone en pausa un fax en el spooler. Si el fax está activado, se pondrá en pausa cuando vuelva al administrador de trabajos de impresión. |

<a name="Use"></a>
<a name="description"></a>
Enumera los directorios y los archivos encontrados:

- en el servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV, 

- en el ordenador del cliente.




**Observación**: Para permitir (o no) el uso de esta función en un servidor FTP/RPC, llame a [NetServerOpción](../WDLang3/3056022.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectRPC is int
ConnectRPC = NetConnect("148.61.125.245", RPCServer, "GUEST", "")
...
// Displays the list of directories and files found on the C drive: of the server
Info(NetDirList(ConnectRPC, "C:\*.*"))
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetDirList([<Connection identifier>, ] <Name of Home Directory>)`
---

**`<Result>: (Cadena de caracteres ANSI)`**

Lista de directorios y archivos (con su tamaño) que se encuentran en el servidor o en el ordenador cliente. Este resultado tiene el siguiente formato: 

```txt
<Directory1> + TAB + <Directory2> + TAB + ... + TAB + <DirectoryN> + CR + ...
<File1/Size of File 1 in KB> + TAB + <File2/Size of File 2 in KB> + ...
TAB + ... + TAB + <FileN/Size of File N in KB>
```


**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión para el ordenador cliente (devuelto por [NetConnect](../WDLang3/3056004.md)). Si no se especifica este parámetro, **NetDirList** devuelve la lista de directorios y archivos en la computadora cliente actual

**`<Name of Home Directory>: (Cadena de caracteres ANSI)`**

Directorio del ordenador servidor o cliente cuyos archivos y subdirectorios deben ser listados. Debe especificarse el filtro utilizado para la búsqueda (\*.\*.. por ejemplo,...).

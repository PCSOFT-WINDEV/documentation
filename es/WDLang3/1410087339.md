
## Descarga_constantes
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de descarga utilizan las siguientes constantes:

| **downloadNotifNone** | Descargar el modo Notification: No se muestran notificaciones. | Tipo [DownloadParameter](../WDLang3/1410087293.md) |
| --- | --- | --- |
| **downloadNotifProgressAndCompletion** | Descargar el modo Notification: Un Notification muestra el progreso de la descarga y se queda Visible cuando la descarga está completa. | Tipo [DownloadParameter](../WDLang3/1410087293.md) |
| **downloadNotifProgress** | Descargar el modo Notification: Una Notification muestra el progreso de la descarga y desaparece cuando ésta se completa. | Tipo [DownloadParameter](../WDLang3/1410087293.md) |
| Constante | Detalles | Tipo de variable |
| **downloadErrorStorage** | Error encontrado: Error de almacenamiento. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadErrorUnknown** | Error encontrado: Error desconocido. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadErrorInsufficientSpace** | Error encontrado: Espacio de almacenamiento insuficiente. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadErrorFileAlreadyExists** | Error encontrado: La URI apunta a un archivo existente. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadStatusPaused** | Estado de la descarga: Descarga interrumpida. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadStatusSuccessful** | Estado de la descarga: Descargar con éxito. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadStatusRunning** | Estado de la descarga: La descarga está en curso. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadStatusFailed** | Estado de la descarga: La descarga ha fallado. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadStatusPending** | Estado de la descarga: Pendiente de descarga. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadPausedWaitingForWifi** | La descarga se detiene hasta que se encuentra una red Wi-Fi. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadPausedWaitingForNetwork** | La descarga se detiene hasta que se encuentra una red. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadPausedUnknown** | La descarga se detiene por una razón desconocida. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |
| **downloadPausedWaitingToRetry** | La descarga se detuvo debido a un error de red. El sistema esperará antes de volver a intentarlo. | Tipo [DownloadInfo](../WDLang3/1410087306.md) |

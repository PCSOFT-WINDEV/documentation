
## geoGetArea( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las coordenadas del área geográfica que corresponde a una determinada description.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Show France in a Map control 
let (p1,p2) = geoGetArea("France")
IF NOT ErrorOccurred() THEN
	MapDisplayArea(MAP_Map1, p1, p2)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`(<NW position>, <SE position>) = geoGetArea(<Description>)`
---

**`<NW position>: (geoPosition Variable)`**

[geoPosition](../WDLang3/1000019191.md) Variable que corresponde a la zona geográfica del Noroeste.

**`<SE position>: (geoPosition Variable)`**

[geoPosition](../WDLang3/1000019191.md) Variable que corresponde a la zona geográfica del sureste.

**`<Description>: (Cadena de caracteres)`**

Área geográfica cuyas coordenadas deben ser recuperadas. Por ejemplo: "París, Francia".

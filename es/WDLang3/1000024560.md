
## saasVisit (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasVisit** permite conocer las características de la visita de un usuario SaaS. Las características de este tipo de variable se pueden conocer mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
T is array of saasVisit 
T = SaaSAdminListVisit("MySaaSSite", "Client1")
```

## Observaciones

### Propiedades específicas de la descripción de variables de tipo saasVisit
<a name="propiedades_especificas_descripcion_variables_tipo_saasvisit_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **saasVisit**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Date | DateTime | Fecha y hora de inicio de sesión del usuario. <br><br>**Esta propiedad es de solo lectura**. |
| Duration | Duration | Duración de la conexión. <br><br>**Esta propiedad es de solo lectura**. |
| Site | [saasSite](../WDLang3/1000019068.md) | Sitio en el que el usuario inició sesión. <br><br>**Esta propiedad es de solo lectura**. |
| User | [saasUser](../WDLang3/1000019083.md) | Usuario que visitó el sitio. <br><br>**Esta propiedad es de solo lectura**. |

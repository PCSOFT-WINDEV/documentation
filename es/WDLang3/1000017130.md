
## BTChangeConnectivity( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite configurar una radio Bluetooth para que acepte (o no) las peticiones de conexión procedentes de los dispositivos..  
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTChangeConnectivity([<Radio identifier>, ] <New status>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si el cambio de conectividad se ha realizado con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Radio identifier>: (Entero opcional)`**

Identificador de la radio Bluetooth a utilizar. Este identificador es devuelto por [BTListRadio](../WDLang3/1000017125.md). Si no se especifica este identificador, se utiliza la radio Bluetooth actual. 

**`<New status>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para que la radio acepte las conexiones,

- <u><u><u><u>False</u></u></u></u> para que la radio rechace las conexiones.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo la pila Microsoft Bluetooth es compatible con esta función. Para conocer la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).  

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

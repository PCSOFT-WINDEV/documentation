


## UPNPModifyVolume (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambiar el volumen de un dispositivo UPnP.

**Atención**: Para utilizar esta función, el dispositivo debe incluir un servicio "RenderingControl".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrDevice is array of upnpDevices 
arrDevice = UPNPListDevice(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
FOR EACH Device of arrDevice
	UPNPModifyVolume(Device, 50)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UPNPModifyVolume(<UPnP device> , <New value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el volumen, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<UPnP device>: (Variable de tipo upnpDevice)`**

Nombre de [upnpDevice](../WDLang3/1000020903.md) Variable correspondiente al dispositivo UPnP a utilizar.

**`<New value>: (Integro)`**

Nuevo volumen a aplicar. Este parámetro debe estar comprendido entre 0 y 100.

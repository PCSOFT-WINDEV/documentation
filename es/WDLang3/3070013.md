
## SocketExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de la toma de corriente.

**Observaciones**: 

- Para comprobar la validez de la conexión, utilice [SocketClientInfo](../WDLang3/3070009.md) para comprobar la IP Address del socket cliente y el número de puerto. 

- El socket existe mientras el socket creado no haya sido cerrado por [SocketClose](../WDLang3/3070015.md). 

- Si se interrumpe la comunicación, la función **SocketExist** podría devolver un resultado positivo incluso si se ha detenido el diálogo.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the existence before the creation
IF SocketExist("MySocket") = False THEN
	IF SocketCreate("MySocket", 8000) = False THEN
		 Error("Creation error" + ErrorInfo(errMessage))
	 END
ELSE
	 SocketRead("MySocket", False)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SocketExist(<Socket name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el socket,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Socket name>: (Cadena de caracteres)`**

Nombre del zócalo a comprobar. 

Para una aplicación WINDEV, se definió este nombre:

- por [SocketCreateInfrarrojo](../WDLang3/3070012.md), [SocketCreate](../WDLang3/3070011.md) o [SocketCreateSSL](../WDLang3/1000017041.md) (para un servidor simplificado). 

- por [SocketAccept](../WDLang3/3070004.md) (para un servidor estándar).

- por [SocketConnect](../WDLang3/3070016.md) (para un sitio WEBDEV).


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Atención: el nombre del socket distingue entre mayúsculas y minúsculas.

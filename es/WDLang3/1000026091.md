
## Funciones Modbus (sintaxis del prefijo)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas son las funciones Modbus (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;modbusSession&gt;.Desconectar](../WDLang3/1000025954.md) | Cierra la sesión de Modbus. |
| [&lt;modbusSession&gt;.ReadBit](../WDLang3/1000026065.md) | Lee uno o más bits de un esclavo Modbus, empezando por una Address específica. |
| [&lt;modbusSession&gt;.ReadFloat](../WDLang3/1000026079.md) | Lee uno o más flotadores de una sola precisión de un esclavo Modbus, a partir de una Address específica. |
| [&lt;modbusSession&gt;.ReadRegister](../WDLang3/1000026080.md) | Lee uno o más registros de un esclavo Modbus, comenzando en una Address específica. |
| [&lt;modbusSession&gt;.WriteBit](../WDLang3/1000025967.md) | Escribe uno o varios bits a una Address específica en un esclavo Modbus. |
| [&lt;modbusSession&gt;.WriteFloat](../WDLang3/1000026035.md) | Escribe uno o más flotadores de precisión simple a un esclavo Modbus, comenzando en una Address específica. |
| [&lt;modbusSession&gt;.WriteRegister](../WDLang3/1000026036.md) | Escribe uno o más registros (valores de 16 bits) a un esclavo Modbus, a partir de una Address específica. |

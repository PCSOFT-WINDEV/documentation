


## UPNPFindDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un dispositivo basado en su nombre único (UDN).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Device is upnpDevice
Device = UPNPFindDevice("uuid:72047cc4-1dd2-11b2-87b7-0000aaae85f5", 51300)
IF ErrorOccurred THEN
	Info(ErrorInfo(errFullDetails))
ELSE
	Info("Device found")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UPNPFindDevice(<Device name> , <Port> [, <Address> [, <Timeout>]])`
---

**`<Result>: (Variable de tipo upnpDevice)`**

Nombre de la [upnpDevice](../WDLang3/1000020903.md) Variable que contiene las características del dispositivo buscado.

**`<Device name>: (Cadena de caracteres)`**

Nombre único (UDN) del dispositivo a buscar.

**`<Port>: (Integro)`**

Número de puerto utilizado para la búsqueda.

**`<Address>: (Cadena de caracteres opcional)`**

Address en la que se crea el zócalo de búsqueda.

**`<Timeout>: (Entero o duración opcional)`**

Tiempo máximo de espera expresado en segundos. Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, "1s", "10cs" o "10ms"). 


Este parámetro corresponde a 5 segundos por Default. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si no se encuentra el dispositivo, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.




## &lt;Mapa&gt;.DeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra lo siguiente de una control Mapa: 

- todos los marcadores,

- todas las formas,

- todas las imágenes.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all markers from the Map control
MAP_MyMap.DeleteAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Map control>.DeleteAll()`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Gestión de marcadores: 

	- Para eliminar una Marker específica de una control Mapa, utilice [&lt;Mapa&gt;.DeleteMarker](../WDLang3/1000022466.md).

	- Para añadir una nueva Marker a una control Mapa, utilice [&lt;Mapa&gt;.AddMarker](../WDLang3/1000022422.md).

	- Para cambiar una Marker en una control Mapa, utilice [&lt;ModificarMarcador](../WDLang3/1000022383.md).




- Gestión de itinerarios: Para eliminar un itinerario concreto, utilice [&lt;Mapa&gt;.DeleteItinerario](../WDLang3/1000022465.md).

- Imágenes: 

	- Para eliminar una Image específica de una control Mapa, utilice [&lt;Mapa&gt;.BorrarImagen](../WDLang3/1000025829.md). 

	- Para añadir una nueva Image a una control Mapa, utilice [&lt;Mapa&gt;.AddImage](../WDLang3/1000025802.md). 

	- Para modificar una Image en una control Mapa, utilice [&lt;Mapa&gt;.ModificarImagen](../WDLang3/1000025818.md). 




- Formas: 

	- Para eliminar una forma específica de una control Mapa, utilice [&lt;Mapa&gt;.BorrarForma](../WDLang3/1000025820.md). 

	- Para añadir una nueva forma a una control Mapa, utilice [&lt;Mapa&gt;.AddShape](../WDLang3/1000025801.md). 

	- Para cambiar una forma en una control Mapa, utilice [&lt;Mapa&gt;.ModifyShape](../WDLang3/1000025805.md).




## grIncreaseData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un valor a un dato en un gráfico. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds the value 7 to the second data of the first series
grIncreaseData("MyChart", 1, 2, 7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grIncreaseData(<Chart name> , <Series number> , <Index> , <Value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie a la que se añadirá un valor. Si la serie especificada no existe, el diseñador de la carta crea tantas series como sean necesarias para get una matriz consistente. Todas las series creadas se inicializan a 0.

**`<Index>: (Integro)`**

Índice de datos de la serie. Si los datos especificados no existen, el diseñador de la carta crea tantos datos como sean necesarios para get una matriz de datos consistente. Los datos creados se inicializan a 0.



**`<Value>: (Entero o real)`**

Valor que se añadirá a los datos especificados.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tipos de gráficos
<a name="tipos_graficos_ELTPARAGRAPHE000129"></a>

Para los gráficos circulares, sólo se utilizarán los datos encontrados en la primera serie.
<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000138"></a>

- Los datos definidos con la función **grIncreaseData** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- Una serie puede contener hasta 8000 elementos.

- No se puede dibujar un gráfico circular si los datos encontrados en la primera serie son nulos.

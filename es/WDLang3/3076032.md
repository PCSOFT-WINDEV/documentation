
## tapiDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona el dispositivo TAPI que se utilizará durante las siguientes operaciones de telefonía:

- [tapiDial](../WDLang3/3076013.md),

- [tapiEscuchar](../WDLang3/3076029.md),



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the list of TAPI devices
sTAPIList is string = tapiDeviceList()

// Displays the list in a Combo Box control
ListAdd(COMBO_TAPI, sTAPIList) 
...
// Selects the TAPI device chosen in the Combo Box control
tapiDevice(COMBO_TAPI[COMBO_TAPI])
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiDevice(<Device name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se seleccionó el dispositivo. El dispositivo seleccionado se utilizará para todas las demás operaciones de telefonía.

- <u><u><u><u>False</u></u></u></u> si se produce un error. 




**`<Device name>: (Cadena de caracteres)`**

Nombre del dispositivo a utilizar. La lista de dispositivos de telefonía es devuelta por [tapiDeviceList](../WDLang3/3076024.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Limitación
<a name="limitacion_ELTPARAGRAPHE000051"></a>

No se puede llamar a la función **tapiDevice** en el procedimiento de detección de llamadas (definido con la función [tapiEscuchar](../WDLang3/3076029.md)).
<a name="NOTE0_2"></a>


### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000190"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.




## EmailMessageLength (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el tamaño del mensaje especificado (sin cargar el mensaje). El tamaño corresponde al espacio que ocupa la Email en el buzón (tamaño del mensaje y de los archivos adjuntos).

**Esta función se puede utilizar con los protocolos POP3 e IMAP.**

**Recordatorio**: Antes de leer una Email, es necesario iniciar una sesión con [EmailStartSession](../WDLang3/3032028.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the emails whose size does not exceed nMaxSize 
I is int
FOR I = 1 TO EmailNbMessage(UserName)
	IF EmailMessageLength(UserName, I) < nMaxSize THEN
		// Read and process the message
		ReadMessage()
	ELSE
		Info("The size of the email exceeds the maximum size and it cannot be read")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = EmailMessageLength(<Session> [, <Message number>])`
---

**`<Result>: (Integro)`**

Tamaño total (en bytes) de Email (mensaje y archivos adjuntos).

**`<Session>: (Cadena de caracteres, o emailPOP3Session o emailIMAPSession Variable)`**

Identifica la sesión de usuario. Corresponde a:

- una cadena de caracteres utilizada como nombre de usuario para iniciar la sesión de email ([EmailStartSession](../WDLang3/3032028.md)).

- una variable de tipo [emailPOP3Session](../WDLang3/1000018759.md).

- una variable de tipo [emailIMAPSession](../WDLang3/1000018957.md).




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) las variables [emailPOP3Session](../WDLang3/1000018759.md) y [emailIMAPSession](../WDLang3/1000018957.md) no están disponibles.

**`<Message number>: (Entero opcional)`**

Identifica el mensaje cuyo tamaño se debe calcular. Si no se especifica este parámetro, &lt;Resultado&gt; devuelve el tamaño del correo electrónico actual.

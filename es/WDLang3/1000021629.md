


## grHMColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica los colores de los diferentes valores en un gráfico Heatmap.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the colors
grHMColor(CHART_HMap, -50, DarkBlue)
grHMColor(CHART_HMap, +50, LightRed)
FOR i = 1 TO 500
	grHMAddData(CHART_HMap, Random(-100, 100), Random(-100, 100), -100)
	grHMAddData(CHART_HMap, Random(-100, 100), Random(-100, 100), 100)
END
grDraw(CHART_HMap)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grHMColor(<Chart name> , <Level> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Level>: (Real o constante)`**

Nivel a partir del cual se aplicará el Color. Corresponde a: 

- un valor real,

- la constante *grHMColorEmptyValue*. En este caso, la Color se aplicará a los datos fuera de nivel o sin valor. Este color es Transparent de Default. 




**`<Color>: (Entero o constante de tipo Integer)`**

Color que se entregará al nivel. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  

- la constante *AutomaticColor*. En este caso, se borra el nivel. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de colores
<a name="gestion_colores_ELTPARAGRAPHE000098"></a>

Los colores se definen por nivel. Para los valores encontrados entre 2 niveles, se aplica un Gradient lineal por Default.

Este comportamiento puede modificarse con la función [grParameter](../WDLang3/1000018337.md) y la constante *grHMDisplayColor*. Se utiliza la siguiente sintaxis:

```txt
grParameter(<Chart name>, grHMDisplayColor, <Display type>)
```



 donde &lt;Tipo de pantalla&gt; corresponde a una de las siguientes constantes:



|   |   |
| --- | --- |
| *grColorGradient*<br>(Valor predeterminado) | Se aplica un degradado lineal para los valores intermedios. |
| *grLowerLevel* | La visualización de los colores se realiza por niveles. Todos los valores menores o iguales al nivel tomarán la Color dada por [grHMColor](../WDLang3/1000021629.md). |
| *grHigherLevel* | La visualización de los colores se realiza por niveles. Todos los valores mayores o iguales al nivel tomarán la Color dada por [grHMColor](../WDLang3/1000021629.md). |


Para las facetas fuera de los límites de color o sin valor, se aplicará el color especificado por *grHMColorEmptyValue*. Este color es Transparent de Default.

Observación: Los colores definidos con la función **grHMColor** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

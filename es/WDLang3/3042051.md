


## grRaised (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un gráfico con efecto de relieve. Basta con especificar la profundidad del efecto. Todos los tipos de gráficos se pueden mostrar con un efecto elevado.

Ejemplo de gráfico con efecto en relieve: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Relief.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the depth of raised effect for "MyChart" (created through programming)
grRaised("MyChart", 5)
```
<a name="Example2"></a>

```wl
// Increase the depth of raised effect for CHART_MyChart
grRaised(CHART_MyChart, 7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grRaised(<Chart name> , <Raised>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Raised>: (Integro)`**

Profundidad del efecto de aumento (en píxeles). Le aconsejamos que no utilice un efecto de alto relieve para que el gráfico se muestre correctamente..
Si este parámetro es igual a 0, el gráfico se muestra en dos dimensiones (sin efecto en relieve)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Todos los tipos de gráficos se pueden mostrar con un efecto elevado.

- Los datos definidos con la función **grRaised** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

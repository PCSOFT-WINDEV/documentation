


## grSeriesLineType (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el tipo de Line para una serie que se encuentra en un gráfico de Line, Área, Dispersión y Radar.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines dotted lines for series 2
grSeriesLineType(CHART_MyChart, 2, LineDotted)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSeriesLineType(<Chart name> , <Series number> , <Type>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Series number>: (Integro)`**

Número de la serie para la que debe definirse el tipo de Line.

**`<Type>: (Constante de tipo Integer)`**

Tipo de Line para la serie: 



|   |   |
| --- | --- |
| *LineDash* | Line con guiones (---). |
| *LineDotAndDash* | Line mezcla puntos y guiones (.-.-.-.-.-). |
| *LineDotted* | Línea de puntos (.......). |
| *LineSolid* | Línea continua. |

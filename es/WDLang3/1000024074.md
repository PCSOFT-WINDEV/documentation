


## &lt;Gráfico&gt;.existente (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un gráfico en la memoria. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks the existence of CHART_MyChart
ResExist = CHART_MyChart.Exist()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.Exist()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el gráfico,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).





<a name="XComponent"></a>

## Componente:
wd280grf.dll




## grColumnSpacing (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el espacio entre cada categoría de datos en: 

- un gráfico de columnas,

- un gráfico de Funnel,

- un gráfico de acciones Mínimo/Máximo.




Ejemplos de gráficos de barras con espacio entre categorías: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EspaceHisto.gif)


**Recordatorio**: El espacio entre dos categorías de datos es igual a 10 píxeles por Default.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Space between two data categories in the "MyChart" column chart: 20 pixels
grColumnSpacing("MyChart", 20)
```
<a name="Example2"></a>

```wl
// Space between two data categories in the "CHART_MyChart" column chart: 15 pixels
grColumnSpacing(CHART_MyChart, 15)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grColumnSpacing(<Chart name> , <Spacing>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Spacing>: (Integro)`**

Espaciado (en píxeles) entre dos categorías.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Espaciado
<a name="espaciado_ELTPARAGRAPHE000134"></a>

Si el destino es suficientemente grande, el espacio se expresa de la siguiente manera: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Grespace.gif)


Si el destino no permite mostrar el espacio, el espacio es set a 0: por lo tanto, las barras están "pegadas".
<a name="NOTE0_2"></a>


### Efecto debido al espaciado
<a name="efecto_debido_espaciado_ELTPARAGRAPHE000146"></a>

- El espaciamiento entre dos categorías de datos no tiene ningún efecto en los gráficos aparte de: 

	- Gráficos de barras, 

	- Cartas de embudo,

	- Gráficos de Acciones Mínimas/Máximas.




- Las modificaciones realizadas con la función **grColumnSpacing** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).


## SaaSAdminPricingEnableService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite un servicio a un precio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite
P is dynamic saasPricing
CRMService is dynamic saasService

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
CRMService = SaaSAdminFindService("CRM", S)

// The subscribers to the "2010 promo" pricing can now access the CRM section of Forum
SaaSAdminPricingEnableService(P, CRMService)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Habilitar un servicio (mediante variables saasxxx)

`<Result> = SaaSAdminPricingEnableService(<Pricing> , <SaaS service>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la activación se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (saasPricing Variable)`**

Nombre de [saasPricing](../WDLang3/1000019091.md) Variable correspondiente a la tarificación a la que se añade un servicio.

**Observación**: El servicio debe haber sido creado previamente por [Servicio SaaSAdminAddService](../WDLang3/1000019107.md).

**`<SaaS service>: (saasService Variable)`**

Nombre del [saasService](../WDLang3/1000019096.md) Variable correspondiente al servicio SaaS que se añadirá a la lista de servicios prestados por la tarificación.


<a name="SYNTAX2"></a>

### Habilitar un servicio identificado por su nombre

`<Result> = SaaSAdminPricingEnableService(<Pricing> , <SaaS service> , <SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la activación se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre de la tarificación a la que se añade un servicio SaaS.

**`<SaaS service>: (Cadena de caracteres)`**

Nombre del servicio SaaS que se añadirá a la lista de servicios proporcionados por la tarificación.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS afectado por el precio y el servicio.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El servicio se pondrá automáticamente a disposición de todos los abonados a esta tarifa (y no sólo de los nuevos).

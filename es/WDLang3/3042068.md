


## grOverlayChart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para mostrar dos tipos diferentes de gráficos en el mismo gráfico. Los dos gráficos están superpuestos.

**Observación**: El primer gráfico debe ser un gráfico de columna (gráfico de columna agrupado o apilado), un Gráfico de cotizaciones (MinMax o Candlestick) o un gráfico Scatter, mientras que el segundo gráfico debe ser un gráfico Line.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF grOverlayChart(CHART_MyColumnChart, CHART_Line) = True THEN
	Info("The charts are overlaid")
END
grDraw(CHART_MyColumnChart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grOverlayChart(<Destination chart> , <Source chart>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se superpusieron los gráficos,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Destination chart>: (Cadena de caracteres)`**

Nombre del gráfico que se utilizará para mostrar los dos gráficos. Este gráfico contiene el primer gráfico (Gráfico de barras, Gráfico de cotizaciones o gráfico de dispersión).

**`<Source chart>: (Cadena de caracteres)`**

Nombre del gráfico que se mostrará en el gráfico de destino. Esta tabla debe ser una tabla de Line. Se superpondrá al gráfico de destino. Esta tabla se tendrá en cuenta como fuente de datos: su título y sus parámetros serán ignorados. Sólo se conservará la información relativa a la leyenda..
El enlace entre los dos gráficos se borra si este parámetro corresponde a una cadena vacía (""").
Este parámetro debe corresponder a un gráfico diferente de &lt;Gráfico de destino&gt; (de lo contrario, los gráficos se sobrescriben entre sí).. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La superposición de gráficos definida con la función **grOverlayChart** se tendrá en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- Atención: **grOverlayChart** no manipular las cruces de los diferentes gráficos superpuestos.

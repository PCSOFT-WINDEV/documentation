
## grDestinationWMF (Función)




	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>14</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Se recomienda utilizar la función [grDestinoEMF](../WDLang3/3042017.md). 

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define un metafichero como destino del gráfico. Este metafichero se crea durante la llamada a [grDraw](../WDLang3/3042026.md).

**Observaciones**:

- Se recomienda utilizar la función [grDestinoEMF](../WDLang3/3042017.md).

- Esta función no se puede utilizar con los controles gráficos. 

- Esta función no se puede utilizar en la editor de reportes.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the "C:\Charts\Chart.WMF" file as destination of "MyChart"
grDestinationWMF("MyChart", "C:\Charts\Chart.WMF")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDestinationWMF(<Chart name> , <Metafile name>)`
---

**`<Chart name>: (Cadena de caracteres)`**

Nombre del gráfico a utilizar, definido por [grCreate](../WDLang3/3042005.md).

**`<Metafile name>: (Cadena de caracteres)`**

Nombre del metafichero donde se guardará el gráfico. Este metafichero se crea cuando se llama a [grDraw](../WDLang3/3042026.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000056"></a>

Para dibujar un gráfico:

- crear el gráfico ([grCreate](../WDLang3/3042005.md)),

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- defina el destino del gráfico (funciones [grDestinoBMP](../WDLang3/3042018.md), [grDestinoControl](../WDLang3/3042020.md), [grDestinoEMF](../WDLang3/3042017.md), [grDestinoWnd](../WDLang3/3042031.md) y **grDestinationWMF**),

- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).



<a name="NOTE0_2"></a>


### Fichero WMF de destino
<a name="fichero_wmf_destino_ELTPARAGRAPHE000097"></a>

- Durante la siguiente llamada a [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en el meta archivo llamado &lt;Nombre del meta archivo&gt;.

- Este archivo puede ser leído por una herramienta que reconoce el formato WMF.

- El tamaño de este meta archivo puede ser definido por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md).

- Le recomendamos que previsualice el gráfico antes de guardarlo en un archivo.



<a name="NOTE0_3"></a>


### Diferencias entre grDestinationWMF y grSaveWMF
<a name="diferencias_entre_grdestinationwmf_grsavewmf_ELTPARAGRAPHE000113"></a>

- La función **grDestinationWMF** permite definir el destino del próximo dibujo del gráfico. Cuando se utiliza [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en el archivo especificado y se tendrán en cuenta todos los cambios realizados desde la última vez que se dibujó el gráfico.

- [grSaveWMF](../WDLang3/3042044.md) se utiliza para guardar el gráfico en un archivo en formato WMF. La Image guardada corresponde al gráfico durante la última llamada a [grDraw](../WDLang3/3042026.md). Los cambios realizados en el gráfico desde la última vez que se llamó a [grDraw](../WDLang3/3042026.md) serán ignorados.




## grScatterSeriesLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la etiqueta de las series de datos en un gráfico "Scatter".

Ejemplo de etiqueta de serie: <br>![Etiqueta de la serie](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)


**Recordatorio**: Por Default, un gráfico no tiene etiqueta.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initializes the "Sales" label of second data series in "MyChart"
grScatterSeriesLabel("MyChart", 2, "Sales")
```
<a name="Example2"></a>

```wl
// Initializes the "Purchase" label of third data series in CHART_MyChart
grScatterSeriesLabel(CHART_MyChart, 3, "Purchase")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grScatterSeriesLabel(<Chart name> , <Series number> , <Label>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie para la que se inicializará una etiqueta.

**`<Label>: (Cadena de caracteres)`**

Leyenda de la etiqueta a inicializar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El tipo de letra de la etiqueta puede ser modificado por [grLabelFont](../WDLang3/3042012.md).

- Los datos definidos con la función **grScatterSeriesLabel** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- La función **grScatterSeriesLabel** es equivalente a la función [grSeriesLabel](../WDLang3/3042024.md). 
	La sintaxis de [grSeriesLabel](../WDLang3/3042024.md) utilizada con los gráficos de "dispersión" es la siguiente:
	
	```txt
	grSeriesLabel(<Chart name>, 2*<Index>-1, <Label>)
	```
**Reminder**: Un gráfico de "dispersión" utiliza los datos en grupos de dos series (para más detalles, véase [grScatterAddDataXY](../WDLang3/3042032.md)).


## NetMACToIP( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la IP Address correspondiente a un determinado MAC Address.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the IP address
Info(NetMACToIP("11:22:33:44:55:66"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetMACToIP(<MAC address> [, <Timeout>])`
---

**`<Result>: (Cadena de caracteres)`**



- IP Address (en formato xxx.xxx.xxx.xxx),

- Vacíe la cadena (""") si se produce un problema.




**`<MAC address>: (Cadena de caracteres)`**

MAC Address en formato "aa:bb:cc:dd:ee:ff" o en formato "aa-bb-cc-dd-ee-ff".

**`<Timeout>: (Entero opcional)`**

Duración máxima (en milisegundos) durante la cual el MAC Address puede ser convertido en un IP Address. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 


Si no se especifica este parámetro, el valor es set a 10000 (10 segundos).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### MAC Address y IP Address
<a name="mac_address_address_ELTPARAGRAPHE000045"></a>

El MAC Address es el Address físico especificado en las tarjetas de red de las computadoras MAC. La dirección IP es la dirección utilizada para comunicarse entre varias computadoras a través del protocolo IP (Protocolo INTERNET).

**Observación**: Esta función sólo puede ser utilizada por el equipo conectada a través de una red local.

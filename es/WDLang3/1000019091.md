
## saasPricing (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasPricing** permite definir las características avanzadas de una tarificación SaaS. 

Las características de los precios pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySite is saasSite dynamic 
MySite = SaaSAdminFindSite("Managing cars")

P is saasPricing
P.Name = "Premium Offer"
P.Price = 150
SaaSAdminAddPricing(P, MySite)

S is saasService
S.Name = "Full Access"

// Adds the service into the site
SaaSAdminAddService(P, MySite)

// The pricing will propose this service
SaaSAdminPricingEnableService(P, S)
```

## Observaciones


### Propiedades específicas de las variables saasPricing
<a name="propiedades_especificas_las_variables_saaspricing_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades se pueden utilizar para las variables manipular **saasPricing**:

| **denominar** | **Tipos** | **Efecto** |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre del precio.<br>Este nombre debe ser único para cada sitio. |
| emplazamiento | Variable de tipo [saasSite](../WDLang3/1000019068.md) | reference hacia el sitio que ofrece este precio.<br>Esta propiedad está disponible en solo lectura. |
| Facturación | Constante de tipo Integer | Especifica si el precio es por usuario o como un paquete.<br><br>Las siguientes constantes están disponibles:<br><br>- *saasOutrightPricing*: El precio se factura como un paquete<br><br>- *saasPricingPerUser*: El precio se factura por usuario<br><br><br> |
| Periodicidad | Entero o constante de tipo Integer | Duración de un período de facturación.<br>Esta duración se expresa en número de días.<br>La constante *saasUnlimitedPeriodicity* permite indicar una tarificación en un período limitado. |
| Precio | Moneda | Costo de la fijación de precios.<br>Si la determinación de precio se factura por período, este precio es por período. |

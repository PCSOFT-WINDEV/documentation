
## TwoFactorAuthenticationGenerateTOTPKey( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera una clave TOTP para un usuario de la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sTOTPKey is ANSI string 
sTOTPKey = TwoFactorAuthenticationGenerateTOTPKey(User.UserID + User.Email)

// Saves the TOTP key in the User data file
User.sTOTPKey = sTOTPKey
HSave(User)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TwoFactorAuthenticationGenerateTOTPKey(<Identifier>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de 16 caracteres ANSI correspondiente a la clave TOTP generada. 

**`<Identifier>: (Cadena de caracteres)`**

Identificador de usuario. **Este identificador debe permitir identificar de forma única a un usuario** (por ejemplo, concatenación de Email, nombre de usuario y clave única de archivo).


## tapiError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la última función tapixxx ([funciones de gestión de las llamadas de Phone](../WDLang3/3076005.md)) ha devuelto un error al módulo TAPI. Esta función debería ser associated con las funciones que no devuelven ningún resultado (ni True ni False).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF tapiCalledID() = "" THEN
	IF tapiError() = True THEN
		Error(ErrorInfo(errMessage))
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiError()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>False</u></u></u></u> si no se produjo ningún error al utilizar una función de telefonía,

- <u><u><u><u>True</u></u></u></u> si se produce un error. El texto o el código del error TAPI es devuelto por [ErrorInfo](../WDLang1/3013008.md).


## BTInfoRadio( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información específica sobre una radio Bluetooth conectada a el equipo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sListRadios is string
sARadio is string
sId is string
sName is string
sMACAddress is string
// Clears the list
TableDeleteAll(TABLE_Radios)
// Returns the list of radios
sListRadios = BTListRadio(btSimpleList)
// Browse the radios
FOR EACH STRING sARadio OF sListRadios SEPARATED BY CR
	// Stores the device information
	sID = ExtractString(sARadio, 1)
	sName = ExtractString(sARadio, 3)
	sMACAddress = ExtractString(sARadio, 2)
	// The name is explicitly requested if it is not specified
	IF sName ~= "" THEN sName = BTInfoRadio(sId, btName) 
	// Fills the Table control with the radios
	TableAddLine(TABLE_Radios, False, sID, sName, sMACAddress)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTInfoRadio(<Radio identifier> , <Requested information>)`
---

**`<Result>: (Cadena de caracteres)`**

Información solicitada sobre la radio especificada.

**`<Radio identifier>: (Integro)`**

Identificador de la radio Bluetooth a utilizar. Este identificador es devuelto por [BTListRadio](../WDLang3/1000017125.md).

**`<Requested information>: (Constante de tipo Integer)`**

Información sobre la radio. El tabla a continuación enumera la información disponible así como los valores que pueden ser devueltos:


|   |   |
| --- | --- |
| *btClassOfDevice* | La clase del dispositivo puede corresponder a una de las siguientes constantes:<br><br>- btCoD_RedAcceso<br><br>- btCoD_AudioVideo<br><br>- btCoD_Miscelánea<br><br>- btCoD_Imagen<br><br>- btCoD_Desconocido<br><br>- btCoD_Juguete<br><br>- btCoD_Computadora<br><br>- btCoD_Dispositivo<br><br>- btCoD_Phone<br><br>- btCoD_Llevable<br><br><br> |
| *btClassOfService* | La clase de servicio puede corresponder a una de las siguientes constantes (o a una combinación de constantes):<br><br>- btCoS_Captura: Dispositivo de adquisición de datos (Scanner, micrófono, ...).<br><br>- btCoS_Audio: Dispositivo de audio (micrófono, altavoz,...).<br><br>- btCoS_Información: Servicio de información (servidor Web o WAP, etc.)<br><br>- btCoS_Posicionamiento: Dispositivo de posicionamiento (GPS, ....).<br><br>- btCoS_Red: Dispositivo de red (punto de acceso, ....).<br><br>- btCoS_Renderización: Impresora, altavoz.<br><br>- btCoS_Telefonía: Dispositivo de telefonía (Phone, módem, ...).<br><br>- btCoS_Transferencia: Medio para tarjeta de memoria Transfer( , ...).<br><br><br> |
| *btIDClassOfService* | Valor numérico (número entero en 11 bits) correspondiente a la clase de servicio. |
| *btIDMajorClassOfDevice* | Valor numérico (número entero en 5 bits) correspondiente al número mayor de la clase de dispositivo. |
| *btIDManufacturer* | Identificador del fabricante. |
| *btIDMinorClassOfDevice* | Valor numérico (número entero en 6 bits) correspondiente al número menor de la clase de dispositivo. |
| *btIDVersion* | El número de versión set del fabricante. |
| *btMACAddress* | MAC Address del dispositivo |
| *btManufacturer* | Nombre del fabricante. |
| *btName* | Nombre de la radio. |

## Observaciones
Esta versión sólo es compatible con las pilas bluetooth de Microsoft, Bluesoleil y Broadcom (Widcomm). Para conocer la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).  

Las pilas Broadcom (Widcomm) no están disponibles en 64 bits.

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)


## WINDEV Mobile y mensajes de texto (SMS)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
WINDEV Mobile te permite enviar y leer fácilmente mensajes SMS con las funciones de WLanguage.

Un SMS (Servicio de Mensajes Cortos) se refiere comúnmente a los mensajes de texto (de hasta 160 caracteres) enviados a un teléfono móvil.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EnvoiSMS.gif)


**Observación**: Las características de los mensajes SMS están recogidas en el documento [Estructura del SMS](../WDLang3/3068003.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000170"></a>
Para poder utilizar las funciones SMS, la aplicación debe estar instalada:

- en una Android Phone.

- en un iOS Phone. 




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El mensaje SMS será preparado por la aplicación WINDEV Mobile, pero será enviado por el Phone.

## Notas
<a name="notas_ELTTEXTE000194"></a>
<a name="NOTE3_2"></a>


### Gestión de los diferentes tipos de números
<a name="gestion_los_diferentes_tipos_numeros_ELTPARAGRAPHE000043"></a>

Se pueden utilizar dos tipos de números para enviar mensajes SMS:

- Números cortos (también llamados números "nacionales"). Estos números sólo pueden ser accedidos desde la Country.

- Números estándar (también llamados números "internacionales"). Se puede acceder a estos números desde cualquier lugar.



<a name="NOTE3_4"></a>


### Diferencias entre el modo GO y el tiempo de ejecución
<a name="diferencias_entre_modo_tiempo_ejecucion_ELTPARAGRAPHE000053"></a>

En el modo **GO** (simulación en el ordenador de desarrollo), se produce un error WLanguage durante la llamada a una función SMS o cuando se utiliza un Variable de la estructura SMS.

### Llamadas Phone
<a name="llamadas_phone_ELTPARAGRAPHE000093"></a>WINDEV Mobile también le permite gestionar fácilmente las llamadas de Phone mediante las siguientes funciones: 



|   |   |
| --- | --- |
| [tapiDialerCall](../WDLang3/1000019235.md) | Abre la aplicación de telefonía por defecto (marcador) que se encuentra en la Phone y marca el número especificado. |
| [tapiDialerDisplay](../WDLang3/1000019236.md) | Abre la aplicación de telefonía Default (marcador) y muestra el número especificado. No se realiza ninguna llamada. |


## SaaSAdminCreateClientDatabase( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea la base de datos de una cuenta de cliente para un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
C is saasClient dynamic 

S = SaaSAdminFindSite("Forum")
C = SaaSAdminFindClient("Hybrid")

IF SaaSAdminCreateClientDatabase(S, C) = False THEN
	IF SaaSAdminCreateClientDatabase(S, C, False) = False THEN
		Error("Unable to create the client database.")
	ELSE
		Info("The database was created empty.")
	END
ELSE
	Info("The database was created from the reference database.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminCreateClientDatabase(<SaaS site> , <Client account> [, <Empty database>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> se se creó la base de datos cliente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS site>: (saasSite Variable o cadena de caracteres)`**

Nombre del [saasSite](../WDLang3/1000019068.md) Variable (o nombre del sitio) que corresponde al sitio SaaS para el que se debe crear la base de datos.

**`<Client account>: (saasClient Variable o cadena de caracteres)`**

Nombre del [saasClient](../WDLang3/1000019072.md) Variable o nombre de la cuenta de cliente SaaS para la que se debe crear la base de datos.

Si este parámetro es una cadena de caracteres, puede representar el nombre o el [Referencia externa](../WDLang3/1000019072.md) Property de la cuenta del cliente.

**`<Empty database>: (Booleano opcional)`**



- <u><u><u><u>False</u></u></u></u> (valor predeterminado), la base de datos de referencia se copiará para crear la nueva base de datos.

- <u><u><u><u>True</u></u></u></u>, la base de datos de referencia definida para el sitio será ignorada y se creará una base de datos vacía.

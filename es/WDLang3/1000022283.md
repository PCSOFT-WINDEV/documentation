


## &lt;emailPOP3Session Variable&gt;.CloseSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la sesión POP3. 

Una sesión de Email que no haya terminado se cerrará automáticamente al final del programa WINDEV o WINDEV Mobile o al final de la sesión de WEBDEV.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a POP3 session
MySession is emailPOP3Session

MySession.ServerAddress = "pop.mydomain.fr"
MySession.Name = "user"
MySession.Password = "secret"

MySession.StartSession()
...
MySession.CloseSession()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<POP3 session>.Sesión cerrada()`
---

**`<POP3 session>: (emailPOP3Session variable)`**

Nombre de la [emailPOP3Session](../WDLang3/1000018759.md) Variable correspondiente a la sesión Email a utilizar.

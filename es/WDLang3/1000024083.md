


## &lt;Gráfico&gt;.HMGetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera datos de un gráfico de Heatmap. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve data
let MyValue = CHART_HMap.HMGetData(1, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Chart control>.HMGetData(<X> , <Y>)`
---

**`<Result>: (Real)`**



- Valor recuperado. 

- 0 si no existe ningún valor en la posición especificada.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (se encuentra en el editor de ventanas, editor de páginas o editor de reportes).


**`<X>: (Integro)`**

Posición de los datos a recuperar en el eje X.

**`<Y>: (Integro)`**

Posición de los datos a recuperar en el eje Y.

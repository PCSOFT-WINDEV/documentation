


## grAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a un gráfico. 

**Atención**: Para los gráficos "avanzados" hay que utilizar una función específica de WLanguage (véase ["Observaciones"](#NOTE0_2)). 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a data into the "CHART_MyChart" control. 
// This data (whose value is 10) is added at the end of the first series.
grAddData(CHART_MyChart, 1, 10)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grAddData(<Chart name> , <Series number> [, <Index>] , <Value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md). Este nombre debe estar entre comillas. 

- el nombre de la control Gráfico (en la ventana, página o editor de reportes). Este nombre no debe estar entre comillas. 




**`<Series number>: (Integro)`**

Número de la serie a la que se añadirá un valor. 
Por ejemplo, si la control Gráfico muestra 2 Line s, esto se usa para identificar la Line para la cual los datos deben ser añadidos. 

Si el número especificado no existe, las series entre la última serie existente y el número especificado se crean automáticamente y se inicializan a 0.

**`<Index>: (Entero opcional)`**

Índice de los datos de la serie a los que hay que añadir un valor. 
Por ejemplo, para un gráfico de Line, este parámetro corresponde al eje X del valor a añadir. 

Si este parámetro:

- no se especifica, el valor se añade al final de la serie.

- se especifica, se sustituye el valor anterior.




**`<Value>: (Entero o real)`**

Valor añadido. 

Por ejemplo, para un gráfico de Line, este parámetro corresponde al eje Y del valor a añadir.

## Observaciones


### Añadir datos
<a name="anadir_datos_ELTPARAGRAPHE000152"></a>

- Los datos agregados con la función **grAddData** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- Para un gráfico circular (constante *grPie*), solo se utilizan los datos de la primera serie. No se puede dibujar un gráfico circular si los datos encontrados en la primera serie son nulos.

- Para los gráficos de "barras apiladas", si el valor de la primera serie es negativo, la barra correspondiente se orienta hacia abajo.. 



<a name="NOTE0_2"></a>


### Añadir datos a los diferentes tipos de gráficos
<a name="anadir_datos_los_diferentes_tipos_graficos_ELTPARAGRAPHE000168"></a>

- Para los gráficos de dispersión, se recomienda utilizar [grScatterAddDataXY](../WDLang3/3042032.md).

- Para los gráficos de acciones, se recomienda utilizar [grStockAddData](../WDLang3/3042009.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [grHMAddData](../WDLang3/1000021628.md).

- Para las cartas Sunburst, se recomienda utilizar [grSunburstAddData](../WDLang3/1000021285.md). 

- Para los gráficos de superficie, se recomienda utilizar [grSuperficieAñadirDatos](../WDLang3/1000020425.md).

- Para los gráficos de barras radiales, se recomienda utilizar [grRBAddData](../WDLang3/1410087792.md).

- Para añadir un valor a los datos de un gráfico, utilice [grIncreaseData](../WDLang3/3042030.md).


## NotifAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Notification en la barra de sistema del dispositivo (Android) o en el centro Notification (iOS).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
notif is Notification
notif.Title = "New incoming message"
notif.Message = "Click here to display the message"
NotifAdd(notif)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NotifAdd(<Notification>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó la notificación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**Observación**: En el simulador, esta función no tiene ningún efecto y siempre devuelve <u><u><u><u>False</u></u></u></u>. 

**`<Notification>: (Notification Variable)`**

Nombre de la variable [Notification](../WDLang3/1000019441.md) que contiene la descripción de la Notification a mostrar.

Si este parámetro corresponde a un Notification que ya está agregado, este Notification será actualizado. En este caso, la función **NotifAdd** y [NotifModify](../WDLang3/1000019443.md) tendrán el mismo efecto.

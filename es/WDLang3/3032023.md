


## EmailReset( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia todos los Variable s de la estructura [correo electrónico](../WDLang3/3032029.md) o un Variable de tipo [Email](../WDLang3/1000018713.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Reset all the variables of the email structure 
EmailReset()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EmailReset([<Email>])`
---

**`<Email>: (Correo electrónico opcional Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable que se va a restablecer. Si no se especifica este parámetro, se utilizará la estructura [Email](../WDLang3/3032029.md).

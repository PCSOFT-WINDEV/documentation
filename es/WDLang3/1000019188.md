


## grSeriesEje secundario (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define si se desea dibujar una serie en el eje secundario.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Places the series 2 of GRF_MyCHart on the secondary axis
grSeriesSecondaryAxis(GRF_MyChart, 2, True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSeriesSecondaryAxis(<Chart name> , <Series number> , <Position>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico.




**`<Series number>: (Integro)`**

Número de la serie a modificar.

**`<Position>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para ubicar la serie en el eje secundario, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función sólo se puede utilizar en las cartas con ejes. No funciona con las cartas Pie, Radar o Funnel.

- Esta función es equivalente a la [SecondaryAxis](../Proprietes/1000019187.md) Property.

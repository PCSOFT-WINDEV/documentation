
## PowerStatus (Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>26</B>, esta función está disponible por compatibilidad con versiones anteriores. Utilice la función [SysPowerStatus](../WDLang1/1000026039.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Utiliza para get diversas informaciones sobre la batería (principal o secundaria) del dispositivo en el que se ejecuta la aplicación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Status of main battery?
Info("The battery is loaded at: " + PowerStatus(ChargePercentage) + " %")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PowerStatus(<Type of information> [, <Battery>])`
---

**`<Result>: (Entero o booleano según el tipo de información)`**



- Información solicitada,

- Vacíe la cadena (""") si la información solicitada no está disponible.




**`<Type of information>: (Constante de tipo Integer)`**

Especifica el tipo de información solicitada: 



|   |   |
| --- | --- |
| *ACLineStatus* | <br><br>- <u><u><u><u>True</u></u></u></u> si el dispositivo está conectado a la corriente alterna,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| *ChargeLevel* | Nivel de carga de la batería:<br><br>- 0: batería no encontrada.<br><br>- 1: alto nivel.<br><br>- 2: bajo nivel.<br><br>- 3: nivel crítico.<br><br><br> |
| *ChargePercentage* | Porcentaje de carga de la batería. |
| *Charging* | <br><br>- <u><u><u><u>True</u></u></u></u> si se está cargando la batería,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| *RemainingLife* | Vida útil restante (en segundos) de la batería.<br><br> |
| *TotalLife* | Vida útil total (en segundos) de la batería.<br><br> |



**`<Battery>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) para obtener información sobre la batería principal del dispositivo,

- <u><u><u><u>True</u></u></u></u> para obtener información sobre la batería secundaria del dispositivo.

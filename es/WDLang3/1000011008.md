
## AppointmentDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la cita actual desde la Calendar de una mensajería de Lotus Notes o Outlook.
	**Recordatorio**: Esta función no puede utilizarse para eliminar la cita actual de la Calendar de una mensajería de Outlook Express.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
	"email/julia.nsf", "C:\Lotus Notes\julia.id")
// Read the last appointment
AppointmentLast(SessionID)
// Delete the appointment
AppointmentDelete(SessionID)
```

<a name="SYNTAX1"></a>

## Sintaxis

### Borrar la cita actual de la Calendar de una mensajería de Lotus Notes o Outlook

`<Result> = AppointmentDelete(<Session identifier> [, <Appointment identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  

**`<Appointment identifier>: (Cadena de caracteres opcional)`**

Identificador de la Appointment que se va a eliminar. Este identificador se encuentra en el mAppointment.ID Variable.

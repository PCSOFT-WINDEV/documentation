
## SaaSReadSiteParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee una información específica del cliente para el sitio SaaS actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use a parameter to customize the logo of the site
IMG_LogoImage = SaaSReadSiteParameter("logo")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSReadSiteParameter(<Parameter to Read> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Valor del parámetro. 

Si el valor no fue personalizado por el cliente a través de [SaaSWriteSiteParameter](../WDLang3/1000019039.md), el valor devuelto corresponde al valor de Default dado por el administrador del sistema SaaS.

**Atención**: La función desencadena un error no fatal si el parámetro de sitio no existe.

**`<Parameter to Read>: (Cadena de caracteres)`**

Nombre del parámetro buscado.

**`<Options>: (Constante de tipo Integer)`**

Opción a tener en cuenta: 



|   |   |
| --- | --- |
| *saasNoCache* | Fuerza el valor a leer en el servidor SaaS. El valor encontrado en la caché no se lee ni se actualiza. |
| *saasReinitCache* | Reinicia toda la caché, luego el valor se lee en el servidor SaaS y la caché se actualiza automáticamente. |
| *saasWithCache*<br>(Valor predeterminado) | El valor se lee desde la caché. Si este valor no existe en la caché, se lee en el servidor SaaS y la caché se actualiza automáticamente. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los parámetros de un sitio son definidos por el administrador del sistema SaaS al configurar el sitio en el sistema [Administrador SaaS](../SaaSAdminWeb/1600000.md). Pueden tener valores de Default.

- Cualquier usuario del sitio puede leer los parámetros. Sólo los usuarios "Administradores" pueden modificarlos.

- Los parámetros del sitio están destinados a permitir la personalización de la UI o el comportamiento de los sitios SaaS. Para más detalles sobre su uso, véase [Desarrollo de un sitio SaaS](../WDLang3/9500214.md).

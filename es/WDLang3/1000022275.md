


## &lt;emailIMAPSession Variable&gt;.CambiarCarpeta (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la carpeta actual en el software de mensajería utilizando el IMAP..   

Esta carpeta se encuentra en la rama "Carpetas personales. 

Atención: Las "Carpetas genéricas" no son compatibles. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// IMAP protocol
EDT_USER is emailIMAPSession
...
IF EmailStartIMAPSession(EDT_USER, EDT_PASSWORD, ...
		"imap.mydomain.fr") = True THEN
	EDT_USER.ChangeFolder("Personal folders\Inbox\Work") 
	...
ELSE
	Error("Unable to establish the connection")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.ChangeFolder(<Folder path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, la ruta especificada no existe). Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailIMAPSession)`**

Nombre de la Variable de tipo [emailIMAPSession](../WDLang3/1000018957.md) que corresponde a la sesión actual de Email.

**`<Folder path>: (Cadena Unicode)`**

Ruta de la nueva carpeta actual.

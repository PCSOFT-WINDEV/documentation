
## MapImage (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Las variables **MapImage** se utilizan para definir las características de un Image que se dibujará en el mapa que se muestra en un control Mapa. Las características de la imagen se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an image to the Map control
i is MapImage
i.Position = MyPosition
i.Image = "image.png"
i.ActionClick = ClickProc
MapAddImage(MAP_MyMap, i)

INTERNAL PROCEDURE ClickProc(i is MapImage)
	ToastDisplay(StringBuild("Click on the <%1> image.", i.Name))
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables MapImage
<a name="propiedades_especificas_las_variables_mapimage_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una imagen en una control Mapa:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionClick | Nombre de Procedure | Nombre del WLanguage Procedure llamado cuando el usuario hace clic en el Image mostrado en el control Mapa. Este procedimiento tiene el siguiente formato: <br><br><pre><code>PROCEDURE <Procedure name>(<Image>)</code></pre><br>donde &lt;Imagen&gt; es una **MapImage** Variable correspondiente a la Image en la que el usuario ha pulsado. <br><br>Si no se especifica esta Property (o corresponde a una cadena vacía ("")), no se realizará ninguna acción al hacer clic.<br><br>Observación: se recomienda utilizar procedimientos globales. Si la Procedure es local, su nombre debe ir precedido del nombre completo del objeto al que pertenece la Procedure (ejemplo para una ventana local Procedure: "WIN_Window1.MyProcedure").<br><br> |
| Alineación | Constante de tipo Integer | Alineación de la imagen con respecto a la posición dada por el **Position** Property : <br><br>- *mkTop*: imagen Above la posición. <br><br>- *mkBottom*: Image debajo de la posición. <br><br>- *mkCenter* (valor Default): Image se centró en la posición. <br><br>- *mkRight*: Image a la derecha de la posición. <br><br>- *mkLeft*: Image a la izquierda de la posición. <br><br><br> |
| Altura | Real | La altura (en metros) que la Image debería ocupar en el mapa. <br><br>Si esta Property no está especificada en o está puesta en 0, la altura se calculará automáticamente según la anchura especificada a través de la **Width** Property, respetando la relación Image. |
| anchura | Real | El ancho (en metros) que la Image debe ocupar en el mapa. <br><br>**Debe especificarse este Property.** |
| Ángulo | Integro | Ángulo de rotación en grados (entre 0 y 360) para ser aplicado a la Image. La rotación se realiza usando la posición dada por la **Position** Property como centro de rotación.<br><br>Este Property es set a 0 por Default. |
| denominar | Cadena de caracteres | Nombre de la imagen. Se asigna un nombre automáticamente cuando se crea la Variable. Puede ser modificado, pero tiene que ser único si se añade la forma a la control Mapa. Este nombre se puede utilizar para eliminar la Image con [MapDeleteImage](../WDLang3/1000025534.md). |
| description | Cadena de caracteres | description de la Image, utilizada para almacenar información adicional, por ejemplo. |
| Image | Cadena de caracteres <br>o [Image](../WDLang1/1000019650.md) Variable | Image para mostrar. Este Property corresponde a: <br><br>- una cadena de caracteres correspondiente a la ruta de la Image,<br><br>- una variable de tipo [Image](../WDLang1/1000019650.md).<br><br><br>Esta Image no puede ser un vector Image (SVG). |
| Opacidad | Integro | Porcentaje de opacidad de Image. Por defecto, esta Property es set a 100. |
| ubicación | [geoPosition](../WDLang3/1000019191.md) | La posición geográfica en la que se dibujará la Image. La alineación de la imagen con respecto a esta posición viene dada por el **Alignment** Property. |
| ZOrder | Integro | Orden Z de la Image en relación con las otras formas de la control Mapa. Le permite manejar las superposiciones. <br><br>Una forma con un orden Z alto se dibujará sobre una forma con un orden Z más bajo. <br><br>Este Property es set a 0 por Default. <br><br> |

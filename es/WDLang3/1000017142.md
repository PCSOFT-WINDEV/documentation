
## BTEnableService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa un servicio proporcionado por un dispositivo Bluetooth.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nDeviceNumber is int
sService is string
// The devices are stored in a Table control
nDeviceNumber = TABLE_Device.COL_ID[TABLE_Device]
// The services are listed in a List Box control
sService = LIST_Services[LIST_Services]
// Enable the selected service in the List Box control
IF BTEnableService(nDeviceNumber, sService) = True THEN
	// Fill the List Box control with the enabled services
	ListAdd(LIST_EnabledServices, sService)
ELSE
	 Trace("Device: " + nDeviceNumber + ", service: " + sService)
	 Info(ErrorInfo(errFullDetails))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTEnableService(<Device identifier> , <Service name> [, <Radio identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se activó el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Device identifier>: (Integro)`**

Identificador del dispositivo Bluetooth a utilizar. Este identificador es devuelto por [BTListDevice](../WDLang3/1000017132.md).

**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a habilitar. El nombre de los servicios es devuelto por [Servicio BTList](../WDLang3/1000017141.md).

**`<Radio identifier>: (Entero opcional)`**

Identificador de la radio Bluetooth a utilizar. Este identificador es devuelto por [BTListRadio](../WDLang3/1000017125.md). Si no se especifica este parámetro, se utilizarán todas las radios disponibles.

<a name="NOTE0_1"></a>

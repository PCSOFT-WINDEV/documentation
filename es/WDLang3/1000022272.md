


## grPointType (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la forma del punto para una categoría o valor específico que se encuentra en una carta de Line, área, dispersión y radar.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Transforms the point (category) 2 of series 1 into circle
grPointType(CHART_MyChart, 1, 2, grPointCircle)

// Transforms the point (category) 3 of all series into square
grPointType(CHART_MyChart, 3, grPointSquare)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Modificación de la forma del punto para una categoría (para todas las series)

`grPointType(<Chart name> , <Category number> , <Type>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Category number>: (Integro)`**

Índice de categoría a modificar. 

**`<Type>: (Constante de tipo Integer)`**

Nuevo tipo de punto para la categoría: 



|   |   |
| --- | --- |
| *grNotDrawn* | Punto no dibujado. |
| *grPointCircle* | Punto en forma de círculo. |
| *grPointCross* | Punta en forma de cruz. |
| *grPointDiamond* | Punta en forma de diamante. |
| *grPointSquare* | Punto en forma de cuadrado. |
| *grPointTriangle* | Punta en forma de triángulo. |




<a name="SYNTAX2"></a>

### Modificar la forma de un punto específico

`grPointType(<Chart name> , <Series number> , <Category number> , <Type>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Series number>: (Integro)`**

Número de la serie en la que se modificará el tipo de punto.

**`<Category number>: (Integro)`**

Índice de categoría a modificar. 

**`<Type>: (Constante de tipo Integer)`**

Nuevo tipo de punto para la categoría: 



|   |   |
| --- | --- |
| *grNotDrawn* | Punto no dibujado. |
| *grPointCircle* | Punto en forma de círculo. |
| *grPointCross* | Punta en forma de cruz. |
| *grPointDiamond* | Punta en forma de diamante. |
| *grPointSquare* | Punto en forma de cuadrado. |
| *grPointTriangle* | Punta en forma de triángulo. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>
Para cambiar el tipo de todos los puntos que se encuentran en una serie, le recomendamos que utilice [grSeriesPointType](../WDLang3/1000022271.md).

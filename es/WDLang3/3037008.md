
## FaxOutbox( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los faxes pendientes o los faxes salientes.

**Observación**: Esta función sólo se puede utilizar en Windows 2000 o XP.. En Windows Vista (y posteriores), la gestión de faxes está disponible con Windows Professional y Windows Integral Edition.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectID is int
ConnectID = FaxConnect()
...
ResSendFax is boolean
ResSendFax = FaxSend(ConnectID, "MyFax", "Message.txt", "0102030405")
...
ListSent is string 
ListSent = FaxOutbox(ConnectID)
...
FaxDisconnect(ConnectID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxOutbox(<Connection identifier>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de identificadores de fax separados por caracteres CR (Carriage Return). Este identificador le permite manipular el fax mediante las funciones de gestión de fax.. Los identificadores de fax corresponden a todos los faxes en espera de ser enviados o actualmente enviados (no sólo los faxes enviados por [FaxSend](../WDLang3/3037006.md)).

**`<Connection identifier>: (Integro)`**

Identificador de la conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md).

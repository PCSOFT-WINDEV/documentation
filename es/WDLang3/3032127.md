
## LDAP constante
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para gestionar los servidores LDAP: 

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **auNegotiate** | Modo de gestión de LDAP con conexión secure SSL. <br><br>Si se usa la constante *auNegotiate*:<br><br>- Si no se especifica el inicio de sesión, se utiliza el inicio de sesión de conexión de Windows.<br><br>- Si se especifica el nombre de usuario y la contraseña, la conexión se establece a través de "Generic Security Services" (GSS) (Snego).. El método de autenticación utilizado es el más apropiado que se encuentra en una lista de posibles autenticaciones.<br><br><br> | [Estructura de LDAPSession](../WDLang3/3032101.md), <br>[gpwOpenLDAPSession](../WDLang6/1000017055.md) |
| **LDAP_CacheDesactivado** | Cada llamada a [LDAPNbValor](../WDLang3/3032106.md) y [Valor LDAPValor](../WDLang3/3032107.md) requiere una conexión con el servidor | [LDAPMode](../WDLang3/3032126.md) |
| **LDAP_CacheHabilitado** | Las llamadas a las funciones LDAP utilizan la caché interna para recuperar los valores de los atributos | [LDAPMode](../WDLang3/3032126.md) |
| **ldapDefaultOption** | Modo de gestión de SSL: LDAP estándar, sin gestión SSL. | [Estructura de LDAPSession](../WDLang3/3032101.md), <br>[gpwOpenLDAPSession](../WDLang6/1000017055.md) |

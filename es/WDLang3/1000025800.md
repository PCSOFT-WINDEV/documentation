


## &lt;Mapa&gt;.Área de visualización (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve el mapa que se muestra en el control Mapa para que el área geográfica especificada se pueda ver en su totalidad.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
p1 is geoPosition
p1.Latitude = 0
p1.Longitude = 0
p2 is geoPosition
p2.Latitude = 3
p2.Longitude = 3
MAP_MyMap.DisplayArea(p1, p2, True, procEndAnimation)

INTERNAL PROCEDURE procEndAnimation(bResult is boolean)
	IF bResult = True THEN
		ToastDisplay("Area displayed")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.DisplayArea(<North West position> , <South East position> [, <Animation> [, <WLanguage procedure>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se solicitaba un cambio de posición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<North West position>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable que corresponde al punto noroeste de la zona a visualizar en el control Mapa.

**`<South East position>: (geoPosition Variable)`**

Nombre de la [geoPosition](../WDLang3/1000019191.md) Variable que corresponde al punto sureste de la zona a mostrar en el control Mapa.

**`<Animation>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor de Default) si la zona se va a mostrar con una animación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

El nombre del WLanguage Procedure ("callback") llamado al final de la animación. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name> (<Displayed area>)
```
donde &lt;Área mostrada&gt; es un booleano: 

- <u><u><u><u>True</u></u></u></u> si la zona pudiera ser mostrada, 

- <u><u><u><u>False</u></u></u></u> de otra manera (por ejemplo, si el usuario movió el mapa durante la animación). 


Si &lt;Animation&gt; se pone en <u><u><u><u>False</u></u></u></u>, el Procedure no se llama.

<a name="NOTE0_1"></a>

## Observaciones


### "Siempre que cambie de posición" control Mapa Event
<a name="siempre_que_cambie_posicion_control_mapa_event_ELTPARAGRAPHE000236"></a>

- Si la control Mapa se mueve al área especificada con una animación, la Event "Siempre que cambie de posición" se ejecutará al final de la animación.. 

- Si el usuario cancela el posicionamiento en la control Mapa antes del final de la animación, la Event "Siempre que cambie la posición" no se ejecutará.





### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000247"></a>

- El nivel de zoom de la control Mapa puede ser set para cambiar automáticamente cuando la posición cambia.

- **Para centrar un mapa en una posición determinada sin animaciones**, puede asignar directamente una variable [geoPosition](../WDLang3/1000019191.md) al [Valor](../Proprietes/2510130.md) Property del control Mapa. 
	Por ejemplo: 
	```wl
	p1 is geoPosition
	p1.Latitude = 3
	p1.Longitude = 3
	MAP_MyMap.Value = p1
	```




## &lt;Parámetro de carga (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura los parámetros utilizados para dibujar un gráfico. Estos parámetros han sido guardados por [&lt;GuardarParámetro](../WDLang3/1000024127.md).

**Recordatorio**: Por Default, los parámetros del gráfico modificados por un usuario se guardan al cerrar la aplicación. Estos parámetros se restauran automáticamente durante el siguiente inicio de la aplicación.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the parameters of a Chart control
INIWrite("Parameters", "CHART_Chart", "Param", CHART_Chart.SaveParameter())

// Restore the parameters of a Chart control
CHART_Chart.LoadParameter(INIRead("Parameters", "CHART_Chart", "Param"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.LoadParameter(<Parámetros>)`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se cargaron los parámetros,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Control name)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Parámetros>: (Ansi character string (with quotes))`**

Parámetro a restaurar. Este parámetro está comprimido. Sólo puede ser devuelto por [&lt;GuardarParámetro](../WDLang3/1000024127.md). El valor de este parámetro se puede guardar en un archivo, en un memo o en el registro.

Si este parámetro corresponde a una cadena vacía (""), la función no tiene ningún efecto y devuelve <u><u><u><u>False</u></u></u></u>.

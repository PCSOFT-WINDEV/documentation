


## &lt;httpRequest Variable&gt;.ResetCookie (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las cookies (globalmente o para un dominio) almacenadas por las llamadas a [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md).




<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

### Borra las cookies almacenadas por HTTPSend

`<HTTP request>.ResetCookie([<Domain>])`
---

**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre del [httpRequest](../WDLang3/1000021158.md) Variable para el que se borrarán las cookies.

**`<Domain>: (Cadena de caracteres opcional)`**

Dominio para el que se borrarán las cookies. Si no se especifica este parámetro (si no se especifica ningún dominio), se borrarán todas las cookies.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un dominio corresponde al final de Address del servidor HTTP (".windev.com" por ejemplo).

- El uso de **&lt;httpRequest Variable&gt;.ResetCookie** influirá en la siguiente llamada a [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md).

- La función **&lt;httpRequest Variable&gt;.ResetCookie** equivale a la función [&lt;httpRequest Variable&gt;.ReemplazarCookie](../WDLang3/1000021623.md) con un array vacío.

- **Atención**: **&lt;httpRequest Variable&gt;.ResetCookie** no borra las cookies de INTERNET Explorer.

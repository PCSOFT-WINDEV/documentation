## Presentación
<a name="presentacion_ELTTEXTE000107"></a>
Para utilizar las notificaciones Push en Android, debes tener una cuenta de Google y debes realizar una configuración específica. 

Esta página de ayuda presenta las diferentes operaciones a realizar. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Etapas a seguir
<a name="etapas_seguir_ELTTEXTE000131"></a>
Los diferentes pasos son los siguientes: 

**1. Creación de un proyecto Google API Project**

- Desde el navegador, abra la consola de [Google API](https://code.google.com/apis/console). 

- Autentifíquese utilizando una cuenta de Google. 

- Crear un nuevo "Proyecto API" o utilizar un proyecto existente. 




**2. Recuperación del número de proyecto Google API**

- Recuperar el número de proyecto de Google API: 

	1. Mostrar la información sobre su proyecto actual. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Push_Google_2.gif)


	2. En la ventana que se muestra, copy el número: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Push_Google_2B.gif)

- Este número debe especificarse en el asistente para generar la aplicación Android. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Push_Google_3.gif)





**3. Generar una "Tecla API"**

- En un navegador, vaya a [https://developers.google.com/mobile/add?platform=android&cntapi=gcm](https://developers.google.com/mobile/add?platform=android&cntapi=gcm)

- Seleccione el proyecto que se creó durante el paso anterior. 

- Especifique el nombre del paquete de la aplicación (definido en Android generación asistente ).

- Haga clic en "Elegir y configurar servicios".

- Haga clic en "Enable Google Cloud Messaging".<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=push_Google_8.gif)


- Recuperar el número de clave API (clave API del servidor). Este número de clave debe pasarse como parámetro a [NotifPushSend](../WDLang3/1000020819.md). A continuación, esta clave aparece en la lista de claves de proyecto en la consola del desarrollador.

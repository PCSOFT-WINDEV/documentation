
## tapiDialerDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la aplicación de telefonía Default (marcador) y muestra el número especificado. No se realiza ninguna llamada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
tapiDialerDisplay("0612345678")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = tapiDialerDisplay(<Phone number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió la aplicación (dialer),

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Phone number>: (Cadena de caracteres)`**

Número de Phone para mostrar en la pantalla de marcado. 

Si este parámetro es una cadena vacía (""), la aplicación de telefonía se abrirá en la pantalla de marcado.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Llamada de automatic
<a name="llamada_automatic_ELTPARAGRAPHE000054"></a>

Para activar automáticamente la llamada al número especificado, utilice [tapiDialerCall](../WDLang3/1000019235.md).




## &lt;emailOutlookSession Variable&gt;.CurrentFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de la carpeta actual en el software de mensajería de Outlook.. Esta carpeta se encuentra en la rama "Carpetas personales".

**Recordatorio**: Esta función no se puede utilizar para encontrar el nombre de la carpeta actual en el software de mensajería Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo



<a name="3032129_Example2"></a>

```wl
// Start the Outlook session
SessionID is EmailOutlookSession
...
// Create a folder in the current folder 
SessionID.AddFolder(SessionID.CurrentFolder() + "\Work")
// Create a folder in the custom folder
SessionID.AddFolder("Personal folders\Inbox\MyOwnWork")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.CurrentFolder()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la carpeta actual,

- Cadena vacía ("") si se produce un error. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailOutlookSession)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la Variable de tipo [emailOutlookSession](../WDLang3/1000018767.md) que corresponde a la sesión actual de Email.


## tapiSendKey( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite simular el uso de las teclas de Phone.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
tapiSendKey("123456789ABCD*#", LIST_CallList[LIST_CallList])
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiSendKey(<Set of Keys> [, <Call identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la marcación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Set of Keys>: (Cadena de caracteres)`**

set de claves para simular.

**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). La llamada actual se gestiona si no se especifica este parámetro.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Limitación
<a name="limitacion_ELTPARAGRAPHE000046"></a>

No se puede llamar a la función **tapiSendKey** en el procedimiento de detección de llamadas (definido con la función [tapiEscuchar](../WDLang3/3076029.md)).

### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000187"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.


## Marcador (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Marker** permite definir las características de un marcador ubicado en un mapa que se muestra en un control Mapa. 

Un marcador se define por un nombre y una posición geográfica (latitud y longitud) y puede ser llenado con una description y una acción en el click. 

Las características de esta Marker pueden definirse y modificarse mediante diferentes propiedades del lenguaje WL.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a marker and runs a procedure during a click on this marker 
MyLocation is geoPosition
MyLocation.Latitude = 43.613708 
MyLocation.Longitude = 3.876972
MyMarker is Marker
MyMarker.Position = MyLocation
MyMarker.ActionClick = ProcMarkerClick

MapAddMarker("MapControl", MyMarker)
```

```wl
//------------------------------------------

PROCEDURE ProcMarkerClick(m is marker)
ToastDisplay(StringBuild("Position: %1 N, %2 E ", ...
	m.Position.Latitude, m.Position.Longitude))
```

## Observaciones


### Propiedades específicas de las variables Marker
<a name="propiedades_especificas_las_variables_marker_ELTPARAGRAPHE000065"></a>

Las siguiente propiedades se pueden utilizar para manipular una variable de tipo **Marker**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionClick | Nombre de Procedure | Nombre del WLanguage Procedure llamado cuando el usuario hace clic en el Marker mostrado en el control Mapa. <br><br>Este procedimiento tiene el siguiente formato: <br><br><br><pre><code>PROCEDURE <Procedure name>(<Marker>)</code></pre><br><br><br>donde &lt;Marcador&gt; es una variable de tipo **Marker** que corresponde al marcador en el que el usuario hizo clic. <br><br>Si esta Property corresponde a una cadena vacía (""), no se realizará ninguna acción al hacer clic.<br><br>Observación: se recomienda utilizar procedimientos globales. Si la Procedure es local, su nombre debe ir precedido del nombre completo del objeto al que pertenece la Procedure (ejemplo para una ventana local Procedure: "WIN_Window1.MyProcedure").<br><br> |
| ActionClickPopup | Cadena de caracteres | El nombre del WLanguage Procedure llamado cuando el usuario hace clic en el popup de Marker. <br><br>Este procedimiento tiene el siguiente formato: <br><br><br><pre><code>PROCEDURE <Procedure name>(<Marker>)</code></pre><br><br><br>donde &lt;Marcador&gt; es una variable de tipo **Marker** que corresponde al marcador asociado a la popup. <br><br>Si esta Property corresponde a una cadena vacía (""), no se realizará ninguna acción al hacer clic.<br><br><br><br> |
| ActionDrag | Cadena de caracteres | Nombre del lenguaje WLanguage Procedure llamado cuando el usuario mueve el marcador mostrado en el control Mapa (si la propiedad **Movable** está establecida en <u><u><u><u>True</u></u></u></u>). <br><br>Este procedimiento tiene el siguiente formato: <br><br><br><pre><code>PROCEDURE <Procedure name>(<Marker>)</code></pre><br><br><br>donde &lt;Marcador&gt; es una variable de tipo **Marker** que corresponde al marcador movido. <br><br>Esta Procedure se llama cuando el usuario libera la Marker. <br><br>Si esta Property corresponde a una cadena vacía (""), no se realizará ninguna acción.<br><br><br><br> |
| Alineación | Constante de tipo Integer | Modo para alinear la Marker cuando se muestra en un mapa: <br><br>- *mkBottom*: Marker mostrada en la parte inferior de la posición. <br><br>- *mkCenter*: Marker se muestra en la posición. <br><br>- *mkRight*: Marker se muestra a la derecha de la posición. <br><br>- *mkLeft*: Marker se muestra a la izquierda de la posición. <br><br>- *mkTop*: Marker mostrada en la parte superior de la posición. <br><br><br> |
| Clúster | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la Marker se va a agrupar en un grupo si es necesario, <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>Observación: Una nueva Event opcional está disponible: "Agrupación de marcadores". <br><br>- Esta Event se ejecuta cada vez que se crea o modifica un cluster de Marker en la control Mapa. <br><br>- Toma como parámetro una array de variables **Marker**. Esta array corresponde a las Marker s del cluster. <br><br>- Este Event puede devolver una imagen o un [MarkerImage](../WDLang3/1000025796.md) Variable que se utilizará para representar el cluster en el mapa. Si esta Event no devuelve un valor (o devuelve una cadena vacía), el cúmulo se mostrará en modo Default.<br><br><br><br><br> |
| denominar | Cadena de caracteres | Nombre de la Marker. <br><br>**Observaciones**: <br><br>- Esta propiedad es opcional: si no se especifica un nombre, se asocia un nombre al azar con la Marker. <br><br>- Si se muestra un popup, mostrará este nombre y el marcador description.<br><br>- Al añadir una Marker con el mismo nombre que una Marker existente, [MapAddMarker](../WDLang3/1000019794.md) falla: el Marker no se añade. <br><br><br> |
| description | Cadena de caracteres | description de la Marker, utilizada por ejemplo para almacenar la dirección correspondiente a la posición de la Marker. <br><br>El texto de esta description se utiliza en el popup de la Marker.<br><br>Esta propiedad es opcional. |
| Image | Cadena de caracteres o [MarkerImage](../WDLang3/1000025796.md) Variable | <br><br>- La trayectoria de la imagen correspondiente al icono asociado a la Marker. <br><br>- [MarkerImage](../WDLang3/1000025796.md) Variable utilizado para personalizar el Marker Image. <br><br><br>Si no se especifica esta Property, se usará la Default Image de la control Mapa. Esta imagen se puede especificar en la pestaña "General" de la ventana de control de description. <br><br>**Observación**: En el mapa, el icono asociado a la Marker se mostrará sólo Above la posición especificada. |
| Móviles | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la Marker puede ser movida por el usuario (Drag and Drop), <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>Observación: Si el marcador es móvil, se dispone de una nueva Event opcional: "Mover marcador". Esta Event toma como parámetro la variable Marker asociada a la Marker movida. La posición geográfica asociada a la Marker se actualiza automáticamente.  <br><br> |
| Nota | Cualquier tipo | Se utiliza para almacenar información adicional (no visible para el usuario) sobre la Marker. <br><br> |
| Opacidad | Integro | Porcentaje de opacidad de la Marker sobre la control Mapa. <br><br><br><br> |
| ubicación | Variable de tipo [geoPosition](../WDLang3/1000019191.md) | Posición geográfica (latitud y longitud) en la que debe mostrarse la Marker. <br><br>Si no hay ninguna posición asociada al marcador, la **Position** Property volverá NULL |
| WithPopup | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si se puede mostrar un popup cuando se hace clic en el Marker,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. El popup muestra el nombre (**Name** Property) y el description del marcador (**description** Property). <br><br><br>La ventana emergente puede personalizarse con las propiedades [InternalPagePopup](../Proprietes/1000026257.md) o [InternalWindowPopup](../Proprietes/1000025608.md). <br><br> |
| ZOrder | Integro | Z-orden de la Marker en relación con las otras Marker s en la control Mapa. Le permite manejar las superposiciones. <br><br>Un Marker con un orden Z alto se dibujará sobre un Marker con un orden Z más bajo. <br><br>Este Property es set a 0 por Default. <br><br> |


## TaskAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega una tarea a la lista de tareas de un mensaje de Lotus Notes o de Outlook.. La [Estructura de la tarea](../WDLang3/1000012010.md) se utiliza para definir los datos de esta tarea.

**Atención**: Esta función **no se puede utilizar** para agregar una tarea a la lista de tareas de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf", ...
				"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Initialize the variables of the Task structure
TaskReset()
// Fill the variables for the new appointment
mTask.Subject = "New task"
mTask.StartDate = "20070125100000"
mTask.NbParticipant = 3
mTask.Participant[1] = "Roman"
mTask.Participant[2] = "Lucas"
mTask.Participant[3] = "Stephen"
// Add the task
IF TaskAdd(SessionID) = True THEN
	Info("Task successfully added")
ELSE
	Error("Error during the addition", ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TaskAdd(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).




## &lt;AutoRefreshSeries (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o define si una serie se actualiza cada vez que se dibuja el gráfico.. En este caso, el contenido de la serie se recalcula automáticamente cada vez que se llama a [&lt;Dibujo](../WDLang3/1000024062.md).

**Ejemplo de uso**: una serie en un gráfico se utiliza como serie reference: esta serie no se actualiza automáticamente.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control
IF CHART_MyChart.AutoRefreshSeries(5) = True THEN
	CHART_MyChart.Draw()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificando el modo utilizado para actualización la serie

`<Result> = <Chart control>.AutoRefreshSeries(<Series number>)`
---

**`<Result>: (booleano)`**

Modo actual para la actualización de la serie: 

- <u><u><u><u>True</u></u></u></u> si la serie se actualiza automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie afectada.


<a name="SYNTAX2"></a>

### Definiendo el modo utilizado para actualización la serie

`<Chart control>.AutoRefreshSeries(<Series number> , <Refresh mode>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie afectada.

**`<Refresh mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la serie debe actualizarse automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## grCategoryColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la categoría color para un gráfico cuyo tipo es: 

- Circular, 

- Columna,

- Gráfico de burbujas, 

- Cascada. 




Por defecto, los colores de las categorías (o los colores de las secciones, burbujas, ...) se seleccionan según un orden preset entre un set de 11 colores. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the color of category 3 in the "CHART_Chart1" control
grCategoryColor(CHART_Chart1, 3, LightRed)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grCategoryColor(<Chart name> , <Category number> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Category number>: (Integro)`**

Número de la categoría a modificar. 
**Para un gráfico circular** (constante *grPie* ), este parámetro corresponde al índice de datos en la serie (el número de sección). En este caso, esta función equivale a [grSeriesColor](../WDLang3/3042013.md). 
**Para un gráfico de columnas** (constante *grColumn*), este parámetro corresponde al índice de la categoría: todas las barras de la categoría estarán coloreadas. Los colores especificados para la serie de categorías serán ignorados.. 
**Para un gráfico de burbujas** (constante *grBubbleChart*), este parámetro corresponde al índice de la categoría: la burbuja correspondiente a la categoría será coloreada. 

**`<Color>: (Entero o constante de tipo Integer)`**

color de categoría especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los colores definidos con la función **grCategoryColor** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).
	

- Para visualizar los degradados color, utilice [grGradient](../WDLang3/3042029.md).

- Los colores especificados con la función **grCategoryColor** se tienen en cuenta en gráficos de cascada. También tiene la posibilidad de definir los colores de las categorías en los gráficos de cascada con [grWaterfallColor](../WDLang3/1000020957.md).

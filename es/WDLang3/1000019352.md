
## WiFiConnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta el dispositivo a la red Wi-Fi especificada.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF WiFiConnect("14758") = True THEN
	Info("Wi-Fi connection established.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WiFiConnect(<Identifier> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se estableció la conexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Identifier>: (Cadena de caracteres)`**

Identificador de la red a la que queremos conectarnos. Este identificador puede ser: 

- el identificador devuelto por [WiFiAddNetwork](../WDLang3/1000019351.md) (para una red recién añadida). 




**`<Password>: (Cadena de caracteres opcional)`**

Contraseña utilizada para conectarse a una red segura. Si no se especifica este parámetro, la contraseña utilizada será la guardada en la configuración de red (si existe esta contraseña).

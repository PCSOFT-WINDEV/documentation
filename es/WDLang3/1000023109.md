
## beaconDetectionInfo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **beaconDetectionInfo** se utiliza para describir la información relativa a una baliza detectada con [BeaconDetectPrecise](../WDLang3/1000023115.md). Las características de este faro pueden ser leídas usando diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start detecting tags
BeaconDetectPrecise(group, ProcDetection)
INTERNAL PROCEDURE ProcDetection(arrInfo is array of beaconDetectionInfo)
	nMinDistance is int 
	NearestBeacon is beaconDetectionInfo	
	FOR EACH MyInfo of arrInfo
		IF nMinDistance = 0 _OR_ MyInfo..Distance < nMinDistance
			NearestBeacon = Info
		END
	END
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo beaconDetectionInfo
<a name="propiedades_especificas_descripcion_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000031"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **beaconDetectionInfo**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Distancia | Real | La distancia (en metros) entre el dispositivo y la baliza. La precisión de la medición puede variar de un dispositivo a otro, pero también si el faro está en un espacio cerrado (por ejemplo, una bolsa).<br>**Esta propiedad es de solo lectura**.<br><br> |
| Mayor | Integro | Número principal del Faro (número entero entre 0 y 65535).<br>**Esta propiedad es de solo lectura**. |
| Menor | Integro | Número menor del Faro (entero entre 0 y 65535).<br>**Esta propiedad es de solo lectura**. |
| Proximidad | Constante de tipo Integer | La proximidad del faro al dispositivo:<br><br>- *biDistant*: más allá de 3 metros.<br><br>- *biNear*: Baliza entre 0,5 m y 3 metros.<br><br>- *biVeryClose*: Faro en un radio de 0,5 m.<br><br><br>**Esta propiedad es de solo lectura**. |
| UUID | Cadena de caracteres | Identificador (o UUID de proximidad) associated con la baliza. En la mayoría de los casos, este identificador corresponde a la organización o a la persona a la que pertenece el Faro.<br>Este identificador incluye 32 dígitos hexadecimales separados en 5 grupos. Cada grupo debe contener el siguiente número de dígitos: <br><br>- 1er grupo: 8 dígitos.<br><br>- 2º grupo: 4 dígitos.<br><br>- 3er grupo: 4 dígitos.<br><br>- 4º grupo: 4 dígitos.<br><br>- 5º grupo: 12 dígitos.<br><br><br>Los diferentes grupos están separados por un guión.  <br><br>Ejemplo de identificador válido: f4231ab6-5ef2-6c99-4229-af6c72e0446e<br><br>**Esta propiedad es de solo lectura**. |

### Reinicio
<a name="reinicio_ELTPARAGRAPHE000108"></a>

Puede utilizar [VariableReset](../WDLang1/3013069.md) para restablecer el contenido de una variable de tipo **beaconDetectionInfo**.

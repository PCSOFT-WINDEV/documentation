
## tapiPlay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reproduce un archivo de sonido (.WAV) para el Line especificado. Tienes la habilidad de hacerlo:

- reproducir el mensaje del contestador automático cuando se recibe una llamada,

- reproducir un mensaje durante una llamada saliente (después de [tapiDial](../WDLang3/3076013.md), por ejemplo).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PROCEDURE CallDetection(nServiceID, nCallID, nStatus)

// Picks up the line to answer a call
tapiAnswerCall(nCallID)

// Trigger the answerphone
tapiPlay("answerphone.wav", tapiPlayLock, nCallID)

// If the star key was pressed
// the answerphone is stopped
IF Position(tapiKeyPressed(nCallID), "*") >0 THEN
	tapiStop(nCallID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiPlay(<Name of WAV file> [, <Option> [, <Call identifier>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se reproduce el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si el archivo no se reproduce, compruebe el formato del archivo WAV. Para obtener más información, consulte [observaciones](#NOTE0_1).




**`<Name of WAV file>: (Cadena de caracteres)`**

Nombre y ruta del archivo.WAV a reproducir. Se debe especificar la extensión del archivo.

**`<Option>: (Entero opcional)`**

Opción utilizada para especificar las opciones de archivo de sonido a reproducir.


|   |   |
| --- | --- |
| *tapiPlayLock* | La función es bloquear: el código continúa ejecutándose sólo cuando se reproduce el sonido. <br>Si no se especifica este parámetro, la función devuelve un resultado tan pronto como el sonido comienza a sonar. |
| *tapiPlayLoop* | El sonido se reproduce en bucle.<br>Si no se especifica este parámetro, el sonido se reproduce una sola vez. |



**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). La llamada actual se gestiona si no se especifica este parámetro.

## Observaciones




### Formato del archivo.WAV
<a name="formato_del_archivowav_ELTPARAGRAPHE000070"></a>

Atención: Algunos formatos WAV no son compatibles con la telefonía. Debe utilizar el códec de compresión CCITT G.711 A-Law, u-Law suministrado por Microsoft.

Observación: Un programa GSpot le permite comprobar el codec del archivo.WAV.

### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000208"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.

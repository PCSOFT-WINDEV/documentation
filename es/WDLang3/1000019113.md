
## SaaSAdminAddClient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una cuenta de cliente SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declares an account for the "Hybrid" company in the database of SaaS Webservice
S is saasClient
S.Name = "Hybrid"
SaaSAdminAddClient(S)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadir una cuenta de cliente (a través de las variables saasxxx)

`<Result> = SaaSAdminAddClient(<New Client>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<New Client>: (Variable de tipo saasClient)`**

Nombre de la variable [saasClient](../WDLang3/1000019072.md) que contiene el description de la cuenta del cliente a añadir.


<a name="SYNTAX2"></a>

### Añadir un nuevo cliente identificado por su nombre

`<Result> = SaaSAdminAddClient(<Client account> [, <External reference>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Client account>: (Cadena de caracteres)`**

Nombre del cliente a añadir a la base de datos SaaS.

**`<External reference>: (Cadena de caracteres opcional)`**

reference externo de la cuenta del cliente. Si no se especifica este parámetro, el valor de reference externo corresponde a una cadena vacía ("").


## Acceso a Lotus Notes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000173"></a>
WINDEV y WINDEV le permiten acceder fácilmente a los datos manejados por Lotus Notes (base de datos Domino).. Usted tiene la posibilidad de recuperar, modificar, añadir o manipular sus contactos, citas, tareas, correos electrónicos, bases de datos, ....

Los datos gestionados por Lotus Notes están organizados de la siguiente manera:

- Una base de datos por usuario que contiene los correos electrónicos, las tareas y las citas de este usuario.

- Una base de datos común a todos los usuarios que contiene los contactos y los grupos de contactos.

- Varias bases de datos que contienen documentos.




Para acceder a Lotus Notes, el cliente de Lotus Notes debe estar instalado en los ordenadores de los usuarios finales.

## Modo de funcionamiento
<a name="modo_funcionamiento_ELTTEXTE000197"></a>


### Método para acceder a Lotus Notes:
<a name="metodo_para_acceder_lotus_notes_ELTPARAGRAPHE000026"></a>

1. Establecimiento de la conexión con el servidor de Domino: [NotasOpenConnection](../WDLang3/1000005102.md).

2. Iniciar una sesión para acceder a los correos electrónicos, tareas, citas, contactos y/o grupos de contactos: [EmailStartNotesSession](../WDLang3/3032121.md) (especificando el identificador de la conexión y la base de datos local o remota de Lotus Notes a utilizar, sintaxis 2) o [EmailStartSession](../WDLang3/3032028.md) asociado a un [emailNotesSession](../WDLang3/1000018768.md) Variable.

3. Iniciar una sesión para acceder a los documentos: [NotasOpenDatabase](../WDLang3/1000005101.md) (especificando el identificador de la conexión y la base de datos local o remota de Lotus Notes a utilizar).

4. Cambiar la base de datos de Lotus Notes para utilizarla: [NotasCerrarBase de datos](../WDLang3/1000005111.md) y [NotasOpenDatabase](../WDLang3/1000005101.md).

5. Cierre de la conexión con el servidor de Domino: [Sesión cerrada](../WDLang3/3032124.md) o [EmailCloseSession](../WDLang3/3032006.md).




**Método rápido si no desea acceder a los documentos**
Si solo desea acceder a los correos electrónicos, tareas, citas, contactos y/o grupos de contactos, se puede establecer la conexión con el servidor Domino y abrir una sesión de acceso a la base de datos de Lotus Notes utilizando directamente la función [EmailStartNotesSession](../WDLang3/3032121.md) (sintaxis 1), sin llamar a la función [NotasOpenConnection](../WDLang3/1000005102.md).
<a name="NOTE2_2"></a>


### Versión de Lotus Notes
<a name="version_lotus_notes_ELTPARAGRAPHE000074"></a>

El acceso a Lotus Notes está disponible para Lotus Notes versión 6.5 o posterior.

## Manejando los datos
<a name="manejando_los_datos_ELTTEXTE000227"></a>
Una vez que se estableció la conexión con el servidor de Domino y una vez que se especificó la base de datos de Lotus Notes, se tiene la capacidad de manipular:

- los correos electrónicos a través de la [Funciones de Email](../WDLang3/3032034.md).

- las tareas a través de la [Funciones de la tarea](../WDLang3/1000012009.md).

- las citas a través de la [Funciones de Appointment](../WDLang3/1000011009.md).

- los contactos a través de la [Funciones de Contact](../WDLang3/1000006110.md).

- los grupos de contactos a través de la [Funciones de grupo](../WDLang3/1000008119.md).

- los documentos a través de la [Funciones de notas](../WDLang3/1000005117.md).



<a name="NOTE3_2"></a>


### Envío de correos electrónicos
<a name="envio_correos_electronicos_ELTPARAGRAPHE000113"></a>

Cuando se envía un Email, si se encuentran uno o más caracteres CR al final del mensaje saliente, estos caracteres se borran automáticamente por Lotus Notes. Esta eliminación se realiza cuando el mensaje se convierte en una cadena de texto enriquecido.

Atención: Para enviar correos electrónicos, Lotus Notes debe estar instalado en modo conectado (y no en modo local).
<a name="NOTE3_3"></a>


### Leyendo emails
<a name="leyendo_emails_ELTPARAGRAPHE000122"></a>

El orden en el que se leen los mensajes de correo electrónico puede diferir del orden en el que se muestran los mensajes de correo electrónico en Lotus Notes.

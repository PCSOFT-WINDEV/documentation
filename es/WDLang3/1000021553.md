


## EmailExpunge (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Destruye todos los correos electrónicos eliminados marcados como eliminados (estado"Eliminado") de la carpeta actual de la sesión IMAP..

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMySession is emailIMAPSession
...
IF EmailStartSession(cMySession) THEN
	cMyEmail is Email
	EmailReadFirst(cMySession, cMyEmail)
	WHILE NOT cMyEmail..Out
		EmailDeleteMessage(cMySession, cMyEmail)
		EmailReadNext(cMySession, cMyEmail)
	END
	// Destroys the deleted emails
	EmailExpunge(cMySession)
	EmailCloseSession(cMySession)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EmailExpunge(<Session>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la destrucción,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Session>: (Variable de tipo emailIMAPSession)`**

Nombre de [emailIMAPSession](../WDLang3/1000018957.md) Variable correspondiente a la sesión en la que se destruirán los mensajes.

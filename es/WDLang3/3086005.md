## Instalación de un servidor SOAP con IIS (versión 4 o 5)
<a name="instalacion_servidor_soap_con_iis_version_4_5_ELTTEXTE000162"></a>
Para instalar el servidor SOAP con IIS 5.1 (o IIS 4) en Windows 2000 (el servidor IIS debe estar instalado en la computadora):

1. En el panel de control, seleccione "Herramientas de gestión" y luego "Gerente de los servicios de INTERNET".

2. En la lista de la izquierda, elija el servidor al que desea añadir soporte SOAP.

3. En el panel de la derecha, elija un sitio Web ("Sitio Web de Default" por ejemplo).

4. Despliega el menú contextual y elige "Propiedades", pestaña "Directorio de inicio".

5. Haga clic en el botón "Configuración" y luego en el botón "Añadir".

6. Haga clic en "Executable" y elija el archivo wdxxxsiis.dll. La ruta no debe contener ningún carácter espacial. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=config_IIS.gif)

	Para las pruebas desde su ordenador, este archivo se encuentra en el subdirectorio "Data" del directorio de instalación de WINDEV.
	Para instalar el servidor SOAP final, este archivo se encuentra en el directorio de instalación de su aplicación.

7. En la extensión control, especifique ".jabón". 
	Observación: Si la extensión ".soap" ya es associated con un módulo en su servidor, use una extensión diferente. En este caso, esta nueva extensión debe ser especificada en las funciones que indican la URL del servidor SOAP (&lt;URL del servidor SOAP&gt; en [SOAPRun](../WDLang3/3069013.md) por ejemplo).

8. Desmarque "Comprobar la existencia del archivo".

9. Haga clic en Aceptar (dos veces). 

10. **Si desea instalar un servidor SOAP estándar,** se completa la configuración del servidor.

## Instalación de un servidor SOAP con IIS (versión 6)
<a name="instalacion_servidor_soap_con_iis_version_6_ELTTEXTE000186"></a>
Para instalar el servidor SOAP con IIS 6 en Windows 2000 (el servidor IIS debe estar instalado en la computadora):

1. En el panel de control, seleccione "Herramientas de gestión" y luego "Gerente de los servicios de INTERNET".

2. En la lista de la izquierda, elija el servidor al que desea añadir soporte SOAP.

3. En el panel de la derecha, elija un sitio Web ("Sitio Web de Default" por ejemplo).

4. Despliega el menú contextual y elige "Propiedades", pestaña "Directorio de inicio".

5. Haga clic en el botón "Configuración" y luego en el botón "Añadir".

6. Haga clic en "Executable" y elija el archivo wdxxxsiis.dll. La ruta no debe contener ningún carácter espacial. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=config_IIS.gif)

	Para las pruebas desde su ordenador, este archivo se encuentra en el subdirectorio "Data" del directorio de instalación de WINDEV.
	Para instalar el servidor SOAP final, este archivo se encuentra en el directorio de instalación de su aplicación.

7. En la extensión control, especifique ".jabón". 
	Observación: Si la extensión ".soap" ya es associated con un módulo en su servidor, use una extensión diferente. En este caso, esta nueva extensión debe ser especificada en las funciones que indican la URL del servidor SOAP (&lt;URL del servidor SOAP&gt; en [SOAPRun](../WDLang3/3069013.md) por ejemplo).

8. Desmarque "Comprobar la existencia del archivo".

9. Haga clic en Aceptar (dos veces).

10. En el panel de la derecha, seleccione "Extensión del servicio web" (o "Extensiones del servicio web" según la versión utilizada).. Esta opción se utiliza para definir los derechos de ejecución de los archivos.soap.

11. Seleccione "Añadir una nueva extensión de servicio web".

12. En la ventana, introduzca:

	- El nombre de la extensión (.soap por ejemplo).

	- Los archivos necesarios. Haga clic en "Añadir" y elija el archivo wdxxxsiis.dll.
			Para las pruebas desde su ordenador, este fichero se encuentra en el subdirectorio Datos del directorio de configuración de WINDEV.
			Para instalar el servidor SOAP final, este archivo se encuentra en el directorio de instalación de su aplicación.




13. Valide.

14. Haga clic en el botón "Autorizar".

## Instalación de un servicio web
<a name="instalacion_servicio_web_ELTTEXTE000210"></a>


### Si desea instalar un servicio web
<a name="desea_instalar_servicio_web_ELTPARAGRAPHE000098"></a>

**15.** Copiar en el directorio especificado en la opción "Access ruta":

- El archivo XML que contiene el servicio web description.

- Las páginas HTML que presentan el servicio web.




Para acceder a este archivo XML, utilice el siguiente comando Line:
"http://&lt;Name or IP Address of Soap Server&gt;/&lt;Name of XML File&gt;.XML"

Para ejecutar la prueba de este servicio web, use el siguiente comando Line
"http://&lt;NAme o IP Address de Soap Server&gt;/index.htm"

**Observación**: Este archivo XML también se puede copiar en otro directorio. Consulte la documentación sobre el servidor HTTP utilizado para obtener más detalles.

**16.** El Webservice está ahora instalado en el servidor SOAP.

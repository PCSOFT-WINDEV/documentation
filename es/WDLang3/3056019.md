
## NetMsgError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el mensaje de error correspondiente al número de error devuelto por [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md) y [NetCloseRemoteAccess](../WDLang3/3056021.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OpeningReport is int = NetOpenRemoteAccess("MyConnection")
IF OpeningReport = 0 THEN
	...
	NetCloseRemoteAccess()
ELSE
	Error("Error during the connection", NetMsgError(OpeningReport))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetMsgError(<Error number>)`
---

**`<Result>: (Cadena de caracteres)`**

Mensaje de la Error( ver las Notas).

**`<Error number>: (Integro)`**

Número de error, devuelto por [NetOpenRemoteAcceso remoto](../WDLang3/3056026.md) y [NetCloseRemoteAccess](../WDLang3/3056021.md).

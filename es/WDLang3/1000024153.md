


## &lt;Gráfico&gt;.AxisTitle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el título del eje X y el título del eje Y en un gráfico.

Ejemplo de títulos para los ejes del gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=titreaxe.gif)


**Observación**: Default no define ningún título para los ejes de un gráfico.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the title of the horizontal axis in "CHART_MyChart"
CHART_MyChart.AxisTitle("Time", grXCoordinate)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.AxisTitle(<Title> , <Axis direction>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico en la ventana, página o editor de reportes.


**`<Title>: (Cadena de caracteres)`**

Título del eje especificado.

**`<Axis direction>: (Constante)`**

Orientación del eje a utilizar:


|   |   |
| --- | --- |
| *grSecondaryYCoordinate* | Marcas en el eje Y secundario (eje vertical). |
| *grXCoordinate*<br>(valor predeterminado) | Marcas en el eje X (eje horizontal). |
| *grYCoordinate* | Marcas en el eje Y (eje vertical). |

<a name="NOTE0_1"></a>

## Observaciones


### Tipos de gráficos
<a name="tipos_graficos_ELTPARAGRAPHE000289"></a>

Dependiendo del tipo de gráfico y de la &lt;Dirección del eje&gt;, la función **&lt;Gráfico&gt;.AxisTitle** se tiene en cuenta o no:

| Tipo de gráfico | Eje horizontal | Eje vertical |
| --- | --- | --- |
| Gráfico de líneas | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |
| Gráfico de cotizaciones (Candelabro y gráfico de barras) | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |
| Gráfico de columnas | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |
| Min/Max Gráfico de cotizaciones | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |
| Dispersión | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |
| Circular | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. | **&lt;Gráfico&gt;.AxisTitle** tenida en cuenta. |


Las modificaciones realizadas con la función **&lt;Gráfico&gt;.AxisTitle** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).
<a name="NOTE0_2"></a>


### Título ejes
<a name="titulo_ejes_ELTPARAGRAPHE000378"></a>El título de los ejes del gráfico se puede mostrar para los diferentes ejes.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)
El contenido del título de los ejes del gráfico se define con la función **&lt;Gráfico&gt;.AxisTitle**. Esta función también se puede utilizar para posicionar este título según el eje deseado. No se puede definir el posicionamiento y/o la orientación del título en relación al eje..  

El tipo de letra utilizado para el título del gráfico puede ser modificado por [&lt;Tabla&gt;.AxisTitleFont](../WDLang3/1000024120.md).
<a name="NOTE0_3"></a>


### Las lenguas que utilizan un carácter no latino set
<a name="las_lenguas_que_utilizan_caracter_latino_set_ELTPARAGRAPHE000399"></a>

**&lt;Gráfico&gt;.AxisTitle** utilizará el carácter no latino seleccionado set sólo si la configuración del proyecto associated utiliza cadenas UNICODE en tiempo de ejecución.

Para comprobar la opción seleccionada en la configuración actual: 

1. En la pestaña "Proyecto", en el grupo "Configuración del proyecto", haga clic en "Configuración actual". 

2. Ir a la pestaña "Unicode": la opción "Utilizar cadenas UNICODE en ejecución" debe estar seleccionada. 

3. Validar si es necesario.

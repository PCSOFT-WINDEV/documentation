
## emailPOP3Session (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **emailPOP3Session** se utiliza para acceder a un servidor de mensajería utilizando el protocolo POP3 (Post Office Protocol) con el fin de recuperar los mensajes que se encuentran en este servidor. Las características de la conexión pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="XExample1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a POP3 session
MySession is emailPOP3Session

MySession.ServerAddress = "pop.mydomain.fr"
MySession.Name = "user"
MySession.Password = "secret"

EmailStartSession(MySession)

// Retrieve all the messages
arrMessage is array of Emails 
arrMessage = EmailGetAll(MySession)

// Browse the array of messages by index
FOR I = 1 _TO_ MySession.EmailBySubscript.Count
	Trace(MySession.EmailBySubscript[i].Subject)
END

// Browse the array of messages with a FOR EACH loop
FOR EACH AMessage OF MySession
	Trace(AMessage.Subject)
END
```

## Observaciones


### Propiedades específicas de las variables emailPOP3Session
<a name="propiedades_especificas_las_variables_emailpop3session_ELTPARAGRAPHE000053"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **emailPOP3Session**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ServerAddress | Cadena de caracteres | Address del servidor Email. Este Address es suministrado por el proveedor de servicios o por el administrador de la red. Esta Address se puede dar en el siguiente formato:<br><br>- IP Address (en formato XXX.XXX.XXX.XXX, 125.5.110.100 por ejemplo).<br><br>- Nombre del servidor (pop3.free.fr por ejemplo). Se recomienda esta sintaxis.<br><br><br> |
| Contraseña | Cadena de caracteres | Contraseña de usuario. Esta contraseña es proporcionada por el proveedor de servicios o por el administrador de la red. |
| denominar | Cadena de caracteres | Identifica al usuario. Este nombre es proporcionado por el proveedor de servicios o por el administrador de la red. |
| Opción | Constante de tipo Integer | Los valores posibles son:<br><br>- *emailOptionDefault* (valor de Default): Conexión mediante el protocolo POP3.<br><br>- *optionSSL*: Conexión mediante el protocolo POP3 dentro de un túnel SSL. Esta opción garantiza la confidencialidad del intercambio pero debe ser soportada por el servidor.<br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este Property no se apoya. |
| Puerto | Integro | Identifica el puerto utilizado para el protocolo POP3.<br>Este Property es set a 110 por Default, correspondiente al número de puerto estándar para este protocolo.<br><br>**Observación**: Para utilizar el protocolo SSL, se debe especificar el número de puerto. En la mayoría de los casos, los servidores de Email esperan las conexiones SSL en el puerto 995. |




## <Chart>.ScatterGetDataXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un dato de un gráfico"Scatter".. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display data in the "CHART_MyChart" control ("Scatter" chart). 
// This data is the first X-coordinate of first series.
Info(CHART_MyChart.ScatterGetDataXY(1, 1, grXValue))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Chart control>.ScatterGetDataXY(<Series number> , <Subscript> , <Value to retrieve>)`
---

**`<Result>: (Real)`**

Valor recuperado.

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (se encuentra en el editor de ventanas, editor de páginas o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie para la que se recuperará un valor. 
Si la serie especificada no existe, el valor recuperado es 0.

**`<Subscript>: (Integro)`**

Índice de los datos de la serie cuyo valor se recuperará.
Si el dato correspondiente al índice especificado no existe en la serie, el valor recuperado es 0.

**`<Value to retrieve>: (Constante de tipo Integer)`**

Indica el valor a recuperar:


|   |   |
| --- | --- |
| *grXValue* | Valor X de la serie. |
| *grYValue* | Valor Y de la serie. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [.StockGetData](../WDLang3/1000023566.md).

- Para recuperar los datos de un gráfico de superficie, se recomienda utilizar [grSurfaceGetData](../ErrCompil/1000001036.md).

- Para recuperar los datos de otro tipo de gráfico (circular, de columnas, Line, etc.), le recomendamos que utilice [&lt;Gráfico&gt;.GetData](../WDLang3/1000024122.md).

- Para recuperar los datos de un gráfico Sunburst, se recomienda utilizar [&lt;Gráfico&gt;.SunburstGetData](../WDLang3/1000024137.md). 

- Para recuperar los datos de un gráfico Heatmap, se recomienda utilizar [&lt;Gráfico&gt;.HMGetData](../WDLang3/1000024083.md).

- Le recomendamos utilizar la función **<Chart>.ScatterGetDataXY** para recuperar datos en gráficos de "dispersión". También se puede utilizar la función [&lt;Gráfico&gt;.GetData](../WDLang3/1000024122.md).
	La siguiente sintaxis:
	
	```wl
	CHART_MyChart.ScatterGetDataXY(i, 1, grXValue)
	CHART_MyChart.ScatterGetDataXY(i, 1, grYValue)
	```
 es equivalente a la sintaxis: 
	
	```wl
	CHART_MyChart.GetData(i*2-1, 1)
	CHART_MyChart.GetData(i*2, 1)
	```


## SaaSAdminDeletePricing( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un precio SaaS. Sus suscripciones se eliminan automáticamente. 

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the "Promo 2009" pricing from the "Forum" site
S is saasSite 
P is saasPricing

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2009", S)

SaaSAdminDeletePricing(P)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminar un precio (utilizando las variables saasxxx)

`<Result> = SaaSAdminDeletePricing(<Pricing>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (saasPricing Variable)`**

Nombre de la [saasPricing](../WDLang3/1000019091.md) Variable correspondiente a la tarificación que se quiere eliminar de la base de datos SaaS.


<a name="SYNTAX2"></a>

### Suprimir un precio identificado por su nombre

`<Result> = SaaSAdminDeletePricing(<Pricing> , <SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre de la tarificación que se va a eliminar de la base de datos de SaaS.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS al que pertenece el precio.


## NetMACAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el MAC Address (físico Address) de una de las tarjetas de red que se encuentran en el equipo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the MAC address of the computer
Info(NetMACAddress())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetMACAddress([<Rank of Network Card>])`
---

**`<Result>: (Cadena de caracteres)`**



- MAC Address (en formato "aa:bb:cc:dd:ee:ff"),

- Vacíe la cadena (""") si se produce un problema.




**`<Rank of Network Card>: (Entero opcional)`**

Se utiliza para especificar el número de la tarjeta de red cuyo MAC Address debe ser recuperado (para los ordenadores equipados con varias tarjetas de red). El rango de la primera carta es 0. 
Si este parámetro es negativo, el Address devuelto será el de la primera tarjeta de red.

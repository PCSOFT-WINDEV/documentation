


## grCategoryLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- Inicializar la etiqueta de una categoría de datos (o sección) en un gráfico. 
	

- get la etiqueta de una categoría de datos (o sección) en un gráfico.




Ejemplo de etiqueta de categoría:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)


**Recordatorio**: Un elemento de la carta no tiene la etiqueta de Default.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the "February" label of the second data category
// in "MyChart"
grCategoryLabel("MyChart", 2, "February")
```
<a name="Example2"></a>

```wl
// Initialize the "February" label of the second data category
// in "MyChart"
grCategoryLabel(CHART_MyChart, 2, "February")
```
<a name="Example3"></a>

```wl
// Read label of second data category
// in "MyChart"
MyLabel is string = grCategoryLabel(CHART_MyChart, 2)
```

## Sintaxis

### Inicialización de la etiqueta de una categoría de datos (o sección) en un gráfico

`grCategoryLabel(<Chart name> , <Index> , <Label>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Index>: (Integro)`**

Índice de datos de la serie.

**`<Label>: (Cadena de caracteres)`**

Leyenda de la etiqueta a inicializar. 
Este parámetro también puede corresponder a una duración, una fecha, una hora o una fecha-hora.. 
**Observación**: Si se utiliza una escala de abscisas temporal, se debe llamar sistemáticamente a **grCategoryLabel** cada vez que se añadan datos al gráfico con [grAddData](../WDLang3/3042023.md).


<a name="SYNTAX2"></a>

### Obtener la etiqueta de una categoría de datos (o sección) en un gráfico

`<Result> = grCategoryLabel(<Chart name> , <Index>)`
---

**`<Result>: (Cadena de caracteres)`**

Leyenda de la etiqueta.

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Index>: (Integro)`**

Índice de datos (con la etiqueta deseada) en la serie.

## Observaciones


### Gestión de las etiquetas de las categorías
<a name="gestion_las_etiquetas_las_categorias_ELTPARAGRAPHE000213"></a>

Se visualizan las etiquetas de categoría:

- por debajo de las marcas horizontales para los gráficos de Line y de acciones (Candlestick y BarCharts),

- debajo del eje X para los gráficos de Columna y MinMax.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteCat.gif)
Las etiquetas de categoría no pueden utilizarse en los gráficos de dispersión.

La posición de las etiquetas de las categorías no se puede cambiar. 

Las etiquetas de categorías se definen con la función **grCategoryLabel**.

Los datos definidos con la función **grCategoryLabel** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

El tipo de letra de la etiqueta puede ser modificado por [grLabelFont](../WDLang3/3042012.md).

Las etiquetas de categorías definidas con la función **grCategoryLabel** se pueden mostrar en las etiquetas de datos del gráfico circular (constante *grShowLabel* utilizada con la función [grLabel](../WDLang3/3042046.md)).

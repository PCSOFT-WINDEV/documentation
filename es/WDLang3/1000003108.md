## Presentación
<a name="presentacion_ELTTEXTE000153"></a>
La automatización del hogar a través del portador de energía-Line (X10) se ha convertido en una realidad. La gestión de los aparatos eléctricos a través del cableado existente prepara el camino para una fácil control remota mediante la programación.

¿Sabía que un sistema eléctrico estándar puede convertirse fácilmente en domótica?

Sin cableado adicional y sin ondas de radio, los aparatos eléctricos pueden ser fácilmente controlados por proyectos WINDEV, WINDEV Mobile o WEBDEV.. Para ello:

- utilizar interruptores específicos o sockets (se utiliza el estándar X10).

- usar un multienchufe específico.




Encender las luces, cerrar las persianas, apagar un dispositivo o averiguar su estado (apagado, encendido) puede realizarse fácilmente desde un proyecto WINDEV, WINDEV Mobile o WEBDEV.

**Observación**: En esta versión, las interfaces CM11 (de Marmitek) y las interfaces CM15 son compatibles..

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000177"></a>


### Control de electrodomésticos
<a name="control_electrodomesticos_ELTPARAGRAPHE000031"></a>

A los aparatos eléctricos de control:

1. Establezca una conexión con una interfaz X10 ([DomoOpen](../WDLang3/1000003102.md)).

2. Conecte los aparatos eléctricos ([DomoSwitchOn](../WDLang3/1000003101.md) o [DomoSend](../WDLang3/1000003104.md)).

3. Enviar el comando a realizar: encender, aumentar o reducir la intensidad, parar ([DomoSend](../WDLang3/1000003104.md)).

4. Comprobar el estado de los dispositivos eléctricos: apagado o encendido ([DomoStatus](../WDLang3/1000003105.md)).

5. Desconecte los aparatos eléctricos ([DomoSwitchOff](../WDLang3/1000003103.md)).

6. Cierre la conexión con la interfaz X10 ([DomoCerrar](../WDLang3/1000003106.md)).




**Observación**: Para que el protocolo X10 funcione correctamente a través de la portadora de potencia-Line, se requiere una potencia continua entre los dispositivos. Por ejemplo, la transmisión no puede pasar a través de un transformador de aislamiento.
<a name="NOTE2_2"></a>


### Control de un módulo CM15
<a name="control_modulo_cm15_ELTPARAGRAPHE000068"></a>

**Condición previa**: 

1. Instale el software "Active Home" suministrado con el módulo CM 15.

2. Instale el programa "ahscript_instalación.exe". Este programa está disponible en el CD "Active Home" suministrado con el módulo CM15.


**Observación**: el cable USB debe estar conectado antes que el cable de alimentación.

**Implementación**

1. Establezca una conexión con una interfaz X10 ([DomoOpen](../WDLang3/1000003102.md)).

2. Conecte los aparatos eléctricos ([DomoSwitchOn](../WDLang3/1000003101.md) o [DomoSend](../WDLang3/1000003104.md)).

3. Enviar el comando a realizar: encender, aumentar o reducir la intensidad, parar ([DomoSend](../WDLang3/1000003104.md)).

4. Desconecte los aparatos eléctricos ([DomoSwitchOff](../WDLang3/1000003103.md)).

5. Cierre la conexión con la interfaz X10 ([DomoCerrar](../WDLang3/1000003106.md)).


**Limitación**: [DomoStatus](../WDLang3/1000003105.md) no está disponible.

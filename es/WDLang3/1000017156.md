
## OBEXGetVCard( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera una VCard de un dispositivo que soporta el protocolo OBEX.

**Observación**: Esta función no se puede utilizar si la transferencia se realiza por infrarrojos.
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OBEXGetVCard(<Identifier of OBEX connection> , <Local file path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la transferencia correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Identifier of OBEX connection>: (Integro)`**

Identificador de la conexión OBEX a utilizar. Este identificador es devuelto por [OBEXConnect](../WDLang3/1000017153.md).

**`<Local file path>: (Cadena de caracteres)`**

ruta completa donde se copiará el archivo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo las pilas de Microsoft bluetooth son soportadas por esta función. Para get la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

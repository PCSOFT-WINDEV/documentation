
## Procedimiento WLanguage asociado a la función AlbumPicker



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [AlbumPicker](../WDLang3/1000020186.md) cuando el usuario selecciona una imagen o video. Este procedimiento puede ser local, global, interno o lambda.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the image
// recommended syntax on iOS and Android
// Caution: On Android 10 and higher, applications must not run in the background. 
bRes is boolean
bRes = AlbumPicker(AlbumPicker_Callback)

INTERNAL PROCEDURE AlbumPicker_Callback(sImagePath = "")
	IMG_MyImage = sImagePath
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AlbumPicker_Callback(<Path>)`
---

**`<Path>: (Cadena de caracteres)`**

Ruta para guardar la imagen o video seleccionado por el usuario.

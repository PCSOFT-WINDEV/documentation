


## &lt;Tabla de herramientas (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pantallas y formatos: 

- la herramienta associated con cada sección del gráfico (sección para un gráfico circular, barra para un gráfico de columnas, etc.). También permite conocer las características de la punta de la herramienta associated con cada sección del gráfico.

- una punta de herramienta personalizada, associated con un punto.




Por defecto, se muestra un tooltip automáticamente (opción *grShowTooltip* establecida en <u><u><u>True</u></u></u>) al pasar el cursor por encima de un gráfico.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control
// Display the percentages on a column chart
CHART_MyChart.Label(grShowPercent, True)
CHART_MyChart.Tooltip(grTooltipFormat, -%"[%SERIES%]" + CR + -%"[%CATEGORY%]" + CR + -%"[%VALUE%]")
CHART_MyChart.Tooltip(grShowTooltip, True)
// Draw the chart
CHART_MyChart.Draw()
```
<a name="Example2"></a>

```wl
// Chart control
// Display a specific tooltip for a point
CHART_MyChart.Tooltip(1, 2, "The content of my tooltip")
// Draw the chart
CHART_MyChart.Draw()
```

## Sintaxis

### Mostrando y formateando la punta de la herramienta associated con un control Gráfico

`<Chart control>.Tooltip(<Tooltip option to configure> , <Tooltip option value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en el editor de ventanas).


**`<Tooltip option to configure>: (Constante de tipo Integer)`**

Habilita o deshabilita el tooltip y configura sus características.


|   |   |
| --- | --- |
| *grShowTooltip* | Habilita o deshabilita el tooltip. |
| *grTooltipFormat* | Describe el formato de la información sobre la herramienta. |
| *grTooltipFormatBubbleChart* | Describe el formato del tooltip para un gráfico de burbujas.<br> |
| *grTooltipFormatCrosshairHeader* | Describe el formato de cabecera para las retículas de gráfico. |
| *grTooltipFormatHeatMap* | Describe el formato de la información de la herramienta para un gráfico de Heatmap.<br> |
| *grTooltipFormatPie* | Describe el formato de la información de herramienta para un gráfico circular. |
| *grTooltipFormatRangeArea* | Describe el formato de la información sobre herramientas para un gráfico de Área de Rango. |
| *grTooltipFormatScatter* | Describe el formato de la información de herramienta para un gráfico de dispersión.<br> |
| *grTooltipFormatStock* | Describe el formato del tooltip para un Gráfico de cotizaciones.<br> |



**`<Tooltip option value>: (Cadena de caracteres o booleana)`**

Valor de la opción para el tooltip seleccionado. Este valor depende de la opción seleccionada (véase el Above tabla).

<a name="SYNTAX3"></a>

### Mostrando y personalizando la punta de la herramienta associated con un punto

`<Chart control>.grTooltip(<Series number> , <Category number> , <Tooltip value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en el editor de ventanas o páginas).


**`<Series number>: (Integro)`**

Número de serie a utilizar.

**`<Category number>: (Integro)`**

Número de la categoría a utilizar.

**`<Tooltip value>: (Cadena de caracteres)`**

Texto personalizado de la punta de la herramienta associated con el punto.

Si se utiliza el formato Default:

- "[%CATEGORY%]" se ignora si no se ha especificado ninguna etiqueta de categoría mediante [.CategoryLabel](../WDLang3/1000024072.md).

- "[%SERIES%]" se ignora si no se ha especificado ninguna etiqueta de serie con [&lt;Tabla&gt;.SeriesLabel](../WDLang3/1000024073.md).


Atención: Si se utiliza la construcción dinámica de cadenas ("Permitir "[% %]" en las cadenas" en la pestaña "Compilación" del proyecto description), se produce un error de compilación (identificador desconocido). En este caso, cada cadena debe ir precedida de '-%'.. Ejemplo: 

```wl
CHART_Deadline.Tooltip(grTooltipFormat, "[%CATEGORY%]" +CR +CR + "[%VALUE%]" + "H")
```
 se convierte en

```wl
CHART_Deadline.Tooltip(grTooltipFormat, -%"[%CATEGORY%]" + CR + CR + -%"[%VALUE%]" + " H")
```

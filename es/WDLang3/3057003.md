
## OLEPaste( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pega el contenido del portapapeles de Windows en un OLE control.

**Observación**: Si el contenido del portapapeles fue creado en un servidor OLE, este servidor es automáticamente associated con el objeto, de lo contrario el objeto no se pegará. Dependiendo del servidor que haya creado el objeto OLE, el objeto puede ser visualizado en la control como:

- su representación de mapa de bits (ejemplo de Image creado con Paint Brush),

- su representación vectorial (ejemplo de hoja de cálculo creada por Excel),

- el Icon que representa el servidor (Icon creado por WinWord),

- texto.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The image section is pasted from Paint Brush into the clipboard.
// The clipboard content is pasted into the control.
// The associated server is Paint Brush.
OLEPaste(OLE_OLE1)
OLESave(OLE_OLE1, "OBJECT1.OLE")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OLEPaste(<OLE control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el objeto fue pegado,

- <u><u><u><u>False</u></u></u></u> si se produce un problema.




**`<OLE control>: (Nombre del control)`**

Nombre de OLE control que recibirá el objeto OLE. Si la control contenía un objeto, éste se sustituye por el nuevo objeto. Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control al que pertenece el actual Event.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Contenido del portapapeles
<a name="contenido_del_portapapeles_ELTPARAGRAPHE000048"></a>

- El contenido del portapapeles puede ser en formato de texto, gráfico o binario.

- El contenido del portapapeles fue inicializado por [OLECopia](../WDLang3/3057004.md), a través de una opción del editor o mediante un comando copy en el portapapeles.



<a name="NOTE0_2"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000060"></a>

- Sólo se admite la versión 1 de OLE.

- El objeto puede ser guardado por [OLESave](../WDLang3/3057011.md).

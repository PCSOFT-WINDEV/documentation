
## SaaSAdminDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la sesión de gestión de una base de datos SaaS.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Management connection to the SaaS database
IF SaaSAdminConnect("admin", "secret") = False THEN
	Error("You are not allowed to connect as administrator")
	RETURN
END

...

// End of management session
SaaSAdminDisconnect()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminDisconnect()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

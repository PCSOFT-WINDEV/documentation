


## &lt;Gráfico&gt;.SunburstGetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un valor numérico de una carta de Sunburst.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyVal is real = CHART_Chart1.SunburstGetData("Europe" + TAB + "France" + TAB + "Montpellier")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Chart control>.SunburstGetData(<Section path>)`
---

**`<Result>: (Real)`**



- Valor a recuperar.

- 0 si la ruta no es válida o si la sección no contiene datos.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección cuyo valor debe ser recuperado. Este parámetro tiene el siguiente formato:


```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [.StockGetData](../WDLang3/1000023566.md).

- Para recuperar los datos de un gráfico de superficie, se recomienda utilizar [grSurfaceGetData](../ErrCompil/1000001036.md).

- Para recuperar los datos de otro tipo de gráfico (circular, de columnas, Line, etc.), le recomendamos que utilice [&lt;Gráfico&gt;.GetData](../WDLang3/1000024122.md).

- Para recuperar los datos de un gráfico Heatmap, se recomienda utilizar [&lt;Gráfico&gt;.HMGetData](../WDLang3/1000024083.md).

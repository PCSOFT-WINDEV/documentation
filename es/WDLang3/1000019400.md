
## DNSAddAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una relación IP/Nombre al servidor DNS.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
c is Duration
c..InDays = 1
DNSResult is boolean

// Add into a specific DNS server
DNSResult = DNSAddAddress("testdoc.doc.local.", "192.168.1.1", c, "Andromeda_2")

IF DNSResult = False THEN
	Error(ErrorInfo(errFullDetails))
END

// Add into the default DNS server
DNSResult = DNSAddAddress("testAp.towerqual.local.", "192.168.1.2")
IF DNSResult = False THEN
	Error(ErrorInfo(errFullDetails))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DNSAddAddress(<Name> , <IP address> [, <Validity period> [, <DNS server> [, <User> [, <Password>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la dirección DNS se agregó correctamente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error. 




**`<Name>: (Cadena Unicode)`**

Nombre completo para asociarse con la IP Address. Este parámetro tiene el siguiente formato: &lt;Nombre &gt; + "." + &lt;Dominio &gt;.. Ejemplo: "testdoc.pcsoft.com"

**`<IP address>: (Cadena de caracteres)`**

Address IP para asociar con &lt;Name&gt;Name. Esta IP Address puede ser una IP Address versión 4 (en el formato xxx.xxx.xxx.xxx) o versión 6 (en el formato aaaa:bbbb:cccc:dddd:eeee:ffff:gggg:hhhh:iiii:jjjjj).

**`<Validity period>: (Duración opcional)`**

Duración (en segundos) de la validez de la solicitud. Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s). 


El valor predeterminado de este parámetro es 0. 

**`<DNS server>: (Cadena de caracteres opcional)`**

Address del servidor DNS a actualizar. La asociación "IP Address/Name" se añadirá a este servidor. Por Default, el actual servidor DNS es usado. 

**`<User>: (Cadena de caracteres Unicode opcional)`**

Nombre del usuario utilizado para firmar la solicitud. 

**`<Password>: (Cadena de caracteres Unicode opcional)`**

Contraseña associated con el usuario para firmar la solicitud. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es compatible con los servidores Linux y Windows DSN que no requieren peticiones firmadas.

- Esta función es compatible con los servidores Linux y Windows DSN que requieren peticiones firmadas.

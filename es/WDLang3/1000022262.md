


## grSeriesTipo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el tipo de serie utilizado en una carta compuesta (carta que contiene varios tipos de cartas).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grSeriesType(CHART_MyChart, 1, grLine)
grSeriesType(CHART_MyChart, 2, grScatter)
grSeriesType(CHART_MyChart, 3, grColumn)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSeriesType(<Chart name> , <Series number> , <Type> [, <Parameter>])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Series number>: (Integro)`**

Número de la serie para la que se modificará el tipo de carta.

**`<Type>: (Constante de tipo Integer)`**

Tipo de gráfico que se utilizará para representar los datos encontrados en la serie:



|   |   |
| --- | --- |
| *grArea* | Gráfico de área. |
| *grBarCharts* | Gráfico de cotizaciones. |
| *grBubbleChart* | Gráfico de burbujas. |
| *grCandleStick* | Gráfico de acciones de Candlestick. |
| *grColumn* | Gráfico de columnas. |
| *grColumnStacked* | Gráfico de columnas apiladas. |
| *grLine* | Gráfico de líneas. |
| *grLineConstant* | Muestra una Line horizontal en la posición especificada en &lt;Parámetros&gt;. |
| *grLineLinearRegression* | Muestra un Line cuya inclinación se calcula realizando una regresión de Line ar en la serie especificada en &lt;Parámetro&gt;. |
| *grLineMeanValue* | Muestra una Line horizontal en la media de las series especificadas en &lt;Parámetros&gt;. |
| *grLineQuadraticRegression* | Gráfico Line en el siguiente formato: "ax2 + bx + c". El Line se define realizando una regresión cuadrática sobre la serie especificada en &lt;Parámetro&gt;. |
| *grMinMax* | Gráfico de cotizaciones que muestra la diferencia entre los valores mínimos y máximos. |
| *grPolarArea* | Gráfico de la zona polar. |
| *grRangeArea* | Gráfico de área de rango. |
| *grScatter* | Gráfico de dispersión. |
| *grWaterfall* | Gráfico de cascada. |



**`<Parameter>: (Constante de tipo Integer)`**

Parámetro requerido para algunos tipos de gráficos. Si &lt;Tipo&gt; corresponde a: 

- *grLineQuadraticRegression*: &lt;Parámetro&gt; corresponde al número de la serie utilizada para realizar la regresión cuadrática.. 

- *grLineConstant*: &lt;Parámetro&gt; corresponde a la posición de visualización de Line horizontal. 

- *grLineMeanValue*: &lt;Parámetro&gt; corresponde al número de la serie utilizada para calcular la media..   

- *grLineLinearRegression*: &lt;Parámetro&gt; corresponde al número de la serie utilizada para realizar la regresión lineal.. 
	Atención: La regresión lineal opera con una sola serie. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>
La función [grType](../WDLang3/3042062.md) cancela todos los tipos de serie específicos fijados con la función **grSeriesTipo**: todas las series tendrán el mismo tipo.

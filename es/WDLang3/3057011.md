
## OLESave( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el objeto encontrado en una control OLE en formato binario. Se invoca el servidor OLE para grabar el objeto (el servidor no aparece).. El OLE control no está actualizado.

A continuación, el objeto que se ha guardado puede volver a cargarse con [OLELoad](../WDLang3/3057002.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OLECreateEmpty(OLE_OLE1, "PBrush")
// Process
// Save the object
OLESave(OLE_OLE1, "OBJECT1.OLE")
OLEPaste(OLE_OLE1, "PBrush")
// Save the object
OLESave(OLE_OLE1, "OBJECT1.OLE")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OLESave(<OLE control> , <File name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha grabado el objeto, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<OLE control>: (Nombre del control)`**

Nombre del control OLE associated con el objeto OLE. Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control al que pertenece el actual Event.

**`<File name>: (Cadena de caracteres)`**

Nombre y ruta (si es necesario) del fichero de destino. Se debe especificar la extensión. Si no se especifica la ruta, el archivo se guarda en el directorio actual..

<a name="NOTE0_1"></a>

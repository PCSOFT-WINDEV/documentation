
## MapPolygon (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **MapPolygon** se utiliza para definir las características de un Polygon que se dibujará en el mapa que se muestra en un control Mapa. Las características del polígono se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a polygon to the Map control
MyPolygon is MapPolygon
FOR EACH PointPosition OF arrPositions
	Add(MyPolygon.Point, PointPosition)
END
MyPolygon.LineColor = LightRed
MyPolygon.ActionClick = ClickProc
MapAddShape(MAP_MyMap, MyPolygon)

INTERNAL PROCEDURE ClickProc(p is MapPolygon)
	ToastDisplay(StringBuild("Click on the <%1> polygon.", p.Name))
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables MapPolygon
<a name="propiedades_especificas_las_variables_mappolygon_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades se pueden utilizar para manipular polígonos:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionClick | Nombre de Procedure | Nombre del WLanguage Procedure llamado cuando el usuario hace clic en el Polygon mostrado en el control Mapa. Este procedimiento tiene el siguiente formato: <br><br><pre><code>PROCEDURE <Procedure name>(<Polygon>)</code></pre><br>donde &lt;Polígono&gt; es una **MapPolygon** Variable correspondiente a la Polygon que el usuario ha pulsado. <br><br>Si no se especifica esta Property (o corresponde a una cadena vacía ("")), no se realizará ninguna acción al hacer clic.<br><br>Observación: se recomienda utilizar procedimientos globales. Si la Procedure es local, su nombre debe ir precedido del nombre completo del objeto al que pertenece la Procedure (ejemplo para una ventana local Procedure: "WIN_Window1.MyProcedure").<br><br> |
| BackgroundColor | [Color](../WDLang1/1000019661.md), entero, Constante entera | Color de fondo del polígono. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br><br><br>Observación: Para cambiar la opacidad del relleno, puedes escribir:<br><br><pre><code>MyMapPolygon.BackgroundColor.Opacity = 50</code></pre><br> |
| denominar | Cadena de caracteres | Nombre de la forma (aquí, nombre de la Polygon). Se asigna un nombre automáticamente cuando se crea la Variable. Puede ser modificado, pero tiene que ser único si se añade la forma a la control Mapa. Este nombre puede ser utilizado para borrar la Polygon con [MapDeleteShape](../WDLang3/1000025510.md). |
| description | Cadena de caracteres | description de la forma, utilizada para almacenar información adicional, por ejemplo. |
| Espesor | Integro | El grosor de la línea usada para la Polygon fuera de la Line. <br>El grosor es de set a 2 por Default. |
| Geodésico | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para indicar que el dibujo de la Polygon debe tener en cuenta la curvatura de la Tierra,<br><br>- <u><u><u><u>False</u></u></u></u> (Default) para trazar una recta Line.<br><br><br><br><br> |
| Line | Constante de tipo Integer | Estilo de línea utilizado entre los vértices de la Polygon: <br><br>- *LineSolid* (valor Default): Línea continua. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitcontinu.gif)<br><br>- *LineDotted*: lunares. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitpointille.gif)<br><br>- *LineDash*: Guiones. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitTiret.gif)<br><br>- *LineDotAndDash*: Puntos y guiones. <br>	![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=traitmixte.gif)<br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) No disponible. |
| LineColor | [Color](../WDLang1/1000019661.md), entero, Constante entera | Esquema de Polygon Color. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br> |
| Punto | Array de [geoPosition](../WDLang3/1000019191.md) | Puntos correspondientes a los vértices de la Polygon. La Polygon se cerrará automáticamente, por lo que los puntos de inicio y final pueden ser diferentes. |
| ZOrder | Integro | El orden Z del círculo en relación con las otras formas de la control Mapa. Le permite manejar las superposiciones. <br><br>Una forma con un orden Z alto se dibujará sobre una forma con un orden Z más bajo. <br><br>Este Property es set a 0 por Default. <br><br> |

|   |   |
| --- | --- |
| [MapAddShape](../WDLang3/1000025509.md) | Añade una nueva forma (Polygon, Line, círculo) al mapa que se muestra en un control Mapa. |
| [MapDeleteShape](../WDLang3/1000025510.md) | Borra una forma dibujada en el mapa que se muestra en una control Mapa. |
| [MapModifyShape](../WDLang3/1000025511.md) | Modifica una forma dibujada en el mapa que se muestra en una control Mapa. |

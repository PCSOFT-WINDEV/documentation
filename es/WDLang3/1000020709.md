
## Constantes para la gestión de las citas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por la función |
| --- | --- | --- |
| **aptContent** | Filtro en las citas: Filtros en la cita description (contiene la búsqueda). | [AppointmentList](../WDLang3/1000020690.md) |
| **aptIdentifier** | Filtro en las citas: Filtros en los identificadores de Appointment (búsqueda de coincidencias exactas). | [AppointmentList](../WDLang3/1000020690.md) |
| **aptLocation** | Filtro en las citas: Filtros en la ubicación de Appointment (contiene la búsqueda). | [AppointmentList](../WDLang3/1000020690.md) |
| **aptTitle** | Filtro en las citas: Filtros en el título de Appointment (contiene la búsqueda). | [AppointmentList](../WDLang3/1000020690.md) |
| **reminderDefault** | El método que se utilizará para realizar la Reminder: El método Default. Este método depende de la configuración del dispositivo. | Tipo [Recordatorio](../WDLang3/1000020688.md) |
| **reminderEmail** | El método que se utilizará para realizar la Reminder: enviar una Email. Este tipo de Reminder sólo está disponible si la cuenta asociada al Appointment Calendar soporta este tipo de Reminder y si esta cuenta está configurada. | Tipo [Recordatorio](../WDLang3/1000020688.md) |
| **reminderNotification** | El método que se utilizará para realizar la Reminder: Envía un Notification. | Tipo [Recordatorio](../WDLang3/1000020688.md) |
| **reminderSMS** | El método que se utilizará para realizar la Reminder: enviar a SMS. Este tipo de Reminder sólo está disponible si la cuenta asociada al Appointment Calendar soporta este tipo de Reminder y si esta cuenta está configurada. | Tipo [Recordatorio](../WDLang3/1000020688.md) |

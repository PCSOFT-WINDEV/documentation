


## Funciones In-App
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones de WLanguage se utilizan para gestionar las compras dentro de la aplicación: 




|   |   |
| --- | --- |
| [inAppCheckSubscription](../WDLang3/1000022719.md) | Al iniciar la aplicación iOS, comprueba si se han renovado las suscripciones In-App. |
| [inAppConsumeProduct](../WDLang3/1000020880.md) | Se utiliza para consumir, en una aplicación, un producto que fue previamente adquirido por el usuario. |
| [inAppListProductInfo](../WDLang3/1000020872.md) | Recupera, de la tienda, información sobre los productos "In-App" associated con la aplicación. |
| [inAppListPurchase](../WDLang3/1000020888.md) | Devuelve la lista de productos de la aplicación comprados por el usuario y no consumidos. |
| [inAppPurchaseProduct](../WDLang3/1000020873.md) | Envía una solicitud de compra de un producto "In-App" associated con la solicitud. |
| [inAppRestorePurchases](../WDLang3/1000020879.md) | Restaura las compras de productos no consumibles realizadas previamente por el usuario en una aplicación iOS. |




## grWaterfallCategoryType (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el tipo de categoría en una tabla de cascada. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// TO 2017
grAddData(CHART_Waterfall, 1, 1, 298000)
grCategoryLabel(CHART_Waterfall, 1, "TO 2017")

// Add the profits
grAddData(CHART_Waterfall, 1, 2, 420000)
grCategoryLabel(CHART_Waterfall, 2, "Profits" + CR + "products")
grAddData(CHART_Waterfall, 1, 3, 210000)
grCategoryLabel(CHART_Waterfall, 3, "Profits" + CR + "services")

// Costs
grAddData(CHART_Waterfall, 1, 4, -170000)
grCategoryLabel(CHART_Waterfall, 4, "Fixed costs")
grAddData(CHART_Waterfall, 1, 5 -140000)
grCategoryLabel(CHART_Waterfall, 5, "Variable costs")

// TO 2018
grAddData(CHART_Waterfall, 1, 6, 0)
grCategoryLabel(CHART_Waterfall, 6, "TO 2018")
grWaterfallCategoryType(CHART_Waterfall, 6, grSubTotal)

// Cumulated TO
grAddData(CHART_Waterfall, 1, 7, 0)
grCategoryLabel(CHART_Waterfall, 7, "Cumulated TO")
grWaterfallCategoryType(CHART_Waterfall, 7, grTotal)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grWaterfallCategoryType(<Chart name> , <Category> , <Type of category>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico que se encuentra en el editor de ventanas, editor de reportes o en el editor de páginas.




**`<Category>: (Integro)`**

Número de la categoría a utilizar. 

**`<Type of category>: (Constante de tipo Integer)`**

Tipo de categoría: 



|   |   |
| --- | --- |
| *grDefault* | Valores positivos o negativos del gráfico. |
| *grSubTotal* | Valores subtotales.<br><br>Si la categoría corresponde a un subtotal, se ignora el valor almacenado para esta categoría.. Para esta categoría, la tabla de cascada muestra una barra correspondiente a la suma de todas las categorías anteriores del último total o subtotal. |
| *grTotal* | Valores totales.<br><br>Si la categoría corresponde a un total, se ignora el valor almacenado para esta categoría.. Para esta categoría, la carta de cascada muestra una barra correspondiente a la suma de todas las categorías anteriores desde el principio de la carta. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Caso especial: la primera categoría**
	La primera categoría es siempre una categoría "total", pero se muestra el valor especificado.: es el valor inicial. 

- Esta función se aplica sólo a las cartas de cascada.

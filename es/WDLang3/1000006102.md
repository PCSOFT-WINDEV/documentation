


## ContactSearch( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un contacto en una libreta de Lotus Notes o Outlook Address.
**Atención**: Esta función **no se puede utilizar** para encontrar un Contact en el libro Address de una mensajería de Outlook Express.

**ContactSearch** no set la posición en el Contact deseado. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ....
		"email/julia.nsf", "C:\Lotus Notes\julia.id")
...
// Find the contacts for the company named "Tames"
IF ContactSearch(SessionID, "Tames", mContactCompany) = True THEN
	// Add the contacts found into a List Box control
	WHILE mContact.Out = False
		ListAdd(LIST_ContactList, mContact.Name + TAB + mContact.FirstName)
		ContactNext(SessionID)
	END
ELSE
	Info("No contact was found")
END
```

## Sintaxis

### Encontrar una Contact en la mensajería de Lotus Notes o Outlook

`<Result> = ContactSearch(<Session identifier> , <Sought value 1> , <Type of value 1> [, <Sought value 2> [, <Type of value 2>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si uno o más contactos corresponden a la búsqueda,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

**`<Sought value 1>: (Tipo de valor buscado)`**

Primer valor a encontrar.

**`<Type of value 1>: (Constante de tipo Integer)`**

Tipo de primer valor a buscar:



|   |   |
| --- | --- |
| *mContactCompany* | Búsqueda realizada en las empresas de contactos |
| *mContactEmail* | Búsqueda realizada en los emails de los contactos |
| *mContactFirstName* | Búsqueda realizada en los nombres de los contactos |
| *mContactLastName* | Búsqueda realizada en los apellidos de los contactos |



**`<Sought value 2>: (Tipo de valor buscado, opcional)`**

Segundo valor a encontrar. Si este parámetro y &lt;Tipo de valor 2&gt; no se especifican, se busca la Contact según &lt;Valor buscado 1&gt;.

**`<Type of value 2>: (Constante opcional de tipo Integer)`**

Tipo del segundo valor a buscar:



|   |   |
| --- | --- |
| *mContactCompany* | Búsqueda realizada en las empresas de contactos |
| *mContactEmail* | Búsqueda realizada en los emails de los contactos |
| *mContactFirstName* | Búsqueda realizada en los nombres de los contactos |
| *mContactLastName* | Búsqueda realizada en los apellidos de los contactos |

Si este parámetro y el parámetro &lt;Valor buscado 2&gt; no se especifican, el contacto solo se busca según el &lt;Valor buscado 1&gt;.

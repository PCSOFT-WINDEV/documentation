<a name="Use"></a>
<a name="description"></a>
Muestra un cuadro de diálogo en cada equipo cliente conectado al servidor.. El servidor es un servidor FTP (File Transfer Protocol) o un servidor RPC (Remote Procedure Call) alimentado por WINDEV.

**Observación**: Esta función se valida cuando todos los cuadros de diálogo se cierran en todos los equipos cliente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003056017&name=netclientmessagebox_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
ServerID is int = NetStartServer()
// Display a YesNo dialog box on the client computer
IF NetClientMessageBox(ServerID, "Do you want to continue?", ...
	"", netYesNo) = 1 THEN
	...
END
```

## Sintaxis

`<Result> = NetClientMessageBox(<Server identifier> , <Message> , <Window title> , <Window type>)`
---

**`<Result>: (Integro)`**

Número del botón pulsado en el cuadro de mensaje por el último ordenador cliente.

**`<Server identifier>: (Integro)`**

Identificador del servidor (devuelto por [NetStartServer](../WDLang3/3056013.md)).

**`<Message>: (Cadena de caracteres ANSI)`**

Texto que debe visualizarse en la ventana de diálogo.

**`<Window title>: (Cadena de caracteres ANSI)`**

Texto mostrado en la barra de título de la ventana.

**`<Window type>: (Constante de tipo Integer)`**

Tipo de cuadro de diálogo que desea visualizar.


|   |   |
| --- | --- |
| *netConfirm* | Buzón de mensajes con los botones Sí, No y Cancel. |
| *netInfo* | Cuadro de mensaje de información (sólo botón OK). |
| *netOkCancel* | Buzón de mensajes con los botones OK y Cancel. |
| *netRetryCancel* | Buzón de mensajes con los botones Retry y Cancel. |
| *netRetryIgnore* | Buzón de mensajes con los botones Cancel, Reintentar e Ignorar. |
| *netYesNo* | Cuadro de mensajes con botones Yes y No. |

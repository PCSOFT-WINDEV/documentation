


## Funciones de gestión de álbumes de fotos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para administrar un álbum de fotos:



|   |   |
| --- | --- |
| [AlbumPicker](../WDLang3/1000020186.md) | Recupera una foto, imagen o video almacenado en el álbum de fotos del dispositivo móvil. |
| [AlbumSave](../WDLang3/1000020191.md) | Guarda un Image, una foto o un vídeo en el álbum de fotos del dispositivo móvil. |
| [PhotoRunApp](../WDLang3/1000021601.md) | Inicia la aplicación de cámara nativa del dispositivo para tomar una foto. |

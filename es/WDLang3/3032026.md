


## EmailMsgError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el mensaje correspondiente al identificador de error.

Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Se recomienda utilizar [ErrorInfo](../WDLang1/3013008.md) en lugar de **EmailMsgError**.

Todas las funciones para la gestión de Email devuelven un resultado que indica si la función fue realizada o no. Si se produce un error, estas funciones devuelven 0 o <u><u><u>False</u></u></u>. El Variable de errores de correo electrónico contiene el identificador del error que se produjo.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032026&name=emailmsgerror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Sends the email
IF EmailSendMessage(UserName) = False THEN
	Error(EmailMsgError(Email.Error))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = EmailMsgError(<Error number>)`
---

**`<Result>: (Cadena de caracteres)`**

Mensaje de error completo.

**`<Error number>: (Integro)`**

Identificador del error. El Email.Error Variable contiene el identificador del último error que se produjo al utilizar las funciones para la gestión de Email.




## &lt;Chart&gt;.ComparativeSeriesPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija la posición (izquierda o derecha) de una serie en un gráfico de tipo Histograma de comparación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_NoName1.ComparativeSeriesPosition(1, grAtRight)
CHART_NoName1.ComparativeSeriesPosition(2, grAtLeft)

CHART_NoName1.SeriesColor(1, GetColor())
CHART_NoName1.SeriesColor(2, GetColor())

rRight is real
rLeft is real

nRadius is int = 250

FOR n = 1 TO 360
	rRight = nRadius + nRadius * Cos(n)
	rLeft = nRadius + nRadius * Sin(n + 180)
	CHART_NoName1.AddData(1, n, rRight)
	CHART_NoName1.AddData(2, n, rLeft)
	CHART_NoName1.CategoryLabel(n, n)
END

CHART_NoName1.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.ComparativeSeriesPosition(<Series number> , <Position>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre del control Gráfico a utilizar en el editor de ventanas.

**`<Series number>: (Integro)`**

Número de la serie a modificar.

**`<Position>: (Integro)`**

Posición de la serie manipulada en el gráfico:


|   |   |
| --- | --- |
| *grAtLeft* | Serie mostrada a la izquierda del gráfico. |
| *grAtRight* | Serie mostrada a la derecha del gráfico. |

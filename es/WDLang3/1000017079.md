
## SNMPOIDDescription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la description de un OID.

**Observación**: Esta función no puede utilizarse mientras no se haya cargado el archivo MIB que describe la correspondencia entre los OID en formato textual y numérico ([SNMPLoadMIB](../WDLang3/1000017074.md)).  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MIBLoaded is boolean
PathMIBs is string = CompleteDir(fExeDir()) + "mibs\"
Mib is string
Mib = "RFC1382-MIB.mib" // MIB of X25 protocol
MyDescription is string
MIBLoaded = SNMPLoadMIB(PathMIBs + Mib)
IF MIBLoaded = False THEN 
	Error("Error MIB: " + ErrorInfo())
ELSE
	MyDescription = SNMPOIDDescription("
			dod.internet.mgmt.mib-2.
			transmission.x25.x25AdmnInterruptTimer
			")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SNMPOIDDescription(<OID to process>)`
---

**`<Result>: (Cadena de caracteres)`**

description del OID. Este description está almacenado en el archivo MIB.

**`<OID to process>: (Cadena de caracteres)`**

Cadena que contiene el OID en formato de texto o numérico.

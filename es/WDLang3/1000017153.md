
## OBEXConnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para conectarse a un dispositivo compatible con el protocolo OBEX.. 

**Recordatorio**: El protocolo OBEX es un protocolo para el intercambio de datos por infrarrojos y Bluetooth.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MacAddress is string
// Retrieve the MAC address of Bluetooth device
// The address is stored in a table column
MacAddress = TABLE_Devices.COL_MACAddr[TABLE_Devices]
// OBEX connection
EDT_OBEXID = OBEXConnect(obexBluetooth, MACAddress)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OBEXConnect(<Type of Device> [, <Device Address>])`
---

**`<Result>: (Integro)`**

Identificador de la conexión. Este identificador será utilizado por las funciones OBEX. Esta función no devuelve ningún error. De hecho, la conexión se establecerá realmente cuando se utilice la siguiente función OBEX.

**`<Type of Device>: (Constante de tipo Integer)`**

Tipo de dispositivo utilizado para intercambiar datos:


|   |   |
| --- | --- |
| *obexBluetooth* | Transporte de datos realizado por Bluetooth |
| *obexInfrared* | Transporte de datos realizado por infrarrojos |



**`<Device Address>: (Cadena de caracteres opcional)`**

Address del dispositivo utilizado (sólo para un dispositivo Bluetooth): este Address corresponde al MAC Address (devuelto por [BTInfoDevice](../WDLang3/1000017133.md) o [BTListDevice](../WDLang3/1000017132.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo las pilas de Microsoft bluetooth son soportadas por esta función. Para get la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)


## Constantes de URI
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes constantes son utilizadas por las funciones URIxxx:

| Constante | Detalles | Función |
| --- | --- | --- |
| **uriAdd** | El contenido del archivo o buffer se añade al final del contenido inicial de la Resource. | [FileToURI](../WDLang3/1410087413.md)<br>[BufferToURI](../WDLang3/1410087415.md) |
| **uriAudio** | Tipos MIME soportados: Permite todo tipo de archivos de audio. | [URISelect](../WDLang3/1000024026.md) |
| **uriCreate** | <br><br>- Abre el selector en modo "Creación" para crear una nueva Document. <br><br>- El contenido inicial de la Resource se sustituye por el contenido del archivo o buffer.<br><br><br> | <br><br>- [URISelect](../WDLang3/1000024026.md)<br><br>- [FileToURI](../WDLang3/1410087413.md)<br>	[BufferToURI](../WDLang3/1410087415.md)<br><br><br> |
| **uriImage** | Tipos MIME soportados: Permite todo tipo de archivos de Image. | [URISelect](../WDLang3/1000024026.md) |
| **uriInfoName** | Información solicitada: Nombre de la Resource (sin extensión). | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoSize** | Información solicitada: Tamaño de la Resource (en bytes). | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoConversionType** | Información solicitada: Tipos de MIME (y las correspondientes extensiones de archivo) en los que se puede convertir el Resource al ser exportado. | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriInfoOriginType** | Información solicitada: Tipo de MIME original del Resource y la correspondiente extensión de archivo. | [URIGetInfo](../WDLang3/1000024037.md) |
| **uriOpen** | Abre el selector en modo "Abrir" para abrir un Document existente. | [URISelect](../WDLang3/1000024026.md) |
| **uriVideo** | Tipos MIME soportados: Permite todo tipo de archivos de Image. | [URISelect](../WDLang3/1000024026.md) |

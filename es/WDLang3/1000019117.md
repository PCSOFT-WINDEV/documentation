
## SaaSAdminDeleteClient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una cuenta de cliente SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is saasClient 
S = SaaSAdminFindClient("Hybrid")

IF SaaSAdminDeleteClient(S) = True THEN
	Info("Client deleted")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminDeleteClient(<Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Client account>: (saasClient Variable o cadena de caracteres)`**

Nombre de la [saasClient](../WDLang3/1000019072.md) Variable o cadena de caracteres correspondiente a la cuenta de cliente que se va a eliminar de la base de datos SaaS.

Si este parámetro es una cadena de caracteres, ésta representa el nombre de la cuenta del cliente o su [Referencia externa](../WDLang3/1000019072.md) Property.

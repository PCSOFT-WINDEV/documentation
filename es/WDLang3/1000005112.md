
## NotesListItem( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de artículos para las actuales Notas Document en Lotus Notes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
...
// Read the first document 
NotesDocumentFirst(SessionID)
// Retrieve the Lotus Notes items found in this document 
ListLotusNotesItem is string
ALotusNotesItem is string
ListLotusNotesItem = NotesListItem(SessionID)
// For each Lotus Notes item
FOR EACH STRING ALotusNotesItem OF ListLotusNotesItem SEPARATED BY CR
	// Add the name of the Lotus Notes item and its value into a List Box control
	ListAdd(LIST_LotusNotesItem, ALotusNotesItem + CR +  ...
		NotesItem(SessionID, ALotusNotesItem))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Devolviendo el nombre de los artículos encontrados en las actuales Notas Document

`<Result> = NotesListItem(<Session identifier>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de los elementos que se encuentran en las actuales Notas Document, separados por caracteres CR (Carriage Return). Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).  


<a name="SYNTAX2"></a>

### Devolviendo el nombre y el tipo de los artículos encontrados en las actuales Notas Document

`<Result> = NotesListItem(<Session identifier> , <notesLstType>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre y tipo de los elementos que se encuentran en las actuales Notas Document, en el siguiente formato:

```txt
<Name of Lotus Notes Item 1> + TAB + <Type of Lotus Notes Item 1> + CR + 
<Name of Lotus Notes Item 2> + TAB + <Type of Lotus Notes Item 2> + CR +...
```

Dónde:

- &lt;Nombre del elemento Lotus Notes X&gt; corresponde al nombre del elemento Lotus Notes.

- &lt;Tipo de elemento Lotus Notes X&gt; corresponde al tipo del elemento Lotus Notes. Este parámetro puede corresponder a:

	- *notesTypeDateTime*: DateTime Lotus Notes item

	- *notesTypeUnknown*: Tipo de artículo de Lotus Notes no admitido

	- *notesTypeNumber*: Número Artículo de Lotus Notes

	- *notesTypeText*: Texto del artículo de Lotus Notes

	- *notesTypeRichText*: El artículo de Rich Text Lotus Notes (que contiene un formato y posibles anexos)







Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<notesLstType>: (Constante)`**

*notesLstType*: Constante utilizada para devolver el tipo de los artículos encontrados en las actuales Notas Document.

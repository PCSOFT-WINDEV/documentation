


## EmailBuildSource (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera el código fuente del mensaje de correo electrónico que se enviará a partir de los Variable s de la estructura [Email](../WDLang3/3032029.md) o de un Variable de tipo [Email](../WDLang3/1000018713.md). 

El código fuente se genera en el **Email.Source** Variable o en el **Source** Property del [Email](../WDLang3/1000018713.md) Variable. Entonces, este código fuente puede ser guardado en un archivo.eml por ejemplo.

**Observación**: Esta función es useful para modificar un elemento encontrado en un Email saliente. La Email puede ser enviada por [EmailSend](../WDLang3/3032003.md). Para obtener más información, consulte [Escribir un Email personalizado](../WDLang3/3032002.md).

<a name="sample_code"></a>

## Ejemplo



### Ejemplo con un correo electrónico Variable
<a name="ejemplo_con_correo_electronico_variable_ELTPARAGRAPHE000107"></a>


```wl
// Initialize the variable
MyMessage is Email

Add(MyMessage.Recipient, "support_test@pcsoft_testfr")
MyMessage.Sender = "customer@company.fr"
MyMessage.Message = "Auto Email Test"

// Create the .eml file
IF EmailBuildSource(MyMessage) = True THEN
	fSaveText("c:\MyEmails\Test.eml", MyMessage.Source)
END
```
<a name="Example2"></a>


### Ejemplo con la estructura Email
<a name="ejemplo_con_estructura_email_ELTPARAGRAPHE000119"></a>


```wl
// Initialize the email structure
Email.NbRecipient = 1
Email.Recipient = "support_test@pcsoft_test.fr"
Email.Sender = "customer@company.fr"
Email.Message = "Auto Email Test"

// Create the .eml file
IF EmailBuildSource() = True THEN
	fSaveText("c:\MyEmails\Test.eml", Email.Source)
END
```

## Sintaxis

`<Result> = EmailBuildSource([<Email> [, <Advanced options>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se generó el código fuente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. 
	Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. El código generado puede leerse en el Email.Source Variable de thread actual.




**`<Email>: (Correo electrónico opcional Variable)`**

Nombre de la variable de tipo [Email](../WDLang3/1000018713.md) que se utilizará.
Si no se especifica este parámetro, se utiliza la estructura [Email](../WDLang3/3032029.md).

**`<Advanced options>: (Constante opcional de tipo Integer)`**

Combinación de las siguientes constantes:


|   |   |
| --- | --- |
| *emailOptionEncodeHeader* | Codifica las cabeceras de los mensajes en Quoted-Printable si es necesario.. <br>El encabezado del mensaje no está codificado por Default. Esta constante evita que los correos electrónicos con caracteres acentuados erróneamente codificados. |
| *emailOptionHeaderBcc* | Se utiliza para añadir los destinatarios no oficiales en el código fuente de la cabecera de Email. |

## Observaciones
Para rellenar la estructura [Email](../WDLang3/3032029.md) o una variable de tipo [Email](../WDLang3/1000018713.md) utilizando el contenido de su **Fuente** Property, utilice [EmailImportSource](../WDLang3/1000017226.md).

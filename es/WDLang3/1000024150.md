


## &lt;SurfaceDeleteMarkCaption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los títulos de las marcas que se encuentran en un gráfico de Superficie. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_MyChart.SurfaceDeleteMarkCaption(grXCoordinate, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceDeleteMarkCaption([<Mark axis> [, <Value>]])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).

**`<Mark axis>: (Constante de tipo Integer)`**

Tipo de eje del que se desea borrar las marcas:



|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |
| *grZCoordinate* | Eje Z. |


Si no se especifica este parámetro, se borran las marcas de todos los ejes..

**`<Value>: (Real)`**

Valor de la marca para la que debe suprimirse la leyenda. 

Si no se especifica este parámetro, se borrarán todos los títulos de las marcas..

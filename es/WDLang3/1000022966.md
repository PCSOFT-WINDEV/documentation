


## grScatter3DDeleteAltitudeColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los colores de altitud especificados por [grScatter3DAltitudeColor](../WDLang3/1000022965.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grScatter3DDeleteAltitudeColor(CHART_My3DScatter, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grScatter3DDeleteAltitudeColor(<Chart name> , <Series number> [, <Altitude>])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas).




**`<Series number>: (Integro)`**

Número de la serie a utilizar. 

**`<Altitude>: (Real opcional)`**

La altitud de Color que será eliminada. Si no se especifica este parámetro, se borran todos los colores de altitud.: el gráfico está usando la serie Color.




## grSuperficieAltitudColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define la Color utilizada para la altitud (eje Z) en un gráfico de superficie.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000020630&name=grsuperficiealtitudcolor_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
grSurfaceAltitudeColor(CHART_Surface, 100, LightRed)
grSurfaceAltitudeColor(CHART_Surface, 0, PastelBlue)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSurfaceAltitudeColor(<Chart name> , <Altitude> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Altitude>: (Real)`**

Altitud para la cual se aplicará el Color. 

**`<Color>: (Entero o constante de tipo Integer)`**

Color para la altitud especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si se especifican colores de altitud, las facetas encontradas a esta altitud se colorearán en consecuencia. 

Para las facetas que se encuentran entre dos colores, se aplicará un Gradient lineal.

Para las facetas fuera de los límites de Color, se aplicará el Color de la serie.

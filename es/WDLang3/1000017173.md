


## Funciones Bluetooth
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los dispositivos Bluetooth:



|   |   |
| --- | --- |
| [BTAcceptConnection](../WDLang3/1000017128.md) | Permite saber si una radio Bluetooth acepta (o no) las peticiones de conexión procedentes de los dispositivos. |
| [BTActivate](../WDLang3/1000019359.md) | Habilita o deshabilita la gestión de Bluetooth en el dispositivo. |
| [BTChangeConnectivity](../WDLang3/1000017130.md) | Permite configurar una radio Bluetooth para que acepte (o no) las peticiones de conexión procedentes de los dispositivos. |
| [BTChangeVisibility](../WDLang3/1000017129.md) | Cambia la visibilidad de una radio Bluetooth. |
| [BTConnectDevice](../WDLang3/1000017146.md) | Le permite ser autenticado hacia un dispositivo Bluetooth. |
| [BTDisableService](../WDLang3/1000017143.md) | Desactiva un servicio en un dispositivo Bluetooth. |
| [BTDisconnectDevice](../WDLang3/1000017147.md) | Cancela la autenticación junto a un dispositivo Bluetooth. |
| [BTEnableService](../WDLang3/1000017142.md) | Activa un servicio proporcionado por un dispositivo Bluetooth. |
| [BTInfoDevice](../WDLang3/1000017133.md) | Devuelve información específica sobre un dispositivo Bluetooth. |
| [BTInfoRadio](../WDLang3/1000017126.md) | Devuelve información específica sobre una radio Bluetooth conectada a el equipo. |
| [BTIsVisible](../WDLang3/1000017127.md) | Se utiliza para averiguar si una radio Bluetooth es Visible. |
| [BTListDevice](../WDLang3/1000017132.md) | Devuelve la lista de dispositivos Bluetooth accesibles. |
| [BTListRadio](../WDLang3/1000017125.md) | Devuelve la lista de radios Bluetooth conectadas a el equipo. |
| [BTListService](../WDLang3/1000017141.md) | Devuelve la lista de servicios proporcionados por un dispositivo Bluetooth. |
| [BTOpenDeviceProperties](../WDLang3/1000017134.md) | Abre la ventana de propiedades de un dispositivo Bluetooth. |
| [BTSelectDevice](../WDLang3/1000017135.md) | Abre una ventana del sistema para seleccionar un dispositivo Bluetooth. |
| [BTStack](../WDLang3/1000017229.md) | Devuelve la pila Bluetooth utilizada actualmente. |
| [BTStatus](../WDLang3/1000019360.md) | Devuelve el estado actual de activación de Bluetooth en el dispositivo o pide que se le notifique cuando cambie el estado de activación. |





**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Sólo la pila de Microsoft y la pila de Bluesoleil son compatibles.

- No es necesario firmar la aplicación para utilizar las funciones de Bluetooth.




Las tomas Bluetooth también se pueden gestionar a través de las siguientes funciones: 



|   |   |
| --- | --- |
| [SocketConnectBluetooth](../WDLang3/1000017145.md) | Conecta una computadora cliente a un enchufe Bluetooth específico. |
| [SocketCreateBluetooth](../WDLang3/1000017167.md) | Crea un socket utilizando un dispositivo Bluetooth. |


## tapiCallIsWaiting( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para saber si la llamada está en espera.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Is the current call on hold?
IF tapiCallIsWaiting() = True THEN
	// Display a message
	Message("Call put on hold")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiCallIsWaiting([<Call identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el número al que se llama o la llamada entrante está en espera,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). Si no se especifica este parámetro, se gestiona la llamada saliente o entrante actual..

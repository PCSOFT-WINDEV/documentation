


## grWaterfallColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la Color para un tipo de categoría en un gráfico de cascada. 

**Observación**: Si no se utiliza esta función, el gráfico se muestra con los colores seleccionados por Default. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grWaterfallColor(CHART_Waterfall, grTotal, RGB(83,129,179))
grWaterfallColor(CHART_Waterfall, grSubTotal, RGB(103,89,159))
grWaterfallColor(CHART_Waterfall, grPositive, RGB(115,179,83))
grWaterfallColor(CHART_Waterfall, grNegative, RGB(179,84,92))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grWaterfallColor(<Chart name> , <Type of category> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre puede corresponder a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Type of category>: (Constante de tipo Integer)`**

Tipo de la categoría para la cual la Color debe ser definida: 



|   |   |
| --- | --- |
| *grNegative* | Valores negativos del gráfico. |
| *grPositive* | Valores positivos del gráfico. |
| *grSubTotal* | Valores subtotales del gráfico. |
| *grTotal* | Valores totales del gráfico. |



**`<Color>: (Entero o constante de tipo Integer)`**

Color para el tipo de categoría especificado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  

- la constante *ColorDefault*: 

	- Si se usa la constante *DefaultColor* para los tipos de categoría *grPositive* y *grNegative*, se usarán los colores de la serie Default. 

	- Si se utiliza la constante *DefaultColor* para el tipo de categoría *grSubTotal* y colores específicos para los tipos de categoría *grPositive* y *grNegative*, se utilizan estos colores específicos para el subtotal. 









<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los colores definidos para las categorías con la función [grCategoryColor](../WDLang3/1000020928.md) tienen prioridad sobre los colores definidos con la función **grWaterfallColor**. 

- Esta función se aplica sólo a las Cartas de cascada.




## zipSelectFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona los archivos de un archivo. Los archivos seleccionados serán procesados por las funciones de gestión de archivos WLanguage.. 

Esto hace posible seleccionar sólo los archivos ".doc" de un archivo, por ejemplo. En este caso, todas las funciones de WLanguage utilizadas para manipular todo el archivo (por ejemplo, [zipExtractAll](../WDLang3/3082011.md)) sólo afectarán a los archivos seleccionados (archivos ".doc" en nuestro ejemplo).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Archive is zipArchive
// Open an archive
IF zipOpen(Archive, "C:\Archives\Archive.zip") = 0 THEN
	// Extract documents only
	sPreviousSelection is string = zipSelectFile(Archive, "*.doc")
	// Extract all documents from the archive
	zipExtractAll(Archive, zipDrive)
	// Restore previous selection
	zipSelectFile(Archive, sPreviousSelection)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = zipSelectFile(<Archive> [, <Selection>])`
---

**`<Result>: (Cadena de caracteres)`**

Selección previa aplicada al archivo.

**`<Archive>: (Cadena de caracteres o zipArchive Variable)`**

Nombre del archivo que se va a utilizar.
Este nombre puede corresponder a:

- un nombre de archivo definido con [zipOpen](../WDLang3/3082044.md) o [zipCreate](../WDLang3/3082003.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [zipArchive](../WDLang3/1000018679.md).




**`<Selection>: (Cadena de caracteres opcional)`**

Nombre genérico de los archivos a seleccionar. 
Los caracteres genéricos (\* y?) están permitidos. 
Si no se especifica este parámetro o si set es una cadena vacía (""), la selección será eliminada y todos los archivos serán procesados.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Principio
<a name="principio_ELTPARAGRAPHE000085"></a>

Después de llamar a **zipSelectFile**, las siguientes funciones de gestión de archivos que manipular todos los ficheros del archivo tendrán en cuenta la selección: estas funciones sólo manipularán los archivos seleccionados. 

Para volver a manipular todos los ficheros del archivo, basta con llamar a **zipSelectFile** sin especificar el parámetro &lt;Selección&gt;..

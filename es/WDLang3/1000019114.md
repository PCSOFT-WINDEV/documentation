
## SaaSAdminFindClient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un cliente SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las cuentas de los clientes. Un gestor de cuenta de cliente sólo tendrá acceso a su propia cuenta de cliente. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Finds information about the "Hybrid" client
S is saasClient dynamic 
S = SaaSAdminFindClient("Hybrid")
IF S = Null THEN
	Error("Customer not found")
ELSE
	IF SaaSAdminDeleteClient(S) = True THEN
		Info("Client deleted")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminFindClient(<Sought client>)`
---

**`<Result>: (Variable de tipo saasClient)`**

variable [saasClient](../WDLang3/1000019072.md) que contiene el description de la cuenta del cliente si la búsqueda fue exitosa.

La Variable devuelta está vacía si el cliente no fue encontrado.

**`<Sought client>: (Cadena de caracteres)`**

Información utilizada para identificar al cliente: 

- Nombre del cliente.

- Nombre de la empresa.

- Referencia externa.


## Constantes para la gestión de las funciones Wi-Fi
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | Detalles | Utilizado por las funciones |
| --- | --- | --- |
| *wifiEnabled* | El Wi-Fi está habilitado en el dispositivo. | [WiFiActivate](../WDLang3/1000019350.md), [WiFiStatus](../WDLang3/1000019354.md) |
| **wifiActivationInProgress** | El Wi-Fi está siendo habilitado en el dispositivo. | [WiFiActivate](../WDLang3/1000019350.md), [WiFiStatus](../WDLang3/1000019354.md) |
| **wifiIPAddress** | IP Address associated con la conexión. | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiMacAddress** | Wi-Fi Mac Address del dispositivo. | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiBSSID** | BSSID (Basic Service set Identifier) del punto de acceso de la conexión. | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiDeactivationInProgress** | El Wi-Fi está siendo desactivado en el dispositivo. | [WiFiActivate](../WDLang3/1000019350.md), [WiFiStatus](../WDLang3/1000019354.md) |
| **wifiDisabled** | El Wi-Fi está desactivado en el dispositivo. | [WiFiActivate](../WDLang3/1000019350.md), [WiFiStatus](../WDLang3/1000019354.md) |
| **wifiError** | No se puede recuperar el estado de activación de Wi-Fi. | [WiFiActivate](../WDLang3/1000019350.md), [WiFiStatus](../WDLang3/1000019354.md) |
| **wifiIDNetwork** | Identificador del dispositivo al que está conectada la red, cadena vacía ("") si el dispositivo está conectado a una red no identificada. | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiSecurityNone** | No se utiliza ningún tipo de seguridad. | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md) |
| **wifiSecurityIEEE8021X** | Seguridad que utiliza el estándar IEEE 802.1x. | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md), [WiFiListNetwork](../WDLang3/1000019356.md) |
| **wifiSecurityWEP** | Seguridad basada en una clave WEP (Wired Equivalent Privacy) | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md), [WiFiListNetwork](../WDLang3/1000019356.md) |
| **wifiSecurityWPA** | Seguridad que utiliza el mecanismo WPA (Wi-Fi Protected Access). | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md), [WiFiListNetwork](../WDLang3/1000019356.md) |
| **wifiSecurityWPA2** | Seguridad que utiliza el mecanismo WPA2 (Wi-Fi Protected Access). | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md), [WiFiListNetwork](../WDLang3/1000019356.md) |
| **wifiSecurityWPA_EAP** | Seguridad que utiliza el mecanismo WPA (Wi-Fi Protected Access) y el mecanismo EAP (Extensible Authentication Protocol). | [WiFiAddNetwork](../WDLang3/1000019351.md), [WiFiDetectAccessPoint](../WDLang3/1000019353.md), [WiFiListNetwork](../WDLang3/1000019356.md) |
| **wifiSSID** | SSID (Service set Identifier) de la red Wi-Fi a la que está conectado el dispositivo. | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiAll** | Información completa sobre la conexión (corresponde a *wifiSSID* + *wifiBSSID* + *wifiIPAddress* + *wifiMacAddress* + *wifiSpeed* + *wifiIDNetwork*). | [WiFiConnectionInfo](../WDLang3/1000019355.md) |
| **wifiSpeed** | Velocidad del enlace Wi-Fi (expresada en Mbps). | [WiFiConnectionInfo](../WDLang3/1000019355.md) |

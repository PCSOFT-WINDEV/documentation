
## Procedimiento WLanguage llamado por la función AndroidActivityResultProcedure



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento WLanguage ("callback") llamado por la función [AndroidActivityResultProcedure](../WDLang3/1410088881.md) cuando se obtiene el resultado de una actividad de terceros.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AndroidActivityResultProcedure((resultCode, requestCode, Uri, extras)=>{
	IF resultCode = -1 _AND_ requestCode = 1024 THEN
		FOR EACH ELEMENT value OF extras 
			...
		END
	END
})
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AndroidActivityResultProcedure_Callback(<Result> , <Request code> , <Uri> , <Extras>)`
---

**`<Result>: (Entero)`**

Resultado de la operación realizada por la actividad que acaba de finalizar.

**`<Request code>: (Entero)`**

Código de solicitud pasado como parámetro al método **startActivityForResult** llamado para iniciar la actividad.

**`<Uri>: (Variable de tipo URI)`**

Nombre de la variable de tipo [URI](../WDLang3/1000024027.md) que corresponde al URI de los datos procesados por la actividad.

**`<Extras>: (Array asociativo de tipo Variant)`**

Pares clave-valor recibidos como resultado de la actividad. Solo se admiten los siguientes tipos de valores: Integer, String, Boolean, Buffer, Real, Array of integer, boolean, string o real. 
Los pares clave-valor de un tipo no admitido serán ignorados.

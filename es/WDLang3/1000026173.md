


## EmailExportHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el código HTML de un correo electrónico para mostrarlo en una control HTML (control HTML , control HTML Display, etc.). Las imágenes se extraen al disco. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
oEmailReceived is Email
//...
// Read the inbox to fill oEmailReceived
//...
HTMEDT_Email.WorkingDirectory = "C:\Temp\mail"
HTMEDT_Email = EmailExportHTML(oEmailReceived, HTMEDT_Email.WorkingDirectory)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EmailExportHTML(<Email> , <Directory>)`
---

**`<Result>: (Cadena de caracteres)`**

Código HTML que puede asignarse a una control HTML (control HTML, control HTML Display, etc.).

**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable que corresponde al Email con el código HTML a recuperar. 

**`<Directory>: (Cadena de caracteres)`**

Ruta del directorio en el que se guardarán las imágenes. 
Observación: Si la Image ya existe, se añade un sufijo al nombre de la Image almacenada en el disco (_1, _2, _n.). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las imágenes extraídas se referencian en el código HTML resultante mediante una etiqueta "./myimage.png".. 

- Las referencias de imágenes de tipo  se sustituyen por .

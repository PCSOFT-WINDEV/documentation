
## BTSelectDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una ventana del sistema para seleccionar un dispositivo Bluetooth.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDevices is string
MyDevices = BTSelectDevice(btStatusAll, "Select the device to connect")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTSelectDevice([<Filter Condition> [, <Text of Picker>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Identificador de los dispositivos Bluetooth seleccionados. Si se seleccionan varios dispositivos, los identificadores se separan mediante una tabulación (TAB).

- Cadena vacía (""") en los siguientes casos: 

	- el recogedor fue cancelado.

	- no se ha seleccionado ningún dispositivo.

	- se ha producido un error. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.







**`<Filter Condition>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Estado de los dispositivos a seleccionar:



|   |   |
| --- | --- |
| *btStatusAll*<br>(Valor predeterminado) | Propone todos los dispositivos. |
| *btStatusAuthenticated* | Propone los dispositivos autentificados. |
| *btStatusRemembered* | Propone los dispositivos recordados. |
| *btStatusUnknown* | Propone los dispositivos desconocidos. |



**`<Text of Picker>: (Cadena de caracteres opcional)`**

Texto a mostrar en la ventana de selección.

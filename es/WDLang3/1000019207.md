


## GPSStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el estado de activación del proveedor de geolocalización o pide ser notificado cuando cambia el estado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks the status of the GPS provider
IF GPSStatus() = gpsDisabled THEN
	Info("Geolocation is not enabled.")
END
```


<a name="Example2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Branches a notification procedure if the status of the GPS is modified
GPSStatus(ProcGPSStatus)
```

```wl
// Procedure called whenever the status of the GPS is modified
PROCEDURE ProcGPSStatus(GPS_Status is int)

IF GPS_Status = gpsUnavailable THEN
	Info("Geolocation not available.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperar el estado de activación del proveedor

`<Result> = GPSStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de activación del proveedor de la ubicación. Se pueden devolver los siguientes valores:


|   |   |
| --- | --- |
| *gpsDisabled* | El proveedor está incapacitado. |
| *gpsEnabled* | El proveedor está habilitado. |
| *gpsError* | Error al recuperar el estado de activación del proveedor. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).<br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |




<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Pedir que se le notifique cuando cambie el estado de activación

`GPSStatus(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

El procedimiento de WLanguage (procedimiento de "devolución de llamada") se llama para cada Notification.

Este procedimiento tiene el siguiente formato: 


```txt
PROCEDURE <Procedure name>(<Status>)
```


&lt;Status&gt; es una constante entera correspondiente al nuevo estado del proveedor.
Puede tomar los siguientes valores:


|   |   |
| --- | --- |
| *gpsAvailable* | El proveedor está disponible. |
| *gpsDisabled* | El proveedor fue desactivado por el usuario. |
| *gpsEnabled* | El proveedor fue habilitado por el usuario. |
| *gpsOffService* | El proveedor está fuera de servicio. |
| *gpsUnavailable* | El proveedor no está disponible temporalmente. |

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000135"></a>

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Para dejar de recibir notificaciones cuando cambia el estado, utilice la función **GPSStatus** con una cadena vacía ("") como parámetro o utilice [GPSEnd](../WDLang3/1000019210.md).



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000388"></a>

Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.

Característica necesaria: Geolocalización

Esta característica permite a las aplicaciones acceder a las características de geolocalización que se encuentran en el dispositivo o en el equipo.


## PermissionList( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve uno o todos los permisos declarados por la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
//Displays the list of application permissions 
FOR EACH ELEMENT Perm of PermissionList()
	ListAdd(LIST_Permission, Perm.Nom)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Listado de todos los permisos declarados por la aplicación

`<Result> = PermissionList()`
---

**`<Result>: (Array de las variables Permission)`**

Array de variables [Permission](../WDLang3/1410087419.md) que contiene la lista de permisos declarados por la aplicación.


<a name="SYNTAX2"></a>

### Obtener una Permission específica declarada por la aplicación

`<Result> = PermissionList(<Permission>)`
---

**`<Result>: (Permission Variable)`**

la variable [Permission](../WDLang3/1410087419.md) que corresponde a la permission especificada. Si el Permission no ha sido declarado por la aplicación, se mostrará un error fatal. 

**`<Permission>: (Cadena de caracteres o constante)`**

Nombre de la Permission a solicitar. Este parámetro puede corresponder a: 

- una cadena de caracteres con el siguiente formato: android.Permission.&lt;NAME&gt;. La lista de permisos del SDK de Android se encuentra en: [https://developer.android.com/reference/android/Manifest.Permission](https://developer.android.com/reference/android/Manifest.permission).

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *permBackgroundLocation* | Permission para acceder a la ubicación del dispositivo cuando la aplicación se ejecuta en el Background. |
| *permCamera* | Permission para acceder a la(s) cámara(s) del dispositivo. |
| *permFineLocation* | Permission para acceder a la ubicación precisa del dispositivo. |
| *permLocation* | Permission para acceder a la localización del dispositivo. |
| *permManageExternalStorage* | Permission para gestionar el almacenamiento externo. |
| *permReadContact* | Permission para leer los contactos. |
| *permReadPhoneState* | Permission para acceder a la información de Phone. |
| *permRecordAudio* | Permiso para los flujos de audio de Record. |
| *permSendSMS* | Permission para el envío de SMS. |
| *permWriteContact* | Permission para modificar los contactos. |
| *permWriteExternalStorage* | Permission para escribir en la memoria externa. |

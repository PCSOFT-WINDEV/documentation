
## SaaSAdminDeleteSite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the "Forum" SaaS site
S is saasSite 
S = SaaSAdminFindSite("Forum")

IF SaaSAdminDeleteSite(S) = True THEN
	Info("Site successfully deleted.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminación de un sitio SaaS (a través de un saasSite Variable)

`<Result> = SaaSAdminDeleteSite(<SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS site>: (Variable de tipo saasSite)`**

Nombre del [saasSite](../WDLang3/1000019068.md) Variable correspondiente al sitio SaaS que se va a eliminar de la base de datos SaaS.


<a name="SYNTAX2"></a>

### Suprimir un sitio SaaS identificado por su nombre

`<Result> = SaaSAdminDeleteSite(<SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio de SaaS que se va a eliminar de la base de datos de SaaS.

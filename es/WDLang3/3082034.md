


## zipMsgError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el mensaje associated con un código de error en un archivo.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003082034&name=zipmsgerror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
MyArchive is zipArchive 
// Open an archive
ResOpenArchive = zipOpen(MyArchive)
// Error while opening an archive
IF ResOpenArchive <> 0 THEN
	// Display an error message
	Info(zipMsgError(ResOpenArchive))
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = zipMsgError(<Error code>)`
---

**`<Result>: (Cadena de caracteres)`**

Mensaje de error associated con el tipo de error especificado.

**`<Error code>: (Integro)`**

Código de error devuelto al manejar un archivo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Códigos de error
<a name="codigos_error_ELTPARAGRAPHE000077"></a>

Las funciones de archivo devuelven los siguientes códigos de error:

- 1: La ruta pasada como parámetro no existe.

- 2: Acceso denegado: el usuario no tiene suficientes derechos.

- 3: El archivo está corrupto.

- 4: La ruta no existe en el archivo.

- 5: No se puede escribir en el archivo.

- 6: Los archivos de los subarchivos no están ordenados (cuando se extraen de un archivo de varias partes).

- 7: El archivo ya se encuentra en el archivo.

- 8: No se puede añadir un archivo comprimido a sí mismo. Por ejemplo, "MyArchive" no puede añadirse a "MyArchive".

- 9: Memoria insuficiente para realizar la operación. 

- 10: No hay espacio suficiente para crear el ejecutable o el usuario no tiene derechos de escritura en el archivo original (.WDZ).

- 11: No se encuentra el Icon o el formato Icon es inválido.

- 12: No hay espacio suficiente para crear el ejecutable o el usuario no tiene derechos de lectura en el archivo inicial (.WDZ).




## grTitleFont (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la fuente utilizada para el título del gráfico.

Ejemplo de título en un gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the font used for the title of CHART_MyChart
// The identifier of font used is 7.
grTitleFont(CHART_MyChart, 7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Definir la fuente utilizada para el título del gráfico

`grTitleFont(<Chart name> , <Font used>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Font used>: (tipo de letra)`**

Fuente a utilizar. Este parámetro corresponde a una variable de tipo [tipo de letra](../Motscles/1514045.md). Las características de la fuente se definieron mediante la función [FontCreate](../WDLang1/3060002.md) o las [propiedades de fuentes](../Proprietes/2515045.md).  


<a name="SYNTAX2"></a>

### Definir la fuente utilizada para el título del gráfico (se mantiene la sintaxis para compatibilidad con versiones anteriores)

`grTitleFont(<Chart name> , <Font identifier>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Font identifier>: (Integro)`**

Identificador de fuente, definido por [grCreateFont](../WDLang3/3042001.md). Un error de WLanguage ocurre si esta fuente no existe.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Título del gráfico
<a name="titulo_del_grafico_ELTPARAGRAPHE000157"></a>El título de la carta puede ser mostrado Above o debajo de la carta.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)
 El contenido del título del gráfico está definido por [grTitle](../WDLang3/3042056.md). Esta función también puede utilizarse para set la posición del título.

La fuente utilizada para el título del gráfico se puede modificar con la función **grTitleFont**.

Los datos definidos con la función **grTitleFont** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).
<a name="NOTE0_2"></a>


### Inicialización de fuentes
<a name="inicializacion_fuentes_ELTPARAGRAPHE000182"></a>

La fuente Default utilizada para el título del gráfico tiene las siguientes características:

- Nombre fuente: Arial

- envergadura: 14

- Estilo: Normal

- Color: Black




Cuando se utiliza [grCreate](../WDLang3/3042005.md), se usa la fuente Default para el título del gráfico.

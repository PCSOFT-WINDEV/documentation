


## grHMDeleteColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los colores especificados por [grHMColor](../WDLang3/1000021629.md) de un gráfico Heatmap. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the colors
grHMDeleteColor(CHART_Chart)
grHMColor(CHART_Chart, -65, RGB(0, 105, 255))
grHMColor(CHART_Chart, 0, RGB(240, 255, 0))
grHMColor(CHART_Chart, 35, RGB(255, 31, 0))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grHMDeleteColor(<Chart name> [, <Level>])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Level>: (Real opcional)`**

Nivel de Color que será eliminado. Si no se especifica este parámetro, se borran todos los niveles de Color s y se utiliza la Color de la primera serie.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los datos modificados con la función **grHMDeleteColor** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).


## Gestionando los calendarios (Android/iOS)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
WINDEV Mobile le permite acceder fácilmente a los calendarios y citas que se encuentran en los teléfonos y tabletas de Android y iOS. Usted tiene la capacidad de: 

- Enumere los calendarios existentes.

- Haga una lista de las citas.

- Añadir una nueva Appointment a una Calendar.

- Modificar los datos de un Appointment.

- Borrar un Appointment.

- Borre todas las citas de un Calendar.

- Define la repetition de una Appointment (todos los días, todos los meses, etc.).

- Agregar/Modificar/Borrar los recordatorios para un determinado Appointment (por ejemplo: enviar un SMS una hora antes de la Appointment).

- Inicie la aplicación nativa de gestión de calendarios para crear, editar o ver una Appointment.

- Varios tipos de datos: 
	


|   |   |
| --- | --- |
| [Appointment](../WDLang1/1000019244.md) | El tipo **Appointment** permite definir las características avanzadas de una cita: hora, lugar, .... |
| [Calendar](../WDLang3/1000020663.md) | El tipo **Calendar** permite manipular un calendario (o agenda) asociado a una cuenta de usuario o que se encuentra en el dispositivo. |


## Aplicación Cliente: Ejecución de procedimientos en un servidor SOAP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000125"></a>
WINDEV incluye varias funciones SOAP. Estas funciones le permiten gestionar la ejecución de procedimientos en un servidor SOAP desde sus aplicaciones WINDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000149"></a>
Todos los parámetros necesarios para ejecutar una Procedure DE JABÓN se suministran en la documentación del servidor de JABÓN.

Para ejecutar un Procedure en un servidor SOAP:

1. Inicializar la estructura de parámetros para la ejecución de Procedure (ver [estructura de una Procedure DE JABÓN](../WDLang3/3069009.md)).
	Observación: Para realizar comprobaciones adicionales en el servidor SOAP, añada una cabecera Procedure ([SOAPAddHeader](../WDLang3/3069001.md)).

2. Ejecute el programa Procedure ([SOAPRun](../WDLang3/3069013.md) o [SOAPRunXML](../WDLang3/3069014.md)). Las siguientes operaciones se realizan automáticamente:

	- conectando la computadora actual y el servidor SOAP.

	- transmitiendo los parámetros del Procedure al servidor SOAP.

	- comprobando la cabecera si es necesario.

	- ...dirigiendo el Procedure.

	- devolviendo el resultado o el error de Procedure al ordenador actual.




3. Revisa el resultado de Procedure.




El resultado de [SOAPRun](../WDLang3/3069013.md) o [SOAPRunXML](../WDLang3/3069014.md) se utiliza para saber si la conexión se ha establecido con éxito.

Si la conexión no se ha establecido, utilice [ErrorInfo](../WDLang1/3013008.md) para averiguar la causa del error.

Si se ha establecido la conexión, compruebe el resultado de [SOAPGetResult](../WDLang3/3069002.md):

- si el resultado no es una cadena vacía (""), la Procedure se ha ejecutado correctamente.

- si el resultado es una cadena vacía (""), la Procedure no se ha ejecutado y/o se ha devuelto un error. Para conocer el error de Procedure, utilice [JABÓNError](../WDLang3/3069005.md).

## Observación
<a name="observacion_ELTTEXTE000173"></a>
Las transferencias no son seguras en este momento (los datos transferidos no están encriptados). El protocolo SOAP no debe utilizarse para transferir datos sensibles.

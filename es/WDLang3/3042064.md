


## grLoadParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura los parámetros utilizados para dibujar un gráfico. Estos parámetros han sido guardados por [grSaveParámetro](../WDLang3/3042065.md).

**Recordatorio**: Por Default, los parámetros del gráfico modificados por un usuario se guardan al cerrar la aplicación. Estos parámetros se restauran automáticamente durante el siguiente inicio de la aplicación.. Para obtener más información, consulte [grCreate](../WDLang3/3042005.md).

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the parameters of a chart created through programming
INIWrite("Parameters", "MyChart", "Param", grSaveParameter("MyChart"))

// Restore the parameters of a chart created through programming
grLoadParameter("MyChart", INIRead("Parameters", "MyChart", "Param"))
```
<a name="Example2"></a>

```wl
// Save the parameters of a Chart control
INIWrite("Parameters", "CHART_Chart", "Param", grSaveParameter(CHART_Chart))

// Restore the parameters of a Chart control
grLoadParameter(CHART_Chart, INIRead("Parameters", "CHART_Chart", "Param"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grLoadParameter(<Chart name> , <Parameters>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cargaron los parámetros,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Parameters>: (Cadena de caracteres ANSI)`**

Parámetro a restaurar. Este parámetro está comprimido. Sólo puede ser devuelto por [grSaveParámetro](../WDLang3/3042065.md). El valor de este parámetro se puede guardar en un archivo, en un memo o en el registro.

Si este parámetro corresponde a una cadena vacía (""), la función no tiene ningún efecto y devuelve <u><u><u><u>False</u></u></u></u>.

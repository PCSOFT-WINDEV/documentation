
## SMSRunApp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la aplicación nativa de envío de SMS que se encuentra en el dispositivo (Android o iPhone/iPad). El contenido y los destinatarios del SMS se inicializan a partir de la información especificada en la [Estructura del SMS](../WDLang3/3068003.md) de WLanguage.

Atención: El SMS no se envía. Sólo se inicializan los controles de la aplicación para el envío de SMS.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SMS.Number = "0610203040" 
SMS.Message = "I am sending SMSs with WINDEV Mobile!"
SMSRunApp()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SMSRunApp()`

<a name="NOTE0_1"></a>

## Observaciones


### Elementos que se tienen en cuenta en la estructura del SMS
<a name="elementos_que_tienen_cuenta_estructura_del_sms_ELTPARAGRAPHE000043"></a>

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo se tienen en cuenta las variables SMS.Number y SMS.Message.


Para más detalles, consulte la [Estructura del SMS](../WDLang3/3068003.md) de WLanguage. 
<a name="NOTE0_2"></a>


### Casos de error
<a name="casos_error_ELTPARAGRAPHE000067"></a>

Si no se ha encontrado ninguna aplicación nativa para el envío de SMS en el dispositivo, la variable [ErrorOccurred](../WDLang1/3087001.md) se establecerá en <u><u><u><u>True</u></u></u></u> después de llamar a la función. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

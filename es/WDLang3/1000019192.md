


## Funciones de geolocalización/GPS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Las siguientes funciones se usan para los datos de geolocalización de manipular y para los dispositivos GPS de control:



|   |   |
| --- | --- |
| [geoAzimuth](../WDLang3/1000019212.md) | Devuelve la dirección representada por un lugar en relación con otro. |
| [geoDistance](../WDLang3/1000019211.md) | Devuelve la distancia entre los dos lugares. Esta distancia está expresada en metros por Default. |
| [geoGetAddress](../WDLang3/1000019797.md) | Recupera una dirección o una lista de direcciones correspondientes a la description de un lugar geográfico. |
| [geoGetArea](../WDLang3/1000025776.md) | Recupera las coordenadas del área geográfica que corresponde a una determinada description. |
| [geoRunApp](../WDLang3/1000019954.md) | Inicia la aplicación nativa de mapas del dispositivo para mostrar: <br><br>- una ubicación geográfica específica, <br><br>- una ruta entre dos lugares.<br><br><br> |
| [geoTrackingDisable](../WDLang3/1000021584.md) | Desactiva el seguimiento de las posiciones de los usuarios de una aplicación WINDEV Mobile. |
| [geoTrackingEnable](../WDLang3/1000021583.md) | Permite seguir la ubicación en una aplicación WINDEV Mobile. |
| [geoTrackingProcedure](../WDLang3/1000021585.md) | Define el WLanguage Procedure que será llamado cuando cambie la ubicación del usuario. |
| [geoTrackingStatus](../WDLang3/1000021592.md) | Recupera el estado del sistema para el seguimiento de posición. |
| [GPSDetectPosition](../WDLang3/1000019209.md) | Pide ser notificado cuando el dispositivo se acerca a una ubicación determinada. |
| [GPSEnd](../WDLang3/1000019210.md) | Indica el fin del uso de las funciones de geolocalización.. |
| [GPSFollowMovement](../WDLang3/1000019206.md) | Solicita que se le notifique periódicamente la ubicación actual del dispositivo. |
| [GPSGetPosition](../WDLang3/1000019205.md) | Recupera información sobre la posición actual del dispositivo. |
| [GPSInfo](../WDLang3/1000019208.md) | Devuelve información sobre el proveedor de ubicación utilizado por la aplicación para las funciones de geolocalización. |
| [GPSInitParameter](../WDLang3/1000019204.md) | Inicializa los parámetros de las funciones de geolocalización de WLanguage y busca un proveedor de ubicación. |
| [GPSLastPosition](../WDLang3/1000019239.md) | Recupera la información sobre la última posición conocida del dispositivo. |
| [GPSStatus](../WDLang3/1000019207.md) | Recupera el estado de activación del proveedor de geolocalización o pide ser notificado cuando cambia el estado. |
| [GPSStopDetection](../WDLang3/1000019240.md) | Detiene la detección de la ubicación activada por [GPSDetectPosition](../WDLang3/1000019209.md). |





Los siguientes tipos avanzados se utilizan para los datos de geolocalización de manipular:



|   |   |
| --- | --- |
| [Dirección](../WDLang3/1000019944.md) | El tipo **Address** se utiliza para manipular el Address de un lugar o persona. |
| [geoPosition](../WDLang3/1000019191.md) | El tipo **geoPosition** se utiliza para las posiciones geográficas de manipular a través de funciones de geolocalización WLanguage. |

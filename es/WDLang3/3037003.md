


## Funciones de fax
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para enviar faxes:



|   |   |
| --- | --- |
| [FaxConnect](../WDLang3/3037005.md) | Establece una conexión con un servidor de fax. |
| [FaxDelete](../WDLang3/3037014.md) | Elimina un fax del administrador de trabajos de impresión y lo cancela. |
| [FaxDisconnect](../WDLang3/3037007.md) | Cierra una conexión con un servidor de fax (establecida previamente por [FaxConnect](../WDLang3/3037005.md)). |
| [FaxInbox](../WDLang3/3037009.md) | Enumera los faxes pendientes o los faxes entrantes. |
| [FaxOutbox](../WDLang3/3037008.md) | Enumera los faxes pendientes o los faxes salientes. |
| [FaxPause](../WDLang3/3037015.md) | Pone en pausa un fax en el spooler. |
| [FaxRestart](../WDLang3/3037012.md) | Reinicia un fax en el administrador de trabajos de impresión. |
| [FaxResume](../WDLang3/3037013.md) | Vuelve a activar un fax en pausa en el administrador de trabajos de impresión (fax pendiente). |
| [FaxSend](../WDLang3/3037006.md) | Envía un fax según la conexión establecida previamente ([FaxConnect](../WDLang3/3037005.md)). |
| [FaxStatus](../WDLang3/3037001.md) | Devuelve el estado de un fax enviado por [FaxSend](../WDLang3/3037006.md). |


## RecorderAction( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta una acción en la grabadora o dictáfono que se encuentra en el dispositivo actual.

**Observación**: Esta función sólo está disponible si el dispositivo actual tiene una grabadora.

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RecorderAction(<Action> [, <File name>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició la acción especificada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.


Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

**`<Action>: (Constante de tipo Integer)`**

Tipo de acción a realizar:


|   |   |
| --- | --- |
| *recorderClose* | Cierra la grabadora. |
| *recorderPlay* | Abre la grabadora y reproduce el archivo de sonido especificado. |
| *recorderRecord* | Abre la grabadora e inicia la grabación.<br>La grabación se inicia cuando el usuario pulsa el botón que llama **RecorderAction**. |
| *recorderStop* | Detiene la grabación actual.<br>La grabación se detiene cuando el usuario suelta el botón que llama **RecorderAction**. |



**`<File name>: (Cadena de caracteres opcional)`**

Nombre y ruta completa del archivo de sonido:

- para grabar (constante *recorderRecord*).

- por reproducir (constante *recorderPlay*).

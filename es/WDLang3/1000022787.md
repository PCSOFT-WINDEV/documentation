
## WebserviceReadHTTPHeader( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la cabecera HTTP recibida en una llamada a un servicio web SOAP o REST. Estas cabeceras HTTP son enviadas por el cliente al servidor en la petición actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Validates the origin of the request (CORS protocol)
sOrigin is string = WebserviceReadHTTPHeader("Origin")
// Validate the origin
...
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Leyendo el encabezado del HTTP identificado por su nombre

`<HTTP header> = WebserviceReadHTTPHeader(<Name of header to read>)`
---

**`<HTTP header>: (Cadena de caracteres)`**

Valor de la cabecera HTTP. El formato de la cabecera es el siguiente:

- El nombre del encabezado se devuelve en mayúsculas.

- Los guiones se sustituyen por caracteres "guión bajo" (el nombre de origen del encabezado HTTP tal y como lo envió el cliente no está disponible). 

- Se devuelve una cadena vacía ("") si no hay ninguna otra cabecera que leer.




**`<Name of header to read>: (Cadena de caracteres)`**

Nombre del encabezado a leer.


<a name="SYNTAX2"></a>

### Lectura de la cabecera HTTP identificada por su índice

`<HTTP header> = WebserviceReadHTTPHeader(<Subscript>)`
---

**`<HTTP header>: (Cadena de caracteres)`**

Valor de la cabecera HTTP. El formato de la cabecera es el siguiente:

- El nombre del encabezado se devuelve en mayúsculas.

- Los guiones se sustituyen por caracteres "guión bajo" (el nombre de origen del encabezado HTTP tal y como lo envió el cliente no está disponible). 

- Se devuelve una cadena vacía ("") si no hay ninguna otra cabecera que leer.




**`<Subscript>: (Integro)`**

Índice de cabecera a leer. Para leer todas las cabeceras HTTP, aumente esta índice hasta que la función devuelva una cadena vacía.

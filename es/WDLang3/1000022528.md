
## SaaSSendSecurityCode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía el código de seguridad al usuario para que pueda cambiar su contraseña.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on "Forgotten password"
IF NOT SaaSSendSecurityCode("Gary") THEN
	Info("Unable to retrieve the password. " + 
		"Did you specify an email address in your profile?")
	RETURN
END

// Validate the form
IF SaaSConnect("Gary", EDT_Code) THEN
	// The code is valid, replace the password by a new one 
	SaaSChangePassword(EDT_NewPassword)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSSendSecurityCode(<Username> [, <Webservice>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código de seguridad se envió correctamente.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Username>: (Cadena de caracteres)`**

Nombre de usuario.

**`<Webservice>: (Cadena de caracteres opcional)`**

Servicio web a utilizar. Este parámetro corresponde a:

- el nombre de host del servicio web SaaS (por ejemplo, "Computer", "Computer.MyNetwork.com", "http://Mydomain.com").

- la URL completa del servicio web SaaS (por ejemplo, "http://domain.com/WDSAAS28 0_WEB/WDSaaS28 0.awws").


Por Default, la conexión se establecerá con el servicio web SaaS instalado con el WEBDEV servidor de aplicaciones en el ordenador local.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El código de seguridad es válido mientras la sesión que solicitó el cambio de contraseña esté habilitada.

- El código de seguridad se envía a la dirección de correo electrónico especificada en el [saasUser](../WDLang3/1000019083.md) Variable correspondiente al usuario (**Email**). Si no se especifica esta Property, el correo electrónico se envía al correo electrónico Address utilizado como nombre de usuario (si es válido). 

- Para poder enviar el código de seguridad, los parámetros SMTP del administrador de SaaS deben estar configurados.: 

	- a través del administrador de SaaS. 

	- mediante [SaaSAdminConfigureSMTP](../WDLang3/1000022592.md).




## Funciones FTP WINDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>



|   |   |
| --- | --- |
| [NetClientMessageBox](../WDLang3/3056017.md) | Muestra un cuadro de diálogo en cada equipo cliente conectado al servidor..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetClientSendMessage](../WDLang3/3056007.md) | Envía un mensaje desde el servidor a todos los ordenadores cliente conectados..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetCloseRemoteAccess](../WDLang3/3056021.md) | Cierra la Line que fue abierta por [NetOpenRemoteAccess](../WDLang3/3056026.md). |
| [NetConnect](../WDLang3/3056004.md) | Conecta con un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV. |
| [NetDirList](../WDLang3/3056016.md) | Enumera los directorios y los archivos encontrados: <br><br>- en el servidor FTP (File Transfer Protocol) o en el servidor RPC (Remote Procedure Call) alimentado por WINDEV,<br><br>- en el ordenador cliente.<br><br><br><br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetDisconnect](../WDLang3/3056011.md) | Se desconecta de un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetDiskList](../WDLang3/3056015.md) | Enumera los discos disponibles:<br><br>- en el servidor FTP (File Transfer Protocol) o en el servidor RPC (Remote Procedure Call) alimentado por WINDEV,<br><br>- en el ordenador cliente.<br><br><br><br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetEndServer](../WDLang3/3056024.md) | Detiene un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV e iniciado por [NetStartServer](../WDLang3/3056013.md)..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetEraseFile](../WDLang3/3056018.md) | Elimina un archivo del servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetExecute](../WDLang3/3056002.md) | Ejecuta un programa específico en un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetGetFile](../WDLang3/3056025.md) | Recupera un archivo existente de un servidor FTP (File Transfer Protocol) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetInfoRemoteAccess](../WDLang3/3056028.md) | Devuelve información sobre el estado de una conexión, así como las estadísticas sobre la misma. |
| [NetIPAddress](../WDLang3/3056023.md) | Devuelve la IP (Protocolo INTERNET) Address de un ordenador. |
| [NetIPToMAC](../WDLang3/1000016005.md) | Devuelve el MAC Address correspondiente a un determinado IP Address. |
| [NetListIPAddress](../WDLang3/1000019683.md) | Devuelve la lista de direcciones IP (Protocolo INTERNET) de un ordenador. |
| [NetListRemoteAccess](../WDLang3/3056027.md) | Devuelve la lista de accesos remotos disponibles. |
| [NetMACAddress](../WDLang3/1000016006.md) | Devuelve el MAC Address (físico Address) de una de las tarjetas de red que se encuentran en el equipo. |
| [NetMachineName](../WDLang3/3056014.md) | Devuelve el nombre del ordenador local o el nombre de un ordenador cuya IP Address se conoce (resolución DNS inversa). |
| [NetMACToIP](../WDLang3/1000016004.md) | Devuelve la IP Address correspondiente a una determinada MAC Address. |
| [NetMsgError](../WDLang3/3056019.md) | Devuelve el mensaje de error correspondiente al número de error devuelto por [NetOpenRemoteAccess](../WDLang3/3056026.md) y [NetCloseRemoteAccess](../WDLang3/3056021.md). |
| [NetOpenRemoteAccess](../WDLang3/3056026.md) | Establece una conexión remota. |
| [NetRemoteMessageBox](../WDLang3/3056020.md) | Muestra un cuadro de diálogo en la pantalla del servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetRenameFile](../WDLang3/3056006.md) | Renombra (o mueve) un archivo encontrado en el servidor FTP (File Transfer Protocol) o en el servidor RPC (Remote Procedure Call) alimentado por WINDEV. |
| [NetSendBuffer](../WDLang3/3056001.md) | Transmite una cadena de caracteres a un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetSendFile](../WDLang3/3056005.md) | Transmite un archivo a un servidor FTP (File Transfer Protocol) alimentado por WINDEV..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetSendMessage](../WDLang3/3056003.md) | Activa un evento Windows en el sistema de un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV.<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetServerOption](../WDLang3/3056022.md) | Gestiona los derechos de los clientes en el servidor (servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV)..<br><br>**Esta función se conserva por motivos de compatibilidad con versiones anteriores**. |
| [NetStartServer](../WDLang3/3056013.md) | Inicia un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV. |

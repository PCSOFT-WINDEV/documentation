


## gr3DSParámetro (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera o modifica un parámetro para dibujar un gráfico en modo espacial 3D.. 

Observaciones: 

- Esta función no debe utilizarse en los controles de gráfico que se encuentran en el editor de reportes.

- Esta función es equivalente a la función [grParameter](../WDLang3/1000018337.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the type of animation played when 
// displaying "CHART_MyChart"
// The bars will be built progressively and
// the chart will appear with a rotation effect
gr3DSParameter(CHART_MyChart, gr3DSPropAnimation, 10)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperar el valor de un parámetro de dibujo

`<Result> = gr3DSParameter(<Chart name> , <Drawing parameter>)`
---

**`<Result>: (Entero, real o booleano)`**

Valor del parámetro de dibujo. El tipo depende del parámetro de dibujo especificado (vea las Notas).

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a: 

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md), 

- el control Gráfico creado en el editor de ventanas.




**`<Drawing parameter>: (Constante de tipo Integer)`**

Parámetro de dibujo cuyo valor se solicita:



|   |   |
| --- | --- |
| *gr3DSPropAnimation* | Tipo de animación que se reproduce al mostrar el gráfico. |
| *gr3DSPropAutoZoom* | automatic zoom del gráfico. |
| *gr3DSPropColorPlaneX* | Color de plano vertical (eje a la derecha o a la izquierda). |
| *gr3DSPropColorPlaneY* | Color de plano horizontal (eje Above o inferior). |
| *gr3DSPropColorPlaneZ* | Color del avión Background. |
| *gr3DSPropDistance* | Zoom del gráfico. |
| *gr3DSPropElevation* | Ángulo de rotación del gráfico alrededor del eje vertical (eje Y). |
| *gr3DSPropLegendOverlap* | La leyenda y el título se superponen al gráfico. |
| *gr3DSPropLightX* | Coordenada X de la luz que ilumina el gráfico. |
| *gr3DSPropLightY* | Coordenada Y de la luz que está iluminando la carta. |
| *gr3DSPropRotation* | Ángulo de rotación del gráfico alrededor del eje horizontal (eje X). |
| *gr3DSPropTransparency* | Nivel de transparencia para las barras o secciones del gráfico. |

## Observaciones


### Parámetro de dibujo
<a name="parametro_dibujo_ELTPARAGRAPHE000197"></a>

Los parámetros para dibujar una carta espacial 3D son los siguientes:


| Constante | Efecto | Valor del parámetro de dibujo |
| --- | --- | --- |
| *gr3DSPropAnimation* | Tipo de animación que se reproduce al mostrar el gráfico | Integro<br>Los valores posibles son:<br><br>- 0: ninguna animación<br><br>- 1: ampliar<br><br>- 2: efecto de rotación<br><br>- 4: dibujo progresivo de barras/secciones<br><br>- 8: visualización progresiva de los colores<br><br><br>Estos valores se pueden sumar.<br>El valor predeterminado es 15. |
| *gr3DSPropAutoZoom* | automatic zoom del gráfico | booleano<br><u><u><u><u>True</u></u></u></u> (predeterminado) si se debe calcular el zoom automáticamente, <u><u><u><u>False</u></u></u></u> en caso contrario.<br>Si se calcula el zoom automáticamente, no se tiene en cuenta el valor de la constante *gr3DSPropDistance*. |
| *gr3DSPropColorPlaneX* | Color de plano vertical (eje a la derecha o a la izquierda) | Integro<br>Componente RGB de Color (devuelto por [RGB](../WDLang1/3029012.md)) o [y la preselección Color](../WDLang5/3010002.md).<br>Para no dibujar este eje, utilice la constante *Transparent*.<br>Esta constante es tomada en cuenta por los gráficos de barras y por los gráficos de Line solamente.<br>El valor predeterminado es *LightGray*. |
| *gr3DSPropColorPlaneY* | Color de plano horizontal (eje Above o inferior) | Integro<br>Componente RGB de Color (devuelto por [RGB](../WDLang1/3029012.md)) o [y la preselección Color](../WDLang5/3010002.md).<br>Para no dibujar este eje, utilice la constante *Transparent*.<br>Esta constante es tomada en cuenta por los gráficos de barras y por los gráficos de Line solamente.<br>El valor predeterminado es *LightGray*. |
| *gr3DSPropColorPlaneZ* | Color del avión Background | Integro<br>Componente RGB de Color (devuelto por [RGB](../WDLang1/3029012.md)) o [y la preselección Color](../WDLang5/3010002.md).<br>Para no dibujar este eje, utilice la constante *Transparent*.<br>Esta constante es tomada en cuenta por los gráficos de barras y por los gráficos de Line solamente.<br>El valor predeterminado es *LightGray*. |
| *gr3DSPropDistance* | Zoom del gráfico | Real<br>Los valores posibles se incluyen entre 0 y el infinito.<br>Esta constante se tiene en cuenta únicamente si el valor de la constante *gr3DSPropAutoZoom* es <u><u><u><u>False</u></u></u></u>.<br>El valor predeterminado es 189. |
| *gr3DSPropElevation* | Ángulo de rotación (en grados) del gráfico alrededor del eje vertical (eje Y). | Real<br>Los valores posibles se incluyen entre 0 y 360.<br>El valor 0 muestra una vista frontal del gráfico.<br>El valor 90 muestra el gráfico de Above.<br>El mejor ángulo de rotación es calculado automáticamente por Default. |
| *gr3DSPropLegendOverlap* | La leyenda y el título se superponen al gráfico. | booleano<br><u><u><u><u>True</u></u></u></u> si la leyenda y el título se superponen a la carta, <u><u><u><u>False</u></u></u></u> (por Default) en caso contrario. |
| *gr3DSPropLightX* | Coordenada X de la luz que ilumina el gráfico. | Real<br>Los valores posibles se incluyen entre 0 y el infinito.<br>El valor predeterminado es 78. |
| *gr3DSPropLightY* | Coordenada Y de la luz que ilumina el gráfico | Real<br>Los valores posibles se incluyen entre 0 y el infinito.<br>El valor predeterminado es 80. |
| *gr3DSPropRotation* | Ángulo de rotación (en grados) del gráfico alrededor del eje horizontal (eje X). | Real<br>Los valores posibles se incluyen entre 0 y 360.<br>El valor 0 muestra una vista frontal del gráfico.<br>El valor 90 muestra una vista derecha del gráfico.<br>El mejor ángulo de rotación es calculado automáticamente por Default. |
| *gr3DSPropTransparency* | Nivel de transparencia (en porcentaje) para las secciones de barras o gráficos. | Integro<br>Los posibles valores se incluyen entre 0 (opaco) y 100 (Transparent/Invisible).<br>El valor predeterminado es 27%. |


<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000398"></a>

Si se utiliza la función **gr3DSParámetro** en un tipo de gráfico que no sea 3D, no se devuelve ningún error. La información se almacena y se utilizará cuando se modifique el tipo de gráfico (por [grType](../WDLang3/3042062.md) o con el menú contextual del gráfico).

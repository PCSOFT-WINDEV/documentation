
## SaaSAdminListClientSubscription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las suscripciones SaaS de una cuenta de cliente.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las suscripciones. Un gestor de cuentas de clientes tendrá acceso a sus propias suscripciones. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the subscriptions of a client in a Table control (linked to an Array variable)
arrSub is array of saasSubscription 
arrSub = SaaSAdminListClientSubscription(EDT_ClientName)
SourceToScreen()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Listado de las suscripciones SaaS de una cuenta de cliente (a través de las variables saasxxxx)

`<Result> = SaaSAdminListClientSubscription(<Client account> [, <SaaS site>])`
---

**`<Result>: (Array de las variables de la saasSubscription)`**

Lista de suscripciones.

**`<Client account>: (Variable de tipo saasClient)`**

Nombre de la [saasClient](../WDLang3/1000019072.md) Variable que corresponde a la cuenta de cliente SaaS para la que se encuentran las suscripciones.

**`<SaaS site>: (Opcional saasSite Variable)`**

Nombre de la variable [saasSite](../WDLang3/1000019068.md) que contiene el description del sitio SaaS a utilizar. Si se especifica este parámetro, sólo se mostrarán las suscripciones al sitio SaaS especificado; de lo contrario, se mostrarán todas las suscripciones de la cuenta de cliente.


<a name="SYNTAX2"></a>

### Enumerar las suscripciones de una cuenta de cliente identificada por su nombre

`<Result> = SaaSAdminListClientSubscription(<Client account> [, <SaaS site>])`
---

**`<Result>: (Array de las variables de la saasSubscription)`**

Lista de suscripciones.

**`<Client account>: (Cadena de caracteres)`**

Nombre de la cuenta de cliente SaaS para la que se solicitan las suscripciones.

**`<SaaS site>: (Cadena de caracteres opcional)`**

Nombre del sitio SaaS a utilizar. Si se especifica este parámetro, sólo se mostrarán las suscripciones al sitio SaaS especificado; de lo contrario, se mostrarán todas las suscripciones de la cuenta de cliente.

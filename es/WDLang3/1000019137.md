
## SaaSAdminFindSubscription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra una suscripción a SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las suscripciones. Un gestor de cuentas de clientes tendrá acceso a sus propias suscripciones. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is saasSite 
P is saasPricing 
C is saasClient 
Sub is saasSubscription 

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")
Sub = SaaSAdminFindSubscription(P, C)
IF ErrorOccurred = False THEN
	Info("The client has subscribed to this pricing")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Encontrar una suscripción (a través de las variables saasxxx)

`<Result> = SaaSAdminFindSubscription(<Pricing> , <Client account>)`
---

**`<Result>: (saasSubscription Variable)`**

variable [saasSubscription](../WDLang3/1000019088.md) que contiene el description de la suscripción buscada.

Si la búsqueda ha fallado, la Variable devuelta está vacía.

**`<Pricing>: (saasPricing Variable)`**

Nombre de [saasPricing](../WDLang3/1000019091.md) Variable correspondiente al precio para el que se solicita la suscripción.

**`<Client account>: (Variable de tipo saasClient)`**

Nombre de [saasClient](../WDLang3/1000019072.md) Variable correspondiente a la cuenta de cliente para la que se solicita la suscripción.


<a name="SYNTAX2"></a>

### Encontrar una suscripción usando el nombre del precio

`<Result> = SaaSAdminFindSubscription(<Pricing> , <SaaS site> , <Client account>)`
---

**`<Result>: (saasSubscription Variable)`**

variable [saasSubscription](../WDLang3/1000019088.md) que contiene el description de la suscripción buscada.

Si la búsqueda ha fallado, la Variable devuelta está vacía.

**`<Pricing>: (Cadena de caracteres)`**

Nombre del precio para el que se solicita la suscripción.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS al que pertenece el precio.

**`<Client account>: (Cadena de caracteres)`**

Nombre de la cuenta de cliente para la que se solicita la suscripción.


## AppointmentFirst( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la primera Appointment encontrada en la Calendar de una mensajería de Lotus Notes o Outlook.

Cuando se lee un Appointment, los datos del Appointment se actualizan en el [Estructura de Appointment](../WDLang3/1000011010.md). Para recuperar los datos de la Appointment actual, basta con recuperar los valores de la [Estructura de Appointment](../WDLang3/1000011010.md).

**Recordatorio**: Esta función no puede utilizarse para leer la primera Appointment encontrada en la Calendar de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf", "C:\Lotus Notes\julia.id")
// Read the first appointment
AppointmentFirst(SessionID)
// Add the appointments found into a List Box control
WHILE mAppointment.Out = False
	ListAdd(LIST_APTList, mAppointment.Subject + TAB + mAppointment.Location)
	AppointmentNext(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppointmentFirst(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la primera cita,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La variable mAppointment.Out permite averiguar si hay un Appointment actual.

- En Outlook, se puede especificar el modo de gestión de los contactos/participantes al iniciar la sesión ([EmailStartOutlookSession](../WDLang3/3032128.md) o [OutlookStartSession](../WDLang3/3032123.md)).

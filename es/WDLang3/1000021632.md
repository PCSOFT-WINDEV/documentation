


## grHMGetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera datos de un gráfico de Heatmap. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve data
let MyValue = grHMGetData(CHART_HMap, 1, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = grHMGetData(<Chart name> , <X> , <Y>)`
---

**`<Result>: (Real)`**



- Valor recuperado. 

- 0 si no existe ningún valor en la posición especificada. 




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md), 

- el nombre de la control Gráfico (en la ventana, página o editor de reportes).




**`<X>: (Integro)`**

Posición de los datos a recuperar en el eje X. 

**`<Y>: (Integro)`**

Posición de los datos a recuperar en el eje Y.

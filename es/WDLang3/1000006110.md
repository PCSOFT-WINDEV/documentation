


## Funciones para la gestión de contactos (Lotus Notes, Outlook y dispositivo móvil)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones de gestión de contactos se utilizan para: 

- gestionar los contactos de una libreta Address de Lotus Notes o Outlook. 







|   |   |
| --- | --- |
| [ContactAdd](../WDLang3/1000006101.md) | Añade un Contact: <br><br>- en un libro de Lotus Notes o Outlook Address.<br><br>- en la lista de contactos del dispositivo (Android, iPhone o iPad, Universal Windows).<br><br><br> |
| [ContactCreate](../WDLang3/1000019556.md) | Muestra la ventana de creación de Contact de la aplicación nativa de gestión de Contact que se encuentra en el dispositivo (Android, iPhone o iPad, Universal Windows). |
| [ContactDelete](../WDLang3/1000006109.md) | Borra la actual Contact: <br><br>- a partir de un libro Address de Lotus Notes o Outlook. <br><br>- de la lista de contactos de la Phone (Android, iPhone o iPad, Universal Windows).<br><br><br> |
| [ContactDisplay](../WDLang3/1000019506.md) | Abre el formulario de una Contact en la aplicación nativa de gestión de Contact del dispositivo (Android, iPhone o iPad, Universal Windows). |
| [ContactEdit](../WDLang3/1000019507.md) | Abre el formulario de una Contact en modo de edición en la aplicación nativa de Contact s del dispositivo (Android, iPhone, iPad, Universal Windows). |
| [ContactFirst](../WDLang3/1000006106.md) | Lee la primera Contact en un libro de Lotus Notes o Outlook Address. |
| [ContactLast](../WDLang3/1000006103.md) | Lee la última Contact en un libro de Lotus Notes o Outlook Address. |
| [ContactList](../WDLang3/1000020266.md) | Enumera los contactos del dispositivo correspondientes a los criterios especificados.. |
| [ContactListSource](../WDLang3/1000020214.md) | Enumera las fuentes (o cuentas) de los contactos guardados en el dispositivo. |
| [ContactModify](../WDLang3/1000006104.md) | Modifica el contacto actual en una libreta de Lotus Notes o Outlook Address. |
| [ContactNext](../WDLang3/1000006108.md) | Lee el Contact después del Contact actual en un libro Address de Lotus Notes o Outlook. |
| [ContactPrevious](../WDLang3/1000006105.md) | Lee el Contact anterior al Contact actual en un libro de Address de Lotus Notes o Outlook. |
| [ContactRead](../WDLang3/1000006112.md) | Lee un contacto en una libreta de Lotus Notes o Outlook Address. |
| [ContactReset](../WDLang3/1000006107.md) | Reinicia: <br><br>- la estructura [Contact](../WDLang3/1000006111.md). <br><br>- a [Contacto](../WDLang3/1000020254.md) Variable. <br><br><br> |
| [ContactSearch](../WDLang3/1000006102.md) | Busca un contacto en una libreta de Lotus Notes o Outlook Address. |
| [ContactSelect](../WDLang3/1000019508.md) | Muestra la lista de contactos encontrados en el dispositivo (Android, iPhone o iPad, Universal Windows). |


## StreamChangeDriver( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica la cámara que se utilizará como fuente para una secuencia y/o una control Conferencia.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on back Camera:
StreamChangeDriver(CONF_Stream1, 2)

// Moves the camera onto another Conference control
StreamChangeDriver(CONF_Stream2, 0) // Frees the camera otherwise the next call fails

StreamChangeDriver(CONF_Stream1, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Definir la cámara fuente a partir del número de cámara

`StreamChangeDriver(<Conference control> , <Camera number>)`
---

**`<Conference control>: (Nombre del control o cadena de caracteres)`**

Nombre de la control Conferencia o nombre del flujo a utilizar. 

**`<Camera number>: (Integro)`**

Número de la cámara a utilizar. 

- Si este parámetro es set a 0, el flujo se desconecta de la cámara associated. 

- Si este parámetro no es válido, la función falla y se produce un error no fatal.. 

- Si el número de cámara no existe, el vapor ya no enviará imágenes.. 





<a name="SYNTAX2"></a>

### Definir la cámara de origen a partir del conductor de la cámara

`StreamChangeDriver(<Conference control> , <Driver name>)`
---

**`<Conference control>: (Nombre del control o cadena de caracteres)`**

Nombre de la control Conferencia o nombre del flujo a utilizar. 

**`<Driver name>: (Cadena de caracteres)`**

Nombre de la cámara a utilizar. Este nombre corresponde a uno de los nombres devueltos por [VideoListDriver](../WDLang1/3079008.md) (utilizado con la constante *viDirectShow*). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si hay una comunicación en curso, se modifica la fuente de vídeo.

- Para averiguar el nombre de los arroyos que se encuentran en una ventana, mostrar la ventana description de los arroyos: 

	- Desde la ventana de description de la control Conferencia: En la pestaña "General", haga clic en "Editar flujos de ventanas". 

	- En el menú contextual de control Conferencia: Seleccione "Editar flujos de ventanas".

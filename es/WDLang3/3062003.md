


## Funciones de gestión de los puertos serie y paralelo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>









|   |   |
| --- | --- |
| [sClose](../WDLang3/3062010.md) | Cierra el puerto serie, el puerto paralelo o el puerto de infrarrojos especificado. |
| [sEndEvent](../WDLang3/3062008.md) | Desactiva la detección de una Event en un puerto serie. |
| [sEscape](../WDLang3/3062006.md) | Ejecuta diferentes funciones que afectan directamente a los bits de puerto serie, puerto paralelo o puerto infrarrojo, independientemente del protocolo de comunicación. |
| [sEvent](../WDLang3/3062014.md) | Se ramifica un Event en un puerto serial. |
| [sInEntryQueue](../WDLang3/3062004.md) | Recupera el número de bytes pendientes en el búfer de entrada de un puerto serie o puerto de infrarrojos especificado. |
| [sInExitQueue](../WDLang3/3062005.md) | Recupera el número de bytes en espera de ser transferidos en el búfer de salida del puerto serie o puerto paralelo especificado. |
| [sListPort](../WDLang3/1000025980.md) | Devuelve la lista de puertos serie y paralelos de el equipo. |
| [sOpen](../WDLang3/3062013.md) | Abre e inicializa:<br><br>- a serial port.<br><br>- un puerto paralelo.<br><br>- un puerto de infrarrojos.<br><br><br> |
| [sParameter](../WDLang3/3062011.md) | Define o modifica los parámetros para configurar el puerto serie, puerto paralelo o puerto de infrarrojos especificado. |
| [sRead](../WDLang3/3062012.md) | Lee una cadena de caracteres en el búfer de entrada del puerto serie especificado. |
| [sWrite](../WDLang3/3062009.md) | Escribe una cadena de caracteres en el búfer de salida del puerto serie, paralelo o de infrarrojos especificado. |



Otras funciones de relacionado: 



|   |   |
| --- | --- |
| [sComputeCrc16](../WDLang1/3062007.md) | Comprueba una cadena de caracteres antes y después de la transmisión entre las aplicaciones WINDEV y los sitios WEBDEV. |
| [sComputeCrc32](../WDLang1/3062015.md) | Calcula la comprobación de redundancia cíclica (CRC) del búfer. |

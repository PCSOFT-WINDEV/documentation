
## PhotoRunApp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la aplicación de cámara nativa del dispositivo para tomar una foto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sPhoto is string = PhotoRunApp()
IF sPhoto <> "" THEN
	Info("Photo saved in the file: " + sPhoto)
END
```


<a name="Example2"></a>



```wl
// Starts the application
PhotoRunApp(CB_PhotoRunApp)

INTERNAL PROCEDURE CB_PhotoRunApp(sPath is string)
// If the return is an empty string vide (""), an error occurred
IF sPath ~= "" THEN
	Error(ErrorInfo())
	RETURN
END
// Message for end of recording
Info("The photo was saved here: " + sPath)
```





<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Iniciando la aplicación de la cámara nativa (llamada de bloqueo)

`<Result> = PhotoRunApp()`
---

**`<Result>: (Cadena de caracteres)`**



- ruta completa del archivo donde se guardó la foto,

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).





<a name="SYNTAX2"></a>

### Iniciando la aplicación nativa de la cámara (llamada sin bloqueo)

`<Result> = PhotoRunApp([<WLanguage procedure>])`
---

**`<Result>: (Cadena de caracteres)`**



- ruta completa del archivo donde se guardó la foto,

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del lenguaje WL Procedure ("callback") llamado cuando se guarda la foto.
Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure name> (<Path>)
```

donde &lt;Path&gt; corresponde a la copia de seguridad ruta del Image.

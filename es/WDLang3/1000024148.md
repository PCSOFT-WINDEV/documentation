


## <Chart>.SurfaceGetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera los datos de un gráfico de superficie.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display a data in the "CHART_MyChart" control. 
// This data is the X-coordinate of the first point.
Info(CHART_MyChart.SurfaceGetData(1, grXCoordinate))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Chart control>.SurfaceGetData(<Subscript> , <Value to retrieve>)`
---

**`<Result>: (Real)`**

Valor recuperado.

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Subscript>: (Integro)`**

Índice de datos definido por el orden de adición en el gráfico. 
Si el dato correspondiente al índice especificado no existe, el valor recuperado es 0.

**`<Value to retrieve>: (Integro)`**

Indica el valor a recuperar:


|   |   |
| --- | --- |
| *grXCoordinate* | Coordenada X del punto a recuperar. |
| *grYCoordinate* | Coordenada Y del punto a recuperar. |
| *grZCoordinate* | Coordenada Z del punto a recuperar. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Recordatorio: Para los datos de una carta de superficie, el plano XY constituye la base de la superficie, mientras que Z indica la altura del punto.

- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [.StockGetData](../WDLang3/1000023566.md).

- Para recuperar los datos de un gráfico de dispersión, se recomienda utilizar [.ScatterGetDataXY](../WDLang3/1000024108.md).

- Para recuperar un dato de un gráfico, utilice [&lt;Gráfico&gt;.GetData](../WDLang3/1000024122.md).

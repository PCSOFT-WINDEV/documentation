
## AppleRequestTrackingAuthorizationAsynchronous (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la solicitud de Permission para el seguimiento de la actividad del usuario (sólo para iOS). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=AppleATT.gif)


Esta autorización es necesaria para utilizar [AppleAdvertisingIdentifier](../WDLang3/1410087936.md).

Observación: la aplicación debe ser totalmente funcional, aunque el usuario no conceda esta Permission: sólo se obstaculizará el seguimiento de la actividad. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SWITCH AppleTrackingAuthorizationStatus()
     CASE atasNotDetermined
             AppleRequestTrackingAuthorizationAsynchronous(...
			AppleRequestTrackingAuthorizationAsynchronous_Callback)
     CASE atasAuthorized
             TrackUser()
END

INTERNAL PROCEDURE AppleRequestTrackingAuthorizationAsynchronous_Callback(AuthorizationStatus)
	 IF AuthorizationStatus = atasAuthorized THEN
                     TrackUser()
       END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AppleRequestTrackingAuthorizationAsynchronous(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del WLanguage Procedure ("callback") llamado en la respuesta del usuario. Este procedimiento tiene el siguiente formato: 

```txt
<Procedure name>(<Status>)
```

donde &lt;Estado&gt; es una constante entera que puede tomar uno de los siguientes valores: 


|   |   |
| --- | --- |
| *atasAuthorized* | Se ha concedido la autorización para rastrear al usuario o al dispositivo. |
| *atasDenied* | Se ha denegado la autorización para el seguimiento del usuario o del dispositivo. |
| *atasNotDetermined* | El usuario aún no ha concedido o denegado la solicitud. |
| *atasRestricted* | El dispositivo se gestiona mediante un "perfil de gestión del dispositivo" que restringe el seguimiento. |

## Observaciones


### Personalización de la ventana de solicitud de Permission
<a name="personalizacion_ventana_solicitud_permission_ELTPARAGRAPHE000061"></a>

El texto de la solicitud de autorización no se puede personalizar. 

No obstante, puede añadir un breve texto explicativo en la ventana. Este mensaje puede introducirse en "Permisos", en la generación de aplicaciones iOS asistente. Simplemente introduzca un texto específico para el "Seguimiento de la utilización de description" Permission. 


### Configurar manualmente el seguimiento de la publicidad
<a name="configurar_manualmente_seguimiento_publicidad_ELTPARAGRAPHE000068"></a>

Para configurar el seguimiento de una aplicación: 

- Antes de iOS 14, el seguimiento es común a todas las aplicaciones. Se puede configurar a través de "Configuración &gt;&gt; Privacidad &gt;&gt; Publicidad". 

- A partir de iOS 14, el seguimiento de la publicidad se puede personalizar: 

	- para el dispositivo y las aplicaciones: "Configuración &gt;&gt; Privacidad &gt;&gt; Publicidad". 

	- para aplicaciones: "Configuración &gt;&gt; "Nombre de la aplicación" &gt;&gt; Permitir seguimiento".

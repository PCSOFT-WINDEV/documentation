


## &lt;emailOutlookSession Variable&gt;.RemoveFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una carpeta del software de mensajería de Outlook.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Start the Outlook session 
SessionID is EmailOutlookSession
...
// Delete a folder
SessionID.RemoveFolder("Personal folders\Inbox\Work")
// Delete a folder from the current folder
SessionID.RemoveFolder(SessionID.CurrentFolder() + "\Work")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.DeleteFolder(<Folder path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailOutlookSession)`**

Nombre de la [emailOutlookSession](../WDLang3/1000018767.md) Variable correspondiente a la sesión Email a utilizar.

**`<Folder path>: (Cadena Unicode)`**

Ruta de la carpeta a eliminar.



<a name="NOTE0"></a>

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Para un servidor de Outlook
<a name="para_servidor_outlook_ELTPARAGRAPHE000283"></a>

La carpeta se encuentra en la rama "Carpetas personales".

Al llamar a la función **&lt;emailOutlookSession Variable&gt;.RemoveFolder**, las siguientes operaciones se realizan automáticamente:

- los mensajes encontrados en la carpeta a eliminar se trasladan a la papelera (carpeta"Elementos borrados").

- las subcarpetas de la carpeta a eliminar, así como su contenido, se trasladan a la papelera (carpeta"Elementos eliminados").

- la carpeta que desea eliminar se elimina de forma permanente.




Si la carpeta que desea eliminar ya se encuentra en la papelera, la carpeta y su contenido se eliminan de forma permanente.

**Recordatorio**: Esta función no se puede utilizar para eliminar una carpeta del software de mensajería Outlook Express.

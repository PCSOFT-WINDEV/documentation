


## HTTPCookieRead (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor de un Cookie recibido tras una petición HTTP.

**Observación**: Las cookies leídas y enviadas por [HTTPRequest](../WDLang3/3043007.md) sólo se almacenan en la memoria de la aplicación. No están relacionados con las cookies de los navegadores Web instalados en el equipo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Performs an HTTP request
HTTPRequest("www.mysite.com")
// Displays the identification cookie returned by the server
sID is string = HTTPCookieRead(".mysite.com", "ID")
Info("Identification cookie = " + sID)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Sintaxis que puede ser usada con httpRequest

`<Result> = HTTPCookieRead(<Domain> , <Name>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de Cookie.

- Cadena vacía ("") si el Cookie no existe. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.




**`<Domain>: (Cadena de caracteres)`**

Nombre del dominio para el que se recibió la Cookie. 

**`<Name>: (Cadena de caracteres)`**

Nombre de Cookie. 


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### La sintaxis que se puede usar con HTTPSend

`<Result> = HTTPCookieRead(<HTTP request> , <Domain> , <Name>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de Cookie.

- Cadena vacía ("") si el Cookie no existe. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.




**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre del [httpRequest](../WDLang3/1000021158.md) Variable para el que se leerá el Cookie.  

**`<Domain>: (Cadena de caracteres)`**

Nombre del dominio para el que se recibió la Cookie. 

**`<Name>: (Cadena de caracteres)`**

Nombre de Cookie. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para utilizar **HTTPCookieRead**, la gestión de las cookies debe haber sido habilitada por [HTTPCookieManage](../WDLang3/1000019218.md).

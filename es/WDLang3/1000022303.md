


## &lt;emailNotesSession Variable&gt;.StartSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia una sesión de Email en un servidor de Lotus Domino para que manipular los mensajes. 

**Observación**:
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para iniciar una sesión con el software de mensajería de Lotus Notes, utilice [EmailStartNotesSession](../WDLang3/3032121.md).
<a name="sample_code"></a>

## Ejemplo

<a name="3032028_Example1"></a>

```wl
// Starts an email session 
MyNotesSession is emailNotesSession
IF MyNotesSession.StartSession() = True THEN
	// Connected
ELSE
	UserName = ""
	Error("Unable to establish the connection")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Session>.OpenSession()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la sesión se abrió correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. 
	Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailNotesSession)`**

Nombre de la [emailNotesSession](../WDLang3/1000018768.md) Variable correspondiente a la sesión de correo electrónico a manipular.

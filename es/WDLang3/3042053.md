


## grPiePullOut (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Saca una sección (o todas las secciones) de un gráfico.. Esta función se puede utilizar en las cartas de pastel, en las cartas de donuts y en las cartas semicirculares.

Ejemplo de sección extraíble en un gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grSecteurPartExcentr%E9e%20-%20HC%20N%B0002.gif)


**Observación**: Cada sección de la carta puede ser extraída independientemente de las otras.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Pull out the second section by 10 pixels in "MyChart"
// (Chart created through programming)
grPiePullOut("MyChart", 2, 10)
```
<a name="Example2"></a>

```wl
// Pull out the third section by 20 pixels in "CHART_MyChart"
grPiePullOut(CHART_MyChart, 3, 20)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Sacar una sección

`grPiePullOut(<Chart name> , <Section subscript> , <Offset>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Section subscript>: (Integro)`**

Índice de la sección a extraer.

**`<Offset>: (Integro)`**

Desplazamiento de la sección especificada (en píxeles).  


<a name="SYNTAX2"></a>

### Sacando todas las secciones

`grPiePullOut(<Chart name> , <Offset>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Offset>: (Integro)`**

Desplazamiento de diferentes secciones (en píxeles).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La extracción de una sección de cartas no tiene ningún efecto sobre las cartas aparte de las cartas Pie, las cartas Donut y las cartas Semicirculares..
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Las cartas semicirculares no están disponibles.

- Las modificaciones realizadas con la función **grPiePullOut** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

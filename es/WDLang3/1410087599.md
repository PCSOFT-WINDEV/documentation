
## WidgetProcedure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica la Procedure que se llamará cuando el usuario haga clic en un widget de iOS 14. Esta función debe ser utilizada en la "Inicialización" de Event associated con el proyecto, especificando el objetivo "iOS". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
<IF ExecutionTarget=iOS>
	WidgetProcedure(OnWidgetClick)
<END>
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WidgetProcedure(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback") llamado cuando se hace clic en un widget de iOS 14. Este procedimiento tiene el siguiente formato : 


```txt
PROCEDURE <Procedure name> (<Element name>)
```
donde &lt;Nombre del elemento&gt; corresponde a: 

- el nombre completo de la control pulsada (widgets medianos o grandes),

- el nombre de la ventana (widgets pequeños). 




Esta Procedure también puede ser una Procedure interna.




## &lt;HMDeleteColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los colores especificados por [&lt;HMColor](../WDLang3/1000024080.md) de un gráfico Heatmap. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the colors
CHART_Chart.HMDeleteColor()
CHART_Chart.HMColor(-65, RGB(0, 105, 255))
CHART_Chart.HMColor(0, RGB(240, 255, 0))
CHART_Chart.HMColor(35, RGB(255, 31, 0))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.HMDeleteColor([<Level>])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).


**`<Level>: (Real opcional)`**

Nivel de Color que será eliminado. Si no se especifica este parámetro, se borran todos los niveles de Color s y se utiliza la Color de la primera serie.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los datos modificados con la función **&lt;HMDeleteColor** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

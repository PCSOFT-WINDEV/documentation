


## grSourceCategoryLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define la fuente de las etiquetas de las categorías.


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003042072&name=grsourcecategorylabel_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
IF grSourceCategoryLabel("MyChart", grTableColumn, "MyTable.MyColumn") = True THEN
	Info("The source of the category labels is MyColumn")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grSourceCategoryLabel(<Chart name> , <Source> [, <Details 1> [... [, <Details N>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se definió la fuente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico.




**`<Source>: (Constante de tipo Integer)`**

Fuente de datos del gráfico:


|   |   |
| --- | --- |
| *grCallback* | Etiquetas de categoría llenadas por una función de WLanguage/Procedure. <br>En este caso, &lt;Detalles 1&gt; corresponde al nombre del WLanguage Procedure a llamar.<br><br>El Procedure se llama para cada número. Este procedimiento tiene el siguiente formato: <br><br><pre><code>PROCEDURE MyProcedure(<Chart name>, <br><Series number>, <Index in series>)</code></pre><br><br>La Procedure devuelve el número deseado. Para indicar el final de la serie, el procedimiento debe devolver <u><u><u><u>False</u></u></u></u>.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *grItem* | Etiquetas de categoría rellenadas a partir de un artículo.<br><br>&lt;Detalles 1&gt; corresponde al nombre del archivo de datos (o consulta), &lt;Detalles 2&gt; corresponde al nombre del artículo. La serie navegará a través de el archivo de datos y tomará todos los valores del artículo.<br><br>Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |
| *grListBox* | Etiquetas de categoría llenadas de un control List Box.<br><br>En este caso, &lt;Detalles 1&gt; corresponde al nombre de un control List Box. El control List Box se navega para llenar la serie. Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |
| *grProgramming* | Las etiquetas de las categorías se rellenan mediante programación.<br><br>&lt;Los detalles 1&gt; pueden corresponder a una serie de números en la misma cadena de caracteres, separados por ";". |
| *grTable* | Etiquetas de categorías rellenadas desde un WLanguage array.<br><br>&lt;Detalles 1&gt; corresponde al nombre de la array. &lt;Detalles 2&gt; puede ser el índice de la columna (array bidimensional ) o el nombre de un miembro (array de estructuras). |
| *grTableColumn* | Etiquetas de categoría rellenadas de una columna de una control Tabla. <br><br>En este caso, &lt;Detalles 1&gt; corresponde al nombre de una columna que se encuentra en un control Tabla. Este nombre debe corresponder al nombre completo de la columna, entre comillas (por ejemplo: "&lt;TableName&gt;.&lt;ColumnName&gt;"). <br>Se navega por la columna para llenar la serie. Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |



**`<Details 1>: (Tipo esperado (parámetro opcional))`**

Primer parámetro esperado, según el tipo de fuente de datos.



**`<Details N>: (Tipo esperado (parámetro opcional))`**

Último parámetro esperado, según el tipo de fuente de datos.

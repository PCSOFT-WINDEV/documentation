
## saasClient (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasClient** permite definir las características avanzadas de una cuenta cliente SaaS. 

Las características de la cuenta de cliente pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
C is saasClient
C.Name = "hybrid"

SaaSAdminAddClient(C)
```

<a name="NOTE0_1"></a>

## Observaciones


### Propiedades específicas de las variables saasClient
<a name="propiedades_especificas_las_variables_saasclient_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una cuenta de cliente SaaS:

| **Property** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre de la cuenta del cliente.<br>En la mayoría de los casos, el nombre de la empresa cliente, el nombre de la empresa o los nombres y apellidos de la persona física..<br>Este Property no puede estar vacío y debe ser único para cada SaaS Webservice. |
| Información | Cadena de caracteres | Diversa información.<br>A este Property sólo puede acceder el administrador del SaaS Webservice (la cuenta "admin"). |
| Referencia externa | Cadena de caracteres | El reference externo permite almacenar un enlace a un Sistema de Información externo al Servidor de Aplicación WEBDEV.<br>El contenido de esta Property es libre y no se requiere su uso. |

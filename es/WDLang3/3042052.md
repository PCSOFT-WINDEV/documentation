


## grGridlines (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra u oculta las líneas de la cuadrícula del gráfico.

Ejemplo de líneas de cuadrícula: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=quadrillage.gif)


**Recordatorio**: Las cuadrículas del gráfico son Invisible de Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the vertical gridlines in "MyChart"
grGridlines("MyChart", True, grYCoordinate)
```
<a name="Example2"></a>

```wl
// Display the horizontal gridlines in CHART_MyChart
grGridlines(CHART_MyChart, True, grXCoordinate)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grGridlines(<Chart name> , <Display> , <Axis direction>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la cuadrícula debe ser mostrada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Axis direction>: (Constante o combinación de constantes de tipo Integer)`**

Eje utilizado para las líneas de la cuadrícula: 



|   |   |
| --- | --- |
| *grXCoordinate*<br>(valor predeterminado) | Marcas en el eje X (eje horizontal). |
| *grYCoordinate* | Marcas en el eje Y (eje vertical). |


La combinación de estas dos constantes se utiliza para get una cuadrícula completa (vertical y horizontal).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se tienen en cuenta las líneas de la cuadrícula:

	- en los gráficos de columna (constantes *grColumn* y *grColumnStacked*), gráficos de dispersión (constante *grScatter*), Gráficos de líneas (constante *grLine*) y gráficos de existencias (constantes *grCandleStick*, *grBarCharts* y *grMinMax*) cuando &lt;La dirección del eje&gt; corresponde a *grXCoordinate*.

	- en los gráficos de dispersión (constante *grScatter*), de líneas (constante *grLine*) y de cotizaciones (constantes *grCandleStick* y *grBarCharts*) cuando el parámetro &lt;Dirección de eje&gt; corresponde a *grYCoordinate*.




- Las cuadrículas no tienen efecto en los gráficos de tarta (constante *grPie*) y en los gráficos de embudo (constante *grFunnel*).

- Los datos definidos con la función **grGridlines** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

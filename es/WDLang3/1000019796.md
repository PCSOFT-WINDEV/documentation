


## MapModifyMarker (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un marcador mostrado en un control Map568




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a marker: the icon changes when the user clicks above it
MyLocation is geoPosition
MyLocation.Latitude = 43.613708 
MyLocation.Longitude = 3.876972
MyMarker is Marker
MyMarker.Position = MyLocation
MyMarker.ActionClick = "ProcMarkerClick"

MapAddMarker(MAP_Position, MyMarker)
```

```wl
PROCEDURE ProcMarkerClick(AMarker is Marker)

AMarker.Image = "marker_click.png"
MapModifyMarker(MAP_Position, AMarker)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapModifyMarker(<Map control> , <Marker>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el marcador,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Marker>: (Marcador Variable)`**

Nombre de la [Marker](../WDLang3/1000019940.md) Variable correspondiente al Marker a modificar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para añadir una nueva Marker a una control Mapa, utilice [MapAddMarker](../WDLang3/1000019794.md).

- Para eliminar una Marker específica de una control Mapa, utilice [MapDeleteMarker](../WDLang3/1000019795.md).

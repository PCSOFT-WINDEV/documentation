## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
WINDEV, WEBDEV y WINDEV Mobile incluyen un control Gráfico (que se puede utilizar en el editor de ventanas, en el editor de páginas y en editor de reportes), así como varias funciones utilizadas para crear gráficos manipular..
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Uso de gráficos en WEBDEV Linux y en WINDEV Linux**
Para utilizar los gráficos en Linux, es necesario instalar la biblioteca QT (Core y GUI) versión 5 (o posterior). 

**En Ubuntu**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```

## ¿Cómo programar los gráficos?
<a name="¿como_programar_los_graficos_ELTTEXTE000149"></a>


### Creación de gráficos mediante programación
<a name="creacion_graficos_mediante_programacion_ELTPARAGRAPHE000172"></a>

WINDEV, WEBDEV y WINDEV Mobile incluyen varias funciones para manejar gráficos. Estas funciones permiten crear los gráficos, modificarlos, cambiar las opciones de visualización....

Las siguientes operaciones se utilizan para crear los gráficos: 

1. Cree el gráfico con [grCreate](../WDLang3/3042005.md).

2. Inicialice los datos del gráfico con las siguientes funciones:

	- [.AddData](../WDLang3/1000023533.md).

	- [.ScatterAddDataXY](../WDLang3/1000024105.md) para los gráficos Scatter.

	- [.StockAddData](../WDLang3/1000023565.md) para los gráficos de acciones.

	- [&lt;Tabla&gt;.SurfaceAddData](../WDLang3/1000024143.md) para las cartas de Superficie. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [&lt;Gráfico&gt;.SunburstAddData](../WDLang3/1000024132.md) para las cartas de Sunburst.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [&lt;Gráfico&gt;.HMAddData](../WDLang3/1000024079.md) para los gráficos de Heatmap.




3. Seleccione el destino de visualización del gráfico con las siguientes funciones:

	- [grDestinoControl](../WDLang3/3042020.md): para visualizar el gráfico en un control Imagen

	- [grDestinoWnd](../WDLang3/3042031.md): mostrar el gráfico en una ventana independiente
			

	- [grDestinationBMP](../WDLang3/3042018.md), [grDestinationEMF](../WDLang3/3042017.md) o [grDestinationWMF](../WDLang3/3042019.md) para guardar el gráfico en archivos en formato BMP, EMF o WMF respectivamente. 

	- [grDestinationImage](../WDLang3/1000020952.md): para guardar el gráfico en una [Image](../WDLang1/1000019650.md) Variable directamente. 




4. Dibuja el gráfico con [&lt;Dibujo](../WDLang3/1000024062.md).




Se utilizan varias funciones para configurar y modificar la apariencia del gráfico. Tiene la posibilidad de mostrar una leyenda, el título de los ejes, ... Para más detalles, consulte la lista de [funciones para la gestión de gráficos](../WDLang3/1000024173.md).

### Gráficos e hilos
<a name="graficos_hilos_ELTPARAGRAPHE000277"></a>

Un solo gráfico puede ser manejado en un Thread.

Consecuencias:

- Dos hilos no pueden hacer manipular el mismo gráfico.

- Un gráfico creado en una Thread se destruye automáticamente al final de esta Thread. El gráfico existe en el hilo que contiene la llamada a [grCreate](../WDLang3/3042005.md).

### Valores Default de un gráfico (en una ventana o en un Page)
<a name="valores_default_grafico_una_ventana_page_ELTPARAGRAPHE000297"></a>

Los valores de Default de un gráfico son los siguientes:

- Menú contextual habilitado (configurable por [&lt;Gráfico&gt;.menú](../WDLang3/1000024094.md) y [&lt;Opción de menú](../WDLang3/1000024112.md)).

- Marcas automatic en los ejes (configurables por [.interval](../WDLang3/1000024076.md)).

- Los colores se seleccionan según un orden preset entre un set de 13 colores (configurable por [&lt;Gráfico&gt;.SeriesColor](../WDLang3/1000023599.md)).

- Los gradientes color están disponibles (pueden ser configurados por [&lt;Tabla&gt;.Gradient](../WDLang3/1000024061.md)).

- Al pasar el ratón por un gráfico, se muestra automáticamente una información sobre la herramienta ([&lt;Tabla de herramientas](../WDLang3/1000024163.md)).

- Tamaño del archivo (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).

- Tamaño de la ventana (configurable por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md)).

- No hay etiqueta para la serie (puede ser configurada por [&lt;Tabla&gt;.SeriesLabel](../WDLang3/1000024073.md)).

- No hay leyenda para los elementos (puede ser configurada por [.CategoryLabel](../WDLang3/1000024072.md) y [&lt;Tabla&gt;.etiqueta](../WDLang3/1000024165.md)).

- Sin leyenda (puede ser configurada por [&lt;Leyenda](../WDLang3/1000024089.md)).

- No hay título para el gráfico (puede ser configurado por [&lt;Gráfico&gt;.título](../WDLang3/1000024166.md)).

- No hay título para los ejes (puede ser configurado por [&lt;Gráfico&gt;.AxisTitle](../WDLang3/1000024153.md)).

- No hay líneas de cuadrícula (puede ser configurado por [&lt;Gráfico&gt;.GridLines](../WDLang3/1000024121.md)).

- Sin datos (los elementos pueden ser añadidos por [.AddData](../WDLang3/1000023533.md)).

- Sin destino (puede ser configurado por [grDestinoControl](../WDLang3/3042020.md), [grDestinoWnd](../WDLang3/3042031.md) y [grDestinationWMF](../WDLang3/3042019.md)).

- No hay visualización de gráficos (para dibujar el gráfico, utilice [&lt;Dibujo](../WDLang3/1000024062.md)).

- Sin efecto de elevación (configurable por [&lt;Gráfico&gt;.levantado](../WDLang3/1000024123.md)).

- Para los gráficos de tarta, el ángulo inicial es igual a 0 (puede ser configurado por [&lt;Gráfico&gt;.PieStartAngle](../WDLang3/1000024129.md)).

- Para los gráficos de barras y los gráficos MinMax, el espacio entre dos categorías es igual a 10 píxeles (puede ser configurado por [&lt;Espaciado de Columnas](../WDLang3/1000024078.md)).

- Para los gráficos de dispersión, los puntos son relacionado y su tamaño es igual a 1 (puede ser configurado por [&lt;Gráfico&gt;.ScatterLinkPoint](../WDLang3/1000024109.md) y [&lt;Gráfico&gt;.ScatterPointSize](../WDLang3/1000024110.md)).

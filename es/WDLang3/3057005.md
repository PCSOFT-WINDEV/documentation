
## OLECreateCopy( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Asocia una control OLE con un objeto OLE especificando un archivo base. **OLECreateCopy** inicia el servidor OLE que creó el archivo. Se edita una copia del archivo. El servidor a ejecutar se deduce de la extensión del archivo base (Paint Brush para la extensión BMP, WinWord para la extensión DOC, etc.).. Durante la salida del servidor, el objeto se dibuja automáticamente en la OLE control de la ventana WINDEV. **Esta función se utiliza para modificar un objeto OLE.**

**Observación**: Dependiendo del servidor que haya creado el objeto OLE, el objeto puede ser visualizado en la control como:

- su representación de mapa de bits (ejemplo de Image creado con Paint Brush),

- su representación vectorial (ejemplo de hoja de cálculo creada por Excel),

- el Icon que representa el servidor (Icon creado por WinWord),

- texto.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The Castle.bmp image was created by Paint Brush.
// Start Paint Brush with the Castle.bmp image loaded.
OLECreateCopy(OLE_OLE1, "C:\Windows\Castle.bmp")
// Process
// Save
OLESave(OLE_OLE1, "OBJECT.OLE")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OLECreateCopy(<OLE control> , <File name> [, <Locks>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó la función,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<OLE control>: (Nombre del control)`**

Nombre de OLE control que recibirá el objeto OLE. Si la control contenía un objeto, éste se sustituye por el nuevo objeto. Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control al que pertenece el actual Event.

**`<File name>: (Cadena de caracteres)`**

Nombre y ruta (si es necesario) del archivo base (con su extensión). Este archivo puede ser cualquier archivo creado con un servidor OLE (imagen BMP de Paint Brush, hoja de cálculo .XLS, .DOC Word Document, etc.).

**`<Locks>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado): Servidor de bloqueo.
	La ventana que contiene el OLE control no puede estar en edición mientras el usuario no haya salido del servidor.

- <u><u><u><u>False</u></u></u></u>: Servidor sin bloqueo.
	El usuario puede realizar entradas en la ventana que contiene el OLE control mientras el servidor está todavía iniciado. Se utiliza para jugar un verbo con [OLEPlay](../WDLang3/3057009.md).

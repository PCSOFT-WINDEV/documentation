
## awSendAndGetInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía datos de un iOS Phone a un Apple Watch (o al contrario) mientras espera una respuesta. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
INTERNAL PROCEDURE GetInfo(b is Buffer)
	Info(b)
END

MyBuffer is Buffer = "Text buffer"
IF NOT awSendAndGetInfo(MyBuffer, GetInfo) THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = awSendAndGetInfo(<Data> , <WLanguage procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el envío se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<Data>: (Buffer)`**

Datos a enviar.

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLanguage Procedure ("callback") llamada cuando se recibe la respuesta. Este procedimiento se utiliza para recuperar y Process la respuesta.
Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure name>(<Response>)
```

donde &lt;Respuesta&gt; es un Buffer Variable.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El receptor recibe los datos enviados a través de la Procedure pasada a [awProcedureGetAndSendInfo](../WDLang3/1000022118.md). Entonces, el receptor debe enviar una respuesta en valor de retorno de este Procedure. 

- Los datos se envían inmediatamente. 

	- De"Reloj a Phone": si la aplicación iOS no se inicia, se iniciará automáticamente en Background para responder a la llamada. 

	- De "Phone to Watch": si no se inicia la aplicación Apple Watch, se genera un error por la llamada. 




- Los mensajes se entregan en el orden en que se han enviado.

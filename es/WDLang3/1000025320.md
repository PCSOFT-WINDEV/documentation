
## GetTheme (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer el tema actual de una aplicación móvil. Se pueden gestionar casos especiales (por ejemplo, seleccionar una imagen específica según el tema actual) mediante el evento "Aplicación del tema claro/oscuro". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Event "Application of light/dark theme"
IF GetTheme() = stThemeDark THEN
	IMG_BackgroundImage = "DarkBackground.png"
ELSE 
	IMG_BackgroundImage = "LightBackground.png"
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GetTheme()`
---

**`<Result>: (Constante de tipo Integer)`**

Tema utilizado actualmente por la aplicación: 



|   |   |
| --- | --- |
| *stThemeDark* | La aplicación utiliza el tema oscuro. |
| *stThemeLight* | <br><br>- La aplicación utiliza el tema claro.<br><br>- La gestión del tema claro/oscuro no está activada en la aplicación (opción "Permitir cambiar al tema oscuro (automáticamente o manualmente con la función ChangeTheme)" en la pestaña "Estilo" de la descripción del proyecto). <br><br><br> |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para modificar el tema de la aplicación, utilice la función [ChangeTheme](../WDLang3/1000025266.md).

- Para conocer el tema activo en el dispositivo móvil, utilice la función [SysTheme](../WDLang1/1000019646.md).


## SaaSDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desconecte al usuario de un sitio SaaS además del servicio web SaaS que gestiona el sitio.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SaaSConnect("login", "password") = True THEN
	IF SaaSDisconnect() = True THEN
		Info("Disconnection successful")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSDisconnect()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se utiliza varias veces la función **SaaSDisconnect** (sin reconexión mediante la función [SaaSConnect](../WDLang3/1000018959.md)), &lt;Resultado&gt; también se establece en <u><u><u><u>False</u></u></u></u>.

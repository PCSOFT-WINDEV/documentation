
## Uso de la Wi-Fi
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000157"></a>
WINDEV y WINDEV Mobile proponen las funciones WiFi para acceder fácilmente a las redes Wi-Fi. Este grupo de funciones puede utilizarse tanto en las aplicaciones WINDEV como en las aplicaciones Android. 





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Wi-Fi y WINDEV: Principio
<a name="wifi_windev_principio_ELTTEXTE000181"></a>
El principio para utilizar las funciones WiFi es el siguiente: 

1. Lista de las tarjetas de red Wi-Fi accesibles desde el ordenador actual ([Tarjeta WiFiListNetworkCard](../WDLang3/1000020693.md)). 

2. Seleccione la tarjeta de red Wi-Fi a utilizar ([Tarjeta WiFiSelectNetworkCard](../WDLang3/1000020694.md)). 

3. Detectar los puntos de acceso Wi-Fi disponibles con [WiFiDetectAccessPoint](../WDLang3/1000019353.md). 

4. Lista de las redes Wi-Fi configuradas en el equipo ([WiFiListNetwork](../WDLang3/1000019356.md)). 

5. Si se encuentra un punto de acceso Wi-Fi disponible en la lista de redes Wi-Fi almacenadas, tiene la posibilidad de conectarse a él ([WiFiConnect](../WDLang3/1000019352.md)). 




Las funciones disponibles son las siguientes: 



|   |   |
| --- | --- |
| [WiFiConnect](../WDLang3/1000019352.md) | Conecta el dispositivo a la red Wi-Fi especificada. |
| [WiFiConnectionInfo](../WDLang3/1000019355.md) | Devuelve la información solicitada sobre la conexión Wi-Fi actual del dispositivo. |
| [WiFiDeleteNetwork](../WDLang3/1000019357.md) | Elimina una red de la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiDetectAccessPoint](../WDLang3/1000019353.md) | Inicia la detección de los puntos de acceso Wi-Fi a los que se puede acceder desde el dispositivo. |
| [WiFiListNetwork](../WDLang3/1000019356.md) | Devuelve la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiListNetworkCard](../WDLang3/1000020693.md) | Devuelve la lista de las tarjetas de red Wi-Fi disponibles actualmente en el ordenador actual. |
| [WiFiSelectNetworkCard](../WDLang3/1000020694.md) | Selecciona la tarjeta de red Wi-Fi que se utilizará para las próximas llamadas a las funciones WiFi. |

## Wi-Fi y Android: Principio
<a name="wifi_android_principio_ELTTEXTE000256"></a>
El principio para utilizar las funciones WiFi es el siguiente: 

1. Compruebe el estado del Wi-Fi en el dispositivo móvil ([WiFiStatus](../WDLang3/1000019354.md)) y habilítelo si es necesario ([WiFiActivate](../WDLang3/1000019350.md)). Si se encuentran varias redes Wi-Fi disponibles durante la activación, el dispositivo se conectará a la mejor red Wi-Fi. 

2. Detectar los puntos de acceso Wi-Fi ([WiFiDetectAccessPoint](../WDLang3/1000019353.md)). Esto le permite conocer todas las redes Wi-Fi accesibles. 

3. Lista de las redes Wi-Fi guardadas ([WiFiListNetwork](../WDLang3/1000019356.md)). 

4. Si se encuentra un punto de acceso Wi-Fi disponible en la lista de redes guardadas, tiene la posibilidad de conectarse a él ([WiFiConnect](../WDLang3/1000019352.md)). 

5. Si un punto de acceso Wi-Fi disponible no se encuentra en la lista de redes guardadas, tiene la posibilidad de configurarlo ([WiFiAddNetwork](../WDLang3/1000019351.md)) y de conectarse a él ([WiFiConnect](../WDLang3/1000019352.md)).




Las funciones disponibles son las siguientes: 



|   |   |
| --- | --- |
| [WiFiActivate](../WDLang3/1000019350.md) | Habilita o deshabilita Wi-Fi en el dispositivo. |
| [WiFiAddNetwork](../WDLang3/1000019351.md) | Añade una nueva red a la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiConnect](../WDLang3/1000019352.md) | Conecta el dispositivo a la red Wi-Fi especificada. |
| [WiFiConnectionInfo](../WDLang3/1000019355.md) | Devuelve la información solicitada sobre la conexión Wi-Fi actual del dispositivo. |
| [WiFiDeleteNetwork](../WDLang3/1000019357.md) | Elimina una red de la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiDetectAccessPoint](../WDLang3/1000019353.md) | Inicia la detección de los puntos de acceso Wi-Fi a los que se puede acceder desde el dispositivo. |
| [WiFiListNetwork](../WDLang3/1000019356.md) | Devuelve la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiStatus](../WDLang3/1000019354.md) | Devuelve el estado actual de activación de Wi-Fi en el dispositivo o solicita que se le notifique cuando cambie el estado de activación de Wi-Fi. |

## Wi-Fi e iOS: Principio
<a name="wifi_ios_principio_ELTTEXTE000338"></a>
En las aplicaciones iOS, las funciones Wi-Fi se utilizan para: 

- Descubre el estado de activación de Wi-Fi. 

- get la lista de redes Wi-Fi. Las funciones disponibles son las siguientes: 





|   |   |
| --- | --- |
| [WiFiListNetwork](../WDLang3/1000019356.md) | Devuelve la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiStatus](../WDLang3/1000019354.md) | Devuelve el estado actual de activación de Wi-Fi en el dispositivo o solicita que se le notifique cuando cambie el estado de activación de Wi-Fi. |

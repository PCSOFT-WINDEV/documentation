


## grMenuOpción (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el texto de una opción del menú contextual de un gráfico.

**Observación**: 

- El menú contextual es activado por Default. Para modificar el estado del menú contextual, utilice [grMenu](../WDLang3/3042035.md).

- Esta función sólo se puede utilizar si los menús del gráfico están en modo de compatibilidad (activado con la constante *grMenuForCompatibility* de la función [grMenu](../WDLang3/3042035.md)).  



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the line 50 in the context menu of "MyChart"
grMenuOption("MyChart", 50, "Print the chart")
```
<a name="Example2"></a>

```wl
// Modify the line 50 in the context menu of CHART_MyChart
grMenuOption(CHART_MyChart, 50, "Print the chart")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grMenuOption(<Chart name> , <Option identifier> , <Text>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico definido por [grCreate](../WDLang3/3042005.md) o nombre del control Gráfico en el editor de ventanas.

**`<Option identifier>: (Integro)`**

Identificador de la opción en el menú contextual.

**`<Text>: (Cadena de caracteres)`**

Nuevo texto para la opción de menú especificada. El menú especificado Line está desactivado si este parámetro corresponde a una cadena vacía (""): esta Line no se mostrará más en el menú.

### Activado/Desactivado Line
<a name="activadodesactivado_line_ELTPARAGRAPHE000346"></a>

- Para desactivar una opción de menú, utilice la función **grMenuOpción** y especifique una cadena vacía ("") en el parámetro &lt;Texto&gt;.

- Para reactivar una opción de menú desactivada con la función **grMenuOpción**, llame a esta función con el mismo identificador de menú pero con un título no vacío.

- Los datos definidos con la función **grMenuOpción** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).



<a name="NOTE0_3"></a>


### Idiomas
<a name="idiomas_ELTPARAGRAPHE000366"></a>

- El menú contextual está disponible en dos idiomas (inglés y francés). No se puede añadir ningún idioma.

- El texto se modifica en el idioma actual. Para gestionar los idiomas extranjeros, modifique los textos predefinidos.

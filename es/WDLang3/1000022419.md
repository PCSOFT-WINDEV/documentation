
## Funciones para la gestión de los controles del Mapa (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de Mapas mediante la sintaxis de los prefijos: 



|   |   |
| --- | --- |
| [&lt;GetPosition](../WDLang3/1000022463.md) | Devuelve la posición geográfica de un punto localizado en el centro del mapa que se está mostrando en un control Map. |
| [&lt;Mapa&gt;.AddImage](../WDLang3/1000025802.md) | Añade una nueva Image al mapa que se muestra en una control Mapa. |
| [&lt;Mapa&gt;.AddItinerario](../WDLang3/1000022420.md) | Añade un itinerario a un control Mapa. |
| [&lt;Mapa&gt;.AddMarker](../WDLang3/1000022422.md) | Añade un nuevo marcador sobre un mapa mostrado en un control Map. |
| [&lt;Mapa&gt;.AddShape](../WDLang3/1000025801.md) | Añade una nueva forma (Polygon, Line, círculo) al mapa que se muestra en un control Mapa. |
| [&lt;Mapa&gt;.Área de visualización](../WDLang3/1000025800.md) | Mueve el mapa que se muestra en el control Mapa para que el área geográfica especificada se pueda ver en su totalidad. |
| [&lt;Mapa&gt;.BorrarForma](../WDLang3/1000025820.md) | Borra una forma dibujada en el mapa que se muestra en una control Mapa. |
| [&lt;Mapa&gt;.BorrarImagen](../WDLang3/1000025829.md) | Borra una Image dibujada en el mapa que se muestra en una control Mapa. |
| [&lt;Mapa&gt;.ChangeStyle](../WDLang3/1000025797.md) | Cambia el estilo del mapa que se muestra en el control Mapa. |
| [&lt;Mapa&gt;.ClosePopup](../WDLang3/1000025803.md) | Cierra la popup de un marcador que se muestra en el control Mapa. |
| [&lt;Mapa&gt;.DeleteAll](../WDLang3/1000022472.md) | Borra lo siguiente de una control Mapa: <br><br>- todos los marcadores, <br><br>- todas las formas,<br><br>- todas las imágenes.<br><br><br> |
| [&lt;Mapa&gt;.DeleteItinerario](../WDLang3/1000022465.md) | Borra un itinerario de una control Mapa. |
| [&lt;Mapa&gt;.DeleteMarker](../WDLang3/1000022466.md) | Elimina una Marker de un mapa mostrado por una control Mapa. |
| [&lt;Mapa&gt;.DisplayPopup](../WDLang3/1000025799.md) | Abre la ventana emergente de una Marker mostrada en una control Mapa. |
| [&lt;Mapa&gt;.DisplayPosition](../WDLang3/1000022394.md) | Centra el mapa visualizado en una control Mapa en una posición geográfica o en un lugar concreto. |
| [&lt;Mapa&gt;.EndOfMove](../WDLang3/1000022437.md) | Se detiene siguiendo el movimiento del dispositivo en una control Mapa. |
| [&lt;Mapa&gt;.ExecuteJS](../WDLang3/1000022384.md) | Ejecuta el código nativo de Google Maps API en un control Mapa en JavaScript. |
| [&lt;Mapa&gt;.FollowMovement](../WDLang3/1000022464.md) | Muestra la ubicación actual del dispositivo en una control Mapa y actualiza la ubicación a medida que se mueve. |
| [&lt;Mapa&gt;.GetImage](../WDLang3/1000022462.md) | Recupera la Image de un mapa mostrado en una control Mapa. Los marcadores y los itinerarios mostrados en este mapa también se encontrarán en la Image. |
| [&lt;Mapa&gt;.GetJSObject](../WDLang3/1000022385.md) | Devuelve la instancia de la API de control Mapa. |
| [&lt;Mapa&gt;.GetStyle](../WDLang3/1000025798.md) | Devuelve el estilo utilizado por el mapa que se muestra en la control Mapa. |
| [&lt;Mapa&gt;.InfoPosición](../WDLang3/1000022447.md) | Devuelve las coordenadas (en píxeles) del punto correspondiente a una posición geográfica (latitud y longitud). |
| [&lt;Mapa&gt;.InfoXY](../WDLang3/1000022448.md) | Devuelve la posición geográfica (latitud y longitud) correspondiente a un punto de una control Mapa. |
| [&lt;Mapa&gt;.ItinerarioDistancia](../WDLang3/1000022431.md) | Calcula la distancia de un itinerario en una control Mapa. |
| [&lt;Mapa&gt;.LimitArea](../WDLang3/1000025804.md) | Limita el área geográfica que se muestra en el control Mapa. |
| [&lt;Mapa&gt;.ModificarImagen](../WDLang3/1000025818.md) | Modifica una Image dibujada en el mapa que se muestra en una control Mapa. |
| [&lt;Mapa&gt;.ModifyShape](../WDLang3/1000025805.md) | Modifica una forma dibujada en el mapa que se muestra en una control Mapa. |
| [&lt;Mapa&gt;.PopupDisplayed](../WDLang3/1000025819.md) | Indica si se muestra la popup de un marcador en un control Mapa. |
| [&lt;ModificarMarcador](../WDLang3/1000022383.md) | Modifica un marcador mostrado en un control Map568 |





Los tipos de variables que manipula el control Mapa o una de sus funciones son los siguientes: 



|   |   |
| --- | --- |
| [MapCircle](../WDLang3/1000025504.md) | El tipo **MapCircle** se utiliza para definir las características de un círculo que se dibujará en el mapa que se muestra en una control Mapa. |
| [MapImage](../WDLang3/1000025523.md) | El tipo **MapImage** se utiliza para definir las características de un Image que se dibujará en el mapa que se muestra en un control Mapa. |
| [MapPolygon](../WDLang3/1000025476.md) | El tipo **MapPolygon** se utiliza para definir las características de un Polygon que se dibujará en el mapa que se muestra en un control Mapa. |
| [MapPolyline](../WDLang3/1000025489.md) | El tipo **MapPolyline** se utiliza para definir las características de una Line compuesta de varios segmentos que se dibujarán en el mapa que se muestra en una control Mapa. |
| [Marcador](../WDLang3/1000019940.md) | El tipo **Marker** permite definir las características de un marcador ubicado en un mapa que se muestra en un control Mapa. |
| [MarkerImage](../WDLang3/1000025796.md) | El tipo **MarkerImage** se utiliza para personalizar la imagen asociada a un cluster Marker o Marker que se muestra en un control Mapa. |

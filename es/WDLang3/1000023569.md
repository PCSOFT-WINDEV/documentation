


## &lt;Gráfico&gt;.BarColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa el color de una barra específica en un gráfico de tipo histograma o un histograma apilado. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes bar 2 of series 1
CHART_MyColumnChart.BarColor(1, 2, RGB(200,150,45))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.BarColor(<Series number> , <Category number> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie a utilizar.

**`<Category number>: (Integro)`**

Número de la categoría a utilizar.

**`<Color>: (Entero o constante de tipo Integer)`**

Bar Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [y la preselección Color](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para modificar la Color de todos los bares encontrados en una categoría, utilice [&lt;Gráfico&gt;.CategoryColor](../WDLang3/1000023570.md).


## BTConnectDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite ser autenticado hacia un dispositivo Bluetooth.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Bluetooth connection with code 123
// COL_ID is the Table control column used to list the devices
IF BTConnectDevice(COL_ID, "123") = True THEN
	// OBEX connection to the device according to the MAC address of the device
	nIDOBEXConnection = OBEXConnect(obexBluetooth, COL_MACAddress)
	...
	// OBEX disconnection
	OBEXDisconnect(nIDOBEXConnection)
	// Bluetooth disconnection
	BTDisconnectDevice(COL_ID)
ELSE
	// The Bluetooth connection failed
	Error("Unable to connect to the Bluetooth device", ErrorInfo(errFullDetails))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTConnectDevice(<Device identifier> , <Password> [, <Radio identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la autenticación fue exitosa,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Device identifier>: (Integro)`**

Identificador del dispositivo Bluetooth a utilizar. Este identificador es devuelto por [BTListDevice](../WDLang3/1000017132.md).

**`<Password>: (Cadena de caracteres)`**

Contraseña utilizada para la autenticación. Esta contraseña debe ser idéntica a la introducida en el dispositivo Bluetooth.

**`<Radio identifier>: (Entero opcional)`**

Identificador de la radio Bluetooth a utilizar. Este identificador es devuelto por [BTListRadio](../WDLang3/1000017125.md). Si no se especifica este parámetro, la elección de la radio se realiza automáticamente.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta versión sólo es compatible con las pilas bluetooth de Microsoft, Bluesoleil y Broadcom (Widcomm). Para conocer la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md). 

Las pilas Broadcom (Widcomm) no están disponibles en 64 bits.

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

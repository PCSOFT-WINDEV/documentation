


## &lt;httpRequest Variable&gt;.WriteCookie (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade o modifica una Cookie en una petición HTTP ejecutada con [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md). 




<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Solicitud ejecutada por HTTPSend

`<HTTP request>.WriteCookie(<Domain> , <Name> , <Value>)`
---

**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre de [httpRequest](../WDLang3/1000021158.md) Variable para el que la cookie será set.

**`<Domain>: (Cadena de caracteres)`**

Dominio de validez de Cookie.

El Cookie se enviará a todas las solicitudes HTTP cuyo dominio termine con el dominio de validez de Cookie.

Por ejemplo, un Cookie cuyo dominio de validez es ".midominio.fr" será enviado durante las siguientes solicitudes:

- http://mydomain.fr

- http://www.mydomain.fr/index.html

- http://server.mydomain.fr/Page.awp


No se enviará en las siguientes solicitudes:

- http://www.mydomain.com

- http://otherdomain.fr




**`<Name>: (Cadena de caracteres)`**

Nombre de Cookie.

Este nombre no distingue entre mayúsculas y minúsculas.

**`<Value>: (Cadena de caracteres)`**

Valor de Cookie.

Observaciones:

- El valor de un Cookie está limitado a 4KB. Más allá de este tamaño, el servidor es libre de rechazar el Cookie.

- Los caracteres de control (TAB, CR, etc.) así como el punto y coma y los ceros binarios no están permitidos.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si la gestión de las cookies no fue habilitada por [&lt;httpRequest Variable&gt;.ManageCookie](../WDLang3/1000021608.md), las cookies no se envían al servidor durante la llamada a [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md).

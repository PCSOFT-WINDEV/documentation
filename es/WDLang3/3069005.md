


## SOAPError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el error del último Jabón Procedure que falló.

**Observación**: Esta Procedure fue dirigida por [SOAPRun](../WDLang3/3069013.md) o [SOAPRunXML](../WDLang3/3069014.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003069005&name=soaperror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Returns the error message of the last SOAP procedure run
ResErrorMessage = SOAPError(SOAPErrMessage)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SOAPError(<Type of error>)`
---

**`<Result>: (Cadena de caracteres)`**



- Error del último Jabón Procedure que falló.

- Una cadena vacía (""") si:

	- el servidor SOAP no ha devuelto ningún error,

	- el servidor SOAP no estableció ninguna conexión.







**`<Type of error>: (Constante)`**

Tipo de error a devolver:


|   |   |
| --- | --- |
| *SOAPErrActor* | URL del servidor SOAP que activó la Error( útil si la Procedure es ejecutada por varios servidores SOAP). |
| *SOAPErrCode* | Error de la última corrida de SOAP Procedure:<br><br>- "VersionMismatch": La versión SOAP utilizada en el ordenador actual difiere de la utilizada en el servidor SOAP.<br><br>- "Debe ser de las afueras.": El servidor SOAP no entiende el parámetro especificado en el encabezado (solo si el parámetro &lt;Ejecución de procedimiento&gt; del encabezado se inicializó en <u><u><u><u>True</u></u></u></u>).<br><br>- "Cliente": Configuración inválida en relación con la ejecución de la Procedure.<br><br>- "Server": El servidor de SOAP no ejecutó el Procedure.<br><br><br>Observación: Esta cadena de caracteres se puede rellenar con información adicional. Se utiliza el siguiente formato: "&lt;Error&gt;.&lt;Información&gt;". Por ejemplo: "Autenticación de cliente" si el error se debe a la autenticación del cliente. |
| *SOAPErrDetails* | Detalles del error. Este mensaje está en formato XML. |
| *SOAPErrMessage* | Mensaje de error devuelto por el servidor de SOAP en la última ejecución de SOAP Procedure. |




## Funciones específicas para la comunicación iOS - Apple Watch
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones WLanguage específicas para la comunicación entre las aplicaciones del iOS y las aplicaciones del Apple Watch de associated son las siguientes: 



|   |   |
| --- | --- |
| [awProcedureGetAndSendInfo](../WDLang3/1000022118.md) | Especifica el WLanguage Procedure que se llamará cuando se reciba información procedente de [awSendAndGetInfo](../WDLang3/1000022116.md). |
| [awProcedureGetInfo](../WDLang3/1000022119.md) | Especifica el WLanguage Procedure que se llamará cuando se reciba información procedente de [awSendInfo](../WDLang3/1000022117.md). |
| [awSendAndGetInfo](../WDLang3/1000022116.md) | Envía datos de un iOS Phone a un Apple Watch (o al contrario) mientras espera una respuesta. |
| [awSendInfo](../WDLang3/1000022117.md) | Envía una información de un iOS Phone a un Apple Watch (o a la inversa). |

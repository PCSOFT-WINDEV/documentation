


## Funciones de archivo (gestión de ZIP, WDZ, CAB, RAR, 7z, TAR y TGZ (TAR.GZ)) (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>










Las siguientes funciones de WLanguage se utilizan para crear y gestionar archivos (sintaxis de prefijo):




|   |   |
| --- | --- |
| [&lt;zipArchive Variable&gt;.Merge](../WDLang3/1000019151.md) | Fusiona los diferentes subarchivos que se encuentran en un archivo de varias partes para crear un archivo de una sola parte. |
| [&lt;zipArchive Variable&gt;.NbPartNeed](../WDLang3/1000019094.md) | Devuelve el número de piezas de un tamaño determinado necesarias para dividir un archivo de una sola pieza. |
| [&lt;zipArchive Variable&gt;.OpenCAB](../WDLang3/1000019148.md) | Abre un archivo existente en formato CAB. |
| [.AddDirectory](../WDLang3/1000019078.md) | Añade todos los archivos encontrados en un directorio y en sus subdirectorios a un archivo y los comprime. |
| [.AddFile](../WDLang3/1000019050.md) | Añade un archivo (de cualquier tipo) a un archivo comprimido en formato CAB, ZIP, WDZ y 7z, TAR o TGZ (TAR.GZ) y lo comprime. |
| [.AddFileList](../WDLang3/1000019159.md) | Añade una lista de archivos (de cualquier tipo) a un archivo comprimido en formato ZIP, CAB, WDZ o 7z y lo comprime. |
| [.ChangePath](../WDLang3/1000026329.md) | Modifica el ruta almacenado de un archivo en un archivo (en formato ZIP, WDZ o 7z). |
| [.Create](../WDLang3/1000019160.md) | Crea un archivo. |
| [.CreateExe](../WDLang3/1000019079.md) | Crea un archivo autoextraíble como un ejecutable de 32 bits (.EXE). |
| [.CurrentFile](../WDLang3/1000019090.md) | Devuelve el nombre del fichero archivado que actualmente gestionan las funciones para añadir y extraer ficheros. |
| [.DeleteAll](../WDLang3/1000019154.md) | Elimina todos los archivos de un archivo comprimido. |
| [.DeleteFile](../WDLang3/1000019152.md) | Elimina un archivo de un archivo comprimido. |
| [.DeleteFileList](../WDLang3/1000019153.md) | Elimina una lista de archivos de un archivo comprimido. |
| [.Exist](../WDLang3/1000026332.md) | Determina si existe un archivo. |
| [.ExtractAll](../WDLang3/1000019085.md) | Extrae todos los archivos de un archivo comprimido y los descomprime. |
| [.ExtractDirectory](../WDLang3/1000025755.md) | Extrae y descomprime automáticamente los archivos en un directorio en el árbol de archivos de un archivo a una ubicación física. |
| [.ExtractFile](../WDLang3/1000019082.md) | Extrae un archivo de un archivo comprimido y lo descomprime automáticamente a una ubicación física o en memoria. |
| [.ExtractFileList](../WDLang3/1000019084.md) | Extrae y descomprime una lista de archivos encontrados en un archivo a una ubicación física. |
| [.ExtractPath](../WDLang3/1000019081.md) | Devuelve el ruta de un fichero encontrado en el archivo. |
| [.FindFile](../WDLang3/1000026330.md) | Encuentra un archivo en un archivo (en formato CAB, ZIP, RAR, WDZ o 7z). |
| [.InfoFile](../WDLang3/1000019092.md) | Devuelve las características de un fichero encontrado en un archivo:<br><br>- El nombre del archivo y su ruta almacenado..<br><br>- El tamaño inicial del archivo.<br><br>- El tamaño comprimido del archivo.<br><br>- La fecha y hora de creación del archivo.<br><br>- La fecha y hora de la modificación del archivo.<br><br>- La fecha y la hora de acceso al archivo.<br><br>- Los atributos del archivo.<br><br>- El modo de encriptación del archivo.<br><br><br> |
| [.IsMulti](../WDLang3/1000019080.md) | Devuelve la clase de archivo: archivo de una sola parte o archivo de varias partes. |
| [.ListFile](../WDLang3/1000019093.md) | Devuelve la lista de ficheros encontrados en un archivo. |
| [.OpenRAR](../WDLang3/1000019150.md) | Abre un archivo existente en formato RAR. |
| [.Split](../WDLang3/1000019161.md) | Divide un archivo en varios subarchivos de un tamaño determinado. |
| [Abrir](../WDLang3/1000019097.md) | Abre un archivo existente. |
| [Cerrar](../WDLang3/1000019086.md) | Cierra un archivo. |
| [FileSize](../WDLang3/1000019155.md) | Devuelve el tamaño de uno de los ficheros encontrados en el archivo (en formato CAB, RAR, WDZ, ZIP, 7z, TAR o TGZ (TAR.GZ)):<br><br>- el tamaño comprimido del archivo.<br><br>- el tamaño del archivo antes de la compresión.<br><br><br> |
| [.SelectFile](../WDLang3/1000026294.md) | Selecciona los archivos de un archivo. |



**Observación**: Las funciones de archivo pueden utilizarse en una Thread sin bloquear las otras Thread s durante su ejecución.

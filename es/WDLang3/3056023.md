


## NetIPAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la IP (Protocolo INTERNET) Address de un ordenador. 

**Atención**: Sólo se devuelven las direcciones de IPv4. Este modo de funcionamiento se mantiene para que sea compatible con las versiones anteriores.. Para conocer las direcciones IPV4 y/o IPv6, utilice [NetListIPAddress](../WDLang3/1000019683.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the IP address of local computer
Info(NetIPAddress())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetIPAddress([<Computer name>] [, <Index>])`
---

**`<Result>: (Cadena de caracteres)`**



- IP Address de la computadora especificada (en formato XXX.XXX.XXX.XXX), 

- Cadena vacía ("") si el equipo tiene menos direcciones IP que el &lt;índice&gt; solicitado.




**`<Computer name>: (Cadena de caracteres opcional)`**

Nombre de el equipo a utilizar. Si no se especifica este parámetro, el ordenador actual será considerado.


**`<Index>: (Entero opcional)`**

Índice de la IP Address en la lista de ordenadores Address es. Por defecto, el Address devuelto es el primer Address encontrado en la lista de Address ES de ordenador. El índice de la primera dirección es set a 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Número de direcciones IP
<a name="numero_direcciones_ELTPARAGRAPHE000065"></a>

- Se encuentra una Address IP para cada protocolo TCP/IP instalado en el mismo ordenador.

- Para listar las diferentes direcciones IP de un ordenador (IPV4 y/o IPv6), utilice [NetListIPAddress](../WDLang3/1000019683.md).

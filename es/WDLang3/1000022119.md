
## awProcedureGetInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica el lenguaje WLanguage Procedure que se llamará al recibir la información procedente de [awSendInfo](../WDLang3/1000022117.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
INTERNAL PROCEDURE GetData(b is Buffer)
	Info(b)
END

awProcedureGetInfo(GetData)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`awProcedureGetInfo(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del WLanguage Procedure ("callback") llamado al recibir información de [awSendInfo](../WDLang3/1000022117.md).
Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure name>(<Response>)
```

donde &lt;Respuesta&gt; es un Buffer Variable. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Consejo**: Llama a esta función en el código de inicialización del proyecto para que el Procedure sea accesible tan pronto como se inicie la aplicación.




## &lt;emailNotesSession Variable&gt;.CloseSession (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sesión abierta con el software de mensajería Lotus Notes.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
MySession is emailNotesSession

MySession..AddressDominoServer = "domino.mycompany.fr"
MySession..LotusNotesDatabase = "email/comptability.nsf"
MySession..CertificatePath = SysDir(srMyDocuments) + ["\"] + "certificates\notes.id"
MySession..Password = EDT_Password

MySession.StartSession()
...
MySession.CloseSession()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Session>.Sesión cerrada()`
---

**`<Session>: (emailNotesSession variable)`**

Nombre de la [emailNotesSession](../WDLang3/1000018768.md) Variable correspondiente a la sesión Email a utilizar.


## AppleAdvertisingIdentifier (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el identificador de publicidad del dispositivo (solo iOS). 

El identificador de publicidad es un UUID específico del dispositivo. Este identificador solo se utiliza para la publicidad. Esta cadena es utilizada por los anunciantes, principalmente para los siguientes fines:

- limitar el número de exposiciones a un anuncio,

- mostrar anuncios en función de los perfiles,

- conversión (anuncios que conducen a una visita al sitio),

- estimar el número de usuarios únicos,

- detectar fraude publicitario,

- depuración.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SWITCH AppleTrackingAuthorizationStatus()
     CASE atasNotDetermined
             AppleRequestTrackingAuthorizationAsynchronous(OnTracking)
     CASE atasAuthorized
             TrackUser(AppleAdvertisingIdentifier())
END
```

## Observaciones


### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000043"></a>

- Para obtener el identificador de publicidad del dispositivo se requiere un permiso específico. Puede comprobar si se ha concedido dicho permiso utilizando la función [AppleTrackingAuthorizationStatus](../WDLang3/1410087934.md). Si no se ha concedido el permiso, puede solicitarlo mediante la función [AppleRequestTrackingAuthorizationAsynchronous](../WDLang3/1410087935.md).

- Si no se ha concedido el permiso, el identificador estará compuesto por ceros. 

- Este identificador no está disponible en el simulador de Xcode.





### Restablecer el identificador de publicidad
<a name="restablecer_identificador_publicidad_ELTPARAGRAPHE000056"></a>

Los usuarios pueden restablecer el identificador de publicidad en los ajustes del dispositivo: 

- A partir de iOS 14: 

	- Vaya a Configuración y seleccione "Privacidad".

	- Desplácese hacia abajo y seleccione "Publicidad de Apple".

	- Desactive "Anuncios personalizados".




- Antes de iOS 14: 

	- Vaya a Configuración y seleccione "Privacidad".

	- Luego vaya a "Publicidad" y seleccione "Restablecer identificador de publicidad...".

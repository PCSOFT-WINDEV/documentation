


## EmailLoadAttachment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un archivo adjunto a un Email.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a file into the attachments of an email
MyMessage is Email
EmailLoadAttachment(MyMessage, ...
	SysDir(srMyDocuments) + ["\"] + "Yearly Report.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EmailLoadAttachment(<Email> , <File name and path>)`
---

**`<Result>: (Integro)`**



- Rango del archivo en la array de archivos adjuntos ([Email.Attach](../WDLang3/1000018713.md) Variable),

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable a la que se debe adjuntar el archivo.

**`<File name and path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo a adjuntar. Se puede utilizar una ruta de tipo UNC. Puede utilizar caracteres comodín ("\*" o "?").

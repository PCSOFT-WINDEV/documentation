
## FileToURI( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el contenido de un archivo en una Resource identificada por su URI.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opens the resource selector in creation mode
uriDocImage is URI = URISelect(uriImage, uriCreate)
IF NOT ErrorOccurred THEN
	// Save the image in the new document 
	FileToURI(image, uriDocImage)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = FileToURI(<File to save> , <Destination URI> [, <Mode>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente,

- <u><u><u><u>false</u></u></u></u> en caso contrario. 


Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

**`<File to save>: (Cadena de caracteres)`**

Ruta del archivo a guardar. Este archivo debe existir.

**`<Destination URI>: (Cadena de caracteres o URI Variable)`**

Resource en la que debe guardarse el archivo. Esta Resource puede ser identificada: 

- por una cadena correspondiente a una URI,

- por una variable de tipo [URI](../WDLang3/1000024027.md). 


Observación: El URI especificado debe ser de tipo "archivo" (archivo externo) o "contenido" (Resource de un proveedor de contenidos) y ser escribible.

**`<Mode>: (Constante opcional de tipo Integer)`**

Guardar modo: 



|   |   |
| --- | --- |
| *uriAdd* | El contenido del archivo se añade al final del contenido inicial de la Resource. |
| *uriCreación*<br>(Valor predeterminado) | El contenido inicial de la Resource se sustituye por el contenido del archivo. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>


## VolumeModify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el ajuste del volumen de una fuente de sonido.

**Observación**: Esta función se puede utilizar en el emulador.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Cuts the phone ring and enables the vibrator mode
VolumeModify(0, volumeRing, volumeVibratorOn)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = VolumeModify(<Volume> [, <Source> [, <Options>]])`
---

**`<Result>: (Integro)`**



- Valor del volumen para la fuente especificada después de la modificación,

- -1 si se produce un error. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Volume>: (Entero o constante de tipo Integer)`**

Nuevo valor para el volumen: 

- Valor incluido entre 0 (mute) y 100 (volumen máximo). 

- Constante utilizada para cambiar el volumen actual: 
	


|   |   |
| --- | --- |
| *volumeMinus* | Disminuye el valor del volumen en un nivel. |
| *volumePlus* | Aumenta el valor del volumen en un nivel. |






**`<Source>: (Constante opcional de tipo Integer)`**

Fuente de sonido a utilizar: 


|   |   |
| --- | --- |
| *volumeAlarm* | Volumen de la alarma. |
| *volumeCall* | Volumen de llamadas. |
| *volumeDefault* | Volumen de la fuente de Default. Esta fuente depende de la configuración de Phone. |
| *volumeMusic* | Volumen utilizado al reproducir un archivo de audio o vídeo. |
| *volumeNotification* | Volumen del sonido emitido durante un nuevo Notification. |
| *volumeRing* | Volumen del anillo de Phone. |
| *volumeSystem* | Volumen de los sonidos del sistema. |



**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Opciones disponibles:


|   |   |
| --- | --- |
| *volumeToast* | Activa la visualización de un toast que indica el nuevo valor del volumen para la fuente de sonido especificada. |
| *volumeVibratorIfSilent* | Habilita el vibrador del dispositivo al recibir un Notification o una llamada si la fuente de sonido correspondiente está en silencio. Esta opción se tiene en cuenta sólo si la fuente de sonido especificada corresponde a la constante *volumeRing* o a la constante *volumeNotification*. |
| *volumeVibratorOff* | Desactiva el vibrador del dispositivo cuando recibe un Notification o una llamada. Esta opción se tiene en cuenta sólo si la fuente de sonido especificada corresponde a la constante *volumeRing* o a la constante *volumeNotification*. |
| *volumeVibratorOn* | Habilita el vibrador del dispositivo al recibir un Notification o una llamada. Esta opción se tiene en cuenta sólo si la fuente de sonido especificada corresponde a la constante *volumeRing* o a la constante *volumeNotification*. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para modificar la fuente de sonido asociada al botón físico utilizado para set el volumen del dispositivo, llame a [VolumePhysicalButton](../WDLang3/1000020168.md).




## Envío de faxes con WINDEV y WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000283"></a>
Para enviar faxes desde una aplicación o desde un sitio, WINDEV y WEBDEV proponen varias funciones que le permiten enviar faxes a través del servidor de fax del sistema disponible en Windows XP y Windows 2000.

**Observación**: En Windows Vista, la gestión de faxes está disponible en Windows Vista Professional Edition y Windows Vista Ultimate Edition.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo enviar faxes?
<a name="¿como_enviar_faxes_ELTTEXTE000307"></a>
Para enviar faxes desde una aplicación WINDEV o desde un sitio WEBDEV, debe hacerlo:

- [configurar el el equipo desde el que se deben enviar los faxes](../WDLang3/3037016.md).

- crear la aplicación WINDEV/WEBDEV para enviar faxes.

- [configurar (si es necesario) las opciones del servidor de fax a través de la programación](../WDLang3/3037016.md).

## Programación del envío de faxes
<a name="programacion_del_envio_faxes_ELTTEXTE000331"></a>


### Envío de un fax desde una aplicación WINDEV o desde un sitio WEBDEV
<a name="envio_fax_desde_una_aplicacion_windev_desde_sitio_webdev_ELTPARAGRAPHE000035"></a>

Para enviar un fax desde una aplicación WINDEV o desde un sitio WEBDEV, debe hacerlo:

1. Preparar el fax, es decir, el archivo que se enviará. También se puede adjuntar al fax una primera página específica (llamada "Portada").. Esta portada es un archivo ".cov" y se puede crear desde el servidor de fax.

2. Utilice la función [FaxConnect](../WDLang3/3037005.md). Esta función se utiliza para conectar la aplicación WINDEV al servidor de fax instalado en el ordenador actual.

3. Envía el fax:

	- con la función [FaxSend](../WDLang3/3037006.md). Al enviar el fax, tiene la posibilidad de especificar el nombre de un WLanguage Procedure. Este Procedure será llamado cada vez que el estado del fax sea modificado por el servidor de fax. El estado del fax es devuelto por [FaxStatus](../WDLang3/3037001.md). Esta función sirve para rellenar la [Estructura de FaxCompleteStatus](../WDLang3/3037011.md) que contiene las características del fax.

	- con [iDestination](../WDLang5/3046074.md) y las funciones de impresión. Esta solución le permite imprimir directamente una reporte creada con el editor de reportes. En este caso, puede dar un nombre al fax enviado. El estado del fax será devuelto por [FaxStatus](../WDLang3/3037001.md). Esta función sirve para rellenar la [Estructura de FaxCompleteStatus](../WDLang3/3037011.md) que contiene las características del fax.
			**Observación**: Se puede enviar un solo fax a la vez. Sin embargo, [FaxSend](../WDLang3/3037006.md) puede ejecutarse varias veces en un fila: los diferentes faxes se añadirán a la cola de faxes salientes. El lenguaje WLanguage Procedure combinado con [FaxStatus](../WDLang3/3037001.md) permite saber qué fax se está procesando actualmente. Esta función sirve para rellenar la [Estructura de FaxCompleteStatus](../WDLang3/3037011.md) que contiene las características del fax.




4. Una vez enviados los faxes, basta con desconectar la aplicación WINDEV del servidor de fax con [FaxDisconnect](../WDLang3/3037007.md). 



<a name="NOTE3_2"></a>


### Envío de un fax creado con la aplicación editor de reportes
<a name="envio_fax_creado_con_aplicacion_editor_reportes_ELTPARAGRAPHE000085"></a>

Para enviar un fax creado con la editor de reportes, puede hacerlo:

- utilizar sólo la función [iDestination](../WDLang5/3046074.md). La reporte impresa por [iPrintReport](../WDLang5/3046032.md) se enviará al número de fax especificado. En este caso, no puede seguir el progreso del fax..
	
	```wl
	// Sends the "RPT_CustInvoice" report by fax
	iDestination(iFax, "0006050402")
	iPrintReport(RPT_CustInvoice)
	```


- utilizar las funciones de fax y [iDestination](../WDLang5/3046074.md). [iDestination](../WDLang5/3046074.md) establece (si es necesario) una conexión con el servidor de fax y devuelve el identificador de esta conexión. Este identificador puede ser utilizado por las funciones de fax para seguir el progreso de la operación de envío..
	
	```wl
	// Sends the "RPT_CustInvoice" report by fax
	ConnectID is int
	ConnectID = iDestination(iFax, "0006050402", "MyFax","\\MyFax\FaxPrinter")
	// "\\MyFax\FaxPrinter" is the share name of the fax printer
	iPrintReport(RPT_CustInvoice)
	```

## Configurar el servidor de fax mediante programación
<a name="configurar_servidor_fax_mediante_programacion_ELTTEXTE000361"></a>


### Opciones del servidor de fax
<a name="opciones_del_servidor_fax_ELTPARAGRAPHE000113"></a>

El servidor de fax estándar de Windows propone varias opciones que se pueden configurar directamente desde la interfaz del servidor de fax. Estas diferentes opciones se pueden configurar mediante programación a través del registro. Usted tiene la capacidad de configurar y descubrirlo:

- información sobre el envío de faxes (Windows 2000): 
	La clave de registro correspondiente es HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax:
	


|   |   |
| --- | --- |
| Archivar los faxes salientes | ArchiveOutgoingFax |
| Directorio de faxes salientes | ArchivoDirectorio |
| Hora de inicio de la tarifa económica | InicioCheapTime |
| Imprimir el encabezado superior | Creación de marca |
| Momento en que finaliza la tarifa económica | StopCheapTime |
| Número de días antes de eliminar los faxes no enviados | Días sucios |
| Número de intentos | reintentos |
| Número de minutos entre dos intentos | Reintentar delay |
| Prohibir las portadas personalizadas | ServerCoverPageOnly |



- información sobre el remitente del fax (Windows 2000): 
	La clave de registro correspondiente es HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax\\UserInfo:
	


|   |   |
| --- | --- |
| Buzón Address | Buzón |
| Código de facturación | Código de facturación |
| Departamento | Departamento |
| Empresa | Empresa |
| Monitor de estado siempre en la parte superior | AlwaysOnTop |
| Muestra el monitor de estado al enviar y recibir faxes. | Notificación visual |
| Negocios Address | Ofimática |
| Negocios Phone | Teléfono de oficina |
| Nombre completo | FullName |
| Número del remitente de fax | Número de fax |
| Sonido Notification | SonidoNotificación |
| Teléfono fijo | Teléfono de casa |
| Título | Título |

### Consejos 
<a name="consejos_ELTPARAGRAPHE000219"></a>

Para crear un servidor de fax y gestionar la rellamada, le recomendamos que especifique las siguientes opciones en el administrador de fax:

- No hay reintento de envío de faxes: 
	RegistrySetValue("HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax","Retries",0)

- Tiempo de espera set a 0 mn entre dos intentos: 
	RegistrySetValue("HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax","Reintentar delay",0)

- 0 día para guardar los faxes no enviados: 
	RegistrySetValue("HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\Fax","Dirty Days",0)




Es posible que un fax cuyo estado cambió de ***FaxStatusInit*** O ***FaxStatusCall*** a ***FaxStatusInactive*** se haya intentado enviar una vez. Entonces, este fax puede ser reenviado (no será automáticamente reenviado porque "No hay reintentos de envío de faxes" es set a True).

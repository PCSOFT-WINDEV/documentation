
## grSaveImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda la imagen de un gráfico en una variable de tipo Image.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
imgChart is Image = grSaveImage(CHART_Chart1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = grSaveImage(<Chart name> [, <Width> , <Height>])`
---

**`<Result>: (Image variable)`**

Variable de tipo [Image](../WDLang1/1000019650.md) en la que se guardó la imagen del gráfico. 

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Width>: (Entero opcional)`**

Ancho de la imagen a generar. Si no se especifican los parámetros &lt;Width&gt; y &lt;Height&gt;, se utilizará el ancho y alto del control Gráfico.

**`<Height>: (Entero opcional)`**

Alto de la imagen a generar. Si no se especifican los parámetros &lt;Width&gt; y &lt;Height&gt;, se utilizará el ancho y alto del control Gráfico.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La imagen generada con la función **grSaveImage** corresponde al gráfico que se muestra en el control Gráfico al llamar a la función **grSaveImage**. 

- El tooltip y la mira se ignoran.

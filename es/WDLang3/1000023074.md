


## MQTTUnsubscribe (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite darse de baja de un MQTT Notification. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MQTTUnsubscribe(gMqttSession, "sensors/sensor1/temperature1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MQTTUnsubscribe(<MQTT session> , <Topic>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se implementa la cancelación de la suscripción,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles. 




**`<MQTT session>: (mqttSession variable)`**

Nombre de [mqttSession](../WDLang3/1000023023.md) Variable correspondiente a la sesión MQTT a utilizar.

**`<Topic>: (Cadena de caracteres)`**

Tema para el que se solicita la baja.

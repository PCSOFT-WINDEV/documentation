
## BTStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado actual de activación de Bluetooth en el dispositivo o pide que se le notifique cuando cambie el estado de activación. Para activar o desactivar el Bluetooth en el dispositivo, utilice [BTActivate](../WDLang3/1000019359.md).

**Observación**: Las funciones de Bluetooth no están disponibles en el emulador de Android.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF BTStatus() = btDisabled THEN
	BTActivate(True)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estado de activación del Bluetooth

`<Result> = BTStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de activación del Bluetooth: 


|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |
| *btError* | No se pudo recuperar el estado de activación del Bluetooth. |




<a name="SYNTAX2"></a>

### Habilitar el Notification cuando la activación del Bluetooth cambia

`BTStatus(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLangage Procedure ("callback" Procedure) que se llamará cuando cambie el estado de activación del Bluetooth en el dispositivo. 
Para obtener más información sobre este procedimiento, consulte [Los parámetros de la Procedure utilizados por el BTStatus](../WDLang3/1000025313.md).

Si este parámetro corresponde a una cadena vacía ("""), las notificaciones ya no se enviarán cuando cambie el estado.. La función no tiene efecto si no se ha realizado ninguna solicitud de Notification.

## Observaciones


### Permisos necesarios

Requerido Permission: BLUETOOTH

Esta Permission permite que las aplicaciones se conecten a los servicios Bluetooth.

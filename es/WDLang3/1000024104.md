


## &lt;Scatter3DDeleteMarkCaption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los subtítulos de las marcas encontradas en un gráfico de dispersión 3D.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_MyChart.Scatter3DDeleteMarkCaption(grXCoordinate, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.Scatter3DDeleteMarkCaption([<Mark axis> [, <Value>]])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en el editor de ventanas).

**`<Mark axis>: (Constante de tipo Integer)`**

Tipo de eje del que deben borrarse las marcas: 



|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |
| *grZCoordinate* | Eje Z. |


Si no se especifica este parámetro, se borran las marcas de todos los ejes..

**`<Value>: (Real)`**

Valor de la marca para la que debe suprimirse la leyenda. 

Si no se especifica este parámetro, se borrarán todos los títulos de las marcas..

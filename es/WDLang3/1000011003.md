
## AppointmentModify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la Appointment actual: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un Lotus Notes o Outlook Calendar.
	Cuando se lee un Appointment, los datos del Appointment se actualizan en el [Estructura de Appointment](../WDLang3/1000011010.md). Para modificar la cita actual, se deben modificar los valores de la [Estructura de Appointment](../WDLang3/1000011010.md) y validar las modificaciones con la función **AppointmentModify**.
	**Recordatorio**: Esta función no puede utilizarse para modificar la cita actual en la Calendar de una mensajería Outlook Express.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
	"email/julia.nsf", "C:\Lotus Notes\julia.id")
// Read the last appointment
AppointmentLast(SessionID)
// Modify the location
mAppointment.Location = "Room 5H"
AppointmentModify(SessionID)
```

## Sintaxis

### Modificación de la cita en la Calendar de una mensajería Lotus Notes o Outlook

`<Result> = AppointmentModify(<Session identifier> [, <Appointment identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  

**`<Appointment identifier>: (Cadena de caracteres opcional)`**

ID de la Appointment a modificar. Este identificador se encuentra en el mAppointment.ID Variable.

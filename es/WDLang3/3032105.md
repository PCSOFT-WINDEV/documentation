
## LDAPListChildren( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los hijos de un objeto en un servidor LDAP.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032105&name=ldaplistchildren_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Find the child elements
sListChildren is string
sListChildren = LDAPListChildren("MySession", "DC=pcs,DC=local")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPListChildren(<LDAP session> , <DN of base object>)`
---

**`<Result>: (Cadena de caracteres)`**



- Lista de nombres distinguidos (DN) de los objetos encontrados,

- cadena vacía ("") si no hay ningún hijo asociado al objeto. 


Los diferentes elementos están separados por caracteres "CR" (Carriage Return). 

Si se produce un error, el valor de &lt;Resultado&gt; es una cadena vacía ("") y la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.

Atención: 

- Si el tamaño de &lt;Result&gt; supera los 2 GB, &lt;Result&gt; se truncará a 2 GB. 

- Dependiendo de las políticas LDAP definidas en el Directorio Active, la importación puede estar limitada a 1.000 usuarios.. En este caso, para eliminar esta limitación, es necesario modificar el parámetro LDAP MaxPageSize. Para obtener más información, consulte [https://support.microsoft.com/kb/315071](https://support.microsoft.com/kb/315071). 




**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP utilizada.

**`<DN of base object>: (Cadena de caracteres)`**

Nombre Distinguido (DN) del objeto cuyos elementos hijo se buscan.

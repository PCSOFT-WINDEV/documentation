
## SaaSAdminModifySite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is saasSite
S = SaaSAdminFindSite("Forum")
S.Name = "Old Forum"

IF SaaSAdminModifySite(S) = True THEN
	Info("Modified site")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Modificar un sitio SaaS (a través de una saasSite Variable)

`<Result> = SaaSAdminModifySite(<SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS site>: (Variable de tipo saasSite)`**

Nombre de la [saasSite](../WDLang3/1000019068.md) Variable correspondiente al sitio SaaS a modificar.


<a name="SYNTAX2"></a>

### Modificar un sitio SaaS identificado por su nombre

`<Result> = SaaSAdminModifySite(<Former name> , <New name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Former name>: (Cadena de caracteres)`**

Nombre del sitio a modificar en la base de datos de SaaS.

**`<New name>: (Cadena de caracteres)`**

Nuevo nombre del sitio SaaS.


## MarkerImage (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **MarkerImage** se utiliza para personalizar la imagen asociada a un cluster Marker o Marker mostrado en un control Mapa. Las características de la imagen se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImg is MarkerImage
MyImg.Size = 60
MyImg.BackgroundColor = LightGreen
MyImg.Text = 100

MyMarker is Marker
MyMarker.Position = MyLocation
MyMarker.Image = MyImg

MapAddMarker(MAP_Map1, MyMarker)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables de MarkerImage
<a name="propiedades_especificas_descripcion_las_variables_markerimage_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades pueden ser utilizadas para manipular el Marker Image:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| BackgroundColor | [Color](../WDLang1/1000019661.md)...entero, constante... | El contorno del círculo. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br>Observación: Para cambiar la opacidad del relleno, puedes escribir:<br><br><pre><code>MyMarkerImage.BackgroundColor.Opacity = 50</code></pre><br>Por defecto Color: Black. |
| Color | [Color](../WDLang1/1000019661.md)...entero, constante... | El color del contorno de la forma y la leyenda de Marker. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md),<br><br>- una variable de tipo [Color](../WDLang1/1000019661.md). <br><br><br>Por defecto Color: Black. |
| envergadura | Integro | Tamaño de la forma (radio para un círculo, lado para un cuadrado), en píxeles. |
| Forma | Constante de tipo Integer | La forma se representará con una de las siguientes constantes: <br><br>- *shapeSquare*: Forma cuadrada. <br><br>- *shapeRoundedSquare*: Forma cuadrada con esquinas redondeadas.<br><br>- *shapeCircle* (valor Default): Forma redonda.<br><br><br> |
| texto | Cadena de caracteres | Pie de foto que se mostrará en la forma. |

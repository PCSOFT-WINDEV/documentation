


## Funciones de gestión de zócalos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>








Las siguientes funciones sirven para gestionar sockets:



|   |   |
| --- | --- |
| [SocketAccept](../WDLang3/3070004.md) | Ejecutar en el servidor, esta función acepta una petición para conectarse a un socket existente en el servidor. |
| [SocketChangeTransmissionMode](../WDLang3/3070005.md) | Cambia el modo de transmisión utilizado en una toma. |
| [SocketClientInfo](../WDLang3/3070009.md) | Permite al servidor de get información sobre el socket cliente actual. |
| [SocketClose](../WDLang3/3070015.md) | Cierra un socket o una conexión a un socket del servidor. |
| [SocketConfigure](../WDLang3/1410088949.md) | Configura el uso de las funciones Socket. |
| [SocketConnect](../WDLang3/3070016.md) | Conecta un ordenador cliente a una toma de corriente determinada. |
| [SocketConnectBluetooth](../WDLang3/1000017145.md) | Conecta una computadora cliente a un enchufe Bluetooth específico. |
| [SocketConnectInfrared](../WDLang3/3070018.md) | Conecta un ordenador cliente a una toma determinada a través del puerto de infrarrojos. |
| [SocketConnectSSL](../WDLang3/1000017042.md) | Se utiliza para establecer una conexión segura con un servidor SSL. |
| [SocketCreate](../WDLang3/3070011.md) | Crea un zócalo. |
| [SocketCreateBluetooth](../WDLang3/1000017167.md) | Crea un socket utilizando un dispositivo Bluetooth. |
| [SocketCreateInfrared](../WDLang3/3070012.md) | Crea una toma que utiliza el puerto de infrarrojos. |
| [SocketCreateSSL](../WDLang3/1000017041.md) | Crea un socket seguro según el protocolo SSL. |
| [SocketCreateUDP](../WDLang3/3070019.md) | Crea un socket que utiliza el protocolo UDP. |
| [SocketDeny](../WDLang3/3070010.md) | Ejecutar en el servidor, esta función rechaza una solicitud de conexión a un socket existente en el servidor. |
| [SocketExist](../WDLang3/3070013.md) | Comprueba la existencia de la toma de corriente. |
| [SocketProxy](../WDLang3/1410088761.md) | Especifica si los sockets TCP y WebSockets deben pasar por un proxy SOCKS5 para ejecutar solicitudes. |
| [SocketRead](../WDLang3/3070014.md) | Recupera un mensaje enviado por otro socket. |
| [SocketWaitForConnection](../WDLang3/3070006.md) | Ejecutar en el servidor, esta función se utiliza para esperar una petición de conexión procedente del cliente sockets. |
| [SocketWrite](../WDLang3/3070002.md) | Escribe un mensaje destinado a otro socket. |
| [WebSocketClientConnect](../WDLang3/1410087751.md) | Establece una conexión entre el cliente y un servidor WebSocket. |
| [WebSocketClientConnectSSL](../WDLang3/1410087752.md) | Establece una conexión SSL entre el cliente y un servidor WebSocket. |

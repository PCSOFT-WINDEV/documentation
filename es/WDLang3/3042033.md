


## grSmoothing (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Suaviza un gráfico de líneas o dispersión con el método "Splines cúbicos". 

**Recordatorio**: Un gráfico no es suavizado por Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Smooth CHART_MyChart
grSmoothing(CHART_MyChart, True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Alisar un gráfico

`grSmoothing(<Chart name> , <Smoothing>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Smoothing>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe activar el suavizado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  





<a name="SYNTAX2"></a>

### Suavizar una serie de gráficos

`grSmoothing(<Chart name> , <Series number> , <Smoothing>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie a alisar.  

**`<Smoothing>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe activar el suavizado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se tiene en cuenta el alisamiento:

- en los gráficos de líneas (constante *grLine*),

- en gráficos de dispersión (*grScatter* constante) con puntos relacionado ([grScatterLinkPoint](../WDLang3/3042040.md)).




**Atención**: Esta función no tiene ningún efecto sobre un gráfico 3D.

Los parámetros definidos por **grSmoothing** se tendrán en cuenta en la siguiente llamada a [grDraw](../WDLang3/3042026.md).

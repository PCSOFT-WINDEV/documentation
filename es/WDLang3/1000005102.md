
## NotesOpenConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece una conexión con un servidor de Domino. Esta sesión puede ser cerrada por [Sesión cerrada](../WDLang3/3032124.md) o [EmailCloseSession](../WDLang3/3032006.md).

**Observaciones**:

- Para acceder a los correos electrónicos, los contactos, los grupos de contactos, las tareas y/o las citas gestionadas por Lotus Notes, hay que especificar la base de datos de Lotus Notes que se va a utilizar (segunda sintaxis de [EmailStartNotesSession](../WDLang3/3032121.md)).

- Para acceder a los documentos gestionados por Lotus Notes, hay que especificar la base de datos de Lotus Notes que se va a utilizar ([NotasOpenDatabase](../WDLang3/1000005101.md)).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the connection
SessionID is int
SessionID = NotesOpenConnection("Password", "MarsServer", ...
		"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Specify the Lotus Notes database to use
IF NotesOpenDatabase(SessionID, "email/julia.nsf") <> 0 THEN
	Info("Session started and database specified")
ELSE
	Error(ErrorInfo())
END
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesOpenConnection(<Password> , <Domino server> [, <Certificate path>])`
---

**`<Result>: (Integro)`**



- Identificador de sesión. 

- 0 si se produce un error (o <u><u><u><u>False</u></u></u></u>). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Password>: (Cadena de caracteres)`**

Contraseña de usuario. Esta contraseña es proporcionada por el proveedor de servicios o por el administrador de la red.

**`<Domino server>: (Cadena de caracteres)`**



- Nombre del servidor Domino,

- Vaciar la cadena (""") si la base de datos es una base de datos local.




**`<Certificate path>: (Cadena de caracteres opcional)`**

Ruta de la Certificate utilizada para identificar al usuario (archivo de identificación). Este archivo lo proporciona el administrador del servidor de Domino. Este archivo es necesario para establecer la conexión con el servidor.
Si no se especifica este parámetro, la ruta de la Certificate se leerá en el archivo de configuración del cliente de Lotus Notes (archivo "Notes.INI"). El Certificate utilizado corresponderá al último Certificate del usuario del cliente de Lotus Notes para el ordenador actual.


## SaaSAdminFindSite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un sitio en la base de datos de SaaS Webservice.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS tendrá acceso a todos los sitios SaaS. Un gestor de cuentas de cliente sólo tendrá acceso a los sitios para los que tiene una suscripción. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the SaaS Forum site
S is dynamic saasSite
S = SaaSAdminFindSite("Forum")
IF ErrorOccurred() THEN
	Error("Site not found")
ELSE
	IF SaaSAdminDeleteSite(S) = True THEN
		Info("Site deleted")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminFindSite(<Site>)`
---

**`<Result>: (Variable de tipo saasSite)`**

variable [saasSite](../WDLang3/1000019068.md) que contiene el description del sitio buscado.

Si no se encuentra el sitio, el Variable devuelto está vacío.

**`<Site>: (Cadena de caracteres)`**

Nombre del sitio SaaS buscado.

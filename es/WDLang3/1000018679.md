
## zipArchive (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **zipArchive** se utiliza para manipular un archivo que corresponde a uno de los siguientes formatos: ZIP, CAB, RAR, WDZ, 7z, TAR o TGZ (TAR.GZ). Las características del archivo pueden definirse y modificarse mediante diferentes propiedades WLanguage.



**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a backup archive for the "My Documents" directory
MyArchive is zipArchive

// Protect the archive by a password
MyArchive.Password = "secret"

// Create the archive
zipCreate(MyArchive, "s:\Backups\" + DateToString(DateSys(), "YYYY-MM-DD") + ".zip")

// Add the files
zipAddDirectory(MyArchive, SysDir(srMyDocuments))

// Close the archive
zipClose(MyArchive)
```

## Observaciones


### Propiedades específicas de las variables zipArchive
<a name="propiedades_especificas_las_variables_ziparchive_ELTPARAGRAPHE000062"></a>

Las siguientes propiedades pueden usarse para manipular una variable de tipo **zipArchive**:

| **denominar** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| ArchiveFormat | Constante de tipo Integer | Formato del fichero de archivo:<br><br>- *zipFormatCAB*: Archivo en formato CAB<br><br>- *zipFormatRAR*: Archivo en formato RAR<br><br>- *zipFormatWDZ*: Archivo en formato WDZ<br><br>- *zipFormatZIP*: Archivo en formato ZIP<br><br>- *zipFormat7Z*: Archivo en formato 7z<br><br>- *zipFormatTAR*: Archivo en formato TAR<br><br>- *zipFormatTARGZ*: Archivo en formato TGZ (TAR.GZ).  <br><br><br> |
| Archivo | Array de [zipArchivedFile](../WDLang3/1000019044.md) | Array de los archivos encontrados en el archivo. |
| CompressedSize | Entero de 8 bytes | Tamaño total comprimido de los archivos encontrados en el archivo comprimido.<br>**Esta propiedad es de solo lectura**. |
| CompressionLevel | Integro | Nivel de compresión que se debe utilizar para añadir archivos al archivo comprimido.<br>Para obtener más información, consulte [zipCompressionLevel](../WDLang3/3082054.md). |
| Contraseña | Cadena de caracteres | Contraseña que se utilizará para agregar archivos al archivo comprimido o para extraer archivos del archivo comprimido.. <br>Para obtener más información, consulte [zipPassword](../WDLang3/3082053.md).<br>Este Property está disponible en modo de escritura.<br><br>formato 7z: No se puede utilizar ninguna contraseña.<br>Formatos TAR y TGZ (TAR.GZ): No se puede utilizar ninguna contraseña. |
| envergadura | Entero de 8 bytes | Tamaño total de los ficheros del archivo (tamaño descomprimido).<br>**Esta propiedad es de solo lectura**. |
| NbFile | Integro | Número de archivos encontrados en el archivo.<br>**Esta propiedad es de solo lectura**. |
| NbPart | Integro | Número de piezas en un archivo de varios volúmenes.<br>Para un archivo de un solo volumen, este Property es igual a 1.<br>**Esta propiedad es de solo lectura**.<br> |




## Funciones Beacon
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las balizas: 



|   |   |
| --- | --- |
| [BeaconDetectBackground](../WDLang3/1000023112.md) | Notifica a la aplicación cuando el dispositivo entra o sale del rango de transmisión de un conjunto de balizas. |
| [BeaconDetectPrecise](../WDLang3/1000023115.md) | Permite buscar balizas cerca del dispositivo. |
| [BeaconListBackgroundDetection](../WDLang3/1000023113.md) | Devuelve la lista de detecciones de la baliza Active iniciada en la Background con [BeaconDetectBackground](../WDLang3/1000023112.md). |
| [BeaconStopBackgroundDetection](../WDLang3/1000023114.md) | Detiene una o más detecciones del faro en la Background. |
| [BeaconStopPreciseDetection](../WDLang3/1000023116.md) | Detiene la detección de la baliza iniciada por [BeaconDetectPrecise](../WDLang3/1000023115.md). |


## SaaSAdminClientConnection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la conexión a la base de datos de una cuenta de cliente para un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las conexiones. Un gestor de cuenta de cliente sólo tendrá acceso a la conexión relativa a su propia cuenta de cliente.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a connection to the database of the "Jet Tames" client for the "CRM" site
ASaaSSite is saasSite
ASaaSClient is saasClient 
AClientDatabase is Connection

ASaaSSite = SaaSAdminFindSite("CRM")
ASaaSClient = SaaSAdminFindClient("Jet Tames")
AClientDatabase = SaaSAdminClientConnection(ASaaSSite, ASaaSClient)

HOpenConnection(AClientDatabase)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperación de los parámetros de conexión (a través de las variables saasxxx)

`<Result> = SaaSAdminClientConnection(<SaaS site> , <Client account>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Connection)`**

Parámetros de conexión a la base de datos de la cuenta del cliente.

**`<SaaS site>: (Variable de tipo saasSite)`**

Nombre de la [saasSite](../WDLang3/1000019068.md) Variable correspondiente al sitio SaaS del que se deben recuperar los parámetros de conexión.

**`<Client account>: (Variable de tipo saasClient)`**

Nombre de la [saasClient](../WDLang3/1000019072.md) Variable correspondiente a la cuenta de cliente SaaS de la que se deben recuperar los parámetros de conexión.


<a name="SYNTAX2"></a>

### Recuperando los parámetros de conexión

`<Result> = SaaSAdminClientConnection(<Site> , <SaaS client>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Connection)`**

Parámetros de conexión a la base de datos de la cuenta del cliente.

**`<Site>: (Cadena de caracteres)`**

Nombre del sitio SaaS cuyos parámetros de conexión serán recuperados.

**`<SaaS client>: (Cadena de caracteres)`**

Nombre de la cuenta de cliente SaaS cuyos parámetros de conexión se recuperarán.

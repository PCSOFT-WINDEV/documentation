


## Funciones de gestión de servicios web
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los servicios web: 



|   |   |
| --- | --- |
| [CancelWebserviceContext](../WDLang3/1000019591.md) | Elimina del contexto del servicio web una Variable añadida por [DeclareWebserviceContext](../WDLang3/1000019460.md). |
| [DeclareWebserviceContext](../WDLang3/1000019460.md) | Permite declarar una lista de variables cuyo valor será persistente entre las sucesivas llamadas a un Webservice. |
| [WebserviceClientIPAddress](../WDLang3/1000019641.md) | Devuelve la IP Address de la el equipo usando y llamando al Webservice. |
| [WebserviceParameter](../WDLang3/1000022786.md) | Obtiene el valor de un parámetro en una llamada a una solicitud de servicio web SOAP o REST. |
| [WebserviceReadHTTPHeader](../WDLang3/1000022787.md) | Lee la cabecera HTTP recibida en una llamada a un servicio web SOAP o REST. |
| [WebserviceWriteCustomHTTPCode](../WDLang3/1410087803.md) | Especifica el código HTTP personalizado que se devolverá cuando finalice la ejecución de la función REST Webservice. |
| [WebserviceWriteHTTPCode](../WDLang3/1000022785.md) | Especifica el código HTTP que se devolverá al final de la ejecución de la función REST Webservice. |
| [WebserviceWriteHTTPHeader](../WDLang3/1000022788.md) | Añade una cabecera HTTP adicional a una solicitud de un servicio web REST. |
| [WebserviceWriteMIMEType](../WDLang3/1000022681.md) | Indica el tipo MIME de la respuesta devuelta por el Webservice (caso de una llamada a un Webservice REST). |

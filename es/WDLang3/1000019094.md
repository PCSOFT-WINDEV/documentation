


## &lt;zipArchive Variable&gt;.NbPartNeed (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de piezas de un tamaño determinado necesarias para dividir un archivo de una sola pieza.. 

**Observación**: Esta función sólo está disponible para los archivos comprimidos en formato ZIP y WDZ.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyArchive is zipArchive
// Open an archive
ResOpenArchive = MyArchive.Open("C:\MyArchives\Archive.zip")

IF ResOpenArchive = 0 THEN
	...
	// Number of parts (1.4 MB) required to contain the archive
	ResNbNecessaryParts = MyArchive.NbPartNeeded(1400*1024)
	Info("To split this archive, " + ResNbNecessaryParts + ...
				" part(s) of 1.4 MB are required")
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Archive>.NbNecessaryPart(<Size of parts>)`
---

**`<Result>: (Integro)`**

Número de piezas de un tamaño determinado que se necesitan para contener un archivo comprimido.

**`<Archive>: (Variable de tipo zipArchive)`**

Nombre de una Variable de tipo [zipArchive](../WDLang3/1000018679.md) correspondiente al archivo a utilizar.

**`<Size of parts>: (Integro)`**

Tamaño de las piezas en bytes. El tamaño de Default es igual a 1,44 MB (1400\*1024 bytes). El tamaño mínimo es igual a 300 KB para cada parte (300\*1024 bytes)..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Dividir un archivo
<a name="dividir_archivo_ELTPARAGRAPHE000225"></a>

[.Split](../WDLang3/1000019161.md) crea un archivo de varias partes. **&lt;zipArchive Variable&gt;.NbPartNeed** se utiliza para determinar el número de partes necesarias para dividir el archivo.<a name="NOTE0_2"></a>


### Archivo de varias partes
<a name="archivo_varias_partes_ELTPARAGRAPHE000242"></a>

El uso de la función **&lt;zipArchive Variable&gt;.NbPartNeed** en un archivo multiparte permite calcular el número de partes necesarias para una realizar nueva división.

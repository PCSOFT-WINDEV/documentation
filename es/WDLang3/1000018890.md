
## HTTPListCertificate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los certificados de cliente encontrados en el equipo para autentificar en un servidor. Se puede utilizar una Certificate específica a través de [Certificado HTTPC](../WDLang3/1000018898.md). 

**Ejemplo de uso**: Esta función puede utilizarse, por ejemplo, para evitar mostrar la ventana estándar de selección de Certificate si el sitio pide al cliente que se autentifique. 

**Observación**: [Lista de certificados](../WDLang1/1000019293.md) se utiliza para listar los certificados disponibles en el almacén de certificados de Windows. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CertifList is string
CertifList = HTTPListCertificate()
FOR EACH STRING Certif OF CertifList SEPARATED BY RC
	ListAdd(LIST_List1, ExtractString(Certif, 1, TAB) + ...
			gStoredValue(ExtractString(Certif, 3, TAB)))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HTTPListCertificate()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de certificados encontrados en el ordenador actual. Los diferentes certificados están separados por un Carriage Return (CR). Esta lista tiene el siguiente formato: 

```txt
<Certificate name> + TAB + <Supplier> + TAB + <Digital print> + CR + ...
```

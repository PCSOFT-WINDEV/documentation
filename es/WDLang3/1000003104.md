
## DomoSend( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía un comando a uno o más dispositivos eléctricos compatibles con el protocolo X10.

**Observación**: Para que el protocolo X10 funcione correctamente a través de la portadora de potencia-Line, se requiere una potencia continua entre los dispositivos. Por ejemplo, la transmisión no puede pasar a través de un transformador de aislamiento.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the X10 connection
ConnectionID is int
ConnectionID = DomoOpen("COM4", "A")  
// Switch on the lamps connected to the A3 and A4 modules
DomoSend(ConnectionID, "A3 A4", domoOn)
// Equivalent to: DomoSwitchOn(ConnectionID, "A3 A4")
// Modify the intensity of these lamps
DomoSend(ConnectionID, "A3 A4", domoDim, -60)
// Equivalent to: DomoSend(ConnectionID, "A3 A4", domoModify, -60)
...
// Switch off the lamps
DomoSend(ConnectionID, "A3 A4", domoOff)
// Equivalent to: DomoSwitchOff(ConnectionID, "A3 A4")
```


<a name="Example2"></a>



```wl
// Switch off all the devices on A
DomoSend(ConnectionID, "A", domoAllOff)
```

## Sintaxis

`<Result> = DomoSend(<Connection identifier> , <Address> , <Command to send> [, <Command value>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el comando se envió correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario, (error de transmisión, el dispositivo no respondió, etc.). Si se produce un error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage* para get más detalles sobre el error.




**`<Connection identifier>: (Integro)`**

Identificador de la conexión a utilizar. Este identificador está definido por [DomoOpen](../WDLang3/1000003102.md).

**`<Address>: (Cadena de caracteres)`**

Address de dispositivos eléctricos compatibles con el protocolo X10 correspondiente.

El formato de Address es el siguiente: &lt;Código de la casa&gt;&lt;Código del módulo&gt;&gt;Código de la casa

Dónde:

&lt;El código de la casa&gt; corresponde a una letra que se encuentra entre A y P.

&lt;El código del módulo&gt; corresponde a un número que se encuentra entre 1 y 16.

Por ejemplo: "A1, B14.

Para especificar varias direcciones, las direcciones deben estar separadas por un carácter de espacio. Por ejemplo: "A1 A3 B15.

**`<Command to send>: (Constante de tipo Integer)`**

Comando que debe ser enviado a los dispositivos eléctricos especificados:


|   |   |
| --- | --- |
| *domoAllOff* | Apaga todos los aparatos eléctricos especificados. En este caso, &lt;Address&gt; sólo debe contener uno o más códigos de casa. |
| *domoBright* | Aumenta la intensidad de los dispositivos eléctricos especificados en &lt;Valor del comando&gt; por ciento (brillo de las lámparas en la mayoría de los casos). |
| *domoDim* | Reduce la intensidad de los dispositivos eléctricos especificados en &lt;Valor del comando&gt; por ciento (brillo de las lámparas en la mayoría de los casos). |
| *domoModify* | Modifica el estado actual de los aparatos eléctricos especificados.. En este caso, &lt;Valor de comando&gt; corresponde a:<br><br>- True: Enciende los aparatos eléctricos especificados.<br><br>- False: Apaga los aparatos eléctricos especificados.<br><br>- &lt;Valor positivo&gt;: Aumenta la intensidad de los dispositivos eléctricos especificados en un &lt;valor positivo&gt; por ciento<br><br>- &lt;Valor negativo&gt;: Reduce la intensidad de los dispositivos eléctricos especificados en un &lt;valor negativo&gt; por ciento<br><br><br> |
| *domoOff* | Apaga los aparatos eléctricos especificados. |
| *domoOn* | Enciende los aparatos eléctricos especificados. |



**`<Command value>: (Opcional booleano o entero)`**

Valor del comando enviado:


|   |   |
| --- | --- |
| <u><u><u><u>True</u></u></u></u> | Enciende los aparatos eléctricos especificados. |
| <u><u><u><u>False</u></u></u></u> | Apaga los aparatos eléctricos especificados. |
| &lt;Valor positivo&gt; | Aumenta la intensidad de los dispositivos eléctricos especificados en un &lt;valor positivo&gt; por ciento. |
| &lt;Valor negativo&gt; | Reduce la intensidad de los dispositivos eléctricos especificados en un &lt;valor negativo&gt; por ciento. |

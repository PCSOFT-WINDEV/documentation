
## GroupDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra el grupo actual de contactos del libro Address de un Lotus Notes o de un mensaje de Outlook.

**Atención**: Esta función **no permite** eliminar el grupo de contactos actual de la libreta de direcciones de una mensajería Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
	"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Read the first group of contacts
GroupFirst(SessionID)
// Delete this group
GroupDelete(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = GroupDelete(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Versión de Outlook utilizada
<a name="version_outlook_utilizada_ELTPARAGRAPHE000058"></a>

Los grupos de contactos no son compatibles con Outlook 97. Por lo tanto, no se puede eliminar ningún grupo de contactos en esta versión de Outlook.

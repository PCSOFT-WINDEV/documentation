
## saasService (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasService** permite definir las características avanzadas de un servicio SaaS. 

Las características de este servicio pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyService is saasService
MyService.Name = "Access"

CurrentSite is saasSite

SaaSAdminAddService(MyService, CurrentSite)
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Propiedades específicas de las variables saasService
<a name="propiedades_especificas_las_variables_saasservice_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden utilizarse para manipular un servicio SaaS:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre del servicio. Este nombre se utilizará para comprobar la autorización de acceso a través de [Servicio SaaSCheckService](../WDLang3/1000018961.md). |
| emplazamiento | Variable de tipo [saasSite](../WDLang3/1000019068.md) | Sitio SaaS al que pertenece el servicio. <br>Esta propiedad está disponible en solo lectura. |


<a name="NOTE0_2"></a>


### Las funciones que utilizan el tipo saasService
<a name="las_funciones_que_utilizan_tipo_saasservice_ELTPARAGRAPHE000063"></a>

Las siguientes funciones están usando un saasService Variable:



|   |   |
| --- | --- |
| [SaaSAdminAddService](../WDLang3/1000019107.md) | Añade un servicio SaaS en la description de un sitio. |
| [SaaSAdminDeleteService](../WDLang3/1000019109.md) | Elimina un servicio SaaS. |
| [SaaSAdminFindService](../WDLang3/1000019110.md) | Encuentra la description de un servicio SaaS. |
| [SaaSAdminModifyService](../WDLang3/1000019108.md) | Modifica un servicio SaaS. |
| [SaaSAdminPricingDisableService](../WDLang3/1000019133.md) | Desactiva un servicio con un precio SaaS. |
| [SaaSAdminPricingEnableService](../WDLang3/1000019132.md) | Permite un servicio a un precio SaaS. |


## LEDSwitchOff( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Apaga el LED (diodo emisor de luz) del aparato.. Este LED debe haber sido encendido por [LEDSwitchOn](../WDLang3/1000019446.md). 

Atención : 

- El dispositivo debe estar equipado con un LED.

- Esta función no tiene ningún efecto en el simulador o en el emulador.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch on the diode in red during 10 second
LEDSwitchOn(LightRed, 1000)

// Turn off LED
LEDSwitchOff()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LEDSwitchOff()`

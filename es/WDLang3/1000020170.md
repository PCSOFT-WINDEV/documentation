
## BrightnessModify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la configuración del brillo de la pantalla del dispositivo.

**Observación**: Esta función se puede utilizar en el emulador.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Applies a maximum brightness to the screen
BrightnessModify(100)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BrightnessModify(<Brightness>)`
---

**`<Brightness>: (Entero o constante de tipo Integer)`**

Nuevo valor para el brillo: 

- valor incluido entre 0 y 100. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales 
<a name="casos_especiales_ELTPARAGRAPHE000067"></a>

- La modificación de la luminosidad afecta a la configuración del dispositivo. Por lo tanto, el nuevo brillo es global para todas las aplicaciones y se almacenará cuando la aplicación esté cerrada.

- Para conocer el valor del ajuste actual del brillo de la pantalla, utilice [BrightnessGet](../WDLang3/1000019810.md).

- Le aconsejamos que no especifique un brillo muy bajo (menos de 10).. De hecho, el brillo mínimo cambia según el dispositivo y la pantalla de la Phone se puede apagar si se alcanza el límite.

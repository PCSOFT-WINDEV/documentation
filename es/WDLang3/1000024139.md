


## &lt;Gráfico&gt;.SourceSeries (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define la fuente de una serie.


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control filled from a Table control column 
IF CHART_Chart1.SourceSeries(3, grTableColumn, "Table1.Column1") = True THEN
	Info("The source of series 3 is MyColumn from now on")
END
```
<a name="Example2"></a>

```wl
// Chart control based on an item of an SQL query
MyQuery is Data Source

MySQLCode is string = QRY_Query1..SQLCode
HExecuteQuery(MyQuery, hQueryDefault, MySQLCode)
CHART_Chart1.SourceSeries(1, grItem, "MyQuery", "TotalBT")
CHART_Chart1.Draw()

CASE ERROR:
	Error(HErrorInfo())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.grSourceSeries(<Series number> , <Source> [, <Details 1> [... [, <Details N>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se definió la fuente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Series number>: (Integro)`**

Número de la serie afectada.

**`<Source>: (Constante de tipo Integer)`**

Fuente de datos del gráfico:


|   |   |
| --- | --- |
| *grCallback* | Serie llena de una función de WLanguage/Procedure. <br>En este caso, &lt;Detalles 1&gt; corresponde al nombre del WLanguage Procedure a llamar.<br><br>El Procedure se llama para cada número. Este procedimiento tiene el siguiente formato: <br><br><br><pre><code>PROCEDURE MyProcedure(<Chart control>, <Series number>, <br><Index in series>)</code></pre><br><br>La Procedure devuelve el número deseado. Para indicar el final de la serie, el procedimiento debe devolver <u><u><u><u>False</u></u></u></u>.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *grItem* | Serie llenada a partir de un artículo.<br><br>&lt;Detalles 1&gt; corresponde al nombre del archivo de datos (o consulta), &lt;Detalles 2&gt; corresponde al nombre del artículo. La serie navegará a través de el archivo de datos y tomará todos los valores del artículo.<br><br>Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |
| *grListBox* | La serie se llenó de un control List Box.<br><br>En este caso, &lt;Detalles 1&gt; corresponde al nombre de un control List Box. El control List Box se navega para llenar la serie. Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |
| *grProgramming* | Serie descrita a través de la programación. <br><br>&lt;Los detalles 1&gt; pueden corresponder a una serie de números en la misma cadena de caracteres, separados por ";". |
| *grTable* | Serie llenada a partir de una array de WLanguage.<br><br>&lt;Detalles 1&gt; corresponde al nombre de la array. &lt;Detalles 2&gt; puede ser el índice de la columna (array bidimensional ) o el nombre de un miembro (array de estructuras). |
| *grTableColumn* | Serie rellenada desde una columna de un control Tabla. <br><br>En este caso, &lt;Detalles 1&gt; corresponde al nombre de una columna que se encuentra en un control Tabla. Este nombre debe corresponder al nombre completo de la columna, entre comillas (por ejemplo: "&lt;TableName&gt;.&lt;ColumnName&gt;").<br>Se navega por la columna para llenar la serie. Si uno de los valores no es un valor numérico, se crea un "agujero" en la serie. |



**`<Details 1>: (Tipo esperado (parámetro opcional))`**

Primer parámetro esperado, según el tipo de fuente de datos.



**`<Details N>: (Tipo esperado (parámetro opcional))`**

Último parámetro esperado, según el tipo de fuente de datos.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El [Fuente](../Proprietes/2510147.md) Property identifica el tipo de fuente de datos de una serie de un gráfico (columna de un control Tabla, control List Box, etc.).

- La [SourceDescription](../Proprietes/2510148.md) Property obtiene el nombre del elemento que corresponde a la fuente de datos de una serie de un gráfico.

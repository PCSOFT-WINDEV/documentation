


## MapDisplayPopup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana emergente de una Marker mostrada en una control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays a marker 
MyLocation is geoPosition
MyLocation.Latitude = 43.613708 
MyLocation.Longitude = 3.876972
MyMarker is Marker
MyMarker.Position = MyLocation
MyMarker.WithPopup = True
MyMarker.Name = "Position"
MyMarker.Description = "Location of point of interest"
MapAddMarker(MAP_MyMap, MyMarker)

// Open marker popup
MapDisplayPopup(MAP_MyMap, MyMarker)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapDisplayPopup(<Map control> , <Marker>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudiera abrir el popup de Marker, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Process.

**`<Marker>: (Marcador Variable)`**

Nombre de la [Marker](../WDLang3/1000019940.md) Variable que describe la posición del Marker a mostrar. Esta Marker debe haber sido añadida en el mapa.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para cerrar la ventana emergente de una Marker ya mostrada, utilice [MapClosePopup](../WDLang3/1000025554.md).

- Para determinar si se muestra una ventana emergente de Marker, utilice [MapPopupDisplayed](../WDLang3/1000025555.md).


## LDAPNbValue( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de valores de un atributo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003032106&name=ldapnbvalue_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Number of values for the "UserPassword" attribute?
NbValue = LDAPNbValue("MySession", "CN=JP", "UserPassword")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPNbValue(<LDAP session> , <Object DN> , <Attribute name>)`
---

**`<Result>: (Integro)`**

Número de valores para el atributo especificado.

**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP utilizada.

**`<Object DN>: (Cadena de caracteres)`**

Nombre Distinguido (DN) del objeto cuyo número de valores para el atributo especificado se solicita.

**`<Attribute name>: (Cadena de caracteres)`**

Nombre del atributo para el que se solicita el número de valores. Los atributos del objeto son devueltos por [Atributo LDAPListAttribute](../WDLang3/3032104.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Servidor LDAP y objeto LDAP local
<a name="servidor_ldap_objeto_ldap_local_ELTPARAGRAPHE000040"></a>

El servidor LDAP contiene objetos LDAP.

El objeto LDAP local es un objeto LDAP que se obtiene del servidor.
<a name="NOTE0_2"></a>




### Actualización del objeto LDAP local
<a name="actualizacion_del_objeto_ldap_local_ELTPARAGRAPHE000050"></a>

Cuando se ejecuta una función LDAP de lectura  ([Atributo LDAPListAttribute](../WDLang3/3032104.md), **LDAPNbValue**, [Valor LDAPValor](../WDLang3/3032107.md)), el objeto del servidor se devuelve completamente al equipo local para limitar la comunicación con el servidor. Por lo tanto, si estas funciones de lectura se llaman varias veces con el mismo DN, la comunicación con el servidor se establece en la primera llamada.. Las otras llamadas leerán el objeto local.

**LDAPNbValue** actualiza el objeto LDAP local sólo si el DN del objeto pasado como parámetro es diferente del último DN pasado como parámetro a una de las siguientes funciones: [Atributo LDAPListAttribute](../WDLang3/3032104.md), **LDAPNbValue**, [Valor LDAPValor](../WDLang3/3032107.md).

**LDAPNbValue** no actualiza el objeto LDAP local si el DN pasado como parámetro es el mismo que el último DN pasado como parámetro a una de las siguientes funciones: [Atributo LDAPListAttribute](../WDLang3/3032104.md), **LDAPNbValue**, [Valor LDAPValor](../WDLang3/3032107.md). Sin embargo, si se han utilizado [LDAPValidateAdd](../WDLang3/3032111.md) y [LDAPValidateModify](../WDLang3/3032117.md) con este mismo DN, el objeto local se actualiza automáticamente.

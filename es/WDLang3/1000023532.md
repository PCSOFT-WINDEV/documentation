


## grAddTimeData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade datos en un gráfico de tiempo. Un gráfico de tiempo es un gráfico en el que la unidad del eje X es la hora (fecha, hora, duración).. 

Esta escala de tiempo puede utilizarse, por ejemplo, para representar un cambio en una medición (sensor de temperatura, por ejemplo) a lo largo de un día.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nSeries is int = 1
rTemperature is array of 24*60 real

FOR i = 1 TO 24*60 // 24h * 60 mn
	d is Duration
	d.Minute = I
	grAddTimeData(CHART_TemperatureSensor, nSeries, d, rTemperature[i])
END
grDraw(CHART_TemperatureSensor)
```

```wl
// Sinusoidal drawing
FOR i = 1 TO 360
	d is Duration
	d.Minute = I
	grAddTimeData(CHART_Sine, 1, d, 2 + Sin(i))
END

grDraw(CHART_Sine)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grAddTimeData(<Chart control> , <Series number> , <Time> , <Value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar.

**`<Series number>: (Integro)`**

Número de la serie a la que se añadirá un valor. Si la serie especificada no existe, el diseñador de la carta crea tantas series como sean necesarias para get una matriz consistente. Todas las series creadas se inicializan a 0.

**`<Time>: (Duración o FechaHora o Fecha u Hora)`**

Posición en el tiempo para la que se añadirá el valor.

**`<Value>: (Entero o real)`**

Valor a agregar. 
Si ya existe un valor para una posición en el tiempo, este valor será sustituido por el nuevo valor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función permite añadir datos a los gráficos que utilizan una escala de tiempo y también a los gráficos en tiempo real.. Para obtener más información, consulte [Gráfico en tiempo real](../WDChamp/1000021037.md).

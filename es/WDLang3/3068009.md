
## SMSNbMessage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de mensajes SMS almacenados en una Android Phone. 

Nota: En Android, las funciones de SMS sólo pueden manipular los mensajes almacenados en el Phone.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of SMS messages stored on the phone's memory
STC_NbSMS = "You have " + SMSNbMessage(smsMemory) + " messages."
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SMSNbMessage(<Location> , <Type of information>)`
---

**`<Result>: (Integro)`**



- Número de SMSs almacenados o número de SMSs que pueden ser almacenados,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Location>: (Constante de tipo Integer)`**

Localización de SMSs:



|   |   |
| --- | --- |
| *smsMemory* | Mensajes SMS almacenados en la memoria del dispositivo. |



**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:



|   |   |
| --- | --- |
| *smsTotalNb* | Número de SMS almacenados. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


### Diferencias entre el modo GO y el tiempo de ejecución
<a name="diferencias_entre_modo_tiempo_ejecucion_ELTPARAGRAPHE000071"></a>

Se produce un error WLanguage cuando se llama a **SMSNbMessage** en el modo de prueba **(GO)** (simulación en el ordenador de desarrollo).

Las funciones de SMS pueden ser usadas durante un GO en el emulador de Android. Para simular el envío de mensajes SMS al emulador, ver la documentación del SDK de Android: [https://developer.android.com/studio/run/emulator#console](https://developer.android.com/studio/run/emulator#console)




### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000081"></a>

La llamada a esta función modifica los permisos requeridos por la aplicación. 

Requerido Permission : READ_SMS


## SocketCreateInfrared( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un socket que utiliza el puerto de infrarrojos. Un socket es un recurso utilizado por las aplicaciones para comunicarse de un equipo a otro, independientemente del tipo de red.

**Observación**: Esta función debe llamarse en el servidor utilizado para la comunicación.. Si el ordenador cliente utiliza una aplicación WINDEV para conectarse a este socket, esta aplicación tendrá que utilizar [SocketConnectInfrarrojo](../WDLang3/3070018.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SocketCreateInfrared("Server", "Service") = False THEN
	Error("Creation error" + ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SocketCreateInfrared(<Socket name> , <Service name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).




**`<Socket name>: (Cadena de caracteres)`**

Nombre que se le dará al zócalo. Este nombre será utilizado por todas las funciones del socket.

**`<Service name>: (Cadena de caracteres)`**

Nombre que se le dará al servicio. Este nombre se utilizará en [SocketConnectInfrarrojo](../WDLang3/3070018.md).

<a name="NOTE0_1"></a>

## Observaciones




### Cómo recrear un zócalo
<a name="como_recrear_zocalo_ELTPARAGRAPHE000054"></a>

Si [SocketClose](../WDLang3/3070015.md) no cierra correctamente un socket (reiniciando el equipo, por ejemplo), la recreación del socket puede tardar bastante tiempo. De hecho, tendrá que esperar hasta que el sistema libere el zócalo.
<a name="NOTE0_2"></a>




### Creación de un socket
<a name="creacion_socket_ELTPARAGRAPHE000065"></a>

Para crear un socket en un servidor, utilice [SocketCreate](../WDLang3/3070011.md).
<a name="NOTE0_3"></a>




### Protocolo IRDA
<a name="protocolo_irda_ELTPARAGRAPHE000076"></a>

**SocketCreateInfrared** utiliza el protocolo IRDA (Infrared Data Association).
<a name="NOTE0_4"></a>




### Límite de ventanas
<a name="limite_ventanas_ELTPARAGRAPHE000086"></a>

El número máximo de sockets infrarrojos utilizados simultáneamente no puede superar los 80.

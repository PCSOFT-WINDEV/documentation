
## SaaSAdminFindService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra la description de un servicio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS tendrá acceso a todos los servicios SaaS. Un gestor de cuentas de clientes sólo tendrá acceso a los servicios associated con los sitios a los que se suscribió. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
S = SaaSAdminFindSite("Forum")

MyService is saasService dynamic 
MyService = SaaSAdminFindService("Post a message", S)
IF ErrorOccurred = False THEN
	Info("Service found")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminFindService(<Department> , <SaaS site>)`
---

**`<Result>: (saasService Variable)`**

variable [saasService](../WDLang3/1000019096.md) que contiene el servicio description. La Variable devuelta está vacía si la búsqueda falló.

**`<Department>: (Cadena de caracteres)`**

Nombre del servicio a encontrar en el sitio.

**`<SaaS site>: (saasSite Variable o cadena de caracteres)`**

Nombre de la [saasSite](../WDLang3/1000019068.md) Variable (o nombre del sitio) correspondiente al sitio SaaS que tiene el servicio deseado.


## SaaSAdminAddSubscription( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una suscripción a un precio de un sitio SaaS para una cuenta de cliente.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todas las suscripciones. Un administrador de cuentas de clientes puede añadir a su propia cuenta una nueva suscripción a un sitio al que ya está suscrito.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite
P is dynamic saasPricing
C is saasClient dynamic

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")

Sub is saasSubscription
Sub.NbLicenses = 5

// Subscribe the "Hybrid" client to the "Promo 2010" pricing of the "Forum" site
SaaSAdminAddSubscription(Sub, P, C)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminAddSubscription(<New Subscription> , <Pricing> , <Client account>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<New Subscription>: (saasSubscription Variable)`**

Nombre de la variable [saasSubscription](../WDLang3/1000019088.md) que contiene el description de la suscripción a añadir.

**`<Pricing>: (saasPricing Variable)`**

Nombre de [saasPricing](../WDLang3/1000019091.md) Variable correspondiente a la tarificación a la que se adscribe el abono.

**`<Client account>: (Variable de tipo saasClient)`**

Nombre de [saasClient](../WDLang3/1000019072.md) Variable correspondiente a la cuenta de cliente a suscribir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La adición de una nueva suscripción a un precio para un cliente sólo se puede hacer si no hay una suscripción actual para este precio y para este cliente.




## &lt;HMCaptionY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de una categoría en el eje Y de un gráfico de Mapa de Calor.

Nota: Por Default, la leyenda de una categoría corresponde al valor numérico de la categoría.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_HMap.HMCaptionY(1, "Y1")
CHART_HMap.HMCaptionY(2, "Y2")
```

```wl
// Caption of axes
FOR EACH ELEMENT nIndex, nTemperature OF gaaTemperature
	CHART_Chart.HMCaptionX(nIndex, "" + nTemperature) 
END
FOR EACH ELEMENT nIndex, nWind OF gaaWind
	CHART_Chart.HMCaptionY(nIndex, "" + nWind)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.HMCaptionY(<Y> , <Caption>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).


**`<Y>: (Integro)`**

Posición de la categoría en el eje Y.

**`<Caption>: (Cadena de caracteres)`**

Nuevo título para la categoría. Si este parámetro corresponde a una cadena vacía ("""), se muestra el valor numérico de la categoría.

<a name="NOTE0_1"></a>

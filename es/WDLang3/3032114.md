
## LDAPDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un objeto del servidor LDAP. Por razones de seguridad, el objeto sólo se borrará si no tiene ningún elemento hijo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the user named Bob
IF LDAPDelete("MySession", "CN=Bob, CN=Users, DC=pcsoft, DC=local") = False THEN
	Error("Error while deleting Bob")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LDAPDelete(<LDAP session> , <DN of base object>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<LDAP session>: (Cadena de caracteres)`**

Nombre de la sesión LDAP utilizada.

**`<DN of base object>: (Cadena de caracteres)`**

Nombre Distinguido (DN) del objeto a borrar

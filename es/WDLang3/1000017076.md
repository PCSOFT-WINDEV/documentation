
## SNMPStringToOID( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un OID de su representación de texto a su representación numérica.

**Observación**: Esta función no puede utilizarse mientras no se haya cargado el archivo MIB que describe la correspondencia entre los OID en formato textual y numérico ([SNMPLoadMIB](../WDLang3/1000017074.md)).  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
bMIBLoaded is boolean
sPathMIBs is string = CompleteDir(fExeDir()) + "mibs\"
sMib is string
sMib = "RFC1382-MIB.mib" // MIB of X25 protocol
sOID is string
bMIBLoaded = SNMPLoadMIB(sPathMIBs + sMib)
IF bMIBLoaded = False THEN 
	Error("MIB error:" + ErrorInfo())
ELSE
	sOID = SNMPStringToOID("dod.internet.mgmt.mib-2.transmission.x25.x25AdmnInterruptTimer")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SNMPStringToOID(<OID to convert>)`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena que contiene el valor del OID en formato numérico,

- Cadena vacía ("") si se produce un error.




**`<OID to convert>: (Cadena de caracteres)`**

Cadena que contiene el OID en formato de cadena para convertir.

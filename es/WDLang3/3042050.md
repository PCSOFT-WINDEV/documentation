


## grSaveBMP (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un gráfico que fue dibujado previamente en un archivo en formato BMP.. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Saves the chart named CHART_MyChart in the "C:\Charts\Chart.BMP" file
grSaveBMP(CHART_MyChart, "C:\Charts\Chart.BMP")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grSaveBMP(<Chart name> , <Name of BMP file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el gráfico,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Name of BMP file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo BMP a crear. La extensión del archivo es obligatoria. Este archivo BMP se crea al llamar a la función **grSaveBMP**. El archivo se sobrescribe y se vuelve a crear si ya existía..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Guardar un gráfico
<a name="guardar_grafico_ELTPARAGRAPHE000102"></a>

- Para guardar un gráfico, éste debe haber sido dibujado previamente ([grDraw](../WDLang3/3042026.md)). Las modificaciones realizadas desde la última llamada a [grDraw](../WDLang3/3042026.md) serán ignoradas.

- El tamaño por defecto de la imagen es set a 639 x 479. Estas dimensiones se tienen en cuenta si la carta nunca fue dibujada o si no es Visible. 




- Si el gráfico se ha creado con [grCreate](../WDLang3/3042005.md), las dimensiones pueden especificarse con [TAMAÑO DE GRINdSize](../WDLang3/3042016.md).



<a name="NOTE0_2"></a>


### Copia de seguridad en formato BMP
<a name="copia_seguridad_formato_bmp_ELTPARAGRAPHE000128"></a>

- Este archivo puede ser leído por una herramienta que reconoce el formato BMP.

- El tamaño del archivo es idéntico al tamaño del destino (tamaño de la ventana, control o archivo).

- Le recomendamos que previsualice el gráfico antes de guardarlo en un archivo.



<a name="NOTE0_3"></a>


### Diferencias entre grDestinationBMP y grSaveBMP
<a name="diferencias_entre_grdestinationbmp_docparampagetitleshort_ELTPARAGRAPHE000139"></a>

- [grDestinationBMP](../WDLang3/3042018.md) se utiliza para definir el destino del siguiente dibujo del gráfico. Cuando se utiliza [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en el archivo especificado y se tendrán en cuenta todos los cambios realizados desde la última vez que se dibujó el gráfico.

- La función **grSaveBMP** permite guardar el gráfico directamente en un archivo en formato BMP. La Image guardada corresponde al gráfico durante la última llamada a [grDraw](../WDLang3/3042026.md). Los cambios realizados en el gráfico desde la última vez que se llamó a [grDraw](../WDLang3/3042026.md) serán ignorados.




## MapDeleteShape (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una forma dibujada en el mapa que se muestra en una control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete a shape
MapDeleteShape(MAP_MyMap, MyShape)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borrar una forma a través de su Variable asociada

`<Result> = MapDeleteShape(<Map control> [, <Shape>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la forma fue borrada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Shape>: (MapCircle, MapPolygon o MapPolyline Variable)`**

La forma que se borrará en el mapa que se muestra en la control Mapa. Esta forma puede describirse utilizando uno de los siguientes tipos de variables: 

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md), 

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md), 

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).


Si no se especifica este parámetro, se borrarán todas las formas del mapa.


<a name="SYNTAX2"></a>

### Borrar una forma a través de su nombre

`<Result> = MapDeleteShape(<Map control> , <Shape identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la forma fue borrada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Cadena de caracteres)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Shape identifier>: (Cadena de caracteres)`**

Identificador de la forma a borrar. Este identificador corresponde al nombre de **** Property de la Variable associated con la forma. Si no se especifica este parámetro, se borrarán todas las formas del mapa.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para añadir una nueva forma a dibujar en una control Mapa, utilice [MapAddShape](../WDLang3/1000025509.md).

- Para modificar una forma dibujada en una control Mapa, utilice [MapModifyShape](../WDLang3/1000025511.md).




## sshSession (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sshSession** permite definir las características de una conexión a un servidor SSH. Las características de esta conexión están descritas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMySession is sshSession
cMySession.Address = "127.0.0.1"
cMySession.Port = 22
cMySession.User = "login"
cMySession.UserPassword = "pass"
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables sshSession
<a name="propiedades_especificas_las_variables_sshsession_ELTPARAGRAPHE000048"></a>

Las siguientes propiedades pueden utilizarse para definir una conexión a un servidor SSH:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Address | Cadena de caracteres | Address del servidor SSH. <br><br>Esta Address se puede dar en el siguiente formato:<br><br>- IP Address en formato XXX.XXX.XXX.XXX (125.5.110.100 por ejemplo).<br><br>- IP Address que contiene el nombre del servidor (SSHServer por ejemplo). Se recomienda esta sintaxis.<br><br><br> |
| ConnectionTimeout | Entero o Duración | Tiempo de espera de la conexión expresado en milisegundos (20 segundos por Default, lo que significa 20000 milisegundos). Esta propiedad puede corresponder a: <br><br>- un número entero que corresponde al número de milisegundos,<br><br>- una variable de tipo Duration,<br><br>- la duración en un formato legible (por ejemplo, 20 s o 20000 ms). <br><br><br> |
| drogadicta | Cadena de caracteres | Nombre de usuario. |
| PrivateKey | Cadena de caracteres o [Certificate](../WDLang1/1000019287.md) Variable | Nombre del Variable de tipo [Certificate](../WDLang1/1000019287.md) que se va a utilizar o archivos de claves en formato OpenSSH (para una conexión por clave privada). <br><br>Si se especifica esta Property, se realiza una autenticación por clave: sólo se tienen en cuenta las propiedades **User**, **PrivateKey** y **PrivateKeyPassword**. |
| PrivateKeyPassword | Cadena de caracteres | Contraseña del archivo de claves (para una conexión por clave privada). La **PrivateKey** Property corresponde a la clave privada que se va a utilizar.  |
| Puerto | Integro | Puerto de conexión (22 de Default). |
| UserPassword | Cadena de caracteres | Contraseña del usuario utilizada en una autenticación con contraseña (la propiedad **User** es el nombre del usuario). |


<a name="NOTE0_2"></a>


### Modo de autenticación
<a name="modo_autenticacion_ELTPARAGRAPHE000123"></a>

La autenticación por clave tiene prioridad sobre la autenticación por contraseña. 

La autenticación con clave se utiliza si: 

- el servidor soporta este modo de autenticación. 

- se especifica el **PrivateKey** Property. 




La autenticación con contraseña se utiliza si: 

- el servidor soporta este modo de autenticación. 

- se especifica la propiedad **UserPassword**.

### Utilización de variables sshSession
<a name="utilizacion_variables_sshsession_ELTPARAGRAPHE000140"></a>

Las variables **sshSession** pueden ser usadas en funciones de WLanguage: 

- sintaxis estándar: 
	


|   |   |
| --- | --- |
| [SSHCommand](../WDLang3/1000021143.md) | Envía un comando a un servidor SSH. La sesión se inicia y se cierra automáticamente tan pronto como se envía el comando. |
| [SSHConnectShell](../WDLang3/1000021147.md) | Inicia una nueva sesión SSH del tipo "Shell". |
| [SSHDisconnectShell](../WDLang3/1000021148.md) | Cierra una sesión de SSH que fue abierta por [SSHConnectShell](../WDLang3/1000021147.md). |
| [SSHRead](../WDLang3/1000021149.md) | Lee los datos encontrados en el buffer de salida de la sesión SSH. |
| [SSHWrite](../WDLang3/1000021150.md) | Escribe los datos en la sesión SSH especificada. |

- sintaxis con prefijo: 
	


|   |   |
| --- | --- |
| [&lt;sshSession Variable&gt;.ConnectShell](../WDLang3/1000021163.md) | Inicia una nueva sesión SSH del tipo "Shell". |
| [&lt;sshSession Variable&gt;.DisconnectShell](../WDLang3/1000021164.md) | Cierra una sesión de SSH que fue abierta por [SSHConnectShell](../WDLang3/1000021147.md). |
| [&lt;sshSession Variable&gt;.Escribir](../WDLang3/1000021166.md) | Escribe los datos en la sesión SSH especificada. |
| [&lt;sshSession Variable&gt;.Leer](../WDLang3/1000021167.md) | Reads the data found on the output buffer of the SSH session. |




## Funciones OLE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>



|   |   |
| --- | --- |
| [GetActiveObject](../WDLang3/3057006.md) | Allows to use an Automation object already started on the current computer. |
| [OLEActivate](../WDLang3/3057008.md) | Edita un objeto OLE contenido en un OLE control. |
| [OLECopy](../WDLang3/3057004.md) | Copia el objeto encontrado en un OLE control en el portapapeles de Windows. |
| [OLECreateCopy](../WDLang3/3057005.md) | Asocia una control OLE con un objeto OLE especificando un archivo base. |
| [OLECreateEmpty](../WDLang3/3057001.md) | Asocia una control OLE con un objeto OLE invocando un servidor OLE específico. |
| [OLELoad](../WDLang3/3057002.md) | Inicializa una control OLE con un objeto OLE. |
| [OLEPaste](../WDLang3/3057003.md) | Pega el contenido del portapapeles de Windows en un OLE control. |
| [OLEPlay](../WDLang3/3057009.md) | Permite:<br><br>- recuperar la lista de operaciones disponibles (que no sean de edición) de un servidor, <br><br>- realizar una operación.<br><br><br> |
| [OLESave](../WDLang3/3057011.md) | Guarda el objeto encontrado en una control OLE en formato binario. |
| [OLEServer](../WDLang3/3057010.md) | Enumera los servidores OLE registrados hacia el sistema actual. |



**Observación**: Las funciones OLE funcionan con Office 2007.

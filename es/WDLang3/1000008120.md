
## La estructura del Grupo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
La estructura de grupo es una estructura preestablecida de WLanguage (no se requiere ninguna declaración).. Esta estructura está acostumbrada:

- crear o modificar un grupo de contactos.

- recuperar el contenido de un grupo de contactos.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las diferentes variables de la estructura del Grupo
<a name="las_diferentes_variables_estructura_del_grupo_ELTTEXTE000142"></a>
La estructura contiene los siguientes miembros:


|   |   |
| --- | --- |
| **mGroup.Contact** | Array de cadenas de caracteres. <br>Nombre de los contactos encontrados en el grupo. <br><br>Para acceder al nombre de los diferentes contactos del grupo, utilice el siguiente formato: <br>mGroup.Contact[&lt;Contact num&gt;]<br>Donde &lt;número de Contact&gt; corresponde al número de la Contact deseada . El número de contactos encontrados en el grupo es devuelto por el mGroup.NbContact Variable. |
| **mGroup.Description** | Cadena de caracteres. <br>description del grupo de contactos. |
| **mGroup.ID** | Cadena de caracteres.<br>Identificador único para el grupo de contactos |
| **mGroup.Name** | Cadena de caracteres.<br>Nombre del grupo de contactos. |
| **mGroup.NbContacto** | Integro. <br>Número de contactos encontrados en el grupo. |
| **mGroup.Out** | booleano. <br>Indica si hay un grupo actual de contactos. |

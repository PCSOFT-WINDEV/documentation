
## Conexión SaaSClientConnection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las características de la conexión a la base de datos de clientes.. Esta función permite a un usuario conectado a través de [SaaSConnect](../WDLang3/1000018959.md) acceder a la base de datos del cliente que le pertenece.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection is Connection

IF SaaSIsConnected() = True THEN
	MyConnection = SaaSClientConnection()
	HChangeConnection("*", MyConnection)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSClientConnexion()`
---

**`<Result>: (Variable de tipo Connection)`**

[Conexión](../WDLang4/1514073.md) variable que contiene las características de la conexión a la base de datos del cliente.

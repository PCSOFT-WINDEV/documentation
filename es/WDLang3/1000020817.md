
## NotifPushDisable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva la gestión de las notificaciones push para una aplicación WINDEV Mobile (Android o IOS).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NotifPushDisable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`NotifPushDisable([<WLanguage procedure>])`
---

**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

Nombre del WLanguage Procedure llamado cuando se realizó la desactivación.

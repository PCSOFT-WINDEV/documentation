


## grAutoRefreshSeries (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o define si una serie se actualiza cada vez que se dibuja el gráfico.. En este caso, el contenido de la serie se recalcula automáticamente cada vez que se llama a [grDraw](../WDLang3/3042026.md).

**Ejemplo de uso**: una serie en un gráfico se utiliza como serie reference: esta serie no se actualiza automáticamente.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
IF grAutoRefreshSeries("MyChart", 5) = True THEN
	grDraw("MyChart")
END
```
<a name="Example2"></a>

```wl
// Chart control
IF grAutoRefreshSeries(CHART_MyChart, 5) = True THEN
	grDraw(CHART_MyChart)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificando el modo utilizado para actualización la serie

`<Result> = grAutoRefreshSeries(<Chart name> , <Series number>)`
---

**`<Result>: (booleano)`**

Modo actual para la actualización de la serie: 

- <u><u><u><u>True</u></u></u></u> si la serie se actualiza automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie afectada.  


<a name="SYNTAX2"></a>

### Definiendo el modo utilizado para actualización la serie

`grAutoRefreshSeries(<Chart name> , <Series number> , <Refresh mode>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Series number>: (Integro)`**

Número de la serie afectada.

**`<Refresh mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la serie debe actualizarse automáticamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

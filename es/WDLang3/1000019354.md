
## WiFiStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado actual de activación de Wi-Fi en el dispositivo o solicita que se le notifique cuando cambie el estado de activación de Wi-Fi.

**Observaciones**: 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF WiFiStatus() = wifiDisabled THEN
	WiFiActivate(True)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estado actual de activación de Wi-Fi en el ordenador actual

`<Result> = WiFiStatus()`
---

**`<Result>: (Integro)`**

Estado actual de activación de Wi-Fi:


|   |   |
| --- | --- |
| *wifiActivationInProgress* | El Wi-Fi está siendo habilitado en el dispositivo.<br><br> |
| *wifiDeactivationInProgress* | El Wi-Fi está siendo desactivado en el dispositivo.<br><br> |
| *wifiDisabled* | El Wi-Fi está desactivado en el dispositivo. |
| *wifiEnabled* | El Wi-Fi está habilitado en el dispositivo. |
| *wifiError* | No se puede recuperar el estado de activación de Wi-Fi.<br><br> |




## <Chart>.SurfaceMarkCaption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite especificar el título de la marca para un gráfico de superficie.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_MyChart.SurfaceMarkCaption(grXCoordinate, 1, "January")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceMarkCaption(<Mark axis> , <Value> , <Caption>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).

**`<Mark axis>: (Constante de tipo Integer)`**

Tipo de eje en el que se visualizarán las marcas:


|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |
| *grZCoordinate* | Eje Z. |



**`<Value>: (Real)`**

Valor de la marca para la que se modificará la leyenda.

**`<Caption>: (Cadena de caracteres)`**

Nueva leyenda para la marca.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de marcas
<a name="gestion_marcas_ELTPARAGRAPHE000209"></a>

Para que se muestre el título de la marca: 

- debe existir una marca con el valor especificado. De lo contrario, no se mostrará el título de la marca.

- la marca debe tener un valor. El título no se mostrará si una marca no tiene valor.




**Recordatorio**: Las marcas de una carta de Superficie pueden ser definidas: 

- en el editor, en la ventana control Gráfico description (pestaña "General"). 

- mediante programación con la función [.interval](../WDLang3/1000024076.md).

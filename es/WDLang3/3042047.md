


## grLegend (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la presencia y posición de la leyenda en un gráfico.

Ejemplo de leyenda en un gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Legende.gif)


**Observación**: La leyenda contiene los colores y las etiquetas de serie (o etiquetas de sección) de un gráfico.. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
grCreate("MyChart", grColumn)
grDestinationWnd("MyChart", "", "Title of the window")
// The legend is displayed on the right of "MyChart"
grLegend("MyChart", grAtRight)
...
grDraw("MyChart")
```
<a name="Example2"></a>

```wl
// Using a Chart control
// The legend is displayed on the left of CHART_MyChart
grLegend(CHART_MyChart, grAtLeft)
...
grDraw(CHART_MyChart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grLegend(<Chart name> , <Position>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Position>: (Integro)`**

Posición de la leyenda del gráfico:


|   |   |
| --- | --- |
| *grAtBottom* | Leyenda mostrada debajo del gráfico. |
| *grAtLeft* | Leyenda mostrada a la izquierda del gráfico. |
| *grAtRight* | Leyenda mostrada a la derecha del gráfico. |
| *grAtTop* | La leyenda del gráfico mostró a Above el gráfico. |
| *grNone* | Sin leyenda. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Leyenda del gráfico
<a name="leyenda_del_grafico_ELTPARAGRAPHE000153"></a>Las etiquetas de la serie set corresponden a la leyenda de la carta.
Las etiquetas de las series de <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 no pueden utilizarse en los gráficos circulares.

Las etiquetas de las series se definen con [grSeriesLabel](../WDLang3/3042024.md). 

La fuente de la leyenda puede cambiarse con [grLegendFont](../WDLang3/3042037.md).  

**grLegend** indica la presencia y la posición de la leyenda (es decir, todas las etiquetas de las series). Las modificaciones realizadas con la función **grLegend** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md). Puede cambiar la posición de la leyenda del gráfico mediante un menú contextual.

**Observación**: Para definir las etiquetas de las series en los gráficos de "dispersión", le recomendamos que utilice [grScatterSeriesLabel](../WDLang3/3042038.md). Para obtener más información, consulte [Leyendas de una carta](../WDChamp/3042069.md). 
<a name="NOTE0_2"></a>


### Posicionamiento manual de la leyenda: grLegend sin efecto
<a name="posicionamiento_manual_leyenda_grlegend_sin_efecto_ELTPARAGRAPHE000192"></a>

Si la leyenda se posiciona manualmente (opción "Posicionamiento manual" de la ventana de la leyenda del gráfico), la función **grLegend** no tiene ningún efecto.

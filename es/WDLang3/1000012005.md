
## TaskFirst( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la primera tarea encontrada en la lista de tareas de un mensaje de Lotus Notes o de Outlook.. Esta tarea se convierte en la tarea actual.

Al leer una tarea, los datos de ésta se actualizan en la estructura [Tarea](../WDLang3/1000012010.md). Para recuperar los datos de la tarea actual, basta con recuperar los valores de la estructura [Tarea](../WDLang3/1000012010.md).

**Atención**: Esta función **no se puede utilizar** para leer la primera tarea encontrada en la lista de tareas de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf",...
					"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Read the first task
TaskFirst(SessionID)
// Add the tasks found into a list
WHILE mTask.Out = False
	ListAdd(LIST_TaskList, mTask.Subject + TAB + mTask.Category)
	TaskNext(SessionID)
END
```


<a name="Example2"></a>



```wl
// Start an Outlook session
SessionID is int
SessionID = OutlookStartSession("MyProfile")
// Read the first task
TaskFirst(SessionID)
// Add the tasks found into a list
WHILE mTask.Out = False
	ListAdd(LIST_TaskList, mTask.Subject + TAB + mTask.Category)
	TaskNext(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TaskFirst(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la primera tarea,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

## Observaciones




### Tarea actual
<a name="tarea_actual_ELTPARAGRAPHE000070"></a>

El mTask.Out Variable se utiliza para averiguar si existe una tarea actual.

## Presentación
<a name="presentacion_ELTTEXTE000073"></a>
Las aplicaciones desarrolladas en WINDEV Mobile se pueden ejecutar en las siguientes plataformas:

- Universal Windows Platform Apps.

- Android versión 5 y posteriores.

- iOS versión 10 y posteriores.




Se pueden utilizar los siguientes procesadores:

- ARM y compatible (Strong ARM, XScale, Samsung, Texas Instrument,...).

- ARM v4T y compatible (XScale, ....).






Windows Mobile, Windows CE y Windows Embedded ya no reciben soporte de Microsoft, pero se siguen utilizando en el sector industrial. 
Si desea seguir desarrollando para Windows Mobile, debe utilizar la versión 26. La versión 28 le da acceso a la versión 26, que se puede descargar [aquí](http://windev.com/dl.htm).




## &lt;Tabla&gt;.ScatterSeriesLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la etiqueta de las series de datos en un gráfico "Scatter".

Ejemplo de etiqueta de serie: <br>![Etiqueta de la serie](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)


**Recordatorio**: Por Default, un gráfico no tiene etiqueta.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initializes the "Purchase" label of third data series in CHART_MyChart
CHART_MyChart.ScatterSeriesLabel(3, "Purchase")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.ScatterSeriesLabel(<Series number> , <Label>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie para la que se inicializará una etiqueta.

**`<Label>: (Cadena de caracteres)`**

Leyenda de la etiqueta a inicializar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El tipo de letra de la etiqueta puede ser modificado por [&lt;Tabla&gt;.LabelFont](../WDLang3/1000024117.md).

- Los datos definidos con la función **&lt;Tabla&gt;.ScatterSeriesLabel** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- La función **&lt;Tabla&gt;.ScatterSeriesLabel** es equivalente a la función [&lt;Tabla&gt;.SeriesLabel](../WDLang3/1000024073.md). 
	La sintaxis de [&lt;Tabla&gt;.SeriesLabel](../WDLang3/1000024073.md) utilizada con los gráficos de "dispersión" es la siguiente:
	
	```txt
	<Chart control>.SeriesLabel(2*<Index>-1, <Label>)
	```
**Reminder**: Un gráfico de "dispersión" utiliza los datos en grupos de dos series (para más detalles, véase [.ScatterAddDataXY](../WDLang3/1000024105.md)).




## &lt;Tabla&gt;.SurfaceAltitudeColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define la Color utilizada para la altitud (eje Z) en un gráfico de superficie.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Surface.SurfaceAltitudeColor(100, LightRed)
CHART_Surface.SurfaceAltitudeColor(0, PastelBlue)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.SurfaceAltitudeColor(<Altitude> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).

**`<Altitude>: (Real)`**

Altitud para la cual se aplicará el Color.

**`<Color>: (Entero o constante de tipo Integer)`**

Color para la altitud especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si se especifican colores de altitud, las facetas encontradas a esta altitud se colorearán en consecuencia. 

Para las facetas que se encuentran entre dos colores, se aplicará un Gradient lineal.

Para las facetas fuera de los límites de Color, se aplicará el Color de la serie.

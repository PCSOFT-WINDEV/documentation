
## AppointmentRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee una cita que se encuentra en el calendario de un mensaje de Lotus Notes o Outlook. 

**Recordatorio**: Esta función no se puede utilizar para leer la cita que se encuentra en la Calendar de un mensaje de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf", "C:\Lotus Notes\julia.id")
// Read the first appointment
AppointmentFirst(SessionID)
...
// Read this same appointment again
AppointmentRead(SessionID, mAppointment.ID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppointmentRead(<Session identifier> , <Appointment identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la cita,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

**`<Appointment identifier>: (Cadena de caracteres)`**

Identificador de la Appointment para leer. Este identificador se encuentra en el mAppointment.ID Variable.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La variable mAppointment.Out permite averiguar si hay un Appointment actual.

- En Outlook, se puede especificar el modo de gestión de los contactos/participantes al iniciar la sesión ([EmailStartOutlookSession](../WDLang3/3032128.md) o [OutlookStartSession](../WDLang3/3032123.md)).

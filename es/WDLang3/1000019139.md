
## SaaSAdminSubscriptionAllocateLicense( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Asigna una licencia de suscriptor a un usuario de SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El Administrador del SaaS tendrá acceso a todas las licencias. Un gestor de cuenta de cliente sólo tendrá acceso a las licencias de sus usuarios.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is saasSite
P is saasPricing
C is saasClient
Sub is saasSubscription
U is saasUser

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
C = SaaSAdminFindClient("Hybrid")
Sub = SaaSAdminFindSubscription(P, C)
U = SaaSAdminFindUser("mark@hybrid.com")

// Allows the usernamed mar@hybrid.com benefit from the promotion 2010
SaaSAdminSubscriptionAllocateLicense(Sub, U)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Asignar una licencia en una suscripción

`<Result> = SaaSAdminSubscriptionAllocateLicense(<Subscription> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la asignación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Subscription>: (saasSubscription Variable)`**

Nombre de la [saasSubscription](../WDLang3/1000019088.md) Variable correspondiente a la suscripción que debe ser modificada en la base de datos SaaS.

**`<User>: (saasUser Variable)`**

Nombre de la [saasUser](../WDLang3/1000019083.md) Variable que identifica al usuario correspondiente. Este usuario recibe una licencia que le permite beneficiarse de la suscripción.





### Asignar una licencia sobre un precio

`<Result> = SaaSAdminSubscriptionAllocateLicense(<Pricing> , <SaaS site> , <User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la asignación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre de la determinación de precio relevante. Una suscripción debe ser associated con este precio para la cuenta de cliente a la que pertenece el usuario.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS al que pertenece el precio.

**`<User>: (Cadena de caracteres)`**

Login del usuario que recibe una licencia que le permite beneficiarse de la suscripción.

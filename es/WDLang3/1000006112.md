
## ContactRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee un contacto en una libreta de Lotus Notes o Outlook Address.
**Atención**: Esta función **no se puede utilizar** para leer un Contact en un libro Address de Outlook Express.

Para ser leído, este Contact debe haber sido: 

- leer ([ContactLast](../WDLang3/1000006103.md), [ContactPrevious](../WDLang3/1000006105.md), [ContactFirst](../WDLang3/1000006106.md) o [ContactNext](../WDLang3/1000006108.md)),

- añadido ([ContactAdd](../WDLang3/1000006101.md)),

- modificado ([ContactModify](../WDLang3/1000006104.md)),

- buscado ([ContactSearch](../WDLang3/1000006102.md)). 




**ContactRead** no set la posición en el Contact leer.   

Al leer un Contact, los datos de este Contact se actualizan en el [Estructura de Contact](../WDLang3/1000006111.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf", ...
		"C:\Lotus Notes\julia.id")
// Read the first contact
ContactFirst(SessionID)
...
// Read this same contact again
ContactRead(SessionID, mContact.ID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ContactRead(<Session identifier> , <Contact identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó el contacto,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).

**`<Contact identifier>: (Cadena de caracteres)`**

Identificador de la Contact para leer. Este identificador se encuentra en el mContact.ID Variable.

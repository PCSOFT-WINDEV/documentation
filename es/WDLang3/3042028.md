


## grLineThickness (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera y/o modifica el valor del grosor de la Line en un gráfico de Line.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
// Modifies the line thickness for all the series of the chart
grLineThickness("MyChart", grAllSeries, 4)
```
<a name="Example2"></a>

```wl
// Chart control
// Modifies the line thickness for all the series of the chart
grLineThickness(CHART_MyChart, grAllSeries, 4)
```

<a name="SYNTAX1"></a>

## Sintaxis

### Recuperando el grosor de la Line (carta de Line)

`<Result> = grLineThickness(<Chart name> , <Series number>)`
---

**`<Result>: (Integro)`**

Grosor de la línea.

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).


Este gráfico debe ser un gráfico Line.



**`<Series number>: (Integro)`**

Número de la serie para la que se solicita el espesor de Line.  


<a name="SYNTAX2"></a>

### Modificando el grosor de la Line (carta de Line)

`grLineThickness(<Chart name> , <Series number> , <New thickness>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).


Este gráfico debe ser un gráfico Line.



**`<Series number>: (Integro)`**



- Número de la serie para la que debe modificarse el espesor de Line. 

- Constante *grAllSeries* para cambiar el grosor de todas las series (incluidas las nuevas series).




**`<New thickness>: (Integro)`**

El nuevo grosor de la Line. Este parámetro puede corresponder a un valor superior a 0. El grosor de la línea es set a 1 de Default.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Unidades que se tienen en cuenta para el espesor
<a name="unidades_que_tienen_cuenta_para_espesor_ELTPARAGRAPHE000176"></a>

El espesor de la Line corresponde a valores mayores o iguales a 0.

- Si el grosor corresponde a 0, el grosor será de 1 píxel en la pantalla (lo que significa 1 pica al imprimir).

- Si el grosor es superior a 1, el grosor será de x píxeles en la pantalla, lo que significa x picas al imprimir.



<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000188"></a>

- Si se utiliza la función **grLineThickness** en un gráfico diferente al gráfico de líneas, no se devuelve ningún error. La información se almacena y se utilizará cuando se modifique el tipo de gráfico (por [grType](../WDLang3/3042062.md) o con el menú contextual del gráfico).

- Equivalencia: la función **grLineThickness** es equivalente a la función [Espesor](../Proprietes/2510144.md).

- Los datos definidos con la función **grLineThickness** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

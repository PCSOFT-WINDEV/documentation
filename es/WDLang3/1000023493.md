


## grScatter3DGridlines (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra las líneas de cuadrícula para los diferentes planos de un gráfico de dispersión 3D.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grSurfaceQuadrillage.gif)
Observación: Las líneas de la cuadrícula de un gráfico de dispersión 3D también pueden ser activadas/desactivadas en la pestaña "General" de la ventana description de control Gráfico. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grScatter3DGridlines(CHART_Chart1, grPlaneXZ, grGridlinesX + grGridlinesZ)
grScatter3DGridlines(CHART_Chart1, grPlaneXY, grNoGridlines)
grScatter3DGridlines(CHART_Chart1, grPlaneXY, grNoGridlines)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grScatter3DGridlines(<Chart name> , <Plane> , <Axis>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas).




**`<Plane>: (Constantes de tipo Integer)`**

Plano sobre el que se aplican las líneas de la cuadrícula:


|   |   |
| --- | --- |
| *grPlaneXY* | Líneas de cuadrícula del plano XY. |
| *grPlaneXZ* | Líneas de cuadrícula del plano XZ. |
| *grPlaneYZ* | Líneas de cuadrícula del plano YZ. |



**`<Axis>: (Constante o combinación de constantes de tipo Integer)`**

Eje sobre el que se aplican las cuadrículas: 


|   |   |
| --- | --- |
| *grGridlinesX* | Las líneas de cuadrícula se muestran en el eje X (disponible si &lt;Plano&gt; corresponde a la constante *grPlaneXZ* o a la constante *grPlaneXY*). |
| *grGridlinesZ* | Las líneas de cuadrícula se muestran en el eje Z (disponible si el parámetro &lt;Plano&gt; corresponde a la constante *grPlaneXZ* o *grPlaneYZ*). |
| *GridlinesY* | Las líneas de cuadrícula se muestran en el eje Y (disponible si &lt;Plano&gt; corresponde a la constante *grPlaneXY* o a la constante *grPlaneYZ*). |
| *grNoGridlines* | Las líneas de la cuadrícula no se muestran. |
| *grPlaneXY* | Las líneas de cuadrícula se muestran en el plano XY (disponible si el parámetro &lt;Plano&gt; corresponde a la constante *grPlaneXY*). |
| *grPlaneXZ* | Las líneas de cuadrícula se muestran en el plano XZ (disponible si el parámetro &lt;Plano&gt; corresponde a la constante *grPlaneXZ*). |
| *grPlaneYZ* | Las líneas de la cuadrícula se muestran en el plano YZ (disponible si &lt;plano&gt; corresponde a la constante *grPlaneYZ*). |


## ¿Cómo incluir los servicios de SESAM VITALE?
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000071"></a>
A partir de la versión 18 de WINDEV, la importación de webservices incluye las herramientas necesarias para permitir el consumo de los Webservices secure utilizados para gestionar las tarjetas Sesam Vitale.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000095"></a>
El uso de una tarjeta Sesam Vitale en una aplicación no requiere una set específica de comandos de WLanguage: no hay función Sesam\*. 

Las tarjetas Sesam Vitale se gestionan a través de Webservices: todo lo que tiene que hacer es importar los servicios web requeridos por la característica a implementar.

Estos servicios web son suministrados por el SESAM-VITALE GIE a través de un acuerdo Process del software que los utilizará. Este acuerdo Process es realizado por la CNDA. Para obtener más información, consulte [http://www.cnda.ameli.fr/](http://www.cnda.ameli.fr/)

Para desarrollar aplicaciones que utilicen los servicios web de Sesam Vitale, debes registrarte en la CNDA para realizar el acuerdo Process. El sitio de la CNDA suministra todos los elementos necesarios para llevar a cabo este Process.


## WebserviceWriteHTTPHeader( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una cabecera HTTP adicional a una solicitud de un servicio web REST.. Esta cabecera HTTP se devolverá al cliente con la respuesta la próxima vez que se llame al servicio web REST.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allows the call to the webservice from any site in a browser.
WebserviceWriteHTTPHeader("Access-Control-Allow-Origin", "*")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WebserviceWriteHTTPHeader(<Name of HTTP Header> , <Header value>)`
---

**`<Name of HTTP Header>: (Cadena de caracteres)`**

Nombre del encabezado HTTP a escribir.

**`<Header value>: (Cadena de caracteres)`**

Valor (o contenido) de la cabecera HTTP a escribir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El WEBDEV servidor de aplicaciones fuerza el valor de algunas cabeceras HTTP ("Content-Length" por ejemplo). En este caso, el valor escrito por **WebserviceWriteHTTPHeader** será ignorado.

- Durante las llamadas sucesivas a esta función para el mismo nombre de cabecera HTTP, sólo se tiene en cuenta el último valor.

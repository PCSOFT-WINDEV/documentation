


## &lt;Gráfico&gt;.suavizado (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Suaviza un gráfico de líneas o dispersión con el método "Splines cúbicos". 

**Recordatorio**: Un gráfico no es suavizado por Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Smooth the chart displayed in CHART_MyChart
CHART_MyChart.Smoothing(True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Alisar un gráfico

`<Chart control>.Smoothing(<Smoothing>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Smoothing>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe activar el suavizado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.





<a name="SYNTAX2"></a>

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se tiene en cuenta el alisamiento:

- en los gráficos de líneas (constante *grLine*),

- en gráficos de dispersión (*grScatter* constante) con puntos relacionado ([&lt;Gráfico&gt;.ScatterLinkPoint](../WDLang3/1000024109.md)).




**Atención**: Esta función no tiene ningún efecto sobre un gráfico 3D.

Los parámetros definidos por **&lt;Gráfico&gt;.suavizado** se tendrán en cuenta en la siguiente llamada a [&lt;Dibujo](../WDLang3/1000024062.md).




## btleDevice (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **btleDevice** le permite manipular un dispositivo que utiliza la tecnología Bluetooth Low Energy.

Las características de este dispositivo pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
myDevice is btleDevice
BTLEConnect(myDevice)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables btleDevice
<a name="propiedades_especificas_las_variables_btledevice_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **btleDevice**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre del dispositivo. Este nombre puede corresponder a una cadena vacía ("""). <br><br>Esta propiedad es de solo lectura. |
| Identificador | Cadena de caracteres | Identificador del dispositivo.<br><br><br><br>Esta propiedad es de solo lectura. |
| Servicio | Array de [btleService](../WDLang3/1000022057.md) | Lista de servicios Bluetooth LE soportados por el dispositivo. Debe estar conectado al dispositivo (con [BTLEConnect](../WDLang3/1000021884.md)) para poder acceder a este Property. De lo contrario, se producirá un error fatal. |


## NotesDocumentDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina el actual Document de la base de datos de Lotus Notes que se ha navegado.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
...
// Read the first document 
NotesDocumentFirst(SessionID)
// Enumerate the documents found
WHILE NotesOut(SessionID) = False
	// Deletes the document
	NotesDocumentDelete(SessionID)
	// Next document found
	NotesDocumentNext(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesDocumentDelete(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el documento actual,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [NotasOpenConnection](../WDLang3/1000005102.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

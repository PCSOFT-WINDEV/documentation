
## NFCStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de activación del sensor NFC que se encuentra en el dispositivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NFCStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de activación del sensor: 


|   |   |
| --- | --- |
| *nfcDisabled* | El sensor NFC está desactivado. |
| *nfcEnabled* | El sensor NFC está habilitado.<br><br> |
| *nfcUnavailable* | El dispositivo no está equipado con un sensor NFC o éste no está disponible. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000059"></a>

- La función **NFCStatus** no se puede utilizar ni en el simulador ni en el emulador: la función devolverá la constante *nfcUnavailable*.


## beaconGroup (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **beaconGroup** se utiliza para definir las características avanzadas para un set de Balizas. Las características de esta set de balizas pueden definirse y modificarse a través de diferentes propiedades del lenguaje WL. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

Recordatorio: Las balizas utilizan la tecnología Bluetooth de baja energía para emitir un triplete de valores: 

- UUID (Universally Unique Identifier, también conocido como UUID de proximidad): identificador único que generalmente representa la organización o la persona a la que pertenece el Faro.

- Mayor: número utilizado para identificar un grupo de balizas dentro de la misma organización (mismo UUID)

- Menor: número utilizado para identificar una baliza específica dentro del mismo grupo (mismo UUID y mismo número de Mayor).


Este trío se utiliza para identificar el Faro de una manera única. 

Utilizando el primer valor o los dos primeros valores del trío, puede determinar si un Faro pertenece a una organización y/o a un subgrupo de esta organización.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Group of all Beacons from the same organization
group is beaconGroup
group.UUID = "f4231ab6-5ef2-6c99-4229-af6c72e0446e" 

// Group of all Beacon tags belonging to the same organization with Major number 5
group is beaconGroup
group.UUID = "f4231ab6-5ef2-6c99-4229-af6c72e0446e" 
group.Major = 5

// Unique Beacon
group is beaconGroup
group.UUID = "f4231ab6-5ef2-6c99-4229-af6c72e0446e" 
group.Major = 5
group.Minor = 1
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables beaconGroup
<a name="propiedades_especificas_las_variables_beacongroup_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un grupo de Faros:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| description | Cadena de caracteres | description del grupo de Faros. |
| Mayor | Integro | Entero incluido entre -1 y 65535 describiendo el número Mayor del grupo de Balizas. <br><br>- El valor por defecto de Property es set a -1. Este valor significa que el número Mayor será ignorado para determinar si un Faro pertenece al grupo. <br><br>- Si se ha especificado el valor de esta Property (distinto de -1), también se debe especificar la **UUID** Property.<br><br><br> |
| Menor | Integro | Entero incluido entre -1 y 65535 describiendo el número Menor del grupo de Balizas. <br><br>- El valor por defecto de Property es set a -1. Este valor significa que el número Menor será ignorado para determinar si un Faro pertenece al grupo. <br><br>- Si se ha especificado el valor de esta Property (distinto de -1), también se deben especificar las propiedades **UUID** y **Major**.<br><br><br> |
| UUID | Cadena de caracteres | Identificador (GUID) común a los faros del grupo. Este identificador incluye 32 dígitos hexadecimales separados en 5 grupos. Cada grupo debe contener el siguiente número de dígitos: <br><br>- 1er grupo: 8 dígitos.<br><br>- 2º grupo: 4 dígitos.<br><br>- 3er grupo: 4 dígitos.<br><br>- 4º grupo: 4 dígitos.<br><br>- 5º grupo: 12 dígitos.<br><br><br>Los diferentes grupos están separados por un guión. <br><br>Ejemplo de identificador válido: f4231ab6-5ef2-6c99-4229-af6c72e0446e<br><br>Este identificador es obligatorio para que el grupo sea válido. |

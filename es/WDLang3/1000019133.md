
## SaaSAdminPricingDisableService( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva un servicio con un precio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
S is dynamic saasSite 
P is dynamic saasPricing 
CRMService is dynamic saasService 

S = SaaSAdminFindSite("Forum")
P = SaaSAdminFindPricing("Promo 2010", S)
CRMService = SaaSAdminFindService("CRM", S)

// Disables the CRM service for the subscribers of "Promo 2010"
SaaSAdminPricingDisableService(P, CRMService)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Desactivar un servicio (mediante variables saasxxx)

`<Result> = SaaSAdminPricingDisableService(<Pricing> , <Department>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la desactivación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (saasPricing Variable)`**

Nombre de la [saasPricing](../WDLang3/1000019091.md) Variable correspondiente a la tarificación a utilizar. El servicio será eliminado de este precio.

**`<Department>: (saasService Variable)`**

Nombre de [saasService](../WDLang3/1000019096.md) Variable que identifica el servicio que será eliminado de la lista de servicios suministrados por la tarificación.


<a name="SYNTAX2"></a>

### Desactivar un servicio identificado por su nombre

`<Result> = SaaSAdminPricingDisableService(<Pricing> , <Department> , <SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la desactivación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pricing>: (Cadena de caracteres)`**

Nombre del precio. El servicio tendrá que ser eliminado de este precio.

**`<Department>: (Cadena de caracteres)`**

Nombre del servicio que será eliminado de la lista de servicios suministrados por el precio.

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS afectado por el precio y el servicio.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El servicio no estará disponible automáticamente para todos los suscriptores a este precio.

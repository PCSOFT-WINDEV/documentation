


## Funciones NFC
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el NFC: 



|   |   |
| --- | --- |
| [NFCReadTag](../WDLang3/1000020416.md) | Inicia la lectura de una etiqueta NFC o permite la detección de etiquetas NFC para la ventana actual. |
| [NFCSendTag](../WDLang3/1000020418.md) | Envía una etiqueta NFC a otro dispositivo. |
| [NFCStatus](../WDLang3/1000020415.md) | Devuelve el estado de activación del sensor NFC que se encuentra en el dispositivo. |
| [NFCWriteTag](../WDLang3/1000020417.md) | Inicia la escritura de datos en una etiqueta NFC. |



Observaciones: 

- Las funciones NFC admiten las etiquetas correspondientes a los tipos estandarizados 1 a 4 definidos por el Foro NFC ([https://nfc-forum.org/build/specifications#tag-type-technical-specifications](https://nfc-forum.org/build/specifications#tag-type-technical-specifications)) y sólo pueden leer y escribir datos codificados en el formato NDEF (NFC Data Exchange Service).

- Las funciones NFC no se pueden utilizar en el simulador ni en el emulador.


## Telefonía: Gestión de las llamadas salientes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Etapas a seguir
<a name="etapas_seguir_ELTTEXTE000098"></a>
Gestionar las llamadas salientes en una aplicación WINDEV:

1. Definir (si es necesario) el dispositivo TAPI en el que se deben marcar las llamadas.. Utilice las funciones:
	


|   |   |
| --- | --- |
| [tapiCapability](../WDLang3/3076014.md) | Devuelve las características de un dispositivo de telefonía. |
| [tapiDevice](../WDLang3/3076032.md) | Selecciona el dispositivo TAPI que se utilizará durante las siguientes operaciones de telefonía: <br><br>	- [tapiDial](../WDLang3/3076013.md),<br><br>	- [tapiListen](../WDLang3/3076029.md).<br><br><br> |
| [tapiDeviceList](../WDLang3/3076024.md) | Enumera los dispositivos compatibles con TAPI 2.0 y TAPI 3.1 instalados en el ordenador actual. |

2. Marque el número de Phone a través del módem ([tapiDial](../WDLang3/3076013.md)).

3. Esta función llama a un WLanguage Procedure específico utilizado para gestionar el progreso de la llamada. En este Procedure, las siguientes variables permiten conocer el estado del Line:

	- *tapiLineBusy*: El Line está actualmente ocupado. 

	- *tapiLineConnected*: El Line está conectado

	- *tapiLineDialing*: Marcación en curso

	- *tapiLineDialTone*: El Line recibe un tono de llamada

	- *tapiLineDisconnected*: El corresponsal ha colgado

	- *tapiLineProceeding*: La llamada se marca: búsqueda del corresponsal

	- *tapiLineRingBack*: Timbre en curso




4. Coge el receptor de Phone para comunicarte.

5. Durante la comunicación, se pueden utilizar las funciones de WLanguage para manejar la llamada (ver abajo).. No es necesario especificar el identificador de la llamada porque la llamada procesada será la llamada actual (es decir, la llamada saliente).

6. Al final de la llamada, cierre la Line con [tapiHangUp](../WDLang3/3076030.md).

7. Cuelga el receptor de Phone.

## Manejo de una llamada saliente
<a name="manejo_una_llamada_saliente_ELTTEXTE000146"></a>
Se pueden realizar las siguientes operaciones en las llamadas salientes:

- averiguar las características de la llamada saliente:
	


|   |   |
| --- | --- |
| [tapiCallDuring](../WDLang3/3076004.md) | Devuelve la duración de la llamada (diferencia entre la fecha y hora de inicio de la llamada y la fecha y hora de finalización de la misma). |
| [tapiCalledID](../WDLang3/3076025.md) | Se utiliza para averiguar el número de Phone llamado. |
| [tapiCallEnd](../WDLang3/3076002.md) | Devuelve la fecha y hora de finalización de la llamada. |
| [tapiCallerID](../WDLang3/3076010.md) | Se utiliza para averiguar el número de Phone que llama (el que llama). |
| [tapiCallIsBusy](../WDLang3/3076017.md) | Se usa para averiguar si el número al que se llama está ocupado. |
| [tapiCallIsOver](../WDLang3/3076003.md) | Permite saber si la llamada entrante o saliente ha finalizado. |
| [tapiCallStart](../WDLang3/3076006.md) | Devuelve la fecha y hora de inicio de la llamada (llamada entrante o saliente). |
| [tapiNoAnswer](../WDLang3/3076008.md) | Se utiliza para averiguar si se ha dado una respuesta a la llamada. |

- realizar operaciones específicas:
	


|   |   |
| --- | --- |
| [tapiKeyPressed](../WDLang3/3076027.md) | Devuelve el historial de las teclas pulsadas en el teclado Phone desde la última llamada a tapiKeyPressed. |
| [tapiPlay](../WDLang3/3076023.md) | Reproduce un archivo de sonido (.WAV) para la Line especificada . |
| [tapiRecord](../WDLang3/3076019.md) | Records the current communication as a".WAV" file. |
| [tapiSendKey](../WDLang3/3076018.md) | Permite simular el uso de las teclas de Phone. |
| [tapiStop](../WDLang3/3076007.md) | Deja de leer un mensaje pregrabado ([tapiPlay](../WDLang3/3076023.md)). |

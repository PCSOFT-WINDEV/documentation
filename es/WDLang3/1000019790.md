


## MapGetPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición geográfica de un punto localizado en el centro del mapa que se está mostrando en un control Map.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 
```wl
// Returns the point in the map corresponding to the current user position
MyLocation is geoPosition
MyPosition = MapGetPosition(MAP_Position)
IF ErrorOccurred = False THEN
	RESULT MapInfoPosition(MAP_Position, MyLocation)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapGetPosition([<Map control>])`
---

**`<Result>: (geoPosition Variable)`**

[geoPosition](../WDLang3/1000019191.md) Variable correspondiente a la posición geográfica del punto actualmente visualizado en el centro del control Mapa.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.

**`<Map control>: (Nombre de control opcional)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, se utilizará la control Mapa a la que pertenece la actual Event.

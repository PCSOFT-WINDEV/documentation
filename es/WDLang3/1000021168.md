


## Funciones SSH (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para conectarse y comunicarse con un servidor SSH: 



|   |   |
| --- | --- |
| [SSHCommand](../WDLang3/1000021143.md) | Envía un comando a un servidor SSH. La sesión se inicia y se cierra automáticamente tan pronto como se envía el comando. |
| [&lt;sshSession Variable&gt;.ConnectShell](../WDLang3/1000021163.md) | Inicia una nueva sesión SSH del tipo "Shell". |
| [&lt;sshSession Variable&gt;.DisconnectShell](../WDLang3/1000021164.md) | Cierra una sesión de SSH que fue abierta por [&lt;sshSession Variable&gt;.ConnectShell](../WDLang3/1000021163.md). |
| [&lt;sshSession Variable&gt;.Leer](../WDLang3/1000021167.md) | Lee los datos encontrados en el buffer de salida de la sesión SSH. |
| [&lt;sshSession Variable&gt;.Escribir](../WDLang3/1000021166.md) | Escribe los datos en la sesión SSH especificada. |



Observación: El protocolo es SSH-2.

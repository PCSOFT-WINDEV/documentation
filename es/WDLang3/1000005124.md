
## NotesModifyAttachment( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un archivo adjunto a un elemento de la actual Document en Lotus Notes.

Para tener en cuenta la modificación del archivo, el Document debe ser guardado por el [NotasGuardar](../WDLang3/1000005110.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the connection
ConnectionID is int
ConnectionID = NotesOpenConnection("Password", "MarsServer", ...
		"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
// Specify the Lotus Notes database to use
DatabaseID is int
DatabaseID = NotesOpenDatabase(ConnectionID, "julia.nsf")
...
// Read the first document
NotesDocumentFirst(DatabaseID)
...
// Attach a file to the current document
FileNum is int
FileNum = NotesAddAttachment(DatabaseID, "MyLotusNotesItem", "C:\MyDocuments\MyFile.TXT")
...
// Modify this file
NotesModifyAttachment(DatabaseID, "MyLotusNotesItem", FileNum, "D:\MyFiles\FileA.TXT")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesModifyAttachment(<Session identifier> , <Lotus Notes item> , <File index> , <New file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<Lotus Notes item>: (Cadena de caracteres)`**

Nombre del elemento de Lotus Notes (Texto enriquecido) al que se adjunta el archivo a modificar.

Para conocer el nombre y el tipo de los elementos de las notas actuales Document, utilice [NotesListItem](../WDLang3/1000005112.md).

**`<File index>: (Integro)`**

Índice del fichero a modificar. Este índice corresponde al rango del archivo en la lista de archivos del elemento de Lotus Notes (por ejemplo, el índice del tercer archivo adjunto será 3).

**Atención**: Los índices de los archivos se modifican cuando se elimina un archivo.

**`<New file>: (Cadena de caracteres)`**

Nombre y ruta completa o relativa del nuevo archivo que se adjuntará al elemento especificado en el documento Notes actual. Este archivo se copiará automáticamente en la base de datos actual de Lotus Notes..

## Observaciones




### Artículo de un Notas Document
<a name="articulo_notas_document_ELTPARAGRAPHE000075"></a>

Cada elemento de un Document de notas puede contener hasta 256 archivos adjuntos.

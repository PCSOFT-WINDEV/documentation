
## ScreenSize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño (en pulgadas) de la diagonal de la pantalla del dispositivo donde se ejecuta la aplicación.

Observaciones : 

- [ScreenType](../WDLang3/1000022210.md) sirve para conocer el tipo de pantalla del dispositivo (Phone, tableta, PC, etc.).

- En modo simulador, el tamaño devuelto no es fiable.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the application is displayed on a large screen (tablet or PC)
IF ScreenSize() >= 7 THEN
	info("The screen is a tablet or a PC")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScreenSize()`
---

**`<Result>: (Real)`**

Tamaño de la diagonal de la pantalla en pulgadas. Por ejemplo : 

- Pantalla de ordenador (diagonal superior a 13 pulgadas).

- Pantalla de visualización (diagonal inferior a 2 pulgadas).

- Pantalla de tabletas (diagonal incluida entre 7 y 13 pulgadas).

- Pantalla Phone (diagonal incluida entre 2 y 7 pulgadas).




## StreamDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la recepción o transmisión de un flujo. La conexión se cierra y se disocia del stream y de los correspondientes controles de Conference.

**Observación**: Para dejar de recibir o emitir un flujo sin disociar este flujo de una control Conferencia, utilice [StreamStop](../WDLang3/3039002.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops receiving the stream and dissociates the stream from the associated control
StreamDisconnect(STREAM_MyStream.Value)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`StreamDisconnect(<Stream identifier>)`
---

**`<Stream identifier>: (Integro)`**

Identificador de la corriente a desconectar. Este identificador se puede encontrar: 

- en la Event "Solicitud de conexión de un nuevo flujo" associated con el flujo.

- utilizando la [Valor](../Proprietes/2510130.md) property en el flujo.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Desconexión de una corriente
<a name="desconexion_una_corriente_ELTPARAGRAPHE000041"></a>

Cuando se desconecta un flujo (por el llamador o el llamado), el "Fin de la conexión" Event se ejecuta automáticamente. 


### Procedure para la detección de corrientes
<a name="procedure_para_deteccion_corrientes_ELTPARAGRAPHE000174"></a>

Un flujo está asociado a la Event "Solicitud de conexión de un nuevo flujo". Este Event se ejecuta cuando se detecta una solicitud de conexión a un flujo entrante.

Este Event se ejecuta en uno de los siguientes casos: 

- si el valor de la propiedad [Detección](../Proprietes/2510031.md) utilizada en el flujo es <u><u><u><u>True</u></u></u></u>.

- si la opción "Detección de corrientes entrantes activas" ha sido marcada en la ventana de la corriente description.




**Observación**: Este Event se ejecuta en un thread específico de WLanguage. Por lo tanto, está sujeto a limitaciones específicas de los hilos de rosca. Para más detalles sobre los hilos, véase [Gestión de hilos con WINDEV](../WDLang1/3077026.md).

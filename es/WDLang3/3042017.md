


## grDestinoEMF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define un meta archivo como destino de la carta (EMF significa Enhanced Meta File). Este archivo se creará durante el siguiente dibujo de la carta ([grDraw](../WDLang3/3042026.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the "C:\Charts\Chart.EMF" file as destination of "MyChart"
grDestinationEMF("MyChart", "C:\Charts\Chart.EMF")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDestinationEMF(<Chart name> , <EMF file name>)`
---

**`<Chart name>: (Cadena de caracteres)`**

Nombre del gráfico a utilizar, definido por [grCreate](../WDLang3/3042005.md).

**`<EMF file name>: (Cadena de caracteres)`**

Nombre del archivo EMF donde se guardará el gráfico. Este archivo se crea durante la llamada a [grDraw](../WDLang3/3042026.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Archivo EMF como destino
<a name="archivo_emf_como_destino_ELTPARAGRAPHE000041"></a>

- Durante la siguiente llamada a [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en el archivo &lt;Nombre del archivo EMF&gt;.

- Este archivo puede ser leído por una herramienta que reconoce el formato EMF.

- El tamaño de este archivo puede ser definido por [TAMAÑO DE GRINdSize](../WDLang3/3042016.md).

- Le recomendamos que previsualice el gráfico antes de guardarlo en un archivo.

- Le recomendamos que utilice **grDestinoEMF** en lugar de [grDestinationWMF](../WDLang3/3042019.md).



<a name="NOTE0_2"></a>


### Diferencias entre grDestinoEMF y grSaveEMF
<a name="diferencias_entre_grdestinoemf_grsaveemf_ELTPARAGRAPHE000063"></a>

- La función **grDestinoEMF** permite definir el destino del próximo dibujo del gráfico. Cuando se utiliza [grDraw](../WDLang3/3042026.md), el gráfico se dibujará en el archivo especificado y se tendrán en cuenta todos los cambios realizados desde la última vez que se dibujó el gráfico.

- [grSaveEMF](../WDLang3/3042049.md) se utiliza para guardar el gráfico en un archivo en formato EMF. La Image guardada corresponde al gráfico durante la última llamada a [grDraw](../WDLang3/3042026.md). Los cambios realizados en el gráfico desde la última vez que se llamó a [grDraw](../WDLang3/3042026.md) serán ignorados.



<a name="NOTE0_3"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000084"></a>

- Esta función no se puede utilizar con los controles gráficos. 

- Esta función no se puede utilizar en la editor de reportes.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Gráfico 3D: Esta función no se puede utilizar: el software DirectX, utilizado para manipular los gráficos 3D, no permite crear archivos en formato EMF.



<a name="NOTE0_4"></a>


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000095"></a>

Para dibujar un gráfico:

- crear el gráfico ([grCreate](../WDLang3/3042005.md)),

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- defina el destino del gráfico ([grDestinationBMP](../WDLang3/3042018.md), [grDestinoControl](../WDLang3/3042020.md), **grDestinoEMF**, [grDestinoWnd](../WDLang3/3042031.md) y [grDestinationWMF](../WDLang3/3042019.md)), 

- definir el destino de la carta con las siguientes funciones: 

	- [grDestinationBMP](../WDLang3/3042018.md), 

	- [grDestinoControl](../WDLang3/3042020.md), 

	- **grDestinoEMF**, 

	- [grDestinoWnd](../WDLang3/3042031.md),

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinationWMF](../WDLang3/3042019.md). 




- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).

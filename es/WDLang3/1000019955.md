


## Funciones de control Mapa
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles de los mapas:



|   |   |
| --- | --- |
| [Área de visualización de mapas](../WDLang3/1000025535.md) | Mueve el mapa que se muestra en el control Mapa para que el área geográfica especificada se pueda ver en su totalidad. |
| [MapAddImage](../WDLang3/1000025532.md) | Añade una nueva Image al mapa que se muestra en una control Mapa. |
| [MapAddItinerario](../WDLang3/1000020383.md) | Añade un itinerario a un control Mapa. |
| [MapAddMarker](../WDLang3/1000019794.md) | Añade una nueva Marker a un mapa mostrado en una control Mapa. |
| [MapAddShape](../WDLang3/1000025509.md) | Añade una nueva forma (Polygon, Line, círculo) al mapa que se muestra en un control Mapa. |
| [MapaItinerarioDistancia](../WDLang3/1000021614.md) | Calcula la distancia de un itinerario en una control Mapa. |
| [MapChangeStyle](../WDLang3/1000025556.md) | Cambia el estilo del mapa que se muestra en el control Mapa. |
| [MapClosePopup](../WDLang3/1000025554.md) | Cierra la popup de un marcador que se muestra en el control Mapa. |
| [MapConfigureGglAPI](../WDLang3/1000021611.md) | Declara parámetros adicionales para el control Map que serán transmitidos a el Google API. |
| [MapDeleteAll](../WDLang3/1000019952.md) | Borra lo siguiente de una control Mapa: <br><br>- todos los marcadores, <br><br>- todas las formas,<br><br>- todas las imágenes.<br><br><br> |
| [MapDeleteImagen](../WDLang3/1000025534.md) | Borra una Image dibujada en el mapa que se muestra en una control Mapa. |
| [MapDeleteItinerario](../WDLang3/1000020384.md) | Borra un itinerario de una control Mapa. |
| [MapDeleteMarker](../WDLang3/1000019795.md) | Elimina una Marker de un mapa mostrado por una control Mapa. |
| [MapDeleteShape](../WDLang3/1000025510.md) | Borra una forma dibujada en el mapa que se muestra en una control Mapa. |
| [MapDisplayPopup](../WDLang3/1000025553.md) | Abre la ventana emergente de una Marker mostrada en una control Mapa. |
| [MapDisplayPosition](../WDLang3/1000019787.md) | Centra el mapa visualizado en una control Mapa en una posición geográfica o en un lugar concreto. |
| [MapEndOfMove](../WDLang3/1000019949.md) | Se detiene siguiendo el movimiento del dispositivo en una control Mapa. |
| [MapExecuteJS](../WDLang3/1000021615.md) | Ejecuta el código nativo de Google Maps API en un control Mapa en JavaScript. |
| [MapFollowMovement](../WDLang3/1000019953.md) | Muestra la ubicación actual del dispositivo en una control Mapa y actualiza la ubicación a medida que se mueve. |
| [MapGetImage](../WDLang3/1000022376.md) | Recupera la Image de un mapa mostrado en una control Mapa. Los marcadores y los itinerarios mostrados en este mapa también se encontrarán en la Image. |
| [MapGetJSObject](../WDLang3/1000021474.md) | Devuelve la instancia de la API de control Mapa. |
| [MapGetPosition](../WDLang3/1000019790.md) | Devuelve la posición geográfica de un punto localizado en el centro del mapa que se está mostrando en un control Map. |
| [MapGetStyle](../WDLang3/1000025557.md) | Devuelve el estilo utilizado por el mapa que se muestra en la control Mapa. |
| [MapInfoPosition](../WDLang3/1000019951.md) | Devuelve las coordenadas (en píxeles) del punto correspondiente a una posición geográfica (latitud y longitud). |
| [MapInfoXY](../WDLang3/1000019950.md) | Devuelve la posición geográfica (latitud y longitud) correspondiente a un punto de una control Mapa. |
| [MapLicenseBing](../WDLang3/1000022006.md) | Indica la clave de licencia del "Bing map" que se utilizará en los controles de Mapas utilizados en el modo de aplicación universal de Windows 10. |
| [MapLicenseGgl](../WDLang3/1000021299.md) | Indica la clave de licencia "Google Maps API" que se utilizará en los controles de mapa (de WINDEV o WEBDEV). |
| [MapLimitArea](../WDLang3/1000025537.md) | Limita el área geográfica que se muestra en el control Mapa. |
| [MapModifyImage](../WDLang3/1000025533.md) | Modifica una Image dibujada en el mapa que se muestra en una control Mapa. |
| [MapModifyMarker](../WDLang3/1000019796.md) | Modifica un marcador mostrado en un control Map568 |
| [MapModifyShape](../WDLang3/1000025511.md) | Modifica una forma dibujada en el mapa que se muestra en una control Mapa. |
| [MapPopupDisplayed](../WDLang3/1000025555.md) | Indica si se muestra la popup de un marcador en un control Mapa. |





Los tipos de variables que manipula el control Mapa o una de sus funciones son los siguientes: 



|   |   |
| --- | --- |
| [MapCircle](../WDLang3/1000025504.md) | El tipo **MapCircle** se utiliza para definir las características de un círculo que se dibujará en el mapa que se muestra en una control Mapa. |
| [MapImage](../WDLang3/1000025523.md) | El tipo **MapImage** se utiliza para definir las características de un Image que se dibujará en el mapa que se muestra en un control Mapa. |
| [MapPolygon](../WDLang3/1000025476.md) | El tipo **MapPolygon** se utiliza para definir las características de un Polygon que se dibujará en el mapa que se muestra en un control Mapa. |
| [MapPolyline](../WDLang3/1000025489.md) | El tipo **MapPolyline** se utiliza para definir las características de una Line compuesta de varios segmentos que se dibujarán en el mapa que se muestra en una control Mapa. |
| [Marcador](../WDLang3/1000019940.md) | El tipo **Marker** permite definir las características de un marcador ubicado en un mapa que se muestra en un control Mapa. |
| [MarkerImage](../WDLang3/1000025796.md) | El tipo **MarkerImage** se utiliza para personalizar la imagen asociada a un cluster Marker o Marker que se muestra en un control Mapa. |

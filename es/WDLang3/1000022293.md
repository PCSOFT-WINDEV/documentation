


## <emailOutlookSession Variable>.Update (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza los mensajes de MS Exchange según el servidor de Email. Se realizan las siguientes operaciones:

- Los mensajes que se encuentran en la bandeja de salida de MS Exchange se envían al servidor de INTERNET Email.

- Los mensajes que esperan en el servidor de INTERNET se transfieren a la bandeja de entrada del cliente de MS Exchange.




La función **<emailOutlookSession Variable>.Update** solo se puede utilizar con el cliente MS Exchange.

**Recordatorio**: Para manipular los Email s, una sesión Email debe ser iniciada por [&lt;emailOutlookSession Variable&gt;.StartSession](../WDLang3/1000022302.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
SessionID is EmailOutlookSession
...
// Retrieve the messages in the in-box of MS Exchange
IF SessionID.Update() = True THEN
	Info("Messages retrieved")
ELSE
	Error()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.Update()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la actualización,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailOutlookSession)`**

Nombre del [emailOutlookSession](../WDLang3/1000018767.md) Variable utilizado para identificar la sesión.

## Observaciones


### Error en la transmisión del Email
<a name="error_transmision_del_email_ELTPARAGRAPHE000218"></a>

Si un mensaje no fue transmitido debido a una mala conexión de INTERNET o a un Address inválido, el mensaje es devuelto al cliente de MS Exchange: el mensaje se muestra en los mensajes no leídos.

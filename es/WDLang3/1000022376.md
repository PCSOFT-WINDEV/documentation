


## MapGetImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la Image de un mapa mostrado en una control Mapa. Los marcadores y los itinerarios mostrados en este mapa también se encontrarán en la Image.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the image displayed in the Map control and assign it to an Image control
MapGetImage(MAP_MyMapControl, CallbackMapGetImage)

INTERNAL PROCEDURE CallbackMapGetImage(MapImage is Image)
	IMG_Image = MapImage
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapGetImage(<Map control> , <WLanguage procedure>)`
---

**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del lenguaje WL Procedure ("callback") llamado cuando el control Mapa Image está disponible. 
Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure name>(<Image>)
```
donde &lt;Imagen&gt; es una [Image](../WDLang1/1000019650.md) Variable que contiene el mapa mostrado en el control.

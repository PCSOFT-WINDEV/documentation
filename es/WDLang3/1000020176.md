


## MobileNetworkConnectionInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la conexión actual a los datos a través de la red móvil.

**Observación**: Esta función no se puede utilizar en el simulador y devolverá 0 en este caso.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Activate the Wi-Fi if the signal intensity is strong enough to access the data 
// via the mobile network
IF Val(MobileNetworkConnectionInfo(networkGSMSignal)) <= 20 _AND_ WiFiStatus() <> wifiEnabled THEN
	IF YesNo("The intensity of the GSM signal is not sufficient. Enable the Wi-Fi?") THEN
		WiFiActivate()
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MobileNetworkConnectionInfo(<Type of information>)`
---

**`<Result>: (Cadena de caracteres o constante entera)`**



- Información solicitada. 

- Si se solicitan varios tipos de información, cada tipo de información está separado por un carácter CR (Carriage Return). 

- El orden en el que se devuelve la información (si se solicita) es el siguiente: 

	1. Tipo de red&gt; &lt;Tipo de red

	2. &lt;Operador&gt;&gt;Operador

	3. &lt;Señal GSM&gt;&gt;.

	4. &lt;Itinerancia&gt;&gt;Itinerancia

	5. &lt;Modo avión&gt;







**`<Type of information>: (Constante de tipo Integer)`**

Información solicitada: 


|   |   |
| --- | --- |
| *networkAirPlaneMode* | Gestión del modo plano.<br><br>**Si se solicita esta información, &lt;Resultado&gt; es set a**:<br><br>- "1" si el dispositivo está en modo avión.<br><br>- "0" en caso contrario.<br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *networkGSMSignal* | Intensidad de la señal GSM entre "0" (sin señal) y "100" (señal óptima).<br><br> |
| *networkOperator* | Nombre del operador de red utilizado.<br><br>Dependiendo del tipo de red o si el modo plano está habilitado, es posible que no se especifique esta información.<br><br>Si no se encuentra la tarjeta SIM, esta información puede corresponder al nombre de un operador si la Phone está "Bloqueada por Sim". |
| *networkRoaming* | Modo de itinerancia de datos.<br><br>**Si se solicita esta información, &lt;Resultado&gt; es set a**:<br><br>- "1" si la conexión actual utiliza la itinerancia de datos<br><br>- "0" en caso contrario.<br><br><br> |
| *networkType* | Tipo de red utilizado para acceder a los datos de la red móvil.<br><br>Modo de itinerancia de datos.<br><br>**Si se solicita esta información, &lt;Resultado&gt; será un entero**:<br><br>- *network2G*: Red 2G<br><br>- *networkEdge*: Red de borde<br><br>- *network3G*: red 3G<br><br>- *network4G*: Red 4G<br><br>- *network5G*: Red 5G<br><br>- *networkUnknown*: Red desconocida (o tarjeta SIM no encontrada). <br><br><br> |

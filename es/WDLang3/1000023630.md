


## Funciones para la gestión de los protocolos OAuth y OpenID (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los protocolos OAuth y OpenID (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;AuthToken Variable&gt;.OpenIDReadIdentity](../WDLang3/1000023629.md) | Recupera la identidad de un usuario contenido en una [AuthToken](../WDLang3/1000022220.md) Variable. |
| [&lt;AuthToken&gt;.RefreshToken](../WDLang3/1000026285.md) | Actualiza un usuario OAuth2 Token cuando expire. |

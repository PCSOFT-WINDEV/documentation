
## Calendar (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Calendar** permite manipular un calendario (o agenda) asociado a una cuenta de usuario o que se encuentra en el dispositivo. Tiene la posibilidad de listar las citas de la Calendar a través de [AppointmentListCalendar](../WDLang3/1000020661.md).

Las características de esta Calendar pueden ser leídas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Fills a list with the calendars available on the device
arrCalendar is array of Calendar = AppointmentListCalendar()
FOR EACH Calendar of arrCalendar
	ListAdd(LIST_List1, Calendar.Name)
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables Calendar
<a name="propiedades_especificas_las_variables_calendar_ELTPARAGRAPHE000041"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a Calendar:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| AccountName | Cadena de caracteres | Nombre de la cuenta de usuario con la que el Calendar es associated. <br><br><br><br>Si esta propiedad corresponde a una cadena vacía (""), la Calendar es associated sin cuenta: se almacena localmente en el dispositivo. |
| AccountType | Cadena de caracteres | Tipo de cuenta. <br><br><br><br><br><br>Si esta propiedad corresponde a una cadena vacía (""), el Calendar es associated sin cuenta y se almacena localmente en el dispositivo. |
| Color | Integro | El color asociado a la Calendar. Este color es el resultado de la siguiente fórmula:<br>Color = 65536 \* Componente Azul + 256 \* Componente Verde + Componente Rojo. |
| denominar | Cadena de caracteres | Nombre de la Calendar. |
| ID | Entero de 8 bytes | Identificador asociado con el Calendar. Esta propiedad se llenará automáticamente cuando se lea el Calendar. No le aconsejamos que guarde este identificador más allá de la vida útil de la aplicación porque este identificador puede ser modificado al sincronizar el Calendar. |


Atención: Las propiedades del tipo **Calendar** son de sólo lectura y se rellenan automáticamente cuando los Calendar son leídos por [AppointmentListCalendar](../WDLang3/1000020661.md).

**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;


## DNSListAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las direcciones IP de associated con un nombre en el servidor DNS.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IPList is string 
IPList = DNSListAddress("testdoc.doc.local", "Andromeda_2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DNSListAddress(<Name> [, <DNS server>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de direcciones IP separadas por caracteres CR (Carriage Return).

**`<Name>: (Cadena Unicode)`**

Nombre completo en el que deben recuperarse las direcciones IP. Este parámetro tiene el siguiente formato: &lt;Nombre &gt; + "." + &lt;Dominio &gt;.. 
Ejemplo: "testdoc.pcsoft.com"

**`<DNS server>: (Cadena de caracteres opcional)`**

Address del servidor DNS para usar. Por Default, se utiliza el actual servidor DNS.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es compatible con los servidores Linux y Windows DSN que no requieren peticiones firmadas.

- Esta función es compatible con los servidores Linux y Windows DSN que requieren peticiones firmadas.




## MapLicenseBing (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la clave de licencia del "Bing map" que se utilizará en los controles de Mapas utilizados en el modo de aplicación universal de Windows 10.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Project initialization
MapLicenseBing("AIzaSyBjsINSH5x39Ks6c0_CoS1yr1Mb3cB3cVo")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapLicenseBing(<key>)`
---

**`<key>: (Cadena de caracteres)`**

Llave suministrada por Bing. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Conseguir una llave Bing
<a name="conseguir_una_llave_bing_ELTPARAGRAPHE000031"></a>

En las aplicaciones de la App Universal de Windows 10, la control Mapa está utilizando el servicio "Bing Map". 

Para utilizar este servicio, debe get una clave de licencia de Bing Map. 

A get una clave de licencia de Bing Map: 

1. Regístrese en [https://www.bingmapsportal.com/](https://www.bingmapsportal.com/). 

2. En el menú "Mi cuenta", seleccione "Crear o ver teclas". 

3. Especifique la información solicitada: 

	- Nombre de la aplicación. 

	- Tipo de clave. 

	- Tipo de aplicación: "Aplicación universal de Windows". 




4. Haga clic en "Crear". 

5. La tecla generada se muestra en el navegador. El enlace "Clave de copy" se utiliza para copy la clave. 




Recordatorio: El uso de un control Mapa está sujeto a términos y condiciones específicos. Para obtener más información, consulte [http://www.microsoft.com/maps/Licensing/licensing.aspx](http://www.microsoft.com/maps/Licensing/licensing.aspx). 
<a name="NOTE0_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000060"></a>

- Esta función permite especificar la licencia de uso para que el proveedor de los mapas de Bing permita a la aplicación acceder a las funciones ampliadas (marcadores, itinerarios, ...). 

- La licencia se aplica a todos los controles del mapa de la aplicación. No se usará para todos los nuevos controles del mapa creados después de la llamada a la función. 

- Se recomienda utilizar **MapLicenseBing** en el código de inicialización del proyecto.

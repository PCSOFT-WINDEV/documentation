


## grRBMaximumValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece el valor máximo en un gráfico de barras radiales. Este valor es set a 100 por Default, por lo que los datos se expresan en porcentaje. Este valor puede modificarse y los porcentajes se recalculan automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grRBMaximumValue(CHART_Chart1, 1212)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grRBMaximumValue(<Chart name> , <Maximum value>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Maximum value>: (Integro)`**

Nuevo valor máximo a aplicar. Este valor debe ser positivo.
Este valor es set a 100 por Default, por lo que los datos se expresan en porcentaje. Cuando se cambia este valor, los datos del gráfico se recalculan automáticamente.


## SaaSAdminListVisit (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de visitas de un sitio SaaS. 

Observación: Esta función se puede ejecutar fuera del sitio SaaS correspondiente. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FOR EACH v OF SaaSAdminListVisit("MySaaS", "", DateSys())
	// Duration of connections of all customers to MySaaS from today
	Trace(v.Duration)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Ver la lista de visitas de un sitio SaaS (utilizando variables de tipo saasxxx)

`<Result> = SaaSAdminListVisit(<SaaS site> , <Client account> [, <Start date> [, <End date>]])`
---

**`<Result>: (Array de saasVisit)`**

Array de una variable de tipo [saasVisit](../WDLang3/1000024560.md). 

**`<SaaS site>: (Variable de tipo saasSite)`**

Nombre de la variable de tipo [saasSite](../WDLang3/1000019068.md) que corresponde al sitio SaaS para el que se filtran las visitas. Si este parámetro corresponde a una cadena vacía (""): 

- el administrador podrá ver la lista de las visitas de todos los sitios SaaS. 

- los gerentes de cuentas podrán ver la lista de todos los sitios SaaS a los que están suscritos. 




**`<Client account>: (Variable de tipo saasClient)`**

Nombre de la variable de tipo [saasClient](../WDLang3/1000019072.md) que corresponde a la cuenta cliente SaaS para la que se filtran las visitas. Si este parámetro corresponde a una cadena vacía, no se tendrá en cuenta en el filtro. 

**`<Start date>: (DateTime opcional)`**

Fecha y hora a partir de la que se filtran las visitas. 

**`<End date>: (DateTime opcional)`**

Fecha y hora hasta la que se filtran las visitas. 


<a name="SYNTAX2"></a>

### Ver la lista de visitas de un sitio SaaS identificado por nombre

`<Result> = SaaSAdminListVisit(<SaaS site> , <Client account> [, <Start date> [, <End date>]])`
---

**`<Result>: (Array de saasVisit)`**

Array de una variable de tipo [saasVisit](../WDLang3/1000024560.md).

**`<SaaS site>: (Cadena de caracteres)`**

Nombre del sitio SaaS para el que se filtran las visitas. Si este parámetro corresponde a una cadena vacía (""): 

- el administrador podrá ver la lista de las visitas de todos los sitios SaaS. 

- los gerentes de cuentas podrán ver la lista de todos los sitios SaaS a los que están suscritos. 




**`<Client account>: (Cadena de caracteres)`**

Nombre de la cuenta cliente SaaS para la que se filtran las visitas. Si este parámetro corresponde a una cadena vacía (""), no se tendrá en cuenta en el filtro. 

**`<Start date>: (DateTime opcional)`**

Fecha y hora a partir de la que se filtran las visitas. 

**`<End date>: (DateTime opcional)`**

Fecha y hora hasta la que se filtran las visitas.

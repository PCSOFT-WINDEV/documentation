


## grClearPict (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra el dibujo del gráfico (en un control Imagen o en una ventana).

**Observación**: El gráfico no se borra. Los datos y parámetros de los gráficos se almacenan en la memoria. Basta con llamar a [grDraw](../WDLang3/3042026.md) para volver a mostrar el gráfico.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Erase the chart named "MyChart"
grClearPict("MyChart")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grClearPict(<Chart name>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a borrar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas o páginas).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el gráfico se dibuja en una ventana independiente, la ventana se cierra.

- La función **grClearPict** no tiene ningún efecto si no se dibuja el gráfico.

- **grClearPict** no tiene efecto en los informes.

- Para eliminar los datos del gráfico, utilice [grDeleteSeries](../WDLang3/3042054.md) o [grDeleteAll](../WDLang3/3042055.md).




## FaxSend( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía un fax en función de la conexión establecida previamente ([FaxConnect](../WDLang3/3037005.md)).

**Observación**: Esta función se puede utilizar en Windows 2000, XP o posterior.. En Windows Vista (y posteriores), la gestión de faxes está disponible con Windows Professional y Windows Integral Edition.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ConnectID is int
ConnectID = FaxConnect()
...
ResSendFax is boolean
ResSendFax = FaxSend(ConnectID, "MyFax", "Message.txt", "0102030405")
...
FaxDisconnect(ConnectID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FaxSend(<Connection identifier> , <Fax name> , <Content file of fax> , <Fax number> [, <Name of cover file> [, <WLanguage procedure>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el fax fue enviado, 

- <u><u><u><u>False</u></u></u></u> si se produce un error (en la mayoría de los casos, este error es causado por un problema de configuración). Para get los detalles del error, utilice [ErrorInfo](../WDLang1/3013008.md).




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [FaxConnect](../WDLang3/3037005.md).

**`<Fax name>: (Cadena de caracteres)`**

Nombre utilizado para identificar el fax saliente. El estado del fax saliente es devuelto por [FaxStatus](../WDLang3/3037001.md).

**`<Content file of fax>: (Cadena de caracteres)`**

Nombre y ruta del archivo que contiene el texto del fax (se puede utilizar un UNC ruta). Todos los tipos de archivos que tienen la opción "Imprimir" en el menú que aparece en el explorador pueden ser enviados por fax.

**Atención**: Dependiendo del tipo de archivo a enviar por fax, los controladores y las aplicaciones necesarias para gestionar este tipo de archivo deben ser instalados en el equipo. Por ejemplo, para enviar un archivo PDF, los controladores necesarios para gestionar el formato PDF deben estar instalados en el equipo. 

**`<Fax number>: (Cadena de caracteres)`**

Número de fax del destinatario. Atención: si se requiere un código de área específico o un código de Country, debe especificarse.

**`<Name of cover file>: (Cadena de caracteres opcional)`**

Nombre y ruta del archivo que contiene la portada del fax. Este archivo debe utilizar el formato de portadas (.cov). Este archivo también puede ser creado por el administrador de fax de Windows. Para obtener más información, consulte [Envío de fax](../WDLang3/3037002.md).

**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

Nombre del WLanguage Procedure ("callback") al que se llama cada vez que cambia el estado del fax saliente. Este Procedure se utiliza para manipular el fax.
Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name>(<Fax name>, <Fax status>)
```

Los parámetros de este Procedure son opcionales.
No hay necesidad de pasarle parámetros a este Procedure. Estos parámetros se rellenan automáticamente para cada fax saliente.

<a name="NOTE0_1"></a>

## Observaciones


### Procedimiento WLanguage
<a name="procedimiento_wlanguage_ELTPARAGRAPHE000076"></a>

Cada vez que el estado del fax saliente cambia, **FaxSend** llama automáticamente al &lt;WLanguage Procedure>. Este Procedure es un Procedure global o local de WINDEV o WEBDEV.

Para crear este Procedure:

1. Crear una Procedure global en el editor de código: en la pestaña "Código", en el grupo "Procedimientos", amplíe "Nuevo" y seleccione "Nuevo global Procedure".

2. Llene la declaración de la Procedure de la siguiente manera: 
	
	```txt
	PROCEDURE <Procedure name> (<Fax name>, <Fax status>)
	```


	- &lt;Nombre de fax&gt; es el nombre del fax utilizado. Este nombre fue definido con **FaxSend**.

	- &lt;Estado del fax&gt; es una de las constantes de estado del fax (ver la función [FaxStatus](../WDLang3/3037001.md)).







**Atención**: Esta Procedure no debe contener ninguna llamada al depurador (ni punto de interrupción, ni palabra clave STOP, ni expresión de autodetención, etc.).

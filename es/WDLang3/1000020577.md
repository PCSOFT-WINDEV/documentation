
## wsRequest (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **wsRequest** define las características avanzadas de un Webservice. Este tipo de Variable se utiliza principalmente para definir los encabezados HTTP utilizados por un servicio web. Los servicios web de EBay y SharePoint requieren este tipo de cabeceras HTTP. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
C is wsRequest
C.HTTPHeader["key"] = "Value"
// Call the Webservice procedure 
// while passing the header and the expected parameters
WebServiceProc(C, param1_WS, param2_WS)
// Displays the request sent
Trace(C.XMLSource)
```

### Propiedades específicas de las variables wsRequest
<a name="propiedades_especificas_las_variables_wsrequest_ELTPARAGRAPHE000037"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a **wsRequest** Variable LE:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| ClientCertificate | Cadena o buffer | Corresponde a: <br><br>- una cadena de caracteres con un acceso ruta al archivo .p12 que contiene el Certificate para adjuntar a la solicitud. El Certificate se cargará automáticamente teniendo en cuenta:<br><br>	- el Certificate en la biblioteca ejecutable (si se ha integrado en la aplicación),<br><br>	- el Certificate en el lugar especificado del disco (si el Certificate no ha sido integrado en la biblioteca ejecutable).<br><br><br><br><br>- un buffer con el Certificate ([fLoadBuffer](../WDLang1/1000019410.md)).<br><br><br><br><br>Si esta propiedad corresponde a una cadena vacía (""), el Certificate por defecto se restablece a "&lt;None&gt;". |
| ClientCertificatePassword | Cadena de caracteres | Contraseña asociada al cliente Certificate (cadena vacía de Default) |
| HTTPHeader | array asociativa de cuerdas de Unicode | Cabeceras HTTP. |
| Timeout | Entero o Duración | Tiempo máximo de respuesta (en milisegundos). Esta propiedad puede corresponder a: <br><br>- un número entero que corresponde al número de milisegundos,<br><br>- una variable de tipo Duration,<br><br>- la duración en un formato legible (por ejemplo, 20 s o 20000 ms). <br><br><br><br><br>Este Property es set a 20 segundos por Default. |
| XMLSource | Binario/Buffer | Fuente XML enviada al servidor.<br><br>**Esta propiedad es de solo lectura**. |

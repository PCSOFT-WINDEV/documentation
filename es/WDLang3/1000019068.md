
## saasSite (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasSite** permite definir las características avanzadas de un sitio WEBDEV SaaS.

Las características de este sitio pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySite is saasSite
MySite.Name = "AVSite"

SaaSAdminAddSite(S)
```

## Observaciones


### Propiedades específicas de las variables saasSite
<a name="propiedades_especificas_las_variables_saassite_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un sitio SaaS:


|   |   |   |
| --- | --- | --- |
| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| denominar | Cadena de caracteres | Nombre del sitio. Este nombre se utilizará en la [Administrador SaaS](../SaaSAdminWeb/1600000.md) y en la programación para identificar el sitio. En la mayoría de los casos, corresponde al nombre del proyecto o al nombre implementación del sitio WEBDEV. |


## &lt;Tabla&gt;.AxisVisible (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la forma en que los ejes se muestran en los gráficos de Line o de columna. El eje puede:

- ...ser Visible,

- ...ser Invisible,

- mostrar sólo la etiqueta.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.AxisVisible(grXCoordinate, grInvisible)
// or for compatibility
// CHART_Chart1.AxisVisible(grXCoordinate, False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.AxisVisible(<Type of axis> , <Visible>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).



**`<Type of axis>: (Constante de tipo Integer)`**

Eje a utilizar:


|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |



**`<Visible>: (booleano)`**

Modo de visualización de ejes: 


|   |   |
| --- | --- |
| *grInvisible* | El eje se convierte en Invisible.<br>Para la compatibilidad retroactiva, también puedes usar &lt;Visible&gt; = False. |
| *grLabelOnly* | Sólo las etiquetas del eje son Visible.<br><br> |
| *grVisible* | El eje se convierte en Visible. <br>Para la compatibilidad retroactiva, también puedes usar &lt;Visible&gt; = True. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el eje de coordenadas Y es Invisible, el eje de coordenadas Y secundario es también Invisible. 

- Los parámetros definidos por **&lt;Tabla&gt;.AxisVisible** se tendrán en cuenta en la siguiente llamada a [&lt;Dibujo](../WDLang3/1000024062.md).




## WidgetRunApp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la aplicación a la que pertenece un widget.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the Android application on the search window by passing as parameter
// the value you want to find
WidgetRunApp(WIN_Find, "value")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Iniciar la aplicación associated Android

`<Result> = WidgetRunApp()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició la aplicación Android que contiene la ventana Widget actual,

- <u><u><u><u>False</u></u></u></u> en caso contrario.





<a name="SYNTAX2"></a>

### Iniciar la aplicación mientras se especifica la ventana a abrir

`<Result> = WidgetRunApp([<Window name> [, <Parameter 1> [... [, <Parameter N>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inició la aplicación Android que contiene la ventana Widget actual,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Window name>: (Cadena de caracteres opcional)`**

Nombre de la ventana de la aplicación que se abrirá al iniciar la aplicación. 

**`<Parameter 1>: (Tipo de valor enviado a la ventana (opcional))`**

Primer parámetro que se pasará al evento "Declaraciones globales" de la ventana a abrir. Este parámetro es pasado por reference y es considerado como un Variable global a la ventana especificada. 
Observación: Sólo los tipos simples pueden ser pasados a la ventana (cadena, entero, booleano, etc.)



**`<Parameter N>: (Tipo de valor enviado a la ventana (opcional))`**

Enésimo parámetro que se pasará al evento "Declaraciones globales" de la ventana que se abrirá. Este parámetro es pasado por reference y es considerado como un Variable global a la ventana especificada.
 Observación: Sólo los tipos simples pueden ser pasados a la ventana (cadena, entero, booleano, etc.)



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **WidgetRunApp** sólo puede utilizarse en modo Widget.

- La aplicación se iniciará en una Process diferente a la que se ejecuta el widget.




## &lt;Gráfico&gt;.HMAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un dato a un gráfico de Heatmap.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.HMAddData(1, 1, 10)
CHART_Chart1.HMAddData(1, 2, 30)
CHART_Chart1.HMAddData(2, 1, 20.5)
CHART_Chart1.HMAddData(2, 2, 5)
```

```wl
// Define the colors
CHART_HMap.HMColor(-50, DarkBlue)
CHART_HMap.HMColor(+50, LightRed)
FOR i = 1 TO 500
	CHART_HMap.HMAddData(Random(-100, 100), Random(-100, 100), -100)
	CHART_HMap.HMAddData(Random(-100, 100), Random(-100, 100), 100)
END
CHART_NoName1.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.HMAddData(<X> , <Y> , <Value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes).


**`<X>: (Integro)`**

Posición de los datos en el eje X.

**`<Y>: (Integro)`**

Posición de los datos en el eje Y.

**`<Value>: (Entero o real)`**

Valor a agregar. Si ya se encuentra un valor en la posición (X, Y) especificada, el valor existente se sustituye por el nuevo valor.

<a name="NOTE0_1"></a>

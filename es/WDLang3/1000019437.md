


## Funciones Wi-Fi
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las redes Wi-Fi:



|   |   |
| --- | --- |
| [WiFiActivate](../WDLang3/1000019350.md) | Habilita o deshabilita Wi-Fi en el dispositivo. |
| [WiFiAddNetwork](../WDLang3/1000019351.md) | Añade una nueva red a la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiConnect](../WDLang3/1000019352.md) | Conecta el dispositivo a la red Wi-Fi especificada. |
| [WiFiConnectionInfo](../WDLang3/1000019355.md) | Devuelve la información solicitada sobre la conexión Wi-Fi actual del dispositivo. |
| [WiFiDeleteNetwork](../WDLang3/1000019357.md) | Elimina una red de la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiDetectAccessPoint](../WDLang3/1000019353.md) | Inicia la detección de los puntos de acceso Wi-Fi a los que se puede acceder desde el dispositivo. |
| [WiFiListNetwork](../WDLang3/1000019356.md) | Devuelve la lista de redes Wi-Fi configuradas en el dispositivo. |
| [WiFiListNetworkCard](../WDLang3/1000020693.md) | Devuelve la lista de las tarjetas de red Wi-Fi disponibles actualmente en el ordenador actual. |
| [WiFiSelectNetworkCard](../WDLang3/1000020694.md) | Selecciona la tarjeta de red Wi-Fi que se utilizará para las próximas llamadas a las funciones WiFi. |
| [WiFiStatus](../WDLang3/1000019354.md) | Devuelve el estado actual de activación de Wi-Fi en el dispositivo o solicita que se le notifique cuando cambie el estado de activación de Wi-Fi. |

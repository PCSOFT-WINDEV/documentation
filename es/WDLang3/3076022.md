


## tapiStopCallDetection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la detección de llamadas especificada que fue iniciada por [tapiEscuchar](../WDLang3/3076029.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Closing the window for call management
// Stop the service for call detection
tapiStopCallDetection("IncomingCall")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`tapiStopCallDetection(<Identifier of call detection>)`
---

**`<Identifier of call detection>: (Cadena de caracteres)`**

Identificador de servicio para la detección de llamadas. Este identificador se especificó cuando la detección de llamadas fue iniciada por [tapiEscuchar](../WDLang3/3076029.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
No se puede llamar a la función **tapiStopCallDetection** en el procedimiento de detección de llamadas (definido con la función [tapiEscuchar](../WDLang3/3076029.md)).

### Configuración necesaria
<a name="configuracion_necesaria_ELTPARAGRAPHE000048"></a>

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.

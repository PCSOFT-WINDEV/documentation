


## &lt;httpRequest Variable&gt;.GetCookie (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las cookies leídas por una llamada a [&lt;httpRequest Variable&gt;.Send](../WDLang3/1000021624.md) para un dominio HTTP. 






<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

### Recuperar las cookies leídas mediante una llamada a HTTPSend

`<Result> = <HTTP request>.GetCookie(<Domain>)`
---

**`<Result>: (Array de las variables de la Cookie)`**

Lista de cookies almacenadas en memoria para el dominio especificado.

**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre de [httpRequest](../WDLang3/1000021158.md) Variable para la que se listarán las cookies.

**`<Domain>: (Cadena de caracteres)`**

Dominio para el que se recuperará la lista de cookies actualmente almacenadas en la memoria.

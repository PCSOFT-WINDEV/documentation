


## <Chart>.StartAngle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece el ángulo inicial de la primera sección en los siguientes tipos de gráfico:

- Pastel, semicírculo y donut.

- Proyección solar.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Gráfico de barras radiales.




Ejemplo de ángulo inicial en una carta semicircular: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grSecteurPartExcentr%E9e%20-%20HC%20N%B0001.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the starting angle of "CHART_MyChart"
CHART_MyChart.StartAngle(45)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart name>.StartAngle(<Angle>)`
---

**`<Chart name>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar en la ventana, página o editor de reportes.


**`<Angle>: (Integro)`**

Ángulo inicial de la primera sección (expresado en grados, en el sentido de las agujas del reloj desde el punto superior). 
Por Default, el ángulo inicial en un gráfico circular, semicircular o de donuts es 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Sólo los siguientes gráficos permiten definir el ángulo inicial de la primera sección: 

	- gráficos circulares (constante *grPie*). 

	- gráficos semicirculares (constante *grSemiCircular*).
			![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este tipo de gráfico no está disponible.

	- gráficos de anillos (constante *grDonut*).
			![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este tipo de gráfico no está disponible.

	- Gráficos de barras radiales (*grRadialBar*constante).
			![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este tipo de gráfico no está disponible.




- Los datos definidos con la función **<Chart>.StartAngle** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

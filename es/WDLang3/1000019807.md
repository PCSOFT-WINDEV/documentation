
## VolumeGet( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el ajuste actual del volumen de una fuente de sonido.

**Observación**: Esta función se puede utilizar en el emulador.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nVolume is int = VolumeGet(volumeRing)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = VolumeGet([<Source> [, <Options>]])`
---

**`<Result>: (Integro)`**



- Valor del volumen para la fuente especificada,

- -1 si se produce un error. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source>: (Constante opcional de tipo Integer)`**

Fuente de sonido a utilizar. 


|   |   |
| --- | --- |
| *volumeAlarm* | Volumen de la alarma. |
| *volumeCall* | Volumen de llamadas. |
| *volumeDefault* | Volumen de la fuente de Default. Esta fuente depende de la configuración de Phone. |
| *volumeMusic* | Volumen utilizado al reproducir un archivo de audio o vídeo. |
| *volumeNotification* | Volumen del sonido emitido durante un nuevo Notification. |
| *volumeRing*<br>(Valor predeterminado) | Volumen del anillo de Phone. |
| *volumeSystem* | Volumen de los sonidos del sistema. |



**`<Options>: (Constante opcional de tipo Integer)`**

Tipo de volumen a recuperar: 


|   |   |
| --- | --- |
| *volumeCurrent*<br>(Valor predeterminado) | Volumen actual de la fuente especificada. Este volumen se incluye entre 0 y el valor máximo para el volumen de la fuente.. |
| *volumeCurrentPercentage* | Volumen actual de la fuente especificada, expresado en porcentaje de su valor máximo. Este volumen se incluye entre 0 y 100. |
| *volumeMax* | Volumen máximo de la fuente especificada. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para modificar el ajuste del volumen de una fuente de sonido, utilice [VolumeModify](../WDLang3/1000020169.md).

- Para modificar la fuente de sonido asociada al botón físico utilizado para set el volumen del dispositivo, llame a [VolumePhysicalButton](../WDLang3/1000020168.md).

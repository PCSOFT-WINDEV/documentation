


## grSuperficieBorrarMarkCaption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los títulos de las marcas que se encuentran en un gráfico de Superficie. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grSurfaceDeleteMarkCaption(CHART_MyChart, grXCoordinate, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSurfaceDeleteMarkCaption(<Chart name> [, <Mark axis> [, <Value>]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Mark axis>: (Constante de tipo Integer)`**

Tipo de eje del que se desea borrar las marcas:



|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |
| *grZCoordinate* | Eje Z. |


Si no se especifica este parámetro, se borran las marcas de todos los ejes.. 

**`<Value>: (Real)`**

Valor de la marca para la que debe suprimirse la leyenda. 

Si no se especifica este parámetro, se borrarán todos los títulos de las marcas..

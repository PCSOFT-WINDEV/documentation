


## grDestinationControl (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define un control Imagen como destino de un gráfico.

**Observaciones**:

- Esta función no se puede utilizar con los controles gráficos. 

- Esta función no se puede utilizar en la editor de reportes.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define the "IMG_ChartImage" control as destination of "MyChart"
grDestinationControl("MyChart", IMG_ChartImage)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDestinationControl(<Chart name> , <Image control>)`
---

**`<Chart name>: (Cadena de caracteres)`**

Nombre del gráfico a utilizar, definido por [grCreate](../WDLang3/3042005.md).  

**`<Image control>: (Nombre del control)`**

Nombre de la control Imagen donde se dibujará el gráfico.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Crear y dibujar un gráfico mediante programación
<a name="crear_dibujar_grafico_mediante_programacion_ELTPARAGRAPHE000086"></a>

Para dibujar un gráfico:

- crear el gráfico ([grCreate](../WDLang3/3042005.md)),

- añadir datos ([grAddData](../WDLang3/3042023.md), [grStockAddData](../WDLang3/3042009.md) y [grScatterAddDataXY](../WDLang3/3042032.md)),

- definir el destino de la carta con las siguientes funciones: 

	- [grDestinoBMP](../WDLang3/3042018.md), 

	- **grDestinationControl**, 

	- [grDestinoEMF](../WDLang3/3042017.md), 

	- [grDestinoWnd](../WDLang3/3042031.md),

	- [grDestinationImage](../WDLang3/1000020952.md),

	- [grDestinoWMF](../WDLang3/3042019.md). 




- dibujar el gráfico ([grDraw](../WDLang3/3042026.md)).


La próxima vez que se llame a [grDraw](../WDLang3/3042026.md), el gráfico se dibujará dentro de &lt;control Imagen&gt;. Otra imagen puede ser associated con este control: el dibujo de la carta se apilará frente a la Image.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Cambio de destino
<a name="cambio_destino_ELTPARAGRAPHE000141"></a>

Si el gráfico se mostraba previamente en una ventana independiente, esta ventana independiente se cerrará durante la siguiente llamada a [grDraw](../WDLang3/3042026.md).


## upnpService (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **upnpService** se utiliza para definir las características avanzadas de un servicio proporcionado por el dispositivo UPnP. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Este tipo de Variable es utilizado por el [upnpDevice](../WDLang3/1000020903.md) Variable s. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).




<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo upnpService
<a name="propiedades_especificas_descripcion_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para manipular, un servicio proporcionado por un dispositivo UPnP:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ControlURL | Cadena de caracteres | URL para enviar los comandos al servicio. |
| escribir a máquina | Cadena de caracteres | Tipo de servicio. |
| Identificador | Cadena de caracteres | Identificador del servicio. |
| URL | Cadena de caracteres | URL que contiene el servicio description. |

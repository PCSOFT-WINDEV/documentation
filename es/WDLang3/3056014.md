
## NetMachineName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del ordenador local o el nombre de un ordenador cuya IP Address se conoce (resolución DNS inversa).  




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the name of local computer
Info(NetMachineName())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetMachineName([<IP address>])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la computadora actual. 


**`<IP address>: (Cadena de caracteres opcional)`**

IP Address (en formato XXX.XXX.XXX.XXX) de el equipo cuyo nombre se solicita. Si no se especifica este parámetro, se devuelve el nombre del ordenador actual.. Este Address debe ser un Address válido.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### IP válido Address
<a name="valido_address_ELTPARAGRAPHE000050"></a>

Para comprobar si una IP Address es válida, utilice [Ping](../WDLang1/3014013.md).


## Funciones de WebSocket (sintaxis de prefijo)

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones pueden utilizarse en un proyecto de "servidor WebSocket" (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;Variable websocketClient&gt;.Execute](../WDLang3/1000026113.md) | Ejecuta una Procedure global en el contexto de un cliente conectado al servidor WebSocket. |
| [&lt;websocketClient Variable&gt;.Desconectar](../WDLang3/1000026067.md) | Desconecta un cliente de un servidor WebSocket. |
| [&lt;websocketClient Variable&gt;.Send](../WDLang3/1000026068.md) | Envía un mensaje del servidor WebSocket al cliente. |

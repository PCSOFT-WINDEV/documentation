


## &lt;btleDevice Variable&gt;.GetAllServices (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera todos los servicios de un dispositivo Bluetooth de baja energía correspondiente a un identificador (UUID).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDevice is btleDevice
...
arrService is array of btleService
arrService = MyDevice.GetAllServices("ACC54-DAD4D5-DADD")
FOR EACH ELEMENT service of arrService 
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Device>.GetAllServices(<Identifier>)`
---

**`<Result>: (Array de las variables de la btleService)`**

Array de las variables de [btleService](../WDLang3/1000022057.md) correspondientes a los servicios encontrados. Si ningún servicio corresponde al identificador especificado, el array estará vacío.

**`<Device>: (btleDevice Variable)`**

Nombre del [btleDevice](../WDLang3/1000022054.md) Variable que corresponde al dispositivo Bluetooth utilizado.

**`<Identifier>: (Cadena de caracteres)`**

Identificador (UUID) de los servicios a recuperar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Varios servicios del mismo dispositivo Bluetooth pueden corresponder al mismo identificador (UUID). Para recuperar el primer servicio correspondiente a un identificador, utilice [&lt;btleDevice Variable&gt;.GetAService](../WDLang3/1000022222.md).

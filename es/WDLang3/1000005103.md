
## NotesItemDimension( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la dimensión de un artículo para el actual Document en Lotus Notes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
...
// Read the first email
EmailReadFirst(SessionID)
// Dimension of the Lotus Notes "INetSendTo" item
ResDimension is int
ResDimension = NotesItemDimension(SessionID, "INetSendTo")
Info("This email was sent to " + ResDimension + " persons")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesItemDimension(<Session identifier> , <Lotus Notes item>)`
---

**`<Result>: (Integro)`**



- Dimensión del elemento Lotus Notes especificado,

- 0 si el elemento de Lotus Notes especificado no existe.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<Lotus Notes item>: (Cadena de caracteres)`**

Nombre del elemento de Lotus Notes cuya dimensión se solicita.

Para conocer el nombre y el tipo de los elementos de las notas actuales Document, utilice [NotesListItem](../WDLang3/1000005112.md).

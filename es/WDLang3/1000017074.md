
## SNMPLoadMIB( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga un archivo MIB en la memoria y lo analiza. Este archivo contiene las definiciones de OID utilizadas para convertir los identificadores numéricos en cadenas.. También contiene un texto que describe los OID.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MIBLoaded is boolean
PathMIBs is string = CompleteDir(fExeDir()) + "mibs\"
Mib is string
Mib = "RFC1382-MIB.mib" // MIB of X25 protocol
MIBLoaded = SNMPLoadMIB(PathMIBs + Mib)
IF MIBLoaded = False THEN 
	Error("MIB error:" + ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SNMPLoadMIB(<Name of MIB file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cargo y analizó el archivo correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Name of MIB file>: (Cadena de caracteres)`**

ruta completa del archivo MIB. Esta ruta puede ser una ruta completa o una ruta relativa al directorio actual..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En Windows, algunos archivos MIB se suministran en el directorio system32.

- Se pueden cargar varios archivos MIB. Las búsquedas se realizarán en todos los archivos MIB cargados.

- Si un archivo MIB accede a otro archivo MIB en su código, el segundo archivo MIB se carga automáticamente (si el nombre especificado en el archivo MIB es idéntico al nombre físico del archivo MIB).

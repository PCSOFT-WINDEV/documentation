


## NetListIPAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de direcciones IP (Protocolo INTERNET) de un ordenador. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the IP addresses of local computer
Info(NetListIPAddress())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NetListIPAddress([<Computer name>] [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de direcciones IP separadas por caracteres CR (Carriage Return). 

**`<Computer name>: (Cadena de caracteres opcional)`**

Nombre del ordenador para el que se enumerarán las direcciones IP. Si no se especifica este parámetro, se devuelve la lista de direcciones IP encontradas en el ordenador local..


**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Tipo de Address a recuperar: 


|   |   |
| --- | --- |
| *IPv4* | Lista de direcciones de IPv4. |
| *IPv6* | Lista de direcciones de IPv6. |


Si no se especifica este parámetro, se devuelve la lista de direcciones IPv4 e IPv6 (combinación de las constantes *IPv6* + *IPv4*).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000063"></a>

Los siguientes códigos son equivalentes: 

```wl
Trace(NetListIPAddress(ipv4))
```


```wl
n is int = 0
WHILE NetIPAddress(n) <> ""
	Trace(NetIPAddress(n))
	n = n + 1
END
```

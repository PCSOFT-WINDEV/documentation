
## WiFiAddNetwork( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva red a la lista de redes Wi-Fi configuradas en el dispositivo.. A continuación, se utiliza [WiFiConnect](../WDLang3/1000019352.md) para conectar el dispositivo a la red Wi-Fi añadida. 

**Observación**: El acceso Wi-Fi no está disponible en el emulador de Android.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Identifier is string 
Identifier = WiFiAddNetwork("SSID", "BSSID", wifiSecurityWPA, "pwd", True)
IF Identifier <> " "  THEN
	WiFiConnect(Identifier)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WiFiAddNetwork(<SSID> , <BSSID> , <Security> [, <Key> [, <Permanent>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Identificador de red de la nueva red. 

- Cadena vacía si no se ha creado el grafo. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**Observación**: Si la red añadida corresponde a una red que ya está configurada en el dispositivo, &lt;Result&gt;corresponde al identificador de la red existente.

**`<SSID>: (Cadena de caracteres)`**

SSID (Service set Identifier) de la red Wi-Fi.

**`<BSSID>: (Cadena de caracteres)`**

BSSID (Basic Service set Identifier) del punto de acceso associated con la red.

**`<Security>: (Constante de tipo Character String)`**

Tipo de seguridad utilizado por la red Wi-Fi: 


|   |   |
| --- | --- |
| *wifiSecurityIEEE8021X* | Seguridad que utiliza el estándar IEEE 802.1x. |
| *wifiSecurityNone* | No se utiliza ningún tipo de seguridad. |
| *wifiSecurityWEP* | Seguridad basada en una clave WEP (Wired Equivalent Privacy) |
| *wifiSecurityWPA2* | Seguridad que utiliza el mecanismo WPA2 (Wi-Fi Protected Access). |
| *wifiSecurityWPA* | Seguridad que utiliza el mecanismo WPA (Wi-Fi Protected Access). |
| *wifiSecurityWPA_EAP* | Seguridad que utiliza el mecanismo WPA (Wi-Fi Protected Access) y el mecanismo EAP (Extensible Authentication Protocol). |



**`<Key>: (Cadena de caracteres opcional)`**

Clave utilizada para conectarse a la red segura. La clave es obligatoria si el tipo de protocolo de seguridad especificado es WPA o WPA2.

**`<Permanent>: (Booleano opcional)`**

Indica si la adición de la red es permanente:

- <u><u><u><u>True</u></u></u></u>: la red añadida se almacenará permanentemente en el dispositivo. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): la red añadida no se mantendrá cuando se reinicie el dispositivo.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000097"></a>

- Si la red añadida se almacena permanentemente en el dispositivo (&lt;Permanente&gt; set a <u><u><u><u>True</u></u></u></u>), los identificadores asignados a cada una de las redes pueden modificarse después de ejecutar **WiFiAddNetwork**. Utilice [WiFiListNetwork](../WDLang3/1000019356.md) para conocer los nuevos identificadores.

- Para eliminar una red de la lista de redes configuradas en el dispositivo, utilice [WiFiDeleteNetwork](../WDLang3/1000019357.md).



<a name="NOTE0_2"></a>


### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000117"></a>

La llamada a esta función modifica los permisos requeridos por la aplicación.

Permisos necesarios :  

- ACCESS_WIFI_STATE: permite a las aplicaciones acceder a la información relativa a las redes Wi-Fi. 

- CAMBIAR_WIFI_STATE: permite a las aplicaciones modificar el estado de la conectividad Wi-Fi.

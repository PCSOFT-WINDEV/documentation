
## NetInfoRemoteAccess( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el estado de una conexión, así como las estadísticas sobre la misma.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CurrentConnection is boolean
ConnectionDuration is duration
CurrentConnection = NetInfoRemoteAccess("MyAccesss", niConnected)
IF CurrentConnection = True THEN
	ConnectionDuration = NetInfoRemoteAccess("MyAccess", niConnectionDuration)
	Info("The MyAccess connection is opened since " + ConnectionDuration)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetInfoRemoteAccess(<Access name> , <Type of information>)`
---

**`<Result>: (Entero o booleano)`**

Información solicitada.

**`<Access name>: (Cadena de caracteres)`**

Nombre de la conexión "Remote network access" definida en Windows.

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información sobre la conexión:


|   |   |
| --- | --- |
| *niConnected* | &lt;El resultado es booleano. <br>Se usa para saber si la conexión está habilitada. <br>Si la conexión está activa, el valor de &lt;Resultado&gt; es *True*. |
| *niConnectionDuration* | &lt;El resultado es una duración. <br>Duración desde que se estableció la conexión. |
| *niNbByteReceived* | &lt;Resultado&gt; es un número entero. <br>Número de bytes recibidos desde que se estableció la conexión. |
| *niNbByteTransmitted* | &lt;Resultado&gt; es un número entero. <br>Número de bytes transmitidos desde que se estableció la conexión. |

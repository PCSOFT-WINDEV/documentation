
## AppointmentCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana de creación de la aplicación nativa Appointment para la gestión de los Appointment que se encuentran en el dispositivo Android.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
apt is Appointment
IF AppointmentCreate(apt) THEN
	Info(StringBuild("The %1 appointment was added to the %2 calendar", ...
			apt.Name, apt.Calendar.Name))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppointmentCreate(<Appointment>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó una cita,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Appointment>: (Appointment Variable)`**

Nombre de la variable [Appointment](../WDLang1/1000019244.md) que se llenará con los datos correspondientes a la Appointment creada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000047"></a>

- Para crear una nueva Appointment mediante programación, sin abrir ninguna ventana, utilice [AppointmentAdd](../WDLang3/1000011001.md).

- **AppointmentCreate** es una función de bloqueo hasta que el usuario valide o cancele la ventana de creación de appointment.

- La función **AppointmentCreate** se puede utilizar en el emulador pero no en el simulador.

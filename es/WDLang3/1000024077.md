


## &lt;Agregación de Columnas (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el algoritmo para agregar datos en un gráfico de columna agrupado o apilado..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code of Radio Button control used to select the aggregation type
// Store the selected type of aggregation
let nAggregation = SWITCH (RADIO_AggregationType, grhaNone, grhaMax, grhaMin, grhaMean, grhaSum)

// Apply the aggregation
CHART_Chart.ColumnAggregation(nAggregation)

// Redraw the example
CHART_Chart.Draw()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.ColumnAggregation(<Algorithm>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Algorithm>: (Constante de tipo Integer)`**

Algoritmo de agregación a utilizar: 


|   |   |
| --- | --- |
| *grhaMax* | Agregación según el máximo de datos. |
| *grhaMean* | Agregación según la media de los datos. |
| *grhaMin* | Agregación según el mínimo de datos. |
| *grhaNone* | Ninguna agregación. |
| *grhaSum* | Agregación según la suma de datos. |

<a name="NOTE0_1"></a>

## Observaciones


### ¿Cuándo debe utilizarse la agregación?
<a name="¿cuando_debe_utilizarse_agregacion_ELTPARAGRAPHE000203"></a>

Cuando se muestra un gráfico de columna, las barras pueden volverse demasiado pequeñas para ser mostradas. Este es el caso de los gráficos que presentan una escala de tiempo. En este caso, se hace necesario mostrar una barra cuyo tamaño mínimo es el tamaño de una marca y agregar los datos de todas las barras potenciales que se encuentran en esta marca.

### Agregación y tipo de gráficos
<a name="agregacion_tipo_graficos_ELTPARAGRAPHE000228"></a>

- Para los gráficos sin escala de tiempo, la agregación no está habilitada por Default.

- Para los gráficos con escala de tiempo, la agregación de Default es la suma de las columnas.

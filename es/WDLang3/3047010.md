
## J2EEError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el error del último J2EE Procedure que falló.

**Observación**: Esta Procedure fue dirigida por [J2EERun](../WDLang3/3047007.md) o [J2EERunXML](../WDLang3/3047008.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the error message of the last J2EE procedure run
ResErrorMessage = J2EEError(J2EEErrMessage)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = J2EEError(<Type of error>)`
---

**`<Result>: (Cadena de caracteres)`**



- Error del último J2EE Procedure que falló.

- Una cadena vacía (""") si:

	- el servidor no ha devuelto ningún error,

	- no se ha establecido ninguna conexión con el servidor.







**`<Type of error>: (Constante)`**

Tipo de error a devolver:


|   |   |
| --- | --- |
| *J2EEErrActor* | URL del servidor que activó la Error( útil si la Procedure es ejecutada por varios servidores). |
| *J2EEErrCode* | Error de la última corrida de J2EE Procedure:<br><br>- "VersionMismatch": La versión J2EE utilizada en el ordenador actual difiere de la utilizada en el servidor.<br><br>- "Debe ser de las afueras.": El servidor no entiende el parámetro especificado en el encabezado (solo si el parámetro &lt;Ejecución del procedimiento&gt; del encabezado se inicializa en <u><u><u><u>True</u></u></u></u>).<br><br>- "Cliente": Configuración inválida en relación con la ejecución de la Procedure.<br><br>- "Server": El servidor no ejecutó el Procedure.<br><br><br>**Observación**: Esta cadena de caracteres se puede rellenar con información adicional. Se utiliza el siguiente formato: "&lt;Error&gt;.&lt;Información&gt;". Por ejemplo: "Autenticación de cliente" si el error se debe a la autenticación del cliente. |
| *J2EEErrDetails* | Detalles del error. Este mensaje está en formato XML. |
| *J2EEErrMessage* | Mensaje de error devuelto por el servidor en la última ejecución de J2EE Procedure. |

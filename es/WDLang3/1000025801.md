


## &lt;Mapa&gt;.AddShape (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva forma (Polygon, Line, círculo) al mapa que se muestra en un control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a polygon to the Map control
Polygon is MapPolygon
FOR EACH PointPosition OF arrPositions
	Polygon.Point.Add(PointPosition)
END
Polygon.LineColor = LightRed
Polygon.ActionClick = ClickProc
MAP_MyMap.AddShape(Polygon)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.AddShape(<Shape>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se pudiera añadir la forma a la control Mapa,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa a utilizar.

**`<Shape>: (MapCircle, MapPolygon o MapPolyline Variable)`**

La forma que se dibujará en el mapa que se muestra en la control Mapa. Esta forma puede describirse utilizando uno de los siguientes tipos de variables: 

- una variable de tipo [MapPolygon](../WDLang3/1000025476.md), 

- una variable de tipo [MapPolyline](../WDLang3/1000025489.md), 

- una variable de tipo [MapCircle](../WDLang3/1000025504.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Una vez que se ha añadido la forma, se puede usar su nombre (**Name** Property del asociado Variable). 

- Si la forma ya ha sido añadida a la control Mapa o si es inválida (por ejemplo.: Polygon con un número insuficiente de puntos), se mostrará un error fatal.

- Para borrar una forma dibujada en una control Mapa, utilice [&lt;Mapa&gt;.BorrarForma](../WDLang3/1000025820.md). 

- Para modificar una forma dibujada en una control Mapa, utilice [&lt;Mapa&gt;.ModifyShape](../WDLang3/1000025805.md).


## TaskLast( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee la última tarea de una lista de tareas de Lotus Notes o Outlook. Esta tarea se convierte en la tarea actual.

Cuando se lee una tarea, los datos de esta tarea se actualizan en el [Estructura de la tarea](../WDLang3/1000012010.md). Para recuperar los datos de la tarea actual, basta con recuperar los valores de la [Estructura de la tarea](../WDLang3/1000012010.md).

**Atención**: Esta función **no se puede utilizar** para leer la última tarea encontrada en la lista de tareas de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", "email/julia.nsf",...
					"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Read the last task
TaskLast(SessionID)
// Add the tasks found into a list
WHILE mTask.Out = False
	ListAdd(LIST_TaskList, mTask.Subject + TAB + mTask.Category)
	TaskPrevious(SessionID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TaskLast(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se leyó la última tarea,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).


## GroupModify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el grupo actual de contactos en el libro de Address de una mensajería de Lotus Notes o de Outlook.

Cuando se lee un grupo de contactos, los datos de este grupo se actualizan en [Estructura del grupo](../WDLang3/1000008120.md). Para modificar el grupo actual, basta con modificar los valores de la [Estructura del grupo](../WDLang3/1000008120.md) y validar las modificaciones con la función  **GroupModify**.

**Atención**: Esta función ** no puede utilizarse** para modificar el grupo actual de contactos en el libro Address de una mensajería de Outlook Express.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
	"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Read the first group of contacts
GroupFirst(SessionID)
// Add a new contact into this group
mGroup.NbContact = 4
mGroup.Contact = "Arthur"
GroupModify(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = GroupModify(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por una de las siguientes funciones: [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Versión de Outlook utilizada
<a name="version_outlook_utilizada_ELTPARAGRAPHE000068"></a>

Los grupos de contactos no son compatibles con Outlook 97. Por lo tanto, ningún grupo de contactos puede ser modificado en esta versión de Outlook.

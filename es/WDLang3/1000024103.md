


## &lt;Gráfico&gt;.Scatter3DDeleteAltitudeColor ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra los colores de altitud especificados por [&lt;Scatter3DAltitudeColor](../WDLang3/1000024098.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_My3DScatter.Scatter3DDeleteAltitudeColor(1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.Scatter3DDeleteAltitudeColor(<Series number> [, <Altitude>])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en el editor de ventanas).

**`<Series number>: (Integro)`**

Número de la serie a utilizar.

**`<Altitude>: (Real opcional)`**

La altitud de Color que será eliminada. Si no se especifica este parámetro, se borran todos los colores de altitud.: el gráfico está usando la serie Color.



<a name="XComponent"></a>

## Componente:
wd280grf.dll

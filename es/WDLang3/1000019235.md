
## tapiDialerCall( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la aplicación de telefonía por defecto (marcador) que se encuentra en la Phone y marca el número especificado.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
tapiDialerCall("0612345678")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = tapiDialerCall(<Phone number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si la aplicación de telefonía se ha iniciado correctamente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puedes get los detalles del mismo con [ErrorInfo](../WDLang1/3013008.md).




**`<Phone number>: (Cadena de caracteres)`**

Número de Phone a marcar. 
Si este parámetro corresponde a una cadena vacía (""), la función devolverá <u><u><u><u>False</u></u></u></u>.
Atención : el número no debe contener ningún carácter de espacio.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000084"></a>

- No es posible llamar automáticamente a los números de emergencia

- No se puede marcar ningún código USSD.

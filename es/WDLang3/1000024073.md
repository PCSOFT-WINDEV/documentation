


## &lt;Tabla&gt;.SeriesLabel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la etiqueta de una serie de datos en un gráfico.

Ejemplo de etiqueta de serie:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 Para mostrar la etiqueta, utilice [&lt;Leyenda](../WDLang3/1000024089.md).

**Recordatorio**: Por Default, un gráfico no tiene etiqueta. Para más detalles sobre la gestión de leyendas, etiquetas, ... consulte [Gestión de leyendas en los gráficos](../WDChamp/3042069.md). 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the "Sales" label of second data series in CHART_MyChart
CHART_MyChart.SeriesLabel(2, "Sales")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.SeriesLabel(<Series number> , <Label>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie para la que se inicializará una etiqueta.

**`<Label>: (Cadena de caracteres)`**

Leyenda de la etiqueta a inicializar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Etiquetas de la serie
<a name="etiquetas_serie_ELTPARAGRAPHE000265"></a>Las etiquetas de la serie set corresponden a la leyenda de la carta.
Las etiquetas de la serie <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 no pueden utilizarse en los gráficos circulares.

Las etiquetas de serie se definen con **&lt;Tabla&gt;.SeriesLabel**. 

Los datos definidos con la función **&lt;Tabla&gt;.SeriesLabel** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

La fuente de la leyenda puede cambiarse con [&lt;LegendFont](../WDLang3/1000024118.md).  

[&lt;Leyenda](../WDLang3/1000024089.md) indica la presencia y la posición de la leyenda (es decir, todas las etiquetas de las series).

**Observación**: Para definir las etiquetas de las series en los gráficos de "dispersión", le recomendamos que utilice [&lt;Tabla&gt;.ScatterSeriesLabel](../WDLang3/1000024107.md).
<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000311"></a>

La función **&lt;Tabla&gt;.SeriesLabel** es equivalente a la función [Leyenda](../Proprietes/2510143.md).

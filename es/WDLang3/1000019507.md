
## ContactEdit( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el formulario de una Contact en modo de edición en la aplicación nativa de Contact s del dispositivo (Android, iPhone, iPad, Universal Windows).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the current contact's form in edit mode
// Global variables
gCurrentContact is Contact

... 

// - Click on BTN_Modify
// Starts editing the contact
ContactEdit(gCurrentContact)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Abrir un formulario Contact en modo de edición

`<Result> = ContactEdit(<Contact>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el contacto,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

- <u><u><u><u>True</u></u></u></u> si se muestra la interfaz de usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Contact>: (Contacte con Variable)`**

Nombre de la [Contact](../WDLang3/1000020254.md) Variable correspondiente al Contact cuyo formulario se abrirá. Esta Contact debe existir en la base de datos de Contact s que se encuentra en el dispositivo.

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000086"></a>

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **ContactEdit** es una función no bloqueante. 

- Para actualizar la [Contact](../WDLang3/1000020254.md) Variable con los datos de la Contact que se editó anteriormente, utilice [ContactRead](../WDLang3/1000006112.md).

- **ContactEdit** puede utilizarse en el emulador. No se puede utilizar en el simulador.

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000283"></a>

Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.

Característica necesaria: Contactos

Esta característica permite a las aplicaciones acceder a los contactos.

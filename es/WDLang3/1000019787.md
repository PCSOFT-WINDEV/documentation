


## MapDisplayPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Centra el mapa visualizado en una control Mapa en una posición geográfica o en un lugar específico. Una animación se realiza automáticamente.

Al final de la animación, se ejecutará la Event "Siempre que cambie de posición" de la control Mapa. Este Event no se ejecutará si el posicionamiento en el mapa es cancelado por el usuario antes del final de la animación.




<a name="sample_code"></a>

## Ejemplo

<a name="Example1"></a>

```wl
// Centers the map displayed by the "MAP_Position" control on a geographical position 
MyPosition is geoPosition
MapDisplayPosition(MAP_Position, MyPosition)

// Equivalent syntax (without animation): MAP_Position = MyPosition
```


<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Centers the map displayed by the "MAP_Position" control on a city identified by its name
MapDisplayPosition(MAP_Position, "Montpellier, France")

// Centers the map displayed by the "MAP_Position" control on a monument identified by its name 
MapDisplayPosition(MAP_Position, "Eiffel Tower")
```

## Sintaxis
<a name="SYNTAX1"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Mostrando una ubicación geográfica

`<Result> = MapDisplayPosition(<Map control> , <Position>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se posicionó el mapa en la posición especificada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) &lt;Resultado&gt; siempre se establece en <u><u><u><u>True</u></u></u></u>, incluso si se produce un error. No hay forma de saber si el posicionamiento se ha realizado con éxito.. 

**`<Map control>: (Nombre del control)`**

Nombre del control Mapa a manipular. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Position>: (geoPosition Variable o cadena de caracteres)`**

description del lugar en el que se centrará el mapa. Este description puede corresponder a: 

- el nombre de la [geoPosition](../WDLang3/1000019191.md) Variable que describe la posición geográfica en la que se centrará el mapa.

- una cadena de caracteres que contiene una Address, una ciudad, el nombre de un monumento, ...
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El uso de una cadena de caracteres puede implicar una posición inconsistente a través del servicio mapping de Bing Map. 

- una [Address](../WDLang3/1000019944.md) Variable.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Mostrando una ubicación geográfica con el cheque Procedure

`MapDisplayPosition(<Map control> , <Position> [, <WLanguage procedure>])`
---

**`<Map control>: (Nombre del control)`**

Nombre del control Mapa a manipular. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Position>: (geoPosition Variable o cadena de caracteres)`**

description del lugar en el que se centrará el mapa. Este description puede corresponder a: 

- el nombre de la [geoPosition](../WDLang3/1000019191.md) Variable que describe la posición geográfica en la que se centrará el mapa.

- una cadena de caracteres que contiene una Address, una ciudad, el nombre de un monumento, ...
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El uso de una cadena de caracteres puede implicar una posición inconsistente a través del servicio mapping de Bing Map. 

- an [Address](../WDLang3/1000019944.md) Variable




**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

Nombre del WLanguage Procedure ("callback" Procedure) llamado una vez que se muestra la nueva posición geográfica. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name> (<Change performed>)
```

donde &lt;Change performed&gt; es un booleano: 

- <u><u><u><u>True</u></u></u></u> si se modificó la posición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **MapDisplayPosition** utiliza el servicio de geolocalización de Google (o Bing Map para la aplicación universal de Windows 10). Las posiciones definidas desde la description de un lugar dependen de la cantidad y de la precisión de la información suministrada en esta description. Si un mismo description corresponde a varios lugares, el servicio de geolocalización devolverá las correspondencias más adecuadas.

- El nivel de zoom del mapa no se modifica al cambiar de posición.

- **Para centrar el mapa en una posición específica sin realizar la animación**, tienes la capacidad de asignar una variable [geoPosition](../WDLang3/1000019191.md) a la [Valor](../Proprietes/2510130.md) Property de un control Mapa.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **Para saber la posición que se muestra actualmente en el centro de la control Mapa**, use [MapGetPosition](../WDLang3/1000019790.md). 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) La control Mapa está usando el servicio de Bing Map.
	 **Atención**: El uso de un control Mapa está sujeto a términos y condiciones específicos. Se debe utilizar una clave Bing (función [MapLicenseBing](../WDLang3/1000022006.md)). Verifique la licencia de Bing antes de implementar sus aplicaciones.

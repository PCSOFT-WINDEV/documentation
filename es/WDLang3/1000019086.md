


## <zipArchive Variable>Cerrar (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un archivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyArchive is zipArchive
// Open an archive
ResOpenArchive = MyArchive.Open("C:\Archives\Archive.zip")

IF ResOpenArchive = 0 THEN
	// Processes
	...
	// Close the archive
	MyArchive.Close()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Archive>.Close()`
---

**`<Archive>: (Variable de tipo zipArchive)`**

Nombre de una Variable de tipo [zipArchive](../WDLang3/1000018679.md) correspondiente al archivo a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000250"></a>

- Un archivo comprimido sólo puede cerrarse si:

	- Se abre el archivo. Los archivos se abren con [Abrir](../WDLang3/1000019097.md).

	- El archivo existe. Los archivos se crean con [.Create](../WDLang3/1000019160.md).




- Todos los archivos se cerrarán automáticamente cuando se detenga la aplicación.

- Se produce un error de WLanguage si el archivo a cerrar no existe. El programa se detiene.


Recordatorio: 

- ![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo están disponibles los archivos ZIP, TAR y TGZ (TAR.GZ).

- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Sólo están disponibles los archivos ZIP.

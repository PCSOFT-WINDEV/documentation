


## BrightnessAuto( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el estado de activación del ajuste automatic del brillo del dispositivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables the automatic adjustment of brightness on the device
BrightnessAuto(True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Habilitando (o no) el ajuste del brillo de automatic

`BrightnessAuto(<Automatic mode>)`
---

**`<Automatic mode>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar el ajuste automático de la luminosidad, 

- <u><u><u><u>False</u></u></u></u> para desactivar el ajuste automático de la luminosidad.





<a name="SYNTAX2"></a>

### Averiguar el estado de activación del ajuste de la luminosidad de automatic

`<Result> = BrightnessAuto()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el ajuste automático de la luminosidad está activado,

- <u><u><u><u>False</u></u></u></u> si el ajuste automático de la luminosidad está desactivado.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000059"></a>

- Si el dispositivo no está equipado con un sensor de luz, la activación del ajuste de brillo de automatic no tiene ningún efecto.

- La modificación de la luminosidad afecta a la configuración del dispositivo. Por lo tanto, el nuevo brillo es global para todas las aplicaciones y se almacenará cuando la aplicación esté cerrada.

- Para conocer el valor del ajuste actual del brillo de la pantalla, utilice [BrightnessGet](../WDLang3/1000019810.md).

- Para modificar el valor del ajuste actual del brillo de la pantalla, utilice [BrightnessModify](../WDLang3/1000020170.md).





### Permisos necesarios
<a name="permisos_necesarios_ELTPARAGRAPHE000073"></a>

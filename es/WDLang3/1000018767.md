
## emailOutlookSession (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **emailOutlookSession** se utiliza para manipular el software de mensajería de Outlook (correos electrónicos, contactos, grupos de contactos, tareas, citas y carpetas). Las características de la conexión pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Atención**: Esta función no se puede utilizar para acceder a los datos que maneja el software de mensajería Outlook Express.

**Observaciones**: 

- Cuando se utiliza [EmailStartOutlookSession](../WDLang3/3032128.md), el directorio actual es modificado automáticamente por Outlook.

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
MySession is emailOutlookSession
MySession..Name = "OutlookProfile"
EmailStartSession(MySession)
```

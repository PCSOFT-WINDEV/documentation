
## FTPName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del último archivo al que se ha accedido mediante una función FTP (File Transfer Protocol).

**Observación**: **FTPName** es useful cuando se navega por los archivos de un servidor FTP con [FTPListFile](../WDLang3/3040018.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of last file accessed by an FTP function
ResName = FTPName()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FTPName()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del último archivo al que se accede mediante una función FTP. Este archivo fue modificado por una de las siguientes funciones FTP: [FTPAttribute](../WDLang3/3040006.md), [FTPDate](../WDLang3/3040015.md), [FTPTime](../WDLang3/3040013.md), [FTPSize](../WDLang3/3040007.md), [FTPListFile](../WDLang3/3040018.md).  

- Cadena vacía ("") si se produce un error.  Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Solicitud al servidor FTP
<a name="solicitud_servidor_ftp_ELTPARAGRAPHE000081"></a>

**FTPName** no envía ninguna solicitud al servidor FTP.
<a name="NOTE0_2"></a>


### Timeout
<a name="timeout_ELTPARAGRAPHE000090"></a>

De forma predeterminada, todas las funciones FTP fallan si no se recibe ninguna respuesta del servidor FTP después de 20 segundos. Este tiempo de espera se puede modificar al conectarse al servidor FTP con la función [FTPConnect](../WDLang3/3040016.md).
<a name="NOTE0_3"></a>
<a name="NOTE0_4"></a>


### Permisos de usuarios
<a name="permisos_usuarios_ELTPARAGRAPHE000118"></a>

Sólo un usuario que tenga derechos de lectura en el servidor FTP puede recuperar los nombres de los archivos que se encuentran en un servidor FTP.. En la mayoría de los casos, los usuarios "anónimos" tienen permisos de lectura.

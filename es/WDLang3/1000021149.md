


## SSHRead (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee los datos encontrados en el buffer de salida de la sesión SSH.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000021162&name=sshread_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// The session was started by SSHConnectShell

// Read the console of SSH shell after sending a command
// with SSHWrite
WHILE True
		
	// Consume the content of the console by leaving 1s rest	
	sStdBuffer = SSHRead(cMySession, 1000, sshStandardOutput)	
		
	// As long as the console is empty
	IF sStdBuffer = "" THEN
		...	
		// See the full example
		...	
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SSHRead(<SSH session> , <Timeout> [, <Output stream>])`
---

**`<Result>: (Buffer)`**

Datos leídos en la sesión de SSH. 

**`<SSH session>: (Variable de tipo sshSession)`**

Nombre de la [sshSession](../WDLang3/1000021157.md) Variable correspondiente a la sesión SSH que se va a utilizar. 

**`<Timeout>: (Entero o Duración)`**

Tiempo máximo de espera para recibir los datos (expresado en milisegundos). Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 


Este parámetro es set a 1000 ms por Default. 

**`<Output stream>: (Constante opcional de tipo Integer)`**

Tipo de stream a leer: 


|   |   |
| --- | --- |
| *sshErrorOutput* | Los datos se leen en la memoria intermedia de salida si se ha producido un error.. |
| *sshStandardOutput*<br> (valor predeterminado) | Los datos se leen en la memoria intermedia de salida estándar. |

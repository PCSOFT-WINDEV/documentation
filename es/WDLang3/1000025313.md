


## WLanguage Procedure llamado por BTStatus



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedure ("Callback") llamada por [BTStatus](../WDLang3/1000019360.md) cuando el estado de activación de Bluetooth cambia en el dispositivo. Este procedimiento puede ser local, global o interno.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
BTStatus(ChangeBT)

INTERNAL PROCEDURE ChangeBT(NewStatus, FormerStatus)
	IF NewStatus <> FormerStatus THEN ToastDisplay("Bluetooth connection changed")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BTStatus_Callback(<New status> , <Former status>)`
---

**`<New status>: (Constante de tipo Integer)`**

Nuevo estado de activación de Bluetooth en el dispositivo. Este estado corresponde a una de las siguientes constantes: 



|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |



**`<Former status>: (Constante de tipo Integer)`**

Estado de activación del Bluetooth en el dispositivo antes del cambio. Este estado corresponde a una de las siguientes constantes: 



|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |




## &lt;sshSession Variable&gt;.Escribir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe los datos en la sesión SSH especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMySession is sshSession
// The session was started by <sshSession>.ConnectShell. 
// Charact(10) is used to end the command line to send. 

// Position in the /var/log folder
cMySession.Write("cd /var/log" + Charact(10))

// Ask to read the lines containing "error"
cMySession.Write("grep -i error /var/log/messages" + Charact(10))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <SSH session>.Write(<Data>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los datos se enviaron correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<SSH session>: (Variable de tipo sshSession)`**

Nombre de la [sshSession](../WDLang3/1000021157.md) Variable correspondiente a la sesión SSH que se va a utilizar.

**`<Data>: (Buffer)`**

Datos que se escribirán en la sesión de SSH.

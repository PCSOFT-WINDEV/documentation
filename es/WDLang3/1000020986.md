


## &lt;upnpDevice Variable&gt;.MediaOpen (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un medio en el dispositivo UPnP.

**Atención**: Para utilizar esta función, el dispositivo debe incluir un servicio "AVTransport"..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrDevice is array of upnpDevices 
arrDevice = UPNPListDevice(...
		"urn:schemas-upnp-org:device:MediaRenderer:1", 5)
FOR EACH Device of arrDevice
	Device.MediaOpen("http://172.17.1.5/mymedia.avi")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <UPnP device>.MediaOpen(<Media>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió el contenido multimedia,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<UPnP device>: (Variable de tipo upnpDevice)`**

Nombre de [upnpDevice](../WDLang3/1000020903.md) Variable correspondiente al dispositivo UPnP a utilizar.

**`<Media>: (Cadena de caracteres)`**

URI de los medios de comunicación para abrir.

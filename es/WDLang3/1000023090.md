


## &lt;mqttSession Variable&gt;.UnsubscribeAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite darse de baja de todas las suscripciones existentes.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gMqttSession is mqttSession

gMqttSession.UnsubscribeAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <MQTT session>.UnsubscribeAll()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizaron todas las suscripciones,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles.




**`<MQTT session>: (mqttSession variable)`**

Nombre de [mqttSession](../WDLang3/1000023023.md) Variable correspondiente a la sesión MQTT a utilizar.

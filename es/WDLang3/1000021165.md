
## httpResponse (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **httpResponse** se utiliza para get las características avanzadas de la respuesta de la petición HTTP/HTTPS. Las características de la solicitud pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMyRequest is httpRequest
cMyResponse is httpResponse
cMyRequest.URL = "http://www.windev.com"
cMyResponse = HTTPSend(cMyRequest)
IF ErrorOccurred THEN
	Error(ErrorInfo(errFullDetails))
ELSE
	Info(cMyResponse.Content)
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables httpResponse
<a name="propiedades_especificas_descripcion_las_variables_httpresponse_ELTPARAGRAPHE000043"></a>

Las siguientes propiedades pueden ser utilizadas para manipular la respuesta de una petición HTTP/HTTTPS:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Cabecera | array asociativa de cadenas de caracteres | Clave/Valor set de los encabezados recibidos. |
| Contenido | Buffer | Cuerpo de respuesta del servidor.<br><br>**Esta propiedad es de solo lectura**. |
| ContentType | Cadena de caracteres | Tipo de contenido para el mensaje HTTP recibido.<br><br>**Esta propiedad es de solo lectura**. |
| DescriptionStatusCode | Cadena de caracteres | description del **StatusCode** Property. <br><br>**Esta propiedad es de solo lectura**. |
| RoughHeader | Cadena de caracteres | Cabecera que se devolvió con la respuesta.<br><br>**Esta propiedad es de solo lectura**. |
| StatusCode | Integro | Estado de la respuesta. Este Property corresponde a uno de los códigos del protocolo HTTP. Los códigos más comunes son: <br><br>- 200: Solicitud procesada con éxito<br><br>- 301: Document se movió permanentemente<br><br>- 302: Document se trasladó temporalmente<br><br>- 403: El servidor ha entendido la petición pero se niega a ejecutarla.<br><br>- 404: No se ha encontrado el recurso<br><br>- 500: Error interno del servidor<br><br>- 503: Servicio temporalmente no disponible o en mantenimiento<br><br><br><br><br>La propiedad **description StatusCode** obtiene el description del código de error. <br><br>**Esta propiedad es de solo lectura**. |






<a name="NOTE0_5"></a>


### Múltiples respuestas HTTP
<a name="multiples_respuestas_http_ELTPARAGRAPHE000114"></a>

Si la respuesta contiene varios resultados HTTP (por ejemplo, si un proxy añade un código HTTP), las propiedades **StatusCode** y **DescriptionStatusCode** devuelven los valores de la primera respuesta HTTP. En este caso, el **RoughHeader** Property contiene todas las respuestas.

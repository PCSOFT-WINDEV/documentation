<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Usando los gráficos en WEBDEV Linux**

Para utilizar los gráficos en WEBDEV Linux, es necesario instalar la biblioteca QT (Core y GUI) versión 5 (o posterior). 

**En Ubuntu**, se puede instalar esta biblioteca con los siguientes comandos:


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```


## WiFiSelectNetworkCard( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona la tarjeta de red Wi-Fi que se utilizará para las próximas llamadas a las funciones WiFi.

**Observación**: La llamada a esta función es opcional. La primera tarjeta de red Wi-Fi disponible es utilizada por Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sMyList is string
nID is int  
sDescription is string

sMyList = WiFiListNetworkCard()
// Retrieves the identifier of the first card found
nID = Val(ExtractString(sMyList, 1, TAB))
// Selects the card 
WiFiSelectNetworkCard (nID)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WiFiSelectNetworkCard(<Identifier of network card>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se seleccionó la tarjeta de red Wi-Fi. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el problema (por ejemplo, si la tarjeta de red Wi-Fi ya no está disponible).




**`<Identifier of network card>: (Cadena de caracteres)`**

Identificador de la tarjeta de red Wi-Fi a seleccionar. Este identificador es devuelto por [Tarjeta WiFiListNetworkCard](../WDLang3/1000020693.md).

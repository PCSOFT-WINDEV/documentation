
## SaaSAdminListClient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las cuentas de clientes de la base de datos SaaS Webservice.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrClient is array of saasClient 
arrClient = SaaSAdminListClient()
FOR EACH Client OF arrClient
	Trace(Customer.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminListClient()`
---

**`<Result>: (Array de las variables de la saasClient)`**

Lista de cuentas de clientes para el SaaS Webservice.

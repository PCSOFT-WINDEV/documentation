
## Galleta (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Cookie** permite manipular las cookies transmitidas en solicitudes HTTP. Las características de las cookies pueden ser definidas y modificadas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the cookies of previous session again
MyCookies is array of cookies
Deserialize(MyCookies, fLoadText("MyCookies.xml"), psdXML)

// Enable the management of cookies
HTTPCookieManage(True)
HTTPCookieReplace(".yahoo.com", MyCookies)

// Request (the cookies will be sent to the Web server)
HTTPRequest("www.yahoo.com")
```

## Observaciones


### Propiedades específicas de las variables Cookie
<a name="propiedades_especificas_las_variables_cookie_ELTPARAGRAPHE000061"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a Cookie:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Camino | Cadena de caracteres | Validez ruta de Cookie.<br><br>Indica al servidor los URLs en los que debe interpretarse la Cookie. |
| comment | Cadena de caracteres | Libere el valor de comment associated con el Cookie.<br><br>Este Property no afecta a las peticiones HTTP. |
| denominar | Cadena de caracteres | Nombre de Cookie.<br><br>Este nombre no distingue entre mayúsculas y minúsculas. |
| Dominio | Cadena de caracteres | Dominio de validez de Cookie.<br><br>El Cookie se enviará a todas las solicitudes HTTP cuyo dominio termine con el dominio de validez de Cookie.<br><br>Por ejemplo, un Cookie cuyo dominio de validez es ".midominio.fr" será enviado durante las siguientes solicitudes:<br><br>- http://mydomain.fr<br><br>- http://www.mydomain.fr/index.html<br><br>- http://server.mydomain.fr/Page.awp<br><br><br>No se enviará en las siguientes solicitudes:<br><br>- http://www.mydomain.com<br><br>- http://otherdomain.fr<br><br><br> |
| Valor | Cadena de caracteres | Valor de Cookie.<br><br>Observaciones:<br><br>- El valor de un Cookie está limitado a 4KB. Más allá de este tamaño, el servidor es libre de rechazar el Cookie.<br><br>- Los caracteres de control (TAB, CR, etc.) así como el punto y coma y los ceros binarios no están permitidos.<br><br><br> |
| Vencimiento | Cadena de caracteres o<br>Variable de tipo DateTime | Fecha y hora de expiración de Cookie.<br><br>En la mayoría de los casos, este Property es set por el servidor Web para evitar recuperar información demasiado antigua en una solicitud HTTP. |

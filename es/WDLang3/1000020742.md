


## xmlSignatureInformationKeyX509 (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **xmlSignatureInformationKeyX509** define la información que se debe incluir en la parte **KeyInfo** de la firma XML. Esto te permite añadir la información sobre un X509 Certificate. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySignature is xmlSignature
Info is xmlSignatureInformationKeyX509
Info.WithCertificate = True
Add(MySignature.KeyInformation, Info)
```

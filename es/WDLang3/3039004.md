


## Funciones de gestión de flujos (vídeo y sonido)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los flujos:



|   |   |
| --- | --- |
| [StreamAccept](../WDLang3/3039003.md) | Asocia una solicitud de conexión de flujo con una control Conferencia. |
| [StreamChangeDriver](../WDLang3/1000021613.md) | Especifica la cámara que se utilizará como fuente para una secuencia y/o un control Conferencia. |
| [StreamConfigure](../WDLang3/1000021645.md) | Especifica los parámetros de red necesarios para utilizar los controles de la Conferencia en INTERNET. |
| [StreamConnect](../WDLang3/3039008.md) | Conecta una corriente (y el control de associated) a un Address específico. |
| [StreamDisconnect](../WDLang3/3039007.md) | Detiene la recepción o transmisión de un stream. |
| [StreamInfo](../WDLang3/3039009.md) | Devuelve el nombre y/o número del emisor del stream. |
| [StreamPlay](../WDLang3/3039010.md) | Reanuda la recepción o la emisión de un tipo de datos en un flujo. |
| [StreamReject](../WDLang3/3039006.md) | Rechaza una solicitud de conexión a la red. |
| [StreamStop](../WDLang3/3039002.md) | Detiene la recepción o transmisión de un tipo de datos en un flujo. |




## grSunburstRootSection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia o especifica el ruta de la raíz en un gráfico de Sunburst.

**Observación**: Esta función se utiliza para desplazarse en la jerarquía del gráfico (desglosar o desglosar).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grSunburstRootSection(CHART_Chart, "Europe" + TAB + "France")
grDraw(CHART_Chart)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSunburstRootSection(<Chart name> , <Section path>)`
---

**`<Chart name>: (Character string (with or without quotes))`**

Nombre del gráfico a manipular. Este nombre corresponde a:

- el nombre del gráfico definido por programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico que se encuentra en el editor de ventanas, editor de páginas o editor de reportes.




**`<Section path>: (Character string)`**

ruta completa de la sección que se utilizará en root. Este parámetro tiene el siguiente formato:


```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```


Esta ruta define la posición de la sección especificando los diferentes niveles jerárquicos. 

Si la ruta no existe, se utiliza la raíz global del gráfico.

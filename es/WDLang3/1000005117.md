


## Notas Funciones de gestión de Document
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los documentos de Notes que se encuentran en Lotus Notes:



|   |   |
| --- | --- |
| [NotesActivateView](../WDLang3/1000005118.md) | Indica la vista que debe manejarse en Lotus Notes. |
| [NotesAddAttachment](../WDLang3/1000005119.md) | Adjunta un archivo a un elemento del actual Document en Lotus Notes. |
| [NotesCloseDatabase](../WDLang3/1000005111.md) | Cierra la base de datos local o remota utilizada (Lotus Notes o Domino). |
| [NotesDeactivateView](../WDLang3/1000005120.md) | Desactiva la vista que se maneja actualmente en Lotus Notes. |
| [NotesDeleteAttachment](../WDLang3/1000005126.md) | Borra un archivo adjunto de un elemento de la actual Document en Lotus Notes. |
| [NotesDeleteItem](../WDLang3/1000005116.md) | Borra el elemento especificado de la actual Document en Lotus Notes. |
| [NotesDocumentDelete](../WDLang3/1000017085.md) | Elimina la Document actual de la base de datos de Lotus Notes consultada. |
| [NotesDocumentFirst](../WDLang3/1000005107.md) | Posiciones sobre la primera Document encontrada en Lotus Notes. |
| [NotesDocumentIdentifier](../WDLang3/1000017084.md) | Devuelve el identificador del actual Document en la base de datos de Lotus Notes que se ha navegado. |
| [NotesDocumentLast](../WDLang3/1000005105.md) | Posiciones sobre la última Document encontrada en Lotus Notes. |
| [NotesDocumentNext](../WDLang3/1000005108.md) | Posiciones en la Document encontradas después de la Document actual en Lotus Notes. |
| [NotesDocumentPrevious](../WDLang3/1000005106.md) | Posiciones en la Document encontradas antes de la Document actual en Lotus Notes. |
| [NotesDocumentSeek](../WDLang3/1000005104.md) | Busca un Document en Lotus Notes. |
| [NotesExtractAttachment](../WDLang3/1000005121.md) | Extrae un archivo adjunto a un elemento de la Document actual en Lotus Notes. |
| [NotesItem](../WDLang3/1000005114.md) | Devuelve el valor de los artículos para los actuales billetes Document en Lotus Notes. |
| [NotesItemDimension](../WDLang3/1000005103.md) | Devuelve la dimensión de un artículo para el actual Document en Lotus Notes. |
| [NotesListAttachment](../WDLang3/1000005122.md) | Devuelve la lista de archivos adjuntos a un elemento del actual Document en Lotus Notes. |
| [NotesListItem](../WDLang3/1000005112.md) | Devuelve la lista de artículos para las actuales Notas Document en Lotus Notes. |
| [NotesListView](../WDLang3/1000005123.md) | Devuelve la lista de vistas que se encuentra en la actual base de datos de Lotus Notes. |
| [NotesModifyAttachment](../WDLang3/1000005124.md) | Modifica un archivo adjunto a un elemento de la actual Document en Lotus Notes. |
| [NotesModifyItem](../WDLang3/1000005113.md) | Modifica el elemento especificado de la Document actual en Lotus Notes. |
| [NotesNbAttachment](../WDLang3/1000005125.md) | Devuelve el número de archivos adjuntos a un elemento del actual Document en Lotus Notes. |
| [NotesOpenConnection](../WDLang3/1000005102.md) | Establece una conexión con un servidor de Domino. |
| [NotesOpenDatabase](../WDLang3/1000005101.md) | Da acceso a los documentos gestionados por Lotus Notes. |
| [NotesOut](../WDLang3/1000005109.md) | Se utiliza para averiguar si existe una Document actual en Lotus Notes. |
| [NotesReset](../WDLang3/1000005115.md) | Crea una Document vacía en Lotus Notes. |
| [NotesSave](../WDLang3/1000005110.md) | Guarda la Document actual en Lotus Notes. |
| [NotesViewListColumn](../WDLang3/1000005129.md) | Enumera las columnas que se encuentran en la vista actual de Lotus Notes. |
| [NotesViewSeekFirst](../WDLang3/1000005128.md) | Busca la primera Document encontrada en la vista actual de Lotus Notes. |
| [NotesViewSeekLast](../WDLang3/1000005127.md) | Busca la última Document encontrada en la vista actual en Lotus Notes. |


## AlbumSave( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un Image, una foto o un video en el álbum de fotos del dispositivo móvil. Esta Image puede ser reutilizada por [AlbumPicker](../WDLang3/1000020186.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save an image found in an image variable
MyImage is Image = "Test.png"
AlbumSave(MyImage)
```


<a name="Example2"></a>



```wl
// Save a photo
IF AlbumSave(VideoStartApp(viPictureCapture)) = False THEN
	Info("Image not saved")
END
```


<a name="Example3"></a>



<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Guarda una imagen que se encuentra en una imagen Variable

`<Result> = AlbumSave(<Image>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el Image fue salvado con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image>: (Imagen Variable)`**

Nombre de la [Image](../WDLang1/1000019650.md) Variable que corresponde a la Image que se va a añadir al álbum.


<a name="SYNTAX2"></a>

### Guardar un Image mostrado en un Image o control Gráfico

`<Result> = AlbumSave(<Control name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el Image fue salvado con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control name>: (Cadena de caracteres)`**

Nombre de la control Imagen o nombre de la control Gráfico que contiene la Image para añadirla al álbum.


<a name="SYNTAX3"></a>

### Guarda un Image encontrado en un archivo

`<Result> = AlbumSave(<Image file name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el Image fue salvado con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image file name>: (Cadena de caracteres)`**

Nombre completo o nombre relativo del archivo Image o de vídeo que se añadirá al álbum (nombre del archivo devuelto por [VideoStartApp](../WDLang1/1000019368.md), por ejemplo).

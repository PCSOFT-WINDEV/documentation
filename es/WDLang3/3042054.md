


## grDeleteSeries (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una serie de datos de un gráfico. Se borran los datos encontrados en la serie.. 

Atención: Las series restantes serán renumeradas.. Por ejemplo, si se borra la serie 1, la serie 2 se convierte en la serie 1, ....



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the second data series from "MyChart"
grDeleteSeries("MyChart", 2)
```
<a name="Example2"></a>

```wl
// Delete the third data series from "CHART_MyChart"
grDeleteSeries(CHART_MyChart, 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDeleteSeries(<Chart name> [, <Series number> [, <Options>]])`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico definido por [grCreate](../WDLang3/3042005.md) o nombre de control Gráfico.


**`<Series number>: (Entero opcional)`**

Número de la serie a borrar. Si no se especifica este parámetro, los datos completos se borran del gráfico.

**`<Options>: (Constante de tipo Integer)`**

Tipo de borrado a realizar: 


|   |   |
| --- | --- |
| *grData* | Borrar datos de la serie. |
| *grDataAndStyle*<br>(Valor predeterminado) | Borrar datos y estilo de la serie. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se borran todas las series de gráficos, se deben añadir nuevos datos para volver a dibujar el gráfico.

- Las modificaciones realizadas con la función **grDeleteSeries** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- Para una serie basada en un elemento, los datos se borran. La fuente de datos se almacena y los datos pueden ser reconstruidos.. Para cortar el enlace a la fuente, utilice [grSourceSeries](../WDLang3/3042067.md) y modifique la fuente de la serie.

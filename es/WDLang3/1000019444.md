
## NotifDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una Notification que aparece en la barra de sistema del dispositivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
notif is Notification
notif.Title = "New incoming message"
notif.Message = "Click here to display the message"
NotifAdd(notif)
...
NotifDelete(notif)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NotifDelete(<Notification>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la notificación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**Observación**: En el simulador, esta función no tiene ningún efecto y siempre devuelve <u><u><u><u>False</u></u></u></u>. 

**`<Notification>: (Notification Variable)`**

Nombre de la variable [Notificación](../WDLang3/1000019441.md) que corresponde al Notification que se va a eliminar.




## grGradient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la visualización de los gradientes de Color en un gráfico. 

**Recordatorio**: Por defecto, los colores que se muestran en un gráfico son los colores de Gradient.
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart created through programming
grCreate("MyChart", grCandlestick)
...
// Disables the color gradients
grGradient("MyChart", False)
```
<a name="Example2"></a>

```wl
// Chart control
// Disables the color gradients
grGradient(CHART_MyChart, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grGradient(<Chart name> , <Gradient>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Gradient>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para mostrar el gráfico utilizando los degradados de colores,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los gráficos muestran los gradientes de Color de Default. Los gradientes de Color pueden ser mostrados por todos los tipos de gráficos.

- Los ajustes definidos con **grGradient** se tendrán en cuenta la próxima vez que se llame a [grDraw](../WDLang3/3042026.md).

- Durante la impresión, el modo Gradient no se tendrá en cuenta para los gráficos de tarta.

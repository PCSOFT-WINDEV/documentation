


## &lt;modbusSession&gt;.ReadRegister (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee uno o más registros de un esclavo Modbus, comenzando en una Address específica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a session and connect to a Modbus server
Session is modbusSession
Session = ModbusConnectTCP("192.168.1.7")

arrResponse is array of 5 2-byte int

// If the connection fails, display the error
IF ErrorOccurred = True THEN
	Trace(ErrorInfo())
ELSE
	// Reads a sequence of 5 registers in read-write mode starting at address 0x42.
	arrResponse = Session.ReadRegister(0x42, 5, False)
	// Display the error, if any.
	IF ErrorOccurred = True THEN
		Trace(ErrorInfo())	
	END
END

// Disconnection
Session.Disconnect()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Leyendo 1 registro de un esclavo Modbus

`<Result> = <Session>.ReadRegister(<Address> , <Read-only>)`
---

**`<Result>: (Integro)`**

Valor del registro leído (entero de 16 bits).

**`<Session>: (modbusSession Variable)`**

Nombre del [modbusSession](../WDLang3/1000025924.md) Variable que corresponde a la sesión de comunicación con el esclavo Modbus.

**`<Address>: (Integro)`**

Address del registro en el esclavo (entre 0 y 65.535).

**`<Read-only>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los registros a leer son parte de los datos de sólo lectura (código de función Modbus 4 (Leer registros de entrada)). 

- <u><u><u><u>False</u></u></u></u> si los registros a leer son parte de los datos de lectura-escritura (código de función Modbus 3 (Leer múltiples registros de retención)).





<a name="SYNTAX2"></a>

### La lectura de múltiples registros de un esclavo Modbus

`<Result> = <Session>.ReadRegister(<Address> , <Number of registers> , <Read-only>)`
---

**`<Result>: (Array)`**

Array de enteros de 16 bits que contiene los valores de los registros leídos.

**`<Session>: (modbusSession Variable)`**

Nombre del [modbusSession](../WDLang3/1000025924.md) Variable que corresponde a la sesión de comunicación con el esclavo Modbus.

**`<Address>: (Integro)`**

Comenzando Address de los registros en el esclavo (entre 0 y 65.535).

**`<Number of registers>: (Integro)`**

Número de registros a leer.

**`<Read-only>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los registros a leer son parte de los datos de sólo lectura (código de función Modbus 4 (Leer registros de entrada)). 

- <u><u><u><u>False</u></u></u></u> si los registros a leer son parte de los datos de lectura-escritura (código de función Modbus 3 (Leer múltiples registros de retención)).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función bloquea la aplicación hasta que el esclavo responda al mensaje o hasta que se supere el tiempo de espera de la sesión. 

- Después de llamar a **&lt;modbusSession&gt;.ReadRegister**, se recomienda comprobar el [ErrorOccurred](../WDLang1/3087001.md) Variable para asegurarse de que no hay errores.




## grSeriesOpacidad (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el factor de opacidad de una serie en un gráfico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sets the percentage of opacity to 50% for series 1
grSeriesOpacity(CHART_MyChart, 1, 50)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSeriesOpacity(<Chart name> , <Series number> , <Percentage of opacity>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de control Gráfico (que se encuentra en el editor de ventanas o en el editor de reportes).




**`<Series number>: (Integro)`**

Número de la serie para la que debe modificarse la opacidad.

**`<Percentage of opacity>: (Integro)`**

Porcentaje de opacidad de la serie. Este porcentaje se incluye entre 0 (serie totalmente Invisible) y 100 (serie totalmente opaca).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


- El cambio de opacidad es tenido en cuenta por todos los tipos de cartas. 

- La opacidad definida para la serie tiene prioridad sobre la opacidad global definida para la control Gráfico ([Opacidad](../Proprietes/2514006.md)).

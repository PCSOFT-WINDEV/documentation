


## &lt;Tabla&gt;.Color (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve e inicializa la Color de diferentes elementos de la carta. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the colors in a Chart control
CHART_Chart1.Color(grColorLegend, LightRed)
CHART_Chart1.Draw() // Border of the legend drawn in red
CHART_Chart1.Color(grColorLegend, LightBlue) // The border is still red
CHART_Chart1.Draw() // The border of the legend drawn in blue
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Descubrir la Color de un elemento en una control Gráfico

`<Result> = <Chart control>.Color(<Element>)`
---

**`<Result>: (Integro)`**

Color del elemento especificado en formato [RGB](../WDLang1/3029012.md). 
**Atención**: El valor devuelto puede variar en función de la configuración de la pantalla (resolución, ....).

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Element>: (Constante de tipo Integer)`**

Elemento de la carta cuya Color se solicita. Este elemento puede corresponder a una de las siguientes constantes:



|   |   |
| --- | --- |
| *grColorAxis* | Color de ejes. |
| *grColorAxisSide* | Color de la parte elevada de los ejes.<br> |
| *grColorBorder* | Color de Border para los elementos de la carta (para las cartas que no sean de 3D). Se utiliza para encontrar el borde Color para una barra en un gráfico de columnas, por ejemplo. |
| *grColorBullishCandlestick* | Color de barras en un candelero cuando el valor de cierre es mayor que el valor de apertura.<br> |
| *grColorGridlines* | Color cuadrícula. |
| *grColorGridlinesSide* | Color de la parte elevada de la rejilla.<br> |
| *grColorLegend* | Color borde leyenda. |
| *grColorLegendBackground* | Color de fondo de la leyenda. |




<a name="SYNTAX2"></a>

### Modificación de la Color de un elemento en una control Gráfico

`<Chart control>.Color(<Element> [, <New color>])`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Element>: (Constante de tipo Integer)`**

Elemento de la carta cuya Color debe ser modificada. Este elemento puede corresponder a una de las siguientes constantes:



|   |   |
| --- | --- |
| *grColorAxis* | Color de ejes. |
| *grColorAxisSide* | Color de la parte elevada de los ejes.<br><br> |
| *grColorBorder* | Color de Border para los elementos de la carta (para las cartas que no sean de 3D). Se utiliza para encontrar el borde Color para una barra en un gráfico de columnas, por ejemplo. |
| *grColorBullishCandlestick* | Color de barras en un candelero cuando el valor de cierre es mayor que el valor de apertura.<br><br> |
| *grColorGridlines* | Color cuadrícula. |
| *grColorGridlinesSide* | Color de la parte elevada de la rejilla.<br><br> |
| *grColorLegend* | Color borde leyenda. |
| *grColorLegendBackground* | Color de fondo de la leyenda. |



**`<New color>: (Entero o constante (opcional))`**

Color del elemento especificado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).

## Observaciones


### ¿Cómo obtener los componentes del color?
<a name="¿como_obtener_los_componentes_del_color_ELTPARAGRAPHE000401"></a>

Obtener los componentes del color devuelto por la función **&lt;Tabla&gt;.Color**:

- Para obtener el componente rojo, use [RGBRed](../WDLang1/3029050.md)  (o realice una división módulo 256 de &lt;Resultado&gt;), 

- Para obtener el componente verde, use [RGBGreen](../WDLang1/3029051.md) (o realice una división módulo 256 de &lt;Resultado&gt;/256),

- Para obtener el componente azul, use [RGBBlue](../WDLang1/3029049.md) (o realice una división módulo 256 de &lt;Resultado&gt;/256/256).



<a name="NOTE0_2"></a>


### Colores de los elementos de la carta (leyendas, ....)
<a name="colores_los_elementos_carta_leyendas_ELTPARAGRAPHE000426"></a>

- Los ajustes definidos con **&lt;Tabla&gt;.Color** se tendrán en cuenta la próxima vez que se llame a [&lt;Dibujo](../WDLang3/1000024062.md) o la próxima vez que se dibuje el gráfico (se redimensiona el host control, se modifica el menú contextual, etc.).

- El color transparente solo se puede utilizar para el color de fondo de la leyenda (constante *grColorLegendBackground*).

- **Imprimir el gráfico**: Si el color blanco está asociado a los ejes (constante *grColorAxis*), a las cuadrículas (constante *grColorGridlines*), a la leyenda (constante *grColorLegend*) y al borde (constante *grColorBorder*), el dibujo se imprimirá en Black.



<a name="NOTE0_3"></a>


### Colores en las cartas (serie, sección, ....)
<a name="colores_las_cartas_serie_seccion_ELTPARAGRAPHE000445"></a>

- La Color de los candelabros Bearish está definida por la Color de la serie.

- [&lt;Gráfico&gt;.SeriesColor](../WDLang3/1000023599.md) se utiliza para modificar el color de una serie. 

- [&lt;Gráfico&gt;.CategoryColor](../WDLang3/1000023570.md) se utiliza para modificar el Color de categorías para los gráficos Pie y Color. 

- [&lt;Gráfico&gt;.ScatterSeriesColor](../WDLang3/1000024106.md) se utiliza para inicializar el Color de una serie en un gráfico de "dispersión". 

- [&lt;Gráfico&gt;.SunburstSectionColor](../WDLang3/1000024134.md) se utiliza para cambiar el Color de una sección en un gráfico Sunburst.

- [&lt;Tabla&gt;.SurfaceAltitudeColor](../WDLang3/1000024144.md) se utiliza para definir el Color utilizado para la altitud (eje Z) en un gráfico de superficie.

- [&lt;Gráfico&gt;.WaterfallColor](../WDLang3/1000024158.md) indica el Color para un tipo de categoría en un gráfico de Cascada.

- [&lt;HMColor](../WDLang3/1000024080.md) se utiliza para definir el Color de los valores en un gráfico Heatmap.

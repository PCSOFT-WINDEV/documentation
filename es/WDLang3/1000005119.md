
## NotesAddAttachment( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adjunta un archivo a un elemento del actual Document en Lotus Notes.

Para tener en cuenta el anexo, la Document debe guardarse con [NotasGuardar](../WDLang3/1000005110.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the connection
ConnectionID is int
ConnectionID = NotesOpenConnection("Password", "MarsServer", ...
		"C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")

// Specify the Lotus Notes database to use
DatabaseID is int
DatabaseID = NotesOpenDatabase(ConnectionID, "julia.nsf")
...
// Read the first document
NotesDocumentFirst(DatabaseID)
...
// Attach a file to the current document 
NotesAddAttachment(DatabaseID, "MyLotusNotesItem", "C:\MyDocuments\MyFile.TXT")
// Take this attachment into account
NotesSave(DatabaseID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesAddAttachment(<Session identifier> , <Lotus Notes item> , <File to attach>)`
---

**`<Result>: (Integro)`**



- Índice del archivo adjunto. Este número corresponde al número de archivos adjuntos al elemento Lotus Notes especificado después de la adición.

- 0 si se produce un error. 




Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<Lotus Notes item>: (Cadena de caracteres)`**

Nombre del elemento de Lotus Notes (Texto enriquecido) al que debe adjuntarse el archivo. El elemento Lotus Notes se crea automáticamente si no existe.

Para conocer el nombre y el tipo de los elementos de las notas actuales Document, utilice [NotesListItem](../WDLang3/1000005112.md).

**`<File to attach>: (Cadena de caracteres)`**

Nombre y ruta completo (o relativo) del archivo a adjuntar al elemento especificado en las actuales Notas Document. Este archivo se copiará automáticamente en la base de datos actual de Lotus Notes..

## Observaciones




### Artículo de un Notas Document
<a name="articulo_notas_document_ELTPARAGRAPHE000067"></a>

Cada elemento de un Document de notas puede contener hasta 256 archivos adjuntos.

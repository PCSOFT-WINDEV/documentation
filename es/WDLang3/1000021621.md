


## &lt;httpRequest Variable&gt;.ReadCookie (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor de un Cookie recibido tras una petición HTTP.






<a name="SYNTAX2"></a>

### La sintaxis que se puede usar con HTTPSend

`<Result> = <HTTP request>.ReadCookie(<Domain> , <Name>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de Cookie.

- Cadena vacía ("") si el Cookie no existe. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u>True</u></u></u>.




**`<HTTP request>: (Variable de tipo httpRequest)`**

Nombre del [httpRequest](../WDLang3/1000021158.md) Variable para el que se leerá el Cookie.

**`<Domain>: (Cadena de caracteres)`**

Nombre del dominio para el que se recibió la Cookie.

**`<Name>: (Cadena de caracteres)`**

Nombre de Cookie.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para utilizar **&lt;httpRequest Variable&gt;.ReadCookie**, la gestión de las cookies debe haber sido habilitada por [&lt;httpRequest Variable&gt;.ManageCookie](../WDLang3/1000021608.md).

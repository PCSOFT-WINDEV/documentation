


## <zipArchive>.SelectFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona los archivos de un archivo. Los archivos seleccionados serán procesados por las funciones de gestión de archivos WLanguage.. 

Esto hace posible seleccionar sólo los archivos ".doc" de un archivo, por ejemplo. En este caso, todas las funciones de WLanguage utilizadas para manipular todo el archivo (por ejemplo, [.ExtractAll](../WDLang3/1000019085.md)) sólo afectarán a los archivos seleccionados (archivos ".doc" en nuestro ejemplo).<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Archive is zipArchive
// Open an archive
IF Archive.Open("C:\Archives\Archive.zip") = 0 THEN
	// Extract documents only
	sPreviousSelection is string = Archive.SelectFile("*.doc")
	// Extract all documents from the archive 
	Archive.ExtractAll(zipDrive)
	// Restore previous selection
	Archive.SelectFile(sPreviousSelection)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Archive>.SelectFile([<Selection>])`
---

**`<Result>: (Cadena de caracteres)`**

Selección previa aplicada al archivo.

**`<Archive>: (Variable de tipo zipArchive)`**

Nombre de una Variable de tipo [zipArchive](../WDLang3/1000018679.md) correspondiente al archivo a utilizar.

**`<Selection>: (Cadena de caracteres opcional)`**

Nombre genérico de los archivos a seleccionar. 
Los caracteres genéricos (\* y?) están permitidos. 
Si no se especifica este parámetro o si set es una cadena vacía (""), la selección será eliminada y todos los archivos serán procesados.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Principio
<a name="principio_ELTPARAGRAPHE000212"></a>

Después de llamar a **<zipArchive>.SelectFile**, las siguientes funciones de gestión de archivos que manipular todos los ficheros del archivo tendrán en cuenta la selección: estas funciones sólo manipularán los archivos seleccionados. 

Para volver a manipular todos los ficheros del archivo, basta con llamar a **<zipArchive>.SelectFile** sin especificar el parámetro &lt;Selección&gt;..

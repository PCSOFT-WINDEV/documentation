


## Funciones MQTTT
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los clientes MQTT: 



|   |   |
| --- | --- |
| [MQTTConnect](../WDLang3/1000023021.md) | Se utiliza para conectarse a un servidor MQTT (también llamado "Broker"). |
| [MQTTDesconectar](../WDLang3/1000023022.md) | Le permite desconectarse de un servidor MQTT (también llamado "Broker"). |
| [MQTTPublicar](../WDLang3/1000023060.md) | Publica un mensaje en el broker MQTT. |
| [MQTTTSubscribe](../WDLang3/1000023073.md) | Se utiliza para suscribirse a un MQTT Notification. |
| [MQTTUnsubscribe](../WDLang3/1000023074.md) | Se utiliza para cancelar la suscripción de un MQTT Notification. |
| [MQTTUnsubscribeAll](../WDLang3/1000023080.md) | Le permite darse de baja de todas las suscripciones existentes. |


## Constantes para las funciones de geolocalización
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Las siguientes constantes son utilizadas por las funciones de geolocalización: 

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **geoKilometer** | Devuelve la distancia en kilómetros. | [geoDistance](../WDLang3/1000019211.md) |
| **geoMap2D** | La ubicación se muestra en un mapa bidimensional en la aplicación nativa de mapas del dispositivo | [geoRunApp](../WDLang3/1000019954.md) |
| **geoMeter** | Devuelve la distancia en metros. | [geoDistance](../WDLang3/1000019211.md) |
| **geoMile** | Devuelve la distancia en millas. | [geoDistance](../WDLang3/1000019211.md) |
| **geoTrackingDisabled** | Estado del sistema de seguimiento: El sistema de localización está desactivado. | [geoTrackingStatus](../WDLang3/1000021592.md) |
| **geoTrackingEnabled** | Estado del sistema de seguimiento: El sistema de localización está activado. | [geoTrackingStatus](../WDLang3/1000021592.md) |
| **geoTrackingError** | Estado del sistema de seguimiento: Error al recuperar el estado de activación del sistema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). | [geoTrackingStatus](../WDLang3/1000021592.md) |
| **geoView360** | La ubicación se muestra como una Image con una vista de 360° en la aplicación de mapas nativa del dispositivo. | [geoRunApp](../WDLang3/1000019954.md) |
| **gpsAltitude** | El proveedor de localización proporciona información sobre la altitud. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsAuto** | Define automáticamente el proveedor de localización. | [GPSInitParameter](../WDLang3/1000019204.md) |
| **gpsAvailable** | El proveedor de localización está disponible. | [GPSStatus](../WDLang3/1000019207.md) |
| **gpsDirection** | El proveedor de localización proporciona información sobre la dirección. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsDisabled** | El proveedor de localización está desactivado. | [GPSStatus](../WDLang3/1000019207.md) |
| **gpsEnabled** | El proveedor de localización está habilitado. | [GPSStatus](../WDLang3/1000019207.md) |
| **gpsEnergy** | Solicita información sobre el consumo de energía del proveedor de la localización. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergyHigh** | El proveedor de la localización puede necesitar una gran cantidad de energía del dispositivo. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergyLow** | El proveedor de la localización debe requerir una baja cantidad de energía del dispositivo. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsEnergyMedium** | El proveedor de la localización puede necesitar una cantidad media de energía del dispositivo. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsError** | Se ha producido un error al recuperar el estado del proveedor de la localización. | [GPSStatus](../WDLang3/1000019207.md) |
| **gpsErrorOK** | No se ha detectado ningún error. | [GPSGetPosition](../WDLang3/1000019205.md), [GPSFollowMovement](../WDLang3/1000019206.md) |
| **gpsErrorPosition** | No se ha podido determinar la ubicación (por ejemplo, error interno del proveedor de acceso). | [GPSGetPosition](../WDLang3/1000019205.md), [GPSFollowMovement](../WDLang3/1000019206.md) |
| **gpsErrorRights** | El navegador no tiene suficientes derechos para utilizar la geolocalización. | [GPSGetPosition](../WDLang3/1000019205.md), [GPSFollowMovement](../WDLang3/1000019206.md) |
| **gpsErrorTimeout** | La ubicación no pudo ser recuperada dentro del tiempo de espera. | [GPSGetPosition](../WDLang3/1000019205.md), [GPSFollowMovement](../WDLang3/1000019206.md) |
| **gpsNetwork** | Calcula la posición utilizando la red de antenas móviles o Wi-Fi. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsOffService** | El proveedor de localización está fuera de servicio. | [GPSStatus](../WDLang3/1000019207.md) |
| **gpsPrecision** | Solicita información sobre la precisión de las mediciones del proveedor de la localización. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsPrecisionHigh** | Calcula la posición con gran precisión. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsPrecisionLow** | Calcula la posición con una baja precisión. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsSatellite** | Calcula la posición usando el GPS. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsSpeed** | El proveedor de localización proporciona información sobre la velocidad. | [GPSInitParameter](../WDLang3/1000019204.md), [GPSInfo](../WDLang3/1000019208.md) |
| **gpsType** | Solicita información sobre el tipo de proveedor de localización. | [GPSInfo](../WDLang3/1000019208.md) |
| **gpsUnavailable** | El proveedor de localización ha dejado de estar disponible temporalmente. | [GPSStatus](../WDLang3/1000019207.md) |

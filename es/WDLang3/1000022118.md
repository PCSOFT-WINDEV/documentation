
## awProcedureGetAndSendInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica el lenguaje WL de Procedure que se llamará al recibir la información procedente de [awSendAndGetInfo](../WDLang3/1000022116.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
INTERNAL PROCEDURE GetData(b is Buffer) : Buffer
	Info(b)
	RESULT "return"
END 

awProcedureGetAndSendInfo(GetData)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`awProcedureGetAndSendInfo(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del WLanguage Procedure ("callback") llamado al recibir información de [awSendAndGetInfo](../WDLang3/1000022116.md).
Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure Name>(<Information>): Buffer
...
RESULT <Answer>
```

donde &lt;Información&gt; y &lt;Respuesta&gt; son variables del Buffer.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Consejo**: Llama a esta función en el código de inicialización del proyecto para que el Procedure sea accesible tan pronto como se inicie la aplicación.

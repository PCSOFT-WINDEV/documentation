| Constante | Detalles | Utilizado por las funciones |
| --- | --- | --- |
| **zipAdministrator** | Obliga al ejecutable creado a ejecutarse como administrador. Este modo se utiliza para evitar la virtualización de archivos desde Windows Vista. | [zipCreateExe](../WDLang3/3082013.md) |
| **zipAnsi** | Se utiliza para forzar la copia de seguridad de los nombres de archivos en formato Ansi en el archivo Archive.<br><br>Esta constante es utilizada por Default en una aplicación que utiliza el modo Ansi. | [zipCreate](../WDLang3/3082003.md) |
| **zipCancel** | Cancela la extracción de archivos del archivo comprimido. | [zipExtractFileList](../WDLang3/1000017234.md), [zipExtractAll](../WDLang3/3082011.md) |
| **zipCreateIfNotFound** | Crea el archivo si no existe. | [zipOpen](../WDLang3/3082044.md) |
| **zipCurrentDirectory** | Todos los archivos de almacenamiento se extraerán al directorio actual. | [zipCreateExe](../WDLang3/3082013.md) |
| **zipDefaultDir** | Utiliza el directorio Default (&lt;directorio de extracción Default). | [zipCreateExe](../WDLang3/3082013.md) |
| **zipDirectory** | Almacena los diferentes directorios que se encuentran en el archivo ruta, así como el nombre y la extensión del archivo.. Por ejemplo: \\Directory\\Files\\NombreArchivo.pdf<br>Devuelve los diferentes directorios del archivo (el nombre y la extensión del archivo no se devuelven). | [zipAddFile](../WDLang3/3082008.md)<br>[zipAddDirectory](../WDLang3/3082014.md)<br>[zipExtractPath](../WDLang3/3082023.md)<br>[zipExtractFile](../WDLang3/3082019.md)<br>[zipExtractAll](../WDLang3/3082011.md) |
| **zipDrive** | Almacenar la ruta completa y absoluta del archivo (nombre del disco, nombre de los directorios, nombre del archivo y extensión del archivo). Por ejemplo: C:\\Directory\\Files\\NombreArchivo.pdf <br>Esta constante no tiene efecto en el formato ZIP.<br>Devuelve sólo el nombre de la unidad. | [zipAddFile](../WDLang3/3082008.md)<br>[zipAddDirectory](../WDLang3/3082014.md)<br>[zipExtractPath](../WDLang3/3082023.md)<br>[zipExtractFile](../WDLang3/3082019.md)<br>[zipExtractAll](../WDLang3/3082011.md) |
| **zipExtension** | Devuelve la extensión del archivo (precedido por "."). | [zipExtractPath](../WDLang3/3082023.md) |
| **zipFilename** | Devuelve el nombre del archivo (sin su extensión). | [zipExtractPath](../WDLang3/3082023.md) |
| **zipFormat7Z** | Archivo de archivos en formato 7z. | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatCAB** | Archivo de archivos en formato CAB. | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatNone** | No se crea ningún fichero de archivo. | Variable de tipo [hBackupDescription](../WDLang4/1000017455.md) |
| **zipFormatRAR** | Fichero de archivo en formato RAR. | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatTAR** | Archivo de archivos en formato TAR (formato sin comprimir). | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatTARGZ** | Archivo en formato TARGZ. | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatWDZ** | Archivo en formato WDZ. | Variable de tipo [zipArchive](../WDLang3/1000018679.md) |
| **zipFormatZIP** | Archivo de archivos en formato ZIP. | Variable de tipo [zipArchive](../WDLang3/1000018679.md)<br>Variable de tipo [hBackupDescription](../WDLang4/1000017455.md) |
| **zipHeader64** | Permite crear un archivo de más de 4 GB en el que se puede utilizar [zipChangePath](../WDLang3/3082027.md). | [zipCreate](../WDLang3/3082003.md), <br>[zipOpen](../WDLang3/3082044.md), <br>[zipOpenCAB](../WDLang3/1000017250.md), <br>[zipOpenRAR](../WDLang3/1000017294.md) |
| **zipIgnore** | Ignora la extracción del archivo por error. | [zipExtractFileList](../WDLang3/1000017234.md), [zipExtractAll](../WDLang3/3082011.md) |
| **zipInMemory** | Permite gestionar los archivos almacenados en memoria (añadir un archivo encontrado en un búfer o extraer un archivo de un búfer). | [zipAddFile](../WDLang3/3082008.md), [zipExtractFile](../WDLang3/3082019.md) |
| **zipNone** | Almacena el nombre y la extensión del archivo. Por ejemplo: NombreDeArchivo.pdf<br>Extracción en el directorio actual sin restaurar la estructura de árbol del archivo (si estaba almacenado). | [zipAddFile](../WDLang3/3082008.md)<br>[zipExtractFile](../WDLang3/3082019.md) |
| **zipOverwrite** | Durante la extracción, los archivos de sólo lectura existentes se sobrescriben automáticamente con los archivos extraídos. | [zipExtractAll](../WDLang3/3082011.md) |
| **zipRead** | Apertura en modo de sólo lectura | [zipOpen](../WDLang3/3082044.md) |
| **zipRelative** | La ruta pasada como parámetro se considera como la raíz del archivo. El ruta almacenado para todos los archivos y subdirectorios se da en relación con esta raíz. | [zipAddDirectory](../WDLang3/3082014.md) |
| **zipRetry** | Vuelve a intentar la extracción de archivos. | [zipExtractFileList](../WDLang3/1000017234.md), [zipExtractAll](../WDLang3/3082011.md) |
| **zipSelectRestorePath** | La autoextracción mostrará la opción "Mantener la estructura de árbol" en el cuadro de diálogo. | [zipCreateExe](../WDLang3/3082013.md) |
| **zipTempDirectory** | Los archivos serán extraídos del directorio temporal de el equipo donde se ejecuta el ejecutable.. Todos los archivos se extraen en la misma ubicación. | [zipCreateExe](../WDLang3/3082013.md) |
| **zipTree** | Los archivos de almacenamiento se extraerán de acuerdo con la estructura de árbol de los directorios que se encuentran en el almacenamiento. | [zipCreateExe](../WDLang3/3082013.md) |
| **zipUncompressed** | El contenido del archivo no se comprime.<br>Este parámetro puede utilizarse con los archivos CAB para utilizar la función de archivo de este formato. | [zipCreate](../WDLang3/3082003.md) |
| **zipUnicode** | Se utiliza para forzar la copia de seguridad de los nombres de archivo en formato Unicode en el archivo Archive.<br><br>Esta constante es utilizada por Default en una aplicación que utiliza el modo Unicode. | [zipCreate](../WDLang3/3082003.md) |
| **zipWithUI** | El programa autoextraíble mostrará un cuadro de diálogo que permite al usuario elegir el directorio de destino.<br><br>Observación: Esta constante corresponde a *zipWithGUI* en las versiones anteriores | [zipCreateExe](../WDLang3/3082013.md) |
| **zipWrite** | Apertura en lectura/escritura. | [zipOpen](../WDLang3/3082044.md) |

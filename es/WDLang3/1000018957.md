
## emailIMAPSession (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **emailIMAPSession** se utiliza para acceder a un servidor de mensajería utilizando el protocolo IMAP (INTERNET Access Message Protocol) con el fin de recuperar los mensajes que se encuentran en este servidor. Las características de la conexión pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the session
MySession is emailIMAPSession

MySession.ServerAddress = "imap.mycompany.us"
MySession.Name = "user"
MySession.Password = EDT_Password

EmailStartSession(MySession)

FOR EACH EmailMessage NOT READ OF MySession
	Trace(EmailMessage.Subject)
END
```

## Observaciones


### Propiedades específicas de las variables emailIMAPSession
<a name="propiedades_especificas_las_variables_emailimapsession_ELTPARAGRAPHE000054"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **emailIMAPSession**:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| AuthToken | Variable de tipo [AuthToken](../WDLang3/1000022220.md) | Acceso al servidor Email Token que se utilizará si se utiliza el protocolo OAuth 2.0 (autenticación de dos factores). Este acceso Token es recuperado por [AuthIdentify](../WDLang3/1000022219.md). |
| Contraseña | Cadena de caracteres | Contraseña de usuario. Esta contraseña es proporcionada por el proveedor de servicios o por el administrador de la red. |
| denominar | Cadena de caracteres | Identifica al usuario. Este nombre es proporcionado por el proveedor de servicios o por el administrador de la red. |
| Opción | Constante de tipo Integer | Los valores posibles son:<br><br>- *emailOptionDefault* (valor de Default): Conexión mediante el protocolo IMAP.<br><br>- *optionSSL*: Conexión utilizando el protocolo IMAP dentro de un canal SSL. Esta opción garantiza la confidencialidad del intercambio pero debe ser soportada por el servidor..  <br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este Property no se apoya. |
| Puerto | Integro | Identifica el puerto utilizado para el protocolo IMAP.<br>Este Property es set a 143 por Default (número de puerto estándar del protocolo IMAP). |
| ServerAddress | Cadena de caracteres | Address del servidor Email. Este Address es suministrado por el proveedor de servicios o por el administrador de la red. Esta Address se puede dar en el siguiente formato:<br><br>- IP Address (en formato XXX.XXX.XXX.XXX, 125.5.110.100 por ejemplo).<br><br>- Nombre del servidor (por ejemplo, imap.free.fr). Se recomienda esta sintaxis.<br><br><br> |

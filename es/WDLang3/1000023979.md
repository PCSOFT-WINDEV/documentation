
## NotifListCategory( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las categorías de Notification de la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display in a List Box control 
// the list of notification categories of the application
arrCategory is array of notificationCategory = NotifListCategory()
FOR EACH Category OF arrCategory
	ListAdd(LIST_Category, Category.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NotifListCategory()`
---

**`<Result>: (Array de notificationCategory)`**

Array de variables [NotificationCategory](../WDLang3/1000023981.md) correspondientes a la lista de categorías Notification de la aplicación. 
Observación: Estas variables tienen en cuenta los cambios realizados por el usuario en la pantalla de configuración de la aplicación Notification.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para eliminar una categoría, utilice [NotifDeleteCategory](../WDLang3/1000023980.md).

- **NotifListCategory** sólo está disponible para los dispositivos que funcionan con Android 8.0 o posterior (nivel 26 de API). 
	Si se utiliza la función con una versión anterior del sistema, se produce un error fatal. 
	Para determinar la versión de Android en la que se está ejecutando la aplicación, utilice la función [SysAndroidVersion](../WDLang1/1000019785.md).


## NotesDeleteItem( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra el elemento especificado de la actual Document en Lotus Notes.

Para tener en cuenta esta supresión, el Document debe ser guardado por el [NotasGuardar](../WDLang3/1000005110.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start a Lotus Notes session
SessionID is int
SessionID = EmailStartNotesSession("Password", "MarsServer", ...
		"mail\julia.nsf", "C:\Users\Julia\AppData\Local\Lotus\Notes\Data\user.id")
...
// Read the first document 
NotesDocumentFirst(SessionID)
// Delete the document subject
NotesDeleteItem(SessionID, "Subject")
// Save the deletion
NotesSave(SessionID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NotesDeleteItem(<Session identifier> , <Lotus Notes item>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el elemento Lotus Notes,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md) o [NotasOpenDatabase](../WDLang3/1000005101.md).

**`<Lotus Notes item>: (Cadena de caracteres)`**

Nombre del elemento de Lotus Notes que desea eliminar. Se produce un error de WLanguage si no existe este elemento de Lotus Notes.

Para conocer el nombre y el tipo de los elementos de las notas actuales Document, utilice [NotesListItem](../WDLang3/1000005112.md).

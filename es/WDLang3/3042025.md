


## grExist (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un gráfico en la memoria. 

El gráfico utilizado puede ser:

- un gráfico creado mediante programación ([grCreate](../WDLang3/3042005.md)).

- un control Gráfico en el editor de ventanas, editor de páginas o editor de reportes.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks the existence of "MyChart"
ResExist = grExist("MyChart")
```
<a name="Example2"></a>

```wl
// Checks the existence of CHART_MyChart
ResExist = grExist(CHART_MyChart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = grExist(<Chart name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el gráfico,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




## &lt;SaveBMP (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un gráfico que fue dibujado previamente en un archivo en formato BMP.. 



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Saves the chart named CHART_MyChart in the "C:\Charts\Chart.BMP" file
CHART_MyChart.SaveBMP("C:\Charts\Chart.BMP")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Chart control>.SaveBMP(<Name of BMP file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el gráfico,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).

**`<Name of BMP file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo BMP a crear. La extensión del archivo es obligatoria. Este archivo BMP se crea al llamar a la función **&lt;SaveBMP**. El archivo se sobrescribe y se vuelve a crear si ya existía..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Guardar un gráfico
<a name="guardar_grafico_ELTPARAGRAPHE000232"></a>

- Para guardar un gráfico, éste debe haber sido dibujado previamente ([&lt;Dibujo](../WDLang3/1000024062.md)). Las modificaciones realizadas desde la última llamada a [&lt;Dibujo](../WDLang3/1000024062.md) serán ignoradas.

- El tamaño por defecto de la imagen es set a 639 x 479. Estas dimensiones se tienen en cuenta si la carta nunca fue dibujada o si no es Visible.



<a name="NOTE0_2"></a>


### Copia de seguridad en formato BMP
<a name="copia_seguridad_formato_bmp_ELTPARAGRAPHE000253"></a>

- Este archivo puede ser leído por una herramienta que reconoce el formato BMP.

- El tamaño del archivo es idéntico al tamaño del destino (tamaño de la ventana, control o archivo).

- Le recomendamos que previsualice el gráfico antes de guardarlo en un archivo.



<a name="NOTE0_3"></a>


### Diferencias entre grDestinationBMP y &lt;SaveBMP
<a name="diferencias_entre_grdestinationbmp_docparampagetitleshort_ELTPARAGRAPHE000265"></a>

- [grDestinationBMP](../WDLang3/3042018.md) se utiliza para definir el destino del siguiente dibujo del gráfico. Cuando se utiliza [&lt;Dibujo](../WDLang3/1000024062.md), el gráfico se dibujará en el archivo especificado y se tendrán en cuenta todos los cambios realizados desde la última vez que se dibujó el gráfico.

- La función **&lt;SaveBMP** permite guardar el gráfico directamente en un archivo en formato BMP. La Image guardada corresponde al gráfico durante la última llamada a [&lt;Dibujo](../WDLang3/1000024062.md). Los cambios realizados en el gráfico desde la última vez que se llamó a [&lt;Dibujo](../WDLang3/1000024062.md) serán ignorados.

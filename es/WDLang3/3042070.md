


## grInfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre las series encontradas en un punto específico de la carta.. Esta información es idéntica a la que se muestra en el tooltip.

**Observación**: Esta función no está disponible para las cartas 3D ni para las cartas de superficie.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Code of click on a CHART_MyChart chart
Information is string = ...
	"(" + MouseXPos() + "," + MouseYPos() + ")" + CR + ...
	"grSeries -->" + TAB + grInfoXY(Myself, grSeries, MouseXPos(), MouseYPos()) + CR + ...
	"grCategory -->" + TAB + grInfoXY(Myself, grCategory, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grValue -->" + TAB + grInfoXY(Myself, grValue, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grXValue -->" + TAB + grInfoXY(Myself, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grMinValue -->" + TAB + grInfoXY(Myself, grMinValue, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grYValue -->" + TAB + grInfoXY(Myself, grYValue, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grMaxValue -->" + TAB + grInfoXY(Myself, grMaxValue, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grStartValue -->" + TAB + grInfoXY(Myself, grStartValue, ...
				MouseXPos(), MouseYPos()) + CR + ...
	"grEndValue -->" + TAB + grInfoXY(Myself, grEndValue, MouseXPos(), MouseYPos())
```

## Sintaxis

`<Result> = grInfoXY(<Chart name> , <Type of information> [, <X> , <Y>])`
---

**`<Result>: (Entero o real)`**



- Información solicitada.

- Si no se encuentra ningún elemento del gráfico debajo del ratón,

	- -1 (constantes *grSeries* y *grCategory*)

	- 0 (*grValueXXXXXX* constante).







**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en el editor de ventanas o páginas).




**`<Type of information>: (Constante)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *grCategory* | Número de categoría puntiaguda. |
| *grTheoreticalCategory* | Número de la categoría que corresponde a una coordenada X señalada (independientemente de la serie) e incluso si el cursor no está en el dibujo del gráfico. <br><br> |
| *grPath* | Trayectoria de la sección en la Carta de Estallido Solar. <br><br> |
| *grXCoord* | Coordenada X del punto del gráfico.<br><br> |
| *grYCoord* | Coordenada Y del punto del gráfico. <br><br> |
| *grSecondaryYCoord* | Coordenada Y según el eje secundario del punto del gráfico. <br><br> |
| *grSeries* | Número de serie.<br>**Observación**: Hay una serie en 4 o en 2 para los gráficos de Stock y para los gráficos Scatter. |
| *grValue* | Valor de la serie. |
| *grStartValue* | Valor inicial de la serie para una Gráfico de cotizaciones.<br><br> |
| *grEndValue* | Valor final de la serie para una Gráfico de cotizaciones.<br><br> |
| *grMaxValue* | Valor máximo de la serie para una Gráfico de cotizaciones.<br><br> |
| *grMinValue* | Valor mínimo de la serie para una Gráfico de cotizaciones.<br><br> |
| *grXValue* | Valor X de la serie para un gráfico de dispersión.<br><br> |
| *grYValue* | Valor Y de la serie para un gráfico de dispersión.<br> |



**`<X>: (Entero opcional)`**

Coordenada X (en píxeles) a analizar. Esta coordenada se expresa en relación con la control. 
Si no se especifica este parámetro, se tiene en cuenta la posición utilizada en la última llamada a **grInfoXY**.

**`<Y>: (Entero opcional)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada se expresa en relación con la control. 
Si no se especifica este parámetro, se tiene en cuenta la posición utilizada en la última llamada a **grInfoXY**.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Posiciones X e Y
<a name="posiciones_ELTPARAGRAPHE000282"></a>

Los parámetros &lt;X&gt; y &lt;Y&gt; deben especificarse al menos una vez.

Si el gráfico cambió entre 2 llamadas a la función **grInfoXY** y si no se especifican las posiciones X e Y, se produce un error WLanguage.
<a name="NOTE0_2"></a>


### Gráfico de columnas
<a name="grafico_columnas_ELTPARAGRAPHE000295"></a>

Al hacer clic en una barra del gráfico de columnas, el área que se tendrá en cuenta será un poco más grande (en la parte superior e inferior).




## WLanguage Procedure llamado por GPSDetectPosition



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
WLanguage Procedure ("callback") llamado por [GPSDetectPosition](../WDLang3/1000019209.md) cuando el dispositivo entra o sale de la región definida por los parámetros especificados.. Este procedimiento puede ser local, global o interno.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable the notification request
GPSDetectPosition(DetectionProc, 1.145258, 30.968745)
```

```wl
// Procedure called during the notification
PROCEDURE DetectionProc(Enter is boolean)
IF Enter = True THEN
	Info("You have reached your destination.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GPSDetectPosition_Callback(<Entry> , <Detection identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para seguir detectando la posición especificada por [GPSDetectPosition](../WDLang3/1000019209.md),

- <u><u><u><u>False</u></u></u></u> para dejar de detectar la posición especificada por [GPSDetectPosition](../WDLang3/1000019209.md).




**`<Entry>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el dispositivo entra en la región definida por los parámetros especificados por [GPSDetectPosition](../WDLang3/1000019209.md),

- <u><u><u><u>False</u></u></u></u> si sale de la región especificada.




**`<Detection identifier>: (Integro)`**

Valor devuelto por [GPSDetectPosition](../WDLang3/1000019209.md). Cuando se utiliza la misma Procedure para detectar varias posiciones, este identificador diferencia las posiciones entre sí.


## Aplicación Cliente: Ejecución de procedimientos en un servidor de servicios J2EE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
WINDEV incluye varias funciones J2EE. Estas funciones le permiten gestionar la ejecución de procedimientos en un servidor de servicios Web J2EE XML desde sus aplicaciones WINDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000141"></a>
Los parámetros necesarios para ejecutar un Procedure se especifican en la documentación sobre el servidor de los servicios web J2EE XML.

Para ejecutar un Procedure en un servidor de servicios web J2EE XML:

1. Inicializar la estructura de parámetros para la ejecución de Procedure ([Estructura J2EE](../WDLang3/3047006.md)).
	Observación: Para realizar comprobaciones adicionales en el servidor de los servicios web J2EE XML, añada una cabecera Procedure ([J2EEAddHeader](../WDLang3/3047001.md)).

2. Ejecute el programa Procedure ([J2EERun](../WDLang3/3047007.md) o [J2EERunXML](../WDLang3/3047008.md)). Las siguientes operaciones se realizan automáticamente:

	- conectando el ordenador actual y el servidor de los servicios web J2EE XML.

	- enviando los parámetros de Procedure al servidor de los servicios web J2EE XML.

	- comprobando la cabecera si es necesario.

	- ...dirigiendo el Procedure.

	- devolviendo el resultado o el error de Procedure al ordenador actual.




3. Revisa el resultado de Procedure.




El resultado de [J2EERun](../WDLang3/3047007.md) o [J2EERunXML](../WDLang3/3047008.md) se utiliza para saber si la conexión se ha establecido con éxito.

Si la conexión no se ha establecido, utilice [ErrorInfo](../WDLang1/3013008.md) para averiguar la causa del error. 
Si se ha establecido la conexión, compruebe el resultado de [J2EEGetResultado](../WDLang3/3047003.md):

- si el resultado no es una cadena vacía (""), la Procedure se ha ejecutado correctamente.

- si el resultado es una cadena vacía (""), la Procedure no se ha ejecutado y/o se ha devuelto un error. Para conocer el error de Procedure, utilice [J2EEError](../WDLang3/3047010.md).

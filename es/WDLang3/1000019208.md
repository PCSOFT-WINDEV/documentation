
## GPSInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el proveedor de ubicación utilizado por la aplicación para las funciones de geolocalización.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Check the type of provider used
IF GPSInfo(gpsType) = gpsSatellite THEN
	Info("The phone geolocation uses the GPS system.")
END
```



```wl
IF GPSInfo(gpsSpeed) = False THEN
	Error("The speed regarding the move of the phone cannot be defined.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GPSInfo(<Information>)`
---

**`<Result>: (Variant)`**

Información solicitada según el parámetro &lt;Información>.

**`<Information>: (Constante de tipo Integer)`**

Tipo de información solicitada.


|   |   |   |
| --- | --- | --- |
| *gpsAltitude* | Altitud del dispositivo disponible. | &lt;El resultado&gt; corresponderá a:<br><br>- <u><u><u><u>True</u></u></u></u> si el proveedor de ubicación permite conocer la altitud del dispositivo,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario o si no se encontró ningún proveedor de ubicación.<br><br><br> |
| *gpsDirection* | Dirección del movimiento realizado por el dispositivo. | &lt;El resultado&gt; corresponderá a:<br><br>- <u><u><u><u>True</u></u></u></u> si el proveedor de ubicación permite conocer la dirección en la que se desplaza el dispositivo,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario o si no se encontró ningún proveedor de ubicación.<br><br><br> |
| *gpsEnergy* | Consumo de energía requerido por el proveedor. | &lt;El resultado&gt; corresponderá a:<br><br>- *gpsEnergyLow*: El proveedor debe requerir una cantidad baja de energía.<br><br>- *gpsEnergyMedium*: El proveedor puede requerir una cantidad media de energía.<br><br>- *gpsEnergyHigh*: El proveedor puede requerir una gran cantidad de energía.<br><br><br> |
| *gpsPrecision* | Precisión de la medición de la posición realizada por el proveedor. | &lt;El resultado&gt; corresponderá a:<br><br>- *gpsPrecisionHigh*: la posición se calcula con gran precisión.<br><br>- *gpsPrecisionLow*: la posición se calcula con poca precisión.<br><br><br> |
| *gpsSpeed* | Velocidad del movimiento realizado por el dispositivo. | &lt;El resultado&gt; corresponderá a:<br><br>- <u><u><u><u>True</u></u></u></u> si el proveedor de ubicación permite conocer la velocidad con la que se desplaza el dispositivo,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario o si no se encontró ningún proveedor de ubicación.<br><br><br> |
| *gpsType* | Tipo de ubicación del proveedor. | &lt;El resultado&gt; corresponderá a:<br><br>- *gpsSatellite*: a través de la red de satélites GPS.<br><br>- *gpsNetwork*: a través de la red de antenas móviles o Wi-Fi.<br><br>- 0 si no se ha definido el tipo de proveedor de ubicación. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta constante no está disponible. |

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000139"></a>

- Se recomienda inicializar la configuración del proveedor de localización con [GPSInitParameter](../WDLang3/1000019204.md) antes de recuperar la información de ese proveedor.

- El estado de activación del proveedor de localización es devuelto por [GPSStatus](../WDLang3/1000019207.md).



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000353"></a>

Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.

Característica necesaria: Geolocalización

Esta característica permite a las aplicaciones acceder a las características de geolocalización que se encuentran en el dispositivo o en el equipo.

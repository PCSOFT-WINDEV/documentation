
## ContactSource (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **ContactSource** permite definir las características avanzadas de una fuente (o cuenta) de contactos (ejemplo: Google, Exchange, Facebook).

Las características de esta fuente pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Lists the Google accounts of the phone
arrSource is array of ContactSource = ContactListSource()
IF NOT ErrorOccurred THEN
	FOR EACH ELEMENT source of arrSource
		IF arrSource.Type = "com.google" THEN
			// Process 	
		END
	END
END
```

## Observaciones


### Propiedades específicas de la descripción de las variables ContactSource
<a name="propiedades_especificas_descripcion_las_variables_contactsource_ELTPARAGRAPHE000045"></a>

Las siguientes propiedades pueden ser utilizadas para manipular la fuente de los contactos:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre de la fuente o cuenta. Este Property puede corresponder a un correo electrónico Address para una cuenta de Google ("lastname.firstname@gmail.com" por ejemplo). |
| escribir a máquina | Cadena de caracteres | Tipo de fuente o cuenta ("com.google" para una cuenta de Google). |


<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000078"></a>

- Para listar las fuentes de contactos en el dispositivo, utilice [ContactListSource](../WDLang3/1000020214.md).

- Puede utilizar [VariableReset](../WDLang1/3013069.md) para restablecer el contenido de un **ContactSource** Variable.

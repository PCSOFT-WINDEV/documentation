
## Funciones de descarga de archivos
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para descargar archivos de una aplicación Android: 



|   |   |
| --- | --- |
| [DownloadAdd](../WDLang3/1410087319.md) | Añade una nueva descarga en el gestor de descargas del dispositivo. |
| [DownloadGetInfo](../WDLang3/1410087320.md) | Recupera los detalles de una descarga presente en el gestor de descargas. |
| [DownloadList](../WDLang3/1410087322.md) | Enumera las descargas de la aplicación actual en el gestor de descargas. |
| [DownloadRemove](../WDLang3/1410087321.md) | Cancela una descarga eliminándola del gestor de descargas, y borra los archivos existentes. |





Estas funciones utilizan los siguientes tipos de variables:   



|   |   |
| --- | --- |
| [DownloadInfo](../WDLang3/1410087306.md) | El tipo **DownloadInfo** se utiliza para leer los diferentes detalles sobre una descarga en curso recuperados por [DownloadGetInfo](../WDLang3/1410087320.md). |
| [DownloadParameter](../WDLang3/1410087293.md) | El tipo **DownloadParameter** se utiliza para definir todas las características avanzadas de una descarga de archivos realizada por [DownloadAdd](../WDLang3/1410087319.md). |

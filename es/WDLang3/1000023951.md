


## Funciones SOAP (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar el protocolo SOAP: 



|   |   |
| --- | --- |
| [&lt;wsRequest Variable&gt;.AddSAMLAssertion](../WDLang3/1000023948.md) | Añade una aserción SAML a una solicitud de servicio web. |
| [&lt;wsRequest Variable&gt;.addXMLFirma](../WDLang3/1000023950.md) | Añade una firma XML a una solicitud de servicio web. |
| [&lt;xsdEntity Variable&gt;.AddAttribute](../WDLang3/1000023949.md) | Añade un atributo a un automatic Variable generado por la importación de un Webservice. |

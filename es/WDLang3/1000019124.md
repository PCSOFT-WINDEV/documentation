
## SaaSAdminFindUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un usuario de una cuenta de cliente SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todos los usuarios de todos los clientes. Un administrador de cuenta de cliente sólo puede buscar usuarios en su propia cuenta de cliente.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyUser is saasUser
MyUser = SaaSAdminFindUser("sam@hybrid.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminFindUser(<Login>)`
---

**`<Result>: (saasUser Variable)`**

variable [saasUser](../WDLang3/1000019083.md) que contiene el description del usuario buscado.

Si la búsqueda fracasa, la Variable devuelta está vacía.

**`<Login>: (Cadena de caracteres)`**

Login de usuario buscado.

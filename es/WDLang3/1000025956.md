
## WebSocketListConnectedUser (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista los clientes conectados al servidor WebSocket.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve list of clients
arrClients is array of websocketClient = WebSocketListConnectedClient()
// To send a message to each client
FOR EACH Client OF arrClients
	WebSocketSend(Client, bufMessage)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WebSocketListUser()`
---

**`<Result>: (Array)`**

Array de las variables de [websocketClient](../WDLang3/1000025952.md) que corresponde a los clientes conectados en el momento de llamar a la función. Los nuevos clientes que se conecten al servidor no se añadirán a la array. Los clientes que se desconecten del servidor no serán eliminados.

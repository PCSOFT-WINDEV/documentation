
## Constantes para la gestión de flujos (sonido y vídeo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones o propiedades |
| --- | --- | --- |
| **StreamAll** | Vídeo + Flujo de sonido (StreamFullDuplexSound + StreamFullDuplexVideo). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamCallerName** | Nombre del emisor de la corriente. | [StreamInfo](../WDLang3/3039009.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamCallerNumber** | Número del emisor del flujo (IP Address por ejemplo). | [StreamInfo](../WDLang3/3039009.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamFullDuplexSound** | Flujo de sonido entrante y saliente (corresponde a StreamOutgoingSound + StreamIncomingSound). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamFullDuplexVideo** | Video streaming entrante y saliente (corresponde a StreamOutgoingVideo + FluxVidéoRéception). | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamIncomingDisplay** | Vídeo visualizado en la control Conferencia: El vídeo visualizado corresponde al vídeo recibido en el ordenador actual. | Propiedad [Pantalla](../Proprietes/2510010.md) |
| **StreamIncomingSound** | Flujo de sonido entrante. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamIncomingVideo** | Transmisión de video entrante. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamNoDisplay** | Vídeo visualizado en la control Conferencia: No se muestra ningún video. | Propiedad [Pantalla](../Proprietes/2510010.md) |
| **StreamOutgoingDisplay** | Vídeo visualizado en la control Conferencia: El vídeo visualizado corresponde al vídeo enviado por el ordenador actual. | Propiedad [Pantalla](../Proprietes/2510010.md) |
| **StreamOutgoingSound** | Flujo de sonido saliente. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamOutgoingVideo** | Transmisión de video saliente. | [StreamPlay](../WDLang3/3039010.md), [StreamStop](../WDLang3/3039002.md) |
| **StreamSound** | Tipo de datos enviados por la corriente: Se emiten los datos de sonido. | Propiedad [Datos de salida](../Proprietes/2510037.md) |
| **StreamVideo** | Tipo de datos enviados por la corriente: Se emiten los datos de vídeo. | Propiedad [Datos de salida](../Proprietes/2510037.md) |

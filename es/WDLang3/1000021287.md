


## grSunburstGetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un valor numérico de una carta de Sunburst.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyVal is real = grSunburstGetData(CHART_Chart1, "Europe" + TAB + "France" + TAB + "Montpellier")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = grSunburstGetData(<Chart name> , <Section path>)`
---

**`<Result>: (Real)`**



- Valor a recuperar.

- 0 si la ruta no es válida o si la sección no contiene datos. 




**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección cuyo valor debe ser recuperado. Este parámetro tiene el siguiente formato:


```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [grStockGetData](../WDLang3/1000021174.md).

- Para recuperar los datos de un gráfico de superficie, se recomienda utilizar [grSurfaceGetData](../ErrCompil/1000001036.md).

- Para recuperar los datos de otro tipo de gráfico (circular, de columnas, Line, etc.), le recomendamos que utilice [grGetDatos](../WDLang3/1000021171.md).

- Para recuperar los datos de un gráfico Heatmap, se recomienda utilizar [grHMGetData](../WDLang3/1000021632.md).

- Por Default, un gráfico creado por [grCreate](../WDLang3/3042005.md) no contiene datos.

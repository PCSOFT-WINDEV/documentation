
## DotNetError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el error para el último .Net Procedure que falló.

**Observación**: Esta Procedure fue dirigida por [DotNetRun](../WDLang3/3031016.md) o [DotNetRunXML](../WDLang3/3031012.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// This example assumes that the eightball module is available from "www.gotdotnet.com".
// The example would no longer operate if this module was not available on the site
DotNet.value[1]= "enter a question in English" + ...
	"just for fun: what would the answer be?"
DotNet.Name[1] = "psQuestion"
DotNet.Type[1] = DotNetStringType
IF DotNetRun("www.gotdotnet.com/playground/services/EightBall/eightballws.asmx", ...
		"Ask", "http://tempuri.org/","http://tempuri.org/Ask") = True THEN
	Info(DotNetGetResult(DotNetResult))
ELSE
	IF DotNetError(DotNetErrMessage) ~= "" THEN
		Error(ErrorInfo()) 
	ELSE
		Error(".Net error: " + DotNetError(DotNetErrMessage))
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DotNetError(<Type of error>)`
---

**`<Result>: (Cadena de caracteres)`**



- Error del último .Net Procedure que falló.

- Una cadena vacía (""") si:

	- el servidor no ha devuelto ningún error,

	- no se ha establecido ninguna conexión con el servidor.







**`<Type of error>: (Constante)`**

Tipo de error a devolver:


|   |   |
| --- | --- |
| *DotNetErrActor* | URL del servidor que activó la Error( útil si la Procedure es ejecutada por varios servidores). |
| *DotNetErrCode* | Error de la última corrida de .Net Procedure:<br><br>- "VersionMismatch": La versión.net utilizada en el ordenador actual difiere de la utilizada en el servidor.<br><br>- "Debe ser de las afueras.": El servidor no entiende el parámetro especificado en el encabezado (solo si el parámetro &lt;Ejecución del procedimiento&gt; del encabezado se inicializa en <u><u><u><u>True</u></u></u></u>).<br><br>- "Cliente": Configuración inválida en relación con la ejecución de la Procedure.<br><br>- "Server": El servidor no ejecutó el Procedure.<br><br><br>**Observación**: Esta cadena de caracteres se puede rellenar con información adicional. Se utiliza el siguiente formato: "&lt;Error&gt;.&lt;Información&gt;". Por ejemplo: "Autenticación de cliente" si el error se debe a la autenticación del cliente. |
| *DotNetErrDetails* | Detalles del error. Este mensaje está en formato XML. |
| *DotNetErrMessage* | Mensaje de error devuelto por el servidor en la última ejecución de .Net Procedure. |

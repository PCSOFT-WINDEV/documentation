


## grAxisVisible (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la forma en que los ejes se muestran en los gráficos de Line o de columna. El eje puede:

- ...ser Visible,

- ...ser Invisible,

- mostrar sólo la etiqueta.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
grAxisVisible(CHART_Chart1, grXCoordinate, grInvisible)
// or for compatibility
// grAxisVisible(CHART_Chart1, grXCoordinate, False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grAxisVisible(<Chart name> , <Type of axis> , <Visible>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Type of axis>: (Constante de tipo Integer)`**

Eje a utilizar:


|   |   |
| --- | --- |
| *grXCoordinate* | Eje X. |
| *grYCoordinate* | Eje Y. |



**`<Visible>: (Constante de tipo Integer)`**

Modo de visualización de ejes: 


|   |   |
| --- | --- |
| *grInvisible* | El eje se convierte en Invisible.<br>Para la compatibilidad retroactiva, también puedes usar &lt;Visible&gt; = False. |
| *grLabelOnly* | Sólo las etiquetas del eje son Visible.<br><br> |
| *grVisible* | El eje se convierte en Visible. <br>Para la compatibilidad retroactiva, también puedes usar &lt;Visible&gt; = True. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el eje de coordenadas Y es Invisible, el eje de coordenadas Y secundario es también Invisible. 

- Los parámetros definidos por **grAxisVisible** se tendrán en cuenta en la siguiente llamada a [grDraw](../WDLang3/3042026.md).

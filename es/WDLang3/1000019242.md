
## Constantes de SaaS
			



| Constante | description | Utilizado por |
| --- | --- | --- |
| **saasAdministrator** | Nivel necesario para la conexión: Administrador | [SaaSAdminConnect](../WDLang3/1000019098.md), [SaaSAdminIsConectado](../WDLang3/1000019101.md) |
| **saasWithCache** | El valor se lee desde la caché. Si este valor no existe en la caché, se lee en el servidor SaaS y la caché se actualiza automáticamente. | [SaaSReadSiteParameter](../WDLang3/1000019037.md) |
| **saasBaseNone** | El sitio SaaS no utiliza ninguna base de datos. | [SaaSAdminConfigureSiteData](../WDLang3/1000019142.md) |
| **saasBaseClient** | Cada cliente del sitio SaaS utiliza su propia base de datos. | [SaaSAdminConfigureSiteData](../WDLang3/1000019142.md) |
| **saasBaseCommon** | La base de datos del sitio SaaS es compartida entre todos los clientes. | [SaaSAdminConfigureSiteData](../WDLang3/1000019142.md) |
| **saasUnlimitedDuration** | Especifica una suscripción SaaS sin fecha de finalización. | [saasSubscription](../WDLang3/1000019088.md) |
| **saasOutrightPricing** | Especifica un precio de paquete (sin contar las licencias). | [saasPricing](../WDLang3/1000019091.md) |
| **saasPricingPerUser** | Especifica un precio por usuario (mientras se cuentan las licencias). | [saasPricing](../WDLang3/1000019091.md) |
| **saasAccountManager** | Nivel necesario para la conexión: Gerente de cuentas senior. | [SaaSAdminConnect](../WDLang3/1000019098.md), [SaaSAdminIsConectado](../WDLang3/1000019101.md) |
| **saasUnlimitedLicenses** | Especifica una suscripción SaaS con precio de paquete. | [saasSubscription](../WDLang3/1000019088.md) |
| **saasUnlimitedPeriodicity** | Especifica un precio único. | [saasPricing](../WDLang3/1000019091.md) |
| **saasNoCache** | Fuerza el valor a leer en el servidor SaaS. El valor encontrado en la caché no se lee ni se actualiza. | [SaaSReadSiteParameter](../WDLang3/1000019037.md) |
| **saasReinitCache** | Reinicia toda la caché, luego el valor se lee en el servidor SaaS y la caché se actualiza automáticamente. | [SaaSReadSiteParameter](../WDLang3/1000019037.md) |


## OBEX constants
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por la función |
| --- | --- | --- |
| **obexBluetooth** | Transporte de datos realizado por Bluetooth | [OBEXConnect](../WDLang3/1000017153.md) |
| **obexInfrared** | Transporte de datos realizado por infrarrojos | [OBEXConnect](../WDLang3/1000017153.md) |
| **obexListDetailed** | Devuelve la lista detallada de archivos OBEX: <br>&lt;Nombre del archivo 1&gt; + TAB + &lt;Tipo 1&gt; + TAB + &lt;Tamaño en bytes 1&gt; + TAB + &lt;Permisos 1&gt; + TAB + &lt;Fecha de modificación 1&gt; + TAB + &lt;Fecha de creación 1&gt; + TAB + &lt;Fecha de acceso 1> + CR +... | [OBEXListFile](../WDLang3/1000017148.md) |
| **obexListDirectory** | Devuelve la lista de archivos y directorios OBEX. | [OBEXListFile](../WDLang3/1000017148.md) |
| **obexListSimple** | Devuelve la lista simple de archivos OBEX: <br>&lt;Nombre del archivo 1&gt; + TAB + &lt;Tipo 1&gt; + CR + ... &lt;Nombre del archivo N&gt; + TAB + &lt;Tipo N&gt; <br>donde &lt;Tipo&gt; corresponde a "F" si el elemento es un fichero y a "D" si el elemento es un directorio. | [OBEXListFile](../WDLang3/1000017148.md) |

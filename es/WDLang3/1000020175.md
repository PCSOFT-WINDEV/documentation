
## MobileNetworkStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado actual de la conexión a los datos de la red móvil o solicita ser notificado cuando cambia el estado de la conexión a los datos.

**Observaciones**: 

- Esta función no se puede utilizar en el simulador. En este caso, siempre devolverá 0. 

- Esta función no afecta a la conexión Wi-Fi, sino a la conexión a través de un operador telefónico.. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF MobileNetworkStatus() = networkConnected THEN
	BrowserRunApp("www.windev.com")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperación del estado de la conexión a los datos de la red móvil

`<Result> = MobileNetworkStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la conexión a los datos:


|   |   |
| --- | --- |
| *networkConnected* | Conectado. |
| *networkConnectionInProgress* | Conectando.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *networkDisconnected* | Desconectado. |




<a name="SYNTAX2"></a>

### Solicitar que se le notifique cuando cambie el estado de la conexión a los datos en la red móvil

`<Result> = MobileNetworkStatus(<WLanguage procedure>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la conexión a los datos:


|   |   |
| --- | --- |
| *networkConnected* | Conectado. |
| *networkConnectionInProgress* | Conectando.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *networkDisconnected* | Desconectado. |



**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback" Procedure) llamado cuando el estado de la conexión a los datos cambia. 

Este procedimiento tiene el siguiente formato: 


```txt
PROCEDURE <Procedure name>(<New status>)
```


&lt;Nuevo estado&gt; es una constante entera que corresponde al nuevo estado de la conexión a los datos: 


|   |   |
| --- | --- |
| *networkConnected* | Conectado. |
| *networkConnectionInProgress* | Conectando.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *networkDisconnected* | Desconectado. |

Si este parámetro corresponde a una cadena vacía ("""), ya no se enviarán las notificaciones relativas al cambio de estado de conexión.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se puede producir un cambio en el estado de conexión cuando cambia el tipo de red utilizado. : por ejemplo, al cambiar de una red 2G a una red 3G (o viceversa). 

Para conocer el tipo de red correspondiente a la conexión actual, utilice [MobileNetworkConnectionInfo](../WDLang3/1000020176.md).

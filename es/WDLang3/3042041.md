


## grScatterPointSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el tamaño de los puntos en un gráfico de "dispersión". 

**Recordatorio**: El tamaño de un punto en un gráfico de "Dispersión" es igual a 1 píxel por Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the size of points in "MyChart".
// This chart is a "Scatter" chart.
grScatterPointSize(CHART_MyChart, 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grScatterPointSize(<Chart name> , <Size>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Size>: (Integro)`**

Tamaño de un punto (en píxeles).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tamaño de un punto
<a name="tamano_punto_ELTPARAGRAPHE000114"></a>El punto mostrado es un cuadrado cuyo tamaño del lado se calcula mediante la siguiente fórmula: (&lt;Tamaño&gt; 2)-1\*píxeles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe2.gif)

<a name="NOTE0_2"></a>


### Puntos en las cartas "Scatter
<a name="puntos_las_cartas_scatter_ELTPARAGRAPHE000124"></a>

- En las cartas "Scatter", un punto se define por sus coordenadas (coordenada X y coordenada Y).

- Los datos definidos con la función **grScatterPointSize** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).

- La función **grScatterPointSize** solo se utiliza para gráficos de "dispersión". Esta función no tiene ningún efecto con las otras clases de gráficos.




## &lt;Gráfico&gt;.GetData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera un dato de un gráfico. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display a data in the "CHART_MyChart" control. 
// This data is the first one of the first series.
Info(CHART_MyChart.GetData(1, 1))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Chart control>.GetData(<Series number> , <Index>)`
---

**`<Result>: (Real)`**

Valor recuperado.

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie para la que se recuperará un valor. 
Si la serie especificada no existe, el valor recuperado es 0.

**`<Index>: (Integro)`**

Índice de los datos de la serie cuyo valor se recuperará.
Si el dato correspondiente al índice especificado no existe en la serie, el valor recuperado es 0.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para recuperar datos de un Gráfico de cotizaciones, se recomienda utilizar [.StockGetData](../WDLang3/1000023566.md).

- Para recuperar los datos de un gráfico de dispersión, se recomienda utilizar [.ScatterGetDataXY](../WDLang3/1000024108.md).

- Para recuperar los datos de un gráfico de superficie, se recomienda utilizar [.SurfaceGetData](../WDLang3/1000024148.md).

- Para recuperar los datos de un gráfico Heatmap, se recomienda utilizar [&lt;Gráfico&gt;.HMGetData](../WDLang3/1000024083.md).

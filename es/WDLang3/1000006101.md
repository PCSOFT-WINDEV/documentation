


## ContactAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un Contact: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un libro de Lotus Notes o Outlook Address. 
	**Atención**: Esta función ** no puede usarse** para añadir un contacto en el libro de Address de una mensajería de Outlook Express.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) en la lista de contactos del dispositivo (Android, iPhone o iPad, Universal Windows).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000020272&name=contactadd_funcion&product=WD">Ver ejemplos adicionales</a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Initialize the variables of Contact structure
ContactReset()
// Fill the variables for the new contact
mContact.Name = "Smith"
mContact.FirstName = "Adriana"
mContact.Email = "adriana.smith@mycompany.com"
// Add the contact
IF ContactAdd(SessionID) = True THEN
	Info("Contact added")
ELSE
	Error("Error during the addition", ErrorInfo())
END
```
<a name="Example2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Local variables
tContact is Contact
tPhone is Phone

// Identity
tContact.Name = EDT_LastName
tContact.FirstName = EDT_FirstName

// Home phone numbers
IF EDT_HomePhone <> "" THEN
	tPhone.PhoneNumber = EDT_HomePhone
	tPhone.Type = phoneHome
	Add(tContact.Phone, tPhone)
END

// Add
IF NOT ContactAdd(tContact, gSource) THEN
	Error("Unable to create the contact.")
ELSE
	WIN_TableContacts.RefreshContactList()
	Close()
END
```

## Sintaxis
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Añadir una Contact en la mensajería de Lotus Notes o Outlook

`<Result> = ContactAdd(<Session identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificación de la sesión que se va a utilizar. Este identificador es devuelto por [EmailStartNotesSession](../WDLang3/3032121.md), [EmailStartOutlookSession](../WDLang3/3032128.md), [NotasOpenDatabase](../WDLang3/1000005101.md) o [OutlookStartSession](../WDLang3/3032123.md).


<a name="SYNTAX2"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Añadir una Contact a la lista de Contact del dispositivo

`<Result> = ContactAdd(<Contact> [, <Source>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Contact>: (Contacte con Variable)`**

Nombre de la [Contact](../WDLang3/1000020254.md) Variable correspondiente al Contact a añadir. 

**`<Source>: (Opcional ContactSource Variable)`**

Nombre de la [ContactSource](../WDLang3/1000020211.md) Variable correspondiente a la fuente o a la cuenta en la que se añadirá el Contact. Si no se especifica este parámetro, el contacto será associated sin cuenta.

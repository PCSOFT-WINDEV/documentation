


## <Chart>.RBRotationDirection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la dirección de rotación en un gráfico de barras radiales.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CHART_Chart1.RBRotationDirection(grAnticlockwise)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.RBRotationDirection(<Rotation>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (en la ventana o editor de reportes).

**`<Rotation>: (Constante de tipo Integer)`**

Dirección de rotación en el gráfico: 


|   |   |
| --- | --- |
| *grAnticlockwise* | Los datos del gráfico de la barra radial se mostrarán en sentido contrario a las agujas del reloj. |
| *grClockwise* | Los datos de la barra radial se mostrarán en el sentido de las agujas del reloj. |

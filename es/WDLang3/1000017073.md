
## SNMPTrapDisable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva la interceptación de una trampa SNMP.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SNMPTrapDisable(nTrapID) = False THEN
	Error("Error while disabling the trap: " + ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SNMPTrapDisable(<Trap identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se desactivó la interceptación de la captura,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Trap identifier>: (Integro)`**

Identificador de la trampa a desactivar. Este identificador fue devuelto por [SNMPTrapHabilitar](../WDLang3/1000017072.md).

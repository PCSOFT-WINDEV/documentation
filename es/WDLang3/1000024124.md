


## <Chart>.Rotación (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- Girar un gráfico alrededor de uno de sus ejes para ver mejor los datos.. 

- Averiguar el ángulo de rotación de un gráfico en uno de sus ejes. 




El gráfico utilizado puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Performs a slight rotation of 35 degrees
// to highlight the raised effect 
// on a Surface chart
CHART_Surface.Rotation(grAxisX, -35)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Girar un gráfico

`<Chart control>.Rotation(<Axis> , <Angle>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes). Esta tabla puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.




**`<Axis>: (Constante de tipo Integer)`**

Eje de rotación: 


|   |   |
| --- | --- |
| *grAxisX* | Rotación en el eje X. |
| *grAxisY* | Rotación en el eje Y. |
| *grAxisZ* | Rotación en el eje Z. |



**`<Angle>: (Real)`**

Ángulo de rotación en grados.


<a name="SYNTAX2"></a>

### Averiguar el ángulo de rotación de un gráfico

`<Result> = <Chart control>.Rotation(<Axis>)`
---

**`<Result>: (Real)`**

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en el editor de ventanas o en el editor de reportes). Esta tabla puede ser: 

- una carta de Superficie,

- un gráfico de la pirámide.




**`<Axis>: (Constante de tipo Integer)`**

Eje de rotación: 


|   |   |
| --- | --- |
| *grAxisX* | Rotación en el eje X. |
| *grAxisY* | Rotación en el eje Y. |
| *grAxisZ* | Rotación en el eje Z. |

<a name="Use"></a>
<a name="description"></a>
Modifica el porcentaje correspondiente al radio del agujero en una tabla de donuts.. 


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The hole will represent 50% of the total radius of the donut
grDonutHolePercentage(CHART_MyChart, 50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grDonutHolePercentage(<Chart name> , <Percentage>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico (en la ventana, página o editor de reportes).




**`<Percentage>: (Integro)`**

Porcentaje representado por el agujero del donut (entre 0 y 70).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los ajustes definidos con **grDonutHolePorcentaje** se tendrán en cuenta la próxima vez que se llame a [grDraw](../WDLang3/3042026.md).

- Esta función sólo se puede utilizar con las cartas de donuts.

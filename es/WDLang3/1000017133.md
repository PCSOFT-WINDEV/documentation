
## BTInfoDevice( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información específica sobre un dispositivo Bluetooth.  






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sListDevices is string
sADevice is string
sId is string
sName is string
sMACAddress is string
// Clears the list
TableDeleteAll(TABLE_Devices)
// Returns the list of nearby Bluetooth devices
sListDevices = BTListDevice(btSimpleList)
// Browse the connected devices
FOR EACH STRING sADevice OF sListDevices SEPARATED BY CR
	// Stores the device information
	sId = ExtractString(sADevice, 1)
	sName = ExtractString(sADevice, 3)
	sMACAddress = ExtractString(sADevice, 2)
	// The name is explicitly requested if it is not specified
	IF sName ~= "" THEN sName = BTInfoDevice(sId, btName) 
	// Fills the Table control with the devices
	TableAddLine(TABLE_Devices, False, sId, sName, sMACAddress)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BTInfoDevice(<Device identifier> , <Requested information>)`
---

**`<Result>: (Cadena de caracteres)`**

Información solicitada sobre el dispositivo especificado.

**`<Device identifier>: (Integro)`**

Identificador del dispositivo Bluetooth a utilizar. Este identificador es devuelto por [BTListDevice](../WDLang3/1000017132.md) o [Dispositivo BTSelectDevice](../WDLang3/1000017135.md).

**`<Requested information>: (Constante de tipo Integer)`**

Información sobre el dispositivo. El tabla a continuación enumera la información disponible así como los valores que pueden ser devueltos:



|   |   |
| --- | --- |
| *btClassOfDevice* | La clase del dispositivo puede corresponder a una de las siguientes constantes:<br><br>- btCoD_RedAcceso<br><br>- btCoD_AudioVideo<br><br>- btCoD_Miscelánea<br><br>- btCoD_Imagen<br><br>- btCoD_Desconocido<br><br>- btCoD_Juguete<br><br>- btCoD_Computadora<br><br>- btCoD_Dispositivo<br><br>- btCoD_Phone<br><br>- btCoD_Llevable<br><br><br> |
| *btLastTimeSeen* | Fecha y hora en que el dispositivo fue visto por última vez por el ordenador actual. Esta fecha está en formato DDMMYYYYYHHHMMSS.<br><br> |
| *btLastTimeUsed* | Fecha y hora en que el dispositivo fue utilizado por última vez por el ordenador actual. Esta fecha está en formato DDMMYYYYYHHHMMSS.<br><br> |
| *btMACAddress* | MAC Address del dispositivo |
| *btName* | Nombre del dispositivo |
| *btStatusAuthenticated* | <br><br>- 1 si el dispositivo está actualmente autenticado junto al ordenador actual,<br><br>- 0 de lo contrario.<br><br><br> |
| *btStatusConnected* | <br><br>- 1 si el dispositivo está actualmente "conectado" al ordenador actual,<br><br>- 0 de lo contrario.<br><br><br><br><br> |
| *btStatusRemembered* | <br><br>- 1 si el dispositivo está siendo "recordado" por el dispositivo Bluetooth del ordenador actual,<br><br>- 0 de lo contrario.<br><br><br><br><br> |

### Limitación
<a name="limitacion_ELTPARAGRAPHE000128"></a>

Esta versión sólo es compatible con las pilas bluetooth de Microsoft, Bluesoleil y Broadcom (Widcomm). Para conocer la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).  

Las pilas Broadcom (Widcomm) no están disponibles en 64 bits.

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

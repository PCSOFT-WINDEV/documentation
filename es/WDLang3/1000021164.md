


## &lt;sshSession Variable&gt;.DisconnectShell (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sesión SSH que fue abierta por [&lt;sshSession Variable&gt;.ConnectShell](../WDLang3/1000021163.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMySession is sshSession
cMySession.Address = "127.0.0.1"
cMySession.Port = 22
cMySession.User = "login"
cMySession.UserPassword = "pass"
IF cMySession.ConnectShell() THEN
	Info("Session started")
	// ...
	cMySession.DisconnectShell()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<SSH session>.DisconnectShell()`
---

**`<SSH session>: (Variable de tipo sshSession)`**

Nombre de la [sshSession](../WDLang3/1000021157.md) Variable correspondiente a la sesión SSH a detener.

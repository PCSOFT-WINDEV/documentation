
## Procedimiento WLanguage llamado por la función geoGetAddress



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento WLanguage ("callback") llamado por la función [geoGetAddress](../WDLang3/1000019797.md). Este procedimiento permite obtener la lista de direcciones que corresponden a la ubicación especificada en la función [geoGetAddress](../WDLang3/1000019797.md).. Este procedimiento puede ser local, global o interno.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the address of a monument
geoGetAddress("Eiffel Tower", geoGetAddress_Callback)

INTERNAL PROCEDURE geoGetAddress_Callback(MyArray)
	FOR I = 1 TO 10
		ToastDisplay(MyArray[I]..Street + CR + MyArray[I]..City + 
		CR + MyArray[I]..Country)
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`geoGetAddress_Callback(<Array of addresses>)`
---

**`<Array of addresses>: (Array)`**

Nombre del array de variables de tipo [Address](../WDLang3/1000019944.md) que se rellenará con la lista de direcciones que corresponden a la ubicación especificada con la función [geoGetAddress](../WDLang3/1000019797.md). Las propiedades de las variables de tipo [Address](../WDLang3/1000019944.md) (calle, ciudad, etc.) se completarán según la precisión de la información devuelta por el servicio de geolocalización.

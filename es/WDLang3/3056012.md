
## Subir y bajar archivos en INTERNET (WINDEV FTP y RPC)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000106"></a>
Se pueden usar varios métodos para subir y bajar archivos por INTERNET:

- El FTP o RPC de WINDEV. Estos protocolos requieren el uso de un servidor FTP/RPC específico potenciado por WINDEV.

- El FTP estándar. Este protocolo requiere el uso de un servidor FTP estándar. Para obtener más información, consulte [Comunicación con un servidor FTP](../WDLang3/3040009.md).




Observaciones:

- RPC: Llamada remota Procedure.

- FTP: Protocolo de transferencia de archivos.

## Carga y descarga de archivos mediante WINDEV FTP o RPC: las reglas a seguir
<a name="carga_descarga_archivos_mediante_windev_ftp_rpc_las_reglas_seguir_ELTTEXTE000130"></a>
Para cargar y descargar archivos, debe cumplir con las siguientes reglas:

1. Conectarse a un servidor FTP WINDEV con [NetConnect](../WDLang3/3056004.md). Esta función establece una conexión entre WINDEV y el servidor y proporciona un identificador de conexión.

2. Cargar, descargar archivos.

3. Cierre la conexión con el servidor con [NetDisconnect](../WDLang3/3056011.md).




**Atención**:

- El protocolo de comunicación TCP/IP debe ser instalado y se debe definir un Address IP.

- Un servidor FTP o RPC WINDEV sólo funciona en modo de 32 bits.

## Otras características
<a name="otras_caracteristicas_ELTTEXTE000154"></a>
También es posible:

- Averigua el nombre y la IP Address de un ordenador.

- Ejecute un programa en un servidor FTP o RPC WINDEV.

- Activar un elemento Windows en el servidor FTP o RPC de WINDEV.

- Transmitir una cadena de caracteres a un servidor FTP o RPC WINDEV.


## SocketConnectInfrared( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta un ordenador cliente a una toma determinada a través del puerto de infrarrojos.

**Recordatorio**: Un socket es un recurso utilizado por las aplicaciones para comunicarse de un equipo a otro, independientemente del tipo de red.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SocketConnectInfrared("customer", "service") = False THEN
	Error("Connection error" + ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SocketConnectInfrared(<Socket name> , <Service name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).




**`<Socket name>: (Cadena de caracteres)`**

Nombre que se le dará a la conexión abierta en el socket del servidor. Este nombre será utilizado por todas las funciones del socket.

**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio especificado en [SocketCreateInfrarrojo](../WDLang3/3070012.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Protocolo IRDA
<a name="protocolo_irda_ELTPARAGRAPHE000051"></a>

**SocketConnectInfrared** utiliza el protocolo IRDA (Infrared Data Association). Por lo tanto, una aplicación WINDEV puede conectarse a cualquier dispositivo que utilice este protocolo (siempre que se conozca el nombre del servicio utilizado).
<a name="NOTE0_2"></a>




### Límite de ventanas
<a name="limite_ventanas_ELTPARAGRAPHE000061"></a>

El número máximo de sockets infrarrojos utilizados simultáneamente no puede superar los 80.
<a name="NOTE0_3"></a>




### Conexión a una toma de corriente determinada
<a name="conexion_una_toma_corriente_determinada_ELTPARAGRAPHE000069"></a>

Para conectar un ordenador cliente a una toma determinada, utilice [SocketConnect](../WDLang3/3070016.md).

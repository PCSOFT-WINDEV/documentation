
## SaaSAdminListSite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los sitios SaaS declarados en la base de datos de un WEBDEV SaaS Webservice.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS tendrá acceso a todos los sitios SaaS. Un gestor de cuentas de cliente sólo tendrá acceso a los sitios para los que tiene una suscripción. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize a list with the SaaS sites
lstSite is array of saasSite
lstSite = SaaSAdminListSite()

FOR EACH SaasSite OF lstSite
	ListAdd(LIST_Sites, SaasSite.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminListSite()`
---

**`<Result>: (Array de las variables de la saasSite)`**

Lista de sitios SaaS declarados en la base de datos Webservice.


## VibrationTrigger( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dispara las vibraciones del dispositivo. 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Estas vibraciones pueden ser detenidas en cualquier momento por [VibrationStop](../WDLang3/1000019449.md).  




**Atención** : 

- El dispositivo debe estar equipado con un sistema de vibración. 

- Esta función no tiene ningún efecto en el simulador o en el emulador.. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// Trigger the vibrations during one second
VibrationTrigger(1000)

// Trigger the vibrations alternatively every 300 milliseconds
arrPeriod is array of int 
arrPeriod = [300,300]
VibrationTrigger(arrPeriod, true)

// Stop the vibrations
VibrationStop()
```

## Sintaxis
<a name="SYNTAX1"></a>

### Desencadenando vibraciones durante un segundo

`VibrationTrigger()`
---


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Desencadenando las vibraciones durante un período determinado

`VibrationTrigger(<Duration>)`
---

**`<Duration>: (Entero o Duración)`**

Duración de las vibraciones en milisegundos. Este parámetro puede corresponder a : 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 





<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Desencadenando repetidos períodos de vibraciones

`VibrationTrigger(<Periods> [, <Recurrence>])`
---

**`<Periods>: (Array de números enteros)`**

Array de entero describiendo la alternancia de períodos para activar y detener las vibraciones de Phone : 

- El primer valor corresponde a la duración antes de la activación de las vibraciones.. 

- El segundo valor corresponde a la duración de las vibraciones.  

- El tercer valor corresponde a la duración antes de la siguiente activación de las vibraciones.. 


Y además.

**`<Recurrence>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si la alternancia de períodos debe repetirse indefinidamente,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para no realizar ninguna repetición.


## OBEXSendFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía un archivo a un dispositivo compatible con el protocolo OBEX.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Bluetooth connection
IF BTConnectDevice(COL_ID, "123") = True THEN
	// OBEX connection
	nIDOBEXCnt = OBEXConnect(obexBluetooth, COL_MAC)
	// Transfer the file
	TABLE_DEVICES[TABLE_DEVICES].BackgroundColor = ...
				  SWITCH (OBEXSendFile(nIDOBEXCnt, sFile), ...
						White, PastelRed)	
	// End of transfer
	OBEXDisconnect(nIDOBEXCnt)
ELSE
	 Error("Unable to connect to " + COL_DEVICE, ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OBEXSendFile(<Identifier of OBEX connection> , <Local file path> [, <Remote file path>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la transferencia correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**Atención**: No se notifica al remitente si el equipo remoto rechaza el archivo.: [ErrorInfo](../WDLang1/3013008.md) no contiene ningún código de error específico. [ErrorInfo](../WDLang1/3013008.md) sólo contiene los mensajes de error de transmisión.

**`<Identifier of OBEX connection>: (Integro)`**

Identificador de la conexión OBEX a utilizar. Este identificador es devuelto por [OBEXConnect](../WDLang3/1000017153.md).

**`<Local file path>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo a enviar.

**`<Remote file path>: (Cadena de caracteres opcional)`**

Nombre y ruta del archivo en el dispositivo remoto. Si no se especifica este parámetro, el archivo conserva el mismo nombre y se copia en el directorio de transferencia configurado en el dispositivo OBEX..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Ubicación del archivo copiado
<a name="ubicacion_del_archivo_copiado_ELTPARAGRAPHE000058"></a>

El archivo se copia en el directorio de transferencia configurado en el dispositivo OBEX utilizado. Si se especifica &lt;Remote File Path&gt;, el ruta especificado será relativo al directorio de transferencia.
<a name="NOTE0_2"></a>




### Transferencia de archivos puertos de infrarrojos
<a name="transferencia_archivos_puertos_infrarrojos_ELTPARAGRAPHE000066"></a>

Cuando se transfieren archivos a través de los puertos de infrarrojos, la transferencia se completa una vez que se ha ejecutado **OBEXSendFile**. Sin embargo, Windows mantiene abierta la ventana de transferencia (para administrar una posible nueva transferencia de archivos).. Esta ventana sólo se cierra al desconectar ([OBEXDesconectar](../WDLang3/1000017154.md)).  
<a name="NOTE0_3"></a>




### Limitación
<a name="limitacion_ELTPARAGRAPHE000079"></a>

Sólo las pilas de Microsoft bluetooth son soportadas por esta función. Para get la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).

Para obtener más información, consulte [¿Qué pilas utilizar?](../WDLang3/1000017231.md)

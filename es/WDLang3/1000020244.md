
## EmailAddress (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **EmailAddress** permite manipular una dirección de correo electrónico y su tipo (casa, oficina, etc.).

Las características de este correo electrónico Address puede ser definido y modificado por varias propiedades de WLanguage. 

Este tipo de Variable es usado por los [Contact](../WDLang3/1000020254.md) Variable s.

**Observación**: Para obtener más información sobre la declaración de este tipo de variables y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the first professional email address of a contact
sAddress is string
FOR EACH Email of MyContact..Email
	SI Email..Type = emailWork THEN
		sAddress = email..Address
		RETURN
	END
END
```

```wl
// Displays the contacts and their different emails
garrContacts is array of Contact
gCurrentContact is Contact

garrContacts = ContactList(Null)
Info("Number of contacts: " + garrContacts..Count)

FOR EACH gCurrentContact OF garrContacts	
	Info(gCurrentContact..DisplayedName)
	FOR EACH Email of gCurrentContact..Email
		Info(Email..Address)	
	END
END
```

## Observaciones


### Propiedades específicas de la descripción de las variables EmailAddress
<a name="propiedades_especificas_descripcion_las_variables_emailaddress_ELTPARAGRAPHE000051"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un correo electrónico Address:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Address | Cadena de caracteres | Correo electrónico. |
| Etiqueta | Cadena de caracteres | Etiqueta para los tipos personalizados de direcciones de Email (opcional).<br>Si **..Type** no corresponde a la constante *emailCustom*, el valor de este Property será ignorado. |
| escribir a máquina | Constante de tipo Integer | Tipo de correo electrónico Address: <br><br>- *emailOther*: otros tipos.<br><br>- *emailWork*: correo electrónico profesional Address.<br><br>- *emailHome*: correo electrónico personal Address.<br><br>- *emailCell*: dirección de correo electrónico de Phone.<br><br>- *emailCustom*: tipo personalizado que corresponde al valor de la propiedad **..Label**.<br><br><br> |


**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;..&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;

### Reinicio
<a name="reinicio_ELTPARAGRAPHE000101"></a>

Para reiniciar el contenido de un **EmailAddress** Variable, tienes la posibilidad de usar [VariableReset](../WDLang1/3013069.md).

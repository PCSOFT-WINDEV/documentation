
## MQTT constants
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **mqttQualityAtLeastOnce** | El mensaje se entregará al menos una vez. El cliente transmitirá el mensaje varias veces si es necesario hasta que el servidor (broker) confirme que el mensaje fue transmitido en la red. | [MQTTSubscribe](../WDLang3/1000023073.md), [MQTTPublish](../WDLang3/1000023060.md) |
| **mqttQualityAtMostOnce** | El mensaje será entregado una vez como máximo. El mensaje no se almacena. Se envía sin garantía de recepción. El MQTTT (broker) no informa al remitente de que el mensaje ha sido recibido.. El mensaje puede perderse si el cliente se desconecta o si el servidor se detiene. | [MQTTSubscribe](../WDLang3/1000023073.md), [MQTTPublish](../WDLang3/1000023060.md) |
| **mqttQualityExactlyOnce** | El mensaje será necesariamente guardado por el emisor. Se transmitirá mientras el receptor no confirme su envío a la red.. El emisor está utilizando un sofisticado reconocimiento con el servidor (broker) para evitar la duplicación de mensajes.. Este modo es más lento pero es más secure. | [MQTTSubscribe](../WDLang3/1000023073.md), [MQTTPublish](../WDLang3/1000023060.md) |


## DomoStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si un dispositivo eléctrico (compatible con el protocolo X10) está encendido o apagado.

Si el dispositivo especificado no está conectado a la corriente, se devuelve el estado almacenado en la interfaz X10.

**Observación**: Para que el protocolo X10 funcione correctamente a través de la portadora de potencia-Line, se requiere una potencia continua entre los dispositivos. Por ejemplo, la transmisión no puede pasar a través de un transformador de aislamiento.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the X10 connection
ConnectionID is int
ConnectionID = DomoOpen("COM4", "A")	
// Switch on the lamps connected to the A3 and A4 modules
DomoSend(ConnectionID, "A3 A4", domoOn)
// Equivalent to: DomoSwitchOn(ConnectionID, "A3 A4")
...
// Status of A3 lamp
IF DomoStatus(ConnectionID, "A3") = True THEN
	Info("The A3 lamp is switched on")
ELSE
	Info("The A3 lamp is switched off")
END
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DomoStatus(<Connection identifier> , <Address>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> el dispositivo especificado está encendido,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si el dispositivo especificado no está conectado a la corriente, se devuelve el estado almacenado en la interfaz X10.




**`<Connection identifier>: (Integro)`**

Identificador de la conexión a utilizar. Este identificador está definido por [DomoOpen](../WDLang3/1000003102.md).

**`<Address>: (Cadena de caracteres)`**

Address de dispositivo eléctrico compatible con el protocolo X10 correspondiente.

El formato de Address es el siguiente: &lt;Código de la casa&gt;&lt;Código del módulo&gt;&gt;Código de la casa

Donde:

&lt;El código de la casa&gt; corresponde a una letra que se encuentra entre A y P.

&lt;El código del módulo&gt; corresponde a un número que se encuentra entre 1 y 16.

Por ejemplo: "A1", "B14".

## Observaciones




### Limitación
<a name="limitacion_ELTPARAGRAPHE000060"></a>

**DomoStatus** no funciona con una interfaz CM15 PC-X10.

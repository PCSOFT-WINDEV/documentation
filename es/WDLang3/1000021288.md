


## grSunburstSectionColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia la Color de una sección en un gráfico de estallido solar.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds the data
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Montpellier", 320000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France" + TAB + "Paris", 789000)
grSunburstAddData(CHART_Chart, "Europe" + TAB + "France", 1109000) 
grSunburstAddData(CHART_Chart, "Europe" + TAB + "Belgium", 900000)
grSunburstAddData(CHART_Chart, "Asia", 250000)

// Customizes the colors of the chart
grSunburstSectionColor(CHART_Chart, "Europe", RGB(106, 188, 227))
grSunburstSectionColor(CHART_Chart, "Asia", RGB(182, 115, 251)) 

grDraw(CHART_Chart)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`grSunburstSectionColor(<Chart name> , <Section path> , <Color>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre del control Gráfico en el editor de ventanas o reportes.




**`<Section path>: (Cadena de caracteres)`**

ruta completa de la sección que debe ser coloreada en la tabla. Este parámetro tiene el siguiente formato: 

```txt
"<Name of base level>" + TAB + ["<Name of 1st level>" + TAB + ...
["<Name of last level>" + TAB + [...]]]"<Section name>"
```


**`<Color>: (Entero o constante de tipo Integer)`**

Color de la sección especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si una sección no tiene una Color específica, se le asigna automáticamente una Color. Esto te permite get un Gradient con las secciones de nivel superior si existen. 

- El Color puede modificarse en cualquier momento antes de dibujar el gráfico.


## NetIPToMAC( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el MAC Address correspondiente a un determinado IP Address.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the MAC address 
Info(NetIPToMAC("11.22.33.44"))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetIPToMAC(<IP address>)`
---

**`<Result>: (Cadena de caracteres)`**



- MAC Address (en formato "aa:bb:cc:dd:ee:ff"),

- Vacíe la cadena (""") si se produce un problema.




**`<IP address>: (Cadena de caracteres)`**

IP Address que debe ser convertido al formato "aa.bb.cc.dd".. Esta IP Address debe ser associated con una tarjeta de red física para ser convertida en una MAC Address.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### MAC Address y IP Address
<a name="mac_address_address_ELTPARAGRAPHE000033"></a>

El MAC Address es el Address físico especificado en las tarjetas de red de las computadoras MAC. La dirección IP es la dirección utilizada para comunicarse entre varias computadoras a través del protocolo IP (Protocolo INTERNET).

**Observación**: 

- Esta función sólo puede ser utilizada por el equipo conectada a través de una red local.

- Las direcciones IP en el formato IPv6 no están soportadas.

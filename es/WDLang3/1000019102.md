
## SaaSAdminAddSite( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El uso de esta función está reservado al administrador del SaaS. Un gestor de cuentas de cliente no puede utilizar esta función. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a "Forum" site into the database of SaaS Webservice
S is saasSite
S.Nom = "Forum"
SaaSAdminAddSite(S)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminAddSite(<SaaS site>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<SaaS site>: (saasSite Variable o cadena de caracteres)`**

description del nuevo sitio SaaS.

Si el parámetro es una cadena de caracteres, este parámetro debe corresponder al nombre del sitio tal como fue desplegado (distinguiendo entre mayúsculas y minúsculas).


## SaaSAdminListPricing( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista los precios de un sitio SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador del SaaS tendrá acceso a todos los precios. Un administrador de cuenta de cliente sólo tendrá acceso a los precios de los sitios a los que se suscribió.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the pricing of the CRM site in a List Box control
arrPricing is array of saasPricing
arrPricing = SaaSAdminListPricing("CRM")

FOR EACH Pricing OF arrPricing
	ListAdd(LIST_Pricing, Pricing..Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminListPricing(<SaaS site>)`
---

**`<Result>: (Array de las variables de la saasPricing)`**

Lista de precios de SaaS definida para el sitio.

**`<SaaS site>: (saasSite Variable o cadena de caracteres)`**

Nombre de la [saasSite](../WDLang3/1000019068.md) Variable correspondiente al sitio SaaS para el que se busca el precio.

Este parámetro también puede ser una cadena de caracteres correspondiente al nombre del sitio SaaS utilizado.

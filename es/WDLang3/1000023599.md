


## &lt;Gráfico&gt;.SeriesColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa el Color de una serie, el Color de una sección de pastel o el Color de una sección de embudo. 

Por defecto, los colores de las series (o los colores de las secciones) se seleccionan según un orden preset entre un set de 11 colores. 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the color of series 2 in the "CHART_MyChart" control
CHART_MyChart.SeriesColor(2, LightYellow)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.SeriesColor(<Series number> , <Color>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (se encuentra en el editor de ventanas, editor de páginas o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie a utilizar. 

**Para un gráfico circular** (constante *grPie*), este parámetro corresponde al índice del dato en la serie (número del sector).
**Para un gráfico de embudo** (constante *grFunnel*), este parámetro corresponde al índice del dato en la serie (número del sector).

**`<Color>: (Entero o constante de tipo Integer)`**

Color de la serie especificada. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [y la preselección Color](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los siguientes colores son usados por Default: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=grCouleurSerie.gif)


- Los colores definidos con la función **&lt;Gráfico&gt;.SeriesColor** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- Para visualizar los gradientes de Color, utilice [&lt;Tabla&gt;.Gradient](../WDLang3/1000024061.md).

- Para definir una Color para las series en los gráficos de "dispersión", se recomienda utilizar [&lt;Gráfico&gt;.ScatterSeriesColor](../WDLang3/1000024106.md).

- El [SeriesColor](../Proprietes/2510145.md) Property permite get e inicializar el Color de una serie o de una sección de pastel (igual que el **&lt;Gráfico&gt;.SeriesColor**).

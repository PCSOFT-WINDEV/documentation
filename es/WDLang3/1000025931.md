
## ModbusDisconnect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la sesión de Modbus. 

Esta función desconecta al cliente del servidor o esclavo Modbus y libera el puerto serie en el caso de una sesión de RTU.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a session and connect to the Modbus slave through the COM4 serial port
Session is modbusSession
Session = ModbusConnectRTU("COM4")

// If the connection fails, display the error
IF ErrorOccurred = True THEN
	Trace(ErrorInfo())
ELSE
	// Write a value to register 0x42 in the slave
	ModbusWriteRegister(Session, 0x42, 0xCAFE)
END

// Disconnection
ModbusDisconnect(Session)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ModbusDisconnect(<Session>)`
---

**`<Session>: (modbusSession Variable)`**

Nombre del [modbusSession](../WDLang3/1000025924.md) Variable que corresponde a la sesión Modbus a cerrar.




## <Chart>.interval (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la interval entre los valores de los ejes vertical y horizontal de un gráfico. 

**Recordatorio**: Por defecto, la interval es set a 0. Esta interval se ajusta al espacio disponible en el destino seleccionado.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Chart control
// Horizontal axis of chart with an interval set to 10
CHART_MyChart.Interval(10, grXCoordinate)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.Mark(<Step> [, <Type of axis>])`
---

**`<Chart control>: (Nombre del control)`**

interval entre dos marcas. Si este parámetro está ajustado a 0, la interval se ajustará al espacio disponible en el destino seleccionado.

**`<Step>: (Real)`**

interval entre dos marcas. Si este parámetro está ajustado a 0, la interval se ajustará al espacio disponible en el destino seleccionado.

**`<Type of axis>: (Constante opcional)`**

Tipo de eje para el que se define la interval:


|   |   |
| --- | --- |
| *grSecondaryYCoordinate* | Marcas en el eje X secundario. |
| *grXCoordinate* <br>(Valor predeterminado) | Marcas en el eje X. |
| *grYCoordinate* | Marcas en el eje Y. |
| *grZCoordinate* | Marcas en el eje Z (sólo en el gráfico de superficie).<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |

<a name="NOTE0_1"></a>

## Observaciones


### Marcas en los ejes
<a name="marcas_los_ejes_ELTPARAGRAPHE000257"></a>

- Limitaciones: 100 marcas Above y 100 marcas debajo del eje X.

- La última marca mostrada es el primer múltiplo de  mayor que el valor más alto de los datos. Se visualizan todos los datos independientemente de la dimensión de destino.

- Para modificar las marcas de inicio y fin del eje horizontal, vertical o secundario de un gráfico, utilice [.Origen](../WDLang3/1000024114.md).

- Los datos definidos con la función **<Chart>.interval** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).


<a name="NOTE0_2"></a>


### Tipos de gráficos
<a name="tipos_graficos_ELTPARAGRAPHE000285"></a>

Las marcas en los ejes no tienen ningún efecto en los gráficos circulares.

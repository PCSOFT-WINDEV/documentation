


## &lt;emailIMAPSession Variable&gt;.CurrentFolder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de la carpeta actual en el software de mensajería usando IMAP (INTERNET Message Access Protocol). Esta carpeta se encuentra en la rama "Carpetas personales".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// IMAP protocol
MySession is emailIMAPSession
...
// Create a folder in the current folder 
MySession.AddFolder(MySession.CurrentFolder() + "\Work")

// Create a folder in the custom folder
MySession.AddFolder("Personal folders\Inbox\MyOwnWork")
```


<a name="3032129_Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.CurrentFolder()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la carpeta actual,

- Cadena vacía ("") si se produce un error. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session>: (Variable de tipo emailIMAPSession)`**

Nombre de la Variable de tipo [emailIMAPSession](../WDLang3/1000018957.md) que corresponde a la sesión actual de Email.

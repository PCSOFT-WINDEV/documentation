


## Contacto (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Contact** se utiliza para definir las características avanzadas de un contacto que se encuentra en el libro de Phone.

Las características de esta Contact pueden definirse y modificarse mediante diferentes propiedades del lenguaje WL.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a phone number to the contact named "John Doe"
arrContact is array of Contact = ContactList()
IF NOT ErrorOccurred THEN
	c is Contact
	FOR EACH c of arrContact
		IF c.LastName ~= " Doe " _AND_ c.FirstName ~= " John " 
			tel is Phone
			tel.Number = " 0467010203 "
			tel.Type = phoneHome
			Add(c.Phone, tel)
			ContactModify(c)
			RETURN
		END
	END
END
```

## Observaciones


### Propiedades específicas de la description de las variables Contact
<a name="propiedades_especificas_description_las_variables_contact_ELTPARAGRAPHE000045"></a>

Las siguientes propiedades pueden ser usadas para manipular a Phone Contact:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Address | Array de [Address](../WDLang3/1000019944.md) | Lista de direcciones postales de Contact. |
| DisplayedName | Cadena de caracteres | Nombre de Contact como debe aparecer en el Phone (en la lista de Contact o al recibir una llamada por ejemplo).<br><br>Este nombre se construye automáticamente a partir de la información dada para el Contact, no puede ser modificado. Típicamente, el nombre mostrado corresponde a la concatenación del nombre y apellido. Si esta información no fue especificada para el Contact, el nombre mostrado puede corresponder a la dirección de correo electrónico o al número de Phone del Contact.<br><br>**Esta propiedad es de solo lectura**. |
| Email | Array de [EmailAddress](../WDLang3/1000020244.md) | Lista de direcciones de correo electrónico de Contact. |
| Event | Array de [Event](../WDLang3/1000020248.md) | Lista de eventos associated con la Contact. |
| Favorito | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el contacto está en la lista de favoritos,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| FirstName | Cadena de caracteres | El nombre de pila de la Contact. |
| Fuente | Array de [ContactSource](../WDLang3/1000020211.md) | Lista de fuentes de Contact (Google, Exchange, cuentas de Facebook) a las que pertenece Contact.<br><br> |
| ID | Integro | Identificador de Contact. <br><br>Atención: **Esta propiedad es de solo lectura**.. |
| InstantMessaging | Array de [InstantMessaging](../WDLang3/1000020247.md) | Lista de direcciones de mensajería instantánea de la Contact.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta propiedad no está disponible. |
| LastName | Cadena de caracteres | El apellido de los Contact. |
| Nota | Array de cadenas de caracteres | Lista de notas associated con la Contact.<br><br> |
| Phone | Array de [Phone](../WDLang3/1000020241.md) | Lista de números de Phone de la Contact. |
| SIP | Array de [SIPAddress](../WDLang3/1000020245.md) | Lista de direcciones SIP de la Contact.<br><br> |
| Sitio Web | Array de [WebAddress](../WDLang3/1000020246.md) | Lista de direcciones web de Contact. |

### Reinicio
<a name="reinicio_ELTPARAGRAPHE000192"></a>

Puede utilizar [VariableReset](../WDLang1/3013069.md) o [ContactReset](../WDLang3/1000006107.md) para restablecer el contenido de un contacto Variable.

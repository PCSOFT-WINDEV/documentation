


## &lt;Tipo de categoría de cascada (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el tipo de categoría en una tabla de cascada. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// TO 2017
CHART_Waterfall.AddData(1, 1, 298000)
CHART_Waterfall.CategoryLabel(1, "TO 2017")

// Add the profits
CHART_Waterfall.AddData(1, 2, 420000)
CHART_Waterfall.CategoryLabel(2, "Profits" + CR + "products")
CHART_Waterfall.AddData(1, 3, 210000)
CHART_Waterfall.CategoryLabel(3, "Profits" + CR + "services")

// Costs
CHART_Waterfall.AddData(1, 4, -170000)
CHART_Waterfall.CategoryLabel(4, "Fixed costs")
CHART_Waterfall.AddData(1, 5, -140000)
CHART_Waterfall.CategoryLabel(5, "Variable costs")

// TO 2018
CHART_Waterfall.AddData(1, 6, 0)
CHART_Waterfall.CategoryLabel(6, "TO 2018")
CHART_Waterfall.WaterfallCategoryType(6, grSubTotal)

// Cumulated TO
CHART_Waterfall.AddData(1, 7, 0)
CHART_Waterfall.CategoryLabel(7, "Cumulated TO")
CHART_Waterfall.WaterfallCategoryType(7, grTotal)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Chart control>.WaterfallCategoryType(<Category> , <Type of category>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico en el editor de ventanas, editor de reportes o editor de páginas.


**`<Category>: (Integro)`**

Número de la categoría a utilizar.

**`<Type of category>: (Constante de tipo Integer)`**

Tipo de categoría: 



|   |   |
| --- | --- |
| *grDefault* | Valores positivos o negativos del gráfico. |
| *grSubTotal* | Valores subtotales.<br><br>Si la categoría corresponde a un subtotal, se ignora el valor almacenado para esta categoría.. Para esta categoría, la tabla de cascada muestra una barra correspondiente a la suma de todas las categorías anteriores del último total o subtotal. |
| *grTotal* | Valores totales.<br><br>Si la categoría corresponde a un total, se ignora el valor almacenado para esta categoría.. Para esta categoría, la carta de cascada muestra una barra correspondiente a la suma de todas las categorías anteriores desde el principio de la carta. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Caso especial: la primera categoría**
	La primera categoría es siempre una categoría "total", pero se muestra el valor especificado.: es el valor inicial. 

- Esta función se aplica sólo a las cartas de cascada.

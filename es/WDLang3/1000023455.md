
## BackgroundTaskDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una tarea Background (que fue añadida por [BackgroundTaskAdd](../WDLang3/1000023454.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the background task for processing emails 
BackgroundTaskDelete(ProcBackgroundEmailProcess)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BackgroundTaskDelete(<WLanguage procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la tarea,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de WLanguage Procedure correspondiente a la tarea de Background a eliminar. El nombre de esta Procedure se especificó cuando la tarea fue añadida por [BackgroundTaskAdd](../WDLang3/1000023454.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Límite
<a name="limite_ELTPARAGRAPHE000051"></a>

Atención: No se puede eliminar mediante programación una tarea en segundo plano añadida automáticamente a través de la automatización de Procedure. 

Para borrar estas tareas, la opción correspondiente debe estar desactivada en la ventana de configuración de la automatización de Procedure. El borrado será efectivo durante el siguiente inicio de la aplicación.




## &lt;Mapa&gt;.ExecuteJS (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta el código nativo de Google Maps API en un control Mapa en JavaScript. 

Esta función se utiliza principalmente para personalizar la control Mapa de WINDEV proponiendo características específicas de la API de Google Maps.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Centers the map on 45.20,26.51
let sCode = [
	var LatLng = new google.maps.LatLng(45.20,26.51);
	MapGetJSObject().panTo(LatLng);
]
MAP_MyMap.ExecuteJS(sCode)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Map control>.ExecuteJS(<JavaScript code>)`
---

**`<Result>: (Cadena de caracteres)`**



- Resultado de la ejecución del código JavaScript a ejecutar, si este código devuelve un resultado. Este resultado se devuelve en formato de cadena. 

- "" (cadena vacía) si el código JavaScript no devuelve ningún resultado.




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar.

**`<JavaScript code>: (Cadena de caracteres)`**

Código JavaScript para ejecutar. En este código, el nombre del objeto JavaScript que representa el mapa de Google es devuelto por [&lt;Mapa&gt;.GetJSObject](../WDLang3/1000022385.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para más detalles, consulte la documentación de Google ([https://developers.google.com/maps/documentation/javascript/](https://developers.google.com/maps/documentation/javascript/)). 

- Si el código genera errores, estos pueden ser mostrados al usuario final y/o impedir que la control Mapa funcione correctamente.. 

- El uso de Google Maps API no está garantizado en el futuro.

- El uso de Google Maps API no es compatible con las aplicaciones iOS/Android.

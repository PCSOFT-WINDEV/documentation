


## grAxisTitle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el título del eje X y el título del eje Y en un gráfico.

Ejemplo de títulos para los ejes del gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=titreaxe.gif)


**Observación**: Default no define ningún título para los ejes de un gráfico.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the title of the vertical axis in "MyChart"
grAxisTitle("MyChart", "Time", grYCoordinate)
```
<a name="Example2"></a>

```wl
// Modify the title of the horizontal axis in "CHART_MyChart"
grAxisTitle(CHART_MyChart, "Time", grYCoordinate)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`grAxisTitle(<Chart name> , <Title> , <Axis direction>)`
---

**`<Chart name>: (Nombre del control o cadena de caracteres)`**

Nombre del gráfico a utilizar. Este nombre corresponde a:

- el nombre del gráfico definido mediante programación con la función [grCreate](../WDLang3/3042005.md).

- el nombre de la control Gráfico en la ventana, página o editor de reportes.




**`<Title>: (Cadena de caracteres)`**

Título del eje especificado.

**`<Axis direction>: (Constante)`**

Orientación del eje a utilizar:


|   |   |
| --- | --- |
| *grSecondaryYCoordinate* | Marcas en el eje Y secundario (eje vertical). |
| *grXCoordinate*<br>(valor predeterminado) | Marcas en el eje X (eje horizontal). |
| *grYCoordinate* | Marcas en el eje Y (eje vertical). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tipos de gráficos
<a name="tipos_graficos_ELTPARAGRAPHE000161"></a>

Dependiendo del tipo de gráfico y de la &lt;Dirección del eje&gt;, la función **grAxisTitle** se tiene en cuenta o no:

| Tipo de gráfico | Eje horizontal | Eje vertical |
| --- | --- | --- |
| Gráfico de líneas | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |
| Gráfico de cotizaciones (Candelabro y gráfico de barras) | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |
| Gráfico de columnas | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |
| Min/Max Gráfico de cotizaciones | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |
| Dispersión | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |
| Circular | **grAxisTitle** tenida en cuenta. | **grAxisTitle** tenida en cuenta. |


Las modificaciones realizadas con la función **grAxisTitle** se tendrán en cuenta la próxima vez que se llame a la función [grDraw](../WDLang3/3042026.md).
<a name="NOTE0_2"></a>


### Título ejes
<a name="titulo_ejes_ELTPARAGRAPHE000247"></a>El título de los ejes del gráfico se puede mostrar para los diferentes ejes.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Titre.gif)
El contenido del título de los ejes del gráfico se define con la función **grAxisTitle**. Esta función también se puede utilizar para posicionar este título según el eje deseado. No se puede definir el posicionamiento y/o la orientación del título en relación al eje..  

El tipo de letra utilizado para el título del gráfico puede ser modificado por [grAxisTitleFont](../WDLang3/3042059.md).
<a name="NOTE0_3"></a>


### Las lenguas que utilizan un carácter no latino set
<a name="las_lenguas_que_utilizan_caracter_latino_set_ELTPARAGRAPHE000265"></a>

**grAxisTitle** utilizará el carácter no latino seleccionado set sólo si la configuración del proyecto associated utiliza cadenas UNICODE en tiempo de ejecución.

Para comprobar la opción seleccionada en la configuración actual: 

1. En la pestaña "Proyecto", en el grupo "Configuración del proyecto", haga clic en "Configuración actual". 

2. Ir a la pestaña "Unicode": la opción "Utilizar cadenas UNICODE en ejecución" debe estar seleccionada. 

3. Validar si es necesario.

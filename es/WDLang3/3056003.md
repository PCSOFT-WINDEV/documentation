<a name="Use"></a>
<a name="description"></a>
Activa un evento Windows en el sistema de un servidor FTP (File Transfer Protocol) o RPC (Remote Procedure Call) alimentado por WINDEV. Este Event será enviado a todas las ventanas abiertas en el servidor.

**Atención**: Esta función debe utilizarse en un ordenador cliente.

**Observación**: Para permitir (o no) el uso de esta función en un servidor FTP/RPC, llame a [NetServerOpción](../WDLang3/3056022.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003056003&name=netsendmessage_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
ConnectRPC is int
ConnectRPC = NetConnect("148.61.125.245", RPCServer, "GUEST", "")
...
// Client side: sending a Windows message to the server
NetSendMessage(ConnectRPC, "MyMessage")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`NetSendMessage(<Connection identifier> , <Event> [, <Parameter 1> [, <Parameter 2>]])`
---

**`<Connection identifier>: (Integro)`**

Identificador de la conexión para el ordenador cliente (devuelto por [NetConnect](../WDLang3/3056004.md)).

**`<Event>: (Cadena de caracteres enteros o Ansi)`**

Identificador del Windows Event enviado a las ventanas del servidor (número de Windows Event o cadena de caracteres específicos).

**`<Parameter 1>: (Entero opcional)`**

Parámetro asociado a la Event.

**`<Parameter 2>: (Entero opcional)`**

Parámetro asociado a la Event.

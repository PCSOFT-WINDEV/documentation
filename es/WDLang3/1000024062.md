


## &lt;Dibujo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja un gráfico de acuerdo con los parámetros especificados.

**Observaciones**: 

- La función **&lt;Dibujo** tiene en cuenta los nuevos parámetros especificados desde la última llamada a esta misma función.

- Esta función no debe utilizarse para los controles de gráfico que se encuentran en la editor de reportes.


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draw a chart in a Chart control
CHART_MyChart.Draw()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.Draw()`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (se encuentra en una ventana o en una página).

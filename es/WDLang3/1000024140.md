


## &lt;OverlayChart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para mostrar dos tipos diferentes de gráficos en el mismo gráfico. Los dos gráficos están superpuestos.

**Observación**: El primer gráfico debe ser un gráfico de columna (gráfico de columna agrupado o apilado), un Gráfico de cotizaciones (MinMax o Candlestick) o un gráfico Scatter, mientras que el segundo gráfico debe ser un gráfico Line.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF CHART_MyColumnChart.OverlayChart(CHART_MyLine) = True THEN
	Info("The charts are overlaid")
END
CHART_MyColumnChart.Draw()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Destination Spreadsheet control>.OverlayChart(<Source Chart control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se superpusieron los gráficos,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Destination Spreadsheet control>: (Nombre del control)`**

Nombre de la control Gráfico que mostrará los dos gráficos al mismo tiempo. Este control Gráfico contiene el primer gráfico (Columna, Stock o Gráfico de dispersión).

**`<Source Chart control>: (Cadena de caracteres)`**

Nombre de la control Gráfico que se mostrará en la carta de destino. Esta tabla debe ser una tabla de Line. Se superpondrá a la lista &lt;Destination control Gráfico&gt;.. Esta tabla se tendrá en cuenta como fuente de datos: su título y sus parámetros serán ignorados. Sólo se conservará la información relativa a la leyenda..
El enlace entre los dos gráficos se borra si este parámetro corresponde a una cadena vacía (""").
Este parámetro debe corresponder a un gráfico diferente de &lt;Destination control Gráfico&gt; (de lo contrario, los gráficos se sobrescriben entre sí).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La superposición de gráficos definida con la función **&lt;OverlayChart** se tendrá en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- Atención: **&lt;OverlayChart** no manipular las cruces de los diferentes gráficos superpuestos.




## Funciones SNMP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de SNMP son las siguientes: 



|   |   |
| --- | --- |
| [SNMPCloseSession](../WDLang3/1000017068.md) | Cierra una sesión SNMP. |
| [SNMPGet](../WDLang3/1000017069.md) | Lee uno o más valores de un agente con el que se ha iniciado una sesión SNMP. |
| [SNMPGetNext](../WDLang3/1000017070.md) | Lee el valor encontrado después del último valor recuperado para un agente SNMP. |
| [SNMPGetTable](../WDLang3/1000017217.md) | Lee un tabla de valores SNMP. |
| [SNMPLoadMIB](../WDLang3/1000017074.md) | Carga un archivo MIB en la memoria y lo analiza. |
| [SNMPOIDAccess](../WDLang3/1000017077.md) | Devuelve los modos de acceso autorizados de un OID específico. |
| [SNMPOIDDescription](../WDLang3/1000017079.md) | Devuelve la description de un OID. |
| [SNMPOIDStatus](../WDLang3/1000017078.md) | Devuelve el estado de un OID. |
| [SNMPOIDToString](../WDLang3/1000017075.md) | Convierte un OID de su representación numérica a su representación textual. |
| [SNMPOIDType](../WDLang3/1000017080.md) | Devuelve el tipo de datos de un OID. |
| [SNMPSet](../WDLang3/1000017071.md) | Escribe uno o más valores en un agente SNMP para el que se ha iniciado una sesión SNMP. |
| [SNMPStartSession](../WDLang3/1000017067.md) | Inicia una sesión SNMP. |
| [SNMPStringToOID](../WDLang3/1000017076.md) | Convierte un OID de su representación de texto a su representación numérica. |
| [SNMPTrapDisable](../WDLang3/1000017073.md) | Desactiva la interceptación de una trampa SNMP. |
| [SNMPTrapEnable](../WDLang3/1000017072.md) | Permite la interceptación de un "trap" SNMP enviado por un agente. |

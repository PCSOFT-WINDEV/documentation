
## notificationFormat (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
A partir de Android 4.1 (Jelly Bean, nivel api 16), las notificaciones mostradas en la lista de notificaciones del dispositivo pueden "ampliarse" para mostrar más contenido (Image, texto multilínea, etc.). 

El sistema contrae o desglosa automáticamente los avisos visualizados en función del espacio disponible en la lista.. Sin embargo, el usuario tiene la capacidad de colapsar o expandir una Notification realizando un gesto de deslizamiento hacia arriba (colapsar) o hacia abajo (expandir) con dos dedos.

El tipo **notificationFormat** describe el formato de visualización de un Notification cuando se expande. Las características del formato de visualización pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

Para asociar un formato a un Notification, utilice el formato **** Property del tipo [Notification](../WDLang3/1000019441.md).

**Observaciones**:

- El formato será ignorado si la Notification se muestra en un dispositivo que ejecuta una versión anterior a Android 4.1.

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
n is notification
n.Title = "Photo saved"
n.Message = "Date taken: 07/18/2014 19:45"
n.LargeIcon = "photo.png"
n.SecondaryMessage = "desert.png"
n.Format.Type = notifImageFormat
n.Format.Content = fCurrentDir + "photos/desert.png"
NotifAdd(n)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables notificationFormat
<a name="propiedades_especificas_descripcion_las_variables_notificationformat_ELTPARAGRAPHE000042"></a>

Las siguientes propiedades pueden ser usadas para manipular un formato Notification:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Contenido | Cadena de caracteres | Contenido de Notification para mostrar. El valor de esta Property depende del tipo de formato utilizado (**Tipo** Property). <br><br>- Si el **Tipo** Property está configurado como *notifImageFormat*, el **Contenido** Property corresponde al ruta del Image a visualizar.<br><br>- Si el **Tipo** Property está configurado como *notifListFormat*, el **Contenido** Property corresponde a los elementos que deben aparecer en la lista, separados por retornos de carro (CR). Tiene la posibilidad de mostrar hasta 5 elementos en la lista. Los elementos adicionales serán ignorados.<br><br>- Si el **Tipo** Property se ajusta a *notifTextFormat*, el **Contenido** Property corresponde al texto multilínea que se va a mostrar. Si el texto especificado es demasiado largo, puede ser truncado cuando se muestra.<br><br><br> |
| escribir a máquina | Constante de tipo Integer | Tipo de formato que se aplicará a la Notification cuando ésta se amplíe: <br><br>- *notifImageFormat*: Mostrar un Image. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NotifFormatimage.gif)<br><br><br>- *notifListFormat*: Mostrar una lista de elementos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NotifFormatListe.gif)<br><br><br>- *notifTextFormat* (valor de Default): Mostrar un texto multilínea.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NotifFormatTexte.gif)<br><br><br><br> |
| LargeIcon | Cadena de caracteres | Ruta de la imagen correspondiente al gran icono asociado a la Notification. Este icono aparecerá a la izquierda del título y del mensaje de Notification. <br><br>Si esta propiedad no se rellena, el gran icono de Notification será idéntico al utilizado cuando se colapsó el Notification.<br><br>Esta propiedad sólo se tiene en cuenta si la propiedad **Type** se establece como *notifImageFormat*. |
| SecondaryMessage | Cadena de caracteres | Mensaje secundario de la Notification. Si esta propiedad no se llena, el mensaje secundario de Notification será idéntico al utilizado cuando se colapsa el Notification. |
| Título | Cadena de caracteres | El título de Notification cuando éste se amplíe. Si esta propiedad no se llena, el título de Notification será idéntico al que se usó cuando se derrumbó el Notification. |

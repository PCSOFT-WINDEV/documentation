


## MapEndOfMove (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se detiene siguiendo el movimiento del dispositivo en una control Mapa. Este seguimiento fue iniciado de antemano por [MapFollowMovement](../WDLang3/1000019953.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start movement tracking in the Map control
MapFollowMovement(MAP_Position)
...
// Stops following the movement in the Map control
MapEndOfMove(MAP_Position)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapEndOfMove([<Map control>])`
---

**`<Map control>: (Nombre de control opcional)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, se utilizará la control Mapa a la que pertenece la actual Event.

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000078"></a>

Cuando se utiliza esta función, se debe declarar una funcionalidad de la aplicación en el asistente de generación de aplicaciones.

Característica necesaria: Geolocalización

Esta función permite a las aplicaciones utilizar las funciones de geolocalización del dispositivo.

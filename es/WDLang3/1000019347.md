
## OutlookDefaultProfile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el perfil de Default definido en Outlook.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the default profile
DefaultProfile is string 
DefaultProfile = OutlookDefaultProfile()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = OutlookDefaultProfile()`
---

**`<Result>: (Cadena de caracteres)`**



- Perfil de Default,

- Vaciar la cadena ("") si no se define un perfil de Default.




**Observación**: Si se definió un perfil de Default, este perfil se devuelve aunque no esté habilitado.


## saasUser (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **saasUser** permite definir las características avanzadas de un usuario de sitio SaaS. 

Las características de este usuario pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NewUser is saasUser
NewUser.Login = "mark@jet-tames.com"
NewUser.Password = "secret"

ClientAccount is saasClient = SaaSAdminFindClient("JetTames")

SaaSAdminAddUser(NewUser, ClientAccount)
```

## Observaciones


### Propiedades específicas de las variables saasUser
<a name="propiedades_especificas_las_variables_saasuser_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un usuario SaaS:

| **denominar** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Cliente | Variable de tipo [saasClient](../WDLang3/1000019072.md) | reference hacia el cliente al cual el usuario es relacionado.<br>**Esta propiedad es de solo lectura**. |
| Contraseña | Cadena de caracteres | Contraseña utilizada para conectarse a los sitios SaaS. |
| ContraseñaPara entrar | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el usuario debe introducir una nueva contraseña,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>Este Property se utiliza para gestionar una vida útil para las contraseñas. |
| de acceso | Cadena de caracteres | Nombre de usuario.<br>Este login debe ser único (entre las diferentes cuentas de clientes) en la base de datos SaaS..<br>Para ello, tienes la posibilidad de usar las direcciones de Email por ejemplo. |
| denominar | Cadena de caracteres | Apellido del usuario. |
| Derechos de Administrador | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el usuario es un "Administrador",<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>Los administradores pueden utilizar el sitio [Administrador SaaS](../SaaSAdminWeb/1600000.md) o aplicaciones equivalentes para asignar licencias a los usuarios de la Cuenta Cliente SaaS a la que pertenecen. |
| Email | Cadena de caracteres | Dirección de correo electrónico del usuario. <br>Este correo electrónico Address se utiliza para enviar el código de seguridad ([SaaSSendSecurityCode](../WDLang3/1000022528.md)). Si este Property no se llena, se envía el correo electrónico al login (si el login corresponde a un correo electrónico válido Address).  |
| FirstName | Cadena de caracteres | Nombre del usuario. |

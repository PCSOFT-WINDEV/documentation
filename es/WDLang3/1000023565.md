


## <Chart>.StockAddData (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade datos en un gráfico de existencias (candelabros, gráficos de barras o minmax).

**Atención**: Para cada tipo de gráfico debe utilizarse una función específica de WLanguage (véase "Observaciones"). 
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="3042009_Example2"></a>

```wl
// Chart control
// Add data into "CHART_MyChart"
CHART_MyChart.StockAddData(1, 5.4, 6.8, 6.1, 6.3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.StockAddData(<Series number> [, <Index>] , <Minimum value> , <Maximum value> , <First value> , <Last value>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar (en la ventana, página o editor de reportes).


**`<Series number>: (Integro)`**

Número de la serie a la que se añadirá un valor. Si la serie especificada no existe, el diseñador de la carta crea tantas series como sean necesarias para get una matriz consistente. Todas las series creadas se inicializan a 0.

**Atención: Sólo se sorteará la primera serie**.

**`<Index>: (Entero opcional)`**

Índice de datos de la serie en la que se añadirán los valores. Si este parámetro:

- no se especifica: se añaden al final de la serie.

- se especifica: se sustituye el valor anterior.




**`<Minimum value>: (Entero o real)`**

Valor mínimo para el día.

**`<Maximum value>: (Entero o real)`**

Valor máximo para el día.

**`<First value>: (Entero o real)`**

Primer valor del día.

**`<Last value>: (Entero o real)`**

Último valor del día.

## Observaciones


### Coherencia de valores
<a name="coherencia_valores_ELTPARAGRAPHE000276"></a>

Si los valores no son consistentes, los datos no se mostrarán en el gráfico:

- &lt;El primer valor&gt;, &lt;Último valor&gt; y &lt;Valor mínimo&gt; debe ser menor que &lt;Valor máximo&gt;.

- &lt;El primer valor&gt;, &lt;Último valor&gt; y &lt;Máximo valor&gt; debe ser mayor que &lt;Mínimo valor&gt;.



<a name="NOTE0_2"></a>


### Añadir datos
<a name="anadir_datos_ELTPARAGRAPHE000289"></a>

- Los datos agregados con la función **<Chart>.StockAddData** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- Sólo se sorteará la primera serie.



<a name="NOTE0_3"></a>


### Añadir datos a los diferentes tipos de gráficos
<a name="anadir_datos_los_diferentes_tipos_graficos_ELTPARAGRAPHE000307"></a>

- Para añadir datos en los gráficos estándar (Line, pastel, ...), utilice [.AddData](../WDLang3/1000023533.md). 

- Para los gráficos de dispersión, se recomienda utilizar [.ScatterAddDataXY](../WDLang3/1000024105.md).

- Para los gráficos de mapas de calor, se recomienda utilizar [&lt;Gráfico&gt;.HMAddData](../WDLang3/1000024079.md).

- Para las cartas Sunburst, se recomienda utilizar [&lt;Gráfico&gt;.SunburstAddData](../WDLang3/1000024132.md). 

- Para los gráficos de superficie, se recomienda utilizar [&lt;Tabla&gt;.SurfaceAddData](../WDLang3/1000024143.md).

- Para añadir un valor a los datos de un gráfico, utilice [.IncreaseData](../WDLang3/1000024086.md).

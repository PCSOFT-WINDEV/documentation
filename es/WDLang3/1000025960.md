


## WebSocketDisconnect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desconecta un cliente de un servidor WebSocket. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve list of clients
arrClients is array of websocketClient = WebSocketListConnectedClient()
// Disconnect some clients
FOR EACH oClient OF arrClients
	IF bDisconnect(oClient) THEN
		WebSocketDisconnect(oClient)
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WebSocketDisconnect(<Client>)`
---

**`<Client>: (websocketClient Variable)`**

Nombre de la [websocketClient](../WDLang3/1000025952.md) Variable que corresponde al cliente a desconectar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La desconexión del cliente Process se llama cuando el cliente está desconectado.

- **Esta función es asíncrona**: simplemente solicita la desconexión del cliente. El cliente puede desconectarse realmente más tarde (después de ejecutar **WebSocketDisconnect**). La función **WebSocketDisconnect** nunca devuelve errores (por ejemplo, si el cliente se desconectó).




## StreamStop( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la recepción o transmisión de un tipo de datos en un flujo. La emisión o la recepción del tipo de datos puede reanudarse a través de [StreamPlay](../WDLang3/3039010.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops receiving sound in the stream
StreamStop(STREAM_MyStream.Value, StreamIncomingSound)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`StreamStop(<Stream identifier> [, <Type of stream>])`
---

**`<Stream identifier>: (Integro)`**

Identificador de la secuencia a utilizar. Este identificador se define en la "Solicitud de conexión del nuevo flujo" Event associated con el flujo. También se puede identificar con la [Valor](../Proprietes/2510130.md) Property en el arroyo.

**`<Type of stream>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Tipo de arroyo a detener:

- Constantes simples
	


|   |   |
| --- | --- |
| *StreamIncomingSound* | Flujo de sonido entrante. |
| *StreamIncomingVideo* | Transmisión de video entrante. |
| *StreamOutgoingSound* | Flujo de sonido saliente. |
| *StreamOutgoingVideo* | Transmisión de video saliente. |



- Constantes correspondientes a una combinación de constantes
	


|   |   |
| --- | --- |
| *StreamAll*<br>(valor predeterminado) | Vídeo + Flujo de sonido (*StreamFullDuplexSound*+ *StreamFullDuplexVideo*). |
| *StreamFullDuplexSound* | Flujo de sonido entrante y saliente (corresponde a *StreamOutgoingSound* + *StreamIncomingSound*). |
| *StreamFullDuplexVideo* | Flujo de tipo video recibido y enviado (corresponde a *StreamOutgoingVideo* + *StreamIncomingVideo*). |

## Observaciones


### Procedure para la detección de corrientes
<a name="procedure_para_deteccion_corrientes_ELTPARAGRAPHE000077"></a>

Un flujo está asociado a la Event "Solicitud de conexión de un nuevo flujo". Este Event se ejecuta cuando se detecta una solicitud de conexión a un flujo entrante.

Este Event se ejecuta en uno de los siguientes casos: 

- si el valor de la propiedad [Detección](../Proprietes/2510031.md) utilizada en el flujo es <u><u><u><u>True</u></u></u></u>.

- si la opción "Detección de corrientes entrantes activas" ha sido marcada en la ventana de la corriente description.




**Observación**: Este Event se ejecuta en un thread específico de WLanguage. Por lo tanto, está sujeto a limitaciones específicas de los hilos de rosca. Para más detalles sobre los hilos, véase [Gestión de hilos con WINDEV](../WDLang1/3077026.md).


## Gestión de las teclas Bluetooth
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000124"></a>
¡Crea un nuevo tipo de aplicación! Sin Contact entre el PC y el dispositivo móvil, se pueden ejecutar varios procesos en el PC utilizando los datos que se encuentran en el dispositivo móvil.

Por ejemplo:

- Una aplicación en un PC (o dispositivo móvil) puede emular servicios en un dispositivo que soporte el estándar bluetooth.

- Una aplicación puede recuperar automáticamente fotos de una célula Phone.

- Una aplicación puede leer la lista de contactos que se encuentra en una célula Phone.

- Un PC puede imprimir los datos encontrados en una PDA.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000148"></a>


### Varios grupos de funciones le permiten comunicarse por Bluetooth:
<a name="varios_grupos_funciones_permiten_comunicarse_por_bluetooth_ELTPARAGRAPHE000025"></a>

- [Las funciones de Bluetooth](../WDLang3/1000017173.md) (BTxxx): Estas funciones le permiten gestionar los dispositivos Bluetooth. Por ejemplo, puede listar los servicios que se encuentran en un dispositivo y habilitarlos o deshabilitarlos.. También puedes autentificar hacia un dispositivo Bluetooth y get varias informaciones sobre el dispositivo.

- [Las funciones de OBEXxxx](../WDLang3/1000017175.md): Estas funciones permiten gestionar los intercambios de ficheros a través del protocolo OBEX. El protocolo OBEX es un protocolo para el intercambio de datos por infrarrojos y Bluetooth.

- [SocketConnectBluetooth](../WDLang3/1000017145.md) y [SocketCreateBluetooth](../WDLang3/1000017167.md) que permiten gestionar los sockets que utilizan los dispositivos Bluetooth.

## ¿Qué llaves se deben utilizar?
<a name="¿que_llaves_deben_utilizar_ELTTEXTE000172"></a>
Las funciones que le permiten manipular las teclas Bluetooth funcionan con las teclas que utilizan una pila de protocolos emitidos por Microsoft o Bluesoleil.. Para get la pila utilizada, llame a [BTStack](../WDLang3/1000017229.md).

Algunas funciones de WLanguage sólo se pueden utilizar con algunos tipos de pilas. Esta información se presenta en detalle en cada una de las funciones relevantes de WLanguage. 

Las siguientes teclas utilizan una pila de Microsoft (sin necesidad de instalar el controlador suministrado con las teclas):

- D-Link DBT-120,

- Broadcom Bluetooth 2.0 EDR USB,

- Dongle EMTEC Bluetooth 2.0 EDR - EKCOB110,

- Adaptador Nano Bluetooth HAMA 2.0 EDR - 10m,

- Dongle Conceptronic Bluetooth 2.0 EDR - CBT200U2A,

- Microadaptador USB Kensington Bluetooth (Bluetooth 2.0 - USB 2.0),

- (\*) Adaptador Bluetooth 2.0 de Belkin - F8T013FR1 - Rango de 10 metros (USB 2.0),

- (\*) Adaptador Bluetooth 2.0 de Belkin - F8T012FR1 - Alcance de 100 metros (USB 2.0),

- TrendNet TBW-104UB - Adaptador USB Bluetooth 2.0,

- (\*) TrendNet TBW-102UB - Adaptador Bluetooth Clase 2 (Bluetooth 1.1), ....




Esta lista no es exhaustiva y se actualizará periódicamente.

(\*) Algunas teclas no funcionan directamente en Windows, es necesario instalar un controlador a través del siguiente modo de funcionamiento:

1. si se instaló, desinstale el controlador suministrado con la llave a través de "Agregar/Quitar programas" que se encuentra en el Panel de control,

2. inserte el CD suministrado con la llave en la unidad, NO inicie su ejecución de automatic,

3. conecte la llave,

4. pida a Windows que busque el controlador en el CD.




Este método le permite tener una llave reconocida por Windows y por lo tanto usar una pila de Microsoft.

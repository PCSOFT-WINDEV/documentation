
## modbusSession (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **modbusSession** se utiliza para identificar una conexión a un servidor Modbus. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a session and connect to the Modbus server at address 192.168.1.18 on port 3275.
Session is modbusSession
Session = ModbusConnectTCP("192.168.1.18",3275)

// If the connection fails, display the error
IF ErrorOccurred = True THEN
	Trace(ErrorInfo())
ELSE
	// Write a value to register 0x42 on the server.
	ModbusWriteRegister(Session, 0x42,0xCAFE)
END

// Disconnect server
ModbusDisconnect(Session)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables modbusSession
<a name="propiedades_especificas_las_variables_modbussession_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser utilizadas para las sesiones de manipular Modbus:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ResponseTimeout | Integro | Tiempo máximo de respuesta de un servidor Modbus. |
| SlaveID | Integro | Identificación del esclavo. Sirve para cambiar el ID del esclavo sin tener que abrir una sesión de nuevo. <br><br> |


<a name="NOTE0_2"></a>


### Las funciones de WLanguage que utilizan las variables de modbusSession
<a name="las_funciones_wlanguage_que_utilizan_las_variables_modbussession_ELTPARAGRAPHE000059"></a>Las variables de **modbusSession** pueden ser manejadas usando las siguientes funciones WLanguage: 



|   |   |
| --- | --- |
| [ModbusConnectRTU](../WDLang3/1000025929.md) | Conecta un cliente a un esclavo Modbus a través de un puerto serie. |
| [ModbusConnectTCP](../WDLang3/1000025930.md) | Conecta un cliente a un servidor Modbus mediante el protocolo de transmisión control (TCP). |
| [ModbusDisconnect](../WDLang3/1000025931.md) | Cierra la sesión de Modbus. |
| [ModbusReadBit](../WDLang3/1000025935.md) | Lee uno o más bits de un esclavo Modbus, empezando por una Address específica. |
| [ModbusReadFloat](../WDLang3/1000025936.md) | Lee uno o más flotadores de una sola precisión de un esclavo Modbus, a partir de una Address específica. |
| [ModbusReadRegister](../WDLang3/1000025937.md) | Lee uno o más registros de un esclavo Modbus, comenzando en una Address específica. |
| [ModbusWriteBit](../WDLang3/1000025932.md) | Escribe uno o varios bits a una Address específica en un esclavo Modbus. |
| [ModbusWriteFloat](../WDLang3/1000025933.md) | Escribe uno o más flotadores de precisión simple a un esclavo Modbus, comenzando en una Address específica. |
| [ModbusWriteRegister](../WDLang3/1000025934.md) | Escribe uno o más registros (valores de 16 bits) a un esclavo Modbus, a partir de una Address específica. |

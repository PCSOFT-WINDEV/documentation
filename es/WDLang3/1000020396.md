Las siguientes constantes se utilizan para gestionar los controles de mapa: 

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **addressCustom** | Tipo de Address: costumbre Address. | Variable de tipo [Address](../WDLang3/1000019944.md) |
| **addressHome** | Tipo de Address: casa Address. | Variable de tipo [Address](../WDLang3/1000019944.md) |
| **addressOther** | Tipo de Address: otro tipo de Address. | Variable de tipo [Address](../WDLang3/1000019944.md) |
| **addressWork** | Tipo de Address: profesional Address (oficina). | Variable de tipo [Address](../WDLang3/1000019944.md) |
| **itineraryCar** | El itinerario se trazará teniendo en cuenta las carreteras accesibles a los coches.<br><br>Características de este modo:<br><br><br><br>- Este modo utiliza el modo de geolocalización de Google. Se requiere un acceso a INTERNET.<br><br>- Puedes reorganizar el orden de las paradas get de la mejor manera posible.<br><br><br> | [MapAddItinerary](../WDLang3/1000020383.md) |
| **itineraryCarNotOptimized** | El itinerario se trazará teniendo en cuenta las carreteras accesibles a los coches.<br><br>Características de este modo:<br><br><br><br>- Este modo utiliza el modo de geolocalización de Google. Se requiere un acceso a INTERNET.<br><br>- El orden de las paradas no será cambiado.<br><br><br> | [MapAddItinerary](../WDLang3/1000020383.md) |
| **itineraryCyclist** | El itinerario será trazado respetando los caminos accesibles a las bicicletas.. Los itinerarios para ciclistas no están disponibles en todos los países.. Consulte la documentación sobre el servicio de geo-localización de Google para más detalles.<br><br>Características de este modo:<br><br><br><br>- Este modo utiliza el modo de geolocalización de Google. Se requiere un acceso a INTERNET.<br><br>- Puedes reorganizar el orden de las paradas get de la mejor manera posible.<br><br><br> | [MapAddItinerary](../WDLang3/1000020383.md) |
| **itineraryManual** | El itinerario se dibujará en una Line recta entre cada posición especificada, ignorando las carreteras.<br><br>Este modo no utiliza necesariamente el modo de geolocalización de Google.. En este caso, el parámetro &lt;Posiciones&gt; debe corresponder a una array de variables geoPosition o Marker.<br><br><br><br>En este modo, los pasos intermedios se tienen en cuenta según el orden de adición en el array. | [MapAddItinerary](../WDLang3/1000020383.md) |
| **itineraryPedestrian** | El itinerario será trazado respetando los caminos accesibles a los peatones.<br><br>Características de este modo:<br><br><br><br>- Este modo utiliza el modo de geolocalización de Google. Se requiere un acceso a INTERNET.<br><br>- Puedes reorganizar el orden de las paradas get de la mejor manera posible.<br><br><br> | [MapAddItinerary](../WDLang3/1000020383.md) |
| **mcsAubergine** | El estilo aplicado al mapa: al estilo "berenjena". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsAutomatic** | El estilo aplicado al mapa: El estilo se aplicará de acuerdo con el modo de luz u oscuridad de la aplicación:<br><br>- Tema claro: se utilizará el estilo "estándar",<br><br>- Tema oscuro: se usará el estilo "nocturno".<br><br><br>Si el tema de la aplicación cambia mientras se muestra la control Mapa, el estilo cambiará automáticamente. | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsCustom** | El estilo aplicado al mapa: Estilo personalizado. | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsDark** | El estilo aplicado al mapa: Estilo "oscuro". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsNight** | El estilo aplicado al mapa: al estilo de la "noche". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsRetro** | El estilo aplicado al mapa: Estilo "Retro". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsSilver** | El estilo aplicado al mapa: estilo "plateado". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mcsStandard** | El estilo aplicado al mapa: Estilo "estándar". | [MapGetStyle](../WDLang3/1000025557.md), [MapChangeStyle](../WDLang3/1000025556.md) |
| **mkBottom** | Modo para alinear la Marker cuando se muestra en un mapa: Marker se muestra en la parte inferior de la posición. | Tipo [Marker](../WDLang3/1000019940.md) |
| **mkCenter** | Modo para alinear la Marker cuando se muestra en un mapa: Marker se muestra en la posición. | Tipo [Marker](../WDLang3/1000019940.md) |
| **mkLeft** | Modo para alinear la Marker cuando se muestra en un mapa: Marker se muestra a la izquierda de la posición. | Tipo [Marker](../WDLang3/1000019940.md) |
| **mkRight** | Modo para alinear la Marker cuando se muestra en un mapa: Marker se muestra a la derecha de la posición. | Tipo [Marker](../WDLang3/1000019940.md) |
| **mkTop** | Modo para alinear la Marker cuando se muestra en un mapa: Marker se muestra en la parte superior de la posición. | Tipo [Marker](../WDLang3/1000019940.md) |


## tapiCalledID( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar el número de Phone llamado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PhoneNum is string 
PhoneNum = tapiCalledID(CallID)
Message("Call in progress: " + PhoneNum + "...")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = tapiCalledID([<Call identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Llamada al número de Phone.

**`<Call identifier>: (Entero opcional)`**

Identificador de la llamada que se va a gestionar. Este identificador se define en el lenguaje WLanguage Procedure llamado por [tapiEscuchar](../WDLang3/3076029.md). 
La llamada actual se gestiona si no se especifica este parámetro. En el caso de una llamada saliente ([tapiDial](../WDLang3/3076013.md)), la llamada actual corresponde a la llamada saliente.

Las funciones de telefonía utilizan tecnología TAPI 2.0 y/o tecnología TAPI 3.1. Estas tecnologías pueden utilizarse en todos los sistemas.




## BTLEStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado actual de activación de Bluetooth en el dispositivo o pide que se le notifique cuando cambie el estado de activación. 

**Observaciones**: 

- Para activar o desactivar el Bluetooth en el dispositivo, utilice [BTActivate](../WDLang3/1000019359.md).

- Las funciones de Bluetooth no están disponibles en el emulador de Android.

- En Windows, utilice [BTStatus](../WDLang3/1000019360.md) para get el estado del Bluetooth. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StatusBTLE is int
StatusBTLE = BTLEStatus()
SWITCH StatusBTLE
	CASE btEnabled
		// Process 
	OTHER CASE
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estado actual de activación del Bluetooth en el dispositivo

`<Result> = BTLEStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de activación del Bluetooth LE: 


|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |
| *btError* | No se pudo recuperar el estado de activación del Bluetooth. |
| *btNotInitialized* | No se ha realizado la inicialización de Bluetooth Low Energy. |




<a name="SYNTAX2"></a>

### Habilitar el Notification cuando la activación del Bluetooth cambia

`BTLEStatus(<WLanguage procedure>)`
---

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLangage Procedure ("callback" Procedure) que se llamará cuando cambie el estado de activación del Bluetooth en el dispositivo. Este procedimiento tiene el siguiente formato : 


```txt
PROCEDURE <Procedure name> (<New status>, <Former status>)
```
donde: 

- &lt;New Status&gt; es un número entero que corresponde al nuevo estado de activación de Bluetooth en el dispositivo.. 

- &lt;El estado anterior&gt; es un número entero que corresponde al estado de activación de Bluetooth en el dispositivo antes de cambiar de estado.


Estos números enteros corresponden a las siguientes constantes: 


|   |   |
| --- | --- |
| *btActivationInProgress* | El Bluetooth se está activando en el dispositivo. |
| *btDeactivationInProgress* | El Bluetooth está siendo desactivado en el dispositivo. |
| *btDisabled* | El Bluetooth está desactivado en el dispositivo. |
| *btEnabled* | El Bluetooth está activado en el dispositivo. |

Los parámetros de este Procedure son opcionales. No hay necesidad de pasarle parámetros a este Procedure. De hecho, estos parámetros se rellenan automáticamente durante cada llamada.

**Si este parámetro corresponde a una cadena vacía (""")**las notificaciones ya no se enviarán cuando cambie el status. La función no tiene efecto si no se ha realizado ninguna solicitud de Notification.

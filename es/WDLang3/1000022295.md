


## &lt;emailNotesSession Variable&gt;.NbMensaje (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de mensajes entrantes en Notas. Este número de mensajes evoluciona en función de los correos electrónicos recibidos o eliminados por [EmailDeleteMessage](../WDLang3/3032027.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySession is emailNotesSession

// Loop for reading all the incoming messages found
// in Notes
I is int
nbMSG is int = MySession.NbMessage()
FOR I = 1 TO nbMSG
	EmailReadMessage(UserName, I)
	IF Email.Out <> True THEN
		CU.NAME = Email.Sender
		CU.MESSAGE_TXT = Email.Message
		...
	ELSE
		Info("This message was deleted")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Session>.NbMessage()`
---

**`<Result>: (Integro)`**



- Número de mensajes en la bandeja de entrada de Notes al momento de llamar a la función **&lt;emailNotesSession Variable&gt;.NbMensaje**,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Session>: (Variable de tipo emailNotesSession)`**

Nombre de la [emailNotesSession](../WDLang3/1000018768.md) Variable correspondiente a la sesión Email a utilizar.

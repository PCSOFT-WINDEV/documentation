


## &lt;LegendFont (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la fuente utilizada para la leyenda del gráfico.

Ejemplo de leyenda en un gráfico: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Legende.gif)




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the font used for the legend of CHART_MyChart.
// The identifier of font used is 7.
CHART_MyChart.LegendFont(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Definir la fuente usada para la leyenda del gráfico

`<Chart control>.LegendFont(<Font used>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar en la ventana, página o editor de reportes.


**`<Font used>: (tipo de letra)`**

Fuente a utilizar. Este parámetro corresponde a una variable de tipo [tipo de letra](../Motscles/1514045.md). Las características de la fuente se definieron mediante la función [FontCreate](../WDLang1/3060002.md) o las [propiedades de fuentes](../Proprietes/2515045.md).


<a name="SYNTAX2"></a>

### Definir la fuente utilizada para la leyenda del gráfico (sintaxis conservada para compatibilidad con versiones anteriores)

`<Chart control>.LegendFont(<Font identifier>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico a utilizar en la ventana, página o editor de reportes.


**`<Font identifier>: (Integro)`**

Identificador de fuente, definido por [grCreateFont](../WDLang3/3042001.md). Un error de WLanguage ocurre si esta fuente no existe.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Manejo de leyendas
<a name="manejo_leyendas_ELTPARAGRAPHE000288"></a>Las etiquetas de la serie set corresponden a la leyenda de la carta.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=EtiquetteSerie.gif)
 Las etiquetas de las series están definidas por [&lt;Tabla&gt;.SeriesLabel](../WDLang3/1000024073.md). 

La fuente de la leyenda se puede modificar con la función **&lt;LegendFont**.  

[&lt;Leyenda](../WDLang3/1000024089.md) indica la presencia y la posición de la leyenda (es decir, todas las etiquetas de las series).
<a name="NOTE0_2"></a>


### Inicialización de fuentes
<a name="inicializacion_fuentes_ELTPARAGRAPHE000315"></a>

La fuente Default utilizada para las leyendas de los gráficos tiene las siguientes características:

- Nombre fuente: Arial, 

- envergadura: 14,

- Estilo: Normal, 

- Color: Black .






Los datos definidos con la función **&lt;LegendFont** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

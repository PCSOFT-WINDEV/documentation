
## TwoFactorAuthenticationGenerateLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera un enlace para registrar una cuenta en una aplicación de autenticación. Cuando se asocia a un control Código de barras, este enlace mostrará un código QR que puede ser escaneado en la app de autentificación instalada en el Phone. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
let sURL = TwoFactorAuthenticationGenerateLink(sTOTPKey, sEmail, sAppName)
BAC_MyQRCode = sURL
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TwoFactorAuthenticationGenerateLink(<TOTP key> , <Account> , <Application>)`
---

**`<Result>: (Cadena de caracteres)`**

URL utilizada para crear una cuenta en una aplicación de autenticación.

**`<TOTP key>: (Cadena de caracteres)`**

Cadena ANSI que contiene la clave TOTP del usuario a autenticar. Esta clave TOTP fue generada con [DoubleAuthGenerateKey](../WDLang3/1000025974.md). 

**`<Account>: (Cadena de caracteres)`**

Nombre de la cuenta a registrar (generalmente se utiliza el correo electrónico Address).

**`<Application>: (Cadena de caracteres)`**

Nombre de la aplicación que se registrará en la aplicación de autenticación.

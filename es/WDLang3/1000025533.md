


## MapModifyImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una Image dibujada en el mapa que se muestra en una control Mapa.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the width of an image displayed in a Map control
MyImage is MapImage

MyImage.Width = 100
MapModifyImage(MAP_MyMap, MyImage)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MapModifyImage(<Map control> , <Image>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la Image fue modificada,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Map control>: (Nombre del control)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Mapa a la que pertenece la actual Event.

**`<Image>: (MapImage Variable)`**

Nombre de la [MapImage](../WDLang3/1000025523.md) Variable que corresponde al Image a modificar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La Image debe haber sido añadida a la control Mapa antes de que pueda ser modificada.

- Para añadir una nueva Image que se dibujará en la control Mapa, utilice [MapAddImage](../WDLang3/1000025532.md).

- Para eliminar una Image dibujada en la control Mapa, utilice [MapDeleteImage](../WDLang3/1000025534.md).

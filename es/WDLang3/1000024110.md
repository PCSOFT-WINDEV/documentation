


## &lt;Gráfico&gt;.ScatterPointSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el tamaño de los puntos en un gráfico de "dispersión". 

**Recordatorio**: El tamaño de un punto en un gráfico de "Dispersión" es igual a 1 píxel por Default.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los gráficos](../WDChamp/3042060.md). 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the size of points in "CHART_MyChart".
// This chart is a "Scatter" chart.
CHART_MyChart.ScatterPointSize(3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Chart control>.ScatterPointSize(<Size>)`
---

**`<Chart control>: (Nombre del control)`**

Nombre de la control Gráfico utilizada (se encuentra en el editor de ventanas, editor de páginas o editor de reportes).


**`<Size>: (Integro)`**

Tamaño de un punto (en píxeles).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tamaño de un punto
<a name="tamano_punto_ELTPARAGRAPHE000239"></a>El punto mostrado es un cuadrado cuyo tamaño del lado se calcula mediante la siguiente fórmula: (&lt;Tamaño&gt; 2)-1\*píxeles.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Graphe2.gif)

<a name="NOTE0_2"></a>


### Puntos en las cartas "Scatter
<a name="puntos_las_cartas_scatter_ELTPARAGRAPHE000250"></a>

- En las cartas "Scatter", un punto se define por sus coordenadas (coordenada X y coordenada Y).

- Los datos definidos con la función **&lt;Gráfico&gt;.ScatterPointSize** se tendrán en cuenta la próxima vez que se llame a la función [&lt;Dibujo](../WDLang3/1000024062.md).

- La función **&lt;Gráfico&gt;.ScatterPointSize** solo se utiliza para gráficos de "dispersión". Esta función no tiene ningún efecto con las otras clases de gráficos.


## SaaSAdminModifyUser( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un usuario de SaaS.

**Observaciones**: 

- Para utilizar esta función, es necesario haber iniciado previamente una sesión de gestión de SaaS con [SaaSAdminConnect](../WDLang3/1000019098.md).

- El administrador de SaaS gestiona todos los usuarios de todos los clientes. Un gestor de cuentas de cliente sólo puede modificar los usuarios de su propia cuenta de cliente.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyUser is saasUser
...
MyUser = SaaSAdminFindUser("mark@hybrid.com")
MyUser.Name = "mark.smith@hybrid.com"

SaaSAdminModifyUser(MyUser)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SaaSAdminModifyUser(<User>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<User>: (saasUser Variable)`**

Nombre del [saasUser](../WDLang3/1000019083.md) Variable correspondiente al usuario que debe ser modificado en la base de datos SaaS.

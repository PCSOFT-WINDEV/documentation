


## &lt;sshSession Variable&gt;.ConnectShell (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia una nueva sesión SSH de tipo "Shell. Esta sesión puede ser cerrada por [&lt;sshSession Variable&gt;.DisconnectShell](../WDLang3/1000021164.md).

**Observación**: Para enviar un comando al servidor SSH directamente, utilice [SSHCommand](../WDLang3/1000021143.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
cMySession is SSHSession
cMySession.Address = "127.0.0.1"
cMySession.Port = 22
cMySession.User = "login"
cMySession.UserPassword = "pass"
IF cMySession.ConnectShell() THEN
	Info("Session started")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <SSH session>.ConnectShell()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la sesión se abrió correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<SSH session>: (Variable de tipo sshSession)`**

Nombre de la [sshSession](../WDLang3/1000021157.md) Variable que contiene los parámetros de conexión de la sesión SSH a abrir.

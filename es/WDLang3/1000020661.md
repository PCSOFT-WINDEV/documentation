
## AppointmentListCalendar( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lista los calendarios disponibles en el dispositivo móvil (Android/iOS).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000020712&name=appointmentlistcalendar_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Fill a List Box control with the calendars available on the device
arrCalendar is array of Calendar 
arrCalendar = AppointmentListCalendar()
FOR EACH Calendar of arrCalendar
	ListAdd(LIST_List1, Calendar.Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Listado de los calendarios disponibles

`<Result> = AppointmentListCalendar()`
---

**`<Result>: (Array de las variables Calendar)`**

Variables Array de [Calendario](../WDLang3/1000020663.md) inicializadas con la lista de calendarios disponibles. 
Si se produce un error, se vacía el array y la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


<a name="SYNTAX2"></a>

### Enumerar los calendarios que corresponden a los filtros especificados

`<Result> = AppointmentListCalendar(<Filter> , <Sought value>)`
---

**`<Result>: (Array de las variables Calendar)`**

Variables Array de [Calendario](../WDLang3/1000020663.md) inicializadas con la lista de calendarios disponibles. 
Si se produce un error, se vacía el array y la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

**`<Filter>: (Constante de tipo Integer)`**

Property en la que se debe filtrar la lista de calendarios: 


|   |   |
| --- | --- |
| *aptAccountName* | Filtros sobre el nombre de la cuenta a la que pertenece el Calendar. Se realiza una búsqueda de "Contiene". |
| *aptAccountType* | Filtros sobre el tipo de cuenta a la que pertenece el Calendar. Se realiza una búsqueda de "Contiene". |
| *aptIdentifier* | Filtros en los identificadores de los calendarios. Se realiza una búsqueda por coincidencia exacta. |
| *aptName* | Filtros sobre el nombre de la Calendar. Se realiza una búsqueda de "Contiene". |



**`<Sought value>: (Cadena de caracteres)`**

El valor buscado en las propiedades filtradas de Calendar. Esta búsqueda no distingue entre mayúsculas y minúsculas.




## Funciones MQTT (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar los clientes MQTT (sintaxis del prefijo):



|   |   |
| --- | --- |
| [&lt;mqttSession Variable&gt;.Conectar](../WDLang3/1000023084.md) | Se utiliza para conectarse a un servidor MQTT (también llamado "Broker"). |
| [&lt;mqttSession Variable&gt;.Darse de baja](../WDLang3/1000023086.md) | Le permite darse de baja de un MQTT Notification. |
| [&lt;mqttSession Variable&gt;.Desconectar](../WDLang3/1000023085.md) | Allows you to disconnect from a MQTT server (also called "Broker"). |
| [&lt;mqttSession Variable&gt;.Publicar](../WDLang3/1000023091.md) | Publica un mensaje en el broker MQTT. |
| [&lt;mqttSession Variable&gt;.Subscribe](../WDLang3/1000023083.md) | Se utiliza para suscribirse a un MQTT Notification. |
| [&lt;mqttSession Variable&gt;.UnsubscribeAll](../WDLang3/1000023090.md) | Le permite darse de baja de todas las suscripciones existentes. |


## DNSDeleteAddress( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una relación Nombre/IP Address de un servidor DNS.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DNSResult is boolean

// Deletes an address
DNSResult = DNSDeleteAddress("testAp.towerqual.local.", "192.168.1.1", "Andromeda_2")
IF DNSResult = False THEN
	Error(ErrorInfo(errFullDetails))
END

// Deletes all addresses
DNSResult = DNSDEleteAddress("testAp.towerqual.local.", "" , "Andromeda_2")
IF DNSResult = False THEN
	Error(ErrorInfo(errFullDetails))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DNSDeleteAddress(<Name> [, <IP address> [, <DNS server> [, <User> [, <Password>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Name>: (Cadena Unicode)`**

Nombre a eliminar del DNS. Este parámetro tiene el siguiente formato: &lt;Nombre del ordenador&gt; + "." + &lt;Dominio&gt;. Ejemplo: "testdoc.pcsoft.com"

**`<IP address>: (Cadena de caracteres opcional)`**

IP Address associated con el nombre a borrar. Esta IP Address puede ser una IP Address versión 4 (en el formato xxx.xxx.xxx.xxx) o versión 6 (en el formato aaaa:bbbb:cccc:dddd:eeee:ffff:gggg:hhhh:iiii:jjjjj).

Si este parámetro corresponde a una cadena vacía (""), todas las asociaciones del nombre con una IP Address serán borradas. 

**`<DNS server>: (Cadena de caracteres opcional)`**

Address del servidor DNS a actualizar. La asociación "IP Address/Name" será eliminada de este servidor.. Por Default, se utiliza el actual servidor DNS.

**`<User>: (Cadena de caracteres Unicode opcional)`**

Nombre del usuario utilizado para firmar la solicitud.

**`<Password>: (Cadena de caracteres Unicode opcional)`**

Contraseña associated con el usuario para firmar la solicitud.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es compatible con los servidores Linux y Windows DSN que no requieren peticiones firmadas.

- Esta función es compatible con los servidores Linux y Windows DSN que requieren peticiones firmadas.




## MapDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra lo siguiente de una control Mapa: 

- todos los marcadores,

- todas las formas,

- todas las imágenes.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all markers from the Map control
MapDeleteAll(MAP_MyMap)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`MapDeleteAll([<Map control>])`
---

**`<Map control>: (Nombre de control opcional)`**

Nombre de la control Mapa que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, se utilizará la control Mapa a la que pertenece la actual Event.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Gestión de marcadores: 

	- Para eliminar una Marker específica de una control Mapa, utilice [MapDeleteMarker](../WDLang3/1000019795.md).

	- Para añadir una nueva Marker a una control Mapa, utilice [MapAddMarker](../WDLang3/1000019794.md).

	- Para cambiar una Marker en una control Mapa, utilice [MapModifyMarker](../WDLang3/1000019796.md).




- Gestión de itinerarios: Para eliminar un itinerario concreto, utilice [MapDeleteItinerary](../WDLang3/1000020384.md).

- Imágenes: 

	- Para eliminar una Image específica de una control Mapa, utilice [MapDeleteImage](../WDLang3/1000025534.md). 

	- Para añadir una nueva Image a una control Mapa, utilice [MapAddImage](../WDLang3/1000025532.md). 

	- Para modificar una Image en una control Mapa, utilice [MapModifyImage](../WDLang3/1000025533.md). 




- Formas: 

	- Para eliminar una forma específica de una control Mapa, utilice [MapDeleteShape](../WDLang3/1000025510.md). 

	- Para añadir una nueva forma a una control Mapa, utilice [MapAddShape](../WDLang3/1000025509.md). 

	- Para cambiar una forma en una control Mapa, utilice [MapModifyShape](../WDLang3/1000025511.md).

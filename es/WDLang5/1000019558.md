


## Funciones de gestión del código de barras
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los códigos de barras: 



|   |   |
| --- | --- |
| [BCCalculateControlCharacter](../WDLang5/1000019909.md) | Devuelve el código de cheque de un código de barras. |
| [BCCapture](../WDLang5/1000019456.md) | Decodifica la información almacenada en un código de barras utilizando la cámara del dispositivo (Android, iPhone, iPad). |
| [BCDecode](../WDLang5/1000019633.md) | Decodifica la información almacenada en un código de barras QR Code. |
| [BCToImage](../WDLang5/1000024060.md) | Genera la Image de un código de barras. |





También puede imprimir los códigos de barras programándolos mediante la siguiente función: 



|   |   |
| --- | --- |
| [iPrintBarCode](../WDLang5/3046020.md) | Imprime un código de barras (dentro de un rectángulo). |

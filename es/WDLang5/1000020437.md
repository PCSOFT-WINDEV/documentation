


## &lt;Variable gglConnection&gt;.GetDocument (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Descarga un Document del servicio de Google Docs.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Cnt is gglConnection
...
IF Cnt.Connect() =False THEN
	Error(ErrorInfo())
END
...
arrDoc is array of gglDocument 
arrDoc = Cnt.ListDocument()
Cnt.GetDocument(arrDoc[1], "c:\temp\document.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Google connection>.GetDocument(<Document to download> , <Destination path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se recuperó el documento,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Google connection>: (Variable de tipo gglConnection)`**

Nombre de la variable de tipo [gglConnection](../WDLang5/1000018009.md) que se utilizará. Esta conexión fue validada por [&lt;gglConnection Variable&gt;.Conectar](../WDLang5/1000021455.md).

**`<Document to download>: (gglDocument Variable)`**

[gglDocument](../WDLang5/1000018001.md) variable correspondiente al document para descargar.

**`<Destination path>: (Cadena Unicode)`**

Nombre y ruta completo correspondiente al archivo a crear a partir del Document descargado. Si se especifica la extensión del archivo, éste se convertirá a este formato cuando se descargue.. 
El formato del archivo fuente se mantiene si no se especifica la extensión.

## Observaciones


### Posibles formatos de conversión según el formato de origen
<a name="posibles_formatos_conversion_segun_formato_origen_ELTPARAGRAPHE000197"></a>

Los formatos soportados dependen del tipo de archivo fuente. Para obtener más información, consulte [Documentación de Google](http://code.google.com/intl/fr-FR/apis/documents/docs/3.0/developers_guide_protocol.html#DownloadingDocsAndFiles). 
Por ejemplo: 

- Los formatos válidos para un texto Document son los siguientes: 

	- doc

	- html

	- jpeg

	- odisea

	- pdf 

	- png

	- rtf

	- txt

	- cremallera. En este caso, el archivo Zip contiene las imágenes (si es necesario) utilizadas en el Document así como los archivos HTML que contienen el texto del Document. 




- Los formatos válidos para los dibujos son los siguientes: 

	- jpeg

	- pdf

	- png

	- svg




- Los formatos válidos para las presentaciones son los siguientes: 

	- pdf

	- png

	- ppt

	- txt




- Los formatos válidos para las hojas de cálculo son los siguientes:  

	- xls

	- csv: sólo se devuelve la primera hoja de cálculo en formato CSV. 

	- pdf

	- ods

	- tsv: sólo se devuelve la primera hoja de cálculo en formato TSV.

	- html

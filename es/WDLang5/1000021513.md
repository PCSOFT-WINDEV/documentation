


## Funciones para la gestión de archivos XLS (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar archivos XLS (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;Variable xlsDocument&gt;.DeleteWorksheet](../WDLang5/1000021512.md) | Borra una hoja de trabajo del Excel Document. |
| [&lt;Variable xlsDocument&gt;.MergeCell](../WDLang5/1410089568.md) | Combina las celdas especificadas en un documento Excel. |
| [&lt;Variable xlsDocument&gt;.RecalculateAll](../WDLang5/1410089739.md) | Recalcula todas las fórmulas de un documento Excel (XLSX). |
| [&lt;Variable xlsDocument&gt;.UnmergeCell](../WDLang5/1410089571.md) | Anula la combinación de una celda determinada. |
| [&lt;xlsDocument Variable&gt;.AddWorksheet](../WDLang5/1000021509.md) | Añade o inserta una nueva hoja de trabajo en un Excel Document. |
| [&lt;xlsDocument Variable&gt;.Cerrar](../WDLang5/1000021510.md) | Cierra un archivo XLS. |
| [&lt;xlsDocument Variable&gt;.Guardar](../WDLang5/1000021511.md) | Guarda un XLS Document. |
| [GetMerge](../WDLang5/1410089569.md) | Determina si una celda está combinada con otras y obtiene el rango de celdas combinadas. |

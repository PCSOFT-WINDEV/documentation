


## GglDisconnect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desconecta al usuario de los servicios de Google utilizados.. Durante la próxima reconexión, se volverán a solicitar las autorizaciones de relacionado a los servicios de Google.. 

**Observación**: Esta función es opcional. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Cnt is gglConnection
// ...
// use the Google connection
// ...
GglDisconnect(Cnt)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GglDisconnect(<Connection> [, <Google service>])`
---

**`<Connection>: (Variable de tipo gglConnection)`**

Nombre de la [gglConnection](../WDLang5/1000018009.md) Variable correspondiente a la conexión con manipular. 

**`<Google service>: (Constante o combinación de constantes de tipo Integer)`**

Servicio de Google para desconectar: 


|   |   |
| --- | --- |
| *gglServiceCalendar* | "Servicio"Google Calendar: Google calendar |
| *gglServiceContacts* | "Servicio"Contactos de Google: Gestión de contactos de Google |
| *gglServiceDocuments* | "Servicio"Lista de documentos de Google: Gestión de documentos |
| *gglServicePicasa* | "Picasa" service |


Si no se especifica este parámetro, se desconectarán todos los servicios..

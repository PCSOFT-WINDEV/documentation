
## iPrintingNation( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el idioma actual para la siguiente impresión (impresión programada o impresión reporte). 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El idioma de visualización de la presentación preliminar no se modifica.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print a report in Spanish
iPrintingNation(nationSpanish)
iPrintReport(Invoice)
```
<a name="Example2"></a>

```wl
// Programmed print in Italian
iPrintingNation(nationItalian)
iPrintWord("<<<multilingual text with flag>>>")
iEndPrinting()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`iPrintingNation(<Nation> [, <Charset> [, <Language> [, <Sub-language> [, <Sort mode>]]]])`
---

**`<Nation>: (Constante de tipo Integer)`**

Nación a utilizar. Para obtener más información sobre estas constantes, consulte [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).

**`<Charset>: (Constante opcional de tipo Integer)`**

El personaje set para usar: 


|   |   |
| --- | --- |
| *charsetAnsi* | Caracteres romanos en estándar ANSI |
| *charsetArabic* | Caracteres arábigos |
| *charsetBaltic* | Caracteres bálticos |
| *charsetChinese* | Caracteres chinos (República Popular China) |
| *charsetDefault* | Utiliza el conjunto de caracteres predeterminado del equipo. No se fuerza ningún conjunto de caracteres. |
| *charsetEastEurope* | Caracteres de Europa del Este (polaco, etc.) |
| *charsetGreek* | Caracteres griegos |
| *charsetHebrew* | Caracteres hebreos |
| *charsetJapanese* | Caracteres japoneses |
| *charsetKorean* | Caracteres coreanos |
| *charsetMac* | Conjunto de caracteres Mac Roman (utilizado en versiones anteriores de Macintosh). |
| *charsetOccidental* | Caracteres romanos en estándar ANSI |
| *charsetRussian* | Caracteres rusos |
| *charsetThai* | Caracteres tailandeses |
| *charsetTraditionalChinese* | Caracteres chino tradicional (República de Taiwán) |
| *charsetTurkish* | Caracteres turcos |
| *charsetUTF8* | Se utiliza para gestionar los países con dos caracteres set (Hong Kong) y los países sin carácter set definidos en Windows (georgiano y armenio). |
| *charsetVietnamese* | Caracteres vietnamitas |


Si no se especifica este parámetro, se utiliza el carácter set associated de Default al Country.  

**`<Language>: (Constante opcional de tipo Integer)`**

Idioma a utilizar. Para obtener más información sobre estas constantes, consulte [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).
Este idioma debe corresponder al parámetro &lt;Charset&gt; especificado.. De lo contrario, se producirá un error de WLanguage.
Este parámetro se utiliza para gestionar las características específicas de cada Country.

Si no se especifica este parámetro, se utiliza el lenguaje associated de Default al Country. 

**`<Sub-language>: (Entero opcional)`**

Subidioma a utilizar. Para más detalles sobre los sublenguajes, véase [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).
Este subidioma debe corresponder al parámetro &lt;Charset&gt; especificado.. De lo contrario, se producirá un error de WLanguage.
Este parámetro se utiliza para gestionar las características específicas de cada Country.

Si no se especifica este parámetro, se utiliza el sub-lenguaje associated de Default al Country.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La nación utilizada para la impresión corresponde al parámetro &lt;Nación&gt; cuando se llama a **iPrintingNation**. Esta nación se utiliza hasta que se llama a una función de fin de impresión (por ejemplo, [iEndPrinting](../WDLang5/3046053.md), [iPrintReport](../WDLang5/3046032.md) o [iSequencingPrint](../WDLang5/1000019903.md)).

- Si se llama a la función [Nación](../WDLang1/3054014.md) durante la impresión, se cancela la nación especificada inicialmente con la función **iPrintingNation**: la impresión vuelve a la nación especificada por [Nación](../WDLang1/3054014.md). En ese caso: 

	- el texto de los informes permanece en la lengua especificada con **iPrintingNation**. 

	- las cadenas multilingües se muestran en el idioma especificado con [Nación](../WDLang1/3054014.md). 




- La función **iPrintingNation** también permite modificar la nación utilizada para un grupo de reportes (funciones [iGroupAñadir](../WDLang5/1000020580.md), [iGroupImprimir](../WDLang5/1000020582.md), [iGroupConfigurar](../WDLang5/1000020581.md)).

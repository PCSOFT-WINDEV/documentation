
## sfMergeRequest (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfMergeRequest** se utiliza para describir los elementos a fusionar durante la llamada a [SFMerge](../WDLang5/1000018509.md). Las características de los elementos que se van a fusionar pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

<a name="NOTE0_1"></a>

## Observaciones




### Propiedades específicas de las variables sfMergeRequest
<a name="propiedades_especificas_las_variables_sfmergerequest_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un elemento a fusionar:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| MasterRecord | [sfObject](../WDLang5/1000018439.md) | Record en la que se fusionarán los datos. |
| RecordToMergeID | Cadena de caracteres | Lista de identificadores de la Fuerza de Ventas de los Record cuyos datos se fusionarán en el maestro Record. Los diferentes identificadores están separados por caracteres CR (Carriage Return). |




## Funciones de gestión de archivos XLS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar archivos XLS:



|   |   |
| --- | --- |
| [xlsAddWorksheet](../WDLang5/1000017285.md) | Añade o inserta una nueva hoja de trabajo en un Excel Document. |
| [xlsCerrar](../WDLang5/3080015.md) | Cierra un archivo XLS. |
| [xlsColumnTitle](../WDLang5/3080012.md) | Retrieves the title of a column found in an XLS file. |
| [xlsColumnType](../WDLang5/3080019.md) | Devuelve el tipo de datos introducidos en una columna de un fichero XLS. |
| [xlsCoordinatesToName](../WDLang5/1000021500.md) | Recupera el nombre de una celda, hoja de cálculo XLS o control Hoja de cálculo según sus coordenadas. |
| [xlsCurrentWorksheet](../WDLang5/3080022.md) | Se utiliza para averiguar y modificar la hoja de trabajo actual en un archivo XLS. |
| [xlsData](../WDLang5/3080021.md) | Recupera el contenido de una celda encontrada en un archivo XLS. |
| [xlsDeleteWorksheet](../WDLang5/1000017288.md) | Borra una hoja de trabajo del Excel Document. |
| [xlsGetMerge](../WDLang5/1410089567.md) | Determina si una celda está combinada con otras y obtiene el rango de celdas combinadas. |
| [xlsMergeCell](../WDLang5/1410089537.md) | Combina las celdas especificadas en un documento Excel. |
| [xlsMsgError](../WDLang5/3080016.md) | Returns the caption of the last error caused by an XLS function. |
| [xlsNameToCoordinates](../WDLang5/1000021501.md) | Devuelve las coordenadas de una celda, archivo XLS u hoja de cálculo control según su nombre. |
| [xlsNbColumn](../WDLang5/3080014.md) | Returns the number of columns found in a worksheet of an XLS file. |
| [xlsNbColumns](../WDLang5/3080007.md) | xlsNbColumns se mantiene sólo para la compatibilidad con versiones anteriores. |
| [xlsNbHoja de trabajo](../WDLang5/3080023.md) | Devuelve el número de hojas de trabajo que se encuentran en un archivo XLS. |
| [xlsNbRow](../WDLang5/3080011.md) | Devuelve el número de líneas que se encuentran en un archivo XLS. |
| [xlsNbRows](../WDLang5/3080013.md) | xlsNbRows se mantiene sólo para compatibilidad con versiones anteriores. |
| [xlsOpen](../WDLang5/3080017.md) | Abre un archivo de Excel (archivos xls o xlsx). |
| [xlsRecalculateAll](../WDLang5/1410089617.md) | Recalcula todas las fórmulas de un documento Excel (XLSX). |
| [xlsSave](../WDLang5/1000018057.md) | Guarda un XLS Document. |
| [xlsUnmergeCell](../WDLang5/1410089538.md) | Anula la combinación de un rango de celdas. |
| [xlsVersion](../WDLang5/3080018.md) | Devuelve la versión de Excel que se utilizó para grabar el fichero. |

## Otras funciones de relacionado a Excel
<a name="otras_funciones_relacionado_excel_ELTTEXTE000295"></a>



|   |   |
| --- | --- |
| [AAFDisable](../WDLang1/1000022018.md) | Desactiva la exportación del contenido control Tabla o de la selección de contenidos a Excel o a OpenOffice Calc (según el software instalado en el ordenador del usuario). |
| [AAFExecute](../WDLang1/1000022099.md) | Exporta el contenido de control Tabla o la selección realizada en control Tabla a Excel o a OpenOffice Calc (dependiendo del software instalado en el ordenador del usuario). |
| [HImportXLS](../WDLang4/1000021197.md) | Importa un archivo XLS en un archivo de datos en formato HFSQL Classic o Cliente/Servidor (archivo de datos descrito en el editor de análisis o declarado por [HDeclare](../WDLang4/3044203.md), [HDeclareExternal](../WDLang4/3044204.md) o [HDescribeFile](../ErrCompil/1000001036.md)). |
| [LooperToExcel](../WDLang2/3083030.md) | Crea un archivo de Excel con los datos de una control Looper (basada en un archivo de datos o rellenada programáticamente). |
| [TablaToExcel](../WDLang1/3074033.md) | Crea un archivo de Excel con los datos de una Tabla o control Tabla TreeView (basada en un archivo de datos o poblada programáticamente). |

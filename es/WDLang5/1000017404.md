
## GglGetStaticMap( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el mapa de una ubicación específica a través del servicio Google Maps..  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000017572&name=gglgetstaticmap_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
MyGoogleKey is string
MyGoogleKey = "Google key example"

MyImage is Image
MyImage = GglGetStaticMap(MyGoogleKey, 40.714728, 73.998672, 14, 400, 400)
```


<a name="Example2"></a>



```wl
// Code used to display a journey

MyImage is Image
```

## Sintaxis
<a name="SYNTAX3"></a>

### Recuperación de un mapa con configuración de centrado

`<Result> = GglGetStaticMap(<Google key> , <Latitude> , <Longitude> , <Zoom> , <Image width> , <Image height> [, <Type of map> [, <Advanced settings>]])`
---

**`<Result>: (Imagen Variable)`**

Nombre de la [Image](../WDLang1/1000019650.md) Variable que contiene el mapa. El mapa se devuelve a las dimensiones especificadas.

**`<Google key>: (Cadena de caracteres)`**

Clave de autenticación para el servicio Google Maps. Esta llave es suministrada por Google.

**Observación**: La clave de Google es necesaria sólo para un uso avanzado (uso con cargo para más de 25.000 solicitudes al día).

**`<Latitude>: (Real)`**

Latitud del punto central del mapa.

**`<Longitude>: (Real)`**

Longitud del punto central del mapa.

**`<Zoom>: (Integro)`**

Nivel de zoom. Este parámetro puede corresponder a:

- un número entero incluido entre 0 (zoom más lejano) y 19 (zoom muy cercano, escala de un edificio por ejemplo).

- la constante *gglAutoZoom*. En este caso, el zoom se elige automáticamente en función de los marcadores o de la zona definida en &lt;Ajustes avanzados&gt;.. Se produce un error de WLanguage si se utiliza esta constante sin &lt;Ajustes avanzados&gt;.




**`<Image width>: (Integro)`**

Ancho (en píxeles) de la Image del mapa para recuperar. El ancho máximo es de set a 640 píxeles. Este límite está definido por el servicio Google Maps en la fecha de publicación de esta página.

**`<Image height>: (Integro)`**

Altura (en píxeles) de la Image del mapa a recuperar. La altura máxima es de set a 640 píxeles. Este límite está definido por el servicio Google Maps en la fecha de publicación de esta página.

**`<Type of map>: (Constante de cadena de caracteres opcional)`**

Tipo de mapa a recuperar. Este parámetro puede corresponder a una de las siguientes constantes:



|   |   |
| --- | --- |
| *gglHybrid* | Combina el mapa de carreteras y la vista de satélite. |
| *gglMobileDevice* | Mapa optimizado para las restricciones específicas de la visualización en un dispositivo móvil |
| *gglRoadmap*<br>(Valor predeterminado) | Hoja de ruta. |
| *gglSatellite* | Vista satelital. |
| *gglTerrain* | Mapa geográfico que muestra el relieve y la vegetación. |



**`<Advanced settings>: (Opcional gglMapParameter Variable)`**

Nombre de la [gglMapParameter](../WDLang5/1000017505.md) Variable que contiene los ajustes avanzados del mapa. Si no se especifica este parámetro, el mapa estará en formato Gif y no contendrá ningún Marker, ningún ruta y ninguna zona.

## Observaciones




### Guardar el mapa de Google
<a name="guardar_mapa_google_ELTPARAGRAPHE000231"></a>

Para guardar el mapa de Google en un archivo, puede hacerlo:

- utilice [dSaveImageBMP](../WDLang1/3029026.md), [dSaveImageGIF](../WDLang1/3029040.md), [dSaveImageJPEG](../WDLang1/3029038.md) y [dSaveImagePNG](../WDLang1/3029027.md): el Image que se encuentra en el control Imagen se guarda en el formato solicitado.

- utilizar directamente [fSaveText](../WDLang1/3036046.md). Por ejemplo:
	
	```wl
	fSaveText("MyFile.gif", GglGetStaticMap(40.714728, 73.998672, 14, 400, 400))
	```




<a name="NOTE0_2"></a>


### Clave para Google Static Map
<a name="clave_para_google_static_map_ELTPARAGRAPHE000258"></a>



Para get la clave requerida para usar el servicio de Google Maps, vaya a la siguiente Address: 

[http://code.google.com/apis/maps/signup.html](http://code.google.com/apis/maps/signup.html). 

Atención: la Address de esta página puede haber sido modificada desde la fecha de publicación de esta página.
<a name="NOTE0_3"></a>


### Posibles casos de error
<a name="posibles_casos_error_ELTPARAGRAPHE000275"></a>

El servicio de Google Maps puede no ser capaz de devolver un Image válido según la ubicación o el zoom solicitado.

No se devuelve ningún dato si se superan las limitaciones definidas por el servicio de Google Maps (tamaño del Image demasiado grande, demasiados puntos en un ruta, etc.). Se devuelve un error "HttpBadRequest".

Para más detalles, consulte la documentación sobre las variables [gglMapParameter](../WDLang5/1000017505.md).


## rssSource (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssSource** permite describir la fuente de la entrada de un flujo RSS. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.  

Este tipo de Variable es el que utiliza [Tipo rssEntry](../WDLang5/1000017792.md).  

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssSource**. De hecho, las propiedades de una variable de tipo [Tipo rssEntry](../WDLang5/1000017792.md) permiten manipular directamente las propiedades de una variable de tipo **rssSource** (especialmente la propiedad .Source).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

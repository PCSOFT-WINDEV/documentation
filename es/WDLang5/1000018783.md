


## Funciones de Salesforce (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las bases de datos de Salesforce (sintaxis del prefijo):



|   |   |
| --- | --- |
| [&lt;sfConnection Variable&gt;.Añadir](../WDLang5/1000018753.md) | Crea una Record en un archivo de una base de datos de Salesforce. |
| [&lt;sfConnection Variable&gt;.Borrar](../WDLang5/1000018776.md) | Elimina registros de una base de datos de Salesforce. |
| [&lt;sfConnection Variable&gt;.Buscar](../WDLang5/1000018772.md) | Realiza una búsqueda entre los registros que se encuentran en una base de datos de Salesforce. |
| [&lt;sfConnection Variable&gt;.Conectar](../WDLang5/1000018756.md) | Le permite autenticarse junto a una plataforma de Salesforce. |
| [&lt;sfConnection Variable&gt;.Desconectar](../WDLang5/1000018760.md) | Desconecta: <br><br>- la plataforma de Salesforce. <br><br>- la sesión actual de Salesforce.<br><br><br> |
| [&lt;sfConnection Variable&gt;.EjecutarProceso](../WDLang5/1000018766.md) | Solicita la ejecución de una Process específica . |
| [&lt;sfConnection Variable&gt;.EmptyRecycleBin](../WDLang5/1000018778.md) | Borra los registros especificados de la papelera de reciclaje de Salesforce. |
| [&lt;sfConnection Variable&gt;.ExecuteQuery](../WDLang5/1000018762.md) | Ejecuta una consulta SOQL (Salesforce Object Query Language) en una base de datos de Salesforce. |
| [&lt;sfConnection Variable&gt;.ForwardQuery](../WDLang5/1000018754.md) | Continúa ejecutando una consulta SOQL (Salesforce Object Query Language) en los registros de una base de datos de Salesforce. |
| [&lt;sfConnection Variable&gt;.Fusión](../WDLang5/1000018769.md) | Fusiona los registros de Salesforce (también llamados objetos de Salesforce). |
| [&lt;sfConnection Variable&gt;.Leer](../WDLang5/1000018770.md) | Recupera los registros de Salesforce de sus identificadores. |
| [&lt;sfConnection Variable&gt;.SeekDeleted](../WDLang5/1000018774.md) | Busca y recupera los registros (objetos de Salesforce) borrados durante el período especificado. |
| [&lt;sfConnection Variable&gt;.SeekModified](../WDLang5/1000018773.md) | Busca y recupera los registros (objetos de Salesforce) modificados durante el período especificado. |
| [&lt;sfConnection Variable&gt;.Undelete](../WDLang5/1000018775.md) | Restaura los registros de Salesforce previamente eliminados. |
| [&lt;Variable sfConnection&gt;.ConvertLead](../WDLang5/1000018757.md) | Convierte los cables. |
| [&lt;Variable sfConnection&gt;.Modify](../WDLang5/1000018771.md) | Modifica los registros de un archivo definido en Salesforce. |




**Advertencia**: Antes de utilizar una función de relacionado para los servicios de Salesforce, le recomendamos encarecidamente que compruebe la licencia para utilizar este servicio.. Se pueden aplicar algunas restricciones. El contenido de las licencias puede cambiar con el tiempo.

PC SOFT no es en ningún caso responsable de la forma en que se utilizan las funciones de acceso nativas.. Por favor, asegúrese de cumplir con la licencia de proveedor de servicios.

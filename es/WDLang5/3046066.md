
## iVLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime una Line vertical en coordenadas especificadas. A diferencia de [iNewLine](../WDLang5/3046035.md), el final de Line se eleva al cuadrado (en lugar de redondearse). La Line se dibuja en la ubicación del cursor (configurada por [iXPos](../WDLang5/3046015.md)).

El trabajo de impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar el trabajo de impresión).

**Observación**: El origen (0,0) se encuentra en el Corner superior izquierdo de la hoja.. Tiene en cuenta los márgenes físicos de la impresora.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draws a vertical line between the line 5 (mm)
// and the line 10 (mm), with a 2-pica thickness
iVLine(5, 10, 2)
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iVLine(<Y1> , <Y2> [, <Thickness> [, <Color> [, <Style>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprimió la línea,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Y1>: (Real)`**

Coordenadas verticales del comienzo de Line (en milímetros). Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Y2>: (Real)`**

Coordenadas verticales del final de Line (en milímetros). Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Thickness>: (Real opcional)`**

Line espesor (en picas). Por Default, este espesor es igual a 1 (un Pica = 0,353 mm).
Para get un grosor de 1 píxel, &lt;Grosor&gt; debe ser igual a 0. Cuando el espesor aumenta, se distribuye proporcionalmente entre los dos lados de Line.

**`<Color>: (Entero opcional)`**

Color usado para el Line. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Style>: (Constante opcional de tipo Integer)`**

Tipo de Line.


|   |   |
| --- | --- |
| *iContinuous*<br>(valor predeterminado) | Línea continua |
| *iDashed* | Dashed Line (-----) |
| *iDotAndDash* | Punto y guión Line (-.-.) |
| *iDotted* | Línea de puntos (....) |


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Atención**: Para get un Line punteado, &lt;Espesor&gt; debe ser igual a 0. Si &lt;Grosor&gt; es mayor o igual a 1, la Line siempre será una Line sólida. Este parámetro puede ignorarse si no es compatible con el controlador de la impresora.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La Line se dibuja en la posición horizontal del cursor (configurada por [iXPos](../WDLang5/3046015.md)).

- La posición del cursor no se modifica cuando se dibuja el contorno.

- La combinación de líneas punteadas (constante *iDotted*) o guiones y puntos (*iDotAndDash*) con bordes gruesos puede causar efectos no deseados.

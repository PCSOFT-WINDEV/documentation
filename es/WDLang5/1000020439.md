<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000072"></a>
Google Docs y Google Spreadsheet son el procesador de textos en línea y la hoja de cálculo en línea propuesta por Google.

WINDEV y WEBDEV le permiten gestionar estos documentos a través de varias funciones de WLanguage. Es posible:

- Cargar documentos en el servidor de Google.

- Borrar documentos.

- Enumere los documentos y realice una búsqueda en los documentos.

- Recuperar documentos.




**Advertencia**: Antes de utilizar una función de relacionado para los servicios de Google, le recomendamos encarecidamente que compruebe la licencia para utilizar este servicio.. Se pueden aplicar algunas restricciones. El contenido de las licencias puede cambiar con el tiempo.

PC SOFT no es en ningún caso responsable de la forma en que se utilizan las funciones de acceso nativas.. Asegúrese de que cumple con la licencia del proveedor de servicios.

## ¿Cómo gestionar los documentos de Google?
<a name="¿como_gestionar_los_documentos_google_ELTTEXTE000111"></a>
Gestionar los documentos de Google que se encuentran en un servidor de Google:

1. Crear una cuenta de Google y un proyecto. Configurar el proyecto en la consola de Google para utilizar el servicio de gestión de documentos.. Para más detalles, consulte la ayuda sobre [&lt;gglConnection Variable&gt;.Conectar](../WDLang5/1000021455.md), "Requisitos previos para la autenticación OAuth 2.0". 

2. En el código de su aplicación, cree una Variable de tipo [gglConnection](../WDLang5/1000018009.md). Este Variable contiene las características de la conexión a su cuenta de Google.

3. Para gestionar los documentos encontrados en el servidor de Google, utilice una de las siguientes funciones:
	


|   |   |
| --- | --- |
| [&lt;gglConnection Variable&gt;.Borrar](../WDLang5/1000021461.md) | Borra un Google Document del servidor. |
| [&lt;gglConnection Variable&gt;.ListDocumento](../WDLang5/1000020436.md) | Recupera la lista de documentos disponibles en el servidor de Google para el usuario actual. |
| [&lt;gglConnection Variable&gt;.UploadDocumento](../WDLang5/1000020424.md) | Sube una Document al servidor de Google. |
| [&lt;Variable gglConnection&gt;.GetDocument](../WDLang5/1000020437.md) | Descarga un Document del servicio de Google Docs. |

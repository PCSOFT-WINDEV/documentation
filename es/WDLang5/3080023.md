


## xlsNbHoja de trabajo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de hojas de trabajo que se encuentran en un archivo XLS.

**Observación**: En el modo compatible, el número de hojas de trabajo es siempre igual a 1.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FileIdentifier is int

// Open an XLS file
FileIdentifier = xlsOpen("C:\My documents\MyFile.XLS")
IF FileIdentifier <> -1 THEN
	// Retrieve the numbers of worksheets found in the file
	Info("The file contains " + xlsNbWorksheet(FileIdentifier) + " worksheets.")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### La sintaxis que está usando el tipo xlsDocument

`<Result> = xlsNbWorksheet(<XLS document>)`
---

**`<Result>: (Integro)`**



- Número de hojas de trabajo que se encuentran en el archivo XLS,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS para cerrar. El archivo físico asociado a esta Variable (si existe) está cerrado. Las modificaciones realizadas en el archivo no se guardarán antes de que se cierre (utilice [xlsGuardar](../WDLang5/1000018057.md) si es necesario)..  


<a name="SYNTAX2"></a>

### Sintaxis conservada por motivos de compatibilidad con la versión 12

`<Result> = xlsNbWorksheet(<XLS file identifier>)`
---

**`<Result>: (Integro)`**



- Número de hojas de trabajo que se encuentran en el archivo XLS,

- -1 si se produce un error. Para obtener los detalles del error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS file identifier>: (Integro)`**

Identificador del archivo XLS a manipular, devuelto por [xlsAbrir](../WDLang5/3080017.md).

## Observaciones


### Gestión de las hojas de trabajo gráficas
<a name="gestion_las_hojas_trabajo_graficas_ELTPARAGRAPHE000076"></a>

Las hojas de trabajo "Gráficas" de Excel son ignoradas. De hecho, estas hojas de trabajo no tienen celda y por lo tanto no se puede acceder a ellas..


## sfSessionInfo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfSessionInfo** se utiliza para get información sobre la sesión actual. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

Este tipo de Variable es utilizado por el [sfConnection](../WDLang5/1000017579.md) Variable s. Las propiedades de este tipo están disponibles después de la llamada a [SFConnect](../WDLang5/1000017583.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
ELSE
	Trace("ID of the connected user: " + Cnt.SessionInfo.UserID)
END
```

### Propiedades específicas de la descripción de las variables sfSessionInfo
<a name="propiedades_especificas_descripcion_las_variables_sfsessioninfo_ELTPARAGRAPHE000050"></a>

Las siguientes propiedades pueden utilizarse para manipular la información sobre la sesión actual:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| IDSesión | Cadena de caracteres | Identificador de sesión. **Esta propiedad es de solo lectura**. |
| UserID | Cadena de caracteres | Identificador de usuario. **Esta propiedad es de solo lectura**. |
| información de usuario | [sfUserInfo](../WDLang5/1000018423.md) | Información sobre el usuario. **Esta propiedad es de solo lectura**. |
| SandboxModo | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el modo "Sandbox" está activado,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>**Esta propiedad es de solo lectura**. |
| ContraseñaVencida | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si caducó la contraseña,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>**Esta propiedad es de solo lectura**. |
| URL del servidor | Cadena de caracteres | URL del servidor. **Esta propiedad es de solo lectura**. |
| MetadatosServidorURL | Cadena de caracteres | URL del servidor de metadatos. **Esta propiedad es de solo lectura**. |

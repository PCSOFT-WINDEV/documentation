
## gglAuthor (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglAuthor** se utiliza para identificar al autor de un elemento de Google. Esta información está disponible en formato de sólo lectura: es especificado por Google durante la creación (de acuerdo con la información de conexión) y no puede ser modificado.

Este tipo de Variable es usado por:

- variables de tipo [gglCalendar](../WDLang5/1000017418.md).

- variables de tipo [gglEvent](../WDLang5/1000017440.md).




**Observaciones**:

- En la mayoría de los casos, no se requiere la declaración de una variable **gglAuthor**. Las propiedades del tipo [gglCalendar](../WDLang5/1000017418.md) o [gglEvent](../WDLang5/1000017440.md) permiten manipular directamente las propiedades del tipo **gglAuthor** (especialmente el **Author** Property).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection to Google
Cnt is gglConnection
Cnt.Email = "balthazar@gmail.com"
Cnt.Password = "qwerty"
Cnt.ApplicationName = "MyApp-01"
IF GglConnect(Cnt) = False THEN
	Error(ErrorInfo())
END
...

// Retrieve the array of available calendars
arrCalendars is array of gglCalendar
arrCalendars = GglListCalendar(Cnt)

// Retrieve the events of the first calendar
GglFillCalendar(Cnt, arrCalendars[1])

// Case 1: Direct use of the name of the author
Trace("Author of the calendar: "+ arrCalendars[1].Author.Name)

// Case 2: Use by passing a gglAuthor variable
CalendarAuthor is gglAuthor
CalendarAuthor = arrCalendars[1].Author
Trace("Author of the calendar: " + CalendarAuthor.Name)
```

## Observaciones




### Propiedades específicas de la description de las variables gglAuthor
<a name="propiedades_especificas_description_las_variables_gglauthor_ELTPARAGRAPHE000051"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **gglAuthor**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Email | Cadena de caracteres | Email del autor. Este Property corresponde al Email del usuario que creó el elemento (nombre utilizado en la información de conexión). <br>**Esta propiedad es de solo lectura**. |
| denominar | Cadena de caracteres | Nombre del autor. Este Property corresponde al nombre del usuario que creó el elemento (nombre utilizado en la información de conexión). <br>**Esta propiedad es de solo lectura**. |

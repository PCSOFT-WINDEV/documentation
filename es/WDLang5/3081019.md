


## XMLRoot( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciones en la etiqueta raíz de un Document XML.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF XMLDocument("MyXMLDoc", XMLCode) = True THEN
	// Find the links 
	XMLFind("MyXMLDoc", "a", XMLTag)
	WHILE XMLFound("MyXMLDoc") = True
		// Process
		// ...
		// Next element in the search
		XMLNext("MyXMLDoc")
	END
	// Position on the root
	XMLRoot("MyXMLDoc")
	// New search
	XMLFind("MyXMLDoc", "Link", XMLTag)
	WHILE XMLFound("MyXMLDoc") = True
		// Process
		// ...
		// Next element in the search
		XMLNext("MyXMLDoc")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLRoot(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se encontró el elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario. En caso de error: 

	- la situación actual permanece inalterada. 

	- la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).







**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Elementos de manipulación en el código XML
<a name="elementos_manipulacion_codigo_xml_ELTPARAGRAPHE000090"></a>

Después de **XMLRoot**, el elemento actual puede ser manejado por [XMLElementName](../WDLang5/3081026.md), [XMLParentName](../WDLang5/3081035.md), [XMLElementType](../WDLang5/3081031.md) y [XMLData](../WDLang5/3081004.md).
<a name="NOTE0_2"></a>


### Utilizar XMLRoot durante una búsqueda
<a name="utilizar_docparampagetitleshort_durante_una_busqueda_ELTPARAGRAPHE000111"></a>

Si hay una búsqueda en curso (función [XMLFind](../WDLang5/3081025.md)), la función **XMLRoot** cancela la búsqueda.
<a name="NOTE0_3"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000260"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).
Si la posición actual en un XML Document se modifica en un thread, la posición actual en este XML Document se modifica para todos los thread s.

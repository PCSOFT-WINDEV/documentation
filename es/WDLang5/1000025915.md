


## HTMLFindElementByID (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca el elemento cuyo atributo "ID" coincide con un valor específico en una Document HTML (o en un nodo HTML).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlDocument = HTMLOpen("https://www.windev.com", fromURL)
e is htmlNode <- HTMLChercheElémentParID(d, "MyID")
IF e <> Null THEN
	Info(e.TagName)
ELSE
	Info("Element not found")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HTMLFindElementByID(<Element to use> , <Identifier>)`
---

**`<Result>: (htmlNode Variable)`**

[htmlNode](../WDLang5/1000026081.md) variable que coincide con la condición de búsqueda. 

**`<Element to use>: (htmlDocument o htmlNode Variable)`**

Elemento donde se realizará la búsqueda: 

- Nombre de la [htmlDocument](../WDLang5/1000025910.md) Variable. 

- Nombre de la [htmlNode](../WDLang5/1000026081.md) Variable.




**`<Identifier>: (Cadena de caracteres)`**

Identificador del elemento a buscar.

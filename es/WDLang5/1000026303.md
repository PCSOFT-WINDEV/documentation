


## Funciones de gestión de la Document HTML
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones manipular documentos HTML: 



|   |   |
| --- | --- |
| [HTMLBuildString](../WDLang5/1000025909.md) | Genera una cadena UTF-8 que contiene un HTML Document. |
| [HTMLEnableDebugger](../WDLang1/1410087849.md) | Activa el depurador HTML/JavaScript en los controles HTML basados en CEF: Visualización de HTML o editor de HTML control. |
| [HTMLFindElementByClass](../WDLang5/1000026177.md) | Busca elementos cuyo atributo "class" coincida con un valor específico en una Document HTML (o en un nodo HTML). |
| [HTMLFindElementByID](../WDLang5/1000025915.md) | Busca el elemento cuyo atributo "ID" coincide con un valor específico en una Document HTML (o en un nodo HTML). |
| [HTMLFindElementByName](../WDLang5/1000025913.md) | Busca elementos cuyo atributo "name" coincida con un valor específico en un Document HTML (o en un nodo HTML). |
| [HTMLFindElementByTag](../WDLang5/1000026272.md) | Busca elementos que correspondan a una etiqueta en un Document HTML (o un nodo HTML). |
| [HTMLInsertAfter](../WDLang5/1410087610.md) | Añade un nodo después del nodo especificado en un HTML Document. |
| [HTMLInsertBefore](../WDLang5/1410087609.md) | Añade un nodo antes del nodo especificado en un HTML Document. |
| [HTMLInsertFirstChild](../WDLang5/1410087615.md) | Añade un hijo al principio de los hijos del nodo actual en un HTML Document. |
| [HTMLInsertLastChild](../WDLang5/1410087616.md) | Añade un hijo después del hijo del nodo actual en un HTML Document. |
| [HTMLModifyAttribute](../WDLang5/1410087618.md) | Añade o cambia un atributo del nodo actual del HTML Document. |
| [HTMLOpen](../WDLang5/1000025907.md) | Crea una variable [htmlDocument](../WDLang5/1000025910.md) a partir de un HTML Document. |
| [HTMLRemove](../WDLang5/1410087614.md) | Elimina el nodo especificado y sus descendientes en un HTML Document. |
| [HTMLRemoveAttribute](../WDLang5/1410087619.md) | Elimina un atributo determinado |
| [HTMLRemoveChild](../WDLang5/1410087617.md) | Elimina un nodo dado en el nodo actual de un HTML Document. |
| [HTMLSave](../WDLang5/1000025908.md) | Guarda el contenido de un [htmlDocument](../WDLang5/1000025910.md) Variable a un archivo HTML. |
| [HTMLToPDF](../WDLang5/1410088984.md) | Convierte una cadena o buffer HTML en un buffer que contiene un documento PDF. |
| [HTMLUnwrapAllChildren](../WDLang5/1410087613.md) | Elimina el nodo especificado en una Document HTML sin borrar sus hijos, que ocupan su lugar. |
| [HTMLWrap](../WDLang5/1410087611.md) | Envuelve el nodo especificado en un nodo dado de un HTML Document. |
| [HTMLWrapAllChildren](../WDLang5/1410087612.md) | Envuelve todos los hijos del nodo dado en un nuevo nodo en un HTML Document. |

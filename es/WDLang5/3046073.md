
## iSkipPage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera un salto de página: el código correspondiente se envía a la cola de impresión. El cursor de impresión se posiciona en las coordenadas (0,0) de la página siguiente.

La impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar la impresión).

**Observación**: Esta función no debería utilizarse en los informes creados con la función editor de reportes. Utilice la función [iPageEnd](../WDLang5/3046038.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print a text
iPrint("Hello")
// Page break
iSkipPage()
// Print a text
iPrint("We are" + DateSys())
// Start the print
iEndPrinting()
```

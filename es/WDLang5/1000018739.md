
## sfLeadConvertResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfLeadConvertResult** se utiliza para conocer el resultado de [SFConvertLead](../WDLang5/1000018498.md). Las características de este resultado son devueltas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de las variables sfLeadConvertResult
<a name="propiedades_especificas_las_variables_sfleadconvertresult_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden utilizarse para manipular el resultado de [SFConvertLead](../WDLang5/1000018498.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Error | Array de [sfError](../WDLang5/1000018455.md) | Estado reporte de errores. |
| AccountID | Cadena de caracteres | Identificador de la cuenta creada o modificada durante la conversión.<br>**Esta propiedad es de solo lectura**. |
| IDContacto | Cadena de caracteres | Identificador de la Contact creado o modificado durante la conversión.<br>**Esta propiedad es de solo lectura**. |
| OpportunityID | Cadena de caracteres | Identificador de la oportunidad posiblemente creada durante la conversión.<br>**Esta propiedad es de solo lectura**. |
| LeadID | Cadena de caracteres | Identificador del cable convertido. <br>**Esta propiedad es de solo lectura**. |
| Éxito | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la conversión se realiza correctamente,<br><br>- <u><u><u><u>False</u></u></u></u> si se produce un error.<br><br><br>**Esta propiedad es de solo lectura**. |

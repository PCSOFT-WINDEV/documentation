


## &lt;htmlNode Variable&gt;.InsertAfter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un nodo después del nodo especificado en un HTML Document.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
hDoc is htmlDocument = "
	<html><head/><body><div id="div"><p>Text</p></div></body></html>
"
hNode is htmlNode <- hDoc.FindElementByID("div")
// body: <div id="div"><p>Text</p></div>
hNode1 is htmlNode
hNode1..TagName = "h1"
hNode1..Text = "h1_text"
hNode.InsertAfter(hNode1)
//<html><head><body><div id="div"><p>Text</p></div><h1>h1_text</h1></body></html>
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Source node>.InsertAfter(<Node to insert>)`
---

**`<Source node>: (htmlNode Variable)`**

Nombre de la variable [htmlNode](../WDLang5/1000026081.md) que corresponde al nodo reference de inserción.

**`<Node to insert>: (htmlNode Variable)`**

Nombre de la [htmlNode](../WDLang5/1000026081.md) Variable que corresponde al nodo que se va a insertar después de &lt;nodo fuente&gt;.

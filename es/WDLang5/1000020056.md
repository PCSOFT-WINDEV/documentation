
## iAddBookmark( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un marcador en el visor de reporte o al exportar a PDF.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En el visor de reporte, una estructura de árbol permite alcanzar la posición de los marcadores generados.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Print in "Report viewer" mode
iDestination(iViewer)

// Add the root bookmark
iAddBookmark("Root", "MyDocument")

// Add a bookmark at the top of first page
iAddBookmark("Page 1", "Page 1", "Root")

// Positions a bookmark and a text at 5 cm from the top of the page
iYPos(50)
iAddBookmark("Page 1 line 1", "First line of the first page", "Page 1")
iPrint("First line of the first page")

// Print the page 2
iSkipPage()

// Add a bookmark at the top of the page
iAddBookmark("Page 2", "Page 2", "Root")

// Print two lines
iPrint("First line of the second page")
iPrint("Second line of the second page")

// Adds a bookmark for the third line
iAddBookmark("Page 2 line 3", "Third line of the second page", "Page 2")
iPrint("Third line of the second page")

// End of print
iEndPrinting()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = iAddBookmark(<Bookmark identifier> , <Caption> [, <Parent identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó el marcador,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. 




**`<Bookmark identifier>: (Cadena de caracteres)`**

Identificador único de marcadores. Se produce un error de WLanguage si este identificador ya existe.

**`<Caption>: (Cadena de caracteres)`**

Leyenda del marcador. Esta leyenda permitirá al usuario acceder directamente a la sección de texto correspondiente al marcador.. 

**`<Parent identifier>: (Cadena de caracteres opcional)`**

Identificador único del marcador padre del marcador para crear. 
Si no se especifica este parámetro, el marcador se crea en la raíz. 
Si el identificador del padre no existe, &lt;Resultado&gt; devuelve <u><u><u><u>False</u></u></u></u> y el marcador se crea en la raíz. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Durante las impresiones programadas**el marcador se crea en la posición actual. 

- **Al imprimir un reporte**, la función **iAddBookmark** debe utilizarse en el código de impresión de un control o bloque. Entonces, el marcador se crea en la posición del correspondiente control o bloque.

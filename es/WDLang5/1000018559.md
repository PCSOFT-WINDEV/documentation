
## sfExecuteProcessResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfExecuteProcessResult** se utiliza para conocer el resultado de [SFExecuteProcess](../WDLang5/1000018523.md). Las características de este resultado son devueltas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="XSYNTAX"></a>


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Propiedades específicas de las variables sfExecuteProcessResult
<a name="propiedades_especificas_las_variables_sfexecuteprocessresult_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden utilizarse para manipular el resultado de [SFExecuteProcess](../WDLang5/1000018523.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Error | Array de [sfError](../WDLang5/1000018455.md) | Estado reporte de errores. |
| ActorID | Cadena de caracteres | Identificador de los actores actualmente asignados al paso de validación.<br>**Esta propiedad es de solo lectura**. |
| IDEntity | Cadena de caracteres | Identificador del objeto de Salesforce actualmente procesado.<br>**Esta propiedad es de solo lectura**. |
| IDInstance | Cadena de caracteres | Identificador de la instancia Process associated con el objeto Salesforce.<br>**Esta propiedad es de solo lectura**. |
| NewWorkitemID | Cadena de caracteres | Identificadores de las nuevas tareas relacionado a la Process. Estos identificadores están separados por caracteres CR (Carriage Return).<br>**Esta propiedad es de solo lectura**. |
| Estado de la Instancia | Cadena de caracteres | Estado de la instancia de Process. Esta Property puede corresponder a "Aprobado", "Rechazado", "Eliminado" o "Pendiente" por ejemplo.<br>**Esta propiedad es de solo lectura**. |
| Éxito | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el proceso se ejecutó correctamente,<br><br>- <u><u><u><u>False</u></u></u></u> si se produce un error.<br><br><br>**Esta propiedad es de solo lectura**. |

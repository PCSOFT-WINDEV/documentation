


## SFConnect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para autenticarse junto a una plataforma de Salesforce.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection

Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
Cnt.LoginServerURL = "https://login.salesforce.com/services/Soap/u/42.0"

IF SFConnect(Cnt) = False THEN
	Error("The connection failed." + CR + ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SFConnect(<Salesforce connection>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la autenticación,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) que se utilizará.

<a name="NOTE0_1"></a>

## Observaciones


### Posibles casos de error : 
<a name="posibles_casos_error_ELTPARAGRAPHE000079"></a>

- La conexión de INTERNET no es válida.

- La autenticación no se realizó correctamente.

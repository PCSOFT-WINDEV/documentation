
## Etapa 3: Imprimir caracteres, imágenes, ....
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000142"></a>
Este paso consiste en "enviar los diferentes elementos a imprimir a la impresora":

- texto, 

- imágenes, 

- dibujos, ....




La impresión real en el soporte solicitado (configurado por [iPreview](../WDLang5/3046069.md) o [iDestination](../WDLang5/3046074.md)) será efectiva al final de la impresión ([iEndPrinting](../WDLang5/3046053.md)).





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Imprimir texto
<a name="imprimir_texto_ELTTEXTE000166"></a>
Las principales funciones de WLanguage utilizadas para imprimir textos son las siguientes.

**Observaciones**:

- Los textos pueden utilizar diferentes tipos de letra: la fuente es seleccionada por [iFont](../WDLang5/3046014.md).

- Los textos se pueden imprimir en posiciones específicas: [iXPos](../WDLang5/3046015.md) y [iYPos](../WDLang5/3046075.md).







|   |   |
| --- | --- |
| [iAreaHeight](../WDLang5/3046062.md) | Calcula la altura de un área de ancho fijo necesaria para imprimir todo el texto que utiliza la fuente actual. |
| [iDidotFont](../WDLang5/3046016.md) | Selecciona la unidad que se utilizará para la altura de las fuentes (creada por [iCreateFont](../WDLang5/3046045.md)): Punto DIDOT o punto PICA. |
| [iFont](../WDLang5/3046014.md) | Selecciona la fuente Default. |
| [iPrint](../WDLang5/3046031.md) | Envía la cadena de caracteres pasada como parámetro al buffer de impresión. |
| [iPrintArea](../WDLang5/3046027.md) | Imprime un texto en un área rectangular. |
| [iPrintWord](../WDLang5/3046034.md) | Envía el string pasado en parámetro a la memoria intermedia de impresión. |
| [iTextHeight](../WDLang5/3046070.md) | Calcula la altura de la fuente para el texto a imprimir (en milímetros). |
| [iTextWidth](../WDLang5/3046033.md) | Calcula el ancho (en milímetros) del texto a imprimir, de acuerdo con la fuente especificada. |
| [iXPos](../WDLang5/3046015.md) | Permite gestionar la posición horizontal (coordenada X) del cursor de impresión en la página. |
| [iYPos](../WDLang5/3046075.md) | Permite gestionar la posición vertical (coordenada Y) del cursor de impresión en la página. |

## Impresión de imágenes
<a name="impresion_imagenes_ELTTEXTE000264"></a>
Se pueden utilizar varias funciones WLanguage para imprimir Image s teniendo en cuenta el tamaño de Image.



|   |   |
| --- | --- |
| [iImageHeight](../WDLang5/3046057.md) | Calcula la altura de la Image a imprimir (en milímetros). |
| [iImageWidth](../WDLang5/3046061.md) | Calcula el ancho de la Image a imprimir (en milímetros). |
| [iPrintBarCode](../WDLang5/3046020.md) | Imprime un código de barras (dentro de un rectángulo). |
| [iPrintImage](../WDLang5/3046017.md) | Envía el archivo Image para imprimirlo al buffer de impresión. |
| [iTransparentMagenta](../WDLang5/3046003.md) | Modifica el modo de gestión del color Magenta para que este color sea considerado como Transparent (o no) en todas las próximas impresiones. |





![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo las imágenes en formato JPEG pueden ser impresas en Linux.

## Impresión de dibujos
<a name="impresion_dibujos_ELTTEXTE000325"></a>
Las siguientes funciones de WLanguage se utilizan para imprimir fácilmente diferentes formas:



|   |   |
| --- | --- |
| [iBorder](../WDLang5/3046044.md) | Imprime una Border en las coordenadas especificadas. |
| [iEllipse](../WDLang5/3046009.md) | Imprime una elipse (en un rectángulo). |
| [iHLine](../WDLang5/3046051.md) | Imprime una línea horizontal en las coordenadas especificadas. |
| [iLine](../WDLang5/3046050.md) | Imprime una Line en las coordenadas especificadas. |
| [iNewLine](../WDLang5/3046035.md) | Imprime una Line (en el estándar de Windows, con un extremo redondeado) en las coordenadas especificadas. |
| [iPrintBarCode](../WDLang5/3046020.md) | Imprime un código de barras (dentro de un rectángulo). |
| [iRoundBorder](../WDLang5/3046024.md) | Imprime una Border con contornos redondeados en coordenadas especificadas. |
| [iTransparentMagenta](../WDLang5/3046003.md) | Modifica el modo de gestión del color Magenta para que este color sea considerado como Transparent (o no) en todas las próximas impresiones. |
| [iVLine](../WDLang5/3046066.md) | Imprime una Line vertical en unas coordenadas determinadas. |
| [iXPos](../WDLang5/3046015.md) | Permite gestionar la posición horizontal (coordenada X) del cursor de impresión en la página. |
| [iYPos](../WDLang5/3046075.md) | Permite gestionar la posición vertical (coordenada Y) del cursor de impresión en la página. |

## Otras funciones
<a name="otras_funciones_ELTTEXTE000428"></a>
Se utilizan varias funciones de WLanguage para optimizar las impresiones realizadas en WLanguage. Se puede gestionar, por ejemplo:

- la pausa Page

- el tamaño de la Page, ...







|   |   |
| --- | --- |
| [función iReset](../WDLang5/3046067.md) | Reinicia los ajustes de impresión (márgenes, fuente, formato de papel, orientación, ...) almacenados en la biblioteca de impresión. |
| [iColumnNum](../WDLang5/1000018825.md) | Devuelve el número de la columna actual en una reporte multicolumna. |
| [iEndPrinting](../WDLang5/3046053.md) | Señala el final de la Document para imprimir y comienza a imprimir los datos almacenados en la cola de la impresora a través de las siguientes funciones: [iPrint](../WDLang5/3046031.md), [iPrintImage](../WDLang5/3046017.md), [iNewLine](../WDLang5/3046035.md), etc. |
| [iPageHeight](../WDLang5/3046052.md) | Calcula la altura de la página imprimible (en milímetros) teniendo en cuenta los márgenes superior e inferior. |
| [iPageHeightLeft](../WDLang5/3046060.md) | Calcula la altura disponible (en milímetros) a la izquierda de la página actual teniendo en cuenta los márgenes definidos (superior o inferior). |
| [iPageNum](../WDLang5/3046013.md) | Devuelve o inicializa el número de la página actualmente impresa. |
| [iPageWidth](../WDLang5/3046036.md) | Calcula el ancho (en milímetros) de la página imprimible teniendo en cuenta los márgenes izquierdo o derecho. |
| [iReportPrintingStatus](../WDLang5/3046043.md) | Devuelve el estado de la impresión actual. |
| [iSkipLine](../WDLang5/3046071.md) | Obliga a una o varias rupturas de Line. |
| [iSkipPage](../WDLang5/3046073.md) | Genera un salto de página: el código correspondiente se envía a la cola de impresión. |

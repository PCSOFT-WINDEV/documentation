


## xlsCoordinatesToName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el nombre de una celda, hoja de cálculo XLS o control Hoja de cálculo según sus coordenadas.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CellName is string = xlsCoordinatesToName(1,1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = xlsCoordinatesToName(<Column> , <Row>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de celda. Este nombre tiene el siguiente tipo: "A1". 

**`<Column>: (Integro)`**

Número de la columna que identifica la celda. 

**`<Row>: (Integro)`**

Número de la fila que identifica la célula.

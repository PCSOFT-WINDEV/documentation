


## XMLFound( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si el elemento actual corresponde a la búsqueda actual o a la búsqueda actual.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the tag that starts with "pri"
XMLFind("XMLDoc", "pri", XMLTag + XMLChildItem, XMLStartWith)
WHILE XMLFound("XMLDoc") = True
	Info("Tag found " + XMLElementName("DocXML"))
	// Next element in the search
	XMLNext("XMLDoc")
END
```
<a name="Example2"></a>

```wl
// Browse a document
XMLFirst("XMLDoc")
// Tag found?
IF XMLFound("XMLDoc") = True THEN
	Info("Current tag: " + XMLElementName("XMLDoc"))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLFound(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si una función de navegación XML se posicionó en un elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario (la posición actual en el documento XML no se modifica). La función **XMLFound** siempre devuelve el resultado opuesto de la función [XMLOut](../WDLang5/3081009.md).




**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Funciones que modifican XMLFound
<a name="funciones_que_modifican_xmlfound_ELTPARAGRAPHE000084"></a>

Las siguientes funciones modifican el valor devuelto por **XMLFound**: 

- [XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md),

- [XMLNext](../WDLang5/3081030.md), [XMLPrevious](../WDLang5/3081023.md),

- [XMLChild](../WDLang5/3081012.md), [XMLParent](../WDLang5/3081022.md), [XMLRoot](../WDLang5/3081019.md),

- [XMLFind](../WDLang5/3081025.md).



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000246"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).Si la posición actual en un XML Document se modifica en un thread, la posición actual en este XML Document se modifica para todos los thread s.

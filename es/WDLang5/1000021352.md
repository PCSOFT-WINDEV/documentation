
## mlEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza una sesión de MATLAB. Esta sesión fue iniciada por [mlInitialize](../WDLang5/1000021351.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nSessionID is int = mlInitialize()
IF ErrorOccurred() THEN
	ErrorInfo(errFullDetails)
	RETURN
END
mlExecute(nSessionID, "A = cosd(180)")
... 
mlEnd(nSessionID)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`mlEnd(<Session identifier>)`
---

**`<Session identifier>: (Integro)`**

Identificador de la sesión de MATLAB a utilizar. Este identificador es devuelto por [mlInitialize](../WDLang5/1000021351.md).

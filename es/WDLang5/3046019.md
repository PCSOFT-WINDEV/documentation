
## iPrintBodyComplement( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fuerza a imprimir un bloque de complemento de cuerpo mientras especifica la altura del bloque de complemento.

Esta función se utiliza para forzar la impresión de un complemento del cuerpo:

- después del final del descanso

- en la última página de reporte




**Observación**: Si no se utiliza esta función, el complemento del cuerpo se imprimirá en cada página después del último bloque de cuerpo y ocupará la altura restante de la página.. El complemento del cuerpo se imprime sólo si se imprime un bloque de cuerpo en la página.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Imprimir en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Before Printing code of document header
iPrintBodyComplement(iPageHeightLeft() - MySelf.Height)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iPrintBodyComplement(<Height of complement block>)`
---

**`<Height of complement block>: (Real)`**

Altura del bloque durante la impresión, expresada en milímetros. Si esta altura es mayor que el espacio dejado en la página, el complemento del cuerpo se imprimirá en 2 páginas..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **iPrintBodyComplement** podría reemplazarse por un bloque impreso (función [iPrintBlock](../WDLang5/3046030.md)) redimensionado con la propiedad [Altura](../Proprietes/2510050.md).

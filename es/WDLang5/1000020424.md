


## &lt;gglConnection Variable&gt;.UploadDocumento (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sube una Document al servidor de Google.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Cnt is gglConnection
...
// Connection
IF Cnt.Connect() =False THEN
	Error(ErrorInfo())
ELSE
	// Upload a Word file
	Cnt.UploadDocument("C:\Docs\file.doc")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Google connection>.UploadDocument(<Document path> [, <Source language> [, <Destination language>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo se subió correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Google connection>: (Variable de tipo gglConnection)`**

Nombre de la variable de tipo [gglConnection](../WDLang5/1000018009.md) que se utilizará. Esta conexión fue validada por [&lt;gglConnection Variable&gt;.Conectar](../WDLang5/1000021455.md).

**`<Document path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo (hasta 260 caracteres). Se puede utilizar una ruta de tipo UNC.

**`<Source language>: (Cadena de caracteres)`**

El lenguaje actual de la Document. Este parámetro se utiliza para definir el idioma de origen para la traducción del Document. Este parámetro corresponde al código Country en la norma ISO639-1. La lista de códigos de país en formato ISO639-1 se puede encontrar en la siguiente Address: [http://www.loc.gov/standards/iso639-2/php/code_list.php](http://www.loc.gov/standards/iso639-2/php/code_list.php)
Si este parámetro corresponde a una cadena vacía (""), Google intenta detectar automáticamente el lenguaje de la Document. 
Atención: Cuando se utiliza el "Drive" API, este parámetro ya no se tiene en cuenta.

**`<Destination language>: (Cadena de caracteres)`**

El lenguaje de la Document que se encuentra en el servidor. Este parámetro se usa para traducir el Document cuando se sube el archivo. Este parámetro corresponde al código Country en la norma ISO639-1. 
La lista de códigos de país en formato ISO639-1 se puede encontrar en la siguiente Address: [http://www.loc.gov/standards/iso639-2/php/code_list.php](http://www.loc.gov/standards/iso639-2/php/code_list.php). 
Atención: Cuando se utiliza el "Drive" API, este parámetro ya no se tiene en cuenta.

## Observaciones


### Posibles casos de error: 
<a name="posibles_casos_error_ELTPARAGRAPHE000225"></a>

- La conexión de INTERNET no es válida.

- La autenticación no se realizó correctamente.

- El ruta del Document no existe, el formato del Document no está soportado o el Document es demasiado grande.



<a name="NOTE0_2"></a>


### Tipos de archivos compatibles con Google
<a name="tipos_archivos_compatibles_con_google_ELTPARAGRAPHE000237"></a>

- Archivos de presentación de PowerPoint (extensiones ppt y pps). Límite de tamaño: 10 MB. 

- Archivos HTML, archivos de texto, archivos de Word (extensión doc), archivos de texto OpenDocument (extensión odt), archivos RTF, archivos Star Office (extensión sxw): 500 KB.

- Archivos correspondientes a hojas de trabajo (extensiones csv, xls y ods): 1 MB.



<a name="NOTE0_3"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000249"></a>

- Las características de una Document encontrada en el servidor de Google se gestionan a través de las variables de [gglDocument](../WDLang5/1000018001.md). Para conocer las características de todos los documentos encontrados en un servidor de Google, utiliza [&lt;gglConnection Variable&gt;.ListDocumento](../WDLang5/1000020436.md): esta función rellena una array de variables gglDocument.

- La interfaz Web está acostumbrada a:

	- incluir un Document en una carpeta.

	- dar un título a la Document.

	- marcar los documentos como "marcados con asterisco".


Estos elementos no se gestionan por programación en esta versión.

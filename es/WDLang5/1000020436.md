


## &lt;gglConnection Variable&gt;.ListDocumento (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la lista de documentos disponibles en el servidor de Google para el usuario actual..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Cnt is gglConnection
...
IF Cnt.Connect() =False THEN
	Error(ErrorInfo())
END
...

// List of all documents
arrDocuments is array of 0 gglDocument dynamic

// List the documents containing the "Test" text
arrDocuments = Cnt.ListDocument("fullText contains 'Test'")

// Display the title of each listed document
Doc is gglDocument dynamic
FOR EACH Doc OF arrDocuments
	 Trace(Doc.Title)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX2"></a>

## Sintaxis

`<Result> = <Google connection>.DocumentList([<Search to perform>])`
---

**`<Result>: (Array de las variables de la gglDocument)`**

Array de variables de tipo [gglDocument](../WDLang5/1000018001.md): cada elemento de array contiene las características de un Document encontrado en el servidor de Google.

**`<Google connection>: (Variable de tipo gglConnection)`**

Nombre de la variable de tipo [gglConnection](../WDLang5/1000018009.md) que se utilizará. Esta conexión fue validada por [&lt;gglConnection Variable&gt;.Conectar](../WDLang5/1000021455.md).

**`<Search to perform>: (Cadena de caracteres)`**

Fórmula de búsqueda. Consulte la documentación sobre el "Accionamiento" API para conocer los parámetros que se pueden utilizar: [https://developers.google.com/drive/web/search-parameters](https://developers.google.com/drive/web/search-parameters)

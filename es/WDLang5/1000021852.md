


## FBSessionStatus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba el estado de una sesión de Facebook. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySession is fbSession

IF FBSessionStatus(MySession) <> fbSessionOpened THEN
	Info("No session is started")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = FBSessionStatus(<Session>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la sesión: 



|   |   |
| --- | --- |
| *fbSessionClosed* | La sesión de Facebook se detuvo. |
| *fbSessionIncorrectPermission* | Sesión de Facebook iniciada con derechos distintos a los de la sesión pasada como parámetro. |
| *fbSessionOpened* | Comenzó la sesión de Facebook. |



**`<Session>: (fbSession variable)`**

Nombre de la [fbSession](../WDLang5/1000021851.md) Variable correspondiente a la conexión con manipular.


## Constantes de Facebook
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por las funciones de gestión de las conexiones de Facebook: 


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **fbEmail** | <br><br>- Información a obtener: Dirección de correo electrónico del usuario. <br><br>- Derechos necesarios durante la conexión a Facebook: "email" Permission: Se utiliza para acceder al primer correo electrónico Address de una persona.<br><br><br> | <br><br>- [FBGetUserInformation](../WDLang5/1000021837.md)<br><br>- [Tipo fbSession](../WDLang5/1000021851.md)<br><br><br> |
| **fbIdentifier** | Información a obtener: Identificador de usuario de Facebook. | [FBGetUserInformation](../WDLang5/1000021837.md) |
| **fbImage** | Información a obtener: Manual de uso de Image. | [FBGetUserInformation](../WDLang5/1000021837.md) |
| **fbInfoFriends** | Derechos necesarios durante la conexión a Facebook: "usuario_amigos" Permission: Se utiliza para acceder a la lista de amigos de su aplicación. | [Tipo fbSession](../WDLang5/1000021851.md) |
| **fbInfoProfile** | Derechos necesarios durante la conexión a Facebook: "public_profile" Permission: Permite acceder al perfil público de una persona. | [Tipo fbSession](../WDLang5/1000021851.md) |
| **fbName** | Información a obtener: Nombre de usuario. | [FBGetUserInformation](../WDLang5/1000021837.md) |
| **fbSessionClosed** | Estado de la sesión: La sesión de Facebook se detuvo. | [FBSessionStatus](../WDLang5/1000021852.md) |
| **fbSessionIncorrectPermission** | Estado de la sesión: La sesión de Facebook se inició con derechos distintos a los de la sesión pasada en parámetro. | [FBSessionStatus](../WDLang5/1000021852.md) |
| **fbSessionOpened** | Estado de la sesión: Comenzó la sesión de Facebook. | [FBSessionStatus](../WDLang5/1000021852.md) |

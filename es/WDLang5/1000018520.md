
## sfProcessWorkitemRequest (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfProcessWorkitemRequest** se utiliza para describir una consulta para el procesamiento de una tarea que se presentará a [SFExecuteProcess](../WDLang5/1000018523.md). Las características de este tipo pueden modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

### Propiedades específicas de las variables sfProcessWorkitemRequest
<a name="propiedades_especificas_las_variables_sfprocessworkitemrequest_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades se pueden utilizar para manipular la consulta para el procesamiento de una tarea para enviar a [SFExecuteProcess](../WDLang5/1000018523.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Acción | Cadena de caracteres | Acción a realizar en la tarea de Salesforce. Esta acción puede corresponder a uno de los siguientes strings de caracteres:<br><br>- Aprobar<br><br>- Rechazar<br><br>- Eliminar<br><br><br> |
| comment | Cadena de caracteres | comment que debe añadirse a la historia de la tarea de la Process. |
| SiguienteAproverID | Cadena de caracteres | Identificadores de los usuarios que deberán realizar la siguiente operación. |
| WorkItemID | Cadena de caracteres | Identificador de la tarea a Process. |

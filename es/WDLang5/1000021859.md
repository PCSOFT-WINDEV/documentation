
## iCurrentFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del archivo generado actualmente.

Observación : Esta función permite conocer el nombre del fichero generado actualmente en los eventos "Antes de volver a imprimir la reporte" o "Cierre".. Esta función está disponible para los informes programados y para los informes creados con la función editor de reportes. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFileName is string
sFileName = iCurrentFile()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = iCurrentFile()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre completo del archivo generado actualmente,

- Cadena vacía (""") en los siguientes casos : 

	- la impresión realizada es una impresión directa en la impresora,

	- la exportación (PDF, HTML, etc.) aún no ha generado un archivo,

	- para las impresiones programadas, la exportación se realiza a través de la vista previa de impresión.

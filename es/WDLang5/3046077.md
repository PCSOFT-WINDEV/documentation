
## iParameterReport( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la impresora abriendo la ventana de configuración. La ventana de configuración se inicializa con los parámetros especificados al crear la reporte (orientación, márgenes, etc.).





![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF iParameterReport(RPT_MyReport) = True THEN
	iPrintReport(RPT_MyReport)
ELSE  
	 Info("The report configuration was canceled")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iParameterReport(<Report name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario validó la información introducida en el cuadro de diálogo,

- <u><u><u><u>False</u></u></u></u> si el usuario canceló la operación,.




**`<Report name>: (Cadena de caracteres)`**

Nombre de reporte usado como reference. Las características de esta reporte (especificadas cuando se creó la reporte) se utilizarán para inicializar la ventana de configuración (orientación, etc.)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Configuración que se tiene en cuenta al imprimir una reporte
<a name="configuracion_que_tiene_cuenta_imprimir_una_reporte_ELTPARAGRAPHE000063"></a>

Sólo se tiene en cuenta la última configuración cuando se imprime una reporte.

No se recomienda utilizar varias veces las funciones **iParameterReport** y [iConfigureReport](../WDLang5/3046076.md) sin imprimir reportes.
<a name="NOTE0_2"></a>




### Reinicio de la configuración
<a name="reinicio_configuracion_ELTPARAGRAPHE000078"></a>

Después de llamar a **iParameterReport** (o [iConfigureReport](../WDLang5/3046076.md)), cualquier llamada a esta función reinicializará la ventana de configuración con los parámetros utilizados en la primera llamada.
<a name="NOTE0_3"></a>




### Número de copias
<a name="numero_copias_ELTPARAGRAPHE000091"></a>

Si el usuario especifica un número de copias (utilizando [iParameter](../WDLang5/3046008.md) o **iParameterReport**) antes de imprimir el reporte, este número de copias tendrá prioridad sobre el número de copias especificado con el [NbCopy](../Proprietes/2511042.md) Property.

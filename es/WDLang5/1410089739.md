


## &lt;Variable xlsDocument&gt;.RecalculateAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recalcula todas las fórmulas de un documento Excel (XLSX).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is xlsDocument

// Make changes to the XLSX document
MyDocument[1,"A"] = 1
// Define the formula
MyDocument[1,"B"].Formula = "A1"
// Recalculates all formulas
MyDocument.RecalculateAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Excel document>.RecalculateAll()`
---

**`<Excel document>: (Variable de tipo xlsDocument)`**

Nombre de la variable de tipo [xlsDocument](../WDLang5/1000017464.md) utilizada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al cambiar el valor o la fórmula de una celda no se recalculan automáticamente las fórmulas del libro. 

- Se debe llamar a la función **&lt;Variable xlsDocument&gt;.RecalculateAll** después de cambiar el valor o la fórmula de una o más celdas. Esta función actualiza los valores de las celdas en un libro que utiliza una fórmula.

- Esta función no puede utilizarse en un documento XLS.


## iLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un enlace a un URL Address en la página HTML actualmente impresa.

**Observación**: **iLink** no tiene efecto si la página no se imprime en un archivo HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iDestination(iHTMLCSS, "MyPage")
iPrint("On the next line, a link to a page")
iLink("Know all about WINDEV", "http://www.WINDEV.com")
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iLink(<Link caption> , <URL address> [, <Current browser>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprimió el enlace,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Link caption>: (Cadena de caracteres)`**

Texto del enlace. Este texto se mostrará como un enlace en la página HTML.

**`<URL address>: (Cadena de caracteres)`**

Address de la página a mostrar. Esta página se abrirá automáticamente cuando el usuario haga clic en &lt;Link Caption>.

**`<Current browser>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado): la página se abre en el navegador actual.

- <u><u><u><u>False</u></u></u></u>: la página se abre en un nuevo navegador.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Imprimir una URL
<a name="imprimir_una_url_ELTPARAGRAPHE000056"></a>

Se genera automáticamente una pausa Line después del título del enlace. El cursor se posiciona en la siguiente Line. El paso al siguiente Line tiene en cuenta la altura del actual Line. Esta altura depende de la altura de la fuente utilizada.

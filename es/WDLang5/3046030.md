
## iPrintBlock( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obliga a imprimir un bloque de reporte. El código WLanguage associated con el bloque impreso se ejecuta.

**Observación** : Esta función permite imprimir cualquier bloque de reporte. Sin embargo, esta función se utiliza principalmente para imprimir un bloque de iteración.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print the END_OF_DOCUMENT block
iPrintBlock(END_OF_DOCUMENT)
// Print an iteration block
iPrintBlock(ITERATION1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iPrintBlock(<Block name>)`
---

**`<Block name>: (Cadena de caracteres)`**

Nombre lógico del bloque a imprimir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **iPrintBlock** no puede utilizarse en los siguientes eventos:

	- "Abrir" el reporte.

	- "Antes de imprimir" de un bloque de encabezado de página.

	- "Antes de imprimir" de un bloque de pie de página.




- En la mayoría de los casos, se recomienda utilizar los planos en la reporte ya que son más fáciles de usar e implementar (en lugar de la sustitución por bloques).. Para obtener más información, consulte [Los aviones](../WDChamp/1013078.md).

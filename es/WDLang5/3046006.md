
## iDocumentCanceled( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la impresión actual ha sido cancelada por el usuario.

**Observación**: Si el usuario cancela una impresión en curso, las funciones de impresión no tienen efecto hasta la siguiente llamada a [iEndPrinting](../WDLang5/3046053.md). Sin embargo, el código de WLanguage sigue funcionando.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046006&name=idocumentcanceled_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
IF iDocumentCanceled() = True THEN
	Info("The current print was canceled")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iDocumentCanceled()`
---

**`<Result>: (booleano)`**

<u><u><u><u>True</u></u></u></u> si el usuario:

- cerró el visor de reporte antes de que se completara la impresión,

- hizo clic en el botón ***Cancelar*** de la ventana de impresión.

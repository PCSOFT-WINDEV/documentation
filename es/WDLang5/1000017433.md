
## gglPath (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglPath** permite definir las características de la ruta a dibujar en un mapa. Las características de la ruta pueden definirse y modificarse mediante diferentes propiedades de WLanguage. El dibujo de la ruta utiliza los puntos especificados (y no las calles del mapa).

Este tipo de variable es utilizado por el tipo [gglMapParameter](../WDLang5/1000017505.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la description de las variables gglPath
<a name="propiedades_especificas_description_las_variables_gglpath_ELTPARAGRAPHE000044"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **gglPath**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Color | Integro | Color de dibujo. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido](../WDLang5/3010002.md).<br><br><br> |
| Espesor | Integro | Espesor de la Line (en píxeles). |
| Opacidad | Integro | Opacidad de la Line (en porcentaje). |
| Punto[ n] | Array de [gglCoordinate](../WDLang5/1000017489.md) | Puntos de la ruta. El máximo número de puntos que se puede usar es set a 100. |
| Punto[ n].Latitud | Real | Latitud del punto. |
| Point[n].Longitude | Real | Longitud del punto. |

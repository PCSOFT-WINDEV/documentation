
## rssEnclosure (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssEnclosure** se utiliza para describir y manipular el adjunto de una entrada de flujo RSS. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Este tipo de Variable es el que utiliza [Tipo rssEntry](../WDLang5/1000017792.md).  

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssEnclosure**. De hecho, las propiedades del [Tipo rssEntry](../WDLang5/1000017792.md) permiten manipular directamente las propiedades de tipo **rssEnclosure**.

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la descripción de las variables rssEnclosure
<a name="propiedades_especificas_descripcion_las_variables_rssenclosure_ELTPARAGRAPHE000059"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **rssEnclosure**

| Property | Tipo utilizado | Efecto |
| --- | --- | --- |
| Tamaño (\*) | Integro | Tamaño de la Document en bytes. |
| Tipo (\*) | Cadena de caracteres | El tipo de MIME de la Document. Más de 150 tipos están definidos en el estándar de comunicación MIME (disponible en INTERNET). Los valores más comunes son:<br><br>- "solicitud/pdf": Document en formato PDF (\*.pdf)<br><br>- "text/html": Página HTML (\*.htm,\*.html)<br><br>- "text/plain": archivo de texto (\*.txt)<br><br>- "Image/gif": Image en formato GIF (\*.gif)<br><br>- "Image/jpeg": Image en formato JPEG (\*.jpg, \*.jpeg)<br><br>- "video/mpeg": vídeo en formato MPEG (\*.mpg,\*.mpeg)<br><br>- "aplicación/desconocido": utilizado para mostrar un cuadro de diálogo proponiendo descargar el archivo en la el equipo de usuario de la Web.<br><br>- "solicitud / palabra clave": Archivo Word.<br><br>- "application/vnd.ms-excel": Excel file.<br><br><br> |
| URL (\*) | Cadena de caracteres | URL utilizado para acceder a la Document. |


**Observaciones**:

- Las propiedades seguidas de una estrella (\*) son obligatorias.

- Todas las propiedades que se encuentran en esta lista están disponibles en lectura/escritura.

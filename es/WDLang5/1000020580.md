


## iGroupAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un reporte a un grupo de informes. 

**Observación**: La implementación de un grupo de informes se utiliza para mostrar un cuadro combinado específico en el visor de reporte. Los informes que se encuentran en el grupo se listan en este cuadro combinado y se pueden visualizar en el visor de reporte. Esta característica le permite proponer al usuario: 

- a reporte en modo vertical y en modo horizontal. 

- una reporte en diferentes formatos: A3, A4, letra US, ... 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print destination
iDestination(iViewer)
// Create the group of reports
iGroupAdd(RPT_TO_Year_A4_Portrait, 2012)
iGroupAdd(RPT_TO_Year_A4_Landscape, 2012)
// Display first report of the group in the report viewer
iGroupPrint()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = iGroupAdd(<Report name> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Result>: (Integro)`**

Índice de la reporte en el grupo de informes.

**`<Report name>: (Cadena de caracteres)`**

Nombre de reporte que se añadirá al grupo.

**`<Parameter 1>: (Parámetros opcionales (el tipo corresponde al tipo de parámetro))`**

Parámetro opcional esperado por la reporte. Este parámetro se definió en el evento "Apertura" del reporte. Este parámetro puede ser el nombre de una vista, el nombre de una zona de memoria, etc..
Este parámetro será pasado a la reporte cuando se muestre en el visor de reporte o cuando se imprima. 



**`<Parameter N>: (Parámetros opcionales (el tipo corresponde al tipo de parámetro))`**

Parámetro opcional esperado por la reporte. Este parámetro se definió en el evento "Apertura" del reporte. Este parámetro puede ser el nombre de una vista, el nombre de una zona de memoria, etc..
Este parámetro será pasado a la reporte cuando se muestre en el visor de reporte o cuando se imprima. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los parámetros pasados por Address no están soportados: cuando se llama, **iGroupAdd** copia el valor de los parámetros.

- Un icono y un pie de foto son añadidos por Default en el cuadro combinado. Esta información puede ser personalizada por [iGroupConfigurar](../WDLang5/1000020581.md). La leyenda de Default corresponde a las dimensiones y orientación de reporte.  

- La reporte seleccionada por defecto puede ser configurada por [iGroupImprimir](../WDLang5/1000020582.md).


## Tipo xlsCell: MaskWD Property
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000153"></a>
La propiedad MaskWD del tipo xlsDocument se utiliza para definir la máscara de visualización en formato WINDEV o WEBDEV que se utilizará en la celda de Excel ("DD/MM/YYYYY", por ejemplo).

La máscara especificada se convierte automáticamente al formato Excel. No se admiten las centésimas de segundo que se encuentran en las máscaras.

Esta página de ayuda presenta las diferentes máscaras que se pueden utilizar.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las máscaras que pueden ser usadas en la máscara Property
<a name="las_mascaras_que_pueden_ser_usadas_mascara_property_ELTTEXTE000177"></a>


### Máscaras de números enteros
<a name="mascaras_numeros_enteros_ELTPARAGRAPHE000022"></a>

Ejemplos de máscaras disponibles para las celdas que muestran números enteros:

- "9"

- "999"

- "999999"

- "999 999"

- "999 999 999"



<a name="NOTE2_2"></a>


### Máscaras reales
<a name="mascaras_reales_ELTPARAGRAPHE000035"></a>

Ejemplos de máscaras disponibles para las celdas que muestran reales:

- "999 999, 999 999"

- "999 999, 999999"

- "999, 999"

- "999999,9"



<a name="NOTE2_3"></a>


### Máscaras de fechas
<a name="mascaras_fechas_ELTPARAGRAPHE000047"></a>

Ejemplos de máscaras disponibles para las celdas que muestran fechas:

- "DD/MM/AAAA"

- "MM/DD/YYYY"

- "DD/MM/AA"

- "DD-MM-AAAA"

- "DD-MM-YY"

- "DD MM YYYYYYY"

- "DD MM YY"

- "Día:DD Mes:MM Año:AAAA"



<a name="NOTE2_4"></a>


### Máscaras de tiempo
<a name="mascaras_tiempo_ELTPARAGRAPHE000063"></a>

Ejemplos de máscaras disponibles para las celdas que muestran los tiempos:

- "HH:MM:SS"

- "HH:MM"

- "HH"

- "HH h MM"

- "HH hr MM min"

- "Horas Horas MM Minutos SS Segundos"



<a name="NOTE2_5"></a>


### Máscaras de fecha y hora
<a name="mascaras_fecha_hora_ELTPARAGRAPHE000077"></a>

Ejemplos de máscaras disponibles para las celdas que muestran fechas y horas:

- "MM/DD/AAAA HH:MM:SS"

- "DD/MM/YYY HH:MM"

- "MM/DD/YYY HH:MM:SS"



<a name="NOTE2_6"></a>


### Máscaras de duración
<a name="mascaras_duracion_ELTPARAGRAPHE000088"></a>

Ejemplos de máscaras disponibles para las celdas que muestran duraciones:

- "HH:MM:SS"

- "D HH:MM:SS"

- "Días Horas Horas MM Minutos"



<a name="NOTE2_7"></a>


### Máscaras de divisas
<a name="mascaras_divisas_ELTPARAGRAPHE000099"></a>

Ejemplos de máscaras disponibles para las celdas que muestran monedas:

- "+999 999,99 $"

- "999999,99$"

- "+999 999,999999 $"

- "999999,999 999$"

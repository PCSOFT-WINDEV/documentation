


## &lt;gglConnection Variable&gt;.Solicitud (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía una solicitud de comunicación (solicitud HTTP) a un servicio de Google.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Cnt is gglConnection
...
IF Cnt.Connect() =False THEN
	Error(ErrorInfo())
END

// Query without parameter
XMLQry is string = Cnt.Request(gglServiceDocuments, ...
		"http://docs.google.com/feeds/documents/private/full") 
// Query with parameters (search for "Casimir")
XMLQry = Cnt.Request(gglServiceDocuments, ...
		"http://docs.google.com/feeds/documents/private/full?q=Casimir")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Google connection>.Request(<Google service> , <HTTP request> [, <Data to send> [, <Type of request>]])`
---

**`<Result>: (Cadena de caracteres ANSI)`**



- Respuesta del servidor,

- Cadena vacía ("") si se produce un error (variable [ErrorOccurred](../WDLang1/3087001.md) establecida en <u><u><u><u>True</u></u></u></u>).




**`<Google connection>: (Variable de tipo gglConnection)`**

Nombre de la variable de tipo [gglConnection](../WDLang5/1000018009.md) que se utilizará. Esta conexión fue validada por [&lt;gglConnection Variable&gt;.Conectar](../WDLang5/1000021455.md).

**`<Google service>: (Constante de tipo Integer)`**

Servicio de Google en el que se realizará la solicitud:


|   |   |
| --- | --- |
| *gglServiceCalendar* | "Servicio"Google Calendar: Google calendar |
| *gglServiceContacts* | "Servicio"Contactos de Google: Gestión de contactos de Google. <br><br><br>	<B><B><B><B>Advertencia</B></B></B></B><br><br>	 Desde el 15 de junio de 2021, la API de contactos de Google utilizada por esta función ya no está disponible ([https://developers.google.com/contacts/v3/announcement](https://developers.google.com/contacts/v3/announcement)). Para obtener más información, consulte [Gestión de contactos de Google](../WDLang5/1000017761.md).<br>
 |
| *gglServiceDocuments* | "Servicio"Lista de documentos de Google: Gestión de documentos |
| *gglServicePicasa* | "Picasa" service |
| *gglServiceStaticMaps* | "Servicio"Google Static Maps: Gestión de mapas |



**`<HTTP request>: (Cadena de caracteres)`**

URL completa de la solicitud (incluyendo los parámetros almacenados en la URL si es necesario). Consulte la documentación de Google correspondiente al servicio utilizado para obtener más información.

**`<Data to send>: (Cadena de caracteres opcional)`**

Datos que deben enviarse al servidor. Consulte la documentación de Google correspondiente al servicio utilizado para obtener más información.

**`<Type of request>: (Constante opcional de tipo Integer)`**

Tipo de petición HTTP a enviar:


|   |   |
| --- | --- |
| *gglHTTPDelete* | Borrar solicitud. |
| *gglHTTPGet* | Solicitud de get. |
| *gglHTTPPost* | Solicitud de Post. |
| *gglHTTPPut* | Poner solicitud. |


Si se especifica &lt;Datos a enviar&gt;, las solicitudes son solicitudes de ***gglHTTPPost*** de Default.
Si no se especifica &lt;Datos a enviar&gt;, las solicitudes son ***ggHTTPGet*** solicitudes de Default.

## Observaciones


### Posibles casos de error: 
<a name="posibles_casos_error_ELTPARAGRAPHE000251"></a>

- La conexión de INTERNET no es válida.

- La autenticación no se realizó correctamente.

- El servidor devuelve un error.




Para saber si se ha producido un error, utilice el programa [ErrorOccurred](../WDLang1/3087001.md) Variable.

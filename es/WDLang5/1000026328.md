
## HTML constante
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para gestionar los archivos HTML:

| Constante | Detalles | Función / Tipo de Variable |
| --- | --- | --- |
| **htmlNodeCDATA** | Tipo de nodo en el HTML Document: Nodo CDATA (compatibilidad con xHTML). | Tipo [htmlNode](../WDLang5/1000026081.md) |
| **htmlNodeComment** | Tipo de nodo en el HTML Document: Nodo comment. | Tipo [htmlNode](../WDLang5/1000026081.md) |
| **htmlNodeElement** | Tipo de nodo en el HTML Document: Nodo de elemento, etiqueta HTML. | Tipo [htmlNode](../WDLang5/1000026081.md) |
| **htmlNodeTexte** | Tipo de nodo en el HTML Document: Nodo de texto. | Tipo [htmlNode](../WDLang5/1000026081.md) |

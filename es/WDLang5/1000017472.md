
## xlsCell (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una variable de tipo **xlsCell** permite describir y modificar una celda de un documento Excel.

Este tipo de Variable se utiliza: 

- por [xlsDocument](../WDLang5/1000017464.md) variables.

- por [xlsColumn](../WDLang5/1000017471.md) variables.

- por [xlsRow](../WDLang5/1000017325.md) variables.




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open an Excel file
sExcelFile is string
sExcelFile = fSelect("", "", "Select an Excel file...", "Excel" + ...
	TAB + "*.xlsx" + CR + "All files (*.*)" + TAB + "*.*", "*.xlsx")
IF sExcelFile = "" THEN RETURN

// Open the Excel document
MyDocument is xlsDocument
MyDocument = xlsOpen(sExcelFile)

// Create an xlsCell variable with red background.
Cell is xlsCell
Cell = "WINDEV is incredible!"
Cell.BackgroundColor = LightRed

// Fill in cell A6 
MyDocument..Cell[6,"A"] = Cell

// Saves the modifications
xlsSave(MyDocument)
xlsClose(MyDocument)
```

## Observaciones




### Propiedades específicas de las variables xlsCell
<a name="propiedades_especificas_las_variables_xlscell_ELTPARAGRAPHE000058"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **xlsCell**:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| AlineaciónH | Constante de tipo Integer | Alineación horizontal de celdas. Los valores posibles son los siguientes:<br><br>- *haCenter*: centrado.<br><br>- *haRight*: alineación correcta.<br><br>- *haLeft*: alineación a la izquierda.<br><br><br> |
| AlineaciónV | Constante de tipo Integer | Alineación vertical de las celdas. Los valores posibles son los siguientes:<br><br>- *vaTop*: alineación en la parte superior.<br><br>- *vaMiddle*: alineación en el centro.<br><br>- *vaBottom*: alineación en la parte inferior.<br><br><br> |
| Franja de orilla | [Variable de tipo Border](../Motscles/15140100.md) | Características del borde de la célula. |
| BackgroundColor | Entero o constante de tipo Integer | Color de fondo de la celda. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- [un color predefinido](../WDLang5/3010002.md).<br><br><br><br><br> |
| Fórmula | Cadena de caracteres | Fórmulas de celdas. <br><br>Atención: Al cambiar la fórmula de una celda no se recalculan automáticamente las fórmulas del libro.. <br>Es necesario llamar a [xlsRecalculateAll](../WDLang5/1410089617.md) después de cambiar la fórmula de una o más celdas para actualizar los valores de las celdas que contienen una fórmula.<br><br>Esta Property sólo puede utilizarse en documentos XLSX (no es compatible con documentos XLS) |
| Máscara | Cadena de caracteres | Máscara de visualización en Excel. Esta máscara debe corresponder a la máscara reconocida por Excel. Consulte la documentación sobre Excel para obtener más detalles. |
| MáscaraWD | Cadena de caracteres | Máscara de visualización en formato WINDEV o WEBDEV que se utilizará en la celda de Excel (por ejemplo,"DD/MM/YYYYY").. La máscara especificada se convierte automáticamente al formato Excel. No se admiten las centésimas de segundo que se encuentran en las máscaras..<br>Para obtener más información, consulte [Máscaras soportadas](../WDLang5/1000017473.md).<br>Este Property está disponible en modo de escritura. |
| Multilínea | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la celda es multilínea,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| tipo de letra | [Variable de tipo Font](../Motscles/1514045.md) | Fuente de la célula. |
| escribir a máquina | Constante de tipo Integer | Tipo de celda de Excel. Los valores posibles son los siguientes:<br><br>- *xlsTypeString*: Columna"Cadena de caracteres" o columna que contiene diferentes tipos de datos.<br><br>- *xlsTypeDate*: Columna"Fecha".<br><br>- *xlsTypeDateTime*: Columna"DateTime".<br><br>- *xlsTypeDouble*: Columna"Numérica".<br><br>- *xlsTypeDuration*: Columna"Duración".<br><br>- *xlsTypeError*: En caso de error. Para obtener más detalles sobre el error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.<br><br>- *xlsTypeTime*: Columna"Tiempo".<br><br>- *xlsTypeCurrency*: Columna"Moneda".<br><br><br>**Esta propiedad es de solo lectura**. |
| Valor | Tipo de valor de celda | Valor de la celda.<br>Cuando se define el estilo de una columna (xlsColumn.DefaultCell), este Property se ignora en el modo de escritura y se inicializa vacío en el modo de lectura). |




## &lt;sfConnection Variable&gt;.SeekModified (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca y recupera los registros (objetos de Salesforce) modificados durante el período especificado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt..Login = "vince@gmail.com"
Cnt..Password = "qwerty" + "XXB12VCZ54"
IF Cnt.Connect() =False THEN
	Error("The connection failed")
	RETURN
END

ResGetModif is sfSeekUpdatedResult
ResGetModif = Cnt.SeekModified("Mileage__c", "20090101", "20090131")

Identifier is string
FOR EACH STRING Identifier OF ResGetModif..ID SEPARATED BY CR
		Trace("--- ID : " + Identifier)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Salesforce connection>.SeekModified(<Salesforce file> , <Start date> , <End date>)`
---

**`<Result>: (sfSeekUpdatedResult Variable)`**

Nombre de la [sfSeekUpdatedResult](../WDLang5/1000018561.md) Variable que contiene los datos del resultado.

**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) que se utilizará.

**`<Salesforce file>: (Cadena de caracteres)`**

Nombre del archivo de la Fuerza de Ventas a Process. Este nombre debe corresponder al nombre definido en la interfaz de Salesforce como "API name".. Para un archivo personalizado, este nombre corresponde al nombre del archivo seguido de 2 caracteres de subrayado y luego de la letra "C".. Por ejemplo : "Mi cuenta__C".

**`<Start date>: (FechaHora)`**

Fecha de inicio del período para el que se solicita el número de objetos modificados. Este parámetro debe estar en formato UTC (utilice [DateTimeLocalToUTC](../WDLang1/3027036.md) si es necesario para realizar la conversión).

**`<End date>: (FechaHora)`**

Fecha final del período para el que se solicita el número de objetos modificados. Este parámetro debe estar en formato UTC (utilice [DateTimeLocalToUTC](../WDLang1/3027036.md) si es necesario para realizar la conversión).

## Observaciones

<a name="posibles_casos_error_ELTPARAGRAPHE000209"></a>

- La autenticación no se realizó correctamente.

- La respuesta XML del servidor no se puede leer.




Para saber si se ha producido un error, utilice el programa [ErrorOccurred](../WDLang1/3087001.md) Variable.

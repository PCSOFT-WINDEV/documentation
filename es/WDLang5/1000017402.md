


## GglDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina datos del servidor de Google. Los datos creados o actualizados están contenidos en un Variable que corresponde al servicio utilizado:

- [gglCalendar](../WDLang5/1000017418.md) para borrar una variable calendar.

- [gglContact](../WDLang5/1000017434.md) variable para borrar un contact.

- [gglContactGroup](../WDLang5/1000017576.md) variable para eliminar un grupo de contactos.





	<B><B><B>Advertencia</B></B></B>

	 Desde el 15 de junio de 2021, la API de Google Contact s utilizada por esta función para gestionar los grupos Contact s y Contact ya no está disponible ([https://developers.google.com/contacts/v3/announcement](https://developers.google.com/contacts/v3/announcement)). 
Para utilizar los contactos de Google, consulte la API de personas publicada por Google como reemplazo :  [https://developers.google.com/people](https://developers.google.com/people). 
[RESTSend](../WDLang3/1000021476.md) le permite integrar fácilmente una API REST en sus proyectos de desarrollo.

## Ejemplo


```wl
// Connection
Cnt is gglConnection
...
IF GglConnect(Cnt) = False THEN
	Error(ErrorInfo())
END

// Retrieve the "Work" calendar
arrCalendar is array of 0 gglCalendar
arrCalendar = GglListCalendar(Cnt, gglUser)
// Display the title of each calendar
Calendar is gglCalendar
FOR EACH Calendar OF arrCalendar
	IF Calendar.Title = "Work" THEN
	 	// Delete the calendar
	 	GglDelete(Cnt, Calendar)
	END
END
```

## Sintaxis

`<Result> = GglDelete(<Google connection> , <Google element>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Google connection>: (Variable de tipo gglConnection)`**

Nombre de la variable de tipo [gglConnection](../WDLang5/1000018009.md) que se utilizará. Esta conexión fue validada por [GglConnect](../WDLang5/1000017332.md).

**`<Google element>: (Variable que corresponde al tipo a borrar)`**

Elemento a borrar. Este parámetro puede corresponder a:

- Nombre de la variable [gglCalendar](../WDLang5/1000017418.md) que corresponde al Calendar que se va a borrar.

- Nombre de la [gglContact](../WDLang5/1000017434.md) Variable que corresponde al Contact que se va a borrar.

- Nombre de la [gglContactGroup](../WDLang5/1000017576.md) Variable que corresponde al grupo Contact que se va a eliminar.

## Observaciones


### Borrar calendarios
<a name="borrar_calendarios_ELTPARAGRAPHE000105"></a>

Para eliminar un calendario, este debe haberse recuperado anteriormente con la función [GglListCalendario](../WDLang5/1000017330.md) y la constante *gglUser*. De lo contrario, el servidor devuelve un "InternalServerError".




## HTMLFindElementByName (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca elementos cuyo atributo "name" coincida con un valor específico en un Document HTML (o en un nodo HTML).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlDocument = HTMLOpen("https://www.windev.com", fromURL)
e is array of htmlNode <- HTMLFindElementByName(d, "Color1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HTMLFindElementByName(<Element to use> , <Name>)`
---

**`<Result>: (Array)`**

Array de los elementos de [htmlNode](../WDLang5/1000026081.md) que coinciden con la condición de búsqueda. 

**`<Element to use>: (htmlDocument o htmlNode Variable)`**

Elemento donde se realizará la búsqueda: 

- Nombre de la [htmlDocument](../WDLang5/1000025910.md) Variable. 

- Nombre de la [htmlNode](../WDLang5/1000026081.md) Variable.




**`<Name>: (Cadena de caracteres)`**

Valor del atributo "nombre" que se busca.

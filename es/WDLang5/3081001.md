


## XMLCancelSearch( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la búsqueda realizada por [XMLFind](../WDLang5/3081025.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the <price> tag
XMLFind("XMLDoc", "price", XMLTag)
WHILE XMLFound("XMLDoc") = True
	Info("Tag found " + XMLElementName("DocXML"))
	XMLNext("XMLDoc")
END

// Cancels the current search
XMLCancelSearch("XMLDoc")

// Resume a standard browse
XMLFirst("XMLDoc")
WHILE XMLOut("XMLDoc") = False
	// Operations
	XMLNext("XMLDoc")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`XMLCancelSearch(<XML document>)`
---

**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document en el que se cancelará la búsqueda. Esta Document contiene el código XML y fue creada por [XMLDocument](../WDLang5/3081008.md).

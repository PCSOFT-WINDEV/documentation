
## rssCategory (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssCategory** permite manipular la categoría de un flujo RSS o de una entrada de flujo. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

Este tipo de Variable es usado por:

- [el tipo rssStream](../WDLang5/1000017795.md).

- [el tipo rssEntry](../WDLang5/1000017792.md).




**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssCategory**. Las propiedades de la [Tipo rssStream](../WDLang5/1000017795.md) o [Tipo rssEntry](../WDLang5/1000017792.md) permiten manipular directamente las propiedades del tipo **rssCategory** (especialmente la categoría **** Property).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la descripción de las variables rssCategory
<a name="propiedades_especificas_descripcion_las_variables_rsscategory_ELTPARAGRAPHE000065"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **rssCategory**:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Dominio | Cadena de caracteres | URL que describe el dominio en el que se aplica la categoría. |
| Valor | Cadena de caracteres | Valor arbitrario que describe la categoría de la entrada. Este valor es utilizado por los programas que agregan los flujos XML. |

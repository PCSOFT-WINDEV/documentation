## Presentación
<a name="presentacion_ELTTEXTE000259"></a>
Este paso le permite definir y elegir las diferentes fuentes que se usarán en el Document impreso. Puede seleccionar y definir:

- fuentes de impresión específicas: estas fuentes estarán disponibles cuando se imprima con las funciones WLanguage.

- fuentes dinámicas, que se pueden utilizar tanto en la pantalla (en ventanas, dibujos) como en la impresión en WLanguage.




**Este paso es opcional**.

Se utiliza la fuente Default si no se especifica ninguna fuente para el trabajo de impresión.

- tipo de letra: No me importa,

- Fuente 0,

- envergadura: 12 spades,

- Al estilo de Normal,

- color: Black .




**Atención**: Sólo puede imprimir utilizando las fuentes de la impresora. Si la fuente utilizada no está disponible, se utilizará la fuente más cercana.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md).

## Imprimir hacer
<a name="imprimir_hacer_ELTTEXTE000283"></a>
Las fuentes de impresión son creadas por [iCreateFont](../WDLang5/3046045.md). Esta función le permite seleccionar:

- el tipo de letra (nombre y familia)
	Constantes para el tipo de letra: 
	


|   |   |
| --- | --- |
| *iCourier* | Fuente Courier (ancho de caracteres fijos) |
| *iDefault* | Cualquier fuente ("Don't care") |
| *iRoman* | Fuente romana (fuente exótica) |
| *iSwiss* | Swiss (fuente de calidad) |



- el tamaño (de los caracteres)

- los atributos (negrita, cursiva, subrayado)
	Constantes para los atributos: 
	


|   |   |
| --- | --- |
| *iBold* | Fuente en negrita |
| *iCondensed* | Caracteres condensados |
| *iExtended* | Caracteres extendidos |
| *iItalic* | Fuente cursiva |
| *iLarge* | Caracteres grandes |
| *iNormal*<br>(valor predeterminado) | Fuente estándar, sin atributos |
| *iStrikeOut* | Fuente tachada |
| *iUnderlined* | Fuente subrayada |



- la color (véase la [constantes preestablecidas](../WDLang5/3010002.md))




Los diferentes parámetros de la fuente se definen mediante varias constantes.

Cada fuente es associated con un número. Para elegir el tipo de letra de imprenta, basta con utilizar [iFont](../WDLang5/3046014.md).

## Fuente dinámica
<a name="fuente_dinamica_ELTTEXTE000307"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) También tiene la capacidad de crear fuentes dinámicas. Una "Fuente" Variable le permite crear una fuente que puede ser utilizada en todos los objetos del proyecto: ventanas, informes, impresiones, gráficos, dibujos, etc.

Las fuentes dinámicas se utilizan para imprimir un texto orientado a 90° por ejemplo (con la [Ángulo](../Proprietes/1410086632.md) Property de la fuente dinámica).

Para definir una fuente dinámica:

1. Crear una Variable de tipo [tipo de letra](../Motscles/1514045.md).

2. Definir las características de la fuente:

	- utilizando [FontCreate](../WDLang1/3060002.md).

	- con la [propiedades de fuentes](../Proprietes/2515045.md): [Ángulo](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Condensado](../Proprietes/2513006.md), [Color](../Proprietes/2510071.md), [Extendido](../Proprietes/2513002.md), [atrevido](../Proprietes/2513001.md), [Cursiva](../Proprietes/2513007.md), [Grande](../Proprietes/2513012.md), [denominar](../Proprietes/2510082.md), [subrayar](../Proprietes/2513013.md) y [envergadura](../Proprietes/2510125.md).







Para elegir un tipo de letra de imprenta, basta con utilizar [iFont](../WDLang5/3046014.md).

## Fuente de impresora
<a name="fuente_impresora_ELTTEXTE000331"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Los fabricantes de impresoras instalan fuentes en sus impresoras. Estas fuentes se denominan fuentes "printer".

A diferencia de las fuentes de software (como "True Type"), estas fuentes pueden ser accedidas directamente por la impresora, no tienen que ser descargadas.

En las listas de fuentes, las fuentes de la impresora se identifican con una "impresora" Icon delante del nombre de la fuente.

Atención: pueden producirse efectos de visualización inesperados al utilizar las fuentes de la impresora.

Si está utilizando fuentes de impresora, estas fuentes serán interpretadas sólo por la impresora.. Durante la vista previa de impresión, Windows gestiona la visualización de las fuentes. Como no admite estas fuentes específicas, Windows utiliza la fuente de visualización más cercana.

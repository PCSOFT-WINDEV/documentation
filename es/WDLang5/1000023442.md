
## XML: Uso de archivos de más de varios gigabytes
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000093"></a>
A partir de la versión 23, WINDEV, WEBDEV y WINDEV Mobile incluyen un analizador XML para los archivos XML de manipular que tienen un tamaño de varios GB: por ejemplo, Open Street Map o Open Data France.

La velocidad de lectura y el consumo de memoria se mejoran en los archivos XML enormemente. De hecho, el archivo no se carga en la memoria: sólo se navega. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000117"></a>


### Método
<a name="metodo_ELTPARAGRAPHE000018"></a>

Para leer un archivo XML grande con el analizador XML: 

1. Abra el archivo XML para leerlo con [XMLOpenReader](../WDLang5/1000023396.md). Esta función se utiliza para inicializar una [xmlReader](../WDLang5/1000023392.md) Variable. 
	Por ejemplo: 
	
	```wl
	Reader is xmlReader = xmlOpenReader("MyFile.xml")
	```


2. Examinar el XML utilizando una sintaxis [FOR EACH](../Motscles/1510022.md).

3. Las propiedades de [xmlReader](../WDLang5/1000023392.md) Variable se utilizan para encontrar las características del elemento leído. 





### Ejemplos
<a name="ejemplos_ELTPARAGRAPHE000043"></a>El siguiente ejemplo presenta un archivo xml y diferentes modos de navegación con la lista de elementos explorados. 

```xml
< ?xml version="1.0"?>
<aa>
	<bb>
		Text1
        
		<cc>
		</cc>
	</bb>
	<bb>
		<cc>
		</cc>
	</bb>
</aa>
```



```wl
// Loop through a level
// Reader is the variable corresponding to the XML file
FOR EACH Reader
	// Read the start aa and end aa tags
	FOR EACH Reader
		// Read the start bb, end bb, start bb, end bb tags
		FOR EACH Reader
		// Read Text1 and the start cc, end cc
		// start cc, end cc tags
		END
	END
END
```



```wl
// Reader is the variable corresponding to the XML file
FOR EACH Reader IN-DEPTH
	// Read the start aa, start bb, text1, 
	// start cc, end cc, end bb, start bb,
	// start cc, end cc, end bb, end aa tags
END
```

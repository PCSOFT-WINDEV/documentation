


## XMLValidDocument (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Valida un Document XML de un esquema XSD.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Validation from the schema defined during the declaration
cMyDoc is xmlDocument, description = "ProjectXSDSchema"
IF XMLValidDocument(cMyDoc) = False THEN
	Error(ErrorInfo())
ELSE
	Info("The XML document is valid") 
END
```
<a name="Example2"></a>

```wl
// Validation with the schema passed in parameter
cMyDoc2 is xmlDocument
IF XMLValidDocument(cMyDoc2, "XSDSchemaOfProject") = False THEN
	Error(ErrorInfo())
ELSE
	Info("The XML document is valid") 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = XMLValidDocument(<XML document> [, <XSD schema>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se validó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, utilice [ErrorInfo](../WDLang1/3013008.md) para obtener más detalles sobre el error. 




**`<XML document>: (xmlDocument)`**

Nombre de la [xmlDocument](../WDLang5/1000018782.md) Variable que contiene el Document a validar.

**`<XSD schema>: (Cadena de caracteres opcional)`**

Nombre del esquema que se utilizará para validar el XML Document. 

- Si se especifica este parámetro, el esquema XSD especificado se utiliza para validar la Document. 

- Si no se especifica este parámetro, se utiliza el esquema especificado durante la declaración (declaración utilizando la siguiente sintaxis : cMyDoc es xmlDocumento, description="MyXSD"). 

- Se visualiza un error WLanguage si el esquema especificado durante la declaración no es válido.. 


Observación : Si el documento XML fue declarado con un esquema description de XSD, la validación se realiza automáticamente durante la copia de seguridad por [XMLSave](../WDLang5/1000019026.md).

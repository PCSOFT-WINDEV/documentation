


## Etapa 1: Configuración de los parámetros de impresión
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000174"></a>
Este paso le permite elegir:

- los ajustes de impresión de Document (calidad, número de copias, etc.).

- los parámetros de configuración de la impresora (controlador de impresora, orientación, formato de papel, etc.).




**Este paso es opcional**.

Este paso debe completarse si los ajustes de Default (definidos en el panel control de Windows) no son adecuados para el trabajo de impresión.



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Las diferentes operaciones
<a name="las_diferentes_operaciones_ELTTEXTE000198"></a>
Las diferentes operaciones son las siguientes:

**1. Selección de los parámetros de impresión de la Document** ([iParameter](../WDLang5/3046008.md)). Es posible:

- abrir la ventana de configuración de la impresión.

- configurar los ajustes de impresión uno por uno (altura y anchura de la página, número de copias, etc.).






**2. Elegir la impresora** ([iConfigure](../WDLang5/3046072.md)). Es posible:

- abrir la ventana de selección de la impresora.

- cambiar la impresora utilizada para el trabajo de impresión actual. Esta modificación puede ser temporal (para el programa actual) o permanente.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) [iConfigure](../WDLang5/3046072.md) no está disponible.

**3. Redefinición de los márgenes de impresión** ([iMargin](../WDLang5/3046011.md)). Los márgenes superior, inferior, izquierdo y derecho se pueden modificar. Si no se llama a esta función, se usarán los márgenes de Default (que difieren según la impresora seleccionada).

**Atención**: Debe respetarse el orden de las operaciones mencionadas de Above. Si hay un trabajo de impresión en curso, [iParameter](../WDLang5/3046008.md) y [iConfigure](../WDLang5/3046072.md) Cancel el trabajo de impresión actual.

## Funciones utilizadas para configurar el destino de la impresión
<a name="funciones_utilizadas_para_configurar_destino_impresion_ELTTEXTE000222"></a>
Las siguientes funciones se utilizan para configurar el destino de impresión y determinar la configuración actual:


|   |   |
| --- | --- |
| [iConfigure](../WDLang5/3046072.md) | Configura la impresora. |
| [iInfoPrinter](../WDLang5/3046022.md) | Recupera las características de la impresora actual o Default. |
| [Impresora iListPrinter](../WDLang5/3046063.md) | Devuelve la lista de impresoras instaladas en el ordenador actual. |
| [iParameter](../WDLang5/3046008.md) | Configura la impresión. |
| [iPreview](../WDLang5/3046069.md) / [iDestination](../WDLang5/3046074.md) | Configura el destino de impresión. |

La configuración y los parámetros definidos con las funciones **iConfigure** y **iParameter** reinicializan el módulo de impresión. Por lo tanto, estas funciones deben ser las primeras funciones de impresión llamadas. Todas las funciones de impresión llamadas antes de estas dos funciones serán ignoradas.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El trabajo de impresión sólo puede realizarse en una impresora.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) La Document sólo puede imprimirse en PDF.

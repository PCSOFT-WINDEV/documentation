<a name="Use"></a>
<a name="description"></a>
Devuelve el texto de la licencia de uso de las bibliotecas de Google.

**Atención**: 

- Desde principios de 2018, Google no le permite recuperar el texto de la licencia de Google para las bibliotecas utilizadas.. Esta licencia para el uso de las bibliotecas de Google ya no se debe mostrar en la aplicación.. El usuario final puede acceder a la licencia en la configuración del teléfono (Configuración -> Google -> menú "Licencias Open Source"). 

- Esta función ya no devuelve la licencia de Google independientemente de la versión de WINDEV Mobile utilizada.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sLicense is string 
sLicense = gglGetLicense(gglPlayServices)
IF sLicense <> "" THEN
	Info(sLicense)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GglGetLicense(<Library>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena vacía (""). 

**`<Library>: (Constante de tipo Integer)`**

Biblioteca de Google para la que se recuperará la licencia: 


|   |   |
| --- | --- |
| *gglPlayServices* | Biblioteca de los Servicios de Google Play. Esta biblioteca se utiliza en Android para los controles del mapa basados en el Google Maps v2 API. |

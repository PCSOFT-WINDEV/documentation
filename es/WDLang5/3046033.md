


## iTextWidth( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el ancho (en milímetros) del texto a imprimir, de acuerdo con la fuente especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
x is int = iTextWidth(iFont(5) + CU.CUSTNAME)
iPrintWord("The width of customer name is equal to: ")
iPrint(x)
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iTextWidth([<Font identifier> + ] <Text to print>)`
---

**`<Result>: (Real)`**



- Ancho (en milímetros) del texto especificado si se realizó el trabajo de impresión,

- -1 si el trabajo de impresión fue cancelado antes de la ejecución de la función.




**`<Font identifier>: (Cadena de caracteres opcional)`**

Identificador de la fuente utilizada, devuelto por [iFont](../WDLang5/3046014.md). Si no se especifica este parámetro, se seleccionará la última fuente utilizada.

**`<Text to print>: (Cadena de caracteres)`**

Cadena de caracteres cuyo ancho se solicita.

### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000065"></a>

Dependiendo del destino (impresora, archivo, etc.), el ancho devuelto por **iTextWidth** puede ser diferente (en una décima de milímetro) para el mismo texto, que se muestra con la misma fuente. Esta diferencia es causada por el mecanismo de visualización utilizado por Windows.
<a name="NOTE0_2"></a>


### Cálculo de la anchura de un texto escrito con fuentes de diferentes tamaños
<a name="calculo_anchura_texto_escrito_con_fuentes_diferentes_tamanos_ELTPARAGRAPHE000074"></a>

Debe llamar a **iTextWidth** para cada texto que utilice una fuente diferente.
<a name="NOTE0_3"></a>


### Calculando el ancho de un texto que contiene caracteres CR
<a name="calculando_ancho_texto_que_contiene_caracteres_ELTPARAGRAPHE000083"></a>

**iTextWidth** ignora los caracteres CR encontrados en el texto. Se debe llamar a la función **iTextWidth** por cada línea de la cadena.
<a name="NOTE0_4"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000094"></a>

Al llamar a la función [iParameter](../WDLang5/3046008.md) después de la función **iTextWidth** no se realiza ninguna acción, dado que la impresión se inicializa.

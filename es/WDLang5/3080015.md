


## xlsCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un archivo XLS.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003080015&name=xlscerrar_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
MyXLSFile is xlsDocument

// Open an XLS file
MyXLSFile = xlsOpen("C:\MyDirectories\File.XLS")
...
// Close the XLS file
xlsClose(MyXLSFile)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### La sintaxis que está usando el tipo xlsDocument

`xlsClose(<XLS document>)`
---

**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS a cerrar. El archivo físico asociado a esta Variable (si existe) está cerrado. Las modificaciones realizadas en el fichero no se guardarán hasta que se cierre (utilice [xlsGuardar](../WDLang5/1000018057.md) si es necesario).  


<a name="SYNTAX2"></a>

### Sintaxis conservada por motivos de compatibilidad con la versión 12

`xlsClose(<XLS file identifier>)`
---

**`<XLS file identifier>: (Integro)`**

Identificador del archivo XLS a cerrar, devuelto por [xlsAbrir](../WDLang5/3080017.md).

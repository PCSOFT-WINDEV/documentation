


## XMLOut( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar si el elemento (etiqueta o atributo) sobre el que queremos posicionarnos está fuera del Document XML.

El uso de la función **XMLOut** es opcional. De hecho, las funciones de navegación XML devuelven un resultado que le permite saber si han tenido éxito o no.. Se recomienda utilizar la función **XMLOut** para que el código sea más fácil de entender.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the first element
XMLFirst("XMLDoc")

// Tag found?
IF XMLOut("XMLDoc") = False THEN
	Info("Current tag: " + XMLElementName("XMLDoc"))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLOut(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si una función de navegación no pudo posicionarse en un elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario (la posición actual en el documento XML no se modifica). 
	La función **XMLOut** siempre devuelve el resultado opuesto de la función [XMLFound](../WDLang5/3081020.md).




**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML para estudiar y fue creado por [xmlDocument](../WDLang5/3081008.md).

## Observaciones


### Funciones que modifican XMLOut
<a name="funciones_que_modifican_xmlout_ELTPARAGRAPHE000090"></a>

Las siguientes funciones modifican el valor devuelto por la función **XMLOut**:

- [XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md),

- [XMLNext](../WDLang5/3081030.md), [XMLPrevious](../WDLang5/3081023.md),

- [XMLChild](../WDLang5/3081012.md), [XMLParent](../WDLang5/3081022.md), [XMLRoot](../WDLang5/3081019.md),

- [XMLFind](../WDLang5/3081025.md).



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000252"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).Si la posición actual en un XML Document se modifica en un thread, la posición actual en este XML Document se modifica para todos los thread s.

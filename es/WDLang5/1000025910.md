
## htmlDocument (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **htmlDocument** se utiliza para definir todas las características avanzadas de un HTML Document. Las características de esta Document pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlDocument
e is htmlNode
e..Type = htmlNodeElement // optional
e..TagName = "H1"
e..Text = "Title"
Add(d.html.body, e)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables Document
<a name="propiedades_especificas_las_variables_document_ELTPARAGRAPHE000035"></a>

Las siguientes propiedades pueden ser utilizadas para las variables manipular **xlsDocument**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| HTMLContent | Cadena de caracteres | El contenido HTML de la Document codificado en UTF-8.  |
| RootNode | Variable de tipo [htmlNode](../WDLang5/1000026081.md) | Array de los nodos de la raíz. |


<a name="NOTE0_2"></a>
Observaciones: 

- El operador "." se utiliza para acceder al elemento raíz "html".

- Los documentos vacíos se crean con el elemento "html".

- Una Document HTML contiene un único elemento [htmlNode](../WDLang5/1000026081.md) llamado "html".. 




- Puedes hacer un bucle a través de todos los elementos de una Document usando "PARA CADA x DE Document Variable EN PROFUNDIDAD".

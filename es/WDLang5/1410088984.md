
## HTMLToPDF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena o buffer HTML en un buffer que contiene un documento PDF.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sHTMLCode is string
// Convert HTML code to a PDF document
My_PDF_Doc is Buffer = HTMLToPDF(sHTMLCode)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HTMLToPDF(<Text in HTML format> [, <Page format> [, <PDF parameters>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Buffer que contiene el documento PDF resultante de la conversión HTML.

- Buffer vacío si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Text in HTML format>: (Cadena de caracteres)`**

Texto a convertir.

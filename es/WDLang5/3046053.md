
## iEndPrinting( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Señala el final de la Document para imprimir y realmente comienza a imprimir los datos almacenados en la cola de impresión utilizando [iPrint](../WDLang5/3046031.md), [iPrintImage](../WDLang5/3046017.md), [iNewLine](../WDLang5/3046035.md), ... Este conjunto de datos constituye la Document a imprimir.

**Observación**: Si no se ejecuta **iEndPrinting**, se perderá todo el Document. Esta función sólo debe utilizarse con los informes creados mediante programación. Esta función no debe ser llamada en los procesos de una reporte creada con editor de reportes en WIINDEV.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iPrint("List of customers")
iSkipLine()
// Process
iEndPrinting()
```

## Observaciones




### Acción realizada por iEndPrinting
<a name="accion_realizada_por_iendprinting_ELTPARAGRAPHE000069"></a>

La función **iEndPrinting** cancela:

- la configuración de [iMargin](../WDLang5/3046011.md),

- las fuentes creadas de antemano.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el modo actual de "vista previa de impresión".

### Procesos a interrumpir durante un trabajo de impresión
<a name="procesos_interrumpir_durante_trabajo_impresion_ELTPARAGRAPHE000089"></a>

Durante la impresión se deben interrumpir los siguientes procesos:

- timer( con [Temporizador final](../WDLang1/3015007.md)),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) gestión de los enlaces DDE (con [DDEUnLink](../WDLang1/3028005.md)),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) event( con [FinalizarEvento](../WDLang1/3015010.md)).

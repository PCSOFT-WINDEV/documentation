


## Funciones de Facebook (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la autenticación de Facebook (sintaxis del prefijo).



|   |   |
| --- | --- |
| [&lt;fbSession Variable&gt;.Estado](../WDLang5/1000021942.md) | Comprueba el estado de una sesión de Facebook. |
| [&lt;fbSession Variable&gt;.GetUserInformation](../WDLang5/1000021945.md) | Recupera la información de Facebook del usuario conectado. |
| [&lt;fbSession Variable&gt;.Solicitud](../WDLang5/1000021946.md) | Realiza una petición de comunicación (petición HTTP) al Facebook Graph API. |
| [&lt;fbSession Variable&gt;.Stop](../WDLang5/1000021943.md) | Detiene la sesión de Facebook que se inició con [&lt;Variable fbSession&gt;.Start](../WDLang5/1000021944.md). |
| [&lt;Variable fbSession&gt;.Start](../WDLang5/1000021944.md) | Permite autenticarse e iniciar sesión en Faceboook. |

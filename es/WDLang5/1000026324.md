


## Funciones de gestión de documentos HTML (sintaxis de prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular documentos HTML (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;htmlDocument Variable&gt;.BuildString](../WDLang5/1000026320.md) | Genera una cadena UTF-8 que contiene un HTML Document. |
| [&lt;htmlDocument Variable&gt;.FindElementByClass](../WDLang5/1000026314.md) | Busca elementos cuyo atributo "class" coincida con un valor específico en un HTML Document. |
| [&lt;htmlDocument Variable&gt;.FindElementByName](../WDLang5/1000026318.md) | Busca elementos cuyo atributo "name" coincida con un valor específico en un HTML Document. |
| [&lt;htmlDocument Variable&gt;.FindElementByTag](../WDLang5/1000026312.md) | Busca elementos que correspondan a una etiqueta en un HTML Document. |
| [&lt;htmlDocument Variable&gt;.Guardar](../WDLang5/1000026321.md) | Guarda el contenido de un [htmlDocument](../WDLang5/1000025910.md) Variable a un archivo HTML. |
| [&lt;htmlDocument Variable&gt;.SearchElementByID](../WDLang5/1000026316.md) | Busca el elemento cuyo atributo "ID" coincide con un valor específico en una Document HTML . |
| [&lt;htmlNode Variable&gt;.FindElementByClass](../WDLang5/1000026315.md) | Busca elementos cuyo atributo "class" coincida con un valor específico en un nodo HTML. |
| [&lt;htmlNode Variable&gt;.FindElementByName](../WDLang5/1000026319.md) | Busca elementos cuyo atributo "name" coincida con un valor específico en un nodo HTML. |
| [&lt;htmlNode Variable&gt;.FindElementByTag](../WDLang5/1000026313.md) | Busca elementos que correspondan a una etiqueta en un nodo HTML. |
| [&lt;htmlNode Variable&gt;.InsertAfter](../WDLang5/1410088110.md) | Añade un nodo después del nodo especificado en un HTML Document. |
| [&lt;htmlNode Variable&gt;.InsertBefore](../WDLang5/1410088111.md) | Añade un nodo antes del nodo especificado en un HTML Document. |
| [&lt;htmlNode Variable&gt;.InsertFirstChild](../WDLang5/1410088113.md) | Añade un hijo al principio de los hijos del nodo actual en un HTML Document. |
| [&lt;htmlNode Variable&gt;.InsertLastChild](../WDLang5/1410088112.md) | Añade un hijo después del hijo del nodo actual en un HTML Document. |
| [&lt;htmlNode Variable&gt;.ModifyAttribute](../WDLang5/1410088114.md) | Añade o cambia un atributo del nodo actual del HTML Document. |
| [&lt;htmlNode Variable&gt;.Remove](../WDLang5/1410088115.md) | Elimina el nodo especificado y sus descendientes en un HTML Document. |
| [&lt;htmlNode Variable&gt;.RemoveAttribute](../WDLang5/1410088118.md) | Elimina un atributo determinado |
| [&lt;htmlNode Variable&gt;.RemoveChild](../WDLang5/1410088116.md) | Elimina un nodo dado en el nodo actual de un HTML Document. |
| [&lt;htmlNode Variable&gt;.SearchElementByID](../WDLang5/1000026317.md) | Busca el elemento cuyo atributo "ID" coincide con un valor específico en un nodo HTML. |
| [&lt;htmlNode Variable&gt;.UnwrapAllChildren](../WDLang5/1410088108.md) | Elimina el nodo especificado en una Document HTML sin borrar sus hijos, que ocupan su lugar. |
| [&lt;htmlNode Variable&gt;.Wrap](../WDLang5/1410088117.md) | Envuelve el nodo especificado en un nodo dado de un HTML Document. |
| [&lt;htmlNode Variable&gt;.WrapAllChildren](../WDLang5/1410088109.md) | Envuelve todos los hijos del nodo dado en un nuevo nodo en un HTML Document. |

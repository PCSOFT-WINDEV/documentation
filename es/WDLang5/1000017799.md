
## rssImage (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssImage** permite describir la imagen de un flujo RSS. Las características de la imagen se pueden definir y cambiar mediante diferentes propiedades WLanguage.  

Este tipo de Variable es el que utiliza el [Tipo rssStream](../WDLang5/1000017795.md).

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssImage**. En efecto, las propiedades de la [Tipo rssStream](../WDLang5/1000017795.md) permiten manipular directamente las propiedades del tipo rssImage (especialmente la .Image Property).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la description de las variables rssImage
<a name="propiedades_especificas_description_las_variables_rssimage_ELTPARAGRAPHE000058"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **rssImage**:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| description | Cadena de caracteres | description de la Image. |
| Altura | Integro | Alto de la imagen en pixeles. Si no se especifica esta Property, la altura se deduce de la Image. |
| anchura | Integro | Ancho de la imagen en pixeles. Si no se especifica este Property, el ancho se deduce del Image. |
| vincular | Cadena de caracteres | Vincular associated con la Image (un sitio web en la mayoría de los casos). |
| Título | Cadena de caracteres | Valor de texto de la Image (usado cuando la Image no se puede visualizar). |
| URL | Cadena de caracteres | URL del Image usado para el canal. |




## XMLPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición actual en la Document en formato XPATH.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is string
Res = XMLPath("MyDocument")
// Res has the following format: /ORDERS/ORDERLINE[2]/@NUMBER on an attribute
// Res has the following format: /ORDERS/ORDERLINE[1] on a child tag
// Res has the following format: /ORDERS on a tag 
// Caution, the XMLPosition function can also return a string
// in the following format (syntactically valid): /*/*[8]/*[1]
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLPath(<XML document>)`
---

**`<Result>: (Cadena de caracteres)`**

Posición de la etapa de corriente, en formato XPATH.

**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000198"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).


## iCreateFont( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una nueva fuente de impresión. Esta fuente se puede utilizar en los diferentes procesos. Esta fuente es referenciada por un número de fuente, elegido durante el desarrollo.. Para elegir este tipo de letra y utilizarlo en las diferentes funciones de impresión, basta con seleccionarlo con [iFont](../WDLang5/3046014.md).

**Observación**: Esta función DEBE llamarse NECESARIAMENTE DESPUÉS DE [iPreview](../WDLang5/3046069.md). De lo contrario, las vistas previas serán ignoradas.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046045&name=icreatefont_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Create a 48-point Pica font
iCreateFont(1, 48, iBold + iItalic, iRoman)
// Print a text with this font
iPrint(iFont(1) + "48-pica text")
```

## Sintaxis

### Crear una nueva fuente

`iCreateFont(<Identifier> , <Size> , <Style> , <Name> [, <Color> [, <Angle>]])`
---

**`<Identifier>: (Integro)`**

Pídales que identifiquen. Este identificador se utilizará en las diferentes funciones de impresión para especificar la fuente utilizada.

**`<Size>: (Integro)`**

Altura de la fuente en puntos Pica (1 punto Pica = 0,353 mm).

**`<Style>: (Constante o combinación de constantes de tipo Integer)`**

Atributos de la fuente.


|   |   |
| --- | --- |
| *iBold* | Fuente en negrita |
| *iCondensed* | Caracteres condensados<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción se ignora. |
| *iExtended* | Caracteres extendidos<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción se ignora. |
| *iItalic* | Fuente cursiva |
| *iLarge* | Caracteres grandes<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta opción se ignora. |
| *iNormal*<br>(valor predeterminado) | Fuente estándar, sin atributos |
| *iStrikeOut* | Fuente tachada |
| *iUnderlined* | Fuente subrayada |



**`<Name>: (Cadena de caracteres)`**

Nombre de la fuente (entre las fuentes instaladas en el ordenador actual).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) **Impresión en PDF**: sólo se admiten las siguientes fuentes: Courier, Helvetica, Times, Symbol y ZapfDingbats. Si el nombre de la fuente especificada no corresponde a una fuente admitida, se utilizará la fuente Helvetica.

**`<Color>: (Entero opcional)`**

Color utilizado para la fuente. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Angle>: (Entero opcional)`**

Ángulo (en grados) según el cual se imprimirá el texto. El ángulo se calcula a partir de una línea horizontal en sentido contrario a las agujas del reloj (dirección trigonométrica).

### Creación de una familia de fuentes

`iCreateFont(<Identifier> , <Size> , <Style> , <Family> [, <Color> [, <Angle>]])`
---

**`<Identifier>: (Integro)`**

Pídales que identifiquen. Este identificador se utilizará en las diferentes funciones de impresión para especificar la fuente utilizada.

**`<Size>: (Integro)`**

Altura de la fuente en puntos Pica (1 punto Pica = 0,353 mm).

**`<Style>: (Constante o combinación de constantes de tipo Integer)`**

Atributos de la fuente.


|   |   |
| --- | --- |
| *iBold* | Fuente en negrita |
| *iCondensed* | Caracteres condensados |
| *iExtended* | Caracteres extendidos |
| *iItalic* | Fuente cursiva |
| *iLarge* | Caracteres grandes |
| *iNormal*<br>(valor predeterminado) | Fuente estándar, sin atributos |
| *iStrikeOut* | Fuente tachada |
| *iUnderlined* | Fuente subrayada |



**`<Family>: (Constante de tipo Integer)`**

Familia de la fuente.


|   |   |
| --- | --- |
| *iCourier* | Fuente Courier (ancho de caracteres fijos) |
| *iDefault* | Cualquier fuente ("Don't care") |
| *iRoman* | Fuente romana (fuente exótica) |
| *iSwiss* | Swiss (fuente de calidad) |



**`<Color>: (Entero opcional)`**

Color utilizado para la fuente. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Angle>: (Entero opcional)`**

Ángulo (en grados) según el cual se imprimirá el texto. El ángulo se calcula a partir de una línea horizontal en sentido contrario a las agujas del reloj (dirección trigonométrica).

## Observaciones




### Fuente predeterminada
<a name="fuente_predeterminada_ELTPARAGRAPHE000238"></a>

Si no se utiliza la función **iCreateFont**, la fuente predeterminada tiene las siguientes características: 

- Fuente #0,

- envergadura: 12, 

- Estilo Normal.

### Fuente y familia
<a name="fuente_familia_ELTPARAGRAPHE000253"></a>

Con algunas configuraciones, la función [iPrint](../WDLang5/3046031.md) puede producir resultados inesperados si se indicó el nombre de una familia de fuentes en la función **iCreateFont**. Se recomienda especificar el nombre de una fuente en **iCreateFont**.
<a name="NOTE0_3"></a>




### Descubrir la lista de fuentes instaladas en un ordenador
<a name="descubrir_lista_fuentes_instaladas_ordenador_ELTPARAGRAPHE000268"></a>

Para conocer la lista de fuentes instaladas en un ordenador:

1. En el administrador de programas de Windows, seleccione el grupo principal.

2. Haga clic en el "Panel de control" Icon y seleccione "Fuentes".

3. En la lista de fuentes instaladas, escriba los nombres sin tener en cuenta los atributos y los tipos.


Observaciones:

### Diferencias entre FontCreate e iCreateFont
<a name="diferencias_entre_fontcreate_icreatefont_ELTPARAGRAPHE000294"></a>

[FontCreate](../WDLang1/3060002.md) crea un tipo de letra Definition que puede utilizarse en todos los elementos de una aplicación WINDEV o WEBDEV. Esta fuente se almacena con el tipo [tipo de letra](../Motscles/1514045.md) Variable y puede utilizarse:

- para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- para seleccionar una fuente de impresión ([iFont](../WDLang5/3046014.md)).

- para establecer el tipo de letra utilizado en los controles de una ventana ([Fuente de los controles de la ventana](../Motscles/1514045.md) property).

- para establecer el tipo de letra utilizado en los controles de un reporte ([Fuente de los controles reporte](../Motscles/1514045.md) property).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) para seleccionar el tipo de letra utilizado en los gráficos ([grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md), [grAxisTitleFont](../WDLang3/3042059.md)).




**iCreateFont** sólo puede crear una fuente que pueda ser utilizada por las funciones de impresión.
<a name="NOTE0_5"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000343"></a>

- con **iCreateFont** se pueden crear tantas fuentes como sea necesario (límite de memoria).

- Debe crear tantas fuentes como el número de colores que se encuentren en el texto.

- Al imprimir en un archivo HTML o RTF ([iPreview](../WDLang5/3046069.md) o [iDestination](../WDLang5/3046074.md)), las fuentes verticales (a 90º) se sustituyen por fuentes horizontales (a 0º): el texto se imprime horizontalmente.



<a name="NOTE0_7"></a>


### Liberando las fuentes creadas por iCreateFont
<a name="liberando_las_fuentes_creadas_por_icreatefont_ELTPARAGRAPHE000360"></a>

Cuando ya no se utilizan, las fuentes creadas con **iCreateFont** pueden ser destruidas por [iReset](../WDLang5/3046067.md) para liberar recursos. Si no se utiliza [iReset](../WDLang5/3046067.md), las fuentes se destruirán automáticamente cuando se cierre la aplicación.

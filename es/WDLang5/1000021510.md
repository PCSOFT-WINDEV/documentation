


## &lt;xlsDocument Variable&gt;.Cerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un archivo XLS.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyXLSFile is xlsDocument

// Open an XLS file
MyXLSFile = xlsOpen("C:\MyDirectories\File.XLS")
...
// Close the XLS file
MyXLSFile.Close()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<XLS document>.Close()`
---

**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS a cerrar. El archivo físico asociado a esta Variable (si existe) está cerrado. Las modificaciones realizadas en el fichero no se guardarán hasta que se cierre (utilice [&lt;xlsDocument Variable&gt;.Guardar](../WDLang5/1000021511.md) si es necesario).

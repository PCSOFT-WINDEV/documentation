


## Funciones para la gestión de documentos de Google (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Para gestionar documentos en un servidor de Google se utilizan las siguientes funciones: 



|   |   |
| --- | --- |
| [GglGetDocument](../WDLang5/1000020437.md) | Descarga un Document del servicio de Google Docs. |
| [GglListDocument](../WDLang5/1000020436.md) | Recupera la lista de documentos disponibles en el servidor de Google para el usuario actual. |
| [GglPrintDocument](../WDLang5/1000020420.md) | Imprime un Document (archivo de texto o PDF, imágenes, Word o Excel Document, ...) usando el servicio Cloud Print de Google. |
| [GglUploadDocument](../WDLang5/1000020424.md) | Sube una Document al servidor de Google. |





También tiene la posibilidad de utilizar las siguientes funciones: 


|   |   |
| --- | --- |
| [&lt;gglConnection Variable&gt;.Borrar](../WDLang5/1000021461.md) | Borra un Google Document del servidor. |
| [&lt;gglConnection Variable&gt;.Solicitud](../WDLang5/1000021462.md) | Envía una solicitud de comunicación (solicitud HTTP) a un servicio de Google. |


**Advertencia**: Antes de utilizar una función de relacionado para los servicios de Google, le recomendamos encarecidamente que compruebe la licencia para utilizar este servicio.. Se pueden aplicar algunas restricciones. El contenido de las licencias puede cambiar con el tiempo.

PC SOFT no es en ningún caso responsable de la forma en que se utilizan las funciones de acceso nativas.. Asegúrese de que cumple con la licencia del proveedor de servicios.

**Atención**: Si utiliza un proxy para acceder a INTERNET, el proxy debe estar configurado ([Proxy](../WDLang3/3043002.md)) para utilizar las funciones de Google.


## rssInitialize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga un flujo RSS en la memoria. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyRSSStream is rssStream
MyRSSStream = rssInitialize("http://blogs.webdev.info/rss.awp?blog=technicalsupport", fromURL)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = rssInitialize(<Source of RSS Stream> [, <Type of source>])`
---

**`<Result>: (rssStream Variable)`**

[rssStream](../WDLang5/1000017795.md) Variable que contiene las características del flujo RSS especificado. Si la inicialización del flujo falla, el flujo está vacío y [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.

**`<Source of RSS Stream>: (Cadena de caracteres)`**

Fuente del flujo RSS utilizado para la inicialización. Este parámetro puede corresponder a:

- Una cadena de caracteres Ansi o un búfer que contiene el contenido de la secuencia RSS. En este caso, el parámetro &lt;Tipo de fuente&gt; debe corresponder a la constante *fromString*.

- Una cadena de caracteres que contiene la ruta completa de un archivo XML.. Este archivo debe ser accesible desde el ordenador actual. En este caso, el parámetro &lt;Tipo de fuente&gt; debe corresponder a la constante *fromFile*.

- Una cadena de caracteres que contiene el URL Address. El contenido del flujo RSS se recupera desde esta URL. En este caso, el parámetro &lt;Tipo de fuente&gt; debe corresponder a la constante *fromURL*.




**`<Type of source>: (Constante opcional de tipo Integer)`**

Indica la fuente del flujo RSS que se va a cargar.  


|   |   |
| --- | --- |
| *fromFile* | El flujo RSS para cargar está contenido en un archivo. |
| *fromString* | El flujo RSS a cargar está contenido en una cadena de caracteres Ansi o en un búfer. |
| *fromURL* | El flujo RSS para cargar es accesible por medio de una URL. |





Si no se especifica este parámetro, se realiza una detección automática. El tipo de la fuente se deduce automáticamente de &lt;Fuente de flujo RSS&gt;:

- si &lt;Fuente de flujo RSS&gt; comienza por "http://", se utilizará la constante *fromURL* automáticamente.

- si &lt;Fuente de flujo&gt; comienza por una letra de unidad o una ruta UNC, se utilizará la constante *fromFile* automáticamente.

- de lo contrario, la constante que se utilizará automáticamente será *fromString* .  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Actualmente sólo se admiten los flujos en formato RSS 2.0.

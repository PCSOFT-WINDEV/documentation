
## rssEntry (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssEntry** se utiliza para manipular una entrada de flujo RSS. El flujo RSS se describe a través del tipo [rssStream](../WDLang5/1000017795.md).

Las características de una entrada de flujo pueden definirse y modificarse mediante las propiedades de WLanguage.  

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyStream is rssStream
MyChannel is rssChannel
MyEntry is rssEntry
MyPath is string

// Characteristics of channel
MyChannel.Title = "My site"
MyChannel.Description = "Example of RSS 2.0 stream"
MyChannel.UpdateDate = DateSys() + TimeSys()
MyChannel.Link = "http://www.example.org"
 
// Characteristics of entries of channel 1
MyEntry.Author = "pcsoft@pcsoft.fr"
MyEntry.Description = "First news"
MyEntry.Title = "News #1"

// Adds the entry into the channel
Add(MyChannel.Entry, MyEntry)

// Adds the channel into the stream
Add(MyStream.Channel, MyChannel)


// Saves the stream in a file
MyPath = CompleteDir(fCurrentDir()) + "Rss.xml"
rssSave(MyStream, MyPath)

// Displays the XML file of stream
IF NOT ShellExecute(MyPath) THEN
	Error("ShellExecute('" + MyPath + "'). '" + ErrorInfo() + "'")
END
```

## Observaciones




### Propiedades específicas de la description de las variables rssEntry
<a name="propiedades_especificas_description_las_variables_rssentry_ELTPARAGRAPHE000058"></a>

Las siguientes propiedades pueden utilizarse para manipular la entrada de un flujo RSS con una variable de tipo **rssEntry**:

   | --- |
| --- | --- | --- |
| Property | Tipo utilizado | Efecto |
| Autor | Cadena de caracteres | Envía un correo electrónico a Address del autor de la entrada. |
| Categoría | [rssCategory](../WDLang5/1000017782.md) | Categoría a la que pertenece la entrada. Puede utilizarse para agregar los elementos sobre el mismo tema. |
| Comentarios | Cadena de caracteres | URL de la página que contiene los comentarios sobre la entrada. |
| Fecha de publicación | FechaHora | Fecha de publicación del contenido de la inscripción. <br>Las fechas leídas se convierten automáticamente en fechas UTC. El valor escrito en el canal está siempre en UTC. |
| description (\*) | Cadena de caracteres | Contenido de la entrada. <br>Las etiquetas HTML están permitidas. |
| Recinto | [rssEnclosure](../WDLang5/1000017789.md) | Describe un Document externo (multimedia en la mayoría de los casos) associated con una entrada. |
| Identificador | [rssIdentifier](../WDLang5/1000017797.md) | Identificador (potencialmente único y permanente) de la entrada. A menudo se utiliza para identificar las entradas ya leídas/procesadas. |
| vincular | Cadena de caracteres | Enlace externo hacia el elemento descrito en la entrada. |
| Fuente | [rssSource](../WDLang5/1000017803.md) | description de la fuente de publicación de la entrada. En el caso del agregado, este valor difiere de la URL del canal que actualmente contiene la entrada. |
| Título (\*) | Cadena de caracteres | Título del canal. <br>El valor de esta Property puede ser recuperado y modificado. |


**Observaciones**: 

- Las propiedades seguidas de una estrella (\*) son obligatorias.

- Todas las propiedades que se encuentran en esta lista están disponibles en lectura/escritura.

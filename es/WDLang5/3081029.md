


## XMLToText (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un string en formato XML en un string estándar.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Convert a character string in XML format
// into a character string in ANSI format
sAnsiString is string
sAnsiString = XMLToText("X is &gt; than Y")
// sAnsiString contains "X is > than Y"
```

MyResult is string

MyResult = XMLExtractString(MyDoc, "NAME", i)

	Trace(MyResult)
	i = i + 1

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLToText(<String to convert>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres. Los siguientes caracteres reservados se sustituyen automáticamente:

- & se vuelve & y

- ' se convierte en '.

- &gt; se convierte en &gt; 

- &lt; se convierte en &lt; 

- "se vuelve" "




**`<String to convert>: (Cadena de caracteres)`**

Cadena de caracteres en formato XML para convertir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para extraer una sección de código XML que se encuentra en una etiqueta XML, utilice [XMLExtractString](../WDLang5/3081017.md).

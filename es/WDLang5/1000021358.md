
## mlLoadWorkspace( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga todas las variables de un archivo de MATLAB en la sesión. El espacio de trabajo de la sesión de MatLab se inicializa cargando todas las variables de archivo (archivo en formato Mat-File cuya extensión es.MAT).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nSessionID is int = mlInitialize()
IF ErrorOccurred THEN
	ErrorInfo(errFullDetails)
	RETURN
END
mlLoadWorkspace(nSessionID, "C:\temp\workspace.mat")
sVarList is string = mlListVariable(nSessionID)
mlEnd(nSessionID)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = mlLoadWorkspace(<Session identifier> , <File name> , <Variables stored>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la carga se realizó correctamente.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Session identifier>: (Integro)`**

Identificador de la sesión de MATLAB. Este identificador es devuelto por [mlInitialize](../WDLang5/1000021351.md). 

**`<File name>: (Cadena de caracteres)`**

Ruta y nombre del archivo Mat-File a utilizar. 

**`<Variables stored>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para conservar las variables existentes. El Variable existente se almacena excepto si se redefine en el archivo. En este caso, se sobrescribe la Variable de la sesión.

- <u><u><u><u>False</u></u></u></u> para no conservarlas (valor predeterminado). Las variables encontradas en la sesión se borran antes de la operación de carga.

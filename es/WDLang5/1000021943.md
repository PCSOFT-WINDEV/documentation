


## &lt;fbSession Variable&gt;.Stop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la sesión de Facebook abierta con [&lt;Variable fbSession&gt;.Start](../WDLang5/1000021944.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySession is fbSession
// Connection to Facebook
...
// Disconnection
MySession.Stop()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Session>.Close()`
---

**`<Session>: (fbSession variable)`**

Nombre de [fbSession](../WDLang5/1000021851.md) Variable correspondiente a la conexión a cerrar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La sesión se detiene automáticamente al cerrar la aplicación. 

- Para volver a utilizar los servicios de Facebook, debes volver a conectarte con [&lt;Variable fbSession&gt;.Start](../WDLang5/1000021944.md).

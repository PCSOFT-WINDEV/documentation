
## iReportPrintingStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de la impresión actual.

Esta función debe utilizarse en el código de cierre de reporte.



**Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// End of print
SWITCH iReportPrintingStatus() 
	CASE -2: Trace("Print canceled")
	CASE 0: Trace("Print in progress")
	CASE 1: Trace("Empty report: no data to print")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iReportPrintingStatus()`
---

**`<Result>: (Integro)`**

Estado de la impresión actual:

- 1: No hay datos para imprimir

- 0: Imprimiendo

- -1: No hay impresión actual

- -2: Cancelación

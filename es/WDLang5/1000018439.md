
## sfObJETO (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfObject** permite describir un registro de un archivo Salesforce. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

Este tipo de Variable es utilizado por los siguientes tipos de Variable s:

- [sfMergeRequest](../WDLang5/1000018506.md).

- [sfExecuteQueryResult](../WDLang5/1000018558.md).

- [sfSearchResult](../WDLang5/1000018525.md).

- [sfSaveResult](../WDLang5/1000018460.md).




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54" 
IF SFConnect(Cnt) = False THEN
	Error("The connection failed") 
	RETURN
END

ResQuery is sfExecuteQueryResult
ResQuery = SFExecuteQuery(Cnt, "SELECT Date__c,Miles_c,Contact__c FROM Mileage__c")

ARecord is sfObject
FOR EACH ARecord OF ResQuery.Object
	 Trace("---- Type: " + ARecord.ObjectType + " | ID: " + ARecord.ID)
	 Trace(ARecord.Date__c + " / " + ARecord.Miles__c + " / " + ...
		ARecord.Contact__c)
END
```

## Observaciones




### Propiedades específicas de las variables sfObject
<a name="propiedades_especificas_las_variables_sfobject_ELTPARAGRAPHE000057"></a>

Las siguientes propiedades pueden ser usadas para manipular un Record de un archivo de Salesforce:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| ID | Cadena de caracteres | Identificador de la Record. |
| CeroItem | Cadena de caracteres | Lista de artículos de Record sin valor. Los diferentes elementos están separados por caracteres CR (Carriage Return). |
| ObjetoTipo | Cadena de caracteres | Nombre del archivo al que pertenece el Record. Este nombre debe corresponder al nombre definido en la interfaz de Salesforce como "API name"..<br>Para un archivo personalizado, este nombre corresponde al nombre del archivo seguido de 2 caracteres de subrayado y luego de la letra "c".. Por ejemplo: "Mi Cuenta__c". |
| &lt;Nombre del artículo&gt;&gt;Nombre del artículo | Tipo de elemento | Permite acceder a cualquier elemento del archivo directamente a través de su nombre..<br>**Observación**: El nombre del elemento debe corresponder al nombre definido en la interfaz de Salesforce como "API name".. Para un archivo personalizado, este nombre corresponde al nombre del elemento seguido de 2 caracteres de subrayado y luego de la letra "c".. Por ejemplo: "OpenDate__c". |

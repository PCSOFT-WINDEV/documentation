
## xmlReader (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **xmlReader** se utiliza para leer un Document XML sin cargarlo en la memoria. Las características de este Document XML pueden ser leídas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyNode is xmlNode
MyNode..Type = XMLTag
MyNode..Name = "MyNode"
MyNode..Text = "Text of node"

Reader is xmlReader = xmlOpenReader("file.xml")

FOR EACH Reader IN-DEPTH
	Trace(Reader.Type)
END

VariableReset(Reader) // Close file

Reader2 is xmlReader = xmlOpenReader("file.xml")

FOR EACH x OF Reader2
	IF Reader2.Type = XMLReaderStartTag _OR_ Reader2.Type = XMLReaderEmptyElementTag THEN
		IF Reader2.Name = "config" THEN
			FOR EACH y OF x
				Trace(x.Name)
			END
		END
	END
END

VariableReset(Reader2) // Close file
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables xmlReader
<a name="propiedades_especificas_description_las_variables_xmlreader_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades pueden ser usadas para manipular un XML Document:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Atributo | Array de [xmlAttribute](../WDLang5/1000018799.md) | Array de atributos XML associated con el nodo. |
| denominar | Cadena Unicode | Nombre del nodo. <br><br>**Esta propiedad es de solo lectura**. |
| escribir a máquina | Integro | Tipo de nodo XML: <br><br>- *XMLReaderNone*: El elemento de corriente no corresponde a un nodo. <br><br>- *XMLReaderStartTag*: Inicio de etiqueta. <br><br>- *XMLReaderEndTag*: Fin de etiqueta. <br><br>- *XMLReaderEmptyElementTag*: Etiqueta vacía. <br><br>- *XMLReaderCDATA*: Sección CDATA.<br><br>- *XMLReaderComment*: Comentarios. <br><br>- *XMLReaderEntity*: Declaración de entidad. <br><br>- *XMLReaderUnknown*: Tipo desconocido. <br><br>- *XMLReaderProcessingInstruction*: Tratamiento de la expresión<br><br>- *XMLReaderEntityReference*: reference a una entidad. <br><br>- *XMLReaderText*: texto. <br><br>- *XMLReaderDocumentType*: Declaración DOCTYPE. <br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Namespace | Variable de tipo [xmlNamespace](../WDLang5/1000018803.md) | Namespace del nodo<br><br>**Esta propiedad es de solo lectura**. |
| NamespaceDeclared | Array de [xmlNamespace](../WDLang5/1000018803.md) | Array de los namespaces XML declarados por el nodo. |
| Profundidad | Integro | Profundidad del elemento de corriente en el árbol XML. <br><br>**Esta propiedad es de solo lectura**. |
| Valor | Cadena Unicode | Valor del nodo. <br><br>Observación: Las referencias a una entidad se interpretan automáticamente. <br><br>**Esta propiedad es de solo lectura**. |

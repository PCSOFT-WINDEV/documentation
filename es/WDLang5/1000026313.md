


## &lt;htmlNode Variable&gt;.FindElementByTag (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca elementos que correspondan a una etiqueta en un nodo HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlNode
// ...
t is array of htmlNode <- d.FindElementByTag("form")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Element to use>.FindElementByTag([<Search element>])`
---

**`<Result>: (Array)`**

Array de los elementos de [htmlNode](../WDLang5/1000026081.md) que coinciden con la condición de búsqueda.

**`<Element to use>: (htmlNode Variable)`**

Nombre de la [htmlNode](../WDLang5/1000026081.md) Variable en la que se realizará la búsqueda.

**`<Search element>: (Cadena de caracteres opcional)`**

Nombre de la etiqueta a buscar. Si este parámetro no se especifica o corresponde a una cadena vacía (""), &lt;Resultado&gt; contendrá todos los elementos.




## XMLExtractDocument( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un nuevo Document XML a partir de un Document XML existente.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En el último XML Document, la lista de atributos está ordenada por orden alfabético.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
XMLSource is string
Res is boolean
// Load the XML content
XMLSource = fLoadText("C:\Temp\example.xml")

// Create the XML document
Res = XMLDocument("XMLDoc", XMLSource)

// Was the document created?
IF Res = False THEN
	Error("The following problem was detected : " + ErrorInfo())
ELSE
	Res = XMLExtractDocument("XMLDoc", "MyNewDoc")
	IF Res = False THEN
		Error("The following problem was detected : " + ErrorInfo())
	ELSE
		Info("New document successfully created")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLExtractDocument(<Source XML document> , <New XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el nuevo documento XML,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el [ErrorOccurred](../WDLang1/3087001.md) Variable es set a <u><u><u><u>True</u></u></u></u>). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source XML document>: (Cadena de caracteres)`**

Nombre del Document XML a partir del cual se creará el nuevo Document XML. El nuevo XML Document se creará a partir de la posición actual en el Document (en el nivel de rama actual).

**`<New XML document>: (Cadena de caracteres)`**

Nombre del Document XML a crear.

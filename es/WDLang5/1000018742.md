
## sfLeadConvert (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfLeadConvert** se utiliza para describir los parámetros de conversión de una pista. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de las variables sfLeadConvert
<a name="propiedades_especificas_las_variables_sfleadconvert_ELTPARAGRAPHE000036"></a>

Las siguientes propiedades pueden ser utilizadas para manipular el description de la conversión de un plomo:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| SobreescribirLeadSource | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para sobrescribir el campo "Origen del candidato" del contacto de destino con el contenido del elemento "Origen del candidato" del candidato a convertir,<br><br>- <u><u><u><u>False</u></u></u></u> para no sobrescribir esta información.<br><br><br> |
| EnviarNotificaciónEmail | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si se debe enviar una notificación de conversión por correo electrónico al administrador del candidato,<br><br>- <u><u><u><u>False</u></u></u></u> para no realizar ninguna acción.<br><br><br> |
| AccountID | Cadena de caracteres | Identificador de la cuenta en la que se fusionará el cliente potencial. Si se especifica esta Property, la cuenta especificada se actualizará. Si este Property corresponde a una cadena vacía, se creará una nueva cuenta.<br>**Importante**: Si se especifica el "AccountID" Property, no es necesario especificar el "ContactID" Property. |
| IDContacto | Cadena de caracteres | Identificador de la Contact en la que se fusionará el plomo. Si se especifica este Property, el Contact especificado será actualizado. Si este Property corresponde a una cadena vacía, se creará un nuevo Contact. |
| LeadID | Cadena de caracteres | Identificador del plomo a convertir. Esta Property debe ser necesariamente especificada. |
| OwnerID | Cadena de caracteres | Identificador de la persona que posee la nueva cuenta, Contact u oportunidad. Por Default, el propietario es el dueño de la pista. |
| DoNotCreateOpportunity | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para no crear oportunidades durante la conversión,<br><br>- <u><u><u><u>False</u></u></u></u> para crear una oportunidad.<br><br><br>Por Default, se crea automáticamente una oportunidad. |
| OportunidadNombre | Cadena de caracteres | Nombre de la oportunidad de crear. Si este Property corresponde a una cadena vacía, se utiliza el nombre de la empresa associated con el plomo. |
| Estado de la conversión | Cadena de caracteres | Valor del status válido para el cliente potencial convertido. El administrador define los status. Para que se proponga, se debe verificar el status para la columna "Convertido.<br>Este Property es obligatorio.<br>**Observación**: Los posibles valores se pueden conocer mediante programación ejecutando la siguiente consulta con [SFExecuteQuery](../WDLang5/1000018438.md): "SELECT Id, MasterLabel FROM LeadStatus WHERE IsConverted=True"). |

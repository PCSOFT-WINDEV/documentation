
## gglEmail (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglEmail** permite describir y modificar la dirección de correo electrónico de un contacto.

Este tipo de Variable es utilizado por el tipo [gglContact](../WDLang5/1000017434.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection to Google
Cnt is gglConnection
Cnt.Email = "balthazar@gmail.com"
Cnt.Password = "qwerty"
Cnt.ApplicationName = "MyApp-01"
IF GglConnect(Cnt) = False THEN
	Error(ErrorInfo())
END
...
// Retrieve the array of contacts
arrContacts is array of gglContact
arrContacts = GglListContact(Cnt)

// Display the emails of the first contact
AContact is gglContact = arrContacts[1]
AnEmail is dynamic gglEmail
FOR EACH AnEmail OF AContact.Email
	Trace(AnEmail.Caption + ": " + AnEmail.Address)
END
```

## Observaciones




### Propiedades específicas de la description de las variables gglEmail
<a name="propiedades_especificas_description_las_variables_gglemail_ELTPARAGRAPHE000036"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **gglEmail**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Address | Cadena Unicode | Correo electrónico de Address de Contact. |
| Pie de foto | Cadena Unicode | Pie de foto para describir la Address.<br><br>Equivalente a un tipo personalizado. Sólo se puede asignar un valor a una de las dos propiedades: **Caption** o **Type**. |
| Primaria | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el correo electrónico corresponde a la dirección de correo electrónico principal del contacto,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| escribir a máquina | Constante de tipo Integer | Tipo del correo electrónico Address correspondiente a una de las siguientes variables:<br><br>- *gglOther*: Otro tipo de Address.<br><br>- *gglWork*: Negocios Address.<br><br>- *gglHome*: Hogar Address.<br><br><br>Sólo se puede asignar un valor a una de las dos propiedades: **Caption** o **Type**. |

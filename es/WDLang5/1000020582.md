


## iGroupPrint( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la impresión de un reporte encontrado en un grupo de informes.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Prepare a group of reports
nSubscript is int = iGroupAdd(RPT_TO_Year_A4_Portrait, 2019)
iGroupConfigure(nSubscript, "Statistics 2019")
nSubscript = iGroupAdd(RPT_TO_Year_A4_Portrait, 2018)
iGroupConfigure(nSubscript, "Statistics 2018")
iGroupPrint()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Imprimir una reporte identificada por su nombre

`<Result> = iGroupPrint([<Report to be printed>])`
---

**`<Result>: (Tipo de resultado devuelto por el reporte (opcional))`**

Valor devuelto por la impresión de la reporte. Este valor es devuelto: 

- por la palabra clave [RETURN](../Motscles/1510007.md) utilizada en el código de cierre de reporte. 

- por [ReturnedValue](../Proprietes/2510134.md) utilizado antes de cerrar el reporte.




**`<Report to be printed>: (Nombre del reporte)`**

Nombre de reporte a imprimir. En el visor de reporte, se mostrará una marca de verificación delante de la leyenda de este reporte. Esta reporte fue añadida al grupo por [iGroupAñadir](../WDLang5/1000020580.md).

Si no se especifica este parámetro, el reporte impreso corresponde a: 

- el último reporte del grupo elegido por el usuario en el visor de reporte. 

- la primera reporte del grupo (en el orden de adición con [iGroupAñadir](../WDLang5/1000020580.md)) si nunca se ha iniciado el visor reporte.





<a name="SYNTAX2"></a>

### Imprimir una reporte identificada por su identificador

`<Result> = iGroupPrint([<Report subscript>])`
---

**`<Result>: (Tipo de resultado devuelto por el reporte (opcional))`**

Valor devuelto por la impresión de la reporte. Este valor es devuelto: 

- por la palabra clave [RETURN](../Motscles/1510007.md) utilizada en el código de cierre de reporte. 

- por [ReturnedValue](../Proprietes/2510134.md) utilizado antes de cerrar el reporte.




**`<Report subscript>: (Entero opcional)`**

Índice de reporte para imprimir. Este índice fue devuelto por [iGroupAñadir](../WDLang5/1000020580.md) cuando se añadió el reporte al grupo. En el visor de reporte, se mostrará una marca de verificación delante de la leyenda de este reporte.

Si no se especifica este parámetro, el reporte impreso corresponde a: 

- el último reporte elegido por el usuario en el visor de reporte. 

- la primera reporte del grupo (en el orden de adición con [iGroupAñadir](../WDLang5/1000020580.md)) si nunca se ha iniciado el visor reporte.

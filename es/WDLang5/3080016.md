


## xlsMsgError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el título del último error causado por una función XLS.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003080016&name=xlsmsgerror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Open an XLS file
XLSFileID = xlsOpen("C:\MyDirectories\File.XLS")
IF XLSFileID <> -1 THEN
	...
ELSE
	// An error occurred in the XLS file
	ResError = xlsMsgError(XLSFileID)
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### La sintaxis que está usando el tipo xlsDocument

`<Result> = xlsMsgError(<XLS document>)`
---

**`<Result>: (Cadena de caracteres)`**



- Subtítulo del último error que se produjo en el archivo XLS,

- Cadena vacía ("") si no se produce ningún error.




**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de la [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS actual.  


<a name="SYNTAX2"></a>

### Sintaxis conservada por motivos de compatibilidad con la versión 12

`<Result> = xlsMsgError(<XLS file identifier>)`
---

**`<Result>: (Cadena de caracteres)`**



- Subtítulo del último error que se produjo en el archivo XLS,

- Cadena vacía ("") si no se produce ningún error.




**`<XLS file identifier>: (Integro)`**

Identificador del archivo XLS actual, devuelto por [xlsAbrir](../WDLang5/3080017.md).

## Observaciones


### Diferentes errores
<a name="diferentes_errores_ELTPARAGRAPHE000059"></a>

Los errores que pueden producirse en las funciones XLS son los siguientes:

- El archivo XLS no se encuentra, no existe, no está abierto, no es válido o no se puede abrir.

- Las columnas y/o fila especificadas están vacías o no existen.

- Insuficiente memoria para abrir el archivo.

- La versión de Excel utilizada es desconocida o demasiado antigua (las funciones XLS no pueden leer los ficheros de la versión 2 de Excel o anterior).



<a name="NOTE0_2"></a>


### Función equivalente
<a name="funcion_equivalente_ELTPARAGRAPHE000071"></a>

[ErrorInfo](../WDLang1/3013008.md) (con la constante *errMessage*) también devuelve los detalles del último error.

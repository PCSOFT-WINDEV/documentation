
## gglDocument (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglDocument** permite manipular un documento Google. Esta Document fue creada por [GglUploadDocumento](../WDLang5/1000017368.md). Las características de esta Document son devueltas por varias propiedades de WLanguage.  

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
Cnt is gglConnection
Cnt.Email = "balthazar@gmail.com"
Cnt.Password = "qwerty"
Cnt.ApplicationName = "MyApp-03"
IF GglConnect(Cnt) = False THEN
	Error(ErrorInfo())
ELSE
	// Retrieve the array of documents
	arrDocs is array of gglDocument = GglListDocument(Cnt)
	// Browse the documents
	Doc is gglDocument
	FOR EACH Doc OF arrDocs
		Trace(Doc.Title)
	END
END
```

## Observaciones




### Propiedades específicas de las variables gglDocument
<a name="propiedades_especificas_las_variables_ggldocument_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades pueden ser usadas para manipular a Google Document (estas propiedades están disponibles en sólo lectura):

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Identificador | Cadena de caracteres | Identificador de Document. |
| MIMEType | Cadena de caracteres | El tipo de MIME de la Document. <br><br>Observación: Para más detalles sobre las clases MIME autorizadas, véase<br><br>- Tipos estándar: [http://fr.wikipedia.org/wiki/Type_MIME](http://fr.wikipedia.org/wiki/Type_MIME)<br><br>- Tipos específicos de Google Drive: [https://developers.google.com/Drive/web/mime-types](https://developers.google.com/drive/web/mime-types)<br><br><br> |
| Protagonistas | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el documento está marcado como "Destacado",<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| Título | Cadena de caracteres | Título de Document. |
| UpdateDate | FechaHora | Fecha de la última actualización de Document.<br><br>**Observación**: La fecha es una fecha UTC (en hora universal). |

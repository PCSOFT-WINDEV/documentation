
## rssCloud (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssCloud** permite describir la manera para suscribirse a las notificaciones de modificación del flujo RSS. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Este tipo de Variable es el que utiliza el [Tipo rssStream](../WDLang5/1000017795.md).

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssCloud**. Las propiedades de la [Tipo rssStream](../WDLang5/1000017795.md) permiten manipular directamente las propiedades del tipo **rssCloud** (especialmente la Nube Property).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la description de las variables rssCloud
<a name="propiedades_especificas_description_las_variables_rsscloud_ELTPARAGRAPHE000058"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **rssCloud**:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Camino | Cadena de caracteres | Acceder a ruta al servicio. |
| Dominio | Cadena de caracteres | Nombre de la empresa de alojamiento que aloja el servidor. |
| Puerto | Integro | Número del puerto TCP en el que el servicio está escuchando. |
| Procedure | Cadena de caracteres | Nombre de la Procedure del servicio web que se llamará a get las notificaciones. |
| Protocolo | Cadena de caracteres | Tipo de protocolo utilizado por el servicio web. |

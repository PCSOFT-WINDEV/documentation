
## iPrintArea( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime un texto en un área rectangular. El trabajo de impresión no se inicia (solo inicia al utilizar la función [iEndPrinting](../WDLang5/3046053.md)).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046027&name=iprintarea_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
iPrintArea("My Text", 0, 0, dHeight, 200, iCenterH, iCenterV)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iPrintArea(<Text> , <X1> , <Y1> , <X2> , <Y2> [, <Alignment> [, <Start> [, <End>]]])`
---

**`<Result>: (Integro)`**

Número de caracteres impresos en el área rectangular. Si se imprimió todo el texto, &lt;Resultado&gt; corresponde al tamaño de &lt;Texto&gt;.

**`<Text>: (Cadena de caracteres)`**

Cadena de caracteres para imprimir en un área específica.

**`<X1>: (Real)`**

Coordenada horizontal de la esquina superior izquierda del área de impresión (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenada vertical de la esquina superior izquierda del área de impresión (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenada horizontal (columna o abscisa) de la esquina inferior derecha del área de impresión (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenada vertical de la esquina inferior derecha del área de impresión (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Alignment>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Configura la alineación del texto en el área (por defeco *iTop* + *iLeft*).


|   |   |
| --- | --- |
| *iBottom* | Texto alineado en la parte inferior. |
| *iCenterH* | Texto centrado horizontalmente. |
| *iCenterV* | Texto centrado verticalmente. |
| *iLeft* | Texto alineado a la izquierda. |
| *iRight* | Texto alineado a la derecha. |
| *iTop* | Texto alineado en la parte superior |



**`<Start>: (Entero opcional)`**

Índice del primer carácter &lt;Text&gt; a imprimir. Si no se especifica este parámetro, el &lt;Texto&gt; se imprime desde el primer carácter.

**`<End>: (Entero opcional)`**

Índice del último carácter &lt;Text&gt; a imprimir. Si no se especifica este parámetro, el &lt;Texto&gt; se imprime hasta el último carácter.


## iConfigureReport( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la impresora abriendo la ventana de configuración (WINDEV y WINDEV Mobile).. La ventana de configuración se inicializa con los parámetros especificados al crear la reporte (orientación, márgenes, etc.).



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Imprimir en Linux](../Editeurs/9000078.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF iConfigureReport(RPT_MyReport) = True THEN
	iPrintReport(RPT_MyReport)
ELSE  
	Info("The report configuration was canceled")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iConfigureReport(<Report name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario validó la información introducida en el cuadro de diálogo,

- <u><u><u><u>False</u></u></u></u> si el usuario canceló la operación,.




**`<Report name>: (Cadena de caracteres)`**

Las características de esta reporte (especificadas cuando se creó la reporte) se utilizarán para inicializar la ventana de configuración (orientación, etc.)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Configuración que se tiene en cuenta al imprimir una reporte
<a name="configuracion_que_tiene_cuenta_imprimir_una_reporte_ELTPARAGRAPHE000062"></a>

Sólo se tiene en cuenta la última configuración cuando se imprime una reporte.

No se recomienda utilizar varias veces las funciones **iConfigureReport** y [iParameterReport](../WDLang5/3046077.md) sin imprimir reportes.
<a name="NOTE0_2"></a>




### Reinicio de la configuración
<a name="reinicio_configuracion_ELTPARAGRAPHE000077"></a>

Después de llamar a **iConfigureReport** (o [iParameterReport](../WDLang5/3046077.md)), cualquier llamada a esta función reinicializará la ventana de configuración con los parámetros utilizados en la primera llamada.

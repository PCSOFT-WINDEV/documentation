
## iDirImageHTML( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite seleccionar el directorio en el que se deben generar las imágenes durante una impresión en formato HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate the images in the "Temp" sub-directory of the "_WEB" directory
iDirImageHTML(iDirectoryWeb, "Temp")

// Print in HTML format
iDestination(iHTML)

// Print the report
iPrintReport(RPT_MyReport)

// Open the generated HTML file in the browser
FileDisplay(iLastFile())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iDirImageHTML(<Directory> [, <Subdirectory>])`
---

**`<Directory>: (Constante de tipo Integer)`**

Directorio en el que se deben generar las imágenes al imprimir en formato HTML:


|   |   |
| --- | --- |
| *iDirectoryHTML* | Directorio donde se genera el archivo HTML. Este directorio es seleccionado por [iDestination](../WDLang5/3046074.md).<br>Las imágenes que se generan en este directorio **no podrán** mostrarse con la función [FileDisplay](../WDLang2/3012005.md). |
| *iDirectoryWeb* | "&lt;MiProyecto&gt;_WEB" directorio del sitio. Este directorio debe ser accesible en modo lectura/escritura.<br>Las imágenes generadas en este directorio se pueden visualizar con [FileDisplay](../WDLang2/3012005.md). |



**`<Subdirectory>: (Cadena de caracteres opcional)`**

Subdirectorio relativo al directorio especificado en &lt;Directorio&gt;. Si no se especifica ningún subdirectorio, las imágenes se generarán en el directorio especificado en &lt;Directory>.


## gglMapParameter (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglMapParameter** permite definir las características avanzadas de un mapa: puntos, carretera, ... Las características de este mapa pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

Entonces, el mapa correspondiente puede ser recuperado por [GglGetStaticMap](../WDLang5/1000017404.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000017570&name=gglmapparameter_tipo_variable&product=WD">Ver ejemplos adicionales</a>

```wl
// Configure a map
Params is gglMapParameter
// Define the markers
M1 is gglMarker
M1.Latitude = 47.022356
M1.Longitude = 52.423654
M1.Color = 0x0000FF // blue
M1.Size = gglSmall
// Add the markers
Add(Params.Marker, M1)
// Various parameters
Params.Format = gglJPG
Params.Border = True
```

## Observaciones




### Propiedades específicas de las variables gglMapParameter
<a name="propiedades_especificas_las_variables_gglmapparameter_ELTPARAGRAPHE000051"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **gglMapParameter**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Área | [gglCoordinate](../WDLang5/1000017489.md) | Especifica el rango en latitud y longitud que debe mostrarse en el mapa.. Para que se tenga en cuenta este parámetro, el parámetro &lt;Zoom&gt; de la función [GglGetStaticMap](../WDLang5/1000017404.md) debe corresponder a la constante *gglAutoZoom*.<br><br> |
| Camino | [gglPath](../WDLang5/1000017433.md) | Camino a dibujar en el mapa. El dibujo de la ruta utiliza los puntos especificados (y no las calles del mapa). Los puntos son relacionado por un Line. |
| envergadura | Constante de tipo String | Formato de Image recibido. Este parámetro puede corresponder a las siguientes constantes:<br><br>- *gglGIF* (valor Default): Image en formato GIF<br><br>- *gglJPG*: Image en formato JPG.<br><br>- *gglJPGNonProgressive*: Image en formato JPG no progresivo.<br><br>- *gglPNG8*: Image en formato PNG 8<br><br>- *gglPNG32*: Image en formato PNG 32<br><br><br> |
| Escala | Integro | Modifica el número de píxeles devueltos. Esta Property es necesaria para una visualización en una pantalla de alta resolución. <br><br>Si el **Scale** Property es set a 2, devuelve el doble de píxeles que si fuera set a 1, para la misma área cubierta y el mismo nivel de detalle (el contenido del mapa no cambia). <br><br>Valores posibles: <br><br>- 1 (valor predeterminado), <br><br>- 2,<br><br>- 4 (disponible si posee una licencia del Plan Premium de APIs de Google Maps). <br><br><br> |
| Franja de orilla | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para agregar bordes alrededor de la imagen,<br><br>- <u><u><u><u>False</u></u></u></u> para no mostrar bordes.<br><br><br>Las características del borde son las siguientes: borde azul de 5 píxeles de ancho, con una opacidad set a 55%.<br><br>**Observación**: desde la versión 2 del servicio Google, este parámetro ha quedado obsoleto y se ignora. |
| Idioma | Cadena de caracteres | Idioma en el que se mostrará la información en el mapa. Este parámetro puede corresponder a "fr", "en", "de", .... |
| Marker | Array de [gglMarker](../WDLang5/1000017442.md) | Marcadores para mostrar en el mapa. El número máximo de marcadores que se pueden utilizar es de set a 50. Este número se reduce si se añaden puntos (de hecho, el tamaño de la URL esperada por Google limita el número máximo de marcadores). |
| MobileDevice | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para obtener un mapa con una visualización optimizada para móviles,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |

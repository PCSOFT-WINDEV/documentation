


## XMLSavePosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Almacena la posición actual en el XML Document. Esto le permite suspender temporalmente el Process actual para que el Document XML pueda ser utilizado en otro lugar (para realizar comprobaciones, por ejemplo). El contexto guardado puede ser restaurado por [XMLRestorePosition](../WDLang5/3081032.md). Entonces, el principal Process puede continuar.

**Observación**: La función **XMLSavePosition** guarda el filtro actual (el filtro se define con la función [XMLFind](../WDLang5/3081025.md)).

<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003081033&name=xmlsaveposition_funcion&product=WD">Ver ejemplos adicionales</a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
IF XMLDocument("MyXMLDoc", XMLCode) = True THEN
	// Find the links
	XMLFind("MyXMLDoc", "a", XMLTag)
	WHILE XMLFound("MyXMLDoc") = True
		IF XMLChildExist("MyXMLDoc", XMLAttribute) = True THEN
			TheXMLPosition is int = XMLSavePosition("MyXMLDoc")
			XMLChild("MyXMLDoc")
			Trace(XMLData("MyXMLDoc")) 
			// Process
			XMLRestorePosition("MyXMLDoc", TheXMLPosition, XMLRPFilter) 
		END
		// Next element in the search
		XMLNext("MyXMLDoc")
	 END
END
```

## Sintaxis

`<Result> = XMLSavePosition(<XML document>)`
---

**`<Result>: (Integro)`**



- Número de la posición que se ha grabado,

- -1 si se produce un error.




**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Guardar varias posiciones
<a name="guardar_varias_posiciones_ELTPARAGRAPHE000090"></a>

La función **XMLSavePosition** puede utilizarse varias veces para guardar varias posiciones en el archivo XML. Para ello, asigne la posición a diferentes variables enteras. Entonces, estas diferentes posiciones pueden ser restauradas por [XMLRestorePosition](../WDLang5/3081032.md).  
<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000229"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).

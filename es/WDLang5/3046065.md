


## Funciones de impresión
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Atención: Los documentos solo pueden imprimirse en PDF. 

Las funciones de impresión son las siguientes: 



|   |   |
| --- | --- |

| [función iVPos](../WDLang5/3046039.md) | iVPos se mantiene para la compatibilidad con versiones anteriores. |

| [iBorder](../WDLang5/3046044.md) | Imprime una Border en las coordenadas especificadas. |
| [iChangeSubReportSource](../WDLang5/1000019621.md) | Modifica mediante programación la reporte asociada a una sub-reporte en un reporte compuesto. |

| [iCreateFont](../WDLang5/3046045.md) | Crea una nueva fuente de impresión. |
| [iCurrentFile](../WDLang5/1000021859.md) | Devuelve el nombre del archivo generado actualmente. |

| [iEndReport](../WDLang5/3046040.md) | Obliga a detener la impresión de reporte. |




## XMLRestorePosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura el contexto previamente guardado de un XML Document ([XMLSavePosition](../WDLang5/3081033.md)). El filtro actual al guardar la posición puede ser restaurado (el filtro es implementado por [XMLFind](../WDLang5/3081025.md)).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003081032&name=xmlrestoreposition_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
IF XMLDocument("MyXMLDoc", XMLCode) = True THEN
	// Find the links
	XMLFind("MyXMLDoc", "a", XMLTag)
	WHILE XMLFound("MyXMLDoc") = True
		IF XMLChildExist("MyXMLDoc", XMLAttribute) = True THEN
			TheXMLPosition is int = XMLSavePosition("MyXMLDoc")
			XMLChild("myXMLDoc")
			Trace(XMLData("MyXMLDoc")) 
			// Process
			XMLRestorePosition("MyXMLDoc", TheXMLPosition, XMLRPFilter) 
		END
		// Next element in the search
		XMLNext("MyXMLDoc")
	 END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLRestorePosition(<XML document> , <Position> [, <Options>])`
---

**`<Result>: (Integro)`**



- <u><u><u><u>True</u></u></u></u> si se restaura la posición,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el [ErrorOccurred](../WDLang1/3087001.md) Variable es set a <u><u><u><u>True</u></u></u></u>). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

**`<Position>: (Integro)`**

Posición a restaurar. Esta posición fue devuelta por [XMLSavePosition](../WDLang5/3081033.md).

**`<Options>: (Constante o combinación de constantes opcional)`**

Permite configurar la operación de restauración:


|   |   |
| --- | --- |
| *XMLRPDefault*<br>(valor predeterminado) | La posición se libera. Se restablece la posición almacenada. |
| *XMLRPFilter* | Restablece el filtro implementado por [XMLSavePosition](../WDLang5/3081033.md). |
| *XMLRPKeep* | La posición no se libera (se podrán realizar otras llamadas a la función **XMLRestorePosition** en esta posición). |

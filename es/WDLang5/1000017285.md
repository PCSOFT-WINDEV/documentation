


## xlsAddWorksheet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade o inserta una nueva hoja de trabajo en un Excel Document. La hoja de trabajo añadida se convierte en la hoja de trabajo actual. 

**Observación** : Este documento debe ser manejado por un [xlsDocument](../WDLang5/1000017464.md) Variable y debe ser abierto en modo de escritura.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyXLSDocument is xlsDocument
MyXLSDocument = xlsOpen("C:\Temp\Example.xls", xlsWrite)
IF xlsNbWorksheet(MyXLSDocument) = 1 THEN
	xlsAddWorksheet(MyXLSDocument, "MyWorksheet 2")
	xlsSave(MyXLSDocument)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = xlsAddWorksheet(<XLS document> , <Worksheet name> [, <Worksheet position>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó la hoja,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 


Si se realiza la suma, la hoja de trabajo añadida se convierte en la hoja de trabajo actual.

**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de la variable de tipo [xlsDocument](../WDLang5/1000017464.md) que se utilizará.

**`<Worksheet name>: (Cadena de caracteres)`**

Nombre de la hoja de trabajo que se debe añadir o insertar. Este nombre puede contener entre 1 y 31 caracteres. Este nombre no distingue entre mayúsculas y minúsculas.

**`<Worksheet position>: (Entero opcional)`**

Posición de la hoja de trabajo que se debe añadir (incluida entre 1 y 255). 
Si se especifica este parámetro, la hoja de cálculo se inserta en la posición especificada. 
Si se especifica este parámetro, la hoja de trabajo se añade después de las hojas de trabajo existentes..

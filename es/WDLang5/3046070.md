


## iTextHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura de la fuente para el texto a imprimir (en milímetros).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
x is int = iTextHeight(iFont(2) + CU.CUSTNAME)
iPrintWord("The height of customer name is equal to: ")
iPrint(x)
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iTextHeight([<Font identifier> + ] <Text to print>)`
---

**`<Result>: (Real)`**



- Altura del texto especificado (en milímetros) si se realiza la impresión,

- -1 si la impresión se canceló antes de ejecutar la función.




**`<Font identifier>: (Cadena de caracteres opcional)`**

Identificador de la fuente utilizada, devuelto por [iFont](../WDLang5/3046014.md). Si no se especifica este parámetro, se seleccionará la última fuente utilizada.

**`<Text to print>: (Cadena de caracteres)`**

Cadena de caracteres cuya altura se solicita.

Atención: se devuelve una altura incluso si el texto a imprimir corresponde a una cadena vacía.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Cálculo de la altura de un texto escrito con fuentes de diferentes tamaños
<a name="calculo_altura_texto_escrito_con_fuentes_diferentes_tamanos_ELTPARAGRAPHE000065"></a>

Debe llamar a **iTextHeight** para cada texto que utilice una fuente diferente.


### Calculando la altura de un texto que contiene caracteres CR
<a name="calculando_altura_texto_que_contiene_caracteres_ELTPARAGRAPHE000072"></a>

**iTextHeight** ignora los caracteres CR encontrados en el texto. Debes contar el número de caracteres CR para añadir tantas veces la altura del texto a get la altura total.

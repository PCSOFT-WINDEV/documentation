
## iPageEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fuerza el movimiento a la página siguiente durante la impresión de reporte.

**Observaciones**:

- Para forzar el salto de página, también puede seleccionar "Salto de página después del bloque" en el bloque description (pestaña "UI").

- Esta función sólo se puede utilizar con los informes creados en el editor de reportes.






**Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Page break
iPageEnd()
```

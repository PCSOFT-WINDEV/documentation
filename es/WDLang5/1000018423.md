
## sfUserInfo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfUserInfo** permite obtener información sobre un usuario. Las características de este tipo de Variable son devueltas por varias propiedades de WLanguage.

Este tipo es utilizado por las variables de [sfSessionInfo](../WDLang5/1000018416.md). Las propiedades de este tipo se rellenan después de la llamada a [SFConnect](../WDLang5/1000017583.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
ELSE
	Trace("Name of connected user: " +  ...
			  Cnt.SessionInfo.UserInfo.FullName)
END
```

<a name="propiedades_especificas_descripcion_las_variables_sfuserinfo_ELTPARAGRAPHE000050"></a>

| --- | --- | --- |

| RoleID | Cadena de caracteres | Identificador de la posición (rol) ocupada por el usuario en la empresa.<br>**Esta propiedad es de solo lectura**. |
| OrganizaciónID | Cadena de caracteres | Identificador de la empresa.<br>**Esta propiedad es de solo lectura**. |
| UserID | Cadena de caracteres | Identificador de usuario.<br>**Esta propiedad es de solo lectura**. |

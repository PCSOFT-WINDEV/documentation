
## iPrintPDF( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime el contenido: 

- de un archivo PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de un búfer que contiene un PDF.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de un lector de PDF control. 




**¿Cuál es el destino de impresión?**

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) La impresora Default se utiliza. La impresora a utilizar puede seleccionarse a través de [iConfigure](../WDLang5/3046072.md).

	- 
			![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [iDestination](../WDLang5/3046074.md) se utiliza para previsualizar el archivo PDF (véase [Observaciones](#NOTE0_4)).










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF iPrintPDF("MyFile.PDF") THEN
	Info("Printed successfully")
END
```


<a name="Example2"></a>




<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Print a buffer
MyPDFBuffer is Buffer
MyPDFBuffer = fLoadBuffer("C:\temp\Myfile.pdf")
IF iPrintPDF(MyPDFBuffer) THEN
	Info("Printed successfully")
END
```

## Sintaxis
<a name="SYNTAX1"></a>

### Imprimir un archivo PDF

`<Result> = iPrintPDF(<Name of PDF file> [, <Password> [, <Sizing> [, <List of pages>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Name of PDF file>: (Cadena de caracteres)`**

Nombre y ruta del archivo PDF a imprimir. Este parámetro puede corresponder a una ruta completa o relativa con respecto al directorio actual (devuelto por la función [fCurrentDir](../WDLang1/3036022.md)). Este parámetro distingue entre mayúsculas y minúsculas. 
**Recordatorio**: En iPhone/iPad, una aplicación solo tiene derecho a escribir en su directorio de instalación o en uno de sus subdirectorios.

**`<Password>: (Cadena de caracteres)`**



- Contraseña para abrir el archivo PDF,

- Cadena vacía ("") si el fichero no tiene contraseña.




**`<Sizing>: (Constante de tipo Integer)`**

Tamaño de impresión: 



|   |   |
| --- | --- |
| *iHomothetic* | Ampliación o reducción homotética de las páginas del PDF Document para adaptarlas al formato de salida, ignorando los márgenes físicos. <br>Esta constante corresponde a la opción "Ajustar al tamaño del papel" disponible en el software de impresión PDF. |
| *iHomotheticNoEnlargement* | Reducción homotética de las páginas del PDF Document para adaptarlas al formato de salida, ignorando los márgenes físicos. Las páginas con un formato más pequeño mantienen su formato y están centradas. <br>Esta constante corresponde a la opción "Ajustar al tamaño del papel" disponible en el software de impresión de PDF. |
| *iHomotheticNoEnlargementPrintableArea* | Reducción homotética de las páginas del PDF Document para adaptarlas al formato de salida, teniendo en cuenta los márgenes físicos. Las páginas con un formato más pequeño mantienen su formato y están centradas. <br>Esta constante corresponde a la opción "Ajustar a los márgenes de la impresora" disponible en el software de impresión de PDF.<br><br> |
| *iRealSize* | Imprimir en tamaño real. Las páginas que utilizan un formato: <br><br>- más pequeños que el formato de salida se centran en el Page. <br><br>- mayores que el formato de salida se truncan y se centran en el Page. <br><br><br> |




**`<List of pages>: (Cadena de caracteres)`**

Lista de páginas para imprimir. Se puede especificar el intervalo de páginas a imprimir separadas por una coma, cada intervalo está en formato "X-Y", donde X e Y son números de páginas.

### Vista previa de la impresión en PDF
<a name="vista_previa_impresion_pdf_ELTPARAGRAPHE000348"></a>

El archivo PDF a imprimir se puede visualizar en el visor de reporte. Simplemente use [iDestination](../WDLang5/3046074.md) con la constante *iViewer* ) antes de **iPrintPDF**. 

Por ejemplo: 


```wl
// Print the PDF in the report viewer
iDestination(iViewer)
iPrintPDF("c:\tmp\Convention.pdf")
```


## xlsLayout (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **xlsLayout** permite definir todas las características de diseño para imprimir una hoja de un documento XLSX en un control Hoja de cálculo o en una variable de tipo [xlsDocument](../WDLang5/1000017464.md). Las características de este diseño se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Current sheet [NoMargins]
xLayout is xlsLayout = PSHEET_SpreadsheetControl.PageLayout
xLayout.LeftMargin = 0
xLayout.RightMargin = 0
PSHEET_SpreadsheetControl.PageLayout = xLayout
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo xlsLayout
<a name="propiedades_especificas_descripcion_variables_tipo_xlslayout_ELTPARAGRAPHE000032"></a>

Las siguientes propiedades pueden utilizarse para manipular diseños:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| HeaderPrinted | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para imprimir el encabezado de la hoja.<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| HeaderVisible | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para que se muestre el encabezado de la hoja en el control Hoja de cálculo.<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| PageHeader | Real | Altura del encabezado de la página en la hoja impresa, expresada en milímetros. |
| PageHeight | Real | Altura de la página impresa, expresada en milímetros. |
| PageWidth | Real | Ancho de la página impresa, expresado en milímetros. |
| BottomMargin | Double real | Margen inferior de la página impresa, expresado en milímetros. |
| RightMargin | Real | Margen derecho de la página impresa, expresado en milímetros. |
| LeftMargin | Real | Margen izquierdo de la página impresa, expresado en milímetros. |
| TopMargin | Real | Margen superior de la página impresa, expresado en milímetros. |
| AdjustmentMode | Constante de tipo Integer | <br><br>- *adjustmentModeNone*: No se realiza ningún tipo de escala. <br><br>- *adjustementModeColumns*: Todas las columnas se ajustan al ancho del control. <br><br>- *adjustementModeRows*: Todas las filas se ajustan a la altura del control. <br><br>- *adjustmentModeRowsAndColumns*: La hoja de trabajo se ajusta a las dimensiones de un control Hoja de cálculo. <br><br><br> |
| LandscapeOrientation | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para definir una orientación horizontal,<br><br>- <u><u><u><u>False</u></u></u></u> para establecer una orientación vertical.<br><br><br> |
| PageFooter | Real | Altura del pie de página en la página impresa, expresada en milímetros. |
| GridlinesPrinted | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para imprimir las líneas de cuadrícula alrededor de las celdas,<br><br>- <u><u><u><u>False</u></u></u></u> para imprimir sin las líneas de cuadrícula.<br><br><br> |
| GridlinesVisible | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para que se muestren las líneas de cuadrícula alrededor de las celdas en el control Hoja de cálculo,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |

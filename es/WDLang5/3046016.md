
## iDidotFont( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona la unidad que se utilizará para la altura de las fuentes (creada por [iCreateFont](../WDLang5/3046045.md)): Punto DIDOT o punto PICA.

El punto PICA es usado por Default.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iDidotFont(True)
// Creates a 48-point Didot font
iCreateFont(1, 48, iBold + iItalic, iRoman)
iDidotFont(False)
// Creates a 32-point Pica font
iCreateFont(2, 32, iBold + iItalic, iRoman)
// Print a text with this font
iPrint(iFont(1) + "Text of 48-point Didot")
// Ends the print job
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iDidotFont([<Didot Unit>])`
---

**`<Didot Unit>: (Booleano opcional)`**

Nueva unidad a utilizar:

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): No apunté.

- <u><u><u><u>False</u></u></u></u>: Punto de pica.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En la mayoría de los casos, **iDidotFont** sólo se llamará una vez, al principio del proyecto.

- 1 punto PICA = 0,353 mm y 1 punto DIDOT = 0,3759 mm (valores aproximados).

- la función **iDidotFont** no tiene ningún efecto: 

	- en los trabajos de impresión realizados con la editor de reportes. 

	- en las impresiones PDF. 
			Estas impresiones utilizan el punto PICA como unidad estándar.


## iFont( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona la fuente Default. Esta fuente será utilizada por [iPrint](../WDLang5/3046031.md), [iPrintWord](../WDLang5/3046034.md), [iTextWidth](../WDLang5/3046033.md), [iTextHeight](../WDLang5/3046070.md), etc.

**Observaciones**:

- Esta fuente debe haber sido creada por [iCreateFont](../WDLang5/3046045.md).

- Esta fuente permanece activa hasta que se vuelva a utilizar la función **iFont**.

- ![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo se administran las fuentes TrueType.






![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Imprimir en Linux](../Editeurs/9000078.md). 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iPrintWord(iFont(2) + CU.LastName + iXPos(200) + CU.Firstname)
iSkipLine
iPrint(iFont(1) + iXPos(100) + CU.Company)
iEndPrinting
```

## Sintaxis

### Usando una fuente dinámica

`<Result> = iFont(<Font name> [, <Immediate calculation>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena control que describe el uso de la fuente especificada.

**`<Font name>: (tipo de letra)`**

Nombre de la fuente Default a utilizar. Este parámetro corresponde a una fuente Variable. Las características de la fuente se definen con la función [FontCreate](../WDLang1/3060002.md) o las [propiedades de gestión de fuentes](../Proprietes/2515045.md).

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo se administran las fuentes TrueType.

**`<Immediate calculation>: (Booleano opcional)`**

Especifica si las fuentes deben seleccionarse inmediatamente (<u><u><u><u>True</u></u></u></u> de Default) o sólo al imprimir (<u><u><u><u>False</u></u></u></u>, si las fuentes deben combinarse, por ejemplo). Para obtener más información, consulte [observaciones](#NOTE0_1).

### Combinación de fuentes
<a name="combinacion_fuentes_ELTPARAGRAPHE000158"></a>

Pueden producirse resultados inesperados al combinar fuentes. Por ejemplo, el siguiente código:


```wl
iFont(2)
iPrint("First Part" + iFont(1) + "Second Part")
```
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_2.GIF)


no es equivalente a:


```wl
iPrint(iFont(2) + "First Part" + iFont(1) + "Second Part")
```
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Police_1.GIF)


En el primer caso, toda la cadena de caracteres se imprimirá con la fuente 1. De hecho, en este caso, **iFont**(1) se ejecuta al construir la cadena a imprimir y por lo tanto antes de que se imprima la cadena de la "Primera parte".

Para obtener un resultado idéntico, se debe utilizar la función **iFont** con el parámetro <u><u><u><u>False</u></u></u></u> (en la primera parte): la función **iFont**(2, False) se ejecutará únicamente durante la impresión.

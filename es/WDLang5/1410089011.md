
## iPrintAreaMarkdown (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime texto Markdown en un área rectangular. El trabajo de impresión no se inicia (solo inicia al utilizar la función [iEndPrinting](../WDLang5/3046053.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iDestination(iViewer)

// Retrieve the height of the area
dHeight is real
dHeight = iAreaHeight(sMarkdownText, 100, iMarkdown)

// Draws the borders
iBorder(0,0, 200, dHeight*2)

// Print the Markdown text inside the borders
iPrintAreaMarkdown(sMarkdownText, 0, 0, 200, dHeight*2)

iEndPrinting()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = iPrintAreaMarkdown(<Markdown text> , <X1> , <Y1> , <X2> , <Y2> [, <Start> [, <End>]])`
---

**`<Result>: (Entero)`**

Índice del último elemento Markdown impreso en el área rectangular (conteo específico de WINDEV y WEBDEV). 

**`<Markdown text>: (Cadena de caracteres)`**

Cadena Markdown a imprimir en un área específica.

**`<X1>: (Real)`**

Coordenada horizontal de la esquina superior izquierda del área de impresión (en milímetros). 
Valor entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenada vertical de la esquina superior izquierda del área de impresión (en milímetros). 
Valor entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenada horizontal (columna o abscisa) de la esquina inferior derecha del área de impresión (en milímetros). 
Valor entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenada vertical de la esquina inferior derecha del área de impresión (en milímetros). 
Valor entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Start>: (Entero opcional)`**

Índice del primer elemento Markdown del &lt;Markdown text&gt; a imprimir. Si no se especifica este parámetro, el &lt;Markdown text&gt; se imprime desde el primer elemento Markdown.

**`<End>: (Entero opcional)`**

Índice del último elemento Markdown del &lt;Markdown text&gt; a imprimir. Si no se especifica este parámetro, el &lt;Markdown text&gt; se imprime hasta llegar al último elemento Markdown.

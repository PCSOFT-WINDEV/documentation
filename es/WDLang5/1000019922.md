
## Watermark (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Watermark** se utiliza para manipular el texto impreso en Watermark en los informes creados con el editor de reportes o mediante programación.

El tipo **Watermark** permite definir las características avanzadas del texto en marca de agua. Las características de este texto pueden ser definidas y modificadas utilizando diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print a watermark in report background
MyWatermark is Watermark

// Text of the watermark
MyWatermark.Text = "Confidential"
// Font used
MyWatermark.Font.Name = "Arial"
// The watermark will be horizontally centered at bottom
MyWatermark.Position = iCenterH + iBottom

// Print in the report viewer
iDestination(iViewer)

// Indicates that the watermark will be printed during the print and in the duplicates
iParameterWatermark(iwatermarkPrinting + iwatermarkDuplicate, MyWatermark)

// Starts printing the report
iPrintReport(RPT_Commercial)
```

## Observaciones


### Propiedades específicas de las variables Watermark
<a name="propiedades_especificas_las_variables_watermark_ELTPARAGRAPHE000041"></a>

Las siguientes propiedades pueden ser usadas para manipular un texto como Watermark:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Ángulo | Constante de tipo Integer | Ángulo de inclinación del texto Watermark. Este ángulo puede corresponder a una de las siguientes constantes: <br><br>- *iFontAngle*: el ángulo de inclinación del texto corresponde al ángulo definido en la fuente. <br><br>- *iDownwardDiagonal*: el texto se imprime en una diagonal de 45° hacia abajo. <br><br>- *iUpwardDiagonal*: el texto se imprime en una diagonal de 45° hacia arriba. <br><br>- *iHorizontal*: el texto se imprime horizontalmente.<br><br><br><br><br> |
| envergadura | Constante de tipo Integer | Tamaño fuente. Este tamaño corresponde a una de las siguientes constantes:  <br><br>- *iAutomatic*: el tamaño de la fuente se elige automáticamente de acuerdo con la longitud del texto de Watermark y el ancho de reporte. <br><br>- *iFontSize*: se utiliza el tamaño definido en la fuente.<br><br><br> |
| Opacidad | Integro | Porcentaje de opacidad del texto Watermark. Este porcentaje varía de 0 (texto de Invisible) a 100 (texto totalmente opaco). |
| texto | Cadena de caracteres | Texto de la Watermark. |
| tipo de letra | Variable de tipo [tipo de letra](../Motscles/1514045.md) | Características del tipo de letra que debe utilizarse para el texto de Watermark. |
| ubicación | Constante de tipo Integer | Constante (o combinación de constantes) usada para definir la posición del texto de Watermark: <br><br>- *iBottom*: el texto se alinea en la parte inferior del área de texto. <br><br>- *iCenterH*: el texto está centrado horizontalmente en el área de texto. <br><br>- *iCenterV*: el texto está centrado verticalmente en el área de texto. <br><br>- *iRight*: el texto se alinea a la derecha del área de texto<br><br><br> |

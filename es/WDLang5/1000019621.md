
## iChangeSubReportSource( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica mediante programación la reporte asociada a una sub-reporte en un reporte compuesto. 

**Observación** : Esta función sólo se puede utilizar cuando se imprime una reporte compuesta.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declaration code of the globals of a composite report
iChangeSubReportSource(SRPT_SubReport1, RPT_Form)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`iChangeSubReportSource(<Subreport> , <Target Report>)`
---

**`<Subreport>: (Cadena de caracteres)`**

Nombre del sub-reporte a modificar. 

**`<Target Report>: (Cadena de caracteres)`**

Nombre de la reporte a asociar con la sub-reporte. Esta reporte se imprimirá cuando se imprima el sub-reporte. Este nombre debe ser un nombre válido. Si este parámetro corresponde a una cadena vacía ("""), no se imprimirá el sub-reporte.

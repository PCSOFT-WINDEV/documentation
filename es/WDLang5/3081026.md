


## XMLElementName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del elemento actual (etiqueta o atributo). Para seleccionar un elemento, utilice las funciones de búsqueda y navegación XML ([XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md), [XMLChild](../WDLang5/3081012.md), ...).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TagName is string
// Retrieves the name of current element
TagName = XMLElementName("XMLDoc")

// Positioned on an element?
IF TagName = "" THEN
	Error("No current tag")
ELSE
	Info("The name of current element is: " + TagName)
END

// Other possibility: 

// Positioned on an element?
IF ErrorOccurred = True THEN
	Error("No current tag")
ELSE
	Info("The name of current element is: " + TagName)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLElementName(<XML document>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del elemento actual.

- Cadena vacía (""") si no hay elemento de corriente. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Identificación del elemento actual
<a name="identificacion_del_elemento_actual_ELTPARAGRAPHE000090"></a>

- El tipo de elemento actual es devuelto por [XMLElementType](../WDLang5/3081031.md).

- El valor del elemento actual es devuelto por [XMLData](../WDLang5/3081004.md).



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000231"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).

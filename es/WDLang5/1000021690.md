
## iPrintSignature ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime una Image de la firma en un área rectangular. El trabajo de impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar el trabajo de impresión).

Observación: Esta firma se utilizará para firmar los archivos PDF generados o los duplicados.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the list of certificates available on the computer
arrCertificate is array of Certificates
arrCertificate = CertificateList()

// Open report viewer
iDestination(iViewer)
iPrintSignature(iSignatureApproval, arrCertificate[1], "My certificate", 10, 10, 200, 200)
iEndPrinting()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Imprimir una firma definida por una Certificate

`<Result> = iPrintSignature(<Type of signature> , <Certificate> , <Label> , <X1> , <Y1> , <X2> , <Y2>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprimió la firma,

- <u><u><u><u>False</u></u></u></u> en caso de error.




**`<Type of signature>: (Constante de tipo Integer)`**

Tipo y formato de la firma: 

- Tipo de firma: 
	


|   |   |
| --- | --- |
| *iSignatureApproval* | Firma de aprobación. Una firma de aprobación corresponde a una marca aplicada por un tercero para indicar que aprueba el contenido del documento. |
| *iSignatureCertification* | Firma de certificación. Una firma de certificación corresponde a un sello aplicado al Document para garantizar su integridad. |



- Formato de la firma (puede combinarse con un tipo de firma): 
	

	- PAdES (PDF Firmas Electrónicas Avanzadas):
			


|   |   |
| --- | --- |
| *iSignature_CAdES_B* | CAdES Básico (Firma Electrónica Avanzada CMS). |
| *iSignature_CAdES_C* | CAdES C (Firma Electrónica Avanzada CMS). |
| *iSignature_CAdES_T* | CAdES Trust Token (CMS Advanced Electronic Signature). |
| *iSignature_CAdES_X* | CAdES X Tipo 2 (Firma Electrónica Avanzada CMS).  |
| *iSignature_CAdES_XL* | CAdES XL Tipo 2 (Firma Electrónica Avanzada CMS). |




**`<Certificate>: (Variable de tipo Certificate)`**

Nombre de la [Certificate](../WDLang1/1000019287.md) Variable que contiene la firma a utilizar.

**`<Label>: (Cadena de caracteres)`**

Texto visualizado en el área de firma. 

**`<X1>: (Real)`**

Coordenada horizontal de la Corner superior izquierda del área de firma (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenada vertical de la Corner superior izquierda del área de la firma (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenada horizontal de la esquina inferior derecha del área de la firma (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenada vertical de la esquina inferior derecha del área de la firma (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

<a name="NOTE0_1"></a>

## Observaciones


- Para imprimir una Image en la zona de la firma, utilice [iPrintImage](../WDLang5/3046017.md): 
	
	```wl
	iPrintSignature(iSignatureApproval, MySignature, "", 150, 200, 250, 250)
	iPrintImage("signature test.jpg", 150, 200, 250, 250)
	```


- En una reporte creada con el editor de reportes, utilice un control Firma para firmar los archivos PDF.. Para obtener más información, consulte [La control Firma](../WDChamp/1410087013.md). 




- **Atención**: Para que Acrobat reconozca un archivo PDF firmado, la organización que crea la firma Certificate debe estar en la lista de confianza aprobada por Adobe ([Lista de confianza autorizada de Adobe](https://helpx.adobe.com/acrobat/kb/trust-services.html)) o en el programa de servicios certificados Document (CDS). 
	Si no se reconoce la Certificate, Acrobat mostrará el siguiente mensaje "Al menos una firma tiene problemas".




## XMLData( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor del elemento actual (etiqueta o atributo XML) o el valor de uno de sus atributos.

Para seleccionar un elemento, utilice las funciones de búsqueda y navegación XML ([XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md), [XMLChild](../WDLang5/3081012.md), ...).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Value of "Product" attribute for the current element
Value is string
Value = XMLData("XMLDoc", "Product")

// Test of value
IF Value = "" THEN
	Error("The Product attribute has no value")
ELSE
	Info("The value of Product attribute is: " + Value)
END
```

## Sintaxis

`<Result> = XMLData(<XML document> [, <Attribute> [, <Search mode>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Valor del elemento actual o valor del atributo especificado.

- Cadena vacía (""):

	- si el elemento actual no tiene valor.

	- si el atributo especificado no es hijo del elemento actual.

	- si no hay ningún elemento de corriente. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

	- si el elemento actual ya es un atributo. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).







**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

**`<Attribute>: (Cadena de caracteres opcional)`**

Nombre del atributo hijo de la etiqueta actual para el que debe devolverse el valor. Este atributo se busca entre los hijos directos del elemento actual. Si no se especifica este parámetro (o corresponde a Null o a una cadena vacía), la función **XMLData** devuelve el valor de la etiqueta actual.

**`<Search mode>: (Constante opcional de tipo Integer)`**

Modo de búsqueda se usa para encontrar el atributo entre los hijos del elemento actual.  


|   |   |
| --- | --- |
| *XMLContains* | Busque el primer atributo cuyo nombre contenga el valor de &lt;Atributo&gt;. |
| *XMLExact*<br>(Valor predeterminado) | Busque el primer atributo cuyo nombre corresponda exactamente al valor de &lt;Atributo&gt;. |
| *XMLIgnoreCase* | Buscar el primer atributo cuyo nombre corresponda al valor de &lt;Atributo&gt; independientemente del caso.<br>Puede combinarse con las constantes *XMLContains*, *XMLStartWith* o *XMLExact.* |
| *XMLStartWith* | Busque el primer atributo cuyo nombre comience con el contenido de &lt;Atributo&gt;. |

## Observaciones


### Identificación del elemento actual
<a name="identificacion_del_elemento_actual_ELTPARAGRAPHE000139"></a>

- El tipo de elemento actual es devuelto por [XMLElementType](../WDLang5/3081031.md).

- El nombre del elemento actual es devuelto por [XMLElementName](../WDLang5/3081026.md).



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000280"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).

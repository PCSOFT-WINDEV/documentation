


## XMLNamespace( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el prefijo del espacio de nombres asociado a la etiqueta actual en un Document XML.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDoc is string
MyDoc = fLoadText("Example.xml")
XMLDocument("example", MyDoc)
IF ErrorOccurred = True THEN
	Error("The file is not in XML format")
	RETURN
END
// Position on the first tag in the file
If XMLFirst() = True THEN
	IF XMLNamespace() <> "" THEN
		Info("The prefix used for the namespace is : " + XMLNamespace(), ...
			"The corresponding URI is : " + XMLNamespaceURI())
	ELSE
		IF ErrorOccurred = True THEN 
			Info("The element read is not a tag")
	  	ELSE
			Info("The tag contains no namespace")
		END
	END
END
XMLClose()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLNamespace([<XML document>])`
---

**`<Result>: (: Cadena de caracteres)`**



- Prefijo del espacio de nombres utilizado para la etiqueta actual,

- Cadena vacía ("") : 

	- si la etiqueta no tiene un prefijo de nombre. 

	- si el XML Document no está posicionado en una etiqueta. La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>.







**`<XML document>: (: Cadena de caracteres opcional)`**

Nombre de XML Document a utilizar. Esta Document contiene el código XML y fue creada por [XMLDocument](../WDLang5/3081008.md). Si no se especifica este parámetro, la función **XMLNamespace** se aplica al último documento XML manipulado en el hilo actual.

## Observaciones


### Usando una etiqueta associated con un espacio de nombre
<a name="usando_una_etiqueta_associated_con_espacio_nombre_ELTPARAGRAPHE000088"></a>

- Una etiqueta asociada a un espacio de nombres puede manipularse con las funciones [XMLData](../WDLang5/3081004.md), [XMLExtractString](../WDLang5/3081017.md)...

- [XMLNamespaceURI](../WDLang5/3081040.md) devuelve el espacio de nombres URI.



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000231"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).

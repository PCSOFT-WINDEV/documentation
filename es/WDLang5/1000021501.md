


## xlsNameToCoordinates( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las coordenadas de una celda, archivo XLS u hoja de cálculo control según su nombre. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
CoordinateCellRow is int 
CoordinateCellRow = xlsNameToCoordinates("C1", xlsRowNum)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = xlsNameToCoordinates(<Name> , <Type>)`
---

**`<Result>: (Integro)`**



- Coordenadas buscadas.

- Cadena vacía si se ha producido un error (coordenadas no válidas). 




**`<Name>: (Cadena de caracteres)`**

Nombre de la celda ("C1" por ejemplo). 

**`<Type>: (Constante de tipo Integer)`**

Tipo de coordenada a devolver: 


|   |   |
| --- | --- |
| *xlsColumnNum* | &lt;El resultado&gt; corresponderá al número de la columna. |
| *xlsRowNum* | &lt;El resultado&gt; corresponderá al número de fila. |




## XMLParent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciones en el elemento padre del elemento actual (tag o atributo). Para seleccionar un elemento, utilice las funciones de búsqueda y navegación XML ([XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md), [XMLChild](../WDLang5/3081012.md), ...).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003081022&name=xmlparent_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Browse performed by XMLFirst and XMLNext ...
// Positions on "<order>" element
XMLLast("XMLOrd")
// moves one level up and positions on the "<number>" element
XMLParent("XMLOrd")
// Retrieves the order number
OrdNum is int = Val(XMLData("XMLDoc")) // contains 1
// Positions on the next element
XMLNext("XMLOrd")
// Retrieves the order date
OrdDate is string = XMLData("XMLOrd") // "11/20/2012"
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLParent(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se encontró el elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario. En caso de error: 

	- La situación actual sigue siendo la misma. 

	- La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).







**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Elementos de manipulación en el código XML
<a name="elementos_manipulacion_codigo_xml_ELTPARAGRAPHE000099"></a>

Después de **XMLParent**, el elemento actual puede ser manejado por [XMLElementName](../WDLang5/3081026.md), [XMLParentName](../WDLang5/3081035.md), [XMLElementType](../WDLang5/3081031.md) y [XMLData](../WDLang5/3081004.md).
<a name="NOTE0_2"></a>


### Uso de XMLParent durante una búsqueda
<a name="uso_xmlparent_durante_una_busqueda_ELTPARAGRAPHE000120"></a>

Si hay una búsqueda en curso, **XMLParent** cancela la búsqueda (ver función [XMLFind](../WDLang5/3081025.md)).
<a name="NOTE0_3"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000266"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).Si la posición actual en un XML Document se modifica en un thread, la posición actual en este XML Document se modifica para todos los thread s.


## sfError (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfError** permite conocer las características de un error devuelto por Salesforce. Las características de estos errores son devueltas por varias propiedades de WLanguage. 

Este tipo de Variable es utilizado por los siguientes tipos de Variable s:

- [sfLeadConvertResult](../WDLang5/1000018739.md).

- [sfDisconnectResult](../WDLang5/1000018560.md).

- [sfExecuteProcessResult](../WDLang5/1000018559.md).

- [sfMergeResult](../WDLang5/1000018499.md).

- [sfUndeleteResult](../WDLang5/1000018526.md).

- [sfSaveResult](../WDLang5/1000018460.md).

- [sfDeleteResult](../WDLang5/1000018546.md).

- [sfEmptyRecycleBinResult](../WDLang5/1000018467.md).




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de la description de las variables sfError
<a name="propiedades_especificas_description_las_variables_sferror_ELTPARAGRAPHE000063"></a>

Las siguientes propiedades pueden utilizarse para detectar un error de Salesforce en manipular a:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Código de error | Cadena de caracteres | Código de error. Este código se utiliza para identificar el error que se ha producido. Para obtener más detalles, consulte la documentación de Salesforce API( palabra clave: StatusCode).<br>**Esta propiedad es de solo lectura**. |
| Mensaje | Cadena de caracteres | Mensaje de error.<br>**Esta propiedad es de solo lectura**. |
| Campo | Cadena de caracteres | Lista de elementos afectados por el error. Los nombres de los artículos están separados por caracteres CR (Carriage Return).<br>**Esta propiedad es de solo lectura**. |

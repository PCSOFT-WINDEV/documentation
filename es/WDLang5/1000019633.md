
## BCDecode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Decodifica la información contenida en un código de barras. Este código de barras está contenido en una Image (Image mostrada en una control Imagen, archivo de una Image o Image en la memoria).

El tipo de código de barras a descodificar depende de la plataforma de ejecución:   

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo se pueden descifrar los códigos QR.




**Atención**: El código de barras debe cubrir una amplia sección de la fuente Image y ser nítido y claramente definido para su descodificación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Decode a QR Code from a Camera control
s is Buffer
s = dSaveImageBMP(CAM_NoName1, inMemory)
bc is BarCode 
bc = BCDecode(s)
IF ErrorOccurred = False _AND_ bc.TypeBarCode = BC_QRCODE 
		_AND_ bc.TypeContent =  bcTypeEmail THEN
    	Info("Email address: " + bc.Content)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BCDecode(<Image>)`
---

**`<Result>: (BarCode Variable)`**

[CódigoDeBarras](../WDLang5/1000019451.md) variable inicializada con la información leída en el código de barras.

**`<Image>: (Cadena de caracteres)`**

Image de código de barras para decodificar. Este parámetro puede corresponder a: 

- el nombre de un control Imagen. 

- el buffer de un image devuelto por [dSaveImagePNG](../WDLang1/3029027.md), [dSaveImageGIF](../WDLang1/3029040.md), [dSaveImageJPEG](../WDLang1/3029038.md) y [dSaveImageBMP](../WDLang1/3029026.md). 

- el nombre de un archivo de Image. 









## Observaciones
<a name="NOTE0_1"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000095"></a>

Para averiguar si la recuperación falló, use: 

- la [ErrorOccurred](../WDLang1/3087001.md) variable,

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) el **TipoCódigoDeBarra** Property del [CódigoDeBarras](../WDLang5/1000019451.md) Variable. Si esta Property difiere del BC_QRCODE, el código de barras no fue detectado o decodificado.



<a name="NOTE0_2"></a>


### Decodificando QRCode
<a name="decodificando_qrcode_ELTPARAGRAPHE000113"></a>

La tabla siguiente presenta la relación entre las propiedades **TypeContent** y **Content** del [CódigoDeBarras](../WDLang5/1000019451.md) Variable: 

| Tipo de contenido | Tipo de valor (..Contenido) | Observación |
| --- | --- | --- |
| *bcTypeText* | Cadena de caracteres | Ejemplo: "¡WINDEV es genial!" |
| *bcTypePhone* | Cadena de caracteres | Ejemplo: "0612345678" |
| *bcTypeSMS* | Cadena de caracteres<br>Los miembros de la [Estructura del SMS](../WDLang3/3068003.md) (SMS.Mensaje y SMS.Número) se rellenarán automáticamente con los valores leídos en el código de barras. El contenido corresponderá a una cadena en el siguiente formato: &lt;Número&gt; + CR + &lt;Mensaje&gt; | Ejemplo: "0612345678&lt;CR&gt;Texto a enviar" |
| *bcTypeURL* | Cadena de caracteres | Ejemplo: "http://www.google.com" |
| *bcTypeMarket* | Cadena de caracteres | Ejemplo: "market://search?q=pnombre:com.company.application" |
| *bcTypeWiFi* | Cadena de caracteres | Cadena de caracteres en el siguiente formato: <br>&lt;SSID&gt; + TAB + &lt;Protocolo de seguridad&gt;+ TAB + &lt;Key&gt;<br>Ejemplo: "MyNetwork&lt;TAB&gt;WPA&lt;TAB&gt;pwd" |
| *bcTypeEmail* | Cadena de caracteres | Ejemplo: "name@Email.com" |
| *bcTypeContact* | Vacío. <br>Los miembros de la [mEstructura de contactos](../WDLang3/1000006111.md) se rellenarán automáticamente con los valores leídos en el código de barras.<br> |   |
| *bcTypeLocation* | variable de tipo [geoPosition](../WDLang3/1000019191.md).<br><br>Las propiedades **Latitud** y **Longitud** de la variable se rellenarán automáticamente con los valores leídos en el BarCode. |   |

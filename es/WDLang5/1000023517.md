
## iPrintXLS( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime la hoja de trabajo actual de una Document en formato "xlsx" o "xls". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyXLSX is string = "c:\temp\Accounts.xlsx"
iDestination(iViewer)
iPrintXLS(MyXLSX)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Imprimir el contenido de un control Hoja de cálculo o xlsDocument Variable

`<Result> = iPrintXLS(<Document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el documento se envió al administrador de trabajos de impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener los detalles del error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Document>: (Tipo correspondiente)`**

XLS o XLSX Document para imprimir. Este Document corresponde a:

- una variable de tipo [xlsDocument](../WDLang5/1000017464.md).

- el ruta de un archivo de hoja de cálculo (archivo XLS o XLSX).

- un memo HFSQL correspondiente a un XLS o XLSX Document.





<a name="SYNTAX2"></a>

### Imprimir el contenido de un "xlsx" o "xls" Document

`<Result> = iPrintXLS(<Path> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el documento se envió al administrador de trabajos de impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener los detalles del error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Path>: (Cadena de caracteres)`**

Ruta de un archivo de hoja de cálculo (archivo XLS o XLSX).

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña para abrir el archivo XLS o XLSX que se va a imprimir. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo se imprime la hoja de trabajo actual de XLS o XLSX Document. Para imprimir otras hojas de trabajo, utilice una variable de tipo [xlsDocument](../WDLang5/1000017464.md) y cambie el número de la hoja de trabajo actual con el valor **Hoja de trabajo** Property.


## iSubstBlock( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sustituye un bloque de la reporte por otro bloque mientras se imprime la reporte.

Una vez que se ha ejecutado **iSubstBlock**, se ejecutarán los siguientes procesos cuando se imprima el bloque inicial:

- "Antes de imprimir" Event del bloque de sustitución.

- imprimir el bloque de sustitución.

- "Después de imprimir" Event del bloque de sustitución.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Substitution of page header
iSubstBlock(PageHeader, Iteration1)
// End of substitution of page header
iSubstBlock(PageHeader, "")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iSubstBlock(<Block to substitute> , <New block>)`
---

**`<Block to substitute>: (Cadena de caracteres)`**

Nombre lógico del bloque a sustituir. Este nombre se definió en la editor de reportes, en el bloque description (pestaña "General", "Nombre" control).

**`<New block>: (Cadena de caracteres)`**

Nombre lógico del bloque utilizado para la sustitución. Este nombre fue definido en el editor de reportes, en el bloque description. 
Si este parámetro corresponde a una cadena vacía (""), la sustitución actual del &lt;Bloque a sustituir&gt; bloque se cancela. 
Si este parámetro es igual a &lt;Bloque a sustituir&gt;, la sustitución actual se cancela.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Sustitución del primer bloque de la cabecera de la página
<a name="sustitucion_del_primer_bloque_cabecera_pagina_ELTPARAGRAPHE000084"></a>

Para sustituir el primer bloque de la cabecera de la página, hay que utilizar **iSubstBlock** en un Event de inicialización del reporte (por ejemplo, en uno de los Event del bloque de cabecera del documento o en el Event de "Apertura" del reporte). Estos eventos se ejecutarán antes de imprimir este bloque.
<a name="NOTE0_2"></a>




### Sustitución del bloque de pie de página
<a name="sustitucion_del_bloque_pie_pagina_ELTPARAGRAPHE000094"></a>

Para evitar problemas de visualización, el bloque Pie de página y su bloque de sustitución deben tener la misma altura.
<a name="NOTE0_3"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000102"></a>

- Usted get el mismo resultado realizando las siguientes operaciones:

	1. Ocultar el bloque ([Visible](../Proprietes/2510138.md) Property).

	2. Utilice la función [iPrintBlock](../WDLang5/3046030.md).




- Para sustituir un bloque de pie de página, se recomienda utilizar bloques de la misma altura.

- En la mayoría de los casos, se recomienda utilizar los planos en la reporte ya que son más fáciles de usar e implementar (en lugar de la sustitución por bloques).. Para obtener más información, consulte [Los aviones](../WDChamp/1013078.md).

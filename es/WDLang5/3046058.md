## Impresión por infrarrojos
<a name="impresion_por_infrarrojos_ELTTEXTE000149"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Si el equipo y la impresora associated están equipadas con un puerto de infrarrojos, el trabajo de impresión se realiza cuando se llama a [iEndPrinting](../WDLang5/3046053.md).

**Recordatorio**: La impresora fue seleccionada previamente por [iDestination](../WDLang5/3046074.md), [iParameter](../WDLang5/3046008.md) o [iConfigure](../WDLang5/3046072.md).

## Impresión mediante un fichero PCL
<a name="impresion_mediante_fichero_pcl_ELTTEXTE000173"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) WINDEV y WEBDEV también permiten imprimir a través de un archivo PCL (Lenguaje de Impresión control).

Para realizar este tipo de impresión:

1. Especifique el archivo PCL a crear ([iDestination](../WDLang5/3046074.md)).
	Por ejemplo:  
	
	```wl
	iDestination(iPCL, "C:\MyDocuments\MyFile.PCL")
	```


2. copy el archivo PCL del PC a la impresora que se utilizará. 
	Por ejemplo:

	- utilizando la función [fCopyFile](../WDLang1/3036004.md).

	- introduciendo el comando Line (en el cuadro de diálogo "Ejecutar" de Windows).
			
		```txt
		Copy "<PCLFile>" "<PrinterUsed>"
		```








**Observación**: El usuario también puede crear un archivo PCL desde la ventana de configuración de la impresora ([iConfigure](../WDLang5/3046072.md) o [iParameter](../WDLang5/3046008.md)).

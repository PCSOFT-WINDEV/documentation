## Presentación
<a name="presentacion_ELTTEXTE000118"></a>
El servicio de contactos de Google se utiliza para gestionar los contactos en INTERNET. WINDEV y WEBDEV permiten crear una aplicación para sincronizar los Contact s de un programa existente con los Contact s de Google Contact.

**Advertencia**: Antes de utilizar una función de relacionado para los servicios de Google, le recomendamos encarecidamente que compruebe la licencia para utilizar este servicio.. Se pueden aplicar algunas restricciones. El contenido de las licencias puede cambiar con el tiempo.

PC SOFT no es en ningún caso responsable de la forma en que se utilizan las funciones de acceso nativas.. Asegúrese de que cumple con la licencia del proveedor de servicios.

## ¿Cómo gestionar los contactos de Google?
<a name="¿como_gestionar_los_contactos_google_ELTTEXTE000148"></a>


### Gestión de los contactos de Google
<a name="gestion_los_contactos_google_ELTPARAGRAPHE000171"></a>

Para gestionar los contactos de Google: 

1. Crear una cuenta de Google si es necesario. Esta cuenta puede ser creada a través de la siguiente Address:  [http://code.google.com/intl/en/apis/maps/signup.html](http://code.google.com/intl/en/apis/maps/signup.html). Atención: el Address de esta página puede haber sido modificado desde la fecha de publicación de esta página.
	La cuenta de Google se identifica con una dirección de correo electrónico y la contraseña de associated. 

2. En el código de su aplicación, cree una Variable de tipo [gglConnection](../WDLang5/1000018009.md). Este Variable contiene las características de la conexión a su cuenta de Google.



<a name="NOTE2_2"></a>


### Creando un Google Contact
<a name="creando_google_contact_ELTPARAGRAPHE000192"></a>

Se puede crear un Contact de Google a través de la interfaz de Google o programando con las funciones de WLanguage.

Para crear un Google Contact con las funciones de WLanguage:

1. Crear una Variable de tipo [gglContact](../WDLang5/1000017434.md).

2. Defina las características de la Contact con las propiedades de la [gglContact](../WDLang5/1000017434.md).

3. Validar la creación de la Contact con [&lt;gglConnection Variable&gt;.Escribir](../WDLang5/1000021460.md).



<a name="NOTE2_3"></a>


### Observación
<a name="observacion_ELTPARAGRAPHE000219"></a>

Si utiliza un proxy para acceder a INTERNET, el proxy debe estar configurado ([Proxy](../WDLang3/3043002.md)) para utilizar las funciones de Google.

## ¿Cómo recuperar un Google Contact?
<a name="¿como_recuperar_google_contact_ELTTEXTE000227"></a>
Para recuperar un Google Contact: 

1. Declarar una array de variables [gglContact](../WDLang5/1000017434.md).

2. Utilice la función [&lt;gglConnection Variable&gt;.ListContact](../WDLang5/1000017757.md). Esta función se utiliza para listar los contactos. Los contactos encontrados se asignan a las variables array de gglContact.




Ejemplo:

```wl
Cnt is gglConnection
...

// Retrieves all the contacts
arrContacts is array of 0 gglContact 
arrContacts = Cnt.ListContact() 
// Browse the contacts
MyContact is gglContact
FOR EACH MyContact OF arrContacts
	Trace(MyContact.Name)
END
```

## ¿Cómo modificar o eliminar los contactos de Google?
<a name="¿como_modificar_eliminar_los_contactos_google_ELTTEXTE000272"></a>


### Principio
<a name="principio_ELTPARAGRAPHE000295"></a>

El principio es muy sencillo: Debe encontrar la Contact a modificar o a suprimir y debe posicionarse sobre esta Contact antes de realizar la operación solicitada.
<a name="NOTE4_2"></a>


### Modificación de un Contact
<a name="modificacion_contact_ELTPARAGRAPHE000305"></a>

Para modificar un Contact:

1. Recuperar la lista de contactos.

2. Encuentra el Contact para modificarlo.

3. Modificar las características de la Contact.

4. Valida las modificaciones con [&lt;gglConnection Variable&gt;.Escribir](../WDLang5/1000021460.md).




Ejemplo:

```wl
Cnt is gglConnection
...

// Retrieves all the contacts
arrContacts is array of 0 gglContact 
arrContacts = Cnt.ListContact() 
// Browse the contacts
MyContact is gglContact
FOR EACH MyContact OF arrContacts
	IF MyContact.Name = "MOORE" THEN
			MyContact.PostalAddress[1].Address = "34080 Montpellier"
			// Actually update the changes on the server
			Cnt.Write(MyContact)
	END
END
```

<a name="NOTE4_3"></a>


### Borrar un Contact
<a name="borrar_contact_ELTPARAGRAPHE000333"></a>

Para borrar un Contact:

1. Recuperar la lista de contactos.

2. Encuentra el Contact para eliminarlo.

3. Eliminar la Contact con [&lt;gglConnection Variable&gt;.Borrar](../WDLang5/1000021461.md).




Ejemplo:

```wl
Cnt is gglConnection
...

// Retrieves all the contacts
arrContacts is array of 0 gglContact 
arrContacts = Cnt.ListContact() 
// Browse the contacts
MyContact is gglContact
FOR EACH MyContact OF arrContacts
	IF MyContact.Name = "MOORE" THEN
			// Deletion
			Cnt.Delete(MyContact)
	END
END
```

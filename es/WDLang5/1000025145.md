


## &lt;Reporte&gt;.InitSubReport (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa los parámetros de un subinforme de un reporte compuesto.. Esta función debe utilizarse cuando un subinforme requiere parámetros como límites inferiores o superiores o parámetros especificados mediante programación.

**Atención**: Esta función sólo se puede llamar cuando se imprime una reporte compuesta..

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opening code of a composite report
SRPT_SubReport1.InitSubReport(nLowerBound, nUpperBound)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Subreport used>.InitSubReport([<Parameter 1> [... [, <Parameter N>]]])`
---

**`<Subreport used>: (Nombre del reporte)`**

Nombre del subinforme a inicializar.

**`<Parameter 1>: (Tipo del parámetro (opcional))`**

Primer parámetro que se debe pasar al subinforme.



**`<Parameter N>: (La enésima parte del parámetro (opcional))`**

Último parámetro que se debe pasar al subinforme.

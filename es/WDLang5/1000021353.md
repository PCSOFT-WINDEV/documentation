
## mlExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Evalúa y ejecuta un comando en una sesión de MATLAB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF gnSessionID = -1 THEN	
	ToastDisplay("MATLAB not initialized")
	RETURN
END

sStringToRun is string = [
a = [ 1 2 3; 2 3 4; 1 2 5]
b = [ 2 1 3 ; 5 0 -2; 2 3 -1]
prod = a * b
]

let sResult = mlExecute(gnSessionID, sStringToRun)

EDT_Result = sResult
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = mlExecute(<Session identifier> , <Command>)`
---

**`<Result>: (Cadena de caracteres)`**

Contenido de la salida estándar devuelta por la ejecución del comando.

**`<Session identifier>: (Integro)`**

Identificador de la sesión de MATLAB. Este identificador es devuelto por [mlInitialize](../WDLang5/1000021351.md). 

**`<Command>: (Cadena de caracteres)`**

MATLAB para ejecutar.

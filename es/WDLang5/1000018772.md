


## &lt;sfConnection Variable&gt;.Buscar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Realiza una búsqueda entre los registros que se encuentran en una base de datos de Salesforce.. La búsqueda se realiza a través de SOSL (Salesforce Object Search Language).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF Cnt.Connect() =False THEN
	Error("The connection failed")
	RETURN
END

Res is sfSearchResult
Res = Cnt.Search("FIND {John Smith} IN NAME FIELDS")
MyRecord is sfObject
FOR EACH MyRecord of Res..Object
	Trace(MyRecord..ID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Salesforce connection>.Search(<Search conditions>)`
---

**`<Result>: (sfSearchResult Variable)`**

Nombre de la [sfSearchResult](../WDLang5/1000018525.md) Variable que contiene los datos del resultado.

**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) que se utilizará.

**`<Search conditions>: (Cadena de caracteres)`**

Código correspondiente a la condición de búsqueda. Esta condición de búsqueda está escrita en SOSL (Salesforce Object Search Language). Para más detalles, consulte la documentación de Salesforce.

## Observaciones


### Posibles casos de error : 
<a name="posibles_casos_error_ELTPARAGRAPHE000195"></a>

- La conexión de INTERNET no es válida.

- La autenticación no se realizó correctamente.

- La respuesta XML del servidor no se puede leer.




Para saber si se ha producido un error, utilice el programa [ErrorOccurred](../WDLang1/3087001.md) Variable.

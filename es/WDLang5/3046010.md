
## iPicaToMM( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte las coordenadas de un punto (expresadas en coordenadas del sistema) en milímetros.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PosX is int = iPicaToMM(50)
PosY is int = iPicaToMM(50, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iPicaToMM(<PICA Position> [, <Direction>])`
---

**`<Result>: (Real)`**

Posición deseada (horizontal o vertical) del punto en milímetros.

**`<PICA Position>: (Real)`**

Posición horizontal o vertical del punto expresada en coordenadas del sistema.

**`<Direction>: (Booleano opcional)`**

Permite configurar la dirección de la conversión:

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): para una posición horizontal.

- <u><u><u><u>False</u></u></u></u>: conversión para una posición vertical.

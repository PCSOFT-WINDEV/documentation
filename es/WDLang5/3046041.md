
## iCloseReport( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la impresión de la reporte actual inmediatamente.. El código WLanguage después de la llamada a la función se ejecuta. Esta función debe utilizarse en uno de los procesos de reporte.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la reporte se imprime en el visor de reporte, la ventana de vista previa se cierra automáticamente.



**Observación**: Esta función sólo se puede utilizar con procesos de informes creados en el editor de reportes.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// End of print 
IF MySelf..Source.CustomerName = "SMITH" THEN
	iCloseReport()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iCloseReport()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para evitar ejecutar el código WLanguage después de **iCloseReport**, utilice la palabra clave [RETURN](../Motscles/1510009.md), por ejemplo.

Ejemplo:


```wl
IF CompletedProcess() = True THEN
	iCloseReport()
	RETURN
END
Info("Hello")
```

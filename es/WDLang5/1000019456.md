
## BCCapture (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Decodifica la información almacenada en un código de barras utilizando la cámara del dispositivo (Android, iPhone, iPad).

**Consejo: Utilizar la control Cámara siempre que sea posible**: 









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Decode a QR Code
bc is BarCode
bc = BCCapture()
IF ErrorOccurred = False _AND_ bc.TypeContent = bcTypeEmail THEN
    	Info("Email address: " + bc.RoughValue)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BCCapture([<Bar code type> [, <Enable flash>]])`
---

**`<Result>: (Variable de tipo BarCode)`**

Variable de tipo [BarCode](../WDLang5/1000019451.md) inicializada con la información leída en el código de barras.



**`<Bar code type>: (Constante opcional de tipo Integer)`**

Tipo de código de barras que se puede decodificar: 


|   |   |
| --- | --- |
| *BC_AZTEC* | Código de barras de tipo Aztec. <br><br> |
| *BC_CODE128* | Código de barras Code128.<br><br> |
| *BC_CODE39* | Código de barras Code39.<br><br> |
| *BC_CODE93* | Código de barras Code93.<br><br> |
| *BC_DATAMATRIX* | Código de barras de tipo DATAMATRIX.<br><br> |
| *BC_EAN128* | Código de barras EAN128.<br><br> |
| *BC_EAN13* | Código de barras EAN13.<br><br> |
| *BC_EAN8* | Código de barras EAN8.<br><br> |
| *BC_INTER2OF5* | Código de barras Interleaved 2 of 5.<br><br> |
| *BC_PDF417* | Código de barras PDF417.<br><br> |
| *BC_QRCODE* | Código de barras de tipo QR Code. |
| *BC_UPCA* | Código de barras UPC-A.<br><br> |
| *BC_UPCE* | Código de barras UPC-E.<br><br> |
| *BC_ALL* | Todos los tipos de códigos de barras soportados por esta función se pueden decodificar. |


Si no se especifica este parámetro, se pueden decodificar todos los tipos de códigos de barras soportados por esta función. 



**`<Enable flash>: (Booleano opcional)`**

### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000262"></a>

Para determinar si se pudo leer el código de barras, utilice la variable [ErrorOccurred](../WDLang1/3087001.md).

La función generará un error en los siguientes casos: 

- si se utiliza en el simulador,

- si se utiliza en el emulador,

- si se utiliza en un dispositivo sin cámara.

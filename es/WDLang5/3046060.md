
## iPageHeightLeft( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura disponible (en milímetros) a la izquierda de la página actual teniendo en cuenta los márgenes definidos (superior o inferior).

**Observación**: Esta función sólo se puede utilizar cuando se imprime un reporte. En caso contrario, la función **iPageHeightLeft** siempre devuelve 0.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Page break if the block does not fit
IF Block..Height > iPageHeightLeft() THEN
	iPageEnd()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iPageHeightLeft()`
---

**`<Result>: (Real)`**

Altura disponible a la izquierda de la página para imprimir (en milímetros).

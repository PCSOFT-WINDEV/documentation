


## iImageHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura de la Image a imprimir (en milímetros). Este cálculo se realiza en función de la resolución de la impresora seleccionada (y no en función de la resolución de la pantalla).. No se aplica ningún factor de ampliación.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo se pueden utilizar imágenes en formato JPEG.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Prints the dimensions of the image (in mm)
iPrint("Width of image: " + iImageWidth("WINDEV.JPG") + "mm")
iPrint("Height of image: " + iImageHeight("WINDEV.JPG") + "mm")
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Calculando la altura de un Image encontrado en un archivo

`<Result> = iImageHeight(<Image name>)`
---

**`<Result>: (Real)`**

Altura de Image (en milímetros).

**`<Image name>: (Cadena de caracteres)`**

Nombre y apellido completo (o familiar) ruta del archivo Image.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo se pueden utilizar imágenes en formato JPEG.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- [iImageWidth](../WDLang5/3046061.md) obtiene el ancho de una página para imprimir.

- **iImageHeight** no puede utilizarse para get la altura de un WMF Image.

- Las dimensiones de una Image son devueltas por [BitmapInfo](../WDLang1/3029015.md).




## iParameterWatermark( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un Watermark en un reporte (reporte creado en el editor de reportes o escrito en WLanguage). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print a watermark in report background
MyWatermark is Watermark

// Text of the watermark
MyWatermark.Text = "Confidential"
// Font used
MyWatermark.Font.Name = "Arial"
// The watermark will be horizontally centered at bottom
MyWatermark.Position = iCenterH + iBottom

// Print in the report viewer
iDestination(iViewer)

// Indicates that the watermark will be printed during the print and in the duplicates
iParameterWatermark(iWatermarkPrinting + iWatermarkDuplicate, MyWatermark)

// Starts printing the report
iPrintReport(RPT_Commercial)
```
<a name="Example2"></a>

```wl
// Syntax available from version 170069
// Print a watermark in report background
// Indicates that the watermark will be printed during the print and in the duplicates
iParameterWatermark("Confidential")

// Starts printing the report
iPrintReport(RPT_Commercial)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Configurando el Watermark a través de un Watermark Variable

`iParameterWatermark([<Destination>, ] <Watermark>)`
---

**`<Destination>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Destino del Watermark: 


|   |   |
| --- | --- |
| *iWatermarkDuplicate* | El Watermark se añade a las copias duplicadas. |
| *iWatermarkPrinting* | El Watermark se añade al imprimir o al exportar en formato PDF. |

Si no se especifica este parámetro, la marca de agua se añade en la copia duplicada y al imprimir (*iWatermarkDuplicate* + *iWatermarkPrinting*).

**`<Watermark>: (Watermark Variable)`**

Nombre de la variable [Watermark](../WDLang5/1000019922.md) correspondiente al texto que se imprimirá en Watermark.


<a name="SYNTAX2"></a>

### Configurar la Watermark mediante una cadena de caracteres

`iParameterWatermark([<Destination>, ] <Text>)`
---

**`<Destination>: (Constante o combinación de constantes de tipo Integer)`**

Destino del Watermark: 


|   |   |
| --- | --- |
| *iWatermarkDuplicate* | El Watermark se añade a las copias duplicadas. |
| *iWatermarkPrinting* | El Watermark se añade al imprimir o al exportar en formato PDF. |

Si no se especifica este parámetro, la marca de agua se añade en la copia duplicada y al imprimir (*iWatermarkDuplicate* + *iWatermarkPrinting*). 

**`<Text>: (Cadena de caracteres)`**

Texto para imprimir en un Watermark. En este caso, se utiliza la fuente "Arial", con un tamaño automatic y con una opacidad set a 20%.




## xlsVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la versión de Excel que se utilizó para grabar el fichero.. El número de la última versión se devuelve si se han utilizado varias versiones.

**Observación**: La versión de Excel puede ser conocida:

- si existe el fichero Excel.

- si el archivo de Excel está abierto.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003080018&name=xlsversion_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Open an XLS file
XLSFileID = xlsOpen("C:\MyDirectory\File.XLS")
...
// Excel version
ResVersion = xlsVersion(XLSFileID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### La sintaxis que está usando el tipo xlsDocument

`<Result> = xlsVersion(<XLS document>)`
---

**`<Result>: (Integro)`**



- Excel vesion (3, 4, 5, 8), 100 para los archivos XLSX. El número de la última versión se devuelve si se han utilizado varias versiones.

- 0 si se produce un error. Para obtener más detalles sobre el error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre del [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS a utilizar.  


<a name="SYNTAX2"></a>

### Sintaxis conservada por motivos de compatibilidad con la versión 12

`<Result> = xlsVersion(<XLS file identifier>)`
---

**`<Result>: (Integro)`**



- Excel vesion (3, 4, 5, 8), 100 para los archivos XLSX. El número de la última versión se devuelve si se han utilizado varias versiones.

- 0 si se produce un error. Para obtener más detalles sobre el error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS file identifier>: (Integro)`**

Identificador del archivo XLS a manipular, devuelto por [xlsAbrir](../WDLang5/3080017.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **xlsVersion** genera un error en los siguientes casos:

- el archivo XLS especificado no se abre.

- la versión de Excel en desconocido.

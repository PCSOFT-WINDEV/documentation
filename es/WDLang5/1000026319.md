


## &lt;htmlNode Variable&gt;.FindElementByName (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca elementos cuyo atributo "name" coincida con un valor específico en un nodo HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlNode
e is array of htmlNode <- d.FindElementByName("Color1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Element to use>.FindElementByName(<Name>)`
---

**`<Result>: (Array)`**

Array de los elementos de [htmlNode](../WDLang5/1000026081.md) que coinciden con la condición de búsqueda.

**`<Element to use>: (htmlNode Variable)`**

Nombre de la [htmlNode](../WDLang5/1000026081.md) Variable en la que se realizará la búsqueda.

**`<Name>: (Cadena de caracteres)`**

Valor del atributo "nombre" que se busca.


## mlInitialize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa una nueva sesión independiente de MATLAB con un espacio de trabajo vacío para ejecutar funciones.. Durante esta inicialización, el shell de MATLAB se muestra durante unos segundos.

Observación : Esta función requiere la versión 7.3 o posterior de MATLAB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gnSessionID is int
gnSessionID = mlInitialize()

IF gnSessionID = -1 THEN
	Error(ErrorInfo(errFullDetails))	
ELSE
	STC_MLSession = "MATLAB initialized"
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = mlInitialize()`
---

**`<Result>: (Integro)`**



- Identificador de la sesión de MATLAB. Este identificador será utilizado por las funciones para la gestión de MATLAB. 

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

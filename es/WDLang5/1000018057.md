


## xlsSave (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un XLS Document. El Document salvado permanece abierto. Para cerrarlo, debe utilizar [xlsCerrar](../WDLang5/3080015.md).  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWorksheet is xlsDocument
FileName is string
FileName = fSelect(fExeDir(), "", ...
	"Select a file...", ...
	"All files (*.*)" + TAB + "*.*", "*", fselOpen + fselExist)
IF FileName = "" THEN RETURN
MyWorksheet = xlsOpen(FileName, xlsWrite)

// Write OK in the A25 cell
MyWorksheet[25,"A"] = "OK"
// Save
xlsSave(MyWorksheet)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = xlsGuardar(<XLS document> [, <File path> [, <Password>]])`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<XLS document>: (xlsDocument variable)`**

Nombre de [xlsDocument](../WDLang5/1000017464.md) Variable a utilizar.

**`<File path>: (Optional character string)`**

ruta completa y nombre del archivo XLS.

**Si se especifica este parámetro,** el archivo se guarda en la ubicación especificada (el archivo existente se sobrescribe si es necesario). 

- Si se guarda con éxito, el **..File** Property del [xlsDocument](../WDLang5/1000017464.md) Variable utilizado contiene &lt;Archivo ruta&gt;. El archivo XLS permanece abierto. 

- Si el XLS Document se guarda con un ruta que difiere del inicial (corresponde a "Save as"), el archivo fuente se cierra, el Document se guarda en el nuevo archivo y el nuevo archivo XLS permanece abierto.  


**Si no se especifica este parámetro**, la ruta que se tiene en cuenta es la especificada en la propiedad **..File** de la variable de tipo [xlsDocument](../WDLang5/1000017464.md) manipulada.  

**`<Password>: (Optional character string (with quotes))`**

Contraseña para proteger el archivo XLS al abrirlo. Esta contraseña debe ser introducida para abrir el archivo XLS.

## Observaciones


### Convertir el archivo XLS en XLSX (y viceversa)
<a name="convertir_archivo_xls_xlsx_viceversa_ELTPARAGRAPHE000095"></a>

Para convertir un archivo XLS en XLSX (y a la inversa), solo debe modificar la extensión en el parámetro &lt;Ruta del archivo&gt;. 

Atención: La conversión pierde la fusión de las celdas, los gráficos, los dibujos, ....

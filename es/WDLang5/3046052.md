
## iPageHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura de la página imprimible (en milímetros) teniendo en cuenta los márgenes superior e inferior.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046052&name=ipageheight_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Print the sheet dimensions (in mm)
iPrint("Width of page: " + iPageWidth() + "mm")
iPrint("Height of page: " + iPageHeight() + "mm")
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iPageHeight(<Font identifier>)`
---

**`<Result>: (Real)`**

Alto página:

- en milímetros si no se especifica el parámetro &lt;Font identifier&gt;, 

- en el número de líneas que se pueden imprimir con la fuente &lt;Font identifier&gt;.




**`<Font identifier>: (Cadena de caracteres)`**

Identificador de la fuente utilizada, devuelto por [iFont](../WDLang5/3046014.md) o especificado cuando la fuente fue creada por [iCreateFont](../WDLang5/3046045.md). Si no se especifica este parámetro, se seleccionará la última fuente utilizada.

### Usar en el código reporte
<a name="usar_codigo_reporte_ELTPARAGRAPHE000072"></a>

Si se utiliza la función **iPageHeight**:

- después del código de apertura de reporte, el resultado corresponde a la altura imprimible de reporte (altura de la página - márgenes superior e inferior).

- en el código de apertura reporte, el resultado corresponde a la altura física imprimible (altura de la página - márgenes físicos superior e inferior de la impresora).

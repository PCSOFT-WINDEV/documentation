
## BCToImage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera la Image de un código de barras.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generates the image of the bar code
MyBarCode is BarCode
MyBarCode.Content = "012345678901"
MyBarCode.TypeBarCode = BC_EAN13
ImageBarCode is Image = BCToImage(MyBarCode)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BCToImage(<Bar code> [, <Maximum width> [, <Maximum height> [, <Background color> [, <Bar code color>]]]])`
---

**`<Result>: (Imagen Variable)`**

[Image](../WDLang1/1000019650.md) Variable correspondiente al código de barras.

**`<Bar code>: (Variable de tipo BarCode)`**

Nombre de la [BarCode](../WDLang5/1000019451.md) Variable que corresponde al Image a generar.

**`<Maximum width>: (Entero opcional)`**

Ancho máximo de la Image. 

- Si el ancho máximo no se especifica o corresponde a la constante *bcDefaultWidth*, el ancho será el mínimo requerido para mostrar la BarCode. 

- Si el ancho especificado es menor que el ancho mínimo requerido para mostrar el código de barras, se generará un error.




**`<Maximum height>: (Entero opcional)`**

Altura máxima de la Image. 

- Si no se especifica el alto máximo o corresponde a la constante *cbDefaultHeight*, el alto será el mínimo necesario para mostrar el código de barras. 

- Si la altura especificada es inferior a la altura mínima requerida para mostrar el código de barras, se generará un error.




**`<Background color>: (Entero opcional)`**

Imagen de fondo Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md),

- una variable de tipo [Color](../WDLang1/1000019661.md). 


Si no se especifica el color de fondo o corresponde a la constante *Transparent*, el fondo será transparente.

**`<Bar code color>: (Entero opcional)`**

Código de barras Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md),

- una variable de tipo [Color](../WDLang1/1000019661.md). 


Si no se especifica la Color del código de barras, el código de barras será Black.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tamaño de Image
<a name="tamano_image_ELTPARAGRAPHE000099"></a>

El tamaño de la Image coincidirá con el tamaño del código de barras.. Por ejemplo, para un Código QR, si el ancho especificado es suficiente y la altura requerida excede el ancho, la Image tendrá dimensiones iguales (la altura será igual al ancho requerido).

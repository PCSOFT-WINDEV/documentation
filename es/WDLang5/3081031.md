


## XMLElementType( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tipo de elemento actual (tag o atributo). Para seleccionar un elemento, utilice las funciones de búsqueda y navegación XML ([XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md), [XMLChild](../WDLang5/3081012.md), ...).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
XMLFirst("XMLDoc")
SWITCH XMLElementType("XMLDoc")
	CASE XMLTag: Info(XMLElementName("XMLDoc") + " is XMLTag")
	CASE XMLAttribute: Info(XMLElementName("XMLDoc") + " is XMLAttribute")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLElementType(<XML document>)`
---

**`<Result>: (Constante de tipo Integer)`**

Tipo de elemento de corriente.


|   |   |
| --- | --- |
| *XMLAttribute* | El elemento actual es un atributo. |
| *XMLError* | Ningún elemento se lee actualmente. La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. |
| *XMLTag* | El elemento actual es una etiqueta. |



**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Características de la etapa de corriente
<a name="caracteristicas_etapa_corriente_ELTPARAGRAPHE000099"></a>

- El valor del elemento actual es devuelto por [XMLData](../WDLang5/3081004.md).

- El nombre del elemento actual es devuelto por [XMLElementName](../WDLang5/3081026.md).



<a name="NOTE0_2"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000240"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).

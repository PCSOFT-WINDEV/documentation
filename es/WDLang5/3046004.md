
## iMMToPica( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte las coordenadas de un punto (expresadas en milímetros) en coordenadas del sistema.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Width is real
PicaWidth is real
Width = iImageWidth("Flower.BMP")
PicaWidth = iMMToPica(Width)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iMMToPica(<Position in MM> [, <Direction>])`
---

**`<Result>: (Real)`**

Posición deseada (horizontal o vertical) del punto en coordenadas del sistema.

**`<Position in MM>: (Real)`**

Posición horizontal o vertical del punto en milímetros.

**`<Direction>: (Booleano opcional)`**

Permite configurar la dirección de la conversión:

- <u><u><u><u>True</u></u></u></u> (valor predeterminado) : para una posición horizontal.

- <u><u><u><u>False</u></u></u></u> : conversión para una posición vertical.

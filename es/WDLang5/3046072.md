
## iConfigure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la impresora para una impresión realizada en WLanguage:

- abriendo la ventana de configuración. Si la información es validada por el usuario, los cambios se realizan sólo para la aplicación actual.. 

- modificando la impresora utilizada para la impresión actual. Esta modificación puede ser temporal (sólo para el programa actual) o permanente (la impresora Default se modifica en el panel control de Windows por ejemplo).




Para configurar la impresora cuando se imprimen informes (creados con el editor de reportes), utilice [iConfigureReport](../WDLang5/3046076.md).

### Abrir la ventana estándar para configurar la impresión

`<Result> = iConfigure()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario validó la información introducida en el cuadro de diálogo,

- <u><u><u><u>False</u></u></u></u> si el usuario canceló la operación,.

### Modificar la impresora de destino

`<Result> = iConfigure([<Printer name> [, <Temporary modification>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se ha realizado el cambio de impresora,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Printer name>: (Cadena de caracteres opcional)`**

Identifica la impresora que se va a utilizar. Su nombre se definió en el panel de control de o en la ventana de configuración de la impresión.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si no se especifica este nombre, se selecciona la impresora definida por Default en Windows.





![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El nombre de la impresora es "sensible a mayúsculas y minúsculas". : debe respetar los caracteres en mayúsculas y minúsculas del nombre.

**`<Temporary modification>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si se debe tener en cuenta la modificación de la impresora para el programa actual. Este cambio se mantendrá hasta el final de la ejecución del programa (excepto si se llama a [iReset](../WDLang5/3046067.md)).

- <u><u><u><u>False</u></u></u></u> si la impresora debe modificarse en el panel de control de Windows o en la ventana de configuración de la impresión. La impresora Default está modificada.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La modificación de la impresora es siempre temporal. Este parámetro no se tiene en cuenta.

### Reinicio del módulo de impresión
<a name="reinicio_del_modulo_impresion_ELTPARAGRAPHE000103"></a>

Si **iConfigure** devuelve <u><u><u><u>True</u></u></u></u>, el módulo de impresión se reinicia. Por lo tanto, se debe utilizar la función **iConfigure** antes de cualquier otra función de impresión. En caso contrario, se ignorarán todas las operaciones realizadas anteriormente por las funciones de impresión (los ajustes realizados por [iParameter](../WDLang5/3046008.md), por ejemplo).

Si hay una impresión en curso, **iConfigure** cierra la impresión. Por lo tanto, se debe llamar a **iConfigure** antes del inicio de la impresión.

<a name="iconfigure_iparameter_ELTPARAGRAPHE000136"></a>

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) El cuadro de diálogo que se muestra con la función **iConfigure** se puede llamar desde el cuadro de diálogo abierto con la función [iParameter](../WDLang5/3046008.md).se puede utilizar [iParameter](../WDLang5/3046008.md) en lugar de **iConfigure**. En este caso, sólo se muestra la ventana de configuración.

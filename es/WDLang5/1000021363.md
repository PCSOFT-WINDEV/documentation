


## Funciones MATLAB
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para acceder a MATLAB: 



|   |   |
| --- | --- |
| [mlDeleteVariable](../WDLang5/1000021356.md) | Borra un Variable de la sesión de MATLAB. |
| [mlEnd](../WDLang5/1000021352.md) | Finaliza una sesión de MATLAB. Esta sesión fue inicializada por [mlInitialize](../WDLang5/1000021351.md). |
| [mlExecute](../WDLang5/1000021353.md) | Evalúa y ejecuta un comando en una sesión de MATLAB. |
| [mlGetVariable](../WDLang5/1000021354.md) | Recupera el valor de una Variable para la sesión de MATLAB. |
| [mlInitialize](../WDLang5/1000021351.md) | Inicia una nueva sesión independiente de MATLAB con un espacio de trabajo vacío para ejecutar funciones. |
| [mlListVariable](../WDLang5/1000021357.md) | Lista las variables encontradas en la sesión de MATLAB. |
| [mlLoadWorkspace](../WDLang5/1000021358.md) | Carga todas las variables de un archivo de MATLAB en la sesión. |
| [mlResetWorkspace](../WDLang5/1000021360.md) | Elimina todas las variables de la sesión de MATLAB. |
| [mlSaveWorkspace](../WDLang5/1000021359.md) | Guarda el espacio de trabajo de la sesión en un archivo de MATLAB. |
| [mlSetVariable](../WDLang5/1000021355.md) | Escribe el valor de una Variable para la sesión de MATLAB. |

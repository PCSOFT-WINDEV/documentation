
## sfProcessSubmitRequest (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfProcessSubmitRequest** se utiliza para describir un Record de Salesforce (también llamado objeto de Salesforce) para enviarlo a [SFExecuteProcess](../WDLang5/1000018523.md). Las características de este tipo pueden modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades específicas de las variables sfProcessSubmitRequest
<a name="propiedades_especificas_las_variables_sfprocesssubmitrequest_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades se pueden utilizar para manipular un registro de Salesforce para enviarlo a [SFExecuteProcess](../WDLang5/1000018523.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| comment | Cadena de caracteres | comment que debe añadirse a la historia de la tarea de la Process. |
| SiguienteAproverID | Cadena de caracteres | Identificadores de los usuarios que deberán realizar la siguiente operación. Estos identificadores están separados por caracteres CR (Carriage Return).<br>Este identificador corresponde a la **UserID** Property del [sfUserInfo](../WDLang5/1000018423.md) Variable. |
| ObjectID | Cadena de caracteres | Identificador del objeto de la Fuerza de Ventas/Record para someterlo al Process. |

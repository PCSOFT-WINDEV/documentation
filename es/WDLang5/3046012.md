## Presentación
<a name="presentacion_ELTTEXTE000195"></a>
WINDEV, WEBDEV y WINDEV Mobile proponen varios métodos de impresión:

- el editor de reportes, que se utiliza para crear "Informes preestablecidos".

- las funciones de impresión en WLanguage.





### Editor de reportes
<a name="editor_reportes_ELTPARAGRAPHE000017"></a>

La editor de reportes se utiliza para crear informes impresos (en formato reporte o de etiqueta) que contienen información procedente de ficheros de datos.

Se recomienda crear los informes impresos con editor de reportes en lugar de con las funciones WLanguage.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los informes creados con editor de reportes no están disponibles.


### Las funciones de impresión en WLanguage
<a name="las_funciones_impresion_wlanguage_ELTPARAGRAPHE000028"></a>

Sin embargo, para un trabajo de desarrollo muy específico, es posible que editor de reportes no le permita crear los informes solicitados.. En este caso, debe utilizar las funciones de impresión de WLanguage.

**Atención**: Este capítulo describe el principio para imprimir en WLanguage.

### Impresión en Linux
<a name="impresion_linux_ELTPARAGRAPHE000093"></a>

Sólo las impresiones en formato PDF están disponibles en Linux. 

Se requiere una configuración específica para imprimir en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md).


## iEndReport( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obliga a detener la impresión de reporte. Se imprime el pie de página y el bloque "final de Document".

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la reporte se imprime en el visor de reporte, la ventana de vista previa no se cierra de nuevo.

**Observación** : Esta función sólo se puede utilizar en los informes creados con la editor de reportes.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Imprimir en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// End of print if the text file is not found
GLOBAL
	// Identifier of text file
	gfnFileID is int = 0

Local
	sTextFile is string = fCurrentDir() + "\Reports\Product.txt"

// Open the text file
gfnFileID = fOpen(sTextFile, foRead)
IF gfnFileID = -1 THEN
	Error(ErrorInfo())
	iEndReport() // Stop printing the report
END
```


<a name="Example2"></a>



```wl
// End of print forced when printing a record
IF MySelf..Source.CustomerName = "SMITH" THEN
	iEndReport()
END
```

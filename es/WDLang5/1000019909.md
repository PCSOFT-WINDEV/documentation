


## BCCalculateControlCharacter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el código de cheque de un código de barras.

**Observación**: Esta función no está disponible para los códigos de barras DataMatrix, PDF417 y QR-Code..  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyBarCode is BarCode
MyBarCode.Content = "012345678901"
MyBarCode.TypeBarCode = BC_EAN13
ControlCharact is string
ControlCharact = BCCalculateControlCharacter(MyBarCode)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Carácter de control de un código de barras encontrado en una Variable de códigos de barras

`<Result> = BCCalculateControlCharacter(<Bar code>)`
---

**`<Result>: (Cadena de caracteres)`**

Caracteres control en el código de barras.

Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).  

**`<Bar code>: (BarCode Variable)`**

Nombre de la variable [CódigoDeBarras](../WDLang5/1000019451.md) que contiene el description del código de barras a analizar. 


<a name="SYNTAX2"></a>

### Carácter control de un código de barras específico

`<Result> = BCCalculateControlCharacter(<Bar code type> , <Value of bar code> , <Extended code>)`
---

**`<Result>: (Cadena de caracteres)`**

Caracteres control en el código de barras.

Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 

**`<Bar code type>: (Integro)`**

Tipo de código de barras a analizar (los diferentes tipos de códigos de barras soportados por WINDEV se presentan en [Gestión de códigos de barras](../WDLang5/3046064.md)).


|   |   |   |
| --- | --- | --- |
| *BC_CODABAR* | Codabar bar code. | Código de barras utilizable con &lt;Código extendido&gt; correspondiente a *BC_Check*. |
| *BC_CODE11* | Código 11, código de barras. |   |
| *BC_CODE39* | Código39 código de barras. | Código de barras utilizable con &lt;Código extendido&gt; correspondiente a <em><em><em><em>BC_Extended</em></em></em></em> o <em><em><em><em>BC_Check</em></em></em></em><br>El valor del código de barras puede contener los caracteres de inicio y fin ("\*" carácter). En este caso, estos caracteres se añadirán al dibujo del código de barras y se imprimirán en el texto que se muestra debajo del código de barras..<br>Si estos caracteres no se encuentran en el valor del código de barras, se añadirán automáticamente al dibujo del código de barras..<br>Sin embargo, no se imprimirán en el texto que se muestra debajo del código de barras. |
| *BC_CODE93* | Código 93 de código de barras. | Código de barras utilizable con &lt;Código extendido&gt; correspondiente a *BC_Extended* o *BC_Check*). |
| *BC_CODE128* | Código 128 de código de barras. |   |

| *BC_INTER2OF5* | Intercalado 2 de 5 códigos de barras. | Código de barras utilizable con &lt;Código extendido&gt; correspondiente a *BC_Check*. |

**`<Value of bar code>: (Cadena de caracteres)`**

Valor del código de barras a utilizar. 

**`<Extended code>: (Constante o combinación de constantes de tipo Integer)`**

Características del código de barras.


|   |   |
| --- | --- |
| *BC_Charsets* | Código de barras que utiliza uno o más conjuntos de caracteres para comprimir el código de barras. Esta set de caracteres debe ser definida en el valor del código de barras.<br>Para obtener más información, consulte [Los diferentes tipos de códigos de barras](../WDLang5/3046064.md). |
| *BC_Check* | Código de barras con caracteres de control (válido solo para códigos de barras de tipo *BC_CODE39*, *BC_CODE93*, *BC_INTER2OF5* y *BC_CODABAR*). |
| *BC_CheckDigits* | Código de barras con caracteres de control (válido solo para códigos de barras de tipo *BC_CODE128* y *BC_EAN128*).<br>Si se selecciona esta opción, las constantes *BC_ControlDigits* y *BC_Charsets* se tienen en cuenta automáticamente. |
| *BC_ControlDigits* | Código de barras con caracteres de control (válido solo para códigos de barras de tipo *BC_CODE128* y *BC_EAN128*). En Default, los caracteres de control están encerrados entre "&lt;" y "&gt;". Para no interpretar estos caracteres como caracteres control, basta con duplicar el primer "&lt;":  <br><br>- "&lt;C40&gt;" = carácter control, <br><br>- "&lt;&lt;AB&gt;" = &lt;AB&gt; a codificar en el código de barras.<br><br><br><br><br>Para obtener más información, consulte [Los diferentes tipos de códigos de barras](../WDLang5/3046064.md). |
| *BC_Extendido* | Código de barras extendido (válido solo para códigos de barras de tipo *BC_CODE39* y *BC_CODE93*). |
| *BC_IgnorarFormato* | Ignora los paréntesis de un código de barras 128 o EAN-128 (válido solo para códigos de barras de tipo *BC_CODE128* y *BC_EAN128*).<br>Los paréntesis permiten a los usuarios leer fácilmente el código de barras, pero no están dibujados en el código. |

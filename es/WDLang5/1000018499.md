
## sfMergeResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfMergeResult** se utiliza para conocer el resultado de [SFMerge](../WDLang5/1000018509.md). Las características de este resultado son devueltas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="XSYNTAX"></a>


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Propiedades específicas de las variables sfMergeResult
<a name="propiedades_especificas_las_variables_sfmergeresult_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden utilizarse para manipular el resultado de [SFMerge](../WDLang5/1000018509.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Error | Array de [sfError](../WDLang5/1000018455.md) | Estado reporte de errores. |
| ID | Cadena de caracteres | Identificador del objeto fusionado.<br>**Esta propiedad es de solo lectura**. |
| UpdatedRelatedRecordID | Cadena de caracteres | Lista de identificadores de Salesforce de los registros que se han modificado tras la operación de fusión (registros relacionado a los registros fusionados).. Los diferentes identificadores están separados por caracteres CR (Carriage Return). <br>**Esta propiedad es de solo lectura**. |
| MergedRecordID | Cadena de caracteres | Lista de identificadores de Salesforce de los registros fusionados. Los diferentes identificadores están separados por caracteres CR (Carriage Return). <br>**Esta propiedad es de solo lectura**. |
| Éxito | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la combinación se realizó correctamente,<br><br>- <u><u><u><u>False</u></u></u></u> si se produce un error.<br><br><br>**Esta propiedad es de solo lectura**. |

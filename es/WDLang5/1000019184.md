
## iColumnEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fuerza un salto de columna en una reporte de varias columnas.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// After Printing code of a break footer
iColumnEnd()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`iColumnEnd()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para una reporte con una sola columna, esta función se comporta como [iPageEnd](../WDLang5/3046038.md).

- Si se llama a esta función en la última columna de la reporte, se realiza un salto de página.




## XMLInsertDocument( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta la etiqueta XML actual que se encuentra en un Document XML en otro Document XML.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Un Document XML sólo puede contener una única etiqueta en su raíz.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Result1 is boolean
// Inserts the current tag found in MySourceDoc into MyTargetDoc
// This tag will be a child tag of the current tag found in MyTargetDoc
Result1 = XMLInsertDocument("MyTargetDoc", "MySourceDoc", XMLChildItem)
IF Result1 = True THEN
	Info("Insertion successful")
ELSE
	Info("The insertion failed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLInsertDocument(<Target XML document> , <Source XML document> [, <Insertion position>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la inserción,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el [ErrorOccurred](../WDLang1/3087001.md) Variable es set a <u><u><u><u>True</u></u></u></u>). 
	Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Target XML document>: (Cadena de caracteres)`**

Nombre del XML Document en el que se realizará la inserción.

**`<Source XML document>: (Cadena de caracteres)`**

Nombre de XML Document que contiene la etiqueta que se insertará en un XML Document.

**`<Insertion position>: (Constante opcional de tipo Integer)`**

Posición en la que la adición se realiza en relación con la etiqueta actual en &lt;Target XML Document&gt;. Este parámetro puede tomar uno de los siguientes valores:


|   |   |
| --- | --- |
| *XMLChildItem* | La inserción se realiza en el nivel actual. La parte insertada será una etiqueta hija de la etiqueta actual. |
| *XMLNextElement*<br>(Valor predeterminado) | La inserción se realiza justo después de la etiqueta actual, en la misma rama. |
| *XMLPreviousElement* | La inserción se realiza justo antes de la etiqueta actual, en la misma rama. |

## Observaciones


### Guardar las modificaciones en el archivo XML
<a name="guardar_las_modificaciones_archivo_xml_ELTPARAGRAPHE000111"></a>

Para guardar las modificaciones realizadas en un fichero XML, debe hacerlo:

1. Utilice la función [XMLBuildString](../WDLang5/3081016.md). Esta función recupera y da formato al contenido de un Document XML.

2. Guardar la cadena de caracteres generada por [XMLBuildString](../WDLang5/3081016.md) en un archivo XML. Para ello, utilice la función [fSaveText](../WDLang1/3036046.md).




Ejemplo:


```wl
// Once the XML document was modified 
XMLSource = XMLBuildString("XMLDoc")
// Save the XML file
fSaveText("ModifiedExample.xml", XMLSource)
```

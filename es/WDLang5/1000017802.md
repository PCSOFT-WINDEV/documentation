
## rssTextInput (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssTextInput** permite describir el área de entrada de un flujo RSS. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Este tipo de Variable es el que utiliza el [Tipo rssStream](../WDLang5/1000017795.md).

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssTextInput**. De hecho, las propiedades de tipo [rssStream](../WDLang5/1000017795.md) permiten manipular directamente las propiedades de tipo **rssTextInput** (especialmente la propiedad ..TextInput).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

### Propiedades específicas de la descripción de las variables rssTextInput
<a name="propiedades_especificas_descripcion_las_variables_rsstextinput_ELTPARAGRAPHE000058"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **rssTextInput**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| description | Cadena de caracteres | Leyenda de control Campo de entrada. |
| vincular | Cadena de caracteres | URL a la que se enviarán los datos. |
| denominar | Cadena de caracteres | Nombre utilizado en la Post para transmitir el valor. |
| Título | Cadena de caracteres | Título de validación del área de entrada. |

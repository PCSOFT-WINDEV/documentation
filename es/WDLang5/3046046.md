
## iForceComplement( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fuerza la impresión de un bloque de complemento de cuerpo. El complemento del cuerpo se imprime en la altura de la página restante, es decir, hasta la parte inferior de la página.

Esta función se utiliza para forzar la impresión de un complemento de cuerpo: 

- después del final del descanso.

- en la última página de reporte.




**Observación**: Si no se utiliza esta función, el complemento del cuerpo se imprime después del último bloque de cuerpo de cada página.. El complemento del cuerpo se imprime sólo si se imprime un bloque de cuerpo en la página.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md).

<a name="sample_code"></a>

## Ejemplo

```wl
iForceComplement()
```

## Observaciones
**iForceComplement** podría sustituirse por la impresión de un bloque ([iPrintBlock](../WDLang5/3046030.md)) redimensionado por el [Altura](../Proprietes/2510050.md) Property.




## iPrintDoc( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime una Document en formato "docx".. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document = "c:\temp\proposal.docx"
iDestination(iViewer)
iPrintDoc(MyDocument)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Imprimir el contenido de un control Procesador de texto o un documento Variable

`<Result> = iPrintDoc(<Document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el documento es válido para la impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Document>: (Nombre del documento Variable o control)`**

Document para imprimir. Este documento corresponde a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de un control Procesador de texto.

- una variable de tipo [Document](../WDLang1/1000022461.md).





<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Imprimir el contenido de un control Procesador de texto con redimensionamiento

`<Result> = iPrintDoc(<Word Processing control> [, <Sizing>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el documento es válido para la impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Word Processing control>: (Nombre del control)`**

Nombre de la control Procesador de texto que muestra la Document para imprimir. 

**`<Sizing>: (Constante de tipo Integer)`**

Tamaño de impresión: 


|   |   |
| --- | --- |
| *iHomothetic* | Ampliación o reducción homotética de las páginas de Document para adaptarlas al formato de salida, ignorando los márgenes físicos. |
| *iHomotheticNoEnlargement* | Reducción homotética de las páginas de Document para adaptarlas al formato de salida, ignorando los márgenes físicos. Las páginas con un formato más pequeño mantienen su formato y están centradas. |
| *iHomotheticNoEnlargementPrintableZone* | Reducción homotética de las páginas de Document para adaptarlas al formato de salida, teniendo en cuenta los márgenes físicos. Las páginas con un formato más pequeño mantienen su formato y están centradas. |
| *iRealSize* | Imprimir en tamaño real. Las páginas que utilizan un formato: <br><br>- más pequeños que el formato de salida se centran en la página. <br><br>- mayores que el formato de salida se truncan y se centran en la página. <br><br><br> |




<a name="SYNTAX3"></a>

### Imprimir el contenido de un archivo "docx", un Buffer Variable o un memo binario

`<Result> = iPrintDoc(<Element to print> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el documento es válido para la impresión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Element to print>: (Cadena, Buffer o memo binario)`**



- Nombre y ruta del archivo "docx" a imprimir. 

- Nombre de Buffer Variable para imprimir. 

- Nombre del elemento binario del memorándum a imprimir. 




**`<Password>: (Cadena de caracteres opcional)`**



- Contraseña para abrir el elemento a imprimir (archivo "docx", contenido del Buffer Variable, contenido del memo binario Variable). 

- Vaciar la cadena ("") si el elemento a imprimir no tiene contraseña.

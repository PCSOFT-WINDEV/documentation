


## iPrintWord( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía el string pasado en parámetro a la memoria intermedia de impresión. El cursor de impresión se sitúa al final del string.. La impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar la impresión).

**Observación**: a diferencia de [iPrint](../WDLang5/3046031.md), no se inserta automáticamente una pausa en line al final de la cadena. Si una cadena se imprime después de **iPrintWord**, se imprimirá en el mismo Line.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iPrintWord(iFont(1) + "Text using font 1")
iPrintWord(iFont(2) + "Text using font 2") 
iPrint()
iEndPrinting()
```

```wl
// Example for printing a secure PDF with password
// In this example, the password is "password"
iParameterPDF("opening", "password", iProtectionPrinting + iProtectionSelection)
// Create the PDF file
iDestination(iPDF, "MyPDF.PDF")
// Print code
iCreateFont(1, 16, iBold, iRoman)
iPrintWord("First line of my PDF" + CR + "Second line of my PDF" + CR + ...
		"Last line of my PDF")
// End of print and close the created PDF file
iEndPrinting()
// Display the PDF
ShellExecute(iLastFile())
```

## Sintaxis

`<Result> = iPrintWord([<Font identifier>+] [ <Horizontal position>+] <Text> [, <Truncated string>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la impresión,

- <u><u><u><u>False</u></u></u></u> si el trabajo de impresión fue cancelado.




**`<Font identifier>: (Entero opcional)`**

Identificador de la fuente utilizada para imprimir el texto. Este identificador es devuelto por [iFont](../WDLang5/3046014.md).
Si no se especifica este parámetro, se seleccionará la última fuente utilizada.

**`<Horizontal position>: (Entero opcional)`**

Coordenada X del punto desde el que se debe realizar la impresión. Esta posición se especifica en número de columnas desde la parte superior izquierda Border de la hoja (con [iXPos](../WDLang5/3046015.md)). 
Si no se especifica este parámetro, el texto se imprime en la posición actual.

**`<Text>: (Cadena de caracteres)`**

Cadena de caracteres a imprimir.

**`<String on next page>: (Booleano opcional)`**

Especifica si el texto impreso al final de la página debe ser truncado o si debe continuar en la página siguiente:

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): la cadena se imprimirá en la página siguiente si no cabe en una sola página. 

- <u><u><u><u>False</u></u></u></u>: el string está truncado: el resto de la cadena no se imprime en la página siguiente.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Características de la impresión
<a name="caracteristicas_impresion_ELTPARAGRAPHE000100"></a>

- El origen (0,0) se encuentra en la parte superior izquierda de la hoja Corner. Tiene en cuenta los márgenes físicos de la impresora.

- La función **iPrintWord** no gestiona la posición vertical.

- Una serie de llamadas a la función **iPrintWord** siempre debe terminar con una llamada a la función [iPrint](../WDLang5/3046031.md) que transferirá las cadenas al buffer de impresión.

- El carácter TAB (Char(9)) no se interpreta. El carácter de ruptura de Line (Char(10)) se interpreta como un salto a la siguiente Line.

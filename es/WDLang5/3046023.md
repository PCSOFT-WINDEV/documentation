
## iPrintZoneRTF( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime el formato de texto enriquecido (RTF) en un área rectangular. El trabajo de impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar el trabajo de impresión).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046023&name=iprintzonertf_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
iPrintZoneRTF(RTFText, 0, 0, dHeight, 200)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iPrintZoneRTF(<RTF text> , <X1> , <Y1> , <X2> , <Y2> [, <Start> [, <End>]])`
---

**`<Result>: (Integro)`**

Índice del último elemento RTF impreso en el área rectangular (recuento específico de WINDEV/WEBDEV). 

**`<RTF text>: (Cadena de caracteres)`**

Cadena de RTF que se imprimirá en un área específica.

**`<X1>: (Real)`**

Coordenada horizontal de la Corner superior izquierda del área de impresión (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenada vertical de la Corner superior izquierda del área de impresión (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenada horizontal (columna o abscisa) de la esquina inferior derecha del área de impresión (en milímetros). 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenada vertical de la Corner inferior derecha del área de impresión (en milímetros). 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Start>: (Entero opcional)`**

Índice del primer carácter en el &lt;RTF texto&gt; a imprimir. Si no se especifica este parámetro, el &lt;RTF text&gt; se imprime desde el primer carácter.

**`<End>: (Entero opcional)`**

Índice del último carácter en el &lt;RTF texto&gt; a imprimir. Si no se especifica este parámetro, el &lt;RTF text&gt; se imprime hasta el último carácter.

## Observaciones




### Impresión de un texto RTF justificado
<a name="impresion_texto_rtf_justificado_ELTPARAGRAPHE000069"></a>

Para imprimir un texto con la función **iPrintZoneRTF** sin cambiar la justificación, se debe utilizar previamente la función [iParameter](../WDLang5/3046008.md) de la siguiente manera:


```wl
iParameter("RTFJustification=Yes")
```


Si se utilizan los parámetros utilizados para dividir el texto RTF (&lt;Inicio&gt; y &lt;Fin&gt;), se almacenará el texto de justificación del texto.
<a name="NOTE0_2"></a>




### Imprimir RTF con saltos de página
<a name="imprimir_rtf_con_saltos_pagina_ELTPARAGRAPHE000086"></a>

El siguiente código imprime RTF que contiene saltos de página programados:


```wl
// Open the report viewer
iDestination(iViewer)
// Convert the text
sText is string = RTFToText(EDT_EditText)
// Text size
nTextSize is int = Length(sText)
// Delete the number of page breaks to the size of the text
nTextSize -= StringCount(sText, Charact(12))
// CR must correspond to one single character
nTextSize -= StringCount(sText, Charact(13))

// Loop on all the pages
nStart is int
nSize is int = -1
LOOP
	// Prints the page
	nStart = nSize + 1
	nSize = iPrintZoneRTF(EDT_EditText, 0, 0, iPageWidth(), iPageHeight(), nStart, -1)
	// Skips the page if other pages are found
	IF nSize < nTextSize AND nSize > nStart THEN 
		iSkipPage()
	END
TO DO WHILE nSize < nTextSize AND nSize > nStart 
// End of print
iEndPrinting()
```

<a name="NOTE0_3"></a>




### Impresión RTF
<a name="impresion_rtf_ELTPARAGRAPHE000096"></a>

Para utilizar **iPrintZoneRTF**, el archivo "RICHED20.DLL" debe estar en el ordenador actual.

En la mayoría de los casos, el archivo "RICHED20.DLL" se encuentra en el directorio del sistema de Windows.

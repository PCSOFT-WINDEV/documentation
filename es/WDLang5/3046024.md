
## iRoundBorder( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime una Border con contornos redondeados en coordenadas especificadas.

**Observación**: El origen (0,0) se encuentra en el Corner superior izquierdo de la hoja.. Tiene en cuenta los márgenes físicos de la impresora.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Start of print: Direct printing
iDestination(iPrinter)
// Print an ellipse
iRoundBorder(50, 50, 130, 150, 10, 10, 1, 100, RGB(0,0,0), iDotted, iDownwardDiagonal)
// Ends the print
iEndPrinting()
```

<a name="SYNTAX1"></a>

## Sintaxis

`iRoundBorder(<X1> , <Y1> , <X2> , <Y2> , <Width of sides> , <Height of sides> [, <Thickness> [, <Fill> [, <Color> [, <Line> [, <Brush>]]]]])`
---

**`<X1>: (Real)`**

Coordenada horizontal de la esquina superior izquierda de la Border (en milímetros). Esta esquina define el lado externo de Border. 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenada vertical de la esquina superior izquierda de la Border (en milímetros). Esta esquina define el lado externo de Border. 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenada horizontal de la esquina inferior derecha de la Border (en milímetros). Esta esquina define el lado externo de Border. 
Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenada vertical de la esquina inferior derecha de la Border (en milímetros). Esta esquina define el lado externo de Border. 
Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Width of sides>: (Real)`**

Ancho de los lados de Border en milímetros. Corresponde a la anchura de la Border sobre la que se aplica el redondeo.

**`<Height of sides>: (Real)`**

La altura de los lados de la Border en milímetros. Corresponde a la altura de Border sobre la que se aplica el redondeo.

**`<Thickness>: (Real opcional)`**

Espesor de Border Line (en picas). Valor de Default = 1 (una Pica = 0.353 mm).

**Atención**: Dependiendo del valor de este parámetro, el espesor se consigue añadiendo varias líneas seguidas desde el exterior hacia el interior..

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro debe corresponder a un valor entero. De lo contrario, el espesor se redondeará al valor entero más cercano.

**`<Fill>: (Entero opcional (incluido entre 1 y 200))`**

Tasa para la línea Color y para el fondo Color. Si este parámetro es:

- Igual a 0 (valor de Default): sin relleno.

- Igual a 100: completamente lleno de la especificada Color.

- Menos de 100: cuanto más se acerque este parámetro a 0, más se iluminará la Color.

- Mayor que 100: cuanto más se acerca este parámetro a 200, más se oscurece el Color.




**`<Color>: (Entero opcional)`**

Color usado para el fondo y para el Border. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Line>: (Constante opcional de tipo Integer)`**

Tipo de Line.


|   |   |
| --- | --- |
| *iContinuous*<br>(valor predeterminado) | Línea continua |
| *iDashed* | Dashed Line (-----) |
| *iDotAndDash* | Punto y guión Line (-.-.) |
| *iDotted* | Línea de puntos (....) |



**`<Brush>: (Constante opcional de tipo Integer)`**

Patrón utilizado para llenar el Border.


|   |   |
| --- | --- |
| *iCrossed* | Líneas cruzadas. |
| *iDiagonalCrossed* | Líneas diagonales cruzadas. |
| *iDownwardDiagonal* | Líneas diagonales descendentes. |
| *iHorizontal* | Líneas horizontales. |
| *iSolid*<br>(valor predeterminado) | Relleno completo. |
| *iUpwardDiagonal* | Líneas diagonales ascendentes. |
| *iVertical* | Líneas verticales. |

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta.

## Observaciones




### La ligereza del negro Color
<a name="ligereza_del_negro_color_ELTPARAGRAPHE000178"></a>

Para el Black Color, la tasa corresponde a una tasa de gris cuyos valores se incluyen entre 1 (White) y 100 (Black sólido). En este caso, los valores superiores a 100 se procesarán como el valor 100.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Impresión en formato HTML o RTF
<a name="impresion_formato_html_rtf_ELTPARAGRAPHE000187"></a>

Los bordes redondeados no se imprimen cuando se imprime en archivos en formato HTML o RTF.. Se sustituyen por bordes estándar.
<a name="NOTE0_3"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000195"></a>

- La posición del cursor no se modifica cuando se dibuja el contorno.

- La combinación de líneas punteadas (*iDotted*) o mixtas (*iDotAndDash*) con bordes gruesos puede causar efectos no deseados.




## &lt;fbSession Variable&gt;.GetUserInformation (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la información de Facebook del usuario conectado.. 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MySession is fbSession
...
sEmail is string = MySession.GetUserInformation(fbEmail)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Session>.GetUserInformation(<Information>)`
---

**`<Result>: (Cadena de caracteres o memoria intermedia)`**

Información solicitada.

**`<Session>: (fbSession variable)`**

Nombre de la [fbSession](../WDLang5/1000021851.md) Variable correspondiente a la conexión de Facebook con manipular.

**`<Information>: (Constante de tipo Character String)`**

Información solicitada: 


|   |   |
| --- | --- |
| *fbEmail* | Dirección de correo electrónico del usuario. |
| *fbIdentifier* | ID de Facebook del usuario. |
| *fbImage* | Manual de uso de Image. |
| *fbName* | Nombre del usuario. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Durante la conexión, se deben haber concedido los derechos (permisos) correspondientes a la información solicitada.. De lo contrario, la función fallará.


## iParameterPreview( Función)

	A partir de la versión <B>25</B>, la vista previa de la impresión ha sido renombrada como "espectador de Reporte". Ahora se recomienda utilizar [iParameterViewer](../WDLang5/1000024634.md).

<a name="Use"></a>
<a name="description"></a>
Configura la vista previa de impresión:

- muestra un mensaje sobre los recursos del equipo actual.

- muestra u oculta las pestañas de la cinta de opciones (así como los botones). Para obtener más información, consulte [Vista previa de impresión](../WDChamp/1011085.md).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Apercu_Volet_Apercu.gif)


- los parámetros de exportación en el sistema [ventana de vista previa en miniatura](../WDChamp/1011094.md).




**Observaciones**:

- Cuando se imprime una reporte programada, todos los botones de impresión están disponibles en la vista previa de impresión.

- De forma predeterminada, se muestran todos los botones para imprimir un reporte creado en el editor de reportes. La función **iParameterPreview** solo permite configurar los botones definidos como visibles en el editor de reportes. 

- Pueden aparecer dos botones adicionales en la vista previa de impresión si la aplicación permite la ejecución de "Informes y Consultas" (proporcionados con WINDEV). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=icone_Apercu_EtatRequete.gif)
Estos botones permiten modificar respectivamente el reporte actual y crear un nuevo reporte con "Reports & Queries". Estos botones solo se muestran:

	- Si el proyecto permite que se ejecute "Reports and Queries" (pestaña "Avanzado" del proyecto description).

	- Si el reporte puede ser modificado por "Reports & Queries" (pestaña "Details" del reporte description).

<a name="sample_code"></a>

## Ejemplo


```wl
// No button in the print preview
iParameterPreview(iButtonNone)
iPreview()
iPrintReport(RPT_MyReport)
```


<a name="Example2"></a>



```wl
// Display the printer buttons only
iParameterPreview(iButtonPrinter)
iPreview()
iPrintReport(RPT_MyReport)
```


<a name="Example3"></a>



```wl
// Display the Printer, PDF and WORD buttons in the print preview
iParameterPreview(iButtonPrinter + iButtonPdf + iButtonWord)
iPreview()
iPrintReport(RPT_MyReport)
```


<a name="Example4"></a>



```wl
// Delete the Email buttons from the print preview 
iParameterPreview(iButtonAll - iButtonEmail - iButtonEmailPdf)
iPreview()
iPrintReport(RPT_MyReport)
```


<a name="Example5"></a>



```wl
// Display the warning message every 2000 pages
iParameterPreview(iButtonAll, 2000)
// Print a report from the print preview
iPrintReport(RPT_MyReport)
```


<a name="Example6"></a>



```wl
// Display the thumbnail pane in the print preview
iParameterPreview(iThumbnailPane, True)
// Print a report from the print preview
iPrintReport(RPT_MyReport)
```

**`<Warning>: (Entero opcional)`**

Número de páginas a partir del que se muestra un mensaje de advertencia. Este mensaje advierte al usuario que los recursos de memoria del equipo actual pueden ser insuficientes si se muestran las páginas restantes. 
El valor predeterminado de este parámetro es 1000. 
Si el valor de este parámetro es 0, no se mostrará el mensaje.

### Configurar las opciones de vista previa de impresión

`iParameterPreview(<Options> , <Display>)`
---

**`<Options>: (Constante de tipo Integer)`**

Elemento a configurar: 


|   |   |
| --- | --- |
| *iStorePaneRibbon* | Le permite almacenar el panel de la cinta que se muestra en la vista previa de impresión.. <br><br>De forma predeterminada, la última pestaña de la cinta de opciones no se conserva; la vista previa se abre en la pestaña "Imprimir". |
| *iPaneMiniatures* | Configurar la visualización de los paneles para mostrar las miniaturas. |



**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el panel de miniaturas se debe mostrar en la vista previa de impresión,

- <u><u><u><u>False</u></u></u></u> si no se debe mostrar el panel de miniaturas. 


Por Default, el panel de miniaturas se muestra en la vista previa de impresión.

## Observaciones


### Configuración de la vista previa de impresión
<a name="configuracion_vista_previa_impresion_ELTPARAGRAPHE000278"></a>

- La configuración definida con la función **iParameterPreview** permanece activa hasta:

	- la próxima vez que se llame a la función [iReset](../WDLang5/3046067.md)

	- la siguiente llamada a **iParameterPreview**.




- Para configurar la vista previa de impresión de los reportes de una aplicación, se debe utilizar la función **iParameterPreview** en el código de inicialización del proyecto.

- Los botones de la vista previa también pueden ser configurados en el editor de reportes (pestaña "UI" del reporte description). En este caso, al llamar a la función **iParameterPreview** antes de imprimir el reporte, se pueden ocultar botones adicionales. Los botones que se definen como invisibles en el reporte siempre permanecerán invisibles.




**Observaciones**: 

- Si la vista previa de impresión permite enviar un Email, las características del Email pueden ser configuradas por el [iParameterExport](../WDLang5/1000017026.md).

- La visualización del panel de miniaturas solo se puede configurar mediante programación. 



<a name="NOTE0_2"></a>


### Reportes no creados en el editor de reportes
<a name="reportes_creados_editor_reportes_ELTPARAGRAPHE000310"></a>

Cuando se imprimen informes programados (no creados en la editor de reportes), todos los botones que se encuentran en la barra de botones son Visible.
<a name="NOTE0_3"></a>




### Mensaje de advertencia (sintaxis 1)
<a name="mensaje_advertencia_sintaxis_1_ELTPARAGRAPHE000318"></a>

El mensaje de advertencia solo aparece cuando se imprime un reporte.

Si el usuario hace clic en "No" para evitar que se muestren las páginas siguientes, la vista previa permanece abierta en las páginas ya mostradas. En este caso, toda la Document se imprime (o exporta) desde la vista previa de impresión (y no sólo las páginas mostradas en la vista previa).
<a name="NOTE0_4"></a>




### Configurar los diferentes modos de impresión
<a name="configurar_los_diferentes_modos_impresion_ELTPARAGRAPHE000328"></a>

Es posible configurar las opciones de los diferentes modos de impresión:

- configurar la impresión en un archivo PDF: [iParameterPDF](../WDLang5/3046007.md).

- configurar el envío de correos electrónicos: [iParameterExport](../WDLang5/1000017026.md).

- configurar la impresión en un archivo XLS: [iParameterXLS](../WDLang5/1000017224.md).

- configurar la impresión de duplicados: [iParameterDuplicate](../WDLang5/1000018845.md). 

- configurar la impresión de un texto en marca de agua: [iParameterWatermark](../WDLang5/1000019939.md).

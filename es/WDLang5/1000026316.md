


## &lt;htmlDocument Variable&gt;.SearchElementByID (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca el elemento cuyo atributo "ID" coincide con un valor específico en una Document HTML .
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is htmlDocument = HTMLOpen("https://www.windev.com", fromURL)
e is htmlNode <- d.FindElementByID("MyID")
IF e <> Null THEN
	Info(e.TagName)
ELSE
	Info("Element not found")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Element to use>.ChercheElémentParID(<Identifier>)`
---

**`<Result>: (htmlNode Variable)`**

[htmlNode](../WDLang5/1000026081.md) variable que coincide con la condición de búsqueda.

**`<Element to use>: (htmlDocument Variable)`**

Nombre de la [htmlDocument](../WDLang5/1000025910.md) Variable en la que se realizará la búsqueda.

**`<Identifier>: (Cadena de caracteres)`**

Identificador del elemento a buscar.

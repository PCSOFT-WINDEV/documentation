


## XMLAddAttribute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un atributo en un Document XML. Este atributo se añade al final de la lista de atributos de la posición actual.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La lista de atributos se clasifica en orden alfabético. Por lo tanto, este atributo se añade según el orden alfabético.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a new order line
XMLParent("XMLOrd")
XMLAddChild("XMLOrd", "orderline", "", True)
XMLAddAttribute("XMLOrd", "number", "2")

// Add the "productcode" tag
XMLAddChild("XMLOrd", "productcode", "sro2125")

// Add the "description" tag
XMLAddChild("XMLOrd", "description", "optical mouse")

// Add the "quantity" tag
XMLAddChild("XMLOrd", "quantity", "15")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`XMLAddAttribute(<Name of XML document> , <Attribute name> [, <Attribute value> [, <Change of position>]])`
---

**`<Name of XML document>: (: Cadena de caracteres)`**

Nombre del XML Document en el que debe realizarse la adición. Esta Document contiene el código XML y fue creada por [XMLDocument](../WDLang5/3081008.md).

**`<Attribute name>: (Cadena de caracteres)`**

Nombre del atributo a añadir.


**`<Attribute value>: (Cadena de caracteres opcional)`**

Valor del nuevo atributo. Este parámetro corresponde a una cadena vacía de Default.

**`<Change of position>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para posicionarse en el elemento agregado. En este caso, el elemento añadido se convierte en la posición actual. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para no modificar la posición en curso al agregar el elemento.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de la adición
<a name="gestion_adicion_ELTPARAGRAPHE000087"></a>

- Si la posición actual es un atributo, el nuevo atributo se añade al final de la lista actual de atributos.

- La posición actual en el documento no cambia al llamar a la función **XMLAddAttribute** (excepto si el parámetro &lt;Cambio de posición&gt; es True).



<a name="NOTE0_2"></a>


### Guardar las modificaciones en el archivo XML
<a name="guardar_las_modificaciones_archivo_xml_ELTPARAGRAPHE000097"></a>

Para guardar las modificaciones realizadas en un fichero XML, debe hacerlo:

1. Utilice la función [XMLBuildString](../WDLang5/3081016.md). Esta función recupera y da formato al contenido de un Document XML.

2. Guardar la cadena de caracteres generada por [XMLBuildString](../WDLang5/3081016.md) en un archivo XML. Para ello, utilice la función [fSaveText](../WDLang1/3036046.md).




Ejemplo:


```wl
// Once the XML document was modified 
XMLSource = XMLBuildString("XMLDoc")
// Save the XML file
fSaveText("ModifiedExample.xml", XMLSource)
```

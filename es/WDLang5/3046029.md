
## iDocumentPrinted( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- determinar si el usuario imprimió la Document desde el visor de reporte.

- get información detallada sobre el trabajo de impresión. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iDestination(iViewer)
iPrint("Text")
iEndPrinting()
IF iDocumentPrinted() = True THEN
	Info("You have requested to print the document")
END
```



```wl
iDestination(iViewer)
iPrintReport(MyReport)

arrOutputInfo is array of iOutputInfo
arrOutputInfo = iDocumentPrinted(iListOutput)

FOR EACH OutputInfo OF arrOutputInfo
	Trace("----------------------------------------")
	Trace("Destination = " + OutputInfo.Destination)
	Trace("Printer = " + OutputInfo.Printer)
	Trace("FileName = " + OutputInfo.FileName)
	Trace("PaperSize = " + OutputInfo.PaperSize)
	Trace("PaperWidth = " + OutputInfo.PaperWidth)
	Trace("PaperHeight = " + OutputInfo.PaperHeight)
	Trace("Orientation = " + OutputInfo.Orientation)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Determinar si el usuario ha iniciado un trabajo de impresión o exportación desde el visor reporte.

`<Result> = iDocumentPrinted()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario solicitó el trabajo de impresión desde el visor reporte, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**Atención**: Esta función no determina si el trabajo de impresión ha finalizado correctamente. Si la impresora tiene un problema de hardware, la función siempre devuelve el mismo resultado.  


<a name="SYNTAX2"></a>

### Obtener información detallada sobre el trabajo de impresión o exportación

`<Result> = iDocumentPrinted(<Options>)`
---

**`<Result>: (Array de iOutputInfo)`**

Variables Array de [iOutputInfo](../WDLang5/1410088954.md) que contienen los diferentes trabajos de impresión iniciados desde el visor reporte. 

**`<Options>: (Constante de tipo Integer)`**

Tipo de cuadro de lista:


|   |   |
| --- | --- |
| *iListOutput* | Lista de los diferentes trabajos de impresión o exportación iniciados desde el visor reporte. |

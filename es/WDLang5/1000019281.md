
## XMLDocument

*** ***
	

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **xmlDocument** puede corresponder a:

- la función **XMLDocument** que permite crear un nuevo documento XML. Para obtener más información, consulte [XMLDocument](../WDLang5/3081008.md).

- una variable de tipo **xmlDocument**. Para obtener más información, consulte [xmlDocument](../WDLang5/1000018782.md).


## iWindowCancel( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura la ventana de impresión de Cancel:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IFENETREABANDON.gif)


Es posible:

- mostrar o no la ventana de Cancel,

- editar el título de la ventana Cancel

- modificar el mensaje de la ventana de Cancel.




**Observación**: Esta función debe llamarse **antes de** el comienzo de una impresión (por ejemplo, antes de [iPrint](../WDLang5/3046031.md) o [iPrintReport](../WDLang5/3046032.md), etc.).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Changes the window title and displays a line
iWindowCancel(True, "Please wait", "Printing page#P")
iPrintReport(RPT_ReportName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iWindowCancel(<Display> , <Title> , <Text>)`
---

**`<Display>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para mostrar la ventana de impresión Cancel, 

- <u><u><u><u>False</u></u></u></u> para evitar que aparezca la ventana de impresión Cancel.




**`<Title>: (Cadena de caracteres)`**

Nuevo título de la ventana de impresión Cancel. 
Si no se especifica este parámetro, se mantiene el título de Default de la ventana.

**`<Text>: (Cadena de caracteres)`**

Mensaje a mostrar en la ventana de impresión de Cancel. 
Si no se especifica este parámetro, se muestra el texto Default. 

Para especificar la página actualmente impresa en este mensaje, utilice la cadena "#P. Esta cadena será sustituida por el número de la página actual en tiempo de ejecución.

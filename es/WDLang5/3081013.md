


## XMLLast( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciones en el último elemento (etiqueta o atributo) del nivel de árbol actual.

En la mayoría de los casos, **XMLLast** se utiliza para posicionar en el XML Document con el fin de realizar un bucle de lectura con [XMLPrevious](../WDLang5/3081023.md).
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las cejas se realizan en orden alfabético de los atributos.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions on the last element
XMLLast("XMLDoc")
WHILE XMLOut("XMLDoc") = False
	// Process the data
	XMLPrevious("XMLDoc")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLLast(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se encontró el elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario. En caso de error : 

	- la posición actual permanece inalterada.

	- la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).







**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [xmlDocument](../WDLang5/3081008.md).

<a name="NOTE0_1"></a>

## Observaciones


### Elementos de manipulación en el código XML
<a name="elementos_manipulacion_codigo_xml_ELTPARAGRAPHE000103"></a>

Después de **XMLLast**, el elemento actual puede ser manejado por [XMLElementName](../WDLang5/3081026.md), [XMLParentName](../WDLang5/3081035.md), [XMLElementType](../WDLang5/3081031.md) y [XMLData](../WDLang5/3081004.md).
<a name="NOTE0_2"></a>


### Uso de XMLLast durante una búsqueda
<a name="uso_xmllast_durante_una_busqueda_ELTPARAGRAPHE000124"></a>

Si hay una búsqueda en curso (función [XMLFind](../WDLang5/3081025.md)), la función **XMLLast** cancela la búsqueda.
<a name="NOTE0_3"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000270"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).La modificación de la posición actual del Document XML en un Thread modifica la posición actual del Document XML en todos los Thread s.

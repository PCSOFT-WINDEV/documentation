
## xlsRow (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **xlsRow** se utiliza para describir y modificar un fila en un Excel Document. Las características de esta fila pueden definirse y modificarse utilizando diferentes propiedades del lenguaje WL.

Este tipo de variable es utilizado por el tipo [xlsDocument](../WDLang5/1000017464.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open an Excel file
sExcelFile is string
sExcelFile = fSelect("", "", "Select an Excel file...", "Excel" + TAB + ...
	"*.xlsx" + CR + "All files (*.*)" + TAB + "*.*", "*.xlsx")
IF sExcelFile = "" THEN RETURN

// Open the Excel document
MyDocument is xlsDocument
MyDocument = xlsOpen(sExcelFile)

// Character font
MyFont is Font
MyFont.Name = "Arial"
MyFont.Size = 12
MyFont.Bold = True
MyFont.Color = White

// Cell by default
MyDefaultCell is xlsCell
MyDefaultCell.BackgroundColor = DarkGreen
MyDefaultCell.Font = MyFont

// xlsRow variable
MyRow is xlsRow
MyRow..Height = 200
MyRow..DefaultCell = MyDefaultCell

// All the cell on row 2 have 
// the characteristics of the cell (color, size, font, etc.)
MyDocument..Row[2] = MyRow

// Saves the modifications.
xlsSave(MyDocument)
xlsClose(MyDocument)
```

## Observaciones




### Propiedades específicas de la description de las variables xlsRow
<a name="propiedades_especificas_description_las_variables_xlsrow_ELTPARAGRAPHE000049"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **xlsRow**:

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Celda | Array de [xlsCell](../WDLang5/1000017472.md) | Celdas de la columna. |
| DefaultCell | xlsCell | El estilo Default de la fila corresponderá al estilo de la celda especificada. |
| Altura | Doblemente real | Altura de fila en píxeles (entre 0 y 545 píxeles). Si esta Property se inicializa a -1, se utilizará la altura por defecto de las filas en el Excel Document. |

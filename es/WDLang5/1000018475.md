


## SFSeekDeleted (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca y recupera los registros (objetos de Salesforce) borrados durante el período especificado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
	RETURN
END

ResGetDel is sfSeekDeletedResult
ResGetDel = SFSeekDeleted(Cnt, "Mileage__c", "20090101", "20090131")

MyRecord is sfDeletedRecord
FOR EACH MyRecord OF ResGetDel.DeletedRecord
	Trace("--- ID: " + MyRecord.ID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SFSeekDeleted(<Salesforce connection> , <Salesforce file> , <Start date> , <End date>)`
---

**`<Result>: (sfSeekDeletedResult Variable)`**

[sfSeekDeletedResult](../WDLang5/1000018562.md) variable que contiene los datos del resultado.

**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) que se utilizará

**`<Salesforce file>: (Cadena de caracteres)`**

Nombre del archivo de la Fuerza de Ventas a Process. Este nombre debe corresponder al nombre definido en la interfaz de Salesforce como "API name".. Para un archivo personalizado, este nombre corresponde al nombre del archivo seguido de 2 caracteres de subrayado y luego de la letra "C".. Por ejemplo : "Mi cuenta__C".

**`<Start date>: (FechaHora)`**

Fecha de inicio del período para el que se solicita el número de objetos borrados. Esta fecha no puede ser más de 30 días antes de la fecha actual.. De hecho, los elementos borrados sólo se guardan durante 30 días en la papelera de reciclaje.. A continuación, Salesforce los elimina de forma permanente.. 
Este parámetro debe estar en formato UTC (utilice [DateTimeLocalToUTC](../WDLang1/3027036.md) si es necesario para realizar la conversión). 

**`<End date>: (FechaHora)`**

Fecha final del período para el que se solicita el número de objetos borrados. Este parámetro debe estar en formato UTC (utilice [DateTimeLocalToUTC](../WDLang1/3027036.md) si es necesario para realizar la conversión).

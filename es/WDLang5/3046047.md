
## iNestedHeaderFooter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime (o no) las cabeceras y pies de página de reporte anidada.

Se puede utilizar esta función:

- en el código de apertura de la reporte principal : será tenido en cuenta por todos los informes anidados que se imprimirán desde la reporte principal.

- antes de imprimir la reporte anidada : será tenido en cuenta por los siguientes reporte anidados.




![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención** : Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Impresión en Linux](../Editeurs/9000078.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print the "RPT_Nest2" while report keeping its page header and footer
iNestedHeaderFooter(True)
iPrintReport(RPT_Nest2)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iNestedHeaderFooter(<Header and footer of nested Report>)`
---

**`<Header and footer of nested Report>: (booleano)`**



- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si el reporte principal debe imprimirse compartiendo el encabezado y pie de página con el reporte anidado. El encabezado y pie de página de la reporte anidada no se imprimirá.

- <u><u><u><u>True</u></u></u></u> si se debe imprimir el encabezado y pie de página del reporte anidado.

### Consejo
<a name="consejo_ELTPARAGRAPHE000102"></a>

Cuando se utiliza **iNestedHeaderFooter** con el parámetro <u><u><u><u>True</u></u></u></u>, se recomienda utilizar [iPageEnd](../WDLang5/3046038.md) antes de [iPrintReport](../WDLang5/3046032.md). Esto le permite imprimir dos informes, uno tras otro, en la misma ventana de vista previa.




### Cambio de orientación
<a name="cambio_orientacion_ELTPARAGRAPHE000120"></a>

Esta función no tiene ningún efecto si una reporte anidada y la reporte principal no tienen la misma orientación. : los bloques de encabezado y pie de página **no** ser compartidos.

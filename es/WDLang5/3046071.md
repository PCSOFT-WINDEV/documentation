
## iSkipLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Forzar una o más rupturas de Line. El cursor de impresión está posicionado en las coordenadas (0, actual Line + 1). Por Default, el salto de Line se realiza según el tamaño (altura) del último Line impreso.

La impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar la impresión).

**Observación**: Esta función no debe utilizarse en los informes creados con el editor de reportes.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Print a text
iPrint("Hello")
// Line break
iSkipLine()
// Print a text
iPrint("We are" + DateSys())
// Start the print
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iSkipLine([<Number of Lines to Skip>])`
---

**`<Number of Lines to Skip>: (Entero opcional)`**

Número de líneas a omitir al imprimir. Por defecto, este número es set a 1.

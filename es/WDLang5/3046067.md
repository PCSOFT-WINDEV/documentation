
## función iReset( )



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia los ajustes de impresión (márgenes, fuente, formato de papel, orientación, ...) almacenados en la biblioteca de impresión.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
iReset()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iReset()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
 La función **función iReset** debe utilizarse para modificar los parámetros de impresión actuales cuando no se pueden usar las funciones [iConfigure](../WDLang5/3046072.md) y [iParameter](../WDLang5/3046008.md).

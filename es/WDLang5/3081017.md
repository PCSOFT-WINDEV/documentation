


## XMLExtractString( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae el contenido de una etiqueta que se encuentra en un Document XML. Para realizar búsquedas más complejas, utilice [XMLDocument](../WDLang5/3081008.md) y [XMLFind](../WDLang5/3081025.md).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En el código XML, la lista de atributos se clasifica en orden alfabético.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load the XML code
XMLSource is string = fLoadText("example.xml")
Result1 is string
i is int = 1

// Extract the XML code found in the first "<operation>" tag
Result1 = XMLExtractString(XMLSource, "operation", i)
WHILE Result1 <> ""
	Trace(Result1)
	i = i + 1
	// Extract the XML code found in the next "<operation>" tag
	Result1 = XMLExtractString(XMLSource, "operation", i)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = XMLExtractString(<XML code> , <Tag> [, <Subscript> [, <Search options>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Etiquetar el contenido.

- Cadena vacía (""):

	- si la etiqueta no se encuentra 

	- si la etiqueta no contiene código XML.







**`<XML code>: (Cadena de caracteres)`**

Código XML que contiene la etiqueta.


**`<Tag>: (Cadena de caracteres)`**

Nombre de la etiqueta cuyo contenido se extraerá.

**`<Subscript>: (Entero opcional)`**

Índice de la etiqueta (si la etiqueta aparece varias veces en el código XML). El valor predeterminado de este parámetro es 1.

**`<Search options>: (Constante opcional)`**

Opciones de búsqueda para la extracción:


|   |   |
| --- | --- |
| *XMLExact*<br>(Valor predeterminado) | El nombre de la etiqueta distingue entre mayúsculas y minúsculas para la búsqueda. |
| *XMLIgnoreCase* | El nombre de la etiqueta no distingue entre mayúsculas y minúsculas en la búsqueda. |

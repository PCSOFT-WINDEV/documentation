


## XMLToJSON (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena XML en JSON.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
xml1 is xmlDocument = XMLOpen([
<document>
  <name>TestName</name>
  <id>2</id>
  <true>true</true>
  <list>
    <item>5</item>
    <item>5</item>
    <item>6</item>
    <item>2</item>
    <item>47</item>
  </list>
</document>
], fromString)

Trace(XMLToJSON(xml1))

// Result displayed: { "name":"TestName", "id":2, "true":true, "list":[ 5, 5, 6, 2, 47 ] }
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = XMLToJSON(<XML to convert> [, <Converter>])`
---

**`<Result>: (Variable de tipo JSON)`**

 
Variable JSON que corresponde al elemento XML convertido. 

**`<XML to convert>: (Variable xmlNode o xmlDocument)`**

Nombre de una variable de tipo [xmlDocument](../WDLang5/1000018782.md) o [xmlNode](../WDLang5/1000018786.md). En el caso de una variable de tipo [xmlDocument](../WDLang5/1000018782.md), solo se convertirá el primer nodo.

**`<Converter>: (Variable JSONXMLConverter opcional)`**

 
Nombre de la variable [JSONXMLConverter](../WDLang5/1410087323.md) que corresponde a la norma de conversión personalizada a aplicar. 
Atención: Esta personalización requiere un buen conocimiento de XML. 

Si no se especifica este parámetro, se utilizará una norma predeterminada para la conversión: se utilizan los valores predeterminados de la variable [JSONXMLConverter](../WDLang5/1410087323.md).

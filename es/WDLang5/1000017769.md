
## rssChannel (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **rssChannel** se utiliza para manipular un canal RSS. Las características del canal pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

Este tipo de Variable es utilizado por el tipo [rssStream](../WDLang5/1000017795.md).

**Observaciones**:

- En la mayoría de los casos, no es necesario declarar una variable de tipo **rssStream**. Las propiedades de [rssStream](../WDLang5/1000017795.md) permiten manipular directamente las propiedades del tipo **rssChannel** (especialmente el **Channel** Property).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyStream is rssStream
MyChannel is rssChannel
MyEntry is rssEntry
MyPath is string

// Characteristics of channel
MyChannel.Title = "My site"
MyChannel.Description = "Example of RSS 2.0 stream"
MyChannel.UpdateDate = DateSys() + TimeSys()
MyChannel.Link = "http://www.example.org"
 
// Characteristics of entries of channel 1
MyEntry.Author = "pcsoft@pcsoft.fr"
MyEntry.Description = "First news"
MyEntry.Title = "News #1"

// Adds the entry into the channel
Add(MyChannel.Entry, MyEntry)

// Adds the channel into the stream
Add(MyStream.Channel, MyChannel)

// Saves the stream in a file
MyPath = CompleteDir(fCurrentDir()) + "Rss.xml"
rssSave(MyStream, MyPath)

// Displays the XML file of stream
IF ShellExecute(MyPath) = False THEN
	Error("ShellExecute('" + MyPath + "'). '" + ErrorInfo() + "'")
END
```

## Observaciones




### Propiedades específicas de la description de las variables rssChannel
<a name="propiedades_especificas_description_las_variables_rsschannel_ELTPARAGRAPHE000064"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un canal RSS:

   | --- |
| --- | --- | --- |
| denominar | Tipo utilizado | Efecto |
| Categoría | [rssCategory](../WDLang5/1000017782.md) | Categoría a la que pertenece el canal. Puede utilizarse para agregar los elementos sobre el mismo tema. |
| Nube | [rssCloud](../WDLang5/1000017785.md) | Describe el servicio Web que se utiliza para suscribirse a las notificaciones de modificaciones realizadas en el canal. |
| Copyright | Cadena de caracteres | Copyright para el contenido del canal. |
| UpdateDate | FechaHora | Fecha de la última actualización del contenido del canal. <br>Las fechas leídas se convierten automáticamente en fechas UTC. El valor escrito en el canal está siempre en UTC. |
| Fecha de publicación | FechaHora | Fecha de publicación del contenido del canal (es decir, fecha del último acceso al contenido del canal). <br>Las fechas leídas se convierten automáticamente en fechas UTC. El valor escrito en el canal está siempre en UTC. |
| description (\*) | Cadena de caracteres | description de canal. <br>En la mayoría de los casos, corresponde a una frase que describe la información dada por el canal. |
| Docs | Cadena de caracteres | description del formato utilizado. |
| TimeToLive | Duración o Entero | Duración de la validez del canal. <br>Esta duración es una indicación para el usuario del canal sobre la duración durante la cual el canal puede ser puesto en caché.. El valor en el canal se redondea al minuto.<br>Esta propiedad puede corresponder a: <br><br>- un número entero correspondiente al número de minutos,<br><br>- una variable de tipo Duration,<br><br>- la duración en un formato legible (por ejemplo, 5 minutos). <br><br><br> |
| Entrada (\*) | [Array de rssEntry](../WDLang5/1000017792.md) | Array de las entradas de canales. |
| Generador | Cadena de caracteres | Nombre del programa utilizado para generar el canal. |
| Image | [rssImage](../WDLang5/1000017799.md) | description de la Image que describe el canal. |
| Idioma | Cadena de caracteres | El lenguaje utilizado para escribir el título, la description y los artículos del canal. Corresponde a una cadena en formato xx-yy o xx_yy.<br><br>- xx es una abreviatura de idioma en formato ISO639.<br><br>- yy es un código Country en formato ISO3166.<br>	Ejemplo: en_FR, en_GB, en_US, etc.<br><br><br><br>El valor de esta Property puede ser recuperado y modificado. |
| Enlace (\*) | Cadena de caracteres | URL utilizada para leer el canal. <br>El valor de esta Property puede ser recuperado y modificado. |
| GestorEditor | Cadena de caracteres | Envía un correo electrónico a Address del director editorial del canal. <br>El valor de esta Property puede ser recuperado y modificado. |
| Entrada de texto | [rssTextInput](../WDLang5/1000017802.md) | Se usa para describir un área de entrada al retorno de get sobre la corriente. <br>*Tsu Property se usa raramente y a veces se ignora*. |
| Título (\*) | Cadena de caracteres | Título del canal. <br>El valor de esta Property puede ser recuperado y modificado. |
| Webmaster | Cadena de caracteres | Envía un correo electrónico a Address del director técnico del canal. <br>El valor de esta Property puede ser recuperado y modificado. |


**Observación**: Las propiedades seguidas de (\*) son obligatorias.

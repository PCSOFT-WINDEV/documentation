


## Funciones Salesforce
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las bases de datos de Salesforce:



|   |   |
| --- | --- |
| [SFAdd](../WDLang5/1000018463.md) | Crea una Record en un archivo de una base de datos de Salesforce. |
| [SFConnect](../WDLang5/1000017583.md) | Le permite autenticarse junto a una plataforma de Salesforce. |
| [SFConvertLead](../WDLang5/1000018498.md) | Convierte los cables. |
| [SFDelete](../WDLang5/1000018465.md) | Deletes records from a Salesforce database. |
| [SFDisconnect](../WDLang5/1000017585.md) | Desconecta: <br><br>- la plataforma de Salesforce. <br><br>- la sesión actual de Salesforce.<br><br><br> |
| [SFEmptyRecycleBin](../WDLang5/1000018466.md) | Borra los registros especificados de la papelera de reciclaje de Salesforce. |
| [SFExecuteProcess](../WDLang5/1000018523.md) | Solicita la ejecución de una Process específica . |
| [SFExecuteQuery](../WDLang5/1000018438.md) | Ejecuta una consulta SOQL (Salesforce Object Query Language) en una base de datos de Salesforce. |
| [SFForwardQuery](../WDLang5/1000018483.md) | Continúa ejecutando una consulta SOQL (Salesforce Object Query Language) en los registros de una base de datos de Salesforce. |
| [SFMerge](../WDLang5/1000018509.md) | Fusiona los registros de Salesforce (también llamados objetos de Salesforce). |
| [SFModify](../WDLang5/1000018478.md) | Modifica los registros de un archivo definido en Salesforce. |
| [SFRead](../WDLang5/1000018524.md) | Recupera los registros de Salesforce de sus identificadores. |
| [SFSearch](../WDLang5/1000018555.md) | Realiza una búsqueda entre los registros que se encuentran en una base de datos de Salesforce. |
| [SFSeekDeleted](../WDLang5/1000018475.md) | Busca y recupera los registros (objetos de Salesforce) borrados durante el período especificado. |
| [SFSeekModified](../WDLang5/1000018554.md) | Busca y recupera los registros (objetos de Salesforce) modificados durante el período especificado. |
| [SFUndelete](../WDLang5/1000018527.md) | Restaura los registros de Salesforce previamente eliminados. |




**Advertencia**: Antes de utilizar una función de relacionado para los servicios de Salesforce, le recomendamos encarecidamente que compruebe la licencia para utilizar este servicio.. Se pueden aplicar algunas restricciones. El contenido de las licencias puede cambiar con el tiempo.

PC SOFT no es en ningún caso responsable de la forma en que se utilizan las funciones de acceso nativas.. Asegúrese de que cumple con la licencia del proveedor de servicios.

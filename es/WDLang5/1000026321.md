


## &lt;htmlDocument Variable&gt;.Guardar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el contenido de una [htmlDocument](../WDLang5/1000025910.md) Variable en un archivo HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load the document
d is htmlDocument = HTMLOpen("..\Templates\Template1.html", fromFile)

// Document modifications
// ...

// Save modified document
d.Save("..\Generation\" + DocumentName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Document>.Save([<File name>])`
---

**`<Document>: (htmlDocument Variable)`**

Nombre de la [htmlDocument](../WDLang5/1000025910.md) Variable que corresponde al documento a manipular.

**`<File name>: (Cadena de caracteres opcional)`**

Nombre del archivo que se va a guardar. Si no se especifica el nombre del archivo y la Document se abrió desde un archivo, el archivo original se sobrescribe.

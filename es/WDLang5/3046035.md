
## iNewLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Imprime una Line (en el estándar de Windows, con un extremo redondeado) en las coordenadas especificadas. La impresión no se inicia (sólo se puede utilizar [iEndPrinting](../WDLang5/3046053.md) para iniciar la impresión).

**Observación**: El origen (0,0) se encuentra en el Corner superior izquierdo de la hoja.. Tiene en cuenta los márgenes físicos de la impresora.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draws a solid line in red
iNewLine(50, 10, 50, 200, iContinuous, 12, LightRed)
// Draws a solid line in black
iNewLine(50, 10, 5, 100, iContinuous, 12, Black)
// Start printing
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iNewLine(<X1> , <Y1> , <X2> , <Y2> [, <Style> [, <Thickness> [, <Color>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se imprimió la línea,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<X1>: (Real)`**

Coordenadas horizontales del comienzo de Line (en milímetros). Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y1>: (Real)`**

Coordenadas verticales del comienzo de Line (en milímetros). Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<X2>: (Real)`**

Coordenadas horizontales del final de Line (en milímetros). Valor incluido entre 0 y el ancho de la página (devuelto por [iPageWidth](../WDLang5/3046036.md)).

**`<Y2>: (Real)`**

Coordenadas verticales del final de Line (en milímetros). Valor incluido entre 0 y la altura de la página (devuelto por [iPageHeight](../WDLang5/3046052.md)).

**`<Style>: (Constante opcional de tipo Integer)`**

Tipo de Line.



|   |   |
| --- | --- |
| *iContinuous* <br>(valor predeterminado) | Línea continua |
| *iDashed* | Dashed Line (-----) |
| *iDotAndDash* | Punto y guión Line (-.-.) |
| *iDotted* | Línea de puntos (....) |


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Atención**: Si &lt;Grosor&gt; es mayor que 1, el Line siempre será un Line sólido. Este parámetro puede ignorarse si no es compatible con el controlador de la impresora.

**`<Thickness>: (Real opcional)`**

Line espesor (en picas). Valor de Default = 1 (una Pica = 0.353 mm).
Para get un grosor de 1 píxel, &lt;Grosor&gt; debe ser igual a 0. Cuando el espesor aumenta, se distribuye proporcionalmente entre los dos lados de Line.

**`<Color>: (Entero opcional)`**

Color utilizado. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La posición del cursor no se modifica cuando se dibuja el contorno.

- La combinación de líneas punteadas (*iDotted*) o mixtas (*iDotAndDash*) con bordes gruesos puede causar efectos no deseados.

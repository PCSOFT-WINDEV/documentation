
## iParameterXLS( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define las opciones para el archivo XLS generado durante la impresión. Se puede tener en cuenta o no la disposición actual (Color y el tipo de letra).

Esta configuración se tiene en cuenta cuando se imprime en un archivo XLS (archivo XLS generado desde la vista previa de impresión o generado por [iPreview](../WDLang5/3046069.md) o [iDestination](../WDLang5/3046074.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Export the colors of cells into the printed XLS file
iParameterXLS(iWithFormatting)
// Create the XLS file
iPreview(iXLS, "C:\Temp\MyXLSFile.XLS")
// Print code
iCreateFont(1, 16, iBold, iRoman)
PrintTitle()
PrintText()
// End of print and close the created XLS file 
iEndPrinting()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`iParameterXLS(<Options>)`
---

**`<Options>: (Constante de tipo Integer)`**

Configura el formato del archivo creado.


|   |   |
| --- | --- |
| *iWithFormatting* | Almacena el Color y la fuente durante la exportación en formato XLS. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **iParameterXLS** se ignora en medio de una impresión.

- La configuración definida con la función **iParameterXLS** se activa:

	- hasta que se cierre la aplicación o el sitio,

	- hasta la próxima vez que se utilice la función **iParameterXLS**,

	- hasta la próxima llamada a [iReset](../WDLang5/3046067.md).

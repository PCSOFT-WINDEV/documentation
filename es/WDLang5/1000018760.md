


## &lt;sfConnection Variable&gt;.Desconectar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se desconecta:

- la plataforma de Salesforce.

- la sesión actual de Salesforce.

- las sesiones de Salesforce especificadas.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF Cnt.Connect() =False THEN
	Error("The connection failed")
	RETURN
END
...
// Disconnection (first syntax)
IF NOT Cnt.Disconnect() THEN
	Error("The disconnection failed")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Desconectando la plataforma de Salesforce

`<Result> = <Salesforce connection>.Disconnects()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 


Si ya se desconectó la sesión o si no se estableció ninguna conexión, &lt;Resultado&gt; se establece en <u><u><u><u>True</u></u></u></u>.

**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) que se utilizará.


<a name="SYNTAX2"></a>

### Desconectando una o más sesiones

`<Result> = <Salesforce connection>.Disconnects(<Session identifier> [, <Status report>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hicieron efectivas las desconexiones,

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, si no se pudo realizar una de las desconexiones).




**`<Salesforce connection>: (sfConnection Variable)`**

Nombre de la variable de tipo [sfConnection](../WDLang5/1000017579.md) a manipular.

**`<Session identifier>: (Variable de tipo Array)`**

Array de cadenas que contienen los identificadores de las sesiones a cerrar.

**`<Status report>: (Opcional array Variable)`**

Array de variables de tipo [sfDisconnectResult](../WDLang5/1000018560.md). Una vez ejecutada la función, esta array contiene el estado reporte de cada desconexión..

## Observaciones


### Posibles casos de error: 
<a name="posibles_casos_error_ELTPARAGRAPHE000244"></a>

- La conexión de INTERNET no es válida.

- La autenticación no se realizó correctamente.

- La respuesta XML del servidor no se puede leer.

- Una de las desconexiones no se realizó.




Para averiguar si se produjo un error, utilice el [ErrorOccurred](../WDLang1/3087001.md) Variable.


## iAreaHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la altura de un área de ancho fijo necesaria para imprimir todo el texto que utiliza la fuente actual.

Observación: Esta función no debe utilizarse en la Event "Apertura" de la reporte. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003046062&name=iareaheight_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Selects a font 
iFont(FontNum)
// Retrieves the height of the column
dHeight is real = iAreaHeight("My Text", 100)
// Draws a border
iBorder(0, 0, dHeight*2, 200)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = iAreaHeight(<Text> , <Area width> [, <Type> [, <Beginning> [, <End>]]])`
---

**`<Result>: (Real)`**

Altura del área (en milímetros) que permite imprimir todo el texto.

**`<Text>: (Cadena de caracteres)`**

Texto a imprimir. Este texto puede estar en RTF o HTML. Esta cadena debe estar codificada con UTF-8. Si es necesario, utilice [StringToUTF8](../WDLang1/3024042.md) para realizar la conversión. 

**`<Area width>: (Real)`**

Ancho del área (en milímetros).

**`<Type>: (Constante opcional)`**

Tipo de texto a imprimir: 



|   |   |
| --- | --- |
| *iHTML* | Texto en formato HTML.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *iMarkdown* | Texto Markdown.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible.<br><br> |
| *iRTF* | Texto en formato RTF.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *iText*<br>(valor predeterminado) | Texto de Normal. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro será ignorado. El texto es siempre de tipo Normal. 

**`<Beginning>: (Entero opcional)`**

Índice del primer carácter &lt;Text&gt; a imprimir. Si no se especifica este parámetro, se utiliza el primer carácter de la cadena.

**`<End>: (Entero opcional)`**

Índice del último carácter &lt;Text&gt; a imprimir. Si no se especifica este parámetro, se utiliza el último carácter de la cadena.

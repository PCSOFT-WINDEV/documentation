
## sfExecuteQueryResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfSaveResult** se utiliza para conocer el resultado de [SFForwardQuery](../WDLang5/1000018483.md) y [SFExecuteQuery](../WDLang5/1000018438.md). Las características de este resultado son devueltas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
	RETURN
END

// Retrieve the data
ResQuery is sfExecuteQueryResult
ResQuery = SFExecuteQuery(Cnt, "SELECT Date__c, Miles__c, Contact__c FROM Mileage__c")

ARecord is sfObject
FOR EACH ARecord OF ResQuery.Object
	 Trace("---- Type: " + ARecord.ObjectType + " | ID: " + ARecord.ID)
	 Trace(ARecord.Date__c + " / " + ARecord.Miles__c + " / " + ...
		ARecord.Contact__c)
END
```

## Observaciones




### Propiedades específicas de las variables sfExecuteQueryResult
<a name="propiedades_especificas_las_variables_sfexecutequeryresult_ELTPARAGRAPHE000047"></a>

Las siguientes propiedades pueden utilizarse para manipular el resultado de [SFForwardQuery](../WDLang5/1000018483.md) y [SFExecuteQuery](../WDLang5/1000018438.md):

| denominar | Tipo utilizado | Efecto |
| --- | --- | --- |
| Hecho | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la consulta se ejecutó completamente,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>Esta Property es useful cuando se ejecuta [SFForwardQuery](../WDLang5/1000018483.md).<br>**Esta propiedad es de solo lectura**. |
| QueryLocator | Cadena de caracteres | Ubicación de la consulta. Esta Property es useful cuando se ejecuta [SFForwardQuery](../WDLang5/1000018483.md).<br>**Esta propiedad es de solo lectura**. |
| Objeto | Array de [sfObject](../WDLang5/1000018439.md) | Registros de Salesforce correspondientes al resultado de la consulta. |
| envergadura | Integro | Número de registros correspondientes al resultado.<br>**Esta propiedad es de solo lectura**. |

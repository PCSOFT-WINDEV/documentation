


## xlsColumnTitle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el título de una columna que se encuentra en un archivo XLS. El resultado depende del modo de apertura del archivo XLS:

- En el modo de varias hojas, se busca el título de la columna para la hoja de trabajo seleccionada por [xlsHoja de trabajo actual](../WDLang5/3080022.md).

- En el modo compatible, siempre se busca el título de la columna para la primera hoja de trabajo del libro de trabajo.




**Observación**: El título de una columna en un archivo XLS se devuelve si:

- El archivo XLS existe.

- Se abre el archivo XLS.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open an XLS file
XLSFileID = xlsOpen("C:\MyDirectory\File.XLS")
IF XLSFileID <> -1 THEN
	// Retrieve the caption of a column
	ResColumnTitle = xlsColumnTitle(XLSFileID, 4)
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### La sintaxis que está usando el tipo xlsDocument

`<Result> = xlsColumnTitle(<XLS document> , <Column number> [, <Empty columns>])`
---

**`<Result>: (Cadena de caracteres)`**



- Leyenda de la columna especificada,

- Constante *xlsTitleError* si se produce un error. Para obtener más detalles sobre el error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS document>: (Variable de tipo xlsDocument)`**

Nombre de la [xlsDocument](../WDLang5/1000017464.md) Variable correspondiente al archivo XLS a utilizar.

**`<Column number>: (Integro)`**

Número de la columna a leer.

**`<Empty columns>: (Booleano opcional)`**

Modo para tener en cuenta las columnas vacías:


|   |   |
| --- | --- |
| *False* | Se tienen en cuenta las columnas que están completamente vacías. |
| *True*<br>(predeterminado) | Las columnas que están completamente vacías se ignoran |




<a name="SYNTAX2"></a>

### Sintaxis conservada por motivos de compatibilidad con la versión 12

`<Result> = xlsColumnTitle(<XLS file identifier> , <Column number> [, <Empty columns>])`
---

**`<Result>: (Cadena de caracteres)`**



- Leyenda de la columna especificada,

- Constante *xlsTitleError* si se produce un error. Para obtener más detalles sobre el error, utilice [xlsMsgError](../WDLang5/3080016.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<XLS file identifier>: (Integro)`**

Identificador del archivo XLS a manipular, devuelto por [xlsAbrir](../WDLang5/3080017.md).

**`<Column number>: (Integro)`**

Número de la columna a leer.

**`<Empty columns>: (Booleano opcional)`**

Modo para tener en cuenta las columnas vacías:


|   |   |
| --- | --- |
| *False* | Se tienen en cuenta las columnas que están completamente vacías. |
| *True*<br>(predeterminado) | Las columnas que están completamente vacías se ignoran |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Leyenda de columna
<a name="leyenda_columna_ELTPARAGRAPHE000112"></a>Las columnas vacías se pueden tener en cuenta o no. Por ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=xlsDonnee.gif)

```wl
xlsColumnTitle(XLSFileID, 3, True) // Returns "Age"
xlsColumnTitle(XLSFileID, 3, False) // Returns ""
```

<a name="NOTE0_2"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000121"></a>

La función **xlsColumnTitle** genera un error en los siguientes casos:

- La columna especificada no existe (columna negativa, por ejemplo).

- El archivo XLS especificado no se abre.

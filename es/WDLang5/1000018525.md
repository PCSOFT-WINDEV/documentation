
## sfSearchResult (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfSearchResult** se utiliza para conocer el resultado de [SFSearch](../WDLang5/1000018555.md). Las características de este resultado son devueltas por varias propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection
Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
	RETURN
END

Res is sfSearchResult
Res = SFSearch(Cnt, "FIND {""John Smith""} IN NAME FIELDS")

MyRecord is sfObject

FOR EACH MyRecord of Res.Object
	Trace(MyRecord.ID)
END
```

## Observaciones




### Propiedades específicas de las variables sfSearchResult
<a name="propiedades_especificas_las_variables_sfsearchresult_ELTPARAGRAPHE000045"></a>

Las siguientes propiedades pueden utilizarse para manipular el resultado de [SFSearch](../WDLang5/1000018555.md):

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Objeto | Array de [sfObject](../WDLang5/1000018439.md) | Registros de Salesforce correspondientes al resultado de la búsqueda. |

|   |   |




## XMLChild( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciones en el primer elemento hijo del elemento de corriente. Para seleccionar un elemento, utilice las funciones de búsqueda y navegación XML ([XMLFirst](../WDLang5/3081024.md), [XMLLast](../WDLang5/3081013.md), ...).

**Recordatorio**: Un elemento es una etiqueta o un atributo.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las cejas se realizan en orden alfabético de los atributos.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create the XML document
XMLDocument("XMLOrd", SourceMyXML)
// Browse performed by XMLFirst and XMLNext
// Positions on "<order>" element
XMLFirst("XMLOrd")
// Goes one level down and positions on "<number>" element
XMLChild("XMLOrd")
// Retrieves the order number
OrdNum is int = Val(XMLData("XMLOrd")) // contains 1
```


<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Retrieve the XML content of "orders.xml" file
SourceMyXML is string = fLoadText("orders.xml")
// Create the XML document
XMLDocument("XMLOrd", SourceMyXML)
IF ErrorOccurred = True THEN
	Error("Error while creating the XML document")
	RETURN
ELSE
	// Browse performed by XMLFirst and XMLNext
	// Positions on "<order>" element
	XMLFirst("XMLOrd")
	// Goes one level down and positions on "<number>" element
	XMLChild("XMLOrd")
	// Retrieves the order number
	OrdNum is int = Val(XMLData("XMLOrd")) // contains 1
END
```

## Sintaxis

`<Result> = XMLChild(<XML document>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se encontró el elemento,

- <u><u><u><u>False</u></u></u></u> en caso contrario. En caso de error: 

	- la situación actual permanece inalterada. 

	- la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).







**`<XML document>: (Cadena de caracteres)`**

Nombre del XML Document utilizado. Este Document contiene el código XML a estudiar y fue creado por [XMLDocument](../WDLang5/3081008.md).

## Observaciones


### Elementos de manipulación en el código XML
<a name="elementos_manipulacion_codigo_xml_ELTPARAGRAPHE000110"></a>

Tras la llamada a **XMLChild**, el elemento actual puede ser manejado por [XMLElementName](../WDLang5/3081026.md), [XMLParentName](../WDLang5/3081035.md), [XMLElementType](../WDLang5/3081031.md) y [XMLData](../WDLang5/3081004.md).

**Observación**: puede utilizar [XMLExtractDocument](../WDLang5/3081010.md) para leer sólo el contenido de una etiqueta y ser notificado cuando haya llegado al final con [XMLOut](../WDLang5/3081009.md).
<a name="NOTE0_2"></a>


### Uso de XMLChild durante una búsqueda
<a name="uso_xmlchild_durante_una_busqueda_ELTPARAGRAPHE000139"></a>

Si hay una búsqueda en curso (función [XMLFind](../WDLang5/3081025.md)), la función **XMLChild** cancela la búsqueda.
<a name="NOTE0_3"></a>


### Funciones y subprocesos XML
<a name="funciones_subprocesos_xml_ELTPARAGRAPHE000285"></a>

Si su aplicación utiliza hilos, el XML Document se comparte entre todos estos hilos. Para más detalles sobre los hilos, véase [Gestión de hilos](../WDLang1/3077026.md).Si la posición actual en un XML Document se modifica en un thread, la posición actual en este XML Document se modifica para todos los thread s.

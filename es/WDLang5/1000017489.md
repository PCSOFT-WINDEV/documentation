
## gglCoordinate (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **gglCoordinate** se utiliza para describir y manipular un punto geográfico en un mapa. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

Este tipo de Variable es usado por:

- el tipo [gglMapParameter](../WDLang5/1000017505.md).

- el tipo [gglPath](../WDLang5/1000017433.md).




**Observaciones**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

## Observaciones




### Propiedades de gestión
<a name="propiedades_gestion_ELTPARAGRAPHE000050"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **gglCoordinate**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Latitud | Real | Latitud del punto. |
| Longitude | Real | Longitud del punto. |

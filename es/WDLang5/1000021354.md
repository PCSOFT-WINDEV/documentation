
## mlGetVariable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el valor de una Variable para la sesión de MATLAB.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
gnSessionID is int = mlInitialize()
IF ErrorOccurred() THEN
	ErrorInfo(errFullDetails)
	RETURN
END
...
aResult is array of 1 by 1 real = [[123]]

mlResetWorkspace(gnSessionID)
aResult = mlGetVariable(gnSessionID, "myRoot")
IF ErrorOccurred THEN
	Trace("Error : " + ErrorInfo())
END

Trace("Root = " + aResult[1,1])
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = mlGetVariable(<Session identifier> , <Variable name>)`
---

**`<Result>: (Array de números enteros, reales, ...)`**

El valor de la búsqueda de Variable.

Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

**`<Session identifier>: (Integro)`**

Identificador de la sesión de MATLAB. Este identificador es devuelto por [mlInitialize](../WDLang5/1000021351.md). 

**`<Variable name>: (Cadena de caracteres)`**

Nombre de la Variable a recuperar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Tipos soportados
<a name="tipos_soportados_ELTPARAGRAPHE000059"></a>

Los tipos soportados son:

- enteros con y sin signo en 1, 2, 4 y 8,

- en el 4 y el 8,

- booleans,

- caracteres.


## sfConnection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sfConnection** permite describir una conexión a la plataforma Salesforce y gestionar la autenticación del usuario. Las características de este tipo de variable se pueden definir y cambiar mediante diferentes propiedades WLanguage.

[SFConnect](../WDLang5/1000017583.md) se utiliza para comprobar la validez de la información de la conexión.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connect to Salesforce
Cnt is sfConnection

Cnt.Login = "balthazar@gmail.com"
Cnt.Password = "qwerty" + "XXB12VCZ54"
Cnt.LoginServerURL = "https://login.salesforce.com/services/Soap/u/42.0"

IF SFConnect(Cnt) = False THEN
	Error("The connection failed")
END
```

## Observaciones




### Propiedades específicas de las variables sfConnection
<a name="propiedades_especificas_las_variables_sfconnection_ELTPARAGRAPHE000046"></a>

Las siguientes propiedades pueden utilizarse para establecer una conexión con manipular a Salesforce:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Información de la sesión | [sfSessionInfo](../WDLang5/1000018416.md) | Datos sobre la sesión actual. Esta Property se llena después de la autentificación.<br>**Esta propiedad es de solo lectura**. |
| Nombre de usuario | Cadena de caracteres | Nombre de usuario. |
| SandboxModo | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el modo "Sandbox" está activado (conexión al servidor de pruebas)<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| Contraseña | Cadena de caracteres | Contraseña de conexión. A esta contraseña hay que añadirle la seguridad Token suministrada por Salesforce. |
| LoginServidorURL | Cadena de caracteres | URL del servidor a utilizar. Esta propiedad es opcional. Si se especifica esta Property, se utiliza para la autenticación. |

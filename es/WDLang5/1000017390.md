
## Creación de un lector RSS
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000129"></a>
WINDEV, WEBDEV y WINDEV Mobile le permiten producir y/o consumir un flujo RSS (Rapid Simple Syndication).

Un flujo RSS se utiliza para producir (hacer disponible) un flujo de datos en formato estándar.. Este flujo de datos puede ser leído (consumido) por una aplicación.

Un lector de flujo RSS se utiliza para mostrar uno o más flujos RSS. 



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000153"></a>
**Para leer un flujo RSS**:

1. Declare una variable de tipo [rssStream](../WDLang5/1000017795.md). Este Variable contendrá la información relativa al flujo RSS a manipular.

2. Cargar el flujo RSS con [rssInicializar](../WDLang5/1000017388.md). El flujo RSS Variable se inicializa con el flujo especificado.
	
	```wl
	MyRSSStream is rssStream
	MyRSSStream = rssInitialize("http://blogs.webdev.info/rss.awp?blog=technicalsupport", fromURL)
	```


3. Comprobar si el canal del stream es válido y cargarlo. Por ejemplo:
	
	```wl
	IF MyRSSStream.Channel.Occurrence >= 1 THEN 
		MyChannel is rssChannel
		MyChannel = MyRSSStream.Channel[1]
		...
	END
	```


4. El contenido del flujo RSS puede mostrarse en un tabla relacionado mediante el enlace de datos al [rssChannel](../WDLang5/1000017769.md) Variable que contiene las entradas del flujo (en nuestro ejemplo, MiCanal.Entrada).  




**Observación**: Se recomienda su uso:

- para cada bucle para leer las entradas de un canal associated con un flujo.

- bucles manuales utilizando el **Count** Property (puede utilizarse en el tipo [rssChannel](../WDLang5/1000017769.md) para get el número de canales y en el tipo [rssEntry](../WDLang5/1000017792.md) para get el número de entradas).

### Base de datos y flujo RSS
<a name="base_datos_flujo_rss_ELTPARAGRAPHE000078"></a>

La base de datos está disponible para las variables rssXXX para que la información RSS se pueda mostrar sin necesidad de programación.

## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
WDHFDiff es una herramienta utilizada para comparar archivos de datos HFSQL.

WDHFDiff se utiliza para comparar:

- [estructura de dos archivos de datos HFSQL](../WDHFDiff/3542002.md).

- [datos encontrados en dos archivos de datos HFSQL](../WDHFDiff/3542002.md).

- [archivos de datos comunes encontrados en dos directorios diferentes](../WDHFDiff/3542002.md).




Los archivos de datos utilizados deben estar en formato HFSQL (HFSQL Classic o Hyper File 5.5).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Iniciando WDHFDiff
<a name="iniciando_wdhfdiff_ELTTEXTE000135"></a>
Para iniciar WDHFDiff:

- desde WINDEV, WEBDEV o WINDEV Mobile: en la pestaña "Herramientas", en el grupo "Base de datos", haga clic en "WDHFDiff".

- ejecutar "WDHFDiff.EXE" que se encuentra en el subdirectorio "Programas" del directorio instalación de WINDEV, WEBDEV o WINDEV Mobile.



<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000054"></a>

La WDHFDiff no puede ser redistribuida. WDHFDiff no se puede instalar con aplicaciones desarrolladas con WINDEV, WEBDEV o WINDEV Mobile.

Para utilizar WDHFDiff, WINDEV, WEBDEV o WINDEV Mobile debe estar necesariamente instalado en el ordenador actual.

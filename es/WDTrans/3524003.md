## Presentación

### Si una transacción está en curso
<a name="una_transaccion_esta_curso_ELTPARAGRAPHE000011"></a>

**Si no hay ninguna transacción en curso**En el caso de que se produzca un fallo en el suministro eléctrico, se restablece la consistencia de la base de datos y se cancela la transacción fallida (por ejemplo, un fallo en el suministro eléctrico).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000138"></a>


### Cancel las operaciones realizadas en un archivo de transacciones
<a name="cancel_las_operaciones_realizadas_archivo_transacciones_ELTPARAGRAPHE000020"></a>

A Cancel las operaciones realizadas en un fichero de transacciones:

1. Inicie WDTrans.

2. Seleccione "Cancel a transaction". 

3. Seleccione el fichero de transacción con el selector de ficheros. 
	Se visualiza la lista de ficheros de datos (con su ruta completa) actualmente en transacción.. Para cada fichero, se muestra el identificador de el equipo que realiza una operación en transacción.

4. Si algunos archivos de datos están protegidos con contraseña, especifique la contraseña. De hecho, esta contraseña es necesaria para que Cancel pueda realizar la transacción.

5. Desactivar (si es necesario) la gestión de duplicados y/o integridad.  

6. Haga clic en el botón "Cancel the transaction. La transacción se cancela.



<a name="NOTE2_2"></a>


### A Cancel las operaciones realizadas en un fichero de transacciones en modo comando Line:
<a name="cancel_las_operaciones_realizadas_fichero_transacciones_modo_comando_line_ELTPARAGRAPHE000036"></a>


```txt
WDTrans /Fic = <File>
		/Mdp = <Password>
		/Option = <Type of action performed>
		/Log = <Log file>
```


El tabla que se muestra a continuación presenta los diferentes elementos que se pueden encontrar en el comando Line:

| paramétrico | Significado |
| --- | --- |
| /Fic = &lt;Archivo&gt; | ruta completa del fichero de transacción (fichero.TRS) |
| /Mdp = &lt;Contraseña&gt; | Contraseña asociada al archivo de Process (un solo archivo) |
| /Opción = &lt;Tipo de acción a realizar&gt;&gt; Tipo de acción | Opción de WDTrans para ejecutar. A Cancel las operaciones realizadas sobre un fichero de transacciones, esta opción corresponde a 6. |
| /Registro = &lt;Archivo de registro&gt; | ruta completa del fichero log (.LOG) a crear. |



**Ejemplo**: Cancel las transacciones guardadas en el archivo "WD Transactions.TRS": 

```txt
WDTRANS.EXE /Fic="C:\Temp\WD Transaction.TRS" /option=6
```

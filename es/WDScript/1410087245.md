
## WDScript: uso en modo interactivo
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000141"></a>
En modo interactivo, el editor WDScript se divide en varias áreas: 

- La cinta de opciones, que contiene las opciones más utilizadas,

- El editor de código, que permite escribir el script WLanguage. 

- La consola de salida, que muestra los resultados de la ejecución del código, así como los errores encontrados durante la ejecución.  




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Cinta de opciones de WDScript
<a name="cinta_opciones_wdscript_ELTTEXTE000165"></a>
La cinta de opciones de WDScript tiene dos pestañas: 

- La pestaña "Inicio", que incluye distintas opciones para manipular scripts WLanguage. 

- La pestaña "Código", que permite manipular el código WLanguage. 





### Pestaña "Inicio"
<a name="pestana_inicio_ELTPARAGRAPHE000026"></a>

La pestaña "Inicio" permite: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0001.gif)


- Crear y manipular un archivo de script WLanguage, mediante las opciones "Nuevo", "Abrir", "Guardar" y "Cerrar". 

- Ejecutar el script actual (opción "GO").  

- Cambiar las opciones de WDScript. Puede: 

	- Elegir el idioma de WDScript. 

	- Asociar los archivos "wl" generados por WDScript a WDScript o WDScriptConsole. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0003.gif)


		- Si el archivo wl está asociado a WDScript, se abrirá automáticamente en la interfaz de WDScript al hacer doble clic sobre él. 

- Si el archivo wl está asociado a WDScriptConsole, se ejecutará automáticamente en modo de consola al hacer doble clic sobre él. 

- incluir la opción "Abrir en WDScript" en el explorador de archivos de Windows. Esta opción aparecerá en el menú contextual del archivo en el explorador de archivos de Windows. 





### Pestaña "Código"
<a name="pestana_codigo_ELTPARAGRAPHE000044"></a>

La pestaña "Código" permite manipular el código que aparece en el editor de código de WDScript. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0002.gif)
Es posible: 

- Comentar / No comentar una o varias líneas de código 

- Buscar en el script (buscar una palabra o ir a una línea específica). 

- Abrir la página de ayuda de la función WLanguage seleccionada en el sitio web de documentación de PC SOFT.

## Editor de código de WDScript
<a name="editor_codigo_wdscript_ELTTEXTE000201"></a>
El editor de código de WDScript es una versión simplificada del editor de código de WINDEV, WEBDEV o WINDEV Mobile. Está pensado para scripts que contengan código más corto y sencillo que el que suele escribirse en el editor de código estándar. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0004.gif)


Características del editor de código de WDScript: 

- Resaltado de sintaxis. 

- Métodos abreviados de teclado simples. 

- Completado de código sencillo. 

- Los errores de compilación se muestran como tooltips, directamente en el código. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0005%201.gif)





Existen algunas limitaciones: 

- El script WLanguage debe contener un solo bloque de código. 

- Los procedimientos deben ser internos. 

- Los procedimientos internos deben declararse antes de ejecutarse. 

- El depurador no está disponible. 

- No se pueden utilizar nombres de controles, ventanas, páginas o reportes.

## Consola de salida de WDScript
<a name="consola_salida_wdscript_ELTTEXTE000225"></a>
La consola de salida permite: 

- Ver los resultados devueltos por el script cuando se ejecuta en modo interactivo (por ejemplo, cuando se usa la función **Trace**). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0004%201.gif)
También se muestra el valor de retorno del código ejecutado. 

- Ver los errores de ejecución cuando el script se ejecuta en modo interactivo. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0005.gif)





Este panel se puede desacoplar y redimensionar. Los errores se muestran en rojo. El botón ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WD_Script%20-%20HC%20N%B0001%201.gif) borra el contenido de la ventana de salida.

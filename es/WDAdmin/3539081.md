


## Servidor Web dedicado
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000064"></a>
Cuando se instala la versión de desarrollo de WEBDEV, si no hay ningún servidor Web disponible en el ordenador actual, WEBDEV instala automáticamente uno (IIS Express versión 10).. Este servidor se configura automáticamente para trabajar con WEBDEV. 

También es posible instalar este servidor Web manualmente, y luego configurarlo a través del administrador WEBDEV.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Consejos
<a name="consejos_ELTTEXTE000092"></a>


### automatic instalación
<a name="automatic_instalacion_ELTPARAGRAPHE000110"></a>

Si no se detecta ningún servidor web al instalar WEBDEV 28, el servidor web IIS Express versión 10 se instala y configura automáticamente para WEBDEV en el ordenador actual. 

Este servidor Web permite probar rápidamente los sitios dinámicos (en modo Sesión o AWP) creados con WEBDEV. 


### Instalación manual
<a name="instalacion_manual_ELTPARAGRAPHE000119"></a>

Es posible instalar manualmente el IIS Express Web Server versión 10 en el ordenador de desarrollo (Windows 7 SP1 mínimo).. Esta instalación se puede encontrar en la siguiente Address: [https://www.microsoft.com/fr-fr/download/details.aspx?id=48264](https://www.microsoft.com/fr-fr/download/details.aspx?id=48264)

Simplemente seleccione la descarga deseada en el idioma deseado: 

- Versión de 64 bits (\*_amd64_\*) para un sistema operativo de 64 bits (el caso más común). 

- Versión de 32 bits (\*_x86_\*) para un sistema operativo de 32 bits.  




A continuación, se puede seleccionar el servidor web: 

- en las opciones de WEBDEV: 

	1. En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WEBDEV". 

	2. En la pestaña "Web", en el área "Servidor web", haga clic en "Editar".. 

	3. Seleccione el servidor Web deseado en la ventana que aparece y valide. 




- en el administrador de WEBDEV: 

	1. Iniciar el administrador de WEBDEV: En la pestaña "Herramientas", en el grupo "Web utilities", haga clic en "WDAdmin". 

	2. En la pestaña "Avanzado" haga clic en "Servidor".. 

	3. Seleccione "Servidor web dedicado (IIS Express)" y valide. 








### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000168"></a>

- No es posible elegir el puerto utilizado para el servidor web dedicado: 80xx, donde xx es la versión principal de WEBDEV. Por ejemplo, con WEBDEV 28, el puerto utilizado es 8028.

- IIS Express no tiene una interfaz de configuración. Es necesario editar el archivo XML. Esto significa que, en la práctica, no es posible modificar directamente la configuración de IIS Express.

- Si el sitio requiere una configuración no trivial (SSL Certificate, etc.); es necesario utilizar un servidor Web Normal (IIS por ejemplo).





### Observación
<a name="observacion_ELTPARAGRAPHE000179"></a>

WEBDEV utiliza una configuración separada por versión principal. Se encuentra en el subdirectorio "&lt;Personal&gt;\\IntegratedWebServer".. Si se produce un problema, este directorio puede ser eliminado: será recreado (así como su contenido).

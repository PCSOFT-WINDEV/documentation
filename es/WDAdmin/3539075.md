## Detalles de la pestaña "Webservices"
<a name="detalles_pestana_webservices_ELTTEXTE000161"></a>
La pestaña "Servicios Web" lista los diferentes servicios web de WEBDEV instalados en el equipo y gestionados por el administrador de WEBDEV. Para cada servicio web, la siguiente información se muestra en un tabla:

- nombre del webservice, 

- servicio web completo del directorio instalación en el equipo,

- nombre del archivo de "proyecto" correspondiente.

- estado del servicio web (bloqueado o no). Puede marcar la casilla para bloquear directamente el acceso al servicio web durante la actualización. 

- número de tareas programadas utilizadas por el servicio web.




La pestaña "Webservices" también le permite hacerlo:

- **Eliminar el/los Webservice(s) seleccionado(s) en la tabla** (botón "Eliminar").
	El servicio web se elimina del administrador: el servicio web de WEBDEV todavía se encuentra en el disco.

- **R refresca el display** de los servicios web que se encuentran en el tabla (botón"Refresh").
	Esta operación de actualización se utiliza para comprobar la presencia de un servicio web en el administrador después de su instalación.

- **Añade un servicio web** en el tabla (botón "Añadir").

- **Configurar los servicios web que se encuentran en el equipo individualmente**.
	La configuración utilizada por Default es la definida en la pestaña ["Configuración"](../WDAdmin/3539046.md).
	También tiene la posibilidad de crear una configuración específica para el servicio web seleccionado.: 

	- el botón "Tareas programadas" se utiliza para gestionar las tareas programadas associated con el servicio web seleccionado. Esta configuración está disponible para los servicios web implementados en el servidor. 
			La ventana que aparece le permite: 

		- Vea las tareas programadas associated con el Webservice. se especifican los siguientes elementos para cada tarea: su nombre, el Procedure asociado, la programación, si la tarea está habilitada o no. 

		- Ejecutar una tarea programada inmediatamente. 

		- Modificar una tarea programada (modificar su programación, por ejemplo). 

		- Eliminar una tarea programada. En este caso, la tarea programada se borra hasta el próximo implementación del Webservice o hasta el próximo reinicio de WEBDEV servidor de aplicaciones.




- el botón "Parámetros" sirve para especificar los parámetros del servicio web. La opción "General" indica que el parámetro corresponde al valor seleccionado en la pestaña "Configuración", la opción "Específico" permite personalizar el parámetro.. Se puede personalizar: 

	- la duración de los contextos del AWWS,

	- la duración máxima de la tarea,

	- las opciones de seguridad. 
						Las opciones de seguridad del servidor se definen en la pestaña ["Avanzado"](../WDAdmin/3539066.md) del administrador WEBDEV (sección "Varios"). 

	- la generación de un fichero log específico. 

	- los destinatarios de los informes de error para este servicio web. Esta opción sólo está disponible si la generación de informes de error está habilitada en la pestaña "Log" del administrador.. Para obtener más información, consulte [Administrador de WEBDEV: Visualización de los registros](../WDAdmin/3539073.md). 

	- la URL del servicio web SaaS que se va a utilizar. Este parámetro corresponde a: 

		- el nombre del host del servicio web SaaS (por ejemplo, "Computer", "Computer.MyNetwork", "http://Mydomain.com").

		- la URL completa del servicio web SaaS (por ejemplo, "http://domain.com/WDSAAS28 0_WEB/WDSaaS28 0.awws").




Para más detalles, consulte la ficha de ["Configuración"](../WDAdmin/3539046.md).

Al validar la ventana de configuración del servicio web, los nuevos parámetros se tienen en cuenta inmediatamente para el servicio web seleccionado.

## Presentación
<a name="presentacion_ELTTEXTE000209"></a>
El Centro control de HFSQL le permite administrar completamente los archivos de datos de HFSQL Client/Server.

Para mostrar las características de un archivo de datos de HFSQL Client/Server: 

1. Conectarse (si es necesario) a un servidor HFSQL. Para obtener más información, consulte [Gestión de bases de datos HFSQL Client/Server](../CCHF/35400013.md). 

2. Seleccione el nombre del servidor HFSQL en la estructura de árbol. 

3. Seleccione el nombre de la base de datos deseada.

4. Haga doble clic en el nombre del archivo de datos deseado.

5. La sección derecha del Centro control muestra una pestaña con el nombre del archivo de datos. Esta pestaña incluye: 

	- varias pestañas utilizadas para manejar las características del archivo de datos HFSQL. Default muestra la pestaña "Contenido . 

	- una cinta usada para configurar el archivo de datos HFSQL. 




<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Fichier_HFCS%20-%20HC%20N%B0002.gif&type=thumb)


**Observación**: Desde el panel del Centro control, el menú contextual del archivo de datos se utiliza para:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Fichier_HFCS%20-%20HC%20N%B0001.gif)


- Optimizar y reparar el archivo de datos. 

- Cerrar el archivo de datos. 

- Cambie la contraseña del archivo de datos. 

- Borrar el archivo de datos. 

- Gestionar los derechos sobre la base de datos.

## Características detalladas de un archivo de datos de HFSQL Client/Server
<a name="caracteristicas_detalladas_archivo_datos_hfsql_clientserver_ELTTEXTE000239"></a>


### "pestañas "description" y "Propiedades
<a name="pestanas_description_propiedades_ELTPARAGRAPHE000047"></a>

La pestaña "description" muestra el description de las diferentes partidas del fichero de datos: 

- El nombre del artículo,

- El tipo de artículo, 

- El tamaño del artículo, 

- El tipo de llave, 

- Si el artículo es Visible en "Reports & Queries".




Si es necesario, se visualiza información adicional.



La pestaña "Propiedades" muestra las propiedades del archivo de datos (nombre físico, formato, protección, etc.).
<a name="NOTE2_2"></a>


### "Pestaña "Conexiones
<a name="pestana_conexiones_ELTPARAGRAPHE000066"></a>

Esta pestaña lista los diferentes usuarios conectados a el archivo de datos.

La lista de conexiones de Active contiene la siguiente información:

- el nombre de usuario, 

- El nombre el equipo,

- el nombre de la aplicación que accede a la base de datos, etc.




Usted tiene la capacidad de administrar las conexiones desde esta pestaña: desconectar a los usuarios o enviar un mensaje a los usuarios.
<a name="NOTE2_3"></a>


### "Pestaña "Cerraduras
<a name="pestana_cerraduras_ELTPARAGRAPHE000081"></a>

Esta pestaña se utiliza para gestionar los bloqueos.

Se muestra la siguiente información: 

- el número de la Record cerrada, 

- el nombre del usuario que está bloqueando, 

- el nombre del ordenador de cierre, 

- la IP Address del ordenador de bloqueo, 

- el nombre de la aplicación de cierre, 

- la fecha de inicio del bloqueo,

- la duración del bloqueo.



<a name="NOTE2_4"></a>


### "Pestaña "Contenido
<a name="pestana_contenido_ELTPARAGRAPHE000098"></a>

Esta pestaña se utiliza para ver los diferentes registros que se encuentran en el archivo de datos.

El menú contextual de los Record s permite manipular el Record seleccionado : borrar, añadir, cambiar el modo de visualización de las notas de texto, etc.

También es posible:

- Eliminar un registro: selecciona la Record y haz clic en el botón "Suprimir" (o pulsa la tecla Supr).

- Modificar o añadir un Record: la entrada se realiza directamente en la tabla. El archivo de datos se modifica automáticamente.




El menú contextual de los registros también se utiliza para: 

- Eliminar un registro, 

- Duplicar a Record,

- Especifique las opciones de visualización,

- Imprimir los registros.



<a name="NOTE2_5"></a>


### "pestaña "Buscar
<a name="pestana_buscar_ELTPARAGRAPHE000119"></a>

Esta pestaña se utiliza para realizar búsquedas en el archivo de datos. Todo lo que tiene que hacer es escribir las condiciones buscadas y los valores en el tabla de condiciones. 

El código SQL correspondiente a la búsqueda puede copiarse a través del "Código SQL copy" que se encuentra en la cinta de opciones. 

Observación: Para realizar una búsqueda SQL, haga clic en "Nueva consulta" en la cinta.
<a name="NOTE2_6"></a>


### "Pestaña "Logs
<a name="pestana_logs_ELTPARAGRAPHE000130"></a>

Esta pestaña se utiliza para ver el contenido de los registros y para realizar una búsqueda en los registros.

Para ello, el archivo de datos debe ser registrado.
<a name="NOTE2_8"></a>


### "Pestaña "Transacciones
<a name="pestana_transacciones_ELTPARAGRAPHE000139"></a>

Esta pestaña se utiliza para liberar un Record o todas las transacciones.

## Otras opciones disponibles
<a name="otras_opciones_disponibles_ELTTEXTE000299"></a>


### Detalles de la cinta
<a name="detalles_cinta_ELTPARAGRAPHE000148"></a>

Para el archivo de datos de HFSQL Client/Server seleccionado, la cinta propone las siguientes acciones: 

- En el grupo "Consulta SQL: 

	- Nueva consulta: Esta opción se utiliza para abrir una pestaña específica que le permite realizar una consulta SQL en la base de datos actual..  

	- Abrir una consulta: Esta opción se utiliza para abrir una pestaña específica correspondiente a una consulta que se haya ejecutado previamente.. 




- En el grupo "Archivo":

	- Optimizar y reparar:  Esta opción se utiliza para abrir la ventana de configuración para optimizar el archivo de datos. 

	- Cerrar archivo: esta opción se utiliza para cerrar el archivo de datos y la pestaña correspondiente. 




- En el grupo "Derechos", la opción "Gestionar derechos" permite gestionar los derechos concedidos a los usuarios y grupos en el archivo de datos. 




Las opciones adicionales pueden aparecer en la cinta de acuerdo con la pestaña seleccionada.




## Configuración del servidor HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000230"></a>
El Centro HFSQL control le permite administrar completamente los servidores HFSQL.

Para mostrar las características de un servidor HFSQL: 

1. Conectarse (si es necesario) a un servidor HFSQL. Para obtener más información, consulte [Gestión de bases de datos HFSQL Client/Server](../CCHF/35400013.md). 

2. Haga doble clic en el nombre del servidor HFSQL en la estructura de árbol. 

3. La sección derecha del Centro control muestra una pestaña con el nombre del servidor. Esta pestaña incluye: 

	- varias pestañas utilizadas para administrar las características del servidor HFSQL.

	- una cinta utilizada para configurar el servidor HFSQL. 





En la cinta de opciones, en el grupo "Parámetros", la opción "Configuración del servidor" permite configurar el servidor HFSQL.  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001%202.gif&type=thumb)

## General
<a name="general_ELTTEXTE000260"></a>
En la ventana de configuración del servidor, haga clic en "General".. Se visualizan las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0007.gif&type=thumb)


Puede configurar: 

- **Ubicación de las bases de datos** , es decir, los archivos de datos de las diferentes aplicaciones HFSQL Client/Server. Se pueden utilizar atajos para indicar los directorios específicos: 

	- "%%EXE%%": Directorio de servicios de Manta.

	- "%%COMMONAPPDATA%%": Directorio de datos comunes a todas las aplicaciones. 




- **Cálculo de estadísticas en las claves**: Habilitar esta opción le permite optimizar automáticamente la ejecución de consultas y filtros.. 

- **Directorio local de almacenamiento**: Este directorio contiene todos los datos recuperados (logs y estadísticas) para realizar consolidaciones durante largos períodos..

## Registros
<a name="registros_ELTTEXTE000308"></a>
En la ventana de configuración del servidor, haga clic en "Registros del servidor".. Se visualizan las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0009.gif&type=thumb)


Usted puede: 

- especificar el directorio de archivos de registro. 

- habilitar la copia de seguridad de las llamadas al servidor en el archivo de registro. 

- habilitar la copia de seguridad de los parámetros de llamada en el archivo de registro. 




También puede especificar el tamaño máximo del archivo de registro.

## Estadísticas de actividad
<a name="estadisticas_actividad_ELTTEXTE000332"></a>
En la ventana de configuración del servidor, haga clic en "Estadísticas de actividad".. Se visualizan las siguientes opciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0010.gif&type=thumb)


Las estadísticas de actividad se utilizan para medir la carga de trabajo del servidor. Usted puede: 

- activar las estadísticas de actividad,

- especifique el directorio de la copia de seguridad,

- especifique el tamaño máximo del archivo,

- especificar el período de cálculo del rendimiento (en segundos).

## Cachés
<a name="caches_ELTTEXTE000380"></a>
En la ventana de configuración del servidor, haga clic en "Cachés".. Se visualizan las siguientes opciones:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0012.gif&type=thumb)


- Tamaño de la caché de índice.
	**Observación**: en 32 bits, el tamaño de la caché de índice está limitado a 1 GB.

- Caché de disco (servidor Windows únicamente): La gestión estándar de las cachés de disco de Windows en modo de 64 bits (independientemente de HFSQL) puede crear saturación de memoria.. Esta opción se utiliza para personalizar la gestión de las cachés de memoria.

## variante
<a name="variante_ELTTEXTE000452"></a>
En la ventana de configuración del servidor, haga clic en "Versión".

Esta opción le permite: 

- Descubra la versión actual del servidor: 

	- Versión principal: Versión correspondiente a la versión del motor HFSQL utilizado por el servidor. 

	- Versión secundaria: Número de la actualización utilizada. 

	- Idioma: Idioma utilizado para el servidor. 


El enlace "Más detalles" sirve para get los detalles de los módulos utilizados. 

- Implementar la actualización del servidor de automatic. Si esta opción está activada, el servidor HFSQL puede actualizarse automáticamente tan pronto como esté disponible una nueva versión.: 

	- en el servidor PC SOFT. 

	- en un servidor personalizado. 


Para obtener más información, consulte [Actualizando el servidor](../CCHF/35400018.md).

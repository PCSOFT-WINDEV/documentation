
## Gestión de las aplicaciones HFSQL Client/Server

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000112"></a>
El Centro control de HFSQL se utiliza para administrar los servidores de HFSQL Client/Server y las bases de datos de HFSQL Client/Server.

Desde el Centro control, puede:

- manipular [un servidor HFSQL](../CCHF/35400014.md).

- manipular [las bases de datos HFSQL Client/Server](../CCHF/35400015.md).

- manipular [los ficheros de datos de HFSQL Client/Server](../CCHF/35400016.md).

## Conectarse a un servidor HFSQL
<a name="conectarse_servidor_hfsql_ELTTEXTE000142"></a>
Para conectarse a un servidor HFSQL, puedes:

- especifique una nueva conexión.

- usar una conexión reciente.




**Para especificar una nueva conexión**: 

1. Iniciar el Centro de Control HFSQL

2. En la pantalla de inicio, haga clic en "Conectar a un servidor HFSQL". 

3. Especifique: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cc_HF_Presentation%20-%20HC%20N%B0001%201.gif&type=thumb)


	- el nombre o la IP Address del servidor,

- el puerto,

- el nombre de la base de datos (opcional),

- el usuario y la contraseña. 

4. Validar. El servidor HFSQL aparece en el Centro HFSQL control. El nombre del servidor y las diferentes bases de datos de HFSQL Client/Server se listan en la sección izquierda de la pantalla.




**Para usar una conexión reciente**: 

1. Iniciar el Centro de Control HFSQL

2. En la pantalla de inicio, haga clic en "Conectar a un servidor HFSQL". 

3. Seleccione la conexión deseada (en "Conexiones recientes") y valide. El servidor HFSQL aparece en el Centro HFSQL control. El nombre del servidor y las diferentes bases de datos de HFSQL Client/Server se listan en la sección izquierda de la pantalla.




Observación: Para cambiar la conexión actual en el HFSQL control Center: 

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Passage%20analyse%20en%20CS%20-%20HC%20N%B0009%201.gif). 

2. En el menú, haga clic en "Conectar a un servidor HFSQL". 

3. Se muestra la ventana de inicio. Conéctese al servidor deseado.

## Presentación
<a name="presentacion_ELTTEXTE000184"></a>
El Centro HFSQL control se utiliza para gestionar las diferentes bases de datos HFSQL (Red, Móvil, Cliente/Servidor) así como las bases de datos externas.

## Características de HFSQL control Center
<a name="caracteristicas_hfsql_control_center_ELTTEXTE000244"></a>
El Centro HFSQL control se utiliza para gestionar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=cc_HF_Presentation%20-%20HC%20N%B0001.gif)


- Las bases de datos de HFSQL Client/Server. Puede manipular y get los detalles de:

	- [el servidor](../CCHF/35400014.md): Detener, reiniciar, configuración del servidor, Definition de los usuarios, tareas programadas, copias de seguridad.

	- [la base de datos](../CCHF/35400015.md): description, búsqueda, copia de seguridad, mantenimiento de archivos de datos, transacciones, etc.

	- [cada archivo de base de datos](../CCHF/35400016.md): description, contenido, búsqueda, registros, transacciones, etc.




- [Archivos HFSQL Classic](../CCHF/35400010.md). Tienes la capacidad de get información y manipular cada archivo de base de datos: description, contenido, búsqueda, registros, transacciones, etc.

- [Un análisis](../CCHF/3540003.md). Tienes la capacidad de get información y manipular todos los archivos de datos del análisis. Estos archivos de datos pueden ser:

	- HFSQL Client/Server, 

	- HFSQL Classic, 

	- Bases de datos externas




- Las bases de datos externas. Tienes la capacidad de información de get y de archivos de datos de manipular que no están en formato HFSQL:

	- Oracle, 

	- SQL Server

	- MySQL

	- etc.




- [Un grupo HFSQL](../CCHF/3540005.md). Usted tiene la capacidad de información de get y de archivos de datos del Cluster manipular HFSQL.

## Opciones del Centro de Control HFSQL
<a name="opciones_del_centro_control_hfsql_ELTTEXTE000268"></a>
HFSQL control Center propone varias opciones generales en cuanto a las funciones disponibles. Para visualizar estas opciones: 

1. Despliegue el menú HFSQL control Center (<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_Suivi_Configuration%20-%20HC%20N%B0001.gif)
). 

2. Seleccione "Opciones".. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Presentation_options%20-%20HC%20N%B0001.gif)
Las opciones disponibles son: 

- **Muestre la "x" Icon para ver las contraseñas especificadas**: Se utiliza para mostrar un ojo junto a los controles de "Contraseña".. Todo lo que tienes que hacer es hacer clic en este Icon para ver la contraseña escrita. 

- **Mostrar los memo Imagen solo si la calidad de la conexión al servidor HFSQL es suficiente**: Si esta opción no está marcada, los memos de Image siempre se mostrarán. Si esta opción está seleccionada, los memos de imágenes sólo se mostrarán si la calidad de la conexión es suficiente.. Esta opción se utiliza para optimizar los accesos y la visualización de los datos.

- **Mostrar la base de datos del sistema de los servidores HFSQL (requiere permisos para conectarse a la base de datos)**:  Se utiliza para mostrar la base de datos del sistema de los servidores HFSQL. 

- **Recuerde las pestañas mostradas (solo conexiones HFSQL Client/Server )**: Esta opción guarda las vistas abiertas (pestañas que muestran información sobre el servidor, la base de datos, el archivo de datos, la consulta, etc.) si el Centro control se cierra o se desconecta. Las pestañas se restauran automáticamente la próxima vez que se abra el Centro control. Esta opción está seleccionada de forma predeterminada. 
	La opción "reset" restablece las vistas guardadas. 

- **Orden de visualización de los campos**: Permite especificar el orden de los elementos utilizados en la descripción el archivo de datos (pestaña "Descripción" del archivo de datos). Puede elegir entre el orden lógico (Default) o el orden alfabético. 

- **Activar gestión de la integridad al ejecutar consultas SQL**: Se utiliza para tener en cuenta la gestión de la integridad cuando se ejecutan consultas SQL desde el HFSQL control Center.

- **Activar gestión de duplicados al ejecutar consultas SQL**: Se utiliza para tener en cuenta la gestión de los duplicados cuando se ejecutan consultas SQL desde el Centro HFSQL control.

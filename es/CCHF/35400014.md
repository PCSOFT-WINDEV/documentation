


## Gestión de los servidores HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000310"></a>
El Centro HFSQL control le permite administrar completamente los servidores HFSQL.

Para mostrar las características de un servidor HFSQL: 

1. Conectarse (si es necesario) a un servidor HFSQL. Para obtener más información, consulte [Gestión de bases de datos HFSQL Client/Server](../CCHF/35400013.md). 

2. Haga doble clic en el nombre del servidor HFSQL en la estructura de árbol. 

3. La sección derecha del Centro control muestra una pestaña con el nombre del servidor. Esta pestaña incluye: 

	- varias pestañas utilizadas para administrar las características del servidor HFSQL.

	- una cinta utilizada para configurar el servidor HFSQL. 









<a name="NOTE2"></a>


## Características detalladas de un servidor HFSQL
<a name="caracteristicas_detalladas_servidor_hfsql_ELTTEXTE000340"></a>


### "Pestaña "Cuadro de mandos
<a name="pestana_cuadro_mandos_ELTPARAGRAPHE000034"></a>

Esta pestaña muestra las principales características del servidor.

Se pueden realizar varias acciones en el servidor desde la cuadro de mando. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001.gif&type=thumb)


Esta pestaña muestra las siguientes características en formato de widget: 

- Nombre y estado del servidor. También se muestran la hora, la cantidad de RAM y el sistema.. 

- Versión servidor, con modo de actualización. Para configurar la actualización del servidor, haga clic en el enlace correspondiente. Para obtener más información, consulte [Actualización automática del servidor](../CCHF/35400018.md). 

- Usuario actual. Tiene la posibilidad de editar el perfil del usuario actual. 

- Usos y grupos definidos en el servidor. Un clic sobre la información mostrada le permite acceder a las características de los usuarios o grupos. 

- Transacciones: Visualiza las operaciones en curso.. 

- Configuración: Muestra las principales características para configurar el servidor: Cálculo de estadísticas clave, estadísticas de actividad, caché de índices, registros, etc.. Haga clic en una opción para abrir la ventana de configuración correspondiente. 

- Actividad del servidor: Muestra las tareas actuales. Un clic en el número de tareas actuales se utiliza para get los detalles de la tarea. 

- Tareas programadas: Muestra los diferentes tipos de tareas programadas que se encuentran en el servidor.. Un clic en estas tareas se utiliza para ver la configuración de las mismas.. 

- Notificaciones: Muestra los avisos recibidos.. Tiene la posibilidad de configurar los destinatarios de las notificaciones directamente. 

- Replicación: Muestra las réplicas de los datos del servidor. Puede configurar directamente la replicación del servidor. 

- reserva: Muestra las copias de seguridad programadas. Tiene la capacidad de configurar las copias de seguridad programadas. 

- Conexiones: Muestra el número de conexiones. El enlace "Más detalles" permite acceder a la pestaña "Conexiones".. 

- Volumetría de las bases de datos: Muestra un gráfico utilizado para averiguar el volumen de cada base de datos instalada en el servidor.. La opción "Mostrar la lista de bases de datos" permite visualizar la pestaña "Bases de datos".. 




Para configurar los widgets que se muestran en el panel de control: 

1. Mostrar el menú contextual tabla de la pestaña "Cuadro de mandos" (en una ubicación que no contenga ningún widget, por ejemplo).. 

2. Seleccione "Modo modificación".. 

3. La cuadro de mando aparece en modo "modificación. Es posible: 

	- Ocultar algunos widgets. 

	- Mostrar los widgets ocultos. 

	- Cambiar el tamaño de los widgets con el ratón. 




4. Para salir del modo de modificación y aplicar los cambios, seleccione "Modo de modificación" en el menú contextual..

### "Pestaña "Conexiones
<a name="pestana_conexiones_ELTPARAGRAPHE000075"></a>Esta pestaña lista los diferentes usuarios conectados a las diferentes bases de datos que se encuentran en el servidor.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0002.gif&type=thumb)
 La lista de conexiones de Active contiene la siguiente información:

- el nombre de la base de datos,

- el nombre de usuario, 

- El nombre el equipo,

- el nombre de la aplicación que accede a la base de datos, etc.




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Conexiones", se utilizan para: 

- permitir o prohibir las conexiones, 

- desconectar usuarios, 

- enviar un mensaje a los usuarios.



<a name="NOTE2_3"></a>


### "Pestaña "Bases de datos
<a name="pestana_bases_datos_ELTPARAGRAPHE000095"></a>Esta pestaña se utiliza para listar las diferentes bases de datos que se encuentran en el servidor. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0003.gif&type=thumb)
 La siguiente información se especifica para cada base de datos: 

- El número de conexiones, 

- el número de cerraduras, 

- el número de tablas. 




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Bases de datos", se utilizan para: 

- acceder a la base de datos seleccionada, 

- mostrar los detalles de la base de datos seleccionada o los detalles de todas las bases de datos. Se abren una o más pestañas nuevas, que muestran los detalles de la base de datos.. 

- eliminar la base de datos seleccionada del servidor. 



<a name="NOTE2_4"></a>


### "Ficha "Usuarios
<a name="ficha_usuarios_ELTPARAGRAPHE000114"></a>

Esta pestaña se utiliza para gestionar los usuarios (creación, modificación, eliminación, búsqueda).

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Usuarios", se utilizan para: 

- crear un nuevo usuario, 

- editar las características de un usuario existente, 

- eliminar un usuario. 




Al crear un usuario, éste puede definirse como superusuario: los usuarios definidos como "superusuarios" pueden realizar todas las acciones en el servidor, las bases de datos y los archivos.
<a name="NOTE2_5"></a>


### "Ficha "Grupo
<a name="ficha_grupo_ELTPARAGRAPHE000129"></a>

Esta pestaña se utiliza para gestionar los grupos de usuarios (creación, modificación, eliminación).

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Grupos", se utilizan para: 

- crear un nuevo grupo, 

- editar las características de un grupo existente, 

- borrar un grupo.



<a name="NOTE2_6"></a>

<a name="pestana_transacciones_ELTPARAGRAPHE000142"></a>

Esta pestaña se utiliza para gestionar las transacciones actuales o las transacciones interrumpidas en el servidor.. Es posible: 

- Vea las transacciones actuales, 

- Eliminar los registros de la transacción. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0004.gif&type=thumb)

### "Pestaña "Elementos programados
<a name="pestana_elementos_programados_ELTPARAGRAPHE000154"></a>

Esta pestaña se utiliza para crear, modificar y eliminar operaciones de programación en el servidor.: 

- respaldos.

- tareas (procedimientos almacenados).

- optimización.

- actualización de vistas materializadas. 




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Elementos programados", se utilizan para: 

- crear una nueva programación, 

- visualizar los detalles de una programación,

- ejecutar una programación,

- Borrar una programación. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0005.gif&type=thumb)
Para obtener más información, consulte [Administrar las tareas programadas](../CCHF/3540009.md).
<a name="NOTE2_8"></a>


### "Pestaña "Copias de seguridad
<a name="pestana_copias_seguridad_ELTPARAGRAPHE000179"></a>

Esta pestaña se utiliza para realizar y restaurar copias de seguridad del servidor y/o copias de seguridad de las bases de datos.. Esta pestaña enumera las copias de seguridad del servidor: 

- actualmente en ejecución,

- terminado. 




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Copias de seguridad", se utilizan para: 

- crear una nueva copia de seguridad (copia de seguridad en caliente o programada),

- reiniciar una copia de seguridad,

- restaurar una copia de seguridad,

- eliminar una copia de seguridad. 




Para obtener más información, consulte [copias](../CCHF/3540006.md).
<a name="NOTE2_9"></a>


### "Pestaña "Notificaciones
<a name="pestana_notificaciones_ELTPARAGRAPHE000201"></a>

En esta pestaña se enumeran las diferentes notificaciones del servidor:

- Tareas programadas fallidas,

- Fallos en las copias de seguridad,

- Servidor de replicación no accesible,

- etc.




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Notificaciones", se utilizan para: 

- Borrar los avisos seleccionados.

- Eliminar todas las notificaciones del servidor.




Para más detalles sobre las notificaciones, véase [Notificaciones de un servidor HFSQL](../WDLang4/1000017311.md).

## Opciones de cinta
<a name="opciones_cinta_ELTTEXTE000412"></a>
Cuando se muestran las características del servidor, la cinta muestra varias opciones: 

- opciones generales. 

- opciones específicas para las diferentes pestañas seleccionadas. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0001%201.gif&type=thumb)
Las opciones generales son las siguientes: 

- En el grupo "Consulta SQL: 

	- Nueva consulta: Esta opción se utiliza para abrir una pestaña específica que le permite realizar una consulta.  

	- Abrir una consulta: Esta opción se utiliza para abrir una pestaña específica correspondiente a una consulta que se haya ejecutado previamente.. 




- En el grupo "Servidor: 

	- Detener servidor: Esta opción se utiliza para detener el servidor. 

	- Iniciar servidor: Esta opción se utiliza para iniciar el servidor (si éste se detuvo de antemano).. 




- En el grupo "Bases de datos: 

	- Nueva base de datos: Esta opción se utiliza para crear una nueva base de datos en el servidor. 

	- Importar una base de datos HFSQL Classic: Esta opción se utiliza para importar una base de datos HFSQL Classic en el servidor en modo Cliente/Servidor. Una asistente permite seleccionar los archivos de datos a importar. 

	- Descargar archivos del servidor: Esta opción se utiliza para los archivos de copy del servidor a el equipo. Una asistente le permite seleccionar las diferentes opciones de copy así como el directorio de copy.




- En el grupo "Derechos: 

	- Administrar permisos: Esta opción se utiliza para gestionar los derechos en el servidor para el usuario o grupo seleccionado.. Tiene la posibilidad de almacenar y restaurar una configuración de derechos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_Gestion_Serveur%20-%20HC%20N%B0006.gif&type=thumb)





- En el grupo "Parámetros", la opción "Configuración del servidor" permite configurar el servidor. Para obtener más información, consulte [Configuración del servidor HFSQL](../CCHF/3540004.md). 

- En el grupo "Registros y estadísticas: 

	- Mostrar registros del servidor: Esta opción se utiliza para get una visión sintética sobre el uso de la base de datos entre dos fechas:

		- consultas más utilizadas, consultas más largas, 

		- aplicaciones con mayor consumo

		- estadísticas por aplicación, Process, etc.


 Esta opción sólo está disponible si los registros están habilitados en el servidor.

- Mostrar estadísticas del servidor: Esta opción sólo está disponible si las estadísticas de actividad están habilitadas en el servidor.

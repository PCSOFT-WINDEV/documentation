
## Centro de Control HFSQL: Creación y ejecución de una consulta
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000097"></a>
El HFSQL control Center le permite crear una consulta basada en los archivos de datos de la base de datos que se muestra actualmente en el control Center. 

Para crear una consulta: 

1. En la cinta de opciones, vaya al grupo "Consulta SQL", expanda "Nueva consulta" y seleccione "Consulta en blanco". 

2. Aparece una nueva pestaña correspondiente a la nueva consulta. 

3. Si es necesario, seleccione la base de datos en la que debe realizarse la consulta y escriba el código SQL de la misma. 

4. Para ejecutar la consulta, seleccione "Ejecutar" en la cinta de opciones. 




Observación: También puede crear una consulta a través de una asistente.

<a name="NOTE2_1"></a>


## interrogar: Las opciones de la cinta
<a name="interrogar_las_opciones_cinta_ELTTEXTE000121"></a>
La cinta de consulta incluye las siguientes opciones: 

El grupo "Consulta SQL" incluye las siguientes opciones: 

- Nueva consulta: Crea una nueva consulta, ya sea escribiendo el código SQL o a través de asistente. 

- Abrir una consulta: Abre una consulta del historial, o una consulta previamente guardada. 

- Sava: Guarda la consulta actual. 

- Cerrar consulta: Cierra la consulta actual. 




El grupo "Ejecución" incluye las siguientes opciones: 

- Corre: Ejecuta la consulta mostrada en la pestaña actual. 

- Sugerir claves: Sugiere las claves para optimizar el rendimiento de la consulta. 

- Mostrar EXPLAIN: Muestra la sentencia EXPLAIN de la consulta como Image o en formato XML. 




El grupo "Transacciones" incluye las siguientes opciones: 

- Iniciar una transacción: Inicia una transacción antes de ejecutar la consulta. Una vez iniciada la transacción, puede validarla en cualquier momento seleccionando "Validar transacción". 

- Cancelar transacción: Cancela la transacción actual si es necesario. 




El grupo "Otras acciones" incluye las siguientes opciones: 

- Historial de consultas: obtiene una lista de las consultas realizadas desde el HFSQL control Center, tanto si se han ejecutado con éxito como si no. 

- Lista de palabras clave SQL: obtiene la lista de palabras clave que se pueden utilizar en las consultas SQL.

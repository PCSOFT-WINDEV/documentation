


## Centro de Control HFSQL: Abrir un análisis
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000181"></a>
El Centro HFSQL control le permite gestionar completamente un análisis. Da acceso a todas las bases de datos que se encuentran en el análisis. : HFSQL Classic, HFSQL Client/Server y bases de datos externas.

Se pueden visualizar las características de un análisis: 

- iniciando el HFSQL control Center desde un proyecto: 

	1. Abrir el proyecto WINDEV, WINDEV Mobile o WEBDEV. 

	2. En la pestaña "Herramientas", en el grupo "Base de datos", haga clic en "HFSQL". 

	3. Se muestra la ventana de inicio de HFSQL control Center. Se selecciona la opción "Abrir un análisis" y el ruta del análisis actual aparece en el campo de entrada. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Passage%20analyse%20en%20CS%20-%20HC%20N%B0008.gif&type=thumb)


	4. Validar. El análisis se abre en el HFSQL control Center. 

- desde el Centro de Control HFSQL: 

	1. Iniciar el Centro de Control HFSQL. 

	2. En la ventana de inicio, haga clic en "Abrir un análisis". 

	3. Especifique el análisis ruta (archivo WDD) y la contraseña del análisis si es necesario.. 

	4. Validar. El análisis se abre en el HFSQL control Center.

## Características del análisis
<a name="caracteristicas_del_analisis_ELTTEXTE000211"></a>


### Pestaña "HFSQL Classic"
<a name="pestana_hfsql_classic_ELTPARAGRAPHE000056"></a>Esta pestaña muestra información sobre los archivos de datos de HFSQL Classic en el análisis.
Se muestra la siguiente información:

- Nombre lógico del archivo de datos,

- ruta completa (si existe),

- Tamaño del archivo de datos,

- Número de registros activos, 

- Número de registros eliminados, 

- Número total de registros.





### "pestaña"HFSQL C/S
<a name="pestanahfsql_ELTPARAGRAPHE000069"></a>Esta pestaña muestra información sobre los archivos de datos de HFSQL Client/Server en el análisis.
Se muestra la siguiente información:

- Nombre lógico del archivo de datos,

- Servidor, 

- Base de datos de associated,

- Tamaño del archivo de datos,

- Número de registros activos, 

- Número de registros eliminados, 

- Número total de registros.





### "Pestaña"Base de datos externa
<a name="pestanabase_datos_externa_ELTPARAGRAPHE000083"></a>Esta pestaña muestra toda la información sobre los archivos de datos no HFSQL presentes en el análisis.
Se muestra la siguiente información:

- Nombre lógico del archivo de datos,

- Tipo,

- Fuente de datos,

- Número total de registros.





### "Pestaña izquierda
<a name="pestana_izquierda_ELTPARAGRAPHE000094"></a>

Esta pestaña muestra toda la información sobre los vínculos entre los archivos de datos del análisis.

Se muestra la siguiente información:

- Nombre del enlace,

- Nombra el archivo de datos de la fuente,

- Cardinalidad fuente, 

- Nombre del archivo de datos de destino,

- Cardenales del destino,

- Regla de integridad para borrado,

- Regla de integridad para la modificación. 




Observación: Los enlaces se crean y se editan en editor de análisis.

## Opciones de cinta
<a name="opciones_cinta_ELTTEXTE000253"></a>
Cuando se abre un análisis en HFSQL control Center, la cinta muestra las siguientes opciones: 

- En el grupo "Gestionar archivos": 

	- **Sincronizar archivos**: Esta opción inicia la modificación de los archivos de datos de automatic para sincronizar los archivos de datos y su description en el análisis. Esta operación se realiza a través de una asistente. 

	- **Agregar un directorio de implementación**: Esta opción se utiliza para seleccionar un directorio donde se buscarán los ficheros de datos correspondientes al análisis.. 

	- **Gestionar archivos .REP**: Esta opción enumera las ubicaciones de los archivos de datos guardados en el archivo .REP relacionado al análisis. Puede añadir y eliminar archivos del archivo .REP. 




- En el grupo "Archivos: Este grupo aparece en las pestañas utilizadas para acceder a los archivos de datos.. En particular, permite el acceso directo a los archivos de datos. Para los archivos de datos HFSQL, también es posible lanzar una optimización y reparación de archivos de datos.

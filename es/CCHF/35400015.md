


## Gestión de bases de datos HFSQL Client/Server
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000353"></a>
El Centro control de HFSQL le permite administrar completamente las bases de datos de HFSQL Client/Server.

**Para visualizar las características de una base de datos HFSQL Client/Server**:

1. Conectarse (si es necesario) a un servidor HFSQL. Para obtener más información, consulte [Gestión de bases de datos HFSQL Client/Server](../CCHF/35400013.md). 

2. En la estructura de árbol del Centro control, amplíe el servidor y sus bases de datos.

3. Haga doble clic en el nombre de la base de datos deseada.

4. Aparece una nueva pestaña correspondiente al nombre de la base de datos seleccionada.. Una cinta y diferentes pestañas se utilizan para gestionar las características de la base de datos HFSQL Client/Server.




**Observación**: Desde el Centro control, el menú contextual de la base de datos se utiliza para:

- Desconecte. 

- Importar una base de datos HFSQL Classic. 

- Iniciar la sincronización de datos. 

- Duplicar la base de datos. 

- Eliminar base de datos. 

- Gestionar los derechos sobre la base de datos. 

- Reparar tablas del sistema.

## Características detalladas de una base de datos de HFSQL Client/Server
<a name="caracteristicas_detalladas_una_base_datos_hfsql_clientserver_ELTTEXTE000383"></a>
<a name="NOTE2_1B"></a>


### Pestaña "General"
<a name="pestana_general_ELTPARAGRAPHE000041"></a>

Esta opción se utiliza para get información general sobre los archivos encontrados en la base de datos: nombre, tamaño, número de registros.

Por Default, sólo se muestran los nombres de los archivos de datos. 

El menú contextual de los archivos se utiliza para: 

- Visualizar los detalles del fichero. La información visualizada se calcula y actualiza para el fichero. 

- Acceder al fichero. El contenido del archivo se muestra en una nueva pestaña. 

- Optimizar y reparar el archivo. 




Observación: La opción " Ocultar directorios de copias de seguridad " sólo muestra los directorios que contienen los ficheros de datos de la base de datos. 

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Archivos", se utilizan para: 

- Acceder al archivo: Esta opción se utiliza para abrir una nueva pestaña que muestra las características del archivo seleccionado.. 

- Ver detalles: Esta opción se utiliza para mostrar los detalles de los archivos seleccionados en la pestaña base de datos.. 

- Optimizar y reparar: Esta opción se utiliza para mostrar la ventana para configurar la optimización y la reparación de los archivos HFSQL.

- suprimir: Esta opción borra los archivos seleccionados. 



<a name="NOTE2_2B"></a>


### "Pestaña "Conexiones
<a name="pestana_conexiones_ELTPARAGRAPHE000078"></a>

Esta pestaña lista los diferentes usuarios conectados a la base de datos.

La lista de conexiones de Active contiene la siguiente información:

- el nombre de usuario, 

- El nombre el equipo,

- el nombre de la aplicación que accede a la base de datos, etc.




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Conexiones", se utilizan para: 

- permitir o prohibir las conexiones, 

- desconectar usuarios, 

- enviar un mensaje a los usuarios.




La opción "No mostrar las conexiones de HFSQL control Center (indicadas en este ordenador)" sirve para mostrar únicamente las conexiones de los usuarios a través de las aplicaciones. 
<a name="NOTE2_3B"></a>


### "Pestaña "Elementos programados
<a name="pestana_elementos_programados_ELTPARAGRAPHE000099"></a>

Esta pestaña se utiliza para crear, modificar y eliminar operaciones de programación de la base de datos.: 

- respaldos.

- tareas (procedimientos almacenados).

- optimización.

- actualización de vistas materializadas. 




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Elementos programados", se utilizan para: 

- crear una nueva programación, 

- visualizar los detalles de una programación,

- ejecutar una programación,

- Borrar una programación. 




Para obtener más información, consulte [Administrar las tareas programadas](../CCHF/3540009.md).
<a name="NOTE2_4b"></a>


### "Pestaña "Copias de seguridad
<a name="pestana_copias_seguridad_ELTPARAGRAPHE000123"></a>

Esta pestaña se utiliza para realizar y restaurar copias de seguridad de bases de datos. Esta pestaña lista las copias de seguridad de la base de datos: 

- actualmente en ejecución,

- terminado. 




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Copias de seguridad", se utilizan para: 

- crear una nueva copia de seguridad (copia de seguridad en caliente o programada),

- reiniciar una copia de seguridad,

- restaurar una copia de seguridad,

- eliminar una copia de seguridad. 




Para obtener más información, consulte [copias](../CCHF/3540006.md).
<a name="NOTE2_5b"></a>


### "Pestaña "Enlaces
<a name="pestana_enlaces_ELTPARAGRAPHE000145"></a>

Esta pestaña presenta la información relativa a los enlaces entre los ficheros de datos que se encuentran en la base de datos HFSQL Client/Server.

Se muestra la siguiente información:

- Nombre del enlace, 

- Nombre del archivo fuente, 

- Cardinalidad fuente, 

- Nombre del archivo de destino, 

- Cardenales del destino, 

- Regla de integridad para borrado, 

- Regla de integridad para la modificación. 




Observación: Los enlaces se crean y se editan en editor de análisis. 
<a name="NOTE2_6B"></a>


### "Pestaña "Procedimientos almacenados
<a name="pestana_procedimientos_almacenados_ELTPARAGRAPHE000164"></a>

Esta pestaña lista los diferentes conjuntos de procedimientos almacenados y los procedimientos almacenados de la base de datos actual.

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Procedimientos almacenados", se utilizan para: 

- Actualizar los conjuntos de procedimientos almacenados.

- Borre una set de procedimientos almacenados.




Para obtener más información, consulte [Procedimientos y consultas almacenados](../WDLang4/3044360.md). 
<a name="NOTE2_7B"></a>


### " Ficha Triggers
<a name="ficha_triggers_ELTPARAGRAPHE000181"></a>

Esta ficha enumera los diferentes desencadenantes de la base de datos actual. Puedes get:

- El nombre del gatillo,

- Los ficheros de datos afectados por el trigger,

- El Procedure almacenado llamó al soltar el gatillo,

- Las funciones HFSQL sirven para liberar el trigger.

- El tipo de disparo (antes/después)




El activador puede activarse o desactivarse haciendo clic en la casilla de la columna "Active".

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Disparadores", se utilizan para: 

- crear un nuevo trigger.

- modificar un trigger.

- borrar un disparador.




Para obtener más información, consulte [Disparadores de servidor](../WDLang4/3044369.md). 
<a name="NOTE2_8B"></a>


### "Pestaña "Consultas almacenadas
<a name="pestana_consultas_almacenadas_ELTPARAGRAPHE000207"></a>

Esta pestaña lista las diferentes consultas almacenadas y los procedimientos almacenados de la base de datos actual.

Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Consultas almacenadas", se utilizan para eliminar una consulta almacenada.

Para obtener más información, consulte [Procedimientos y consultas almacenados](../WDLang4/3044360.md). 
<a name="NOTE2_9B"></a>


### "Pestaña "Parámetros
<a name="pestana_parametros_ELTPARAGRAPHE000221"></a>

Esta pestaña lista los diferentes parámetros de la base de datos actual.

Los parámetros se utilizan para almacenar:

- La fecha y hora de la última conexión del usuario,

- Los parámetros de entorno de una aplicación,

- El estado reporte de un Procedure almacenado,

- etc.




Para cada paraméter, tienes la capacidad de averiguarlo:

- El nombre del parámetro,

- El valor del parámetro.




Cuando se selecciona esta ficha, se muestran opciones adicionales en la cinta.. Estas opciones, que se encuentran en el grupo "Parámetros", se utilizan para: 

- crear un nuevo parámetro.

- modificar un parámetro.

- delete a parameter.




Para manipular los parámetros de la base de datos a través de la programación, tiene la posibilidad de utilizar [HSaveParameter](../WDLang4/1000017313.md), [HLoadParameter](../WDLang4/1000017314.md) y [HDeleteParameter](../WDLang4/1000017315.md). 
<a name="NOTE2_10B"></a>


### "Pestaña "Cerraduras
<a name="pestana_cerraduras_ELTPARAGRAPHE000257"></a>

Esta pestaña se utiliza para gestionar los bloqueos de todos los archivos de datos de la base de datos actual.. 

Usted puede averiguar: 

- el nombre del archivo en el que se aplica el bloqueo, 

- el número de la Record cerrada, 

- el nombre del usuario que está bloqueando, 

- el nombre del ordenador de cierre, 

- la IP Address del ordenador de bloqueo, 

- el nombre de la aplicación de cierre, 

- el tipo de cerradura,

- la fecha de inicio del bloqueo,

- la duración del bloqueo.




Esta información se puede recuperar mediante la programación con [HInfoLock](../WDLang4/3044307.md). 

Para desbloquear un Record desde el Centro de Control HFSQL, debes desconectar al usuario que bloqueó el Record. 

Desconectar usuarios:

1. Mostrar el panel de la base de datos (haga doble clic en su nombre en la estructura de árbol). 

2. Haz clic en la pestaña "Conexiones".

3. En la cinta de opciones, en el grupo "Conexiones", haga clic en "Desconectar usuarios".

4. En la ventana que se muestra, compruebe que los usuarios se desconecten.

5. Haga clic en "Desconectar a los usuarios seleccionados".

## Opciones de cinta
<a name="opciones_cinta_ELTTEXTE000461"></a>
Para la base de datos HFSQL Client/Server seleccionada, la cinta propone las siguientes acciones: 

- En el grupo "Consulta SQL: 

	- Nueva consulta: Esta opción se utiliza para abrir una pestaña específica que le permite realizar una consulta en la base de datos actual..  

	- Abrir una consulta: Esta opción se utiliza para abrir una pestaña específica correspondiente a una consulta que se haya ejecutado previamente.. 




- En el grupo "Bases de datos":

	- Importar una base de datos HFSQL Classic

	- Copiar archivos desde el servidor

	- Sincronización de la base de datos

	- Duplicar base de datos

	- Cerrar conexión a la base de datos




- En el grupo "Derechos", la opción "Gestión de derechos" permite gestionar los derechos concedidos a los usuarios y a los grupos en la base de datos.

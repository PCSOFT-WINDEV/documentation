
## Gestión de archivos de datos HFSQL Classic
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000236"></a>
El Centro de Control HFSQL propone una gestión completa de los archivos de datos HFSQL Classic.

**To muestra las características de un archivo de datos** clásico de HFSQL:

1. Mostrar (si es necesario) la ventana de inicio de HFSQL Control Center. Esta ventana se muestra: 

	- al iniciar el Centro de Control HFSQL. 

	- expandiendo ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Passage%20analyse%20en%20CS%20-%20HC%20N%B0009%201.gif) y seleccionando "Abrir un archivo HFSQL Classic".. 




2. Aparece la ventana de inicio del Centro de control.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CC_HF_FichierHFSQLClassic%20-%20HC%20N%B0001.gif&type=thumb)


3. Especifique: 

	- la ruta de la HFSQL Classic archivo de datos para abrir,

	- su contraseña (si es necesario). Usted tiene la capacidad de almacenar la contraseña. 




4. Validar. 

5. El Centro de Control HFSQL muestra el contenido de la archivo de datos seleccionada.. 

	- Las diferentes pestañas utilizadas para gestionar las características de HFSQL Classic archivo de datos.

	- La cinta propone las diferentes acciones que se pueden realizar sobre el archivo. Las opciones pueden variar según la pestaña seleccionada.

## Características detalladas de un HFSQL Classic archivo de datos
<a name="caracteristicas_detalladas_hfsql_classic_archivo_datos_ELTTEXTE000272"></a>


### "pestaña"Descripción
<a name="pestanadescripcion_ELTPARAGRAPHE000079"></a>

Esta pestaña presenta la información relativa a los artículos de la archivo de datos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0001.gif&type=thumb)


Se muestra la siguiente información:

- El nombre del artículo,

- El tipo de artículo,

- El tamaño del artículo,

- El tipo de llave,

- Las estadísticas, 

- Si el artículo está visible en "Informes y consultas".




Si es necesario, se visualiza información adicional.
<a name="NOTE2_2"></a>


### "Ficha"Propiedades
<a name="fichapropiedades_ELTPARAGRAPHE000099"></a>La pestaña "Propiedades" muestra las propiedades de la archivo de datos (nombre físico, formato, protección,...).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0002.gif&type=thumb)

<a name="NOTE2_3"></a>


### "Pestaña"Contenido
<a name="pestanacontenido_ELTPARAGRAPHE000108"></a>

Esta pestaña se utiliza para ver los diferentes registros que se encuentran en el archivo archivo de datos. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0003.gif&type=thumb)


El menú desplegable de registros se utiliza para manipular el registro seleccionado.: borrado, adición, modo de visualización de los memos de texto, ....

Tú también puedes:

- Eliminar un registro: seleccione el registro y haga clic en el botón "Borrar" (o pulse la tecla Supr).

- Modificar o añadir un registro: la entrada se realiza en la lista de registros. Todo lo que tiene que hacer es posicionarse en la primera fila vacía e introducir la información del registro.. La archivo de datos se modifica automáticamente.



<a name="NOTE2_4"></a>


### "Ficha"Buscar
<a name="fichabuscar_ELTPARAGRAPHE000134"></a>

La pestaña "Buscar" sirve para realizar búsquedas en la archivo de datos.

La pestaña "Buscar" sirve para escribir las condiciones de búsqueda de forma explícita (búsquedas QBE).. Para la posición solicitada, todo lo que tiene que hacer es seleccionar una condición y un valor para verificar. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0004.gif&type=thumb)



<a name="NOTE2_6"></a>


### "Ficha Logs
<a name="ficha_logs_ELTPARAGRAPHE000156"></a>

Esta pestaña se utiliza para ver el contenido de los registros y para realizar una búsqueda en los registros.

Para ello, se debe grabar en log el archivo de datos.
<a name="NOTE2_7"></a>


### "Ficha"Transacciones
<a name="fichatransacciones_ELTPARAGRAPHE000165"></a>

Esta pestaña se utiliza para liberar un registro o todas las transacciones.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0005.gif&type=thumb)

## Opciones propuestas por la cinta
<a name="opciones_propuestas_por_cinta_ELTTEXTE000332"></a>
Para el archivo HFSQL Classic seleccionado, la cinta propone las siguientes acciones: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=CCHF_Fichiers_HFClassic%20-%20HC%20N%B0001%201.gif)


- En el grupo "Consulta SQL: 

	- Nueva consulta: Esta opción se utiliza para abrir una pestaña específica que le permite realizar una consulta sobre la archivo de datos actual..  

	- Open a query: Esta opción se utiliza para abrir una pestaña específica correspondiente a una consulta que se haya ejecutado previamente.. 




- En el grupo "Archivo: 

	- Optimizar y reparar: Esta opción se utiliza para mostrar la ventana para configurar la optimización y la reparación de los archivos HFSQL.. 

	- Cerrar el archivo: Esta opción se utiliza para cerrar el archivo y la pestaña asociada..

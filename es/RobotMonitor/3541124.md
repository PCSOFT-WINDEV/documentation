<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000150"></a>
Puede utilizar los modelos de cheques para crear fácilmente cheques. Estos modelos definen la configuración del programa de control. 

Al crear un cheque, en lugar de configurar el calendario individualmente, puede seleccionar simplemente el modelo de cheque correspondiente. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Gestión de los modelos de cheques
<a name="gestion_los_modelos_cheques_ELTTEXTE000174"></a>
Para gestionar los diferentes modelos de cheques, vaya al menú de RobotMonitor, al grupo "Categories and checks" y haga clic en "Check model"..  
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0001.gif)


Esta es la ventana de gestión de cheques:  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0004.gif)


Esta ventana permite: 

- Crea un nuevo modelo,

- Editar un modelo. 

- Ver los controles associated con un modelo. 

- Borrar un modelo. 





### Crear un modelo
<a name="crear_modelo_ELTPARAGRAPHE000041"></a>

Para crear un nuevo modelo: 

1. Haga clic en "Add a new model. Los campos de la ventana se borran.

2. Especificar las diferentes características del modelo: 

	- Nombre del modelo: Este nombre se utilizará en la comprobación de description para seleccionar las opciones de horario deseadas. 

	- interval entre dos cheques: interval entre dos ejecuciones del cheque. Por ejemplo, si la interval es set a 2 horas, la comprobación se realizará cada 2 horas.

	- Número de intentos: Número de veces que se reinicia la comprobación antes de considerarla fallida.

	- Espera entre dos intentos: Las pruebas se lanzan cada dos minutos. El tiempo entre dos intentos no debe superar los 2 minutos (120 segundos).

	- Franjas horarias de ejecución: Periodo de tiempo para realizar la comprobación. Para ejecutar el control durante todo el día, la hora de inicio debe ser "00:00" y la hora de finalización "23:59".

	- Días de ejecución: Días de la semana en los que se debe realizar el control.




3. Haga clic en "Validar". El modelo de comprobación se crea automáticamente. Se añade a la lista de modelos y puede ser associated con un cheque. 





### Editar un modelo
<a name="editar_modelo_ELTPARAGRAPHE000065"></a>

Para editar un modelo: 

1. Seleccione el modelo deseado de la lista. 

2. Editar las características deseadas. 

3. Haga clic en "Validar". El modelo de comprobación se actualiza automáticamente. También se actualizarán las comprobaciones de associated con el modelo.  





### Ver los controles associated con un modelo
<a name="ver_los_controles_associated_con_modelo_ELTPARAGRAPHE000078"></a>

Para ver los cheques associated con un modelo: 

1. Seleccione el modelo deseado de la lista. 

2. Las características del modelo se muestran en la sección "Parámetros del modelo", a la izquierda de la ventana. 

3. Las comprobaciones associated con el modelo aparecen en "Pruebas associated con el modelo", a la derecha de la ventana. 







### Borrar un modelo
<a name="borrar_modelo_ELTPARAGRAPHE000088"></a>

Para eliminar un modelo: 

1. Seleccione el modelo deseado de la lista. 

2. Haga clic en el botón <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0004%201.gif)
.




Atención: Los modelos sólo pueden ser eliminados si no son associated con ningún control. 

Observación: " Delete models without checks" borra todos los modelos que ya no son associated a un control.

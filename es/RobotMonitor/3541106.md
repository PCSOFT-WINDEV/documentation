## Presentación

Desde el monitor, puedes cambiar las opciones de configuración:

- del robot,

- del contra-robot,

- del monitor.

## Configuración de los parámetros del contra-robot
<a name="configuracion_los_parametros_del_contrarobot_ELTTEXTE000173"></a>
Los parámetros del contra-robot corresponden a los parámetros especificados en la asistente para configurar el contra-robot.

**Para visualizar los parámetros del contra-robot**: 

1. En el menú del monitor, vaya al grupo "Ajustes", seleccione "Configurar" y, a continuación, "Parámetros del robot contador".

2. Se abre una ventana que muestra las características del contra-robot:

	- Nombre, ordenador.

	- Opciones de alerta.

## Opciones de alerta
<a name="opciones_alerta_ELTTEXTE000197"></a>


### Alerta de WDBal
<a name="alerta_wdbal_ELTPARAGRAPHE000037"></a>

La pestaña "WDBal" se utiliza para configurar el envío de informes de error a través de la mensajería de PC SOFT denominada WDBal.. Para utilizar este modo de mensaje de alerta, la base de datos de los centros de control debe ser accesible desde la el equipo donde el contra-robot está instalado.

La sección "Configuración de WDBal" sirve para definir el modo de acceso a los datos de los Centros control:

- En el modo de red compartida: directorio de los archivos de datos.

- En modo HFSQL Client/Server: 

	- servidor utilizado, puerto y base de datos.

	- usuario y contraseña. 







Puede definir el o los destinatarios de Default del mensaje enviado a través del sistema de mensajería ("Definir destinatarios") en "Colaboradores". Los destinatarios mostrados son los usuarios que han habilitado WDBal. 
<a name="NOTE3_2"></a>


### Alerta de Email
<a name="alerta_email_ELTPARAGRAPHE000054"></a>

La pestaña "Email" se utiliza para configurar el envío de informes de error por parte de Email. Para utilizar este modo de mensaje de alerta, un servidor SMTP debe ser accesible desde el equipo donde el contra-robot está instalado.

La sección "Configuración de Email" permite definir las características del servidor SMTP utilizado:

- Servidor y puerto SMTP

- Usuario y contraseña de associated para usar el servidor SMTP.

- Tipo de seguridad utilizada. 




La sección "Colaboradores" le permite definir:

- El remitente del mensaje, es decir, el correo electrónico Address utilizado como remitente.

- Los destinatarios de Default del mensaje enviado. Basta con especificar las direcciones Email de los destinatarios.



<a name="NOTE3_3"></a>


### Usando un programa específico
<a name="usando_programa_especifico_ELTPARAGRAPHE000072"></a>

La pestaña "Otro programa" permite enviar informes de error a través de un programa específico. Para utilizar este modo de mensaje de alerta, especifique la ruta del ejecutable deseado. Este ejecutable debe ser accesible desde el ordenador instalación del contra-robot. 

En este modo, el error reporte es un fichero XML. El ruta del error reporte será pasado en el comando Line al programa específico y tendrá que ser procesado por este programa.

El formato del archivo XML es el siguiente:


```xml
<Test>
<TestNumber>%1</TestNumber>
<TestName>%2</TestName>
<TestCategory>%3</TestCategory>
<TestPriority>%4</TestPriority>
<TestSyntax>
%5
</TestSyntax>
<TestSolution>
%6
</TestSolution>
<ErrorDate>%7</ErrorDate>
<ErrorTime>%8</ErrorTime>
<CustomErrorMessage>
%9
</CustomErrorMessage>
<ErrorMessage>
%10
</ErrorMessage>
</Test>
<Test>
...
</Test>
```


En este código:

- %1 = Número que identifica la prueba

- %2 = Nombre de la prueba

- %3 = Nombre de la categoría de prueba

- %4 = Prioridad de la prueba

- %5 = Sintaxis de la prueba (parámetros de associated)

- %6 = Solución propuesta para el ensayo

- %7 = Fecha del error

- %8 = Hora del error

- %9 = Mensaje de error especificado en el formulario de la prueba

- %1 0 = Mensaje de error devuelto por el robot



<a name="NOTE3_4"></a>


### Configurar el status reporte
<a name="configurar_status_reporte_ELTPARAGRAPHE000098"></a>

La pestaña "Configurar el estado reporte" sirve para configurar el asunto del mensaje enviado (por Email o por WDBal).

La sección "Objeto de Email enviado" se utiliza para construir el título del mensaje. Las macros se usan para obtener información específica de get (nombre del contra-robot, ...).

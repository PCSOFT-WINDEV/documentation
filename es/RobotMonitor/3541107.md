
## Motor de monitoreo: Creación de un cheque
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000158"></a>
El monitor del robot de monitorización se utiliza para crear las diferentes comprobaciones que debe realizar el robot de monitorización.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000182"></a>
Las categorías se utilizan para clasificar los cheques por tema. Cada cheque es associated con una categoría.

Si se ha instalado un contra-robot, se crea automáticamente una categoría "Contra-robot".. Esta categoría contiene una prueba para monitorizar el contra-robot.
<a name="NOTE2_2"></a>


### Crear una categoría
<a name="crear_una_categoria_ELTPARAGRAPHE000025"></a>

Para crear una categoría:

1. En el menú, vaya al grupo "Categorías y controles", haga clic en "Añadir" y seleccione "Añadir una categoría" (o seleccione "Crear una categoría" en el menú contextual de la tabla).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0002%201.gif)


2. En la ventana que aparece, introduzca el nombre de la categoría.

3. Validar. La nueva categoría se visualiza en el monitor.



<a name="NOTE2_3"></a>


### Creación de un cheque
<a name="creacion_cheque_ELTPARAGRAPHE000037"></a>

Para crear un cheque:

1. En el menú, vaya al grupo "Categorías y cheques", haga clic en "Añadir" y seleccione "Añadir un cheque" (o seleccione "Crear un cheque" en el menú contextual de la tabla). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0002.gif)


2. Aparece la ventana de configuración de la comprobación. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0003.gif&type=thumb)


3. Especifique: 

	- el nombre del cheque.

	- la categoría del cheque.

	- el tipo de comprobación a realizar.

	- Si el cheque depende de otro cheque.
			Observación: Si la comprobación depende de otra comprobación, la primera sólo se ejecutará si la segunda se ha completado con éxito.  




4. Especifique la [parámetros generales](../RobotMonitor/3541108.md) del cheque.

5. Especificar los parámetros específicos del cheque. Estos parámetros dependen de la clase de verificación seleccionada.

	- [Conectividad informática (ping)](../RobotMonitor/3541113.md).

	- [Conectividad FTP](../RobotMonitor/3541111.md).

	- [Conectividad HTTP](../RobotMonitor/3541112.md).

	- [Conectividad HFSQL Client/Server](../RobotMonitor/3541115.md).

	- [Conectividad con las bases de datos externas (por ODBC)](../RobotMonitor/3541110.md).

	- [Conectividad NNTP](../RobotMonitor/3541114.md).

	- [Actividad de una aplicación WINDEV](../RobotMonitor/3541109.md).

	- [Enviar una Email](../RobotMonitor/3541117.md).

	- [Registro de un sitio WEBDEV](../RobotMonitor/3541119.md).

	- [SNMP](../RobotMonitor/3541121.md).

	- [Espacio en el disco (mediante SNMP)](../RobotMonitor/3541118.md).

	- [WLanguage (código)](../RobotMonitor/3541122.md).

	- [Servidor WEBDEV](../RobotMonitor/3541120.md).

	- [Conectividad SMTP](../RobotMonitor/3541116.md).

	- Estado de un clúster.

	- Actividad de un servidor de telemetría.

	- Sonda de temperatura.

	- Sonda de temperatura (por archivo).

	- [Estado SMART de una Drive](../RobotMonitor/3541125.md).




6. El botón "Prueba" ejecuta la prueba que se está editando. La prueba se ejecuta desde el monitor.

7. Validar. La prueba se muestra en el monitor del robot de monitorización.

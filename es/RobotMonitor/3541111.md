
## Cheque: Conectividad FTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
El robot de monitorización intentará conectarse al servidor FTP con los parámetros del servidor. Puede comprobar la existencia de un archivo en el servidor FTP, así como el número de usuarios conectados. 

El cheque será:

- exitosa si la conexión es efectiva (y si existe el archivo especificado).

- sin éxito de otra manera.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación de la conectividad FTP
<a name="parametros_especificos_comprobacion_conectividad_ftp_ELTTEXTE000120"></a>


### Información para conectarse al servidor FTP
<a name="informacion_para_conectarse_servidor_ftp_ELTPARAGRAPHE000023"></a>

Debe especificar las características para conectarse al servidor FTP:

- la Address del servidor FTP.

- el número de puerto del servidor y el modo de conexión (pasivo o no).

- el usuario y la contraseña de associated.



<a name="NOTE2_2"></a>


### Información sobre el fichero a comprobar
<a name="informacion_sobre_fichero_comprobar_ELTPARAGRAPHE000034"></a>

Tiene la posibilidad de comprobar la existencia de un archivo en el servidor, así como su descarga.. Se debe especificar la siguiente información:

- el nombre completo del archivo en el servidor FTP.

- si el archivo debe ser descargado o no.  



<a name="NOTE2_3"></a>


### Comprobar el número de usuarios conectados
<a name="comprobar_numero_usuarios_conectados_ELTPARAGRAPHE000044"></a>

Puedes comprobar el número de usuarios conectados. Se debe especificar la siguiente información:

- la Address de la página que devuelve la lista de usuarios.

- el número máximo de usuarios.

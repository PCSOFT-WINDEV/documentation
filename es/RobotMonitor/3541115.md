
## Cheque: Conectividad con HFSQL Client/Server
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000095"></a>
El robot de vigilancia intentará conectarse a un servidor HFSQL Client/Server. También tiene la capacidad de comprobar la adición de una Record en el servidor.

El cheque será:

- con éxito si se establece la conexión.

- sin éxito de otra manera.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación relativos a la conectividad con el servidor HFSQL
<a name="parametros_especificos_comprobacion_relativos_conectividad_con_servidor_hfsql_ELTTEXTE000125"></a>


### Información sobre el servidor HFSQL
<a name="informacion_sobre_servidor_hfsql_ELTPARAGRAPHE000025"></a>

Para comprobar un servidor HFSQL, debe especificar:

- el servidor Address,

- puerto de conexión,

- el login utilizado y la contraseña de associated.



<a name="NOTE2_2"></a>


### Comprobación de la adición de registros
<a name="comprobacion_adicion_registros_ELTPARAGRAPHE000036"></a>

Para comprobar la adición de una Record en un servidor HFSQL, debe especificar:

- la base de datos utilizada,

- el archivo de datos donde se debe realizar la adición,

- la contraseña del archivo de datos si es necesario,

- la lista de valores de cada elemento que debe añadirse a el archivo de datos.




**Atención**: 

- **La Record añadida durante esta prueba no será eliminada.**

- El Record se añadirá con los derechos associated con el usuario especificado en la información de conexión (login y contraseña associated).

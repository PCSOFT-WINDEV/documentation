
## Verificar: Conectividad con las bases de datos externas (por ODBC)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000074"></a>
El robot de monitorización intentará conectarse a una base de datos externa (MySQL, SQL Server, ...) mediante ODBC.

El cheque será:

- exitosa si la conexión es efectiva.

- sin éxito de otra manera.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación relativos a la conectividad con las bases de datos externas
<a name="parametros_especificos_comprobacion_relativos_conectividad_con_las_bases_datos_externas_ELTTEXTE000098"></a>
Debe especificar el nombre de la fuente ODBC que desea verificar. Esta fuente de ODBC debe ser definida en la computadora instalación del robot de monitoreo.

**Observación**: Para ejecutar una prueba localmente, desde el monitor, se debe definir la misma fuente ODBC en este ordenador.

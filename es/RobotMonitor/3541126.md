## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
El robot de supervisión le permite definir los grupos de usuarios que recibirán los diferentes informes de errores de comprobación. 

Para gestionar grupos de usuarios: 

1. En el menú del monitor, vaya al grupo "Configuración" y seleccione "Gestionar grupos".. 

2. La ventana que aparece le permite: 

	- Añadir o eliminar un usuario de un grupo. 

	- Agregar un grupo.

	- Cambiar el nombre de un grupo.

	- Borrar un grupo.

## Características de un grupo
<a name="caracteristicas_grupo_ELTTEXTE000101"></a>
Cuando añades un nuevo grupo, se abre una ventana que te permite introducir un nombre. El nuevo grupo aparecerá en la lista de grupos. 
Para asociar usuarios a un grupo, basta con seleccionar el grupo en la lista y marcar los usuarios deseados. 

Una vez creado el grupo, puede ser set como destinatario de los informes de error de comprobación (en los tings set generales de comprobación)..

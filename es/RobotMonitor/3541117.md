## Presentación
<a name="presentacion_ELTTEXTE000090"></a>
El robot de monitoreo lo hará:

- enviar un correo electrónico a un correo electrónico Address.

- leer la bandeja de entrada del Email Address para comprobar si el Email fue recibido.




El cheque será:

- ...con éxito si el Email fue enviado y si se encontró en el buzón de entrada.

- sin éxito de otra manera.

## Parámetros específicos de la comprobación de los correos electrónicos salientes
<a name="parametros_especificos_comprobacion_los_correos_electronicos_salientes_ELTTEXTE000114"></a>


### Parámetros SMTP (para enviar el mensaje)
<a name="parametros_smtp_para_enviar_mensaje_ELTPARAGRAPHE000026"></a>

Los parámetros SMTP son los del servidor utilizado para el envío de correos electrónicos:

- el correo electrónico Address al que se enviará el mensaje de prueba.

- el servidor SMTP a utilizar y el puerto associated.

- el usuario y la contraseña de associated.



<a name="NOTE2_2"></a>


### Parámetros POP3 (para leer el mensaje entrante)
<a name="parametros_pop3_para_leer_mensaje_entrante_ELTPARAGRAPHE000037"></a>

Los parámetros SMTP son los del servidor utilizado para el envío de correos electrónicos:

- el correo electrónico Address al que se leerá el mensaje de prueba.

- el servidor POP3 a utilizar y el puerto associated. La conexión puede ser una conexión SSL.

- el usuario y la contraseña de associated.

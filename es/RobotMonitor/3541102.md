
## Instalando el robot controller
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000137"></a>
El robot de monitorización incluye tres ejecutables.

El primer ejecutable en instalar es el [el ejecutable correspondiente al robot](../RobotMonitor/3541101.md). El robot de monitorización se inicia siempre en un ordenador y realiza todas las pruebas necesarias.. Debe estar instalado en un ordenador que nunca se apagará.

El segundo ejecutable es el robot ** controller (o contra-robot)**. **Su configuración es opcional**. El contra-robot es una herramienta de seguridad adicional. Se utiliza para comprobar si el robot de monitorización funciona correctamente.. Si está instalado, el robot para contador debe instalarse en un ordenador que nunca estará apagado, a diferencia del ordenador de configuración del robot de monitorización..  

El tercer ejecutable es el [monitor del robot](../RobotMonitor/3541103.md). El monitor se puede instalar en cualquier ordenador. Permite editar las pruebas y los diferentes parámetros del robot.

## Asistente para la instalación del robot controller
<a name="asistente_para_instalacion_del_robot_controller_ELTTEXTE000161"></a>


### Instalando el robot controller
<a name="instalando_robot_controller_ELTPARAGRAPHE000030"></a>

Para instalar el robot controller:

1. Inicie el paquete de instalación (WX28 0PACKROBOT.EXE) en el ordenador donde se instalará el robot controller. Recordatorio: El paquete de instalación (WX28 0PACKROBOT.EXE) está disponible en el subdirectorio "Instalar Robot _ASLASH_" del producto utilizado.

2. Valide la licencia.

3. Seleccione "Instalar el controlador del robot" y vaya al paso siguiente.

4. Seleccione el directorio de configuración del robot controller. Pase a la etapa siguiente.  

5. La instalación ha finalizado. Valide. El robot controller está instalado. Todo lo que tienes que hacer ahora es configurar el robot controller. 



<a name="NOTE2_2"></a>


### Configuración del contra-robot
<a name="configuracion_del_contrarobot_ELTPARAGRAPHE000047"></a>

Una vez instalado el robot, se muestra automáticamente un asistente para configurar el robot.. En este asistente: 

1. Especifique el nombre del contra-robot. Este nombre se utilizará en los diferentes informes de estado que se envíen. Le aconsejamos que utilice un nombre que pueda ser fácilmente identificable. Pase a la etapa siguiente.

2. Especificar la ubicación de los archivos de robot. Este directorio corresponde al directorio compartido creado en el ordenador en el que se instaló el robot.. Pase a la etapa siguiente.

3. Seleccione las opciones de alerta del contra-robot. Estas opciones serán usadas por Default para todas las pruebas realizadas por el robot de monitorización. Tú puedes:

	- utilizar la mensajería interna de WINDEV y WEBDEV: WDBal. Esta opción asume que la base de datos de los Centros Control es accesible desde la computadora del robot de monitoreo.

	- enviar los mensajes por Email. Esta opción asume que la computadora del robot de monitoreo puede acceder a un servidor de Email.

	- use otro programa, que realizará un Process específico de los mensajes. Este otro programa debe ser accesible desde el ordenador del robot de monitorización..
			Las alertas del contra-robot también se pueden seleccionar en el monitor. Le aconsejamos que seleccione al menos una de las opciones de alerta para el robot.




4. Pase a la etapa siguiente. Este plano depende de las opciones que se hayan seleccionado previamente.

	- **Si se debe usar la mensajería de WDBal**Especifique

		- el nombre del remitente del mensaje (y la contraseña de associated si es necesario). Este nombre debe corresponder al nombre de un colaborador que se encuentre en la base de datos de los Centros Control.

		- los receptores de Default. El nombre especificado en el tabla debe corresponder al nombre de un colaborador que se encuentra en la base de datos de los Centros Control.
						Observación: Tendrá la posibilidad de añadir nuevos destinatarios desde el monitor.

		- los parámetros para acceder a la base de datos de los Centros Control:

			- de la base de datos si los archivos utilizan una red compartida.

			- características de la base de datos HFSQL Client/Server (servidor, puerto, base de datos).




- **Si se debe enviar un Email**, especifique:

	- el correo electrónico Address del remitente. Este Address es el que aparece en el Email.

	- las características del servidor SMTP que se utilizará para enviar los mensajes: servidor, puerto, usuario y contraseña asociada (estas características son suministradas por el proveedor INTERNET).

	- las direcciones de correo electrónico de los receptores de Default.
						Observación: Tendrá la posibilidad de añadir nuevos destinatarios desde el monitor.

5. Especifique la cuenta de Windows (y la contraseña de associated) utilizada para crear la tarea programada que iniciará el contra-robot. Atención: esta cuenta debe tener los derechos de gestión para realizar esta operación. Si se utiliza un dominio, especifique el nombre de dominio. 
	**Observación**: Si se utilizan nombres de dominio, los tres programas (robot, contra-robot y monitor) deben estar instalados en el mismo dominio..
	Pase a la etapa siguiente.  

6. La instalación ha finalizado. Valide. Se realiza el ajuste del contra-robot.




Usted tiene la capacidad de [instalar un monitor](../RobotMonitor/3541102.md).

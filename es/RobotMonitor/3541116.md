
## Cheque: conectividad SMTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
El robot de monitorización intentará conectarse a un servidor SMTP con el usuario especificado.

El cheque será:

- exitosa si la conexión es efectiva.

- sin éxito de otra manera.

<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación de la conectividad SMTP
<a name="parametros_especificos_comprobacion_conectividad_smtp_ELTTEXTE000100"></a>


### Información para la conexión al servidor SMTP
<a name="informacion_para_conexion_servidor_smtp_ELTPARAGRAPHE000023"></a>

Debe definir las características para conectarse al servidor SMTP:

- la Address del servidor SMTP.

- el número de puerto para la conexión.

- el usuario y la contraseña de associated.

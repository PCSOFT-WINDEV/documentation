
## Verificar: Conectividad NNTP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000075"></a>
El robot de monitorización intentará conectarse al servidor NNTP especificado (servidor de noticias).

El cheque será:

- exitosa si la conexión es efectiva.

- sin éxito de otra manera.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación de la conectividad NNTP
<a name="parametros_especificos_comprobacion_conectividad_nntp_ELTTEXTE000099"></a>


### Información sobre el servidor NNTP
<a name="informacion_sobre_servidor_nntp_ELTPARAGRAPHE000023"></a>

Para comprobar un servidor NNTP, debe especificar:

- la Address del servidor NNTP.

- el puerto de conexión.


## Verificar: SNMP
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000091"></a>
El robot de vigilancia leerá el valor del OID especificado y comparará su valor con el valor de reference.

El cheque será:

- exitoso si se leyó el valor del OID y si corresponde al valor de reference.

- sin éxito de otra manera.

<a name="NOTE2_1"></a>


## Parámetros específicos de la comprobación SNMP
<a name="parametros_especificos_comprobacion_snmp_ELTTEXTE000115"></a>


### Información sobre el agente SNMP
<a name="informacion_sobre_agente_snmp_ELTPARAGRAPHE000023"></a>

Para realizar una comprobación SNMP, especifique:

- la Address del agente SNMP para usar,

- la comunidad del agente.



<a name="NOTE2_2"></a>


### Información sobre el OID para comprobar
<a name="informacion_sobre_oid_para_comprobar_ELTPARAGRAPHE000033"></a>

Para realizar una comprobación SNMP, especifique:

- el OID preestablecido si existe,

- las condiciones de ensayo del valor. 




**Observación**: Para añadir un OID preestablecido:

1. Introduzca su nombre ("Preset OIDs" control).

2. Introduzca su valor ("OID a leer" control).

3. Haga clic en el botón "+".

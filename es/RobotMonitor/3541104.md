## Presentación
<a name="presentacion_ELTTEXTE000111"></a>
El robot de supervisión permite gestionar los usuarios relacionados con los diferentes informes de errores de comprobación. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Cr%E9er%20un%20controle%20-%20HC%20N%B0001.gif)


Para gestionar los usuarios: 

1. En el menú del monitor, vaya al grupo "Configuración" y seleccione "Gestionar usuarios".. 

2. La ventana que aparece le permite: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Utilisateur%20-%20HC%20N%B0001.gif)


	- Fusionar dos usuarios. 

- Agregar un usuario. 

- Editar un usuario. 

- Eliminar un usuario.

## Características de un usuario
<a name="caracteristicas_usuario_ELTTEXTE000135"></a>
La ventana de parámetros de usuario se muestra cuando se añade o edita un usuario. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Robot_Utilisateur%20-%20HC%20N%B0002.gif&type=thumb)


Es posible definir: 

- **las opciones de alerta**: 
	Estas opciones permiten emitir alertas visuales o sonoras si se produce un error. Si se elige una alerta de sonido, puede seleccionar un archivo personalizado Sound( ".Wav") a través del botón "Examinar".. 

- **las opciones de usuario**: 
	Estas opciones le permiten definir: 

	- el nombre de usuario WDBal associated con el usuario. Este nombre de usuario se utilizará para enviar informes a través de la aplicación de mensajería WDBal. 

	- el correo electrónico Address del usuario. Esta Address se utilizará para enviar informes por Email.  




- **opciones para la síntesis reporte**:
	La síntesis reporte contiene:

	- el historial de las modificaciones realizadas en los controles.

	- los errores de los diferentes controles.  


 Esta reporte está recomendada para los administradores. Es posible indicar: 

- la frecuencia con la que se envía la reporte,

- el modo de envío utilizado (WDBal o Email). 

- **las opciones de resumen**: 
	El resumen contiene una lista de comprobaciones con errores. Proporciona una revisión periódica de los problemas encontrados. Es posible indicar: 

	- a qué hora se envía el resumen,

	- la lista de controles que se incluirán en el resumen. 

	- cómo se envía el resumen (WDBal o Email). 




- **niveles de error para el usuario**: 
	Cada usuario puede optar por recibir la reporte o no, en función del nivel de error encontrado en la comprobación. Por ejemplo, el administrador de la red puede no querer la información de depuración.

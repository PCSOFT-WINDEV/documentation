## Presentación
<a name="presentacion_ELTTEXTE000159"></a>

- Las columnas "Número de Record" y "Lupa".

- La visualización de las notas y de las teclas compuestas.

## Columnas específicas
<a name="columnas_especificas_ELTTEXTE000189"></a>


### "Columna"Número de Record
<a name="columnanumero_record_ELTPARAGRAPHE000023"></a>

Al visualizar un archivo de datos HFSQL, aparece la columna "Número de Record".. El número Record se utiliza para identificar y/o manipular cada uno de los Record que se encuentran en un archivo de datos HFSQL.

**Observación**: No puedes:

- Modificar los números de Record.

- Ordena los números de Record (en orden ascendente o descendente).

- Mueve la columna "Número de Record".

- Encuentra un número de Record.



<a name="NOTE2_2"></a>


### "Magnifier" column
<a name="magnifier_column_ELTPARAGRAPHE000037"></a>

La columna "Lupa" sirve para visualizar los datos de relacionado. Para un pedido, usted tiene la posibilidad de ver las líneas de pedido y los productos correspondientes.

Para ver los datos de relacionado: 

1. Haga clic en la columna "Lupa. Usted get la lista de archivos de relacionado y el nombre del elemento sobre el que se realiza el enlace.

2. Seleccione el archivo para ver y validar. Aparece la lista de registros para el fichero relacionado correspondiente a la posición seleccionada en la carpeta principal.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Fichier_lie.gif)

## Casos especiales

### Caso especial: Posición de nota de texto
<a name="caso_especial_posicion_nota_texto_ELTPARAGRAPHE000052"></a>

La opción "Parameters .. Multiline memo column" se utiliza para visualizar:

- La primera Line de los memorandos de texto. Esta opción se utiliza para optimizar la visualización de todos los registros del archivo.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MemoTexte1.gif)


- El contenido completo de las notas de texto. Las columnas correspondientes se convierten en multilínea. La altura de las filas de tabla es proporcional al tamaño de los memos.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=MemoTexte2.gif)





Para ver el texto completo de la nota y/o modificar el contenido de una nota de texto:

1. Seleccione la columna de notas de la Record solicitada.

2. Select "Edit .. Memo column".

3. El contenido de la nota de texto seleccionada aparece en una ventana específica.


**Observación**: El contenido de una nota de texto también se puede modificar desde la visualización de tabla de WDMap.

### Caso especial: Binario Partida pro memoria
<a name="caso_especial_binario_partida_pro_memoria_ELTPARAGRAPHE000072"></a>

WDMap le permite cambiar, exportar y reproducir los memos binarios.

Para cambiar el memo binario associated con el artículo:

1. Haga doble clic en la celda que contiene la nota binaria. Se muestra la ventana de modificación de la nota binaria.

2. Haga clic en el botón "Importar. Se muestra un selector de archivos.

3. Seleccione el nuevo memo binario.




Para exportar un memo binario a un nuevo archivo:

1. Haga doble clic en la celda que contiene la nota binaria. Se muestra la ventana de modificación de la nota binaria.

2. Haga clic en el botón "Exportar. Se muestra un selector de archivos.

3. Introduzca el nombre del fichero a generar. Por defecto, se proponen el nombre y la extension del fichero associated con la nota binaria.




Para tocar un memo binario:

1. Haga doble clic en la celda que contiene la nota binaria. Se muestra la ventana de modificación de la nota binaria.

2. Haga clic en el botón "Reproducir.
	**Observación**: Este botón es Active sólo si el memo puede ser ejecutado.

### Caso especial: Claves compuestas
<a name="caso_especial_claves_compuestas_ELTPARAGRAPHE000098"></a>

Por Default, los elementos de "clave compuesta" no se muestran en WDMap.

**Para mostrar los elementos de tipo "clave compuesta"**, seleccione la opción "Vista .. Mostrar claves compuestas".

Los diferentes valores de una "clave compuesta" están separados por comas.

En el siguiente ejemplo, el elemento "clave compuesta" denominado *DATECODENAME* está hecho de la *ORDEN DE FECHA*, *ORDCODO* y *NOMBRE* artículos.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=clecomposee.gif)


**Para evitar que se muestren los elementos de la "clave compuesta**Seleccione "Visualizar. Retire las teclas compuestas".

**Observación**: Para modificar el valor de un elemento de "clave compuesta", modifique los valores de los diferentes elementos que se encuentran en la "clave compuesta".

## Presentación
<a name="presentacion_ELTTEXTE000185"></a>
WDMap es una herramienta utilizada para ver y manipular un archivo de datos mientras se desarrolla una aplicación. **Esta herramienta sólo está disponible en modo de 32 bits.**

WDMap puede ser utilizado para:

- Compruebe rápidamente el contenido de una Record.

- Probar una aplicación con datos significativos.




El archivo de datos utilizado puede ser: 

- en formato HFSQL (Clásico o Cliente/Servidor). 

- en formato Hyper File 5.5

- acceso a través de un proveedor de OLE DB, un controlador ODBC o un acceso nativo. 


También tiene la capacidad de manipular un alias de un archivo de datos HFSQL.

**Importante**: WDMap sólo está disponible en modo de 32 bits.. Si WINDEV, WEBDEV o WINDEV Mobile están instalados en modo de 64 bits, WDMap se ejecuta en modo de 32 bits. Para un acceso a través de un proveedor OLE DB, un controlador ODBC o un Conector Nativo, puede utilizar uno de los siguientes métodos: 

- get un cliente de 32 bits para la base de datos,

- utilizar el sistema [Centro de Control HFSQL](../CCHF/3540001.md).

## Iniciar WDMap
<a name="iniciar_wdmap_ELTTEXTE000215"></a>


### Iniciar WDMap
<a name="iniciar_wdmap_ELTPARAGRAPHE000041"></a>

WDMap se puede iniciar:

- desde el editor de análisis: en la pestaña "Análisis", en el grupo "Archivo", haga clic en "Editar con WDMap", 

- de la cinta: en la pestaña "Herramientas", en el grupo "Base de datos", haga clic en "WDMap", 

- ejecutando "WDMAP.EXE" directamente.




**Observación**: Si WDMap se inicia desde editor de análisis:

- Si es necesario, se creará el archivo físico correspondiente a la description del archivo de datos seleccionado.

- Para tener en cuenta las modificaciones realizadas en el fichero description, le aconsejamos que genere el análisis antes de ver el contenido del fichero.



<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000081"></a>

WDMap no puede ser redistribuido. WDMap no se puede instalar con las aplicaciones desarrolladas con WINDEV o WEBDEV.

Para utilizar WDMap , WINDEV, WEBDEV o WINDEV Mobile deben estar instalados en el ordenador actual.

## Notas
<a name="notas_ELTTEXTE000269"></a>


- WDMap puede ser usado para ver un solo archivo a la vez.

- Si el análisis está protegido por una contraseña (contraseña de tiempo de ejecución definida en el análisis description), esta contraseña se solicita cuando se selecciona el análisis en WDMap.

- Si el archivo de datos está protegido por una contraseña (contraseña definida por [HPass](../WDLang4/3044108.md), [HCreation](../WDLang4/3044255.md) o [HCreationIfNotFound](../WDLang4/3044254.md), o durante el [Modificación de datos de automatic](../WDLang4/3044195.md)), esta contraseña se solicitará cuando se seleccione el archivo físico en WDMap.

- Tiene la posibilidad de ver los datos que se encuentran en un archivo de datos HFSQL Client/Server: por ejemplo, si los archivos de análisis se definen en modo Cliente/Servidor, con una conexión, la opción "Editar el archivo de datos con WDMAP" abre el archivo de datos de servidor según la conexión especificada.

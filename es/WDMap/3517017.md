
## WDMap: Añadir, modificar y borrar los registros de un archivo de datos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
WDMap permite realizar operaciones simples (adición, modificación o eliminación) en los registros que se encuentran en un archivo de datos.

Estas operaciones sólo se pueden realizar durante el desarrollo de la aplicación (WDMap no se puede redistribuir).

Por ejemplo, puede añadir, modificar o eliminar registros antes de que se creen las ventanas correspondientes en su aplicación.. También puede ejecutar directamente la prueba de sus procesos.

**Atención**: Si las operaciones afectan a elementos que se encuentran en archivos relacionado (en el editor de análisis), le recomendamos que habilite la gestión de la integridad y los duplicados.. 
La gestión de integridad y la gestión de duplicados están habilitadas por Default. 
La gestión de la integridad sólo está habilitada si se conoce el análisis de origen del archivo.: la gestión de la integridad no está habilitada si el archivo se abre directamente en el explorador mediante WDMap.

**Observaciones**:

- Para permitir la gestión de la integridad, seleccione "Parámetros .. Gestión de la integridad".. 
	**Atención**: Es posible que a veces desee localizar manualmente los archivos relacionado en el archivo utilizado. Para localizar manualmente los archivos que faltan, haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeGestionIntegrite.gif) y seleccione el directorio de cada archivo que participa en la gestión de la integridad del archivo actual.

- Para permitir la gestión de los duplicados, seleccione "Parámetros .. Gestión de duplicados".

## Añadir un Record en un archivo de datos
<a name="anadir_record_archivo_datos_ELTTEXTE000176"></a>
Para añadir un Record en un archivo de datos:

1. Muestra la el archivo de datos a la que se añadirá la Record (para más detalles, véase [Viendo los registros encontrados en un archivo de datos](../WDMap/3517012.md)).

2. Seleccione "Editar... Agregar un Record" (o haz clic en el primer fila vacío de ver tabla). Se añade una nueva fila a la visualización de tabla.

3. En este nuevo fila, introduzca los valores correspondientes al nuevo Record.
	**Observación**: Las teclas de flecha le permiten moverse de una columna a otra.

4. Haga clic en otro fila para validar la adición..
	**Observación**: El mecanismo de seguridad HFSQL se activará si se produce un error de duplicado mientras la gestión de duplicados está habilitada.. Modificar los valores y validar.




**Observación**: El WDMap también puede usarse para duplicar un Record:

1. Selecciona la Record para duplicar.

2. Seleccione la opción "Edición .. Duplicar un registro" (o presione Ctrl + D).

## Modificación de un Record en un archivo de datos
<a name="modificacion_record_archivo_datos_ELTTEXTE000200"></a>
Para modificar un Record en un archivo de datos:

1. Mostrar el archivo de datos donde se modificará el Record (para más detalles, ver [Viendo los registros encontrados en un archivo de datos](../WDMap/3517012.md)).

2. Selecciona la Record a modificar en la visualización de tabla.

3. Haga clic en la columna para modificar. El cursor cambia a un cursor de texto.

4. Introduzca el nuevo valor.

5. Haga clic en otro fila para validar la modificación..
	**Observación**: El mecanismo de seguridad HFSQL se activará si se produce un error de duplicado mientras la gestión de duplicados está habilitada.. Modificar los valores y validar.




**Procesamiento por lotes**: WDMap le permite modificar todos los registros mostrados en una sola operación:

1. Seleccione "Editar... Modificar estos registros".

2. Seleccione "&lt;Inicialización&gt;" en las filas correspondientes a los elementos a modificar.

3. Introduzca los nuevos valores.

## Borrar un Record de un archivo de datos
<a name="borrar_record_archivo_datos_ELTTEXTE000224"></a>
Para borrar un Record de un archivo de datos:

1. Mostrar el archivo de datos de la que hay que borrar el Record (para más detalles, ver [Viendo los registros encontrados en un archivo de datos](../WDMap/3517012.md)).

2. Selecciona el Record a borrar en el tabla de visualización.

3. Seleccione "Editar... Borrar la Record" (o pulsar la tecla Supr).

4. Confirmar el borrado.
	**Observación**: Se muestra un mensaje de error si se aplica una regla de integridad al Record borrado mientras la gestión de la integridad está activada. El Record no será borrado.




**Consejo**: Todos los registros visualizados pueden ser borrados en una sola operación a través de "Editar .. Borrar todos los registros".

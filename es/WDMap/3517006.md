
## WDMap: Reindexación de un archivo de datos HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000122"></a>
WDMap se utiliza para averiguar si un archivo de datos HFSQL debe ser reindexado y reindexar un archivo de datos HFSQL.

Existen dos tipos de operaciones de reindexación:

- ***Reindexación estándar***: Reindexación estándar (con estadísticas y sin compresión).

- ***Reindexación con compresión***: Reindexación con compresión (sólo mantiene los registros de Active. Todos los registros cruzados o eliminados se eliminan de forma permanente).




**Recordatorio**:

- Los índices se reorganizan mediante la operación de reindexación. Después de la operación de indexación, se optimiza el acceso a los índices. La operación de indexación puede ser necesaria si el archivo de indexación se pierde o se daña.

- WDMap te permite reindexar fácilmente a un archivo de datos. Sin embargo, para reindexar un archivo de forma más completa, utilice [WDOptimizer](../WDOptimiseur/3519002.md).

- También puede reindexar un archivo desde el explorador de Windows. Para ello, seleccione "Optimizar" en el menú contextual del archivo a reindexar. A continuación, la reindexación es realizada por [WDOptimizer](../WDOptimiseur/3519002.md).

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000152"></a>


### WDMap comenzó desde el editor
<a name="wdmap_comenzo_desde_editor_ELTPARAGRAPHE000037"></a>

Para reindexar a un archivo de datos:

1. Mostrar el archivo de datos para reindexar (para más detalles, ver [Viendo los registros encontrados en un archivo de datos](../WDMap/3517012.md)).

2. Para actuar:

	- a ***reindexación estándar***: Seleccione "Herramientas .. Reindexación.... Estándar".

	- a ***reindexación con compresión***: seleccione "Herramientas .. Reindexación.... Con compresión".






<a name="NOTE2_2"></a>


### WDMap iniciado por el programa "WDMAP.EXE
<a name="wdmap_iniciado_por_programa_wdmapexe_ELTPARAGRAPHE000053"></a>

Para reindexar a un archivo de datos:

1. En la asistente, seleccione "Reindexar un archivo" y vaya al siguiente plano.

2. Seleccione el archivo a reindexar y valide.




Observaciones:

- Si el archivo está protegido por contraseña (contraseña definida por [HPass](../WDLang4/3044108.md), [HCreation](../WDLang4/3044255.md) o [HCreationIfNotFound](../WDLang4/3044254.md), o durante el [Modificación de datos de automatic](../WDLang4/3044195.md)), especifique esta contraseña.

- Si el archivo proviene de un análisis en formato 5.5, especifique la ruta completa del análisis de origen y su contraseña.


## WDMap: Exportación de registros de un archivo de datos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000089"></a>
WDMAP le permite exportar registros a:

- un archivo de Excel. Los registros estarán disponibles para ser utilizados por Excel (estadísticas, cálculos,...).

- un archivo XML. Entonces, los registros pueden ser mostrados en una página de INTERNET.

- un archivo de texto.




Los registros exportados corresponden a los que se visualizan en WDMap. Para exportar una selección de registros, basta con [seleccionar los registros vistos](../WDMap/3517011.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000113"></a>
Para exportar todos los registros visualizados:

1. Visualizar el formulario el archivo de datos al que deben exportarse los registros (para más detalles, véase [Viendo los registros encontrados en un archivo de datos](../WDMap/3517012.md)).

2. Seleccione los registros a exportar. Para obtener más información, consulte [Selección de los registros vistos](../WDMap/3517011.md).
	**Recordatorio**: Sólo se exportarán los registros visualizados en WDMap.

3. Seleccione "Archivo... Exportar" y seleccione el tipo de exportación deseado (a Excel, a un archivo XML o a un archivo de texto).. Se muestra un selector de archivos.

4. Introduzca el nombre (y ruta) del fichero generado.. No es necesario introducir la extensión del fichero generado.




**Observación**: Para exportar todos los registros vistos, también tienes la posibilidad de seleccionar "Exportar" de el menú contextual del tabla (clic con el botón derecho del ratón).


## WDMap: Características de los archivos encontrados en una base de datos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000232"></a>
WDMap se utiliza para get varias características específicas:

- cada archivo encontrado en una base de datos utilizada por un proyecto WINDEV, WEBDEV o WINDEV Mobile, 

- todos los archivos encontrados en una base de datos utilizada por un proyecto WINDEV, WEBDEV o WINDEV Mobile.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000262"></a>


### WDMap comenzó desde el editor
<a name="wdmap_comenzo_desde_editor_ELTPARAGRAPHE000023"></a>

A get información sobre los archivos encontrados en una base de datos:

1. En WDMap, seleccione el análisis WINDEV, WEBDEV o WINDEV Mobile (archivo ".WDD") que contiene la description del archivo de datos solicitado (haga clic en el botón "Examinar" si es necesario).

2. Si es necesario, indique la contraseña del análisis (contraseña en tiempo de ejecución definida en la descripción del análisis.

3. Seleccione "Mostrar .. Información general sobre la base de datos" (o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeInfo.gif)). Se visualiza la ventana que contiene las características de la base de datos.



<a name="NOTE2_2"></a>


### WDMAP iniciado por "WDMAP.EXE" directamente
<a name="wdmap_iniciado_por_wdmapexe_directamente_ELTPARAGRAPHE000036"></a>

A get información sobre los archivos encontrados en una base de datos:

1. En asistente, seleccione "Seleccionar un análisis" y vaya al paso siguiente.

2. Seleccione el análisis WINDEV, WEBDEV o WINDEV Mobile (archivo ".WDD") que contiene la description del archivo de datos solicitado (utilice el botón "Examinar" si es necesario) y vaya al paso siguiente. Se inicia WDMap.

3. Si es necesario, indique la contraseña del análisis (contraseña en tiempo de ejecución definida en la descripción del análisis.

4. Seleccione "Mostrar .. Información general sobre la base de datos" (o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeInfo.gif)). Se visualiza la ventana que contiene las características de la base de datos.




<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## ¿Cuáles son las características visualizadas?
<a name="¿cuales_son_las_caracteristicas_visualizadas_ELTTEXTE000292"></a>
Una base de datos utilizada por un proyecto WINDEV, WEBDEV o WINDEV Mobile puede incluir archivos en formato HFSQL así como archivos en otro formato.

La ventana que contiene las características se divide por la mitad:

- La sección superior contiene las características específicas de los archivos HFSQL.

- La sección inferior contiene las características específicas de los otros tipos de archivos de datos (no en formato HFSQL) a los que se accede a través de un proveedor de OLE DB, un controlador ODBC o un acceso nativo.




**Observación**: Si la base de datos actual sólo incluye archivos HFSQL, sólo se mostrará la sección superior.. Si la base de datos actual sólo incluye archivos que no están en formato HFSQL, sólo se mostrará la sección inferior.

| **Información adicional relacionada con el archivo físico al hacer doble clic en un archivo de datos HFSQL** |
| Directorio de archivos | Directorio del archivo físico. |
| Fecha de la última modificación |   |
| Vía de análisis en la versión 5.5 | Sólo si el archivo seleccionado proviene de un análisis de la versión 5.5. |

Para todos los archivos de datos HFSQL que se encuentran en la base de datos, se realizan automáticamente los siguientes cálculos:

- El tamaño total (en KB) de todos los archivos de datos.

- El tamaño (en KB) de los archivos de registro associated con los archivos de datos.

- El número de registros encontrados en todos los archivos de datos.

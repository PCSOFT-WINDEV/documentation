
## WDMap: Modificar el modo de visualización de los registros
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000193"></a>
Por Default, WDMap muestra los registros encontrados en un archivo de datos usando las siguientes reglas:

- Todos los registros encontrados en el archivo de datos se muestran.

- Los registros se clasifican según su orden de creación en el archivo de datos.

- Todos los elementos que se encuentran en el archivo de datos se muestran (excepto los de "Clave compuesta").

- Las posiciones están organizadas en orden lógico. El orden lógico corresponde al orden de visualización definido en la editor de análisis.




Sin embargo, usted tiene la capacidad de modificar:

- La selección de la información a mostrar. De hecho, WDMap le permite seleccionar:

	- los registros para ver,

	- los elementos a visualizar.




- Las opciones de visualización de tabla:

	- búsqueda de registros,

	- tipo de registros mostrados,

	- modificación del orden de presentación de los artículos,

	- columnas de anclaje a la izquierda de tabla.







<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Visualización de una selección de registros únicamente
<a name="visualizacion_una_seleccion_registros_unicamente_ELTTEXTE000217"></a>
Para simplificar sus búsquedas, WDMap le permite mostrar los registros correspondientes a un criterio específico.
Por ejemplo, puede visualizar los clientes que salen de "Boston" o los proveedores cuyo nombre empieza por la letra "F".

Existen dos métodos:

- Entrada directa de uno o más criterios de selección en las posiciones de fichero ("Visualizar... Selección de registros... Asistente" o Ctrl + Q).

- Utilizando una consulta SQL en los registros del archivo ("Mostrar... Selección de registros... Consulta SQL").




**Observación**: Estos criterios de selección no se graban. Durante un próximo uso de WDMap, tendrá que redefinir estos criterios para realizar una nueva selección.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Mostrando sólo algunos artículos
<a name="mostrando_solo_algunos_articulos_ELTTEXTE000241"></a>
Si sus archivos de datos contienen varios elementos, WDMap le permite mostrar sólo los elementos requeridos por su búsqueda.

Por ejemplo, si su archivo "Cliente" contiene más de 20 elementos, puede elegir ver el apellido, el nombre y la ciudad de cada cliente.

Para seleccionar los elementos a visualizar, seleccione "Visualizar... Selección de columnas a visualizar" (o pulse Ctrl + O) y compruebe los elementos a visualizar.

**Observación**: Este criterio de selección no se graba. Durante el próximo uso de WDMap, tendrá que redefinir este criterio para realizar una nueva selección.

## Configurar la visualización de tabla
<a name="configurar_visualizacion_tabla_ELTTEXTE000265"></a>


### Presentación
<a name="presentacion_ELTPARAGRAPHE000065"></a>

En WDMap, los registros se visualizan en un tabla. Por lo tanto, todas las opciones específicas de este tipo de control están disponibles.
<a name="NOTE4_2"></a>


### Búsqueda de registros
<a name="busqueda_registros_ELTPARAGRAPHE000072"></a>

Se puede realizar una búsqueda "Starts with" (Comienza con) sobre un elemento clave.

Las columnas en las que se puede realizar una búsqueda están identificadas por ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeRecherche.gif).

Para buscar registros según el valor de un artículo:

1. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeRecherche.gif) que se encuentra en la cabecera de la columna. Se visualiza un área de entrada.

2. Introduzca el valor del artículo a buscar. Los registros que comienzan con las letras escritas se visualizan dinámicamente.




**Observación**: Este criterio de búsqueda no se guarda. Durante el próximo uso de WDMap, tendrá que redefinir este criterio de búsqueda.
<a name="NOTE4_3"></a>


### Ordenar los registros vistos
<a name="ordenar_los_registros_vistos_ELTPARAGRAPHE000092"></a>

Se puede realizar una clasificación simple en una posición (orden ascendente o descendente).

Por ejemplo, puede clasificar los proveedores según su ciudad.

Las columnas clasificables se identifican con la ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeTri.gif) Icon.

Para clasificar los registros visualizados según un artículo, haga clic en la cabecera de la columna.

**Observación**: Este criterio de clasificación no se graba. Durante el próximo uso de WDMap, tendrá que redefinir este criterio para poder realizar una nueva ordenación.
<a name="NOTE4_4"></a>


### Modificar el orden de visualización de los artículos
<a name="modificar_orden_visualizacion_los_articulos_ELTPARAGRAPHE000109"></a>

Por Default, el orden de visualización corresponde al orden lógico de los artículos. El orden lógico corresponde al orden de visualización definido en la editor de análisis.

WDMap le permite modificar fácilmente el orden de visualización de los elementos.

Por ejemplo, puede organizar las columnas en orden de importancia o agrupar las columnas que contienen el mismo tipo de información.

Para mover una columna:

1. Haga clic en la cabecera de la columna y mantenga pulsado el botón izquierdo del ratón.

2. Mueva la columna a su nueva posición y suelte el botón izquierdo del ratón.




**Observaciones**:

- Este orden de visualización no se graba. Durante el próximo uso de WDMap, el orden utilizado será el orden Default (orden lógico de los artículos).

- La columna "Rec #" no se puede mover (esta columna corresponde al número de los registros de el archivo de datos).



<a name="NOTE4_5"></a>


### Columnas de anclaje a la izquierda de tabla
<a name="columnas_anclaje_izquierda_tabla_ELTPARAGRAPHE000130"></a>

Por Default, para ver todas las columnas, debes usar la barra de desplazamiento horizontal.

WDMap le permite fijar las columnas a la izquierda de la visualización de tabla. Cuando se utiliza la barra de desplazamiento horizontal, sólo se moverán las columnas no fijas.

Por lo tanto, las columnas que contienen la información más importante siempre pueden ser Visible.

Por ejemplo, si la columna "Nombre del cliente" está fija a la izquierda, el nombre del cliente seguirá siendo Visible cuando se utilice la barra de desplazamiento horizontal.

Para arreglar una columna, pase el ratón por la izquierda de la cabecera de la columna a arreglar y haga clic cuando aparezca ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeCadenas.gif).

Para desanclar una columna, pase el ratón por la izquierda de la cabecera de la columna a fijar y haga clic cuando aparezca ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconecadenasFerme.gif).

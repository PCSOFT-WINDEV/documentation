## Presentación
<a name="presentacion_ELTTEXTE000135"></a>
WDStatistic es una herramienta que se utiliza para realizar cálculos estadísticos sobre el tráfico de sus sitios WEBDEV dinámicos (sitios estándar y AWP) y los servicios Web desplegados en un servidor de aplicaciones WEBDEV.

Estas estadísticas se calculan a partir de los ficheros log (extensión".LOG") generados por el servidor de aplicaciones WEBDEV. Estos archivos se pueden guardar para todos los sitios o para un sitio específico.

Algunos ejemplos de estadísticas calculadas automáticamente por WDStatistic:

- Lista de conexiones: detalles de todas las conexiones con el servidor Web (IP Address, fecha y hora, navegador utilizado, sistema operativo, etc.).

- Conexiones por sitio: número de conexiones por sitio WEBDEV y duración.

- Lista de páginas visitadas por un usuario Web.

- Número de conexiones por tipo de navegador (Explorador INTERNET, ...).

- Número de conexiones por franja horaria.

- Número de conexiones por sistema operativo.

- Lista de sitios donde estaban los usuarios antes de llegar a su sitio.

- Lista de palabras clave utilizadas en los motores de búsqueda que permiten a los usuarios de la Web llegar a su sitio.

- Estadísticas sobre el origen geográfico de los visitantes (por Country).




Estas estadísticas se muestran en:

- Tabla de resumen. Estas tablas se pueden guardar en formato Word, Excel o XML (haga clic con el botón derecho del ratón en el tabla).

- Gráfico. Estos gráficos se pueden personalizar (haga clic con el botón derecho del ratón en el gráfico). Tiene la posibilidad de cambiar el tipo de gráfico, añadir leyendas, modificar la fuente utilizada, etc.

- reporte imprimible. Tiene la posibilidad de elegir el tipo de reporte a imprimir así como el periodo solicitado.. Estos informes pueden ser impresos, guardados en formato PDF, enviados por Email, ...




Para iniciar WDStatistic, ejecute "WDStatistic.EXE" directamente.

Estadística WDS **no puede redistribuirse**. WDStatistic no se puede instalar en el servidor Web donde se encuentra desplegado su sitio.

Para utilizar WDStatistic, la versión de desarrollo de WEBDEV debe estar necesariamente instalada en el ordenador actual.

Observación: WDStatistic también está disponible como una herramienta de on-Line, del administrador remoto. Para obtener más información, consulte [Mostrando las estadísticas en el administrador remoto](../WDAdminWeb/6466.md).

## Uso de WDStatistic

### Archivo de registro
<a name="archivo_registro_ELTPARAGRAPHE000069"></a>

Las estadísticas relativas al tráfico de sus sitios dinámicos se calculan a partir de los archivos de registro (extensión".LOG").. Estos archivos son generados por el motor WEBDEV.

Estos archivos de registro se pueden crear:

- ***Para cada uno de los sitios WEBDEV dinámicos*** instalado en el servidor Web. En este caso, cada día se creará un único archivo de registro en un directorio específico de cada sitio.. Este archivo contendrá el historial de las operaciones realizadas en el sitio WEBDEV dinámico correspondiente.

- ***Para todos los sitios WEBDEV dinámicos*** instalado en el servidor Web. En este caso, se creará un único archivo de registro todos los días en un directorio común. Este archivo contendrá el historial de las operaciones realizadas en todos los sitios WEBDEV dinámicos instalados en el servidor Web.




Estos archivos de registro se guardan continuamente tan pronto como se utilizan los sitios associated con el archivo.

El formato para el nombre de estos archivos de registro es el siguiente:  "WDSession_&lt;BackupYear&gt;&lt;BackupMonth&gt;&lt;BackupDay>"
<a name="NOTE2_3"></a>




### Archivos de datos
<a name="archivos_datos_ELTPARAGRAPHE000086"></a>

Los archivos de datos de WDStatistic se encuentran en el directorio "C:\\Documents and Settings\\User\\Application Data\\PC SOFT\\WDSTATISTIC\\".. Estos archivos se pueden transferir de un ordenador a otro.

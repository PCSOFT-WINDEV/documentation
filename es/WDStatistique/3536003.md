
## WDStatistic: Configuración del administrador de WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000112"></a>
Para crear los archivos de registro necesarios para calcular las estadísticas de tráfico del sitio, se debe realizar una configuración específica en el administrador de WEBDEV.

Este ajuste se puede realizar:

- desde el administrador remoto de WEBDEV, accesible desde el ordenador actual (recomendado).

- del administrador de WEBDEV instalado en su empresa de alojamiento (suministrado con el WEBDEV servidor de aplicaciones).




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Configuración del administrador remoto de WEBDEV
<a name="configuracion_del_administrador_remoto_webdev_ELTTEXTE000136"></a>
Para crear los archivos de registro para cada uno de los sitios WEBDEV dinámicos instalados en el servidor Web: 

1. Mostrar la página principal del administrador remoto de WEBDEV.
	Recordatorio: El administrador remoto de WEBDEV puede ser ejecutado desde cualquier ordenador equipado con un navegador INTERNET. Para ello, utilice la siguiente Address: http://&lt;WebServer&gt;/WD28 0AWP/WD28 0AWP.EXE/CONNECT/WDAdminWeb

2. Seleccione el sitio WEBDEV solicitado.

3. Marque "Generar el fichero log".

4. Especifique el directorio donde se deben crear los archivos de registro.




**Observación**: El administrador remoto de WEBDEV no puede utilizarse para configurar la creación de un archivo de registro común para todos los sitios WEBDEV dinámicos instalados en el servidor Web.

## Configuración del administrador de WEBDEV (WEBDEV Application Server)
<a name="configuracion_del_administrador_webdev_webdev_application_server_ELTTEXTE000160"></a>


### Creación de los ficheros log para cada uno de los sitios WEBDEV dinámicos instalados en el servidor Web
<a name="creacion_los_ficheros_log_para_cada_uno_los_sitios_webdev_dinamicos_instalados_servidor_web_ELTPARAGRAPHE000043"></a>

Para crear los archivos de registro para cada uno de los sitios WEBDEV dinámicos instalados en el servidor Web: 

1. Mostrar la pestaña "Sitios" del administrador de WEBDEV (versión utilizada en la servidor de aplicaciones).

2. Seleccione el sitio WEBDEV solicitado.

3. Compruebe los "Parámetros personalizados".

4. Marque "Generar un archivo de registro específico para este sitio para las estadísticas de tráfico (.log)".

5. Especifique el directorio donde se deben crear los archivos de registro.

6. Haga clic en el botón "Aplicar".

7. Repita estas operaciones para cada uno de sus sitios (si es necesario).



<a name="NOTE3_2"></a>


### Creación de los archivos de registro comunes para todos los sitios WEBDEV dinámicos instalados en el servidor Web
<a name="creacion_los_archivos_registro_comunes_para_todos_los_sitios_webdev_dinamicos_instalados_servidor_web_ELTPARAGRAPHE000058"></a>

Para crear los archivos de registro comunes a todos los sitios WEBDEV dinámicos instalados en el servidor Web: 

1. Mostrar la pestaña "Configuración" del administrador de WEBDEV (versión utilizada con la servidor de aplicaciones).

2. Compruebe "Generar un archivo de registro para las estadísticas de tráfico (.log)".

3. Especifique el directorio donde se deben crear los archivos de registro.

4. Haga clic en el botón "Aplicar".

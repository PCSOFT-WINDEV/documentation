


## WDLog: Visualización de un archivo de registro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000173"></a>
WDLog le permite ver el contenido de un archivo de registro:

- en modo HFSQL Classic. El archivo visualizado en WDLog es &lt;NameLoggedFile>JNL.FIC.

- en modo HFSQL Client/Server. Sin embargo, le recomendamos que utilice el HFSQL control Center.




Recordatorio: 

- El modo de funcionamiento del registro Process presenta diferencias entre HFSQL Classic y HFSQL Client/Server.

- En programación, [HHistoryModification](../WDLang4/3044344.md) devuelve las modificaciones realizadas en uno o varios elementos de una determinada Record.

### Ver el archivo JournalOpération.FIC (HFSQL Classic)
<a name="ver_archivo_journaloperationfic_hfsql_classic_ELTPARAGRAPHE000033"></a>

Para ver el archivo JournalOpération.FIC (HFSQL Classic)

1. Inicie el registro WDLog.

2. Seleccione "Archivo... Gestionar los registros" (o haga clic en el libro).

3. Haga clic en "Abrir un registro de HFSQL Classic".

4. Seleccione el fichero log de su aplicación (fichero XXXJNL.FIC).

5. Se muestra el contenido del archivo.

### Visualización del fichero log (HFSQL Client/Server)
<a name="visualizacion_del_fichero_log_hfsql_clientserver_ELTPARAGRAPHE000048"></a>

Para visualizar el fichero Log (HFSQL Client/Server)

1. Inicie el registro WDLog.

2. Seleccione "Archivo... Gestionar los registros" (o haga clic en el libro).

3. Haga clic en "Abrir un registro de HFSQL Client/Server".

4. Describa la conexión con el servidor HFSQL en el asistente.

5. Seleccione el archivo registrado.

6. Se muestra el contenido del archivo.

### La información mostrada
<a name="informacion_mostrada_ELTPARAGRAPHE000062"></a>


|   |   |
| --- | --- |
| Aplicación | Nombre de la aplicación que ha realizado la operación (useful cuando los archivos se comparten entre varias aplicaciones).<br>Esta información es una cadena de 12 caracteres. |
| Clave | Valor de la primera clave única del fichero. |
| Dirección IP | IP Address de la computadora que realizó la operación. |
| HComputer | Identificador de el equipo que realizó la modificación (identificador definido por ***hComputer***). <br>Esta información es una cadena de 16 caracteres. |
| momento | Hora en que se realizó la operación. |
| Nombre de la red | Nombre de el equipo (hasta 12 caracteres). Si este nombre contiene más de 12 caracteres, el comienzo del nombre se trunca (de hecho, el final del nombre de el equipo es la sección más determinante).<br>![HFSQL Client/Server](https://doc.pcsoft.fr/ext/images/us/HFCS.png) El archivo JNLUSER.Fic permite conocer el nombre del ordenador del usuario. |
| Notas | Notas personalizadas definidas por la función ***HLogInfo***.<br>Esta información es una cadena de 12 caracteres. |
| Operación | Operación realizada en el archivo de datos. En el archivo de datos, esta operación se identifica con un número. La correspondencia "Número - Función HFSQL" está disponible en el archivo ListeDéfinitionHF.WL ubicado en el subdirectorio \\Personal\\Extern de WINDEV. |
| salir con | Fecha en la que se realizó la operación. |
| Valores | La lupa se utiliza para ver los diferentes valores de la Record:<br><br>- Durante una adición, se visualiza el valor añadido.<br><br>- Durante una modificación, se visualizan los valores antes y después de la modificación.<br><br>- Durante un borrado, se visualizan los valores borrados.<br><br><br> |

### Posibles operaciones
<a name="posibles_operaciones_ELTPARAGRAPHE000124"></a>

Esta ventana se utiliza para realizar búsquedas avanzadas en todos los elementos guardados en el registro.. Por ejemplo, puede buscar todas las modificaciones realizadas por un ordenador durante un período específico o buscar todas las operaciones realizadas por un ordenador específico.

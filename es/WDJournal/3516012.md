## Presentación
<a name="presentacion_ELTTEXTE000232"></a>


### WDLog
<a name="wdlog_ELTPARAGRAPHE000011"></a>

Las diferentes operaciones que se pueden realizar con WDLog son las siguientes:

- [Guardar archivos de datos HFSQL relacionados con una aplicación WINDEV, WEBDEV o WINDEV Mobile](../WDJournal/3516006.md)
	Crea una copia de seguridad de los archivos de datos. Si se produce un problema, se pueden restaurar los archivos de datos.

- [Restaurar archivos de datos guardados previamente](../WDJournal/3516009.md)
	Restaura una copia de seguridad de un archivo de datos HFSQL realizada previamente con WDLog.

- [Actualizar archivos de datos mediante un archivo de registro](../WDJournal/3516001.md)
	Aplica todas las operaciones de un archivo de registro a un archivo de datos.

- [Ver archivos de registro](../WDJournal/3516007.md)
	Muestra la información guardada en un archivo de registro. También puede buscar información específica en el archivo de registro.




**Observación**: Cuando se modifica el análisis, el archivo de registro se guarda automáticamente al realizar la modificación automática de los archivos de datos.. El archivo de registro se vaciará más tarde.

## Uso
<a name="uso_ELTTEXTE000262"></a>


### Iniciar WDLog
<a name="iniciar_wdlog_ELTPARAGRAPHE000041"></a>

Para iniciar WDLog:

- En modo interactivo: 

	- desde WINDEV, WEBDEV o WINDEV Mobile: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDTool" y seleccione "WDTool". 

	- ejecute el programa "WDLOG.EXE" ubicado en el subdirectorio "Programs" del directorio de instalación de WINDEV, WEBDEV y WINDEV Mobile.




- Desde una línea de comandos.




En modo interactivo, las diferentes opciones del menú desplegable permiten acceder a todas las funcionalidades de WDLog.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=wdjournal.gif)

<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000069"></a>

WDLog puede ser redistribuido. WDLog puede instalarse con las aplicaciones desarrolladas en WINDEV y/o WEBDEV. Los términos de licencia de WINDEV, WEBDEV o WINDEV Mobile se aplican plenamente.

WDLog se puede incluir en las aplicaciones al crear el procedimiento de instalación.

**Recordatorio**: El procedimiento de instalación se puede crear: 

- mediante el asistente. Para abrir el asistente, vaya a la pestaña "Proyecto", grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación". 

- desde el editor de instalación. Para abrir el editor de instalación, en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".

## Recordatorio: Registro de los archivos de datos
<a name="recordatorio_registro_los_archivos_datos_ELTTEXTE000298"></a>
WINDEV, WEBDEV y WINDEV Mobile permiten crear un archivo de registro para cada archivo de datos. Este archivo de registro puede contener:

- Operaciones de escritura

- Operaciones de lectura y escritura




Los archivos de registro contienen el historial de las operaciones realizadas en los archivos de datos:

- Los valores de los campos del registro antes de la modificación del archivo de datos,

- Los valores de los campos del registro después de la modificación del archivo de datos,

- El autor de la operación,

- La fecha de la operación,

- El equipo en el que se realiza la operación,

- Comentarios,

- La dirección IP del equipo en el que se realiza la operación,

- El nombre de la aplicación, 

- El nombre de red del usuario,

- El tipo de operación (agregar, modificar, eliminar, leer, etc.).




Los registros de los archivos de datos se crean en el editor de análisis de WINDEV, WEBDEV o WINDEV Mobile.

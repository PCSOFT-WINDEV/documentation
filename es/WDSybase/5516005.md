
## Conector Sybase nativo para WINDEV y WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000130"></a>
WINDEV y WEBDEV proponen un Conector Nativo opcional para Sybase (también llamado Native Access to Sybase). Esta Conector Nativo le permite acceder a una base de datos Sybase desde un programa WLanguage sin utilizar ningún controlador externo.

Esta versión también se utiliza para:

- Ejecute la misma consulta varias veces.

- Recuperar el valor de salida de una ejecución de Procedure almacenada en el servidor




Estas funciones están disponibles en la versión 12.5 de Sybase Adaptive Server Enterprise (ASE).

**Atención**: Esta versión del Conector Nativo Sybase funciona con WINDEV 28 y/o WEBDEV 28.





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000154"></a>


### Condiciones para utilizar el conector Sybase nativo
<a name="condiciones_para_utilizar_conector_sybase_nativo_ELTPARAGRAPHE000032"></a>

- Sybase Adaptive Server Enterprise (ASE) versión 10 o posterior (versión 12.5 para recuperar el resultado de una Procedure almacenada).

- Capa de cliente de Sybase instalada.

- El ***LIBSYBDB.DLL*** DLL debe ser accesible:

	- del ordenador de desarrollo.

	- de la computadora implementación.






<a name="NOTE3_2"></a>


### Los archivos necesarios
<a name="los_archivos_necesarios_ELTPARAGRAPHE000044"></a>

Los siguientes archivos son necesarios para utilizar el conector Sybase nativo:

- los ***LIBSYBDB.DLL*** DLL. Esta DLL es instalada por la capa de cliente de Sybase. Consulte el contrato de licencia de Sybase para obtener más detalles sobre las condiciones de distribución de esta DLL.

- ***wd28 0syb.dll***: Requerido para la Conector Nativo a una base de datos Sybase. Este archivo se suministra con el conector nativo WINDEV/WEBDEV para Sybase.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000184"></a>


### Uso del conector Sybase nativo en sus aplicaciones WINDEV y sitios WEBDEV
<a name="uso_del_conector_sybase_nativo_sus_aplicaciones_windev_sitios_webdev_ELTPARAGRAPHE000058"></a>

Para utilizar Sybase en modo nativo, debe programar:

- con las funciones "SQL" de WLanguage (véase [Programación con funciones SQL](../WDSybase/5516001.md)).

- con las funciones "HFSQL" de WLanguage (véase [Programación con funciones HFSQL](../WDSybase/5516006.md)).




**Observaciones**: 

- No se olvide de leer el documento [limitaciones con respecto al uso del conector Sybase nativo](../WDSybase/5516004.md).

- El Conector Nativo SQL Sybase tiene en cuenta la opción "Crear automáticamente los archivos de datos cuando se abran" (disponible en la pestaña "Archivos" de la descripción del proyecto).


## MyPage (Palabra reservada)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **MyPage** permite manipular la página actual. Al ejecutar la página, la palabra clave **MyPage** se reemplaza por la página y no por el nombre de la página. Así, la palabra clave **MyPage** puede utilizarse como una página.

Ventaja: La palabra clave **MyPage** permite que un código local (control, botón, etc.) o un código global (procedimiento global, clase, etc.) se vuelva independiente de la página actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the title of the current page
PageTitle = MyPage..Title
```


<a name="Example2"></a>



```wl
// Button for changing the language
Nation(nationEnglish)
PageDisplay(MyPage)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MyPage`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Manejo de la página actual
<a name="manejo_pagina_actual_ELTPARAGRAPHE000036"></a>

- La palabra clave **MyPage** siempre se reemplaza por la página actual.

- La palabra clave **MyPage** solo puede utilizarse en los procesos que manipulan la página actual (por ejemplo, en los procesos asociados a un control, a una página, etc.). **MyPage** no se puede utilizar en un reporte.

- La palabra clave **MyPage** solo puede utilizarse desde un proceso o procedimiento de la página.



<a name="NOTE0_2"></a>




### Usando MyPage en un Procedure
<a name="usando_mypage_procedure_ELTPARAGRAPHE000046"></a>

La palabra clave **MyPage** puede utilizarse en un procedimiento (local o global) solo si el procedimiento manipula la página actual. En este caso, la palabra clave **MyPage** hace referencia a la página actual.

Para un Procedure local, **MyPage** corresponde a la página a la que pertenece el Procedure.
<a name="NOTE0_3"></a>




### Funciones de WLanguage y página actual
<a name="funciones_wlanguage_pagina_actual_ELTPARAGRAPHE000056"></a>

Para especificar la página actual en las funciones de WLanguage que aceptan un nombre de página como parámetro, utilice

- una cadena vacía,

- **MyPage**.

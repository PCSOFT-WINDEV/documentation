


## MyChildWindow (Palabra reservada)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**MyChildWindow** se utiliza para manipular la ventana secundaria de la ventana actual.. Cuando se ejecuta la ventana, **MiInfantilVentana** es reemplazado por la ventana (y no por el nombre de la ventana).

**Ventaja**: **MyChildWindow** sólo se puede usar en el Event "Cerrar una ventana para niños". Esta palabra clave se utiliza para manipular en este código las características de la ventana secundaria que se cerró. Esto le permite recuperar: 

- el nombre de la ventana hija que fue cerrada por la [denominar](../Proprietes/2510082.md) property. 

- el valor devuelto por la ventana hija a través de la [ReturnedValue](../Proprietes/2510134.md) property. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Event "Closing a child window of WIN_List_Customers"
// -------------------------------------------------------------
// Refreshes the list of customers
LooperDisplay(LOOP_Customers, taReExecuteQuery)

// For any return (except for the deletion)
IF MyChildWindow.ReturnedValue <> -1 THEN
	nIndex is int
	// Finds the customer row
	nIndex = LooperSearch(ATT_CustomerNum, Customer.CustomerNum)
	IF nIndex > 0 THEN
		// Positions on the customer row
		LooperPosition(LOOP_Customers, nIndex)
	END
END
```

## Observaciones


### Descubrir el valor devuelto por una ventana secundaria
<a name="descubrir_valor_devuelto_por_una_ventana_secundaria_ELTPARAGRAPHE000052"></a>

En las aplicaciones móviles, las ventanas se abren con [OpenMobileWindow](../WDLang1/1000021018.md) (o [OpenChild](../WDLang1/3038021.md)) en lugar de [entablar](../WDLang1/3038035.md). 

**MiVentanaInfantil** permite obtener el nombre de la ventana hija cerrada, así como el valor que ha devuelto, cuando se utiliza en el "Cierre de una ventana hija" Event de la ventana padre (la ventana en la que se llamó a [OpenMobileWindow](../WDLang1/1000021018.md) o [OpenChild](../WDLang1/3038021.md)).

Por ejemplo:

- En el código de cierre de la ventana hija, se utiliza [Cerrar](../WDLang1/3038018.md) para devolver un parámetro: 
	
	```wl
	Close("", MyParameter)
	```


- El nombre de la ventana infantil así como el valor devuelto puede ser recuperado en la Event "Cerrar una ventana infantil" de la ventana de llamada: 
	
	```wl
	// Close a child window
	NameChildWindow = MyChildWindow.Name
	ValueChildWindow = MyChildWindow.ReturnedValue
	```

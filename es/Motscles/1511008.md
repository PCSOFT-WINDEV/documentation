
## Designación del elemento de corriente (Palabra reservada)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las palabras clave utilizadas para identificar el elemento actual son:


|   |   |
| --- | --- |
| [MiVentana](../Motscles/1511004.md) | Le permite manipular la ventana actual. |
| [Ventana MyRightSliding](../Motscles/1511023.md) | Le permite a manipular la ventana corrediza derecha associated con la ventana actual. |
| [MyLeftSlidingWindow](../Motscles/1511022.md) | Le permite a manipular la ventana corrediza izquierda associated con la ventana actual. |
| [MiVentanaInterna](../Motscles/1511024.md) | Le permite manipular la ventana interna actual. |
| [MyPage](../Motscles/1511017.md) | Le permite manipular la página actual |
| [MiFuente](../Motscles/1511001.md) | Le permite manipular la fuente de datos actual |
| [MySelf](../Motscles/1511007.md) | Le permite a manipular el actual control |
| [MiPopupControl](../Motscles/1511005.md) | Le permite a manipular el control que abrió una ventana emergente |
| [MiInforme](../Motscles/1511010.md) | Le permite manipular el reporte actual |
| [MiArchivo](../Motscles/1511019.md) | Le permite manipular el archivo que liberó el trigger del código de trigger. |
| [MiPadre](../Motscles/1511016.md) | Le permite manipular el supercontrol actual desde uno de sus controles. |
| [Objeto](../POO/6010002.md) / [Este](../POO/6010002.md) | Le permite a manipular la clase actual. |
| MiHilo | Permite manipular el objeto thread que ejecuta el código actual. |

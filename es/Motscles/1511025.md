
## Lista de atributos de extension
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los atributos de extensión son declaraciones especiales de WLanguage que permiten añadir propiedades o funcionalidades específicas a las variables o procedimientos. Estos atributos extension ayudan a simplificar el código y facilitan el mantenimiento de sus aplicaciones.

WINDEV, WEBDEV y WINDEV Mobile admiten varios atributos extension. Esta Page presenta los diferentes atributos de extensión disponibles, agrupados por temas.

Las categorías de atributos de extension son: 

- [Atributos generales de extension](#NOTE2_1),

- [Atributos extension específicos de OOP y MVP](#NOTE3_1),

- [Atributos extension específicos de las variables](#NOTE4_1),

- [Atributos de extensión específicos de los procedimientos locales, globales o internos](#NOTE5_1).

- [Atributos extension para gestionar temporizadores](#NOTE6_1). 

- [Atributos extension para gestionar los hilos](#NOTE7_1).




Para simplificar la entrada de atributos de extensión, el editor de código propone automáticamente la lista de atributos de extensión disponibles cuando se presiona la tecla "&lt;". Para obtener más información, consulte [Autocompletado](../Editeurs/2013001.md).

## Observación
<a name="observacion_ELTTEXTE000871"></a>
Puede combinar varios atributos extension utilizando la sintaxis, según el tipo de declaración y el objeto utilizado: 
```txt
Declaration of my object <Name Attribute 1=value1, Name Attribute 2=value2, ...>
```

Por ejemplo, para una declaración de un tampón Variable: 
```txt
Class
	m_buffer is Buffer <serialize=false,mapping="ImageMemo">
END
```


## Font (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una variable de tipo **Font** permite crear una fuente mediante programación. Las características del tipo de letra pueden establecerse mediante diferentes propiedades WLanguage.

Una variable de tipo **Font** puede utilizarse: 

- en dibujos,

- en gráficos,

- en trabajos de impresión,

- en los controles de una ventana (propiedad [Font](../Motscles/1514045.md)), 

- en los controles de una página (propiedad [Font](../Motscles/1514045.md)),

- en los controles de un reporte (propiedad [Font](../Motscles/1514045.md)).




**Observaciones**: 

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para cambiar las características de una celda en un control Hoja de cálculo, utilice la función [SpreadsheetStyleSelection](../WDLang1/1000021338.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare the Font variables
MyFont is Font
MyFont1, MyFont2 are Font
```


<a name="Example2"></a>



```wl
// Variable initialized with the font created by FontCreate
MyFont = FontCreate("Arial", 12, iBold)
```


<a name="Example3"></a>



```wl
// Modify the characteristics of the font
MyFont.Name = "Arial"
MyFont.Size = 12
MyFont.Bold = True
```


<a name="Example4"></a>





```wl
// Use a font on a window control
EDT_LastName.Font = MyFont

// Retrieve the font of a control
MyFont1 = EDT_FirstName.Font
```

## Observaciones




### Características de las fuentes
<a name="caracteristicas_las_fuentes_ELTPARAGRAPHE000109"></a>

De forma predeterminada, una variable de tipo Font tiene las siguientes características:

- "Arial", 

- tamaño 12,

- color negro,

- sin atributos.




Para establecer las características de una variable de tipo Font mediante programación, utilice:

- La función [FontCreate](../WDLang1/3060002.md).

- Las propiedades de gestión de fuentes (véase más abajo)

- La función [FontSelect](../WDLang1/3060004.md), que abre el cuadro de diálogo de selección de fuentes de Windows.



<a name="NOTE0_2"></a>




### Propiedades de gestión de fuentes
<a name="propiedades_gestion_fuentes_ELTPARAGRAPHE000134"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **Font**:

| Nombre de la propiedad | Efecto |
| --- | --- |
| [Bold](../Proprietes/2513001.md) | Comprueba si una variable de tipo Font está en negrita y cambia el atributo de negrita de una variable de tipo Font. |
| [CharacterSpacing](../Proprietes/1410088367.md) | Obtiene y establece el espaciado entre caracteres de una variable de tipo Font. |
| [Charset](../Proprietes/2512008.md) | Devuelve y cambia el conjunto de caracteres utilizado por una variable de tipo Font.<br><br> |
| [Color](../Proprietes/2510071.md) | Comprueba y cambia el color de una variable de tipo Font. |
| [Italic](../Proprietes/2513007.md) | Comprueba si una variable de tipo Font está en cursiva y cambia el atributo de cursiva de una variable de tipo Font. |
| [LineSpacing](../Proprietes/1410088366.md) | Obtiene y establece el interlineado de una variable de tipo Font.  |
| [Name](../Proprietes/2510082.md) | Devuelve y cambia el nombre de una variable de tipo Font. |
| [Orientation](../Proprietes/1410086632.md) | Devuelve y cambia el ángulo (en grados) de una variable de tipo Font.<br><br> |
| [Size](../Proprietes/2510125.md) | Devuelve y cambia el tamaño (en picas) de una variable de tipo Font. |
| [SizeUnit](../Proprietes/1410088365.md) | Obtiene y establece la unidad de tamaño utilizada para una variable de tipo Font. |
| [StrikeOut](../Proprietes/2513005.md) | Comprueba si una variable de tipo Font está tachada y tacha una variable de tipo Font. |
| [Underline](../Proprietes/2513013.md) | Comprueba si una variable de tipo Font está subrayada y subraya una variable de tipo Font. |

Estas propiedades se pueden utilizar directamente en la variable de tipo **Font**. Ejemplo:


```wl
MyFont is Font
MyFont.Bold = True
MyFont.Color = LightRed
```


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Solo se pueden utilizar las siguientes propiedades con el tipo **Font**: [Orientation](../Proprietes/1410086632.md), [StrikeOut](../Proprietes/2513005.md), [Color](../Proprietes/2510071.md), [Bold](../Proprietes/2513001.md), [Italic](../Proprietes/2513007.md), [Name](../Proprietes/2510082.md), [Underline](../Proprietes/2513013.md), [Size](../Proprietes/2510125.md).

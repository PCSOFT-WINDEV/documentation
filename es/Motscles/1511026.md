
## BLOQUE

*** ***
				

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **&lt;BLOCK&gt;** se utiliza en el editor de código para declarar bloques de código personalizados.. Estos bloques de código se utilizan para resaltar los diferentes pasos de un algoritmo y para agrupar las declaraciones de variables o miembros.. 

El bloque de código definido se puede expandir o contraer. 

Un comment puede ser associated con el bloque. Esta comment siempre se mostrará en modo colapsado. 

Esta palabra clave no influye en el tiempo de ejecución. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
<BLOCK Structures for control serialization>

ControlObject is Structure
	Name is string
	Type is int
	Properties is associative array of Variant
END

MemoryControlObject is Structure
	NumberOccurrence is int
	ListObjects is array of ControlObject
END

<END>
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Declarar un bloque de código

`<BLOCK>
   <Code to run>
<END>
`
---

**`<Code to run>: ()`**

Código de idioma para ejecutar associated con el bloque.  


<a name="SYNTAX2"></a>

### Declarar un bloque de código mientras se muestra un comment en modo colapsado

`<BLOCK Comment always displayed>
   <Code to run>
<END>
`
---

**`<Code to run>: ()`**

Código de idioma para ejecutar associated con el bloque.

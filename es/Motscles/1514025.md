
## array fijo (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una array fija es un tipo "avanzado" de array: las dimensiones de esta array se definen durante la compilación y no pueden ser modificadas.

Las dimensiones de una array fija se definen durante la compilación, sólo si las dimensiones de esta array corresponden a:

- una integridad,

- una constante que fue creada de antemano.




De lo contrario, se producirá un error de WLanguage durante la compilación del proyecto.

**Recordatorio**: Un array es un tipo estructurado que permite agrupar una serie de elementos del mismo tipo. Se puede acceder a cada elemento de la array por su índice.

Se recomienda su uso:

- Un **fijo array** para pasar un array como parámetro a las funciones API de Windows.

- Un [array dinámica](../Motscles/1514057.md) o un ["simple" array](../Motscles/1514030.md) cuando el tamaño del array debe ser modificado durante la ejecución del programa.

- Una [asociativa array](../Motscles/1514058.md) para almacenar los elementos indexados en cualquier tipo de información.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare a fixed array
CustomerArray is fixed array of 5 by 7 by 3 int
// Equivalent to: CustomerArray is fixed array of 5,7,3 int
```


<a name="Example2"></a>



```wl
// Refer to a fixed array
CustomerArray[2,5,3] = 47
// Equivalent to: CustomerArray[2][5][3] = 47
```

## Observaciones




### Pasar una array fija como parámetro a una Procedure
<a name="pasar_una_array_fija_como_parametro_una_procedure_ELTPARAGRAPHE000135"></a>

Se puede pasar una array fija como parámetro a una Procedure. Para ello, utilice la siguiente sintaxis:


```txt
<Procedure name>(<Array name>)
```


Por ejemplo:


```wl
SupplierArray is fixed array of 10 by 50 string
// Call to the DisplayArray procedure
DisplayArray(SupplierArray)
```

<a name="NOTE0_2"></a>




### Declarar un miembro fijo de array
<a name="declarar_miembro_fijo_array_ELTPARAGRAPHE000149"></a>

Un miembro "fijo array" puede ser declarado en:

- ***Una clase***. Esta array fija se asigna directamente en la zona de memoria de esta clase.

- ***A compuesto Variable***. Esta array fija se asigna directamente en la zona de memoria de esta Variable compuesta.

- ***Una estructura*** &lt;Nombre de la estructura&gt;. Esta array fija se asigna directamente en la zona de memoria de cada &lt;nombre de la estructura&gt; Variable.




Por ejemplo:


```wl
Struct is structure
	n1 is int
	nArray is fixed array of 2 int
	n2 is int
END
MyStructure is Struct
```


Representación de la zona de memoria de "MiEstructura":
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ZoneMemoireStruct.gif)


Esta representación de memoria es compatible con las APIs de Windows. Por lo tanto, un array de tamaño fijo puede ser transmitido a una función de las APIs de Windows.

### Dimensión de una array fija
<a name="dimension_una_array_fija_ELTPARAGRAPHE000171"></a>

La función [Dimensión](../WDLang1/3013022.md) y la propiedad [Conde](../Proprietes/2510097.md) se utilizan para obtener el número de elementos en un array fijo.
**Recordatorio**: Una array fija no puede redimensionarse.

### Limitaciones: Elementos de array fija
<a name="limitaciones_elementos_array_fija_ELTPARAGRAPHE000200"></a>

- Un array fijo puede incluir clases sólo si estas clases tienen un constructor sin parámetro (o con parámetros opcionales).




- Un array fijo no puede estar hecho de variables compuestas.

- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Una array fija no puede ser hecha de arrays.




- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No se puede crear arrays fija de clases o estructuras.

- El tamaño de una array fija no puede exceder los 2GB.

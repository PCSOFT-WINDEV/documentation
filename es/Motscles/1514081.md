
## Parámetros útiles de una Procedure
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Una Procedure puede declarar algunos parámetros que no se utilizan inmediatamente. En este caso, aparece una advertencia en el editor de códigos, indicando que la Variable local no se utiliza. 

Para evitar esta advertencia, el nombre Variable debe ser seguido por un atributo de extensión específico. Se debe utilizar la siguiente sintaxis: 


```txt
PROCEDURE <Procedure name>(<Parameter 1> <useful[="Reason"]>, ...
	<Optional Parameter 1> = <Value> <useful [="Reason"]>,
	<Optional Parameter 2> = <Value> <useful [="Reason"]>, ...)
```


donde "Reason" corresponde a una explicación dada por el desarrollador. 

Por ejemplo:


```wl
// -- Declaration of Drawing procedure
PROCEDURE Drawing(coorLine <useful = "line coordinate">, coorColumn <useful = "column coordinate">)
...
```

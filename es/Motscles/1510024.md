
## PARA CADA UNO/PARA TODOS (búsqueda de combinaciones)

*** ***
				

<a name="description"></a>
La sentencia **FOR EACH** se utiliza para navegar por los valores disponibles en las combinaciones.  

**Observación**: Se aceptan las frases **PARA TODOS, PARA CADA UNO**.. En esta documentación se utilizará la sentencia **FOR EACH** , pero puede sustituirse por **FOR ALL**.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declaration code
// Declare a combination
CombinationType is Combination
	CreateFile
	ReadFile
	WriteFile
	DeleteFile
END
```

```wl
// Syntax 1: Browse the combination options
FOR EACH eOption OF CombinationType
	Trace(eOption..Name)
END
```

```wl
// Assign the variable
FileAction = ReadFile + WriteFile

// Syntax 2: Browse the active options
FOR EACH eOption OF FileAction
	Trace(eOption..Name)
END
```

## Sintaxis

### Sintaxis simplificada

`FOR EACH <MyOption> OF <Combination>
    	 ...
END
`
---

**`<FOR EACH>: ()`**

Marca el inicio del bloque de instrucciones.

**`<MyOption>: ()`**

[Option Description](../WDLang1/1000023072.md) variable correspondiente a la opción de combinación. No hay necesidad de declarar este Variable. 

**`<Combination>: ()`**



- [Combinación](../Motscles/1514074.md) que contiene los valores a explorar. 

- [Combinación](../Motscles/1514074.md) variable que contiene las opciones para navegar. Se navega por todas las opciones (activas o inactivas). 





<a name="SYNTAX2"></a>

### Navegando por las opciones activas de un Variable de combinación

`FOR EACH <MyOption> OF <Combination>
    	 ...
END
`
---

**`<FOR EACH>: ()`**

Marca el inicio del bloque de instrucciones.

**`<MyOption>: ()`**

[Option Description](../WDLang1/1000023072.md) variable correspondiente a la opción de combinación. No hay necesidad de declarar este Variable. 

**`<Combination>: ()`**

[Combinación](../Motscles/1514074.md) variable que contiene las opciones para navegar. Sólo se navega por las opciones de Active.

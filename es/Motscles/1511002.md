
## Modulo( Palabra reservada)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el resto de una división.

Observación: **Modulo** puede utilizarse como palabra clave o como función de WLanguage.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the remainder of division
DivisionRemainder = Modulo(21, 4) // DivisionRemainder is set to 1
// Equivalent to: DivisionRemainder = 21 Modulo 4
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Función de módulo

`<Result> = Modulo(<Dividend> , <Divisor>)`
---

**`<Result>: (Integro)`**

Resto de la división.

**`<Dividend>: (Integro)`**

Dividendo de la división.

**`<Divisor>: (Integro)`**

Divisor de división. Se produce un error de WLanguage si este parámetro es set a 0.


<a name="SYNTAX2"></a>

### Palabra clave del módulo

`<Division Remainder> = <Dividend> Modulo <Divisor>`
---

**`<Division Remainder>: (Integro)`**

Resto de la división.

**`<Dividend>: (Integro)`**

Dividendo de la división.

**`<Divisor>: (Integro)`**

Divisor de división. Se produce un error de WLanguage si este parámetro es set a 0.

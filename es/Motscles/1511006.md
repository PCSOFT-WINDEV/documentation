
## Lista de palabras reservadas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las palabras reservadas de WLanguage son las siguientes:


|   |   |
| --- | --- |
| [nuevo](../Motscles/1511011.md) | Permite asignar espacio de memoria para un array dinámico, un objeto de clase dinámico o una estructura dinámica. |
| [BLOQUE](../Motscles/1511026.md) | Permite definir un bloque de código en el editor de código. |
| [EXTERN](../Motscles/1511013.md) | Se utiliza para incluir un archivo de texto que contiene comandos de WLanguage en una aplicación o para declarar un objeto externo. |
| [Eliminar](../Motscles/1511012.md) | Permite liberar explícitamente el espacio de memoria ocupado por un array dinámico, un objeto de clase dinámico o una estructura dinámica. |
| [MiVentana](../Motscles/1511004.md) | Le permite manipular la ventana actual. |
| [Ventana MyRightSliding](../Motscles/1511023.md) | Le permite a manipular la ventana corrediza derecha associated con la ventana actual. |
| [MyLeftSlidingWindow](../Motscles/1511022.md) | Le permite a manipular la ventana corrediza izquierda associated con la ventana actual. |
| [MiInfantilVentana](../Motscles/1511021.md) | Le permite manipular la ventana secundaria de la ventana actual. |
| [MiVentanaInterna](../Motscles/1511024.md) | Le permite manipular la ventana interna actual. |
| [MyPage](../Motscles/1511017.md) | Le permite manipular la página actual. |
| [MiFuente](../Motscles/1511001.md) | Le permite manipular la fuente de datos actual (archivo, vista o consulta). |
| [MyParameters](../Motscles/1511018.md) | Permite a manipular los parámetros de un WLanguage Procedure (Procedure estándar o Procedure con un número de parámetros Variable) |
| [Módulo](../Motscles/1511002.md) | Devuelve el resto de una división. |
| [MySelf](../Motscles/1511007.md) | Le permite a manipular el actual control. |
| [MiInforme](../Proprietes/2510086.md) | Le permite manipular el reporte actual |
| [MiPadre](../Motscles/1511016.md) | Le permite manipular el supercontrol actual desde uno de sus controles. |
| [Null](../Proprietes/2512064.md) | Permite:<br><br>- especificar que un parámetro debe ser ignorado en una consulta: Null y las consultas<br><br>- especifican que una variante de Variable no contiene ningún valor: Nulo y las variantes<br><br>- comparar un valor con 0: Nulo y los números<br><br>- en algunas funciones de WLanguage, especifique que el parámetro debe ser ignorado: Null y las funciones WLanguage<br><br><br> |
| [PARAR](../Motscles/1511014.md) | Se usa para llamar al depurador de WLanguage cuando se ejecuta una prueba en el editor.. Una vez llamada esta palabra clave, la prueba actual se ejecuta en el depurador. |


A esta lista hay que añadir: 

- tipos de variables, 

- las diferentes funciones de WLanguage,

- las diferentes propiedades de WLanguage.

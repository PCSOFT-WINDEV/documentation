### Uso de cadenas
<a name="uso_cadenas_ELTPARAGRAPHE000579"></a>



- Cadenas de sólo lectura en el parámetro de entrada
	


|   |   |   |
| --- | --- | --- |
| **Tipo C** | **C cómo** | **Tipo WLanguage** |
| char\* | cadena ANSI de sólo lectura | entero del sistema (que contiene la Address de una cadena ANSI: &string[ NSI]) |
| achar\* | cadena UNICODE de sólo lectura | entero del sistema (que contiene la Address de una cadena Unicode: &string unicode) |






- Lectura/escritura de cadenas en los parámetros de entrada
	


|   |   |   |
| --- | --- | --- |
| **Tipo C** | **C cómo** | **Tipo WLanguage** |
| char\* | leer/escribir una cadena ANSI | entero del sistema (que contiene la Address de una cadena Ansi fija: &string[ ANSI] en &lt;Longitud Máxima...) |
| achar\* | leer/escribir la cadena UNICODE | entero del sistema (que contiene la dirección de una cadena Unicode fija: &string unicode en &lt;Max Length&gt;) |



- Strings en el valor de retorno
	


|   |   |   |   |
| --- | --- | --- | --- |
| **Tipo C** | **C cómo** | **Tipo WLanguage** | **WLanguage cómo** |
| char\* | Cadena ANSI | system integer | utilizar [StringRetrieve](../WDLang1/3024005.md)(system integer, *srASCIIZAddress*) |
| achar\* | UNICODE string | system integer | utilizar [StringRetrieve](../WDLang1/3024005.md)(system integer, *srUNICODEAddress*) |



- Cadenas en el parámetro de salida
	


|   |   |   |   |
| --- | --- | --- | --- |
| **Tipo C** | **C cómo** | **Tipo WLanguage** | **WLanguage cómo** |
| char\*\* | Cadena ANSI | system integer | dirección de otro system integer que contiene la dirección que se utilizará con [StringRetrieve](../WDLang1/3024005.md)(system integer, *srASCIIZAddress*) |
| achar\*\* | UNICODE string | system integer | dirección de otro system integer que contiene la dirección que se utilizará con [StringRetrieve](../WDLang1/3024005.md)(system integer, *srUNICODEAddress*) |

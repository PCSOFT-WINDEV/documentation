
## DateTime (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **DateTime** permite manipular fácilmente una fecha y una hora. El tipo DateTime se puede utilizar para realizar cálculos (resta, suma,...) de las fechas, horas y duraciones.. Para obtener más información, consulte [Operaciones que se pueden realizar en el tipo DateTime](../Motscles/1514040.md).

**Observaciones**:

- Se pueden utilizar varias propiedades con el tipo **DateTime**, para obtener la fecha o la hora, una parte de la fecha o una parte de la hora.

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartDate is DateTime = "200112311524"     // 31/12/2001 at 15:24
```

## Observaciones




### Declaración e inicialización
<a name="declaracion_inicializacion_ELTPARAGRAPHE000072"></a>

- Un **DateTime** Variable puede ser inicializado con un valor del siguiente tipo:

	- **Date**: las horas y los minutos se inicializarán a 0.

	- **DateTime**

	- **Cadena de caracteres**en el siguiente formato

		- AAAAMMDD

		- AAAAMMDDHH

		- AAAAMMDDHHmm

		- AAAAMMDDHHmmSS

		- AAAAMMDDHHmmSSCC

		- AAAAMMDDHHmmSSCCC
						(AAAA: año, MM: mes, DD: Día, HH: hora, mm: minutos, SS: segundos, CC: centésimas de segundo, CCC: milésimas de segundo)




- Por defecto, una variable de tipo **DateTime** se inicializa con la fecha del día (fecha del sistema) y la hora del sistema. Para definir el valor predeterminado asignado a las variables de tipo [salir con](../Motscles/1514066.md), [momento](../Motscles/1514068.md) y **DateTime**, utilice la función [DateTimeByDefault](../WDLang1/3027015.md).




Observación: También tiene la posibilidad de inicializar un **DateTime** Variable con la fecha y hora del sistema utilizando [Now](../WDLang1/3027024.md).

```wl
StartDate is DateTime = "200112011530"
// Adds 5 years to the date
StartDate.Year += 5
// Modifies the month
StartDate.Month = 5
// Displays the date
Info("Date " + StartDate<§Date)
// Displays the time
Info("Time " + StartDate<§Time)
```

### interval de datos
<a name="interval_datos_ELTPARAGRAPHE000165"></a>

El **FechaHora** se utiliza para gestionar los datos incluidos entre el 01/01/0001 a las 00:00 y el 31/12/9999 a las 23:59.




## Duración (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Duration** se utiliza para manipular fácilmente los intervalos de fechas y horas. El tipo DateTime se puede utilizar para realizar cálculos (resta, suma,...). Para obtener más información, consulte [Operaciones que se pueden realizar en la clase Duración](../Motscles/1514040.md).

**Observaciones**:

- Se pueden utilizar varias propiedades con el **Duración** para recuperar el número de días o minutos, por ejemplo.

- Para asignar una duración procedente del resultado de una función, utilice [StringToDuration](../WDLang1/3027002.md).

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartTime is Time = "1330"
EndTime is Time = "1730"     
ConferenceDuration is Duration = EndTime - StartTime
```

```wl
TaskDuration is Duration = 3 min 30s
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Declarar e inicializar una variable

`<Variable name> is Duration = <Value>
OR

 <Variable name> is Duration
<Variable name> = <Value>`
---

**`<Variable name>: ()`**

Nombre de la variable a declarar. Cuando se declaran e inicializan varias Variable s del mismo tipo en la misma línea, sólo se inicializa la última Variable.

**`<Value>: ()`**

En la mayoría de los casos, una duración es el resultado de una operación realizada sobre las variables [salir con](../Motscles/1514066.md), [momento](../Motscles/1514068.md) o [FechaHora](../Motscles/1514070.md). 
Este valor puede corresponder a: 

- una cadena de caracteres, 

- un entero (duración en milisegundos),

- una duración (por ejemplo, otra duración Variable),

- la duración en un formato legible (por ejemplo: 30 s). 


0 se utiliza para reiniciar la duración.

<a name="NOTE0_1"></a>

## Observaciones


### Valores de una duración
<a name="valores_una_duracion_ELTPARAGRAPHE000105"></a>

Si se maneja una duración Variable: 

- usando una cadena, esta cadena tiene el siguiente formato: "+DHHMMSSCCC" (días, horas, minutos, segundos, milésimas de segundo).  

- usando un número entero, este número entero corresponde a la duración en milisegundos.

- utilizando una duración, esta duración puede corresponder a una variable Duración o a una description clara de la duración. 


En la mayoría de los casos, una duración es el resultado de una operación realizada sobre las variables [salir con](../Motscles/1514066.md), [momento](../Motscles/1514068.md) o [FechaHora](../Motscles/1514070.md). Este valor puede ser una duración.

0 se utiliza para reiniciar la duración.


<a name="NOTE0_2"></a>


### Las propiedades que se pueden utilizar en las duraciones
<a name="las_propiedades_que_pueden_utilizar_las_duraciones_ELTPARAGRAPHE000135"></a>

Las siguientes propiedades pueden ser utilizadas en las duraciones: 


|   |   |
| --- | --- |
| [InHours](../Proprietes/1000017296.md) | Devuelve o modifica el número de horas correspondiente a una duración.. El número de horas se expresa en formato real. |
| [InDays](../Proprietes/1000017295.md) | Devuelve o modifica el número de días correspondiente a una duración. El número de días se expresa en formato real. |
| [InMilliseconds](../Proprietes/1000017299.md) | Devuelve o modifica el número de milisegundos correspondiente a una duración.. El número de milisegundos se expresa en formato entero. |
| [InMinutes](../Proprietes/1000017297.md) | Devuelve o modifica el número de minutos correspondiente a una duración. El número de minutos se expresa en formato real. |
| [InSeconds](../Proprietes/1000017298.md) | Devuelve o modifica el número de segundos correspondiente a una duración.. El número de segundos se expresa en formato real. |
| [momento](../Proprietes/2512038.md) | Devuelve o modifica el número de horas. El número de horas se expresa en formato entero. |
| [jornada](../Proprietes/2512043.md) | Devuelve o modifica el número de días. El número de días se expresa en formato entero. |
| [Milisegundo](../Proprietes/2512062.md) | Devuelve o modifica el número de milisegundos. El número de milisegundos se expresa en formato entero. |
| [Minuto](../Proprietes/2512050.md) | Devuelve o modifica el número de minutos. El número de minutos se expresa en formato entero. |
| [Segundo](../Proprietes/2512081.md) | Devuelve o modifica el número de segundos. El número de segundos se expresa en formato entero. |


Para más detalles sobre las posibles operaciones con duraciones (sumar, restar, comparar), véase [Operaciones que se pueden realizar en la clase Duración](../Motscles/1514040.md).

```wl
StartTime is Time = "1330"
EndTime is Time = "1730"     
ConferenceDuration is Duration = EndTime - StartTime
Info("The conference lasted: " + ConferenceDuration.Hour + " hours")
```

**Importante**: Para almacenar los valores de las propiedades listadas en la tabla anterior, debe respetar el tipo de variable devuelta por cada Property. Por ejemplo:

```wl
MyDuration is Duration
HowManyHours is real

MyDuration.Day = 5
HowManyHours = MyDuration.InHours

Info(MyDuration.Day + " days = " + HowManyHours + " hours")
```

En este ejemplo, la **HowManyHours** Variable debe declararse como real (y no como entero). Si esta Variable se declara como un número entero, su valor será aproximado.
<a name="NOTE0_3"></a>


### Valores admitidos
<a name="valores_admitidos_ELTPARAGRAPHE000207"></a>

El tipo **Duration** permite gestionar duraciones comprendidas entre -2 mil millones y 2 mil millones de días. La precisión es de hasta un milisegundo.
<a name="NOTE0_4"></a>


### Mostrando una duración en un control Campo de entrada
<a name="mostrando_una_duracion_control_campo_entrada_ELTPARAGRAPHE000214"></a>

Para mostrar una duración en una control Campo de entrada, tiene la posibilidad de utilizar [DisplayMask](../Proprietes/1000020008.md). Para obtener más información, consulte [Máscara de visualización para duraciones](../Proprietes/1000020008.md).

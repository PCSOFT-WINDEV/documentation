
## array dinámica (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una array dinámica es un tipo "avanzado" de array: las dimensiones de esta array se asignan a petición durante la ejecución del programa. En la mayoría de los casos, una array "simple" es suficiente.

**Recordatorio**: Un array es un tipo estructurado que permite agrupar una serie de elementos del mismo tipo. Se puede acceder a cada elemento de la array por su índice.

Se recomienda su uso:

- Un **array dinámico** o un ["simple" array](../Motscles/1514030.md) cuando debe modificarse el tamaño del array durante la ejecución del programa.

- Una [fijo array](../Motscles/1514025.md) para las APIs de Windows.

- Una [asociativa array](../Motscles/1514058.md) para almacenar los elementos indexados en cualquier tipo de información.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare a dynamic array
CustomerArray is dynamic array
```


<a name="Example2"></a>



```wl
// Allocate a dynamic array
CustomerArray is dynamic array

CustomerArray = new array of 4 by 7 int
// Equivalent to: CustomerArray = new array of 4,7 int
// (From version 17) Equivalent to: CustomerArray = new array [4,7] int
```


<a name="Example3"></a>





```wl
// Refer to a dynamic array
CustomerArray[2,5,3] = 47
// Equivalent to: CustomerArray[2][5][3] = 47
```

## Observaciones




### Borrar una array dinámica (opcional)
<a name="borrar_una_array_dinamica_opcional_ELTPARAGRAPHE000139"></a>

Una array dinámica se borra automáticamente al final de la vida de la Variable (por ejemplo, cuando se cierra la ventana) o cuando se asignan nuevas dimensiones.

Para eliminar una array dinámica, utilice la siguiente sintaxis:


```txt
delete <Name of dynamic array>
```


Si la array dinámica está declarada y asignada en el mismo Line de código, no debe utilizar la palabra clave [suprimir](../Motscles/1511012.md) para eliminar la array dinámica. Se producirá un error de ejecución si se utiliza [suprimir](../Motscles/1511012.md).
<a name="NOTE0_2"></a>




### Pasar una array dinámica como parámetro a una Procedure
<a name="pasar_una_array_dinamica_como_parametro_una_procedure_ELTPARAGRAPHE000159"></a>

Se puede pasar una array dinámica como parámetro a una Procedure. Para ello, utilice la siguiente sintaxis:


```txt
<Procedure name>(<Array name>)
```


Por ejemplo:


```wl
SupplierArray is dynamic array
SupplierArray = new array of 10 by 50 character strings
// Call to the DisplayArray procedure
DisplayArray(SuppArray)
```

<a name="NOTE0_3"></a>




### Declarar un miembro dinámico de array
<a name="declarar_miembro_dinamico_array_ELTPARAGRAPHE000173"></a>

Un miembro "dinámico de array" puede ser declarado en:

- una estructura,

- un compuesto Variable,

- una clase.




El array dinámico debe asignarse después de la declaración de la estructura o Variable compuesto.

Por ejemplo:


```wl
// Declare a structure
Struct is structure
	x1 is int
	x2 is dynamic array		// Declare the array
END

// Declare a structure variable
MyStruct is struct
// Allocate the array
x2 = new dynamic array of 4,7 int
```


**Para una clase**la declaración debe hacerse en el código de declaración de la clase y la asignación debe hacerse en el constructor de la clase.
<a name="NOTE0_4"></a>




### Funciones WLanguage y arrays dinámica
<a name="funciones_wlanguage_arrays_dinamica_ELTPARAGRAPHE000193"></a>

Se pueden utilizar varias funciones de WLanguage para manipular el arrays dinámico. Usted tiene la capacidad de realizar clasificaciones, búsquedas, ... Para obtener más información, consulte [Funciones de gestión de Array](../WDLang1/3075006.md).
<a name="NOTE0_5"></a>




### Límites: Elementos de una array dinámica
<a name="limites_elementos_una_array_dinamica_ELTPARAGRAPHE000204"></a>

- Una array dinámica puede incluir clases sólo si estas clases tienen un constructor sin parámetro (o con parámetros opcionales).

- Una array dinámica no puede incluir:

	- variables compuestas, 

	- arrays.


## MyHostControl (Palabra reservada)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
La palabra clave **MyHostControl** permite manipular el control Página WEBDEV desde el código de la página que se muestra en dicho control. 

Las únicas sintaxis posibles son: 

- comparar con valores NULL. 

- llamar a un procedimiento. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If integrated in WINDEV or WINDEV Mobile
IF InPageWithinWindowMode() THEN
	// Call the WINDEV application to include the selected option in the invoice
	MyHostControl.RequestBilling(ATT_Sender[gnRowNum], ...
			ATT_Summary[gnRowNum], gnRowNum, ATT_Report[gnRowNum], sGoodieName)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Comprobar si la página se utiliza en un control Página WEBDEV

`IF <WEBDEV control>.MyHostControl = Null THEN
    ...
    // The page is not displayed in a WEBDEV Page control
    ...
END

 IF <WEBDEV control>.MyHostControl <> Null THEN
    ...
    // The page is displayed in a WEBDEV Page control
    ...
END`
---


<a name="SYNTAX2"></a>

### Llamar a un procedimiento de la ventana

`MyHostControl.ProcedureName()`
---


<a name="SYNTAX3"></a>

### Llamar a un procedimiento de la ventana con un valor de retorno

`AFTER MyResult = MyHostControl.ProdcedureName() DO
	// Use of the result
END`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Desde la ventana, es posible llamar a los procedimientos Navegador de la página utilizando la propiedad Page. 

- Llamada a un procedimiento: Es posible pasar parámetros al procedimiento llamado. 

- En la sintaxis 3 ("Llamar a un procedimiento de la ventana con un valor de retorno"), el valor de retorno se recibe de forma asíncrona

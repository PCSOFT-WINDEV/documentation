
## WITH

*** ***
				

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Factoriza la expresión de un elemento en el que se accede a varios subelementos o propiedades.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WITH File
	.Item = 1
END
```


<a name="Example2"></a>



```wl
WITH Control
	..X = 5
	..Y = 6
END
```




<a name="Example3"></a>





```wl
WITH ClassVariable
	.Member = 5
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WITH <Expression representing a base element>
    .<Sub-element>
    ..<Property>
   :<Member or attribute>
END
`
---

**`<WITH>: ()`**

Marca el inicio del bloque de instrucciones.

**`<Expression representing a base element>: ()`**

Elemento utilizado. Se pueden utilizar los siguientes elementos: 

- los controles,

- los archivos HFSQL,

- las fuentes de datos,

- los tipos avanzados (Record, xmlNode, ...),

- las estructuras,

- las clases,

- los objetos.NET y los objetos de la aplicación Windows Store. 




Atención: La expresión que representa el elemento base se evalúa una sola vez. 

**`<END>: ()`**

Marca el final del bloque de declaraciones.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Nidificación
<a name="nidificacion_ELTPARAGRAPHE000066"></a>

Los bloques WITH pueden ser anidados. Sin embargo, el elemento sobre el que se realizan los accesos a los subelementos, propiedades o barras es siempre el elemento del bloque más interior.


```wl
oo is gglCalendar
sTitle, sAuthorName are strings
WITH oo
	sTitle = ..Title
	WITH ..Author
		sAuthorName = ..Name
	END
END
```

<a name="NOTE0_2"></a>




### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000076"></a>

- En un método de clase, no se puede acceder a los miembros de la clase con la sintaxis &lt;:Miembro&gt; dentro de un bloque **WITH**.. El operador "':' está reservado al elemento del bloque **WITH**. 

- El depurador se utiliza para visualizar los subelementos del bloque **WITH** actual.

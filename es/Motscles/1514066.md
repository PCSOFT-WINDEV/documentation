
## Fecha (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Date** permite manipular las fechas fácilmente. Este tipo se utiliza para gestionar las conversiones de forma casi automática ([StringToDate](../WDLang1/3027013.md), [DateToString](../WDLang1/3027025.md), [IntegerToDate](../WDLang1/3027009.md), [DateToInteger](../WDLang1/3027014.md), ...).

Se puede utilizar este tipo:

- para recuperar y manipular los ítems HFSQL en formato Date.

- para recuperar y manipular los controles de edición de la fecha.

- en las funciones WLanguage utilizadas para gestionar las fechas.

- para realizar cálculos en fechas (diferencia, suma,...). Para obtener más información, consulte [Operaciones que se pueden realizar en el tipo Fecha](../Motscles/1514040.md).




**Observaciones**:

- Se pueden utilizar varias propiedades con el tipo **Date**, para obtener solo una parte de la fecha, por ejemplo.

- Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartDate is Date = 1205
EndDate is Date = "20011231"     // 31/12/2001
TodaysDate is Date = DateSys()
```

## Observaciones




### Declaración e inicialización
<a name="declaracion_inicializacion_ELTPARAGRAPHE000098"></a>

Un **Date** Variable puede ser inicializado con un valor cuyo tipo es:

- **Date**

- **DateTime**: sólo se tendrá en cuenta la fecha.

- **String**de caracteres, en formato AAAAAMMDDD (AAAAA: año, MM: Mes, DD: Día)

- **Integer**: la fecha corresponderá al número de días transcurridos entre el 1 de enero de 1800 y la fecha especificada.


Por defecto, una variable de tipo **Date** se inicializa con la fecha del día (fecha del sistema). Para definir el valor predeterminado asignado a las variables de tipo **Date**, [momento](../Motscles/1514068.md) y [FechaHora](../Motscles/1514070.md), utilice [DateTimeByDefault](../WDLang1/3027015.md).

Observación: También es posible inicializar una variable de tipo **Date** con la fecha del sistema utilizando la función [Now](../WDLang1/3027024.md). 
<a name="NOTE0_2"></a>




### Propiedades disponibles para las fechas
<a name="propiedades_disponibles_para_las_fechas_ELTPARAGRAPHE000129"></a>

Las siguientes propiedades pueden ser utilizadas en las fechas: 


|   |   |
| --- | --- |
| [año](../Proprietes/2512095.md) | Devuelve o modifica el año de la fecha especificada (en formato entero). |
| [jornada](../Proprietes/2512043.md) | Devuelve o modifica el día de la fecha especificada (en formato entero). |
| [Mes](../Proprietes/2512052.md) | Devuelve o modifica el mes de la fecha especificada (en formato entero). |

Para más detalles sobre las posibles operaciones con fechas (sumar, restar, comparar), véase [Operaciones que se pueden realizar en el tipo Fecha](../Motscles/1514040.md).


```wl
StartDate is Date = "20011201"
// Adds 5 years to the date
StartDate.Year += 5
// Modifies the month
StartDate.Month = 5
// Calculates the end (30 days later)
let EndDate = StartDate
EndDate.Month++
EndDate.Day--
```

<a name="NOTE0_3"></a>




### Intervalo de fechas
<a name="intervalo_fechas_ELTPARAGRAPHE000160"></a>

El tipo **Date** permite gestionar fechas comprendidas entre el 01/01/0001 y el 12/31/9999.

**Observación**: Los cálculos sobre las fechas realizadas con las [Funciones de WLanguage](../WDLang1/3027029.md) o las [Propiedades de WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.


## Operador cambiario (<=>)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El **&lt;=&gt;** se utiliza para intercambiar el contenido de dos elementos. Las conversiones de tipo se realizan si es necesario. Devuelve la dirección de un Variable como un número entero.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
I is int = 5 
J is int = 6 
I <=> J
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Element 1> <=> <Element 2>
`
---

**`<Element 1>: (Tipo simple)`**

Nombre del elemento Variable, control, HFSQL cuyo contenido debe ser intercambiado con el contenido de &lt;Elemento 2&gt;.

**`<Element 2>: (Tipo simple)`**

Nombre del elemento Variable, control, HFSQL cuyo contenido debe ser intercambiado con el contenido de &lt;Elemento 1&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El operador de la centralita sólo funciona con los tipos simples.

|   |   |

## Sentencias de bucle
<a name="sentencias_bucle_ELTTEXTE000159"></a>
Las sentencias de bucle son las siguientes: 



|   |   |
| --- | --- |
| [FOR](../Motscles/1510001.md) | El bloque de expresión se repite hasta que se alcanza un valor límite |
| [FOR EACH](../Motscles/1510013.md) | Navegación HyperFileSQL (navegación completa, navegación con búsqueda, navegación con filtro) |
| [FOR EACH](../Motscles/1510016.md) | Iterar a través de la arrays |
| [FOR EACH](../Motscles/1510018.md) | Iterar sobre los caracteres de una cadena |
| [FOR EACH](../Motscles/1510017.md) | Iterar a través de los controles |
| [LOOP](../Motscles/1510004.md) | El bloque de instrucciones se repite sin parar |
| [WHILE](../Motscles/1510010.md) | El bloque de expresión se repite según una condición |


La instrucción FOR EACH también se puede utilizar para navegar por los objetos.net que implementan la interfaz IEnumerable.

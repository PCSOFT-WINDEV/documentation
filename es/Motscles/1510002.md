


## Declaraciones simples
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Los enunciados simples de WLanguage son los siguientes:



|   |   |
| --- | --- |
| [BREAK](../Motscles/1510012.md) | Salir de un bloque de declaraciones y ejecutar el resto de la actual Process. |
| [CONTINUE](../Motscles/1510003.md) | Pasa al principio de la siguiente iteración sin terminar el código de la iteración actual. |
| [RETURN](../Motscles/1510009.md) | Salir de un bloque de declaración y salir de la actual Process (o Procedure). |
| [RETURN](../Motscles/1510007.md) | Sale de la Process actual (o Procedure) y devuelve un código de estado. |


## Conector Natif DB2 para WINDEV y WEBDEV
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000120"></a>
WINDEV y WEBDEV proponen un Conector Nativo opcional para DB2 (también llamado "Native Access to DB2"). Este Conector Nativo le permite acceder a una base de datos DB2 desde un programa WLanguage sin utilizar ningún controlador externo.

**Atención**: Esta versión del Conector DB2 nativo sólo funciona con WINDEV y WEBDEV.





<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Configuración necesaria
<a name="configuracion_necesaria_ELTTEXTE000144"></a>


### Condiciones de uso del conector DB2 nativo
<a name="condiciones_uso_del_conector_db2_nativo_ELTPARAGRAPHE000020"></a>

- DB2 versión 7.1 o posterior (Conector Nativo probado por nosotros con las versiones 8.2 y 10). 

- El cliente DB2 en cada ordenador que accede a DB2.

- ***DB2CLI.DLL*** debe ser accesible:

	- del ordenador de desarrollo,

	- de la computadora implementación.






<a name="NOTE3_2"></a>


### Los archivos necesarios
<a name="los_archivos_necesarios_ELTPARAGRAPHE000032"></a>

Los siguientes archivos son necesarios para utilizar el conector DB2 nativo:

- en 32 bits: 

	- ***DB2CLI.DLL***. Esta DLL es instalada por la capa de cliente de 32 bits de DB2. Vea el acuerdo de licencia de DB2 para más detalles sobre las condiciones de distribución de esta DLL.

	- ***wd28 0db2.dll***. Requerido para Conector Nativo a una base de datos DB2 en modo 32-bit. Este archivo se suministra con Conector Nativo para DB2.









- en 64 bits: 

	- ***DB2CLIO.DLL***. Esta DLL es instalada por la capa de cliente de 64 bits de DB2. Vea el acuerdo de licencia de DB2 para más detalles sobre las condiciones de distribución de esta DLL.

	- ***wd28 0db2_64.dll***. Requerido para Conector Nativo a una base de datos DB2 en modo 64-bit. Este archivo se suministra con Conector Nativo para DB2.

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000174"></a>


### Para usar el conector nativo DB2 en sus aplicaciones WINDEV o en sus sitios WEBDEV
<a name="para_usar_conector_nativo_db2_sus_aplicaciones_windev_sus_sitios_webdev_ELTPARAGRAPHE000057"></a>

Para utilizar la DB2 en modo nativo, debe programar:

- Con las funciones "SQL" de WLanguage (véase [Programación con funciones SQL](../WDDB2/5510002.md)).

- Con las funciones "HFSQL" de WLanguage (véase [Programación con funciones HFSQL](../WDDB2/5510003.md)).




Observación: Para obtener más información, consulte [las limitaciones con respecto al uso del conector DB2 nativo](../WDDB2/5510007.md).

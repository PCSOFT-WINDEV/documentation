## Presentación
<a name="presentacion_ELTTEXTE000247"></a>
WDSQL está acostumbrado a:

- Cree y ejecute consultas SQL en una base de datos desde el ordenador del desarrollador, así como desde el ordenador del usuario final.

- Compruebe la validez de los parámetros de conexión y su modo de funcionamiento.

- Convertir la estructura de una base de datos HFSQL en un script SQL. Luego, este script puede ejecutarse en una base de datos SQL para crear la base de datos correspondiente al análisis HFSQL.




Observación: WDSQL está disponible en versiones de 32 y 64 bits. La versión de 64 bits se utiliza para acceder a bases de datos externas para las que sólo está disponible el controlador ODBC en 64 bits..

## aprovechamiento
<a name="aprovechamiento_ELTTEXTE000271"></a>


### Inicio de WDSQL
<a name="inicio_wdsql_ELTPARAGRAPHE000026"></a>

Para iniciar WDSQL si WINDEV está instalado en su ordenador:

- En el editor: en la pestaña "Herramientas", en el grupo "Base de datos", haga clic en "WDSql".

- Ejecute "WDSQL.EXE" directamente (versión de 32 bits).

- Ejecute "WDSQL64.EXE" directamente (versión de 64 bits).




Si WINDEV no está instalado en su ordenador, puede iniciar WDSQL ejecutando "WDSQL.EXE" directamente.

Tan pronto como se inicia el WDSQL, una conexión asistente permite definir la base de datos a utilizar.
<a name="NOTE2_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000055"></a>

WDSQL puede ser redistribuido. Por lo tanto, WDSQL puede ser instalado con las aplicaciones desarrolladas con WINDEV. El acuerdo de licencia de WINDEV se aplica en su totalidad.

Al crear el instalación Procedure, WDSQL puede ser suministrado con sus aplicaciones.

**Recordatorio**: El procedimiento de instalación se puede crear: 

- mediante el asistente: en la pestaña "Proyecto", en el grupo "Generación", despliegue "Procedimiento de instalación" y seleccione "Crear procedimiento de instalación".

- desde el editor de instalación: en la pestaña "Herramientas", en el grupo "Utilidades", haga clic en "WDInst".


## WDSQL: Exportar e imprimir el resultado de una consulta SQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000106"></a>
El resultado de una consulta de selección puede ser:

- exportado a un archivo (texto, Excel, XML o Word),

- impreso.


**Observación**: Para las siguientes operaciones, consideramos que el resultado de la consulta se muestra en WDSQL.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Exportar el resultado de una consulta selecta
<a name="exportar_resultado_una_consulta_selecta_ELTTEXTE000130"></a>


### Exportar el resultado de una consulta selecta a un archivo de texto
<a name="exportar_resultado_una_consulta_selecta_archivo_texto_ELTPARAGRAPHE000023"></a>

Para exportar el resultado de una consulta selecta a un archivo de texto:

1. Haga clic en el botón "Exportar. Aparece la ventana para seleccionar el archivo de texto.

2. Especifique el nombre y la ubicación del archivo de texto (extensión "TXT" de Default).

3. Elija el tipo de separador utilizado entre los diferentes elementos de cada Record (opción "Tipo"):

	- ***Separador: Tabulación***: Los elementos estarán separados por fichas.

	- ***Separador: Punto y coma***: Los ítems estarán separados por punto y coma.




4. Valide. Los registros encontrados en el resultado de la consulta se exportan automáticamente al archivo de texto.

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ExporterFichierTexte.gif)

<a name="NOTE2_2"></a>


### Exportación del resultado de una consulta seleccionada a Excel, Word o XML
<a name="exportacion_del_resultado_una_consulta_seleccionada_excel_word_xml_ELTPARAGRAPHE000040"></a>

Para exportar el resultado de una consulta seleccionada a Excel, Word o XML:

1. Haga clic con el botón derecho del ratón en la tabla que contiene el resultado de la consulta.

2. Seleccione el modo de exportación deseado: Word, Excel o XML.

3. Especifique el nombre del fichero de exportación y valide.

## Imprimir el resultado de una consulta de selección
<a name="imprimir_resultado_una_consulta_seleccion_ELTTEXTE000160"></a>
Para imprimir el resultado de una consulta de selección:

1. Haga clic en el botón "Imprimir. Aparece la ventana para seleccionar los elementos que se van a imprimir.

2. Especifique los elementos que desea imprimir y validar. El resultado de la consulta se muestra en una presentación preliminar.

3. Haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%202.gif) para imprimir el resultado de la consulta.

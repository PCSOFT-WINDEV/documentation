


## WDSQL: Convertir la estructura de una base de datos HFSQL
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000103"></a>
WDSQL se utiliza para convertir la estructura de una base de datos HFSQL en un script SQL. Luego, este script puede ejecutarse en una base de datos SQL para crear la base de datos correspondiente al análisis HFSQL.

Esta opción se utiliza, por ejemplo, para simplificar la creación de un análisis en una base de datos Oracle a partir de un análisis existente en formato WINDEV.

**Observación**: Para convertir la estructura de una base de datos HFSQL, debe estar conectado a una base de datos SQL. Esta conexión se utiliza para identificar los tipos de columnas reconocidas por la base de datos SQL seleccionada.. Estos tipos de columnas se utilizarán al exportar la base de datos HFSQL a la base de datos SQL.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000133"></a>


### Convertir la estructura de una base de datos desde WDSQL
<a name="convertir_estructura_una_base_datos_desde_wdsql_ELTPARAGRAPHE000024"></a>

Para convertir la estructura de una base de datos de WDSQL:

1. Conectarse a una base de datos SQL. Para obtener más información, consulte [Conexión a una base de datos](../WDSQL/3521010.md).

2. Seleccione "Herramientas ... Análisis conversión" o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%206.gif). Comienza la conversión de la asistente para el análisis.

3. Seleccione el archivo de análisis WINDEV a Convert(".WDD").

4. Si es necesario, indique la contraseña de este análisis (contraseña de tiempo de ejecución definida en la descripción del análisis.

5. Seleccione el directorio que contiene los archivos de datos del análisis a convertir y vaya al siguiente paso.

6. Seleccione los archivos de datos que deben ser convertidos a formato SQL y vaya al siguiente paso.

7. El guión de conversión generado se muestra en la asistente.
	**Observación**: En la asistente, puedes:

	- modificar este script, 

	- encontrar una expresión en este guión, 

	- reemplazar las expresiones encontradas por otra expresión.




8. Validar. El código SQL del script generado se muestra automáticamente en la ventana principal de WDSQL.




**Observación**: Este script puede ser:

- modificado en la ventana principal de WDSQL,

- guardada ("Archivo .. Guardar una consulta" o ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%201.gif)),

- ejecutar ("Ejecución .. Ejecutar un script por lotes" o ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%203.gif)).

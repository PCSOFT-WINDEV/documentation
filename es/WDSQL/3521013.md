
## WDSQL: Creación y ejecución de un script
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000146"></a>
Un script es un set de comandos SQL utilizados para realizar una operación específica (crear un nuevo tabla en la base de datos, insertar registros, ...).

Un guión incluye varias líneas. Cada script Line contiene un código SQL que puede ser ejecutado independientemente.

Un guión debe ser ejecutado por Line por Line, en modo Batch.

**Observación**: Las siguientes operaciones sólo se pueden realizar en archivos a los que se accede a través de un proveedor de OLE DB, un controlador ODBC o un acceso nativo:

- Creación de la estructura de un fichero/tabla.

- Modificación de la estructura de un fichero/tabla.

- Borrar la estructura de un fichero/tabla.

## Creación de un script
<a name="creacion_script_ELTTEXTE000170"></a>
Se puede crear un script:

- de la ventana principal de WDSQL,

- de la asistente para la creación de la tabla (guión para la creación de la tabla).




**Observaciones**:

- Para guardar un guión, seleccione "Archivo... Guardar una consulta" o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%201.gif).

- Para imprimir el código SQL de un script, seleccione "Archivo .. Imprimir el código fuente de la consulta" o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%202.gif).

### Creación de un script para crear una nueva tabla en la base de datos actual
<a name="creacion_script_para_crear_una_nueva_tabla_base_datos_actual_ELTPARAGRAPHE000065"></a>

Crear un script para crear una nueva tabla en la base de datos actual:

1. Seleccione "Archivo .. Crear una consulta" (o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%205.gif)). El asistente se abre. 

2. Seleccione "Un guión para crear una nueva tabla".. Pase a la etapa siguiente.

3. Escriba el nombre de la tabla a crear (opción"Nombre de tabla").

4. Para cada artículo de tabla:

	- Escriba el nombre del artículo (opción "nombre de control").

	- Haga clic en el botón "Agregar".

	- Especifique el tipo de artículo. Los tipos disponibles dependen de la base de datos actual.

	- Especifique el tamaño del artículo.

	- Especifique si el artículo debe indexarse.

	- Especifique si el artículo puede ser nulo.




5. Comprobar (si es necesario) "Generar el comando DROP TABLE" para generar automáticamente el comando "DROP TABLE"..
	**Recordatorio**: El comando DROP TABLE se utiliza para borrar un tabla.

6. Valide el asistente. El código SQL del script para la creación de tabla se muestra automáticamente en la ventana principal de WDSQL.

## Ejecución de un script
<a name="ejecucion_script_ELTTEXTE000206"></a>
Para ejecutar un script:

1. Cargue (si es necesario) el script a ejecutar en WDSQL (seleccione "Archivo .. Cargar una consulta" o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%207.gif)).

2. Seleccione "Ejecución .. Ejecutar un script por lotes" (o haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=WDSQL%20-%20HC%20N%B0006%203.gif)).

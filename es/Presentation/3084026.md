## Compartir elementos entre WINDEV
<a name="compartir_elementos_entre_windev_ELTTEXTE000094"></a>
Un sitio WEBDEV puede compartir elementos con una aplicación WINDEV o una aplicación WINDEV Mobile.

Los elementos que se pueden compartir son los siguientes:


|   |   |
| --- | --- |
| Análisis | El análisis encontrado en un proyecto WINDEV o en un proyecto WINDEV Mobile puede ser compartido con un proyecto WEBDEV. |
| Clases | Las clases que se encuentran en un proyecto WINDEV o en un proyecto WINDEV Mobile pueden ser compartidas con un proyecto WEBDEV.. <br>Atención: En la clase compartida sólo se debe utilizar el código compatible con WEBDEV. |
| Conjuntos de procedimientos | Los conjuntos de procedimientos que se encuentran en un proyecto WINDEV o en un proyecto WINDEV Mobile pueden ser compartidos con un proyecto WEBDEV.. <br>Atención: sólo el código compatible con WEBDEV debe ser usado en la set compartida de procedimientos. |
| Consultas | <br><br>- Las consultas encontradas en un proyecto WINDEV o en un proyecto WINDEV Mobile pueden ser compartidas con un proyecto WEBDEV.<br><br>- Las consultas encontradas en un proyecto WINDEV o en un proyecto WINDEV Mobile pueden ser cargadas en un proyecto WEBDEV.. <br>	Estas consultas son consultas de sólo lectura. Pueden ser ejecutados y el código SQL de associated puede ser visto. No se puede ver el gráfico de la consulta.<br><br><br> |
| Reportes | <br><br>- Los informes encontrados en un proyecto WINDEV pueden ser compartidos con un proyecto WEBDEV.<br><br>- Los informes encontrados en un proyecto WINDEV no pueden ser compartidos con un proyecto WINDEV.<br><br><br> |

**Observación**: Las ventanas de una aplicación WINDEV pueden ser importadas directamente a un proyecto WEBDEV.. Para obtener más información, consulte [Importar una ventana WINDEV](../Editeurs/2030034.md).

## Presentación
<a name="presentacion_ELTTEXTE000081"></a>
Informes y Consultas está acostumbrado a:

- crear, manipular y/o ejecutar informes. Para obtener más información, consulte [Versión de usuario de editor de reportes](../Presentation/3088004.md).

- crear, manipular y/o ejecutar consultas. Para obtener más información, consulte [Versión de usuario del editor de consultas](../Presentation/3088006.md).




**Observación**: Reportes y Consultas reemplaza al Usuario de WDEtat.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Condiciones de uso
<a name="condiciones_uso_ELTTEXTE000105"></a>
Reports & Queries es un módulo independiente de WINDEV. Este módulo está destinado a los usuarios de las aplicaciones WINDEV. Con este módulo, los usuarios tienen la posibilidad de crear sus propios informes y consultas:

- de una aplicación WINDEV existente.

- independientemente de cualquier aplicación WINDEV.

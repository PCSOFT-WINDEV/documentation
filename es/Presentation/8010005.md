
## Los diferentes servidores de aplicaciones WEBDEV
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000162"></a>
WEBDEV incluye varios servidores de aplicaciones WEBDEV: 

- **Servidor de aplicaciones WEBDEV completo** (versión implementación ): Este servidor de aplicaciones le ofrece todas las funciones de WEBDEV: 

	- Conexiones ilimitadas para sitios WEBDEV y servicios web, 

	- Aplicaciones específicas (servidor de replicación, servidor de telemetría, módulos BaaS, servicios web de acceso a bases de datos de terceros).   

	- Compatibilidad con WEBDEV Cluster.  

	- Administrador web: SaaS, HFSQL, Servidor de aplicaciones WEBDEV.




- **WEBDEV Application Server - 10 conexiones**: Este servidor de aplicaciones admite hasta 10 conexiones simultáneas a sitios WEBDEV y servicios web. 

- **El servidor reservado**: Este servidor de aplicaciones no es compatible con sitios WEBDEV y servicios web. Sólo están disponibles aplicaciones específicas (servidor de replicación, servidor de telemetría, módulos BaaS, servicios web para acceder a bases de datos de terceros).. 




Observaciones: 

- Puede utilizar el Servidor de Aplicaciones WEBDEV - 10 conexiones junto con el Servidor Reservado. 

- A partir de la versión 28, se utiliza un único programa instalación para instalar tanto la versión completa de WEBDEV Application Server como WEBDEV Application Server - 10 conexiones..

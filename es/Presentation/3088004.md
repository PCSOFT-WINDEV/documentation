## Presentación
<a name="presentacion_ELTTEXTE000139"></a>
Un reporte proporciona una vista personalizada de los datos: datos introducidos en la base de datos, de un archivo de texto, de un control Tabla, etc, ... Para obtener más información, consulte [Informes](../WDChamp/1011023.md).

Una reporte puede ser creada y modificada en el editor de reportes. Para obtener más información, consulte [Entorno de la editor de reportes](../Editeurs/2029004.md).

Dos versiones de editor de reportes están disponibles:

- ***el editor de reportes integrado directamente***.
	Esta versión se suministra con WINDEV, WEBDEV o WINDEV Mobile. Esta versión sólo se puede utilizar al desarrollar una aplicación WINDEV o WINDEV Mobile, o un sitio WEBDEV.. Para obtener más información, consulte [Editor de reportes](../WDChamp/1011069.md).

- ***el editor de reportes del Usuario*** (también llamado ***Reports & Queries***).
	Este editor sólo se suministra con WINDEV.
	Esta versión es independiente de WINDEV y puede ser utilizada por los usuarios finales.. 
	Algunas características de editor de reportes no están disponibles en esta versión. Para obtener más información, consulte [Limitaciones de la versión de usuario de editor de reportes](../Presentation/3088005.md).


Contact PC SOFT Departamento de Ventas para más detalles sobre Informes y Consultas.

## Versión de usuario de editor de reportes (Informes y Consultas)
<a name="version_usuario_editor_reportes_informes_consultas_ELTTEXTE000163"></a>
Reports & Queries le permite crear y ejecutar informes desde el editor (WINDEV no es necesario).

Los informes no pueden contener líneas de código para modificar los archivos de datos (funciones utilizadas para realizar operaciones de escritura en los archivos).

Los informes creados pueden ejecutarse desde editor de reportes o desde una aplicación WINDEV (si esta aplicación soporta los informes creados con Reports & Queries).

Algunas características de editor de reportes no están disponibles en esta versión. Para obtener más información, consulte [Limitaciones de la versión de usuario de editor de reportes](../Presentation/3088005.md).

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_nondispo4.gif) Icon indica las diferencias entre la versión de usuario del editor de reportes y el editor de reportes estándar.

No se requiere ningún dongle para utilizar Reports & Queries.

Algunas operaciones son necesarias para crear nuevos informes o para modificar los informes existentes con Informes y Consultas. Para obtener más información, consulte [Distribución de informes y consultas](../Presentation/3088002.md).

## Migración de los informes creados en la versión de usuario de WDEtat
<a name="migracion_los_informes_creados_version_usuario_wdetat_ELTTEXTE000187"></a>
Los informes creados en la versión de usuario de WDEtat se pueden abrir en Informes y Consultas. Cuando se abren, estos informes se migran automáticamente a la versión 28.

Le recomendamos que guarde estos informes en Informes y consultas. Para guardar los reportes, en la pestaña "Inicio", en el grupo "General", despliegue "Guardar" y seleccione "Guardar".

**Observación**: Los informes creados en WDQBE no se pueden abrir en Informes y Consultas. Estos informes deben volver a crearse en Informes y consultas.

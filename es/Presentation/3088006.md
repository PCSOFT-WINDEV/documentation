## Presentación
<a name="presentacion_ELTTEXTE000131"></a>
Una consulta se utiliza para interrogar una base de datos con el fin de ver, insertar, modificar o eliminar datos. La estructura de consulta define los datos manipulados. Para obtener más información, consulte [Consultas](../Editeurs/2032052.md).

**Se puede crear y manejar una consulta en el editor de consultas**.

Existen dos versiones del editor de consultas:

- ***el editor de consultas incluido en el producto***.
	Esta versión se proporciona con WINDEV, WEBDEV o WINDEV Mobile. Esta versión sólo se puede utilizar al desarrollar una aplicación WINDEV o WINDEV Mobile, o un sitio WEBDEV.. Para obtener más información, consulte [Editor de consultas](../Editeurs/2032063.md).

- ***el editor de consultas Usuario*** (también llamado ***Reports & Queries***).
	Este editor sólo se proporciona con WINDEV.
	Esta versión es independiente de WINDEV y puede ser utilizada por los usuarios finales.. 
	Esta versión se incluye en Reports &Queries. Para obtener más información, consulte [Informes y Consultas](../Presentation/3088003.md).
	Algunas características del editor de consultas no están disponibles en esta versión. Para obtener más información, consulte [Limitaciones de la versión de usuario del editor de consultas](../Presentation/3088001.md).




Contact PC SOFT Departamento de Ventas para más detalles sobre la versión de usuario del editor de consultas.

## Versión de usuario del editor de consultas (Informes y Consultas)
<a name="version_usuario_del_editor_consultas_informes_consultas_ELTTEXTE000155"></a>
Reports & Queries le permite crear y ejecutar consultas desde el editor (WINDEV no es necesario).

Las consultas creadas pueden ejecutarse desde el editor de consultas o desde una aplicación WINDEV (si esta aplicación soporta las consultas creadas por Reports & Queries, a través de [HExecuteQuery](../WDLang4/3044080.md)).

Algunas características del editor de consultas no están disponibles en esta versión. Para obtener más información, consulte [Limitaciones de la versión de usuario del editor de consultas](../Presentation/3088001.md).

El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=picto_nondispo4.gif) Icon indica las diferencias entre la versión de usuario del editor de consultas y el editor de consultas estándar.

No es necesario un dongle para utilizar la versión de usuario del editor de consultas.

Algunas operaciones son necesarias para crear nuevas consultas o para modificar las consultas existentes con Reports & Queries. Para obtener más información, consulte [Distribución de informes y consultas](../Presentation/3088002.md).

## Migración de las consultas creadas en la versión de usuario de WDEtat
<a name="migracion_las_consultas_creadas_version_usuario_wdetat_ELTTEXTE000179"></a>
Las consultas creadas en la versión de usuario de WDEtat se pueden abrir en Informes y Consultas. Estas consultas se migran automáticamente a la versión 16. cuando se abren.

Para poder utilizar estas consultas, le aconsejamos que las guarde con Informes y Consultas (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ico_Enregistrer.gif)).

**Observación**: Las consultas creadas en WDQBE no se pueden abrir en Informes y Consultas. Estas consultas deben volver a crearse en Informes y consultas.

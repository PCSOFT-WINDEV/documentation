## Características específicas
<a name="caracteristicas_especificas_ELTTEXTE000139"></a>


### Tipo de consultas
<a name="tipo_consultas_ELTPARAGRAPHE000026"></a>

La versión de usuario del editor de consultas no se puede utilizar para crear:

- insertar consultas.

- consultas de actualización.

- delete queries.

- en código SQL que utilizan los siguientes comandos: [INSERTAR](../Editeurs/2034001.md), [ACTUALIZACIÓN](../Editeurs/2034001.md), [BORRAR](../Editeurs/2034001.md)




**Recordatorio**: La versión de usuario del editor de consultas no permite modificar la base de datos del proyecto actual.
<a name="NOTE2_2"></a>


### Fuente de datos de una consulta
<a name="fuente_datos_una_consulta_ELTPARAGRAPHE000048"></a>

Los archivos y los elementos que pueden ser utilizados en una consulta están definidos en la editor de análisis de WINDEV. Para más detalles sobre los requisitos para utilizar una aplicación WINDEV con Reports and Queries, véase [Distribución de informes y consultas](../Presentation/3088002.md).
<a name="NOTE2_3"></a>


### Cristales acoplables
<a name="cristales_acoplables_ELTPARAGRAPHE000058"></a>

En la versión de usuario del editor de consultas, los paneles acoplables le permiten acceder:

- las principales características del editor de consultas del usuario.

- los elementos accesibles (informes, consultas,...) de su proyecto.




Para más detalles sobre los paneles acoplables, véase [Cristales acoplables](../Editeurs/2027001.md).




## Informes y Consultas: Funciones WLanguage no soportadas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000256"></a>
Algunas funciones de WLanguage no están disponibles en Reports & Queries. Estas funciones son las siguientes:


|   |   |
| --- | --- |
| [HAdd](../WDLang4/3044147.md) | Añade el registro encontrado en memoria a la tabla (consulta o vista). |
| [HAddGroup](../WDLang4/3044280.md) | Añade un grupo de usuarios (conexión HFSQL Client/Server) |
| [HAddLink](../WDLang4/3044312.md) | Agrega una regla de integridad entre dos archivos en el servidor. |
| [HAddUser](../WDLang4/3044281.md) | Añade un usuario (conexión HFSQL Client/Server) |
| [HChangeLogDir](../WDLang4/3044164.md) | Modifica la ubicación de los archivos de registro correspondientes a un archivo HFSQL. (Archivo\*JNL, así como archivos JournalOpération y JournalIdentification si existen). |
| [HCopyFile](../WDLang4/3044282.md) | Copia un archivo HFSQL del cliente al servidor (y a la inversa) |
| [HCreation](../WDLang4/3044255.md) | Crea un archivo de datos vacío (extensión .FIC) con el archivo de índice y el archivo de notas si es necesario. |
| [HCreationIfNotFound](../WDLang4/3044254.md) | La función **HCreationIfNotFound** realiza las siguientes operaciones:<br><br>- Si el archivo no existe, crea un archivo de datos vacío (archivo "FIC") con el archivo de índice y el archivo de notas si es necesario.. <br>	La función equivale a [HCreation](../WDLang4/3044255.md).<br><br>- Si el archivo existe, abre el archivo. <br>	En este caso, la función equivale a [HOpen](../WDLang4/3044120.md).  <br>	Para optimizar la llamada a la función **HCreationIfNotFound**, es posible abrir el archivo solo la primera vez que se accede a él.<br><br><br> |
| [HCross](../WDLang4/3044121.md) | Cruza un archivo Record. La Record se borra lógicamente y puede ser recuperada si es necesario. |
| [HDBCreation](../WDLang4/3044171.md) | Termina la description dinámica de la estructura de archivos. El archivo que se acaba de describir se crea en el disco en el ruta especificado en [HDBDescribeFile](../WDLang4/3044172.md). Los archivos de datos, notas e índices se crean en el disco. |
| [HDelete](../WDLang4/3044018.md) | Borra una Record de un archivo (de una consulta o de una vista). |
| [HDeleteDatabase](../WDLang4/3044332.md) | Elimina una base de datos de un servidor HFSQL. Todas las conexiones que utilizan la base de datos eliminada están cerradas. |
| [HDeleteFile](../WDLang4/3044294.md) | Elimina los archivos HFSQL (archivos.fic,.ndx y.mmo si existen) del servidor. |
| [HDeleteGroup](../WDLang4/3044295.md) | Elimina del servidor a un grupo de usuarios asociados a una conexión . |
| [HDeleteLink](../WDLang4/3044314.md) | Elimina una regla de integridad entre dos archivos del servidor. |
| [HDeleteUser](../WDLang4/3044296.md) | Elimina (del servidor) un usuario asociado con una conexión. |
| [HFree](../WDLang4/3044064.md) | Transforma los registros cruzados (HCross) en registros eliminados. El espacio ocupado por los registros cruzados se libera y puede ser reutilizado durante las siguientes adiciones (HAdd). |
| [HImportHF55](../WDLang4/3044055.md) | Importa un archivo Hyper File 5.5 a un archivo HFSQL Classic. |
| [HImportText](../WDLang4/3044011.md) | Importa un archivo de texto a un archivo en formato HFSQL Classic. |
| [HImportXML](../WDLang4/3044007.md) | Importa un archivo XML dentro de un archivo en formato HFSQL Classic. |
| [HMigrateLinkedCompositeKey](../WDLang4/3044038.md) | Migra el valor de las claves compuestas relacionadas que provienen de un archivo en formato Hyper File 5.5 al formato HFSQL Classic. |
| [HModify](../WDLang4/3044042.md) | Modifica el registro especificado o el que esta en la memoria en el archivo de datos (consulta o vista). |
| [HModifyDatabaseProperty](../WDLang4/3044329.md) | Modifica las propiedades de una base de datos ubicada en un servidor HFSQL. |
| [HModifyDatabaseRights](../WDLang4/3044326.md) | Modifica los permisos otorgados a un usuario o grupo para una base de datos HFSQL Client/Server. |
| [HModifyFileProperty](../WDLang4/3044330.md) | Modifica las propiedades de un archivo HFSQL ubicado en un servidor. |
| [HModifyFileRights](../WDLang4/3044327.md) | Modifica los permisos otorgados a un usuario o grupo en un archivo de datos HFSQL Client/Server. |
| [HModifyGroup](../WDLang4/3044308.md) | Modifica la información del grupo especificado según la información de la estructura. |
| [HModifyServerProperty](../WDLang4/3044331.md) | Modifica las propiedades de un servidor HyperFileSQL. |
| [HModifyServerRights](../WDLang4/3044328.md) | Modifica los permisos otorgados a un usuario o grupo en un servidor HFSQL. |
| [HModifyUser](../WDLang4/3044309.md) | Modifica la información del usuario según la información de la estructura. |
| [HRegenerateFile](../WDLang4/3044122.md) | Regenera un archivo a partir de su registro. |
| [HStopServer](../WDLang4/3044317.md) | Detiene un servidor HFSQL. Para realizar esta acción, el usuario debe tener suficientes derechos en el servidor. |
| [HViewToFile](../WDLang4/3044021.md) | Guarda las modificaciones realizadas en una vista (por [HModify](../WDLang4/3044042.md), [HDelete](../WDLang4/3044018.md) o [HCross](../WDLang4/3044121.md)) en el archivo correspondiente. |
| [HWrite](../WDLang4/3044092.md) | Escribe un registro en un archivo de datos sin actualizar los índices que corresponden a todas las claves utilizadas en el archivo. |

## Presentación
<a name="presentacion_ELTTEXTE000161"></a>
Esta página de ayuda presenta las características específicas de "Reports & Queries" (que significa la versión de usuario de editor de reportes):

- Fuente de datos de reporte.

- Funciones para modificar registros.

- Acristalamientos verticales.

- Plantillas personalizadas de informes tema.

- Modificaciones de los informes encontrados en un componente.




**Observación**: El ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=er.png) Icon (que se encuentra en la ayuda en línea) indica las diferencias entre la versión de usuario del editor de reportes y el editor de reportes estándar.

## Características específicas
<a name="caracteristicas_especificas_ELTTEXTE000185"></a>


### Fuente de datos de reporte
<a name="fuente_datos_reporte_ELTPARAGRAPHE000028"></a>

La versión de usuario de editor de reportes no se puede utilizar para crear:

- ***informa sobre una vista HFSQL***. Sin embargo, tienes la posibilidad de personalizar los informes basados en una vista HyperFileSQL creada en el estándar editor de reportes (Color, fuentes, tamaño, ...).

- ***reportes en un control Tabla***. Sin embargo, tienes la posibilidad de personalizar los informes de un control Tabla creado en el editor de reportes estándar (Color, fuentes, tamaño, columna relacionado, ...).

- ***reportes en un control Agenda o Planificador***. Sin embargo, tienes la posibilidad de personalizar los informes de un Organizador o control Planificador creado en el estándar editor de reportes (Color, fuentes, tamaño, columna relacionado, ...).

- ***informes sobre una fuente de datos programada***.




**Observación**: Los archivos de datos y los elementos que pueden ser utilizados en un reporte basado en un archivo de datos están definidos en el editor de análisis de WINDEV. Para más detalles sobre los requisitos para utilizar una aplicación WINDEV con el usuario editor de reportes, véase [Distribución de informes y consultas](../Presentation/3088002.md).

Para más detalles sobre las fuentes de datos de reporte, véase [Fuentes de datos de reporte](../WDChamp/1011055.md).
<a name="NOTE2_2"></a>


### Funciones para modificar registros
<a name="funciones_para_modificar_registros_ELTPARAGRAPHE000052"></a>

Las funciones para modificar los registros ([HAdd](../WDLang4/3044147.md), [HModify](../WDLang4/3044042.md), [HDelete](../WDLang4/3044018.md), ...) no pueden ser llamadas en la versión de usuario del editor de reportes.
<a name="NOTE2_3"></a>


### Cristales acoplables
<a name="cristales_acoplables_ELTPARAGRAPHE000068"></a>

En el usuario editor de reportes, los paneles acoplables le permiten acceder únicamente:

- las principales características del usuario editor de reportes.

- los elementos accesibles (informes, consultas,...) de su proyecto.




Para más detalles sobre los paneles acoplables, véase [Cristales acoplables](../Editeurs/2027001.md).
<a name="NOTE2_4"></a>


### Plantillas personalizadas de informes tema
<a name="plantillas_personalizadas_informes_tema_ELTPARAGRAPHE000083"></a>

La versión de usuario de la editor de reportes no permite crear sus propias plantillas de informes de tema.

Para más detalles sobre las plantillas reporte tema, véase [Creación de una plantilla de reporte](../Editeurs/2016005.md).
<a name="NOTE2_5"></a>


### Modificación de los informes encontrados en un componente externo
<a name="modificacion_los_informes_encontrados_componente_externo_ELTPARAGRAPHE000095"></a>

"Reportes y Consultas" no permite al usuario final modificar un reporte encontrado en un componente externo.

Para más detalles sobre los componentes externos, véase [Componente externo](../Editeurs/2014011.md).
<a name="NOTE2_6"></a>


### Modificación de los informes encontrados en una biblioteca externa
<a name="modificacion_los_informes_encontrados_una_biblioteca_externa_ELTPARAGRAPHE000107"></a>

"Reports and Queries" no permite al usuario final modificar un reporte encontrado en una biblioteca externa.

Para más detalles sobre los componentes, véase [Biblioteca](../Editeurs/2030022.md).

## Novedades de la actualización 1 (versión 280075)
<a name="novedades_actualizacion_1_version_280075_ELTTEXTE000356"></a>
**Entorno**

- Nueva plantilla táctil (controles más grandes, ideales para aplicaciones táctiles). 




**Editor de ventanas**

- **Espaciar**: Ahora puede espaciar una selección de controles, es decir, aumentar el espacio horizontal y vertical entre los controles.. Atención: la posición de los controles junto a la selección no se modifica. 

- **Ampliar ventanas**: Ahora puede ampliar una selección, una ventana o todo el proyecto seleccionando un factor de escala. Este factor de escala se aplica a los controles de la ventana, a las fuentes y a los estilos (opcional). También puede crear una plantilla"ampliada" . Esto permite aplicar la ampliación simplemente aplicando este nuevo plantilla. 

- **Edit control**: Un nuevo tipo de Edición control está disponible: Markdown en solo lectura. Este tipo de Edición control le permite utilizar el formato Markdown. Los usuarios podrán seleccionar y desplazarse por el contenido.

- **Modelos**: Actualización optimizada de plantillas. 




**SCM / GIT**

- Conexión a un servidor Git a través de SSH (sin volver a solicitar nombre de usuario o contraseña). 

- Más rápido SCM: Las siguientes acciones son ahora más rápidas: 

	- Utilización de SCM en la nube. 

	- Comprobación de elementos en. 

	- Crear ramas. 

	- Abrir el administrador SCM. 

	- Eliminación de archivos. 

	- Eliminación de directorios. 

	- Acciones en proyectos con un gran número de archivos. 







**WLanguage** 

- **DisplayAsMarkdown**: Indica si una control Edit muestra o no su contenido en formato Markdown.




**Características específicas de Android** 

- MarkdownToPDF: Convierte una cadena (o buffer) Markdown en un buffer que contiene un documento PDF.

- Nuevo nivel API: Cuando se genera una aplicación para Android, el APK o AAB resultante contiene un nivel API de destino (o TargetSdkVersion) del SDK Android. Las nuevas solicitudes se dirigen ahora al nivel 33 de API. 

- Notificaciones Push: **NotifPushSend** ahora admite notificaciones que utilizan el FCM v1 API.

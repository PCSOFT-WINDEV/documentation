


## TreePosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un control TreeView de un elemento especificado (nodo u hoja) o devuelve el nombre del primer elemento mostrado en un control TreeView.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of first element displayed in the "TREE_TVRecipe" TreeView control
ResElementName = TreePosition(TREE_TVRecipe)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre del primer elemento que aparece en un control TreeView

`<Result> = TreePosition(<TreeView control> [, <Duplicate identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

ruta completa del primer elemento mostrado en el control TreeView. 
Este resultado tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).

**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Duplicate identifier>: (Booleano opcional)`**

Especifica el modo de gestión de los duplicados en los elementos de &lt;Resultado&gt;:

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): no se realiza ninguna gestión.

- <u><u><u><u>True</u></u></u></u>: los nombres de los elementos ruta se construyen para contener su identificador. Para obtener más información, consulte [Gestión de duplicados en los controles de la vista de árbol](../WDChamp/1013053.md).


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta.


<a name="SYNTAX2"></a>

### Mostrando una control TreeView de un elemento específico

`<Result> = TreePosition(<TreeView control> , <Node path>)`
---

**`<Result>: (Cadena de caracteres)`**

ruta completa del primer elemento mostrado en la control TreeView tras la llamada a **TreePosition**. 
Este resultado tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Node path>: (Cadena de caracteres)`**

ruta completa del nodo que se visualizará en primera posición en el control TreeView. 
Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).

## Observaciones


### Visualización de un elemento al inicio de la control TreeView (sintaxis 2)
<a name="visualizacion_elemento_inicio_control_treeview_sintaxis_2_ELTPARAGRAPHE000108"></a>

El elemento visualizado en primer lugar puede diferir del solicitado. De hecho, si no hay suficientes elementos mostrados debajo del elemento &lt;Camino de nodos&gt;, este elemento no puede ser mostrado al principio de control TreeView

Por ejemplo, el nodo "Cold starters" no se puede visualizar al principio de control TreeView: no hay suficientes elementos mostrados debajo de este nodo.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ArbrePosition.gif)

<a name="NOTE0_2"></a>


### Seleccionar el elemento de la fila &lt;Índice&gt;
<a name="seleccionar_elemento_fila__ELTPARAGRAPHE000120"></a>

Para seleccionar el elemento en fila &lt;Index&gt;, utilice:

- la asignación directa: 
	
	```txt
	<TreeView Control> = Subscript
	```
Si se visualiza la fila en el control TreeView, se selecciona el elemento correspondiente.

- la función [TreeSelectPlus](../WDLang1/3018002.md):
	
	```txt
	TreeSelectPlus(<TreeView Control>, <Full Element Path>)
	```

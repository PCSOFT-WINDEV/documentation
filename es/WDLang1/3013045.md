
## SelectColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana de selección de color mientras se selecciona el color especificado.  



Hasta la versión 17, se abre la ventana estándar para la selección del color de Windows:<br>![Selector de color estándar de Windows](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SELECTCOULEUR.gif)


A partir de la versión 18, se muestra una ventana más moderna para la selección de colores (según las opciones seleccionadas)..  <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Selectcouleur18.gif)


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Observación: El recogedor corresponde al recogedor de color suministrado con el JRE.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Value is int 
SelectionResult is boolean

Value =  RGB(10, 17, 69)
Info("Initial color: " + Value)
SelectionResult = SelectColor(Value, scInit)
Info("Final color: " + Value)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SelectColor(<Initial value> [, <Options> [, <Custom Colors>]])`
---

**`<Result>: (booleano)`**

Salir de la ventana para la selección del color: 

- <u><u><u><u>True</u></u></u></u> si el usuario ha hecho clic en el botón OK,

- <u><u><u><u>False</u></u></u></u> en caso contrario (botón Cancelar).




**`<Initial value>: (Integro)`**

Color inicial seleccionado en la ventana de selección de color. 

Atención: No se debe utilizar directamente una constante de color (*LightRed*, etc.) o la función [RGB](../WDLang1/3029012.md). Puede utilizar un **variable** igual a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)), 

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)), 

- a [color predefinido de WLanguage](../WDLang5/3010002.md).




No hacer: 


```wl
SelectionResult is boolean = SelectColor(RGB(10, 17, 69), scInit)
```


Hacer: 


```wl
Value is int = RGB(10, 17, 69)
SelectionResult is boolean = SelectColor(Value, scInit)
```


**Cuando la ventana de selección de color se cierra**, &lt;Valor inicial&gt; contiene el valor del color seleccionado si el usuario ha validado la ventana con el botón OK.

**`<Options>: (Constante opcional o combinación opcional de constantes)`**

Opciones para mostrar la ventana de selección de color


|   |   |
| --- | --- |
| *sscHelp* | Habilita el botón de ayuda.<br><br>Ventana mostrada: Ventana estándar para la selección del color de Windows. |
| *scPersoColor* | <br><br><br><br>- Si se especifica &lt;Colores personalizados&gt;, se muestra la ventana de selección de colores estándar de Windows y se activa el botón para crear colores personalizados.<br><br>- Si no se especifica &lt;Colores personalizados&gt;, se muestra la ventana de selección de colores.<br><br><br> |
| *scSolidColor* | Sólo se permiten los colores no sombreados.<br><br>Ventana mostrada: Ventana de selección de color. |
| *scInit*<br>(Valor predeterminado) | El color seleccionado por defecto en la ventana de selección de color corresponde al color pasado como parámetro a la función **SelectColor**.<br><br>Ventana mostrada: Ventana de selección de color. |
| *scNotPersoColor* | El botón de creación de color personalizado se atenúa.<br><br>Ventana mostrada: Ventana estándar para la selección del color de Windows. |
| *scSystem* | Muestra la ventana estándar para la selección del color de Windows. |
| *scAnyColor* | Todos los colores están permitidos.<br><br>Ventana mostrada: Ventana de selección de color. |


Observación: Cuando se combinan constantes, la nueva ventana de selección de color se muestra sólo si todas las características solicitadas están disponibles en esta ventana.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta.

**`<Custom Colors>: (Dirección opcional)`**

Dirección de un array de 16 enteros que contiene los códigos RGB de los colores personalizados. Si no se especifica este parámetro, se proponen 16 tonos de gris.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta.

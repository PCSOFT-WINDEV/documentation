
## PanelSaveConfiguration( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda la configuración actual de los controles del panel acoplable de una ventana.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the current configuration
sConf is string = PanelSaveConfiguration()


// Restore the configuration saved in sConf
PanelLoadConfiguration(sConf)
IF ErrorOccurred THEN ToastDisplay(ErrorInfo())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PanelSaveConfiguration()`
---

**`<Result>: (Cadena de caracteres)`**

Configuración de todos los control Panel acoplable en la ventana actual.

Esta configuración puede ser guardada por [fSaveText](../WDLang1/3036046.md) o [SaveParameter](../WDLang1/3025046.md).

Esta configuración puede ser restaurada por [PanelLoadConfiguración](../WDLang1/1000023995.md). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La siguiente información se guarda para cada control Panel acoplable en la ventana: 

- Tamaño,

- Posición,

- Adosado / Separado,

- Visible / Invisible

- Colapsado / expandido. 




Para guardar/restablecer la posición de un panel acoplable en la ventana, utilice [Posición del panel](../WDLang1/1000023992.md) y [PanelAttach](../WDLang1/1000023990.md).

Las siguientes constantes se utilizan para administrar los controles de TreeView:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **tizButton** | Tipo de superficie flotante: Se muestra el botón"+/-" delante de la leyenda del elemento TreeView. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizCaption** | Tipo de superficie flotante: Título elementos. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizCheckmark** | Tipo de superficie flotante: Marca de verificación del elemento. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizImage** | Tipo de superficie flotante: Image se muestra delante de la leyenda del elemento. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizLeft** | Tipo de superficie flotante: El espacio se encuentra a la izquierda de la leyenda del elemento. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizOut** | Tipo de superficie flotante: Espacio que se encuentra debajo de los elementos de la vista de árbol. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tizRight** | Tipo de superficie flotante: El espacio se encuentra a la derecha de la leyenda del elemento. | [TreeInfoXY](../WDLang1/3018026.md) |
| **treeBranchID** | Tipo de información solicitada: Identificador del elemento encontrado en (X, Y). | [TreeInfoXY](../WDLang1/3018026.md) |
| **treeBranchName** | Tipo de información solicitada: Nombre del elemento encontrado en (X, Y). | [TreeInfoXY](../WDLang1/3018026.md) |
| **treeLineNumber** | Tipo de información solicitada: Número de la fila que se encuentra en (X, Y). | [TreeInfoXY](../WDLang1/3018026.md) |
| **treeOriginScreen** | Si se especifica esta opción, el origen de las coordenadas especificadas corresponde a la pantalla (y no a la control TreeView). | [TreeInfoXY](../WDLang1/3018026.md) |
| **treeZone** | Tipo de información solicitada: Tipo de área flotante. | [TreeInfoXY](../WDLang1/3018026.md) |
| **tvAcceptDuplicate** | Posición de la hoja en relación con los elementos del mismo nivel: los duplicados son aceptados. | [TreeAdd](../WDLang1/3018012.md), [TreeMoveItem](../WDLang1/3018024.md) |
| **tvAfter** | Posición de la hoja insertada: hoja insertada después del elemento "hermano" especificado. | [TreeInsert](../WDLang1/3018023.md) |
| **tvAll** | Nivel de elementos a tener en cuenta: todos los "niños" son tenidos en cuenta. | [TreeCount](../WDLang1/3018009.md) |
| **tvAlphaSort** | Posición de la hoja en relación con los elementos del mismo nivel: hoja insertada o movida según el orden alfabético. | [TreeAdd](../WDLang1/3018012.md), [TreeMoveItem](../WDLang1/3018024.md) |
| **tvBefore** | Posición de la hoja insertada: hoja insertada antes del elemento "hermano" especificado. | [TreeInsert](../WDLang1/3018023.md) |
| **tvCollapse** | Estado del nodo: nodo colapsado | [TreeStatus](../WDLang1/3018015.md) |
| **tvDefault** | Tipo de imagen associated con la hoja: imagen predeterminada. | [TreeAdd](../WDLang1/3018012.md), [TreeInsert](../WDLang1/3018023.md), [TreeModify](../WDLang1/3018013.md) |
| **tvDown** | Tipo de tipo: en orden alfabético inverso. | [TreeSort](../WDLang1/3018020.md) |
| **tvError** | Status de nodo/Tipo de elemento: o elemento no encontrado. | [TreeStatus](../WDLang1/3018015.md), [TreeTypeItem](../WDLang1/3018014.md) |
| **tvExpand** | Estado del nodo: nodo expandido | [TreeStatus](../WDLang1/3018015.md) |
| **tvFirst** | Posición de la hoja en relación a los elementos del mismo nivel / Indica el elemento "hijo" solicitado: hoja insertada o movida al primer punto de nivel / Primer elemento hijo. | [TreeAdd](../WDLang1/3018012.md), [TreeMoveItem](../WDLang1/3018024.md)<br>[TreeGiveChild](../WDLang1/3018010.md) |
| **tvLast** | Posición de la hoja en relación con los elementos del mismo nivel: hoja insertada o movida hasta el último punto del nivel. | [TreeAdd](../WDLang1/3018012.md), [TreeMoveItem](../WDLang1/3018024.md) |
| **tvLeaf** | Tipo de elemento: elemento hoja. | [TreeTypeItem](../WDLang1/3018014.md), [TreeFind](../WDLang1/3018030.md) |
| **tvNext** | Indica el "niño" solicitado: siguiente elemento hijo. | [TreeGiveChild](../WDLang1/3018010.md) |
| **tvNode** | Tipo de elemento: node element. | [TreeTypeItem](../WDLang1/3018014.md), [TreeFind](../WDLang1/3018030.md) |
| **tvOneLevel** | Nivel de elementos a tener en cuenta: sólo se tienen en cuenta los elementos "hijo" inmediatos. | [TreeCount](../WDLang1/3018009.md), [TreeFind](../WDLang1/3018030.md) |
| **tvUp** | Tipo de tipo: ordenar por orden alfabético. | [TreeSort](../WDLang1/3018020.md) |

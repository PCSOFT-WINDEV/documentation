


## &lt;Tabla&gt;.ExpandAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se expande:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la jerarquía completa (todos los nodos) de una control Tabla TreeView.

- todas las pausas de una control Tabla. 




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las pausas no están disponibles para controles Tabla en esta versión.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Expands all nodes found in the "TVT_Schedule" TreeView Table control
TVT_Schedule.ExpandAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.ExpandAll()`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este nombre puede corresponder a:

- el nombre de control Tabla TreeView para el que se expandirán todos los nodos.

- el nombre de la control Tabla para la que hay que ampliar todas las roturas.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000223"></a>

La función **&lt;Tabla&gt;.ExpandAll** puede utilizarse en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Tabla TreeView.

- un control Tabla rellenado mediante programación. 

- un control Tabla con fuente de datos en memoria.  




Esta función no se puede utilizar en controles Tabla archivo de datos (acceso directo).
<a name="NOTE0_2"></a>


### Control Tabla TreeView
<a name="control_tabla_treeview_ELTPARAGRAPHE000251"></a>

Cuando se llama a **&lt;Tabla&gt;.ExpandAll**, si la rama especificada y/o uno de sus hijos tienen un Procedure asociado a través de la propiedad [AddChildDelayed](../Proprietes/1000020927.md), se ejecutará este Procedure.

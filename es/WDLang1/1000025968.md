


## HTMLControlValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para identificar o modificar una control en un formulario HTML contenido en una control de visualización HTML. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF HTMLControlValue(HTM_Display, "LOGIN") <> "" THEN
	HTMLControlValue(HTM_Display, "LOGIN", NetworkUser())
END
```

```wl
HTMLControlValue(HTM_Sites, "input4")
IF ErrorOccurred THEN
	Trace(ErrorInfo()) // ex: The <"input4"> control does not exist in the page.
	RETURN
ELSE
	HTMLControlValue(HTM_Sites, "input4", sControlValue)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Identificar el valor de una control en un formulario HTML

`<Result> = HTMLControlValue(<HTML Display control> , <Control in the page>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de la control en la pantalla HTML control,

- Cadena vacía ("") si se produce un error (por ejemplo, si la control no se encuentra en la página). En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML utilizada por control. 

**`<Control in the page>: (Cadena de caracteres)`**

Nombre de la control contenida en la página HTML mostrada por el Display HTML control. 


<a name="SYNTAX2"></a>

### Modificar el valor de una control en un formulario HTML

`HTMLControlValue(<HTML Display control> , <Control in the page> , <Value>)`
---

**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML utilizada por control. 

**`<Control in the page>: (Cadena de caracteres)`**

Nombre de la control contenida en la página HTML mostrada por el Display HTML control. Si no se encuentra la control, se producirá un error no fatal. 

**`<Value>: (Cadena de caracteres)`**

El valor de control será modificado. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Importante** : Para que esta función se ejecute en el contexto de la página HTML mostrada en la control, la control debe ser accesible mediante JavaScript. No debería haber restricciones de acceso en esta página.

- Esta función permite automatizar los procesos en la página Web que se muestra en un Display HTML control.

- La control se busca por su nombre y luego por su ID. El nombre de la control (o su ID) puede conocerse editando el código fuente de la página HTML.

- Atención: Los nombres HTML distinguen entre mayúsculas y minúsculas.

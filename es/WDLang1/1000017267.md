


## MenúInsertarMenú (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un menú antes que otro menú en una ventana. El menú insertado puede contener otros menús, opciones de menú, separadores, ...

Para añadir un menú al final de un menú existente, utilice [MenuAddMenu](../WDLang1/1000017266.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert a new menu
MenuInsertMenu(_Menu, "MNU_Schedule", "MNU_Monitoring", "Project monitoring")
// Add an option into the new menu
MenuAddOption("MNU_Monitoring", "MNU_Prj", "Sales management", Sales_Mgt)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un menú (posición definida por un nombre de menú)

`<Result> = MenuInsertMenu(<Source menu> , <Insert menu> , <Menu to create> , <Caption>)`
---

**`<Result>: (Integro)`**



- Posición del menú en el sub-menú, 

- -1 si se produce un error. 




**`<Source menu>: (Nombre del menú)`**

Nombre del menú al que se añadirá el nuevo menú. El nuevo menú se añadirá en la posición especificada.

**`<Insert menu>: (Nombre del menú)`**

Nombre del menú utilizado como posición base para la inserción. El nuevo menú se insertará antes de esta posición.

**`<Menu to create>: (Cadena de caracteres)`**

Nombre del menú para crear e insertar. Este nombre se utilizará para manipular el menú mediante programación. Si este nombre corresponde a un menú que ya existe, se produce un error fatal.

**`<Caption>: (Cadena de caracteres)`**

Texto del nuevo menú. Esta leyenda se mostrará en la ventana.

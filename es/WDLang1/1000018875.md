
## SysIconDeleteImage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra el Icon que superpone la aplicación Icon en la barra de tareas. Esta Icon fue mostrada por [SysIconAddImage](../WDLang1/1000018874.md). 

**Importante**: La barra de tareas debe estar configurada para mostrar el modo "Iconos grandes".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// During the process, adds an hourglass icon over the icon of the application
SysIconAddImage("hourglass.ico")
HourGlass()

Long_process()

HourGlass(False)

// Delete the icon at the end of the process
SysIconDeleteImage()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysIconDeleteImage()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el ícono superpuesto,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

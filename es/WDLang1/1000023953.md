


## EditAddToken (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Token a un "Texto Token" Editar control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [All recipients] 
EditDeleteAllToken(EDT_Recipient)
FOR EACH mail OF tabMail
	EditAddToken(EDT_Recipient, mail.Name, mail.Address)
END
```

```wl
// Syntax with the Token type
EditDeleteAllToken(EDT_Recipient)
FOR EACH mail OF tabMail
	MyToken is Token
	MyToken.Caption = mail.Address
	MyToken.Value = mail.Name
	EditAddToken(EDT_Recipient, MyToken)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadiendo un Token directamente

`<Result> = EditAddToken(<Edit control> , <Value> [, <Caption>])`
---

**`<Result>: (Integro)`**



- Índice de agregado Token,

- 0 si se produce un error:

	- Si el control Edición no es un control "Fichero de texto" (la propiedad [TokenEnabled](../Proprietes/1000023782.md) está establecida en <u><u><u><u>False</u></u></u></u>). 

	- Si se ha detectado un duplicado y la propiedad [TokenAllowDuplicate](../Proprietes/1000023784.md) tiene el valor <u><u><u><u>False</u></u></u></u>.







**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" Editar control. 

**`<Value>: (Cadena de caracteres)`**

Valor de la Token. 

**`<Caption>: (Cadena de caracteres opcional)`**

Pie de foto de la Token. Si no se especifica este valor, corresponderá al valor de la Token.


<a name="SYNTAX2"></a>

### Añadiendo un Token a través del tipo Token

`<Result> = EditAddToken(<Edit control> , <Token>)`
---

**`<Result>: (Integro)`**



- Índice de agregado Token,

- 0 si se produce un error:

	- Si el control Edición no es un control "Fichero de texto" (la propiedad [TokenEnabled](../Proprietes/1000023782.md) está establecida en <u><u><u><u>False</u></u></u></u>). 

	- Si se ha detectado un duplicado y la propiedad [TokenAllowDuplicate](../Proprietes/1000023784.md) tiene el valor <u><u><u><u>False</u></u></u></u>.







**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" Editar control. 

**`<Token>: (Ficha Variable)`**

Nombre del [Token](../WDLang1/1000023942.md) WLanguage Variable que describe el Token a añadir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La Token se añade en la posición del cursor. Si el cursor se coloca entre dos Token, el nuevo Token se insertará entre ellos. 

- Para agregar un Token después de todos los demás Token s, use [EditInsertToken](../WDLang1/1000023954.md). 

- El control Campo de entrada [StoredValue](../Proprietes/2510127.md)Property devuelve los valores almacenados para los tokens, separados por los caracteres  (Retorno de carro).

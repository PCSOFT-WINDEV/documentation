
## fResourceDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ruta de los recursos de sólo lectura de la aplicación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The function returns the content of the file
fLoadBuffer(fResourceDir() + ["\"] + "data.bin")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fResourceDir()`
---

**`<Result>: (Cadena de caracteres)`**

ruta completa de los recursos de sólo lectura que se encuentran en la aplicación. Corresponde al directorio ".app" de la aplicación. No se puede escribir en este directorio.


## fTrackStopAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene todas las tareas de supervisión en curso sobre archivos y directorios. La tarea de supervisión podría haberse iniciado con [fTrackFile](../WDLang1/1000020522.md) o [fTrackDirectory](../WDLang1/1000020523.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stop all current monitoring tasks
IF NOT fTrackStopAll() THEN
	Error("No monitoring task has been started.")
ELSE
	Info("Monitoring stopped successfully.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fTrackStopAll()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se detuvieron todos los seguimientos activados previamente, 

- <u><u><u><u>False</u></u></u></u> en caso contrario (ningún seguimiento encontrado).




## Funciones de gestión de fuentes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las fuentes (fuentes de dibujo, de impresión, etc.):  



|   |   |
| --- | --- |
| [dFont](../WDLang1/3029020.md) | Declara la fuente que utilizará la función [dText](../WDLang1/3029032.md): <br><br>- en un control Imagen, <br><br>- en una variable de tipo [Image](../WDLang1/1000019650.md), <br><br>- en una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo),<br><br>- en una variable de tipo [picLayer](../WDLang1/1000024604.md). <br><br><br> |
| [FontCreate](../WDLang1/3060002.md) | Crea un tipo de letra Definition. |
| [FontList](../WDLang1/3060003.md) | Devuelve la lista de fuentes válidas instaladas en el ordenador actual (TrueType, Postscript, ...). |
| [FontSelect](../WDLang1/3060004.md) | Abre la ventana estándar para la selección de la fuente y selecciona la fuente especificada. |
| [gFont](../WDLang1/3029047.md) | Construye una secuencia de caracteres que describen la fuente que se utilizará para el texto. |
| [gFontBold](../WDLang1/3029061.md) | Construye una secuencia de caracteres utilizados para especificar si el texto que sigue a la llamada a esta función está en negrita o no. |
| [gFontItalic](../WDLang1/3029062.md) | Construye una secuencia de caracteres que se utiliza para especificar si el texto que sigue a la llamada a esta función está en cursiva o no. |
| [gFontSize](../WDLang1/3029064.md) | Construye una secuencia de caracteres que se utiliza para especificar el tamaño de la fuente para un texto que se encuentra en una cadena gráfica. |
| [gFontUnderlined](../WDLang1/3029063.md) | Construye una secuencia de caracteres para especificar si el texto que sigue a la llamada a esta función está o no subrayado. |
| [grAxisTitleFont](../WDLang3/3042059.md) | Modifica la fuente utilizada para los títulos de los ejes del gráfico. |
| [grLabelFont](../WDLang3/3042012.md) | Modifica la fuente utilizada para las etiquetas de categoría de un gráfico. |
| [grLegendFont](../WDLang3/3042037.md) | Modifica la fuente utilizada para la leyenda del gráfico. |
| [grTitleFont](../WDLang3/3042048.md) | Modifica la fuente utilizada para el título del gráfico. |
| [iCreateFont](../WDLang5/3046045.md) | Crea una nueva fuente de impresión. |
| [iDidotFont](../WDLang5/3046016.md) | Selecciona la unidad que se utilizará para la altura de las fuentes (creada por [iCreateFont](../WDLang5/3046045.md)): Punto DIDOT o punto PICA. |
| [iFont](../WDLang5/3046014.md) | Selecciona la fuente Default. |
| [iSelectFont](../WDLang5/3046055.md) | Crea una nueva fuente de impresión desde la ventana estándar para la selección de fuentes.... |

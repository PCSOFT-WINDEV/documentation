


## &lt;Procesamiento de palabras&gt;.UndoCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de acciones que se pueden deshacer en una control Procesador de texto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the number of actions that can be undone in the control
Info(WP_MyDocument.UndoCount())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.UndoCount()`
---

**`<Result>: (Integro)`**

Número de acciones que se pueden deshacer.

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.



<a name="XComponent"></a>

## Componente:
wd280mdl.dll

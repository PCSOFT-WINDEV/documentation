


## Funciones DDE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar DDE:



|   |   |
| --- | --- |
| [DDEConnect](../WDLang1/3028015.md) | Establece una conexión DDE entre el programa actual y el destinatario según un tema determinado. |
| [DDEDisconnect](../WDLang1/3028003.md) | Elimina una conexión entre el programa actual y un destinatario. |
| [DDEError](../WDLang1/3028006.md) | Devuelve el status de tiempo de ejecución reporte de una función DDE. |
| [DDEEvent](../WDLang1/3028009.md) | Asocia un procedimiento WLanguage con un DDE Event: cuando el evento DDE ocurre, el asociado WLanguage Procedure se ejecuta automáticamente. |
| [DDEExecute](../WDLang1/3028008.md) | Envía un comando que se ejecutará en el servidor. |
| [DDEItem](../WDLang1/3028007.md) | Identifica el artículo afectado por un DDE Event. |
| [DDELink](../WDLang1/3028010.md) | Crea un enlace directo con un dato. |
| [DDERecipient](../WDLang1/3028001.md) | Identifica el destinatario de una conexión DDE. |
| [DDERetrieve](../WDLang1/3028017.md) | Recupera los datos enviados por un programa (el destinatario de la conexión para el objeto especificado). |
| [DDESend](../WDLang1/3028004.md) | Envía un dato al programa conectado vía DDE. |
| [DDEStart](../WDLang1/3028013.md) | Inicia la ejecución de un programa (un ejecutable por ejemplo) desde la aplicación actual. |
| [DDEString](../WDLang1/3028002.md) | Devuelve la información recuperada por [DERetrieve](../WDLang1/3028017.md). |
| [DDETopic](../WDLang1/3028019.md) | Identifica al sujeto de la conversación associated con una conexión DDE. |
| [DDEUnLink](../WDLang1/3028005.md) | Interrumpe un enlace entre un elemento y un dato. |
| [DDEUpdate](../WDLang1/3028011.md) | Modifica un dato de relacionado. |
| [DDEWarmLink](../WDLang1/3028014.md) | Crea un enlace cálido con un dato. |

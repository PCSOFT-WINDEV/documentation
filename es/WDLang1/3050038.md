
## BinaryOR( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el resultado del OR lógico (a nivel de bits) entre dos valores.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the result of a logical OR
ResBinaryOR = BinaryOR(6, 6)  // Returns 6
ResBinaryOR = BinaryOR(6, 2)  // Returns 6
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BinaryOR(<Value 1> , <Value 2>)`
---

**`<Result>: (Integro)`**

Resultado de la lógica OR.

**`<Value 1>: (Integro)`**

Primer valor a probar.

**`<Value 2>: (Integro)`**

Segundo valor a probar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Cálculo de bit a bit
<a name="calculo_bit_bit_ELTPARAGRAPHE000065"></a>

**BinaryOR** realiza el cálculo a nivel de bits entre &lt;Valor 1&gt; y &lt;Valor 2&gt; según el siguiente tabla:


|   |   |   |
| --- | --- | --- |
|   | **0** | **1** |
| **0** | 0 | 1 |
| **1** | 1 | 1 |


<a name="NOTE0_2"></a>




### Operador binario
<a name="operador_binario_ELTPARAGRAPHE000092"></a>

**BinaryOR** es equivalente al operador binario **|**. Para obtener más información, consulte [Operadores binarios](../Motscles/1512009.md).

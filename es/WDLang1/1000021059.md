
## Constantes del tablero de instrumentos

Las siguientes constantes se utilizan para gestionar los controles del Tablero:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **dashMove** | Cambiar a modo de edición: el usuario puede mover los widgets en el control Cuadro de mando. | [DashMode](../WDLang1/1000020956.md) |
| **dashResize** | Cambiar a modo de edición: el usuario puede cambiar el tamaño de los widgets en el control Cuadro de mando. | [DashMode](../WDLang1/1000020956.md) |
| **dashNoModif** | Cerrar el modo de edición: el usuario no puede modificar los widgets en el control Cuadro de mando. | [DashMode](../WDLang1/1000020956.md) |
| **dashNoReorganization** | Redimensiona el widget sin reorganizar los widgets adyacentes. | [DashResizeWidget](../WDLang1/1000021286.md),<br>[DashMoveWidget](../WDLang1/1000021289.md) |
| **dashDelete** | Cambiar a modo de edición: el usuario puede ocultar los widgets en el control Cuadro de mando. | [DashMode](../WDLang1/1000020956.md) |
| **dashUnitCell** | La unidad utilizada para el tamaño del widget o cuando se mueve el widget es un número de celdas. | [DashResizeWidget](../WDLang1/1000021286.md),<br>[DashMoveWidget](../WDLang1/1000021289.md)<br>[DashGetWidgetPosition](../WDLang1/1000022498.md) |
| **dashUnitPixel** | La unidad utilizada para el tamaño del widget o cuando se mueve el widget es un número de píxeles. | [DashResizeWidget](../WDLang1/1000021286.md),<br>[DashMoveWidget](../WDLang1/1000021289.md)<br>[DashGetWidgetPosition](../WDLang1/1000022498.md) |


## WHelp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un archivo o una página de ayuda en formato HLP o CHM.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open "NotePad.hlp" on the cut/paste page
WHelp("C:\Windows\NotePad.hlp", "PASTING TEXT")
```


<a name="Example2"></a>

```wl
// Open "NotePad.hlp" on the page whose context number is 35 
// or on the index if the number does not exist
WHelp("C:\Windows\NotePad.hlp", 35)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Abrir la ayuda sobre la ayuda (WinHelp.hlp)

`WHelp()`
---


<a name="SYNTAX2"></a>

### Abrir un archivo de ayuda

`WHelp(<Name of help file> [, <Keyword>])`
---

**`<Name of help file>: (Cadena de caracteres)`**

Nombre y ruta del archivo de ayuda (.HLP o.CHM). Si no se encuentra el archivo de ayuda, la función no tiene efecto.

**`<Keyword>: (Cadena de caracteres opcional)`**

Palabra clave utilizada para encontrar la página de ayuda. 
Este parámetro corresponde a una cadena vacía (""") o no se especifica, se abre el índice del archivo de ayuda y el usuario podrá seleccionar la palabra clave que desee.. Si el archivo de ayuda no tiene índice, el archivo se abre en la primera página de ayuda.


<a name="SYNTAX3"></a>

### Mostrando una página de ayuda específica

`WHelp(<Name of help file> , <Context number>)`
---

**`<Name of help file>: (Cadena de caracteres)`**

Nombre y ruta del archivo de ayuda (.HLP o.CHM). Si no se encuentra el archivo de ayuda, la función no tiene efecto.

**`<Context number>: (Integro)`**

Número de contexto associated con la página de ayuda y con el control para el que se debe mostrar la página. El número de ayuda associated con un control es devuelto por [NumHelp](../WDLang1/3013027.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Cambiar el idioma y el archivo de ayuda
<a name="cambiar_idioma_archivo_ayuda_ELTPARAGRAPHE000065"></a>

En el editor de ayuda de WINDEV, el nombre del archivo de ayuda puede construirse a partir de un identificador numérico correspondiente a [Nación](../WDLang1/3054014.md).

En este caso, para iniciar el archivo de ayuda correspondiente al idioma actual, todo lo que tiene que hacer es utilizar el siguiente código:


```wl
// Start the help in CHM format
WHelp("MyFile" + NumToString(Nation(), "03d") + ".chm")
```

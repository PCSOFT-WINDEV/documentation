
## Chrono (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Chrono** permite definir todas las características avanzadas de un cronómetro. Las características del cronómetro se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
c is Chrono
ChronoStart(c)
...
// Stopwatch code
...
ChronoEnd(c)
```





<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Chrono
<a name="propiedades_especificas_descripcion_variables_tipo_chrono_ELTPARAGRAPHE000043"></a>

Las siguientes permiten manipular una variable de tipo **Chrono**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Started | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> si ya se inició el cronómetro,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Value | Duration | Tiempo transcurrido. <br><br>Observación: Esta propiedad corresponde al resultado de la función [ChronoValue](../WDLang1/3027033.md). <br><br>**Esta propiedad es de solo lectura**. |


<a name="NOTE0_2"></a>


### Funciones WLanguage que manipulan el tipo Chrono
<a name="funciones_wlanguage_que_manipulan_tipo_chrono_ELTPARAGRAPHE000085"></a>

- Las siguientes funciones utilizan el tipo **Chrono**: 
	


|   |   |
| --- | --- |
| [ChronoEnd](../WDLang1/3027032.md) | Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [ChronoStart](../WDLang1/3027031.md)). |
| [ChronoPause](../WDLang1/1000018829.md) | Pone en pausa un cronómetro para medir el tiempo de un proceso. |
| [ChronoReset](../WDLang1/1000020474.md) | Detiene y reinicia un cronómetro en marcha. |
| [ChronoResume](../WDLang1/1000018830.md) | Reinicia un cronómetro que se detuvo con las funciones [ChronoEnd](../WDLang1/3027032.md) o [ChronoPause](../WDLang1/1000018829.md). |
| [ChronoStart](../WDLang1/3027031.md) | Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. |
| [ChronoValue](../WDLang1/3027033.md) | Indica el tiempo transcurrido (en milisegundos) desde que se llamó a la función [ChronoStart](../WDLang1/3027031.md). |

- Las siguientes funciones utilizan el tipo **Chrono** (sintaxis con prefijo): 
	


|   |   |
| --- | --- |
| [&lt;Chrono&gt;.End](../WDLang1/1000023789.md) | Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md)). |
| [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md) | Pone en pausa un cronómetro para medir el tiempo de un proceso. |
| [&lt;Chrono&gt;.Reset](../WDLang1/1000023791.md) | Detiene y reinicia un cronómetro en marcha. |
| [&lt;Chrono&gt;.Resume](../WDLang1/1000023792.md) | Reanuda un cronómetro detenido previamente con las funciones [&lt;Chrono&gt;.Fin](../WDLang1/1000023789.md) o [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md). |
| [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) | Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. |

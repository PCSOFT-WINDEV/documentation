


## ChronoReset (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene y reinicia un cronómetro en marcha. El valor del tiempo transcurrido se pierde.  




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code of BTN_Reset
// Resets the stopwatch
ChronoReset()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Detener y reiniciar el cronometraje iniciado con un número de cronómetro

`ChronoReset([<Stopwatch number>])`
---

**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro a reiniciar. Si no se especifica este parámetro, el valor predeterminado es 1.

### Detener y reiniciar el cronometraje iniciado con una variable de tipo Chrono

`ChronoReset(<Stopwatch>)`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.

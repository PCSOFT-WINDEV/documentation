


## &lt;Hoja de cálculo&gt;.SelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona una o más celdas de un control Hoja de cálculo que se encuentra en una ventana..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PSHEET_Spreadsheet.SelectMinus()
PSHEET_Spreadsheet.SelectPlus("A2", "B5")
PSHEET_Spreadsheet.StyleSelection(psheetBackgroundColor, LightRed)
```
<a name="Example2"></a>

```wl
// Select a range of cells to merge them
PSHEET_Spreadsheet.SelectPlus("B1", "D1")
// Merge the cells 
AAFExecute(PSHEET_Spreadsheet, aafSpreadsheetMerge)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.SelectPlus([<Start cell> [, <End cell>]])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Start cell>: (Cadena de caracteres opcional)`**

Nombre de la celda a seleccionar. Este parámetro está en formato &lt;Letra de la columna&gt;&lt;Número de la fila&gt;. Por ejemplo: "A32". 

Si este parámetro corresponde: 

- al nombre de una columna (por ejemplo,"A"), la columna especificada se selecciona por completo. 

- al número de una fila ("2" por ejemplo), el fila especificado se selecciona por completo. 

- a una cadena vacía ("""), se seleccionan todas las celdas encontradas en la control Hoja de cálculo.


Si no se especifica este parámetro, se seleccionan todas las celdas, así como las cabeceras de columna y fila.. 

Si se especifica el parámetro &lt;Celda fin&gt;, &lt;Celda inicio&gt; corresponde a la primera celda que se seleccionará de un rango de celdas.  

Si no se especifica &lt;Celda final&gt;, sólo se selecciona &lt;Celda inicial&gt;..

**`<End cell>: (Cadena de caracteres opcional)`**

Nombre de la celda correspondiente al final del rango de celdas a seleccionar. Este parámetro está en formato &lt;Letra de la columna&gt;&lt;Número de la fila&gt;. Por ejemplo: "A32".  

Si no se especifica este parámetro, sólo se selecciona &lt;Start Cell&gt;.. 

Si &lt;Celda final&gt; corresponde a una celda situada antes de &lt;Celda inicial&gt;, se invierten las coordenadas y se realiza la selección.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La selección realizada se añade a la selección actual. 

- La selección la realizan [&lt;Hoja de cálculo&gt;.BorderSelection](../WDLang1/1000023717.md), [&lt;Hoja de cálculo&gt;.TypeAndMaskSelection](../WDLang1/1000023745.md), [&lt;Hoja de cálculo&gt;.StyleSelection](../WDLang1/1000023739.md). También puede utilizar [AAFExecute](../WDLang1/1000022099.md) (por ejemplo, con la constante *aafSpreadsheetMerge* para fusionar las celdas seleccionadas).

- Para seleccionar una sola celda y deseleccionar las demás, utilice la siguiente sintaxis:
	
	```wl
	PSHEET_Spreadsheet = "A3"
	```


- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

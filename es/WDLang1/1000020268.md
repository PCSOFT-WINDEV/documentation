


## VideoGenerateThumbnail( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera una miniatura de la foto o el vídeo especificado. Esta miniatura se genera en formato JPEG. 

Observaciones: 

- Si las fotos se van a utilizar en aplicaciones Android, le recomendamos trabajar con miniaturas y no con las fotos originales tomadas por el dispositivo. 

- Esta función es equivalente a la función [CameraGenerateThumbnail](../WDLang1/1000022456.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000020273&name=videogeneratethumbnail_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
sPhoto is string = VideoStartApp(viPictureCapture)
IF NOT ErrorOccurred THEN
	bThumbnail is boolean
	bThumbnail = VideoGenerateThumbnail(sPhoto, ...
			fExtractPath(sPhoto, fDirectory + fFileName) + "_thumb.jpg")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Generar una miniatura de tamaño automático de una foto o un vídeo

`<Result> = VideoGenerateThumbnail(<Source> , <Destination>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la generación. La altura de la miniatura será de 96 píxeles y su anchura se calculará según la relación anchura/altura de la fuente Image o vídeo. 

- <u><u><u><u>False</u></u></u></u> si se produce un error. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>False</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source>: (Cadena de caracteres)`**

ruta completa de foto o video para la cual se generará una miniatura. Para un vídeo, el archivo debe estar en una ubicación compartida del dispositivo (en la tarjeta sd, por ejemplo). 

**`<Destination>: (Cadena de caracteres)`**

ruta completa del archivo donde se guardará la miniatura.


<a name="SYNTAX2"></a>

### Generar la miniatura de una foto con un tamaño determinado

`<Result> = VideoGenerateThumbnail(<Source> , <Destination> , <Width> , <Height>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la generación,

- <u><u><u><u>False</u></u></u></u> si se produce un error. En este caso, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>False</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source>: (Cadena de caracteres)`**

ruta completa de foto o video para la cual se generará una miniatura. Para un vídeo, el archivo debe estar en una ubicación compartida del dispositivo (en la tarjeta sd, por ejemplo). 

**`<Destination>: (Cadena de caracteres)`**

ruta completa del archivo donde se guardará la miniatura.

**`<Width>: (Integro)`**

Ancho (en píxeles) de la miniatura para generar. 

**`<Height>: (Integro)`**

Altura (en píxeles) de la miniatura a generar.

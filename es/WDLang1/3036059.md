
## fDirectoryExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un directorio.

Observación: Esta función es equivalente a la función [fDirExist](../WDLang1/1000021899.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyDirectory is string = "C:\DataDirectory"
IF fDirectoryExist(MyDirectory) = True THEN 
	Info("The directory " + MyDirectory + " exists on disk")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fDirectoryExist(<Directory name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el directorio,

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, el nombre del directorio corresponde al nombre de un archivo).




**`<Directory name>: (Cadena de caracteres)`**

Nombre y apellidos y ruta completa (o relativa) del directorio. No se permiten los caracteres comodín (\*,?).

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) El directorio ruta corresponde a un ruta relativo en relación al directorio actual. El directorio actual corresponde al directorio de datos de la aplicación. 
**Recordatorio**: Sólo se pueden utilizar los directorios que se encuentran en el directorio de datos associated con la aplicación.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para comprobar si un archivo existe, utilice [fFileExist](../WDLang1/3036015.md).

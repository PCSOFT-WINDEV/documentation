


## &lt;Imagen&gt;.EndDrawing (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los dibujos realizados en la control Imagen desde la última llamada a [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md). 
**Observación**: Llamar a la función **&lt;Imagen&gt;.EndDrawing** es opcional. Los recursos se liberan automáticamente cuando se cierra la ventana o Page.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Destroy the drawing found in the "IMG_Drawing" control
IMG_Drawing.EndDrawing()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Image control>.EndDrawing()`
---

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

<a name="NOTE0_1"></a>

## Observaciones


### Uso en una sintaxis con prefijo
<a name="uso_una_sintaxis_con_prefijo_ELTPARAGRAPHE000379"></a>

Se debe llamar a la función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) antes de utilizar otras funciones de dibujo. La función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) permite definir las opciones de dibujo utilizadas en el control Imagen. 

Para un control Imagen: 

- Las funciones de dibujo trabajan en una copia ("bitmap") de la imagen. Se pueden utilizar las funciones de dibujo de Windows (con la función [API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md)), pero estas funciones deben utilizar el DC (Device Context) devuelto por la función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md).

- No se debe llamar a las funciones **&lt;Imagen&gt;.EndDrawing** y [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) en el mismo proceso.
	Si se llama a las funciones **&lt;Imagen&gt;.EndDrawing** y [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) en el mismo proceso, no se mostrará el dibujo: se borrará automáticamente.

### Funciones de dibujo, barras de desplazamiento y zoom automáticos
<a name="funciones_dibujo_barras_desplazamiento_zoom_automaticos_ELTPARAGRAPHE000540"></a>

La opción "Barras de desplazamiento y zoom automáticos", disponible en los controles Imagen, no es compatible con la función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md). Si se utiliza esta opción en el control Imagen, la función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) inmoviliza la imagen en el control. El zoom y las barras de desplazamiento también se inmovilizan. El comportamiento estándar se restablece al llamar a la función **&lt;Imagen&gt;.EndDrawing**.


## MCIMsgErr( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el título del último error ocurrido en una función MCI (Media control Interface).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Caption of the last error
ResErrorMsg = MCIMsgErr()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIMsgErr()`
---

**`<Result>: (Cadena de caracteres)`**

Leyenda del último error que se produjo en una función MCI.  



<a name="XComponent"></a>

## Componente:
wd280std.dll

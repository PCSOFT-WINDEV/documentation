


## InitParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la gestión de los valores persistentes. Especifica la ubicación utilizada para almacenar:

- los valores de los controles persistentes.

- el tamaño y la posición de los mandos y de las ventanas.

- los valores gestionados por [SaveParameter](../WDLang1/3025046.md) y [LoadParameter](../WDLang1/3025047.md).




**Observación**: La inicialización de los valores persistentes es "global".. Por lo tanto, si un componente inicializa la gestión de valores persistentes, la gestión de valores persistentes se inicializará para la aplicación que utiliza el componente.

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Parameters of persistent values 
// in a .XML file that depends on the user 
InitParameter(paramXML, "Param_" + USER + ".xml")
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Parameters of persistent values
// in a registry key with a version number
InitParameter(paramRegistry, ProjectInfo(piRegistry) + "\3.51")
// other possibility:
InitParameter(paramRegistry, "HKEY_CURRENT_USER\Software\DDSoft\DDTool\3.51")
```

```wl
MyString is string
MyString = fLoadText("MyFile.xml")

```

## Sintaxis

`<Result> = InitParameter(<Initialization mode> [, <Source of values>])`
---

**`<Result>: (Cadena de caracteres)`**

Modo de inicialización utilizado antes de llamar a la función **InitParameter**. &lt;El resultado tiene el siguiente formato: 

```txt
<Initialization Mode> + TAB + <Source of Values>
```


**`<Initialization mode>: (Constante de tipo Integer)`**

Tipo de copia de seguridad para los datos persistentes:



|   |   |
| --- | --- |
| *paramIni* | Los parámetros se guardarán en la sección ] Global[ de un archivo en formato .INI.<br>Si se selecciona esta opción:<br><br>- el archivo INI se llama &lt;nombre del proyecto&gt;.INI por Default. Este archivo se crea en el directorio del ejecutable. <br>	<br>	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este archivo se crea en la carpeta "$HOME/.local/share".<br><br>- &lt;Fuente de valores&gt; corresponde al nombre del archivo .INI a utilizar.<br>	Si no se especifica este directorio, el archivo se crea en el directorio ejecutable.<br>	<br>	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este archivo se crea en la carpeta "$HOME/.local/share".<br><br><br> |
| *paramRegistry* | Los parámetros se guardarán en el registro.<br>Si se selecciona esta opción:<br><br>- por defecto, la entrada utilizada corresponde al resultado de ProjectInfo( piRegistry) (que significa HKEY_CURRENT_USER\\Software\\&lt;Empresa&gt;\\&lt;Nombre del proyecto&gt;)<br>	&lt;Empresa&gt; corresponde a la empresa especificada al crear el ejecutable.. Si no se especifica este nombre, &lt;Compañía&gt; corresponde a "Aplicaciones WINDEV".<br><br>- en modo Test, el ruta utilizado en el registro es el siguiente: HKEY_CURRENT_USER\\SOFTWARE\\PC SOFT\\WinDev\\28\\TestMode\\&lt;Nombre del proyecto&gt;. <br><br>- &lt;Fuente de valores&gt; corresponde a la entrada del registro a utilizar.<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) &lt;Fuente de valores&gt; será ignorada. Si se elige este modo para almacenar los datos persistentes, el soporte utilizado para la copia de seguridad depende del sistema operativo en el que se ejecuta la aplicación.. | *paramXML* | Los parámetros se guardarán en un archivo en formato.XML.<br>Si se selecciona esta opción:<br><br>- el archivo XML se llama &lt;nombre del proyecto&gt;.XML por Default. Este archivo se crea en el directorio del ejecutable. <br>	<br>	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este archivo se crea en la carpeta "$HOME/.local/share".<br><br>- &lt;Fuente de valores&gt; corresponde al nombre del archivo .XML a utilizar. <br>	Si no se especifica este directorio, el archivo se crea en el directorio ejecutable. <br>	<br>	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este archivo se crea en la carpeta "$HOME/.local/share".<br><br><br> |



**`<Source of values>: (Cadena de caracteres opcional)`**

Este parámetro depende del modo de inicialización seleccionado.

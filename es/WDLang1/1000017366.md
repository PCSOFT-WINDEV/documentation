


## Push (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Empuja un elemento a la pila. El elemento se inserta en la parte superior de la pila.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyStack is stack of int

// Push the values 1, 2 and 3
Push(MyStack, 1)
Push(MyStack, 2)
Push(MyStack, 3)

// Display the trace: 3, 2, 1
x is int
WHILE Pop(MyStack, x)
	Trace(x)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Push(<Stack> , <Value>)`
---

**`<Stack>: (Pila Variable)`**

Nombre de la variable de tipo [apilar](../WDLang1/1514022.md) que se utilizará.

**`<Value>: (Tipo de elementos manejados por la pila)`**

Valor que se añadirá en la parte superior de la pila.

## Observaciones


### Modo de funcionamiento multihilo

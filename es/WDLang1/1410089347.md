
## France.CheckSIRET (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número de SIRET francés (Système d'Identification du Répertoire des Etablissements).

ATENCIÓN: Esta información es específica de la administración francesa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MySIRETNumber is a string = "33031827000064"
France.CheckSIRET(MySIRETNumber) // Returns True

// Check a control
EDT_SIRETNumber = "ABC"
France.CheckSIRET(EDT_SIRETNumber) // Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = France.CheckSIRET(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número de SIRET válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número de SIRET a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

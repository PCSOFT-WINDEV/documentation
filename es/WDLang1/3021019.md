## Presentación
<a name="presentacion_ELTTEXTE000140"></a>
Los cuadros de diálogo avanzados son ventanas que le permiten comunicarse con el usuario. Estas ventanas se utilizan para gestionar:

- una interrogativa directiva (función: el usuario responde a una pregunta a través de los botones que contienen el texto de la acción a realizar.

- la entrada inmediata: el usuario introduce el valor deseado en el cuadro de diálogo.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Interrogativas directas
<a name="interrogativas_directas_ELTTEXTE000164"></a>
La seguridad de los datos de la aplicación depende de las respuestas dadas por los usuarios finales a las preguntas formuladas en los mensajes.. En la mayoría de los casos, estas preguntas son más o menos claras. Por ejemplo, "¿Quieres Cancel la eliminación actual? " con el botón "Sí" o "No".

Para simplificar la escritura de estos mensajes, siempre que se llame a una función utilizada para dialogar con el usuario final, WINDEV propone hacerlo:

- seleccionar un mensaje existente

- crear un nuevo mensaje a través de una simple ventana.




El código correspondiente se genera automáticamente cuando se introducen las diferentes características del mensaje.

**Observación**: El código generado utiliza [coloquio](../WDLang1/3021015.md).

## Base de datos de mensajes
<a name="base_datos_mensajes_ELTTEXTE000212"></a>
WINDEV propone una lista de mensajes. Estos mensajes no se pueden borrar. Constituyen la base de datos de mensajes. 

Todos los mensajes nuevos se añaden automáticamente a la base de datos de mensajes.. Por Default, la base de datos de mensajes se encuentra en el directorio "Personal\\Messages" de WINDEV. 

Este directorio puede ser modificado en las opciones de WINDEV: 

- en la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de WINDEV". 

- en la pestaña "Directorio", especifique el directorio deseado para la base de datos de mensajes.




Para eliminar un mensaje de la base de datos de mensajes:

1. Visualizar el editor de código.

2. En la pestaña "Código", en el grupo "Idiomas", despliegue "Traducir cadenas" y seleccione "Interrogativas directas".

3. Seleccione el mensaje que desea eliminar y haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeSupprimeFavoris.gif).

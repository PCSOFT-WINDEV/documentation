


## Charact (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el carácter que corresponde al código ASCII especificado. El código ASCII utilizado corresponde al estándar ANSI utilizado en Windows.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ASCIIChar = Charact(49)    // Returns "1"
ASCIIChar = Charact(87)    // Returns "W"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Charact(<ASCII code to convert>)`
---

**`<Result>: (Cadena de caracteres)`**

Carácter que corresponde al código ASCII especificado.

**`<ASCII code to convert>: (Entero)`**

Valor a convertir (entre 0 y 255). Si el valor no es válido, se produce un error de WLanguage.

## Observaciones


### Alternativas a la función **Charact**
<a name="alternativas_funcion_docparampagetitleshort_ELTPARAGRAPHE000064"></a>

Las siguientes constantes predefinidas evitan el uso de la función **Charact**:

| Constante | Valor | Función |
| --- | --- | --- |
| CR | Charact(13) + Charact(10) | Retorno de carro |
| TAB | Charact(9) | Tabulador |
| ESC | Charact(27) | Tecla Esc |
| SPC | Charact(160) | Espacio no separable |


## IntegerPart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la parte entera de un número. La parte entera de un número es el mayor número entero relativo menor o igual que el número. Por ejemplo:

- la parte entera de 3,5 es 3.

- la parte entera de -3,5 es -4.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Integer part of 16.999
ResIntegerPart = IntegerPart(16.999)
// Returns 16
// To get 17, use the Round function
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IntegerPart(<Number>)`
---

**`<Result>: (Real)`**

Parte entera solicitada. Corresponde al mayor número entero menor o igual a &lt;Número>.

**`<Number>: (Real)`**

Número a utilizar.


## AutoRunAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite iniciar automáticamente una aplicación WINDEV cuando se produce una Event específica. La aplicación a ejecutar puede corresponder a la aplicación actual.

**Observación**: [AutoRunDelete](../WDLang1/3019001.md) se utiliza para cancelar la ejecución de automatic de una aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Automatically run the "MyApp" application
// at startup
AutoRunAdd(alStartup, "MyApp", fExeDir() + "\CustomerManagement.exe")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AutoRunAdd(<Run event> , <Application> , <Executable>)`
---

**`<Run event>: (Constante de tipo Integer)`**

Event que desencadena la ejecución de la aplicación:


|   |   |
| --- | --- |
| *alStartup* | Ejecutar la aplicación cuando se inicia el PC. |



**`<Application>: (Cadena de caracteres)`**

Nombre de la aplicación a ejecutar. Este nombre se utilizará en [AutoRunDelete](../WDLang1/3019001.md).

**`<Executable>: (Cadena de caracteres)`**

Nombre y ruta completa del ejecutable de la aplicación a ejecutar. Este ejecutable está en el ordenador actual.
Este parámetro puede contener el comando Line que se pasará al ejecutable.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nombre y directorio del ejecutable
<a name="nombre_directorio_del_ejecutable_ELTPARAGRAPHE000049"></a>

- Al utilizar la función [ExeInfo](../WDLang1/3035001.md) con la constante *exeName*, se devuelve el nombre completo del ejecutable de la aplicación actual.

- [fExeDir](../WDLang1/3036049.md) devuelve el directorio que contiene el ejecutable de la aplicación actual.



<a name="NOTE0_2"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000064"></a>

**Atención**: La función **AutoRunAdd** no devuelve ningún código de error. Para determinar si esta función ha generado un error, utilice la función [ErrorOccurred](../WDLang1/3087001.md) Variable. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).

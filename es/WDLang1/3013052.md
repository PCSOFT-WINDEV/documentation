


## TraceStart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el ventana de salida o guarda la información de la traza en un archivo de texto.

**Observaciones**:

- La función **TraceStart** no tiene efecto si la ventana de salida ya está abierta.

- La función **TraceStart** es opcional. La primera llamada a [rastro](../WDLang1/3013050.md) o a [TraceBuild](../WDLang1/1000021683.md) abre automáticamente el ventana de salida.


## Sintaxis

`TraceStart([<Characteristic> [, <Trace file>]])`
---

**`<Characteristic>: (Constante opcional)`**

Características de la traza:



|   |   |
| --- | --- |
| *trCreation* | Borra el archivo de rastreo. |
| *trDisable* | Todas las llamadas a [rastro](../WDLang1/3013050.md) serán ignoradas. |
| *trFile* | El ventana de salida no se mostrará más y la información encontrada en el ventana de salida se guardará en el archivo llamado &lt;Archivo de rastreo&gt;..<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo esta constante está disponible. Por Default, este archivo se crea en el directorio temporal de UNIX y se llama "Trace_&lt;Project>.txt".<br> |
| *trTopMost* | El ventana de salida se mostrará Above todas las ventanas del sistema.<br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *trWindow* <br>(Valor predeterminado) | El ventana de salida se muestra en la Corner superior izquierda Corner de la pantalla.<br><br><br> |
| *trWindow* + *trFile* | El ventana de salida se mostrará y el contenido del ventana de salida también se guardará en el archivo llamado &lt;Archivo de rastreo&gt;..<br> |



**`<Trace file>: (Cadena de caracteres opcional)`**

Nombre y ruta completa del archivo donde se guardará la información encontrada en la ventana de salida. Si no se especifica este parámetro, el archivo de traza se creará en la raíz de la unidad "C" y su nombre tendrá el siguiente formato "Trace_&lt;ProjectName>.txt".

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Si no se especifica este parámetro, el archivo de traza se creará en el directorio UNIX temporal como "Trace_&lt;ProjectName>.txt".![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Si no se especifica este parámetro, el archivo se crea en el directorio actual de la aplicación (conocido a través de [fCurrentDir](../WDLang1/3036022.md)) con el nombre "Trace_&lt;NombredelProyecto&gt;.txt". 
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Si no se especifica este parámetro, el archivo se crea en el directorio de datos de la aplicación (devuelto por [fDataDir](../WDLang1/3036001.md)) y se denomina "Trace_&lt;NombreProyecto&gt;.txt".

## Observaciones


### Condiciones de uso según las plataformas
<a name="condiciones_uso_segun_las_plataformas_ELTPARAGRAPHE000182"></a>

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) La función **TraceStart** permite guardar la información solicitada solo en un archivo. Por Default, este archivo se crea en el directorio temporal de UNIX y se llama "Trace_&lt;Project>.txt".
<a name="NOTE0_2"></a>


### Cierre de la ventana de salida
<a name="cierre_ventana_salida_ELTPARAGRAPHE000219"></a>

[TraceEnd](../WDLang1/3013044.md) se utiliza para cerrar el ventana de salida.

Si se especifica la constante *trFile*, [TraceEnd](../WDLang1/3013044.md) deja de guardar en el archivo de seguimiento.

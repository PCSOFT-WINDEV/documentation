


## Posición de la barra de herramientas (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el modo de visualización de una barra de herramientas control.

**Recordatorio**: Una barra de herramientas control puede "flotar" o puede ser acoplada a los bordes de la ventana por uno de sus lados
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ToolbarPosition(TBAR_Toolbar1) = barUndocked THEN
	ToolbarDock(TBAR_Toolbar1, barLeft)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ToolbarPosition(<Toolbar control>)`
---

**`<Result>: (Constante opcional)`**

Posición de la barra de herramientas control: 


|   |   |
| --- | --- |
| *barDown* | La barra de herramientas control está acoplada en la parte inferior de la ventana principal. |
| *barLeft* | La barra de herramientas control está acoplada a la izquierda de la ventana principal. |
| *barRight* | La barra de herramientas control está acoplada a la derecha de la ventana principal. |
| *barUndocked* | Barra de herramientas "flotante" control. |
| *barUp*<br>(Valor predeterminado) | La barra de herramientas control está acoplada en la parte superior de la ventana principal. |



**`<Toolbar control>: (Nombre del control)`**

Nombre de la barra de herramientas control a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La palabra clave [MiVentana](../Motscles/1511004.md) no debe utilizarse: 

- en los eventos associated con la barra de herramientas control. 

- en los eventos associated con los controles en la barra de herramientas.

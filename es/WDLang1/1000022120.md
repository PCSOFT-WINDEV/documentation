
## WinEdSaveNote( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta las notas reposicionables creadas por el usuario a un archivo. Entonces, estas notas pueden ser importadas: 

- directamente en el editor. 

- mediante [WinEdLoad](../WDLang1/1000021849.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Export my notes]
IF WinEdSaveNote("Export.wdnote") = False THEN
	ErrorInfo()
END

// Click [Import my notes]
WinEdLoad(WIN_NoteWindow, "Export.wdnote")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WinEdSaveNote(<Backup file> [, <Window>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardaron las notas reposicionables,

- <u><u><u><u>False</u></u></u></u> si se produjo un error o si no se ha creado ninguna nota. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


Observación: Si no se ha creado ninguna nota, el valor de la variable [ErrorOccurred](../WDLang1/3087001.md) es <u><u><u><u>False</u></u></u></u>. 

**`<Backup file>: (Cadena de caracteres)`**

Nombre del archivo donde se realizará la copia de seguridad. La extensión de este archivo es "wdnote".. 

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna que contiene las notas a guardar. Si no se especifica este parámetro, se exportan todas las notas creadas por el usuario en la aplicación.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**WinEdSaveNote** es equivalente a la función de exportación de notas automatic (AAF). La única diferencia es que el selector de archivos no está abierto.

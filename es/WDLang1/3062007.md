
## sComputeCrc16( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba una cadena de caracteres antes y después de la transmisión entre las aplicaciones WINDEV y los sitios WEBDEV.

Esta función se utiliza para verificar si una cadena de caracteres se transmitió correctamente entre las aplicaciones WINDEV y los sitios WEBDEV durante las transmisiones con riesgo de pérdida de información (por ejemplo, módem).. Para ello, calcule el valor "Crc16" antes de transmitir la cadena y compruebe su valor al recibir la cadena.: los dos valores deben ser idénticos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nCrcRead is int	// Control character read
LineRead is string	// String read
...
IF nCrcRead <> sComputeCrc16(LineRead) THEN
	Info("Error while retrieving the string")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = sComputeCrc16(<String>)`
---

**`<Result>: (Integro)`**

Valor de control de la cadena.

**`<String>: (Cadena de caracteres)`**

Cualquier cadena de caracteres.  






## Observaciones
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta función puede devolver un resultado diferente en la versión WINDEV. Para mejorar la reutilización del código, le recomendamos que utilice [sComputeCrc32](../WDLang1/3062015.md).

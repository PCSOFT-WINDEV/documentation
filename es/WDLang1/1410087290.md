


## Funciones de compresión/cifrado (sintaxis de prefijos)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para el cifrado y la compresión manipular, utilizando la sintaxis de prefijos:



|   |   |
| --- | --- |
| [.DecryptAsymmetric](../WDLang1/1410087282.md) | Descifra un buffer con un algoritmo de clave privada/pública según el cifrado RSA. |
| [.DecryptAsymmetricEnvelopped](../WDLang1/1410087283.md) | Realiza el cifrado asimétrico de un sobre que ha sido cifrado con [EncrypteAsymmetricEnveloped](../WDLang1/1410087280.md). |
| [.EncryptAsymmetric](../WDLang1/1410087279.md) | Cifra un buffer utilizando un algoritmo de cifrado asimétrico (RSA) que requiere una clave pública y una clave privada. |
| [.EncryptAsymmetricEnvelopped](../WDLang1/1410087280.md) | Realiza una encriptación asimétrica de un sobre. |
| [.EncryptStandard](../WDLang1/1410087281.md) | Cifra una cadena de caracteres o un búfer binario utilizando un algoritmo de cifrado simétrico (AES, DES, etc.). |

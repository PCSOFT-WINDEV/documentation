


## VideoListDriver( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los controladores de captura de vídeo instalados en el ordenador actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the video drivers installed
// Display them in LIST_DriverList
ListAdd(LIST_DriverList, VideoListDriver(viDirectShow))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = VideoListDriver([<Technology used>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de controladores de vídeo instalados en el equipo, en el siguiente formato:

```txt
<Name of driver 1> + TAB + <Version of driver 1> + CR + ... 
<Name of driver N> + TAB + <Version of driver N>
```


Donde: 

- &lt;Nombre del conductor 1&gt; ... &lt;Nombre del controlador N&gt; corresponden al nombre de los controladores de vídeo, 

- &lt;Versión del conductor 1&gt; ... &lt;Versión del conductor N&gt; corresponden a la versión del conductor.




**`<Technology used>: (Constante de tipo Integer)`**

Tipo de tecnología utilizada para listar los drivers: 


|   |   |
| --- | --- |
| *viDirectShow* | La tecnología utilizada: Mostrar directamente. <br><br>Esta constante debe utilizarse: <br><br>- para listar los controladores correspondientes a las diferentes cámaras instaladas (en una tableta, por ejemplo). <br><br>- para seleccionar un controlador utilizado en un control Conferencia ([StreamChangeDriver](../WDLang3/1000021613.md)).<br><br><br> |
| *viVideoForWindows*<br>(Valor predeterminado) | La tecnología utilizada: Video para Windows |






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para mostrar la lista de controladores de vídeo en un List Box o control Combo Box, utilice la siguiente sintaxis: 
	
	```txt
	ListAdd(<Name of List Box control>, <Result)
	```

	donde
	
	```txt
	ListAdd(<Name of List Box control>, VideoListDriver())
	```


- Esta función es equivalente a la función [CameraListDriver](../WDLang1/1000022457.md).

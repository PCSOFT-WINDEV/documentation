


## Hoja de cálculoInsertarColumna (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una columna vacía en una control Hoja de cálculo que se encuentra en una ventana.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Inserts several columns
FOR I = 2 TO 5
	SpreadsheetInsertColumn(PSHEET_Spreadsheet, I)
 END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetInsertColumn(<Spreadsheet control> [, <Column>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Column>: (Entero opcional)`**

Número de la columna ante la que se insertará la nueva columna. Por ejemplo, si este parámetro es set a 2, la nueva columna se insertará en la posición 2. Las columnas existentes se moverán a la derecha y las fórmulas se actualizarán.. 
Si no se especifica este parámetro y si se selecciona una columna en la control, la inserción se realizará antes de la columna seleccionada. De lo contrario (sin columna seleccionada), la función no tiene ningún efecto.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La operación de inserción se añade a la lista de operaciones de Cancel: el usuario puede deshacer esta operación a través del menú contextual o pulsando Ctrl + Z.

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

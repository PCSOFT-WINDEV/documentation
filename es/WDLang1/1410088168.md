


## DiagramSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona una forma en un editor Diagram control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DiagramSelectMinus(DIAGEDT_MyDiagram, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DiagramSelectMinus(<Diagram> [, <Shape index>])`
---

**`<Diagram>: (Nombre del control)`**

Nombre del editor de Diagram control a utilizar.  

**`<Shape index>: (Entero opcional)`**

Índice de la forma a deseleccionar (si hay varios elementos seleccionados). Si no se especifica este parámetro, deselecciona el elemento actual. 
Observación: Si la forma no existe en la Diagram, la función no tiene efecto.

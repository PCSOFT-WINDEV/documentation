
## ControlCurrent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del control actualmente en edición.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the control currently in edit
ResControl = ControlCurrent()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlCurrent()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la control actualmente en edición,

- Vaciar la cadena ("") si no hay ningún control en edición (ningún control tiene el foco). 




Si la control actualmente en edición no pertenece a la ventana que posee el código, la cadena devuelta tiene el siguiente formato: "&lt;nombre de la ventana&gt;.&lt;nombre de la control&gt;".
Si la control corresponde a una célula tabla, &lt;Resultado&gt; contiene el nombre de la columna.
Si el control pertenece a un supercontrol, &lt;Resultado&gt; tiene el siguiente formato: &lt;Supercontrol nombre&gt;.&lt;control nombre&gt;. 
**Atención**: El caso del nombre de control (caracteres en mayúsculas/minúsculas) corresponde al utilizado cuando se creó la control en el editor.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En esta versión no se procesa el caso de los controles pertenecientes a supercontroles.

<a name="NOTE0_1"></a>

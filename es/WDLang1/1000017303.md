


## BankHoliday (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si un día es festivo o no. Las fechas que corresponden a los días festivos se especifican con la función [BankHolidayAdd](../WDLang1/1000017304.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Define public holidays
BankHolidayAdd("0714")
// Check a public holiday
IF BankHoliday("19700714") = True THEN
	Info(DateToString("19700714", maskDateSystem) + " is a public holiday")
END
// Check a date in an Edit control
IF BankHoliday(EDT_Date) = True THEN
	Info(DateToString(EDT_Date, maskDateSystem) + " is a public holiday")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BankHoliday(<Date to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fecha especificada corresponde a un día festivo definido con la función [BankHolidayAdd](../WDLang1/1000017304.md), 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Date to check>: (Cadena de caracteres o variable de tipo Date)`**

Fecha a comprobar. Si la fecha corresponde a una cadena, debe estar en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).

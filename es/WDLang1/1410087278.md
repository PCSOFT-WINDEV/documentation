


## &lt;HTML Display control&gt;.FromEmail (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Importa el contenido HTML de una Email y sus imágenes en una control de visualización HTML. Las imágenes contenidas en la Email se muestran también en la control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read a message and display it in the HTML_View control
MyMessage is Email
EmailReadMessage(SessionID, MyMessage)
HTM_HTMLDisplay.ToEmail(MyMessage)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Display control>.FromEmail(<Email>)`
---

**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML utilizada por control.

**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable a mostrar en la pantalla HTML control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El contenido HTML del correo electrónico se asigna a la pantalla HTML control junto con sus imágenes. 

- La propiedad [WorkingDirectory](../Proprietes/1000026169.md) se asigna con el ruta del directorio temporal que contiene las imágenes associated con el Email.

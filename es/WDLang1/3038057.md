


## WinRateDDDW (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obtiene y establece la tasa de atenuación para las ventanas que utilizan la función OVI (Oscurecimiento de Ventanas Inactivas).

La función OVI se utiliza para atenuar automáticamente las ventanas desactivadas. Para obtener más información, consulte [Oscurecimiento de ventanas inactivas](../Editeurs/9000071.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Project initialization
// Change dimming rate
WinRateDDW(40)
// Enable the DDW feature for all the application windows
WinActivateDDW(True)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtención de la tasa de atenuación

`<Result> = WinRateDDW()`
---

**`<Result>: (Integro)`**

Tasa de atenuación actual (entre 0 y 100). La tasa de atenuación por defecto es de set a 18%.  


<a name="SYNTAX2"></a>

### Cambio de la tasa de atenuación

`<Result> = WinRateDDW(<Dimming rate>)`
---

**`<Result>: (Integro)`**

Tasa de atenuación inicial (entre 0 y 100).

**`<Dimming rate>: (Integro)`**

Nueva tasa de atenuación (entre 0 y 100). El valor máximo (100) se utilizará si el valor especificado es superior a 100. La función **WinRateDDDW** no tendrá ningún efecto si se indica un valor inferior a 0. La tasa de atenuación por defecto es de set a 18%.

<a name="NOTE0_1"></a>

## Observaciones


### Configuración de los ordenadores del usuario final
<a name="configuracion_los_ordenadores_del_usuario_final_ELTPARAGRAPHE000045"></a>

Dependiendo de la configuración de los ordenadores del usuario final, la función OVI puede no tener ningún efecto.

De hecho, la función OVI se ve afectada por las características de la tarjeta de vídeo, la velocidad del procesador, ....

Si se produce un problema, pida a los usuarios finales que actualicen el controlador de su tarjeta de vídeo.

**Observación**: Para funcionar, los ordenadores de los usuarios finales deben:

- tienen un modo de visualización superior a 256 colores.

- estar en modo de 32 ó 64 bits.

- estar ejecutando Windows 2000 o posterior.

- utilice el nivel máximo de "Aceleración de hardware" para la tarjeta de vídeo (propiedades avanzadas de visualización de el equipo).



<a name="NOTE0_2"></a>


### Activación de la función OVI (OVI)
<a name="activacion_funcion_ovi_ovi_ELTPARAGRAPHE000063"></a>

Para activar o desactivar la función OVI, utilice [WinActivateDDW](../WDLang1/3038056.md).


## DDEItem( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el artículo afectado por un DDE Event.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Equivalent to: Item = _DDE.Item
Item = DDEItem()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEItem([<Connection identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Artículo afectado por el DDE Event.

**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, **DDEItem** utiliza la última conexión gestionada.  



<a name="XComponent"></a>

## Componente:
wd280std.dll


## Parámetro description (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Las variables **del parámetro description** se utilizan para describir un parámetro mediante programación. 

Las características de este tipo son devueltas por varias propiedades de WLanguage. Este tipo de Variable es utilizado por [Sintaxis description](../WDLang1/1410089154.md) Variable s. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Description_report is Definition = GetTypeDefinition(RPT_CustomerOrders)
Syntax_description is Syntax Description = Description_report.DeclarationSyntax
// Iterates over the parameters of the report
FOR EACH x OF Syntax_description.Parameter
	Trace(x.Name)
END
```

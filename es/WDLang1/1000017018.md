
## SysListScreenResolution( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las posibles resoluciones de una pantalla.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds into a table containing 3 columns 
// the list of available resolutions for the main screen
ListAdd(TABLE_Resolution, SysListScreenResolution())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysListScreenResolution([<Screen name>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres que contiene la lista de resoluciones disponibles, cadena vacía ("") si se ha producido un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.

Esta cadena tiene el siguiente formato:

```txt
<Horizontal Resolution 1> + TAB + <Vertical Resolution 1> + TAB +
<Number of Bits per Pixels 1> + TAB + <Refresh Frequency> + CR + ... + 
<Horizontal Resolution N> + TAB + <Vertical Resolution N> + TAB + 
<Number of Bits per Pixels N> + TAB + <Refresh Frequency N>
```


**`<Screen name>: (Cadena de caracteres opcional)`**

Nombre de la pantalla para la que se solicitan las resoluciones disponibles. Si no se especifica este parámetro, se buscan las resoluciones disponibles para la pantalla principal. Si el nombre de la pantalla no existe, &lt;Result&gt; corresponde a una cadena vacía (""").

**Observación**: La lista de nombres de pantalla es devuelta por [Pantalla SysListScreen](../WDLang1/1000017014.md).




## TimelineDeleteTrack (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una pista de una control Línea de tiempo. 

**Atención**: Si el control Línea de tiempo es relacionado a una fuente de datos, la eliminación se realiza en el control solamente: los eventos relacionado a la pista no se borran de la fuente de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the SOUND track
TimelineDeleteTrack(TL_MYTIMELINE, "Sound")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TimelineDeleteTrack(<TimeLine control> , <Track>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó una pista,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo del que se borrará la pista.

**`<Track>: (Cadena de caracteres)`**

Nombre de la pista a borrar.


## ThreadPause( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pausa el hilo actual por el tiempo especificado.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ThreadPause(100)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ThreadPause(<Timeout>)`
---

**`<Timeout>: (Integro)`**

Tiempo de espera definido para el hilo (en centésimas de segundo). Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- una variable de tipo Duration,

- una duración en un formato legible (por ejemplo, "1s" o "10cs"). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se recomienda utilizar **ThreadPause** en lugar de [Multitask](../WDLang1/3015004.md) o [Timeout](../WDLang1/3015012.md) cuando se utilizan varios hilos (incluido el hilo principal de la aplicación) si el tiempo de espera no necesita Process acciones del usuario.

- En una aplicación de Servicio, **ThreadPause** debe sustituirse por [ServicioEspera](../WDLang1/1000018712.md).

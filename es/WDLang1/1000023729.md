


## &lt;Recalcular todo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recalcula todas las fórmulas de un control Hoja de cálculo encontrado en una ventana (incluyendo las fórmulas que utilizan funciones/procedimientos de WLanguage)..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HReadNext(TurnoverFile)
PSHEET_Spreadsheet.RecalculateAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.RecalculateAll()`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.



<a name="XComponent"></a>

## Componente:
wd280obj.dll

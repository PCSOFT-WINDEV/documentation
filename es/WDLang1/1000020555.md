
## SysIOSVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la versión de iOS utilizada por la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF SysIOSVersion() = "6.0" THEN
	// Process specific to the version 6.0
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysIOSVersion()`
---

**`<Result>: (Cadena de caracteres)`**

Versión actual del sistema expresada en formato "X.Y.

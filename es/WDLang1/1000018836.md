
## dbgAuditStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera y modifica el estado de la auditoría dinámica.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts the dynamic audit
dbgEnableAudit()
...
// Pauses the dynamic audit
dbgAuditStatus(AuditPause)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el estado de la auditoría dinámica

`<Result> = dbgAuditStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Situación actual de la auditoría dinámica: 


|   |   |
| --- | --- |
| *AuditDisabled* | La auditoría dinámica no fue habilitada. |
| *AuditEnabled* | La auditoría dinámica está habilitada. |
| *AuditPause* | La auditoría dinámica se interrumpe. |




<a name="SYNTAX2"></a>

### Modificación de la situación de la auditoría dinámica

`<Result> = dbgAuditStatus(<New status>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la actividad de auditoría dinámica antes de los cambios realizados por **dbgAuditStatus**. 


|   |   |
| --- | --- |
| *AuditDisabled* | La auditoría dinámica no fue habilitada. |
| *AuditEnabled* | La auditoría dinámica está habilitada. |
| *AuditPause* | La auditoría dinámica se interrumpe. |



**`<New status>: (Constante de tipo Integer)`**

Nuevo estado de la auditoría dinámica: 


|   |   |
| --- | --- |
| *AuditDisabled* | Detiene permanentemente la auditoría dinámica. Para volver a activarlo, tendrá que utilizar [dbgEnableAudit](../WDLang1/1000018835.md). |
| *AuditEnabled* | Habilita la auditoría dinámica (después de una pausa). |
| *AuditPause* | Detiene temporalmente la auditoría dinámica. |

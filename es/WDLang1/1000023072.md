
## Opción description (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Una variable de tipo **Option Description** permite manipular las características de una opción de combinación mediante programación.

Las características de este tipo son devueltas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the definition of d type
d is Definition
d = GetDefinition(FileAction)
// Displays the name of d type
Trace(d.Name)

// Solution 1: list the combination options
IF d.Type = wlCombination THEN
	FOR EACH MyElement OF d.Option
		Trace(MyElement.Name)	
	END
END
// Solution 2: list the combination options 
// (by using the Option Description type)
IF d.Type = wlCombination THEN
	D1 is Option Description
	FOR EACH D1 OF d.Option
		Trace(D1.Name)	
	END
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables description de la opción description
<a name="propiedades_especificas_las_variables_description_opcion_description_ELTPARAGRAPHE000031"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una opción de combinación:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Alias | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la opción es un alias de una opción o de una combinación de opciones existente,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**. |
| AssociatedValue | Todos los tipos | Valor de la opción associated. <br><br>**Esta propiedad es de solo lectura**. |
| Declaración | Variable de tipo [Definition](../WDLang1/1000019540.md) | Declaración de opciones. <br><br>**Esta propiedad es de solo lectura**. |
| denominar | Cadena de caracteres | Nombre de la opción. <br><br>**Esta propiedad es de solo lectura**. |
| Heredado | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la combinación se hereda de una combinación extendida,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**. |
| Valor | Todos los tipos | Valor de la opción. <br><br>**Esta propiedad es de solo lectura**. |

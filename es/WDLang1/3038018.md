


## Cerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una ventana WINDEV (y devuelve un valor si es necesario).

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta función solo está disponible para simplificar la Webificación de proyectos WINDEV. En un sitio WEBDEV, esta función tiene el mismo comportamiento que [Diálogo PageCloseDialog](../WDLang2/1000017166.md) o [ContextoCerrar](../WDLang2/3058003.md), según el caso. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the "WIN_Edit" window 
Close(WIN_Edit)
```


<a name="Example2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Close the "WIN_EditUsername" window 
// and retrieve several return values
Close(WIN_EditUsername, LastName, FirstName)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Close([<Window> [, <Returned value> [... [, <Returned value N>]]]])`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana a cerrar. Si no se especifica este parámetro o si corresponde a una cadena vacía ("""), se cierra la ventana actual.

**`<Returned value>: (Tipo correspondiente al valor devuelto (opcional))`**

Valor devuelto por la ventana cuando está cerrada. 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este valor se recupera sólo si la ventana fue abierta por [entablar](../WDLang1/3038035.md). 
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este valor se recupera sólo si la ventana fue abierta por [OpenMobileWindow](../WDLang1/1000021018.md) (o por [OpenChild](../WDLang1/3038021.md)).

Usted tiene la capacidad de regresar: 

- tipos simples (cadena de caracteres, booleano, ....). 

- tipos avanzados: estructura, estructura dinámica, clase, tipo avanzado, array, asociativo array, cola, pila, lista.






**`<Returned value N>: (Tipo correspondiente al valor devuelto (opcional))`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Valor N devuelto por la ventana cuando está cerrada (si se devuelven varios valores).  
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Este valor se recupera sólo si la ventana fue abierta por [entablar](../WDLang1/3038035.md). 


Usted tiene la capacidad de regresar: 

- tipos simples (cadena de caracteres, booleano, ....). 

- tipos avanzados: estructura, estructura dinámica, clase, tipo avanzado, array, asociativo array, cola, pila, lista.

## Observaciones


### Códigos ejecutados en el cierre
<a name="codigos_ejecutados_cierre_ELTPARAGRAPHE000152"></a>

- Al cerrar una ventana:

	1. La actual Process está permanentemente interrumpida.

	2. Se ejecuta la Event de cierre de la ventana.




- Si se llama a [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md) en el "cierre" de Event, la ventana no se cierra.

- Si se ejecuta la función **Cerrar** en un procedimiento llamado con la función [Timer](../WDLang1/3015008.md), se debe verificar la existencia de la ventana que se va a cerrar con las funciones [WinStatus](../WDLang1/3038030.md) o [CurrentWin](../WDLang1/3038033.md), de lo contrario todas las ventanas abiertas podrían cerrarse. Por ejemplo:
	
	```wl
	// Procedure called by timer
	Window is string 
	Window = CurrentWin()
	IF Window <> "Window name" THEN
		Close()
	END
	```




<a name="NOTE0_2"></a>


### Cierre según el tipo de ventana
<a name="cierre_segun_tipo_ventana_ELTPARAGRAPHE000180"></a>

- Si la ventana a cerrar es una ventana padre con ventanas hijas, estas ventanas hijas se cerrarán primero (su código de cierre se ejecuta).

- Si la ventana para cerrar es una ventana secundaria, la ventana principal se convierte en la ventana actual.. Se ejecuta el código de "ganancia de enfoque" de la ventana principal.



<a name="NOTE0_3"></a>


### Última ventana de proyecto
<a name="ultima_ventana_proyecto_ELTPARAGRAPHE000188"></a>

Si se cierra la última ventana, se ejecuta la Event de "Cierre" del proyecto antes de finalizar el programa.

### Valor devuelto a la función Abrir
<a name="valor_devuelto_funcion_abrir_ELTPARAGRAPHE000208"></a>

Se puede utilizar la función **Cerrar** para devolver uno o más valores a la función [entablar](../WDLang1/3038035.md).

Recordatorio: Un valor también puede ser devuelto por el [ReturnedValue](../Proprietes/2510134.md) Property (&lt;nombre de la ventana&gt;.*ReturnedValue*) ejecutado antes del **Cerrar**. En este caso, se puede devolver un valor individual. 

Ejemplos: 

- El usuario debe escribir una contraseña en una ventana. Esta contraseña se recupera cuando se cierra esta ventana y se devuelve a [entablar](../WDLang1/3038035.md):
	
	```wl
	MyPassword = Open(WIN_EditPassword)
	IF MyPassword <> "" THEN
		Info("Wrong password") 
	END
	
	// -- Click code on "OK" button in the "WIN_EditPassword" window
	// Users enter their password in the EDT_Password control 
	// and validates the window.
	Close("", EDT_Password)
	```


- El usuario debe introducir un nombre de usuario y una contraseña en una ventana. El nombre de usuario y la contraseña se recuperan cuando se cierra la ventana y se devuelve a [entablar](../WDLang1/3038035.md): 
	
	```wl
	(MyUsername, MyPassword) = Open(WIN_EditPassword)
	IF MyUsername <> "" OR MyPassword <> "" THEN 
		Info("Wrong username or password")
	END
	
	// -- Click code on "OK" button in the "WIN_EditPassword" window
	// Users enter their username and password 
	// in the EDT_Password control and validates the window.
	Close("", EDT_Username, EDT_Password)
	```

### Valor devuelto al cerrar una ventana secundaria
<a name="valor_devuelto_cerrar_una_ventana_secundaria_ELTPARAGRAPHE000248"></a>

La función **Cerrar** permite devolver un valor a la ventana padre que abrió la ventana hija.

Recordatorio: Un valor también puede ser devuelto por el [ReturnedValue](../Proprietes/2510134.md) Property (&lt;nombre de la ventana&gt;.*ReturnedValue*) ejecutado antes del **Cerrar**.

El valor devuelto por la ventana hija puede leerse en el Event "Cierre de una ventana hija" de la ventana que la abrió (la que llamó a [OpenMobileWindow](../WDLang1/1000021018.md) o [OpenChild](../WDLang1/3038021.md)): simplemente utilice el [ReturnedValue](../Proprietes/2510134.md) Property en la palabra clave [MiInfantilVentana](../Motscles/1511021.md).

Por ejemplo:

- En el código de cierre de la ventana hija, se utiliza [Cerrar](../WDLang1/3038018.md) para devolver un parámetro: 
	
	```wl
	Close("", MyParameter)
	```


- El nombre de la ventana hija, así como el valor devuelto, pueden ser recuperados en la Event "Cierre de una ventana hija" de la ventana que llama, utilizando la palabra clave [MiInfantilVentana](../Motscles/1511021.md): 
	
	```wl
	// Close a child window
	NameChildWindow = MyChildWindow.Name
	ValueChildWindow = MyChildWindow.ReturnedValue
	```




<a name="NOTE0_7"></a>


### Teclas Alt + F4
<a name="teclas_alt_ELTPARAGRAPHE000293"></a>

Presionar Alt + F4 puede desencadenar varios comportamientos:

- Si la ventana contiene un botón de "Abandono", se ejecuta el WLanguage Event associated con el botón de abandono.

- Si la ventana tiene un botón con Alt + F4 como atajo de teclado, se ejecuta el WLanguage Event associated con este botón.

- Si la ventana no contiene un botón "abandonar", o un botón con Alt + F4 como atajo de teclado, se ejecuta el Process de cierre de la ventana y se cierra la ventana (excepto si se llama a [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md)).

- Si se abre una ventana desde un programa en lenguaje externo, Alt + F4 devuelve "ESC" en WdKey.

### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000308"></a>La siguiente Line de código: 


```txt
WinStatus(<Window>, NotFound)
```
equivale a: 


```txt
Close(<Window>)
```

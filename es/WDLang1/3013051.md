


## CommandLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica y recupera los diferentes elementos del comando Line pasado como parámetro al programa actual.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The executable is called by the following command line:
// project.exe param1 /opt1=param2 "param31 param32" -opt2="param41 param42"
CommandLine()	// Returns the entire command line
CommandLine("")    // ERROR: CommandLine() and CommandLine("") are not equivalent
CommandLine(1)	// Returns 'param1'
CommandLine(2)	// Returns '/opt1 = param2'
CommandLine(3)	// Returns 'param31 param32'
CommandLine(4)	// Returns '-opt2 = "param41 param42"'
CommandLine(5)	// Returns ' '
CommandLine("opt1")	// Returns 'param2'
CommandLine("opt2")	// Returns 'param41 param42'
CommandLine("opt3")	// Returns ' '
CommandLine("opt3", "Empty")	// Returns 'Empty'
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperando todo el comando Line

`<Result> = CommandLine()`
---

**`<Result>: (Cadena de caracteres)`**

El comando Line pasó al ejecutable (sin el nombre del mismo).  


<a name="SYNTAX2"></a>

### Recuperando el valor del parámetro Nth en el comando Line

`<Result> = CommandLine(<Option index>)`
---

**`<Result>: (Cadena de caracteres)`**

Parámetro correspondiente al índice solicitado.

**`<Option index>: (Integro)`**

Índice de la opción a recuperar. 
**Observación**: El carácter de espacio corresponde al separador.  


<a name="SYNTAX3"></a>

### Recuperar el valor de un parámetro específico en el comando Line

`<Result> = CommandLine(<Option name> , <Default value>)`
---

**`<Result>: (Cadena de caracteres)`**

Valor correspondiente al parámetro solicitado.

**`<Option name>: (Cadena de caracteres)`**

Nombre de la opción a recuperar. Se aceptan las siguientes sintaxis:

- /&lt;nombre&gt; = valor

- &lt;nombre&gt; = valor




El carácter de espacio corresponde al separador. Para utilizar un espacio en el valor de un parámetro nombrado, el parámetro debe estar entre comillas ("&lt;Parámetro con espacio&gt;").
Ejemplo: project.exe param1 /opt1="param2 con espacios".

**`<Default value>: (Cadena de caracteres)`**

Valor de Default devuelto si la opción no se encuentra o no existe. 
Por Default, el valor devuelto por es una cadena vacía ("").  

Ejemplo:


```wl
CommandLine("select", "False")
```
Comandos:

- program.exe -> La función **CommandLine** devuelve "False".

- program.exe /select -> La función **CommandLine** devuelve "False".

- program.exe /select=abc -> La función **CommandLine** devuelve "abc".






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En modo de prueba, puede especificar el comando Line en el editor (en la pestaña "Proyecto", en el grupo "Modo de prueba", expanda "Modo de prueba" y seleccione "Configurar modo de prueba").

- A diferencia de la función **CommandLine**, la función [dbgStandardOutput](../WDLang1/3014020.md) permite al ejecutable devolver un valor a la aplicación que la inició (escritura en STDOUT).




## OrganizerSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un Appointment de un control Agenda.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an appointment into an Organizer control
nAppointmentIndex is int
nAppointmentIndex = OrganizerAddAppointment(ORG_MyOrganizer, ...
				"Meeting with sales team", ...
				DateSys() + "1400", DateSys() + "1600")
// Select the added appointment
OrganizerSelectPlus(ORG_MyOrganizer, nAppointmentIndex)
...
// Click the "Deselect" Button control
OrganizerSelectMinus(ORG_MyOrganizer)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Desmarcar una Appointment identificada por su índice

`OrganizerSelectMinus(<Organizer control> [, <Appointment index>])`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.




**`<Appointment index>: (Entero opcional)`**

Índice de la Appointment a deseleccionar. Los índices de Appointment comienzan a partir de 1.
Si no se especifica este parámetro, la Appointment que está actualmente seleccionada se deselecciona.
Si se especifica este parámetro y no coincide con el índice de la Appointment actualmente seleccionada en la control Agenda, la función no tiene efecto.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Deselección de una Appointment identificada por una Appointment Variable

`OrganizerSelectMinus(<Organizer control> [, <Appointment to deselect>])`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda a utilizar (control en una ventana). 

**`<Appointment to deselect>: (Opcional Appointment Variable)`**

Nombre de la [Appointment](../WDLang1/1000019244.md) que corresponde a la Appointment a deseleccionar. 
Si no se especifica este parámetro, la Appointment que está actualmente seleccionada se deselecciona.
Si se especifica este parámetro y no corresponde al Appointment actualmente seleccionado en el control Agenda, la función no tiene ningún efecto.

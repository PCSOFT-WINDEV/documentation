


## Funciones de gestión de controles Botón de opción
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles de los botones de radio:



|   |   |
| --- | --- |
| [RadioButtonAdd](../WDLang1/1000020598.md) | Añade una nueva opción a una control Botón de opción. |
| [RadioButtonCount](../WDLang1/1000021184.md) | Devuelve el número de opciones encontradas en una control Botón de opción. |
| [RadioButtonDelete](../WDLang1/1000020600.md) | Elimina una opción de un control Botón de opción. |
| [RadioButtonInsert](../WDLang1/1000020599.md) | Inserta una nueva opción en una control Botón de opción. |

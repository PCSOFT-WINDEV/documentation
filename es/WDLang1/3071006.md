
## CursorPos( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve y modifica la posición del cursor del ratón.

**Observación**: Para get la posición del cursor del ratón, se recomienda utilizar [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is int = CursorPos(cpClient)
IF Res <> -1 THEN
	Message("Position X = " + LoWord(Res) + "Position Y = " + HiWord(Res))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CursorPos([<Marker>, ] <X Position> , <Y Position>)`
---

**`<Result>: (Integro)`**

Posición del ratón (las coordenadas son devueltas por [HiWord](../WDLang1/3014017.md) y [LoWord](../WDLang1/3014018.md)).

- Posición actual si &lt;X Posición&gt; y &lt;Y Posición&gt; no se especifican.

- Posición anterior si se especifican &lt;X Posición&gt; y &lt;Y Posición&gt;.

- -1 si se produce un error (posición del ratón fuera de la Marker solicitada por ejemplo).




**`<Marker>: (Constante opcional de tipo Integer)`**

Indica la Marker utilizada para calcular la posición:


|   |   |
| --- | --- |
| *cpClient* | Posición del ratón en relación con el área del cliente de la ventana con foco.<br>Área de clientes = ventana sin barra de título, barra de menú y Border. |
| *cpScreen*<br>(Valor predeterminado) | Posición del ratón respecto a la pantalla. |
| *cpWindow* | Posición del cursor con respecto a la ventana con foco. |



**`<X Position>: (Real)`**

Nueva posición horizontal en la Marker seleccionada.

**`<Y Position>: (Real)`**

Nueva posición vertical en la Marker seleccionada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Cálculo de la posición del ratón
<a name="calculo_posicion_del_raton_ELTPARAGRAPHE000073"></a>

Para averiguar la posición horizontal y vertical del ratón a partir del resultado, hay que utilizar [HiWord](../WDLang1/3014017.md) y [LoWord](../WDLang1/3014018.md).

El byte bajo de &lt;Resultado&gt; se utiliza para get la posición horizontal (X) del ratón.

El byte alto de &lt;Resultado&gt; se utiliza para get la posición vertical (Y) del ratón. Si se produce un error, X e Y son iguales a -1.

Las coordenadas son relativas a la esquina superior izquierda del Marker seleccionado.
<a name="NOTE0_2"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000093"></a>

- [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md) devuelven, respectivamente, las posiciones horizontal y vertical del ratón en relación con el control en el que se hizo un clic con el botón izquierdo del ratón.

- Por ejemplo, se puede calcular la posición del cursor del ratón en una timer para mostrar siempre esta posición en la pantalla.

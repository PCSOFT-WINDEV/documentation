
## USBDetectRemovableStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detecta si se ha añadido o eliminado una unidad de almacenamiento extraíble (CD, llave USB, cámara USB,...).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF USBDetectRemovableStorage("USBProcedure") = False THEN
	Error(ErrorInfo())
END
```

```wl
// ----------------------------------------
PROCEDURE USBProcedure(Letter, Action)
IF Action = RemovableStorageInserted THEN
	Info(Letter + ": Inserted")
ELSE
	Info(Letter + ": Ejected")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = USBDetectRemovableStorage(<WLanguage procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la detección. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<WLanguage procedure>: (Nombre de Procedure)`**



- Nombre de WLanguage Procedure que se ejecutará durante la detección,

- Cadena vacía (""") para desactivar la detección. 




Esta Procedure se llama siempre que se introduce o se expulsa una unidad de almacenamiento.

Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <WLanguage procedure>(<Drive letter>, <Type of action>)
```


&lt;Carta de la unidad&gt; es una cadena usada para encontrar la letra de la unidad associated con la unidad de almacenamiento. 
Observación: Si la unidad de almacenamiento desencadena la aparición de varias letras de unidad, éstas se separarán con tabulaciones (&lt;Letra de unidad 1&gt; + TAB + &lt;Letra de unidad 2&gt; + ... + TAB + &lt;Letra de unidad N>).

El parámetro &lt;Tipo de acción&gt; es una constante entera que sirve para saber si el dispositivo fue insertado o expulsado. Este parámetro puede tomar uno de los siguientes valores:


|   |   |
| --- | --- |
| *RemovableStorageInserted* | Se ha insertado una unidad de almacén extraíble. |
| *RemovableStorageRemoved* | Se ha expulsado una unidad de almacén extraíble. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Múltiples llamadas a USBDetectRemovableStorage
<a name="multiples_llamadas_usbdetectremovablestorage_ELTPARAGRAPHE000064"></a>

Si esta función se ejecuta varias veces con diferentes Procedure s pasadas como parámetro, la última Procedure utilizada reemplazará todas las Procedure s previamente llamadas: los primeros procedimientos ya no serán llamados.
<a name="NOTE0_2"></a>


### Procedure local a la ventana
<a name="procedure_local_ventana_ELTPARAGRAPHE000071"></a>

Si el lenguaje Procedure es local a una ventana, la detección se detendrá cuando la ventana se cierre. Le recomendamos que use un Procedure global.
<a name="NOTE0_3"></a>




### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000079"></a>

La función **USBDetectRemovableStorage** no debe ejecutarse desde un hilo secundario.
<a name="NOTE0_4"></a>




### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000089"></a>

**USBDetectRemovableStorage** y [fDetectRemovibleStorage](../WDLang1/3036061.md) son equivalentes.

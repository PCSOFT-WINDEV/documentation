
## WinChangeAlias( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el alias de una ventana.

**Recordatorio**: Un alias se utiliza para diferenciar entre la misma ventana abierta varias veces (frecuente en una aplicación MDI).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opens a new MDI window.
// WINDEV chooses an alias.
sAliasName = MDIOpen(WIN_EditCustomer, "New customer", Value1, Value2)
// Use a custom alias
WinChangeAlias(sAliasName, "TEMP0001")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinChangeAlias(<Former alias> , <New alias>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio de nombre,

- <u><u><u><u>False</u></u></u></u> si no se pudo realizar la operación (ya existe una ventana con el mismo alias).




**`<Former alias>: (Cadena de caracteres)`**

Alias que deben ser modificados.

**`<New alias>: (Cadena de caracteres)`**

Nuevo alias para la ventana.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función es equivalente a la [Alias](../Proprietes/2510004.md) Property.


```txt
{<Former alias>}..Alias = <New alias>
```

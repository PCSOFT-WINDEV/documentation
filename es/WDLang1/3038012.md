


## Funciones de gestión de ventanas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las ventanas WINDEV y WINDEV Mobile mediante programación.

Para más detalles:

- para saber cómo programar los menús de manipular, véase [Funciones de gestión de menús](../WDLang1/3038016.md)

- sobre cómo manipular las ventanas MDI mediante programación, véase [Funciones de la ventana MDI](../WDLang1/3052003.md)

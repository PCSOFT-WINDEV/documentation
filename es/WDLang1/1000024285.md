


## &lt;Tabla&gt;.SelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desmarca un fila de una tabla o de un control Tabla TreeView.. La fila seleccionada corresponde a la fila actual en la que se muestra la banda de selección.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselect row #5 i n the Table control "TABLE_Product"
TABLE_Product.SelectMinus(5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.SelectMinus([<Index 1> [... [, <Index N>]]])`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla. 

- un control Tabla TreeView.




**`<Index 1>: (Entero opcional)`**

Índice de la primera fila a deseleccionar. Si no se especifica este parámetro, todas las filas que se encuentran en el control especificado se deseleccionan. **&lt;Tabla&gt;.SelectMinus** no tiene efecto si este parámetro corresponde a un fila que no está seleccionado. Puede get el índice del fila seleccionado con [&lt;Tabla&gt;.Select](../WDLang1/1000024284.md).
La función no tiene efecto si este parámetro corresponde a -1.




**`<Index N>: (Entero opcional)`**

Índice de la enésima fila a deseleccionar. Si no se especifica este parámetro, se desmarcan todas las líneas que se encuentran en el control Tabla especificado.. **&lt;Tabla&gt;.SelectMinus** no tiene efecto si este parámetro corresponde a un fila que no está seleccionado. Puede get el índice del fila seleccionado con [&lt;Tabla&gt;.Select](../WDLang1/1000024284.md).

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000236"></a>

La función **&lt;Tabla&gt;.SelectMinus** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.

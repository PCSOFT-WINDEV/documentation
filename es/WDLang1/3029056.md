
## ColorHue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el tono de un color.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SelectColor(MyColor)
Info("Hue: " + ColorHue(MyColor) + "°", ...
	"Lightness: " + ColorLightness(MyColor) + "%", ...
	"Saturation: " + ColorSaturation(MyColor) + "%")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ColorHue(<Color>)`
---

**`<Result>: (Entero)`**

Valor del tono del color especificado. Este valor se expresa como un ángulo de 0 a 360° en el círculo cromático (0 = Rojo).

**`<Color>: (Entero)`**

Color utilizado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

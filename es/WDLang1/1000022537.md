
## Funciones para la gestión de los controles del cuadro de mandos (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del Cuadro de Mando (sintaxis del prefijo):



|   |   |
| --- | --- |
| [&lt;Panel de control&gt;.AddWidget](../WDLang1/1000022502.md) | Añade y muestra un widget (ventana interna) en un control Cuadro de mando. |
| [&lt;Tablero de instrumentos&gt;.count](../WDLang1/1000022538.md) | Devuelve el número de widgets encontrados en una control Cuadro de mando. |
| [.Delete](../WDLang1/1000022535.md) | Elimina un widget mostrado en un control Cuadro de mando. El control Cuadro de mando se actualiza y una ubicación libre reemplaza al widget eliminado. |
| [&lt;Panel de control&gt;.DeleteAll](../WDLang1/1000022536.md) | Elimina todos los widgets mostrados en un control Cuadro de mando. |
| [.Display](../WDLang1/1000022500.md) | Elementos Actualiza en un control Cuadro de mando. |
| [&lt;GetWidgetPosition](../WDLang1/1000022533.md) | Devuelve la posición de un widget que se encuentra en una control Cuadro de mando. |
| [&lt;Panel de control&gt;.InfoXY](../WDLang1/1000022529.md) | Permite saber:<br><br>- si un widget se encuentra en una posición determinada en un control Cuadro de mando.<br><br>- cuyo widget se encuentra en una posición determinada en un control Cuadro de mando.<br><br><br> |
| [&lt;Tablero de control&gt;Configuración inicial](../WDLang1/1000022525.md) | Modifica la configuración de una control Cuadro de mando (definida en el editor) antes de su primera visualización. |
| [&lt;Tablero de mandos&gt;.LoadConfiguration](../WDLang1/1000022506.md) | Restaura la configuración de una control Cuadro de mando. |
| [&lt;Tablero de instrumentos&gt;.modo](../WDLang1/1000022532.md) | Abre o cierra el modo de edición de control Cuadro de mando. |
| [.MoveWidget](../WDLang1/1000022527.md) | Cambia la posición de un widget que se encuentra en un control Cuadro de mando. |
| [&lt;Panel de control&gt;.ResizeWidget](../WDLang1/1000022504.md) | Cambia las dimensiones de un widget que se encuentra en un control Cuadro de mando. |
| [&lt;SaveConfiguration](../WDLang1/1000022534.md) | Guarda la configuración de una control Cuadro de mando en formato de cadena. |

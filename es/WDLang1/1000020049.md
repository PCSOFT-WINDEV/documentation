


## Entrada asistidaCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la lista de valores propuestos por la entrada asistida para una control Campo de entrada. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF EDT_Direction = "" THEN
	AssistedInputClose(EDT_Direction)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AssistedInputClose(<Edit control>)`
---

**`<Edit control>: (Nombre del control)`**

Nombre de la Edición control a utilizar.

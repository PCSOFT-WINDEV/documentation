


## &lt;Tipo String&gt;.Increment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Incrementa un número al inicio o al final de una cadena.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
String1 is string = "String1"
// Displays "String2"
Info(String1.Increment())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Initial string>.Increment([<Position>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena inicial con el número (o dígito) incrementado.

**`<Initial string>: (Cadena de caracteres)`**

Cadena a incrementar.

**`<Position>: (Entero opcional)`**

Posición del número (o dígito) a incrementar: 


|   |   |
| --- | --- |
| *FromBeginning* | Incrementa el número al inicio de la cadena.<br><br>Si la cadena no comienza con un número, se añade el dígito 1 al inicio de la cadena. |
| *FromEnd* | Incrementa el número al final de la cadena.<br><br>Si la cadena no termina con un número, se añade el dígito "1" al final de la cadena. |


Si no se especifica este parámetro, hay tres casos posibles:

- la cadena termina con un número: en este caso, se incrementa el número al final,

- la cadena inicia con un número: en este caso, se incrementa el número al inicio,

- la cadena no inicia ni termina con un número: en este caso, se añade el número "1" al final de la cadena.

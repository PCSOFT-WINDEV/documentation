


## AddColumn( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una columna al final de una array bidimensional Property ([xlsDocument](../WDLang5/1000017464.md) Variable por ejemplo).






<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AddColumn(<WLanguage array> [, <Element row 1> [... [, <Element row N>]]])`
---

**`<Result>: (Integro)`**



- Índice de la nueva columna,

- -1 si se produce un error.




**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional o bidimensional.

**`<Element row 1>: (Tipo de elementos del array, opcional)`**

Elemento de la columna que se añadirá a la array especificada. Cada elemento corresponde a una fila en el array. El tipo de elemento añadido debe ser compatible con el tipo de la fila correspondiente..
Si no se especifica ninguno de estos parámetros, se añade una columna vacía a la columna array. Las filas se inicializan con el valor Default del tipo de los otros elementos array.
Si no se especifica uno de estos parámetros, el correspondiente fila se inicializa con el valor Default del tipo de los otros elementos array.



**`<Element row N>: (Tipo de elementos del array, opcional)`**

Elemento de la columna que se añadirá a la array especificada. Cada elemento corresponde a una fila en el array. El tipo de elemento añadido debe ser compatible con el tipo de la fila correspondiente..
Si no se especifica ninguno de estos parámetros, se añade una columna vacía a la columna array. Las filas se inicializan con el valor Default del tipo de los otros elementos array.
Si uno de estos parámetros no se especifica, el correspondiente fila se inicializa con el valor Default del tipo de los otros elementos array.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se llama a la función **AddColumn**:

- se debe crear la variable de tipo avanzado.

- el tipo avanzado debe tener un enumerador de tipo de colección modificable.

- el tipo avanzado se amplía automáticamente para recibir la nueva columna.

- las filas se inicializan con los valores pasados como parámetro. Si no se pasa ningún valor en el parámetro, las filas se inicializan con el valor Default del tipo de los elementos array.




## Funciones de gestión de los menús (sintaxis de los prefijos)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los menús (sintaxis del prefijo): 




|   |   |
| --- | --- |
| [&lt;Menú&gt;. existente](../WDLang1/1000020605.md) | Indica si existe una opción de menú en un menú.. |
| [&lt;Menú&gt;.add](../WDLang1/1000020591.md) | Añade un menú emergente existente al menú principal de la ventana actual. |
| [&lt;Menú&gt;.AddMenu](../WDLang1/1000020592.md) | Añade un nuevo menú a un menú desplegable o contextual existente de una ventana o Page. |
| [&lt;Menú&gt;.AddOption](../WDLang1/1000020593.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. |
| [&lt;Menú&gt;.AddPopup](../WDLang1/1000020602.md) | Transforma una opción de menú de una Page para que esta opción abra una ventana emergente. |
| [&lt;Menú&gt;.InsertarMenú](../WDLang1/1000020606.md) | Inserta un menú antes que otro menú en una ventana. |
| [&lt;Menú&gt;.InsertarOpción](../WDLang1/1000020609.md) | Inserta una nueva opción en una posición específica. |
| [.AddSeparator](../WDLang1/1000020603.md) | Añade un nuevo separador en un menú (desplegable o contextual). |
| [.AddURLOption](../WDLang1/1000020594.md) | Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú abre la página que corresponde a la URL especificada. |
| [.Delete](../WDLang1/1000020614.md) | Borra un menú o una opción de menú. |
| [.InsertSeparator](../WDLang1/1000020610.md) | Inserta un separador en un menú. |





Las siguientes funciones también se pueden utilizar para gestionar menús: 


|   |   |
| --- | --- |
| [&lt;Gráfico&gt;.menú](../WDLang3/1000024094.md) | Activa o desactiva el menú contextual de un gráfico. |
| [ControlContextMenu](../WDLang1/3025012.md) | Identifica la control en la que se ha abierto el menú contextual. |
| [EnumMenu](../WDLang1/3038039.md) | En un bucle, esta función se utiliza para enumerar las opciones de menú de una ventana, así como las subopciones de una opción de menú. |
| [EnumSubElement](../WDLang1/3064002.md) | En un bucle, esta función enumera los menús desplegables o contextuales de una ventana o control. |
| [Menú Emular](../WDLang1/1000009010.md) | Emula el siguiente menú que se abrirá y ejecuta automáticamente la opción de menú pasada como parámetro. |
| [OpenContextMenu](../WDLang1/3025029.md) | Abre automáticamente un menú contextual para la control o ventana actual. |




## TableGiveChild (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve los "hijos" de un elemento (nivel inferior) en una control Tabla TreeView.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// First "child" of first node in the "TVT_RecipeTV" control
Res is int
Res = TableGiveChild(TVT_RecipeTV, 1, taFirst)
WHILE Res <> -1
	Trace(Res)
	Res = TableGiveChild(TVT_RecipeTV, 1, taNext)
END
```


<a name="Example2"></a>

```wl
// Recursive procedure used to list all children 
// of all parents from a row
PROCEDURE ListChildren(i=0)
j is int
// If no row number was passed as parameter,
// gets the index of selected row
IF i = 0 THEN i = TableSelect(TVT_MyTreeViewTable)
j = TableGiveChild(TVT_MyTreeViewTable, i, taFirst)
WHILE j <> -1
	Info("Row: " + j + ", Code: " + COL_Code[j])
	// Call the procedure by specifying 
	// the current row number
	ListChildren(j)
	j = TableGiveChild(TVT_MyTreeViewTable, i, taNext)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Devolver al hijo de un elemento especificando el número de fila

`<Result> = TableGiveChild(<TreeView Table control> , <Row number> , <Child>)`
---

**`<Result>: (Integro)`**



- Número de la fila para el "niño" que se encontró,

- -1 si no se encuentra ningún "niño".




**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Row number>: (Integro)`**

Número de la fila que contiene la rama a utilizar.

**`<Child>: (Constante)`**

Indica el "niño" solicitado.


|   |   |
| --- | --- |
| *taFirst* | Primer elemento hijo. |
| *taNext* | Siguiente elemento hijo. |




<a name="SYNTAX2"></a>

### Devolver al hijo de un elemento especificando su nombre

`<Result> = TableGiveChild(<TreeView Table control> [, <Element name>] , <Child>)`
---

**`<Result>: (Integro)`**



- Número de la fila del "niño" encontrado,

- -1 si no se encuentra ningún "niño".




**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Element name>: (Cadena de caracteres opcional)`**

Nombre del elemento a utilizar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Element name>"
```
Se produce un error de WLanguage si este parámetro no corresponde a un elemento existente.

**`<Child>: (Constante)`**

Indica el "niño" solicitado.


|   |   |
| --- | --- |
| *taFirst* | Primer elemento hijo. |
| *taNext* | Siguiente elemento hijo. |

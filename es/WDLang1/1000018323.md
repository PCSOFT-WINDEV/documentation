


## En espera (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un elemento a la cola. El elemento se añade al final de la cola.  




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyQueue is queue of int

// Enqueue the values 1, 2 and 3
Enqueue(MyQueue, 1)
Enqueue(MyQueue, 2)
Enqueue(MyQueue, 3)

// Display the trace: 1, 2, 3
x is int
WHILE Dequeue(MyQueue, x)
	Trace(x)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Enqueue(<Queue> , <Value>)`
---

**`<Queue>: (Cola de Variable)`**

Nombre de la variable de tipo [Cola](../WDLang1/1514021.md) que se utilizará.

**`<Value>: (Tipo de elementos manejados por la cola)`**

Valor que se añadirá al final de la cola. Todos los tipos de variables pueden utilizarse EXCEPTO para la arrays, la arrays asociativa, las colas, las pilas y las listas..

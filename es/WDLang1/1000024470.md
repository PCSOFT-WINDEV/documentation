


## &lt;Variable picLayer&gt;.StartDrawing (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica que las funciones de dibujo que se utilizarán están destinadas a una variable de tipo [picLayer](../WDLang1/1000024604.md).

**Observación**: La función **&lt;Variable picLayer&gt;.StartDrawing** inicializa todos los parámetros necesarios para crear un dibujo de un elemento determinado.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is WDPic = "Test.bmp"
MyLayer is picLayer
// Start drawing in the layer
ResDrawing = MyLayer.StartDrawing()
// Drawings in MyLayer
...
// End of drawings in MyLayer
IMG_MyDrawing = MyImage
```

```wl
MyWDPic is WDPic = "Test.bmp"
MyLayer is picLayer

MyLayer.StartDrawing()
MyLayer.Font("Arial", 12, iNormal, 0)
MyLayer.DrawText(4, 0, "I'm writing in a picLayer variable!")
IMG_MyDrawing = MyWDPic
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

`<Result> = <picLayer image>.StartDrawing([<Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u>True</u></u></u> si se inicializó el dibujo,

- <u><u><u>False</u></u></u> en caso contrario.




**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Opciones de dibujo:


|   |   |
| --- | --- |
| *dGridlinesOpacity* | El dibujo se muestra sobre las líneas de cuadrícula para resaltar las secciones transparentes o semitransparentes del dibujo. Las líneas de cuadrícula no se guardan en la imagen.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *dNoErase* | No borra el dibujo de la variable. |
| *dOnChart* | Permite dibujar en una variable en la que ya se ha dibujado un gráfico. En este caso, no se borra el dibujo del gráfico.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *dWithOpacity* | Permite gestionar la opacidad en las funciones de dibujo utilizadas con la variable. Permite, por ejemplo, crear un dibujo guardado en formato PNG con canales alfa. |


Si no se especifica este parámetro, no se utiliza ninguna opción de dibujo.

## Observaciones


### Uso en una sintaxis con prefijo
<a name="uso_una_sintaxis_con_prefijo_ELTPARAGRAPHE000488"></a>

Se debe llamar a la función **&lt;Variable picLayer&gt;.StartDrawing** antes de utilizar otras funciones de dibujo. La función **&lt;Variable picLayer&gt;.StartDrawing** permite definir las opciones de dibujo utilizadas en una variable de tipo [picLayer](../WDLang1/1000024604.md).

### Dibujo con opacidad
<a name="dibujo_con_opacidad_ELTPARAGRAPHE000571"></a>

Al crear un dibujo que admita la opacidad, puede guardar fácilmente imágenes PNG con canales alfa.

**Para guardar un dibujo PNG con canales Alfa**:

1. Utilice la función **&lt;Variable picLayer&gt;.StartDrawing**. Esta función permite definir, si es necesario, el elemento sobre el que se aplicarán las funciones de dibujo, así como el modo de dibujo. La constante *dWithOpacity* indica que el dibujo gestionará la opacidad. La constante *dGridlinesOpacity* permite mostrar una cuadrícula debajo de las diferentes áreas del dibujo.

2. Configure la opacidad de los dibujos con las siguientes funciones:
	


|   |   |
| --- | --- |
| [&lt;Variable picLayer&gt;.Background](../WDLang1/1000024480.md) | Declara el color de fondo y su opacidad para los siguientes dibujos. |
| [&lt;Variable picLayer&gt;.DrawPoint](../WDLang1/1000024500.md) | Declara el color de los puntos y su opacidad para los siguientes dibujos. |
| [&lt;Imagen&gt;.Pen](../WDLang1/1000024016.md) | Declara el color de las líneas y su opacidad para los siguientes dibujos. |


	**Observación**: La función [&lt;Variable picLayer&gt;.PixelOpacity](../WDLang1/1000024499.md) permite conocer la opacidad de un punto.

3. Utilice la función [&lt;Image&gt;.SavePNG](../WDLang1/1000024345.md) para guardar el dibujo en formato PNG con canales alfa (opacidad).




**Limite**: En modo "Gestión de la opacidad", la función [&lt;Variable picLayer&gt;.DrawText](../WDLang1/1000024657.md) se limita al uso de fuentes True Type.

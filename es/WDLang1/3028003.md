
## DDEDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una conexión entre el programa actual y un destinatario.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection = DDEConnect("Excel", Subject)		// Connection
Counter = 3			// Repeats 3 times if necessary
LOOP				// Sends the data
	IF DDESend(MyConnection, Item, Data, 10) THEN BREAK
	Counter --
	IF Counter = 0 THEN
		Error("The information was not sent to Excel")
		BREAK
	END
END
DDEDisconnect(MyConnection)		// Disconnection
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEDisconnect([<Connection identifier>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se hizo efectiva la desconexión,

- <u><u><u><u>False</u></u></u></u> si se ha producido un error ([DDEError](../WDLang1/3028006.md) devuelve los detalles del error).




**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, se desconecta la conexión actual..

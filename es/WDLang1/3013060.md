


## ExecuteAAF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta una característica de aplicación automática (AAF) en un control o ventana.. Esta función se puede utilizar para iniciar el corrector ortográfico en un control Campo de entrada, por ejemplo.

**Observación**: esta función es equivalente a [AAFExecute](../WDLang1/1000022099.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the "Spelling checker" button
// => Starts the spelling checker
// on the "EDT_ProdDescription" edit control
ExecuteAAF(EDT_ProdDescription, aafSpellCheck)
```
<a name="Example2"></a>

```wl
// Click the "Cancel" button
// => restores the previous values
ExecuteAAF(WIN_CustomerInput, aafRestoreValue)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = EjecutarAAF(<Control or Window> , <Action to perform>)`
---

**`<Result>: (booleano)`**

Valor que puede ser explotado sólo si el valor *aafClosingDialog* se utiliza la constante. Esto permite identificar la acción realizada por el usuario cuando se muestra el cuadro de diálogo. &lt;El resultado&gt; está configurado en: 

- <u><u><u><u>False</u></u></u></u> si el usuario pide no cerrar el documento,

- <u><u><u><u>True</u></u></u></u> De lo contrario. 




**`<Control or Window>: (Nombre del control o nombre de la ventana)`**

Nombre de la ventana o control al que se aplica la funcionalidad automática. También es posible utilizar un alias de ventana.

**`<Action to perform>: (Constante de tipo Integer)`**

Característica automática a realizar. Esta acción depende del elemento utilizado (control o ventana). En el caso de un control, esta acción depende del tipo de control utilizado. Si el tipo de control no es válido para la constante seleccionada, la función no tiene ningún efecto .

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para desactivar una función automática, use [AAFDisable](../WDLang1/1000022018.md) (o [DisableAAF](../WDLang1/3013061.md)). La opción correspondiente se borra automáticamente de el menú contextual.

- Para borrar las opciones que corresponden a los AAF de el menú contextual de un control, utilice [..PopupMenu](../Proprietes/2510077.md).




## PDFReaderSeekNext (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la posición en el siguiente resultado de una búsqueda iniciada por [PDFReaderSeekFirst](../WDLang1/1000023409.md) en el contenido de un PDF Reader control (si hay varias ocurrencias del texto buscado).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Seeks a text in a PDF Reader control and displays it
PDFReaderSeekFirst(PDF_MyPDFReader, "company")
// Finds the next element
IF PDFReaderSeekNext(PDF_MyPDFReader) = False THEN
	Info("Search ended")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFReaderSeekNext(<PDF Reader control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto especificado en [PDFReaderSeekFirst](../WDLang1/1000023409.md) se encontró en el actual Document en PDF Reader control ,

- <u><u><u><u>False</u></u></u></u> si no se encontró el texto. 




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El texto y las opciones de búsqueda se han especificado en [PDFReaderSeekFirst](../WDLang1/1000023409.md).




## Funciones para la gestión de los controles TimeLine (sintaxis de prefijo)
			



Las siguientes funciones se utilizan para gestionar los controles de TimeLine (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;Línea de tiempo&gt;.AddEvent](../WDLang1/1000020776.md) | Añade una Event a una control Línea de tiempo. |
| [TimelineAddTrack](../WDLang1/1000020777.md) | Añade una nueva pista a una control Línea de tiempo. |
| [&lt;TimeLine&gt;.ChangeMode](../WDLang1/1000020778.md) | Modifica el período (es decir, el número de segundos, milisegundos, microsegundos,...) que muestra un control Línea de tiempo. |
| [TimelineDeleteAll](../WDLang1/1000020782.md) | Borra todos los eventos y pistas de un control Línea de tiempo. |
| [.DeleteEvent](../WDLang1/1000020780.md) | Borra un Event de un control Línea de tiempo. |
| [TimelineDeleteTrack](../WDLang1/1000020781.md) | Borra una pista de una control Línea de tiempo. |
| [TimelineDisplay](../WDLang1/1000020775.md) | Reinicializa el contenido de una control Línea de tiempo a partir de su fuente de datos (archivo de datos HFSQL por ejemplo). |
| [&lt;Línea de tiempo&gt;.listEvent](../WDLang1/1000020779.md) | Devuelve: <br><br>- la lista de eventos encontrados entre dos veces en un control Línea de tiempo, <br><br>- todos los eventos de un control Línea de tiempo,<br><br>- una Event( específica seleccionada o suspendida). <br><br><br> |

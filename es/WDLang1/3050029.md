
## Exp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el exponencial de un valor numérico (al revés del logaritmo napieriano).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the exponential of 5
ResExponential = Exp(5)
// Returns 148.413159103
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Exp(<Numeric value>)`
---

**`<Result>: (Real)`**

Busco exponencial.

**`<Numeric value>: (Entero o real)`**

Valor numérico a utilizar.

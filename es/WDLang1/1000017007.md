


## ArrayToString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un array de una o dos dimensiones en una cadena de caracteres.

**Observación**: Esta función sólo puede utilizarse en [WLanguage arrays](../Motscles/1514030.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// One-dimensional array
sMyValues is string 
// Initial array
MyArray is array of 3 strings
MyArray[1] = "A"
MyArray[2] = "B"
MyArray[3] = "C"
sMyValues = ArrayToString(MyArray)
// sMyValues contains: "A" + CR + "B" + CR + "C"
```
<a name="Example2"></a>

```wl
// Two-dimensional array
sMyValues is string 
// Initial array
MyArray is array of 2 by 3 strings
MyArray[1,1] = "A"; MyArray[1,2] = "B"
MyArray[1,3] = "C"; MyArray[2,1] = "D"
MyArray[2,2] = "E"; MyArray[2,3] = "F"

sMyValues = ArrayToString(MyArray)
// sMyValues contains: "A" + TAB + "B" + TAB + "C" +CR+ "D" + TAB + "E" + TAB + "F"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Convertir una array unidimensional

`<Result> = ArrayToString(<WLanguage array> [, <Element separator>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres que contiene los diferentes valores que se encuentran en la array. Los elementos array se convierten en cadenas de caracteres y se añaden a la cadena mientras están separados por el separador &lt;Element separator&gt;.

**`<WLanguage array>: (Variable de tipo Array)`**

Nombre de la array unidimensional que debe convertirse en un string.. Hay que crear esta array.

**`<Element separator>: (Cadena de caracteres opcional)`**

Separador utilizado para separar los valores de los diferentes elementos array. El elemento separador utilizado por Default es el carácter CR (Carriage Return).  


<a name="SYNTAX2"></a>

### Convertir una array bidimensional

`<Result> = ArrayToString(<WLanguage array> [, <Row separator> [, <Column separator>]])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres que contiene los diferentes valores que se encuentran en la array. Los elementos array se convierten en cadenas de caracteres y se añaden a la cadena.. Las filas están separadas por &lt;Fila separator&gt; y las columnas por &lt;Column separator&gt;.

**`<WLanguage array>: (Variable de tipo Array)`**

Nombre de la array bidimensional que debe convertirse en un string.. Esta array debe ser creada. 

**`<Row separator>: (Cadena de caracteres opcional)`**

Separador utilizado para separar los valores de las diferentes filas. El separador fila utilizado por Default es el personaje CR (Carriage Return).

**`<Column separator>: (Cadena de caracteres opcional)`**

Separador utilizado para separar los valores de diferentes columnas. El separador de columna usado por Default es TAB.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La estructura arrays soportada es la única estructura arrays unidimensional.. En este caso, el orden de los parámetros define el orden en la cadena de resultados.

- No se soportan los arrays que contienen strings o variantes de UNICODE.

- Para el tipo Duración, el valor se convierte en miles de segundos.

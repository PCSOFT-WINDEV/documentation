


## TablaFormulaDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las fórmulas añadidas con [TableFormulaAdd](../WDLang1/1000022361.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TableFormulaDeleteAll(TABLE_MyTable)
nRow is int
nRow = TableFormulaAdd(TABLE_MyTable.COL_Num, "Positive mean", ProcInit, ProcAdd, ProcEnd)
// Change the background color of the row for custom calculation
COL_Num[nRow].BackgroundColor = LightRed

nCounter is int
INTERNAL PROCEDURE ProcInit() 
	nCounter = 0
	RESULT 0
END

INTERNAL PROCEDURE ProcAdd(Accumulator, ColValue) 
	// Ignores the negative numbers or NULL
	IF (ColValue <= 0) RESULT Accumulator
	nCounter++
	RESULT Accumulator + ColValue
END
INTERNAL PROCEDURE ProcEnd(Accumulator) 
	IF nCounter = 0 THEN RESULT 0
	// Calculate the mean
	RESULT Accumulator/nCounter
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TableFormulaDeleteAll(<Table control>)`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.


Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los cálculos preestablecidos (total, media, recuento, mínimo, máximo) no se ven afectados..

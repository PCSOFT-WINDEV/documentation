
## WinUtilSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica el tamaño útil de una ventana interna con barras de desplazamiento automatic. El tamaño de useful corresponde al tamaño completo de la ventana interna:

- Ancho de la ventana interna.

- Altura de la ventana interna.




Las barras de desplazamiento se mostrarán automáticamente si el tamaño de la control que contiene la ventana interna es demasiado pequeño.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenTailleUtile.gif)


**Observación**: Esta función también se utiliza para especificar el tamaño de useful de los supercontroles con barras de desplazamiento.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sImage is string = SC_FilePicker.EDT_FILE
nWidth is int = ExtractString(BitmapInfo(sImage), 2)
nHeight is int = ExtractString(BitmapInfo(sImage), 3)

IW_Internal1.IMG_Image1 = sImage
IW_Internal1.IMG_Image1.Width = nWidth
IW_Internal1.IMG_Image1.Height = nHeight
WinUsefulSize(IW_Internal1, nWidth, nHeight)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WinUsefulSize(<Internal window> , <Width> , <Height>)`
---

**`<Internal window>: (Nombre de ventana)`**

Nombre de la ventana interna.

**`<Width>: (Integro)`**

Nuevo ancho (en píxeles) de la ventana interna.

**`<Height>: (Integro)`**

Nueva altura (en píxeles) de la ventana interna.

<a name="NOTE0_1"></a>

## Observaciones




### Ventana interna con barras de desplazamiento automatic
<a name="ventana_interna_con_barras_desplazamiento_automatic_ELTPARAGRAPHE000064"></a>

**WinUtilSize** permite cambiar dinámicamente el tamaño de la ventana interna mostrada en una "Ventana interna" control.. Si es necesario, las barras de desplazamiento pueden aparecer o desaparecer al cambiar el tamaño.

**Observaciones**: 

- Las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md) modifican el tamaño exterior del control Ventana interna, no el tamaño de la ventana interna que se utilizará en el control.

- Las propiedades [UsefulWidth](../Proprietes/1000021503.md) y [UsefulHeight](../Proprietes/1000021504.md) se utilizan para get y cambiar el tamaño de la ventana interna que se utilizará en el control.

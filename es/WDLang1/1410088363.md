
## gStoredValue( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un valor "Invisible" a un elemento de un cuadro de lista, Combo Box o control Planificador. Este valor será el valor almacenado cuando el elemento control sea seleccionado por el usuario.

**Observaciones**:

- Para obtener el valor almacenado asociado con un elemento del control, use el [StoredValue](../Proprietes/2510127.md) Property .

- Para recuperar el valor definido por **gStoredValue**:

	- utilice la función [gStoredValueEnable](../WDLang1/1410088364.md).

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) marque "Devolver el valor especificado con la función gStoredValue" (pestaña "Detalles" del control description). 


En este caso, el control devolverá directamente el valor del **gStoredValue** que corresponda a la opción seleccionada.

- Esta función es equivalente a la función [gEnlace](../WDLang1/3029043.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Button control code to populate the Combo Box control
// ----------------------------------------------
// The "COMBO_Title" Combo Box displays personal titles:
// "Mister", "Madam" and "Miss".
// An integer representing a gender will be returned to the language
ListeAdd(COMBO_Title, "Mister" + gStoredValue("1"))
// Only the string "1" is returned to the language.
// This string is not displayed in the Combo Box control.
```

```wl
// Button control code to get the value of gStoredValue
// -------------------------------------------------------------------------------
// Code to get the value of gStoredValue for the selected option 
Info(COMBO_Title[COMBO_Title].StoredValue)
// Displays "1" in our example.
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gStoredValue(<String to return>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena control que debe utilizarse en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

**`<String to return>: (Cadena de caracteres)`**

Valor de texto a devolver. Sólo esta cadena será devuelta al idioma. Esta cadena no se mostrará en la pantalla de fila.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Dónde utilizar gStoredValue
<a name="donde_utilizar_docparampagetitleshort_ELTPARAGRAPHE000072"></a>

**gStoredValue** se puede llamar en cualquier lugar del fila que se añada.

Por ejemplo:


```wl
ListAdd(COMBO_COMBO1, gStoredValue("1") + "Mister")
// or
ListAdd(COMBO_COMBO1, "Mister" + gStoredValue("1"))
```

<a name="NOTE0_2"></a>


### Valor añadido
<a name="valor_anadido_ELTPARAGRAPHE000086"></a>

Los valores añadidos con **gStoredValue** no se tienen en cuenta para las operaciones de ordenación y los atajos de teclado.

Si se añaden varios valores a un elemento de un cuadro de lista con **gStoredValue**, sólo se devolverá el primer valor.


```wl
ListAdd(LIST_LIST1, "MyElement" + gStoredValue("Link1") + gStoredValue("Link2"))
```


En este ejemplo, sólo se devolverá el valor "Link1".

[ListSeek](../WDLang1/3049003.md) tiene en cuenta los valores añadidos por **gStoredValue**.
<a name="NOTE0_3"></a>




### Sensibilidad a los casos
<a name="sensibilidad_los_casos_ELTPARAGRAPHE000110"></a>

**gStoredValue** no distingue entre mayúsculas y minúsculas.

Por lo tanto, el siguiente código: 


```wl
ListAdd(LIST_LIST1, "MyElement" + gStoredValue("AAA"))
```
será equivalente a:


```wl
ListAdd(LIST_LIST1, "MyElement" + gStoredValue("aaa"))
```

<a name="NOTE0_4"></a>




### Combo Box completado automáticamente
<a name="combo_box_completado_automaticamente_ELTPARAGRAPHE000127"></a>

**gStoredValue** se utiliza para gestionar los controles "auto-rellenados Combo Box".
<a name="NOTE0_5"></a>




### Elemento recuperado
<a name="elemento_recuperado_ELTPARAGRAPHE000137"></a>

Cuando se recupera un elemento de un List Box o control Combo Box, el orden de prioridad es el siguiente:

1. Valores añadidos por **gStoredValue**.

2. Valores añadidos por [gToRetrieve](../WDLang1/3029025.md).




Para obtener el valor almacenado asociado con un elemento del control, use el [StoredValue](../Proprietes/2510127.md) Property .
<a name="NOTE0_6"></a>




### Control Planificador
<a name="control_planificador_ELTPARAGRAPHE000158"></a>

Si los Resource s del programa han sido creados con [SchedulerAddResource](../WDLang1/1000019279.md) mediante **gStoredValue**, el nombre del Resource obtenido con [SelectedResource](../Proprietes/1000019445.md) Property corresponderá al valor especificado en la función **gStoredValue** para este Resource.

Ejemplo: 


```wl
// Adds a resource (a meeting room) into the Scheduler control
// Meeting is the data file that describes the available meeting rooms
SchedulerAddResource(SCH_RoomOccupancy, Meeting.Name + gStoredValue(Meeting.MeetingID))

...

// Click code on BTN_NEW_APT button
MyAPT is Appointment
IF SCH_RoomOccupancy.SelectedResource = "" THEN
        RETURN
END

MyAPT.StartDate = SCH_RoomOccupancy.StartDate
MyAPT.EndDate = SCH_RoomOccupancy.EndDate
MyAPT.Resource = SCH_RoomOccupancy.SelectedResource
// The name of the resource corresponds to the value specified 
// in gStoredValue for this resource
```

<a name="NOTE0_7"></a>


### Caso especial: control Combo Box o Combo Box en un control Tabla
<a name="caso_especial_control_combo_box_combo_box_control_tabla_ELTPARAGRAPHE000179"></a>

Si **gStoredValue** debe devolver una cadena vacía (""), el resultado será diferente si el control utilizado es una columna control Combo Box o Combo Box en un control Tabla. 

Se utiliza el siguiente código: 

```wl
ListAdd(COMBO, "MyElement" + gStoredValue(""))
```
el valor recuperado será: 

1. una cadena vacía ("") en el caso de una control Combo Box,

2. 0 (cero) en el caso de una columna Combo Box en un control Tabla.




Para utilizar el mismo código para recuperar el valor de **gStoredValue** para una columna control Combo Box y una Combo Box, se recomienda utilizar un control Combo Box en un [columna portacontenedores](../WDChamp/1013263.md) en el control Tabla.




## docTable (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docTable** se utiliza para definir las características avanzadas de un párrafo de la tabla que se encuentra en un procesador de textos Document. Las características de esta tabla pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no debe utilizarse directamente.** Sólo debe utilizarse a través de las variables [docParagraph](../WDLang1/1000022483.md) correspondientes a un tabla. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the document
MyDocument is Document <- WP_Table
// Retrieve the table, here the paragraph 1
pTable is docParagraph <- MyDocument.Paragraph[1]

// Browses the rows
FOR I = 1 _TO_ pTable.Table.Rows.Occurrence
	// Browses the columns
	FOR I = 1 _TO_ pTable.Table.Columns.Occurrence
		Trace("Value of cell [[%i%]][[%j%]] = " + 
		pTable.Table.Cells[I, j].Content.Text)
	END
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables docTable
<a name="propiedades_especificas_description_las_variables_doctable_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docTable**:

   | --- |
| --- | --- | --- |
| Nombre de la propiedad | Tipo utilizado | Efecto |
| Células[Fila,Columna] | Tabla de celdas tabla ([docCell](../WDLang1/1000022927.md)) | Acceso a las celdas de un párrafo de Tabla. El tipo de células tabla es [docCell](../WDLang1/1000022927.md). |
| Columnas |   | Variable de tipo [docColumn](../WDLang1/1000023160.md) | Utilizado para manipular las columnas tabla. <br><br>Por ejemplo, para insertar una columna en la tercera columna de un tabla: <br><br><pre><code>Insert(pTable.Table.Columns, 3)</code></pre><br> |
|   | Columnas.AnchoMilímetros | Real | Ancho de fila (en milímetros). |
|   | Columns.WidthMode | Constante de tipo Integer | Modo para calcular el ancho de columna: <br><br>- *docWidthModeNone*: Caso especial. <br><br>- *docWidthModeAutomatic*: El ancho de columna se calcula automáticamente según el texto que se encuentra en la columna actual y el que se encuentra en las otras columnas.. <br><br>- *docWidthModeUndefined*: El modo de cálculo del ancho de columna no está definido porque las celdas de columna tienen anchos diferentes.. <br><br>- *docWidthModePercent*: El modo de cálculo del ancho de columna se realiza en porcentaje del ancho total de tabla.. <br><br>- *docWidthModeValue*: El modo de cálculo de la anchura de las columnas se realiza según la anchura suministrada en milímetros.. Este modo se selecciona automáticamente si se especifica Columns.WidthMillimeters. <br><br><br><br><br>Esta propiedad es de solo lectura. |
| Filas |   | Variable de tipo [docRow](../WDLang1/1000022925.md) | Utilizado para manipular las filas de tabla. <br><br>Por ejemplo, para insertar un fila en el 3er fila de un tabla: <br><br><pre><code>Insert(pTable.Table.Rows, 3)</code></pre><br> |
|   | Filas.Altura | Integro | Altura de fila (en milímetros). |
|   | Rows.HeightMode | Constante de tipo Integer | Modo de cálculo de la altura fila: <br><br>- *docHeightModeAtLeast*: La altura fila será como mínimo la definida con el **Altura** Property. <br><br>- *docHeightModeAutomatic*: La altura de fila se calcula automáticamente en función del texto de las celdas de fila. <br><br>- *docHeightModeExact*: La altura fila será la definida con la altura **** Property. <br><br>- *docHeightModeUndefined*: El modo de cálculo de la altura de fila no está definido (el modo de cálculo de automatic lo realiza Default). <br><br><br> |
|   | Rows.CellCount | Integro | Número de células fila. <br><br>Esta propiedad es de solo lectura. |

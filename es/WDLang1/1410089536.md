


## &lt;Lector PDF&gt;.GoToBookmark (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la posición directamente en el marcador deseado en el documento PDF.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyPDF is pdfDocument = "test.pdf"
PDF_MyReader = MyPDF

// Set the position on the first bookmark
IF NOT MyPDF.Bookmark.Empty THEN
	Bookmark is pdfBookmark = MyPDF.Bookmark[1]
	PDF_MyReader.GoToBookmark(Bookmark)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<PDF Reader control>.GoToBookmark(<Bookmark>)`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<Bookmark>: (Variable de tipo pdfBookmark)`**

Nombre de la variable de tipo [pdfBookmark](../WDLang6/1410089195.md) que corresponde al marcador que se mostrará en el control Lector PDF.

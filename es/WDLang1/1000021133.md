


## WinCurrentLayout (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del layout actual: 

- en la ventana actual.

- en la ventana especificada o ventana interna.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF WinCurrentLayout(WIN_Details) = 2 THEN
	STC_DETAILS = CUSTOMER.DETAILS
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el diseño de la ventana actual

`<Result> = WinCurrentLayout()`
---

**`<Result>: (Integro)`**



- Índice del layout actual.

- 0 si la ventana no incluye ninguna disposición. 





<a name="SYNTAX2"></a>

### Obtener la disposición de una ventana específica

`<Result> = WinCurrentLayout(<Window used>)`
---

**`<Result>: (Integro)`**



- Índice del layout actual.

- 0 si la ventana interna no incluye ninguna disposición. 




**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna utilizada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los diseños se numeran en el orden en que se crearon.

- El layout cambia si la orientación del dispositivo cambia o si se ejecuta la función [WinChangeLayout](../WDLang1/1000021056.md).

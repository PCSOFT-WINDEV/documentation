


## VideoStopDecoding (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la decodificación de códigos de barras en una control Cámara.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops decoding bar codes in the Camera control
VideoStopDecoding(CAM_Camera1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`VideoStopDecoding(<Camera control>)`
---

**`<Camera control>: (Cadena de caracteres)`**

Nombre del control Cámara a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Cuando la control Cámara está configurada para permitir la decodificación de los códigos de barras (en la ventana de la description), la decodificación se inicia automáticamente cuando se abre la ventana. Para prEvent este comportamiento, se puede llamar a **VideoStopDecoding** en la "Inicialización" Event del control Cámara.

- Cuando se detiene la decodificación de los códigos de barras, la retícula de decodificación ya no aparece en la control Cámara.

- Si el control Cámara no ha sido configurado para soportar la descodificación de códigos de barras (en la ventana description) o si no hay descodificación en curso, la función no tiene efecto.

- Esta función es equivalente a la función [CameraStopDecoding](../WDLang1/1000022449.md).

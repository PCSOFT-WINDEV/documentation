
## SysIdentifier( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el identificador único del dispositivo móvil actual.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sIdentifier is string 
sIdentifier = SysIdentifier()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysIdentifier()`
---

**`<Result>: (Cadena de caracteres)`**

Identificador único del dispositivo.


## StatCovariance( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la covarianza entre dos series de valores.

**Observación**: La covarianza corresponde a la media del producto de las desviaciones de cada serie.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003050020&name=statcovariance_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Calculate the covariance between row 1 and row 2 of a matrix
ResCovariance = StatCovariance("MyMatrix", 1, 2, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = StatCovariance(<Matrix> [, <1st row/1st column> [, <2nd row/2nd column> [, <Row/Column>]]])`
---

**`<Result>: (Real)`**



- Covarianza entre las dos series,

- 0 si la covarianza es nula.




**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz que contiene los datos, definida por [MatCreate](../WDLang1/3050058.md).

**`<1st row/1st column>: (Entero opcional)`**

1a fila o 1a columna que contiene los datos (1 de Default).

**`<2nd row/2nd column>: (Entero opcional)`**

2ª fila o 2ª columna que contiene los datos (2 de Default).

**`<Row/Column>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si las series de valores corresponden a columnas de la matriz,

- <u><u><u><u>False</u></u></u></u> si las series de valores corresponden a las filas de la matriz.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Número de elementos encontrados en la matriz
<a name="numero_elementos_encontrados_matriz_ELTPARAGRAPHE000096"></a>

El número de elementos encontrados en la matriz se toma como reference. Este número de elementos corresponde a:

- el número de filas de la matriz, devuelto por [MatNbLine](../WDLang1/3050060.md).

- el número de columnas de la matriz, devuelto por [MatNbColumn](../WDLang1/3050073.md).




Si la serie contiene menos elementos que la matriz, los valores que faltan se rellenan automáticamente con 0. Estos valores se tienen en cuenta durante los cálculos.
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000115"></a>

**Atención**: La función **StatCovariance** no devuelve ningún código de error. Para saber si se han generado errores al calcular la covarianza, utilice [StatError](../WDLang1/3050019.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

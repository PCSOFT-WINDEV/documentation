
## RandomCryptoInteger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un número aleatorio que puede ser utilizado con las tecnologías criptográficas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve an integer included between 0 and 2 147 483 647
ResRandom = RandomCryptoInteger()
// Retrieve an integer included between 0 and 34
ResRandom = RandomCryptoInteger(34)
// Retrieve an integer included between 7 and 21
ResRandom = RandomCryptoInteger(7, 21)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Calculando un número al azar...

`<Result> = RandomCryptoInteger()`
---

**`<Result>: (Integro)`**



- Entero aleatorio (incluido entre 0 y 2 147 483 647),

- -1 si se produce un error. 





<a name="SYNTAX2"></a>

### El cálculo de un número aleatorio con un límite superior

`<Result> = RandomCryptoInteger(<Upper bound>)`
---

**`<Result>: (Integro)`**



- Entero aleatorio (incluido entre 0 y el límite superior, inclusive)

- -1 si se produce un error.




**`<Upper bound>: (Integro)`**

Límite superior del número devuelto (incluido entre 1 y 2 147 483 647).


<a name="SYNTAX3"></a>

### El cálculo de un número aleatorio con un límite inferior y otro superior

`<Result> = RandomCryptoInteger(<Lower bound> , <Upper bound>)`
---

**`<Result>: (Integro)`**



- Entero aleatorio (incluido entre los límites inferior y superior, ambos inclusive),

- -1 si se produce un error.




**`<Lower bound>: (Integro)`**

Límite inferior del número devuelto (incluido entre 0 y 2 147 483 647)

**`<Upper bound>: (Integro)`**

Límite superior del número devuelto (incluido entre 1 y 2 147 483 647)



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La primera llamada a la función puede ser bastante larga (alrededor de un segundo) porque inicializar el mecanismo aleatorio es muy complejo.

- La función [InitRandom](../WDLang1/3013030.md) no tiene ningún efecto en el resultado de la función **RandomCryptoInteger**.

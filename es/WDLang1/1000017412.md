


## TableSavePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra: 

- la posición de la barra de desplazamiento en una Tabla o control Tabla TreeView,

- los elementos seleccionados en el control. 




Estos elementos pueden ser restaurados por [TableRestorePositionAndSelection](../WDLang1/1000017413.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SaveTable is string
SaveTable = TableSavePositionAndSelection(TABLE_Table1)
TableDisplay(TABLE_Table1, taInit)
TableRestorePositionAndSelection(TABLE_Table1, SaveTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableSavePositionAndSelection(<Table control>)`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [TableRestorePositionAndSelection](../WDLang1/1000017413.md). 

- Cadena vacía ("") si la control Tabla está vacía. 




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000076"></a>

La función **TableSavePositionAndSelection** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.



<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000091"></a>

Las funciones **TableSavePositionAndSelection** y [TableRestorePositionAndSelection](../WDLang1/1000017413.md) equivalen a la opción "Recordar posición y selección" disponible en la pestaña "Detalles" de la descripción del control Tabla.
<a name="NOTE0_3"></a>


### Limitación
<a name="limitacion_ELTPARAGRAPHE000109"></a>

La función **TableSavePositionAndSelection** no funciona en controles Tabla archivo de datos con rebote (es decir, sin barra de desplazamiento proporcional).

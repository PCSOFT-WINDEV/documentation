
## Constantes para la gestión de proyectos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes constantes se utilizan para gestionar los proyectos:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **enumAll** | Enumera todos los elementos del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumContextMenu** | Enumera los menús emergentes que se encuentran en una ventana. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumDynamicPage** | Enumera todas las páginas dinámicas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumExternalComponent** | Enumera los elementos públicos de un componente externo incluido en el proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumFirst** | Inicializa la enumeración. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumGPU** | Constante destinada al User Groupware. | [EnumElement](../WDLang1/3064001.md) |
| **enumGroup** | Enumera todos los grupos de ventanas. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumInternalPage** | Enumera todas las páginas internas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumInternalWindow** | Enumera todas las ventanas internas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumMainMenu** | Enumera los menús desplegables que se encuentran en una ventana. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumPage** | Enumera todas las páginas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumPageTemplate** | Enumera las plantillas de página del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumPane** | Enumera los paneles de pestañas. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumQuery** | Enumera todas las consultas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumReport** | Enumera todos los informes de proyectos. | [EnumElement](../WDLang1/3064001.md) |
| **enumReportTemplate** | Enumera las plantillas de informes del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumSideBar** | Enumera los paneles que se encuentran en una barra lateral. | [EnumSubElement](../WDLang1/3064002.md) |
| **enumWindow** | Enumera todas las ventanas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **enumWindowTemplate** | Enumera las plantillas de ventanas del proyecto. | [EnumElement](../WDLang1/3064001.md) |
| **piAnalysis** | Tipo de información a devolver: Nombre físico sin extensión del archivo.WDDD utilizado por el proyecto o por el componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationCCDataPath** | Tipo de información a devolver: Ruta del directorio de datos de los Centros control definida al crear el ejecutable, la biblioteca o el componente.<br>Si esta constante se utiliza en un componente, da la ruta del directorio de datos para la aplicación que la utiliza. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationEmail** | Tipo de información a devolver: Email definido al crear el ejecutable, la biblioteca o el componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piApplicationInternetAddress** | Tipo de información a devolver: INTERNET Address definido al crear el ejecutable, la biblioteca o el componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCCDataPath** | Tipo de información a devolver: Ruta del directorio de datos de los Centros control definida al crear el ejecutable, la biblioteca o el componente.<br>Si esta constante se utiliza en el proyecto, da el ruta del directorio de datos correspondiente al proyecto.. Si esta constante se utiliza en un componente, da el ruta del directorio de datos correspondiente al componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCompany** | Tipo de información a devolver: Nombre de la empresa especificada al crear el ejecutable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piConfiguration** | Tipo de información a devolver: Nombre de la configuración del proyecto utilizada para generar el ejecutable o el componente actualmente en ejecución. | [ProjectInfo](../WDLang1/3064004.md) |
| **piCopyright** | Tipo de información a devolver: Copyright definido al crear el ejecutable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piDescription** | Tipo de información a devolver: description de compañía definida al crear el ejecutable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piEmail** | Tipo de información a devolver: Email definido al crear el ejecutable, la biblioteca o el componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piEXEName** | Tipo de información a devolver: Nombre del ejecutable especificado al crear el ejecutable. | [ProjectInfo](../WDLang1/3064004.md) |
| **piGenerationDate** | Tipo de información a devolver: Fecha de generación de la aplicación o biblioteca de componentes (BDM). | [ProjectInfo](../WDLang1/3064004.md) |
| **piInternetAddress** | Tipo de información a devolver: INTERNET Address definido al crear el ejecutable, la biblioteca o el componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNameCurrentProject** | Tipo de información a devolver: Nombre del proyecto en el que se llama ProjectInfo.<br>Si esta constante se utiliza en el código del proyecto principal, el resultado es el nombre del proyecto principal.<br>Si esta constante se utiliza en un componente, el resultado corresponde al nombre del proyecto de componente. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNameDeployedSite** | Tipo de información a devolver: Nombre real del sitio desplegado (para un sitio WEBDEV durante un multi-sitio implementación). | [ProjectInfo](../WDLang1/3064004.md) |
| **piNewFeaturesHelpContext** | Tipo de información a devolver: Número de contexto de la página a mostrar en el archivo de ayuda para las nuevas funciones. | [ProjectInfo](../WDLang1/3064004.md) |
| **piNewFeaturesHelpName** | Tipo de información a devolver: Nombre del archivo de ayuda para las nuevas funciones. | [ProjectInfo](../WDLang1/3064004.md) |
| **piProjectName** | Tipo de información a devolver: Nombre de proyecto. | [ProjectInfo](../WDLang1/3064004.md) |
| **piRegistry** | Tipo de información a devolver: Ruta en el registro donde se puede guardar la información específica del proyecto. | [ProjectInfo](../WDLang1/3064004.md) |

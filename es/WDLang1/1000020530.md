
## RibbonHighlightPane (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Destaca una pestaña en una control Cinta de opciones. Resaltar una pestaña consiste en mostrar una nueva pestaña sobre la pestaña especificada, con el texto especificado y Color.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on an image
RibbonHighlightPane(RIBBON_Menu, 5, "Image tools", LightMagenta)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RibbonHighlightPane([<Ribbon control>, ] <Index> , <Text> [, <Color>])`
---

**`<Ribbon control>: (Nombre de control opcional)`**

Nombre de la control Cinta de opciones a utilizar. Si no se especifica este parámetro, manipula la control Cinta de opciones de la ventana actual. 

**`<Index>: (Integro)`**

Índice de la pestaña a utilizar. 

**`<Text>: (Cadena de caracteres)`**

Texto a mostrar durante el resaltado. Si este parámetro corresponde a una cadena vacía ("""), se eliminará el resaltado.. 

**`<Color>: (Entero o constante de tipo Integer (opcional))`**

Destacando a Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).  


Esta Color se aplica a toda la pestaña adicional. Si no se especifica este parámetro, la Color utilizada corresponde a la pestaña Color. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La pestaña resaltada no está seleccionada. 

- La control Cinta de opciones debe encontrarse en la ventana actual.

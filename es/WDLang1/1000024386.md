


## Funciones de gestión de hilos (sintaxis con prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar subprocesos: 



|   |   |
| --- | --- |
| [&lt;AutomaticEvent Variable&gt;.Abrir](../WDLang1/1000024385.md) | Abre una sincronización Event entre varios hilos. |
| [&lt;AutomaticEvent Variable&gt;.Cerrar](../WDLang1/1000024383.md) | Cierra una sincronización Event entre varios hilos. |
| [&lt;Hilo&gt;.RequestStop](../WDLang1/1000024924.md) | Envía una solicitud para detener un hilo. |
| [&lt;Hilo&gt;.SendSignal](../WDLang1/1000024926.md) | El hilo actual envía una señal al hilo que está en espera para desbloquearlo. |
| [&lt;ManualEvent Variable&gt;.Cerrar](../WDLang1/1000024382.md) | Cierra una sincronización Event entre varios hilos. |
| [&lt;ManualEvent Variable&gt;.Open](../WDLang1/1000024384.md) | Abre una sincronización Event entre varios hilos. |
| [&lt;Variable Thread&gt;.Wait](../WDLang1/1000024880.md) | Espera a que finalice la ejecución del hilo especificado. |





Las siguientes variables permiten gestionar eventos: 



|   |   |
| --- | --- |
| [AutomaticEvent](../WDLang1/1000021270.md) | El tipo **AutomaticEvent** permite gestionar un evento automático. |
| [ManualEvent](../WDLang1/1000021275.md) | El tipo **ManualEvent** permite manipular un evento manual. |

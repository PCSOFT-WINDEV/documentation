
## SysSerialNum( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número IMEI del dispositivo actual.

Esta función es equivalente a la función [SysIMEI](../WDLang1/3073029.md). 

Atención: Esta información está disponible hasta Android 9. A partir de Android 10, esta información ya no está disponible: esta función causa un error fatal.  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF Val(SysAndroidVersion(sysVersionNumber))<10 THEN	  
	// Retrieve the serial number of the current phone
	Info("Phone serial number: " + SysSerialNumber())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysSerialNum()`
---

**`<Result>: (Cadena de caracteres)`**



- Número de serie de la actual Android Phone,

- Cadena vacía ("") si se produce un error.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000054"></a>

- [SysIMEI](../WDLang1/3073029.md) obtiene el número IMEI de un Phone. 

- Los dispositivos que no tienen una ranura para tarjeta SIM no tienen un número IMEI (la mayoría de las tabletas)..


## FullScreen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa o desactiva el modo de pantalla completa en las aplicaciones en modo Universal Windows 10 App. 

El modo de pantalla completa consiste en ocultar: 

- la barra del sistema cuando la aplicación se ejecuta en un PC,

- la barra del sistema y la barra de navegación en un dispositivo móvil. 


Estas barras pueden volver a mostrarse al pasar el ratón por la zona. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
FullScreen(True)
FullScreen(False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`FullScreen([<Activation>])`
---

**`<Activation>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para activar el modo de pantalla completa, 

- <u><u><u><u>False</u></u></u></u> para desactivar el modo de pantalla completa.


## Procedimiento description (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo de procedimiento **description** se utiliza para get información sobre un método de clase. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the definition of type class of variable x
MyDefinition is Definition
MyDefinition = GetDefinition(x)

// Retrieves the description of Calc method
MyProcedure is procedure description 
MyProcedure = MyDefinition.Procedure["Calc"]
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Propiedades específicas de las variables description de Procedimiento description
<a name="propiedades_especificas_las_variables_description_procedimiento_description_ELTPARAGRAPHE000046"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un procedimiento description Variable:


|   |   |   |
| --- | --- | --- |
| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| associated | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el método proviene de una clase asociada,<br><br>- <u><u><u><u>False</u></u></u></u> si el método proviene de una clase base. <br><br><br> |
| Declaración | Definition | Definition de la clase que contiene el método. <br>**Esta propiedad es de solo lectura**. |
| Global | booleano | <br><br>- <u><u><u><u>True STRIMEULB</u></u></u></u> si el método es global,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**. |
| Heredado | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el método se hereda de una clase base,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**. |
| denominar | Cadena de caracteres | Nombre del método. <br>**Esta propiedad es de solo lectura**. |
| Virtual | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el método es virtual,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**. |


<a name="NOTE0_2"></a>


### Llamada al método
<a name="llamada_metodo_ELTPARAGRAPHE000136"></a>

El método cuya description fue recuperada puede ser llamado por [ExecuteProcess](../WDLang1/3013031.md).

- Llamada a un método estático
	
	```txt
	p is Procedure Description
	p = ...
	ExecuteProcess(Null, p, <param1>, <param2>, ...)
	```


- Llamada a un método de instancia
	
	```txt
	p is Procedure Description
	p = ...
	ExecuteProcess(<object variable>, p, <param1>, <param2>, ...)
	```


## SchedulerTaskProperties( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee las propiedades de una tarea programada y actualiza la estructura [Tareas programadas](../WDLang1/3089004.md).

**Recordatorio**: Los eventos que desencadenarán una tarea (los horarios de la tarea) son devueltos por [SchedulerTriggerProperties](../WDLang1/3089010.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the properties of MyTask
SchedulerTaskProperties("MyTask")
Info("The MyTask task is associated with the application " + ScheduledTask.Application)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SchedulerTaskProperties(<Task name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las propiedades de la tarea se leyeron correctamente,

- <u><u><u><u>False</u></u></u></u> si se produjo un error (por ejemplo, si la tarea no existe).




**`<Task name>: (Cadena de caracteres)`**

Nombre de la tarea associated con el programa de recuperación. Este nombre fue definido cuando la tarea fue creada por [ProgramadorAñadirTarea](../WDLang1/3089003.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Información no recuperada: La contraseña
<a name="informacion_recuperada_contrasena_ELTPARAGRAPHE000060"></a>

Al utilizar la función **SchedulerTaskProperties**, la variable ScheduledTask.Password se inicializa automáticamente con una cadena vacía ("").

Durante la siguiente llamada a [ProgramadorModificarTareas](../WDLang1/3089008.md), la contraseña se almacenará si no se ha asignado específicamente un nuevo valor a la tarea programada.Contraseña Variable (por asignación directa o a través de [ProgramadorReiniciar](../WDLang1/3089012.md)).
<a name="NOTE0_2"></a>




### Información adicional rellenada sólo por SchedulerTaskProperties
<a name="informacion_adicional_rellenada_solo_por_docparampagetitleshort_ELTPARAGRAPHE000078"></a>

Al utilizar **SchedulerTaskProperties**, se actualizan automáticamente varias variables de la estructura [Tareas programadas](../WDLang1/3089004.md). Estas variables son de sólo lectura. Estas variables son las siguientes:


|   |   |
| --- | --- |
| Código de error de tarea programada | Integro.<br>Último código de error devuelto por la tarea (0 si la tarea se completó con éxito o si el programa nunca se ha iniciado).<br>Esta Variable es de sólo lectura y se actualiza con **SchedulerTaskProperties**. |
| Cuenta de desencadenamiento de tareas programadas | Integro.<br>Número de programas de tareas (número de eventos que activarán la tarea).<br>Esta Variable es de sólo lectura y se actualiza con **SchedulerTaskProperties**. |
| Fecha de última ejecución de la tarea programada | FechaHora.<br>Fecha de la última ejecución de la tarea.<br>Esta Variable es de sólo lectura y se actualiza con **SchedulerTaskProperties**. |
| Tareas programadas SiguienteEjecutarHora | FechaHora.<br>Fecha de la siguiente ejecución de la tarea programada.<br>Esta Variable es de sólo lectura y se actualiza con **SchedulerTaskProperties**. |

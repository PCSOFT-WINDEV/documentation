
## MDIMother( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el nombre de la ventana principal de MDI.

**Observación**: **MDIMother** sólo puede utilizarse para las ventanas definidas como "MDI padre" en el editor de ventanas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the MDI parent window
ResParentWindow = MDIMother()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MDIMother()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la ventana principal de MDI,

- Cadena vacía (""") si no existe ninguna ventana padre MDI.


## EmulateWindow( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la prueba que se ejecutará durante la siguiente apertura de la ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the window named "WIN_MainWindow"
// Run "Scenario1"
EmulateWindow(WIN_MainWindow, "Scenario1")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EmulateWindow(<Window name> , <Test Name>)`
---

**`<Window name>: (Cadena de caracteres)`**

Nombre de la ventana para la que se ejecutará &lt;Test Name&gt; durante la próxima apertura.

**`<Test Name>: (Cadena de caracteres)`**

Nombre de la prueba que se ejecutará durante la próxima apertura de &lt;Nombre de la ventana&gt;..

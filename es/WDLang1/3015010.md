
## EndEvent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la interceptación de un Event de Windows en un control o ventana de WINDEV (implementado por [Evento](../WDLang1/3015011.md)). La Procedure asociada a una Event de Windows ya no se llamará más.

**Observación**: La interceptación de eventos se detiene automáticamente cuando se cierra la ventana de associated.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Capture the mouse click on an edit control
WM_LBUTTONDOWN is int = 513
EventID is int
EventID = Event("MouseClick", "EDT_EDIT1", WM_LBUTTONDOWN)
...
// Stop intercepting WM_LBUTTONDOWN on the "EDT_EDIT1" control
EndEvent(EventID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = EndEvent(<Event number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se canceló la interceptación del evento,

- <u><u><u><u>False</u></u></u></u> si se presenta un problema (número de evento incorrecto, por ejemplo).




**`<Event number>: (Integro)`**

Número de la Event a Cancel (devuelto por [Evento](../WDLang1/3015011.md)).

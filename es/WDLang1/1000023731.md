


## &lt;Hoja de cálculo&gt;.GetMerge (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite averiguar si una célula que se encuentra en un control Hoja de cálculo se fusiona con otras células y a get el nombre de las células fusionadas. 

Observación: Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sMerge is string = PSHEET_Spreadsheet.GetMerge("B1")
IF sMerge = "" THEN
	Info("No merge")
ELSE
	sMin is string = ExtractString(sMerge, 1, ":")
	sMax is string = ExtractString(sMerge, 2, ":")
	Info("Merge between " + sMin + " and " +  sMax)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperando las características de una célula fusionada identificada por su nombre

`<Result> = <Spreadsheet control>.GetMerge(<Cell>)`
---

**`<Result>: (Cadena de caracteres)`**



- Vaciar la cadena (""") si la celda no está fusionada,

- Rango de celdas combinadas en el siguiente formato: 
	
	```txt
	"<Name top left cell>:<Name bottom right cell>"
	```
Por ejemplo: "A2:C5"




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Cell>: (Cadena de caracteres)`**

Nombre de la celda a utilizar. Por ejemplo: "A4".


<a name="SYNTAX2"></a>

### Recuperando las características de una célula fusionada identificada por su columna y por su fila

`<Result> = <Spreadsheet control>.GetMerge(<Column> , <Row>)`
---

**`<Result>: (Cadena de caracteres)`**



- Vaciar la cadena (""") si la celda no está fusionada,

- Rango de celdas combinadas en el siguiente formato: 
	
	```txt
	"<Name top left cell>:<Name bottom right cell>"
	```
Por ejemplo: "A2:C5"




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Column>: (Integro)`**

Número de columna para la celda a utilizar.

**`<Row>: (Integro)`**

Número de Fila de la celda a utilizar.

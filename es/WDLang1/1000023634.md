


## &lt;WinRedraw (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vuelve a dibujar inmediatamente los controles de ventana para los que se requiere una actualización gráfica.

**Observación**: Esta función es equivalente a la función [MultitaskRedraw](../WDLang1/3038004.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
i is int 
FOR i = 1 TO 1000
	LongProcess()
	WIN_Customer.Redraw()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Window>.Redraw()`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana a redibujar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### comparison con Multitask
<a name="comparison_con_multitask_ELTPARAGRAPHE000207"></a>

A diferencia de la función [Multitask](../WDLang1/3015004.md), el código actual no se interrumpe al llamar a la función **&lt;WinRedraw** (no se llama a las funciones Timer, por ejemplo).

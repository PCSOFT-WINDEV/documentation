


## VideoStop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la reproducción o grabación de una secuencia de vídeo en una control Cámara.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops playing the video sequence currently transmitted
// in the "CAM_Kitchen" control
ResStop = VideoStop(CAM_Kitchen, viStopDisplay)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = VideoStop(<Camera control> [, <Action to stop>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si la visualización y/o la grabación en el control Cámara se detiene correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<Action to stop>: (Constante opcional de tipo Integer)`**

Tipo de acción a detener:


|   |   |
| --- | --- |
| *viStopAll*<br>(Valor predeterminado) | Detiene la visualización y grabación del vídeo. El fichero generado está disponible. La control Cámara especificada se vacía. |
| *viStopCapture* | Detiene la grabación del vídeo. El fichero generado está disponible. |
| *viStopDisplay* | Detiene la visualización del vídeo. La control Cámara especificada se vacía. |

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000096"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una "cámara" al ordenador actual.


**Observación**: Esta función es equivalente a la función [CameraStop](../WDLang1/1410086596.md).
<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000123"></a>

- Para guardar el vídeo transmitido por una "cámara web", utilice [VideoCapture](../WDLang1/3079007.md)

- Para visualizar la secuencia de vídeo transmitida por una "cámara web" en un control Cámara, utilice [VideoDisplay](../WDLang1/3079004.md).




## TableAdjust (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ajusta el título y el tamaño de las columnas que se encuentran en una Tabla o control Tabla TreeView. Es posible definir: 

- el título de la columna,

- el ancho de columna,

- la alineación de los elementos de la columna,

- el tamaño de entrada de la columna.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the titles and width of the two
// first columns found in the "TABLE_ProductTable" control
TableAdjust(TABLE_ProductTable, "Ref" + TAB + "50" + TAB + ...
	"C" + TAB + "12" + RC + "Designation" + TAB + "200" + ...
	TAB + "G" + TAB + "12")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TableAdjust(<Table control> [, <Format>])`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla, 

- un control Tabla TreeView.


Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control al que pertenece el actual Event.

**`<Format>: (Cadena de caracteres opcional)`**

Este parámetro tiene el siguiente formato: 

```txt
<Title of column 1> + TAB + <Size of column 1> + TAB + ...
<Alignment of elements> + TAB + <Input size> + CR + ...
...
<Title of column N> + TAB + <Size of column N> + TAB + ...
<Alignment of elements> + TAB + <Input size>
```
donde:


|   |   |   |
| --- | --- | --- |
| &lt;Título de la columna&gt; | Cadena de caracteres | Modifica el título de la columna. |
| &lt;Tamaño de la columna&gt; | Integro | Modifica el tamaño de la columna (en caracteres). |
| &lt;Alineación de elementos&gt;&gt;Alineación de elementos | Carácter | Modifica la alineación de los elementos de la columna:<br><br>- "C": Centraliza los elementos de la columna<br><br>- "G": Alinea los elementos a la izquierda<br><br>- "D": Alinea los elementos a la derecha<br><br><br> |
| &lt;Tamaño entrada&gt; | Integro | Modifica el número de caracteres que se pueden escribir en la columna (en número de caracteres y no en píxeles).. <br>Observación: El ancho de un carácter corresponde al ancho de la letra "E" en la fuente utilizada para el título de control. |


Si no se especifica este parámetro, el ancho de las columnas se ajusta al menor ancho posible para que los datos sean Visible.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro es obligatorio.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000113"></a>

La función **TableAdjust** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.



<a name="NOTE0_2"></a>


### Creación/eliminación de columnas
<a name="creacioneliminacion_columnas_ELTPARAGRAPHE000128"></a>

La función **TableAdjust** no crea y no elimina columnas. Las columnas de control deben haber sido descritas en el editor de la ventana de antemano. Asegúrese de que existe un número suficiente de columnas antes de manipularlas con la función **TableAdjust**.

Si el número de columnas del control es superior al número de columnas manipuladas con la función **TableAdjust**, las últimas columnas no tendrán título y tendrán un tamaño nulo.
<a name="NOTE0_3"></a>


### Columnas Invisible
<a name="columnas_invisible_ELTPARAGRAPHE000145"></a>

**Si las columnas manipuladas con la función TableAdjust** son invisibles, estas columnas se harán visibles automáticamente.
<a name="NOTE0_4"></a>


### Equivalencia entre TableAdjust con las propiedades de control
<a name="equivalencia_entre_docparampagetitleshort_con_las_propiedades_control_ELTPARAGRAPHE000156"></a>

Las propiedades [Título](../Proprietes/2510090.md), [anchura](../Proprietes/2510059.md), [envergadura](../Proprietes/2510125.md) y [HorizontalAlignment](../Proprietes/2510023.md) tendrán el mismo efecto que la función **TableAdjust**.

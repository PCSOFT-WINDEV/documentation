
## WinGiveSuitableHeight (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la altura de una ventana, adaptada al contenido de los controles.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the Table control
TableDisplay(WIN_Price.TABLE_Price)
// Calculates the proper height
nHeightOK is int
nHeightOK = WinGiveSuitableHeight(WIN_Price)
// Resizes the window
IF nHeightOK < 500 THEN
	WIN_Price..Height = nHeightOK
ELSE
	WIN_Price..Height = 500
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WinGiveSuitableHeight([<Window>])`
---

**`<Result>: (Integro)`**

Altura adaptada al contenido de los mandos para optimizar la visualización de los mandos en la ventana. El alto se expresa en pixeles. 

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana a utilizar. La ventana actual se utilizará si no se especifica este parámetro.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El cálculo de la altura adaptada de una ventana sólo es posible si la ventana contiene uno de los siguientes controles anclados en altura:

- Un control List Box, 

- Control Tabla, 

- Control Looper, 

- Control TreeView,

- Control ListView, 

- Control Campo de entrada multilínea.




## &lt;SelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un elemento de un Cuadro de Listas, ListView o control Combo Box.

**Recordatorio**: El elemento seleccionado corresponde al elemento actual en el que se visualiza el banda de selección.

**Observación**: La función **&lt;SelectMinus** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselect the element at index 7 from "LIST_CustomerList"
LIST_CustomerList.SelectMinus(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<List Box control>.SelectMinus([<Index 1> [... [, <Index N>]]])`
---

**`<List Box control>: (Nombre del control)`**

Nombre de la caja de la lista (o Combo Box) control a utilizar.

**`<Index 1>: (Entero opcional)`**

Índice del primer elemento a deseleccionar. 
Si no se especifica ningún índice, se deseleccionan todos los elementos del cuadro de lista o control Combo Box especificados. 
Si estos índices corresponden a elementos no seleccionados, la función **&lt;SelectMinus** no tiene ningún efecto.



**`<Index N>: (Entero opcional)`**

Índice del Nº elemento a deseleccionar. 
Si no se especifica ningún índice, se deseleccionan todos los elementos del cuadro de lista o control Combo Box especificados. 
Si estos índices corresponden a elementos no seleccionados, la función **&lt;SelectMinus** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Control Combo Box
<a name="control_combo_box_ELTPARAGRAPHE000244"></a>

Si se utiliza la función **&lt;SelectMinus** con un control Combo Box:

- no se muestra ningún elemento de control Combo Box,

- si la control Combo Box es editable, se borra la control Campo de entrada.



<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000259"></a>

- Para anular la selección de un elemento de una control Tabla, utilice [&lt;Tabla&gt;.SelectMinus](../WDLang1/1000024285.md).

- **&lt;SelectMinus** puede utilizarse en una columna "Combo Box" tabla.

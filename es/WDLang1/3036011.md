
## fTempFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de un archivo temporal único.

**Observación**: **fTempFile** no crea el archivo temporal correspondiente. Para crear y abrir un archivo temporal, utilice [fOpenTempFile](../WDLang1/1000017289.md).

<a name="sample_code"></a>

## Ejemplo

## Sintaxis

`<Result> = fTempFile([<Prefix of temporary file> [, <Directory of temporary file>]])`
---

**`<Result>: (Cadena de caracteres)`**



- ruta completa de un archivo temporal único. 

- Vaciar la cadena ("") si se produce un error (no se puede get la ruta temporal del sistema). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. 




**`<Prefix of temporary file>: (Cadena de caracteres opcional)`**

Prefijo (3 letras) para el nombre del archivo temporal. Este parámetro se utiliza para prefijar el nombre del archivo temporal. Sólo se tendrán en cuenta las tres primeras letras.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Directory of temporary file>: (Cadena de caracteres opcional)`**

Directorio completo o relativo del fichero temporal. Se puede utilizar una ruta UNC. Este nombre de directorio puede (o no) terminar con el carácter "\\".. Este directorio debe existir. De lo contrario, se producirá un error de WLanguage.
Si no se especifica este parámetro, el directorio Default será:

- el directorio temporal de Windows (si este directorio existe en el ordenador actual),

- el directorio actual (si el directorio temporal de Windows no existe en el ordenador actual).




![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **fTempFile** solo permite obtener el nombre de un archivo temporal único. Esta función no se puede utilizar para crear el archivo temporal correspondiente.. Para ello, utilice [fCreate](../WDLang1/3036017.md) o [fOpenTempFile](../WDLang1/1000017289.md).

Atención: Por razones de seguridad, **fTempFile** borra el archivo temporal si ya existe. Si el archivo temporal es utilizado por otro Process, se producirá un error no fatal.

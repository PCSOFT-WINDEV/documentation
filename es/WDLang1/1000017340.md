


## CalendarioSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desmarca una fecha en una selección múltiple control Calendario. 

 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselects the day corresponding to today
CalendarSelectMinus(CAL_Calendar1, Today())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`CalendarSelectMinus(<Calendar control> , <Date to delete>)`
---

**`<Calendar control>: (Nombre del control)`**

Nombre de control Calendario utilizado. "Selección múltiple" debe estar marcada para este control (pestaña "Detalles" del control description). La función no tiene ningún efecto si esta opción no está seleccionada.

**`<Date to delete>: (Cadena de caracteres)`**

Fecha en formato AAAAMMDDD que debe borrarse de la selección. La función no tiene ningún efecto si no se selecciona esta fecha..


## Mensaje (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra o elimina un mensaje de la barra de estado de la ventana actual o del navegador.

**Observación**: La ventana o la ventana del navegador deben tener una barra de estado.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la barra de estado está dividida en varias celdas, puede especificar la celda en la que se muestra el mensaje.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Message("Processing...")
MyProcess() // Procedure used to perform the requested process
Message("Ready")
```




<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Message in the "MCB_Info" cell
Message(MCB_Info, "Enter your name")
```

## Sintaxis

### Mostrar o borrar un mensaje en la barra de estado

`Message([<Text>])`
---

**`<Text>: (Cadena de caracteres opcional)`**

Texto a mostrar (hasta 127 caracteres).
Si no se especifica este parámetro, se borra el mensaje mostrado anteriormente en la barra de estado..
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la barra de estado está dividida en varias celdas, este texto se mostrará en la primera celda.



<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Mostrar o borrar un mensaje en una celda de la barra de estado

`Message(<Cell name> , <Text>)`
---

**`<Cell name>: (Cadena de caracteres)`**

Nombre de la celda a utilizar. Si la celda fue creada:

- en el editor de ventanas, el nombre de la celda se puede especificar sin comillas.

- por [StatusBarAddCell](../WDLang1/3021001.md), el nombre de la celda debe especificarse entre comillas (también tiene la posibilidad de utilizar la palabra clave [Externo](../Motscles/1511013.md)).




**`<Text>: (Cadena de caracteres)`**

Texto a mostrar (hasta 127 caracteres).
Si no se especifica este parámetro, se borra el mensaje mostrado anteriormente en la barra de estado..

## Observaciones


### Características del mensaje
<a name="caracteristicas_del_mensaje_ELTPARAGRAPHE000119"></a>

El mensaje está alineado a la izquierda.

El texto que aparece en la barra de estado o en la primera celda de una barra de estado de varias celdas se borra automáticamente cuando el usuario selecciona un control associated con un mensaje de ayuda: el mensaje de ayuda de este control se mostrará en la barra de estado.
<a name="NOTE0_2"></a>




### Error generado
<a name="error_generado_ELTPARAGRAPHE000129"></a>

Se produce un error de WLanguage si se llama a **Mensaje** desde un Process del "Código del proyecto" (inicialización del proyecto, cierre del proyecto o cierre de una sesión).
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Mensaje y ventana secundaria MDI
<a name="mensaje_ventana_secundaria_mdi_ELTPARAGRAPHE000140"></a>

**Mensaje** , usado desde una ventana secundaria MDI sin barra de estado, usa la barra de estado de la ventana principal.

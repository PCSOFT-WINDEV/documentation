


## &lt;Deshacer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deshace la última acción realizada en una control Procesador de texto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Undo the last action performed on the control
WP_MyDocument.Undo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.Undo()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el último cambio fue deshecho, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.



<a name="XComponent"></a>

## Componente:
wd280mdl.dll

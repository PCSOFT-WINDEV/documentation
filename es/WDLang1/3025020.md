


## ControlNext( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la próxima control en edición.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the next control in edit
ResControl = ControlNext()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlNext()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la próxima Control en edición. El caso del nombre de Control (caracteres en mayúsculas/minúsculas) corresponde al utilizado cuando se creó la Control en el editor.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **ControlNext** sólo puede ser llamado en la "Salida" Event de una Control. Si se llama a **ControlNext** en otro evento o Process, el nombre del control devuelto no tendrá sentido.

- Cuando el usuario pulsa la tecla TAB, **ControlNext** devuelve el nombre de control que sigue al actual en el orden de tabulación definido en el editor. 
	Recordatorio: Para set la orden de tabulación, en el panel "Ventana", en el grupo "Orden", expanda "Orden tab." y seleccione "Editar".

- Por el contrario, cuando el usuario hace clic en un control, **ControlNext** devuelve el nombre del control que ha sido pulsado por el usuario.

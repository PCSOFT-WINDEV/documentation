
## BreakDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las pausas de un control Tabla o de un control Looper.. Estas pausas han sido añadidas previamente por [BreakAdd](../WDLang1/1000022737.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Groups the customers by Zip Code (ZC)
// 1. Delete the former breaks
BreakDeleteAll(TABLE_Customer)
// 2. Create the new break
BreakAdd(TABLE_Customer.ZC, "Brk_ZC")
TableSort(TABLE_Customer, "ZC")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BreakDeleteAll(<Control to use>)`
---

**`<Control to use>: (Nombre del control)`**

Nombre del control a manipular: 

- Control Tabla.

- Control Looper. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **BreakDeleteAll** elimina todas las rupturas creadas con la función [BreakAdd](../WDLang1/1000022737.md). La función no tiene ningún efecto si no se ha creado una ruptura. 

- Para eliminar una sola interrupción, utilice [RomperBorrar](../WDLang1/1000022738.md).

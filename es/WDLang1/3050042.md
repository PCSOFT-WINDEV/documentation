
## FinPeriodInterest( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula para un período determinado el importe de los intereses de un préstamo amortizado mediante pagos periódicos fijos con un tipo de interés fijo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Amount of interests due for the second year of a loan.
// This is a loan of $25000 with a 5% interest rate over 6 years.
ResInterestPeriod = FinPeriodInterest(25000, 5, 2, 6)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinPeriodInterest([<Capital> [, <Rate> [, <Period> [, <Duration>]]]])`
---

**`<Result>: (Real o moneda)`**

Importe de los intereses. Esta cantidad puede ser:

- negativo: se debe hacer un pago;

- constructivo: un crédito debe ser hecho.




**`<Capital>: (Real o moneda opcional)`**

Cantidad del préstamo (0 por Default).

**`<Rate>: (Real opcional)`**

Tasa de interés, expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set a 5.

**`<Period>: (Entero opcional)`**

Período sobre el que debe calcularse el importe de los intereses (1 de Default).

**`<Duration>: (Real opcional)`**

Número de períodos para el préstamo (1 por Default).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Unidad de tiempo
<a name="unidad_tiempo_ELTPARAGRAPHE000078"></a>

Los cálculos se pueden realizar según meses o años. Los valores de &lt;Tasa&gt;, &lt;Duración&gt; y &lt;Período&gt; deben corresponder a la misma unidad.

Por lo tanto, a get el importe de los intereses adeudados en el 5º mes de un préstamo a 3 años, con un tipo de interés anual de 5%, que puede utilizar:

- unidades expresadas en meses (por ejemplo, para un pago mensual):

	- &lt;Duración&gt; = 3 x 12 meses = 36

	- Período &lt;período&gt; = 5

	- &lt;Tasa&gt; = 5% / 12




- unidades expresadas en años (por ejemplo, para un pago anual):

	- &lt;Duración&gt; = 3 años

	- Período &lt;período&gt; = 5/36

	- &lt;Tasa&gt; = 5%

<a name="gestion_errores_ELTPARAGRAPHE000099"></a>

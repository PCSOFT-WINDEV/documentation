


## &lt;Scheduler&gt;.DateVisible ( Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Hace visible un día o Invisible en un control Planificador. 

Observación: El día que se hizo Invisible ya no aparecerá si el control Planificador se muestra semanalmente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Don't display Wednesday
ThisWednesday is Date = DateSys()
ThisWednesday.Day += 3 - DateToDay(ThisWednesday) 
SCH_MySchedule.DateVisible(ThisWednesday, False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Haciendo visible un día o Invisible

`<Scheduler control>.DateVisible(<Date> , <Visible>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha que se hará visible o Invisible. Esta fecha está en formato "AAAAMMDD".

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer visible la fecha,

- <u><u><u><u>False</u></u></u></u> para que la fecha Invisible.





<a name="SYNTAX2"></a>

### Averiguar si un día es visible o Invisible

`<Result> = <Scheduler control>.DateVisible(<Date>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fecha es visible,

- <u><u><u><u>False</u></u></u></u> si la fecha es invisible.




**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha a manipular. Esta fecha está en formato "YYYYMMDD"..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los días de Invisible se "saltan" cuando se dibuja el control Planificador.
	Por ejemplo, si un miércoles es Invisible, el control Planificador muestra el martes y luego el jueves. 

- La pantalla se refresca inmediatamente o al cambiar de período: 

	- Si el control Planificador actualmente muestra sólo un día y ese día se convierte en Invisible, el día se convierte en Invisible cuando se cambia el día o el período. 

	- Si el control Planificador muestra 1 semana o 1 mes, los días del Invisible no se muestran pero el control Planificador no muestra más días.

	- Si el control Planificador muestra X días, los días del Invisible no se cuentan en los X días. Por ejemplo, si el control Planificador muestra 4 días mientras que el miércoles es Invisible, los 4 días mostrados serán: Lunes, martes, jueves, viernes.  




- Si un Appointment dura varios días (entre ellos 1 día Invisible), visualmente aparecerá como más corto.

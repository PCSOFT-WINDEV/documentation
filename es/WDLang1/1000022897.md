


## &lt;Posición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un control Agenda a partir de una fecha especificada o devuelve la fecha del primer día (o la fecha del último día) que aparece en un control Agenda.





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación universal de Windows 10, esta función sólo se puede utilizar para los controles del organizador manipular en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions the organizer on today's date
ORG_Organizer1.Position(Today())
// Positions the organizer on December 20, 2012
ORG_Organizer1.Position("20121220")
```
<a name="Example2"></a>

```wl
// Find out the first date displayed in the organizer
OrganizerDate is Date
OrganizerDate = ORG_MyOrganizer.Position()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Colocación del organizador en una fecha determinada

`<Organizer control>.Position(<Start date>)`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.




**`<Start date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de inicio de la visualización.


<a name="SYNTAX2"></a>

### Averiguar la primera o la última fecha mostrada por la control Agenda

`<Result> = <Organizer control>.Position(<Sought date>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha visualizada por el control Agenda.

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.




**`<Sought date>: (Constante opcional de tipo Integer)`**

Tipo de fecha buscada:


|   |   |
| --- | --- |
| *orgCurrentMonth* | Primer día del mes mostrado en el control Agenda. |
| *orgDateTimeFirstVisible* | Primera fecha y hora que aparecen en la control Agenda (según la posición de la barra de desplazamiento). |
| *orgEndDate* | Última fecha mostrada en el control Agenda. |
| *orgStartDate* <br>(Valor predeterminado) | Primera fecha mostrada en el control Agenda.<br><br> |


**Observación**: Si el control Agenda se muestra en modo "por día", las constantes *orgStartDate* y *orgEndDate* son equivalentes.

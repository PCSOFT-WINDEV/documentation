
## MatDeterminant( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el determinante de una matriz cuadrada (el número de filas es igual al número de columnas).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the determinant of a matrix
ResDeterminant = MatDeterminant("MyMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatDeterminant(<Matrix>)`
---

**`<Result>: (Real)`**

Determinante de la matriz. El determinante de la matriz es nulo si este valor es igual a 0.

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a utilizar, definida con [MatCreate](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000077"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000088"></a>

**Atención**: La función **MatDeterminant** no devuelve ningún código de error. Para saber si se han generado errores al calcular el determinante, utilice [MatError](../WDLang1/3050044.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

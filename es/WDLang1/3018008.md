


## TreeCollapse (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Colapsa una rama que se expandió de antemano.. Los nodos "niños" ya no son Visible en el control TreeView. Los nodos hijo permanecen en su estado anterior (contraído o expandido).

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)
**Reminder**: En tiempo de ejecución, un nodo se colapsa cuando el nombre del nodo está precedido por "+".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Collapse the "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TreeCollapse(TREE_TVRecipe, "Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeCollapse(<TreeView control> , <Node to collapse>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nodo se colapsó,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Node to collapse>: (Cadena de caracteres)`**

Ruta del nodo a colapsar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nodo a colapsar
<a name="nodo_colapsar_ELTPARAGRAPHE000068"></a>

Si el nodo se colapsa:

- no se encuentra, **TreeCollapse** devuelve <u><u><u><u>False</u></u></u></u>.

- es una hoja, **TreeCollapse** devuelve <u><u><u><u>True</u></u></u></u>.

- ya está colapsado, **TreeCollapse** devuelve <u><u><u><u>True</u></u></u></u>.



<a name="NOTE0_2"></a>


### Banda de selección
<a name="banda_seleccion_ELTPARAGRAPHE000099"></a>

- Si la banda de selección se posicionó en uno de los elementos del nodo a colapsar, se posiciona en el nodo colapsado después de la llamada a **TreeCollapse**.

- El código de modificación se llama cada vez que se mueve la banda de selección. Si dos niveles están contraídos y la banda de selección ser posiciona en el nodo superior, se ejecutan los códigos de modificación de cada nivel.

- La ruta completa del elemento seleccionado es devuelta por [TreeSelect](../WDLang1/3018003.md).


## NoteLoadAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Importa y muestra las notas reposicionables de la ventana actual guardadas previamente por [NoteSaveAll](../WDLang1/1000022396.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Export my notes]
NoteSaveAll()
IF ErrorOccurred = True THEN
	Error(ErrorInfo())
END

// Click [Import my notes]
NoteLoadAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`NoteLoadAll([<File>])`
---

**`<File>: (Cadena de caracteres opcional)`**

Nombre del archivo que contiene las notas reposicionables a cargar. 

Si no se especifica este parámetro, las notas cargadas son las que corresponden al archivo: &lt;Nombre de la ventana&gt;.wdnote encontrado en el directorio de datos. El archivo buscado corresponde a: 
```txt
fDataDir()\<MyWindow..Name>.wdnote
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es equivalente a la AAF (automatic Application Features) para importar notas sin mostrar el selector de archivos. Para obtener más información, consulte [note reposicionable (AAF)](../Editeurs/1000021820.md). 

- Los elementos restaurados son: la posición de la nota, su contenido, su fondo Color y las características de la fuente utilizada.

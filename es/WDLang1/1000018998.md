


## Funciones de gestión de claves de activación
			



Las siguientes funciones se utilizan para gestionar las claves de activación:



|   |   |
| --- | --- |
| [KeyCalcActivationKey](../WDLang1/1000018849.md) | Calcula la clave de activación de la aplicación a partir de la clave inicial. |
| [KeyCompareKey](../WDLang1/1000018850.md) | Compara una clave inicial con una clave de activación. |
| [KeyGenerateInitialKey](../WDLang1/1000018848.md) | Genera la clave inicial (número de licencia único) para el sistema de activación de la aplicación. |
| [KeyGetIdentifier](../WDLang1/1000018851.md) | Recupera el identificador que se utilizó para generar una clave inicial. |




## &lt;Hoja de cálculo&gt;.AddWorksheet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una hoja de trabajo a un control Hoja de cálculo que se encuentra en una ventana. 

Observación: Para utilizar esta función, el control Hoja de cálculo debe permitir la gestión de varias hojas de trabajo: 

- En el editor de ventanas: En la pestaña "Detalles" de la ventana del control description, marque "Permitir varias hojas de trabajo". 

- A través de la programación, utilice el [MultiWorksheet](../Proprietes/1000021639.md) Property.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a worksheet
nWorksheet is int 
nWorksheet = PSHEET_MySpreadsheet.AddWorksheet()
// Select the worksheet
PSHEET_MySpreadsheet.CurrentWorksheet = nWorksheet
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.AddWorksheet([<Worksheet name>])`
---

**`<Result>: (Integro)`**

Número de la hoja de trabajo creada en el control Hoja de cálculo.

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Worksheet name>: (Cadena de caracteres opcional)`**

Nombre de la hoja de trabajo a crear. Si no se especifica este parámetro, la hoja de trabajo se crea con un nombre Default ("WorksheetX"). Este nombre: 

- aparece en la pestaña utilizada para seleccionar la hoja de trabajo (en la parte inferior de control Hoja de cálculo). 

- se utiliza para identificar la hoja de trabajo en las fórmulas. Por ejemplo, una fórmula que utiliza una celda de "Hoja de trabajo 2" puede ser: "=SUMA(Hoja de trabajo 2!B4:D4)".






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función falla y muestra un error de WLanguage en los siguientes casos: 

	- La control Hoja de cálculo no gestiona varias hojas de trabajo. 

	- El nombre de la hoja de trabajo no es válido o ya está siendo utilizado por otra hoja de trabajo.. 




- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..




## &lt;Hoja de cálculo&gt;.InsertRow (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una fila vacía en una control Hoja de cálculo que se encuentra en una ventana..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Inserts several rows
FOR I = 2 TO 5
	PSHEET_Spreadsheet.InsertRow(I)
 END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.InsertLine([<Row>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Row>: (Entero opcional)`**

Número de la fila ante la cual se insertará la nueva fila. Por ejemplo, si este parámetro es set a 2, el nuevo fila se insertará en la posición 2. Las filas existentes se moverán al final y las fórmulas se actualizarán.. 
Si no se especifica este parámetro y si se selecciona un fila en el control, la inserción se realizará antes del fila seleccionado. De lo contrario (sin fila seleccionada), la función no tiene ningún efecto..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La operación de inserción se añade a la lista de operaciones de Cancel: el usuario puede deshacer esta operación a través del menú contextual o pulsando Ctrl + Z.

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..




## Funciones de gestión de eventos de Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones permiten gestionar eventos: 



|   |   |
| --- | --- |
| [EndEvent](../WDLang1/3015010.md) | Cancela la interceptación de una Event de Windows en un control o ventana WINDEV (implementado por [Event](../WDLang1/3015011.md)). |
| [EndTimer](../WDLang1/3015007.md) | Finaliza la ejecución de una timer lanzada por [timer](../WDLang1/3015008.md). |
| [EndTimerSys](../WDLang1/3015009.md) | Termina la ejecución de una timer activada por [timer Sys](../WDLang1/3015006.md). |
| [Event](../WDLang1/3015011.md) | Intercepta una Event de Windows en una control, un grupo de control s o una ventana WINDEV. |
| [EventExist](../WDLang1/1000024424.md) | Indica si se ejecuta un procedimiento específico cuando se envía un evento de Windows a un control o ventana. |
| [Handle](../WDLang1/3015005.md) | Devuelve el "Handle" del sistema (HWND) de un control o ventana de WINDEV. |
| [HandleParentNext](../WDLang1/1000017312.md) | Fuerza al padre de la siguiente ventana a abrirse. |
| [Multitask](../WDLang1/3015004.md) | MultiTask se utiliza para: <br><br>- define un timeout,<br><br>- devuelve control a Windows, <br><br>- devuelve control a Windows y a WLanguage.<br><br><br> |
| [PostMessage](../WDLang1/3015001.md) | Envía un mensaje Windows a un control o ventana. |
| [SendMessage](../WDLang1/3015003.md) | Envía un mensaje Windows a un control o ventana. |
| [Temporizador](../WDLang1/3015008.md) | Periódica y automáticamente llama a un WLanguage Procedure. |
| [TimerSys](../WDLang1/3015006.md) | Periódica y automáticamente llama a un WLanguage Procedure. |
| [Wait](../WDLang1/3015012.md) | Detiene temporalmente la ejecución del programa. |

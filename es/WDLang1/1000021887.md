
## FirstDayOfYear (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del primer día del año.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the customers created since the beginning of the year
HReadFirst(Customer, Name)
WHILE HOut() = False
	// Process the record
	IF (Customer.DateCreated >= FirstDayOfYear()) THEN
		Trace(Customer.Name)
	END
	HReadNext(Customer, Name)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el primer día del año a partir de una fecha

`<Result> = FirstDayOfYear([<Date>])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Primer día del año (en formato AAAAMMDD) para la fecha especificada.

**`<Date>: (Cadena de caracteres, variable de tipo Date o DateTime opcional)`**

Fecha para la que debe calcularse el primer día del año. Esta fecha puede corresponder a : 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  

- una cadena en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo (utilice CCC para las milésimas de segundo)). En este caso, solo se tiene en cuenta la fecha. 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.


Si no se especifica este parámetro, se utiliza la fecha actual de forma predeterminada.


<a name="SYNTAX2"></a>

### Obtener el primer día del año basado en un año

`<Result> = FirstDayOfYear(<Year>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Primer día del año (en formato AAAAMMDD).

**`<Year>: (Entero)`**

Año para el que debe calcularse el primer día.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función permite calcular rangos de fechas fácilmente.

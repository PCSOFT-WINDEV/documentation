


## &lt;Hilo&gt;.RequestStop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía una solicitud para detener un hilo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

<a name="Example2"></a>

```wl
// Build a thread
t is Thread(Proc, ("First parameter", 2))
// Trigger the execution of the thread
ThreadExecute(t)
...
t.RequestStop()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

### Solicitar que se detenga un hilo identificado por una variable de tipo Thread

`<Thread>.RequestStop()`
---

**`<Thread>: (Variable de tipo Thread)`**

Nombre de la variable de tipo [Thread](../WDLang1/1000024879.md) que corresponde al hilo que se detendrá.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El hilo no se detiene automáticamente. El código del hilo debe verificar si se solicitó la interrupción con la función [ThreadStopRequested](../WDLang1/1000021219.md), y finalizar correctamente.

- Es una función no bloqueante y no espera a que el hilo se detenga. Se debe utilizar la función [&lt;Variable Thread&gt;.Wait](../WDLang1/1000024880.md) para esperar a que el hilo se detenga.




## &lt;Tipo Ansi string&gt;.DeleteDuplicate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los duplicados entre las subcadenas de una cadena, basándose en uno o varios separadores.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Removes duplicates
s is string
s = "France, Italy, Germany, Spain, France"
s.DeleteDuplicate(", ") 
// Returns "France, Italy, Germany, Spain"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Initial string>.DeleteDuplicate(<Separator> [, <Comparison>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena sin duplicados.

**`<Initial string>: (Nombre del control)`**

Cadena con duplicados.

**`<Separator>: (Cadena o array de cadenas)`**

Este parámetro puede corresponder a:

- La cadena que separa las subcadenas. Este separador distingue entre mayúsculas y minúsculas.

- Un array de cadenas. Las diferentes cadenas del array delimitan las subcadenas. Estos separadores distinguen entre mayúsculas y minúsculas.


Si no se especifica este parámetro, el separador predeterminado es TAB.

**`<Comparison>: (Entero opcional)`**

Opciones de comparación: 



|   |   |
| --- | --- |
| *ccIgnoreAccent* | Comparación sin tener en cuenta los acentos |
| *ccIgnoreCase* | Comparación sin distinguir entre mayúsculas y minúsculas. |
| *ccIgnoreInsideSpace* | Comparación sin tener en cuenta los espacios dentro de las cadenas. |
| *ccIgnorePunctuationAndSpace* | Comparación sin tener en cuenta los espacios y la puntuación. |
| *ccIgnoreSpace* | Comparación sin tener en cuenta los espacios antes y después de las cadenas. |
| *ccNormal*<br>(Valor predeterminado) | Comparación estándar, equivale al operador '='. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- En caso de duplicados, solo se conserva en la cadena resultante la primera subcadena de la cadena original.

- Los caracteres que se tienen en cuenta para la puntuación y los espacios son proporcionados por el sistema. Para obtener la lista de estos caracteres, escriba el siguiente código WLanguage: 
	
	```wl
	s is string
	FOR i = 0 TO 255
		IF Charact(i) <> StringFormat(Charact(i), ccIgnorePunctuationAndSpace) THEN
			 s += Charact(i)
		END
	END
	Info(s)
	ToClipboard(s)
	```

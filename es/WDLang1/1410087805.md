
## WLanguage Procedure llamado por OpenAsynchronous



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
WLanguage Procedure ("callback") llamado por [OpenAsynchronous](../WDLang1/1410087804.md) después de cerrar la ventana. Este procedimiento puede ser local, global o interno.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open asynchronously without passing parameters 
OpenAsynchronous(WIN_DateSelect, (), CloseDateSelect)
INTERNAL PROCEDURE CloseDateSelect(SelectedDate)
	EDT_DATE = SelectedDate
END
```

```wl
// Open asynchronously without passing parameters and lambda
OpenAsynchronous(WIN_DateSelect, (), (SelectedDate) => {EDT_DATE = SelectedDate})
```

```wl
// Open asynchronously with 2 parameters
OpenAsynchronous(WIN_DateSelect, (DateSys(), "Select a date"), CloseDateSelect)
INTERNAL PROCEDURE CloseDateSelect(SelectedDate)
	EDT_DATE = SelectedDate
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OpenAsynchronous_Callback(<Return value>)`
---

**`<Return value>: (Tipo del valor de retorno)`**

Corresponde al valor que devolverá la ventana. Este valor puede ser devuelto: 

- a través de la sentencia [RETURN](../Motscles/1510007.md) en el código de cierre de la ventana,

- a través de [Cerrar](../WDLang1/3038018.md) (Close("", Value)),

- a través de la [ReturnedValue](../Proprietes/2510134.md) Property de la ventana.

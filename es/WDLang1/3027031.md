


## ChronoStart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. El cronómetro se detiene: 

- con la función [ChronoEnd](../WDLang1/3027032.md).

- con la función [ChronoPause](../WDLang1/1000018829.md). 


Si el cronómetro se detiene, se puede reiniciar con la función [ChronoResume](../WDLang1/1000018830.md). 

El valor del cronómetro se puede leer en cualquier momento: 

- con la función [ChronoValue](../WDLang1/3027033.md).

- con la propiedad **Value** de la variable de tipo [Chrono](../WDLang1/1000023560.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the stopwatch 
ChronoStart()
MyFunction()
// Stop the stopwatch 
DurationMyFunction is Duration = ChronoEnd()
Info("The MyFunction() process lasted " + DurationMyFunction)
```


<a name="Example2"></a>

```wl
// Start the first stopwatch 
ChronoStart(1)
// Start the second stopwatch 
ChronoStart(2)
// Stop the first stopwatch
DurationChronoStart is Duration = ChronoEnd(1)
MyFunction()
// Stop the second stopwatch
DurationMyFunction is Duration = ChronoEnd(2)
Info("MyFunction() process lasted" + DurationMyFunction, ...
	"The whole process lasted " + DurationChronoStart)
```

## Sintaxis

### Cronometraje con un número de cronómetro

`ChronoStart([<Stopwatch number>])`
---

**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro utilizado. 

Si no se especifica este parámetro, el valor predeterminado es 1. 

Si este número corresponde a un cronómetro en marcha, este último se reinicia.

### Cronometraje con una variable de tipo Chrono

`ChronoStart(<Stopwatch>)`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Error
<a name="error_ELTPARAGRAPHE000204"></a>

Si no se llama a la función **ChronoStart** antes de las funciones [ChronoValue](../WDLang1/3027033.md) y [ChronoEnd](../WDLang1/3027032.md), se produce un error WLanguage (error de tipo "No se ha iniciado el cronómetro").
<a name="NOTE0_2"></a>


### Hilos y tareas paralelas
<a name="hilos_tareas_paralelas_ELTPARAGRAPHE000221"></a>

Cada hilo (y cada tarea paralela) tiene sus propios cronómetros. 

Dos hilos distintos (o dos tareas paralelas distintas) pueden realizar cronometrajes separados utilizando el mismo número de cronómetro. 

El error WLanguage "No se ha iniciado el cronómetro" solo se mostrará si no se llama a la función **ChronoStart** en el mismo hilo (o en la misma tarea paralela) que las funciones [ChronoValue](../WDLang1/3027033.md) y [ChronoEnd](../WDLang1/3027032.md).

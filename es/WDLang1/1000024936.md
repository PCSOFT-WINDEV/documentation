


## &lt;Array&gt;.AllTrue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve <u><u><u>True</u></u></u> si todos los elementos de un array valen <u><u><u>True</u></u></u>.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrProduct is array of Product
// Return True if no product is out of stock
IF arrProduct.AllTrue(AProduct => AProduct.Stock > 0) THEN
	// Everything is OK
ELSE
	// Alert
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Source>.AllTrue([<Operation>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si todos los elementos del array especificado valen <u><u><u><u>True</u></u></u></u>,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Source>: (Variable de tipo Array)`**

Nombre de la variable de tipo Array a verificar.

**`<Operation>: (Procedimiento WLanguage)`**

Procedimiento WLanguage a ejecutar: 

- Si no se especifica este parámetro, el parámetro &lt;Source&gt; debe corresponder a un array de booleanos. 

- Si se especifica este parámetro, se llama al procedimiento WLanguage por cada elemento del array y debe devolver un booleano. 


Este parámetro puede corresponder a un procedimiento WLanguage local o global, a un procedimiento interno o a un procedimiento lambda.

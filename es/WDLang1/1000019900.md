
## RestartProgram( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la aplicación y la reinicia.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Restart the application
RestartProgram()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RestartProgram([<Code Execution>])`
---

**`<Code Execution>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> para ejecutar el código de finalización del proyecto.. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para no ejecutar el código de cierre del proyecto.

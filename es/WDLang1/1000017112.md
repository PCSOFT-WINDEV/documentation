
## ServiceRestart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reiniciar un servicio (ya iniciado). Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.. Se tienen en cuenta automáticamente las opciones especificadas cuando se inició el servicio anteriormente (el comando Line por ejemplo).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Restart the service in the TABLE_Service control
IF ServiceRestart(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceRestart(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se solicitó reiniciar el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a reiniciar.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para reiniciar un servicio en otro ordenador.. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el servicio se gestiona localmente..

### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000055"></a>

La función **ServiceRestart** envía una solicitud al administrador de servicios para reiniciar un servicio. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).

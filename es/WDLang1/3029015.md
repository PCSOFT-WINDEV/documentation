
## BitmapInfo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las características de un archivo de imagen o de un archivo PDF:

- formato, 

- ancho y alto, 

- número de bits por pixel.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los archivos PDF no están soportados.  







![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
ImageInfo, Format are strings
Height, Width are int
// Retrieve the characteristics of the image
ImageInfo = BitmapInfo("C:\Windows\TILES.BMP")
Format = ExtractString(ImageInfo, 1)
Width = Val(ExtractString(ImageInfo, 2))
Height = Val(ExtractString(ImageInfo, 3))
// Display the characteristics
Info("Format of the image: " + Format, "Width: " + Width, "Height: " + Height)
```

## Sintaxis

`<Result> = BitmapInfo(<Image file name>)`
---

**`<Result>: (Cadena de caracteres)`**



- Características de la imagen en el siguiente formato:
	
	```txt
	<Format> + TAB + <Width> + TAB + <Height> + TAB + <Bits/pixel>
	```

	


|   |   |   |
| --- | --- | --- |
| &lt;Format&gt; | Cadena de caracteres | <br><br>	- Tres caracteres permiten identificar el formato de la imagen: "BMP", "GIF", "PDF", etc.<br><br>	- "BAD" si el nombre de la imagen es un archivo que no existe o si el formato es desconocido<br><br><br> |
| &lt;Width&gt; | Entero | Ancho de la imagen en pixeles.<br>Observación: La altura y el ancho pueden invertirse en función de la posición del punto (0,0) de la imagen. Para obtener la altura y el ancho teniendo en cuenta la orientación de la imagen, utilice una variable de tipo [Image](../WDLang1/1000019650.md). |
| &lt;Height&gt; | Entero | Altura de la imagen en pixeles.<br>Observación: La altura y el ancho pueden invertirse en función de la posición del punto (0,0) de la imagen. Para obtener la altura y el ancho teniendo en cuenta la orientación de la imagen, utilice una variable de tipo [Image](../WDLang1/1000019650.md). |
| &lt;Bits/pixel&gt; | Cadena de caracteres | Número de bits por pixel:<br><br>	- 1 (imagen monocroma)<br><br>	- 4 (paleta de 16 colores)<br><br>	- 8 (paleta de 256 colores)<br><br>	- 24 (sin paleta, 16 millones de colores)<br><br>	- 32 (sin paleta, color verdadero)<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) El número de bits por pixel siempre es 0.<br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) El número de bits por pixel siempre es 32. |



- Si se produce un error, se devuelve la siguiente cadena: 
	```txt
	"BAD" + TAB + TAB + TAB
	```





**`<Image file name>: (Cadena de caracteres)`**



- Nombre del archivo de imagen o del archivo PDF con la ruta completa o relativa (hasta 260 caracteres). Se puede utilizar una ruta de tipo UNC.

- Nombre del control enlazado a un campo de tipo memo imagen.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los archivos PDF no están soportados.

## Observaciones




### Varios
<a name="varios_ELTPARAGRAPHE000152"></a>

- La función **BitmapInfo** puede utilizarse con archivos PDF. 
	Atención: [el framework GDI +](../Editeurs/2020007.md) debe estar instalado en el equipo.
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta funcionalidad no está disponible. 

- Para las imágenes en formato WMF o EMF, &lt;Bits/pixel&gt; es igual a 0.

- Los siguientes formatos de archivos de imagen son compatibles:

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Todos los formatos descritos en la función [fGraphicFilter](../WDLang1/3036009.md).

	- ![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) JPEG, PNG, GIF o BMP.

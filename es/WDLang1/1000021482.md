


## Hoja de cálculoCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el XLSX Document que fue abierto por [Hoja de cálculoCarga](../WDLang1/1000021340.md). 
<a name="sample_code"></a>

## Ejemplo


```wl
SpreadsheetLoad(PSHEET_Spreadsheet, "doc1.XLS")
SpreadsheetClose(PSHEET_Spreadsheet)
ShellExecute("doc1.xlsx")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetClose(<Spreadsheet control>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Por Default, [Hoja de cálculoCarga](../WDLang1/1000021340.md) deja el archivo XLSX abierto en modo de escritura. Cuando se guarda el Document, sólo se sobrescriben las celdas que han sido cambiadas en el control Hoja de cálculo.. 

- Para abrir el archivo ".xlxs" con otra aplicación, hay que cerrarlo con **Hoja de cálculoCerrar**.

- Si no se abrió ningún archivo ".xlsx", la función **Hoja de cálculoCerrar** no realiza ninguna acción.

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000199"></a>
Para ver los resultados de los datos de telemetría para sus aplicaciones, WINDEV y WINDEV Mobile proponen un editor de telemetría. 

**Para visualizar el editor de telemetría**: 

1. En la pestaña "Proyecto", en el grupo "Auditoría y rendimiento", haga clic en "Telemetría". 

2. Especifique los parámetros para la conexión con el servidor de telemetría: 

	- Servidor: servidor Address, nombre de usuario y contraseña. 
			Observación: El nombre de usuario y la contraseña se han especificado al instalar el servidor de telemetría. 

	- Plataforma CLOUD: nombre de la plataforma, nombre de usuario y contraseña.




3. Compruebe la conexión si es necesario. 

4. Valide.

## Editor de telemetría
<a name="editor_telemetria_ELTTEXTE000223"></a>


### Interfaz general
<a name="interfaz_general_ELTPARAGRAPHE000042"></a>

El editor de telemetría se utiliza para mostrar estadísticas sobre todas las aplicaciones que envían sus datos al mismo servidor de telemetría.. 

Se utilizan varias fichas para agrupar las estadísticas de cada aplicación. La mayoría de las estadísticas se pueden ver: 

- para cada aplicación (o para cada plataforma de aplicación si el proyecto contiene varias plataformas desplegadas)

- no importa si el proyecto está abierto en el editor o no (se muestra un mensaje si el proyecto debe abrirse para poder ver algunos datos). 




Se pueden ver las siguientes pestañas: 

- **Sumario**: resumen relativo a la utilización de la solicitud durante un período determinado: número de usuarios, elementos más utilizados, elementos menos utilizados, .....

- **Windows**: resumen sobre el uso de ventanas durante un período determinado: los elementos más utilizados, los elementos menos utilizados, los procesos más utilizados, los procesos más largos, .....

- **Reportes/consultas**: resumen sobre el uso de informes y consultas para un período determinado: los elementos más utilizados, los elementos menos utilizados, los procesos más utilizados, los procesos más largos, .....

- **Usuarios**: estadísticas sobre el número de usuarios de la aplicación. 

- **Dispositivo PC**: estadísticas sobre el dispositivo en el que se implementa la aplicación. Esta pestaña afecta a las aplicaciones WINDEV desplegadas en el PC.  

- **Dispositivo móvil**: estadísticas sobre el dispositivo móvil en el que se implementa la aplicación. Esta pestaña afecta a las aplicaciones WINDEV Mobile desplegadas en dispositivos móviles.  

- **Anomalías**: estadísticas sobre las anomalías que se han producido en el uso de la aplicación (los errores más comunes de WLanguage, ....). 

- pestañas correspondientes a la [telemetría personalizada](../WDLang1/1410086591.md).




**Observación**: Los elementos mostrados en estas diferentes pestañas pueden reorganizarse u ocultarse según sus necesidades.. Para ello: 

- Haga clic derecho para abrir el menú contextual. 

- Seleccione "Modo de modificación".. 

- Usted tiene la capacidad de ocultar áreas, de moverlas, ....

- Para volver al modo "Normal", seleccione de nuevo "Modo de modificación" en el menú contextual. 





### Opciones disponibles si el proyecto correspondiente se abre en el editor de WINDEV o WINDEV Mobile
<a name="opciones_disponibles_proyecto_correspondiente_abre_editor_windev_windev_mobile_ELTPARAGRAPHE000077"></a>

**Cuando se abre el proyecto correspondiente a la telemetría en el editor de WINDEV o WINDEV Mobile**tiene la posibilidad de ver directamente la telemetría de un elemento de proyecto. Por ejemplo, el el menú contextual de las ventanas más utilizadas se utiliza para ver la telemetría de la ventana (opción "Ver la telemetría"). 

Cuando se muestra la telemetría de un elemento, usted tiene la capacidad de ejecutar un GO de esta ventana en modo telemetría: en la pestaña "Telemetría", en el grupo "General", haga clic en "GO". 

El "Telemetría GO" se utiliza para iniciar el elemento mostrando las estadísticas de uso en la ventana a través de áreas coloreadas.. Usted tiene la capacidad de ver: 

- los mandos más utilizados. Un control deslizante se utiliza para definir el nivel con respecto al número de usos. 

- los controles no utilizados,

- los errores de WLanguage,

- el tiempo medio de ejecución de los procesos.

## Opciones específicas del editor de telemetría
<a name="opciones_especificas_del_editor_telemetria_ELTTEXTE000253"></a>


### Conexión
<a name="conexion_ELTPARAGRAPHE000103"></a>

La opción "Conexión" (en la pestaña "Telemetría", en el grupo "General") permite cambiar la conexión actual. Esto le permite cambiar el servidor de telemetría visto. Todo lo que tiene que hacer es especificar las características del nuevo servidor de telemetría. 


### Parámetros
<a name="parametros_ELTPARAGRAPHE000116"></a>

La opción "Parámetros" (en la pestaña "Telemetría", en el grupo "General") permite definir los parámetros globales de todas las aplicaciones que utilizan el servidor actual. 

Este ajuste afecta: 

- **El almacenamiento de los datos agregados**: El servidor recibe los datos regularmente y los agrega cada dos horas para proponer estadísticas completas.. 
	Se realizan estadísticas diarias, semanales, mensuales y anuales.. Tiene la capacidad de especificar la duración del almacenamiento de los datos: 

	- para los datos diarios: Estos datos son almacenados durante 1 año por Default. 

	- para los datos semanales y mensuales. Estos datos son almacenados durante 3 años por Default. 




- **El servidor HFSQL utilizado para almacenar los datos**: Esta información corresponde al servidor HFSQL utilizado para almacenar la base de datos de telemetría.. Si se modifica este parámetro, la base de datos se modificará para todas las aplicaciones que utilizan la telemetría en este servidor.. 

- **La modificación de la contraseña de telemetría**. El nombre de usuario y la contraseña se han especificado al instalar el servidor de telemetría. Esta información puede ser modificada. 
	Atención: La modificación de esta información no tiene ningún efecto sobre las aplicaciones implementadas.. La modificación de esta información desencadena la modificación de esta información en las aplicaciones que utilizan este servidor de telemetría durante su próxima actualización.: el nuevo nombre de usuario y la nueva contraseña se solicitarán al crear el ejecutable. 





### Estadísticas de un elemento
<a name="estadisticas_elemento_ELTPARAGRAPHE000142"></a>

La opción "Estadísticas de un elemento" (en la pestaña "Telemetría", en el grupo "General") permite ver las estadísticas de un elemento del proyecto actual. Simplemente seleccione el elemento deseado.

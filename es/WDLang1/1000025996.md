


## HTMLEditorFormatSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el estilo de los elementos seleccionados en un editor HTML control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLEditorFormatSelection(HTMEDT_MyHTMLEditor, htmlItalic)
HTMLEditorFormatSelection(HTMEDT_MyHTMLEditor, htmlFontSize, "14px")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLEditorFormatSelection(<HTML Editor control> , <Attribute> [, <Value>])`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del control Editor HTML utilizado. 

**`<Attribute>: (Constante de tipo Integer)`**

Atributo de estilo a modificar:


|   |   |
| --- | --- |
| *htmlAlignment* | Alinea la selección horizontalmente. Si se utiliza esta constante, se puede utilizar el parámetro  para especificar la alineación deseada. Los valores posibles son: <br><br>- *haCenter*: La selección se centra. <br><br>- *haRight*: La selección está alineada a la derecha. <br><br>- *haLeft*: La selección está alineada a la izquierda. <br><br><br> |
| *htmlBackgroundColor* | Cambia el fondo color de la selección. Si se utiliza esta constante, se puede utilizar el parámetro  para especificar el fondo deseado color. El parámetro  puede corresponder a: <br><br>- un entero. En ese caso, se utilizará una color RGB (por ejemplo, la devuelta por [RGB](../WDLang1/3029012.md)). <br><br>- una cuerda. En ese caso, se utilizará una Color en formato HTML (por ejemplo, "00ff00" o "verde"). <br><br><br> |
| *htmlBold* | Hace la selección audaz. Si la selección ya está en negrita, el formato en negrita será eliminado. Si una parte de la selección es en negrita, toda la selección será set a negrita.   |
| *htmlColor* | Cambia la Color de la selección. Si se utiliza esta constante, se puede utilizar el parámetro  para especificar el color deseado . El parámetro  puede corresponder a: <br><br>- un entero. En ese caso, se utilizará una color RGB (por ejemplo, la devuelta por [RGB](../WDLang1/3029012.md)). <br><br>- una cuerda. En ese caso, se utilizará una Color en formato HTML (por ejemplo, "00ff00" o "verde"). <br><br><br> |
| *htmlCreateLink* | Crea un enlace a la URL especificada en el parámetro . |
| *htmlDeleteLink* | Elimina el enlace de la selección.   |
| *htmlFontName* | Cambia el nombre de la fuente utilizada para la selección. Si se utiliza esta constante, se puede utilizar el parámetro  para especificar la familia de fuentes a utilizar. Por ejemplo: "Arial, Helvetica, sans-serif". |
| *htmlFontSize* | Cambia el tamaño de la fuente utilizada en la selección. Si se utiliza esta constante, se puede utilizar el parámetro  para especificar el tamaño de la fuente a utilizar. Este tamaño se expresa en unidades HTML. Por ejemplo: "12px" o "2.5em".  |
| *htmlFontSizeDecrease* | Reduce el tamaño de la letra de la selección. |
| *htmlFontSizeIncrease* | Aumenta el tamaño de la letra de la selección. |
| *htmlItalic* | Italiza la selección. Si la selección ya está en cursiva, el formato en cursiva se eliminará. Si parte de la selección está en cursiva, toda la selección estará en cursiva.   |
| *htmlNameCssStyle* | Modifica el estilo CSS usado por la selección. Si se utiliza esta constante, el parámetro  puede utilizarse para especificar el nombre del estilo CSS que se va a utilizar. |
| *htmlStrikeThrough* | Tacha la selección. Si la selección ya está tachada, la Line será eliminada. Si una parte de la selección es tachada, la selección entera será tachada.  |
| *htmlSubscript* | Aplica el formato índice a la selección. Si el formato de índice ya se ha aplicado a la selección, se eliminará. Si el formato de índice se ha aplicado a una parte de la selección, entonces se aplicará a toda la selección |
| *htmlSuperscript* | Aplica el formato de superíndice a la selección. Si el formato del superíndice ya se ha aplicado a la selección, se eliminará. Si el formato del superíndice se ha aplicado a una parte de la selección, se aplicará entonces a toda la selección.   |
| *htmlUnderlined* | Subraya la selección. Si la selección ya está subrayada, el subrayado se eliminará. Si una parte de la selección está subrayada, la selección completa estará subrayada.   |



**`<Value>: (Variant opcional)`**

Valor a utilizar en función de la constante utilizada en el parámetro .


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Después de utilizar esta función, se ejecuta automáticamente la "Modificar selección" Event del mando (a menos que se utilice la función en este Event). 

- El cambio no es inmediato: el valor del mando se leerá de nuevo después de llamar a la Event "Modificar selección".




## PVTSave (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el resultado del cálculo realizado en un control Tabla dinámica (en un archivo o en la memoria).. Esta copia de seguridad puede ser recargada en un control Tabla dinámica por [PVTLoad](../WDLang1/1000020507.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFile is string
// Builds the file name
sFile = SysDir(srAppData) + ["\"] + "StatsByCountry.tcd"

// The file exists?
IF fFileExist(sFile) THEN
	// Yes, load it 
	PVTLoad(PVT_Statistics, sFile)
	IF YesNo("Do you want to recalculate everything?") THEN PVTCalculateAll(PVT_Statistics)
ELSE
	// The file does not exist, calculate it
	PVTCalculateAll(PVT_Statistics)
END
PVTSave(PVT_Statistics, sFile)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Guardar un control Tabla dinámica con su interfaz y una contraseña

`<Result> = PVTSave(<Pivot Table control> , <Backup media> [, <Backup mode> [, <Password>]])`
---

**`<Result>: (El tipo depende del medio de respaldo)`**



- Copia de seguridad en memoria: Buffer que contiene la copia de seguridad. A continuación, esta copia de seguridad puede guardarse en un archivo ([fSaveBuffer](../WDLang1/1000019412.md)), ...

- Copia de seguridad en un archivo: 

	- <u><u><u><u>True</u></u></u></u> si se realizó la copia de seguridad,

	- <u><u><u><u>False</u></u></u></u> en caso contrario.  







**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Backup media>: (Cadena de caracteres o constante)`**



- Copia de seguridad en memoria: *inMemory* constant: Guarda el contenido de la control Tabla dinámica en la memoria. &lt;El resultado contiene el resultado de la copia de seguridad. 

- Copia de seguridad en un archivo: Nombre y ruta completa del archivo donde se guardará el contenido de control Tabla dinámica.




**`<Backup mode>: (Constante opcional de tipo Integer)`**

Por Default, sólo se guardan los datos. También tiene la posibilidad de guardar la interfaz de control Tabla dinámica: 


|   |   |
| --- | --- |
| *pvtWithUI* | Guarda la configuración de control Tabla dinámica:<br><br>- tamaño de las columnas,<br><br>- tamaño de las filas,<br><br>- Cabeceras de fila colapsadas/expandidas o cabeceras de columna.<br><br><br><br><br>Observación: Antes de la versión 25, esta constante se llamaba *pvtWithGUI*. |

Si no se especifican &lt;Backup Mode&gt; y &lt;Password&gt;, sólo se guardarán los datos encontrados en control Tabla dinámica y la copia de seguridad no estará protegida por contraseña.. Para guardar sólo los datos y proteger la copia de seguridad con una contraseña, utilice la sintaxis 2. 

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña de seguridad. Si la copia de seguridad está protegida por una contraseña, ésta tendrá que ser especificada cuando la copia de seguridad sea cargada por [PVTLoad](../WDLang1/1000020507.md).

## Observaciones


### Guardar y cargar una control Tabla dinámica
<a name="guardar_cargar_una_control_tabla_dinamica_ELTPARAGRAPHE000142"></a>

Cuando se calcula el contenido de una control Tabla dinámica con [PVTCcalcularTodos](../WDLang1/1000020539.md), el resultado puede guardarse con **PVTSave**. A continuación, se puede volver a cargar con [PVTLoad](../WDLang1/1000020507.md): en este caso, no se recalcula el contenido de la control Tabla dinámica.




## ListMoveLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desplaza una fila o intercambia dos filas en un cuadro de lista o control Combo Box. 

**Observaciones**:            

- Para mover una fila en una control Tabla, utilice [TableMoveLine](../WDLang1/1000017247.md).

- Para desplazar una fila en una control Looper, utilice [LooperMoveLine](../WDLang2/1000017248.md). 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Swap the current row and row 5
ListMoveRow(LIST_List1, rowSelected , 5, amSwap)
// Move the current row to row 3
ListMoveRow(LIST_List1, rowSelected , 3, amMove)
// Swap the current row with the next row
ListMoveLine (LIST_List1, rowSelected, amNext, amSwap)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Moverse a una posición específica

`<Result> = ListMoveLine(<List Box control> , <Index of row to move> , <Destination row index> [, <Operation to perform>])`
---

**`<Result>: (Integro)`**

Nuevo índice de la fila que se trasladó.

**`<List Box control>: (Nombre del control)`**

Nombre de la Caja de Listas o control Combo Box que se va a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Index of row to move>: (Integro)`**

Posición de la fila para moverse. Esta posición debe estar comprendida entre 1 y el número de filas que se encuentren en la Caja de Listas o control Combo Box (devuelta por [ListCount](../WDLang1/3049014.md) o [Conde](../Proprietes/2510097.md), por ejemplo).

Para mover la fila seleccionada, utilice la constante *rowSelected*.

**`<Destination row index>: (Integro)`**

Nueva posición de fila. Esta posición debe estar comprendida entre 1 y el número de filas de la Caja de Listas o control Combo Box (devuelta por [ListCount](../WDLang1/3049014.md) o [Conde](../Proprietes/2510097.md), por ejemplo). Se produce un error fatal si este parámetro no es válido.

Para manipular la fila seleccionada, utilice la constante *rowSelected*.

**`<Operation to perform>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Cambio de posición que se realizará para la fila:


|   |   |
| --- | --- |
| *amDisplayDestination* | La fila modificada se muestra en la pantalla control List Box (o Combo Box).. Se realiza un desplazamiento de las filas de automatic si es necesario para mostrar el fila modificado. La fila seleccionada (banda de selección) no se modifica.<br>Esta constante puede combinarse con las constantes *amMove* y *amSwap*.<br><br><br> |
| *amMove*<br>(valor predeterminado) | La fila se traslada al índice de la fila de destino. |
| *amSwap* | Se intercambian las líneas de origen y de destino. |




<a name="SYNTAX2"></a>

### Moverse relativamente a la posición actual

`<Result> = ListMoveLine(<List Box control> , <Index of row to move> , <Relative move> [, <Operation to perform>])`
---

**`<Result>: (Integro)`**



- Nuevo índice de la fila que se trasladó,

- 0 si se produce un error.




**`<List Box control>: (Nombre del control)`**

Nombre de la control List Box que se va a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Index of row to move>: (Integro)`**

Posición de la fila para moverse. Esta posición debe estar incluida entre 1 y el número de filas de la control List Box (devuelta por [ListCount](../WDLang1/3049014.md) o [Conde](../Proprietes/2510097.md), por ejemplo).

Para manipular la fila seleccionada, utilice la constante *rowSelected*.

**`<Relative move>: (Constante de tipo Integer)`**

Se utiliza para mover o un intercambio a una posición relativa al índice de la fila a mover.


|   |   |
| --- | --- |
| *amFirst* | Mueve el fila a la primera posición. |
| *amLast* | Mueve el fila a la última posición. |
| *amNext* | Mueve el fila una posición hacia abajo. |
| *amNextPage* | Mueve la fila una página hacia abajo (una página corresponde al número de filas que se pueden mostrar simultáneamente en el control List Box)..<br><br><br> |
| *amPrevious* | Mueve la fila una posición hacia arriba |
| *amPreviousPage* | Mueve la fila una página hacia arriba (una página corresponde al número de filas que se pueden mostrar simultáneamente en el control List Box)..<br><br><br> |


Si este parámetro no es válido (*amPrevious* utilizado en el primer fila de control List Box por ejemplo), el fila se mueve "a la posición más cercana".

**`<Operation to perform>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Cambio de posición que se realizará para la fila:


|   |   |
| --- | --- |
| *amDisplayDestination* | La fila modificada se muestra en la pantalla control List Box. Se realiza un desplazamiento de las filas de automatic si es necesario para mostrar el fila modificado. La fila seleccionada (banda de selección) no se modifica.<br>Esta constante puede combinarse con las constantes *amMove* y *amSwap*.<br><br><br> |
| *amMove*<br>(valor predeterminado) | La fila se traslada al índice de la fila de destino. |
| *amSwap* | Se intercambian las líneas de origen y de destino. |

<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento en una control Combo Box
<a name="modo_funcionamiento_una_control_combo_box_ELTPARAGRAPHE000231"></a>

Cuando se utiliza la función **ListMoveLine** en un control Combo Box, el elemento seleccionado permanece igual. Se modifica el índice del elemento seleccionado en la lista desplegable (es decir, el valor devuelto por la control Combo Box).
<a name="NOTE0_2"></a>


### Multiselección control List Box
<a name="multiseleccion_control_list_box_ELTPARAGRAPHE000242"></a>

Si se utiliza la función **ListMoveLine** en un control List Box de selección múltiple:

- al desplazar filas (constante *amMove*): todas las filas seleccionadas se moverán.

- al intercambiar las filas (constante *amSwap*): sólo la primera fila será cambiada

- no se puede utilizar la constante *rowSelected*. De lo contrario, se produce un error fatal y no se mueve ningún fila.



<a name="NOTE0_3"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000257"></a>

**Atención**: La función **ListMoveLine** no devuelve ningún código de error. Para determinar si esta función ha generado un error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

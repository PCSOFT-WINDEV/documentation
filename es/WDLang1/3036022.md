
## fCurrentDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o modifica el directorio actual.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036022&name=fcurrentdir_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Find out the current directory
ResCurrentDir = fCurrentDir()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Encontrar el directorio actual en el disco especificado

`<Result> = fCurrentDir([<Disk> [, <Option>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Ruta del directorio actual en el disco especificado (hasta 260 caracteres). Esta ruta termina con un carácter "\\" en la raíz de un disco solamente ("A: _ASLASH_" por ejemplo).
	
	
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En Default (si no se especifica &lt;Disco&gt;), este directorio corresponde al directorio de instalación de la aplicación. No puede leer ni escribir en este directorio.

- **-1** si se produce un error. Para obtener más detalles sobre este error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Disk>: (Cadena de caracteres opcional)`**

Nombre del disco donde se buscará el directorio actual. Este nombre tiene el siguiente formato:

- &lt;Letra de unidad&gt;&gt;Características de la unidad**:**. Por ejemplo: "**A:**".

- Ruta UNC. Por ejemplo: "_ASLASH_\\&lt;NombreServidor&gt;\\&lt;NombreCompartido&gt;".




Si no se especifica este parámetro, el disco corresponde al disco actual.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

**`<Option>: (Constante opcional de tipo Integer)`**

Se usa para especificar el formato del resultado de la función. Por Default, **fCurrentDir** devuelve una cadena de caracteres en formato Ansi. 


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.


<a name="SYNTAX2"></a>

### Modificar el directorio actual

`<Result> = fCurrentDir(<Path of new current directory> [, <Option>])`
---

**`<Result>: (Cadena de caracteres)`**



- Ruta del directorio actual (hasta 260 caracteres). Esta ruta termina con un carácter "\\" en la raíz de un disco solamente ("A: _ASLASH_" por ejemplo). 

- **-1** si se produce un error. Para obtener más detalles sobre este error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Path of new current directory>: (Cadena de caracteres)`**

ruta completa (o relativa) del nuevo directorio actual (hasta 260 caracteres). Se puede utilizar una ruta UNC: debe tener el siguiente formato: "ASLASH_\\&lt;ServerName&gt;\\&lt;ShareName&gt;\\&lt;DirectoryName&gt;".
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Option>: (Constante opcional de tipo Integer)`**

Se usa para especificar el formato del resultado de la función. Por Default, **fCurrentDir** devuelve una cadena de caracteres en formato Ansi. 


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

## Observaciones




### Espacio
<a name="espacio_ELTPARAGRAPHE000144"></a>

Los espacios finales de &lt;Ruta del nuevo directorio actual&gt; son eliminados automáticamente por **fCurrentDir**:


```wl
fCurrentDir("C:\MyDirectory\    ")
```


equivale a:


```wl
fCurrentDir("C:\MyDirectory")
```

<a name="NOTE0_2"></a>


### Modificar el disco y el directorio actual
<a name="modificar_disco_directorio_actual_ELTPARAGRAPHE000159"></a>

Para cambiar el disco, le recomendamos que utilice:

1. [fCurrentDrive](../WDLang1/3036023.md) para modificar el disco actual,

2. La función **fCurrentDir** para modificar el directorio actual en el nuevo disco.




Si el disco solicitado por la función **fCurrentDir** no corresponde al disco actual, el cambio de disco se realiza automáticamente.
<a name="NOTE0_3"></a>




### Directorio actual durante las pruebas
<a name="directorio_actual_durante_las_pruebas_ELTPARAGRAPHE000178"></a>

Cuando se ejecutan pruebas desde el editor WINDEV (ventana o proyecto GO), el directorio actual corresponde a: 

- el directorio de generación de la [configuración del proyecto](../Editeurs/9000030.md) actual,

- el subdirectorio EXE de su proyecto.





<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### SOAP server
<a name="soap_server_ELTPARAGRAPHE000197"></a>

Cuando se utiliza **fCurrentDir** en un servidor SOAP, devuelve el directorio actual del servidor INTERNET utilizado. Este directorio no debe ser modificado.

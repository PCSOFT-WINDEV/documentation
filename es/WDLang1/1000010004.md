


## TableAddChild (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una fila a una control Tabla TreeView, al final de un nivel jerárquico determinado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1001000010004&name=tableaddchild_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Adds a root ("Desserts") to the TVT_Recipe TreeView Table control
TableAddChild(TVT_Recipe, Null, "Desserts")

// Adds a leaf ("Ice Creams") below the "Desserts" node
TableAddChild(TVT_Recipe, "Desserts", "Ice Creams")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Añadir una fila especificando el índice del elemento padre

`<Result> = TableAddChild(<TreeView Table control> , <Index of the parent element> [, <Element column 1> [... [, <Element column N>]]])`
---

**`<Result>: (Integro)`**



- Índice de la fila añadida,

- -1 si se produce un error.




**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Tabla TreeView a la que pertenece el Process actual.

**`<Index of the parent element>: (Integro)`**

Índice del fila padre en la jerarquía o *NULL* constante para insertar el fila en la raíz del control Tabla TreeView (en este caso, esta función es equivalente a [TableAddLine](../WDLang1/3074006.md)).

**`<Element column 1>: (Tipo de columna associated (opcional))`**

Elementos de la fila a añadir a la control Tabla TreeView especificada. Cada elemento corresponde a una columna control. El tipo del elemento agregado debe ser compatible con el tipo de la columna asociada..

Si no se especifica ningún parámetro, se añade un fila vacío al control Tabla TreeView.



**`<Element column N>: (Tipo de columna associated (opcional))`**

Elementos de la fila a añadir a la control Tabla TreeView especificada. Cada elemento corresponde a una columna control. El tipo del elemento agregado debe ser compatible con el tipo de la columna asociada..

Si no se especifica ningún parámetro, se añade un fila vacío al control Tabla TreeView.


<a name="SYNTAX2"></a>

### Añadir una fila especificando el elemento padre

`<Result> = TableAddChild(<TreeView Table control> , <Parent element> [, <Element column 1> [... [, <Element column N>]]])`
---

**`<Result>: (Integro)`**



- Índice de la fila añadida, 

- -1 si se produce un error.




**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

Si este parámetro corresponde a una cadena vacía (""), se utilizará la control Tabla TreeView a la que pertenece el Process actual.

**`<Parent element>: (Cadena de caracteres)`**

Contenido de la primera columna del elemento padre en la control Tabla TreeView. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


**`<Element column 1>: (Tipo de columna associated (opcional))`**

Elementos de la fila a añadir a la control Tabla TreeView especificada. Cada elemento corresponde a una columna. El tipo del elemento agregado debe ser compatible con el tipo de la columna asociada..
Si no se especifica ningún parámetro, se añade un fila vacío al control Tabla TreeView.



**`<Element column N>: (Tipo de columna associated (opcional))`**

Elementos de la fila a añadir a la control Tabla TreeView especificada. Cada elemento corresponde a una columna. El tipo del elemento agregado debe ser compatible con el tipo de la columna asociada..
Si no se especifica ningún parámetro, se añade un fila vacío al control Tabla TreeView.




## Funciones Hoja de cálculo
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular el control Hoja de cálculo: 



|   |   |
| --- | --- |
| [Hoja de cálculoAñadir Fórmula](../WDLang1/1000021373.md) | Añade una fórmula de cálculo a un control Hoja de cálculo que se encuentra en una ventana. |
| [Hoja de cálculoAñadir Hoja de trabajo](../WDLang1/1000021642.md) | Añade una hoja de trabajo a un control Hoja de cálculo que se encuentra en una ventana. |
| [Hoja de cálculoBordeSelección](../WDLang1/1000021480.md) | Modifica la Border de las celdas seleccionadas en una control Hoja de cálculo encontrada en una ventana. |
| [Hoja de cálculoBorrarColumna](../WDLang1/1000022406.md) | Borra una columna de un control Hoja de cálculo. |
| [Hoja de cálculoCerrar](../WDLang1/1000021482.md) | Cierra la XLSX Document que fue abierta por [SpreadsheetLoad](../WDLang1/1000021340.md). |
| [Hoja de cálculoCongelar](../WDLang1/1000022519.md) | Se utiliza para congelar la visualización de una fila o columna en un control Hoja de cálculo. |
| [Hoja de cálculoControlFormula](../WDLang1/1000021647.md) | Permite utilizar un control para crear una barra de fórmula personalizada para el control Hoja de cálculo. |
| [Hoja de cálculoEstiloSelección](../WDLang1/1000021338.md) | Devuelve o modifica el estilo de las celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana. |
| [Hoja de cálculoGetFormula](../WDLang1/1000021507.md) | Recupera la fórmula de cálculo de una celda encontrada en un control Hoja de cálculo de una ventana. |
| [Hoja de cálculoGetMerge](../WDLang1/1000021740.md) | Se utiliza para averiguar si una célula se fusiona con otras células y a get el nombre de células fusionadas. |
| [Hoja de cálculoInsertarColumna](../WDLang1/1000021518.md) | Inserta una columna vacía en un control Hoja de cálculo que se encuentra en una ventana. |
| [Hoja de cálculoInsertRow](../WDLang1/1000021456.md) | Inserta una fila vacía en una control Hoja de cálculo que se encuentra en una ventana. |
| [Hoja de cálculoObtenerSelección](../WDLang1/1000021499.md) | Recupera la lista de celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana. |
| [Hoja de cálculoPermitirProcedimiento](../WDLang1/1000021473.md) | Permite el uso de una función WLanguage/Procedure en una fórmula de una control Hoja de cálculo. |
| [Hoja de cálculoRecalcularTodo](../WDLang1/1000021457.md) | Recalcula todas las fórmulas de un control Hoja de cálculo encontrado en una ventana (incluyendo las fórmulas que utilizan funciones/procedimientos de WLanguage). |
| [Hoja de cálculoRestaurarPosición y Selección](../WDLang1/1000023334.md) | restaura: <br><br>- la posición de la barra de desplazamiento en un control Hoja de cálculo, <br><br>- la hoja de trabajo actual, <br><br>- los elementos seleccionados en la hoja de trabajo actual de control Hoja de cálculo. <br><br><br>Estos elementos han sido guardados por [Hoja de cálculoGuardarPosiciónYSelección](../WDLang1/1000023333.md). |
| [Hoja de cálculoSeleccionar Hoja de trabajo](../WDLang1/1000021649.md) | Selecciona una hoja de trabajo (identificada por su nombre) en una control Hoja de cálculo. |
| [Hoja de cálculoSelectMinus](../WDLang1/1000021494.md) | Desmarca una o más celdas de un control Hoja de cálculo encontrado en una ventana. |
| [Hoja de cálculoSelectPlus](../WDLang1/1000021493.md) | Selecciona una o más celdas de un control Hoja de cálculo encontrado en una ventana. |
| [Hoja de cálculoSortSelection](../WDLang1/1000023463.md) | Ordena los valores seleccionados en una control Hoja de cálculo. |
| [Posición de la hoja de cálculo](../WDLang1/1000021609.md) | Muestra la celda especificada en una control Hoja de cálculo que se encuentra en una ventana.: <br><br>- o el área de Visible de control Hoja de cálculo se mueve para mostrar la célula. <br><br>- o la celda se muestra en la esquina superior izquierda del área Visible de control Hoja de cálculo.<br><br><br> |
| [SpreadsheetAddData](../WDLang1/1000021365.md) | Añade un dato a un control Hoja de cálculo que se encuentra en una ventana. |
| [SpreadsheetAddImage](../WDLang1/1000023335.md) | Añade una Image a la hoja de trabajo actual de una control Hoja de cálculo. |
| [SpreadsheetAddName](../WDLang1/1000022523.md) | Especifica un nombre para identificar una celda que se encuentra en una control Hoja de cálculo. |
| [SpreadsheetDeleteAll](../WDLang1/1000021495.md) | Borra todos los datos de un control Hoja de cálculo encontrado en una ventana (celdas, fórmulas, tamaño y disposiciones de cabecera).. |
| [SpreadsheetDeleteImage](../WDLang1/1000023546.md) | Borra una Image de la hoja de trabajo actual en una control Hoja de cálculo. |
| [SpreadsheetDeleteName](../WDLang1/1000023447.md) | Borra uno de los nombres usados para identificar una celda encontrada en una control Hoja de cálculo. |
| [SpreadsheetDeleteRow](../WDLang1/1000022405.md) | Borra un fila de un control Hoja de cálculo. |
| [SpreadsheetDeleteWorksheet](../WDLang1/1000021643.md) | Elimina una hoja de trabajo de un control Hoja de cálculo encontrado en una ventana. |
| [SpreadsheetEndModification](../WDLang1/1000023461.md) | <br><br> |
| [SpreadsheetGetSortInfo](../WDLang1/1000023462.md) | Devuelve información diversa sobre la clasificación realizada en una control Hoja de cálculo. |
| [SpreadsheetInputFunction](../WDLang1/1000023412.md) | Inserta un nombre de función en la posición del cursor en la celda editable de un control Hoja de cálculo que se encuentra en una ventana. |
| [SpreadsheetInsertWorksheet](../WDLang1/1000021648.md) | Inserta una hoja de trabajo en un control Hoja de cálculo que se encuentra en una ventana. |
| [SpreadsheetListComment](../WDLang1/1000024881.md) | Crea una lista de todos los comentarios de la hoja actual de un control Hoja de cálculo. |
| [SpreadsheetListImage](../WDLang1/1000023547.md) | Enumera las imágenes que se encuentran en la hoja de trabajo actual de una control Hoja de cálculo. |
| [SpreadsheetListName](../WDLang1/1000023448.md) | Devuelve la lista de nombres que identifican una celda en una control Hoja de cálculo. |
| [SpreadsheetListWorksheet](../WDLang1/1000023450.md) | Devuelve la lista de hojas de trabajo que se encuentran en una control Hoja de cálculo. |
| [SpreadsheetLoad](../WDLang1/1000021340.md) | Carga un archivo en formato XLSX en un control Hoja de cálculo que se encuentra en una ventana. |
| [SpreadsheetMergeCell](../WDLang1/1000026181.md) | Fusiona células especificadas o seleccionadas en una control Hoja de cálculo. |
| [SpreadsheetModifyName](../WDLang1/1000023464.md) | Modifica el nombre utilizado para identificar una celda en una control Hoja de cálculo. |
| [SpreadsheetMoveWorksheet](../WDLang1/1410087257.md) | Mueve una hoja de trabajo dentro de una control Hoja de cálculo en una ventana. |
| [SpreadsheetReplace](../WDLang1/1000023436.md) | Encuentra y reemplaza un valor en un control Hoja de cálculo. |
| [SpreadsheetSave](../WDLang1/1000021341.md) | Guarda el contenido de una control Hoja de cálculo en un archivo en formato XLSX. |
| [SpreadsheetSavePositionAndSelection](../WDLang1/1000023333.md) | Ahorra: <br><br>- la posición de la barra de desplazamiento en un control Hoja de cálculo para la hoja de trabajo actual, <br><br>- la hoja de trabajo actual en el control Hoja de cálculo,<br><br>- los elementos seleccionados en el control Hoja de cálculo para la hoja de trabajo actual.<br><br><br> |
| [SpreadsheetSeek](../WDLang1/1000021610.md) | Busca un valor en las celdas de un control Hoja de cálculo encontrado en una ventana.. |
| [SpreadsheetSelectComment](../WDLang1/1000024883.md) | Permite: <br><br>- saber si se seleccionó un comentario en un control Hoja de cálculo, <br><br>- seleccionar o deseleccionar un comentario en un control Hoja de cálculo.<br><br><br> |
| [SpreadsheetStartModification](../WDLang1/1000023460.md) | <br><br> |
| [SpreadsheetUnmergeCell](../WDLang1/1410089203.md) | Anula la combinación de la celda especificada o seleccionada en un control Hoja de cálculo. |
| [Tipo de hoja de cálculo y selección de máscara](../WDLang1/1000021349.md) | Devuelve o modifica el tipo y la máscara de las celdas seleccionadas en una control Hoja de cálculo. |



    
También se pueden utilizar otras funciones para manipular las coordenadas de una celda: 



|   |   |
| --- | --- |
| [xlsCoordinatesToName](../WDLang5/1000021500.md) | Recupera el nombre de una celda, hoja de cálculo XLS o control Hoja de cálculo según sus coordenadas. |
| [xlsNameToCoordinates](../WDLang5/1000021501.md) | Devuelve las coordenadas de una celda, archivo XLS u hoja de cálculo control según su nombre. |

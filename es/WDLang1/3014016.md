
## IconInstall( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un icono (associated con un programa) en un grupo de programas de Windows, con el comando associated Line.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates a shortcut for a program
IconInstall("Examples", "WD Example", "D:\WINDEV\Examples\SALESMGT\SALESMGT.EXE")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IconInstall(<Group name> , <Caption> , <Command line>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se instaló el ícono,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Group name>: (Cadena de caracteres)`**

Nombre del grupo en el que debe crearse la Icon. No se puede crear ningún subgrupo. Si el grupo especificado no existe, se crea.

**`<Caption>: (Cadena de caracteres)`**

Pie de foto mostrado debajo de la Icon o en el menú "Inicio".. Si la leyenda contiene espacios en blanco, debe estar entre comillas dobles.

**`<Command line>: (Cadena de caracteres)`**

Línea de mando associated con la Icon. Contiene la ruta completa de la aplicación associated con la Icon y los parámetros de inicio (si es necesario).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para una gestión más avanzada de los iconos del programa, utilice [CreateShortcut](../WDLang1/3014008.md).

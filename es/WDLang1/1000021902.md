
## AttachPointAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define un punto de unión para una de las dos extremidades de una control de alambre.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
StartAttachPointID is int
EndAttachPointID is int

WIRE_Link_wire is Control

WIRE_Link_wire <- ControlCreate("Link", typWire)


WIRE_Link_wire.StartControl = IMG_Start
WIRE_Link_wire.EndControl = IMG_End

StartAttachPointID = AttachPointAdd(IMG_Start, 100, 25))
EndAttachPointID = AttachPointAdd(IMG_End, 0, 75))

WIRE_Link_wire.AttachPointStart = StartAttachPointID
WIRE_Link_wire.AttachPointEnd = EndAttachPointID
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Definir un punto de unión a través de sus coordenadas

`<Result> = AttachPointAdd(<Control used> , <X> , <Y>)`
---

**`<Result>: (Integro)`**

Identificador del punto de unión. Este identificador puede utilizarse con las propiedades [AttachPointStart](../Proprietes/1000021907.md) y [AttachPointEnd](../Proprietes/1000021908.md).

**`<Control used>: (Nombre del control)`**

Nombre del comienzo o del final control del alambre control. 

**`<X>: (Integro)`**

Posición horizontal del punto de unión en % (ver las Notas). El valor de este parámetro debe estar comprendido entre 1 y 100. 

**`<Y>: (Integro)`**

Posición vertical del punto de unión en % (ver las Notas). El valor de este parámetro debe estar comprendido entre 1 y 100.


<a name="SYNTAX2"></a>

### Definir un punto de unión a través de un punto Variable

`<Result> = AttachPointAdd(<Control used> , <Point>)`
---

**`<Result>: (Integro)`**

Identificador del punto de unión. Este identificador puede utilizarse con las propiedades [AttachPointStart](../Proprietes/1000021907.md) y [AttachPointEnd](../Proprietes/1000021908.md).

**`<Control used>: (Nombre del control)`**

Nombre del comienzo o del final control del alambre control. 

**`<Point>: (Variable de tipo Point)`**

Nombre del punto Variable correspondiente al punto de unión. Las coordenadas del punto se expresan en % (ver las Notas).

## Observaciones


### Definir un punto de unión
<a name="definir_punto_union_ELTPARAGRAPHE000060"></a>

Para definir un punto de enganche, debes saber: 

- El control relacionado al punto de unión. Este control corresponde al punto de inicio o final de Wire control.

- La posición del punto de enganche en uno de los lados de control.




El punto de unión puede corresponder al punto de inicio o final de la control de alambre.
<a name="NOTE0_2"></a>




### Posición del punto de unión
<a name="posicion_del_punto_union_ELTPARAGRAPHE000073"></a>

La posición del punto de enganche está definida por 2 valores que representan en porcentaje la posición X e Y (horizontal y vertical) en relación con la control.

Por ejemplo: 

- si &lt;X&gt; es set a 0, el punto de unión comienza desde el lado izquierdo del control. 

- si &lt;X&gt; es set a 100, el punto de unión comienza desde el lado derecho del control.

- si &lt;Y&gt; es set a 0, el punto de unión comienza en la parte superior del control.

- si &lt;Y&gt; es set a 100, el punto de unión comienza desde el lado inferior del control.




Otro ejemplo: Para iniciar el alambre control desde el punto de unión que se encuentra en la parte inferior y en el centro del control: 

- &lt;X&gt; debe ser igual a 50, 

- &lt;Y&gt; debe ser igual a 100.


## Spain.CheckDNI (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número de DNI (número de documento nacional de identidad en España).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyDNINumber is string = "12345678Z"
Spain.CheckDNI(MyDNINumber)	// Returns True

// Check a control
EDT_DNINumber = "ABC"
Spain.CheckDNI(EDT_DNINumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Spain.CheckDNI(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número de DNI válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número de DNI a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

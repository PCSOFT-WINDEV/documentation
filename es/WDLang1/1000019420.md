
## RGBToHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el color HTML que corresponde al color RGB. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RGBToHTML(255) // Returns "#0000FF"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RGBToHTML(<RGB color>)`
---

**`<Result>: (Cadena de caracteres)`**

Color HTML expresado en un valor hexadecimal.



**`<RGB color>: (Entero)`**

Color RGB a convertir en color HTML. Este color puede corresponder a: 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

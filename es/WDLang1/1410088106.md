


## <Camera>.VideoStart (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la grabación de video a través de un control Cámara en una aplicación Android o iOS. Las diferentes opciones especificadas para el control Cámara (flash, cámara trasera o delantera, etc.) se tienen en cuenta.  

Para detener la grabación de video, utilice la función [.VideoStop](../WDLang1/1410088061.md). El video se guarda en el directorio predeterminado de la aplicación, o en el directorio especificado con la propiedad [WorkingDirectory](../Proprietes/1000026169.md).

Observación: Esta función solo puede utilizarse con un control Cámara de la versión 27 o posterior (la opción "Modo compatible con la versión 26" no debe estar marcada en la pestaña "General" de la ventana de descripción del control).<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// BTN_Start_video click code
ChronoStart()

TimerSys(TimerSys_Callback,1s)
INTERNAL PROCEDURE TimerSys_Callback()
	dDuration is Duration = ChronoValue()
	STC_CPT_VIDEO = dDuration.ToString("HH:MM:SS")
END

CAM_Camera.VideoStart(CameraVideoStart_Callback)


INTERNAL PROCEDURE CameraVideoStart_Callback(bSuccess is boolean, VideoPath is string)
	ChronoEnd()
	EndTimerSys()
	STC_CPT_VIDEO = "00:00:00"
	IF bSuccess THEN
		STC_VIDEO_PATH = VideoPath
		MyWindow.Plane = 4
	ELSE
		ToastDisplay("Cannot record video." + CR + ErrorInfo())
	END
END
```

```wl
// Stop video 
CAM_Camera.VideoStop()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Camera control>.VideoStart(<WLanguage procedure>)`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara que graba el video.

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del procedimiento WLanguage ("callback") al que se llama una vez que se ha detenido la grabación de video (con la función [.VideoStop](../WDLang1/1410088061.md)).
Para obtener más información sobre este procedimiento, consulte [Parámetros del procedimiento utilizado por la función <Camera>.VideoStart](../WDLang1/1410087850.md).

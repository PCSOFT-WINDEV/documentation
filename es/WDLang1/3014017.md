
## HiWord( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve los dos bytes altos de un entero.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is int
Res = 0x000A001F
Trace(HiWord(Res)) // Display 10 (corresponds to 0x000A)
Trace(LoWord(Res)) // Display 31 (corresponds to 0x001F)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HiWord(<Value> [, <Size>])`
---

**`<Result>: (Integro)`**

Los dos bytes altos.

**`<Value>: (Integro)`**

Valor cuyos bytes altos se buscan. 
Si este valor es un número distinto a un número entero, este valor se convierte automáticamente en un número entero.. 
Se producirá un error si este valor es un string.

**`<Size>: (Entero opcional)`**

Tamaño del entero: 

- 1 para un número entero en 1 byte,

- 2 para un número entero de 2 bytes,

- 4 para un número entero en 4 bytes.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El operador [&](../Motscles/1512002.md) se utiliza para conocer la dirección de un Variable.

- La función **HiWord** puede utilizarse para obtener el bit más significativo de un número entero devuelto por la función [API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md).

- [LoWord](../WDLang1/3014018.md) se utiliza para recuperar los bytes bajos de un entero.

- Para construir un entero de 4 bytes a partir de un entero de 2 bytes, utilice [MakeInteger](../WDLang1/3062016.md).

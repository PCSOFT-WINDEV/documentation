


## DocAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega: 

- al final de un procesador de textos Document: un elemento, una Document, un párrafo, un texto, una Image o un fragmento.

- al final de un fragmento: un elemento, un texto o un Image.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds an image
// Select the image file 
sFile is string  
sFile = fImageSelect(fCurrentDir(), "", "Add an image",
	"JPEG (*.JPG)" + TAB + "*.jpg" + CR + "PNG (*.PNG)" + TAB + 
	"*.png" + CR + "All files (*.*)" + TAB + "*.*")
	
IF sFile = "" THEN
	RETURN
END

// The image file is transferred into an Image variable 
img is Image = sFile

// Add the image at the end of document 
DocAdd(WP_NoName1, img)
```
<a name="Example2"></a>

```wl
// Add a page break
DocAdd(WP_NoName, Charact(12))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadiendo al final de un procesador de textos Document

`<Result> = DocAdd(<Document> , <Object to add>)`
---

**`<Result>: (Variable de tipo DocFragment)`**

[docFragment](../WDLang1/1000022482.md) variable con el fragmento añadido. 

**`<Document>: (Nombre de control o documento Variable)`**

Document para usar. Este Document corresponde a : 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de un control Procesador de texto. 

- una variable de tipo [Document](../WDLang1/1000022461.md).




**`<Object to add>: (Tipo de objeto añadido)`**

Objeto a añadir: 

- aspecto: Nombre de la [docElement](../WDLang1/1000022484.md) Variable que corresponde al elemento a añadir.

- Document: Nombre de la variable [Document](../WDLang1/1000022461.md) que corresponde al Document a añadir.

- Párrafo: Nombre de la [docParagraph](../WDLang1/1000022483.md) Variable que corresponde al párrafo a añadir. Este párrafo debe haber sido recuperado de un Document, no puede ser creado de la nada porque el array de los elementos del párrafo es de sólo lectura.

- Texto en formato de cadena de caracteres: Texto a añadir.
	Puede utilizar las siguientes constantes en el texto para añadir para gestionar las diferentes pausas posibles: 
	


|   |   |
| --- | --- |
| *docColumnBreak* | Añade un salto de columna en una sección multicolumna. Si la sección no es multicolumna, se añade un salto de página. |
| *docLineBreak* | Añade una ruptura de Line. |
| *docPageBreak* | Añade un salto de página. |
| *docParagraphBreak* | Añade una ruptura de párrafo. |



- Image: Nombre de [Image](../WDLang1/1000019650.md) Variable a añadir.

- Fragment: Nombre de [docFragment](../WDLang1/1000022482.md) Variable a añadir.





<a name="SYNTAX2"></a>

### Añadiendo al final de un fragmento

`<Result> = DocAdd(<Fragment> , <Object to add>)`
---

**`<Result>: (Variable de tipo DocFragment)`**

[docFragment](../WDLang1/1000022482.md) variable con el fragmento añadido. 

**`<Fragment>: (Variable de tipo DocFragment)`**

Nombre de la variable de tipo [docFragment](../WDLang1/1000022482.md) que se utilizará.

**`<Object to add>: (Tipo de objeto añadido)`**

Objeto a añadir: 

- aspecto: Nombre de la [docElement](../WDLang1/1000022484.md) Variable que corresponde al elemento a añadir.

- Texto en formato de cadena de caracteres: Texto a añadir.
	Puede utilizar las siguientes constantes en el texto para añadir para gestionar las diferentes pausas posibles: 
	


|   |   |
| --- | --- |
| *docColumnBreak* | Añade un salto de columna en una sección multicolumna. Si la sección no es multicolumna, se añade un salto de página. |
| *docLineBreak* | Añade una ruptura de Line. |
| *docPageBreak* | Añade un salto de página. |
| *docParagraphBreak* | Añade una ruptura de párrafo. |



- Image: Nombre de [Image](../WDLang1/1000019650.md) Variable a añadir.

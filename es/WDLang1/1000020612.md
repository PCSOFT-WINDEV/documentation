
## EventTimeline (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **EventTimeline** permite definir todas las características avanzadas de un evento. Este Event será disipado dentro de un control Línea de tiempo. Las características de esta Event pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

El Event correspondiente puede entonces ser usado con las funciones de WLanguage para la gestión del control de la línea de tiempo.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TimelineDeleteAll(TL_Robots)
Evt is EventTimeline

TimelineAddTrack(TL_Robots, "Robot 1")
Evt.Track = "Robot 1"
Evt.Title = "Start"
Evt.Start = 10
Evt.End = 150
Evt.BackgroundColor = LightGreen
TimelineAddEvent(TL_Robots, Evt)

Evt.Track = "Robot 1"	
Evt.Title = "Special process"
Evt.Start = 150
Evt.End = 450
Evt.BackgroundColor = PastelBlue
TimelineAddEvent(TL_Robots, Evt)

Evt.Track = "Robot 1"	
Evt.Title = "Pause"
Evt.Start = 450
Evt.End = 700
Evt.BackgroundColor = LightRed
TimelineAddEvent(TL_Robots, Evt)

Evt.Track = "Robot 1"	
Evt.Title = "Stop"
Evt.Start = 700
Evt.End = 750
Evt.BackgroundColor = PastelRed
TimelineAddEvent(TL_Robots, Evt)
```

### Propiedades específicas de las variables TimelineEvent
<a name="propiedades_especificas_las_variables_timelineevent_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden ser utilizadas para manipular y Event:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| Autor | Cadena de caracteres | Nombre del autor de Event. Si no se especifica esta propiedad, la Event no tiene autor. |
| BackgroundColor | Integro | El color de fondo utilizado para mostrar el Event en el control Línea de tiempo.<br>Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)), <br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)), <br><br>- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).<br><br><br>Si no se especifica esta Property, se calculará automáticamente una color. Los eventos de la misma categoría tendrán la misma color.<br><br> |
| Categoría | Cadena de caracteres | Categoría de la Event. Si no se especifica esta Property, la Event es associated sin categoría. |
| circuito | Cadena de caracteres | Pista asociada a la Event. |
| Contenido | Cadena de caracteres | Detalle de la description de la Event. El contenido se muestra en la página web de control Línea de tiempo. <br><br>Si no se especifica esta propiedad, el evento no tiene description. |
| estreno | Integro | Inicio de Event. Este Property debe usar la misma unidad que la usada para mostrar el control Línea de tiempo (segundo, milisegundo o microsegundo).<br>Esta Property debe ser necesariamente especificada. |
| ID | Cadena de caracteres | Identificador asociado con el Event. Permite almacenar el identificador de un elemento que debe ser associated con el elemento para usarlo más tarde en la programación. <br><br>Por ejemplo, este identificador puede corresponder al identificador de un Record en una base de datos. |
| Image | Cadena de caracteres | La imagen asociada con la Event. Esta Image se muestra en la control Línea de tiempo (además del logo que representa un elemento importante si es necesario). <br><br>Esta propiedad puede corresponder a: <br><br>- el ruta a un Image accesible desde el ordenador actual,<br><br>- un archivo de Image que se encuentra en la biblioteca de aplicaciones,<br><br>- un control Imagen que contiene un Image,<br><br>- un dibujo realizado en una control Imagen con las funciones de dibujo y guardado en la memoria.<br><br><br>Si no se especifica esta propiedad, el Event no tiene associated Image.<br><br>**Observación**: Se debe utilizar una ventana interna para mostrar una imagen en la Event. El estilo simple (propuesto por Default) no puede ser usado para mostrar un Image. |
| Importancia | Integro | Importancia de la Event. <br><br>Si este Property es igual o mayor que 1, se mostrará un icono (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=agRendezVous_IconeImportant.gif)) en el control Línea de tiempo para indicar un Event importante. <br><br>Si no se especifica este Property, su valor es set a 0. |
| meta | Integro | Fin de Event.  Este Property debe usar la misma unidad que la usada para mostrar el control Línea de tiempo (segundo, milisegundo o microsegundo).<br>Esta Property debe ser necesariamente especificada. |
| Nota | Cadena de caracteres | Nota asociada a la Event. <br><br>Si no se especifica esta Property, no se asocia ninguna nota a la Event. |
| Repetition | Variable de tipo [Repetition](../WDLang1/1000019262.md) | Parámetros avanzados de Repetition. Esta Property se tiene en cuenta sólo si la **con la repetición** Property es set a <u><u><u><u>True</u></u></u></u>. |
| Título | Cadena de caracteres | Título de la Event. El título se muestra en la línea de tiempo control. <br><br>Si no se especifica esta propiedad, la Event no tiene título. |
| Tooltip | Cadena de caracteres | Tooltip que se muestra al deslizar el cursor sobre el evento. Por defecto, este valor incluye las horas de inicio y fin de la Event así como el título de Event. |
| WithRepetition | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si se debe repetir el evento,<br><br>- <u><u><u><u>False</u></u></u></u> si el evento tiene lugar una sola vez. <br><br><br>Si el evento se repite, la propiedad **Repetition** permite especificar la frecuencia del Repetition. <br><br>Por defecto, el valor de esta propiedad es <u><u><u><u>False</u></u></u></u>. |

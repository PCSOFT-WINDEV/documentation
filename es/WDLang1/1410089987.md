


## <Diagram Variable>Guardar (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una Diagram como archivo "diag.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDiagram is Diagram
MyDiagram.Save("diagram1.wddiag")
IF ErrorOccurred THEN
	Error("File not saved: " + ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Diagram>.Save(<File>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<Diagram>: (Diagram Variable)`**

Nombre de la variable de tipo [Diagram](../WDLang1/1410088055.md) que se utilizará.

**`<File>: (Cadena de caracteres)`**

ruta completa del fichero "diag".




## SpreadsheetDeleteRow (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un fila de un control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the rows 2 and 3
SpreadsheetDeleteRow(PSHEET_MySpreadsheet, 2)
SpreadsheetDeleteRow(PSHEET_MySpreadsheet, 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetDeleteRow(<Spreadsheet control> , <Row> [, <Worksheet>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.  

**`<Row>: (Integro)`**

Número de la fila a borrar, por ejemplo: 1. 
Los números de fila comienzan desde 1. 
La función no tiene ningún efecto si este parámetro no es válido.. 

**`<Worksheet>: (Entero opcional)`**

Número de la hoja de trabajo a utilizar. Si no se especifica este parámetro, se utiliza la hoja de trabajo actual. 
La función no tiene ningún efecto si este parámetro no es válido.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Después de la ejecución de la función: 

- las filas que se encuentran debajo de la fila borrada se desplazan a la parte superior. 

- el contenido de las expresiones cambia de nombre.

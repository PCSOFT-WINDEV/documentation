
## ArcTan2( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el arco tangente 2 de los valores pasados en parámetro.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ArcTan2(1, 1) 	// Returns 45
ArcTan2(1, -1) 	// Returns 135
ArcTan2(-1, 1) 	// Returns -45
ArcTan2(-1, -1) 	// Returns -135
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ArcTan2(<Y> , <X>)`
---

**`<Result>: (Real)`**



- Ángulo (en grados) entre: 

	- la parte positiva del eje X de un plano,

	- y el punto de este plano en coordenadas (X,Y). 
			Este ángulo es positivo para los ángulos en sentido contrario a las agujas del reloj, también llamado sentido trigonométrico (mitad superior del plano, Y &gt; 0) y negativo en el otro (mitad inferior del plano, Y &lt; 0).
			Este ángulo está incluido en el [-180, 180] interval.




- 0 si los parámetros &lt;X&gt; y &lt;Y&gt; son set a 0. 




**`<Y>: (Real)`**

Valor Y. 

**`<X>: (Real)`**

Valor de X.

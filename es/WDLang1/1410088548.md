
## diagWaypoint (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagWaypoint** se utiliza para definir todas las características avanzadas de un waypoint conector.. Las características de este waypoint pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Shape is diagShape
dConnector is diagConnector
dWaypoint is diagWaypoint
dWaypoint.X = 100
dWaypoint.Y = 50
dConnector.Waypoint.Add(dWaypoint)
```

<a name="PROPS"></a>

## Properties


### Propiedades específicas de las variables diagWaypoint 
<a name="propiedades_especificas_las_variables_diagwaypoint_ELTPARAGRAPHE000025"></a>

Las siguientes propiedades se pueden utilizar para manipular un waypoint:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| vincular | [diagConnector](../WDLang1/1410088454.md) | Conector utilizado. |
| X | Real | Coordenada X del waypoint en la Diagram. Esta posición se expresa en píxeles. |
| Y | Real | Coordenada Y del waypoint en la Diagram. Esta posición se expresa en píxeles. |

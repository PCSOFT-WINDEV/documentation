
## AppDeleteVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una versión del historial de versiones encontradas en el servidor reference instalación.

**Atención**: Esta función es irreversible: los archivos instalación de la versión correspondiente se borrarán del servidor instalación de referencia. 

**Observación**: Esta función sólo puede ser utilizada con una red instalación en una red local. No puede ser usado con una instalación por IP.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates the list of available versions
ListVersions is string 
ListVersions = AppListVersion()
// Extract the first version
FirstVersion is string
FirstVersion = ExtractString(ListVersions, 1, CR)
// Delete version
IF AppDeleteVersion(FirstVersion) = False THEN
	Error("Error deleting the version " +  ...
			FirstVersion + CR + ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppDeleteVersion(<Version> [, <Executable>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la eliminación se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**Observaciones**: Las posibles causas de error son las siguientes: 

- el historial de versiones no está habilitado. 

- la versión solicitada no existe.

- la versión borrada corresponde a la versión actualmente utilizada. En este caso, la versión de Active puede ser modificada por [AppActivateVersion](../WDLang1/1000018846.md). 




**`<Version>: (Cadena de caracteres)`**

Número de la versión a borrar. 

**`<Executable>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable de la aplicación para la que se debe borrar la versión. Si no se especifica este parámetro, se gestiona la aplicación actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El usuario que tiene los derechos para acceder al servidor de instalación de referencia en modo de escritura debe utilizar la función **AppDeleteVersion**.

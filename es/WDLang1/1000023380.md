


## PicCreate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una nueva imagen en un editor de imágenes control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create the image in the Image Editor control
IF PicCreate(IE_MyEditor, picImage, 48, 24) = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PicCreate(<Image Editor control> [, <Creation mode> [, <Width> [, <Height>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Image Editor control>: (Nombre del control)`**

Nombre del editor de imágenes control a utilizar. 

**`<Creation mode>: (Constante opcional de tipo Integer)`**

Modo de creación de Image: 


|   |   |
| --- | --- |
| *picIcon* | Se crea un nuevo Icon. En este caso, no se deben especificar los parámetros &lt;Ancho&gt; y &lt;Altura&gt;.. De lo contrario, se muestra un error fatal. |
| *picImage*<br>(Valor predeterminado) | Se crea un nuevo Image. En este caso, se deben especificar los parámetros &lt;Ancho&gt; y &lt;Altura&gt;. |



**`<Width>: (Entero opcional)`**

Ancho de la Image a crear, expresado en píxeles. Este ancho es set a 800 pixeles por Default. 
Atención: Este parámetro debe ser especificado cuando se utiliza la constante *picImage*. Se produce un error fatal si este parámetro se especifica con la constante *picIcon*. 

**`<Height>: (Entero opcional)`**

Altura de Image a crear, expresada en píxeles. Esta altura es set a 600 pixeles por Default. 
Atención: Este parámetro debe ser especificado cuando se utiliza la constante *picImage*. Se produce un error fatal si este parámetro se especifica con la constante *picIcon*.




## Funciones del control Kanban
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
 
Las siguientes funciones permiten manipular controles Kanban:



|   |   |
| --- | --- |
| [KanbanAddCard](../WDLang1/1410089140.md) | Agrega una tarjeta a una lista en un control Kanban. |
| [KanbanAddList](../WDLang1/1410089137.md) | Agrega una nueva lista a un control Kanban. |
| [KanbanCount](../WDLang1/1410089240.md) | Devuelve el número de tarjetas o listas en un control Kanban. |
| [KanbanDeleteAllCard](../WDLang1/1410089143.md) | Elimina todas las tarjetas de un control Kanban o de un control Lista Kanban. |
| [KanbanDeleteCard](../WDLang1/1410089141.md) | Elimina una tarjeta de un control Kanban o de un control Lista Kanban. |
| [KanbanDeleteList](../WDLang1/1410089239.md) | Elimina una lista de un control Kanban. |
| [KanbanDisplay](../WDLang1/1410089522.md) | Actualiza el control Kanban o una tarjeta del control Kanban. |
| [KanbanFilter](../WDLang1/1410089534.md) | Muestra solo las tarjetas que coinciden con el filtro especificado. |
| [KanbanInfoXY](../WDLang1/1410089224.md) | Devuelve una de las siguientes opciones para una posición determinada (coordenadas de un punto) en un control Kanban:<br><br>- el número de la tarjeta que se encuentra en la posición especificada, <br><br>- el número de la lista que se encuentra en la posición especificada<br><br><br> |
| [KanbanInsertCard](../WDLang1/1410089316.md) | Inserta una tarjeta en una lista Kanban. |
| [KanbanInsertList](../WDLang1/1410089539.md) | Inserta una lista en un control Kanban. |
| [KanbanListIndex](../WDLang1/1410089317.md) | Devuelve el índice de una lista de un control Kanban. |
| [KanbanMoveCard](../WDLang1/1410089144.md) | Mueve una tarjeta en un control Kanban. |
| [KanbanMoveList](../WDLang1/1410089540.md) | Mueve una lista en un control Kanban. |
| [KanbanPositionList](../WDLang1/1410089241.md) | Permite desplazarse por la lista de un control Kanban para mostrar una tarjeta específica. |
| [KanbanSave](../WDLang1/1410089562.md) | Actualiza el registro enlazado a la tarjeta actual en el control Kanban. |

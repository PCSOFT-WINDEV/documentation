
## docNumbering (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docNumbering** se utiliza para definir las características avanzadas de una numeración utilizada en un procesador de textos Document. Las características de esta numeración pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDoc is Document = WP_NoName1
MyNumbering is docNumbering
Level is docNumberingLevel

// Define the numbering

// 1st numbering level
Level.Text = "%1) "
Level.Text = "%1) "
Level.InitialValue = 1
Level.PageLayout.IndentFirstLine = 10
Level.Format = numfRomanUppercase
Add(MyNumbering.Level, Level)

// Second numbering level
Level.Text = "%2] "
Level.InitialValue = 1
Level.PageLayout.IndentFirstLine = 20
Level.Format = numfUppercaseLetter
Add(MyNumbering.Level, Level)

// Third numbering level
Level.Text = "%3\ "
Level.InitialValue = 1
Level.PageLayout.IndentFirstLine = 30
Level.Format = numfLowercaseLetter
Add(MyNumbering.Level, Level)


// Add the numbering to the document
let id = Add(MyDoc.Numbering, MyNumbering)

// Define the numbering associated with paragraph 1
MyDoc.Paragraph[1].Numbering.Identifier = id
MyDoc.Paragraph[1].Numbering.Level = 1

// Display the document in the Word Processing control
WP_NoName1 = MyDoc
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docNumbering
<a name="propiedades_especificas_descripcion_las_variables_docnumbering_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una numeración description:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| denominar | Cadena de caracteres | Nombre de la numeración. |
| Identificador | Integro | Identificador de la numeración. Este identificador puede utilizarse para especificar la numeración utilizada por un párrafo.<br><br>Este Property se define cuando la numeración se añade al Document.  <br><br>**Esta propiedad es de solo lectura**. |
| Nivel | Array de [docNumberingLevel](../WDLang1/1000022799.md) | Lista de descripciones de los niveles de numeración. |
| Tmpl | Cadena de caracteres | Identificador específico para utilizar la numeración en MS Word. <br><br>**Esta propiedad es de solo lectura**. |

**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;



<a name="NOTE0_2"></a>


### Añadir una numeración description a un Document
<a name="anadir_una_numeracion_description_document_ELTPARAGRAPHE000084"></a>

Para añadir una numeración description a una Document, tiene la posibilidad de utilizar [añadir](../WDLang1/1000017348.md) con la siguiente sintaxis: 

```txt
Add(Document.Numbering, MyNumbering)
```
 donde: 

- El documento es una [Document](../WDLang1/1000022461.md) Variable. 

- MyNumbering es una Variable de tipo [docNumbering](../WDLang1/1000022794.md).




## &lt;Editor de HTML&gt;.get (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera asincrónicamente el valor de una de las propiedades de un editor de HTML control. 

**Observación**: Esta función es particularmente useful para las aplicaciones Android e iOS para las que las propiedades [Cuerpo](../Proprietes/1000025914.md) , [Valor](../Proprietes/2510130.md) y [TextWithoutFormat](../Proprietes/1000019118.md) WLanguage no están disponibles (ya que sólo pueden utilizarse de forma sincrónica).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the entered data in a file
HTMEDT_MyEditor.Get(hegValue, HTMLEditorGet_Callback)
INTERNAL PROCEDURE HTMLEditorGet_Callback(value)
	fSaveText("save.html", value)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Editor control>.Retrieves(<Property> , <WLanguage procedure>)`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control.

**`<Property>: (Constante de tipo Integer)`**

Property para la cual el valor debe ser recuperado; 



|   |   |
| --- | --- |
| *hegBody* | Usó para get el código HTML del cuerpo.<br><br>Esta constante equivale a la [Cuerpo](../Proprietes/1000025914.md) Property. |
| *hegTextWithoutFormat* | Sirve para get el texto contenido en el control sin las etiquetas HTML. <br><br>Esta constante equivale a la [TextWithoutFormat](../Proprietes/1000019118.md) Property. |
| *hegValue* | Usó para get el valor de la control. <br><br>Esta constante equivale a la [Valor](../Proprietes/2510130.md) Property. |



**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback") ejecutado a get el valor buscado. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name>(Value)
```
donde &lt;Valor&gt; es el valor buscado. En este parámetro, los caracteres se codifican en UTF-8.

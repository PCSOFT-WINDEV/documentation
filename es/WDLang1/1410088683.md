
## PicRectangularSnip (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Captura un área rectangular con el cursor y almacena la imagen en una variable de tipo [Image](../WDLang1/1000019650.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCapture is Image
MyCapture = PicRectangularSnip(True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PicRectangularSnip([<Minimize>])`
---

**`<Result>: (Variable de tipo Image)`**

Variable [Image](../WDLang1/1000019650.md) que contiene el resultado de la captura. 

**`<Minimize>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para minimizar la aplicación mientras se realiza la captura de pantalla,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

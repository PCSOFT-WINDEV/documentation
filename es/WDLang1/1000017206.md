
## MultimediaAbrir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un archivo multimedia en una control Multimedia. El archivo se carga en la memoria. Para reproducir el archivo, debe utilizar [MultimediaPlay](../WDLang1/1000017202.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"
MultimediaOpen(MM_Multimedia1, MyFile)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MultimediaOpen(<Multimedia control> , <File to open>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo se abrió correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control utilizado para reproducir el archivo multimedia.

**`<File to open>: (Cadena de caracteres)`**

Nombre y ruta del archivo multimedia a abrir.




## &lt;Hoja de cálculo&gt;.RestorePositionAndSelection ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
restaura: 

- la posición de la barra de desplazamiento en una control Hoja de cálculo,

- la hoja de trabajo actual,

- los elementos seleccionados en la hoja de trabajo actual de control Hoja de cálculo. 


Estos elementos han sido guardados por [&lt;Hoja de cálculo&gt;.SavePositionAndSelection](../WDLang1/1000023735.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Let MySave = PSHEET_Spreadsheet1.SavePositionAndSelection() 

// Displays the B1 cell in bold
PSHEET_Spreadsheet.SelectPlus("B1")
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)

// Restores the position/selection 
PSHEET_Spreadsheet1.RestorePositionAndSelection(MySave)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.RestorePositionAndSelection([<Save>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Save>: (Cadena de caracteres opcional)`**

Guardar características devueltas por [&lt;Hoja de cálculo&gt;.SavePositionAndSelection](../WDLang1/1000023735.md). 

- Si no se especifica este parámetro, se utiliza la última copia de seguridad realizada por [&lt;Hoja de cálculo&gt;.SavePositionAndSelection](../WDLang1/1000023735.md) en la control (si existe).

- Se produce un error si no se ha realizado ninguna copia de seguridad o si la cadena especificada no es válida..






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000191"></a>

- El nombre de la control utilizada para guardar ([&lt;Hoja de cálculo&gt;.SavePositionAndSelection](../WDLang1/1000023735.md)) puede diferir del nombre utilizado para la operación de restauración. Esta característica puede ser useful para duplicar la posición y la selección en dos controles de la hoja de cálculo que se muestran alternativamente, por ejemplo.

- Si el control está vacío, la función **&lt;Hoja de cálculo&gt;.RestorePositionAndSelection** no tiene ningún efecto.

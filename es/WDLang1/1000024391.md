


## &lt;Procesamiento de palabras&gt;.redoCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de acciones canceladas que se pueden rehacer en un control Procesador de texto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the number of canceled actions that can be redone
Info(WP_MyDocument.RedoCount())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.RedoCount()`
---

**`<Result>: (Integro)`**

Número de acciones canceladas que se pueden rehacer.

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 



<a name="XComponent"></a>

## Componente:
wd280mdl.dll


## Entrada asistida en una control Campo de entrada (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000096"></a>
Implementar sugerencias de entrada en una edición control es muy sencillo: cuando el usuario empieza a escribir en una control de edición, se abre automáticamente una lista debajo de la control y muestra los valores que empiezan por los caracteres que se están escribiendo. 

Las sugerencias de entrada pueden implementarse en el editor o personalizarse mediante programación.. 



<a name="NOTE2_Window"></a>
<a name="NOTE2_Window_1"></a>


## Principio
<a name="principio_ELTTEXTE000140"></a>
WINDEV, WINDEV Mobile y WEBDEV incluyen varios tipos de sugerencias de entrada: 

- Sugerencias de entrada en memoria: La Edición control no está vinculada a un elemento de un archivo de datos. La entrada de automatic se realiza a partir de una lista de datos que se proporciona en el código ([&lt;AddAssistedInput](../WDLang1/1000023671.md)). Los datos de la lista se visualizan según la entrada.  

- Entrada predictiva con enlace de datos: La Edición control está vinculada a un elemento de un archivo de datos o a una consulta. Los datos de la lista se visualizan según la entrada.

- Sugerencias de entrada definidas mediante programación: Las sugerencias proceden de una set personalizada de datos. La visualización de los datos de la lista en función de la entrada se personaliza.

## Sugerencias de entrada en memoria
<a name="sugerencias_entrada_memoria_ELTTEXTE000179"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000202"></a>

Para aplicar sugerencias de entrada en memoria: 

1. Utilice [&lt;AddAssistedInput](../WDLang1/1000023671.md) para añadir los elementos que aparecerán en las sugerencias de entrada. 
	Puede añadir: 

	- un valor simple, que aparecerá en la lista de sugerencias de entrada,

	- un valor que se mostrará y otro que se almacenará.

	- una variable de tipo [AssistedInput](../WDLang1/1410088983.md).




2. Si es necesario, utilice [&lt;ConfigureAssistedInput](../WDLang1/1000024064.md) para configurar las sugerencias de entrada: filtro, apertura de lista, número de caracteres escritos antes de abrir la lista, ....




**Observación**: El valor almacenado permite asociar un valor específico a una sugerencia de entrada. Puede utilizar este valor para realizar una Process específica según el tipo de valor seleccionado.


### Ejemplos
<a name="ejemplos_ELTPARAGRAPHE000242"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Ejemplo de una ventana**: 
	
	```wl
	// End of window initialization
	EDT_Find.ConfigureAssistedInput(aiMinSize, 3)
	EDT_Find.ConfigureAssistedInput(aiFilter, filterContains)
	
	FOR EACH STRING sTitle OF gsPhotoTitles SEPARATED BY CR
		EDT_Find.AssistedInputAdd(sTitle)
	END
	```






### Observaciones
<a name="observaciones_ELTPARAGRAPHE000273"></a>

- La Event opcional "Seleccionar un valor en la lista de sugerencias de entrada" se ejecuta cuando se selecciona un elemento. Esta Event toma como parámetro una Variable de tipo [AssistedInput](../WDLang1/1410088983.md). Esta Variable le permite get el valor seleccionado y el valor associated almacenado, si existe.

- Por Default, el valor que aparece en el Editar control es el valor está seleccionado en las sugerencias de entrada. Para modificar el valor en el control Edición, utilice la función "Seleccionar un valor en la lista de sugerencias de entrada" Event. Puede elegir mostrar el valor almacenado en su lugar, por ejemplo. 


Para obtener más información, consulte [Eventos asociados a los controles Campo de entrada](../WDChamp/1013229.md).

## "Entrada asistida por"navegación
<a name="entrada_asistida_pornavegacion_ELTTEXTE000287"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000310"></a>

Poner en práctica las sugerencias de entrada basadas en un archivo de datos: 

1. En la ventana Editar control description: 

	- Compruebe si la Edición control está vinculada a un elemento de un archivo de datos o de consulta (pestaña "Vinculación"). 
			Atención: La posición debe ser una clave. 

	- En la pestaña "Detalles", seleccione la opción "Entrada asistida basada en enlace de datos". 




2. Validar la ventana Editar control description. 




En el momento de la ejecución, en cuanto se teclea la primera letra en la control: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) se muestra el primer resultado que coincide con el carácter introducido. Los datos de coincidencia cambian a medida que se escribe. 




Observaciones: 

- Si la Edición control está vinculada a un elemento de consulta, la consulta se ejecuta automáticamente. Sin embargo, si la consulta espera parámetros, debe especificarlos (por ejemplo, en los eventos "Declaraciones globales" o "Fin de la inicialización" de la ventana) y luego ejecutar la consulta con [&lt;Fuente&gt;.ExecuteQuery](../WDLang4/1000024944.md).  

- Las sugerencias de entrada se muestran mediante un filtro "Empieza por".. Para utilizar otro filtro, debe definir las sugerencias de entrada mediante programación.

## Sugerencias de entrada definidas mediante programación
<a name="sugerencias_entrada_definidas_mediante_programacion_ELTTEXTE000347"></a>


### ¿Cómo proceder?
<a name="¿como_proceder_ELTPARAGRAPHE000370"></a>

Al definir las sugerencias de entrada mediante programación, puede utilizar una fuente de datos personalizada. 

Para utilizar sugerencias de entrada definidas mediante programación: 

1. Desactive el filtro predeterminado con la función [&lt;ConfigureAssistedInput](../WDLang1/1000024064.md) (utilizando la constante *filterNone*). De este modo, la lista de sugerencias de entrada puede abrirse y mostrar todos los elementos especificados.  

2. En la "Siempre que modifique" Event del Editar control: 

	- Borrar la lista de sugerencias ([&lt;Edición&gt;.DeleteAllAssistedInput](../WDLang1/1000024065.md)). 

	- Reconstruir la lista de elementos propuestos según los caracteres tecleados ([&lt;AddAssistedInput](../WDLang1/1000023671.md)).

	- Utilice [&lt;OpenAssistedInput](../WDLang1/1000024063.md) para abrir la lista de sugerencias de entrada.








### Ejemplo
<a name="ejemplo_ELTPARAGRAPHE000409"></a>


```wl
// "Contains" search performed in the name 
// and code of an array of stock shares
FOR EACH sName, sCode of gArrShare
	IF Contains(sCode, sSoughtValue) _OR_ ...
		Contains(sName, sSoughtValue) THEN
		SAI_Programmed_mode.AssistedInputAdd(sName)
	END
END
```


## ActionBarSearchVisible( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite get y cambiar el estado de visibilidad del control de búsqueda associated con el Action Bar en la ventana actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ActionBarSearchVisible(True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Modificando el estado de visibilidad del control de búsqueda associated con el Action Bar

`ActionBarSearchVisible(<Visible> [, <Indication>])`
---

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control de búsqueda debe hacerse visible. 

- <u><u><u><u>False</u></u></u></u> si el control de búsqueda debe convertirse en Invisible. 




**`<Indication>: (Cadena de caracteres opcional)`**

Texto de indicación que se mostrará en la búsqueda control.


<a name="SYNTAX2"></a>

### Averiguar el estado de visibilidad del control de búsqueda associated con el Action Bar

`<Result> = ActionBarSearchVisible()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control de búsqueda es visible,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- También se mostrará un historial de búsqueda autocompletado según las opciones del Action Bar control.

- Esta función se aplica a la ventana actual (lo que significa la ventana Active durante la llamada). 

- Para que se aplique esta función, la ventana actual debe tener un control Action Bar con la opción "Permitir búsqueda en el Action Bar" seleccionada. De lo contrario, se producirá un error fatal. Para obtener más información, consulte [Descripción del control Action Bar](../WDChamp/1000025005.md).

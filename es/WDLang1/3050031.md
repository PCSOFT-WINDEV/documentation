
## MatDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una matriz existente. La eliminación de una matriz no utilizada (o nula) de una aplicación se utiliza para liberar la huella de memoria ocupada por esta matriz.

**Observación**: Todas las matrices utilizadas en una aplicación se borran automáticamente al cerrar la aplicación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete a matrix
MatDelete("MyMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MatDelete(<Matrix>)`
---

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a eliminar, definida por [MatCreate](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz a borrar
<a name="existencia_matriz_borrar_ELTPARAGRAPHE000076"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md). Si la matriz no existe, **MatDelete** no tiene ningún efecto.




## Funciones de control Cuadro de mando
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del Tablero:



|   |   |
| --- | --- |
| [DashAddWidget](../WDLang1/1000020953.md) | Añade y muestra un widget (ventana interna) en un control Cuadro de mando. |
| [DashCount](../WDLang1/1000021042.md) | Devuelve el número de widgets encontrados en una control Cuadro de mando. |
| [dashDelete](../WDLang1/1000020955.md) | Elimina un widget mostrado en un control Cuadro de mando. El control Cuadro de mando se actualiza y una ubicación libre reemplaza al widget eliminado. |
| [DashDeleteAll](../WDLang1/1000020954.md) | Elimina todos los widgets mostrados en un control Cuadro de mando. |
| [DashDisplay](../WDLang1/1000020970.md) | Elementos Actualiza en un control Cuadro de mando. |
| [DashGetWidgetPosition](../WDLang1/1000022498.md) | Devuelve la posición de un widget que se encuentra en una control Cuadro de mando. |
| [DashInfoXY](../WDLang1/1000020974.md) | Permite saber:<br><br>- si un widget se encuentra en una posición determinada en un control Cuadro de mando.<br><br>- cuyo widget se encuentra en una posición determinada en un control Cuadro de mando.<br><br><br> |
| [DashInitialConfiguration](../WDLang1/1000021043.md) | Modifica la configuración de una control Cuadro de mando (definida en el editor) antes de su primera visualización. |
| [DashLoadConfiguration](../WDLang1/1000020961.md) | Restaura la configuración de una control Cuadro de mando. |
| [DashMode](../WDLang1/1000020956.md) | Abre o cierra el modo de edición de control Cuadro de mando. |
| [DashMoveWidget](../WDLang1/1000021289.md) | Cambia la posición de un widget que se encuentra en un control Cuadro de mando. |
| [DashResizeWidget](../WDLang1/1000021286.md) | Cambia las dimensiones de un widget que se encuentra en un control Cuadro de mando. |
| [DashSaveConfiguration](../WDLang1/1000020960.md) | Guarda la configuración de una control Cuadro de mando en formato de cadena. |





Otras funciones disponibles: 



|   |   |
| --- | --- |
| [DnDCacheDashElement](../WDLang1/1000020971.md) | Añade un widget en una control Cuadro de mando durante la función de arrastrar y soltar. |

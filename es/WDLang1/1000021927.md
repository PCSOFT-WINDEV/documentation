
## Usar la modificación de UI cuando se ejecuta un proyecto o una prueba de ventana
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
Los usuarios finales pueden modificar directamente la interfaz de la aplicación (según las opciones especificadas por el desarrollador). 

Estas modificaciones también están disponibles cuando se prueba una ventana o un proyecto: 

- Creación de una note reposicionable ,

- Crear un control Conector, 

- Modificando la punta de la herramienta de un control,

- Mover uno o más controles, ....




Al volver del modo GO, si se han realizado modificaciones en la interfaz, el editor de ventanas propone aplicar automáticamente estas modificaciones a la interfaz de ventanas.. 

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Ejemplo de uso
<a name="ejemplo_uso_ELTTEXTE000129"></a>
Para recuperar las modificaciones realizadas en el proyecto o en la ventana IR: 

1. Ejecutar el GO de proyecto (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_GO_Projet_WD.gif)) o ventana (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ICO_GO_Fenetre_WD_bl.gif)). 

2. Seleccione las ventanas solicitadas.

3. Modificar la UI de las ventanas deseadas. Por ejemplo: 

	- Crear una nota reposicionable control. 

	- Cambiar al modo de modificación y: 

		- Mueve los controles,

		- Crear un control Conector.




- Modificar el texto de la información sobre la herramienta.

4. Finalizar prueba. 

5. Al volver al editor, se muestra una ventana específica. Esta ventana se utiliza para recuperar las modificaciones de la UI realizadas en el modo GO. 

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMH_Modifiable_3%20-%20HC%20N%B0001.gif)


Esta ventana presenta, para cada ventana modificada, las modificaciones que se pueden aplicar: 

- Modificar el tamaño y la posición: se ha modificado el tamaño y/o la posición de uno o más controles. 

- Crear un control: se han creado uno y/o varios control (Wire control, Note control). 

- Modificación de propiedades: se han modificado una y/o varias propiedades de control. Las modificaciones de la información sobre herramientas se tienen en cuenta en esta categoría. 




Con los botones "Todos" y "Ninguno" se seleccionan las modificaciones que se aplicarán o no.. Al expandir el botón, tiene la posibilidad de seleccionar el tipo de modificación a tener en cuenta: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMH_Modifiable_3%20-%20HC%20N%B0002.gif)


**Observación**: Esta ventana también se muestra cuando se importa un archivo "\*.wdmodif" en el editor.. Este archivo se genera al exportar las modificaciones de la interfaz ("Personalizar la interfaz... Exportar la personalización de esta ventana" en el menú contextual de la ventana). Para obtener más información, consulte [UI modificable por el usuario final](../WDLang1/1000021853.md).




## <Menu>.InsertSeparator (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un separador en un menú. Para insertar un separador al final del menú, utilice [.AddSeparator](../WDLang1/1000020603.md).

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a new menu
_Menu.AddMenu("MNU_Schedule", "Schedule")
// Add options into the new menu
MNU_Schedule.AddOption("MNU_Create", "Create", Create_Schedule)
"MNU_Schedule".AddOption("MNU_View", "View", View_schedule)
// Insert a separator
MNU_Schedule.InsertSeparator("MNU_Create")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un separador (posición definida por un nombre de opción de menú)

`<Source menu>.InsertSeparator(<Insertion option>)`
---

**`<Source menu>: (Nombre del menú)`**

Nombre del menú donde se debe insertar el separador.

**`<Insertion option>: (Cadena de caracteres)`**

Nombre de la opción de menú utilizada como posición base para la inserción. El nuevo separador se insertará antes de esta posición.


<a name="SYNTAX2"></a>

### Insertar un separador (posición definida por un índice)

`<Source menu>.InsertSeparator(<Subscript>)`
---

**`<Source menu>: (Nombre del menú)`**

Nombre del menú donde se debe insertar el separador.

**`<Subscript>: (Integro)`**

Subíndice de la opción de menú utilizada como posición base para la inserción. El nuevo separador se insertará antes de esta posición.

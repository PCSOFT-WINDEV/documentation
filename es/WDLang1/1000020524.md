
## gToText( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los elementos gráficos de una cadena de caracteres que utiliza [gStoredValue](../WDLang1/1410088363.md), [gImage](../WDLang1/3029042.md), ...


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
s is string = gImage("Blog.bmp") + "Test" + gStoredValue("3")
ListAdd(LIST_List1, s)

IF gToText(s) = "Test" THEN
	Info("Test")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gToText(<String to process>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres con los elementos gráficos. 

**`<String to process>: (Cadena de caracteres)`**

Cadena de caracteres que contiene (o no) elementos gráficos ([gStoredValue](../WDLang1/1410088363.md), [gImage](../WDLang1/3029042.md), [gRectangle](../WDLang1/3029045.md), ....

Si la cadena no contiene ningún elemento gráfico, &lt;Resultado&gt; será la cadena a Process.


## Portapapeles (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera el texto o Image que se encuentra en el portapapeles del sistema.

Observaciones: 

- Para identificar el contenido del portapapeles, utilice [ClipboardFormat](../WDLang1/3063002.md).

- Para recuperar un texto en RTF, utilice [PortapapelesRTF](../WDLang1/1000017200.md).

- Para recuperar un texto en formato Unicode, utilice [PortapapelesUnicode](../WDLang1/1000018858.md).




![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo puedes recuperar el texto del portapapeles.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Text is string
IF ClipboardFormat(cfText) = True THEN
	Text = Clipboard()  // Clipboard content in Text
END
```

## Sintaxis

### Recuperar un texto

`<Result> = Clipboard()`
---

**`<Result>: (Cadena de caracteres)`**

Texto en el portapapeles.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Recuperar un Image en un control Imagen

`<Result> = Clipboard(<Image control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha obtenido la imagen del portapapeles,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image control>: (Nombre del control)`**

Nombre de la control Imagen (en la ventana actual) a la que se copiará la Image en el portapapeles.

`<Result> = Clipboard(<Image>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha obtenido la imagen del portapapeles,

Nombre de la [Image](../WDLang1/1000019650.md) Variable a la que debe copiarse el portapapeles Image.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **Portapapeles** lee la información de tipo texto o imagen que se encuentra en el portapapeles del sistema actual. **Portapapeles** no se puede utilizar para leer cualquier tipo de información que no sea texto o image (por ejemplo, binario, etc.) en el portapapeles.. Para leer un texto en RTF, utilice [PortapapelesRTF](../WDLang1/1000017200.md).

- La función **Portapapeles** puede utilizarse para cortar/copiar/pegar la información de tipo texto o imagen.

- Si el contenido del portapapeles debe asignarse varias veces a una control Imagen, debe utilizarse [dEndDrawing](../WDLang1/3029019.md) entre las distintas asignaciones.

- [ToClipboard](../WDLang1/3063005.md) se utiliza para escribir en el portapapeles.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si la información de transparencia de la imagen se mantiene al copiar la imagen en el portapapeles ([ToClipboard](../WDLang1/3063005.md)), esta información también se mantiene al copiar la imagen en el control Imagen y/o en el [Image](../WDLang1/1000019650.md) Variable. 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo se puede recuperar la información de Texto (Sintaxis 1).

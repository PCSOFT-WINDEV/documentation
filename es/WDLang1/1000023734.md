


## &lt;Hoja de cálculo&gt;.Save (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el contenido de una control Hoja de cálculo en un archivo en formato XLSX.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the current file
PSHEET_Spreadsheet.Save("Workbook1.XLSX")
```

```wl
// Load the file
PSHEET_NoName1.Load("test.xlsx")
// Modify the file
PSHEET_NoName1["A","1"] = "modified"
// Save under a different name
// The "text1.xlsx" file already exists
IF PSHEET_NoName1.Save("test1.xlsx") = False THEN
	// Before version 200051, the existing xlsx file is overwritten. 
	// From version 200051, the existing xlsx file is not modfified. 
	// The function returns False and an error 
	// To overwrite the file, use the psheetOverwriteAllowed constant
	Error("File not saved " + ErrorInfo()
ELSE
	Info("File saved")
END
```


## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.Save(<File to save> [, <Option> [, <Password>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se ha guardado el fichero de control Hoja de cálculo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<File to save>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo XLSX donde se guardará el contenido de control Hoja de cálculo.

**`<Option>: (Constante opcional de tipo Integer)`**

Modo de guardar el archivo si el archivo ya existe: 


|   |   |
| --- | --- |
| 0 (o nada)<br>(Comportamiento de Default) | Si el archivo XLSX ya existe y no está abierto (en control Hoja de cálculo o en otra aplicación), no se guarda. |
| *psheetOverwriteAllowed* | Si el archivo XLSX ya existe y no está abierto (en control Hoja de cálculo o en otra aplicación), se sobrescribe. |



**`<Password>: (Cadena de caracteres opcional)`**

Contraseña para proteger el archivo XLSX al abrirlo. Esta contraseña debe ser especificada para abrir el archivo XLSX.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para abrir el archivo ".xlxs" guardado con otra aplicación, debe ser cerrado por [&lt;Hoja de cálculo&gt;.close](../WDLang1/1000023722.md). 

- El archivo guardado puede volver a cargarse en un control Hoja de cálculo a través de [&lt;Hoja de cálculo&gt;.load](../WDLang1/1000023719.md) y [xlsAbrir](../WDLang5/3080017.md).

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana.. 

- La función **&lt;Hoja de cálculo&gt;.Save** actualiza la propiedad [FilePath](../Proprietes/1000023104.md) del control Hoja de cálculo asociando el nombre del archivo cargado al control Hoja de cálculo.




## GetColor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite obtener una set de colores armoniosos sin tener que usar una tabla de colores.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Fill in the data of the Chart control
FOR I = 1 TO 12
	// Adds the turnover of month number I
	grAddData(CHART_MyChart, I, arrValues[I])
	
	// Assigns a color to the series of month number I
	grSeriesColor(CHART_MyChart, I, GetColor())	
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = GetColor([<Subscript> [, <Serial>]])`
---

**`<Result>: (Integro)`**

Color en formato RGB.



**`<Subscript>: (Entero opcional)`**

Número de la color a recuperar. 
Si no se especifica este parámetro, el Color regresó en un Color que nunca fue usado antes. 
Si se especifica este parámetro, siempre se devolverá la misma color para una índice dada .


**`<Serial>: (Constante opcional de tipo Integer)`**

Constante que indica el estilo de color set para recuperar: 


|   |   |
| --- | --- |
| *colSetDark* | El color devuelto se elige entre un set de color s más oscuros (adecuados para el control TreeMap por ejemplo). |
| *colSetDefault*<br>(Valor predeterminado) | La color devuelta se elige entre una set de color s estándar. |
| *colSetPalette* | El color devuelto es elegido entre los colores de la set definidos para la Palette actual. <br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *colSetPastel* | La color devuelta se elige entre una set de color s más ligeros. |
| *colSetTemplate* | El color devuelto se elige entre los set de color s definidos para el plantilla actual . <br><br> |

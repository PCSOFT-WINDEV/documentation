
## RegistryDeleteKey( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una subclave del registro de Windows.

**Observación**: La función **RegistryDeleteKey** elimina la subclave, los valores y los descendientes de la clave especificada.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Path is string
// Find out the name of "root" key
Path = RegistryFirstSubKey()
// Delete the key for my application
Path = Path + "\MyApp\"
IF RegistryDeleteKey(Path) = False THEN
	Error("Error while deleting the key")
	Return
END
```


<a name="Example2"></a>



```wl
// Deletes the "HKEY_LOCAL_MACHINE\SYSTEM\App" subkey
ResDelKey = RegistryDeleteKey("HKEY_LOCAL_MACHINE\SYSTEM\App")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RegistryDeleteKey([<Access mode>, ] <Sub-key path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la subclave o si la clave no existe,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Sub-key path>: (Cadena de caracteres)`**

ruta completa de la subclave a eliminar.

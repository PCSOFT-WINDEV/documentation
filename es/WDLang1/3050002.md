
## DecimalPart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la parte decimal de un número.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Decimal part of 16.487
ResDecimalPart = DecimalPart(16.487)
// Returns 0.487
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DecimalPart(<Number>)`
---

**`<Result>: (Real)`**

Parte decimal solicitada.

**`<Number>: (Real)`**

Número a utilizar.

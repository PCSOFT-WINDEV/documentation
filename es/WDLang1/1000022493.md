
## docElementText (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docElementText** se utiliza para manipular las características de un elemento Text que se encuentra en una variable [docElement](../WDLang1/1000022484.md).

Las características de este tipo de variables pueden ser manejadas por varias propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no se puede utilizar directamente. Debe utilizarse mediante el tipo [docElement](../WDLang1/1000022484.md).**


<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docElementText
<a name="propiedades_especificas_descripcion_las_variables_docelementtext_ELTPARAGRAPHE000030"></a>Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docElementText**:


   | --- |
| --- | --- | --- |
| **Nombre de la propiedad** | Tipo utilizado | Efecto |
| Formato | [docFormato](../WDLang1/1000022817.md) variable | Información sobre el formato de los elementos. <br><br> |
| Valor | Cadena de caracteres | Texto del elemento. |


**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;..&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;

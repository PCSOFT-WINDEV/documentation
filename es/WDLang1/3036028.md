
## fLongPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la ruta completa y larga de un fichero o directorio.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036028&name=flongpath_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Retrieve the long path of a file
ResLongPath = fLongPath("C:\MyDir~1\Account~1.doc")
// fLongPath returns "C:\MyDirectories\Report March 2000.doc"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fLongPath(<File path> [, <Option>])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre largo correspondiente a la ruta completa del fichero o directorio. Si &lt;Archivo ruta&gt; termina con el carácter "\\", este carácter se mantiene.

- Vaciar la cadena (""") si se ha producido un error (por ejemplo, si el archivo o directorio especificado no existe).. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<File path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del fichero o directorio (hasta 260 caracteres). Se puede utilizar una ruta UNC. 
Este parámetro puede estar en formato Ansi o Unicode.

**`<Option>: (Constante de tipo Integer)`**

Se usa para especificar el formato del resultado de la función. 


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. |


**Observación**: Si no se especifica la constante *fPathUNICODE* y si la función debe devolver una cadena Unicode, &lt;Resultado&gt; corresponderá al corto ruta del archivo.

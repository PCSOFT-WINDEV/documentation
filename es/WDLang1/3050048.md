
## MatCopy( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea la copy de una matriz. La nueva matriz y la matriz fuente tienen las mismas dimensiones. Todos los elementos encontrados en la matriz fuente se copian en la nueva matriz.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copy a matrix
ResCopy = MatCopy("MyMatrix", "MyCopiedMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatCopy(<Matrix to copy> , <Copied matrix>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se copió la matriz,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Matrix to copy>: (Cadena de caracteres)`**

Nombre de la matriz a copiar, definida con [MatCreate](../WDLang1/3050058.md).

**`<Copied matrix>: (Cadena de caracteres)`**

Nombre de la matriz copiada, creada automáticamente durante la copy.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz copiada
<a name="existencia_matriz_copiada_ELTPARAGRAPHE000089"></a>

No se verifica la existencia de la matriz.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).




## &lt;Vista de Árbol&gt;.ColapsoTodo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Colapsa una rama previamente expandida (así como sus elementos hijos): 

- Los nodos "niños" ya no son Visible en el control TreeView. 

- Los nodos hijos están colapsados.




Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)
**Reminder**: En tiempo de ejecución, un nodo se colapsa cuando el nombre del nodo está precedido por "+".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Collapse the "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TREE_TVRecipe.CollapseAll("Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeView control>.CollapseAll([<Node to collapse>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nodo y sus hijos han sido colapsados,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Node to collapse>: (Cadena de caracteres opcional)`**

Ruta del nodo a colapsar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si no se especifica este parámetro, control TreeView se colapsa desde la raíz.

Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nodo a colapsar
<a name="nodo_colapsar_ELTPARAGRAPHE000197"></a>

Si el nodo se colapsa:

- no se encuentra, **&lt;Vista de Árbol&gt;.ColapsoTodo** devuelve <u><u><u><u>False</u></u></u></u>.

- no se especifica, **&lt;Vista de Árbol&gt;.ColapsoTodo** devuelve <u><u><u><u>True</u></u></u></u>. La control TreeView está colapsada desde su raíz.

- es una hoja, **&lt;Vista de Árbol&gt;.ColapsoTodo** devuelve <u><u><u><u>True</u></u></u></u>.

- ya está colapsado, **&lt;Vista de Árbol&gt;.ColapsoTodo** devuelve <u><u><u><u>True</u></u></u></u>.



<a name="NOTE0_2"></a>


### Banda de selección
<a name="banda_seleccion_ELTPARAGRAPHE000236"></a>

- Si la banda de selección se posicionó en uno de los elementos del nodo a colapsar, se posiciona en el nodo colapsado después de la llamada a **&lt;Vista de Árbol&gt;.ColapsoTodo**.

- El código de modificación se llama cada vez que se mueve la banda de selección. Si dos niveles están contraídos y la banda de selección ser posiciona en el nodo superior, se ejecutan los códigos de modificación de cada nivel.

- La ruta completa del elemento seleccionado es devuelta por [&lt;TreeView&gt;.Select](../WDLang1/1000023767.md).

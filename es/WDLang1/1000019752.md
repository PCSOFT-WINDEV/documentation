


## &lt;TreeMap&gt;.Count (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos que se encuentran en una control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements in TMAP_File
ResNumberElements = TMAP_File.Count()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeMap control>.Conde()`
---

**`<Result>: (Integer)`**

Número de elementos encontrados en el TreeMap (0 si el control está vacío). 
**Observación**: Se tienen en cuenta todos los elementos (incluso los elementos cuyo peso es 0).

**`<TreeMap control>: (Control name)`**

Nombre de control TreeMap a utilizar.

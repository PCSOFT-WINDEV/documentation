
## fTempDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del directorio utilizado para almacenar los archivos temporales.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the name of temporary directory
ResTempDir = fTempDir()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fTempDir()`
---

**`<Result>: (Cadena de caracteres)`**



- Ruta del directorio utilizado para almacenar los archivos temporales (hasta 260 caracteres). Esta ruta termina con el carácter "\\".. 
	

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.   


**Observación**: El resultado de la función **fTempDir** siempre es una cadena de caracteres en formato Ansi. Sin embargo, el resultado puede almacenarse en una cadena de caracteres Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Directorio temporal
<a name="directorio_temporal_ELTPARAGRAPHE000042"></a>

Si el directorio temporal soportado por el sistema no existe en el equipo actual, la función **fTempDir** devuelve: 

- en Windows 98 o Me: el directorio actual de el equipo. 

- en Windows 2000, NT o CE: el nombre del directorio temporal Default soportado por el sistema. No se realiza ninguna prueba sobre la existencia del directorio temporal.




**Atención**: En Windows Vista y posteriores (cuando la [Sistema UAC](../Editeurs/2025014.md) está activada), el directorio temporal utilizado depende de si se han concedido privilegios de administrador al usuario o no.


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000057"></a>

La función **fTempDir** es equivalente a la función [fTempPath](../WDLang1/3036044.md).


## ContinentGet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la variable de tipo [Continent](../WDLang1/1000024919.md) que corresponde a uno de los siguientes valores: 

- una constante de continente (código de continente de 2 caracteres),

- un nombre de continente.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve information about Europe
oContinent is Continent = ContinentGet(continentEurope)
// Display the caption 
STC_Continent = oContinent.Name
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ContinentGet([<Caption, constant or code of the continent>])`
---

**`<Result>: (Variable de tipo Continent)`**

Variable de tipo [Continent](../WDLang1/1000024919.md) que corresponde a un nombre o código de continente. 

**`<Caption, constant or code of the continent>: (Cadena de caracteres opcional)`**



- Nombre del continente buscado (en español, inglés o francés). 

- Constante que identifica el continente (o código de continente de 2 caracteres):
	


|   |   |
| --- | --- |
| *continentAfrica* | África (la constante vale "AF"). |
| *continentAntarctica* | Antártida (la constante vale "AN"). |
| *continentAsia* | Asia (la constante vale "AS"). |
| *continentEurope* | Europa (la constante vale "EU"). |
| *continentNorthAmerica* | América del Norte (la constante vale "NA"). |
| *continentOceania* | Oceanía (la constante vale "OC"). |
| *continentSouthAmerica* | América del Sur (la constante vale "SA"). |





Si no se especifica este parámetro, se utilizará el continente del sistema.

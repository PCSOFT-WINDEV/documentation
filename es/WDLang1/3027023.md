


## TimeToString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una hora en una cadena en el formato especificado.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Convert the specified time according to the "HH:MM:SS:CC" format
TimeToString("12005956")   // Returns "12:00:59:56"
Left(TimeToString("12005956"), 5) // Returns "12:00"
```

## Sintaxis

`<Result> = TimeToString(<Time to convert> [, <Format>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres en el formato especificado.

**`<Time to convert>: (Cadena de caracteres, variable de tipo Date o DateTime)`**

Hora a manipular. Esta hora puede corresponder a: 

- una cadena de caracteres en formato HHMMSSCC (HH corresponde a horas, MM a minutos, SS a segundos y CC a centésimas de segundo). 

- una cadena de caracteres en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo). En este caso, solo se tiene en cuenta la hora. 

- una variable de tipo Time. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la hora.




**`<Format>: (Cadena de caracteres opcional)`**

Formato que debe aplicarse a la hora. Este parámetro corresponde a: 

- una cadena de caracteres (una frase, por ejemplo). Los caracteres que representan los elementos de la hora (HH, MM, SS, CC o AP) se reemplazarán automáticamente por el valor del parámetro &lt;Hora&gt;. En esta cadena:

	- HH representa el número de horas,

	- MM representa el número de minutos,

	- SS representa el número de segundos,

	- CC representa el número de centésimas de segundo,

	- AP representa "AM" o "PM" (si se usa "AP", la hora se muestra en formato de 12 horas).




- la constante *maskSystemTime*. Esta constante permite utilizar directamente la máscara utilizada por la hora del sistema. 
	![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible.




Si no se especifica este parámetro, el formato del resultado es HH:MM:SS:CC.

## Observaciones


### Validez de la hora
<a name="validez_hora_ELTPARAGRAPHE000116"></a>

La validez de la hora pasada como parámetro se verifica. Si la hora no es válida, se muestra un mensaje. La validez de las horas se puede comprobar con la función [TimeValid](../WDLang1/3027021.md).

**Consejo**: Para obtener una cadena con una hora válida, la hora debe introducirse en un control Campo de entrada de tipo Hora.
<a name="NOTE0_2"></a>


### Formato de la hora
<a name="formato_hora_ELTPARAGRAPHE000128"></a>

La función **TimeToString** permite convertir una hora devuelta por la función [IntegerToTime](../WDLang1/3027006.md) o [TimeSys](../WDLang1/3027019.md). El resultado puede asignarse a un control de texto, por ejemplo.

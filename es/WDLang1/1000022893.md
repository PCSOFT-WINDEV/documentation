
## NoteBorrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una note reposicionable que se encuentra en la ventana actual. Esta note reposicionable puede ser una control o una note creada mediante programación con [WinEdCreateNote](../WDLang1/1000022270.md), [NoteCreate](../WDLang1/1000022395.md) o [ControlCreate](../WDLang1/1000020876.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [New note]
gMyNote <- NoteCreate()

// Click on [Delete]
IF gMyNote <> NULL THEN
	NoteDelete(gMyNote)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`NoteDelete(<Note to use>)`
---

**`<Note to use>: (Cadena de caracteres)`**

Nombre de la note reposicionable a eliminar. Este nombre puede corresponder a: 

- el nombre de una Nota Reposicionable control encontrada en la ventana. 

- a Control Variable associated con un note reposicionable creado con una de las siguientes funciones: [WinEdCreateNote](../WDLang1/1000022270.md), [NoteCreate](../WDLang1/1000022395.md) o [ControlCreate](../WDLang1/1000020876.md).

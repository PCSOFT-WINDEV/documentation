
## Root( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la raíz de un número.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Square root of 25
ResRoot = Root(25)
// Returns 5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Root(<Number> , <Root Factor>)`
---

**`<Result>: (Real)`**

Raíz del número.

**`<Number>: (Entero o real)`**

Número a utilizar.

**`<Root Factor>: (Entero o real)`**

Factor de la Root( 2 para una raíz cuadrada, 3 para una raíz cúbica, ...). 
Si no se especifica este parámetro, se devuelve la raíz cuadrada de &lt;Number&gt;..

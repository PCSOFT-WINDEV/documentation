
## HexaToInt (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena que corresponde a un número hexadecimal en un entero sin signo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is unsigned int
Res = HexaToInt("1F") 	
// Returns an unsigned 4-byte integer containing the value 31.

Res = HexaToInt("FF") 	
// Returns an unsigned 4-byte integer containing the value 255.
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HexaToInt(<Hexadecimal string>)`
---

**`<Result>: (Entero)`**

Resultado de la conversión. Este resultado corresponde a un entero de 4 bytes sin signo. En caso de desbordamiento, puede corresponder a un entero de 8 bytes o a un numérico.

**`<Hexadecimal string>: (Cadena de caracteres)`**

Cadena hexadecimal que se convertirá en un número entero.

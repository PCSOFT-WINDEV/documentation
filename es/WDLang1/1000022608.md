
## ActionRedo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta la última acción cancelada. Esta función equivale a un "Redo" (Ctrl + Y) realizado por el usuario.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Restore] button
ActionRedo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ActionRedo([<Window to use>])`
---

**`<Window to use>: (Nombre de la ventana opcional)`**

Nombre de la ventana o ventana interna a utilizar.

- Si no se especifica este parámetro, la acción se ejecuta en la ventana actual.. 

- Si se especifica este parámetro, la función **ActionRedo** es equivalente a la función [WinEdRedo](../WDLang1/1000021905.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000040"></a>

- **ActionRedo** no tiene efecto si no se ha cancelado ninguna acción. 

- Si la última acción es una acción personalizada ejecutada por [ActionDo](../WDLang1/1000022676.md), se ejecuta el Procedure declarado en esta función.

- La acción restaurada puede deshacerse de nuevo pulsando Ctrl + Z o mediante la programación con [ActionUndo](../WDLang1/1000022677.md).

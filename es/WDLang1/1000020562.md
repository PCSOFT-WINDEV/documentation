
## IWListPosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana interna correspondiente a la posición especificada (en la lista de ventanas internas exploradas por un control Ventana interna) o devuelve la posición de la ventana interna actualmente mostrada (en la lista de ventanas internas exploradas por un control Ventana interna).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the first internal window associated with the control 
IWListPosition(IWC_InternalWindowControl, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Mostrando las ventanas internas desde una posición determinada

`IWListPosition(<Internal Window control> , <Position>)`
---

**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.

**`<Position>: (Integro)`**

Índice de la ventana interna que se mostrará en el control.


<a name="SYNTAX2"></a>

### Averiguar la posición de la ventana interna que se muestra actualmente

`<Result> = IWListPosition(<Internal Window control>)`
---

**`<Result>: (Integro)`**



- Posición de la ventana interna que se muestra actualmente en la "Ventana interna" control. 

- -1 si no hay ventana interna es associated con el control.




**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
No se utiliza ninguna animación para cambiar la ventana interna mostrada en el control con **IWListPosition**. Para utilizar una animación al cambiar la ventana interna, llame a [IWListNext](../WDLang1/1000020564.md) y [IWListPrevious](../WDLang1/1000020565.md).




## <Check box>.Delete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una opción de un control Casilla de verificación. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes option #3 from a Check Box control
CBOX_Options.Delete(3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Check Box control>.Delete(<Subscript>)`
---

**`<Check Box control>: (Nombre del control)`**

Nombre de control Casilla de verificación del que se eliminará la opción.

**`<Subscript>: (Integro)`**

Índice de la opción a eliminar. Este índice debe estar comprendido entre 1 y el número de opciones de control Casilla de verificación. El [Conde](../Proprietes/2510097.md) Property obtiene el número de opciones en un control Casilla de verificación.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Cuando se eliminan las opciones de una control Casilla de verificación, el tamaño de la control no se modifica. El tamaño de control puede modificarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Tienes la capacidad de compactar las opciones para optimizar el aspecto de la control una vez que las opciones han sido eliminadas. 
	En el modo compactado, las opciones de una control Casilla de verificación se acercan.: 

	- cuando se hace una opción Invisible. 

	- cuando se elimina una opción.




- Para utilizar el modo compactado: 

	- marque "Compactar opciones invisibles" en la pestaña "Detalles" del control description.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilice la propiedad [CompactOption](../Proprietes/1000020608.md).




- Cuando se elimina una opción, los índices de las opciones que siguen a la opción eliminada se renumeran automáticamente.


## NumHelp (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número del contexto de ayuda associated con el control especificado. Este número de ayuda se especificó cuando se creó el control en el editor de la ventana.

**Observación**: Este número de ayuda se puede utilizar con [WHelp](../WDLang1/3013053.md) para mostrar la página de ayuda (en formato HLP o CHM) correspondiente al control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the MyHelp.HLP help on the page
// whose context number is NumHelp(NameCurrentControl)
// NameCurrentControl is a variable containing the name of the current control
WHelp("MyHelp.hlp", NumHelp(NameCurrentControl))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NumHelp(<Control name>)`
---

**`<Result>: (Integro)`**

Número de la ayuda sensible al contexto.

**`<Control name>: (Cadena de caracteres)`**

Nombre de la control cuyo número de ayuda sensible al contexto se solicita. Para las opciones de una casilla de verificación o un botón de radio, el número de la opción debe ir entre paréntesis (CheckBox[ 1] para la primera opción de una casilla de verificación, por ejemplo).

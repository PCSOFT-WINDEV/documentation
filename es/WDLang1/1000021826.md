
## ScrollbarMaxPosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la posición máxima de una de las barras de desplazamiento horizontales o verticales que se encuentran en un control de una aplicación WINDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Goes to the last elements of the Table control
ScrollbarPosition(TABLE_MyTable, sbVert, ScrollbarMaxPosition(TABLE_MyTable))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ScrollbarMaxPosition(<Control used> [, <Orientation>])`
---

**`<Result>: (Integro)`**



- Posición máxima de la caja de desplazamiento (en unidades lógicas).

- 0 si la barra de desplazamiento no existe o si es Grayed.




**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.

**`<Orientation>: (Constante opcional de tipo Integer)`**

Orientación de la barra de desplazamiento para la que debe recuperarse la posición: 


|   |   |
| --- | --- |
| *sbHorz* | La posición se calcula para la barra de desplazamiento horizontal de control. |
| *sbVert*<br>(Valor predeterminado) | La posición se calcula para la barra de desplazamiento vertical de control. |

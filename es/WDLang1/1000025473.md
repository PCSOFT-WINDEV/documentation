
## Procedimiento WLanguage llamado por la función HashCheckString



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [HashCheckString](../WDLang1/1000007113.md) al comprobar el algoritmo de hash. Este procedimiento permite obtener el resultado del hash. 

Este procedimiento debe ser Navegador. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sHash is string = "9F 04 F4 1A 84 85 14 16 20 50 E3 D6"
HashCheckString(HA_SHA_160, "abc", HexaToBuffer(sHash), HashCheckString_Callback)

INTERNAL PROCEDURE HashCheckString_Callback(bRes boolean)
	IF bRes = False THEN Trace("Error during verification")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HashCheckString_Callback(<Result>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el resultado de la cadena al aplicar el algoritmo de hash especificado corresponde al parámetro &lt;Hash&gt; de la función [HashCheckString](../WDLang1/1000007113.md),

- <u><u><u><u>False</u></u></u></u> en caso contrario.

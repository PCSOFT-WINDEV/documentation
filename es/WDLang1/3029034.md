
## gCoord( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres que se utiliza para definir la posición del cursor de entrada de un texto.. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En una ventana, esta secuencia de caracteres se utiliza para definir la posición del cursor de entrada de un texto: 

- en un elemento que se encuentra en un List Box o control Combo Box.

- en la barra de título y/o en la barra de estado de una ventana.

- in a control Estático.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Add the word "Raised" into the "LIST_WritingType" control
ListAdd(LIST_WritingType, gCoord(2,2) + "Raised" + 18 + gCoord(1,1) + "Raised")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gCoord(<X> , <Y>)`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que definen la posición de escritura.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<X>: (Real)`**

Coordenada X (en píxeles) de la nueva posición del cursor para el siguiente texto a escribir. Si este parámetro corresponde a una cadena vacía ("""), la coordenada x correspondiente a la posición del cursor no se modificará.

**`<Y>: (Real)`**

Coordenada Y (en píxeles) de la nueva posición del cursor para el siguiente texto a escribir. Si este parámetro corresponde a una cadena vacía (""), la coordenada Y correspondiente a la posición del cursor no se modificará.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todas las coordenadas se expresan en píxeles en relación con la Corner superior izquierda del elemento utilizado (la Corner superior izquierda de la fila que se encuentra en el Cuadro de Lista o control Combo Box por ejemplo).

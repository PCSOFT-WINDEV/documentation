
## LastDayOfMonth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del último día del mes especificado. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the last day for the month of February 2017
MyDate is Date
MyDate = LastDayOfMonth(2017, 2)
```
<a name="Example2"></a>

```wl
// Returns the last day of the current month
MyDate is Date
MyDate = LastDayOfMonth()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el último día del mes a partir de una fecha determinada

`<Result> = LastDayOfMonth([<Date>])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha del último día del mes especificado (en formato AAAAMMDDD).

**`<Date>: (Cadena de caracteres, variable de tipo Date o DateTime opcional)`**

Fecha para la que debe calcularse el último día del mes. Esta fecha puede corresponder a : 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  

- una cadena en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo (utilice CCC para las milésimas de segundo)). En este caso, solo se tiene en cuenta la fecha. 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.


Si no se especifica este parámetro, se utiliza la fecha actual de forma predeterminada.


<a name="SYNTAX2"></a>

### Obtener el último día del mes a partir de un año y un mes

`<Result> = LastDayOfMonth(<Year> , <Month>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha del último día del mes especificado.

**`<Year>: (Entero)`**

Año para el que debe calcularse el último día del mes. 

**`<Month>: (Entero)`**

Mes para el que debe calcularse el último día.

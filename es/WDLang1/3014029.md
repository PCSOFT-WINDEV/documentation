


## InStoredProcedureMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el código se ejecuta en un servidor HFSQL.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run on the HFSQL server?
IF InStoredProcedureMode() = True THEN
	// Stored Procedure mode
	// Process to perform
	...
ELSE
	// Different mode
	// Process to perform
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InStoredProcedureMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código actual se ejecuta en un servidor HFSQL,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000230"></a>

La función **InStoredProcedureMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InStoredProcedureMode** pueden compartirse entre varios productos sin generar errores de compilación.

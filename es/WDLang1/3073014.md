
## SysEnvironment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las variables de entorno del sistema operativo para el ordenador actual.



**Observación**: Para ver y/o modificar las variables de entorno de la computadora actual, abra el panel de control y haga clic en el "Sistema" Icon.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the environment variables of the operating system
ResEnvironment = SysEnvironment()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysEnvironment([<Environment variable>])`
---

**`<Result>: (Character string)`**



- Entorno completo del ordenador actual (o servidor) si no se especifica un &lt;Entorno Variable&gt;.. Las líneas de entorno están separadas por caracteres CR.

- Línea de ambiente cuya primera palabra corresponde al parámetro &lt;Environment Variable&gt;.

- Vacíe la cadena (""") si se produce un problema.




**`<Environment variable>: (Optional character string (with quotes))`**

Variable para encontrar en el entorno del ordenador actual ("PATH" o "USERNAME" por ejemplo).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Buscar
<a name="buscar_ELTPARAGRAPHE000049"></a>

- La búsqueda en el parámetro &lt;Environment Variable&gt; es genérica. Por ejemplo, SysEnvironment("PR") devuelve el Line correspondiente al aviso (si existe en el entorno).

- Si varias líneas coinciden con la búsqueda, sólo se devuelve la primera.



<a name="NOTE0_2"></a>




### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000058"></a>

La función **SysEnvironment** devuelve el mismo resultado que el comando SET de la línea de comandos (DOS).

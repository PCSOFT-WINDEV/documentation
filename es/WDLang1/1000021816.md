
## SysNbExternalStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de espacios de almacenamiento externos encontrados en un dispositivo Android.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the path of the directory that will be used to store the application files 
// on the secondary external storage space of the application
nNB is int = SysNbExternalStorage()
IF nNb > 1 THEN
	RESULT SysDirExternalStorage(2, sseAppFile)
ELSE
	RETURN ""
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysNbExternalStorage()`
---

**`<Result>: (Integro)`**

Número de espacios de almacenamiento externos que se encuentran en el dispositivo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El mismo dispositivo puede contener varias áreas de almacenamiento externo (por ejemplo, un área corresponde a la partición "externa" de la memoria permanente y otra área corresponde a la tarjeta SDCard).

- [SysRemovableExternalStorage](../WDLang1/1000021818.md) se utiliza para averiguar si un espacio de almacenamiento corresponde a la memoria móvil (SDCard) o no.
	Antes de acceder a un directorio situado en el área de almacenamiento externo del dispositivo, se recomienda comprobar su disponibilidad con [SysStatusStorageCard](../WDLang1/1000019554.md).

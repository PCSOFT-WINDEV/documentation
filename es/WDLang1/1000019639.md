


## &lt;Stack Variable&gt;.Info (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las características de una pila: tipos de elementos y número de elementos.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
stMyClass is Structure
	MyClassProp is int
END

// Stack whose elements are instances of stMyClass
MyStack is Stack of stMyClass
// Retrieve the definition of the type of elements
MyDefinition is Definition 
MyDefinition = MyStack.Info(tiElementDefinition)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Stack name>.Info(<Información>)`
---

**`<Result>: (Variant)`**

Información solicitada.

**`<Stack name>: (Stack)`**

Nombre de la variable de tipo [apilar](../WDLang1/1514022.md) que se utilizará.

**`<Información>: (Integer constant)`**

Información por recuperar:


|   |   |
| --- | --- |
| *tiElementDefinition* | &lt;Resultado&gt; es una variable [Definition](../WDLang1/1000019540.md) y contiene el Definition del tipo de elementos.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *tiElementType* | &lt;Resultado&gt; es un entero Variable y contiene el tipo de los elementos encontrados en la pila. Este resultado es compatible con [TypeVar](../WDLang1/3013056.md). |
| *tiTotalNumber* | &lt;Resultado&gt; es un entero Variable y contiene el número total de elementos en la pila. |

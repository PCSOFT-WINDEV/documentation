
## SysNameExe( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre y la ruta completa de una aplicación.

**Observación**: Para recuperar el nombre del ejecutable actual, utilice [ExeInfo](../WDLang1/3035001.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the name and the full path of the current application
ExeNameRes = SysNameExe(SysInstance())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysNameExe(<Executable Instance>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre y ruta completa de la aplicación especificada.

**`<Executable Instance>: (Integro)`**

Instancia del ejecutable de la aplicación a utilizar. La instancia de una aplicación es devuelta por [SysInstance](../WDLang1/3073018.md).

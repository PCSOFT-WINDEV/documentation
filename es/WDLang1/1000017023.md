
## TestCheck( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe un resultado en el estado reporte de un test según el resultado de una expresión booleana.

**Observaciones**: 

- Esta función sólo puede ser usada en el código de una prueba de automatic.

- Para comprobar el resultado de una expresión booleana en todas las plataformas, utilice las funciones dbgCheckXXX. Para obtener más información, consulte [Funciones de depuración](../WDLang1/3014025.md). 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the validity of an email address
// We assume that a bEmailValid function is used to check the validity of an email address
// Call this procedure with test values that represent valid addresses, 
// via the sEmailAddress parameter
// In the status report of the test, a line will appear for each address checked 
TestCheck(bValidEmail(sEmailAddress), "Invalid address", "OK", ...
	"test about the validity of email address '" + sAddress + "': ")

// To get the invalid addresses only in the status report of test
TestCheck(bEmailValid(sEmailAddress), sAddress + ": Invalid address")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TestCheck(<Boolean expression> [, <Caption if error> [, <Caption if success> [, <Common caption>]]])`
---

**`<Result>: (booleano)`**

Resultado de &lt;expresión booleana&gt;.

**`<Boolean expression>: (booleano)`**

Expresión que devuelve un booleano (True o False).

**`<Caption if error>: (Cadena de caracteres opcional)`**

Título que se mostrará si &lt;expresión booleana&gt; es set a False.

**`<Caption if success>: (Cadena de caracteres opcional)`**

Título que se mostrará si &lt;expresión booleana&gt; es set a True.

**`<Common caption>: (Cadena de caracteres opcional)`**

Leyenda que se mostrará en todos los casos.

<a name="NOTE0_1"></a>

## Observaciones




### Mensaje mostrado en el estado reporte de las pruebas
<a name="mensaje_mostrado_estado_reporte_las_pruebas_ELTPARAGRAPHE000059"></a>

- Si el valor de la expresión booleana es <u><u><u><u>True</u></u></u></u>: 

	- si la suma de &lt;Título común>+&lt;Título en caso de éxito&gt; no está vacía, la función **TestCheck** equivale a:
			TestWriteResult( twrInfo, &lt;Capítulo común&gt;+&lt;Capítulo en caso de éxito&gt;)

	- si la suma de &lt;Capítulo común&gt;+&lt;Capítulo si éxito&gt; está vacía, la función no tiene ninguna acción.




- Si el valor de la expresión booleana es <u><u><u><u>False</u></u></u></u>: 

	- si la suma de &lt;Título común>+&lt;Título en caso de error&gt; no está vacía, la función **TestCheck** equivale a:
			TestWriteResult( twrError, &lt;Capítulo común&gt;+&lt;Capítulo si error&gt;)

	- si la suma de &lt;Título común>+&lt;Título en caso de error&gt; está vacía, la función **TestCheck** equivale a:
			TestWriteResult( twrError, &lt;Mensaje de error estándar&gt;)

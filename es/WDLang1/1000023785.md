
## GesturePointerType( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tipo de puntero utilizado en el dispositivo: dedo, ratón o stylus.

Observación: Esta función está disponible en WINDEV sólo para ser utilizada en el simulador.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nType is int = GesturePointerType()
IF nType = typePointerFinger THEN Info("Finger is the pointer being used.")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GesturePointerType([<Pointer>])`
---

**`<Result>: (Integro)`**



- Tipo de puntero utilizado,

- -1 si se produce un error. 




El tipo de puntero puede corresponder a una de las siguientes constantes:



|   |   |
| --- | --- |
| *typePointerFinger* | El puntero que se utiliza es el dedo. |
| *typePointerMouse* | El puntero que se utiliza es el ratón. |
| *typePointerStylus* | El puntero que se utiliza es el lápiz táctil. |



**`<Pointer>: (Entero opcional)`**

Puntero índice para averiguar. El índice del primer puntero es set a 1. 

- Si no se especifica este parámetro, la función devolverá la posición del primer puntero.. 

- Si el índice no corresponde a un puntero válido, la función devolverá -1.








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **GesturePointerType** debe utilizarse en los eventos relacionados con los movimientos del puntero: haga clic, mueva, presione el puntero, suelte el puntero, desplácese, etc.. Si se llama a la función **GesturePointerType** en otro evento o proceso, &lt;Resultado&gt; corresponderá al valor '-1'.

- Para obtener el índice del puntero que desencadenó la ejecución de un Event relacionado a un movimiento en la pantalla, utilice [GestureCurrentPointer](../WDLang1/1000019751.md).


## NetworkConnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Asociar una unidad a un directorio de red compartido.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Associates the letter "E:" with the network directory \\SharedDocuments\Files
ResConnect is int = NetworkConnect("E:", "\\SharedDocuments\Files")

SWITCH ResConnection 
	CASE 1
		Error("The connection is not supported")
	CASE 2
		Error("Insufficient memory.")
	CASE 3
		Error("Wrong pointer.")
	CASE 4
		Error("Wrong type of device.")
	CASE 5
		Error("Wrong user profile.")
	CASE 7
		Error("An entry is already found for this device in the user profile (already connected)")
	CASE 8
		Error("Invalid network or wrong path.")
	CASE 9
		Error("No network.")
	CASE 10
		Error("Network error.")
	CASE 11
		Error("Wrong network name.")
	CASE 12
		Error("The drive letter is invalid.")
	CASE 13
		Error("Wrong password.")
	CASE 14
		Error("Access denied.")			
	OTHER CASE	
		Info("Drive successfully created")		
END
```

## Sintaxis

`<Result> = NetworkConnect(<Drive letter> , <Network directory> [, <Options>] [, <User> [, <Password>]])`
---

**`<Result>: (Integro)`**



- 0 si se ha establecido la conexión

- un código de error (valor superior a 0).




**`<Drive letter>: (Cadena de caracteres)`**

Unidad de red a crear, con el siguiente formato: "&lt;Letra de la unidad>**:**".

**`<Network directory>: (Cadena de caracteres)`**

Directorio de red que debe ser associated con la unidad de red.

**`<Options>: (Constante opcional de tipo Integer)`**

Establece la persistencia de la conexión: 


|   |   |
| --- | --- |
| *ncOptionNone* | Sin opción de persistencia. |
| *ncOptionPersistent*<br>(valor predeterminado) | Conexión persistente (la conexión se mantiene después de reiniciar el equipo). |



**`<User>: (Cadena de caracteres opcional)`**

Nombre del usuario al que se le permite acceder al directorio especificado. Este parámetro corresponde a *Null* de Default.

**`<Password>: (Cadena de caracteres opcional)`**

Contraseña utilizada para acceder al directorio especificado. Si &lt;Usuario&gt; corresponde al nombre del usuario actual, Windows ignora la contraseña.. Este parámetro corresponde a una cadena vacía ("") de Default.

## Observaciones


### Códigos de error
<a name="codigos_error_ELTPARAGRAPHE000056"></a>

Se devuelven los siguientes códigos de error:

- 1: La conexión no es compatible.

- 2: Memoria insuficiente.

- 3: Puntero equivocado.

- 4: Tipo de dispositivo incorrecto.

- 5: Perfil de usuario incorrecto.

- 7: Ya se ha encontrado una entrada para este dispositivo en el perfil de usuario (Ya conectado).

- 8: Red inválida o ruta incorrecta.

- 9: No hay red.

- 10: Error de red.

- 11: Nombre de red incorrecto.

- 12: La letra de la unidad no es válida.

- 13: Contraseña incorrecta.

- 14: Acceso denegado.




El mensaje del código de error puede recuperarse mediante la función [ErrorInfo](../WDLang1/3013008.md) con la constante *rrMessage*.
<a name="NOTE0_2"></a>


### Uso en Windows Vista (y posterior)
<a name="uso_windows_vista_posterior_ELTPARAGRAPHE000082"></a>

En Windows Vista (y versiones posteriores), al iniciar la aplicación en modo "Administrador", es posible que se pierdan las conexiones de red de la aplicación.. Si se activa [Sistema UAC](../Editeurs/2025014.md), las unidades de red se desconectan cuando se aumentan los privilegios del usuario. Para poder utilizar las unidades, debe volver a conectarlas con **NetworkConnect**.




## &lt;Document Variable&gt;.Deshacer el recuento (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de acciones que se pueden deshacer en un [Document](../WDLang1/1000022461.md) Variable.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

// Displays the number of actions that can be undone on a document
Info(MyDocument.UndoCount())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Document>.UndoCount()`
---

**`<Result>: (Integro)`**

Número de acciones que se pueden deshacer.

**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

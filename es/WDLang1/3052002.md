
## MDIActive( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o muestra una ventana secundaria de MDI en primer plano.

**Observación**: **MDIActive** sólo puede utilizarse:

- para las ventanas definidas como "ventanas secundarias MDI" en el editor de ventanas,

- si existe una ventana MDI padre y si está abierta.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the MDI child window displayed in the foreground
ResForegroundWindow = MDIActive()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificar la ventana del niño MDI que se muestra en primer plano

`<Result> = MDIActive()`
---

**`<Result>: (Cadena de caracteres)`**



- Alias de la ventana MDI hija mostrada en primer plano,

- Vaciar la cadena ("") si no hay una ventana de niño MDI es Active.  





<a name="SYNTAX2"></a>

### Posicionando una ventana para niños MDI en el primer plano

`<Result> = MDIActive(<Window Alias>)`
---

**`<Result>: (Cadena de caracteres)`**

Alias de la ventana MDI hija mostrada en primer plano (después de la modificación).

**`<Window Alias>: (Cadena de caracteres)`**

Alias de la ventana MDI hija que se debe posicionar en primer plano. Se produce un error de WLanguage si este parámetro no corresponde a un alias de ventana válido..

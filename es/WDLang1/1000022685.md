


## docSection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docSection** se utiliza para definir las características avanzadas de una sección que se encuentra en un Document o párrafo. 

A nivel de Document: 

- el tipo **docSection** se utiliza para averiguar la información general de Document.

- el tipo **docSection** se utiliza para modificar toda la información de todas las secciones de Document.




A nivel de párrafo: 

- el tipo **docSection** permite obtener información sobre los párrafos que pertenecen a la misma sección.

- el tipo **docSection** permite modificar información de los párrafos de la misma sección.




Las características de esta sección pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the Word Processing control
MyDoc is Document <- WP_MyDoc
// Retrieve the document orientation
Info(MyDoc.Section.Orientation)
// Switch the document orientation to landscape
MyDoc.Section.Orientation = docLandscape
```




## Funciones de video
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los vídeos:



|   |   |
| --- | --- |
| [VideoCapture](../WDLang1/3079007.md) | Captura las imágenes recibidas de la cámara:<br><br>- como imagen: toma una foto.<br><br>- como video: graba un video.<br><br><br><br><br> |
| [VideoConfigure](../WDLang1/3079009.md) | Permite configurar el controlador de captura de vídeo. |
| [VideoGenerateThumbnail](../WDLang1/1000020268.md) | Genera una miniatura de la foto o el vídeo especificado. |
| [VideoListDriver](../WDLang1/3079008.md) | Enumera los controladores de captura de vídeo instalados en el ordenador actual. |
| [VideoParameter](../WDLang1/1000019694.md) | Devuelve o modifica el valor de un parámetro de la cámara del dispositivo.<br><br> |
| [VideoRequestPermission](../WDLang1/1000022355.md) | Solicita a Permission que utilice la cámara del dispositivo. |
| [VideoSetDriver](../WDLang1/3079010.md) | Conecta la "Cámara" control a un controlador específico para la captura de vídeo. |
| [VideoStartApp](../WDLang1/1000019368.md) | Inicia la aplicación de la cámara nativa del dispositivo para Record un video o para tomar una foto. |
| [VideoStartDecoding](../WDLang1/1000022163.md) | Inicia la descodificación de códigos de barras en una control Cámara. |
| [VideoStopDecoding](../WDLang1/1000022164.md) | Detiene la decodificación de un código de barras en una control Cámara. |





<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


### Observaciones: 
<a name="observaciones_ELTPARAGRAPHE000036"></a>

- Las funciones de Videoxxx y Cameraxxx son idénticas. 

- [dSaveImageJPEG](../WDLang1/3029038.md) guarda la imagen de un control Cámara:

	- en un archivo JPEG.

	- en la memoria.




- [dSaveImageBMP](../WDLang1/3029026.md) guarda la imagen de un control Cámara:

	- en un archivo BMP.

	- en la memoria.


## Gestión de los ejecutables (funciones WLanguage)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000071"></a>
WINDEV, WEBDEV y WINDEV Mobile proponen diferentes funciones para gestionar los ejecutables desde una aplicación WINDEV, desde una aplicación WINDEV Mobile o desde un sitio WEBDEV. Estos ejecutables pueden ser creados por WINDEV o por otra aplicación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Funciones para el manejo de los ejecutables
<a name="funciones_para_manejo_los_ejecutables_ELTTEXTE000095"></a>
Las siguientes funciones se utilizan para manipular los ejecutables y para get la información sobre los ejecutables que se encuentran en el ordenador actual:



|   |   |
| --- | --- |
| [AppOpenForm](../WDLang1/1000020980.md) | inaugura: <br><br>- el formulario de una aplicación en Android (Play Store), iOS (App Store) o Windows (Windows Store) para que el usuario le dé una marca y/o a comment la.<br><br>- el formulario para configurar las compras In-App de una aplicación. <br><br><br> |
| [AutoRunAdd](../WDLang1/3019002.md) | Permite iniciar automáticamente una aplicación WINDEV cuando se produce una Event específica. |
| [AutoRunDelete](../WDLang1/3019001.md) | Detiene la ejecución de automatic de una aplicación WINDEV. |
| [ExeGetPID](../WDLang1/1000017239.md) | Devuelve información sobre el actual Process. |
| [ExeInfo](../WDLang1/3035001.md) | Recupera la información especificada sobre la versión de un ejecutable o DLL. |
| [ExeListDLL](../WDLang1/3035009.md) | Devuelve la lista de bibliotecas (archivos".DLL") utilizadas por una de las aplicaciones que se están ejecutando actualmente.. |
| [ExeListProcess](../WDLang1/3035008.md) | Devuelve la lista de aplicaciones que se están ejecutando actualmente. |
| [ExePriority](../WDLang1/3035010.md) | Devuelve o modifica la prioridad de una aplicación actualmente en ejecución. |
| [ExeRemoteRun](../WDLang1/1000018695.md) | Inicia la ejecución de un programa remoto desde la aplicación actual. |
| [ExeRun](../WDLang1/3035006.md) | Inicia la ejecución de un programa (un ejecutable por ejemplo) desde la aplicación actual. |
| [ExeRunning](../WDLang1/3035003.md) | Le permite saber si ya se está ejecutando una aplicación de 32 bits. |
| [ExeTerminate](../WDLang1/3035004.md) | Finaliza la ejecución de una aplicación de 32 bits en el equipo actual.: todas las instancias del ejecutable son asesinadas. |
| [RestartProgram](../WDLang1/1000019900.md) | Finaliza la aplicación y la reinicia. |
| [ShellExecute](../WDLang1/3035007.md) | Abre un Document en su aplicación associated. |
| [StartShare](../WDLang1/1000020620.md) | Abre una ventana emergente que permite seleccionar la aplicación que se utilizará para compartir una Resource (texto o Document). |





También se pueden utilizar otras funciones: 



|   |   |
| --- | --- |
| [GetIdentifier](../WDLang1/3013028.md) | Calcula un identificador único (entero) para un ejecutable o para una sesión. |
| [ListDLL](../WDLang1/3064006.md) | Devuelve:<br><br>- **en tiempo de ejecución**: la lista de bibliotecas (archivos".DLL") utilizadas por la aplicación WINDEV o WINDEV Mobile actual o por el sitio WEBDEV actual. Sólo se listan las bibliotecas cargadas en memoria.<br><br>- **en modo de prueba**: todas las bibliotecas instaladas en el ordenador actual.<br><br><br> |

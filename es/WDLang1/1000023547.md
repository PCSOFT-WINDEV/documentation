
## SpreadsheetListImage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las imágenes que se encuentran en la hoja de trabajo actual de un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all images
Id is string 
FOR EACH Id OF SpreadsheetListImage(PSHEET_Spreadsheet) 
	// Delete the image 
	SpreadsheetDeleteImage(PSHEET_Spreadsheet, Id) 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetListImage(<Spreadsheet control>)`
---

**`<Result>: (Array)`**

Array de strings de caracteres que contienen los identificadores de las imágenes que se encuentran en el control Hoja de cálculo. Este identificador puede utilizarse en [Hoja de cálculoBorrarImagen](../WDLang1/1000023546.md). 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


## Sin( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el seno de un ángulo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sine of 30
ResSine = Sin(30)
// Returns 0.5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Sin(<Angle>)`
---

**`<Result>: (Real)`**

Seno del ángulo especificado.

**`<Angle>: (Entero o real)`**

Ángulo (en grados) de uso.

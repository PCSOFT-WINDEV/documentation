


## Funciones de gestión de controles Casilla de verificación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles de casillas de verificación:



|   |   |
| --- | --- |
| [CheckBoxAdd](../WDLang1/1000020595.md) | Añade una nueva opción a una control Casilla de verificación. |
| [CheckBoxBorrar](../WDLang1/1000020596.md) | Elimina una opción de un control Casilla de verificación. |
| [CheckBoxCount](../WDLang1/1000021274.md) | Devuelve el número de opciones encontradas en una control Casilla de verificación. |
| [CheckBoxInsert](../WDLang1/1000020597.md) | Inserta una nueva opción en una control Casilla de verificación. |

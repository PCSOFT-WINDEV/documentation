
## eRoundDifference( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la diferencia de redondeo entre:

- el valor mostrado en un control "Moneda + Euro",

- el associated control utilizado para la doble visualización.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the rounding difference between the "EDT_PriceFranc" control
// that displays the values in francs and the "EDT_PriceEuro" control that displays
// the same values converted into Euros
ResRoundingDifference = eRoundDifference(EDT_PriceFranc)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = eRoundDifference(<Control used>)`
---

**`<Result>: (Moneda)`**



- La diferencia de redondeo entre el control especificado y el associated control usado para la doble visualización. 
	Esta diferencia se expresa en la moneda con el valor de moneda más bajo.. Este valor de moneda depende del tipo de cambio y del número de decimales.

- 0 si no se encuentra una diferencia de redondeo o si el control especificado no tiene associated control para la doble indicación.




**`<Control used>: (Nombre del control)`**

Nombre de la columna "Moneda + Euro" o control Campo de entrada.

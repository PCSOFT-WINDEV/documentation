


## ThreadRequestStop (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía una solicitud para detener un hilo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ThreadExecute("T1", threadNormal, ThreadProcedure)
ThreadRequestStop ("T1")
```
<a name="Example2"></a>

```wl
// Build a thread
t is Thread(Proc, ("First parameter", 2))
// Trigger the execution of the thread
ThreadExecute(t)
...
ThreadRequestStop(t)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Solicitar que se detenga un hilo identificado por su nombre

`ThreadRequestStop(<Thread name>)`
---

**`<Thread name>: (Cadena de caracteres)`**

Nombre del hilo a detener. Este nombre se asigna al ejecutar el hilo (con la función [ThreadExecute](../WDLang1/3077024.md)). 


<a name="SYNTAX2"></a>

### Solicitar que se detenga un hilo identificado por una variable de tipo Thread

`ThreadRequestStop(<Thread>)`
---

**`<Thread>: (Variable de tipo Thread)`**

Nombre de la variable de tipo [Thread](../WDLang1/1000024879.md) que corresponde al hilo que se detendrá. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El hilo no se detiene automáticamente. El código del hilo debe verificar si se solicitó la interrupción con la función [ThreadStopRequested](../WDLang1/1000021219.md), y finalizar correctamente.

- Es una función no bloqueante y no espera a que el hilo se detenga. Se debe utilizar la función [ThreadWait](../WDLang1/3077020.md) para esperar a que el hilo se detenga.

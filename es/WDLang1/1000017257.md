


## MenuClone (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Clona un menú o una opción de menú así como el código de associated.

Crea un nuevo menú o una nueva opción de menú. Este nuevo menú (u opción de menú) es una copy de un menú existente. El nuevo menú o la nueva opción de menú se crea necesariamente en el mismo menú que el elemento fuente.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Clone a menu option
MenuClone("MNU_Schedule", "MNU_Schedule2008", "Schedule 2008")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un menú clonado o una opción clonada en la posición especificada

`<Result> = MenuClone(<Source element> , <Cloned element> , <Text of cloned element> [, <Insertion position>])`
---

**`<Result>: (Integro)`**

Posición del menú clonado o posición de la opción de menú clonado. Si no se realiza la clonación, se produce un error fatal.

**`<Source element>: (Nombre del menú o nombre de la opción)`**

Nombre del menú o de la opción de menú de copy.

**`<Cloned element>: (Cadena de caracteres)`**

Nombre del menú clonado o de la opción de menú. Este nombre se utilizará para manipular el elemento clonado a través de la programación.

**`<Text of cloned element>: (Cadena de caracteres)`**

Pie de foto del menú clonado o de la opción de menú. Este subtítulo permitirá al usuario manipular el elemento.

**`<Insertion position>: (Entero o constante de tipo Integer opcional)`**

Posición donde se insertará el menú clonado o la opción clonada. Este parámetro puede corresponder a:

- un entero: en este caso, el menú clonado o la opción de menú clonado se insertará antes de la posición especificada.

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *menuInsertBeforeSeparator* <br>(Valor predeterminado) | Añade el elemento clonado antes del próximo separador. |
| *menuInsertEnd* | Añade el elemento clonado al final del menú |







<a name="SYNTAX2"></a>

### Insertar un menú clonado o una opción clonada en la posición relativa

`<Result> = MenuClone(<Source element> , <Cloned element> , <Text of cloned element> [, <Insertion element>])`
---

**`<Result>: (Integro)`**

Posición del menú clonado o posición de la opción de menú clonado. Si no se realiza la clonación, se produce un error fatal.

**`<Source element>: (Nombre del menú o nombre de la opción)`**

Nombre del menú o de la opción de menú de copy.

**`<Cloned element>: (Cadena de caracteres)`**

Nombre del menú clonado o de la opción de menú. Este nombre se utilizará para manipular el elemento mediante programación. Se produce un error fatal si este nombre corresponde a un elemento existente.

**`<Text of cloned element>: (Cadena de caracteres)`**

Pie de foto del menú clonado o de la opción de menú. Este subtítulo permitirá al usuario manipular el elemento.

**`<Insertion element>: (Cadena de caracteres opcional)`**

Nombre del elemento ante el que debe insertarse el elemento clonado. Si no se especifica este nombre, el elemento clonado se insertará antes del siguiente separador de menú.

<a name="NOTE0_1"></a>

## Observaciones


### Características del elemento clonado
<a name="caracteristicas_del_elemento_clonado_ELTPARAGRAPHE000108"></a>

El elemento clonado comparte los parámetros iniciales con el elemento fuente:

- su código

- sus atajos




El elemento clonado no comparte con el elemento fuente:

- los parámetros modificados después de su creación

- el nombre

- La opción de menú clonado o el menú clonado se hará automáticamente Visible.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No se puede clonar una opción de menú creada mediante programación.

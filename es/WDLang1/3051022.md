
## MCIRetrieve( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el último resultado del comando MCI ejecutado por [MCIEjecutar](../WDLang1/3051005.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Last result of MCIExecute
ResRetrieve = MCIRetrieve()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIRetrieve()`
---

**`<Result>: (Cadena de caracteres)`**



- Resultado devuelto por [MCIEjecutar](../WDLang1/3051005.md),

- Cadena vacía ("") si [MCIEjecutar](../WDLang1/3051005.md) no devuelve ningún resultado.  






<a name="XComponent"></a>

## Componente:
wd280std.dll




## &lt;Variable Image&gt;.ModifySaturation (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la saturación de una imagen en un control Imagen.![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the saturation of the image found in the IMG_Image1 control
IMG_Image1.ModifySaturation(50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Image variable>.ModifySaturation(<Variation of saturation> [, <Destination image>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image variable>: (Image variable)`**

Nombre del control Imagen a manipular.

**`<Variation of saturation>: (Entero)`**

Valor comprendido entre -100 y +100. Este valor se agregará a la saturación actual de cada pixel para obtener una nueva saturación.

**`<Destination image>: (Nombre de control o variable de tipo Image)`**

Imagen en la que se dibujará la imagen transformada. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).
	




Si no se especifica este parámetro, la transformación se realizará en &lt;Control Imagen&gt;

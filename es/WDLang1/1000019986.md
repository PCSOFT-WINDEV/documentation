
## Funciones para gestionar los controles del organigrama (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del organigrama (mediante una sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;Organigrama&gt;.add](../WDLang1/1000019975.md) | Añade un nuevo elemento raíz a una control Organigrama. |
| [&lt;Organigrama&gt;.AddChild](../WDLang1/1000019976.md) | Añade un elemento hijo a una control Organigrama. |
| [.Delete](../WDLang1/1000019984.md) | Elimina un elemento y los "hijos" de un control Organigrama. |
| [&lt;Organigrama&gt;.DeleteAll](../WDLang1/1000019985.md) | Borra todos los elementos de un control Organigrama. |
| [&lt;Organigrama de la organización&gt;.display](../WDLang1/1000019974.md) | Reinicializa el contenido de una control Organigrama a partir de su fuente de datos (archivo de datos HFSQL por ejemplo). |
| [&lt;Organigrama&gt;.InfoXY](../WDLang1/1000019977.md) | Devuelve la índice del elemento que se encuentra en una posición dada en un control Organigrama. |
| [OrgModify](../WDLang1/1000019978.md) | Modifica un elemento en una control Organigrama. |
| [&lt;Gráfico de organización&gt;.Occurrence](../WDLang1/1000019979.md) | Devuelve el número de elementos que se encuentran en una control Organigrama. |
| [&lt;Organigrama&gt;.SaveImage](../WDLang1/1000019980.md) | Crea una Image que representa a toda la control Organigrama. |
| [&lt;Organigrama&gt;.Select](../WDLang1/1000019981.md) | Devuelve el índice del elemento seleccionado en una control Organigrama. |
| [.SelectMinus](../WDLang1/1000019982.md) | Desmarca un elemento de un control Organigrama. |
| [.SelectPlus](../WDLang1/1000019983.md) | Selecciona un elemento en una control Organigrama. |

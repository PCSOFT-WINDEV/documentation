


## NationToSubLanguage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el sub-lenguaje Default correspondiente a una nación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res = NationToSubLanguage(nationAustralian)
// Res = 3
```

<a name="XSYNTAX"></a>

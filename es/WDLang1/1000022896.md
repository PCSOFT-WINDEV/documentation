


## &lt;Organizador&gt;.DisplayNextPeriodo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve el área de tiempo que se muestra a la siguiente área (por ejemplo, las próximas semanas) en un control Agenda. El control Agenda se visualiza a partir del día correspondiente al área de tiempo siguiente. 

**Observaciones**: 

- Las funciones **&lt;Organizador&gt;.DisplayNextPeriodo** y [&lt;Organizador&gt;.DisplayPeríodo anterior](../WDLang1/1000022878.md) permiten establecer fácilmente los botones "Adelante/Atrás". Estos controles le permiten desplazarse por las áreas de tiempo que muestra la pantalla de control Agenda. 

- El periodo mostrado también puede ser modificado por [&lt;Organizador&gt;.ChangeMode](../WDLang1/1000022879.md) y [&lt;Posición](../WDLang1/1000022897.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to monthly display
ORG_MyOrganizer.ChangeMode(orgzMonth)
// Display the current month
ORG_MyOrganizer.Position(DateSys())
// Switch to the next month
ORG_MyOrganizer.DisplayNextPeriod()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Organizer control>.DisplayNextPeriod()`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

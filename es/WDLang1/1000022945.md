
## docProperty (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docProperty** se utiliza para definir las propiedades de un procesador de textos Document. Las características de estas propiedades pueden ser definidas y modificadas por varias propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no debe utilizarse directamente.** Sólo debe utilizarse a través de las variables [Document](../WDLang1/1000022461.md). 

Observación: Las propiedades del Document se usan para algunos controles calculados que pueden ser incluidos en el Document.






<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docProperty
<a name="propiedades_especificas_descripcion_las_variables_docproperty_ELTPARAGRAPHE000031"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docProperty**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Autor | Cadena de caracteres | Autor de Document. |
| Categoría | Cadena de caracteres | Categoría de Document. |
| comment | Cadena de caracteres | comment asociada a Document. |
| ContenidoEstado | Cadena de caracteres | Estado de Document: borrador, revisado, .... |
| DateCreated | Fecha y hora | Fecha y hora de la creación de Document. <br><br>**Esta propiedad es de solo lectura.**<br><br>Esta propiedad corresponde a "CreationDate" en las versiones anteriores. |
| Empresa | Cadena de caracteres | La compañía que creó la Document. <br><br> |
| LastModificationDate | Fecha y hora | Fecha y hora de la última modificación de Document. <br><br>**Esta propiedad es de solo lectura.** |
| Palabra clave | Cadena de caracteres | Palabra clave asociada a Document. |
| Revisión | Cadena de caracteres | Número de revisiones de Document. <br><br>**Esta propiedad es de solo lectura.** |
| Tema | Cadena de caracteres | Asunto del documento. |
| Título | Cadena de caracteres | Título de Document. |

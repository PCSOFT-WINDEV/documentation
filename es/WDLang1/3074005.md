
## FileToMemoryTable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Rellena un control Tabla programáticamente con los registros de un archivo de datos, vista HFSQL o consulta (consulta creada en el editor de consultas o con [HExecuteSQLQuery](../WDLang4/3044084.md)).

La control Tabla que se rellena mediante programación debe contener las columnas que se van a vincular a los artículos. La función **FileToMemoryTable** va a: 

- vaciar el control Tabla. 

- vincular las columnas de control Tabla a los elementos de un archivo de datos, vista o consulta HFSQL. Los elementos se vincularán según el orden lógico definido en el archivo. 

- rellenar el control Tabla con el contenido del archivo de datos o consulta.




**ATENCIÓN**: Esta operación puede tardar bastante tiempo y saturar la memoria si el archivo de datos es muy grande.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Populate TABLE_MyTable with the data from QRY_Query1
FileToMemoryTable(TABLE_MyTable, QRY_Query1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`FileToMemoryTable(<Table control> , <Data file>)`
---

**`<Table control>: (Nombre del control)`**

Nombre de la control Tabla a rellenar.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo de datos, vista HFSQL o consulta utilizada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Tipos de consultas
<a name="tipos_consultas_ELTPARAGRAPHE000057"></a>

La función **FileToMemoryTable** puede utilizarse con las consultas creadas en el editor de consultas o con la función [HExecuteSQLQuery](../WDLang4/3044084.md).

Para asociar una consulta creada con la función [SQLExec](../WDLang4/3072007.md) a un control Tabla rellenado mediante programación, utilice la función [SQLTable](../WDLang4/3072031.md).
<a name="NOTE0_2"></a>




### Navegación del archivo de datos
<a name="navegacion_del_archivo_datos_ELTPARAGRAPHE000078"></a>

El archivo de datos se navega de acuerdo con el contenido de la propiedad [BrowsedItem](../Proprietes/2510111.md).

Para que la función **FileToMemoryTable** tenga en cuenta el filtro aplicado al archivo de datos, es necesario:

1. Modificar la propiedad [BrowsedItem](../Proprietes/2510111.md) asignándole el elemento devuelto por la función [HFilter](../WDLang4/3044100.md).

2. Utilizar la función **FileToMemoryTable**.



<a name="NOTE0_3"></a>




### Función FileToMemoryTable y Nueva Property
<a name="funcion_docparampagetitleshort_nueva_property_ELTPARAGRAPHE000104"></a>

La función **FileToMemoryTable** asigna la propiedad [nuevo](../Proprietes/2510073.md) a <u><u><u><u>True</u></u></u></u>.

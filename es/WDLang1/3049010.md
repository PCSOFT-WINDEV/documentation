


## ListSelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos seleccionados en un cuadro de lista, ListView o control Combo Box.

**Recordatorio**: El elemento seleccionado corresponde al elemento actual en el que se visualiza el banda de selección.

**Observación**: La función **ListSelectCount** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements selected in the "LIST_CustomerList" control
ResSelectedElements = ListSelectCount(LIST_CustomerList)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ListSelectCount(<List Box control>)`
---

**`<Result>: (Integro)`**

Número de elementos seleccionados en el Cuadro de Lista especificado (o Combo Box) control. Este parámetro:

- puede ser mayor que 1 sólo si la control List Box es multiselección,

- es igual a 1 si el control Combo Box no es editable o si el control Combo Box es editable y si se selecciona un elemento,

- es igual a 0 si el control Combo Box es editable y no hay ningún elemento seleccionado.




**`<List Box control>: (Nombre del control)`**

Nombre de la casilla de la lista (o Combo Box) control a utilizar.
Si este parámetro corresponde a una cadena vacía (""), se devuelve el número de elementos seleccionados en el control List Box (o Combo Box) al que pertenece el Event actual.

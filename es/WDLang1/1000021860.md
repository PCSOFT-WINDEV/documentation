
## gImageExists( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si un Image fue cargado en la memoria. 

**Recordatorio**: [LoadImage](../WDLang1/3029022.md) o [CargarImagenSheet](../WDLang1/1000018826.md) se utilizan para cargar imágenes en memoria. Estas imágenes pueden ser numeradas del 1 al 9. **gImageExists** se utiliza para determinar si Image N se ha cargado con éxito en la memoria. Entonces, esta Image puede ser utilizada por [gImage](../WDLang1/3029042.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Loads the image #3 if it was not loaded yet
IF NOT gImageExists(3) THEN
	LoadImage(3, "Logo.png")
END
STC_TITLE = "Our logo: " = gImage(3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = gImageExists(<Image identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la imagen se cargó en memoria. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image identifier>: (Integro)`**

Identificador de Image buscado en la memoria (incluido entre 1 y 9). Este identificador es devuelto por [LoadImage](../WDLang1/3029022.md) o [CargarImagenSheet](../WDLang1/1000018826.md).

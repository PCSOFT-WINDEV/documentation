


## Funciones de gestión de servicios
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los servicios:



|   |   |
| --- | --- |
| [EndService](../WDLang1/1000018711.md) | Finaliza la ejecución del servicio actual. |
| [ServiceContinue](../WDLang1/1000017105.md) | Reinicia un servicio pausado. |
| [ServiceControl](../WDLang1/1000017108.md) | Envía un mensaje control a un servicio. |
| [ServiceExist](../WDLang1/1000018936.md) | Comprueba si un servicio está instalado. |
| [ServiceInfo](../WDLang1/1000017103.md) | Devuelve información sobre la configuración de un servicio. |
| [ServiceInstall](../WDLang1/1000017099.md) | Instala un servicio de Windows o una daemon de Linux de acuerdo con la información dada en los miembros del [Servicio Variable](../WDLang1/1000017118.md). |
| [ServiceList](../WDLang1/1000017104.md) | Devuelve la lista de servicios encontrados en el ordenador actual o en el ordenador remoto especificado. |
| [ServiceModify](../WDLang1/1000017100.md) | Modifica la configuración de un servicio de acuerdo con la información dada en los miembros del [Servicio Variable](../WDLang1/1000017118.md). |
| [ServicePause](../WDLang1/1000017109.md) | Pone en pausa un servicio. |
| [ServiceRefresh](../WDLang1/1000017107.md) | Solicita a un servicio que relea la información de configuración. |
| [ServiceRestart](../WDLang1/1000017112.md) | Reinicia un servicio (ya iniciado). |
| [ServiceStart](../WDLang1/1000017110.md) | Inicia un servicio. |
| [ServiceStatus](../WDLang1/1000017102.md) | Devuelve el status actual de un servicio. |
| [ServiceStop](../WDLang1/1000017111.md) | Detiene un servicio. |
| [ServiceUninstall](../WDLang1/1000017098.md) | Desinstala un servicio. |
| [ServiceWait](../WDLang1/1000018712.md) | Pausa el servicio actual durante la duración especificada. |
| [ServiceWaitStatus](../WDLang1/1000017101.md) | Espera a que un servicio se encuentre en un estado específico. |
| [ServiceWriteEventLog](../WDLang1/1000018807.md) | Escribe un Event en el registro de los Event s de Windows. |




## Funciones de búsqueda de la ventana control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000076"></a>
WINDEV permite al usuario realizar una búsqueda entre todos los textos encontrados en una ventana o en un control. 

El cálculo se puede realizar: 

- a través de la aplicación automatic para buscar en una ventana. Para obtener más información, consulte [Buscar en una ventana](../Editeurs/9500101.md). 

- a través de las funciones de búsqueda de WLanguage. 




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Funciones para realizar una búsqueda en los controles
<a name="funciones_para_realizar_una_busqueda_los_controles_ELTTEXTE000100"></a>
Las siguientes funciones se utilizan para buscar un texto en los diferentes controles de ventana: 



|   |   |
| --- | --- |
| [ControlFindHighlight](../WDLang1/1000020535.md) | Destaca, en una ventana o control, todas las leyendas y textos correspondientes a un texto dado. |
| [ControlFindTextFirst](../WDLang1/1000020763.md) | Encuentra y selecciona el primer elemento correspondiente al texto buscado en una ventana o control. |
| [ControlFindTextNext](../WDLang1/1000020764.md) | Encuentra y selecciona el siguiente elemento correspondiente al texto buscado en una ventana o en una control. |
| [ControlFindTextPrevious](../WDLang1/1000020765.md) | Encuentra y selecciona el elemento anterior correspondiente al texto buscado en una ventana o control. |

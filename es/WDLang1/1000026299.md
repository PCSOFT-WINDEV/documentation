


## &lt;Editor de HTML&gt;.Insert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta texto o código HTML en la posición del cursor en un editor HTML control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert today's date in bold
HTMEDT_MyHTMLEditor.Insert("'''" + DateToString(DateSys()) + "'''")

// Insert a smiley
HTMEDT_MyHTMLEditor.Insert("<p style=""font-size:64px"">&#128512;</p>")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Editor control>.Insert(<HTML to insert>)`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control.

**`<HTML to insert>: (Cadena de caracteres)`**

Texto o código HTML para insertar en la posición del cursor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se selecciona algún elemento en el Editor HTML control, esta selección se elimina. 

- El usuario puede deshacer la inserción a través de la opción Deshacer o pulsando Ctrl + Z. 

- La posición de inserción actual es devuelta por las propiedades [Selección](../Proprietes/1000023033.md) o [Cursor](../Proprietes/2510025.md).

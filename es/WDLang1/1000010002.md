


## Funciones de gestión de controles Tabla TreeView
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el TreeView controles Tabla:



|   |   |
| --- | --- |
| [TablaRefreshVisible](../WDLang1/1000021196.md) | Modifica o devuelve el estado de visibilidad de la barra actualización en una Tabla o control Tabla TreeView. |
| [TablaRestaurarPosición YSelección](../WDLang1/1000017413.md) | Restaura la posición de la barra de desplazamiento en una Tabla o control Tabla TreeView, así como los elementos seleccionados. |
| [TablaToWord](../WDLang1/3074036.md) | Crea un archivo Word (.RTF) a partir de los datos encontrados en una Tabla o control Tabla TreeView. |
| [TableAdd](../WDLang1/3074017.md) | Agrega una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView, <br><br>- un control Combo Box "con tabla" (control Tabla en un control Combo Box).<br><br><br> |
| [TableAddChild](../WDLang1/1000010004.md) | Añade una fila a una control Tabla TreeView, al final de un nivel jerárquico determinado. |
| [TableAddLine](../WDLang1/3074006.md) | Agrega una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView. <br><br><br> |
| [TableAdjust](../WDLang1/3074004.md) | Ajusta el título y el tamaño de las columnas que se encuentran en una Tabla o control Tabla TreeView. |
| [TableChildCount](../WDLang1/1000020504.md) | Devuelve el número de hijos directos de un elemento de una control Tabla TreeView. |
| [TableCollapse](../WDLang1/1000010008.md) | Colapsos:<br><br>- una sucursal que fue ampliada previamente en un control Tabla TreeView. <br><br>- una ruptura que se expandió previamente en un control Tabla. <br><br>- una ventana detallada que se muestra para un fila.<br><br><br> |
| [TableCollapseAll](../WDLang1/1000010003.md) | Colapsos:<br><br>- la jerarquía completa (todos los nodos) de un control Tabla TreeView.<br><br>- todas las roturas de un control Tabla. <br><br><br> |
| [TableColumnIndex](../WDLang1/3074016.md) | Devuelve el índice de una columna en una Tabla o Treeview control Tabla. |
| [TableCount](../WDLang1/3074021.md) | Devuelve: <br><br>- el número de columnas que se encuentran en una tabla o control Tabla TreeView.<br><br>- el número de filas que se encuentran en una tabla o control Tabla TreeView. <br><br><br> |
| [TableDelete](../WDLang1/3074032.md) | Borra un fila de: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView, <br><br>- una tabla que se muestra en un control Combo Box.<br><br><br> |
| [TableDeleteAll](../WDLang1/3074034.md) | Borra todas las líneas de: <br><br>- control Tabla rellenado mediante programación, <br><br>- a control Tabla TreeView rellenado programáticamente, <br><br>- una tabla que se muestra en un control Combo Box.<br><br><br> |
| [TableDeleteChild](../WDLang1/1000010006.md) | Borra todas las líneas secundarias de una rama de la jerarquía.. |
| [TableDeleteSelect](../WDLang1/1000020506.md) | Borra las filas seleccionadas de una tabla o vista en árbol control Tabla. |
| [TableDisplay](../WDLang1/3074003.md) | Actualiza la visualización de un control Tabla o Tabla TeeView a partir de una posición determinada:<br><br>- para un control relacionado a un archivo: los cambios realizados en el archivo de datos relacionado se reflejan en el control.<br><br>- para un control manejado programáticamente: las columnas calculadas se vuelven a calcular.<br><br><br> |
| [TableEnumColumn](../WDLang1/3074012.md) | Devuelve el nombre de una columna que se encuentra en una tabla o en una tabla de Treeview control Tabla. |
| [TableExpandAll](../WDLang1/1000010001.md) | Se expande:<br><br>- la jerarquía completa (todos los nodos) de un control Tabla TreeView.<br><br>- todas las roturas de un control Tabla. <br><br><br> |
| [TableFilteredColumn](../WDLang1/1000020517.md) | Devuelve la lista de columnas filtradas por el usuario en una Tabla o control Tabla TreeView.. |
| [TableFixColumn](../WDLang1/3074014.md) | Fija una columna a la izquierda o libera una columna fijada a la izquierda. |
| [TableGiveChild](../WDLang1/1000010009.md) | Devuelve los "hijos" de un elemento (nivel inferior) en una control Tabla TreeView. |
| [TableGiveParent](../WDLang1/1000010011.md) | Devuelve el "padre" de un elemento en una control Tabla TreeView. |
| [TableInfoXY](../WDLang1/3074007.md) | Vuelve para una posición dada en una Tabla o control Tabla TreeView (coordenadas de un punto control):<br><br>- el nombre de la columna mostrada para la posición especificada.<br><br>- el índice de la fila o columna para la posición especificada.<br><br><br> |
| [TableInputInProgress](../WDLang1/3074023.md) | Indica si una Tabla o control Tabla TreeView está actualmente en edición. |
| [TableInsert](../WDLang1/3074025.md) | Inserta una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView,<br><br>- una tabla que se muestra en un control Combo Box.<br><br><br> |
| [TableInsertChild](../WDLang1/1000010005.md) | Inserta una fila en un control Tabla TreeView, en un nivel jerárquico determinado. |
| [TableInsertLine](../WDLang1/3074026.md) | Inserta una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView.<br><br><br> |
| [TableItemStatus](../WDLang1/1000010012.md) | Devuelve el estado de un nodo en un control Tabla de Treeview.: colapsado, expandido o no encontrado. |
| [TableListChild](../WDLang1/1000021787.md) | Enumera los "hijos" de un nodo y los elementos "hijo" de estos "hijos" en un control Tabla TreeView. |
| [TableModify](../WDLang1/3074019.md) | Modifica una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView, <br><br>- una tabla que se muestra en un control Combo Box.<br><br><br> |
| [TableModifyLine](../WDLang1/3074028.md) | Modifica una fila en: <br><br>- un control Tabla, <br><br>- un control Tabla TreeView. <br><br><br> |
| [TableMoveBranch](../WDLang1/1000020611.md) | Mueve el contenido de una fila así como toda su estructura de árbol descendente en una control Tabla TreeView. |
| [TableMoveColumn](../WDLang1/3074011.md) | Mueve una columna en una Tabla o control Tabla TreeView. |
| [TablePosition](../WDLang1/3074022.md) | Muestra un control Tabla o Tabla TreeView a partir de una fila específica o devuelve el índice de la primera fila que se muestra en la Tabla o Tabla TreeView. |
| [TableSavePositionAndSelection](../WDLang1/1000017412.md) | Ahorra: <br><br>- la posición de la barra de desplazamiento en una tabla o control Tabla TreeView, <br><br>- los elementos seleccionados en el control.<br><br><br> |
| [TableSearchChild](../WDLang1/1000017064.md) | Busca un valor en una columna de un control Tabla TreeView, en una rama especificada. |
| [TableSelect](../WDLang1/3074020.md) | Devuelve el índice del elemento seleccionado en la Tabla o control Tabla TreeView. |
| [TableSelectCount](../WDLang1/3074030.md) | Devuelve el número de elementos seleccionados en una Tabla o control Tabla TreeView (especialmente los controles de selección múltiple). |
| [TableSelectMinus](../WDLang1/3074029.md) | Desmarca un fila de una tabla o de un control Tabla TreeView. |
| [TableSelectPlus](../WDLang1/3074031.md) | Selecciona una fila en una tabla o control Tabla TreeView. |
| [TableSortChild](../WDLang1/1000017025.md) | Ordena una rama de una control Tabla TreeView en una o más columnas. |
| [TableSwapBranch](../WDLang1/1000020505.md) | Intercambia el contenido de dos filas (así como su respectiva estructura de árbol) en un control Tabla TreeView. |
| [TableSwapLine](../WDLang1/1000020627.md) | canjes: <br><br>- dos filas en una control Tabla rellenada programáticamente. <br><br>- dos ramas en un Treeview control Tabla rellenado programáticamente.<br><br><br> |
| [TableToClipboard](../WDLang1/1000017062.md) | Copia el contenido de una Tabla o control Tabla TreeView al portapapeles. |
| [TableToExcel](../WDLang1/3074033.md) | Crea un archivo Excel con los datos de una Tabla o control Tabla TreeView. |
| [TableToText](../WDLang1/1000017061.md) | Crea un string a partir de los datos encontrados en una Tabla o control Tabla TreeView. |
| [TableToXML](../WDLang1/3074037.md) | Crea un archivo XML a partir de los datos encontrados en una Tabla o control Tabla TreeView. |
| [TableTypeItem](../WDLang1/1000010010.md) | Comprueba la existencia de un elemento en una tabla TreeView y devuelve su tipo (nodo u hoja) si el elemento existe. |




## <Window>.CurrentLayout (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del diseño actual en la ventana o ventana interna.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF WIN_Details.CurrentLayout() = 2 THEN
	STC_DETAILS = CUSTOMER.DETAILS
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX2"></a>

## Sintaxis

### Obtener la disposición de una ventana específica

`<Result> = WinCurrentLayout(<Window used>)`
---

**`<Result>: (Integro)`**



- Índice del layout actual.

- 0 si la ventana interna no incluye ninguna disposición. 




**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna utilizada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los diseños se numeran en el orden en que se crearon.

- El layout cambia si la orientación del dispositivo cambia o si se ejecuta la función [.ChangeLayout](../WDLang1/1410088842.md).

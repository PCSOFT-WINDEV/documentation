
## InfoVersion (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **InfoVersion** permite conocer las características de una versión específica de una aplicación. Las características de esta versión son devueltas por varias propiedades de WLanguage.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sRes is string = AppliListVersion()
IV is InfoVersion

sDisplay is string = [
Number = %1
Date = %2
Size = %3 bytes
Comment = %4
Enabled = %5
Forced = %6
Forbidden = %7
]

FOR EACH STRING MyString OF sRes SEPARATED BY CR
	IV = AppVersionInfo(MyString)
	Trace(StringBuild(sDisplay, ...
		IV.VersionNumber, IV.Date, ...
		IV.Size, IV.Comment, ...
		IV.Enabled, IV.Forced, IV.Forbidden))
	Trace("===============================================")
END
```

## Observaciones


### Propiedades específicas de las variables InfoVersion
<a name="propiedades_especificas_las_variables_infoversion_ELTPARAGRAPHE000040"></a>

Las siguientes propiedades pueden utilizarse para manipular una variable de tipo **InfoVersion**:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Habilitado | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la versión está habilitada (es decir, si es visible para los equipos cliente),<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>**Esta propiedad es de solo lectura**.<br><br> |
| comment | Cadena de caracteres | Información sobre las novedades de la versión. Esta información fue especificada por el desarrollador. El contenido de esta Property puede ser: <br><br>- un mensaje de texto,<br><br>- un archivo de texto, RTF o CHM,<br><br>- un "enlace" a una ayuda Page (ayuda sobre las novedades generadas al crear el ejecutable). <br><br><br>**Esta propiedad es de solo lectura**. |
| salir con | FechaHora | Fecha de lanzamiento de la versión. <br>**Esta propiedad es de solo lectura**. |
| Forzado | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la versión está forzada,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br>**Esta propiedad es de solo lectura**.<br><br> |
| Prohibido | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si está versión está prohibida,<br><br>- <u><u><u><u>False</u></u></u></u> si la versión está permitida. <br><br><br>**Esta propiedad es de solo lectura**.<br><br> |
| VersionNumber | Cadena de caracteres | Número de la versión. <br>**Esta propiedad es de solo lectura**. |
| envergadura | Integro | Tamaño de la versión en disco (en bytes). <br>**Esta propiedad es de solo lectura**. |

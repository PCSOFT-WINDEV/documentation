
## fIsImage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar si un archivo encontrado en un disco o un archivo contenido en un búfer corresponde a un formato Image reconocido por WINDEV, WEBDEV o WINDEV Mobile.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF fIsImage(MyBuffer) = True THEN
	Info("The variable corresponds to an image.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Examinar un archivo externo

`<Result> = fIsImage(<File name and path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo es una imagen que puede cargarse con WINDEV, WEBDEV o WINDEV Mobile,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<File name and path>: (Cadena de caracteres Ansi o Unicode)`**

Nombre y ruta completa (o relativa) del fichero a identificar. Se puede utilizar una ruta de tipo UNC. 


<a name="SYNTAX2"></a>

### Analizando un Buffer Variable

`<Result> = fIsImage(<Variable name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el buffer corresponde a una imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Variable name>: (Buffer o memorándum binario Variable)`**

Nombre del Buffer o Memo Binario Variable que contiene el Image a analizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Consejo
<a name="consejo_ELTPARAGRAPHE000071"></a>

La función **fIsImage** puede ser relativamente lenta porque intenta cargar la imagen. Para comprobar rápidamente si un archivo es un Image, también tienes la posibilidad de comprobar la extensión del archivo directamente.




## ArrayInfo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las características de una array: tipo de elementos, número de dimensiones, dimensiones, etc.

**Observación**: Esta función sólo puede utilizarse en [WLanguage arrays](../Motscles/1514030.md). 










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Total number of elements found in a one-dimensional array
MyArray is array of 5 int
...
nbDim is int
nbDim = ArrayInfo(MyArray, tiTotalNumber)
// nbDim is equal to 5
```
<a name="Example2"></a>

```wl
// Array dimensions
MyArray is array of 5 by 2 int
nbRowDim is int 
nbRowDim = ArrayInfo(MyArray, tiNumberRows)
// NbRowDim is equal to 5
nbColumnDim is int 
nbColumnDim = ArrayInfo(MyArray, tiNumberColumns)
// NbColumnDim is equal to 2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArrayInfo(<WLanguage array> , <Type of information> [, <Additional parameter>])`
---

**`<Result>: (Entero o booleano)`**

Información solicitada.

Observación: La función no devuelve ningún resultado en una array dinámica que no se haya creado.

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a manipular.

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información a recuperar:


|   |   |
| --- | --- |
| *tiAssociativeWithDuplicate* | Booleano.<br><br>- <u><u><u><u>True</u></u></u></u> si el array asociativo admite duplicados<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| *tiDimension* | Integer. Número de elementos que se encuentran en una dimensión de la array.<br>&lt;Parámetro adicional&gt; debe contener el índice de la dimensión cuyo número de elementos se solicita. <br>Valor de Default para &lt;parámetro adicional&gt;: 1<br>Array asociativo: Número de elementos encontrados en la array. |
| *tiDynamic* | Booleano.<br><br>- <u><u><u><u>True STRIMEULB</u></u></u></u> si el array es un array dinámico.<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br> |
| *tiElementDefinition* | Variable de tipo [Definición](../WDLang1/1000019540.md). Definition del tipo de elementos.<br> |
| *tiElementSize* | Integer. Tamaño (en bytes) de un elemento array.<br> |
| *tiElementType* | Integer. Tipo de elementos array. &lt;Resultado&gt; es un entero que corresponde a las constantes de la función [TypeVar](../WDLang1/3013056.md).<br> |
| *tiKeyType* | Integer. El tipo de claves que se encuentran en la asociación array. &lt;Resultado&gt; es un entero que corresponde a las constantes de la función [TypeVar](../WDLang1/3013056.md).<br>array no asociativa: &lt;El resultado es igual a 0. |
| *tiNumberColumns* | Integer. Número de columnas encontradas en una array bidimensional. El número de columnas corresponde a la 2ª dimensión de la array. <br>Si la array no es una array bidimensional, tiNumberColumns = 0.<br>Array asociativo: 0. |
| *tiNumberDimensions* | Integer. Número de dimensiones de array (entre 1 y 10)<br>Array asociativo: 1. |
| *tiNumberRows* | Integer. Número de filas encontradas en una array bidimensional. El número de filas corresponde a la 1ª dimensión de la array. <br>Si la array no es una array bidimensional, tiNumberRows = 0.<br>Array asociativo: 0. |
| *tiTotalNumber* | Integer. Número total de elementos encontrados en la array. |
| *tiTotalSize* | Integer. Tamaño total (en bytes) de array.<br> |



**`<Additional parameter>: (Opcional)`**

Parámetro adicional que debe especificarse de acuerdo con la información solicitada.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Equivalencias
<a name="equivalencias_ELTPARAGRAPHE000232"></a>

- La constante *tiNumberRows* es equivalente a la constante *tiDimension* utilizada con &lt;Parámetro adicional&gt; = 1. Los siguientes códigos son idénticos:
	
	```wl
	ArrayInfo(MyArray, tiNumberRows)
	ArrayInfo(MyArray, tiDimension, 1)
	```


- La constante *tiNumberColumns* es equivalente a la constante *tiDimension* utilizada con &lt;Parámetro adicional&gt; = 2.
	
	```wl
	ArrayInfo(MyArray, tiNumberColumns)
	ArrayInfo(MyArray, tiDimension, 2)
	```




<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000256"></a>

Esta función se puede utilizar con:

- simple arrays.

- arrays fijos.

- dinámica arrays.

- associative arrays.

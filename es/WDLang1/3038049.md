
## WinAdaptSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adapta el tamaño de la ventana al contenido de los controles. La ventana se redimensiona para que los controles se muestren de la mejor manera posible (sin fila vacía y sin barra de desplazamiento).

**Observación**: La ventana se redimensiona automáticamente si se selecciona la opción "Adaptar tamaño según el contenido" en la pestaña "UI" de la descripción de la ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the Table control
TableDisplay(TABLE_Table1)
// Adapts the size of the window
WinAdaptSize()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WinAdaptSize([<Window used>])`
---

**`<Window used>: (Nombre de la ventana opcional)`**

Nombre o alias de la ventana que se va a utilizar. Si este parámetro corresponde a una cadena vacía ("""), se utiliza la ventana actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Condiciones necesarias para redimensionar la ventana
<a name="condiciones_necesarias_para_redimensionar_ventana_ELTPARAGRAPHE000038"></a>

El dispositivo de escape debe contener al menos uno de los siguientes mandos anclado en anchura y/o en altura:

- Control List Box, 

- Control Tabla, 

- Control Looper, 

- Control TreeView,

- Control ListView, 

- Control Campo de entrada multilínea.




La función **WinAdaptSize** no tiene efecto si la ventana no contiene ningún control que corresponda a los criterios especificados.
<a name="NOTE0_2"></a>




### Efectos de la operación de redimensionamiento en la ventana
<a name="efectos_operacion_redimensionamiento_ventana_ELTPARAGRAPHE000057"></a>

- La ventana se reduce si contiene líneas vacías.

- La ventana se amplía para que todas las filas sean Visible (incluyendo las filas que se muestran mediante una barra de desplazamiento).




 Observaciones: 

- La ventana no está ampliada:

	- Above su tamaño máximo (definido por su anchura máxima y por su altura máxima).

	- Above el tamaño de la pantalla.




- El dispositivo de escape no se reduce por debajo de su tamaño mínimo (definido por su anchura y altura mínimas).



<a name="NOTE0_3"></a>




### Redimensionamiento de la ventana de automatic
<a name="redimensionamiento_ventana_automatic_ELTPARAGRAPHE000074"></a>

La ventana se redimensiona automáticamente si se selecciona la opción "Adaptar tamaño según el contenido" en la descripción de la ventana (pestaña "UI").

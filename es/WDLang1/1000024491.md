


## &lt;Variable picLayer&gt;.AlphaBlend (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Combina dos imágenes. Esto permite crear un efecto de fundido entre 2 imágenes.



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
MyLayer is picLayer
// Declare the promotion image
MyPromoImage is Image = "PromoImage.png"
// Blend the promotion image in the product image with an opacity set to 80%
MyLayer.AlphaBlend(MyPromoImage, 80)
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Destination image>.AlphaBlend(<Source image> [, <Opacity>])`
---

**`<Destination image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) en la que se realizará la combinación. Durante la combinación, esta imagen será considerada como la imagen de fondo.

**`<Source image>: (Control Imagen o variable de tipo Image, WDPic o picLayer)`**

Imagen a combinar. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [WDPic](../WDLang1/1000023503.md). Solo se manipulará la capa de fondo. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [picLayer](../WDLang1/1000024604.md).




**`<Opacity>: (Real opcional)`**

Porcentaje de opacidad a aplicar a &lt;Imagen de origen&gt; antes de realizar la combinación (100% por defecto, no se aplica ninguna opacidad).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El tamaño de la imagen de destino corresponde al tamaño de referencia.

- La función **&lt;Variable picLayer&gt;.AlphaBlend** combina las imágenes y conserva el canal alfa de la imagen de destino. Por lo tanto, no se utiliza el canal alfa de la imagen de origen.

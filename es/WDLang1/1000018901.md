
## TreeMapSearch( Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [TreeMapSeek](../WDLang1/1000019739.md).



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra la índice de un elemento en un control TreeMap a partir del elemento ruta.. Esta ruta se especificó en [TreeMapAñadir](../WDLang1/1000018814.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The TMAP_Wine control is filled from a table. 
// Find the element corresponding to the selected element in the table
SelectionPath is string
SelectionPath = COL_WineType[TABLE_WINE] + "\" + COL_Owner[TABLE_WINE]
ElementSubscript = TreeMapSearch(TMAP_Wine, SelectionPath)
// Select the element in the TreeMap control
IF ElementIndex <> - 1 THEN
	TMAP_Wine = ElementIndex
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TreeMapSearch(<TreeMap control> , <Element path> [, <Start index>])`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado. 

- -1 si ningún elemento corresponde con la búsqueda. 


**Observación**: El elemento encontrado puede ser manejado directamente usando su índice. Puede, por ejemplo, modificar las propiedades de este elemento o seleccionarlo en el control TreeMap.

**`<TreeMap control>: (Nombre del control)`**

Nombre de la control TreeMap utilizada. 
Si este parámetro corresponde a una cadena vacía (""), se utilizará la control TreeMap a la que pertenezca la Process actual.

**`<Element path>: (Cadena de caracteres)`**

Ruta y título del elemento buscado. Esta ruta debe ser idéntica a la que se utilizó cuando se añadió el elemento en la control TreeMap mediante [TreeMapAñadir](../WDLang1/1000018814.md). 
El separador de los diferentes elementos de ruta puede ser:

- el carácter "\\".. Por ejemplo: "MiDirectorio\\MiArchivo". 

- el carácter "TAB. Por ejemplo: "Postres "+TAB+"Helados".




**`<Start index>: (Entero opcional)`**

Índice del elemento desde el que se realizará la búsqueda. Si no se especifica este parámetro, la búsqueda se realiza en toda la control TreeMap. Este parámetro se utiliza para encontrar todas las ocurrencias de una ruta.

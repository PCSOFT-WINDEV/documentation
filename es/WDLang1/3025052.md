


## ConfigureSpellCheck( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura el corrector ortográfico para los controles de edición y para las columnas control Tabla en toda la aplicación.. Puede especificar el idioma del corrector ortográfico.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable the spell check for the entire application
ConfigureSpellCheck(spellcheckEnabled, True)
// Select the English dictionary
ConfigureSpellCheck(spellcheckNation, nationEnglish)
```
<a name="Example2"></a>

```wl
// Find out whether the spell check is enabled for the entire application
IF ConfigureSpellCheck(spellcheckEnabled) = True THEN
	Info("The spelling checker is enabled")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Configurando el corrector ortográfico

`ConfigureSpellCheck(<Option> , <Value>)`
---

**`<Option>: (Constante de tipo Integer)`**

Opción para configurar el corrector ortográfico de OpenOffice: 


|   |   |
| --- | --- |
| *spellcheckCheckerType* | Le permite seleccionar el tipo de ficha utilizada. <br>Si &lt;Valor&gt; corresponde a: <br><br>- *spellcheckIntegratedChecker*: el corrector ortográfico integrado se utilizará para todos los controles de edición de la aplicación.<br><br>- *spellcheckOpenOfficeChecker*: el corrector ortográfico de OpenOffice se utilizará para todos los controles de edición de la aplicación.<br><br><br> |
| *spellcheckEnabled* | Si &lt;Valor&gt; corresponde a <u><u><u><u>True</u></u></u></u>, el corrector ortográfico se activa para todos los controles Campo de entrada de la aplicación. <br>**Atención**: La opción "Revisar ortografía" debe estar seleccionada en el editor.<br>Si &lt;Valor&gt; corresponde a <u><u><u><u>False</u></u></u></u>, el corrector ortográfico se desactiva para todos los controles Campo de entrada de la aplicación. |
| *spellcheckNation* | Especifica el idioma del diccionario que se utilizará. &lt;Value&gt; puede corresponder a una de las constantes que permite identificar los idiomas (ver [Observaciones](#NOTE0_2)). El corrector ortográfico no se propondrá si el diccionario no está instalado.<br><br>Observación: Esta constante se tiene en cuenta sólo para el corrector ortográfico de OpenOffice. |



**`<Value>: (Variant)`**

Valor de la opción configurada (ver &lt;Opciones&gt;).  


<a name="SYNTAX2"></a>

### Averiguar cómo se configuró el corrector ortográfico

`<Result> = ConfigureSpellCheck(<Option>)`
---

**`<Result>: (Variant)`**

Valor de la opción solicitada.

**`<Option>: (Integro)`**

Opción para configurar el corrector ortográfico de OpenOffice:


|   |   |
| --- | --- |
| *spellcheckCheckerType* | Si &lt;Resultado&gt; corresponde a: <br><br>- *spellcheckIntegratedChecker*: el corrector ortográfico integrado se utilizará para todos los controles de edición de la aplicación.<br><br>- *spellcheckOpenOfficeChecker*: el corrector ortográfico de OpenOffice se utilizará para todos los controles de edición de la aplicación.<br><br><br> |
| *spellcheckEnabled* | Si &lt;Resultado&gt; corresponde a <u><u><u><u>True</u></u></u></u>, el corrector ortográfico de OpenOffice se activa para todos los controles Campo de entrada de la aplicación<br>Si &lt;Resultado&gt; corresponde a <u><u><u><u>False</u></u></u></u>, el corrector ortográfico de OpenOffice se desactiva para todos los controles Campo de entrada de la aplicación. |
| *spellcheckNation* | Especifica el idioma del diccionario que se utilizará. &lt;El resultado corresponde a un identificador de idioma (ver Notas). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Notas
<a name="notas_ELTPARAGRAPHE000111"></a>

- El corrector ortográfico de Word también puede ser activado por [AAFExecute](../WDLang1/1000022099.md) y desactivado por [AAFDisable](../WDLang1/1000022018.md).

- Los diccionarios de OpenOffice deben estar instalados antes de poder utilizar el corrector ortográfico de OpenOffice.. Para obtener más información, consulte [Comprobación de la ortografía con OpenOffice](../Editeurs/1013146.md).

- El uso del corrector ortográfico integrado requiere el uso de diccionarios Hunspell.. Para obtener más información, consulte [Revisar ortografía con diccionarios Hunspell](../WDChamp/1410087030.md).

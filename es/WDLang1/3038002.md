


## MenuUnMark( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina la marca ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Marque.gif) que aparece delante de la opción de menú (menú emergente o menú desplegable).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Remove the checkmark displayed in front of "OPT_IconBar"
MenuUnmark(OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuUnmark([<Window>, ] <Menu option>)`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú para desmarcar. 
Si no se especifica este parámetro, **MenuUnMark** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a desmarcar. Este nombre se define en la ventana de description de la opción de menú. 
Observación: Este nombre debe ser especificado sin comillas.

<a name="NOTE0_1"></a>

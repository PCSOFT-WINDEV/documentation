


## &lt;Tablero de instrumentos&gt;.count (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de widgets encontrados en una control Cuadro de mando. Se tienen en cuenta los siguientes widgets: 

- los widgets declarados en el editor de ventanas o en el editor de páginas,

- los widgets añadidos por [&lt;Panel de control&gt;.AddWidget](../WDLang1/1000022502.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the number of displayed widgets and the total number of widgets
InfoBuild("The dashboard contains %1 widgets among which %2 are displayed", ...
	DASH_Dashboard.Count(toTotal), 
	DASH_Dashboard.Count(toShown))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Dashboard control>.Conde(<Type of widget>)`
---

**`<Result>: (Integer)`**

Número de widgets encontrados en la control Cuadro de mando.

**`<Dashboard control>: (Control name)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Type of widget>: (Integer constant)`**

Tipo de widget a tener en cuenta:


|   |   |
| --- | --- |
| *toShown* | Sólo se tienen en cuenta los widgets mostrados. |
| *toTotal* | Todos los Widgets son tenidos en cuenta (no importa si se muestran o no). |

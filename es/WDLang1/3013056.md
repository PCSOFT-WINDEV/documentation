


## TypeVar( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el tipo de una expresión, una variable (durante una llamada a un Procedure por ejemplo) o un control.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PROCEDURE MyProcedure(Parameter)
TypeVar(Parameter)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TypeVar(<Expression>)`
---

**`<Result>: (Integro)`**

Identifica el tipo de valor de Variable o control. 

0 para los control que no devuelven ningún valor (botón, OLE control, ...).

<a name="NOTE0_1"></a>

## Observaciones


### Tipo de cadena y cadena Unicode
<a name="tipo_cadena_cadena_unicode_ELTPARAGRAPHE000540"></a>En Móvil, una cadena Unicode se considera como una cadena:


```wl
MyString is Unicode string
TypeVar(MyString) // returns wlString
```
En WINDEV o WEBDEV, una cadena Unicode se considera como una cadena Unicode:


```wl
MyString is Unicode string
TypeVar(MyString) // returns wlUnicodeString
```

<a name="NOTE0_2"></a>


### Entero del sistema
<a name="entero_del_sistema_ELTPARAGRAPHE000552"></a>

- El entero del sistema se adapta automáticamente al tamaño soportado por el modo de compilación (4 bytes para un programa compilado en modo 32 bits, 8 bytes para un programa compilado en modo 64 bits).

- Para aplicaciones compiladas en modo de 32 bits, **TypeVar** devuelve 8 ( wlInt ).

- Para aplicaciones compiladas en modo de 64 bits, **TypeVar** devuelve 9 ( wlInt _8).


## CameraParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el valor de un parámetro para la cámara del dispositivo.


	<B><B><B>Advertencia</B></B></B>

	 está funcionalidad no estará disponible.



**Observaciones**: 

- Para conocer la lista de valores que admite la cámara para un determinado parámetro, utilice [CameraListParameter](../WDLang1/1000022458.md).

- Esta función es equivalente a la función [VideoParameter](../WDLang1/1000019694.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use the maximum resolution to take a photo
sListResolution is string 
sListResolution = CameraListParameter(vipPhotoResolution)
IF ErrorOccurred = False THEN
	CameraParameter(vipPhotoResolution, ExtractString(sListResolution, lastRank, CR))
ELSE
	Error("Error while retrieving the resolutions supported by the camera.")
END

// Use the maximum zoom value
sListZoom is string
sListZoom = CameraListParameter(vipZoom)
IF ErrorOccurred = False THEN
	nZoom is int = Val(ExtractString(sListZoom, lastRank, CR))
	IF CameraParameter(vipZoom, nZoom) = False THEN
		Error("The camera cannot zoom.")
	END
END

// Geolocation of photos taken by the camera
MyPosition is geoPosition = GPSGetPosition()
CameraParameter(vipGeoPosition, MyPosition)

// Set the ISO sensitivity
CameraParameter(vipISO, 800)
```

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperar el valor de un parámetro para la cámara del dispositivo

`<Result> = CameraParameter(<Parameter>)`
---

**`<Result>: (Tipo de parámetro recuperado)`**



- Valor del parámetro,

- Vacíe la cadena (""") si la cámara no admite el parámetro especificado o si se ha producido un error al recuperar los parámetros de la cámara.




Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

**`<Parameter>: (Constante de la cadena de caracteres o cadena de caracteres)`**

Parámetro para el que se recuperará el valor. Este parámetro puede corresponder a: 

- una de las constantes preestablecidas. Estas constantes corresponden a los parámetros más comunes. 

- una Cadena correspondiente al nombre del parámetro deseado. 
	**Atención**: el nombre del parámetro distingue entre mayúsculas y minúsculas y acentos. Vea la documentación sobre el SDK de Android o sobre el dispositivo para saber el nombre de los parámetros adicionales soportados. Si el nombre especificado no corresponde a un parámetro soportado por la cámara, la función devolverá una cadena vacía.


Las constantes disponibles son las siguientes: 


|   |   |
| --- | --- |
| *vipBalance* | Modo de balance de blancos actualmente soportados por la cámara: automatic, luz del día, nublado, ...<br> |
| *vipCamera* | Índice de la cámara utilizada actualmente:<br><br>- El subíndice incluye entre 1 y el número de cámaras que se encuentran en el dispositivo.<br><br>- -1 si no se utiliza actualmente ninguna cámara.<br><br><br> |
| *vipCompressionJPEG* | Compresión de imágenes JPEG, correspondientes a un número entero incluido entre 1 y 100.<br><br>- 1: Máxima compresión (baja calidad)<br><br>- 100: Sin compresión (muy alta calidad)<br><br><br> |
| *vipEffect* | Efecto de los colores que se tienen en cuenta actualmente: negro y White, cartel, sepia, ...<br> |
| *vipFlash* | Estado actual del flash de la cámara: automatic, sin, ojos anti rojos, ... |
| *vipFlicker* | Ajuste del parpadeo (anti-banding) utilizado actualmente por la cámara: automatic, ninguno, 50Hz, 60Hz, ...<br> |
| *vipFocus* | Estado actual del enfoque de la cámara: automatic, macro, fijo, ...<br> |
| *vipGeoPosition* | Información de geolocalización actual: el valor devuelto es un [geoPosition](../WDLang3/1000019191.md) Variable. Las siguientes propiedades pueden ser especificadas de acuerdo con los elementos soportados por la cámara: latitud, longitud, altitud, fecha de medición.<br> |
| *vipISO* | Sensibilidad ISO actualmente utilizada por la cámara: automatic , 100, 200, 400, ...<br> |
| *vipPhotoResolution* | Resolución de las fotos tomadas por la cámara. <br>El valor se devuelve en el siguiente formato: <br><br><pre><code><Width><TAB><Height></code></pre><br><br><br>Ejemplo: <br><br><pre><code>640<TAB>480</code></pre><br> |
| *vipRotationAuto* | automatic rotación de las fotos según la orientación del dispositivo durante la captura.<br><br>- <u><u><u><u>True</u></u></u></u>: la rotación de automatic está activada.<br><br>- <u><u><u><u>False</u></u></u></u>: la rotación de automatic está desactivada.<br><br><br>Si se activa la rotación de la automatic, la foto se guardará teniendo en cuenta la orientación del dispositivo durante la captura.<br><br>La rotación de automatic está habilitada por Default; sin embargo, algunos dispositivos no soportan la toma de fotos en modo retrato. En este caso, le aconsejamos que desactive la rotación de automatic y que gestione la rotación de Image mediante la programación.<br> |
| *vipScene* | Tipos de escenas que utiliza actualmente la cámara: automatic, noche, deportes, ...<br> |
| *vipZoom* | Valor de zoom utilizado actualmente por la cámara.<br>Los valores devueltos tienen el siguiente formato:<br><br>- 100 (1x zoom = no zoom)<br><br>- 200 (zoom 2x)<br><br>- 300 (zoom 3x)<br><br>- 350 (zoom de 3,5 aumentos)<br><br>- ...<br><br><br>El primer valor es siempre 100 y el último valor corresponde al zoom máximo.. <br>Ejemplo: <br><br><pre><code>100<CR>200<CR>300<CR>350</code></pre><br><br><br>**Observación**: La función devolverá una cadena vacía ("") si la cámara del dispositivo no admite el zoom.<br> |




<a name="SYNTAX2"></a>

### Modificar el valor de un parámetro para la cámara del dispositivo

`<Result> = CameraParameter(<Parameter> , <Value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la modificación del parámetro se realizó con éxito, 

- <u><u><u><u>False</u></u></u></u> de lo contrario o si se produce un error al recuperar los parámetros de la cámara. Por ejemplo, la función **CameraParameter** devolverá <u><u><u><u>False</u></u></u></u> si el parámetro especificado (o su valor) no es compatible con la cámara.




Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

**`<Parameter>: (Constante de la cadena de caracteres o cadena de caracteres)`**

Parámetro para el que se modificará el valor. Este parámetro puede corresponder a: 

- una de las constantes preestablecidas. Estas constantes corresponden a los parámetros más comunes. 

- una Cadena correspondiente al nombre del parámetro deseado. 
	**Atención**: el nombre del parámetro distingue entre mayúsculas y minúsculas y acentos. Vea la documentación sobre el SDK de Android o sobre el dispositivo para saber el nombre de los parámetros adicionales soportados. Si el nombre especificado no corresponde a un parámetro soportado por la cámara, la función devolverá una cadena vacía.

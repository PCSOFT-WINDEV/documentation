


## TableSwapBranch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Intercambia el contenido de dos filas (así como su respectiva estructura de árbol) en una control Tabla TreeView. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Swap branches 5 and 20 in a TreeView Table control
TableSwapBranch(TVT_MENU, 5, 20)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TableSwapBranch(<TreeView Table control> , <First element to swap> , <Second element to swap>)`
---

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

Si este parámetro corresponde a una cadena vacía (""), el control Tabla TreeView utilizado es el control al que pertenece el Event actual.

**`<First element to swap>: (Entero o cadena de caracteres)`**

La primera rama que se utilizará para el intercambio. Esta rama puede ser identificada: 

- por el índice de la rama a mover. Este índice debe estar comprendido entre 1 y el número de filas del control Tabla TreeView (devuelto por la función [TableCount](../WDLang1/3074021.md) o el [Conde](../Proprietes/2510097.md) Property). 

- por la ruta del elemento a mover. Este parámetro tiene el siguiente formato: 
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```





**`<Second element to swap>: (Entero o cadena de caracteres)`**

La segunda rama a mover. Esta rama puede ser identificada: 

- por el índice de la rama a mover. Este índice debe estar comprendido entre 1 y el número de filas del control Tabla TreeView (devuelto por la función [TableCount](../WDLang1/3074021.md) o el [Conde](../Proprietes/2510097.md) Property). 

- por la ruta del elemento a mover. Este parámetro tiene el siguiente formato:
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```




## ListSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un elemento de un Cuadro de Listas, ListView o control Combo Box.

**Recordatorio**: El elemento seleccionado corresponde al elemento actual en el que se visualiza el banda de selección.

**Observación**: La función **ListSelectMinus** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselect the element at index 7 from "LIST_CustomerList"
ListSelectMinus(LIST_CustomerList, 7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ListSelectMinus(<List Box control> [, <Index 1> [... [, <Index N>]]])`
---

**`<List Box control>: (Nombre del control)`**

Nombre de la casilla de la lista (o Combo Box) control a utilizar. Si este parámetro corresponde a una cadena vacía (""), el elemento especificado por su índice se deselecciona en el control List Box (o Combo Box) al que pertenece el Event actual.



**`<Index 1>: (Entero opcional)`**

Índice del primer elemento a deseleccionar. 
Si no se especifica ningún índice, se deseleccionan todos los elementos del cuadro de lista o control Combo Box especificados. 
Si estos índices corresponden a elementos no seleccionados, la función **ListSelectMinus** no tiene ningún efecto.



**`<Index N>: (Entero opcional)`**

Índice del Nº elemento a deseleccionar. 
Si no se especifica ningún índice, se deseleccionan todos los elementos del cuadro de lista o control Combo Box especificados. 
Si estos índices corresponden a elementos no seleccionados, la función **ListSelectMinus** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Control Combo Box
<a name="control_combo_box_ELTPARAGRAPHE000120"></a>

Si se utiliza la función **ListSelectMinus** con un control Combo Box:

- no se muestra ningún elemento de control Combo Box,

- si la control Combo Box es editable, se borra la control Campo de entrada.



<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000134"></a>

- Para anular la selección de un elemento de una control Tabla, utilice [TableSelectMinus](../WDLang1/3074029.md).

- **ListSelectMinus** puede utilizarse en una columna "Combo Box" tabla.

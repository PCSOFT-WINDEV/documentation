
## SysUnlinkExtension( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina la asociación entre una extensión y un comando que fue creado por [SysLinkExtension](../WDLang1/1000021892.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Links the ".txt" extension to the "app.exe" executable
SysLinkExtension(".txt", "C:\Program Files\MyApp\app.exe ""%1""", sysCurrentUser)
...
SysUnlinkExtension(".txt", sysCurrentUser)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysUnlinkExtension(<Extension> [, <User>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la disociación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles.




**`<Extension>: (Cadena de caracteres)`**

Extensión a libre. 

**`<User>: (Constante opcional de tipo Integer)`**

La asociación entre la aplicación y la extensión se guarda en el registro.. Hay dos opciones disponibles: 


|   |   |
| --- | --- |
| *sysAllUsers*<br>(Valor predeterminado) | La asociación será borrada para todos los usuarios de la computadora actual.. <br><br>Observación: Debe tener los derechos de "Administrador" para ejecutar esta operación. |
| *sysCurrentUser* | La asociación se borrará sólo para el usuario actual.. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Una vez liberada la extensión, el explorador puede almacenar temporalmente la asociación o proponer el programa que antes era relacionado (datos independientes de registros).

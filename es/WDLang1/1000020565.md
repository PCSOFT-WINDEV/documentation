
## IWListPrevious( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana interna anterior en la lista de ventanas internas navegadas por un control Ventana interna.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IWListPrevious(IW_InternalWindowControl)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = IWListPrevious(<Internal Window control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró la ventana interna anterior,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se utiliza una animación cuando se cambia una ventana interna con [IWListNext](../WDLang1/1000020564.md) y **IWListPrevious**. Para cambiar una ventana interna sin utilizar animaciones, llame a [IWListPosition](../WDLang1/1000020562.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Las animaciones no están disponibles.

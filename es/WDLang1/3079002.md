


## Gestión de vídeos (transmitidos por una cámara web)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000167"></a>
Para gestionar la secuencia de vídeo actualmente transmitida por la cámara web instalada en el ordenador actual, WINDEV propone:

- la [Control Cámara Web](../WDChamp/1013188.md).

- [Funciones de WLanguage](../WDLang1/3079003.md).




Tienes la habilidad de hacerlo:

- ver el vídeo transmitido por una cámara web.

- guardar un Image del video transmitido en un archivo Image (BMP, JPEG, etc.).

- guardar una sección del vídeo transmitido en un archivo de vídeo.

- ...




**Observación**: Para ver un vídeo en formato AVI, utilice un [control Imagen](../WDChamp/1013218.md) y el [funciones MCI](../WDLang1/3051010.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio
<a name="principio_ELTTEXTE000191"></a>
A manipular los videos:

1. Crear una ventana que contenga una control Cámara Web .

2. Mostrar la ventana description de este control ("description" en el menú contextual).

3. En esta ventana, seleccione la cámara web que desea utilizar. 
	**Observación**: La cámara web a utilizar también se puede especificar a través de la programación ([VideoSetDriver](../WDLang1/3079010.md)).

4. Mediante programación:

	- Muestra la secuencia de vídeo que está siendo transmitida por la cámara web especificada en la control Cámara web ([VideoDisplay](../WDLang1/3079004.md)).

	- Guarde (si es necesario) el vídeo transmitido actualmente por la cámara web especificada como un archivo Image (BMP) o como una secuencia de vídeo (archivo AVI) con [VideoCapture](../WDLang1/3079007.md).

	- Modificar (si es necesario) la cámara web utilizada ([VideoSetDriver](../WDLang1/3079010.md)).

	- Configure (si es necesario) el controlador de captura de vídeo ([VideoConfigure](../WDLang1/3079009.md)).

	- Enumerar (si es necesario) los controladores de captura de vídeo instalados en el ordenador actual ([VideoListDriver](../WDLang1/3079008.md)).

	- Devuelve (si es necesario) el estado de la web control Cámara ([VideoStatus](../WDLang1/3079006.md)).

	- Dejar de visualizar o guardar una secuencia de vídeo en la web control Cámara ([VideoStop](../WDLang1/3079001.md)).






<a name="NOTE2_2"></a>


### Observaciones
<a name="observaciones_ELTPARAGRAPHE000091"></a>

- En tiempo de ejecución, la secuencia de vídeo transmitida por la Cámara Web associated se muestra automáticamente tan pronto como se abre la ventana que contiene la Web control Cámara. En este caso, no es necesario utilizar [VideoDisplay](../WDLang1/3079004.md).

- Si no se ha associated con la cámara web control Cámara, [VideoDisplay](../WDLang1/3079004.md) y/o [VideoCapture](../WDLang1/3079007.md) manipular la secuencia de vídeo transmitida por la cámara web Default (en el ordenador actual).

- Para guardar el video que actualmente transmite la cámara web especificada como un archivo Image, puede utilizar:

	- la función [VideoCapture](../WDLang1/3079007.md).

	- [dSaveImageBMP](../WDLang1/3029026.md), [dSaveImageGIF](../WDLang1/3029040.md) y [dSaveImageJPEG](../WDLang1/3029038.md).

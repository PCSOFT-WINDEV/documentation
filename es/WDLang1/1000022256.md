
## SystemIntegerToInteger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un entero del sistema en un entero de 4 bytes.

Observación: El entero del sistema se codifica en 4 u 8 bytes según el modo de compilación del proyecto (32 o 64 bits). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HandleNum is system int

GetModuleHandle is API Description
GetModuleHandle.DLLName = "kernel32"
GetModuleHandle.FunctionName = "GetModuleHandleA"
GetModuleHandle.ReturnType = apiSystemInt
GetModuleHandle.Parameter[1].Type = apiString

// Gets the window handle via an API
HandleNum = API(GetModuleHandle, "MyWindow")

Num is int
Num = SystemIntegerToInteger(HandleNum)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SystemIntegerToInteger(<Value>)`
---

**`<Result>: (Entero)`**

Entero codificado en 4 bytes independientemente del modo de compilación.

**`<Value>: (Entero del sistema)`**

Entero del sistema a convertir. 
Recordatorio: 

- Un entero del sistema está codificado en 4 bytes en el modo de compilación de 32 bits.

- Un entero del sistema está codificado en 8 bytes en el modo de compilación de 64 bits.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el valor del entero del sistema es demasiado grande para ser convertido en entero, se producirá un error fatal. En este caso, se debe procesar una excepción.

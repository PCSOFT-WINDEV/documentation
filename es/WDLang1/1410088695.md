


## DiagramSearch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca formas que contengan un texto específico en un diagrama. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDiag is Diagram <- DIAGEDT_NoName1

arrIndex is array of int
arrIndex = DiagramSearch(DIAGEDT_NoName1, "Example")
FOR EACH i OF arrIndex
	Trace(MyDiag[i].Text.Caption)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DiagramSearch(<Diagram> , <Searched text> [, <Search options>])`
---

**`<Result>: (Array de números enteros)`**

Array de enteros con los índices de las formas que contienen el texto especificado. 

**`<Diagram>: (Nombre del control o variable de tipo Diagram)`**

Diagrama a utilizar. Este documento corresponde a:

- el nombre de un control Editor de diagramas.

- una variable de tipo [Diagram](../WDLang1/1410088055.md).




**`<Searched text>: (Cadena de caracteres)`**

Texto a buscar en el diagrama. 

**`<Search options>: (Constante opcional de tipo Integer)`**

Indica las opciones de búsqueda:


|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios) |


De forma predeterminada, no se selecciona ninguna opción: la búsqueda distingue entre mayúsculas y minúsculas, y se realiza sobre palabras incompletas. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el mismo texto se encuentra varias veces en una forma, el índice de la forma se devuelve solo una vez.

- Las formas ordenan según su posición en el eje X y luego en el eje Y.


## FinInterestRate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la tasa de interés para un préstamo durante un período específico de tiempo con pagos fijos.

**Observación**: Se supone que el tipo de interés es fijo durante el período.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Interest rate for a loan of 25000 Euros. Six payments
// of 5000 Euros are required to pay off the loan.
ResInterestRate = FinInterestRate(6, -5000, 25000)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinInterestRate(<Number of Necessary Payments> [, <Payment Value> [, <Capital>]])`
---

**`<Result>: (Real o moneda)`**

Tipo de interés buscado.

**`<Number of Necessary Payments>: (Real o moneda)`**

Número total de pagos que deben realizarse.

**`<Payment Value>: (Real opcional)`**

Valor del pago (0 por Default). Este valor debe ser negativo.

**`<Capital>: (Real o moneda opcional)`**

Cantidad del préstamo (1 por Default).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Atención**: La función **FinInterestRate** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

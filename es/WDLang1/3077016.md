
## ThreadState( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado actual de un thread.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ThreadState("Thread1") = threadSuspended THEN
	ThreadResume("Thread1")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ThreadState(<Thread name>)`
---

**`<Result>: (Constante)`**

Estado actual de thread:


|   |   |
| --- | --- |
| *threadRunning* | Se está ejecutando el hilo. |
| *threadStopRequested* | Se ha hecho una solicitud de detención para este thread. |
| *threadSuspended* | El hilo está suspendido. |
| *threadUnknown* | El Thread no fue creado o fue destruido. |



**`<Thread name>: (Cadena de caracteres)`**

Nombre de la Thread cuyo estado se solicita. Este nombre se asigna al ejecutar el hilo (con la función [ThreadExecute](../WDLang1/3077024.md)).

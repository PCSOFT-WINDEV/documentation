


## &lt;Procesamiento de palabras&gt;.exitEdit (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sale de la edición actual. La entrada en edición fue realizada por [&lt;Procesamiento de palabras&gt;.GoIntoEdit](../WDLang1/1000024263.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WP_MyDocControl.ExitEdit()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Word Processing control>.ExitEdit()`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.



<a name="XComponent"></a>

## Componente:
wd280mdl.dll

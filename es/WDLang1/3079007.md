


## VideoCapture (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Captura las imágenes recibidas de la cámara: 

- como imagen: toma una foto. 

- como video: graba un video.   






Dependiendo de la plataforma, se utilizan los siguientes formatos de foto y video:  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) imágenes BMP o videos AVI. 
	Si la extensión del archivo es .jpg o .jpeg, la imagen se guarda en formato JPEG









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Records the video captured in the "CAM_Kitchen" control
// as a 1-minute AVI file ("C:\Temp\MyVideo.AVI")
ResSave = VideoCapture(CAM_Kitchen, "C:\Temp\MyVideo.AVI", viVideoCapture, 60)
IF ResSave = True THEN
	Info("Video successfully saved")
ELSE
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = VideoCapture(<Camera control> , <File to create> [, <Type of capture> [, <Capture duration> [, <Number of images per second> [, <Display during capture>]]]])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el video se grabó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar. No es necesario que se muestre el video en un control Cámara para guardarlo.

**`<File to create>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo a crear.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Cuando se toma una foto, se crea un archivo BMP. Cuando se graba un video, se crea un archivo AVI**Atención**: No se pueden utilizar rutas UNC.

**`<Type of capture>: (Constante opcional de tipo Integer)`**

Tipo de captura:


|   |   |
| --- | --- |
| *viPictureCapture* | ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Captura una imagen y la guarda como un archivo BMP.<br> |
| *viVideoCapture*<br>(Valor predeterminado) | ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Captura un video y lo guarda como un archivo AVI.<br> |



**`<Capture duration>: (Entero opcional)`**

Duración de la grabación de video (en segundos). Si este parámetro se establece en 0 o no se especifica, la grabación de video se detendrá:

- al llamar a la función [VideoStop](../WDLang1/3079001.md).

- cuando no haya más espacio en el disco.


Este parámetro puede ser: 

- un número entero que corresponde al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 




**`<Number of images per second>: (Entero opcional)`**

Número de imágenes guardadas por segundo (15 de forma predeterminada). Este parámetro solo se tiene en cuenta si &lt;Type of capture&gt; es igual a *viVideoCapture*.

**`<Display during capture>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si la secuencia de vista previa debe mostrarse en el control Cámara especificado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

## Observaciones


### Guardar una foto
<a name="guardar_una_foto_ELTPARAGRAPHE000190"></a>

Solo se guardará en formato BMP la imagen que se muestra al ejecutar la función **VideoCapture**. Para guardar esta imagen, también se puede utilizar:

- la función [dSaveImageBMP](../WDLang1/3029026.md): la foto se guardará como un archivo BMP.

- la función [dSaveImageGIF](../WDLang1/3029040.md): la foto se guardará como un archivo GIF.

- la función [dSaveImageJPEG](../WDLang1/3029038.md): la foto se guardará como un archivo JPEG.





### Condiciones de uso: 
<a name="condiciones_uso_ELTPARAGRAPHE000212"></a>

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una cámara Web al equipo actual.


Observación: Esta función es equivalente a la función [CameraCapture](../WDLang1/1000022450.md).

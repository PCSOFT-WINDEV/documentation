
## LuhnCalculate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el siguiente dígito de control de Luhn para una secuencia dada. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sNumberToCheck is string = "120 027 016 0056"
cCheckDigit is character = LuhnCalculate(sNumberToCheck)
Info("The check digit is [%cCheckDigit%]")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LuhnCalculate(<Identifier>)`
---

**`<Result>: (Cadena de caracteres)`**

Dígito de control de Luhn para la secuencia dada

**`<Identifier>: (Cadena de caracteres)`**

Secuencia a partir de la cual se determinará el dígito Luhn. Esta secuencia debe estar compuesta únicamente por números y espacios (los espacios no son relevantes para la fórmula).

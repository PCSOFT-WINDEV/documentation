


## DashResizeWidget (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia las dimensiones de un widget que se encuentra en un control Cuadro de mando. Este widget ya está configurado o mostrado. 

Recordatorio: Un widget corresponde a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una ventana interna.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Places the current widget on two columns to display the details
DashResizeWidget(2, 1)
STC_Details.Visible = True
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Redimensionando el widget actual

`DashResizeWidget(<Width> , <Height> [, <Options>])`
---

**`<Width>: (Integro)`**

Ancho del widget actual en número de celdas o en píxeles. El tamaño mínimo corresponde a una celda. El valor en píxeles se redondea para corresponder a un número entero de celdas. 

El widget utilizado corresponde a la control a la que pertenece el Event actual (Widget o control perteneciente al widget). 

**`<Height>: (Integro)`**

Altura del widget actual en número de celdas o en píxeles. El tamaño mínimo corresponde a una celda. El valor en píxeles se redondea para corresponder a un número entero de celdas.

El widget utilizado corresponde a la control a la que pertenece el Event actual (Widget o control perteneciente al widget). 

**`<Options>: (Constante opcional de tipo Integer)`**

Permite definir la unidad utilizada y el modo de reorganización.: 


|   |   |
| --- | --- |
| *dashNoReorganization* | Redimensiona el widget sin reorganizar los widgets adyacentes.<br><br>Atención: El uso de la constante *dashNoReorganization* puede producir una visualización incorrecta (con widgets superpuestos). Al utilizar esta constante, es importante volver a organizar completamente el control Cuadro de mando con las funciones **DashResizeWidget** y [DashMoveWidget](../WDLang1/1000021289.md). |
| *dashUnitCell*<br>(Valor predeterminado) | La unidad utilizada para el tamaño del widget es un número de celdas. |
| *dashUnitPixel* | La unidad utilizada para el tamaño del widget es el píxel. |




<a name="SYNTAX2"></a>

### Cambiar el tamaño de un widget identificado por su índice

`DashResizeWidget(<Dashboard control> , <Widget index> , <Width> , <Height> [, <Options>])`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Widget index>: (Integro)`**

Índice del widget a utilizar. Este índice corresponde a:

- el número de adición del widget definido en el editor de ventanas o en el editor de páginas.

- el número de adición del widget devuelto por [DashAddWidget](../WDLang1/1000020953.md).




**`<Width>: (Integro)`**

Ancho del widget en número de celdas o en píxeles. El tamaño mínimo corresponde a una celda. El valor en píxeles se redondea para corresponder a un número entero de celdas.

**`<Height>: (Integro)`**

Altura del widget en número de celdas o en píxeles. El tamaño mínimo corresponde a una celda. El valor en píxeles se redondea para corresponder a un número entero de celdas.

**`<Options>: (Constante opcional de tipo Integer)`**

Permite definir la unidad utilizada y el modo de reorganización.: 


|   |   |
| --- | --- |
| *dashNoReorganization* | Redimensiona el widget sin reorganizar los widgets adyacentes.<br><br>Atención: El uso de la constante *dashNoReorganization* puede producir una visualización incorrecta (con widgets superpuestos). Al utilizar esta constante, es importante volver a organizar completamente el control Cuadro de mando con las funciones **DashResizeWidget** y [DashMoveWidget](../WDLang1/1000021289.md). |
| *dashUnitCell*<br>(Valor predeterminado) | La unidad utilizada para el tamaño del widget es un número de celdas. |
| *dashUnitPixel* | La unidad utilizada para el tamaño del widget es el píxel. |

## Observaciones


### Regla utilizada para calcular celdas
<a name="regla_utilizada_para_calcular_celdas_ELTPARAGRAPHE000150"></a>

Veamos algunos ejemplos de un widget cuyo tamaño es igual a 100 px por 100 px con un margen set a 10 px: 


```wl
DashResizeWidget(DASH_TimeZones, 1, 80, 80, dashUnitPixel)
// the widget will be set to 1 x 1

DashResizeWidget(DASH_TimeZones, 1, 200, 200, dashUnitPixel)
// the widget will be set to 2 x 2

DashResizeWidget(DASH_TimeZones, 1, 201, 201, dashUnitPixel)
// the widget will be set to 3 x 3

DashResizeWidget(DASH_TimeZones, 1, 400, 500, dashUnitPixel)
// the widget will be set to 4 x 5
```

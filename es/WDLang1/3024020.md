
## Manipular cadenas de caracteres
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000152"></a>
WINDEV, WINDEV Mobile y WEBDEV permiten realizar diferentes tipos de operaciones en las cadenas de caracteres mediante las funciones WLanguage. Es posible:

- manipular el contenido de la cadena

- realizar conversiones y cifrados.




Unicode está soportado. Para obtener más información, consulte [Uso de Unicode en cadenas de caracteres](../WDLang1/3024038.md).

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Manipular el contenido de una cadena
<a name="manipular_contenido_una_cadena_ELTTEXTE000176"></a>
Las funciones WLanguage permiten extraer parte de una cadena (funciones [Left](../WDLang1/3024001.md), [Right](../WDLang1/3024016.md) o [Middle](../WDLang1/3024023.md)) y realizar búsquedas en una cadena (función [Position](../WDLang1/3024027.md) o [Replace](../WDLang1/3024022.md)).

Algunas de las funcionalidades propuestas por las funciones WLanguage también se pueden realizar con los [operadores de cadenas de WLanguage](../Motscles/1512001.md).

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Convertir, comprimir y cifrar
<a name="convertir_comprimir_cifrar_ELTTEXTE000200"></a>
Varias funciones WLanguage permiten convertir, comprimir y cifrar fácilmente cadenas de caracteres. Puede, por ejemplo:

- convertir una cadena a ANSI u OEM (funciones [AnsiToOem](../WDLang1/3024007.md) y [OemToAnsi](../WDLang1/3024021.md)),

- convertir una cadena a ANSI o UNICODE (funciones [AnsiToUnicode](../WDLang1/3024002.md) y [UnicodeToAnsi](../WDLang1/3024025.md)),

- comprimir y descomprimir una cadena (funciones [Compress](../WDLang1/3024012.md) y [Uncompress](../WDLang1/3024009.md)),

- cifrar o descifrar una cadena de caracteres (funciones [Encrypt](../WDLang1/3024014.md) y [Decrypt](../WDLang1/3024035.md)),

- cifrar o descifrar una cadena de caracteres mediante un algoritmo de cifrado simétrico (funciones [EncryptStandard](../WDLang1/1000021293.md) y [DecryptStandard](../WDLang1/1000021294.md)),

- convertir una cadena de caracteres en una transcripción fonética (función [Phonetic](../WDLang1/3024026.md)).




**Observación**: Para las conversiones de tipo Cadena/Fecha o Cadena/Hora, consulte [Manipular fechas y horas](../WDLang1/3027022.md).


## kbCard (Tipo de variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **kbCard** define todas las características avanzadas de una tarjeta en un control Kanban. Las características de la tarjeta se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Las tarjetas corresponden a los elementos que se muestran en las listas de un control Kanban. Estas tarjetas pueden: 

- agregarse a una lista de un control Kanban (función [KanbanAddCard](../WDLang1/1410089140.md)). 

- pasarse como parámetros a eventos WLanguage (por ejemplo, el evento "Desplazamiento de una tarjeta" del control Kanban). 




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCard is kbCard
MyCard.Title = "File"
MyCard.Text = "Customer file management"
MyCard.Extra.ID = GetIdentifier()
KanbanAddCard(KANBAN_Scheduler.LIST_ToDO, MyCard)
```

## Properties


### Propiedades específicas de las variables de tipo kbCard
<a name="propiedades_especificas_las_variables_tipo_kbcard_ELTPARAGRAPHE000034"></a>

Las siguientes propiedades pueden utilizarse para manipular tarjetas en un control Kanban:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionAllowed | [kbActionAllowed](../WDLang1/1410089541.md) | Acción permitida en la tarjeta.<br><br> |
| Contributor | Character string | Nombre del colaborador asociado a la tarjeta. |
| Date | DateTime | Fecha y hora de la tarjeta. De forma predeterminada, esta propiedad se inicializa con la fecha y hora de creación de la tarjeta. |
| Extra | Associative array | Array asociativo para guardar la información del usuario. |
| Index | Integer | Prioridad de la tarjeta. Corresponde al orden en la lista. <br>Para cambiar la prioridad de una tarjeta, utilice la función [KanbanMoveCard](../WDLang1/1410089144.md). <br><br>**Esta propiedad es de solo lectura.** |
| InternalPage | Character string | Nombre de la página interna que se utilizará para mostrar la tarjeta. Si este parámetro es una cadena vacía (""), se utilizará la tarjeta predeterminada del control Kanban. |
| InternalWindow | Character string | Nombre de la ventana interna que se utilizará para mostrar la tarjeta. Si este parámetro es una cadena vacía (""), se utilizará la tarjeta predeterminada del control Kanban. |
| ListName | Character string | Nombre de la lista que contiene la tarjeta. <br>Para mover la tarjeta a otra lista, utilice la función [KanbanMoveCard](../WDLang1/1410089144.md). <br><br>**Esta propiedad es de solo lectura.** |
| Text | Character string | Texto de la tarjeta. De forma predeterminada, el texto está en formato Markdown. |
| Title | Character string | Título de la tarjeta. |
| Visible | Boolean | <br><br>- <u><u><u>True</u></u></u> si la tarjeta debe ser visible,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br><br><br> |

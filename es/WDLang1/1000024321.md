
## TrustedCertificateReset( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todos los certificados añadidos por [TrustedCertificateAdd](../WDLang1/1000024320.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TrustedCertificateAdd("MyAC.pem")
// code
TrustedCertificateReset()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TrustedCertificateReset()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se han eliminado los certificados de confianza (añadidos por [TrustedCertificateAdd](../WDLang1/1000024320.md)),

- <u><u><u><u>False</u></u></u></u> si se produce un error.

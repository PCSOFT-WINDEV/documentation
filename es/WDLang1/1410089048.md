


## &lt;Variable DiskFile&gt;.Seek (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve y modifica la posición actual en un fichero externo.

Al abrir un archivo, la posición actual corresponde a:

- el primer byte de archivo,

- al último byte del archivo si se abre en modo "adición" (constante *foAdd*).




La utilización de las funciones de lectura ([&lt;DiskFile Variable&gt;.Leer](../WDLang1/1410089044.md), [&lt;Variable DiskFile&gt;.ReadLine](../WDLang1/1410089045.md)) y de escritura ([&lt;DiskFile Variable&gt;.Write](../WDLang1/1410089041.md), [&lt;DiskFile Variable&gt;.WriteLine](../WDLang1/1410089042.md)) modifica la posición actual.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Open an external file
MyFile is DiskFile
MyFile = fOpen("C:\MyDirectories\File.txt", foReadWrite)
IF NOT ErrorOccurred THEN
	...
	// Get the current position
	ResPosition = MyFile.Seek(0, fpCurrent)
	...
END
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <File used>.SetPosition(<Move> , <Source position>)`
---

**`<Result>: (Entero)`**



- Posición actual después de la modificación,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.






**`<File used>: (Variable de tipo DiskFile)`**

Nombre de la variable [DiskFile](../WDLang1/1410088883.md) inicializada con [fOpen](../WDLang1/3036036.md) o [fCreate](../WDLang1/3036017.md).

**`<Move>: (Entero)`**

Cantidad de bytes (caracteres) a compensar en el archivo. Para retroceder desde el final del archivo (constante *fpEnd*), este número debe ser negativo.

**`<Source position>: (Constante)`**

Especifica la posición de la fuente:


|   |   |
| --- | --- |
| *fpBeginning* | La posición de la fuente corresponde al primer byte de archivo (posición 0). |
| *fpCurrent* | La posición de la fuente corresponde a la posición actual. |
| *fpEnd* | La posición de la fuente corresponde al último byte del archivo. |

<a name="NOTE0_1"></a>

## Observaciones


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000305"></a>

La función **&lt;Variable DiskFile&gt;.Seek** genera un error en los siguientes casos:

- el archivo no se abre,

- el archivo está bloqueado por otro ordenador o por otra aplicación.



<a name="NOTE0_2"></a>


### Posición actual
<a name="posicion_actual_ELTPARAGRAPHE000320"></a>Para recuperar la posición actual en el fichero externo, utilice la siguiente sintaxis:

```wl
FileName is DiskFile
...
CurrentPosition = MyFile.Seek(0, fpCurrent)
```


No se realiza ningún movimiento en el archivo especificado.

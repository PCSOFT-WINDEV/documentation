


## InSimulatorMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la aplicación se inicia en "modo prueba simulador" de WINDEV Mobile.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InSimulatorMode() = True THEN 
	Info("You will notice differences with the real mode")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InSimulatorMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecuta la aplicación en modo GO simulador desde WINDEV Mobile. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000223"></a>

La función **InSimulatorMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InSimulatorMode** pueden compartirse entre varios productos sin generar errores de compilación.

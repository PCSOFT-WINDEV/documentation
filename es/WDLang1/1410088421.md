
## DiagramGroup( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un grupo a partir de los elementos seleccionados en el editor de Diagram control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a group from the selected elements
IF DiagramGroup(DIAGEDT_MyDiagram) THEN
	ToastDisplay("Group created. To undo, press Ctrl+Z.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DiagramGroup(<Diagram>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha creado el grupo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles sobre el error.




**`<Diagram>: (Nombre del control o variable de tipo Diagram)`**

Diagrama a utilizar. Este parámetro corresponde a: 

- el nombre de un control Editor de diagramas. 

- el nombre de una variable de tipo [Diagrama](../WDLang1/1410088055.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se crea una forma de tipo "Grupo" y se añade a la Diagram. 

- Esta forma se crea a partir de los elementos actualmente seleccionados. 

- Si la selección contiene menos de 2 formas, la función no crea el grupo y devuelve <u><u><u><u>False</u></u></u></u>. 

- Los usuarios finales pueden deshacer la creación de un grupo si el Diagram está en modo de edición.

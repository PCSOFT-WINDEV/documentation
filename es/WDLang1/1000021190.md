


## Funciones para la gestión de los controles de los botones de radio (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de los botones de radio mediante la sintaxis de los prefijos:



|   |   |
| --- | --- |
| [&lt;Botón de radio&gt;.add](../WDLang1/1000021186.md) | Añade una nueva opción a una control Botón de opción. |
| [&lt;Botón Radio&gt;.Count](../WDLang1/1000021188.md) | Devuelve el número de opciones encontradas en una control Botón de opción. |
| [.Delete](../WDLang1/1000021189.md) | Elimina una opción de un control Botón de opción. |
| [&lt;Botón de radio&gt;.Insertar](../WDLang1/1000021187.md) | Inserta una nueva opción en una control Botón de opción. |




## Funciones de gestión de máscaras
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar máscaras de entrada: 



|   |   |
| --- | --- |
| [MaskFormat](../WDLang1/1000024738.md) | Define el formato de una cadena de caracteres utilizando una variable de tipo [InputMask](../WDLang1/1000024886.md). |
| [MaskPhoneNumber](../WDLang1/1000024741.md) | Recupera la máscara de entrada del número de teléfono de un país determinado. |
| [MaskZipCode](../WDLang1/1000024742.md) | Recupera la máscara de entrada del código postal adaptado a un país determinado. |

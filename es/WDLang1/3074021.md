


## TableCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve: 

- el número de columnas que se encuentran en una Tabla o control Tabla TreeView.

- el número de filas que se encuentran en una Tabla o control Tabla TreeView. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of rows found in the "TABLE_Product" control
ResNumberRow = TableCount(TABLE_Product)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableCount(<Table control> [, <Row/Column>])`
---

**`<Result>: (Integro)`**

Número de filas o columnas que se encuentran en la control especificada .

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.


Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Row/Column>: (Constante entera (opcional))`**

Se usa para especificar el tipo de resultado devuelto: 

- las constantes *toShown*, *toTotal* y *toVisible* permiten obtener un número de filas. 

- la constante *toColumn* permite obtener el número de columnas del control.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000200"></a>

La función **TableCount** puede utilizarse en:

- controles Tabla o Tabla TreeView basados en un archivo de datos.

- controles Tabla o Tabla TreeView rellenados mediante programación.

- controles de selección única o múltiple.

### Equivalencia entre la función TableCount y el tamaño Property
<a name="equivalencia_entre_funcion_docparampagetitleshort_tamano_property_ELTPARAGRAPHE000238"></a>

La función **TableCount** y la [envergadura](../Proprietes/2510125.md) Property son equivalentes:

```txt
ResNbColumn = TableCount(<Table control>, toColumn)

ResNbColumn = <Table control>.Size
```




## <Window>.WinEnableDDW (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>


La función OVI se utiliza para atenuar automáticamente las ventanas desactivadas. Para obtener más información, consulte [Oscurecimiento de ventanas inactivas](../Editeurs/9000071.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code of "BTN_Validate"
// Enable the DDW feature for the "WIN_EditCustomer" window
WIN_EditCustomer.EnableDDW(True)
// Open a dialog box
Dialog("Save changes?")
// As long as the dialog box is displayed, 
// the "WIN_EditCustomer" window is dimmed.
```


<a name="3038056_Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Window>.EnableDDW(<DDW activation>)`
---

**`<Result>: (booleano)`**

Anterior Estado de OVI:

- <u><u><u><u>True</u></u></u></u>: se habilitó la función OVI para la ventana especificada.

- <u><u><u><u>False</u></u></u></u>: la función OVI no estaba habilitada para la ventana especificada.




**`<Window>: (Nombre del control)`**

Nombre o alias de la ventana que se va a utilizar. Esta ventana debe abrirse.

**`<DDW activation>: (booleano)`**

Nuevo estado de OVI:

- <u><u><u><u>True</u></u></u></u>: la función OVI está habilitada para la ventana especificada.

- <u><u><u><u>False</u></u></u></u>: la función OVI no está habilitada para la ventana especificada.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Configuración de los ordenadores del usuario final
<a name="configuracion_los_ordenadores_del_usuario_final_ELTPARAGRAPHE000213"></a>

Dependiendo de la configuración de los ordenadores del usuario final, la función OVI puede no tener ningún efecto.

En efecto, la función OVI se ve afectada por las características de la tarjeta de vídeo, la velocidad del procesador, etc.

Si se produce un problema, pida a los usuarios finales que actualicen el controlador de su tarjeta de vídeo.

**Observación**: Para funcionar, los ordenadores de los usuarios finales deben:

- tienen un modo de visualización superior a 256 colores.

- estar en modo de 32 ó 64 bits.

- estar ejecutando Windows 2000 o posterior.

- utilice el nivel máximo de "Aceleración de hardware" para la tarjeta de vídeo (propiedades avanzadas de visualización de el equipo).



<a name="NOTE0_2"></a>


### Velocidad de atenuación
<a name="velocidad_atenuacion_ELTPARAGRAPHE000234"></a>

La tasa de atenuación por defecto en las ventanas es de set a 18 %. Para cambiar esta tasa, utilice [WinRateDDW](../WDLang1/3038057.md).

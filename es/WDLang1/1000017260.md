
## AppChangeParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia una información en el mecanismo de actualización de la aplicación automatic.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
IF AppParameter(appUPDMode) = 3 THEN
	AppChangeParameter(appUPDMsgStop, ...
			"The application " + ProjectInfo(piEXEName) + ...
			"will be automatically stopped.")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppChangeParameter(<Information> , <New value> [, <Application path>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la información solicitada,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Para más detalles sobre el problema, utilice [ErrorInfo](../WDLang1/3013008.md).




**`<Information>: (Constante de tipo Integer)`**

Característica de aplicación para modificar:  


|   |   |
| --- | --- |
| *appUPDAutoCheck* | Activa la comprobación de actualizaciones de automatic: <br><br>- <u><u><u><u>True</u></u></u></u> para habilitar,<br><br>- <u><u><u><u>False</u></u></u></u> para desactivar. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Constante no disponible. |
| *appUPDDisplayError* | Permite mostrar un mensaje si se encuentra un problema al comprobar las actualizaciones: <br><br>- <u><u><u><u>True</u></u></u></u> para mostrar un mensaje, <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Constante no disponible. |
| *appUPDDisplayMode* | Opciones de visualización de las notificaciones de actualización:<br><br>- 0: Se muestra un mensaje de actualización en una ventana de la aplicación.<br><br>- 1: Se muestra un mensaje de actualización como Notification.<br><br><br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Constante no disponible. |
| *appUPDLocalVersion* | Número de versión del ejecutable local. |
| *appUPDMode* | Modo de funcionamiento actual de la aplicación:<br><br>- 0: Funcionamiento estándar.<br><br>- 1: Prohibida la puesta en marcha: los ejecutables actuales siguen funcionando.<br><br>- 2: Prohibida la puesta en marcha: los ejecutables actuales reciben un mensaje de advertencia de forma regular.<br><br>- 3: Prohibida la puesta en marcha: los ejecutables actuales se detienen automáticamente después de un tiempo muerto dado.<br><br><br> |
| *appUPDMsgForbid* | Mensaje que aparece cuando el inicio de la aplicación está prohibido. |
| *appUPDMsgStop* | Mensaje mostrado antes de que la aplicación se detenga automáticamente (cuando appUPDMode = 3). |
| *appUPDMsgWarning* | Mensaje mostrado regularmente (cuando appUPDMode = 2). |
| *appUPDNetwork* | Nivel de actualización:<br><br>- 0: Ninguna actualización. <br>	No se utiliza la actualización en el grafo.<br><br>- 1: Actualización de red habilitada. <br>	Se comprueban las actualizaciones disponibles y se muestra el cuadro de diálogo de actualización de automatic.<br><br>- 2: Actualización de red habilitada y registro de aplicaciones hacia el servidor de actualización. <br>	Se comprueban las actualizaciones disponibles y se muestra el cuadro de diálogo de actualización de automatic. <br>	Tan pronto como se inicia, el ejecutable crea un archivo ".LOK" en el servidor. Este archivo permite a [WDAdminExe](../WDAdminExe/2028052.md) identificar a los clientes que utilizan la aplicación.<br><br><br> |
| *appUPDPath* | Ruta de la información de actualización en el servidor. |
| *appUPDProtocol* | Protocolo de red utilizado para buscar actualizaciones. Este protocolo puede corresponder a "LOCAL" o a "HTTP". |
| *appUPDServer* | Servidor donde se buscarán las actualizaciones.<br>Si se utiliza el protocolo HTTP, IP Address o nombre del ordenador.<br>Si se utiliza el protocolo LOCAL, nombre de la red compartida. |
| *appUPDTimeout* | Tiempo de espera entre dos mensajes (si appUPDMode = 2) o antes de que la aplicación se detenga automáticamente (si appUPDMode = 3). Este tiempo muerto se expresa en segundos. |



**`<New value>: (Tipo esperado)`**

Nuevo valor que se dará a la información especificada.

**`<Application path>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable a utilizar. Si no se especifica este parámetro, se gestionará la aplicación actual..


## fWebDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre físico del directorio que contiene las imágenes, los archivos Java Script y los archivos Java del sitio WEBDEV. Este directorio corresponde al directorio &lt;Nombre del proyecto&gt;_WEB. Este directorio se encuentra tanto en la computadora de desarrollo como en la computadora implementación.

Esto le permite leer, escribir o modificar los archivos que se encuentran en este directorio.. Atención, las modificaciones sólo serán posibles si el host concede derechos de escritura (por razones de seguridad).



Observación: Para get el directorio "&lt;nombre del proyecto&gt;_WEB", puede utilizar: 

- la función [FolderWeb](../WDLang2/3012015.md): esta función devuelve una ruta relativa para ser interpretada por el navegador. 

- la función **fWebDir**: esta función devuelve un ruta absoluto para ser utilizado desde el servidor con el [funciones de gestión de archivos externos (fxxx)](../WDLang1/3036034.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Runtime directory
Info("Image files found in the directory: " + fWebDir())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fWebDir()`
---

**`<Result>: (Cadena de caracteres)`**

Nombre físico completo del directorio que contiene las imágenes, los archivos Java Script y Java del sitio WEBDEV. Este resultado tiene el siguiente formato: "C:\\Mi Proyecto\\Horario\\Horario_WEB". No se encuentra ningún carácter "\\" al final de este resultado.

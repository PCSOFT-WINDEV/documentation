
## AppVersionInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre una versión disponible en el servidor de instalación.

**Observación**: Esta función sólo puede ser utilizada con una red instalación en una red local. No puede ser usado con una instalación por IP.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is string = AppListVersion()
IV is InfoVersion

DisplayInfo is string = [
Number = %1
Date = %2
Size = %3 bytes
Comment = %4
Enabled = %5
Forced = %6
Forbidden = %7
]

FOR EACH STRING MyString OF Res SEPARATED BY CR
	IV = AppVersionInfo(MyString)
	Trace(StringBuild(DisplayInfo, ...
		IV.VersionNumber, IV.Date, ...
		IV.Size, IV.Comment, ...
		IV.Enabled, IV.Forced, IV.Forbidden))
	Trace("===============================================")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppVersionInfo(<Application version> [, <Application path>])`
---

**`<Result>: (InfoVersion Variable)`**

[InfoVersion](../WDLang1/1000018905.md) variable que contiene la información relativa a la versión de la aplicación especificada. 

**`<Application version>: (Cadena de caracteres)`**

Número de versión de la solicitud cuya información se solicita. 

**Observación**: Los números de versión disponibles son devueltos por [AppListVersion](../WDLang1/1000018843.md). 

**`<Application path>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable a utilizar. Si no se especifica este parámetro, se gestionará la aplicación actual.

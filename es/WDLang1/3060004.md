
## FontSelect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre la ventana estándar para la selección de la fuente y selecciona la fuente especificada.

Se puede utilizar la fuente seleccionada por el usuario:

- para escribir en dibujos ([dFont](../WDLang1/3029020.md)).

- para escribir en gráficos ([grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md), [grAxisTitleFont](../WDLang3/3042059.md)).

- para imprimir (con la función [iFont](../WDLang5/3046014.md)).

- en los controles de una ventana o reporte ([tipo de letra](../Proprietes/2515045.md) property).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a default font
DefaultF is font = FontCreate("Arial", 12)
// Use a variable to store the user font
UserF is font
// Ask the user to modify his font
FontSelect(UserF, True, DefaultF)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Abrir el selector de fuentes especificando una fuente Default

`<Result> = FontSelect(<Selected font> [, <Option> [, <Default font>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario ha seleccionado una fuente,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el usuario ha hecho clic en el botón "Cancelar").




**`<Selected font>: (tipo de letra)`**

Nombre de la [tipo de letra](../Motscles/1514045.md) Variable que contendrá las características de la fuente seleccionada por el usuario.

**`<Option>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para mostrar las opciones avanzadas de la fuente (subrayado, color),

- <u><u><u><u>False</u></u></u></u> para no mostrar estas opciones en el selector de fuentes.




**`<Default font>: (Fuente opcional Variable)`**

Nombre de la [tipo de letra](../Motscles/1514045.md) Variable que corresponde a las características de la fuente a seleccionar por Default en el selector de fuentes de Windows.


<a name="SYNTAX2"></a>

### Abrir el selector de fuentes especificando todas las características de la fuente Default

`<Result> = FontSelect(<Selected font> [, <Option> [, <Font name> [, <Size> [, <Style> [, <Color>]]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el usuario ha seleccionado una fuente,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el usuario ha hecho clic en el botón CANCELAR).




**`<Selected font>: (tipo de letra)`**

Nombre de la [tipo de letra](../Motscles/1514045.md) Variable que contendrá las características de la fuente seleccionada por el usuario. Si sólo se especifica este parámetro, la fuente Default seleccionada en el selector de fuentes tendrá las características de esta fuente.

**`<Option>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para mostrar las opciones avanzadas de la fuente (subrayado, color),

- <u><u><u><u>False</u></u></u></u> para no mostrar estas opciones en el selector de fuentes.




**`<Font name>: (Cadena de caracteres opcional)`**

Nombre de la fuente seleccionada por Default en el selector de fuentes. Este nombre debe corresponder a una de las fuentes instaladas en el ordenador actual.

**`<Size>: (Entero opcional)`**

Tamaño de la fuente seleccionada por Default en el selector de fuentes. Este tamaño se expresa en picas (1 pica = 0,353 mm).

**`<Style>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Atributos de la fuente seleccionada por Default:



|   |   |
| --- | --- |
| *iBold* | Fuente en negrita |
| *iItalic* | Fuente cursiva |
| *iNormal* | Fuente estándar, sin atributos |
| *iStrikeOut* | Fuente tachada |
| *iUnderlined* | Fuente subrayada |



**`<Color>: (Entero opcional)`**

Color de la fuente seleccionada por Default. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)).

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- y [color predefinido de WLanguage](../WDLang5/3010002.md).

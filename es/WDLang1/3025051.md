
## ToolTipDelay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se usa para modificar:

- el tiempo de espera antes de mostrar las sugerencias.

- la duración de visualización de las indicaciones de las herramientas. El tooltip actual se cierra automáticamente cuando el usuario mueve el cursor del ratón.




Estas modificaciones se realizan para la aplicación actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The tooltips will be displayed when the controls 
// are hovered during more than 1 second
ToolTipDelay(tooltipDelayBeforeOpening, 100)
// The tooltips will be displayed during 5 seconds
ToolTipDelay(tooltipDelayDisplay, 500)
```


<a name="Example2"></a>





```wl
// Restore the default time-out before 
// displaying the tooltips
ToolTipDelay(tooltipDelayBeforeOpening, -1)
// Restore the default display duration 
// of control tooltips
ToolTipDelay(tooltipDelayDisplay, -1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ToolTipDelay(<Type of Timeout> , <New timeout>)`
---

**`<Type of Timeout>: (La constante booleana)`**

Tipo de tiempo de espera a modificar:


|   |   |
| --- | --- |
| *tooltipDelayBeforeOpening* | Tiempo de espera antes de mostrar los datos de las herramientas. |
| *tooltipDelayDisplay* | Duración de la visualización de los consejos de las herramientas de los controles. |



**`<New timeout>: (Entero o Duración)`**



- Nuevo tiempo de espera (en centésimas de segundo). Este parámetro puede corresponder a: 

	- un número entero que corresponde al número de centésimas de segundo,

	- una variable de tipo Duration,

	- la duración en un formato legible (por ejemplo, 1 s o 10 ms). 




- -1 para restaurar los valores de Default (gestionado por Windows).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los tooltips son habilitados por Default en los controles. Para desactivarlas, utilice [ShowToolTip](../WDLang1/3025014.md).

- La duración máxima de la visualización de los tooltips es de set a 32765 ms (32,6 segundos). Consejo para utilizar una mayor duración: usar un control Estático que se hará visible o Invisible. 

- Esta función configura tanto los tooltips estándar como los personalizados.

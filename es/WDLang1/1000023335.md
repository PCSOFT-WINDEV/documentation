


## SpreadsheetAddImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un Image en la hoja de trabajo actual de un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add the logo.png image between A2 and B5
SpreadsheetAddImage(PSHEET_Spreadsheet, "c:\temp\logo.png", "A2", "B5") 

// Add the banner.png image at C6 position
SpreadsheetAddImage(PSHEET_Spreadsheet, "c:\temp\banner.png", "C6")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetAddImage(<Spreadsheet control> , <Image> , <Insertion cell> [, <End cell>])`
---

**`<Result>: (Cadena de caracteres)`**

Identificador del agregado Image.

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Image>: (Cadena de caracteres)`**

ruta completa del archivo de imagen que se añadirá a la control. 

**`<Insertion cell>: (Cadena de caracteres)`**

Coordenada de la celda donde debe situarse la esquina superior izquierda de la Image. 

**`<End cell>: (Cadena de caracteres)`**

Coordenada de la celda donde debe situarse la esquina inferior derecha de la Image. El Image cubre la célula. Si no se especifica este Property, la imagen se inserta con su tamaño set a 100%.




## DrawingAvailable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el navegador soporta las funcionalidades de dibujo.

**Observación**: Las funcionalidades de dibujo en el código Navegador solo están disponibles en navegadores recientes (compatibles con el estándar HTML 5).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF DrawingAvailable() = False THEN 
	Info("Drawing not available.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DrawingAvailable()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el navegador soporta las funcionalidades de dibujo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

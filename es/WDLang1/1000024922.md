


## &lt;Variable WDPic&gt;.Rotation (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Gira una imagen en formato WDPic. Todas las capas de la imagen giran. 

Observación: Para girar una sola capa, utilice la función [&lt;Imagen&gt;.Rotation](../WDLang1/1000024338.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a source image
MyWDPicImage is WDPic
MyWDPicImage.FileName = fExeDir() + ["\"] + "Source.png"

// Rotate the image
MyWDPicImage.Rotation(45)

// Save the result
MyWDPicImage.Save(fExeDir() + ["\"] + "Result.png")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Hacer un giro simple

`<Result> = <WDPic image>.Rotation(<Angle> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el giro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que contiene la imagen a manipular.

**`<Angle>: (Real)`**

Ángulo de rotación (en grados). 

- Si el ángulo es positivo, la imagen gira en el sentido de las agujas del reloj. 

- Si el ángulo es negativo, la imagen gira en sentido contrario a las agujas del reloj.




**`<Options>: (Constante de tipo Integer)`**

Opciones de rotación: 


|   |   |
| --- | --- |
| *drAdapt* | La imagen se gira y se amplía. Luego, se reduce para que corresponda al tamaño inicial de la imagen. |
| *drDefault*<br>(valor predeterminado) | La imagen se amplía (si es necesario) para que corresponda a las dimensiones de la imagen girada. |
| *drNoEnlarging* | La imagen gira pero no se amplía: la imagen puede truncarse. |




<a name="SYNTAX2"></a>

### Hacer un giro modificando el centro de rotación

`<Result> = <WDPic image>.Rotation(<Angle> , <X> , <Y> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el giro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que contiene la imagen a manipular.

**`<Angle>: (Real)`**

Ángulo de rotación (en grados). 

- Si el ángulo es positivo, la imagen gira en el sentido de las agujas del reloj. 

- Si el ángulo es negativo, la imagen gira en sentido contrario a las agujas del reloj.




**`<X>: (Real)`**

Nueva coordenada X del centro de rotación.

**`<Y>: (Real)`**

Nueva coordenada Y del centro de rotación.

**`<Options>: (Constante opcional de tipo Integer)`**

Opciones de rotación: 


|   |   |
| --- | --- |
| *drDefault*<br>(Valor predeterminado) | La imagen se amplía (si es necesario) para que corresponda a las dimensiones de la imagen girada. |
| *drNoEnlarging* | La imagen gira pero no se amplía |

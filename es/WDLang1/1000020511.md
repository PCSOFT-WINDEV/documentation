
## DecimalToSexagesimal( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ángulo sexagesimal (en base 60) correspondiente a un ángulo decimal.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Finds the sexagesimal angle corresponding to an angle of 128.91°
rAngle is real = 128.91
sSexagesimalAngle is string = DecimalToSexagesimal(rAngle)
Info("An angle of " + rAngle + "° corresponds to: " + sSexagesimalAngle)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DecimalToSexagesimal(<Angle>)`
---

**`<Result>: (Cadena de caracteres)`**

Resultado de la conversión en formato sexagesimal. Esta cadena tiene el siguiente formato: 
```txt
XXX°XX'XX''
```


**`<Angle>: (Real)`**

Ángulo en formato decimal que debe convertirse.

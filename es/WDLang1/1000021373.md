


## Hoja de cálculoAñadir Fórmula (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una fórmula de cálculo a un control Hoja de cálculo que se encuentra en una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
//  Initialize the 1st column with numbers from 1 to 10
FOR I = 1 TO 10
	SpreadsheetAddData(PSHEET_Spreadsheet, 1, i, "Row " + I)
END

SpreadsheetAddData(PSHEET_Spreadsheet, 1, 1, 13)
SpreadsheetAddData(PSHEET_Spreadsheet, 1, 2, 20)
SpreadsheetAddFormula(PSHEET_Spreadsheet, 1, 3, "A1+A2/2") // Displays 23
SpreadsheetAddFormula(PSHEET_Spreadsheet, "A4", "A1+A2")
SpreadsheetAddData(PSHEET_Spreadsheet, 1, 2, 33) // Displays 33 in the A3 cell
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadiendo una fórmula en una celda identificada por sus coordenadas

`SpreadsheetAddFormula(<Spreadsheet control> , <Column> , <Row> , <Formula>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Column>: (Integro)`**

El número de columna de la célula que contiene la fórmula. 

**`<Row>: (Integro)`**

Número de Fila de la celda que contiene la fórmula. 

**`<Formula>: (Cadena de caracteres)`**

Fórmula para añadir. El primer carácter "=" no debe incluirse en la fórmula. Por ejemplo: "A1+B2-A2"
En caso de error de sintaxis o de identificador inválido, se muestra "###" en la celda y se muestra un error no fatal.


<a name="SYNTAX2"></a>

### Añadir una fórmula en una célula identificada por su nombre

`SpreadsheetAddFormula(<Spreadsheet control> , <Cell name> , <Formula>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Cell name>: (Cadena de caracteres)`**

Nombre de la celda a utilizar. Por ejemplo: "A4". 

**`<Formula>: (Cadena de caracteres)`**

Fórmula para añadir. El primer carácter "=" no debe incluirse en la fórmula. Por ejemplo: "A1+B2-A2"
En caso de error de sintaxis o de identificador inválido, se muestra "###" en la celda y se muestra un error no fatal.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la celda (Columna, Fila) no existe, se crea la celda.

- Si la celda (Columna, Fila) ya existe, se borra el contenido.. Se sustituye por la fórmula &lt;Fórmula&gt;.. 

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana.. 

- Las celdas pueden ser renombradas ([SpreadsheetAddName](../WDLang1/1000022523.md)). Tienes la habilidad de usar el nombre de una célula en la fórmula.

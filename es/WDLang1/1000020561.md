
## IWListDeleteAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las ventanas internas de la lista de ventanas internas buscadas por un control Ventana Interna..

**Observación**: Si se define una ventana interna en edición para la control Ventana interna, esta ventana se mostrará después de la llamada a esta función.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IWListDeleteAll(IW_InternalWindowControl)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`IWListDeleteAll(<Internal Window control>)`
---

**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.

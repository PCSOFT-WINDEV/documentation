
## FirstDayOfMonth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del primer día del mes.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Returns the first day of February 2000
MyDate is Date
MyDate = FirstDayOfMonth(2000, 2)
```
<a name="Example2"></a>

```wl
// Returns the first day of the current month
MyDate is Date
MyDate = FirstDayOfMonth()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Obtener el primer día del mes a partir de una fecha

`<Result> = FirstDayOfMonth([<Date>])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha del primer día del mes especificado (en formato AAAAMMDDD).

**`<Date>: (Cadena de caracteres, variable de tipo Date o DateTime opcional)`**

Fecha para la que debe calcularse el primer día del mes. Esta fecha puede corresponder a : 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  

- una cadena en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo (utilice CCC para las milésimas de segundo)). En este caso, solo se tiene en cuenta la fecha. 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.


Si no se especifica este parámetro, se utiliza la fecha actual de forma predeterminada.


<a name="SYNTAX2"></a>

### Obtener el primer día del mes a partir de un año y un mes

`<Result> = FirstDayOfMonth(<Year> , <Month>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha que corresponde al primer día del mes especificado.

**`<Year>: (Entero)`**

Año para el que debe calcularse el primer día del mes. 

**`<Month>: (Entero)`**

Mes para el que debe calcularse el primer día.

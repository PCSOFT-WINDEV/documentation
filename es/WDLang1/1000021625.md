
## ControlPane( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del panel estático o dinámico actual que se encuentra en un control Pestaña associated con un control. 

Observación: Para los paneles de pestañas dinámicas, esta función puede ser usada en el código de la ventana interna para averiguar el nombre del panel de pestañas associated. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on "BTN _New" button
MyPane is Control 
MyPane <- ControlPane(MySelf)
MyPane.Caption = "New document.txt"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ControlPane(<Control>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del panel de pestañas al que pertenece el control (alias). 

- "" (cadena vacía) si la &lt;control&gt; no es un panel de pestañas. 




**`<Control>: (Nombre del control)`**

El nombre de control se encuentra en un panel estático o dinámico de control Pestaña.




## Funciones de gestión de mensajes toast
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar mensajes toast: 



|   |   |
| --- | --- |
| [ToastDeleteAll](../WDLang1/1000020336.md) | Borra todos los brindis mostrados por [ToastDisplay](../WDLang1/1000019358.md). |
| [ToastDisplay](../WDLang1/1000019358.md) | Muestra un mensaje "Toast". |
| [ToastDisplayIW](../WDLang1/1000024643.md) | Abre un toast cuyo contenido se basa en una ventana interna. |
| [ToastDisplayPopup](../WDLang1/1000020625.md) | Muestra una página emergente durante una duración determinada para mostrar un mensaje "Toast". |

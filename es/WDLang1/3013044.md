
## TraceEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el ventana de salida si se abrió esta ventana o deja de guardar la información de la traza en un archivo de texto. 

**Observación**: Cuando se cierra el proyecto, el ventana de salida se cierra automáticamente y se detiene la grabación del archivo de texto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Closing the trace window
TraceEnd()
```

<a name="SYNTAX1"></a>

## Sintaxis

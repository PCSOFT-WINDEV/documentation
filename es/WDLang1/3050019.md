
## StatError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el número de error del último error causado por una función Statxx ([funciones estadísticas](../WDLang1/3050027.md)).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003050019&name=staterror_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Identify the type of error
ResError = StatError()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = StatError()`
---

**`<Result>: (Integro)`**

Número del último error causado por una función estadística:


|   |   |   |
| --- | --- | --- |
| 0 | Ningún error | Todas las funciones |
| 1 | Matriz vacía | Todas las funciones |
| 2 | Se requieren por lo menos dos valores en la matriz fila o en la columna de matriz. | [StatVariance](../WDLang1/3050033.md),<br>[StatAverageDeviation](../WDLang1/3050024.md) |
| 3 | Media geométrica negativa | [StatAverage](../WDLang1/3050036.md),<br>[StatStandardDeviation](../WDLang1/3050025.md) |
| 4 | División por cero | [StatAverage](../WDLang1/3050036.md),<br>[StatCorrelation](../WDLang1/3050022.md) |
| 5 | La matriz no existe o se superan los valores límite. | Todas las funciones |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

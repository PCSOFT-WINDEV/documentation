


## XamlDisable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Simula la ausencia de .Net 3.0 framework.

El .Net 3.0 framework es necesario para usar el control Carrusel, el Cubo control y el Xaml control. Cuando se desarrolla una aplicación que contiene una control Carrusel, un control de cubo y/o un control Xaml, puede ser useful para saber cómo reaccionará la aplicación en un ordenador que no esté equipado con la framework .Net 3.0 (o posterior). La función **XamlDisable** permite realizar esta prueba.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Simulate the absence of .Net 3.0 framework
IF InTestMode() = True THEN
	XamlDisable()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`XamlDisable()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **XamlDisable** es equivalente al cambio de nombre de la clave de registro:

```txt
"HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\v3.0"
```

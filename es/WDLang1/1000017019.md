
## SysScreenResolution( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la resolución de una pantalla.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the resolution of the main screen
ResString is string
ResString = SysScreenResolution()
Trace("Width resolution = " + ExtractString(ResString, 1) + CR + ...
	"Height resolution = " + ExtractString(ResString, 2) + CR + ...
	"Nb bits per pixel = " + ExtractString(ResString, 3) + CR + ...
	"Refresh rate = " + ExtractString(ResString, 4))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysScreenResolution([<Screen name>])`
---

**`<Result>: (Cadena de caracteres)`**



- Resolución de la pantalla en el siguiente formato:
	
	```txt
	<Width Resolution> + TAB + <Height Resolution> + TAB + 
	<Number of Bits per Pixel> + TAB + <Refresh Rate>
	```


- Cadena vacía ("") si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Screen name>: (Cadena de caracteres opcional)`**

Nombre de la pantalla cuya resolución se solicita.

Si no se especifica este parámetro, se devuelve la resolución de la pantalla principal.. Si el nombre especificado no existe, &lt;Result&gt; corresponde a una cadena vacía (""").

**Observación**: La lista de nombres de pantalla es devuelta por [Pantalla SysListScreen](../WDLang1/1000017014.md).

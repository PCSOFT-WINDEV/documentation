


## WinEdCreateNote (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una Nota Reposicionable control mediante programación. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NOTE_MyNote is Control
NOTE_MyNote <- WinEdCreateNote(MyWindow, "New note")
NOTE_MyNote.BackgroundColor = LightBlue
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`[<Result> <- ] WinEdCreateNote(<Window> , <Initial text> [, <X> [, <Y>]])`
---

**`<Result>: (Control Variable)`**

El control de la Nota creada puede ser manejado por un [Control](../WDLang1/1000020967.md) Variable. En este caso, los valores deben asignarse con el operador **'<-'**. 

Si la función **WinEdCreateNote** se utiliza directamente, el control Nota se crea en la ventana y es visible por defecto. 

**`<Window>: (Nombre de ventana)`**

Nombre o alias de la ventana donde se creará la Nota control. 
Si no se especifica este parámetro o si corresponde a una cadena vacía (""), se creará la nota control en la ventana actual.

**`<Initial text>: (Cadena de caracteres)`**

Texto que aparece en la Nota control.

**`<X>: (Entero opcional)`**

Coordenada X de la Nota control a crear (posición en el eje X) en píxeles. Posición horizontal de la Corner superior izquierda del control, en relación con la Corner superior izquierda del área de cliente de la ventana (es decir, la ventana sin barra de título, barra de menú o bordes). Este parámetro corresponde a la [XInitial](../Proprietes/2510140.md) Property. 

Si no se especifica este parámetro, la coordenada X de la Nota control corresponde a la posición X del ratón. 

Este valor puede modificarse con el programa [X](../Proprietes/2510135.md) Property.

**`<Y>: (Entero opcional)`**

Coordenada Y de la Nota control a crear (posición en el eje Y) en píxeles. Posición vertical de la Corner superior izquierda del control, en relación con la Corner superior izquierda del área de cliente de la ventana (es decir, la ventana sin barra de título, barra de menú o bordes). Este parámetro corresponde a la [YInitial](../Proprietes/2510087.md) Property. 

Si no se especifica este parámetro, la coordenada Y de la Nota control corresponde a la posición Y del ratón. 

Este valor puede modificarse con el programa [Y](../Proprietes/2510137.md) Property.

## Observaciones


### Características de la nota reposicionable control
<a name="caracteristicas_nota_reposicionable_control_ELTPARAGRAPHE000071"></a>

- La creación puede ser cancelada por Ctrl + Z. 

- Una vez creada, la Nota Reposicionable control creada es persistente: si se cierra la ventana y se vuelve a abrir, se restablece la note. 
	Para crear un control Nota no persistente, utilice la función [ControlCreate](../WDLang1/1000020876.md) con la constante *typRepositionableNote*. 

- La Nota Reposicionable control se guardará durante la llamada a las funciones: 

	- [WinEdSave](../WDLang1/1000021848.md),

	- [WinEdSaveNote](../WDLang1/1000022120.md),

	- [NoteSaveAll](../WDLang1/1000022396.md). 






<a name="NOTE0_2"></a>


### Equivalencias
<a name="equivalencias_ELTPARAGRAPHE000095"></a>

- Esta función equivale a una nota creada en la ventana por el usuario final a través de AAF (automatic Application Feature). Para obtener más información, consulte [note reposicionable (AAF)](../Editeurs/1000021820.md). 

- Se utiliza para crear una nota reposicionable control en la ventana actual, esta función es equivalente a [NotaCrear](../WDLang1/1000022395.md).

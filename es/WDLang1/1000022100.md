
## AAFChangeCaption( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Personaliza el texto de una opción de menú para los menús contextuales correspondientes a una AAF (característica de la aplicación automatic). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AAFChangeCaption(aafTableToExcel, "Save in an Excel file")

AAFChangeCaption(TAB_DOC, aafTabNew, "New document")
AAFChangeCaption(TAB_CUSTOMER, aafTabNew, "New customer")
```

<a name="XSYNTAX"></a>

## Sintaxis

**`<Control or window>: (Cadena de caracteres opcional)`**

Nombre de control o nombre de la ventana abierta. 

Si no se especifica este parámetro, la modificación se realiza para todos los controles y para todas las ventanas.

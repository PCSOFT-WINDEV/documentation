
## docFormatting (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docFormatting** se utiliza para definir las características de formato de un elemento en un tratamiento de texto Document. Las características de formato pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no se puede utilizar directamente. Debe utilizarse a través de**: 

- el tipo [docElementFormula](../WDLang1/1000022937.md). 

- el tipo [docElementText](../WDLang1/1000022493.md). 







<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docFormatting

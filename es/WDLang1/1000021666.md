<a name="Utilisation"></a>

<a name="Exemple1"></a>

## Ejemplo


```wl
// Déclenche l'exécution d'une tâche
t est une TâcheParallèle = TâcheParallèleExécute(Proc)
...
// Annule l'exécution de la tâche
TâcheParallèleDemandeAnnulation(t)
```

<a name="XSYNTAXE"></a>

## Sintaxis
<a name="SYNTAXE1"></a>

### Demander l'annulation d'une tâche parallèle

`TâcheParallèleDemandeAnnulation(<Tâche>)`
---

**`<Tâche>: (Variable de type TâcheParallèle)`**




<a name="SYNTAXE2"></a>

### Demander l'annulation des tâches parallèles présentes dans un tableau

`TâcheParallèleDemandeAnnulation(<Tableau de tâches>)`
---

**`<Tableau de tâches>: (Tableau de variables de type TâcheParallèle)`**


## MatReadLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee el valor de todos los elementos que se encuentran en una matriz fila.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the elements found in row 5
ResReadRow = MatReadLine("MyMatrix", 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatReadLine(<Matrix> , <Row to read> [, <Total element size> [, <Number of decimals> [, <Element format>]]])`
---

**`<Result>: (Cadena de caracteres)`**

Valor de todos los elementos que se encuentran en el fila especificado, en el siguiente formato:

```txt
<Value of element 1> + TAB + <Value of element 2> + TAB + ...
+ TAB + <Value of element N>
```


**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a leer, definida con [MatCreate](../WDLang1/3050058.md).

**`<Row to read>: (Integro)`**

Número de la fila a leer.

**`<Total element size>: (Entero opcional)`**

Tamaño total de un elemento a manipular. Este tamaño incluye el signo, el separador de mil, el punto decimal, el exponente, las partes enteras y las partes decimales del valor del elemento.

- Si el elemento tiene más de &lt;Tamaño total del elemento&gt; dígitos, &lt;Tamaño total del elemento&gt; se incrementa.

- Si el elemento tiene menos de &lt;Tamaño total del elemento&gt; dígitos, &lt;Resultado&gt; se llena de espacios.

- Si no se especifica &lt;Tamaño total del elemento&gt;, el valor del elemento leído tendrá el tamaño de su parte significativa.

- &lt;El tamaño total de los elementos no debe exceder de 100.

- Si el valor del &lt;Tamaño total de un elemento&gt; es 0, el valor del elemento tendrá el tamaño de su parte significativa.




**`<Number of decimals>: (Entero opcional)`**

Número de decimales a leer. Este parámetro siempre comienza con un punto ("."). El último decimal se redondeará. Este parámetro sólo afecta a las notaciones de flotador ("f") y a las notaciones exponenciales ("e" y "E").. &lt;Tamaño total del elemento&gt; se ignora si &lt;Número de decimales&gt; es mayor que &lt;Tamaño total del elemento&gt;.

| *&lt;Número de decimales&gt;* | *Tipo"d" (entero)* | *tipo "e" o "E" (notación exponencial) y tipo "f" (notación flotante)* |
| --- | --- | --- |
| .0 | - | 0 |
| .N | - | N (si N es mayor que el número de decimales, la parte decimal se rellenará con ceros) |
| (no especificado) | - | 6 |



**`<Element format>: (Carácter opcional)`**

Formato para representar un elemento:

- d: el valor del elemento se expresa como un entero con signo. Los dígitos que se encuentran después del punto decimal se truncan y &lt;Número de decimales&gt; se ignoran.

- e o E: notación exponencial con 1 dígito antes del punto decimal, &lt;Número de decimales&gt; dígitos después del punto decimal y al menos 2 dígitos para el exponente.

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000120"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Tamaño de &lt;Resultado&gt;.
<a name="tamano_ELTPARAGRAPHE000131"></a>

**Atención**: El tamaño de la cadena de caracteres devuelta en &lt;Result&gt; no puede exceder los 260 caracteres.. Si la matriz fila contiene más caracteres, la cadena devuelta en &lt;Result&gt; contendrá sólo los primeros 260 caracteres.
<a name="NOTE0_3"></a>




### Extracción de un valor de elemento de &lt;Resultado&gt;.
<a name="extraccion_valor_elemento_ELTPARAGRAPHE000139"></a>

Para extraer uno o más valores de elementos de la cadena de caracteres &lt;Resultado&gt;, utilice la función [ExtractString](../WDLang1/3024017.md).
<a name="NOTE0_4"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000150"></a>

**Atención**: La función **MatReadLine** no devuelve ningún código de error. Para saber si se han generado errores al leer los elementos fila, utilice [MatError](../WDLang1/3050044.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

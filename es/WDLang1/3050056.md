
## FinFutureVal( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor futuro de una inversión. Los pagos son regulares y fijos. El tipo de interés es fijo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Future value of an investment. The initial capital is set to 100000 Euros.
// 3000 Euros are paid each month during 2 years. The interest rate is set to 8%.
ResFutureVal = FinFutureVal(-100000, -3000, 24, 8)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinFutureVal([<Initial capital> [, <Periodic Deposit> [, <Number of periods> [, <Rate> [, <Start>]]]]])`
---

**`<Result>: (Real o moneda)`**

Valor futuro de la inversión.

**`<Initial capital>: (Real o moneda opcional)`**

Capital inicial (0 por Default).

**`<Periodic Deposit>: (Real o moneda opcional)`**

Depósito periódico (0 por Default). Este valor se fija para toda la duración de la operación.. En la mayoría de los casos, este importe incluye el capital y los intereses.

**`<Number of periods>: (Entero opcional)`**

Número total de períodos para la operación (1 de Default).

**`<Rate>: (Real opcional)`**

Tasa de interés de la inversión expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set al 5. Esta tasa no puede ser nula.

**`<Start>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si los pagos deben realizarse al inicio del período (valor predeterminado),

- <u><u><u><u>False STRIMEULB</u></u></u></u> si los pagos deben efectuarse al final del período.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Unidad de tiempo
<a name="unidad_tiempo_ELTPARAGRAPHE000085"></a>

Los cálculos se pueden realizar según meses o años. Los valores de &lt;Tarifa&gt; y &lt;Período&gt; deben corresponder a la misma unidad.

Por lo tanto, para los pagos mensuales de un préstamo de 5 años con un tipo de interés anual set a 8%, uso:

- &lt;Tasa&gt; = 8% / 12, 

- &lt;Número de períodos&gt; = 5\*12




Para el mismo préstamo con pagos anuales, use:

- &lt;Tasa&gt; = 8%, 

- &lt;Número de períodos&gt; = 5




Para todos los parámetros, los pagos están representados por un número negativo mientras que los cobros están representados por un número positivo.
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000105"></a>

**Atención**: La función **FinFutureVal** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

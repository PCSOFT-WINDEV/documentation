


## &lt;Scheduler&gt;.ChangeMode ( Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el período (es decir, el número de días) que muestra un control Planificador.



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on BTN_Day
SCH_Scheduler.ChangeMode(schModeDay)

// Click on BTN_Week
SCH_Scheduler.ChangeMode(schModeWeek)

// Click on BTN_Month
SCH_Scheduler.ChangeMode(schModeMonth)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Scheduler control>.ChangeMode(<Display mode> [, <Number of days>])`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Display mode>: (Constante de tipo Integer)`**

Rango de tiempo que debe mostrarse en la pantalla de control Planificador: 


|   |   |
| --- | --- |
| *schModeAccordingApt* | Muestra el control Planificador durante el número de días que muestran todas las citas disponibles.<br><br> |
| *schModeCustomNbDay* | Muestra la control Planificador sobre el número de días especificado en &lt;Número de días&gt;. |
| *schModeDay* | Muestra la control Planificador durante un día. |
| *schModeMonth* | Muestra el control Planificador durante un mes. |
| *schModeWeek* | Muestra la control Planificador durante una semana de 5 o 7 días. El número de días se especifica en &lt;Número de días&gt;. El primer día que se muestra es el primer día de la semana definido en la ventana de description del control Planificador. |



**`<Number of days>: (Entero opcional)`**

Número de días mostrados. Este parámetro se tiene en cuenta sólo si &lt;Modo de visualización&gt; corresponde a la constante *schModeCustomNbDay* o a la constante *schModeWeek*. De lo contrario, se produce un error WLanguage.




## HTMLEditorToEmail (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta el contenido de un control de un editor HTML a un correo electrónico Variable (para enviar la Email). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyMessage is Email
HTMLEditorToEmail(HTMEDT_MyHTMLEditor, MyMessage)
MyMessage.Recipient = "flo-us@gmal.com"
MyMessage.Subject = "Test"
EmailSendMessage(MySession, MyMessage, emailOptionEncodeHeader)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLEditorToEmail(<HTML Editor control> , <Email> [, <Option>])`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control. 

**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la variable [Email](../WDLang3/1000018713.md) en la que debe importarse el HTML Document del editor HTML control. Si este parámetro no se especifica o es una cadena vacía (""), se utiliza la estructura [Email](../WDLang3/3032029.md).
 
Observación: El contenido HTML del control HTML Editor se asigna automáticamente al **HTML** Property de la estructura [Email](../WDLang3/1000018713.md) Variable (o al **Email.HTML** Variable de la estructura [Email](../WDLang3/3032029.md)). Los caracteres se codifican en UTF-8 y se inserta el conjunto de caracteres correspondiente.

**`<Option>: (Constante opcional de tipo Integer)`**

Opciones de exportación, válidas sólo cuando se utiliza la estructura de Email: 


|   |   |
| --- | --- |
| *emailOptionDefault*<br>(Valor predeterminado) | Se genera un identificador único para identificar cada Image encontrada en el archivo HTML. **Este identificador es único independientemente del archivo HTML exportado**.<br><br>Modo de operación de Default. |
| *emailOptionNoCID* | Se genera un identificador para las imágenes en el archivo HTML. **Este identificador no es único. Si se exportan varios archivos HTML, pueden utilizar los mismos identificadores Image**. |

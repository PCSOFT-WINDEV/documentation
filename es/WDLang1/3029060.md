
## ArrowEnd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece la posición del extremo de destino de una flecha en un control "Forma".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Move ends of an arrow
ArrowStart(SHA_Arrow1, 5, 5)
ArrowEnd(SHA_Arrow1, 5, 15)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ArrowEnd(<Shape control> , <X> , <Y>)`
---

**`<Shape control>: (Nombre del control)`**

Nombre del control Forma a manipular. El control Forma debe ser una flecha.

**`<X>: (Entero)`**

Posición horizontal (o coordenada X) en pixeles del extremo de destino de la flecha.

**`<Y>: (Entero)`**

Posición vertical (o coordenada Y) en pixeles del extremo de destino de la flecha.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Atención**: No se detecta ningún error de compilación si el nombre del control Forma pasado como parámetro no es de tipo flecha. Sin embargo, se mostrará un error WLanguage en tiempo de ejecución.

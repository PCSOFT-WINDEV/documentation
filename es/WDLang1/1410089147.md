
## JSONValid (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de una cadena JSON. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check if the JSON string is valid 
IF JSONValid(sJSON) THEN
	// Yes => Initialize JSON variable
	oJSON is JSON = StringToJSON(sJSON)
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = JSONValid(<JSON string>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la cadena JSON es válida,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, utilice la función [ErrorInfo](../WDLang1/3013008.md) para obtener más información sobre el error.




**`<JSON string>: (Cadena de caracteres)`**

Cadena que contiene los datos JSON (ASCII de 7 bits + codificación JSON o UTF-8).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Codificación
<a name="codificacion_ELTPARAGRAPHE000044"></a>

La codificación utilizada corresponde al estándar JSON: 

- codificación ASCII de 7 bits para los primeros 128 caracteres, es decir, los caracteres sin acentos

- codificación JSON de los otros caracteres: "\\u" seguido del código del carácter en formato hexadecimal de 4 bytes. Ejemplo: para el carácter "é" (código ASCII 233, código hexadecimal E9), la codificación corresponde a "\\u00E9".

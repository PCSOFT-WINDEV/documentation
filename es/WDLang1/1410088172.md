
## diagSelection (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagSelection** se utiliza para definir todas las características avanzadas de una selección realizada en un Diagram. Las características de esta selección pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the selected elements in a Diagram Editor control
MySelection is diagSelection <- DIAGEDT_MyDiagram.Selection
IF MySelection.Shape.Count > 0 THEN
	// Get the shapes
	FOR EACH stShape OF MySelection.Shape	
		STC_SELECTION_INFO = stShape.Name + "has been selected." + 
		" [ " + stShape.X + ", " + stShape.Y + " - " + stShape.Width + 
		"x" + stShape.Height + " ]"	
	END
ELSE
	STC_SELECTION_INFO = "Click a shape in the diagram to get the selection"
END
```

## Properties


### Propiedades específicas de las variables diagSelection
<a name="propiedades_especificas_las_variables_diagselection_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades pueden utilizarse para manipular una selección:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Altura | Real | <br><br>- Altura de la selección, expresada en píxeles,<br><br>- -1 si no hay selección. <br><br><br> |
| anchura | Real | <br><br>- Ancho de la selección, expresado en píxeles,<br><br>- -1 si no hay selección. <br><br><br> |
| Ángulo | Real | Ángulo de rotación de la selección. |
| Background |   | <br><br>- Fondo Color de la forma seleccionada,<br><br>- -1 si no se selecciona ninguna forma o si se seleccionan varias formas.  <br><br><br> |
| comment | Cadena de caracteres | comment associated con la selección. |
| escribir a máquina | Constante de tipo Integer | Tipo de elemento seleccionado |
| Forma | Array de [diagShape](../WDLang1/1410088082.md) | Formas en la selección. |
| Franja de orilla | [diagBorder](../WDLang1/1410088179.md) | Características del borde de la forma seleccionada. |
| Opacidad | Integro | <br><br>- Porcentaje de opacidad de la forma seleccionada. Este valor oscila entre invisible (0) y completamente opaco (100).<br><br>- -1 si no se selecciona ninguna forma.<br><br><br> |
| Redondo | Real | <br><br>- Curvatura de la forma seleccionada,<br><br>- -1 si la forma no tiene esquinas redondeadas o si se seleccionan varias formas. <br><br><br> |
| Sombra | booleano | <br><br>- <u><u><u>True</u></u></u> si la forma tiene sombra,<br><br>- <u><u><u>False</u></u></u> en caso contrario.<br><br><br> |
| texto | [diagText](../WDLang1/1410088393.md) | Características del texto de la forma seleccionada. |
| vincular | [diagConnector](../WDLang1/1410088454.md) | Características del conector seleccionado. |
| X | Real | <br><br>- Posición X de la caja que contiene la selección (expresada en píxeles),<br><br>- -1 si no hay selección.<br><br><br> |
| Y | Real | <br><br>- Posición Y de la caja que contiene la selección (expresada en píxeles),<br><br>- -1 si no hay selección.<br><br><br> |
| ZOrder | Integro | Orden Z de la selección. |

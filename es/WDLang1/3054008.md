
## Desarrollo de aplicaciones multilingües
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000079"></a>
Una aplicación multilingüe puede distribuirse en varios idiomas. WINDEV, WEBDEV y WINDEV Mobile tienen en cuenta los diferentes lenguajes de la aplicación durante todos los pasos de desarrollo de una aplicación.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Principio para el desarrollo de una aplicación multilingüe
<a name="principio_para_desarrollo_una_aplicacion_multilingue_ELTTEXTE000103"></a>
Las principales etapas para desarrollar una aplicación multilingüe son las siguientes: 

1. [Seleccionar los idiomas del proyecto y el análisis.](../WDLang1/3054010.md)

2. [Traducir los elementos del proyecto (ventana, mensajes de código, etc.) a los diferentes idiomas del proyecto.](../WDLang1/3054009.md)

3. [Definir el idioma del proyecto mediante programación](../WDLang1/3054012.md).

4. [Gestionar conjuntos de caracteres específicos en los archivos de datos](../WDLang1/3054017.md).

5. [Crear el ejecutable y el programa de instalación](../WDLang1/3054006.md).

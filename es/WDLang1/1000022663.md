
## ControlExecuteModification( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Considera que la control ha sido modificada por el usuario (la propiedad [Modified](../Proprietes/2510080.md) se establece en <u><u><u>True</u></u></u> para la control y para la ventana si es necesario) y ejecuta el WLanguage "Siempre que se modifique" Event associated con la control. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ControlExecuteModification(EDT_Edit)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ControlExecuteModification(<Control used>)`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular.

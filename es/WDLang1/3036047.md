
## fDriveReady( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber:

- si una Drive determinada (tarjeta de almacenamiento, CD o DVD) está disponible,

- si existe la Drive.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036047&name=fdriveready_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Drive ready?
ResReady = fDriveReady("E")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fDriveReady(<Disk path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el Drive está listo, accesible y disponible, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Disk path>: (Cadena de caracteres)`**

Ruta del disco a utilizar en el siguiente formato:

- &lt;Carta de Drive&gt;. Por ejemplo: "a", "a:" o "a:\\".

- Ruta UNC. Por ejemplo: "_ASLASH_\\&lt;NombreServidor&gt;\\&lt;NombreCompartido&gt;".




Si el Drive es un disquete Drive, una tarjeta Drive, un CD Drive o un DVD Drive, el Drive sólo está listo si contiene un disquete, una tarjeta, un CD o un DVD.

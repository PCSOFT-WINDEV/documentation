


## ArrayCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve:

- el número de elementos que se encuentran en una array unidimensional.

- el número de filas que se encuentran en un array bidimensional.

- el número de elementos que se encuentran en la primera dimensión de un array tridimensional o N-dimensional.




**Observación**: Esta función sólo puede utilizarse en [WLanguage arrays](../Motscles/1514030.md).








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyArray is array of 2 strings
ArrayInsert(MyArray, 1, "WINDEV")
ArrayInsert(MyArray, 2, "WEBDEV")
ArrayInsert(MyArray, 3, "WINDEV and WEBDEV")
// Number of elements?
Trace(ArrayCount(MyArray)) // Displays 5
```
<a name="Example2"></a>

```wl
MyArray is array of 2 by 3 strings
ArrayInsertLine(MyArray, 1, "A", "B", "C")
ArrayInsertLine(MyArray, 2, "D", "E", "F")
// Number of rows?
Trace(ArrayCount(MyArray)) // Displays 4
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArrayCount(<WLanguage array>)`
---

**`<Result>: (Integro)`**



- Número de elementos encontrados en la array si &lt;WLanguage array&gt; es una array unidimensional .

- Número de filas que se encuentran en el array si &lt;WLanguage array&gt; es un array bidimensional.

- Número de elementos encontrados en la primera dimensión de la array si &lt;WLanguage array&gt; es una array tridimensional (o N-dimensional) .




**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar.

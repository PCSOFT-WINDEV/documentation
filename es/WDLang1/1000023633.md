


## &lt;Navegar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la navegación en una pantalla HTML control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code of Forward button
HTM_HTMLControl1.Navigate(htmlForward)
```
<a name="Example2"></a>

```wl
// Code of Back button
HTM_HTMLControl1.Navigate(htmlBack)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<HTML control>.Navigate(<Navigation option>)`
---

**`<HTML control>: (Nombre del control)`**

Nombre de la pantalla HTML control a utilizar.

**`<Navigation option>: (Constante)`**

Dirección para la navegación:


|   |   |
| --- | --- |
| *htmlBack* | Muestra la página anterior (equivalente al botón "Atrás/Anterior" del navegador). |
| *htmlForward* | Muestra la página siguiente (equivalente al botón "FORWARD" del navegador). |
| *htmlHome* | Muestra la página de inicio (equivalente al botón "Inicio" del navegador). Para obtener el valor inicial del control de la pantalla HTML, utilice la función [InitialValue](../Proprietes/2510128.md) Property.<br> |
| *htmlRefresh* | Actualiza la página actual (equivalente al botón "Actualizar" del navegador). |
| *htmlStop* | Detiene la carga de los recursos utilizados por la página. La Event "Cargar recursos" de la control no se ejecuta. |

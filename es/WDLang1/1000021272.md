


## eventClose (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra una sincronización Event entre varios hilos.


<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EventClose(<Event>)`
---

**`<Event>: (Cadena de caracteres, o AutomaticEvent o ManualEvent Variable)`**

Nombre de la Event a cerrar. Este nombre puede corresponder a: 

- una cadena de caracteres, 

- el nombre de una variable de tipo [ManualEvent](../WDLang1/1000021275.md), 

- el nombre de una variable de tipo [AutomaticEvent](../WDLang1/1000021270.md). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función equivale a la siguiente sintaxis de [EventChange](../WDLang1/3077017.md): 
	
	```txt
	EventChange(<Event>, eventClose)
	```


- **Para un automatic Event**: El Event permanece cerrado. 
	**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera.

- **Para un evento manual**: El Event está cerrado. 
	**Nuevos hilos**: Los nuevos temas se añaden a la cola de espera.

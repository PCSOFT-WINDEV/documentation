
## CameraStatus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de un control Cámara.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the status of "CAM_Kitchen" control
// The explanation is displayed in the status bar. 
ResStatus = CameraStatus(CAM_Kitchen)
SWITCH ResStatus
	CASE viStatusDisplayCapture:
		Message("The video is displayed in the Camera control. " + ...
			"Video recording in progress")
	CASE viStatusDisplayNoCapture:
		Message("The video is displayed in the Camera control. " + ...
			"No recording in progress")
	CASE viStatusCaptureNoDisplay:
		Message("A video is currently saved. "+ ...
			"This video is not displayed in the Camera control")
	CASE viStatusNoCaptureNoDisplay:
		Message("No video being displayed and/or " + ...
			"recorded in the Camera control")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CameraStatus(<Camera control>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la control Cámara especificada:


|   |   |
| --- | --- |
| *viStatusCaptureNoDisplay* | Se está grabando un vídeo. Este vídeo no se muestra en la control Cámara especificada. |
| *viStatusDisplayCapture* | El vídeo se muestra en el formato especificado control Cámara. El vídeo se está grabando. |
| *viStatusDisplayNoCapture* | El vídeo se muestra en el formato especificado control Cámara. El vídeo no se está grabando. |
| *viStatusNoCaptureNoDisplay* | No se está visualizando y/o grabando ningún vídeo en la control Cámara especificada. |



**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

## Observaciones




### Condiciones de uso: 
<a name="condiciones_uso_ELTPARAGRAPHE000070"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una cámara Web al equipo actual.

- Esta función es equivalente a la función [VideoStatus](../WDLang1/3079006.md).  







<a name="NOTE0_2"></a>




### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000091"></a>

- Para capturar el vídeo recibido de una cámara web, utilice [CameraCapture](../WDLang1/1000022450.md).

- Para que se muestre el video capturado por una "cámara Web" en un control Cámara, utilice la función [CameraDisplay](../WDLang1/1410086595.md).




## InPrelaunchedSessionMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si el sitio se inicia en modo "sesión preiniciada" de WEBDEV. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InPrelaunchedSessionMode() = True THEN
	Info("Site in pre-launched session mode")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InPrelaunchedSessionMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecuta el sitio en modo sesión preiniciada. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000203"></a>

La función **InPrelaunchedSessionMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InPrelaunchedSessionMode** pueden compartirse entre varios productos sin generar errores de compilación.

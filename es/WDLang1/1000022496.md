


## DocReplace (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra todas las instancias de una cadena de caracteres y las reemplaza con otra cadena.: 

- en todo un procesador de textos Document.

- en un fragmento.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Replace all "%CustomerName%" by the customer name
DocReplace(WP_MyDocument, "%CustomerLastName%", Customer.LastName)
```
<a name="Example2"></a>

```wl
// Replace "Presentation" by "PRESENTATION"
DocReplace(WP_NoName1, "Presentation", "PRESENTATION", WholeWord)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DocReplace(<Search scope> , <Text to replace> , <Replacement text> [, <Options>])`
---

**`<Search scope>: (Documento Variable, docFragment Variable o control nombre)`**

Elemento en el que debe realizarse la búsqueda y sustitución. Este elemento puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de un control Procesador de texto.

- una variable de tipo [Document](../WDLang1/1000022461.md). 

- una variable de tipo [docFragment](../WDLang1/1000022482.md).




**`<Text to replace>: (Cadena de caracteres)`**

Texto para encontrar y reemplazar en la Document. Si este parámetro corresponde a una cadena vacía (""), la Document no se modificará.

**`<Replacement text>: (Cadena de caracteres)`**

Cadena de caracteres que reemplazará &lt;Text to replace&gt;&gt;. Si este parámetro corresponde a una cadena vacía ("""), se eliminarán todas las ocurrencias de &lt;Text to replace&gt;.

**`<Options>: (Constante o combinación de constantes opcional)`**

Opciones de búsqueda para el texto a reemplazar:


|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busque la palabra completa (encerrada en caracteres de puntuación o en caracteres espaciados). |


De forma predeterminada, la búsqueda distingue entre mayúsculas y minúsculas: el texto buscado y el texto a encontrar deben tener el mismo caso. Para realizar una búsqueda que no distinga entre mayúsculas y minúsculas, utilice la constante *IgnoreCase*.

Atención: La sustitución no tiene en cuenta la dirección de escritura.

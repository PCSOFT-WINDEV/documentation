
## Belgium.CheckCBE (Función)



<a name="description"></a>
Comprueba la validez de un número CBE belga (Crossroads Bank for Enterprises).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyCBENumber is string = "0436649062"
Belgium.CheckBCE(MyBCENumber)	// Returns True

// Check a control
EDT_CBENumber = "ABC"
Belgium.CheckCBE(EDT_CBENumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Belgium.CheckCBE(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número CBE válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número CBE a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.

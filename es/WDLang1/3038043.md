
## NextTitle( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de la siguiente ventana a abrir.

**Recordatorio**: Por defecto, el título de la ventana es el definido en el editor.












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Modify the title of next window to open
NextTitle("Modifying a customer form")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`NextTitle(<New title>)`
---

**`<New title>: (Cadena de caracteres)`**

Nuevo título de la siguiente ventana o cuadro de diálogo a abrir.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Título de los cuadros de diálogo
<a name="titulo_los_cuadros_dialogo_ELTPARAGRAPHE000090"></a>

Por Default, las ventanas abiertas por [Advertencia](../WDLang1/3021009.md), [Confirme](../WDLang1/3021007.md), [coloquio](../WDLang1/3021015.md), [Error](../WDLang1/3021013.md), [ErrorInfo](../WDLang1/3013008.md), [Info](../WDLang1/3021011.md), [OKCancel](../WDLang1/3021004.md) o [YesNo](../WDLang1/3021005.md) tienen el mismo título que la ventana anterior. La función **NextTitle** permite modificar este título. 


<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Presentación del título
<a name="presentacion_del_titulo_ELTPARAGRAPHE000146"></a>

La [funciones de dibujo](../WDLang1/3029033.md) en los cuadros de lista (que comienzan con la letra "g") se utiliza para:

- añadir dibujos al título,

- modificar el formato del título (fondo color, texto color, etc.) con [gBackground](../WDLang1/3029037.md) y [gPen](../WDLang1/3029039.md).




Por ejemplo: para modificar el texto Color y el fondo Color de título:


```wl
NextTitle(EOT + gBackground(RGB(0, 255, 255)) + gPen(RGB(255, 0, 0)) + "Color test")
```


**Caso especial en Windows Vista y posteriores**: Las ventanas que usan el estilo "Aero sistema" no pueden tener un título gráfico: se ignora la parte gráfica. Si "Aero" no está habilitado, el título del gráfico se mostrará como de costumbre.

### Ventana minimizada
<a name="ventana_minimizada_ELTPARAGRAPHE000173"></a>

Cuando se minimiza una ventana, el título de esta ventana se muestra al lado de la Icon en la barra de tareas.

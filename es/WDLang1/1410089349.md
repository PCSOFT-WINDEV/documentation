
## Belgium.CheckNISS (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número NISS (número de seguridad social en Bélgica).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyNISSNumber is string = "85073003328"
Belgium.CheckNISS(MyNISSNumber)	// Returns True

// Check a control
EDT_NISSNumber = "ABC"
Belgium.CheckNISS(EDT_NISSNumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Belgium.CheckNISS(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número NISS válido, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número NISS a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.

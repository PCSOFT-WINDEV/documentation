


## InVGAMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la aplicación se ejecuta en un dispositivo móvil en modo VGA. 

**Observación**: Esta función solo funciona en WINDEV Mobile (ejecutable o Go en modo simulador). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If we are in VGA mode
IF InVGAMode() = True THEN
	// Open the window intended for the VGA mode
	Open(WIN_VGAMode)
ELSE
	// Open the window intended for the non-VGA mode
	Open(WIN_NonVGAMode)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InVGAMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cumplen las siguientes condiciones: 

	- la aplicación se ejecuta en un móvil VGA.

	- la aplicación se ejecuta en modo VGA (y no en emulación no VGA).




- <u><u><u><u>False</u></u></u></u> en caso contrario.




## &lt;Array&gt;.DeleteLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un fila en una posición determinada:

- de un array bidimensional.

- de la propiedad avanzada array (array de eventos gglCalendar, etc.)




El tamaño del array se reduce automáticamente.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyArray is array of 2 by 3 strings
MyArray.InsertLine(1, "A", "B", "C")
MyArray.InsertLine(2, "a", "b", "c")
MyArray.InsertLine(3, "D", "E", "F")
// Delete the 1st row
MyArray.DeleteLine(1) // Deletes "A B C"
// Display the 1st row
Trace(MyArray[1,1], MyArray[1,2], MyArray[1,3])
// Displays "a b c"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<WLanguage array>.DeleteLine(<Row index>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser bidimensional.

**`<Row index>: (Integro)`**

Índice de la fila a eliminar. Se produce un error WLanguage si este parámetro es mayor que el número de líneas array..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Borrar la última fila
<a name="borrar_ultima_fila_ELTPARAGRAPHE000220"></a>

Si se borra el último Line, el array se redimensiona a 0. No se ha borrado el array: Las nuevas incorporaciones pueden realizarse sin necesidad de volver a crear el array.
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000230"></a>

Esta función no se puede utilizar con la arrays fija.

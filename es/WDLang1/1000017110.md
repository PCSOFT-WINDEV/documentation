
## ServiceStart( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia un servicio. Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts the service found in the TABLE_Service table
IF ServiceStart(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceStart(<Service name> [, <Command line> [, <Remote computer>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se envió la solicitud para iniciar el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a iniciar.

**`<Command line>: (Cadena de caracteres opcional)`**

Ordene a Line que pase al servicio antes de que empiece. Este parámetro no debe especificarse si se utiliza un [servicio creado con WINDEV](../WDLang1/1000017114.md).

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para iniciar un servicio en otro ordenador. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía, el servicio se gestiona localmente..

### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000060"></a>

La función **ServiceStart** envía una solicitud para iniciar un servicio al administrador de servicios. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).




## MenuLabel( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obtiene o cambia la etiqueta de una opción de menú (menú contextual o desplegable).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the label of "OPT_IconBar" 
ResLabel = MenuLabel(OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la etiqueta de una opción de menú

`<Result> = MenuLabel([<Window>, ] <Menu option>)`
---

**`<Result>: (Cadena de caracteres)`**

Etiqueta de la opción de menú actual.

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana que contiene la opción de menú. 
Si no se especifica este parámetro, **MenuLabel** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a utilizar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe ser especificado sin comillas.  


<a name="SYNTAX2"></a>

### Cambiar la etiqueta de una opción de menú

`<Result> = MenuLabel([<Window>, ] <Menu option> , <New caption>)`
---

**`<Result>: (Cadena de caracteres)`**

Antigua etiqueta de opción de menú.

**`<Window>: (Cadena de caracteres opcional)`**

Nombre de la ventana que contiene la opción de menú. 
Si no se especifica este parámetro, **MenuLabel** manipular la ventana actual.

**`<Menu option>: (Opción de menú)`**

Nombre de la opción de menú a utilizar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe especificarse sin comillas.

**`<New caption>: (Cadena de caracteres)`**

Nueva etiqueta de opción de menú. Puede añadir un mensaje de ayuda a la opción de menú. Este parámetro tiene el siguiente formato: "&lt;Nuevo texto&gt;:&lt;Mensaje de ayuda&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Caracteres especiales
<a name="caracteres_especiales_ELTPARAGRAPHE000066"></a>

- Para que el carácter ":" no sea considerado como un separador de ayuda, este carácter ":" debe ir precedido de una barra invertida ("\\").

- El carácter "TAB" permite alinear los siguientes caracteres a la izquierda ("\\t" en el editor de menús).

- El carácter "Char(8)" permite alinear los siguientes caracteres a la derecha (" _ASLASH_a" en el editor de menús).

- El carácter "&" situado delante de una letra se utiliza para describir esta letra como un acceso directo a la opción de menú.


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Los caracteres especiales se ignoran. Los atajos de teclado no son soportados por las opciones de menú

<a name="equivalencia_ELTPARAGRAPHE000080"></a>

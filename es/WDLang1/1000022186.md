


## &lt;Tabla dinámica&gt;.Filter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita un filtro en un control Tabla dinámica. 

Atención: Sólo se puede utilizar un filtro definido en la pestaña "Contenido" de la ventana de description de control Tabla dinámica. Al utilizar la función **&lt;Tabla dinámica&gt;.Filter**, solo se modifica la visualización del control Tabla dinámica, los totales no se actualizan.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Selecting a row of COMBO_Filter_by_family
// Filters the Pivot Table control on the family of products that must be taken into account
PVT_Statistics.Filter(COL_Caption, MySelf)

IF PVT_Statistics.Count = 0 THEN
	STC_NO_DATA.Visible = True
ELSE
	STC_NO_DATA.Visible = False
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Pivot Table control>.Filter(<Display filter> , <Value>)`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Display filter>: (Cadena de caracteres)`**

Nombre del filtro de visualización a utilizar. Este filtro se definió en la pestaña "Contenido" de la ventana de description de control Tabla dinámica.

**`<Value>: (El tipo correspondiente al filtro)`**



- Valor del filtro a implementar. 

- Vaciar la cadena (""") para eliminar el filtro.

## Observaciones


### Implementación
<a name="implementacion_ELTPARAGRAPHE000177"></a>

Para implementar un filtro en una control Tabla dinámica, debe hacerlo:: 

1. Cree la control Tabla dinámica. 

2. En la ventana de description de control Tabla dinámica, en la pestaña "Contenido", añada uno o más filtros de visualización. 

3. Crear un control que permita al usuario seleccionar el valor del filtro. Al seleccionar el valor se ejecutará la función **&lt;Tabla dinámica&gt;.Filter**. 




**Ejemplo**:   

Se define un control Tabla dinámica para visualizar estadísticas de ventas.: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TCD_ExempleFiltre.gif)


Se define un filtro en el título de la familia de productos. El filtro por familia se utiliza para tener en cuenta únicamente la familia especificada en el cálculo de los volúmenes de negocios.. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TCDFiltre.gif)


La ventana propone un cuadro combinado que permite al usuario elegir la familia de productos a filtrar: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TCD_ExempleFiltreExe.gif)

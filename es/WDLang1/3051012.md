
## MCIVolume (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica o devuelve el volumen del sonido.

**Observación**: Si el ordenador actual no tiene tarjeta de sonido, la función no tiene efecto y siempre devuelve 0.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the sound volume for the "WAV" files
ResVolume = MCIVolume("WAV", 200)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar y modificar el volumen del sonido (todos los archivos)

`<Result> = MCIVolume(<Type of files> , <Element> [, <Volume> [, <Parameters> [, <Name of message window>]]])`
---

**`<Result>: (Integer)`**

Corresponde a:

- el volumen de sonido actual.

- el volumen del sonido antes de la modificación.

- 0 si se produce un error.




**`<Type of files>: (Character string)`**

Nombre o alias del archivo o dispositivo a abrir.

**`<Element>: (Constant)`**

Elemento cuyo volumen de sonido debe ser modificado:


|   |   |
| --- | --- |
| *MciRightVolume* | Volumen derecho |
| *MciLeftVolume* | Volumen izquierdo |
| *MciTreble* | Volumen de agudos |
| *MciBass* | Volumen de graves |



**`<Volume>: (Optional integer)`**

Nuevo volumen de sonido para el elemento especificado (incluido entre 0 y 65535).

**`<Parameters>: (Optional character string (with quotes))`**

Parámetros adicionales si es necesario. Estos parámetros se pueden conocer consultando la documentación de la función "Set" de Windows MCI..
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of message window>: (Optional character string (with or without quotes))`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxxxx y se pueden recuperar mediante [Evento](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.  


<a name="SYNTAX2"></a>

### Averiguar y modificar el volumen del sonido (sólo WAV y MID)

`<Result> = MCIVolume(<Type of files> , <New volume> , <New Volume 2>)`
---

**`<Result>: (Integer)`**

Corresponde a:

- el volumen de sonido actual.

- el volumen del sonido antes de la modificación.

- 0 si se produce un error.




**`<Type of files>: (Character string)`**

Tipo de archivos:

- "WAV" para los archivos ".WAV".

- "MID" para los archivos ".MID".

- Nombre o alias del archivo o dispositivo a abrir.




**`<New volume>: (Integer)`**

Nuevo volumen de sonido (incluido entre 0 y 65535). Si no se especifica &lt;Nuevo volumen 2&gt;, corresponde al volumen general del sonido; de lo contrario, corresponde al volumen del sonido izquierdo.. Sólo se tiene en cuenta en los archivos WAV y MID.

**`<New Volume 2>: (Integer)`**

Nuevo volumen de sonido (incluido entre 0 y 65535). Corresponde al volumen de sonido correcto. Sólo se tiene en cuenta en los archivos WAV y MID.

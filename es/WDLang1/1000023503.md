


## WDPic (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **WDPic** permite definir las características avanzadas de una imagen en formato ".wdpic". Las características de la imagen se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

Este tipo se utiliza principalmente para: 

- hacer visibles las capas/Invisible mediante programación ([ImageLayerVisible](../WDLang2/1000023504.md)). 

- Imágenes de manipular en formato WDPic así como sus capas.




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the product image in an Image control
PROCEDURE DisplayProduct(ImageControl is Control, sWDPicName is string, bPromo is boolean)

MyImage is WDPic
MyImage.FileName = sWDPicName
ImageLayerVisible(MyImage, "Promo layer", bPromo) 

ImageControl.Value = MyImage
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables WDPic
<a name="propiedades_especificas_las_variables_wdpic_ELTPARAGRAPHE000046"></a>

Las siguientes propiedades pueden ser usadas para manipular una imagen en formato "WDPic":

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Altura | Integro | La altura de Image expresada en píxeles. <br><br>Por defecto, al asignar una Image al WDPic Variable, corresponde a la altura de la Image.<br><br>Este Property está disponible en modo de lectura y escritura. |
| anchura | Integro | El ancho de Image expresado en píxeles. <br><br>Por defecto, cuando se asigna un Image al WDPic Variable, corresponde al ancho del Image. <br><br>Este Property está disponible en modo de lectura y escritura. |
| Capa | Array de [picLayer](../WDLang1/1000024604.md) | Capas definidas en el "wdpic" Image. Las capas se identifican por sus nombres. |
| FileName | Cadena de caracteres | Nombre del archivo ".WDPic". |
| Image |   | La imagen asociada con la Variable. <br><br>Esta imagen se actualiza, por ejemplo, cuando se asigna un control Imagen al Variable.<br><br>Este Property está disponible en modo de lectura y escritura. |

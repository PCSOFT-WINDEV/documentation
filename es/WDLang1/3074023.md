


## TableInputInProgress (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si una Tabla o control Tabla TreeView está actualmente en edición.

**Observación**: Si el usuario utiliza la Búsqueda Icon en la cabecera de la columna, se considera que la Tabla o control Tabla TreeView está en modo de edición.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The user is not performing an input in the control
IF TableInputInProgress(TABLE_TABLE1) = False THEN
	// Updates the Table control
	TableDisplay(TABLE_Table1)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableInputInProgress(<Table control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control está en modo de entrada (es decir, si el usuario está ingresando información en el control),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Tabla TreeView. 
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Los TreeView controles Tabla no están disponibles.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000097"></a>

Esta función está disponible en: 

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

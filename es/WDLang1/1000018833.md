
## FileToArray( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Rellena una array de clases o estructuras con el contenido de un archivo de datos o consulta HFSQL. Los valores de los miembros de las instancias de las estructuras o clases se rellenan con los valores de los elementos de los registros encontrados en el archivo de datos o en la consulta.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Declare a MyContacts class
MyContacts is class
 	FamilyName is string
 	FirstName is string
 	EmailAddress is string
END

// Load records from the Contacts.fic file
// in an array of objects of MyContacts class
arrContact is array of MyContacts

HOpen(Contacts)
FileToArray(arrContact, Contacts)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`FileToArray(<Array> [, <Data file> [, <Search key>]])`
---

**`<Array>: (Array de estructuras u objetos)`**

Nombre de la array unidimensional en la que se añadirán los elementos encontrados en el archivo de datos o en la consulta. Esta array debe asignarse a una estructura o a una categoría de clase.

**`<Data file>: (Cadena de caracteres)`**

Nombre del archivo o consulta de datos HFSQL utilizado.

**`<Search key>: (Cadena de caracteres)`**

Nombre del elemento de búsqueda utilizado. Este parámetro se utiliza cuando un filtro es definido por [HFilter](../WDLang4/3044100.md). Este parámetro corresponde a la clave de búsqueda óptima a utilizar, devuelta por [HFilter](../WDLang4/3044100.md).







## Observaciones


- La función **FileToArray** vacía completamente el array.

- Los automatic arrays son automatic aliados en función de las necesidades.

- La búsqueda de archivos se realiza respetando el filtro actual en el archivo de datos.

- La posición y los valores del registro actual no se modifican con la función **FileToArray**.

- Para cada Record encontrada en el archivo de datos o en la consulta:

	- para cada miembro de la estructura o de la clase con el mismo nombre, o el mismo atributo "[mapping](../Motscles/1515003.md)", que un elemento del fichero, el valor del elemento se copia en el valor del miembro. 

	- si una estructura o miembro de una clase no tiene ningún elemento con el mismo nombre en el archivo de datos, su valor no se modifica.

	- si un elemento del archivo de datos no tiene ningún miembro con el mismo nombre en la clase, no se realizará ninguna copy de valor.




- Atención: **FileToArray** es una función de bloqueo para todos los hilos.

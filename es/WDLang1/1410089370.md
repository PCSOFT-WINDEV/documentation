


## Funciones para el tratamiento de directorios y ficheros externos (sintaxis de prefijos)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular archivos externos a través de la sintaxis de prefijo:



|   |   |
| --- | --- |
| [&lt;DiskFile Variable&gt;.ChangeSize](../WDLang1/1410089038.md) | Cambia el tamaño de un archivo externo. |
| [&lt;DiskFile Variable&gt;.Close](../WDLang1/1410089043.md) | Cierra un fichero externo. |
| [&lt;DiskFile Variable&gt;.Find](../WDLang1/1410089039.md) | Encuentra una cadena de caracteres o un búfer en un archivo abierto por [fOpen](../WDLang1/3036036.md). |
| [&lt;DiskFile Variable&gt;.Leer](../WDLang1/1410089044.md) | Lee:<br><br>- un bloque de bytes (caracteres) en un archivo externo (ANSI o Unicode),<br><br>- el contenido de un archivo externo (ANSI o Unicode) y lo asigna a una zona de memoria.<br><br><br> |
| [&lt;DiskFile Variable&gt;.Lock](../WDLang1/1410089037.md) | Bloquea total o parcialmente un archivo externo. |
| [&lt;DiskFile Variable&gt;.ReadLineRegularExpression](../WDLang1/1410089046.md) | Lee una línea en un archivo externo, obtiene secciones de esta línea según una expresión regular y las asigna a un Variable. |
| [&lt;DiskFile Variable&gt;.Unlock](../WDLang1/1410089040.md) | Desbloquea total o parcialmente un archivo externo. |
| [&lt;DiskFile Variable&gt;.Write](../WDLang1/1410089041.md) | Escribe:<br><br>- una cadena de caracteres en un fichero externo.<br><br>- una sección de memoria.<br><br><br> |
| [&lt;DiskFile Variable&gt;.WriteLine](../WDLang1/1410089042.md) | Escribe una línea en un archivo de texto (ANSI o UNICODE). |
| [&lt;Variable DiskFile&gt;.FileName](../WDLang1/1410089047.md) | Devuelve el nombre de un archivo externo actualmente abierto. |
| [&lt;Variable DiskFile&gt;.ReadLine](../WDLang1/1410089045.md) | Lee una línea de un archivo externo (ANSI o UNICODE). |
| [&lt;Variable DiskFile&gt;.Seek](../WDLang1/1410089048.md) | Obtiene y establece la posición actual en un archivo externo. |
| [&lt;Variable DiskFile&gt;.Size](../WDLang1/1410089049.md) | Devuelve el tamaño de un archivo (en bytes). |

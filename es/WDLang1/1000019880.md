
## fBuildPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye un fichero completo ruta a partir del directorio y nombre corto de un fichero.. Esta función se utiliza principalmente para get un nombre de archivo válido sin tener que preocuparse por la presencia de separadores en el ruta ('/' o '\\') o por su dirección según la plataforma. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sLog is string
sLog = fBuildPath(fExeDir(), CUSTOMER.NAME, ".log")

Trace(fBuildPath("C:\My documents", "MyFile", "fic"))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fBuildPath(<Path> , <Short name> [, <Extension>])`
---

**`<Result>: (Cadena de caracteres)`**

Ficha válida ruta. 

**`<Path>: (Cadena de caracteres)`**

Directorio de archivos. Este directorio puede incluir caracteres '\\' en el ruta. 

**`<Short name>: (Cadena de caracteres)`**

Denominación breve del fichero. 

**`<Extension>: (Cadena de caracteres opcional)`**

Extensión del archivo. 
Si el carácter '.' no se especifica al principio de la extensión, éste se añade automáticamente. 
Si no se especifica este parámetro, la extensión se considera incluida en el parámetro &lt;Short Name&gt;..

<a name="NOTE0_1"></a>

## Observaciones


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000080"></a>El siguiente código: 


```txt
fBuildPath(<Directory>, <File>, <Extension>)
```


reemplaza el siguiente código: 


```txt
CompleteDir(<Directory>) + <File> + ["."] + <Extension>
```




## ChronoResume (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicia un cronómetro que se detuvo con las funciones [ChronoEnd](../WDLang1/3027032.md) o [ChronoPause](../WDLang1/1000018829.md). La duración cronometrada a partir de la función **ChronoResume** se acumulará a la duración calculada previamente.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ProcessDuration is Duration
ChronoStart()
MyFunction1()
ProcessDuration = ChronoPause()
Info("The MyFunction1 process lasted " + ProcessDuration) 
ChronoResume() 
MyFunction2() 
ProcessDuration = ChronoEnd()
Info("The MyFunction1 and MyFunction2 processes lasted " + ProcessDuration)
```

## Sintaxis

### Reanudar el cronometraje iniciado con un número de cronómetro

`ChronoResume([<Stopwatch number>])`
---

**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro utilizado. Si no se especifica este parámetro, el valor predeterminado es 1.

### Reanudar el cronometraje iniciado con una variable de tipo Chrono

`ChronoResume(<Stopwatch>)`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000183"></a>

 La función **ChronoResume** no tiene ningún efecto si se utiliza en un cronómetro que ya está en marcha. 
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000194"></a>

Si no se llama a la función [ChronoStart](../WDLang1/3027031.md) antes de la función **ChronoResume**, se produce un error WLanguage.
<a name="NOTE0_3"></a>


### Hilos
<a name="hilos_ELTPARAGRAPHE000208"></a>

Las funciones de cronometraje (Chronoxxx) relacionadas con el mismo cronómetro deben utilizarse en el mismo hilo.

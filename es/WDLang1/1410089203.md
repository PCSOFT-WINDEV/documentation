


## SpreadsheetUnmergeCell (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Anula la combinación de la celda especificada o seleccionada en un control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Unmerges cell B2
SpreadsheetUnmergeCell(PSHEET_MySpreadsheet, "B2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetUnmergeCell(<Spreadsheet control> [, <Cell>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Cell>: (Cadena de caracteres opcional)`**

Nombre de una de las celdas del rango de celdas a separar. Este parámetro tiene la siguiente sintaxis: &lt;Column letter&gt;&lt;Row number&gt;. Por ejemplo: "A32".
Si no se especifica este parámetro, se separará la celda seleccionada en el control Hoja de cálculo.

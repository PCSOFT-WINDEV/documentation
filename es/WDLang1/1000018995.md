


## Funciones de gestión de controles TreeMap
			



Las siguientes funciones se utilizan para gestionar los controles del TreeMap: 



|   |   |
| --- | --- |
| [TreeMapAdd](../WDLang1/1000018814.md) | Añade un elemento a una control TreeMap. |
| [TreeMapCount](../WDLang1/1000018928.md) | Devuelve el número de elementos que se encuentran en una control TreeMap. |
| [TreeMapDelete](../WDLang1/1000018903.md) | Borra un elemento de un control TreeMap. |
| [TreeMapDeleteAll](../WDLang1/1000018914.md) | Borra todos los elementos de un control TreeMap. |
| [TreeMapInfoXY](../WDLang1/1000018899.md) | Devuelve el índice del elemento que aparece en una posición determinada en una control TreeMap. |
| [TreeMapSearch](../WDLang1/1000018901.md) | Busca el índice de un elemento en una control TreeMap a partir del elemento ruta. |
| [TreeMapSeek](../WDLang1/1000019739.md) | Busca el índice de un elemento en una control TreeMap a partir del elemento ruta. |
| [TreeMapSelectGroup](../WDLang1/1000018902.md) | Selecciona visualmente un grupo de elementos en un control TreeMap. |

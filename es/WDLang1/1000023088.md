


## DocExitEdit (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sale de la edición actual. La entrada en edición fue realizada por [DocGoIntoEditar](../WDLang1/1000023077.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DocExitEdit(WP_MyDocControl)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DocExitEdit(<Word Processing control>)`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.



<a name="XComponent"></a>

## Componente:
wd280mdl.dll


## Hora de trabajo del programador (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite indicar las horas de trabajo específicas por día en una control Planificador. Este período de tiempo puede ser especificado:

- para la mañana.

- para la tarde.


Los períodos de tiempo se pueden definir por día de la semana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// From 9:00 to 12:00 then from 13:40 to 19:15
SchedulerWorkingHour(SCH_MySchedule, "0900", "1200", "1330", "1915")

// No appointment on Monday morning
SchedulerWorkingHour(SCH_MySchedule, "1330", "1915", "", "", 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerWorkingHour(<Scheduler control> , <Morning start time> , <Morning end time> [, <Afternoon start time> [, <Afternoon end time> [, <Day of week>]]])`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. 

**`<Morning start time>: (Hora o cadena de caracteres)`**

Hora de inicio de la mañana (en formato "HHMM"). 

**`<Morning end time>: (Hora o cadena de caracteres)`**

Hora de finalización de la mañana (en formato "HHMM").

**`<Afternoon start time>: (Hora o Cadena de caracteres opcional)`**

Hora de inicio de la tarde (en formato "HHMM").

**`<Afternoon end time>: (Hora o Cadena de caracteres opcional)`**

Hora de finalización de la tarde (en formato "HHMM"). 

**`<Day of week>: (Entero opcional)`**

Indica el número del día afectado por los horarios especificados.: 

- 1 para el lunes,

- 2 para el martes,

- 3 para el miércoles,

- 4 para el jueves,

- 5 para el viernes,

- 6 para el sábado,

- 7 para el domingo. 


Si no se especifica este parámetro o si se pone a 0, la Definition de horas de trabajo se aplica a todos los días de la semana.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las horas de trabajo están resaltadas por un Color diferente. Este Color se elige en las opciones de estilo del control. 

- **Hora de trabajo del programador** sustituye las horas de trabajo definidas por [WorkingHourStart](../Proprietes/1000018958.md) y [WorkingHourEnd](../Proprietes/1000019056.md).

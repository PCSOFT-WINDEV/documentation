


## Hoja de cálculoBorrarColumna (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una columna de un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes columns B and C from the current worksheet
SpreadsheetDeleteColumn(PSHEET_MySpreadsheet, 2)
SpreadsheetDeleteColumn(PSHEET_MySpreadsheet, "C")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetDeleteColumn(<Spreadsheet control> , <Column> [, <Worksheet>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.  

**`<Column>: (Entero o cadena de caracteres)`**

Columna a borrar. Este parámetro puede corresponder a: 

- el número de columna, por ejemplo: 1. 
	Los números de columna comienzan desde 1

- el nombre de columna, "A" por ejemplo. 


La función no tiene ningún efecto si este parámetro no es válido.. 

**`<Worksheet>: (Entero opcional)`**

Número de la hoja de trabajo a utilizar. Si no se especifica este parámetro, se utiliza la hoja de trabajo actual. 
La función no tiene ningún efecto si este parámetro no es válido.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Después de la ejecución de la función: 

- las columnas que se encuentran a la derecha de la columna eliminada se desplazan a la izquierda. 

- el contenido de las expresiones cambia de nombre.

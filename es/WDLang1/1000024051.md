


## &lt;Tipo Date&gt;.ToMonthInAlpha (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del mes de una fecha determinada.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDate is Date = DateSys()
EDT_Date = MyDate.ToDayInAlpha() + " " + MyDate.Day + ...
	" " + MyDate.ToMonthInAlpha() + " at " + ...
	TimeToString(TimeSys(), "HH:MM:SS")
// The EDT_Date control contains for example: Monday 13 August at 14:44:17
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Date>.ToMonthInAlpha()`
---

**`<Result>: (Cadena de caracteres)`**

Mes que corresponde a la fecha especificada: Enero, febrero, marzo, abril, mayo, junio, julio, agosto, septiembre, octubre, noviembre, diciembre.
**Observación**: El mes devuelto depende de las opciones de idioma especificadas para las fechas en la pestaña "Idiomas" de la descripción del proyecto (en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción"):

- Si el proyecto utiliza las opciones de idioma de Windows para las fechas, el mes será devuelto en el idioma de Windows.

- Si el proyecto utiliza parámetros específicos para las fechas (en uno o más idiomas), el mes devuelto corresponderá al que se especificó en la descripción del proyecto.




**`<Date>: (Variable de tipo Date o DateTIme)`**

Fecha a manipular. Esta fecha puede corresponder a : 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999.

Los cálculos sobre las fechas realizadas con las [funciones WLanguage](../WDLang1/1000023943.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.


## Cos( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el coseno de un ángulo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the cosine of 60
ResCosine = Cos(60)
// Returns 0.5
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Cos(<Angle>)`
---

**`<Result>: (Real)`**

Coseno del ángulo especificado.

**`<Angle>: (Entero o real)`**

Ángulo (en grados) de uso. Este parámetro puede tener cualquier valor: el cálculo del coseno se realiza "modulo 360"..


## ControlInfoXY( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de la control situada en una posición determinada. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click on [HOME]

// Find the control in the upper-left corner of the window
sControl is string
sControl = ControlInfoXY(10, 10, WIN_Management)
IF sControl <> "" THEN
	SetFocus(sControl)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ControlInfoXY(<X> , <Y> [, <Parent>])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre de la control a encontrar. 

**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. 

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. 

**`<Parent>: (Cadena de caracteres opcional)`**

Nombre del padre del elemento buscado. Este parámetro puede corresponder al nombre de una control o ventana. 

- Si no se especifica este parámetro, las coordenadas X e Y son relativas a la pantalla. 

- Si este parámetro corresponde al nombre de una ventana, las coordenadas X e Y son relativas al área del cliente de la ventana. 

- Si este parámetro corresponde al nombre de una control, las coordenadas X e Y son relativas a la zona cliente de la control. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los controles de Invisible no se devuelven. 

- Si se especifica &lt;Parent&gt;, los controles que se encuentran en ubicaciones fuera del área de cliente no se devuelven..

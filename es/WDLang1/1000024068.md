
## Funciones de gestión de entradas asistidas (sintaxis de prefijos)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar la entrada asistida en los controles de edición (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;AddAssistedInput](../WDLang1/1000023671.md) | Añade una fila a la lista de valores disponibles en las sugerencias de autocompletar de un Editar control. |
| [&lt;CloseAssistedInput](../WDLang1/1000023946.md) | Cierra la lista de valores propuestos por la entrada asistida para una control Campo de entrada. |
| [&lt;ConfigureAssistedInput](../WDLang1/1000024064.md) | Permite configurar el filtro y la apertura de la entrada asistida. |
| [&lt;Edición&gt;.DeleteAllAssistedInput](../WDLang1/1000024065.md) | Borra la lista de valores propuestos para la entrada asistida de una control Campo de entrada. |
| [&lt;OpenAssistedInput](../WDLang1/1000024063.md) | Abre la lista de valores propuestos por la entrada asistida para una control Campo de entrada. |

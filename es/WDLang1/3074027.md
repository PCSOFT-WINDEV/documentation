


## TableRecNum (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la Record actual en una control Tabla basada en un archivo de datos.

**Observación**: La función **TableRecNum** puede utilizarse en un control Tabla de selección única o múltiple.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Current record number in the "TABLE_ProductTable" control
ResRecNum = TableRecNum(TABLE_ProductTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableRecNum(<Table control>)`
---

**`<Result>: (Integro)`**



- El número actual de Record en el archivo de datos relacionado al control Tabla,

- 0 si no hay ningún fila actual o si el fila actual no está guardado todavía en el archivo de datos relacionado al control Tabla.




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.
Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control Tabla al que pertenece el Event actual.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la consulta se ejecutó con la constante *hConFiltro* (opción marcada en el editor de consultas), **TableRecNum** devuelve el número del Record actual en la consulta.

- Si la consulta se ejecutó sin la constante *hWithFilter*, **TableRecNum** devuelve un resultado indefinido.

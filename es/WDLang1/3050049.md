
## MatAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade dos matrices de las mismas dimensiones (el mismo número de filas y el mismo número de columnas).
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Matadd0.gif)





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add two matrices
ResAdd = MatAdd("MyMatrix1", "MyMatrix2", "MyResultMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatAdd(<1st matrix> , <2nd matrix> , <Result matrix>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la suma, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<1st matrix>: (Cadena de caracteres)`**

Nombre de la primera matriz a añadir, definida por [MatCreate](../WDLang1/3050058.md).

**`<2nd matrix>: (Cadena de caracteres)`**

Nombre de la segunda matriz a añadir, definida por [MatCreate](../WDLang1/3050058.md).

**`<Result matrix>: (Cadena de caracteres)`**

Nombre de la matriz que contiene el resultado de la suma. Esta matriz se crea automáticamente cuando se añaden las dos matrices..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz de resultados
<a name="existencia_matriz_resultados_ELTPARAGRAPHE000096"></a>

No se verifica la existencia de la matriz.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Matrices idénticas
<a name="matrices_identicas_ELTPARAGRAPHE000107"></a>

Si &lt;1a matriz&gt; es idéntica a &lt;2a matriz&gt;, la matriz de resultados contiene el doble de la matriz fuente.
<a name="NOTE0_3"></a>




### Matrices de resta
<a name="matrices_resta_ELTPARAGRAPHE000115"></a>

Para restar una matriz de otra matriz, debe hacerlo:

- multiplicar la matriz a restar por -1 con [MatFloatMultiply](../WDLang1/3050040.md),

- añadir esta matriz a la primera matriz.

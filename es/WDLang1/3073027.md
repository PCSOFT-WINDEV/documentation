
## SysIconModify( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el archivo Icon y/o el texto hover del Icon en la barra de tareas (Corner inferior derecha de la pantalla). Esta Icon fue añadida en la barra de tareas por [SysIconAdd](../WDLang1/3073001.md).

![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Icone.gif) o ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconePPC.gif)

**Observación**: El modo de funcionamiento del icono (menú emergente, llamada a un Procedure) no se modifica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the hover text of the icon
// that was previously added into the taskbar
ResModifyIcon = SysIconModify("", "Comparison of sales")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysIconModify(<Icon> [, <Hover text>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el ícono en la barra de tareas,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Icon>: (Cadena de caracteres o Icono Variable)`**

Corresponde a uno de los siguientes elementos: 

- Nombre y ruta completo (o relativo) del archivo Icon para agregar. Se permite el uso de un horario UNC ruta.

- Nombre de una variable de tipo [Icon](../WDLang1/1000020811.md). 


Si este parámetro corresponde a una cadena vacía (""), el archivo Icon no se modifica.

**`<Hover text>: (Cadena de caracteres opcional)`**

Texto que aparece al pasar el ratón por encima de la Icon. 
Si este parámetro no se especifica o es una cadena vacía (""), el texto que se muestra por encima no cambia.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un solo Icon puede ser añadido a una aplicación. No se pueden añadir varios.

- [SysIconAdd](../WDLang1/3073001.md) se utiliza para añadir un icon.

- [SysIconBorrar](../WDLang1/3073022.md) se utiliza para borrar el icon. El Icon se borra automáticamente al cerrar la aplicación.

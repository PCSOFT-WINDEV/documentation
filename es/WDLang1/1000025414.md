
## &lt;Array&gt;.InsertReference (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega la referencia de un objeto en la posición dada en un array unidimensional.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AnArray is array of AClass
AnObject is AClass
AnArray.InsertReference(InsertionIndex, AnObject)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<WLanguage array>.InsertReference(<Index> , <Object>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Index>: (Integro)`**

Índice donde se insertará el elemento en la array.

**`<Object>: (Objeto)`**

Nombre del objeto cuya referencia se agrega al array.


## ControlExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un control (o grupo de control s) en una ventana, en una página o en un reporte.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the existence of the "EditName" control
ResExist = ControlExist("EditName")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlExist(<Control name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control (o grupo de control s) existe, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Control name>: (Cadena de caracteres)`**

Nombre del control que debe ser marcado **entre comillas**. Este parámetro también puede corresponder a un grupo de controles. 
Para comprobar la existencia de control s en otras ventanas, páginas o informes, &lt;Nombre de control&gt; debe tener el siguiente formato:
"&lt;Nombre de la ventana, página o Reporte&gt;.&lt;Nombre de control&gt;".

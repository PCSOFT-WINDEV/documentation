


## Funciones Calendario
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para manipular, los controles Calendar y los controles de edición de fecha:



|   |   |
| --- | --- |
| [CalendarioAbrir](../WDLang1/1000017291.md) | En una fecha control Campo de entrada, abre una ventana emergente que muestra una Calendar. |
| [CalendarioPosición](../WDLang1/1000017281.md) | Devuelve el mes mostrado en un control Calendario. |
| [CalendarioSelectCount](../WDLang1/1000017282.md) | Devuelve el número de días seleccionados en una control Calendario. |
| [CalendarioSelectMinus](../WDLang1/1000017340.md) | Desmarca una fecha en una control Calendario multiselección. |
| [CalendarioSelectPlus](../WDLang1/1000017339.md) | Selecciona una fecha en una control Calendario. |
| [CalendarPicker](../WDLang1/1000017300.md) | Abre un diálogo modal con una Calendar para introducir una fecha. |
| [CalendarSelect](../WDLang1/1000017283.md) | Devuelve la fecha seleccionada en un control Calendario. |


## AnimationGetParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee el valor de uno de los parámetros de animación definidos para una Image o control Estático.



![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En esta versión, solo se puede leer el valor de los parámetros de animación definidos para un control Estático.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
If AnimationGetParameter(IMG_Image1, acpLoop) = True THEN
	AnimationSetParameter(IMG_Image1, acpLoop, False)
END
```


<a name="Example2"></a>


![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
IF AnimationGetParameter(STC_CustomerNameSta, acpColor) = Transparent THEN
	AnimationSetParameter(STC_CustomerNameSta, acpColor, LightYellow)
END
```

## Sintaxis
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Leyendo los parámetros de animación de una Image (control Imagen)

`<Result> = AnimationGetParameter(<Image control> , <Parameter name>)`
---

**`<Result>: (Tipo de parámetro esperado)`**

Valor del parámetro buscado.

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<Parameter name>: (Constante o cadena de caracteres)`**

Nombre del parámetro a leer.


|   |   |
| --- | --- |
| *acpDirection* (o "Direction") | Dirección de desplazamiento. <br>**Animación de tipo "Desplazamiento" únicamente.**<br>Estos parámetros se pueden combinar.<br>1 (por defecto): de izquierda a derecha<br>2: de derecha a izquierda<br>4: de abajo a arriba<br>8: de arriba a abajo<br><br>Ejemplo de combinación: <br>3 (2+1): izquierda derecha: ampliar progresivamente desde el centro <br>5 (4+1) de abajo a arriba y de derecha a izquierda<br><br> |
| *acpDirectionAlternate* (o "DirecciónAlternativa") | Lee los dibujos de la imagen desde el primero hasta el último, y luego desde el último hasta el primero de forma continua (este parámetro es set a <u><u><u><u>False</u></u></u></u> por Default). |
| *acpDuration* (o "Duration") | Tiempo durante el cual se muestra cada dibujo animado en milisegundos (300 ms por defecto). |
| *acpLoop* (o "Loop") | Leer los dibujos de la imagen desde el primero hasta el último de forma continua (este parámetro es set a <u><u><u><u>True</u></u></u></u> por Default). |
| *acpNbImageX* (o "NumberOfImagesX") | Número de dibujos a lo ancho del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br>**Animación de tipo "Imagen a animar" únicamente.**<br><br> |
| *acpNbImageY* (o "NumberOfImagesY") | Número de dibujos a lo alto del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br>**Animación de tipo "Imagen a animar" únicamente.**<br><br> |
| *acpNbStep* (o "NumberOfSteps") | 10 por defecto<br>**Animación de tipo "Desplazamiento" únicamente.**<br><br> |
| *acpReverse* (o "Reverse") | Lee los dibujos de la imagen desde el último (abajo a la derecha) hasta el primero (arriba a la izquierda) (este parámetro es de set a <u><u><u><u>False</u></u></u></u> por Default). |

### Lectura de los parámetros de animación de un texto estático (control Estático)

`<Result> = AnimationGetParameter(<Static control> , <Parameter name>)`
---

**`<Result>: (Tipo de parámetro esperado)`**

Valor del parámetro buscado.

**`<Static control>: (Nombre del control)`**

Nombre de la control Estático a utilizar.

**`<Parameter name>: (Constante o cadena de caracteres)`**

Nombre del parámetro a leer.


|   |   |
| --- | --- |
| *acpColor* (o "Color") | Color parpadeante. Este parámetro puede corresponder a una [constante de color](../WDLang5/3010002.md). Por defecto, este color corresponde a *Transparent*. <br>** Sólo animación"Parpadeante"** |
| *acpDirection* (o "Direction") | Dirección de desplazamiento.<br> **Animación de tipo "Desplazamiento" únicamente.**<br>Estos parámetros no se pueden combinar.<br>1 (por defecto): de izquierda a derecha<br>2: de derecha a izquierda<br>4: de abajo a arriba<br>8: de arriba a abajo |
| *acpDuration* (o "Duration") | Tiempo durante el cual se muestra cada elemento animado en milisegundos (300 ms por defecto) |
| *acpNbPixel* (o "NumberOfPixels") | Número de pixeles para cada intervalo de desplazamiento. <br>**Animación de tipo "Desplazamiento" únicamente.** |

### control Imagen: Parámetros disponibles y valores
<a name="control_imagen_parametros_disponibles_valores_ELTPARAGRAPHE000198"></a>

Hay dos tipos de animaciones disponibles para los controles de Image:

- Imagen a animar (animación realizada por WINDEV o WINDEV Mobile), 

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Scrolling.





|   |   |
| --- | --- |
| **Parámetros comunes a las diferentes animaciones** |
| presilla | Leer los dibujos de la imagen desde el primero hasta el último de forma continua (este parámetro es set a <u><u><u><u>True</u></u></u></u> por Default). |
| Invertir | Leer los dibujos de la imagen desde el último (abajo a la derecha) hasta el primero (arriba a la izquierda) (este parámetro es set a <u><u><u><u>False</u></u></u></u> por Default). |
| DirectionAlternate | Lee los dibujos de la imagen desde el primero hasta el último, y luego desde el último hasta el primero de forma continua (este parámetro es set a <u><u><u><u>False</u></u></u></u> por Default). |
| Duración | Tiempo durante el cual se muestra cada dibujo animado en milisegundos (300 ms por defecto). |
| **Parámetro específico a "Imagen a animar"** |
| NumberOfImagesX | Número de dibujos a lo ancho del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br><br> |
| NumberOfImagesY | Número de dibujos a lo alto del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br><br> |
| **Parámetro específico a "Desplazamiento"** |
| Dirección | Dirección de desplazamiento. Estos parámetros se pueden combinar.<br>1 (por defecto): de izquierda a derecha<br>2: de derecha a izquierda<br>4: de abajo a arriba<br>8: de arriba a abajo<br><br>Ejemplo de combinación:<br>3 (2+1): izquierda derecha: ampliar progresivamente desde el centro<br>5 (4+1) de abajo a arriba y de derecha a izquierda |
| NumberOfSteps | 10 por defecto. |

### Control Estático: Parámetros disponibles y valores
<a name="control_estatico_parametros_disponibles_valores_ELTPARAGRAPHE000285"></a>

Hay dos tipos de animaciones disponibles para los controles estáticos:

- Parpadeo, 

- Scrolling.





|   |   |
| --- | --- |
| **Parámetros comunes a las diferentes animaciones.** |
| Duración | Tiempo durante el cual se muestra cada dibujo animado en milisegundos (300 ms por defecto). |
| **Parámetro específico a "Parpadeo"** |
| Color | Color parpadeante. Este parámetro puede corresponder a una [constante de color](../WDLang5/3010002.md).<br>Por defecto, este color corresponde a *Transparent*. |
| **Parámetro específico a "Desplazamiento"** |
| NumberOfPixels | Número de pixeles para cada intervalo de desplazamiento. |
| Dirección | Dirección de desplazamiento. Estos parámetros no se pueden combinar.<br>1 (por defecto): de izquierda a derecha<br>2: de derecha a izquierda<br>4: de abajo a arriba<br>8: de arriba a abajo. |

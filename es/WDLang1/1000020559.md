
## IWListDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una ventana interna que se encuentra en la lista de ventanas internas buscadas por un control Ventana Interna.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the first internal window added into the control
IWListDelete(IW_InternalWindowControl, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = IWListDelete(<Internal Window control> [, <Position>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó la ventana interna,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.

**`<Position>: (Entero opcional)`**

Posición de la ventana interna a borrar. Si no se especifica este parámetro, se borrará la ventana interna que se muestra actualmente.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la ventana interna borrada corresponde a la ventana interna que se muestra actualmente en la control, se mostrará la siguiente ventana interna (o la anterior si la ventana interna borrada fue la última). 

- Cuando se elimina una ventana interna de la lista de ventanas internas, las ventanas internas que siguen a la ventana eliminada se renumeran..

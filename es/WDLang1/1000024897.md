
## DocUpdateFormula (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza las fórmulas de un documento. Una fórmula puede estar en: 

- una variable de tipo Document,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Procesador de texto,

- una variable de tipo docFragment. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document
...

DocUpdateFormula(MyDocument)
```





<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Actualizar las fórmulas de una variable Document

`DocUpdateFormula(<Document>)`
---

**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Actualizar las fórmulas de un control Procesador de texto

`DocUpdateFormula(<Word Processing control>)`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.


<a name="SYNTAX3"></a>

### Actualizar las fórmulas de una variable docFragment

`DocUpdateFormula(<Fragment>)`
---

**`<Fragment>: (Variable de tipo DocFragment)`**

Nombre de la variable de tipo [docFragment](../WDLang1/1000022482.md) que contiene las fórmulas.

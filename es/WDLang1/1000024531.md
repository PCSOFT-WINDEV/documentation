
## StringToJSON (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena JSON en una variable de tipo JSON.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sJZon is ANSI string = [
{
"glossary": {
"title": "example glossary",
"GlossDiv": {
"title": "S",
"GlossList": {
"GlossEntry": {
"ID": "SGML",
"SortAs": "SGML",
"GlossTerm": "Standard Generalized Markup Language",
"Acronym": "SGML",
"Abbrev": "ISO 8879:1986",
"GlossDef": {
"para": "A meta-markup language, used to create markup languages such as DocBook.",
"GlossSeeAlso": ["GML", "XML"]
},
"GlossSee": "markup"
}
}
}
}
}
]

jz is JSON = StringToJSON(sJZon)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = StringToJSON(<JSON string>)`
---

**`<Result>: (Variable de tipo JSON)`**

Variable de tipo [JSON](../Motscles/1000023611.md) que contiene el resultado de la conversión.
Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) permite conocer los detalles del error.

**`<JSON string>: (Cadena de caracteres)`**

Cadena que contiene los datos en formato JSON.

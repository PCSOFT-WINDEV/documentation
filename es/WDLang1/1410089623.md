


## &lt;Kanban&gt;.MoveList (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve una lista en un control Kanban.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Moves the LIST_To_Review list to position 2
KANBAN_Planning.MoveList(LIST_To_Review.ListIndex(), 2)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Kanban control>.MoveList(<Position before moving> , <Position after moving>)`
---

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<Position before moving>: (Entero)`**

Posición de la lista Kanban antes de moverla. Esta posición es devuelta por [&lt;Lista Kanban&gt;.ListIndex](../WDLang1/1410089485.md).

**`<Position after moving>: (Entero)`**

Nueva posición de la lista Kanban. 

- Si esta posición es menor o igual a 1, la lista Kanban se mueve a la primera posición. 

- Si esta posición es mayor o igual que el número de listas en el control Kanban, la lista se mueve a la última posición. 

- Si &lt;Position before moving&gt; = &lt;Position after moving&gt;, la lista permanece en la misma posición.

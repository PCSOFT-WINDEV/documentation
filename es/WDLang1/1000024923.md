


## &lt;Variable WDPic&gt;.Save (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda la imagen encontrada en una Variable de tipo [WDPic](../WDLang1/1000023503.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic
...
// Asks to save the image
IF MyWDPicImage.Save("C:\temp\test.png") = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <WDPic image>.Save(<File>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario (error, cancelación del selector de archivos, etc.). Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que contiene la imagen a manipular.

**`<File>: (Cadena de caracteres)`**

Nombre y ruta del archivo Image a guardar. La extensión del archivo define el tipo de Image a guardar.




## &lt;Panel acoplable&gt;.Activar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa un panel cuando varios paneles están acoplados entre sí..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Array to know the number of errors
arrErrors is array of int
... 
// Force the display of the error panel if there is at least 1 error
IF arrErrors.Count > 0 THEN
	DOPA_Errors.Activate()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Dockable Panel control>.Activate()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró el panel,

- <u><u><u><u>False</u></u></u></u> en caso contrario (el panel ya está activo o no está acoplado con otros paneles).




**`<Dockable Panel control>: (Nombre del control)`**

control Panel acoplable a utilizar.

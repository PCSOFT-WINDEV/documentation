
## DnDEvent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la Procedure a ejecutar durante un Event de Arrastre/Suelta.

**Observación**: Para determinar el comportamiento de un control desde el código cuando se arrastra y se suelta, utilice las propiedades [DndTarget](../Proprietes/2510016.md) y [DndSource](../Proprietes/2510014.md) con la constante *dndProgram*.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Call the OnDrop procedure when the left mouse button 
// is released on the target control named "EDT_Find"
DnDEvent("OnDrop", EDT_Find, dndDrop)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`DnDEvent(<Procedure> , <Control name> , <Type of event>)`
---

**`<Procedure>: (Cadena de caracteres)`**

Nombre del Procedure que debe ser llamado durante el &lt;Tipo de Event&gt; Event en el &lt;Nombre del control&gt; control.

**`<Control name>: (Cadena de caracteres)`**

Nombre del control en el que debe procesarse el Event.

**`<Type of event>: (Constante)`**

Tipo de evento que desencadena la llamada al &lt;Procedimiento&gt;:


|   |   |
| --- | --- |
| *dndBeginDrag* | Inicio de un Drag/Drop en una fuente control. |
| *dndDragEnter* | Entrada en un objetivo control. |
| *dndDragLeave* | Salida de un objetivo control. |
| *dndDragOver* | Coloca un blanco, control. |
| *dndDrop* | Suelte el botón izquierdo del ratón en un objetivo control. |
| *dndEndDrag* | Fin de arrastrar / soltar. |

## Observaciones




### Variables
<a name="variables_ELTPARAGRAPHE000078"></a>

Se pueden utilizar las siguientes variables:

| Nombre de la variable | description |
| --- | --- |
| **Acción _DND.acción** | Acción especificada en [DnDAccepto](../WDLang1/3030005.md).<br><br>Los valores posibles son: *dndCopy*, *dndMove* y *dndNone*.<br><br>Esta variable no se asigna al principio de la acción Arrastrar y soltar en un control fuente (constante *dndBeginDrag*) o al salir de un control de destino(constante *dndDragLeave*). |
| **DND.TargetControl** | Nombre del objetivo control.<br><br>Esta variable no se asigna al principio de la acción Arrastrar y soltar en un control fuente (constante *dndBeginDrag*) o al salir de un control de destino(constante *dndDragLeave*). |
| **DND.SourceControl** | Nombre de la fuente control. |
| **CtrlDown** | Estado de la tecla Ctrl:<br><br>- True: se pulsa la tecla Ctrl.<br><br>- False: la tecla Ctrl no está pulsada.<br><br><br>Esta variable no se asigna al principio de la acción Arrastrar y soltar en un control fuente (constante *dndBeginDrag*) o al salir de un control de destino(constante *dndDragLeave*). |
| **DND.SourceWin** | Nombre de la ventana fuente.<br><br>Esta variable no se especifica al salir de un control de destino (constante *dndDragLeave*). |
| **DND.MouseXPos** | Posición horizontal (X) del cursor del ratón en relación con el control manejado durante la Event.<br><br>Esta variable no se especifica al salir de un control de destino (constante *dndDragLeave*). |
| **DND.MouseYPos** | Posición vertical (Y) del cursor del ratón en relación con el control manejado durante la Event.<br><br>Esta variable no se especifica al salir de un control de destino (constante *dndDragLeave*). |


El **DND.SourceControl** y **DND.SourceWin** devuelven una cadena vacía (""") cuando la función de arrastrar y soltar proviene de una aplicación distinta a la actual.

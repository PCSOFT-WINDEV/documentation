


## &lt;Procesamiento de palabras&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve uno de los siguientes valores para una posición determinada en un control Procesador de texto (coordenadas de un punto del control):

- el número de la página correspondiente,

- la posición X correspondiente (en mm),

- la posición Y correspondiente (en mm).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Page number
nPage is int
nPage = WP_MyDocument.InfoXY(docPageNumber, MouseXPos(), MouseYPos())
Trace("Page number: ", nPage)

// X position
rXPos is real
rXPos = WP_MyDocument.InfoXY(docXCoord, MouseXPos(), MouseYPos())
Trace("X position in mm: ", rXPos)

// Y position
rYPos is real
rYPos = WP_MyDocument.InfoXY(docYCoord, MouseXPos(), MouseYPos())
Trace("Y position in mm: ", rYPos)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.InfoXY(<Type of information> , <X> , <Y>)`
---

**`<Result>: (Real)`**

Información solicitada.

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 

Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *docOriginScreen* | Por Default, (0,0) corresponde a fila 1, columna 1 de la control Procesador de texto (barras de desplazamiento ubicadas en el origen).<br>Si la constante *docOriginScreen* se combina con las constantes anteriores, (0,0) corresponde al origen de la pantalla.<br>Esta constante no puede utilizarse por sí sola. |
| *docPageNumber* | Número de la página que contiene la posición especificada. |
| *docXCoord* | Coordenadas X de la posición especificada en la página (expresada en milímetros). |
| *docYCoord* | Coordenada Y de la posición especificada en la página (expresada en milímetros). |



**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. Esta coordenada es relativa al control (si no se especifica la constante *docOriginScreen*).

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada es relativa al control (si no se especifica la constante *docOriginScreen*).

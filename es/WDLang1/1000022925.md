


## docRow (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docRow** permite definir las características avanzadas de una fila de tabla de un documento de tipo Procesador de texto. Las características de esta tabla fila pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no debe utilizarse directamente.** Sólo debe utilizarse a través de las variables [docParagraph](../WDLang1/1000022483.md) correspondientes a un tabla. 


<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la description de las variables docRow
<a name="propiedades_especificas_description_las_variables_docrow_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docRow**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Altura | Integro | Altura de fila (en milímetros). |
| Modo Altura | Constante de tipo Integer | Modo de cálculo de la altura fila: <br><br>- *docHeightModeAtLeast*: La altura fila será como mínimo la definida con el **Altura** Property. <br><br>- *docHeightModeAutomatic*: La altura de fila se calcula automáticamente en función del texto de las celdas de fila. <br><br>- *docHeightModeExact*: La altura fila será la definida con la altura **** Property. <br><br>- *docHeightModeUndefined*: El modo de cálculo de la altura de fila no está definido (el modo de cálculo de automatic lo realiza Default). <br><br><br> |
| Recuento de células | Integro | Número de células fila. <br><br>Esta propiedad es de solo lectura. |

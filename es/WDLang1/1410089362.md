
## InPageWithinWindowMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si la página se muestra en un control Página WEBDEV en una ventana. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InPageWithinWindowMode() THEN
	MyHostControl.ProcedureOfHostControl()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InPageWithinWindowMode()`
---

**`<Result>: (Boolean)`**



- <u><u><u><u>True</u></u></u></u> si la página se utiliza en un control Página WEBDEV en una ventana, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

<a name="disponibilidad_funcion_ELTPARAGRAPHE000164"></a>

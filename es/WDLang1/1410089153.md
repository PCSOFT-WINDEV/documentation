
## MarkdownToDOCX (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena Markdown en un archivo DOCX.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document
MyInitialString is string = "*Italic text*"

MyDocument = MarkdownToDOCX(MyInitialString)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MarkdownToDOCX(<Markdown string>)`
---

**`<Result>: (Variable de tipo Document)`**

Variable de tipo [Document](../WDLang1/1000022461.md) que contiene el resultado de la conversión. 

**`<Markdown string>: (Cadena de caracteres)`**

Cadena Markdown a manipular.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El tamaño de página es el tamaño predeterminado de una variable de tipo [Document](../WDLang1/1000022461.md) (tamaño A4).

- Si el texto Markdown contiene: 

	- texto sin etiquetas Markdown, se aplicará el estilo predeterminado de las variables de tipo [Document](../WDLang1/1000022461.md) (estilo "Normal"). 

	- texto con etiquetas Markdown, se aplicará el estilo de texto predeterminado y, a continuación, los atributos del texto Markdown correspondiente (negrita, subrayado, etc.).




- El texto Markdown entre etiquetas de estilo (Título nivel 1, Título nivel 2, etc.) tendrá un estilo equivalente ("Título 1", "Título 2") en el documento resultante.

- Puede editar todas las características del documento resultante (formato de página, estilo, etc.) utilizando las funciones y propiedades asociadas a las variables de tipo [Document](../WDLang1/1000022461.md).

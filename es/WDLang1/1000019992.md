
## Modo de programación (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el modo de visualización de un control Planificador: día, semana, mes, ....

**Observación**: [SchedulerChangeMode](../WDLang1/1000019440.md) se utiliza para modificar el modo de visualización de control Planificador. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nMode is int = SchedulerMode(SCH_MySchedule)
IF nMode = schModeDay THEN
	SCH_MyScheduler..GranularityAppointment = 15 // 15 minutes in Day mode
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SchedulerMode(<Scheduler control>)`
---

**`<Result>: (Integro)`**

El tiempo interval mostrado por el control Planificador: 


|   |   |
| --- | --- |
| *schModeAccordingApt* | Programa que muestra un número de días que le permite ver todas las citas disponibles.<br><br> |
| *schModeCustomNbDay* | Programe para un número específico de días. |
| *schModeDay* | Horario para un día. |
| *schModeMonth* | Calendario para un mes. |
| *schModeWeek* | Horario para una semana de 5 días o para una semana de 7 días. |



**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a:

- el control Planificador de una ventana,

- la control Planificador de un reporte.


## RegistryExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de una clave en el registro de Windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Checks the existence of the "HKEY_LOCAL_MACHINE\SOFTWARE\MyApp" key
ResExist = RegistryExist("HKEY_LOCAL_MACHINE\SOFTWARE\MyApp")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Comprobar la existencia de una clave o valor clave

`<Result> = RegistryExist([<Access mode>, ] <Key path> [, <Value name>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe la clave o el valor especificado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Key path>: (Cadena de caracteres)`**

Ruta de la clave a comprobar.

**`<Value name>: (Cadena de caracteres opcional)`**

Nombre del valor a verificar. Si no se especifica este parámetro, el valor comprobado será el valor "&lt;Default&gt;".. Para comprobar el valor "&lt;Default&gt;", &lt;Nombre del valor&gt; debe ser "0".  


<a name="SYNTAX2"></a>

### Comprobación de la existencia de un valor clave identificado por su índice

`<Result> = RegistryExist([<Access mode>, ] <Key path> , <Value index>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el valor especificado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Key path>: (Cadena de caracteres)`**

Ruta de la clave a comprobar.

**`<Value index>: (Integro)`**

Índice del valor a comprobar. Si no se especifica este parámetro, el valor comprobado será el valor "&lt;Default&gt;".. Para comprobar el valor "&lt;Default&gt;", &lt;Índice de valor&gt; debe ser "0".

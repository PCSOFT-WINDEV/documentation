
## OrganizerChangeZoom( Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [OrganizerChangeMode](../WDLang1/1000019497.md).



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el modo de visualización de un control Agenda. El intervalo de tiempo mostrado puede ser el día, la semana o el mes.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the first four days of the week
OrganizerChangeZoom(ORG_MyOrganizer, orgzWeek, 4)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`OrganizerChangeZoom(<Organizer control> , <Display range> [, <Number of days>])`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de control Agenda. Este control puede corresponder a: 

- una control Agenda en una ventana o en una página,

- una control Agenda en una reporte.




**`<Display range>: (Constante de tipo Integer)`**

Rango de tiempo que debe mostrarse en el control Agenda:


|   |   |
| --- | --- |
| *orgzDay* | Muestra la agenda de un día. |
| *orgzMonth* | Muestra la agenda de un mes. |
| *orgzWeek* | Muestra el organizador en el número de días especificado en &lt;Número de días&gt;. |
| *orgzWeekOn5* | Mostrar el organizador en 5 días (no se muestra el fin de semana). |



**`<Number of days>: (Entero opcional)`**

Número de días mostrados en la semana. Si no se especifica este parámetro, el organizador se mostrará en una semana completa.. Este parámetro solo se tiene en cuenta si el parámetro &lt;Intervalo de visualización&gt; es igual a *orgzWeek*.

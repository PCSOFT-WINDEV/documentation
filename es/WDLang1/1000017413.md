


## TablaRestaurarPosición YSelección (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la posición de la barra de desplazamiento en una Tabla o control Tabla TreeView así como los elementos seleccionados.. Estos elementos han sido guardados por [TableSavePositionAndSelection](../WDLang1/1000017412.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SaveTable is string
SaveTable = TableSavePositionAndSelection(TABLE_Table1)
TableDisplay(TABLE_Table1, taInit)
TableRestorePositionAndSelection(TABLE_Table1, SaveTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TableRestorePositionAndSelection(<Table control> [, <Save>])`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.




**`<Save>: (Cadena de caracteres opcional)`**

Guardar características devueltas por [TableSavePositionAndSelection](../WDLang1/1000017412.md). Si no se especifica este parámetro:

- Se utiliza el último guardado realizado por [TableSavePositionAndSelection](../WDLang1/1000017412.md) en la control (si existe).

- fSi la opción "Recordar posición y selección" está marcada en el control description (pestaña "Detalles"), se utiliza el último valor guardado.

- De lo contrario, se produce un error.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000082"></a>

La función **TablaRestaurarPosición YSelección** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- una selección única o selección múltiple control.



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000097"></a>

- El nombre de la control utilizada para guardar ([TableSavePositionAndSelection](../WDLang1/1000017412.md)) puede diferir del nombre utilizado para la operación de restauración. Esta característica puede ser useful para duplicar la posición y la selección en dos controles de la Tabla que se muestran alternativamente por ejemplo.

- Si el control está vacío, la función **TablaRestaurarPosición YSelección** no tiene ningún efecto.

- La función **TablaRestaurarPosición YSelección** no funciona en controles Tabla basados en archivo de datos con rebote (es decir, controles Tabla sin barra de desplazamiento proporcional).



<a name="NOTE0_3"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000115"></a>

Las funciones [TableSavePositionAndSelection](../WDLang1/1000017412.md) y **TablaRestaurarPosición YSelección** equivalen a la opción "Recordar posición y selección" disponible en la pestaña "Destalles" de la descripción del control.

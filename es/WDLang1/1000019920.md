
## WinAdaptWidth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adapta el ancho de la ventana al contenido de los controles. Los controles se muestran de forma óptima en la ventana (las barras de desplazamiento se borran si es posible).. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WinAdaptWidth(WIN_DetailForm)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinAdaptWidth([<Window>])`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana a redimensionar. La ventana actual se utilizará si no se especifica este parámetro.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El cambio de tamaño de una ventana en anchura sólo es posible si la ventana contiene uno de los siguientes controles anclado en anchura:

- Control List Box, 

- Control Tabla,

- Control Looper, 

- Control TreeView, 

- Control ListView, 

- Control Campo de entrada multilínea.




## GetIdentifier( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula un identificador único (entero) para un ejecutable o para una sesión.. Estos datos no se almacenan: se pierde al final de la ejecución de la aplicación o sesión.

## Sintaxis

`<Result> = GetIdentifier()`
---

**`<Result>: (Integro)`**

Identificador generado. La primera llamada a la función **GetIdentifier** devuelve 1, la segunda devuelve 2, ...

## Observaciones


### Límites
<a name="limites_ELTPARAGRAPHE000077"></a>

- El identificador generado es un número entero, por lo que está limitado a 231. El identificador se vuelve negativo si se excede este límite.

- La función **GetIdentifier** puede utilizarse en una tarea paralela.





### Cálculo de un identificador
<a name="calculo_identificador_ELTPARAGRAPHE000093"></a>

También tiene la posibilidad de get un identificador global (GUID) a través de [GetGUID](../WDLang1/1000018857.md).

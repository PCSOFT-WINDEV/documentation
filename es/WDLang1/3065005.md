
## RegistrySeek( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca una cadena de caracteres en el registro.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve all the "FullScreen" values 
// of the "HKEY_CURRENT_USER\Console" key
Key is string = "HKEY_CURRENT_USER\Console"
WHILE Key <> ""
	Key = RegistrySeek(Key, "FullScreen")
	Info(Key)
END
```

```wl
// Retrieve all the "FullScreen" values 
// of the "HKEY_CURRENT_USER\Console" key
// Forced "64-bit application" access mode
Key is string = "HKEY_CURRENT_USER\Console"
WHILE Key <> ""
	Key = RegistrySeek(registryMode64, Key, "FullScreen")
	Info(Key)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RegistrySeek([<Access mode>, ] <Search base path> , <Search string> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Resultado de la búsqueda (cadena vacía si se ha producido un error). El resultado tiene el siguiente formato: 

```txt
<Key path> + TAB + <Subkey> [ + TAB + <Value> [ + TAB + <Data>]]
```


**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Search base path>: (Cadena de caracteres)`**

Ruta de la clave desde la que se realiza la búsqueda. Esta ruta tiene el siguiente formato: 

```txt
<Key path>[ + TAB + <Subkey> [ + TAB + <Value> [ + TAB + <Data>]]]
```


**`<Search string>: (Cadena de caracteres)`**

Cadena de caracteres buscada en el registro.

**`<Options>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Modo de búsqueda. El modo de búsqueda de Default corresponde a: 
RegistrySeekKey + RegistrySeekValue + RegistrySeekData + registrySeekRecursive.



|   |   |
| --- | --- |
| *registrySeekData* | Realiza una búsqueda en los datos (String, Multiple String o Extensible String data only). |
| *registrySeekKey* | Realiza una búsqueda en el nombre de las teclas. |
| *registrySeekRecursive* | Realiza una búsqueda recursiva desde la clave base de la búsqueda. |
| *registrySeekValue* | Realiza una búsqueda en los nombres de los valores. |
| *registrySeekWholeWord* | Busca la palabra completa especificada en el parámetro &lt;Cadena de búsqueda&gt;. |


## SysRecentDocAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un archivo a la lista de documentos abiertos recientemente. Esta lista es administrada por el shell de Windows.

**Observación**: Entonces, esta Document puede ser abierta por [ShellExecute](../WDLang1/3035007.md) si su extensión era associated con una aplicación (a través del menú emergente "Abrir con" o a través del registro).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF OpenMyDocument(DocPath) = True THEN
	SysRecentDocAdd(DocPath)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysRecentDocAdd(<File path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se agregó el documento a la lista de documentos abiertos recientemente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve la causa del error.  




**`<File path>: (Cadena de caracteres)`**

Ruta del archivo que debe añadirse a la lista de documentos abiertos recientemente. Esta ruta puede ser una cadena de caracteres en formato Unicode o Ansi.. 






## Observaciones


### Asociar una aplicación con una extensión de archivo
<a name="asociar_una_aplicacion_con_una_extension_archivo_ELTPARAGRAPHE000046"></a>

Para asociar una aplicación con una extensión de archivo a través del registro, debe crear las siguientes claves:

- "HKEY_CLASSES_ROOT\\.ext": 
	dónde **anexo** es la extensión. 
	El valor de Default corresponde al tipo de archivo (cadena libre).

- "HKEY_CLASSES_ROOT\\type_name": 
	dónde **type_name** es el nombre del tipo dado a la extensión.
	El valor por defecto corresponde a la description que se mostrará en la columna "Tipo" del explorador de Windows.

- "HKEY_CLASSES_ROOT\\type_name\\shell\\open\\command": 
	El valor de Default es el comando que ejecutará [ShellExecute](../WDLang1/3035007.md) sobre este tipo de archivo. En el comando Line, la cadena %1 será reemplazada por la ruta del archivo.

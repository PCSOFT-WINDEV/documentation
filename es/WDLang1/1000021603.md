


## Funciones de gestión de controles Action Bar
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la Action Bar: 



|   |   |
| --- | --- |
| [ActionBarDeleteSearchHistory](../WDLang1/1000021600.md) | Borra el historial de la búsqueda control que se encuentra en la barra de acciones de una ventana. |
| [ActionBarSearchVisible](../WDLang1/1000021599.md) | Permite get y cambiar el estado de visibilidad del control de búsqueda associated con el Action Bar en la ventana actual. |


## GetCountryName( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>25</B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Se recomienda utilizar la función [Funciones de gestión de continentes y países](../WDLang1/1000024945.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de un Country que corresponde a su código ISO según la norma ISO 3166-1.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// France caption
sCaptionFrance is string = GetCountryName("FRA")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GetCountryName(<ISO code>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre del país.

**`<ISO code>: (Cadena de caracteres)`**

Código ISO de Country. Este código ISO puede ser el código ISO alfabético (2 o 3 caracteres) o el código ISO numérico de la Country.

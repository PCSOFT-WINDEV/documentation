
## IsNumeric( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite averiguar si un Variable o un control es:

- un numérico (entero, real, moneda o numérico).

- una cadena que puede ser convertida en numérica.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IsNumeric("143")    // returns True
IsNumeric("1.67")   // returns True
IsNumeric("ABC")    // returns False
IsNumeric("3plus2") // returns True
IsNumeric("7,5")    // returns True
IsNumeric("1D2")    // returns True
IsNumeric("2.5e-2") // returns True
IsNumeric("69 Lyon Rhône") // returns True
IsNumeric("09/01/2007") // returns True
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IsNumeric(<Variable or control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la variable o el control especificado es: 

	- un numérico 

	- una cadena que puede ser convertida en numérica (por [Val](../WDLang1/3024037.md), por ejemplo),




- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Variable or control>: (Cualquier tipo)`**

Elemento a manipular.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un búfer no se considera un búfer numérico.

- Una duración no se considera numérica.

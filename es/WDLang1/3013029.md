
## HelpFile( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el nombre del archivo de ayuda utilizado por la ayuda contextual de una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use the "D:\WINDEV\Help\Help.chm" help file
// for the context-sensitive help of "WIN_Window1"
HelpFile(WIN_Window1, "D:\WINDEV\Help\Help.chm")
```


<a name="Example2"></a>



```wl
// Use the "Help.hlp" help file
// for the context-sensitive help of the current window
HelpFile("", "Help.hlp")
```


<a name="Example3"></a>



```wl
// Find out the help file used
// for the context-sensitive help of the current window
ResHelpFile = HelpFile("")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el nombre del archivo de ayuda utilizado para la ayuda sensible al contexto

`<Result> = HelpFile(<Name of the window>)`
---

**`<Result>: (Cadena de caracteres)`**

Nombre y ruta del archivo de ayuda (.HLP o .CHM) associated con la ventana de la ayuda sensible al contexto.

**`<Name of the window>: (Cadena de caracteres)`**

Nombre de la ventana cuyo archivo de ayuda associated se solicita. Si este parámetro corresponde a una cadena vacía (""), la ventana utilizada es la ventana a la que pertenece la Process actual.


<a name="SYNTAX2"></a>

### Modificar el archivo de ayuda utilizado para la ayuda sensible al contexto

`<Result> = HelpFile(<Name of the window> [, <Name of help file>])`
---

**`<Result>: (Cadena de caracteres)`**

Nombre y ruta del archivo de ayuda (.HLP o .CHM) associated con la ventana de ayuda contextual <strong><strong><strong><strong>antes de la modificación</strong></strong></strong></strong> .

**`<Name of the window>: (Cadena de caracteres)`**

Nombre de la ventana utilizada. Si este parámetro corresponde a una cadena vacía (""), la ventana utilizada es la ventana a la que pertenece la Process actual.

**`<Name of help file>: (Cadena de caracteres opcional)`**

Nombre y ruta del archivo de ayuda (.HLP o.CHM) utilizado para la ayuda sensible al contexto. Se puede utilizar una ruta relativa o una UNC ruta.

## Observaciones




### Ayuda contextual
<a name="ayuda_contextual_ELTPARAGRAPHE000056"></a>

La ayuda contextual se muestra cuando el usuario hace clic en "¿Qué es?" Icon en la ventana (? Icon). ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=aidecontext.gif)

Esta ayuda sensible al contexto está basada:

- En el estilo de la ventana ("Icon de ayuda contextual" marcada).

- En el archivo para la ayuda contextual definida: 

	- en el editor de ventanas (pestaña "Ayuda" de la ventana de description). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fichierAide%20-%20HC%20N%B0001.gif&type=thumb)


	- mediante la función **HelpFile**. 

- En el número de ayuda definido en el control description.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fichierAide%20-%20HC%20N%B0002.gif&type=thumb)

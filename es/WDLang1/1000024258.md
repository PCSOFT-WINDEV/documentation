


## &lt;Tabla&gt;.BreakIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de la cabecera y el pie de página de una pausa dada en una control Tabla.

El índice de la cabecera de ruptura de una determinada ruptura corresponde también al índice de la primera fila de esta ruptura.

El índice del pie de página de una ruptura determinada también corresponde al índice de la última fila de esta ruptura.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HeaderIndex is int

// Retrieves the index of the break header for the selected row
HeaderIndex = BRK_BreakHeader.BreakIndex()
```


<a name="Example2"></a>

```wl
// Adds a row into the Table control
RowIndex is int
RowIndex = TABLE_Articles.AddLine(ProductName, ...
		SupplierName, UnitPrice)

// Retrieves the index of the break header for the added row
HeaderIndex = BRK_BreakHeader.BreakIndex(RowIndex)

// Fixes the value of the text in the break
TABLE_Articles[HeaderIndex].STC_Supplier = SupplierName
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Break header/footer>.BreakIndex([<Index>])`
---

**`<Result>: (Integro)`**



- Índice del encabezado/pie de ruptura especificado en la ruptura que contiene la fila cuyo índice es &lt;Índice&gt;, 

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Break header/footer>: (Nombre del control)`**

Nombre de la cabecera o pie de página de la pausa que se va a utilizar.

**`<Index>: (Entero opcional)`**

Índice de una de las filas en la pausa deseada.
Si no se especifica este parámetro, se devuelve el índice del encabezado/pie de ruptura que contiene la fila actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000233"></a>

**&lt;Tabla&gt;.BreakIndex** se puede utilizar en un control Tabla basado en un archivo de datos o rellenarse mediante programación.




<a name="NOTE0_2"></a>
Se recomienda llamar a **&lt;Tabla&gt;.BreakIndex** una vez que el contenido del control Tabla se muestre por completo. De lo contrario, los índices devueltos por **&lt;Tabla&gt;.BreakIndex** pueden no corresponder a los índices reales mostrados en el control

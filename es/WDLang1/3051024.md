
## Reproducción de un vídeo con las funciones MCI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000143"></a>
Las funciones MCI se utilizan para facilitar los archivos multimedia de manipular. Esta página de ayuda presenta las diferentes operaciones que pueden ser realizadas por las funciones MCI.

Para manipular fácilmente un video en WINDEV, todo lo que tienes que hacer es usar un control Imagen.

Se pueden realizar las siguientes operaciones:

- Abre y reproduce un video en un control Imagen,

- Haga una pausa o detenga el video,

- control el volumen.

<a name="NOTE2_1"></a>


## Abrir y reproducir un vídeo en una control Imagen
<a name="abrir_reproducir_video_una_control_imagen_ELTTEXTE000167"></a>


### Abrir un archivo de vídeo
<a name="abrir_archivo_video_ELTPARAGRAPHE000026"></a>

Para abrir un archivo de vídeo, basta con llamar a [MCIOpen](../WDLang1/3051017.md). La mayoría de los archivos de vídeo pueden ser manejados por las funciones MCI.

Ejemplo:


```wl
// Select the video file
sFile is string
sFile = fSelect(EDT_FILE, "Select a video file...",...
	"Video files (*.avi)"+ TAB + "*.avi" + CR + ...
	"MPEG video files (*.mpg)" + TAB + "*.mpg" + CR + ...
	"Quick Time files (*.mov)" + TAB + "*.mov" + CR + ...
	"All files (*.*)" + TAB + "*.*", ...
	"*", fselOpen + fselExist)
// If a file was selected
IF sFile <> "" THEN
	 EDT_FILE = sFile
	 // Close the previous Video file if there is one
	 // No action otherwise 
	 MCIClose("MyVideo") 
	 // Open the selected Video file 
	 // "MyVideo" is the logical name (alias) chosen arbitrarily 
	 // this name will be used to handle the file thereafter 
	 // IMG_VIDEO is the Image control where the video will be displayed 
	 MCIOpen(EDT_FILE, "MyVideo", MciStyleChild, IMG_VIDEO) 
	 // If the opening failed
	 IF ErrorOccurred = True THEN
		 // Display the status report
		  Error(ErrorInfo()) 
	 END
END
```

<a name="NOTE2_2"></a>


### Reproducción de un archivo de vídeo
<a name="reproduccion_archivo_video_ELTPARAGRAPHE000040"></a>

Para reproducir un archivo de vídeo, basta con utilizar [MCIPlay](../WDLang1/3051001.md).


```wl
// Play the selected video
// Reminder: "MyVideo" is the logical name chosen for this file when it is opened
MCIPlay("MyVideo")
```


**Observación**: Para reproducir el vídeo (tipo AVI) en bucle, utilice [MCIEjecutar](../WDLang1/3051005.md) y especifique "reproducir repetición AVI" en el comando Line:


```wl
// Play the AVI video in loop
MCIExecute("play AVI repeat")
```

## Pausar o detener el vídeo
<a name="pausar_detener_video_ELTTEXTE000197"></a>


### Detener el vídeo
<a name="detener_video_ELTPARAGRAPHE000061"></a>

Para pausar el vídeo, basta con llamar a [MCIPause](../WDLang1/3051015.md).


```wl
// Pause the selected video
// Reminder: "MyVideo" is the logical name chosen for this file when it is opened
MCIPause("MyVideo")
```

<a name="NOTE3_2"></a>


### detenerse
<a name="detenerse_ELTPARAGRAPHE000073"></a>

Para detener la reproducción, basta con utilizar [MCIStopReproducción](../WDLang1/3051003.md).


```wl
// Stop the selected video
// Reminder: "MyVideo" is the logical name chosen for this file when it is opened
MCIStopPlaying("MyVideo")
```


**Consejo**: No olvide reposicionar el vídeo al principio del archivo. Por lo tanto, el vídeo comenzará desde el principio la próxima vez que se reproduzca.


```wl
// Reposition at the beginning of the file
// When the user asks to read the file again, the read operation will start from the beginning
MCIBegining("MyVideo")
```

## Control del volumen
<a name="control_del_volumen_ELTTEXTE000227"></a>
Para control el volumen, basta con llamar a [Volumen de MCI](../WDLang1/3051012.md). El volumen derecho, el volumen izquierdo, el volumen de los agudos y el volumen de los graves se pueden subir y bajar.

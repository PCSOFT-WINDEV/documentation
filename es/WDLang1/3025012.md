
## ControlContextMenu( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la control en la que se ha abierto el menú contextual.

**Atención**: **ControlContextMenu** sólo deben utilizarse en los eventos WLanguage de los menús contextuales y en los procedimientos a los que puedan llamar.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Code of the menu option
// Identify the control on which the popup menu was opened
ResControl = ControlContextMenu()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlContextMenu()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la control (en mayúsculas) en la que se abrió el menú contextual,

- Vaciar la cadena (""") si no se ha abierto ningún menú emergente.




Si el control pertenece a un supercontrol, &lt;Resultado&gt; tiene el siguiente formato: &lt;Supercontrol nombre&gt;.&lt;control nombre&gt;.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se asigna el mismo menú contextual a varios controles, **ControlContextMenu** identifica el control en el que se ha expandido el menú para ejecutar el Process adecuado.




## &lt;Document Variable&gt;.Rehacer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la última acción no realizada en una [Document](../WDLang1/1000022461.md) Document.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

// Redoes the last undone action on the document
MyDocument.Redo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Document>.Redo()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la última acción deshecha ha sido restaurada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

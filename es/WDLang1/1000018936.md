
## ServiceExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si un servicio está instalado.. La existencia del servicio se puede comprobar en el ordenador local o en un ordenador remoto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check the existence of the SIS collector
IF ServiceExist("WDCollector") = True THEN
	Info("The collector service of SIS is installed on this computer.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ServiceExist(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a verificar

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para control un servicio en otro ordenador. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el servicio se gestiona localmente.

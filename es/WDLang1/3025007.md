


## Funciones de gestión de controles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles:



|   |   |
| --- | --- |
| [ActiveXEvent](../WDLang1/3025010.md) | Asocia una Procedure escrita en WLanguage con una Event de ActiveX control. |
| [AddLink](../WDLang1/1000019038.md) | Añade un enlace a una sección de texto en un control Campo de entrada. |
| [Altura del texto](../WDLang1/3025011.md) | Calcula la altura (en píxeles) del texto mostrado en una control Campo de entrada, en una control Estático o en una control Combo Box.. |
| [AssistedInputAdd](../WDLang1/1000020043.md) | Añade una fila a la lista de valores disponibles en las sugerencias de autocompletar de un Editar control. |
| [AssistedInputDeleteAll](../WDLang1/1000020044.md) | Borra la lista de valores propuestos para la entrada asistida de una control Campo de entrada. |
| [AssistedInputOpen](../WDLang1/1000020048.md) | Abre la lista de valores propuestos por la entrada asistida para una control Campo de entrada. |
| [AttachPointAdd](../WDLang1/1000021902.md) | Define un punto de unión para una de las dos extremidades de una control de alambre. |
| [AttachPointDeleteAll](../WDLang1/1000021903.md) | Borra todos los puntos de conexión de una control. |
| [BorrarParámetro](../WDLang1/3025048.md) | Borra un parámetro (o una set de parámetros) guardado por [SaveParameter](../WDLang1/3025046.md), o automáticamente a través de la persistencia de datos en los controles. |
| [ChangeStyle](../WDLang1/1000017171.md) | Modifica el estilo de un control dinámicamente. |
| [ConfigureSpellCheck](../WDLang1/3025052.md) | Configura el corrector ortográfico para los controles de edición y para las columnas control Tabla en toda la aplicación. |
| [ControlAlias](../WDLang1/3025016.md) | Obtiene, modifica o restaura la vinculación de datos a una control o grupo de control s. |
| [ControlClone](../WDLang1/3025017.md) | Crea un nuevo control a partir de un control existente. |
| [ControlContextMenu](../WDLang1/3025012.md) | Identifica la control en la que se ha abierto el menú contextual. |
| [ControlCreate](../WDLang1/1000020876.md) | Crea una nueva control (de cualquier tipo) en una ventana mediante programación. |
| [ControlCurrent](../WDLang1/3025019.md) | Devuelve el nombre del control actualmente en edición. |
| [ControlDelete](../WDLang1/3025037.md) | Borra permanentemente una control de una ventana, página o reporte. |
| [ControlEnabled](../WDLang1/1000017377.md) | Activa una control o un grupo de control s en una ventana. Una animación puede ser set en los control s durante esta operación. |
| [ControlError](../WDLang1/3025036.md) | Personaliza el valor que se muestra en los controles cuando el valor no se puede mostrar. |
| [ControlExist](../WDLang1/3025035.md) | Comprueba la existencia de un control (o grupo de control s) en una ventana, en una página o en un reporte. |
| [ControlFirst](../WDLang1/3025022.md) | Indica el nombre del primer campo especificado en la orden de tabulación para la ventana especificada. |
| [ControlGrayed](../WDLang1/1000017375.md) | Grises un control o un grupo de control s. Una animación puede ser set en los controles durante esta operación. |
| [ControlInfoXY](../WDLang1/1000020503.md) | Devuelve el nombre de la control situada en una posición determinada. |
| [ControlInOutReason](../WDLang1/3025013.md) | Indica el origen de la entrada en la control. |
| [ControlInvisible](../WDLang1/1000017369.md) | Hace un control (o un grupo de controles) Invisible en una ventana. Una animación puede ser set en los controles durante esta operación. |
| [ControlNext](../WDLang1/3025020.md) | Identifica la próxima control en edición. |
| [ControlNoSpace](../WDLang1/3025008.md) | Especifica si se borran los caracteres espaciales a la derecha del valor que se encuentra en la control. |
| [ControlOver](../WDLang1/3025021.md) | Identifica la control sobre la que se pasa el cursor del ratón. |
| [ControlPrevious](../WDLang1/3025033.md) | Especifica el nombre de control que estaba en edición antes del actual control. |
| [ControlTypeInfo](../WDLang1/1000020520.md) | Devuelve el icono correspondiente al tipo especificado de control. |
| [ControlVisible](../WDLang1/1000017338.md) | Hace un control (o un grupo de controles) Visible en una ventana. Durante esta operación, se puede realizar una animación en los controles. |
| [CoordinateEditorToScreen](../WDLang1/1000021277.md) | Convierte un tamaño o una posición especificada en los ppp de reference en el editor (160 ppp) a los ppp nativos de la pantalla en el dispositivo donde se ejecuta la aplicación. |
| [CoordinateImageControlToImage](../WDLang1/1000026037.md) | Convierte las coordenadas de un rectángulo en el control Imagen a las coordenadas correspondientes en el Image que se muestran en el control. |
| [CoordinateImageToImageControl](../WDLang1/1000026038.md) | Convierte las coordenadas de un rectángulo en la Image en las coordenadas correspondientes en la control Imagen. |
| [CoordinateScreenToEditor](../WDLang1/1000021276.md) | Convierte un tamaño o una posición especificada en los ppp nativos de la pantalla del dispositivo donde se ejecuta la aplicación a los ppp de reference en el editor (160 ppp). |
| [Count](../WDLang1/3025009.md) | Devuelve el número de apariciones de una control en una ventana, en una página o en un grupo de control s. |
| [CurrentColumn](../WDLang1/3025006.md) | Devuelve:<br><br>- el índice de la columna indexada actual en una control Tabla (basada en un archivo de datos o rellenada mediante programación).<br><br>- el número de la columna actualmente en modo de edición en la control Tabla especificada (basada en un archivo de datos o poblada programáticamente).<br><br><br> |
| [CurrentIndex](../WDLang1/3025040.md) | Dependiendo de la control especificada, regresa:<br><br>- el número del fila actual mostrado en el control Tabla actual.,<br><br>- el número de la opción con foco en un control Casilla de verificación,<br><br>- el número del fila actual en el control Looper actual,<br><br>- el índice de la actual control indexada .<br><br><br> |
| [DictionaryLoad](../WDLang1/1000022958.md) | Indica y carga el diccionario correspondiente a un idioma determinado. |
| [EditCaret](../WDLang1/1000022421.md) | Modifica el cursor de desplazamiento estándar para todos los controles de edición de la aplicación. |
| [Entrada asistidaCerrar](../WDLang1/1000020049.md) | Cierra la lista de valores propuestos por la entrada asistida para una control Campo de entrada. |
| [EnumControl](../WDLang1/3025005.md) | Devuelve el nombre de Nth control que se encuentra en el elemento especificado. |
| [HTMLNavigate](../WDLang1/3025042.md) | Inicia la navegación en una pantalla HTML control. |
| [InitParameter](../WDLang1/3025044.md) | Inicializa la gestión de los valores persistentes. |
| [LinkColor](../WDLang1/1000022354.md) | Modifica la Color de los enlaces en los controles de edición y en los controles estáticos. |
| [LoadParameter](../WDLang1/3025047.md) | Lee un valor persistente. |
| [NextIndex](../WDLang1/3025030.md) | Devuelve el número de la siguiente fila visualizada en el control Tabla actual.. |
| [Objeto padre](../WDLang1/3025031.md) | Identifica el "padre" de un objeto gráfico: control, columna, ventana, página, ... |
| [OpenContextMenu](../WDLang1/3025029.md) | Abre automáticamente un menú contextual para la ventana control o Page actual. |
| [Restablecer](../WDLang1/3025026.md) | Reinicia:<br><br>- todos los controles que se encuentran en la ventana actual o Page (incluidos los grupos de controles, los supercontroles y las ventanas internas).<br><br>- los controles que se encuentran en un grupo de controles, un supercontrol, una ventana interna o un Page interno.<br><br><br> |
| [ReturnToCapture](../WDLang1/3025027.md) | Detiene el Process actual y fuerza la entrada en el control, ventana o página especificada. |
| [SaveParameter](../WDLang1/3025046.md) | Guarda un valor persistente en el registro o en otro archivo especificado por [InitParameter](../WDLang1/3025044.md). |
| [ScreenFirst](../WDLang1/3025039.md) | Define el próximo control que estará en edición en la ventana actual. |
| [ScrollbarIsVisible](../WDLang1/1000021644.md) | Se usa para averiguar si la barra de desplazamiento de un control en una aplicación WINDEV es Visible. |

| [ScrollbarPosition](../WDLang1/1000021057.md) | Permite gestionar la posición de las barras de desplazamiento: <br><br>- Modifica o recupera la posición de una de las barras de desplazamiento horizontal o vertical que se encuentran en una control de una aplicación WINDEV o WINDEV Mobile. <br><br>- Recupera la posición de una de las barras de desplazamiento horizontal o vertical que se encuentran en una control de un sitio WEBDEV.<br><br><br> |
| [ScrollBarShow](../WDLang1/3025018.md) | Muestra (o no) una barra de desplazamiento en un control WINDEV (control Campo de entrada o cuadro de lista) aunque no se haya especificado una barra de desplazamiento durante el control description. |
| [SelecciónColor](../WDLang1/3025003.md) | Modifica las características de la banda de selección en una control Tabla, control Combo Box o control List Box. |
| [SetFocus](../WDLang1/3025001.md) | Enfoque de set:<br><br>- a un control (que se encuentra en una ventana, en una página o en un conjunto de marcos).<br><br>- a una ventana.<br><br><br> |
| [ShapePolygon](../WDLang1/1000022000.md) | Modifica la forma que se muestra actualmente en una control Forma para obtener una forma personalizada (especificando las coordenadas de los puntos de la Polygon). |

| [ToolTipMargin](../WDLang1/1000021844.md) | Modifica el margen del texto en la información de las herramientas. |


## Aleatorio (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un número aleatorio.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve a real included between 0 and 1
ResRandom = Random()
// Retrieve an integer included between 1 and 33
ResRandom = Random(33)
// Retrieve an integer included between 7 and 21
ResRandom = Random(7, 21)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperando un número aleatorio incluido entre 0 y 1

`<Result> = Random()`
---

**`<Result>: (Real)`**

Número aleatorio incluido entre 0 y 1.  


<a name="SYNTAX2"></a>

### Recuperar un número aleatorio incluido entre 1 y un valor

`<Result> = Random(<Upper bound>)`
---

**`<Result>: (Integro)`**

Número aleatorio incluido entre 1 y un valor especificado (inclusive).

**`<Upper bound>: (Integro)`**

Límite superior del número devuelto (valor incluido entre 1 y 2 147 483 647).  


<a name="SYNTAX3"></a>

### Recuperar un número aleatorio incluido entre dos valores

`<Result> = Random(<Lower bound> , <Upper bound>)`
---

**`<Result>: (Integro)`**

Número aleatorio incluido entre dos valores especificados (inclusive).

**`<Lower bound>: (Integro)`**

Límite inferior del número devuelto (valor incluido entre 0 y 2 147 483 647).

**`<Upper bound>: (Integro)`**

Límite superior del número devuelto (valor incluido entre 0 y 2 147 483 647). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Pool aleatorio
<a name="pool_aleatorio_ELTPARAGRAPHE000089"></a>

El "pool" del que se "elige" el número aleatorio puede ser inicializado por [InitRandom](../WDLang1/3013030.md).

Si no se llama a la función [InitRandom](../WDLang1/3013030.md), los valores aleatorios generados por la función **Aleatorio** serán los mismos cada vez que se ejecute el programa.




## <TimeLine>.DeleteEvent (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un Event de un control Línea de tiempo. Si el control Línea de tiempo es relacionado a una fuente de datos, el Appointment también se borra de la fuente de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the selected event from the TimeLine control
TL_TimeLine.DeleteEvent()

// Deletes the first added event
TL_TimeLine.DeleteEvent(1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Suprimir una Event identificada por su índice

`<TimeLine control>.TimelineDeleteEvent([<Event subscript>])`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de la control Línea de tiempo que se va a utilizar.

**`<Event subscript>: (Entero opcional)`**

Índice de Event para borrar. La índice de la primera Event es set a 1. Si no se especifica este parámetro, se borrará la Event seleccionada.



<a name="SYNTAX2"></a>

### Borrar un evento (EventTimeline Variable)

`TimelineDeleteEvent(<TimeLine control> , <Event>)`
---

**`<TimeLine control>: (Nombre del control)`**

Índice de Event para borrar. La índice de la primera Event es set a 1. Si no se especifica este parámetro, se borrará la Event seleccionada.

**`<Event>: (EventTimeline Variable)`**

Nombre de la control Línea de tiempo que se va a utilizar.


<a name="SYNTAX3"></a>

### Borrar una set de eventos 

`TimelineDeleteEvent(<TimeLine control> , <Array of events>)`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de la variable [EventTimeline](../WDLang1/1000020612.md) que describe las características de la Event a eliminar.

**`<Array of events>: (Array de las variables de la EventTimeline)`**

Nombre de la control Línea de tiempo que se va a utilizar.

## ¿Qué es arrastrar y soltar?
<a name="¿que_arrastrar_soltar_ELTTEXTE000131"></a>
Arrastrar y soltar se utiliza para transferir datos entre varios controles con el ratón.. Estos datos pueden moverse entre varios controles que se encuentran en la misma aplicación o en aplicaciones diferentes.

Hay que distinguir entre:

- La fuente, objeto que contiene los datos a copy o a cortar.

- El destino, objeto en el que se deben copiar los datos.




Para realizar la función de arrastrar y soltar:

1. Seleccione los datos en la fuente con el ratón.

2. Presione el botón izquierdo del ratón y manténgalo presionado mientras mueve los datos al destino.

3. Suelte el botón del ratón para validar la transferencia de datos.




Si hay que copiar datos, hay que mantener pulsada la tecla Ctrl durante la operación.

## WINDEV y "Arrastrar y soltar"
<a name="windev_arrastrar_soltar_ELTTEXTE000155"></a>
Se pueden realizar varios tipos de arrastrar y soltar en sus aplicaciones:

- [automatic Arrastrar y soltar](../WDLang1/3030012.md)en los controles de edición, los list boxes (incluyendo las vistas de lista), las vistas de árbol, las tablas y las tablas de vista de árbol.

- [Arrastrar y soltar programados](../WDLang1/3030013.md)en varios tipos de controles.

- [Arrastrar y soltar programado entre el explorador de Windows](../WDLang1/3030014.md)y una aplicación WINDEV.




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En Java, sólo está disponible el Arrastrar y Soltar de automatic.


### Caso especial: Conector Nativo SQL Server y Drag and Drop
<a name="caso_especial_conector_nativo_sql_server_drag_and_drop_ELTPARAGRAPHE000058"></a>

La función de arrastrar y soltar puede no funcionar cuando se utiliza el conector del servidor SQL (a través de la base de datos OLE o a través de ODBC).. 
Para restablecer un modo de funcionamiento correcto, deben añadirse las siguientes líneas de código en el código de inicialización del proyecto: 


```wl
LoadDLL("ole32")
API("ole32","ColnitializeEx",0,2)
```


## CalendarPicker( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un diálogo modal con un Calendar para introducir una fecha. Una ventana permite al usuario seleccionar una fecha en un Calendar. Esta función no requiere control Campo de entrada de fecha ni control Calendario.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ResultDate is string 
ResultDate = CalendarPicker()
```


<a name="Example2"></a>



```wl
ResultDate is string 
ResultDate = CalendarPicker("20090101", "Choose a date:")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CalendarPicker([<Initial date> [, <Window title>]])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**



- Fecha seleccionada por el usuario en formato AAAAMMDD,

- Cadena vacía ("") si el usuario ha cerrado la ventana mediante el botón "X".




**`<Initial date>: (Cadena de caracteres opcional o Fecha Variable)`**

La fecha se muestra por defecto en el Calendar (en formato AAAAMMDD). 
La fecha del día es seleccionada por Default si este parámetro no se especifica o si corresponde a una cadena vacía ("").

**`<Window title>: (Cadena de caracteres opcional)`**

Título mostrado en la barra de título de la ventana para seleccionar la fecha.

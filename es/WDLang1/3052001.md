
## MDIEnumChild( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el alias de la ventana MDI hija especificada.

En un bucle, **MDIEnumChild** puede utilizarse para get los alias de las ventanas MDI hijas
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AliasName is string
i is int = 1
AliasName = MDIEnumChild(i)
WHILE AliasName <> ""
	Trace("MDI child # " + i + ": " + AliasName)
	// Next child window
	i++
	AliasName = MDIEnumChild(i)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MDIEnumChild(<Number of Child Window>)`
---

**`<Result>: (Cadena de caracteres)`**



- Alias de la ventana secundaria MDI especificada

- Cadena vacía ("") si se produce un error.




**`<Number of Child Window>: (Integro)`**

Número de la ventana secundaria utilizada para encontrar su alias. Si este parámetro es mayor que el número de ventanas MDI hijas abiertas, &lt;Result&gt; corresponderá a una cadena vacía (""").

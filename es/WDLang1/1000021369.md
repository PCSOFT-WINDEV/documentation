


## TabOpen (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un nuevo panel en una control Pestaña dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NewPane is Control
NewPane <- TabOpen(TAB_DynTab, "New document", IW_Document)
NewPane..Image = "new.png"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TabOpen(<Tab control> , <Caption> [, <Internal window> [, <Parameter 1> [... [, <Parameter N>]]]])`
---

**`<Result>: (Cadena de caracteres)`**

Alias del panel de pestañas creado. Este alias se usa para manipular el panel de pestañas dinámico. 

**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. Si el control Pestaña no es dinámico, la función falla y devuelve un error WLanguage.

**`<Caption>: (Cadena de caracteres)`**

Título del botón en el panel de pestañas creado.

**`<Internal window>: (Cadena de caracteres)`**

Nombre de la ventana interna que se utilizará en el panel de pestañas. Si no se especifica este parámetro, se utiliza la ventana interna por defecto (definida en la pestaña "Detalles" de la ventana description o con el [WindowIfNew](../Proprietes/1000021381.md) Property).
Si la ventana interna está vacía, se genera un panel vacío.

**`<Parameter 1>: (Tipo de parámetro (opcional))`**

Primer parámetro a pasar a la ventana interna. Este parámetro se pasa al Process para declarar las variables globales de la ventana interna.



**`<Parameter N>: (Tipo de parámetro (opcional))`**

Enésimo parámetro a pasar a la ventana interna. Este parámetro se pasa al Process para declarar las variables globales de la ventana interna.

<a name="NOTE0_1"></a>

## Observaciones


- La imagen asociada al panel de pestañas debe ser modificada con la [Image](../Proprietes/2510034.md) Property. Por ejemplo: 
	
	```wl
	TAB_Tab[AliasName]..Image = "xxxx.png"
	```


- La leyenda asociada al panel de pestañas puede modificarse con la función [Pie de foto](../Proprietes/2510053.md) Property. Por ejemplo: 
	
	```wl
	TAB_Tab[AliasName].Caption = "Customer xxxx *"
	```
La [MaxButtonWidth](../Proprietes/1000022169.md) Property se utiliza para definir la anchura máxima de la leyenda mostrada. Si el título excede este ancho, se muestra una elipsis al final ("....").

- La ventana interna utilizada en el panel puede ser modificada por [ChangeSourceWindow](../WDLang1/3038053.md). Por ejemplo: 
	
	```wl
	ChangeSourceWindow(TAB_DynTab[AliasName], "MyIW")
	```


- El alias del panel actual puede recuperarse o modificarse con la función [Valor](../Proprietes/2510130.md) Property. 

- Para conocer los alias de los paneles dinámicos abiertos, basta con llamar a [EnumControl](../WDLang1/3025005.md) en la control Pestaña: 
	
	```wl
	// Fill a combo box with the list of window controls 
	i is int = 1
	ResControl is string
	ResControl = EnumControl(TAB_MyTab, i)
	WHILE ResControl <> ""
		i++
		Trace(ResControl)
		ResControl = EnumControl(TAB_MyTab, i)
	END
	```


- Tiene la posibilidad de manipular un control que se encuentra en una ventana interna de un panel tan pronto como se abre el panel. 
	Ejemplo: 
	
	```wl
	// Handle a control found in an internal window of a pane
	gsAlias is string = TabOpen(TAB_NoName1, "Initial caption", IW_Internal)
	x is control <- TAB_NoName1[gsAlias]
	x.IW_Internal.EDT_NoName1.Width = 500
	```


- Para ejecutar una Procedure de una ventana interna de un panel, utilice [ExecuteProcess](../WDLang1/3013031.md). 
	Ejemplo: 
	
	```wl
	// Run a procedure of an internal window of a pane
	gsAlias is string = TabOpen(TAB_NoName1, "Initial caption", IW_Internal)
	ExecuteProcess(gsAlias + ".MyInternalProcedure", trtProcedure)
	```

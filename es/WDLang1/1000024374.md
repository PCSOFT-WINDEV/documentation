
## Lector de PDF control y constantes de gestión de archivos PDF
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por las funciones para gestionar los controles de PDF Reader:

| Constante | Detalle | Función o tipo de Variable |
| --- | --- | --- |
| **dispWithCoverPage** | Modo de visualización de la página en el lector de PDF: Establece la primera página del archivo PDF como portada. La primera Page aparece a la derecha cuando se visualiza en modo 2-Page. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **dispEnableScrolling** | Modo de visualización de la página en el lector de PDF: Visualización con desplazamiento progresivo: Muestra las páginas en una columna vertical ininterrumpida que ocupa el ancho de una página. El desplazamiento se puede activar tanto en el modo de visualización de 1 página como en el de 2 páginas. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **dispDefault** | Modo de visualización de la página en el lector de PDF: Visualización predeterminada. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **dispTwoPage** | Modo de visualización de la página en el lector de PDF: Mostrar dos páginas una al lado de la otra en el lector de PDF. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **dispSinglePage** | Modo de visualización de la página en el lector de PDF: Muestra una página del archivo PDF a la vez en el lector de PDF. No se muestran los márgenes. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **pdfXCoord** | Coordenada X de la posición especificada en la Page (expresada en milímetros). | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfYCoord** | Coordenada Y de la posición especificada en la Page (expresada en milímetros). | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfColor** | Gestión del color en el archivo PDF: Crear un archivo PDF a color. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **pdfGrayscale** | Gestión del color en el archivo PDF: Crear un archivo PDF en blanco y negro. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **pdfPageNumber** | Número de la Page que contiene la posición especificada. | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **pdfOriginScreen** | Cambia el origen para calcular una posición en el PDF Reader control. | [PDFReaderInfoXY](../WDLang1/1000025332.md) |
| **rotationAnticlockwise** | Gira la Page -90 grados. | [PDFReaderRotatePage](../WDLang1/1000023940.md) |
| **rotationClockwise** | Gira la Page 90 grados. | [PDFReaderRotatePage](../WDLang1/1000023940.md) |
| **zoomFitVisible** | Zoom predeterminado al abrir el archivo PDF: Ampliar el contenido. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **zoomDefault** | Zoom predeterminado al abrir el archivo PDF: Zoom predeterminado. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **zoomFullPage** | Zoom predeterminado al abrir el archivo PDF: Zoom para mostrar toda la página. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **zoomFullHeight** | Zoom predeterminado al abrir el archivo PDF: Zoom para mostrar todo el espacio vertical de la página. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **zoomFullWidth** | Zoom predeterminado al abrir el archivo PDF: Zoom para mostrar todo el espacio horizontal de la página. | Tipo [pdfParameter](../WDLang6/1410087857.md) |
| **zoomActualSize** | Zoom predeterminado al abrir el archivo PDF: Zoom al 100%/ | Tipo [pdfParameter](../WDLang6/1410087857.md) |


## Transfer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia un bloque de bytes o una cadena de longitud fija de una memoria Address a otra. El interés de esta función reside en su velocidad cuando copy LA las cuerdas manejando las direcciones de las cuerdas a copy.

**Observación**: Esta función es una función avanzada. Puede encontrar problemas con el comportamiento de su aplicación si el tamaño o las direcciones especificadas no son válidas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DestString is fixed string on 12000
SourceString is fixed string on 24000
TransferLength is int
TransferLength = 4096
Transfer(&DestString, &SourceString, TransferLength)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Transfer(<Destination block> , <Source block> , <Length>)`
---

**`<Destination block>: (Address)`**

Address del bloque de destino. Un error de WLanguage ocurre si la Address es inválida (NULL específicamente). Los bloques de memoria pueden solaparse.

**`<Source block>: (Address)`**

Address del bloque fuente. Un error de WLanguage ocurre si la Address es inválida (NULL específicamente). Los bloques de memoria pueden solaparse.

**`<Length>: (Integro)`**

Número de bytes a copy.

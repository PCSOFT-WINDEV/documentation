
## PVTCompareDateRange (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula y muestra una comparación de rango de fechas en un control Tabla dinámica. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Compare 1st quarters 2018 to 2020
sDateRange is string 
FOR nYear = 2018 TO 2020
	// from January 1 (0101) to March 31 (0331)
	sDateRange += [CR] + "" + nYear + "0101" + TAB + nYear + "0331"
END
PVTCompareDateRange(PVT_Sales, sDateRange, pvtAbsoluteEvolution + pvtPercentageEvolution)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PVTCompareDateRange(<Pivot Table control> , <Range> , <Display mode>)`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Range>: (Cadena de caracteres)`**

Rangos de fechas a comparar. Este parámetro tiene el siguiente formato: 

```txt
<Min date Range 1> + TAB + <Max date Range 1> + CR + ...
<Min date Range N> + TAB + <Max date Range N>
```
 donde: 

- &lt;Fecha Mín Rango&gt; corresponde a la fecha mínima del rango,

- &lt;Fecha Máx Rango&gt; corresponde a la fecha máxima del rango. 


Los rangos de fechas deben estar en el mismo orden (ascendente o descendente) que las dimensiones de las fechas.

**`<Display mode>: (Constante de tipo Integer)`**

Indica si la evolución se debe mostrar en porcentajes y/o en valores absolutos:


|   |   |
| --- | --- |
| *pvtAbsoluteEvolution* | Muestra la evolución entre los períodos en valores absolutos. <br><br>Esta constante se puede combinar con la constante *pvtPercentageEvolution*. |
| *pvtPercentageEvolution* | Muestra la evolución entre los períodos en porcentajes. <br><br>Esta constante se puede combinar con la constante *pvtAbsoluteEvolution*. |
| *pvtWithoutEvolution*<br>(Valor predeterminado) | No muestra la evolución entre los períodos. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función permite comparar rangos de fechas. Equivale a la Funcionalidad Automática de la Aplicación "Comparar rangos de fechas", del menú contextual del control Tabla dinámica. Para obtener más información, consulte [Funcionalidades automáticas de los controles Tabla dinámica](../WDChamp/1000029008.md). 

- El control Tabla dinámica no debe estar vacío, de lo contrario la función no tendrá ningún efecto.

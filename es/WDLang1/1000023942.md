
## Ficha (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Token** permite definir todas las características avanzadas de un token que se muestra en un control Campo de entrada. Las características de esta Token pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a token to an Edit control
MyToken is Token 
MyToken.Caption = Customer.Name
MyToken.Value = Customer.IdCustomer
EditAddToken(EDT_Customer, MyToken)

// Modify the background color of a token
FOR EACH MyToken OF EDT_Customer.Token
	IF MyToken.Value = Customer.IdCustomer THEN
		MyToken.BackgroundColor = LightRed
	END
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables Token
<a name="propiedades_especificas_las_variables_token_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a Token:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| BackgroundColor | Variable de tipo [Color](../WDLang1/1000019661.md) | Antecedentes color del Token.<br><br> |
|   | Cadena de caracteres | Código HTML de la leyenda del token tal y como se mostrará en la edición de control. Si no se especifica esta Property, se utiliza el título especificado con la **Caption** Property. |
| Color | Variable de tipo [Color](../WDLang1/1000019661.md) | color del texto Token.<br><br> |
| Pie de foto | Cadena de caracteres | Leyenda de la Token tal y como se mostrará en Editar control. Si no se especifica esta Property, la leyenda de la Token corresponderá a su valor almacenado.<br><br> |
| StoredValue | Cadena de caracteres | Valor almacenado de la Token. Almacena cualquier tipo de valor. |
| Tooltip | Cadena de caracteres | Tooltip aparece al pasar el ratón por encima de Token. |
| Valor | Cadena de caracteres | Valor de la Token. |


<a name="NOTE0_2"></a>


### Manipulación de tokens
<a name="manipulacion_tokens_ELTPARAGRAPHE000109"></a>

- Para manipular los tokens de una Edición control, es posible utilizar: 

	- la [funciones de gestión de tokens](../WDLang1/1000024042.md). 

	- la propiedad [Token](../Proprietes/1000023976.md) del control Campo de entrada que devuelve una lista modificable que contiene todos los tokens mostrados en el control.




- Puede utilizar [VariableReset](../WDLang1/3013069.md) para restablecer el contenido de un Token Variable.

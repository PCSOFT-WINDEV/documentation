
## ServiceUninstall( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desinstala un servicio. Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Desactiva todos los "niveles de ejecución" del servicio pero no borra ningún archivo (incluyendo los scripts de inicio).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Uninstalls the service found in the TABLE_Service table
IF ServiceUninstall(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceUninstall(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se desinstaló el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a desinstalar.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para desinstalar un servicio en otro ordenador.. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía, el servicio se gestiona localmente..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Caso especial
<a name="caso_especial_ELTPARAGRAPHE000068"></a>

El servicio no se desinstala inmediatamente. Se envía una solicitud al sistema. Esta solicitud será procesada cuando el servicio haya terminado.. El servicio sólo se puede terminar cuando no se utiliza ninguna instancia del servicio.

Si el servicio se está ejecutando actualmente, se detendrá automáticamente antes de desinstalarlo.

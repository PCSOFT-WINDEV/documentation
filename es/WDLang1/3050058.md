
## MatCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una matriz que contiene 0 columna fila y 0.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a matrix
ResCreate = MatCreate("MyMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatCreate(<Matrix>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la matriz,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a crear. Este nombre será utilizado por todas las funciones para la gestión de matrices.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000083"></a>

No se verifica la existencia de la matriz.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).

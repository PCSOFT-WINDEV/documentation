


## &lt;Image editor&gt;.Cerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la imagen encontrada en un contro «Image Editor»
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IE_MyControl.Close()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Image Editor control>.Close()`
---

**`<Image Editor control>: (Nombre del control)`**

Nombre del editor de imágenes control a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función no tiene ningún efecto si no se abre ninguna Image en el editor de Image control.

## Componente:
wd280pic.dll

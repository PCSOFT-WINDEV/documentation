
## MatFloatAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un valor a cada elemento de matriz. Esta adición se realiza en la matriz especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add the value 3 to a matrix
MatFloatAdd("MyMatrix", 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MatFloatAdd(<Matrix> , <Value>)`
---

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a utilizar, definida con [MatCreate](../WDLang1/3050058.md).

**`<Value>: (Real)`**

Valor que se añadirá a la matriz.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000076"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000087"></a>

**Atención**: La función **MatFloatAdd** no devuelve ningún código de error. Para saber si se ha generado algún error, utilice [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

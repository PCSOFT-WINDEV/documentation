


## dModifyContrast (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el contraste de una imagen: 

- en un control Imagen,

- en una variable de tipo [Image](../WDLang1/1000019650.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change contrast
dModifyContrast(IMG_Image1, 50)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = dModifyContrast(<Source image> , <Contrast variation> [, <Destination image>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el contraste,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Source image>: (Nombre de control o variable de tipo Image)`**

Imagen a manipular. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).




**`<Contrast variation>: (Real)`**

Valor comprendido entre -100 y +100. Los valores negativos disminuyen el contraste y los positivos lo aumentan. Este valor se sumará al contraste actual para obtener un nuevo contraste.

**`<Destination image>: (Nombre de control o variable de tipo Image opcional)`**

Imagen en la que se dibujará la imagen transformada. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).


Si no se especifica este parámetro, la transformación se realizará en &lt;Source image&gt;.




## VideoDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra en una control Cámara la secuencia de vídeo actualmente transmitida por una "cámara web" o por la cámara que se encuentra en el dispositivo móvil.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the video sequence currently transmitted in the "CAM_Camera" control
ResFilm = VideoDisplay(CAM_Camera)
IF ResFilm = True THEN
	// Displays a caption
	CAM_Camera.Caption = "Now in the kitchen"
ELSE
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = VideoDisplay(<Camera control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la secuencia de video transmitida actualmente se muestra correctamente en el control Cámara,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000075"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una "cámara" al ordenador actual.




**Observación**: Esta función es equivalente a la función [CámaraPantalla](../WDLang1/1410086595.md).
<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000103"></a>

- En tiempo de ejecución, la secuencia de vídeo transmitida por la Cámara associated se muestra automáticamente tan pronto como se abre la ventana que contiene el control Cámara. En este caso, no es necesario llamar a la función **VideoDisplay**.

- Para detener la reproducción de una secuencia de vídeo, utilice [VideoStop](../WDLang1/3079001.md).

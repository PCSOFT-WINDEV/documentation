
## CameraStartDecoding( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la descodificación de códigos de barras en un control Cámara. Tiene la capacidad de especificar el tipo de códigos de barras a decodificar.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts decoding QRCode bar codes in the Camera control
CameraStartDecoding(CAM_Camera1, BC_QRCODE)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`CameraStartDecoding(<Camera control> [, <Type of bar code>])`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<Type of bar code>: (Constante opcional de tipo Integer)`**

Tipo de códigos de barras que pueden ser decodificados: 


|   |   |
| --- | --- |
| *BC_AZTEC* | Código de barras azteca. <br> |
| *BC_CODE128* | Código 128 código de barras.<br> |
| *BC_CODE39* | Código39 código de barras.<br> |
| *BC_CODE93* | Código 93 código de barras.<br> |
| *BC_DATAMATRIX* | DATAMATRIX bar code.<br> |
| *BC_EAN13* | EAN13 bar code.<br> |
| *BC_EAN8* | EAN8 bar code.<br> |
| *BC_INTER2OF5* | Intercalado 2 de 5 códigos de barras.<br> |
| *BC_PDF417* | Código de barras PDF417.<br> |
| *BC_QRCODE* | Código QR Código de barras. |
| *BC_UPCA* | Código de barras UPC-A.<br> |
| *BC_UPCE* | UPC-E bar code.<br> |
| *BC_ALL* | Todos los tipos de códigos de barras soportados por esta función pueden ser decodificados. |


Si no se especifica este parámetro: 

- si se decodificarán los códigos de barra especificados al llamar por primera vez a la función **CameraStartDecoding**. 

- los códigos de barras especificados en la ventana de description de control Cámara serán decodificados. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El control Cámara debe haber sido configurado para permitir la decodificación de códigos de barras (en la ventana description). De lo contrario, se producirá un error fatal al llamar a la función **CameraStartDecoding**.

- Cuando la control Cámara está configurada para permitir la decodificación de los códigos de barras (en la ventana de la description), la decodificación se inicia automáticamente cuando se abre la ventana. Para prEvent este comportamiento, se puede llamar a [CameraStopDecoding](../WDLang1/1000022449.md) en el Event "Inicialización" del control Cámara.

- Si se llama a la función **CameraStartDecoding** cuando ya hay un proceso de decodificación en curso en el control Cámara, se detiene la decodificación y se reinicia para tener en cuenta el nuevo tipo de código de barras a decodificar. Si el tipo de código de barras es idéntico, la función no tiene ningún efecto.

- Cuando se detecta un código de barras, se llama al Event "Decodificación de un código de barras" del control Cámara con un [CódigoDeBarras](../WDLang5/1000019451.md) Variable como parámetro, que corresponde al código de barras detectado. Si este evento devuelve <u><u><u><u>True</u></u></u></u>, la decodificación de códigos de barras se detendrá automáticamente.

- Esta función es equivalente a la función [VideoStartDecoding](../WDLang1/1000022163.md).




## &lt;Tabla&gt;.StartFilling (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia el Process para poblar una Tabla o control Tabla TreeView programáticamente. Se representa el progreso de esta operación:

- mediante una barra de progreso de Infinite que se muestra en la parte inferior de control.

- por una ventana interna.




**Atención**: Antes de utilizar esta función, es necesario habilitar el mecanismo de gestión de la operación de llenado en la ventana de control description. Para obtener más información, consulte [Características de los controles Tabla en una ventana](../WDChamp/1013200.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TABLE_Mem.StartFill()

FOR i = 1 TO 10000
	TABLE_Mem.AddLine()
END

TABLE_Mem.EndFilling() // Fill completed
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Table control>.StartFilling()`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Combo Box con tabla.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Al final del Process para cargar el control Tabla, se debe llamar a [&lt;Tabla&gt;.EndFilling](../WDLang1/1000024252.md) para especificar que la operación de carga ha finalizado.

- Esta función no tiene ningún efecto sobre la Tabla controla relacionado a una fuente de datos de Archivo o Consulta.

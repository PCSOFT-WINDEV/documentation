


## VideoRequestPermission( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Solicita a Permission que utilice la cámara del dispositivo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF VideoRequestPermission() = True THEN
	OpenMobileWindow(WIN_Camera)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = VideoRequestPermission()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se otorgó el permiso para utilizar la cámara,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

<a name="NOTE0_1"></a>

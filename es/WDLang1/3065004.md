


## Funciones de gestión de registros
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para administrar el registro de Windows:



|   |   |
| --- | --- |
| [RegistryCopyKey](../WDLang1/1000019418.md) | Copia una clave del registro con todas sus subclaves y valores. |
| [RegistryCreateKey](../WDLang1/3065002.md) | Crea una clave en el registro de Windows. |
| [RegistryDeleteKey](../WDLang1/3065010.md) | Elimina una subclave del registro de Windows. |
| [RegistryDeleteValue](../WDLang1/3065011.md) | Elimina un valor del registro de Windows. |
| [RegistryExist](../WDLang1/3065007.md) | Comprueba la existencia de una clave en el registro de Windows. |
| [RegistryFirstSubKey](../WDLang1/3065008.md) | Identifica la ruta de la primera subclave de la clave especificada en el registro de Windows. |
| [RegistryListkey](../WDLang1/1000019881.md) | Enumera las subclaves de una clave de registro. |
| [RegistryListValue](../WDLang1/3065013.md) | Devuelve el nombre (y posiblemente el tipo) de los valores de una clave que se encuentra en el registro. |
| [RegistryNextKey](../WDLang1/3065009.md) | Identifica la clave que se encuentra después de la clave especificada en el registro de Windows. |
| [RegistryQueryValue](../WDLang1/3065012.md) | Lee un valor en el registro de Windows. |
| [RegistryRename](../WDLang1/1000019419.md) | Renombra una clave en el registro. |
| [RegistrySeek](../WDLang1/3065005.md) | Busca una cadena de caracteres en el registro. |
| [RegistrySetValue](../WDLang1/3065006.md) | Escribe un valor a una clave en el registro de Windows. |
| [RegistryValueType](../WDLang1/1000017280.md) | Devuelve el tipo de valor encontrado en el registro. |

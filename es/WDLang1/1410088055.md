
## Diagram (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Diagram** se utiliza para definir todas las características avanzadas de un Diagram que se pueden editar en el Editor Diagram control. Las características de esta Diagram pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a rectangle shape
MyShape is diagRectangle

// Set the dimensions, position and background color
MyShape.X = 100
MyShape.Y = 100
MyShape.Width = 350
MyShape.Height = 150
MyShape.Background.Color = PastelRed
MyShape.Comment = "Rectangle"

// Reference the displayed diagram
MyDiagram is Diagram <- DIAGEDT_MyDiagram

// Add the rectangle shape to the diagram
Add(MyDiagram.Shape, MyShape)

// Display the comment of the shape
Trace(MyDiagram.Shape[1].Comment)
```

```wl
// Move all shapes down
MyDiagram is Diagram <- DIAGEDT_MyDiagram.Value
FOR EACH ShapeInDiagram OF MyDiagram.Shape
	ShapeInDiagram.Y += 10
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables Diagram
<a name="propiedades_especificas_las_variables_diagram_ELTPARAGRAPHE000030"></a>

Las siguientes propiedades pueden utilizarse para manipular a Diagram:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| BackgroundColor | Integro | Color de fondo de la Diagram. <br><br>Esta propiedad puede corresponder a: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br>- a Variable de tipo [Color](../WDLang1/1000019661.md). En este caso, se tiene en cuenta la opacidad especificada en la variable.<br><br><br> |
| Forma | Array de [diagShape](../WDLang1/1410088082.md) | Array que contiene las diferentes formas en el Diagram. <br><br>**Esta propiedad es de solo lectura**. |
|   | Real | Espacio entre las líneas de la cuadrícula. |
| PageLayout | [diagLayout](../WDLang1/1410088610.md) | Diseño de Diagram. Esta disposición se utiliza para imprimir la Diagram. |
| Valor | String, Buffer, etc. | Valor de la Diagram. Esta propiedad puede corresponder a: <br><br>- a Diagram Editor control,<br><br>- la ruta de un archivo Diagram,<br><br>- una variable de tipo Buffer que contiene una Diagram. <br><br><br> |

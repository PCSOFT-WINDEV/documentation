
## InSysNotificationMode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para determinar si la aplicación se lanza desde una Notification interactiva en "Windows 10".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
If InSysNotificationMode() = True THEN 
	Trace("Application launched from a Windows 10 notification")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InSysNotificationMode()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la aplicación se lanza desde un "Windows 10" Notification, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## IsOdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica un número impar.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Odd number?
ResOdd = IsOdd(65)      // Returns True
ResOdd = IsOdd(5 + 11)      // Returns False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IsOdd(<Number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el número especificado es impar,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number>: (Expresión numérica o entero entero)`**

Expresión numérica entera o número entero a identificar.




## &lt;Kanban&gt;.Filter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra solo las tarjetas que coinciden con el filtro especificado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the EDT_Filter control
KANBAN_Example.Filter(EDT_Filter, ccIgnoreCase + ccIgnoreAccent)
```


<a name="1410089534_Example2"></a>

```wl
// Display tasks without contributors
KANBAN_Example.Filter(Proc_Filter)
	INTERNAL PROCEDURE Proc_Filter(Card is kbCard)
		IF Card.Contributor = "" THEN
			RETURN True
		END
	END
// Identical to 
// KANBAN_Example.Filter((Card)=> { RETURN Card.Contributor = "" } )
```


<a name="1410089534_Example3"></a>

```wl
// Remove filter
KANBAN_Example.Filter("")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Filtrar tarjetas según una cadena contenida en el título

`<Kanban control>.Filter(<Filter> [, <Option>])`
---

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<Filter>: (Cadena de caracteres)`**

Filtro a utilizar. Solo se seleccionarán las tarjetas que incluyan la cadena &lt;Filter&gt; en el título o en el contenido. Si este parámetro es una cadena vacía (""), no se aplica ningún filtro.

**`<Option>: (Constante opcional de tipo Integer)`**

Opciones de comparación: 


|   |   |
| --- | --- |
| *ccIgnoreAccent* | Compara las cadenas sin tener en cuenta los caracteres acentuados. |
| *ccIgnoreCase* | Compara las cadenas sin distinguir entre mayúsculas y minúsculas. |
| *WholeWord* | Compara palabras completas, es decir, palabras entre signos de puntuación (caracteres distintos de una letra o un número) o espacios. |




<a name="SYNTAX2"></a>

### Filtrar tarjetas utilizando un procedimiento de filtro

`<Kanban control>.Filter(<WLanguage procedure>)`
---

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<WLanguage procedure>: (Nombre de procedimiento)`**

Nombre del procedimiento WLanguage ("callback") al que se llama para filtrar las tarjetas que se muestran en el control Kanban. Para obtener más información sobre este procedimiento, consulte [Parámetros del procedimiento utilizado por la función &lt;Kanban&gt;.Filter](../WDLang1/1410089535.md).
Si este parámetro es una cadena vacía (""), no se aplica ningún filtro.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las tarjetas que se agregan después de llamar a la función **&lt;Kanban&gt;.Filter** no se tienen en cuenta.

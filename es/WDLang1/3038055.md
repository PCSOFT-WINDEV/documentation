
## WinInactiveEffect( Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17 (76)</B></B>, está funcionalidad no estará disponible.



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita la función OVI (Dim Disabled Window):

- para una ventana específica.

- para todas las ventanas de la aplicación. En este caso, las ventanas set hasta **WinInactiveEffect** serán ignoradas.




La función OVI se usa para hacer grises automáticamente las ventanas Inactive. Para obtener más información, consulte [Oscurecimiento de ventanas inactivas](../Editeurs/9000071.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code of "BTN_Validate"
// Enable the DDW feature for the "WIN_EditCustomer" window
WinInactiveEffect(WIN_EditCustomer, True)
// Open a dialog box
Dialog("Do you want to save the modifications?")
// As long as the dialog box is displayed, 
// the "WIN_EditCustomer" window is dimmed
```


<a name="Example2"></a>



```wl
// -- Project initialization
// Disable the DDW features for all the application windows
WinInactiveEffect(False)
```

## Sintaxis

### Activar o desactivar la función OVI para una ventana específica

`<Result> = WinInactiveEffect(<Window used> , <Activation of DDW>)`
---

**`<Result>: (booleano)`**

Estado de activación previa de OVI:

- <u><u><u><u>True</u></u></u></u>: se habilitó la función OVI para la ventana especificada.

- <u><u><u><u>False</u></u></u></u>: la función OVI no estaba habilitada para la ventana especificada.




**`<Window used>: (Nombre de ventana)`**

Nombre o alias de la ventana que se va a utilizar. Esta ventana debe abrirse.

**`<Activation of DDW>: (booleano)`**

Nuevo estado de activación de OVI:

- <u><u><u><u>True</u></u></u></u>: la función OVI está habilitada para la ventana especificada.

- <u><u><u><u>False</u></u></u></u>: la función OVI no está habilitada para la ventana especificada.  





<a name="SYNTAX2"></a>

### Activar o desactivar la función OVI para todas las ventanas de la aplicación

`<Result> = WinInactiveEffect(<Activation of DDW> [, <Rate of gray>])`
---

**`<Result>: (booleano)`**

Estado de activación previa de OVI:

- <u><u><u><u>True</u></u></u></u>: se habilitó la función OVI para las ventanas de la aplicación.

- <u><u><u><u>False</u></u></u></u>: la función OVI no estaba habilitada para las ventanas de la aplicación.




**`<Activation of DDW>: (booleano)`**

Nuevo estado de activación de OVI:

- <u><u><u><u>True</u></u></u></u>: la función OVI está habilitada para las ventanas de la aplicación.

- <u><u><u><u>False</u></u></u></u>: la función OVI no está habilitada para las ventanas de la aplicación.




**`<Rate of gray>: (Entero opcional)`**

Porcentaje de gris que se aplicará (entre 0 y 100). La tasa de gris es set a 25 por Default.

## Observaciones




### Configuración de los ordenadores del usuario final
<a name="configuracion_los_ordenadores_del_usuario_final_ELTPARAGRAPHE000098"></a>

Dependiendo de la configuración de los ordenadores del usuario final, la función OVI puede no tener ningún efecto.

De hecho, la función OVI se ve afectada por las características de la tarjeta de vídeo, la velocidad del procesador, ....

Si se produce un problema, pida a los usuarios finales que actualicen el controlador de su tarjeta de vídeo.

**Observación**: Para funcionar, los ordenadores de los usuarios finales deben:

- tienen un modo de visualización superior a 256 colores.

- estar en modo de 32 ó 64 bits.

- ejecutarse en Windows Vista o posterior.

- utilizar el nivel máximo de "Aceleración de hardware" para la tarjeta de vídeo (propiedades de visualización avanzadas de el equipo).



<a name="NOTE0_2"></a>




### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000117"></a>

La visualización de la función OVI también puede ser configurada por [WinDisableEffect](../WDLang1/1000017172.md).

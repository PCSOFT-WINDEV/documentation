


## HTMLEditorFromEmail (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Importa el contenido HTML de un correo electrónico y sus imágenes en un editor HTML control. Las imágenes contenidas en la Email se muestran también en la control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read a message and display it in the HTML Editor control to write a reply
MyMessage is Email
EmailReadMessage(SessionID, MyMessage)
HTMLEditorFromEmail(HTMEDT_MyHTMLEditor, MyMessage, emailOptionReplyBefore)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLEditorFromEmail(<HTML Editor control> , <Email> [, <Option>])`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control. 

**`<Email>: (Envía un correo electrónico a Variable)`**

Nombre de la [Email](../WDLang3/1000018713.md) Variable que se mostrará en el editor HTML control. 

**`<Option>: (Constante opcional de tipo Integer)`**

Opciones de visualización de Email: 


|   |   |
| --- | --- |
| *emailOptionForwarded* | Añade un encabezado de respuesta al principio de la Email. |
| *emailOptionReplyAfter* | El contenido de la Email está marcado. La respuesta se escribirá después del contenido. |
| *emailOptionReplyBefore* | El contenido de la Email está marcado. La respuesta será escrita antes del contenido. |


Si no se especifica este parámetro o se set a 0, se puede modificar directamente el correo electrónico que aparece en la control.

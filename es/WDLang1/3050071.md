
## FinError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número del último error causado por una función Finxxx (véase la lista de [funciones financieras](../WDLang1/3050054.md)).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the type of error
ResError = FinError()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinError()`
---

**`<Result>: (Integro)`**

Número del último error causado por una función financiera:

| Número | Tipo de error | Funciones afectadas |
| --- | --- | --- |
| 0 | Ningún error | Todas las funciones |
| 1 | La duración no puede ser nula | [FinDecreasingRedemption](../WDLang1/3050068.md),<br>[FinLinearRedemption](../WDLang1/3050046.md) |
| 2 | La matriz está vacía | [FinNetCurrentVal](../WDLang1/3050053.md) |
| 4 | El período no puede ser superior a la duración | [FinPeriodInterest](../WDLang1/3050042.md) |
| 5 | El tipo de amortización no es correcto | [FinRedemption](../WDLang1/3050061.md) |
| 6 | El importe del reembolso es nulo | [FinPaymentNb](../WDLang1/3050041.md) |
| 7 | Data inconsistency | [FinInterestRate](../WDLang1/3050047.md),<br>[FinPaymentNb](../WDLang1/3050041.md) |
| 8 | El coste de adquisición no puede ser nulo | [FinDecreasingRedemption](../WDLang1/3050068.md), |
| 9 | La matriz no existe | [FinNetCurrentVal](../WDLang1/3050053.md) |
| 10 | Valores máximos excedidos | Todas las funciones |
| 11 | No se puede calcular (división por cero) | [FinRedemption](../WDLang1/3050061.md),<br>[FinPeriodInterest](../WDLang1/3050042.md),<br>[FinCurrentVal](../WDLang1/3050055.md),<br>[FinNetCurrentVal](../WDLang1/3050053.md),<br>[FinRepayVal](../WDLang1/3050039.md) |
| 12 | Infinite número de pagos | [FinPaymentNb](../WDLang1/3050041.md) |
| 13 | No se encontró la tasa | [FinInterestRate](../WDLang1/3050047.md) |
| 14 | El período es más largo que la duración | [FinDecreasingRedemption](../WDLang1/3050068.md) |
| 15 | Número de meses del primer año no incluidos entre 0 y 12 | [FinDecreasingRedemption](../WDLang1/3050068.md) |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.


## Belgium.CheckEU (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número EU (número de unidad de establecimiento para identificar empresas en Bélgica).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyEUNumber is string = "7123654957"
Belgium.CheckEU(MyEUNumber)	// Returns True

// Check a control
EDT_EUNumber = "ABC"
Belgium.CheckEU(EDT_EUNumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Belgium.CheckEU(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número EU válido, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número EU a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.

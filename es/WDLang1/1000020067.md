


## Funciones de gestión del reconocimiento de voz
			



Las siguientes funciones se utilizan para gestionar el reconocimiento de voz:



|   |   |
| --- | --- |
| [SpeechRecognitionAddCommand](../WDLang1/1000020014.md) | Agrega un comando de voz en la ventana actual. |
| [SpeechRecognitionDeleteCommand](../WDLang1/1000020017.md) | Elimina un comando de voz añadido previamente con [SpeechRecognitionAddCommand](../WDLang1/1000020014.md). |
| [SpeechRecognitionTrigger](../WDLang1/1000020013.md) | Activa el servicio de reconocimiento de voz del dispositivo. |







![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) **Observación**: Para la compatibilidad con las versiones anteriores, también están disponibles las siguientes funciones: 



|   |   |
| --- | --- |
| [AddVoiceCommand](../WDLang1/1000017106.md) | Agrega un comando de voz en la ventana actual. A partir de la versión 17, esta función se sustituye por [SpeechRecognitionAddCommand](../WDLang1/1000020014.md). |
| [DeleteVoiceCommand](../WDLang1/1000017187.md) | Elimina un comando de voz que fue agregado por [AddVoiceCommand](../WDLang1/1000017106.md). A partir de la versión 17, esta función se sustituye por [SpeechRecognitionDeleteCommand](../WDLang1/1000020017.md). |


## IsEven( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica un número par.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Even number?
ResEven = IsEven(66)     // Returns True
ResEven = IsEven(6 + 13) // Returns False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IsEven(<Number>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el número especificado es par,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number>: (Expresión numérica o entero entero)`**

Expresión numérica entera o número entero a identificar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El cero se considera un número par.

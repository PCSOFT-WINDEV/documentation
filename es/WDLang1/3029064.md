
## gFontSize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres que se utiliza para especificar el tamaño de la fuente para un texto que se encuentra en una cadena gráfica.

**Observación**: Todo el texto escrito **después de** la función se ve afectada.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En una ventana, esta secuencia de caracteres se utiliza para indicar el tamaño de la fuente:  

- en un elemento que se encuentra en un List Box o control Combo Box.

- en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Display the "WINDEV, IDE #1 in France" title of next window
NextTitle(gFontSize(12) + "WINDEV" + gFontSize(10) + ", #1 in France.")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gFontSize(<Font size>)`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que definen el tamaño de fuente para el resto del texto.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<Font size>: (Integro)`**

Nuevo tamaño de letra para el texto después de la llamada a **gFontSize**.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La fuente Default definida para el control utilizado se restablece tras la llamada a [gFontXXX](../WDLang1/3029028.md).

- Todas las demás características de la fuente se mantienen (cursiva, etc.).

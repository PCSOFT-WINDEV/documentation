
## LoadParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee un valor persistente. El valor persistente fue guardado por [SaveParameter](../WDLang1/3025046.md): 

- en el registro (o archivo equivalente en la plataforma en tiempo de ejecución). Por ejemplo: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) En una aplicación Linux, los parámetros se guardan en la carpeta $HOME/.local/share.




- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) en el archivo especificado por [InitParameter](../WDLang1/3025044.md).











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save
SaveParameter("LOGIN", EDT_Edit1)

// Restore
EDT_Edit1 = LoadParameter("LOGIN", "")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LoadParameter(<Parameter name> [, <Default value>])`
---

**`<Result>: (Cadena de caracteres o entero)`**



- Valor del parámetro buscado. 

- Si no se encuentra el parámetro buscado, corresponde al &lt;Valor predeterminado&gt; o a una cadena vacía ("") si no se especificó ningún valor predeterminado.




**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro a encontrar. Este parámetro no debe corresponder a una cadena vacía.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Si el modo de inicialización seleccionado es el registro, **LoadParameter** no distingue entre mayúsculas y minúsculas ni entre acentos (Parámetro y parámetro son equivalentes).

**`<Default value>: (Cadena de caracteres opcional o entero opcional)`**

Default valor del parámetro. Este valor se devolverá si no se encuentra el valor del parámetro.

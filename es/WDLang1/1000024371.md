


## &lt;Variable Image&gt;.VerticalSymmetry (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Voltea la imagen de un control Imagen con respecto a un eje vertical (simetría con respecto a un eje vertical).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Vertical symmetry of the IMG_Image1 control
MyResult is boolean
MyResult = IMG_Image1.VerticalSymmetry()
IF MyResult = False THEN Error("The operation failed")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Image variable>.VerticalSymmetry()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se volteó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image variable>: (Variable de tipo Image)`**

Nombre del control Imagen a manipular.

La simetría se realiza con respecto al eje vertical central del control. La imagen se modifica directamente.

## Observaciones


### Fin del dibujo
<a name="fin_del_dibujo_ELTPARAGRAPHE000347"></a>

Si se llama a las funciones [&lt;Imagen&gt;.EndDrawing](../WDLang1/1000024327.md) y **&lt;Variable Image&gt;.VerticalSymmetry** en el mismo proceso, no se mostrarán los cambios de la imagen.


## RGBBlue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el componente azul de un color RGB.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RGBBlue(RGB(128, 38, 15)) // Returns 15
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RGBBlue(<RGB color>)`
---

**`<Result>: (Entero)`**

Componente azul del color RGB entre 0 y 255.

**`<RGB color>: (Entero)`**

Color (24 bits). Este color es el resultado de la siguiente fórmula:

Color = 65536 \* Componente Azul + 256 \* Componente Verde + Componente Rojo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Varios
<a name="varios_ELTPARAGRAPHE000080"></a>

- La función [RGB](../WDLang1/3029012.md) devuelve el identificador del color especificado a partir de los componentes Rojo, Verde y Azul.

- La función [RGBRed](../WDLang1/3029050.md) devuelve el componente rojo de un color RGB. La función [RGBGreen](../WDLang1/3029051.md) devuelve el componente verde de un color RGB

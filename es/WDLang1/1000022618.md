


## Funciones de gestión de controles Procesador de texto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del tratamiento de textos: 



|   |   |
| --- | --- |
| [DocAdd](../WDLang1/1000022919.md) | Agrega: <br><br>- al final de un tratamiento de palabras Document: un elemento, una Document, un párrafo, un texto, una Image o un fragmento.<br><br>- al final de un fragmento: un elemento, un texto o un Image.<br><br><br> |
| [DocChangeZOrder](../WDLang1/1000023970.md) | Cambia el orden Z de un elemento de Imagen o Área de Texto en un DocX Document. |
| [DocClose](../WDLang1/1000022621.md) | Cierra un archivo docx y libera la Document. |
| [DocDelete](../WDLang1/1000024172.md) | Elimina un elemento de un Document. Este elemento es eliminado de su Document asociado. |
| [DocDeleteTableOfContents](../WDLang1/1000024916.md) | Elimina una tabla de contenido de un documento. |
| [DocEndModification](../WDLang1/1000022423.md) | Indica el final de un grupo de operaciones en un procesador de textos Document en un solo Event (gestión "deshacer/rehacer"). |
| [DocExitEdit](../WDLang1/1000023088.md) | Sale de la edición actual. La entrada en edición fue realizada por [DocGoIntoEdit](../WDLang1/1000023077.md). |
| [DocFind](../WDLang1/1000022388.md) | Encuentra una cadena de caracteres: <br><br>- en un procesador de textos completo Document.<br><br>- en un fragmento.<br><br><br> |
| [DocFindElementByName](../WDLang1/1410088857.md) | Busca un elemento por su nombre en un documento de un procesador de texto. |
| [DocGoIntoEditar](../WDLang1/1000023077.md) | Entra en edición en la control Procesador de texto en una sección específica de Document: cuerpo, encabezado o pie de página. |
| [DocInfoXY](../WDLang1/1000025333.md) | Devuelve uno de los siguientes valores para una posición determinada en un control Procesador de texto (coordenadas de un punto del control):<br><br>- el número de la página correspondiente, <br><br>- la posición X correspondiente (en mm), <br><br>- la posición Y correspondiente (en mm)<br><br><br> |
| [DocInsert](../WDLang1/1000022732.md) | Inserta un objeto en un procesador de textos Document o sustituye el contenido del fragmento actual. Los objetos que se pueden utilizar son: <br><br>- un elemento,<br><br>- a Document, <br><br>- a paragraph, <br><br>- un texto, <br><br>- una imagen, etc, <br><br>- un fragmento.<br><br><br> |
| [DocInsertTable](../WDLang1/1000022967.md) | Inserta una tabla en un procesador de textos Document. |
| [DocInsertTableOfContents](../WDLang1/1000024884.md) | Inserta una tabla de contenido en un documento de tipo Procesador de texto. |
| [DocOpen](../WDLang1/1000022622.md) | Abre un archivo de procesamiento de texto (formato docx) en modo Lectura/Escritura. El archivo docx se abre y se bloquea hasta que se cierra. |
| [DocPosition](../WDLang1/1000022743.md) | Activa el scroll de control Procesador de texto para que el texto solicitado se muestre en su totalidad. |
| [DocPositionTopPage](../WDLang1/1410088921.md) | Devuelve la posición del primer carácter de una página determinada en un control Procesamiento de texto. |
| [DocPrint](../WDLang1/1000022640.md) | Imprime una Document en formato "docx". |
| [DocRedo](../WDLang1/1000022425.md) | Restaura la última acción deshecha en un procesador de textos Document. |
| [DocRedoCount](../WDLang1/1000022427.md) | Devuelve el número de acciones canceladas que se pueden rehacer en un procesador de textos Document. |
| [DocReplace](../WDLang1/1000022496.md) | Encuentra todas las instancias de una cadena de caracteres y las reemplaza con otra cadena.: <br><br>- en un procesador de textos completo Document.<br><br>- en un fragmento.<br><br><br> |
| [DocSave](../WDLang1/1000022387.md) | Guarda una Document como archivo "docx". |
| [DocSelect](../WDLang1/1000022391.md) | Selecciona un fragmento de un control Procesador de texto. |
| [DocStartModification](../WDLang1/1000022620.md) | Señala el comienzo de la agrupación de un conjunto de operaciones en un documento en una sola gestión de Event( de "deshacer/rehacer"). |
| [DocToHTML](../WDLang1/1000022393.md) | Convierte un procesador de textos Document en un archivo HTML. |
| [DocToImage](../WDLang1/1000022808.md) | Exporta una página que se encuentra en un procesador de textos Document en formato Image. |
| [DocToPDF](../WDLang1/1000022531.md) | Convierte un procesador de textos Document en un archivo PDF. |
| [DocToText](../WDLang1/1000022389.md) | Crea un string a partir de los datos: <br><br>- se encuentra en una control Procesador de texto o en un procesador de textos Document. <br><br>- encontrada en un párrafo de un control Procesador de texto.<br><br><br> |
| [DocUndo](../WDLang1/1000022424.md) | Deshace la última acción realizada en un procesador de textos Document. |
| [DocUndoCount](../WDLang1/1000022426.md) | Devuelve el número de acciones que se pueden deshacer en una Document. |
| [DocUndoRedoDeleteAll](../WDLang1/1000022734.md) | Borra toda la historia del mecanismo de Deshacer/Rehacer para un procesador de textos Document. |
| [DocUpdateFormula](../WDLang1/1000024897.md) | Actualiza las fórmulas de un documento. |
| [DocUpdateTableOfContents](../WDLang1/1000024885.md) | Actualiza la tabla de contenido de un documento. |

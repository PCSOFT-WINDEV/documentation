
## fSelect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el selector de archivos del sistema actual. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036053&name=fselect_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Select a file
SelectedFile is string
SelectedFile = fSelect("C:\Directories", "File.txt", ...
	"Select a file", ...
	"Text file (*.TXT)" + TAB + "*.txt" + CR + "All" + TAB + "*.*", "txt")
```

## Sintaxis

### Abrir un selector de archivos especificando el directorio inicial y el archivo seleccionado por Default

`<Result> = fSelect(<Initial directory> , <File selected by default> , <Picker title> , <Types of files> , <Default extension> [, <Selection mode>])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre completo de los archivos seleccionados. Si se seleccionan varios archivos, los nombres de los archivos se separan con caracteres CR (Carriage Return).

- Nombre completo del archivo a crear.

- Cadena vacía ("") si no se ha seleccionado o creado ningún archivo.




**`<Initial directory>: (Cadena de caracteres)`**

ruta completa o relativa del directorio inicial (hasta 260 caracteres). Este nombre de directorio puede terminar con "\\" o no. Los archivos encontrados en este directorio se mostrarán en el selector de archivos. Si este parámetro es una cadena vacía (""), el directorio inicial corresponderá al directorio actual.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<File selected by default>: (Cadena de caracteres)`**

Nombre y extensión del archivo seleccionado por Default. Si este parámetro es una cadena vacía (""), no se seleccionará ningún archivo de forma predeterminada.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Picker title>: (Cadena de caracteres)`**

Título que aparece en la barra de título del selector de archivos. Si este parámetro es una cadena vacía (""), se mostrará el título de la ventana WINDEV actual.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Types of files>: (Cadena de caracteres)`**

description de tipos de archivos que pueden ser seleccionados. Este description tiene el siguiente formato:

```txt
<Types Files> + TAB + <List Extensions> [ + CR + ...
<Types Files 2> + TAB + <List Extensions 2>]
```

Donde:

- &lt;Types Files&gt;: cadena de caracteres que describe el tipo de archivo ("BMP (\*.BMP;\*.RLE)", por ejemplo).

- &lt;List Extensions&gt;: extensiones que corresponden al tipo descrito ("\*.BMP;\*.RLE", por ejemplo).




Por ejemplo:
"BMP (\*.BMP;\*.RLE)"+TAB+"\*.BMP;\*.RLE"+RC+"Ícono (\*.ICO)"+TAB+"\*.ICO"+RC+"Otros"+TAB+"\*.GIF;\*.TIF;\*.JPG"

**`<Default extension>: (Cadena de caracteres)`**

Extensión asignada por Default al archivo seleccionado (ver las Notas).

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro no se tiene en cuenta.

**`<Selection mode>: (Constante o combinación de constantes opcional)`**

Permite definir el modo de selección del picker:


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. <br><br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no se tiene en cuenta. |
| *fselChange* | Envía el mensaje <u><u><u><u>MessageSelect</u></u></u></u> a la ventana WINDEV llamadora por cada clic en un archivo que se encuentra en el selector de archivos. |
| *fselCreate* | Abre el selector de archivos en modo creación de archivo (botón "Guardar"). |
| *fselExist* | Si se combina esta constante:<br><br>- con *fselCreate*: Devuelve el nombre del archivo seleccionado. Si el archivo ya existe, aparece un cuadro de diálogo que permite al usuario sobrescribir el archivo anterior.<br><br>- con *fselOpen*: Devuelve el nombre del archivo seleccionado si existe.<br><br><br> |
| *fselMulti* | Abre el selector de archivos en modo de selección múltiple. El usuario puede seleccionar varios archivos utilizando las teclas "Ctrl" y "Shift". |
| *fselOpen* | Abre el selector de archivos en modo de apertura de archivos (botón "Abrir"). |


Por defecto, el selector se llama con la combinación *fselOpen* + *fselExist*.

## Observaciones


### Extensión por defecto (cuando se crea un archivo con la constante fselCreate)
<a name="extension_por_defecto_cuando_crea_archivo_con_constante_fselcreate_ELTPARAGRAPHE000246"></a>

La extensión de un archivo seleccionado (archivo a crear) varía en función de:

- Parámetro &lt;Extensión predeterminada&gt;, 

- la extensión introducida por el usuario.




Por ejemplo, el tabla a continuación presenta la extensión que será tomada en cuenta de acuerdo a &lt;Default extension&gt; y a la extensión introducida por el usuario:

| Parámetro &lt;Extensión predeterminada&gt; | Extensión introducida por el usuario | Extensión que se tiene en cuenta al crear el fichero |
| --- | --- | --- |
| ".TXT" | "Archivo.BMP"<br>BMP es una extensión válida y reconocida. | "Archivo.BMP" |
| ".TXT" | "Archivo.PAF" <br>PAF es una extensión inválida. El nombre del archivo se completará con la extensión &lt;Default&gt;. | "Archivo.PAF.TXT" |
| ".TXT" | "Archivo" <br>No se introduce ninguna extensión. Se utiliza la extensión &lt;Default extension&gt;. | "Archivo.TXT" |
| "" (cadena vacía) | "Archivo.BMP"<br>BMP es una extensión válida y reconocida. | "Archivo.BMP" |
| "" (cadena vacía) | "Archivo.PAF" <br>PAF es una extensión inválida. El nombre del archivo se completará con la primera extensión del "Tipo de archivo" que se muestra.<br>Tipo de archivo visualizado: "Texto (\*.TXT;\*.DOC)" | "Archivo.PAF.TXT" |
| "" (cadena vacía) | "Archivo.PAF" <br>PAF es una extensión inválida. Dado que el tipo de archivo es "\*.\*", se conserva la extensión inválida. <br>Tipo de archivo visualizado: "Todos los archivos (\*.\*)" | "Archivo.PAF" |
| "" (cadena vacía) | "Archivo"<br>No se introduce ninguna extensión. La extensión utilizada es la primera que se encuentra en el "Tipo de archivo" que se muestra.<br>Tipo de archivo visualizado: "Texto (\*.TXT;\*.DOC)" | "Archivo.TXT" |
| "" (cadena vacía) | "Archivo"<br>No se introduce ninguna extensión. Dado que el tipo de archivo es "\*.\*", el nombre del archivo no lleva extensión. <br>Tipo de archivo visualizado: "Todos los archivos (\*.\*)" | "Archivo" |


<a name="NOTE0_2"></a>


### Extensión Default al abrir un archivo
<a name="extension_default_abrir_archivo_ELTPARAGRAPHE000337"></a>

La extensión de un archivo seleccionado (archivo a abrir) corresponde a la extensión seleccionada por Default al abrir el selector de archivos. La primera &lt;Extensión predeterminada&gt; debe corresponder solo a las letras que corresponden a la extensión (por ejemplo, "lst" para seleccionar "\*.lst" de manera predeterminada).
<a name="NOTE0_3"></a>


### Hacer clic en un archivo encontrado en el selector de archivos
<a name="hacer_clic_archivo_encontrado_selector_archivos_ELTPARAGRAPHE000344"></a>

Para ejecutar una Process específica al hacer clic en un archivo encontrado en el selector de archivos, bifurque una Event en el mensaje <u><u><u><u>WM_USER + 600</u></u></u></u> (1624). Cuando se recibe este mensaje:

- el ***lParam*** Variable contiene el Address del archivo seleccionado

- el ***wParam*** Variable contiene el Address del archivo seleccionado.




Por ejemplo, se puede reproducir el sonido seleccionado en el selector de archivos (ver Ejemplo 2, disponible al hacer clic en el enlace "Ejemplo" en la parte superior de la página de ayuda).
<a name="NOTE0_4"></a>


### Selección de directorios
<a name="seleccion_directorios_ELTPARAGRAPHE000360"></a>

La función **fSelect** permite seleccionar archivos. Para seleccionar un directorio, utilice [fSelectDir](../WDLang1/3036050.md).
<a name="NOTE0_7"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Unidades de red, manifiestos y UAC
<a name="unidades_red_manifiestos_uac_ELTPARAGRAPHE000374"></a>

En Windows Vista y versiones posteriores, si el UAC está activado y si se general el ejecutable con un manifiesto que solicita privilegios (ver [Generación del ejecutable](../Editeurs/2025013.md)), el selector de archivos abierto con la función **fSelect** no tendrá acceso a las unidades de red asociadas con el usuario que abrió la sesión de Windows. Para obtener más información, consulte [Preguntas frecuentes sobre Windows Vista y versiones posteriores](../Editeurs/2025016.md).

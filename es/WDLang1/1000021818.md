
## SysRemovableExternalStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si el espacio de almacenamiento externo corresponde a la memoria móvil (tarjeta SD).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copy the file into the application directory on the phone SDCard
FOR i = 1 _TO_ SysNbExternalStorage()
	IF SysRemovableExternalStorage(i) _AND_ SysStatusExternalStorage(i) = sseAvailable THEN
		fCopyFile(sFile, SysDirExternalStorage(i, sseAppFile))
		BREAK
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysRemovableExternalStorage(<Subscript>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el espacio de almacenamiento externo corresponde a memoria móvil,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Subscript>: (Integro)`**

Índice del espacio de almacenamiento externo que se va a utilizar (si el dispositivo incluye varios espacios de almacenamiento externo). 
Observación: [SysNbExternalStorage](../WDLang1/1000021816.md) devuelve el número de espacios de almacenamiento externos encontrados en el dispositivo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un mismo dispositivo puede contener varias áreas de almacenamiento externo (por ejemplo, un área corresponde a la partición "externa" de la memoria permanente y otra área corresponde a la SDCard).

- Antes de acceder a un directorio situado en el área de almacenamiento externo del dispositivo, se recomienda comprobar su disponibilidad con [SysStatusStorageCard](../WDLang1/1000019554.md).

- Atención: Esta función está disponible a partir de Android 5.0.

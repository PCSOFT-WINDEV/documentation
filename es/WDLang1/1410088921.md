
## DocPositionTopPage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición del primer carácter de una página determinada en un control Procesamiento de texto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert an image on page 2
nPosPage2 is int = DocPositionTopPage(WP_MyDocument, 2)
DocInsert(WP_MyDocument, nPosPage2, ElementHeader)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DocPositionTopPage(<Word Processing control> , <Page number>)`
---

**`<Result>: (Entero)`**

Posición del primer carácter de la página especificada. Esta posición se puede utilizar con la función [DocInsert](../WDLang1/1000022732.md) para insertar un elemento en el control Procesador de texto. 

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<Page number>: (Entero)`**

Número de la página en la que se busca la posición.




## &lt;Organigrama&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la índice del elemento que se encuentra en una posición dada en un control Organigrama.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the subscript of the element found at cursor position in the Organization Chart control
ElementNum is int = ORG_MyOrganizationChart.InfoXY(MouseXPos(), MouseYPos())
IF ElementNum <> -1 THEN
	MyWindow.Title = ORG_MyOrganizationChart[ElementNum].Title
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Organization Chart control>.InfoXY(<X> , <Y>)`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado. 

- -1 si no existe ningún elemento en las coordenadas especificadas.




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar.

**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar.

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a estudiar.

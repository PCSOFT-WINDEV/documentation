


## La estructura del archivo de control remoto
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000117"></a>
La gestión de control a distancia utiliza un archivo en formato INI. Este archivo control le permite gestionar fácilmente el modo de ejecución de una aplicación, así como los mensajes mostrados según el modo. Por Default, este archivo se llama WDUPDATE.NET.

WDADMINEXE le permite gestionar fácilmente la creación y el mantenimiento de este archivo control.

<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Estructura del archivo control
<a name="estructura_del_archivo_control_ELTTEXTE000141"></a>
La estructura del archivo de control es la siguiente:


```txt
[<Name of the EXE file to control>]
MODE = 
MSG_INTERDIT = 
MSG_AVERTISSEMENT =
MSG_ARRET =
DELAI =
TEMPS_AFFICHAGE =
```
Esta estructura se repite para cada ejecutable que debe ser control led por el archivo control.

<a name="NOTE3"></a>
<a name="NOTE3_1"></a>


## Las diferentes palabras clave del archivo control
<a name="las_diferentes_palabras_clave_del_archivo_control_ELTTEXTE000165"></a>
El archivo control contiene los siguientes elementos:


|   |   |
| --- | --- |
| Nombre del archivo EXE a verificar | Nombre del ejecutable a control. Si el ejecutable es SALESMGT.EXE, el nombre del archivo EXE a control será SALESMGT. |
| modelar | Modo de la aplicación controlada:<br><br>- 0: modo de funcionamiento estándar de la aplicación<br><br>- 1 (*acForbidden* mode): acceso denegado a la solicitud<br><br>- 2 (*acWarning* mode): advertencia de parada inminente<br><br>- 3 (*acStop* mode): desconexión de aplicaciones<br><br><br> |
| MSG_INTERDIT | Mensaje que se mostrará si el acceso a la aplicación no está permitido (modo *acForbidden*).<br>Este mensaje se mostrará a cada nuevo usuario que intente conectarse a la aplicación. |
| MSG_AVERTISSEMENT | Mensaje que se mostrará para señalar una interrupción inminente de la aplicación (modo *acWarning*).<br>Este mensaje se mostrará a intervalos regulares para informar al usuario y permitirle completar la tarea actual antes de que la aplicación se detenga. |
| MSG_ARRET | Mensaje que se mostrará para señalar la interrupción de la aplicación (modo *acStop*).<br>Este mensaje se mostrará antes de que la aplicación se detenga. |
| DELAY | Tiempo de espera para comprobar el archivo control (expresado en minutos). Este tiempo de espera es de set a 5 minutos de Default. |
| TIEMPO_DE_PANTALLA | Duración de la visualización (expresada en segundos) de las ventanas de advertencia. Si el usuario no realiza ninguna acción, la ventana se cerrará automáticamente.. Por defecto, esta duración es set a 10 segundos. |


## WinEdSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva una control cuando la interfaz es modificada por el usuario final (modo "modificación de UI por el usuario final" activado). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselects the current control
X is Control
X <- WinEdSelect(WIN_Form) 
WInEdSelectMinus(WIN_Form, X)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinEdSelectMinus(<Window> , <Control used>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. Esta ventana debe estar en modo de modificación ([WinEdActive](../WDLang1/1000021843.md)).  

**`<Control used>: (Opcional control)`**

control para anular la selección en la ventana. La control seleccionada es devuelta por [WinEdSelect](../WDLang1/1000021871.md). 

- Si se permite la selección múltiple, la control especificada se elimina de la selección. 

- La función no tiene efecto si no se selecciona la control especificada. 




Si no se especifica este parámetro: 

- si se permite la selección múltiple, se deseleccionan todos los controles de ventana.

- si la selección múltiple no está permitida, deselecciona la control seleccionada.

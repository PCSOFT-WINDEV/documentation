


## OrgAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un nuevo elemento raíz a una control Organigrama. Si la control Organigrama es relacionado para una fuente de datos, el elemento también se añade a la fuente de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a "Root" element into the Organization Chart control (Syntax 1)
Boss is int 
Boss = OrgAdd(ORG_MyOrganizationChart, "Vince Boss", "Big boss", "Goldbar.gif")
// Modify the background color of this element
ORGA_MyOrganizationChart[Boss].BackgroundColor = iLightYellow
```
<a name="Example2"></a>

```wl
// Add elements into the Organization Chart control (syntax 2)
// Root element
ElementCEO is OrgElement
ElementCEO.Title = "Big Boss"
OrgAdd(ORG_EXECUTIVE, ElementCEO)

// Child elements
Element2 is OrgElement
Element2.Title = "Production"
OrgAddChild(ORG_EXECUTIVE, 1, Element2)

Element3 is OrgElement
Element3.Title = "Sales"
OrgAddChild(ORG_EXECUTIVE, 1, Element3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadir un elemento "raíz" especificando sus características

`<Result> = OrgAdd(<Organization Chart control> , <Title> [, <Content> [, <Image>]])`
---

**`<Result>: (Integro)`**



- Índice del elemento añadido. 

- -1 si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a la que se añadirá el elemento.

**`<Title>: (Cadena de caracteres)`**

Título del elemento.

**`<Content>: (Cadena de caracteres opcional)`**

Contenido del elemento.

**`<Image>: (Cadena de caracteres opcional)`**

Imagen associated con el elemento. Esta Image se muestra a la izquierda del título del elemento. Este parámetro corresponde al nombre y a la ruta completa (o relativa) de Image. Se puede utilizar una ruta de tipo UNC.


<a name="SYNTAX2"></a>

### Añadiendo un elemento raíz usando el tipo OrgElement

`<Result> = OrgAdd(<Organization Chart control> , <Element>)`
---

**`<Result>: (Integro)`**



- Índice del elemento añadido.

- -1 si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a la que se añadirá el elemento.

**`<Element>: (OrgElement Variable)`**

Nombre de la [OrgElement](../WDLang1/1000019713.md) Variable que describe las características del elemento a añadir.

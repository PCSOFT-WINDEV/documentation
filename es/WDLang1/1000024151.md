


## Funciones de control Panel acoplable
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para administrar los controles del panel acoplable: 



|   |   |
| --- | --- |
| [PanelActivar](../WDLang1/1000024396.md) | Activa un panel cuando varios paneles están acoplados entre sí. |
| [PanelAttach](../WDLang1/1000023990.md) | Conecta o cambia la posición de conexión de una control Panel acoplable. |
| [PanelCreate](../WDLang1/1000023989.md) | Crea una nueva control Panel acoplable en la ventana actual. |
| [PanelDetach](../WDLang1/1000023991.md) | Desprende un control Panel acoplable para transformarlo en una ventana flotante. |
| [PanelEnumerate](../WDLang1/1000023993.md) | Enumera los controles del panel acoplable en una ventana. |
| [PanelLoadConfiguration](../WDLang1/1000023995.md) | Restaura la configuración de los controles del Panel acoplable en una ventana. |
| [PanelPosición](../WDLang1/1000023992.md) | Recupera la posición de un control Panel acoplable. |
| [PanelSaveConfiguration](../WDLang1/1000023994.md) | Guarda la configuración actual de los controles del panel acoplable de una ventana. |

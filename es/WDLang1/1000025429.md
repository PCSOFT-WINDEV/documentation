
## Procedimiento WLanguage llamado por la función OKCancelAsynchronous

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [OKCancelAsynchronous](../WDLang1/1000025308.md) cuando el usuario hace clic en uno de los botones del cuadro de diálogo. Este procedimiento permite saber qué botón se pulsó, y ejecutar un proceso específico, si es necesario. 
Observación: Este procedimiento puede ser local, global, interno o lambda. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OKCancelAsynchronous("Which button do you choose?", MyProcedure_OKCancel)

INTERNAL PROCEDURE MyProcedure_OKCancel(nResponse int){
	SWITCH nResponse
		CASE 1: ToastDisplay("OK button")
		CASE 0: ToastDisplay("Cancel button")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OKCancelAsynchronous_Callback(<Selected button>)`
---

**`<Selected button>: (Entero)`**

Identifica el botón seleccionado por el usuario:


|   |   |
| --- | --- |
| *False* (o 0) | El botón seleccionó el botón "Cancelar". |
| *True* (o 1) | El botón seleccionó el botón "OK". |

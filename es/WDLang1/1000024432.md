


## &lt;Variable picLayer&gt;.ChangeMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el modo de dibujo en una variable de tipo [picLayer](../WDLang1/1000024604.md). Este modo es utilizado por las funciones de dibujo ([&lt;Variable picLayer&gt;.DrawLine](../WDLang1/1000024495.md), [&lt;Variable picLayer&gt;.DrawRectangle](../WDLang1/1000024564.md), etc.).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Draws a rectangle in a layer and erases it
MyLayer.ChangeMode(drawXOR)
// Displays the rectangle (in XOR mode)
MyLayer.DrawRectangle(0, 0, 100, 100)
Multitask(10)
// Erases the rectangle
MyLayer.DrawRectangle(0, 0, 100, 100)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<picLayer image>.ChangeMode(<New mode>)`
---

**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<New mode>: (Constante de tipo Integer)`**

Modo de dibujo en la imagen:



|   |   |
| --- | --- |
| *drawAntiAliasing* | Utiliza el GDI+ y/o las funcionalidades de la tarjeta gráfica para dibujar líneas y círculos sin efecto de dientes de sierra. Este modo de dibujo es más lento pero el resultado es mejor que con el modo estándar.<br> |
| *drawDefault* | Colores estándar. |
| *drawXOR* | Colores aplicados mediante una operación de "O Exclusivo" con el fondo. Borra un dibujo si esta opción se utiliza dos veces.<br>ATENCIÓN: Este modo de dibujo no es compatible con la gestión de la opacidad (el dibujo no se borra)..<br>**Atención**: <br><br>- El dibujo en modo XOR solo está disponible para controles Imagen. El dibujo en modo XOR no está disponible para variables de tipo [Image](../WDLang1/1000019650.md). <br><br>- La constante *drawXOR* no puede combinarse con la constante *drawAntiAliasing*. <br><br><br> |

### Antialiasing
<a name="antialiasing_ELTPARAGRAPHE000422"></a>El antialiasing consiste en dibujar líneas y círculos sin efecto de dientes de sierra. Esta funcionalidad requiere el framework GDI+ (es decir, el framework de WINDEV).
Ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dchangemode.gif)
Límites:

- La función [&lt;Variable picLayer&gt;.DrawChord](../WDLang1/1000024469.md) no está soportada en este modo.

- Para evitar que los bordes tengan un efecto de dientes de sierra, los colores y las líneas de los elementos se suavizan. Algunas funciones de dibujo pueden tener un comportamiento inesperado porque ya no se encuentran los colores especificados. Por ejemplo, si se especifica un color para el borde con la función [&lt;Variable picLayer&gt;.Fill](../WDLang1/1000024588.md), y este color no se encuentra: se rellena toda el área.

- Para un dibujo con antialiasing, las funciones [&lt;Variable picLayer&gt;.DrawPolygon](../WDLang1/1000024504.md) y [&lt;Variable picLayer&gt;.DrawSlice](../WDLang1/1000024561.md) tienen el siguiente comportamiento. El contorno se dibuja por encima del interior del polígono: por lo tanto, la línea se combina parcialmente con el interior del polígono. Si la línea es muy gruesa, se mostrará en dos colores.

- Para obtener más información sobre el uso de GDI+, consulte [Framework GDI+](../Editeurs/2020007.md).

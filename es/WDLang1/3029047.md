
## gFont( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres que describen la fuente que se utilizará para el texto.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **En una ventana**, esta cadena define la fuente del texto:  

- en un elemento de un List Box o control Combo Box.

- en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 
	Observación: Puede ser necesario ajustar el texto una vez definido el contenido.









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Writes "Chicken" in Arial
ListAdd(LIST_Animals, gFont("Arial") + "Chick")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gFont(<Font name> [, <Charset used>])`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que describen la fuente que se utilizará para el texto. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<Font name>: (Cadena de caracteres)`**

Nombre exacto de la fuente. Si este parámetro corresponde a una cadena vacía (""), la función **gFont** no tiene ningún efecto.

**`<Charset used>: (Constante opcional de tipo Integer)`**

Constante correspondiente al carácter set a utilizar. El carácter por defecto set se utilizará si este parámetro no se especifica.  


|   |   |
| --- | --- |
| *charsetAnsi* | Caracteres romanos en estándar ANSI |
| *charsetArabic* | Caracteres arábigos |
| *charsetBaltic* | Caracteres bálticos |
| *charsetChinese* | Caracteres chinos (República Popular China) |
| *charsetDefault* | Utiliza el conjunto de caracteres predeterminado del equipo. No se fuerza ningún conjunto de caracteres. |
| *charsetEastEurope* | Caracteres de Europa del Este (polaco, etc.) |
| *charsetGreek* | Caracteres griegos |
| *charsetHebrew* | Caracteres hebreos |
| *charsetJapanese* | Caracteres japoneses |
| *charsetKorean* | Caracteres coreanos |
| *charsetMac* | Conjunto de caracteres Mac Roman (utilizado en versiones anteriores de Macintosh). |
| *charsetOccidental* | Caracteres romanos en estándar ANSI |
| *charsetRussian* | Caracteres rusos |
| *charsetThai* | Caracteres tailandeses |
| *charsetTraditionalChinese* | Caracteres chino tradicional (República de Taiwán) |
| *charsetTurkish* | Caracteres turcos |
| *charsetVietnamese* | Caracteres vietnamitas |






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La fuente Default se restablece después de la llamada a esta función.

- Todas las demás características de la fuente se mantienen (negrita, ....).

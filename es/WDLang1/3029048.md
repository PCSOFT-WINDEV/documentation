
## ColorLightness (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la luminosidad de un color.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SelectColor(MyColor)
Info("Hue: " + ColorHue(MyColor) + "°", ...
	"Lightness: " + ColorLightness(MyColor) + "%", ...
	"Saturation: " + ColorSaturation(MyColor) + "%")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ColorLightness(<Color>)`
---

**`<Result>: (Entero)`**

Valor de la saturación del color especificado (porcentaje comprendido entre 0 y 100).

**`<Color>: (Entero)`**

Color a utilizar. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

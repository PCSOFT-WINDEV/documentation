
## SchedulerModifyTrigger( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica los parámetros de una tarea programada, es decir, los parámetros que definirán la ejecución de una tarea. Los parámetros utilizados para modificar la programación de la tarea son los especificados en la estructura [Activación de tareas programadas](../WDLang1/3089002.md).

**Observación**: Para rellenar la estructura TriggerScheduledTask con los parámetros de una programación de tareas, utilice [SchedulerTriggerProperties](../WDLang1/3089010.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The task is a daily task, run every day at 10:00 from today
TriggerScheduledTask.Type = schedDaily
TriggerScheduledTask.Time = 1000
TriggerScheduledTask.BeginDate = Today()
ScheduleNum = SchedulerAddTrigger("MyTask")

...

// Modify the execution time
TriggerScheduledTask.Time = 1030
SchedulerModifyTrigger("MyTask", ScheduleNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SchedulerModifyTrigger(<Task name> , <Schedule identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la operación,

- <u><u><u><u>False</u></u></u></u> si se produce un error.




**`<Task name>: (Cadena de caracteres)`**

Nombre de la tarea a utilizar.

**`<Schedule identifier>: (Integro)`**

Identificador del programa a utilizar. Este identificador se definió en el momento de la creación del calendario por parte de [SchedulerAddTrigger](../WDLang1/3089001.md).

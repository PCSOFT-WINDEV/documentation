
## Italy.CheckIVA (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número de IVA (número de identificación de empresa en Italia).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyIVANumber is string = "IT07643520567"
Italy.CheckIVA(MyIVANumber)	// Returns True

// Check a control
EDT_IVANumber = "ABC"
Italy.CheckIVA(EDT_IVANumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Italy.CheckIVA(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número de IVA válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número de IVA a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

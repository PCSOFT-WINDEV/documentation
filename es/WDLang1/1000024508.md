


## &lt;Document Variable&gt;.Cerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el archivo docx asociado a la [Document](../WDLang1/1000022461.md) Variable y libera la Document. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

MyDocument.Close()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Document>.Close()`
---

**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

El archivo físico asociado a la Variable (si existe) se cierra. Las modificaciones realizadas en el archivo no se guardarán hasta que se cierre (utilice [&lt;Document Variable&gt;.Guardar](../WDLang1/1000024537.md) si es necesario).

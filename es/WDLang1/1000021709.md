
## Funciones para la gestión de los controles de las pestañas (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de las pestañas (mediante una sintaxis de prefijo): 



|   |   |
| --- | --- |
| [ControlPane](../WDLang1/1000021625.md) | Devuelve el nombre del panel estático o dinámico que se encuentra en una control Pestaña associated con una control. |
| [ControlTab](../WDLang1/3025028.md) | Devuelve el nombre del control Pestaña que contiene el control especificado. |
| [&lt;Cerrar](../WDLang1/1000021702.md) | Cierra un panel dinámico en un control Pestaña. |
| [TabCloseAll](../WDLang1/1000021703.md) | Cierra todos los cristales dinámicos de una control Pestaña. |
| [&lt;LoadConfiguration](../WDLang1/1000021699.md) | Restaura la configuración de los paneles dinámicos en un control Pestaña. |
| [&lt;Abrir](../WDLang1/1000021704.md) | Abre un nuevo panel en una control Pestaña dinámica. |
| [&lt;SaveConfiguration](../WDLang1/1000021708.md) | Guarda la configuración de los paneles dinámicos que se encuentran en un control Pestaña en formato de cadena de caracteres. |
| [&lt;Estado](../WDLang1/1000021700.md) | Devuelve el estado de un panel de pestañas dinámico (identificado por su alias). |

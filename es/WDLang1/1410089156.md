


## Funciones de identificadores administrativos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de números de identificación administrativa son las siguientes: 



|   |   |
| --- | --- |
| [Belgium.CheckCBE](../WDLang1/1410089348.md) | Comprueba la validez de un número CBE belga (Crossroads Bank for Enterprises). |
| [Belgium.CheckEU](../WDLang1/1410089350.md) | Comprueba la validez de un número EU (número de unidad de establecimiento para identificar empresas en Bélgica). |
| [Belgium.CheckNISS](../WDLang1/1410089349.md) | Comprueba la validez de un número NISS (número de seguridad social en Bélgica). |
| [Brazil.CheckCNPJ](../WDLang1/1410089249.md) | Comprueba la validez de un número CNPJ (número de identificación de empresa en Brasil). |
| [Brazil.CheckCPF](../WDLang1/1410089245.md) | Comprueba la validez de un número CPF (número de identificación fiscal en Brasil). |
| [Canada.CheckBN15](../WDLang1/1410089352.md) | Comprueba la validez de un número BN15 (número de empresa canadiense de 15 dígitos). |
| [Canada.CheckBN9](../WDLang1/1410089353.md) | Comprueba la validez de un número BN9 (número de empresa canadiense de 9 dígitos). |
| [Canada.CheckSIN](../WDLang1/1410089351.md) | Comprueba la validez de un número SIN (número de seguridad social canadiense). |
| [France.CheckINSEE](../WDLang1/1410089315.md) | Prueba la validez de un número INSEE (número de identificación de personas físicas asignado por el Instituto Nacional de Estadística y Estudios Económicos). |
| [France.CheckSIREN](../WDLang1/1410089365.md) | Comprueba la validez de un número de SIREN francés (Système d'Identification du Répertoire des Entreprises). |
| [France.CheckSIRET](../WDLang1/1410089347.md) | Comprueba la validez de un número de SIRET francés (Système d'Identification du Répertoire des Etablissements). |
| [Germany.CheckIdNr](../WDLang1/1410089244.md) | Comprueba la validez de un número IdNr (número de identificación fiscal en Alemania). |
| [Germany.CheckUStIdNr](../WDLang1/1410089248.md) | Comprueba la validez de un número USt-IdNr (número de identificación de empresa en Alemania). |
| [Italy.CheckCF](../WDLang1/1410089243.md) | Comprueba la validez de un número de CF (código fiscal italiano). |
| [Italy.CheckIVA](../WDLang1/1410089247.md) | Comprueba la validez de un número de IVA (número de identificación de empresa en Italia). |
| [Spain.CheckDNI](../WDLang1/1410089242.md) | Comprueba la validez de un número de DNI (número de documento nacional de identidad en España). |
| [Spain.CheckNIF](../WDLang1/1410089246.md) | Comprueba la validez del NIF (número de identificación fiscal en España). |
| [Switzerland.CheckBER](../WDLang1/1410089356.md) | Comprueba la validez de un número BER (número de identificación en el registro de empresas en Suiza). |
| [Switzerland.CheckOASI](../WDLang1/1410089355.md) | Comprueba la validez de un número OASI (número de pensión pública en Suiza). |
| [Switzerland.CheckUID](../WDLang1/1410089354.md) | Comprueba la validez de un número UID (número de identificación de empresa en Suiza). |
| [UnitedStates.CheckEIN](../WDLang1/1410089226.md) | Comprueba la validez de un número EIN (número de identificación del empleador en Estados Unidos). |
| [UnitedStates.CheckFEI](../WDLang1/1410089250.md) | Comprueba la validez de un número FEI (identificador de establecimiento estadounidense). |
| [UnitedStates.CheckSSN](../WDLang1/1410089225.md) | Comprueba la validez de un número SSN (Número de seguro social estadounidense). |

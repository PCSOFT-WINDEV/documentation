


## PVTExpandAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Expande todas las cabeceras de fila y/o de columna en un control Tabla dinámica. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Loads the data file
PVTLoad(PVT_Statistics, sFile)
// Expands all headers of the Pivot Table control
PVTExpandAll(PVT_Statistics)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PVTExpandAll(<Pivot Table control> [, <Type of header>])`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar. Esta control Tabla dinámica debe haber sido rellenada por [PVTCcalcularTodos](../WDLang1/1000020539.md) o por [PVTLoad](../WDLang1/1000020507.md). 

**`<Type of header>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Clase de la cabecera a desglosar: 


|   |   |
| --- | --- |
| *pvtColumnHeader* | Despliega las cabeceras de las columnas. |
| *pvtRowHeader* | Desglosa las cabeceras de las filas. |


Si no se especifica este parámetro, se expanden los encabezados de filas **Y** columnas.




## MenuInsertSeparator (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un separador en un menú. Para insertar un separador al final del menú, utilice [MenuAddSeparator](../WDLang1/1000017270.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a new menu
MenuAddMenu(_Menu, "MNU_Schedule", "Schedule")
// Add options into the new menu
MenuAddOption("MNU_Schedule", "MNU_Create", "Create", Create_Schedule)
MenuAddOption("MNU_Schedule", "MNU_View", "View", View_schedule)
// Insert a separator
MenuInsertSeparator("MNU_Schedule", "MNU_Create")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un separador (posición definida por un nombre de opción de menú)

`MenuInsertSeparator(<Source menu> , <Insertion option>)`
---

**`<Source menu>: (Nombre del menú)`**

Nombre del menú donde se debe insertar el separador.

**`<Insertion option>: (Cadena de caracteres)`**

Nombre de la opción de menú utilizada como posición base para la inserción. El nuevo separador se insertará antes de esta posición.  


<a name="SYNTAX2"></a>

### Insertar un separador (posición definida por un índice)

`MenuInsertSeparator(<Source menu> , <Subscript>)`
---

**`<Source menu>: (Nombre del menú)`**

Nombre del menú donde se debe insertar el separador.

**`<Subscript>: (Integro)`**

Subíndice de la opción de menú utilizada como posición base para la inserción. El nuevo separador se insertará antes de esta posición.

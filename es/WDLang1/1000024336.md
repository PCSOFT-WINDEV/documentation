


## &lt;Imagen&gt;.PixelOpacity (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la opacidad de un pixel en una imagen (en un control Imagen) que admite canales Alfa.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the opacity of the pixel (15, 25)
// on the "IMG_ImageDrawing" control
ResOpacity = IMG_ImageDrawing.PixelOpacity(15, 25)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Image control>.PixelOpacity(<X-coordinate> , <Y-coordinate>)`
---

**`<Result>: (Entero)`**

Opacidad del pixel especificado (comprendido entre 0 para transparente y 255 para opaco).

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<X-coordinate>: (Entero)`**

Posición horizontal del pixel a manipular.

**`<Y-coordinate>: (Entero)`**

Posición vertical del pixel a manipular.

<a name="NOTE0_1"></a>

## Observaciones


### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000348"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dpixel.gif)
Las coordenadas se indican en pixeles con respecto a la esquina superior izquierda de la imagen (coordenadas (0,0)).

Para obtener la posición de un clic en un control Imagen, utilice las funciones [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md).
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>


### Dibujo en una imagen con opacidad
<a name="dibujo_una_imagen_con_opacidad_ELTPARAGRAPHE000449"></a>

Para especificar que el dibujo de la imagen admite la opacidad (canal Alfa), utilice la función [&lt;Imagen&gt;.StartDrawing](../WDLang1/1000024326.md) y la constante ***dWithOpacity***.

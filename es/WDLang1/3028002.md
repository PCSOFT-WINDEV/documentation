
## DDEString( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la información recuperada por [Recuperación de DDER](../WDLang1/3028017.md).

**Recordatorio**: [Recuperación de DDER](../WDLang1/3028017.md) se utiliza para recuperar un dato. Estos datos se almacenan en **DDEString**.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Runs a server action
DDEExecute(ConnectionNum, "EXE1")
IF DDERetrieve(ConnectionNum, "Item1") THEN
	Info("The retrieved data is" + DDEString())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEString([<Connection identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Información recuperada.

**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, **DDEString** utiliza la última conexión gestionada.

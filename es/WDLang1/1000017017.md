
## SysNameMainScreen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de la pantalla principal de el equipo.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the name of the main screen
Info(SysNameMainScreen()) 
// Displays for example: \\.\DISPLAY2\Monitor2 Plug-and-Play screen
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysNameMainScreen()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la pantalla principal,

- Cadena vacía ("") si se produce un error.


## MatInvert( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la inversa de una matriz cuadrada.

**Observación**: La matriz inversa puede ser calculada:

- de una matriz cuadrada (el número de filas es igual al número de columnas)

- si el determinante (devuelto por [MatDeterminant](../WDLang1/3050045.md)) no es nulo.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the inverse of a square matrix
InverseResult = MatInvert("MyMatrix", "MyInverseMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatInvert(<Inverse matrix to calculate> , <Calculated inverse matrix>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la inversión,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Inverse matrix to calculate>: (Cadena de caracteres)`**

Nombre de la matriz inversa a calcular, definida con [MatCreate](../WDLang1/3050058.md).

**`<Calculated inverse matrix>: (Cadena de caracteres)`**

Nombre de la matriz inversa, creada automáticamente durante la inversión.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz inversa
<a name="existencia_matriz_inversa_ELTPARAGRAPHE000097"></a>

No se verifica la existencia de la matriz.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Matrices idénticas
<a name="matrices_identicas_ELTPARAGRAPHE000108"></a>

Se produce un error de WLanguage si &lt;Matriz inversa a calcular&gt; y &lt;Matriz inversa calculada&gt; son idénticas.

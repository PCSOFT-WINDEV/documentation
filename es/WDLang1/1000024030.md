


## &lt;Tipo Date&gt;.ToDay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el día que corresponde a una fecha determinada.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDate is Date
MyDate = "20011225"
MyDate.ToDay() // Returns 2

MyDate = DateSys
IF MyDate.ToDay() > 5 THEN
	Info("It's the weekend")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Date>.ToDay()`
---

**`<Result>: (Entero)`**

Número del día de la semana que corresponde a la fecha especificada:
1 - Lunes
2 - Martes
3 - Miércoles
4 - Jueves
5 - Viernes
6 - Sábado
7 - Domingo

**`<Date>: (Variable de tipo Date o DateTIme)`**

Fecha a manipular. Esta fecha puede corresponder a : 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999.

Los cálculos sobre las fechas realizadas con las [funciones WLanguage](../WDLang1/1000023943.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.


## ScrollbarPosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite gestionar la posición de las barras de desplazamiento: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Modifica o recupera la posición de una de las barras de desplazamiento horizontales o verticales que se encuentran en una control de una aplicación WINDEV o WINDEV Mobile. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Synchronize the scrollbars of two Table controls
nPos = ScrollbarPosition(TABLE_MyTable, sbVert)
ScrollbarPosition(TABLE_MyTable2, sbVert, nPos)
```





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la posición de la barra de desplazamiento

`<Result> = ScrollbarPosition(<Element used> [, <Orientation>])`
---

**`<Result>: (Integro)`**



- Posición de la caja de la barra de desplazamiento. 

- 0 si la barra de desplazamiento no existe o si es Grayed.  




**`<Element used>: (control o nombre de la página)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Nombre del control utilizado. 
	




**`<Orientation>: (Constante opcional de tipo Integer)`**

Orientación de la barra de desplazamiento para la que debe recuperarse la posición: 


|   |   |
| --- | --- |
| *sbHorz* | La posición se calcula para la barra de desplazamiento horizontal de control. |
| *sbVert*<br>(Valor predeterminado) | La posición se calcula para la barra de desplazamiento vertical de control. |




<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
### Modificar la posición de la barra de desplazamiento

`<Result> = ScrollbarPosition(<Control used> , <Orientation> , <New position>)`
---

**`<Result>: (Integro)`**

Nueva posición de la barra de desplazamiento.  

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado.


**`<Orientation>: (Constante opcional de tipo Integer)`**

Orientación de la barra de desplazamiento para la que se debe modificar la posición: 


|   |   |
| --- | --- |
| *sbHorz* | La posición se modifica para la barra de desplazamiento horizontal de la control. |
| *sbVert* | La posición se modifica para la barra de desplazamiento vertical de la control. |



**`<New position>: (Entero opcional)`**

Nueva posición de la barra de desplazamiento. La posición de la barra de desplazamiento comienza a partir de 1. 

Una posición correcta es forzada si el valor especificado no está incluido dentro de los límites especificados para la barra de desplazamiento de control. 

La nueva posición se ignora si la barra de desplazamiento no existe o si es Grayed: la función no tiene ningún efecto.

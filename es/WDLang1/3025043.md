
## RTFSearch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra una cadena de caracteres en un RTF control (que se encuentra en una ventana o en un WINDEV reporte) o en una cadena de caracteres que contiene texto en formato RTF.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch all the occurrences of "WINDEV" to red and underlined
I is int
I = RTFSearch(EDT_Edit1, "WINDEV")
WHILE I > 0
	// Switch to red
	RTFSelection(EDT_Edit1, rtfColor, LightRed, I, 6)
	// Switch to underlined
	RTFSelection(EDT_Edit1, rtfUnderlined, True, I, 6)
	// Find the next "WINDEV"
	I = RTFSearch(EDT_Edit1, "WINDEV", rtfCaseSensitive, I + 1)
END
```


<a name="Example2"></a>



```wl
nRes is int
sString is string = EDT_Edit1
// ****** USES A STRING VARIABLE
// Find the words containing the "RTF" string
nRes = RTFSearch(sString, "RTF", rtfFullWord, 1)
WHILE nRes <> 0
	nRes = RTFSearch(sString, "RTF", rtfFullWord, nRes + 3)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RTFSearch(<RTF control or string> , <String to find> [, <Options> [, <Start>]])`
---

**`<Result>: (Integro)`**



- Posición del primer carácter de la cadena buscada en la cadena inicial. 

- 0 si no se encuentra la cadena buscada. 


El formato RTF se ignora.

**`<RTF control or string>: (Cadena de caracteres)`**

Nombre de la cadena RTF control o RTF donde se realizará la búsqueda.

**`<String to find>: (Cadena de caracteres)`**

Cadena de caracteres que debe encontrarse en la control o en la cadena.

**`<Options>: (Constante o combinación de constantes opcional)`**

Indica los parámetros para la búsqueda: 



|   |   |
| --- | --- |
| *rtfCaseSensitive* | La búsqueda distingue entre mayúsculas y minúsculas. |
| *rtfFullWord* | Sólo se buscan las palabras completas. |



**`<Start>: (Entero opcional)`**

Posición del carácter desde el que se busca el string especificado. 
Si no se especifica este parámetro, la búsqueda comienza en la posición del cursor (o al final del texto seleccionado) y continúa hasta el final del texto en la control o cadena de caracteres.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Encontrar una cadena de caracteres
<a name="encontrar_una_cadena_caracteres_ELTPARAGRAPHE000058"></a>

- Por Default, la búsqueda no distingue entre mayúsculas y minúsculas y se buscan todos los caracteres especificados.

- La búsqueda se realiza desde el final de la selección (posición del cursor del ratón) hasta el final del texto que se encuentra en la control o cadena.

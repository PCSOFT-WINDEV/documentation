


## SpreadsheetDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los datos de un control Hoja de cálculo encontrado en una ventana (celdas, fórmulas, tamaño y disposiciones de cabecera).. La lista de operaciones almacenadas en "undo/redo" también se borra.: por lo tanto, el borrado no se puede cancelar. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SpreadsheetDeleteAll(PSHEET_Spreadsheet)
SpreadsheetAddData(PSHEET_Spreadsheet, "A2", 2)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetDeleteAll(<Spreadsheet control>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


## InWebSocketServerGlobalServiceMode( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el código actual se está ejecutando en el servicio global del servidor WebSocket.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InWebSocketServerGlobalServiceMode() THEN
	// Code specific to the global service of the WebSocket server
ELSE
	// Code to execute in other cases
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InWebSocketServerGlobalServiceMode()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código se ejecuta en el servicio global del servidor WebSocket, 

- <u><u><u><u>False</u></u></u></u> en los demás casos. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000160"></a>

La función **InWebSocketServerGlobalServiceMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InWebSocketServerGlobalServiceMode** pueden compartirse entre varios productos sin generar errores de compilación.

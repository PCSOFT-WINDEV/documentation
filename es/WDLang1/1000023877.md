
## Funciones de gestión del lector de PDF control (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de PDF Reader (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [PDFReaderAbrir](../WDLang1/1000023776.md) | Abre un archivo PDF que se muestra en un lector de PDF control. |
| [PDFReaderCerrar](../WDLang1/1000023695.md) | Cierra y libera el archivo PDF que se muestra en un lector de PDF control. |
| [&lt;Lector PDF&gt;.GoToBookmark](../WDLang1/1410089536.md) | Establece la posición directamente en el marcador deseado en el documento PDF. |
| [.Highlight](../WDLang1/1000023545.md) | Encuentra un texto en el Document mostrado por un lector de PDF control y resalta las ocurrencias de este texto en el control. |
| [.InfoXY](../WDLang1/1000025359.md) | Para una posición dada (coordenadas de un punto) en un PDF Reader control, devuelve una de las siguientes opciones: <br><br>- el número Page para la posición especificada, <br><br>- la posición X (en mm) para la posición especificada, <br><br>- la posición Y (en mm) para la posición especificada.<br><br><br> |
| [.RotatePage](../WDLang1/1000024412.md) | Gira una página mostrada en el lector de PDF control. |
| [&lt;Lector PDF&gt;.Save](../WDLang1/1000024900.md) | Guarda el contenido de un control Lector PDF en un archivo. |
| [.SeekFirst](../WDLang1/1000023778.md) | Inicializa la búsqueda de un texto en un archivo PDF visualizado por el lector PDF control y se posiciona en la primera ocurrencia encontrada (a partir del Page actual). |
| [.SeekNext](../WDLang1/1000023779.md) | Establece la posición en el siguiente resultado de una búsqueda en el contenido de un lector de PDF control. |
| [PDFReaderSeekPrevio](../WDLang1/1000023777.md) | Establece la posición en el resultado anterior de una búsqueda iniciada con [.SeekFirst](../WDLang1/1000023778.md) en el contenido de un lector de PDF control. |





Observación: [iPrintPDF](../WDLang5/1000020519.md) puede utilizarse para imprimir el contenido de un lector de PDF control.

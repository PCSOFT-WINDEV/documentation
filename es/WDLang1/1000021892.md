
## SysLinkExtension( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Asocia una extensión (.txt, .avi, etc.) a un programa de ejecución por defecto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Links the ".txt" extension to the "app.exe" executable
SysLinkExtension(".txt", "C:\Program Files\MyApp\app.exe ""%1""", sysCurrentUser)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysLinkExtension(<Extension> , <Command> [, <User>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la asociación,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles.




**`<Extension>: (Cadena de caracteres)`**

Extensión del enlace. 

**`<Command>: (Cadena de caracteres)`**

Comando para ejecutar. Este comando tiene el siguiente formato: "FullPathOfExecutable %1"
donde %1 representa el archivo seleccionado para pasar en el argumento. Le aconsejamos que lo adjunte entre comillas para garantizar su correcta ejecución..
Observación: Este parámetro puede contener atajos de sistema como %systemroot%.

**`<User>: (Constante opcional de tipo Integer)`**

La asociación entre la aplicación y la extensión se guarda en el registro.. Hay dos opciones disponibles: 


|   |   |
| --- | --- |
| *sysAllUsers* | La asociación se tendrá en cuenta para todos los usuarios del ordenador actual.. <br><br>Observación: Debe tener los derechos de "Administrador" para ejecutar esta operación. |
| *sysCurrentUser*<br>(Valor predeterminado) | La asociación se tendrá en cuenta sólo para el usuario actual.. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se acepta un solo comando para cada extensión. Si varios comandos son associated con la misma extensión, sólo se tendrá en cuenta el último comando.

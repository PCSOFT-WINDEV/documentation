


## &lt;Hoja de cálculo&gt;.SavePositionAndSelection ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra: 

- la posición de la barra de desplazamiento en una control Hoja de cálculo para la hoja de trabajo actual,

- la hoja de trabajo actual en el control Hoja de cálculo,

- los elementos seleccionados en el control Hoja de cálculo para la hoja de trabajo actual. 




Estos elementos pueden ser restaurados por [&lt;Hoja de cálculo&gt;.RestorePositionAndSelection](../WDLang1/1000023733.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Let MySave = PSHEET_Spreadsheet1.SavePositionAndSelection() 

// Displays the B1 cell in bold
PSHEET_Spreadsheet.SelectPlus("B1")
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)

// Restores the position/selection 
PSHEET_Spreadsheet1.RestorePositionAndSelection(MySave)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.SavePositionAndSelection()`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [&lt;Hoja de cálculo&gt;.RestorePositionAndSelection](../WDLang1/1000023733.md). 

- Cadena vacía ("") si el control Hoja de cálculo está vacío.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

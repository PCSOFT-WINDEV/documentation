
## SysThumbnailDeleteAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los botones associated con la miniatura de la aplicación de la barra de tareas. A continuación, tendrá la posibilidad de añadir nuevos botones con [Botón SysThumbnailAddButton](../WDLang1/1000018871.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all the buttons linked to the application thumbnail
SysThumbnailDeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysThumbnailDeleteAll()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminaron todos los botones asociados a la miniatura,

- <u><u><u><u>False</u></u></u></u> en caso contrario.


## Emulación de constantes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **eabLeftButton** | Simula una pulsación en el botón izquierdo de Action Bar. | [EmulateActionBar](../WDLang1/1000021199.md) |
| **eabOption** | Simula una pulsación sobre una de las opciones de la Action Bar. | [EmulateActionBar](../WDLang1/1000021199.md) |
| **eabView** | Simula una pulsación sobre una de las vistas que se encuentran en la Action Bar. | [EmulateActionBar](../WDLang1/1000021199.md) |
| **eiAlt** | Mantiene presionada la tecla Alt. <br><br> | [EmulateMouse](../WDLang1/1000009011.md), [EmulateInput](../WDLang1/1000009012.md) |
| **eiControl** | Mantiene apretada la tecla Ctrl. <br><br> | [EmulateMouse](../WDLang1/1000009011.md), [EmulateInput](../WDLang1/1000009012.md) |
| **eiShift** | Mantenga apretada la tecla Mayúsculas. <br><br> | [EmulateMouse](../WDLang1/1000009011.md), [EmulateInput](../WDLang1/1000009012.md) |
| **emLeftButtonDown** | Presiona el botón izquierdo. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emLeftButtonUp** | Suelta el botón izquierdo. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emLeftClick** | Realiza un clic izquierdo. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emLeftDoubleClick** | Realiza un doble clic izquierdo. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emRightButtonDown** | Presiona el botón derecho. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emRightButtonUp** | Suelta el botón derecho. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emRightClick** | Realiza un clic derecho. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **emRightDoubleClick** | Realiza un doble clic con el botón derecho del ratón. <br><br> | [EmulateMouse](../WDLang1/1000009011.md) |
| **esBack** | Simula una pulsación sobre el botón Atrás del simulador. | [EmulateSimulador](../WDLang1/1000021151.md) |
| **esClose** | Cierra el simulador. | [EmulateSimulador](../WDLang1/1000021151.md) |
| **esDefault** | Velocidad de ejecución para las funciones de emulación: La velocidad de Normal. | [EmulateSpeed](../WDLang1/1000022075.md) |
| **esMenu** | Simula una pulsación sobre el botón Menú del simulador. | [EmulateSimulador](../WDLang1/1000021151.md) |
| **esSlow** | Velocidad de ejecución para las funciones de emulación: velocidad lenta para ver las acciones realizadas. | [EmulateSpeed](../WDLang1/1000022075.md) |
| **esTurnScreen** | Activa la pantalla del simulador: cambia del modo horizontal al modo vertical (o viceversa). | [EmulateSimulador](../WDLang1/1000021151.md) |

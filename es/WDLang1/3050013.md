
## CoTan( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la cotangente de un ángulo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the cotangent of 20
CoTangentRes = CoTan(20)
// Returns 2.74747741945
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CoTan(<Angle>)`
---

**`<Result>: (Real)`**

Cotangente del ángulo especificado.

**`<Angle>: (Entero o real)`**

Ángulo (en grados) de uso. Se produce un error de WLanguage si este parámetro es set a 0°, 180° o -180°..

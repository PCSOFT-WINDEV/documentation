


## Funciones de gestión de la telemetría
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Para la gestión de la telemetría se utilizan las siguientes funciones: 



|   |   |
| --- | --- |
| [TelemetryAddMeasure](../WDLang1/1000022168.md) | Añade una medida para un sensor de valor o de tiempo. |
| [TelemetryAddPassage](../WDLang1/1000022174.md) | Señala la ejecución del código para un sensor de paso. |
| [TelemetryConfigure](../WDLang1/1000021913.md) | Permite conocer y modificar los parámetros de telemetría. |
| [TelemetryDisable](../WDLang1/1000021912.md) | Desactiva la telemetría. Los datos de telemetría ya no se envían al servidor. |
| [TelemetryEnable](../WDLang1/1000021900.md) | Activa la telemetría manualmente. |
| [TelemetryIdentifyUser](../WDLang1/1000022160.md) | Identifica al usuario actual para el que se guardan los datos de telemetría. |
| [TelemetryIsEnabled](../WDLang1/1000021911.md) | Permite saber si la telemetría está habilitada. |
| [TelemetrySendData](../WDLang1/1000024736.md) | Fuerza el envío inmediato de datos de la telemetría. |

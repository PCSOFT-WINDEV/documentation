
## TraceDeleteAll( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Limpia el contenido de la ventana de trace abierta en paralelo con la ventana actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Clears the content of trace window
TraceDeleteAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TraceDeleteAll()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Archivo de rastreo
<a name="archivo_rastreo_ELTPARAGRAPHE000029"></a>

La traza se puede realizar en un fichero en los siguientes casos: 

- en Linux,

- a través de [TraceStart](../WDLang1/3013049.md) asociada a la constante *trFile*. 


En este caso, la función **TraceDeleteAll** elimina el archivo de traza.




## Funciones de hoja de cálculo (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la control Hoja de cálculo (sintaxis del prefijo): 



|   |   |
| --- | --- |
| [&lt;Hoja de cálculo&gt;.AddData](../WDLang1/1000023711.md) | Añade un dato a un control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Hoja de cálculo&gt;.AddFormula](../WDLang1/1000023713.md) | Añade una fórmula de cálculo a un control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Hoja de cálculo&gt;.AddImage](../WDLang1/1000023714.md) | Añade una Image a la hoja de trabajo actual de una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.AddName](../WDLang1/1000023715.md) | Especifica un nombre para identificar una celda que se encuentra en una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.AddWorksheet](../WDLang1/1000023712.md) | Añade una hoja de trabajo a un control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Hoja de cálculo&gt;.AllowProcedure](../WDLang1/1000023716.md) | Permite el uso de una función WLanguage/Procedure en una fórmula de una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.BorderSelection](../WDLang1/1000023717.md) | Modifica la Border de las celdas seleccionadas en una control Hoja de cálculo encontrada en una ventana. |
| [&lt;Hoja de cálculo&gt;.close](../WDLang1/1000023722.md) | Cierra la XLSX Document que fue abierta por [SpreadsheetLoad](../WDLang1/1000023719.md). |
| [&lt;Hoja de cálculo&gt;.ControlFormula](../WDLang1/1000023718.md) | Permite utilizar un control para crear una barra de fórmula personalizada para el control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.DeleteAll](../WDLang1/1000023743.md) | Borra todos los datos de un control Hoja de cálculo encontrado en una ventana (celdas, fórmulas, tamaño y disposiciones de cabecera).. |
| [&lt;Hoja de cálculo&gt;.DeleteRow](../WDLang1/1000023742.md) | Borra un fila de un control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.DeleteWorksheet](../WDLang1/1000023741.md) | Elimina una hoja de trabajo de un control Hoja de cálculo encontrado en una ventana. |
| [&lt;Hoja de cálculo&gt;.Freeze](../WDLang1/1000023723.md) | Se utiliza para congelar la visualización de una fila o columna en un control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.GetFormula](../WDLang1/1000023730.md) | Recupera la fórmula de cálculo de una celda encontrada en un control Hoja de cálculo de una ventana. |
| [&lt;Hoja de cálculo&gt;.GetMerge](../WDLang1/1000023731.md) | Se utiliza para averiguar si una célula se fusiona con otras células y a get el nombre de células fusionadas. |
| [&lt;Hoja de cálculo&gt;.GetSelection](../WDLang1/1000023732.md) | Recupera la lista de celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana. |
| [&lt;Hoja de cálculo&gt;.InsertRow](../WDLang1/1000023727.md) | Inserta una fila vacía en una control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Hoja de cálculo&gt;.load](../WDLang1/1000023719.md) | Carga un archivo en formato XLSX en un control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Hoja de cálculo&gt;.mergeCell](../WDLang1/1000026322.md) | Fusiona células especificadas o seleccionadas en una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.MoveWorksheet](../WDLang1/1410087262.md) | Mueve una hoja de trabajo dentro de una control Hoja de cálculo en una ventana. |
| [&lt;Hoja de cálculo&gt;.posición](../WDLang1/1000023728.md) | Muestra la celda especificada en una control Hoja de cálculo que se encuentra en una ventana.: <br><br>- o el área de Visible de control Hoja de cálculo se mueve para mostrar la célula. <br><br>- o la celda se muestra en la esquina superior izquierda del área Visible de control Hoja de cálculo.<br><br><br> |
| [&lt;Hoja de cálculo&gt;.Reemplazar](../WDLang1/1000023898.md) | Encuentra y reemplaza un valor en un control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.RestorePositionAndSelection](../WDLang1/1000023733.md) | restaura: <br><br>- la posición de la barra de desplazamiento en un control Hoja de cálculo, <br><br>- la hoja de trabajo actual, <br><br>- los elementos seleccionados en la hoja de trabajo actual de control Hoja de cálculo. <br><br><br>Estos elementos han sido guardados por [Hoja de cálculoGuardarPosiciónYSelección](../WDLang1/1000023735.md). |
| [&lt;Hoja de cálculo&gt;.Save](../WDLang1/1000023734.md) | Guarda el contenido de una control Hoja de cálculo en un archivo en formato XLSX. |
| [&lt;Hoja de cálculo&gt;.SavePositionAndSelection](../WDLang1/1000023735.md) | Ahorra: <br><br>- la posición de la barra de desplazamiento en un control Hoja de cálculo para la hoja de trabajo actual, <br><br>- la hoja de trabajo actual en el control Hoja de cálculo,<br><br>- los elementos seleccionados en el control Hoja de cálculo para la hoja de trabajo actual.<br><br><br> |
| [&lt;Hoja de cálculo&gt;.SelectMinus](../WDLang1/1000023737.md) | Desmarca una o más celdas de un control Hoja de cálculo encontrado en una ventana. |
| [&lt;Hoja de cálculo&gt;.SelectPlus](../WDLang1/1000023738.md) | Selecciona una o más celdas de un control Hoja de cálculo encontrado en una ventana. |
| [&lt;Hoja de cálculo&gt;.SelectWorksheet](../WDLang1/1000023736.md) | Selecciona una hoja de trabajo (identificada por su nombre) en una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.SortSelection](../WDLang1/1000023744.md) | Ordena los valores seleccionados en una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.StyleSelection](../WDLang1/1000023739.md) | Devuelve o modifica el estilo de las celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana. |
| [&lt;Hoja de cálculo&gt;.TypeAndMaskSelection](../WDLang1/1000023745.md) | Devuelve o modifica el tipo y la máscara de las celdas seleccionadas en una control Hoja de cálculo. |
| [&lt;Hoja de cálculo&gt;.UnmergeCell](../WDLang1/1410089570.md) | Anula la combinación de la celda especificada o seleccionada en un control Hoja de cálculo. |
| [&lt;InsertWorksheet](../WDLang1/1000023726.md) | Inserta una hoja de trabajo en un control Hoja de cálculo que se encuentra en una ventana. |
| [&lt;Recalcular todo](../WDLang1/1000023729.md) | Recalcula todas las fórmulas de un control Hoja de cálculo encontrado en una ventana (incluyendo las fórmulas que utilizan funciones/procedimientos de WLanguage). |

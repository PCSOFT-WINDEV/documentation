


## &lt;Variable picLayer&gt;.HorizontalSymmetry (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Voltea la imagen que se encuentra en una variable de tipo [picLayer](../WDLang1/1000024604.md) con respecto a un eje horizontal (simetría con respecto a un eje horizontal).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dSymetrieHorizontale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Symmetry of layer with respect to a horizontal axis
MyResult is boolean
MyResult = MyLayer.HorizontalSymmetry()
IF MyResult = False THEN Error("The operation failed")
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = dHorizontalSymmetry(<Image>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se volteó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

La simetría se realiza con respecto al eje horizontal central de la imagen. La imagen se modifica directamente.

## Observaciones


### Fin del dibujo
<a name="fin_del_dibujo_ELTPARAGRAPHE000346"></a>

Si se llama a las funciones [&lt;Variable picLayer&gt;.EndDrawing](../WDLang1/1000024473.md) y **&lt;Variable picLayer&gt;.HorizontalSymmetry** en el mismo proceso, no se mostrarán los cambios de la imagen.

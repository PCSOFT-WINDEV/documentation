


## Funciones de gestión de tareas paralelas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar tareas paralelas: 



|   |   |
| --- | --- |
| [ParallelTaskCancel](../WDLang1/1000021278.md) | Finaliza inmediatamente la ejecución de la tarea actual.: la tarea se marca como *ptsCanceled*. |
| [ParallelTaskConfigure](../WDLang1/1000021284.md) | Modifica los parámetros de tareas paralelas. |
| [ParallelTaskExecute](../WDLang1/1000021211.md) | Activa la ejecución de una tarea paralela. |
| [ParallelTaskExecuteAfter](../WDLang1/1000021214.md) | Indica una tarea paralela de continuación que se ejecutará cuando se termine una de las tareas paralelas especificadas. |
| [ParallelTaskExecuteAfterAll](../WDLang1/1000021217.md) | Indica una tarea paralela de continuación que se ejecutará cuando todas las tareas de un array de tareas paralelas hayan finalizado. |
| [ParallelTaskExecuteAfterOne](../WDLang1/1000021216.md) | Indica una tarea paralela de continuación que se ejecutará una vez finalizada la primera tarea de un array de tareas paralelas. |
| [ParallelTaskRequestCancellation](../WDLang1/1000021213.md) | Solicita la cancelación de una tarea paralela. |
| [ParallelTaskWait](../WDLang1/1000021212.md) | Espera la ejecución de una tarea paralela. |
| [ParallelTaskWaitAll](../WDLang1/1000021297.md) | Espera al final de la ejecución de todas las tareas paralelas que se encuentran en una array. |

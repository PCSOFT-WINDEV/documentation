


## CertificateList( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de certificados disponibles en el equipo. 



**Observación**: [Certificado HTTPList](../WDLang3/1000018890.md) sólo muestra los certificados con función de "Autenticación de clientes".. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the list of certificates available on the computer
// (Windows personal certificate store)
arrCertificate is array of Certificates
arrCertificate = CertificateList()

// Browse the certificates
ACertificate is Certificate
i is int
i = 1
FOR EACH ACertificate OF arrCertificate
	// Checks whether the certificate is valid for signing
	IF ACertificate.ValidForSignature = True THEN
		// Adds the certificate into a Combo Box control
		ListAdd(COMBO_CERTIFICATECHOICE, ACertificate.Name + ...
				 "(" + ACertificate.Issuer + ")" + gStoredValue(i))
		i++
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CertificateList([<Store location> [, <Store name>]])`
---

**`<Result>: (Array de las variables de la Certificate)`**

Nombre de la variable array de [Certificate](../WDLang1/1000019287.md) que contiene la lista de certificados disponibles en el equipo.

**`<Store location>: (Constante opcional de tipo Integer)`**

Ubicación de la tienda Certificate donde se deben recuperar los Certificate: 



|   |   |
| --- | --- |
| *certLocalMachineStore* | Ubicación correspondiente: <br>HKEY_LOCAL_MACHINE/Software/Microsoft/<br>SistemaCertificados |
| *certLocalMachineStoreEnterprise* | Ubicación correspondiente: <br>Cargado desde los dominios de la empresa. |
| *certCurrentServiceStore* | Ubicación correspondiente: <br>HKEY_LOCAL_MACHINE/Software/Microsoft/Cryptography/<br>Servicios/NombreDeServicio/CertificadosDelSistema |
| *certServiceStore* | Ubicación correspondiente: <br>HKEY_LOCAL_MACHINE/Software/Microsoft/Cryptography/<br>Servicios/NombreDeServicio/CertificadosDelSistema |
| *certLocalMachineStorePolicy* | Ubicación correspondiente: HKEY_LOCAL_MACHINE/Software/Policy/Microsoft/<br>SistemaCertificados |
| *certCurrentUserStorePolicy* | Ubicación correspondiente: HKEY_CURRENT_USER/Software/Policy/Microsoft/ |
| *certCurrentUserStore* <br>(Valor predeterminado) | Ubicación correspondiente: HKEY_CURRENT_USER/Software/Microsoft/<br>SistemaCertificados |
| *certUsersStore* | Ubicación correspondiente: HKEY_USERS/NombreDeUsuario/Software/Microsoft/<br>SistemaCertificados |



**`<Store name>: (Cadena de caracteres opcional o constante opcional)`**

Nombre del almacén en el que se deben recuperar los certificados. Este parámetro puede corresponder a: 

- una cadena de caracteres, 

- una de las siguientes constantes: 
	


|   |   |
| --- | --- |
| *certAuthority* | Almacén de autoridades de certificación. |
| *certTrust* | Tienda de confianza. |
| *certPersonal* <br>(Valor predeterminado) | Tienda personal. |
| *certRoot* | Tienda de raíces |


## fMerge( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fusiona varios archivos en uno solo. Los archivos a utilizar han sido creados durante una operación de división anterior por [fSplit](../WDLang1/1000021019.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Splits the Video.avi file
fSplit("Video.avi", 100*1024*1024, "SplitVideo")
IF ErrorOccurred THEN
	// Displays the error message
	Error(ErrorInfo())
	RETURN
END

// The "SplitVideo.001" and "SplitVideo.002" files have been created
// Delete the source file ("Video.avi")
fDelete("Video.avi")

// Uses the "SplitVideo.001" and "SplitVideo.002" files 
// to re-create the "Video.avi" file 
fMerge("SplitVideo", "Video.avi")
IF ErrorOccurred THEN
	// Displays the error message
	Error(ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fMerge(<File name> [, <Output file> [, <Option> [, <Progress Bar control>]]])`
---

**`<Result>: (Integro)`**



- Número de archivos fusionados,

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
	Observación: en caso de error, los archivos de salida deben encontrarse en el disco. 




**`<File name>: (Cadena de caracteres)`**

Nombre completo del archivo a fusionar (sin la extensión ".001"). 

**`<Output file>: (Cadena de caracteres opcional)`**

ruta completa del archivo a crear. 
En Default, &lt;Fichero de salida&gt; corresponde a &lt;Nombre del fichero&gt;.

**`<Option>: (Constante opcional de tipo Integer)`**

Por Default: 

- los archivos fuente se mantienen durante la operación de fusión. 

- si el fichero de salida ya existe, la función devuelve un error.




Puede especificar las siguientes opciones de fusión: 


|   |   |
| --- | --- |
| *fDeleteDestination* | Si el archivo de destino ya existe, se eliminará y se volverá a crear. |
| *fDeleteSource* | Los archivos fuente se borrarán a medida que avance la operación de fusión.<br><br>Atención: Si se produce un error, los archivos fuente se pierden y no se pueden recuperar. |



**`<Progress Bar control>: (Nombre de control opcional)`**

Nombre de la control Barra de progreso utilizada para seguir el progreso de la operación de fusión para las diferentes partes del archivo.

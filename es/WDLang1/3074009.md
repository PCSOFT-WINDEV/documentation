


## Funciones de gestión de controles Tabla
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones sirven para gestionar controles Tabla:



|   |   |
| --- | --- |

| [TablaAñadirColumna](../WDLang1/1000017168.md) | Añade elementos a la derecha de una control Tabla horizontal. |

| [TablaFórmulaAñadir](../WDLang1/1000022361.md) | Añade una fila calculada a una control Tabla proporcionando procedimientos de cálculo personalizados. |

| [TablaRefreshVisible](../WDLang1/1000021196.md) | Modifica o devuelve el estado de visibilidad de la barra actualización en una Tabla o control Tabla TreeView. |

| [TablaToWord](../WDLang1/3074036.md) | Crea un archivo Word (.RTF) a partir de los datos encontrados en una Tabla o control Tabla TreeView. |

| [TableCollapseExpand](../WDLang1/1000024474.md) | Contrae o expande un elemento de un control Tabla o Tabla TreeView. |
| [TableColumnIndex](../WDLang1/3074016.md) | Devuelve el índice de una columna en una Tabla o Treeview control Tabla. |

| [TableEndFilling](../WDLang1/1000022369.md) | Finaliza la Process para poblar una Tabla o control Tabla TreeView programáticamente. |
| [TableEnumBreak](../WDLang1/1000022418.md) | Enumera las rupturas encontradas en un control «Table». |

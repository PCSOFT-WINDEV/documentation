
## Appointment (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Appointment** permite definir las características avanzadas de una cita: hora, lugar, ... 

Este tipo de variable se puede utilizar:  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una control Agenda o en una control Planificador. El tipo **Appointment** se manipulará con las funciones WLanguage de gestión de controles Agenda o Planificador.




Las características de esta Appointment pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación de Windows 10 Universal, este tipo de Variable sólo puede usarse para manipular las citas de los controles del Planificador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Build an Appointment variable
MyAppointment is Appointment

// The appointment starts now and it lasts two hours
MyAppointment.StartDate = DateSys() + TimeSys()
MyAppointment.EndDate = MyAppointment.StartDate
MyAppointment.EndDate.Time = MyAppointment.EndDate.Time + 2

MyAppointment.Title = "Appointment with the sales director"
MyAppointment.Importance = 1
MyAppointment.Guest = "Tommy, Vince, Sandra"
MyAppointment.Content = "Establishing the new price list"
MyAppointment.Note = "Remember to bring the blue folder."
MyAppointment.Tooltip = MyAppointment.Title + CR + MyAppointment.Note

// Add the appointment into the organizer
OrganizerAddAppointment(ORG_MyOrganizer, MyAppointment)
```


<a name="Example2"></a>

```wl
// Adds an appointment by using the Appointment type and with repetition
// Every first Monday of the month 
// a Monday found in days from 1 to 7 of the month = 1st of month.

x is Appointment
x.StartDate = Today() + "1200"
x.EndDate = Today() + "1400"
x.WithRepetition = True
y is Repetition
y.Type = schEveryDay
y.DayOfWeek = 1 
y.DayOfMonth = "1-7" 
y.DayOfMonthOrDayOfWeek = False
x.Repetition = y

OrganizerAddAppointment(ORG_Organizer, x)
```

## Observaciones

El tipo Appointment permite gestionar los controles Organizador y Programador y gestionar los contactos a través de las funciones Appointment..

### Propiedades específicas para la gestión de los controles del organizador y del programador
<a name="propiedades_especificas_para_gestion_los_controles_del_organizador_del_programador_ELTPARAGRAPHE000090"></a>

Las siguientes propiedades pueden ser utilizadas para manipular y Appointment:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Autor | Cadena de caracteres | Nombre del autor de Appointment. Si no se especifica esta propiedad, la Appointment no tiene autor.<br><br> |
| BackgroundColor | Integro | El color de fondo utilizado para mostrar el Appointment en el control Agenda o en el control Planificador.<br>Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)), <br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)), <br><br>- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).<br><br><br>Si no se especifica esta Property, se calculará automáticamente una color. Los nombramientos que se encuentran en la misma categoría tendrán la misma Color.<br><br> |
| Categoría | Cadena de caracteres | Categoría de Appointment. Si no se especifica esta propiedad, la Appointment es associated sin categoría. |
| Contenido | Cadena de caracteres | Detalle de la description de la Appointment. <br><br>Para un control Planificador (o para un control Agenda), el contenido se muestra en el control.<br><br>Si no se especifica esta propiedad, la cita no tiene description. |
| EndDate | Cadena de caracteres o variable de tipo DateTime | Fecha y hora de finalización de la Appointment.  Se debe asignar una fecha válida a este Property antes de usar el Variable.<br><br><br>Esta Property debe ser necesariamente especificada. |
| ID | Cadena de caracteres | Identificador asociado con el Appointment. Permite almacenar el identificador de un elemento que debe asociarse a la Appointment para utilizarlo más tarde en la programación. <br><br>Por ejemplo, este identificador puede corresponder al identificador de un Record en una base de datos. |
| Image | Cadena de caracteres | La imagen asociada con la Appointment. Esta imagen se muestra en el control Agenda o en el control Planificador (además del logo que representa un Appointment importante si es necesario). <br><br>Esta propiedad puede corresponder a: <br><br>- el ruta a un Image accesible desde el ordenador actual,<br><br>- un archivo de Image que se encuentra en la biblioteca de aplicaciones,<br><br>- un control Imagen que contiene un Image,<br><br>- un dibujo realizado en una control Imagen con las funciones de dibujo y guardado en la memoria.<br><br><br>Si no se especifica esta propiedad, la Appointment no tiene associated Image.<br><br> |
| Importancia | Integro | Importancia de Appointment. <br><br>Si esta Property es mayor o igual a 1, se mostrará un icono (![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=agRendezVous_IconeImportant.gif)) en el Organizador o control Planificador para indicar un Event importante. <br><br>Si no se especifica este Property, su valor es set a 0.<br><br> |
| Nota | Cadena de caracteres | Nota asociada a la Appointment. <br><br>Si no se especifica esta propiedad, no se asocia ninguna nota a la Appointment.<br><br> |
| Pedido | Integro | Orden de las citas (para las citas que se superponen). <br><br> |
| Recurso | Cadena de caracteres | Recurso asociado con el Appointment. Esta propiedad se tiene en cuenta sólo cuando el Appointment es utilizado por un control Planificador. |
| Repetition | Variable de tipo [Repetition](../WDLang1/1000019262.md) | Parámetros avanzados de Repetition. Esta Property se tiene en cuenta sólo si la **con la repetición** Property es set a <u><u><u><u>True</u></u></u></u>.<br><br> |
| StartDate | Cadena de caracteres o variable de tipo DateTime | Fecha y hora de inicio de la Appointment. Se debe asignar una fecha válida a este Property antes de usar el Variable.<br><br>Esta Property debe ser necesariamente especificada. |
| Título | Cadena de caracteres | Título de Appointment. El título se muestra en el control Agenda o en el control Planificador. <br><br>Si no se especifica esta propiedad, la Appointment no tiene título. |
| ToolTip | Cadena de caracteres | Tooltip que se muestra al deslizar el cursor sobre una cita. Por defecto, este valor incluye la hora de inicio y la hora de fin de Appointment, así como el título de Appointment.<br><br> |
| Ubicación | Cadena de caracteres | Ubicación de la Appointment. <br><br>Si no se especifica esta propiedad, la Appointment no tiene ubicación.<br><br> |
| WholeDay | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la cita ocupa todo el día. <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario (valor predeterminado). <br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Atención**: Para que la cita para todo el día se muestre correctamente en el control, "Mostrar y permitir las citas para todo el día" debe estar marcado en la pestaña "Detalles" del control description. |
| WithRepetition | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si debe repetirse la cita,<br><br>- <u><u><u><u>False</u></u></u></u> si la cita tiene lugar una sola vez. <br><br><br>Si la cita se repite, la propiedad **repetition** permite especificar la frecuencia del repetition. <br><br>Por defecto, el valor de esta propiedad es <u><u><u><u>False</u></u></u></u>.<br><br> |

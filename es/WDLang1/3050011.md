
## BinaryNOT( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el resultado de un NOT lógico (bit a bit) realizado sobre un valor.

**Recordatorio**: La operación lógica NOT es equivalente a la de "Complemento a uno".. La negación lógica se aplica a TODOS los bits del valor.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the result of a logical NOT
ResBinaryNOT = BinaryNOT(6)  // Returns -7
ResBinaryNOT = BinaryNOT(2)  // Returns -3
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BinaryNOT(<Value>)`
---

**`<Result>: (Integro)`**

Resultado de la lógica NOT.

**`<Value>: (Integro)`**

Valor a probar.

## Observaciones




### Cálculo de bit a bit
<a name="calculo_bit_bit_ELTPARAGRAPHE000075"></a>

**BinaryNOT** realiza el cálculo a nivel de bits de &lt;Valor&gt; basado en el siguiente tabla:


|   |   |   |
| --- | --- | --- |
| **Valor** | 0 | 1 |
| **Result** | 1 | 0 |


<a name="NOTE0_2"></a>




### Operador binario
<a name="operador_binario_ELTPARAGRAPHE000098"></a>

La función **BinaryNOT** es equivalente al operador binario **~**. Para obtener más información, consulte [Operadores binarios](../Motscles/1512009.md).

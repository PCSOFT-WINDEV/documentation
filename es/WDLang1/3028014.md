
## DDEWarmLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un enlace cálido con un dato.

Crear un enlace "caliente" con un dato significa que el programa será notificado por el servidor (programa conectado) cada vez que se modifiquen los datos de relacionado pero sin recibir su valor.. Por lo tanto, debe bifurcarse a un procedimiento en respuesta al DDE Event utilizado para modificar un dato relacionado ([DDEEvent](../WDLang1/3028009.md) con el *DDEtoModify* Event).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyConnection = DDEConnect("Excel", "Worksheet1")
DDEEvent("ModifProc", DDEtoModify)
DDEWarmLink(MyConnection, "L1C1", 1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEWarmLink(<Connection identifier> , <Object> [, <Timeout>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el enlace,

- <u><u><u><u>False</u></u></u></u> si se ha producido un error ([DDEError](../WDLang1/3028006.md) devuelve los detalles del error).




**`<Connection identifier>: (Integro)`**

Identificador de conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md).

**`<Object>: (Cadena de caracteres)`**

Objeto de conexión.

**`<Timeout>: (Entero o duración opcional)`**

Duración máxima de espera (en segundos). Si el comando no se ejecutaba:

- Si se especifica &lt;Timeout&gt;, hay un delay de &lt;Timeout&gt; segundos antes de salir de **DDEWarmLink**.

- Si no se especifica el tiempo de espera, hay una espera interminable.... 


Observación: Este parámetro puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en claro ('1s' por ejemplo). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Un enlace DDE no debe estar habilitado durante una impresión: debe ser desactivado por [DDEUnLink](../WDLang1/3028005.md).

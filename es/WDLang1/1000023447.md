
## SpreadsheetDeleteName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra uno de los nombres usados para identificar una celda encontrada en una control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Replace a name
SpreadsheetDeleteName(PSHEET_MySpreadsheet, "PriceBT")
SpreadsheetAddName(PSHEET_MySpreadsheet, "PriceBT", "$A$1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetDeleteName(<Spreadsheet control> , <Cell name>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Cell name>: (Cadena de caracteres)`**

Nombre de la celda a borrar. Se devuelve un error no fatal si no existe este nombre..

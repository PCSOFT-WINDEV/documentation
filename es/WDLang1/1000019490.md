


## InFactoryMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el código es ejecutado por un plan de acción de la fábrica de software. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run in an action plan 
IF InFactoryMode() = True THEN
	// Code specific to the software factory
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InFactoryMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código se ejecuta desde un plan de acción de la fábrica de software,

- <u><u><u><u>False</u></u></u></u> en caso contrario.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000204"></a>

La función **InFactoryMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InFactoryMode** pueden compartirse entre varios productos sin generar errores de compilación.


## WinEdRestore (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela todas las modificaciones realizadas por el usuario final (Modo "Modificación de UI por el usuario final" activado). La ventana se restablece a su estado inicial. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the [Cancel all the modiifications] button
WinEdRestore(WIN_Form)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinEdRestore(<Window>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función no tiene ningún efecto: 

- Si la ventana especificada no es editable. 

- Si la ventana especificada no fue modificada por el usuario final.




## &lt;Tabla dinámica&gt;.HeaderValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor mostrado en una cabecera de control Tabla dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
i is int = 1
s is string = PVT_Stat.HeaderValue("DPVT_Family", i)
WHILE s <> "" 
	Trace(s)
	s = PVT_Stat.HeaderValue("DPVT_Family", i)
	i++
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Pivot Table control>.HeaderValue(<Header> , <Subscript>)`
---

**`<Result>: (Cadena de caracteres)`**



- Valor de la cabecera especificada.

- Cadena vacía ("") si la índice especificada no es válida.




**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar

**`<Header>: (Cadena de caracteres)`**

Nombre de una cabecera de la control Tabla dinámica.

**`<Subscript>: (Integro)`**

Subíndice del valor a recuperar.

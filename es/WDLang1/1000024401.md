


## &lt;Procesamiento de palabras&gt;.ToImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta una página de un control Procesador de texto en formato Image.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is Image = WP_MyDocument.ToImage(1)
dSaveImageJPEG(d, "test")
ShellExecute("test.jpg")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.ToImage(<Page>)`
---

**`<Result>: (Imagen Variable)`**

Nombre de la [Image](../WDLang1/1000019650.md) Variable que corresponde a la página en el tratamiento de textos Document.

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<Page>: (Integro)`**

Número de la página a exportar en formato Image.




## &lt;Variable WDPic&gt;.DrawTextRTF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja texto RTF en una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"

// Draw RTF text
MyWDPicImage.DrawTextRTF(10, 20, MyTextRTF)

IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<WDPic image>.DrawTextRTF(<X-coordinate> , <Y-coordinate> , <RTF text> [, <Width> [, <Height>]])`
---

**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.  

**`<X-coordinate>: (Entero)`**

Coordenada X del texto RTF a dibujar (en pixeles).

**`<Y-coordinate>: (Entero)`**

Coordenada Y del texto RTF a dibujar (en pixeles).

**`<RTF text>: (Cadena de caracteres)`**

Texto RTF a dibujar.

**`<Width>: (Entero opcional)`**

Ancho máximo del texto. De forma predeterminada, no hay ningún ancho máximo del texto definido.

**`<Height>: (Entero opcional)`**

Alto máximo del texto. De forma predeterminada, no hay ningún alto máximo del texto definido.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000306"></a>

Las coordenadas se especifican con respecto a la esquina superior izquierda de la imagen (coordenadas (0,0)).

Las coordenadas corresponden a la esquina superior izquierda del rectángulo en el que se dibujará el texto.

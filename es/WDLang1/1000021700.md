


## &lt;Estado (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de un panel de pestañas dinámico (identificado por su alias)..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the selected row from a Table control
nIndex is int = TABLE_Customer.Select()
IF nIndex = -1 THEN
	ToastDisplay("Select a customer in the Table control")
	RETURN
END

// Gets the customer identifier
nCustomerNum is int = TABLE_Customer.CUSTOMERID

// Defines the tab alias
sAlias is string = gaaTab[nCustomerNum]
gaaTab is associative array of strings

// Checks whether the customer form is not already opened
IF sAlias = "" _OR_ TAB_MyTab.Status(sAlias) = paneNotFound THEN
	// Opens the window in modification	
	gaaTab[nCustomerNum] = ...
		TAB_MyTab.Open(StringBuild("Customer form of: %1", ...
			TABLE_Customer.Name[nIndex]), IW_Form, nCustomerNum)
ELSE
	ToastDisplay("Form already opened")
	// Activates the tab
	TAB_MyTab = sAlias
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Tab control>.Status(<Alias>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la pestaña dinámica: 


|   |   |
| --- | --- |
| *paneActive* | El panel de pestañas existe y se adjunta a su pestaña. |
| *paneFloating* | El panel de pestañas existe y está separado. |
| *paneNotFound* | El panel de pestañas no existe o se ha cerrado. |



**`<Tab control>: (Nombre del control)`**

Nombre del control Pestaña a utilizar.

**`<Alias>: (Cadena de caracteres)`**

Alias del panel de pestañas dinámico que se va a utilizar. Este alias puede ser recuperado por: 

- la propiedad [Valor](../Proprietes/2510130.md). 

- la propiedad [Alias](../Proprietes/2510004.md). 

- la función [&lt;Abrir](../WDLang1/1000021704.md). 


Si este parámetro no corresponde a ninguna ficha dinámica, la función no tiene acción y se produce un error no fatal.

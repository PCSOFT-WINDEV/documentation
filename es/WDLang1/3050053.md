
## FinNetCurrentVal( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor actual neto de la inversión. Esta inversión se basa en los flujos financieros de Variable.

**Recordatorio**: El valor actual neto corresponde al valor en la fecha actual de una serie de desembolsos futuros (valores negativos) y cobros (valores positivos).

**Observación**: Desde un punto de vista financiero, una inversión se considerará rentable si su valor actual neto es positivo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003050053&name=finnetcurrentval_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Net current value of investment. The interest rate is set to 8%.
ResNetCurrentVal = FinNetCurrentVal(8, "Matrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinNetCurrentVal(<Rate> [, <Matrix name> [, <Row/Column number> [, <Row/Column>]]])`
---

**`<Result>: (Real o moneda)`**

Valor neto actual de la inversión.

**`<Rate>: (Real)`**

Tasa de interés de la inversión expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set al 5. Esta tasa no puede ser nula.

**`<Matrix name>: (Cadena de caracteres opcional)`**

Nombre de la matriz (definida por [MatCreate](../WDLang1/3050058.md)) que contiene los desembolsos y los cobros originados por la inversión.

**`<Row/Column number>: (Entero opcional)`**

Número de la fila o columna que contiene los datos (1 de Default).

**`<Row/Column>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si la serie de valores corresponde a una columna de tabla,

- <u><u><u><u>False</u></u></u></u> si la serie de valores corresponde a una fila de tabla.

## Observaciones




### Elemento de matriz
<a name="elemento_matriz_ELTPARAGRAPHE000089"></a>

La fila o columna de la tabla analizada con la función **FinNetCurrentVal** debe contener los siguientes elementos:


|   |   |
| --- | --- |
| Número de la fila/columna | Contenido |
| 1 | Importe invertido (desembolsado): negativo |
| 2 | Ingresos del ejercicio 1: abonado o cargado |
| 3 | Ingresos del año 2: abonado o cargado |
| ..... |   |
| N-1 | Ingresos del ejercicio N-2: abonado o cargado |
| N | Ingresos del ejercicio N-1: abonado o cargado |


<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000126"></a>

**Atención**: La función **FinNetCurrentVal** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

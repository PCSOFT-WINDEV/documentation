


## &lt;Image Editor&gt;.Abrir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una imagen en un editor de imágenes control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFile is string 

// Opens the image picker
sFile = fImageSelect("", "", "Select an image...")

IF sFile = "" THEN
	RETURN
END

// Open the image in the Image Editor control
IF IE_MyEditor.Open(sFile) = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Image Editor control>.Open(<Image> [, <Opening mode>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Image Editor control>: (Nombre del control)`**

Nombre del control del Editor de Image usado para mostrar el Image.

**`<Image>: (Cadena de caracteres o variable de tipo Image)`**

Image para mostrar en el editor Image control. Este parámetro puede ser: 

- una variable de tipo [Image](../WDLang1/1000019650.md). 

- el archivo ruta de Image.




**`<Opening mode>: (Constante opcional de tipo Integer)`**

Modo de apertura Image: 



|   |   |
| --- | --- |
| *odOptionHidePane* | La imagen se visualiza en el editor de imágenes en modo lectura/escritura y el panel derecho (que muestra las características de color) no es visualizado por Default.  |
| *odOptionNone*<br>(valor predeterminado) | El Image se muestra en el Editor Image control en modo lectura/escritura. |
| *odOptionReadOnly* | El Image se muestra en modo de sólo lectura en el editor Image.  <br><br>En este modo, sólo se podrá acceder a las opciones de visualización y al OCR. El usuario también podrá imprimir y guardar la Document utilizando la opción "Guardar como"..  |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función es equivalente a las siguientes sintaxis: 

```txt
<Image Editor control> = <Image path>
<Image Editor control> = <Image variable>
```

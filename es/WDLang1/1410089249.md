
## Brazil.CheckCNPJ (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número CNPJ (número de identificación de empresa en Brasil).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyCNPJNumber is string = "43337004000172"
Brazil.CheckCNPJ(MyCNPJNumber)	// Returns True

// Check a control
EDT_CNPJNumber = "ABC"
Brazil.CheckCNPJ(EDT_CNPJNumber)	// Returns False
```



```wl
// Check a variable
MyCNPJNumber is string
MyResult is boolean
MyResult = Input("Enter CNPJ number: ", MyCNPJNumber)
IF Brazil.CheckCNPJ(MyCNPJNumber) = False THEN
	Info("Invalid CNPJ number")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Brazil.CheckCNPJ(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número CNPJ válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número CNPJ a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

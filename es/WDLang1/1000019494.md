
## IntToHexa (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la cadena hexadecimal de un valor numérico.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IntToHexa(31) 	// Returns "0000001F"
IntToHexa(31, 2) 	// Returns "00 00 00 1F"
IntToHexa(-1, 2) 	// Returns "FF FF FF FF"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = IntToHexa(<Value> [, <Word size>])`
---

**`<Result>: (Cadena de caracteres)`**

Resultado de la conversión a formato hexadecimal. 

**`<Value>: (Entero)`**

Número a convertir en cadena.

**Observación**: Los enteros de 1 y 2 bytes se procesan de la misma manera que los enteros de 4 bytes. 

**`<Word size>: (Entero opcional)`**

Tamaño de palabra separado por espacios. Una palabra es un "grupo" de dígitos hexadecimales. Por ejemplo, un byte es una palabra de 2 dígitos hexadecimales.

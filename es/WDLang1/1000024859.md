


## CheckAnyTrue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve <u><u><u>True</u></u></u> si al menos un elemento de un array vale <u><u><u>True</u></u></u>. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrProduct is array of Product
// Return True if at least one product is out of stock to notify the manager
IF CheckAnyTrue(arrProduct, AProduct => AProduct.Stock = 0) THEN
	// alert
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CheckAnyTrue(<Source> [, <Operation>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si al menos un elemento del array especificado vale <u><u><u><u>True</u></u></u></u>,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Source>: (Array)`**

Nombre de la variable de tipo Array a verificar. 

**`<Operation>: (Procedimiento WLanguage)`**

Procedimiento WLanguage a ejecutar: 

- Si no se especifica este parámetro, el parámetro &lt;Source&gt; debe corresponder a un array de booleanos. 

- Si se especifica este parámetro, se llama al procedimiento WLanguage por cada elemento del array y debe devolver un booleano. 


Este parámetro puede corresponder a un procedimiento WLanguage local o global, a un procedimiento interno o a un procedimiento lambda.

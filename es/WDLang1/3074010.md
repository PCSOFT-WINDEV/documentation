


## TableSortedColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de columnas ordenadas que se encuentran en una control Tabla. 

Estas columnas se han clasificado: 

- por el usuario directamente. 

- con la función [TableSort](../WDLang1/3074035.md). 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ListColumns is string
ListColumns = TableSortedColumn(TABLE_Table1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableSortedColumn(<Table control>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de columnas ordenadas con su dirección de clasificación. Esta lista tiene el siguiente formato: 

```txt
<Direction><Column1> + TAB + <Direction><Column2> + TAB + ...
```


- &lt;Dirección&gt; es la dirección de clasificación. Cuando la ordenación es ascendente, este parámetro corresponde a "-". 

- &lt;Columna&gt; es el nombre de la columna. 




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.




## Funciones de gestión del Programador de Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para administrar el programador de Windows:



|   |   |
| --- | --- |
| [SchedulerAddTask](../WDLang1/3089003.md) | Creates a task in the Windows scheduler (if this task does not already exist). |
| [SchedulerAddTrigger](../WDLang1/3089001.md) | Crea una programación para una tarea programada. |
| [SchedulerDeleteTask](../WDLang1/3089014.md) | Elimina una tarea programada. |
| [SchedulerDeleteTrigger](../WDLang1/3089013.md) | Elimina un activador de una tarea programada. |
| [SchedulerModifyTask](../WDLang1/3089008.md) | Modifica los parámetros de una tarea programada. |
| [SchedulerModifyTrigger](../WDLang1/3089007.md) | Modifica los parámetros de una tarea programada, es decir, los parámetros que definirán la ejecución de una tarea. |
| [SchedulerOpenTaskProperties](../WDLang1/3089009.md) | Abre el Property Page de una tarea programada. |
| [SchedulerReset](../WDLang1/3089012.md) | Rellena las estructuras [ScheduledTask](../WDLang1/3089004.md) y/o [TriggerScheduledTask](../WDLang1/3089002.md) con los valores de Default. |
| [SchedulerTaskList](../WDLang1/3089006.md) | Recupera la lista de tareas programadas. |
| [SchedulerTaskProperties](../WDLang1/3089011.md) | Lee las propiedades de una tarea programada y actualiza la [Estructura de tareas programadas](../WDLang1/3089004.md). |
| [SchedulerTaskStatus](../WDLang1/3089005.md) | Obtiene o establece el estado de una tarea creada en el planificador de Windows. |
| [SchedulerTriggerProperties](../WDLang1/3089010.md) | Lee las propiedades de una tarea programada y actualiza la estructura [TriggerScheduledTask](../WDLang1/3089002.md). |




## TreeGiveChild (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve los "hijos" de un elemento que se encuentra en una control TreeView.

Ejemplo de control TreeView: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// First "child" of "Desserts" node in the "TREE_TVRecipe" TreeView
Res = TreeGiveChild(TREE_TVRecipe, "Recipes" +TAB + "Desserts", tvFirst)
// Res is equal to: "Warm Desserts"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeGiveChild(<TreeView control> , <Node path> , <Child to get>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del elemento "hijo" encontrado,

- Vacíe la cadena ("") si no se encuentra ningún elemento "hijo".




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Node path>: (Cadena de caracteres)`**

ruta completa al nodo cuyos elementos "hijo" se solicitan. Este parámetro:

- tiene el siguiente formato: 
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```


- corresponde a la constante *NULL* si los "hijos" de la raíz del control TreeView deben ser devueltos.




**`<Child to get>: (Constante de tipo Integer)`**

Indica el "niño" solicitado: 



|   |   |
| --- | --- |
| *tvFirst* | Primer elemento hijo. |
| *tvNext* | Niño que sigue al último niño que fue recuperado. |

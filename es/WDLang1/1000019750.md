
## GestureNbPointer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de punteros en Contact con la pantalla. 

El valor devuelto depende de cómo se utilice la función:

- Usado en un Event relacionado al movimiento del puntero (click, mover, presionar, soltar, etc.), devuelve el número de punteros (dedo o lápiz) actualmente en Contact con la pantalla

- Usado en otro Event o Process, devuelve el último número conocido de puntos Contact.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code on an Image control
nbContactPoints is int
nbContactPoints = GestureNbPointer()

ToastDisplay("You are currently using " + nbContactPoints + " finger(s) on the screen")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GestureNbPointer()`
---

**`<Result>: (Integro)`**

Número de punteros (dedos o estiletes) que tocan la pantalla.

### Varios
<a name="varios_ELTPARAGRAPHE000094"></a>

- Para conocer la posición vertical (Y) de un puntero, utilice [GesturePosY](../WDLang1/1000019745.md).

- Para conocer la posición horizontal (X) de un puntero, utilice [GesturePosX](../WDLang1/1000019741.md). 

- Para obtener el índice del puntero que desencadenó la ejecución de un Event relacionado a un movimiento en la pantalla, utilice [GestureCurrentPointer](../WDLang1/1000019751.md).

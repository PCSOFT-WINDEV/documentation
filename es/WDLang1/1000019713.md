


## OrgElement (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **OrgElement** se utiliza para definir las características avanzadas de un elemento que se encuentra en un control Organigrama. Las características de este elemento pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

Entonces, el elemento correspondiente puede ser utilizado por el [Funciones WLanguage para la gestión de control Organigrama](../WDLang1/1000019817.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add elements to the Organization Chart control
// Root element
ElementCEO is OrgElement
ElementCEO.Title = "Big Boss"
OrgAdd(ORG_EXECUTIVE, ElementCEO)

// Child elements
Element2 is OrgElement
Element2.Title = "Production"
OrgAddChild(ORG_EXECUTIVE, 1, Element2)

Element3 is OrgElement
Element3.Title = "Sales"
OrgAddChild(ORG_EXECUTIVE, 1, Element3)
```

## Observaciones


### Propiedades específicas de las variables OrgElement
<a name="propiedades_especificas_las_variables_orgelement_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades pueden utilizarse para manipular, un elemento que se encuentra en un control Organigrama:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| BackgroundColor | Entero o constante de tipo Integer | Fondo color utilizado para visualizar el elemento.<br>Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)), <br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)), <br><br>- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).<br><br><br>Si no se especifica esta Property, se calculará automáticamente una color.<br><br> |
| BackgroundImage | Cadena de caracteres | Nombre y ruta completo (o relativo) del archivo Image mostrado como fondo Image del elemento. Se puede utilizar una ruta de tipo UNC. |
| Contenido | Cadena de caracteres | Texto visualizado en el borde del elemento. |
| Contenido de la fuente | [tipo de letra](../Motscles/1514045.md) | Características de la fuente utilizada para mostrar el contenido del elemento. |
| FiDrawing | Cadena de caracteres | Nombre de la ventana interna utilizada para dibujar el elemento. De forma predeterminada, esta propiedad corresponde a una cadena vacía (""): la ventana interna utilizada es la definida en el editor para la control Organigrama. |
| Franja de orilla | [Franja de orilla](../Motscles/15140100.md) | Tipo de borde para el elemento. |
| ID | Todos los tipos | Identificador libre, utilizado para vincular el elemento a un archivo de datos por ejemplo. |
| Image | Cadena de caracteres | Nombre y ruta completo (o relativo) del archivo Image mostrado a la izquierda del título del elemento. Se puede utilizar una ruta de tipo UNC. |
| Nota | Cadena de caracteres | Se utiliza para almacenar información de usuario en tiempo de ejecución. |
| Orientación90 | booleano | Permite gestionar la orientación del elemento: por lo tanto, un niño puede ser mostrado (o no) a la derecha de sus padres: <br><br>- <u><u><u><u>True</u></u></u></u>: El elemento hijo se muestra a la derecha de su padre.<br><br>- <u><u><u><u>False</u></u></u></u> (valor predeterminado): El elemento hijo se muestra debajo de su padre.<br><br><br> |
| Título | Cadena de caracteres | Texto correspondiente al título del elemento. |
| Título de la fuente | [tipo de letra](../Motscles/1514045.md) | Características de la fuente utilizada para mostrar el título del elemento. |
| ToolTip | Cadena de caracteres | Tooltip se muestra al pasar el ratón por encima del elemento. Default no muestra ningún dato de herramienta. |

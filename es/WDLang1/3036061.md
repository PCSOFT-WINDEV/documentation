


## fDetectRemovableStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detecta si se ha añadido o eliminado una unidad de almacenamiento extraíble (CD, llave USB, cámara USB,...).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF fDetectRemovableStorage("MyProcedure") = False THEN
	Error(ErrorInfo())
END
```

```wl
//----------------------------------------
PROCEDURE Myprocedure(Letter, Action)
IF Action = RemovableStorageInserted THEN
	Info(Letter + ": Inserted")
ELSE
	Info(Letter + ": Ejected")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fDetectRemovableStorage(<WLanguage procedure>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la detección. 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<WLanguage procedure>: (Nombre de Procedure)`**



- Nombre de WLanguage Procedure que se ejecutará durante la detección. Esta Procedure se llama siempre que se introduce o se expulsa una unidad de almacenamiento.

- Cadena vacía (""") para desactivar la detección. 




Este procedimiento tiene el siguiente formato:

```txt
PROCEDURE <Procedure name>(<Drive letter>, <Type of action>)
```


&lt;La letra del disco&gt; es una cadena que se utiliza para averiguar la letra del disco associated con la unidad de almacenamiento.

&lt;El tipo de acción&gt; es un número entero que se utiliza para saber si el dispositivo fue insertado o expulsado.. Este parámetro puede tomar uno de los siguientes valores:


|   |   |
| --- | --- |
| *RemovableStorageInserted* | Se ha insertado una unidad de almacén extraíble. |
| *RemovableStorageRemoved* | Se ha expulsado una unidad de almacén extraíble. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Múltiples llamadas a fDetectRemovableStorage
<a name="multiples_llamadas_fdetectremovablestorage_ELTPARAGRAPHE000058"></a>

Si esta función se ejecuta varias veces con diferentes Procedure s pasadas como parámetro, la última Procedure utilizada reemplazará todas las Procedure s previamente llamadas: los primeros procedimientos ya no serán llamados.
<a name="NOTE0_2"></a>


### Procedure local a la ventana
<a name="procedure_local_ventana_ELTPARAGRAPHE000065"></a>

Si la &lt;WLanguage Procedure&gt; es local a una ventana, la detección se detendrá cuando la ventana se cierre. Le recomendamos que use un Procedure global.
<a name="NOTE0_3"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000072"></a>

La función **fDetectRemovableStorage** no debe ejecutarse desde un hilo secundario.
<a name="NOTE0_4"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000081"></a>

**fDetectRemovableStorage** y [USBDetectRemovibleStorage](../WDLang1/3090007.md) son equivalentes.

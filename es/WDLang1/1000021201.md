
## Botón EmulateRadioButton (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula la selección de una opción en un control Botón de opción.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the 5th option of the radio button
EmulateRadioButton(RADIO_RadioButton, 5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulateRadioButton(<Control> , <Option number>)`
---

**`<Control>: (Cadena de caracteres)`**

Nombre de la control Botón de opción.

**`<Option number>: (Integro)`**

Índice de la opción a seleccionar en la control.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función solo se debe utilizar en el código de las pruebas automatizadas.

## Componente:

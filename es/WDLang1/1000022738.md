
## BreakDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una pausa de una control Tabla o de una control Looper. Esta pausa fue añadida de antemano por [BreakAdd](../WDLang1/1000022737.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a break on the date
xBreak is Control
xBreak <- BreakAdd(TABLE_Order.COL_Date, "BrkDate", brkHeader, 50) 

//...

// Deletes the "xBreak" break
BreakDelete(xBreak)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`BreakDelete(<Break>)`
---

**`<Break>: (Control Variable)`**

Nombre de la [Control](../WDLang1/1000020967.md) Variable correspondiente a la interrupción que se desea eliminar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **BreakDelete** elimina el encabezado y el pie de página de ruptura, si existen.

- Para ocultar una pausa definida en el editor, utilice el botón [Visible](../Proprietes/2510138.md) Property.

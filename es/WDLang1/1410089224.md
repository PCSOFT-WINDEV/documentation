
## KanbanInfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve una de las siguientes opciones para una posición determinada (coordenadas de un punto) en un control Kanban:

- el número de la tarjeta que se encuentra en la posición especificada,

- el número de la lista que se encuentra en la posición especificada. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ListIndex is int = KanbanInfoXY(KANBAN_Planning, kbListInfo, MouseXPos, MouseYPos)
Trace(KANBAN_Planning.List[ListIndex].Title)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = KanbanInfoXY(<Kanban control> , <Type of information> , <X> , <Y>)`
---

**`<Result>: (Entero)`**

Información solicitada: 

- Número de la tarjeta en la posición especificada,

- Número de la lista Kanban en la posición especificada,

- -1 si la posición no corresponde a un elemento del control Kanban. 




**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban a utilizar. 
Si este parámetro es una cadena vacía (""), se utilizará el control al que pertenece el evento actual. 

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *kbCardInfo* | Índice de la tarjeta en la posición especificada. |
| *kbListInfo* | Índice de la lista en la posición especificada. |



**`<X>: (Entero)`**

Coordenada X (en pixeles) a analizar. Esta coordenada es relativa al control.

**`<Y>: (Entero)`**

Coordenada Y (en pixeles) a analizar. Esta coordenada es relativa al control.

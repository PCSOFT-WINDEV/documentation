
## TrustedCertificateAdd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una Certificate a la lista de autoridades raíz de confianza utilizadas para validar las cadenas de certificación. 

Atención: Esta adición se hace para la aplicación actual. No se hacen adiciones en las tiendas de Windows Certificate.

Observación: Esta función tiene un impacto en: 

- Solicitudes HTTP que utilizan variables [httpRequest](../WDLang3/1000021158.md),

- Gestión de archivos por FTPS,

- SSL sockets,

- MQTTT vía SSL,

- la verificación de las firmas (2ª sintaxis de [CertificateCheckString](../WDLang1/1000019298.md)).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TrustedCertificateAdd("MyAC.pem")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TrustedCertificateAdd(<Certificate>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la adición,

- <u><u><u><u>False</u></u></u></u> si se produce un error. 




**`<Certificate>: (Cadena de caracteres)`**

Ruta del archivo Certificate a añadir a la lista de autoridades raíz de confianza. 

Observación: Este archivo puede incluirse en la biblioteca de la aplicación (archivo WDL).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Se admiten certificados en los formatos cer, pem, der y p12.

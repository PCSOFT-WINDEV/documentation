


## <Dashboard>.Display (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elementos Actualiza en un control Cuadro de mando.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an order and refresh the Dashboard control.
HAdd(Orders)
DASH_Main.Display()
```


<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Refreshes the widget of recalls
DASH_Dashboard.Display(IW_Widget_Recalls)
ToastDisplay("The IW_Widget_Recalls widget was refreshed", toastShort, vaMiddle, haCenter)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Refrescando la visualización de una peluca identificada por su nombre

`<Dashboard control>.Display([<Widget>])`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Widget>: (Nombre opcional de la ventana interna o Page interna)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la ventana interna a actualizar.




Si no se especifica este parámetro, se actualizan todas las ventanas/páginas internas gestionadas por control Cuadro de mando..


<a name="SYNTAX2"></a>

### Actualizar la visualización de un wigdet identificado por su índice

`<Dashboard control>.Display([<Subscript>])`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Subscript>: (Entero opcional)`**

Subíndice de la Widget para actualizar.

Si no se especifica este parámetro, se actualizan todas las ventanas/páginas internas gestionadas por control Cuadro de mando.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **<Dashboard>.Display** ejecuta el "widget de actualización" Event para el widget especificado o para todos los widgets que se muestran en control Cuadro de mando. 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) "Refresh widget" es una Event opcional de las ventanas internas utilizadas como widgets en una control Cuadro de mando.

- El "widget Actualizar" de Event no puede ejecutarse inmediatamente: se ejecuta cuando el usuario retira control (fin del código que se está ejecutando o llamada a [Multitask](../WDLang1/3015004.md), por ejemplo). 

- Si se llama varias veces a la función **<Dashboard>.Display** antes de devolver el control, los eventos de actualización se ejecutarán una sola vez.

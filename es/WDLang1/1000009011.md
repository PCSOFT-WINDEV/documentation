


## EmulateMouse (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Simula un ratón Event (clic izquierdo, clic derecho, doble clic, ...).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on EDT_TypeLogin edit control
EmulateMouse(EDT_TypeLogin, emLeftClick, 3, 10)

// Type "MARK" in this control
EmulateInput(EDT_TypeLogin, "Mark", eiShift)

// Click the Validate Button control
EmulateMouse(EDT_ValidateButton, emLeftClick, 5, 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EmulateMouse(<Window or control> , <Action to perform> [, <X> , <Y>])`
---

**`<Window or control>: (El nombre de la ventana o el nombre de control)`**

Nombre de la ventana o control en el que se realizará el ratón Event.

**`<Action to perform>: (Constante o combinación de constantes de tipo Integer)`**

Indica al ratón Event para simular:


|   |   |
| --- | --- |
| *eiAlt* | Mantiene presionada la tecla Alt durante la acción.<br>Esta constante se puede combinar con otras constantes.<br><br> |
| *eiControl* | Mantiene presionada la tecla Ctrl durante la acción.<br>Esta constante se puede combinar con otras constantes.<br><br> |
| *eiShift* | Mantiene apretada la tecla Mayúsculas durante la acción.<br>Esta constante se puede combinar con otras constantes.<br><br> |
| *emLeftButtonDown* | Presiona el botón izquierdo.<br><br> |
| *emLeftButtonUp* | Suelta el botón izquierdo.<br><br> |
| *emLeftClick* | Realiza un clic izquierdo.<br><br> |
| *emLeftDoubleClick* | Realiza un doble clic con el botón izquierdo.<br><br> |
| *emRightButtonDown* | Presiona el botón derecho.<br><br> |
| *emRightButtonUp* | Suelta el botón derecho.<br><br> |
| *emRightClick* | Realiza un clic derecho.<br><br> |
| *emRightDoubleClick* | Realiza un doble clic con el botón derecho del ratón.<br><br> |



**`<X>: (Entero opcional)`**

Posición horizontal (o coordenada X) en píxeles donde debe realizarse la acción. Esta posición se da en relación con la esquina superior izquierda de la ventana o control utilizada.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los parámetros &lt;X&gt; y &lt;Y&gt; son opcionales. Si no se especifican estos parámetros, el ratón Event para simular se ejecuta en el centro de la ventana o control.![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro es obligatorio.

**`<Y>: (Entero opcional)`**

Posición vertical (o coordenada Y) en píxeles donde debe realizarse la acción. Esta posición se da en relación con la esquina superior izquierda de la ventana o control utilizada. 
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los parámetros &lt;X&gt; y &lt;Y&gt; son opcionales. Si no se especifican estos parámetros, el ratón Event para simular se ejecuta en el centro de la ventana o control.![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro es obligatorio.

## Observaciones


### Limitaciones de los controles utilizados
<a name="limitaciones_los_controles_utilizados_ELTPARAGRAPHE000156"></a>

La función **EmulateMouse** no es compatible con los siguientes controles:

- Control Imagen.

- Control Barra de progreso.

- Control Static.

- control OLE.

- Control Forma, 

- Control HTML,

- Control Cámara Web.

- Control Conferencia.

- Control Código de barras.

- Control Ventana interna

- Control Gráfico.




**Atención**: **EmulateMouse** no funciona en una sesión TSE no interactiva.

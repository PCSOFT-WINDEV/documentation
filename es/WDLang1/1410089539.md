


## KanbanInsertList (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una lista en un control Kanban. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete list 1
KanbanDeleteList(KANBAN_List, 1)
// Insert 2 other lists instead
KanbanInsertList(KANBAN_List, 1, "To review")
KanbanInsertList(KANBAN_List, 2, "To do")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = KanbanInsertList(<Kanban control> , <Position> , <List name>)`
---

**`<Result>: (Control)`**

Control Lista Kanban. 

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado. 

**`<Position>: (Entero)`**

Posición de inserción. Si este parámetro es menor o igual que 1, la lista se inserta en la posición 1. Si este parámetro es mayor o igual que el número de listas del control Kanban, la lista se inserta después de la última lista. 

**`<List name>: (Cadena de caracteres)`**

Nombre de la nueva lista Kanban.

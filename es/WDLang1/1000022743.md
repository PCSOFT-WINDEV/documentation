


## DocPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa el scroll de control Procesador de texto para que el texto solicitado se muestre en su totalidad.. Tiene la capacidad de definir la posición de este texto en relación con los bordes de control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the seection if it was not visible
DocPosition(WP_MyDoc)

// Positions the selection at the top of control
DocPosition(WP_MyDoc, 0)

// Positions the selection at the bottom of control horizontally centered
DocPosition(WP_MyDoc, 100, 50)

// Positions the document section found between the characters 123 and 158 
// at 30% of the remaining space in height and at 40% in width
DocPosition(WP_MyDoc, 123, 158, 30, 40)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Mostrando el texto seleccionado en la posición solicitada

`DocPosition(<Word Processing control> [, <Vertical positioning> [, <Horizontal positioning>]])`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 

**`<Vertical positioning>: (Entero opcional)`**

Posición vertical del texto seleccionado. Esta posición corresponde al porcentaje en altura del espacio restante (una vez deducida la altura del texto seleccionado) en el control Procesador de texto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DocPositionne_Hauteur1.gif)


Si no se especifica este parámetro (o si corresponde a la constante *dpDefault*), la función activará el desplazamiento mínimo para mostrar el texto seleccionado. 

Corresponde a: 

- valor incluido entre 0 y 100. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DocPositionne_Hauteur2.gif)


- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *dpBottom* | El texto seleccionado se muestra en la parte inferior de la pantalla control Procesador de texto. |
| *dpCenter* | El texto seleccionado se muestra en el centro vertical de la pantalla control Procesador de texto. |
| *dpDefault* <br>(Valor predeterminado) | Desplazamiento vertical mínimo para que el texto seleccionado se muestre en el control Procesador de texto. |
| *dpTop* | El texto seleccionado se muestra en la parte superior de control Procesador de texto. |






**`<Horizontal positioning>: (Entero opcional)`**

Posición horizontal del texto seleccionado. Esta posición corresponde al porcentaje en ancho del espacio restante (una vez deducido el ancho del texto seleccionado) en el control Procesador de texto. <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DocPositionne_Largeur1.gif)


Si no se especifica este parámetro (o si corresponde a la constante *dpDefault*), la función activará el desplazamiento mínimo para mostrar el texto seleccionado. 

Corresponde a: 

- valor incluido entre 0 y 100. Por ejemplo:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DocPositionne_Largeur2.gif)


- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *dpCenter* | El texto seleccionado se muestra en el centro horizontal de control Procesador de texto. |
| *dpDefault*<br>(Valor predeterminado) | Desplazamiento horizontal mínimo para que se muestre el texto seleccionado. |
| *dpLeft* | El texto seleccionado se muestra a la izquierda de control Procesador de texto. |
| *dpRight* | El texto seleccionado se muestra a la derecha de control Procesador de texto. |







<a name="SYNTAX2"></a>

### Mostrando una sección de texto en la posición solicitada

`DocPosition(<Word Processing control> , <Start of text to display> , <End of text to display> , <Vertical positioning> [, <Horizontal positioning>])`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular. 

**`<Start of text to display>: (Integro)`**

Posición inicial (en número de caracteres) de la sección de texto a visualizar. 

**`<End of text to display>: (Integro)`**

Posición final (en número de caracteres) de la sección de texto a visualizar. 

**`<Vertical positioning>: (Entero opcional)`**

Corresponde a: 

- valor incluido entre 0 y 100. Por ejemplo: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DocPositionne_Hauteur2.gif)

|   |   |
| --- | --- |

| *dpDefault* <br>(Valor predeterminado) | Desplazamiento vertical mínimo para que la sección de texto se muestre en el control Procesador de texto. |


## LanguageToName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de un idioma.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the name of the Armenian language
ResName is string
ResName = LanguageToName(languageArmenian)
```


<a name="Example2"></a>



```wl
// Find out the name of the English language in Australia
ResName is string
ResName = LanguageToName(languageEnglish, 3)
// Returns "Australian"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LanguageToName(<Language> [, <Sub-language>])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre del idioma especificado,

- Vacíe la cadena (""") si se desconoce el idioma especificado.




**`<Language>: (Constante de tipo Integer)`**

Idioma cuyo nombre se solicita. Para obtener más información sobre estas constantes, consulte [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).

**`<Sub-language>: (Entero opcional)`**

Subidioma cuyo nombre se solicita. Para más detalles sobre los sublenguajes, véase [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).
Si no se especifica este parámetro, se devuelve el nombre estándar del &lt;Idioma&gt; especificado..

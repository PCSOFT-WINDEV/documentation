
## PicRectangularSnipOCR (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta el OCR y extrae el texto de un área rectangular definida con el ratón.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sImageText is string
sImageText = PicRectangularSnipOCR()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PicRectangularSnipOCR([<Minimize>])`
---

**`<Result>: (Cadena de caracteres)`**

Texto extraído de la imagen capturada. 

**`<Minimize>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para minimizar la aplicación mientras se realiza la captura de pantalla,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

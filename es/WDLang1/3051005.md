
## MCIExecute( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ejecuta un comando MCI (Media control Interface).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the "C:\MyFiles\File.AVI" file
// in the "IMG_ImageAVI" Image control
ResExecuteMCI = MCIExecute("Open C:\MyFiles\File.AVI" + ...
	"Alias AVI Style Child Parent" + Handle(IMG_ImageAVI))
ResExecuteMCI2 = MCIExecute("Play AVI Notify")
```


<a name="Example2"></a>



```wl
// Play a video "C:\MyFiles\File.AVI" in repeat mode
// Open the video
MCIOpen("C:\MyFiles\File.AVI", "AVI", MciStyleChild, IMG_Video, MciModeNormal)
// Play the video
MCIExecute("Play AVI repeat")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIExecute(<Command to Run> [, <Window name>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó el comando MCI,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para averiguarlo:

	- el número de error, utilice [MCIErr](../WDLang1/3051011.md).

	- los detalles de los errores, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) asociados a la constante *errMessage*.







**`<Command to Run>: (Cadena de caracteres)`**

MCI para ejecutar.

Se puede acceder a los diferentes comandos MCI con Windows a través de la DLL MMSYSTEM. Esta biblioteca, proporcionada por Microsoft, contiene todas las funciones necesarias para utilizar los dispositivos multimedia. Para más detalles, consulte la documentación de Microsoft.
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Window name>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.

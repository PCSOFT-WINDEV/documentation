
## WinForceDDW (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fuerza la función OVI (Oscurecimiento de Ventanas Inactivas) en una ventana, incluso si esta ventana no es accesible.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Hourglass(True)
// Force DDW on the main window
WinForceDDW(WIN_Main, True)
// Start an important calculation
ImportantCalculation()
// Stop DDW on the main window
WinForceDDW(WIN_Main, False)
HourGlass(False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinForceDDW(<Window> , <Activation of DDW>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana a utilizar. 

**`<Activation of DDW>: (booleano)`**

Nuevo estado de OVI:

- <u><u><u><u>True</u></u></u></u>: OVI se activa instantáneamente en la ventana especificada. La ventana sale por Grayed. 

- <u><u><u><u>False</u></u></u></u>: OVI se desactiva instantáneamente en la ventana especificada. La ventana se muestra normalmente entonces (excepto si está desactivada: permanece Grayed fuera).

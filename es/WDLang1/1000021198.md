
## EmulatePane (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula la selección de un panel en una Pestaña, Cinta o control Barra lateral.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the 5th tab of the TAB_Tab control
EmulatePane(TAB_Tab, 5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulatePane(<Control> , <Pane number>)`
---

**`<Control>: (Cadena de caracteres)`**

Nombre del control a manipular. Este control debe ser un Tab, Cinta o control Barra lateral.

**`<Pane number>: (Integro)`**

Subíndice del panel para seleccionar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función solo se debe utilizar en el código de las pruebas automatizadas.

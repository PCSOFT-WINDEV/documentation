
## StatusBarAddCell( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva celda a la barra de estado de la ventana actual.

**Observación**: La ventana debe tener una barra de estado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds "MyCell" to the status bar of the current window
StatusBarAddCell("MyCell", 100, LightRed)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`StatusBarAddCell(<Cell name> [, <Cell Size> [, <Cell Color>]])`
---

**`<Cell name>: (Cadena de caracteres)`**

Nombre de la celda a crear. Este nombre se utilizará para manipular la célula a través de la programación (con las propiedades por ejemplo). La primera celda se llama "". Si la célula ya existe, su description se modificará automáticamente.

**`<Cell Size>: (Entero opcional)`**

Tamaño de la célula en píxeles. Si este parámetro corresponde a -1, la celda ocupará todo el espacio disponible en la barra de estado.

**`<Cell Color>: (Entero o constante opcional)`**

Color de fondo de la celda. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Usando las propiedades WLanguage con las celdas en una barra de estado
<a name="usando_las_propiedades_wlanguage_con_las_celdas_una_barra_estado_ELTPARAGRAPHE000050"></a>

Se pueden aplicar varias propiedades de WLanguage a las celdas de la barra de estado.

Atención: Para usar una Property en una celda de una barra de estado, se debe usar la siguiente sintaxis:


```txt
{"Cell name"}..<Property name>
```


Para obtener más información, consulte [Lista de propiedades que se pueden utilizar con las celdas de la barra de estado](../WDChamp/1010035.md).

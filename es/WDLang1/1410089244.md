
## Germany.CheckIdNr (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número IdNr (número de identificación fiscal en Alemania).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyIdNr is string = "36574261809"
Germany.CheckIdNr(MyIdNr)	// Returns True

// Check a control
EDT_IdNr = "ABC"
Germany.CheckIdNr(EDT_IdNr)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Germany.CheckIdNr(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número IdNr válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número IdNr a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.




## DateTimeToEpoch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una fecha y hora local en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays "1501056581" (in France)
Info(DateTimeToEpoch("20170726100941"))
```
<a name="Example2"></a>

```wl
MyDay is DateTime
MyDay = "20170726100941"
// Displays "1501056581" (in France)
Info(DateTimeToEpoch(MyDay))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DateTimeToEpoch(<Data to convert> [, <Conversion options>])`
---

**`<Result>: (Entero de 8 bytes)`**



- Fecha y hora en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC).

- -1 si se produce un error (fecha y hora no válidas).




**`<Data to convert>: (Cadena de caracteres o variable de tipo DateTime)`**

Fecha y hora local a convertir. Puede corresponder a:

- una variable de tipo DateTime.

- una cadena de caracteres con uno de los siguientes formatos:

	- AAAAMMDDHHmmSSCC

	- AAAAMMDDHHmmSSCC

	- AAAAMMDDHHmmSS

	- AAAAMMDDHHmm

	- AAAAMMDDHH

	- AAAAMMDD





Este parámetro no será considerado como válido en los siguientes casos: 

- si la fecha no es válida o es anterior a 01/01/1970. La validez de las fechas se puede comprobar con la función [DateValid](../WDLang1/3027003.md).

- si la hora no es válida. La validez de las horas se puede comprobar con la función [TimeValid](../WDLang1/3027021.md).


Observación: Es posible utilizar fechas más allá de 2038. 

**`<Conversion options>: (Constante opcional de tipo Integer)`**

Opciones para convertir la fecha y hora en formato Epoch:



|   |   |
| --- | --- |
| *epochUnixMillisecond* | El resultado se expresa en milisegundos desde el 01/01/1970 a las 00h00 UTC (hora 0 en Unix). |
| *epochUnixSecond*<br>(valor predeterminado) | El resultado se expresa en segundos desde el 01/01/1970 a las 00h00 UTC (hora 0 en Unix). |

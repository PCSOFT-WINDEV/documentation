


## TreeDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Actualiza la visualización de un control TreeView desde una posición determinada. 

**Observación**: La función **TreeDisplay** puede utilizarse en:

- una control TreeView de navegación o de memoria. Para una navegación por control TreeView, las modificaciones realizadas en el archivo de datos de relacionado se aplican al control TreeView.

- una selección única o una selección múltiple control TreeView.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an element into an HFSQL data file
HAdd(FILE_Element)
// Reinitialize the TreeView control
TreeDisplay(TREE_MyTreeview)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TreeDisplay(<TreeView control> [, <Position>])`
---

**`<TreeView control>: (Nombre del control)`**

Nombre de control TreeView a actualización.
Si este parámetro corresponde a una cadena vacía (""), se refresca el control TreeView al que pertenece el Event actual.

**`<Position>: (Entero, carácter o constante (opcional))`**

Indica la posición de inicio para la actualización. Si no se especifica este parámetro:

- ***Para un control TreeView archivo de datos***, la función se comporta como si la constante *taInit* se hubiese pasado como parámetro.

- ***Para un control TreeView memoria***, la función no tiene ningún efecto.




Este parámetro puede corresponder a:

- Un entero. La actualización se realiza a partir de esta posición. El Line seleccionado no se modifica.
	***Para un control TreeView archivo de datos***, el evento "Visualización de una fila" se ejecuta para la fila &lt;Posición&gt;. El elemento actual en el control TreeView es el mismo que el registro actual en el archivo de datos.
	***Para un control TreeView memoria***, la fila &lt;Posición&gt; vuelve a mostrarse.

- Un carácter o una constante: 
	

|   | Navegación por control TreeView cargada en memoria | Memoria control TreeView |
| --- | --- | --- |
| *taCurrentSelection* | <br><br>	- Vuelve a mostrar la control TreeView.<br><br>	- El Line seleccionado no ha cambiado.<br><br><br> | <br><br>	- Vuelve a mostrar la control TreeView.<br><br>	- El Line seleccionado no ha cambiado.<br><br><br> |
| *taCurrentRecord* | <br><br>	- Vuelve a ejecutar la consulta relacionada si es necesario.<br><br>	- Tiene en cuenta los registros agregados, modificados o eliminados por otros equipos.<br><br>	- Selecciona la actual Record.<br><br>	- Se conserva la posición de la barra de desplazamiento mientras la selección siga siendo visible.<br><br>	- Ejecuta las Event s de "Inicialización" y "Fin de inicialización" del control, luego la Event de "Selección de una fila" para la fila seleccionada (aunque no esté modificada).<br><br><br> | Ningún efecto. |
| *taStart* | <br><br>	- Actualiza el control TreeView posicionándose al principio de control.<br><br>	- El primer Line es seleccionado. Esta fila corresponde al primer registro.<br><br><br> | Ningún efecto. |
| *taInit* | <br><br>	- Ejecuta el código de inicialización de control TreeView (código de inicialización y final del código de inicialización).<br><br>	- Llena la control TreeView.<br><br>	- La primera Line de control TreeView es seleccionada. Esta línea corresponde al actual Record al final del código de inicialización de control TreeView.<br><br><br> | Ningún efecto. |
| *taReExecuteQuery* | <br><br>	- Vuelve a ejecutar la consulta relacionado para control TreeView.<br><br>	- Actualiza el control TreeView posicionándose al principio de control.<br><br>	- La primera Line de control TreeView es seleccionada. Esta fila corresponde al primer registro.<br><br><br>**Atención**: Sólo se ejecuta la consulta relacionado al control TreeView. Si se han especificado opciones específicas con [HExecuteQuery](../WDLang4/3044080.md) (*hModifyFile*, por ejemplo), estas opciones se ignoran. | Ningún efecto. |

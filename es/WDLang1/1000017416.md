


## &lt;Caja de lista&gt;.Borrar todo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un cuadro de lista, ListView o control Combo Box rellenado mediante programación.

**Observación**: **&lt;Caja de lista&gt;.Borrar todo** puede utilizarse en los controles de cuadro de lista de selección única y de selección múltiple.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all elements from the "LIST_CustomerList" control
LIST_CustomerList.DeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<List Box control>.DeleteAll()`
---

**`<List Box control>: (Nombre del control)`**

Nombre de la caja de la lista (o Combo Box) control a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000257"></a>

- Para eliminar todas las filas de una control Tabla, utilice [&lt;Tabla&gt;.DeleteAll](../WDLang1/1000024292.md).

- **&lt;Caja de lista&gt;.Borrar todo** puede utilizarse en una columna "Combo Box" tabla.

- Para los controles editables de Combo Box, el área de entrada está despejada.

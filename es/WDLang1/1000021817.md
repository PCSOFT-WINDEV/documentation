
## SysMemoryExternalStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la cantidad de memoria disponible en el área de almacenamiento externo del dispositivo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the amount of available memory on the external storage space
nFreeSpace is 8-byte int 
nFreeSpace = SysMemoryExternalStorage(1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysMemoryExternalStorage(<Subscript> [, <Type>])`
---

**`<Result>: (Integro)`**



- Cantidad de memoria (en bytes), 

- -1 si se produce un error.




**`<Subscript>: (Integro)`**

Índice del espacio de almacenamiento externo que se va a utilizar (si el dispositivo incluye varios espacios de almacenamiento externo). 
Observación: [SysNbExternalStorage](../WDLang1/1000021816.md) devuelve el número de espacios de almacenamiento externos encontrados en el dispositivo.

**`<Type>: (Constante opcional de tipo Integer)`**

Tipo de memoria a calcular: 


|   |   |
| --- | --- |
| *sysStorageSpace* | Total de savia disponible para el almacenamiento. |
| *sysFreeStorageSpace*<br>(Valor predeterminado) | Espacio de almacenamiento disponible. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Un mismo dispositivo puede contener varias áreas de almacenamiento externo (por ejemplo, un área corresponde a la partición "externa" de la memoria permanente y otra área corresponde a la SDCard).

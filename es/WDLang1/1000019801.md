
## Funciones para la gestión de los controles TreeMap (sintaxis de prefijo)
			

Las siguientes funciones se utilizan para gestionar los controles de TreeMap (sintaxis de prefijo):



|   |   |
| --- | --- |
| [&lt;Borrar todo](../WDLang1/1000019800.md) | Borra todos los elementos de un control TreeMap. |
| [&lt;Mapa de árbol&gt;.InfoXY](../WDLang1/1000019743.md) | Devuelve el índice del elemento que aparece en una posición determinada en una control TreeMap. |
| [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md) | Añade un elemento a una control TreeMap. |
| [&lt;Seleccione Grupo](../WDLang1/1000019793.md) | Selecciona visualmente un grupo de elementos en un control TreeMap. |
| [&lt;TreeMap&gt;.Seek](../WDLang1/1000019742.md) | Busca el índice de un elemento en una control TreeMap a partir del elemento ruta. |
| [.Delete](../WDLang1/1000019799.md) | Borra un elemento de un control TreeMap. |

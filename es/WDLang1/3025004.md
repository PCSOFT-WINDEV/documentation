
## TextWidth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el ancho (en píxeles) del texto mostrado en una control Campo de entrada, en una control Combo Box o en una control Estático.. Esto le permite predecir el ancho ocupado por el texto para ampliar la control si es necesario.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ResNbPixels is real = TextWidth(EDT_Edit1, "Hello")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TextWidth(<Control name> , <Text> [, <Options>])`
---

**`<Result>: (Real)`**

Ancho del texto (en píxeles) en la control especificada (según el estilo de la control).

Puedes get el ancho en milímetros usando la siguiente fórmula: 
&lt;N° pixeles&gt; = &lt;Resolución pantalla&gt;\*&lt;N° milímetros&gt;/25.4


**`<Control name>: (Cadena de caracteres)`**

Nombre de la edición control, control Estático o control Combo Box.


**`<Text>: (Cadena de caracteres)`**

Texto cuyo ancho debe ser calculado.

**`<Options>: (Constante opcional de tipo Integer)`**

Configura el modo para calcular el ancho del texto:


|   |   |
| --- | --- |
| *tWithMargin* | Permite tener en cuenta los márgenes set para el control (sangría izquierda, sangría derecha, etc.). <br>Esta constante puede ser combinada con la constante *tMultiline*. <br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *tDefault*<br>(valor predeterminado) | Anchura del texto en la control. |
| *tMultiline* | Anchura del texto en la control. El ancho máximo corresponderá al ancho de la control.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TexteLargeur.gif)<br> |
| *tSingleLine* | Anchura del texto sin tener en cuenta la anchura de la control. Esta constante se utiliza para get una precisión mayor que la dada por la constante *tDefault*.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TexteLargeur.gif)<br> |




## SchedulerTimeDisplayed (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el intervalo de tiempo que muestra control Planificador en el modo Día o en el modo Semana.. La barra de desplazamiento y la altura de las filas de la control Planificador se modifican para mostrar el intervalo de tiempo especificado.

Si el intervalo de tiempo de la control Planificador es mayor que el que se muestra, la barra de desplazamiento le permitirá acceder a los tiempos antes y después del intervalo.



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions the scrollbar in order for the visible time range to go from 9:00 to 19:00 
SchedulerTimeDisplayed(SCH_Schedule1, "0900", "1900")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerTimeDisplayed(<Scheduler control> , <Start time> , <End time>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Start time>: (Cadena de caracteres)`**

Hora de inicio de la visualización (en formato HHMM o HH).

**`<End time>: (Cadena de caracteres)`**

Hora de finalización de la visualización (en formato HHMM o HH).

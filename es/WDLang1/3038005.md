


## MenuSelectPlus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita una opción de menú (menú emergente o menú desplegable). Se puede seleccionar la opción de menú especificada. Esta función vuelve a activar una opción del menú Grayed con [MenuSelectMinus](../WDLang1/3038010.md).

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Re-enable the "OPT_IconBar" option in 
// the drop-down menu ("Menu") of the current window
MenuSelectPlus(OPT_IconBar) 
// Re-enables the "OPT_IconBar" option in the "Menu" of the "WIN_Win1" window
// The WIN_Win1 window is not the current window
MenuSelectPlus(WIN_Win1, OPT_IconBar)
// Re-enables the "OPT_IconBar" option in "PopupMenu1" 
MenuSelectPlus(WIN_Win1, PopupMenu1.OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuSelectPlus([<Window name>, ] <Name of the menu option>)`
---

**`<Window name>: (Cadena de caracteres opcional)`**

Nombre de la ventana que contiene la opción de menú para habilitar. 
Si no se especifica este parámetro, **MenuSelectPlus** manipular la ventana actual.

**`<Name of the menu option>: (Cadena de caracteres)`**

Nombre de la opción de menú que se va a habilitar. Este nombre se define en la ventana de description de la opción de menú. Este nombre debe ser especificado sin comillas. 
Si este parámetro corresponde al nombre de un menú, todas las subopciones de este menú también estarán habilitadas (recursivas).. Para volver a activar una opción en un menú emergente, utilice la siguiente notación: &lt;Nombre del menú emergente&gt;.&lt;Nombre de la opción&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Consejo: Volver a habilitar una set de opciones
<a name="consejo_volver_habilitar_una_set_opciones_ELTPARAGRAPHE000048"></a>

Para habilitar una set de opciones, debes:

1. Asigne las opciones a un grupo de controles.

2. Vuelva a habilitar las opciones (y los demás controles del grupo) ajustando el [Estado](../Proprietes/2510039.md) Property a Active y aplicándolo al grupo de controles.

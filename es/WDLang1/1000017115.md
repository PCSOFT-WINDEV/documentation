
## Constantes de servicio
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para gestionar los servicios:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **AccountLocalService** | Cuenta local que tiene los derechos recomendados para ejecutar un servicio. En la red, esta cuenta será considerada como huésped. | [Servicio Variable](../WDLang1/1000017118.md) |
| **AccountNetworkService** | Cuenta local que tiene los derechos recomendados para ejecutar un servicio. En la red, esta cuenta tendrá los derechos del ordenador actual. | [Servicio Variable](../WDLang1/1000017118.md) |
| **AccountLocalSystem** | Cuenta local que representa al sistema operativo. Esta cuenta tiene todos los derechos. En la red, esta cuenta tendrá los derechos del ordenador actual. | [Servicio Variable](../WDLang1/1000017118.md) |
| **esStop** | El servicio se detiene correctamente, sin provocar ningún error.. Se ejecutan los diferentes procesos finales y se detiene el servicio. | [Servicio final](../WDLang1/1000018711.md) |
| **esFailure** | El servicio se detiene y desencadena un fallo. Los diferentes procesos finales no se ejecutan. | [Servicio final](../WDLang1/1000018711.md) |
| **elWarning** | Advertencia Event | [ServicioWriteEventLog](../WDLang1/1000018807.md) |
| **elError** | Error Event | [ServicioWriteEventLog](../WDLang1/1000018807.md) |
| **elInformation** | Información Event | [ServicioWriteEventLog](../WDLang1/1000018807.md) |
| **serviceCmdStop** | Detiene el servicio (equivalente a [ServiceStop](../WDLang1/1000017111.md)). | [ServicioControl](../WDLang1/1000017108.md) |
| **serviceCmdPause** | Pone en pausa el servicio (equivalente a [ServicePause](../WDLang1/1000017109.md)). | [ServicioControl](../WDLang1/1000017108.md) |
| **serviceCmdRestart** | Reinicia un servicio (equivalente a [ServicioContinuar](../WDLang1/1000017105.md)). | [ServicioControl](../WDLang1/1000017108.md) |
| **serviceCmdRereadConfig** | Pide al servicio que relea sus ajustes de configuración (equivalente a [ServicioActualización](../WDLang1/1000017107.md)). | [ServicioControl](../WDLang1/1000017108.md) |
| **serviceStartupAutomatic** | Iniciar automáticamente una vez que se inicializa el sistema | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceStartupDisabled** | El servicio está desactivado. Cualquier intento de arrancarlo desencadena un error. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceStartupManual** | Arranque manual | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceStartupSystem** | automatic se inicia al mismo tiempo que el sistema operativo (utilizado por los controladores). | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceErrorCritical** | El sistema escribe el fallo para iniciar el servicio en el registro del observador de Event y se reinicia en el modo "Última configuración adecuada conocida". | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceErrorSerious** | El sistema escribe el fallo para iniciar el servicio en el registro del observador de Event y se reinicia en el modo "Última configuración adecuada conocida" si este modo no está ya utilizado. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceErrorIgnore** | El sistema no realiza ninguna acción si el servicio no se inicia. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceErrorLog** | El sistema escribe el fallo para iniciar el servicio en el registro del observador de Event. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceStatusStopped** | El servicio está interrumpido. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusStopping** | El servicio recibió la orden de detenerse, pero aún no se ha detenido. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusStarting** | El servicio ha recibido una orden de inicio, pero aún no se ha iniciado. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusPausing** | El servicio recibió un comando de pausa pero aún no ha sido pausado. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusRestarting** | El servicio recibió un comando para reiniciar pero aún no ha sido reiniciado. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusStarted** | El servicio está actualmente en funcionamiento. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusPaused** | El servicio está en pausa. | [ServicioEsperandoEstado](../WDLang1/1000017101.md), [Estado del servicio](../WDLang1/1000017102.md) |
| **serviceStatusTimeout** | El servicio no ha alcanzado el estado solicitado al final del tiempo de espera. | [ServicioEsperandoEstado](../WDLang1/1000017101.md) |
| **serviceTypeInteractiveShared** | Servicio que permite interactuar con el escritorio y compartir su Process con otros servicios. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceTypeInteractiveSimple** | El servicio permite interactuar con el escritorio y operar por sí mismo en su Process. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceTypeShared** | Servicio estándar que comparte su proceso con otros servicios. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceTypeDriverKernel** | Driver del kernel. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceTypeDriverFileSystem** | Controlador del sistema de archivos. | [Servicio Variable](../WDLang1/1000017118.md) |
| **serviceTypeSimple** | Servicio estándar que opera por su cuenta en su Process. | [Servicio Variable](../WDLang1/1000017118.md) |

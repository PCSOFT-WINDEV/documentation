## Presentación
<a name="presentacion_ELTTEXTE000082"></a>
Los dongles de la familia Hasp (Hasp3, Hasp4, NetHasp, memoHasp y Hasp HL) se utilizan para proteger las aplicaciones desarrolladas con WINDEV o WEBDEV.

Las funciones Hasp permiten restringir el uso de una aplicación a las personas que poseen un dongle válido.

Otro método de protección consiste en utilizar [las funciones de gestión de las claves de activación](../WDLang1/1000018998.md).

**Observaciones**: 

- Para utilizar las funciones Hasp, debe conocer el modo de funcionamiento de las teclas Hasp y disponer de su documentación.

- [Hasp](../WDLang1/3013038.md) no está disponible en 64 bits, utilice las funciones HaspHLxxx en su lugar.

## Funciones Hasp
<a name="funciones_hasp_ELTTEXTE000106"></a>
Las funciones Hasp son las siguientes:



|   |   |
| --- | --- |
| [Hasp](../WDLang1/3013038.md) | Interroga una llave electrónica HASP (HASP-3, HASP-4, MemoHasp o NetHasp). |
| [HaspHLDecrypt](../WDLang1/1000018894.md) | Descifra una cadena de caracteres utilizando los algoritmos de la clave Hasp. |
| [HaspHLEncrypt](../WDLang1/1000018893.md) | Cifra una cadena de caracteres utilizando los algoritmos de la clave Hasp especificada. |
| [HaspHLInfo](../WDLang1/1000018895.md) | Devuelve información sobre la llave electrónica Hasp. |
| [HaspHLLogin](../WDLang1/1000018891.md) | Conecta la aplicación con una tecla Hasp. |
| [HaspHLLogout](../WDLang1/1000018892.md) | Cierra la conexión de la aplicación a una clave Hasp. |
| [HaspHLRead](../WDLang1/1000018896.md) | Lee datos (cadena de caracteres o memoria intermedia) en la memoria interna de la tecla Hasp. |
| [HaspHLWrite](../WDLang1/1000018897.md) | Escribe datos (cadena de caracteres o memoria intermedia) en la memoria interna de la tecla Hasp. |

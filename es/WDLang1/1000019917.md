
## StringDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina: 

- todas las instancias de una subcadena en una cadena. 

- parte de una cadena dentro de otra cadena. La parte de la cadena a eliminar se identifica según su posición y longitud. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString = StringDelete("Antananarivo", "a")
// MyString = Antnnrivo
```

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminar todas las instancias de una subcadena en una cadena

`<Result> = StringDelete(<Initial string> , <Substring> [, <Options>])`
---

**`<Result>: (Cadena de caracteres Ansi, Unicode o Binaria)`**

Cadena resultante de la eliminación de las subcadenas. 

**`<Initial string>: (Cadena de caracteres Ansi, Unicode o Binaria)`**

Cadena inicial a manipular. Esta cadena de caracteres no se modifica. 

**`<Substring>: (Cadena de caracteres o array de caracteres Ansi, Unicode o Binarios)`**

Subcadena que se eliminará de la &lt;Cadena&gt;. Esta cadena puede corresponder a: 

- una cadena de caracteres. 

- un array de cadenas de caracteres. En este caso, todas las subcadenas se eliminarán de la .




**`<Options>: (Constante opcional de tipo Integer)`**

Modo utilizado para encontrar la subcadena: 



|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios). |


De forma predeterminada, la búsqueda distingue entre mayúsculas y minúsculas y busca todas las cadenas especificadas (incluso si forman parte de una palabra).

### Eliminar parte de una cadena según su posición y longitud

`<Result> = StringDelete(<Initial string> , <Position> , <Length>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena resultante de la eliminación de una parte de la cadena.

**`<Initial string>: (Cadena de caracteres)`**

Cadena inicial a manipular (tamaño máximo: 2 GB). Esta cadena de caracteres no se modifica.

**`<Position>: (Entero)`**

Posición de la parte de la cadena a eliminar.

**`<Length>: (Entero)`**

Longitud de la parte de la cadena a eliminar (en número de caracteres).


## ChangeSeparator( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el valor de los separadores (separador decimal o separador de mil) utilizados en las máscaras numéricas para la internacionalización.

Modificar el separador:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) de la ventana actual: utilizar **ChangeSeparator** y volver a mostrar la ventana actual con [Uso](../WDLang1/3038044.md).

## Sintaxis

`ChangeSeparator(<Separator Type> , <New value>)`
---

**`<Separator Type>: (Constante de tipo Integer)`**

Especifica el tipo de separador a modificar:


|   |   |
| --- | --- |
| *sepCURRENCY* | Símbolo utilizado para las monedas. |
| *sepDECIMAL* | Separador de decimales (si el sistema operativo utilizado es la versión francesa de Windows, el separador de decimales es la coma (", ") de Default). |
| *sepTHOUSAND* | Separador de mil (si el sistema operativo utilizado es la versión francesa de Windows, el separador es un carácter espacial ("") de Default). |



**`<New value>: (Carácter)`**

Nuevo valor para el separador especificado.

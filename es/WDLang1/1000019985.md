


## &lt;Organigrama&gt;.DeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los elementos de un control Organigrama. La control Organigrama está completamente despejada. 

**Observación**: Si el control Organigrama está relacionado con una fuente de archivo, la función **&lt;Organigrama&gt;.DeleteAll** no tiene ningún efecto sobre la fuente: no se borra ningún Record si la fuente no se borra de antemano.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all elements from the Organization Chart control
ORG_EXECUTIVE.DeleteAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Organization Chart control>.DeleteAll()`
---

**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama cuyos elementos serán borrados.

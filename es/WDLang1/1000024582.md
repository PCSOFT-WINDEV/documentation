


## &lt;Variable picLayer&gt;.Resize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Redimensiona una imagen en una variable de tipo [picLayer](../WDLang1/1000024604.md).



![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Zoom image: enlarge image twice
Result1 is boolean
Result1 = MyLayer.Resize(MyLayer.Width * 2, MyLayer.Height * 2)
IF Result1 = False THEN Error("The operation failed")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <picLayer image>.Resize(<New width> , <New height> [, <Option>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el redimensionamiento se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<New width>: (Entero)`**

Nuevo ancho de la imagen expresado en pixeles.

**`<New height>: (Entero)`**

Nuevo alto de la imagen expresado en pixeles.

**`<Option>: (Constante opcional de tipo Integer)`**

Opciones de redimensionamiento: 


|   |   |
| --- | --- |
| *drHighQuality* | Esta constante puede combinarse con las opciones de redimensionamiento homotético.<br><br>En modo homotético, esta constante permite mejorar la calidad de la imagen cuando esta se reduce con respecto a su tamaño inicial.<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *drHomothetic* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse. Esta ocupa:<br><br>- todo el alto especificado.<br><br>- todo el ancho especificado.<br><br><br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *drHomotheticCentered* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse y se centra. Esta ocupa:<br><br>- todo el alto especificado: la imagen se centra horizontalmente.<br><br>- todo el ancho especificado: la imagen se centra verticalmente.<br><br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *drHomotheticExtended* | La imagen se amplía o reduce para que el lado más pequeño de la imagen se muestre completamente.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *drStretched*<br>(Valor predeterminado) | La imagen se amplía si es necesario para ajustarse al nuevo ancho y alto especificados.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *drTiled* | La imagen se repite para ocupar el nuevo ancho y alto especificados.<br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |




## StringToTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una hora en formato de cadena de caracteres en una hora en formato HHMMSSCC que puede ser utilizada por una variable de tipo [Time](../Motscles/1514068.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyTime is Time
MyTime = StringToTime("12:10:25:78")	 // MyTime = "12102578"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = StringToTime(<String> [, <Format>])`
---

**`<Result>: (Cadena de caracteres o variable de tipo Time)`**



- Hora en formato HHMMSSCC (HH corresponde a las horas, MM a los minutos, SS a los segundos y CC a las centésimas de segundo).

- Cadena vacía ("") si se produce un problema.




**`<String>: (Cadena de caracteres)`**

Hora en el formato especificado en el parámetro &lt;Formato>.

**`<Format>: (Cadena de caracteres opcional)`**

Formato de la cadena que debe convertirse en hora. Puede utilizar:

- Los caracteres que representan los elementos de la hora (HH, MM, SS, CC o AP) se reemplazarán automáticamente por el valor del parámetro &lt;Cadena&gt;. 
	En esta cadena:

	- HH representa el número de horas,

	- MM representa el número de minutos,

	- SS representa el número de segundos,

	- CC representa el número de centésimas de segundo,

	- AP representa "AM" o "PM" (si se usa "AP", la hora se muestra en formato de 12 horas).




- La siguiente constante:
	


|   |   |
| --- | --- |
| *maskSystemTime* | Permite utilizar directamente la máscara utilizada por la hora del sistema. Esta máscara depende de la configuración del proyecto. <br>Para abrir las opciones de hora del proyecto: <br><br>	1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". La ventana de descripción del proyecto se muestra en pantalla. <br><br>	2. En la pestaña "Idiomas", seleccione la pestaña "Hora". Puede utilizar las opciones de Windows o parámetros específicos. <br><br><br> |




Si no se especifica este parámetro, se utiliza el formato HH:MM:SS:CC de forma predeterminada.

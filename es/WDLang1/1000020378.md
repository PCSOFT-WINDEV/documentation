


## RibbonEndExpandedMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Colpasa una cinta colapsada que se expande temporalmente. 

**Recordatorio**: [Colapsado](../Proprietes/2513014.md) se utiliza para colapsar una cinta. En este caso, sólo se muestran los títulos de los paneles (como en un menú estándar).. El contenido del panel se muestra cuando el usuario hace clic en el título del panel.. Si el usuario hace clic de nuevo en el título del panel o en un botón, el contenido del panel ya no se muestra automáticamente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Collapse the ribbon once the size was selected in a Combo Box control. 
// Selection of COMBO_Size
nSize is int = COMBO_Size[COMBO_Size]
RTFSelection(EDT_RTFTEXT, rtfFontSize, nSize)
RibbonEndExpandedMode()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RibbonEndExpandedMode([<Ribbon control>])`
---

**`<Ribbon control>: (Nombre de control opcional)`**

Nombre de la control Cinta de opciones a utilizar. Si no se especifica este parámetro, manipula la control Cinta de opciones de la ventana actual.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función no tiene ningún efecto si la cinta no está colapsada.. 

- Esta función no tiene efecto si no se encuentra ninguna cinta en la ventana actual.

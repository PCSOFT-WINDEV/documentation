


## &lt;DiskFile Variable&gt;.Close (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un fichero externo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
ResClose is int

// Open an external file
MyFile is DiskFile
MyFile = fOpen("C:\MyDirectories\File.txt", foReadWrite)
IF NOT ErrorOccurred THEN
	...
	// Close the external file
	ResClose = MyFile.Close()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <File used>.Close()`
---

**`<Result>: (Integro)`**



- 0 es que el archivo externo está cerrado,

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. **&lt;DiskFile Variable&gt;.Close** arroja un error si el archivo no se abre.




**`<File used>: (DiskFile Variable)`**

Nombre de la variable [DiskFile](../WDLang1/1410088883.md) inicializada con [fOpen](../WDLang1/3036036.md) o [fCreate](../WDLang1/3036017.md).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Cerrar un archivo abierto varias veces
<a name="cerrar_archivo_abierto_varias_veces_ELTPARAGRAPHE000272"></a>

El mismo archivo se puede abrir varias veces simultáneamente. Cada versión del archivo debe cerrarse individualmente mediante su propio identificador.
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Cierre de un archivo bloqueado
<a name="cierre_archivo_bloqueado_ELTPARAGRAPHE000284"></a>

Un archivo bloqueado se desbloqueará automáticamente cuando lo cierre.

**Recordatorio**: [&lt;DiskFile Variable&gt;.Lock](../WDLang1/1410089037.md) se utiliza para bloquear un archivo externo. [&lt;DiskFile Variable&gt;.Unlock](../WDLang1/1410089040.md) se utiliza para desbloquear un archivo externo.

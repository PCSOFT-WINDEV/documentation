


## &lt;Chrono&gt;.Start (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. El cronómetro se detiene: 

- con la función [&lt;Chrono&gt;.End](../WDLang1/1000023789.md).

- con la función [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md). 


Si el cronómetro se detiene, se puede reiniciar con la función [&lt;Chrono&gt;.Resume](../WDLang1/1000023792.md). 

El valor del cronómetro se puede leer en cualquier momento: 

- con la función [ChronoValue](../WDLang1/3027033.md).

- con la propiedad **Value** de la variable de tipo [Chrono](../WDLang1/1000023560.md).









<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyChrono1 is Chrono
MyChrono2 is Chrono
// Start the first stopwatch 
MyChrono1.Start()
// Start the second stopwatch 
MyChrono2.Start()
// Stop the first stopwatch
DurationChronoStart is Duration = MyChrono1.End()
MyFunction()
// Stop the second stopwatch
DurationChronoStart is Duration = MyChrono2.End()
Info("The MyFunction() process lasted " + DurationMyFunction, ...
	"The whole process lasted " + DurationChronoStart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cronometraje con una variable de tipo Chrono

`<Stopwatch>.Start()`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Error
<a name="error_ELTPARAGRAPHE000315"></a>

Si no se llama a la función **&lt;Chrono&gt;.Start** antes de las funciones [ChronoValue](../WDLang1/3027033.md) y [&lt;Chrono&gt;.End](../WDLang1/1000023789.md), se produce un error WLanguage (error de tipo "No se ha iniciado el cronómetro").
<a name="NOTE0_2"></a>


### Hilos y tareas paralelas
<a name="hilos_tareas_paralelas_ELTPARAGRAPHE000335"></a>

Cada hilo (y cada tarea paralela) tiene sus propios cronómetros. 

Dos hilos distintos (o dos tareas paralelas distintas) pueden realizar cronometrajes separados utilizando el mismo número de cronómetro. 

El error WLanguage "No se ha iniciado el cronómetro" solo se mostrará si no se llama a la función **&lt;Chrono&gt;.Start** en el mismo hilo (o en la misma tarea paralela) que las funciones [ChronoValue](../WDLang1/3027033.md) y [&lt;Chrono&gt;.End](../WDLang1/1000023789.md).


## gLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres que contiene una Line. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres se utiliza para dibujar una Line:  

- en un elemento de un List Box o control Combo Box.

- en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Draw a green line in an element of "LIST_Figure" control
ListAdd(LIST_Figure, gLine(0, 0, 25, 18, LightGreen))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gLine(<X1> , <Y1> , <X2> , <Y2> [, <Color>])`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que contienen una Line.  

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<X1>: (Real)`**

Coordenada X (en pixeles) del punto inicial de la línea.

**`<Y1>: (Real)`**

Coordenada Y (en píxeles) del punto de inicio de la Line.

**`<X2>: (Real)`**

Coordenada X (en pixeles) del punto final de la línea. Este punto final no se dibuja.

**`<Y2>: (Real)`**

Coordenada Y (en píxeles) del punto final de la Line. Este punto final no se dibuja.

**`<Color>: (Entero o constante (opcional))`**

Color de la línea. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el Color de la línea corresponde al Color del texto actual (definido por [gPen](../WDLang1/3029039.md)).

## Observaciones




### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000118"></a>

Todas las coordenadas se expresan en píxeles en relación con la Corner superior izquierda del elemento utilizado (en relación con la Corner superior izquierda de la fila que se encuentra en el Cuadro de Lista o control Combo Box por ejemplo).

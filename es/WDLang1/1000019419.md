
## RegistryRename( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Renombra una clave y/o un valor en el registro.

**Observación**: El cambio de nombre de una clave consiste en: 

- Copiar la clave. 

- Cambiar el nombre de la clave copiada. 

- Borrar la clave anterior. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Renames the "MyApp" key to "MyApplication"
RegistryRename("HKEY_LOCAL_MACHINE\SOFTWARE\MyApp", ...
		"HKEY_LOCAL_MACHINE\SOFTWARE\MyApplication")

// Renames the "Auto" value to "Automatic"
RegistryRename("HKEY_LOCAL_MACHINE\SOFTWARE\MyApp", "Auto", "Automatic")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Renombrar una clave en el registro

`<Result> = RegistryRename([<Access mode>, ] <Name of key to rename> , <New key name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cambió el nombre de la clave de registro,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Name of key to rename>: (Cadena de caracteres)`**

ruta completa de la clave a renombrar. Si no existe esta clave, &lt;Resultado&gt; devolverá <u><u><u><u>False</u></u></u></u>.

**`<New key name>: (Cadena de caracteres ()`**

Nuevo ruta y nombre de la llave. Si esta clave ya existe, &lt;Resultado&gt; devolverá <u><u><u><u>False</u></u></u></u>.


<a name="SYNTAX2"></a>

### Renombrar una clave de registro así como su valor

`<Result> = RegistryRename([<Access mode>, ] <Name of key to rename> , <Name of value to rename> [, <New key name>] , <New value name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cambió el nombre de la clave de registro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Access mode>: (Constante de tipo Integer)`**

Modo de acceso al registro:



|   |   |
| --- | --- |
| *registryMode32* | Modo forzado para acceder al registro como un programa de 32 bits. |
| *registryMode64* | Modo forzado para acceder al registro como un programa de 64 bits. |
| *registryModeAuto*<br>(Valor predeterminado) | Modo de acceso al registro de automatic: <br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 32 bits manipula el registro como un programa de 32 bits.<br><br>- una aplicación de 32 bits que se ejecuta en un sistema de 64 bits manipula el registro desde la siguiente rama:<br>	<br><pre><code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node</code></pre><br><br><br>- una aplicación de 64 bits que se ejecuta en un sistema de 64 bits manipula el registro como un programa de 64 bits.<br><br><br> |



**`<Name of key to rename>: (Cadena de caracteres)`**

ruta completa de la clave a renombrar. Si no existe esta clave, &lt;Resultado&gt; devolverá <u><u><u><u>False</u></u></u></u>.

**`<Name of value to rename>: (Cadena de caracteres)`**

Nombre del valor a renombrar en el registro para la clave especificada. Si no existe este valor, &lt;Resultado&gt; devolverá <u><u><u><u>False</u></u></u></u>.

**`<New key name>: (Cadena de caracteres opcional)`**

Nuevo ruta y nombre de la llave. Si este parámetro corresponde a una cadena vacía ("") o no se especifica, el nombre de la clave no se modifica. Si esta clave ya existe, &lt;Resultado&gt; devolverá <u><u><u><u>False</u></u></u></u>.

**`<New value name>: (Cadena de caracteres)`**

Nuevo nombre del valor en el registro para la clave especificada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **RegistryRename** realiza un copy recursivo y luego una eliminación recursiva.

- A partir de Windows Vista, las operaciones de cambio de nombre se realizan en una transacción. Si una operación falla, se cancelan todas las operaciones anteriores.

- El propietario de la clave se pierde cuando se cambia el nombre de la clave.


## Constantes para la gestión de telemetría
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes constantes se utilizan para administrar la telemetría:

| Constante | Detalles | Función |
| --- | --- | --- |
| **tsAbandon** | Estado actual de la telemetría: la telemetría se abandona después de demasiados errores de envío. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tsCollect** | Estado actual de la telemetría: la telemetría está activada y se está realizando una recolección. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tsSend** | Estado actual de la telemetría: la telemetría está activada y se envió la información. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tsError** | Estado actual de la telemetría: la telemetría está habilitada pero no se ha enviado información al servidor. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tsInactive** | Estado actual de la telemetría: la telemetría es Inactive. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcEnableLog** | Configure el registro de telemetría. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcIdleTimeout** | Configurar la telemetría (contar el tiempo de actividad en una ventana): el cronómetro de conteo se detiene automáticamente después de este tiempo de inactividad. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcStatus** | Estado actual de la telemetría. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcFileLog** | Configurar el registro de telemetría: La telemetría está usando un registro de archivos. El archivo correspondiente se guarda en la ventana de diálogo " AppData\\Itinerancia\\Itinerancia _ASLASH_.  NO, NO, NO, NO, NO. "directorio. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcMemoryLog** | Configurar el registro de telemetría: La telemetría está usando un registro de memoria. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcOfflineMode** | Configuración de la telemetría: Modo de funcionamiento de la telemetría. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcNumberSendAttempts** | Configuración de la telemetría: Número máximo de intentos de enviar datos de telemetría en modo online | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcMaximumPeriodCollection** | Configuración de la telemetría: Periodicidad máxima para la recogida de los datos de telemetría (en segundos). Los datos se enviarán al final de este período, independientemente del tamaño de los datos recogidos. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcMinimumPeriodCollection** | Configuración de la telemetría: Periodicidad mínima para la recogida de los datos de telemetría (en segundos). Los datos pueden enviarse al final de este período si se alcanza el tamaño mínimo. | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcMinimumSizeCollection** | Configuración de la telemetría: Tamaño mínimo de los datos recogidos durante el período mínimo de recogida (en kilobytes). | [TelemetryConfigure](../WDLang1/1000021913.md) |
| **tcGetMemoryLog** | Devuelve el contenido del registro de memoria. | [TelemetryConfigure](../WDLang1/1000021913.md) |




## Funciones List Box, ListView y control Combo Box
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles List Box, ListView y Combo Box:



|   |   |
| --- | --- |
| [FileToMemoryList](../WDLang1/3049007.md) | Rellena un cuadro de lista o control Combo Box de forma programada con los registros de un archivo de datos o de la consulta. |
| [ListAdd](../WDLang1/3049004.md) | Añade un elemento a un cuadro de lista, ListView o control Combo Box rellenado mediante programación. |
| [ListarModificar](../WDLang1/3049009.md) | Modifica un elemento de una Caja de Listas, ListView o control Combo Box poblada programáticamente. |
| [ListCount](../WDLang1/3049014.md) | Devuelve el número de elementos de un cuadro de lista, ListView o control Combo Box. |
| [ListDelete](../WDLang1/3049016.md) | Elimina un elemento de un cuadro de lista, ListView o control Combo Box rellenado mediante programación. |
| [ListDeleteAll](../WDLang1/3049013.md) | Borra un cuadro de lista, ListView o control Combo Box rellenado mediante programación. |
| [ListDisplay](../WDLang1/3049005.md) | Actualiza un cuadro de lista, ListView o control Combo Box basado en un archivo de datos |
| [ListInfoXY](../WDLang1/3049006.md) | Devuelve el índice de la fila (o Image) en una posición dada en un Cuadro de Lista o control ListView. |
| [ListInsert](../WDLang1/3049002.md) | Inserta un elemento en una Caja de Listas, ListView o control Combo Box poblada programáticamente. |
| [ListMoveLine](../WDLang1/1000017279.md) | Mueve un fila o intercambia dos filas en un List Box o control Combo Box. |
| [ListPosition](../WDLang1/3049015.md) | Muestra un cuadro de lista o control ListView de un elemento especificado o devuelve el índice del primer elemento mostrado en un control List Box (si se utilizó la barra de desplazamiento). |
| [ListSeek](../WDLang1/3049003.md) | Busca un elemento en un cuadro de lista, ListView o control Combo Box (incluyendo los valores descritos por [gStoredValue](../WDLang1/1410088363.md)). |
| [ListSelect](../WDLang1/3049011.md) | Devuelve el índice del elemento seleccionado en un Cuadro de Lista, ListView o control Combo Box. |
| [ListSelectCount](../WDLang1/3049010.md) | Devuelve el número de elementos seleccionados en un cuadro de lista, ListView o control Combo Box. |
| [ListSelectMinus](../WDLang1/3049008.md) | Deselecciona un elemento de un Cuadro de Listas, ListView o control Combo Box. |
| [ListSelectPlus](../WDLang1/3049012.md) | Selecciona uno o varios elementos de un Cuadro de Lista, ListView o control Combo Box. |
| [ListSort](../WDLang1/1000017022.md) | Ordena los elementos de una Caja de Listas, ListView o control Combo Box poblada programáticamente. |

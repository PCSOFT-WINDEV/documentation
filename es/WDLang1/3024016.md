


## Right (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae los últimos caracteres de una cadena o de un búfer de cadenas.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Extract characters
Right("The cuckoo", 6)   // Returns "cuckoo"
Right("ABC", 50)   // Returns "ABC"
Right(" AZELMA")   // Returns "AZELMA"

MyString is string = "Antananarivo - Madagascar"
Right(MyString, 10) // Returns "Madagascar"
```

## Sintaxis

### Extracción de los últimos caracteres de una cadena

`<Result> = Right(<Initial string> [, <Length>])`
---

**`<Result>: (Cadena de caracteres)`**



- Últimos caracteres de . Corresponde a &lt;Cadena inicial&gt; si el parámetro &lt;Longitud&gt; es mayor que la longitud de &lt;Cadena inicial&gt;.

- EOT si el parámetro &lt;Longitud&gt; es negativo. Se produce un error de WLanguage (error no fatal). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Initial string>: (Cadena de caracteres)`**

Cadena de la que se extraerán los últimos caracteres. Esta cadena no se modifica.

**`<Length>: (Entero opcional)`**

Número de **caracteres** a extraer. Si no se especifica este parámetro, &lt;Resultado&gt; corresponde a &lt;Cadena inicial&gt; sin espacios al inicio.

### Extracción de los últimos caracteres de un búfer de cadena 

`<Result> = Right(<Initial buffer> [, <Length>])`
---

**`<Result>: (Buffer)`**



- Últimos caracteres de . Corresponde a &lt;Buffer inicial&gt; si el parámetro &lt;Tamaño&gt; es mayor que el tamaño de &lt;Buffer inicial&gt;.

- EOT si el parámetro &lt;Longitud&gt; es negativo. Se produce un error de WLanguage (error no fatal). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

- Buffer no válido si &lt;Buffer inicial&gt; contiene una cadena UNICODE.




**`<Initial buffer>: (Buffer)`**

Nombre del buffer del que se extraerá la parte derecha. Este buffer no se modifica.

**`<Length>: (Entero opcional)`**

Número de **bytes** a extraer. Si no se especifica este parámetro, &lt;Resultado&gt; corresponde a &lt;Buffer inicial&gt; sin espacios al inicio.

## Observaciones


### Función Right y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000130"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Recordatorio**: En UNICODE, cada carácter ocupa dos bytes de memoria. Por lo tanto, el tamaño de memoria de la cadena UNICODE es el doble del número de caracteres que contiene realmente la cadena.

El primer carácter de la cadena está en el índice 1.

**Observación**: Si el resultado en una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

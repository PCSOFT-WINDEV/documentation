


## <Menu>.AddURLOption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú abre la página que corresponde a la URL especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a new menu
_Menu.AddMenu("MNU_Site", "My Site")
// Add an option into the new menu
MNU_Site.AddURLOption("OPT_PCSOFT", "PC Soft", "http://www.windev.com")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Source menu>.AddURLOption(<Option to create> , <Caption> , <Option URL>)`
---

**`<Result>: (Integro)`**

Posición de la opción en el menú. Si no se agrega la opción, se produce un error fatal.

**`<Source menu>: (Nombre del menú)`**

Nombre del menú al que debe agregarse la opción de menú. La opción se añadirá después de la última opción de este menú.
Observación: Este nombre puede corresponder a un menú contextual. 

**`<Option to create>: (Cadena de caracteres)`**

Nombre de la opción de menú a agregar. Este nombre se utilizará para manipular la opción de menú mediante programación. Si existe una opción con el mismo nombre, se produce un error fatal.

**`<Caption>: (Cadena de caracteres)`**

Texto de la nueva opción de menú. Este texto aparecerá en la página y permitirá a los usuarios seleccionar la opción.

**`<Option URL>: (Cadena de caracteres)`**

Dirección de la página que se abrirá al seleccionar la opción.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Abrir una página AWP
<a name="abrir_una_pagina_awp_ELTPARAGRAPHE000193"></a>

Puede utilizar la función **<Menu>.AddURLOption** para abrir una página AWP. En este caso, puede utilizar la función [PageAddress](../WDLang2/1000021660.md) para construir la URL de la página. Por ejemplo:


```wl
MENU_Menu1.AddURLOption("OPT_AWPPage", "Active WEBDEV Page", PAGE_AWP.Address())
```

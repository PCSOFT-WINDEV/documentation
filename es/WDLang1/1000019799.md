


## <TreeMap>.Delete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un elemento de un control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the "Recipes\Dessert\Vanilla ice cream" element
TMAP_Recipes.Delete("Recipes\Dessert\Vanilla ice cream")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Eliminar un elemento del TreeMap especificando su índice

`<TreeMap control>.Delete(<Element index>)`
---

**`<TreeMap control>: (Nombre del control)`**

Nombre de control TreeMap a utilizar.

**`<Element index>: (Integro)`**

Índice del elemento a suprimir. La función no tiene efecto si este índice no corresponde a un elemento control.


<a name="SYNTAX2"></a>

### Eliminar un elemento del TreeMap especificando su ruta

`<TreeMap control>.Delete(<Element path>)`
---

**`<TreeMap control>: (Nombre del control)`**

Nombre de control TreeMap a utilizar.

**`<Element path>: (Cadena de caracteres)`**

Ruta del elemento a borrar. La función no tiene efecto si este ruta no corresponde a un elemento control.
**Observación**: Esta ruta debe ser idéntica a la que se utilizó cuando se añadió el elemento en la control TreeMap mediante [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md). 
El separador de diferentes elementos ruta puede ser: 

- el carácter "\\".. Por ejemplo: "MiDirectorio\\MiArchivo". 

- el carácter "TAB. Por ejemplo: "Postres "+TAB+"Helados".




## &lt;PositionDateTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que se muestra en una control Planificador. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Position the Scheduler control on today's date
SCH_Schedule.PositionDateTime(Today())

// Position the Scheduler control on December 20, 2012
SCH_Schedule.PositionDateTime("20181220")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Posicionar el programa en una fecha determinada

`<Scheduler control>.PositionDateTime(<Start date>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- la control Planificador de un reporte.




**`<Start date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de inicio de la visualización.


<a name="SYNTAX2"></a>

### Averiguar la primera o la última fecha mostrada por la control Planificador

`<Result> = <Scheduler control>.PositionDateTime([<Sought date>])`
---

**`<Result>: (Cadena de caracteres, o Fecha o DateTime Variable)`**

Fecha visualizada por el control Planificador.

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- la control Planificador de un reporte.




**`<Sought date>: (Constante opcional de tipo Integer)`**

Tipo de fecha solicitada: 


|   |   |
| --- | --- |
| *schCurrentMonth* | Primer día del mes mostrado en el control Planificador. |
| *schEndDate* | Última fecha mostrada en el control Planificador. |
| *schStartDate* <br>(Valor predeterminado) | Primera fecha mostrada en el control Planificador. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Posicionamiento en la control Planificador
<a name="posicionamiento_control_planificador_ELTPARAGRAPHE000235"></a>

Al utilizar la función **&lt;PositionDateTime** para mostrar una fecha, el control Planificador conserva los ajustes de visualización. 

Por ejemplo, si el control Planificador se muestra por semana a partir del Lunes, la función **&lt;PositionDateTime** utilizada en la fecha del viernes muestra la planificación del Lunes al Domingo de la semana que contiene el viernes especificado.




## Funciones de gestión de Pivot controles Tabla (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para manipular el Pivote controles Tabla (sintaxis del prefijo):



|   |   |
| --- | --- |
| [&lt;Pivot tabla&gt;.CollapseAll](../WDLang1/1000022185.md) | Oculta todas las cabeceras de fila y/o de columna en un control Tabla dinámica. |
| [&lt;Pivot tabla&gt;.ExpandAll](../WDLang1/1000022183.md) | Expande todas las cabeceras de fila y/o de columna en un control Tabla dinámica. |
| [&lt;Pivot tabla&gt;.expandir](../WDLang1/1000022182.md) | Desglosa los datos encontrados en una control Tabla dinámica según una cabecera fila o a una cabecera de columna. |
| [&lt;Pivot tabla&gt;.InfoXY](../WDLang1/1000022187.md) | Devuelve los valores de las cabeceras correspondientes a una posición en una control Tabla dinámica en píxeles. |
| [&lt;Pivot tabla&gt;.Save](../WDLang1/1000022189.md) | Guarda el resultado del cálculo realizado en un control Tabla dinámica (en un archivo o en la memoria). |
| [&lt;Pivot tabla&gt;.ToExcel](../WDLang1/1000022193.md) | Crea un fichero Excel con los datos de un control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.AxisXY](../WDLang1/1000022178.md) | Permite: <br><br>- Especifique las cabeceras de fila y de columna visualizadas en un control Tabla dinámica. <br><br>- Descubra las cabeceras de columna y fila mostradas en un control Tabla dinámica.<br><br><br> |
| [&lt;Tabla dinámica&gt;.CalculateAll](../WDLang1/1000022180.md) | Calcula o recalcula completamente un control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.CalculateUpdate](../WDLang1/1000022179.md) | Actualiza algunos de los datos encontrados en una control Tabla dinámica a partir de una fecha determinada. |
| [&lt;Tabla dinámica&gt;.Collapse](../WDLang1/1000022184.md) | Oculta los datos encontrados en una control Tabla dinámica según una cabecera fila o una cabecera de columna.. |
| [&lt;Tabla dinámica&gt;.Filter](../WDLang1/1000022186.md) | Habilita o deshabilita un filtro en una control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.HeaderValue](../WDLang1/1000022192.md) | Devuelve el valor mostrado en una cabecera de control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.ListPositionHeader](../WDLang1/1000022188.md) | Devuelve todos los valores associated con un fila o un encabezado de columna en un control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.Load](../WDLang1/1000022181.md) | Carga el resultado del cálculo realizado en una control Tabla dinámica desde una copia de seguridad (archivo encontrado en el disco o copia de seguridad en un búfer). |
| [&lt;Tabla dinámica&gt;.Select](../WDLang1/1000022190.md) | Le permite conocer la posición de las celdas seleccionadas y seleccionar celdas en un control Tabla dinámica. |
| [&lt;Tabla dinámica&gt;.SelectCount](../WDLang1/1000022191.md) | Devuelve el número de celdas seleccionadas en una control Tabla dinámica. |


## ParallelTaskCancel( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza inmediatamente la ejecución de la tarea actual.: la tarea se marca como *ptsCanceled*.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Ends the execution of the current task
ParallelTaskCancel()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ParallelTaskCancel()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **ParallelTaskCancel** finaliza inmediatamente la ejecución de la tarea en curso.

- **ParallelTaskCancel** sólo puede utilizarse en el código ejecutado por una tarea. La llamada a la función finaliza inmediatamente la ejecución de la medida.

- [ParallelTaskRequestCancellation](../WDLang1/1000021213.md) no puede ser utilizada en el código ejecutado por una tarea.

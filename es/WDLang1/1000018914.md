


## TreeMapDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los elementos de un control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all elements from the TreeMap control
TreeMapDeleteAll(TMAP_Wine)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TreeMapDeleteAll(<TreeMap control>)`
---

**`<TreeMap control>: (Nombre del control)`**

Nombre de la control TreeMap utilizada.
Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control TreeMap al que pertenece el Event actual.

## Componente:

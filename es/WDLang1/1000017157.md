
## RecycleBinClear( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra la papelera de reciclaje del sistema.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RecycleBinClear()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RecycleBinClear()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se vació la papelera de reciclaje,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

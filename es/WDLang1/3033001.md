
## eToCurrency( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un valor expresado en euros a otra moneda.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Convert Euros into Marks
ResConversion = eToCurrency(150, eDEM)
```




<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Convert Euros into Irish punts and manage the rounding difference
// Two decimal places are used for the conversion
ResConversion = eToCurrency(10000, eIEP, 2, ResDiff)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = eToCurrency(<Value to convert> [, <Destination currency> [, <Number of decimals> [, <Rounding difference>]]])`
---

**`<Result>: (Moneda)`**

Valor convertido a la moneda especificada.



**`<Value to convert>: (Moneda)`**

Importe en euros a convertir.



**`<Destination currency>: (Entero o constante (opcional))`**

Identificador de la moneda de destino o associated constante con la moneda de destino. Si no se especifica este parámetro, la función convierte el valor especificado en la moneda cuyo identificador es "1" (el franco francés por Default).

**`<Number of decimals>: (Entero opcional)`**

Número de decimales utilizados para el resultado de la Conversion( 2 por la Default).

**`<Rounding difference>: (Moneda opcional)`**

Variable recuperando la diferencia de redondeo causada por la conversión. Esta diferencia se expresa en la moneda de origen.
Este parámetro es useful para las monedas cuyo tipo de cambio es menor que el Euro (Punt Irlandés). Para las otras monedas, este parámetro es siempre nulo.

## Observaciones




### Intercambio tabla
<a name="intercambio_tabla_ELTPARAGRAPHE000082"></a>

Una ***tabla de cambio*** se almacena en la memoria. Cada moneda es con associated:

- Un número de moneda utilizado para identificar la moneda por programación.

- Una constante utilizada para identificar la moneda por programación.

- Una leyenda (hasta 7 caracteres): esta leyenda se muestra en la control "Moneda + Euro" si la máscara de entrada lo permite (se indica el signo $).

- Un tipo de cambio. Este tipo de cambio corresponde al tipo de cambio de la moneda en relación con el euro.

### Diferencia de redondeo
<a name="diferencia_redondeo_ELTPARAGRAPHE000397"></a>La &lt;Diferencia de redondeo&gt; se calcula mediante la siguiente fórmula:


```txt
Round(<Value to convert>, <Result in source currency>, <Number of decimals>)
```


Por ejemplo:


```wl
Difference, ResConversion are currencies
ResConversion = eToCurrency(50, eIEP, 2, Difference)
```


ResConversión es set a 34.94 y Diferencia es set a 0.01.

La siguiente Line de código permite recuperar el importe inicial:


```wl
InitAmount = eToEuro(ResConversion, eIEP, 2) - Difference
```

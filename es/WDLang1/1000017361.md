


## &lt;Calendar&gt;.Posición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el mes mostrado en un control Calendario.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the calendar month in letters
Info("The month displayed in the Calendar control is: " + ...
	DateToMonthInAlpha(CAL_Calendar1.Position()))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Calendar control>.Position()`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Primer día del mes mostrado. Este parámetro tiene el siguiente formato: "YYYYYMM01". Por ejemplo, si el mes mostrado es noviembre de 2008, &lt;Resultado&gt; será igual a "20081101".

**Atención**: &lt;Resultado&gt; corresponde al primer día del mes mostrado en el Calendar y no al primer día mostrado en el Calendar para el mes seleccionado (para noviembre de 2008, el primer día mostrado es el lunes 27 de octubre de 2008).

**`<Calendar control>: (Nombre del control)`**

Nombre de control Calendario utilizado.

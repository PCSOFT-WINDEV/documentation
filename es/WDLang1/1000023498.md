
## psheetSelection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **psheetSelection** se utiliza para averiguar las características avanzadas de una selección en un control Hoja de cálculo. Las características de esta selección son devueltas por varias propiedades de WLanguage. 

La propiedad [.selección](../Proprietes/1000023033.md) obtiene una variable de tipo **psheetSelection**. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sel is psheetSelection = PSHEET_MySpreadsheet.Selection
IF sel.Type = psheetSelSimpleCell THEN
	// Display the cell name in the status bar
	STATUSBAR = sel.MinCell
END
```



```wl
t is psheetSelection
t = PSHEET_NoName1.Selection
SWITCH t.Type
	CASE psheetSelSimpleCell: Trace("You have selected only one cell.")
	CASE psheetSelSimpleHeader: Trace("You have selected only one header.")
	CASE psheetSelMultiCell: Trace("You have selected several cells.")
	CASE psheetSelMultiHeader: Trace("You have selected several headers.")	
	CASE psheetSelRectCell: Trace("You have selected several cells (rectangle).")
	CASE psheetSelEmpty: Trace("You haven't selected any cells.")
	CASE psheetSelAll: Trace ("You have selected the entire worksheet.")	
	OTHER CASE	
END

te is array of strings = t.Header
tc is array of strings = t.Cell
Trc is array of strings = t.RectCell
Trace("-----------------------------")
Trace("")

Trace("Min cell:", t.MinCell)
Trace("Max cell:", t.MaxCell)
Trace("-----------------------------")
Trace("")

Trace("Browse of headers")
FOR I = 1 TO te.Occurrence
	Trace("header " + i + ":", te[i])
END
Trace("-----------------------------")
Trace("")

Trace("Browse of cells")
FOR I = 1 TO tc.Occurrence
	Trace("cell " + i + ":", tc[i])
END
Trace("-----------------------------")
Trace("")

Trace("Browse of rectangle of cells")
FOR I = 1 TO Trc.Occurrence
	Trace("rect cell " + i + ":", Trc[i])
END
Trace("-----------------------------")
Trace("")
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables de psheetSelection
<a name="propiedades_especificas_descripcion_las_variables_psheetselection_ELTPARAGRAPHE000035"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una selección en un control Hoja de cálculo:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Cabecera | Array de cuerdas | Cabeceras seleccionadas (cabeceras de filas y/o columnas seleccionadas). <br><br>**Esta propiedad es de solo lectura**. |
| Celda | Array de cuerdas | Array de células seleccionadas. Las células se clasifican según el orden de selección. <br><br>**Esta propiedad es de solo lectura**. |
| escribir a máquina | Constante de tipo Integer | Tipo de selección: <br><br>- *psheetSelMultiCell*: Se seleccionan varias celdas. <br><br>- *psheetSelMultiHeader*: Se seleccionan varias cabeceras. <br><br>- *psheetSelRectCell*: Se seleccionan varias celdas y definen un rectángulo. <br><br>- *psheetSelSimpleCell*: Se selecciona una sola celda. <br><br>- *psheetSelSimpleHeader*: Se selecciona una sola cabecera. <br><br>- *psheetSelAll*: Se selecciona toda la hoja de trabajo.<br><br>- *psheetSelEmpty*: Sin selección.<br><br><br>**Esta propiedad es de solo lectura**. |
| MaxCell | Cadena de caracteres | Máxima celda seleccionada, es decir, la celda que se encuentra en la parte inferior derecha del rectángulo que define la selección de las celdas.  <br><br>**Esta propiedad es de solo lectura**. |
| MinCell | Cadena de caracteres | Mínimo de células seleccionadas, lo que significa que la célula que se encuentra en la parte superior izquierda del rectángulo que define la selección de las células.  <br><br>**Esta propiedad es de solo lectura**. |
| RectCell | Array de cuerdas | Array de rectángulos de células seleccionadas. Ejemplo: "A1:B2"<br><br>**Esta propiedad es de solo lectura**. |


## BankHolidayDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra la lista de todos los días festivos. Los días festivos están definidos por la función [BankHolidayAdd](../WDLang1/1000017304.md). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Clears the list of public holidays
BankHolidayDeleteAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`BankHolidayDeleteAll()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función BankHolidayDeleteAll siempre devuelve <u><u><u><u>False</u></u></u></u> después de llamar a la función **[BankHoliday](../WDLang1/1000017303.md)**.

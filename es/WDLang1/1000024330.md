


## &lt;Imagen&gt;.ModifyLightness (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la luminosidad de una imagen en un control Imagen.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the lightness of the image found in IMG_Image1
IMG_Image1.ModifyLightness(50)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Image control>.ModifyLightness(<Variation of lightness> [, <Destination image>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<Variation of lightness>: (Entero)`**

Valor comprendido entre -100 y +100. Este valor se agregará a la luminosidad actual de cada pixel para obtener una nueva luminosidad.

**`<Destination image>: (Cadena de caracteres opcional o variable de tipo Image)`**

Imagen en la que se dibujará la imagen transformada. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).
	




Si no se especifica este parámetro, la transformación se realizará en &lt;Control Imagen&gt;

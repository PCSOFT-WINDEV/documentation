
## MCISize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el tamaño de un archivo multimedia (previamente abierto con [MCIOpen](../WDLang1/3051017.md)). La unidad utilizada es la especificada por [MCIPositionFormat](../WDLang1/3051004.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MCIOpen("Sound.wav", "MySound")
MCIFormatPosition("MySound", MciFormatMS)
...
MCIPlay("MySound")
...
Res_Size = MCISize("MySound")
...
MCIClose("MySound")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCISize(<Media file or alias> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Tamaño del archivo expresado en el formato especificado por [MCIPositionFormat](../WDLang1/3051004.md). 

- Cadena vacía si se ha producido un error. Para get el número de error, utilice [MCIErr](../WDLang1/3051011.md). Para obtener más detalles sobre el error, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
	El estado reporte del comando es devuelto por [MCIRetrieve](../WDLang1/3051022.md).




**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario. Estos parámetros pueden conocerse consultando la documentación de la función Windows MCI denominada "status"..
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.

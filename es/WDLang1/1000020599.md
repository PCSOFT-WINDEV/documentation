


## RadioButtonInsert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una nueva opción en una control Botón de opción.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Inserts an option at position 2 
RadioButtonInsert(RADIO_Options, 2, "Read a book on a tablet")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RadioButtonInsert(<Radio Button control> , <Element index> , <Caption> [, <Sub-caption> [, <Image>]])`
---

**`<Result>: (Integro)`**



- Índice de la opción insertada. 

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Radio Button control>: (Nombre del control)`**

Nombre de control Botón de opción en la que se insertará la opción.

**`<Element index>: (Integro)`**

Índice del elemento a insertar. 

Si este parámetro:

- es mayor que el número de opciones que se encuentran en el botón de radio, la opción se añade después de la última opción en la control Botón de opción. 

- es igual a 1 o 0, el elemento se añade antes de la primera opción que se encuentra en la control Botón de opción.




**`<Caption>: (Cadena de caracteres)`**

Texto de la opción a añadir.

**`<Sub-caption>: (Cadena de caracteres opcional)`**

Subtítulo de la opción a añadir. 
Si no se especifica este parámetro, la nueva opción no tiene subtexto. 
Si se especifica este parámetro, el subtítulo se mostrará incluso si el botón de radio utiliza un modo de visualización sin subtítulo..


**`<Image>: (Cadena de caracteres opcional)`**

Imagen associated con la opción de añadir. 
Si no se especifica este parámetro, la opción añadida no tiene Image.
Si se especifica este parámetro, la Image se mostrará sólo si el botón de radio utiliza un modo de visualización "estilo Vista" (con Image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las características de la opción añadida también pueden modificarse mediante las siguientes propiedades utilizadas en la opción: 

	- [Pie de foto](../Proprietes/2510053.md) para modificar el título. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) [SubCaption](../Proprietes/1000017232.md) modificar el subtítulo. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Image](../Proprietes/2510034.md) para modificar el Image. 




- Al insertar las opciones en un control Botón de opción, el tamaño del control no se modifica. Las opciones añadidas pueden no ser Visible si el control no es lo suficientemente grande. La control puede ampliarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Para añadir una opción después de las opciones existentes, utilice [RadioButtonAdd](../WDLang1/1000020598.md). 

- Al insertar una opción: 

	- el índice de la opción insertada es &lt;índice del elemento&gt;. 

	- si el botón de radio utiliza la numeración de opciones, las opciones que siguen a la opción insertada se renumeran automáticamente. 




- **Utilización de valores devueltos**: Si el botón de radio utiliza los valores devueltos, tiene la posibilidad de definir el valor devuelto por la opción insertada a través de [ReturnedValue](../Proprietes/2510134.md). 
	Por ejemplo: 
	
	```wl
	nIndex is int
	nIndex = RadioButtonInsert(RADIO_Option, 2, "Read a book on a tablet")
	RADIO_Option[nIndex].ReturnedValue = "Tablet"
	// Select the new option
	RADIO_Option = "Tablet"
	```

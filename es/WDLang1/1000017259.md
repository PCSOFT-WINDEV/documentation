


## Funciones de gestión de aplicaciones con "Live update"
			



Las siguientes funciones se utilizan para gestionar "actualizaciones en vivo":



|   |   |
| --- | --- |
| [AppActivateVersion](../WDLang1/1000018846.md) | Activa una de las versiones de la aplicación disponible en el historial de versiones del servidor de reference instalación. |
| [AppChangeParameter](../WDLang1/1000017260.md) | Cambia una información en el mecanismo de actualización de la aplicación automatic. |
| [AppControl](../WDLang1/3013003.md) | Activa (o desactiva) la control remota para la aplicación WINDEV multiusuario actual. Esta es una función avanzada. |
| [AppDeleteVersion](../WDLang1/1000018847.md) | Elimina una versión del historial de versiones encontradas en el servidor reference instalación. |
| [AppDetectUPDAvailable](../WDLang1/1410087705.md) | Desactiva o personaliza la comprobación de actualizaciones disponibles en automatic. |
| [AppInstallUPD](../WDLang1/1000017261.md) | Activa la actualización de automatic de la aplicación actual. |
| [AppInstallVersion](../WDLang1/1000018918.md) | Instala una versión específica de la aplicación. |
| [AppListVersion](../WDLang1/1000018843.md) | Enumera las versiones disponibles en el servidor de reference instalación. |
| [AppParameter](../WDLang1/1000017268.md) | Devuelve información sobre el mecanismo de actualización de la aplicación automatic. |
| [AppUPDAvailable](../WDLang1/1000017293.md) | Le permite saber si una actualización está disponible para la aplicación especificada. |
| [AppVersionInfo](../WDLang1/1000018844.md) | Devuelve información sobre una versión disponible en el servidor de instalación. |

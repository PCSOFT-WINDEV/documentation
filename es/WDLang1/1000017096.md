
## ArrayToCSV( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una array unidimensional o una array bidimensional en un string en formato CSV.

**Observación**: Esta función sólo puede utilizarse en [WLanguage arrays](../Motscles/1514030.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// One-dimensional array
sMyValues is string 
// Initial array
MyArray is array of 3 strings
MyArray[1] = "A"
MyArray[2] = "B"
MyArray[3] = "C"
sMyValues = ArrayToCSV(MyArray)
// sMyValues contains: "A" + CR + "B" + CR + "C"
```


<a name="Example2"></a>



```wl
// Two-dimensional array
sMyValues is string 
// Initial array
MyArray is array of 2 by 3 strings
MyArray[1,1] = "A"; MyArray[1,2] = "B"
MyArray[1,3] = "C"; MyArray[2,1] = "D"
MyArray[2,2] = "E"; MyArray[2,3] = "F"

sMyValues = ArrayToCSV(MyArray)
// sMyValues contains: "A;B;C"+CR+"D;E;F"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArrayToCSV(<WLanguage array> [, <Column separator>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres que contiene los diferentes valores encontrados en la array.

**Para un array unidimensional**, los elementos del array se convierten en cadenas de caracteres y después se agregan a la cadena separados por un CR (Carriage Return).

**Para un array bidimensional**, los elementos del array se convierten en una cadena de caracteres y después se agregan a la cadena. Las filas están separadas por caracteres CR (Carriage Return) y las columnas están separadas por &lt;Column separator&gt;.

**`<WLanguage array>: (Variable de tipo Array)`**

Nombre de array unidimensional o bidimensional que se convertirá en string.. Esta array debe ser creada. 

**`<Column separator>: (Cadena de caracteres opcional)`**

Separador utilizado para separar los valores de diferentes columnas. El separador de columna utilizado por Default es ";".



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El separador utilizado en el &lt;Resultado&gt; para separar los valores encontrados en las diferentes líneas es CR (Carriage Return).

- La estructura arrays soportada es la arrays unidimensional.

- No se soportan los arrays que contienen strings o variantes de UNICODE.

- Para el tipo Duración, el valor se convierte en miles de segundos.

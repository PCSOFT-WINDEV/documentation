


## TreeDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un elemento y sus "hijos" de un control TreeView.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the "Desserts" node and all its
// "children" from the "TREE_TVRecipe" TreeView control
Res = TreeDelete(TREE_TVRecipe, "Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeDelete(<TreeView control> , <Element path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la eliminación correctamente,

- <u><u><u><u>False</u></u></u></u> si el elemento especificado no existe.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Element path>: (Cadena de caracteres)`**

ruta completa del elemento a borrar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).

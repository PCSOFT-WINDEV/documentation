
## CurrentTitle( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el título de la ventana actual (devuelto por [CurrentWin](../WDLang1/3038033.md)) y la barra de título.

**Recordatorio**: Por defecto, el título de la ventana es el definido en el editor.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Replace the title of the current window
CurrentTitle(gBackground(LightRed) + gPen(Black) + "Modifying a customer form")
```
<a name="Example2"></a>

```wl
// Modify the title of the current window
// Adds "in progress"
CurrentTitle(EOT + gBackground(LightRed) + gPen(Black) + " in progress")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificar el título de la ventana

`CurrentTitle(<New window title>)`
---

**`<New window title>: (Cadena de caracteres)`**

Nuevo título para la ventana actual.  


<a name="SYNTAX2"></a>

### Modificando el título y la barra de título de la ventana

`CurrentTitle([<Background color> [, <Text color>, ]] <New window title>)`
---

**`<Background color>: (Entero opcional)`**

Color usado para el Background de la barra de título. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Text color>: (Entero opcional)`**

Color utilizado para el texto mostrado en la barra de título. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<New window title>: (Cadena de caracteres)`**

Nuevo título para la ventana actual. Si se especifica la constante EOT, el nuevo título se añadirá al título existente.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Título de los cuadros de diálogo
<a name="titulo_los_cuadros_dialogo_ELTPARAGRAPHE000110"></a>

Las ventanas abiertas por [Advertencia](../WDLang1/3021009.md), [Confirme](../WDLang1/3021007.md), [coloquio](../WDLang1/3021015.md), [Error](../WDLang1/3021013.md), [ErrorInfo](../WDLang1/3013008.md), [Info](../WDLang1/3021011.md), [OKCancel](../WDLang1/3021004.md) o [YesNo](../WDLang1/3021005.md) tienen el mismo título que la ventana anterior.

Para cambiar el título de la siguiente ventana que se abra, utilice [NextTitle](../WDLang1/3038043.md).
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

### Título de las ventanas recortadas
<a name="titulo_las_ventanas_recortadas_ELTPARAGRAPHE000148"></a>

**CurrentTitle** también puede utilizarse para cambiar el título de las ventanas recortadas. Sin embargo, este título no se muestra en la ventana.

Si se utilizó una plantilla recortada, el título de la ventana se muestra en un control Estático. La función **CurrentTitle**no tiene en cuenta el contenido de este control. Debe ser modificado específicamente. 
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Presentación del título
<a name="presentacion_del_titulo_ELTPARAGRAPHE000163"></a>

Los [funciones de gestión de cadenas gráficas](../WDLang1/3029033.md) (que empiezan por "g") se utilizan para:

- añadir dibujos al título,

- modificar el formato del título (fondo color, texto color, etc.) con [gBackground](../WDLang1/3029037.md) y [gPen](../WDLang1/3029039.md).




Por ejemplo: para cambiar el texto Color y el fondo Color de un título:


```wl
CurrentTitle(EOT + gBackground(RGB(0, 255, 255)) + gPen(RGB(255, 0, 0)) + "Color test")
```


**Caso especial en Windows Vista y 7**: Las ventanas que usan el estilo "Aero sistema" no pueden tener un título gráfico: se ignora la parte gráfica. Si "Aero" no está habilitado, el título del gráfico se mostrará como de costumbre.
<a name="NOTE0_4"></a>




### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000189"></a>El siguiente código: 


```txt
CurrentTitle(<New window title>)
```

equivale a la [Título](../Proprietes/2510090.md) property:


```txt
MyWindow.Title = <New window title
```

### Ventana minimizada
<a name="ventana_minimizada_ELTPARAGRAPHE000207"></a>

Cuando se minimiza una ventana, el título de esta ventana se muestra al lado de la Icon en la barra de tareas.
<a name="NOTE0_6"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Visualización del título
<a name="visualizacion_del_titulo_ELTPARAGRAPHE000216"></a>

Para mostrar un título, la ventana debe incluir un Action Bar control.




## InWebserviceMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la ejecución está en curso en el motor AWWS.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run from a Webservice? 
IF InWebserviceMode() = True THEN
	// Processes to perform
	...
ELSE
	// Processes to perform
	...
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InWebserviceMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecuta el código en el motor AWWS,

- <u><u><u><u>False</u></u></u></u> en los demás casos. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000204"></a>

La función **InWebserviceMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InWebserviceMode** pueden compartirse entre varios productos sin generar errores de compilación.

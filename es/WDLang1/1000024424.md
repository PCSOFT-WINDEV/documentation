
## EventExist (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si se ejecuta un procedimiento específico cuando se envía un evento de Windows a un control o ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Request to receive the click if not done already
WM_LBUTTONDOWN is int = 513
IF NOT EventExist(EDT_MyControl, WM_LBUTTONDOWN) THEN
   Event(MouseClick, "EDT_MyControl", WM_LBUTTONDOWN)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = EventExist(<Element> , <Windows event>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si: 

	- Se llamó a la función [Event](../WDLang1/3015011.md) en el control,

	- Se agregó un evento opcional para el control en el editor de código. 




- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Element>: (Nombre de control o ventana)`**

Nombre del control o ventana con el evento deseado. 

**`<Windows event>: (Entero)`**

Número o nombre del evento de Windows a manipular. Consulte la [lista de eventos de Windows](../AppelDLL/6510001.md). 
Observación: Las constantes de Windows se pueden integrar directamente en el código WLanguage mediante la palabra clave [EXTERN](../Motscles/1511013.md).

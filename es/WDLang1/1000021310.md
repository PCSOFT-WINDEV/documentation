


## Funciones para la gestión de controles de casillas de verificación (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de casillas de verificación mediante la sintaxis de los prefijos:



|   |   |
| --- | --- |
| [&lt;Caja de verificación&gt;.Insertar](../WDLang1/1000021307.md) | Inserta una nueva opción en una control Casilla de verificación. |
| [&lt;Marque la casilla de verificación&gt;.](../WDLang1/1000021291.md) | Añade una nueva opción a una control Casilla de verificación. |
| [.Delete](../WDLang1/1000021309.md) | Elimina una opción de un control Casilla de verificación. |

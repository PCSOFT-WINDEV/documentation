


## NotaCrear (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una Nota Reposicionable control en la ventana de curernt mediante programación. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates a colored Repositionable Note control
NewNote is Control
NewNote <- NoteCreate("Note your remarks here")
NewNote.BackgroundColor = GetColor()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NoteCreate([<Note text> [, <X> [, <Y>]]])`
---

**`<Result>: (Control Variable)`**

El control de la Nota Reposicionable creado puede ser manejado por un [Control](../WDLang1/1000020967.md) Variable. En este caso, los valores deben asignarse con el operador **'<-'**. 

Si se utiliza directamente **NotaCrear**, el control se crea en la ventana y es Visible por Default. 

**`<Note text>: (Cadena de caracteres opcional)`**

Texto a mostrar en la Nota Reposicionable control. 

**`<X>: (Entero opcional)`**

Coordenada X de la Nota Reposicionable control a crear (posición en el eje X) en píxeles. Posición horizontal de la esquina superior izquierda del control Repositionable Note en relación con la esquina superior izquierda del área de cliente de la ventana (lo que significa la ventana sin barra de título, barra de menú y Border). Este parámetro corresponde a la [XInitial](../Proprietes/2510140.md) Property. 

Si no se especifica este parámetro, la coordenada X del control es set a 0. 

Este valor puede modificarse con el programa [X](../Proprietes/2510135.md) Property.

**`<Y>: (Entero opcional)`**

Coordenada Y de la Nota Reposicionable control a crear (posición en el eje Y) en píxeles. Posición vertical de la esquina superior izquierda del control Repositionable Note en relación con la esquina superior izquierda del área de cliente de la ventana (lo que significa la ventana sin barra de título, barra de menú y Border). Este parámetro corresponde a la [YInitial](../Proprietes/2510087.md) Property. 

Si no se especifica este parámetro, la coordenada Y del control es set a 0. 

Este valor puede modificarse con el programa [Y](../Proprietes/2510137.md) Property.

## Observaciones


### Características de la note reposicionable
<a name="caracteristicas_note_reposicionable_ELTPARAGRAPHE000079"></a>

- La creación puede ser cancelada por Ctrl + Z. 

- La note reposicionable creada es una note persistente: si se cierra la ventana y se vuelve a abrir, se restablece la note. 
	Para crear una nota no persistente, utilice la función [ControlCreate](../WDLang1/1000020876.md) con la constante *typRepositionableNote*. 

- Todas las notas reposicionables que se encuentran en la ventana pueden ser guardadas y restauradas por [NoteLoadAll](../WDLang1/1000022397.md) y [NoteSaveAll](../WDLang1/1000022396.md). 





### Equivalencias
<a name="equivalencias_ELTPARAGRAPHE000099"></a>

- Esta función equivale a una nota creada en la ventana por el usuario final a través de AAF (automatic Application Feature). Para obtener más información, consulte [note reposicionable (AAF)](../Editeurs/1000021820.md). 

- Esta función es equivalente a [WinEdCreateNote](../WDLang1/1000022270.md) que se utiliza para crear un note en la ventana actual.

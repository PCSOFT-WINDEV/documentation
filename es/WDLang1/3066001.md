


## Funciones de gestión de red
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar las redes:



|   |   |
| --- | --- |
| [NetworkConnect](../WDLang1/3066002.md) | Asociar una unidad a un directorio de red compartido. |
| [NetworkDirName](../WDLang1/3066005.md) | Identifica el directorio compartido associated con una unidad de red. |
| [NetworkDisconnect](../WDLang1/3066003.md) | Desconecta una unidad de red. |
| [NetworkDomainName](../WDLang1/1000018859.md) | Devuelve el nombre del dominio associated con el equipo. |
| [NetworkUser](../WDLang1/3066004.md) | Identifica el nombre del usuario conectado. |




## &lt;Multimedia&gt;.altura (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la altura (en píxeles) de la imagen de origen del archivo multimedia reproducido en la control Multimedia. Esta función puede utilizarse para los archivos de vídeo, por ejemplo.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Multimedia control>.Height()`
---

**`<Result>: (Integro)`**



- Altura (en píxeles) de la Image del archivo multimedia,

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **&lt;Multimedia&gt;.altura** obtiene la altura de un archivo multimedia. 

- [&lt;Multimedia&gt;.Width](../WDLang1/1000020093.md) obtiene la anchura de un archivo multimedia.

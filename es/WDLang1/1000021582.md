
## EmulatePaneDock (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula la integración de un panel desacoplado en el associated dinámico control Pestaña.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Integrates the "Alias_1" pane in the Tab control at 3rd position
EmulatePaneDock(TAB_MyDynTab, "Alias_1", 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulatePaneDock(<Control name> , <Alias> , <Pane number>)`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de la control Pestaña dinámica a la que se debe fijar el panel. 

**`<Alias>: (Cadena de caracteres)`**

Alias del panel para integrar en la dinámica de associated control Pestaña. 

**`<Pane number>: (Integro)`**

Número correspondiente a la posición deseada del panel en la control Pestaña dinámica después de la integración. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función solo se debe utilizar en el código de las pruebas automatizadas.




## Funciones de gestión de controles Organigrama
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del organigrama: 



|   |   |
| --- | --- |
| [OrgAdd](../WDLang1/1000019729.md) | Añade un nuevo elemento raíz a una control Organigrama. |
| [OrgAddChild](../WDLang1/1000019763.md) | Añade un elemento hijo a una control Organigrama. |
| [OrgDelete](../WDLang1/1000019758.md) | Elimina un elemento y los "hijos" de un control Organigrama. |
| [OrgDeleteAll](../WDLang1/1000019749.md) | Borra todos los elementos de un control Organigrama. |
| [OrgDisplay](../WDLang1/1000019764.md) | Reinicializa el contenido de una control Organigrama a partir de su fuente de datos (archivo de datos HFSQL por ejemplo). |
| [OrgInfoXY](../WDLang1/1000019759.md) | Devuelve la índice del elemento que se encuentra en una posición dada en un control Organigrama. |
| [OrgModify](../WDLang1/1000019921.md) | Modifica un elemento en una control Organigrama. |
| [OrgOccurrence](../WDLang1/1000019971.md) | Devuelve el número de elementos que se encuentran en una control Organigrama. |
| [OrgSaveImage](../WDLang1/1000019972.md) | Crea una Image que representa a toda la control Organigrama. |
| [OrgSelect](../WDLang1/1000019762.md) | Devuelve el índice del elemento seleccionado en una control Organigrama. |
| [OrgSelectMinus](../WDLang1/1000019761.md) | Desmarca un elemento de un control Organigrama. |
| [OrgSelectPlus](../WDLang1/1000019760.md) | Selecciona un elemento en una control Organigrama. |




## DocChangeZOrder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica intuitivamente el orden Z de un elemento de Imagen o de un área de texto en un docx Document.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
f is docFragment <- TT_Document.Selection.Fragment
e is docElement <- f.Element[1]
IF e..Image <> Null THEN
	DocChangeZOrder(e, docZOrderFront)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DocChangeZOrder(<Element> , <Modification>)`
---

**`<Element>: (docElement Variable)`**

Nombre de la variable de tipo [docElement](../WDLang1/1000022484.md) que se utilizará. Debe ser un elemento "Image" o "Área de texto".. De lo contrario, se muestra un error fatal. 

**`<Modification>: (Constante de tipo Integer)`**

Nuevo orden Z:


|   |   |
| --- | --- |
| *docZOrderBackground* | Envía el elemento a la Background. |
| *docZOrderBackward* | Desplaza el elemento al nivel inferior. |
| *docZOrderForward* | Desplaza el elemento al nivel superior. |
| *docZOrderFront* | Trae el elemento al primer plano. |

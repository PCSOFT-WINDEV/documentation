
## AppActivateVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa una de las versiones de la aplicación disponible en el historial de versiones del servidor de reference instalación.

**Observación**: Esta función sólo puede ser utilizada con una red instalación en una red local. No puede ser usado con una instalación por IP.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates the list of available versions
ListVersions is string
ListVersions = AppListVersion()
// Extracts the last version
LastVersion is string 
LastVersion = ExtractString(ListVersions, 1, CR, FromEnd)
// Activate the version
IF AppActivateVersion(LastVersion) = False THEN
	Error("Error while activating the version " + LastVersion +  ...
		CR + ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AppActivateVersion(<Version> [, <Forced version> [, <Executable>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la activación se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).
	Observación: las posibles causas de error son las siguientes: 

	- el historial de versiones no está habilitado. 

	- la versión solicitada no existe. 







**`<Version>: (Cadena de caracteres)`**

Número de versión a activar. 

**`<Forced version>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para forzar la versión que se activará. Todos los equipos se actualizarán con la versión especificada en &lt;Version&gt;, incluyendo los equipos que utilizan una versión más reciente.. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) en caso contrario. 




**`<Executable>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable correspondiente a la aplicación a actualizar. Si no se especifica este parámetro, se actualiza la aplicación actual..

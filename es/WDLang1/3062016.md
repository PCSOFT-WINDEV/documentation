
## MakeInteger( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un entero de 4 bytes a partir de dos enteros de 2 bytes.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Value is int = 5000
Low is int on 2 = LoWord(Value)
High is int on 2 = HiWord(Value)
// Rebuilds the value from these high bytes and low bytes
ValueBuilt is int = MakeInteger(High, Low)
// ValueBuilt is equal to Value
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MakeInteger(<High Byte> , <Low Byte>)`
---

**`<Result>: (Integro)`**

Valor entero (4 bytes) construido a partir de la concatenación de &lt;High Byte&gt; y &lt;Low Byte>.

**`<High Byte>: (2-byte integer)`**

Valor que definirá los 2 bytes altos de &lt;Resultado>.

**`<Low Byte>: (2-byte integer)`**

Valor que definirá los 2 bytes bajos de &lt;Resultado&gt;..

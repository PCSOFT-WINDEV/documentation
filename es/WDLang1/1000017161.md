
## dbgEndRecording( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la grabación del escenario. La grabación fue iniciada por [dbgStartRecording](../WDLang1/1000017160.md).

**Observación**: Esta función debe utilizarse para [crear una prueba automatizada por el usuario final](../Editeurs/2019031.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
dbgEndRecording()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`dbgEndRecording()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**dbgEndRecording** no tiene efecto si no se está registrando ningún escenario.

**Observación**: La grabación del escenario también se puede iniciar a través del archivo ".wx" que se encuentra junto al ejecutable. Para obtener más información, consulte [Escenario creado por el usuario](../Editeurs/2019031.md).

## Componente:
wd280vm.dll

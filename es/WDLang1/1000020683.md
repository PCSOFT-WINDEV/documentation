


## TimelineDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Reinicializa el contenido de una control Línea de tiempo a partir de su fuente de datos (archivo de datos HFSQL por ejemplo). Todos los eventos se borran de control Línea de tiempo y se vuelven a crear desde la fuente de datos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an event to an HFSQL data file
HAdd(File_Event)
// Reinitialize the TimeLine control
TimelineDisplay(TL_Music)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TimelineDisplay(<TimeLine control> [, <Options>])`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de la control Línea de tiempo que se va a utilizar. La control Línea de tiempo debe ser relacionado para una fuente de datos.. En caso contrario (si la control se rellena mediante programación), **TimelineDisplay** no tiene ningún efecto.

**`<Options>: (Constante de tipo Integer)`**

Opción para visualizar la control Línea de tiempo:


|   |   |
| --- | --- |
| *schWithoutExecuteQuery* | Se utiliza para evitar que se repita la consulta relacionado a la control Línea de tiempo (para un control Línea de tiempo relacionado a una consulta). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la fuente de datos relacionado a control Línea de tiempo es una consulta, esta consulta se reinicializa.. 

- La posición en el control Línea de tiempo no se modifica.

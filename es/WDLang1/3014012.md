
## DeleteShortCut( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina un acceso directo previamente creado por [CreateShortcut](../WDLang1/3014008.md) desde el escritorio de Windows, desde el menú "Inicio" de Windows o desde un directorio específico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Creates the "Test" shortcut for the "C:\Test\Test.exe" program
// on the desktop, with Ctrl-Alt-A as shortcut 
CreateShortcut(shortcutDesktop, "Test", "C:\Test\Test.exe", "", swsNormal, "A")
...
DeleteShortcut(shortcutDesktop, "Test")
```
<a name="Example2"></a>

```wl
// Create the shortcut in "Start/Programs/Accessories"
CreateShortcut(shortcutProgram, "Test", "C:\Test\Test.exe", "Accessories", swsNormal, "A")
...
DeleteShortcut(shortcutProgram, "Test")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DeleteShortCut(<Location> , <Name> [, <Path>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el atajo,

- <u><u><u><u>False</u></u></u></u> si se produce un problema.




**`<Location>: (Constante de tipo Integer)`**

Ubicación relativa del acceso directo ruta:


|   |   |
| --- | --- |
| *shortcutDesktop*<br>(Valor predeterminado) | Atajo en el escritorio de Windows.<br>En Windows Vista, la aplicación que elimina el acceso directo debe tener un manifiesto de administrador. |
| *shortcutFree* | Atajo en el directorio especificado por &lt;Path>. |
| *shortcutProgram* | Acceso directo en el menú "Programas de inicio..." de Windows.<br>En Windows Vista (y superior), la aplicación que elimina el acceso directo debe tener un manifiesto de administrador. |
| *shortcutQuickLaunch* | Atajo en la barra de inicio rápido (Windows 98 y posteriores). |
| *shortcutStart* | Atajo en el menú "Inicio" de Windows para todos los usuarios de el equipo. |
| *shortcutStartup* | Atajo en el menú "Inicio .. Programas .. Inicio" de Windows para todos los usuarios de el equipo. |
| *shortcutUserDesktop* | Atajo en el escritorio de Windows para el usuario actual de el equipo. |
| *shortcutUserProgram* | Atajo en el menú "Inicio .. Programas" de Windows para el usuario actual de el equipo. |
| *shortcutUserStart* | Atajo en el menú "Inicio" de Windows para el usuario actual de el equipo. |
| *shortcutUserStartup* | Atajo en el menú "Inicio .. Programas .. Inicio" de Windows para el usuario actual de el equipo. |



**`<Name>: (Cadena de caracteres)`**

Nombre del acceso directo.

**`<Path>: (Cadena de caracteres opcional)`**

Directorio relativo al &lt;Location&gt; de donde se debe eliminar el acceso directo. Si &lt;Ubicación&gt; = *shortcutFree*, la ruta es absoluta.

Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el acceso directo se eliminará del &lt;Localización&gt; especificado.

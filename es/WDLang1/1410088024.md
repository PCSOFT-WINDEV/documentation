
## sysNotificationInput (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sysNotificationInput** se utiliza para definir todas las características avanzadas de un control de entrada, como una entrada de texto o un control de cuadro combinado, en un Notification interactivo . Las características de esta control pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

El tipo **sysNotificationInput** es utilizado por las variables [sysNotification](../WDLang1/1410087893.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a notification that allows the user to enter their last name, a first name and a service
MyNotif is sysNotification
MyNotif.Title = "Register"

MyNotif.Inputs[1].Type = sysNotifInputText
MyNotif.Inputs[1].Name = "User_LastName"
MyNotif.Inputs[1].Caption = "Last name"

MyNotif.Inputs[2].Type = sysNotifInputText
MyNotif.Inputs[2].Name = "User_FirstName"
MyNotif.Inputs[2].Caption = "First name"

MyNotif.Inputs[3].Type = sysNotifInputCombo
MyNotif.Inputs[3].Name = "User_Service"

MyNotif.Inputs[3].ComboItems = ["1" + TAB + "Development", ...
				"2" + TAB + "Direction", ...
				"3" + TAB + "Translation"]
								
MyNotif.Buttons[1].Caption = "Register"
MyNotif.Buttons[1].Note = "btn_register"

MyNotif.Buttons[2].Caption = "Decline"
MyNotif.Buttons[2].Type = sysNotifCloseButton

SysNotificationSend(MyNotif)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables sysNotificationInput
<a name="propiedades_especificas_las_variables_sysnotificationinput_ELTPARAGRAPHE000034"></a>

Las siguientes propiedades pueden utilizarse para manipular una entrada de usuario control:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ComboItems | Array de cadenas de caracteres | Artículos de control Combo Box. Cada cadena de la array debe tener el siguiente formato:<br><br><pre><code>Identifier + TAB + Caption</code></pre><br> donde: <br><br>- Identificador: valor recuperado por la aplicación.<br><br>- Pie de foto: texto del elemento en el cuadro combinado.<br><br><br><br><br>Esta propiedad sólo es necesaria si la propiedad **Type** se establece como *sysNotifInputCombo*. |
| DefaultIndex | Integro | Número de la fila que aparece por defecto en la control Combo Box. <br><br>Esta propiedad sólo es necesaria si la propiedad **Type** se establece como *sysNotifInputCombo*. |
| denominar | Cadena de caracteres | Nombre recuperado por la aplicación para determinar el contenido de la control. |
| escribir a máquina | Constante de tipo Integer | Tipo de la control que permite la interacción del usuario:<br><br>- *sysNotifInputCombo*: Control Combo Box.<br><br>- *sysNotifInputText*: texto control Campo de entrada.<br><br><br> |
| Pie de foto | Cadena de caracteres | Leyenda de control.<br><br>Esta propiedad sólo es necesaria si la propiedad **Type** se establece como *sysNotifInputText*. |

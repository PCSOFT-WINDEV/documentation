


## &lt;Multimedia&gt;.Estado (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el estado del archivo multimedia reproducido en el control Multimedia.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"

MM_Multimedia1.Open(Myfile)
...
IF MM_Multimedia1.Status() = MultimediaStatusPause THEN
	MM_Multimedia1.Beginning()
	MM_Multimedia1.Play()
END
...
MM_Multimedia1.Close()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Multimedia control>.Status()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado del archivo multimedia que se encuentra actualmente en el control Multimedia.


|   |   |
| --- | --- |
| *MultimediaStatusPause* | El archivo está en pausa. |
| *MultimediaStatusPlay* | El archivo se abre y se reproduce actualmente. |
| *MultimediaStatusStop* | El archivo se detiene.<br> |



**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar.

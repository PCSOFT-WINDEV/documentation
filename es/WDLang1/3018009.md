
## TreeCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos que se encuentran en una rama de una control TreeView.

Ejemplo de control TreeView:
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)


Este número se tiene en cuenta:

- todos los elementos "hijo" (desde un nodo hasta todas sus hojas),

- sólo los elementos "hijo" directos (nivel siguiente).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements found in the "Desserts" node 
// in the "TREE_TVRecipe" control
ResNumber = TreeCount(TREE_TVRecipe, "Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeCount(<TreeView control> [, <Source node> [, <Elements taken into account>]])`
---

**`<Result>: (Integro)`**



- Número de elementos encontrados,

- -1 si se produce un error.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Source node>: (Cadena de caracteres opcional)`**

ruta completa del nodo desde el cual se realizará el cálculo. Este nodo se ignora en el recuento. Este nodo puede ser Invisible. 
Si este parámetro no se especifica o si corresponde a la constante *NULL*, se tiene en cuenta todo el control TreeView. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


**`<Elements taken into account>: (Constante opcional)`**

Nivel de elementos a tener en cuenta en el conteo.


|   |   |
| --- | --- |
| *tvAll*<br>(Valor predeterminado) | Todos los "niños" son tenidos en cuenta. |
| *tvOneLevel* | Sólo se tienen en cuenta los "niños" cercanos. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las siguientes sintaxis son equivalentes:


```txt
ResNumber = TreeCount(<TreeView control>)

ResNumber = TreeCount(<TreeView control>, NULL)

ResNumber = <TreeView control>.Count
```

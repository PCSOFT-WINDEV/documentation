
## MatRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee el valor de un elemento de matriz.

**Recordatorio**: Un elemento de matriz se identifica por su fila y por su columna.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Read the element (row 3, column 5)
ResRead = MatRead("MyMatrix", 3, 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatRead(<Matrix> , <Element row> , <Element column>)`
---

**`<Result>: (Real)`**

Valor del elemento leído.

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a leer, definida con [MatCreate](../WDLang1/3050058.md).

**`<Element row>: (Integro)`**

Número de la fila del elemento a leer.

**`<Element column>: (Integro)`**

Número de la columna para el elemento a leer.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000083"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000094"></a>

**Atención**: La función **MatRead** no devuelve ningún código de error. 

Para saber si se han generado errores al leer un elemento, utilice [MatError](../WDLang1/3050044.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

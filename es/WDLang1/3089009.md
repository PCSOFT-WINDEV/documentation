
## SchedulerOpenTaskProperties( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el Property Page de una tarea programada. Por lo tanto, la tarea programada puede ser configurada por el programador de Windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open the property page for MyTask
SchedulerOpenTaskProperties("MyTask")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SchedulerOpenTaskProperties(<Task name>)`
---

**`<Task name>: (Cadena de caracteres)`**

Nombre de la tarea a utilizar.


## SchedulerWorkingHourResource (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite indicar las horas de trabajo específicas por día y por Resource en un control Planificador. Este período de tiempo puede ser especificado:

- para la mañana.

- para la tarde.


Los períodos de tiempo pueden definirse por Resource y por día de la semana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Doctor Moore works from 9:00 to 14:00
SchedulerWorkingHourResource(SCH_MySchedule, "Moore", "0900", "1400")
// Doctor Brown works from 9:00 to 12:00 and from 13:40 to 19:15
SchedulerWorkingHourResource(SCH_MySchedule, "Brown", "0900", "1200", "1330", "1915")
// Doctor Brown works from 9:00 to 12:00 on Wednesdays
SchedulerWorkingHourResource(SCH_MySchedule, "Brown", "0900", "1200", "", "", 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerWorkingHourResource(<Scheduler control> , <Resource name> , <Morning start time> , <Morning end time> [, <Afternoon start time> [, <Afternoon end time> [, <Day of week>]]])`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. 

**`<Resource name>: (Cadena de caracteres)`**

Nombre de Resource a utilizar. 

**`<Morning start time>: (Hora o cadena de caracteres)`**

Hora de inicio de la mañana (en formato "HHMM"). 

**`<Morning end time>: (Hora o cadena de caracteres)`**

Hora de finalización de la mañana (en formato "HHMM"). 

**`<Afternoon start time>: (Hora o Cadena de caracteres opcional)`**

Hora de inicio de la tarde (en formato "HHMM").

**`<Afternoon end time>: (Hora o Cadena de caracteres opcional)`**

Hora de finalización de la tarde (en formato "HHMM"). 

**`<Day of week>: (Entero opcional)`**

Indica el número del día afectado por los horarios especificados.: 

- 1 para el lunes,

- 2 para el martes,

- 3 para el miércoles,

- 4 para el jueves,

- 5 para el viernes,

- 6 para el sábado,

- 7 para el domingo.


Si no se especifica este parámetro o si se pone a 0, la Definition de horas de trabajo se aplica a todos los días de la semana.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las horas de trabajo están resaltadas por un Color diferente. Este Color se elige en las opciones de estilo del control. 

- **SchedulerWorkingHourResource** sustituye las horas de trabajo definidas con las propiedades [WorkingHourStart](../Proprietes/1000018958.md) y [WorkingHourEnd](../Proprietes/1000019056.md) y la función [SchedulerWorkingHour](../WDLang1/1000022404.md).

- [Hora de trabajo de Gantt](../WDLang1/1000021117.md) es la función equivalente para los controles del diagrama de Gantt.

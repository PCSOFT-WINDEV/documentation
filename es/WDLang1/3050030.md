
## StatVarianceP( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la desviación para una serie completa de valores.

**Observación**: **StatVarianceP** asume que los argumentos representan a toda la población. Para calcular la varianza de una muestra de población, utilice [StatVariance](../WDLang1/3050033.md). Con muestras de gran tamaño, la funciones **StatVarianceP** y [StatVariance](../WDLang1/3050033.md) devuelven aproximadamente el mismo valor.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003050030&name=statvariancep_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Calculate the variance of the elements in row 3
ResVarianceP = StatVarianceP("MyMatrix", 3, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = StatVarianceP(<Matrix> [, <Row/Column number> [, <Row/Column>]])`
---

**`<Result>: (Real)`**

Desviación para una serie completa de valores.

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz que contiene los datos, definida por [MatCreate](../WDLang1/3050058.md).

**`<Row/Column number>: (Entero opcional)`**

Número de la fila o columna que contiene los datos (1 de Default).

**`<Row/Column>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (predeterminado) si la serie de valores corresponde a una columna de la matriz,

- <u><u><u><u>False</u></u></u></u> si la serie de valores corresponde a una fila de tabla.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Número de elementos encontrados en una matriz fila o en una columna de matriz
<a name="numero_elementos_encontrados_una_matriz_fila_una_columna_matriz_ELTPARAGRAPHE000103"></a>

El número de elementos que se encuentran en una matriz fila o en una columna de matriz se toma como un reference. Este número de elementos corresponde a:

- el número total de filas de la matriz, devuelto por [MatNbLine](../WDLang1/3050060.md) (si el cálculo se realiza sobre una columna);

- el número total de columnas de la matriz, devuelto por [MatNbColumn](../WDLang1/3050073.md) (si el cálculo se realiza sobre una fila).




Si la serie contiene menos elementos que la matriz, los valores que faltan se rellenan automáticamente con 0. Estos valores se tienen en cuenta durante los cálculos.
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000122"></a>

**Atención**: La función **StatVarianceP** no devuelve ningún código de error. Para saber si se han generado errores al calcular la varianza, utilice [StatError](../WDLang1/3050019.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

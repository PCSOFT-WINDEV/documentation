


## &lt;Multimedia&gt;.Width (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la anchura (en píxeles) de la imagen de origen del archivo multimedia reproducido en la control Multimedia. Esta función puede utilizarse para los archivos de vídeo, por ejemplo.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Multimedia control>.Width()`
---

**`<Result>: (Integro)`**



- Ancho de la Image del archivo multimedia (en píxeles),

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- [&lt;Multimedia&gt;.altura](../WDLang1/1000019999.md) obtiene la altura de un archivo multimedia. 

- **&lt;Multimedia&gt;.Width** obtiene el ancho de un archivo multimedia.

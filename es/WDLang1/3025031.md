
## Objeto padre (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el "padre" de un objeto gráfico: control, columna, ventana, página, ...






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// COL_Column1 is in TABLE_Table1, which is in the WIN_Window2 window,
// which is the child of the WIN_Window1 window
ParentObject(COL_Column1)    // Returns "TABLE_Table1"
ParentObject(TABLE_Table1)      // Returns "WIN_Window2"
ParentObject(WIN_Window2)    // Returns "WIN_Window1"
ParentObject(WIN_Window1)    // Returns ""
```


<a name="Example2"></a>

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ParentObject(<Object name>)`
---

**`<Result>: (Cadena de caracteres)`**



- Si &lt;Nombre del objeto&gt; es un control, &lt;Resultado&gt; corresponde al nombre de la ventana (o página) a la que pertenece el control (aunque el control esté en una pestaña).

- Si &lt;Nombre del objeto&gt; es una columna, &lt;Resultado&gt; corresponde al nombre del control Tabla.

- Si &lt;Nombre de objeto&gt; es una ventana, &lt;Resultado&gt; corresponde al nombre de la ventana padre (o a una cadena vacía ("") si &lt;Nombre de objeto&gt; no tiene ventana padre).

- Si &lt;Nombre de objeto&gt; es una página, &lt;Resultado&gt; corresponde al nombre del conjunto de marcos (o a una cadena vacía ("") si &lt;Nombre de objeto&gt; no tiene conjunto de marcos).




**`<Object name>: (Cadena de caracteres)`**

Nombre del objeto (control, ventana, página, columna) cuyo padre debe ser encontrado. Se produce un error de WLanguage si el objeto es desconocido..

### ParentObject y las ventanas MDI
<a name="parentobject_las_ventanas_mdi_ELTPARAGRAPHE000076"></a>

La función **Objeto padre** utilizada en una ventana abierta desde una ventana MDI hija devuelve el alias de la ventana MDI hija.

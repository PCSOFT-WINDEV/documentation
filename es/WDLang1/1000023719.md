


## &lt;Hoja de cálculo&gt;.load (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga un archivo en formato XLSX en un control Hoja de cálculo que se encuentra en una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load the second worksheet of the XLSX file
PSHEET_Spreadsheet.Load("Workbook1.XLSX", 2)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.Load(<File to load> [, <Worksheet number> [, <Option> [, <Password>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cargó el archivo en el control Hoja de cálculo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo utilizada.

**`<File to load>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo (en formato XLSX) a cargar.
Atención: No se puede cargar un archivo XLSX que contenga más de 2 millones de celdas.

**`<Worksheet number>: (Entero opcional)`**



- Número de la hoja de trabajo a cargar. 

- -1 para usar el comportamiento de Default: 

	- Si control Hoja de cálculo está en modo multihoja, se cargarán todas las hojas de trabajo.

	- Si control Hoja de cálculo no está en modo multihoja, sólo se cargará la primera hoja de trabajo..







**`<Option>: (Constante de tipo Integer)`**

Modo para abrir el archivo XLSX para cargar: 


|   |   |
| --- | --- |
| *psheetCargarMantenerContraseña* | La contraseña del archivo XLSX cargado en la control Hoja de cálculo se guarda en la memoria.. <br><br>Al guardar el archivo ("Guardar como" o [&lt;Hoja de cálculo&gt;.Save](../WDLang1/1000023734.md)), no será necesario volver a especificar la contraseña del archivo.. La contraseña se mantendrá. |
| *psheetLoadDefault*<br>(Valor predeterminado) | El archivo XLSX cargado en la control Hoja de cálculo se abre en modo escritura. <br><br>En este modo, para abrir el archivo ".xlxs" con otra aplicación, debe ser cerrado por [&lt;Hoja de cálculo&gt;.close](../WDLang1/1000023722.md). |
| *psheetLoadUnlinked* | El archivo XLSX cargado en la control Hoja de cálculo se carga sólo en la memoria.. El archivo ".xlxs" puede ser abierto por otra aplicación. <br><br>Para guardar los datos modificados, utilice la función [&lt;Hoja de cálculo&gt;.Save](../WDLang1/1000023734.md) con la constante *psheetOverwriteAllowed*. En este caso, los datos modificados por otra aplicación se sobrescribirán.. |



**`<Password>: (Cadena de caracteres opcional)`**

Contraseña para abrir el archivo XLSX a utilizar (si el archivo está protegido por una contraseña). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El contenido del archivo se carga en la control Hoja de cálculo.

- Se elimina el contenido anterior.

- Se calculan las fórmulas.

- Algunas funciones de Excel no son compatibles.

- El tamaño de los archivos XLSX que se pueden cargar está limitado a 2 millones de celdas.. 

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana.. 

- La función **&lt;Hoja de cálculo&gt;.load** actualiza la propiedad [FilePath](../Proprietes/1000023104.md) del control Hoja de cálculo asociando el nombre del archivo cargado al control Hoja de cálculo.

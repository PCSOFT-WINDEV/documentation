
## MaskZipCode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la máscara de entrada del código postal adaptado a un país determinado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use Spain postal code mask on EDT_ZipCode
EDT_ZipCode.InputMask = MaskZipCode("ES")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MaskZipCode(<Country> [, <Option>])`
---

**`<Result>: (Variable de tipo InputMask)`**

Variable de tipo [InputMask](../WDLang1/1000024886.md) que contiene las características de la máscara de entrada del código postal del país especificado. Este resultado puede asignarse directamente a la propiedad [InputMask](../Proprietes/2510033.md) .  

**`<Country>: (Cadena o variable de tipo Country)`**

País cuya máscara de entrada del código postal debe recuperarse. Este parámetro puede corresponder a: 

- El código ISO alfabético o numérico del país deseado (por ejemplo, el código alfabético de Colombia es "CO").

- Una variable de tipo [Country](../WDLang1/1000024921.md). 


Si el país está vacío o es desconocido, se activa una excepción. 

**`<Option>: (Entero opcional)`**

Modo de gestión para salir del control: 



|   |   |
| --- | --- |
| *mzcFreeExit*<br>(Valor predeterminado) | Permite salir del control incluso si no se ha verificado la máscara de entrada. |
| *mzcLockExit* | No permite salir del control si el valor introducido no es válido (por ejemplo, 5 dígitos para los códigos postales en Francia).  |


Este parámetro solo se tiene en cuenta si se selecciona la opción "Impedir salir del control" en la pestaña "Detalles" de la ventana de descripción del control. Si no se selecciona esta opción, la aplicación no se bloquea cuando el usuario sale del control y no se muestra ningún valor.


## &lt;Array&gt;.Min (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor más bajo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Smallest value?
myarray is array of 5 int
myarray = [1,2,3,4,5]
Trace(myarray.Min())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Calcula el valor más bajo de la array

`<Result> = <WLanguage array>.Min()`
---

**`<Result>: (Cadena de caracteres, entera o real)`**

El valor más pequeño pasado como parámetro.

**`<WLanguage array>: (Array de cadenas de caracteres, enteros o reales)`**

Nombre del Array Variable unidimensional que contiene los valores a comparar.

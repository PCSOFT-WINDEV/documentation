


## &lt;Hoja de cálculo&gt;.posición (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la celda especificada en una control Hoja de cálculo que se encuentra en una ventana.: 

- o bien el área de la Visible de la control Hoja de cálculo se mueve para mostrar la célula. 

- o la celda aparece en la esquina superior izquierda del área Visible de control Hoja de cálculo.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCell is string = PSHEET_SPREEADSHEET1.Seek("CAUTION")
// If the cell is found
IF sCell <> "" THEN
	PSHEET_SPREADSHEET1.Position(sCell, psheetVisible)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.Position(<Cell> , <Type of positioning>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Cell>: (Cadena de caracteres)`**

Nombre de la celda a utilizar. Por ejemplo: "A4".

**`<Type of positioning>: (Integro)`**

Tipo de posicionamiento a realizar: 


|   |   |
| --- | --- |
| *psheetTopLeft* | La visualización de control Hoja de cálculo se modifica para que la celda especificada se sitúe en la esquina superior izquierda del área Visible control. |
| *psheetVisible* | La visualización de control Hoja de cálculo se modifica para que la celda especificada sea visible en el control Hoja de cálculo. La visualización del control no se modifica si la celda especificada ya es Visible en el control Hoja de cálculo. |

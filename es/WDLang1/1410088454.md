
## diagConnector (Tipo de Variable)

*** ***
Inherits from [diagShape (Tipo de variable)](../WDLang1/1410088082.md)
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagConnector** se utiliza para definir todas las características avanzadas de un conector en un Diagram. Las características de este conector pueden definirse y modificarse mediante diferentes propiedades de WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
N is int

// Create a rectangle shape
MyShape1 is diagRectangle
// Set the dimensions, position and background color
MyShape1.X = 100
MyShape1.Y = 100
MyShape1.Width = 50
MyShape1.Height = 50
MyShape1.Background.Color = PastelRed

// Reference the displayed diagram
MyDiagram is Diagram <- DIAGEDT_MyDiagram

// Add the rectangle shape to the diagram
N = Add(MyDiagram.Shape, MyShape1)

// Create an ellipse
MyShape2 is diagOval

// Set the dimensions, position and background color
MyShape2.X = 200
MyShape2.Y = 200
MyShape2.Width = 50
MyShape2.Height = 50
MyShape2.Background.Color = PastelGreen

// Add the ellipse to the diagram
Add(MyDiagram.Shape, MyShape2)

// Creates a connector
MyConnector is diagConnector
Add(MyDiagram.Shape, MyConnector)

// Reference added objects
MyShape1 <- MyDiagram.Shape[ N ]
MyShape2 <- MyDiagram.Shape[ N + 1 ]
MyConnector <- MyDiagram.Shape[ N + 2 ]

MyConnector.Beginning.ConnectionPoint = MyShape1.ConnectionPoint[1]
MyConnector.Beginning.ConnectionPoint.X = 100
MyConnector.Beginning.ConnectionPoint.Y = 50

MyConnector.Ending.ConnectionPoint = MyShape2.ConnectionPoint[1]
MyConnector.Ending.ConnectionPoint.X = 0
MyConnector.Ending.ConnectionPoint.Y = 50
```

## Properties


### Propiedades específicas de las variables de tipo **diagConnector**
<a name="propiedades_especificas_las_variables_tipo_docparampagetitleshort_ELTPARAGRAPHE000026"></a>

Observación: El tipo **diagConnector** hereda del tipo [diagShape](../WDLang1/1410088082.md) . Esto significa que todas las propiedades del tipo [diagShape](../WDLang1/1410088082.md) están disponibles para el tipo **diagConnector**. Estas propiedades aparecen en cursiva en la siguiente lista.

Las siguientes propiedades pueden utilizarse para manipular un conector:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| *ActionAllowed* | [diagActionAllowed](../WDLang1/1410088840.md) o constante | <br><br>- Variable de tipo [diagActionAllowed](../WDLang1/1410088840.md) que permite definir los cambios que el usuario puede realizar en la forma.<br><br>- Constante *diagActionNone*: ninguna acción disponible. <br><br>- Constante *diagActionAll* (Valor predeterminado): todas las acciones están disponibles. <br><br><br> |
| *Altura* | Real | Altura de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. |
| *Ancho* | Real | Ancho de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. |
| *Ángulo* | Real | Ángulo de rotación de la forma (expresado en grados). |
| *Background* | Constante, entero o [Color](../WDLang1/1000019661.md) | Color de fondo de la forma. <br>La propiedad **Color** define el color de fondo de la forma. <br>Esta propiedad puede corresponder a: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br>- a Variable de tipo [Color](../WDLang1/1000019661.md). En este caso, se tiene en cuenta la opacidad especificada en la variable.<br><br><br> |
| *comment* | Cadena de caracteres | Nota sobre la forma. Esta nota no se muestra. |
| *ConnectionPoint* | Array de [diagConnectionPoint](../WDLang1/1410088457.md) | Lista de puntos de conexión de la forma. |
| *Nombre* | Cadena de caracteres | Nombre de la forma. Este nombre se utilizará para la búsqueda, por ejemplo, si la forma está en una biblioteca. |
| Enrutamiento | Constante de tipo Integer | Tipo de recorrido del conector: <br><br>- *routeStraight*: El conector, que va desde el punto inicial al punto final y pasa por los puntos de paso, es una recta Line. <br><br>- *rutaOrtogonal*: El conector, que va desde el punto inicial al punto final y pasa por los puntos de paso, tiene curvas en ángulo recto. <br><br><br> |
| estreno | [diagHead](../WDLang1/1410088470.md) | Extremo de origen del conector. |
| Fin de | [diagHead](../WDLang1/1410088470.md) | Extremo de destino del conector. |
| *Borde* | [diagBorder](../WDLang1/1410088179.md) | Características del borde de la forma. <br>Las propiedades disponibles son: <br><br>- Color: color del borde. <br><br>- Espesor: grosor del borde. <br><br>- escribir a máquina: tipo de borde.<br><br><br> |
| *note* | Cadena de caracteres | Nota asociada a la forma. |
| *Opacidad* | Integro | Porcentaje de opacidad de la forma. Este valor oscila entre invisible (0) y completamente opaco (100). |
| *Sombra* | booleano | <br><br>- <u><u><u>True</u></u></u> si la forma tiene sombra,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br> |
| *Texto* | [diagText](../WDLang1/1410088393.md) | Texto (con formato) dentro de la forma. |
| WayPoint | Array de [diagWaypoint](../WDLang1/1410088548.md) | Puntos de conexión. |
| *X* | Real | Coordenada X de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. Esta coordenada X es relativa al padre de la forma. |
| *Y* | Real | Coordenada Y de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. Esta coordenada Y es relativa al padre de la forma. |

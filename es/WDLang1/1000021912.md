
## TelemetryDisable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva la telemetría. Los datos de telemetría ya no se envían al servidor.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TelemetryDisable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TelemetryDisable()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para desactivar la telemetría en una aplicación, puede: 

- utilizar la función **TelemetryDisable**. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Vuelva a instalar la aplicación y desactive la casilla "Participar en la mejora...".. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) modificar el archivo &lt;Nombre ejecutable&gt;.WX. Este archivo debe contener las siguientes líneas: 
	
	```txt
	[WDANALYTICS]
	COLLECTE=0
	```

	Observación: Esta entrada no se encuentra en el archivo &lt;Nombre del ejecutable&gt;.WX si la telemetría está activada.




## &lt;Lector PDF&gt;.Save (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda el contenido de un control Lector PDF en un archivo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save file 
sFileName is string = "C:\temp\MyFile.pdf"

PDF_MyPDFReader.Save(sFileName)

IF ErrorOccurred THEN
	Error(StringBuild("The %1 file was not saved.", ...
			sFileName), ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <PDF Reader control>.Save([<File path>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<File path>: (Cadena de caracteres opcional)`**

Ruta completa y nombre del archivo "PDF".

**Si se especifica este parámetro,** el archivo se guarda en la ubicación especificada (el archivo existente se sobrescribe si es necesario). 

- Si se guarda con éxito, el archivo "PDF" permanece abierto. 

- Si el documento "PDF" se guarda con una ruta diferente a la original (corresponde a "Guardar como"), el archivo de origen se cierra, el PDF se guarda en un nuevo archivo, y el nuevo "PDF" permanece abierto.  


**Si no se especifica este parámetro**, la ruta que se tendrá en cuenta es la del archivo PDF que se muestra en el control Lector PDF. Si esta ruta no existe, se muestra un error.

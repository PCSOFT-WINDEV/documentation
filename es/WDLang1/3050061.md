
## FinRedemption( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una matriz de amortización para un préstamo de tipo fijo durante un período de tiempo definido.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Amortization matrix for an "In Fine" loan of $25000
// with a fixed interest rate of 5% over 6 years
ResAmortization = FinRedemption("AmortMatrix", finInFine, 25000, 5, 6)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinRedemption(<Matrix name> , <Type of amortization> [, <Capital> [, <Rate> [, <Duration>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha creado la tabla de amortización,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Matrix name>: ()`**

Nombre de la matriz de amortización. Esta matriz se crea automáticamente al calcular la amortización.. Esta matriz contiene 5 columnas. Cada columna corresponde a una información específica:



|   |   |
| --- | --- |
| Columna 1 | año |
| Columna 2 | Saldo a pagar |
| Columna 3 | Ventaja |
| Columna 4 | Amortización |
| Columna 5 | Pago anual |



**`<Type of amortization>: (Constante)`**

Indica el tipo de amortización:



|   |   |
| --- | --- |
| *finConstantAnnualPayment* | ***Pago anual constante:*** Cada año, el prestatario paga la misma cantidad al prestamista, incluyendo el capital y los intereses. |
| *finConstantRedemption* | ***Amortización constante: ***Cada año, el valor de la amortización es fijo; los demás parámetros se calculan en consecuencia. |
| *finInFine* | ***Reembolso de"In Fine":*** Durante toda la duración del préstamo, el prestatario sólo paga los intereses, pagando el capital el último año del préstamo. |



**`<Capital>: (Real o moneda opcional)`**

Total del capital prestado (0 por Default).

**`<Rate>: (Real opcional)`**

Tasa de interés del préstamo, expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set a 5.

**`<Duration>: (Real opcional)`**

Duración del préstamo, expresada en años (1 por Default).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
No se verifica la existencia de la matriz de amortización.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).

Entonces, el contenido de la matriz de amortización puede ser leído por las siguientes funciones:

- [MatRead](../WDLang1/3050074.md): Lee el valor de un elemento de matriz.

- [MatReadColumn](../WDLang1/3050070.md): Lee el valor de todos los elementos que se encuentran en una columna de matriz.

- [MatReadLine](../WDLang1/3050032.md): Lee el valor de todos los elementos que se encuentran en una matriz fila.


## RibbonAdaptSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adapta el tamaño de los mandos que se encuentran en una control Cinta de opciones en función del tamaño de su pie de foto.. 

**Observación**: Esta función es useful para adaptar automáticamente la interfaz al cambiar de idioma.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Nation(nationEnglish)
RibbonAdaptSize(RIBBON_MyMenu)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RibbonAdaptSize(<Ribbon>)`
---

**`<Ribbon>: (Cadena de caracteres)`**

Nombre de la control Cinta de opciones a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El tamaño de los controles y agrupaciones se modifica de acuerdo con los siguientes epígrafes. 

- Los botones de las 3 líneas "grandes" se amplían o se reducen. Las alineaciones se mantienen. 

- Los botones rectangulares grandes se amplían o se reducen. Su tamaño mínimo corresponde a [InitialWidth](../Proprietes/2510067.md).

- Para limitar la evolución del tamaño de los controles, se pueden definir límites con [MinWidth](../Proprietes/2510062.md) y [MaxWidth](../Proprietes/2510068.md).




## ListInfo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera las características de una lista WLanguage: tipos de elementos y número de elementos.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// List whose elements are instances of MyClass
MyList is list of MyClass
// Retrieve the definition of the type of elements
MyDefinition is Definition 
MyDefinition = ListInfo(MyList, tiElementDefinition)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ListInfo(<WLanguage list> , <Information>)`
---

**`<Result>: (Variant)`**

Información solicitada. 

**`<WLanguage list>: (jugarreta)`**

Nombre de la variable de tipo [jugarreta](../WDLang1/1514023.md) que se utilizará.

**`<Information>: (Constante de tipo Integer)`**

Información a obtener:


|   |   |
| --- | --- |
| *tiElementDefinition* | &lt;Resultado&gt; es una variable [Definition](../WDLang1/1000019540.md) y contiene el Definition del tipo de elementos.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *tiElementType* | &lt;Resultado&gt; es un entero Variable y contiene el tipo de los elementos encontrados en la lista. Este resultado es compatible con [TypeVar](../WDLang1/3013056.md). |
| *tiTotalNumber* | &lt;Resultado&gt; es un entero Variable y contiene el número total de elementos de la lista. |

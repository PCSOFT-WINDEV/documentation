
## AutoRunDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la ejecución de automatic de una aplicación WINDEV. La ejecución de automatic fue definida por [AutoRunAdd](../WDLang1/3019002.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- From a WINDEV application
// Automatically run "MyApp" at startup
AutoRunAdd(alStartup, "MyApp", fExeDir() + "\CustomerManagement.exe")
...
// Stop the automatic run of "MyApp"
AutoRunDelete(alStartup, "MyApp")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AutoRunDelete(<Run event> , <Application>)`
---

**`<Run event>: (Constante de tipo Integer)`**

Event que iniciaría la ejecución de la aplicación:


|   |   |
| --- | --- |
| *alStartup* | Ejecutar la aplicación cuando se inicia el PC. |



**`<Application>: (Cadena de caracteres)`**

Nombre de la aplicación, definido con [AutoRunAdd](../WDLang1/3019002.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000164"></a>

**Atención**: La función **AutoRunDelete** no devuelve ningún código de error. Para determinar si esta función ha generado un error, utilice la función [ErrorOccurred](../WDLang1/3087001.md) Variable. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).

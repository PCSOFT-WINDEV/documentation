


## &lt;Tabla&gt;.InputSearch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite la entrada en el área de búsqueda en el título de la columna de la Tabla o Treeview control Tabla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En WINDEV: ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TableSaisieLoupe2.gif)![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=TableSaisieLoupe1.gif)




Esta función tiene el mismo efecto que cuando el usuario hace clic en la búsqueda Icon. Puede iniciar una búsqueda o implementar un filtro en la control.

Para cancelar un filtro iniciado con la función **&lt;Tabla&gt;.InputSearch**, utilice la función [&lt;Table&gt;.DisableFilter](../WDLang1/1000024246.md).

**Observación**: Esta función se ignora si el control Tabla especificado utiliza una barra de desplazamiento no proporcional ("Barra de desplazamiento proporcional" desmarcada en la pestaña "Detalles" de la ventana del control description). Para más detalles sobre la lupa, consulte [Características de las columnas de una control Tabla](../WDChamp/1013333.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switches the COL_Name column to edit mode
COL_NAME.InputSearch()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Column>.InputSearch([<Type of filter or search> [, <Initial value>]])`
---

**`<Column>: (Nombre del control)`**

Nombre de la columna que será cambiada al modo de búsqueda. La función no tiene ningún efecto si no se puede realizar esta operación en esta columna (columna no clave, por ejemplo).

El nombre de la columna puede ser dado directamente o en el siguiente formato: &lt;control Tabla&gt;.&lt;Columna&gt;.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La columna utilizada debe ser una columna "Buscable" (pestaña "General" de la columna description).

**`<Type of filter or search>: (Constante opcional de tipo Integer)`**

Tipo de filtro a aplicar o tipo de búsqueda a realizar:


|   |   |
| --- | --- |
| *filterContains* | Muestra las líneas cuyo valor para la columna especificada "Contiene" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterDifferent* | Muestra las líneas cuyo valor para la columna especificada "difiere" de los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterDoesNotContain* | Muestra las líneas cuyo valor para la columna especificada "no contiene" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterDoesNotEndWith* | Muestra las líneas cuyo valor para la columna especificada "no termina con" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterDoesNotStartWith* | Muestra las líneas cuyo valor para la columna especificada "No comienza con" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterEndsWith* | Muestra las líneas cuyo valor para la columna especificada "termina con" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterEqual* | Muestra las líneas cuyo valor para la columna especificada "es igual" a los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterGreater* | Muestra las líneas cuyo valor para la columna especificada "es mayor" que los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterGreaterOrEqual* | Muestra las líneas cuyo valor para la columna especificada "es mayor o igual que" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterLess* | Muestra las líneas cuyo valor para la columna especificada "es menor" que los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterLessOrEqual* | Muestra las líneas cuyo valor para la columna especificada "es inferior o igual a" los caracteres especificados en &lt;Valor inicial&gt;. |
| *filterStartsWith* | Muestra las líneas cuyo valor para la columna especificada "Comienza con" los caracteres especificados en &lt;Valor inicial&gt;. |
| *searchDefault*<br>(Valor predeterminado) | Clasifica la tabla según la columna especificada y cambia la columna a modo de búsqueda. |
| *searchStartsWith* | Ordena la tabla según la columna especificada, cambia la columna a modo de búsqueda y llena la control Campo de entrada de la columna con los caracteres especificados en &lt;Valor inicial&gt;.. Muestra las líneas a partir de las que el valor de la columna especificada "comienza con" los caracteres especificados en &lt;Valor inicial&gt;. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

**`<Initial value>: (Cadena de caracteres opcional)`**

Valor inicial que se debe especificar en el área de entrada del modo de búsqueda.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Efecto de la función
<a name="efecto_funcion_ELTPARAGRAPHE000298"></a>

Esta función no tendrá ningún efecto si no se puede cambiar al modo de búsqueda.. Por ejemplo, el modo de búsqueda no estará activado:

- Si la columna no fue definida como columna con búsqueda (ventana de description del control).

- Si la columna no es associated con un elemento clave.



<a name="NOTE0_2"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000311"></a>

Esta función está disponible en: 

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

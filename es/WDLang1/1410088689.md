


## <Diagram editor>.SelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona una forma en un editor Diagram control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DIAGEDT_MyDiagram.SelectPlus(1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Diagram Editor>.SelectPlus([<Shape index>])`
---

**`<Diagram Editor>: (Nombre del control)`**

Nombre del editor de Diagram control a utilizar.

**`<Shape index>: (Entero opcional)`**

Índice de la forma a seleccionar (si se seleccionan varios elementos). Si no se especifica este parámetro, selecciona el elemento actual. 
Observación: Si la forma no existe en la Diagram, la función no tiene efecto.

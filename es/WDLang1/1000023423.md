


## &lt;Buscar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un elemento (hoja o nodo) en una control TreeView y devuelve su ruta.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the element whose identifier is "1" in the "TREE_CustomerTreeView" TreeView control
// Remark: all the elements of the TreeView control are taken into account
ResPath = TREE_CustomerTreeView.Find()
```
<a name="Example2"></a>

```wl
// Find the element whose identifier is "1" in the "TREE_CustomerTreeView" TreeView control
// Search from the root of the TreeView control and only on the leaves
ResPath = TREE_CustomerTreeView.Find(1, Null, tvLeaf)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeView control>.Find(<Element to find> [, <Path of source element> [, <Options>]])`
---

**`<Result>: (Cadena de caracteres)`**



- El camino del elemento encontrado. Esta cadena puede contener caracteres específicos de control. Esta cadena se puede utilizar en todas las funciones que gestionan los controles de TreeView. Esta ruta tiene el siguiente formato: 
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```


- Cadena vacía (""") si ningún elemento corresponde a la búsqueda.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Element to find>: (Cadena de caracteres o entero)`**

Identificador del elemento a encontrar. Este identificador debe ser idéntico al utilizado en [&lt;Vista de árbol&gt;.add](../WDLang1/1000023422.md), [&lt;Insertar](../WDLang1/1000023761.md) o [&lt;Modificar](../WDLang1/1000023763.md).

**`<Path of source element>: (Cadena de caracteres opcional)`**

Vía de acceso del elemento fuente para la búsqueda en la control TreeView. Si no se especifica este parámetro o si corresponde a la constante *Null*, la búsqueda se realiza desde la raíz de control TreeView.

**`<Options>: (Constante o combinación de constantes opcional)`**

Especifica las características de búsqueda:


|   |   |
| --- | --- |
| *tvLeaf* | Búsqueda realizada sobre las hojas de control TreeView. |
| *tvNode* | Búsqueda realizada en los nodos de control TreeView. |
| *tvOneLevel* | Búsqueda realizada en un único nivel de estructura de árbol (se ignoran los elementos que se encuentran debajo de la rama fuente). La búsqueda se realiza en los elementos hijo del elemento fuente. |


Por defecto, la búsqueda se realiza con *tvLeaf* + *tvNode* (se exploran todos los elementos del control TreeView).

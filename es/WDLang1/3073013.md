
## SysShutdown( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene el sistema o cierra la sesión actual de Windows.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops the system and turns the power off
SysShutdown(sysPoweroff)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysShutdown(<Shutdown mode>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si no se ha encontrado ningún problema,

- <u><u><u><u>False</u></u></u></u> en caso contrario. [ErrorInfo](../WDLang1/3013008.md) devuelve más detalles.




El resultado se envía inmediatamente. Entonces, el cierre de Process se realiza en la tarea de Background.

**`<Shutdown mode>: (Constante)`**

Indica el modo de apagado solicitado:


|   |   |
| --- | --- |
| *sysLockSession* | Bloquea la sesión actual. |
| *sysLogoff* | Termina la sesión actual |
| *sysPoweroff* | Detiene el sistema y apaga la energía. |
| *sysReboot* | Detiene y reinicia el sistema. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### No se permite el apagado por software
<a name="permite_apagado_por_software_ELTPARAGRAPHE000065"></a>

Si el sistema no admite el apagado por software, **SysShutdown** detiene Windows y muestra el siguiente mensaje en el equipo: "Ahora puedes apagar tu ordenador".

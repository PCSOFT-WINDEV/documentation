


## Propiedades relacionado a la dirección del Euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes propiedades se utilizan para gestionar el Euro en:

- Los controles de edición "Moneda + Euro" y las columnas tabla se encuentran en las ventanas y páginas.

- La partida "Moneda + Euro" controla y calcula los controles que se encuentran en los informes.





|   |   |
| --- | --- |
| [DisplayCurrency](../Proprietes/2510030.md) | Permite conocer y modificar la divisa utilizada al mostrar e introducir un valor en un control Campo de entrada de tipo "Monetario + Euro" |
| [MemoryCurrency](../Proprietes/2510013.md) | Permite conocer y modificar la divisa utilizada al manipular el control Campo de entrada de tipo "Monetario + Euro" mediante programación |

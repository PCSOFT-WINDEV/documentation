


## HTMLOnErrorJS (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica la Procedure a la que hay que llamar si se produce un error durante la ejecución del código JavaScript en una pantalla HTML control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLOnErrorJS(HTM_Display, HTMLOnErrorJS_Callback)
INTERNAL PROCEDURE HTMLOnErrorJS_Callback()
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLOnErrorJS(<HTML Display control> , <WLanguage procedure>)`
---

**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML control a utilizar. 

**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback") llamado cuando se encuentra un error en el código JavaScript de la página. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name>(sError is string)
```
donde &lt;sError&gt; es una cadena de caracteres que corresponde al error de JavaScript encontrado.
En el código de esta Procedure, se puede utilizar [ErrorInfo](../WDLang1/3013008.md) para identificar el error encontrado.

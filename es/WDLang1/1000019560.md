
## description variable (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Variable description** se utiliza para get información sobre una clase o miembro de la estructura.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the definition of type class or structure of variable x
MyDefinition is Definition
MyDefinition = GetDefinition(x)
// Retrieves the description of nValue member of class or structure
m is Variable Description 
m = MyDefinition.Variable["nValue"]
```

## Observaciones


### Propiedades específicas de la variable description Variable s
<a name="propiedades_especificas_variable_description_variable_ELTPARAGRAPHE000046"></a>

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |

| AttributeMapping | Cadena de caracteres | <br><br>- Valor del atributo &lt;Mapping&gt; extension para una clase o estructura,<br><br>- "" (cadena vacía) si no se especifica el atributo (o para otros tipos). <br><br><br>**Esta propiedad es de solo lectura**. |
| AttributeNote | Cadena de caracteres | <br><br>- Valor del atributo &lt;Note&gt; extension para una clase o estructura, o para sus miembros.<br><br>- "" (cadena vacía) si no se especifica el atributo (o para otros tipos). <br><br><br><br><br>**Esta propiedad es de solo lectura**. |

### Operación
<a name="operacion_ELTPARAGRAPHE000205"></a>

El miembro descrito puede ser manejado a través de la indirección:: 

- {&lt;objeto&gt;, &lt;Variable description&gt;}

- {&lt;objeto&gt;, &lt;Variable description&gt;} = ...

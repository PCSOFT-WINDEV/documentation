


## Length (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve:

- la longitud de una cadena, es decir, el número de caracteres de la cadena (incluyendo espacios y ceros binarios).

- el tamaño de un buffer, es decir, el número de bytes en el buffer.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Length("Quasimodo")      // Returns 9
Length(" Quasimodo ")    // Returns 11
Length("A" + Charact(0))  // Returns 2
Length("")               // Returns 0
Length(" ")              // Returns 1
```

## Sintaxis

### Obtener la longitud de una cuerda

`<Result> = Length(<Initial string>)`
---

**`<Result>: (Entero)`**

Número de **caracteres** en la cadena.

**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres cuyo tamaño será calculado.  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Obtener el tamaño de un buffer

`<Result> = Length(<Initial buffer>)`
---

**`<Result>: (Entero)`**

Número de **bytes** en el buffer.

**`<Initial buffer>: (Buffer)`**

Buffer cuyo tamaño será calculado. 
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) En UNICODE, cada carácter ocupa cuatro bytes de memoria. Por lo tanto, el tamaño de un buffer que contiene una cadena UNICODE corresponde a cuatro veces el número de caracteres que realmente se encuentran en la cadena en Windows.

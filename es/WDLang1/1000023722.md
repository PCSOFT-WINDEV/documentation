


## &lt;Hoja de cálculo&gt;.close (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el XLSX Document que fue abierto por [&lt;Hoja de cálculo&gt;.load](../WDLang1/1000023719.md).
<a name="sample_code"></a>

## Ejemplo


```wl
PSHEET_Spreadsheet.Load("doc1.xlsx")
PSHEET_Spreadsheet.Close()
ShellExecute("doc1.xlsx")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.Close()`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Por Default, [&lt;Hoja de cálculo&gt;.load](../WDLang1/1000023719.md) deja el archivo XLSX abierto en modo de escritura. Cuando se guarda el Document, sólo se sobrescriben las celdas que han sido cambiadas en el control Hoja de cálculo.. 

- Para abrir el archivo ".xlxs" con otra aplicación, hay que cerrarlo con **&lt;Hoja de cálculo&gt;.close**.

- Si no se abrió ningún archivo ".xlsx", la función **&lt;Hoja de cálculo&gt;.close** no realiza ninguna acción.

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

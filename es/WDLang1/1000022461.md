


## Documento (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Document** permite manipular un documento de tipo "docx". El contenido de esta Document puede ser visto y modificado por varias funciones y propiedades de WLanguage. 

A **Document** Variable se puede inicializar: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) con el contenido de una control Procesador de texto. 

- con un archivo ".docx" o un archivo ".rtf. 

- con un buffer (que contiene un docx o rtf).

- con un memo binario de un archivo de datos. 

- con un string. Por ejemplo: 
	```wl
	MyDoc is document = "a paragraph." + CR + "a second paragraph."
	```





**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000023102&name=documento_tipo_variable&product=WD">Ver ejemplos adicionales</a>

```wl
// Load the document
MyDoc is Document = "c:\temp\doc.docx"
// Browse the paragraphs of the document
FOR EACH para OF MyDoc.Paragraph
	FOR EACH MyElement OF para.Element
		Trace(MyElement.Texte)
	END
END
```

### Propiedades específicas de las variables Document
<a name="propiedades_especificas_las_variables_document_ELTPARAGRAPHE000039"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **Document**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Contraseña | Cadena de caracteres | Contraseña para el archivo docx. Esta contraseña se utiliza para encriptar el archivo docx. Esta contraseña se solicitará al abrir el archivo docx. |
| Estilo | Array de variables de tipo [docStyle](../WDLang1/1000022486.md) | Información sobre los estilos relacionado al Document (información global al Document). |
| EvenOddPageDiferente | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el documento debe tener encabezados y pies de página diferentes para las páginas pares e impares,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. En este caso, los encabezados y pies de página son idénticos para las páginas pares e impares.. <br><br><br> |
| FilePath | Cadena de caracteres | Nombre y ruta completo del archivo docx asociado con el Document. <br>Esta propiedad es de solo lectura. Se asigna cuando se utiliza [DocOpen](../WDLang1/1000022622.md). <br><br>Observaciones: <br><br>- Cuando se asigna una variable **Document** a otra, esta Property no se copia (excepto durante la llamada a [DocOpen](../WDLang1/1000022622.md)).<br><br>- Si el documento proviene de un buffer, este Property corresponde a una cadena vacía (""). <br><br><br> |
| Marcador[ Nombre del marcador] | array asociativa de variables [DocFragment](../WDLang1/1000022482.md) | Fragmento associated con el marcador. |
| Modified | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si se ha modificado el documento desde que se abrió (mediante programación o por el usuario),<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>Esta propiedad es de solo lectura. |
| Numeración | Array de [docNumbering](../WDLang1/1000022794.md) | Las numeraciones asociadas a la Document. La numeración de Document puede ser consultada a través de la declaración FOR EACH. |
| PáginaColor | Variable de tipo [Color](../WDLang1/1000019661.md) | Antecedentes color del Page. |
| Párrafo | Array de [docParagraph](../WDLang1/1000022483.md) | Párrafos incluidos en el cuerpo de Document. Los párrafos de Document pueden ser consultados a través de la declaración de CADA UNO. |
| Property | Variable de tipo [docProperty](../WDLang1/1000022945.md) | Propiedades del documento. |
| ReadOnly | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si el documento es de solo lectura,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br><br><br>Esta propiedad es de solo lectura. |
| Sección | Variable de tipo [docSection](../WDLang1/1000022685.md) | Información de la sección (información global a la Document). |


## TelemetryIsEnabled( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite saber si la telemetría está habilitada.. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
If TelemetryIsEnabled() = False THEN TelemetryEnable()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TelemetryIsEnabled()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la telemetría está activada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

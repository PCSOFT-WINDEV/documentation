


## WLanguage Procedure llamado por SelectColor



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lenguaje WL del navegador Procedure ("callback") ejecutado por [SelectColor](../WDLang1/3013045.md) cuando se cierra la ventana de selección Color. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo




<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SelectColor_Callback(<Color>)`
---

**`<Color>: (Color Variable)`**

[Color](../WDLang1/1000019661.md) variable que corresponde al color seleccionado en la ventana de selección de color. 



<a name="XComponent"></a>

## Componente:
WDJS.DLL

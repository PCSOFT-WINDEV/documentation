


## fLoadText( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga el contenido de un archivo de texto en cualquier control de texto o Variable (cadena Variable, control Campo de entrada en una ventana, control Estático en una reporte, etc.). [fSaveBuffer](../WDLang1/1000019412.md) se utiliza para extraer este archivo del buffer.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Load the content of "C:\Temp\MyFile.txt" into
// the "EDT_Edit1" edit control
EDT_Edit1 = fLoadText("C:\Temp\MyFile.txt")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fLoadText(<Name and path of text file> [, <Load mode>])`
---

**`<Result>: (Cadena de caracteres)`**



- Contenido del archivo de texto,

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Name and path of text file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo de texto. Se puede utilizar una ruta UNC. Este archivo no es necesariamente un archivo en formato TXT: de hecho, los ceros binarios están permitidos.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Load mode>: (Constante de tipo Integer)`**

Modo de carga del archivo: 


|   |   |
| --- | --- |
| *foAnsi* | Archivo Ansi. En este caso, &lt;Result&gt; corresponderá a una cadena de caracteres Ansi. |
| *foUnicode* | Archivo Unicode. En este caso, &lt;Result&gt; corresponderá a una cadena de caracteres Unicode. |

### Visualización del contenido de un archivo en formato UNICODE
<a name="visualizacion_del_contenido_archivo_formato_unicode_ELTPARAGRAPHE000120"></a>

Para ver el contenido de un archivo en formato UNICODE, debe realizar la siguiente Process: 

```wl
// Load the content of "C:\MyFiles\MyFile.XML" into MyString 
MyBuffer is Buffer 
MyBuffer = fLoadText("C:\MyFiles\MyFile.XML")
MyString is UNICODE string 
MyString = MyBuffer
Info(MyString)
```


## UnitedStates.CheckEIN (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número EIN (número de identificación del empleador en Estados Unidos).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyEIN is string = "322124444"
UnitedStates.CheckEIN(MyEIN)	// Returns True

// Check a control
EDT_EIN = "ABC"
UnitedStates.CheckEIN(EDT_EIN)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = UnitedStates.CheckEIN(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número EIN válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número EIN a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

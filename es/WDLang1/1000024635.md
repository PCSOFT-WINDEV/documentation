


## PicResize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Redimensiona una imagen en formato WDPic. Todas las capas de la imagen se redimensionan. 

Observación: Para redimensionar una sola capa, utilice la función [dResize](../WDLang1/3029011.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a source image
MyWDPicImage is WDPic
MyWDPicImage.FileName = fExeDir() + ["\"] + "Source.png"

// Resize the image
PicResize(MyWDPicImage, 100, 100, drHomothetic)

// Save the result
PicSave(MyWDPicImage, fExeDir() + ["\"] + "Result.png")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PicResize(<WDPic image> , <New width> , <New height> [, <Option>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el redimensionamiento se realizó correctamente,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que contiene la imagen a manipular. 

**`<New width>: (Entero)`**

Nuevo ancho de la imagen expresado en pixeles.

**`<New height>: (Entero)`**

Nuevo alto de la imagen expresado en pixeles.

**`<Option>: (Constante opcional de tipo Integer)`**

Opciones de redimensionamiento: 


|   |   |
| --- | --- |
| *drHighQuality* | Esta constante puede combinarse con las opciones de redimensionamiento homotético.<br><br>En modo homotético, esta constante permite mejorar la calidad de la imagen cuando esta se reduce con respecto a su tamaño inicial |
| *drHomothetic* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse. Esta ocupa:<br><br>- todo el alto especificado.<br><br>- todo el ancho especificado.<br><br><br> |
| *drHomotheticCentered* | La imagen se amplía o reduce para ocupar de forma óptima el espacio definido sin distorsionarse y se centra. Esta ocupa:<br><br>- todo el alto especificado: la imagen se centra horizontalmente.<br><br>- todo el ancho especificado: la imagen se centra verticalmente.<br><br><br> |
| *drHomotheticExtended* | La imagen se amplía o reduce para que el lado más pequeño de la imagen se muestre completamente. |
| *drStretched*<br>(Valor predeterminado) | La imagen se amplía si es necesario para ajustarse al nuevo ancho y alto especificados. |
| *drTiled* | La imagen se repite para ocupar el nuevo ancho y alto especificados. |

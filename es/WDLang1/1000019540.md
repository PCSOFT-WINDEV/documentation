
## Definition (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Definition**permite obtener las características de un tipo: 

- tipo simple,

- tipo avanzado,

- class, 

- estructura,

- combinación,

- enumeración.




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the definition of x type
d is Definition
d = GetDefinition(x)
// Displays the name of x type
Trace(d.Name)
```

## Observaciones


### Propiedades específicas de las variables Definition
<a name="propiedades_especificas_las_variables_definition_ELTPARAGRAPHE000057"></a>

Las siguientes propiedades pueden ser usadas para manipular a Definition Variable:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| AsociaciónDefinición | Array de variables de tipo [Definition](../WDLang1/1000019540.md) | Definition de los tipos associated. <br><br>En este caso, las variables **Associated** Property de [Property Description](../WDLang1/1000019324.md), [Variable Description](../WDLang1/1000019560.md) y [Procedure Description](../WDLang1/1000019317.md) devuelven <u><u><u><u>True</u></u></u></u>. |
| AttributeMapping | Cadena de caracteres | <br><br>- Valor del atributo &lt;Mapping&gt; extension para una clase o estructura, <br><br>- "" (cadena vacía) si no se especifica el atributo (o para otros tipos).<br><br><br>**Esta propiedad es de solo lectura**. |
| AttributeNote | Cadena de caracteres | Contenido del atributo &lt;Note&gt; extension.<br><br>**Esta propiedad es de solo lectura**. |
| AttributeSerialize | Cadena de caracteres | Nombre de serialización del tipo según su nombre y el atributo &lt;serialize&gt;.: <br><br>- Nombre de la clase, si no se especifica el atributo,<br><br>- Contenido de la cadena si el atributo contiene una cadena,<br><br>- "" (cadena vacía) si el atributo es set a False o si el tipo no es una clase.<br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| BaseDefinición | Array de variables de tipo [Definition](../WDLang1/1000019540.md) | Definition de tipos base. |
| DeclaraciónSintaxis | [Sintaxis description](../WDLang1/1410089154.md) | description de la declaración de variables globales para ventanas, páginas e informes. <br><br>**Esta propiedad es de solo lectura**. |
| DefiniciónFamilia | Integro | Definition de los tipos de base. <br>Este Property es accesible en sólo lectura. |
| denominar | Cadena de caracteres | Nombre del tipo. <br>Este Property es accesible en sólo lectura. |
| escribir a máquina | Integro | Tipo WLanguage. <br>Esta Property es accesible en modo sólo lectura. Los valores disponibles son idénticos a los devueltos por [TypeVar](../WDLang1/3013056.md). |
| Opción | Array de [Option Description](../WDLang1/1000023072.md) | description de opciones para un Definition de combinación.<br><br>Observación: Información disponible para las combinaciones definidas en la editor de análisis y para las combinaciones definidas en WLanguage. |
| Procedure | Array de [Procedure Description](../WDLang1/1000019317.md) | Métodos de clase si el tipo definido es una clase.<br><br>El **Asociado** Property de las variables [Procedure Description](../WDLang1/1000019317.md) devuelve <u><u><u><u>False</u></u></u></u> en el caso de los métodos de la clase base. |
| Property | Array de [Property Description](../WDLang1/1000019324.md) | Propiedades de la clase si el tipo definido es una clase.<br><br>El **Asociado** Property de las variables [Property Description](../WDLang1/1000019324.md) devuelve <u><u><u><u>False</u></u></u></u> en el caso de las propiedades de la clase base. |
| Valor | Array de [Value description](../WDLang1/1000023061.md) | description de valores para una Definition de enumeración.<br><br>Observación: Información disponible para las enumeraciones definidas en la editor de análisis y para las enumeraciones definidas en WLanguage.  |
| Variable | Array de [Variable Description](../WDLang1/1000019560.md) | Miembros de clase o estructura si el tipo definido es una clase o una estructura.<br><br>El **Asociado** Property de las variables [Variable Description](../WDLang1/1000019560.md) devuelve <u><u><u><u>False</u></u></u></u> en el caso de los miembros de la clase base. |


## dClear (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vacía el contenido de una variable de tipo Image o control Imagen y lo rellena con el color especificado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
dStartDrawing(IMG_Drawing)

// Draw a rectangle with a light yellow background
// and light blue borders
dRectangle(10, 10, 50, 50, LightYellow, LightBlue)

// Clear the drawing and apply blue
dClear(LightBlue)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Vaciar el contenido de la imagen actual y rellenarla con el color especificado

`dClear([<Color>])`
---

**`<Color>: (Constante o Variable de de tipo Color opcional)`**

Color utilizado para rellenar el elemento. Este color puede corresponder a: 

- una de las constantes WLanguage de color,

- una variable de tipo [Color](../WDLang1/1000019661.md). 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).


Si no se especifica este parámetro, la imagen se restablecerá a su estado inicial: 

- transparente para una variable de tipo [Image](../WDLang1/1000019650.md),

- color de fondo de la ventana para un control Imagen. 





<a name="SYNTAX2"></a>

### Vaciar el contenido de la variable de tipo Image o control Imagen y rellenarla con el color especificado

`dClear(<Image> [, <Color>])`
---

**`<Image>: (Nombre de control o variable de tipo Image)`**

Nombre del control Imagen o de la variable de tipo [Image](../WDLang1/1000019650.md) a manipular. 

**`<Color>: (Constante o Variable de de tipo Color opcional)`**

Color utilizado para rellenar el elemento. Este color puede corresponder a: 

- una de las constantes WLanguage de color,

- una variable de tipo [Color](../WDLang1/1000019661.md). 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color predefinido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).


Si no se especifica este parámetro, la imagen se restablecerá a su estado inicial: 

- transparente para una variable de tipo [Image](../WDLang1/1000019650.md),

- color de fondo de la ventana para un control Imagen.

## Observaciones


### Opacidad
<a name="opacidad_ELTPARAGRAPHE000095"></a>

- Si se activa el modo de dibujo en el control sin un valor de opacidad (función dStartDrawing sin la constante *dWithOpacity*): 

	- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) En Windows, cuando se utiliza la función **dClear** con un color, no se tiene en cuenta la opacidad. 

	- ![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) En Linux, solo se pueden utilizar colores con opacidad máxima.




- Si se activa el modo de dibujo en el control con un valor de opacidad (función dStartDrawing con la constante *dWithOpacity*), esta se tiene en cuenta cuando se utiliza la función **dClear** con un color.

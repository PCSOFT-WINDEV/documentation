


## DeleteColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una columna en una posición determinada en la propiedad array de un tipo avanzado (array de eventos de gglCalendar, etc.).

El tamaño del array se reduce automáticamente.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`DeleteColumn(<WLanguage array> , <Column index>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser bidimensional.

**`<Column index>: (Integro)`**

Índice de la columna que se eliminará. Si este parámetro es mayor que el número de columnas en el array, se produce un error WLanguage.


## SchedulerAddTrigger( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un programa para una tarea programada. Un programa es uno de los eventos que desencadenarán la ejecución de una tarea. Antes de crear la programación, todas las configuraciones de programación deben definirse en la estructura [LanzarEstructura de tareas programadas](../WDLang1/3089002.md)


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Daily task, performed every day at 10:00 AM
// from today
TriggerScheduledTask.Type = schedDaily
TriggerScheduledTask.Time = 1000
TriggerScheduledTask.BeginDate = Today()
SchedulerAddTrigger("MyTask")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SchedulerAddTrigger(<Task name>)`
---

**`<Result>: (Integro)`**



- Número del programa creado. Este identificador puede ser utilizado por las funciones de gestión de horarios.

- 0 si se produce un error. 




**`<Task name>: (Cadena de caracteres)`**

Nombre de la tarea a utilizar. Esta tarea fue creada por [SchedulerAddTask](../WDLang1/3089003.md).

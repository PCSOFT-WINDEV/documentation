
## MarkdownToHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena Markwdown en un buffer HTML (codificado en UTF-8). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyBuffer is buffer
MyInitialString is string = "*Italic text*"

MyBuffer = MarkdownToHTML(MyInitialString)
Trace(MyBuffer)
// Display <em>Text in italics</em>
```



```wl
// Save to an HTML file
HTML_File is ANSI string = [
<html>
    <head><meta charset="UTF-8"> </head>
<body>	
	]
HTML_File += CR + MarkdownToHTML(STC_MarkdownText) + "</body></html>"
fSaveText("C:\temp\myFile.html", HTML_File)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MarkdownToHTML(<Markdown string> , <CSS style>)`
---

**`<Result>: (Buffer)`**

Buffer HTML (codificado en UTF-8). 

**`<Markdown string>: (Cadena de caracteres)`**

Cadena Markdown. 

**`<CSS style>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u>: el valor devuelto contendrá el estilo necesario para que el texto se muestre correctamente (con las reglas CSS que se aplicarán a las etiquetas HTML).  

- <u><u><u><u>False</u></u></u></u> (valor predeterminado): el valor devuelto no contendrá estilos CSS.

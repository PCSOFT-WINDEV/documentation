
## fSizeUncompressed( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño de un archivo antes de la compresión.

Cuando un archivo es descomprimido por [fUncompress](../WDLang1/3036021.md), le recomendamos que compare el tamaño no comprimido del archivo con el espacio disponible en el disco.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036054&name=fsizeuncompressed_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Size of file before compression
ResDecompressedSize = fSizeUncompressed("C:\MyDirectories\File.doc")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fSizeUncompressed(<File path>)`
---

**`<Result>: (Integro)`**



- Tamaño del archivo antes de la compresión (en bytes),

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

- -2 si el archivo no fue comprimido por [fCompress](../WDLang1/3036016.md) o si el archivo fue comprimido por WINDEV versión 4 (4.0 o 4.1).




**`<File path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo comprimido (hasta 260 caracteres). Se puede utilizar una ruta UNC. Este archivo debe haber sido comprimido por [fCompress](../WDLang1/3036016.md). Los caracteres comodín (\* y?) no están permitidos.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

## Observaciones




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000070"></a>

La función **fSizeUncompressed** genera un error en los siguientes casos:

- el archivo especificado no existe,

- el nombre del archivo especificado no es válido,

- el archivo ya está bloqueado por otra aplicación.

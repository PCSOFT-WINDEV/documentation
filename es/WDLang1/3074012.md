


## TableEnumColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de una columna que se encuentra en una tabla o en una tabla de Treeview control Tabla.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of third column found in the "TABLE_ProductTable" control
ResColumnName = TableEnumColumn(TABLE_ProductTable, 3)
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Handle the column via a Control variable
pCol is Control
pCol <- TableEnumColumn(TABLE_ProductTable, 3)
```
<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Enumerate all columns found in the Table control "TABLE_ProductTable"
AColumn is Control
FOR nColumn = 1 _TO_ TableCount(TABLE_ProductTable, toColumn)
	AColumn <- TableEnumColumn(TABLE_ProductTable, nColumn)
	Trace(AColumn.FullName + "//" + AColumn.Title)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableEnumColumn(<Table control> , <Column index>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la columna. 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Este resultado puede asignarse a una [Control](../WDLang1/1000020967.md) Variable.

- Cadena vacía ("") si el parámetro &lt;Índice de la columna&gt; es igual al número de columnas del control Tabla + 1.




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla a manipular.

Si este parámetro corresponde a una cadena vacía (""), se utilizará el control Tabla al que pertenece el Process actual.

**`<Column index>: (Integro)`**

Índice de la columna, entre 1 y el número de columnas del control Tabla + 1. 
Se produce un error WLanguage si este parámetro es mayor que el número de columnas que se encuentran en control Tabla + 1.

Para conocer el número de columnas de una control Tabla, utilice [TableCount](../WDLang1/3074021.md).

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000126"></a>

La función **TableEnumColumn** puede utilizarse en:

- una control Tabla basada en un archivo de datos o rellenada mediante programación. 

- a control Tabla TreeView basada en un archivo de datos o rellenada mediante programación.

- una selección única o selección múltiple control,





<a name="NOTE0_2"></a>


### Los diferentes tipos de índices de columna
<a name="los_diferentes_tipos_indices_columna_ELTPARAGRAPHE000145"></a>

Existen dos tipos de índices para las columnas:

- Índice de la posición Visible: índice de la columna cuando se ejecuta la ventana.

- Índice de creación: índice de la columna cuando se crea la control en el editor de ventanas.




Estos dos tipos de índices son devueltos por [TableColumnIndex](../WDLang1/3074016.md).

Estos índices cambian si el usuario mueve las columnas.

**TableEnumColumn** utiliza el índice de creación de columnas.

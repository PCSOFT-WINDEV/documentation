
## WinInitialized( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si el código de "fin de inicialización" se ha ejecutado en una ventana..


Esta función se puede utilizar para evitar llenar algunos controles cuyo código de modificación se ejecuta antes del final de la apertura (por ejemplo, al restaurar los valores almacenados).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modification code of the CBOX_Filtering check box
IF WinInitialized() = True THEN
	FillList(CBOX_Filtering)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinInitialized()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecutó el código de inicialización de la ventana,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

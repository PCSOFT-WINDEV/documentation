


## TreeExpand (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desglosa un nodo en un control TreeView. Los elementos "hijo" (nodos u hojas) de este nodo se pueden ver.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)
**Reminder**: En tiempo de ejecución, un nodo se expande cuando el nombre del nodo va precedido de "-".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Expand the "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TreeExpand(TREE_TVRecipe, "Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TreeExpand(<TreeView control> , <Node to expand>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nodo se expandió,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Node to expand>: (Cadena de caracteres)`**

Ruta del nodo a expandir. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nodo a expandir
<a name="nodo_expandir_ELTPARAGRAPHE000068"></a>

Si el nodo a expandir:

- no se encuentra, **TreeExpand** devuelve <u><u><u><u>False</u></u></u></u>.

- es una hoja, **TreeExpand** devuelve <u><u><u><u>True</u></u></u></u>.

- ya está expandido, **TreeExpand** devuelve <u><u><u><u>True</u></u></u></u>.

- no es visible, **TreeExpand** hace que este nodo sea visible. También se muestran sus elementos "padre" y "hermano".. Los elementos "padre" se expanden.


Cuando se llama a **TreeExpand**, si la rama especificada tiene un Procedure asociado a través del [AddChildDelayed](../Proprietes/1000020927.md) Property, se ejecutará este Procedure.
<a name="NOTE0_2"></a>


### Banda de selección
<a name="banda_seleccion_ELTPARAGRAPHE000111"></a>

La función **TreeExpand** no cambia la posición de la banda de selección.

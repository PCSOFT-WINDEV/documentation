
## AnimationSetParameter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia uno de los parámetros de animación utilizados para:

- un Image animado (en un control Imagen),

- un texto animado (en un control Estático).






![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En esta versión, sólo puede cambiar los parámetros de animación utilizados para un texto animado (en un control Estático).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Change the animation parameters of an image
IMG_Image1.Animation = True
AnimationSetParameter(IMG_Image1, acpDirectionAlternate, True)
```


<a name="Example2"></a>


![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Change the animation parameters of a Static control
IMG_Image1.Animation = True
AnimationSetMode(STC_MyStatic, actBlinking)
AnimationSetParameter(STC_MyStatic, acpDuration, 100)
AnimationSetParameter(STC_MyStatic, acpColor, DarkBlue)
```

## Sintaxis
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Cambiar los parámetros de animación de un Image

`AnimationSetParameter(<Image control> , <Parameter to modify> , <Parameter value>)`
---

**`<Image control>: (Cadena de caracteres)`**

Nombre del control Imagen a manipular.

**`<Parameter to modify>: (Constante o cadena de caracteres)`**

Nombre del parámetro a modificar.


|   |   |
| --- | --- |
| *acpDirection* (o "Direction") | Dirección de desplazamiento.<br>**Animación de tipo "Desplazamiento" únicamente.**<br>Estos parámetros se pueden combinar.<br><br>- 1 (por defecto): de izquierda a derecha<br><br>- 2: de derecha a izquierda<br><br>- 4: de abajo a arriba<br><br>- 8: de arriba a abajo<br><br><br>Ejemplo de combinación:<br><br>- 3 (2+1): izquierda derecha: ampliar progresivamente desde el centro<br><br>- 5 (4+1) de abajo a arriba y de derecha a izquierda<br><br><br><br><br> |
| *acpDirectionAlternate* (o "DirecciónAlternativa") | Lectura continua de los dibujos de la imagen, del primero al último y del último al primero (por defecto, el valor de este parámetro es <u><u><u><u>False</u></u></u></u>). |
| *acpDuration* (o "Duration") | Tiempo durante el cual se muestra cada dibujo animado en milisegundos (300 ms por defecto). |
| *acpLoop* (o "Loop") | Lectura continua de los dibujos de la imagen, del primero al último (por defecto, el valor de este parámetro es <u><u><u><u>True</u></u></u></u>). |
| *acpNbImageX* (o "NumberOfImagesX") | Número de dibujos a lo ancho del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br>**Animación de tipo "Imagen a animar" únicamente.**<br><br> |
| *acpNbImageY* (o "NumberOfImagesY") | Número de dibujos a lo alto del archivo imagen. Dependiendo del número especificado, la imagen se dividirá automáticamente en tantos dibujos como sea necesario (1 por defecto).<br>**Animación de tipo "Imagen a animar" únicamente.**<br><br> |
| *acpNbStep* (o "NumberOfSteps") | 10 por defecto<br>**Animación de tipo "Desplazamiento" únicamente.**<br><br> |
| *acpReverse* (o "Reverse") | Lectura de los dibujos de la imagen, del último dibujo abajo a la derecha al primero arriba a la izquierda (por defecto, el valor de este parámetro es <u><u><u><u>False</u></u></u></u>) |



**`<Parameter value>: (Tipo de parámetro esperado)`**

Nuevo valor para el parámetro.

### Cambiar los parámetros de animación de un control Estático

`AnimationSetParameter(<Static control> , <Parameter to modify> , <Parameter value>)`
---

**`<Static control>: (Cadena de caracteres)`**

Nombre de la control Estático a utilizar.

**`<Parameter to modify>: (Constante o cadena de caracteres)`**

Nombre del parámetro a modificar.


|   |   |
| --- | --- |
| *acpColor* (o "Color") | Color parpadeante. Este parámetro puede corresponder a una [constante de color](../WDLang5/3010002.md). Por defecto, este color corresponde a *Transparent*.<br>** Sólo animación"Parpadeante"** |
| *acpDirection* (o "Direction") | Dirección de desplazamiento.<br>**Animación de tipo "Desplazamiento" únicamente.**<br>Estos parámetros no se pueden combinar.<br><br>- 1 (por defecto): de izquierda a derecha<br><br>- 2: de derecha a izquierda<br><br>- 4: de abajo a arriba<br><br>- 8: de arriba a abajo<br><br><br> |
| *acpDuration* (o "Duration") | Tiempo durante el cual se muestra cada elemento animado en milisegundos (300 ms por defecto). |
| *acpNbPixel* (o "NumberOfPixels") | Número de pixeles para cada intervalo de desplazamiento.<br>**Animación de tipo "Desplazamiento" únicamente.** |



**`<Parameter value>: (Tipo de parámetro esperado)`**

Nuevo valor para el parámetro.


## BinaryAND( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el resultado de un AND lógico (poco a poco) entre dos valores.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the result of a logical AND
ResBinaryAND = BinaryAND(6, 6)  // Returns 6
ResBinaryAND = BinaryAND(6, 2)  // Returns 2
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = BinaryAND(<Value 1> , <Value 2>)`
---

**`<Result>: (Integro)`**

Resultado de la lógica AND.

**`<Value 1>: (Integro)`**

Primer valor a probar.

**`<Value 2>: (Integro)`**

Segundo valor a probar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Cálculo de bit a bit
<a name="calculo_bit_bit_ELTPARAGRAPHE000075"></a>

**BinaryAND** realiza el cálculo a nivel de bits entre &lt;Valor1&gt; y &lt;Valor2&gt; basado en el siguiente tabla:


|   |   |   |
| --- | --- | --- |
|   | 0 | 1 |
| 0 | 0 | 0 |
| 1 | 0 | 1 |


<a name="NOTE0_2"></a>




### Operador binario
<a name="operador_binario_ELTPARAGRAPHE000102"></a>

- La función **BinaryAND** es equivalente al operador binario **&**. Para obtener más información, consulte [Operadores binarios](../Motscles/1512009.md).

- El operador binario **&** se utiliza para realizar operaciones con enteros de 8 bytes.

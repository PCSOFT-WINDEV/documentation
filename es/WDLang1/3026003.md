


## ComboOpen (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Despliega la lista de un control Combo Box (Combo Box de una o varias columnas).

**Observaciones**: 

- La control Combo Box cuya lista se expande gana foco y se ejecuta el código de entrada de esta Combo Box.

- Si se desglosa otra control Combo Box, esta Combo Box se cierra primero.






![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta función no puede utilizarse en los cuadros combinados nativos ("Combo Box nativo
(Windows Store Apps)" marcado en la pestaña "Detalles" de la ventana del control description).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Expands the list of "COMBO_FrenchCity" control
ComboOpen(COMBO_FrenchCity)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ComboOpen(<Combo Box control>)`
---

**`<Combo Box control>: (Nombre del control)`**

Nombre de control Combo Box a utilizar.
Se produce un error de WLanguage si este parámetro no corresponde a un nombre válido de control.

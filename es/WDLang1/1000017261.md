
## AppInstallUPD( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa la actualización de automatic de la aplicación actual.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF AppUPDAvailable() = appUPDAvailMandatory THEN
	IF YesNo("An update is available for the application.", ...
		  "Do you want to perform this update?") = Yes THEN
			  AppInstallUPD()
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppInstallUPD()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se inicia la actualización,

- <u><u><u><u>False</u></u></u></u> si no se inició la actualización. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 




Si se activa la actualización, la aplicación finaliza y se instala la actualización..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Al ejecutar la función **AppInstallUPD**:

- la aplicación actual está cerrada. Se ejecuta el código de cierre del proyecto.

- la aplicación se reinicia (sin el comando Line). La actualización se detecta e instala.

- la aplicación se reinicia al final de la actualización, sin el comando Line.

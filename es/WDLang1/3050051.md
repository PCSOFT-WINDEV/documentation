


## Funciones de gestión de matrices
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar matrices:



|   |   |
| --- | --- |
| [MatAdd](../WDLang1/3050049.md) | Añade dos matrices de las mismas dimensiones (el mismo número de filas y el mismo número de columnas). |
| [MatCopy](../WDLang1/3050048.md) | Crea la copy de una matriz. |
| [MatCreate](../WDLang1/3050058.md) | Crea una matriz que contiene 0 columna fila y 0. |
| [MatDelete](../WDLang1/3050031.md) | Borra una matriz existente. |
| [MatDeterminant](../WDLang1/3050045.md) | Calcula el determinante de una matriz cuadrada (el número de filas es igual al número de columnas). |
| [MatError](../WDLang1/3050044.md) | Identifica el tipo de último error causado por una función Matxxx ([funciones para la gestión de matrices](../WDLang1/3050051.md)). |
| [MatExist](../WDLang1/3050076.md) | Comprueba si la matriz existe en la memoria. |
| [MatFill](../WDLang1/3050057.md) | Inicializa todos los elementos que se encuentran en una matriz de un tamaño determinado. |
| [MatFloatAdd](../WDLang1/3050059.md) | Añade un valor a cada elemento de matriz. |
| [MatFloatMultiply](../WDLang1/3050040.md) | Multiplica cada elemento de matriz por un valor. |
| [MatInvert](../WDLang1/3050075.md) | Calcula la inversa de una matriz cuadrada. |
| [MatMultiply](../WDLang1/3050072.md) | Multiplica dos matrices. |
| [MatNbColumn](../WDLang1/3050073.md) | Devuelve el número de columnas que se encuentran en una matriz. |
| [MatNbLine](../WDLang1/3050060.md) | Devuelve el número de filas que se encuentran en una matriz. |
| [MatRead](../WDLang1/3050074.md) | Lee el valor de un elemento de matriz. |
| [MatReadColumn](../WDLang1/3050070.md) | Lee el valor de todos los elementos que se encuentran en una columna de matriz. |
| [MatReadLine](../WDLang1/3050032.md) | Lee el valor de todos los elementos que se encuentran en una matriz fila. |
| [MatStack](../WDLang1/3050043.md) | Comprime la huella de memoria ocupada por una matriz. |
| [MatTranspose](../WDLang1/3050001.md) | Calcula la matriz transpuesta. |
| [MatWrite](../WDLang1/3050077.md) | Escribe un elemento en una matriz. |

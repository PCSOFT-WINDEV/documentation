
## SysNumberScreen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de pantallas actualmente conectadas al ordenador actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Return the number of connected screens
Info(SysNumberScreen())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysNumberScreen()`
---

**`<Result>: (Integro)`**

Número de pantallas actualmente conectadas.

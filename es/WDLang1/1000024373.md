
## CriticalSection( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa un nombre o sección crítica on-Variable en la declaración [USO DE ... EN](../Motscles/1510025.md). 
<a name="sample_code"></a>

## Ejemplo

<a name="Example1"></a>

```wl
// Critical section on a variable
t is array <critical section> of int
USING CriticalSection(t) IN
	// Operations on the array by one only thread at a time
	...
END
```

### Habilitando una sección crítica en una Variable

`CriticalSection([<Variable name>])`
---

**`<Variable name>: (Cualquier tipo)`**

Nombre de WLanguage Variable que debe ser protegido en una sección crítica. Esta variable debe definirse con el atributo &lt;sección crítica&gt;.


<a name="SYNTAX2"></a>

### Habilita una sección crítica nombrada en una parte del código

`CriticalSection([<Section name>])`
---

**`<Section name>: (Cadena de caracteres opcional)`**

Identifica la sección crítica. Este parámetro se utiliza para identificar las secciones de código protegidas. 

Si no se especifica este parámetro, se implementa una sección crítica de automatic. 
Observación: la sección crítica automática y la sección crítica que corresponde a una cadena vacía ("") son diferentes.


## DDEs constantes
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **DDEActive** | Modo de inicio del programa: el programa que se ejecuta es Active y "toma control" sobre el programa actual. | [DDEStart](../WDLang1/3028013.md) |
| **DDEIconize** | Modo de inicio del programa: la ejecución del programa es Inactive y se minimiza. | [DDEStart](../WDLang1/3028013.md) |
| **DDEInactive** | Modo de inicio del programa: el programa que se ejecuta es Inactive. | [DDEStart](../WDLang1/3028013.md) |
| **DDEMaximize** | Modo de inicio del programa: el programa que se ejecuta es Active y "toma control" sobre el programa actual (la ventana de ejecución se maximiza). | [DDEStart](../WDLang1/3028013.md) |
| **DDEtoAnswer** | DDE Event: el programa recibe una solicitud de datos de otra aplicación. | [DDEEvent](../WDLang1/3028009.md) |
| **DDEtoExecute** | DDE Event para gestionar: el programa recibe una orden de ejecución que proviene de otra aplicación. | [DDEEvent](../WDLang1/3028009.md) |
| **DDEtoLink** | DDE Event: el programa recibe una solicitud de enlace "caliente" o "caliente" a un dato (esta solicitud proviene de otra aplicación). | [DDEEvent](../WDLang1/3028009.md) |
| **DDEtoModify** | DDE Event: el programa recibe una señal de modificación del valor de un dato relacionado (esta señal viene de otra aplicación). | [DDEEvent](../WDLang1/3028009.md) |
| **DDEtoReceive** | DDE Event: el programa recibe datos de otra aplicación. | [DDEEvent](../WDLang1/3028009.md) |

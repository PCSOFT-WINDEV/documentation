
## LargeFontFactor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ratio de ampliación actual de una ventana WINDEV (sólo si Windows está en modo "fuentes grandes").

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Esta función solo está disponible para simplificar la Webificación de proyectos WINDEV.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enlarge the "EDT_CustomerName" edit control
// according to the enlargement ratio of the window
EDT_CustomerName.X = EDT_CustomerName.X * LargeFontFactor()
EDT_CustomerName.Width = EDT_CustomerName.Width * LargeFontFactor()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LargeFontFactor([<Window used>])`
---

**`<Result>: (Real)`**



- Relación de ampliación actual de la ventana especificada.

- 1 si:

	- Windows no está en modo "fuentes grandes",

	- la ventana no se agranda en fuentes grandes.




- 0 si se produce un error. 




**`<Window used>: (Nombre de la ventana opcional)`**

Nombre de la ventana a utilizar. Si no se especifica este parámetro, se gestionará la ventana actual.

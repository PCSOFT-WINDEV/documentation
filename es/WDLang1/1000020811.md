
## Icon (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Icon** permite definir las características avanzadas de un ícono. Las características del ícono se pueden definir y cambiar mediante diferentes propiedades WLanguage.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load an icon
MyIcon is Icon = "Logo.ico"
// Modify the first image of the icon
MyIcon..Image[1]..Image = "C:\MyImages\logo48.bmp"
MyIcon.TransparencyColor[1] = LightMagenta
MyIcon.Format[1] = icoFormatBMP
// Save the modified icon
dSaveImageICO(MyIcon, "Logo.ico")
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Icon
<a name="propiedades_especificas_descripcion_variables_tipo_icon_ELTPARAGRAPHE000027"></a>

Las siguientes propiedades permiten manipular un ícono:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| BitPerPixel | Array de integers | Array que contiene, para cada imagen del ícono, el número de bits por pixel de la imagen. |
| Format | Array | Array con el formato de cada imagen del ícono. Los formatos admitidos son: <br><br>- BMP: la constante correspondiente es *icoFormatBMP*. <br><br>- PNG: la constante correspondiente es *icoFormatPNG*.<br><br><br> |
| Image | Array de [Image](../WDLang1/1000019650.md) | Array con las imágenes de cada formato del ícono. <br><br>Se conservarán las dimensiones de las imágenes especificadas. El sistema elegirá la imagen más adecuada según las dimensiones y el número de colores (se seleccionará la primera imagen que se encuentre si existen dos imágenes con las mismas dimensiones y número de colores). |
| Palette | Array de [Palette](../WDLang1/1000021347.md) | Paleta de colores de cada imagen del ícono. |
| TransparencyColor | Array de [Color](../WDLang1/1000019661.md) | Array con el color de transparencia de cada imagen del ícono. |
| Value | Image (memo) | Valor del ícono. Esta propiedad permite cargar un ícono a partir de un buffer, archivo o memo. |

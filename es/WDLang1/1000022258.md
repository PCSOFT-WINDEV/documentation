
## Codificar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Codifica una cadena de caracteres o un buffer. Para descodificar el resultado, basta con utilizar [Decodifique](../WDLang1/1000022259.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Encode an image in Base64
Res is string = Encode(fLoadBuffer("MyImage.jpg"), encodeBASE64)
```
<a name="Example2"></a>

```wl
// Encode a URL
Info(Encode("https://www.google.fr/?", encodeURLFromAnsi) + ...
	Encode(Encode("https://www.google.fr/?é", encodeURLFromAnsi), encodeURLParameter))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Encode(<Data> [, <Encoding format>])`
---

**`<Result>: (Cadena de caracteres)`**

Datos codificados según el formato seleccionado.

**`<Data>: (Cadena de caracteres o memoria intermedia)`**

Datos a codificar.

**`<Encoding format>: (Constante opcional de tipo Integer)`**

Formato de los datos encoding:


|   |   |
| --- | --- |
| *encodeBASE64*<br>(Valor predeterminado) | Formato BASE 64 (cumple con el estándar RFC 4648). El texto codificado contiene sólo caracteres imprimibles y CRs (Carriage Returns).<br><br> |
| *encodeBASE64NoCR* | Formato BASE 64 (cumple con el estándar RFC 4648) sin RCs (Carriage Returns). El texto codificado sólo contiene caracteres imprimibles.<br><br> |
| *encodeBASE64URL* | Formato BASE 64 URL (cumple con el estándar RFC 4648). El texto codificado sólo contiene caracteres imprimibles.<br><br> |
| *encodeBASE85* | Formato BASE 85. El texto codificado sólo contiene caracteres imprimibles.<br><br><br><br> |
| *encodePCS* | Formato ASCII. El texto codificado sólo contiene caracteres imprimibles.<br><br> |
| *encodeURLFromAnsi* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto a codificar es una cadena ANSI.<br><br> |
| *encodeURLFromUnicode* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto a codificar es una cadena Unicode.<br><br> |
| *encodeURLFromUTF8* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto a codificar es una cadena UTF-8.<br><br> |
| *encodeURLParameter* | Esta constante se utiliza cuando es necesario pasar una determinada URL (URL A) como parámetro a otra URL (URL B). Los caracteres reservados de la URL A (//,?...) están codificados.<br><br>La URL A (a codificar) debe ser una URL válida. Si tiene sus propios parámetros, deben haber sido codificados con una de las siguientes constantes: *encodeURLFromAnsi* , *encodeURLFromUnicode* o *encodeURLFromUTF8* .<br><br> |
| *encodeUUEncode* | Formato UUEncode. El texto codificado sólo contiene caracteres imprimibles.<br><br> |

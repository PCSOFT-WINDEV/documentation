


## PDFReaderCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra y libera el archivo PDF que se muestra en un lector de PDF control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
PDFReaderClose(PDF_MyPDFReader)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PDFReaderClose(<PDF Reader control>)`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del lector de PDF control que contiene el archivo PDF a cerrar.

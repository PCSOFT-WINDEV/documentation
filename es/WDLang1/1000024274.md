


## &lt;Tabla&gt;.ModifyColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica una columna en una "Tabla horizontal" control.

**Observación**: La función **&lt;Tabla&gt;.ModifyColumn** puede utilizarse en:

- una control Tabla horizontal basada en un archivo de datos o rellenada mediante programación.

- una selección única o una selección múltiple control Tabla.





<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.ModifyColumn(<Index> [, <Element row 1> [... [, <Element row N>]]])`
---

**`<Table control>: (Nombre del control)`**

Nombre de la control Tabla Horizontal a utilizar.

**`<Index>: (Integro)`**

Índice de la columna a modificar. El índice de la primera columna es set a 1.


|   |   |
| --- | --- |
| Índice igual a -1. | Se modifica la columna actual. No hay efecto si no se selecciona ninguna columna. |
| Índice mayor que el número de columnas de la control Tabla. | Se produce un error del WLanguage. |



**`<Element row 1>: (Cadena de caracteres opcional)`**

Primer elemento de la primera fila a modificar en la columna. El valor de este elemento sustituye al valor existente. Cada elemento corresponde a una fila en el control Tabla. Cada tipo de elemento modificado debe ser compatible con el tipo de fila correspondiente..
Si no se especifica ninguno de los parámetros &lt;Elemento fila X&gt;, **&lt;Tabla&gt;.ModifyColumn** no tiene ningún efecto.



**`<Element row N>: (Cadena de caracteres opcional)`**

Enésimo elemento de la Nª fila a modificar en la columna. El valor de este elemento sustituye al valor existente. Cada elemento corresponde a una fila en el control Tabla. Cada tipo de elemento modificado debe ser compatible con el tipo de fila correspondiente..
Si no se especifica ninguno de los parámetros &lt;Elemento fila X&gt;, **&lt;Tabla&gt;.ModifyColumn** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Características de los elementos de columna
<a name="caracteristicas_los_elementos_columna_ELTPARAGRAPHE000198"></a>

- Para modificar el valor de un elemento de una columna, utilice la siguiente sintaxis:
	
	```txt
	TableControl.RowName[ColumnIndex] = NewValue
	```


- Si un elemento corresponde al resultado de un cálculo, la expresión numérica debe estar entre paréntesis.. Por ejemplo:
	
	```wl
	TABLE_CalculationTable.ModifiyColumn((53+29),(83-21))
	```




<a name="NOTE0_2"></a>


### Tabla archivo de datos: Modificaciones en el archivo de datos relacionado
<a name="tabla_archivo_datos_modificaciones_archivo_datos_relacionado_ELTPARAGRAPHE000218"></a>

Si la entrada en cascada está activada, de el archivo de datos relacionado a la control Tabla se modifica automáticamente cuando el usuario pasa a la siguiente columna.

Si la entrada en cascada no está habilitada, no se modifica el el archivo de datos relacionado al control Tabla basado en un archivo de datos. Para escribir el registro en el archivo de datos, la función [&lt;Tabla&gt;.Save](../WDLang1/1000024249.md) debe ejecutarse después de la función **&lt;Tabla&gt;.ModifyColumn**.

**Recordatorio**: Se puede seleccionar la opción "entrada en cascada":

- Al crear una control Tabla.

- En la pestaña "Detalles" de la ventana de description de control Tabla en el editor de la ventana.



<a name="NOTE0_3"></a>


### Modificaciones realizadas en una control Tabla basada en un archivo de datos
<a name="modificaciones_realizadas_una_control_tabla_basada_archivo_datos_ELTPARAGRAPHE000242"></a>

Para modificar una columna en una control Tabla basada en un archivo de datos, le aconsejamos:

- modificar la record en el archivo con [&lt;Fuente&gt;.Modify](../WDLang4/1000025038.md),

- utilizar [&lt;Table&gt;.Display](../WDLang1/1000023505.md) para actualización el control Tabla.



<a name="NOTE0_4"></a>


### Función &lt;Tabla&gt;.ModifyColumn y Property modificada
<a name="funcion_docparampagetitleshort_property_modificada_ELTPARAGRAPHE000265"></a>

**&lt;Tabla&gt;.ModifyColumn** no influye en el [Modified](../Proprietes/2510080.md) property. El [Modified](../Proprietes/2510080.md) Property se utiliza para saber si la columna actual ha sido modificada directamente por el usuario (entrada directa en el control Tabla).

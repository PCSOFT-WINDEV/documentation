
## Funciones de gestión de Token (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los tokens en los controles de edición mediante la sintaxis de los prefijos:



|   |   |
| --- | --- |
| [&lt;Edición&gt;.DeleteAllToken](../WDLang1/1000024047.md) | Borra todos los Token de un Token "Texto control Campo de entrada". |
| [&lt;Edición&gt;.TokenCount](../WDLang1/1000024045.md) | Devuelve el número de Token s presentes en un "Texto Token" control Campo de entrada. |
| [&lt;Editar&gt;.AddToken](../WDLang1/1000024043.md) | Añade una Token a un "Texto Token" Editar control. |
| [&lt;InsertToken](../WDLang1/1000024044.md) | Inserta un token en un control Campo de entrada de tipo "Tokens de texto". |
| [.DeleteToken](../WDLang1/1000024046.md) | Elimina una Token de un "Texto Token" Editar control. |

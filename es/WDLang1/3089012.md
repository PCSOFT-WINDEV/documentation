
## SchedulerReset( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Rellena las estructuras [Tareas programadas](../WDLang1/3089004.md) y/o [Activación de tareas programadas](../WDLang1/3089002.md) con los valores Default.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the ScheduledTask and TriggerScheduledTask structures
SchedulerReset()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SchedulerReset([<Structure to reset>])`
---

**`<Structure to reset>: (Constante o combinación de constantes opcional)`**

Tipo de la estructura a restablecer.


|   |   |
| --- | --- |
| *schedulerResetTask* | Restablece la estructura ScheduledTask. |
| *schedulerResetTask* + *schedulerResetTiming*<br>(predeterminado) | Restablece ambas estructuras. |
| *schedulerResetTiming* | Restablece la estructura TriggerScheduledTask. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Restablecer las estructuras
<a name="restablecer_las_estructuras_ELTPARAGRAPHE000060"></a>

Las estructuras se restablecen para crear una tarea Default (idéntica a la creada por "Nueva tarea" en el planificador de Windows): una tarea que se realiza todos los días a las 9:00 de la mañana, que se detiene si el equipo pasa a funcionar con batería y que no puede superar las 72 horas

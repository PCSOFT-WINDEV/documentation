


## PVTLoad (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga el resultado del cálculo realizado en una control Tabla dinámica desde una copia de seguridad (archivo encontrado en el disco o copia de seguridad en un búfer).. Este cálculo fue guardado de antemano por [PVTSave](../WDLang1/1000020471.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFile is string
// Builds the file name
sFile = SysDir(srAppData) + ["\"] + "StatsByCountry.tcd"

// The file exists?
IF fFileExist(sFile) THEN
	// Yes, load it 
	PVTLoad(PVT_Statistics, sFile)
	IF YesNo("Do you want to recalculate everything?") THEN PVTCalculateAll(PVT_Statistics)
ELSE
	// The file does not exist, calculate it
	PVTCalculateAll(PVT_Statistics)
END
PVTSave(PVT_Statistics, sFile)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PVTLoad(<Pivot Table control> , <Backup media> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si se ha cargado el contenido del fichero control Tabla dinámica. 

- <u><u><u><u>False</u></u></u></u> en caso contrario




**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Backup media>: (Cadena de caracteres o memoria intermedia)`**



- Copia de seguridad en un archivo: Nombre y ruta completa del archivo a cargar. Este archivo corresponde a un archivo previamente guardado con [PVTSave](../WDLang1/1000020471.md).

- Copia de seguridad en memoria: Buffer de reserva devuelto por [PVTSave](../WDLang1/1000020471.md). 




**`<Password>: (Cadena de caracteres opcional)`**

Contraseña associated con el respaldo de la control Tabla dinámica. Esta contraseña fue especificada cuando el control Tabla dinámica fue guardado por [PVTSave](../WDLang1/1000020471.md).

## Observaciones


### Guardar y cargar el contenido de una control Tabla dinámica
<a name="guardar_cargar_contenido_una_control_tabla_dinamica_ELTPARAGRAPHE000080"></a>

Cuando la control Tabla dinámica es calculada por [PVTCcalcularTodos](../WDLang1/1000020539.md), el resultado puede ser guardado por [PVTSave](../WDLang1/1000020471.md). A continuación, se puede volver a cargar con **PVTLoad**: en este caso, no se recalcula el contenido de la control Tabla dinámica.

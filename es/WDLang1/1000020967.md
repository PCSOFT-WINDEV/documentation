
## Control (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **control** se utiliza para definir las características de un control. Esta control puede ser creada dinámicamente por [ControlCreate](../WDLang1/1000020876.md) o puede ser una control encontrada en una ventana o en una reporte. 

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) [ControlCreate](../WDLang1/1000020876.md) no está disponible.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Dynamically create a control in a window
c is Control
c <- ControlCreate("BUTTON_1", typBouton, 30, 40, 140, 20) // Caution, not '='

c.Caption = "Click here"

c.Process[trtClick] = myProcedure
```

<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>


### Declarando un control Variable
<a name="declarando_control_variable_ELTPARAGRAPHE000052"></a>

La siguiente sintaxis se utiliza para declarar un Variable de control: 


```txt
<Variable Name> is Control
```

<a name="NOTE0_2"></a>


### Asignando un control Variable
<a name="asignando_control_variable_ELTPARAGRAPHE000061"></a>

Un **Control** Variable puede ser usado para manipular: 

- un control encontrado en una ventana o en un reporte. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) a control creada por [ControlCreate](../WDLang1/1000020876.md). 




La asignación de un Control Variable debe ser realizada por el operador '<-'. 

Ejemplos: 


```wl
// Assignment with an existing control
c is Control
c <- EDT_Edit1
```



```wl
// Assignment by using the indirection
// The variable references the EDT_EDIT2 control
c <- {"EDT_EDIT2"}
```


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Assignment with a control created by ControlCreate
c is Control
c <- ControlCreate("BUTTON_1", typButton, 30, 40, 140, 20)
```


**Atención**: El operador "=" no debe ser usado para asignar un **Control** Variable. Este operador se utiliza para transferir el valor de un control a otro control del mismo tipo. Por ejemplo: 


```wl
// Assignment with an existing control
c is Control
c <- EDT_Edit1


// Modify the value 
c = EDT_Edit3 // Equivalent code: EDT_Edit1.Value = EDT_Edit3.Value

c = "Text" // Equivalent code: EDT_Edit1.Value = "Text"
```


**Observación**: Para saber si se ha asignado un valor al Control Variable, utilice el [escribir a máquina](../Proprietes/2510131.md) Property en el Variable. 
<a name="NOTE0_3"></a>


### Usando una variable de control en un Procedure
<a name="usando_una_variable_control_procedure_ELTPARAGRAPHE000100"></a>

El tipo **Control** puede ser un parámetro de un Procedure. Puedes escribir por ejemplo: 

- el código de la Procedure: 
	
	```wl
	PROCEDURE MyProcedure(C is Control)
	C = "It is " + TimeSys()
	```


- la llamada a la Procedure: 
	
	```wl
	MyProcedure(EDT_Edit1)
	```




<a name="NOTE0_4"></a>


### Optimización de los procesos con indirección
<a name="optimizacion_los_procesos_con_indireccion_ELTPARAGRAPHE000114"></a>

El tipo **control** puede utilizarse para optimizar los procesos que utilizan las indirecciones. Puedes escribir por ejemplo: 


```wl
C is Control <-{s, indControl}
C.Width = 10
C.Height = 20
```


en lugar del siguiente código: 


```wl
{s, indControl}..Width = 10
{s, indControl}..Height = 20
```





<a name="NOTE0_5"></a>


### Propiedades asociadas a un Variable de control
<a name="propiedades_asociadas_variable_control_ELTPARAGRAPHE000128"></a>

El tipo **control** se utiliza para manipular todos los tipos de control s que se encuentran en una ventana o en un reporte. Las propiedades disponibles dependen del tipo de control utilizado. 

Por lo tanto, si se usa un Control Variable para manipular un control Campo de entrada, todas las propiedades de WLanguage disponibles para el control Campo de entrada serán utilizables en el **Control** Variable. 

Para get las propiedades de un control, consulte la página de ayuda de control: el enlace "Ver también" se utiliza para acceder a todas las propiedades de control. Para obtener más información, consulte [Controles de ventanas](../WDChamp/1013081.md). 
<a name="NOTE0_6"></a>


### Las funciones que manipular a Control Variable
<a name="las_funciones_que_manipular_control_variable_ELTPARAGRAPHE000142"></a>

Todas las funciones WLanguage que manipulan controles pueden utilizarse en una variable de tipo **Control**. Sin embargo, sólo estarán disponibles las funciones WLanguage correspondientes al tipo de control utilizado. 

Por lo tanto, las funciones de la TableXXX pueden ser usadas en un Variable de control que maneje un control Tabla.

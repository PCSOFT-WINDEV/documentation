
## DateDifference (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el número de días entre dos fechas.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculates the number of days elapsed since 01/01/1998
nbDays is int
nbDays = DateDifference("19980101", Today())
// Displays the result
Info(nbDays + " days elapsed")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DateDifference(<Start date> , <End date>)`
---

**`<Result>: (Entero)`**

Número de días transcurridos entre dos fechas. El número de días es el resultado de la siguiente operación: 

```txt
<End date> - <Start date>
```

El número de días es negativo si la &lt;Fecha de inicio&gt; está después de la &lt;Fecha de finalización&gt;.

**`<Start date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de inicio del cálculo en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).

**`<End date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de finalización del cálculo en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La validez de las fechas pasadas como parámetros se verifica. Si la fecha no es válida, se muestra un mensaje. La validez de las fechas se puede comprobar con la función [DateValid](../WDLang1/3027003.md).

Esta función no permite calcular la diferencia entre dos fechas anteriores al 14 de octubre de 1582 (el cambio del calendario juliano al gregoriano no se tendrá en cuenta).

El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999.

Los cálculos sobre las fechas realizadas con las [funciones WLanguage](../WDLang1/3027029.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.


## DatePicker (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el selector de fecha del sistema.

**Observación**: Para seleccionar una fecha mediante el selector, le recomendamos utilizar directamente la máscara de entrada "Selector de fecha" en el control Campo de entrada de tipo Fecha. Esta máscara de entrada también está disponible para aplicaciones iPhone/iPad. 
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=DateSelecteur.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is Date = DatePicker()
```



```wl
InitialDate is Date = "20170101"
MyDate is Date
MyDate = DatePicker(InitialDate, "Start date")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DatePicker([<Initial date> [, <Title>]])`
---

**`<Result>: (Fecha o cadena de caracteres)`**



- Fecha seleccionada en el selector,

- Cadena vacía ("") si el usuario cancela la selección




**`<Initial date>: (Fecha o cadena de caracteres opcional)`**

Fecha que aparece al abrir el selector. Si no se especifica esta fecha, el selector mostrará la fecha actual.

**`<Title>: (Cadena de caracteres opcional)`**

Título que se mostrará en el selector. Si no se especifica este parámetro, el título será la fecha que aparece en el selector en formato de texto.

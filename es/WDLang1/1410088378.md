


## &lt;Table&gt;.ToXML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un archivo XML a partir de los datos encontrados en una Tabla o control Tabla TreeView.

**Observaciones**: 

- Por Default, una hoja de estilo (archivo "XSL") se genera automáticamente al crear el archivo XML.

- La entidad Document del archivo XML generado se llama WINDEV_TABLE. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
sFile is string
// File name
sFile = fSelect("", "", "Choose the export file", ...
	"XML files (*.XML) *.xml", "xml", fSelCreate + fSelExist)
// The export is canceled if no name was specified
// Otherwise, export the content of TABLE_TABLE1
IF sFile <> "" THEN TableToXML(TABLE_TABLE1, sFile, taNoTitle)
```

## Sintaxis

`<Result> = <Table control>.ToXML(<Name of the XML file> [, <Options> [, <Start row> [, <End row>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla. 

- un control Tabla TreeView.




**`<Name of the XML file>: (Cadenas de caracteres)`**

Nombre y ruta completa del archivo XML a crear. El archivo se sustituye por el nuevo si ya existe.

**`<Options>: (Constante opcional de tipo Integer)`**

Configura el archivo XML a crear.


|   |   |
| --- | --- |
| *taColumnDisplayedOrder* | Exporta las columnas según el orden de las columnas que aparecen actualmente en la control (y no según el orden de las columnas definido en el editor).<br> |
| *taColumnsTitles*<br>(valor predeterminado) | El título de las columnas se inserta en el archivo. |
| *taNoTitle* | Sólo se copian los datos en el archivo. |
| *taNoTotal* | No exporta: <br><br>- las filas que contienen los totales, la media y la automática Count( estas filas son exportadas por Default).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) las filas que contienen cálculos personalizados (estas filas son exportadas por Default).<br><br><br> |
| *taNoXSL* | La hoja de estilo (archivo".XSL") no se crea. |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |
| *taWithInvisibleColumns* | También exporta las columnas de Invisible control.<br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Las columnas definidas como no imprimibles y no exportables se exportan. Estas columnas se definen: <br><br>- mediante programación: La propiedad [VisibleInExportAndPrint](../Proprietes/1000022320.md) se establece en <u><u><u><u>False</u></u></u></u>. <br><br>- en la pestaña "General" de la columna de la ventana de description: "Exportar e imprimir" set a "Nunca".<br><br><br> |



**`<Start row>: (Entero opcional)`**

Número de la fila donde se iniciará la exportación. 
Si no se especifica este parámetro, el fila de inicio es el primer fila del control.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. 
Si no se especifica este parámetro, el fila final es el último fila del control.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000322"></a>

La función **&lt;Table&gt;.ToXML** puede utilizarse en: 

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Columna de tipo memo binario (imagen)
<a name="columna_tipo_memo_binario_imagen_ELTPARAGRAPHE000340"></a>

Para las columnas Binary Memo, el archivo XML contiene una cadena vacía.
<a name="NOTE0_3"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000350"></a>

Las columnas combinadas se ignoran.

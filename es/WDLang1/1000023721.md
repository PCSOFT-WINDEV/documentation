


## &lt;Hoja de cálculo&gt;.StartModificación (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el inicio de un grupo de operaciones en un control Hoja de cálculo en un solo Event (gestión de "deshacer/rehacer").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The selection is displayed in bold and italic. A single Ctrl + Z cancels both operations
PSHEET_Spreadsheet.StartModification()
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)
PSHEET_Spreadsheet.StyleSelection(psheetItalic, True)
PSHEET_Spreadsheet.EndModification()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Control Hoja de cálculo>.StartOfModification()`
---

**`<Control Hoja de cálculo>: (Control name)`**

Nombre de control Hoja de cálculo a utilizar.

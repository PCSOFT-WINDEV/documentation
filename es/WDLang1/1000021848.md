
## WinEdSave (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda las modificaciones realizadas en el modo "Modificación de UI por el usuario final". Estas modificaciones se pueden guardar en la memoria o en un archivo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on [Save]
WinEdSave(WIN_Overview, "My modifications.wdmodif")


// Click on [Load]
WinEdLoad(WIN_Overview, "My modifications.wdmodif")
IF ErrorOccurred = True THEN
	Info("Unable to load" + CR + ErrorInfo()) 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Guardar las modificaciones en un buffer

`<Result> = WinEdSave(<Window> , <inMemory> , <Save mode>)`
---

**`<Result>: (Buffer)`**

Buffer que contiene el resultado de la copia de seguridad. La memoria intermedia está vacía si no se ha realizado ninguna modificación.

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar.

**`<inMemory>: (Constante)`**

*inMemory*: Constante utilizada para guardar las modificaciones en memoria.

**`<Save mode>: (Constante de tipo Integer)`**

Modo para guardar las modificaciones: 



|   |   |
| --- | --- |
| *weModeDifferential*<br>(Valor predeterminado) | Guardar todas las operaciones de modificación, incluidas las operaciones personalizadas ([WinEdDo](../WDLang1/1000021897.md)). Todas estas operaciones se reproducirán durante la operación de recarga (como en el caso de un registro).. |
| *weModeFull* | Guardar los cambios de todos los controles y sus propiedades actuales. Esta constante sólo se puede utilizar en una ventana interna.  |
| *weModeReposionableNote* | Guardar sólo las notas reposicionables creadas por el usuario. Esta constante equivale a la función "Exportar las notas" del menú contextual de las notas reposicionables (AAF). |




<a name="SYNTAX2"></a>

### Guardar las modificaciones en un archivo

`<Result> = WinEdSave(<Window> , <File name> , <Save mode>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardaron las modificaciones,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).


Observación: Si no se ha realizado ninguna modificación, &lt;Resultado&gt; es igual a <u><u><u><u>False</u></u></u></u>. 

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. 

**`<File name>: (Cadena de caracteres)`**

Nombre del archivo en el que deben guardarse las modificaciones. La extensión de este archivo es "wdmodif".. 

**`<Save mode>: (Constante de tipo Integer)`**

Modo para guardar las modificaciones:



|   |   |
| --- | --- |
| *weModeDifferential*<br>(Valor predeterminado) | Guardar todas las operaciones de modificación, incluidas las operaciones personalizadas ([WinEdDo](../WDLang1/1000021897.md)). Todas estas operaciones se reproducirán durante la operación de recarga (como en el caso de un registro).. |
| *weModeFull* | Guardar los cambios de todos los controles y sus propiedades actuales. Esta constante sólo se puede utilizar en una ventana interna.  |
| *weModeReposionableNote* | Guardar sólo las notas reposicionables creadas por el usuario. Esta constante equivale a la función "Exportar las notas" del menú contextual de las notas reposicionables (AAF).<br><br>También puede utilizar [WinEdSaveNote](../WDLang1/1000022120.md) para guardar sólo las notas reposicionables creadas por el usuario. |

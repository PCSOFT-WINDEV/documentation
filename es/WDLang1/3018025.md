


## Funciones de gestión de controles TreeView
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles TreeView: 



|   |   |
| --- | --- |
| [Colapso del árbolTodos](../WDLang1/3018011.md) | Colapsa una rama previamente expandida (así como sus elementos hijos). |
| [TreeAdd](../WDLang1/3018012.md) | Agrega un elemento "hoja" (y todos los nodos necesarios) en un control TreeView. |
| [TreeCollapse](../WDLang1/3018008.md) | Colapsa una rama previamente expandida. |
| [TreeCopyItem](../WDLang1/3018006.md) | Copia un elemento (hoja o nodo) en un control TreeView. |
| [TreeCount](../WDLang1/3018009.md) | Devuelve el número de elementos que se encuentran en una rama de una control TreeView. |
| [TreeDelete](../WDLang1/3018001.md) | Elimina un elemento y todos sus "hijos" de un control TreeView. |
| [TreeDeleteAll](../WDLang1/3018018.md) | Borra todos los elementos de un control TreeView. |
| [TreeDeleteChild](../WDLang1/3018017.md) | Borra los hijos de un elemento de un control TreeView sin borrar el elemento mismo.. |
| [TreeDisplay](../WDLang1/1000019565.md) | Actualiza la visualización de un control TreeView desde una posición determinada. |
| [TreeExpand](../WDLang1/3018007.md) | Desglosa un nodo en una control TreeView. |
| [TreeExpandAll](../WDLang1/3018022.md) | Expande un nodo y todos sus elementos hijo en una control TreeView. |
| [TreeFind](../WDLang1/3018030.md) | Encuentra un elemento (hoja o nodo) en una control TreeView y devuelve su ruta. |
| [TreeGetIndex](../WDLang1/1000023418.md) | Devuelve el índice de un elemento (nodo u hoja) en un control TreeView. |
| [TreeGiveChild](../WDLang1/3018010.md) | Devuelve los "hijos" de un elemento (nivel inmediatamente inferior) en un control TreeView. |
| [TreeID](../WDLang1/3018027.md) | Se utiliza para construir la ruta de los elementos que se encuentran en un control TreeView diferenciando los duplicados. |
| [TreeIdentifier](../WDLang1/3018004.md) | Devuelve el identificador associated con un elemento (nodo u hoja) en una control TreeView. |
| [TreeInfoXY](../WDLang1/3018026.md) | Devuelve el elemento TreeView que se encuentra en una posición determinada. |
| [TreeInsert](../WDLang1/3018023.md) | Inserta una hoja en una control TreeView en una posición específica. |
| [TreeListItem](../WDLang1/3018016.md) | Enumera los "hijos" de un nodo y los elementos "hijo" de estos "hijos" en un control TreeView. |
| [TreeModify](../WDLang1/3018013.md) | Modifica un elemento (nodo u hoja) en un control TreeView. |
| [TreeMoveItem](../WDLang1/3018024.md) | Mueve un elemento (hoja o nodo) en un control TreeView. |
| [TreePosition](../WDLang1/3018005.md) | Muestra un control TreeView de un elemento especificado (nodo u hoja) o devuelve el nombre del primer elemento mostrado en un control TreeView. |
| [TreeSelect](../WDLang1/3018003.md) | Devuelve la ruta completa del elemento seleccionado en un control TreeView. |
| [TreeSelectCount](../WDLang1/1000019376.md) | Devuelve el número de elementos seleccionados en una control TreeView (para una control TreeView multiselección). |
| [TreeSelectMinus](../WDLang1/1000019636.md) | Desmarca un elemento de un control TreeView. |
| [TreeSelectPlus](../WDLang1/3018002.md) | Selecciona un elemento en una control TreeView. |
| [TreeSort](../WDLang1/3018020.md) | Ordena los elementos que se encuentran en una control TreeView. |
| [TreeStatus](../WDLang1/3018015.md) | Devuelve el status de un nodo.: colapsado, expandido o no encontrado. |
| [TreeTypeItem](../WDLang1/3018014.md) | Comprueba la existencia de un elemento en una control TreeView y devuelve su tipo (nodo u hoja) si existe el elemento. |

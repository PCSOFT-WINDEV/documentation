
## TextToHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte el texto a HTML (los saltos de línea se reemplazan por &lt;BR&gt;, los acentos se reemplazan por su equivalente en HTML, etc.).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// EDT_HTML is edit control
EDT_HTML = TextToHTML("WINDEV is a great tool" + CR + ...
	"Developing applications is really fun")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TextToHTML(<Text> [, <Charset used>])`
---

**`<Result>: (Cadena de caracteres)`**

Texto en formato HTML. Atención: las etiquetas &lt;html&gt; y &lt;/html&gt; no se incluyen en el resultado. Se deben agregar si es necesario mostrar el resultado en un control HTML.

**`<Text>: (Cadena de caracteres)`**

Nombre del control de texto o cadena de caracteres en formato de texto.  

**`<Charset used>: (Constante opcional de tipo Integer)`**

Constante que identifica el conjunto de caracteres utilizado para escribir el &lt;Texto&gt;. De forma predeterminada, se utiliza el conjunto de caracteres actual de WINDEV o WEBDEV (constante *charsetCurrent*). 
Para obtener más información sobre estas constantes, consulte [Correspondencia entre idiomas, subidiomas, conjuntos de caracteres y naciones](../WDLang1/3054022.md).





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para convertir una cadena RTF a HTML, utilice la función [RTFToHTML](../WDLang1/3025049.md).

- Para convertir una cadena RTF a Texto, utilice la función [RTFToText](../WDLang1/3025032.md).


## SysStatusExternalStorage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de disponibilidad del espacio de almacenamiento externo del dispositivo.. 

Esta función debe utilizarse antes de las operaciones de lectura/escritura en el espacio de almacenamiento para comprobar si éste está disponible.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the path of the directory that will be used to store the application files 
// on the primary external storage space of application
IF SysStatusExternalStorage(1) = sseAvailable THEN
	RESULT SysDirExternalStorage(1, sseAppFile)
ELSE
	RETURN ""
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysStatusExternalStorage(<Subscript>)`
---

**`<Result>: (Constante de tipo Integer)`**

Estado actual de la tarjeta de almacenamiento en el dispositivo:


|   |   |
| --- | --- |
| *sseAvailable* | Tiene la capacidad de leer y escribir en el espacio de almacenamiento externo. |
| *sseUnavailable* | No se puede leer ni escribir en el espacio de almacenamiento externo. |
| *sseReadOnly* | Tiene la capacidad de leer en el espacio de almacenamiento externo. |



**`<Subscript>: (Integro)`**

Índice del espacio de almacenamiento externo que se va a utilizar (si el dispositivo incluye varios espacios de almacenamiento externo). 
Observación: [SysNbExternalStorage](../WDLang1/1000021816.md) devuelve el número de espacios de almacenamiento externos encontrados en el dispositivo.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un mismo dispositivo puede contener varias áreas de almacenamiento externo (por ejemplo, un área corresponde a la partición "externa" de la memoria permanente y otra área corresponde a la SDCard).

- [SysRemovableExternalStorage](../WDLang1/1000021818.md) se utiliza para averiguar si un espacio de almacenamiento corresponde a la memoria móvil (SDCard) o no.

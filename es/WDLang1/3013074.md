


## InAJAXMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica si el código se ejecuta desde:

- un proceso en modo AJAX.

- un procedimiento llamado con las funciones [AJAXExecute](../WDLang2/3055114.md) o [AJAXExecuteAsynchronous](../WDLang2/3055115.md).




Para obtener más información, consulte [Presentación de AJAX](../WDLang2/3055101.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run from an AJAX process?
IF InAJAXMode() = True THEN
	// AJAX mode
	// Processes to perform
	...
ELSE
	// Non-AJAX mode
	// Processes to perform
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InAJAXMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el código se ejecuta desde un proceso en modo AJAX o desde un procedimiento llamado con las funciones [AJAXExecute](../WDLang2/3055114.md) o [AJAXExecuteAsynchronous](../WDLang2/3055115.md),

- <u><u><u><u>False</u></u></u></u> en caso contrario.




Si el proceso actual no está en modo AJAX, pero se llamó desde un proceso en modo AJAX (con la función [ExecuteProcess](../WDLang1/3013031.md), por ejemplo), la función **InAJAXMode** devuelve <u><u><u><u>True</u></u></u></u>.

## Observaciones
<a name="NOTE0_2"></a>
<a name="NOTE0_3"></a>


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000296"></a>

La función **InAJAXMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InAJAXMode** pueden compartirse entre varios productos sin generar errores de compilación.

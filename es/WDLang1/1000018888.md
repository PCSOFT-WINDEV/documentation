


## OrganizerTimeDisplayed (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el rango de tiempo mostrado por la control Agenda en el modo Día o en el modo Semana. La barra de desplazamiento y la altura de las filas de la control Agenda se modifican para mostrar el intervalo de tiempo especificado.. 

Si el intervalo de tiempo de la control Agenda es mayor que el que se muestra, la barra de desplazamiento le permitirá acceder a los tiempos antes y después del intervalo.







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación universal de Windows 10, esta función sólo se puede utilizar para los controles del organizador manipular en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions the scrollbar in order for the visible time period 
// to go from 9.00 to 19:30 
OrganizerTimeDisplayed(ORG_Organizer1, "0900", "1930")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`OrganizerTimeDisplayed(<Organizer control> , <Start time> , <End time>)`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.




**`<Start time>: (Cadena de caracteres)`**

Hora de inicio de la visualización (en formato HHMM o HH).

**`<End time>: (Cadena de caracteres)`**

Hora de finalización de la visualización (en formato HHMM o HH).

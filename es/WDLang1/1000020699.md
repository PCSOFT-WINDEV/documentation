


## PVTExpand (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desglosa los datos encontrados en una control Tabla dinámica según una cabecera fila o a una cabecera de columna.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Expands all quarters
PVTExpand(PVT_Sales, COL_OrderDate_Quarter)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PVTExpand(<Pivot Table control> , <Header to expand>)`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Header to expand>: (Cadena de caracteres)`**

Nombre de la cabecera fila o de la cabecera de columna a desglosar.

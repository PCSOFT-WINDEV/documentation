


## Funciones para la gestión de las combinaciones y enumeraciones
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar las Combinaciones y Enumeraciones: 



|   |   |
| --- | --- |
| [CombinationCheckName](../WDLang1/1000020772.md) | Comprueba si una combinación de una o más opciones conocidas por sus nombres es válida. |
| [CombinationFromName](../WDLang1/1000020771.md) | Devuelve una combinación de una o más opciones conocidas por su nombre.. |
| [CombinationFromValue](../WDLang1/1000020770.md) | Devuelve una combinación de una o más opciones conocidas por sus valores de associated. |
| [EnumerationCheckName](../WDLang1/1000020767.md) | Comprueba si un valor de enumeración conocido por su nombre es válido. |
| [EnumerationCheckValue](../WDLang1/1000020866.md) | Comprueba si un valor de enumeración conocido por su valor associated es válido. |
| [EnumerationFromName](../WDLang1/1000020769.md) | Devuelve un valor de enumeración conocido por su nombre. |
| [EnumerationFromValue](../WDLang1/1000020768.md) | Devuelve un valor de enumeración conocido por su valor associated. |

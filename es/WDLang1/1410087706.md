
## WLanguage Procedure llamado por AppDetectUPDAvailable



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedure ("Callback") llamada por [AppDetectUPDAvailable](../WDLang1/1410087705.md) cuando se detecta una actualización de la aplicación Active. Esta Procedure puede ser una Procedure global o interna.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AppDetectUPDAvailable(UpdateAvaliable)
	INTERNAL PROCEDURE UpdateAvailable(Update, Version)
		IF Update = appUPDAvailMandatory THEN
			ToastDisplay("A mandatory update is available (" + Version + ").")
		END
	END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AppDetectUPDAvailable_Callback(<Level of update> , <Version>)`
---

**`<Level of update>: (Constante de tipo Integer)`**

Tipo de actualización disponible: 



|   |   |
| --- | --- |
| *appUPDAvailMandatory* | Una actualización obligatoria está disponible. |
| *appUPDAvailOptional* | Una actualización opcional está disponible. |



**`<Version>: (Cadena de caracteres)`**

Número de la versión disponible. 







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Configuración de la comprobación de actualizaciones de automatic
<a name="configuracion_comprobacion_actualizaciones_automatic_ELTPARAGRAPHE000047"></a>

Esta Procedure se utiliza para configurar la acción que se realiza cuando se detecta la actualización. (Para más detalles, véase [automatic comprueba las actualizaciones disponibles](../WDLang1/1410087705.md)). Esta acción sustituye a la Notification de Windows. Esta Procedure puede, por ejemplo: 

- no realizar ninguna acción: en este caso, no se notificará al usuario que hay una actualización disponible. 

- realizar una acción específica (mostrar un mensaje explicativo como toast, por ejemplo). Esta acción sustituirá a la Notification de Windows por defecto.

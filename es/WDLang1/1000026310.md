


## Funciones control del editor HTML (sintaxis de prefijos)



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones de WLanguage se utilizan para gestionar los controles del editor HTML (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;Editor de HTML&gt;. Búsqueda](../WDLang1/1000026301.md) | Busca una cadena en un editor HTML control. |
| [&lt;Editor de HTML&gt;.EjecutarJS](../WDLang1/1000026297.md) | Ejecuta el código JavaScript en un editor HTML control. |
| [&lt;Editor de HTML&gt;.FormatSelection](../WDLang1/1000026298.md) | Modifica el estilo de los elementos seleccionados en un editor HTML control. |
| [&lt;Editor de HTML&gt;.get](../WDLang1/1000026302.md) | Recupera asincrónicamente el valor de una de las propiedades de un editor de HTML control. |
| [&lt;Editor de HTML&gt;.GetDocument](../WDLang1/1000026304.md) | Recupera de forma asíncrona la Document HTML que se está mostrando en una control de edición HTML . |
| [&lt;Editor de HTML&gt;.GetSelection](../WDLang1/1000026305.md) | Recupera de forma asíncrona la selección actual de un editor HTML control. |
| [&lt;Editor de HTML&gt;.Insert](../WDLang1/1000026299.md) | Inserta texto o código HTML en la posición del cursor en un editor HTML control. |
| [&lt;Editor de HTML&gt;.Reemplazar](../WDLang1/1000026306.md) | Sustituye una cadena por otra en un editor HTML control. |
| [&lt;Editor de HTML&gt;.Seleccionar](../WDLang1/1000026307.md) | Selecciona un fragmento de texto en un editor de HTML control. |
| [&lt;Editor HTML&gt;.FromEmail](../WDLang1/1000026296.md) | Importa el contenido HTML de un correo electrónico y sus imágenes en un editor HTML control. Las imágenes contenidas en la Email se muestran también en la control. |
| [&lt;Editor HTML&gt;.InsertTable](../WDLang1/1000026300.md) | Inserta una tabla en la posición del cursor en un editor de HTML control. |
| [&lt;Editor HTML&gt;.ToEmail](../WDLang1/1000026308.md) | Exporta el contenido del control de un editor HTML a un correo electrónico Variable (para enviar la Email). |
| [.ToPDF](../WDLang1/1000026309.md) | Exporta el contenido de un editor HTML control a un archivo PDF. |

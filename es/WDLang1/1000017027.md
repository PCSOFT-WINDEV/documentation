


## InUMCMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el código se ejecuta desde un código escrito en un Macro-Código de Usuario. Por ejemplo, permite saber si en el código de un procedimiento, éste se ejecutó desde un Macro-Código de Usuario o no.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run from UMC?
PROCEDURE Check()
...
IF InUMCMode() = True THEN
	// Procedure run from UMC
	...
ELSE
	// Standard case
	// Process to perform
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InUMCMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ejecuta el código desde un Macro-Código de Usuario,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000223"></a>

La función **InUMCMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InUMCMode** pueden compartirse entre varios productos sin generar errores de compilación.

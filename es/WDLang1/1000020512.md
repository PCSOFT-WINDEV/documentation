
## SexagesimalToDecimal ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ángulo decimal correspondiente a un ángulo sexagesimal.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Finds the angle in degrees corresponding to the latitude 45°53'36"
sLatitude is string = "45°53'36''"
rLatitudeAngle is real = SexagesimalToDecimal(sLatitude)
Info("The latitude " + sLatitude + " corresponds to " + rLatitudeAngle + " °")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SexagesimalToDecimal(<Sexagesimal angle>)`
---

**`<Result>: (Real)`**

Resultado de la conversión en formato decimal. 

**`<Sexagesimal angle>: (Cadena de caracteres)`**

Ángulo en formato sexagesimal. Esta cadena tiene el siguiente formato
```txt
XXX°XX'XX''
```

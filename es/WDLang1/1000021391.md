


## Funciones de gestión de controles Pestaña
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular los controles de la pestaña: 



|   |   |
| --- | --- |
| [ControlPane](../WDLang1/1000021625.md) | Devuelve el nombre del panel estático o dinámico que se encuentra en una control Pestaña associated con una control. |
| [ControlTab](../WDLang1/3025028.md) | Devuelve el nombre del control Pestaña que contiene el control especificado. |
| [TabClose](../WDLang1/1000021362.md) | Cierra un panel dinámico en un control Pestaña. |
| [TabCloseAll](../WDLang1/1000021372.md) | Cierra todos los cristales dinámicos de una control Pestaña. |
| [TabLoadConfiguration](../WDLang1/1000021370.md) | Restaura la configuración de los paneles dinámicos en un control Pestaña. |
| [TabOpen](../WDLang1/1000021369.md) | Abre un nuevo panel en una control Pestaña dinámica. |
| [TabSaveConfiguration](../WDLang1/1000021371.md) | Guarda la configuración de los paneles dinámicos que se encuentran en un control Pestaña en formato de cadena de caracteres. |
| [TabStatus](../WDLang1/1000021637.md) | Devuelve el estado de un panel de pestañas dinámico (identificado por su alias). |

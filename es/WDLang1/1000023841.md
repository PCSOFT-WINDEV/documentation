


## &lt;Tabla&gt;.CellSelect (Función)




	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>16 (56F)</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [&lt;Tabla&gt;.Select](../WDLang1/1000024284.md).

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las coordenadas (número de fila y número de columna) de una celda en un control Tabla.. 

**Observaciones**: 

- La control Tabla debe permitir el modo de selección por celda. Esta opción se puede configurar en la pestaña "GUI" de control Tabla description. 

- Para conocer el número de celdas seleccionadas, utilice [&lt;Tabla&gt;.CellSelectOccurrence](../WDLang1/1000023842.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbSelectedCells is int
Total is real
RowNum is int
ColNum is int

NbSelectedCell = TABLE_TABLE1.CellSelectOccurrence()
FOR i = 1 TO NbSelectedCells
	RowNum = TABLE_TABLE1.CellSelect(i, tcsLine)
	ColNum = TABLE_TABLE1.CellSelect(i, tcsColumn)
	Total = Total + TABLE_TABLE1[RowNum,ColNum] 
	
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Table control>.CellSelect(<Index> , <Type of information>)`
---

**`<Result>: (Integro)`**



- Número de la columna o fila seleccionada,

- -1 si se produce un problema. 




**Observación**: El modo de selección por celda debe haber sido seleccionado para la control Tabla (pestaña "GUI" en el control description). De lo contrario, &lt;Resultado&gt; es set a -1.

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.

**`<Index>: (Integro)`**

Número de la columna seleccionada. Si este índice no corresponde a una columna seleccionada (por ejemplo, índice mayor que el número de columnas), &lt;Resultado&gt; se set a -1.

**`<Type of information>: (Constante de tipo Integer)`**

Se utiliza para especificar si el resultado debe ser el número de fila o el número de columna.: 


|   |   |
| --- | --- |
| *tcsColumn* | Se utiliza para recuperar el número de columna. El número de la primera columna es set a 1. |
| *tcsLine* | Se utiliza para recuperar el número de fila. El número del primer fila es set a 1. |

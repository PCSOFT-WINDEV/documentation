


## &lt;Document Variable&gt;.ToText (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una cadena de caracteres con los datos de una [Document](../WDLang1/1000022461.md) Variable.

Esta cadena se puede guardar en un formato específico a partir de entonces.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

MyString is string
MyString = MyDocument.ToText()
// Save in a text file
fSaveText("C:\temp\Content.txt", MyString)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Document>.ToText()`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena de caracteres que contiene los datos. 

- Vacíe la cadena ("") si se ha producido un error o si no hay datos que extraer.




**`<Document>: (Documento o cadena Variable)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

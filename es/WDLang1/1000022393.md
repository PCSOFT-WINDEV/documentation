


## DocToHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un procesador de textos Document en un archivo HTML.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
d is Document = "text.docx"
IF DocToHTML(d, "text.htm") THEN
	Info("Document converted to HTML format")
END
```
<a name="Example2"></a>

```wl
// Exports the content of a Word Processing control to HTML
// Click code of "HTML export" button 
DocToHTML(WP_MyDoc, fExeDir() + [fSep] + "wp.htm")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DocToHTML(<Document to convert> [, <Path>] [, <Managing the CSS styles>])`
---

**`<Result>: (Booleana o Buffer)`**



- Si se especifica &lt;Path&gt;: 

	- <u><u><u><u>True</u></u></u></u> si se generó el archivo,

	- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




- Si no se especifica &lt;Ruta&gt;, &lt;Resultado&gt; corresponde a un buffer que contiene el código HTML. 




**`<Document to convert>: (Nombre del documento Variable o control)`**

Document para usar. Este Document puede corresponder a: 

- una variable de tipo [Document](../WDLang1/1000022461.md). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Procesador de texto.




**`<Path>: (Cadena de caracteres opcional)`**

Ruta del archivo HTML a generar. Si no se especifica el ruta, el código HTML se devuelve en formato de búfer. 

**`<Managing the CSS styles>: (Constante opcional)`**

Atributos de estilo a utilizar:



|   |   |
| --- | --- |
| *dthWithCSS*<br>(Valor predeterminado) | Aplica **ajeno** Estilos de CSS<br>El código CSS se genera en un archivo ".CSS" externo a la página HTML. Este archivo se crea en el mismo directorio que la página HTML generada.. Tiene el mismo nombre que la página HTML, con la extensión ".CSS. Se añade un enlace a este archivo en el código HTML de la página.<br><br>Si el Document contiene imágenes, éstas se guardan en un subdirectorio con el mismo nombre que el archivo HTML generado. |
| *dthWithoutCSS* | Aplica **en línea** Estilos de CSS: el código CSS es generado e integrado en el código HTML de la página generada. Cada etiqueta contiene los estilos CSS que se le aplican. <br><br>Si el Document contiene imágenes, están integradas y codificadas en base64. <br><br>Esta constante permite, por ejemplo, crear una generación HTML adaptada a la nueva mensajería Email. <br><br>Observación: si una imagen contenida en una Email es de más de 100 KB, se mostrará como un archivo adjunto, incluso si ha sido codificado en base64. |

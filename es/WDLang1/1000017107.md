
## ServiceRefresh( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Solicita a un servicio que relea su información de configuración. Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Refreshes the service found in TABLE_Service
IF ServiceRefresh(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceRefresh(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True STRIMEULB</u></u></u></u> si la solicitud de actualización se ha transmitido al responsable del servicio, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a actualización.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para actualización un servicio en otro ordenador.. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía, el servicio se gestiona localmente..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000065"></a>

La función **ServiceRefresh** le pide al administrador de servicios actualizar el servicio. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).

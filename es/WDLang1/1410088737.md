


## <Diagram editor>.Diagram InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica la presencia de una forma en una posición determinada del editor de Diagram control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
let nShapeIndex = DIAGEDT_Diagram.InfoXY(MouseXPos, MouseYPos)
gDiag is Diagram <- DIAGEDT_Diagram.Value
FOR i = 1 _TO_ gDiag.Shape.Count 
	// Change the opacity of the hovered shape
	IF nShapeIndex = i THEN 
		gDiag.Shape[nShapeIndex].Opacity = 50 
	ELSE 
		gDiag.Shape[nShapeIndex].Opacity = 100
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Diagram Editor>.InfoXY(<X> , <Y>)`
---

**`<Result>: (Integro)`**



- Índice de la forma si hay una forma en la posición especificada,

- 0 de lo contrario.




**`<Diagram Editor>: (Nombre del control)`**

Nombre del editor de Diagram control a utilizar.

**`<X>: (Integro)`**

Coordenada X de la posición especificada.

**`<Y>: (Integro)`**

Coordenada Y de la posición especificada.

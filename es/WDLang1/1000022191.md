


## &lt;Tabla dinámica&gt;.SelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de celdas seleccionadas en una control Tabla dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Loop through the selected cells
pvtPos is pvtPosition of PVT_MyPVT

nbSel is int = PVT_MyPVT.SelectOccurrence()
FOR i = 1 TO nbSel
	pvtPos = PVT_MyPVT.Select(i)
	Trace(pvtPos.COL_Today_Date + ":" + pvtPos.COL_Product))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Pivot Table control>.SelectCount()`
---

**`<Result>: (Integro)`**

Número de celdas seleccionadas en el control Tabla dinámica.

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

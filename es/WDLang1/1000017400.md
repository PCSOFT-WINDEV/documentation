
## AnimationMinFPS( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura el número mínimo de imágenes por segundo para que las animaciones (efectos de transición) estén habilitadas en los controles.

Si el equipo no es lo suficientemente rápida para mostrar el mínimo número de imágenes por segundo, los efectos de animación no se mostrarán en los controles para evitar un efecto entrecortado. En este caso, el control pasa directamente del estado inicial al estado final.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Minimum animation: 25 images per second
AnimationMinFPS(25)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AnimationMinFPS(<FPS>)`
---

**`<FPS>: (Integro)`**

Número mínimo de imágenes por segundo.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función sólo afecta a la animación de los controles de ventana. Las animaciones de imágenes (GIF animado) y las animaciones de ventanas no son modificadas por esta función.

- FPS significa "cuadros por segundo".

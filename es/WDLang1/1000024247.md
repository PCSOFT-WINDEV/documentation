


## <Table>.SetFocus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Da enfoque a una fila que se encuentra en una Tabla o control Tabla TreeView: el control cambia a editar para el fila y la columna especificados.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a row and switch to edit on "New" (COL_Name column)
let nRow = TABLE_MyTable.AddLine("New", 2)
TABLE_MyTable.SetFocus(nRow, COL_Nom)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Table control>.SetFocus([<Subscript> [, <Column>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el control Table ha cambiado a modo de entrada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.




**`<Subscript>: (Entero opcional)`**

Índice de la fila que ganará atención.
Si no se especifica este parámetro, el enfoque se da a la fila actual.. Si no hay ninguna fila seleccionada, la función **<Table>.SetFocus** no tiene ningún efecto y devuelve <u><u><u><u>False</u></u></u></u>.

**`<Column>: (Nombre del control)`**

Nombre de la columna tabla que recibirá el foco.
Si no se especifica este parámetro, el foco se da a la primera columna válida en el orden de visualización.. 
Si no hay ninguna columna válida (o si el nombre de la columna no corresponde a una columna del control Tabla), la función **<Table>.SetFocus** no tiene ningún efecto y devuelve <u><u><u><u>False</u></u></u></u>.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el control Tabla no tiene foco, la función **<Table>.SetFocus** dará el foco al control. 

- Si control Tabla está atenuado o Inactive , **<Table>.SetFocus** no tiene efecto y devuelve <u><u><u><u>False</u></u></u></u> .

- control, fila y los eventos de entrada y salida de la columna no se ejecutan.

- Si la control Tabla ya está en modo de edición, se modifica el número de la fila o de la columna en modo de edición.

- Las filas que aparecen en el control Tabla pueden moverse para que el fila con nuevo enfoque sea Visible.




## OrganizerMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el modo de visualización de un control Agenda: día, semana, mes, ....

Observación: [OrganizerChangeMode](../WDLang1/1000019497.md) se utiliza para cambiar el modo de visualización del control Agenda. 

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación universal de Windows 10, esta función sólo se puede utilizar para los controles del organizador manipular en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Toggles between daily display and monthly display
IF OrganizerMode(ORG_MyOrganizer) = orgzDay THEN
	OrganizerChangeMode(ORG_MyOrganizer, orgzMonth)
ELSE
	OrganizerChangeMode(ORG_MyOrganizer, orgzDay)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OrganizerMode(<Organizer control>)`
---

**`<Result>: (Constante de tipo Integer)`**

Rango de tiempo mostrado por la control Agenda:


|   |   |
| --- | --- |
| *orgzCustomNbDay* | Muestra la control Agenda durante un número de días especificado por [OrganizerChangeMode](../WDLang1/1000019497.md).<br><br> |
| *orgzDay* | El control Agenda muestra un día. |
| *orgzMonth* | El control Agenda muestra un mes completo. |
| *orgzWeek* | El control Agenda muestra una semana completa (o una semana Partial si se especificó un número de días con [OrganizerChangeMode](../WDLang1/1000019497.md)). |
| *orgzWeekOn5* | El control Agenda muestra una semana de 5 días (sin fin de semana). |



**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.

## Observaciones


### Compatibilidad
<a name="compatibilidad_ELTPARAGRAPHE000140"></a>

La función **OrganizerMode** reemplaza a la función [OrganizadorZoom](../WDLang1/1000018884.md) (conservada por motivos de compatibilidad).



### Equivalencias
<a name="equivalencias_ELTPARAGRAPHE000152"></a>Las siguientes líneas de código son equivalentes: 

```wl
OrganizerChangeMode(ORG_Organizer, orgzWeek, 5)
```
 y

```wl
OrganizerChangeMode(ORG_Organizer, orgzWeekOn5)
```
 

En ambos casos, la función **OrganizerMode** devuelve la constante *orgzWeekOn5*.

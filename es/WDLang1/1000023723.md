


## &lt;Hoja de cálculo&gt;.Freeze (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite congelar la visualización de una fila o columna en una control Hoja de cálculo.. Las filas y columnas congeladas seguirán siendo Visible cuando se muevan en la hoja de trabajo. 

Es posible: 

- congelar una o más columnas a la izquierda (o liberarlas). 

- congelar una o más filas en la parte superior (o liberarlas).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Freezes the 1st row
PSHEET_MySpreadsheet.Freeze(1, 0)

// Freezes the 1st column
PSHEET_MySpreadsheet.Freeze(0, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.Freeze(<Row> , <Column>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Row>: (Integro)`**



- Número de la última fila a congelar. Todas las filas especificadas (desde la primera hasta el número especificado) serán congeladas. 

- 0 para liberar las filas congeladas.




**`<Column>: (Integro)`**



- El número de la última columna que se congeló. Todas las columnas especificadas (desde la primera hasta el número especificado) serán congeladas. 

- 0 para liberar las columnas congeladas.

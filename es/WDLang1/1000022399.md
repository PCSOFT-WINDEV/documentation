


## OrganizerSelectedRange (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha y hora de inicio o de fin de la franja horaria seleccionada en una control Agenda.. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the range selected in an Organizer control
myStartDate is DateTime = OrganizerSelectedRange(ORG_MyOrganizer, orgStartDate)
myEndDate is DateTime = OrganizerSelectedRange(ORG_MyOrganizer, orgEndDate)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = OrganizerSelectedRange(<Organizer control> [, <Start or end of range>])`
---

**`<Result>: (FechaHora)`**



- Fecha y hora solicitadas (para el inicio o fin del rango),

- Cadena vacía ("") si no se ha realizado ninguna selección en la control Agenda.




**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda a utilizar (se encuentra en una ventana)

**`<Start or end of range>: (Constante de tipo Integer)`**

Indica la información a devolver: 


|   |   |
| --- | --- |
| *orgEndDate* | Información devuelta: Fecha y hora de finalización de la franja horaria seleccionada. |
| *orgStartDate* | Información devuelta: Fecha y hora de inicio de la franja horaria seleccionada. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función equivale a las propiedades [StartDate](../Proprietes/1000017426.md) y [EndDate](../Proprietes/1000017427.md) de control Agenda.

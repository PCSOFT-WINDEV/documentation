
## NetworkDisconnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desconecta una unidad de red.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Disconnect the network drive "E:"
ResDisconnect is int = NetworkDisconnect("E:")

SWITCH ResDisconnection
	CASE 16
		Error("Wrong value for disconnection")
	CASE 17
		Error("Connection currently used")
	CASE 18
		Error("A file is opened, unable to disconnect")
	CASE 19
		Error("Attempt to disconnect without a connection")
	OTHER CASE
		Info("Drive successfully disconnected")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NetworkDisconnect(<Drive letter>)`
---

**`<Result>: (Integro)`**



- 0 si se ha realizado la desconexión,

- un código de error (valor superior a 0).




**`<Drive letter>: (Cadena de caracteres)`**

Unidad de red para desconectar, en el siguiente formato: "&lt;Carta de unidad&gt;**:**".  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Códigos de error
<a name="codigos_error_ELTPARAGRAPHE000035"></a>

Se devuelven los siguientes códigos de error:

- 16: Valor incorrecto de desconexión.

- 17: Conexión utilizada actualmente.

- 18: Se abre un archivo que no puede desconectarse.

- 19: Intente desconectarse sin conexión.




El mensaje del código de error puede recuperarse mediante la función [ErrorInfo](../WDLang1/3013008.md) con la constante *rrMessage*.

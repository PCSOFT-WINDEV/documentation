


## TestCheckDifferent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Verifica si las expresiones especificadas son diferentes en la programación asertiva y en el modo de prueba automática.

Observación: Esta función es equivalente a la función [dbgCheckDifferent](../WDLang1/1000019777.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The function should not return 0
TestCheckDifferent(CalcFunction(), 0)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TestCheckDifferent(<Expression 1> , <Expression 2> [, <Comments> [, <Parameter 1> [... [, <Parameter N>]]]])`
---

**`<Expression 1>: (Cadena de caracteres)`**

Cualquier expresión. 

**`<Expression 2>: (Cadena de caracteres)`**

Cualquier expresión.

**`<Comments>: (Cadena de caracteres)`**

Cadena de caracteres en el formato definido por [StringBuild](../WDLang1/3024006.md). 

**`<Parameter 1>: (Cualquier tipo)`**

Primer parámetro esperado por &lt;Comentarios&gt;.



**`<Parameter N>: (Cualquier tipo)`**

El enésimo parámetro esperado por &lt;Comentarios&gt;.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000047"></a>

Cuando las expresiones son iguales: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **en modo de prueba automatizada**, se agrega un error al informe de la prueba (equivalente a la función [Resultado de TestWriteResult](../WDLang1/1000013001.md)).

- **en modo prueba manual**, se muestra una aserción (equivalente a la función [dbgAssert](../WDLang1/3014022.md)).

- **en modo ejecución**, se muestra una aserción si se han habilitado las aserciones (equivalente a la función [dbgAssert](../WDLang1/3014022.md)).




Cuando se ejecuta una aplicación, la expresión no se evalúa si las aserciones no están habilitadas.

[dbgOnError](../WDLang1/1000020722.md) permite iniciar automáticamente un procedimiento WLanguage utilizado para recuperar el comentario relacionado al error.


## Procedimiento WLanguage llamado por la función HashString



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [HashString](../WDLang1/1000007111.md) al aplicar el algoritmo de hash. Este procedimiento permite obtener el resultado del hash. 

Este procedimiento debe ser Navegador. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HashString(HA_SHA_160, "abc", HashString_Callback)

INTERNAL PROCEDURE HashString_Callback(buffer)
	// This code is used to view the buffer
	Trace(BufferToHexa(buffer))
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HashString_Callback(<Result>)`
---

**`<Result>: (Buffer)`**

Resultado del hash de la cadena con el algoritmo especificado en la función [HashString](../WDLang1/1000007111.md).

Atención: este resultado puede contener caracteres que no se pueden mostrar. Si desea mostrar el resultado, lo puede convertir con la función [BufferToHexa](../WDLang1/1000019916.md).




## &lt;Vista de árbol&gt;.SelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desmarca un elemento de un control TreeView. 

Ejemplo de control TreeView: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselect the "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TREE_TVRecipe.SelectMinus("Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <TreeView control>.SelectMinus(<Element path>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se deseleccionó el elemento,

- <u><u><u><u>False</u></u></u></u> si el elemento especificado no existe.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Element path>: (Cadena de caracteres opcional)`**

ruta completa del elemento a utilizar. Este parámetro tiene el siguiente formato: 

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)). 

Si se especifica este parámetro, se deselecciona el elemento especificado.. 

Si no se especifica este parámetro, se deselecciona la etapa actual..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Vista de árbol multiselección
<a name="vista_arbol_multiseleccion_ELTPARAGRAPHE000191"></a>

Para una selección múltiple de control: 

- si se especifica &lt;Elemento ruta&gt;, sólo se deselecciona el elemento especificado. 

- si no se especifica &lt;Elemento ruta&gt;, se deseleccionan todos los elementos seleccionados.

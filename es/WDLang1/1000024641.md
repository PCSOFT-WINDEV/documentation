


## &lt;Variable WDPic&gt;.VerticalSymmetry (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Voltea la imagen de una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo) con respecto a un eje vertical (simetría con respecto a un eje vertical).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dSymetrieVerticale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
// Vertical symmetry of the image contained in the WDPic variable
MyResult is boolean
MyResult = MyImage.VerticalSymmetry()
IF MyResult = False THEN Error("The operation failed")
IMG_MyDrawing = MyImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <WDPic image>.VerticalSymmetry()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se volteó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo. 

La simetría se realiza con respecto al eje vertical central de la imagen. La imagen se modifica directamente.

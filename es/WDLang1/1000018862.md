
## UUEncode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Codifica una cadena de caracteres según el algoritmo UUEncode.. Esta cadena codificada puede ser descodificada por [UUDecode](../WDLang1/1000018864.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Encode a string
Res = UUEncode("My account number is 0123456789")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = UUEncode(<String to encode>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena codificada según el algoritmo UUEncode. 

**`<String to encode>: (Cadena de caracteres)`**

Texto a codificar.

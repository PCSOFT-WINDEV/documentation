


## Funciones de gestión de controles Agenda
			







Las siguientes funciones se utilizan para gestionar los controles del Organizador:



|   |   |
| --- | --- |
| [iCalendarToOrganizer](../WDLang1/1000021918.md) | Importa datos en formato iCalendar a un control Agenda. |
| [OrganizadorAñadir Cita](../WDLang1/1000018877.md) | Añade una nueva Appointment a una control Agenda. |
| [OrganizadorBorrarTodo](../WDLang1/1000018882.md) | Borra todas las citas de una control Agenda. |
| [OrganizadorFondoColor](../WDLang1/1000020004.md) | Define un fondo Color para una franja horaria en un control Agenda. |
| [OrganizadorToiCalendar](../WDLang1/1000021919.md) | Exporta las citas encontradas en un control Agenda en formato iCalendar. |
| [OrganizerChangeMode](../WDLang1/1000019497.md) | Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). |
| [OrganizerChangeZoom](../WDLang1/1000018883.md) | Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). Esta función se conserva por motivos de compatibilidad con versiones anteriores. |
| [OrganizerDateVisible](../WDLang1/1000022875.md) | Hace visible un día o Invisible en una control Agenda. |
| [OrganizerDeleteAppointment](../WDLang1/1000018881.md) | Borra una Appointment de una control Agenda. |
| [OrganizerDisplay](../WDLang1/1000019425.md) | Reinicia el contenido de un control Agenda desde su fuente de datos (archivo de datos HFSQL, por ejemplo). |
| [OrganizerDisplayNextPeriod](../WDLang1/1000018885.md) | Mueve el área de tiempo que se muestra a la siguiente área (por ejemplo, las próximas semanas) en un control Agenda. |
| [OrganizerDisplayPreviousPeriod](../WDLang1/1000018886.md) | Desplaza el área de tiempo mostrada al área anterior (semanas anteriores, por ejemplo) en un control Agenda. |
| [OrganizerInfoXY](../WDLang1/1000018887.md) | Retornos para una posición determinada en la control Agenda (coordenadas de un punto en el organizador): <br><br>- el índice de la Appointment para la posición especificada. <br><br>- la fecha correspondiente a la posición especificada.<br><br><br> |
| [OrganizerListAppointment](../WDLang1/1000018880.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Agenda, <br><br>- todas las citas encontradas en un control Agenda,<br><br>- una Appointment específica (seleccionada o suspendida).<br><br><br> |
| [OrganizerMode](../WDLang1/1000019545.md) | Devuelve el modo de visualización de un control Agenda: día, semana, mes, ... |
| [OrganizerPosition](../WDLang1/1000018889.md) | Muestra un control Agenda a partir de una fecha especificada o devuelve la fecha del primer día (o la fecha del último día) que aparece en un control Agenda. |
| [OrganizerSelectedRange](../WDLang1/1000022399.md) | Devuelve la fecha y hora de inicio o de fin de la franja horaria seleccionada en una control Agenda. |
| [OrganizerSelectMinus](../WDLang1/1000019147.md) | Deselecciona un Appointment de un control Agenda. |
| [OrganizerSelectPlus](../WDLang1/1000019003.md) | Selecciona un Appointment en un control Agenda. |
| [OrganizerTimeDisplayed](../WDLang1/1000018888.md) | Define el intervalo de tiempo que muestra control Agenda en el modo Día o en el modo Semana. |
| [OrganizerWorkingHour](../WDLang1/1000022530.md) | Se utiliza para indicar las horas de trabajo específicas por día en una control Agenda. |
| [OrganizerZoom](../WDLang1/1000018884.md) | Devuelve el intervalo de tiempo mostrado por un control Agenda. |



**Observación**: También tiene la posibilidad de administrar los organizadores de Lotus Notes y los organizadores de Outlook.. Para obtener más información, consulte: 

- [Acceso a Outlook](../WDLang3/3032137.md).

- [Acceso a Lotus Notes](../WDLang3/3032136.md).

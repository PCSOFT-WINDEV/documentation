


## ComboCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra la lista desplegable de una control Combo Box (control Combo Box simple o multicolumna).

**Observaciones**:

- **ComboCerrar** no tiene efecto si la lista de control Combo Box no se expande.

- En la mayoría de los casos, la lista de control Combo Box se cierra automáticamente cuando el usuario selecciona un valor o cuando hace clic fuera de la lista.






![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta función no puede utilizarse en los cuadros combinados nativos ("Combo Box nativo
(Windows Store Apps)" marcado en la pestaña "Detalles" de la ventana del control description).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Close the drop-down list of "COMBO_FrenchCity" control
ComboClose(COMBO_FrenchCity)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ComboClose(<Combo Box control>)`
---

**`<Combo Box control>: (Nombre del control)`**

Nombre de control Combo Box a utilizar.
Se produce un error de WLanguage si este parámetro no corresponde a un nombre válido de control.

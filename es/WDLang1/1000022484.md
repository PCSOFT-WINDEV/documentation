
## docElement (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docElement** se utiliza para manipular cada elemento que se encuentra en una variable [docPárrafo](../WDLang1/1000022483.md). 

Las características de este elemento pueden ser definidas y modificadas por varias funciones y propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variables y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opens a docx file in the Word Processing control
WP_MyDoc = DocOpen(fExeDir() + fSep() + "MyDoc.docx")

// Fragment positioned at the end of document
fEnd is docFragment(WP_MyDoc, -1)
// Selects the document
TextFragment is docFragment(WP_MyDoc, 1, fEnd..EndPosition)

FOR EACH MyCurrentElement OF TextFragment..Element
	MyParagraph is docParagraph = MyCurrentElement..Paragraph
	Trace(MyParagraph..StartPosition)
	Trace(MyParagraph..EndPosition)
END
```

## Observaciones

### Propiedades específicas de la descripción de las variables docElement
<a name="propiedades_especificas_descripcion_las_variables_docelement_ELTPARAGRAPHE000036"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docElement**:

| **Nombre de la propiedad** | Tipo utilizado | Efecto |
| --- | --- | --- |
| Celda | [docCell](../WDLang1/1000022927.md) variable | Características de la célula (sólo si el elemento corresponde a una célula tabla). |
| Fórmula | [docElementFormula](../WDLang1/1000022937.md) variable | Características de la fórmula: fórmula, formato de fórmula, resultado de fórmula (sólo para un elemento de fórmula). |
| retrato | [docElementImage](../WDLang1/1000022696.md) variable | Características de la imagen (sólo se puede utilizar para un elemento Imagen). |
| vincular | Variable de tipo [docElementLink](../WDLang1/1000022970.md) | Características del enlace (sólo si el elemento corresponde a un enlace). <br><br>Estas características están disponibles en formato de sólo lectura. |
| Párrafo | [docPárrafo](../WDLang1/1000022483.md) variable | Párrafo que contiene el elemento. |
| Padre | Variable de tipo [docElement](../WDLang1/1000022484.md) | Elemento que contiene el elemento.<br>Esta propiedad está disponible en solo lectura. |
| Posición de inicio | Integro | Posición inicial del elemento en el documento.<br>Esta propiedad está disponible en solo lectura. |
| Posición final | Integro | Posición final del elemento en el documento.<br>Esta propiedad está disponible en solo lectura. |
| TableOfContents | [docTableOfContents](../WDLang1/1000024893.md) variable | Características de la tabla del contenido del documento. |
| texto | [docElementText](../WDLang1/1000022493.md) variable | Características de texto (sólo se pueden utilizar para un Elemento de texto). |
| escribir a máquina | Integro | Tipo de elemento correspondiente a una de las siguientes constantes: <br><br>- *typeDocElementCell*: Elemento de la célula de la mesa.<br><br>- *typeDocElementDocument*: Elemento de documento.<br><br>- *typeDocElement Header*: Elemento de cabecera.<br><br>- *typeDocElementFormula*: Formula element. <br><br>- *typeDocElementImage*: Elemento de imagen.<br><br>- *typeDocElementDesconocido*: Elemento no reconocido. <br><br>- *typeDocElementLink*: Elemento de enlace.<br><br>- *typeDocElementPárrafo*: Elemento de párrafo sin texto (texto vacío).<br><br>- *typeDocElementFooter*: Elemento de pie de página.<br><br>- *tipoDocumentoTabla de Contenidos*: Elemento de la tabla de contenido.<br><br>- *typeDocElementText*: Elemento de texto.<br><br>- *typeDocElementTextArea*: Elemento de área de texto. <br><br><br><br><br>Esta propiedad está disponible en solo lectura. |
| ÁreaDeTexto | [docElementTextArea](../WDLang1/1000023420.md) variable | Características del área de texto (sólo se puede utilizar para un elemento de Área de texto). |


**Observación**: Estas propiedades se pueden utilizar con una de las siguientes sintaxis:

- &lt;Nombre de variable&gt;..&lt;Nombre de propiedad&gt;

- &lt;Nombre de variable&gt;.&lt;Nombre de propiedad&gt;

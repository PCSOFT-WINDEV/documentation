


## InAutomaticTestMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si se está ejecutando o no una prueba automatizada.

**Observación**: Las pruebas automatizadas solo se pueden ejecutar en ventanas, clases o procedimientos.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InAutomaticTestMode() = True THEN 
	Trace("CustomerNum variable: " + CustomerNum)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InAutomaticTestMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se está realizando una prueba automatizada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000215"></a>

La función **InAutomaticTestMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InAutomaticTestMode** pueden compartirse entre varios productos sin generar errores de compilación.




## TableTypeItem (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un elemento en un árbol tabla y devuelve su tipo (nodo u hoja) si existe el elemento.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Type of element on row 3 
// in the "TVT_RecipeTable" TreeView Table control
IF TableTypeItem(TVT_RecipeTable, 3) = taNode THEN
	// Expand this node
	TableExpand(TVT_RecipeTable, 3)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Comprobar la existencia de un elemento especificando el número de fila

`<Result> = TableTypeItem(<TreeView Table control> [, <Row number>])`
---

**`<Result>: (Constante)`**

Tipo de elemento: 



|   |   |
| --- | --- |
|  *taError* | El elemento especificado no existe. |
| *taLeaf* | El elemento especificado es una hoja. |
| *taNode* | El elemento especificado es un nodo. |



**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Row number>: (Entero opcional)`**

Número de la fila que contiene el elemento a utilizar. Si no se especifica este parámetro, se devuelve el tipo de elemento que se encuentra en la fila actual..  


<a name="SYNTAX2"></a>

### Comprobar la existencia de un elemento especificando su nombre

`<Result> = TableTypeItem(<TreeView Table control> [, <Element name>])`
---

**`<Result>: (Constante)`**

 Tipo de elemento: 



|   |   |
| --- | --- |
|  *taError* | El elemento especificado no existe. |
| *taLeaf* | El elemento especificado es una hoja. |
| *taNode* | El elemento especificado es un nodo. |



**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Element name>: (Cadena de caracteres opcional)`**

Nombre del elemento a ensayar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Element name>"
```
Se produce un error de WLanguage si este parámetro no corresponde a un elemento existente.

Si no se especifica este parámetro, se devuelve el tipo de elemento que se encuentra en la fila actual..


## MultimediaCerrar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el archivo multimedia que se está reproduciendo en la control Multimedia. Se libera la memoria asignada al archivo multimedia.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"

MultimediaPlay(MM_Multimedia1, MyFile)
...
MultimediaClose(MM_Multimedia1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MultimediaClose(<Name of Multimedia control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el archivo reproducido en el control Multimedia se cierra correctamente,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).






**`<Name of Multimedia control>: (Cadena de caracteres)`**

Nombre del Multimedia control que se va a utilizar.

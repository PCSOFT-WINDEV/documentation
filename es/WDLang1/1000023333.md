


## SpreadsheetSavePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra: 

- la posición de la barra de desplazamiento en una control Hoja de cálculo para la hoja de trabajo actual,

- la hoja de trabajo actual en el control Hoja de cálculo,

- los elementos seleccionados en el control Hoja de cálculo para la hoja de trabajo actual. 




Estos elementos pueden ser restaurados por [Hoja de cálculoRestaurarPosición y Selección](../WDLang1/1000023334.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Let MySave = SpreadsheetRestorePositionAndSelection(PSHEET_Spreadsheet1) 

// Displays the B1 cell in bold
SpreadsheetSelectPlus(PSHEET_Spreadsheet, "B1")
SpreadsheetStyleSelection(PSHEET_Spreadsheet, psheetBold, True)

// Restores the position/selection 
SpreadsheetRestorePositionAndSelection(PSHEET_Spreadsheet1, MySave)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetSavePositionAndSelection(<Spreadsheet control>)`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [Hoja de cálculoRestaurarPosición y Selección](../WDLang1/1000023334.md). 

- Cadena vacía ("") si el control Hoja de cálculo está vacío. 




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

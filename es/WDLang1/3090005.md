


## Funciones USB
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los dispositivos USB:



|   |   |
| --- | --- |
| [USBDetectRemovableStorage](../WDLang1/3090007.md) | Detecta si se ha añadido o eliminado una unidad de almacenamiento extraíble (CD, llave USB, cámara USB,...). |
| [USBDisconnect](../WDLang1/3090001.md) | Desconecta o expulsa un dispositivo USB. |
| [USBFind](../WDLang1/3090004.md) | Encuentra un dispositivo USB según las palabras clave. |
| [USBListDevice](../WDLang1/3090002.md) | Devuelve la lista de dispositivos USB conectados actualmente al ordenador actual. |
| [USBProperty](../WDLang1/3090003.md) | Recupera el valor de un Property para un dispositivo USB. |


## ClipboardFormat( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el formato de la información que se encuentra en el portapapeles.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF ClipboardFormat(cfText) = True THEN
	BTN_Paste.State = Active
ELSE
	BTN_Paste.State = Grayed
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ClipboardFormat(<Sought Format>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el formato solicitado está disponible,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Sought Format>: (Constante de tipo Integer)`**

Formato encontrado en el portapapeles:


|   |   |
| --- | --- |
| *cfBitmap* | Formato Image |
| *cfRtf* | Formato RTF |
| *cfText* | Formato de texto |
| *cfUnicodeText* | Formato de texto Unicode. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Lectura del portapapeles: 

	- [Portapapeles](../WDLang1/3063004.md) lee el texto o la información de Image del portapapeles de Windows. 

	- [PortapapelesRTF](../WDLang1/1000017200.md) lee la información de texto RTF que se encuentra en el portapapeles de Windows (el contenido del portapapeles está en RTF).

	- [PortapapelesUnicode](../WDLang1/1000018858.md) lee la información de texto Unicode del portapapeles. 




- [ToClipboard](../WDLang1/3063005.md) escribe en el portapapeles. [ToClipboardRTF](../WDLang1/1000017199.md) escribe texto RTF en el portapapeles.

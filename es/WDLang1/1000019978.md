


## <Organization Chart>Modificar (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un elemento en una control Organigrama.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Syntax 1: Modifying the title only
Boss is int 
Boss = ORG_MyOrganizationChart.Add("Vince Boss", "big boss", "Goldbar.gif")

ORG_MyOrganizationChart.Modify(1, "Tim C.", "CEO")
```
<a name="Example2"></a>

```wl
// Syntax 2: Modifying the element
ElementCEO is OrgElement
ElementCEO.Title = "Big Boss"
ORG_EXECUTIVE.Add(ElementCEO)

ElementCEO.Title = "Tim C."
ElementCEO..Image = "Goldbar.gif"
ORG_EXECUTIVE.Modify(1, ElementCEO)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Modificar un elemento que se encuentra en una control Organigrama especificando sus características

`<Organization Chart control>.Modifies(<Subscript> , <Title> [, <Content> [, <Image>]])`
---

**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama donde se modificará el elemento.

**`<Subscript>: (Integro)`**

Índice del elemento a modificar. Este índice fue devuelto por [&lt;Organigrama&gt;.add](../WDLang1/1000019975.md).

**`<Title>: (Cadena de caracteres)`**

Nuevo título del elemento.

**`<Content>: (Cadena de caracteres opcional)`**

Nuevo contenido del elemento. 
Si no se especifica este parámetro, el contenido existente no se modificará..

**`<Image>: (Cadena de caracteres opcional)`**

Nueva imagen associated con el elemento. Esta Image se muestra a la izquierda del título del elemento. Este parámetro corresponde al nombre y a la ruta completa (o relativa) de Image. Se puede utilizar una ruta de tipo UNC. 
Si no se especifica este parámetro, el Image existente no se modificará.


<a name="SYNTAX2"></a>

### Modificar un elemento encontrado en un control Organigrama (usando el tipo OrgElement)

`<Organization Chart control>.Modifies(<Subscript> , <Element>)`
---

**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama donde se modificará el elemento.

**`<Subscript>: (Integro)`**

Índice del elemento a modificar. Este índice fue devuelto por [&lt;Organigrama&gt;.add](../WDLang1/1000019975.md).

**`<Element>: (OrgElement Variable)`**

Nombre de la [OrgElement](../WDLang1/1000019713.md) Variable que describe las características completas del elemento a modificar.

## Observaciones


### Utilizando el tipo OrgElement (2ª sintaxis)
<a name="utilizando_tipo_orgelement_2ª_sintaxis_ELTPARAGRAPHE000227"></a>

El elemento en el índice &lt;Índice&gt; es reemplazado por &lt;Elemento&gt;. En este caso, la función **<Organization Chart>Modificar** es equivalente a la siguiente sintaxis:


```txt
OrganizationChart_Control[Index] = Element
```

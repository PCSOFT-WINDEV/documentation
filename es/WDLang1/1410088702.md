
## CapitalizeEachWord (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pone en mayúsculas la primera letra de todas las palabras de una cadena.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString is string = CapitalizeEachWord("first letter of each word in uppercase")
// MyString will contain "First Letter Of Each Word In Uppercase"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CapitalizeEachWord(<String to convert> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Cadena con la primera letra de cada palabra en mayúsculas.

**`<String to convert>: (Cadena de caracteres)`**

Cadena de caracteres a manipular.
Observación: Todos los caracteres que no sean numéricos o alfabéticos se consideran separadores de palabras.

**`<Options>: (Entero opcional)`**

Opciones para convertir la primera letra de cada palabra en mayúsculas: 



|   |   |
| --- | --- |
| *ccNormal*<br>(Valor predeterminado) | Las tildes se conservan al convertir las letras en mayúsculas. |
| *ccIgnoreAccent* | La primera letra de cada palabra se convierte en mayúsculas sin tildes. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Función **CapitalizeEachWord** y Unicode
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000049"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **CapitalizeEachWord** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).


## Errores DDE
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000414"></a>
Cuando una función DDE falla (devuelve False, por ejemplo), [ErrorInfo](../WDLang1/3013008.md) devuelve el número de error correspondiente. Tienes la capacidad de Process el error y de reintentar la operación.

Esta página presenta los principales errores de DDE que pueden ocurrir. Para cada error, usted encontrará:

- su número,

- las funciones DDE afectadas por el error,

- los detalles del error.

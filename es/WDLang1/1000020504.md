


## TableChildCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de hijos directos de un elemento de una control Tabla TreeView.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Count the number of desserts found in the restaurant menu
nNumberDesserts is int 
nNumberDesserts = TableChildCount(TVT_Menu, "Desserts")
Info("There are: " + nNumberDesserts + " on the menu.")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Los hijos de una fila identificados por su número

`<Result> = TableChildCount(<TreeView Table control> , <Row number>)`
---

**`<Result>: (Integro)`**



- Número de hijos del elemento.

- -1 si el número de la fila especificada no existe.  




**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular. 

**`<Row number>: (Integro)`**

Número de fila para los que se calculará el número de elementos hijo. 


<a name="SYNTAX2"></a>

### Hijos de un fila identificado por su ruta

`<Result> = TableChildCount(<TreeView Table control> , <Branch name>)`
---

**`<Result>: (Integro)`**

Número de hijos para el elemento

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular. 

**`<Branch name>: (Cadena de caracteres)`**

Nombre de la rama para la que se debe calcular el número de elementos hijo.. Este parámetro tiene el siguiente formato: 

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```
Si este parámetro no corresponde a una rama existente, se produce un error WLanguage.

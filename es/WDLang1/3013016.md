<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000155"></a>
Las aplicaciones que usted desarrolla pueden reaccionar a la voz del usuario. Por ejemplo, algunas acciones (Scrolling a tabla, cerrar ventanas, seleccionar una casilla de verificación o hacer clic en un botón) se apoyan automáticamente.

Todos los comandos estándar de Windows son compatibles.

También puede añadir sus propios comandos de voz a sus aplicaciones. Por ejemplo, diga "Buscar el formulario del Sr. John Smith" y se llevará a cabo la acción correspondiente.

## ¿Cómo proceder?
<a name="¿como_proceder_ELTTEXTE000179"></a>


### Condiciones necesarias
<a name="condiciones_necesarias_ELTPARAGRAPHE000037"></a>

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar los comandos de voz en sus aplicaciones, su ordenador debe estar equipado con:

- Windows Vista (o posterior).

- un programa de reconocimiento de voz (Dragon Naturally Speaking por ejemplo, pruebas realizadas con la versión 9 "Preferred" o "standard").





<a name="NOTE2_2"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) 

### Uso de los comandos de voz en la aplicación
<a name="uso_los_comandos_voz_aplicacion_ELTPARAGRAPHE000065"></a>

Para usar los comandos de voz en la aplicación: 

1. Active el programa de reconocimiento de voz.

2. Dicte los subtítulos de las opciones del menú o de los botones que desee utilizar.



<a name="NOTE2_3"></a>


### Proponer opciones adicionales o macrocomandos
<a name="proponer_opciones_adicionales_macrocomandos_ELTPARAGRAPHE000075"></a>

Los procedimientos o funciones pueden ser associated con un comando de voz. Lo único que hay que hacer es asociar la Procedure con el comando a través de [SpeechRecognitionAddCommand](../WDLang1/1000020014.md). Este Procedure es relacionado al Procedure para la ventana actual.

En la Procedure, la frase que desencadenó el comando de voz puede ser recuperada. Esto le permite aislar la información solicitada (nombre del cliente, nombre de la ciudad,...).

Tiene la posibilidad de eliminar esta asociación con [SpeechRecognitionDeleteCommand](../WDLang1/1000020017.md).

Ejemplo:


```wl
// Start the "Find" procedure if the voice command is recognized
nVoiceID is int 
nVoiceID = SpeechRecognitionAddCommand("Find", "Finds the form of")
```

```wl
PROCEDURE Find(Param1)
sCustomer is string = Param1[[21 TO ]]
HReadSeekFirst(sCustomer, FIRSTNAMELASTNAME, sCustomer)
```

### Desactivar el reconocimiento de voz
<a name="desactivar_reconocimiento_voz_ELTPARAGRAPHE000099"></a>

El reconocimiento de voz se habilita automáticamente por Default.

Para desactivar el reconocimiento de voz en su aplicación, utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafSpeechRecognition*.


## CameraStopDecoding (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la decodificación de códigos de barras en un control Cámara.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops decoding bar codes in the Camera control
CameraStopDecoding(CAM_Camera1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`CameraStopDecoding(<Camera control>)`
---

**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si el control Cámara está configurado (en la ventana de descripción) para permitir la decodificación de los códigos de barras, la decodificación inicia automáticamente cuando se abre la ventana. Para evitar este comportamiento, llame a la función **CameraStopDecoding** en el evento "Inicialización" del control Cámara.

- Cuando se detiene la decodificación del código de barras, el visor deja de aparecer en el control Cámara.

- Si el control Cámara no está configurado (en la ventana de descripción) para soportar la decodificación de códigos de barras o si no hay una decodificación en curso, la función no tiene ningún efecto.

- Esta función es equivalente a la función [VideoStopDecoding](../WDLang1/1000022164.md).

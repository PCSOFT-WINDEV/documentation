
## CursorDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra u oculta el cursor del ratón. 

Observación: El alfombrado del ratón (utilizado durante la entrada) no se modifica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- "Entry" event associated with the edit control
// Hide the mouse cursor (the default arrow)
// when the edit control has focus
CursorDisplay(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`CursorDisplay(<Displayed cursor>)`
---

**`<Displayed cursor>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se debe mostrar el cursor del ratón,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **CursorDisplay** no tiene ningún efecto si no hay un ratón instalado.

- Esta función sólo afecta al cursor de la aplicación actual.: las ventanas de otras aplicaciones mantienen el estado de visualización de su cursor.

- El cursor, incluso oculto, sigue siendo Active: siempre puedes hacer clic y mover el ratón.

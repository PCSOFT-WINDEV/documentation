
## CompleteDir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un separador de directorios ("\\" o "/" según la plataforma de ejecución) al final de una cadena, si es necesario.

Esta función es útil para construir nombres de archivos completos cuando se desconoce el formato de la ruta (por ejemplo, cuando el usuario introduce una ruta de archivo).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
Res = CompleteDir(FilePath) + "Document.Doc"
Res = CompleteDir("c:\temp")           // Returns "c:\temp\"
Res = CompleteDir("c:\temp\")          // Returns "c:\temp\"
Res = CompleteDir("\CCS\\Dir2\user\")  // Returns "\CCS\\Dir2\user\"
Res = CompleteDir("")                  // Returns ""
```


<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) 
```wl
Res = CompleteDir(" /root/file ")      // Returns "/root/file/"
Res = CompleteDir("")                  // Returns ""
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CompleteDir(<String to check>)`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena que termina con un separador de directorios. 

- Cadena vacía si &lt;Cadena a analizar&gt; corresponde a una cadena vacía ("").




**`<String to check>: (Cadena de caracteres)`**

Cadena a analizar y completar, si es necesario. Esta cadena de caracteres no se modifica. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Espacios
<a name="espacios_ELTPARAGRAPHE000078"></a>

**Atención**: Los espacios al inicio y al final de la &lt;Cadena a analizar&gt; se eliminan (igual que la función [NoSpace](../WDLang1/3024034.md)).
<a name="NOTE0_2"></a>


### Función CompleteDir y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000088"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **CompleteDir** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

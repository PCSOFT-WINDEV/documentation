
## Funciones para la gestión de los controles del Programador (sintaxis del prefijo)
			

Las siguientes funciones se utilizan para gestionar los controles del Programador (sintaxis del prefijo):



|   |   |
| --- | --- |
| [&lt;AddAppointment](../WDLang1/1000022904.md) | Añade una nueva Appointment a una control Planificador. |
| [&lt;AddResource](../WDLang1/1000022905.md) | Añade un nuevo Resource en un control Planificador. |
| [&lt;DeleteAppointment](../WDLang1/1000022912.md) | Borra una Appointment de una control Planificador. |
| [&lt;DisplayNextPeriod](../WDLang1/1000022902.md) | Mueve el área de tiempo que se muestra en el área siguiente (por ejemplo, las próximas semanas) en un control Planificador.  |
| [&lt;DisplayPreviousPeriod](../WDLang1/1000022901.md) | Desplaza el área de tiempo mostrada sobre el área anterior (semanas anteriores, por ejemplo) en una control Planificador. |
| [&lt;InfoXY](../WDLang1/1000022908.md) | Devuelve el Appointment, el Resource o la fecha encontrada en un puesto determinado en un control Planificador. |
| [&lt;Lista de Citas](../WDLang1/1000022909.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Planificador, <br><br>- todas las citas encontradas en un control Planificador,<br><br>- un Appointment específico (seleccionado o suspendido). <br><br><br> |
| [&lt;PositionDateTime](../WDLang1/1000022920.md) | Muestra un control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que aparece en un control Planificador. |
| [&lt;PositionResource](../WDLang1/1000022921.md) | Muestra una Resource dada en una control Planificador o devuelve la posición de la primera Resource visible en una control Planificador. |
| [&lt;Programador&gt;.DeleteAll](../WDLang1/1000022914.md) | Borra todas las citas y recursos de un control Planificador. |
| [&lt;Scheduler&gt;.ChangeMode](../WDLang1/1000022906.md) | Modifica el período (es decir, el número de días) que muestra un control Planificador. |
| [&lt;Scheduler&gt;.DateVisible](../WDLang1/1000022918.md) | Hace visible un día o Invisible en una control Planificador. |
| [&lt;Scheduler&gt;.DeleteResource](../WDLang1/1000022913.md) | Borra una Resource de una control Planificador. |
| [&lt;SelectMinus](../WDLang1/1000022910.md) | Deselecciona un Appointment de un control Planificador. |
| [&lt;SelectPlus](../WDLang1/1000022911.md) | Selecciona un Appointment en un control Planificador. |
| [&lt;TimeDisplayed](../WDLang1/1000022907.md) | Define el intervalo de tiempo que muestra control Planificador en el modo Día o en el modo Semana.. |
| [&lt;Visualización](../WDLang1/1000022900.md) | Reinicia el contenido de un control Planificador desde su fuente de datos (archivo HFSQL por ejemplo). |

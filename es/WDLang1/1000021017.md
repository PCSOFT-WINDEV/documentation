


## ArrayShuffle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Baraja los elementos de un WLanguage array.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize shuffle
InitRandom()

// Fill the array
PlayList is array of strings
...

// Shuffle the array elements
ArrayShuffle(PlayList)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ArrayMix(<WLanguage array>)`
---

**`<WLanguage array>: (Variable de tipo Array)`**

WLanguage array para usar. Esta array puede contener de 1 a N dimensiones.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La barajadura del elemento array se basa en la generación de números aleatorios. Debe utilizar [InitRandom](../WDLang1/3013030.md) para reiniciar la baraja.

- Esta función no se puede utilizar en la función asociativa arrays. 

- Si la array incluye varias dimensiones, la barajadura se realiza a través de las diferentes dimensiones.

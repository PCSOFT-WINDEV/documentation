
## MCIErr( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número del último error causado por una función MCI (Media control Interface).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number for the last error
ResError = MCIErr()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIErr()`
---

**`<Result>: (Integro)`**

Número del último error causado por una función MCI.

## Componente:
wd280std.dll


## ChronoValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el tiempo transcurrido (en milisegundos) desde que se llamó a la función [ChronoStart](../WDLang1/3027031.md). El cronómetro no ha detenido.

Para detener el cronómetro, utilice la función [ChronoEnd](../WDLang1/3027032.md).

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Observación: Esta función es equivalente a la propiedad **Value** de las variables de tipo [Chrono](../WDLang1/1000023560.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DurationMyFunction1 is int
DurationMyFunction2 is int 
TotalDuration is int
// Start the stopwatch
ChronoStart()
MyFunction1()
DurationMyFunction1 = ChronoEnd()
Info("The MyFunction1() process lasted " + DurationMyFunction1 + " milliseconds")

ChronoStart()
MyFunction2()
DurationMyFunction2 = ChronoEnd()
Info("The MyFunction2() process lasted " + DurationMyFunction2 + " milliseconds")

TotalDuration = DurationMyFunction1 + DurationMyFunction2
Info("The total process lasted " + TotalDuration + " milliseconds")
```

```wl
MyFunction2()

Trace("The total process lasted (including the display) " + TotalDuration + " milliseconds")
```

## Sintaxis

### Indicar el valor de un cronometraje iniciado con un número de cronómetro

`<Result> = ChronoValue([<Stopwatch number>])`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el número de cronómetro especificado.  



**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro utilizado. Si no se especifica este parámetro, el valor predeterminado es 1.

### Indicar el valor de un cronometraje iniciado con una variable de tipo Chrono

`<Result> = ChronoValue(<Stopwatch>)`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el cronómetro especificado.  

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.

## Observaciones




### Varios
<a name="varios_ELTPARAGRAPHE000149"></a>

Después de llamar a la función [ChronoEnd](../WDLang1/3027032.md), todas las llamadas a la función **ChronoValue** devolverán el valor del cronómetro cuando se llamó a [ChronoEnd](../WDLang1/3027032.md).
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000164"></a>

Si no se llama a la función [ChronoStart](../WDLang1/3027031.md) antes de la función **ChronoValue**, se produce un error WLanguage (error de tipo "No se ha iniciado el cronómetro").
<a name="NOTE0_3"></a>




### Hilos y tareas paralelas
<a name="hilos_tareas_paralelas_ELTPARAGRAPHE000177"></a>

Cada hilo (y cada tarea paralela) tiene sus propios cronómetros. 

Dos hilos distintos (o dos tareas paralelas distintas) pueden realizar cronometrajes separados utilizando el mismo número de cronómetro. 

El error WLanguage "No se ha iniciado el cronómetro" solo se mostrará si no se llama a las funciones [ChronoStart](../WDLang1/3027031.md) y **ChronoValue** en el mismo hilo.

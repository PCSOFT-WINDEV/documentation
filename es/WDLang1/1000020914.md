


## Funciones Diagrama de Gantt
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular una columna de diagrama de Gantt que se encuentra en control Tabla (o en un control Tabla TreeView): 



|   |   |
| --- | --- |
| [GanttAddHoliday](../WDLang1/1000021116.md) | Declara un nuevo período de vacaciones para un fila en una columna del diagrama de Gantt que se encuentra en una Tabla (o Tabla de Vista de Árbol) control. |
| [GanttAddLink](../WDLang1/1000021075.md) | Crea un enlace "End to Start" entre dos tareas mostradas en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |
| [GanttAddTask](../WDLang1/1000020904.md) | Añade una tarea en una columna del Diagrama de Gantt en una Tabla o control Tabla TreeView. |
| [GanttChangeMode](../WDLang1/1000021030.md) | Modifica el período (que significa el número de días) que se muestra en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |
| [GanttDateTimePosition](../WDLang1/1000021036.md) | Configura el primer día y la primera vez que se muestra en una columna de diagrama de Gantt (que se encuentra en una tabla o control Tabla TreeView). |
| [GanttDeleteAll](../WDLang1/1000020924.md) | Borra todas las tareas de una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |
| [GanttDeleteLink](../WDLang1/1000021080.md) | Elimina los enlaces de una columna del diagrama de Gantt en una tabla o control Tabla TreeView. |
| [GanttDeleteTask](../WDLang1/1000020923.md) | Elimina una tarea de una columna del diagrama de Gantt en una Tabla o control Tabla TreeView. |
| [GanttGetTask](../WDLang1/1000021026.md) | Recupera una tarea de su identificador en una columna del diagrama de Gantt (que se encuentra en una Tabla o control Tabla TreeView). |
| [GanttListLink](../WDLang1/1000021079.md) | Enumera los enlaces que se encuentran en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |
| [GanttListTask](../WDLang1/1000021073.md) | Devuelve: <br><br>- la lista de tareas incluidas entre dos fechas determinadas en una columna del diagrama de Gantt, <br><br>- todas las tareas que se encuentran en una columna del diagrama de Gantt,<br><br>- una tarea específica (seleccionada o suspendida). <br><br><br> |
| [GanttOrganizeTask](../WDLang1/1000020890.md) | Reorganiza las tareas de un diagrama de Gantt según las restricciones (dependencias y recursos).. |
| [GanttTimeDisplayed](../WDLang1/1000021081.md) | Define el rango de tiempo mostrado en una columna de una Tabla o control Tabla TreeView de tipo Diagrama de Gantt. |
| [GanttWorkingWeekDay](../WDLang1/1000021035.md) | Indica si un día de la semana es un día laborable en una columna del diagrama de Gantt en una Tabla o control Tabla TreeView. |
| [Hora de trabajo de Gantt](../WDLang1/1000021117.md) | Sobrecarga las horas de trabajo de una fila en una columna de Diagrama de Gantt que se encuentra en una Tabla (o Tabla TreeView) control. |



Se pueden utilizar dos tipos de variables para manipular los elementos que se encuentran en los diagramas de Gantt: 



|   |   |
| --- | --- |
| [GanttLink](../WDLang1/1000021076.md) | El tipo **GanttLink** permite definir las características avanzadas de una relación entre dos tareas de un diagrama de Gantt. |
| [GanttTask](../WDLang1/1000020881.md) | El tipo **GanttTask** permite definir las características avanzadas de una tarea para un diagrama de Gantt. |

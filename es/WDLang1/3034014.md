
## ExceptionPropagate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Propaga una excepción. Esta función debe ejecutarse en una excepción Process y se utiliza para reiniciar el mecanismo de excepción Process.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ExceptionPropagate([<Message>])`
---

**`<Message>: (Cadena de caracteres opcional)`**

Mensaje adicional que se puede añadir al error.

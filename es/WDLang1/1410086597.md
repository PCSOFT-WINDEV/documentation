


## Funciones de la cámara
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la control Cámara: 




|   |   |
| --- | --- |
| [CameraChangeDriver](../WDLang1/1000022451.md) | Conecta la "Cámara" control a un conductor específico para la captura de vídeo. |
| [CameraConfigure](../WDLang1/1000022452.md) | Permite configurar el controlador de captura de vídeo. |
| [CameraDisplay](../WDLang1/1410086595.md) | Muestra el video capturado por una cámara web o por la cámara de un dispositivo móvil en un control Cámara.<br><br> |
| [CameraGenerateThumbnail](../WDLang1/1000022456.md) | Genera una miniatura de la foto o el vídeo especificado. |
| [CameraListDriver](../WDLang1/1000022457.md) | Enumera los controladores de captura de vídeo instalados en el ordenador actual. |
| [CameraPhoto](../WDLang1/1410087225.md) | Captura una foto a través de un control Cámara en una aplicación Android o iOS. |
| [CameraRequestPermission](../WDLang1/1000022453.md) | Solicita a Permission que utilice la cámara del dispositivo. |
| [CameraRunApp](../WDLang1/1000022470.md) | Inicia la aplicación de la cámara nativa del dispositivo para Record un video o para tomar una foto. |
| [CameraStartDecoding](../WDLang1/1000022454.md) | Inicia la descodificación de códigos de barras en una control Cámara. |
| [CameraStatus](../WDLang1/1000022455.md) | Devuelve el estado de un control Cámara.<br><br> |
| [CameraStop](../WDLang1/1410086596.md) | Detiene la secuencia de vista previa o grabación de un video en un control Cámara.<br><br> |
| [CameraStopDecoding](../WDLang1/1000022449.md) | Detiene la decodificación de códigos de barras en un control Cámara. |
| [CameraVideoStart](../WDLang1/1410087227.md) | Inicia la grabación de video a través de un control Cámara en una aplicación Android o iOS. |
| [CameraVideoStop](../WDLang1/1410087229.md) | Detiene la grabación de video iniciada previamente con la función [CameraVideoStart](../WDLang1/1410087227.md). |

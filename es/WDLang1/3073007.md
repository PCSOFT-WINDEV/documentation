
## SysWinHandle( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la manipular de una ventana identificada por su título. Esta ventana no fue creada por WINDEV.

**Observación**: Para conocer la manipular de una ventana WINDEV, utilice [Maneje](../WDLang1/3015005.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the handle of the "Sales 2011 - Microsoft Word.DOC" window
ResWinHandle is int
ResWinHandle = SysWinHandle("Sales 2011 - Microsoft Word.DOC")

IF ResWinHandle = Null THEN Trace("Window not found")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysWinHandle(<Window title>)`
---

**`<Result>: (Integro)`**



- Manipulación de la ventana especificada. 

- Constante nula o 0 si no se encontró la ventana.




**`<Window title>: (Cadena de caracteres)`**

Título de la ventana a utilizar. Este parámetro debe ser idéntico al título que se muestra en la barra de título de la ventana..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Ventana manipular
<a name="ventana_manipular_ELTPARAGRAPHE000039"></a>

[SysWinShow](../WDLang1/3073003.md), [Título de SysWin](../WDLang1/3073008.md) y [Mango SysValidHandle](../WDLang1/3073009.md) utilizan el manipular de las ventanas que no fueron creadas por WINDEV.

Para manipular las ventanas creadas por WINDEV, recomendamos que utilice las siguientes funciones: las funciones [restituir](../WDLang1/3038042.md), [Maximizar](../WDLang1/3038001.md), [WinStatus](../WDLang1/3038030.md), [Iconizar](../WDLang1/3038015.md)...

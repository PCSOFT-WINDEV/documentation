


## HTMLValidateForm (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Valida un formulario en una página HTML mostrada en una pantalla HTML control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLControlValue(HTM_Display, "LOGIN", sLogin)
HTMLControlValue(HTM_Display, "PASSWD", sPassword)
HTMLValidateForm(HTM_Display)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLValidateForm(<HTML Display control> [, <Form>])`
---

**`<HTML Display control>: (Nombre del control)`**

Nombre de la pantalla HTML utilizada por control. 

**`<Form>: (Cadena de caracteres opcional)`**

Nombre del formulario a validar. Si no se especifica este parámetro, se validará el primer formulario de la página que aparece en la control.

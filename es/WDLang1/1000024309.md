


## &lt;Array&gt;.Insert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un elemento en una posición determinada:

- en una array unidimensional.

- en una array asociativa.

- en una propiedad array avanzada (array de eventos gglCalendar, etc.).

- en una lista WLanguage. 




Esta función también permite concatenar dos arrays o dos listas.












<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyArray is array of 2 strings
MyArray.Insert(1, "WINDEV")
MyArray.Insert(2, "WEBDEV")
MyArray.Insert(3, "WINDEV and WEBDEV")
// Display the content of 3rd element
Trace(MyArray[3]) // Displays "WINDEV and WEBDEV"
```
<a name="Example2"></a>

```wl
MyArray is array of 2 strings
MyArray[1] = "WINDEV"
MyArray[2] = "WEBDEV"
MyArray.Insert(3, "WINDEV and WEBDEV")
// Display the content of 3rd element
Trace(MyArray[3]) // Displays "WINDEV and WEBDEV"
```
<a name="Example3"></a>

```wl
// Associative array Last Name - First Name
aaLastNameFirstName is associative array of strings
aaLastNameFirstName.Insert("Smith", "Tommy")
aaLastNameFirstName.Insert("Montgomery", "Julia")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar un elemento en un array o en un array Property avanzado

`<WLanguage array>.Insert(<Insertion index> [, <Element value>])`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Insertion index>: (Integro)`**

Índice donde se insertará el elemento en la array. A partir de este índice, todos los elementos se desplazarán en 1 elemento. 
Si &lt;índice de inserción&gt; es igual al número de elementos de la array +1, el elemento se añade al final de la array (equivalente a [&lt;Array&gt;.Add](../WDLang1/1000024299.md)). 
Se produce un error de WLanguage si &lt;índice de inserción&gt; es mayor que el número de elementos del array +1.

**`<Element value>: (Cualquier tipo, opcional)`**

Elemento que se insertará en el array especificado, en la posición dada.. Si no se especifica este parámetro, el array se amplía con el valor Default del tipo de los otros elementos array.


<a name="SYNTAX2"></a>

### Insertar un elemento en una array asociativa

`<WLanguage array>.Insert(<Element key> , <Element value>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Element key>: (Tipo de clave en la asociación array)`**

Valor de la clave para la que se insertará el elemento. En una array asociativa sin duplicado, se produce un error de WLanguage si el elemento ya existe.

**`<Element value>: (Cualquier tipo)`**

Elemento que debe añadirse a la array especificada.


<a name="SYNTAX3"></a>

### Insertar un array en un array

`<WLanguage array>.Insert(<Insertion index> [, <Array to insert>])`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Insertion index>: (Integro)`**

Índice donde se insertará el elemento en la array. A partir de este índice, todos los elementos se desplazarán en 1 elemento. 
Si &lt;índice de inserción&gt; es igual al número de elementos de la array +1, el elemento se añade al final de la array (equivalente a [&lt;Array&gt;.Add](../WDLang1/1000024299.md)). 
Se produce un error de WLanguage si &lt;índice de inserción&gt; es mayor que el número de elementos del array +1.

**`<Array to insert>: (Opcional array)`**

Nombre de la Array Variable a insertar en el índice de inserción especificado en &lt;WLanguage array&gt;.


<a name="SYNTAX4"></a>

### Insertar un elemento en una lista

`<WLanguage list>.Insert(<Insertion index> , <Element value>)`
---

**`<WLanguage list>: (jugarreta)`**

Nombre de la Lista Variable a utilizar.

**`<Insertion index>: (Constante de tipo Integer)`**

Índice donde se insertará el elemento en la lista. Se pueden utilizar las siguientes constantes:


|   |   |
| --- | --- |
| *AfterCurrentElement* | El elemento se inserta después del elemento de corriente.<br>Esta constante sólo está disponible para iterar sobre los elementos de la lista.<br>**Atención**: El elemento se leerá en la siguiente iteración en un bucle ascendente. |
| *BeforeCurrentElement* | El elemento se inserta antes del elemento de corriente.<br>Esta constante sólo está disponible para iterar sobre los elementos de la lista.<br>**Atención**: El elemento se leerá en la siguiente iteración en un bucle descendente. |
| *FirstElement* | El elemento se inserta al principio de la lista. |
| *LastElement* | El elemento se inserta al final de la lista. |



**`<Element value>: (Cualquier tipo)`**

Elemento que se insertará en la lista en la posición especificada.


<a name="SYNTAX5"></a>

### Insertar una lista en una lista

`<WLanguage list>.Insert(<Insertion index> , <List to be inserted>)`
---

**`<WLanguage list>: (jugarreta)`**

Nombre de la Lista Variable a utilizar.

**`<Insertion index>: (Constante de tipo Integer)`**

Índice en el que se insertará la lista. Se pueden utilizar las siguientes constantes:


|   |   |
| --- | --- |
| *AfterCurrentElement* | La lista se inserta después del elemento actual.<br>Esta constante sólo está disponible para iterar sobre los elementos de la lista. |
| *BeforeCurrentElement* | La lista se inserta antes del elemento actual.<br>Esta constante sólo está disponible para iterar sobre los elementos de la lista. |
| *FirstElement* | La lista se inserta al principio de &lt;Nombre de lista&gt;. |
| *LastElement* | La lista se inserta al final de &lt;Nombre de lista&gt;. |




**`<List to be inserted>: (jugarreta)`**

Nombre de la lista que se insertará en el índice de inserción especificado.

## Observaciones


### Insertar un elemento en una array
<a name="insertar_elemento_una_array_ELTPARAGRAPHE000453"></a>

Cuando se llama a la función **&lt;Array&gt;.Insert**:

- el array se amplía automáticamente para recibir el nuevo elemento.

- si es necesario, el elemento se convierte en el tipo de los otros elementos del array.




Cuando se declara una array de N elementos, esta array contiene N elementos vacíos. Por ejemplo, el array que se declara a continuación contiene 3 cadenas vacías.


```wl
MyArray is array of 3 strings
```
Los elementos insertados mediante **&lt;Array&gt;.Insert** se insertan automáticamente entre los elementos existentes en el array.

En nuestro ejemplo, la array contendrá 4 elementos una vez realizada la inserción.
<a name="NOTE0_2"></a>


### Agregar un elemento a una propiedad array de tipo avanzado
<a name="agregar_elemento_una_propiedad_array_tipo_avanzado_ELTPARAGRAPHE000477"></a>

Cuando se llama a la función **&lt;Array&gt;.Insert**:

- se debe crear la variable de tipo avanzado.

- el tipo avanzado debe tener un enumerador de tipo de colección modificable.

- el tipo avanzado se amplía automáticamente para recibir los nuevos elementos.

- el elemento se inicializa con el valor pasado como parámetro. Si no se pasa ningún valor en el parámetro, el elemento se inicializa con el valor Default del tipo de los elementos array.



<a name="NOTE0_3"></a>


### Insertar un elemento en una lista
<a name="insertar_elemento_una_lista_ELTPARAGRAPHE000494"></a>

Cuando se llama a la función **&lt;Array&gt;.Insert**:

- la lista se amplía automáticamente para recibir el nuevo elemento.

- si es necesario, el elemento se convierte en el tipo de los otros elementos de la lista.



<a name="NOTE0_4"></a>


### Inserción de una tabla
<a name="insercion_una_tabla_ELTPARAGRAPHE000513"></a>

Cuando se llama a la función **&lt;Array&gt;.Insert**:

- el array se amplía automáticamente para recibir los nuevos elementos. Los elementos del &lt;Nombre del array por insertar&gt; se agregan en la posición especificada en &lt;Nombre del array&gt;.

- los dos arrays deben ser del mismo tipo.

- los arrays deben tener la misma dimensión.

- los valores de las dimensiones de array (excepto la primera) deben ser idénticos.



<a name="NOTE0_5"></a>


### Insertar una lista
<a name="insertar_una_lista_ELTPARAGRAPHE000530"></a>

Cuando se llama a la función **&lt;Array&gt;.Insert**:

- la lista se amplía automáticamente para recibir los nuevos elementos. Los elementos de &lt;Nombre de la lista a insertar&gt; se añaden en la posición especificada.

- ambas listas deben ser del mismo tipo.



<a name="NOTE0_6"></a>


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000549"></a>

Esta función se puede utilizar con las estructuras. En este caso, se debe:

1. Declarar una variable del mismo tipo que la estructura).

2. Inicializar cada miembro.

3. Pasar la variable estructura como parámetro a la función **&lt;Array&gt;.Insert**.




Esta función no puede utilizarse:

- los arrays no creados

- arrays fijos.



<a name="NOTE0_7"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000570"></a>



- Para añadir un elemento al final de una array unidimensional, utilice [&lt;Array&gt;.Add](../WDLang1/1000024299.md).

- Para agregar un elemento a un array ordenado (respetando el orden), utilice la función [&lt;Array&gt;.AddSorted](../WDLang1/1000024301.md).

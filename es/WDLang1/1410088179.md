
## diagBorder (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagBorder** se utiliza para definir todas las características avanzadas de un borde (o bordes) de una forma utilizada en un Diagram. Las características de esta Border pueden definirse y modificarse mediante diferentes propiedades del lenguaje WL.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).

**PRECAUCIÓN: Este tipo no se puede utilizar directamente. Sólo debe utilizarse a través del tipo [diagShape](../WDLang1/1410088082.md) (o uno de sus tipos derivados).**
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a rectangle shape
MyShape is diagRectangle

// Set the dimensions, position and background color
MyShape.X = 100
MyShape.Y = 100
MyShape.Width = 350
MyShape.Height = 150
MyShape.Background.Color = PastelRed
MyShape.Border.Color = Black
MyShape.Border.Thickness = 2
MyShape.Border.Type = LineDash

// Reference the displayed diagram
MyDiagram is Diagram <- DIAGEDT_MyDiagram

// Add the rectangle shape to the diagram
Add(MyDiagram.Shape, MyShape)
```

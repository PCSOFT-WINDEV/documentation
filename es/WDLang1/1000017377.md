
## ControlEnabled (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Activa una control o un grupo de control en una ventana. Una animación puede ser set en los controles durante esta operación.  

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) No se puede reproducir ninguna animación cuando los controles están activados.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables the GR_Name group of controls
ControlEnabled(GR_Name, animFadeIn, 200)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ControlEnabled(<Control name> [, <Type of animation> [, <Animation duration>]])`
---

**`<Control name>: (Nombre de la control o nombre de un grupo de control s)`**

Nombre de la control o del grupo de control que se va a habilitar.

**`<Type of animation>: (Constante opcional de tipo Integer)`**

Tipo de animación a reproducir:


|   |   |
| --- | --- |
| *animCoverDown* | El nuevo elemento aparece desde arriba y cubre el elemento inicial. |
| *animCoverFromCenter* | El nuevo elemento aparece desde el centro y cubre el elemento inicial. |
| *animCoverFromCorners* | El nuevo elemento aparece por las esquinas y cubre el elemento inicial. |
| *animCoverLeft* | El nuevo elemento aparece por la derecha y cubre el elemento inicial. |
| *animCoverRight* | El nuevo elemento aparece por la izquierda y cubre el elemento inicial. |
| *animCoverUp* | El nuevo elemento aparece desde abajo y cubre el elemento inicial. |
| *animFadeIn* | Animación de fade-in. |
| *animFlipDownwardDiagonal* | Da la vuelta a la diagonal que va de la parte superior izquierda a la inferior derecha. |
| *animFlipUpwardDiagonal* | Da la vuelta a la diagonal que va de la parte superior derecha a la inferior izquierda. |
| *animShrinkAndCoverFromCorners* | El elemento inicial se encoge y el nuevo elemento aparece por las esquinas. |
| *animSlideDown* | Deslizar hacia abajo. |
| *animSlideLeft* | Deslizar hacia la izquierda. |
| *animSlideRight* | Deslizar hacia la derecha. |
| *animSlideUp* | Deslizar hacia arriba. |

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no se tiene en cuenta. No se puede reproducir ninguna animación cuando la control está activada.

**`<Animation duration>: (Entero o duración opcional)`**

Duración de la animación en centésimas de segundo. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, "1cs" o "10 ms"). 


Recomendamos utilizar una duración corta (hasta 1 segundo)..
La aplicación está bloqueada mientras la animación no esté terminada.. Si el usuario hace clic o pulsa una tecla, la animación se detiene y los controles se muestran en su estado final.![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no se tiene en cuenta. No se puede reproducir ninguna animación cuando la control está activada.

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Animación de un grupo de controles
<a name="animacion_grupo_controles_ELTPARAGRAPHE000124"></a>

Si &lt;nombre de control&gt; corresponde a un grupo de control s, la animación se reproduce en el rectángulo definido por el control en la parte superior izquierda y el de la parte inferior derecha.

También se animarán los controles que no pertenecen al grupo de controles pero que se encuentran en esta área.
<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000133"></a>

- **ControlEnabled** no tiene ningún efecto si el control o el grupo de control s no está grayed fuera.

- El [Estado](../Proprietes/2510039.md) Property también puede utilizarse para habilitar controles. Esta Property no puede ser usada para reproducir una animación durante esta operación. En WINDEV Mobile, las propiedades [Estado](../Proprietes/2510039.md) y **ControlEnabled** son equivalentes.

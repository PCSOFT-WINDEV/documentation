
## Gestión de hilos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000132"></a>
WINDEV y WINDEV Mobile proponen varias funciones utilizadas para gestionar hilos de forma avanzada.

Los hilos permiten ejecutar código (o procesos) en paralelo con la aplicación principal. Por lo tanto, se pueden ejecutar varios procesos en segundo plano sin bloquear la aplicación principal.

Los hilos reemplazan algunos tipos de temporizadores (Timers).

Un hilo eficiente espera un evento, como una acción del usuario, un correo electrónico entrante, una llamada telefónica entrante, etc.

**Ejemplos de uso de hilos:**

- Recuperar correos electrónicos en segundo plano mientras se redacta un nuevo correo electrónico, por ejemplo.

- Aplicación de comunicación: gestión de llamadas telefónicas, comunicación mediante sockets, etc.

## Gestión simple y avanzada de hilos
<a name="gestion_simple_avanzada_hilos_ELTTEXTE000156"></a>
WINDEV propone varios métodos para gestionar hilos:

- Ejecutar hilos. 
	Para obtener más información, consulte [Principio de ejecución de hilos](../WDLang1/3077004.md)

- Gestionar los **semáforos** en los hilos, es decir, limitar la ejecución simultánea de un código a uno o más hilos en un momento dado. Un semáforo se puede compartir entre varias aplicaciones. 
	Para obtener más información, consulte [Gestionar los semáforos en hilos: limitar la ejecución simultánea de un código](../WDLang1/3077013.md). 

- Gestionar los **mutex** para limitar la ejecución simultánea de un código a un solo hilo en un momento dado. Un Mutex se puede compartir entre varias aplicaciones. 
	Para obtener más información, consulte [Gestionar la exclusión mutua en hilos](../WDLang1/1000019475.md). 

- Gestionar las **secciones críticas** para limitar la ejecución simultánea de un código a **un hilo** en un momento dado, en una sola aplicación. 

- Gestionar los **eventos** para sincronizar varios hilos.
	 Para obtener más información, consulte [Sincronizar hilos mediante eventos](../WDLang1/3077018.md).

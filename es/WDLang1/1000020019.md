
## TimePicker (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el selector de hora del sistema.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=HeureSelecteur.gif)


**Observación**: Para seleccionar una hora con el selector, le recomendamos utilizar directamente la **máscara de entrada de tipo "Selector de hora"** para el control Campo de entrada de tipo "Hora". Esta máscara de entrada también está disponible para aplicaciones iPhone/iPad. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
h is Time = TimePicker()
```



```wl
StartTime is Time = "0730"
TimeSelected is Time = TimePicker(StartTime, True, "Enter the end time")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TimePicker([<Initial time> [, <24-hour mode> [, <Title>]]])`
---

**`<Result>: (Hora o cadena de caracteres)`**



- Hora seleccionada en el selector,

- Cadena vacía ("") si el usuario cancela la selección




**`<Initial time>: (Hora o cadena de caracteres opcional)`**

Hora que se muestra al abrir el selector. Si no se especifica esta hora, el selector mostrará la hora actual.

**`<24-hour mode>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado) para mostrar la hora en formato de 24 horas.

- <u><u><u><u>False</u></u></u></u> para mostrar la hora en formato AM/PM. 




**`<Title>: (Cadena de caracteres opcional)`**

Título que se mostrará en el selector. Si no se especifica este parámetro, el título será la hora que aparece en el selector en formato de texto.

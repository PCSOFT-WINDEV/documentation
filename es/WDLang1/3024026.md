
## Phonetic (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la transcripción fonética de la cadena pasada como parámetro. Esto permite saber si dos cadenas de caracteres son fonéticamente similares (según la fonética francesa).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Compare two character strings
MyString1 is string = Phonetic("volant") 
MyString2 is string = Phonetic("volent")

IF MyString1 = MyString2 THEN
	Trace("Same phonetic transcription")
ELSE
	Trace("Different phonetic transcription")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Phonetic(<Initial string>)`
---

**`<Result>: (Cadena de caracteres)`**

Transcripción pseudo-fonética de &lt;Cadena inicial&gt;.

**`<Initial string>: (Cadena de caracteres)`**

Cadena a convertir en una transcripción fonética. Esta cadena no se modifica.

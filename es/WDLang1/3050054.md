


## Funciones financieras
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones financieras son las siguientes:



|   |   |
| --- | --- |
| [FinCurrentVal](../WDLang1/3050055.md) | Devuelve el valor actual de la inversión. |
| [FinDecreasingRedemption](../WDLang1/3050068.md) | Calcula el valor de amortización de un bien durante un período de tiempo específico, mediante la fórmula de amortización decreciente de tipo fijo. |
| [FinError](../WDLang1/3050071.md) | Devuelve el número del último error causado por una función Finxxx ([funciones financieras](../WDLang1/3050054.md)). |
| [FinFutureVal](../WDLang1/3050056.md) | Devuelve el valor futuro de una inversión. |
| [FinInterestRate](../WDLang1/3050047.md) | Calcula la tasa de interés para un préstamo durante un período específico de tiempo con pagos fijos. |
| [FinLinearRedemption](../WDLang1/3050046.md) | Calcula el valor de la amortización lineal de un bien durante un período de tiempo específico. |
| [FinNetCurrentVal](../WDLang1/3050053.md) | Devuelve el valor actual neto de la inversión. |
| [FinPaymentNb](../WDLang1/3050041.md) | Devuelve el número de pagos necesarios para amortizar un capital de acuerdo con una tasa determinada. |
| [FinPeriodInterest](../WDLang1/3050042.md) | Calcula para un período determinado el importe de los intereses de un préstamo amortizado mediante pagos periódicos fijos con un tipo de interés fijo. |
| [FinRedemption](../WDLang1/3050061.md) | Crea una matriz de amortización para un préstamo de tipo fijo durante un período de tiempo definido. |
| [FinRepayVal](../WDLang1/3050039.md) | Devuelve el importe de cada pago regular para una inversión con tipo de interés fijo y pago fijo. |


## SpellAddDictionary( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una palabra al diccionario de OpenOffice: 

- para el caso de la aplicación actual. 

- para todas las aplicaciones.




Después de esta adición, la palabra será considerada como correcta por [SpellCheck](../WDLang1/1000018923.md). 

**Observación**: La mayoría de los controles de una aplicación WINDEV ofrecen la posibilidad de utilizar el corrector ortográfico ("Revisar ortografía" en la pestaña "Detalles" del control description). 

**Consejo**: Se recomienda utilizar **SpellAddDictionary** en la "Inicialización" Event del proyecto. Así, el corrector ortográfico de OpenOffice no considerará ciertas palabras como errores, como el nombre de la aplicación, y términos específicos de la aplicación.  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SpellAddDictionary("WINDEV")
SpellAddDictionary("WEBDEV")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SpellAddDictionary(<Word to Add> [, <Range>])`
---

**`<Word to Add>: (Cadena de caracteres)`**

Palabra que debe añadirse al diccionario de Open Office. 
**Atención**: El idioma del diccionario a tener en cuenta debe ser configurado por [ConfigureSpellCheck](../WDLang1/3025052.md).

**`<Range>: (Constante de tipo Integer)`**

Alcance de la adición al diccionario. Este parámetro corresponde a una de las siguientes constantes: 



|   |   |
| --- | --- |
| *spellScopePersistent* | La palabra se añade al diccionario de OpenOffice. <br>Esta constante tiene el mismo efecto que la opción "Añadir al diccionario" del menú contextual de AAF con el corrector ortográfico. |
| *spellScopeProcess*<br>(Valor predeterminado) | La palabra no se añade al diccionario de OpenOffice.<br>Por lo tanto, seguirá mostrándose como "False" si se reinicia la aplicación. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para saber si el corrector ortográfico de OpenOffice está instalado en el ordenador actual, utilice [HechizoDisponible](../WDLang1/1000019019.md).

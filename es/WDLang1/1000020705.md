<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
[Mostrar solo las constantes utilizadas por las sintaxis con prefijo](#" onclick="$('.object').show();$('.not_object').hide();)

[Mostrar solo las constantes utilizadas por las sintaxis sin prefijo](#" onclick="$('.object').hide();$('.not_object').show();)Las siguientes constantes se utilizan para gestionar los controles de la tabla dinámica:

| Constante | Detalles | Función |
| --- | --- | --- |
| **pvtAbsoluteEvolution** | Muestra la evolución entre los períodos en valores absolutos. | [PVTCompareDateRange](../WDLang1/1000024869.md) |
| **pvtColumnHeader** | Maneja los encabezados de las columnas. | [PVTExpandTodos](../WDLang1/1000020623.md), [Colapso de PVTCTodos](../WDLang1/1000020622.md) |
| **pvtPercentageEvolution** | Muestra la evolución entre los períodos en porcentajes. | [PVTCompareDateRange](../WDLang1/1000024869.md) |
| **pvtRowHeader** | Maneja las cabeceras fila. | [PVTExpandTodos](../WDLang1/1000020623.md), [Colapso de PVTCTodos](../WDLang1/1000020622.md) |
| **pvtWithoutEvolution** | No muestra la evolución entre los períodos. | [PVTCompareDateRange](../WDLang1/1000024869.md) |
| **pvtWithUI** | Guarda la configuración de control Tabla dinámica:<br><br>- tamaño de las columnas,<br><br>- tamaño de las filas,<br><br>- Cabeceras de fila colapsadas/expandidas o cabeceras de columna.<br><br><br> | [PVTSave](../WDLang1/1000020471.md) |




Las siguientes constantes se utilizan para gestionar los controles de la tabla dinámica (sintaxis del prefijo):

| **pvtRowHeader** | Maneja las cabeceras fila. | [&lt;Tabla pivotante&gt;.ExpandAll](../WDLang1/1000022183.md), [&lt;Tabla pivotante&gt;.colapsoTodos](../WDLang1/1000022185.md) |
| **pvtWithUI** | Guarda la configuración de control Tabla dinámica:<br><br>- tamaño de las columnas,<br><br>- tamaño de las filas,<br><br>- Cabeceras de fila colapsadas/expandidas o cabeceras de columna.<br><br><br> | [&lt;Tabla pivotante&gt;.Save](../WDLang1/1000022189.md) |

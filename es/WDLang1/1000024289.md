


## <Table>.Delete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un fila de: 

- un control Tabla, 

- un control Tabla TreeView, 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una tabla que se muestra en un control Combo Box.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Observación: Para eliminar una fila con una animación, utilice [&lt;Tabla&gt;.DeleteWithAnimation](../WDLang1/1000024290.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete row #5 from the "TABLE_Product" control
TABLE_Product.Delete(5)
```
<a name="Example2"></a>

```wl
// Delete all selected rows from a Table control
// <Table>.DeleteSelect can also be used
i is int
NbSelection is int = TABLE_MyTable.SelectCount()
FOR i = NbSelection TO 1 STEP -1
	TABLE_MyTable.Delete(TABLE_MyTable.Select(i))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.Delete([<Subscript>])`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Combo Box con tabla.




**`<Subscript>: (Entero opcional)`**

Índice de la fila a eliminar. 
Si no se especifica este parámetro, se borran las líneas seleccionadas.. Si no se especifica este parámetro y no se selecciona ninguna fila, la función **<Table>.Delete** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000277"></a>

La función **<Table>.Delete** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.
	



<a name="NOTE0_2"></a>


### Eliminación de registros en una tabla o control Tabla TreeView basada en un archivo de datos
<a name="eliminacion_registros_una_tabla_control_tabla_treeview_basada_archivo_datos_ELTPARAGRAPHE000314"></a>

La función **<Table>.Delete** elimina el registro correspondiente en el archivo de datos asociado al control. Si los elementos son relacionado para otros archivos de datos, no se realiza ninguna modificación automáticamente en estos archivos de datos de relacionado.
<a name="NOTE0_3"></a>


### Integridad referencial y Tabla o TreeView controles Tabla basado en un archivo de datos
<a name="integridad_referencial_tabla_treeview_controles_tabla_basado_archivo_datos_ELTPARAGRAPHE000326"></a>

Si la gestión de la integridad referencial está activada ([&lt;Variable Connection&gt;.SetIntegrity](../WDLang4/1000023960.md)), se debe llamar a [HErrorIntegrity](../WDLang4/3044086.md) después de **<Table>.Delete** para comprobar la integridad. 

Si se detecta un error de integridad, el fila no se borra y permanece Visible en el control.
<a name="NOTE0_4"></a>


### Eliminación de registros en una control Tabla basada en un archivo de datos y Trigger
<a name="eliminacion_registros_una_control_tabla_basada_archivo_datos_trigger_ELTPARAGRAPHE000348"></a>

Tiene la posibilidad de activar un disparador al borrar un Record de un control Tabla basado en un archivo de datos. Para obtener más información, consulte la documentación de [&lt;Fuente&gt;.DescribeTrigger](../WDLang4/1000024216.md).
<a name="NOTE0_5"></a>


### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000363"></a>

La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> si la eliminación falla. Para get los detalles del error, utilice [ErrorInfo](../WDLang1/3013008.md).  

Ejemplo de casos de error: Intento de eliminar una fila cuyo índice corresponde a una fila no seleccionada en la parte Visible del control Tabla.
<a name="NOTE0_6"></a>


### Bloquear el archivo de datos de relacionado
<a name="bloquear_archivo_datos_relacionado_ELTPARAGRAPHE000385"></a>

Si el archivo de datos está bloqueado en el proceso actual, la función **<Table>.Delete** elimina la fila especificada y desbloquea el archivo de datos.
<a name="NOTE0_7"></a>


### Eliminación de registros en una control Tabla rellenada programáticamente con un filtro "Contiene"
<a name="eliminacion_registros_una_control_tabla_rellenada_programaticamente_con_filtro_contiene_ELTPARAGRAPHE000397"></a>

Cuando se utiliza en una control Tabla rellenada programáticamente con un filtro "Contiene", **<Table>.Delete** muestra los registros que no coinciden con el filtro. Los filtros definidos por los AAF (automatic Application Features) se cancelan.

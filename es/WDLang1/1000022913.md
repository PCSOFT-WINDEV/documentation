


## &lt;Scheduler&gt;.DeleteResource ( Function)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un Resource de un control Planificador. 

**Atención**: Si el control Planificador es relacionado a una fuente de datos, la eliminación se realiza en el control solamente: los nombramientos de relacionado a Resource no se borran de la fuente de datos. 



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds an appointment for tonight
MyTitle is string
StartAPT is DateTime
EndAPT is DateTime
NbResource is int

MyTitle = "Sales meeting"
StartAPT = DateSys() + "17000000"
EndAPT = DateSys() + "17300000"

ResourceNum = SCH_Schedule1.AddResource("Florence" + gStoredValue("Flo"))
SCH_Scheduler1.AddAppointment("Flo", MyTitle, StartAPT)

MyTitle = "Meeting 2"
StartAPT = DateSys() + "17300000"
EndAPT = DateSys() + "18000000"
SCH_Schedule1.AddResource("Thomas" + gStoredValue("Tom"))
SCH_Scheduler1.AddAppointment("Tom", MyTitle, StartAPT)

// Deletes one of the resources
ResDel is boolean
ResDel = SCH_Scheduler1.DeleteResource("Flo")
IF ResDel = True THEN
	Info("Resource deleted")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Scheduler control>.DeleteResource(<Resource>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó un recurso,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador del que se borrará el Resource. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Resource>: (Cadena de caracteres)`**

Nombre de Resource a borrar.  

**Observación**: Si los Resource s de la control Planificador han sido creados con [&lt;AddResource](../WDLang1/1000022905.md) utilizando [gStoredValue](../WDLang1/1410088363.md), este parámetro debe corresponder al valor especificado en [gStoredValue](../WDLang1/1410088363.md) para este Resource.

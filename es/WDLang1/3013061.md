


## DisableAAF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva una función de aplicación de automatic (AAF) en una ventana de control o en la aplicación actual.

Por ejemplo, se puede desactivar la exportación de un control Tabla a Excel. La opción del menú contextual correspondiente se elimina automáticamente.

Observación: Esta función es equivalente a la función [AAFDisable](../WDLang1/1000022018.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Click the Cancel button
// => the values typed by the user will not be saved
DisableAAF(MyWindow, aafSaveValue)
Close()
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// 'Restore the default parameters' menu option
DeleteParameter()
DisableAAF(MyWindow, aafSaveSizeAndPosition) 
DisableAAF(MyWindow, aafSaveValue) 
Use(WIN_MainWindow)
```
<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// -- Project initialization
// Disables the ability to select the columns to display 
// for all the application tables
DisableAAF(aafColumnSelection)
```

## Sintaxis

`DisableAAF([<Control or Window>, ] <Feature to disable>)`
---

**`<Control or Window>: (Nombre de control o ventana opcional)`**

Nombre de la ventana o control al que se aplica la funcionalidad automática. También es posible utilizar un nombre de alias de ventana.
Si no se especifica este parámetro, la funcionalidad automática indicada se desactivará en toda la aplicación.

**`<Feature to disable>: (Constante de tipo Integer)`**

La función de automatic para desactivar. Esta acción depende del tipo de control utilizado. Si el tipo de control no es válido para la constante seleccionada, la función no tiene ningún efecto .

## Observaciones


### Ejecutar una funcionalidad automática
<a name="ejecutar_una_funcionalidad_automatica_ELTPARAGRAPHE001101"></a>

Para ejecutar una funcionalidad automática, utilice la función [AAFExecute](../WDLang1/1000022099.md).
<a name="NOTE0_2"></a>


### Eliminar las AAF
<a name="eliminar_las_aaf_ELTPARAGRAPHE001111"></a>

Para eliminar las opciones que corresponden a los AAF en el menú contextual de una control, utilice [ContextMenu](../Proprietes/2510077.md).




## TreeSelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos seleccionados en una control TreeView (para una selección múltiple control TreeView). Las líneas seleccionadas corresponden a las líneas actuales en las que se visualiza el banda de selección.

**Observación**: La función **TreeSelectCount** puede utilizarse en:

- un control TreeView poblado programáticamente, basado en un archivo de datos, o basado en un Variable. 

- una selección única o una selección múltiple control TreeView.. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nbOccurrences is int 
nbOccurrences = TreeSelectCount(TREE_TreeView1)
FOR i = 1 TO nbOccurrences 
	Trace("Selected node:" + TreeSelect(TREE_TreeView1, False, i))  
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TreeSelectCount(<TreeView control>)`
---

**`<Result>: (Integro)`**



- Número de filas seleccionadas en la control TreeView especificada. Este número puede ser mayor que 1 para los controles TreeView de selección múltiple. 

- 0 si no se selecciona fila.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView que se va a utilizar.
Se produce un error WLanguage si este parámetro no corresponde al nombre de un control TreeView.
Si este parámetro corresponde a una cadena vacía (""), el control utilizado será el control TreeView al que pertenece el Event actual.




## Funciones de corrección ortográfica
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para administrar el corrector ortográfico:



|   |   |
| --- | --- |
| [SpellAddDictionary](../WDLang1/1000018921.md) | Añade una palabra al diccionario de OpenOffice: <br><br>- para el caso de la aplicación actual. <br><br>- para todas las aplicaciones.<br><br><br> |
| [SpellAvailable](../WDLang1/1000019019.md) | Indica si hay un diccionario disponible para el corrector ortográfico (comprobación realizada en los controles que utilizan la corrección integrada o la comprobación realizada por las funciones WLanguage). |
| [SpellCheck](../WDLang1/1000018923.md) | Indica si la ortografía de una palabra es correcta (es decir, si se encuentra en el diccionario de OpenOffice instalado en el equipo). |
| [SpellSuggest](../WDLang1/1000018920.md) | Devuelve la lista de posibles palabras encontradas en el diccionario de OpenOffice que pueden corregir una palabra mal escrita. |

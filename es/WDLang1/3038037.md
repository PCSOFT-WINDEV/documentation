
## WinInHeight (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la altura interna de una ventana (sin barra de título y Border).
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenExtInt.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
//Internal height of the "WIN_EditCustomer" window
ResHeight = WinInHeight(WIN_EditCustomer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinInHeight([<Window>])`
---

**`<Result>: (Integro)`**

Altura interna de la ventana (en píxeles).

**`<Window>: (Nombre de la ventana opcional)`**

Nombre o alias de la ventana que se va a utilizar. Si no se especifica este parámetro o si se trata de una cadena vacía ("""), se devuelve la altura interna de la ventana actual..

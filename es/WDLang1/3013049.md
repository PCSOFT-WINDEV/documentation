
## Constantes utilizadas por la función TraceStart
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por [TraceStart](../WDLang1/3013052.md):

| Constante | Detalle |
| --- | --- |
| **trCreation** | Despeja la ventana de salida |
| **trTopMost** | La ventana de salida se muestra Above todas las demás ventanas del sistema |
| **trDisable** | Todas las llamadas a [rastro](../WDLang1/3013050.md) serán ignoradas. |
| **trWindow** | El ventana de salida se muestra en la Corner superior izquierda Corner de la pantalla.<br><br><br> |
| **trFile** | El ventana de salida no se mostrará más y la información encontrada en el ventana de salida se guardará en el archivo llamado &lt;Archivo de rastreo&gt;..<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Sólo esta constante está disponible. Por Default, este archivo se crea en el directorio temporal de UNIX y se llama "Trace_&lt;Project>.txt". |
| **trWindow** + **trFile** | Se mostrará el ventana de salida y la información encontrada en el ventana de salida se guardará en el archivo llamado &lt;Fichero de Rastreo&gt;. |

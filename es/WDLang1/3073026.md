
## SysDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ruta de un directorio del sistema.





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En esta versión, esta función sólo puede utilizarse para devolver el directorio que utilizarán las aplicaciones para almacenar los datos (específicos del usuario).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Retrieve the path of system directory
ResDir = SysDir(srSystem)
```

## Observaciones


### Directorio temporal
<a name="directorio_temporal_ELTPARAGRAPHE000871"></a>

Para conocer el directorio temporal de Windows, utilice [fTempPath](../WDLang1/3036044.md).
<a name="NOTE0_2"></a>


### Papelera
<a name="papelera_ELTPARAGRAPHE000881"></a>

El "Bin" corresponde a un directorio virtual gestionado por Windows. La función **SysDir** no permite conocer la ruta de la "Papelera".
<a name="NOTE0_3"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000894"></a>

El uso de esta función desencadena la declaración de varias características de la aplicación en la generación de aplicaciones asistente.
Funcionalidades requeridas: 

- Biblioteca de imágenes

- Videoteca

- Mediateca

- Biblioteca de documentos

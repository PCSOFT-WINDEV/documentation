


## TableCellSelectOccurrence (Función)




	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>16 (56F)</B></B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Esta función se sustituye por [TableSelectCount](../WDLang1/3074030.md).

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de celdas seleccionadas que se encuentran en una control Tabla. 

**Observación**: La control Tabla debe permitir el modo de selección por celda. Esta opción se puede configurar en la pestaña "GUI" de control Tabla description. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbSelectedCells is int
Total is real
RowNum is int
ColNum is int

NbSelectedCells = TableCellSelectOccurrence(TABLE_TABLE1)
FOR i = 1 TO NbSelectedCells
	RowNum = TableCellSelect(TABLE_TABLE1, i, tcsLine)
	ColNum = TableCellSelect(TABLE_TABLE1, i, tcsColumn)
	Total = Total + TABLE_TABLE1[RowNum, ColNum] 
	
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TableCellSelectOccurrence(<Table control>)`
---

**`<Result>: (Integro)`**



- Número de celdas seleccionadas,

- 0 si el modo de selección por celda no está seleccionado para el control Tabla (pestaña "GUI" de la ventana de descripción del control),

- 1 si el control Tabla permite la selección de una sola célula. 




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.

<a name="NOTE0_1"></a>


## MCIWindowSize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el tamaño y la posición de la ventana de visualización de un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)).

**Atención**: Durante la apertura, se debe haber especificado previamente el nombre de la ventana del contenedor.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MCIOpen("Film.avi", "MyFilm", MciStyleChild, IMG_Image)
...
MCIWindowSize("MyFilm", MciModeStretched)

MCIPlay("MyFilm")
...
MCIClose("MyFilm")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Cambiar el tamaño de la ventana de visualización

`<Result> = MCIWindowSize(<Media file or alias> , <X Position> , <Y Position> , <Width> , <Height> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para get el número de error, utilice [MCIErr](../WDLang1/3051011.md). 
	Para obtener más detalles sobre el error, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
	El estado reporte del comando es devuelto por [MCIRetrieve](../WDLang1/3051022.md).




**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<X Position>: (Integro)`**

Nueva posición horizontal (coordenada X) de la ventana de visualización.

**`<Y Position>: (Integro)`**

Nueva posición vertical (coordenada Y) de la ventana de visualización.

**`<Width>: (Integro)`**

Nuevo ancho de la ventana de visualización.

**`<Height>: (Integro)`**

Nueva altura de la ventana de visualización.

**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario. Estos parámetros se pueden conocer consultando la documentación de la función "open" de Windows MCI.
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre el manejo de CD ROM o Joystick.  


<a name="SYNTAX2"></a>

### Modificar el modo de visualización

`<Result> = MCIWindowSize(<Media file or alias> , <Display mode> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (Cadena de caracteres)`**

Posición en el fichero. Esta posición se expresa en el formato especificado por [MCIPositionFormat](../WDLang1/3051004.md).

**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Display mode>: (Constante)`**

Modo de visualización del archivo en la ventana: 


|   |   |
| --- | --- |
| *MciModeCentered* | Image centrada en la ventana |
| *MciModeHomotheticCentered* <br>(Valor predeterminado) | Image estirada proporcionalmente |
| *MciModeNormal* | Visualización en 100% |
| *MciModeStretched* | Image estirada para ocupar toda la superficie |



**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario. Estos parámetros se pueden conocer consultando la documentación de la función "open" de Windows MCI.
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre el manejo de CD ROM o Joystick.

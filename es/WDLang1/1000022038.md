
## Background (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Background** define las características avanzadas del fondo de un elemento. Las características del fondo se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draw a rectangle with blue borders, green background and rounded corners
// in an Image control

// Initializes the drawing in the Image control
dStartDrawing(IMG_Drawing)

Rect is Rectangle
Rect.X = 10
Rect.Y = 15
Rect.Width = 100
Rect.Height = 150
BorderStyle is Border
BorderStyle.Color = LightBlue
BorderStyle.Round = 30
BorderStyle.Thickness = 1
BackgroundStyle is Background
BackgroundStyle.Color = LightGreen
dBorder(Rect, BorderStyle, BackgroundStyle)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Background
<a name="propiedades_especificas_descripcion_variables_tipo_background_ELTPARAGRAPHE000044"></a>

Las siguientes propiedades se pueden utilizar para manipular el fondo de un elemento:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Color | Variable de tipo [Color](../WDLang1/1000019661.md) | Color de fondo. |
| Gradient |   | Degradado del color de fondo: <br><br>- Propiedad **ColorStart** (tipo [Color](../WDLang1/1000019661.md)): Color inicial del degradado. <br><br>- Propiedad **ColorEnd** (tipo [Color](../WDLang1/1000019661.md)): Color final del degradado.<br><br>- Propiedad **Angle** (entero): Ángulo a utilizar para la selección de colores en el degradado. El valor del ángulo está comprendido entre 0 (ángulo horizontal, valor predeterminado) y 360°.<br><br>- Propiedad **Color3** (tipo [Color](../WDLang1/1000019661.md)): Color a utilizar en el degradado.<br><br>- Propiedad **DistanceColor3** (entero): Porcentaje que separa el color inicial del color 3. <br><br>- Propiedad **Color4** (tipo [Color](../WDLang1/1000019661.md)): Color a utilizar en el degradado.<br><br>- Propiedad **DistanceColor4** (entero): Porcentaje que separa el color inicial del color 4.  <br><br><br><br><br>Los parámetros de color se utilizan de la siguiente manera:<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=drectangledegrade.gif)<br><br><br>El ángulo permite definir la orientación del degradado.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=angle_degrade.gif)<br><br><br>Los degradados de 3 o 4 colores solo están disponibles para ángulos de 0 o 90 grados.<br><br>**Observaciones**: <br><br>- En algunos sistemas (por ejemplo, uso de 256 colores en modo de escritorio remoto), los degradados no están disponibles. En este caso, se utiliza el color definido mediante la propiedad **ColorStart**.<br><br>- En documentos docx, los fondos degradados no están disponibles. <br><br><br> |
| Hatch | Constante de tipo Integer | Tipo de sombreado utilizado: <br><br>- *HatchNone*: Sin sombreado. <br><br>- *HatchCrossed*: Sombreado cruzado.<br><br>- *HatchCrossedDiagonal*: Sombreado diagonal cruzado<br><br>- *HatchDiagonal*: Sombreado diagonal de 60%.<br><br>- *HatchDiagonalBottom*: Sombreado diagonal de 120%.<br><br>- *HatchHorizontal*: Sombreado horizontal.<br><br>- *HatchVertical*: Sombreado vertical. <br><br><br> |
| Type | Constante de tipo Integer | Tipo de fondo: <br><br>- *BackgroundSolid*: Fondo sólido.<br><br>- *BackgroundTransparent*: Fondo transparente (se ignora la propiedad **Color**).<br><br>- *BackgroundHatched*: Fondo sombreado (se tiene en cuenta la propiedad **Hatch**). <br><br>- *BackgroundGradient*: Fondo con degradado de color (se tiene en cuenta la propiedad **Gradient**).<br><br><br> |

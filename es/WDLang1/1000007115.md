## Presentación
<a name="presentacion_ELTTEXTE000077"></a>
El hashing se utiliza para resumir un archivo o una cadena de caracteres. Este resumen se llama Hash. Un único "hash" puede ser associated con un texto en lenguaje natural: cuando se modifica el texto inicial, también se modifica el hash correspondiente.

La trituración se utiliza principalmente para:

- comprobar la integridad de un Document cuando se transmite (el Document se transmite con el resultado del hash).

- firmar electrónicamente un Document (codificando el resultado del hash con una clave, por ejemplo).

## Funciones Hash
<a name="funciones_hash_ELTTEXTE000101"></a>
Las funciones de hash son las siguientes:



|   |   |
| --- | --- |
| [HashCheckFile](../WDLang1/1000007114.md) | Comprueba el Hash de un archivo para un tipo específico de algoritmo. |
| [HashCheckString](../WDLang1/1000007113.md) | Comprueba el guión de una cadena de caracteres para un tipo específico de algoritmo. |
| [HashFile](../WDLang1/1000007112.md) | Calcula el Hash de un archivo según un algoritmo específico. |
| [HashString](../WDLang1/1000007111.md) | Calcula el Hash de una cadena de caracteres según un algoritmo específico. |





Estas funciones se utilizan para gestionar los hashes simples y los hashes con autenticación de mensajes.

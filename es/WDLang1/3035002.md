


## EXE constants
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las constantes para la gestión de ejecutables son las siguientes: 


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **alStartup** | Ejecutar la aplicación al iniciar el PC | [AutoRunAdd](../WDLang1/3019002.md), [AutoRunDelete](../WDLang1/3019001.md) |
| **exeAboveNormalPriority** | Prioridad de aplicación: Prioridad superior a *exeNormalPriority* e inferior a *exeRealTimePriority*. | [ExePrioridad](../WDLang1/3035010.md) |
| **exeActive** | Modo de inicio del programa: el programa que se ejecuta es Active | [ExeRun](../WDLang1/3035006.md) |
| **exeBelowNormalPriority** | Prioridad de aplicación: Prioridad superior a *exeLowPriority* y menor a *exeNormalPriority*. | [ExePrioridad](../WDLang1/3035010.md) |
| **exeCommandLine** | Comando Line de ejecutable. | [Proceso ExeList](../WDLang1/3035008.md) |
| **exeCompany** | Información a obtener: nombre introducido para la empresa al crear el ejecutable. | [ExeInfo](../WDLang1/3035001.md) |
| **exeCopyright** | Información a obtener: derechos de autor del programa | [ExeInfo](../WDLang1/3035001.md) |
| **exeDeepLink** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeDescription** | Información a obtener: Programa description | [ExeInfo](../WDLang1/3035001.md) |
| **exeDirectory** | Información a obtener: directorio del ejecutable (sin el \\final pero con el ":"). | [ExeInfo](../WDLang1/3035001.md) |
| **exeDontWait** | Tiempo de espera antes de que se reanude el programa actual: el programa actual y el programa lanzado se ejecutan en paralelo | [ExeRun](../WDLang1/3035006.md) |
| **exeDownloadComplete** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeGeoTracking** | <br><br>Observación: Esta constante se mantiene para compatibilidad con versiones anteriores de la versión 230050. Es idéntica a la constante *exeInBackground*. | [ExeInfo](../WDLang1/3035001.md) |
| **exeHighPriority** | Prioridad de aplicación: Prioridad alta | [ExePrioridad](../WDLang1/3035010.md) |
| **exeIconize** | Modo de inicio del programa: la ejecución del programa es Inactive y minimizado | [ExeRun](../WDLang1/3035006.md) |
| **exeInactive** | Modo de inicio del programa: el programa que se ejecuta es Inactive | [ExeRun](../WDLang1/3035006.md) |
| **exeInBackground** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeLaunch** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeLongName** | Nombre y apellidos y ruta completa del ejecutable de la aplicación | [Proceso ExeList](../WDLang1/3035008.md) |
| **exeLowPriority** | Prioridad de aplicación: Bajo | [ExePrioridad](../WDLang1/3035010.md) |
| **exeMaximize** | Modo de inicio del programa: la ejecución del programa es Active, "toma control" sobre el programa actual (la ventana de ejecución se maximiza) | [ExeRun](../WDLang1/3035006.md) |
| **exeMaxMemory** | Cantidad máxima de memoria consumida por la aplicación | [Proceso ExeList](../WDLang1/3035008.md) |
| **exeMemory** | Cantidad de memoria consumida actualmente por la aplicación | [Proceso ExeList](../WDLang1/3035008.md) |
| **exeMode** | Información a obtener: modo de compilación de ejecutables (16 bits, 32 bits o DOS) | [ExeInfo](../WDLang1/3035001.md) |
| **exeName** | Información a obtener: nombre completo del ejecutable | [ExeInfo](../WDLang1/3035001.md) |
| **exeNoHandleInheritance** | Indica que el ejecutable no debe heredar los manejadores del padre (se aplica a TODOS los manejadores, incluidos los archivos). El ejecutable es independiente de la aplicación que lo lanzó.  | [DDEStart](../WDLang1/3028013.md)<br>[ExeRun](../WDLang1/3035006.md) |
| **exeNormalPriority** | Prioridad de aplicación: Prioridad superior a *exeBelowNormalPriority* e inferior a *exeAboveNormalPriority* | [ExePrioridad](../WDLang1/3035010.md) |
| **exeParentPID** | PID de padre (que significa el Process que inició el actual Process). | [ExeGetPID](../WDLang1/1000017239.md), [Proceso ExeList](../WDLang1/3035008.md) |
| **exePID** | Identificador de la aplicación Process.<br>Este identificador puede utilizarse, por ejemplo, en algunas APIs. | [ExeGetPID](../WDLang1/1000017239.md), [ExeRun](../WDLang1/3035006.md), [Proceso ExeList](../WDLang1/3035008.md) |
| **exePushNotification** | Modo de inicio de la aplicación: la aplicación es lanzada automáticamente por el sistema tras recibir un push Notification. | [ExeInfo](../WDLang1/3035001.md) |
| **exeRealTimePriority** | Prioridad de aplicación: Prioridad en tiempo real. | [ExePrioridad](../WDLang1/3035010.md) |
| **exeReturnValue** | El programa actual reanuda su ejecución cuando finaliza la ejecución del programa.. &lt;Resultado&gt; es el valor de retorno del ejecutable. | [ExeRun](../WDLang1/3035006.md) |
| **exeShortName** | Nombre y extension del ejecutable de la aplicación | [Proceso ExeList](../WDLang1/3035008.md) |
| **exeStartup** |   | [ExeInfo](../WDLang1/3035001.md) |
| **exeTID** | Identificador de la thread actual . | [ExeGetPID](../WDLang1/1000017239.md) |
| **exeVersion** | Información a obtener: versión del programa | [ExeInfo](../WDLang1/3035001.md) |
| **exeVersionWindowsFormat** | Información a obtener: en formato estándar de Windows. | [ExeInfo](../WDLang1/3035001.md) |
| **exeWait** | Tiempo de espera antes de que se reanude el programa actual: el programa actual reanuda su ejecución cuando finaliza la ejecución del programa. | [ExeRun](../WDLang1/3035006.md) |
| **formApplication** | Tipo de formulario a abrir: Abre el formulario de solicitud. | [AppOpenForm](../WDLang1/1000020980.md) |
| **formInAppPurchase** | Tipo de formulario a abrir: Abre la ventana para configurar las compras dentro de la aplicación. | [AppOpenForm](../WDLang1/1000020980.md) |
| **shareDocument** | Uno o más documentos (Image, video, pdf, ...) son compartidos. Estos documentos se identifican por su ruta completa, pasada como parámetro. | [StartShare](../WDLang1/1000020620.md) |
| **shareText** | El texto es compartido. | [StartShare](../WDLang1/1000020620.md) |


## Procedimiento WLanguage asociado a la función YesNoAsynchronous



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Procedimiento ("Callback") llamado por la función [YesNoAsynchronous](../WDLang1/1000025272.md) cuando el usuario hace clic en uno de los botones del cuadro de diálogo. Este procedimiento puede ser local, global, interno o lambda.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
YesNoAsynchronous("Do you want to close this window?", Procedure_YesNo)

INTERNAL PROCEDURE Procedure_YesNo(nResponse int)
	IF nResponse = Yes THEN
		ToastDisplay("You want to close the window!")
	ELSE
		ToastDisplay("You don't want to close the window!")
	END
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`YesNoAsynchronous_Callback(<Selected button>)`
---

**`<Selected button>: (Entero o constante de tipo Integer)`**

Identifica el botón seleccionado por el usuario:


|   |   |   |
| --- | --- | --- |
| *No* | 0 | El usuario seleccionó el botón [NO]<br>o cerró el cuadro de diálogo con el botón 'X'. |
| *Sí* | 1 | El usuario seleccionó el botón [SÍ] |

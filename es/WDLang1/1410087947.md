
## sysNotificationButton (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sysNotificationButton** se utiliza para definir todas las características avanzadas de control Botón en un Notification interactivo . Las características de esta control Botón pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

El tipo **sysNotificationButton** es utilizado por las variables [sysNotification](../WDLang1/1410087893.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a notification with a button of each type.
notif is sysNotification
notif.Title = "Many buttons"
notif.Text1 = "Download now to get the latest features."
notif.Note = "UPD_notif"
//normal button
notif.Buttons[1].Caption = "Normal"
notif.Buttons[1].Note = "btn_normal"
//button in the context menu of the notification
notif.Buttons[2].Caption = "Context"
notif.Buttons[2].Note = "btn_context"
notif.Buttons[2].Type = sysNotifContextButton
//button with a wait animation
notif.Buttons[3].Caption = "Wait"
notif.Buttons[3].Note = "btn_wait"
notif.Buttons[3].PendingUpdate = True
//reminder button
notif.Buttons[4].Caption = "Reminder"
notif.Buttons[4].Type = sysNotifReminderButton
//closing system button
notif.Buttons[5].Caption = "Close"
notif.Buttons[5].Type = sysNotifCloseButton

SysNotificationSend(notif)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables de sysNotificationButton
<a name="propiedades_especificas_descripcion_las_variables_sysnotificationbutton_ELTPARAGRAPHE000034"></a>

Las siguientes propiedades pueden utilizarse para manipular un botón en un Notification:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Acción | Global Procedure solamente | Nombre de la Procedure global que se ejecutará cuando el usuario haga clic en el botón.<br><br>Si no se especificó el procedimiento o no se puede encontrar, se ejecuta el evento opcional del proyecto "Recepción de una notificación". |
| escribir a máquina | Constante de tipo Integer | Comportamiento específico o posicionamiento del botón: <br><br>- *sysNotifContextButton* : Botón que se añade al menú contextual de Notification. <br><br>- *sysNotifCloseButton* : Botón del sistema que cierra automáticamente el Notification.<br><br>- *sysNotifStandardButton* (valor Default): Botón simple. <br><br>- *sysNotifReminderButton* : "Botón del sistema "Recordar más tarde. <br><br><br> |
| ImagePath | Cadena de caracteres | ruta absoluta de un Image en disco que puede añadirse al contenido del botón, junto al texto. |
| Nota | Cadena de caracteres | Información que se enviará a la aplicación cuando el usuario haga clic en el botón. |
| PendingUpdate | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para mostrar una animación de espera en el botón cuando el usuario haga clic en él, al tiempo que se evita que el Notification se cierre hasta que se envíe un nuevo Notification con el mismo ID. <br><br>- <u><u><u><u>False</u></u></u></u> (valor por defecto) a set sin animaciones de espera en el botón. La Notification se cierra al pulsar el botón. <br><br><br> |
| Pie de foto | Cadena de caracteres | Título del botón. |

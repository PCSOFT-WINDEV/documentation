
## Truncate (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Trunca la parte derecha de una cadena o buffer. Es posible indicar: 

- el número de caracteres de la cadena truncada. 

- el número de caracteres que se van a eliminar. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sString is string = " Turlututu"

// Truncate the string to 9 characters
Trace(Truncate(sString, 9))	// Displays " Turlutut"
Trace(Truncate(sString, 9, numberOfCharactersToStore))	 // Displays " Turlutut"

// Truncate the string to 4 characters
Trace(Truncate(sString, 4, numberOfCharactersToDelete))	
// Displays " Turlu"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Truncate(<Source> , <Number of characters> [, <Truncate method>])`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena truncada.

- EOT si el parámetro &lt;Número de caracteres&gt; es negativo. Se produce un error de WLanguage (error no fatal). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Source>: (Cadena de caracteres o Buffer)`**

Elemento a manipular: Cadena o buffer. Este elemento no se modifica.

**`<Number of characters>: (Entero)`**

Corresponde a: 

- el número de caracteres que deben conservarse en la cadena &lt;Resultado&gt; (si se utiliza la constante *numberOfCharactersToStore*). 

- el número de caracteres o bytes que deben eliminarse de la &lt;Fuente&gt; (si se utiliza la constante *numberOfCharactersToDelete*). 




**`<Truncate method>: (Constante opcional de tipo Boolean)`**

Tipo de truncamiento a utilizar: 


|   |   |
| --- | --- |
| *numberOfCharactersToDelete* | El parámetro &lt;Número de caracteres&gt; corresponde al número de caracteres que debe eliminarse de la &lt;Fuente&gt;. |
| *numberOfCharactersToStore*<br>(Valor predeterminado) | El parámetro &lt;Número de caracteres&gt; corresponde al número de caracteres que debe conservarse en la cadena &lt;Resultado&gt;. |

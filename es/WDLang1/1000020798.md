


## dSaveImageICO (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una imagen en formato de ícono en un archivo o en la memoria.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create an icon
MyIcon is Icon
MyIcon.Image[1].Image = "C:\MyImages\logo48.bmp"
MyIcon.TransparencyColor[1] = LightMagenta
MyIcon.Format[1] = icoFormatBMP
MyIcon.Image[2].Image = "C:\MyImages\logo256.png"
MyIcon.TransparencyColor[2] = Transparent
MyIcon.Format[2] = icoFormatPNG
dSaveImageICO(MyIcon, "Logo.ICO")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Guardar un ícono en un archivo en formato ICO

`<Result> = dSaveImageICO(<Icon> , <File name>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Icon>: (Variable de tipo Icon)`**

Nombre de la variable de tipo [Icon](../WDLang1/1000020811.md) que se utilizará.

**`<File name>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo ICO a crear. Se puede utilizar una ruta de tipo UNC. La extensión del archivo es opcional.

**Atención**:

- Para guardar la imagen, debe existir el directorio de destino.

- No se comprueba si existe el archivo ICO. Si ya existe, se sobrescribe automáticamente.





<a name="SYNTAX2"></a>

### Guardar un ícono en la memoria

`<Result> = dSaveImageICO(<Icon> , <inMemory>)`
---

**`<Result>: (Buffer)`**

Cadena de bytes que contiene la imagen en formato ICO. Esta cadena de bytes puede asignarse a un control Imagen, guardarse en un archivo (función ([fSaveText](../WDLang1/3036046.md)), etc.

**`<Icon>: (Variable de tipo Icon)`**

Nombre de la variable de tipo [Icon](../WDLang1/1000020811.md) que se utilizará.

**`<inMemory>: (Constante)`**

*inMemory*: Constante utilizada para guardar una imagen en la memoria.

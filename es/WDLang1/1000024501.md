


## &lt;Variable WDPic&gt;.DrawPoint (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja un punto en una variable de tipo [WDPic](../WDLang1/1000023503.md).

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
// Draw a light yellow point in a WDPic variable
MyWDPicImage.DrawPoint(10, 50, LightYellow)
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Dibujar un punto especificando las coordenadas

`<WDPic image>.DrawPoint(<X-coordinate> , <Y-coordinate> [, <Point color> [, <Opacity>]])`
---

**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.

**`<X-coordinate>: (Entero)`**

Posición horizontal (en pixeles) del punto a dibujar.

**`<Y-coordinate>: (Entero)`**

Posición vertical (en pixeles) del punto a dibujar.

**`<Point color>: (Entero o constante (opcional))`**

Color a asignar al punto. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el color del punto:

- es *Transparente* si no se ha utilizado la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md) previamente, 

- corresponde al color especificado en la última llamada a la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md).




**`<Opacity>: (Entero opcional)`**

Opacidad del color, comprendido entre 0 (transparente) y 255 (opaco). Una opacidad de 128 indica que el pixel correspondiente tendrá una transparencia del 50% cuando se dibuje en la pantalla o se guarde en una imagen en formato PNG. 

**Observación**: Esta opacidad se tendrá en cuenta si el dibujo de la Imagen se inicializó con la constante *dWithOpacity* (función [&lt;Variable WDPic&gt;.StartDrawing](../WDLang1/1000024471.md)).


<a name="SYNTAX2"></a>

### Dibujar un punto con una variable de tipo Point

`<WDPic image>.DrawPoint(<Point> [, <Point color> [, <Opacity>]])`
---

**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.

**`<Point>: (Variable de tipo Point)`**

Nombre de la variable [Point](../Motscles/1000018908.md) que contiene las coordenadas del punto a dibujar.

**`<Point color>: (Entero o constante (opcional))`**

Color a asignar al punto. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el color del punto:

- es *Transparente* si no se ha utilizado la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md) previamente, 

- corresponde al color especificado en la última llamada a la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md).




**`<Opacity>: (Entero opcional)`**

Opacidad del color, comprendido entre 0 (transparente) y 255 (opaco). Una opacidad de 128 indica que el pixel correspondiente tendrá una transparencia del 50% cuando se dibuje en la pantalla o se guarde en una imagen en formato PNG. 

**Observación**: Esta opacidad se tendrá en cuenta si el dibujo de la Imagen se inicializó con la constante *dWithOpacity* (función [&lt;Variable WDPic&gt;.StartDrawing](../WDLang1/1000024471.md)).

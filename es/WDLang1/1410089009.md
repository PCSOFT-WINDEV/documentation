


## &lt;picLayer Variable&gt;.Blur (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desenfoca toda el área especificada en una variable de tipo [picLayer](../WDLang1/1000024604.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Blurs the layer
MyLayer.Blur()

IMG_MyDrawing = MyImage
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Desenfocar la imagen completa

`<picLayer image>.Blur([<Radius>])`
---

**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<Radius>: (Entero opcional)`**

Radio alrededor de un punto en el que se tienen en cuenta los puntos cercanos para el desenfoque. Cuanto mayor sea el radio, más borrosa será la imagen y más tiempo durará el desenfoque. El valor predeterminado es 10.


<a name="SYNTAX2"></a>

### Desenfocar un área de una imagen

`<picLayer image>.Blur(<Area to blur> [, <Radius>])`
---

**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<Area to blur>: (Variable de tipo Rectangle o Polygon)`**

Área de la imagen que se va a desenfocar. Esta área puede ser definida por 

- una variable de tipo [Rectangle](../Motscles/1514079.md) (para áreas simples).

- una variable de tipo [Polygon](../WDLang1/1000025624.md) (para áreas complejas). 
	Observaciones: 

	- No es necesario cerrar el polígono (las coordenadas del último punto no tienen que ser iguales a las coordenadas del primer punto). El polígono se "cierra" automáticamente.

	- Un polígono puede tener un máximo de 100 000 puntos.





Las coordenadas se especifican con respecto a la esquina superior izquierda de la imagen (coordenadas (0,0)).

**`<Radius>: (Entero opcional)`**

Radio alrededor de un punto en el que se tienen en cuenta los puntos cercanos para el desenfoque. Cuanto mayor sea el radio, más borrosa será la imagen y más tiempo durará el desenfoque. El valor predeterminado es 10.

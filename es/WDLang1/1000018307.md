


## Funciones de gestión de colas, pilas y listas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los tipos Cola, Pila y Lista:



|   |   |
| --- | --- |
| [Dequeue](../WDLang1/1000018322.md) | Elimina un elemento de la cola. |
| [En espera](../WDLang1/1000018323.md) | Añade un elemento a la cola. |
| [ListInfo](../WDLang1/1000019548.md) | Recupera las características de una lista WLanguage: tipos de elementos y número de elementos. |
| [Pop](../WDLang1/1000017372.md) | Hace estallar un elemento de una pila. |
| [Push](../WDLang1/1000017366.md) | Empuja un elemento a la pila. |
| [QueueInfo](../WDLang1/1000019547.md) | Recupera las características de una cola: tipos de elementos y número de elementos. |
| [StackInfo](../WDLang1/1000019546.md) | Recupera las características de una pila: tipos de elementos y número de elementos. |

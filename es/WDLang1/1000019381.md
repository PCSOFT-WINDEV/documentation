


## TableColumnSelect (Función)




	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>16 (56)</B></B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [TableSelect](../WDLang1/3074020.md).

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de columna correspondiente a una de las columnas seleccionadas en un control Tabla. 

**Observaciones**:  

- La control Tabla debe permitir el modo de selección por columna. Esta opción se puede configurar en la pestaña "GUI" de control Tabla description. 

- Para conocer el número de columnas seleccionadas, utilice [TablaColumnaSelectOccurrence](../WDLang1/1000019380.md).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbColSelection is int 
ColNum is int 

NbColSelection = TableColumnSelectOccurrence(TABLE_TABLE1)
Trace("TableColumnSelectOccurrence() = " + NbColSelection)

FOR i = 1 TO NbColSelection
	ColNum = TableColumnSelect(TABLE_TABLE1,i)
	Trace("The column " + ColNum + " is selected")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableColumnSelect(<Table control> , <Index>)`
---

**`<Result>: (Integro)`**



- Número de la columna seleccionada. El número de la primera columna es set a 1. 

- -1 si se produce un error. 




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular. 

**`<Index>: (Integro)`**

Índice de la selección. El índice del primer elemento seleccionado es set a 1. Si &lt;Index&gt; es mayor que el número de columnas seleccionadas, &lt;Resultado&gt; se set a -1.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para permitir la selección de una columna, debe habilitar en las características de control Tabla: 

- el modo de selección "Columnas" (selección simple o múltiple) en la pestaña "UI" de la ventana description de control Tabla. 

- el modo de selección "Cells" (selección múltiple) en la pestaña "UI" de la ventana description de control Tabla. 




El usuario final puede seleccionar las columnas haciendo clic en el encabezado de la columna (como para una clasificación).

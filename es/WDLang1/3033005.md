
## eToEuro( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte a euros un valor expresado en una moneda específica.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Convert Marks to Euro
ResConversion = eToEuro(150, eDEM)
```




<a name="Example2"></a>

// Two decimal places are used for the conversion
ResConversion = eToEuro(10000, eITL, 2, resDiff)

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = eToEuro(<Value to convert> [, <Source currency> [, <Number of decimals> [, <Rounding difference>]]])`
---

**`<Result>: (Moneda)`**

Valor convertido en euros.

**`<Value to convert>: (Moneda)`**

Importe a convertir.

**`<Source currency>: (Entero o constante (opcional))`**

Identificador de la moneda de origen o associated constante con la moneda de origen. Si no se especifica este parámetro, la moneda a convertir es la moneda cuyo identificador es "1" (por Default el franco francés).

**`<Number of decimals>: (Entero opcional)`**

Número de decimales utilizados para el resultado de la Conversion( 2 por la Default).

**`<Rounding difference>: (Moneda opcional)`**

Variable recuperando la diferencia de redondeo causada por la conversión. Esta diferencia se expresa en la moneda de origen.
Este parámetro es sólo useful para las monedas cuyo tipo de cambio es mayor que el Euro (Franco Belga, Peseta, etc.). Para las otras monedas, este parámetro es siempre nulo.

## Observaciones




### Intercambio tabla
<a name="intercambio_tabla_ELTPARAGRAPHE000070"></a>

Una ***tabla de cambio*** se almacena en la memoria. Cada moneda es con associated:

- Un número de moneda utilizado para identificar la moneda por programación.

- Una constante utilizada para identificar la moneda por programación.

- Una leyenda (hasta 7 caracteres): esta leyenda se muestra en la control "Moneda + Euro" si la máscara de entrada lo permite (se indica el signo $).

- Un tipo de cambio. Este tipo de cambio corresponde al tipo de cambio de la moneda en relación con el euro.

- añadir nuevas monedas modificando las monedas personalizadas si es necesario ("monedas definidas por el usuario" de 16 a 25) con [eInitCurrency](../WDLang1/3033003.md),

### Diferencia de redondeo
<a name="diferencia_redondeo_ELTPARAGRAPHE000385"></a>

La &lt;Diferencia de redondeo&gt; se calcula mediante la siguiente fórmula:


```txt
Round(<Value to convert>, <Result in source currency>, <Number of decimals>)
```


Por ejemplo:


```wl
Difference and ResConversion are currencies
ResConversion = eToEuro(1.52, eFRF, 2, Difference)
```


ResConversión es set a 0.45 y Diferencia es set a 0.01.

La siguiente Line de código permite recuperar el importe inicial:


```wl
InitialAmount = eToCurrency(ResConversion, eFRF, 2) + Difference
```




## &lt;Hoja de cálculo&gt;.SelectWorksheet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona una hoja de trabajo (identificada por su nombre) en un control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Go to Worksheet] button
PSHEET_MySpreadsheet.SelectWorksheet(EDT_NameWorksheet_To_Select)
IF ErrorOccurred = True THEN
	Error(ErrorInfo())
ELSE
	STC_MyWorksheet = PSHEET_MySpreadsheet.CurrentWorksheet
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.SelectWorksheet(<Worksheet>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Worksheet>: (Cadena de caracteres)`**

Nombre de la hoja de trabajo a seleccionar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La índice de la hoja de trabajo actual es devuelta por [Hoja de trabajo actual](../Proprietes/1000021640.md). 

- El nombre de la hoja de trabajo actual es devuelto por [Nombre de la hoja de trabajo](../Proprietes/1000021646.md). 

- Si la hoja de trabajo no existe, la función desencadena un error no fatal.: la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error. 

- Esta función sólo puede utilizarse en una control Hoja de cálculo que se encuentre en una ventana.




## &lt;Mapa de árbol&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del elemento que aparece en una posición determinada en una control TreeMap.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Optional code of mouse rollover for the TMAP_TreeMap1 control
HoveredElement is int
HoveredElement = TMAP_TreeMap1.InfoXY( MouseXPos(), MouseYPos())
// Changes the element tooltip
IF HoveredElement > 0 THEN
	TMAP_TreeMap1[HoveredElement].Tooltip = ...
		"Hovered element: " + TMAP_TreeMap1[HoveredElement].Caption
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeMap control>.InfoXY(<X> , <Y>)`
---

**`<Result>: (Integro)`**



- Índice del elemento mostrado en la posición especificada. 

- -1 si ningún elemento corresponde a la posición especificada.




**`<TreeMap control>: (Nombre del control)`**

Nombre de control TreeMap a utilizar.

**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. Esta coordenada se da en relación con la control.

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada se da en relación con la control.

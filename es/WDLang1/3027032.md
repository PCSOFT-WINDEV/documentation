


## ChronoEnd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [ChronoStart](../WDLang1/3027031.md)). El cronómetro se puede reiniciar con la función [ChronoResume](../WDLang1/1000018830.md), si es necesario. 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the stopwatch 
ChronoStart()
MyFunction()
// Stop the stopwatch 
DurationMyFunction is Duration = ChronoEnd()
Info("The MyFunction() process lasted " + DurationMyFunction)
```


<a name="Example2"></a>

```wl
// Start the first stopwatch 
ChronoStart(1)
// Start the second stopwatch 
ChronoStart(2)
// Stop the first stopwatch
DurationChronoStart is Duration = ChronoEnd(1)
MyFunction()
// Stop the second stopwatch
DurationMyFunction is Duration = ChronoEnd(2)
Info("MyFunction() process lasted" + DurationMyFunction, ...
	"The whole process lasted " + DurationChronoStart)
```

## Sintaxis

### Detener el cronometraje iniciado con un número de cronómetro

`<Result> = ChronoEnd([<Stopwatch number>])`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el número de cronómetro especificado.


**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro utilizado. Si no se especifica este parámetro, el valor predeterminado es 1.

### Detener el cronometraje iniciado con una variable de tipo Chrono

`<Result> = ChronoEnd(<Stopwatch>)`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el cronómetro especificado.

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.   



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000172"></a>

Si se llama a la función **ChronoEnd** dos veces en el mismo cronómetro sin haber llamado a la función [ChronoStart](../WDLang1/3027031.md), la duración devuelta será idéntica para las dos llamadas (el cronómetro se detiene al llamar a la función **ChronoEnd** por primera vez).
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000188"></a>

Si no se llama a la función [ChronoStart](../WDLang1/3027031.md) antes de la función **ChronoEnd**, se produce un error WLanguage (error de tipo "No se ha iniciado el cronómetro").
<a name="NOTE0_3"></a>


### Hilos y tareas paralelas
<a name="hilos_tareas_paralelas_ELTPARAGRAPHE000202"></a>

Cada hilo (y cada tarea paralela) tiene sus propios cronómetros. 

Dos hilos distintos (o dos tareas paralelas distintas) pueden realizar cronometrajes separados utilizando el mismo número de cronómetro. 

El error WLanguage "No se ha iniciado el cronómetro" solo se mostrará si no se llama a las funciones [ChronoStart](../WDLang1/3027031.md) y **ChronoEnd** en el mismo hilo.

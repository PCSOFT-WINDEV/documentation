


## &lt;Procesamiento de palabras&gt;.Save (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una Document como un archivo "docx". El Document salvado permanece abierto. Para cerrarlo, debe utilizar [&lt;Procesamiento de palabras&gt;.close](../WDLang1/1000024264.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Choose the directory and the backup name 
sFileName is string = fSelect(CompleteDir(fExeDir()),"presentation.docx", ...
	"Selecting DocX files","Docx file (*.docx)" + TAB + "*.docx" , "*.docx", fselCreate)

// Save the file
WP_NoName1.Save(sFileName)

IF ErrorOccurred THEN
	Error(StringBuild("The %1 file was not saved.", sFileName), ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.Save([<File path>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la copia de seguridad,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<File path>: (Cadena de caracteres opcional)`**

ruta completa y nombre del archivo "docx.

**Si se especifica este parámetro,** el archivo se guarda en la ubicación especificada (el archivo existente se sobrescribe si es necesario). 
Observación: SI el Document "docx" se guarda con un ruta que difiere del inicial (corresponde a "Save as"), el archivo fuente se cierra, el Document se guarda en el nuevo archivo y el nuevo archivo "docx" permanece abierto.  

**Si no se especifica este parámetro**, el ruta que se tiene en cuenta es el ruta especificado en el valor **Property** del control Procesador de texto manejado.. Si esta propiedad está vacía, la función devuelve <u><u><u><u>False</u></u></u></u>.

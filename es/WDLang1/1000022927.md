
## docCell (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docCell** se utiliza para definir las características avanzadas de una célula tabla que se encuentra en un procesador de textos Document. Las características de esta célula pueden definirse y modificarse utilizando diferentes propiedades del lenguaje WLanguage. 

**PRECAUCIÓN: Este tipo no debe utilizarse directamente.** Sólo se debe utilizar: 

- a través del tipo [docElement](../WDLang1/1000022484.md) (para un elemento Cell). 

- mediante el tipo [docParagraph](../WDLang1/1000022483.md) (para un párrafo de la tabla). 


Para más detalles sobre los diferentes tipos relacionados con los documentos de tratamiento de textos, véase [Teclea associated con el procesador de textos](../WDLang1/1000022736.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the document
MyDocument is Document <- WP_Table
// Retrieve the table, here the paragraph 1
pTable is docParagraph <- MyDocument.Paragraph[1]

// Browses the rows
FOR I = 1 _TO_ pTable.Table.Rows.Occurrence
	// Browses the columns
	FOR I = 1 _TO_ pTable.Table.Columns.Occurrence
		Trace("Value of cell [[%i%]][[%j%]] = " + 
		pTable.Table.Cells[i, j].Content.Text)
	END
END
```

### Propiedades específicas de la description de las variables docCell
<a name="propiedades_especificas_description_las_variables_doccell_ELTPARAGRAPHE000045"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docCell**:

   | --- |
| --- | --- | --- |
| **Nombre de la propiedad** | Tipo utilizado | Efecto |
| Contenido | Variable de tipo [docFragment](../WDLang1/1000022482.md) | Contenido de la célula tabla. |
| ColSpan | Integro | Índice de la columna actual. |
| RowSpan | Integro | Índice de la corriente fila. |
| ÍndiceColumna | Integro | Índice de la columna actual. |
| RowIndex | Integro | Índice de la corriente fila. |
| PageLayout |   | Disposición de la celda actual. <br><br> |
|   | PageLayout.Border | Variable de tipo [Border](../Motscles/15140100.md) | Identificador de Border utilizado para el párrafo.  <br><br>Observación: las esquinas redondeadas no están soportadas. |
|   | PageLayout.BackgroundColor | Integro | Color de fondo de la celda. Este color puede corresponder a:<br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)), <br><br>- a [color preestablecido](../WDLang5/3010002.md). <br><br><br> |
|   | PageLayout.BackgroundPattern | Variable de tipo [Background](../WDLang1/1000022038.md) | Características de la célula Background. <br><br> |

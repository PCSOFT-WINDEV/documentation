


## &lt;Variable picLayer&gt;.Rotation (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Gira una imagen en una variable de tipo [picLayer](../WDLang1/1000024604.md).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dRotation.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
MyLayer is picLayer

// Rotate layer by 45 degrees
Result1 is boolean
Result1 = MyLayer.Rotation(45)
IF Result1 = False THEN Error("The operation failed")
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Hacer un giro simple

`<Result> = <picLayer image>.Rotation(<Rotation angle> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el giro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<picLayer image>: (Variable de tipo picLayer)`**

Nombre de la variable de tipo [picLayer](../WDLang1/1000024604.md) que se utilizará.

**`<Rotation angle>: (Real)`**

Ángulo de rotación (en grados). 

- Si el ángulo es positivo, la imagen gira en el sentido de las agujas del reloj. 

- Si el ángulo es negativo, la imagen gira en sentido contrario a las agujas del reloj.




**`<Options>: (Constante de tipo Integer)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Opciones de rotación: 


|   |   |
| --- | --- |
| *drAdapt* | La imagen se gira y se amplía. Luego, se reduce para que corresponda al tamaño inicial de la imagen. |
| *drDefault* | La imagen se amplía (si es necesario) para que corresponda a las dimensiones de la imagen girada.<br><br>**Constante utilizada de forma predeterminada para las variables de tipo [picLayer](../WDLang1/1000024604.md)**. |
| *drNoEnlarging* | La imagen gira pero no se amplía: la imagen puede truncarse.<br><br> |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no está disponible.

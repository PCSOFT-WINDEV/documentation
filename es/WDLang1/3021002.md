
## StatusBarDeleteCell( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una celda de la barra de estado de la ventana actual.

**Observación**: La ventana debe tener una barra de estado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds "MyCell" to the status bar of the current window
StatusBarAddCell("MyCell", 100, LightRed)
...
StatusBarDeleteCell("MyCell")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`StatusBarDeleteCell(<Cell name>)`
---

**`<Cell name>: (Cadena de caracteres)`**

Nombre de la celda a borrar. Este nombre puede corresponder a una celda de la barra de estado creada en el editor o mediante programación ([StatusBarAddCell](../WDLang1/3021001.md)). Un error de WLanguage ocurre si la celda no existe.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Borrar una celda de la barra de estado
<a name="borrar_una_celda_barra_estado_ELTPARAGRAPHE000037"></a>

Le recomendamos que haga la célula Invisible en lugar de borrarla.
<a name="NOTE0_2"></a>




### Usando las propiedades WLanguage con las celdas en una barra de estado
<a name="usando_las_propiedades_wlanguage_con_las_celdas_una_barra_estado_ELTPARAGRAPHE000045"></a>

Se pueden aplicar varias propiedades de WLanguage a las celdas de la barra de estado.

**Atención**: Para usar una Property en una celda de una barra de estado, se debe usar la siguiente sintaxis:


```txt
{"Cell name"}..<Property name>
```


Para obtener más información, consulte [Lista de propiedades que se pueden utilizar con las celdas de las barras de estado](../WDChamp/1010035.md).


## AnimationSetMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia el tipo de animación:

- para un Image en un control Imagen.

- para un texto en un control Estático.




Los parámetros específicos del nuevo tipo pueden ser modificados por [AnimationSetParameter](../WDLang1/3017002.md).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) En esta versión, sólo puede cambiar el tipo de animación utilizado para un texto en un control Estático.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the animation of the IMG_Image1 control
IMG_Image1.Animation = True
AnimationSetMode(IMG_Image1, actScrolling)
```


<a name="Example2"></a>



```wl
// Change the animation parameters of a Static control
IMG_Image1.Animation = True
AnimationSetMode(STC_MyStatic, actBlinking)
AnimationSetParameter(STC_MyStatic, acpDuration, 100)
AnimationSetParameter(STC_MyStatic, acpColor, DarkBlue)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AnimationSetMode(<Control used> , <Type of animation>)`
---

**`<Control used>: (Nombre del control)`**

Nombre de la imagen control o control estático a utilizar.

**`<Type of animation>: (Constante o cadena de caracteres)`**

Animación para interpretar.



|   |   |
| --- | --- |
| *actBlinking* (o "Parpadeo") | La estática se muestra alternativamente en su Color predeterminada y en su Color intermitente. |
| *actScrolling* (o "Scrolling") | La Image (o la leyenda) se muestra progresivamente, desde la derecha, desde la izquierda, ...<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante sólo se puede utilizar con los controles estáticos. |
| *actWinDevAnimation* (o "Imagen a animar") | La Image está dividida en varios dibujos. Cada dibujo se muestra sucesivamente, dando una ilusión de vídeo. <br>Esta constante sólo se puede utilizar con los controles Image.<br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Cambiar el tipo de animación
<a name="cambiar_tipo_animacion_ELTPARAGRAPHE000075"></a>

Si el nuevo tipo corresponde al tipo actual, los parámetros específicos de este tipo se reinician con los parámetros de Default.
<a name="NOTE0_2"></a>




### Detener o reiniciar una animación
<a name="detener_reiniciar_una_animacion_ELTPARAGRAPHE000083"></a>

La animación puede activarse o desactivarse con el programa [Animación](../Proprietes/2510006.md) Property.
<a name="NOTE0_3"></a>
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Imágenes que utilizan un formato animado
<a name="imagenes_que_utilizan_formato_animado_ELTPARAGRAPHE000095"></a>

**AnimationSetMode** no tiene efecto en las imágenes que utilizan un formato animado. Por lo tanto, para una Image en formato GIF animado, los parámetros se leen en el archivo y no pueden ser modificados.




## GanttChangeMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el período (que significa el número de días) que se muestra en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on BTN_Week
GanttChangeMode(COL_Gantt, ganttModeInDay)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`GanttChangeMode(<Gantt column> , <Display mode> [, <Number of days>])`
---

**`<Gantt column>: (Nombre del control)`**

Nombre de la columna (de un control Tabla o Tabla TreeView) de tipo Diagrama de Gantt a utilizar.

**`<Display mode>: (Constante de tipo Integer)`**

Intervalo de tiempo que debe visualizarse en la columna Diagrama de Gantt: 


|   |   |
| --- | --- |
| *ganttModeAccordingToTask* | Muestra el diagrama de Gantt desde la primera tarea hasta la última. |
| *ganttModeInDay* | Visualiza el diagrama de Gantt según el ancho de la columna.. En este caso, debe especificar el número de días a visualizar. |



**`<Number of days>: (Entero opcional)`**

Número de días mostrados. Este parámetro solo se tiene en cuenta si el parámetro &lt;Modo de visualización&gt; corresponde a la constante *ganttModeInDay*. De lo contrario, se produce un error WLanguage.

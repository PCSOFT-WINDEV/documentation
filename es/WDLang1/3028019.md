
## DDETopic( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica al sujeto de la conversación associated con una conexión DDE.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a DDE connection
ConnectionNum = DDEConnect(ServerName, "MyTopic")

...

// Conversation subject?
Subject = DDETopic(ConnectionNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDETopic([<Connection identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Tema de conversación.

**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, **DDETopic** utiliza la última conexión gestionada.

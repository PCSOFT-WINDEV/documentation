
## ErrorRestoreParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura el error Process de un Process. Este error Process se modificó en la actual Process mediante una llamada a [ErrorChangeParameter](../WDLang1/3034009.md).


<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ErrorRestoreParameter()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si el número de llamadas a la función **ErrorRestoreParameter** supera el número de llamadas a la función [ErrorChangeParameter](../WDLang1/3034009.md), las llamadas adicionales no se tendrán en cuenta.

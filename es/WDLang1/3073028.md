
## SysVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la versión PHP utilizada en el servidor actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the PHP version
 Info("PHP version " + SysVersion())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysVersion()`
---

**`<Result>: (Cadena de caracteres)`**



- Información solicitada sobre la versión PHP,

- Cadena vacía ("") si se produce un error.

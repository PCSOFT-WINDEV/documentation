
## Lower (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte todos los caracteres de una cadena a minúsculas según la configuración regional definida en Windows.

**Observación**: La función **Lower** se conserva por motivos de compatibilidad y solo se puede utilizar con conjuntos de caracteres latinos. Para configurar el formato de una cadena de forma más precisa o manipular otros conjuntos de caracteres, utilice la función [StringFormat](../WDLang1/3024019.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Lower("ABCD") 		// Returns "abcd"
Lower("HELLO")            // Returns "hello"
Lower("Peter and Paul")		// Returns "peter and paul"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Lower(<String to convert>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres en minúsculas.

**`<String to convert>: (Cadena de caracteres)`**

Cadena a convertir en minúsculas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Reglas de conversión
<a name="reglas_conversion_ELTPARAGRAPHE000077"></a>

- Los caracteres en minúsculas sin acentos permanecen iguales.

- Los caracteres en minúsculas con acentos permanecen iguales.

- Los caracteres en mayúsculas sin acentos se convierten en minúsculas sin acentos.

- Los caracteres en mayúsculas con acentos permanecen iguales.
	




**Observación**: La función **Lower** se conserva por motivos de compatibilidad y solo se puede utilizar con conjuntos de caracteres latinos. Para configurar el formato de una cadena de forma más precisa o manipular otros conjuntos de caracteres, utilice la función [StringFormat](../WDLang1/3024019.md).
<a name="NOTE0_2"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000094"></a>

- Los "espacios en blanco" se conservan.

- El motor HFSQL realiza automáticamente búsquedas que no distinguen entre mayúsculas y minúsculas. Por lo tanto, la función **Lower** no tiene ningún efecto al realizar este tipo de operación.

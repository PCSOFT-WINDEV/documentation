


## DashMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre o cierra el modo de edición de control Cuadro de mando.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Informs the user if the Dashboard control is already in edit mode
IF DashMode(DASH_Dashboard) <> dashNoModif THEN
	Error("The dashboard is in edit mode already")
	RETURN
END

nOption is int

// Defines the edit option
IF CBOX_EditOptions[1] THEN nOption += dashMove
IF CBOX_EditOptions[2] THEN nOption += dashResize
IF CBOX_EditOptions[3] THEN nOption += dashDelete

// Checks that at least one option is checked
IF nOption = 0 THEN
	Error("At least one edit option must be checked")
	SetFocusAndReturnToUserInput(CBOX_EditOptions)
END

// Modifies the edit mode
DashMode(DASH_Dashboard, nOption)
```
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Disable the deletion of widgets 
// but not their move or resizing:
DashMode(DASH_NoName1, dashMove + dashResize)
// Caution, to avoid displaying the toolbar, 
// you must use DissableAAF (or AAFDisable) with the constant aafDashEditTdbMode.
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Cambiar el modo de la control Cuadro de mando

`DashMode(<Dashboard control> , <Mode>)`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar. 

**`<Mode>: (Constante o combinación de constantes de tipo Integer)`**

Tratar las opciones disponibles para el usuario: 


|   |   |
| --- | --- |
| *dashDelete* | Cambiar a modo de edición: el usuario puede ocultar los widgets en el control Cuadro de mando. |
| *dashMove* | Cambiar a modo de edición: el usuario puede mover los widgets en el control Cuadro de mando. |
| *dashNoModif* | Cerrar el modo de edición: el usuario no puede modificar los widgets en el control Cuadro de mando.<br><br>Esta constante no se puede combinar con las otras constantes. |
| *dashResize* | Cambiar a modo de edición: el usuario puede cambiar el tamaño de los widgets en el control Cuadro de mando. |




<a name="SYNTAX2"></a>

### Averiguar el modo actual de la control Cuadro de mando

`<Result> = DashMode(<Dashboard control>)`
---

**`<Result>: (Constante de tipo Integer)`**

Modo de edición actual de la control Cuadro de mando: 


|   |   |
| --- | --- |
| *dashDelete* | Modo de edición: el usuario puede ocultar los widgets en el control Cuadro de mando. |
| *dashMove* | Modo de edición: el usuario puede mover los widgets en el control Cuadro de mando. |
| *dashNoModif* | Modo estándar: el usuario no puede modificar los widgets en el control Cuadro de mando. |
| *dashResize* | Modo de edición: el usuario puede cambiar el tamaño de los widgets en el control Cuadro de mando. |



**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando que se va a utilizar.

## Observaciones


### Borrar el modo de edición para el usuario final (a través de AAF)
<a name="borrar_modo_edicion_para_usuario_final_traves_aaf_ELTPARAGRAPHE000149"></a>

La función **DashMode** permite cambiar al modo de edición mediante programación. 

Para evitar que el usuario pase al modo de edición (a través del menú contextual o con un clic largo), utilice [AAFDisable](../WDLang1/1000022018.md) con la constante *aafDashEditMode*.

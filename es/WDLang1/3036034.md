También se pueden utilizar las siguientes funciones para gestionar los ficheros externos: 


|   |   |
| --- | --- |
| [CompleteDir](../WDLang1/3024018.md) | Agrega una barra invertida al final de la cadena, si es necesario. Esta función es útil para construir nombres de archivos completos cuando se desconoce el formato de la ruta (por ejemplo, cuando el usuario introduce una ruta de archivo). |
| [UncompleteDir](../WDLang1/3024044.md) | Elimina el carácter "/" o "\\" del final de una cadena, si es necesario.. Esta función es útil para construir nombres de archivos completos cuando se desconoce el formato de la ruta (por ejemplo, cuando el usuario introduce una ruta de archivo). |


## GenerateGeometricImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera una imagen con formas geométricas de forma aleatoria. Esta imagen puede utilizarse para rellenar temporalmente un control Imagen, por ejemplo. Esto permite rellenar la "foto" de un usuario, mientras el usuario sube una propia.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate a geometric image 
MyImage is Image = GenerateGeometricImage()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GenerateGeometricImage([<Width> [, <Height> [, <Color> [, <Background color>]]]])`
---

**`<Result>: (Variable de tipo Image)`**

Variable de tipo [Image](../WDLang1/1000019650.md) que contiene la imagen generada.

**`<Width>: (Entero opcional)`**

Ancho de la imagen generada, en pixeles. De forma predeterminada, el ancho de la imagen generada es de 80 pixeles.

**`<Height>: (Entero opcional)`**

Altura de la imagen generada, en pixeles. De forma predeterminada, la altura de la imagen generada es la misma que el ancho.

**`<Color>: (Entero opcional)`**

Color de fondo predeterminado de las diferentes formas geométricas. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

- una variable de tipo [Color](../WDLang1/1000019661.md). En este caso, se tiene en cuenta la opacidad especificada en la variable.


Si no se especifica este parámetro, se utilizará un color aleatorio.

**`<Background color>: (Entero opcional)`**

Color de fondo predeterminado de las diferentes formas geométricas. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

- una variable de tipo [Color](../WDLang1/1000019661.md). En este caso, se tiene en cuenta la opacidad especificada en la variable.


Si no se especifica este parámetro, se utilizará un fondo transparente.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La aleatoriedad de la generación de imágenes se basa en la función [Random](../WDLang1/3013037.md). Para evitar obtener las mismas imágenes cada vez que se ejecute el programa, llame a la función [InitRandom](../WDLang1/3013030.md) en el evento de inicialización del proyecto.

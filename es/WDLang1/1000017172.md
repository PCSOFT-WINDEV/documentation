
## WinDisableEffect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desactiva los efectos visuales del motor gráfico de WINDEV (para un uso en modo TSE por ejemplo). Esta función es useful cuando una aplicación debe ser ejecutada en un ordenador con capacidades gráficas limitadas o para el modo TSE si el ancho de banda es limitado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF InTSEMode() = True THEN
	WinDisableEffect(effectAnimation + effectTranslucentBorder + effectDDW + effectBlur)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`WinDisableEffect(<Effect to disable>)`
---

**`<Effect to disable>: (Constante o combinación de constantes de tipo Integer)`**

El efecto a desactivar corresponde a una o más de las siguientes constantes:


|   |   |
| --- | --- |
| *effectAnimation* | Desactiva las animaciones de las ventanas. |
| *effectBlur* | Desactiva el halo alrededor de los botones. |
| *effectDDW* | Desactiva la función OVI. |
| *effectTranslucentBorder* | Deshabilita el Border translúcido de las ventanas (Border utilizado en Windows XP o 2003). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **WinDisableEffect** es equivalente a:

	- [Estilo de dibujo](../WDLang1/3014030.md) sirve para configurar el efecto translúcido de las ventanas.

	- [WinInactiveEffect](../WDLang1/3038055.md) se utiliza para configurar la función OVI.




- En el TSE, por Default: 

	- las animaciones de las ventanas están desactivadas. 

	- la animación relacionado to the OVI (Dim Disabled Window) está desactivada. El OVI está habilitado por Default en TSE.

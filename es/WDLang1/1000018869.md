
## SysRecentDocList( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de documentos abiertos recientemente.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the list of recently opened documents in LIST_RecentDoc
LIST_RecentDoc = SysRecentDocList()
```



```wl
// Display in LIST_RecentDoc 
// the list of recently opened documents and directories
// sorted by date and with the displayed date
Trace(SysRecentDocList(srdlSortDate + srdlDate))
// Displays for example
// C:\MyDoc	20180322105130
// C:\MyDoc\Post.jpg	20180322105130
// C:\MyDoc\Post2.jpg	20180322095020
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysRecentDocList([<Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de documentos abiertos recientemente. Esta lista tiene el siguiente formato:

```txt
<Path document 1> + CR + <Path document 2> + CR + ... + <Path document N>
```
Según las opciones utilizadas, esta lista también puede visualizar: 

- la fecha del último acceso en formato AAAAMMDDHHmmSS.  

- el nombre de los directorios a los que se accede.




**`<Options>: (Constantes enteras combinables)`**



- Modo de senderos Process: 
	


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode.<br><br>ATENCIÓN: Si no se especifica esta opción, las rutas Document que contengan cadenas que no puedan cambiarse a ANSI se sustituirán por las correspondientes ruta cortas. |



- Información visualizada en la lista: 
	


|   |   |
| --- | --- |
| *srdlDate* | Además de la información solicitada, &lt;Resultado&gt; contiene la fecha de último acceso al fichero en formato AAAAMMDDHHmmSS (separada del elemento ruta por un tabulador). |
| *srdlDefault* | &lt;Resultado&gt; contiene la lista de los últimos Document abiertos (archivo ruta correspondiente al Document abierto). |
| *srdlSortDate* | &lt;Resultado&gt; contiene la lista de los últimos directorios a los que se ha accedido y de los últimos documentos abiertos (separados por CR), ordenados desde el más reciente hasta el más antiguo.. <br><br>Esta constante puede combinarse con la constante *srdlDate*: en este caso, se añade la fecha del último acceso (separada de la ruta por un CR). |

## Observaciones


### Asociar una aplicación con una extensión de archivo
<a name="asociar_una_aplicacion_con_una_extension_archivo_ELTPARAGRAPHE000072"></a>

Para asociar una aplicación con una extensión de archivo a través del registro, debe crear las siguientes claves:

- "HKEY_CLASSES_ROOT\\.ext": 
	dónde **anexo** es la extensión. 
	El valor de Default corresponde al tipo de archivo (cadena libre).

- "HKEY_CLASSES_ROOT\\type_name": 
	dónde **type_name** es el nombre del tipo dado a la extensión.
	El valor por defecto corresponde a la description que se mostrará en la columna "Tipo" del explorador de Windows.

- "HKEY_CLASSES_ROOT\\type_name\\shell\\open\\command": 
	El valor de Default es el comando que ejecutará [ShellExecute](../WDLang1/3035007.md) sobre este tipo de archivo. En el comando Line, la cadena %1 será reemplazada por la ruta del archivo.


## RecycleBinDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje de confirmación para borrar un archivo de la papelera de reciclaje. Si el usuario lo valida, el archivo se elimina; de lo contrario, el archivo permanece en la papelera de reciclaje.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Button for filling a List Box control
// Display the elements of the recycle bin in a List Box control
sMyBinList is string
ListDeleteAll(LIST_Bin)
sMyBinList = RecycleBinListFile()
ListAdd(LIST_Bin, sMyBinList)

// -- Button for deleting the element selected in the List Box control
RecycleBinDelete(LIST_Bin[LIST_Bin])
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RecycleBinDelete(<File name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró el mensaje de confirmación de la eliminación,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<File name>: (Cadena de caracteres)`**

Nombre completo de la fuente del archivo (ruta completo y nombre del archivo). Si el archivo se encuentra varias veces en la papelera de reciclaje, todos los archivos serán eliminados. 
Para borrar un solo archivo, el nombre del archivo debe ir seguido de la hora de borrado en el siguiente formato: 

```txt
<File Name> + TAB + <Erase Time>
```

Este formato es devuelto por [ReciclarBinListaArchivo](../WDLang1/1000017162.md).

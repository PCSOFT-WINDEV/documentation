
## SpreadsheetSelectComment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- saber si se seleccionó un comentario en un control Hoja de cálculo,

- seleccionar o deseleccionar un comentario en un control Hoja de cálculo. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the comment in cell A1
SpreadsheetSelectComment(PSHEET_MySpreadsheet, "A1")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Saber si se seleccionó un comentario

`<Result> = SpreadsheetSelectComment(<Spreadsheet control>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la celda en la que se seleccionó un comentario (por ejemplo, "A1"),

- Cadena vacía ("") si no hay ningún comentario seleccionado.  




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 


<a name="SYNTAX2"></a>

### Seleccionar o deseleccionar un comentario

`SpreadsheetSelectComment(<Spreadsheet control> , <Position>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Position>: (Cadena de caracteres)`**



- Nombre de la celda (por ejemplo, "A1") cuyo comentario debe ser seleccionado, visible y editable. Si la celda no tiene comentarios, se crea uno nuevo. 

- Cadena vacía (""). En este caso, si hay un comentario seleccionado, se deselecciona y se valida la entrada realizada en el comentario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Solo se puede seleccionar un comentario a la vez en un control Hoja de cálculo.

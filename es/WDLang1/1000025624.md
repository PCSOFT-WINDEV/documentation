
## Polygon (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Polygon** permite definir los puntos de un polígono. Estos puntos se ordenan y conectan según el orden definido.  

Las características del polígono se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
polTriangle is Polygon
oPoint is Point
oPoint.X = 460
oPoint.Y = 10
Add(polTriangle.Point, oPoint)
oPoint.X = 500
oPoint.Y = 80
Add(polTriangle.Point, oPoint)
oPoint.X = 420
oPoint.Y = 80
Add(polTriangle.Point, [oPoint.X, oPoint.Y])
dPolygon(polTriangle, PastelBlue, LightGreen)


polRectangle is Polygon
polRectangle.Point = [[420,90], [500,90], [500,150], [420,150]]
dPolygon(polRectangle, DarkBrown)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Polygon
<a name="propiedades_especificas_descripcion_variables_tipo_polygon_ELTPARAGRAPHE000031"></a>

Las siguientes propiedades se pueden utilizar para manipular polígonos:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Point | Array de variables de tipo [Point](../Motscles/1000018908.md) | Puntos del polígono. |

|   |   |
| --- | --- |
| [dPolygon](../WDLang1/3029006.md) | Dibuja un polígono: <br><br>- en un control Imagen, <br><br>- en una variable de tipo [Image](../WDLang1/1000019650.md), <br><br>- en una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo),<br><br>- en una variable de tipo [picLayer](../WDLang1/1000024604.md). <br><br><br> |




## &lt;Tipo String&gt;.Reverse (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Invierte los caracteres de una cadena. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString is string = "ABC"
Trace(MyString.Reverse())
// The trace window displays "CBA"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <String>.Reverse()`
---

**`<Result>: (Cadena de caracteres Ansi, Unicode o Binaria)`**

Cadena de caracteres invertida.

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a invertir.

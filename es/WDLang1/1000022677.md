
## ActionUndo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la última acción realizada por el usuario final.. Esta función equivale a un "Deshacer" (Ctrl + Z) realizado por el usuario.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Cancel] button
ActionUndo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ActionUndo([<Window to use>])`
---

**`<Window to use>: (Nombre de la ventana opcional)`**

Nombre de la ventana o ventana interna a utilizar. Si no se especifica este parámetro, se utiliza la ventana actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000032"></a>

- **ActionUndo** no tiene efecto si no hay ninguna acción para cancel.

- Si la última acción es una acción personalizada ejecutada por [ActionDo](../WDLang1/1000022676.md), se ejecuta el procedimiento definido por [ActionDeclareUndo](../WDLang1/1000022678.md).

- La acción cancelada puede ser restaurada: 

	- por Ctrl + Y,

	- mediante la programación con [ActionRedo](../WDLang1/1000022608.md).

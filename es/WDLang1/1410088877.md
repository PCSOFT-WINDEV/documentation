


## <Diagram Variable>.StartModification (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el inicio de una serie de acciones realizadas en una variable de tipo Diagrama en una única Event("deshacer/rehacer").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo




```wl
MyDiagram is Diagram <- DIAGEDT_MyDiagram
MySelection is diagSelection <- DIAGEDT_MyDiagram.Selection
MyDiagram.StartModification()
MySelection.Background.Color = White
MySelection.Border.Color = Black
MyDiagram.EndModification()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Diagram>.StartModification()`
---

**`<Diagram>: (Diagram Variable)`**

Nombre de la variable de tipo [Diagram](../WDLang1/1410088055.md) que se utilizará.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todos los cambios realizados entre la llamada a **<Diagram Variable>.StartModification** y [&lt;Variable Diagram&gt;.EndModification](../WDLang1/1410088878.md) se agrupan en un acción única que se puede deshacer fácilmente.

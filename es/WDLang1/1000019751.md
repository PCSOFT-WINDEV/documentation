
## GestureCurrentPointer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice del puntero (dedo o lápiz óptico) que desencadenó la llamada de un Event multitáctil específico en un control Imagen.

Este índice puede ser utilizado por [GesturePosX](../WDLang1/1000019741.md) y/o [GesturePosY](../WDLang1/1000019745.md) para get la posición vertical y/o horizontal del puntero correspondiente.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the horizontal position of the pointer that triggered
// the call to the "Pressed" event associated to an Image control 
PositionX is int
PositionX = GesturePosX(GestureCurrentPointer())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GestureCurrentPointer()`
---

**`<Result>: (Integro)`**

Índice del puntero (dedo o lápiz) que desencadenó la llamada de un evento "Presionado" o "Liberado" asociado a un control Imagen. El primer puntero que toca la pantalla corresponde al índice 1.

**Observación**: En el simulador o emulador Android, este número es siempre igual a 1.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000066"></a>



- **Condiciones de uso**: 

	- Esta función debe llamarse en los eventos de una control Imagen que desencadenó la llamada a los eventos de cada puntero.: opción "Volver a llamar a los procesos en cada pulsación (varios punteros)" en la pestaña "Detalles" del control description.  

	- Esta función debe ser llamada en uno de los siguientes eventos de la control: "Presionado", "Soltado", "Desplazamiento con el dedo", "Desplazamiento horizontal y vertical" o "Zoom con el dedo". De lo contrario, la función siempre devolverá 1.










<a name="NOTE0_3"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000100"></a>

- Para conocer la posición horizontal (X) de un puntero, utilice [GesturePosX](../WDLang1/1000019741.md).

- Para conocer la posición vertical (Y) de un puntero, utilice [GesturePosY](../WDLang1/1000019745.md).

- Para conocer el número de punteros en Contact con la pantalla, utilice [GestureNbPointer](../WDLang1/1000019750.md).

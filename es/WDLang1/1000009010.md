
## EmulateMenu (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula el siguiente menú que se abrirá y ejecuta automáticamente la opción de menú pasada como parámetro.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Runs the "Exit" option from "_Menu"
EmulateMenu("_Menu", "Exit")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EmulateMenu(<Menu name> , <Option name>)`
---

**`<Menu name>: (Cadena de caracteres)`**



- Nombre del menú a emular,

- Vaciar la cadena ("") para emular el menú de Default.




**`<Option name>: (Cadena de caracteres)`**



- Nombre de la opción que se ejecutará durante la emulación,

- Vaciar la cadena (""") si no hay ninguna opción seleccionada (el menú se cancela)..  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Atención**: La función **EmulateMenu** se ejecutará solo cuando se abra un menú. Si este menú es el especificado en &lt;Nombre del menú&gt;, se ejecutará la opción seleccionada.. De lo contrario, se producirá un error.

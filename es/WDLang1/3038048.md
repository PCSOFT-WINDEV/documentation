


## MenuAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un menú emergente existente en el menú principal de la ventana actual.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add the "MENU_PopupMenu1" menu into the menu option "OPT_File"
MenuAdd(OPT_File, MENU_PopupMenu1)

// Add the "MENU_PopupMenu2" menu into the main menu "_Menu"
MenuAdd(_Menu, MENU_PopupMenu2)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuAdd(<Destination menu> , <Popup menu> [, <Position>])`
---

**`<Destination menu>: (El nombre del menú)`**

Nombre de la opción de menú en la que se debe realizar la inserción.
Si este parámetro corresponde a un menú, se añade el menú contextual a las opciones de este menú.
Si este parámetro corresponde a una opción de menú, esta opción de menú mostrará las opciones de el menú contextual.
Si este parámetro corresponde a una cadena vacía (""), el menú contextual se añade al menú principal.  

**`<Popup menu>: (El nombre del menú)`**

Nombre de el menú contextual que debe ser insertado.

**`<Position>: (Entero opcional)`**

Posición en la que se insertará el menú. 
Si este parámetro no se especifica o si es igual a -1, se añade el menú contextual al final. 
Si este parámetro es igual a 1, el menú contextual se inserta al principio.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Si la ventana no incluye ningún menú principal, se crea este menú.

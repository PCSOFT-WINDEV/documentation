


## Selección de los idiomas soportados por la aplicación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000230"></a>
La elección de los idiomas soportados por la aplicación WINDEV o WINDEV Mobile o por el sitio WEBDEV se realiza en varios niveles:

- elegir los idiomas soportados por el proyecto.

- elegir los idiomas soportados por el análisis.

- elegir los idiomas soportados por los diferentes elementos del proyecto.




<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Idiomas del proyecto
<a name="idiomas_del_proyecto_ELTTEXTE000254"></a>
Los idiomas se definen en la descripción del proyecto: 

- En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

- Seleccione la pestaña "Idioma".

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Projet_multilangue%20-%20HC%20N%B0001.gif&type=thumb)
Los idiomas seleccionados se propondrán al introducir la información sobre los diferentes elementos traducibles (subtítulos de control s, opciones de menú, mensaje de ayuda associated con un control, ...).

Al cambiar el idioma del proyecto, los cambios se tendrán en cuenta automáticamente:

- para cualquier objeto nuevo creado en los diferentes editores (Ventana, Reporte, control, etc.).

- para cualquier elemento abierto en los diferentes editores (Ventana, Reporte, etc.).




El lenguaje de la Default (identificado con el número "1") corresponde al lenguaje utilizado por la Default en tiempo de ejecución. Para modificarlo, use el menú contextual del lenguaje que usará Default.
<a name="NOTE2_2"></a>


### Opciones lingüísticas
<a name="opciones_linguisticas_ELTPARAGRAPHE000052"></a>

Puede definir cómo mostrar los datos numéricos (número, moneda, fecha, etc.) en el proyecto para cada idioma soportado.

Los parámetros utilizados de forma predeterminada son los definidos en las opciones de idioma de Windows (disponibles en el panel de control de Windows).

Estos ajustes se definen en la pestaña "Idiomas" del proyecto description. Para ello:

- Seleccione el idioma deseado. 

- Seleccione el elemento a configurar (número, divisa, ....)

- Modificar los parámetros deseados y validar. 




Observación: Se utiliza un botón para reiniciar los parámetros con los valores de Default.

**Al crear un control Campo de entrada o una columna en un control Tabla** que muestra datos numéricos (número, moneda, fecha, hora, duración, etc.), la máscara de entrada utilizada será automáticamente la máscara definida en las opciones de idioma del proyecto. Esta opción está disponible en ventanas y páginas.

**En tiempo de ejecución**, cuando una columna control Campo de entrada o tabla tiene una máscara "Definida por el proyecto", la máscara de entrada/visualización se adaptará automáticamente de acuerdo con las opciones seleccionadas en el proyecto para el idioma mostrado en tiempo de ejecución.

**Observación**: La opción "Notas adicionales" permite definir: 

- la dirección de la escritura.

- el personaje que usó set.

- el archivo de traducción de WDM.

### Programa ejecutable
<a name="programa_ejecutable_ELTPARAGRAPHE000083"></a>

Al crear el programa ejecutable, tiene la capacidad de especificar los idiomas que serán tenidos en cuenta por el ejecutable.

Puede crear un archivo ejecutable diferente por cada idioma del proyecto. Esta opción permite para reducir el tamaño del ejecutable.

Para obtener más información, consulte [Creación del ejecutable de una aplicación multilingüe](../WDLang1/3054006.md).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Todos los idiomas del proyecto están incluidos en el archivo de Java.

## Idiomas del análisis
<a name="idiomas_del_analisis_ELTTEXTE000290"></a>
Si el proyecto utiliza un análisis, los idiomas del análisis se seleccionan en el editor de análisis. Un mismo análisis puede ser compartido entre diferentes proyectos que no utilizan los mismos idiomas. Por lo tanto, el análisis podría utilizar más idiomas que el proyecto.

Para configurar los idiomas del análisis: 

1. Mostrar el análisis description: en la pestaña "Análisis", en el grupo "Elemento actual", haga clic en "Descripción". 

2. Seleccione la pestaña "Idioma".




Los diferentes idiomas definidos en el análisis estarán disponibles:

- al configurar la información compartida de los campos. La descripción de los controles enlazados a los campos (opciones, títulos, etc.) puede introducirse en los diferentes idiomas que admite el análisis. Al generar un "RAD Aplicación completa" o un RAD de ventanas, esta información se utiliza automáticamente para todos los idiomas comunes al análisis y al proyecto.

- en la información impresa en la documentación técnica (información escrita en la pestaña "Notas" del archivo o elemento description).

- en la información gestionada por "Reports & Queries". De hecho, si "Reports & Queries" se suministra con su aplicación WINDEV, usted tiene la capacidad de traducir el nombre del archivo y el nombre de cada elemento.. Los usuarios podrán utilizar los nombres en el idioma deseado al crear los reportes y consultas. Esta información puede ser introducida en la pestaña "Informes y Consultas" del archivo y el elemento description. 
	**Observación**: "Informes y Consultas" está disponible sólo para aplicaciones WINDEV.

## Idiomas de los diferentes elementos del proyecto
<a name="idiomas_los_diferentes_elementos_del_proyecto_ELTTEXTE000314"></a>
En Default, los diferentes elementos del proyecto (ventanas, páginas, informes, código, clases, conjuntos de procedimientos, etc.) soportan los mismos idiomas que el proyecto en el que fueron creados.

En algunos casos, un elemento puede admitir más idiomas que el proyecto (por ejemplo, cuando el elemento es compartido entre varios proyectos que utilizan diferentes idiomas).
<a name="NOTE4_2"></a>


### Visualización de los idiomas soportados por un elemento
<a name="visualizacion_los_idiomas_soportados_por_elemento_ELTPARAGRAPHE000142"></a>

Para saber qué idiomas están soportados por un elemento:

- para las ventanas, las páginas y los informes: mostrar la pestaña "Idioma" de la ventana de description.

- para las clases y los conjuntos de procedimientos:

	1. Seleccione la clase o la set de procedimientos en el panel "Explorador del proyecto".

	2. Seleccione "Idiomas" de el menú contextual.







En la ventana para la gestión de los idiomas de los elementos, tiene la posibilidad de añadir un nuevo idioma.. Este idioma será soportado por el elemento seleccionado. Para añadir un idioma a todo el proyecto, el idioma debe añadirse al proyecto description (véase "Idiomas apoyados por el proyecto").
<a name="NOTE4_3"></a>


### Sincronización de los lenguajes de los elementos con el proyecto
<a name="sincronizacion_los_lenguajes_los_elementos_con_proyecto_ELTPARAGRAPHE000157"></a>

El botón "Sincronizar con el proyecto" que se encuentra en la ventana de gestión de los idiomas de los elementos sirve para sincronizar los idiomas del elemento actual con los definidos para el proyecto.

Después de la sincronización, sólo se seleccionarán los idiomas compatibles con el proyecto actual.. Estos idiomas no se pueden deseleccionar. Los otros idiomas (no soportados por el proyecto) serán eliminados y las traducciones existentes se perderán.
<a name="NOTE4_4"></a>


### Cambiar el idioma mostrado en los editores
<a name="cambiar_idioma_mostrado_los_editores_ELTPARAGRAPHE000166"></a>

Para cambiar el idioma mostrado en los editores (ventana, página, código, reporte), vaya a la pestaña "Vista", grupo "Opciones" y haga clic en "Idioma mostrado".. El acceso directo correspondiente es:

- Ctrl + Alt + L para ir al siguiente idioma. 

- Mayúsculas + Alt + L para ir al idioma anterior.

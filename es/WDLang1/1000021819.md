
## fCacheDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la ruta de directorio que se utilizará para almacenar los datos en caché en el espacio de almacenamiento interno de la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save a file in cache on the internal storage space
sCacheDir is string = fCacheDir()
IF sCacheDir <> "" THEN
	fSaveText(sCacheDir + "file.txt", MyString)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fCacheDir()`
---

**`<Result>: (Cadena de caracteres)`**

Ruta del directorio de caché de la aplicación en el espacio de almacenamiento interno de la aplicación.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Gestionar el directorio de caché interno de la aplicación
<a name="gestionar_directorio_cache_interno_aplicacion_ELTPARAGRAPHE000032"></a>

Los datos encontrados en este directorio serán borrados automáticamente por el sistema.: 

- si el sistema requiere espacio adicional en el disco. 

- si el usuario desea borrar los datos de la caché.

- si la aplicación está desinstalada.



<a name="NOTE0_2"></a>


### Consejos para usar el directorio de caché interno de la aplicación
<a name="consejos_para_usar_directorio_cache_interno_aplicacion_ELTPARAGRAPHE000043"></a>

- Le aconsejamos que limite el volumen de datos almacenados en la caché y que los elimine tan pronto como dejen de utilizarse.

- Los datos almacenados en el directorio de caché de la aplicación en el espacio de almacenamiento interno no pueden ser accedidos por las otras aplicaciones.

- [SysCacheExternalStorage](../WDLang1/1000021815.md) se utiliza para averiguar el ruta del directorio de caché de la aplicación en la memoria externa del dispositivo.

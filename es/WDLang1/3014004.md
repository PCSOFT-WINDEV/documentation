
## Constantes utilizadas por ExecutePresetAction
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por [ExecutePresetAction](../WDLang1/3014028.md):

| Constante | Detalles |
| --- | --- |
| **actPaste** | Pegar contenido del portapapeles |
| **actCopy** | copy el contenido seleccionado en el portapapeles |
| **actCut** | Cortar el contenido seleccionado del portapapeles |
| **actEraseRWDisk** | Borrar el contenido de un directorio en CD-RW/DVD-RW |
| **actSendEmail** | Enviar un email |
| **actBrowseDirectory** | Abrir un explorador en un directorio |
| **actBurnADirectory** | Grabar el contenido de un directorio en un CD-R/DVD-R |
| **actMinimize** | Minimizar la ventana actual |
| **actWinImageToClipboard** | Copia la Image de la ventana actual en el portapapeles |
| **actRunContextSensitiveHelp** | Cambiar al modo "Ayuda sensible al contexto |
| **actRunInternetSite** | Abrir un navegador INTERNET en una URL |
| **actReadLast** | Lee la última Record (en el archivo de datos actualmente en uso). |
| **actReadPrevious** | Lee la anterior Record (en el archivo de datos actualmente en uso). |
| **actReadFirst** | Lee la primera Record (en el archivo de datos actualmente en uso). |
| **actReadNext** | Lee la siguiente Record (en el archivo de datos actualmente en uso). |
| **actOpenReport** | Abrir la vista previa de la impresión de un reporte |

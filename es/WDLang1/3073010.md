


## Funciones del sistema
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones del sistema son las siguientes:



|   |   |
| --- | --- |
| [SysAndroidVersion](../WDLang1/1000019785.md) | Devuelve la información sobre la versión de Android utilizada por la aplicación. |
| [SysBatteryOptimBackground](../WDLang1/1000026008.md) | Se utiliza para gestionar la optimización de la batería: <br><br>- Identifica el modo de optimización de la batería para la aplicación actual. <br><br>- Indica al usuario que deshabilite la optimización de la batería para una aplicación determinada en el dispositivo.<br><br><br>Esta función puede utilizarse para indicar al sistema operativo que deje de impedir que la aplicación se ejecute en la Background. |
| [SysCacheExternalStorage](../WDLang1/1000021815.md) | Devuelve el ruta del directorio que se utilizará para almacenar los datos en caché en el espacio de almacenamiento externo de la aplicación. |
| [SysChangeEnvironment](../WDLang1/1000017241.md) | Modifica las variables de entorno del sistema operativo que se encuentran en el ordenador actual. |
| [SysChangeScreenResolution](../WDLang1/1000017020.md) | Modifica la resolución de una pantalla. |
| [SysColor](../WDLang1/3073011.md) | Identifica o modifica la Color de un elemento de Windows. |
| [SysColorRes](../WDLang1/3073012.md) | Devuelve la resolución de la pantalla en número de colores. |
| [SysDetectLockedSession](../WDLang1/1000021185.md) | Detecta el cambio de bloqueo de la sesión de Windows. |
| [SysDeviceInfo](../WDLang1/1000023103.md) | Recupera información del dispositivo móvil donde se ejecuta la aplicación. |
| [SysDir](../WDLang1/3073026.md) | Devuelve el ruta de un directorio del sistema. |
| [SysDirExternalStorage](../WDLang1/1000021793.md) | Devuelve la ruta del directorio que se utilizará para almacenar archivos en el espacio de almacenamiento externo del dispositivo Android. |
| [SysDirStorageCard](../WDLang1/3073034.md) | Devuelve la ruta del directorio raíz del espacio de almacenamiento externo primario. |
| [SysEnvironment](../WDLang1/3073014.md) | Devuelve las variables de entorno del sistema operativo para el ordenador actual. |
| [SysErrorMode](../WDLang1/3073020.md) | Modifica el modo de visualización de los errores. |
| [SysGetDC](../WDLang1/3073023.md) | Recupera el DC (Contexto del Dispositivo) de una ventana, control o pantalla. |
| [SysGetExtLink](../WDLang1/1000021896.md) | Devuelve el comando relacionado a una extensión. |
| [SysIconAdd](../WDLang1/3073001.md) | Añade una bandeja de sistema Icon (en la parte inferior derecha Corner de la pantalla). |
| [SysIconAddImage](../WDLang1/1000018874.md) | Añade una Icon que superpone la aplicación Icon en la barra de tareas. |
| [SysIconBadge](../WDLang1/1000020796.md) | Devuelve o modifica el valor mostrado en la placa associated con el Icon de una aplicación iOS o Universal Windows 10 App. |
| [SysIconDelete](../WDLang1/3073022.md) | Borra una Icon de la barra de tareas (junto al reloj). |
| [SysIconDeleteImage](../WDLang1/1000018875.md) | Borra la Icon que superpone la aplicación Icon en la barra de tareas. |
| [SysIconDisplayTooltip](../WDLang1/1000021767.md) | Muestra un tooltip para una Icon que se encuentra en la barra de tareas. |
| [SysIconize](../WDLang1/3073017.md) | Permite saber si una ventana está minimizada. |
| [SysIconModify](../WDLang1/3073027.md) | Modifica el archivo Icon y/o el texto hover del Icon en la barra de tareas (Corner inferior derecha de la pantalla). |
| [SysIdentifier](../WDLang1/1000020619.md) | Devuelve el identificador único del dispositivo móvil actual. |
| [SysIMEI](../WDLang1/3073029.md) | Devuelve el número IMEI de un Android Phone. |
| [SysInstance](../WDLang1/3073018.md) | Devuelve la instancia de un módulo. |
| [SysIOSVersion](../WDLang1/1000020555.md) | Devuelve información sobre la versión de iOS utilizada por la aplicación. |
| [SysLinkExtension](../WDLang1/1000021892.md) | Asocia una extensión (.txt, .avi, etc.) a un programa de ejecución por defecto. |
| [SysListScreen](../WDLang1/1000017014.md) | Devuelve la lista de pantallas conectadas a una o más tarjetas de vídeo. |
| [SysListScreenResolution](../WDLang1/1000017018.md) | Devuelve las posibles resoluciones de una pantalla. |
| [SysListVideoCard](../WDLang1/1000017016.md) | Devuelve el nombre de las tarjetas de vídeo conectadas al ordenador actual. |
| [SysMemoryExternalStorage](../WDLang1/1000021817.md) | Devuelve la cantidad de memoria disponible en el área de almacenamiento externo del dispositivo. |
| [SysMetric](../WDLang1/3073019.md) | Devuelve la resolución de un elemento específico. |
| [SysNameExe](../WDLang1/3073015.md) | Devuelve el nombre y la ruta completa de una aplicación. |
| [SysNameMainScreen](../WDLang1/1000017017.md) | Devuelve el nombre de la pantalla principal de el equipo. |
| [SysNation](../WDLang1/1000021876.md) | Devuelve la nación actualmente utilizada por el sistema. |
| [SysNbExternalStorage](../WDLang1/1000021816.md) | Devuelve el número de espacios de almacenamiento externos encontrados en un dispositivo Android. |
| [SysNumberScreen](../WDLang1/1000017021.md) | Devuelve el número de pantallas actualmente conectadas al ordenador actual. |
| [SysPowerStatus](../WDLang1/1000026039.md) | Utiliza para get diversas informaciones sobre la batería (principal o secundaria) del dispositivo en el que se ejecuta la aplicación. |
| [SysRecentDocAdd](../WDLang1/1000018867.md) | Añade un archivo a la lista de documentos abiertos recientemente. |
| [SysRecentDocList](../WDLang1/1000018869.md) | Devuelve la lista de documentos abiertos recientemente. |
| [SysReleaseDC](../WDLang1/3073021.md) | Libera la corriente DC (Device Context) recuperada por [SysGetDC](../WDLang1/3073023.md). |
| [SysRemovableExternalStorage](../WDLang1/1000021818.md) | Permite saber si el espacio de almacenamiento externo corresponde a la memoria móvil (tarjeta SD). |
| [SysScreenOrientation](../WDLang1/3073031.md) | Permite: <br><br>- obtiene la orientación de la pantalla del dispositivo (Android, iOS o Windows Universal Phone).<br><br>- establece la orientación de la pantalla del dispositivo (Android Phone).<br><br><br> |
| [SysScreenRectangle](../WDLang1/1000019421.md) | Devuelve las coordenadas del rectángulo de visualización correspondiente a todas las pantallas.. |
| [SysScreenResolution](../WDLang1/1000017019.md) | Devuelve la resolución de una pantalla. |
| [SysSerialNum](../WDLang1/3073033.md) | Devuelve el número IMEI del dispositivo actual. |
| [SysSetFocus](../WDLang1/3073004.md) | Fija el foco en una ventana o en control. |
| [SysShutdown](../WDLang1/3073013.md) | Detiene el sistema o cierra la sesión actual de Windows. |
| [SysSpace](../WDLang1/3073002.md) | Devuelve el espacio de memoria física del ordenador actual. |
| [SysStandby](../WDLang1/3073032.md) | Permite: <br><br>- activar y desactivar el modo de espera (Android)<br><br>- muestra la pantalla del sistema utilizada para configurar el modo de espera (Universal Windows Apps)<br><br><br> |
| [SysStatusExternalStorage](../WDLang1/1000021798.md) | Devuelve el estado de disponibilidad del espacio de almacenamiento externo del dispositivo. |
| [SysStatusStorageCard](../WDLang1/1000019554.md) | Devuelve el estado de disponibilidad de la tarjeta de almacenamiento en el dispositivo. |
| [SysTheme](../WDLang1/1000019646.md) | Devuelve el tema actual utilizado en el dispositivo móvil. |
| [SysThumbnailAddButton](../WDLang1/1000018871.md) | Añade un botón a la miniatura de la aplicación en la barra de tareas. |
| [SysThumbnailDeleteAll](../WDLang1/1000018873.md) | Borra todos los botones associated con la miniatura de la aplicación de la barra de tareas. |
| [SysThumbnailDeleteButton](../WDLang1/1000018872.md) | Elimina uno de los botones que se encuentran en la miniatura de la aplicación de la barra de tareas. |
| [SysThumbnailModifyButton](../WDLang1/1000018956.md) | Modifica uno de los botones que se encuentran en la miniatura de la aplicación en la barra de tareas. |
| [SysUnlinkExtension](../WDLang1/1000021914.md) | Elimina la asociación entre una extensión y un comando creado por [SysLinkExtension](../WDLang1/1000021892.md). |
| [SysValidHandle](../WDLang1/3073009.md) | Verifica la validez de una ventana manipular. |
| [SysVersion](../WDLang1/3073028.md) | Devuelve información sobre la versión PHP utilizada en el servidor actual. |
| [SysWinActive](../WDLang1/3073005.md) | Devuelve la manipular de la ventana de la Active o activa una ventana. |
| [SysWindowsVersion](../WDLang1/3073025.md) | Devuelve información sobre la versión de Windows utilizada en el equipo actual. |
| [SysWinHandle](../WDLang1/3073007.md) | Devuelve la manipular de una ventana identificada por su título. |
| [SysWinShow](../WDLang1/3073003.md) | Modifica el estado de visualización de una ventana. |
| [SysWinTitle](../WDLang1/3073008.md) | Devuelve el título de la ventana. |
| [SysXRes](../WDLang1/3073024.md) | Devuelve la resolución horizontal de la pantalla. |
| [SysYRes](../WDLang1/3073006.md) | Devuelve la resolución vertical de la pantalla. |

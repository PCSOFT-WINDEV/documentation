
## RibbonCloseGrouping (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el grupo actualmente abierto en una control Cinta de opciones. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF Length(EDT_InRibbon)>= 3 THEN
	RibbonCloseGrouping()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`RibbonCloseGrouping()`

<a name="NOTE0_1"></a>

## Observaciones


### Cómo funcionan los grupos
<a name="como_funcionan_los_grupos_ELTPARAGRAPHE000030"></a>

En un control Cinta de opciones, los grupos pequeños muestran un Icon y una flecha que sirve para abrir una ventana emergente que contiene todos los controles.

Por Default, este popup está cerrado: 

- cuando el usuario hace clic en un botón,

- cuando el usuario realiza una selección en una control Combo Box.




Para establecer una acción diferente que cierre el grupo (por ejemplo, introducir 3 caracteres), utilice **RibbonCloseGrouping** en el Process deseado. 

**Observación**: Si no hay ninguna ventana popup abierta, la función **RibbonCloseGrouping** no realiza ninguna acción y no devuelve errores.

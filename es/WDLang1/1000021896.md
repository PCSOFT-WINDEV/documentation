
## SysGetExtLink( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el comando relacionado a una extensión. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Links the ".txt" extension to the "app.exe" executable
SysLinkExtension(".txt", "C:\Program Files\MyApp\app.exe ""%1""", sysCurrentUser)
...
SysGetExtLink(".txt")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysGetExtLink(<Extension> [, <User>])`
---

**`<Result>: (Cadena de caracteres)`**



- Orden relacionado a la extensión,

- Vaciar la cadena (""") si no hay ningún comando relacionado a la extensión. 




**`<Extension>: (Cadena de caracteres)`**

Extensión de uso. 

**`<User>: (Constante opcional de tipo Integer)`**

La asociación entre la aplicación y la extensión se guarda en el registro.. Hay dos opciones disponibles: 


|   |   |
| --- | --- |
| *sysAllUsers* | La asociación se busca para todos los usuarios de la computadora actual. <br><br>Observación: Debe tener los derechos de "Administrador" para ejecutar esta operación. |
| *sysCurrentUser*<br>(Valor predeterminado) | La asociación se busca sólo para el usuario actual. |

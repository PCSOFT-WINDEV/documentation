
## SelecciónColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las características de la banda de selección en un control Tabla, control Combo Box o control List Box.. Las nuevas características se tienen en cuenta inmediatamente.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Changes the color of selection bar in "TABLE_Orders" control
SelectionColor(TABLE_Orders, RGB(255,200,200), LightGreen)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SelectionColor(<Control used> , <Text color> , <Background color>)`
---

**`<Control used>: (Nombre del control)`**

Nombre del control a manipular: Tabla, List Box o control Combo Box. El Color del banda de selección se modificará para este control.

**`<Text color>: (Integro)`**

Nueva Color para el texto de los elementos seleccionados en la control. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Background color>: (Integro)`**

Nuevo fondo Color para la banda de selección. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Error
<a name="error_ELTPARAGRAPHE000068"></a>

Se produce un error de WLanguage en tiempo de ejecución si el nombre del control no se corresponde con un control Tabla, un control List Box o un control Combo Box.
<a name="NOTE0_2"></a>




### Varios
<a name="varios_ELTPARAGRAPHE000076"></a>

**SelecciónColor** puede utilizarse para definir las características banda de selección de un control Combo Box en un control Tabla.

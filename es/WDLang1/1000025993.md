


## HTMLEditorGetDocument (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera de forma asíncrona la Document HTML que se está mostrando en una control de edición HTML. 

**Observación**: Esta función es especialmente útil para las aplicaciones Android e iOS para las que el lenguaje [Document](../Proprietes/1000026145.md) WLanguage Property no está disponible en modo de lectura. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
HTMLEditorGetDocument(HTMEDT_MyHTMLEditor, HTMLEditorGetDocument_Callback)
INTERNAL PROCEDURE HTMLEditorGetDocument_Callback(doc is htmlDocument)
	STC_Title = doc.title
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HTMLEditorGetDocument(<HTML Editor control> , <WLanguage procedure>)`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control. 

**`<WLanguage procedure>: (Nombre de Procedure)`**

WLanguage Procedure ("callback") ejecutado una vez recuperada la Document. Este procedimiento tiene el siguiente formato: 

```txt
PROCEDURE <Procedure name>(MyDocument is htmlDocument)
```
donde &lt;MiDocumento&gt; es una [htmlDocument](../WDLang5/1000025910.md) Variable que corresponde a la Document HTML mostrada en el Editor HTML control.

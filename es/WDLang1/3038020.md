
## WinSize (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**WinSize**:

- devuelve o modifica el modo de visualización de la ventana (maximizado, minimizado o Normal)
	![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Funcionalidad no disponible.

- modifica la altura y/o el ancho de la ventana,

- mueve la ventana y modifica la altura y/o el ancho de una ventana.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Increase the width and height of "WIN_Edit" window
WinSize(WIN_Edit, WIN_Edit.Width + 20, WIN_Edit.Height + 30)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificar la altura y/o anchura de la ventana

`WinSize(<Window> , <Width> , <Height>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre o alias de la ventana que se va a utilizar. 
Si este parámetro corresponde a una cadena vacía ("""), se modifica la anchura y/o la altura de la ventana actual.



**`<Width>: (Integro)`**

Nuevo ancho total de ventana. Este valor debe estar entre la anchura mínima y máxima (devuelta por las propiedades [MinWidth](../Proprietes/2510062.md) y [MaxWidth](../Proprietes/2510068.md)).
Para get la anchura actual de una ventana, utilice [anchura](../Proprietes/2510059.md).
Para no modificar este parámetro, utilice la constante *Default*.

**`<Height>: (Integro)`**

Nueva altura total de la ventana. Este valor debe estar entre la altura mínima y máxima (devuelta por las propiedades [MinHeight](../Proprietes/2510035.md) y [MaxHeight](../Proprietes/2510065.md)).
Para get la altura actual de una ventana, utilice [Altura](../Proprietes/2510050.md).
Para no modificar el parámetro, utilice la constante *Default*.  


<a name="SYNTAX2"></a>

### Mover una ventana y/o modificar su altura y/o anchura

`WinSize(<Window> , <Horizontal position> , <Vertical position> , <Width> , <Height>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre o alias de la ventana que se va a utilizar. 
Si este parámetro corresponde a una cadena vacía ("""), se modifica el ancho y/o alto de la ventana actual y se mueve la ventana actual.



**`<Horizontal position>: (Integro)`**

Nueva posición horizontal de la ventana en relación con la parte superior izquierda de la pantalla Corner.
Para no modificar este parámetro, utilice la constante *Default*.

**`<Vertical position>: (Integro)`**

Nueva posición vertical de la ventana en relación con la parte superior izquierda de la pantalla Corner.
Para no modificar este parámetro, utilice la constante *Default*.

**`<Width>: (Integro)`**

Nuevo ancho total de ventana. Este valor debe estar entre la anchura mínima y máxima (devuelta por las propiedades [MinWidth](../Proprietes/2510062.md) y [MaxWidth](../Proprietes/2510068.md)).
Para get la anchura actual de una ventana, utilice [anchura](../Proprietes/2510059.md).
Para no modificar este parámetro, utilice la constante *Default*.

**`<Height>: (Integro)`**

Nueva altura total de la ventana. Este valor debe estar entre la altura mínima y máxima (devuelta por las propiedades [MinHeight](../Proprietes/2510035.md) y [MaxHeight](../Proprietes/2510065.md)).
Para get la altura actual de una ventana, utilice [Altura](../Proprietes/2510050.md).
Para no modificar el parámetro, utilice la constante *Default*.

### Obtener o ajustar el modo de visualización de una ventana

`<Result> = WinSize([<Window> [, <Display mode>]])`
---

**`<Result>: (Constante)`**

El modo de visualización de ventanas corresponde a uno de los siguientes modos de visualización:

- actual si no se especifica el modo de visualización &lt;Display mode&gt;,

- antes de la modificación si se especifica &lt;Modo de visualización&gt;.




Este parámetro puede tener valor:


|   |   |
| --- | --- |
| *MaxSize* | Ventana maximizada. |
| *MinSize* | Ventana minimizada.<br><br> |
| *NormalSize* | Ventana no maximizada, ni minimizada. |



**`<Window>: (Nombre de ventana)`**

Nombre o alias de la ventana que se va a utilizar. 
Si no se especifica este parámetro o si corresponde a una cadena vacía ("""), se vuelve al modo de visualización de la ventana actual.



**`<Display mode>: (Constante opcional)`**

Especifica el nuevo modo de visualización de ventanas:


|   |   |
| --- | --- |
| *MaxSize* | Ventana maximizada. |
| *MinSize* | Ventana minimizada.<br><br> |
| *NormalSize* | Ventana no maximizada, ni minimizada. |

## Observaciones




### Ventanas no redimensionables
<a name="ventanas_redimensionables_ELTPARAGRAPHE000222"></a>

La función **WinSize** permite modificar el tamaño de una ventana, incluso si esta ventana es de tipo "No redimensionable".
<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Modo de visualización
<a name="modo_visualizacion_ELTPARAGRAPHE000233"></a>

Si el tamaño de la ventana excede la resolución de la pantalla y si "barras de desplazamiento de automatic" está marcada (pestaña "Estilo" en la descripción de la ventana), algunas barras de desplazamiento se añaden a automatic para desplazar la ventana.

Por ejemplo, se creó una ventana en 800\*600 a pantalla completa (o 240\*600).. Algunas barras de desplazamiento aparecerán cuando esta ventana se muestre en 640\*480 (o 240\*320).
<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Botones de la barra de título
<a name="botones_barra_titulo_ELTPARAGRAPHE000244"></a>

Sólo las ventanas que contienen un botón de maximizar en su barra de título pueden ser redimensionadas con el ratón.

Las ventanas sin botón de maximizar en su barra de título no se pueden maximizar. En este caso, se generará un error.

Las ventanas sin botón de minimizar en su barra de título no se pueden minimizar. En este caso, se generará un error.
<a name="NOTE0_4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Ventana superior/ventana inferior
<a name="ventana_superiorventana_inferior_ELTPARAGRAPHE000257"></a>

Cuando se minimiza una ventana principal, todas las ventanas secundarias se minimizan al mismo tiempo. En el escritorio de Windows sólo se encuentra la Icon de la ventana principal.


## SysListScreen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de pantallas conectadas a una o más tarjetas de vídeo..  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the list of screens for the current PC
ListAdd(LIST_ScreenList, SysListScreen())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysListScreen([<Name of video card>])`
---

**`<Result>: (Cadena de caracteres)`**



- Lista de nombres de pantalla separados por caracteres CR. Esta lista tiene el siguiente formato:
	
	```txt
	<Name Screen 1> + CR + <Name Screen 2> + CR + ... + CR + <Name Screen N>
	```

	Si la función del sistema no puede ser manejada, &lt;Result&gt; corresponde a una cadena vacía (""").

- Cadena vacía ("") si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.




**`<Name of video card>: (Cadena de caracteres opcional)`**

Nombre de la tarjeta de vídeo para la que deben listarse las pantallas. El nombre de las tarjetas de vídeo es devuelto por [SysListVideoCard](../WDLang1/1000017016.md). Si no se especifica este parámetro, se listan las pantallas de todas las tarjetas de vídeo.

Si el nombre especificado no existe, &lt;Result&gt; corresponde a una cadena vacía (""").


## Power( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elevar un número a una potencia.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// 8 to the power of 5
ResPower = Power(8, 5)
// Returns 32768
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Power(<Number to raise> , <Exponent>)`
---

**`<Result>: (Real)`**

Resultado de &lt;Número a elevar&gt; a la potencia de &lt;Exponente&gt;.

**`<Number to raise>: (Entero o real)`**

Número que será elevado al poder.

**`<Exponent>: (Entero o real)`**

Exponente.

### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000076"></a>

**Power** es equivalente al operador aritmético **^**. La sintaxis de este operador es la siguiente: 


```txt
<Number to Raise> ^ <Exponent>
```

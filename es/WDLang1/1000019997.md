
## PlanificadorFondoColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define una Color de fondo para un periodo de tiempo en una control Planificador (periodo de tiempo correspondiente a la hora de comer, un día festivo, etc.). 

**Atención**: El color de fondo modificado no es el de Appointment sino el del período de tiempo especificado. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// November 12, 13 and 14, 2017 will be displayed on a red background
SchedulerBackgroundColor(SCH_MySchedule, "20171012", "20171015", "Room ABC", iLightRed) 
SchedulerBackgroundColor(SCH_MySchedule, "20171012", "20171015", "Green room", iLightRed) 
SchedulerBackgroundColor(SCH_MySchedule, "20171019", "20171020", "Green Room", iLightGreen)
SchedulerBackgroundColor(SCH_MySchedule, "20171017", "20171019", "", iDarkBlue)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerBackgroundColor(<Scheduler control> , <Start of time period> , <End of time period> , <Resource> , <Color>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de la control Planificador utilizada, que se encuentra en una ventana.

**`<Start of time period>: (Cadena de caracteres o variable de tipo DateTime)`**

Fecha y hora de inicio del período de tiempo a Color.

- Si el formato de fecha es 0000000000HHMMSS, todos los días se ven afectados.

- Si el formato de fecha es AAAAMMDDD, todas las horas del día se ven afectadas.




**`<End of time period>: (Cadena de caracteres o variable de tipo DateTime)`**

Fecha y hora de finalización del período de tiempo a Color.

**`<Resource>: (Cadena de caracteres)`**



- Nombre de la Resource afectada por el cambio de fondo Color para el período de tiempo. 

- Cadena vacía ("") para modificar la Color de fondo del período de tiempo para todos los recursos. 




**`<Color>: (Entero o constante de tipo Integer)`**

Antecedentes Color de la época. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Observación: Si este parámetro corresponde a -1 o a la constante *transparent*, el color de fondo predeterminado se restaura.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La granularidad del período de tiempo utilizando un fondo específico Color se limita a la granularidad del dibujo.

- Los períodos de tiempo no deben solaparse. De lo contrario, el resultado puede diferir del esperado..

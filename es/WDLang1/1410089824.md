


## &lt;Tipo ANSI String&gt;.Contains (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite comprobar si una cadena contiene una subcadena.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sMyString is string = "Antananarivo"
IF sMyString.Contains("NA") = False THEN
	Info("This string does not contain the NA syllabus")
END
```


<a name="1000019893_Example2"></a>

```wl
arrString is array of strings = ["WINDEV","WEBDEV"]
sTest is string = "WEBDEV 28"

IF sTest.Contains(arrString) THEN
	Info("This string contains WINDEV or WEBDEV")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Comprobar si una cadena contiene una subcadena

`<Result> = <String to analyze>.Contains(<Substring> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la cadena contiene una subcadena. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<String to analyze>: (Cadena a manipular)`**

Cadena de caracteres a manipular.

**`<Substring>: (Cadena de caracteres ANSI, Unicode o Binary)`**

Subcadena buscada.

**`<Options>: (Constante de tipo Integer)`**

Método para buscar la subcadena: 



|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios). |


De forma predeterminada, la búsqueda distingue entre mayúsculas y minúsculas, y tiene en cuenta todas las cadenas especificadas (incluso si forman parte de una palabra).


<a name="SYNTAX2"></a>

### Comprobar si una cadena contiene cadenas de un array

`<Result> = <String to analyze>.Contains(<Array of search strings> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la cadena contiene una de las cadenas del array. 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<String to analyze>: (Cadena a manipular)`**

Cadena de caracteres a manipular.

**`<Array of search strings>: (Array)`**

Nombre del array que contiene las cadenas buscadas.

**`<Options>: (Constante de tipo Integer)`**

Método para buscar la subcadena: 



|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios). |


De forma predeterminada, la búsqueda distingue entre mayúsculas y minúsculas, y tiene en cuenta todas las cadenas especificadas (incluso si forman parte de una palabra).


## AppUPDAvailable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Le permite saber si una actualización está disponible para la aplicación especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF AppUPDAvailable() = appUPDAvailMandatory THEN
	IF YesNo("An update is available for the application.", ...
		  "Do you want to perform this update?") = Yes THEN
			  AppInstallUPD()
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppUPDAvailable([<Application path>])`
---

**`<Result>: (Constante de tipo Integer)`**

Devuelve:

- 0 si se ha producido un error al buscar la actualización. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *appUPDAvailMandatory* | Una actualización obligatoria está disponible. |
| *appUPDAvailOptional* | Una actualización opcional está disponible. |
| *appUPDNotAvail* | No hay actualizaciones disponibles. |






**`<Application path>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable a utilizar. Si no se especifica este parámetro, se gestionará la aplicación actual..

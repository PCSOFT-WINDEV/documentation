


## &lt;Hoja de cálculo&gt;.DeleteWorksheet (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una hoja de trabajo de un control Hoja de cálculo encontrado en una ventana..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all worksheets except for the first one
FOR I = 1 TO PSHEET_MySpreadsheet.NbWorksheet TO 2 STEP -1
	PSHEET_MySpreadsheet.DeleteWorksheet(i)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Suprimir una hoja de trabajo identificada por su número

`<Spreadsheet control>.DeleteWorksheet([<Worksheet number>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Worksheet number>: (Entero opcional)`**

Número de la hoja de trabajo a eliminar. Si no se especifica este parámetro, se elimina la hoja de trabajo actual..


<a name="SYNTAX2"></a>

### Suprimir una hoja de trabajo identificada por su nombre

`<Spreadsheet control>.DeleteWorksheet(<Worksheet name>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Worksheet name>: (Cadena de caracteres)`**

Nombre de la hoja de trabajo a borrar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si sólo queda una hoja de trabajo en el control Hoja de cálculo, **&lt;Hoja de cálculo&gt;.DeleteWorksheet** no tiene ningún efecto. 

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

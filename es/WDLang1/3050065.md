
## ArcTan( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la tangente de arco de un valor numérico (entero o real).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the arc tangent of 12
ResTang = ArcTan(12)
// Returns 85.2363583093
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArcTan(<Numeric value>)`
---

**`<Result>: (Real)`**

Arco Tangent( ángulo en grados de la tangente) del valor numérico especificado, incluido en el [-90, 90] interval (excluyendo los límites).

**`<Numeric value>: (Entero o real)`**

Valor numérico (tangente) a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### ¿Cómo se encuentra un ángulo usando su tangente?
<a name="¿como_encuentra_angulo_usando_tangente_ELTPARAGRAPHE000069"></a>

En un triángulo rectángulo, la tangente de un ángulo es igual a la relación del lado opuesto al adyacente. ¿Cómo encuentro el ángulo? Simplemente utiliza **ArcTan**. 

Ejemplo: El siguiente código: 

```wl
Trace("ArcTan(1) = " + ArcTan(1))
Trace("ArcTan(3/4) = " + ArcTan(3/4))
```
muestra los siguientes elementos en la ventana de salida: 

```txt
ArcTan(1) = 45
ArcTan(3/4) = 36.86989764584
```




## Funciones de gestión de valores binarios
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar valores binarios:



|   |   |
| --- | --- |
| [BinaryAND](../WDLang1/3050010.md) | Devuelve el resultado de un AND lógico (poco a poco) entre dos valores. |
| [BinaryNOT](../WDLang1/3050011.md) | Devuelve el resultado de un NOT lógico (bit a bit) realizado sobre un valor. |
| [BinaryOR](../WDLang1/3050038.md) | Devuelve el resultado del OR lógico (a nivel de bits) entre dos valores. |
| [BinaryXOR](../WDLang1/3050023.md) | Devuelve el resultado del OR exclusivo lógico (a nivel de bits) entre dos valores. |




## Restaurar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una ventana WINDEV a su tamaño inicial.

**Observación**: **Restaurar** realiza la misma acción que el botón de restauración situado en la parte superior derecha corner de las ventanas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the "WIN_EditCustomer" window in its initial size
Restore(WIN_EditCustomer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Restore([<Window>])`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que se debe mostrar en su tamaño inicial.
Si no se especifica este parámetro o si corresponde a una cadena vacía ("""), la ventana actual se muestra en su tamaño inicial.



<a name="XComponent"></a>

## Componente:
wd280obj.dll

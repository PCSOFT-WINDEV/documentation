
## SpreadsheetInputFunction( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un nombre de función en la posición del cursor en la celda editable de un control Hoja de cálculo que se encuentra en una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert the SUM function as formula
// an opening bracket is added and switch to assisted cell selection 
// to choose the cells onto which the sum will be applied
SpreadsheetInputFunction(PSHEET_Spreadsheet, "SUM")
```
<a name="Example2"></a>

```wl
// Insert an entire formula
// The formula in B3 corresponds to : "=SUM(ABS(-25);LOG(10))"
PSHEET_Spreadsheet = "B3"
SpreadsheetInputFunction(PSHEET_Spreadsheet, "SUM")
SpreadsheetInputFunction(PSHEET_Spreadsheet, "ABS")
SendKey("-25);")
SpreadsheetInputFunction(PSHEET_Spreadsheet, "LOG")
SendKey("10)){ENTER}")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetInputFunction(<Spreadsheet control> , <Function name>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo a utilizar, que se encuentra en una ventana. 

**`<Function name>: (Cadena de caracteres)`**

Nombre de la función que se insertará en la posición del cursor en la celda actualmente en edición. Para obtener más información, consulte [Control Hoja de cálculo: Fórmulas disponibles](../WDChamp/1000033005.md). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Una vez que se inserta el nombre de la función, se añade automáticamente un corchete de apertura y el usuario pasa a la selección asistida de las celdas en las que se aplica la función, como si el nombre de la función se escribiera en el teclado. La selección asistida de celdas se detiene cuando el usuario teclea un carácter (cierre de corchete o cualquier otro carácter).

- Si el nombre de la función se inserta al principio de la celda, se inserta un signo " = " antes del nombre de la función (para indicar que el contenido de la celda es una fórmula).

- Esta función cambia el control Hoja de cálculo para editar el contenido de la celda actual (si el control Hoja de cálculo no estaba en edición).. 

- Esta función llena la lista de funciones recientemente utilizadas, disponibles en la cinta de control.

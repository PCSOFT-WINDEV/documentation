
## Funciones para la gestión de los controles TreeView (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles TreeView (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [Colapso del árbolTodos](../WDLang1/1000023759.md) | Colapsa una rama previamente expandida (así como sus elementos hijos). |
| [&lt;Vista de árbol&gt;.add](../WDLang1/1000023422.md) | Agrega un elemento "hoja" (y todos los nodos necesarios) en un control TreeView. |
| [&lt;Vista de árbol&gt;.colapso](../WDLang1/1000023758.md) | Colapsa una rama previamente expandida. |
| [&lt;Copiar elemento](../WDLang1/1000023424.md) | Copia un elemento (hoja o nodo) en un control TreeView. |
| [TreeCount](../WDLang1/3018009.md) | Devuelve el número de elementos que se encuentran en una rama de una control TreeView. |
| [&lt;Vista de árbol&gt;.Delete](../WDLang1/1000023771.md) | Elimina un elemento y todos sus "hijos" de un control TreeView. |
| [&lt;Vista de árbol&gt;.Borrar todo](../WDLang1/1000023773.md) | Borra todos los elementos de un control TreeView. |
| [&lt;Vista de árbol&gt;.DeleteChild](../WDLang1/1000023772.md) | Borra los hijos de un elemento de un control TreeView sin borrar el elemento mismo.. |
| [&lt;Vista de árbol&gt;.display](../WDLang1/1000023421.md) | Actualiza la visualización de un control TreeView desde una posición determinada. |
| [&lt;Vista de árbol&gt;.expandir](../WDLang1/1000023756.md) | Desglosa un nodo en una control TreeView. |
| [&lt;Vista de árbol&gt;.ExpandAll](../WDLang1/1000023757.md) | Expande un nodo y todos sus elementos hijo en una control TreeView. |
| [&lt;Buscar](../WDLang1/1000023423.md) | Encuentra un elemento (hoja o nodo) en una control TreeView y devuelve su ruta. |
| [TreeGetIndex](../WDLang1/1000023418.md) | Devuelve el índice de un elemento (nodo u hoja) en un control TreeView. |
| [&lt;Vista de árbol&gt;.GiveChild](../WDLang1/1000023765.md) | Devuelve los "hijos" de un elemento (nivel inmediatamente inferior) en un control TreeView. |
| [TreeID](../WDLang1/3018027.md) | Se utiliza para construir la ruta de los elementos que se encuentran en un control TreeView diferenciando los duplicados. |
| [&lt;Identificador](../WDLang1/1000023766.md) | Devuelve el identificador associated con un elemento (nodo u hoja) en una control TreeView. |
| [&lt;Vista de árbol&gt;.InfoXY](../WDLang1/1000023760.md) | Devuelve el elemento TreeView que se encuentra en una posición determinada. |
| [&lt;Insertar](../WDLang1/1000023761.md) | Inserta una hoja en una control TreeView en una posición específica. |
| [&lt;Vista de árbol&gt;.listItem](../WDLang1/1000023762.md) | Enumera los "hijos" de un nodo y los elementos "hijo" de estos "hijos" en un control TreeView. |
| [&lt;Modificar](../WDLang1/1000023763.md) | Modifica un elemento (nodo u hoja) en un control TreeView. |
| [&lt;Vista de árbol&gt;.MoveItem](../WDLang1/1000023755.md) | Mueve un elemento (hoja o nodo) en un control TreeView. |
| [&lt;Posición](../WDLang1/1000023764.md) | Muestra un control TreeView de un elemento especificado (nodo u hoja) o devuelve el nombre del primer elemento mostrado en un control TreeView. |
| [&lt;TreeView&gt;.Select](../WDLang1/1000023767.md) | Devuelve la ruta completa del elemento seleccionado en un control TreeView. |
| [TreeSelectCount](../WDLang1/1000023769.md) | Devuelve el número de elementos seleccionados en una control TreeView (para una control TreeView multiselección). |
| [TreeSelectMinus](../WDLang1/1000023768.md) | Desmarca un elemento de un control TreeView. |
| [&lt;Vista de árbol&gt;.SelectPlus](../WDLang1/1000023770.md) | Selecciona un elemento en una control TreeView. |
| [&lt;Vista de árbol&gt;.ordenar](../WDLang1/1000023774.md) | Ordena los elementos que se encuentran en una control TreeView. |
| [&lt;Treeview&gt;.ItemStatus](../WDLang1/1000023878.md) | Devuelve el status de un nodo.: colapsado, expandido o no encontrado. |
| [&lt;Vista de árbol&gt;.TypeItem](../WDLang1/1000023775.md) | Comprueba la existencia de un elemento en una control TreeView y devuelve su tipo (nodo u hoja) si existe el elemento. |

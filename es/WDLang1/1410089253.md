## Sintaxis
<a name="SYNTAX1"></a>

`<Image control>.DrawTextMarkdown(<X-coordinate> , <Y-coordinate> , <Markdown text> [, <Text color>])`
---

**`<Image control>: (Nombre del control)`**



**`<X-coordinate>: (Entero)`**

Coordenadas (abscisa) del texto a dibujar (en pixeles).

**`<Y-coordinate>: (Entero)`**

Coordenadas (ordenada) del texto a dibujar (en pixeles).

**`<Markdown text>: (Tipo de texto a dibujar)`**

Texto Markdown a dibujar.

**`<Text color>: (Entero o constante (opcional))`**

Color a utilizar para el texto Markdown. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el color del texto:

- es Negro si no se ha utilizado la función [&lt;Imagen&gt;.Pen](../WDLang1/1000024016.md) previamente,

- corresponde al color especificado en la última llamada a la función [&lt;Imagen&gt;.Pen](../WDLang1/1000024016.md).

## Observaciones


### Fuente
<a name="fuente_ELTPARAGRAPHE000347"></a>

De forma predeterminada:

- Se utiliza la fuente "MS Sans Serif" (tamaño 13).

- La dirección de escritura es horizontal (de izquierda a derecha).

- Las coordenadas de la esquina superior izquierda de la imagen son (0,0).

- El carácter "&" no se interpreta.




Para cambiar estos parámetros, utilice la función [&lt;Variable Image&gt;.Font](../WDLang1/1000024232.md).
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La fuente predeterminada puede ser diferente dependiendo del sistema en el que se ejecuta la aplicación.
<a name="NOTE0_2"></a>


### Color de fondo
<a name="color_fondo_ELTPARAGRAPHE000372"></a>

De forma predeterminada, el color de fondo de un control Imagen es transparente. Para cambiar el color de fondo de un control Imagen, utilice la propiedad [BackgroundColor](../Proprietes/2510022.md).
<a name="NOTE0_3"></a>


### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000385"></a>

Las coordenadas se especifican con respecto a la esquina superior izquierda del control Imagen o de la imagen (coordenadas (0,0)).

Las coordenadas corresponden a la esquina superior izquierda del rectángulo en el que se dibujará el texto. La ubicación exacta del texto en el rectángulo depende del tamaño, la forma y los atributos de la fuente.

### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000468"></a>

En modo antialiasing ([&lt;Variable Image&gt;.ChangeMode](../WDLang1/1000024039.md)) o en modo opacidad ([&lt;Variable picLayer&gt;.StartDrawing](../WDLang1/1000024470.md)), la función **&lt;Image&gt;.DrawTextMarkdown** solo está disponible para las fuentes TrueType.

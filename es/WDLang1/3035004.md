


## ExeTerminate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la ejecución de una aplicación de 32 ó 64 bits en el equipo actual.: todas las instancias del ejecutable son asesinadas. La aplicación puede ser identificada por su nombre o por su PID.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png)

```wl
bRes is boolean
bRes = ExeTerminate("C:\Draw\psp\psp.exe")
```

## Sintaxis

### Terminar una aplicación identificada por su nombre

`<Result> = ExeTerminate(<Executable name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminaron todas las instancias del programa,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Esta función puede fallar:

	- si el ejecutable no está instanciado (el ejecutable no se ejecuta). 

	- si el usuario no tiene suficientes derechos para destruir una de las instancias. 







**`<Executable name>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del ejecutable (hasta 260 caracteres).


<a name="SYNTAX2"></a>

### Terminar una aplicación identificada por su ID de Process (PID)

`<Result> = ExeTerminate(<Process ID>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminaron todas las instancias del programa,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Esta función puede fallar:

	- si el ejecutable no está instanciado (el ejecutable no se ejecuta). 

	- si el usuario no tiene suficientes derechos para destruir una de las instancias. 







**`<Process ID>: (Integro)`**

Process ID (PID) de la aplicación. Este identificador es devuelto por [ExeRun](../WDLang1/3035006.md) (associated con la constante *exePID*) o por [ExeGetPID](../WDLang1/1000017239.md).

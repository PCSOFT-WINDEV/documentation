


## &lt;Tabla&gt;.CollapseAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Colapsos:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) la jerarquía completa (todos los nodos) de una control Tabla TreeView.

- todas las pausas de una control Tabla. 




![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Las pausas no están disponibles en las tablas de esta versión.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Collapses all nodes in the "TVT_Schedule" TreeView Table control
TVT_Schedule.CollapseAll()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.CollapseAll()`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este nombre puede corresponder a:

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de control Tabla TreeView para el que se colapsarán todos los nodos.

- el nombre de la control Tabla para la que se deben ocultar todas las pausas.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000218"></a>

La función **&lt;Tabla&gt;.CollapseAll** puede utilizarse en: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) un control Tabla TreeView.

- un control Tabla rellenado mediante programación. 

- un control Tabla con fuente de datos en memoria.  




Esta función no se puede utilizar en controles Tabla archivo de datos (acceso directo).

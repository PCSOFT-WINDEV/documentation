
## Funciones de Calendar (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para manipular los controles Calendar, así como los controles de edición de fecha mediante la sintaxis de prefijos:



|   |   |
| --- | --- |
| [&lt;Calendar&gt;.Posición](../WDLang1/1000017361.md) | Devuelve el mes mostrado en un control Calendario. |
| [&lt;Calendario&gt;.Select](../WDLang1/1000017385.md) | Devuelve la fecha seleccionada en un control Calendario. |
| [&lt;Calendario&gt;.SelectCount](../WDLang1/1000017362.md) | Devuelve el número de días seleccionados en una control Calendario. |
| [&lt;Calendario&gt;.SelectMinus](../WDLang1/1000017363.md) | Desmarca una fecha en una control Calendario multiselección. |
| [&lt;Calendario&gt;.SelectPlus](../WDLang1/1000017343.md) | Selecciona una fecha en una control Calendario. |
| [&lt;Entrada&gt;.CalendarioAbierto](../WDLang1/1000017423.md) | En una fecha control Campo de entrada, abre una ventana emergente que muestra una Calendar. |


## dCopyBlt (Función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>18</B></B>, esta función se conserva por motivos de compatibilidad con la mayoría de las plataformas. Se recomienda utilizar la función [dCopy](../WDLang1/1000020799.md). De hecho:

- la función [dCopy](../WDLang1/1000020799.md) gestiona la transparencia,

- los modos para realizar copias de **dCopyBlt** son reemplazados por: 

	- las funciones que se aplican al tipo [Image](../WDLang1/1000019650.md) (funciones [dEncrustColor](../WDLang1/1000019672.md), [dInvertColor](../WDLang1/1000019672.md), etc.),

	- el acceso a los pixeles.









<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia una imagen: 

- de un control Imagen a otro control Imagen.

- en una variable de tipo [Image](../WDLang1/1000019650.md).
	






![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo se puede utilizar la primera sintaxis con el tipo Image.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003029007&name=dcopyblt_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Simple copy of content from the "IMG_ImageDrawing" control
// into the "IMG_ImageCopy" Image control
ResCopy = dCopyBlt(IMG_ImageDrawing, IMG_ImageCopy)
```




<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Copy the image of the screen into an Image control
dCopyBlt(copyScreen, IMG_MyImage)
```

## Sintaxis

### Copiar imágenes de los controles Imagen

`<Result> = dCopyBlt(<Source image control> , <Destination Image control> [, <Copy mode> [, <Source image X-coordinate> [, <Source image Y-coordinate> [, <Source image height> [, <Source image width> [, <Destination image X-coordinate> [, <Destination image Y-coordinate> [, <Destination image height> [, <Destination image width>]]]]]]]]])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se copió la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**Observación**: Puede que la función **dCopyBlt** devuelva <u><u><u><u>False</u></u></u></u> si el controlador de pantalla no admite esta función. En este caso, se muestra un mensaje en la imagen copiada. Le recomendamos actualizar el controlador de pantalla para obtener un comportamiento adecuado.

**`<Source image control>: (Nombre del control)`**

Nombre del control Imagen de origen a manipular.
**Para realizar una captura de pantalla**, utilice la constante *copyScreen*: se copia todo el contenido de la pantalla. 
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) La constante *copyScreen* no está disponible.![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro puede corresponder a: 

- el nombre de un control Imagen, 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md). 




**`<Destination Image control>: (Nombre del control)`**

Nombre del control Imagen de destino en el que se copiará la imagen.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro puede corresponder a: 

- el nombre de un control Imagen, 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).

## Observaciones




### Zoom en una imagen
<a name="zoom_una_imagen_ELTPARAGRAPHE000356"></a>

Se puede ampliar o reducir una sección de la imagen de origen al copiarla a la imagen de destino. Para ello:

- Defina el área que desea ampliar (o reducir) en la imagen de origen (con los parámetros &lt;Coordenada X de imagen de origen&gt;, &lt;Coordenada Y de imagen de origen&gt;, &lt;Alto de imagen de origen&gt; y &lt;Ancho de imagen de origen&gt;),

- Defina el área de visualización de la imagen final. Esta área debe ser más grande para ampliar la imagen y más pequeña para reducirla. De forma predeterminada, se tiene en cuenta toda la imagen de destino. Si es necesario, utilice los parámetros &lt;Coordenada X de imagen de destino&gt;, &lt;Coordenada Y de imagen de destino&gt;, &lt;Alto de imagen de destino&gt; y &lt;Ancho de imagen de destino&gt;.




El porcentaje del zoom depende del coeficiente multiplicador entre las dimensiones de las dos áreas.

Por ejemplo:


|   |   |
| --- | --- |
| **1.** El área a ampliar está dentro de un rectángulo de 30 pixeles de ancho y 40 pixeles de alto. | **2.** Para aplicar un zoom del 200%, el área de destino debe ser dos veces más grande que el área de origen: rectángulo de 60 pixeles de ancho y 80 pixeles de alto. |
| <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChampSource.gif)<br> | <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ChampDestinationpsd.gif)<br> |

En este caso, la línea de código que permite realizar el zoom al copiar la imagen es la siguiente:


```wl
dCopyBlt(IMG_SourceControl, IMG_DestinationControl, copySrcCopy, 10, 10, 30, 40, 10, 10, 60, 80)
```




## ControlCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una nueva control (de cualquier tipo) en una ventana mediante programación. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
c is Control
c <- ControlCreate("BUTTON_1", typButton, 30, 40, 140, 20)
c.Caption = "Click here"
c.Process[trtClick] = myProcedure
```
<a name="Example2"></a>

```wl
ControlCreate("EDIT_1", typDate, 30, 40, 140, 20)
ctrl is Control
ctrl <- ControlCreate("DATE", typDate, 150, 150, 80, 20)
ctrl.CaptionWidth = 40
ctrl.Caption = "Date"
```
<a name="Example3"></a>

```wl
// Create a Table control with its columns
cTable is Control
cTable <- ControlCreate("cTable", typTable, 0, 0, 300, 250, True)
cTable.Anchor = anchorWidth + anchorHeight
col1 is Control <- ControlCreate("cTable.COL1", typColumn)
col1.InputType = typInputText
col2 is Control <- ControlCreate("cTable.COL2", typColumn)
col2.InputType = typInputTime
col3 is Control <- ControlCreate("cTable.COL3", typColumn)
col3.InputType = typInputNum
TableAddLine(c, "cTable", "Col 1", "1224", "25")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ControlCreate(<Name> , <Type> [, <X> [, <Y> [, <Width> [, <Height> [, <Visible>]]]]])`
---

**`<Result>: (Control Variable)`**

El nuevo control puede manejarse con una Variable de tipo [Control](../WDLang1/1000020967.md). En este caso, los valores deben asignarse con el operador **'<-'**. 

Si se utiliza directamente **ControlCreate**, el control se crea en la ventana y es Visible por Default. 

**`<Name>: (Cadena de caracteres)`**

Nombre que se le dará a la nueva control. Este nombre se utilizará para manipular el control a través de la programación. 
Se produce un error de WLanguage si este nombre ya existe. 
Si este parámetro corresponde a una cadena vacía ("""), se crea automáticamente un nombre único. 

Casos especiales: 

- Creación de una control en una control"madre": especificar el nombre completo de la control que se va a crear. Por ejemplo: "Pestaña[1].Botón2".

- Creación de una columna de edición en un control Tabla: especificar el tipo *typColumn* y el nombre del control Tabla antes del nombre de la columna. Por ejemplo: 
	
	```wl
	// Edit + Date column 
	c <- ControlCreate("TABLE1.COL3", typColumn)
	c.Width = 100
	c.InputType = typInputDate
	```


- Creación de una columna simple en una control Tabla: especifique el tipo de columna y el nombre de control Tabla antes del nombre de columna. Por ejemplo:
	
	```wl
	// Image column 
	c2 is Control <- ControlCreate("Table1.Col3", typImage)
	c2.Width = 50
	```


- Creación de paneles de pestañas : especificar el tipo de *typTabPane* y el nombre de la ficha. Por ejemplo:
	
	```wl
	ControlCreate("TAB1", typTabPane)
	```


- Creación de un texto Token control Campo de entrada: Simplemente crea un texto control Campo de entrada y habilita las fichas. Por ejemplo: 
	
	```wl
	MyControl is Control <- ControlCreate("test", typText, 10, 10, 250, 25)
	MyControl.TokenEnabled = True
	```

**`<X>: (Entero opcional)`**

Coordenada X de control a crear (posición en el eje X) en píxeles. Posición horizontal de la Corner superior izquierda del control, en relación con la Corner superior izquierda del área de cliente de la ventana (es decir, la ventana sin barra de título, barra de menú o bordes). Este parámetro corresponde a la [XInitial](../Proprietes/2510140.md) Property. 

Si no se especifica este parámetro, la coordenada X del control es set a 0. 

Este valor puede modificarse con el programa [X](../Proprietes/2510135.md) Property.

**`<Y>: (Entero opcional)`**

Coordenada Y de control a crear (posición en el eje Y) en píxeles. Posición vertical de la Corner superior izquierda del control, en relación con la Corner superior izquierda del área de cliente de la ventana (es decir, la ventana sin barra de título, barra de menú o bordes). Este parámetro corresponde a la [YInitial](../Proprietes/2510087.md) Property. 

Si no se especifica este parámetro, la coordenada Y del control es set a 0. 

Este valor puede modificarse con el programa [Y](../Proprietes/2510137.md) Property.

**`<Width>: (Entero opcional)`**

Ancho de control a crear (expresado en píxeles). Este parámetro corresponde a la [InitialWidth](../Proprietes/2510067.md) Property. 

Si no se especifica este parámetro, el ancho de control es set a 0. 

Este valor puede modificarse con el programa [Altura](../Proprietes/2510050.md) Property.

**`<Height>: (Entero opcional)`**

Altura de control a crear (expresada en píxeles). Este parámetro corresponde a la [InitialHeight](../Proprietes/2510043.md) Property. 

Si no se especifica este parámetro, la altura de control es set a 0. 

Este valor puede modificarse con el programa [anchura](../Proprietes/2510059.md) Property.

**`<Visible>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado) si el control debe ser visible,

- <u><u><u><u>False</u></u></u></u> para crear un Invisible control. 




Este valor puede modificarse con el programa [Visible](../Proprietes/2510138.md) Property.

## Observaciones


- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Si se crean varios controles en la misma ventana en una sola operación, se recomienda desactivar el [DisplayEnabled](../Proprietes/2510015.md) Property antes de crear los controles y volver a activarlo después: se optimizará la visualización de los nuevos controles y se eliminará el efecto de la apariencia progresiva. 

- Para configurar las características del nuevo control, utilice las propiedades associated con su tipo: 

	- Si el control se crea directamente, se puede utilizar el operador de indirección para manipular el control por su nombre: 
			
		```wl
		ControlCreate("BUTTON_1", typButton, 30, 40, 140, 20) 
		
		{"BUTTON_1"}..Caption = "Send an email"
		{"BUTTON_1"}..Process[trtClick] = myProcedure
		```


	- Si el control creado está asociado a Variable de tipo [Control](../WDLang1/1000020967.md), utilice las propiedades del control en el Variable: 
			
		```wl
		c is Control
		c <- ControlCreate("BUTTON_1", typButton, 30, 40, 140, 20) 
		c.Caption = "Click here"
		c.Process[trtClick] = myProcedure
		```





- Para cambiar el estilo de la control creada: 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) utilice la función [ChangeStyle](../WDLang1/1000017171.md). Basta con asignar un estilo en la hoja de estilo del proyecto a la control. 

	- utilice la propiedad [Estilo](../Proprietes/1000021472.md). Esta Property copia el estilo de una control existente a otra. 




- Para definir los diferentes procesos/eventos asociados al control, utilice el programa [Process](../Proprietes/1000020878.md) Property. 

- Para eliminar un control creado con la función **ControlCreate**, utilice la función [ControlDelete](../WDLang1/3025037.md).

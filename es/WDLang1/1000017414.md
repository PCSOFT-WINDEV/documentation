


## &lt;Caja de lista&gt;.SelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona uno o varios elementos de un Cuadro de Lista, ListView o control Combo Box.

El elemento se selecciona pero no se muestra si no es Visible. Para que el elemento seleccionado sea Visible, utilice [&lt;Caja de lista&gt;.Posición](../WDLang1/1000017335.md). 

**Recordatorio**: El elemento seleccionado corresponde al elemento actual en el que se visualiza el banda de selección.

**Observación**: La función **&lt;Caja de lista&gt;.SelectPlus** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the element at index 7 in "LIST_CustomerList"
LIST_CustomerList.SelectPlus(7)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<List Box control>.SelectPlus([<Index 1> [... [, <Index N>]]])`
---

**`<List Box control>: (Nombre del control)`**

Nombre de la caja de la lista (o Combo Box) control a utilizar.

**`<Index 1>: (Enteros opcionales)`**

Índice del primer elemento a seleccionar. 
Si no se especifica ningún índice, se seleccionan todos los elementos de la Caja de Lectura o control Combo Box especificada. 
Si estos parámetros corresponden a elementos ya seleccionados, la función **&lt;Caja de lista&gt;.SelectPlus** no tiene ningún efecto.



**`<Index N>: (Entero opcional)`**

Índice del Nº elemento a seleccionar. 
Si no se especifica ningún índice, se seleccionan todos los elementos de la Caja de Lectura o control Combo Box especificada. 
Si estos parámetros corresponden a elementos ya seleccionados, la función **&lt;Caja de lista&gt;.SelectPlus** no tiene ningún efecto.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Último elemento
<a name="ultimo_elemento_ELTPARAGRAPHE000259"></a>Para posicionarse en el último elemento de control List Box, utilice la siguiente sintaxis:

```txt
<List Box control>.SelectPlus(<List Box control>.Count)
```
o


```txt
<List Box control> = <List Box control>.Count
```

<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000284"></a>

Para una selección única control List Box sin llamar a [gStoredValue](../WDLang1/1410088363.md), las dos sintaxis siguientes son equivalentes:

```txt
<List Box control>.SelectPlus(<Index>)
```


y


```txt
<List Box control> = <Index>
```

<a name="NOTE0_3"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000307"></a>

- Para seleccionar un elemento en una control Tabla, utilice [&lt;Tabla&gt;.SelectPlus](../WDLang1/1000024287.md).

- **&lt;Caja de lista&gt;.SelectPlus** puede utilizarse en una columna "Combo Box" tabla.

- [&lt;SelectMinus](../WDLang1/1000017393.md) permite anular la selección de los elementos de un cuadro de lista o control Combo Box.

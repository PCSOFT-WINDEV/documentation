


## <Diagram Variable>.ToImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta un Diagram a un Variable de tipo [Image](../WDLang1/1000019650.md). A continuación, la Image generada puede guardarse o visualizarse en otra control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save a diagram to a PNG file
DiagImage is Image
DiagImage = DIAGEDT_MyDiagram.ToImage()
IF ErrorOccurred THEN
	Error()
	RETURN
END
DiagImage.SavePNG("DiagramExport.png")
```

```wl
MyDiagram is Diagram <- DIAGEDT_MyDiagram

// Retrieves the generated image
MyImage is Image 
MyImage <- MyDiagram.ToImage()

// Open the image
OpenDocument(MyImage)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Diagram>.ToImage([<Background color>])`
---

**`<Result>: (Imagen Variable)`**

Variable de tipo [Image](../WDLang1/1000019650.md) correspondiente a la imagen del Diagram.

**`<Diagram>: (Nombre del control o variable de tipo Diagram)`**

[Diagram](../WDLang1/1410088055.md) Variable para manipular

**`<Background color>: (Entero opcional)`**

Imagen de fondo Color. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md),

- a Variable de tipo [Color](../WDLang1/1000019661.md). 


Si no se especifica el fondo color, el Diagram se exportará tal cual, con el fondo color ya definido (si existe).

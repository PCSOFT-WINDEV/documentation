


## Funciones de gestión de controles Zona multilínea
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar los controles de la zona multilínea:



|   |   |
| --- | --- |
| [mlzAddLine](../WDLang1/1000020199.md) | Añade una nueva dinámica fila a un control Zona multilínea. |
| [mlzDelete](../WDLang1/1000020208.md) | Borra un fila dinámico de un control Zona multilínea. |
| [mlzDeleteAll](../WDLang1/1000020209.md) | Elimina todas las filas dinámicas de una Zona Mutilina control. |
| [mlzInsertLine](../WDLang1/1000020207.md) | Inserta una nueva dinámica fila en un control Zona multilínea. |

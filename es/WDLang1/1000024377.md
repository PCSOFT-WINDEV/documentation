
## AAFImportParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga todas las personalizaciones realizadas a través de AAF (automatic Applications Features) que han sido previamente guardadas en un archivo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on "Import my parameters"
FileName is string 
FileName = fSelect("", "", "Select a file...", "parameters" + ...
	TAB + "*.param" + CR + "All files (*.*)" + ...
	TAB + "*.*", "*.*", fselExist)
IF FileName = "" THEN RETURN
AAFImportParameter(FileName)
IF ErrorOccurred THEN
	Error()
ELSE
	ToastDisplay("Parameters retrieved successfully.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`AAFImportParameter(<File to import>)`
---

**`<File to import>: (Cadena de caracteres)`**

Nombre y ruta completo del archivo en el que se han guardado previamente las personalizaciones de la AAF.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Personalizaciones guardadas
<a name="personalizaciones_guardadas_ELTPARAGRAPHE000032"></a>

Todas las personalizaciones hechas por el usuario se guardan: 

- Tamaños y posiciones de las ventanas,

- Almacena los valores de control,

- Historial de entradas, 

- Notas personalizadas, 

- Personalización de los controles del tablero,

- Personalización del panel acoplable, ...




Estos parámetros pueden restablecerse con **AAFImportParameter** o a través de la función de importación de parámetros de la aplicación automatic (disponible en el menú contextual de una ventana de la aplicación).




## <Certificate Variable>.ExtractExtensionValue (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor de una Certificate extension.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Extract the value of the "1.3.6.1.4.1.311.10.3.4" certificate extension
cert is Certificate = CertificateSelect()

bufValue is Buffer = cert.ExtractExtensionValue("1.3.6.1.4.1.311.10.3.4")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Certificate>.ExtractExtensionValue(<OID>)`
---

**`<Result>: (Buffer)`**

Valor de extension. Si la extensión no existe, el buffer devuelto está vacío.

**`<Certificate>: (Variable de tipo Certificate)`**

Nombre de la variable [Certificate](../WDLang1/1000019287.md) que corresponde al Certificate en el que se debe buscar el extension.

**`<OID>: (Cadena de caracteres)`**

extension OID.




## PDFReaderInfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Para una posición dada (coordenadas de un punto) en un PDF Reader control, devuelve una de las siguientes opciones: 

- el número Page de la posición especificada,

- la posición X (en mm) para la posición especificada,

- la posición Y (en mm) para la posición especificada.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Page number
nPage is int
nPage = PDFReaderInfoXY(PDF_NoName1, pdfPageNumber, MouseXPos(), MouseYPos())

// X-position in millimeters
rXPos is real
rXPos = PDFReaderInfoXY(PDF_NoName1, pdfXCoord, MouseXPos(), MouseYPos())

// Y-position in millimeters
rYPos is real
rYPos = PDFReaderInfoXY(PDF_NoName1, pdfYCoord, MouseXPos(), MouseYPos())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFReaderInfoXY(<PDF Reader control> , <Type of information> , <X> , <Y>)`
---

**`<Result>: (Real)`**

Información solicitada. 

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular. 

Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *pdfOriginScreen* | Por Default, (0,0) corresponde a fila 1, columna 1 de la control Procesador de texto (barras de desplazamiento ubicadas en el origen).<br>Si se combina la constante *pdfOriginScreen* con las constantes anteriores, (0,0) corresponde al origen de la pantalla.<br>Esta constante no puede utilizarse por sí sola. |
| *pdfPageNumber* | Número de la Page que contiene la posición especificada. |
| *pdfXCoord* | Coordenada X de la posición especificada en la Page (expresada en milímetros). |
| *pdfYCoord* | Coordenada Y de la posición especificada en la Page (expresada en milímetros). |



**`<X>: (Integro)`**

Coordenada X (en pixeles) a analizar. Esta coordenada es relativa al control (si no se especifica la constante *pdfOriginScreen* ).

**`<Y>: (Integro)`**

Coordenada Y (en pixeles) a analizar. Esta coordenada es relativa al control (si no se especifica la constante *pdfOriginScreen* ).

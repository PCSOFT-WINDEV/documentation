
## EmulatePaneUndock (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula el desacoplamiento de un panel en una control Pestaña dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Undocks the "Alias_1" pane from the TAB_Tab control
EmulatePaneUndock(TAB_Tab, "Alias_1", 105, 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulatePaneUndock(<Control name> , <Alias> , <X> , <Y>)`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de la control Pestaña dinámica de la que se debe desacoplar el panel. 

**`<Alias>: (Cadena de caracteres)`**

Alias del panel para desacoplar de la dinámica associated control Pestaña. 

**`<X>: (Integro)`**

Coordenada X en la que debe situarse el panel. 

**`<Y>: (Cadena de caracteres)`**

Coordenada Y sobre la que se debe posicionar el cristal.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función solo se debe utilizar en el código de las pruebas automatizadas.




## fFileExist( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la existencia de un fichero externo.

**Observación**: En Windows de 64 bits, puede acceder a un directorio del sistema desde un ejecutable de 32 bits en un directorio diferente. Para obtener más información, consulte [Nativo de 64 bits y nativo de 32 bits](../Editeurs/2030048.md).

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
AFile is string
AFile = "C:\Temp\Example.txt"
IF fFileExist(AFile) = True THEN 
	Info("The file named " + AFile + " exists on the disk")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fFileExist(<File name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si existe el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario (por ejemplo, si el nombre del archivo corresponde al nombre de un directorio).




**`<File name>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo. Los caracteres comodín (\*,?) no están permitidos. Puede llamar a [fDir](../WDLang1/3036056.md) para utilizar caracteres comodín. Si sólo se especifica el nombre del archivo (sin ruta), el archivo se busca en el directorio actual (por ejemplo: fFileExist("Test.jpg"))
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.

## Observaciones


### Comprobación de la existencia de un directorio
<a name="comprobacion_existencia_directorio_ELTPARAGRAPHE000098"></a>

Para comprobar la existencia de un directorio, utilice [fDirectoryExist](../WDLang1/3036059.md).

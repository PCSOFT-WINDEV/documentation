
## fTempPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del directorio utilizado para almacenar los archivos temporales.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036044&name=ftemppath_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Retrieve the name of temporary directory
ResTempDir = fTempPath()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fTempPath()`
---

**`<Result>: (Cadena de caracteres)`**



- Ruta del directorio utilizado para almacenar los archivos temporales (hasta 260 caracteres). Esta ruta termina con el carácter "\\".. 
	

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.   


**Observación**: El resultado de la función **fTempPath** siempre es una cadena de caracteres en formato Ansi. Sin embargo, el resultado puede almacenarse en una cadena de caracteres Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Directorio temporal
<a name="directorio_temporal_ELTPARAGRAPHE000070"></a>

En Windows Vista (y posteriores), cuando se habilita la [Sistema UAC](../Editeurs/2025014.md), el directorio temporal es diferente dependiendo de si el usuario tiene privilegios de administrador o no.


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000080"></a>

La función **fTempPath** es equivalente a la función [fTempDir](../WDLang1/1000021768.md).

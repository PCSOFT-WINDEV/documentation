
## Funciones de notificaciones interactivas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten manipular las notificaciones interactivas: 



|   |   |
| --- | --- |
| [SysNotificationClear](../WDLang1/1410087937.md) | Elimina todas las notificaciones interactivas enviadas previamente por la aplicación actual. |
| [SysNotificationCompatible](../WDLang1/1410087932.md) | Comprueba si el sistema operativo soporta notificaciones interactivas. |
| [SysNotificationRemove](../WDLang1/1410087907.md) | Elimina un Notification interactivo enviado anteriormente. |
| [SysNotificationRemoveID](../WDLang1/1410087939.md) | Elimina todas las notificaciones interactivas enviadas anteriormente que coincidan con el identificador (ID) especificado. |
| [SysNotificationSend](../WDLang1/1410087906.md) | Envía una Notification interactiva al sistema para que sea visualizada. |





Las funciones de notificaciones interactivas utilizan los siguientes tipos de variables: 



|   |   |
| --- | --- |
| [sysNotification](../WDLang1/1410087893.md) | El tipo **sysNotification** se utiliza para definir todas las características avanzadas de un sistema interactivo Notification. |
| [sysNotificationButton](../WDLang1/1410087947.md) | El tipo **sysNotificationButton** se utiliza para definir todas las características avanzadas de control Botón en un Notification interactivo. |
| [sysNotificationInput](../WDLang1/1410088024.md) | El tipo **sysNotificationInput** se utiliza para definir todas las características avanzadas de un control de entrada, como una entrada de texto o un control de cuadro combinado, en un Notification interactivo. |

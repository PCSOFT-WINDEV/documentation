
## NoRightCharacter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve una cadena después de eliminar los caracteres especificados a la derecha de la cadena inicial.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sString is string = "<<I am a character string>>"
sResult is string = NoRightCharacter(sString, ">")
// sResult contains "<<I am a character string"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NoRightCharacter(<Initial string> , <Characters to remove>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres sin los caracteres especificados a la derecha de la cadena inicial. 

**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres inicial. Esta cadena no se modifica. 

**`<Characters to remove>: (Cadena de caracteres)`**

Lista de caracteres que deben eliminarse a la derecha de la cadena inicial. 

Si este parámetro corresponde a una cadena vacía (""), &lt;Resultado&gt; será igual a la cadena inicial. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Función NoRightCharacter y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000056"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **NoRightCharacter** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

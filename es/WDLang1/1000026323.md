


## Funciones para la gestión de los controles de visualización de HTML (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de visualización de HTML (sintaxis del prefijo): 




|   |   |
| --- | --- |
| [&lt;HTML Display control&gt;.ExecuteBrowserProcedure](../WDLang1/1000026265.md) | Ejecuta un navegador WLanguage Procedure en un HTML Display control que muestra una página creada con WEBDEV. |
| [&lt;HTML Display control&gt;.FromEmail](../WDLang1/1410087278.md) | Importa el contenido HTML de una Email y sus imágenes en una control de visualización HTML. Las imágenes contenidas en la Email se muestran también en la control. |
| [&lt;HTML Display control&gt;.OnErrorJS](../WDLang1/1000026267.md) | Especifica la Procedure a la que hay que llamar si se produce un error al ejecutar el código JavaScript en una control de visualización de HTML. |
| [&lt;HTML Display control&gt;.ValidateForm](../WDLang1/1000026311.md) | Valida un formulario en una página HTML mostrada en una pantalla HTML control. |
| [&lt;HTML Display control&gt;.Valor control](../WDLang1/1000026268.md) | Se utiliza para identificar o modificar una control en un formulario HTML contenido en una control de visualización HTML . |
| [&lt;Navegar](../WDLang1/1000023633.md) | Inicia la navegación en una pantalla HTML control. |
| [.Executecontrol Process](../WDLang1/1000026266.md) | Ejecuta un Event associated con un control en una página HTML mostrada por un control. |

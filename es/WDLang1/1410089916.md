


## Funciones de cadenas (sintaxis con prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten manipular cadenas, utilizando la sintaxis con prefijo:




|   |   |
| --- | --- |
| [&lt;Tipo ANSI string&gt;.AnsiToOem](../WDLang1/1000024891.md) | Convierte una cadena ANSI (Windows) en una cadena OEM (DOS). |
| [&lt;Tipo ANSI String&gt;.Contains](../WDLang1/1410089824.md) | Permite comprobar si una cadena contiene una subcadena. |
| [&lt;Tipo Ansi string&gt;.DeleteDuplicate](../WDLang1/1410087983.md) | Elimina los duplicados entre las subcadenas de una cadena, basándose en uno o varios separadores. |
| [&lt;Tipo ANSI string&gt;.ToNum](../WDLang1/1410087984.md) | Devuelve el valor numérico de una cadena de caracteres. Esta función permite realizar cálculos numéricos en las cadenas de caracteres. |
| [&lt;Tipo Buffer&gt;.AnsiToUnicode](../WDLang1/1000026033.md) | Convierte un buffer que contiene una cadena ANSI (Windows) en un buffer que contiene una cadena UNICODE. |
| [&lt;Tipo String&gt;.AnsiToUnicode](../WDLang1/1000024892.md) | Convierte una cadena ANSI (Windows) en una cadena UNICODE. |
| [&lt;Tipo String&gt;.BufferToHexa](../WDLang1/1000024894.md) | Convierte una cadena Ansi o Unicode en una cadena hexadecimal (por ejemplo: "4A 5B 00"). |
| [&lt;Tipo String&gt;.CharactTypeCount](../WDLang1/1000024895.md) | Devuelve el número de caracteres que coinciden con un tipo determinado. |
| [&lt;Tipo String&gt;.EndsWith](../WDLang1/1000024903.md) | Verifica si una cadena de caracteres termina: <br><br>- por una cadena de caracteres específica.<br><br>- por una de las cadenas de caracteres de un array.<br><br><br> |
| [&lt;Tipo String&gt;.ExtractString](../WDLang1/1000024876.md) | Permite:<br><br>- extraer una subcadena de una cadena en función de un separador de cadenas especificado.<br><br>- buscar subcadenas en una cadena en función de un separador de cadenas especificado.<br><br><br> |
| [&lt;Tipo String&gt;.Format](../WDLang1/1000024877.md) | Define el formato de una cadena de caracteres según las opciones seleccionadas. |
| [&lt;Tipo String&gt;.Increment](../WDLang1/1000024905.md) | Incrementa un número al inicio o al final de una cadena. |
| [&lt;Tipo String&gt;.Insert](../WDLang1/1000024906.md) | Inserta una cadena en otra cadena de caracteres en una posición determinada. |
| [&lt;Tipo String&gt;.Replace](../WDLang1/1000024890.md) | Reemplaza todas las instancias especificadas de una subcadena en una cadena por otra subcadena. |
| [&lt;Tipo String&gt;.Reverse](../WDLang1/1000024908.md) | Invierte los caracteres de una cadena. |
| [&lt;Tipo String&gt;.Split](../WDLang1/1000024902.md) | Extrae las subcadenas de una cadena de caracteres en función de uno o más separadores. |
| [&lt;Tipo String&gt;.StartsWith](../WDLang1/1000024901.md) | Verifica si una cadena de caracteres comienza: <br><br>- por una cadena de caracteres específica. <br><br>- por una de las cadenas de caracteres de un array.<br><br><br> |
| [&lt;Variable JSON&gt;.ToString](../WDLang1/1410089810.md) | Convierte el contenido de una variable de tipo JSON a una cadena JSON. |
| [.completa](../WDLang1/1410089823.md) | Devuelve una cadena de caracteres de una longitud especificada. |
| [.derecha](../WDLang1/1410089825.md) | Extrae la parte derecha (es decir, los últimos caracteres) de una cadena o buffer. |

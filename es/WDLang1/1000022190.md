


## &lt;Tabla dinámica&gt;.Select (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer la posición de la celda seleccionada y seleccionar una celda en un control Tabla dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Get the current position
pvtPos is pvtPosition
pvtPos = PVT_Stats.Select()
IF NOT pvtPos.Out THEN
	TRACE(pvtPos.Country + " " + pvtPos.City)
END
```

```wl
// Loop through the selected cells
pvtPos is pvtPosition of PVT_MyPVT

nbSel is int = PVT_MyPVT.SelectOccurrence()
FOR i = 1 TO nbSel
	pvtPos = PVT_MyPVT.Select(i)
	Trace(pvtPos.COL_Today_Date + ":" + pvtPos.COL_Product))
END
```

```wl
pvtPos is pvtPosition
// Select "France / 2014" 
pvtPos.Country = "FRANCE"
pvtPos.Year = "2014"
PVT_Stats.Select(pvtPos)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Recuperar la posición de la célula seleccionada en un control Tabla dinámica

`<Result> = <Pivot Table control>.Select([<Rank>])`
---

**`<Result>: (pvtPosition Variable)`**

[pvtPosition](../WDLang1/1000020726.md) Variable que contiene la posición de la celda seleccionada en el control. Si no se selecciona ningún elemento, el **Out** Property del [pvtPosition](../WDLang1/1000020726.md) Variable devuelto se establece en <u><u><u><u>True</u></u></u></u>.

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Rank>: (Entero opcional)`**

Rango de selección para una multiselección control. Si no se especifica este parámetro, se devuelve la posición de la primera celda seleccionada.


<a name="SYNTAX2"></a>

### Seleccionar un puesto en un control Tabla dinámica

`<Pivot Table control>.Select(<Position> [, <Value>])`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Position>: (pvtPosition Variable)`**

Nombre de la [pvtPosition](../WDLang1/1000020726.md) Variable correspondiente a la posición a seleccionar. Se produce un error de WLanguage si la posición no corresponde a una célula de Visible.

**`<Value>: (Cadena de caracteres)`**

Nombre del valor a seleccionar. Si no se especifica este parámetro y se encuentran varios valores en la misma celda, se selecciona el primer valor.




## Instance( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el "sistema manipular" (HINSTANCE) del programa actual.. Entonces, esta manipular puede ser utilizada en las APIs del sistema que requieren este parámetro.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the instance of the program
hInstance is int = Instance()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Instance()`
---

**`<Result>: (Integro)`**

Sistema manipular del programa.

## Componente:

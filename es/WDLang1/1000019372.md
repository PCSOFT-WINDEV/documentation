
## Posición del programador (función)






	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>17</B></B>, esta función se mantiene para la compatibilidad en WINDEV. En WINDEV, le recomendamos que utilice [SchedulerPositionDateTime](../WDLang1/1000019998.md). 



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que se muestra en una control Planificador. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions the schedule on today's date
SchedulerPosition(SCH_Schedule, Today())

// Positions the schedule on December 20, 2016
SchedulerPosition(SCH_Schedule, "20161220")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Posicionar el programa en una fecha determinada

`SchedulerPosition(<Scheduler control> , <Start date>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Start date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha de inicio de la visualización.


<a name="SYNTAX2"></a>

### Averiguar la primera o la última fecha mostrada por la control Planificador

`<Result> = SchedulerPosition(<Scheduler control> [, <Sought date>])`
---

**`<Result>: (Cadena de caracteres)`**

Fecha visualizada por el control Planificador.

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Sought date>: (Constante opcional de tipo Integer)`**

Tipo de fecha solicitada: 


|   |   |
| --- | --- |
| *schCurrentMonth* | Primer día del mes mostrado en el control Planificador. |
| *schEndDate* | Última fecha mostrada en el control Planificador. |
| *schStartDate* <br>(Valor predeterminado) | Primera fecha mostrada en el control Planificador. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Posicionamiento en la control Planificador
<a name="posicionamiento_control_planificador_ELTPARAGRAPHE000078"></a>

Al utilizar la función **Posición del programador** para mostrar una fecha, el planificador conserva los ajustes de visualización. 

Por ejemplo, si el planificador se muestra por semana a partir del Lunes, la función **Posición del programador** utilizada en la fecha del viernes muestra la planificación del Lunes al Domingo de la semana que contiene el viernes especificado.

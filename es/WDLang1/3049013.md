


## ListDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un cuadro de lista, ListView o control Combo Box rellenado mediante programación.

**Observación**: **ListDeleteAll** puede utilizarse en los controles de cuadro de lista de selección única y de selección múltiple.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all elements from the "LIST_CustomerList" control
ListDeleteAll(LIST_CustomerList)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ListDeleteAll(<List Box control>)`
---

**`<List Box control>: (Nombre del control)`**

Nombre de la casilla de la lista (o Combo Box) control a utilizar. Si este parámetro corresponde a una cadena vacía (""), se borran todos los elementos de la control List Box (o Combo Box) a la que pertenece la Event actual.





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000132"></a>

- Para eliminar todas las filas de una control Tabla, utilice [TableDeleteAll](../WDLang1/3074034.md).

- **ListDeleteAll** puede utilizarse en una columna "Combo Box" tabla.

- Para los controles editables de Combo Box, el área de entrada está despejada.

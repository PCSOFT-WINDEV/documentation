
## InvalidInputShowIcon (Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>26</B>, Se recomienda utilizar la nueva funcionalidad para comprobar la entrada obligatoria o no válida. Para obtener más información, consulte [Entrada obligatoria o no válida](../WDChamp/9500154.md).
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un ícono de error al lado de un control Campo de entrada para indicar que la entrada no es válida.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Exit from EDT_PinCode
IF Length(EDT_PinCode)<4 THEN
	InvalidInputShowIcon("The PIN code must contain at least 4 characters.", 
		icoLeft + icoTop)
	SetFocusAndReturnToUserInput(EDT_PinCode) 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`InvalidInputShowIcon(<Tooltip text> [, <Position> [, <Icon>]])`
---

**`<Tooltip text>: (Cadena de caracteres)`**

Contenido del tooltip que se muestra al pasar el cursor sobre el ícono o al hacer clic en él. 

**`<Position>: (Entero opcional)`**

Posición del ícono que se muestra: 


|   |   |
| --- | --- |
| *icoBottom* | El ícono se muestra debajo del control. |
| *icoLeft* | El ícono se muestra a la izquierda del control. |
| *icoRight*<br>(Valor predeterminado) | El ícono se muestra a la derecha del control. |
| *icoTop* | El ícono se muestra encima del control. |



**`<Icon>: (Cadena de caracteres opcional)`**

Nombre completo de la imagen a mostrar. Este parámetro puede corresponder a cualquier tipo de imagen. Esta se mostrará como un ícono de 15 x 15. 
Si no se especifica este parámetro, se mostrará el ícono predeterminado ("!" en un círculo rojo). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se debe llamar a la función **InvalidInputShowIcon** desde un control Campo de entrada, de lo contrario no tendrá ningún efecto.

- Si se modifica el valor del control, se borrará el ícono

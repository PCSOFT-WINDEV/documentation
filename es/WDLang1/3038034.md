
## WinInput( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la ventana que contiene el código WLanguage que se está ejecutando actualmente.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of the window whose code is currently run
ResWinCurrentlyRun = WinInput()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinInput()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre (o alias) de la ventana si el código que se está ejecutando actualmente lo es:

	- un código de ventana,

	- un código control,

	- un local de Procedure a la ventana.




- Vaciar la cadena (""") si el código que se está ejecutando actualmente es:

	- un código associated con el proyecto,

	- el Procedure global,

	- un código de inicialización o un código de finalización de una clase,

	- Un método de objeto.  









<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Ventana en ejecución
<a name="ventana_ejecucion_ELTPARAGRAPHE000085"></a>

- **WinInput** devuelve un valor invariable independientemente de cómo se llame al código que contiene esta función.

- La función **WinInput** no depende del modo de apertura de la ventana ni del tipo de ventana (modal, no modal, MDI padre o MDI hija).


## MemCreate( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20</B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [Associative arrays](../Motscles/1514058.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una zona de memoria. Esta zona de memoria puede contener texto o información numérica.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a memory area
MemCreate("MyMemoryZone")
 
// When typing the following MemXXX functions, 
// "MyMemoryZone" will be automatically proposed in the completion
```


<a name="Example2"></a>



```wl
sString is string = "A"
sString = Complete(sString, 10000, "B")
Trace(Length(sString))

MemCreate("MZ")
MemAdd("MZ", "1", sString)
sString = MemFirst("MZ")
Trace(Length(sString))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MemCreate(<Memory area>)`
---

**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria a crear (hasta 20 caracteres). Este nombre será utilizado por todas las funciones de gestión de las zonas de memoria.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de zona de memoria
<a name="existencia_zona_memoria_ELTPARAGRAPHE000079"></a>

No se comprueba la existencia de la zona de memoria. Si la zona de memoria ya existía, esta zona de memoria se sobrescribe. Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Finalización de automatic en el nombre del área de memoria
<a name="finalizacion_automatic_nombre_del_area_memoria_ELTPARAGRAPHE000090"></a>

En el editor de código, después de manipular un área de memoria mediante una función Memxxx (**MemCreate**, por ejemplo), su nombre se propondrá automáticamente cuando se escriban funciones de WLanguage que acepten el nombre del área de memoria como parámetro.
<a name="NOTE0_3"></a>




### Tamaño de la información
<a name="tamano_informacion_ELTPARAGRAPHE000100"></a>

Si el tamaño de la información a almacenar en la zona de memoria es mayor que el espacio de memoria disponible, se utiliza un "disco de intercambio": se crean uno o más ficheros temporales para almacenar la información adicional. Estos archivos temporales se crean en el disco local, en el directorio especificado por el entorno "TEMP" Variable.




## &lt;Hoja de cálculo&gt;.EndModification ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Señala el fin de la agrupación de un conjunto de operaciones en un control Hoja de cálculo en un solo Event (gestión de "deshacer/rehacer").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The selection is displayed in bold and italic. A single Ctrl + Z cancels both operations
PSHEET_Spreadsheet.StartModification()
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)
PSHEET_Spreadsheet.StyleSelection(psheetItalic, True)
PSHEET_Spreadsheet.EndModification()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Spreadsheet control>.EndModification()`
---

**`<Spreadsheet control>: (Control name)`**

Nombre de control Hoja de cálculo a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Las modificaciones realizadas en la control Hoja de cálculo se agrupan desde la llamada a [&lt;Hoja de cálculo&gt;.StartModificación](../WDLang1/1000023721.md). Si se realizan varias modificaciones Cancel B en el control Hoja de cálculo, simplemente use Ctrl + Z para Cancel todas las operaciones realizadas.

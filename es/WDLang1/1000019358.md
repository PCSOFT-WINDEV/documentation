


## ToastDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje "Toast".

Los Toast son mensajes cortos que aparecen en la pantalla durante unos segundos antes de desvanecerse sin ninguna intervención del usuario. Un mensaje Toast no obtiene el foco, no es modal y no interrumpe la ejecución de la aplicación.

Se recomienda utilizar los mensajes Toast para transmitir información al usuario sin bloquear la ejecución de la aplicación (por ejemplo: mostrar un mensaje para indicar que una descarga ha finalizado).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=ToastAffiche.gif)
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta función solo está disponible para sitios WEBDEV en Linux.

## Ejemplo


```wl
ToastDisplay("This is a Toast message.")
```
<a name="Example2"></a>

```wl
ToastDisplay("This is a Toast message.", toastLong, vaMiddle, haCenter, LightGreen)
```
<a name="Example3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Define the font used (gxxx functions)
ToastDisplay(gFontBold() + gFont("Batang") + gFontSize(18) + ...
	gPen(LightYellow) + "My custom message", ...
	toastShort, vaBottom, haRight, DarkRed)
```
<a name="Example4"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
MyToast is Toast
MyToast.Title = "Did you know?"
MyToast.Text = "AAFs are really cool."
MyToast.Button[1].Caption = "Learn more..."
MyToast.Button[1].ActionClick = FuncHelp
MyToast.ActionClose = clickClose
ToastDisplay(MyToast)
INTERNAL PROCEDURE FuncHelp
	Open ("WIN_Help")
END
INTERNAL PROCEDURE clickClose(nButton)
	Trace("Click to close. Button: " + nButton)
END
```

## Sintaxis
<a name="SYNTAX1"></a>

### Definir y mostrar un toast

`ToastDisplay(<Message> [, <Display duration> [, <Vertical alignment> [, <Horizontal alignment> [, <Background color>]]]])`
---

**`<Message>: (Cadena de caracteres)`**

Mensaje a mostrar. Este mensaje puede ser multilínea

**`<Display duration>: (Entero opcional)`**

Constante que indica la duración de visualización del mensaje Toast: 


|   |   |
| --- | --- |
| *toastLong* | El mensaje Toast se mostrará durante un largo período de tiempo. |
| *toastShort*<br>(Valor predeterminado) | El mensaje Toast se mostrará durante un corto período de tiempo. |


**Observación**: La duración exacta de visualización depende del dispositivo utilizado.

**`<Vertical alignment>: (Entero opcional)`**

Constante que indica la alineación vertical del mensaje. Esta alineación es relativa: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) a la pantalla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) a la ventana.





|   |   |
| --- | --- |
| *vaBottom*<br>(Valor predeterminado) | El mensaje Toast se muestra abajo. |
| *vaMiddle* | El mensaje Toast se muestra en el centro. |
| *vaTop* | El mensaje Toast se muestra arriba. |




**`<Horizontal alignment>: (Entero opcional)`**

Constante que indica la alineación horizontal del mensaje. Esta alineación es relativa: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) a la pantalla.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) a la ventana.





|   |   |
| --- | --- |
| *haCenter*<br>(Valor predeterminado) | El mensaje Toast se muestra en el centro.<br><br> |
| *haLeft* | El mensaje Toast se muestra a la izquierda.<br><br> |
| *haRight* | El mensaje Toast se muestra a la derecha.<br><br> |




**`<Background color>: (Entero opcional)`**

Color de fondo utilizado para mostrar el mensaje. Este parámetro puede ser: 

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no está disponible.

### Mostrar un toast basado en una variable de tipo Toast

`ToastDisplay(<Toast>)`
---

**`<Toast>: (Variable de tipo Toast)`**

Nombre de la variable de tipo [Toast](../WDLang1/1000024638.md) que define el toast a mostrar. 

Observación: La variable de tipo [Toast](../WDLang1/1000024638.md) permite definir parámetros adicionales.

## Observaciones


### Gestión de subprocesos
<a name="gestion_subprocesos_ELTPARAGRAPHE000233"></a>

Se puede llamar a la función **ToastDisplay** desde un subproceso WLanguage (por ejemplo, para notificar al usuario sobre los eventos que se produjeron al ejecutar del subproceso).


### Llamadas sucesivas
<a name="llamadas_sucesivas_ELTPARAGRAPHE000242"></a>

Si se llama varias veces a la función **ToastDisplay**, cada toast se muestra una vez que el toast anterior ha desaparecido. Este comportamiento puede cambiarse utilizando la propiedad **Overlayable** de la variable de tipo [Toast](../WDLang1/1000024638.md). Esta propiedad le permite obtener el mismo comportamiento que en la versión 24:  los toast se superponen unos a otros.

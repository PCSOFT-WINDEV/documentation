
## ArcSin( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el seno del arco de un valor numérico (entero o real).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the arc sine of 0.5
ArcSinRes = ArcSin(0.5)
// Returns 30
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArcSin(<Numeric value>)`
---

**`<Result>: (Real)`**

Arco del seno (ángulo del seno en grados) del valor numérico especificado, que se encuentra en el [-90, 90] interval.

**`<Numeric value>: (Entero o real)`**

Valor numérico (seno) a utilizar. Este valor debe encontrarse en el [-1,1] interval. De lo contrario, se produce un error WLanguage.

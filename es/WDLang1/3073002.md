
## SysSpace( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el espacio de memoria física del ordenador actual.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the physical memory space
ResMemorySpace = SysSpace()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysSpace([<Type of memory>])`
---

**`<Result>: (Integro)`**

Espacio deseado (en bytes) en el ordenador actual (o servidor).



**Observación**: **SysSpace** devuelve un entero de 8 bytes si el tamaño deseado es superior a 2 GB.

**`<Type of memory>: (Constante opcional de tipo Integer)`**

Tipo de memoria a devolver:


|   |   |
| --- | --- |
| *sysFreeProgramSpace* | Espacio de memoria física no utilizado.<br><br> |
| *sysFreeStorageSpace* | Espacio de almacenamiento disponible<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta constante no está disponible. |
| *sysProgramSpace*<br>(Valor predeterminado) | Espacio físico de memoria.<br><br> |
| *sysStorageSpace* | Espacio utilizado para el almacenamiento<br><br><br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta constante no está disponible. |

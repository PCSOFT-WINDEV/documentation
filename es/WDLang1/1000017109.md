


## ServicePause( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pone en pausa un servicio.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Pauses the service found in TABLE_Service
IF ServicePause(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServicePause(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se solicitó poner en pausa el servicio,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a pausar.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para pausar un servicio en otro ordenador.. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el servicio se gestiona localmente..

### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000054"></a>

La función **ServicePause** envía una solicitud para poner en pausa el servicio al administrador de servicios. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).
<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000066"></a>

- Algunos servicios no gestionan el estado de pausa.




## &lt;Variable Diagram&gt;.EndModification (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el final de una serie de acciones realizadas sobre una variable de tipo [Diagram](../WDLang1/1410088055.md) en un único Event("deshacer/rehacer").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo




```wl
MyDiagram is Diagram <- DIAGEDT_MyDiagram
MySelection is diagSelection <- DIAGEDT_MyDiagram.Selection
MyDiagram.StartModification()
MySelection.Background.Color = White
MySelection.Border.Color = Black
MyDiagram.EndModification()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Diagram>.EndModification()`
---

**`<Diagram>: (Diagram Variable)`**

Nombre de la variable de tipo [Diagram](../WDLang1/1410088055.md) que se utilizará.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todos los cambios realizados entre la llamada a [.StartModification](../WDLang1/1410088877.md) y **&lt;Variable Diagram&gt;.EndModification** se agrupan fácilmente en una sola acción que se puede deshacer fácilmente.

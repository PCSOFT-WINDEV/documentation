


## docParagraph (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docParagraph** se utiliza para manipular las características de un párrafo en un DOCX Document. Las características de este apartado pueden definirse y modificarse mediante diferentes funciones y propiedades de WLanguage. 

Este tipo de Variable es utilizado por [Document](../WDLang1/1000022461.md) Variable s. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WP_MyDoc = DocOpen(fExeDir() + fSep() + "MyDoc.docx") 

TextFragment is docFragment(WP_MyDoc, 1, Length(DocToText(WP_MyDoc)))

FOR EACH MyCurrentElement OF TextFragment.Element
	MyParagraph is docParagraph = MyCurrentElement.Paragraph
	Trace(MyParagraph.StartPosition)
	Trace(MyParagraph.EndPosition)
END
```




## &lt;Organizador&gt;.DisplayPeríodo anterior (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desplaza el área de tiempo mostrada al área anterior (semanas anteriores, por ejemplo) en un control Agenda. El control Agenda se visualiza a partir del día correspondiente al área horaria anterior.. 

**Observaciones**: 

- Las funciones **&lt;Organizador&gt;.DisplayPeríodo anterior** y [&lt;Organizador&gt;.DisplayNextPeriodo](../WDLang1/1000022896.md) permiten establecer fácilmente los botones "Adelante/Atrás". Estos controles le permiten desplazarse por las áreas de tiempo que muestra la pantalla de control Agenda. 

- El periodo mostrado también puede ser modificado por [&lt;Organizador&gt;.ChangeMode](../WDLang1/1000022879.md) y [&lt;Posición](../WDLang1/1000022897.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to monthly display
ORG_MyOrganizer.ChangeMode(orgzMonth)
// Display the current month
ORG_MyOrganizer.Position(DateSys())
// Switch to previous month
ORG_MyOrganizer.DisplayPreviousPeriod()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Organizer control>.DisplayPreviousPeriod()`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

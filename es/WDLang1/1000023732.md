


## &lt;Hoja de cálculo&gt;.GetSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera la lista de celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculates the sum of all selected cells
// (Numeric cells)
Total is int
FOR EACH sCellName Of PSHEET_Spreadsheet.GetSelection()
	Total += PSHEET_Spreadsheet[sCellName]
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.GetSelection()`
---

**`<Result>: (Array of strings)`**



- Array de cadenas que contienen el nombre de las celdas seleccionadas en el control Hoja de cálculo.

- Cadena vacía (""") si no hay ninguna celda seleccionada.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.


## BorrarParámetro (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un parámetro o una set de parámetros guardados: 

- con la función [SaveParameter](../WDLang1/3025046.md), 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) automáticamente a través de la [persistencia de los datos](../WDChamp/9000019.md) en los controles.




**Atención**: Esta operación es irreversible y no se puede deshacer.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete a parameter (syntax 1)
DeleteParameter("MyParameter")
```


<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Delete a parameter (syntax 1)
DeleteParameter("WIN_Window1\EDT_Edit1")
```


<a name="Example3"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 
```wl
// Delete parameters from the "WIN_Win1" window (syntax 2)
DeleteParameter(WIN_Win1)
// equivalent to: DeleteParameter(WIN_Win1..FullPath)
```

<a name="SYNTAX1"></a>

## Sintaxis

### Borrar un parámetro guardado

`<Result> = DeleteParameter(<Parameter name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).




**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro a borrar. Este parámetro se busca: 

- en el registro (o archivo equivalente en la plataforma en tiempo de ejecución),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) en el archivo de parámetros especificado por [InitParameter](../WDLang1/3025044.md). 


Este nombre puede ser un valor o una ruta completa (los separadores aceptados son el punto (.) y la barra (\\)). Para un ruta completo, todos los valores que se encuentran "debajo" de este ruta serán eliminados (ver ejemplo en la sección de Notas).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Si el modo de inicialización seleccionado es el registro, **BorrarParámetro** no distingue entre mayúsculas y minúsculas ni entre acentos (Parámetro y parámetro son equivalentes).


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Borrar todos los parámetros guardados para el elemento especificado

`<Result> = DeleteParameter(<Element name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md).




**`<Element name>: (Cadena de caracteres)`**

Nombre de la control o ventana cuyos parámetros guardados deben ser borrados. Este elemento se busca en el archivo de parámetros especificado por [InitParameter](../WDLang1/3025044.md) (en el registro por Default).  


<a name="SYNTAX3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Borrar todos los parámetros guardados

`<Result> = DeleteParameter()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la función se ejecuta con éxito,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Si se produce un error, puede get más detalles sobre el error con [ErrorInfo](../WDLang1/3013008.md). 




Se borran todos los parámetros guardados por la aplicación.. Durante el siguiente inicio, la aplicación se comportará como si fuera su primer inicio.
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) El archivo de parámetros de la aplicación especificada por [InitParameter](../WDLang1/3025044.md) se borra por completo. Si los parámetros se han guardado en un archivo INI o XML, este archivo se borra.


## AnimationPlayOnControlProperty (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Anima la modificación de una propiedad entera entre 2 valores. 

Por ejemplo, esta función anima la modificación del [X](../Proprietes/2510135.md) Property para mover un control o anima el [anchura](../Proprietes/2510059.md) Property para cambiar el tamaño de un control. La animación no se puede interrumpir si se está ejecutando.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Animates the modification of control height
AnimationPlayOnControlProperty(EDT_Details, "Height", 16, 200)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AnimationPlayOnControlProperty(<Control> , <Property> , <Start value> , <End value> [, <Duration>])`
---

**`<Control>: (Nombre del control)`**

Nombre del control a manipular.

**`<Property>: (Cadena de caracteres)`**

Nombre de la lengua WLanguage Property a animar. Las propiedades disponibles actualmente utilizan números enteros. Por ejemplo: [X](../Proprietes/2510135.md), [Y](../Proprietes/2510137.md), [anchura](../Proprietes/2510059.md), [Altura](../Proprietes/2510050.md), [Color](../Proprietes/2510071.md), [BackgroundColor](../Proprietes/2510022.md), [Opacidad](../Proprietes/2514006.md), etc.

**`<Start value>: (Integro)`**

Valor inicial de la propiedad.

**`<End value>: (Integro)`**

Valor final de la propiedad.

**`<Duration>: (Entero o duración opcional)`**

Duración de la animación en centésimas de segundo. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de centésimas de segundo,

- una variable de tipo Duration,

- duración en un formato legible (por ejemplo, "1s" o "10ms").

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000123"></a>

La propiedad pasa de &lt;Valor inicial&gt; a &lt;Valor final&gt; durante la &lt;Duración&gt; indicada. Las diferentes etapas se muestran al usuario.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Limitación
<a name="limitacion_ELTPARAGRAPHE000130"></a>

La animación solo está disponible para las siguientes propiedades: 

- propiedad [Altura](../Proprietes/2510050.md), 

- propiedad [anchura](../Proprietes/2510059.md), 

- propiedad [X](../Proprietes/2510135.md), 

- propiedad [Y](../Proprietes/2510137.md),

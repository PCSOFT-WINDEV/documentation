
## MemKeyVal( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20</B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [Associative arrays](../Motscles/1514058.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor (añadido o modificado por [MemAdd](../WDLang1/3053002.md) o [MemModify](../WDLang1/3053007.md)) del elemento actual en una zona de memoria.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the value of the current element
// in "MyMemoryZone"
ResValue = MemKeyVal("MyMemoryZone")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MemKeyVal(<Memory area>)`
---

**`<Result>: (Cadena de caracteres)`**

Valor añadido por [MemAdd](../WDLang1/3053002.md) o modificado por [MemModify](../WDLang1/3053007.md).

**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria, definida por [MemCreate](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Existencia de zona de memoria
<a name="existencia_zona_memoria_ELTPARAGRAPHE000243"></a>

No se comprueba la existencia de la zona de memoria.

Si la zona de memoria especificada no existe, se produce un error WLanguage cuando se utiliza **MemKeyVal**. 

Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Llamada **MemKeyVal**
<a name="llamada_docparampagetitleshort_ELTPARAGRAPHE000098"></a>

**MemKeyVal** debe llamarse únicamente después de las funciones [MemSeek](../WDLang1/3053015.md), [MemSetPosition](../WDLang1/3053018.md), [MemFirst](../WDLang1/3053014.md), [MemLast](../WDLang1/3053004.md), [MemPrevious](../WDLang1/3053011.md) y [MemNext](../WDLang1/3053017.md).


## ExtractLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae una línea específica de una cadena de caracteres.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Extract line 5 from the "test.txt" file
sContent is string = fLoadText("C:\test.txt")
sLine5 is string = ExtractLine(sContent, 5)
```


<a name="Example2"></a>

```wl
// Extract line 5 from the EDT_Info multiline Edit control
sLine5 is string = ExtractLine(EDT_Info, 5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ExtractLine(<Initial string> , <Line number>)`
---

**`<Result>: (Cadena de caracteres)`**



- Contenido de la línea extraída. 

- EOT si el número de línea es desconocido. 




**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres en la que se busca la línea. 

**`<Line number>: (Entero)`**

Número de la línea buscada. Este número de línea está comprendido entre 1 y el número total de líneas en &lt;Cadena inicial&gt;. Las líneas se dividen con retornos de carro (CR).

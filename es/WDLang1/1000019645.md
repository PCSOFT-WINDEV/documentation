
## TrialVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la ejecución actual de la aplicación está en modo "versión de prueba".. 

**Observación**: La Microsoft Store propone un modo de ejecución para las versiones de prueba de las aplicaciones. Esto le permite publicar una versión de prueba gratuita de una aplicación al limitar el acceso a algunas funciones a través de **TrialVersion**. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF TrialVersion() = True THEN
	Info("This feature is not available in the trial version")
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TrialVersion()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la aplicación se ejecuta en modo "versión de prueba",

- <u><u><u><u>False</u></u></u></u> en caso contrario.




## PanelAttach (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta o cambia la posición de conexión de una control Panel acoplable. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a configuration with 2 panels at the bottom, one next to the other
PanelAttach(DOPA_Left, dopaLeft)
PanelAttach(DOPA_Right, dopaTop + dopaRight)

// Create a configuration with 2 panels at the bottom in the same area 
PanelAttach(DOPA_Details, dopaRight)
PanelAttach(DOPA_Info, dopaRight)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PanelAttach(<Dockable Panel control> [, <Reference Panel control>] [, <Position>])`
---

**`<Dockable Panel control>: (Nombre del control)`**

Nombre del control Panel acoplable a utilizar. 

**`<Reference Panel control>: (Nombre de control opcional)`**

Nombre de la control Panel acoplable que se utilizará como reference. Si se especifica este parámetro: 

- la posición set con el parámetro &lt;Posición&gt; será relativa a este control.  

- si no se especifica el parámetro &lt;Posición&gt;, &lt;control Panel acoplable&gt; se posicionará en la misma pestaña que &lt;reference Panel control&gt;.




**`<Position>: (Constante de cuerda opcional (o combinación de constantes))`**

Describe la posición en la que se debe colocar el control Panel acoplable: 


|   |   |
| --- | --- |
| *dopaBottom* | Fijación de la control Panel acoplable en el fondo. |
| *dopaLeft* | Conecta la control Panel acoplable a la izquierda. |
| *dopaRight* | Conecta la control Panel acoplable a la derecha. |
| *dopaTop* | Conecta la control Panel acoplable a la parte superior. |


Las constantes se pueden combinar en pares, donde una constante da la posición Izquierda/Derecha y otra da la posición Inferior/Superior.. 
Si no se especifica este parámetro y el control es un control Panel acoplable, se adjunta a su última posición conocida. 
Si la posición ya está ocupada, se añade el control Panel acoplable al área y se crea y muestra automáticamente un panel.. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La posición de la control Panel acoplable puede conocerse mediante [Posición del panel](../WDLang1/1000023992.md). 

- La función **PanelAttach** provoca un error no fatal si la posición no es válida.

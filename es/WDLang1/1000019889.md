
## WinGiveSuitableWidth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ancho de una ventana, adaptado al contenido de los controles.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the Table control
TableDisplay(WIN_Price.TABLE_Price)
// Calculates the proper width
nWidthOK is int
nWidthOK = WinGiveSuitableWidth(WIN_Price)
IF nWidthOK < 640 THEN
	WIN_Price.Width = nWidthOK
ELSE
	WIN_Price.Width = 640
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WinGiveSuitableWidth([<Window>])`
---

**`<Result>: (Integro)`**

Anchura adaptada al contenido de los controles para optimizar la visualización de los controles en la ventana. El ancho se expresa en pixeles. 

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana a utilizar. La ventana actual se utilizará si no se especifica este parámetro.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El cálculo de la anchura adaptada a la ventana sólo es posible si la ventana contiene uno de los siguientes controles anclado en la anchura:

- Un control List Box, 

- Control Tabla, 

- Control Looper, 

- Control TreeView,

- Control ListView, 

- Control Campo de entrada multilínea.




## OrgSelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona un elemento en una control Organigrama. Si el elemento a seleccionar no se encuentra en el área de Visible del control Organigrama, se realiza automáticamente un desplazamiento.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the subscript of selected element in an Organization Chart control
ElementNum is int 
ElementNum = OrgInfoXY(ORG_MyOrganizationChart, MouseXPos(), MouseYPos())
IF ElementNum <> -1 THEN
	OrgSelectPlus(ORG_MyOrganizationChart, ElementNum)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OrgSelectPlus(<Organization Chart control> , <Subscript>)`
---

**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar.

**`<Subscript>: (Integro)`**

Subíndice del elemento a seleccionar. Este índice es devuelto por [OrgInfoXY](../WDLang1/1000019759.md), por ejemplo.

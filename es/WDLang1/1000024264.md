


## &lt;Procesamiento de palabras&gt;.close (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el archivo docx asociado con el control Procesador de texto y libera el Document. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WP_MyDocument.Close()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Word Processing control>.Close()`
---

**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

El archivo físico associated con este control (si existe) está cerrado. Las modificaciones realizadas en el archivo no se guardarán hasta que se cierre (utilice [&lt;Procesamiento de palabras&gt;.Save](../WDLang1/1000024393.md) si es necesario).

## Componente:
wd280mdl.dll




## OpenContextMenu (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre automáticamente un menú contextual para la ventana control o Page actual.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Click code of BTN_Action button
OpenContextMenu(MenuDateChoice)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`OpenContextMenu([<Context menu name> [, <X> , <Y >]])`
---

**`<Context menu name>: (Cadena de caracteres opcional)`**

Nombre del menú contextual que debe abrirse para la control actual. 
Si no se especifica este parámetro, se abre por defecto el menú contextual associated con el control.


**`<X>: (Entero (opcional))`**

Coordenada X de la posición de apertura del menú contextual. Esta coordenada X se expresa en píxeles, en relación con la Corner superior izquierda de la ventana. Si no se especifican &lt;X&gt; y &lt;Y&gt;, el menú contextual se abre en la posición del ratón.


**`<Y >: (Entero (opcional))`**

Coordenada Y de la posición de apertura del menú contextual. Esta coordenada Y se especifica en píxeles, en relación con la Corner superior izquierda de la ventana. Si no se especifican &lt;X&gt; y &lt;Y&gt;, el menú contextual se abre en la posición del ratón.




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función debe ejecutar el código de una ventana control, o Page. 
	Esta función no debe utilizarse en el código para redimensionar una ventana.

- Esta función es una función de bloqueo hasta que se cierra el menú contextual.
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para una ventana MDI hija, puede ser interesante abrir un menú contextual definido en la ventana MDI padre. En este caso, &lt;Nombre del menú contextual&gt; debe tener el siguiente formato: 
	
	```txt
	NameOfWindow.NameOfContextMenu
	```

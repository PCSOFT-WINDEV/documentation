


## funciones MCI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones MCI son las siguientes:



|   |   |
| --- | --- |
| [MCIBeginning](../WDLang1/3051002.md) | Posiciones al principio de un archivo multimedia que fue abierto previamente [MCIOpen](../WDLang1/3051017.md). |
| [MCIClose](../WDLang1/3051006.md) | Cierra un archivo multimedia. |
| [MCIEnd](../WDLang1/3051007.md) | Posiciones al final de un archivo multimedia que fue abierto previamente [MCIOpen](../WDLang1/3051017.md). |
| [MCIErr](../WDLang1/3051011.md) | Devuelve el número del último error causado por una función MCI (Media control Interface). |
| [MCIExecute](../WDLang1/3051005.md) | Ejecuta un comando MCI (Media control Interface). |
| [MCIHeight](../WDLang1/3051008.md) | Devuelve la altura (en píxeles) de un archivo multimedia utilizado para mostrar imágenes. |
| [MCIMsgErr](../WDLang1/3051021.md) | Devuelve el título del último error ocurrido en una función MCI (Media control Interface). |
| [MCIOpen](../WDLang1/3051017.md) | Abre un archivo multimedia. |
| [MCIPause](../WDLang1/3051015.md) | Pone en pausa la reproducción del archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)). |
| [MCIPlay](../WDLang1/3051001.md) | Reproduce un archivo multimedia. |
| [MCIPosition](../WDLang1/3051013.md) | Devuelve o cambia la posición actual en un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)). |
| [MCIPositionFormat](../WDLang1/3051004.md) | Corrige el formato utilizado para definir la posición en un archivo multimedia (abierto por [MCIOpen](../WDLang1/3051017.md)). |
| [MCIRetrieve](../WDLang1/3051022.md) | Devuelve el último resultado del comando MCI ejecutado por [MCIExecute](../WDLang1/3051005.md). |
| [MCISize](../WDLang1/3051018.md) | Devuelve o cambia el tamaño de un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)). |
| [MCIStatus](../WDLang1/3051009.md) | Devuelve el estado de un archivo multimedia que fue abierto previamente [MCIOpen](../WDLang1/3051017.md). |
| [MCIStopPlaying](../WDLang1/3051003.md) | Detiene la ejecución de un archivo multimedia que fue abierto previamente ([MCIOpen](../WDLang1/3051017.md)). |
| [MCITrack](../WDLang1/3051016.md) | Devuelve la pista actual de un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)). |
| [MCITrackCount](../WDLang1/3051014.md) | Devuelve el número de pistas encontradas en un archivo multimedia que fue abierto previamente ([MCIOpen](../WDLang1/3051017.md)). |
| [MCIVolume](../WDLang1/3051012.md) | Modifica o devuelve el volumen del sonido. |
| [MCIWidth](../WDLang1/3051020.md) | Devuelve la anchura (en píxeles) de un archivo multimedia utilizado para mostrar imágenes. |
| [MCIWindowSize](../WDLang1/3051019.md) | Cambia el tamaño y la posición de la ventana de visualización de un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)). |

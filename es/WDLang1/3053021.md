
## MemFound( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>

<a name="Use"></a>
<a name="description"></a>
Comprueba si se ha encontrado el elemento buscado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Was the element found in the memory area named "MemoryZone"?
ResFound = MemFound("MemoryZone")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MemFound(<Memory area>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento fue encontrado, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria, definida por [MemCreate](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Llamada a MemFound
<a name="llamada_docparampagetitleshort_ELTPARAGRAPHE000087"></a>

La función **MemFound** debe llamarse únicamente después de las funciones [MemNext](../WDLang1/3053017.md), [MemPrevious](../WDLang1/3053011.md), [MemSeek](../WDLang1/3053015.md), [MemFirst](../WDLang1/3053014.md) y [MemLast](../WDLang1/3053004.md).
<a name="NOTE0_2"></a>




### Cheques
<a name="cheques_ELTPARAGRAPHE000113"></a>

**MemFound** no se puede utilizar para realizar una comprobación varias veces en un fila. Por ejemplo, **MemFound** no puede utilizarse de la siguiente manera:


```wl
MemSeek("DayZone", SoughtValue)
WHILE MemFound("DayZone") = True
	MemNext("DayZone")
END
```


En efecto, las búsquedas realizadas en una zona de memoria son búsquedas de coincidencia exacta. En este caso, **MemFound** y [MemOut](../WDLang1/3053009.md) tienen el mismo comportamiento.
 En nuestro ejemplo, **MemFound** devuelve <u><u><u><u>False</u></u></u></u> (no se llega al final de la zona de memoria). No se realizarán las siguientes búsquedas.
<a name="NOTE0_3"></a>


### Existencia de zona de memoria
<a name="existencia_zona_memoria_ELTPARAGRAPHE000275"></a>

No se comprueba la existencia de la zona de memoria.

Si la zona de memoria especificada no existe, se produce un error WLanguage cuando se utiliza **MemFound**. 

Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).

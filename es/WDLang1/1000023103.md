
## SysDeviceInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Recupera información del dispositivo móvil donde se ejecuta la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
let sModelName = SysDeviceInfo(sysModel)
Trace(sModelName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysDeviceInfo(<Type of information>)`
---

**`<Result>: (Cadena de caracteres)`**

Información solicitada. 

**`<Type of information>: (Constante de tipo Integer)`**

Tipo de información solicitada:



|   |   |
| --- | --- |
| *sysManufacturer* | Fabricante del dispositivo |
| *sysModel* | Modelo del dispositivo. |

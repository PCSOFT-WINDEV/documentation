
## ThreadPriority( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el nivel de prioridad de un thread.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperar el nivel de prioridad de una Thread

`<Result> = ThreadPriority(<Thread name>)`
---

**`<Result>: (Constante)`**



- -1 si el hilo no existe

- Nivel de prioridad actual del hilo: 
	


|   |   |
| --- | --- |
| *PriorityHigh* | Prioridad alta . |
| *PriorityLow* | Prioridad baja. |
| *PriorityNormal* | Prioridad normal. |






**`<Thread name>: (Cadena de caracteres)`**

Nombre de thread cuya prioridad se solicita. Este nombre se asigna al ejecutar el hilo (con la función [ThreadExecute](../WDLang1/3077024.md)).  


<a name="SYNTAX2"></a>

### Modificación del nivel de prioridad de una Thread

`<Result> = ThreadPriority(<Thread name> , <Priority>)`
---

**`<Result>: (Constante)`**

Nivel de prioridad de thread antes de ejecutar la función:


|   |   |
| --- | --- |
| *PriorityHigh* | Prioridad alta . |
| *PriorityLow* | Prioridad baja. |
| *PriorityNormal* | Prioridad normal. |



**`<Thread name>: (Cadena de caracteres)`**

Nombre de thread cuya prioridad debe ser modificada. Este nombre se asigna al ejecutar el hilo (con la función [ThreadExecute](../WDLang1/3077024.md)).

**`<Priority>: (Constante)`**

Nuevo nivel de prioridad para el hilo:


|   |   |
| --- | --- |
| *PriorityHigh* | Prioridad alta . |
| *PriorityLow* | Prioridad baja. |
| *PriorityNormal* | Prioridad normal. |

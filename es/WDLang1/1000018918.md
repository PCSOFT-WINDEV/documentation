
## AppInstallVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Instala una versión específica de la aplicación.

**Observación**: Esta función sólo puede ser utilizada con una red instalación en una red local. No puede ser usado con una instalación por IP.




<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`AppInstallVersion(<Version number> [, <Application path>])`
---

**`<Version number>: (Cadena de caracteres)`**

Número de versión de la aplicación a instalar. 

**Observación**: Los números de versión disponibles son devueltos por [AppListVersion](../WDLang1/1000018843.md).

**`<Application path>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable a utilizar. Si no se especifica este parámetro, la aplicación utilizada corresponde a la aplicación actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de instalación de la versión
<a name="condiciones_instalacion_version_ELTPARAGRAPHE000035"></a>

Se comprueba la validez de instalación. No se puede instalar: 

- Una versión prohibida. 

- Una versión idéntica a la versión actual. 

- Una versión si el historial no está habilitado. 




**Si la instalación no puede realizarse**, la aplicación sigue funcionando. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). 

**Si se puede realizar la instalación**, se realizan las siguientes operaciones: 

- la solicitud está cerrada. 

- se instala la versión solicitada. 

- se reinicia la aplicación.


## SysColorRes( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la resolución de la pantalla en número de colores.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the resolution of the screen
ResColorRes = SysColorRes()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysColorRes()`
---

**`<Result>: (Integro)`**

Número de colores para la resolución de la pantalla. Este número está definido en el panel de control de Windows.  







<a name="XComponent"></a>

## Componente:
wd280std.dll

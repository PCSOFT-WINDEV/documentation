
## WINBackgroundImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el fondo Image de una ventana y especifica el modo de visualización de este Image.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Modify the background image of "WIN_EditImage"
// The new image is "C:\Documents\Bkgd.GIF"
// This image will be centered in the window
Res = WinbackgroundImage(WIN_EditImage, "C:\Documents\Bkgd.GIF", imgCentered)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinBackgroundImage(<Window used> , <Image name> , <Display>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó la imagen de fondo,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Window used>: (Nombre de ventana)`**

Nombre de la ventana a utilizar.
Si este parámetro corresponde a una cadena vacía ("""), se utiliza la ventana actual.

**`<Image name>: (Cadena de caracteres)`**

Nombre y ruta completo (o relativo) del Image para mostrar en la ventana. Se puede utilizar una ruta de tipo UNC. No se muestra ningún Image en la ventana si este parámetro no corresponde a un Image existente.

**`<Display>: (Constante)`**

Indica el modo de visualización de la Image:


|   |   |
| --- | --- |
| *imgCentered* | centrado: Image centrada en la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Centre.GIF)<br> |
| *imgH* | Homotético: Image ampliado o reducido (respetando las proporciones del Image inicial) para ocupar toda la superficie de la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HOMO.GIF)<br> |
| *imgHC* | Homotético centrado: Image ampliado o reducido (respetando las proporciones del Image inicial) y centrado en la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HomoCentre.GIF)<br> |
| *imgHCE* | Homotético expandido centrado : Image ampliado o reducido (respetando las proporciones del Image inicial) de manera que el lado más pequeño del Image se muestre completamente en la ventana y centrado en la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HomoCentreEtendu.GIF)<br> |
| *imgHCWIS* | Homotético centrado sin aumento: Si la Image se puede mostrar completamente en la ventana, se muestra al 100% y centrada en la ventana. De lo contrario, la Image se reduce proporcionalmente para que se pueda visualizar y centrar completamente en la ventana. |
| *imgHE* | Homotético expandido: Image ampliado o reducido (respetando las proporciones del Image inicial) de manera que el lado más pequeño del Image se muestra completamente en la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_HOMOETENDU.GIF)<br> |
| *imgHWIS* | Homotético sin aumento: Si la imagen se puede mostrar completamente en la ventana, se muestra al 100%. De lo contrario, la Image se reduce proporcionalmente para poder ser visualizada en su totalidad. |
| *imgNormal* | 100%: Image se muestra al 100% en la parte superior izquierda de Corner de la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_100.GIF)<br> |
| *imgStretched* | Expandido: Image se extiende por toda la superficie de la ventana.<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Etiree.GIF)<br> |
| *imgTiled* | Mosaico: Image se muestra al 100% y Tile d para ocupar toda la superficie de control (Tile).<br><br><br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IMG_Mosaique.GIF)<br> |




## &lt;Hoja de cálculo&gt;.Reemplazar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra y reemplaza un valor en un control Hoja de cálculo. 

Observación: Esta operación puede ser cancelada por el usuario vía Ctrl + Z.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Replace VAT by Value A.T
nRepl is int = PSHEET_MySpreadsheet.Replace("VAT" , "Value A.T.")
IF nRepl = 0 THEN
	Info("Not found")
ELSE
	Info("[%nRepl%] elements have been replaced.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Spreadsheet control>.Replace(<Sought value> , <ReplaceBy> [, <Cell> [, <Options>]])`
---

**`<Result>: (Integro)`**



- Número de reemplazos realizados.

- 0 si no se ha realizado ninguna sustitución.




**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Sought value>: (Real o cadena de caracteres)`**

Valor a encontrar en el control Hoja de cálculo para ser reemplazado.

**`<ReplaceBy>: (Real o cadena de caracteres)`**

Nuevo valor que sustituirá al valor buscado.

**`<Cell>: (Cadena de caracteres opcional)`**

Nombre de la celda a modificar. 

- Si se especifica este parámetro, sólo se sustituirán los valores correspondientes en esta celda.. 

- Si no se especifica este parámetro, se modificarán todos los valores correspondientes (en la hoja de trabajo actual o en todas las hojas de trabajo, según los parámetros &lt;Opciones&gt;)..




**`<Options>: (Constante opcional de tipo Integer)`**

Opciones de búsqueda: 



|   |   |
| --- | --- |
| *psheetSearchWithCase* | Buscar teniendo en cuenta el caso. |
| *psheetSearchContain* | "Contiene" búsqueda. |
| *psheetSearchInFormula* | Buscar sólo en las fórmulas. |
| *psheetSearchExact* | "Búsqueda"exacta. |
| *psheetSearchCurrentWorksheet* | Buscar en la hoja de trabajo actual. |
| *psheetSearchNoCase* | Buscar ignorando el caso. |
| *psheetSearchAnyWorksheet* | Buscar en todas las hojas de trabajo del libro de trabajo. |


Por Default, la búsqueda se realiza según la siguiente combinación de constantes: *psheetSearchWithCase* + *psheetSearchCurrentWorksheet* + *psheetSearchExact*

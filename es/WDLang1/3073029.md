
## SysIMEI( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número IMEI de un Android Phone.

**Observación**: [SysSerialNum](../WDLang1/3073033.md) obtiene el número IMEI de un Android Phone. 
Atención: Esta información está disponible hasta Android 9. A partir de Android 10, esta información ya no está disponible: esta función provoca un error fatal.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF Val(SysAndroidVersion(sysVersionNumber))<10 THEN	
	// Retrieve the IMEI number of the device
	Info("The IMEI number of the device is " + SysIMEI("Cellular Line"))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysIMEI()`
---

**`<Result>: (Cadena de caracteres)`**



- Número IMEI,

- Cadena vacía ("") si se produce un error.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### IMEI
<a name="imei_ELTPARAGRAPHE000055"></a>

La Identidad Internacional de Equipo Móvil (IMEI) es un número que identifica de manera única a cada dispositivo móvil GSM o UMTS.

Este número permite al operador de Phone identificar el dispositivo móvil que llama para que pueda ser autorizado (o no) a conectarse. Se utiliza para conectar las llamadas de emergencia sin SIM. Este número también es útil para pedir a los operadores que bloqueen un móvil robado Phone.

Los dispositivos que no tienen una ranura para tarjeta SIM no tienen un número IMEI (la mayoría de las tabletas).. Para los dispositivos con una tarjeta SIM, el número IMEI puede ser recuperado hasta Android 9. A partir de Android 10, esta información ya no está disponible.

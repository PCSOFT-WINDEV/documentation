


## Dimension( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el tamaño de una Variable (cuerda, numérica, array, ...) o redimensiona una array dinámica .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Resize an array (dynamic array)
MyArr is array dynamic of 5 by 3 int
...
// Resize the array
Dimension(MyArr, 8, 6)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Recuperando el tamaño de un Variable

`<Result> = Dimension(<Variable name>)`
---

**`<Result>: (Integro)`**

Tamaño solicitado.


|   |   |
| --- | --- |
| Cadena Pascal | Número de caracteres de useful en la cadena.<br><br>Ejemplo: Para una cadena de Pascal en 10, el resultado será 10. |
| Cadena ASCIIZ | Número de caracteres de useful en la cadena.<br><br>Ejemplo: Para una cadena ASCIIZ en 10, el resultado será 9 porque hay un 0 binario al final. |
| Variable compuesta y estructura | Suma del tamaño (en bytes) de cada componente.<br><br>**Atención**: Este caso no se aplica a las Clases. |
| Array | Número total de elementos en la array. Esta función es equivalente a la [Conde](../Proprietes/2510097.md) Property.<br><br>Ejemplo: Para una array de 3 cuerdas, el resultado será 3.<br>Ejemplo: Para un array de 5 por 3 enteros, el resultado será 15 (5 \* 3)<br><br>Observación: Para conocer las dimensiones de array, utilice [ArrayInfo](../WDLang1/3075005.md). |
| Otros | Tamaño en bytes del área de memoria correspondiente a la Variable. Para las cadenas de caracteres, utilice [Longitud](../WDLang1/3024031.md). |



**`<Variable name>: (Cadena de caracteres)`**

Nombre de la Variable (cualquier tipo de Variable).



<a name="SYNTAX2"></a>

### Redimensionando una array dinámica de N dimensiones

`Dimension(<Array name> , <Number of elements in dimension 1> [, <Number of elements in dimension 2> [... [, <Number of elements in dimension N>]]])`
---

**`<Array name>: (Cadena de caracteres)`**

Nombre del Variable que corresponde al dinámico array.
**Observación**: La arrays fija no puede redimensionarse.

**`<Number of elements in dimension 1>: (Integro)`**

Número de elementos en la primera dimensión de la array. 
**Atención**: El número de dimensiones de array no puede ser modificado, ni el tipo de los elementos.. Una array bidimensional sigue siendo una array bidimensional.

**`<Number of elements in dimension 2>: (Entero opcional)`**

Número de elementos en la segunda dimensión de la array. 
**Atención**: El número de dimensiones de array no puede ser modificado, ni el tipo de los elementos.. Una array bidimensional sigue siendo una array bidimensional.



**`<Number of elements in dimension N>: (Entero opcional)`**

Número de elementos en la dimensión N de la array. 
**Atención**: El número de dimensiones de array no puede ser modificado, ni el tipo de los elementos.. Una array bidimensional sigue siendo una array bidimensional.


## MatFloatMultiply( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Multiplica cada elemento de matriz por un valor. Esta multiplicación se realiza en la matriz especificada.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Multiply the matrix elements by 3
MatFloatMultiply("MyMatrix", 3)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MatFloatMultiply(<Matrix> , <Value>)`
---

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a utilizar, definida con [MatCreate](../WDLang1/3050058.md).

**`<Value>: (Real)`**

Valor utilizado para la multiplicación.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Dividir todos los elementos de la matriz
<a name="dividir_todos_los_elementos_matriz_ELTPARAGRAPHE000077"></a>

Para dividir todos los elementos de la matriz por un valor dado, multiplique la matriz por: 1/&lt;Valor>.
<a name="NOTE0_2"></a>




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000085"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_3"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000096"></a>

**Atención**: La función **MatFloatMultiply** no devuelve ningún código de error. Para saber si se han generado errores en la multiplicación, utilice [MatError](../WDLang1/3050044.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

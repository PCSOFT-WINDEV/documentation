


## &lt;List Caja&gt;.Cuenta (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos de un cuadro de lista, ListView o control Combo Box.

**Observación**: La función **&lt;List Caja&gt;.Cuenta** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of elements found in the "LIST_CustomerList" control
ResNumberElements = LIST_CustomerList.Count()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <List Box control>.Conde()`
---

**`<Result>: (Integer)`**

Número de elementos (filas) encontrados en la control especificada .

**`<List Box control>: (Control name)`**

Nombre de la Caja de Listas o control Combo Box que se va a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000236"></a>

Si el control List Box (o Combo Box) especificado está relacionado con un archivo que contiene más de 2.147.483.647 registros, el valor devuelto se trunca en este valor.

### control List Box basada en un archivo de datos
<a name="control_list_box_basada_archivo_datos_ELTPARAGRAPHE000248"></a>

Para un control List Box basado en un archivo de datos, la función **&lt;List Caja&gt;.Cuenta** devuelve:

- el número total de registros encontrados en el archivo relacionado si no se ha definido ningún filtro.

- el número de registros que coinciden con el filtro si se ha definido un filtro.

- el número de registros encontrados en la consulta relacionado a la control.




El código de fin de inicialización de control se ejecuta una vez que se calcula el total.

**Atención**: El tiempo para calcular el número de registros puede ser bastante largo
<a name="NOTE0_3"></a>


### Último elemento
<a name="ultimo_elemento_ELTPARAGRAPHE000269"></a>Para posicionarse en el último elemento de control List Box, utilice la siguiente sintaxis:

```txt
<List Box control>.SelectPlus(<List Box control>.Count)
```


o


```txt
<List Box control> = <List Box control>.Count
```
o

```txt
<List Box control> = <List Box control>.Count()
```

<a name="NOTE0_4"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000296"></a>

- Para conocer el número de filas de una control Tabla, utilice [&lt;Tabla&gt;.cuenta](../WDLang1/1000024277.md).

- **&lt;List Caja&gt;.Cuenta** puede utilizarse en una columna "Combo Box" tabla.

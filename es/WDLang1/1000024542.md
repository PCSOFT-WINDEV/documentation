


## &lt;Documento Variable&gt;.ToImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Exporta una página contenida en una [Document](../WDLang1/1000022461.md) Variable en formato Image.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDoc is Document <- WP_MyDocument
d is Image = Mydoc.ToImage(1)
dSaveImageJPEG(d, "test")
ShellExecute("test.jpg")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Document>.ToImage(<Page>)`
---

**`<Result>: (Imagen Variable)`**

Nombre de la [Image](../WDLang1/1000019650.md) Variable que corresponde a la página en el tratamiento de textos Document.

**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

**`<Page>: (Integro)`**

Número de la página a exportar en formato Image.




## <PDF Reader>.Highlight (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Encuentra un texto en el Document mostrado por un lector de PDF control y resalta las ocurrencias de este texto en el control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Highlights all "WINDEV" occurrences found in a PDF Reader control in yellow
PDF_READER_PDF.Highlight("WINDEV", LightYellow, WholeWord)
// Highlights all "WEBDEV" occurrences in blue
PDF_READER_PDF.Highlight("WEBDEV", LightBlue, WholeWord)
// Highlights all "WINDEV Mobile" occurrences in orange
PDF_READER_PDF.Highlight("WINDEV Mobile", LightOrange, WholeWord)
// Deletes all highlighting
PDF_READER_PDF.Highlight("")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <PDF Reader control>.Highlight(<Searched text> [, <Color> [, <Search options>]])`
---

**`<Result>: (Integro)`**



- Número de ocurrencias de texto encontradas en la control.

- 0 si ningún elemento contiene el texto buscado.




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<Searched text>: (Cadena de caracteres)`**



- Texto que debe encontrarse en el elemento especificado.

- "" (cadena vacía) a Cancel la búsqueda.




**`<Color>: (Entero, constante u opcional Color Variable)`**

Color usado para resaltar las ocurrencias encontradas. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

- una variable de tipo [Color](../WDLang1/1000019661.md).




**Observación**: La color utilizada por Default es de color amarillo claro.

**`<Search options>: (Constante opcional de tipo Integer)`**

Indica las opciones de búsqueda:


|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios). |


De forma predeterminada, no se selecciona ninguna opción: la búsqueda distingue entre mayúsculas y minúsculas, y se realiza sobre palabras incompletas.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Mientras no se llame a esta función con una cadena vacía ("") para detener el resaltado, todas las nuevas llamadas añadirán un resaltado adicional.

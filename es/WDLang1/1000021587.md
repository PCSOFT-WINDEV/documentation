
## Constantes de tareas paralelas
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones que gestionan las tareas paralelas utilizan las siguientes constantes: 

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **ptoDelayedCopyHFSQLContext** | Dispara un ligero copy de contexto HFSQL **durante el primer acceso a los datos HFSQL**. | [Descripción de ParallelTask](../WDLang1/1000021586.md), variable de tipo [ParallelTask](../WDLang1/1000021220.md), <br>[ParallelTaskExecute](../WDLang1/1000021211.md), <br>[ParallelTaskExecuteAfter](../WDLang1/1000021214.md), <br>[ParallelTaskExecuteAfterAll](../WDLang1/1000021217.md)<br>[ParallelTaskExecuteAfterOne](../WDLang1/1000021216.md) |
| **ptoFullCopyHFSQLContext** | Activa la copy inmediata del contexto HFSQL actual.<br>Se recomienda, por ejemplo, si la tarea paralela debe tener en cuenta las posiciones actuales en los archivos y consultas del contexto de la persona que llama. | [Descripción de ParallelTask](../WDLang1/1000021586.md), variable de tipo [ParallelTask](../WDLang1/1000021220.md), <br>[ParallelTaskExecute](../WDLang1/1000021211.md), <br>[ParallelTaskExecuteAfter](../WDLang1/1000021214.md), <br>[ParallelTaskExecuteAfterAll](../WDLang1/1000021217.md)<br>[ParallelTaskExecuteAfterOne](../WDLang1/1000021216.md) |
| **ptoLightCopyHFSQLContext** | Activa la copy inmediata de una parte del contexto HFSQL actual.<br>Solo se almacenan los directorios que contienen los archivos de datos en modo HFSQL Classic y/o las conexiones en modo HFSQL Client/Server. | [Descripción de ParallelTask](../WDLang1/1000021586.md), variable de tipo [ParallelTask](../WDLang1/1000021220.md), <br>[ParallelTaskExecute](../WDLang1/1000021211.md), <br>[ParallelTaskExecuteAfter](../WDLang1/1000021214.md), <br>[ParallelTaskExecuteAfterAll](../WDLang1/1000021217.md)<br>[ParallelTaskExecuteAfterOne](../WDLang1/1000021216.md) |
| **ptoMainThread** | Ejecuta la tarea en el principal Thread.<br>Atención: Esta constante no puede ser usada para ejecutar una tarea paralela en la thread principal de la thread principal. | [Descripción de ParallelTask](../WDLang1/1000021586.md), variable de tipo [ParallelTask](../WDLang1/1000021220.md), <br>[ParallelTaskExecute](../WDLang1/1000021211.md), <br>[ParallelTaskExecuteAfter](../WDLang1/1000021214.md), <br>[ParallelTaskExecuteAfterAll](../WDLang1/1000021217.md)<br>[ParallelTaskExecuteAfterOne](../WDLang1/1000021216.md) |
| **ptpMaximumNumberThread** | Número máximo de hebras que se pueden ejecutar al mismo tiempo. | [ParallelTaskConfigure](../WDLang1/1000021284.md) |
| **ptsCanceled** | Status de la tarea paralela: La tarea paralela se cancela ([ParallelTaskCancel](../WDLang1/1000021278.md)). | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsCancellationRequested** | Status de la tarea paralela: Se realizó una solicitud de cancelación en la tarea paralela ([ParallelTaskRequestCancellation](../WDLang1/1000021213.md)). | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsCompleted** | Status de la tarea paralela: La tarea paralela ha finalizado. | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsExecutionInProgress** | Status de la tarea paralela: La tarea paralela se está ejecutando actualmente. | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsNotScheduled** | Status de la tarea paralela: La tarea paralela no está programada. | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsWaitingExecution** | Status de la tarea paralela: La tarea paralela está a la espera de ser ejecutada. | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |
| **ptsWaitingPrevious** | Status de la tarea paralela: La tarea paralela espera la ejecución de una tarea paralela anterior. | Variable de tipo [ParallelTask](../WDLang1/1000021220.md) |

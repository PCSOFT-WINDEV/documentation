
## LoadImageSheet( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Carga una imagen set en memoria para que pueda ser utilizada posteriormente por [gImage](../WDLang1/3029042.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The MULTI.BMP image includes 130 components
// No border will be displayed around each one of these components
LoadImageSheet(2, "MULTI.BMP", 16, 16)
// Display the images in a graphic list
FOR Subscript = 1 TO 10*13
	ListAdd(LIST_GraphicList, gImage(2, Subscript))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = LoadImageSheet(<Image identifier> , <Image name> , <Image width> , <Image height> [, <Border>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el conjunto de imágenes se cargó en memoria,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image identifier>: (Integro)`**

Identificador de Image cargado en la memoria (incluido entre 1 y 9). Si este identificador ya se utiliza, el Image cargado reemplaza al Image anterior. Este identificador puede ser utilizado por la función [gImage](../WDLang1/3029042.md).

**`<Image name>: (Cadena de caracteres)`**

Nombre y apellido (o familiar) ruta de Image para cargar. Se puede utilizar una ruta de tipo UNC. 

**`<Image width>: (Integro)`**

Ancho de una imagen encontrada en la imagen set. 

**`<Image height>: (Integro)`**

La altura de una Image encontrada en la hoja de Image.

**`<Border>: (Carácter opcional)`**

"C" si no se debe mostrar Border alrededor de cada dibujo. Mediante Default, se muestra un Border de 1 píxel alrededor de cada dibujo que se encuentra en la hoja Image.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Dibujos numerados
<a name="dibujos_numerados_ELTPARAGRAPHE000054"></a>

Los juegos Image incluyen varios dibujos organizados en filas y columnas. Todos estos dibujos deben tener el mismo tamaño y pueden ser encerrados en un Border de 1 píxel.

Los dibujos están numerados del 1 al &lt;Número de filas&gt; \* &lt;Número de columnas&gt; de izquierda a derecha y de arriba a abajo.

Observaciones: 

- A diferencia de [LoadImage](../WDLang1/3029022.md), si el número de dibujos cambia en Image, los Image s mostrados no se truncarán y no habrá que modificar el código correspondiente. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [gImageExists](../WDLang1/1000021860.md) se utiliza para determinar si una imagen fue cargada en memoria por [LoadImage](../WDLang1/3029022.md) o **LoadImageSheet**.



<a name="NOTE0_2"></a>


### Límites
<a name="limites_ELTPARAGRAPHE000082"></a>

**LoadImageSheet** no debe utilizarse en el código del proyecto. De hecho, esta función debe ser usada en la ventana donde el Image debe ser manejado.

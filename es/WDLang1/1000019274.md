


## SchedulerDeleteAppointment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un Appointment de un control Planificador. Si el control Planificador es relacionado a una fuente de datos, el Appointment también se borra de la fuente de datos. 



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes the selected appointment from the SCH_Schedule control
SchedulerDeleteAppointment(SCH_Schedule)

// Deletes the first added appointment
SchedulerDeleteAppointment(SCH_Schedule, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Borrar una Appointment identificada por su índice

`SchedulerDeleteAppointment(<Scheduler control> [, <Appointment index>])`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Appointment index>: (Entero opcional)`**

Índice de la Appointment a eliminar. El índice de la primera Appointment es set a 1. Si no se especifica este parámetro, se borrará la Appointment seleccionada.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Suprimir un Appointment identificado por un Appointment Variable

`SchedulerDeleteAppointment(<Scheduler control> , <Appointment used>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




**`<Appointment used>: (Appointment Variable)`**

variable [Appointment](../WDLang1/1000019244.md) correspondiente a la appointment a eliminar.

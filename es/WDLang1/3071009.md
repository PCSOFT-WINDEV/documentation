


## Funciones de gestión del ratón
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las siguientes funciones se utilizan para gestionar el ratón: 



|   |   |
| --- | --- |
| [ControlOver](../WDLang1/3025021.md) | Identifica la control sobre la que se pasa el cursor del ratón. |
| [CursorCapture](../WDLang1/3071004.md) | Dirige todas las Event s asociadas al ratón hacia una ventana o control específico, independientemente del objeto de la pantalla donde se encuentre la Event del ratón. |
| [CursorDisplay](../WDLang1/3071003.md) | Muestra u oculta el cursor del ratón. |
| [CursorPos](../WDLang1/3071006.md) | Devuelve y modifica la posición del cursor del ratón. |
| [HourGlass](../WDLang1/3071011.md) | Transforma el puntero del ratón en HourGlass( y a la inversa).<br>En Android e iOS, muestra una barra de progreso interminable en una ventana. |
| [HourglassNetwork](../WDLang1/1000020797.md) | Muestra u oculta una barra de progreso sin fin: <br><br>- en la barra de estado de una aplicación iOS. <br><br>- en la barra de título de las ventanas de una aplicación Android.<br><br><br> |
| [MouseXPos](../WDLang1/3071007.md) | Devuelve la posición horizontal (X) del cursor del ratón respecto a la ventana o control especificado. Se utiliza en una Process de ratón o lápiz óptico (clic, hover, botón izquierdo o derecho pulsado, etc.). |
| [MouseYPos](../WDLang1/3071008.md) | Devuelve la posición vertical (Y) del cursor del ratón en relación con la ventana o el control especificado. Se utiliza en una Process de ratón o lápiz óptico (clic, hover, botón izquierdo o derecho pulsado, etc.). |

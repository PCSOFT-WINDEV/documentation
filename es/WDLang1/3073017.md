
## SysIconize( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar si una ventana está minimizada.. Esta ventana no fue creada por WINDEV.

**Observación**: Para saber si una ventana de WINDEV está minimizada, utilice [WinSize](../WDLang1/3038020.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out whether "Microsoft Word - Sales.DOC" is minimized
ResHandle = SysWinHandle("Microsoft Word - Sales.DOC")
WindowMinimized = SysIconize(ResHandle)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysIconize(<Window handle>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la ventana especificada está minimizada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Window handle>: (Integro)`**

Mango de la ventana a ensayar. Esta manipular es devuelta por [SysWinHandle](../WDLang1/3073007.md).

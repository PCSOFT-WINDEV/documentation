


## &lt;Maximizar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una ventana WINDEV en su tamaño máximo.

**Observación**: **&lt;Maximizar** realiza la misma acción que el botón de maximizar situado en la parte superior derecha corner de las ventanas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the "WIN_EditCustomer" window in its maximum size
WIN_CustomerInput.Maximize()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Window>.Maximize()`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana que se debe mostrar en su tamaño máximo.



<a name="XComponent"></a>

## Componente:
wd280obj.dll

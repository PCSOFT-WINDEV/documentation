


## fSelectDir( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un selector de directorios.

El selector de directorios es el siguiente (el aspecto puede cambiar según el sistema utilizado)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fRepS%E9lecteur%20-%20HC%20N%B0001.gif)
**Observación**: Tiene la capacidad de seleccionar un directorio local o un directorio encontrado en un disco de red.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036050&name=fselectdir_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Select a directory
Directory = fSelectDir("C:\Directories", "Select a directory", ...
	"Select the directory that will be copied")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fSelectDir(<Directory selected by default> , <Picker title> [, <Picker comments> [, <Root directory> [, <Options>]]])`
---

**`<Result>: (Cadena de caracteres)`**



- Ruta y nombre completo del directorio seleccionado,

- Cadena vacía (""") si no se ha seleccionado ningún directorio. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Directory selected by default>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del directorio seleccionado por Default. Se puede utilizar una ruta UNC. Este parámetro debe terminar con "\\".. Si este parámetro es una cadena vacía (""), ningún directorio será seleccionado por Default.
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Picker title>: (Cadena de caracteres)`**

Texto que aparece en la barra de título del selector de directorios. 
Si este parámetro es una cadena vacía ("""), se mostrará el título "Buscar una carpeta".

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Picker comments>: (Cadena de caracteres opcional)`**

Los comentarios aparecen en la parte superior del selector de directorios.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Root directory>: (Cadena de caracteres opcional)`**

Nombre y ruta completa del directorio "raíz" en la estructura de árbol sugerida. Se puede utilizar una ruta UNC. Si este parámetro es una cadena vacía ("""), el directorio "Escritorio" iniciará la estructura de árbol propuesta..
Este parámetro se utiliza para limitar la selección a los directorios de un disco determinado o a los subdirectorios de un directorio determinado.

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Este parámetro puede estar en formato Ansi o Unicode.

Si este parámetro difiere de la cadena vacía ("""), se utilizará el anterior selector de directorio.

**`<Options>: (Constante opcional de tipo Integer)`**

Permite especificar: 

- el formato del resultado de la función. Por Default, **fSelectDir** devuelve una cadena de caracteres en formato Ansi. 

- el estilo del selector de directorios. 





|   |   |
| --- | --- |
| *fdForbidVirtualFolder* | &lt;Resultado&gt; no contendrá los directorios virtuales.<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *fdFormerAspect* | Le permite utilizar el antiguo selector de directorios (estilo antiguo). <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=SelectRep.gif)<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode.<br>![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Esta constante no está disponible. |

## Observaciones


### Selección de archivos
<a name="seleccion_archivos_ELTPARAGRAPHE000124"></a>

La función **fSelectDir** permite seleccionar un directorio. Para seleccionar un archivo, utilice [fSelect](../WDLang1/3036053.md).
<a name="NOTE0_3"></a>
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) 

### Modo de funcionamiento en Java
<a name="modo_funcionamiento_java_ELTPARAGRAPHE000138"></a>

**En Java, en Windows**, al utilizar la función **fSelectDir** se muestra una ventana Java que simula el aspecto y el modo de funcionamiento del selector de directorios nativo de Windows.

Para forzar el uso del selector de directorios Java, utilice &lt;wd.prop.sel_rep_defaut&gt; al iniciar la aplicación y especifique uno de los siguientes valores: sí/o/sí/y/verdadero/True. 
**Ejemplo** del comando Line utilizado para iniciar una aplicación Java forzando el uso del selector de directorio Java:


```txt
java -Dwd.prop.sel_rep_defaut=y -jar MyApplication.jar
```

 
**En Java, en los de más OS (Linux, Mac, etc.**, al utilizar la función **fSelectDir** se muestra el selector de directorios Java.

Para forzar la visualización de una ventana Java que simule el aspecto y el comportamiento del selector de directorios nativo de Windows, utilice &lt;wd.prop.sel_rep_windows&gt; al iniciar la aplicación y especifique uno de los siguientes valores: sí/o/sí/y/verdadero/True. 
**Ejemplo** del comando Line utilizado para iniciar una aplicación Java forzando el uso del selector de directorio de Windows:


```txt
java -Dwd.prop.sel_rep_windows=y -jar MyApplication.jar
```

### Antiguo y nuevo selector de directorios
<a name="antiguo_nuevo_selector_directorios_ELTPARAGRAPHE000168"></a>

A partir de la versión 23, el nuevo selector de directorios es usado por Default. 

Si el nuevo selector no está soportado (el nuevo selector está soportado desde Windows Vista) o si el parámetro &lt;Root&gt; no está vacío (el directorio raíz no está soportado por el nuevo selector), el selector anterior se muestra incluso si el parámetro *antiguo espectador* no se especifica.

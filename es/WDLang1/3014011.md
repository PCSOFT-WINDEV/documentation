


## Funciones varias de Windows
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones varias de Windows son las siguientes:  



|   |   |
| --- | --- |
| [API](../WDLang1/3014005.md) | Ejecuta una función que se encuentra en una DLL externa. |
| [APIConfigure](../WDLang1/3014026.md) | Modifica las opciones para llamar a [API](../WDLang1/3014005.md) y [CallDLL32](../WDLang1/3014003.md). |
| [BitcoinValidAddress](../WDLang1/1410088858.md) | Comprueba si la dirección de un monedero Bitcoin es válida. |
| [CallDLL32](../WDLang1/3014003.md) | Ejecuta una función que se encuentra en una DLL externa. |
| [CallInterface](../WDLang1/1000017088.md) | Ejecuta un método de una interfaz de un objeto implementado en una DLL externa a la WINDEV framework. |
| [COMCreateInstance](../WDLang1/1000018831.md) | Instancia un objeto COM (Component Object Model). |
| [COMQueryInterface](../WDLang1/1000019784.md) | Llama al método QueryInterface del objeto COM especificado a get una interfaz específica de este objeto para que pueda ser usado con [COMCallMethod](../WDLang1/1000018832.md). |
| [CreateShortCut](../WDLang1/3014008.md) | Crea un acceso directo en el escritorio de Windows, en el menú "Inicio" de Windows o en un directorio específico.. |
| [DeleteShortCut](../WDLang1/3014012.md) | Elimina un acceso directo creado previamente por [CreateShortcut](../WDLang1/3014008.md) desde el escritorio de Windows, desde el menú "Inicio" de Windows o desde un directorio específico. |
| [FreeDLL](../WDLang1/3014009.md) | Libera la librería de 32 bits (DLL) que fue cargada en memoria por [LoadDLL](../WDLang1/3014006.md). |
| [HiWord](../WDLang1/3014017.md) | Devuelve los dos bytes altos de un entero. |
| [IBANValid](../WDLang1/1410089019.md) | Comprueba la validez de un número IBAN (número de cuenta bancaria internacional). |
| [IconInstall](../WDLang1/3014016.md) | Crea un icono (associated con un programa) en un grupo de programas de Windows, con el comando associated Line. |
| [Instance](../WDLang1/3014019.md) | Devuelve el "sistema manipular" (HINSTANCE) del programa actual.. |
| [KeyPressed](../WDLang1/3014007.md) | Comprueba qué tecla está pulsada. |
| [LoadDLL](../WDLang1/3014006.md) | Carga la biblioteca especificada (DLL) en la memoria. |
| [LoWord](../WDLang1/3014018.md) | Devuelve los dos bytes bajos de un entero. |
| [MakeInteger](../WDLang1/3062016.md) | Crea un entero de 4 bytes a partir de dos enteros de 2 bytes. |
| [Método COMCall](../WDLang1/1000018832.md) | Ejecuta un método de una interfaz de un objeto COM instanciado por [COMCreateInstance](../WDLang1/1000018831.md). |
| [OpenSSLCheck](../WDLang1/1000020432.md) | Comprueba, mediante una clave public, que la firma de los datos realizada con la librería OpenSSL es correcta. |
| [Ping](../WDLang1/3014013.md) | Comprueba si se puede llegar a una red Address utilizando el protocolo ICMP (similar a la utilidad de la red PING). |
| [sComputeCrc16](../WDLang1/3062007.md) | Comprueba una cadena de caracteres antes y después de la transmisión entre las aplicaciones WINDEV y los sitios WEBDEV. |
| [sComputeCrc32](../WDLang1/3062015.md) | Calcula la comprobación de redundancia cíclica (CRC) del búfer. |
| [SendKey](../WDLang1/3014014.md) | Simula las pulsaciones de teclas y las acciones del ratón. |
| [Transfer](../WDLang1/3014015.md) | Copia un bloque de bytes o una cadena de longitud fija de una memoria Address a otra. |
| [WindowsVersion](../WDLang1/3014002.md) | Devuelve el modo de ejecución de Windows. |





También se pueden utilizar los siguientes tipos de variables: 



|   |   |
| --- | --- |
| [API description](../WDLang1/1000019149.md) | Un **API description** Variable se utiliza para describir un API, sus parámetros, su valor de retorno y sus opciones de llamada mediante programación. |

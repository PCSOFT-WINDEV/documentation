
## Factorial( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el factorial de un número entero.

En matemáticas, el factorial de un número natural n (¡escrito n!) es el producto de los números enteros estrictamente positivos menores o iguales a n. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res = Factorial(5) // Returns 120
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Factorial(<n>)`
---

**`<Result>: (Entero o real)`**

Factorial del número. 

**`<n>: (Integro)`**

Número entero cuyo factorial debe calcularse.

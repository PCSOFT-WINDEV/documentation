


## &lt;Pivot tabla&gt;.CollapseAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta todas las cabeceras de fila y/o de columna en un control Tabla dinámica..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Loads the data file
PVT_Statistics.Load(sFile)
// Collapses all headers of the Pivot Table control
PVT_Statistics.CollapseAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Pivot Table control>.CollapseAll([<Type of header>])`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar. Esta control Tabla dinámica debe haber sido rellenada por [&lt;Tabla dinámica&gt;.CalculateAll](../WDLang1/1000022180.md) o por [&lt;Tabla dinámica&gt;.Load](../WDLang1/1000022181.md).

**`<Type of header>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Tipo de encabezado a colapsar: 


|   |   |
| --- | --- |
| *pvtColumnHeader* | Oculta las cabeceras de columna. |
| *pvtRowHeader* | Colapsa las cabeceras de fila. |


Si no se especifica este parámetro, los encabezados de filas **Y** columnas se contraen.

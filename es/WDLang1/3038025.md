
## WinIconBarHeight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la altura actual de la zona "Icon bar" que se encuentra en la ventana padre MDI actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modifies the height of the area
WinIconBarHeight(50)
```


<a name="Example2"></a>



```wl
// Retrieves the height of the area
AreaHeight is int 
AreaHeight = WinIconBarHeight()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener el alto del área "Barra de íconos"

`<Result> = WinIconBarHeight()`
---

**`<Result>: (Integro)`**

Tamaño actual de la barra de Icon (en píxeles).  


<a name="SYNTAX2"></a>

### Modificar el alto del área "Barra de íconos"

`<Result> = WinIconBarHeight(<New Height>)`
---

**`<Result>: (Integro)`**

Tamaño de la barra de Icon antes de la modificación (en píxeles).

**`<New Height>: (Integro)`**

Nuevo tamaño de la barra de Icon (en píxeles). Si este parámetro no se especifica o si es negativo, la altura de la barra de Icon permanece sin cambios.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**WinIconBarHeight** no tiene efecto si no hay ninguna ventana madre MDI abierta.

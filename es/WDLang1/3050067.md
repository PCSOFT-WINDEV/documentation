
## ArcCos( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el coseno del arco de un valor numérico (entero o real).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve the arc cosine of 0.5
ArcCosineRes = ArcCos(0.5)
// Returns 60
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ArcCos(<Numeric value>)`
---

**`<Result>: (Real)`**

Arccoseno (ángulo del coseno en grados) del valor numérico especificado, en el [ 0,180] interval.

**`<Numeric value>: (Entero o real)`**

Valor numérico (coseno) a utilizar. Este valor debe encontrarse en el [-1,1] interval. De lo contrario, se produce un error WLanguage.

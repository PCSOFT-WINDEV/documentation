
## LuhnCheck( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba que la secuencia dada es válida según la fórmula de Luhn. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sIDNumber is string = EDT_IDNum
IF LuhnCheck(sIDNumber) THEN
	Info("Valid number")
ELSE
	Info("Invalid number")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LuhnCheck(<Identifier>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la secuencia es válida según la fórmula de Luhn,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Identifier>: (Cadena de caracteres)`**

Secuencia compuesta únicamente por números y espacios (los espacios no son relevantes para la fórmula).




## &lt;Hoja de cálculo&gt;.SortSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ordena los valores seleccionados en un control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Sorts the [A2:B5] cells of current worksheet
PSHEET_Spreadsheet.SelectPlus("A2", "B5") 
PSHEET_Spreadsheet.SortSelection(asAscending)
```
<a name="Example2"></a>

```wl
// Sorts the entire worksheet
// Selects a cell
PSHEET_Spreadsheet = "A1" 
// Sorts all the rows and columns with data below and on the right
PSHEET_Spreadsheet.SortSelection(asAscending)
IF ErrorOccurred = True THEN
	Error(ErrorInfo())
END
```
<a name="Example3"></a>

```wl
// Sorts the selection according to different criteria
// Column A: Ascending sort
// Column B: Descending sort
PSHEET_Spreadsheet.SortSelection("A" + TAB + "B" + asDescending)
IF ErrorOccurred = True THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Clasificar la selección según un criterio de clasificación

`<Spreadsheet control>.SortSelection(<Options>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Options>: (Constante o combinación de constantes de tipo Integer)`**

Opciones de orden: 


|   |   |
| --- | --- |
| *asAscending*<br>(Valor predeterminado) | Orden ascendente. |
| *asDescending* | Orden descendente. |
| *tccIgnoreAccent* | Ordenar sin tener en cuenta los caracteres acentuados. |
| *tccIgnoreCase* | Ordene mientras ignora el caso. |
| *tccIgnoreInsideSpace* | Ordenar sin tener en cuenta los caracteres de espacio que se encuentran dentro de las cadenas. |
| *tccIgnorePonctuationAndSpace* | Ordenar sin tener en cuenta el espacio y los caracteres de puntuación. |
| *tccIgnoreSpace* | Ordenar sin tener en cuenta los caracteres de espacio que se encuentran al principio y al final de las cadenas. |
| *tccLexicographicOrder* | Ordenar según el orden lingüístico. |
| *tccRespectNumeric* | Ordenar respetando el orden de los valores numéricos. |




<a name="SYNTAX2"></a>

### Clasificando la selección según varios criterios de clasificación

`<Spreadsheet control>.SortSelection(<Criteria>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Criteria>: (Cadena de caracteres)`**

Clasificar criterios en el siguiente formato: 

```txt
"<Name column1>[;< Option>]" + TAB + "<Name column2>[;< Option>]" + TAB + ...
```
donde: 

- &lt;Nombre columna&gt; corresponde al nombre de la columna a ordenar. 

- &lt;Opción&gt; corresponde a una constante (o a una combinación de constantes) utilizada para definir el orden que se aplicará a la columna.. Si no se especifica este parámetro, se utilizará la constante *asAscending*. Este parámetro puede corresponder a: 
	


|   |   |
| --- | --- |
| *asAscending*<br>(Valor predeterminado) | Orden ascendente. |
| *asDescending* | Orden descendente. |
| *tccIgnoreAccent* | Ordenar sin tener en cuenta los caracteres acentuados. |
| *tccIgnoreCase* | Ordene mientras ignora el caso. |
| *tccIgnoreInsideSpace* | Ordenar sin tener en cuenta los caracteres de espacio que se encuentran dentro de las cadenas. |
| *tccIgnorePonctuationAndSpace* | Ordenar sin tener en cuenta el espacio y los caracteres de puntuación. |
| *tccIgnoreSpace* | Ordenar sin tener en cuenta los caracteres de espacio que se encuentran al principio y al final de las cadenas. |
| *tccLexicographicOrder* | Ordenar según el orden lingüístico. |
| *tccRespectNumeric* | Ordenar respetando el orden de los valores numéricos. |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La clasificación se realiza en la selección actual. 

- Si se selecciona una sola celda, la función busca la mejor selección: todas las filas y columnas con datos abajo y a la derecha. 

- Si no se puede realizar la ordenación, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u> y los detalles del error se devuelven con [ErrorInfo](../WDLang1/3013008.md). 

- La primera columna se utiliza como criterio de clasificación.

- La clasificación puede ser cancelada por el usuario a través de Ctrl + Z.

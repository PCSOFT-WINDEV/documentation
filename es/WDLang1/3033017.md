
## Creación de una aplicación que gestiona el euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Etapas a seguir
<a name="etapas_seguir_ELTTEXTE000099"></a>
El método para crear una aplicación que soporte el cambio al Euro es tan simple como el método para crear una aplicación WINDEV.. Las operaciones a realizar son las siguientes:

1. Cree su proyecto.

2. Crear el análisis de associated. Las posiciones utilizadas para almacenar monedas deben ser posiciones de "Moneda".

3. Generar el análisis.

4. Describa las ventanas, informes, ... Para cada valor de moneda que se debe soportar:

	- especificar que la columna associated control (control Campo de entrada, tabla o reporte control) debe ser una "Moneda+Euro" control,

	- especifique la moneda de visualización,

	- especifique la moneda almacenada.




5. Inicializar los tipos de cambio de las monedas no utilizadas en el programa a través de [eInitCurrency](../WDLang1/3033003.md): utilizar un tipo de cambio igual a 0.

6. Pruebe la aplicación.

7. Cree la biblioteca (si es necesario) y el ejecutable.

8. Crea la instalación de la aplicación y distribuye tus programas.

## El euro en sus archivos de datos
<a name="euro_sus_archivos_datos_ELTTEXTE000123"></a>


### Su aplicación gestiona euros y otra moneda
<a name="aplicacion_gestiona_euros_otra_moneda_ELTPARAGRAPHE000034"></a>

Para una mayor seguridad, le recomendamos que mantenga todos los valores de sus divisas tanto en francos como en euros.. Por lo tanto, si su archivo utiliza cantidades, le recomendamos que utilice los siguientes elementos:

- importe en francos,

- importe en euros,

- La moneda reference (en la que se realiza la transacción),

- diferencia de redondeo.




Este método de almacenamiento de datos se utiliza para facilitar todos los tipos de procesos futuros.
<a name="NOTE2_2"></a>


### Su aplicación sólo gestiona euros
<a name="aplicacion_solo_gestiona_euros_ELTPARAGRAPHE000048"></a>

Su aplicación está configurada para ejecutarse después del 28 de febrero de 2002.. La visualización en francos se da sólo a título informativo: sus archivos de datos sólo pueden almacenar valores expresados en euros.

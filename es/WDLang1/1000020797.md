


## HourglassNetwork( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra u oculta una barra de progreso sin fin: 

- en la barra de estado de una aplicación iOS. 

- en la barra de título de las ventanas que se encuentran en una aplicación Android.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the progress bar
HourglassNetwork(True)
// Process
...
Multitask(-1)
...
// Hide the progress bar
HourglassNetwork(False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`HourglassNetwork(<Activation>)`
---

**`<Activation>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar la barra de progreso infinita, 

- <u><u><u><u>True</u></u></u></u> para ocultar la barra de progreso infinita.

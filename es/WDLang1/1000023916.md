
## GetUUID256( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula un identificador único universal de 256 bits ("UUID"). 

Este identificador es único en el mundo, independientemente de la máquina en la que se haya fabricado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyUUID256 is 256-bit UUID
MyUUID256 = GetUUID256()
// Example of the UUID256 calculated: cc0ca42fd00f354e9f0ed1386590e0fd5d75a90f28ec822d138c00ba89021956
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GetUUID256()`
---

**`<Result>: (256-bit UUID)`**

Identificador generado.
El formato UUID no tiene separadores.
Por ejemplo: "cc0ca42fd00f354e9f0ed1386590e0fd5d75a90f28ec822d138c00ba89021956".

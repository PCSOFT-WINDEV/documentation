


## &lt;Restaurar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una ventana WINDEV a su tamaño inicial.

**Observación**: **&lt;Restaurar** realiza la misma acción que el botón de restauración situado en la parte superior derecha corner de las ventanas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the "WIN_EditCustomer" window in its initial size
WIN_CustomerInput.Restore()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Window>.Restore()`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana que se debe mostrar en su tamaño inicial.



<a name="XComponent"></a>

## Componente:
wd280obj.dll


## SpellSuggest( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de posibles palabras encontradas en el diccionario de OpenOffice que pueden corregir una palabra mal escrita.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If the word is not found in the OpenOffice dictionary, proposes the closest words
IF SpellAvailable() = True AND SpellCheck(Word) = False THEN
	ListSugg is string = SpellSuggest(Word) 
	Info(Word + "? Don't you want to say <" + CR + ListSugg + CR + "> instead?" )
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SpellSuggest(<Word to Check>)`
---

**`<Result>: (Cadena de caracteres)`**

Puede corresponder a: 

- La lista de palabras correctas cerca de la palabra a verificar. La lista tiene el siguiente formato: 
	
	```txt
	<Correct Word 1> +  CR + <Correct Word 2>  + CR + ...
	```


- Una cadena vacía (""") si no se ha producido ningún error o si no hay ninguna proposición disponible. Si ocurrió un error, [ErrorInfo](../WDLang1/3013008.md) se usa para averiguar el motivo del Error( si el diccionario de OpenOffice no está instalado, por ejemplo). 




**`<Word to Check>: (Cadena de caracteres)`**

Palabra que debe ser revisada por el corrector ortográfico de OpenOffice. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Lista de palabras sugeridas
<a name="lista_palabras_sugeridas_ELTPARAGRAPHE000040"></a>

La lista de palabras sugeridas corresponde a la lista propuesta por el diccionario de OpenOffice cuando se realiza un clic derecho sobre una palabra mal escrita (en un control Campo de entrada que propone el corrector ortográfico, por ejemplo).).  


### Comprobación de la presencia del diccionario de OpenOffice
<a name="comprobacion_presencia_del_diccionario_openoffice_ELTPARAGRAPHE000045"></a>

Para saber si el corrector ortográfico de OpenOffice está instalado en el ordenador actual, utilice [HechizoDisponible](../WDLang1/1000019019.md).


## sComputeCrc32( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la comprobación de redundancia cíclica (CRC) del búfer.

**Observación**: El CRC es equivalente a una firma de búfer. Cualquier modificación realizada en el buffer cambia el valor CRC. El uso de CRC permite detectar las posibles modificaciones de archivos durante una transferencia a través de la red.. Puede comparar el CRC del archivo antes y después de la transmisión. Si son idénticos, es probable que el archivo no haya sido modificado durante la transferencia.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculates the CRC of a string
CRC is int
CRC = sComputeCrc32(MyString)

// Transmits the string and the CRC

// Check the integrity of string
If sComputeCrc32(MyString) <> CRC THEN Info("Modified string")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = sComputeCrc32(<Buffer> [, <Partial CRC>])`
---

**`<Result>: (Integro)`**

Valor control de la cadena (CRC)

**`<Buffer>: (Buffer o cadena de caracteres)`**

Cadena o buffer utilizado para calcular el valor de control (CRC)

**`<Partial CRC>: (Entero opcional)`**

Resultado de un cálculo anterior de CRC. Este parámetro se utiliza para calcular el CRC de dos tampones en un fila: no es necesario concatenar los búferes. Este parámetro corresponde a 0 (se gestiona un solo búfer) por Default.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no se tiene en cuenta. Se gestiona un único búfer.

## Observaciones
**sComputeCrc32** es idéntico a [sComputeCrc16](../WDLang1/3062007.md), pero es más rápido y eficiente.

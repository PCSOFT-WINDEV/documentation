
## ServiceControl( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Envía un mensaje de control a un servicio. Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Pauses the service found in TABLE_Service
IF ServiceControl(COL_Name[TABLE_Service], serviceCmdPause) = False THEN
	Error(ErrorInfo(errMessage))
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceControl(<Service name> , <Command to send> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se envió el comando al administrador de servicios,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a comprobar.

**`<Command to send>: (Constante entera o entero)`**

Comando que será enviado al servicio:


|   |   |
| --- | --- |
| *serviceCmdPause* | Pone en pausa el servicio (equivalente a [ServicePause](../WDLang1/1000017109.md)). |
| *serviceCmdRereadConfig* | Pide al servicio que relea sus ajustes de configuración (equivalente a [ServicioActualización](../WDLang1/1000017107.md)). |
| *serviceCmdRestart* | Reinicia un servicio (equivalente a [ServicioContinuar](../WDLang1/1000017105.md)). |
| *serviceCmdStop* | Detiene el servicio (equivalente a [ServiceStop](../WDLang1/1000017111.md)). |


También tiene la posibilidad de utilizar un número entero incluido entre 128 y 255. Este rango de valores está destinado a los comandos definidos por el propio servicio (véase la documentación sobre el servicio para más detalles).

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para control un servicio en otro ordenador. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía, el servicio se gestiona localmente..

## Observaciones




### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000099"></a>

La función **ServiceControl** envía un comando al administrador de servicios. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).




## TimelineAddEvent(Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un Event en un control Línea de tiempo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a 5-second event into a TimeLine control configured to the second
MyTrack is string
MyTitle is string
EvtStart is int
EvtEnd is int 

MyTrack = "Strings"
MyTitle = "Start"
EvtStart = 4
EvtEnd = 9

TimelineAddEvent(TL_Music, MyTrack, MyTitle, EvtStart, EvtEnd)
```
<a name="Example2"></a>

```wl
TimelineDeleteAll(TL_Robots)
Evt is EventTimeline

TimelineAddTrack(TL_Robots, "Robot 1")
Evt.Track = "Robot 1"
Evt.Title = "Start"
Evt.Start = 10
Evt.End = 150
Evt.BackgroundColor = LightGreen
TimelineAddEvent(TL_Robots, Evt)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadir un elemento especificando sus características

`<Result> = TimelineAddEvent(<TimeLine control> , <Track> , <Title> , <Start> [, <End> [, <Category>]])`
---

**`<Result>: (Integro)`**



- Índice de la nueva Event,

- -1 si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**Observación**: Puede manipular el nuevo Event directamente con su índice, utilizando la siguiente notación: &lt;Control Línea de tiempo&gt;[&lt;Índice&gt;]. Puede modificar las propiedades de este elemento.

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo en el que se añadirá el Event.

**`<Track>: (Cadena de caracteres)`**

Nombre de la pista asociada con el Event. 

**`<Title>: (Cadena de caracteres)`**

Título de la Event. 

**`<Start>: (Integro)`**

Inicio de Event. Este parámetro se expresa en la unidad elegida para la resolución de pantalla de control Línea de tiempo. Si la resolución de la pantalla es la segunda, este parámetro corresponderá a la hora de inicio de la Event expresada en segundos. 

Observación: La resolución de la pantalla se puede modificar: 

- en la ventana de control Línea de tiempo description (pestaña "Detalles"). 

- con la función [TimelineChangeMode](../WDLang1/1000020397.md). 




**`<End>: (Entero opcional)`**

Fin de Event. Este parámetro se expresa en la unidad elegida para la resolución de pantalla de control Línea de tiempo.
Si no se especifica este parámetro, la duración por defecto de Event será set a 1 unidad.

**`<Category>: (Cadena de caracteres opcional)`**

Nombre de la categoría asociada a la Event. Si no se especifica este parámetro, el Event es associated sin categoría. La categoría se crea automáticamente si no existe.


<a name="SYNTAX2"></a>

### Añadir un evento en una control Línea de tiempo utilizando el tipo EventTimeline

`<Result> = TimelineAddEvent(<TimeLine control> , <Event>)`
---

**`<Result>: (Entero opcional)`**



- Índice de la nueva Event,

- -1 si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**Observación**: Puede manipular el nuevo Event directamente con su índice, utilizando la siguiente notación: &lt;Nombre del control Línea de tiempo&gt;[&lt;Índice&gt;]. Puede modificar las propiedades de este elemento. 

**`<TimeLine control>: (Nombre del control)`**

Nombre de control Línea de tiempo en el que se añadirá el Event. 

**`<Event>: (EventTimeline Variable)`**

Nombre de la variable [EventTimeline](../WDLang1/1000020612.md) que describe las características de la Event a añadir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La función **TimelineAddEvent** permite asociar un evento a una sola pista. Para agregar el mismo evento a varias pistas, se debe llamar a la función **TimelineAddEvent** para cada pista.

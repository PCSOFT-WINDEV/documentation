


## Funciones de gestión de continentes y países
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de gestión de continentes y países son las siguientes:



|   |   |
| --- | --- |
| [ContinentGet](../WDLang1/1000024933.md) | Devuelve la variable de tipo [Continent](../WDLang1/1000024919.md) que corresponde a uno de los siguientes valores: <br><br>- una constante de continente (código de continente de 2 caracteres), <br><br>- un nombre de continente.<br><br><br> |
| [ContinentList](../WDLang1/1000024932.md) | Devuelve la lista de continentes. |
| [CountryGet](../WDLang1/1000024931.md) | Devuelve la variable de tipo Country que corresponde a un código ISO o al nombre de un país según la norma ISO 3166-1. |
| [CountryList](../WDLang1/1000024930.md) | Devuelve la lista de todos los países o países de un continente determinado según la norma ISO 3166-1. |



Estas funciones manipulan los siguientes tipos de variables: 



|   |   |
| --- | --- |
| [Continent](../WDLang1/1000024919.md) | El tipo **Continent** permite conocer las características de un continente. |
| [Country](../WDLang1/1000024921.md) | El tipo **Country** permite conocer las características de un país. |

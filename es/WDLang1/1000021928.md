
## Constantes para gestionar la modificación de ventanas por parte del usuario
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por propiedades/funciones |
| --- | --- | --- |
| **weAddRepositionableNote** | Autorización para modificar: El usuario final puede añadir una nota reposicionable en la ventana. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weAddWire** | Autorización para modificar: El usuario final puede añadir un cable entre varios controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weAll** | Autorización para modificar: Todas las modificaciones están permitidas. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weChangeTooltip** | Autorización para modificar: El usuario final puede modificar los consejos de herramientas de los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weChangeWire** | Autorización para modificar: El usuario final puede modificar los cables entre los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weModeDifferential** | Tipo de backup a realizar: Guardar todos los cambios, incluyendo las acciones personalizadas (función [WinEdDo](../WDLang1/1000021897.md)). Todas estas operaciones se reproducirán durante la operación de recarga (como en el caso de un registro).. | [WinEdSave](../WDLang1/1000021848.md) |
| **weModeFull** | Tipo de backup a realizar: Guardar todos los controles y sus propiedades actuales. Esta constante sólo puede ser usada en una ventana interna. | [WinEdSave](../WDLang1/1000021848.md) |
| **weModeReposionableNote** | Tipo de backup a realizar: Guardar las notas reposicionables creadas sólo por el usuario. Esta constante equivale a la función "Exportar las notas" del menú contextual de las notas reposicionables (AAF). | [WinEdSave](../WDLang1/1000021848.md) |
| **weMoveControl** | Autorización para modificar: El usuario final puede mover los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weMultiSelection** | Autorización para modificar: El usuario final puede seleccionar varios controles al mismo tiempo. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weNone** | Autorización para modificar: No se permite ninguna modificación. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **wePosition** | Autorización para modificar: El usuario final puede mover los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weResizeControl** | Autorización para modificar: El usuario final puede cambiar el tamaño de los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weSelection** | Autorización para modificar: El usuario final puede seleccionar los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |
| **weSize** | Autorización para modificar: El usuario final puede modificar el tamaño de los controles. | [WinEditPermitido](../Proprietes/1000021850.md), [WinEditMode](../Proprietes/1000021824.md) |


## edtHTMLSelection (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **edtHTMLSelection** se utiliza para identificar todas las características avanzadas de una selección en un Editor HTML control. Las características de esta selección son devueltas por varias propiedades de WLanguage. 

Las variables **edtHTMLSelection** se utilizan en los siguientes casos: 

- "Modificar la selección" Event associated con el editor HTML control. 

- función [HTMLEditorGetSelection](../WDLang1/1000025992.md). Esta función se utiliza para recuperar la selección actual en un editor HTML control. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Selección](../Proprietes/1000023033.md) Property utilizado en un editor HTML control. 




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gray out the "Open" button (or not) according to the selection
HTMLEditorGetSelection(HTMEDT_MyHTMLEditor, HTMLEditorGetSelection_Callback)
INTERNAL PROCEDURE HTMLEditorGetSelection_Callback(Selection is edtHtmlSelection)
   	BTN_Open.Grayed = NOT Left(selection.Text, "https:\\" )
END
```

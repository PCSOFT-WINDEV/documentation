


## &lt;Barra de herramientas&gt;.dock (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Acopla una barra de herramientas control a una ventana o cambia la posición de anclaje de una barra de herramientas acoplada control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TBAR_Toolbar1.Dock(barUp)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Toolbar control>.Dock([<Docked side> [, <X> , <Y>]])`
---

**`<Toolbar control>: (Nombre del control)`**

Nombre de la barra de herramientas control a utilizar.

**`<Docked side>: (Constante opcional)`**

Barra de herramientas control lado de acoplamiento:


|   |   |
| --- | --- |
| *barDown* | La barra de herramientas control está acoplada en la parte inferior de la ventana principal. |
| *barLeft* | La barra de herramientas control está acoplada a la izquierda de la ventana principal. |
| *barRight* | La barra de herramientas control está acoplada a la derecha de la ventana principal. |
| *barUp*<br>(Valor predeterminado) | La barra de herramientas control está acoplada en la parte superior de la ventana principal. |



**`<X>: (Entero opcional)`**

Coordenada X de la esquina superior izquierda de la barra de herramientas control. Esta coordenada X es igual a 0 por Default. 
Si se usa la constante *barLeft*, &lt;X&gt; se pasa sólo para información. La barra de herramientas control siempre está acoplada a la izquierda o a la derecha. Si se acoplan varias barras de herramientas control, la barra de herramientas especificada control se inserta lo más cerca posible de la posición &lt;Y&gt; especificada.

**`<Y>: (Entero opcional)`**

Coordenada Y de la Corner superior izquierda de la barra de herramientas. Esta coordenada Y es igual a 0 por Default. 
Si se usa la constante *barUp*, &lt;Y&gt; se pasa sólo para información. La barra de herramientas control siempre está acoplada en la parte superior o inferior. Si se acoplan varias barras de herramientas control, la barra de herramientas especificada control se inserta lo más cerca posible de la posición &lt;Y&gt; especificada.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La palabra clave [MiVentana](../Motscles/1511004.md) no debe utilizarse: 

- en los eventos associated con la barra de herramientas control. 

- en los eventos associated con los controles en la barra de herramientas.

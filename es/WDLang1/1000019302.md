


## Funciones de gestión de certificados
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular los certificados de Windows:



|   |   |
| --- | --- |
| [CertificateCheckDuplicate](../WDLang1/1000019306.md) | Comprueba la firma de un duplicado. |
| [CertificateCheckFileSignatureBuffer](../WDLang1/1000019500.md) | Verifica la correspondencia entre una firma y un fichero.. La firma fue almacenada como un Buffer Variable. |
| [CertificateCheckFileSignatureFile](../WDLang1/1000019499.md) | Verifica la correspondencia entre una firma y un fichero.. La firma se mantuvo como un archivo externo. |
| [CertificateCheckString](../WDLang1/1000019298.md) | Verifica la correspondencia entre una firma y un string. |
| [CertificateExtract](../WDLang1/1000019300.md) | Extrae una Certificate de un buffer de firmas o de un duplicado de impresión. |
| [CertificateExtractExtensionValue](../WDLang1/1410087221.md) | Devuelve el valor de una Certificate extension. |
| [CertificateList](../WDLang1/1000019293.md) | Devuelve la lista de certificados disponibles en el equipo. |
| [CertificateLoad](../WDLang1/1000020851.md) | Carga una Certificate desde un archivo o un buffer. |
| [CertificateOpenProperties](../WDLang1/1000019301.md) | Abre la ventana de Windows que se usa para averiguar las propiedades de un Certificate. |
| [CertificateSelect](../WDLang1/1000019295.md) | Abre una ventana para seleccionar los certificados. Los certificados listados son los certificados instalados en la tienda personal. |
| [CertificateSignExecutable](../WDLang1/1000020560.md) | Firma un archivo ejecutable con un Certificate. |
| [CertificateSignFile](../WDLang1/1000019297.md) | Crea la firma de un archivo. Esta firma puede ser almacenada en un Buffer Variable o en un archivo de texto. |
| [CertificateSignString](../WDLang1/1000019296.md) | Crea la firma de un string. |
| [TrustedCertificateAdd](../WDLang1/1000024320.md) | Añade una Certificate a la lista de autoridades raíz de confianza utilizadas para validar las cadenas de certificación. |
| [TrustedCertificateReset](../WDLang1/1000024321.md) | Elimina todos los certificados añadidos por [TrustedCertificateAdd](../WDLang1/1000024320.md). |

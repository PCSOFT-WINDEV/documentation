


## MenuMark( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Coloca la marca de verificación ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Marque.gif) delante de la opción de menú (menú emergente o menú desplegable).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display a checkmark in front of "OPT_Option1" found in "PopupMenu1"
MenuMark(OPT_Option1)
// Equivalent to: MenuMark(PopupMenu1.OPT_Option1)
```
<a name="Example2"></a>

```wl
// Display a checkmark in front of "OPT_IconBar"
MenuMark(OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuMark([<Window>, ] <Menu option>)`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú a marcar. 
Si no se especifica este parámetro, **MenuMark** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a marcar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe ser especificado sin comillas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Mostrando una marca de verificación en una opción associated con un Image
<a name="mostrando_una_marca_verificacion_una_opcion_associated_con_image_ELTPARAGRAPHE000050"></a>

Si una marca de verificación es associated con una opción de menú que contiene un Image, el Image no es reemplazado por la marca de verificación.
<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000057"></a>

Esta función es equivalente a la [Comprobado](../Proprietes/2510019.md) Property.

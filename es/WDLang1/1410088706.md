
## RoundToMultiple (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un valor numérico redondeado al múltiplo más cercano de otro valor numérico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
RoundToMultiple(10, 9)
// returns 9 

RoundToMultiple(-10, -3)
// Returns -9

RoundToMultiple(1.3, 0.2)
// Returns 1.4

RoundToMultiple(2.64, 0.25)
// Returns 2.75
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RoundToMultiple(<Numeric value> , <Significance>)`
---

**`<Result>: (Variant)`**

Valor de &lt;Numeric value&gt; redondeado al múltiplo más cercano de &lt;Significance&gt;.

**`<Numeric value>: (Numérico)`**

Valor numérico a redondear.

**`<Significance>: (Numérico)`**

Múltiplo hacia el que se redondeará el número.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El redondeo con el tipo "Real" no es preciso. Las operaciones con el tipo "Real" no son precisas debido a la representación de los números reales en la memoria. Para obtener un redondeo exacto, se recomienda utilizar los tipos Currency o Numeric, que utilizan una representación exacta en la memoria. Para obtener más información, consulte [el tipo Real](../Motscles/1514048.md).

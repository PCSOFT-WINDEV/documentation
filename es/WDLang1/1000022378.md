


## &lt;Cámara&gt;.SetDriver (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Conecta la "Cámara" control a un controlador específico para la captura de vídeo. 

**Atención**: 

- Para utilizar esta función, se debe conectar una "cámara" al ordenador actual.

- Para conocer la lista de controladores de captura de vídeo instalados en el ordenador actual, utilice [VideoListDriver](../WDLang1/3079008.md) (asociado a la constante *viDirectShow*). Para determinar qué conductor está actualmente asociado a un control Cámara, utilice el [Controlador](../Proprietes/2510061.md) Property.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the list of available video drivers
// in the LIST_VideoDrivers control
LIST_VideoDriver.Add(VideoListDriver(viDirectShow))
// Uses the second video driver
IF CAM_Camera1.SetDriver(LIST_VideoDrivers[2]) = False THEN
	Error("Unable to change driver: " + CR + ...
	ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Camera control>.SetDriver(<Identifier of video driver>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se modificó el controlador de video asociado al control,

- <u><u><u><u>False</u></u></u></u> si se produce un problema. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<Identifier of video driver>: (Cadena de caracteres)`**

Identificador del controlador de vídeo a utilizar. Este identificador tiene el siguiente formato:


```txt
<Driver name> + TAB + <Driver version>
```


Este identificador corresponde al formato de lista devuelto por [VideoListDriver](../WDLang1/3079008.md) (asociado a la constante *viDirectShow*).


## SysMetric( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la resolución de un elemento específico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Width of the cursor
ResMetric = SysMetric(metCxCursor)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysMetric(<Element>)`
---

**`<Result>: (Integro)`**

Resolución del elemento especificado.

**`<Element>: (Constante de tipo Integer)`**

Especifica el elemento para el que se debe definir la resolución:


|   |   |
| --- | --- |
| *metCxBorder* | Ancho del borde fijo de la ventana. |
| *metCxCursor* | Ancho del cursor. |
| *metCxDoubleClk* | Ancho del rectángulo que rodea la ubicación del primer clic durante un doble clic. El segundo clic debe producirse en la misma área para que el sistema detecte un doble clic. |
| *metCxFixedFrame* | Ancho del borde cuando el estilo de la ventana es WS_DLGFRAME. |
| *metCxFullScreen* | Ancho del área de clientes en una ventana de pantalla completa. |
| *metCxHScroll* | Ancho de flecha que se encuentra en una barra de desplazamiento horizontal. |
| *metCxIcon* | Anchura de los iconos. |
| *metCxIconSpacing* | Anchura de los rectángulos utilizados para posicionar los iconos uno al lado del otro. |
| *metCxMin* | Ancho mínimo de ventana. |
| *metCxMinTrack* | Ancho mínimo cuando el usuario cambia el tamaño de una ventana. |
| *metCxScreen* | Ancho de la pantalla. |
| *metCxSize* | Ancho de los botones que se encuentran en la barra de título de una ventana. |
| *metCxSizeFrame* | Ancho del borde de la ventana que puede ser modificado. |
| *metCxVScroll* | Ancho de la Image de la flecha que se encuentra en la barra de desplazamiento vertical. |
| *metCxVThumb* | Ancho de la caja (en pulgadas) en la barra de desplazamiento horizontal. |
| *metCyBorder* | Altura del borde fijo de la ventana. |
| *metCyCaption* | Altura del título de la ventana. Esta altura corresponde a la altura del título más la altura del borde de la ventana que no se puede redimensionar. |
| *metCyCursor* | Altura del cursor. |
| *metCyDoubleClk* | Altura del rectángulo que rodea la ubicación del primer clic durante un doble clic. El segundo clic debe producirse en la misma área para que el sistema detecte un doble clic. |
| *metCyFixedFrame* | La altura del borde cuando el estilo de la ventana es WS_DLGFRAME. |
| *metCyFullScreen* | Altura del área de cliente en una ventana de pantalla completa (equivalente a la altura de la pantalla menos la altura de la barra de título). |
| *metCyHScroll* | Altura de la flecha que se encuentra en una barra de desplazamiento horizontal. |
| *metCyIcon* | Altura de los iconos. |
| *metCyIconSpacing* | Altura de los rectángulos utilizados para posicionar los iconos uno al lado del otro. |
| *metCyKanji* | Altura de la ventana "Kanji". |
| *metCyMenu* | Altura de la barra para el menú desplegable. Esta altura corresponde a la altura del menú más la altura del borde de la ventana que no se puede redimensionar. |
| *metCyMin* | Altura mínima de la ventana. |
| *metCyMinTrack* | Altura mínima cuando el usuario cambia el tamaño de una ventana. |
| *metCyScreen* | Altura de la pantalla. |
| *metCySize* | Altura de las imágenes que se encuentran en la barra de título. |
| *metCySizeFrame* | Altura del borde de la ventana modificable. |
| *metCyVScroll* | Ancho (en píxeles) de las barras de desplazamiento verticales. |
| *metCyVThumb* | Altura de la caja (en pulgadas) en la barra de desplazamiento horizontal. |
| *metDBCSEnabled* | Diferente de cero si la versión actual de Windows utiliza caracteres codificados en 2 bits; de lo contrario, devuelve cero. |
| *metMenuDropAlignment* | Menú para alinear "Popups".. Si este valor es igual a cero, el lado izquierdo del menú contextual se alinea con el lado izquierdo del objeto correspondiente en la barra de menús. Si este valor es diferente de cero, el lado izquierdo del menú "popup" se alinea con el lado derecho del objeto correspondiente en la barra de menús. |
| *metMousePresent* | Diferente de cero si el controlador del ratón está instalado. |
| *metPenWindows* | Maneje (identificador) de "Pen Windows" DLL si este manipular está instalado. |
| *metSwapButton* | Diferente de cero si los botones izquierdo y derecho del ratón están invertidos. |

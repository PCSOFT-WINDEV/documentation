
## HaspHLInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la llave electrónica Hasp.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace("Information about the key:")
Trace(HaspHLInfo(EDT_Session_ID, haspInfoKey))
Trace("Information about the session:")
Trace(HaspHLInfo(EDT_Session_ID, haspInfoSession))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HaspHLInfo(<Session identifier> , <Information> [, <File identifier>])`
---

**`<Result>: (Cuerda o entero (según la información solicitada))`**

Información recuperada.

**`<Session identifier>: (Integro)`**

Identificador de la sesión Hasp, devuelto por [HaspHLLogin](../WDLang1/1000018891.md).

**`<Information>: (Constante de tipo Integer)`**

Información por recuperar:


|   |   |
| --- | --- |
| *haspInfoFileSize* | &lt;Resultado&gt; es un número entero correspondiente al tamaño (en bytes) del archivo cuyo identificador corresponde a &lt;Identificador de archivo&gt;. |
| *haspInfoKey* | &lt;Resultado&gt; es una cadena de caracteres que contiene la información sobre la clave electrónica en formato XML. |
| *haspInfoSession* | &lt;Resultado&gt; es una cadena de caracteres que contiene la información sobre la sesión Hasp en formato XML. |



**`<File identifier>: (Entero opcional o constante entera opcional)`**

Identificador del archivo cuyo tamaño debe calcularse. Este identificador puede corresponder a:

- un entero. 

- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *haspLicenseFile* | Fichero"Área de licencia" de la clave. |
| *haspMainFile* | Fichero"Área de memoria principal" de la llave. |

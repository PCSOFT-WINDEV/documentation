
## Funciones para la gestión de los controles de la barra de herramientas (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para manipular la barra de herramientas control mediante la sintaxis de prefijos:



|   |   |
| --- | --- |
| [&lt;Barra de herramientas&gt;.desacoplar](../WDLang1/3020008.md) | Transforma una barra de herramientas acoplada control en una barra de herramientas flotante control. |
| [&lt;Barra de herramientas&gt;.dock](../WDLang1/3020006.md) | Acopla una barra de herramientas control a una ventana o cambia la posición de anclaje de una barra de herramientas acoplada control. |
| [&lt;Barra de herramientas&gt;.posición](../WDLang1/3020007.md) | Indica el modo de visualización de una barra de herramientas control. |


## EmulateInput (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Realiza una entrada por emulación en un control mientras comprueba si este control es un control editable.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on EDT_TypeLogin edit control
EmulateMouse(EDT_TypeLogin, emLeftClick, 3, 10)

// Type "MARK" in this control
EmulateInput(EDT_TypeLogin, "Mark", eiShift)

// Click on BTN_ValidateButton
EmulateMouse(EDT_ValidateButton, emLeftClick, 5, 5)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EmulateInput(<Control name> , <Text to type> [, <Additional keys>])`
---

**`<Control name>: (Cadena de caracteres)`**

Nombre de control Campo de entrada donde &lt;Text to type&gt; será escrito.
**Observación**: Para realizar una entrada con la función **EmulateInput**, el control &lt;Nombre del control&gt; debe estar activo, estar en modo de entrada y tener el foco al realizar la llamada.

**`<Text to type>: (Cadena de caracteres)`**

Texto a escribir en el &lt;control nombre&gt; control.

**`<Additional keys>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Indica las teclas del teclado que se deben pulsar durante la entrada.


|   |   |
| --- | --- |
| *eiAlt* | Presione la tecla Alt. <br><br> |
| *eiControl* | Presione la tecla Ctrl. <br><br> |
| *eiShift* | Pulsar la tecla Mayúsculas. <br><br> |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Limitaciones de los controles utilizados
<a name="limitaciones_los_controles_utilizados_ELTPARAGRAPHE000072"></a>

La función **EmulateInput** no es compatible con los siguientes controles:

- control Imagen.

- Control Barra de progreso.

- Control Estático.

- Control OLE.

- Control Forma, 

- Control HTML,

- Control Cámara Web.

- Control Conferencia.

- Control Código de barras.

- Control Ventana interna.

- control Gráfico.




**Atención**: **EmulateInput** no funciona en una sesión TSE no interactiva. 
<a name="NOTE0_2"></a>




### Enviando una set de llaves
<a name="enviando_una_set_llaves_ELTPARAGRAPHE000098"></a>

Algunas claves pueden no ser tenidas en cuenta cuando se envía un set de claves y cuando el control utilizado contiene llamadas a [Multitask](../WDLang1/3015004.md) en su código de modificación.

En este caso, es preferible dividir la serie de teclas en varias llamadas a la función **EmulateInput**.


<a name="NOTE0_3"></a>

| AYUDA | {AYUDA} | INICIO | {HOME} |

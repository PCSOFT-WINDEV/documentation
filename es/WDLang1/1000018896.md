
## HaspHLRead( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Lee algunos datos (cadenas de caracteres o búferes) en la memoria interna de la tecla Hasp.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieves the initial activation key in the memory of the Hasp key
InitialKey is string 
InitialKey = HaspHLRead(EDT_Session_ID, haspMainFile, 0, 30)

// Compares the initial key to the activation key that was received
KeyCompareKey(InitialKey, ActivationKey)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HaspHLRead(<Session identifier> , <File identifier> , <Position> , <Size>)`
---

**`<Result>: (Cadena de caracteres)`**

Información leída en la memoria de la clave Hasp.

**`<Session identifier>: (Integro)`**

Identificador de la sesión Hasp, devuelto por [HaspHLLogin](../WDLang1/1000018891.md).

**`<File identifier>: (Entero o constante de tipo Integer)`**

Identificador del fichero en el que deben leerse los datos. Este identificador puede corresponder a:

- un entero. 

- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *haspLicenseFile* | Lee en el archivo "área de licencia" de la clave. |
| *haspMainFile* | Lee en el archivo "área de memoria principal" de la llave. |






**`<Position>: (Integro)`**

Posición (en bytes) en la que se leerán los datos en el fichero.

**Atención**: El inicio del archivo comienza en la posición 0.

**`<Size>: (Integro)`**

Tamaño (en bytes) de los datos a leer.

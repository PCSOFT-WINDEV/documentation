
## MaskFormat (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el formato de una cadena de caracteres utilizando una variable de tipo [InputMask](../WDLang1/1000024886.md). La cadena de caracteres tiene el mismo formato que se mostraría en un control Campo de entrada que utiliza una máscara.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyMask is InputMask = MaskPhoneNumber("FR")
Trace(MaskFormat(MyMask, "+33467032032"))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = MaskFormat(<Mask to apply> , <String>)`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena con el formato de la máscara especificada.

- Cadena vacía ("") si se produce un error. 




**`<Mask to apply>: (Variable de tipo InputMask)`**

Nombre de la variable de tipo [InputMask](../WDLang1/1000024886.md) que se utilizará para definir el formato de la cadena de caracteres. 

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres cuyo formato se definirá. si este parámetro es no es válido, &lt;Resultado&gt; corresponde a una cadena vacía ("").


## Constantes de serialización
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las constantes de serialización son las siguientes:

| Constante | Descripción | Función |
| --- | --- | --- |
| **psdBinary** | Serialización binaria. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |
| **psdBinaryFormat16** | Serialización binaria en formato compatible con la versión 16. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |
| **psdJSON** | Serialización JSON. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |
| **psdFormatting** | Serialización JSON con nuevas líneas y sangría. | [Serialize](../WDLang1/3013065.md)<br>[VariantToJSON](../WDLang1/1000021051.md) |
| **psdXML** | Serialización XML con referencia a subobjetos. Este tipo de serialización permite utilizar el formato XML como medio de almacenamiento e intercambio entre aplicaciones escritas en WLanguage. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |
| **psdXML_FormerFormat** | Hasta la versión 100050, la constante psdXML no generaba un archivo XML válido. El formato generado produjo problemas en los dispositivos móviles. Este problema fue solucionado.<br><br>Un archivo XML creado con una versión posterior a la versión 100050 no puede ser leído por un programa compilado con una versión anterior. Para mantener el modo de funcionamiento de la versión 100050, utilice la constante psdXML_FormerFormat. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |
| **psdXMLAggregated** | Serialización XML con agregación directa de subobjetos. Este tipo de serialización permite generar fácilmente un archivo XML en un formato estándar para realizar intercambios con otros sistemas. | [Deserialize](../WDLang1/3013066.md)<br>[Serialize](../WDLang1/3013065.md) |

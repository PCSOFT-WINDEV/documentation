


## SchedulerAddResource (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un nuevo Resource en un control Planificador.



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds a resource (a meeting room) into the Scheduler control
// Meeting is the data file that describes the available meeting rooms
SchedulerAddResource(SCH_RoomOccupancy, Meeting.Name + gStoredValue(Meeting.MeetingID))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SchedulerAddResource(<Scheduler control> , <Resource> [, <Image>])`
---

**`<Result>: (Integro)`**



- Índice de agregado Resource,

- -1 si se produce un error. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador en el que se añadirá el Resource. Este control puede corresponder a: 

- el control Planificador de una ventana, 

- la control Planificador de un Page,

- la control Planificador de un reporte.




**`<Resource>: (Cadena de caracteres)`**

Nombre del agregado Resource. Un valor Invisible puede ser añadido por [gStoredValue](../WDLang1/1410088363.md).

**`<Image>: (Cadena de caracteres opcional)`**

Ruta y nombre de la imagen de fondo para el título de la Resource. Esta imagen se muestra debajo del encabezado de la fila o columna que contiene el nombre de la Resource.
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro no está disponible.

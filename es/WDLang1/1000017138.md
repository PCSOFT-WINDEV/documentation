
## dbgLogStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve y modifica el status del log de tiempo de ejecución.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Starts the runtime log
dbgEnableLog()
...
// Pauses the runtime log
dbgLogStatus(LogPause)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el estado del registro de tiempo de ejecución

`<Result> = dbgLogStatus()`
---

**`<Result>: (Constante de tipo Integer)`**

Status actual del log de tiempo de ejecución:


|   |   |
| --- | --- |
| *LogDisabled* | El log de tiempo de ejecución no está habilitado.<br><br> |
| *LogEnabled* | El log de tiempo de ejecución está activado. |
| *LogPause* | El log de tiempo de ejecución se pausa. |




<a name="SYNTAX2"></a>

### Modificación del estado del registro de tiempo de ejecución

`dbgLogStatus(<Former status> , <New status>)`
---

**`<Former status>: (Constante de tipo Integer)`**

Estado de la actividad del registro en tiempo de ejecución antes de los cambios realizados por **dbgLogStatus**.


|   |   |
| --- | --- |
| *LogDisabled* | El log de tiempo de ejecución no está habilitado.<br><br> |
| *LogEnabled* | El log de tiempo de ejecución está activado. |
| *LogPause* | El log de tiempo de ejecución se pausa. |



**`<New status>: (Constante de tipo Integer)`**

Nuevo status de log de tiempo de ejecución:


|   |   |
| --- | --- |
| *LogDisabled* | Detiene permanentemente el registro de tiempo de ejecución. Para volver a activarlo, tendrá que utilizar [dbgEnableLog](../WDLang1/1000017137.md).<br><br> |
| *LogEnabled* | Activa el tiempo de ejecución Log( después de una pausa). |
| *LogPause* | Detiene temporalmente el log de tiempo de ejecución. |




## Funciones de compresión / encriptación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el cifrado y la compresión:



|   |   |
| --- | --- |
| [Codificar](../WDLang1/1000022258.md) | Codifica una cadena de caracteres o un buffer. Para decodificar el resultado, simplemente use [Decodificar](../WDLang1/1000022259.md). |
| [Comprimir](../WDLang1/3024012.md) | Comprime una cadena o un bloque de memoria (buffer) en formato binario. |
| [Decodificar](../WDLang1/1000022259.md) | Decodifica una cadena de caracteres que fue codificada por [Encode](../WDLang1/1000022258.md). |
| [DecryptAsymmetric](../WDLang1/1000024050.md) | Descifra un buffer con un algoritmo de clave privada/pública según el cifrado RSA. |
| [DecryptAsymmetricEnveloped](../WDLang1/1000024058.md) | Realiza el cifrado asimétrico de un sobre que ha sido cifrado con [EncrypteAsymmetricEnveloped](../WDLang1/1000024057.md). |
| [DecryptStandard](../WDLang1/1000021294.md) | Descifra un mensaje cifrado con un algoritmo de cifrado simétrico (AES, DES, etc.). |
| [Descifrar](../WDLang1/3024035.md) | Descifra una cadena de caracteres que fue encriptada por [Encriptar](../WDLang1/3024014.md). |
| [Descomprimir](../WDLang1/3024009.md) | Descomprime un bloque de memoria (buffer) que fue comprimido por [Comprimir](../WDLang1/3024012.md). |
| [Encriptar](../WDLang1/3024014.md) | Cifra una cadena de caracteres en formato binario o en formato ASCII. |
| [EncryptAsymmetric](../WDLang1/1000024049.md) | Cifra un buffer utilizando un algoritmo de cifrado asimétrico (RSA) que requiere una clave pública y una clave privada. |
| [EncryptAsymmetricEnveloped](../WDLang1/1000024057.md) | Realiza una encriptación asimétrica de un sobre. |
| [EncryptGenerateRSAKey](../WDLang1/1000024052.md) | Genera una pareja clave privada/clave pública RSA. |
| [EncryptStandard](../WDLang1/1000021293.md) | Cifra una cadena de caracteres o un búfer binario utilizando un algoritmo de cifrado simétrico (AES, DES, etc.). |
| [LuhnCalculate](../WDLang1/1410087276.md) | Calcula el siguiente dígito de control de Luhn para una secuencia dada. |
| [LuhnCheck](../WDLang1/1410087277.md) | Comprueba que la secuencia dada es válida según la fórmula de Luhn. |
| [UUDecode](../WDLang1/1000018864.md) | Decodifica una cadena de caracteres en formato UUEncode. |
| [UUEncode](../WDLang1/1000018862.md) | Codifica una cadena de caracteres según el algoritmo UUEncode. |


## NumToFinancialWritingChinese (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un valor numérico (entero, real o monetario) en una cadena en formato de escritura financiera china.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NumToFinancialWritingChinese(123.00)        // Returns "????????	??"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NumToFinancialWritingChinese(<Number>)`
---

**`<Result>: (Cadena de caracteres)`**

Número expresado como una cadena de caracteres en formato de escritura financiera china. 
Si el parámetro &lt;Número&gt; es una cadena, la cadena devuelta es la misma que la cadena inicial.

**`<Number>: (Entero, real o monetario)`**

Valor numérico a convertir en una cadena de caracteres en formato de escritura financiera china.
Observación: Si este parámetro es negativo, la función devuelve una cadena vacía ("").

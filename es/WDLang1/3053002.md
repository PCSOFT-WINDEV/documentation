
## MemAdd( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20</B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [Associative arrays](../Motscles/1514058.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade un elemento a una zona de memoria.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Adds the value of the Customer.Name item to "MemArea"
// The returned value will be the record number of this item
MemAdd("MemArea", Customer.Name, HRecNum(Customer))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MemAdd(<Memory area> , <Added value> , <Return value>)`
---

**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria, definida por [MemCreate](../WDLang1/3053005.md).

**`<Added value>: (Cadena de caracteres, entera o real)`**

Valor añadido a la zona de memoria (tamaño máximo: 65521 octetos). Este valor corresponde al argumento de ordenación utilizado por [MemSort](../WDLang1/3053020.md). Este valor es devuelto por [MemKeyVal](../WDLang1/3053016.md).

**`<Return value>: (Cadena de caracteres)`**

Valor que devolverá [MemRetrieve](../WDLang1/3053012.md) o [MemSetPosition](../WDLang1/3053018.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Existencia de zona de memoria
<a name="existencia_zona_memoria_ELTPARAGRAPHE000225"></a>

No se comprueba la existencia de la zona de memoria.

Si la zona de memoria especificada no existe, se produce un error WLanguage cuando se utiliza **MemAdd**. 

Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).
<a name="NOTE0_2"></a>




### Clasificación de los elementos
<a name="clasificacion_los_elementos_ELTPARAGRAPHE000099"></a>

El elemento añadido a la zona de memoria no está ordenado. Para ordenar toda la zona de memoria, utilice [MemSort](../WDLang1/3053020.md).

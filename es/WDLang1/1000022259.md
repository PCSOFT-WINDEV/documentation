
## Decodificar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Decodifica una cadena de caracteres que fue codificada por [Codifica](../WDLang1/1000022258.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Encode an image in base64
Res is string = Encode(fLoadBuffer("MyImage.jpg"), encodeBASE64)

// ...

// Decode the image
DecodedImage is Image = Decode(Res, encodeBASE64)
```



```wl
// Decode a buffer received in an HTTP request
bufRead is Buffer = HTTPResponse.Content
<COMPILE IF ConfigurationUNICODE>
 	bufResult = Decode(AnsiToUnicode(bufRead), encodeBASE64) 
<ELSE>
 	bufResult = Decode(bufRead, encodeBASE64) 
<END>
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Decode(<Element to decode> [, <Encoding/decoding format>])`
---

**`<Result>: (Cadena de caracteres o memoria intermedia)`**

Resultado de la descodificación.

**`<Element to decode>: (Cadena de caracteres)`**

Datos a decodificar. Estos datos fueron codificados por [Codifica](../WDLang1/1000022258.md).
Atención: La cadena a decodificar debe tener el mismo formato que las cadenas de Default del proyecto: ANSI o UNICODE.

**`<Encoding/decoding format>: (Constante de tipo Integer)`**

Especifica el tipo de codificación. Este parámetro debe ser el mismo que se utilizó en la codificación.


|   |   |
| --- | --- |
| *encodeBASE64*<br>(Valor predeterminado) | Formato BASE 64 (cumple con el estándar RFC 4648). El texto codificado contiene sólo caracteres imprimibles y CRs (Carriage Returns).<br><br> |
| *encodeBASE64NoCR* | Formato BASE 64 (cumple con el estándar RFC 4648) sin RCs (Carriage Returns). El texto codificado sólo contiene caracteres imprimibles.<br><br> |
| *encodeBASE64URL* | Formato BASE 64 URL (cumple con el estándar RFC 4648). El texto codificado sólo contiene caracteres imprimibles.<br><br> |
| *encodeBASE85* | Formato BASE 85. El texto codificado sólo contiene caracteres imprimibles.<br><br><br><br> |
| *encodePCS* | Formato ASCII. El texto codificado sólo contiene caracteres imprimibles.<br><br><br><br> |
| *encodeURLToAnsi* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena ANSI. Esta constante corresponde al la constante *encodeURLFromAnsi* utilizada en la función [Codifica](../WDLang1/1000022258.md). <br><br> |
| *encodeURLToUnicode* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena Unicode. Esta constante corresponde al la constante *encodeURLFromUnicode* utilizada en la función [Codifica](../WDLang1/1000022258.md). <br><br> |
| *encodeURLToUTF8* | URL format. El texto codificado solo contiene caracteres permitidos en una URL. El texto decodificado es una cadena UTF-8. Esta constante corresponde al la constante *encodeURLFromUTF8* utilizada en la función [Codifica](../WDLang1/1000022258.md). <br><br> |
| *encodeUUEncode* | Formato UUEncode. El texto codificado sólo contiene caracteres imprimibles.<br><br> |

<a name="NOTE0_1"></a>

## Observaciones


### Formato de las cadenas de caracteres a decodificar
<a name="formato_las_cadenas_caracteres_decodificar_ELTPARAGRAPHE000165"></a>

La cadena a decodificar debe tener el mismo formato que las cadenas de Default de la configuración actual del proyecto: ANSI o UNICODE.

Para conocer o modificar el formato de la cadena de Default: 

1. Abra la ventana de descripción de la configuración actual: en la pestaña "Proyecto", en el grupo "Configuración del proyecto", haga clic en "Configuración actual". 

2. La pestaña "Unicode" le permite saber qué tipo de cadenas de caracteres utiliza Default.

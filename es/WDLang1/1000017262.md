


## MenuAddOption (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva opción después de la última opción de un menú desplegable o contextual. Esta opción de menú ejecuta un lenguaje WLanguage Procedure. Para insertar una opción de menú en un lugar específico, utilice la función [MenuInsertOption](../WDLang1/1000017265.md).

**Recordatorio**: El nombre del menú principal se define: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en la ventana de descripción de la ventana.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Adds a new menu to MENU_MyMenu
MenuAddMenu(MENU_MyMenu, "OPT_Schedule", "Schedule")
// Adds an option to the new OPT_Schedule menu
MenuAddOption("OPT_Schedule", "OPT_Create", "Create", Create_Schedule)
// Associates an image with the OPT_Create menu option
{"OPT_Create", indControl}..Image = "schedule.png"
// Add a separator
MenuAddSeparator("OPT_Schedule")
// Adds a new option (OPT_View)
MenuAddOption("OPT_Schedule", "OPT_View", "View", View_schedule)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MenuAddOption(<Source menu> , <Option to create> , <Caption> , <WLanguage procedure> [, <Procedure parameter>])`
---

**`<Result>: (Entero)`**

Posición de la opción en el menú. Si no se agrega la opción, se produce un error fatal.

**`<Source menu>: (Nombre del menú)`**

Nombre del menú al que debe agregarse la opción de menú. La opción se añadirá después de la última opción de este menú.
Observación: Este nombre puede corresponder a un menú contextual. 

**`<Option to create>: (Cadena de caracteres)`**

Nombre de la opción de menú a agregar. Este nombre se utilizará para manipular la opción de menú mediante programación. 
Si existe una opción con el mismo nombre, se produce un error fatal.

**`<Caption>: (Cadena de caracteres)`**

Texto de la nueva opción de menú. Este texto aparecerá en la ventana y permitirá a los usuarios seleccionar la opción.

**`<WLanguage procedure>: (Nombre de procedimiento)`**

Nombre del procedimiento WLanguage al que se llama cuando se selecciona la opción de menú.

**`<Procedure parameter>: (Variant opcional)`**

Parámetro que se puede pasar a &lt;WLanguage procedure&gt; cuando el usuario selecciona la opción de menú.

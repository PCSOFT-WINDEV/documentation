
## ExePriority( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la prioridad de una aplicación actualmente en ejecución.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the application to use
nIndex is int = TableSelect(TABLE_Table1)
IF nIndex <> -1 THEN
	// Priority of the selected application?
	i is int = ExePriority(TABLE_Table1.PID[nIndex])
	SWITCH i
		CASE exeLowPriority
			Info("Low priority")
		CASE exeHighPriority
			Info("High priority")
		CASE exeBelowNormalPriority
			Info("Priority less than standard")
		CASE exeNormalPriority
			Info("Standard priority")
		CASE exeAboveNormalPriority
			Info("Priority greater than standard")
		CASE exeRealTimePriority
			Info("Real-time priority")
	END 
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificación de la prioridad de una solicitud que se esté ejecutando actualmente

`ExePriority(<Application identifier> , <New priority>)`
---

**`<Application identifier>: (Constante de tipo Integer)`**

Identificador de la aplicación a utilizar. Este identificador es devuelto por [Proceso ExeList](../WDLang1/3035008.md).

**`<New priority>: (Constante de tipo Integer)`**

Nueva prioridad para la aplicación especificada:


|   |   |
| --- | --- |
| *exeAboveNormalPriority* | Prioridad superior a *exeNormalPriority* e inferior a *exeRealTimePriority*.<br>Esta constante está disponible para Windows 2000 y XP. |
| *exeBelowNormalPriority* | Prioridad superior a *exeLowPriority* y menor a *exeNormalPriority*<br>Esta constante está disponible para Windows 2000 y XP. |
| *exeHighPriority* | Prioridad alta . |
| *exeLowPriority* | Prioridad baja. |
| *exeNormalPriority* | Prioridad superior a *exeBelowNormalPriority* e inferior a *exeAboveNormalPriority* |




<a name="SYNTAX2"></a>

### Averiguar la prioridad de una solicitud que se esté ejecutando actualmente

`<Result> = ExePriority(<Application identifier>)`
---

**`<Result>: (Constante de tipo Integer)`**

Prioridad de la aplicación especificada:


|   |   |
| --- | --- |
| *exeAboveNormalPriority* | Prioridad superior a *exeNormalPriority* e inferior a *exeRealTimePriority*.<br>Esta constante está disponible para Windows 2000 y XP. |
| *exeBelowNormalPriority* | Prioridad superior a *exeLowPriority* y menor a *exeNormalPriority*.<br>Esta constante está disponible para Windows 2000 y XP. |
| *exeHighPriority* | Prioridad alta |
| *exeLowPriority* | Prioridad baja |
| *exeNormalPriority* | Prioridad superior a *exeBelowNormalPriority* e inferior a *exeAboveNormalPriority* |
| *exeRealTimePriority* | Prioridad en tiempo real. |



**`<Application identifier>: (Constante de tipo Integer)`**

Identificador de la aplicación a utilizar. Este identificador es devuelto por [Proceso ExeList](../WDLang1/3035008.md).

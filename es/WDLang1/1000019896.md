
## LineToPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición del primer carácter de una línea en un bloque de texto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize a multiline string
s is string = [
AA
BB
CC
]

// the function returns 1
LineToPosition(s, 1)
// the function returns 5
LineToPosition(s, 2)
// the function returns 9
LineToPosition(s, 3)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LineToPosition(<String> , <Line>)`
---

**`<Result>: (Entero)`**

Posición del primer carácter en la línea especificada. 

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a manipular. 

**`<Line>: (Entero)`**

Número de la línea buscada. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El separador de líneas es un retorno de carro (CR). Corresponde a 2 caracteres. 

- Los caracteres que constituyen el separador de línea pertenecen a la línea anterior.

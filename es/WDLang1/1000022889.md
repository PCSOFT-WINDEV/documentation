


## &lt;Organizador&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Retornos para una posición determinada en la control Agenda (coordenadas de un punto en la control Agenda): 

- el índice de la Appointment para la posición especificada. 

- la fecha correspondiente a la posición especificada. 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the date located at cursor position 
// in the Organizer control
dDate is Date
dDate = ORG_MyOrganizer.InfoXY(oriDateTime, MouseXPos(), MouseYPos())
// The hovered date becomes the first date displayed by the organizer
ORG_MyOrganizer.Position(dDate)
```

```wl
// Display the location of the appointment in the rollover tooltip
// Mouse rollover code (WM_MOUSEMOVE) of Organizer control:
nAppointmentIndex is int
nAppointmentIndex = ORG_Organizer.InfoXY(oriAppointment, ...
		MouseXPos(), MouseYPos())
IF nAppointmentIndex <> -1 THEN
	// An appointment is under the mouse cursor ...
	ORG_ORGANIZER.ToolTip = "Appointment at " + ORG_ORGANIZER[nAppointmentIndex].Location
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Organizer control>.InfoXY(<Type of information> , <X> , <Y>)`
---

**`<Result>: (Entero, cadena de caracteres o Fecha Variable)`**



- Información solicitada. 

- Una cadena vacía ("") si se utiliza la constante *oriDateTime* y si no se encuentra ninguna fecha en las coordenadas especificadas.

- -1 si se utiliza la constante *oriAppointment* y si no se encuentra ninguna cita en las coordenadas especificadas.




**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda a utilizar (se encuentra en una ventana)

**`<Type of information>: (Constante o combinación de constantes de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *oriAppointment* | Índice de la Appointment en las coordenadas especificadas. |
| *oriDateTime* | La fecha y la hora de Appointment corresponden a las coordenadas especificadas. |
| *oriScreenOrigin* | Por Default, (0,0) corresponde a la parte superior izquierda Corner del control Agenda utilizado (las barras de desplazamiento se encuentran en el origen).<br>Si se combina la constante *oriScreenOrigin* con las otras constantes, el punto (0,0) corresponde al origen de la pantalla.<br>Esta constante no puede utilizarse por sí sola. |



**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. Esta coordenada es relativa al control Agenda (si no se especifica la constante *oriScreenOrigin*).

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada es relativa al control Agenda (si no se especifica la constante *oriScreenOrigin*).

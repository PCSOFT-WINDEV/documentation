


## Posición Multimedia (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la posición actual del archivo multimedia que se está reproduciendo en la control Multimedia.







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
MyFile is string = "C:\My music\mysong.mp3"
MultimediaPlay(MM_Multimedia1, MyFile)
// Positions 10 seconds after the beginning of file
MultimediaPosition(MM_Multimedia1, 10000) 
MultimediaStop(MM_Multimedia1)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar la posición en el archivo

`<Result> = MultimediaPosition(<Multimedia control>)`
---

**`<Result>: (Integro)`**

Posición actual en el fichero, expresada en milisegundos.

**`<Multimedia control>: (Nombre del control)`**

Nombre del Multimedia control que se va a utilizar.  


<a name="SYNTAX2"></a>

### Modificando la posición en el archivo

`<Result> = MultimediaPosition(<Multimedia control> , <New position>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el puesto era set,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.

**`<New position>: (Entero o Duración)`**

Nueva posición en el archivo multimedia, expresada en milisegundos. Este parámetro puede corresponder a: 

- un número entero que corresponde al número de milisegundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo, 1 s o 10 ms).

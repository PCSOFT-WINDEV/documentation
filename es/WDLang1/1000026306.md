


## &lt;Editor de HTML&gt;.Reemplazar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Sustituye una cadena por otra en un editor HTML control. 

Observación: Por Default, la búsqueda ignora el formato de la cadena.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Replace WINDEV with WEBDEV
HTMEDT_MyHTMLEditor.Replace("WINDEV", "WEBDEV", htmlWholeWord + htmlCaseSensitive)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Editor control>.Replace(<Text to replace> , <Replacement text> [, <Options>])`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor HTML utilizado por control.

**`<Text to replace>: (Nombre del control)`**

Texto para buscar y reemplazar en el Editor HTML control.

**`<Replacement text>: (Cadena de caracteres)`**

Cadena de caracteres que reemplazará &lt;Text to replace&gt;&gt;. Si este parámetro corresponde a una cadena vacía ("""), se eliminarán todas las ocurrencias de &lt;Text to replace&gt;.

**`<Options>: (Constante o combinación de constantes opcional)`**

Opciones de búsqueda para el texto a reemplazar:


|   |   |
| --- | --- |
| *htmlCaseSensitive* | Realiza una búsqueda sensible a mayúsculas y minúsculas. |
| *htmlWholeWord* | Busque la palabra completa (encerrada en caracteres de puntuación o en caracteres espaciados). |


Por Default, la búsqueda no distingue entre mayúsculas y minúsculas.

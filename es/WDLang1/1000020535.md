
## ControlFindHighlight( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Destaca, en una ventana o control, todas las leyendas y textos correspondientes a un texto dado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Whenever EDT_Search is modified
ControlFindHighlight(WIN_Search, EDT_Search)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ControlFindHighlight(<Element> , <Searched text> [, <Search options> [, <Highlight color>]])`
---

**`<Result>: (Integro)`**



- Número de ocurrencias encontradas en la control o ventana. 

- 0 si ningún elemento contiene el texto buscado. 




**`<Element>: (Cadena de caracteres)`**

Nombre del elemento donde se realizará la búsqueda. Este elemento puede corresponder a: 

- un nombre de ventana. La búsqueda se realiza en el título de los controles, el texto de los botones, el texto de los controles de edición, el texto de las filas de tabla, ... La búsqueda se realiza en todos los controles visibles que no tienen ningún padre de Invisible. 

- un nombre de control. Si este elemento corresponde a un control Cinta de opciones, la búsqueda se realiza en todos los controles de la cinta.. 




**`<Searched text>: (Cadena de caracteres)`**



- Texto que debe encontrarse en el elemento especificado. 

- "" (cadena vacía) a Cancel la búsqueda. 




**`<Search options>: (Constante opcional de tipo Integer)`**

Indica las opciones de búsqueda:


|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos |
| *WholeWord* | Busca una palabra completa (entre caracteres de puntuación o espacios) |
| 0 | Busque teniendo en cuenta el caso (mayúsculas / minúsculas) en un texto |


Ninguna opción es seleccionada por Default: la búsqueda es sensible al caso y se realiza sobre palabras incompletas. 

**`<Highlight color>: (Entero o constante (opcional))`**

Color usado para resaltar las ocurrencias encontradas. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**Observación**: El Color usado por Default es amarillo claro. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se puede realizar una sola búsqueda en un momento dado en un control o en una ventana. 

- La búsqueda no se realiza en los siguientes controles: 

	- HTML control y control Campo de entrada en formato HTML. 

	- Control ActiveX.

	- Zona no mostrada en los controles Looper, List Box y Table.

	- Lista, tabla o ventana emergente mostrada por los combo boxes.




## &lt;Tabla&gt;.RecNum (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la Record actual en una control Tabla basada en un archivo de datos.

**Observación**: La función **&lt;Tabla&gt;.RecNum** puede utilizarse en un control Tabla de selección única o múltiple.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Current record number in the "TABLE_ProductTable" control
ResRecNum = TABLE_TableProduct.RecNum()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Table control>.RecNum()`
---

**`<Result>: (Integro)`**



- El número actual de Record en el archivo de datos relacionado al control Tabla,

- 0 si no hay ningún fila actual o si el fila actual no está guardado todavía en el archivo de datos relacionado al control Tabla.




**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si la consulta se ejecutó con la constante *hConFiltro* (opción marcada en el editor de consultas), **&lt;Tabla&gt;.RecNum** devuelve el número del Record actual en la consulta.

- Si la consulta se ejecutó sin la constante *hWithFilter*, **&lt;Tabla&gt;.RecNum** devuelve un resultado indefinido.

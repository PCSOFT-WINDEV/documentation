
## ChangeKeyboard( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obtiene o establece el idioma associated con el teclado. Esta función permite gestionar los teclados que utilizan conjuntos de caracteres específicos.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switches the display language of the application to Greek
Nation(nationGreek) 
// Switches the fonts to Greek
ChangeCharset(charsetGreek) 

// Changes the keyboard
ChangeKeyboard(languageGreek)

// Redisplays the "WIN_Window1" window
Use(WIN_Window1)
```


<a name="Example2"></a>



```wl
// Switches the display language of the application to Arabic
Nation(nationArabic) 
// Switches the fonts to Arabic
ChangeCharset(charsetArabic) 

// Changes the keyboard
// (sub-language: Egyptian = 3)
ChangeKeyboard(languageArabic, 3)

// Redisplays the "WIN_Window1" window
Use(WIN_Window1)
```

## Observaciones


### Cambiar el teclado
<a name="cambiar_teclado_ELTPARAGRAPHE000468"></a>

- **ChangeKeyboard** se tiene en cuenta inmediatamente. El idioma del teclado se puede cambiar mientras se escribe.

- **ChangeKeyboard** sólo puede utilizarse si el teclado está instalado en el ordenador actual.



<a name="NOTE0_2"></a>




### Teclado utilizado por Default
<a name="teclado_utilizado_por_default_ELTPARAGRAPHE000481"></a>

Si varios teclados correspondientes al mismo idioma están instalados en el ordenador actual y si no se especifica &lt;Sub-lenguaje&gt; (Argelino y Egipcio para árabe por ejemplo), WINDEV utiliza el primer teclado encontrado.

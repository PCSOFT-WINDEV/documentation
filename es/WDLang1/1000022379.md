


## &lt;Configurar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite configurar el controlador de captura de vídeo. Esta configuración puede ser realizada por el usuario (a través de un cuadro de diálogo) o mediante programación (configuración de la resolución).

Atención: Esta función sólo se puede usar con los controles de la cámara en modo compatible ("Use the former 'Video for Windows' APIs (compatibility)" comprobado en el control description). 

Observación: Esta función es equivalente a la función [RoomConfigurations](../WDLang1/1000022452.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the standard dialog box for configuring the driver for video capture 
// used for the "CAM_MyVideo" control
CAM_MyVideo.Configure()
```
<a name="Example2"></a>

```wl
// Specifies a resolution of 640 by 480
CAM_VIDEO1.Configure(viResolution, 640, 480)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Camera control>.Configures([<Type of configuration> [, <Width resolution> , <Height resolution>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se muestra el cuadro de diálogo estándar,

- <u><u><u><u>False</u></u></u></u> si se produce un error. El mensaje de error correspondiente es devuelto por [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

**`<Type of configuration>: (Constante opcional de tipo Integer)`**

Tipo de configuración a realizar:


|   |   |
| --- | --- |
| *viConfigureFormat* | Mostrar un cuadro de diálogo (si existe) que permita al usuario especificar la resolución de la cámara, el número de colores utilizados, etc. |
| *viConfigureSource*<br>(valor predeterminado) | Mostrar un cuadro de diálogo (si existe) que permite al usuario especificar el nombre de la cámara a utilizar y los parámetros de esta cámara (saturación, por ejemplo)..<br>Este cuadro de diálogo depende del controlador utilizado por control Cámara. |
| *viResolution* | Permite especificar una resolución para la cámara mediante los parámetros &lt;Resolución horizontal&gt; y &lt;Resolución vertical&gt;. |



**`<Width resolution>: (Entero opcional)`**

Ancho de resolución de la cámara (en píxeles). Este parámetro debe especificarse si &lt;Tipo de configuración&gt; vale *viResolution*.

**`<Height resolution>: (Entero opcional)`**

Altura de la resolución de la cámara (en píxeles). Este parámetro debe especificarse si &lt;Tipo de configuración&gt; vale *viResolution*.

## Observaciones


### Modo de funcionamiento de la control Cámara durante la visualización de las ventanas de diálogo
<a name="modo_funcionamiento_control_camara_durante_visualizacion_las_ventanas_dialogo_ELTPARAGRAPHE000231"></a>

- Si se muestra el panel para configurar el formato, el control Cámara se congela.

- Si el panel de configuración es mostrado por la constante *viConfigureSource*, el control Cámara continúa operando. Las modificaciones de los parámetros se tienen en cuenta inmediatamente y se aplican a la control Cámara en "direct live".

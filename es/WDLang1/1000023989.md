


## PanelCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una nueva control Panel acoplable en la ventana actual. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a configuration with 2 panels on the left in a pane
PanelCreate("", dopaLeft, IW_PRODUCTLIST)
PanelCreate("", dopaLeft, IW_DETAILS)
```

```wl
// Create a Dockable Panel control
myPanelcontrol is Control
myPanelcontrol <- PanelCreate("", dopaLeft, IW_PRODUCTLIST)
```

```wl
// Create an empty Dockable Panel control
p is Control <- PanelCreate("p", dopaLeft)
p.Width = 200
// Adds a control in the Dockable panel control
b is Control <- ControlCreate(p.FullName+ ".b", typButton, 10, 20, 100, 24)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PanelCreate(<Title> , <Position> [, <Internal window> [, <Parameter 1> [... [, <Parameter N>]]]])`
---

**`<Result>: (Cadena de caracteres)`**

Panel acoplable creado.  

**`<Title>: (Cadena de caracteres)`**

Título del Panel acoplable creado. Si este parámetro corresponde a una cadena vacía (""), un título es automáticamente associated con el control creado. 

**`<Position>: (Constante de cuerda opcional (o combinación de constantes))`**

Describe la posición de la control Panel acoplable.: 


|   |   |
| --- | --- |
| *dopaBottom* | Fijación de la control Panel acoplable en el fondo. |
| *dopaFloating* | Crea una control Panel acoplable flotante. <br><br> |
| *dopaLeft* | Conecta la control Panel acoplable a la izquierda. |
| *dopaRight* | Conecta la control Panel acoplable a la derecha. |
| *dopaTop* | Conecta la control Panel acoplable a la parte superior. |


Las constantes se pueden combinar.
Si la posición ya está ocupada, se añade el control Panel acoplable al área y se crea y muestra automáticamente un panel.. 

**`<Internal window>: (Nombre de ventana)`**

Nombre de la ventana interna a utilizar en el panel acoplable. El tamaño de la control Panel acoplable corresponde al tamaño de la ventana en el editor. 
Si no se especifica este parámetro, se crea un panel acoplable vacío.. 

**`<Parameter 1>: (Tipo de parámetro (opcional))`**

Primer parámetro a pasar a la ventana interna. Este parámetro se pasa a la Event "Declaraciones globales" de la ventana interna. 



**`<Parameter N>: (Tipo de parámetro (opcional))`**

Enésimo parámetro a pasar a la ventana interna. Este parámetro se pasa a la Event "Declaraciones globales" de la ventana interna.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **PanelCreate** provoca un error no fatal si la posición no es válida.

- **Parámetros pasados a la ventana interna**: Los parámetros se recuperan en la Event "Declaraciones globales" de la ventana interna. Simplemente escriba la siguiente línea de código al inicio del evento:
	
	```txt
	PROCEDURE <Window name>(<Parameter1> [, <Parameter2> [, ...]])
	```

	Para obtener más información, consulte [Ventana con parámetros](../WDChamp/1010026.md).

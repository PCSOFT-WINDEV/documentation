


## mlzDelete (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un fila dinámico de un control Zona multilínea. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on [delete]
nRow is int = MZ_Network.Value
mlzDelete(MZ_Network, nRow)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`mlzDelete(<Multiline Zone control> [, <Position>])`
---

**`<Multiline Zone control>: (Nombre del control)`**

Nombre de la control Zona multilínea a utilizar.

**`<Position>: (Entero opcional)`**

Índice del fila a suprimir. 

- Si no se especifica este parámetro, el borrado se realizará en la fila actual (si es una fila dinámica).. Si no se selecciona ninguna fila dinámica, la función **mlzDelete** provoca un error fatal.

- Si &lt;Posición&gt; es superior al número de filas, la función **mlzDelete** provoca un error fatal.

- El índice debe coincidir con el índice de una fila dinámica (añadida por [mlzAddLine](../WDLang1/1000020199.md) o [mlzInsertLine](../WDLang1/1000020207.md)). Si el control Zona multilínea contiene filas estáticas y dinámicas, se debe tener en cuenta el número de filas estáticas.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El tamaño de la control Zona multilínea se modifica para tener en cuenta la fila eliminada: la control se reduce desde el fondo. Si los controles están situados debajo de la control Zona multilínea, se mueven hacia arriba.. La altura de la ventana de useful se reduce en consecuencia.

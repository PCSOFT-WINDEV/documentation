


## &lt;DisplayPreviousPeriod (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Desplaza el área de tiempo mostrada en el área anterior (semanas anteriores, por ejemplo) en un control Planificador. El control Planificador se visualiza a partir del día correspondiente al área horaria anterior.. 

**Observación**: Las funciones **&lt;DisplayPreviousPeriod** y [&lt;DisplayNextPeriod](../WDLang1/1000022902.md) permiten establecer fácilmente los botones "Adelante/Atrás". Estos controles le permiten desplazarse por las áreas de tiempo que muestra la pantalla de control Planificador. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the [PREVIOUS] Button control
SCH_RoomScheduler.DisplayPreviousPeriod()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Scheduler control>.DisplayPreviousPeriod()`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

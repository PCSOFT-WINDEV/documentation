


## Funciones de gestión de UMC
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Para gestionar el UMC se utilizan las siguientes funciones: 



|   |   |
| --- | --- |
| [InUMCMode](../WDLang1/1000017027.md) | Indica si el código se ejecuta desde un código escrito en un Macro-Código de Usuario. |
| [UMCActive](../WDLang1/3030020.md) | Permite conocer y modificar la activación de la gestión UMC (User Macro-Code). |
| [UMCOpen](../WDLang1/3030021.md) | Abre la ventana de edición de Código de macro de usuario para la ventana actual. |




## NationToLanguage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el idioma correspondiente a una nación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the language corresponding to the Italian nation
ResLanguage is int
ResLanguage = NationToLanguage(nationItalian) 
SWITCH ResLanguage
	CASE languageGerman:
	 ...
	CASE languageItalian:
	 ...
END
```

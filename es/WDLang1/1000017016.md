
## SysListVideoCard( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de las tarjetas de vídeo conectadas al ordenador actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the list of video cards installed on the current PC
ListAdd(LIST_Screen, SysListVideoCard())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysListVideoCard()`
---

**`<Result>: (Cadena de caracteres)`**

Lista de tarjetas de vídeo instaladas en el ordenador actual, cadena vacía (""") si se ha producido un error. [ErrorInfo](../WDLang1/3013008.md) devuelve más información sobre el error.

Esta lista tiene el siguiente formato:


```txt
<Name of Screen 1> + TAB + <Name of Video Card 1> + CR + 
<Name of Screen 2> + TAB + <Name of Video Card 2> + CR + ... 
+ CR + <Name of Screen N> + TAB + <Name of Video Card N>
```


Por ejemplo: 


```txt
\\.\DISPLAY1<TAB>NVIDIA GeForce 210 <\r><\n>\\.\DISPLAY2<TAB>NVIDIA GeForce 210
```

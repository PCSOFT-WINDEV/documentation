


## &lt;Chrono&gt;.Pause (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pone en pausa un cronómetro para medir el tiempo de un proceso. El cronometraje se puede reiniciar con la función [&lt;Chrono&gt;.Resume](../WDLang1/1000023792.md).






<a name="sample_code"></a>

## Ejemplo

<a name="Example1"></a>


![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
ProcessDuration is Duration 
MyChrono is Chrono 
MyChrono.Start()
MyFunction1()
ProcessDuration = MyChrono.Pause()
Info("The MyFunction1 process lasted " + ProcessDuration)
MyChrono.Resume()
MyFunction2()
ProcessDuration = MyChrono.End()
Info("The MyFunction1 and MyFunction2 processes lasted " + ProcessDuration)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Stopwatch>.Pause()`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) para el cronómetro especificado.

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000300"></a>

La función **&lt;Chrono&gt;.Pause** no tiene ningún efecto si se utiliza en un cronómetro en pausa.
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000312"></a>

Si no se llama a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) antes de la función **&lt;Chrono&gt;.Pause**, se produce un error WLanguage.
<a name="NOTE0_3"></a>


### Hilos
<a name="hilos_ELTPARAGRAPHE000329"></a>

Las funciones de cronometraje (Chronoxxx) relacionadas con el mismo cronómetro deben utilizarse en el mismo hilo.




## dDominantImageColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el color dominante (es decir, el más frecuente) de una imagen que se encuentra en una variable de tipo [Image](../WDLang1/1000019650.md) o en un control Imagen.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyColor is Color = dDominantImageColor(IMG_MyImage)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = dDominantImageColor(<Image>)`
---

**`<Result>: (Entero)`**

Color dominante de la imagen. 

**`<Image>: (Cadena de caracteres o variable de tipo Image)`**

Imagen a analizar. Este parámetro puede corresponder a:

- el nombre de un control Imagen,

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los pixeles transparentes o semitransparentes de la imagen se ignoran.

- La función **dDominantImageColor** no puede utilizarse en imágenes vectoriales (SVG, PDF, etc.).

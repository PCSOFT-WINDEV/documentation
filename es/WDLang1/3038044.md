
## Uso (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una ventana WINDEV y cierra todas las demás ventanas que se abrieron previamente.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Open the "WIN_EditCustomer" window and close
// all other previously opened windows
Use(WIN_EditCustomer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Use(<Window name>)`
---

**`<Window name>: (Cadena de caracteres)`**

Nombre de la ventana a abrir.

Por defecto, la ventana se muestra según la posición definida en el editor. Se puede definir la posición en la que se mostrará la ventana. El nombre de la ventana se completa con las coordenadas de visualización de la ventana al abrirse. Este parámetro tiene el siguiente formato: "&lt;Nombre de la ventana&gt;, &lt;Fila&gt;, &lt;Columna&gt;" donde:

- &lt;Nombre de ventana&gt;: nombre de la ventana que se va a abrir.

- &lt;Fila&gt;: coordenada horizontal de visualización de la ventana (en píxeles, en relación con la Corner superior izquierda de la pantalla o de la ventana principal).

- &lt;Columna&gt;: coordenada vertical de visualización de la ventana (en píxeles, relativa a la Corner superior izquierda de la pantalla o de la ventana principal).

## Observaciones
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Abrir la ventana
<a name="abrir_ventana_ELTPARAGRAPHE000073"></a>

- Las ventanas previamente abiertas se cierran. El código de cierre no se ejecuta.

- El código de inicialización del proyecto (si se está ejecutando) se interrumpe.

- Se muestra la ventana abierta y se activan sus controles.

- Se ejecuta la ventana de apertura de Process (si la hay).



<a name="NOTE0_2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Características de la ventana abierta
<a name="caracteristicas_ventana_abierta_ELTPARAGRAPHE000085"></a>

La ventana que hay que abrir no puede:

- devuelve un resultado (será ignorado),

- aceptar parámetros obligatorios.



<a name="NOTE0_3"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 

### Título de la ventana a abrir
<a name="titulo_ventana_abrir_ELTPARAGRAPHE000097"></a>

Por defecto, el título de la ventana es el definido en el editor.

Para modificar el título de la ventana, utilice las funciones [NextTitle](../WDLang1/3038043.md) o [CurrentTitle](../WDLang1/3038023.md).

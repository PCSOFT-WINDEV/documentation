


## CheckBoxAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva opción a una control Casilla de verificación. La opción se añade después de las opciones existentes.  




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add 2 options 
CheckBoxAdd(CBOX_Options, "Read a book on a tablet")
CheckBoxAdd(CBOX_Options, "Read a paperback")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CheckBoxAdd(<Check Box control> , <Caption> [, <Sub-caption> [, <Image>]])`
---

**`<Result>: (Integro)`**



- Índice de la nueva opción. 

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Check Box control>: (Nombre del control)`**

Nombre de control Casilla de verificación a la que se añadirá la opción. La opción se añade después de las opciones existentes.

**`<Caption>: (Cadena de caracteres)`**

Texto de la opción a añadir.

**`<Sub-caption>: (Cadena de caracteres opcional)`**

Subtexto de la opción a añadir. 
Si no se especifica este parámetro, la nueva opción no tiene subtexto. 
Si se especifica este parámetro, el subtexto se mostrará incluso si la casilla de verificación utiliza un modo de visualización sin subtexto.


**`<Image>: (Cadena de caracteres opcional)`**

Imagen associated con la opción de añadir. 
Si no se especifica este parámetro, la opción añadida no tiene Image.
Si se especifica este parámetro, la Image se mostrará sólo si la casilla de verificación está usando un modo de visualización "estilo Vista" (con Image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las características de la opción añadida también pueden modificarse mediante las siguientes propiedades utilizadas en la opción: 

	- [Pie de foto](../Proprietes/2510053.md) property para cambiar el texto. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) [SubCaption](../Proprietes/1000017232.md) property para cambiar el subtexto. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Image](../Proprietes/2510034.md) property para modificar el image. 




- Al añadir opciones en una control Casilla de verificación, el tamaño de la control no se modifica. Las opciones añadidas pueden no ser Visible si el control no es lo suficientemente grande. La control puede ampliarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Para añadir una opción en una posición específica, utilice [CheckBoxInsert](../WDLang1/1000020597.md).




## &lt;Procesamiento de palabras&gt;.redoblar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la última acción deshecha en una control Procesador de texto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Redoes the last undone action on the control
WP_MyDocument.Redo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.Redo()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la última acción deshecha ha sido restaurada, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.



<a name="XComponent"></a>

## Componente:
wd280mdl.dll

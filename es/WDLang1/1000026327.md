
## Constantes del editor HTML control
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones control del editor HTML utilizan las siguientes constantes:

| Constante | Detalles | Función/Tipo |
| --- | --- | --- |
| **edthtmlSelMultiple** | Tipo de selección: Se seleccionan varios elementos HTML. | Tipo [edtHTMLSelection](../WDLang1/1000026093.md) |
| **edthtmlSelSingle** | Tipo de selección: Sólo se selecciona un elemento HTML. | Tipo [edtHTMLSelection](../WDLang1/1000026093.md) |
| **edthtmlSelEmpty** | Tipo de selección: No se realiza ninguna selección. | Tipo [edtHTMLSelection](../WDLang1/1000026093.md) |
| **hegBody** | Usó para get el código HTML del cuerpo. | [HTMLEditorGet](../WDLang1/1000025981.md) |
| **hegTextWithoutFormat** | Se utiliza para get el texto contenido en la control sin las etiquetas HTML. | [HTMLEditorGet](../WDLang1/1000025981.md) |
| **hegValue** | Se utiliza para get el valor del control. | [HTMLEditorGet](../WDLang1/1000025981.md) |
| **emailOptionReplyAfter** | Opciones de visualización de Email: El contenido de la Email está sangrado. La respuesta se escribirá después del contenido. | [HTMLEditorFromEmail](../WDLang1/1000026172.md) |
| **emailOptionReplyBefore** | Opciones de visualización de Email: El contenido de la Email está sangrado. La respuesta se escribirá antes del contenido. | [HTMLEditorFromEmail](../WDLang1/1000026172.md) |
| **emailOptionForwarded** | Opciones de visualización de Email: Añade una cabecera de respuesta al principio de la Email. | [HTMLEditorFromEmail](../WDLang1/1000026172.md) |
| **htmlAlignment** | Alinea la selección horizontalmente. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlStrikeTrough** | Tacha la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlColor** | Cambia la Color de la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlBackgroundColor** | Cambia el fondo color de la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlCreateLink** | Crea un enlace a la URL especificada en el parámetro. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlSuperscript** | Aplica el formato de superíndice a la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlBold** | Hace la selección en negrita. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlSubscript** | Aplica el formato índice a la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlItalic** | Italiza la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlWholeWord** | Busca una palabra completa (entre caracteres de puntuación o espacios). | [HTMLEditorReplace](../WDLang1/1000025995.md) |
| **htmlFontName** | Cambia el nombre de la fuente utilizada para la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlNameCssStyle** | Modifica el estilo CSS usado por la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlCaseSensitive** | Realiza una búsqueda sensible a mayúsculas y minúsculas. | [HTMLEditorReplace](../WDLang1/1000025995.md) |
| **htmlUnderlined** | Subraya la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlDeleteLink** | Elimina el enlace de la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlFontSize** | Cambia el tamaño de la fuente utilizada en la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlFontSizeDecrease** | Reduce el tamaño de la letra de la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **htmlFontSizeIncrease** | Aumenta el tamaño de la letra de la selección. | [HTMLEditorFormatSelection](../WDLang1/1000025996.md) |
| **typehtmlOther** | Tipo de Resource para descargar: Tipo desconocido | " [Cargar los recursos](../WDChamp/1410087152.md)" Event del editor HTML control. |
| **typehtmlCssRes** | Tipo de Resource para descargar: estilo CSS | " [Cargar los recursos](../WDChamp/1410087152.md)" Event del editor HTML control. |
| **typehtmlImageRes** | Tipo de Resource para descargar: Image | " [Cargar los recursos](../WDChamp/1410087152.md)" Event del editor HTML control. |
| **typehtmlScriptRes** | Tipo de Resource para descargar: Script | " [Cargar los recursos](../WDChamp/1410087152.md)" Event del editor HTML control. |
| **typehtmlXMLHttpRequestRes** | Tipo de Resource para descargar: XML Document generado en respuesta a una petición HTTP | " [Cargar los recursos](../WDChamp/1410087152.md)" Event del editor HTML control. |

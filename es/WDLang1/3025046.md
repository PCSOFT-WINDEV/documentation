


## SaveParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda un valor persistente: 

- en el registro o en el archivo equivalente de la plataforma en tiempo de ejecución.

	- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En Universal Windows 10 App, los valores persistentes se guardan en los parámetros de la aplicación ("Isolated Storage Settings").

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) En una aplicación Linux, los valores persistentes se guardan en la carpeta $HOME/.local/share.




- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) en el archivo especificado por [InitParameter](../WDLang1/3025044.md).




Este valor puede ser leído por [LoadParameter](../WDLang1/3025047.md) la próxima vez que se inicie la aplicación.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save
SaveParameter("LOGIN", EDT_Edit1)

// Restore
EDT_Edit1 = LoadParameter("LOGIN")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SaveParameter(<Parameter name> , <Parameter value>)`
---

**`<Parameter name>: (Cadena de caracteres)`**

Nombre del parámetro a guardar. Este parámetro no debe corresponder a una cadena vacía.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Se puede especificar una ruta completa si este parámetro se ha guardado en el registro.. Los separadores aceptados son "...", " / " y "\\" ".. La propiedad [FullName](../Proprietes/2514005.md) devuelve el ruta completo de un control. Para obtener más información, consulte [los comentarios](#NOTE0_1).

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Si el modo de inicialización seleccionado es el registro, **SaveParameter** no distingue entre mayúsculas y minúsculas ni entre acentos (por ejemplo, "Parámetro" y "parámetro" son equivalentes).

**`<Parameter value>: (Cadena de caracteres o entero)`**

Valor a ahorrar.

## Observaciones


### Nombre del parámetro en un registro
<a name="nombre_del_parametro_registro_ELTPARAGRAPHE000129"></a>

- **Caso 1: el nombre del parámetro es "Param1".:**
	Este parámetro se guarda en el registro utilizando el siguiente ruta:
	HKEY_CURRENT_USER\\Software\\&lt;Company&gt;\\&lt;Application&gt;\\Param1

- **Caso 2: el nombre del parámetro es "WIN_Window1\\EDT_Edit2\\Param1".:**
	Este parámetro se guarda en el registro utilizando el siguiente ruta:
	HKEY_CURRENT_USER\\Software\\&lt;Company&gt;\\&lt;Application&gt;\\WIN_Window1\\EDT_Edit2\\Param1

- **Caso 3: el nombre del parámetro es "WIN_Window1.EDT_Edit2.Param1".:**
	Este parámetro se guarda en el registro utilizando el siguiente ruta:
	HKEY_CURRENT_USER\\Software\\&lt;Company&gt;\\&lt;Application&gt;\\WIN_Window1\\EDT_Edit2\\Param1




**Recordatorio**: Para obtener el registro ruta donde se almacena la información de la aplicación, utilice [ProjectInfo](../WDLang1/3064004.md) con la constante *piRegistry*.

&lt;Empresa&gt; corresponde a la empresa especificada al crear el ejecutable.. Si no se especifica este nombre, &lt;Compañía&gt; corresponde a "Aplicaciones WINDEV".

**Observación**: En el modo Test, el ruta utilizado en el registro es el siguiente: HKEY_CURRENT_USER\\SOFTWARE\\PC SOFT\\WinDev\\28.0\\TestMode\\&lt;project&gt;.

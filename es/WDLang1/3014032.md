
## ConfigureAAF( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite especificar el comportamiento de algunas características de la aplicación automatic (AAF). Por ejemplo, puede set la opción de exportación a Excel que utiliza [el menú automatic de las mesas](../Editeurs/9500102.md).

Observación: esta función es equivalente a la función [AAFConfigurar](../WDLang1/1000022108.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Configures the mode for exporting the content of Table control to Excel
ConfigureAAF(aafTabletoExcel, taWithLayout)

// With combination of options: 
ConfigureAAF(aafTableToExcel, taColumnsTitles + taColumnDisplayedOrder + taWithLayout)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ConfigureAAF(<Relevant AAF> , <Parameter>)`
---

**`<Result>: (Variant)`**

Valor anterior del ajuste de las Fuerzas Armadas Americanas.

**`<Relevant AAF>: (Constante entera o combinación de constantes enteras)`**

Constante correspondiente a la AAF para configurar: 



|   |   |
| --- | --- |
| *aafToastDurationRestoreFilter* | Configura la duración de visualización de la toast indicando la restauración de los filtros almacenados por el usuario en una control Tabla. <br>Recordatorio: El usuario puede almacenar los filtros definidos en una columna de control Tabla a través de "Guardar el filtro" en el menú contextual.<br><br>Elemento afectado: control Tabla |
| *aafPrintReportOnTable* | Configura la impresión de reporte en control Tabla. El formato de las celdas (colores, ...) definido en la control Tabla se almacena cuando se utiliza [TablaImprimir](../WDLang1/1000017237.md). <br><br>Elemento afectado: control Tabla |
| *aafInternetSearch* | Configura el motor utilizado para realizar la búsqueda en INTERNET. El motor de búsqueda utilizado por Default es Google. <br><br>Elemento afectado: <br><br>- Editar control<br><br>- Controles Tabla, Tabla TreeView, Hoja de cálculo y Tabla dinámica<br><br><br> |
| *aafDirectoryTableExport* | Configura el directorio Default donde se creará el archivo de exportación. |
| *aafColumnSelection* | Selección de columnas de Visible a través del menú contextual de control Tabla.<br><br>Elemento afectado: control Tabla |
| *aafTableToExcel* | Exporta el contenido de control Tabla o la selección realizada en el control a Excel.<br><br>Elemento afectado: control Tabla |
| *aafTableToWord* | Exporta el contenido de control Tabla o la selección realizada en el control a Word.<br><br>Elemento afectado: control Tabla |
| *aafTableToXML* | Exporta el contenido de control Tabla o la selección realizada en el control a XML.<br><br>Elemento afectado: control Tabla |



**`<Parameter>: (Cadena de caracteres o constante entera)`**

El escenario de la AAF. 

- **To configura la exportación a Excel, Word o XML**, utilizando las siguientes constantes: 
	


|   |   |
| --- | --- |
| *taWithInvisibleColumns* | También exporta las columnas de Invisible de control Tabla. |
| *taWithLayout* | Exporta el Color y la fuente de control Tabla. |
| *taColumnDisplayedOrder* | Exporta las columnas según el orden de las columnas que se muestran actualmente en el control Tabla (y no según el orden de las columnas definido en el editor). |
| *taMerge* | **Only si el archivo XLS ya existe.** <br><br>Fusiona el contenido existente con el nuevo contenido:<br><br>	- Las fórmulas, el formato, ... de las células existentes se almacenan.<br><br>	- Los datos existentes no se almacenan. Sólo se almacenan los datos de las columnas que se encuentran también en el archivo XLS existente.<br><br><br> |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). No exporta los totales. |
| *taNoTitle* | Sólo se copian los datos en el archivo de exportación. |
| *taNoTotal* | No exporta filas con totales, promedio o automático Count( estas filas son exportadas por Default). |
| *taColumnsTitles* | El título de las columnas se inserta en el archivo de exportación. |



- **To configurar la selección de columnas de Visible**, &lt;Parámetros&gt; es un número entero que indica el número de columnas del control Tabla de las que el menú contextual permanece visualizado después de una selección del usuario. El valor predeterminado de este parámetro es 4.

- **To configura el directorio de exportación**, &lt;Parámetro&gt; es una cadena que corresponde al directorio ruta.

- **To configurar el motor de búsqueda utilizado para la búsqueda INTERNET**, &lt;Parámetro&gt; es una cadena correspondiente al comando Line del motor de búsqueda. Por ejemplo: 

	- Google: "http://www.google.com/#q=%1"

	- Bing: "http://www.bing.com/search?q=%1"

	- Yahoo: "http://fr.search.yahoo.com/search?p=%1"


Si &lt;Parámetro&gt; = "" (cadena vacía), se usará el motor Default (Google). 

- **Para configurar la impresión del reporte** (y especialmente cómo se conserva el formato de las celdas), &lt;Parámetro&gt; es un entero que vale 1.

- **Para definir el tiempo de visualización de un mensaje temporal (filtros en los controles de la Tabla)**, &lt;Parámetro&gt; indica el tiempo de visualización expresado en centésimas de segundo. Esta duración es set a 7 segundos por Default. Este parámetro puede corresponder a: 

	- un número entero que corresponde al número de centésimas de segundo,

	- una variable de tipo Duration,

	- la duración en un formato legible (por ejemplo, "1 s" o "10 cs"). 

	- -1 para usar la duración de Default.




## OrganizerChangeMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes).







![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo de aplicación universal de Windows 10, esta función sólo se puede utilizar para los controles del organizador manipular en un reporte. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the first four days of the week
OrganizerChangeMode(ORG_MyOrganizer, orgzWeek, 4)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`OrganizerChangeMode(<Organizer control> , <Display range> [, <Number of days>])`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

- una control Agenda en una reporte.




**`<Display range>: (Constante de tipo Integer)`**

Rango de tiempo que debe mostrarse en el control Agenda:


|   |   |
| --- | --- |
| *orgzCustomNbDay* | Muestra el organizador en el número de días especificado en &lt;Número de días&gt;.<br><br> |
| *orgzDay* | Muestra la agenda de un día. |
| *orgzMonth* | Muestra la agenda de un mes. |
| *orgzWeek* | Muestra el organizador en el número de días especificado en &lt;Número de días&gt;. |
| *orgzWeekOn5* | Mostrar el organizador en 5 días (no se muestra el fin de semana). |



**`<Number of days>: (Entero opcional)`**

Número de días mostrados. Si el parámetro &lt;Rango de visualización&gt; es igual a : 

- *orgzWeek*: Número de días mostrados en la semana. Si no se especifica este parámetro, el organizador se mostrará en una semana completa.. 

- *orgzCustomNbDay*: Número de días que se muestran en el control Agenda. 


En otros casos, se muestra un error WLanguage.

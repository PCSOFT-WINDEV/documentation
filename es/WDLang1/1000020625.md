
## ToastDisplayPopup (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una página emergente durante una duración determinada para mostrar un mensaje "Toast".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ToastDisplayPopup(POPUP_Toast)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ToastDisplayPopup(<Popup> [, <Display duration> [, <Vertical alignment> [, <Horizontal alignment>]]])`
---

**`<Popup>: (Nombre del control)`**

Nombre del control Popup a mostrar.

**`<Display duration>: (Entero opcional)`**

Constante que indica la duración de visualización del mensaje Toast: 


|   |   |
| --- | --- |
| *toastLong* | El mensaje Toast se mostrará durante un largo período de tiempo. |
| *toastShort*<br>(Valor predeterminado) | El mensaje Toast se mostrará durante un corto período de tiempo. |


**Observación**: La duración exacta de visualización depende del dispositivo utilizado.

**`<Vertical alignment>: (Entero opcional)`**

Constante que indica la alineación vertical del mensaje. Esta alineación es relativa a la página.


|   |   |
| --- | --- |
| *vaBottom*<br>(Valor predeterminado) | El mensaje Toast se muestra abajo. |
| *vaMiddle* | El mensaje Toast se muestra en el centro. |
| *vaTop* | El mensaje Toast se muestra arriba. |



**`<Horizontal alignment>: (Entero opcional)`**

Constante que indica la alineación horizontal del mensaje. Esta alineación es relativa a la página.


|   |   |
| --- | --- |
| *haCenter*<br>(Valor predeterminado) | El mensaje Toast se muestra en el centro. |
| *haLeft* | El mensaje Toast se muestra a la izquierda. |
| *haRight* | El mensaje Toast se muestra a la derecha. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000082"></a>

El contenido de los controles de la ventana popup (texto, color de fondo, etc.) debe inicializarse antes de llamar a la función **ToastDisplayPopup**. 

La función **ToastDisplayPopup** puede utilizarse para un solo toast a la vez. Al llamar a la función **ToastDisplayPopup** por segunda vez, se ocultará el toast anterior.

Esta función es compatible con [ToastDisplay](../WDLang1/1000019358.md). Los toasts que se muestran con la función [ToastDisplay](../WDLang1/1000019358.md) aparecen por encima de las ventanas emergentes (independientemente de que se muestren con la función [PopupDisplay](../WDLang2/1000019788.md), **ToastDisplayPopup**, etc.).
<a name="NOTE0_2"></a>


### Características de las ventanas emergentes
<a name="caracteristicas_las_ventanas_emergentes_ELTPARAGRAPHE000110"></a>

- La posición de visualización de la ventana emergente es relativa al área de visualización de la página.

- La ventana emergente está fijada en la página : la ventana emergente permanece en la misma posición aunque se desplace la página o cambie de tamaño.

- La ventana emergente se muestra sin habilitar el OVI (Oscurecimiento de Ventanas Inactivas) : el resto de la página (que no está cubierta por la ventana emergente) permanece accesible.

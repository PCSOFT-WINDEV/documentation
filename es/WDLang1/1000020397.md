


## TimelineChangeMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica el período (es decir, el número de segundos, milisegundos, microsegundos,...) que muestra un control Línea de tiempo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on BTN_Seconds
// Displays 2 minutes 
TimelineChangeMode(TL_TimeLine, schModeSecond, 120)

TL_TimeLine.GranularityMovement = 5
TL_TimeLine.GranularityDuration = 1
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TimelineChangeMode(<TimeLine control> , <Display mode> [, <Range>])`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de la control Línea de tiempo utilizada.

**`<Display mode>: (Constante de tipo Integer)`**

Rango de tiempo que debe mostrarse en la pantalla de control Línea de tiempo: 



|   |   |
| --- | --- |
| *schModeCustomNbDay* | Muestra el control Línea de tiempo en el número de días especificado en &lt;Rango>. |
| *schModeDay* | Muestra la control Línea de tiempo mientras toma el día como resolución de pantalla. |
| *schModeMicrosecond* | Muestra el control Línea de tiempo mientras toma el microsegundo como resolución de pantalla. |
| *schModeMillisecond* | Muestra el control Línea de tiempo mientras toma el milisegundo como resolución de pantalla. |
| *schModeMonth* | Muestra el control Línea de tiempo mientras se toma el mes como resolución de pantalla. |
| *schModeSecond* | Muestra la control Línea de tiempo mientras toma la segunda como resolución de pantalla. |
| *schModeWeek* | Muestra el control Línea de tiempo en una semana de 5 días o en una semana de 7 días. El número de días se especifica en &lt;Rango>. |



**`<Range>: (Entero opcional)`**

Rango total a mostrar en la control Línea de tiempo teniendo en cuenta la nueva unidad.

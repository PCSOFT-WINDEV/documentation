
## OemToAnsi (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena OEM (DOS) en una cadena ANSI (Windows). Los caracteres acentuados son diferentes en ambos estándares. La conversión consiste en modificar los caracteres acentuados.

**Recordatorio**:

- ANSI es el conjunto de caracteres utilizado en Windows.

- OEM es el conjunto de caracteres utilizado en DOS (o consola de Windows).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Str is string
Str = Charact(130) + "t" + Charact(130)  // "été" in OEM
Info(OemToAnsi(Str))
// displays "été" in ANSI in the Windows information window
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = OemToAnsi(<OEM character string>)`
---

**`<Result>: (Cadena de caracteres ANSI)`**

Cadena ANSI.

**`<OEM character string>: (Cadena de caracteres)`**

Cadena OEM que se va a convertir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **OemToAnsi** no convierte caracteres con un código ASCII entre 0 y 31 (caracteres de control).

- **Atención**: No utilice la función **OemToAnsi** con una cadena que contenga caracteres específicos de ANSI (caracteres con acentos): estos caracteres se convertirán y el resultado será incorrecto.

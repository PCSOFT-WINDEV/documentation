
## Constantes para la gestión del registro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **registrySeekKey** | Realiza una búsqueda en el nombre de las claves del registro. | [RegistrySeek](../WDLang1/3065005.md) |
| **registrySeekData** | Realiza una búsqueda en los datos del registro (String, Multiple String o Extensible String data only). | [RegistrySeek](../WDLang1/3065005.md) |
| **registrySeekWholeWord** | Realiza una búsqueda en el registro de la palabra completa especificada en &lt;Sought String>. | [RegistrySeek](../WDLang1/3065005.md) |
| **registrySeekRecursive** | Realiza una búsqueda recursiva en el registro desde la clave base de la búsqueda. | [RegistrySeek](../WDLang1/3065005.md) |
| **registrySeekValue** | Realiza una búsqueda en el registro con los nombres de los valores. | [RegistrySeek](../WDLang1/3065005.md) |
| **registryModeAuto** | Modo de acceso al registro: Modo automatic | [Funciones de RegistryXXX](../WDLang1/3065004.md) |
| **registryMode32** | Modo de acceso al registro: Modo forzado para acceder al registro como un programa de 32 bits. | [Funciones de RegistryXXX](../WDLang1/3065004.md) |
| **registryMode64** | Modo de acceso al registro: Modo forzado para acceder al registro como un programa de 64 bits. | [Funciones de RegistryXXX](../WDLang1/3065004.md) |
| **registryTypeBinary** | Clave de registro binario | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeString** | Clave de registro de la cadena | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeStringEnv** | Clave de registro de cadena que contiene variables de entorno. | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeInt** | Clave de registro de 4 bytes enteros | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeIntegerBigEndian** | Clave de registro de números enteros de gran tamaño | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeInt_8** | Clave de registro de 8 bytes enteros | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |
| **registryTypeMultiString** | Clave de registro multistring | [RegistrySetValue](../WDLang1/3065006.md), [RegistryValueType](../WDLang1/1000017280.md) |

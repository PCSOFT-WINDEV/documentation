
## AAFPossible ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si se puede ejecutar una AAF (característica de la aplicación automatic) en un control específico.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gray "BTN_Cancel" out if no action can be undone
IF AAFPossible(PSHEET_MySpreadsheet, aafUndo) THEN
	BTN_Cancel.Grayed = False
ELSE
	BTN_Cancel.Grayed = True
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AAFPossible(<Control> , <Feature to use>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si las AAF (Funcionalidades Automáticas de la Aplicación) pueden utilizarse,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 




**`<Control>: (Nombre del control)`**

Nombre del control que se va a utilizar, para el que se solicita una AAF (característica de aplicación de automatic) disponible. Este parámetro puede corresponder a: 

- a control Hoja de cálculo. 

- una ventana interna que se ejecuta en modo de edición. 




**`<Feature to use>: (Constante de tipo Integer)`**

Aplicación automatic Característica a utilizar: 


|   |   |
| --- | --- |
| *aafPaste* | Se utiliza para averiguar si "Pegar" está disponible para el control. <br><br>Control utilizado: control Hoja de cálculo. |
| *aafCopy* | Sirve para saber si "copy" está disponible para el control. <br><br>Control utilizado: control Hoja de cálculo. |
| *aafCut* | Se utiliza para averiguar si "Corte" está disponible para el control. <br><br>Control utilizado: control Hoja de cálculo. |
| *aafSaveAs* | Se utiliza para averiguar si el contenido de la control puede ser guardado bajo otro nombre.  |
| *aafRedo* | Se utiliza para averiguar si "Redo" (rehacer la última acción) está disponible para la control. |
| *aafUndo* | Se utiliza para averiguar si "Deshacer" (deshacer la última acción) está disponible para la control. |


Si se usa otra constante *aafxxx*, &lt;Resultado&gt; es set a <u><u><u><u>False</u></u></u></u>.


## WLanguage Procedure llamado por ExplorerAccept



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
WLanguage Procedure ("Callback") llamado por [ExplorerAcceptar](../WDLang1/3030015.md) durante una operación de arrastrar y soltar desde el explorador de Windows. Este procedimiento puede ser local, global o interno.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Drag and Drop with the Windows explorer: activation
ExplorerAccept(True, STC_ZONE_DROP, Call_backExplorerAccept)

	INTERNAL PROCEDURE Call_backExplorerAccept(NumberOfFiles is int, 
					ListOfFiles is string)
		i is int
		
		FOR i = 1 TO NumberOfFiles
			FileName is string
			FileName = ExtractString(ListOfFiles, i, CR)	
			TableAddLine(TABLE_DND_WINDOWS, FileName)
		END
	END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ExplorerAccept_Callback(<Number of files> , <List of files>)`
---

**`<Number of files>: (Integro)`**

Número de archivos que se han eliminado del Explorador de Windows. 

**`<List of files>: (Cadena de caracteres)`**

Lista de los archivos que se han eliminado del Explorador de Windows. Los diferentes archivos están separados por un retorno de carro (CR). Esta lista tiene el siguiente formato: 

```txt
<File 1> + CR + ... + CR + <File N>
```

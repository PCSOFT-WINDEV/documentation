
## gRectángulo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres utilizados para dibujar un rectángulo.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En una ventana, esta secuencia de caracteres se utiliza para dibujar un rectángulo:  

- en un elemento que se encuentra en un List Box o control Combo Box.

- en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Draw a rectangle in an element of "LIST_Shape"
// The rectangle background is light green and the border is light yellow
ListAdd(LIST_Shape, gRectangle(0, 0, 25, 18, LightGreen, LightYellow))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gRectangle(<X1> , <Y1> , <X2> , <Y2> [, <Background color> [, <Line color>]])`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres utilizados para dibujar un rectángulo. 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<X1>: (Real)`**

Coordenada X (en píxeles) de la Corner superior izquierda del rectángulo a dibujar.

**`<Y1>: (Real)`**

Coordenada Y (en píxeles) de la Corner superior izquierda del rectángulo a dibujar.

**`<X2>: (Real)`**

Abscisa (en pixeles) de la esquina inferior derecha del rectángulo a dibujar.

**`<Y2>: (Real)`**

Ordenada (en pixeles) de la esquina inferior derecha del rectángulo a dibujar.

**`<Background color>: (Entero o constante (opcional))`**

Color de fondo del rectángulo. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el fondo Color corresponde al fondo Color del texto actual (definido por [gBackground](../WDLang1/3029037.md)).

**`<Line color>: (Entero o constante (opcional))`**

Color de la línea del rectángulo. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, el Color de la línea corresponde al Color del texto actual (definido por [gPen](../WDLang1/3029039.md)).

<a name="NOTE0_1"></a>

## Observaciones




### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000133"></a>

Todas las coordenadas se expresan en pixeles con respecto a la esquina superior izquierda del elemento en el que se dibujará el rectángulo (por ejemplo, esquina superior izquierda de la fila del control List Box o Combo Box).
<a name="NOTE0_2"></a>




## &lt;Tabla&gt;.SavePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Ahorra: 

- la posición de la barra de desplazamiento en una Tabla o control Tabla TreeView,

- los elementos seleccionados en el control. 




Estos elementos pueden ser restaurados por [&lt;Tabla&gt;.RestorePositionAndSelection](../WDLang1/1000024280.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SaveTable is string
SaveTable = TABLE_Table1.SavePositionAndSelection()
TABLE_Table1.Display(taInit)
TABLE_Table1.RestorePositionAndSelection(SaveTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Table control>.SavePositionAndSelection()`
---

**`<Result>: (Cadena de caracteres)`**



- Información que se enviará a [&lt;Tabla&gt;.RestorePositionAndSelection](../WDLang1/1000024280.md). 

- Cadena vacía ("") si la control Tabla está vacía.




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000216"></a>

La función **&lt;Tabla&gt;.SavePositionAndSelection** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- un control de selección única o de selección múltiple.



<a name="NOTE0_2"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000232"></a>

Las funciones **&lt;Tabla&gt;.SavePositionAndSelection** y [&lt;Tabla&gt;.RestorePositionAndSelection](../WDLang1/1000024280.md) equivalen a la opción "Recordar posición y selección" disponible en la pestaña "Detalles" de la descripción del control Tabla.
<a name="NOTE0_3"></a>


### Limitación
<a name="limitacion_ELTPARAGRAPHE000253"></a>

La función **&lt;Tabla&gt;.SavePositionAndSelection** no funciona en controles Tabla archivo de datos con rebote (es decir, sin barra de desplazamiento proporcional).


## SpreadsheetListWorksheet( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de hojas de trabajo que se encuentran en una control Hoja de cálculo.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ArrName is array of strings
SpreadName = SpreadsheetListWorksheet(PSHEET_Spreadsheet1) 
FOR EACH sWorksheetName OF SpreadName
	ListAdd(LIST_MyWorksheets, sWorksheetName)	
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetListWorksheet(<Spreadsheet control>)`
---

**`<Result>: (Array of strings)`**

Array que contiene el nombre de cada hoja de trabajo que se encuentra en el control Hoja de cálculo. 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.




## Funciones de gestión de cadenas gráficas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones para gestionar strings de gráfico se utilizan principalmente para visualizar los strings de gráfico en: 

- las líneas que se encuentran en los controles List Box o Combo Box,

- Controles estáticos. 

- la barra de título o la barra de estado de una ventana. 




Las siguientes funciones de WLanguage se utilizan para crear cadenas de gráficos y manipular: 



|   |   |
| --- | --- |
| [gAlignment](../WDLang1/3029046.md) | Construye una secuencia de caracteres que contiene un texto alineado o una Image alineada. |
| [gBackground](../WDLang1/3029037.md) | Construye una secuencia de caracteres utilizados para definir la pantalla Color (fondo Color de un texto, círculo, rectángulo, ...). |
| [gBackgroundGradient](../WDLang1/1000017060.md) | Describe una pantalla Color en modo Gradient. |
| [gCircle](../WDLang1/3029041.md) | Construye una secuencia de caracteres que contiene un círculo o una elipse. |
| [gClickableLink](../WDLang1/1000021774.md) | Añade un enlace en el que se puede hacer clic en una cadena de gráficos. |
| [gCoord](../WDLang1/3029034.md) | Construye una secuencia de caracteres que se utiliza para definir la posición del cursor de entrada de un texto. |
| [gFont](../WDLang1/3029047.md) | Construye una secuencia de caracteres que describen la fuente que se utilizará para el texto. |
| [gFontBold](../WDLang1/3029061.md) | Construye una secuencia de caracteres utilizados para especificar si el texto que sigue a la llamada a esta función está en negrita o no. |
| [gFontItalic](../WDLang1/3029062.md) | Construye una secuencia de caracteres que se utiliza para especificar si el texto que sigue a la llamada a esta función está en cursiva o no. |
| [gFontSize](../WDLang1/3029064.md) | Construye una secuencia de caracteres que se utiliza para especificar el tamaño de la fuente para un texto que se encuentra en una cadena gráfica. |
| [gFontStrikeout](../WDLang1/1000023596.md) | Construye una secuencia de caracteres que se utiliza para especificar si el texto que sigue a la llamada a esta función está tachado o no. |
| [gFontUnderlined](../WDLang1/3029063.md) | Construye una secuencia de caracteres para especificar si el texto que sigue a la llamada a esta función está o no subrayado. |
| [gImage](../WDLang1/3029042.md) | Construye una secuencia de caracteres que contiene una Image. |
| [gImageExists](../WDLang1/1000021860.md) | Comprueba si una Image fue cargada en la memoria. |
| [gLine](../WDLang1/3029044.md) | Construye una secuencia de caracteres que contiene una Line. |
| [gLink](../WDLang1/3029043.md) | Agrega un valor "invisible" a un elemento de un control List Box o Combo Box.<br><br>Se mantiene la función por compatibilidad. Se recomienda utilizar [gStoredValue](../WDLang1/1410088363.md). |
| [gLinkActivate](../WDLang1/3029036.md) | Permite (o no) recuperar el valor definido por [gLink](../WDLang1/3029043.md) en un cuadro de lista o una control Combo Box.<br><br>Se mantiene la función por compatibilidad. Se recomienda utilizar [gStoredValueEnable](../WDLang1/1410088364.md). |
| [gPen](../WDLang1/3029039.md) | Construye una secuencia de caracteres para describir el color del texto y/o del contorno de una figura (rectángulo o círculo). |
| [gRectángulo](../WDLang1/3029045.md) | Construye una secuencia de caracteres utilizados para dibujar un rectángulo. |
| [gStoredValue](../WDLang1/1410088363.md) | Añade un valor "Invisible" a un elemento de un cuadro de lista, Combo Box o control Planificador. |
| [gStoredValueEnable](../WDLang1/1410088364.md) | Permite (o no) recuperar el valor definido por [gStoredValue](../WDLang1/1410088363.md) en un Cuadro de Lista o en una control Combo Box. |
| [gToRetrieve](../WDLang1/3029025.md) | Describe el valor de texto que se debe mostrar y devolver cuando se recupera el valor de un elemento que se encuentra en un control List Box o en un control Combo Box. |
| [gToText](../WDLang1/1000020524.md) | Elimina los elementos gráficos de una cadena de caracteres que utiliza [gStoredValue](../WDLang1/3029043.md), [gImage](../WDLang1/3029042.md), ... |
| [LoadImage](../WDLang1/3029022.md) | Carga un Image en la memoria para que pueda ser usado más tarde por [gImage](../WDLang1/3029042.md). |
| [LoadImageSheet](../WDLang1/1000018826.md) | Carga una imagen set en memoria para que pueda ser utilizada posteriormente por [gImage](../WDLang1/3029042.md). |

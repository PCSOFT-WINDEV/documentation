
## WinPasteForm (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pega el contenido de un formulario almacenado en el portapapeles.

**Observación**: El contenido del formulario se ha copiado previamente: 

- por la función [Formulario WinCopy](../WDLang1/1000018992.md).

- por la AAF (automatic Application Feature) para copiar formularios.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copies the form values
WinPasteForm()  // equivalent to WinPasteForm(MyWindow)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinPasteForm([<Window name>])`
---

**`<Window name>: (Cadena de caracteres opcional)`**

Nombre o alias de la ventana que se va a utilizar. Si no se especifica este parámetro o si corresponde a una cadena vacía ("""), se gestiona la ventana actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Tipos de controles que se tienen en cuenta
<a name="tipos_controles_que_tienen_cuenta_ELTPARAGRAPHE000040"></a>

Los tipos de controles soportados son:

- Controles de edición,

- Botones de opción/Cajas de verificación,

- Deslizadores, deslizadores redondos,

- Calendarios, 

- Combo boxes editables.

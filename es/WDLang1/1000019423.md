
## ThreadCurrent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre de el hilo actualmente en funcionamiento.




<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ThreadCurrent()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de el hilo que se ejecuta actualmente,

- "" (cadena vacía) si la el hilo actual es la el hilo principal.


## EmulateCombo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula una de las siguientes acciones en un control Combo Box: 

- clic en el botón que permite desplegar la lista del control Combo Box. 

- seleccionar un elemento en un control Combo Box.



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Select the 5th element of the Combo Box control
EmulateCombo(COMBO_Choice, 5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Emular el control Combo Box con el identificador del elemento

`EmulateCombo(<Combo Box control> [, <Element index>])`
---

**`<Combo Box control>: (Nombre del control)`**

Nombre del control Combo Box.

**`<Element index>: (Entero opcional)`**

Índice del elemento a seleccionar en el control Combo Box. Si no se especifica este parámetro, se simula un clic en el botón que despliega la lista del control Combo Box.


<a name="SYNTAX2"></a>

### Emular el control Combo Box con el título del elemento

`EmulateCombo(<Combo Box control> [, <Element caption>])`
---

**`<Combo Box control>: (Nombre del control)`**

Nombre del control Combo Box.

**`<Element caption>: (Cadena de caracteres opcional)`**

Título del elemento a seleccionar en el control (se seleccionará el primer elemento que contenga el título).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función solo se debe utilizar en el código de las pruebas automatizadas.

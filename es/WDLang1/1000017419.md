


## Funciones para el manejo de los controles List Box, ListView y Combo Box (sintaxis de prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones manipular List Box, ListView y Combo Box controles (sintaxis de prefijo): 



|   |   |
| --- | --- |
| [&lt;Caja de lista&gt;.add](../WDLang1/1000017319.md) | Añade un elemento mediante programación a un cuadro de lista, ListView, Combo Box o control Carrusel. |
| [&lt;Caja de lista&gt;.Borrar](../WDLang1/1000017415.md) | Elimina un elemento de un cuadro de lista, ListView o control Combo Box rellenado mediante programación. |
| [&lt;Caja de lista&gt;.Borrar todo](../WDLang1/1000017416.md) | Borra un cuadro de lista, ListView o control Combo Box rellenado mediante programación. |
| [&lt;Caja de lista&gt;.display](../WDLang1/1000017318.md) | Actualiza un cuadro de lista, ListView o control Combo Box basado en un archivo de datos. |
| [&lt;Caja de Lista&gt;.InfoXY](../WDLang1/1000017323.md) | Devuelve el índice de la fila (o Image) en una posición dada en un Cuadro de Lista o control ListView. |
| [&lt;Caja de lista&gt;.Ordenar](../WDLang1/1000017417.md) | Ordena los elementos de una Caja de Listas, ListView o control Combo Box poblada programáticamente. |
| [&lt;Caja de lista&gt;.Posición](../WDLang1/1000017335.md) | Muestra un cuadro de lista (o un ListView) control de un elemento especificado o devuelve el índice del primer elemento mostrado en un control List Box (si se utilizó la barra de desplazamiento). |
| [&lt;Caja de Lista&gt;.SelectCount](../WDLang1/1000017394.md) | Devuelve el número de elementos seleccionados en un cuadro de lista, ListView o control Combo Box. |
| [&lt;Caja de lista&gt;.SelectPlus](../WDLang1/1000017414.md) | Selecciona uno o varios elementos de un Cuadro de Lista, ListView o control Combo Box. |
| [&lt;Insertar](../WDLang1/1000017326.md) | Inserta un elemento en una Caja de Listas, ListView o control Combo Box poblada programáticamente. |
| [&lt;Modificar](../WDLang1/1000017327.md) | Modifica un elemento de una Caja de Listas, ListView o control Combo Box poblada programáticamente. |
| [&lt;Seleccione](../WDLang1/1000017336.md) | Devuelve el índice del elemento seleccionado en un Cuadro de Lista, ListView o control Combo Box. |
| [&lt;SelectMinus](../WDLang1/1000017393.md) | Deselecciona un elemento de un Cuadro de Listas, ListView o control Combo Box. |

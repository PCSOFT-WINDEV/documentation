
## ToClipboard( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe el texto o la información de Image en el portapapeles del sistema.

Para escribir información de texto RTF, utilice [ToClipboardRTF](../WDLang1/1000017199.md).

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Sólo tiene la capacidad de escribir información de texto en el portapapeles de Windows.










<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Text is string
Res is int
// The content of the control is copied into the clipboard
Text = EDT_EDIT1
Res = ToClipboard(Text)
IF Res = False THEN
	Error("Error while writing into the clipboard")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Escribir un texto en el portapapeles

`<Result> = ToClipboard(<Text>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha copiado el texto en el portapapeles,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Text>: (Cadena de caracteres)`**

Texto que se escribirá en el portapapeles. 


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Copiar una Image en el portapapeles

`<Result> = ToClipboard(<Image> , <Transparency>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha copiado la imagen en el portapapeles,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image>: (Cadena de caracteres o variable de tipo Image)`**



- Nombre de control Imagen que se copiará en el portapapeles. Si el control Imagen no pertenece a la ventana actual, utilice la siguiente anotación: &lt;Nombre de la ventana&gt;.&lt;Nombre del control Imagen&gt;.  

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la [Image](../WDLang1/1000019650.md) Variable que se va a copiar en el portapapeles. 




**`<Transparency>: (booleano)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 

- <u><u><u><u>True</u></u></u></u> (Default) para mantener la transparencia del Image (si es posible) cuando se copie en el portapapeles, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **ToClipboard** puede utilizarse para cortar/copiar/pegar información de tipo texto o imagen (BMP).

- [Portapapeles](../WDLang1/3063004.md) se utiliza para leer el contenido del portapapeles. [Portapapeles](../WDLang1/3063004.md) no puede utilizarse para leer cualquier tipo de información que no sea texto o image (por ejemplo, binario, etc.) en el portapapeles.

- [ToClipboardRTF](../WDLang1/1000017199.md) y [PortapapelesRTF](../WDLang1/1000017200.md) se utilizan para gestionar la información en RTF.


## TooltipMaxWidth( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la anchura máxima de los tooltips. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Width of tooltips set to 300 pixels
TooltipMaxWidth(300)

// Display large tooltips
TooltipMaxWidth(1280)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TooltipMaxWidth(<Width>)`
---

**`<Width>: (Integro)`**

Nuevo ancho máximo de la información de herramienta en píxeles. 
Este ancho es de set a 640 por Default. Esta anchura no puede ser inferior a 100.

**Observación**: En la versión 21, el ancho máximo de la punta de la herramienta era de set a 300 píxeles. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función configura tanto los tooltips estándar como los personalizados.

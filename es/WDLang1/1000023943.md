
## Funciones de gestión de fechas y horas (sintaxis con prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones permiten gestionar fechas y horas mediante una sintaxis con prefijo:



|   |   |
| --- | --- |
| [&lt;Chrono&gt;.End](../WDLang1/1000023789.md) | Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md)). |
| [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md) | Pone en pausa un cronómetro para medir el tiempo de un proceso. |
| [&lt;Chrono&gt;.Reset](../WDLang1/1000023791.md) | Detiene y reinicia un cronómetro en marcha. |
| [&lt;Chrono&gt;.Resume](../WDLang1/1000023792.md) | Reanuda un cronómetro detenido previamente con las funciones [&lt;Chrono&gt;.Fin](../WDLang1/1000023789.md) o [&lt;Chrono&gt;.Pause](../WDLang1/1000023790.md). |
| [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) | Inicia un cronómetro para medir la duración de un proceso (en milisegundos) y reinicia un cronómetro en marcha. |
| [&lt;Tipo Date&gt;.ToDay](../WDLang1/1000024030.md) | Devuelve el día que corresponde a una fecha determinada. |
| [&lt;Tipo Date&gt;.ToDayInAlpha](../WDLang1/1000024031.md) | Devuelve el nombre del día de una fecha determinada. |
| [&lt;Tipo Date&gt;.ToInteger](../WDLang1/1000024029.md) | Convierte una fecha en un número entero. |
| [&lt;Tipo Date&gt;.ToMonthInAlpha](../WDLang1/1000024051.md) | Devuelve el nombre del mes de una fecha determinada. |
| [&lt;Tipo Date&gt;.ToString](../WDLang1/1000023952.md) | Convierte: <br><br>- una fecha en cadena en el formato especificado.<br><br>- una fecha/hora en una cadena en el formato especificado.<br><br><br> |
| [&lt;Tipo Date&gt;.ToWeekNumber](../WDLang1/1000024160.md) | Devuelve el número de la semana del año que corresponde a una fecha determinada. |
| [&lt;Tipo DateTime&gt;.ToEpoch](../WDLang1/1000024161.md) | Convierte una fecha y hora local en formato Epoch (número de unidades de tiempo desde el 01/01/1970 a las 00h00 UTC). |
| [&lt;Tipo DateTime&gt;.ToReal](../WDLang1/1000024190.md) | Convierte una fecha y hora en un real de 8 bytes. |
| [&lt;Tipo DateTime&gt;.ToString](../WDLang1/1000025807.md) | Convierte una fecha/hora en una cadena en el formato especificado. |
| [&lt;Tipo Duration&gt;.ToString](../WDLang1/1000023947.md) | Convierte una duración en una cadena de caracteres en el formato especificado. |

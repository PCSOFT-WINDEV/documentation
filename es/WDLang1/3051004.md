
## MCIPositionFormat( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Corrige el formato utilizado para definir la posición en un archivo multimedia (abierto por [MCIOpen](../WDLang1/3051017.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MCIOpen("Film.avi", "MyFilm")
...
MCIPositionFormat("MyFilm", MciFormatTMSF)
...
MCIClose("MyFilm")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIPositionFormat(<Media file or alias> , <Format> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el cambio de formato,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para get el número de error, utilice [MCIErr](../WDLang1/3051011.md). 
	Para obtener más detalles sobre el error, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
	El estado reporte del comando es devuelto por [MCIRetrieve](../WDLang1/3051022.md).




**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Format>: (Constante de tipo Character String)`**

Formato de la posición: 


|   |   |
| --- | --- |
| *MciFormatByte* | Número de bytes |
| *MciFormatFrame* | Número de imágenes |
| *MciFormatHMS* | Número de horas, minutos y segundos expresados como "hh:mm:ss". |
| *MciFormatMS* | Milisegundos |
| *MciFormatMSF* | Formato: Minuto, segundo, cuadro (":mm:ss:ff") |
| *MciFormatTMSF* | Formato: Pista, Minuto, Segundo, Cuadro ("tt:mm:ss:ff") |
| *MciFormatTrack* | Número de pistas |



**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario.
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.

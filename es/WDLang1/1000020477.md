


## StringInsert (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una cadena en otra cadena de caracteres en una posición determinada. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add Sol to SiLaFaMiReDo
sNotes is string = "SiLaFaMiReDo"
sFullNotes is string = StringInsert(sNotes, "Sol", 5)
// sFullNotes contains "SiLaSolFaMiReDo"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = StringInsert(<Initial string> , <String to insert> , <Insertion position>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres en la que se insertó la cadena en la posición especificada. 

**`<Initial string>: (Cadena de caracteres o Buffer)`**

Cadena de caracteres en la que se insertará otra cadena. 
ATENCIÓN: Esta cadena no se modificará.

**`<String to insert>: (Cadena de caracteres o Buffer)`**

Cadena a insertar en la &lt;Cadena inicial&gt;. 

**`<Insertion position>: (Entero)`**

Posición en la que se añadirá la &lt;Cadena a insertar&gt;.

- Si la posición especificada es negativa, la cadena se insertará al inicio de la cadena inicial.

- Si la posición especificada es mayor que la longitud de la cadena inicial, la cadena se insertará al final de la cadena inicial.

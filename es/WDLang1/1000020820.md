


## NationRightToLeft( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para averiguar si el lenguaje actual se escribe de derecha a izquierda. Esta información se almacena en la información lingüística del proyecto (pestaña "Idiomas" de la ventana description del proyecto, subpestaña "Notas adicionales"): <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NationDroiteAGauche%20-%20HC%20N%B0001.gif)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF NationRightToLeft() THEN
	// Code specific to languages written from right to left
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = NationRightToLeft()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el idioma actual se escribe de derecha a izquierda,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

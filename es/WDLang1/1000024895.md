


## &lt;Tipo String&gt;.CharactTypeCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de caracteres que coinciden con un tipo determinado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbChar is int
MyString is string = "AbcdeF"
NbChar = MyString.CharactTypeCount(ctUppercase)
// NbChar = 2
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <String>.CharactTypeCount(<Type>)`
---

**`<Result>: (Entero)`**

Número de caracteres en la cadena que corresponden al tipo especificado.

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a manipular.

**`<Type>: (Constante o combinación de constantes de tipo Integer)`**

Tipo de caracteres a buscar: 


|   |   |
| --- | --- |
| *ctAccent* | Caracteres con acentos y acentos diacríticos. |
| *ctAlpha* | Letras. |
| *ctLowercase* | Caracteres en minúsculas. |
| *ctNumeric* | Caracteres numéricos. |
| *ctPunctuation* | Caracteres de puntuación. |
| *ctSpace* | Espacios. |
| *ctUppercase* | Caracteres en mayúsculas. |


Observaciones: 

- La combinación de las constantes *ctUppercase* + *ctAccent* permite buscar caracteres en mayúscula con acentos. 

- La combinación de las constantes *ctLowercase* + *ctAccent* permite buscar caracteres en minúscula con acentos.

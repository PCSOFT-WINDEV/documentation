
## ParallelTaskConfigure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica los parámetros de tareas paralelas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Limits the maximum number of simultaneous tasks to 8
ParallelTaskConfigure(ptpMaximumNumberThread, 8)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Conocer el valor de un parámetro

`<Result> = ParallelTaskConfigure(<Parameter>)`
---

**`<Result>: (Tipo correspondiente al parámetro buscado)`**

Valor del parámetro.

**`<Parameter>: (Constante de tipo Integer)`**

Parámetro cuyo valor se solicita: 


|   |   |
| --- | --- |
| *ptpMaximumNumberThread* | Número máximo de hebras que se pueden ejecutar al mismo tiempo.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Si el número especificado de hilos es menor que el número mínimo de hilos del carrete, el valor especificado será ignorado.. El número mínimo de hilos de bobina depende de la implementación de la JVM. |




<a name="SYNTAX2"></a>

### Modificar el valor de un parámetro

`<Result> = ParallelTaskConfigure(<Parameter> , <Value>)`
---

**`<Result>: (Tipo correspondiente al parámetro buscado)`**

Valor del parámetro antes de la modificación.

**`<Parameter>: (Constante de tipo Integer)`**

Parámetro cuyo valor debe ser modificado: 


|   |   |
| --- | --- |
| *ptpMaximumNumberThread* | Número máximo de hebras que se pueden ejecutar al mismo tiempo. |



**`<Value>: (El tipo correspondiente al parámetro)`**

Nuevo valor para el parámetro.

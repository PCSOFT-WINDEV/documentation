


## dRectangleGradient (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja un rectángulo: 

- en un control Imagen,

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) en una variable de tipo [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una variable de tipo [picLayer](../WDLang1/1000024604.md).




Observación: Para dibujar rectángulos personalizados, le recomendamos utilizar la función [dBorder](../WDLang1/1000021894.md). 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draw a rectangle whose background changes from red to blue
// via yellow
dStartDrawing(IMG_Image1)
dRectangleGradient(10, 20, 100, 150, LightRed, LightBlue, 0, LightYellow, 30)
```

## Sintaxis

### Dibujar un rectángulo especificando los puntos del rectángulo y el degradado

`dRectangleGradient([<Image>, ] <X1> , <Y1> , <X2> , <Y2> , <Start color> , <End color> [, <Angle> [, <Color 3> [, <Distance color 3> [, <Color 4> [, <Distance color 4>]]]]])`
---

**`<Image>: (Nombre de control opcional o variable de tipo Image, WDPic o picLayer opcional)`**

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Imagen a manipular. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [WDPic](../WDLang1/1000023503.md). Solo se manipulará la capa de fondo. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [picLayer](../WDLang1/1000024604.md).




Si no se especifica este parámetro, se debe definir el destino del dibujo con la función [dStartDrawing](../WDLang1/3029009.md).

**`<X1>: (Entero)`**

Coordenada X (en pixeles) de la esquina superior izquierda del rectángulo.

**`<Y1>: (Entero)`**

Coordenada Y (en pixeles) de la esquina superior izquierda del rectángulo.

**`<X2>: (Entero)`**

Coordenada X (en pixeles) de la esquina inferior derecha del rectángulo.

**`<Y2>: (Entero)`**

Coordenada Y (en pixeles) de la esquina inferior derecha del rectángulo.

**`<Start color>: (Entero o constante)`**

Color inicial del degradado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).




**`<End color>: (Entero o constante)`**

Color final del degradado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).




**`<Angle>: (Entero opcional)`**

Ángulo a utilizar para la selección de colores en el degradado. El valor del ángulo está comprendido entre 0 (ángulo horizontal, valor predeterminado) y 360°.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo se pueden utilizar los siguientes ángulos: 0,90,180,270,360.

**`<Color 3>: (Entero o constante opcional)`**

Color a utilizar en el degradado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

**`<Distance color 3>: (Entero opcional)`**

Porcentaje que separa el color inicial del color 3.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

**`<Color 4>: (Entero o constante opcional)`**

Color a utilizar en el degradado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- un color preestablecido (ver [colores predefinidos de WLanguage](../WDLang5/3010002.md)).


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

**`<Distance color 4>: (Entero opcional)`**

Porcentaje que separa el color inicial del color 4.
![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro no está disponible.

## Observaciones


### Parámetros del degradado
<a name="parametros_del_degradado_ELTPARAGRAPHE000674"></a>Los parámetros de color se utilizan de la siguiente manera:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=drectangledegrade.gif)
El ángulo permite definir la orientación del degradado.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=angle_degrade.gif)
Los degradados de 3 o 4 colores solo están disponibles para ángulos de 0 o 90 grados.

**Observación**: Los colores de degradado no están disponibles en algunos sistemas (Windows 98 o Me, por ejemplo) o en TSE 256 colores. En este caso, se muestra un rectángulo cuyo color corresponde a &lt;Color inicial&gt;.

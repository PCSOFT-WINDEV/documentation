
## diagShape (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagShape** permite definir todas las características avanzadas de la forma original utilizada en un diagrama. 

**ATENCIÓN**: 

- Una variable de tipo **diagShape** no debe asignarse directamente y no puede agregarse directamente a un diagrama. Solo los tipos que derivan de este tipo de variable pueden asignarse o agregarse a un diagrama: 

	- Variable de tipo [diagPath](../WDLang1/1410088384.md), 

	- Variable de tipo [diagGroup](../WDLang1/1410088127.md), 

	- Variable de tipo [diagImage](../WDLang1/1410088414.md), 

	- Variable de tipo [diagConnector](../WDLang1/1410088454.md), 

	- Variable de tipo [diagOval](../WDLang1/1410088135.md), 

	- Variable de tipo [diagPolygon](../WDLang1/1410088137.md), 

	- Variable de tipo [diagRectangle](../WDLang1/1410088083.md).




- Las variables de tipo **diagShape** permiten manipular los tipos de formas, en particular los arrays de formas. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Iterate over the shapes in a diagram
ShapeName is string
diagControl is Diagram <- DIAGEDT_DIAG

FOR EACH ShapeInDiagram OF diagControl.Shape
	ShapeName = ShapeInDiagram.Name // Gets the name of the shape
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables de tipo diagShape
<a name="propiedades_especificas_las_variables_tipo_diagshape_ELTPARAGRAPHE000057"></a>

Las siguientes propiedades pueden utilizarse para manipular una forma original:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ActionAllowed | [diagActionAllowed](../WDLang1/1410088840.md) o constante | <br><br>- Variable de tipo [diagActionAllowed](../WDLang1/1410088840.md) que permite definir los cambios que el usuario puede realizar en la forma.<br><br>- Constante *diagActionNone*: ninguna acción disponible. <br><br>- Constante *diagActionAll* (Valor predeterminado): todas las acciones están disponibles. <br><br><br> |
| Altura | Real | Altura de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. |
| anchura | Real | Ancho de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. |
| Ángulo | Real | Ángulo de rotación de la forma (expresado en grados). |
| Background | Constante, entero o [Color](../WDLang1/1000019661.md) | Color de fondo de la forma. <br>La propiedad **Color** define el color de fondo de la forma. <br>Esta propiedad puede corresponder a: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br>- a Variable de tipo [Color](../WDLang1/1000019661.md). En este caso, se tiene en cuenta la opacidad especificada en la variable.<br><br><br> |
| comment | Cadena de caracteres | Nota sobre la forma. Esta nota no se muestra. |
| ConnectionPoint | Array de [diagConnectionPoint](../WDLang1/1410088457.md) | Lista de puntos de conexión de la forma. |
| denominar | Cadena de caracteres | Nombre de la forma. Este nombre se utilizará para la búsqueda, por ejemplo, si la forma está en una biblioteca. |
| Franja de orilla | [diagBorder](../WDLang1/1410088179.md) | Características del borde de la forma. <br>Las propiedades disponibles son: <br><br>- Color: color del borde. <br><br>- Espesor: grosor del borde. <br><br>- escribir a máquina: tipo de borde.<br><br><br> |
| note | Cadena de caracteres | Nota asociada a la forma. |
| Opacidad | Integro | Porcentaje de opacidad de la forma. Este valor oscila entre invisible (0) y completamente opaco (100). |
| Sombra | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> si la forma tiene sombra,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br> |
| texto | [diagText](../WDLang1/1410088393.md) | Texto (con formato) dentro de la forma. |
| X | Real | Coordenada X de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. Esta coordenada X es relativa al padre de la forma. |
| Y | Real | Coordenada Y de la caja que contiene la forma antes de la rotación o zoom, expresada en pixeles. Esta coordenada Y es relativa al padre de la forma. |

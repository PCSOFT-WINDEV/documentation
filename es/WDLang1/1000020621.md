


## Funciones de control Línea de tiempo
			



Las siguientes funciones se utilizan para gestionar los controles de TimeLine:



|   |   |
| --- | --- |
| [TimelineAddEvent](../WDLang1/1000020400.md) | Añade una Event a una control Línea de tiempo. |
| [TimelineAddTrack](../WDLang1/1000020405.md) | Añade una nueva pista a una control Línea de tiempo. |
| [TimelineChangeMode](../WDLang1/1000020397.md) | Modifica el período (es decir, el número de segundos, milisegundos, microsegundos,...) que muestra un control Línea de tiempo. |
| [TimelineDeleteAll](../WDLang1/1000020404.md) | Borra todos los eventos y pistas de un control Línea de tiempo. |
| [TimelineDeleteEvent](../WDLang1/1000020403.md) | Borra un Event de un control Línea de tiempo. |
| [TimelineDeleteTrack](../WDLang1/1000020760.md) | Borra una pista de una control Línea de tiempo. |
| [TimelineDisplay](../WDLang1/1000020683.md) | Reinicializa el contenido de una control Línea de tiempo a partir de su fuente de datos (archivo de datos HFSQL por ejemplo). |
| [TimelineListEvent](../WDLang1/1000020401.md) | Devuelve: <br><br>- la lista de eventos encontrados entre dos veces en un control Línea de tiempo, <br><br>- todos los eventos de un control Línea de tiempo,<br><br>- una Event( específica seleccionada o suspendida). <br><br><br> |

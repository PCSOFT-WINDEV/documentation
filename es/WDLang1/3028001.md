
## DDERecipient( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica el destinatario de una conexión DDE.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a DDE connection
ConnectionNum = DDEConnect(ServerName, "MyTopic")
...

// Recipient of the current connection
Recipient = DDERecipient(ConnectionNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDERecipient([<Connection identifier>])`
---

**`<Result>: (Cadena de caracteres)`**

Destinatario de la conexión DDE.

**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, **DDERecipient** utiliza la última conexión gestionada.

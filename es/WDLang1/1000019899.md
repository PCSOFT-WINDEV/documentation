
## PositionToLine (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de la línea que contiene un carácter identificado por su posición en un bloque de texto.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize a multiline string
s is string = [
AA
BB
CC
]

PositionToLine(s, 1) // Returns 1
PositionToLine(s, 2) // Returns 1
PositionToLine(s, 3) // Returns 1 because of CR
PositionToLine(s, 5) // Returns 2
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PositionToLine(<String> , <Position>)`
---

**`<Result>: (Entero)`**

Número de la línea que contiene el carácter especificado. 

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a manipular. 

**`<Position>: (Entero)`**

Posición del carácter. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El separador de líneas es un retorno de carro (CR).

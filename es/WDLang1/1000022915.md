


## &lt;Organizador&gt;.DateVisible ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Hace visible un día o Invisible en un control Agenda. 

Observación: El día que se hizo Invisible ya no aparecerá si el organizador se muestra semanalmente. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Don't display Wednesday
ThisWednesday is Date = DateSys()
ThisWednesday.Day += 3 - ThisWednesday.ToDay() 
ORG_MyOrganizer.DateVisible(ThisWednesday, False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Haciendo visible un día o Invisible

`<Organizer control>.DateVisible(<Date> , <Visible>)`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda a utilizar (se encuentra en una ventana)

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha que se hará visible o Invisible. Esta fecha está en formato "YYYYMMDD"..

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para hacer visible la fecha,

- <u><u><u><u>False</u></u></u></u> para que la fecha Invisible.





<a name="SYNTAX2"></a>

### Averiguar si un día es visible o Invisible

`<Result> = <Organizer control>.DateVisible(<Date>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si la fecha es visible,

- <u><u><u><u>False</u></u></u></u> si la fecha es invisible.




**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda a utilizar (se encuentra en una ventana)

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha a manipular. Esta fecha está en formato "YYYYMMDD"..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los días de Invisible se "saltan" al dibujar el organizador.
	Por ejemplo, si un miércoles es Invisible, el control Agenda muestra el martes y luego el jueves. 

- La pantalla se refresca inmediatamente o al cambiar de período: 

	- Si el control Agenda actualmente muestra sólo un día y ese día se convierte en Invisible, el día se convierte en Invisible cuando se cambia el día o el período. 

	- Si el control Agenda muestra 1 semana, los días de Invisible no se muestran pero el control Agenda no muestra más días. 

	- Si la control Agenda muestra 1 mes, se muestran los días invisibles (no hay "agujeros" en la Calendar). 




- Si un Appointment dura varios días (entre ellos 1 día Invisible), visualmente aparecerá como más corto.




## &lt;Window&gt;.WindowBitmap (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea la Image de la ventana especificada en un archivo BMP.

**Atención**: La Image de la ventana corresponde a la ventana tal como se muestra en el editor de la ventana. Para crear una Image de la ventana en tiempo de ejecución, utilice [dCopyWindowImage](../WDLang1/1000019912.md) y el tipo [Image](../WDLang1/1000019650.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create images of the WIN_EditCustomer window
// in the "C:\Images\EditCustomer" file
Res = WIN_EditCustomer.WindowBitmap("C:\Images\EditCustomer")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Window>.WindowBitmap(<Image file>)`
---

**`<Result>: (Integro)`**

Número de imágenes creadas. Para la ventana especificada, **&lt;Window&gt;.WindowBitmap** crea tantas imágenes (archivos BMP) como el número de planos de la ventana. Si la ventana incluye pestañas, se crea una sola Image.

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana a utilizar.

**`<Image file>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo resultante (sin especificar la extensión). Se puede utilizar una ruta de tipo UNC.

Si se crea un solo archivo Image, su extensión será ".BMP".

Si se crean varios archivos Image, tendrán para la extensión ".B01", ".B02", ...



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La Image se crea con la información actual del sistema: colores, fuentes,...

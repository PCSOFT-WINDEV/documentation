


## InsertReference (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega la referencia de un objeto en la posición dada en un array unidimensional.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AnArray is array of AClass
AnObject is AClass
InsertReference(AnArray, InsertionIndex, AnObject)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`InsertReference(<WLanguage array> , <Subscript> , <Object>)`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Subscript>: (Integro)`**

Índice donde el elemento será insertado en el array. 

**`<Object>: (Objeto)`**

Nombre del objeto cuya referencia se agrega al array.

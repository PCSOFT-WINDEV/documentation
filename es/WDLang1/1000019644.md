
## SensorEnd( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el fin del uso de las funciones del sensor. Todas las detecciones actuales se detienen: 

- detección de gestos de agitación iniciados con [SensorDetectBeginShake](../WDLang1/1000019363.md) y/o [SensorDetectEndShake](../WDLang1/1000019364.md),

- detección de cambios de aceleración iniciada con [SensorDetectChangeAcceleration](../WDLang1/1000019362.md),

- detección de cambios de orientación iniciada con [SensorDetectChangeOrientation](../WDLang1/1000019361.md).







<a name="sample_code"></a>

## Ejemplo


```wl
SensorEnd()
```

## Sintaxis

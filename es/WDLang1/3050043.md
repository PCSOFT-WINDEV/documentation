
## MatStack( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprime la huella de memoria ocupada por una matriz.

**Observación**: La compresión es necesaria cuando manipular una matriz grande en la que se han hecho varios cálculos. Si esta matriz contiene varias columnas y/o filas nulas, la compresión se utiliza para reducir significativamente la huella de memoria de la matriz.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Compress a matrix
MatStack("MyMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MatStack(<Matrix>)`
---

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a comprimir, definida por [MatCreate](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de matriz para comprimir
<a name="existencia_matriz_para_comprimir_ELTPARAGRAPHE000080"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000091"></a>

**Atención**: La función **MatStack** no devuelve ningún resultado. Para saber si se han generado errores por la compresión, utilice [MatError](../WDLang1/3050044.md) en esta matriz. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

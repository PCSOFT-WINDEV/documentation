
## Funciones para la gestión de los controles del Organizador (sintaxis del prefijo)
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>




Las siguientes funciones se utilizan para manipular la control Agenda mediante la sintaxis de prefijos: 



|   |   |
| --- | --- |
| [&lt;Color de fondo](../WDLang1/1000022895.md) | Define un fondo Color para una franja horaria en un control Agenda. |
| [&lt;Organización&gt;Hora de trabajo](../WDLang1/1000022916.md) | Se utiliza para indicar las horas de trabajo específicas por día en una control Agenda. |
| [&lt;Organizador&gt;.AddAppointment](../WDLang1/1000022894.md) | Añade una nueva Appointment a una control Agenda. |
| [&lt;Organizador&gt;.ChangeMode](../WDLang1/1000022879.md) | Modifica el rango de tiempo mostrado por un control Agenda (Día/Semana/Mes). |
| [&lt;Organizador&gt;.DateVisible](../WDLang1/1000022915.md) | Hace visible un día o Invisible en una control Agenda. |
| [&lt;Organizador&gt;.DeleteAll](../WDLang1/1000022898.md) | Borra todas las citas de una control Agenda. |
| [&lt;Organizador&gt;.DeleteAppointment](../WDLang1/1000022877.md) | Borra una Appointment de una control Agenda. |
| [&lt;Organizador&gt;.DisplayNextPeriodo](../WDLang1/1000022896.md) | Mueve el área de tiempo que se muestra a la siguiente área (por ejemplo, las próximas semanas) en un control Agenda. |
| [&lt;Organizador&gt;.DisplayPeríodo anterior](../WDLang1/1000022878.md) | Desplaza el área de tiempo mostrada al área anterior (semanas anteriores, por ejemplo) en un control Agenda. |
| [&lt;Organizador&gt;.InfoXY](../WDLang1/1000022889.md) | Retornos para una posición determinada en la control Agenda (coordenadas de un punto en el organizador): <br><br>- el índice de la Appointment para la posición especificada. <br><br>- la fecha correspondiente a la posición especificada.<br><br><br> |
| [&lt;Organizador&gt;.lista de citas](../WDLang1/1000022886.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Agenda, <br><br>- todas las citas encontradas en un control Agenda,<br><br>- una Appointment específica (seleccionada o suspendida).<br><br><br> |
| [&lt;Organizador&gt;.modo](../WDLang1/1000022885.md) | Devuelve el modo de visualización de un control Agenda: día, semana, mes, ... |
| [&lt;Organizador&gt;.SelectedRange](../WDLang1/1000022883.md) | Devuelve la fecha y hora de inicio o de fin de la franja horaria seleccionada en una control Agenda. |
| [&lt;Organizador&gt;.SelectMinus](../WDLang1/1000022882.md) | Deselecciona un Appointment de un control Agenda. |
| [&lt;Organizador&gt;.SelectPlus](../WDLang1/1000022881.md) | Selecciona un Appointment en un control Agenda. |
| [&lt;Organizador&gt;.ToiCalendar](../WDLang1/1000022899.md) | Exporta las citas encontradas en un control Agenda en formato iCalendar. |
| [&lt;Posición](../WDLang1/1000022897.md) | Muestra un control Agenda a partir de una fecha especificada o devuelve la fecha del primer día (o la fecha del último día) que aparece en un control Agenda. |
| [&lt;TiempoDisplayed](../WDLang1/1000022892.md) | Define el intervalo de tiempo que muestra control Agenda en el modo Día o en el modo Semana. |
| [&lt;Visualización](../WDLang1/1000022880.md) | Reinicia el contenido de un control Agenda desde su fuente de datos (archivo de datos HFSQL, por ejemplo). |

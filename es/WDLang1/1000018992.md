
## WinCopyForm (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia el contenido de un formulario en el portapapeles.

Observaciones: 

- El contenido copiado puede pegarse en otro formulario a través de [Formulario WinPaste](../WDLang1/1000018993.md).

- También puede copy el contenido de un formulario a través de la AAF (automatic Application Feature) para formularios copy ing. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copies the form values
WinCopyForm(WIN_EnterName)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinCopyForm([<Window name>])`
---

**`<Window name>: (Cadena de caracteres opcional)`**

Nombre o alias de la ventana que se va a utilizar. Si no se especifica este parámetro o si corresponde a una cadena vacía ("""), se gestiona la ventana actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los tipos de controles soportados son:

- Controles de edición,

- Botones de opción/Cajas de verificación,

- Deslizadores, deslizadores redondos,

- Calendarios, 

- Combo boxes editables.

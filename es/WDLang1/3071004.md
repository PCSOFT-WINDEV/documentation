
## CursorCapture( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dirige todas las Event s asociadas al ratón hacia una ventana o control específico, independientemente del objeto de la pantalla donde se encuentre la Event del ratón.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003071004&name=cursorcapture_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// All the mouse events are directed to the "WIN_WindowName" window
CursorCapture(True, Handle(WIN_WindowName))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CursorCapture([<Redirection>, ] <Object identifier>)`
---

**`<Result>: (Integro)`**



- Identificador (manipular) de la ventana que recibe previamente todos los mensajes del ratón,

- NULL si ninguna ventana específica estaba recibiendo los eventos del ratón (modo estándar).




**`<Redirection>: (Booleano opcional)`**

Especifica el modo de gestión de los eventos del ratón:

- <u><u><u><u>True</u></u></u></u> (valor predeterminado): los mensajes se dirigen a una ventana o a una control.

- <u><u><u><u>False</u></u></u></u>: gestión estándar de eventos.




**`<Object identifier>: (Integro)`**

Mango de la ventana o control. Este identificador es devuelto por [Handle](../WDLang1/3015005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**CursorCapture** no tiene efecto cuando se utiliza en el código de inicialización de la ventana.

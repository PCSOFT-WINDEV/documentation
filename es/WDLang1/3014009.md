
## FreeDLL( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Libera la biblioteca de 32 bits (DLL) que fue cargada en memoria por [CargaDLLL](../WDLang1/3014006.md). La memoria se libera.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
hInst = LoadDLL("MyDLL.DLL")
IF hInst = 0 THEN
	Error("Error during the unload operation")
ELSE
	CallDLL32("MyDLL", "FunctionA", par1, par2)
	CallDLL32("MyDLL", "FunctionB", par1)
	FreeDLL(hInst)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`FreeDLL(<DLL Instance>)`
---

**`<DLL Instance>: (Integro)`**

Instancia de la DLL a descargar, devuelta por [CargaDLLL](../WDLang1/3014006.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
[API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md) carga automáticamente la DLL si es necesario, y luego la descarga (si se ha cargado). Este mecanismo puede ser bastante lento, excepto para las DLLs del sistema (KERNEL, USER, GDI).

Para optimizar la velocidad de ejecución, le aconsejamos cargar una vez la DLL con la función [CargaDLLL](../WDLang1/3014006.md) y descargarla con la función**FreeDLL** cuando ya no se utilice.




## Funciones de gestión de proyectos
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para facilitar la manipular de los elementos que se encuentran en un proyecto WINDEV:




|   |   |
| --- | --- |
| [ComponenteCarga](../WDLang1/3014023.md) | Carga una Component externa. Esta Component puede ser:<br><br>- un Component externo presente en el proyecto actual. <br><br>- un Component externo independiente del proyecto en curso.<br><br><br> |
| [ComponentExecute](../WDLang1/1410089107.md) | Ejecuta una Procedure global presente en una Component externa. Esta Component externa se carga con [Component Load](../WDLang1/3014023.md). |
| [ComponentList](../WDLang1/3064007.md) | Devuelve la lista de componentes externos (archivos".WDK") utilizados por la aplicación WINDEV o WINDEV Mobile actual o por el sitio WEBDEV actual. |
| [ComponentLoadAll](../WDLang1/1000018926.md) | Recursivamente carga en memoria todos los componentes externos utilizados en el proyecto. |
| [ComponentLocate](../WDLang1/3064009.md) | Especifica el acceso ruta a un componente externo del proyecto. |
| [EnumElement](../WDLang1/3064001.md) | Se utiliza para enumerar los elementos del proyecto: ventanas, páginas, informes, consultas,.... |
| [EnumSubElement](../WDLang1/3064002.md) | Se utiliza para enumerar los subelementos de un elemento de proyecto. |
| [ListDLL](../WDLang1/3064006.md) | Devuelve:<br><br>- **en tiempo de ejecución**: la lista de bibliotecas (archivos".DLL") utilizadas por la aplicación WINDEV o WINDEV Mobile actual o por el sitio WEBDEV actual. Sólo se listan las bibliotecas cargadas en memoria.<br><br>- **en modo de prueba**: todas las bibliotecas instaladas en el ordenador actual.<br><br><br> |
| [ProjectInfo](../WDLang1/3064004.md) | Devuelve información específica sobre el proyecto en curso. |
| [WindowCount](../WDLang1/3064005.md) | Calcula el número de ventanas abiertas actualmente en la aplicación actual. |

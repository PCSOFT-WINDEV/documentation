


## &lt;Pivot tabla&gt;.ToExcel (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un fichero Excel con los datos de un control Tabla dinámica.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sFile is string

// Builds the file name
sFile = SysDir(srAppData) + ["\"] + "StatsByCountry.xls"

PVT_Statistics.ToExcel(sFile)
ShellExecute(Sysdir(srAppData) + ["\"] + "StatsByCountry.xls")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Pivot Table control>.ToExcel(<XLS file> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<XLS file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo Excel a crear. El archivo se sustituye por el nuevo si ya existe.

**`<Options>: (Constante opcional)`**

Configuración del archivo XLS a crear.


|   |   |
| --- | --- |
| *taColumnsTitles*<br>(valor predeterminado) | El título de las columnas se inserta en el archivo. |
| *taNoTitle* | Sólo se copian los datos en el archivo. |
| *taWithLayout* | Exporta el Color y la fuente del control Tabla dinámica. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los archivos de Excel no pueden exceder 65536 filas y 256 columnas si la extensión utilizada es ".xls".. Este límite puede ser excedido utilizando la extensión ".xlsx" para crear archivos en formato Excel 2007.. Atención: en este caso, las versiones anteriores de Excel deben estar equipadas con el "Compatibility Pack Office 2007" que puede descargarse gratuitamente desde el sitio de Microsoft para abrir los documentos generados.

- El archivo creado puede ser abierto por Excel 97 (o posterior) o por OpenOffice Calc 2 (o posterior).

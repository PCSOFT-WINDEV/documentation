
## Toast (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Toast** permite definir las características avanzadas de una notificación toast. Las características del toast se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyToast is Toast
MyToast.Title = "Did you know?"
MyToast.Text = "AAFs are really cool."
MyToast.Button[1].Caption = "Learn more..."
MyToast.Button[1].ActionClick = FuncHelp
MyToast.ActionClose = clickClose
ToastDisplay(MyToast)
INTERNAL PROCEDURE FuncHelp
	Open ("WIN_Help")
END
INTERNAL PROCEDURE clickClose(nButton)
	Trace("Click to close. Button: " + nButton)
END
```

```wl
// Displays a toast without the product name
MyToast is Toast
MyToast.DisplayDuration = toastLong
MyToast.VerticalAlignment = vaBottom 
MyToast.HorizontalAlignment = haCenter
ToastDisplayIW(IW_TOAST_DELETE, MyToast, PRODUCT.NAME)

//------------------------------------
// Global declarations of the internal window
PROCEDURE IW_TOAST_DELETE(ProductName)
STC_TEXT = StringBuild("The product %1 has been deleted", ProductName)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Toast
<a name="propiedades_especificas_descripcion_variables_tipo_toast_ELTPARAGRAPHE000057"></a>

Las siguientes propiedades permiten manipular un toast:

   | --- |
| --- | --- | --- |
| **Nombre de la propiedad** | Tipo utilizado | Efecto |
| ActionClose |   |   | Acción a realizar cuando se cierra la toast. Esta Property corresponde al nombre del Procedure que se ejecutará al pulsar el botón del toast. <br>Esta Procedure toma el número del botón pulsado como parámetro opcional. Si no se ha hecho clic en ningún botón para cerrar la toast (toast cerrada mediante un temporizador, clic en el toast, ventana padre cerrada, etc.), este parámetro se set a 0. |
| Button[n] |   |   | Botones mostrados en el toast. <br><br> |
|   | Button[n].ActionClick |   | Acción a realizar al hacer clic en el botón. <br>Esta Property corresponde al nombre del Procedure que se ejecutará al pulsar el botón del toast. |
|   | Button[n].Caption |   | Título del botón. |
| HorizontalAlignment |   | Constante de tipo Integer | Alineación horizontal del mensaje con respecto a la ventana: <br><br>- *haCenter* (Valor predeterminado): El mensaje Toast se muestra en el centro.<br><br>- *haRight*: El mensaje Toast se muestra a la derecha.<br><br>- *haLeft*: El mensaje Toast se muestra a la izquierda.<br><br><br><br><br> |
| VerticalAlignment |   | Constante de tipo Integer | Alineación vertical del mensaje con respecto a la ventana: <br><br>- *vaBottom* (Valor predeterminado): El mensaje Toast se muestra abajo.<br><br>- *vaTop*: El mensaje Toast se muestra arriba.<br><br>- *vaMiddle*: El mensaje Toast se muestra en el centro.<br><br><br> |
| BackgroundColor |   | Integro | Color de fondo utilizado para mostrar el mensaje. Este parámetro puede ser: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br><br> |
| DisplayDuration |   | Constante de tipo Integer | Duración de visualización del mensaje toast: <br><br>- *toastShort* (Valor predeterminado): El mensaje Toast se mostrará durante un corto período de tiempo.<br><br>- *toastLong*: El mensaje Toast se mostrará durante un largo período de tiempo.<br><br><br><br><br>**Observación**: La duración exacta de visualización depende del dispositivo utilizado. |
| CloseOnClick |   | booleano | Cierre del Toast: <br><br>- <u><u><u><u>True</u></u></u></u>: el Toast se cierra automáticamente si el usuario hace clic sobre el mensaje,<br><br>- <u><u><u><u>False</u></u></u></u>: el Toast se cierra después de la duración especificada. <br><br><br><br><br> |
| Opacidad |   | Integro | Nivel de opacidad del Toast con un valor entre 1 y 100: <br><br>- 1, Toast sin opacidad,<br><br>- 100, Toast completamente opaco. <br><br><br><br><br>De forma predeterminada, el nivel de opacidad es 0 (corresponde a una opacidad de 100). |
| Superponible |   | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> (valor predeterminado): si se muestran varios toast, se superponen entre sí. <br><br>- <u><u><u><u>False</u></u></u></u>: si se muestran varios toast, se superponen entre sí (compatible con la versión 24). <br><br><br><br><br> |
| texto |   | Cadena de caracteres | Mensaje a mostrar. |
| Título |   | Cadena de caracteres | Título del Toast. <br><br> |

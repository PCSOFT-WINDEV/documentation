
## &lt;Imagen&gt;.DrawBorder (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja los bordes en un control Imagen.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Draw a rectangle with blue borders, green background and rounded corners
// in an Image control

Rect is Rectangle
Rect.X = 10
Rect.Y = 15
Rect.Width = 100
Rect.Height = 150
BorderStyle is Border
BorderStyle.Color = LightBlue
BorderStyle.Round = 30
BorderStyle.Thickness = 1
BackgroundStyle is Background
BackgroundStyle.Color = LightGreen
IMG_Drawing.DrawBorder(Rect, BorderStyle, BackgroundStyle)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Image control>.DrawBorder(<Border> [, <Border style> [, <Background style>]])`
---

**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular.

**`<Border>: (Variable de tipo Rectangle)`**

Nombre de la variable de tipo [Rectangle](../Motscles/1514079.md) que especifica los límites de los bordes a dibujar.

**`<Border style>: (Variable de tipo Border opcional)`**

Nombre de la variable de tipo [Bordes](../Motscles/15140100.md) que describe el estilo de los bordes. 

Si no se especifica este parámetro o no se definen las propiedades **Color**, **Thickness** y **Line** de la variable de tipo [Bordes](../Motscles/15140100.md), se utiliza la información del estilo actual (definida por la función [&lt;Imagen&gt;.Pen](../WDLang1/1000024016.md)). Si no se ha utilizado la función [&lt;Imagen&gt;.Pen](../WDLang1/1000024016.md): 

- Color de los bordes: corresponde al color de fondo del rectángulo. 

- Grosor:  1 

- Línea: corresponde a la constante *LineSolid*.




**`<Background style>: (Variable de tipo Background opcional)`**

Nombre de la variable de tipo [Background](../WDLang1/1000022038.md) que describe el estilo de fondo. 

Si no se especifica este parámetro, se utiliza la información del pincel actual.


## DocDeleteTableOfContents (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una tabla de contenido de un documento de tipo Procesador de texto. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DocDeleteTableOfContents(WP_NoName1.Selection.Fragment)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DocDeleteTableOfContents(<Fragment>)`
---

**`<Fragment>: (Variable de tipo DocFragment)`**

Nombre de la variable de tipo [docFragment](../WDLang1/1000022482.md) que corresponde a la tabla de contenido a eliminar.

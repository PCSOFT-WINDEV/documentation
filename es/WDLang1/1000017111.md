
## ServiceStop( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene un servicio. Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stops the service found in TABLE_Service
IF ServiceStop(COL_Name[TABLE_Service]) = True THEN
	IF ServiceWaitStatus(COL_Name[TABLE_Service], serviceStatusStopped, 10) = -1 THEN
		 Error(ErrorInfo())
	END
ELSE
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceStop(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se envió la solicitud para interrumpir el servicio al administrador de servicios,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a interrumpir.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para detener un servicio en otro ordenador. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía, el servicio se gestiona localmente..

### Modo de funcionamiento
<a name="modo_funcionamiento_ELTPARAGRAPHE000055"></a>

La función **ServiceStop** solicita al administrador de servicios detener el servicio. Esta operación no se puede realizar inmediatamente.. Para conocer el estado del servicio, utilice [ServicioEsperandoEstado](../WDLang1/1000017101.md).

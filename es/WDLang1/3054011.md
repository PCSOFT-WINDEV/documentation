### 1. Selección de los diferentes idiomas
<a name="1_seleccion_los_diferentes_idiomas_ELTPARAGRAPHE000052"></a>


|   |   |
| --- | --- |
| <br><br>- **Proyecto**<br><br><br> | Puede seleccionar los idiomas del proyecto en el proyecto description:<br><br>1. En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".<br><br>2. Seleccione la pestaña "Idiomas" y haga clic en "Añadir".. <br><br>3. Las lenguas seleccionadas se tienen en cuenta para todos los elementos (los elementos abiertos se cierran automáticamente).<br><br><br> |
| <br><br>- **Ventanas, páginas, informes y controles (texto, máscaras de entrada e imágenes)**<br><br><br> | Los diferentes idiomas del proyecto se gestionan automáticamente. <br>Elementos compartidos entre proyectos que utilizan diferentes idiomas: cuando el elemento se abre en uno de los proyectos, las lenguas adicionales se añaden automáticamente. |
| <br><br>- **Sistema de ayuda (hlp o chm)**<br><br><br> | Los diferentes idiomas del proyecto se gestionan automáticamente. <br>Sistemas de ayuda compartidos entre proyectos que utilizan diferentes idiomas: cuando el elemento se abre en uno de los proyectos, las lenguas adicionales se añaden automáticamente. |
| <br><br>- **Mensajes en el código (código, procedimientos, clases)**<br><br><br> | Los diferentes idiomas del proyecto se gestionan automáticamente.<br>Clases y procedimientos compartidos entre proyectos: cuando el elemento se abre en uno de los proyectos, las lenguas adicionales se añaden automáticamente. . |

| <br><br>- **Mensajes en el código (código, procedimientos, clases)**<br><br><br> | La traducción debe realizarse en el editor de mensajes (Ctrl + T). |

### examinar
<a name="examinar_ELTPARAGRAPHE000243"></a>




|   |   |
| --- | --- |
| <br><br>- **Proyecto**<br><br><br> | La prueba del proyecto se ejecuta utilizando el lenguaje seleccionado como "lenguaje Default en tiempo de ejecución". El idioma por defecto utilizado en tiempo de ejecución se selecciona en la ventana del proyecto description (pestaña "Idiomas"). |
| <br><br>- **Ventanas, páginas y controles (texto, máscaras de entrada e imágenes)**<br><br><br> | La prueba de la ventana se realiza en el lenguaje Default del proyecto. |

| <br><br>- **Sistema de ayuda (hlp o chm)**<br><br><br> | Compilación y ejecución de prueba en el lenguaje de visualización del sistema de ayuda. <br>Debe crear tantos archivos de ayuda como el número de idiomas. |
| <br><br>- **Mensajes en el código (código, procedimientos, clases)**<br><br><br> | Cuando se prueba una ventana o un proyecto, los mensajes del código se muestran en el lenguaje Default del proyecto. |

En la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción".<br><br>2. Seleccione la pestaña "Idioma".<br><br>3. Seleccione el idioma deseado. <br><br>4. En la pestaña "Varios", seleccione el archivo de traducción WDM. <br><br><br> |

## Caso especial: Traduciendo las ventanas y páginas propuestas por Default
<a name="caso_especial_traduciendo_las_ventanas_paginas_propuestas_por_default_ELTTEXTE000927"></a>
Los siguientes elementos pueden ser incluidos automáticamente en sus aplicaciones o sitios: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ventana de vista previa.

- Ventanas HFSQL para automatic gestión de errores de programación HFSQL.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ventanas o páginas de groupware usuario. Estos elementos se utilizan para gestionar la conexión de un usuario a una aplicación y los diferentes niveles de acceso para cada usuario.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) ventanas de AAF (Características de la aplicación automatic). 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el programa instalación, permitiendo al usuario instalar su aplicación o su sitio.



<a name="NOTE2_2"></a>


### Incluyendo y traduciendo las ventanas y páginas propuestas por Default
<a name="incluyendo_traduciendo_las_ventanas_paginas_propuestas_por_default_ELTPARAGRAPHE000505"></a>

En la opción "Visor de reportes", seleccione "Personalizado (incluido en el proyecto)".<br><br>4. Seleccione el tipo de personalización que desea realizar:<br><br>	- &lt;Visor de reportes estándar (sin plantilla)&gt;: esta opción importa las ventanas de impresión estándar a su proyecto.<br><br>	- &lt;Visor estándar de reporte con el plantilla del proyecto&gt;: esta opción importa las ventanas de impresión correspondientes a la plantilla de su proyecto.<br><br><br><br><br><br> |

### Windows de AAF (Características de la aplicación automatic)
<a name="windows_aaf_caracteristicas_aplicacion_automatic_ELTPARAGRAPHE000756"></a>

Algunos AAF (automatic Application Features) usan ventanas específicas. Estas ventanas se pueden incluir en la aplicación a traducir. Lista de las principales características que requieren una integración específica para ser traducidas: 

- [Control Planificador](../WDChamp/1000019498.md) y [Control Agenda](../WDChamp/1016329.md): Posibilidad de personalizar y traducir la ventana de Appointment. 

- [Control Organigrama](../WDChamp/1000019822.md): Posibilidad de personalizar y traducir la ventana de elementos.

- [Bloqueo de la aplicación](../WDLang6/3061101.md): Posibilidad de personalizar y traducir la ventana de bloqueo. 

- [Gestión de los métodos abreviados de teclado](../Editeurs/9500108.md): Posibilidad de personalizar y traducir la ventana de acceso directo. 

- [Mostrando una Line de una control Campo de entrada](../Editeurs/9500120.md): Posibilidad de personalizar y traducir la ventana de entrada de una determinada Line. 

- [Definiendo una timer](../Editeurs/9500100.md): Posibilidad de personalizar y traducir la ventana de configuración de timer. 

- [Búsqueda de Multi-control](../Editeurs/9500101.md): Posibilidad de personalizar y traducir la ventana de búsqueda.

- [Historial de modificaciones](../WDLang4/3044344.md): Posibilidad de personalizar y traducir la ventana del historial. 

- [Calendar](../WDChamp/1016300.md): Posibilidad de personalizar y traducir el menú contextual de Calendar. 

- [Calculadora](../Editeurs/9500122.md): Posibilidad de personalizar la calculadora y traducir sus opciones.

- [Impresión en automatic de los controles del Scheduler](../WDChamp/1000019561.md): Posibilidad de personalizar y traducir la ventana de ajustes de impresión.

- [Impresión en automatic de los controles del Organizador](../WDChamp/1016327.md): Posibilidad de personalizar y traducir la ventana de ajustes de impresión.


## Tangent( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la tangente de un ángulo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Tangent of 45
TangentRes = Tangent(45)
// Returns 1
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Tangent(<Angle>)`
---

**`<Result>: (Real)`**

Tangente al ángulo especificado.

**`<Angle>: (Entero o real)`**

Ángulo (en grados) de uso. 
Se produce un error de WLanguage si este parámetro es set a 90°, -90°, 270° o -270°..

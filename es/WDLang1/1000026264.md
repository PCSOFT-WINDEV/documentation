


## Visualización HTML Funciones control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de visualización de HTML: 



|   |   |
| --- | --- |
| [HTMLControlValue](../WDLang1/1000025968.md) | Se utiliza para identificar o modificar una control en un formulario HTML contenido en una control de visualización HTML . |
| [HTMLExecuteBrowserProcedimiento](../WDLang1/1000025920.md) | Ejecuta un navegador WLanguage Procedure en un HTML Display control que muestra una página creada con WEBDEV. |
| [HTMLExecuteControlProcess](../WDLang1/1000025969.md) | Ejecuta un Event associated con un control en una página HTML mostrada por un control. |
| [HTMLFromEmail](../WDLang1/1410087263.md) | Importa el contenido HTML de una Email y sus imágenes en una control de visualización HTML. Las imágenes contenidas en la Email se muestran también en la control. |
| [HTMLNavigate](../WDLang1/3025042.md) | Inicia la navegación en una pantalla HTML control. |
| [HTMLOnErrorJS](../WDLang1/1000026259.md) | Especifica la Procedure a la que hay que llamar si se produce un error al ejecutar el código JavaScript en una control de visualización de HTML. |
| [HTMLValidateForm](../WDLang1/1000025970.md) | Valida un formulario en una página HTML mostrada en una pantalla HTML control. |

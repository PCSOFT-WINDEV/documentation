
## MemSeek( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>

<a name="Use"></a>
<a name="description"></a>
Busca un elemento en una zona de memoria:

- sobre el valor añadido por [MemAdd](../WDLang1/3053002.md) o sobre el valor modificado por [MemModify](../WDLang1/3053007.md),

- sobre el valor de retorno.




**Observación**: Esta búsqueda es una búsqueda por coincidencia exacta.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003053015&name=memseek_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Seeks the first value equal to "M" in the "MemZone" memory area
ResSeeek = MemSeek("MemZone", "M")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MemSeek(<Memory area> , <Sought value> [, <Element>])`
---

**`<Result>: (Cualquier tipo)`**



- Si &lt;Elemento&gt; es set a <u><u><u><u>True</u></u></u></u> (por Default): 

	- Valor del primer elemento encontrado,

	- Vaciar la cadena (""") si no se ha encontrado ningún elemento.




- Si &lt;Elemento&gt; es set a <u><u><u><u>False</u></u></u></u>: 

	- Índice del primer elemento encontrado,

	- -1 si no se ha encontrado ningún elemento.







**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria, definida por [MemCreate](../WDLang1/3053005.md).

**`<Sought value>: (Cadena de caracteres)`**

Valor buscado en la zona de memoria. Cuando la búsqueda se realiza sobre el valor añadido, este valor y el valor añadido por [MemAdd](../WDLang1/3053002.md) deben tener el mismo tipo.

**`<Element>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> (valor predeterminado): la búsqueda se realiza sobre el valor añadido por [MemAdd](../WDLang1/3053002.md) o sobre el valor modificado por [MemModify](../WDLang1/3053007.md).

- <u><u><u><u>False</u></u></u></u>: la búsqueda se realiza según el valor de retorno.

## Observaciones




### Lectura de una zona de memoria
<a name="lectura_una_zona_memoria_ELTPARAGRAPHE000126"></a>

**MemSeek** se utiliza para iniciar la lectura del área de memoria.

Tras la llamada a **MemSeek**:

- [MemNext](../WDLang1/3053017.md) se posicionará en el siguiente elemento buscado.

- [MemPrevious](../WDLang1/3053011.md) se posicionará en el elemento anterior buscado.




Para reanudar una iteración estándar, debe utilizar [MemFirst](../WDLang1/3053014.md) o [MemLast](../WDLang1/3053004.md).
<a name="NOTE0_2"></a>




### Error
<a name="error_ELTPARAGRAPHE000155"></a>

Se genera un error WLanguage cuando se ejecuta **MemSeek**:

- si &lt;área de memoria&gt; no existe. Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).

- si &lt;área de memoria&gt; no se ha ordenado previamente. Para ordenar una zona de memoria, utilice [MemSort](../WDLang1/3053020.md).

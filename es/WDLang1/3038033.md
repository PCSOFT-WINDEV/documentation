
## CurrentWin( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la ventana actualmente en edición.

**Observación**: La ventana actual es la ventana que tiene el foco: esta ventana está en edición y su barra de título es Active.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of current widow
ResCurrentWin = CurrentWin()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CurrentWin()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la ventana actualmente en edición,

- Alias de la ventana actualmente en edición (si este alias se especificó cuando se abrió la ventana).  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Caracteres en mayúsculas/minúsculas
<a name="caracteres_mayusculasminusculas_ELTPARAGRAPHE000078"></a>

El nombre de la ventana puede escribirse en mayúsculas o minúsculas.. El operador "~=" (flexible equality) debe utilizarse para comparar el nombre de la ventana devuelto por **CurrentWin** con un nombre de ventana codificado.

Por ejemplo:


```wl
IF CurrentWin() ~= "EditWindow" THEN
	...
END
```

<a name="NOTE0_2"></a>




### Ventana actual
<a name="ventana_actual_ELTPARAGRAPHE000092"></a>

El &lt;Resultado&gt; de la función **CurrentWin** depende del evento en el que se llama a la función **CurrentWin**:

| Llamando a Event | Resultado de CurrentWin. |
| --- | --- |
| La ventana obtiene el foco | Nombre de la ventana actual (ventana que toma el foco). |
| La ventana pierde el foco | Nombre de la ventana actual (ventana que pierde el enfoque). |
| Cambiar el tamaño de la ventana | Nombre de la ventana actual (la ventana se redimensiona). |
| Inicialización de la ventana | Cadena vacía (""). |
| Declaraciones globales | Nombre de la ventana anterior (ventana donde se abrirá la ventana secundaria). |
| Cerrar la ventana | Nombre de la ventana actual (ventana que se cierra). |
| Inicialización del control | <br><br>- Nombre de la ventana anterior.<br><br>- Una cadena vacía (""") si no hay ninguna ventana anterior.<br><br><br> |


Para conocer la ventana que se está ejecutando, puede utilizar la palabra clave [MiVentana](../Motscles/1511004.md).

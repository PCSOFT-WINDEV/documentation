


## Hoja de cálculoPermitirProcedimiento (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite el uso de un WLanguage Procedure en una fórmula de un control Hoja de cálculo. 

**Observación**: Por Default, los procedimientos de WLanguage no están permitidos en las fórmulas por razones de seguridad. De hecho, un usuario malintencionado podría, por ejemplo, llamar a una inicialización Procedure que restablezca todos los archivos de datos a 0.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Allows the CalculateVAT() procedure in the formulas
SpreadsheetAllowProcedure(PSHEET_Spreadsheet, "CalculateVAT" + CR + "MaxPrice" + CR + "MinPrice")
SpreadsheetAddFormula(PSHEET_Spreadsheet, 1,1, "CalculateVAT(C2)")
SpreadsheetAddFormula(PSHEET_Spreadsheet, 2,1, "(MaxPrice() + MinPrice())/2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SpreadsheetAllowProcedure(<Spreadsheet control> , <WLanguage procedure>)`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLanguage Procedure que se permite en una fórmula. Este Procedure puede ser un Procedure local o global. 
Para especificar varios procedimientos, los nombres deben estar separados por un carácter CR (Carriage Return). 
Esta Procedure debe devolver el valor para mostrarlo en la celda mediante la fórmula.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Atención**: Si se permite una Procedure de WLanguage, esta Procedure aparecerá en la lista de fórmulas propuestas al usuario. Por lo tanto, el usuario final podrá utilizar este Procedure. No olvide proporcionar una documentación sobre las posibles fórmulas y los parámetros esperados.. 

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..


## GeneratePassword( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Genera automáticamente una contraseña con características específicas (tamaño, caracteres utilizados,...). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate a password with 8 characters
sPwd is string = GeneratePassword(8)

// Generate a password containing vowels only
sPwd2 is string = GeneratePassword(8, "aeiouyAEIOUY")

// Generate a password with all the characters from the alphabet + numbers
MySet is string = "abcdefghijklmnopqrstuvwxyz0123456789"
sPwd3 is string = GeneratePassword(8, MySet)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GeneratePassword(<Length> [, <Set of characters>])`
---

**`<Result>: (Cadena de caracteres)`**



- Contraseña generada. 

- Vacíe la cadena (""") si la longitud de la contraseña corresponde a 0.




**`<Length>: (Integro)`**

Tamaño de la contraseña. Si este parámetro es set a 0, la contraseña corresponde a una cadena vacía ("").

**`<Set of characters>: (Cadena de caracteres opcional)`**

Caracteres que deben incluirse en la contraseña. 

Si no se especifica este parámetro, la contraseña generada será "pronunciable".. Esta contraseña está compuesta de caracteres: 

- entre la a y la z.

- entre la A y la Z. 

- entre 0 y 9.


## CharactUnicode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el carácter que corresponde al código Unicode especificado. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyCharacter is UNICODE string
MyCharacter = CharactUnicode(49)    // Returns "1"
MyCharacter = CharactUnicode(87)    // Returns "W"

Trace(Asc(Middle("????", 1, 1))) // Displays 35501
Trace(CaractUnicode(35501)) // ?
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CharactUnicode(<Unicode code to convert>)`
---

**`<Result>: (Cadena Unicode)`**

Carácter que corresponde al código Unicode especificado.

**`<Unicode code to convert>: (Entero)`**

Valor a convertir. Si el valor no es válido, se produce un error de WLanguage.

<a name="NOTE0_1"></a>

## Observaciones




### Evitar el uso de la función CharactUnicode
<a name="evitar_uso_funcion_docparampagetitleshort_ELTPARAGRAPHE000073"></a>

Las siguientes constantes predefinidas evitan el uso de la función **CharactUnicode**:

| Constante | Valor | Función |
| --- | --- | --- |
| CR | Charact(13) + Charact(10) | Retorno de carro |
| ESC | Charact(27) | Tecla Esc |
| SPC | Charact(160) | Espacio no separable |
| TAB | Charact(9) | Tabulador |




## &lt;GetWidgetPosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la posición de un widget que se encuentra en una control Cuadro de mando.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the position of widget 2 in number of cells
(x,y) = DASH_MyDashboard.GetWidgetPosition(2)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX2"></a>

### Recuperación de la posición de un widget en una control Cuadro de mando

`(<X>, <Y>) = <Dashboard control>.GetWidgetPosition(<Widget index> [, <Options>])`
---

**`<X>: (Integro)`**

Posición horizontal del widget en número de celdas o en píxeles.

**`<Y>: (Integro)`**

Posición vertical del widget en número de celdas o en píxeles.

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Widget index>: (Integro)`**

Índice del widget a utilizar. Este índice corresponde a:

- el número de adición del widget definido en el editor de ventanas o en el editor de páginas.

- el número de adición del widget devuelto por [&lt;Panel de control&gt;.AddWidget](../WDLang1/1000022502.md).




**`<Options>: (Constante opcional de tipo Integer)`**

Unidad utilizada:


|   |   |
| --- | --- |
| *dashUnitCell* <br>(Valor predeterminado) | La unidad utilizada para la posición del widget es un número de celdas. La primera celda es la celda [ 1, 1]. |
| *dashUnitPixel* | La unidad utilizada para la posición del widget es el píxel. |

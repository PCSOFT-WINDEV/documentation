
## ExceptionEnable( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Vuelve a habilitar la actual excepción Process si la excepción fue corregida. 

**Recordatorio**: Una excepción Process de WLanguage puede ser utilizada una sola vez (para evitar bucles interminables si la excepción no es fija). La función **ExceptionEnable** permite reactivar el proceso de excepción: otra excepción puede ser Process ed por el Process.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Re-enables the exception process
ExceptionEnable()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ExceptionEnable()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **ExceptionEnable** debe llamarse solo en un proceso de excepción (proceso desencadenado por la palabra clave: "WHEN EXCEPTION").

- La función no debe utilizarse en una Procedure o en un método llamado desde una excepción Process.

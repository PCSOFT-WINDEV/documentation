


## DateToDayInAlpha (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre del día de una fecha determinada.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DateToDayInAlpha("20011225") // Returns "Tuesday"
MyDate is Date = "20011225"
DateToDayInAlpha(MyDate)      // Returns "Tuesday"
```
<a name="Example2"></a>

```wl
MyDate is Date = DateSys()
EDT_Date = DateToDayInAlpha(MyDate)+ " " + MyDate.Day + ...
	" " + DateToMonthInAlpha(MyDate) + " at " + ...
	TimeToString(TimeSys(), "HH:MM:SS")
// The EDT_Date control contains for example: Monday 13 August at 14:44:17
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DateToDayInAlpha(<Date>)`
---

**`<Result>: (Cadena de caracteres)`**

Día de la semana que corresponde a la fecha especificada: lunes, martes, miércoles, jueves, viernes, sábado, domingo.
**Observación**: El día de la semana depende de las opciones de idioma especificadas para las fechas en la pestaña "Idiomas" de la descripción del proyecto (en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción"):

- Si el proyecto utiliza las opciones de idioma de Windows para las fechas, el día será devuelto en el idioma de Windows.

- Si el proyecto utiliza parámetros específicos para las fechas (en uno o más idiomas), el día devuelto corresponderá al que se especificó en la descripción del proyecto.




**`<Date>: (Cadena de caracteres, variable de tipo Date o DateTime)`**

Fecha a manipular. Esta fecha puede corresponder a : 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  

- una cadena en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo (utilice CCC para las milésimas de segundo)). En este caso, solo se tiene en cuenta la fecha.

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999.

Los cálculos sobre las fechas realizadas con las [funciones WLanguage](../WDLang1/3027029.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.

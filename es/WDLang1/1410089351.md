
## Canada.CheckSIN (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número SIN (número de seguridad social canadiense). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MySIN is string = "046454286"
Canada.CheckSin(MySIN)		// Returns True

// Check a control
EDT_SIN = "ABC"
Canada.CheckSIN(EDT_SIN)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Canada.CheckSIN(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número de seguridad social válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número de seguridad social a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.

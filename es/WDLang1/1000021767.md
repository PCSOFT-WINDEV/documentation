


## SysIconDisplayTooltip( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un tooltip para un Icon que se encuentra en la barra de tareas. Esta Icon debe haber sido añadida en la barra de tareas por [SysIconAdd](../WDLang1/3073001.md).

Observación: Desde Windows 10, el centro Notification integrado en Windows muestra el tooltip como un nuevo Notification.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Informs that the process has ended
SysIconDisplayTooltip("The process is ended.")
```

```wl
// Display a warning with a click procedure
SysIconDisplayTooltip("The process is quite long. " + ...
	"Click the tooltip to see the details of operations.", ...
	sysTooltipTypeWarning, "", ProcedureDetails)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysIconDisplayTooltip(<Message> [, <Title> [, <Type> [, <Icon> [, <WLanguage procedure>]]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se mostró el tooltip,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Message>: (Cadena de caracteres)`**

Texto mostrado en el tooltip (hasta 255 caracteres).

**`<Title>: (Cadena de caracteres opcional)`**

Título de la información sobre la herramienta (hasta 63 caracteres).

**`<Type>: (Constante opcional de tipo Integer)`**

Tipo de información sobre la herramienta : 



|   |   |
| --- | --- |
| *sysTooltipTypeWarning* | Herramienta de advertencia. Una advertencia Icon se muestra en el tooltip (panel amarillo). |
| *sysTooltipTypeDefault*<br>(Valor predeterminado) | Tooltip mostrado por Default (que significa el tooltip definido por [SysIconAdd](../WDLang1/3073001.md)). |
| *sysTooltipTypeError* | Sugerencia de error. Un icono de error se muestra en el tooltip (cruz de White en rojo Background). |
| *sysTooltipTypeInfo* | Info tooltip. Una información Icon se muestra en el tooltip (White "i" en Background azul). |
| *sysTooltipTypeCustom* | Tooltip personalizado: el parámetro &lt;Icon&gt; se utiliza para definir el Icon que se muestra en el tooltip. |



**`<Icon>: (Cadena de caracteres opcional)`**

Ícono (16x16) que se muestra en el tooltip si el valor del parámetro &lt;Tipo&gt; es *sysTooltipTypeCustom*. 
Corresponde a:

- el nombre y la ruta completa (o relativa) del archivo Icon para añadir. Se permite el uso de un horario UNC ruta.

- el nombre de un Icono Variable.


Este parámetro corresponde a una cadena vacía ("") de Default. 

**`<WLanguage procedure>: (Nombre opcional de Procedure)`**

WLanguage Procedure que será llamado durante el clic en la punta de la herramienta (Procedure sin parámetro). Este parámetro se tiene en cuenta desde Windows XP.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La punta de la herramienta se cierra cuando se hace clic en ella.

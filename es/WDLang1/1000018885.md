


## OrganizerDisplayNextPeriod (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve el área de tiempo que se muestra a la siguiente área (por ejemplo, las próximas semanas) en un control Agenda. El control Agenda se visualiza a partir del día correspondiente al área de tiempo siguiente. 

**Observaciones**: 

- Las funciones **OrganizerDisplayNextPeriod** y [OrganizerDisplayPreviousPeriod](../WDLang1/1000018886.md) permiten establecer fácilmente los botones "Adelante/Atrás". Estos controles le permiten desplazarse por las áreas de tiempo que muestra la pantalla de control Agenda. 

- El periodo mostrado también puede ser modificado por [OrganizerChangeMode](../WDLang1/1000019497.md) y [OrganizerPosition](../WDLang1/1000018889.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Switch to monthly display
OrganizerChangeMode(ORG_MyOrganizer, orgzMonth)
// Display the current month
OrganizerPosition(ORG_MyOrganizer, DateSys())
// Switch to the next month
OrganizerDisplayNextPeriod(ORG_MyOrganizer)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OrganizerDisplayNextPeriod(<Organizer control>)`
---

**`<Organizer control>: (Nombre del control)`**

Nombre de la control Agenda que se va a utilizar. Este control puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una control Agenda en una ventana.

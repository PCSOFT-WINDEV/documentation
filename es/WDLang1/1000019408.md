
## ExceptionDisplay( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra la ventana estándar de excepciones con el contenido de la excepción actual.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WHEN EXCEPTION IN
	Proc()
DO
	// Display the exception
	ExceptionDisplay()
	// Specific code after the display
	...
	// End of application
	EndProgram()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ExceptionDisplay()`
---

**`<Result>: (Constante de tipo Integer)`**

Operación realizada en la ventana por el usuario: 


|   |   |
| --- | --- |
| *edEndProgram* | El usuario no ha comprobado la opción "Reiniciar la aplicación". |
| *edRelaunchProgram* | El usuario ha comprobado la opción"Reiniciar la aplicación". |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La ventana estándar de excepciones sólo se visualiza en modo de test. La ventana no se muestra durante la ejecución del sitio. En este caso, &lt;Resultado&gt; se establece en *edEndProgram*.

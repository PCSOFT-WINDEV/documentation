
## dHalo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Agrega un efecto de halo a una imagen en una variable de tipo [Image](../WDLang1/1000019650.md). 

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is Image
MyImage = "image.png"

// Adds a 5-pixel wide yellow halo
dHalo(MyImage, LightYellow, 5)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`dHalo(<Image> , <Halo color> [, <Thickness> [, <Clipping>]])`
---

**`<Image>: (Variable de tipo Image)`**

Nombre de la variable de tipo [Image](../WDLang1/1000019650.md) que corresponde a la imagen a manipular. 

**`<Halo color>: (Entero)`**

Color del halo a agregar a la imagen. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Thickness>: (Entero opcional)`**

Grosor del halo en pixeles (2 pixeles por defecto). 

**`<Clipping>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si se debe recortar el halo,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) en caso contrario.

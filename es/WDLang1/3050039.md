
## FinRepayVal( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el importe de cada pago regular para una inversión con tipo de interés fijo y pago fijo.

**Observación**: El valor devuelto por **FinRepayVal** incluye el pago del principal y los intereses.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Amount of regular payments. The initial capital is set to 100000 Euros.
// The payments are paid each month during 2 years.
// The interest rate is set to 8%.
ResRepayVal = FinRepayVal(100000, 24, 8)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinRepayVal([<Initial capital> [, <Number of periods> [, <Rate>]]])`
---

**`<Result>: (Real o moneda)`**

Importe de los pagos regulares.

**`<Initial capital>: (Real o moneda opcional)`**

Capital inicial (0 por Default). Este parámetro indica la cantidad sobre la que se debe realizar el cálculo.

En el caso "estándar" de un prestatario, este capital se toma prestado y por lo tanto corresponde a un valor negativo.

Si este parámetro es positivo, el cálculo se realiza para un prestamista. El valor devuelto es negativo: este valor corresponde al importe que debe pagarse.

**`<Number of periods>: (Entero opcional)`**

Número total de períodos para la operación (1 de Default).

**`<Rate>: (Real opcional)`**

Tasa de interés de la inversión expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set al 5.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Unidad de tiempo
<a name="unidad_tiempo_ELTPARAGRAPHE000081"></a>

Los cálculos se pueden realizar según meses o años. Los valores de &lt;Tarifa&gt; y &lt;Período&gt; deben corresponder a la misma unidad.

Por lo tanto, para los pagos mensuales de un préstamo de 4 años con una tasa de interés anual set a 12%, uso:

- &lt;Rate&gt; = 12% / 12, 

- &lt;Número de períodos&gt; = 4\*12




Para el mismo préstamo con pagos anuales, use:

- &lt;Tasa&gt; = 12%, 

- &lt;Número de períodos&gt; = 4




Para todos los parámetros, los pagos están representados por un número negativo mientras que los cobros están representados por un número positivo.
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000101"></a>

**Atención**: La función **FinRepayVal** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

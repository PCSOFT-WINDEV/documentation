
## Constantes para la gestión de eventos de Windows
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
 
Las siguientes constantes se utilizan para administrar los eventos de Windows:

| Constante | Detalles | Función |
| --- | --- | --- |
| **HandleHScrollbar** | Mango de la barra de desplazamiento horizontal de la control. | [Handle](../WDLang1/3015005.md) |
| **HandleVScrollbar** | Mango de la barra de desplazamiento vertical de la control. | [Handle](../WDLang1/3015005.md) |
| **HandleNativeControl** | El nativo iOS control associated con el control especificado: control Mapa (MKMapView) y HTML control (UIWebView). | [Handle](../WDLang1/3015005.md) |
| **HandleComboBoxList** | Maneje la lista desplegable associated con un cuadro combinado (el control debe ser un control Combo Box). | [Handle](../WDLang1/3015005.md) |
| **HandleAutocompleteList** | Devuelve la manipular de la lista desplegable associated con un control que tiene la función de autocompletar. | [Maneje](../WDLang1/3015005.md) |
| **HandleComboBoxEdit** | Manejar el control Campo de entrada associated con una caja combo (el control debe ser un control Combo Box). | [Maneje](../WDLang1/3015005.md) |
| **HandleUIViewController** | Devuelve el UIViewController (iOS) associated con el control especificado. | [Maneje](../WDLang1/3015005.md) |
| **waitRedraw** | Tipo de Event que se puede ejecutar durante la espera: Las ventanas y los controles pueden ser redibujados. | [esperar](../WDLang1/3015012.md) |
| **waitNothing** | Tipo de Event que se puede ejecutar durante la espera: No se puede dirigir ningún Event. | [esperar](../WDLang1/3015012.md) |
| **waitMouseAndKeyboard** | Tipo de Event que se puede ejecutar durante la espera:<br><br>- Las ventanas y los controles pueden ser redibujados.<br><br>- Los temporizadores pueden funcionar.<br><br>- Se pueden ejecutar las acciones del ratón (por ejemplo, hacer clic en los controles) o los eventos del teclado.<br><br><br> | [esperar](../WDLang1/3015012.md) |
| **waitTimer** | Tipo de Event que se puede ejecutar durante la espera:<br><br>- Las ventanas y los controles pueden ser redibujados.<br><br>- Los temporizadores pueden funcionar.<br><br><br> | [esperar](../WDLang1/3015012.md) |


## LinkColorRTF( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cambia la Color de los enlaces en todos los controles de edición RTF de la aplicación. Por defecto, los enlaces en los controles de edición de RTF son azules, o utilizan la Color especificada con [LinkColor](../WDLang1/1000022354.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Customize color of links (Edit and Static controls)
LinkColor(RGB(250,80,84), RGB(255,90,89), RGB(250,70,84))
// Customize color of links in RTF Edit controls
LinkColorRTF(RGB(25,80,84), RGB(25,90,89), RGB(25,70,84))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`LinkColorRTF(<Default Color> [, <Rollover Color> [, <Clicked link color> [, <Color of Visited Link>]]])`
---

**`<Default Color>: (Entero o constante de tipo Integer)`**

Color de enlace por defecto. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md). 


Si este parámetro corresponde a la constante *DefaultColor*, se restauran los colores del estilo inicial. 

**`<Rollover Color>: (Entero o constante de tipo Integer opcional)`**

Color del eslabón durante la prórroga. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md). 


Si no se especifica este parámetro, el color del enlace al pasar el cursor es el &lt;Color predeterminado&gt;.

**`<Clicked link color>: (Entero o constante de tipo Integer opcional)`**

Color de enlace durante el clic. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md). 


Si no se especifica este parámetro, el color del enlace en el que se hace clic es el &lt;Color predeterminado&gt;.

**`<Color of Visited Link>: (Entero o constante de tipo Integer opcional)`**

Color de enlace ya visitado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md). 


Si no se especifica este parámetro, el color del enlace visitado es el &lt;Color predeterminado&gt;.

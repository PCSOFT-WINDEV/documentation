
## Constantes del sensor
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>






| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **snrAzimuth** | Notification cuando la orientación cambia alrededor del eje vertical (Z). | [SensorDetectChangeOrientation](../WDLang1/1000019361.md), [SensorGetOrientation](../Editeurs/2030039.md) |
| **snrFrequencyGame** | Frecuencia de Notification requerida cuando se usa la función en un juego. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md), [SensorDetectChangeOrientation](../WDLang1/1000019361.md) |
| **snrFrequencyNormal** | Frecuencia estándar de Notification. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md), [SensorDetectChangeOrientation](../WDLang1/1000019361.md) |
| **snrFrequencyQuick** | Frecuencia rápida de Notification. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md), [SensorDetectChangeOrientation](../WDLang1/1000019361.md) |
| **snrLateral** | Notification cuando el dispositivo acelera sobre el eje horizontal. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md) |
| **snrLongitudinal** | Notification cuando el dispositivo acelera en el eje longitudinal. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md) |
| **snrPitch** | Notification cuando la orientación cambia alrededor del eje horizontal (X). | [SensorDetectChangeOrientation](../WDLang1/1000019361.md), [SensorGetOrientation](../Editeurs/2030039.md) |
| **snrRoll** | Notification cuando la orientación cambia alrededor del eje longitudinal (Y). | [SensorDetectChangeOrientation](../WDLang1/1000019361.md), [SensorGetOrientation](../Editeurs/2030039.md) |
| **snrSensitivityHigh** | Alta sensibilidad a los gestos de agitación. Incluso una baja sacudida del dispositivo disparará el Notification. | [SensorDetectBeginShake](../WDLang1/1000019363.md), [SensorDetectEndShake](../WDLang1/1000019364.md) |
| **snrSensitivityLow** | Baja sensibilidad a los gestos de agitación. Sólo una fuerte sacudida del aparato disparará el Notification. | [SensorDetectBeginShake](../WDLang1/1000019363.md), [SensorDetectEndShake](../WDLang1/1000019364.md) |
| **snrSensitivityNormal** | Sensibilidad estándar a los gestos de agitación. | [SensorDetectBeginShake](../WDLang1/1000019363.md), [SensorDetectEndShake](../WDLang1/1000019364.md) |
| **snrVertical** | Notification cuando el dispositivo acelera sobre el eje vertical. | [SensorDetectChangeAcceleration](../WDLang1/1000019362.md) |

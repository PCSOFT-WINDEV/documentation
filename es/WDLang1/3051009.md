
## MCIStatus( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de un archivo multimedia que fue abierto previamente ([MCIOpen](../WDLang1/3051017.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MCIOpen("Sound.wav", "MySound")
...
IF MCIStatus("MySound") = MciStatusPause THEN
	MCIBeginning("MySound")
	MCIPlay("MySound")
END
...
MCIClose("MySound")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MCIStatus(<Media file or alias> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (Constante de tipo Character String)`**

Estado del archivo (véase más abajo el tabla) o cadena vacía si se ha producido un error.. Para get el número de error, utilice [MCIErr](../WDLang1/3051011.md). Para obtener más detalles sobre el error, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
El estado reporte del comando es devuelto por [MCIRetrieve](../WDLang1/3051022.md).


|   |   |
| --- | --- |
| *MciStatusNotReady* | La unidad no está lista (por ejemplo, no hay ningún CD en la unidad) |
| *MciStatusOpen* | Archivo abierto solamente. |
| *MciStatusParked* | El jugador está estacionado: las cabezas de los jugadores aparcadas. |
| *MciStatusPause* | El archivo está en pausa |
| *MciStatusPlay* | El archivo se abre y se reproduce actualmente |
| *MciStatusRecord* | El archivo está grabado actualmente |
| *MciStatusSeek* | Buscar/buscar en curso en el archivo |
| *MciStatusStop* | El archivo se detiene |



**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario. Estos parámetros pueden conocerse consultando la documentación de la función Windows MCI denominada "status"..
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.




## &lt;Document Variable&gt;.RedoCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de acciones canceladas que se pueden rehacer en una Variable de tipo [Document](../WDLang1/1000022461.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

// Displays the number of canceled actions that can be redone
Info(MyDocument.RedoCount())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Document>.RedoCount()`
---

**`<Result>: (Integro)`**

Número de acciones canceladas que se pueden rehacer.

**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.




## Continent (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **Continent** permite conocer las características de un continente. Las características de este continente se pueden: 

- obtener con las funciones [ContinentGet](../WDLang1/1000024933.md) o [ContinentList](../WDLang1/1000024932.md). 

- conocer con diferentes propiedades WLanguage. 




**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve information about Europe
oContinent is Continent = ContinentGet(continentEurope)
// Display the caption 
STC_Continent = oContinent.Name
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo Continent
<a name="propiedades_especificas_descripcion_variables_tipo_continent_ELTPARAGRAPHE000048"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **Continent**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Code | Constante de tipo Character String | Constante que identifica el continente:<br><br>- *continentAfrica*: África (la constante vale "AF")<br><br>- *continentNorthAmerica*: América del Norte (la constante vale "NA")<br><br>- *continentSouthAmerica*: América del Sur (la constante vale "SA")<br><br>- *continentAntarctica*: Antártida (la constante vale "AN")<br><br>- *continentAsia*: Asia (la constante vale "AS")<br><br>- *continentEurope*: Europa (la constante vale "EU")<br><br>- *continentUndefined*: El continente no pudo ser identificado. <br><br>- *continentOceania*: Oceanía (la constante vale "OC")<br><br><br><br><br>**Esta propiedad es de solo lectura**. |
| Name | Character string | Nombre del continente (en el idioma del framework). <br>**Esta propiedad es de solo lectura**. |


<a name="NOTE0_2"></a>


### Funciones WLanguage que manipulan variables de tipo Continent
<a name="funciones_wlanguage_que_manipulan_variables_tipo_continent_ELTPARAGRAPHE000087"></a>




|   |   |
| --- | --- |
| [ContinentGet](../WDLang1/1000024933.md) | Devuelve la variable de tipo [Continent](../WDLang1/1000024919.md) que corresponde a uno de los siguientes valores: <br><br>- una constante de continente (código de continente de 2 caracteres), <br><br>- un nombre de continente.<br><br><br> |
| [ContinentList](../WDLang1/1000024932.md) | Devuelve la lista de continentes. |
| [CountryList](../WDLang1/1000024930.md) | Devuelve la lista de todos los países o países de un continente determinado según la norma ISO 3166-1. |


## Move (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite:

- mover un elemento en un array WLanguage unidimensional o en una propiedad array de un tipo avanzado (array de eventos de gglCalendar, etc.). Los otros elementos del array se desplazan.

- intercambiar dos elementos en un array WLanguage unidimensional o en una propiedad array de un tipo avanzado (array de eventos de gglCalendar, etc.).




**Observación**: Esta función es equivalente a la función [ArrayMove](../WDLang1/3075011.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyArray is array of 6 strings
MyArray[1] = "A"
MyArray[2] = "B"
MyArray[3] = "C"
MyArray[4] = "D"
MyArray[5] = "E"
MyArray[6] = "F"
// Move the element 2 to 5th position
Move(MyArray, 2, 5)
// The array contains the following elements: "A", "C", "D", "E", "B", "F"
```


<a name="Example2"></a>



```wl
MyArray is array of 6 strings
MyArray[1] = "A"
MyArray[2] = "B"
MyArray[3] = "C"
MyArray[4] = "D"
MyArray[5] = "E"
MyArray[6] = "F"
// Swap the element 2 with element 5
Move(MyArray, 2, 5, amSwap)
// The array contains the following elements: "A", "E", "C", "D", "B", "F"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Mover un elemento en un array WLanguage o en la propiedad array de un tipo avanzado

`Move(<Array name> , <Start index> , <End subscript> [, <amMove>])`
---

**`<Array name>: (Array WLanguage)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Start index>: (Entero)`**

Índice del elemento que se desplazará.

**`<End subscript>: (Entero)`**

Nuevo índice del elemento movido.

**`<amMove>: (Constante opcional)`**

*amMove*: Constante utilizada para mover un elemento en un array WLanguage.  


<a name="SYNTAX2"></a>

### Intercambiar dos elementos en un array WLanguage o en la propiedad array de un tipo avanzado

`Move(<Array name> , <Index of 1st element> , <Index of 2nd element> , <amSwap>)`
---

**`<Array name>: (Array WLanguage)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

**`<Index of 1st element>: (Entero)`**

Índice del primer elemento a intercambiar.

**`<Index of 2nd element>: (Entero)`**

Índice del segundo elemento a intercambiar.

**`<amSwap>: (Constante)`**

*amSwap*: Constante utilizada para intercambiar dos elementos en un array WLanguage.

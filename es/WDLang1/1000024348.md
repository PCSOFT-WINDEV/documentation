


## &lt;Variable Image&gt;.DominantImageColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el color dominante (es decir, el más frecuente) de una imagen de un control Imagen.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyColor is Color = IMG_MyImage.DominantImageColor()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Image variable>.DominantImageColor()`
---

**`<Result>: (Entero)`**

Color dominante de la imagen.

**`<Image variable>: (Image variable)`**

Nombre del control Imagen a manipular.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los pixeles transparentes o semitransparentes de la imagen se ignoran.

- La función **&lt;Variable Image&gt;.DominantImageColor** no puede utilizarse en imágenes vectoriales (SVG, PDF, etc.).

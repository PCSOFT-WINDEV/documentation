
## AnimationEnabled( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Habilita o deshabilita las animaciones en los controles.

**Observación**: Sólo se ven afectadas las animaciones de los controles: animaciones automatic (configuradas en el proyecto o en la ventana) o programadas (con [AnimationPrepare](../WDLang1/1000017386.md), [AnimaciónJuego](../WDLang1/1000017387.md), ...). Las animaciones de los controles Image y de los controles estáticos, así como las animaciones de las ventanas no se ven afectadas..  
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The animations will never be enabled
AnimationEnabled(False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AnimationEnabled([<Activation mode>])`
---

**`<Result>: (booleano)`**

Corresponde al antiguo modo de gestión de la animación control (antes de utilizar **AnimationEnabled**): 

- <u><u><u><u>True</u></u></u></u> si la gestión de las animaciones estaba activa,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Activation mode>: (Booleano opcional)`**

Modo de activación de las animaciones:



|   |   |
| --- | --- |
| *False* | Las animaciones nunca se muestran. |
| *True* | Las animaciones están en modo automatic. La animación se realiza según la potencia de el equipo utilizada.<br>Las animaciones no serán mostradas si las APIs requeridas por las animaciones no se encuentran en el equipo. |

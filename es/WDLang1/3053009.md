
## MemOut( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>20</B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [Associative arrays](../Motscles/1514058.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Se utiliza para saber si la exploración realizada en una zona de memoria está fuera de esta zona de memoria.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MemFirst("MyMemoryZone")
// Browse outside "MyMemoryZone"?
WHILE MemOut("MyMemoryZone") = False
	// Processes in the memory area
	...
	// End of processes
	MemNext("MyMemoryZone")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MemOut(<Memory area>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha alcanzado el final de la zona de memoria, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Memory area>: (Cadena de caracteres)`**

Nombre de la zona de memoria, definida por [MemCreate](../WDLang1/3053005.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Llamada a MemOut
<a name="llamada_docparampagetitleshort_ELTPARAGRAPHE000087"></a>

**MemOut** debe utilizarse después de la llamada a [MemSeek](../WDLang1/3053015.md), [MemPrevious](../WDLang1/3053011.md), [MemFirst](../WDLang1/3053014.md), [MemLast](../WDLang1/3053004.md) y [MemNext](../WDLang1/3053017.md) para comprobar si se ha alcanzado el final de la zona de memoria.
<a name="NOTE0_2"></a>


### Existencia de zona de memoria
<a name="existencia_zona_memoria_ELTPARAGRAPHE000240"></a>

No se comprueba la existencia de la zona de memoria.

Si la zona de memoria especificada no existe, se produce un error WLanguage cuando se utiliza **MemOut**. 

Para comprobar la existencia de una zona de memoria, utilice [MemExist](../WDLang1/3053006.md).

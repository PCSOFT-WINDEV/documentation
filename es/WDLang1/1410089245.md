
## Brazil.CheckCPF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número CPF (número de identificación fiscal en Brasil).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyCPFNumber is string = "16650440396"
Brazil.CheckCPF(MyCPFNumber)	// Returns True

// Check a control
EDT_CPFNumber = "ABC"
Brazil.CheckCPF(EDT_CPFNumber)	// Returns False
```



```wl
// Check a variable
MyCPFNumber is string
MyResult is boolean
Input("Enter the CPF number: ", MyCPFNumber)
IF MyResult = False THEN
	Info("Invalid CPF number")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Brazil.CheckCPF(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número CPF válido, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número CPF a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.


## FinCurrentVal( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el valor actual de la inversión.

**Recordatorio**: El valor actual corresponde a la suma que representa un set de pagos futuros en la fecha de hoy. Por ejemplo, cuando get un préstamo, la cantidad del préstamo representa el valor actual para el prestamista.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Current value of investment. The repayment value is set to 5000 Euros.
// This investment must bring in 50000 Euros per year during 10 years.
// The interest rate is set to 8%.
ResCurrentVal = FinCurrentVal(-5000, 50000, 10, 8)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinCurrentVal([<Payment Value> [, <Future Value> [, <Number of periods> [, <Rate> [, <Start>]]]]])`
---

**`<Result>: (Real o moneda)`**

Valor actual de la inversión.

**`<Payment Value>: (Real o moneda opcional)`**

Valor de pago para cada período (0 por Default). Este pago es idéntico durante toda la duración. Este valor debe ser negativo.

**`<Future Value>: (Real o moneda opcional)`**

Valor futuro (valor capitalizado). Esta es la cantidad después del último pago (0 por Default).

**`<Number of periods>: (Entero opcional)`**

Número total de períodos en la inversión (1 por Default).

**`<Rate>: (Real opcional)`**

Tasa de interés de la inversión expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set a 5.

**`<Start>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si los pagos deben realizarse al inicio del período (valor predeterminado),

- <u><u><u><u>False STRIMEULB</u></u></u></u> si los pagos deben efectuarse al final del período.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Unidad de tiempo
<a name="unidad_tiempo_ELTPARAGRAPHE000086"></a>

Los cálculos se pueden realizar según meses o años. Los valores de &lt;Tarifa&gt; y &lt;Período&gt; deben corresponder a la misma unidad.

Por lo tanto, para obtener el valor actual de un préstamo a 4 años, pagado mensualmente, con una tasa de interés anual set a 10%, las unidades pueden ser expresadas en meses:

- &lt;Período&gt; = 4\*12, 

- &lt;Rate&gt; = 10% / 12



<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000099"></a>

**Atención**: La función **FinCurrentVal** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

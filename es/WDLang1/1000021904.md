
## WinEdUndo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cancela la última acción realizada por el usuario final al modificar la interfaz (Modo "Modificación de UI por el usuario final" activado). Esta función equivale a un "Deshacer" (Ctrl + Z) realizado por el usuario.. 

Esta función se utiliza principalmente cuando se crea un editor mediante programación (véase "Observaciones").
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// [Cancel] button
WinEdUndo(WIN_Form)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinEdUndo(<Window>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. Esta ventana debe estar en modo de modificación ([WinEdActive](../WDLang1/1000021843.md)). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000037"></a>

- **WinEdUndo** no tiene efecto si no hay ninguna acción para Cancel.

- Si la última acción es una acción personalizada ejecutada por [WinEdDo](../WDLang1/1000021897.md), se ejecuta el procedimiento definido por [WinEdDeclareUndo](../WDLang1/1000021898.md).

- La acción cancelada puede ser restaurada: 

	- por Ctrl + Y,

	- mediante la programación con [WinEdRedo](../WDLang1/1000021905.md). 






<a name="NOTE0_2"></a>


### Ejemplo de uso
<a name="ejemplo_uso_ELTPARAGRAPHE000060"></a>

[WinEdDo](../WDLang1/1000021897.md) se utiliza para crear acciones personalizadas (creación de control, por ejemplo) que admiten la función de deshacer/rehacer.

Veamos un ejemplo simple: creación de un editor de objetos gráficos. 
 
Un botón se utiliza para crear una forma. El usuario debe tener la capacidad de Cancel la creación de la forma. Se realizan las siguientes operaciones: 

- La ventana pasa al modo de edición ([WinEdActive](../WDLang1/1000021843.md)). 

- La función [WinEdDo](../WDLang1/1000021897.md) se llama. La Procedure llamada por esta función se utiliza para crear la forma. Esta Procedure también contiene la llamada a [WinEdDeclareUndo](../WDLang1/1000021898.md). [WinEdDeclareUndo](../WDLang1/1000021898.md) se utiliza para definir el comportamiento en caso de cancelación del usuario. 

- La ventana pasa al modo estándar ([WinEdActive](../WDLang1/1000021843.md)). 




El usuario final puede deshacer esta acción (Ctrl + Z o **WinEdUndo**) y eventualmente rehacerla (Ctrl + Y o [WinEdRedo](../WDLang1/1000021905.md)).

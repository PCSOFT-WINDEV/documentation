


## AssistedInputAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una fila a la lista de valores disponibles en las sugerencias de autocompletar de un Editar control. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
AssistedInputDeleteAll(EDT_Direction)
AssistedInputAdd(EDT_Direction, "Left")
AssistedInputAdd(EDT_Direction, "Left to the top")
AssistedInputAdd(EDT_Direction, "Left to the bottom")
AssistedInputAdd(EDT_Direction, "Right")
AssistedInputAdd(EDT_Direction, "Right to the top")
AssistedInputAdd(EDT_Direction, "Right to the bottom")
AssistedInputAdd(EDT_Direction, "Center")
AssistedInputOpen(EDT_Direction)
```



```wl
AssistedInputAdd(EDT_Edit1, "Displayed value" + gLink("Stored value"))
Trace(EDT_Edit1.StoredValue) 
// Returns "stored value" if the row is selected

AssistedInputAdd(EDT_Edit1, "Displayed value", "Stored value") 
Trace(EDT_Edit1.StoredValue) 
// Returns "stored value" if the row is selected
```



```wl
// Add a suggestion
_AutocompleteSuggestion is AssistedInput
_AutocompleteSuggestion.Caption = "Alpharetta, GA 30022"
_AutocompleteSuggestion.StoredValue = 30022
AssistedInputAdd(EDT_PostalCode, _AutocompleteSuggestion)

// -- Event "Select a value in the list of input suggestions"
PROCEDURE Selection(MySelection is AssistedInput)
EDT_PostalCode = MySelection.StoredValue
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Añadir una fila a la lista de valores disponibles en las sugerencias de autocompletar

`<Result> = AssistedInputAdd(<Edit control> , <Text> [, <Stored value>])`
---

**`<Result>: (Integro)`**

Número de la fila añadida. 

**`<Edit control>: (Nombre del control)`**

Nombre de la Edición control a utilizar. La opción de "entrada asistida" se activa automáticamente en este control.

**`<Text>: (Cadena de caracteres)`**

Valor que debe añadirse a las sugerencias de autocompletar. 

**`<Stored value>: (Tipo de valor)`**

Valor almacenado associated con el valor añadido.




<a name="SYNTAX2"></a>

### Añadir un fila (AssistedInput Variable) a la lista de valores disponibles

`<Result> = AssistedInputAdd(<Edit control> , <Autocomplete suggestion>)`
---

**`<Result>: (Integro)`**

Número de la fila añadida. 

**`<Edit control>: (Nombre del control)`**

Nombre de la Edición control a utilizar. La opción de "entrada asistida" se activa automáticamente en este control.

**`<Autocomplete suggestion>: (AssistedInput Variable)`**

Nombre de la [AssistedInput](../WDLang1/1410088983.md) Variable que contiene los datos de la sugerencia de autocompletar que se va a añadir.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para abrir la lista con las distintas opciones:  

- el usuario puede introducir un valor en la control Campo de entrada (empezando por las letras de una de las opciones disponibles). 

- programáticamente, puede utilizar [AssistedInputOpen](../WDLang1/1000020048.md). Se recomienda filtrar la lista de valores y abrirla desde la Event "Siempre que se modifique" de la control Campo de entrada.

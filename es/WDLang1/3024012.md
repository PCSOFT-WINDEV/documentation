


## Comprimir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprime una cadena o un bloque de memoria (buffer) en formato binario.. A continuación, la descompresión será realizada por [Descomprimir](../WDLang1/3024009.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Compresses then decompresses a string
Comp is Buffer = Compress("String to compress")
Info(Uncompress(Comp))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Compress(<Character String/Buffer> [, <Type of Compression>])`
---

**`<Result>: (Buffer)`**



- Búfer comprimido. 

- Cadena vacía ("") si se produce un error. [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.




**`<Character String/Buffer>: (Cadena de caracteres o memoria intermedia)`**



- Cadena de caracteres para comprimir.

- Tampón para comprimir.




**`<Type of Compression>: (Constante opcional de tipo Integer)`**

Tipo de compresión a realizar: 


|   |   |
| --- | --- |
| *compressGZIP* | Compresión con el algoritmo GZIP. |
| *compressLZW*<br>(Valor predeterminado) | Compresión con el algoritmo LZW.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No disponible. |
| *compressNone* | No se realiza ninguna compresión.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No disponible. |
| *compressShortString* | Compresión mediante un algoritmo optimizado para las cadenas de caracteres cortos.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No disponible. |


![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Este parámetro es obligatorio.

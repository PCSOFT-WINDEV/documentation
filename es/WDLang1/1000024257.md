


## &lt;Tabla&gt;.ColumnIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de una columna en una Tabla o Treeview control Tabla.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
// Index of "Ref" column in the "TABLE_Product" control
// The display order is taken into account
ResColumnIndex = TABLE_Product.ColumnIndex("Ref", tcpDisplay)
```


<a name="Example2"></a>

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Table control>.ColumnIndex(<Column> , <Order>)`
---

**`<Result>: (Integro)`**

Índice de la columna: 

- El índice de la primera columna es set a 1. 

- Las columnas de Invisible también están listadas.




**`<Table control>: (Nombre del control)`**

Nombre del control Tabla o Tabla TreeView a manipular.

**`<Column>: (Cadena de caracteres)`**

Nombre de la columna a utilizar.

**`<Order>: (Constante de tipo Integer)`**

Especifica el orden de las columnas relevantes:


|   |   |
| --- | --- |
| *tcpCreation* | Orden en el que se han creado las columnas al describir el control Tabla. |
| *tcpDisplay* | Orden en el que se visualizan las columnas en tiempo de ejecución. |

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000252"></a>

La función **&lt;Tabla&gt;.ColumnIndex** puede utilizarse en:

- controles Tabla o Tabla TreeView basados en un archivo de datos.

- controles Tabla o Tabla TreeView rellenados mediante programación.

- controles de selección única o múltiple.

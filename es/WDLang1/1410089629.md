


## &lt;Kanban&gt;.ListIndex (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el índice de una lista de un control Kanban. 

Observación: La visibilidad de las listas no se tiene en cuenta.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Index is int 
Index = KANBAN_MyKanban.ListIndex(LIST_To_Review)
IF Index = 1 THEN
	KANBAN_MyKanban.AddList("To start")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Kanban control>.ListIndex(<Kanban list>)`
---

**`<Result>: (Entero)`**



- Índice de la lista en el control Kanban,

- 0 si la lista Kanban no existe.


Observación: Este índice se define en el editor.

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<Kanban list>: (Nombre del control)`**

Nombre del control Lista Kanban a manipular. Este nombre puede corresponder a: 

- el nombre del control Lista Kanban definido en el editor,

- el nombre del control Lista Kanban definido mediante programación.

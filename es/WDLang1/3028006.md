
## DDEError( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el status de tiempo de ejecución reporte de una función DDE.

De hecho, la mayoría de las funciones DDE devuelven <u><u><u>True</u></u></u> si tiene éxito y <u><u><u>False</u></u></u> si se produce un problema. **DDEError** devuelve los detalles del error. También puede utilizar [ErrorInfo](../WDLang1/3013008.md) para get el código de error.

**Atención**: **DDEError** y [ErrorInfo](../WDLang1/3013008.md) no devuelven los mismos códigos de error si se produce un error DDE. En efecto, **DDEError** devuelve el código de error de Windows mientras que [ErrorInfo](../WDLang1/3013008.md) devuelve un código de error específico de WINDEV o WEBDEV.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Connection
MyConnection = DDEConnect("Excel", Subject)
// Error occurred?
IF MyConnection = 0 THEN
	SWITCH DDEError()
		CASE 1: Error("Initialization error")
		CASE 6: Error("The recipient is unknown")
	END
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEError([<Connection identifier>])`
---

**`<Result>: (Integro)`**

Código del último error DDE que se produjo en la conexión DDE especificada.


|   |   |
| --- | --- |
| -1 | Sintaxis Error( cliente = \*, sujeto = \*, etc.) |
| 1 | Error de inicialización |
| 2 | El programa ya está detenido |
| 3 | El programa que se ejecuta actualmente se inicia mientras está registrado en WINDEV/WEBDEV (el programa no llamó a [DDEDDesconectar](../WDLang1/3028003.md) durante una ejecución anterior) |
| 4 | Hay demasiados programas registrados hacia WDDE.DLL (hasta 50 programas) |
| 5 | Se encuentran demasiados mensajes en la memoria (hasta 400 mensajes) |
| 6 | Destinatario desconocido |
| 7 | Ningún mensaje a recuperar: la pila de mensajes está vacía para el asunto solicitado |
| 8 | Insuficiente memoria para iniciar el servidor |
| 9 | El programa llamado es desconocido (ruta no válido) |
| 10 | El programa llamado ya se está ejecutando |
| 11 | Error al iniciar el programa |
| 12 | Error durante una asignación de memoria |
| 13 | Error durante la inicialización del DDE |
| 14 | Error al conectarse a la aplicación (servidor) |
| 15 | Error durante la transacción DDE |
| 16 | Error al recuperar un dato DDE |



**`<Connection identifier>: (Entero opcional)`**

Identificador de la conexión, devuelto por la función [DDEConectar](../WDLang1/3028015.md). Si no se especifica este parámetro, **DDEError** utiliza la última conexión gestionada.


## ControlInOutReason( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el origen de la entrada en la control.

**Atención**: Esta función sólo debe usarse en el código de entrada o en el código de salida de la control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Entry code of EDT_EDIT1 control
// If entry in the control via the keyboard
IF ControlInOutReason() = reasonKeyboard THEN
	// The control is cleared
	MySelf = ""
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlInOutReason()`
---

**`<Result>: (Constante de tipo Integer)`**

Origen de la entrada en el control actualmente en edición:


|   |   |
| --- | --- |
| *reasonClick* | Entrada en la control con un clic del ratón. |
| *reasonCode* | Entrada en la control actual a través de [SetFocusAndReturnToUserInput](../WDLang1/1410088107.md) (o función equivalente). |
| *reasonKeyboard* | Entrada en el control a través de la tecla TAB, un atajo de teclado o navegando a través de un control Tabla con las teclas de flecha. |
| *reasonOpening* | Entrada en la actual control después de la apertura de la ventana. |

## Presentación
<a name="presentacion_ELTTEXTE000105"></a>
WINDEV, WINDEV Mobile y WEBDEV proponen varias funciones utilizadas en las matrices de manipular.

Las matrices son herramientas matemáticas utilizadas para simplificar y resolver varios problemas. Los cálculos de las matrices se utilizan en muchos campos: la economía, la física, ....

En estos campos, los problemas pueden ser a menudo formulados como una set de ecuaciones. El resultado de esta set de ecuaciones puede ser calculado a través de las matrices.

## Definición
<a name="definicion_ELTTEXTE000129"></a>
Una matriz es un conjunto ordenado de "**n p**" números, que se muestran en un array de **n** filas y **p** columnas.

Los números incluidos en la matriz son los siguientes ***elementos de matriz***. Cada elemento se identifica por su fila y por su columna.

Un ***value*** es associated con cada elemento.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Matlit1.gif)


En esta matriz, n=2 filas y p=4 columnas. El valor del elemento (fila 2, columna 3) es 7

La ***Dimension( o tamaño) de la matriz*** corresponde al número de filas y al número de columnas que se encuentran en la matriz.

WINDEV, WINDEV Mobile y WEBDEV le permiten gestionar dinámicamente el tamaño de sus matrices.. De hecho, una matriz está vacía cuando se crea: contiene 0 fila y 0 columna. La dimensión de esta matriz es 0 x 0.

Cuando se inicializa un elemento en fila n y la columna p, el tamaño de la matriz pasa a ser n x p.

Por ejemplo, cuando se inicializa un elemento en fila 6 y en la columna 12, el tamaño de la matriz es de 6 x 12.

## ¿Cómo se hacen las matrices de manipular?
<a name="¿como_hacen_las_matrices_manipular_ELTTEXTE000153"></a>
WINDEV, WINDEV Mobile y WEBDEV proponen unas 20 funciones WLanguage para la gestión de matrices. Estas funciones están acostumbradas a:

- manipular las matrices: creación, copy, inversión, compresión, eliminación.

- inicializar los elementos de las matrices.

- leer el valor de los elementos que se encuentran en las matrices.

- realizar varios cálculos en las matrices: suma, multiplicación, transposición, determinante, ....

- get información sobre las matrices: número de filas, número de columnas, ....

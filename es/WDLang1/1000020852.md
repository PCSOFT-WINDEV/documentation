


## dSaveImageTIFF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda una imagen en formato TIFF en un archivo o en la memoria.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a TIFF file containing two pages
arrImage is array of 2 Images
arrImage[1].Image = "C:\MyImages\Image1.bmp"
arrImage[2].Image = "C:\MyImages\Image2.bmp"
dSaveImageTIFF(arrImage, "MyFile.TIFF")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Guardar una imagen TIFF en un archivo

`<Result> = dSaveImageTIFF(<Images> , <File name>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Images>: (Array de imágenes o variable de tipo Tiff)`**



- Array con las imágenes de cada página del archivo TIFF.

- Nombre de la variable de tipo [Tiff](../WDLang1/1000021025.md) que se utilizará.




**`<File name>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del archivo TIFF a crear. Se puede utilizar una ruta de tipo UNC. La extensión del archivo es opcional.

**Atención**:

- Para guardar la imagen, debe existir el directorio de destino.

- No se comprueba si existe el archivo TIFF. Si ya existe, se sobrescribe automáticamente.





<a name="SYNTAX2"></a>

### Guardar una imagen TIFF en la memoria

`<Result> = dSaveImageTIFF(<Images> , <inMemory>)`
---

**`<Result>: (Buffer)`**

Cadena de bytes que contiene la imagen en formato TIFF. Esta cadena de bytes puede asignarse a un control Imagen, guardarse en un archivo (función ([fSaveText](../WDLang1/3036046.md)), etc.

**`<Images>: (Array de imágenes o variable de tipo Tiff)`**



- Array con las imágenes de cada página del archivo TIFF.

- Nombre de la variable de tipo [Tiff](../WDLang1/1000021025.md) que se utilizará.




**`<inMemory>: (Constante)`**

*inMemory*: Constante utilizada para guardar una imagen en la memoria.

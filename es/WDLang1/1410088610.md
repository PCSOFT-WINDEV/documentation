
## diagLayout (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagLayout** se utiliza para definir todas las características de diseño de un Diagram. Las características de este diseño se pueden definir y cambiar mediante diferentes propiedades WLanguage. **Este diseño se utiliza cuando se muestra la cuadrícula y cuando se imprime el Diagram .**

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
myDiagram is Diagram

// Layout for A4
myLayout is diagLayout <- myDiagram.PageLayout
myLayout.PageWidth = 210.0
myLayout.PageHeight = 297.0
```

<a name="PROPS"></a>

## Properties


### Propiedades específicas de las variables diagLayout
<a name="propiedades_especificas_las_variables_diaglayout_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades pueden utilizarse para manipular la disposición de un Diagram:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| PageHeight | Real | Altura de Page (en milímetros). |
| PageWidth | Real | Anchura Page (en milímetros). |


<a name="NOTE0"></a>

## Observaciones
<a name="NOTE0_1"></a>
Si las propiedades **PageHeight** y **PageWidth** son set a 0, se utiliza un diseño Infinite.

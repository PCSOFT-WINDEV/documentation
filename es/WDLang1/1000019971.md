


## OrgOccurrence (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos que se encuentran en una control Organigrama. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the element according to its name
NbElements is int = OrgOccurrence(ORG_Executive)
FOR i = 1 TO NbElements
	IF ORG_Executive[i].Title = "Vince" THEN
		ORG_Executive[i].SubTitle = "Employee of the month"
	END 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = OrgOccurrence(<Organization Chart control>)`
---

**`<Result>: (Integro)`**

Número de elementos encontrados en la control Organigrama. Este número corresponde a: 

- el número de elementos añadidos por [OrgAdd](../WDLang1/1000019729.md) y [OrgAddChild](../WDLang1/1000019763.md). 

- el número de elementos añadidos a través del enlace de archivo si el control es relacionado a una fuente de datos. En este caso, estos elementos han sido mostrados por [OrgDisplay](../WDLang1/1000019764.md). 




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar.

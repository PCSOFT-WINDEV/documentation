


## &lt;Hoja de cálculo&gt;.BorderSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer y modificar los bordes de las celdas seleccionadas en un control Hoja de cálculo de una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines the border to draw
xBorder is Border
xBorder.Thickness = 2
xBorder.Color = Black

// Selects the cells to frame
PSHEET_Spreadsheet.SelectPlus("A3", "B12")
// Traces the border of the group of cells
PSHEET_Spreadsheet.BorderSelection(xBorder, False)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Modificación de las características de las fronteras

`<Spreadsheet control>.BorderSelection(<Border> [, <Selection>])`
---

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de la control Hoja de cálculo utilizada.

**`<Border>: (Variable de tipo Border)`**

Nombre de la [Border](../Motscles/15140100.md) Variable que contiene las características del Border que se va a utilizar.

**`<Selection>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si los bordes deben aplicarse a cada celda seleccionada,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si los bordes deben aplicarse a todas las celdas seleccionadas.





<a name="SYNTAX2"></a>

### Descubriendo las características de la Border

`<Border> = <Spreadsheet control>.BorderSelection()`
---

**`<Border>: (Variable de tipo Border)`**

Nombre de la [Border](../Motscles/15140100.md) Variable que contiene las características de la Border.

**`<Spreadsheet control>: (Nombre del control)`**

Nombre de control Hoja de cálculo utilizado.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La operación de modificación se añade a la lista de operaciones de Cancel: el usuario puede Cancel esta operación a través de el menú contextual o a través de Ctrl + Z. 

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

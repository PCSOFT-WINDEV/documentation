
## MatNbLine( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de filas que se encuentran en una matriz.. El número de líneas de matriz cambia dinámicamente. Corresponde al número de filas inicializadas. El número de filas puede ser modificado por:

- [MatWrite](../WDLang1/3050077.md),

- [MatFill](../WDLang1/3050057.md).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of rows?
ResNbRows = MatNbLine("MyMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatNbLine(<Matrix>)`
---

**`<Result>: (Integro)`**

Número de filas que se encuentran en la matriz especificada.

**`<Matrix>: (Cadena de caracteres)`**

Nombre de la matriz a utilizar, definida con [MatCreate](../WDLang1/3050058.md).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz
<a name="existencia_matriz_ELTPARAGRAPHE000084"></a>

No se verifica la existencia de la matriz.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000095"></a>

**Atención**: La función **MatNbLine** no devuelve ningún código de error. Para saber si se han generado errores al devolver el número de filas, utilice [MatError](../WDLang1/3050044.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

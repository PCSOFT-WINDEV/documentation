
## SysNotificationClear( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las notificaciones interactivas enviadas previamente por la aplicación actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// -- Application closing code 
// Removes all notifications from the application before closing it
SysNotificationClear()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SysNotificationClear()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Cuando se eliminan, las notificaciones de la aplicación desaparecen de la pantalla y se eliminan del Centro de Acción de Windows.

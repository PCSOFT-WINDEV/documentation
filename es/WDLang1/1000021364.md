
## RealToDateTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un real de 8 bytes en una fecha y hora. Este real corresponde a:

- el número de días transcurridos desde el 1 de enero de 1900 (compatible con Excel, comienza a partir de 1)

- el número de días transcurridos desde el 1 de enero de 1904 (compatible con Excel seleccionado la opción "Usar sistema de fechas 1904", comienza a partir de 0).




Observación: La opción "Usar sistema de fechas 1904" se puede configurar en las opciones avanzadas de Microsoft Excel. En Open Office, esta opción está disponible en las opciones de Open Office Calc (sección "Calcular", opción "Fecha"). 






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays "19000103060000000"
Info(RealToDateTime(3.25))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RealToDateTime(<Value> [, <From 1904>])`
---

**`<Result>: (Variable de tipo DateTime o cadena de caracteres)`**

Valor de la fecha y hora. Puede corresponder a: 

- una variable de tipo DateTime. 

- una cadena de caracteres en formato AAAAMMDDHHmmSSCCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CCC a los milisegundos). 

- una cadena vacía ("") si se produce un error. 




**`<Value>: (Real)`**

Valor a convertir. 

**`<From 1904>: (Booleano opcional)`**

Modo de cálculo: 

- <u><u><u><u>True</u></u></u></u> si la fecha se calcula a partir de 1904. 

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) si la fecha se calcula a partir de 1900. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Modo de cálculo
<a name="modo_calculo_ELTPARAGRAPHE000094"></a>

La parte entera del número real da el número de serie que corresponde a la fecha del resultado: 

- comienza a contar a partir de 1 para el 01/01/1900

- comienza a contar a partir de 0 para el 01/01/1904 si el parámetro &lt;Desde 1904&gt; se establece en <u><u><u><u>True</u></u></u></u>. 


La parte decimal da la fracción de día que corresponde a la hora del resultado (p. ej., 0,25 corresponde a "060000000").

Atención: Cualquier conversión de tipo puede causar una pérdida de precisión.

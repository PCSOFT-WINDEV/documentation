
## SpreadsheetGetSortInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información diversa sobre la ordenación realizada en un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Shows the area that will be sorted
sSel is string 
sSel = SpreadsheetGetSortInfo(PSHEET_Spreadsheet, psheetInfoSortSelection) 
IF sSel = "" THEN
	RETURN // not sortable
END
sMin is string = ExtractString(sSel, 1, ":")
sMax is string = ExtractString(sSel, 2, ":")

// Selects the area 
SpreadsheetSelectPlus(PSHEET_Spreadsheet, sMin, sMax)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetGetSortInfo(<Spreadsheet control> , <Type of information>)`
---

**`<Result>: (Cadena de caracteres)`**

Información solicitada. 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 

**`<Type of information>: (Constante de tipo Integer)`**

Información solicitada: 


|   |   |
| --- | --- |
| *psheetInfoSortCriteria* | Se utiliza para averiguar los últimos criterios de clasificación utilizados para clasificar la control Hoja de cálculo. &lt;El resultado tiene el siguiente formato: <br><br><pre><code>"<Name of column1>[;< Option>]" + TAB + <br>"<Name of column2>" + TAB + ...</code></pre><br>donde: <br><br>- &lt;Nombre de la columna&gt; es el nombre de la columna ordenada,<br><br>- &lt;Opción&gt; corresponde a una de las constantes de ordenación definidas para [Hoja de cálculoSortSelection](../WDLang1/1000023463.md). <br><br><br><br><br>Ejemplo: "A&lt;Tab&gt;B;3" |
| *psheetInfoSortSelection* | Se utiliza para conocer la selección que será clasificada por la AAF (automatic Application Features) para la clasificación de la control Hoja de cálculo. Ejemplo: "A2:B4". <br><br>Observación: Si la selección no permite la clasificación, &lt;Resultado&gt; corresponde a una cadena vacía ("""). |
| *psheetInfoSortWithTitle* | Se utiliza para saber si la selección clasificada por la AAF (automatic Application Features) para la clasificación de la control Hoja de cálculo contiene un título. &lt;El resultado puede corresponder a: <br><br>- 1 si la selección a ordenar incluye un título,<br><br>- 0 de lo contrario. <br><br><br> |

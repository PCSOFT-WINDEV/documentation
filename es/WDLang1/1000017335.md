


## &lt;Caja de lista&gt;.Posición (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un cuadro de lista o control ListView de un elemento especificado o devuelve el índice del primer elemento mostrado en un control List Box (si se utilizó la barra de desplazamiento).

**Observación**: La función **&lt;Caja de lista&gt;.Posición** puede utilizarse en:

- Controles de la caja de lista basados en un archivo de datos o rellenados mediante programación,

- Controles de cuadro de lista de selección única o de selección múltiple.













<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Index of the first element displayed in "LIST_CustomerList"
ResPosition = LIST_CustomerList.Position()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Mostrando una control List Box de un elemento específico

`<List Box control>.Position(<Element index>)`
---

**`<List Box control>: (Nombre del control)`**

Nombre de control List Box a utilizar.

**`<Element index>: (Integro)`**

Índice del elemento que debe aparecer al principio de la control List Box. 
Se produce un error de WLanguage si este parámetro es mayor que el número de elementos encontrados en el control List Box (devuelto por [&lt;List Caja&gt;.Cuenta](../WDLang1/1000017334.md)).

Si este parámetro es igual a 1, el control List Box se reposiciona al principio..


<a name="SYNTAX2"></a>

### Obtención del índice del primer elemento mostrado en un Cuadro de Listas o control Combo Box

`<Result> = <List Box control>.Position()`
---

**`<Result>: (Integro)`**

Índice del primer elemento mostrado en la control List Box especificada .

**`<List Box control>: (Nombre del control)`**

Nombre de control List Box a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Visualización de un elemento al principio de una control List Box (sintaxis 1)
<a name="visualizacion_elemento_principio_una_control_list_box_sintaxis_1_ELTPARAGRAPHE000268"></a>

El elemento que aparece al principio de la control List Box puede diferir del elemento deseado. De hecho, la función **&lt;Caja de lista&gt;.Posición** ubica al principio del control el elemento que permite mostrar un máximo de filas en el control. Si no hay suficientes elementos mostrados debajo del elemento especificado, este elemento no puede mostrarse al principio de la control.

Por ejemplo, el elemento "Smith Jimmy" no puede aparecer al principio de la control List Box: no hay suficientes elementos visualizados debajo de este elemento.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=LISTEPOSITION.gif)

<a name="NOTE0_2"></a>


### Limitaciones
<a name="limitaciones_ELTPARAGRAPHE000283"></a>

**&lt;Caja de lista&gt;.Posición** no puede utilizarse en los controles de cuadro de lista basados en un archivo de datos en modo "manual" que utilizan una barra de desplazamiento de rebote.

Reminder:

- Un control List Box basado en un archivo de datos en modo "manual" es un control List Box relacionado con un archivo de datos mediante programación.

- Con una barra de desplazamiento rebotante, puedes hacer que la barra de desplazamiento se desplace manteniendo el botón del ratón pulsado en el botón de la barra de desplazamiento o manteniendo las teclas de flecha (flecha superior o inferior) pulsadas.



<a name="NOTE0_3"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000300"></a>

- Para mostrar un elemento al principio de un control Tabla o para get el índice del primer elemento mostrado en un control Tabla, utilice [.Posición](../WDLang1/1000024278.md).

- **&lt;Caja de lista&gt;.Posición** puede utilizarse en una columna "Combo Box" tabla.




## &lt;Chrono&gt;.Reset (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene y reinicia un cronómetro en marcha. El valor del tiempo transcurrido se pierde.  




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyStopwatch is chrono
...

// -- Click code of BTN_Reset

// Resets MyStopwatch
MyStopwatch.Reset()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Stopwatch>.Reset()`
---

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.


## SysColor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o modifica la Color de un elemento de Windows.

![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En esta versión, sólo se puede identificar la Color de un elemento de Windows.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Identify the color of active title bar
ResColor = SysColor(scActiveCaption)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Identificar la Color de un elemento de Windows

`<Result> = SysColor(<Windows element>)`
---

**`<Result>: (Integro)`**

Color del elemento especificado en formato RGB. Para recuperar los componentes Color, utilice [RGBBlue](../WDLang1/3029049.md), [RGBRed](../WDLang1/3029050.md) y [RGBGreen](../WDLang1/3029051.md). 

**`<Windows element>: (Constante)`**

Elemento de Windows a utilizar:


|   |   |
| --- | --- |
| *scActiveBorder* | Frontera de la ventana de Active.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scActiveCaption* | Barra de título Active. |
| *scAppWorkSpace* | Espacio de trabajo de las aplicaciones MDI.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scBackground* | Escritorio. |
| *scBtnFace* | Border de botón. |
| *scBtnHighLight* | Botón seleccionado (corresponde a los botones 3D de Windows 95). |
| *scBtnShadow* | Sombra de botones.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scBtnText* | Texto de los botones de texto. |
| *scCaptionText* | Texto de la barra de título. |
| *scGrayText* | Texto Grayed (devuelve cero si el controlador de pantalla no contiene escala de grises). |
| *scHighLight* | Background del objeto seleccionado.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scHighLightText* | Texto del objeto seleccionado. |
| *scInactiveBorder* | Frontera de la ventana de Inactive.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scInactiveCaption* | Barra de título Inactive. |
| *scInactiveCaptionText* | Barra de título Inactive. |
| *scMenu* | Background de menús.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scMenuText* | Texto de los menús.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scScrollBar* | Barra de desplazamiento.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scWindow* | Background de ventana. |
| *scWindowFrame* | Border de ventana.<br><br>![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta constante no está disponible. |
| *scWindowText* | Texto de la ventana. |

### Modificar la Color de un elemento de Windows

`SysColor(<Windows element> , <Color>)`
---

**`<Windows element>: (Constante)`**

Elemento de Windows a utilizar:


|   |   |
| --- | --- |
| *scActiveBorder* | Frontera de la ventana de Active. |
| *scActiveCaption* | Barra de título Active. |
| *scAppWorkSpace* | Espacio de trabajo de las aplicaciones MDI. |
| *scBackground* | Escritorio. |
| *scBtnFace* | Border de botón. |
| *scBtnHighLight* | Botón seleccionado (corresponde a los botones 3D de Windows 95). |
| *scBtnShadow* | Sombra de botones. |
| *scBtnText* | Texto de los botones de texto. |
| *scCaptionText* | Texto de la barra de título. |
| *scGrayText* | Texto Grayed (devuelve cero si el controlador de pantalla no contiene escala de grises). |
| *scHighLight* | Background del objeto seleccionado. |
| *scHighLightText* | Texto del objeto seleccionado. |
| *scInactiveBorder* | Frontera de la ventana de Inactive. |
| *scInactiveCaption* | Barra de título Inactive. |
| *scInactiveCaptionText* | Barra de título Inactive. |
| *scMenu* | Background de menús. |
| *scMenuText* | Texto de los menús. |
| *scScrollBar* | Barra de desplazamiento. |
| *scWindow* | Background de ventana. |
| *scWindowFrame* | Border de ventana. |
| *scWindowText* | Texto de la ventana. |



**`<Color>: (Integro)`**

Nueva Color para el elemento especificado. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

<a name="NOTE0_1"></a>

## Observaciones




### Color
<a name="color_ELTPARAGRAPHE000287"></a>

El Color depende del número de Color s utilizados por el sistema y de la resolución de la pantalla. Este número de colores es devuelto por [SysColorRes](../WDLang1/3073012.md).


## fAddBuffer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade el contenido de un búfer al final de un archivo. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyBuffer is Buffer 
MyBuffer = fLoadBuffer("C:\tmp\Page1.html")

IF fAddBuffer("C:\test\MyFile.html", MyBuffer)
	Info("Addition successful")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fAddBuffer(<File used> , <Buffer to add>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó la suma, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<File used>: (Cadena de caracteres)`**

Archivo a utilizar (hasta 260 caracteres). Este parámetro puede ser: 

- un nombre de archivo. Este archivo se buscará automáticamente en el directorio actual (devuelto por [fCurrentDir](../WDLang1/3036022.md)).

- un nombre de archivo con su ruta completa o relativa. Se puede utilizar una ruta de tipo UNC.




**Atención**: este archivo es modificado por la función **fAddBuffer**. 

![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.





**`<Buffer to add>: (Buffer)`**

Nombre del buffer Variable cuyo contenido debe ser añadido al archivo.

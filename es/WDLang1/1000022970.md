


## docElementLink (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docElementLink** se utiliza para obtener las características avanzadas de un elemento de enlace en un [docElement](../WDLang1/1000022484.md) Variable. 

Las características del **docElementLink** Variable pueden ser identificadas y modificadas mediante diferentes propiedades del WLanguage. 

**PRECAUCIÓN: Este tipo no se puede utilizar directamente. Debe utilizarse mediante el tipo [docElement](../WDLang1/1000022484.md).**
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Load the document
WP_MyDoc = ""

// Create a link element
elt is docElement
elt.Type = typeDocElementLink
elt.Link.Link = "https://www.windev.com"
elt.Link.Type = typeDocLinkURL

// Create a document fragment linked to the control used
f is docFragment(WP_MyDoc, 1)

// Add the link in the document
DocInsert(f, elt)

// Search for the link to add the text and define the style
FOR EACH eltInsert OF f.Element
	set parent = eltInsert.Parent
	parent.Link.Content.Text = "PC SOFT website"
	parent.Link.Content.Formatting.TextColor = DarkBlue
	parent.Link.Content.Formatting.UnderlineColor = DarkBlue
	parent.Link.Content.Formatting.FontUnderlined = True
	BREAK
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables docElementLink
<a name="propiedades_especificas_las_variables_docelementlink_ELTPARAGRAPHE000033"></a>

Las siguientes propiedades pueden ser utilizadas para manipular una lila en un Document: 


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Contenido | Variable de tipo [docFragment](../WDLang1/1000022482.md) | Texto associated con el enlace.  |
| escribir a máquina | Integro | Tipo de enlace: <br><br>- *typeDocLinkBookmark*: Enlace a favoritos. <br><br>- *typeDocLinkURL*: Enlace URL utilizado para abrir una Document.<br><br><br> |
| Tooltip | Cadena de caracteres | Contenido del tooltip associated con el enlace. |
| vincular | Cadena de caracteres | Contenido del enlace (archivo ruta, nombre del sitio, ....). El título del enlace no puede ser conocido. |

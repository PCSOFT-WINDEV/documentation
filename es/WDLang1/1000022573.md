


## docTabulation (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docTabulation** permite definir las características de una tabulación personalizada. Esta tabulación personalizada está asociada a un párrafo incluido en un procesador de textos Document. Las características de esta tabulación pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDoc is Document = MyDocControl
// Retrieve the first paragraph
Parag is docParagraphe = MyDoc.Paragraph[1]
// Tabulation at 3 cm
Tabul is docTabulation
Tabul.Position = 30
// Add the tabulation to the custom tabulations of paragraph
Add(Parag.PageLayout.CustomTabulation, Tabul)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docTabulation
<a name="propiedades_especificas_descripcion_las_variables_doctabulation_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades se pueden utilizar para manipular una variable de tipo **docTabulation**:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| escribir a máquina | Constante opcional de tipo Integer | Tipo de tabulación a añadir:<br><br>- *tabTypeBar*: Se mostrará una barra vertical para la tabulación.<br><br>- *tabTypeCenter*: El centro del texto se alineará en la tabulación.<br><br>- *tabTypeDecimal*: El separador decimal se alineará en la tabulación.<br><br>- *tabTypeRight*: El texto correcto Border se alineará en la tabulación.<br><br>- *tabTypeLeft* (valor de Default): El texto izquierdo Border se alineará en la tabulación.<br><br><br> |
| FillCharacter | Cadena de caracteres o constante | Carácter de relleno para el espacio antes de la tabulación. <br><br>Esta Property puede corresponder a una de las siguientes constantes: <br><br>- *docTabulationCharacterNone*: Carácter vacío (no hay carácter de relleno).<br><br>- *docTabulationCharacterDot*: Dot.<br><br>- *docTabulationCharacterMiddleDot*: Punto central.<br><br>- *docTabulationCharacterUnderscore*: Subrayado.<br><br>- *docTabulationCharacterHyphen*: Hifas.<br><br><br>Este Property corresponde a un personaje vacío de Default. |
| ubicación | Real | Posición de la tabulación (en milímetros) desde el borde izquierdo del párrafo (0 por Default). |

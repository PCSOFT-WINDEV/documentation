


## TabClose (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra un panel dinámico en un control Pestaña. El Event "Cerrar un panel" no se llama. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click the Close Button control
TabClose(TAB_DynTab, TAB_DynTab.Value)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Cerrar un panel de pestañas dinámico específico

`TabClose(<Tab control> , <Alias>)`
---

**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. 

**`<Alias>: (Cadena de caracteres)`**

Alias del panel dinámico de pestañas para cerrar. Este alias puede ser recuperado por: 

- la [Valor](../Proprietes/2510130.md) property. 

- la [Alias](../Proprietes/2510004.md) property. 

- la función [TabOpen](../WDLang1/1000021369.md). 


Si este parámetro no corresponde a ningún panel dinámico, la función no tiene acción y se produce un error no fatal. 


<a name="SYNTAX2"></a>

### Cerrando el actual panel de pestañas dinámicas

`TabClose(<Tab control>)`
---

**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar. El actual panel dinámico se cerrará.

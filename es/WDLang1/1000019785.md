
## SysAndroidVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la información sobre la versión de Android utilizada por la aplicación.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The Bluetooth functions are available 
// from the Api Level 5 (Android "Eclair" 2.0)
IF Val(SysAndroidVersion(sysVersionApiLevel)) >= 5 THEN
	// Enables the bluetooth on the device
	BTActivate(True)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysAndroidVersion([<Type of information>])`
---

**`<Result>: (Cadena de caracteres)`**



- Solicitó información sobre la versión de Android. 

- Cadena vacía ("") si se produce un error. 




**`<Type of information>: (Constante opcional de tipo Integer)`**

Indica la información solicitada: 


|   |   |
| --- | --- |
| *sysVersionApiLevel* | Nivel de API de Android. |
| *sysVersionNumber* | Número de versión de Android.<br>Por ejemplo: "1.5", "2.2", "2.3.3", ...). |
| *sysVersionPlatform* | Nombre de la versión principal de Android:<br><br>- "Pastelito"<br><br>- "Anillo"<br><br>- "Relámpago"<br><br>- "Froyo"<br><br>- "Pan de jengibre"<br><br>- "Nido de abeja"<br><br>- "Sándwich de helado"<br><br><br>Si no se conoce el nombre de la versión, &lt;Result&gt; corresponde a una cadena vacía ("""). |

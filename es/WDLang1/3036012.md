
## fShortPath( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la abreviatura ruta completa de un fichero o directorio.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036012&name=fshortpath_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Retrieve the short path of a file
ResShortPath = fShortPath("C:\MyDirectories\Report March 2000.doc")
// fShortPath returns: "C:\MyDir~1\Account~1.doc"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fShortPath(<File path> [, <Option>])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre corto correspondiente a la ruta completa del fichero o directorio (cadena vacía si se ha producido un error). Si &lt;Archivo ruta&gt; termina con un carácter _EASLASH_, este carácter se conserva.

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*. Se genera un error, por ejemplo, si el archivo o directorio especificado no existe.




**`<File path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del fichero o directorio (hasta 260 caracteres). Se puede utilizar una ruta UNC.
Este parámetro puede estar en formato Ansi o Unicode.

**`<Option>: (Constante opcional de tipo Integer)`**

Se usa para especificar el formato del resultado de la función. Por Default, **fShortPath** devuelve una cadena de caracteres en formato Ansi. 


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. |


**Observación**: El resultado de **fShortPath** es siempre una cadena de caracteres ANSI, la constante *fPathUNICODE* sólo se utiliza para almacenar el resultado en una cadena de caracteres Unicode.




## <Edit>.DeleteToken (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un Token de un Token "Texto control Campo de entrada".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Delete the test robot]
EDT_Recipient.DeleteToken("robotmail@address.test.com")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Suprimir una Token identificada por su valor

`<Result> = <Edit control>.DeleteToken(<Value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el token,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" control Campo de entrada.

**`<Value>: (Cadena de caracteres)`**

El valor almacenado del Token para borrar. Si varios Token tienen el mismo valor, sólo se borrará el primer Token.


<a name="SYNTAX2"></a>

### Suprimir una Token identificada por su índice

`<Result> = <Edit control>.DeleteToken(<Subscript>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el token,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" control Campo de entrada.

**`<Subscript>: (Integro)`**

Índice de Token para borrar.

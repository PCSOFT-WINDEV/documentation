
## RepeatString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Concatena un número determinado de copias de la misma cadena o buffer especificado.








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res = RepeatString("x", 4)     // Returns "xxxx"
Res = RepeatString("Boby", 2)  // Returns "BobyBoby"
Res = RepeatString("", 2)      // Returns ""
Res = RepeatString("", 0)      // Returns ""
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Concatenar un número especificado de copias de la misma cadena

`<Result> = RepeatString(<Initial string> , <Number of copies>)`
---

**`<Result>: (Cadena de caracteres)`**



- Nueva cadena que contiene el número especificado de copias de la cadena inicial,

- Cadena vacía ("") si &lt;Número de copias&gt; es negativo o nulo.




**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres a repetir.

**`<Number of copies>: (Entero)`**

Número de veces que se debe repetir la &lt;Cadena inicial&gt;.  


<a name="SYNTAX2"></a>

### Concatenar un número especificado de copias del mismo buffer

`<Result> = RepeatString(<Initial buffer> , <Number of copies>)`
---

**`<Result>: (Buffer)`**



- Nuevo buffer que contiene el número especificado de copias del buffer inicial,

- Buffer vacío si &lt;Número de copias&gt; es negativo o nulo.




**`<Initial buffer>: (Buffer)`**

Buffer a repetir.

**`<Number of copies>: (Entero)`**

Número de veces que se debe repetir el &lt;Buffer inicial&gt;.

## Observaciones




### Función RepeatString y UNICODE (sintaxis 1)
<a name="funcion_docparampagetitleshort_unicode_sintaxis_1_ELTPARAGRAPHE000093"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **RepeatString** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

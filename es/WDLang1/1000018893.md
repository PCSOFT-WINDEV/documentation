
## HaspHLEncrypt( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cifra una cadena de caracteres utilizando los algoritmos de la clave Hasp especificada..

**Observación** : Esta cadena puede ser descifrada por [HaspHLDescifrar](../WDLang1/1000018894.md) utilizando la misma clave.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Encrypts the content of the edit control (password)
EncryptedPassword is string 
EncryptedPassword = HaspHLEncrypt(EDT_Session_ID, EDT_Password)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HaspHLEncrypt(<Session identifier> , <String to encrypt>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena encriptada. Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.

**`<Session identifier>: (Integro)`**

Identificador de la sesión Hasp, devuelto por [HaspHLLogin](../WDLang1/1000018891.md).

**`<String to encrypt>: (Cadena de caracteres)`**

Cadena de caracteres a cifrar (15 caracteres mínimo).


**Atención**: Se producirá un error si esta cadena contiene menos de 15 caracteres.

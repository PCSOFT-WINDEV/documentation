
## IWListCount( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Retorna el número de ventanas internas que se encuentran actualmente en la lista de ventanas internas buscadas en un control Ventana Interna.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IWListCount(IW_InternalWindowControl)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = IWListCount(<Internal Window control>)`
---

**`<Result>: (Integro)`**

Número de ventanas internas actualmente associated con el control Ventana interna especificado.

**`<Internal Window control>: (Nombre del control)`**

Nombre de la control Ventana interna a utilizar.

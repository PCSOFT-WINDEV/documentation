
## WordToArray (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Rellena un array con las palabras de una cadena. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Split string in the array
MyArray is array of strings
WordToArray("Example function WordCount", MyArray)
// the array contains
//		"Example"
//		"function"
//		"WordCount"
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WordToArray(<String> , <Array>)`
---

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres a manipular. 

**`<Array>: (Variable de tipo Array)`**

Nombre de la variable de tipo "Array of strings" que contendrá las diferentes palabras. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todos los caracteres que no sean numéricos o alfabéticos se consideran separadores de palabras.

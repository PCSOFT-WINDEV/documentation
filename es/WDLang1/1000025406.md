


## &lt;Array&gt;.Distinct (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un array sin duplicados.

Se pueden manipular los siguientes arrays:

- Array unidimensional de elementos simples (int, real, string).

- Array unidimensional de estructuras. Los duplicados se eliminan según el valor de uno o más miembros.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// First array of integers
arrMyArray1 is array of int
arrMyArray1.Add(1)
arrMyArray1.Add(2)
arrMyArray1.Add(1)
arrMyArray1.Add(3)

// "Distinct" array
arrMyDistinct is array of int
arrMyDistinct = arrMyArray1.Distinct()
// the result is 1; 2; 3
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <WLanguage array>.Distinct()`
---

**`<Result>: (Array)`**

Variable de tipo Array que contiene el array sin duplicados.

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Este array debe ser unidimensional.

<a name="NOTE0_1"></a>

## Observaciones
**Notas adicionales**
También puede eliminar los duplicados de un array con la función [.DeleteDuplicate](../WDLang1/1000024315.md). 

- La función [.DeleteDuplicate](../WDLang1/1000024315.md) manipula directamente el array pasado como parámetro. 

- La función **&lt;Array&gt;.Distinct** no modifica el array pasado como parámetro y devuelve un nuevo array sin duplicados. 
	
	**Secuencia de funciones**
	Puede utilizar las funciones de array en una secuencia. 
	Las siguientes funciones pueden utilizarse en una secuencia:

	- [Distinto](../WDLang1/1000025406.md).

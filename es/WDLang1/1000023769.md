


## &lt;SelectCount (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos seleccionados en una control TreeView (para una selección múltiple control TreeView). Las líneas seleccionadas corresponden a las líneas actuales en las que se visualiza el banda de selección.

**Observación**: La función **&lt;SelectCount** puede utilizarse en:

- un control TreeView poblado programáticamente, basado en un archivo de datos, o basado en un Variable. 

- una selección única o una selección múltiple control TreeView..



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nbOccurrences is int 
nbOccurrences = TREE_TreeView1.SelectCount()
FOR i = 1 TO nbOccurrences 
	Trace("Selected node:" + TREE_TreeView1.Select(False, i))  
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <TreeView control>.SelectCount()`
---

**`<Result>: (Integro)`**



- Número de filas seleccionadas en la control TreeView especificada. Este número puede ser mayor que 1 para los controles TreeView de selección múltiple. 

- 0 si no se selecciona fila.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

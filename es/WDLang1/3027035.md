
## Easter (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha de Pascua del año especificado según el calendario cristiano occidental.

**Recordatorio**: La fecha de Pascua corresponde al primer domingo después de la primera luna llena tras el equinoccio de primavera.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Easter(2006) // returns "20060416"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Easter(<Year>)`
---

**`<Result>: (Cadena de caracteres)`**



- Fecha de Pascua para el año especificado en formato AAAAMMDD,

- Cadena vacía ("") si se produce un error.




**`<Year>: (Cadena de caracteres, variable de tipo Date o entero)`**

Año para el que debe calcularse la fecha de Pascua. El año debe estar entre 1582 (calendario gregoriano) y 9999.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta fecha no es válida para el calendario ortodoxo.

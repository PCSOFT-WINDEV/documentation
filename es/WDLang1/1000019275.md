


## SchedulerDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las citas y recursos de un control Planificador.

Observación: 

- En el caso de una control Planificador rellenada programáticamente, se borran todas las citas y recursos. 

- Para un control Planificador basado en un archivo de datos o en un Variable, las citas y los recursos sólo se borran del control Planificador. Los nombramientos y los recursos se mantienen en la fuente de datos relacionado a la control. 






![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En el modo Universal Windows 10 App, esta función sólo se puede utilizar para manipular los controles del Programador que se encuentran en un reporte. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the entire content from the SCH_Schedule control
SchedulerDeleteAll(SCH_Schedule)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`SchedulerDeleteAll(<Scheduler control>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador del que se borrarán todas las citas y recursos. Este control puede corresponder a: 

- el control Planificador de una ventana,

- el control Planificador de una página,

- la control Planificador de un reporte.




## Funciones MDI
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones MDI son las siguientes:



|   |   |
| --- | --- |
| [MDIActive](../WDLang1/3052002.md) | Identifica o muestra una ventana secundaria de MDI en primer plano. |
| [MDIEnumChild](../WDLang1/3052001.md) | Devuelve el alias de la ventana MDI hija especificada. |
| [MDIMother](../WDLang1/3052006.md) | Identifica el nombre de la ventana principal de MDI. |
| [MDIOpen](../WDLang1/3052005.md) | Abre y habilita una ventana secundaria de MDI. |
| [MDIWindowMenu](../WDLang1/3052004.md) | Modifica la disposición de las ventanas MDI hijas. |

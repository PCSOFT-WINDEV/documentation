
## FontCreate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un tipo de letra Definition. Esta fuente se puede utilizar:

- para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- para seleccionar el tipo de letra utilizado en los gráficos ([grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md), [grAxisTitleFont](../WDLang3/3042059.md)).

- para seleccionar una fuente de impresión ([iFont](../WDLang5/3046014.md)).

- para establecer el tipo de letra utilizado en el control de una ventana, página o reporte ([tipo de letra](../Proprietes/2515045.md) property).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a font
MyFont is Font 
MyFont = FontCreate("Arial", 12, iBold, LightYellow)
// Use this font in the "CustomerName" control
CustomerName.Font = MyFont
```

```wl
// Create a standard font
StandardF is font = FontCreate("Arial", 12)
// Create a bold font
BoldF is font = FontCreate("Arial", 12, iBold)
// Prepare the print
iDestination(iPDF)
// Browse an array
i is int
FOR i = 1 TO 100
	// Use the standard font for the first 50 array elements
	IF i<=50 THEN
		iFont(StandardF)
	ELSE
		// Use the bold font for the last 50 array elements
		iFont(BoldF)
	END
END
// Start the print job
iEndPrinting()
```

## Sintaxis

`<Result> = FontCreate(<Font name> [, <Size> [, <Style> [, <Color> [, <Angle>]]]])`
---

**`<Result>: (Variable de tipo Font)`**

[tipo de letra](../Motscles/1514045.md) variable que se puede utilizar en las diferentes funciones de dibujo, funciones de gráfico o funciones de impresión que utilizan una fuente.

**`<Font name>: (Cadena de caracteres)`**

Nombre de la fuente utilizada. Este nombre debe corresponder a una de las fuentes instaladas en el ordenador actual.

**`<Size>: (Entero opcional)`**

Tamaño de fuente expresado en picas (1 Pica = 0.353 mm). El tamaño de la letra es set a 12 por Default.

**`<Style>: (Constante o combinación de constantes de tipo Integer (opcional))`**

Atributos de la fuente:


|   |   |
| --- | --- |
| *iBold* | Fuente en negrita |
| *iCondensed* | Caracteres condensados |
| *iExtended* | Caracteres extendidos |
| *iItalic* | Fuente cursiva |
| *iLarge* | Caracteres grandes |
| *iNormal*<br>(valor predeterminado) | Fuente estándar, sin atributos |
| *iStrikeOut* | Fuente tachada |
| *iUnderlined* | Fuente subrayada |



**`<Color>: (Entero opcional)`**

Color utilizado para la fuente. Si no se especifica este parámetro, el Color utilizado es Black. 
Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)).

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- y [color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Angle>: (Entero opcional)`**

Ángulo (en grados) según el cual se imprimirá el texto. El ángulo se calcula a partir de una línea horizontal en sentido contrario a las agujas del reloj (dirección trigonométrica).

## Observaciones




### Diferencias FontCreate e iCreateFont
<a name="diferencias_docparampagetitleshort_icreatefont_ELTPARAGRAPHE000141"></a>

**FontCreate** crea un tipo de letra Definition que puede utilizarse en todos los elementos de una aplicación WINDEV o WEBDEV. Este tipo de letra se almacena en la [Variable de tipo Font](../Motscles/1514045.md) y se puede utilizar:

- para escribir texto en dibujos (con la función [dFont](../WDLang1/3029020.md)).

- para seleccionar el tipo de letra utilizado en los gráficos ([grLabelFont](../WDLang3/3042012.md), [grLegendFont](../WDLang3/3042037.md), [grTitleFont](../WDLang3/3042048.md), [grAxisTitleFont](../WDLang3/3042059.md)).

- para seleccionar una fuente de impresión ([iFont](../WDLang5/3046014.md)).

- para establecer el tipo de letra utilizado en los controles de una ventana ([tipo de letra](../Proprietes/2515045.md) property).

- para establecer el tipo de letra utilizado en los controles de una página ([tipo de letra](../Proprietes/2515045.md) property).

- para establecer el tipo de letra utilizado en los controles de un reporte ([tipo de letra](../Proprietes/2515045.md) property).




[iCreateFont](../WDLang5/3046045.md) crea una fuente que puede ser utilizada por las funciones de impresión.

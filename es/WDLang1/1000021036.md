


## GanttDateTimePosition (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Configura el primer día y la primera vez que se muestra en una columna de diagrama de Gantt (que se encuentra en una tabla o control Tabla TreeView).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
GanttDateTimePosition(COL_Gantt, DateSys())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`GanttDateTimePosition(<Gantt column> , <Start date>)`
---

**`<Gantt column>: (Nombre del control)`**

Nombre de la columna (de un control Tabla o Tabla TreeView) de tipo Diagrama de Gantt a utilizar.

**`<Start date>: (Cadena de caracteres o Fecha/Hora Variable)`**

Fecha de inicio para la visualización en el diagrama de Gantt. Este parámetro puede ser: 

- una fecha en formato "AAAAMMDD",

- una fecha y una hora en formato "AAAAMMDDHHmmSSCC".






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
En un reporte, la función **GanttDateTimePosition** debe utilizarse en el código de inicialización de la columna de tipo Gantt.

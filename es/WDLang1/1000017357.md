


## DeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los elementos:

- de un WLanguage array unidimensional o bidimensional.

- de un array asociativo.

- de una propiedad array avanzada (array de eventos gglCalendar, ....).

- de una cola de WLanguage.

- de una pila de WLanguage.

- de una lista WLanguage.




**Observaciones**: 

- Esta función es equivalente a la función [ArrayDeleteAll](../WDLang1/3075010.md).











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyArray is array of 2 strings
MyArray[1] = "WINDEV"
MyArray[2] = "WEBDEV"
// Delete all the elements
DeleteAll(MyArray)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`DeleteAll(<WLanguage variable>)`
---

**`<WLanguage variable>: (Array, Cola, Pila o Lista)`**

Nombre de la variable que se utilizará. Esta variable puede ser:

- a Array. Este array debe ser unidimensional o bidimensional.

- una Lista.

- un Stack.

- una cola.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Borrar elementos
<a name="borrar_elementos_ELTPARAGRAPHE000127"></a>

- **Array unidimensional**: la array se redimensiona a 0. No se ha borrado el array: se pueden realizar nuevas adiciones sin necesidad de volver a crear la array.

- **Array bidimensional**: sólo se borran las líneas de array. Las columnas se mantienen. Por lo tanto, se puede utilizar la función [ArrayAddLine](../WDLang1/3075013.md) después de la función **ArrayDeleteAll**.

- **Asociativa array**: se borran todos los elementos de array.

- **Cola**: se borran todos los elementos de cola.

- **Pila**: se borran todos los elementos de la pila.

- **List**: se borran todos los elementos de la lista.




**Observación**: Para reinicializar (poner a 0 o a la cadena vacía "") los elementos array sin redimensionar el array, utilice [VariableReset](../WDLang1/3013069.md).

<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000152"></a>

Esta función no se puede utilizar con la arrays fija.

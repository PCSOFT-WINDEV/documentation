
## NoteSaveAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda las notas reposicionables de la ventana actual en un archivo. Todas las notas reposicionables se guardan: 

- Control Nota reposicionable. 

- Nota reposicionable creada a través de la AAF (automatic Application Feature). 

- note reposicionable creada mediante programación ([NoteCreate](../WDLang1/1000022395.md), [WinEdCreateNote](../WDLang1/1000022270.md), [ControlCreate](../WDLang1/1000020876.md)). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Export my notes]
NoteSaveAll()
IF ErrorOccurred = True THEN
	Error(ErrorInfo())
END

// Click [Import my notes]
NoteLoadAll()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`NoteSaveAll([<File>])`
---

**`<File>: (Cadena de caracteres opcional)`**

Nombre del archivo que contiene las notas reposicionables para guardar. 

Si no se especifica este parámetro, las notas reposicionables se guardan en el archivo: &lt;Nombre de la ventana&gt;.wdnote encontrado en el directorio de datos. El archivo de respaldo corresponde a: 
```txt
fDataDir()\<MyWindow..Name>.wdnote
```




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función es equivalente a la AAF (automatic Application Features) para importar notas sin mostrar el selector de archivos. Para obtener más información, consulte [note reposicionable (AAF)](../Editeurs/1000021820.md). 

- Los elementos guardados son: la posición de la nota, su contenido, su fondo Color y las características de la fuente utilizada.

- Luego, las notas reposicionables guardadas pueden ser importadas: 

	- En el editor (mediante arrastrar y soltar o abriendo el archivo de notas). 

	- Mediante la programación a través de [NoteLoadAll](../WDLang1/1000022397.md). 




- La función no tiene efecto si no se ha creado ninguna note.

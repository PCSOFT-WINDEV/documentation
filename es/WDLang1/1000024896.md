
## InvalidInputShowTooltip (Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>26</B>, se recomienda utilizar la nueva funcionalidad para comprobar la entrada obligatoria o no válida. Para obtener más información, consulte [Entrada obligatoria o no válida](../WDChamp/9500154.md).
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un aviso de error en un tooltip debajo de un control Campo de entrada para indicar por qué la entrada no es válida.  


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Exit from EDT_PinCode
IF Length(EDT_PinCode)<4 THEN
	InvalidInputShowTooltip("The PIN code must contain at least 4 characters.")
	SetFocusAndReturnToUserInput(EDT_PinCode) 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`InvalidInputShowTooltip(<Tooltip text>)`
---

**`<Tooltip text>: (Cadena de caracteres)`**

Contenido del tooltip de error que se muestra debajo del control Campo de entrada en caso de entrada no válida. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El aviso desaparece cuando el usuario hace clic en el tooltip o pulsa una tecla del teclado. 

- Se debe llamar a la función **InvalidInputShowTooltip** desde un control Campo de entrada, de lo contrario no tendrá ningún efecto.

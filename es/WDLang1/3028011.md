
## DDEUpdate( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica un dato de relacionado.

**DDEUpdate** envía el nuevo valor de un dato de programa a todas las aplicaciones que han creado un enlace "caliente" o "tibio" con este dato. Por lo tanto, el programa debe haber aceptado un enlace a estos datos y debe haber ramificado un Procedure en respuesta al DDE Event para solicitar el enlace (con [DDEEvent](../WDLang1/3028009.md) con el *DDEtoLink* Event).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DDEEvent("LinkProc", DDEtoLink)
DDEUpdate("MySubject", "MyItem", "abcd")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEUpdate(<Topic> , <Object> , <Value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los datos fueron modificados,

- <u><u><u><u>False</u></u></u></u> si se produce un error. [DDEError](../WDLang1/3028006.md) devuelve los detalles del error.




**`<Topic>: (Cadena de caracteres)`**

Tema de conexión.

**`<Object>: (Cadena de caracteres)`**

Objeto de conexión.

**`<Value>: (Cadena de caracteres)`**

Nuevo valor para los datos relacionado.

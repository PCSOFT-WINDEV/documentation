
## Error (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra un mensaje de error personalizado en una ventana de error del sistema.
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=fonctions_dialogue%20-%20HC%20N%B0003.gif)


Observaciones: 

- **Esta ventana de advertencia es bloqueante**. Para continuar ejecutando la aplicación, el usuario debe hacer clic en "OK". 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para no bloquear la aplicación, utilice la función [ErrorAsynchronous](../WDLang1/1000025306.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) WINDEV permite aplicar la plantilla del proyecto al cuadro de diálogo. Para obtener más información, consulte [Personalizar cuadros de diálogo](../WDLang1/3021003.md).

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) En un Webservice, la función **Error** permite definir el error devuelto por el Webservice. Si se llama varias veces a la función **Error**, los mensajes de error se concatenan.

## Sintaxis

### Mostrar un mensaje de error personalizado

`Error(<Text> [, <Line 2> [... [, <Line n>]]])`
---

**`<Text>: (Cadena de caracteres)`**

Texto a mostrar.

**`<Line 2>: (Cadena de caracteres opcional)`**

Texto que aparece en la segunda línea.



**`<Line n>: (Cadena de caracteres opcional)`**

Texto que aparece en la enésima línea.

## Observaciones




### Características del cuadro de diálogo
<a name="caracteristicas_del_cuadro_dialogo_ELTPARAGRAPHE000121"></a>



- El título del cuadro de diálogo corresponde al título de la ventana (o página) actual.
	

- Para modificar o definir el título del cuadro de diálogo, utilice la función [NextTitle](../WDLang1/3038043.md).




- El mensaje está alineado a la izquierda.

- El texto del botón depende del idioma de ejecución de Windows.

- El número máximo de caracteres está limitado a 4096. Si se pasa una cadena más grande como parámetro, se truncará.

- El ícono que se muestra (cruz blanca) no se puede modificar.




![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) **Para personalizar este cuadro de diálogo** (y todos los cuadros de diálogo del sistema en la aplicación), seleccione la opción "Personalizar ventanas del sistema (Info, YesNo, Confirm, Dialog)" en la pestaña "Estilo" de la descripción del proyecto. Para obtener más información, consulte [Personalizar cuadros de diálogo](../WDLang1/3021003.md).
<a name="NOTE0_2"></a>




### Salto de línea en el cuadro de diálogo
<a name="salto_linea_cuadro_dialogo_ELTPARAGRAPHE000136"></a>

La cadena predefinida CR (Carriage Return) se puede utilizar para pasar a la línea siguiente. Por ejemplo:

```wl
Error("An error was detected while entering the address." + CR + ...
	"Please modify the customer details.")
```

equivale a: 

```wl
Error("An error was detected while entering the address. ", ...
	"Please modify the customer details.")
```


También se puede usar la sintaxis de las cadenas multilínea. Por ejemplo: 

```wl
Error(
[
An error was detected while entering the address. 

Please modify the customer details.
])
```

### Mostrar el último mensaje de error
<a name="mostrar_ultimo_mensaje_error_ELTPARAGRAPHE000155"></a>

Al llamar a la función **Error** sin ningún parámetro (sintaxis 2) se muestra el mensaje de error de la última función WLanguage que provocó un error.

En este caso, el siguiente código:


```wl
Error()
```


equivale a:


```wl
Error(ErrorInfo())
```


**Atención**: La función **Error** debe utilizarse inmediatamente después de llamar a la función que falló. Si se llama a una función WLanguage, ésta puede restablecer y cambiar el informe de error devuelto por la función **Error**.

### Varios
<a name="varios_ELTPARAGRAPHE000180"></a>

- Los timers (función [TimerSys](../WDLang1/3015006.md)) no se detienen al llamar a esta función.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Los eventos de Windows no se detienen al llamar a esta función.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Puede usar el asistente de código para introducir la función en el editor de código. Según las características introducidas en el asistente, la función utilizada puede ser diferente en el código que se genera automáticamente.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) La función [DelayBeforeClosing](../WDLang1/3038040.md) limita el tiempo de visualización de todos los cuadros de diálogo. El cuadro de diálogo se cierra automáticamente. Para los cuadros de diálogo de pregunta o confirmación, la respuesta esperada corresponde al botón predeterminado. 
	![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para limitar solo el tiempo de un cuadro de diálogo de tipo Error en la pantalla, utilice la función [ErrorWithTimeout](../WDLang1/1000020529.md).

- El texto de los botones se muestra en el idioma utilizado por Windows en el equipo actual.


## ErrorOccurred (Variable)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
**ErrorOccurred** es un Variable preestablecido de WLanguage que indica el resultado de la última función WLanguage utilizada.

**ErrorOcurrió** corresponde a:

- <u><u><u>False</u></u></u> si la última función WLanguage funcionó correctamente.

- <u><u><u>True</u></u></u> si la última función WLanguage utilizada provocó un error no fatal.




Observación: Los detalles del error ocurrido son devueltos por [ErrorInfo](../WDLang1/3013008.md) o [HErrorInfo](../WDLang4/3044071.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
nFile is int
nFile = fOpen("C:\MyDirectories\File.txt", foReadWrite)
IF nFile <> -1 AND ErrorOccurred = False THEN
	...
ELSE
	Error(ErrorInfo())
END
```

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para gestionar los errores fatales, utilice [WHEN EXCEPTION](../WDLang1/3034001.md).

- Los errores fatales que no se procesan hacen que la aplicación se detenga inmediatamente. Aparece automáticamente un mensaje de error.

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Este mensaje se puede configurar al crear el ejecutable.

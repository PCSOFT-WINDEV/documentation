<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000272"></a>
WINDEV, WEBDEV y WINDEV Mobile incluyen varias funciones para mostrar, crear, copiar y guardar dibujos e imágenes desde sus aplicaciones o sitios.

WINDEV, WEBDEV y WINDEV Mobile proponen funciones que permiten:

- realizar varios dibujos,

- mostrar todo tipo de imágenes,

- copiar dibujos e imágenes,

- guardar dibujos e imágenes.

### Uso de las funciones de dibujo en Linux
<a name="uso_las_funciones_dibujo_linux_ELTPARAGRAPHE000083"></a>

Para utilizar las funciones de dibujo en Linux, se debe instalar la librería QT (Core y GUI) versión 5 (o posterior). 

**En Ubuntu**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
sudo apt-get install libqtcore5
sudo apt-get install libqtgui5
```


**En mandriva 2010 Spring**, se puede instalar esta biblioteca con los siguientes comandos: 


```txt
urpmi libqtgui5
```

## ¿Cómo dibujar?
<a name="¿como_dibujar_ELTTEXTE000344"></a>


### Dibujar en un control Imagen
<a name="dibujar_control_imagen_ELTPARAGRAPHE000113"></a>

Para dibujar en un control Imagen: 

1. Inicialice el dibujo en el control Imagen (función [dStartDrawing](../WDLang1/3029009.md)). Esta inicialización permite asignar un área de memoria específica para la gestión de dibujos.

2. Dibuje con las [funciones de dibujo](../WDLang1/3029033.md).

3. Si es necesario, inserte texto en los dibujos con la función [dText](../WDLang1/3029032.md).

4. Amplíe y copie las imágenes con la función [dCopyImage](../WDLang1/3029007.md).

5. Guarde el dibujo con las siguientes funciones [dSaveImageBMP](../WDLang1/3029026.md), [dSaveImageGIF](../WDLang1/3029040.md), [dSaveImageJPEG](../WDLang1/3029038.md), [dSaveImagePNG](../WDLang1/3029027.md).

6. Imprima el archivo con la función [iPrintImage](../WDLang5/3046017.md).

7. Si es necesario, destruya el dibujo en la memoria con la función [dEndDrawing](../WDLang1/3029019.md).





### Dibujar en una variable de tipo Image
<a name="dibujar_una_variable_tipo_image_ELTPARAGRAPHE000162"></a>

Para dibujar en una variable de tipo Image, se deben cumplir las siguientes condiciones:

1. Declarar una variable de tipo [Image](../WDLang1/1000019650.md). 

2. Inicialice el dibujo en la variable (función [dStartDrawing](../WDLang1/3029009.md)). Esta inicialización permite asignar un área de memoria específica para la gestión de dibujos.

3. Dibuje con las [funciones de dibujo](../WDLang1/3029033.md).

4. Si es necesario, inserte texto en los dibujos con la función [dText](../WDLang1/3029032.md).

5. Guarde el dibujo con las siguientes funciones [dSaveImageBMP](../WDLang1/3029026.md), [dSaveImageGIF](../WDLang1/3029040.md), [dSaveImageJPEG](../WDLang1/3029038.md), [dSaveImagePNG](../WDLang1/3029027.md).

6. Imprima el archivo con la función [iPrintImage](../WDLang5/3046017.md).

7. Si es necesario, destruya el dibujo en la memoria con la función [dEndDrawing](../WDLang1/3029019.md).

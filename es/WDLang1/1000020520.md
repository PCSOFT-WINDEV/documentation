


## ControlTypeInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el icono correspondiente al tipo especificado de control.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
nType is int = EDT_Name.Type
ControlImg is Image = ControlTypeInfo(nType)
IMG_Control = ControlImg
```
<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
ControlImg is String = ControlTypeInfo(typCalendar)
IMG_Control = ControlImg
```

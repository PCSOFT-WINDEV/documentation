


## &lt;Tipo Date&gt;.ToInteger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una fecha en un número entero. Este entero corresponde a:

- el número de días transcurridos entre el 1 de enero de 1800 y una fecha especificada.

- el número de días transcurridos entre el 1 de enero de una fecha especificada y otra fecha especificada.











<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDate is Date
// Number of days elapsed between 01/01/1800 and 12/25/2001
MyDate = "20011225"
MyDate.ToInteger()   // Returns 73773

// Number of days elapsed between 01/01/1800 and 01/03/1800
MyDate = "18000103"
MyDate.ToInteger()   // Returns 3
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Date>.ToInteger([<From beginning of year>])`
---

**`<Result>: (Entero)`**

Número de días transcurridos entre: 

- El 1 de enero de una fecha determinada y otra fecha especificada.

- El 1 de enero de 1800 y una fecha determinada




**`<Date>: (Variable de tipo Date o DateTime)`**

Fecha a convertir. Esta fecha puede corresponder a : 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.




**`<From beginning of year>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> para obtener el número de días transcurridos entre el 1 de enero de una fecha determinada y otra fecha especificada,

- <u><u><u><u>False</u></u></u></u> (valor predeterminado) para obtener el número de días transcurridos entre el 1 de enero de 1800 y la fecha especificada.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La validez de la fecha pasada como parámetro se verifica. Si la fecha no es válida, se muestra un mensaje de error. La validez de las fechas se puede comprobar con la función [DateValid](../WDLang1/3027003.md).

- Las funciones **&lt;Tipo Date&gt;.ToInteger** y [DateDifference](../WDLang1/3027001.md) permiten calcular el número de días transcurridos entre dos fechas. 

- El formato de almacenamiento de fechas permite almacenar fechas del 01/01/0001 al 31/12/9999. Los cálculos sobre las fechas realizadas con las [funciones WLanguage](../WDLang1/1000023943.md) o las [propiedades WLanguage](../Proprietes/2514010.md) son precisas a partir del 1 de enero de 1583.

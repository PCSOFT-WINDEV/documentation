
## Beep( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emite un pitido. Este sonido corresponde al sonido del sistema de Windows (ajustable en el panel de control o eligiendo un tema de escritorio).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF HFound() = False THEN
	Message("Not found")
	Beep()
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Beep()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si no se encuentra (o instala) ninguna tarjeta de sonido, el altavoz que se encuentra en la placa base se utiliza para emitir un pitido.

- La función no se bloquea: en cuanto se inicia el sonido, se ejecuta el resto del programa. El programa no espera al final del sonido.

- El usuario puede deshabilitar el sistema de sonido a través del panel control.

- Si se realizan varias llamadas consecutivas a la función **Beep** en el mismo proceso, se escuchará un solo sonido.

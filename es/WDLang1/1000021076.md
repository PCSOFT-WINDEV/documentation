


## GanttLink (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **GanttLink** permite definir las características avanzadas de una relación entre dos tareas de un diagrama de Gantt. Las características de este enlace pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Retrieve all the links that point to "ID1"
arrLink is Array of GanttLink
arrLink = GanttListLink(COL_Gantt, "ID1")
Link is GanttLink
FOR EACH Link of arrLink
	Trace("Linked to " + Link.SourceID)
END
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de las variables GanttLink
<a name="propiedades_especificas_las_variables_ganttlink_ELTPARAGRAPHE000028"></a>

Las siguientes propiedades pueden ser utilizadas para manipular un enlace entre dos tareas Gantt:

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| escribir a máquina | Constante de tipo Integer | Tipo de enlace. Los tipos de enlaces disponibles son los siguientes: <br><br>- *gltStartToStart*: "Enlace"Start to Start: La tarea de destino no puede iniciarse mientras no se inicie la tarea fuente.<br><br>- *gltStartToEnd*: "Enlace"De principio a fin: La tarea de destino no puede finalizar mientras no se inicie la tarea de origen (caso poco frecuente).<br><br>- *gltEndToStart* (valor de Default): "Enlace"End to Start: La tarea de destino no puede iniciarse mientras no haya finalizado la tarea de origen.. <br><br>- *gltEndToEnd*: "Enlace"End to End: La tarea de destino no puede finalizar mientras no haya finalizado la tarea de origen.<br><br><br> |
| ID de destino | Cadena de caracteres | Identificador de la tarea de destino. Este identificador corresponde al **ID** Property definido en el [GanttTask](../WDLang1/1000020881.md) Variable. |
| ID de fuente | Cadena de caracteres | Identificador de la tarea fuente. Este identificador corresponde al **ID** Property definido en el [GanttTask](../WDLang1/1000020881.md) Variable. |


<a name="NOTE0_2"></a>


### Notas
<a name="notas_ELTPARAGRAPHE000071"></a>

- Un enlace entre 2 tareas se representa con una flecha.

- Este tipo de Variable: 

	- se utiliza para añadir un enlace a través de [EnlaceGanttAddLink](../WDLang1/1000021075.md),

	- puede ser devuelto por [Enlace GanttListLink](../WDLang1/1000021079.md). 




- Default crea un enlace "End to Start" entre dos tareas: la tarea correspondiente a la **SourceID** Property debe haber finalizado antes de que la tarea **DestinationID** pueda comenzar. 

- Según su tipo, los enlaces se dibujan en los bordes de inicio o final de las tareas.. 



<a name="NOTE0_3"></a>


### Utilización de variables GanttLink
<a name="utilizacion_variables_ganttlink_ELTPARAGRAPHE000090"></a>Las variables **GanttLink** son utilizadas por las siguientes funciones: 



|   |   |
| --- | --- |
| [GanttAddLink](../WDLang1/1000021075.md) | Crea un enlace "End to Start" entre dos tareas mostradas en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |
| [GanttDeleteLink](../WDLang1/1000021080.md) | Elimina los enlaces de una columna del diagrama de Gantt en una tabla o control Tabla TreeView. |
| [GanttListLink](../WDLang1/1000021079.md) | Enumera los enlaces que se encuentran en una columna del diagrama de Gantt que se encuentra en una tabla (o tabla de vista de árbol) control. |

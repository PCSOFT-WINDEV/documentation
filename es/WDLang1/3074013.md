


## TableEnd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene la iteración manual del archivo de datos relacionado a un control Tabla basado en un archivo de datos. Después de ejecutar la función **TableEnd**, los registros que se encuentran después del registro actual del control Tabla no se leerán.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Stop browsing the file linked to the Table control "TABLE_ProductTable"
TableEnd(TABLE_ProductTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TableEnd([<Table control>])`
---

**`<Table control>: (Nombre de control opcional)`**

Nombre de control Tabla a manipular.

Si este parámetro no se especifica o es una cadena vacía (""), el control Tabla utilizado será el relacionado con el Event actual.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000059"></a>

- La función **TableEnd** puede utilizarse en un control Tabla de selección única o múltiple.

- No se debe utilizar la función **TableEnd** en los controles Tabla archivo de datos que utilizan una navegación automática.





<a name="NOTE0_2"></a>


### Eventos de la control Tabla
<a name="eventos_control_tabla_ELTPARAGRAPHE000077"></a>

La función **TableEnd** solo puede utilizarse en los siguientes eventos del control Tabla:

- Inicialización.

- Leyendo el anterior Record.

- Leyendo el próximo Record.



<a name="NOTE0_3"></a>


### Reinicializar un control Tabla
<a name="reinicializar_control_tabla_ELTPARAGRAPHE000098"></a>

Después de llamar a **TableEnd** en un evento de control Tabla, volver a mostrar los datos de control Tabla. Por ejemplo:

```wl
TableDisplay(TABLE_TABLE1, taInit)
```

<a name="NOTE0_4"></a>


### Visualización de los registros encontrados entre dos valores
<a name="visualizacion_los_registros_encontrados_entre_dos_valores_ELTPARAGRAPHE000115"></a>

Para ver los registros que corresponden a un intervalo de valores en un control Tabla, se recomienda utilizar la función [HFilter](../WDLang4/3044100.md) (en lugar de la función **TableEnd**).

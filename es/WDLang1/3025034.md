


## Funciones de gestión de los controles RTF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de edición RTF:



|   |   |
| --- | --- |
| [RTFAdd](../WDLang1/3025024.md) | Añade una cadena de caracteres (en RTF o no) al final de una control Campo de entrada RTF , un control RTF en una reporte o una cadena de caracteres que contenga un texto en RTF. |
| [RTFAddLink](../WDLang1/1000021062.md) | Añade un enlace a una sección de texto en un RTF Editar control. |
| [RTFInsert](../WDLang1/3025025.md) | Inserta una cadena de caracteres en RTF en el contenido de una control RTF (que se encuentra en una ventana o en una reporte) o en el contenido de una cadena de caracteres en RTF. |
| [RTFLoad](../WDLang1/3025041.md) | Carga un archivo en RTF en una control RTF (que se encuentra en una ventana o en una reporte). |
| [RTFSearch](../WDLang1/3025043.md) | Busca una cadena de caracteres en una control RTF (que se encuentra en una ventana o en una reporte) o en una cadena de caracteres que contiene texto en formato RTF. |
| [RTFSelección](../WDLang1/3025023.md) | Cambia o enumera los atributos del texto actualmente seleccionado en un RTF control (que se encuentra en una ventana o en un reporte). |
| [RTFSustituir](../WDLang1/3025038.md) | Reemplaza o inserta una cadena de caracteres en una control RTF (que se encuentra en una ventana o en una reporte) o en una cadena en RTF (que contiene un texto en RTF). |
| [RTFToHTML](../WDLang1/3025049.md) | Devuelve la cadena HTML correspondiente a la cadena RTF especificada. |
| [RTFToText](../WDLang1/3025032.md) | Devuelve el texto sin las características de formato específicas del formato RTF. |


## ControlOver( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la control sobre la que se pasa el cursor del ratón.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Display the name of the hovered controls in a trace

// -- Initialization code of window
// Take the window and the window controls into account
Event(Hover, "*.* ", 512)
```

```wl
// -- "Hover" procedure
PROCEDURE HoverOver()
Trace(ControlOver())
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlOver([<Invisible Controls> [, <Hovered Window>]])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la columna o ventana de control, sobre la que se pasa el cursor del ratón. 

- Cadena vacía ("") si no se pasa el cursor del ratón por encima de control.




**`<Invisible Controls>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si se deben tener en cuenta los controles de Invisible.

- <u><u><u><u>False</u></u></u></u> (por defecto) si los controles de Invisible no deben tenerse en cuenta. 




**`<Hovered Window>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u> si el nombre de la ventana debe ser devuelto cuando no se pasa el cursor del ratón por encima de control,

- <u><u><u><u>False</u></u></u></u> (Default) si se devuelve una cadena vacía cuando no se pasa el cursor del ratón por encima de control.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### control pasó por encima con el cursor del ratón
<a name="control_paso_por_encima_con_cursor_del_raton_ELTPARAGRAPHE000052"></a>

- Si se pasa el cursor del ratón por encima de varios control s, se devuelve el nombre del control con la posición de orden Z más alta (lo que significa el nombre del control en primer plano).

- Si se pasa por encima de una columna tabla con el cursor del ratón, se devuelve el nombre de la columna.

- Los controles que no se encuentran en el plano activo se consideran como Invisible.

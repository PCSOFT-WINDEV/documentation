


## Funciones para la gestión de los controles de la Zona Multilínea (sintaxis del prefijo)
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de la Zona Multilínea mediante la sintaxis de los prefijos:



|   |   |
| --- | --- |
| [&lt;Zona multilínea&gt;.AddLine](../WDLang1/1000020216.md) | Añade una nueva dinámica fila a un control Zona multilínea. |
| [Borrar](../WDLang1/1000020218.md) | Borra un fila dinámico de un control Zona multilínea. |
| [&lt;DeleteAll](../WDLang1/1000020219.md) | Elimina todas las filas dinámicas de una Zona Mutilina control. |
| [InsertLine](../WDLang1/1000020217.md) | Inserta una nueva dinámica fila en un control Zona multilínea. |

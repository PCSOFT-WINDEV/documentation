


## &lt;Tipo ANSI string&gt;.AnsiToOem (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena ANSI (Windows) en una cadena OEM (DOS). Los caracteres acentuados son diferentes en ambos estándares. La conversión consiste en modificar los caracteres acentuados.

**Recordatorio**:

- ANSI es el conjunto de caracteres utilizado en Windows.

- OEM es el conjunto de caracteres utilizado en DOS (o consola de Windows).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is string
StringToConvert is string = "Crème brûlée"
Res = StringToConvert.AnsiToOem()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <String to convert>.AnsiToOem()`
---

**`<Result>: (Cadena de caracteres ANSI)`**

Cadena de caracteres OEM.

**`<String to convert>: (Cadena de caracteres ANSI)`**

Cadena de caracteres ANSI a convertir.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función **&lt;Tipo ANSI string&gt;.AnsiToOem** no convierte caracteres con un código ASCII entre 0 y 31 (caracteres de control).

- **Atención**: No utilice la función **&lt;Tipo ANSI string&gt;.AnsiToOem** con una cadena que contenga caracteres específicos de OEM (caracteres con acentos): estos caracteres se convertirán y el resultado será incorrecto.

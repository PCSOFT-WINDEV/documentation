
## FinPaymentNb( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de pagos necesarios para amortizar un capital de acuerdo con una tasa determinada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Number of necessary payments: the amount of payments
// is set to 7500 Euros,
// the interest rate is set to 5% and the amount borrowed is 
// set to 42000 Euros.
ResNbPayment = FinPaymentNb(-7500, 5, 42000)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinPaymentNb([<Value of payment> [, <Rate> [, <Capital>]]])`
---

**`<Result>: (Real)`**

Número de pagos a realizar. Este número puede ser:

- negativo: se debe hacer un pago;

- constructivo: un crédito debe ser hecho.




**`<Value of payment>: (Real o moneda opcional)`**

Valor del pago (0 por Default). Para un crédito, este valor debe ser negativo.

**`<Rate>: (Real opcional)`**

Tasa de interés, expresada en porcentaje (1 por Default). Por ejemplo, para una tasa del 5%, este parámetro será set a 5.

**`<Capital>: (Real o moneda opcional)`**

Cantidad del préstamo (1 por Default).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
**Atención**: La función **FinPaymentNb** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

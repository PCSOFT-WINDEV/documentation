
## ToolTipColor( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica los colores utilizados para mostrar los consejos de la herramienta.. El Color del texto y el Color de fondo del tooltip pueden ser modificados. Esta modificación se realiza para la aplicación actual.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ToolTipColor(YellowToolhelp, DarkBlue)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`ToolTipColor(<Text color> , <Background color>)`
---

**`<Text color>: (Integro)`**

Color del texto mostrado en las tooltips de la ventana actual. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




**`<Background color>: (Integro)`**

Color de fondo de los tooltips que se encuentran en la ventana actual. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los tooltips son habilitados por Default en los controles. Para desactivarlas, utilice [ShowToolTip](../WDLang1/3025014.md).

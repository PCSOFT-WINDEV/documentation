


## TablaColumnaSelectOccurrence (Función)




	<B><B>Advertencia</B></B>

	A partir de la versión <B><B>16 (56)</B></B>, esta funcionalidad se conserva por motivos de compatibilidad. Se recomienda utilizar la función [TableSelectCount](../WDLang1/3074030.md).

<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de columnas seleccionadas que se encuentran en una control Tabla.

**Observación**: La control Tabla debe permitir el modo de selección por columna. Esta opción se puede configurar en la pestaña "UI" de la ventana description de control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NbColSelection is int 
ColNum is int

NbColSelection = TableColumnSelectOccurrence(TABLE_TABLE1)
Trace("TableColumnSelectOccurrence = " + NbColSelection)

FOR i = 1 TO NbColSelection
	ColNum = TableColumnSelect(TABLE_TABLE1,i)
	Trace("The column " + ColNum + " is selected")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TableColumnSelectOccurrence(<Table control>)`
---

**`<Result>: (Integro)`**

Número de columnas seleccionadas.

**`<Table control>: (Nombre del control)`**

Nombre de control Tabla a manipular. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para permitir la selección de una columna, debe habilitar en las características de control Tabla: 

- el modo de selección "Columnas" (selección simple o múltiple) en la pestaña "UI" de la ventana description de control Tabla. 

- el modo de selección "Cells" (selección múltiple) en la pestaña "UI" de la ventana description de control Tabla. 




El usuario final puede seleccionar las columnas haciendo clic en el encabezado de la columna (como para una clasificación).

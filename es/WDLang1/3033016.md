
## eDefaultMemCurrency( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica o modifica la moneda almacenada que utiliza Default.

**Observaciones**:

- La función **eDefaultMemCurrency** define la divisa almacenada solo para los controles "Monetario + Euro" cuya divisa almacenada corresponde a la opción "Divisa predeterminada" en el editor de ventanas o páginas.

- La moneda almacenada que usa Default es el Euro.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Modify the stored currency used by default
ResDefaultStoredCurrency = eDefaultMemCurrency(eDEM)
// The currency stored used by default is the German Mark
```

<a name="SYNTAX1"></a>

## Sintaxis

### Modificar la moneda almacenada que utiliza Default

`<Result> = eDefaultMemCurrency(<New currency>)`
---

**`<Result>: (Integro)`**

Identificador de la moneda almacenada utilizada por Default antes de la modificación.

**`<New currency>: (Entero o constante)`**

Identificador o constante associated con la nueva moneda almacenada por Default.  

Esta moneda se tendrá en cuenta: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) durante la siguiente apertura de la ventana (realizada por [entablar](../WDLang1/3038035.md) o [aprovechamiento](../WDLang1/3038044.md)).





<a name="SYNTAX2"></a>

### Identificar la moneda almacenada utilizada por Default

`<Result> = eDefaultMemCurrency()`
---

**`<Result>: (Integro)`**

Identificador de la moneda almacenada utilizada por Default.

## Observaciones




### Modificación de la moneda almacenada en el Default (sintaxis 1)
<a name="modificacion_moneda_almacenada_default_sintaxis_1_ELTPARAGRAPHE000083"></a>

La función **eDefaultMemCurrency** debe llamarse en el código de inicialización del proyecto. La moneda almacenada no debe modificarse varias veces en la misma aplicación.
<a name="NOTE0_2"></a>




### Intercambio tabla
<a name="intercambio_tabla_ELTPARAGRAPHE000093"></a>

Una ***tabla de cambio*** se almacena en la memoria. Cada moneda es con associated:

- Un número de moneda utilizado para identificar la moneda por programación.

- Una constante utilizada para identificar la moneda por programación.

- Una leyenda (hasta 7 caracteres): esta leyenda se muestra en la control "Moneda + Euro" si la máscara de entrada lo permite (se indica el signo $).

- Un tipo de cambio. Este tipo de cambio corresponde al tipo de cambio de la moneda en relación con el euro.

En este intercambio tabla, usted tiene la capacidad de:

- añadir nuevas monedas modificando las monedas personalizadas si es necesario ("monedas definidas por el usuario" de 16 a 25) con [eInitCurrency](../WDLang1/3033003.md),

- especificar el tipo de cambio de cada moneda con [eInitCurrency](../WDLang1/3033003.md).




**Atención**: Las monedas cuyo tipo de cambio es set a 0 no se mostrarán en el menú contextual de la "Moneda + Euro" control. Para definir el tipo de cambio de estas monedas, debe redefinirlas en el código de inicialización del proyecto con [eInitCurrency](../WDLang1/3033003.md).

### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000405"></a>

La función **eDefaultMemCurrency** es equivalente a la función [MemoryCurrency](../Proprietes/2510013.md) Property (&lt;nombre del control&gt;.MemoryCurrency).


## SchedulerSelectedRange (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha y hora de inicio o de fin de la franja horaria seleccionada en una control Planificador.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Gets the selected range in a Scheduler control
dtStartDate is DateTime = SchedulerSelectedRange(SCH_SchedulePlanning, schStartDate)
dtEndDate is DateTime = SchedulerSelectedRange(SCH_SchedulePlanning, schEndDate)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SchedulerSelectedRange(<Scheduler control> [, <Start or end of range>])`
---

**`<Result>: (FechaHora)`**



- Fecha y hora solicitadas (para el inicio o fin del rango),

- Vaciar la cadena (""") si no se ha realizado ninguna selección en la control Planificador. 




**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar. 

**`<Start or end of range>: (Constante de tipo Integer)`**

Indica la información a devolver: 


|   |   |
| --- | --- |
| *schEndDate* | Información devuelta: Fecha y hora de finalización de la franja horaria seleccionada. |
| *schStartDate* | Información devuelta: Fecha y hora de inicio de la franja horaria seleccionada. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función es equivalente a las propiedades [StartDate](../Proprietes/1000017426.md) y [EndDate](../Proprietes/1000017427.md) del control Planificador.

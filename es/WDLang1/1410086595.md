
## CameraDisplay (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra el video capturado por una cámara web o por la cámara de un dispositivo móvil en un control Cámara.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the video captured by the "CAM_Camera" control
ResFilm = CameraDisplay(CAM_Camera)
IF ResFilm = True THEN
	// Displays a caption
	CAM_Camera.Caption = "Now in the kitchen"
ELSE
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CameraDisplay(<Camera control>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el video capturado se muestra correctamente en el control Cámara,

- <u><u><u><u>False</u></u></u></u> si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000050"></a>

- ![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una cámara al equipo actual.




**Observación**: Esta función es equivalente a la función [VideoDisplay](../WDLang1/3079004.md). 




<a name="NOTE0_2"></a>




### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000072"></a>

- En tiempo de ejecución, el video capturado por la cámara se muestra automáticamente en cuanto se abre la ventana que contiene el control Cámara. En este caso, no es necesario llamar a la función **CameraDisplay**.

- Para detener la grabación de video, utilice la función [CameraStop](../WDLang1/1410086596.md).


## SysIconDelete( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra una Icon de la barra de tareas (junto al reloj). Esta Icon fue añadida en la barra de tareas por [SysIconAdd](../WDLang1/3073001.md).

![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Icone.gif) o ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconePPC.gif)

**Observación**: El Icon se borra automáticamente al cerrar la aplicación.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete the icon that was previously added into the taskbar
ResDelIcon = SysIconDelete()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysIconDelete()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se eliminó el ícono de la barra de tareas,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="XComponent"></a>

## Componente:
wd280std.dll




## <PDF Reader>.SeekFirst (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa la búsqueda de un texto en un fichero PDF visualizado por el lector PDF control y se posiciona en la primera ocurrencia encontrada (a partir del Page actual ). Se selecciona el texto encontrado.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Seeks a text in a PDF Reader control and displays it
PDF_MyPDFReader.SeekFirst("company")
// Finds the next element
IF PDF_MyPDFReader.SeekNext() = False THEN
	Info("Search ended")
END

// Finds the previous element
PDF_MyPDFReader.SeekPrevious()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <PDF Reader control>.SeekFirst(<Text to find> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se encontró el texto en el documento,

- <u><u><u><u>False</u></u></u></u> si no se encontró el texto.




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<Text to find>: (Cadena de caracteres)`**

Texto que se encuentra en el contenido del lector de PDF control.

**`<Options>: (Constante o combinación de constantes opcional)`**

Opciones de búsqueda:


|   |   |
| --- | --- |
| *IgnoreCase* | Busca sin distinguir entre mayúsculas, minúsculas o acentos. |
| *WholeWord* | Busque la palabra completa (encerrada en caracteres de puntuación o en caracteres espaciados). |


De forma predeterminada, la búsqueda distingue entre mayúsculas y minúsculas: el texto buscado y el texto a encontrar deben tener el mismo caso. Para realizar una búsqueda que no distinga entre mayúsculas y minúsculas, utilice la constante *IgnoreCase*.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El texto y las opciones de búsqueda especificadas se tienen en cuenta automáticamente al ejecutar las funciones [.SeekNext](../WDLang1/1000023779.md) y [.SeekPrevious](../WDLang1/1000023777.md) después de la función **<PDF Reader>.SeekFirst**.

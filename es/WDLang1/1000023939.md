
## GetCountryISO( Función)

<DIV class="specObsolete">
	<IMG style="float:left;margin-right:10px;" src="https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=warning_big.png"><B>Advertencia</B><BR>
	A partir de la versión <B>25</B>, esta función se conserva por motivos de compatibilidad con versiones anteriores. Se recomienda utilizar la función [Funciones de gestión de continentes y países](../WDLang1/1000024945.md). 
</DIV><a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el código ISO de una Country que corresponde a su nombre según la ISO 3166-1.

Observación: También es posible para get el código ISO de un Country correspondiente a un tipo diferente de código ISO (por ejemplo, el código ISO de 2 letras correspondiente a un código ISO numérico). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// France ISO code
sFranceISOCode is string = GetCountryISO("France")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = GetCountryISO(<Country name or ISO code> [, <Type of ISO code>])`
---

**`<Result>: (Cadena de caracteres)`**

Código ISO de Country.

**`<Country name or ISO code>: (Cadena de caracteres)`**



- Nombre de Country ISO. 

- Código ISO de Country. La codificación de este código ISO es diferente del tipo de codificación deseado. 




**`<Type of ISO code>: (Constante opcional de tipo Integer)`**

Tipo de código ISO devuelto:



|   |   |
| --- | --- |
| *lcAlpha2Code* | Código ISO alfabético en 2 caracteres. |
| *lcAlpha3Code* | Código ISO alfabético en 3 caracteres. |
| *lcNumericCode* | Código ISO numérico. |


## DateSys (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la fecha del sistema (fecha actual definida en el equipo).

**Observaciones**: 

- Esta función es equivalente a la función [Today](../WDLang1/3027016.md).





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res = DateSys()   // Res = "20011225"
Res = DateToString(DateSys()) // Res = "25/12/2001"
Year = Left(DateSys(), 4)
// Year returns the current year (2001 for example)
```

## Sintaxis

### Obtener la fecha del sistema

`<Result> = DateSys()`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha del sistema en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Modificar la fecha del sistema

`DateSys(<New date>)`
---

**`<New date>: (Cadena de caracteres, variable de tipo Date o DateTime)`**

Nueva fecha del sistema. Esta fecha puede corresponder a : 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).  

- una cadena en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo (utilice CCC para las milésimas de segundo)). En este caso, solo se tiene en cuenta la fecha. 

- una variable de tipo Date. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la fecha.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para obtener la fecha del sistema en un formato distinto a AAAAMMDD, se debe definir el formato de la cadena con las funciones [DateToString](../WDLang1/3027025.md), [Right](../WDLang1/3024016.md), [Left](../WDLang1/3024001.md) o las [propiedades de fecha](../Proprietes/2514010.md). Se eligió el formato AAAAMMDD porque permite comprobar el orden de las fechas. Esto no sería posible con los formatos AAAADDMM o DDMMAAAA, por ejemplo. 

- En los sistemas actuales, el año de la fecha del sistema puede estar comprendido entre 1920 y 2120.

- **Atención**: Para modificar la fecha del sistema, debe tener los permisos de administrador en el equipo actual.

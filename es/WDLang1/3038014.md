
## PreviousWin( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la ventana que estaba en edición antes de la ventana que está actualmente en edición (para el mismo proyecto).. Esta función se utiliza para las ventanas sin modelo.

**Recordatorio**: La ventana actual es la ventana que tiene el foco: esta ventana está en edición y su barra de título es Active.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Name of the previous window
ResPreviousWin = PreviousWin()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = PreviousWin()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de la ventana que estaba previamente en edición,

- Alias de la ventana que estaba previamente en edición (si este alias fue especificado cuando se abrió la ventana).

- Cadena vacía ("") si no existe una ventana anterior, o si la ventana editada previamente se cerró antes de llamar a **PreviousWin**.

### Caracteres en mayúsculas/minúsculas
<a name="caracteres_mayusculasminusculas_ELTPARAGRAPHE000054"></a>

El nombre de la ventana puede escribirse en mayúsculas o minúsculas.. El operador "~=" (flexible equality) debe utilizarse para comparar el nombre de la ventana devuelto por **PreviousWin** con un nombre de ventana codificado.

Por ejemplo:


```wl
IF PreviousWin() ~= "EditWindow" THEN
	...
END
```

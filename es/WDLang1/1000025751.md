
## StyleDrawCaptionGrayed( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Personaliza cómo se dibuja el texto en los controles de salida de Grayed. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF LightSkinTemplateMode() THEN
	StyleDrawCaptionGrayed(sdcgDefault)
ELSE
	StyleDrawCaptionGrayed(sdcgMonochrome, LightGray)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`StyleDrawCaptionGrayed(<Style> [, <Color>])`
---

**`<Style>: (Constante de tipo Integer)`**

Estilo a utilizar: 



|   |   |
| --- | --- |
| *sdcgDefault* | El texto se dibuja con el estilo Default (texto gris en relieve). |
| *sdcgMonochrome* | El texto se dibuja utilizando la Color especificada con el parámetro &lt;Color&gt;. |



**`<Color>: (Entero opcional)`**

Color del texto que se utilizará si se especifica la constante *sdcgMonochrome*. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

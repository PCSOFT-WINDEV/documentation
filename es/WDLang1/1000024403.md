


## &lt;Procesamiento de palabras&gt;.toText (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un string a partir de los datos de una control Procesador de texto.

Esta cadena se puede guardar en un formato específico a partir de entonces.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString is string
MyString = WP_MyDocument.ToText()
// Save in a text file
fSaveText("C:\temp\Content.txt", MyString)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Word Processing control>.ToText()`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena de caracteres que contiene los datos. 

- Vacíe la cadena ("") si se ha producido un error o si no hay datos que extraer.




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HaspHLLogin(<Area identifier> , <Vendor code> [, <Options>])`
---

**`<Result>: (Integro)`**



- Identificador de sesión. Este identificador se utilizará en el programa [funciones para la gestión de claves Hasp](../WDLang1/1000018999.md).

- -1 si se produce un error. [ErrorInfo](../WDLang1/3013008.md) se utiliza para identificar el error que se ha producido. 




**`<Area identifier>: (Integro)`**

Nombre del área de la clave Hasp a la que debe conectarse la aplicación.

**`<Vendor code>: (Cadena de caracteres)`**

Código de proveedor obtenido del proveedor de la clave.

**`<Options>: (Combinación opcional de constantes de tipo Integer)`**

Modo de conexión a utilizar: 


|   |   |
| --- | --- |
| *haspCompatibleMode* | Esta constante permite utilizar claves HASP de tipo HASP3 y HASP4. |
| *haspIgnoreTSE* | Esta constante se utiliza para desactivar la comprobación del entorno "Terminal Server". |
| *haspLicensePerProcess* | El número de licencias se reducirá por cada Process que se inicie. Por ejemplo, dos instancias de la misma aplicación usarán dos licencias. |
| *haspLocalLicense* | Sólo se usarán las teclas Hasp conectadas a el equipo. |
| *haspNetworkLicense* | Sólo se utilizarán las claves HASP de red accesibles desde el ordenador actual. |


Por Default, no se utiliza ninguna constante.

**Atención**: No está permitido combinar las constantes *haspLocalLicense* + *haspNetworkLicense*.

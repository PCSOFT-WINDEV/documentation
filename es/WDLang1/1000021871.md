
## WinEdSelect (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve uno de los controles seleccionados cuando la interfaz es modificada por el usuario final (Modo "Modificación de UI por el usuario final" activado). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deselects the current control
X is Control
X <- WinEdSelect(WIN_Form) 
WInEdSelectMinus(WIN_Form, X)
```


<a name="Example2"></a>

```wl
// Enumerate the selected controls
X is Control
FOR i = 1 _TO_ WInEdSelectOccurrence(WIN_Form)
	X <- WinEdSelect(WIN_Form, i)
	Trace(x.Name)	
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WinEdSelect(<Window> [, <Index>])`
---

**`<Result>: (Control)`**



- Si la selección múltiple no está activada, este parámetro corresponde a la control seleccionada. 

- Si la selección múltiple está activada, este parámetro corresponde al Enésimo control de la selección (parámetro &lt;Índice&gt;). 


Este resultado puede asignarse a una [Control](../WDLang1/1000020967.md) Variable para ser utilizado. 

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. Esta ventana debe estar en modo de modificación ([WinEdActive](../WDLang1/1000021843.md)). 

**`<Index>: (Entero opcional)`**

Índice del elemento en una selección múltiple. Si no se especifica este parámetro, se tiene en cuenta la primera control seleccionada.


## Middle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae: 

- una subcadena de una cadena a partir de una posición especificada.

- parte de un buffer a partir de una posición especificada.






**Observación**: La misma funcionalidad se puede obtener con el operador [[]].




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Extract characters
Middle("Eponine", 2, 3)   // Returns "pon"
Middle("Eponine", 2)    // Returns "ponine"
Middle("Eponine", 50)    // Returns ""
Middle("Eponine", 2, 50)   // Returns "ponine"
// Equivalent to 
// MyString is string = Eponine
// MyString = MyString[[2,50]]

MyString is string = "Antananarivo - Madagascar"
Middle(MyString, 10, 7)   // Returns "ivo - M"
```

## Sintaxis

### Extraer una subcadena de una cadena

`<Result> = Middle(<Initial string> , <Start position> [, <Length>])`
---

**`<Result>: (Cadena de caracteres)`**



- Parte de &lt;Cadena inicial&gt;.

- Cadena vacía ("") si el parámetro &lt;Posición inicial&gt; es mayor que la longitud de &lt;Cadena inicial&gt;.

- EOT si el parámetro &lt;Posición inicial&gt; y/o &lt;Longitud&gt; son negativos. Se produce un error de WLanguage (error no fatal). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres que contiene la subcadena que se va a extraer. Esta cadena no se modifica.

**`<Start position>: (Entero)`**

Posición del primer carácter que se va a extraer. 
Si &lt;Posición inicial&gt; es un valor nulo, la extracción comienza al inicio de la cadena.

**`<Length>: (Entero opcional)`**

Número de **caracteres** a extraer. 
Si no se especifica este parámetro o es mayor que el número de caracteres que se encuentran después de &lt;Posición inicial&gt;, la cadena se devuelve a partir de &lt;Posición inicial&gt;.

### Extraer parte de un buffer

`<Result> = Middle(<Initial buffer> , <Start position> [, <Size>])`
---

**`<Result>: (Cadena de caracteres)`**



- Parte de &lt;Buffer inicial&gt;.

- Cadena vacía ("") si el parámetro &lt;Posición inicial&gt; es mayor que la longitud de &lt;Buffer inicial&gt;.

- EOT si el parámetro &lt;Posición inicial&gt; y/o &lt;Tamaño&gt; son negativos. Se produce un error de WLanguage (error no fatal). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

- Buffer no válido si &lt;Buffer inicial&gt; contiene una cadena UNICODE.




**`<Initial buffer>: (Buffer)`**

Buffer que contiene la parte que se va a extraer. Este buffer no se modifica.

**`<Start position>: (Entero)`**

Posición del primer byte que se va a extraer. 
Si &lt;Posición inicial&gt; es un valor nulo, la extracción comienza al inicio del buffer.

**`<Size>: (Entero opcional)`**

Número de **bytes** a extraer. 
Si no se especifica este parámetro o es mayor que el número de caracteres que se encuentran después de &lt;Posición inicial&gt;, la cadena se devuelve a partir de &lt;Posición inicial&gt;.

## Observaciones


### Función Middle y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000130"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Recordatorio**: En UNICODE, cada carácter ocupa dos bytes de memoria. Por lo tanto, el tamaño de memoria de la cadena UNICODE es el doble del número de caracteres que contiene realmente la cadena.

El primer carácter de la cadena está en el índice 1.

**Observación**: Si el resultado en una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

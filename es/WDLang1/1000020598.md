


## RadioButtonAdd (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una nueva opción a una control Botón de opción. La opción se añade después de las opciones existentes. 




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add 2 options 
RadioButtonAdd(RADIO_Options, "Read a book on a tablet")
RadioButtonAdd(RADIO_Options, "Read a paperback")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RadioButtonAdd(<Radio Button control> , <Caption> [, <Sub-caption> [, <Image>]])`
---

**`<Result>: (Integro)`**



- Índice de la nueva opción. 

- -1 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Radio Button control>: (Nombre del control)`**

Nombre de la control Botón de opción a la que se añadirá la opción. La opción se añade después de las opciones existentes.

**`<Caption>: (Cadena de caracteres)`**

Texto de la opción a añadir.

**`<Sub-caption>: (Cadena de caracteres opcional)`**

Subtexto de la opción a añadir. 
Si no se especifica este parámetro, la nueva opción no tiene subtexto. 
Si se especifica este parámetro, el subtítulo se mostrará incluso si el botón de radio utiliza un modo de visualización sin subtítulo..


**`<Image>: (Cadena de caracteres opcional)`**

Imagen associated con la opción de añadir. 
Si no se especifica este parámetro, la opción añadida no tiene Image.
Si se especifica este parámetro, la Image se mostrará sólo si el botón de radio utiliza un modo de visualización "estilo Vista" (con Image).




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las características de la opción añadida también pueden modificarse mediante las siguientes propiedades utilizadas en la opción: 

	- [Pie de foto](../Proprietes/2510053.md) para modificar el título. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) [SubCaption](../Proprietes/1000017232.md) modificar el subtítulo. 

	- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) [Image](../Proprietes/2510034.md) para modificar el Image. 




- Cuando se añaden opciones a un control Botón de opción, el tamaño del control no se modifica. Las opciones añadidas pueden no ser Visible si el control no es lo suficientemente grande. La control puede ampliarse mediante las propiedades [anchura](../Proprietes/2510059.md) y [Altura](../Proprietes/2510050.md).

- Para añadir una opción en una posición específica, utilice [RadioButtonInsert](../WDLang1/1000020599.md).  

- **Utilización de valores devueltos**: Si el botón de radio utiliza los valores devueltos, tiene la posibilidad de definir el valor devuelto por la opción añadida mediante [ReturnedValue](../Proprietes/2510134.md). 
	Por ejemplo: 
	
	```wl
	nIndex is int
	nIndex = RadioButtonAdd(RADIO_Option, "Read a book on a tablet")
	RADIO_Option[nIndex].ReturnedValue = "Tablet"
	// Select the new option
	RADIO_Option = "Tablet"
	```

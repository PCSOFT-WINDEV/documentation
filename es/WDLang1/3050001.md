
## MatTranspose( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula la matriz transpuesta.

La matriz transpuesta se obtiene cambiando las filas en columnas. si la matriz fuente es una matriz que contiene **n** filas y **m** la matriz transpuesta es una matriz que contiene **m** filas y **n** columnas.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Transposed matrix
ResTranspose = MatTranspose("MyMatrix", "MyTransposedMatrix")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MatTranspose(<Matrix to transpose> , <Transposed matrix>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha calculado la matriz transpuesta,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<Matrix to transpose>: (Cadena de caracteres)`**

Nombre de la matriz a transponer, definido por [MatCreate](../WDLang1/3050058.md).

**`<Transposed matrix>: (Cadena de caracteres)`**

Nombre de la matriz transpuesta, creada automáticamente al calcular la matriz transpuesta..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Existencia de la matriz transpuesta
<a name="existencia_matriz_transpuesta_ELTPARAGRAPHE000091"></a>

No se verifica la existencia de la matriz.. Esta matriz se sobrescribe si existía de antemano.. Para comprobar la existencia de la matriz, utilice [MatExiste](../WDLang1/3050076.md).
<a name="NOTE0_2"></a>




### Matrices idénticas
<a name="matrices_identicas_ELTPARAGRAPHE000102"></a>

Se produce un error de WLanguage si &lt;Matriz a transponer&gt; y &lt;Matriz transpuesta&gt; son idénticas.

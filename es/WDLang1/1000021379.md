<a name="Use"></a>
<a name="description"></a>
Ejecuta inmediatamente el evento "Solicitud para actualizar la visualización" de la ventana, página, ventana interna, página interna, modelo de controles o supercontrol.

Si se llama a esta función varias veces durante un proceso, el evento de actualización se ejecutará varias veces.

**Observación**: Esta función se utiliza principalmente: 

- cuando se actualiza la interfaz mediante hilos. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) cuando se utiliza la programación MVP (Modelo - Vista - Presentador). Para obtener más información, consulte [Presentación MVP: Modelo - Vista - Presentador](../Editeurs/1000021496.md). 







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ExecuteRefreshUI()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`ExecuteRefreshUI([<Parameter 1> [... [, <Parameter N>]]])`
---

**`<Parameter 1>: (Tipo de valor esperado por el evento "Solicitud para actualizar la visualización")`**

Primer parámetro que se pasará al evento "Solicitud para actualizar la visualización" de la ventana o página. 



**`<Parameter N>: (Tipo de valor esperado por el evento "Solicitud para actualizar la visualización")`**

Enésimo parámetro que se pasará al evento "Solicitud para actualizar la visualización" de la ventana o página.

## Observaciones


### Evento "Solicitud para actualizar la visualización"
<a name="evento_solicitud_para_actualizar_visualizacion_ELTPARAGRAPHE000075"></a>

"Solicitud para actualizar la visualización" es uno de los eventos asociados a la ventana o página. Este evento permite agrupar el código de actualización de los controles de la ventana o página: contenido de los controles, estado (visible, en gris, etc.), título a actualizar, etc.

Este evento puede recibir parámetros. Solo se debe declarar un procedimiento en el evento. Por ejemplo: 


```wl
// Request for refreshing the display
PROCEDURE AskForRefresh(RefreshType is Boolean)
```


Este evento se ejecuta mediante solicitud previa con las siguientes funciones: 

- [RequestUpdateUI](../WDLang1/1000021378.md): Ejecuta el evento "Solicitud para actualizar la visualización" al final del proceso en curso (ejecución asíncrona). 

- **ExecuteUpdateUI**: Ejecuta inmediatamente el evento "Solicitud para actualizar la visualización". 

- [RequestUpdateUIParent](../WDLang1/1000022961.md): Ejecuta el evento "Solicitud para actualizar la visualización" del nivel superior al final del proceso en curso (ejecución asíncrona).

- [ExecuteRefreshUIParent](../WDLang1/1000022961.md): Ejecuta inmediatamente el evento "Solicitud para actualizar la visualización" del nivel superior.


Al llamar a estas funciones, se puede especificar el valor de los parámetros esperados por el evento "Solicitud para actualizar la visualización". 

Observación: El evento "Solicitud para actualizar la visualización" también puede ejecutarse inmediatamente con la función [ExecuteProcess](../WDLang1/3013031.md) utilizando la siguiente sintaxis: 


```txt
ExecuteProcess(<Element name>, trtRequestRefresh)
```

### MVP: Atributo "Presentador"
<a name="mvp_atributo_presentador_ELTPARAGRAPHE000114"></a>

El atributo **presentador** permite asociar una clase de la capa del presentador a una vista (ventana, página o reporte). Este atributo se utiliza principalmente en la declaración global de las ventanas generadas por el RAD MVP.

Por ejemplo: 


```wl
PROCEDURE WIN_Table_MyExampleFile(...
	gclPresenter is PTableMyExampleFile dynamic <presenter>=Null)
```


Con este atributo, se activará la llamada al evento "Solicitud para actualizar la visualización" de la ventana (o página) al realizar:

- una llamada a la función [RequestUpdateUI](../WDLang1/1000021378.md) desde una clase declarada como &lt;presentador&gt;, 

- una llamada desde cualquier clase asociada a la clase &lt;presentador&gt;. 




Para obtener más información, consulte [RAD MVP: Atributo "Presentador"](../Editeurs/1000021508.md).

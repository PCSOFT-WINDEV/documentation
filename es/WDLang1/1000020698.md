


## Colapso del PVTC (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Oculta los datos encontrados en una control Tabla dinámica según una cabecera fila o una cabecera de columna.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Collapses all quarters
PVTCollapse(PVT_Sales, COL_OrderDate_Quarter)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PVTCollapse(<Pivot Table control> , <Header to collapse>)`
---

**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<Header to collapse>: (Cadena de caracteres)`**

Nombre de la cabecera de fila o de la cabecera de columna a ocultar.

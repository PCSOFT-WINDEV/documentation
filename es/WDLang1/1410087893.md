
## sysNotification (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **sysNotification** se utiliza para definir todas las características avanzadas de un sistema interactivo Notification. Las características de esta Notification pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Example: Create and send an update notification
notifUPD is sysNotification
notifUPD.Title = "Update available!"
notifUPD.Text1 = "Download now to get the latest features."
notifUPD.Note = "UPD_notif"
notifUPD.Buttons[1].Caption = "Download"
notifUPD.Buttons[1].Note = "UPD_download_button"
notifUPD.Buttons[2].Caption = "Later..."
notifUPD.Buttons[2].Type = sysNotifReminderButton
SysNotificationSend(notifUPD)
```

## Observaciones
<a name="NOTE0_1"></a>


### Ejemplo de Notification
<a name="ejemplo_notification_ELTPARAGRAPHE000028"></a>

Este es un ejemplo de Notification: <br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=NOTIF_FR.gif)


- 1: Aplicación Icon y nombre: Esta información no puede ser modificada. 

- 2: Imagen de cabecera de la notificación (propiedad **HeaderImagePath** del **sysNotification** Variable). 

- 3: Logotipo Image: Logotipo de la aplicación Default. Este logotipo puede ser sustituido por una imagen (**LogoImagePath** propiedad del **sysNotification** Variable). También puede recortar esta imagen en un círculo (propiedad **LogoImageCrop** del **sysNotification** Variable). 

- 4: Título: Título o nombre de la notificación (2 líneas como máximo, **Título** propiedad del **sysNotification** Variable).

- 5: Texto adicional: 2 áreas de texto adicionales, que no ocupan más de un total de 4 líneas (**Texto1** y **Texto2** propiedades del **sysNotification** Variable). 

- 6: Fecha y hora de envío de la notificación (propiedad **Date** del **sysNotification** Variable).

- 7: Atribución, que indica la fuente de la información: SMS, ...  (**Atribución** propiedad del **sysNotification** Variable). 

- 8: Imagen contenida en la Notification. Esta imagen opcional se muestra cuando se expande la notificación (propiedad **ContentImagePath** del **sysNotification** Variable).

- 9: Control de edición donde el usuario puede introducir texto (definido por el [sysNotificationInput](../WDLang1/1410088024.md) Variable). 

- 10: Combo Box donde el usuario puede seleccionar una opción (definida por el [sysNotificationInput](../WDLang1/1410088024.md) Variable).

- 11: Botón simple, que puede realizar una acción definida por el sistema o enviar información a la aplicación que llama (definida por el [sysNotificationButton](../WDLang1/1410087947.md) Variable). También puede definir un botón contextual, que se añade al menú contextual de la Notification (clic derecho).




Observación: los botones sencillos aparecen siempre en la parte inferior de Notification. Sin embargo, los controles de Edición y Combo Box aparecen en el orden en que han sido definidos.

### Propiedades específicas de la descripción de las variables de sysNotification
<a name="propiedades_especificas_descripcion_las_variables_sysnotification_ELTPARAGRAPHE000059"></a>

Las siguientes propiedades pueden utilizarse para manipular un sistema Notification:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Acción | Global Procedure solamente | Nombre del procedimiento global que se ejecutará cuando el usuario haga clic en la propia Notification (no en un control dentro de la Notification).<br><br>Si no se especificó el procedimiento o no se puede encontrar, se ejecuta el evento opcional del proyecto "Recepción de una notificación". |
| Arrancadores | Array de [sysNotificationInput](../WDLang1/1410088024.md) | Controles de "entrada del usuario" de la Notification. Pueden ser controles de "entrada de texto" o "cuadro combinado".. <br><br>Los controles aparecen de arriba a abajo en el orden en que fueron añadidos al array.<br><br>Atención : No puede haber más de 4 controles de entrada en un Notification. Los controles adicionales sustituirán a los anteriores. |
| Atribución | Cadena de caracteres | Texto que debe aparecer en la pequeña zona de texto situada debajo de la Notification (por ejemplo, esta casilla puede utilizarse para indicar la fuente de la información). <br><br>Esta Property puede estar vacía. |
| Botones | Array de [sysNotificationButton](../WDLang1/1410087947.md) | Controles de los botones de la Notification. Los botones aparecen siempre en la parte inferior del Notification, de izquierda a derecha siguiendo su orden en el array (excepto los botones contextuales, que se añaden al menú contextual del Notification, de arriba a abajo).<br><br>Atención: No puede haber más de 5 botones en una Notification. |
| ContentImagePath | Cadena de caracteres | ruta absoluta de una imagen en disco que puede añadirse al contenido de la Notification. <br><br>Esta Property puede estar vacía. |
| Escenario | Constante de tipo Integer | Comportamiento específico de la Notification : <br><br>- *sysNotifAlarm*: La Notification permanece en pantalla hasta que el usuario interactúa con ella. Una alerta sonora "Reminder" definida por el sistema se reproduce en bucle.<br><br>- *sysNotifCall*: La Notification permanece en pantalla hasta que el usuario interactúa con ella. Se reproduce en bucle un sonido de llamada entrante definido por el sistema y se utiliza un estilo diferente para los botones.<br><br>- *sysNotifStandard* (valor Default): La Notification aparece en pantalla durante unos segundos antes de entrar en el Centro de Acción de Windows.<br><br>- *sysNotifReminder*: La Notification permanece en pantalla hasta que el usuario interactúa con ella. Cuando aparece la Notification, se reproduce una alerta sonora "recordatoria" definida por el sistema.<br><br><br> |
| HeaderImagePath | Cadena de caracteres | ruta absoluta de una imagen en disco que puede utilizarse como cabecera de la Notification. <br><br>El tamaño de la Image mostrada es de 364 x 180 píxeles (la Image se reduce automáticamente si es necesario). |
| ID | Cadena de caracteres | Identificador único asociado a la Notification. Dos notificaciones de la misma aplicación no pueden tener el mismo identificador. Si dos Notification s tienen el mismo identificador, el nuevo Notification sustituirá al anterior. <br><br>Esta Property puede estar vacía. |
| LogoImageCrop | Constante de tipo Integer | Recorte que se aplicará al logotipo: <br><br>- *sysNotifNoCrop* (valor Default): Sin recorte, cuadrado Image. <br><br>- *sysNotifCircleCrop*: Image con un círculo.<br><br><br> |
| LogoImagePath | Cadena de caracteres | ruta absoluta de una imagen en disco que puede utilizarse como logotipo de la Notification. <br><br>El tamaño de la Image mostrada es de 48 x 48 píxeles en el Notification (el Image se reduce automáticamente si es necesario). Puede recortar el logotipo utilizando la función **LogoImageCrop** Property.<br><br>Esta Property puede estar vacía. |
| Nota | Cadena de caracteres | Cadena que se devolverá a la aplicación cuando el usuario haga clic en la Notification (no es un botón). Puede utilizarse para identificar la Notification pulsada, por ejemplo. <br><br>Esta Property puede estar vacía. |
| salir con | FechaHora | Fecha que se mostrará en la Notification. <br><br>Esta Property puede estar vacía. |
| Texto1 | Cadena de caracteres | Texto opcional que aparece debajo del título. El texto combinado de las propiedades **Text1** y **Text2** no puede ocupar más de 4 líneas. |
| Texto2 | Cadena de caracteres | Texto opcional que aparece debajo del texto 1. <br><br>Si el **Texto1** Property está vacío, se mostrará el **Texto2** Property en su lugar. <br><br>El texto combinado de las propiedades **Text1** y **Text2** no puede ocupar más de 4 líneas. |
| Título | Cadena de caracteres | El título de Notification aparece en 2 líneas como máximo. |
| XML | Cadena de caracteres | Código XML que corresponde a la configuración actual del Notification.<br><br>En modo de escritura : Esta propiedad le permite especificar una descripción XML del Notification. Si se especifica esta propiedad, se ignorarán todas las demás propiedades del tipo **sysNotification**. <br><br>En modo de lectura: Esta propiedad se utiliza para obtener el código XML generado por WINDEV correspondiente a la Notification. <br><br>Para más información sobre el formato XML de las notificaciones de Windows, consulte : [https://docs.microsoft.com/en-us/windows/uwp/design/shell/tiles-and-notifications/adaptive-interactive-toasts?tabs=xml](https://docs.microsoft.com/en-us/windows/uwp/design/shell/tiles-and-notifications/adaptive-interactive-toasts?tabs=xml) |

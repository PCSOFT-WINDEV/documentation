
## &lt;Tipo String&gt;.Format (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Define el formato de una cadena de caracteres según las opciones seleccionadas.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyString1 is string = "Developing is fantastic"
MyString1 = MyString1.Format(ccUpCase)
// MyString1 contains "DEVELOPING IS FANTASTIC"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <String>.Formats(<Options>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena con el formato especificado.

**`<String>: (Cadena de caracteres)`**

Cadena de caracteres cuyo formato se definirá.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Se pueden utilizar cadenas Unicode.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) No se pueden utilizar cadenas Unicode.

**`<Options>: (Constante o combinación de constantes)`**

Opciones de formato:               


|   |   |
| --- | --- |
| *ccIgnoreAccent* | Formato sin caracteres acentuados. |
| *ccIgnoreInsideSpace* | Elimina los siguientes caracteres dentro de la cadena:<br><br>- espacio (carácter 32)<br><br>- tabulador (carácter 9)<br><br>- retorno de carro (carácter 13) y salto de línea (carácter 10)<br><br>- caracteres de control 11 y 12<br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *ccIgnorePunctuationAndSpace* | Elimina los espacios y los caracteres de puntuación. |
| *ccIgnoreSpace* | Elimina los siguientes caracteres al inicio y al final de la cadena:<br><br>- espacio (carácter 32)<br><br>- tabulador (carácter 9)<br><br>- retorno de carro (carácter 13) y salto de línea (carácter 10)<br><br>- caracteres de control 11 y 12<br><br><br> |
| *ccLowCase* | Convierte la cadena en caracteres en minúsculas. |
| *ccNormal* | No se define ningún formato. |
| *ccUpCase* | Convierte la cadena en caracteres en mayúsculas (incluyendo caracteres con acentos). |

## Observaciones


### Función &lt;Tipo String&gt;.Format y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000264"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **&lt;Tipo String&gt;.Format** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

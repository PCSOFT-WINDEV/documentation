


## dHorizontalSymmetry (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Voltea la imagen con respecto a un eje horizontal (simetría con respecto a un eje horizontal). Esta imagen puede corresponder a: 

- un control Imagen, 

- una variable de tipo [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [WDPic](../WDLang1/1000023503.md) (en la capa de fondo),

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) una variable de tipo [picLayer](../WDLang1/1000024604.md).

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dSymetrieHorizontale.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Symmetry of IMG_IMAGE1 with respect to a horizontal axis
MyResult is boolean
MyResult = dHorizontalSymmetry(IMG_IMAGE1)
IF MyResult = False THEN Error("The operation failed")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = dHorizontalSymmetry(<Image>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se volteó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image>: (Nombre de control o variable de tipo Image, WDPic o picLayer)`**

Imagen a manipular. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [WDPic](../WDLang1/1000023503.md). Solo se manipulará la capa de fondo. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [picLayer](../WDLang1/1000024604.md).




La simetría se realiza con respecto al eje horizontal central del control o de la imagen. La imagen se modifica directamente.

## Observaciones


### Fin del dibujo
<a name="fin_del_dibujo_ELTPARAGRAPHE000217"></a>

Si se llama a las funciones [dEndDrawing](../WDLang1/3029019.md) y **dHorizontalSymmetry** en el mismo proceso, no se mostrarán los cambios de la imagen.

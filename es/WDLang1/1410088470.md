
## diagHead (Tipo de Variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **diagHead** se utiliza para definir todas las características avanzadas de un extremo de un conector en un Diagram. Las características de este extremo pueden definirse y modificarse mediante diferentes propiedades de WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Create a link between rectangle1 and rectangle2
rectangle1 is diagRectangle
rectangle2 is diagRectangle
Connector is diagConnector

Connector.Beginning.ConnectionPoint = rectangle1.ConnectionPoint(100, 50) 
Connector.Beginning.Drawing = shapeNone
Connector.Ending.ConnectionPoint = rectangle2.ConnectionPoint(0, 50) 
Connector.Beginning.Drawing = shapeTriangle
```





<a name="PROPS"></a>

## Properties


### Propiedades específicas de las variables diagEnd
<a name="propiedades_especificas_las_variables_diagend_ELTPARAGRAPHE000026"></a>

Las siguientes propiedades se pueden utilizar para manipular el final de un conector:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ConnectionPoint | [diagConnectionPoint](../WDLang1/1410088457.md) | Punto en el que el conector se encaja. |
| Dibujo | Constante de tipo Integer | Dibujo de la flecha: <br><br>- *shapeNone*: El final no tiene ninguna forma particular. <br><br>- *shapeSquare*: El final es un cuadrado. <br><br>- *shapeRoundedSquare*: El extremo es un cuadrado redondeado. <br><br>- *shapeArrow*: El final es una flecha. <br><br>- *shapeDiamond*: El final es un diamante. <br><br>- *shapeCircle*: El final es un círculo. <br><br>- *shapeTriangle*: El final es un triángulo. <br><br><br> |
| SnapMode | Constante de tipo Integer | Cómo se ajustará el conector a la forma de destino:  <br><br>- *connectorSnapToAuto*: El punto de conexión se determina automáticamente según la posición del conector y la forma. <br><br>- *connectorSnapToShape*: El extremo del conector se ajustará al centro de la forma. <br><br>- *connectorSnapToConnectionPoint*: El conector se ajustará a un punto de conexión. <br><br>- *connectorSnapToBoundingBox*: El extremo del conector se ajustará a un punto del cuadro delimitador de la forma. <br><br><br> |
| X | Real | Coordenada X del extremo en la Diagram. |
| Y | Real | Coordenada Y del extremo en la Diagram. |

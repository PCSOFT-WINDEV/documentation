
## TimeToInteger (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una hora en un número entero. Este número entero corresponde al número de centésimas de segundo transcurridas entre el parámetro &lt;Hora&gt; y 00 horas 00 minutos 00 segundos y 00 centésimas de segundo.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Wait 4 seconds
Start, t are int
Start = TimeToInteger(TimeSys())
LOOP
	t = TimeToInteger(TimeSys()) - Start
	// Display the time elapsed between 0 and 4s
	Trace(IntegerToTime(t))
	// Exit if the 4 seconds have elapsed
	IF t > 400 THEN BREAK
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TimeToInteger(<Time>)`
---

**`<Result>: (Entero)`**



- Número de centésimas de segundo transcurridas entre &lt;Hora&gt; y 00 horas 00 minutos 00 segundos y 00 centésimas de segundo,

- 0 si se produce un error.




**`<Time>: (Cadena de caracteres, variable de tipo Date o DateTime)`**

Hora a convertir. Esta hora puede corresponder a: 

- una cadena de caracteres en formato HHMMSSCC (HH corresponde a horas, MM a minutos, SS a segundos y CC a centésimas de segundo). 

- una cadena de caracteres en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo). En este caso, solo se tiene en cuenta la hora. 

- una variable de tipo Time. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la hora.


La función devuelve 0 si este parámetro corresponde a una cadena vacía ("").  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La validez de la hora pasada como parámetro se verifica. Si la hora no es válida, se muestra un mensaje de error

- Las funciones **TimeToInteger** y [TimeDifference](../WDLang1/3027018.md) permiten calcular el número de centésimas de segundo transcurridas entre dos fechas. 

- La medianoche corresponde a 00 horas, 00 minutos, 00 segundos y 00 centésimas de segundo.

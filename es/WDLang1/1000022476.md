


## &lt;Cámara&gt;.estado (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el estado de una control Cámara.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the status of "CAM_Kitchen" control
// The explanation is displayed in the status bar. 
ResStatus = CAM_Kitchen.Status()
SWITCH ResStatus
	CASE viStatusDisplayCapture:
		Message("The video is displayed in the Camera control. " + ...
			"Backup of this video in progress")
	CASE viStatusDisplayNoCapture:
		Message("The video is displayed in the Camera control. " + ...
			"No backup is currently performed")
	CASE viStatusCaptureNoDisplay:
		Message("A video is currently saved. "+ ...
			"This video is not displayed in the Camera control")
	CASE viStatusNoCaptureNoDisplay:
		Message("No video is displayed and/or currently " + ...
			"saved in the Camera control")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <Camera control>.Report()`
---

**`<Result>: (Constante de tipo Integer)`**

Estado de la control Cámara especificada:


|   |   |
| --- | --- |
| *viStatusCaptureNoDisplay* | Un vídeo está actualmente guardado. Este vídeo no se muestra en la control Cámara especificada. |
| *viStatusDisplayCapture* | El vídeo se muestra en el formato especificado control Cámara. Este vídeo está actualmente guardado. |
| *viStatusDisplayNoCapture* | El vídeo se muestra en el formato especificado control Cámara. Actualmente no se realiza ninguna copia de seguridad. |
| *viStatusNoCaptureNoDisplay* | No se muestra ningún vídeo y/o no se guarda actualmente en la control Cámara especificada. |



**`<Camera control>: (Nombre del control)`**

Nombre del control Cámara a utilizar.

## Observaciones

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png) Para utilizar esta función, se debe conectar una "cámara web" al ordenador actual.

- Esta función es equivalente a la función [CameraStatus](../WDLang1/1000022455.md). 



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000245"></a>

- Para guardar el vídeo transmitido por una "cámara web", utilice [&lt;Cámara&gt;Captura](../WDLang1/1000022382.md)

- Para visualizar la secuencia de vídeo transmitida por una "cámara web" en un control Cámara, utilice [&lt;Cámara&gt;.display](../WDLang1/1000022380.md).

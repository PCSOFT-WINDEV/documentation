


## WinEdSelectOccurrence (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica el número de controles seleccionados cuando la interfaz es modificada por el usuario final (Modo "Modificación de UI por el usuario final" activado). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enlarge all the controls by 10 pixels
FOR i = 1 _TO_ WInEdSelectOccurrence(WIN_Form)
	X is Control <- WinEdSelect(WIN_Form)
	X..Width += 10
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = WinEdSelectOccurrence(<Window>)`
---

**`<Result>: (Integro)`**



- Número de controles seleccionados por el usuario final (Modo "Modificación de UI por el usuario final")

- 0 si no se ha seleccionado ninguna control o si la selección múltiple no está activada. 




**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. Esta ventana debe estar en modo de modificación ([WinEdActive](../WDLang1/1000021843.md)). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Habilitación del modo "multiselección" para el usuario final
<a name="habilitacion_del_modo_multiseleccion_para_usuario_final_ELTPARAGRAPHE000037"></a>

Se puede activar el modo "multiselección" para el usuario final: 

- en el editor: 

	- a través de las opciones de configuración específicas del proyecto: 

		- en la pestaña "Proyecto", en el grupo "Proyecto", haga clic en "Descripción". 

		- en la pestaña "Avanzado", haga clic en el botón "Opciones" para definir las opciones autorizadas. 




- a través de las opciones de configuración específicas de la ventana: 

	- Abre la ventana description ("description" del menú de la ventana).  

	- En la pestaña "Detalles", seleccione la opción "Permitir que el usuario final modifique la UI". 

	- El botón "Opciones" sirve para especificar las opciones autorizadas. 

- mediante programación con la propiedad [WinEditMode](../Proprietes/1000021824.md).

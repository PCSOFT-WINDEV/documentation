


## KanbanDeleteAllCard (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina todas las tarjetas de un control Kanban o de un control Lista Kanban.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all cards from the Kanban control
KanbanDeleteAllCard(KANBAN_MyKanban)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`KanbanDeleteAllCard(<Control used>)`
---

**`<Control used>: (Nombre del control)`**

Nombre del control utilizado: 

- Nombre del control Kanban.

- Nombre del control Lista Kanban.




## &lt;Editor HTML&gt;.InsertTable (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una tabla en la posición del cursor en un editor de HTML control.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert a 4 x 3 table
HTMEDT_MyHTMLEditor.InsertTable(4, 3, "border=2")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<HTML Editor control>.InsertTable(<Number of columns> , <Number of rows> [, <CSS style>])`
---

**`<HTML Editor control>: (Nombre del control)`**

Nombre del editor de HTML utilizado por control.

**`<Number of columns>: (Integro)`**

Número de columnas a insertar.

**`<Number of rows>: (Integro)`**

Número de filas a insertar.

**`<CSS style>: (Cadena de caracteres opcional)`**

Nombre CSS del estilo a aplicar a la tabla insertada. Este parámetro también puede corresponder a un estilo incrustado en una etiqueta &lt;tabla&gt; (por ejemplo.: "Border=1").

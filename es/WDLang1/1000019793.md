


## &lt;Seleccione Grupo (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona visualmente un grupo de elementos en un control TreeMap. Un grupo de elementos se identifica por su ruta. La ruta se especificó cuando el elemento fue añadido a la control TreeMap por [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The TMAP_Wine control is filled from a Table control. 
// Select the group corresponding to the type of wine selected in the Table control
SelectionDirectory is string
SelectionDirectory = COL_WineType[TABLE_WINE]
TMAP_Wine.SelectGroup(SelectionDirectory)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<TreeMap control>.SelectGroup(<Group path>)`
---

**`<TreeMap control>: (Nombre del control)`**

Nombre de control TreeMap a utilizar.

**`<Group path>: (Cadena de caracteres)`**

Ruta del grupo a seleccionar. El separador de los diferentes elementos de ruta puede ser:

- el carácter "\\" ".. Por ejemplo: "MyDirectory\\MyFile". 

- el carácter "TAB. Por ejemplo: "Postres"+TAB+"Helado". 




Esta ruta se ha especificado al añadir el elemento en la control TreeMap con [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md).

Si este parámetro corresponde a una cadena vacía ("""), no se selecciona ningún grupo..



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si un grupo ya estaba seleccionado con **&lt;Seleccione Grupo**, se deselecciona automáticamente. 

- El estilo del grupo seleccionado se define en la pestaña "Estilo" de la ventana de description del control TreeMap.

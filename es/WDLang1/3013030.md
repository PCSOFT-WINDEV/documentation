
## InitRandom( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicializa el generador de números aleatorios.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize the generator of random numbers
InitRandom()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`InitRandom(<n>)`
---

**`<n>: (Integro)`**

Número base para el generador de números aleatorios. 
Si no se especifica este parámetro, el generador de números aleatorios se basa en la hora del sistema (centésimas de segundo)..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si no se llama a la función **InitRandom**, los valores aleatorios generados por la función [Al azar](../WDLang1/3013037.md) serán los mismos cada vez que se ejecute el programa.

- Si se llama sucesivamente a **InitRandom**, se devolverán secuencias idénticas porque **InitRandom** utiliza el tiempo del sistema para generar secuencias de números aleatorios.

- El uso del parámetro &lt;n&gt; no garantiza necesariamente a get diferentes secuencias.

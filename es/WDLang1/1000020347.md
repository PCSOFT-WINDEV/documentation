
## OpenMainMenu( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre el menú principal de la ventana actual. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
OpenMainMenu()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`OpenMainMenu()`
---



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función no tiene efecto si no hay ninguna ventana actual durante la llamada a la función o si la ventana actual no contiene ningún menú principal.

- A partir de la versión 3.0 de Android, el botón de menú no aparecerá sistemáticamente en la barra de navegación del dispositivo.
	Para visualizar el menú principal de la ventana, le aconsejamos que lo haga:

	- utilizar una [Action Bar](../WDChamp/1000025001.md) en la ventana: en este caso, el menú se puede abrir desde el Action Bar.

	- llamar a la función **OpenMainMenu** desde un botón de la ventana (por ejemplo).




- Esta función se puede utilizar en el simulador.

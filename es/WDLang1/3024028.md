
## Reverse (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el carácter que corresponde a la diferencia entre el código ASCII de un carácter específico en una cadena y 255. La cadena se cifra.

Para una cadena Unicode (o en WINDEV Mobile), la diferencia se calcula con respecto a 65535.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
EDT_Edit2 = Reverse(EDT_Edit1)
// If EDT_Edit1 contains TEST, EDT_Edit2 will contain "º¬"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Reverse(<Initial string>)`
---

**`<Result>: (Cadena de caracteres)`**

Contiene la cadena convertida.

**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres a cifrar. Esta cadena no se modifica.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta cadena puede ser una cadena Unicode.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Las cadenas Unicode no están soportadas.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Función Reverse y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000076"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **Reverse** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

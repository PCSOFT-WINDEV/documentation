


## TimelineDeleteAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todos los eventos y pistas de un control Línea de tiempo.

Observaciones: 

- En el caso de una control Línea de tiempo rellenada programáticamente, se borran todos los eventos y pistas. 

- En el caso de un control Línea de tiempo basado en un archivo de datos o en un Variable, los eventos y las pistas sólo se borran del control. Los eventos y las pistas se mantienen en la fuente de datos relacionado a la control. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete all events and tracks from the TimeLine control populated programmatically
TimeLineDeleteAll(TL_TimeLine)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TimelineDeleteAll(<TimeLine control>)`
---

**`<TimeLine control>: (Nombre del control)`**

Nombre de la control Línea de tiempo que se va a utilizar.

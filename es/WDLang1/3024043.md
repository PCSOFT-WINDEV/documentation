
## UTF8ToString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una cadena UTF-8 a ANSI o UNICODE.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Convert a UTF-8 string to ANSI
ConversionResult = UTF8ToString(UTF8String, charsetAnsi)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = UTF8ToString(<UTF-8 string to convert> [, <Destination charset>])`
---

**`<Result>: (Cadena de caracteres)`**



- En ANSI: cadena de caracteres ANSI. 

	- Si no se especifica el conjunto de caracteres de destino, la cadena se convierte a ANSI utilizando el conjunto de caracteres actual. 

	- Si se especifica el conjunto de caracteres de destino, la cadena se convierte a ANSI utilizando el conjunto de caracteres especificado.




- En Unicode: cadena de caracteres UNICODE con el resultado de la conversión. El conjunto de caracteres de destino se ignora.

- EOT si se producen problemas de conversión.




**`<UTF-8 string to convert>: (Cadena de caracteres)`**

Cadena a convertir.

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta cadena es una cadena ANSI en formato UTF-8.



**`<Destination charset>: (Entero opcional)`**

Conjunto de caracteres que se usará para crear la cadena. De forma predeterminada, este parámetro corresponde al conjunto de caracteres actual (especificado con la función [ChangeCharset](../WDLang1/3054001.md)).


|   |   |
| --- | --- |
| *charsetAnsi* | Caracteres romanos en estándar ANSI |
| *charsetArabic* | Caracteres arábigos |
| *charsetBaltic* | Caracteres bálticos |
| *charsetChinese* | Caracteres chinos (República Popular China) |
| *charsetCurrent*<br>(Valor predeterminado) | Utiliza el conjunto de caracteres actual especificado con la función [ChangeCharset](../WDLang1/3054001.md) (*charsetOccidental* predeterminado) |
| *charsetDefault* | Utiliza el conjunto de caracteres predeterminado del equipo. No se fuerza ningún conjunto de caracteres.<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Solo está disponible este valor. |
| *charsetEastEurope* | Caracteres de Europa del Este (polaco, etc.) |
| *charsetGreek* | Caracteres griegos |
| *charsetHebrew* | Caracteres hebreos |
| *charsetJapanese* | Caracteres japoneses |
| *charsetKorean* | Caracteres coreanos |
| *charsetRussian* | Caracteres rusos |
| *charsetThai* | Caracteres tailandeses |
| *charsetTraditionalChinese* | Caracteres chino tradicional (República de Taiwán) |
| *charsetTurkish* | Caracteres turcos |
| *charsetUTF8* | Permite gestionar el formato UTF-8 (utilizado en XML, por ejemplo). |
| *charsetVietnamese* | Caracteres vietnamitas |





![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no está disponible.




## &lt;Tabla TreeView Table&gt;.InsertChild (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una fila en un control Tabla TreeView, en un nivel jerárquico determinado.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert a leaf ("Pancakes") at position 2 below the "Desserts" node
TVT_Recipe.InsertChild("Desserts", 2, "Pancakes")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar una fila especificando el índice del elemento padre

`<Result> = <TreeView Table control>.InsertChildren(<Index of the parent element> , <Position> [, <Element column 1> [... [, <Element column N>]]])`
---

**`<Result>: (Integro)`**

Índice de inserción fila.

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Index of the parent element>: (Integro)`**

Índice del fila padre en la jerarquía o *NULL* constante para insertar el fila en la raíz del control Tabla TreeView (en este caso, esta función es equivalente a [&lt;Tabla&gt;.InsertLine](../WDLang1/1000023407.md)).

**`<Position>: (Integro)`**

Posición en la que la fila se insertará en relación con su padre.

**`<Element column 1>: (Tipo de columna associated (opcional))`**

Elementos de la fila para insertar en la control Tabla TreeView especificada. Cada elemento corresponde a una columna de control Tabla TreeView. El tipo de elemento insertado debe ser compatible con el tipo de columna relevante..

Si no se especifica este parámetro, se inserta un fila vacío en el control Tabla TreeView.



**`<Element column N>: (Tipo de columna associated (opcional))`**

Elementos de la fila para insertar en la control Tabla TreeView especificada. Cada elemento corresponde a una columna de control Tabla TreeView. El tipo de elemento insertado debe ser compatible con el tipo de columna relevante..

Si no se especifica este parámetro, se inserta un fila vacío en el control Tabla TreeView.


<a name="SYNTAX2"></a>

### Insertar una fila especificando el elemento padre

`<Result> = <TreeView Table control>.InsertChildren(<Parent element> , <Position> [, <Element column 1> [... [, <Element column N>]]])`
---

**`<Result>: (Integro)`**

Índice de inserción fila

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.

**`<Parent element>: (Cadena de caracteres)`**

Contenido de la primera columna del elemento padre en la control Tabla TreeView.

**`<Position>: (Integro)`**

Posición en la que la fila se insertará en relación con su padre.

**`<Element column 1>: (Tipo de columna associated (opcional))`**

Elementos de la fila para insertar en la control Tabla TreeView especificada. Cada elemento corresponde a una columna en el control Tabla TreeView. El tipo de elemento insertado debe ser compatible con el tipo de columna relevante..

Si no se especifica este parámetro, se inserta un fila vacío en el control Tabla TreeView.



**`<Element column N>: (Tipo de columna associated (opcional))`**

Elementos de la fila para insertar en la control Tabla TreeView especificada. Cada elemento corresponde a una columna en el control Tabla TreeView. El tipo de elemento insertado debe ser compatible con el tipo de columna relevante..

Si no se especifica este parámetro, se inserta un fila vacío en el control Tabla TreeView.

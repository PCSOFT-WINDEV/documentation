


## InAWPMode (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la página que se está ejecutando es una página AWP. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Code run from an Active WEBDEV Page? 
IF InAWPMode() = True THEN
	// AWP mode
	// Processes to perform
	...
ELSE
	// Non-AWP mode
	// Processes to perform
	...
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = InAWPMode()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la página ejecutada es una página AWP,

- <u><u><u><u>False</u></u></u></u> en los demás casos.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Disponibilidad de la función
<a name="disponibilidad_funcion_ELTPARAGRAPHE000207"></a>

La función **InAWPMode** se puede utilizar en todos los IDE para escribir código multiplataforma. Las funciones o procedimientos que utilizan la función **InAWPMode** pueden compartirse entre varios productos sin generar errores de compilación.


## ControlPrevious( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Especifica el nombre de control que estaba en edición antes del actual control.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// EDT_CITY entry code
// if coming from the "EDT_ZIP" control
IF ControlPrevious() = EDT_ZIP THEN
	// Fill the name according to ZC
	EDT_CITY = ZIPToCity(EDT_ZIP)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ControlPrevious()`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre de control que estaba previamente en edición (en la ventana actual). 

- Cadena vacía ("") si el actual control es la primera ventana control.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- [ControlCurrent](../WDLang1/3025019.md) se utiliza para averiguar el nombre de control que se está editando.

- El nombre de la control es siempre relativo a la ventana actual.

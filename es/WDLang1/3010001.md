
## Constantes de Notas adicionales
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes pueden ser utilizadas en los procesos sin ser declaradas. Tienen un valor preestablecido.



| Constante | Uso | Valor |
| --- | --- | --- |
| **CERO** | Entero inicializado a 0. Por ejemplo, se puede utilizar un parámetro "null" en una función API ([API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md)) | 0 |
| **CR** | "Carácter"Carriage Return: ir a la siguiente Line (usada en las cadenas de caracteres) | Charact(13) + Charact(10) |
| **CRLF** | Ir a la siguiente Line<br><br> | Charact(13) + Charact(10) |
| **EOT** | Carácter que se encuentra al final de los archivos externos | Charact(4) |
| **ESC** | Carácter correspondiente a la tecla "Esc".<br><br> | Charact(27) |
| **ESP** | Espacio no separable.<br><br> | Charact(160) |
| **False** | booleano | 0 |
| **TAB** | Carácter"Tabulación" (utilizado en las cadenas de caracteres) | Charact(9) |
| **True** | booleano | 1 |
| **ValPI** | Valor aproximado de Pi.<br><br> | 3,14159265358979323 |


## COMQueryInterface (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Llama al método QueryInterface del objeto COM especificado para get una interfaz específica de este objeto para que pueda ser utilizada con [COMCallMétodo](../WDLang1/1000018832.md). 




<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = COMQueryInterface(<COM Object> , <GUID>)`
---

**`<Result>: (COMObject Variable)`**

variable [COMObject](../WDLang1/1000018983.md) que puede ser utilizada por [COMCallMétodo](../WDLang1/1000018832.md). 

**`<COM Object>: (Objeto COM)`**

Objeto COM desde el que se debe recuperar la interfase. 

**`<GUID>: (Cadena de caracteres)`**

GUÍA de la interfaz del objeto a get. Las cadenas GUID deben incluir 32 caracteres hexadecimales separados (o no) por guiones (por ejemplo: 265EBF50-D4A0-448D-ADC6-EAD971CFB537).

## Componente:
wd280vm.dll

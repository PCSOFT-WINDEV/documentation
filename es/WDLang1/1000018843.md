
## AppListVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera las versiones disponibles en el servidor de reference instalación. Estas versiones están disponibles en la historia de la reference instalación. 

**Observación**: Esta función sólo puede ser utilizada con una red instalación en una red local. No puede ser usado con una instalación por IP.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res is string = AppListVersion()
IV is InfoVersion

DisplayInfo is string = [
Number = %1
Date = %2
Size = %3 bytes
Comment = %4
Enabled = %5
Forced = %6
Forbidden = %7
]

FOR EACH STRING MyString OF Res SEPARATED BY CR
	IV = AppVersionInfo(MyString)
	Trace(StringBuild(DisplayInfo, ...
		IV.VersionNumber, IV.Date, ...
		IV.Size, IV.Comment, ...
		IV.Enabled, IV.Forced, IV.Forbidden))
	Trace("===============================================")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = AppListVersion([<Application path>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de versiones disponibles separadas por caracteres CR (Carriage Return). Esta lista tiene el siguiente formato: 

```txt
<Version number 1> + CR + <Version number 2> + CR + ... 
+ CR + <Version number N>
```


**`<Application path>: (Cadena de caracteres opcional)`**

ruta completa del ejecutable a utilizar. Si no se especifica este parámetro, se gestionará la aplicación actual.

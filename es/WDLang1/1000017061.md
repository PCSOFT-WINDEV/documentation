


## TableToText (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un string a partir de los datos encontrados en una Tabla o control Tabla TreeView.. Esta cadena se puede guardar en un formato específico a partir de entonces.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sText is string
sText = TableToText(TABLE_TABLE1, taNoTitle, ";")
fSaveText("Export.csv", sText)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableToText(<Table control> [, <Options> [, <Column separator> [, <Row separator> [, <Start row> [, <End row>]]]]])`
---

**`<Result>: (Cadena de caracteres)`**



- Cadena de caracteres que contiene los datos de la control Tabla. En esta cadena, las columnas están separadas por &lt;Separator&gt; y las filas están separadas por caracteres CR (Carriage Return). 

- Cadena vacía ("") si se produce un error o si la control Tabla está vacía




**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla. 

- un control Tabla TreeView.




**`<Options>: (Constante opcional)`**

Configura la información que debe tenerse en cuenta en la cadena.


|   |   |
| --- | --- |
| *taColumnDisplayedOrder* | Exporta las columnas según el orden de las columnas que aparecen actualmente en la control (y no según el orden de las columnas definido en el editor).<br> |
| *taColumnsTitles*<br>(Valor predeterminado) | Se inserta el título de las columnas. |
| *taNoTitle* | Sólo se copian los datos. |
| *taNoTotal* | No exporta: <br><br>- las filas que contienen los totales, la media y la automática Count( estas filas son exportadas por Default).<br><br>- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) las filas que contienen cálculos personalizados (estas filas son exportadas por Default).<br><br><br> |
| *taSelectedLines* | Exporta sólo las filas seleccionadas (todas las filas son exportadas por Default). |
| *taWithInvisibleColumns* | También exporta las columnas de Invisible control.<br>![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Las columnas definidas como no imprimibles y no exportables se exportan. Estas columnas se definen: <br><br>- mediante programación: La propiedad [VisibleInExportAndPrint](../Proprietes/1000022320.md) se establece en <u><u><u><u>False</u></u></u></u>. <br><br>- en la pestaña "General" de la columna de la ventana de description: "Exportar e imprimir" set a "Nunca".<br><br><br> |



**`<Column separator>: (Cadena de caracteres opcional)`**

Separador utilizado entre las columnas. El separador Default es la tabulación (TAB).

**`<Row separator>: (Cadena de caracteres opcional)`**

Separador utilizado entre las filas. El separador utilizado por Default es el carácter CR (Carriage Return).

**`<Start row>: (Entero opcional)`**

Número de la fila donde se iniciará la exportación. 
Si no se especifica este parámetro, el fila de inicio es el primer fila del control.

**`<End row>: (Entero opcional)`**

Número de la fila donde terminará la exportación. 
Si no se especifica este parámetro, el fila final es el último fila del control.

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000127"></a>

La función **TableToText** puede utilizarse en: 

- controles Tabla o Tabla TreeView basados en un archivo de datos.

- controles Tabla o Tabla TreeView rellenados mediante programación.

- controles de selección única o múltiple. 



<a name="NOTE0_2"></a>


### Límite
<a name="limite_ELTPARAGRAPHE000142"></a>

Las columnas combinadas se ignoran.


## Constantes interactivas de gestión de Notification
			

<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para las notificaciones interactivas de manipular:

| **sysNotifStandard** | Comportamiento específico de la Notification : La Notification aparece en pantalla durante unos segundos antes de entrar en el Centro de Acción de Windows. | [sysNotification](../WDLang1/1410087893.md) |
| --- | --- | --- |
| **sysNotifReminder** | Comportamiento específico de la Notification : La Notification permanece en pantalla hasta que el usuario interactúa con ella. Cuando aparece la Notification, se reproduce una alerta sonora "recordatoria" definida por el sistema. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifCall** | Comportamiento específico de la Notification : La Notification permanece en pantalla hasta que el usuario interactúa con ella. Se reproduce en bucle un sonido de llamada entrante definido por el sistema y se utiliza un estilo diferente para los botones. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifAlarm** | Comportamiento específico de la Notification : La Notification permanece en pantalla hasta que el usuario interactúa con ella. Una alerta sonora "Reminder" definida por el sistema se reproduce en bucle. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifReminderButton** | Comportamiento específico o posicionamiento del botón: "Botón del sistema "Recordar más tarde | [sysNotificationButton](../WDLang1/1410087947.md) |
| **sysNotifCloseButton** | Comportamiento específico o posicionamiento del botón: Botón del sistema que cierra automáticamente el Notification. | [sysNotificationButton](../WDLang1/1410087947.md) |
| **sysNotifContextButton** | Comportamiento específico o posicionamiento del botón: Botón que se añade al menú contextual de la Notification. | [sysNotificationButton](../WDLang1/1410087947.md) |
| **sysNotifStandardButton** | Comportamiento específico o posicionamiento del botón: Botón simple. | [sysNotificationButton](../WDLang1/1410087947.md) |
| Constante | Detalles | Tipo de variable |
| **sysNotifCircleCrop** | Recorte que se aplicará al logotipo: Image con un círculo. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifNoCrop** | Recorte que se aplicará al logotipo: Sin recorte, cuadrado Image. | [sysNotification](../WDLang1/1410087893.md) |
| **sysNotifInputCombo** | Tipo de control que permite la interacción del usuario : Control Combo Box. | [sysNotificationInput](../WDLang1/1410088024.md) |
| **sysNotifInputText** | Tipo de control que permite la interacción del usuario : texto control Campo de entrada. | [sysNotificationInput](../WDLang1/1410088024.md) |

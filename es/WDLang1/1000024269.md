


## &lt;Procesamiento de palabras&gt;.open (función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre un archivo de procesamiento de texto (formato docx) en modo Lectura/Escritura. El archivo docx se abre y se bloquea hasta que se cierra.

**Observación**: Para crear un archivo docx sin utilizar un archivo inicial, se debe crear una variable de tipo [Document](../WDLang1/1000022461.md) (en este caso, no es necesario utilizar la función **&lt;Procesamiento de palabras&gt;.open**).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
WP_MyWP.Open("C:\Users\test\Documents\file.docx")

IF ErrorOccurred() THEN
	Error(ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.Open(<Path> [, <Password>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se abrió el docx en el control Procesador de texto,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Word Processing control>: (Nombre del control)`**

Nombre de control Procesador de texto donde se abrirá el archivo docx.

**`<Path>: (Cadena de caracteres)`**

Corresponde a uno de los siguientes elementos: 

- ruta completa o relativa de archivo docx para abrir (hasta 260 caracteres).

- Nombre de la variable Buffer con el docx Document para mostrar en el control Procesador de texto. 

- Nombre de la nota con el documento Document para mostrar en el control Procesador de texto. Este parámetro tiene el siguiente formato: &lt;Archivo de datos&gt; &lt;Elemento&gt;.




**`<Password>: (Cadena de caracteres opcional)`**

Contraseña para abrir el archivo docx a utilizar (si el archivo está protegido por una contraseña).

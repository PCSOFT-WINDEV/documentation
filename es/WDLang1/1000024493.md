


## &lt;Variable WDPic&gt;.InvertColor (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Invierte los colores en una variable de tipo [WDPic](../WDLang1/1000023503.md).<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dInverseCouleur.gif)
![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
// Inverts the colors of MyWDPicImage
Result1 is boolean
Result1 = MyWDPicImage.InvertColor()
IF Result1 = False THEN Error("The operation failed")
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <WDPic image>.InvertColor()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se invirtieron los colores,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.

Cada pixel de la imagen toma el color opuesto a su color inicial (negación bit a bit).. La imagen manipulada se modifica directamente.

### Fin del dibujo

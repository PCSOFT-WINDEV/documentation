
## NoAccent (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte los caracteres acentuados de una cadena en caracteres no acentuados.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
NoAccent("élan")    // Returns "elan"
NoAccent("fiancée")   // Returns "fiancee"
NoAccent("")          // Returns ""
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = NoAccent(<Initial string>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres en la que todos los caracteres acentuados han sido reemplazados por caracteres no acentuados.

**`<Initial string>: (Cadena de caracteres)`**

Cadena de caracteres inicial.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Función NoAccent y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000068"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **NoAccent** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

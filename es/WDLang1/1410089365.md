
## France.CheckSIREN (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número de SIREN francés (Système d'Identification du Répertoire des Entreprises).

ATENCIÓN: Esta información es específica de la administración francesa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MySIRENNumber is string = "330318270"
France.CheckSIREN(MySIRENNumber) // Returns True

// Check a control
EDT_SIRENNumber = "ABC"
France.CheckSIREN(EDT_SIRENNumber) // Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = France.CheckSIREN(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número de SIREN válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número de SIREN a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

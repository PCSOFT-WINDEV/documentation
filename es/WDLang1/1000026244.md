
## InvalidInputListControl( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una lista de controles que requieren entrada de datos/contienen datos no válidos. Los controles no se marcan como que contienen errores.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
arrInvalidControls is array of Control = InvalidInputListControl()
IF NOT arrInvalidControls.Empty THEN
	Error("Problem")
END

ControlList is array of Control = InvalidInputListControl()
FOR EACH ELEMENT c OF ControlList
	c.BackgroundColor = LightRed
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = InvalidInputListControl([<Element used>])`
---

**`<Result>: (Array de control)`**

Array de las variables de [Control](../WDLang1/1000020967.md) que contienen la lista de controles que requirieron la introducción de datos/contienen datos no válidos. Esta array está ordenada según la orden de tabulación de los controles.

**`<Element used>: (Nombre de control o ventana opcional)`**

Nombre de la control o de la ventana que hay que comprobar. Si el control es un contenedor ( control Pestaña , Super control , etc.), se comprobarán control. 
Si no se especifica este parámetro, se comprobarán todos los controles de la ventana actual.

## Observaciones


### Características de la verificación/detección de los controles que requieren la introducción de datos/contienen datos no válidos
<a name="caracteristicas_verificaciondeteccion_los_controles_que_requieren_introduccion_datoscontienen_datos_validos_ELTPARAGRAPHE000038"></a>

La entrada se comprobará en los siguientes tipos de controles: 

- **Control Botón de opción/Casilla de verificación**: si se ha habilitado la opción de entrada requerida (o la propiedad [MandatoryInput](../Proprietes/1000026174.md) se establece como <u><u><u><u>True</u></u></u></u>), comprueba que al menos una opción está seleccionada.

- **Control Valoración**: si se ha habilitado la opción de entrada requerida (o la propiedad [MandatoryInput](../Proprietes/1000026174.md) está establecida en <u><u><u><u>True</u></u></u></u>), comprueba que se ha seleccionado una calificación.

- **Combo Box con entrada**: si se ha habilitado la opción de entrada requerida (o la propiedad [MandatoryInput](../Proprietes/1000026174.md) está establecida como <u><u><u><u>True</u></u></u></u>), comprueba que se ha introducido o seleccionado un valor en el cuadro combinado. 
	Si se ha utilizado una máscara de entrada, la función comprueba que los datos introducidos coinciden con esta máscara. 
	Si se utiliza una máscara personalizada con una variable [InputMask](../WDLang1/1000024886.md), la verificación se realiza ejecutando el procedimiento asociado al **CheckIfValidInput** Property de esta máscara.

- **Combo Box sin entrada**: si se ha habilitado la opción de entrada requerida (o la propiedad [MandatoryInput](../Proprietes/1000026174.md) está establecida como <u><u><u><u>True</u></u></u></u>), comprueba que se ha seleccionado un valor en el cuadro combinado. 

- **Control Campo de entrada**: si se ha habilitado la opción de entrada requerida (o la propiedad [MandatoryInput](../Proprietes/1000026174.md) está establecida como <u><u><u><u>True</u></u></u></u>), comprueba que se ha introducido o seleccionado un valor en el campo control. 
	Si se ha utilizado una máscara de entrada, la función comprueba que los datos introducidos coinciden con esta máscara. Si se utiliza una máscara personalizada con una variable [InputMask](../WDLang1/1000024886.md), la verificación se realiza ejecutando el procedimiento asociado al **CheckIfValidInput** Property de esta máscara.


Sólo se comprueban los controles activos y Visible.
<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000100"></a>

- Para resaltar los errores en los controles que requieren la introducción de datos/contienen datos no válidos, utilice [InvalidInputDetect](../WDLang1/1000026180.md) o [InvalidInputShowMessage](../WDLang1/1000026247.md).

- Caso especial: Control de edición o control Combo Box editable con máscara personalizada con un [InputMask](../WDLang1/1000024886.md) Variable. Si el control no tiene el foco cuando se llama a **InvalidInputListControl**, el texto mostrado en el control se formateará primero con el **FormatDuringEntry** Property de la máscara antes de comprobar su validez.


## CameraListDriver( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los controladores de captura de vídeo instalados en el ordenador actual.

Observación: Esta función es equivalente a la función [VideoListDriver](../WDLang1/3079008.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the video drivers installed
// Display them in LIST_DriverList
ListAdd(LIST_DriverList, CameraListDriver(viDirectShow))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = CameraListDriver([<Technology used>])`
---

**`<Result>: (Cadena de caracteres)`**

Lista de controladores de vídeo instalados en el equipo, en el siguiente formato:

```txt
<Name of driver 1> + TAB + <Version of driver 1> + CR + ... 
<Name of driver N> + TAB + <Version of driver N>
```


Donde: 

- &lt;Nombre del conductor 1&gt; ... &lt;Nombre del controlador N&gt; corresponden al nombre de los controladores de vídeo, 

- &lt;Versión del conductor 1&gt; ... &lt;Versión del conductor N&gt; corresponden a la versión del conductor.




**`<Technology used>: (Constante de tipo Integer)`**

Tipo de tecnología utilizada para listar los drivers: 


|   |   |
| --- | --- |
| *vidDirectShow* | La tecnología utilizada: Mostrar directamente. <br><br>Esta constante debe utilizarse para seleccionar un controlador utilizado en un control Conferencia ([StreamChangeDriver](../WDLang3/1000021613.md)). |
| *vidVideoForWindows*<br>(Valor predeterminado) | La tecnología utilizada: Video para Windows |

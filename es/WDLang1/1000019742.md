


## &lt;TreeMap&gt;.Seek (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca el índice de un elemento en una control TreeMap a partir del elemento ruta. Esta ruta se especificó en [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The TMAP_Wine control is filled from a Table control.
// Find the element corresponding to the selected element in the Table control
SelectionPath is string
SelectionPath = COL_WineType[TABLE_WINE] + "\" + COL_Owner[TABLE_WINE]
ElementIndex = MAP_Wine.Seek(SelectionPath)
// Select the element in the TreeMap control
IF ElementIndex <> - 1 THEN
	TMAP_Wine = ElementIndex
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <TreeMap control>.Seek(<Element path> [, <Start index>])`
---

**`<Result>: (Integro)`**



- Índice del elemento encontrado,

- -1 si ningún elemento corresponde con la búsqueda. 


**Observación**: El elemento encontrado puede ser manejado directamente usando su índice. Puede, por ejemplo, modificar las propiedades de este elemento o seleccionarlo en el control TreeMap.

**`<TreeMap control>: (Nombre del control)`**

Nombre de la control TreeMap utilizada.

**`<Element path>: (Cadena de caracteres)`**

Ruta y título del elemento buscado. Esta ruta debe ser idéntica a la que se utilizó cuando se añadió el elemento en la control TreeMap mediante [&lt;Mapa del árbol&gt;.add](../WDLang1/1000019740.md). 
El separador de los diferentes elementos de ruta puede ser:

- el carácter "\\" ".. Por ejemplo: "MyDirectory\\MyFile". 

- el carácter "TAB. Por ejemplo: "Postres" +TAB+"Helados".




**`<Start index>: (Entero opcional)`**

Índice del elemento a partir del cual se realizará la búsqueda. Si no se especifica este parámetro, la búsqueda se realiza en toda la control TreeMap. Este parámetro se utiliza para encontrar todas las ocurrencias de una ruta.


## WinInWidth (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el ancho interno de una ventana (sin Border).
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=FenExtInt.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Internal width of the "WIN_EditCustomer" window
ResInternalWidth = WinInWidth(WIN_EditCustomer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinInWidth([<Window used>])`
---

**`<Result>: (Integro)`**

Ancho interno de la ventana (en píxeles).

**`<Window used>: (Nombre opcional de la ventana o cadena opcional)`**

Nombre o alias de la ventana que se va a utilizar. Si este parámetro no se especifica o corresponde a una cadena vacía (""), se devuelve la anchura interna de la ventana actual.

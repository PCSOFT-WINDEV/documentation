
## RandomCryptoBuffer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve un buffer al tamaño especificado lleno de datos aleatorios, que puede ser utilizado con las tecnologías criptográficas.

Observación: El búfer devuelto se puede utilizar como clave para las funciones de encriptación, por ejemplo. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Generate a 32-byte random key (256 bits)
RandomKey is Buffer = RandomCryptoBuffer(32)
// Encryption with the generated key
EncryptedMessage is Buffer = CryptStandard(MyMessage, RandomKey)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = RandomCryptoBuffer([<Buffer size>])`
---

**`<Result>: (Buffer)`**



- Buffer lleno de datos aleatorios,

- Buffer vacío si se produce un error. 




**`<Buffer size>: (Entero opcional)`**

Tamaño (en número de bytes) de la memoria intermedia para inicializar (32 por Default). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La primera llamada a la función puede ser bastante larga (un segundo o más si el tamaño del búfer a inicializar es importante).. De hecho, la inicialización del mecanismo aleatorio es muy compleja. 

- La función [InitRandom](../WDLang1/3013030.md) no tiene ningún efecto en el resultado de la función **RandomCryptoBuffer**.

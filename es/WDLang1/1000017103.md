
## ServiceInfo( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre la configuración de un servicio.. La configuración del servicio se asigna a la [Servicio Variable](../WDLang1/1000017118.md).

Este servicio se puede encontrar en el ordenador local o en un ordenador remoto.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find out the characteristics of the service found in TABLE_Service
IF ServiceInfo(COL_Name[TABLE_Service]) = False THEN
	Error(ErrorInfo(errMessage))
ELSE
	Info("The Service variable is updated")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ServiceInfo(<Service name> [, <Remote computer>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se solicitó la información,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md). El servicio Variable está lleno de información sobre la configuración del servicio.




**`<Service name>: (Cadena de caracteres)`**

Nombre del servicio a utilizar.

**`<Remote computer>: (Cadena de caracteres opcional)`**

IP Address del ordenador remoto en formato xxx.xxx.xxx.xxx (devuelto por [NetIPAddress](../WDLang3/3056023.md)) o nombre del ordenador remoto (devuelto por [NetMachineName](../WDLang3/3056014.md)). Este parámetro se utiliza para get información sobre un servicio en otro ordenador. Para utilizar esta opción, el usuario debe ser administrador del equipo remoto.. 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), el servicio se gestiona localmente..

<a name="NOTE0_1"></a>

## Observaciones




### Servicio Variable
<a name="servicio_variable_ELTPARAGRAPHE000059"></a>

El Servicio Variable se utiliza para averiguar los siguientes elementos. Para más detalles, consulte la página de ayuda de [Servicio Variable](../WDLang1/1000017118.md). 


|   |   |
| --- | --- |
| Error de puesta en marcha del servicio | Los errores que se produjeron al iniciar un servicio son procesados por Windows (para los servicios con inicio de automatic). |
| Línea de comandos de servicio | Comando completo Line del servicio (ruta del ejecutable y parámetros). |
| Puesta en marcha del servicio | Modo de inicio del servicio |
| Servicio. description | Descripción del servicio |
| Tipo de servicio | Tipo de servicio |
| Usuario de servicio | Cuenta de usuario que ejecutará el servicio. |

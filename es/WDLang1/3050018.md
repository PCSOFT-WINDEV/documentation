
## Ln( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el logaritmo napieriano (inverso de exponencial) de un valor numérico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Calculate the napierian logarithm of 12
ResNapierianLog = Ln(12)
// Returns 2.48490664979
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = Ln(<Numeric value>)`
---

**`<Result>: (Real)`**

Logaritmo napieriano solicitado.

**`<Numeric value>: (Entero o real)`**

Valor numérico a utilizar. Se produce un error de WLanguage si este parámetro es un número negativo o un número nulo..

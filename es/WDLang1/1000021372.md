


## TabCloseAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra todos los cristales dinámicos de un control Pestaña. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
TabCloseAll(TAB_DynTab)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TabCloseAll(<Tab control>)`
---

**`<Tab control>: (Nombre del control)`**

Nombre de la control Pestaña que se va a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Todas las ventanas internas associated con los paneles dinámicos se descargan y los paneles dinámicos se borran del control Pestaña especificado.

## Componente:
wd280obj.dll

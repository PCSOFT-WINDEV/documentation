


## DocClose (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Cierra el archivo docx y libera el procesador de textos Document. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
DocClose(WP_MyDocument)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`DocClose(<Document>)`
---

**`<Document>: (Nombre del documento Variable o control)`**

Document para usar. Este Document corresponde a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de un control Procesador de texto.

- una variable de tipo [Document](../WDLang1/1000022461.md).


El archivo físico asociado al control o a la Variable (si existe) se cierra. Las modificaciones realizadas en el archivo no se guardarán hasta que se cierre (utilice [DocSave](../WDLang1/1000022387.md) si es necesario).

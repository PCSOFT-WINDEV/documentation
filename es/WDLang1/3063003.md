
## Constantes para la gestión del portapapeles
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes son utilizadas por las funciones para la gestión del portapapeles: 

| Constante | description | Detalles | Utilizado por la función |
| --- | --- | --- | --- |
| **cfBitmap** | Formato encontrado en el portapapeles | Image format | [ClipboardFormat](../WDLang1/3063002.md) |
| **cfRtf** | Formato encontrado en el portapapeles | Formato RTF | [ClipboardFormat](../WDLang1/3063002.md) |
| **cfText** | Formato encontrado en el portapapeles | Formato texto | [ClipboardFormat](../WDLang1/3063002.md) |
| **cfUnicodeText** | Formato encontrado en el portapapeles | Formato de texto Unicode | [ClipboardFormat](../WDLang1/3063002.md) |




## SpreadsheetSeek (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Busca un valor en las celdas de un control Hoja de cálculo encontrado en una ventana.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sCell is string = SpreadsheetSeek(PSHEET_NoName1, "%CUSTOMERNAME%")
// Processes the result of the search
IF sCell <> "" THEN
	 // Replaces %CUSTOMERNAME% by Doe
	 PSHEET_NoName1[sCell] = "Doe"
END
```

```wl
// Browse all cells with "VAT"
sCell is string = SpreadsheetSeek(PSHEET_NoName1, "VAT")
WHILE  sCell <> ""
	// Replaces VAT by PSHEET_NoName1[sCell] = 20.6
	Trace(sCell)
	// Next cell
	sCell = SpreadsheetSeek(PSHEET_NoName1, "VAT", sCell)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Hoja de cálculoSeek(<Spreadsheet control> , <Sought value> [, <Start> [, <Options>]])`
---

**`<Result>: (Character string)`**



- Nombre de la celda encontrada ("A1" por ejemplo). Si se encuentran varias celdas, &lt;Result&gt; contiene el nombre de la primera celda encontrada. 

- "" (cadena vacía) si no se encuentra ninguna celda. 




**`<Spreadsheet control>: (Control name)`**

Nombre de control Hoja de cálculo a utilizar. 

**`<Sought value>: (Type of sought value)`**

Valor a encontrar en las celdas de control Hoja de cálculo. Este valor puede corresponder a un número o a un string. 

**`<Start>: (Optional character string)`**

Nombre de la persona desde la que se realizará la búsqueda. Esta celda está excluida de la búsqueda. 
Observación: este parámetro se usa para navegar por todas las celdas que contienen un valor. 

Si no se especifica este parámetro, la búsqueda se realiza desde la celda "A1. 

**`<Options>: (Optional Integer constant)`**

Opciones de búsqueda: 


|   |   |
| --- | --- |
| *psheetSearchWithCase* | Buscar teniendo en cuenta el caso. |
| *psheetSearchContain* | "Contiene" búsqueda. |
| *psheetSearchInFormula* | Buscar sólo en las fórmulas. |
| *psheetSearchExact* | "Búsqueda"exacta. |
| *psheetSearchCurrentWorksheet* | Buscar en la hoja de trabajo actual. |
| *psheetSearchNoCase* | Buscar ignorando el caso. |
| *psheetSearchAnyWorksheet* | Buscar en todas las hojas de trabajo del libro de trabajo. |


Por Default, la búsqueda se realiza según la siguiente combinación de constantes: *psheetSearchWithCase* + *psheetSearchCurrentWorksheet* + *psheetSearchExact*



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La búsqueda se realiza en el siguiente orden: A1, B1, C1, ..., A2, B2, C2, ...

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..

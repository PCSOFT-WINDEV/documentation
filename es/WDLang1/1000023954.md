


## EditInsertToken (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta un token en un control Campo de entrada de tipo "Tokens de texto". 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click [Insert the test robot] 
EditInsertToken(EDT_Recipient, 1, "ROBOT", "robotmail@address.test.com")
```

```wl
// Syntax using the Token type
MyToken is Token
MyToken.DisplayedValue = "ROBOT"
MyToken.StoredValue = "robotmail@address.test.com"
EditInsertToken(EDT_Recipient, 1, MyToken)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Insertar un Token directamente

`<Result> = EditInsertToken(<Edit control> , <Position> , <Value> [, <Caption>])`
---

**`<Result>: (Integro)`**



- índice real de la Token insertada,

- 0 si se produce un error:

	- Si el control Edición no es un control "Fichero de texto" (la propiedad [TokenEnabled](../Proprietes/1000023782.md) está establecida en <u><u><u><u>False</u></u></u></u>). 

	- Si se ha detectado un duplicado y la propiedad [TokenAllowDuplicate](../Proprietes/1000023784.md) tiene el valor <u><u><u><u>False</u></u></u></u>.







**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" Editar control. 

**`<Position>: (Integro)`**

Inserción de Token índice. 

- Si  es mayor que el número de Token s en el control, el Token se añade al final de los Token s existentes ([EditAddToken](../WDLang1/1000023953.md)).

- Si  es set a 0, el Token se añade al principio. 

- Si  es set a -1,  es 0.




**`<Value>: (Cadena de caracteres)`**

Valor de la Token. 

**`<Caption>: (Cadena de caracteres opcional)`**

Pie de foto de la Token. Si no se especifica este valor, corresponderá al valor de la Token.

<a name="SYNTAX2"></a>

### Insertar un Token a través del tipo Token

`<Result> = EditInsertToken(<Edit control> , <Position> , <Token>)`
---

**`<Result>: (Integro)`**



- índice real de la Token insertada,

- 0 si se produce un error:

	- Si el control Edición no es un control "Fichero de texto" (la propiedad [TokenEnabled](../Proprietes/1000023782.md) está establecida en <u><u><u><u>False</u></u></u></u>). 

	- Si se ha detectado un duplicado y la propiedad [TokenAllowDuplicate](../Proprietes/1000023784.md) tiene el valor <u><u><u><u>False</u></u></u></u>.







**`<Edit control>: (Nombre del control)`**

Nombre del "Texto Token" Editar control. 

**`<Position>: (Integro)`**

Inserción de Token índice. 

- Si  es mayor que el número de Token s en el control, el Token se añade al final de los Token s existentes ([EditAddToken](../WDLang1/1000023953.md)).

- Si  es set a 0, el Token se añade al principio. 

- Si  es set a -1,  es 0.




**`<Token>: (Ficha Variable)`**

Nombre de la [Token](../WDLang1/1000023942.md) Variable que describe la Token a añadir.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Para añadir una Token en la posición del cursor, utilice [EditAddToken](../WDLang1/1000023953.md). 

- El control Campo de entrada [StoredValue](../Proprietes/2510127.md)Property devuelve los valores almacenados para los tokens, separados por los caracteres  (Retorno de carro).

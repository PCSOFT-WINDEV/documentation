
## gBackground( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres utilizados para definir la pantalla Color (fondo Color de un texto, círculo, rectángulo, ...). 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En una ventana, esta secuencia de caracteres se utiliza para definir la Color de fondo de un texto o de una forma (rectángulo o círculo): 

- en un elemento que se encuentra en un List Box o control Combo Box.

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 




**Observación**: Para utilizar un gradiente Color, llame a [gBackgroundGradient](../WDLang1/1000017060.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Add the word "Yellow" on a yellow background into "LIST_ColorList"
ListAdd(LIST_ColorList, gBackground(LightYellow) + "Yellow")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gBackground(<Color>)`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que definen el fondo Color del texto, círculo, rectángulo, ... 

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) usado en la barra de título o barra de estado de una ventana. 




**`<Color>: (Entero o constante)`**

Mostrar Color para la Background del texto o para la sección interior de una forma (rectángulo o círculo). Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).  

- la constante *Transparent* (utilizada para superponer un texto y un gráfico por ejemplo).

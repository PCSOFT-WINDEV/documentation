
## PanelLoadConfiguration( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la configuración de los controles del Panel acoplable en una ventana.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Save the current configuration
sConf is string = PanelSaveConfiguration()


// Restore the configuration saved in sConf
PanelLoadConfiguration(sConf)
IF ErrorOccurred THEN ToastDisplay(ErrorInfo())
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PanelLoadConfiguration(<Configuration>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cargó la configuración.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Configuration>: (Cadena de caracteres)`**

Configuración para restaurar. Esta configuración es el resultado de [PanelSaveConfiguration](../WDLang1/1000023994.md).

Atención: Si el formato de este parámetro no es válido o no puede aplicarse al control Panel acoplable, &lt;Resultado&gt; se establece en <u><u><u><u>False</u></u></u></u>.

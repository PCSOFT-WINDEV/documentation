


## TelemetryAddPassage( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Señala la ejecución del código para un sensor de paso.

Esto le permite a get estadísticas de uso sobre una característica de la aplicación : 

- número total de usuarios o porcentaje de usuarios que han utilizado la función durante un período,

- número mínimo, medio o máximo de usuarios que han utilizado la función todos los días, ....





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// An "OrderCancellation" sensor was defined in the editor.
// Whenever an order is canceled, we identify the use of the feature 
// by the user
TelemetryAddPassage(OrderCancellation)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`TelemetryAddPassage(<Sensor> [, <Category value>])`
---

**`<Sensor>: (Cadena de caracteres)`**

Nombre del sensor de paso definido en el editor. 

**`<Category value>: (Cadena de caracteres opcional)`**

Valor de la categoría definida en el editor.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Los sensores deben definirse en una telemetría personalizada.

- Se produce un error de WLanguage si el sensor no es un sensor de paso.

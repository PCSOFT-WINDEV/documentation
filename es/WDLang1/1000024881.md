
## SpreadsheetListComment (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea una lista de todos los comentarios de la hoja actual de un control Hoja de cálculo. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Make all comments visible
FOR EACH Comment OF SpreadsheetListComment(PSHEET_MySpreadsheet)
	Comment..Visible = True
END
```

```wl
// Delete all comments
FOR EACH Comment OF SpreadsheetListComment(PSHEET_MySpreadsheet)
	Comment..Text = ""
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SpreadsheetListComment(<Spreadsheet control>)`
---

**`<Result>: (Array)`**

Array con los diferentes comentarios y sus características. 

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se modifica un comentario en el array, también se modifica en el control Hoja de cálculo. 

- Agregar o eliminar un comentario en el array, no produce ningún efecto en los comentarios del control Hoja de cálculo. 

- Si no hay comentarios en la hoja, el array está vacío.

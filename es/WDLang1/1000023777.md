


## <PDF Reader>.SeekPrevious (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija la posición sobre el resultado anterior de una búsqueda iniciada con [.SeekFirst](../WDLang1/1000023778.md) en el contenido de un lector PDF control. 

Observación: [.SeekNext](../WDLang1/1000023779.md) encuentra la siguiente instancia del texto buscado. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Seeks a text in a PDF Reader control and displays it
PDF_MyPDFReader.SeekFirst("company")
// Finds the next element
IF PDF_MyPDFReader.SeekNext() = False THEN
	Info("Search ended")
END

// Finds the previous element
PDF_MyPDFReader.SeekPrevious()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <PDF Reader control>.SeekPrevious()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto especificado en [.SeekFirst](../WDLang1/1000023778.md) se encontró en el actual Document en PDF Reader control ,

- <u><u><u><u>False</u></u></u></u> si no se encontró el texto.




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El texto y las opciones de búsqueda se han especificado en [.SeekFirst](../WDLang1/1000023778.md).

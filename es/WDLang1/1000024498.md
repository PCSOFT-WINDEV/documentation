
## &lt;Variable WDPic&gt;.PixelOpacity (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Identifica la opacidad de un pixel en una imagen (capa de fondo de una variable de tipo [WDPic](../WDLang1/1000023503.md)) con canales Alfa.

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyWDPicImage is WDPic = "Test.bmp"
// Identify the opacity of the pixel (15, 25)
// on MyWDPicImage
ResOpacity = MyWDPicImage.PixelOpacity(15, 25)
IMG_MyDrawing = MyWDPicImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <WDPic image>.PixelOpacity(<X-coordinate> , <Y-coordinate>)`
---

**`<Result>: (Entero)`**

Opacidad del pixel especificado (comprendido entre 0 para transparente y 255 para opaco).

**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.

**`<X-coordinate>: (Entero)`**

Posición horizontal del pixel a manipular.

**`<Y-coordinate>: (Entero)`**

Posición vertical del pixel a manipular.

## Observaciones


### Coordenadas
<a name="coordenadas_ELTPARAGRAPHE000347"></a>
<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dpixel.gif)
Las coordenadas se indican en pixeles con respecto a la esquina superior izquierda de la imagen (coordenadas (0,0)).

<a name="NOTE0_3"></a>


### Dibujo en una imagen con opacidad
<a name="dibujo_una_imagen_con_opacidad_ELTPARAGRAPHE000448"></a>

Para especificar que el dibujo de la imagen admite la opacidad (canal Alfa), utilice la función [&lt;Variable WDPic&gt;.StartDrawing](../WDLang1/1000024471.md) y la constante ***dWithOpacity***.

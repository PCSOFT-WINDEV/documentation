


## ChronoPause (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Pone en pausa un cronómetro para medir el tiempo de un proceso. El cronometraje se puede reiniciar con la función [ChronoResume](../WDLang1/1000018830.md).






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ProcessDuration is Duration
ChronoStart()
MyFunction1()
ProcessDuration = ChronoPause()
Info("The MyFunction1 process lasted " + ProcessDuration) 
ChronoResume() 
MyFunction2() 
ProcessDuration = ChronoEnd()
Info("The MyFunction1 and MyFunction2 processes lasted " + ProcessDuration)
```

## Sintaxis

### Pausar el cronometraje iniciado con un número de cronómetro

`<Result> = ChronoPause([<Stopwatch number>])`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el número de cronómetro especificado.

**`<Stopwatch number>: (Entero opcional)`**

Identificador del cronómetro utilizado. Si no se especifica este parámetro, el valor predeterminado es 1.

### Pausar el cronometraje iniciado con una variable de tipo Chrono

`<Result> = ChronoPause(<Stopwatch>)`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [ChronoStart](../WDLang1/3027031.md) para el cronómetro especificado.

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000171"></a>

La función **ChronoPause** no tiene ningún efecto si se utiliza en un cronómetro en pausa. 
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000182"></a>

Si no se llama a la función [ChronoStart](../WDLang1/3027031.md) antes de la función **ChronoPause**, se produce un error WLanguage.
<a name="NOTE0_3"></a>


### Hilos
<a name="hilos_ELTPARAGRAPHE000196"></a>

Las funciones de cronometraje (Chronoxxx) relacionadas con el mismo cronómetro deben utilizarse en el mismo hilo.

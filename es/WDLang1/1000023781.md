
## LeapYear (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Indica si el año especificado es un año bisiesto o no.

**Recordatorio** : un año bisiesto es un año que tiene 366 días en lugar de 365. El día adicional es el 29 de febrero.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Indicates whether it's a leap year
IF LeapYear() THEN
	Info("It's a leap year.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = LeapYear([<Date>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si es un año bisiesto,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Date>: (Cadena de caracteres o entero opcionales)`**

Fecha para la que se debe identificar si corresponde a un año bisiesto. Este parámetro puede corresponder a: 

- una fecha o año como cadena de caracteres,

- una fecha, un DateTime o un año como número entero.


Si no se especifica este parámetro, se utilizará el año actual.  

Observación: El año debe estar entre 0 y 9999.

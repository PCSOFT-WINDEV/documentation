
## ProgressBar( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Muestra una barra de progreso en la barra de estado de la ventana actual.

**Observación**: La ventana debe tener una barra de estado.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Maxim is int = 234
FOR i = 1 TO Maxim
	ProgressBar(i, Maxim, "There is " + (Maxim-i)*100/Maxim + "% left")
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Mostrando la barra de progreso

`ProgressBar(<Progress Bar value> [, <Maximum value> [, <Progress Bar text> [, <Color>]]])`
---

**`<Progress Bar value>: (Integro)`**

Incremento del valor de la barra de progreso. Este valor debe estar comprendido entre 0 y .

**`<Maximum value>: (Entero opcional)`**

Valor máximo posible de la barra de progreso. Si no se especifica este parámetro, el valor máximo de la barra de progreso es set a 100.

**`<Progress Bar text>: (Cadena de caracteres opcional)`**

Pie de foto a la derecha de la barra de progreso. Si no se especifica este parámetro, el texto que se muestra tiene el siguiente formato: "NN%" donde NN representa el porcentaje de  en relación con .
Por ejemplo: 45%.

**`<Color>: (Entero o constante (opcional))`**

Color barra de progreso. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).




Si no se especifica este parámetro, la color utilizada corresponde a la color de los elementos seleccionados. Esta color se define en las propiedades de visualización del ordenador actual (panel control, opción "Pantalla", pestaña "Aspecto" en Windows NT).

## Observaciones


### Borrar la barra de progreso
<a name="borrar_barra_progreso_ELTPARAGRAPHE000081"></a>

Si se ha visualizado un texto en la barra de estado antes de que se visualizara la barra de progreso, este texto se restaura cuando se borra la barra de progreso.
<a name="NOTE0_2"></a>


### Barra de estado
<a name="barra_estado_ELTPARAGRAPHE000088"></a>

Para mostrar la barra de estado en una ventana, marque "Mostrar barra de estado" en la pestaña "Estilo" de la ventana description.

### Barra de progreso y ventana secundaria MDI
<a name="barra_progreso_ventana_secundaria_mdi_ELTPARAGRAPHE000101"></a>

**ProgressBar** , usado desde una ventana secundaria MDI sin barra de estado, usa la barra de estado de la ventana principal.

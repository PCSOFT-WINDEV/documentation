
## HTMLToRGB (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte el color HTML en color RGB.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Trace(HTMLToRGB("AliceBlue"))	// Returns 16775408
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = HTMLToRGB(<Color>)`
---

**`<Result>: (Integro)`**

Color RGB que corresponde al color HTML especificado. Este color es el resultado de la siguiente fórmula: 
Color = 65536 \* Componente azul + 256 \* Componente verde + Componente rojo.

**`<Color>: (Cadena de caracteres)`**

Color HTML expresado en uno de los siguientes formatos:  

- notación hexadecimal (por ejemplo #f00 o #ff0000).

- notación RGB (por ejemplo, RGB(100%, 0%, 0%)).

- nombre HTML del color (ver lista de abajo).

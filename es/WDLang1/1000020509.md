
## BufferToReal (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Extrae un número real de un buffer binario en una posición determinada. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Extract real at byte 52 from the Xyl.raw file

// Load buffer that corresponds to the Xyl.raw file
bMyBuffer is Buffer = fLoadBuffer("C:\Test\Xyl.raw")
// Check loading
IF bMyBuffer = "" THEN
	Error("Error reading buffer")
	RETURN
END
// Extract real at byte 52
rMyReal is real = BufferToReal(bMyBuffer, 52)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BufferToReal(<Buffer to use> [, <Position> [, <Number of bytes to read>]])`
---

**`<Result>: (Real)`**

Real extraído del buffer. 

**`<Buffer to use>: (Buffer)`**

Nombre de la variable buffer cargada en memoria que contiene el valor real. Este parámetro también puede corresponder a una cadena Ansi o Unicode.

**`<Position>: (Entero opcional)`**

Posición en la que se buscará el número real. La primera posición del buffer corresponde a 0.
El valor predeterminado de &lt;Posición&gt; es 0.

**`<Number of bytes to read>: (Entero opcional)`**

Número de bytes en los que está codificado el número real. El valor de este parámetro puede ser 4 u 8. 
De forma predeterminada, &lt;Número de bytes a leer&gt; se establece en 4. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El real se lee al inicio del buffer como un real de 4 bytes.

- Los bytes se leen en modo Little-Endian (el byte menos significativo se lee primero).

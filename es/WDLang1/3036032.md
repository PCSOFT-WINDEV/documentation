


## fShortName( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el nombre corto de un archivo o directorio.

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003036032&name=fshortname_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Retrieve the short name of a file
ResShortName = fShortName("C:\MyDirectories\Report March 2001.doc")
// fShortName returns "March~1.doc"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = fShortName(<File path> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**



- Nombre corto del fichero o directorio. Sólo se devuelve el nombre corto del fichero o directorio.

- Vaciar la cadena (""") si se ha producido un error (archivo o directorio no encontrado). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<File path>: (Cadena de caracteres)`**

Nombre y ruta completa (o relativa) del fichero o directorio (hasta 260 caracteres). Se puede utilizar una ruta UNC.  
![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) Este parámetro puede estar en formato Ansi o Unicode.

**`<Options>: (Constante opcional de tipo Integer)`**

Se usa para especificar el formato del resultado de la función. Por Default, **fShortName** devuelve una cadena de caracteres en formato Ansi. 


|   |   |
| --- | --- |
| *fPathUNICODE* | &lt;Resultado&gt; será una cadena de caracteres en formato Unicode. |


**Observación**: El resultado de **fShortName** es siempre una cadena de caracteres ANSI, la constante *fPathUNICODE* sólo se utiliza para almacenar el resultado en una cadena de caracteres Unicode.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nombres largos
<a name="nombres_largos_ELTPARAGRAPHE000067"></a>

Vuelve **fShortName**:

- *El nombre corto del fichero o directorio* si &lt;File ruta&gt; se expresa en nombres largos (nombres que superan los 8 caracteres).

- *El nombre del archivo especificado* si el nombre del archivo en &lt;File ruta&gt; es:

	- expresado como nombre corto (nombre que contiene hasta 8 caracteres, conteniendo una tilde "~"),

	- de 8 caracteres o menos.




- *El nombre del directorio especificado* si el nombre del directorio en &lt;File ruta&gt; es:

	- expresado como nombre corto (nombre que contiene hasta 8 caracteres, conteniendo una tilde "~"),

	- de 8 caracteres o menos.

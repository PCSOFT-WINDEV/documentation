


## Funciones Notas reposicionables
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones pueden utilizarse para gestionar mediante programación los controles reposicionables de note: 



|   |   |
| --- | --- |
| [NotaCrear](../WDLang1/1000022395.md) | Crea una Nota Reposicionable control en la ventana de curernt mediante programación. |
| [NoteBorrar](../WDLang1/1000022893.md) | Elimina una note reposicionable que se encuentra en la ventana actual. |
| [NoteLoadAll](../WDLang1/1000022397.md) | Importa y muestra las notas reposicionables de la ventana actual guardadas previamente por [NoteSaveAll](../WDLang1/1000022396.md). |
| [NoteSaveAll](../WDLang1/1000022396.md) | Guarda las notas reposicionables de la ventana actual en un archivo. |




## StartShare( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Abre una ventana emergente que permite seleccionar la aplicación que se utilizará para compartir un Resource (texto o Document). La aplicación se abrirá con la Resource especificada. 

Por ejemplo, un Image puede ser compartido con la aplicación de envío de correos electrónicos. En este caso, se creará una nueva Email con la Image especificada en el anexo. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo



<a name="Example3"></a>

```wl
// Share a text displayed in an edit control
IF StartShare(shareText, EDT_Edit) = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = StartShare([<Type>, ] <Parameter 1> [, <Parameter 2> [... [, <Parameter N>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los recursos especificados pueden compartirse. 
	 Observaciones: 

	- Esta función no está bloqueando: el código que sigue a la llamada a esta función continuará ejecutándose tan pronto como aparezca la ventana emergente.

	- No es posible saber si una aplicación ha sido seleccionada por el usuario para realizar el compartimiento. 




- <u><u><u><u>False</u></u></u></u> si no se encuentra ninguna aplicación que permita compartir los recursos. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Type>: (Constante opcional de tipo Integer)`**

Tipo de Resource a compartir: 


|   |   |
| --- | --- |
| *shareDocument* <br>(Valor predeterminado) | Uno o más documentos (Image, video, pdf, ...) son compartidos. Estos documentos están identificados por su ruta completa, especificada como parámetro (&lt;Parámetro 1&gt; a &lt;Parámetro N>).<br><br>El Document debe estar ubicado en la memoria compartida del dispositivo (por ejemplo: la tarjeta SDCard) para ser utilizada por la aplicación externa seleccionada para la acción.<br> |
| *shareText* | El texto es compartido.<br><br>Si se especifican varias cadenas de caracteres como parámetros (&lt;Parámetro 1&gt; a &lt;Parámetro N&gt;), se concatenan y se separan mediante retornos de carro (CR).<br> |



**`<Parameter 1>: (Cadena de caracteres)`**

El primer Resource en compartir. Corresponde a: 

- un archivo ruta (si se utiliza la constante *shareDocument*). 

- una cadena de caracteres (si se usa la constante *shareText*). 




**`<Parameter 2>: (Cadena de caracteres opcional)`**

Segundo Resource para compartir. Corresponde a: 

- un archivo ruta (si se utiliza la constante *shareDocument*). 

- una cadena de caracteres (si se usa la constante *shareText*). 






**`<Parameter N>: (Cadena de caracteres opcional)`**

El último Resource para compartir. Corresponde a: 

- un archivo ruta (si se utiliza la constante *shareDocument*). 

- una cadena de caracteres (si se usa la constante *shareText*).

## Observaciones


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000105"></a>

- Para personalizar el título de la ventana popup de selección de la aplicación, utilice la función[NextTitle](../WDLang1/3038043.md) antes de la función **StartShare**. 

- Si una sola aplicación te permite compartir el Resource en el móvil, el popup no se muestra y la aplicación se inicia directamente.

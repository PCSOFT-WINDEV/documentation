
## docElementImage (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docElementImage** se utiliza para manipular las características de un elemento de Imagen que se encuentra en un [docElement](../WDLang1/1000022484.md) Variable. 

Las características de **docElementImage** Variable pueden ser manejadas por varias propiedades de WLanguage. 

**PRECAUCIÓN: Este tipo no se puede utilizar directamente. Debe utilizarse mediante el tipo de [docElement](../WDLang1/1000022484.md)**.

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Inserts an image cursor location
// Select the image file
sFile is string 
sFile = fImageSelect(fCurrentDir(), "", ...
	"Insert an image","JPEG (*.JPG)" + TAB + "*.jpg" + CR + ...
	"PNG (*.PNG)" + TAB + "*.png" + CR + "All files (*.*)" + TAB + "*.*")
	
IF sFile = "" THEN
	RETURN
END

// The image file is transferred into an Image variable
img is Image
img = sFile

// Declare an Image element (that will be inserted into the document)
elt is docElement
elt.Type = typeDocElementImage
elt.Image.BufferImage = img
elt.Image.Wrapping = docWrappingAlignedOnText


// Calculate the width and height of the image found in the element
// The image size is expressed in pixels in the Image variable. 
// This size is converted into mm in the element (unit used in a document)
elt.Image.Width = img.Width * 25.4 / 96.0
elt.Image.Height = img.Height * 25.4 / 96.0
// Image position
elt.Image.X = 0
elt.Image.Y = 0
	
DocInsert(WP_MyDocument, WP_MyDocument.Cursor, elt)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de las variables docElementImage
<a name="propiedades_especificas_descripcion_las_variables_docelementimage_ELTPARAGRAPHE000041"></a>

Las siguientes propiedades pueden ser utilizadas para manipular a **docElementImage** Variable LE:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| Ajuste | Constante de tipo Integer | Tipo de envoltura Image: <br><br>- *docWrappingAlignedOnText*: El Image insertado en el Line y es relacionado a la ubicación del texto donde se insertó. Se procesa como cualquier carácter. Se mueve con el texto. <br><br>- *docWrappingThrough*: Se utiliza para personalizar la envoltura. <br><br>- *docWrappingBehindTheText*: Le permite usar la imagen en Watermark detrás del texto.<br><br>- *docWrappingBeforeTheText*: Se utiliza para mostrar la imagen Above el texto. <br><br>- *docWrappingFramed*: Envuelve el texto alrededor del borde de Image (recomendado para un Image cuadrado).  <br><br>- *docWrappingTopAndBottom*: La imagen permanece en su Line y está ubicada entre dos secciones de texto en la parte superior e inferior.<br><br>- *docWrappingNear*: Envuelve el Image según sus contornos (recomendado para un Image redondo o para un Image de forma irregular). <br><br><br> |
| Altura | Real | La altura de Image en milímetros. <br><br>Atención: la unidad utilizada es el milímetro en lugar del píxel (no hay suficiente precisión). Si es necesario, convierta el alto de la imagen de pixeles a milímetros. |
| anchura | Real | Anchura de la Image en milímetros.<br><br>Atención: la unidad utilizada es el milímetro en lugar del píxel (no hay suficiente precisión). Si es necesario, convierta el ancho de la imagen de pixeles a milímetros. |
| ImageBuffer | [Image](../WDLang1/1000019650.md) | tampón de Image. |
| ImageLink | Cadena de caracteres | Enlace al expediente Image. El Image real no se integrará en el Document, sino que sólo será un enlace al Image. |
| X | Real | Posición X de la Image en relación con su punto de anclaje (en milímetros). <br><br>Esta propiedad se utiliza sólo para las imágenes para las que la propiedad **Wrapping** se establece en una constante distinta de *docWrappingAlignedOnText*. |
| Y | Real | Posición Y de la Image en relación con su punto de anclaje (en milímetros). <br><br>Esta propiedad se utiliza sólo para las imágenes para las que la propiedad **Wrapping** se establece en una constante distinta de *docWrappingAlignedOnText*. |
| ZOrder | Integro | Número correspondiente al orden Z de la Image. <br><br>Esta Property se modifica automáticamente cuando se utiliza [DocChangeZOrder](../WDLang1/1000023970.md). |

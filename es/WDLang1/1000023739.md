


## &lt;Hoja de cálculo&gt;.StyleSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica el estilo de las celdas seleccionadas en un control Hoja de cálculo encontrado en una ventana.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// CLICK on [BOLD]
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)


// Displays the B1 cell in bold
PSHEET_Spreadsheet.SelectPlus("B1")
PSHEET_Spreadsheet.StyleSelection(psheetBold, True)
```

```wl
sRes is boolean = PSHEET_Spreadsheet.StyleSelection(psheetBold)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estilo de selección

`<Result> = <Spreadsheet control>.StyleSelection(<Attribute>)`
---

**`<Result>: (Tipo de resultado esperado)`**

Atributo solicitado.

**`<Spreadsheet control>: (Nombre del control)`**

Nombre del control Hoja de cálculo a utilizar.

**`<Attribute>: (Constante de tipo Integer)`**

Atributo para averiguar: 


|   |   |
| --- | --- |
| *psheetAlignmentH* | Alineación horizontal. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a: <br><br>- *haCenter*: Alineación centrada. <br><br>- *haRight*: Alineación correcta. <br><br>- *haLeft*: Alineación a la izquierda.<br><br><br> |
| *psheetAlignmentV* | Alineación vertical. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a: <br><br>- *vaBottom*: Alineación en el fondo. <br><br>- *vaTop*: Alineación en la parte superior. <br><br>- *vaMiddle*: Alineación en el centro.<br><br><br> |
| *psheetAutomaticLineWrap* | Gestión de la envoltura de automatic Line. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a: <br><br>- <u><u><u><u>True</u></u></u></u> para utilizar el ajuste de líneas automático,<br><br>- <u><u><u><u>False</u></u></u></u> para no utilizar el ajuste de línea automático. <br><br><br> |
| *psheetBackgroundColor* | Antecedentes Color de texto. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br><br> |
| *psheetBold* | Carácter audaz. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a:<br><br>- <u><u><u><u>True</u></u></u></u> para poner el texto en negrita,<br><br>- <u><u><u><u>False</u></u></u></u> para no poner el texto en negrita.<br><br><br> |
| *psheetFontAngle* | Ángulo de rotación del texto. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a un entero incluido entre -180 y +180. |
| *psheetFontName* | Nombre de la fuente utilizada. <br><br>En este caso, &lt;Result&gt; puede corresponder a una cadena de caracteres que contiene el nombre de la fuente. |
| *psheetFontSize* | Tamaño fuente. <br><br>En este caso, &lt;Result&gt; corresponde al tamaño de la fuente (entero). |
| *psheetItalic* | Carácter itálico. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a:<br><br>- <u><u><u><u>True</u></u></u></u> para poner el texto en cursiva,<br><br>- <u><u><u><u>False</u></u></u></u> para no poner el texto en cursiva,<br><br><br> |
| *psheetStrikeOut* | Carácter Struck-out. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a:<br><br>- <u><u><u><u>True</u></u></u></u> para tachar,<br><br>- <u><u><u><u>False</u></u></u></u> para no tachar.<br><br><br> |
| *psheetTextColor* | Texto Color. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a: <br><br>- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),<br><br>- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),<br><br>- [un color predefinido de WLanguage](../WDLang5/3010002.md).<br><br><br> |
| *psheetUnderlined* | Carácter subrayado. <br><br>En este caso, &lt;Resultado&gt; puede corresponder a:<br><br>- <u><u><u><u>True</u></u></u></u> para subrayar,<br><br>- <u><u><u><u>False</u></u></u></u> para no subrayar.<br><br><br> |

## Observaciones


- La operación de modificación se añade a la lista de operaciones de Cancel: el usuario puede deshacer esta operación a través del menú contextual o pulsando Ctrl + Z.

- Esta función sólo se puede utilizar en un control Hoja de cálculo que se encuentre en una ventana..




## PDFReaderSeekPrevio (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija la posición sobre el resultado anterior de una búsqueda iniciada con [PDFReaderSeekFirst](../WDLang1/1000023409.md) en el contenido de un lector PDF control. 

Observación: [PDFReaderSeekNext](../WDLang1/1000023410.md) encuentra la siguiente instancia del texto buscado. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Seeks a text in a PDF Reader control and displays it
PDFReaderSeekFirst(PDF_MyPDFReader, "company")
// Finds the next element
IF PDFReaderSeekNext(PDF_MyPDFReader) = False THEN
	Info("Search ended")
END

// Finds the previous element
PDFReaderSeekPrevious(PDF_MyPDFReader)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PDFReaderSeekPrevious(<PDF Reader control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el texto especificado en [PDFReaderSeekFirst](../WDLang1/1000023409.md) se encontró en el actual Document en PDF Reader control ,

- <u><u><u><u>False</u></u></u></u> si no se encontró el texto. 




**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- El texto y las opciones de búsqueda se han especificado en [PDFReaderSeekFirst](../WDLang1/1000023409.md).


## France.CheckINSEE (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Prueba la validez de un número INSEE (número de identificación de personas físicas asignado por el Instituto Nacional de Estadística y Estudios Económicos).

ATENCIÓN: Esta información es específica de la administración francesa. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyINSEENumber is string = "285063410315510"
France.CheckINSEE(MyINSEENumber)		// Returns True

// Check a control
EDT_INSEENumber = "ABC"
France.CheckINSEE(EDT_INSEENumber)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = France.CheckINSEE(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número INSEE válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número INSEE a comprobar. Este parámetro puede corresponder a:

- el nombre de una variable de tipo String.

- el nombre de un control.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Un número INSEE se considera válido incluso sin la clave de control de 2 dígitos al final.

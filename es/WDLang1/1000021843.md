
## WinEdActive (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- \* averigua si el modo "Modificación de UI por el usuario final" está habilitado en una ventana (o ventana interna).

- habilitar (o deshabilitar) el modo "Modificación de UI por el usuario final" en una ventana (o ventana interna).



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enable the mode "UI modification by end user"
// Allow the end user to move all the controls except for the OK Button control
WIN_OverviewWindow.WinEditMode = wePosition
BTN_OK.WinEditAllowed = weNone
WInEdActive(WIN_OverviewWindow, True)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averigüe si el modo "Modificación de UI por el usuario final" está activado

`<Result> = WinEdActive(<Window>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el modo "Modificación de UI por el usuario final" está activado,

- <u><u><u><u>False</u></u></u></u> si el modo "Modificación de UI por el usuario final" está desactivado.  




**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. 


<a name="SYNTAX2"></a>

### Activar o desactivar el modo "Modificación de UI por el usuario final"

`WinEdActive(<Window> , <Enable>)`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. 

**`<Enable>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para habilitar el modo "Modificación de UI por el usuario final". Las autorizaciones de modificaciones deben ser definidas por [WinEditMode](../Proprietes/1000021824.md) y [WinEditPermitido](../Proprietes/1000021850.md). 

- <u><u><u><u>False</u></u></u></u> para desactivar el modo "Modificación de UI por el usuario final".


## UMCActive( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite conocer y modificar la activación de la gestión UMC (User Macro-Code).
<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar si la gestión del UMC está habilitada

`<Result> = UMCActive()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el UMC está activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.  





<a name="SYNTAX2"></a>

### Modificación del modo de gestión del UMC

`<Result> = UMCActive(<Enable>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el UMC está activado,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Enable>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para activar el UMC, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Modo de activación de UMC
<a name="modo_activacion_umc_ELTPARAGRAPHE000055"></a>

Los efectos de la función **UMCActive** dependen de la configuración del UMC en el proyecto y en las ventanas.


|   |   |   |   |   |   |   |
| --- | --- | --- | --- | --- | --- | --- |
|   | **UMC permitido en el proyecto** | **UMC no está permitido en el proyecto** |
| La función UMCActive no se utiliza | UMCActive(True) | UMCActive(lse) | La función UMCActive no se utiliza | UMCActive(True) | UMCActive(lse) |
| UMC permitido en la ventana | UMC posible | UMC posible | UMC no es posible | UMC no es posible | UMC posible | UMC no es posible |
| UMC no está permitido en la ventana | UMC no es posible | UMC no es posible | UMC no es posible | UMC no es posible | UMC no es posible | UMC no es posible |




## &lt;Variable WDPic&gt;.DrawCircle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Dibuja un círculo o una elipse en una variable de tipo [WDPic](../WDLang1/1000023503.md).![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyImage is WDPic = "Test.bmp"
// Draw a circle whose background is light yellow and 
// whose border is light blue in a WDPic variable
MyImage.DrawCircle(10, 10, 50, 50, LightYellow, LightBlue)
IMG_MyDrawing = MyImage
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<WDPic image>.DrawCircle(<X1> , <Y1> , <X2> , <Y2> [, <Background color> [, <Line color>]])`
---

**`<WDPic image>: (Variable de tipo WDPic)`**

Nombre de la variable de tipo [WDPic](../WDLang1/1000023503.md) que se utilizará. Solo se manipulará la capa de fondo.

**`<X1>: (Entero)`**

Coordenada X de la esquina superior izquierda del rectángulo que contiene el círculo. Estas coordenadas se expresan en pixeles.

**`<Y1>: (Entero)`**

Coordenada Y de la esquina superior izquierda del rectángulo que contiene el círculo. Estas coordenadas se expresan en pixeles.

**`<X2>: (Entero)`**

Coordenada X de la esquina inferior derecha del rectángulo que contiene el círculo. Estas coordenadas se expresan en pixeles.

**`<Y2>: (Entero)`**

Coordenada Y de la esquina inferior derecha del rectángulo que contiene el círculo. Estas coordenadas se expresan en pixeles.

**`<Background color>: (Entero o constante (opcional))`**

Color de fondo del círculo. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md), 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) una variable de tipo [Color](../WDLang1/1000019661.md).


Si no se especifica este parámetro, el color de fondo:

- es *Transparente* si no se ha utilizado la función [&lt;Variable WDPic&gt;.Background](../WDLang1/1000024481.md) previamente, 

- corresponde al color especificado en la última llamada a la función [&lt;Variable WDPic&gt;.Background](../WDLang1/1000024481.md).




**`<Line color>: (Entero o constante (opcional))`**

Color de la línea del círculo. Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md), 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) una variable de tipo [Color](../WDLang1/1000019661.md).


Si no se especifica este parámetro, el color de la línea:

- corresponde al color especificado en la última llamada a la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md),

- es el mismo que el color de fondo si no se ha utilizado la función [&lt;Variable WDPic&gt;.Pen](../WDLang1/1000024636.md).

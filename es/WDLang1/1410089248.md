
## Germany.CheckUStIdNr (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba la validez de un número USt-IdNr (número de identificación de empresa en Alemania).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Check a variable
MyUStIdNr is string = "DE262843185"
Germany.CheckUStIdNr(MyUStIdNr)	// Returns True

// Check a control
EDT_UStIdNr = "ABC"
Germany.CheckUStIdNr(EDT_UStIdNr)	// Returns False
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Germany.CheckUStIdNr(<Number to check>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si el elemento especificado contiene un número USt-IdNr válido.

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Number to check>: (Nombre de control o variable de tipo String)`**

Número USt-IdNr a comprobar. Este parámetro puede corresponder a: 

- el nombre de una variable de tipo String.

- el nombre de un control.

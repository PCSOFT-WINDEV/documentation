


## APIConfigure( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica las opciones de llamada de [API](../WDLang1/3014005.md) y [CallDLL32](../WDLang1/3014003.md).

Predeterminada:

- [API](../WDLang1/3014005.md) y [CallDLL32](../WDLang1/3014003.md) no bloquean los otros hilos.

- Si la función API llamada modifica los ajustes regionales del sistema, se restauran los ajustes regionales anteriores.




La función **APIConfigure** permite configurar el comportamiento predeterminado de estas funciones.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Lock the threads
APIConfigure(apiLockThreads, True)

// Don't restore the regional settings
APIConfigure(apiRestoreSystemSettings, False)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Averiguar el valor de una opción para API y CallDLL32

`<Result> = APIConfigure(<Option>)`
---

**`<Result>: (booleano)`**

Valor de la opción solicitada: 

- <u><u><u><u>True</u></u></u></u> si la opción solicitada está activada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Option>: (Constante de tipo Integer)`**

Parámetro cuyo valor se solicita:


|   |   |
| --- | --- |
| *apiLockThreads* | Sirve para saber si [API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md) bloquea los hilos durante su ejecución. <br>Esta opción está desactivada por Default. |
| *apiRestoreSystemSettings* | Permite saber si [API](../WDLang1/3014005.md) o [CallDLL32](../WDLang1/3014003.md) restablece la configuración regional del sistema (idioma, visualización de decimales, ...) tras la ejecución del API de Windows . <br>Esta opción está habilitada por Default. |




<a name="SYNTAX2"></a>

### Modificando el valor de una opción para API y CallDLL32

`<Result> = APIConfigure(<Option> , <Value>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se había activado la opción solicitada antes de utilizar la función **APIConfigure**,

- <u><u><u><u>False</u></u></u></u> si no se había activado la opción solicitada antes de utilizar la función **APIConfigure**.




**`<Option>: (Constante de tipo Integer)`**

Parámetro cuyo valor debe ser modificado:


|   |   |
| --- | --- |
| *apiLockThreads* | Si el parámetro &lt;Valor&gt; se establece en <u><u><u><u>True</u></u></u></u>, los subprocesos se bloquean durante la ejecución de las funciones [API](../WDLang1/3014005.md) y [CallDLL32](../WDLang1/3014003.md).. <br>Si el parámetro &lt;Valor&gt; se establece en <u><u><u><u>False</u></u></u></u>, los subprocesos no se bloquean durante la ejecución de las funciones [API](../WDLang1/3014005.md) y [CallDLL32](../WDLang1/3014003.md). <br>Esta opción está desactivada por Default. |
| *apiRestoreSystemSettings* | Si el parámetro &lt;Valor&gt; se establece en <u><u><u><u>True</u></u></u></u>, los parámetros regionales del sistema (idioma, decimales, etc.) se restauran después de ejecutar la API de Windows. <br>Si el parámetro &lt;Valor&gt; se establece en <u><u><u><u>False</u></u></u></u>, los parámetros regionales del sistema (idioma, decimales, etc.) no se restauran después de ejecutar la API de Windows. En este caso, las modificaciones se almacenarán si estos parámetros son modificados por el sistema operativo Windows API. <br>Esta opción está habilitada por Default. |



**`<Value>: (booleano)`**

Valor de la opción solicitada: 

- <u><u><u><u>True</u></u></u></u> si la opción solicitada debe activarse,

- <u><u><u><u>False</u></u></u></u> en caso contrario.

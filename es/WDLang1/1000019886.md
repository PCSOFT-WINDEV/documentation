
## WinAdaptHeight (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Adapta la altura de la ventana al contenido de los controles. Los controles se muestran de forma óptima en la ventana (las barras de desplazamiento se borran si es posible).. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays the TABLE_Price control
TableDisplay(WIN_Price.TABLE_Price)
// Adapts the windows height
WinAdaptHeight(WIN_Price)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinAdaptHeight([<Window>])`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana a redimensionar. La ventana actual se utilizará si no se especifica este parámetro. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Una ventana puede redimensionarse en altura sólo si la ventana contiene uno de los siguientes controles anclados en altura: 

- Control List Box.

- Control Tabla.

- Control Looper.

- Control TreeView.

- Control ListView

- Control Campo de entrada multilínea.

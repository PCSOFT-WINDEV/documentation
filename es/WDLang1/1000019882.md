
## FirstDayOfWeek (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del primer día de la semana (es decir, el lunes). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Click on "BTN_beginning_of_week"
EDT_START_DATE = FirstDayOfWeek(EDT_START_DATE)
```
<a name="Example2"></a>

```wl
NbWorkedDays = DateDifference(EDT_Date, FirstDayOfWeek(EDT_Date), EDT_Date)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = FirstDayOfWeek(<Date>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Date)`**

Fecha que corresponde al primer día de la semana especificada (en formato AAAAMMDD). 

**`<Date>: (Cadena de caracteres o variable de tipo Date)`**

Fecha para la que debe calcularse el primer día de la semana. Esta fecha está en formato AAAAMMDD.

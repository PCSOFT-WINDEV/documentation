
## ThreadStopRequested (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Verifica si se ha enviado una solicitud al hilo que se está ejecutando. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// If a stop request was sent to the thread
IF ThreadStopRequested() THEN
	// Free thread resources
	...
	// End thread
	ThreadEnd()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = ThreadStopRequested()`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se envió una solicitud al hilo que se está ejecutando para que se detenga,

- <u><u><u><u>False</u></u></u></u> en caso contrario. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Un hilo solicita que otro hilo se detenga mediante la función [ThreadRequestStop](../WDLang1/1000021218.md).

- El código del hilo actual debe verificar de forma regular (bucles, ... ) si se ha solicitado que se detenga un hilo y finalizar correctamente, si es necesario.

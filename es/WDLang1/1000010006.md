


## TableDeleteChild (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra todas las líneas secundarias de una rama de la jerarquía.. La propia rama no se borra.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Deletes all child rows of "Desserts" element
TableDeleteChild(TVT_Recipe, "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Borrar las filas hijas de una rama especificando su índice

`TableDeleteChild(<TreeView Table control> , <Index of the parent element>)`
---

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular. 
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Index of the parent element>: (Integro)`**

Índice de la fila cuyas filas hijas serán eliminadas.  


<a name="SYNTAX2"></a>

### Borrar las filas de niños de una rama especificando su nombre

`TableDeleteChild(<TreeView Table control> , <Name of parent element>)`
---

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular. 
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Name of parent element>: (Cadena de caracteres)`**

Contenido de la primera columna del elemento cuyas filas inferiores deben ser borradas.

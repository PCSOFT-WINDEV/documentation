


## &lt;Tablero de control&gt;Configuración inicial (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica la configuración de una control Cuadro de mando (definida en el editor) antes de su primera visualización.. Esto permite configurar y tener en cuenta los widgets declarados en la programación por [&lt;Panel de control&gt;.AddWidget](../WDLang1/1000022502.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// -- Initialize DASH_Dashboard
// Adds information into the initial configuration:
// - key numbers for the last 3 months
nX = 1 ; nY = 3
FOR i = 1 TO 3
	// Adds the widget
	nIndex = DASH_Dashboard.AddWidget(IW_Widget_KeyNumber, ...
			StringBuild("Key numbers for %1", DateToString(dDate, "Mmm YYYY")), ...
			dDate)
	// Configures the widget (placed on the left)
	DASH_Dashboard.InitialConfiguration(nIndex, nX, nY)
	// The next one will be lower
	nY++
	dDate.Month--
END
// - stock alert
// Adds the widget
nIndex = DASH_Dashboard.AddWidget(IW_Widget_StockAlert, "Stock alert")
// Configures the widget (placed on the left) 
DASH_Dashboard.InitialConfiguration(nIndex, 5, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Configuración inicial: manejar el widget por su índice

`<Dashboard control>.InitialConfiguration(<Widget index> [, <X> [, <Y> [, <Width> [, <Height>]]]])`
---

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Widget index>: (Integro)`**

Índice del widget a utilizar. Este índice corresponde a: 

- el número de adición del widget definido en el editor de ventanas o en el editor de páginas. 

- el número de adición del widget devuelto por [&lt;Panel de control&gt;.AddWidget](../WDLang1/1000022502.md).




**`<X>: (Entero opcional)`**

Posición horizontal del widget en el número de celdas.

Si no se especifica [&lt;X&gt;, &lt;Y&gt;] o si su valor es igual a [ 0,0], el widget se posiciona en la primera ubicación libre. 

La primera celda es la celda [ 1, 1].

**`<Y>: (Entero opcional)`**

Posición vertical del widget en el número de celdas.

Si no se especifica [&lt;X&gt;, &lt;Y&gt;] o si su valor es igual a [ 0,0], el widget se posiciona en la primera ubicación libre. 

La primera celda es la celda [ 1, 1].

**`<Width>: (Entero opcional)`**

Ancho del widget en número de celdas.

Si no se especifica [&lt;anchura&gt;, &lt;altura&gt;] o si su valor es igual a [ 0, 0]: 

- durante la primera llamada a **&lt;Tablero de control&gt;Configuración inicial**, el tamaño del widget corresponde al tamaño de la ventana o página interna de origen (al menos 1 celda).

- al llamar a la función **&lt;Tablero de control&gt;Configuración inicial** las próximas veces, el tamaño del Widget corresponde al tamaño definido durante la primera llamada.




**`<Height>: (Entero opcional)`**

Altura del widget en el número de celdas.

Si no se especifica [&lt;anchura&gt;, &lt;altura&gt;] o si su valor es igual a [ 0, 0]: 

- durante la primera llamada a **&lt;Tablero de control&gt;Configuración inicial**, el tamaño del widget corresponde al tamaño de la ventana o página interna de origen (al menos 1 celda).

- al llamar a la función **&lt;Tablero de control&gt;Configuración inicial** las próximas veces, el tamaño del Widget corresponde al tamaño definido durante la primera llamada.

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se llama a la función **&lt;Tablero de control&gt;Configuración inicial** después de inicializar el control Cuadro de mando, se produce un error.

- Los Widgets configurados con la función **&lt;Tablero de control&gt;Configuración inicial** se cargarán y se harán visibles después de la ejecución de los procesos de inicialización.

- La función **&lt;Tablero de control&gt;Configuración inicial** no se tiene en cuenta en los siguientes casos: 

	- La opción "Recordar configuración de Widgets" está marcada en la ventana control Cuadro de mando description (pestaña "Detalles") y el usuario hace el widget Invisible. 

	- El usuario hizo el widget Invisible y la configuración de la control Cuadro de mando fue guardada y luego recargada.

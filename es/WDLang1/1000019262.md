


## repetition (Tipo de Variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **repetition** se utiliza para describir y modificar las características de un repetition para un Appointment a través de las diferentes propiedades de WLanguage. Entonces, la Appointment se mostrará en una control Agenda o en una control Planificador.

Este tipo de Variable es utilizado por el tipo [Appointment](../WDLang1/1000019244.md).

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Defines an appointment for the morning
MyAppointment is Appointment
MyAppointment.StartDate = DateSys() + "090000"
MyAppointment.EndDate = DateSys() + "093000"
MyAppointment.Title = "Appointment with the development team"
MyAppointment.Guest = "Jack, Mark, Sandra"
MyAppointment.Content = "Daily meeting regarding the progress of the projects."
MyAppointment.WithRepetition = True

// Configures a daily repetition of the appointment
RepetitionParameter is Repetition
RepetitionParameter.Type = schEveryDay
RepetitionParameter.TypeEnd = schNoEnd
MyAppointment.Repetition = ConfigureRepetition

OrganizerAddAppointment(ORG_Organizer1, MyAppointment)
```

## Observaciones


### Propiedades específicas de la description de las variables Repetition
<a name="propiedades_especificas_description_las_variables_repetition_ELTPARAGRAPHE000045"></a>

Las siguientes propiedades pueden usarse para manipular una variable **Repetition**: 

| **Nombre de la propiedad** | **Tipo utilizado** | **Efecto** |
| --- | --- | --- |
| DayOfMonth | Cadena de caracteres | Número del día para el que está programado el Appointment (por ejemplo, "31"). Este número se incluye entre 0 y 31.<br>Este Property también puede corresponder:<br><br>- una lista de números separados por comas para especificar varios meses.<br><br>- a interval. Por ejemplo,  "9-13" (la cita se repite del 9 al 13 del mes).<br><br>- una estrella ("\*" ") para especificar el rango completo.<br><br>- un interval con un paso. Por ejemplo, "\*/2" para repetir la cita todos los días pares del mes.<br><br><br><br><br> |
| DayOfMonthOrDayOfWeek | booleano | <br><br>- <u><u><u><u>True</u></u></u></u> para repetir la cita todos los lunes Y el primer día del mes (condiciones acumuladas).<br><br>- <u><u><u><u>False</u></u></u></u> para repetir la cita solo el lunes 1 del mes (condiciones exclusivas).<br><br><br><br><br> |
| DayOfWeek | Cadena de caracteres | Número del día para el que está programado el Appointment (por ejemplo, "2"). Este número se incluye entre 1 y 7 (1 = Lunes, 2 = Martes, ....).<br>Este Property también puede corresponder:<br><br>- una lista de números separados por comas para especificar varios días.<br><br>- a interval. Por ejemplo, "1-3" (el Appointment se repite el lunes, martes y miércoles).<br><br>- una estrella ("\*" ") para especificar el rango completo.<br><br>- un interval con un paso. Por ejemplo, "\*/2" para repetir la cita todos los días pares de la semana.<br><br>- el día de la semana en letras ("lunes", "martes", "miércoles", "jueves", "viernes").<br><br><br><br><br> |
| EndDate | Cadena de caracteres | Fecha de finalización de la repetition. Esta propiedad se utiliza si la propiedad **TypeEnd** es igual a la constante *schEndDate*. |
| escribir a máquina | Constante de tipo Integer | Tipo de repetition. <br>Esta Property corresponde a una de las siguientes constantes:<br><br>- *schEveryDay*: el Appointment se repetirá todos los días a la misma hora.<br><br>- *schEveryWeek*: la Appointment se repetirá cada semana, el mismo día y a la misma hora.<br><br>- *schEveryMonth*: la Appointment se repetirá cada mes, el mismo día y a la misma hora.<br><br>- *schEveryYear*: el Appointment se repetirá cada año, el mismo día y a la misma hora.<br><br><br> |
| Hora | Cadena de caracteres | Hora de la repetition en formato UTC (hora universal). Los valores posibles son:<br><br>- un dígito comprendido entre 0 y 23.<br><br>- una lista de números separados por comas para especificar varias horas.<br><br>- a interval. Por ejemplo, "9-13".<br><br>- una estrella ("\*" ") para especificar el rango completo.<br><br>- un interval con un paso. por ejemplo, "\*/2" para repetir la cita cada hora par.<br><br><br> |
| Mes | Cadena de caracteres | Número del mes para la cita repetition. Los valores posibles son:<br><br>- un dígito comprendido entre 1 y 12.<br><br>- una lista de números separados por comas para especificar varios meses.<br><br>- a interval. Por ejemplo, "9-11".<br><br>- una estrella ("\*" ") para especificar el rango completo.<br><br>- un interval con un paso. Por ejemplo, "\*/2" para repetir la cita cada mes par.<br><br><br> |
| Minuto | Cadena de caracteres | Minutos del tiempo para la Appointment Repetition. Los valores posibles son:<br><br>- un dígito comprendido entre 0 y 59.<br><br>- una lista de números separados por comas para especificar varios minutos.<br><br>- a interval. Por ejemplo, "9-13".<br><br>- una estrella ("\*" ") para especificar el rango completo.<br><br>- un interval con un paso. Por ejemplo, "\*/2" para repetir la cita cada minuto par.<br><br><br> |
| NbOccurrence | Integro | Número máximo de repeticiones. Esta propiedad se tiene en cuenta cuando la propiedad **TypeEnd** es igual a la constante *schMaxOccurrence*.<br><br> |
| TypeEnd | Constante de tipo Integer | Condición para detener el Repetition. <br>Esta Property corresponde a una de las siguientes constantes:<br><br>- *schEndDate*: la Appointment ya no se repetirá a partir de la fecha especificada en la **EndDate**Property .<br><br>- *schMaxOccurrence*: la Appointment ya no se repetirá después del número de repeticiones especificado en la **NbOccurrence**Property. <br>	<br><br>- *schNoEnd*: la Appointment se repetirá sin cesar.<br><br><br> |

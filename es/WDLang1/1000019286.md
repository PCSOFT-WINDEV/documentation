


## SchedulerSelectMinus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deselecciona un Appointment de un control Planificador.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add an appointment into a Scheduler control
nAppointmentIndex is int
nAppointmentIndex = SchedulerAddAppointment(SCH_Schedule, "Vince", ...
				"Meeting with sales team", ...
				DateSys() + "1400", DateSys() + "1600")
// Select the added appointment
SchedulerSelectPlus(SCH_Schedule, nAppointmentIndex)
...
// Click the "Deselect" button
SchedulerSelectMinus(SCH_Schedule)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Desmarcar una Appointment identificada por su índice

`SchedulerSelectMinus(<Scheduler control> [, <Appointment index>])`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<Appointment index>: (Entero opcional)`**

Índice del Appointment para deseleccionar. Los índices de Appointment comienzan a partir de 1.
Si no se especifica este parámetro, la Appointment que está actualmente seleccionada se deselecciona.
Si se especifica este parámetro y si no corresponde a la índice del Appointment actualmente seleccionado en el programa, la función no tiene ningún efecto.


<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
### Deselección de una Appointment identificada por una Appointment Variable

`SchedulerSelectMinus(<Scheduler control> , <Appointment used>)`
---

**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<Appointment used>: (Appointment Variable)`**

Variable [Appointment](../WDLang1/1000019244.md) correspondiente al Appointment a deseleccionar.




## DashLoadConfiguration (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la configuración de una control Cuadro de mando. Esta configuración incluye los siguientes elementos: 

- posición de las ventanas o páginas internas en el control Cuadro de mando.

- tamaño de las ventanas o páginas internas de la control Cuadro de mando.

- visibilidad de las ventanas o páginas internas de la control Cuadro de mando.




Observación: Para guardar o restaurar automáticamente la configuración del usuario, marque "Recordar configuración de Widgets" en la pestaña "Detalles" de la ventana de control de description.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Asks the user for the configuration that will be loaded
sFile is string

// Opens the file picker
sFile = fSelect("", "", "Select a configuration file...",...
			"Dashboard configuration" + TAB + "*.dashconf", "*.dashconf")
// If a file is selected
IF sFile <> "" THEN
	// Retrieves the configuration found in the file
	sConfiguration is string = fLoadText(sFile)
	// Loads the configuration
	IF DashLoadConfiguration(DASH_Dashboard, sConfiguration) THEN
		ToastDisplay("The configuration was loaded", toastShort, vaMiddle, haCenter)
	ELSE
		Error("Error while loading the configuration", ErrorInfo())
	END
END
```

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = DashLoadConfiguration(<Dashboard control> , <Configuration>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se cargó la configuración.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Configuration>: (Cadena de caracteres)`**

Configuración para restaurar. Esta configuración es el resultado de [DashSaveConfiguration](../WDLang1/1000020960.md). 

Atención: Si el formato de este parámetro no es válido o no se puede aplicar al control Cuadro de mando (porque las ventanas/páginas internas o el número de columnas no se corresponden, por ejemplo), el valor de &lt;Resultado&gt; es <u><u><u><u>False</u></u></u></u>. 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Los widgets declararon que Visible desde la última copia de seguridad debe ser Visible al cargar la configuración.

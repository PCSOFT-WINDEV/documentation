
## FinDecreasingRedemption( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el valor de amortización de un bien durante un período de tiempo específico, mediante la fórmula de amortización decreciente de tipo fijo.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Amortization value of a good bought for 25000 Euros
// whose residual value will be 15000 Euros after 6 years.
ResAmortization = FinDecreasingRedemption(25000, 15000, 6)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinDecreasingRedemption(<Acquisition cost> , <Residual value> , <Duration> [, <Period> [, <Number of months year 1>]])`
---

**`<Result>: (Real o moneda)`**

Valor de la amortización para el &lt;Período&gt; dado o 0 si la amortización es nula.

**`<Acquisition cost>: (Real o moneda)`**

Costo de adquisición del bien. Este costo no puede ser nulo.

**`<Residual value>: (Real o moneda)`**

Valor residual del bien después de la amortización.

**`<Duration>: (Real)`**

Número de períodos durante los cuales se reembolsa el bien. Esta duración no puede ser nula.

**`<Period>: (Entero opcional)`**

Período sobre el que debe calcularse la amortización (1 de Default).

**`<Number of months year 1>: (Entero opcional)`**

Número de meses en el primer año de amortización (1 por Default). Este número debe incluirse entre 0 y 12.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Amortización a tipo fijo
<a name="amortizacion_tipo_fijo_ELTPARAGRAPHE000077"></a>

El método de amortización a tipo fijo consiste en calcular la amortización de un bien mediante un tipo fijo durante el período de amortización.

Esta tasa se aplica al valor actual del bien en cada período.. El valor actual del bien corresponde a:


```txt
<Acquisition cost> - <Amortization total>
```


donde &lt;Amortización total&gt; representa la suma de la amortización de los períodos anteriores.

Este tipo se calcula en función del valor residual, del coste de adquisición del bien y de la duración de la amortización.

Se debe realizar un cálculo específico con respecto al valor de amortización para el primer año y para el último año:

- para el primer año, sólo se tiene en cuenta el número de meses amortizados (&lt;Número de meses para el año 1&gt;);

- para el último año, el número de meses amortizados corresponde a:
	
	```txt
	12 - <Number of months for year 1>
	```





**Observación**: Si el número de meses del primer año es igual a 12, este año se incluye en la duración.

### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000102"></a>

**Atención**: La función **FinDecreasingRedemption** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

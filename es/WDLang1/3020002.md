


## Funciones de la barra de herramientas control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles de la barra de herramientas:



|   |   |
| --- | --- |
| [Posición de la barra de herramientas](../WDLang1/3020004.md) | Indica el modo de visualización de una barra de herramientas control. |
| [ToolbarDock](../WDLang1/3020003.md) | Acopla una barra de herramientas control a una ventana o cambia la posición de anclaje de una barra de herramientas acoplada control. |
| [ToolbarUndock](../WDLang1/3020001.md) | Transforms a docked Toolbar control into a floating Toolbar control. |

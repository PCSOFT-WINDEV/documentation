
## docTableOfContents (Tipo de variable)

*** ***
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **docTableOfContents** permite definir las opciones para generar un tabla de contenido en un documento de tipo Procesador de texto. Las características de la tabla de contenido se pueden definir y cambiar mediante diferentes propiedades WLanguage. 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
doc is Document
...
// Define the table of contents
toc is docTableOfContents
toc.WithPageNumber = True

// Paragraphs that use the default style for level 1 (Heading 1) 
// are used for level 1 entries in the table of contents
toc.StylesPerLevel[1] = styleTOCDefault1

// Paragraphs that use the default style for level 2 (Heading 2) 
// and the paragraphs that use "MyCustomStyle"
// are used for level 2 entries in the table of contents
toc.StylesPerLevel[2] = styleTOCDefault2 + CR + "MyCustomStyle"

// Paragraphs that use "MyCustomStyle3"
// are used for level 3 entries in the table of contents
toc.StylesPerLevel[3] = "MyCustomStyle3"

// Insert table of contents at the beginning of the document
fDocBeginning is docFragment = doc[1 ON 0]
DocInsert(fDocBeginning, toc)
```

## Observaciones
<a name="NOTE0_1"></a>


### Propiedades específicas de la descripción de variables de tipo docTableOfContents
<a name="propiedades_especificas_descripcion_variables_tipo_doctableofcontents_ELTPARAGRAPHE000029"></a>

Las siguientes propiedades se pueden utilizar para manipular una tabla de contenido:


| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| StylesPerLevel | Array of strings | Permite especificar el o los estilos que se utilizarán al generar los niveles de la tabla de contenido.<br><br>Si se utilizan varios estilos, deben estar separados por retornos de carro (CR). <br><br>Es posible utilizar: <br><br>- uno o más estilos personalizados,<br><br>- las constantes que corresponden a los estilos predeterminados: <br><br>	- *styleTOCDefault1*: Estilo de párrafo predeterminado para el nivel 1 de la tabla de contenido (Título 1). <br><br>	- *styleTOCDefault2*: Estilo de párrafo predeterminado para el nivel 2 de la tabla de contenido (Título 2). <br><br>	- *styleTOCDefault3*: Estilo de párrafo predeterminado para el nivel 3 de la tabla de contenido (Título 3). <br><br>	- *styleTOCDefault4*: Estilo de párrafo predeterminado para el nivel 4 de la tabla de contenido (Título 4). <br><br>	- *styleTOCDefault5*: Estilo de párrafo predeterminado para el nivel 5 de la tabla de contenido (Título 5). <br><br>	- *styleTOCDefault6*: Estilo de párrafo predeterminado para el nivel 6 de la tabla de contenido (Título 6). <br><br>	- *styleTOCDefault7*: Estilo de párrafo predeterminado para el nivel 7 de la tabla de contenido (Título 7). <br><br>	- *styleTOCDefault8*: Estilo de párrafo predeterminado para el nivel 8 de la tabla de contenido (Título 8). <br><br>	- *styleTOCDefault9*: Estilo de párrafo predeterminado para el nivel 9 de la tabla de contenido (Título 9). <br><br><br><br><br><br> |
| WithPageNumber | Boolean | <br><br>- <u><u><u><u>True</u></u></u></u> si se debe mostrar el número de página en cada línea de la tabla de contenido, <br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario. <br><br><br> |


<a name="NOTE0_2"></a>


### Varios
<a name="varios_ELTPARAGRAPHE000080"></a>

La función [DocInsert](../WDLang1/1000022732.md) permite agregar una tabla de contenido al documento.


## gPen( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Construye una secuencia de caracteres para describir el color del texto y/o del contorno de una figura (rectángulo o círculo).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **En una ventana**, esta secuencia de caracteres fija el texto y/o el contorno Color:  

- en un elemento de un List Box o control Combo Box.

- en la barra de título y/o barra de estado de una ventana. 

- en una control Estático. 
	Observación: Puede ser necesario ajustar el texto una vez definido el contenido.





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Add "Yellow" colored in yellow in "LIST_ListColors"
ListAdd(LIST_ListColors, gPen(LightYellow) + "Yellow")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = gPen(<Color>)`
---

**`<Result>: (Cadena de caracteres)`**

Secuencia de caracteres que definen el color del texto o del contorno de una figura (círculo, rectángulo, etc.).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Esta secuencia de caracteres puede ser: 

- utilizado en [ListAdd](../WDLang1/3049004.md), [ListInsert](../WDLang1/3049002.md) o [ListModify](../WDLang1/3049009.md).

- asignado a un control Estático. 

- usado en la barra de título o barra de estado de una ventana. 




**`<Color>: (Entero o constante)`**

Mostrar color para el texto o para el contorno de una forma (rectángulo o círculo). Este color puede corresponder a:

- un color RGB (devuelto por la función [RGB](../WDLang1/3029012.md)),

- un color HSL (devuelto por la función [HSL](../WDLang1/3029057.md)),

- [un color predefinido de WLanguage](../WDLang5/3010002.md).

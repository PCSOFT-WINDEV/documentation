


## Yesterday (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la fecha del día anterior al día actual o una fecha determinada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays yesterday's date
Info("Yesterday was " + DateToString(Yesterday()))
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = Yesterday([<Date>])`
---

**`<Result>: (Cadena de caracteres)`**

Fecha del día anterior a &lt;Fecha&gt; en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día).

**`<Date>: (Cadena de caracteres, variable de tipo Date o DateTime opcional)`**

Fecha a manipular. Este parámetro puede corresponder a: 

- una cadena en formato AAAAMMDD (AAAA corresponde al año, MM al mes y DD al día), 

- una variable de tipo Date o DateTime. 


Si no se especifica este parámetro, la fecha manipulada corresponde a la fecha actual.

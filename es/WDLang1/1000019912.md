
## dCopyWindowImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Copia la imagen de una ventana WINDEV en una variable de tipo [Image](../WDLang1/1000019650.md) o en un control Imagen.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Copies the image of the current window into an image variable
MyImage is Image = dCopyWindowImage("")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = dCopyWindowImage(<Window> [, <Shadow management>])`
---

**`<Result>: (Variable de tipo Image o Buffer binario)`**



- Nombre de la variable de tipo [Image](../WDLang1/1000019650.md) que contendrá la captura de pantalla.

- Buffer binario que corresponde a la imagen. Este buffer puede asignarse directamente a un control Imagen. 




**`<Window>: (Cadena de caracteres)`**

Nombre de la ventana de WINDEV o WINDEV Mobile cuya imagen será copiada. Si este parámetro corresponde a una cadena vacía (""), se copiará la ventana actual.. 

**`<Shadow management>: (Constante de tipo Integer)`**

Modo de gestión de la sombra del borde de la ventana (especialmente en Windows 10): 



|   |   |
| --- | --- |
| *dWithShadow*<br>(Valor predeterminado) | La sombra de la ventana se conserva al copiar la ventana. |
| *dNoShadow* | La sombra de la ventana no se conserva al copiar la ventana. |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La ventana a copiar debe mostrarse en la pantalla. 

- Si la ventana está recortada, se copia toda el área que corresponde a la ventana: por lo tanto, los elementos que aparecen en el fondo de la ventana recortada también se copian. 

- Atención: Si la opción "Permitir captura de pantalla de la ventana" (pestaña "Detalles" de la ventana de descripción) no está seleccionada, la imagen de la ventana aparecerá completamente oscurecida. Esta opción puede modificarse en el código mediante la propiedad [ScreenshotAllowed](../Proprietes/1000025336.md).

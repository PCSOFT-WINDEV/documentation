
## CameraListParameter( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Enumera los valores admitidos por la cámara del dispositivo para un parámetro determinado.

**Observaciones**: 

- Para modificar o conocer el valor actual de un parámetro de la cámara del aparato, utilice [CameraParameter](../WDLang1/1000022459.md).

- La función **CameraListParameter** puede utilizarse en el emulador de Android, pero no en el simulador.

- Esta función es equivalente a la función [VideoListParameter](../WDLang1/1000019695.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Use the maximum resolution to take a photo
sListResolution is string 
sListResolution = CameraListParameter(vipPhotoResolution)
IF ErrorOccurred = False THEN
	CameraParameter(vipPhotoResolution, ExtractString(sListResolution, lastRank, CR))
ELSE
	Error("Error while retrieving the resolutions supported by the camera.")
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = CameraListParameter(<Parameter>)`
---

**`<Result>: (Cadena de caracteres)`**



- Lista de valores soportados por la cámara para el &lt;Parámetro&gt; especificado. Esta lista tiene el siguiente formato: 
	
	```txt
	<Value 1>+ CR + <Value 2> + CR + ... + <Value N>
	```


- Vaciar la cadena (""") si el parámetro no es compatible con la cámara o si se ha producido un error durante la recuperación de los parámetros..
	Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se pone en <u><u><u><u>True</u></u></u></u>. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Parameter>: (Constante de la cadena de caracteres o cadena de caracteres)`**

Parámetro para el que deben recuperarse los valores soportados. Este parámetro puede corresponder a: 

- una de las constantes preestablecidas. Estas constantes corresponden a los parámetros más comunes. 

- una Cadena correspondiente al nombre del parámetro deseado. 
	**Atención**: el nombre del parámetro distingue entre mayúsculas y minúsculas y acentos. Vea la documentación sobre el SDK de Android o sobre el dispositivo para saber el nombre de los parámetros adicionales soportados. Si el nombre especificado no corresponde a un parámetro soportado por la cámara, la función devolverá una cadena vacía.




Las constantes disponibles son las siguientes: 



|   |   |
| --- | --- |
| *vipBalance* | Modos de balance para los blancos soportados por la cámara: automatic, luz del día, nublado, ...<br> |
| *vipCamera* | Subíndices y tipos de cámaras disponibles en el dispositivo. La lista tiene el siguiente formato: <br><br><pre><code><Subscript><TAB><Type></code></pre><br>donde:<br><br>- &lt;Índice&gt; es un número entero incluido entre 1 y el número de cámaras encontradas en el dispositivo, <br><br>- &lt;Type&gt; es una de las siguientes constantes:<br><br>	- *vipCameraBack*: Cámara trasera opuesta a la pantalla.<br><br>	- *vipCameraFront*: Cámara frontal de cara a la pantalla.<br><br><br><br><br><br> |
| *vipEffect* | Efectos de Color: negro y White, cartel, sepia, ...<br> |
| *vipFlash* | Modos de flash compatibles con la cámara: automatic, sin, ojos anti rojos, ... |
| *vipFlicker* | Ajustes de parpadeo (anti-banding) soportados por la cámara: automatic, ninguno, 50Hz, 60Hz, ...<br> |
| *vipFocus* | Modos de enfoque compatibles con la cámara: automatic, macro, fijo, ...<br> |
| *vipISO* | Sensibilidades ISO soportadas por la cámara: automatic , 100, 200, 400, ...<br> |
| *vipPhotoResolution* | Resoluciones soportadas por la cámara para tomar fotos. <br>La lista está ordenada en orden ascendente, desde la resolución más baja hasta la más alta..<br>Los valores devueltos tienen el siguiente formato: <br><br><pre><code><Width><TAB><Height></code></pre><br>Ejemplo: <br><br><pre><code>640<TAB>480<CR>1024<TAB>760<CR>2048<TAB>1240</code></pre><br> |
| *vipScene* | Tipos de escenas soportadas por la cámara: automatic, noche, deportes, ...<br> |
| *vipZoom* | Valores de zoom soportados por la cámara.<br>Los valores devueltos tienen el siguiente formato:<br><br>- 100 (1x zoom = no zoom)<br><br>- 200 (zoom 2x)<br><br>- 300 (zoom 3x)<br><br>- 350 (zoom de 3,5 aumentos)<br><br>- ...<br><br><br>El primer valor es siempre 100 y el último corresponde al zoom máximo.. <br>Ejemplo: <br><br><pre><code>100<CR>200<CR>300<CR>350</code></pre><br>**Observación**: La función devolverá una cadena vacía (""") si la cámara del dispositivo no soporta el zoom.<br> |

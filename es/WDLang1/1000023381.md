


## PicSave (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda la imagen encontrada: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en un control Editor de imágenes.

- en una variable de tipo [WDPic](../WDLang1/1000023503.md). 





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Request to save the image found in an Image Editor control
IF PicSave(IE_MyEditor) = False THEN
	Error(ErrorInfo())
END
```

```wl
MyWDPicImage is WDPic
...
// Asks to save the image
IF PicSave(MyWDPicImage, "C:\temp\test.png") = False THEN
	Error(ErrorInfo())
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = PicSave(<Image> [, <File>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario (error, cancelación del selector de archivos, etc.). Si se produce un error, la variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve los detalles del error.  




**`<Image>: (Nombre del control o WDPic Variable)`**

Image para salvar. Esta imagen puede corresponder a: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre del Editor de Image para usar.

- el nombre de la [WDPic](../WDLang1/1000023503.md) variable a utilizar.




**`<File>: (Cadena de caracteres opcional)`**

Nombre y ruta del archivo Image a guardar. La extensión del archivo define el tipo de Image a guardar.
Si no se especifica este parámetro: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Para un editor de imágenes control: la acción realizada corresponde a la función de guardar del editor de imágenes (disquete Icon en la cinta).
	Si la Image nunca fue guardada, el selector de archivos se muestra. 

- Para una Variable de tipo [WDPic](../WDLang1/1000023503.md) , se muestra un error fatal.




## &lt;Kanban&gt;.DeleteList (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina una lista de un control Kanban.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Delete list 1
KANBAN_List.DeleteList(1)
// Insert 2 other lists instead
KANBAN_List.InsertList(1, "To review")
KANBAN_List.InsertList(2, "To do")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Eliminar una lista utilizando su índice

`<Kanban control>.DeleteList(<Index>)`
---

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<Index>: (Entero)`**

Índice de la lista a eliminar. 



<a name="SYNTAX2"></a>

### Eliminar una lista utilizando su nombre

`<Kanban control>.DeleteList(<Kanban list>)`
---

**`<Kanban control>: (Nombre del control)`**

Nombre del control Kanban utilizado.

**`<Kanban list>: (Nombre del control)`**

Nombre del control Lista Kanban a eliminar. 




<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La lista Kanban y todas sus tarjetas se eliminan del control Kanban. 

- Si el control Kanban está vinculado a un archivo de datos, los datos del archivo de datos no se modifican. 

- La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> si se produce un error durante la eliminación. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).

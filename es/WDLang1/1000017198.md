
## WinScreenRectangle (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve las coordenadas de la pantalla que contiene una ventana.. A continuación, tiene la posibilidad de gestionar las posiciones o los tamaños de las ventanas en un entorno de pantalla única o multipantalla.. Por ejemplo, puede averiguar si una ventana excede el tamaño de la pantalla o si la ventana está centrada en la pantalla.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sRect is string = WinScreenRectangle(WIN_Window1, screenWithoutTaskbar)
nYMax is int = ExtractString(sRect, 4)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = WinScreenRectangle([<Window used> [, <Options>]])`
---

**`<Result>: (Cadena de caracteres)`**

Dimensiones de la pantalla donde se muestra la ventana. Estas coordenadas se devuelven en el siguiente formato: 

```txt
<X1> + TAB + <Y1> + TAB + <X2> + TAB + <Y2>
```


- Las coordenadas (X1, Y1) corresponden a la Corner superior izquierda de la pantalla.

- Las coordenadas (X2, Y2) corresponden a la Corner inferior derecha de la pantalla.




Si se utilizan 2 pantallas y se visualiza la ventana en la segunda pantalla, las coordenadas (X1, Y1) corresponden a las coordenadas de la segunda pantalla (1280, 0 por ejemplo).

**`<Window used>: (Nombre de la ventana opcional)`**

Nombre o alias de la ventana que se va a utilizar. 
Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), se utiliza la ventana de llamada. 
La pantalla principal se utiliza si no se abre ninguna ventana.

**`<Options>: (Constante opcional de tipo Integer)`**

Elementos que se tienen en cuenta para el tamaño de la pantalla:


|   |   |
| --- | --- |
| *screenAll* <br>(Valor predeterminado) | Tamaño total de la pantalla. |
| *screenWithoutTaskbar* | Si la barra de tareas es Active en la pantalla, se resta su tamaño. |

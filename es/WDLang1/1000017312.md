
## HandleParentNext (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Obliga al padre de la siguiente ventana a abrirse. Esta función debe utilizarse justo antes de [Abrir](../WDLang1/3038035.md).

Esta función permite una ventana WINDEV:

- ser hijo de otra aplicación.

- ser hijo de una ventana que no sea de WINDEV.




**Atención: Esta es una función avanzada.**
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Opening code of the application project
ParentHandle is system int = CommandLine("HPARENT")
IF ParentHandle <> 0 THEN
	Trace(ParentHandle)
	HandleParentNext(ParentHandle)
END
Open(WIN_Window1)
EndProgram()

// Code of the caller
Parent is system int = Handle("")
ExeRun("MyApplication.exe" + " " + "/HPARENT=" + Parent)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`HandleParentNext(<Window handle>)`
---

**`<Window handle>: (Integro)`**

Mango de la ventana padre a utilizar.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Recordatorio: La manipular de una ventana es devuelta por [Handle](../WDLang1/3015005.md) o [SysWinHandle](../WDLang1/3073007.md).

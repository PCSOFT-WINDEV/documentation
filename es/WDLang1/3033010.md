


## Funciones del euro
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar el euro:



|   |   |
| --- | --- |
| [eChangeCurrency](../WDLang1/3033012.md) | Convierte una moneda a otra moneda. |
| [eCurrencyName](../WDLang1/3033006.md) | Devuelve el título de una moneda. |
| [eDefaultMemCurrency](../WDLang1/3033016.md) | Identifica o modifica la moneda almacenada que utiliza Default. |
| [eDefaultViewCurrency](../WDLang1/3033009.md) | Identifica o modifica la moneda de visualización utilizada por Default. |
| [eExchangeRate](../WDLang1/3033004.md) | Devuelve el valor del tipo de cambio de una moneda. |
| [eInitCurrency](../WDLang1/3033003.md) | Añade o cambia las características de una moneda en el cambio de moneda tabla. |
| [eRoundDifference](../WDLang1/3033015.md) | Devuelve la diferencia de redondeo entre:<br><br>- el valor mostrado en un control "Moneda + Euro",<br><br>- el associated control usado para la doble visualización.<br><br><br> |
| [eToCurrency](../WDLang1/3033001.md) | Convierte un valor expresado en euros a otra moneda. |
| [eToEuro](../WDLang1/3033005.md) | Convierte a euros un valor expresado en una moneda específica. |

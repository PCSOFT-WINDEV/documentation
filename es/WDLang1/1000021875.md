
## TextEllipsis (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el texto con elipsis mostrado en una ventana de Edición, Combo Box o control Estático.

**Atención**: 

- Esta función no puede ser usada para mostrar el texto en la control. 

- Esta función está disponible para los controles de una ventana. Esta función no se puede utilizar para los controles en una reporte. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sTextWithEllipsis is string = TextEllipsis(EDT_Edit1, "Welcome to your new office")
// sTextWithEllipsis is set to: "Welcome to..."
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TextEllipsis(<Control name> , <Text> [, <Management of ellipsis>])`
---

**`<Result>: (Cadena de caracteres)`**

Texto con elipsis correspondiente al texto que se muestra en la control especificada (según el estilo control).

**`<Control name>: (Cadena de caracteres)`**

Nombre de la Edición, Estática o control Combo Box utilizada para calcular la elipsis del texto. Esta control debe estar en una ventana.  

**`<Text>: (Cadena de caracteres)`**

Texto cuyo texto con elipsis debe ser calculado.

**`<Management of ellipsis>: (Constante opcional de tipo Integer)`**

Modo de gestión de la elipsis.


|   |   |
| --- | --- |
| *ellipsisDisabled* | Los caracteres "..." no se utilizan. No se añade la elipsis. |
| *ellipsisEnd*<br>(Valor predeterminado) | Los caracteres "..." sustituyen a los caracteres que no pueden aparecer al final de la control. |
| *ellipsisPath* | Los caracteres "..." sustituyen a los caracteres en el centro del texto. Esta constante debe utilizarse para mostrar una ruta a un archivo. |

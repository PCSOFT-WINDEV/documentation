


## &lt;Panel de control&gt;.AddWidget (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade y muestra un widget (ventana o página interna) en un control Cuadro de mando.

**Atención**: Los widgets añadidos mediante programación no se muestran automáticamente. Debe utilizar [&lt;Tablero de control&gt;Configuración inicial](../WDLang1/1000022525.md) antes de que aparezca el control Cuadro de mando por primera vez.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// Add 2 elements
DASH_MyDashboard.AddWidget(IW_Sales, "Sales")
DASH_MyDashboard.AddWidget(IW_MonthlyStat, "Monthly statistics for July", "201307")
```


<a name="Example2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
nIndex is int 

// Adds a widget that expects parameters
nIndex = DASH_Dashboard.AddWidget(IW_Widget_Clock, "Clock "+ ...
	COMBO_AddWidget[COMBO_AddWidget].DisplayedValue, ...
	COMBO_AddWidget[COMBO_AddWidget].DisplayedValue)

// Display this widget upon request
IF YesNo(Yes, "Do you want to display this widget?") = Yes THEN
	DASH_Dashboard[nIndex].Visible = True
ELSE
 	ToastDisplay("The widget was added to the dashboard as ""hidden""." + ...
		"Switch to edit mode to add it.", ...
		toastShort, vaMiddle, haCenter)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Dashboard control>.AddWidget(<Widget> , <Caption> [, <Parameter 1> [... [, <Parameter N>]]])`
---

**`<Result>: (Integro)`**

Índice del elemento añadido. Este índice puede utilizarse para manipular el widget.

**`<Dashboard control>: (Nombre del control)`**

Nombre de la control Cuadro de mando a utilizar.

**`<Widget>: (Nombre de la ventana interna o página interna)`**



- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Nombre de la ventana interna que se debe añadir a la control Cuadro de mando.




**`<Caption>: (Cadena de caracteres)`**

Título del widget. Esta leyenda aparecerá en el menú contextual de la control Cuadro de mando y permitirá al usuario manipular el widget.

**`<Parameter 1>: (Tipo de valor enviado a la ventana o página (opcional))`**

Primer parámetro que se pasará a las "Declaraciones Globales" Event de la ventana o página interna para abrir. Este parámetro es pasado por reference y es considerado como un global de Variable a la ventana o página interna.
Atención: Sólo están disponibles los tipos simples.



**`<Parameter N>: (Tipo de valor enviado a la ventana o página (opcional))`**

Nth parámetro que se pasará a las "Declaraciones Globales" Event de la ventana interna o página interna para abrir. Este parámetro es pasado por reference y es considerado como un global de Variable a la ventana o página interna.
Atención: Sólo están disponibles los tipos simples.

## Observaciones


### Nuevo widget
<a name="nuevo_widget_ELTPARAGRAPHE000252"></a>

El widget se declara además de los widgets definidos en la description de control Cuadro de mando. 

Este widget se puede hacer Visible: 

- utilizando [&lt;Tablero de control&gt;Configuración inicial](../WDLang1/1000022525.md) antes de la primera visualización.

- por el usuario final a través de el menú contextual.

- utilizando la siguiente sintaxis: 
	
	```txt
	DASH_Dashboard[Index].Visible = True
	```
donde &lt;Index&gt; es el número de widget devuelto por **&lt;Panel de control&gt;.AddWidget**.

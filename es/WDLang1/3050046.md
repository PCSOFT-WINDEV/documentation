
## FinLinearRedemption( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Calcula el valor de la amortización lineal de un bien durante un período de tiempo específico.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Amortization value of a good bought for 25000 Euros whose
// residual value will be 15000 Euros after 6 years.
ResAmortization = FinLinearRedemption(25000, 15000, 6)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = FinLinearRedemption(<Acquisition cost> , <Residual value> , <Duration>)`
---

**`<Result>: (Real o moneda)`**



- Valor de la amortización,

- 0 es que la amortización es nula.




**`<Acquisition cost>: (Real o moneda)`**

Coste de adquisición del bien.

**`<Residual value>: (Real o moneda)`**

Valor residual del bien después de la amortización.

**`<Duration>: (Real)`**

Número de períodos durante los cuales se reembolsa el bien. Esta duración no puede ser nula.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Amortización lineal
<a name="amortizacion_lineal_ELTPARAGRAPHE000074"></a>

En una amortización lineal, el valor de amortización es el mismo para cada período de amortización.
<a name="NOTE0_2"></a>




### Gestión de errores
<a name="gestion_errores_ELTPARAGRAPHE000082"></a>

**Atención**: La función **FinLinearRedemption** no devuelve ningún código de error. Para determinar si esta función ha generado errores, utilice [FinError](../WDLang1/3050071.md). Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.

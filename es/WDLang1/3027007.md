


## DurationToString (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte una duración en una cadena de caracteres en el formato especificado. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start time and end time
StartTime is Time = "125324"
EndTime is Time = "140852"
// Calculate the duration
MyDuration is Duration = EndTime - StartTime
// Format the duration in order to get a result in the same format as DateTimeDifference
Res = DurationToString(MyDuration, durationCenti) // Res = "001152800"
// Custom formatting
Res = DurationToString(MyDuration, "D-HH:MM")    // Res =  "0-01:15"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DurationToString(<Duration> , <Format>)`
---

**`<Result>: (Cadena de caracteres)`**

Cadena de caracteres en el formato especificado.

**`<Duration>: (Variable de tipo Duration)`**



- Variable de tipo Duration que contiene la duración en formato "+DHHMMSSCCC" (D corresponde al número de días, HH a las horas, MM a los minutos, SS a los segundos y CCC a los milisegundos).

- Duración en un formato legible (por ejemplo, 1 d 2 h 3 min 10 s 10 ms). 




**`<Format>: (Cadena de caracteres o constante)`**

Formato a aplicar a la duración. Este parámetro puede corresponder a:

- una cadena de caracteres (una frase, por ejemplo). Los caracteres que representan los elementos de la duración (D, HH, MM, SS o CCC) se reemplazarán automáticamente por el valor del parámetro &lt;Duración&gt;. Los caracteres de la cadena son los siguientes:

	- + permite tener en cuenta los números negativos,

	- D representa el número de días,

	- HH representa el número de horas,

	- MM representa el número de minutos,

	- SS representa el número de segundos,

	- CCC representa el número de milisegundos.




- una de las siguientes constantes:
	


|   |   |
| --- | --- |
| *durationCenti* | Formato utilizado: "+DHHMMSSCC". Este formato permite recuperar el resultado en el formato utilizado por la función [DateTimeDifference](../WDLang1/3027020.md). |
| *durationMilli* | Formato utilizado: "+DHHMMSSCCC" |








<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La asignación directa &lt;Cadena&gt; = &lt;Duración&gt; es equivalente a la función **DurationToString** con el formato "+DHHMMSSCCC".

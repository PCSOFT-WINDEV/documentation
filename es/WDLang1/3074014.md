


## TableFixColumn (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Fija una columna a la izquierda o libera una columna fijada a la izquierda.. Esta función también se puede utilizar para averiguar si la columna está fijada a la izquierda o no.

**Observaciones**: 

- Las columnas fijadas a la izquierda no se desplazan con la barra de desplazamiento horizontal.. 

- Esta función es equivalente a la función: 

	- la cerradura set por el usuario.

	- la opción "Congelar columnas a la izquierda" en el menú contextual del título de la columna.


 Para obtener más información, consulte [AAF: Congelación de las columnas de una control Tabla](../WDChamp/9500109.md)



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Fixes/Frees a column with a button
// Click on the button
IF TableFixColumn(COL_COLUMN) THEN
	// The column was fixed
	TableFixColumn(COL_COLUMN, False) // Free it
ELSE
	// The column was freed
	TableFixColumn(COL_COLUMN, True) // Fix it
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = TableFixColumn(<Column> [, <Fix / Free>])`
---

**`<Result>: (booleano)`**

Status anterior de la columna: 

- <u><u><u><u>True</u></u></u></u>, columna fija,

- <u><u><u><u>False</u></u></u></u>, columna libre.




**`<Column>: (Nombre del control)`**

Nombre de la columna a utilizar. 
Si varias controles Tabla incluyen columnas con el mismo nombre, el nombre de la columna debe ir precedido del nombre de la control Tabla: &lt;control Tabla&gt;.&lt;Columna&gt;.

**`<Fix / Free>: (Booleano opcional)`**



- <u><u><u><u>True</u></u></u></u>: la columna debe ser fija

- <u><u><u><u>False</u></u></u></u>: la columna debe ser liberada. 


Si no se especifica este parámetro, la función **TableFixColumn** devuelve el estado actual de la columna.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Si se utiliza la función **TableFixColumn** para fijar una columna que ya está fija, esta función no tiene ningún efecto.

- Esta función no puede utilizarse con columnas indexadas.

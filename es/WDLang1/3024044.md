
## UncompleteDir (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Elimina los caracteres '/' o '\\' del final de una cadena, si es necesario.

Esta función es útil para manipular nombres de archivos completos cuando se desconoce el formato de la ruta (por ejemplo, cuando el usuario introduce una ruta de archivo).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
Res = UncompleteDir("c:\temp\")         // Returns "c:\temp"
Res = UncompleteDir("c:\temp")          // Returns "c:\temp"
Res = UncompleteDir("\\CCS\Dir2\User\") // Returns "\\CCS\Dir2\User"
Res = UncompleteDir("/root/file/")      // Returns "/root/file"
Res = UncompleteDir("")                 // Returns ""
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = UncompleteDir(<String to check>)`
---

**`<Result>: (Cadena de caracteres)`**



- &lt;Cadena a analizar&gt; sin los caracteres '\\' o '/' al final,

- Cadena vacía si &lt;Cadena a analizar&gt; corresponde a una cadena vacía ("").




**`<String to check>: (Cadena de caracteres)`**

Cadena de caracteres a analizar.

## Observaciones




### Espacios
<a name="espacios_ELTPARAGRAPHE000067"></a>

**Atención**: Los espacios al inicio y al final de la &lt;Cadena a analizar&gt; se eliminan (igual que la función [NoSpace](../WDLang1/3024034.md)).
<a name="NOTE0_2"></a>




### Función UncompleteDir y UNICODE
<a name="funcion_docparampagetitleshort_unicode_ELTPARAGRAPHE000078"></a>

Si la cadena pasada como parámetro es una cadena ANSI, el resultado será una cadena ANSI.

Si la cadena pasada como parámetro es una cadena UNICODE, el resultado será una cadena UNICODE.

**Observación**: Si el resultado de la función **UncompleteDir** de una cadena ANSI se asigna a una cadena UNICODE (y viceversa), la conversión se realizará implícitamente.

Para obtener más información, consulte [Gestión de UNICODE](../WDLang1/3024038.md).

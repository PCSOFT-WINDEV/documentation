
## HSL (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un color a partir del tono, saturación y luminosidad.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Color an area in light blue
dFill(10, 50, HSL(180, 100, 50))
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = HSL(<Color Hue> , <Color Saturation> , <Color Lightness>)`
---

**`<Result>: (Entero)`**

Valor del color. Este valor se puede utilizar en todas las funciones WLanguage que manipulan un color.

**`<Color Hue>: (Entero)`**

Valor comprendido entre 0 y 360 (ángulo del círculo cromático). El rojo corresponde a 0°.

**`<Color Saturation>: (Entero)`**

Valor comprendido entre 0 y 100 (porcentaje de saturación).

**`<Color Lightness>: (Entero)`**

Valor comprendido entre 0 y 100 (porcentaje de luminosidad).

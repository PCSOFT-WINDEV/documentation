


## Funciones de gestión del control Editor de imágenes y de las variables WDPic
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones de gestión del control Editor de imágenes son las siguientes: 



|   |   |
| --- | --- |
| [PicClose](../WDLang1/1000023382.md) | Cierra la imagen encontrada en un contro «Image Editor» |
| [PicCreate](../WDLang1/1000023380.md) | Crea una imagen en el control «Editor control». |
| [PicMergeLayer](../WDLang1/1000024865.md) | Combina las capas de una imagen en formato WDPic. |
| [PicOpen](../WDLang1/1000023379.md) | Abre una imagen en un editor de imágenes control. |
| [PicRectangularSnip](../WDLang1/1410088683.md) | Captura un área rectangular con el cursor y almacena la imagen en una variable de tipo [Image](../WDLang1/1000019650.md). |
| [PicRectangularSnipOCR](../WDLang1/1410088839.md) | Ejecuta el OCR y extrae el texto de un área rectangular definida con el ratón. |
| [PicResize](../WDLang1/1000024635.md) | Redimensiona una imagen en formato WDPic. |
| [PicRotation](../WDLang1/1000024633.md) | Gira una imagen en formato WDPic. |
| [PicSave](../WDLang1/1000023381.md) | Guarda la imagen encontrada: <br><br>- en un control Editor de imágenes.<br><br>- en un WDPic Variable. <br><br><br> |

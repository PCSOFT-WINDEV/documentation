
## SysYRes( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la resolución vertical de la pantalla.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Vertical screen resolution
ResVertRes = SysYRes()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = SysYRes()`
---

**`<Result>: (Integro)`**

Resolución de pantalla vertical.

### Resolución de las configuraciones multipantalla
<a name="resolucion_las_configuraciones_multipantalla_ELTPARAGRAPHE000179"></a>

En las configuraciones multipantalla, hay que utilizar [SysListVideoCard](../WDLang1/1000017016.md), [Pantalla SysListScreen](../WDLang1/1000017014.md) y [SysScreenResolución](../WDLang1/1000017019.md) para get la resolución máxima en una dimensión determinada.

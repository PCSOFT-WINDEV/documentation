


## Funciones de gestión de controles Lector PDF
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar los controles del lector de PDF: 



|   |   |
| --- | --- |
| [PDFReaderAbrir](../WDLang1/1000023452.md) | Abre un archivo PDF que se muestra en un lector de PDF control. |
| [PDFReaderCerrar](../WDLang1/1000023453.md) | Cierra y libera el archivo PDF que se muestra en un lector de PDF control. |
| [PDFReaderGoToBookmark](../WDLang1/1410089202.md) | Establece la posición directamente en el marcador deseado en el documento PDF. |
| [PDFReaderHighlight](../WDLang1/1000023544.md) | Encuentra un texto en el Document mostrado por un lector de PDF control y resalta las ocurrencias de este texto en el control. |
| [PDFReaderInfoXY](../WDLang1/1000025332.md) | Para una posición dada (coordenadas de un punto) en un PDF Reader control, devuelve una de las siguientes opciones: <br><br>- el número Page para la posición especificada, <br><br>- la posición X (en mm) para la posición especificada, <br><br>- la posición Y (en mm) para la posición especificada.<br><br><br> |
| [PDFReaderRotatePage](../WDLang1/1000023940.md) | Gira una página mostrada en el lector de PDF control. |
| [PDFReaderSave](../WDLang1/1000024898.md) | Guarda el contenido de un control Lector PDF en un archivo. |
| [PDFReaderSeekFirst](../WDLang1/1000023409.md) | Inicializa la búsqueda de un texto en un archivo PDF visualizado por el lector PDF control y se posiciona en la primera ocurrencia encontrada (a partir del Page actual). |
| [PDFReaderSeekNext](../WDLang1/1000023410.md) | Establece la posición en el siguiente resultado de una búsqueda en el contenido de un lector de PDF control. |
| [PDFReaderSeekPrevio](../WDLang1/1000023411.md) | Establece la posición en el resultado anterior de una búsqueda iniciada con [PDFReaderSeekFirst](../WDLang1/1000023409.md) en el contenido de un lector de PDF control. |



Observación: [iPrintPDF](../WDLang5/1000020519.md) puede utilizarse para imprimir el contenido de un lector de PDF control.

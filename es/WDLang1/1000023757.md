


## &lt;Vista de árbol&gt;.ExpandAll (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Expande un nodo y todos sus elementos hijo en una control TreeView.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)
**Reminder**: En tiempo de ejecución, un nodo se expande cuando el nombre del nodo va precedido de "-".
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Expand the "Desserts" node in the "TREE_TVRecipe" TreeView control
// All child elements are also expanded
Res = TREE_TVRecipe.ExpandAll("Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeView control>.ExpandAll([<Node to expand>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el nodo y sus elementos hijos se han expandido,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView que se va a utilizar.

**`<Node to expand>: (Cadena de caracteres opcional)`**

Ruta del nodo a expandir (así como sus elementos hijos). Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```


Si no se especifica este parámetro, control TreeView se expande desde la raíz.

Si hay dos elementos con el mismo nombre en la ruta, puede añadir los ID de los elementos (después de sus nombres utilizando [TreeID](../WDLang1/3018027.md)).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Nodo a expandir
<a name="nodo_expandir_ELTPARAGRAPHE000195"></a>

Si el nodo a expandir:

- no se encuentra, **&lt;Vista de árbol&gt;.ExpandAll** devuelve <u><u><u><u>False</u></u></u></u>.

- no se especifica, **&lt;Vista de árbol&gt;.ExpandAll** devuelve <u><u><u><u>True</u></u></u></u> y el control TreeView se expande desde su raíz.

- es una hoja, **&lt;Vista de árbol&gt;.ExpandAll** devuelve <u><u><u><u>True</u></u></u></u>.

- ya está expandido, **&lt;Vista de árbol&gt;.ExpandAll** devuelve <u><u><u><u>True</u></u></u></u>, y los elementos hijos se expanden si es necesario.

- no es visible, **&lt;Vista de árbol&gt;.ExpandAll** hace que este nodo sea visible. También se muestran sus elementos "padre" y "hermano".. Los elementos "padre" se expanden.


Cuando se llama a **&lt;Vista de árbol&gt;.ExpandAll**, si la rama especificada y/o uno de sus hijos tienen un Procedure asociado a través de la propiedad [AddChildDelayed](../Proprietes/1000020927.md), se ejecutará este Procedure. 
<a name="NOTE0_2"></a>


### Banda de selección
<a name="banda_seleccion_ELTPARAGRAPHE000246"></a>

La función **&lt;Vista de árbol&gt;.ExpandAll** no cambia la posición de la banda de selección.

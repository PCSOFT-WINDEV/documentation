


## &lt;Organigrama&gt;.SaveImage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Crea un Image que representa a todo el control Organigrama.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Saves the image of Organization Chart control
ORG_Direction.SaveImage("OrganizationChartImage.png")
// Print the image in a report (image passed as parameter)
RPT_Org.Print("OrganizationChartImage.png")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Organization Chart control>.SaveImage(<Image name>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó la imagen,

- <u><u><u><u>False</u></u></u></u> en caso contrario. La función [ErrorInfo](../WDLang1/3013008.md) permite identificar el error.




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama cuyo Image será salvado.

**`<Image name>: (Cadena de caracteres)`**

Nombre del archivo Image a crear. El formato de la Image se deduce de la extensión (".gif", ".BMP", ".png", ".jpg").



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
La Image creada en disco es una representación de todo el organigrama. Si se encuentran barras de desplazamiento, las áreas ocultas también se incluyen en la Image. Por lo tanto, esta Image puede ser grande.




## Funciones del editor HTML control
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones WLanguage se utilizan para gestionar los controles del editor HTML: 



|   |   |
| --- | --- |
| [HTMLEditor-EjecutarJS](../WDLang1/1000026260.md) | Ejecuta el código JavaScript en un editor HTML control. |
| [HTMLEditorFormatSelection](../WDLang1/1000025996.md) | Modifica el estilo de los elementos seleccionados en un editor HTML control. |
| [HTMLEditorFromEmail](../WDLang1/1000026172.md) | Importa el contenido HTML de un correo electrónico y sus imágenes en un editor HTML control. Las imágenes contenidas en la Email se muestran también en la control. |
| [HTMLEditorGet](../WDLang1/1000025981.md) | Recupera asincrónicamente el valor de una de las propiedades de un editor de HTML control. |
| [HTMLEditorGetDocument](../WDLang1/1000025993.md) | Recupera de forma asíncrona la Document HTML que se está mostrando en una control de edición HTML . |
| [HTMLEditorGetSelection](../WDLang1/1000025992.md) | Recupera de forma asíncrona la selección actual de un editor HTML control. |
| [HTMLEditorInsert](../WDLang1/1000025994.md) | Inserta texto o código HTML en la posición del cursor en un editor HTML control. |
| [HTMLEditorInsertTable](../WDLang1/1000025917.md) | Inserta una tabla en la posición del cursor en un editor de HTML control. |
| [HTMLEditorSearch](../WDLang1/1000025918.md) | Busca una cadena en un editor HTML control. |
| [HTMLEditorSelect](../WDLang1/1000025926.md) | Selecciona un fragmento de texto en un editor de HTML control. |
| [HTMLEditorSustituir](../WDLang1/1000025995.md) | Sustituye una cadena por otra en un editor HTML control. |
| [HTMLEditorToEmail](../WDLang1/1000026171.md) | Exporta el contenido del control de un editor HTML a un correo electrónico Variable (para enviar la Email). |
| [HTMLEditorToPDF](../WDLang1/1000025927.md) | Exporta el contenido de un editor HTML control a un archivo PDF. |

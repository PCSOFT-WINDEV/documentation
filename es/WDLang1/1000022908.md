


## &lt;InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el Appointment, el Resource o la fecha encontrada en un puesto determinado en un control Planificador.

**Atención**:  [MouseXPos](../WDLang1/3071007.md) y [MouseYPos](../WDLang1/3071008.md) devuelven una información relevante en los procesos relacionado al ratón o al puntero (código de clic, código de rollover, código de botón izquierdo o derecho pulsado, ...). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Displays in the trace the date found at hovered location 
// (optional WM_MOUSEMOVE event)
// The result would also be obtained in the code of left button up (WM_LBUTTONUP)
// Not to be used in the code for "Selecting a period" in the Scheduler control 
// that is not linked to the mouse

MyDate is string
MyDate = SCH_Schedule1.InfoXY(schDateTime + schScreenOrigin, MouseXPos(), MouseYPos())
Trace(MyDate)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Scheduler control>.InfoXY(<Type of information> , <X> , <Y>)`
---

**`<Result>: (Cadena de caracteres o entero)`**

Información solicitada: 

- Índice de Appointment en un punto específico, -1 si no hay Appointment. 

- Fecha y hora (en formato AAAAMMDDHHSSCC) de la cita en el punto especificado, cadena vacía ("") si no hay fecha. 

- Nombre de Resource en el punto especificado, cadena vacía ("") si no hay Resource.




**`<Scheduler control>: (Nombre del control)`**

Nombre de control Planificador a utilizar.

**`<Type of information>: (Constante o combinación de constantes de tipo Integer)`**

Tipo de información solicitada:


|   |   |
| --- | --- |
| *schAppointment* | Índice del Appointment en las coordenadas especificadas. |
| *schDateTime* | Fecha y hora de la Appointment correspondiente a las coordenadas especificadas. |
| *schResource* | Nombre de Resource correspondiente a las coordenadas especificadas. |
| *schScreenOrigin* | Por Default, el punto (0,0) corresponde al punto superior izquierdo de control Planificador utilizado (las barras de desplazamiento se encuentran en el origen).<br>Si la constante *schScreenOrigin* se combina con las otras constantes, el punto (0,0) corresponde al origen de la pantalla.<br>Esta constante no puede utilizarse por sí sola.<br><br> |



**`<X>: (Integro)`**

Coordenada X (en pixeles) a analizar. Esta coordenada es relativa al control Planificador (si no se especifica la constante *schScreenOrigin*).

**`<Y>: (Integro)`**

Coordenada Y (en pixeles) a analizar. Esta coordenada es relativa al control Planificador (si no se especifica la constante *schScreenOrigin*).




## TableSelectToFile (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Posiciones en el registro el archivo de datos correspondientes a un fila seleccionado en la navegación selección múltiple de control Tabla.

**Observación**: En una control Tabla de selección única basada en un archivo de datos, el Record actual en el archivo de datos siempre corresponde al elemento seleccionado en el control Tabla.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
i is int
FOR I = 1 TO TableSelectCount(TABLE_Product)
	TableSelectToFile(TABLE_Product, i)
	Trace(Product.ProductID)
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TableSelectToFile(<Table control> , <Rank>)`
---

**`<Table control>: (Nombre del control)`**

Nombre de la control Tabla basada en un archivo de datos que se va a utilizar. 
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control Tabla al que pertenece el Event actual.
Se produce un error de WLanguage si este parámetro no corresponde a un control Tabla de navegación.

**`<Rank>: (Integro)`**

Rango de la selección. Se produce un error de WLanguage si este parámetro no corresponde a un rango existente..  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función puede utilizarse con una control Tabla de selección única basada en un archivo de datos si y sólo si &lt;Rango&gt; es igual a 1

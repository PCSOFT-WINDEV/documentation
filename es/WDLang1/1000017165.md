


## Funciones de gestión de la papelera de reciclaje
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes funciones se utilizan para gestionar la papelera de reciclaje:



|   |   |
| --- | --- |
| [RecycleBinClear](../WDLang1/1000017157.md) | Borra la papelera de reciclaje del sistema. |
| [RecycleBinDelete](../WDLang1/1000017158.md) | Muestra un mensaje de confirmación para borrar un archivo de la papelera de reciclaje. |
| [RecycleBinListFile](../WDLang1/1000017162.md) | Enumera los archivos encontrados en la papelera de reciclaje del sistema. |
| [RecycleBinRestore](../WDLang1/1000017159.md) | Restaura un archivo que se encuentra en la papelera de reciclaje del sistema. |

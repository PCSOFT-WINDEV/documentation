


## InsertColumn( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una columna en una posición dada en una propiedad array bidimensional avanzada (xlsDocument, ....). 






<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`InsertColumn(<WLanguage array> , <Insertion index> [, <Element row 1> [... [, <Element row N>]]])`
---

**`<WLanguage array>: (Array)`**

Nombre de la variable de tipo Array a utilizar. Esta Variable debe corresponder a una array bidimensional.

**`<Insertion index>: (Integro)`**

Subíndice en el que debe insertarse la columna. A partir de esta índice, todas las columnas se desplazan en 1 columna. 
Si &lt;Índice de inserción&gt; es igual al número de columnas del array + 1, la columna se agrega al final del array (equivalente a la función [AñadirColumna](../WDLang1/1000017350.md)). 
Se produce un error WLanguage si  es mayor que el número de columnas array + 1.

**`<Element row 1>: (Tipo de elementos del array, opcional)`**

Elemento de la columna que se insertará en la array especificada. Cada elemento corresponde a una fila en el array. El tipo de elemento añadido debe ser compatible con el tipo de la fila correspondiente..
Si no se especifica ninguno de estos parámetros, se inserta una columna vacía en la ventana de diálogo array. Las filas se inicializan con el valor Default del tipo de los otros elementos array.
Si no se especifica uno de estos parámetros, el correspondiente fila se inicializa con el valor Default del tipo de los otros elementos array.



**`<Element row N>: (Tipo de elementos del array, opcional)`**

Elemento de la columna que se insertará en la array especificada. Cada elemento corresponde a una fila en el array. El tipo de elemento añadido debe ser compatible con el tipo de la fila correspondiente..
Si no se especifica ninguno de estos parámetros, se inserta una columna vacía en la ventana de diálogo array. Las filas se inicializan con el valor Default del tipo de los otros elementos array.
Si uno de estos parámetros no se especifica, el correspondiente fila se inicializa con el valor Default del tipo de los otros elementos array.

## Observaciones


### Insertar una columna en un array Property avanzado
<a name="insertar_una_columna_array_property_avanzado_ELTPARAGRAPHE000075"></a>

Cuando se llama a la función **InsertColumn**:

- se debe crear la variable de tipo avanzado.

- el tipo avanzado debe tener un enumerador de tipo de colección modificable.

- el tipo avanzado se amplía automáticamente para recibir la nueva columna.

- las filas se inicializan con los valores pasados como parámetro. Si no se pasa ningún valor en el parámetro, las filas se inicializan con el valor Default del tipo de los elementos array.

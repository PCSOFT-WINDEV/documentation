
## kbActionAllowed (Tipo de variable)

*** ***
			
				



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
El tipo **kbActionAllowed** permite definir qué acciones están permitidas para el usuario final cuando manipula tarjetas en un control Kanban. Estos permisos pueden definirse y cambiarse mediante diferentes propiedades WLanguage. 

Atención: Este tipo de variable solo puede utilizarse con variables de tipo [kbCard](../WDLang1/1410089167.md). 

**Observación**: Para obtener más información sobre la declaración de este tipo de variable y el uso de propiedades WLanguage, consulte [Declarar una variable](../Motscles/1514032.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Prevent user from deleting a task
TheTask is kbCard
TheTask.Title = "IMPORTANT"
TheTask.ActionAllowed.Remove = False
KanbanAddCard(LIST_TODO, TheTask)
```





<a name="PROPS"></a>

## Properties


### Propiedades específicas de las variables de tipo kbActionAllowed:
<a name="propiedades_especificas_las_variables_tipo_kbactionallowed_ELTPARAGRAPHE000031"></a>

Las siguientes propiedades se pueden utilizar para definir las acciones permitidas en una tarjeta de un control Kanban:

| Nombre de la propiedad | Tipo utilizado | Efecto |
| --- | --- | --- |
| ViewDetails | Boolean | <br><br>- <u><u><u>True</u></u></u> para permitir la ejecución del evento "Visualización de los detalles de una tarjeta" cuando el usuario pulsa una o dos veces una tarjeta en un dispositivo móvil,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br> |
| ChangeList | Boolean | <br><br>- <u><u><u>True</u></u></u> para permitir que el usuario final mueva la tarjeta de una lista a otra,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br> |
| Move | Boolean | <br><br>- <u><u><u>True</u></u></u> para permitir que el usuario final mueva la tarjeta en la lista,<br><br>- <u><u><u>False</u></u></u> en caso contrario. <br><br><br> |
| Remove | Boolean | <br><br>- <u><u><u>True</u></u></u> para permitir que el usuario final elimine la tarjeta arrastrándola y soltándola al final de la lista,<br><br>- <u><u><u>False</u></u></u> en caso contrario.  <br><br><br> |




## &lt;Pivot tabla&gt;.InfoXY (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve los valores de las cabeceras correspondientes a una posición en una control Tabla dinámica en píxeles..
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
pos is pvtPosition of PVT_Statistics
pos = PVT_Statistics.InfoXY(MouseXPos(), MouseYPos())
IF pos.Out = True THEN 
	STC_Caption  = ""
ELSE
	STC_Caption = "Year =" + pos.Year + ...
		"Product = " + pos.Product + ...
		"Sales  = " + PVT_Statistics.VAL_Sales[pos] 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Pivot Table control>.InfoXY(<X> , <Y>)`
---

**`<Result>: (pvtPosition Variable)`**

[pvtPosition](../WDLang1/1000020726.md) Variable que contiene las características de la célula seleccionada. 

Observaciones: 

- Si la posición no corresponde a ninguna celda del pivote tabla (-1,-1), el **Out** Property del [pvtPosition](../WDLang1/1000020726.md) Variable es set a <u><u><u><u>True</u></u></u></u>.

- La posición (0,0) corresponde a la 1ª celda en la parte superior izquierda.




**`<Pivot Table control>: (Nombre del control)`**

Nombre del control Tabla dinámica a utilizar.

**`<X>: (Integro)`**

Coordenada X (en píxeles) a analizar. Esta coordenada es relativa al área de clientes de la control.

**`<Y>: (Integro)`**

Coordenada Y (en píxeles) a analizar. Esta coordenada es relativa al área de clientes de la control.




## MenuInvisible( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Hace una opción de menú Invisible (menú emergente o menú desplegable). Las opciones encontradas después de la opción modificada se mueven una Line a la parte superior.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The "OPT_IconBar" option becomes invisible
MenuInvisible(OPT_IconBar)
```
<a name="Example2"></a>

```wl
// Makes the "Save" sub-option invisible 
// in the "File" menu of the "WIN_WinMenu" window 
// from the "Customer" window
MenuInvisible(WIN_WinMenu, File.Save)
// equivalent to WIN_WinMenu.File.Save..Visible = False
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`MenuInvisible([<Window>, ] <Menu option>)`
---

**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú. 
Si no se especifica este parámetro, **MenuInvisible** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a utilizar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe ser especificado sin comillas.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Equivalencia: La siguiente sintaxis: 
	
	```txt
	MenuInvisible(<Menu option>)
	```

	equivale a la [Visible](../Proprietes/2510138.md) property: 
	
	```txt
	Menu option>.Visible
	```


- Cuando se hace una opción de menú Invisible, el atajo associated ya no se puede usar.

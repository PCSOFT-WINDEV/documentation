


## &lt;Menú&gt;.InsertarOpción (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inserta una nueva opción en una posición específica. Esta opción de menú ejecuta una Procedure pasada en parámetro.

Para añadir una opción de menú al final de un menú, utilice [&lt;Menú&gt;.AddOption](../WDLang1/1000020593.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Insert an option into the new menu
MNU_Schedule.InsertOption("MNU_Consult", "MNU_Create", "Create", Create_Schedule)
// Associate an image with the menu option
{"MNU_Create", indControl}..Image = "See.png"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Insertar una opción de menú (posición definida por una opción de menú)

`<Result> = <Source menu>.InsertOption(<Insertion option> , <Option to create> , <Caption> , <WLanguage procedure> [, <Procedure parameter>])`
---

**`<Result>: (Integro)`**

Posición de la opción en el menú. Se produce un error fatal si no se ha insertado la opción de menú.

**`<Source menu>: (El nombre del menú)`**

Nombre del menú donde la opción de menú debe ser creada y añadida. La opción del menú se añadirá al final de este menú.

**`<Insertion option>: (Cadena de caracteres)`**

Nombre de la opción de menú utilizada como posición base para la inserción. La nueva opción de menú se insertará antes de esta posición.

**`<Option to create>: (Cadena de caracteres)`**

Nombre de la opción de menú para crear e insertar. Este nombre se utilizará para manipular la opción de menú mediante programación. Si existe una opción con el mismo nombre, se produce un error fatal.

**`<Caption>: (Cadena de caracteres)`**

Texto de la nueva opción de menú. Este título se mostrará en la ventana y permitirá al usuario seleccionar la opción.

**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre del procedimiento WLanguage al que se llama cuando se selecciona la opción de menú.

**`<Procedure parameter>: (Variante opcional)`**

Parámetro que se puede pasar a &lt;WLanguage procedure&gt; cuando el usuario selecciona la opción de menú.

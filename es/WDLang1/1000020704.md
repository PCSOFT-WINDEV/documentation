


## Funciones de gestión de controles Tabla dinámica
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Estas funciones se utilizan para manipular el Pivote controles Tabla:



|   |   |
| --- | --- |
| [Colapso del PVTC](../WDLang1/1000020698.md) | Oculta los datos encontrados en una control Tabla dinámica según una cabecera fila o una cabecera de columna.. |
| [PVTAxisXY](../WDLang1/1000021113.md) | Permite: <br><br>- Especifique las cabeceras de fila y de columna visualizadas en un control Tabla dinámica. <br><br>- Descubra las cabeceras de columna y fila mostradas en un control Tabla dinámica.<br><br><br> |
| [PVTCalcularActualizar](../WDLang1/1000020534.md) | Actualiza algunos de los datos encontrados en una control Tabla dinámica a partir de una fecha determinada. |
| [PVTCalculateAll](../WDLang1/1000020539.md) | Entirely calculates or recalculates a Pivot Table control. |
| [PVTCollapseAll](../WDLang1/1000020622.md) | Oculta todas las cabeceras de fila y/o de columna en un control Tabla dinámica. |
| [PVTCompareDateRange](../WDLang1/1000024869.md) | Calcula y muestra una comparación de rango de fechas en un control Tabla dinámica. |
| [PVTExpand](../WDLang1/1000020699.md) | Desglosa los datos encontrados en una control Tabla dinámica según una cabecera fila o a una cabecera de columna. |
| [PVTExpandAll](../WDLang1/1000020623.md) | Expande todas las cabeceras de fila y/o de columna en un control Tabla dinámica. |
| [PVTFiltro](../WDLang1/1000020531.md) | Habilita o deshabilita un filtro en una control Tabla dinámica. |
| [PVTHeaderValue](../WDLang1/1000021135.md) | Devuelve el valor mostrado en una cabecera de control Tabla dinámica. |
| [PVTInfoXY](../WDLang1/1000020735.md) | Devuelve los valores de las cabeceras correspondientes a una posición en una control Tabla dinámica en píxeles. |
| [PVTListPosition Header](../WDLang1/1000020975.md) | Devuelve todos los valores associated con un fila o un encabezado de columna en un control Tabla dinámica. |
| [PVTLoad](../WDLang1/1000020507.md) | Carga el resultado del cálculo realizado en una control Tabla dinámica desde una copia de seguridad (archivo encontrado en el disco o copia de seguridad en un búfer). |
| [PVTSave](../WDLang1/1000020471.md) | Guarda el resultado del cálculo realizado en un control Tabla dinámica (en un archivo o en la memoria). |
| [PVTSelect](../WDLang1/1000021024.md) | Le permite conocer la posición de las celdas seleccionadas y seleccionar celdas en un control Tabla dinámica. |
| [PVTSelectOccurrence](../WDLang1/1000022175.md) | Devuelve el número de celdas seleccionadas en una control Tabla dinámica. |
| [PVTToExcel](../WDLang1/1000020703.md) | Crea un fichero Excel con los datos de un control Tabla dinámica. |

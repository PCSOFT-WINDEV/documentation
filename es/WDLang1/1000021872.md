
## WinEdSelectPlus (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Selecciona una control cuando la interfaz es modificada por el usuario final (modo "modificación de UI por el usuario final" activado).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Add a control and select the control
X is Control
X -> ControlCreate("STC" + GetIdentifier(), typStatic, 10, 10, 100, 24, True)
X.Caption = "New text"
WInEdSelectPlus(WIN_Form, X)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`WinEdSelectPlus(<Window> [, <Control used>])`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana o ventana interna a utilizar. Esta ventana debe estar en modo de modificación ([WinEdActive](../WDLang1/1000021843.md)). 

**`<Control used>: (Opcional control)`**

control para seleccionar en la ventana. 

- Si la selección múltiple no está activada, la selección actual se sustituye por la control especificada. 

- Si la selección múltiple está activada, la control especificada se añade a la selección. 


La función no tiene efecto si no se selecciona la control especificada.

Si no se especifica este parámetro: 

- si se activa la selección múltiple, se seleccionan todos los controles de la ventana.

- si la selección múltiple no está activada, la función no tiene efecto.   






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La control seleccionada es devuelta por [WinEdSelect](../WDLang1/1000021871.md).  

- Si la selección múltiple está activada, puede determinar el número de controles seleccionados con [WinEdSelectOccurrence](../WDLang1/1000021870.md) y recorrerlos con [WinEdSelect](../WDLang1/1000021871.md).

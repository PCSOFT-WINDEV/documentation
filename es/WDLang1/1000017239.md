


## ExeGetPID( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve información sobre el actual Process.






<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// PID of the application
MyPID is int
MyPID = ExeGetPID()
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ExeGetPID([<Sought information>])`
---

**`<Result>: (Entero del sistema)`**

Información solicitada.

**`<Sought information>: (Constante opcional de tipo Integer)`**

Información sobre el actual Process:


|   |   |
| --- | --- |
| *exeParentPID* | PID de padre (que significa el Process que inició el actual Process).<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |
| *exePID*<br>(valor predeterminado) | PID (Identificador de Process us) del actual Process. |
| *exeTID* | Identificador de la Thread actual .<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Esta constante no está disponible. |

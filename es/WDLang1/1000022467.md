
## AAFContextMenuToString( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la lista de opciones del menú contextual que muestra realmente control. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sAll is string = AAFContextMenuToString(TABLE_NoName1)	

FOR EACH STRING sOption OF sAll SEPARATED BY CR
	sCaption is string = ExtractString(sOption, 1, TAB)
	nID is string = ExtractString(sOption, 2, TAB)
	sPos is string = ExtractString(sOption, 3, TAB)
	Trace(sCaption + " - " + nID + " - " + sPos)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = AAFContextMenuToString(<Control>)`
---

**`<Result>: (Cadena de caracteres)`**

Lista de opciones del menú contextual de control. Esta lista tiene el siguiente formato: 

```txt
<Caption 1> + TAB + <Identifier 1> + TAB + <Position 1> + CR + ... 
+ CR + <Caption N> + TAB + <Identifier N> + TAB + <Position N>
```
donde: 

- Pie de foto: Texto de opción. 

- Identificador: Identificador de opción. 
	Observación: Este identificador puede ser utilizado por [AAFExecute](../WDLang1/1000022099.md) para ejecutar la opción directamente.

- ubicación: Índice de opciones en el menú. Si la opción está en un submenú, este índice tiene el siguiente formato: 
	
	```txt
	<Number of main option>.<Number of submenu option>
	```
Por ejemplo, "4.5" para la opción 5 del submenú 4. 




**`<Control>: (Nombre del control)`**

Nombre del control utilizado. 







<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Se tienen en cuenta todas las opciones del menú contextual: 

	- Opciones del menú del sistema. 

	- Opciones de menú mostradas específicas para los AAF (Características de la aplicación automatic). 

	- Opciones personalizadas del menú contextual. 




- Sólo se enumeran las opciones sobre las que se puede hacer clic. 

- Si la control puede mostrar varios menús, se vuelve al último menú mostrado. 
	Por ejemplo, en una control Tabla: 

	- Por Default, las opciones del menú de control Tabla están listadas. 

	- Si se muestra el menú contextual de una columna, la función devolverá las opciones del menú contextual de la columna.

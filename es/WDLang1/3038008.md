


## MenuIsMarked( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite saber si la marca ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Marque.gif) aparece delante de una opción de menú (menú emergente o menú desplegable).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo
<a class="notetitle" target="_blank" href="$DOC$=1000003038008&name=menuismarked_funcion&product=WD">Ver ejemplos adicionales</a>

```wl
// Is a checkmark displayed in front of "OPT_IconBar"?
ResMarking = MenuIsMarked(OPT_IconBar)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = MenuIsMarked([<Window>, ] <Menu option>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se selecciona la opción de menú especificada,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Window>: (Nombre de la ventana opcional)`**

Nombre de la ventana que contiene la opción de menú. 
Si no se especifica este parámetro, **MenuIsMarked** manipular la ventana actual.

**`<Menu option>: (Nombre de la opción)`**

Nombre de la opción de menú a utilizar. Este nombre se define en la ventana de description de la opción de menú. 
Este nombre debe ser especificado sin comillas. Esta opción también puede corresponder a la secuencia de accesos directos utilizados para acceder a la opción.

<a name="NOTE0_1"></a>


## Constantes de depuración
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las siguientes constantes se utilizan para gestionar las operaciones de depuración:

| Constante | description | Utilizado por las funciones |
| --- | --- | --- |
| **AuditAll** | Todos los eventos de auditoría se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md) |
| **AuditAssertion** | Las afirmaciones están escritas en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditComment** | Los comentarios de auditoría se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditDisabled** | La auditoría dinámica no fue habilitada. | [dbgAuditStatus](../WDLang1/1000018836.md) |
| **AuditEnabled** | La auditoría dinámica se activa inmediatamente. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditStatus](../WDLang1/1000018836.md) |
| **AuditError** | Los errores no fatales no procesados se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditException** | Los errores fatales, procesados por WHEN EXCEPTION o no procesados, se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditPause** | La auditoría dinámica se interrumpe. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditStatus](../WDLang1/1000018836.md) |
| **AuditWarningAnomaly** | Los avisos de ejecución relativos a las anomalías detectadas se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditWarningPerformance** | Las advertencias de ejecución relativas al rendimiento se registran en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **AuditWarningUser** | Las advertencias de ejecución personalizadas se escriben en la auditoría dinámica. | [dbgEnableAudit](../WDLang1/1000018835.md), [dbgAuditOption](../WDLang1/1000018837.md) |
| **dbgActivateSystemDump** | Información de depuración: Habilita o deshabilita la generación de volcado de sistema para transmitir algunos problemas al Soporte Técnico. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgActivateUserDump** | Información de depuración: Activa o desactiva el volcado de memoria del usuario (Ctrl + Pausa). | [dbgInfo](../WDLang1/3014024.md) |
| **dbgCallingProcess** | Información de depuración: información sobre la llamada a Process. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgCurrentProcess** | Información de depuración: información sobre el actual Process. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgDebuggerEnabled** | Información de depuración: Se usa para saber si el depurador está habilitado o no. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgDirectorySystemDump** | Información de depuración: Directorio para la generación de los volcados del sistema. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgElement** | Información de depuración: información sobre el Process actualmente en funcionamiento. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgElementShortName** | Información de depuración: <br><br>- Nombre del control asociado al WLanguage Event que se está ejecutando,<br><br>- Nombre de la Procedure que se ejecuta. <br><br><br> | [dbgInfo](../WDLang1/3014024.md) |
| **dbgFullComponentName** | Información de depuración: Nombre completo del componente que se está ejecutando actualmente. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgLine** | Información de depuración: Número del actual Line. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgProcess** | Información de depuración: información sobre el Process. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgStack** | Información de depuración: pila de llamadas. | [dbgInfo](../WDLang1/3014024.md) |
| **dbgTypeName** | Información de depuración: deletreado nombre del tipo. | [dbgInfo](../WDLang1/3014024.md) |
| **LogAll** | El log de tiempo de ejecución tiene en cuenta todas las opciones.: Funciones WLanguage, parámetros de función, procesos. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogOption](../WDLang1/1000017136.md) |
| **LogDisabled** | El log de tiempo de ejecución no está habilitado. | [dbgLogStatus](../WDLang1/1000017138.md) |
| **LogEnabled** | El log de tiempo de ejecución está activado. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogStatus](../WDLang1/1000017138.md) |
| **LogParameters** | Los parámetros de las funciones se registran en el log de tiempo de ejecución. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogOption](../WDLang1/1000017136.md) |
| **LogPause** | El log de tiempo de ejecución no está habilitado. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogStatus](../WDLang1/1000017138.md) |
| **LogProcessOnly** | El log de tiempo de ejecución sólo recupera información sobre los procesos. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogOption](../WDLang1/1000017136.md) |
| **LogWLanguageFunctions** | Las funciones WLanguage se registran en el log de tiempo de ejecución. | [dbgEnableLog](../WDLang1/1000017137.md), [dbgLogOption](../WDLang1/1000017136.md) |

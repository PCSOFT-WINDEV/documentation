
## dbgWriteCommentAudit( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Escribe una comment en la actual auditoría dinámica. La función **dbgWriteCommentAudit** no tiene ningún efecto si la auditoría no está habilitada. 

**Observación**: La auditoría dinámica puede activarse mediante [dbgEnableAudit](../WDLang1/1000018835.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Enables the dynamic audit
dbgEnableAudit()

// Writes a comment to remind that if the fDelete function generates an error, 
// it does not matter
dbgWriteCommentAudit("The call to fDelete can generate an error " + ...
	 "when the file does not exist; this error can be ignored.")
fDelete("dontexist.txt")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`dbgWriteCommentAudit(<Comment>)`
---

**`<Comment>: (Cadena de caracteres)`**

comment que debe ser escrita en el archivo de auditoría.

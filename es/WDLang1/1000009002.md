


## Pruebas automatizadas: Funciones de emulación
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Las funciones de emulación se utilizan para emular los diferentes tipos de ventanas que pueden aparecer durante una prueba automatizada. Estas funciones, utilizadas en el código de la prueba, se utilizan para realizar una acción. Evitan que la prueba se bloquee. 

Las funciones de emulación son las siguientes:



|   |   |
| --- | --- |
| [Botón EmulateRadioButton](../WDLang1/1000021201.md) | Emula la selección de una opción en un control Botón de opción. |
| [EmulateActionBar](../WDLang1/1000021199.md) | Emula un Action Bar durante la ejecución de las pruebas automatizadas en un dispositivo móvil (Android o iOS). |
| [EmulateCheckBox](../WDLang1/1000021200.md) | Emula la selección de una opción en un control Casilla de verificación. |
| [EmulateCombo](../WDLang1/1000025309.md) | Emula una de las siguientes acciones en un control Combo Box: <br><br>- clic en el botón que permite desplegar la lista del control Combo Box. <br><br>- seleccionar un elemento en un control Combo Box.<br><br><br> |
| [EmulateConfirm](../WDLang1/1000009003.md) | Emula la siguiente llamada a [Confirmar](../WDLang1/3021007.md) y automáticamente hace clic en el botón pasado en parámetro. |
| [EmulateDialog](../WDLang1/1000009004.md) | Emula la siguiente llamada a [Dialog](../WDLang1/3021015.md) y automáticamente hace clic en el botón pasado en el parámetro. |
| [EmulateDnD](../WDLang1/1410088663.md) | Emula la acción de "arrastrar y soltar" en las pruebas automatizadas. |
| [EmulateError](../WDLang1/1000009005.md) | Emula la siguiente llamada a [Error](../WDLang1/3021013.md) y automáticamente hace clic en el botón "OK"". |
| [EmulateInfo](../WDLang1/1000009006.md) | Emula la siguiente llamada a [Info](../WDLang1/3021011.md) y hace clic automáticamente en el botón "OK"". |
| [EmulateInput](../WDLang1/1000009012.md) | Realiza una entrada por emulación en un control mientras comprueba si este control es un control editable. |
| [EmulateMenu](../WDLang1/1000009010.md) | Emula el siguiente menú que se abrirá y ejecuta automáticamente la opción de menú pasada como parámetro. |
| [EmulateMouse](../WDLang1/1000009011.md) | Simula un ratón Event (clic izquierdo, clic derecho, doble clic, ...). |
| [EmulateOKCancel](../WDLang1/1000009007.md) | Emula la siguiente llamada a [OKCancel](../WDLang1/3021004.md) y automáticamente hace clic en el botón pasado en parámetro. |
| [EmulatePane](../WDLang1/1000021198.md) | Emula la selección de un panel en una Pestaña, Cinta o control Barra lateral. |
| [EmulatePaneDock](../WDLang1/1000021582.md) | Emula la integración de un panel desacoplado en el associated dinámico control Pestaña. |
| [EmulatePaneUndock](../WDLang1/1000021581.md) | Emula el desacoplamiento de un panel en una control Pestaña dinámica. |
| [EmulateSimulator](../WDLang1/1000021151.md) | Emula la ventana del simulador de móvil (Android/iOS). |
| [EmulateSpeed](../WDLang1/1000022075.md) | Define la velocidad de ejecución de las funciones de emulación. |
| [EmulateWarning](../WDLang1/1000009001.md) | Emula la siguiente llamada a [Aviso](../WDLang1/3021009.md) y hace clic automáticamente en el botón "OK"". |
| [EmulateWindow](../WDLang1/1000009009.md) | Indica la prueba que se ejecutará durante la siguiente apertura de la ventana. |
| [EmulateYesNo](../WDLang1/1000009008.md) | Emula la siguiente llamada a [SíNo](../WDLang1/3021005.md) y automáticamente hace clic en el botón especificado en el parámetro. |

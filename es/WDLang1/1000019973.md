
## MultimediaControlVisible( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la visibilidad de los botones de control (Play, Pausa, Avance rápido, ...) que se encuentran en el control Multimedia.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IF MultimediaControlVisible(MM_Video) = False THEN
	MultimediaControlVisible(MM_Video) = True
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Averiguar el estado de visibilidad de los botones de control

`<Result> = MultimediaControlVisible(<Multimedia control>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los botones de control se muestran actualmente,

- <u><u><u><u>False</u></u></u></u> si los botones de control están ocultos actualmente.




**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.


<a name="SYNTAX2"></a>

### Modificando el estado de visibilidad de los botones de control

`<Result> = MultimediaControlVisible(<Multimedia control> , <Visible> [, <Timeout>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si los botones eran visibles antes de la modificación.

- <u><u><u><u>False</u></u></u></u> si los botones eran invisibles antes de la modificación.




**`<Multimedia control>: (Nombre del control)`**

Nombre de Multimedia control para usar.

**`<Visible>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> para mostrar los botones de control,

- <u><u><u><u>False</u></u></u></u> para ocultar los botones de control. 




**`<Timeout>: (Entero o duración opcional)`**

Tiempo de inactividad (en segundos) después del cual los botones de la control se ocultarán automáticamente. 

- Si este tiempo de espera es set a 0, los botones de control siempre serán Visible (o hasta que sean ocultados manualmente por el usuario). 

- Si no se especifica este tiempo de espera, la duración de la inactividad será de set a 3 segundos.


Este parámetro de duración puede corresponder a: 

- un número entero correspondiente al número de segundos,

- una variable de tipo Duration,

- la duración en un formato legible (por ejemplo: '1 s'). 


**Atención**: Este parámetro solo se tiene en cuenta si el valor de &lt;Visible&gt; es <u><u><u><u>True</u></u></u></u>.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La función no tiene ningún efecto si no se encuentra ningún botón de control en el control Multimedia (esta opción se puede configurar en el control description).

- Los botones de control que se muestran en el control pueden definirse en el description de Multimedia control (pestaña "General").

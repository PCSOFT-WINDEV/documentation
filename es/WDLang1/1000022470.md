
## CameraRunApp( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Inicia la aplicación de la cámara nativa del dispositivo para Record un video o para tomar una foto.

**Observaciones**: 

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **CameraRunApp** debe ejecutarse en un thread. 

- En una aplicación WINDEV, [CámaraCaptura](../WDLang1/1000022450.md) puede utilizarse para Record un vídeo o para tomar una foto con la cámara del dispositivo. 

- Esta función es equivalente a la función [VideoStartApp](../WDLang1/1000019368.md). 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
sPhoto is string
sPhoto = CameraRunApp(viPictureCapture)
IF sPhoto <> "" THEN
	Info("Photo saved in the file " + sPhoto)
END
```


<a name="Example2"></a>

```wl
nRecordingType is int

// Let the user choose the type of capture
FI Dialog("Do you want to take a photo or record a video?", ["A photo", ...
	"A video"], 1, 2, dlgIconQuestion) = 1 THEN
	nRecordingType = viPictureCapture
ELSE
	nRecordingType = viVideoCapture
END

// Starts the application
CameraRunApp(nRecordingType, CameraRunApp_Callback)

INTERNAL PROCEDURE CameraRunApp_Callback(sPath is string)

	// If the return is an empty string vide (""), an error occurred
	IF sPath ~= "" THEN
		Error(ErrorInfo())
		RETURN
	END

	// Message for end of recording
	Info("The file was saved here: " + sPath)
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

### Iniciando la aplicación de la cámara nativa (llamada de bloqueo)

`<Result> = CameraRunApp([<Type of capture>])`
---

**`<Result>: (Cadena de caracteres)`**



- ruta completa del archivo donde se guardó la foto o el vídeo,

- Cadena vacía ("") si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Type of capture>: (Constante opcional de tipo Integer)`**

Tipo de captura:


|   |   |
| --- | --- |
| *viPictureCapture* | Guardar una foto. |
| *viVideoCapture*<br>(Valor predeterminado) | Captura una secuencia de vídeo. |




<a name="SYNTAX2"></a>

### Iniciando la aplicación nativa de la cámara (llamada sin bloqueo)

`<Result> = CameraRunApp([<Type of capture> [, <WLanguage procedure>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardó la foto o el video.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Type of capture>: (Constante opcional de tipo Integer)`**

Tipo de captura:


|   |   |
| --- | --- |
| *viPictureCapture* | Guardar una foto. |
| *viVideoCapture*<br>(Valor predeterminado) | Captura una secuencia de vídeo. |



**`<WLanguage procedure>: (Nombre de Procedure)`**

Nombre de la WLanguage Procedure ("callback") llamada cuando se graba la foto o el vídeo.
Para obtener más información sobre este procedimiento, consulte [Parámetros del procedimiento utilizado por la función CameraRunApp](../WDLang1/1000024790.md). 



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Fotos y videos guardados
<a name="fotos_videos_guardados_ELTPARAGRAPHE000111"></a>

Los ajustes de foto y vídeo (formato, resolución, tamaño) son los definidos en la aplicación nativa de la cámara.

Para que se muestre la foto en un control Imagen, abra la descripción del control Imagen, vaya a la pestaña "Detalles" y seleccione "Orientar según los datos Exif de la imagen".
<a name="NOTE0_2"></a>


### Respaldar directorios
<a name="respaldar_directorios_ELTPARAGRAPHE000120"></a>



![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) **En las aplicaciones en el modo de aplicaciones de Windows Store**las fotos y los vídeos se guardan en el directorio "TempState" de la aplicación. 
<a name="NOTE0_3"></a>
![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 

### Funcionalidad de aplicación requerida
<a name="funcionalidad_aplicacion_requerida_ELTPARAGRAPHE000159"></a>

El uso de esta función desencadena la declaración de varias características de la aplicación en la generación de la aplicación asistente.

Funcionalidades requeridas: 

- Micrófono: Esta función permite a las aplicaciones realizar grabaciones de audio. 

- WebCam: Esta característica permite a las aplicaciones capturar vídeos o fotos.


## MCIPosition( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o cambia la posición actual en un archivo multimedia previamente abierto ([MCIOpen](../WDLang1/3051017.md)).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MCIOpen("Sound.wav", "MySound")
...
MCIPlay("MySound")
...
MCIPosition("MySound", 10)
...
MCIClose("MySound")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Modificando la posición en el archivo

`<Result> = MCIPosition(<Media file or alias> , <Position> [, <Parameters> [, <Name of the message window>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el puesto era set,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para get el número de error, utilice [MCIErr](../WDLang1/3051011.md). 
	Para obtener más detalles sobre el error, utilice [MCIMsgErr](../WDLang1/3051021.md) o [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.
	El estado reporte del comando es devuelto por [MCIRetrieve](../WDLang1/3051022.md).




**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a utilizar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Position>: (Cadena de caracteres)`**

Nueva posición en el archivo. Esta posición se expresa en el formato especificado por [MCIPositionFormat](../WDLang1/3051004.md).

**`<Parameters>: (Cadena de caracteres opcional)`**

Parámetros adicionales si es necesario. Estos parámetros se pueden conocer consultando la documentación de la función "seek" de Windows MCI..
La constante *MciWait* puede ser añadida a esta cadena. Esta constante se utiliza para indicar que el programa debe esperar al final del comando MCI para ejecutar el resto de la aplicación.. 

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.  


<a name="SYNTAX2"></a>

### Averiguar la posición en el archivo

`<Result> = MCIPosition(<Media file or alias> [, <Name of the message window>])`
---

**`<Result>: (Cadena de caracteres)`**

Posición en el fichero. Esta posición se expresa en el formato especificado por [MCIPositionFormat](../WDLang1/3051004.md).

**`<Media file or alias>: (Cadena de caracteres)`**

Nombre del archivo multimedia que se va a cerrar (AVI, MP3, etc.) o alias definido al abrir el archivo ([MCIOpen](../WDLang1/3051017.md)).

**`<Name of the message window>: (Cadena de caracteres opcional)`**

Nombre de la ventana utilizada para interceptar y gestionar los diferentes mensajes enviados por MMSYSTEM.DLL. Estos mensajes están en formato MM_xxxxxx y pueden recuperarse con [Event](../WDLang1/3015011.md). Puede por ejemplo mensajes de get sobre la gestión del CD ROM o del Joystick.  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Una vez que se ha llamado a **MCIPosition** , para asegurarse de que la posición ha sido set, se recomienda utilizar el siguiente Line de código:


```txt
WHILE MCIStatus(<File or Alias>) <> MciStatusStop
END
```

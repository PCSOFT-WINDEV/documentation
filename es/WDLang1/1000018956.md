


## SysThumbnailModifyButton( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Modifica uno de los botones que se encuentran en la miniatura de la aplicación en la barra de tareas.. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Initialize a multimedia reader
SysThumbnailAddButton("reading.ico", "Reading", Reading)
SysThumbnailAddButton("pause.ico", "Pause", Pause, sysbtnGrayed)
SysThumbnailAddButton("stop.ico", "Stop", "", sysbtnGrayed)
SysThumbnailAddButton(CompleteDir(fExeDir()) + "close.gif", "Close", "", sysbtnCloseThumbnail)

// Modify a multimedia reader
SysThumbnailModifyButton(1, "fastReading.ico", "Reading", Reading)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = SysThumbnailModifyButton(<Button number> , <Image to display> [, <Tooltip> [, <WLanguage procedure> [, <Options>]]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se ha realizado la modificación, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Button number>: (Integro)`**

Rango del botón a modificar (incluido entre 1 y 7). Al añadir un botón, [Botón SysThumbnailAddButton](../WDLang1/1000018871.md) devuelve el rango del botón añadido.

**`<Image to display>: (Cadena de caracteres Ansi o Unicode)`**

Ruta del archivo Icon que se va a mostrar. Este archivo se puede encontrar en el disco o en la biblioteca de aplicaciones (archivo WDL). 
Si no se especifica este parámetro, la Icon no se modificará. 

**Observaciones**:

- Si la Icon no es de 16x16, será redimensionada automáticamente por Windows. 

- Los siguientes formatos de Image están soportados en Windows:

	- Mapa de bits (\*.BMP)

	- Formato de intercambio de gráficos (\*.gif)

	- Grupo Mixto de Expertos en Fotografía (\*.jpg \*;.jpeg)

	- CD de fotos Kodak (\*.pcd)

	- Pincel de pintura (\*.pcx)

	- Formato Adobe Photoshop (\*.psd)

	- TrueVision TARGA (\*.tga)

	- Etiquetado en el formato de archivo Image (\*.tif ; \*.Tiff)
			Observación: la opción "tiff de mosaico" del estándar tiff 6.0 no está soportada. En este caso, le aconsejamos que guarde el Image en el estándar Tiff 5.0. 

	- Gráficos de red portátiles (\*.png)

	- MetaFicheros de Windows (\*.emf;\*.wmf)

	- Iconos (\*.ico \*;.icw)
			Observaciones: 

		- Pantalla HQ (High Quality) disponible.

		- Manejo de hojas en formato Transparent BMP (32 bits).




- Cursores (\*.cur)

- Formato portátil de Document (\*.pdf)

- Gráficos vectoriales escalables (\*.svg)

- WINDEV Image s (generado por el editor de Image, \*.WDPic).




**`<Tooltip>: (Cadena de caracteres opcional en formato Ansi o Unicode)`**

Texto de la información de la herramienta de rollover del botón. 
Si no se especifica este parámetro, el texto de la información sobre la herramienta no se modifica.. 
Si este parámetro corresponde a una cadena vacía ("""), se borrará la información de herramienta anterior.. 

**`<WLanguage procedure>: (Nombre de la Procedure opcional)`**

El nombre de WLanguage Procedure llamado durante un clic en el botón. 
No se iniciará ningún Procedure si este parámetro corresponde a una cadena vacía ("").
Si este parámetro no se especifica o si es igual a <u><u><u><u>Null</u></u></u></u>, el nombre del Procedure no será modificado. 

**`<Options>: (Constante opcional de tipo Integer)`**

Modo de gestión del botón. Las constantes disponibles son las siguientes: 


|   |   |
| --- | --- |
| *sysbtnActive*<br>(Valor predeterminado) | El botón es Active. |
| *sysbtnCloseThumbnail* | Al hacer clic en el botón se cierra la miniatura de la aplicación. |
| *sysbtnGrayed* | El botón es Grayed. |
| *sysbtnInactive* | No se puede hacer clic en el botón (esta constante sólo se puede usar cuando el botón se usa como Notification Icon). |
| *sysbtnInvisible* | El botón es Invisible (su ubicación está vacía en la barra de botones). |
| *sysbtnNoBackground* | Sólo se dibuja el icono del botón (no el botón Border). Esta constante puede ser usada cuando el Icon contiene el dibujo de Border. |


Si no se especifica este parámetro, las opciones siguen siendo idénticas a las definidas por [Botón SysThumbnailAddButton](../WDLang1/1000018871.md).



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Sólo 7 botones pueden ser associated con una miniatura.

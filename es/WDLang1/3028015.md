
## DDEConnect( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Establece una conexión DDE entre el programa actual y el destinatario según un tema determinado.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Open a DDE connection
ConnectionNum = DDEConnect(ServerName, "MyTopic")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = DDEConnect(<Recipient name> , <Topic>)`
---

**`<Result>: (Integro)`**



- Identificador de conexión (utilizado en las funciones DDE),

- 0 si la conexión ha fallado ([DDEError](../WDLang1/3028006.md) contiene el error reporte).




**`<Recipient name>: (Cadena de caracteres)`**

Nombre del programa destinatario (sin extensión o ruta). El programa receptor debe ser accesible (en el directorio actual o en un directorio accesible por PATH).

**`<Topic>: (Cadena de caracteres)`**

Tema de conexión.

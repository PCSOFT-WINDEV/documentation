


## TableMoveBranch (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Mueve el contenido de una fila así como toda su estructura de árbol descendente en una control Tabla TreeView. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Positions the content of branch 1 
// as child of branch 19 in the TreeView Table control
TableMoveBranch(TVT_Customer, 1, 19, tmInChild)
```



```wl
// Positions the content of "My Documents\To Do" branch 
// as child of branch 1 in the TreeView Table control
TableMoveBranch(TVT_FOLDERS, "My documents" + TAB + ...
		"To do", 1, tmInChild)
```



```wl
// Positions the content of branch 1 as root of the TreeviewTable control
TableMoveBranch(TVT_Customer, 1, Null, tmInChild)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = TableMoveBranch(<TreeView Table control> , <Element to move> , <Destination element> , <Type of move>)`
---

**`<Result>: (Integro)`**

Posición real en la que se movió el fila. Esta posición es igual a la índice del destino fila sólo si el destino es anterior al origen en el control Tabla TreeView.

**`<TreeView Table control>: (Nombre del control)`**

Nombre del control Tabla TreeView a manipular.
Si este parámetro corresponde a una cadena vacía (""), se utilizará el control al que pertenece el evento actual.

**`<Element to move>: (Entero o cadena de caracteres)`**

Rama a mover. Esta rama puede ser identificada: 

- por el índice de la rama a mover. Este índice debe estar entre 1 y el número de filas de control (devuelto por la función [TableCount](../WDLang1/3074021.md) o la [Conde](../Proprietes/2510097.md) Property). 

- por la ruta del elemento a mover. Este parámetro tiene el siguiente formato:
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```





**`<Destination element>: (Entero o cadena de caracteres)`**

La sucursal de destino. Esta rama puede ser identificada: 

- por el índice de la rama de destino. Este índice debe estar entre 1 y el número de filas de control (devuelto por la función [TableCount](../WDLang1/3074021.md) o la [Conde](../Proprietes/2510097.md) Property). 

- por la ruta del elemento de destino. Este parámetro tiene el siguiente formato:
	
	```txt
	"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
	["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
	```


- por 'Null' o cadena vacía ("") para especificar la raíz de la estructura de árbol.  




**`<Type of move>: (La constante booleana)`**

Tipo de movimiento a realizar: 


|   |   |
| --- | --- |
| *tmInChild* | La rama se mueve y será la primera rama hija de destino.. La rama desplazada se posicionará un nivel por debajo de la rama de destino en la jerarquía. |
| *tmInSibling* | La rama se mueve y será el siguiente hermano de la rama de destino.. La rama desplazada se posicionará en el mismo nivel jerárquico que la rama de destino, pero en la siguiente posición. |

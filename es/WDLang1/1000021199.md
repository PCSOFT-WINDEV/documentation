
## EmulateActionBar( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula un Action Bar durante la ejecución de las pruebas automatizadas en un dispositivo móvil (Android o iOS).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta función está disponible en WINDEV y sólo se utiliza en modo simulador. En una aplicación WINDEV, esta función no tiene ningún efecto. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Press the left button of the Action Bar
EmulateActionBar(WIN_MainWindow.ACTB_ActionBar, eabLeftButton)

// Select the top right option OPT_OptPlus
EmulateActionBar(WIN_MainWindow.ACTB_ActionBar, eabOption, OPT_OptPlus)

// Select the option at the bottom in the OPT_Menu
EmulateActionBar(WIN_MainWindow.ACTB_ActionBar, eabOption, OPT_Menu)

// Select the view #1 in the Action Bar
EmulateActionBar(WIN_MainWindow.ACTB_ActionBar, eabView, 1)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulateActionBar(<Action Bar> , <Element> [, <Advanced option>])`
---

**`<Action Bar>: (Cadena de caracteres)`**

Nombre de la Action Bar a emular.

**`<Element>: (Constante de tipo Integer)`**

Elemento a emular entre las siguientes constantes: 


|   |   |
| --- | --- |
| *eabLeftButton* | Simula una pulsación en el botón izquierdo de Action Bar. |
| *eabOption* | Simula una pulsación en una de las opciones que se encuentran en el Action Bar. &lt;Opción avanzada&gt; se utiliza para especificar el nombre de la opción a seleccionar. |
| *eabView* | Simula una prensa en una de las vistas que se encuentran en el Action Bar. &lt;Opción avanzada&gt; se utiliza para especificar el número de la vista a seleccionar. |



**`<Advanced option>: (Opcional Variable)`**

Opción adicional para especificar. 

- Si &lt;Elemento&gt; se establece en *eabOption*, este parámetro corresponde al nombre de la opción a seleccionar. 

- Si &lt;Elemento&gt; es set a *eabView*, este parámetro corresponde al número de la vista a seleccionar. 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función sólo debe utilizarse en el código de las pruebas automatizadas ejecutadas en el simulador Android o iOS.

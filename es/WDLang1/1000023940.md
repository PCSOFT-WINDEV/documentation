


## PDFReaderRotatePage (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Gira una página mostrada en el lector de PDF control. 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Rotate page 4 of the PDF_Reader control
PDFReaderRotatePage(PDF_Reader, 4, rotationClockwise)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`PDFReaderRotatePage(<PDF Reader control> , <Page> , <Direction>)`
---

**`<PDF Reader control>: (Nombre del control)`**

Nombre del control Lector PDF a manipular.

**`<Page>: (Integro)`**

Número de la Page a girar. 

**`<Direction>: (Constante de tipo Integer)`**

Sentido de giro de Page: 


|   |   |
| --- | --- |
| *rotationAnticlockwise* | Gira la Page -90 grados. |
| *rotationClockwise* | Gira la Page 90 grados. |




## Funciones de gestión de varios idiomas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>
Para gestionar las aplicaciones multilingües se utilizan las siguientes funciones:  



|   |   |
| --- | --- |
| [ChangeCharset](../WDLang1/3054001.md) | Recupera y/o cambia el personaje que set utilizó:<br><br>- por todas las fuentes de pantalla.<br><br>- por todas las operaciones realizadas en los strings de caracteres (funciones y/o operadores).<br><br><br> |
| [ChangeKeyboard](../WDLang1/3054002.md) | Obtiene o establece el lenguaje associated con el teclado. |
| [ChangeSeparator](../WDLang1/3054004.md) | Cambia el valor de los separadores (separador decimal o separador de mil) utilizados en las máscaras numéricas para la internacionalización. |
| [LanguageToCharset](../WDLang1/3054018.md) | Devuelve el carácter set correspondiente a un idioma. |
| [LanguageToName](../WDLang1/3054020.md) | Devuelve el nombre de un idioma. |
| [LanguageToNation](../WDLang1/3054019.md) | Devuelve la nación correspondiente a un idioma. |
| [LoadError](../WDLang1/3054003.md) | Carga en memoria el archivo que contiene los mensajes de error en el idioma deseado. |
| [MessTranslate](../WDLang1/3054015.md) | Recupera un mensaje encontrado en el código de un programa en el idioma actual. |
| [Nation](../WDLang1/3054014.md) | Define o identifica el idioma de visualización (subtítulos, mensajes,...) utilizado para las ventanas, páginas, informes y controles. |
| [NationFramework](../WDLang1/1000024376.md) | Define o identifica el lenguaje de la framework. |
| [NationRightToLeft](../WDLang1/1000020820.md) | Permite averiguar si el lenguaje actual está escrito de derecha a izquierda. |
| [NationToLanguage](../WDLang1/3054021.md) | Devuelve el idioma correspondiente a una nación. |
| [NationToName](../WDLang1/1000022324.md) | Devuelve el nombre de la lengua correspondiente a una nación. |
| [NationToSubLanguage](../WDLang1/1000022209.md) | Devuelve el sub-lenguaje Default correspondiente a una nación. |

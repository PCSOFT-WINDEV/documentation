


## Funciones de gestión de controles Planificador
			





Las siguientes funciones se utilizan para gestionar los controles del Programador:



|   |   |
| --- | --- |
| [Cita previa](../WDLang1/1000019273.md) | Añade una nueva Appointment a una control Planificador. |
| [Hora de trabajo del programador](../WDLang1/1000022404.md) | Se utiliza para indicar las horas de trabajo específicas por día en una control Planificador. |
| [iCalendarToScheduler](../WDLang1/1000021917.md) | Importa datos en formato iCalendar a un control Calendario. |
| [Lista de programación de citas](../WDLang1/1000019276.md) | Devuelve: <br><br>- la lista de citas encontradas entre dos fechas en un control Planificador, <br><br>- todas las citas encontradas en un control Planificador,<br><br>- un Appointment específico (seleccionado o suspendido). <br><br><br> |
| [Modo de programación](../WDLang1/1000019992.md) | Devuelve el modo de visualización de un control Planificador: día, semana, mes, .... |
| [PlanificadorBorrarRecurso](../WDLang1/1000019282.md) | Borra una Resource de una control Planificador. |
| [PlanificadorFondoColor](../WDLang1/1000019997.md) | Define una Color de fondo para un periodo de tiempo en una control Planificador (periodo de tiempo correspondiente a la hora de comer, un día festivo, etc.). |
| [PlanificadorPosiciónRecurso](../WDLang1/1000019969.md) | Muestra una Resource dada en una control Planificador o devuelve la posición de la primera Resource visible en una control Planificador. |
| [Posición del programador](../WDLang1/1000019372.md) | Muestra un control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que aparece en un control Planificador. |
| [SchedulerAddResource](../WDLang1/1000019279.md) | Añade un nuevo Resource en un control Planificador. |
| [SchedulerChangeMode](../WDLang1/1000019440.md) | Modifica el período (es decir, el número de días) que muestra un control Planificador. |
| [SchedulerDateVisible](../WDLang1/1000022876.md) | Hace visible un día o Invisible en una control Planificador. |
| [SchedulerDeleteAll](../WDLang1/1000019275.md) | Borra todas las citas y recursos de un control Planificador. |
| [SchedulerDeleteAppointment](../WDLang1/1000019274.md) | Borra una Appointment de una control Planificador. |
| [SchedulerDisplayNextPeriod](../WDLang1/1000019496.md) | Mueve el área de tiempo que se muestra en el área siguiente (por ejemplo, las próximas semanas) en un control Planificador.  |
| [SchedulerDisplayPreviousPeriod](../WDLang1/1000019495.md) | Desplaza el área de tiempo mostrada sobre el área anterior (semanas anteriores, por ejemplo) en una control Planificador. |
| [SchedulerInfoXY](../WDLang1/1000019553.md) | Devuelve el Appointment, el Resource o la fecha encontrada en un puesto determinado en un control Planificador. |
| [SchedulerPositionDateTime](../WDLang1/1000019998.md) | Muestra un control Planificador a partir de una fecha especificada o devuelve la fecha del primer día (o último día) que aparece en un control Planificador. |
| [SchedulerSelectedRange](../WDLang1/1000022550.md) | Devuelve la fecha y hora de inicio o de fin de la franja horaria seleccionada en una control Planificador. |
| [SchedulerSelectMinus](../WDLang1/1000019286.md) | Deselecciona un Appointment de un control Planificador. |
| [SchedulerSelectPlus](../WDLang1/1000019285.md) | Selecciona un Appointment en un control Planificador. |
| [SchedulerTimeDisplayed](../WDLang1/1000019373.md) | Define el intervalo de tiempo que muestra control Planificador en el modo Día o en el modo Semana.. |
| [SchedulerToiCalendar](../WDLang1/1000021920.md) | Exporta las citas desde un control Planificador en formato iCalendar. |
| [SchedulerWorkingHourResource](../WDLang1/1000022403.md) | Se utiliza para indicar las horas de trabajo específicas por día y por Resource en una control Planificador. |
| [Visualización del planificador](../WDLang1/1000019426.md) | Reinicia el contenido de un control Planificador desde su fuente de datos (archivo HFSQL por ejemplo). |

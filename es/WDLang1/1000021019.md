
## fSplit( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Divide un archivo en varios archivos. Los archivos creados pueden ser fusionados en un solo archivo por [fMerge](../WDLang1/1000021020.md). 


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Splits the Video.avi file
fSplit("Video.avi", 100*1024*1024, "SplitVideo")
IF ErrorOccurred THEN
	// Displays the error message
	Error(ErrorInfo())
	RETURN
END

// The "SplitVideo.001" and "SplitVideo.002" files have been created
// Delete the source file ("Video.avi")
fDelete("Video.avi")

// Uses the "SplitVideo.001" and "SplitVideo.002" files 
// to re-create the "Video.avi" file 
fMerge("SplitVideo", "Video.avi")
IF ErrorOccurred THEN
	// Displays the error message
	Error(ErrorInfo())
	RETURN
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = fSplit(<File name> , <Maximum size> [, <Output file> [, <Option> [, <Progress Bar control>]]])`
---

**`<Result>: (Integro)`**



- Número de archivos creados,

- 0 si se produce un error. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md) con la constante *errMessage*.




**`<File name>: (Cadena de caracteres)`**

Nombre completo del archivo a dividir. 

**`<Maximum size>: (Integro)`**

Tamaño máximo (en bytes) de cada archivo creado. Este tamaño debe ser superior a 0. 

**`<Output file>: (Cadena de caracteres opcional)`**

ruta completa de los archivos a crear. Los ficheros creados corresponderán al nombre especificado seguido de un ".xxx" extension donde xxx corresponde al índice del fichero creado.
Por Default,  corresponde a .

**`<Option>: (Constante opcional de tipo Integer)`**

El archivo fuente se guarda durante la operación de partición. Tiene la posibilidad de especificar la siguiente opción de partición: 


|   |   |
| --- | --- |
| *fDeleteSource* | El archivo fuente se borrará a medida que avance la operación de división.<br><br>Atención: Si se produce un error, el archivo fuente se perderá y no podrá ser recuperado. |



**`<Progress Bar control>: (Nombre de control opcional)`**

Nombre de control Barra de progreso, utilizado para mostrar el progreso de la operación de división. 


<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- [fOpen](../WDLang1/3036036.md) no puede utilizarse en un fichero de varias partes. Antes de utilizar [fOpen](../WDLang1/3036036.md), hay que volver a crear el archivo original con [fMerge](../WDLang1/1000021020.md). 
	Observación: Cuando se utiliza sobre un trozo de expediente, [fOpen](../WDLang1/3036036.md) permite acceder al subfichero correspondiente al trozo de expediente. 

- La función devolverá un error si un fichero existente y uno de los subficheros a crear tienen el mismo nombre.

- El número de archivos que se pueden crear no puede ser superior a 10 000.

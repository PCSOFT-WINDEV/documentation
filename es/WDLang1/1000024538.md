


## &lt;Document Variable&gt;.Deshacer (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Deshace la última acción realizada sobre una Variable de tipo [Document](../WDLang1/1000022461.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
MyDocument is Document

// Undo the last action performed on the document
MyDocument.Undo()
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Document>.Undo()`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si el último cambio fue deshecho, 

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Document>: (Variable de tipo Document)`**

Nombre de la variable de tipo [Document](../WDLang1/1000022461.md) que se utilizará.

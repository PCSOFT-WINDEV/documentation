
## SpeechRecognitionDeleteCommand( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Borra un comando de voz previamente añadido con [SpeechRecognitionAddCommand](../WDLang1/1000020014.md).

**Observaciones**: 

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Se debe instalar un programa de reconocimiento de voz en el ordenador en tiempo de ejecución.. Un programa de reconocimiento de voz se suministra con Windows Vista. Para los demás sistemas, se debe instalar "Dragon Naturally Speaking" (las pruebas se ejecutan con la versión 9 "Preferred" o "standard").





<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Start the "Proc_Anim_Window" procedure if the voice command is recognized
Voice1ID is int
Voice1ID = SpeechRecognitionAddCommand("Proc_Anim_Window", "Animate the window")
...
// Delete the voice command
SpeechRecognitionDeleteCommand(Voice1ID)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`SpeechRecognitionDeleteCommand(<Identifier of the voice command>)`
---

**`<Identifier of the voice command>: (Integro)`**

Identificador del comando de voz a eliminar. Este identificador fue devuelto por [SpeechRecognitionAddCommand](../WDLang1/1000020014.md).

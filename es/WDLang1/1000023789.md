


## &lt;Chrono&gt;.End (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Detiene un cronómetro y devuelve el tiempo transcurrido (en milisegundos) desde que se inició el cronometraje (con la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md)). El cronómetro se puede reiniciar con la función [&lt;Chrono&gt;.Resume](../WDLang1/1000023792.md), si es necesario. 








<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo






```wl
MyChrono1 is Chrono
MyChrono2 is Chrono
// Start the first stopwatch 
MyChrono1.Start()
// Start the second stopwatch 
MyChrono2.Start()
// Stop the first stopwatch
DurationChronoStart is Duration = MyChrono1.End()
MyFunction()
// Stop the second stopwatch
DurationChronoStart is Duration = MyChrono2.End()
Info("The MyFunction() process lasted " + DurationMyFunction, ...
	"The whole process lasted " + DurationChronoStart)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Detener el cronometraje iniciado con una variable de tipo Chrono

`<Result> = <Stopwatch>.End()`
---

**`<Result>: (Duración)`**

Tiempo transcurrido (en milisegundos) desde la última llamada a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) para el cronómetro especificado.

**`<Stopwatch>: (Variable de tipo Chrono)`**

Nombre de la variable de tipo [Chrono](../WDLang1/1000023560.md) que corresponde al cronómetro a utilizar.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Varios
<a name="varios_ELTPARAGRAPHE000303"></a>

Si se llama a la función **&lt;Chrono&gt;.End** dos veces en el mismo cronómetro sin haber llamado a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md), la duración devuelta será idéntica para las dos llamadas (el cronómetro se detiene al llamar a la función **&lt;Chrono&gt;.End** por primera vez).
<a name="NOTE0_2"></a>


### Error
<a name="error_ELTPARAGRAPHE000322"></a>

Si no se llama a la función [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) antes de la función **&lt;Chrono&gt;.End**, se produce un error WLanguage (error de tipo "No se ha iniciado el cronómetro").
<a name="NOTE0_3"></a>


### Hilos y tareas paralelas
<a name="hilos_tareas_paralelas_ELTPARAGRAPHE000339"></a>

Cada hilo (y cada tarea paralela) tiene sus propios cronómetros. 

Dos hilos distintos (o dos tareas paralelas distintas) pueden realizar cronometrajes separados utilizando el mismo número de cronómetro. 

El error WLanguage "No se ha iniciado el cronómetro" solo se mostrará si no se llama a las funciones [&lt;Chrono&gt;.Start](../WDLang1/1000023788.md) y **&lt;Chrono&gt;.End** en el mismo hilo.

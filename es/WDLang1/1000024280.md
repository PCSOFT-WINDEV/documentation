


## &lt;Tabla&gt;.RestorePositionAndSelection (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Restaura la posición de la barra de desplazamiento en una Tabla o control Tabla TreeView así como los elementos seleccionados.. Estos elementos han sido guardados por [&lt;Tabla&gt;.SavePositionAndSelection](../WDLang1/1000024283.md).


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
SaveTable is string
SaveTable = TABLE_Table1.SavePositionAndSelection()
TABLE_Table1.Display(taInit)
TABLE_Table1.RestorePositionAndSelection(SaveTable)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Table control>.RestorePositionAndSelection([<Save>])`
---

**`<Table control>: (Nombre del control)`**

Nombre del control a manipular. Este control puede corresponder a: 

- un control Tabla.

- un control Tabla TreeView.




**`<Save>: (Cadena de caracteres opcional)`**

Guardar características devueltas por [&lt;Tabla&gt;.SavePositionAndSelection](../WDLang1/1000024283.md). Si no se especifica este parámetro:

- Se utiliza el último guardado realizado por [&lt;Tabla&gt;.SavePositionAndSelection](../WDLang1/1000024283.md) en la control (si existe).

- fSi la opción "Recordar posición y selección" está marcada en el control description (pestaña "Detalles"), se utiliza el último valor guardado.

- De lo contrario, se produce un error.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


### Condiciones de uso
<a name="condiciones_uso_ELTPARAGRAPHE000217"></a>

La función **&lt;Tabla&gt;.RestorePositionAndSelection** puede utilizarse en:

- una Tabla o control Tabla TreeView basada en un archivo de datos.

- una Tabla o control Tabla TreeView rellenada programáticamente.

- una selección única o selección múltiple control.



<a name="NOTE0_2"></a>


### Casos especiales
<a name="casos_especiales_ELTPARAGRAPHE000233"></a>

- El nombre de la control utilizada para guardar ([&lt;Tabla&gt;.SavePositionAndSelection](../WDLang1/1000024283.md)) puede diferir del nombre utilizado para la operación de restauración. Esta característica puede ser useful para duplicar la posición y la selección en dos controles de la Tabla que se muestran alternativamente por ejemplo.

- Si el control está vacío, la función **&lt;Tabla&gt;.RestorePositionAndSelection** no tiene ningún efecto.

- La función **&lt;Tabla&gt;.RestorePositionAndSelection** no funciona en controles Tabla basados en archivo de datos con rebote (es decir, controles Tabla sin barra de desplazamiento proporcional).



<a name="NOTE0_3"></a>


### Equivalencia
<a name="equivalencia_ELTPARAGRAPHE000254"></a>

Las funciones [&lt;Tabla&gt;.SavePositionAndSelection](../WDLang1/1000024283.md) y **&lt;Tabla&gt;.RestorePositionAndSelection** equivalen a la opción "Recordar posición y selección" disponible en la pestaña "Destalles" de la descripción del control.

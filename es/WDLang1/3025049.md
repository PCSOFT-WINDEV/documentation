
## RTFToHTML (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve la cadena HTML correspondiente a la cadena RTF especificada.


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) 
```wl
// MyRTF_String is RTF string
// sHTML is HTML string
// HTM_HTML1 is HTML control
sHTML = RTFToHTML(MyRTF_String)
sTmp is string = "<HTML>" + sHTML + "</HTML>"
HTM_HTML1 = sTmp
```


<a name="Example2"></a>


![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) 
```wl
// EDT_RTF is RTF control 
// sHTML is HTML string
// HTM_HTML1 is HTML control
// Use the management of tables and images (version 23 and later)
sHTML = RTFToHTML(EDT_RTF, rthOptionNone)
sTmp is string = "<HTML>" + sHTML + "</HTML>"
HTM_HTML1 = sTmp
```

<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = RTFToHTML(<RTF text or Name of RTF control> [, <Options>])`
---

**`<Result>: (Cadena de caracteres)`**

Texto en formato HTML

**`<RTF text or Name of RTF control>: (Cadena de caracteres)`**



- Nombre de RTF control, 
	![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este tipo de parámetro no es compatible. 

- Cadena de caracteres en RTF.  




**`<Options>: (Constante de tipo Integer)`**

Opciones de conversión: 


|   |   |
| --- | --- |
| *rthOptionCompatible* | Permite restablecer el modo de funcionamiento de la versión 22 (y anteriores). En ese caso: <br><br>- No se soportan las tablas. <br><br>- Si las imágenes han sido incluidas en el RTF (y no relacionado al RTF Document), aparecerán como un conjunto de números en el HTML Document.<br><br><br> |
| *rthOptionNone*<br>(Valor predeterminado) | Nuevo modo de conversión utilizado para gestionar las imágenes y las tablas. <br><br>Observaciones: <br><br>- El nuevo modo de conversión sólo está disponible en Windows. <br><br>- Las tablas anidadas no son compatibles. <br><br><br> |


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no está disponible. En estas plataformas: 

- No se soportan las tablas. 

- Si las imágenes se han integrado en el contenido RTF (y no relacionado al RTF Document), aparecerán como una serie de números en el HTML Document.

## Observaciones


### Codificación de caracteres
<a name="codificacion_caracteres_ELTPARAGRAPHE000109"></a>

Mediante Default, los caracteres especiales y acentuados no se codifican en la cadena HTML devuelta por **RTFToHTML**. Dependiendo de cómo se utilice el contenido HTML, puede ser necesario codificarlo. Por ejemplo, si el contenido HTML devuelto se asigna al editor HTML control, habrá que convertirlo a UTF-8 con [StringToUTF8](../WDLang1/3024042.md).

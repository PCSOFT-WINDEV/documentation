
## BitcoinValidAddress (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Comprueba si la dirección de un monedero Bitcoin es válida.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Verify Bitcoin wallet address 
IF NOT BitcoinValidAddress(EDT_Address) THEN
	Error()
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = BitcoinValidAddress(<Address>)`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si la dirección es válida, 

- <u><u><u><u>False</u></u></u></u> en caso contrario. La variable [ErrorOccurred](../WDLang1/3087001.md) se establece en <u><u><u><u>True</u></u></u></u> y la función [ErrorInfo](../WDLang1/3013008.md) devuelve el mensaje de error correspondiente.




**`<Address>: (Cadena de caracteres)`**

Dirección del monedero Bitcoin a verificar. Esta dirección puede ser de cualquier tipo:

- P2PKH (Pay To Public Key Hash, comienza por "1"),

- P2SH (Pay To Script Hash, comienza por "3"),

- P2WSH (Pay To Witness Script Hash, comienza por "3"),

- Segwit v0 o P2WPKH (Pay To Witness Public Key Hash, comienza por "bc1q"),

- Segwit v1 o P2TR (Pay to Taproot, comienza por "bc1p"). 






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Esta función solo comprueba la validez de la dirección de un monedero Bitcoin, no la validez de la clave pública o privada de un monedero Bitcoin.

- La función no comprueba si la dirección corresponde a un monedero Bitcoin que ya existe.

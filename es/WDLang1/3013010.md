
## WINDEVVersion( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de versión de la DLL wdxxxvm.

![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Devuelve el número de versión del archivo wdxxxjava.jar.

**Observación**: Esta función es equivalente a la función [ExeInfo](../WDLang1/3035001.md) con la constante *exeVersion* .


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
```wl
Info("WINDEV version: " + WINDEVVersion())
```

## Sintaxis

`<Result> = WINDEVVersion([<Version>])`
---

**`<Result>: (Cadena de caracteres o constante)`**



- Número de versión del elemento solicitado.

- Constante que corresponde a la framework utilizada. 




**`<Version>: (Constante opcional de tipo Integer)`**

Versión solicitada:


|   |   |
| --- | --- |
| *versionFramework* | Devuelve el número de versión de PC SOFT framework ("01F28 0039g" por ejemplo).<br><br> |
| *versionLanguage* | Devuelve una constante que corresponde al idioma de la framework: <br><br>- *languageEnglish*: Marco en inglés.<br><br>- *languageChinese*: Marco de trabajo en chino. <br><br>- *languageFrench*: Marco de trabajo en francés. <br><br><br><br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) No disponible. |
| *versionVM*<br>(valor predeterminado) | Devuelve el número de versión de la DLL wdxxxvm ("28.01Dn" por ejemplo).<br><br>![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png) Devuelve el número de versión del archivo wdxxxjava.aar.<br>Para get la versión de este archivo, simplemente:<br><br>1. Abra el archivo ".jar" o ".aar" (archivo zip),<br><br>2. Vaya a la carpeta "\\res\\raw\\",<br><br>3. Abrir "build_info.properties". Este archivo de texto contiene (por ejemplo): <br>	<br><pre><code>major.number=25<br>	minor.number=0<br>	build.number=341<br>	revision.number=8</code></pre><br><br><br><br><br><br>Si se genera un archivo Android APK/AAB con esta versión framework , **WINDEVVersion** devuelve "25.0.341.8". |

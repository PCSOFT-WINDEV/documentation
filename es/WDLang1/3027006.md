
## IntegerToTime (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte enteros a horas en formato HHMMSSCC.




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
IntegerToTime(0)	 // Returns "00000000"

// Wait 4 seconds
Start, t are int
Start = TimeToInteger(TimeSys())
LOOP
	t = TimeToInteger(TimeSys()) - Start
	// Display the time elapsed between 0 and 4s
	Trace(IntegerToTime(t))
	// Exit if the 4 seconds have elapsed
	IF t > 400 THEN BREAK
END
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = IntegerToTime(<Value>)`
---

**`<Result>: (Cadena de caracteres o variable de tipo Time)`**

Cadena de caracteres en formato HHMMSSCC donde HH representa las horas, MM los minutos, SS los segundos y CC las centésimas de segundo.

**`<Value>: (Entero)`**

Número de centésimas de segundo transcurridas desde la medianoche (00:00:00:00). El 0 corresponde a la medianoche. &lt;Valor&gt; no puede exceder 8 639 999 (24 horas - 1 centésima de segundo).  



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones




### Formato HHMMSSCC
<a name="formato_hhmmsscc_ELTPARAGRAPHE000081"></a>

Para obtener la hora en un formato distinto a HHMMSSCC, debe definir el formato de la cadena utilizando las funciones [TimeToString](../WDLang1/3027023.md), [Right](../WDLang1/3024016.md), [Left](../WDLang1/3024001.md), etc.

Se eligió el formato HHMMSSCC porque permite comparar dos horas con una precisión de centésimas de segundo.
<a name="NOTE0_2"></a>




### Validez de la hora
<a name="validez_hora_ELTPARAGRAPHE000100"></a>

La validez de la hora pasada como parámetro se verifica. Si la hora no es válida, se muestra un mensaje. La validez de las horas se puede comprobar con la función [TimeValid](../WDLang1/3027021.md).

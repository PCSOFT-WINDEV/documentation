## Presentación
<a name="presentacion_ELTTEXTE000160"></a>
Al escribir un nuevo mensaje destinado al usuario final, WINDEV o WINDEV Mobile muestra automáticamente la siguiente ventana.<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant_question.gif)<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Assistant_Saisie.gif)
 Esta ventana permite configurar:

1. La imagen del cuadro de diálogo. Este Image puede ser un Image preestablecido o un Image específico, accesible desde su ordenador.

2. El texto de la pregunta, en los diferentes idiomas del proyecto.

3. Los diferentes botones que aparecen en el cuadro de diálogo.

	- El botón "Añadir" sirve para añadir un botón.

	- El botón "Borrar" sirve para borrar el botón seleccionado.

	- El botón "Buscar" sirve para buscar una respuesta entre las existentes.

	- Los botones de flecha se utilizan para mover el botón seleccionado.




4. Las características del botón seleccionado:

	- Leyenda del botón en los diferentes idiomas: Por Default, la leyenda de los botones utiliza automáticamente el verbo que se encuentra en la pregunta.

	- Valor de retorno. Este valor se utiliza para definir la acción que se realizará cuando se haga clic en el botón.

	- Botón Validar: El botón sirve para validar el cuadro de diálogo

	- Botón Cancel: Este botón se utiliza para Cancel el cuadro de diálogo.  




5. Las características de la control Campo de entrada se pueden definir al escribir una pregunta de directiva.

## Base de datos de mensajes
<a name="base_datos_mensajes_ELTTEXTE000214"></a>
WINDEV y WINDEV Mobile proponen una lista de mensajes. Estos mensajes no se pueden borrar.

Todos los mensajes nuevos se añaden automáticamente a la base de datos de mensajes.. En Default, la base de datos de mensajes se encuentra en el directorio "Personal\\Messages" de WINDEV o WINDEV Mobile. 
Este directorio se puede modificar en las opciones del producto utilizado: 

1. En la pestaña "Inicio", en el grupo "Entorno", despliegue "Opciones" y seleccione "Opciones generales de xxx". 

2. Visualice la pestaña "Directorios".




**Para eliminar un mensaje de la base de datos de mensajes**:

1. Visualizar el editor de código.

2. En la pestaña "Código", en el grupo "Idiomas", despliegue "Traducir cadenas" y seleccione "Interrogativas directas".

3. Seleccione el mensaje que desea eliminar y haga clic en ![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=IconeSupprimerPropriete.gif).

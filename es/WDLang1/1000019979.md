


## &lt;Gráfico de organización&gt;.Occurrence ( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el número de elementos que se encuentran en una control Organigrama.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Find the element according to its name
NbElements is int = ORG_Direction.Occurrence()
FOR i = 1 TO NbElements
	IF ORG_Executive[i].Title = "Vince" THEN
		ORG_Executive[i].SubTitle = "Employee of the month"
	END 
END
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Organization Chart control>.Occurrence()`
---

**`<Result>: (Integro)`**

Número de elementos encontrados en la control Organigrama. Este número corresponde a: 

- el número de elementos añadidos por [&lt;Organigrama&gt;.add](../WDLang1/1000019975.md) y [&lt;Organigrama&gt;.AddChild](../WDLang1/1000019976.md). 

- el número de elementos añadidos a través del enlace de archivo si el control es relacionado a una fuente de datos. En este caso, estos elementos han sido mostrados por [&lt;Organigrama de la organización&gt;.display](../WDLang1/1000019974.md).




**`<Organization Chart control>: (Nombre del control)`**

Nombre de control Organigrama a utilizar.

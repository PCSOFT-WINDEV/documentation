
## SysScreenOrientation( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Permite: 

- obtener la orientación de la pantalla del dispositivo (Android, iOS o Windows Universal Phone).

- establecer la orientación de la pantalla del dispositivo (Android Phone).







<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Fill the "LIST_Orientations" control with the possibles orientations
ListAdd(LIST_Orientations, SysListScreenOrientation())

// Select the current orientation of the screen in the List Box control
LIST_Orientations = sysScreenOrientation()
```


<a name="Example2"></a>



<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

### Obtener la orientación de la pantalla

`<Result> = SysScreenOrientation()`
---

**`<Result>: (Constante de tipo Character String)`**

Valor de la orientación actual de la pantalla:


|   |   |
| --- | --- |
| *Orient0* | La pantalla está orientada en la dirección de Default. |
| *Orient90* | La pantalla gira 90° en el sentido de las agujas del reloj. |
| *Orient180* | La pantalla gira 180°. |
| *Orient270* | La pantalla se gira 90° en sentido contrario a las agujas del reloj (o 270° en sentido de las agujas del reloj). |




<a name="SYNTAX2"></a>

<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- La orientación de la pantalla no se puede cambiar en todos los tipos de dispositivos móviles. Cuando esta función no está disponible, **SysScreenOrientation** siempre devuelve *Orient0*.




## Funciones de gestión de animaciones
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000237"></a>
Se utilizan varias funciones de WLanguage para gestionar las animaciones. Es posible:

- gestionar la animación en los controles Image y en los controles Static.

- animar la modificación de los controles. 

- animar el paso de una ventana a otra.






<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Animación de los controles Image y de los controles estáticos
<a name="animacion_los_controles_image_los_controles_estaticos_ELTTEXTE000261"></a>
Estas funciones se utilizan para gestionar la animación de un Image en un control Imagen o la animación de un texto en un control Estático.


|   |   |
| --- | --- |
| [AnimationGetParameter](../WDLang1/3017003.md) | Lee el valor de uno de los parámetros de animación definidos para una Image o control Estático |
| [AnimationSetMode](../WDLang1/3017001.md) | Cambia el tipo de animación de un Image en un control Imagen o de un texto en un control Estático (los parámetros específicos del nuevo tipo se pueden cambiar con [AnimationSetParameter](../WDLang1/3017002.md)) |
| [AnimationSetParameter](../WDLang1/3017002.md) | Cambia el parámetro de animación para un Image mostrado en un control Imagen o para un texto mostrado en un control Estático |
| [WinDisableEffect](../WDLang1/1000017172.md) | Desactiva los efectos visuales del motor gráfico de WINDEV (para un uso en modo TSE por ejemplo). Esta función es useful cuando una aplicación debe ser ejecutada en un ordenador con capacidades gráficas limitadas o para el modo TSE si el ancho de banda es limitado. |


Para obtener más información, consulte [Animación de una control Estático](../WDChamp/1013160.md) y [Animar una imagen](../WDChamp/1013244.md).

## Animación de controles
<a name="animacion_controles_ELTTEXTE000285"></a>
Estas funciones se utilizan para animar la modificación de uno o más controles por programación.




|   |   |
| --- | --- |
| [AnimaciónJuego](../WDLang1/1000017387.md) | Reproduce una animación cuando se modifica un control, un grupo de control s o una ventana. |
| [AnimaciónMinFPS](../WDLang1/1000017400.md) | Configura el número mínimo de imágenes por segundo para que las animaciones (efectos de transición) estén habilitadas en los controles. |
| [AnimaciónPermitida](../WDLang1/1000017399.md) | Habilita o deshabilita las animaciones en los controles (animaciones definidas en el proyecto o por programación). |
| [AnimationPlayOnControlProperty](../WDLang1/1000017398.md) | Anima la modificación de una propiedad entera entre 2 valores. |
| [AnimationPrepare](../WDLang1/1000017386.md) | Prepara la animación en un control, un grupo de control s o una ventana. A continuación, la animación será reproducida por [AnimaciónJuego](../WDLang1/1000017387.md). |


Se utilizan varias funciones para reproducir una animación cuando cambia el estado de un control (o grupo de control s):


|   |   |
| --- | --- |
| [ControlEnabled](../WDLang1/1000017377.md) | Activa una control o un grupo de control en una ventana. Durante esta operación, se puede realizar una animación en los controles. |
| [ControlGrayed](../WDLang1/1000017375.md) | Grises un control o un grupo de control s. Durante esta operación, se puede realizar una animación en los controles. |
| [ControlInvisible](../WDLang1/1000017369.md) | Hace un control (o un grupo de controles) Invisible en una ventana. Durante esta operación, se puede realizar una animación en los controles. |
| [ControlVisible](../WDLang1/1000017338.md) | Hace un control (o un grupo de controles) Visible en una ventana. Durante esta operación, se puede realizar una animación en los controles. |

## Animación de ventanas
<a name="animacion_ventanas_ELTTEXTE000309"></a>
Estas funciones permiten animar el paso de una ventana a otra.. Para obtener más información, consulte [Animación de ventanas](../WDLang1/3038054.md).


|   |   |
| --- | --- |
| [Cierre de WinAnimation](../WDLang1/3038050.md) | Modifica el tipo y la duración de la animación utilizada al cerrar las ventanas de la aplicación. |
| [WinAnimationOpening](../WDLang1/3038051.md) | Modifica el tipo y la duración de las animaciones utilizadas al abrir las ventanas de la aplicación. |
| [WinAnimationSiguiente](../WDLang1/3038052.md) | Modifica el tipo y la duración de las animaciones utilizadas al abrir y cerrar la siguiente ventana. |

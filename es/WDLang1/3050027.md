


## Funciones estadísticas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


Las funciones estadísticas son las siguientes:



|   |   |
| --- | --- |
| [StatAverage](../WDLang1/3050036.md) | Calcula la media aritmética, geométrica o armónica de una serie de valores. |
| [StatAverageDeviation](../WDLang1/3050024.md) | Calcula la desviación media de los valores en relación con su media aritmética. |
| [StatCorrelation](../WDLang1/3050022.md) | Calcula el coeficiente de correlación entre dos series de valores. |
| [StatCovariance](../WDLang1/3050020.md) | Calcula la covarianza entre dos series de valores. |
| [StatError](../WDLang1/3050019.md) | Devuelve el número de error del último error causado por una función Statxxx ([funciones estadísticas](../WDLang1/3050027.md)). |
| [StatMax](../WDLang1/3050034.md) | Calcula el valor máximo para una serie de valores. |
| [StatMin](../WDLang1/3050037.md) | Calcula el valor mínimo para una serie de valores. |
| [StatStandardDeviation](../WDLang1/3050025.md) | Calcula la desviación estándar para una serie de valores. |
| [StatStandardDeviationP](../WDLang1/3050026.md) | Calcula la desviación estándar para una serie completa de valores. |
| [StatSum](../WDLang1/3050035.md) | Calcula la suma de una serie de valores. |
| [StatVariance](../WDLang1/3050033.md) | Calcula la desviación para una serie de valores. |
| [StatVarianceP](../WDLang1/3050030.md) | Calcula la desviación para una serie completa de valores. |

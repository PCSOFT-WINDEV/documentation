


## &lt;Identificador (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve el identificador associated con un elemento (nodo u hoja) en una control TreeView.

Ejemplo de control TreeView:<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=Arbre_Vocabulaire.gif)

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Return the identifier of "Desserts" node in the "TREE_TVRecipe" TreeView control
Res = TREE_TVRecipe.Identifier("Recipes" + TAB + "Desserts")
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = <TreeView control>.GetIdentifier(<Element path>)`
---

**`<Result>: (Cadena de caracteres)`**



- Identificador del elemento especificado,

- Cadena vacía ("") si no hay un identificador associated con el elemento o si el elemento no existe.




Si el identificador (definido por [&lt;Vista de árbol&gt;.add](../WDLang1/1000023422.md) o [&lt;Modificar](../WDLang1/1000023763.md)) no es una cadena de caracteres sino un número entero, es posible que tenga que convertir el resultado en un número entero para poder manipular este identificador ([Val](../WDLang1/3024037.md)).

**`<TreeView control>: (Nombre del control)`**

Nombre de la control TreeView a utilizar.

**`<Element path>: (Cadena de caracteres)`**

ruta completa del elemento a utilizar. Este parámetro tiene el siguiente formato:

```txt
"<Root name>" + TAB + ["<Name of 1st node>" + TAB + ...
["<Name of 2nd node>" + TAB + [...]]]"<Leaf name>"
```

## Observaciones


### Recuperación del identificador de un elemento con gestión de duplicados
<a name="recuperacion_del_identificador_elemento_con_gestion_duplicados_ELTPARAGRAPHE000191"></a>Para recuperar el identificador del elemento seleccionado teniendo en cuenta los duplicados, &lt;Elemento ruta&gt; debe corresponder al siguiente código:

```txt
<TreeView control>.Select(True)
```


Por ejemplo:

```wl
Trace(TREE_TreeView1.Select() + "/" + ...
	TREE_TreeView1.Identifier(TREE_TreeView1.Select(True)))
```

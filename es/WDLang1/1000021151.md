
## EmulateSimulator( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Emula la ventana del simulador de móvil (Android/iOS).

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) Esta función está disponible en WINDEV y sólo se utiliza en modo simulador. En una aplicación WINDEV, esta función no tiene ningún efecto. 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Change the orientation of the simulator
EmulateSimulator(esTurnScreen)
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`EmulateSimulator(<Action>)`
---

**`<Action>: (Constante de tipo Integer)`**

Acción a realizar: 


|   |   |
| --- | --- |
| *esBack* | Simula que se pulsa el botón Atrás.<br><br> |
| *esClose* | Cierra el simulador. |
| *esMenu* | Simula que se pulsa el botón Menú.<br><br> |
| *esTurnScreen* | Vuelve la pantalla redonda: cambia del modo horizontal al modo vertical (o viceversa). |





<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Esta función sólo debe utilizarse en el código de las pruebas automatic que se ejecutan en el simulador Android o iOS.




## <Word Processing>.ToPDF (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Convierte un procesador de textos Document en un archivo PDF. 

<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Exports the content of a Word Processing control to PDF
// Click code of "PDF export" button 
WP_MyDoc.ToPDF(fExeDir() + [fSep] + "wp.pdf")
```

<a name="XSYNTAX"></a>

## Sintaxis
<a name="SYNTAX1"></a>

`<Result> = <Word Processing control>.ToPDF(<Name of PDF file>)`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se creó el archivo PDF.

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Word Processing control>: (Nombre del control)`**

Nombre del control Procesador de texto a manipular.

**`<Name of PDF file>: (Cadena de caracteres)`**

Nombre y ruta completa del archivo PDF a crear. El archivo se sustituye por el nuevo si ya existe.



<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- Las fuentes OpenType (.otf) no son compatibles con la generación de PDF. 

- Las marcas de agua no son compatibles con la generación de PDF.

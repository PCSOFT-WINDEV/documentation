


## Iconizar (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Minimiza una ventana WINDEV.

**Observación**: La función **Iconizar** realiza la misma acción que el botón minimizar que se encuentra en la esquina superior derecha de las ventanas.
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Minimizes the "WIN_EditCustomer" window
Iconize(WIN_EditCustomer)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`Iconize([<Window>])`
---

**`<Window>: (Nombre de ventana)`**

Nombre de la ventana a minimizar. 

- Si este parámetro no se especifica o si corresponde a una cadena vacía ("""), la ventana actual se minimiza.

- Si la ventana minimizada corresponde a una ventana secundaria, también se minimiza la ventana principal..  






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones
Para saber si una ventana está minimizada, utilice [WinSize](../WDLang1/3038020.md).

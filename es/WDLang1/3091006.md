
## ImageInfoSave (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Guarda las modificaciones realizadas en las etiquetas Exif de un Image por [ImageInfoModify](../WDLang1/3091005.md).

**Observaciones**:

- La información Exif (EXchangeable Image File) se incluye en el mismo archivo de imagen. Esta norma define un set de etiquetas usadas para almacenar información sobre el Image: fecha, anchura, orientación, modelo de cámara, velocidad de obturación, distancia, tiempo de exposición, ....

- [dSaveImageJPEG](../WDLang1/3029038.md) también puede utilizarse para guardar los cambios realizados en las etiquetas Exif de una imagen en un control Imagen o un Variable de tipo [Image](../WDLang1/1000019650.md).

- Para manipular las etiquetas Exif de una imagen, utilice las variables **Exif** Property de [Image](../WDLang1/1000019650.md). Esta Property puede utilizarse en aplicaciones Android. 



<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
ImgPath is string = "C:\Travel\Mauritius1.JPG" 
// Modify the copyright and the description
ImageInfoModify(ImgPath, imgCopyright, "PC Soft" + DateSys())
ImageInfoModify(ImgPath, imgDescription, "Breathtaking landscape")
// Save the modification in the
// the same image file
ImageInfoSave(ImgPath)
// Close the image file
ImageInfoClose(ImgPath)
```


<a name="Example2"></a>



```wl
// Modify the comments of the image
// displayed in the "IMG_MyImageControl" control
ImageInfoModify(IMG_MyImageControl, imgComment, "Coral Reef")
// Save the modification in the
// "C:\Diving\Coral.JPG" file
ImageInfoSave(MyImageControl, "C:\Diving\Coral.JPG", 100)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`<Result> = ImageInfoSave(<Image> [, <New image> [, <Quality>]])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se guardaron las modificaciones,

- <u><u><u><u>False</u></u></u></u> en caso contrario. Para obtener más información sobre el error, utilice la función [ErrorInfo](../WDLang1/3013008.md).




**`<Image>: (Cadena de caracteres)`**

Imagen a manipular. Esta imagen puede corresponder a:

- un archivo Image (formato JPEG): especificar el nombre y el ruta completo (o relativo) del fichero. Se puede utilizar una ruta de tipo UNC. 
	En este caso, el archivo debe ser cerrado por [ImageInfoCerrar](../WDLang1/3091001.md).

- una Image mostrada en una control Imagen: especifique el nombre de la control Imagen. Si este Image no está en formato JPEG, se convertirá automáticamente a este formato.

- un Image (formato JPEG) que se encuentra en un memo de Image: especifica este memorándum de Image.

- una Image (formato JPEG) que se encuentra en una cadena binaria: especifique esta cadena binaria.




**`<New image>: (Cadena de caracteres opcional)`**

Nombre y ruta completa (o relativa) del nuevo archivo JPEG. Se puede utilizar una ruta de tipo UNC.

Si &lt;Image&gt; corresponde a un archivo Image, este parámetro es opcional. En este caso, el archivo &lt;Image&gt; será modificado.

Si &lt;Imagen&gt; corresponde a una imagen mostrada en un control Imagen, sólo se guardará la sección Visible de la imagen mostrada en el control Imagen. Las etiquetas Exif se conservarán.

**`<Quality>: (Entero opcional)`**

Calidad de la imagen guardada. Valor entre 1 y 100 (80 de forma predeterminada). Cuanto más baja sea la calidad del Image, más alta será la tasa de compresión del Image y más pequeño será el tamaño del archivo.. 

Este parámetro se tiene en cuenta sólo si &lt;Image&gt; corresponde a un Image mostrado en un control Imagen. De lo contrario, este parámetro se ignora.

## Observaciones




### Image mostrada en una control Imagen
<a name="image_mostrada_una_control_imagen_ELTPARAGRAPHE000085"></a>

[dSaveImageJPEG](../WDLang1/3029038.md) también puede utilizarse para guardar las modificaciones realizadas en las etiquetas Exif de una imagen mostrada en un control Imagen.

Las siguientes sintaxis son equivalentes:


```wl
// Save the modifications
ImageInfoSave(MyImageControl, "C:\My Images\Mauritius.JPG", 100)
```



```wl
// Save the modifications
dSaveImageJPEG(MyImageControl, "C:\My Images\Mauritius.JPG", 100, imgSaveTag)
```


**Recordatorio**: Sólo se guarda la sección Visible del Image que aparece en el control Imagen. Las etiquetas Exif se conservarán.




## dRotation (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Realiza una rotación: 

- en un control Imagen,

- en una variable de tipo [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) en una variable de tipo [picLayer](../WDLang1/1000024604.md).

<br>![](https://doc.pcsoft.fr/en-US/images/image.awp?langid=7&name=dRotation.gif)


![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png) **Atención**: Se requiere una configuración específica para usar esta función en Linux. Para obtener más información, consulte [Los dibujos](../WDLang1/3029035.md). 
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Rotate the IMG_Image1 control by 45 degrees
Result1 is boolean
Result1 = dRotation(IMG_Image1, 45)
IF Result1 = False THEN Error("The operation failed")
```

<a name="SYNTAX1"></a>

## Sintaxis

### Hacer un giro simple

`<Result> = dRotation(<Image> , <Rotation angle> [, <Options>])`
---

**`<Result>: (Booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el giro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image>: (Nombre de control, variable de tipo Image o variable de tipo picLayer)`**

Imagen a manipular. Esta imagen puede corresponder a: 

- el nombre de un control Imagen. 

- el nombre de una variable de tipo [Image](../WDLang1/1000019650.md).
	

- ![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png) el nombre de una variable de tipo [picLayer](../WDLang1/1000024604.md).




La rotación se realiza con respecto al centro de la imagen. La imagen se modifica directamente.

**`<Rotation angle>: (Real)`**

Ángulo de rotación (en grados). 

- Si el ángulo es positivo, la imagen gira en el sentido de las agujas del reloj. 

- Si el ángulo es negativo, la imagen gira en sentido contrario a las agujas del reloj.




**`<Options>: (Constante de tipo Integer)`**

![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Windows](https://doc.pcsoft.fr/ext/images/us/WINDOWS.png)![Linux](https://doc.pcsoft.fr/ext/images/us/LX.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) Opciones de rotación: 


|   |   |
| --- | --- |
| *drAdapt* | La imagen se gira y se amplía. Luego, se reduce para que corresponda al tamaño inicial de la imagen. |
| *drDefault* | La imagen se amplía (si es necesario) para que corresponda a las dimensiones de la imagen girada.<br><br>**Constante utilizada de forma predeterminada para las variables de tipo [Image](../WDLang1/1000019650.md) o [picLayer](../WDLang1/1000024604.md)**. |
| *drNoEnlarging* | La imagen gira pero no se amplía: la imagen puede truncarse.<br><br>**Constante utilizada de forma predeterminada para los controles Imagen**. |

### Hacer un giro modificando el centro de rotación

`<Result> = dRotation(<Image control> , <Rotation angle> , <X of rotation center> , <Y of rotation center> [, <Options>])`
---

**`<Result>: (booleano)`**



- <u><u><u><u>True</u></u></u></u> si se realizó el giro,

- <u><u><u><u>False</u></u></u></u> en caso contrario.




**`<Image control>: (Nombre del control)`**

Nombre del control Imagen a manipular. La rotación se realiza con respecto al centro del control Imagen. La imagen contenida en el control Imagen se modifica.

**`<Rotation angle>: (Real)`**

Ángulo de rotación (en grados).

**`<X of rotation center>: (Integro)`**

Coordenada X (en píxeles) del nuevo centro de rotación.

**`<Y of rotation center>: (Integro)`**

Coordenada Y (en píxeles) del nuevo centro de rotación. 

**`<Options>: (Constante de tipo Integer)`**

Opciones de rotación: 


|   |   |
| --- | --- |
| *drDefault* | La imagen se amplía (si es necesario) para que corresponda a las dimensiones de la imagen girada.<br><br>**Constante utilizada de forma predeterminada para las variables de tipo [Image](../WDLang1/1000019650.md) o [picLayer](../WDLang1/1000024604.md)**. |
| *drNoEnlarging* | La imagen gira pero no se amplía<br><br>**Constante utilizada de forma predeterminada para los controles Imagen**. |


![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) Este parámetro no está disponible.

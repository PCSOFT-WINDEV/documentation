


## Estructura de tareas programadas
			



<a name="NOTE1"></a>
<a name="NOTE1_1"></a>


## Presentación
<a name="presentacion_ELTTEXTE000317"></a>
El **Tareas programadas** se utiliza para definir las características de una tarea programada. Las variables de esta estructura son reiniciadas por [ProgramadorReiniciar](../WDLang1/3089012.md).

**Observación**: Al crear una nueva tarea, se utiliza la estructura **ScheduledTask** y la estructura [Activación de tareas programadas](../WDLang1/3089002.md).



<a name="NOTE2"></a>
<a name="NOTE2_1"></a>


## Estructura de tareas programadas
<a name="estructura_tareas_programadas_ELTTEXTE000341"></a>
Las diferentes variables de la estructura de **ScheduledTask** son las siguientes:


|   |   |
| --- | --- |
| Aplicación de tareas programadas | Cadena de caracteres.<br><br>Nombre y ruta del programa a ejecutar. |
| Tareas programadas ParadaParadaFinalización | Booleano.<br><br><u><u><u><u>True</u></u></u></u> si debe detenerse la tarea tan pronto como el sistema reanuda una actividad. <br>Por defecto, el valor de esta variable es <u><u><u><u>False</u></u></u></u>. |
| Tareas programadas DetenerseEnBaterías | Booleano.<br><br><u><u><u><u>True</u></u></u></u> si el sistema debe detener la tarea cuando el equipo funciona con batería. <br>Por defecto, el valor de esta variable es <u><u><u><u>True</u></u></u></u>. |
| Código de error de tarea programada | Entero.<br><br>Último código de error devuelto por la tarea (0 si la tarea se completó con éxito o si el programa nunca se ha iniciado). <br>Esta Variable es de sólo lectura y se actualiza con [Programador de tareasPropiedades](../WDLang1/3089011.md). |
| ScheduledTask.comment | Cadena de caracteres.<br><br>comment associated con la tarea. Este comment será Visible en las propiedades de la tarea. Este comment puede ser multilingüe. |
| Tareas programadas.IdleWaitTimeout | Entero en minutos. <br><br>Tiempo de espera durante el cual el sistema esperará un tiempo de inactividad antes de realizar la tarea. <br>Si ScheduledTask.IdleTimeout no es set a 0, esta Variable se utiliza para especificar la cantidad de tiempo necesaria antes de que el sistema cancele la ejecución de la tarea. <br>Esta variable es set a 60 minutos por Default. |
| Tareas Programadas.IdleTimeout | Entero en minutos.<br><br>Cantidad de tiempo de inactividad del sistema requerido antes de realizar la tarea. La tarea se realizará sólo si el sistema es Inactive. <br>Ejemplo: Un escaneo antivirus se realiza todos los días a la 1:00 PM si el sistema es Inactive desde hace 30 minutos. <br>Esta variable es set a 0 (tiempo de espera inmediato) por Default. |
| Fecha de última ejecución de la tarea programada | FechaHora.<br><br>Fecha de la última ejecución de la tarea. <br>Esta Variable es de sólo lectura y se actualiza con [Programador de tareasPropiedades](../WDLang1/3089011.md). |
| ScheduledTask.MaxRunTime | Duración.<br><br>Duración máxima de la tarea. Cantidad de tiempo necesario antes de que el sistema finalice la tarea. <br>Si este Variable es set a 0, la tarea se ejecutará tanto tiempo como sea necesario. <br>Esta variable es set a 72 horas por Default.<br><br>Observación: Este Variable puede corresponder a: <br><br>- un número entero que corresponde al número de centésimas de segundo,<br><br>- una variable de tipo Duration,<br><br>- la indicación directa de la duración (1 h por ejemplo). <br><br><br> |
| Sistema de tareas programadas | Booleano.<br><br>- <u><u><u><u>True</u></u></u></u> si el equipo debe estar suspendido para ejecutar esta tarea (el sistema no debe estar suspendido),<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>Por defecto, el valor de esta variable es <u><u><u><u>False</u></u></u></u>. |
| Ejecución de tareas programadasIfLoggedOn | Booleano.<br><br>- <u><u><u><u>True</u></u></u></u> si debe ejecutarse la tarea solo cuando se inicia una sesión interactiva (con identificación del usuario) en el equipo del usuario ScheduledTask.User,<br><br>- <u><u><u><u>False</u></u></u></u> en caso contrario.<br><br><br>Esta función le permite no especificar la contraseña del usuario (sin embargo, se debe especificar el nombre de usuario).<br>Por defecto, el valor de esta variable es <u><u><u><u>False</u></u></u></u>: la tarea se ejecutará incluso si la sesión ha finalizado. <br>Observación: Este Variable requiere Windows XP o posterior (no funciona en Windows 2000). |
| Línea de comandos de tareas programadas | Cadena de caracteres. <br><br>Parámetros para pasar en el comando Line a la aplicación que se ejecutará. |
| Contraseña para tareas programadas | Cadena de caracteres.<br><br>Contraseña de usuario. Este usuario se especificó en ScheduledTask.User. Se utiliza para realizar una tarea programada de acuerdo con una cuenta de usuario.. <br>Si se modifica esta contraseña, esta modificación también se aplicará a todas las tareas de relacionado para este usuario. |
| Cuenta de desencadenamiento de tareas programadas | Entero.<br><br>Número de planes de tareas (lo que significa el número de eventos que desencadenarán la tarea). <br>Esta Variable es de sólo lectura y se actualiza con [Programador de tareasPropiedades](../WDLang1/3089011.md). |
| Tareas programadas: No ejecutar si las baterías están agotadas | Booleano.<br><br><u><u><u><u>True</u></u></u></u> si el sistema debe detener la tarea cuando el equipo funciona con batería.<br>Por defecto, el valor de esta variable es <u><u><u><u>True</u></u></u></u>. |
| Tareas programadas SiguienteEjecutarHora | FechaHora.<br><br>Fecha de la siguiente ejecución de la tarea programada. <br>Esta Variable es de sólo lectura y se actualiza con [Programador de tareasPropiedades](../WDLang1/3089011.md). |
| Directorio de trabajo de tareas programadas | Cadena de caracteres.<br><br>Directorio de trabajo de la aplicación a ejecutar (cadena vacía si el directorio de trabajo corresponde al directorio del programador (MSTASK.EXE)). <br>**Atención: Si no se especifica este directorio, corresponde al directorio de ScheduledTask.Application.** |
| Tareas programadasBorrarCuando se realiza | Booleano.<br><br><u><u><u><u>True</u></u></u></u> si debe eliminarse la tarea automáticamente cuando no hay más condiciones posibles para ejecutarla (por ejemplo, si se ha alcanzado la fecha limite de cada horario). <br>Por defecto, el valor de esta variable es <u><u><u><u>False</u></u></u></u>. |
| Usuario de tareas programadas | Cadena de caracteres.<br><br>Nombre del usuario cuya cuenta se utiliza para realizar la tarea.<br>Observación: si está usando un dominio, este dominio debe ser especificado. |

## Ejemplo
<a name="ejemplo_ELTTEXTE000365"></a>


### Ejemplo para añadir una tarea
<a name="ejemplo_para_anadir_una_tarea_ELTPARAGRAPHE000268"></a>


```wl
SchedulerReset()
// Create the scheduled task
ScheduledTask.Application = "MyExecutable.exe"
ScheduledTask.CommandLine = QryAllAboutBackup.Name
ScheduledTask.SystemOn = True
ScheduledTask.Comment = "WINDEV program"

// Create the schedules for the scheduled task
// nBackupType is a radio button proposing a backup 
// single, weekly or monthly backup...

TriggerScheduledTask.Type = nBackupType

// Execution date of the single backup
IF nBackupType = schedDaily THEN
	IF EDT_SingleDate <> "" THEN
		IF EDT_SingleTime <> "" THEN
			TriggerScheduledTask.Date = EDT_SingleDate 
			TriggerScheduledTask.Time = EDT_SingleTime
		ELSE
			Info("Enter a valid backup time.")
			RETURN
		END
	ELSE
		Info("Enter a valid backup date.")
		RETURN
	END
END
// Monthly backup
IF nBackupType = schedMonthly THEN
	IF EDT_SingleTime <> "" THEN
		TriggerScheduledTask.Day = RADIO_DayOfMonth
		TriggerScheduledTask.Time = EDT_SingleTime
	ELSE
		Info("Enter a valid backup time.")
		RETURN
	END
END
// Weekly backup
IF nBackupType = schedWeekly THEN
	IF EDT_SingleTime <> "" THEN
		TriggerScheduledTask.DayOfWeek = RADIO_DayOfWeek
		TriggerScheduledTask.Time = EDT_SingleTime
	ELSE
		Info("Enter a valid backup time.")
		RETURN
	END
END
IF NOT SchedulerAddTask("MyTask") THEN 
	Error(ErrorInfo())
ELSE
	Info("The application was added to the task manager of Windows")
END
```

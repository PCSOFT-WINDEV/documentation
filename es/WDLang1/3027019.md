
## TimeSys (Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Devuelve o modifica la hora del sistema (hora actual definida en el equipo).

**Observación**: Esta función es equivalente a la función [Now](../WDLang1/3027024.md).




<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
Res = TimeSys()        // Res = "15332105"
Res = TimeToString(TimeSys())   // Res = "15:33:21:05"
CurrentTime = Left(TimeSys(), 4)  // CurrentTime = "1533"
```

## Sintaxis

### Obtener la hora del sistema

`<Result> = TimeSys()`
---

**`<Result>: (Cadena de caracteres o variable de tipo Time)`**

Hora del sistema en formato HHMMSSCC (HH corresponde a las horas, MM a los minutos, SS a los segundos y CC a las centésimas de segundo).  




<a name="SYNTAX2"></a>
![WINDEV](https://doc.pcsoft.fr/ext/images/us/WD.png)![Java](https://doc.pcsoft.fr/ext/images/us/JAVA.png)![Código de Usuario (UMC)](https://doc.pcsoft.fr/ext/images/us/MCU.png) 
### Modificar la hora del sistema

`TimeSys(<New time>)`
---

**`<New time>: (Cadena de caracteres, variable de tipo Date o DateTime)`**

Nueva hora del sistema. Esta hora puede corresponder a: 

- una cadena de caracteres en formato HHMMSSCC (HH corresponde a horas, MM a minutos, SS a segundos y CC a centésimas de segundo). 

- una cadena de caracteres en formato AAAAMMDDHHmmSSCC (AAAA corresponde al año, MM al mes, DD al día, HH a las horas, mm a los minutos, SS a los segundos y CC a las centésimas de segundo). En este caso, solo se tiene en cuenta la hora. 

- una variable de tipo Time. 

- una variable de tipo DateTime. En este caso, solo se tiene en cuenta la hora.






<a name="NOTE0"></a>
<a name="NOTE0_1"></a>

## Observaciones


- **Atención**: Para modificar la hora del sistema, debe tener los permisos de administrador en el equipo actual. 

- Para obtener la hora en un formato diferente a HHMMSSCC, se debe definir el formato de la cadena con las funciones [DateToString](../WDLang1/3027025.md), [Right](../WDLang1/3024016.md), [Left](../WDLang1/3024001.md), etc.

- Se eligió el formato HHMMSSCC porque permite comparar dos horas con una precisión de centésimas de segundo.

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En esta versión, la función **TimeSys** solo devuelve la hora del sistema (hora en el equipo actual o en el equipo en el que se está ejecutando el navegador de Internet).

- ![Universal Windows 10 App](https://doc.pcsoft.fr/ext/images/us/UNIVERSALAPP.png) En modo de simulación, se utiliza la hora del PC en el que se ejecuta la simulación. En tiempo de ejecución, se utiliza la hora del dispositivo móvil.




## TestAddIteration( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Añade una iteración a la prueba actual y da los valores de los parámetros de prueba para esta iteración.

**Observación**: Esta función sólo puede utilizarse en el código de una prueba automatizada. Debe ser usado en la Event "Antes de la prueba".


<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// The current test performs an addition, it is described with 3 parameters (p1, p2 and res)
// The code of the test scenario is res=p1+p2
// In the code before test, add the test data
TestAddIteration(1, 0, 1)
TestAddIteration(2, 2, 4)
TestAddIteration("a", "b", "ab")
// The test scenario will be run 3 times and it will check that
// 1 + 0 = 1
// 2 + 2 = 4
// "a" + "b "= "ab"
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`TestAddIteration(<Parameter 1> [, <Parameter 2> [... [, <Parameter N>]]])`
---

**`<Parameter 1>: (Variant)`**

Primer parámetro que pasará a la prueba. Este parámetro fue definido en el "Escenario de prueba" de Event.

**`<Parameter 2>: (Variante opcional)`**

El segundo parámetro a pasar a la prueba. Este parámetro fue definido en el "Escenario de prueba" de Event.



**`<Parameter N>: (Variante opcional)`**

Parámetro Nth que se pasará a la prueba. Este parámetro fue definido en el "Escenario de prueba" de Event.


## EndTimer( Función)



<a name="XUse"></a>
<a name="Use"></a>
<a name="description"></a>
Finaliza la ejecución de una timer activada por [Timer](../WDLang1/3015008.md).
<a name="Example1"></a>
<a name="sample_code"></a>

## Ejemplo


```wl
// Procedure used to regularly display the time in TIME1
PROCEDURE Display_Time()
TIME1 = TimeSys()

// Process when opening the window or the page
// Display_Time will be automatically called every second
TimerNum is int
TimerNum = Timer("Display_Time", 100, 1)
IF TimerNum = 0 THEN
	Error("Unable to create the timer")
END
```



```wl
// Stop the timer during a click on the "StopTimer" button
EndTimer(TimerNum)
```

<a name="XSYNTAX"></a>
<a name="SYNTAX1"></a>

## Sintaxis

`EndTimer([<Timer number>])`
---

**`<Timer number>: (Entero opcional (entero obligatorio en el navegador WEBDEV))`**

Número de la timer a destruir (devuelto por [Timer](../WDLang1/3015008.md)).

Si se especifica este número, la timer seleccionada se interrumpe. La función no tiene ningún efecto si la timer no existe o si ya ha sido interrumpida.

Si no se especifica este número y si se utiliza **EndTimer** en el código de un Procedure llamado por un timer, el timer se interrumpe. De lo contrario, la función **EndTimer** no tiene ningún efecto.
